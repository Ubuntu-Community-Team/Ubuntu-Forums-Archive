### Post by liorwohl on 2010-08-22
Today while not sleeping because of jet leg I thought about my next smartphone and how nice it will be if it will have the same OS as in my desktop and netbook, with the same interface instead of the limited android or iOS.
So I came up with an idea of a new interface which simply looks good on all display sizes and i have made a quick mockup.

the problem was: the regular ubuntu interface (gnome) look good only on very wide displays and is generally not very smart at all at using limited screen space and is ugly and booring like MS Windows. the ubuntu netbook interface is way too big to fit on a smartphone because of the categories on the left, and the iOS/android interface is not very good for more then a tablet, they just dont work with real multi task and are not easy to use fast like a desktop pc.

My idea is like a combination of all of them:
instead of each app icon to be on its own like the iOS menu, the icons contained in a categories, like in netbook remix, but organized in a transparent "windows" on the desktop, with no seprate categories menu. that way you dont end up with thousands of unorganized icons on the desktop/menu (like in the iOS) and you use just a bit more space then the iOS does.

the user can right click on the desktop and create a new "window" for widgets like weather and pc temps.

this categories/widgets "windows" can be moved around workspaces, and the position of each "window" will be saved according to the resolution, so when you use your netbook display the windows will be sorted in your minimalistic way across workspaces and when plugging your 22 inch display you will see all categories and unnecessary widgets on just one big display.

when clicking on an app icon it will open just like in netbook remix: if it's too big for the display maximize it and remove all borders, and if it small, display like in regular ubuntu.
that way it will work in any display size.

the last thing is the open app list, on very small display it will collapse to a menu (like gnome Windows Selector menu) on netbook/tablet it wont show the app name (just the icon, like in netbook remix) and on very wide ones it will be just like in ubuntu.

Some of this interface advantages are very smart, like for example using your smartphone as you main computer :eek: just plug it with wireless HDMI to your big screen, and with bluetooth to your mouse and keyboard and it's just like any computer.
or buy a dumb "laptop", which don't have any processor, it will just display what your phone displays, wireless. :shock: 

here are quick mocups for desktop and smartphone displays, they are just the same and will be easy to use on both of them, no more 3 different OSs.

please tell what you think about the general idea, in my opinion its very simple and elegant solution, but maybe there is a reason why i'm the first (?) to draw it.
