### Post by Duncan J Murray on 2011-01-10
Figured it out, and thought I would post it here due to the overwhelming requests for the same unanswered problem.

Ok...

System>Administration>Synaptic package-manager -> search for human-theme

Install it and then go to

System>Preferences>Appearance and select the theme.

Hurrah!
