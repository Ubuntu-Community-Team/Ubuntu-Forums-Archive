### Post by smartboyathome on 2008-01-20
Well, I would say try GIMP, and look around for tutorials on it. Yes, it is VERY different from Photoshop, but it is just as powerful. I would advise you to use Gutsy for GIMP as well, since there are some huge differences between GIMP 2.2 (which is included with Feisty) and GIMP 2.4 (which is included with Gutsy).

Also, I would advise against GIMPshop, since it is still based on GIMP 2.2, and lacks a bunch of useful features which GIMP 2.4 has.
