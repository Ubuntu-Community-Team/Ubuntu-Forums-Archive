### Post by Swiss on 2006-03-05
Ok it goes something like this...

Goto: System>Administration>Synaptic.

Now that you are in Synaptic, goto: Settings>Repositories>And click Universe. Save, reload, and you're good.

Then after restarting Synaptic search for Kguitar. When found, check the box beside it, then click "apply" 

It will install program!

If it doesn't show up in the programs menu, goto:

Accecories>Terminal. in terminal, type the name of the program after the "username@whatever:~$" and press enter.
