### Post by hw-tph on 2006-07-28
It is actually not very hard. Install Inkscape (**sudo aptitude install inkscape**) and open it from the Graphics menu. Import your image and then select Path --> Trace Bitmap. Play around a little with the settings and you should have a decent vector version of the image.

I usually find that it is easier to trace color images as black and whites and the colorize them afterwards.
