### Post by transformania on 2012-02-12
I'm not sure how good the tutorial is for the kurianos link above is, but it's now password protected so I can't test it anyway.

I've had luck with this guide...

[http://stefanobolli.blogspot.com/2011/05/maya-2012-x64-on-ubuntu-1104.html](http://stefanobolli.blogspot.com/2011/05/maya-2012-x64-on-ubuntu-1104.html)

...including on 11.10.

His guide is for HotFix4.  I got SP1 working using this guide, and I imagine any point release of Maya 2012 will work just as well (at SP2 at the time of this post).  So, just get the latest HotFix or SP directly from Autodesk's support download page...

[http://usa.autodesk.com/adsk/servlet/ps/dl/index?siteID=123112&id=2334435&linkID=9242259](http://usa.autodesk.com/adsk/servlet/ps/dl/index?siteID=123112&id=2334435&linkID=9242259)

...as even though they are *called* HotFixes and Service Packs, they actually contain the entire app (which is why the downloads are so big).

You still need a valid license to run Maya, regardless of whether you get it like this or copy it off your install CD.
