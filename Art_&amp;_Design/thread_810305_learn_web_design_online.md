### Post by lisati on 2008-05-28
A good starting point is "anywhere" - by observing the kind of web pages that look good, and the kind of web pages that look "yuck". The next step is to ask if you want to learn HTML (or any of the other tools used to specify web page content), or if you would be content to use some kind of software-aided design (note: Yahoo/Geocities pagebuilder isn't too Ubuntu-friendly).

Perhaps some more clued-up person can help with more specific pointers.
