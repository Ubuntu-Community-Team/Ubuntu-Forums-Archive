### Post by sbussy89 on 2008-12-01
I would like to make the background of my desktop the same color as my (solid-colored) toolbar.  Is there some way I can go into GIMP and get the RGB components of my toolbar?  I've experimented with the "Select by Color" tool, which will find everything else the same color as the toolbar (using a screen shot), but I can't seem to figure out how to get it to give me the components. ### Post by oedipuss on 2008-12-01
Use the color picker tool ('O' key) for that. Check out its options too, there are some interesting ones like sample average.

Alternatively, you can install and use gcolor2 (it's in synaptic) which is a simple gnome tool that does just that, and without the need of a screenshot.

### Post by volanin on 2008-12-01
Take a screenshot with Print Screen, open the image on GIMP, select the
Color Picker Tool (click on the Toolbox Window and press **O**) and
click on the desired color. This color will become the main color.

Just click once on the color square on the Toolbox Window to open the
Color Selection Dialog. There you have a lot of information about the
color, including the hexadecimal code.

### Post by CarpKing on 2008-12-01
Open up the wallpaper changer and select "No Wallpaper."  At the bottom of this dialog there should be a square to select the color of the background; click it and then click the eyedropper button in the dialog that comes up.  When your cursor is an eyedropper, click on your panel to select that color.  This same button is found in GIMP's color dialog next to the foreground and background colors.
