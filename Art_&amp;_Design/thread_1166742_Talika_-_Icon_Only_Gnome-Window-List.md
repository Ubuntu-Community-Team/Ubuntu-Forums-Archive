### Post by dahiyavinod on 2010-01-09
Remove previous version with this command:

> sudo rm /usr/lib/gnome-panel/talika

If you build from source run configure with --prefix=/usr. Without it applet won't be listed.

> ./configure --prefix=/usr
make
sudo make install



Logout from session and login again.
