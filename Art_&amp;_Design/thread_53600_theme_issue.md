### Post by super on 2005-08-01
are they gtk or metacity themes?
then they must be packaged as tar.gz (tar+gzip) and they can be installed via the gnome theme manager. you can either point the theme manager to the file or you can drag and drop the file in the theme manager window.

i've had problems installing icon themes in gnome thru the theme manager. usually i have to extract the icon themes directly into $HOME/.icons
