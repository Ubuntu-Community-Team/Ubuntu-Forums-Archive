### Post by bvc on 2005-04-21
You can't 'install' these with the gnome-theme-manager.
The easiest way is to;
-uncompress the tarball with file-roller
-click (open) the uncompressed directory (example= fancylooks)
- press/hold Ctrl and select the first theme, then press/hold Shift and select the last theme
-drag and drop the selected themes into /home/username/.themes
