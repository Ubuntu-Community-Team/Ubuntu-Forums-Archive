### Post by suchawato on 2008-04-13
It's called a photomosaic.

You need a program to render it.
There are free one's available.
There are two parts to this:
One you need the original image you want to turn into a photomosaic, and the other is you need a library of images for the program to use for the tiles.
Look arround on Google for Photomosaic programs and you should get a hit.
I know of some windows one's however do not know of what the linux one would be.
The one I used to use was a simple .exe windows program that would probably run fine under wine.

I would reccomend getting a search started and find the program you want.

Also, once you start to render, the effect can take a very long time to render depending on your computer.
Just to keep in mind.

Cool effect though,
