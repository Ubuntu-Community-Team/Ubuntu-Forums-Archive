### Post by Szise on 2010-09-18
Ok, found the solution: 

Edit Menus
Graphics
New Item
Type - Application 
Name - Gimp (the icon should appear automatically)
Command - gimp
Ok

Now the Gimp icon can be found under Graphics in Menu
