### Post by Meep3D on 2008-01-07
I'd tend to agree.  It's best to learn to walk before you run!

I'd find a nice CSS layout template site you can use as a starting point and by playing with that you should be able to get something made quite quickly, and more importantly learn a bit about what you're doing.  View Source and steal code off of sites you like too and see how they work.

Avoid animation.  Create some jazzy graphics in Gimp and basically go to town.  It'll probably take quite a few attempts to get something you like and to learn enough to be able to achieve what you want.

Standards compliance and CSS layouts, rather than table based, are the way forwards.  It may be tempting to learn table based layouts due to the easier learning curve, but don't - take the time to learn CSS, it's well worth it in the long run.
