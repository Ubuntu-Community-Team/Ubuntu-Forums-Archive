### Post by Merk42 on 2010-01-09
Well they're both made by Adobe and only come in Windows or Mac flavors so neither run natively in Linux, but regardless of that...

Illustrator (Inkscape native equivalent) is a vector editor, deals with shapes.  The images created in it can be scaled to any size without losing quality. (look up an .svg for an example)

Photoshop (GIMP native equivalent) is a raster editor, it edits by the pixels. Most image manipulation you see is done with a raster editor.
