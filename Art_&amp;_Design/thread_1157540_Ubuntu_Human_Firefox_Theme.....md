### Post by m_2009 on 2009-05-14
I can also create a green (XP) styled theme for ubuntu if there are any requests for that.

All of the themes I have made follow the GTK/GNOME dialog box styles which Ive noticed a lot of other themes don't, meaning mine contain all the icons on the menu items still, and the icons that are on buttons within dialog boxes, as can be seen in the screenshot of the preferences window with the icons on the close button. This goes for all other buttons that should have icons such as Clear private data, the colours buttons etc.. and exit, cancel buttons.

Also I have set the menu and toolbar buttons to be handled by the OS colour scheme and not by the skin css files, meaning if you were to use the theme with the "DarkRoom" or "Dust" appearance in ubuntu then the menu bar will change colour accordingly, including any auto complete drop down lists such as forms, the search bar and the address bar drop down. The menu drop down lists follow the same rules.

Tabs will also change their colour depending on what appearance setting you have the OS set to.
