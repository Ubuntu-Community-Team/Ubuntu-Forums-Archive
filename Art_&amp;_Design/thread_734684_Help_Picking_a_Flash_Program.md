### Post by MarkBremmer on 2008-12-20
Anime Studio Pro is perfect for this kind of work. However, it does have some special effects that don't export well to .swf formats. However, it does export the gradients and that's a huge time saver when animating it. 

If you're interested, I do have a [**_tutorial series_**]("http://www.markbremmer.com/3Bpages/animestudiopro.html") on it with some free lessons. You'll be able to see it in action and  maybe see if it's something worth picking up. Anime isn't very expensive for everything that it can do. The automatic lip syncing is worth it's weight in gold.
