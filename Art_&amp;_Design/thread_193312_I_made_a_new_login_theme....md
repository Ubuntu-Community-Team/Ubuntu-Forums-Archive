Its optimized for (contrary to the original theme "LiNsta") the 1280x1024 resolution. I haven't tried it on any other resolution. If you would like to try it out, download from here: (right click save as)


**Files:**

[GdmGreeterTheme]("http://www.freemediaresources.com/GdmGreeterTheme.desktop")
[Layout XML]("http://www.freemediaresources.com/LiNsta-Gnome.xml")
[Background Image]("http://www.freemediaresources.com/bg.jpg") (from [http://aeiko.net/desktopography/](http://aeiko.net/desktopography/))
[Login BG]("http://www.freemediaresources.com/login.png")
[GIMP File for Login BG (.xcf)]("http://www.freemediaresources.com/login.xcf")

[B]
Install:[/B]

1. Create a new folder in your /usr/share/gdm/themes
2. Download all files and place in folder
3. Nav to System>Administration>Login Window, and select newly installed theme.
4. To view, logout.


If this is in the wrong section, sorry, but I hope you like it as much as I do.
