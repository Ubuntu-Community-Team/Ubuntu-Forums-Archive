### Post by Ynem on 2007-06-01
Bricscad ( [www.bricscad.com](www.bricscad.com) ) is very nice alternative. It's very easy to use if you are used to the autocad interface.It uses the same commands, icons and structures, but it's a lot cheaper.They're planning to make a native linux version of their upcomming V8 edition. For now you have to use wine.

Qcad isn't ready for more complicated drawings: it takes to much time to do simple operations, even if you know the right procedure and commands.
