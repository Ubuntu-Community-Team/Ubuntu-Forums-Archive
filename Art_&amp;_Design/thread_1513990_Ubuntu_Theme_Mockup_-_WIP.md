### Post by Siberia on 2010-06-20
I had a little free time the past few days so wanted to start on a new little design project and the first thing that came into my head was an Ubuntu redesign purely because the community are really helpful when giving feedback so I though I'd be able to get some suggestions on the first mockup version of a design I have started.

Note this is only after an hour or twos work but I'm looking to improve it as much as possible with the help of the community.

I'm trying to take it more minimal and suitable for home/corporate use simultaneously and although this is a dark theme just now I think it would equally transform into a nice light theme too. I'm trying to use as little 'shine' and gradients as I can get away with without losing any dimension to the design.

All feedback and criticism is much appreciated!

P.S. Could anyone also recommend an icon set that might blend well with this design?
