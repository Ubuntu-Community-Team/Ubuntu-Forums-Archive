### Post by smartboyathome on 2008-04-14
You can install cmatrix from the repos, and run it in a terminal and take a screenshot. Then take out the background and overlay it on your image. This can also be done with the preinstalled matrix screensaver.

### Post by smartboyathome on 2008-04-14
