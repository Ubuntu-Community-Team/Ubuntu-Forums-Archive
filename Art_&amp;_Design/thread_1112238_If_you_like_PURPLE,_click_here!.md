### Post by vegetarianshrimp on 2009-03-31
I love purple. I love it so much, I'm going to show you how to totally **[COLOR=darkorchid]PURPLE[/COLOR]** your **Ubuntu Linux** desktop.
It will end something like this:
[[IMG]http://ubuntuforums.org/attachment.php?attachmentid=108192&stc=1&thumb=1&d=1238521289[/IMG]]("http://ubuntuforums.org/attachment.php?attachmentid=108192&d=1238521289")
 
Here here's how you do it:
 
 
1. First, you go [here]("http://linux.softpedia.com/progDownload/HumanPurple-Download-46386.html") to download the purple icon set (HumanPurple) Save it to Desktop (Thanks to **[Vlad Albulescu]("http://www.gnome-look.org/usermanager/search.php?username=mrkane27&PHPSESSID=21350e85a6e335e26934391525902c50")** for making these)
 
2. Then, copy and paste each line one at a time into the Terminal (Applications > Accessories > Terminal). Replace *[username] *with your username. 
```
cd Desktop
cp HumanPurple.tar.gz /home/[username]/.icons
cd /home/[username]/.icons
tar xfz HumanPurple.tar.gz
```3. Now go to System > Preferences > Appearence. Select the Human theme (if not already selected) click Customize.
 
4. In the Colors tab, select Windows: Background. In the Color Name box, put in #EED7FD
 
5. In the Colors tab, select Selected Items: Background. In the Color Name box, put in #BE5EFD
 
6. In the Icons tab, select HumanPurple. 
 
7. That's it! you're done!
 
If you want the wallpaper that is in the thumbnail, click [here]("http://v-nessa.net/wallpaper/purple_orb.jpg"). Right click on the image, and select Set as Desktop Background.
 
-VegetarianShrimp
