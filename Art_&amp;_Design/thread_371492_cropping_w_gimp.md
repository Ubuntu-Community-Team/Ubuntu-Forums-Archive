### Post by tiggs_the_cat on 2007-02-27
> **tjtansey said:**
> Quick question,
Can you, and if so how do you crop a photo to end up with a round end product?

There is no way (that I know of) to crop an image to have rounded corners.

What you could do is to add a (white) Layer Mask to the layer, then with the Layer Mask selected pick the Rectangle Select Tool and select the area that you would like to keep. In the option of the Rectangle Select Tool chose Rounded Corners and round them as much as desired. If you are satisfied with the result you can fill the selection with black on the layer mask, clear the selection and invert the Layer Mask. Finally save your image in a file type that supports alphas (e.g. PNG). This works in the GIMP 2.3.14. :D

The first sentence in tiggs_the_cat's post is because *for your computer* all images are rectangles, *there is no such things as images with rounded corners*.
