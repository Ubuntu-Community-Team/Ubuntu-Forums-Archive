### Post by m.rankovic on 2010-11-17
Icons are Faenza [http://tiheum.deviantart.com/art/Faenza-Icons-173323228](http://tiheum.deviantart.com/art/Faenza-Icons-173323228)
To install, open Applications > Accessories > Terminal an type
```

sudo add-apt-repository ppa:tiheum/equinox
sudo apt-get update
sudo apt-get install faenza-icon-theme

```
And theme looks like Elementary, but I'm not shure...
