### Post by Darkstar_444 on 2008-10-11
Hi

Go to System/Administration/Synaptic Packet Manager and enter your password. Search for kxstitch and install.

Link to kxstitch website for screenshots/handbook/Q+A etc
[http://kxstitch.sourceforge.net/]("http://kxstitch.sourceforge.net/")

To run software you will need to go to Applications/Accessories/Terminal and type kxstitch (no caps).

Don't know if it will be able to save in the file format of your machine, but probably best bet.

Hope this helps

Darren

### Post by Sue007 on 2008-11-15
whatever anyone else may say the industry standard is Wilcom Digitizing software their beginnings were in developing software for Melco and then went out on thier own to then supply software to Barudan and Tajima under thier own respective brands. This ceased some years ago and they did a great buy back offer of theses softwares so customers could continue to upgrade thier software and get product support from Wilcom. Currently they have a new product that has intergated with corel draw its really great and can also be used to prepare artwork for screen printing. there a 3 entry levels so you can start with a basic system and up grade if you feel the need to be more creative. Im not a sales person although I might sound like one but I have used the software for some 10 years now and have looked at others but stil have not found anything better :)
