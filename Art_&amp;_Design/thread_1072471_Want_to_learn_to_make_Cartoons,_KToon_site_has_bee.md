### Post by MarkBremmer on 2009-02-19
Pencil is pretty decent - especially considering the price. If you're just animating for fun you really can't go wrong with it. 

If you plan on doing any lip syncing to dialog, it will a load of work though. If you think you might be doing that, Anime Studio Pro is a good fit. It will prevent some major headaches when working with audio because it makes it so easy.

I've actually done some tutorials in it so you can see what the programs about if you want.
