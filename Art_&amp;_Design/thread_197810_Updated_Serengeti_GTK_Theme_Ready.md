It all started with the Serengeti GTK theme and that to has received some polish.

Attached is the pallet guide and as many screen shots as space allows.
[B]
To Install:[/B] Download the attached archive and extract. Run script by executing *sudo sh install-theme.sh*.

This script will install the theme globally and modify the gdm.conf file as
required.

Optionally, you may install my exit icon (**>|**) to replace the standard bright
red button of the Human icon theme.

Enjoy!

*BTW:* To see a screen shot of the exit icon view the Serengeti wallpaper post and look to the 
upper right hand corner of the screenshot image.
