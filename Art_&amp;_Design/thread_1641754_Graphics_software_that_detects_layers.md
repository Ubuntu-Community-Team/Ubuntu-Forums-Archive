well yeah if you saved your project as *.xcf your layers would still be there when you will open it again inside gimp. Usually layers are supported by the same programs where they have been done if the save has been done into the programs specific format, like gimp's *.xcf for example without being merged previously. If the save was done in *.jpg, *.png,*.gif and so on the layers were auto merged and there are no more layers to be recognized by any programs.

If you want to remove some aspects from the picture try different types of selections like selecting by color, or freehand selection, even cloning at some extent could help you achieving that.
