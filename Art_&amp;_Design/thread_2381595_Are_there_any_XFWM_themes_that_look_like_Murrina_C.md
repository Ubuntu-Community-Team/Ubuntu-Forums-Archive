### Post by vasa1 on 2018-01-03
XFWM refers to the window manager of XFCE. And any XFWM "theme" would only style the window decorations. 

If I understand correctly, you seem to want a gtk3 theme for "Murrina Crystal". I think there's no gtk3 theme available. 

This link indicates that even the gtk2 theme was last updated eight years ago: [https://www.xfce-look.org/content/show.php/Murrina+Crystal?content=96065&action=fan&page=1](https://www.xfce-look.org/content/show.php/Murrina+Crystal?content=96065&action=fan&page=1)

If you want a consistent look for your gtk2 and gtk3 applications, you may need to use themes that are fully updated to accommodate changes introduced by GNOME frequently.
