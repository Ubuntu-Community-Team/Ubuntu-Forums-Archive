### Post by nalinde on 2007-07-08
Hi -=Viper=-. Recognize your problem actually. Do a lot o designing for web etc. and unfortunately gnomes prog. just don't have those prog and features i need.

I think that I've tried almost every program out there to build my pages with. My Personal opinion is that [screem]("http://www.screem.org") is very competent.. easy to use etc. aptana as Hairy_Palms suggested is one of my favorites also.
But if you are not satisfied with this you can do as i did: Throw away that nasty windows portion and then install vmware player or vmware server (both free as in free beer) and run your favorite web editor and design tools there in windows. Then you don't have to boot every time you want to work!
Tip: if you choose this alternative. have a look at [devilspie]("http://wiki.foosel.net/linux/devilspie") to make vm-player/server start up in a certain "working area?".

### Post by Rohen on 2007-07-13
Tried Screem... but it hasn't been updated in sometime.
Tried Bluefish... but I don't like the interface.
Tried Aptana... but it consumes too much CPU.
Tried Kompozer... but I don't like WYSIWYG editors, plus it won't open CSS documents.

After you study W3C, read up on some free e-Books to toughen up your skills, and then I recommend go looking at a website that you really like, and study it's code.
