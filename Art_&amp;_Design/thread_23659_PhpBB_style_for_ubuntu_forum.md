### Post by HungSquirrel on 2005-04-04
Install the [EditCSS](http://editcss.mozdev.org/) Firefox extension.  It allows you to preview changes to a stylesheet in realtime without actually writing to the CSS file.  Once you are satisfied with the results, copy the CSS from EditCSS and paste it into the CSS file on your server.
