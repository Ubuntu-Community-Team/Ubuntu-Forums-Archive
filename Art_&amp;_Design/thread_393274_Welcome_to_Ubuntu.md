### Post by Cheizzz on 2007-04-04
While I think it is in fact a good idea to include some welcome upon first boot, I strongly oppose the kind of welcome the mockup proposes. 
The "welcome" shown in the mockup is just some links to the various items found in the administration panel, which can be found with two clicks, even by the newest of users.

We should only do a first-boot screen *if it really adds something.* No welcome-center so we can be just like windows vista, please!
I propose a short introduction in ubuntu to explain its most basic inner workings (apt-get, etc) and it's limitations like the codec and restricted driver issues, so people  won't get lost on the first boot and come to the forums to post another "I'm going back to windows because I can't install any programs!"-topic.

A funny thing to keep in you mind: in vista this screen will come up after you've installed windows and are ready to app-install frenzy. In ubuntu however, this would be your second boot given you have installed from the CD. This could be confusing. Should the screen jump up when you boot from the CD (when you cant install anything, so it would be kind of worthless) or on the first boot after installation?

What do you guys think?
