> **WarrenSH said:**
> Nice can you make this in 1920x1080 please ;)
Hmm, not sure I have the patience to do a wallpaper that size on my tiny screen ;)

I can tell you how I did it instead. It's pretty straight forward.

1) follow this tutorial [http://www.youtube.com/watch?v=cA8haSgN1Ks](http://www.youtube.com/watch?v=cA8haSgN1Ks)
I stopped at 6mins

2) create an empty selection with rounded edges
Flood fill with a very, very light blue

3) contract selection by 2px and delete.
This should leave you with a nice outlined box. Add a vertical bar around the middle.

4) select the box with the magic wand, and expand selection by 2.
Create a new layer under the box, and flood fill selection and add gaussian blur.

5) Merge box down, and set opacity to 14,5
You should now have an empty box containing 2 smaller squares. 

6) autocrop layer
Copy this layer until you have one for each dir you want in your WP. Each box should be in it's own autocropped layer. 

7) Fill in text as desired, mergeing textlayers down as you go, like this:
Add dir text > add dir description text > merge textlayers only > set layer to 14,5 opacity

8 ) merge textlayer down with box

9) Rinse and reapeat for all boxes.
