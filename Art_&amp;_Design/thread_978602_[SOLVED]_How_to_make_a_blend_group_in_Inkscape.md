### Post by smartboyathome on 2008-11-11
I don't understand what you want. You want to blend from one shape to another? If so, you can intersect the two by copying both shapes, using ctrl+alt+v to paste them where they originally were, and then using path > intersect to intersect them. Copy and paste like before this new path, make it the color you want, set the blur you want with ctrl+shift+f, and then make sure the non blurred path is on top (you can use home on the non-blurred path). Finally, set the clip for the path by selecting both the blurred and non blurred paths, and using object > clip > set.
