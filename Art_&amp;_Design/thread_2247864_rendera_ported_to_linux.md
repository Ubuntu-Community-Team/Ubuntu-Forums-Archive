I recently decided to rewrite this old Win32 program of mine. It does what I originally wanted it to do, and is pretty much usable besides lack of 8-bit file format support and bugs I may not have found. Since I'm using Ubuntu as the dev platform I thought I would share it here. Should compile on any Linux distro, or even Win32/Mac for that matter with some effort (and a performance penalty due to native Xlib support on Linux).

The program was designed mostly with photo restoration in mind, and a lot of effort was put into the cloning, colorizing, and filters to that end.

Cheers.
