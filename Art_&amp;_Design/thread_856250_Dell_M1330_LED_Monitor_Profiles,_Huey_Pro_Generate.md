For anyone with a Dell M1330 with the LED back light display, these are monitor profiles I generated with a Huey Pro and should be close enough for someone else to with this same laptop. I have the 128mb NVIDIA card. 
The profiles are labelled b(brightness eg 100=100%) t(temperature eg 65 = 6500K) g(gamma eg 22=2.2).icc b100t65g22.icc would probably be the standard one to use.
let me know how these go you you if you use them.
Is there a more appropriate place to put these, someone should build a library website where people can upload profiles they've generated for others to download and use.
