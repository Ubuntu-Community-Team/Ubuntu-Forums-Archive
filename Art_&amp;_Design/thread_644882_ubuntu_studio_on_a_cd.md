### Post by Drunky on 2007-12-19
You can always upgrade from a current version of Ubuntu 7.10 Gusty.

[UbuntuStudio/UpgradingFromGusty:]("https://help.ubuntu.com/community/UbuntuStudio/UpgradingFromGutsy")

Open a terminal then cut and paste:
```
sudo apt-get update && sudo apt-get install ubuntustudio-desktop ubuntustudio-audio ubuntustudio-audio-plugins ubuntustudio-graphics ubuntustudio-video linux-rt
```

This will install a full blown Ubuntu Studio installation...or you can pick and choose which applications you like.
