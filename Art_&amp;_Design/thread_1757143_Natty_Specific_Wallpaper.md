### Post by sectshun8 on 2011-05-13
After just installing 11.04 I really wanted a Ubuntu wallpaper to reflect the OS I was using.  There quite a few half decent Ubuntu specific wallpapers out there, but the ones with the silhouetted Narhwal for the Natty release just werent cutting it for me.

I went around the net, found two existing brown colored Ubuntu wallapapers, snipped the Ubuntu logo from one, pasted it in another, then found a rad Narwhal illustration which is from the Unicorn vs. Narwhal flash game... removed the existing "horn" or tooth, pasted a different one from a seperate Narwhal illustration, colored it... found some cool tribal designs... added those... and this is what I have now.

Didn't know if it really qualified as "original" so I didn't post it in that large thread.. but if it belongs there, mods please move it :)

Thoughts?
