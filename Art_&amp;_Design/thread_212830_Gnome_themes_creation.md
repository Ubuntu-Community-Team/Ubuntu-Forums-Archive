### Post by mostwanted on 2006-07-10
Metacity is the window border theme, Gtk+ is the colours used in the theme and look of the widgets, icons is *drumroll* the icon set used. Themes are split this way because Gnome is split into a window manager (Metacity) and a toolkit (Gtk+). It's possible to use a different toolkit than Gtk+ in Gnome (like when you run a KDE app), but that app would still have a Metacity window border.
