### Post by TJ1916 on 2012-03-07
If you are still stuck I ran into problems with Ubuntu Tweak too but solved it by changing the permissions of the image I wanted to use. In the Others Access menu I had to set the permissions to read and write. 

The original background is /usr/share/backgrounds/warty-final-ubuntu.png. If you want to use your own image with Ubuntu Tweak it has to be in the same folder, as you mentioned. If you want your default background back just choose the above image in Ubuntu Tweak and adjust permissions if necessary.
