### Post by _sAm_ on 2010-02-15
Hi zlatarev

Raw Therapee is very good and works fine on Ubuntu(I use the 64bit version). I use it for NEF files from my D300s. RT changed from closed to open source (under GPL), so it might turn up in Synaptic in a later version of Ubuntu. [http://www.rawtherapee.com/?mitem=2](http://www.rawtherapee.com/?mitem=2) 

RT is easy to start; download the *.tgz file and unpack it. Start by doublecklick the rt file inside the new folder. You can add it to the Applications menu if you want. 
This version of RT is alpha, so not all stuff is in yet(and some don't work).

Latest version of Digikam looks better then before(with a new vignetting and HDR tool).

Latest version of Bible works also on Ubuntu; its very fast and have all the stuff you mentioned.
