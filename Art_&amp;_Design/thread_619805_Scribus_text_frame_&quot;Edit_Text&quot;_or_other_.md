### Post by ImpressMe on 2007-11-22
If you have time to fool around they try using OpenOffice Writer or even Draw. I hate the lack of aliasing of text and vector graphics, but the output can be pretty sweet. Try exporting to PDF and you'll see.

Scribus... it is a very promising program, but still not stable nor ready for prime time. Yet. They are working on it though. 

I am not sure if I understand your problems. If I do this in 1.3.3.10 everything works as expected:

1) Create text frame
2) Double click it
3) Enter some text
4) Right click box and select "Edit text"
5) Work on some more text

The style editor is nothing like Word or Writer. You are expected to use styles for formating text, not direct formating.

You create and select styles in the frame/column to the LEFT.

Styles and the text editor is among the features that the developers are working on as we speak. I am looking forward to 1.3.5 veryyy muuuch (Borat).
