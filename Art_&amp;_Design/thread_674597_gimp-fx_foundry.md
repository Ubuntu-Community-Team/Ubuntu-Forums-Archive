### Post by loell on 2008-01-21
if you like to make effects on gimp then gimpfx foundry is for you :)

to install

download the tarball or zip from [http://sourceforge.net/project/showfiles.php?group_id=208874](http://sourceforge.net/project/showfiles.php?group_id=208874)

then extract it to **/.gimp-2.4/scripts**  in your home directory

start gimp then you should see the **FX Foundry** menu in the image window ;)
