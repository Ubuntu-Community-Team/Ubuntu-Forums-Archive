### Post by Phire on 2009-02-23
ctrl+shift+[U2008] is one character. It may not be possible to ahve more than one item with the same name, so it may be better do do as the guide suggested.

For every item after the first, repeat the special character one more time.

For instance, lets suppose I'm making a third icon nameless, having done so to 2 prior ones. In this case, i would need to rename the icon ```
ctrl+shift+[U2008]ctrl+shift+[U2008]ctrl+shift+[U2008]
```to give "_ _ _" 3 blank spaces. Hope this helps.
