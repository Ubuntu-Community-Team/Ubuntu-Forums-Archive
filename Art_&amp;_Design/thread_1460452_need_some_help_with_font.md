### Post by Newfoundlander on 2010-04-24
It would be wise to go with [Weiss]("http://new.myfonts.com/fonts/linotype/weiss/weiss/") for "EvergreenPlace".

Swis721 BT looks very close for "ASSISTED LIVING" but it is difficult because the text is so small to pick out details.

The "ever caring. ever giving" font looks like one of the Garamonds but it has diamond dots and periods which is different.

Hope that helps.
