### Post by Newfoundlander on 2010-02-15
[*Scribus*]("http://www.scribus.net") is a great desktop publishing package that you can use to design all kinds of printed material.  *OpenOffice Writer* and *Inkscape* are two other applications that may work but Scribus offers the best control over text and layout.  Do a search on YouTube and you'll find lots of demonstrations and tutorials.

Install it through the Ubuntu Software Centre.

> **Gnosiz said:**

They are incompatible with no way of easily transferring files unless Publisher can save in Word format, you could load and convert it to odf with OpenOffice, and then import it into Scribus.

Inkscape is very good for one-page publications.  I use it a lot myself but it does not handle lots of text as well as a true desktop publishing app.
