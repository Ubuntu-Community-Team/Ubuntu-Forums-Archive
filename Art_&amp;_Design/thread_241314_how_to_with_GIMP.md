### Post by Aelfric5578 on 2006-08-26
Sometimes a picture in GIMP is automatically completely selected.  You can't make a smaller selection inside of a bigger one.  Try doing "Select None" before you try to use the selector tool.  You can do this by pressing "Ctrl + Shift + A" or by using the menu, Select -> None
