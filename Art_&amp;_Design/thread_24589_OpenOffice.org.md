### Post by Underdog45 on 2005-04-07
Hello Ubuntusers! I'm Dave, the proud owner of a semi-fresh Hoary install (does last weekend still count as fresh?). Anyway, I made a splash screen for OpenOffice with Inkscape and the GIMP for those who haven't turned it off or use the quick starter, and you can find it here: 

[http://www.gnome-look.org/content/show.php?content=22816](http://www.gnome-look.org/content/show.php?content=22816)

You can find instructions on how to install it for OOo 1.* or the new 2.0 if you have it already. It took a while to hack up the Ubuntu default text to make the letters in "OpenOffice.org" that aren't in the word "Ubuntu" (e.g. flipping the t upside down and moving the crossbar for the f). I think for the most part it turned out nicely, but I think the g still looks a little off.

EDIT: I made a Kubuntu version and it is here:

[http://www.kde-look.org/content/show.php?content=22830](http://www.kde-look.org/content/show.php?content=22830)
