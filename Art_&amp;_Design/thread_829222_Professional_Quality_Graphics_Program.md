### Post by Xfcn on 2008-06-18
Tutorials are a help, yes, but honestly you've GOT to play around or you'll never get anywhere. As long as you're staying away from the "System" menu and the Terminal, you won't damage your setup at all. If a tutorial points you to using the Terminal, what they're explaining is either bloody old or bleeding edge. You want to be neither, really, until you're really comfortable. Then you can feel free, but realize you're likely putting more effort into it than is really needed.

I highly encourage you to play around, ask many many questions, and look for tutorials on Google.

[This should help](http://www.google.com/search?source=ig&hl=en&rlz=&=&q=GIMP+Bevel+Tutorial&btnG=Google+Search).

### Post by marialice on 2008-06-20
Try Inkscape.

See [Inkscape for Adobe Illustrator Users](http://wiki.inkscape.org/wiki/index.php/Inkscape_for_Adobe_Illustrator_users)

> Things Illustrator can do that Inkscape cannot do:

    * Gradient meshes
    * Multiple strokes and fills for one object
    * Color management for print (ICC Profiles, etc.)
    * PMS color
    * Natively work with graphs based on data
    * Free transform and perspective transform (only via extension)
    * Blends (only via extension) 

Things Inkscape can do that Illustrator cannot:

    * Edit SVG source directly
    * Clones, tiled clones, edit clones on canvas
    * Keys to move/rotate/scale by screen pixels
    * Shapes as objects
    * Edit gradients with handles on-canvas
    * Edit nodes with keyboard
    * One-click paint bucket fill
    * Color painting over objects
