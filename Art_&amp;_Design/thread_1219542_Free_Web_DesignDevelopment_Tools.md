### Post by ajy0852 on 2009-07-23

If you're going to be in XP you may want to use Virtual PC as that makes the VHDs easier. You can use Virtualbox in either Linux or Windows though last time I tried using the vhds it asked for a Windows XP disc.

There's a nice program called "IETester" that combines the layout and javascript engines of IE 5.5, 6, 7, and 8 into one program.([link]("http://www.my-debugbar.com/wiki/IETester/HomePage")). I wish this program would work under Wine, I run it on my wife's Vista computer.

You might check out Evernote as well, having a note-taking program like this helps me be more productive. I was using this under Wine and it works pretty well, but there were just enough inconveniences to make me go back to Zim for my note-taking and clipping (with a couple custom bash scripts I cobbled together). Evernote works fine on Windows. I know Tomboy is available on Windows too, I've never tried it on that platform.

I like looking through the list of software at portableapps.com, there are a lot of useful programs there even if you're not planning to run them portably.
