First things first! This line art was created by my friend Steven Sommers aka Zeratanus. Check him out on Deviant Art here: [http://zeratanus.deviantart.com/](http://zeratanus.deviantart.com/)

The is a sort of announcement that I plan to get back into Blender Game Engine and actually use it to create a game with it's own universe, characters and everything. This is a wallpaper I made throwing together drawings of the whole cast and coloring it. There are characters created by Steve, my wife and myself. SpiL has been an on and off again interest ever since high school though it has changed many times over.

My goal with this is to create 3D backgrounds so Steve can get into creating some SpiL comics if he chooses. Soon I would like to create a 3D game that can be freely distributed. Something made with open source tools, but that also has personality and characters you can remember if not from unique design then by personality.

I'd like to get a Patreon set up for this project eventually. It would be great to start getting feedback after a few development updates.
