### Post by ericesque on 2007-11-13
So I can't stand the new Rhythmbox icon.  Yes, it was time for the old one to go.  But what we ended up with looks more like a school bell from the 50s than anything that has to do with music. 

I've put inkscape to the test to put together my own visual metaphor.   I didn't want to recycle the cd, play button, or music note metaphor that is tired at best by now.  So when I thought rhythm I thought bass.  When I thought box,  I thought speaker.  Put the two together and you get subwoofer.

The design isn't finalized.  I may still use a music note coming out of the sub or next to it to ensure new users will know that we're talking about playing music.  I also need to add the highlight stroke and tweak here or there, but it already looks nice in AWN.


[IMG]http://ekaschykresume.googlepages.com/RBpreview.png[/IMG]


Attached is the SVG.  Feel free to tweak and post your results.

### Post by ericesque on 2007-11-14
Well, to replace the default icon for Rhythmbox, the process depends on what icon set you're using.  Also, using this icon as a systray icon is going to leave you with nothing but a dark gray blob because the system would have to scale a 128x128 icon to 22x22.  Honestly, my favorite systray icon for a music player has always been a simple black music note.  But I may be able to adapt this icon to 22x22 in the near future.

For now, you could use the SVG in a dock or if you have a desktop icon for Rhytmbox, you should be able to right click on the icon, select properties, and change the icon from there.  If you post what icon set you're using I can probably help you set everything else up in the next few days.
