### Post by Glynnux on 2013-05-14
Hi OmmoJommo,
you basically use compiz to allow you to install different wallpapers for the different workspaces. Each face of the cube shows a different workspace.
PLEASE NOTE! you will not be able to see folders etc that you drop to the desktop as the wallpapers will hide the desktop (you actually need to disable showing desktop).
You should be able to configure it with the help of this video (I couldn't find the NixiePixel one that I used a few years back)

[http://www.youtube.com/watch?v=mwKfmKibmKM](http://www.youtube.com/watch?v=mwKfmKibmKM)
