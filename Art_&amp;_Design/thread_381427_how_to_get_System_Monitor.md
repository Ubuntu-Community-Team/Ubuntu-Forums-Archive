I is Beryl. Beryl is the only way I know to get reliable transparency in windows AND window borders. The only caveat is that your system will need some good graphics to get a good effect with Beryl, but it doesn't have to be over the top. I also have a PIII desktop system that has a 128 mb PCI nVidia graphics card that gets Beryl good enough to make it fun.

The PC in the pic is my laptop. 

Specs:
Intel Core 2 Duo at 2 Ghz
2 Gig RAM
nVidia graphics - 256 MB RAM
Edgy 6.10

Beryl runs very well natively on Edgy and used the nVidia driver out of the box, so to speak, and not XGL.

[Go here]("http://www.beryl-project.org/index.php") to learn more about Beryl and look around the beryl forums and WIKI. this is where you will find the most info about Beryl. Also look around the Ubuntu Forums for Beryl and you will get the idea. For some it seems hard because there is a fair amount of code to input into the xorg.conf file, but if you run Edgy and have a good graphics card, try [this link]("http://forum.beryl-project.org/viewtopic.php?f=43&t=51"). This is all I did on my lappie and it looks beautimus, as you can see.

Good luck.
