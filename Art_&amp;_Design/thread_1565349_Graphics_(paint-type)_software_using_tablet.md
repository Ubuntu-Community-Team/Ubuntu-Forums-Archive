### Post by kayosiii on 2010-09-02
a list.

Inkscape -- really a vector illustration package but has an ink tool that works great with tablet.
Gimp -- pretty much the most feature complete image editor on Ubuntu.
Mypaint -- as mentioned above &
Krita --([www.krita.org](www.krita.org))another paint program which is getting a lot of love from it's developers at the moment and is targeted towards painting.

If you are into 3D at all then Blender has tablet support and some nice sculpting tools.

Also note that Krita,Mypaint,Gimp(I think)... Support the open raster format which is a great way of getting paintings from one application to another.
