### Post by MadsRH on 2009-03-16
I got so inspired by Andreas Nilssons graphical representation of the
GNOME Release Process that I wanted to create one for Ubuntu. The
image represents the 6 months cycle, like a timeline.

I've only added actual releases and left out freezes and deadlines,
although the "Debian snail" should have shown where DebianImportFreeze
occurs, but it was by accident placed one month later! Perhaps I'll
add more details (and move the snail ;)). I started out using Andreas
Nilssons .svg image as skeleton.

[http://anotherubuntu.blogspot.com/2009/03/karmic-koala-timeline.html](http://anotherubuntu.blogspot.com/2009/03/karmic-koala-timeline.html)

[IMG]https://lists.ubuntu.com/archives/ubuntu-art/attachments/20090315/a682260b/attachment-0001.jpg[/IMG]


Best regards
