### Post by jbrice on 2011-03-01
A slightly indirect answer to your question, but if you want to learn how to build something there's no better starting point than tearing apart someone else's work to see how they did it.

And the best tool for doing that is Firefox with the Firebug plugin. Also allows you to edit the local copy of the HTML and CSS code and see what effect that has. Mess it up and a page refresh will take you back to where you were. (There is a similar thing buried in "Customize and Control" on Chrome(ium) called Developer Tools if you prefer that browser.)

Once you start building your own web pages (and even the Ubuntu default text editor - GEdit - is fine for writing/editing HTML and CSS), the same tools are brilliant for debugging your own web code and general web page forensics.
