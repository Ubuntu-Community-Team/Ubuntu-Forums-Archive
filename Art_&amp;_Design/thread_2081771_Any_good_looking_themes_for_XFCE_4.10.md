### Post by slickymaster on 2012-11-12
Take a look at Greybird-Mac 1.0 together with Faenza Cupertino icons theme.

You can download the theme [here]("http://xfce-look.org/content/download.php?content=149800&id=1&tan=45352163&PHPSESSID=fce9ea8404c8d4b65c73a4659e449e83").

And to install the icons themes just type in a terminal:

sudo add-apt-repository ppa:tiheum/equinox 
sudo apt-get update 
sudo apt-get install faenza-icon-theme
