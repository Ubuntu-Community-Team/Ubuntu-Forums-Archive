I understand what you mean about footer text not necessarily needing to "stand out" but it should still be readable.  I like the colors you're using now alot better.  much easier on the eyes

if I may make a couple more suggestions.  personally, I would trim much of the black from around the banner graphic, as your content is already pretty far down the page.  you could save nearly an inch of screen real estate just by trimming that down.  I use 1024x768 screen resolution and I can't see any of your content without scrolling down, just the headers, like "site news" on your home page or "mission statement" on your about page.  also, I notice that you are using roll-over sub navigation links on your about page.  is there a particular reason you didn't use the same color scheme as you do in the main navigation?  not that there's anything wrong with the way you have it per se, just wondering why you wouldn't be consistent with your navigation style / color scheme.

also, css provides ways to modify the display of an "active" link

take [[COLOR="Red"]_my site_[/COLOR]]("http://www.athometech.info") for example.  I use a gray background for my navigation links, and a blue background for the "active" link, or the page you are currently on.

the html for my navigation:
```

<ul class="nav">
    <li><a href="index.php">home</a></li>
    <li class="current"><a href="services.php">services</a></li>
    <li><a href="contact.php">contact</a></li>
</ul>
<ul class="subNav">
    <li class="current"><a href="services.php">repair</a></li>
    <li><a href="services.php?p=upgrade">upgrade</a></li>
    <li><a href="services.php?p=instruction">instruction</a></li>
    <li><a href="services.php?p=networking">networking</a></li>
</ul>

```

the corresponding css:
```

ul.nav {
	width: 6em;
	margin: 2.65em 1em 1em 1em;
	float: left;
	padding: 0;
	list-style-type: none;
}

ul.subNav {
	width: 30em;
	margin: 1.25em 0 -.75em -2em;
	float: left;
	list-style-type: none;
	display: inline;
}

ul.nav a {
	display: block;
	font-weight: bold;
	padding: .25em .25em .25em 1em;
	text-decoration: none;
	background-color: #1C1C1C;
	color: #FFFFFF;
}

ul.subNav li {
	display: inline;
}

ul.subNav a {
	font-weight: bold;
	padding: .25em 1em 1em 1em;
	text-decoration: none;
	background-color: #1C1C1C;
	color: #FFFFFF;
}

li.current a {
	background-color: #294366;
}

li.subNav a.current {
	background-color: #294366;
}

li.nav a:hover {
	background-color: #294366;
}

li.subNav a:hover {
	background-color: #294366;
}

```

one of my most used resources is w3schools.  they have references for many languages and very thorough examples.
more information on styling links can be found here - [http://www.w3schools.com/css/css_link.asp](http://www.w3schools.com/css/css_link.asp)

I hope I'm not coming across as rude, although I've been told I usually do.  just trying to help out
