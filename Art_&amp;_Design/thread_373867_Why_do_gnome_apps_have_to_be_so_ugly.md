### Post by jem7v on 2007-03-03
Also, not everybody writes programs with GTK2 interfaces using default icons (such as OpenOffice, which is still undergoing Gnome/KDE integration), which thus means that when you change your icon set in the theme manager it won't affect the program (because the program only uses proprietary icons).  This is more an issue with the program developers than with the Gnome developers.  Most of the Gnome programs themselves integrate very well with each other.
