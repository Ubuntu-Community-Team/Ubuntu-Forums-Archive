### Post by cph05a on 2010-06-01
I've found that there aren't any linux video editors that have everything I want by themselves, so end up using more than one depending on what I'm doing. For example, I usually do my cutting and splicing in kino because it's really really simple, but for more complex things (text overlay, chroma key, etc) I usually use cinalerra. I use kdenlive occasionally.
