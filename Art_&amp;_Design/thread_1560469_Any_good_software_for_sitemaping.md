Open a terminal.
Just write:
"sudo apt-get install sitemap"
Then go to the directory, through cd command...
Then just write "sitemap" into the terminal.
And the program generates a sitemap for the chosen folder! :D

### Post by ajy0852 on 2010-08-28
You could try searching through the software center for mind mapping. One that comes to mind is Freemind, though I haven't used it.

Evolus Pencil is cool - [http://pencil.evolus.vn/en-US/Home.aspx](http://pencil.evolus.vn/en-US/Home.aspx) - it's a wireframing tool but it also has stencils for flowcharts, so maybe it kills two birds with one stone. I use it mainly for the sketchy wireframes in my workflow since I don't normally do the flowchart thing.

For a different take on this, maybe Zim would be good, since you can do hierarchical wiki-linking. That makes it nice for stream-of-consciousness type work. Zim also has a Dia plugin.
