### Post by smartboyathome on 2008-11-12
Theres already a thread on that theme, and personally I don't like the "sticky note" look it has. Also, it isn't possible to create that yet, as you can't have rounded menus, nor is Ubuntu going with a dock since Apple patented it.

EDIT ABOUT YOUR EDIT:
Ubuntu *can't* come with mp3 codecs, as its patented in some countries and thus would be illegal. Ubuntu encourages the use of native programs and not wine so it won't come with that. There is currently no paint software which matches kolourpaint which is written in GTK, so if you want it create it. Lastly, Ubuntu won't come with a whole bunch of programs because it tries to keep on one CD. To come with a whole bunch of software, as you say it should, would mean that it would have to be DVD sized (and not everyone everywhere in the world can download DVDs very quickly nor has a DVD drive to burn it).
