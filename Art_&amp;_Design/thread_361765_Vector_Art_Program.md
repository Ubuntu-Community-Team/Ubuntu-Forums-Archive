### Post by kebes on 2007-02-16
For some instructions on how to do various kinds of install in Ubuntu, see here:
[http://www.cutlersoftware.com/ubuntuinstall/](http://www.cutlersoftware.com/ubuntuinstall/)


You can use the tarball from the Inkscape site ([see here]("http://www.cutlersoftware.com/ubuntuinstall/#source") for help). Alternately, if you want to stick to "Ubuntu approved" packages, you can manually download and install the .deb file. First go to the Ubuntu packages:
[http://packages.ubuntu.com/](http://packages.ubuntu.com/)

Select your distro, search it, and then download the one for your architecture. Then on the commandline use something like ([see here]("http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_install.2Funinstall_.deb_files")):
sudo dpkg -i package_file.deb


You can also install the downloaded .deb file using the GUI ([see here]("http://www.cutlersoftware.com/ubuntuinstall/#deb")).


Hope that helps.


P.S.: I use Inkscape and it's great!
