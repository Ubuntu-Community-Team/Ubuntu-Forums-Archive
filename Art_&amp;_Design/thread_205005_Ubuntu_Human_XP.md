### Post by mhancoc7 on 2006-07-02
[QUOTE=Mr.X]Wow, thats really cool:D.
How do I install it :p?[/QUOTE]

Thanks!!

To install it:
1. Download the themes.
2. Open System > Preferences > Theme
3. Drag and Drop all the themes except the GDM Theme: Energy Bliss Ubuntu.
4. Now Click Theme Details and select the new themes from each of the tabs.
5. Save the theme as Ubuntu Human-Orange XP.
6. Select Ubuntu Human-Orange XP and exit.
7. Open System > Administration > Login Window.
8. Drag and Drop the GDM Theme: Energy Bliss Ubuntu.
9 Select Energy Bliss Ubuntu and exit.
10. For instructions on using the "Ubuntu Start Button" refer to: [http://www.ubuntuforums.org/showpost...0&postcount=13]("http://www.ubuntuforums.org/showpost...0&postcount=13")

Sorry if this seems a bit complicated. It is actually quite easy.

Thanks, Jereme
