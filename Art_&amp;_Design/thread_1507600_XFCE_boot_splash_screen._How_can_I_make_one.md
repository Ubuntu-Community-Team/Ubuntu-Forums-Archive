I'm not finding any documentation out there, and this is what I have found myself.

Seems there are several splash engines out there. Balou being the one most people seem to release splash screens for. For some reason they don't work for me, but that's probably another issue.
 
The Xubuntu splash, mouse running in a Xubuntu logo, is done in the 'Mice2 Splash Engine'.

These can be found here:
/usr/lib/xfce4/splash/engines
