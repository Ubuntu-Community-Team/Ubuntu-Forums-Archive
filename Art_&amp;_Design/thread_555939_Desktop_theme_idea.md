### Post by VictorR on 2007-09-20
I had been working with implementation of the concept for Gnome desktop theme (GTK + (Metacity or Emerald) + icon set + mouse cursors) until found that the interim result was rather boring. So I decided to put it here for discussion.

The idea was that the operation system which is just a tool, should be separated visually from its data - the reflection of our world, by mean of colour (in my case). So I tried to make a black and white desktop theme. It is actually not a true B&W one as pure gray looks like dead, so used colours have minute prevalence of red and green over blue. Other things taken to considerations were:
1. the active window should be easily distinguished from others and be free of any obstacles, so it has a dark border and no transparent elements
2. the Close button is considered dangerous (can cause loss of data), so it is the only which has red glow when one is trying to click it
3. the background picture is assumed to be monochrome or low saturated
4. action buttons like Close, Exit or Save, should have monochrome or low saturated images
5. application icons are part of data so they can be coloured as well as emblems and mimetypes

The screenshot is here: [http://home.graffiti.net/vrull/Screenshot.jpg]("http://home.graffiti.net/vrull/Screenshot.jpg")

GTK+ theme is based on Ubuntulooks engine, Emerald - truglass engine, icon set - Etiquette by Andrew Fitzsimon and mouse cursors - Polar Cursor Theme Blue by Eric Matthews.
