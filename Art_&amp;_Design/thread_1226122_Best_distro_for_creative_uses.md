### Post by Incense on 2009-08-01
Ubuntu with the ubnutustudio-graphics package installed isn't a bad start. If you're working with a lot of raw files, you may want to invest in [Bibble Pro]("http://bibblelabs.com/") since f-spot, and digikam don't really provide a great workflow for raw conversion, though digikam is getting there. [Raw Studio]("http://rawstudio.org/") is also in the repos (I think) and it does an ok job too. I guess at the end of the day, the software is going to be available for pretty much any distro. So whatever YOU want to use should work just fine.
