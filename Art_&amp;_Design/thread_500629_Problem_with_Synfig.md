### Post by digitalis_vulgaris on 2007-07-14
Solved problem with Synfig ! It looks like to be a common problem during installation, so I find solution on some forum.
If somebody confront with this kind of problem, solution is to find graphical files (set of *.png images) for Synfig and to copy them to /usr/share/pixmaps folder.
