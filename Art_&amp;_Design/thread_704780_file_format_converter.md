### Post by mendozaro on 2008-02-22
gThumb.
It is a default app installed in ubuntu.
Just right click an image file -> open with -> gThumb

Also you could change your settings so that all images open with gThumb.
For that you must right click a image flie go to properties and change the "opens with" application.

In order to convert format for any image file using gThumb just open the file and hit Save As. There you will find all the formats you may want to use.

### Post by erginemr on 2008-02-22
Nice to have another Irfanview user here. Together with XMPlay and Total Commander, it was the pearl of the Windows world for me. I miss it a lot, too. 

+1 for GThumb, but also the default image viewer in Ubuntu, the Eye of Gnome, can also convert images. All you need to do is to select "save as,,," from the menu and give an extension to the file name. So, if you want to save it as jpeg, fill in the dialog as file_name.jpg, if you want is as tiff, file_name.tiff, and so on. 

You can also use GIMP for that purpose, which supports much more picture formats. But using it for a simple png <-> jpg conversion would be an overkill of course.
