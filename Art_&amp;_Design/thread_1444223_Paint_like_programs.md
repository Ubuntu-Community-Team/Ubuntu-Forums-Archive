### Post by aklo on 2010-04-06
Honestly i've never tried other linux version of painting programs.

I only use GIMP since it is feature rich...with enough knowledge you can do anything you want. 

Although not too easy to learn for beginners, i highly recommend new users to spend time getting familiarise with GIMP. How difficult is it to select the paint brush and start drawing away?

Also we do not really need a basic mode...well the basic tools are already there...**is the pen/paint brush tool so difficult to spot?**

As for layers, definitely something new to beginners also...well if they don't want to use it they will never know how to and treat it as if the feature itsn't there.

So i dont' see how GIMP new users will be overwhelmed by GIMP's interface/tools at the first place.

Also since GIMP can do many things, you do not need multiple programs like some of the users suggest...i don't use the programs they sugguested so i can't say that those programs are bad but why get 1 program for painting, 1 for photomanipulation, 1 which is suitable for pixel art....heck GIMP has all of them. Just 1 program to take care of those task you need and it seriously is not difficult to learn...of course to reach master levels of painting standards/photomanipulation will require practice and natural talent.
However getting familiar with the tools are just a few days of playing around and you should know what every tool does...or most of the basic tools.

I hightly suggest every gimp user to try gimp paint studio. It is a package which installs additional brushes and arrange the layout to make your work easier.
To install:
Install GIMP first.
Download GIMP Paint studio zip file.
Read which contents to copy into your GIMP folder.
