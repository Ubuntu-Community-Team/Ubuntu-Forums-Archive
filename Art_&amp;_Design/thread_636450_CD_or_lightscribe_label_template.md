### Post by reiki on 2007-12-09
I made this simple template with instructions using Inkscape. Using it you can make a CD label or... as I use if for... create full disk lightscribe labels to be burned using Lacie's 4L-gui

Instructions are a layer in the file and can be turned off when you export you label to bitmap. I export as .png for burning lightscribe labels.

Please let me know how this works out for you. It's not fancy, but it seems to work out fine for this purpose.
