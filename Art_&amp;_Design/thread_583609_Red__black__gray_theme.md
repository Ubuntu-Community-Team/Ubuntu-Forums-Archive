### Post by smartboyathome on 2007-10-20
try taking icons you like from one theme and putting them in the one you are using. Those icons are like that because your current icon theme doesn't include those icons, so it falls back on the standard GNOME theme. Alternatively, you can edit the icon's .desktop file (easier to fill in gaps, btw) using the link for theming in my sig, and set another icon set that is red as a fall back. This would fill in the gaps.
