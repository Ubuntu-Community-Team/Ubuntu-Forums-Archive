### Post by rsambuca on 2007-12-03
You can either install it using the Synaptic Package Manager (System -> Administration -> Synaptic Package Manager).  Just select Avidemux for installation and press apply.  It will install the program and all required dependencies for you.  You will find a launcher in your Applications menu under "Sound and Video".

Alternatively, you can install it via the command line with this command```
sudo apt-get install avidemux
```
