### Post by bvc on 2006-01-12
I don't really understand the gtk theme concept. Most of the theme can be done with engines. The scrollbars can be done exactly the same with the smooth engine, and the buttons are 90% clearlooks, so go all the way with them and the other few widgets and you have an engine theme (?square and round?...if that floats your boat) and it's much faster. It's not likely that a pixmap gtk theme will ever be included in a default install. If it was there is far better out there.

The clearlooks2 metacity theme doesn't even match the gtk scrollbar. That has been mentioned about 10 times now, so I'm not the only one that sees that. Makes you wonder what's happening there.

The redish brown wasn't very popular either, though I grew to like it. That said, I think the depressing tones of brown should be ditched for something brighter. [Yasis is fine in its original color]("http://kwh.kernow-gb.com/~bvc/theme/screenshots/Yasis1.jpg") with that faster metacity theme :smile: ...oh and look, the metacity matches the scrollbar.....

The problem with the yasis icon theme is that it is not packaged in freedesktop standards, but that wouldn't be hard to change.
