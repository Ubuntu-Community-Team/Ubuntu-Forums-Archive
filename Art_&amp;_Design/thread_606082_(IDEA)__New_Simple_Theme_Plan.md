### Post by BradwJensen on 2007-11-07
I think it would be amazing if we could somehow have a new way of customizing/making theme colors/looks..

**My Idea** is to incorporate a web CSS type scripts to change every aspect of a desktops look.  We could add new/extra codes to it to add gradients.  And images can be used just like in web designing..  This would really get more people who are into theming websites to making themes for Linux.


**Examples:**

**Examp 1: **[SIZE=1](this would use an image "title.png" made in Photoshop or Gimp or something as the titlebar, simple.)[/SIZE]

titlebar { bg-image: title.png;}

**Examp 2:** [SIZE=1](this makes the titlebar 21 pixels high, black, have a white to transparent gradient from top to bottom, and make the gradient from top to bottom only 10% seeable so it blends into the black bg color more, so its not soo hard/ugly looking. lol)[/SIZE]

titlebar {
height: 21px;
bg-color: black;
gradient-top: white, transparent;
gradient-top-transparency: 10%;
}


Something like this would be easy to learn and more customizable..  we could always add more aspects to the CSS like code later for more features. 

**NOTE:** This would only be used to control the colors/main looks.  _not how_ menus drop down or anything like that cause that could get messy.
