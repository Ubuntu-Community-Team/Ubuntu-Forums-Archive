### Post by Garyu on 2007-02-20
To create icon themes or window manager themes or GTK themes... One way I find useful is to just copy an existing theme and then modify it. Use one of the default ones that already has a lot of definitions set. To be able to install a theme it needs to be compressed though, so after you are done modifying it, just right-click in nautilus and choose "Create archive".

One way to make the things you do available to other users is to add it here at ubuntuforums.org, just click "Gallery" on the menu-bar close to the top of each page. Or you could add them to gnome-look.org where lots of other linux users will benefit.

Good luck. Submitting art should be easier (as well as submitting software), because right now there isn't any really good way I think, but the important thing is that people start using their creativity and letting others benefit. :)
