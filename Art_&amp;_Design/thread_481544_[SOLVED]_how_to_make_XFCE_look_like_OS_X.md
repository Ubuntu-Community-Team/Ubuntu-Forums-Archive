### Post by avik on 2007-06-22
XFCE uses GTK, so GTK themes (that change things like buttons and other controls) will work.

XFCE doesn't use Metacity, so you can't use Metacity themes to change the window decorations.

If you're running Beryl, however, you have more options.  Because you can use the Emerald window manager, you can theme the windows with OSX Emerald themes.  You can get docks such AWN that require a composited environment.

However, you can't change the top panel, as far as I know, because that's only for GNOME.

Hope that helps.
