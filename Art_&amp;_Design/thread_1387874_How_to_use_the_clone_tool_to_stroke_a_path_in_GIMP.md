### Post by abelard on 2010-12-23
This problem has plagued me for months, when attempting to use the Clone Tool in GIMP while running Ubuntu.  Just as I was about to call the whole thing quits, I added one key to the path and it started working for me.  Normally, to select a little area to "clone," you are supposed to use CNTL + Left-Click. ** Instead of this (in Ubuntu), try:  _CNTL/ALT + Left-Click_ (or whichever click you have set to function as the standard LEFT click.)  **** Eureka!  Works like magic.:popcorn:**
