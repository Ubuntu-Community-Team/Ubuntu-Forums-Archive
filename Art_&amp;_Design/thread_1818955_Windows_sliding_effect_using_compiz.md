### Post by Copper Bezel on 2011-08-08
Hey, sorry, I know this might be a bit late, but you need the compiz-plugins-extra package. If you turn off the tesselation (by setting the values to 1) and rotation (by setting the value to 0) on Skewer in the Animations Add-On plugin's Effect Settings and set the directions you want, it'll give you what you want. I've used this as a minimize animation in the past, and it's really quite neat.
