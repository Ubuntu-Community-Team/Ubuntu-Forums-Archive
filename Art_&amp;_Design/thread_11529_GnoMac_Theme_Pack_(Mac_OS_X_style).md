### Post by sas on 2005-07-05
[QUOTE=Skel]I feel kinda dumb but i wanna know that also LOL[/QUOTE]
 If you download the file there's a README, it has the directories you need to copy things too. One thing to note is that to copy anything into /usr/share/ then you must use sudo. Also one of the directories it tells you to put things in has a typo. It should say "gdm" instead of "fdm".

Once they're installed press the system button, then preferences, then themes, for wallpapers follow the same steps but press "wallpaper" instead of "themes".
