### Post by Woolio1 on 2010-02-15
Okay, here's the deal. Right click menus. They're boring, am I right? Gray or brown, and nobody really thinks about them...

I've found a game that thinks about them. It's a web-based game, but it's not important. What is important is that we should try to get our context menus to look like this.

[IMG]http://img.photobucket.com/albums/v313/Afoxmore/Runescape/nokickass.png[/IMG] 

See the little brown box with the text? It actually looks NICE, instead of the average gray context menu we're used to... I'd like to see this in some sort of theme, I may even start making my own! 

Because a little more eyecandy never hurt. Windows has had the same context menus since 95, why should we, the Linux users, the pinnacle of freedom, have to stick in the same category as Micro$oft? 

Bottom line: We need to improve our Right Click Menus.
