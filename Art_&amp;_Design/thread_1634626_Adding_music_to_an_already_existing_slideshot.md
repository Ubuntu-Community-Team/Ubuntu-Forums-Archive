### Post by I'mGeorge on 2010-12-05
> **toolmania1 said:**
> [http://ubuntuforums.org/showthread.php?t=1530741](http://ubuntuforums.org/showthread.php?t=1530741)

Using the method mentioned in that post will give you a slide show.  How can you add music to it?

The method, from that link you've gave implies creating a .gif file, composed from sliding images. That's not such a big deal, but the thing it's gif's are animated images and don't support sound streams. If you wanna add sound to a sliding image project use Open Office's Presentation, that's the clone of MS PowerPoint or Smile an open source software especially designed for slideshows. Or bind the images into a video format, mpeg, avi and so on so you could add a sound sequence to them using Kdenlive for example. Or to keep it really simple you can use the gimp's creating gifs method, and just play the sound file as the animated gif unfolds.
