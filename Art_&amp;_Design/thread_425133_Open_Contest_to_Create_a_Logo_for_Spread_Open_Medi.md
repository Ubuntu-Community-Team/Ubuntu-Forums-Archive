### Post by Saoshyant on 2007-04-27
Hello everyone,

While this forum seems more focused on graphic software/theme discussion than art creation, I think this thread belongs here.

Spread Open Media is a project by the Xiph.Org Foundation to help promote the different open media formats that exist like Vorbis, FLAC, Speex, Theora, XSPF and even SVG.  The Open Clipart Library is hosting a public contest where anyone may submit an entry to create the logo that will be used throughout all the campaigns, including in web banners, t-shirts, fliers and other stuff.

So, if you are talented, and you either want to help the cause of free software and promote better alternatives to media formats, or you want fame and glory, this is for you.

Be sure to check [http://www.openclipart.org/xiph](http://www.openclipart.org/xiph) even if just to look for more information.

I'm pointing this out, because this project really needs help to take off, so even if you aren't going to participate do mention this contest to people you know that may want to join.  Please.
