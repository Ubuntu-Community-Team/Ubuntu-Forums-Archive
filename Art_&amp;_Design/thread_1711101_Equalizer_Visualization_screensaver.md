### Post by Myoh on 2011-03-23
You might want to look into ProjectM. One of the best, if not THE best audio visualization for Linux. It can run as a standalone program so you should be able to run that after a certain amount of idle time. Thats where the biggest issue lies. There doesn't seem to be an option to run a command as "screensaver".
This seems promising though: [http://ubuntuforums.org/showthread.php?t=305357&page=2](http://ubuntuforums.org/showthread.php?t=305357&page=2)
I'll test it tomorrow.

One other pretty large downside: you'll have to compile it from source, it's not included in the software repos. See if this guide works: 
[http://www.ubuntugeek.com/ubuntu-howto-install-projectm-audio-visualizer.html](http://www.ubuntugeek.com/ubuntu-howto-install-projectm-audio-visualizer.html)
