Using xwinwrap and glmatrix

Code if anyone wants to play:

```

xwinwrap -ni -fs -s -st -sp -b -nf -- /usr/lib/xscreensaver/glmatrix -window-id WID -delay 50000 -density 60

Obviously you need xwinwrap installed.
