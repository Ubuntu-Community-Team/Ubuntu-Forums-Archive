A quick GIMP'ing.

Method :



Made a circle on one layer.

Copy pasted it onto 3 more layers.

Moved each one into position.

Filled each one with colour.

Filter and added a bump to each.

Made a grey to white gradient going down for the background (shows badly on that JPG).

Made each layer 80% ish transparency.

Added 1 plain transparent layer on top.

Added text to top layer.
