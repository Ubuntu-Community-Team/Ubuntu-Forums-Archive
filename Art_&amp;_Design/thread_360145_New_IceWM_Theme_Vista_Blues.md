### Post by Albi on 2007-02-12
A hybrid between Beryl and Vista looks, but with many of my own touches.  This took me longer to make than the other themes. Made after my discovery of "gradients" in IceWM, which let me do the Beryl-like look.

My only wish was that IceWM had png and alpha transparency support; that would rock, but this is still a far cry from a window manager made to resemble Win95.

If the screenshot looks familiar, it is.  This was my theme in October/November, although with Beryl, not IceWM.

Also, to set background: Edit the line near the end of the default.theme file to the correct path to your background, or remove it altogether if you're using idesk or something similar.

Anyways, enjoy!
