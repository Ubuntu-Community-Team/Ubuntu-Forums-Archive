### Post by Paul820 on 2007-11-18
The image you are working on probably doesn't have an alpha channel. Right click on the layer in the layers palette that your image is on and select **add alpha channel**. Now you can remove the parts you do not want.

You can also make a new transparent layer below the layer you are working on but the above solution is probably the best way.

### Post by nikoPSK on 2007-11-18
black background white text size 50 bold, then type it out position it in the center. Use the fuzzy smudge tool size 7 blurry. Blur all the letters so that no pure white is left then smdge up in wavy lines.

goto colours:

Shadows
~---------~
+100
+30
-30

Midtones
~----------~
+100
+30
-30

Highlights
~----------~
+100
0
-100

there you go! quite simple but a bit time consuming. :)
