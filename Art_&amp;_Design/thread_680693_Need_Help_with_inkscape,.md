### Post by junior aspirin on 2008-01-28
you should be able to use the trace bitmap tool

1.import the png into inkscape
2.Path>Trace Bitmap
3. fiddle with the settings :)

since it is all black and white it should work with no problems.

edit: brightness cutoff, with colours:2 worked good for me.
