### Post by tretle on 2007-07-04
With the recent discussion on mobile phone synchronization in the developement gutsy gibbon thread I started thinking about the design aspect of the idea. Whether using conduit, open sync or whatever form of applictaion to synchronize data to your mobile phone having a coherent look with respect to icons is an important thing. There are two methods which could be used to insure this coherency. 1. Make a default tango phone icon to be used for all mobile phones, 2. Make tango icons for all mobile phones. That is what this thread is here for, the developement of tango stylized icons for mobile phones. So post them here. If you just want to make an icon for your mobile then do so, if you want to make multiple icons then may I point you towards the [mobile compatability list for opensync]("http://www.opensync.org/wiki/DeviceCompatibilityList") to help you make up your mind on which phones you want to make icons for. For those who dont know what conduit is,  you can find information [here]("http://www.conduit-project.org/"). Conduit is essentially a synchronization tool not specific to mobile phones and it is based on opensync. EnJoY!
