### Post by cottfcfan on 2012-12-25
For those of you who aren't overly impressed with the dirty grey/blue look of the plymouth theme in kubuntu. I have made it more blue in line with the kde look.
You can download the 2 files from the following link & copy them to /lib/plymouth/themes/kubuntu-logo. (You might want to rename the original files 1st in case you want them back)
It goes quite well with the Glassified-eos splash screen, which you can get in system settings/workspace appearance/splash screen, & the Elegant-eos kdm theme which you can download through system settings/login screen/theme.

ps. the login screen has to be changed as root - kdesudo systemsettings in a terminal. 

[https://www.dropbox.com/sh/a4x7cedz1rsd2qx/m8zo6SL7n1/KDE-Plymouth](https://www.dropbox.com/sh/a4x7cedz1rsd2qx/m8zo6SL7n1/KDE-Plymouth)

Enjoy.

edit... you will have to run sudo update-initramfs -u in a terminal for it to take effect.
