### Post by alex.rayu on 2009-05-22
> **Merk42 said:**
> Could someone tell me what the point of Fireworks is?
I had it at work since I got one of the CS3 collections but never needed to use it.

Fireworks is an app for web design. It is like Photoshop, but with much more leaning towards web design. Uses more vector than Photoshop, has many built-in features that facilitates creation of web elements. Besides, it supports an array of formats, which makes it slowly into a more and more demanded tool for opening different formats for the pro web design industry.

This also answers the previous post. You can pretty much go with anything to draw, even MS Paint, but if you can not open an AI or layered PNG files that other pro developers send to you, if you can't open photoshop PSD files decently, if you send them your designs in a format that is not industry accepted, then you become a garage coder/designer, making websites for your neighbour's pet. So this is quite serious, I absolutely don't recommend to listen to "We have GIMP" talks when it comes to this.

### Post by sabre2th on 2009-06-08
> **MadnessRed said:**
> what do you mean by that, have wine open in its own window, emulate a virtual desktop.
Sounds like he means to install Windows in a virtual machine.  I do this occasionally when I find something that doesn't work in WINE.  Easy to set up using VirtualBox.  

I can't guarantee that would work, but I see no reason why it shouldn't.  You'll need a full, legal copy of Windows, but it should let you keep using Fireworks (great program, from my limited experience with it).

[http://en.wikipedia.org/wiki/VirtualBox](http://en.wikipedia.org/wiki/VirtualBox)
[http://www.virtualbox.org/](http://www.virtualbox.org/)

If you choose to go that route, the latest VirtualBox can be installed from their repository or from a .deb.  I'm pretty sure it's in the Ubuntu repositories too, but that may be an older version.  Either should work, regardless.  I suggest trying it out with an unlicensed version of Windows first (in case it doesn't work like you want), then take the plunge if everything seems okay.

Edit: It does take some resources to run virtual machines (it's kind of like running multiple computers on one set of hardware).  That's one big reason why I say you should try it before you buy anything.  Make sure you have decent hardware if you want to try that, and be sure to allocate enough RAM in the VirtualBox settings (not too much, though).
