### Post by ad_267 on 2009-05-09
You can select the blend tool (gradient) and select the conical shape to blend the colours together like that. Then do a Gaussian blur with a pretty big radius.

To make the engraved text you can create some text and convert it to a path, then a selection. Then use Filters - Layer Effects - Drop Shadow to get that effect. I've attached a screenshot showing what I've done in GIMP.
