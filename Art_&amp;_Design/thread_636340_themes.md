### Post by smartboyathome on 2007-12-09
An emerald theme requires the Emerald theme manager, which is not installed by default. You have to install it using Synaptic and searching for emerald. Then, the emerald theme chooser is in System > Preferences. To use the emerald theme manager, run this command via alt-f2:
emerald --replace

Also, you need compiz fusion to run emerald.
