### Post by mcduck on 2009-08-25
Gnome-Colors/Shiki Dark/Arc-colors.

You'll be able to find icons, GTK2 themes, Metacity themes (I'm not using Emerald but since there's also Fluxbox & Openbox themes available I bet you'll find Emerald theme as well), GDM themes, Gnome-Screensaver's lock dialogs, wallpapers and pretty much everything to fit the theme.

There's even a PPA repository that includes all the most important components.

[http://code.google.com/p/gnome-colors/]("http://code.google.com/p/gnome-colors/")

### Post by mcduck on 2009-08-26
> **Shiv4m said:**
> I need an already made FULL theme, so I just have to download and install it all together. Anyone?

Add the Gnome-colors repository, and install the packages they have. Then you will have pretty much a full theme (actually several full themes, as it comes in many color versions). That way you'll get wallpapers, icons and both GTK & Metacity themes.

Making a theme that includes more than that in single package isn't really even possible, so whatever you do you'll still have to install GDM and lock dialog themes separately.
