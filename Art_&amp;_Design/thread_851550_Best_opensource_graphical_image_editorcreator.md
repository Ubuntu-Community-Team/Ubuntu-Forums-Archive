### Post by reya276 on 2008-07-16
In the GIMP try going into File>Preferences, there you will see a section called Window Management. Once you select this you will see something that says "Window Manager Hints" Select Keep Above for form select menus and that would do it. 

Oh also make sure you check the Focus box (Active the focused image) and the Window Positions box(Save windows positions on exit). While you are in Preferences you might also want to take a look at the Color Management section. 

There are also tons of plugins for the GIMP that give you much of Photoshop functionality.

If you need any help using the GIMP check out the online tutorial:
[http://docs.gimp.org/en/gimp-pimping.html#gimp-prefs-dialog](http://docs.gimp.org/en/gimp-pimping.html#gimp-prefs-dialog)

For illustration use inkscape, if you have not installed it run this on your terminal(command line tool) 

sudo apt-get install inkscape

I actually wanted to like the GIMP but when it happened that I had to work on 16-bit photos I just could not continue using it. So I looked around and tested Krita. I usually don't install software that was not made for GNOME (actually that's a bit annoying to have so many DEs around and programs made specifically with one DE in mind), but I had to.
