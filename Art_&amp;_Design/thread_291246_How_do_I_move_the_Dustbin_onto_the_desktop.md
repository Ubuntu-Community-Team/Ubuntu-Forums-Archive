### Post by bihe on 2006-11-02
* start gconf-editor
* navigate to /apps/nautilus/desktop
* check 'trash_icon_visible'

### Post by NeoLithium on 2006-11-02
Type alt+F2, type in gconf-editor; in the window on the side, select applications -> then go to nautilus -> then to desktop; and check the box with trash_icon_visible; and exit.
