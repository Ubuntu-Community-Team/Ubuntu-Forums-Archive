### Post by UbuWu on 2005-10-04
There are five different parts of the official repositories:

Main: officially supported software
Universe: the rest
Restricted: drivers etc. that are not open source or have restricted licenses
Multiverse: programs that are not open source or have restricted licenses
Backports: newer versions of programs backported from the development release

Only main is enabled by default.
