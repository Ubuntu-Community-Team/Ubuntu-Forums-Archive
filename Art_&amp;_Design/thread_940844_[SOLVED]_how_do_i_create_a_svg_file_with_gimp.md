### Post by smartboyathome on 2008-10-07
You cannot create SVG files in the GIMP. I suggest Inkscape for that. Also, raster files (the ones that are created in GIMP) cannot be converted to SVG very well, as SVG is equation based, and Raster is pixel-based.

### Post by airtonix on 2008-10-08
should also mention that inkscape also makes use of a 'bmp 2 ps' plugin...

1 Open up inkscape, import a photo by drag'n & drop'n from a folder
2 select the bitmap, and goto : path -> trace bitmap
3 play with the options there and have fun.

result should be a vectorised version of your bitmap...which can be saved as a svg.

Remember : svg = Scalable Vector Graphic...and its a ascii based file format using the xml markup language.
