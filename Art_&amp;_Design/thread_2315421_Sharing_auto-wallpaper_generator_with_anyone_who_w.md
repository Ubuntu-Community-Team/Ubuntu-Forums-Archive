### Post by dvdbillen on 2016-02-28
I wrote this wallpaper generator and I've been liking it a lot and thought I'd share it.

It's attached, but here's a link in case there's problems:

[https://drive.google.com/file/d/0B8K-vp0sl8NZMk9aSTlaME1Fc3M/view?usp=sharing](https://drive.google.com/file/d/0B8K-vp0sl8NZMk9aSTlaME1Fc3M/view?usp=sharing)

It's source code, but it's just one source file and no dependencies and very easy to make (simple instructions included). Plus since it's source code you don't have to wonder if it's something seedy :)

You will only be interested in it if you're like me and want a simple, pretty much solid wallpaper, but with some niceness about it. It's great with a "flat" look. I found that by modeling the gradient that light actually uses to decay it made a soft gradient look like real lighting variations and for some reason more pleasing to look at.
