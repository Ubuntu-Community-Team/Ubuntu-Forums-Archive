### Post by rouge568 on 2008-10-18
So I've been spending the last couple hours trying to get the now stable Photosynth working through wine, to no avail. Apparently the team is working on cross-platform compatibility, but considering that this is Microsoft, I think they will be catering to mac users, not linux. However, during my search for a solution, I found a program that aims to do the same thing. Introducing Bundler, the open-source alternative to Photosynth (and actually the basis for the Microsoft code). It's not terribly pretty, or feature-fledged, and requires some hacking to get it configured, but definitely has potential if some developers jumped on this. I couldn't actually get it working, due to my own shortcomings, but please post if you do.

Links:

The project homepage: [http://phototour.cs.washington.edu/](http://phototour.cs.washington.edu/)
