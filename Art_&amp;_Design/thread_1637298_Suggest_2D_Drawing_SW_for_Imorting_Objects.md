### Post by lykeion on 2010-12-04
You can use GIMP image editor to import scanner images, just install the **xsane** and the **sane** packages.
Then you can acquire images in GIMP from File > Create > XSane menu.
You might want to check this page to setup your scanner in Ubuntu: [https://help.ubuntu.com/community/ScanningHowTo](https://help.ubuntu.com/community/ScanningHowTo)
