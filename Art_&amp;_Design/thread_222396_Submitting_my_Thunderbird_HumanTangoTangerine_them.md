### Post by marbleheadman on 2006-07-24
I noticed in the Knot 1 screenshots that Firefox now has a default Human theme, which I'm assuming comes from the firefox-themes-ubuntu package. I looked at the source package (ubuntu-php-firefox-human), and saw that it was generated by a set of nifty command line PHP scripts. I wondered why there wasn't the same for Thunderbird, so I took matters into my own hands and did some extensive modifications to the scripts to generate Thunderbird themes.

My question: where do I go from here? Should I repackage it and submit it to the Universe? I'm sure the community would love this integration -- and I'm sure the original author of the scripts (Frank Schoep) would like some credit, too.
