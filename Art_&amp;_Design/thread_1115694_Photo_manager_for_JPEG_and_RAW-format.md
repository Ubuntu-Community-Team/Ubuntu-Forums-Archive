### Post by Martin-Herrmann on 2009-04-24
Mapivi helps you to handle pairs of JPEG and RAW files. It is not suited to process RAW files or to add meta information to RAW files directly, but if you e.g. copy, move or rename a JPEG file Mapivi is able to copy, move or rename the corresponding RAW file.
So if you add e.g. keywords to the JPEG file you are able to find it later and also to find the corresponding RAW file.
