### Post by malrost on 2009-07-05
If you want a text only lightscribe label. This layout will get about ~7,000 characters. The font is legible, though tiny. I tried a few combinations and eventually went with DejaVu Sans Mono 65px. 

The first image is a screenshot of the LaCie Lightscribe preview. The source .jpg itself was too large to upload here, but if you want to burn the calibration image yourself to test the font size, use this .jpg as a label. [http://www.flickr.com/photos/22943887@N03/3690608554/]("http://www.flickr.com/photos/22943887@N03/3690608554/")

The attached archive is an ODS spreadsheet of the green character grid image. For proper text alignmnet, create a gimp image file 7000 x 7000 pixels. In gimp, insert a text area and drag the edges to align it with the corner of the image. The green characters will be visible on the disc when you use the fit to width or height button on Lacie.
