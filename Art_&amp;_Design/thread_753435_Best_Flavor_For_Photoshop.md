### Post by suchawato on 2008-04-13
No, it does matter, I disagree.

The best environment for using photoshop is going to be any distro using the xfce desktop environment.

You will find Photoshop runs a lot faster as it is already system heavy and having a lighter desktop environment will boost performance.

Personally, if I need a fast environment I either install the xubuntu desktop into ubuntu or I use Zenwalk.

Zenwalk is _very_ fast.
