### Post by Vadi on 2009-04-11
Hi all,

I'm starting a new project - it's a program that will help you create Ubuntu packages (.debs). This'll be useful for theme artists (no more odd shell scripts or manual installation), software developers (packaging the program wouldn't involve hours spent on learning debian packagers) and people who don't want to install from source but a package.

The first release, 0.1, is pretty much ready - but I forgot that there is no logo :p . **Would anyone be interested in doing a logo?**

Sizes needed: SVG + the standard icon sizes: 16x16, 24x24, 32x32, and 48x48.

In regards to the style, not something too cartoony or silly please (I'm hoping that the application will benefit a wide variety of users) - otherwise you're free on your own.

**Project website**: [http://giftwrap.tuxfamily.org/index.php?](http://giftwrap.tuxfamily.org/index.php?)

**0.1 announcement**: [http://giftwrap.tuxfamily.org/index.php?post/2009/05/28/GiftWrap-0.1-is-here%21](http://giftwrap.tuxfamily.org/index.php?post/2009/05/28/GiftWrap-0.1-is-here%21)
