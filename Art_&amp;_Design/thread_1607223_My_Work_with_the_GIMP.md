### Post by LMP900 on 2010-10-31
> **ironic.demise said:**
> Was the bubble image done in gimp, at all?

Yep. I overlayed a color layer over a texture to get the construction paper look and cut out the holes with the circle select tool. I used a photo of Marissa Miller and added a white border to make it look like a photograph. Then, I positioned the cut-out circles along with some stock images (cut, rotated, resized, blurred). I used a slight drop shadow on each component of the image to get a more realistic look.

The only part that wasn't done with the GIMP was the lighting effect, which was done with Core Image Fun House - a hidden tool on Mac OS X requiring the developer tools found on the installation disc.
