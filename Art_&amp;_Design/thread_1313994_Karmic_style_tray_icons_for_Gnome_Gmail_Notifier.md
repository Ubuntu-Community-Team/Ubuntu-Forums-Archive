### Post by verbme on 2009-11-04
I have made some icons new for those of you that like Gnome Gmail Notifier, but dislike how it don't match the new gray karmic style tray icons.

[IMG]http://imgur.com/18qxS.png[/IMG]
[IMG]http://imgur.com/Z7gct.png[/IMG]

Download the file to the desktop and follow these steps:

```
cd ~/Desktop
sudo rename -v 's/\.svg$/\.svg.bak/' /usr/share/gnome-gmail-notifier/*.svg
sudo tar -C /usr/share/gnome-gmail-notifier -xvzf ggn-karmic-style-icons.tar.gz
```

To get the old icons back:

```
sudo rename -vf 's/\.svg.bak$/\.svg/' /usr/share/gnome-gmail-notifier/*.svg.bak
```
