### Post by smartboyathome on 2007-11-11
I haven't done anything with PDFs, but if you are able to get it into GIMP, you should be able to save it as something else (JPG and PNG comes to mind, JPG because it is small with a tradeoff of not as much quality, and PNG because it preserves almost all details, while keeping the size average).

### Post by Tux Aubrey on 2007-11-11
Resizing is easy - Image>Scale Image

Save in whatever format suits (see above) - jpg, png or gif

Other things are certainly possible - index image, reduce colors etc.

Oh, work on a copy!

### Post by maruchan on 2007-11-12
Just save in JPG format after resizing the image. In JPG "advanced options", be sure to set Subsampling to 1x1, 1x1, 1x1 so you don't lose color saturation when you save the file. That should do what you need.
