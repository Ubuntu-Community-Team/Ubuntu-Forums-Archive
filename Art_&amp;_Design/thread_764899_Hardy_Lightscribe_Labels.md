### Post by BadServo on 2008-04-27
Here's the skinny on Lightscribe for Linux.

You're correct, there are only 2 native applications that use Lightscribe on Linux.  You have Lacie's "4L", and lightscribe.com's "Simple Labeler".

Here's where it gets odd.  If you go to the Lacie site, and download the lightscribe host software they provide along with the labeler, you can use both 4L or the simple labeler.  However, the contrast is not as nice as in Windows using the same application.

If you instead install the newest host software from lightscribe.com, the Simple Labeler will work, but the 4L application will not be able to detect your drive.  On the bright side, there's a trick with the newer host software that enables it to burn much darker labels.

run:
```
sudo /usr/lib/lightscribe/elcu.sh
```

And it will give you the option of normal or darker contrast.  Make your choice, and it will be reflected in future burns using the Simple Labeler.

Sadly, the Linux choices for Lightscribe are so poor at this time that unless I just need a quick 'n' dirty label, I boot into Windows and use "SureThing" labeler.  SureThing is what I'm using to make the templates for the above-posted labels.

Hope this info helps.
