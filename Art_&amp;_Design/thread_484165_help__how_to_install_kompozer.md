### Post by linux_virus on 2010-10-20
I found one from [http://www.mepisguides.com/Mepis-6/html/kompozer/kompozer-install-3.html](http://www.mepisguides.com/Mepis-6/html/kompozer/kompozer-install-3.html)

install on ubuntu 10.04:
(1) download kompozer from [http://kompozer.net/](http://kompozer.net/)
(2) unzip the install package "kompozer-0.8b3.en-US.gcc4.2-i686.tar.gz"
$ tar -xvzf kompozer-0.8b3.en-US.gcc4.2-i686.tar.gz
(3) then cd to the unzipped folder called "kompozer", type
$ sudo apt-get install kompozer

after it's done, type
$ kompozer
to run NVU(kompozer)
