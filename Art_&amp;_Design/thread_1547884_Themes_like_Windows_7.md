AWN is pretty great(it's what I use), but I don't think it has an option to use multi-window previews.
With DockbarX, you can get multi-window previews, but I think you may have to enable some settings in CCSM- compiz config settings manager(System menu> Preferences> Compizconfig Settings Manager, if you don't have it installed you can install it using the Ubuntu Software Centre). In CCSM, there is an entry titled KDE compatibility, with a sub-entry 'Support Plasma Thumbnails'. You might need to log out and in again.

Edit:
From the Dockbarx page at Gnome-look
*'NOTE! To use previews you need to activate KDE Compability in compiz settings manager and under KDE Compability check "Support Plasma Thumbnails". (Alternatively you can use kwin as window manager instead of compiz. That will give you previews for minimized windows as well.)'*
You don't get previews for minimized windows with Compiz, but for unminimized ones it works fine.
