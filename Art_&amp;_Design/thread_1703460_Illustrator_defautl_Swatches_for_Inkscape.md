Hi, I want to share this palette. This are the colors from the CMYK Basic swatch from Adobe Illustrator. Remember that Inkscape juts works with RGB palettes (.gpl) so this is not cmyk.

When I used Illustrator I used a lot the default colors, so I was missing some of them in Inkscape.

Just put the illustrator-cmyk.gpl in you Palettes folder /home/USERNAME/.inkscape/palette
