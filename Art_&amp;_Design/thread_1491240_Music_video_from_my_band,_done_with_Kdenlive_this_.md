Whilst I don't understand the lyrics I do like the sound, mellow.  The aspect ratio left black borders on my screen, however as this is a wide screen I'm guessing you are hopeing for some TV airtime.
Kdenlive is good but, according to my own personal taste, is still only the second best video editor for linux.  I concider Cinelerra to be the run away best available though it does have the downside of crashing quite lot, not a major problem if one remembers to save frequently.  One nice thing about cinelerra is that it runs very well on Puppy linux.  For a band on tour this could be handy as one can borrow any pc, load to ram on boot do the editing on a memory stick leaving the borrowed machine unaffected. Being music creating people   I imagine you already know about "rosegarden", if not and you have time that's worth looking at too.
