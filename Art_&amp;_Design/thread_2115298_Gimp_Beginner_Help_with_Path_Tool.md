I assume this is for Gimp? "Select/From path" uses the  active path. When you have several paths it's hard to tell which is active. You can display the Paths list ("Windows/Dockable dialogs/Paths). If you right click on a path in that list, one of the choices is "Path to selection", so you can set the selection from any path you want.
