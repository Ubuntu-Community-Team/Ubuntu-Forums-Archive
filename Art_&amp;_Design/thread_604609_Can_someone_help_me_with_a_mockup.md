### Post by gekaare on 2007-11-06
I am trying to create a mockup, but I'm not very good at it. This is what I have come up with, but it does not display all of my ideas. The reason for me wanting to do this is that I have a widescreen laptop, and hardly ever use the whole width. While using Openoffice.org there is lot of dead space on the sides of the screen, and also in a lot of webpages. Why not use this space more efficiently? When I tried to set this up on my Gutsy I get some weird looking elements, like the systemtray, it displays the icons above eachother, taking up a lot space. 

[IMG]http://farm3.static.flickr.com/2075/1888914275_327646df1b.jpg?v=0[/IMG]

And so on to the main idea that I have, but could not make myself in this mockup. Instead of a dock consistent with the one in OS X, and instead of the regular way to display open applications on the panel, I would like to have live screenshots of the different applications on the sidepanel. They should be organised in an eyecandy way, perhaps something like the Super+Tab way Compiz does the task switch. I hope you understand :P

On to the other panels I have drawn. The idea of these was that applications could use these to insert its menus and title of the app, like in OS X. This would make a more clean desktop I think. The small panel on the side could be used as a scroller for instance. And the panel on the bottom could be used to display messages from the application. Like when firefox tells you it's done loading. Also I would like the corners on the panels to be rounded, making the Desktop imitate an old tv sort of.

Main Menu: When pressing Ubuntu I would like a large window coming out of it with a fixed size, showing on the side nearest to the main panel all the menu categories, and on the next row the submenues. I Would also like a search bar in the top of the menu, searching both apps and docs and web via Firefox for instance. Would also like a recently used apps entry. This window will have an X button to close it, and dissapears if the mouse is no longer in its area, or if the Ubuntu-button is pressed again.

This mockup is based on the beautiful mockup made by floodcasso2, found on this forum.

Can anyone help make a better mockup? Or perhaps making my dream come true?
