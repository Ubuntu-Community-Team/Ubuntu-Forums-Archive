### Post by ola on 2005-04-27
One way to go would be to start the screensaver program from System / Preferences / Screensavers and select the advance tab where you can choose your own images (for example a ubuntu image).

Then you select a screensaver (for example SlideScreen) that modifies those images the way you want. 

Hope it helps!
