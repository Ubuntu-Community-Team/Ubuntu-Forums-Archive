### Post by mcduck on 2009-02-15
In Tool Options, right under brush scale slider, is a small arrow and text "Brush Dynamics". Click that, and you'll see an array of check boxes that you can use to select which settings (opacity, hardness, size and color) are controlled by which things (pressure, velocity or random).

Pressure, of course, only works if you have a pressure sensitive tablet, but both velocity and random can be used even with a mouse.

On the far right end you also have a button that lets you change how big influence these have on the brush (from 0% t 100%).

If you can't see the Tool Options anywhere, go to Windows/Dockable Dialogs/Tool Options to open it.
