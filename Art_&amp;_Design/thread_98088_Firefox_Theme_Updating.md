### Post by 23meg on 2005-12-02
Only if it has a compatibility update for Firefox 1.5. You can see its current compatibility status in its page on addons.mozilla.org, and if you do a check for updates and you have the "Installed extensions and themes" option checked in Preferences / Advanced / Update, it will automatically check anyway.
