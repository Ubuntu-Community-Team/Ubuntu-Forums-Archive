### Post by Flimm on 2009-10-31
For the open and close effects, it's Compiz's beam effect.
Install all the extra packages for Compiz: ```
sudo apt-get install compiz-plugins compiz-fusion-plugins-main compiz-fusion-plugins-extra compizconfig-settings-manager
```
Launch CompizConfig Settings Manager. Enable both Animations and Animations Add-on. Open Animations, edit the first animation on the list (I think it's Glide by default), and change it to Beam up. Do the same for "Close Animation" tab.
To make the beam green, enter Animations Add-on plugin, expand Beam and change the beam colour.
