Process:

1. Create or rename pictures as indicated in the files section (ensuring     that the sizes are as shown below

2. go to /usr/share/images/xsplash and copy all the files in this directory     to another safe location as a back-up.

3. open a Terminal and type code "gksu nautilus"

4. Copy all files from list below to /usr/share/images/xsplash

5. test this by opening a terminal and typing the command "sudo xsplash"     press esc to exit (will exit on it's own)
