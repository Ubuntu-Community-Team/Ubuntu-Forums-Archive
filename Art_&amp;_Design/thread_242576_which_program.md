### Post by huwshimi on 2006-08-25
Inkscape is probably your best option. It is a vector graphics editor that in my opinion is one of the best pieces of open source software ever. There are some great tutorials in the help menu that are actually really helpful for learning how to use the program.

If you want to make raster images then use The GIMP which is similar to Photoshop.

Cheers.
