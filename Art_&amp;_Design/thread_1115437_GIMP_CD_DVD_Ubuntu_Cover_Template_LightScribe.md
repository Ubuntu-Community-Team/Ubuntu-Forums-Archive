UPDATES: New covers and improved template.

I created a template for GIMP for making a Ubuntu CD/DVD cover and wondered if others might want to use it. 

[Free Cover Template]("http://www.vitalbodies.net/site/tech-web/tech/111-open-source-cddvd-cover-for-open-source-projects.html")

There is a link to the Ubuntu font, a down-load-able cover in GIMPs native format and a link to a blog post I did on getting LightScribe to work. 

The GIMP template is in layers so you can easily change it to suit your needs. For example my favorite one has a moon background but that can be changed. 

[CENTER][IMG]http://www.vitalbodies.net/site/images/uploads/Screenshot_LaCie%20LightScribe%20Labeler.png[/IMG][/CENTER]

If you use the template, or customize the template, post an image or link on this thread - would be fun for me and others to see....

**Here is mine:**

[CENTER][IMG]http://www.vitalbodies.net/site/images/uploads/ubuntu_disk.jpg[/IMG][/CENTER]

Note that this image does not look as nice or as interesting as the disk does when held in the hand as light plays upon the disk when your holding it and moving it.
