### Post by super on 2006-01-30
the windows core fonts package are called 'msttcorefonts' and as far as i know, it is available via apt-get if you have the ubuntu universe repositories enabled.
it provides:

Andale Mono
Arial
Comic Sans MS
Courier New
Georgia
Impact
Times New Roman
Trebuchet
Verdana
Webdings

alternately you can download the debian package [here]("http://packages.debian.org/unstable/x11/msttcorefonts").

or follow these complicated but effective instructions found [here]("http://corefonts.sourceforge.net/").

if you are looking for some specific font let me know, maybe i can send it to you.
