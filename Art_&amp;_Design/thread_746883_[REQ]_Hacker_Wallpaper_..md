### Post by FALSEFLAG on 2008-04-07
matrix or pi theme is the archetype for this. I think its been done to death at this point But if you must reinvent the squarewheel do this:
Find a random X3d file and open one window in emacs, lime text background back, font=system.hit prscrn.open GIMP ctrl v and save as jpeg. Set bacground texture in x3d file ito name and path of saved jpeg Open same file in x3d browser of choice, set view setting to wireframe. hit prscrn, repeat as necessary...
