### Post by cammin on 2008-10-24
You have a lot of color/contrast changes, which isn't bad on it's own.
However, when using them as a wallpaper, it makes it hard to choose a color desktop items as some section of them will end up blending with the background. It's not a huge deal with icons, but it makes text hard to read.

Usually that sort of stuff is placed near the edges of the screen, so keeping the edges similar in color/contrast helps. You don't have to do it all the way around the image, but you'll want to do it in a way that doesn't throw your picture off balance.
