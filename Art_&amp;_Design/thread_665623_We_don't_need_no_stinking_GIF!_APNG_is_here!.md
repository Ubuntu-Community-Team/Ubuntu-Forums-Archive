### Post by RemmyLee on 2008-01-12
If you are viewing this post in Firefox 3 Beta, you'll see a rotating Firefox icon. This is the new APNG extension to Firefox which completely eliminates the need for the gif format. APNG is similar to MNG, but it offers a smaller library size and is compatible with older versions of the PNG implementations. If you are using an older version of Firefox, you'll simply see a static frame of the Firefox icon.

[IMG]http://farm3.static.flickr.com/2348/2187394259_e9af2af8ef_o.png[/IMG]

About the animation. It's heavily dithered to 64 colors just for quick loading, but like the PNG format, it supports full alpha channeling, so just think of the possibilities.
