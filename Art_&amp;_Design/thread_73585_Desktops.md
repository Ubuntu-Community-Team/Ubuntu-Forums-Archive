### Post by Zelut on 2005-10-09
Well as far as transparent terminals you can change that in your own terminal settings.  Open it, select Edit > Current Profile and you can change the effects (transparent, colors, etc).

Other options are using the System > Preferences > Themes for a different style & look.  Also look into gdesklets for some cool desktop apps & effects.  (to install that, if you haven't, check out [www.ubuntuguide.org](www.ubuntuguide.org))
