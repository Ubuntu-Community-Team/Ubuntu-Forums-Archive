### Post by Newfoundlander on 2010-07-10
Most of Ubuntu's icons are either SVG or PNG but it can use almost any image format including ICO.

If you right-click on an icon then select Properties, you will see an icon 
preview in the top left corner.  Click on the preview and you can then select a new image.

If you still have trouble viewing an ICO file then open it in GIMP and convert it to another image format like PNG.
