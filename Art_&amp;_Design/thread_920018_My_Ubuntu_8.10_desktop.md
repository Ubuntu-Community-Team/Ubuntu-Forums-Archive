### Post by ramjet_1953 on 2009-03-25
Ash R and Ben Crisford.

The screenlets are set-up by first loading the package [COLOR="Blue"]screenlets[/COLOR] by either using Synaptic or:

```
sudo apt-get install screenlets
```

When the daemon is running, it will put an icon in your taskbar.
You then just select the screenlet you want and configure it.

The selection bar at the bottom of the screen is [COLOR="Blue"]Avant Window Navigator[/COLOR].
Again, you can install it using Synaptic or:

```
sudo apt-get install avant-window-navigator
sudo apt-get install awn-manager
```

You also need to ensure that libawn0 is installed.

[COLOR="Red"]Note:[/COLOR][COLOR="Black"]You need to be running Compiz Fusion to use AWN.[/COLOR]

Regards,
Roger :)
