### Post by Corey on 2005-07-30
I find it wierd for that most of the progress on these icons is so behind the scenes, especially in a community oriented distro such as ubuntu. The default icons are very important in terms of usability and branding so its a critical part of ubuntu. I really like how the gentoo people have set up a community to work on [lila](http://lila-theme.berlios.de/). I wish humanity and ubuntu's artwork in general was more available like that. I know their is progress, but the icon theme in particular still seems kinda secretive. With a community like that all the talented artists using ubuntu could help out and seriously increase the amount of icons available. Icon sets can and **should** be massive. When you install new programs you want their icons to fit in, and if there isn't one available you want enough miscellaneous icons that a suitable and unique one can be found. On top of that there are gazillions of mime types that need to be themed. There arealso lots of other little details that really make a theme shine such as a matching set of gaim emoticons. When the whole community gets involved this extra stuff can be created with relative ease. Making new mime type icons is *mostly* a copy and paste affair. Its a task many people just starting out with inkscape could help accomplish, especially if the final products required some sort of peer review by the creator of the theme. If a few talented inkscapers (is that what you call 'em? ) were able to join in as well the overall progress would move much faster.

Anywho, I'm not complaining, just suggesting. Art can be very personal and I definetly understand the many reasons one would want complete control over it, especially in the begining. Once it is officially released with breezy though I'm sure people will start making edits and new icons for their favorite apps immediately. Having a website like the lila center would make sure that all that various work is collected and [b[easily[/b] available to everyone else. It would also help organize what work needs to be done and prevent people from doing the same work.
