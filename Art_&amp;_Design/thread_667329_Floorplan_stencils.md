### Post by jetole on 2010-11-17
I found this thread while looking for a way to draw a floor plan myself and will be looking into qcad now.

I wanted to mention though, dia is really not the right program for this. I have used Dia a lot and it is a great program but not only is it not designed for creating a floor plan but you will also find that it lacks features which you will often need in creating one.

Dia is a great tool if you need to create diagrams, flow charts, etc but I believe you should use something CAD based for a floor plan.
