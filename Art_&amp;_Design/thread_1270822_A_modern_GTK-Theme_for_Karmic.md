### Post by Exodist on 2009-09-20
Its very dark IMHO.

A theme should incorporate 3 things.

- The theme should contrast well preventing eye strain on the user.
- Theme should interest the user at the same time not totally distracting the user from the task at hand.
- Be as compatible as possible with most all applications, no theme should hinder or prevent the user from completing a task due to incorrectly formatted eye-candy.


Here is a link to my most recent themes for reference:
[http://www.gnome-look.org/content/show.php?content=110684]("http://www.gnome-look.org/content/show.php?content=110684")
