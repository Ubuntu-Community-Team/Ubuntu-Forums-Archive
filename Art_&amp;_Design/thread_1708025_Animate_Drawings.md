### Post by SeijiSensei on 2011-03-16
I've made a number of animated GIFs using the GIMP.  You build the animation as a series of layers, then save to GIF.   You can optimize the file size using a special GIF animation tool as well.  In addition, there's a package called [GAP]("http://www.gimp.org/tutorials/Using_GAP/") that installs inside GIMP and provides additional animation capabilities.  You can install both the gimp and gimp-gap packages from the repositories.

For drawing, your wife should take a look at [Inkscape]("http://sourceforge.net/projects/inkscape/").  It's in the repositories as well.  A quick Google search suggests it works with Wacom tablets in Linux.
