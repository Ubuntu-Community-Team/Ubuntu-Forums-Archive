### Post by wjuniorr on 2007-09-10
> **mastergunner said:**
> I am using Kubuntu 7.04 Feisty Fawn. So will this program work and how do you use it? Any docs on it?

The program is very easy and work with kubuntu. After installed it allows that you modify without complications the most varied configurations of grub and splash screens. Install the program and any doubts you ask to me ok?! At the home-page of the program  you can find screen shots  and docs that may be help you!!  

You can download the grub-backgrounds and [**splash-screens**]("http://www.gnome-look.org/index.php?xcontentmode=160") at [**http://www.gnome-look.org/**](**http://www.gnome-look.org/**). You'll find a lot of stuff!

After install:
--> In Kubuntu you'll find it in KMenu -> System
--> in Ubuntu you can find in Applications -> System Tools

If you have any problem post it here! Good Luck!!!


[COLOR="Red"][B]Features:

Change

        * Grub timeout
        * default boot title
        * number of kernels in bootloader menu
        * enable/disable boot option for memtest86
        * enable/disable boot option for "rescue mode"
        * if the default boot option should be automatically updated
        * bootup resolution and color depth
        * grub menu colors and background, and usplash theme

Create

        * rescue floppy

Change visibility of

        * Grub menu
        * colors
        * background image
        * usplash and usplash text

Password protect

        * grub menu from editing
        * "alternate" boot options
        * old boot options

Themes

        * Install new grub backgrounds (either from a png, or a correctly sized and colored .xpm or xpm.gz)
        * Remove grub backgrounds
        * Install new usplash themes (from a correctly made .so file)
        * Remove Usplash themes[/B][/COLOR]
