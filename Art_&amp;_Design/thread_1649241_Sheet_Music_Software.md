### Post by macneil22 on 2011-04-16
Musescore is awesome and supports midi.  I use it a lot for transcribing but never tried it with a midi file tho.  Takes a while to get your head around how to enter notes and their lengths but once you do you can work pretty fast...

Check out Widisoft automatic transcription software... It's called - MP3 to Midi VST - not sure about a Linux version.

"**WIDI Audio To MIDI VST** plugin is designed for real-time Audio to  MIDI conversion. The plugin receives input from an audio channel and  performs real-time music recognition. Resulting MIDI events can be sent  to the VST host, System MIDI Out Device or saved as a MIDI file with a  simple MIDI recorder. The spectrum window displays streaming spectrogram  in real-time with recognized notes above it. Also the plugin **recognizes chords** and displays chord name at the  top of the spectrogram window. It is possible to control the delay of  chord indication. Adjustable recognition parameters allow to improve  recognition quality."

### Post by macneil22 on 2011-04-19
I just tried Widi Audio Standard download under Wine.  It installed ok but kept crashing on playback but it worked fine in windows.

It is actually pretty good, worth trying just to see what it does.  It was a bit confused with lots of stuff happening, but I tried it with a Duke Ellington piano solo too and I was impressed with the midi transcription. What it messes up you could work out or clean up in the score.   It definitely picked up on complicated chords too, tells you the tuning and what key it's in. The only thing is with the free version you can't save the midi file, so can't load it into Musescore.  It will only let you save 10 seconds unless you buy the $99 license.  VST was $59.

Worth it maybe if Musescore or something else will print it ok.
