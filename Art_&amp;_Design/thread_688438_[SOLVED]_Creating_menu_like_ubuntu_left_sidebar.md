The css for the sidebar container div is specified in [[COLOR="Blue"]_layout.css_[/COLOR]]("http://www.ubuntu.com/themes/ubuntu07/layout.css") - the item is named #leftsidebar.
 
More of the css rendering of container items is found in other css files, which can easily be found using a highly recommended Firefox addon called [[COLOR="Blue"]_Firebug_[/COLOR]]("https://addons.mozilla.org/en-US/firefox/addon/1843"), an indispensable tool for web developers.
