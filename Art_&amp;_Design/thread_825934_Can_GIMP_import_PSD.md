### Post by alex.rayu on 2010-04-20
Gimp can import PSD files, losing the following:
1. Layer comps
2. Layer effects (shadow, glow, etc)
3. Vector objects
4. Layer masks

So, if you have a web design in PSD, you can almost 100% be sure that the imported file will not be as intended.

Photoshop exports to a number of intermediary formats, reducing the functionality and rasterizing the effects, where possible.
