### Post by mikethedj4 on 2009-12-28
I didn't read many topics in here so this in my intake on PhotoShop, and Gimp.

Now everybody functions different, learns a different speeds, and different ways. However some say PhotoShop is better then gimp while others say Gimp can do the same things as PhotoShop.

Now I first started out using PhotoShop, and moved to Gimp. Now PS takes up 1GB of space, now installing more scripts, plugins, brushes, etc: Makes it take up more space.

Gimp takes up 17MB on my pc (and for Linux 15MB), and like PS installing scripts, brushes, etc: takes up space as well, but the PS brushes are combined into 1 abr file bringing it's size higher then Gimp's (.gbr).

Although I have noticed that PS takes up 512MB of Ram, and 1GB is recommended. However Gimp takes up 128MB of Ram, which is still high, but much smaller then PhotoShop's which is another big plus.

One thing I can think of being both a former PS user, and a Gimp user is that PS's one  plus with me would be [this](http://www.blog.spoongraphics.co.uk/tutorials/create-a-realistic-page-curl-in-photoshop).

Now PS comes with the Blending options for Bevel & Emboss, Inner Shadow, Inner Glow, etc: that does the effect for you. Unlike in Gimp you would haft to do it manually, because it doesn't come with the inner shadows, and glows filter, but you can download the [url=http://registry.gimp.org/node/186]layer effects filter to get it, but still it doesn't come with it is the point.

Now Gimp can not save .avi or .flv files like PhotoShop, which is a downer (I did create a site called [Gimp Video Effects](http://gimpvideoeffects.isgreat.org/) that you can go to, to learn that process though) Still like before it didn't come with it.

Now Gimp, and PhotoShop both have some great plugins, and scripts out there, like saving cursors (.cur and/or .ani) for example. However I personally prefer Gimp more due to it's Map Object filter, that comes in handy quite a lot with me. Along with some great scripts, and plugins some users have created like Fensepost's scripts, G'MIC, Gap, and so on.

So there's my intake on this debate.
