### Post by Elle Stone on 2008-01-11
The OP said

"I don't have a very powerful pc (1gb ram, ati x1400, 1.83ghz centrino duo) so I m not sure if VMware will be an option (the adobe stuff uses alot of resources in itself."

Can you upgrade your ram?  I found that a fresh install of windows as a guest in VirtualBox actually works better than a "regular" windows installation.  I doubled my ram so I could give the VBox virtual install the same amount of ram as the real windows install had before, and ditched my original windows install completely.  The neat thing is, is windows in a VirtualBox vm doesn't need any anti-spyware (assuming you do NOT allow it to access the internet - necessary things like activation can be done, then turn off dhcp and dns and VirtualBox network connection and NEVER connect windows directly to the internet again).  And most of the windows background services can be shut down, giving more resources under the virtual machine to the program that you want to run, than you ever had running windows directly.  You'll need to learn enough about windows to know what can be shut down and still keep windows happy.  

There is a ton of neat imaging software in the linux world.  I've always liked the gimp but for some reason they've never gone down the 16-bit route.  Krita crashes on me as soon as I try to open a tiff, but apparently that is a bug related to amd.  It's the other stuff, stuff you may have never heard of living in the windows world, that you might really like under linux.  

Doubling your ram and setting up VirtualBox (from [http://www.virtualbox.org/wiki/Downloads](http://www.virtualbox.org/wiki/Downloads)) will allow you to stay productive under linux (well, under windows under linux) while you explore this wonderful new world of linux.

Also, how you set up your ubuntu installation will determine how much ram and resource ubuntu itself uses.  I did a minimal command line installation and then installed a lightweight desktop (icewm).  It is definitely not as convenient out of the box, but if resources are a concern, it makes a big difference.  And the minimal desktop install will FORCE you to learn how to do things from the command line, which in the long run is a very good thing. 

On the other hand, simply buying more powerful hardware is LOT quicker than learning to get the most out of the resources at hand,   Time really is money, when you think about it.

Elle
