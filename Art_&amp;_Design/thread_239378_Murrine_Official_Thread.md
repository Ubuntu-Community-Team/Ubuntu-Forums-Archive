**Adding repository**
Edit /etc/apt/sources.list and add:
[INDENT]
*Dapper*
> deb [http://malteo.homelinux.net](http://malteo.homelinux.net) dapper-malteo extras
*Edgy*
> deb [http://malteo.homelinux.net](http://malteo.homelinux.net) edgy-malteo extras[/INDENT]


**Authenticate packages**
Packages is this repository can be gpg authenticated. The key that is being used for signing the packages is B54820BC. You can enter this key into the APT trusted keys database with the following command:
> wget [http://malteo.homelinux.net/B54820BC.gpg](http://malteo.homelinux.net/B54820BC.gpg) -O- | sudo apt-key add -

**Installation**
Install through synaptic or with sudo apt-get install gtk2-engines-murrine

### Post by Athropos on 2007-01-06
> **Colonel Kilkenny said:**
> 
You need to set the root user to also use the theme of your choice.
(in terminal "sudo gnome-theme-manager") 


Another way to do that is to create links to your ".themes" and ".icons" directories in "/root", so that all the themes installed for you are also available for root. This way, root applications will always use the same theme as yours without having to "sudo gnome-theme-manager" on every theme change.

To create the links, simply type this in a terminal:

```

sudo ln -sf ~/.themes /root/.themes
sudo ln -sf ~/.icons /root/.icons 

```

```
sudo prevu-init
prevu murrine
sudo dpkg -i /var/cache/prevu/edgy-debs/gtk2-engines-murrine_0.41-0ubuntu1~6.10prevu1_i386.deb

```
