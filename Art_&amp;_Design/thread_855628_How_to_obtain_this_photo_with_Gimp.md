### Post by Niva on 2008-07-10
Well you can create an image like this with GIMP if you're very patient, it's easier if you try a 3d program.  The light effects are render based, you'll need a renderer which supports ray tracing and caustics (the way light bends and focuses to create the highlights below the shapes through refraction.)  I'm not sure Blender's renderer supports such effects but the major 3d packages like maya, 3dsmax and xsi all work with mental ray as a renderer and can easily create such an image.
