### Post by smartboyathome on 2008-07-05
I have made a new theme for Ubuntu Intrepid called Human-Clearlooks2, which is, as the name suggests, an evolution of Human-Clearlooks theme. The difference between these themes is that I gave this theme more color, and based it more on the newer Clearlooks theme engine. I also modified the newer clearlooks metacity theme in order for it to look nice with this theme. Please leave your bug reports or opinions of the theme so that it can be improved. :)

A screenshot is included with the theme archive. It should install like normal with the Appearance tool.

EDIT: I made an alternate window border for this theme, it is available in the Test2 package below. Please tell me if you like the origonal window border or this one better, and if there are any improvements you could recommend.
