-- Install the window border and icon packages at Preferences->Theme.
-- Install the login-GDM package at Administration->Login Window. Be sure to also change the default background colour to something that matches!
-- Installing the splash screen required me (a 7.04 user) to install gnome-splashscreen-manager using the Synaptic Package Manager. Once installed, use Preferences->Splash Screen to install and activate the splash image.

Unfortunately, not all icons are replaced with the green equivalent... If you have any comments, recommendations, or problems, post here!
