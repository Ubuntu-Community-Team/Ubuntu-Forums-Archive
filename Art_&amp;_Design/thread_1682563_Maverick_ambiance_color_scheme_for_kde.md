### Post by alaukikyo on 2011-02-06
If you use kde apps in gnome then you can make them look native by following [this]("http://askubuntu.com/questions/24780/how-to-make-kde-applications-look-native-in-gnome/24786#24786") and changing all fonts (except fixed width) to ubuntu font.

You will also need a matching color scheme but i coudn't find one for maverick so i made one taking [this]("http://kde-look.org/content/show.php/Ambiance?content=123960") as the base .

First extract the archive.
Then go to System -> Preferences -> System Settings (you should have [systemsettings]("apt://systemsettings") installed)  then go to Appearance then colors  then 'import scheme' now select  the extracted file.


Screenshot 
[IMG]http://i.imgur.com/XtEv0.png[/IMG]

It is released under gpl.
