### Post by bvc on 2005-09-08
On the desktop you can rt-click>properties>remove custom icon. Everything else you'll have to do manually.

I see that in the gperfection theme all different sizes are specified but there is only a gtkpod.png for 24x24 and 48x48. So, if the menu is looking for a 32x32 it will revert to the default because it thinks you prefer to use the correct size icon over scalling the rt icon.
