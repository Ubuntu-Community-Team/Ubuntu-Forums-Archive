### Post by hok00age on 2010-08-13
Islamic plymouth theme

Install:
sudo add-apt-repository ppa:tldm217/tahutek.net
sudo apt-get update
sudo apt-get install basmalah-plymouth-theme

After install:
sudo update-alternatives --config default.plymouth
#(choose the number of basmalah theme)

Then:
sudo update-initramfs -u

Hope you like it.
