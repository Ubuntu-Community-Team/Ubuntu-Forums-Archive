### Post by rsambuca on 2007-11-17
Just add an alpha channel, then use the selection editor to select the white background.  You may have to play a bit with the threshold levels to select everything perfectly.  Then "Colour to Alpha" and save as a png.

EDIT:  It doesn't show on the forum's white background, but the image white background is now transparent.

### Post by libertypo on 2010-07-13
Use the path tool > make the selection > click on path to selection > right click on the layer > add layer mask > from selection > right click again on the layer > apply layer mask. There, you extracted the image. You better zoom the image so you can see which pixel is which
