### Post by skfin on 2011-09-25
Hey,
I wish to thank you very much about the SMF theme you've made. We are  currently making a new site for Ubuntu Finland LoCo, based on Drupal 7.  Since there are no any Drupal 7 - SMF migration plugins, I took your  theme as a base for our forums, and manually added the header, menus and  footer generated by Drupal.

It was relatively pain-free since I got it  done in just few hours. There are still some conflicts between naming in  Drupal CSS files and SMF theme files, but it wont take too much time to  fix them.

I have modified some things from the theme, so it fits inside the content are of the site. Basically I removed the header and background stuff so it fits inside of the content area.

You can see the modified theme here: [https://skfin.viuhka.fi/ubuntu-smf/](https://skfin.viuhka.fi/ubuntu-smf/)

What license your theme has? Can I publish source codes of our version of the theme? Of course we would mention the original developer :)
