### Post by maruchan on 2010-09-22
Presumably you'll be using bitmaps. Blender would work, but you'd need to enable the orthographic camera feature in order to get really straight grid lines in your renders.

I would tend to recommend GIMP. I've [used it for game graphics]("http://www.flickr.com/photos/porkfriedrice/sets/72157624196786634/") and it works great. It does have guides (like gridlines) that you can use, and it comes with a filter (Filter --> Render) that will draw grids. You could draw a grid layer pretty easily and use it as an overlay.

Inkscape is another one to look into, but GIMP is probably the most common to use in this situation.
