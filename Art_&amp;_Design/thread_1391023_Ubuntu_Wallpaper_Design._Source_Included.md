### Post by skymera on 2010-01-26
Hello :)

Being bored I decided to make my first ever wallpaper. While i was messing around with the colours, i pasted the Ubuntu logo for some colour inspiration, and it looked pretty cool! (Well i think it looks cool). So I decided to post here.

Here it is:
[[IMG]http://img97.imageshack.us/img97/8568/ubuntulogos.th.jpg[/IMG]](http://img97.imageshack.us/i/ubuntulogos.jpg/)

I have also uploaded the .PSD in a ZIP file [**->HERE<-**]("http://rapidshare.com/files/341407167/final.zip") so anyone can edit this to their tastes. It has all seperate layers. 

To get colours, create a new layer and set the mode to "color". Then simply apply a gradient. Voila.

Credits goes to my friend **Teo** who created a tut originally in [**THIS**]("http://www.ps-garage.com/forum/viewtopic.php?t=3804") thread 

I am a Photoshop amateur, so don't shoot me down :KS

If anyone is actually interested :O I can post higher res
