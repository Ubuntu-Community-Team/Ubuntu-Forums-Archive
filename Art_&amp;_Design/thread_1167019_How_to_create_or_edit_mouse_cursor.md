### Post by Krom2k on 2009-05-26
OK, here some news.
I found the program called **Xcursorgen** which is part of the **x11-apps** package for Ubuntu (*see Synaptic Package Manager*). This program allow you to create x11 cursor files from PNG files. It is possible to create fixed and animated cursors.
