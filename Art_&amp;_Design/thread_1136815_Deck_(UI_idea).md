This is a mockup of little UI idea. In Ubuntu brainstorm [http://brainstorm.ubuntu.com/idea/19424/](http://brainstorm.ubuntu.com/idea/19424/).

This, the transparent thing, is deck (dock+desk=deck). It's a way to navigate windows.

First, you need to hover bar, then deck pop-ups (like Windows autohide bar). While hovering program, the desktop "minimap" and howered window model are light-up. Also, while hovering, for example, minimap of desktop 3 &#8212; only its programs are showed in deck.

The point is - this deck allows to store huge amount of items, and navigate them. If there is'not enougth space, deck may rise to the top of monitor. On 800x600 it maximally gives space for over 100 items - witch is definatly enough :-).

It also may open on hotkey. Or, witch would be great, on mouse extra button (for example I have a4tech x750bf).

That's pretty much it. Ask qestions, if it's not completly clear.
