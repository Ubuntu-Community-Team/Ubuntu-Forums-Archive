### Post by lyceum on 2009-02-26
to change the look (icons, windows colors etc..) download the ones you like to your desktop, go to Systems (top bar) Preferences, Appearance (make sure you are on Themes) and drag the files from you desktop and drop them onto the themes box. It should ask you if you want to use what was just loaded. If you want to mix and match to see what you like after installing more than on set of icons, etc.. click on customize in the theme tab. 

Hope that helps.
