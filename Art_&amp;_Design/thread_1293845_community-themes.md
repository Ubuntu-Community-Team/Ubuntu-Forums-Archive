### Post by bpickel on 2009-10-17
This may already be old news here but i just had to post expess how awesome I think the community themes package is. I just stumbled on it today and after a quick install I have the look on ubuntu I have spent countless hours trying to find only to discover some weird display issue that was a deal breaker. 

These themes are as tight as anything shipped with the OS. The Hanso is my fav. If you don't have these yet then



sudo apt-get install community-themes


Then check System > Preferences > Appearance 


Have fun.
