### Post by timetunnel on 2006-07-16
No need for the whole kdesktop. The package "kcontrol" contains the KDE Control Center which, among others, let you change the Look & Feel of KDE apps. IIRC the package won't install menu item in GNOME, so just start it from the command line with "kcontrol". 

BTW: I use this ubuntu color scheme for KDE apps:
[http://www.kde-look.org/content/show.php?content=39402](http://www.kde-look.org/content/show.php?content=39402)
