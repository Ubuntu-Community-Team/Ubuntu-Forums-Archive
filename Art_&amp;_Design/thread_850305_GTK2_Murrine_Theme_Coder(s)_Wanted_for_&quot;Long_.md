### Post by linbai on 2008-07-05
The Long Term Vision (LTV) theme needs one or more (experienced) theme programmers to collaborate in developing a new Ubuntu theme concept.  All progress on this theme will be regularly submitted to the Ubuntu Art Team via the Wiki site.  "LTV's" intention should be to be the best possible user interface and theme design, not necessarily to become a default.

Visit the Launchpad team site to join the project.

Wiki: [https://wiki.ubuntu.com/Artwork/Incoming/Long_Term_Vision](https://wiki.ubuntu.com/Artwork/Incoming/Long_Term_Vision)
Launchpad Team:[https://edge.launchpad.net/~long-term-vision-theme-team](https://edge.launchpad.net/~long-term-vision-theme-team)
Launchpad Project:[https://launchpad.net/long-term-vision](https://launchpad.net/long-term-vision)

The theme design is very flexible in terms of changing direction right now, so anybody that is interested will have quite a bit of influence.  First phase of theme development (after defining a milestone schedule) will focus on exchanging ideas and concept art, with a healthy dose of reality-check from programmers as to what is really feasible.  Nothing in LTV is written in stone, but it has to be relatively easily implementable.  The theme should hold close to Ubuntu design norms, but also push for new UI innovations.  

For more information, please see the attached images, or visit the wiki site.
