### Post by AlexEagar on 2007-06-29
The Human icons look great, but unfortunately they clash very badly with other colors. The way that icons are handled should be refactored so that the orange in the Human theme can be replaced with any color via a color picker. This would allow a user to choose a color that matches or complements the computer's desktop picture and theme. I think that a much larger percentage of people would be pleased with this solution. I am suggesting this because of the frustration I have had trying to find a decent icon replacement. It should be a one step process of replacing the orange in a color picker but instead is a long and tedious process that for me ultimately ended in failure. Here are some of the steps that I and probably many others have gone through to find replacement icons.

1. Search for web site that contains icon themes
2. Search for a good icon theme
3. Pick an icon theme based on a preview of 3 to 6 of the icons in the theme
4. Download icon theme
5. Install icon theme
6. Find that some icons look great but the icon set is not as complete as Human icons
7. Become frustrated
8. Return to step 1 or 2 many times
9. Mix and match icons for a couple hours to try and replace the clashing orange icons
10. Fail to make a good set of icons by mixing and matching
11. Become more frustrated
12. Give up and start using Human icons again
13. Wait a few months and start at step one again

What is the name of the project or organization who controls how icons work. I will also suggest this idea to them.
