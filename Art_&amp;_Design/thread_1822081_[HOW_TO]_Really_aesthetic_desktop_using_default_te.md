### Post by bruno philipe on 2011-08-10
Some people (until recently, including me), says that the Apple OSX is the only good looking operating system.

The first time I installed Ubuntu, the first thing I did was install the Macbuntu theme on it.
It looked great, but I started to find it boring after some time.

Recently I had a problem with my HDD and had to reformat it. It was my chance to have fun with the Ubuntu possibilities again.

Using mixed default (Ubuntu 10.10) themes, Docky, the Edward Hopper's painting [Nighthawks]("http://en.wikipedia.org/wiki/Nighthawks") and the [Faenza icon pack]("http://tiheum.deviantart.com/art/Faenza-Icons-173323228"), I archived the best looking desktop I have ever used.

Here comes a preview (click to view large):
[[IMG]http://i56.tinypic.com/21l8kyc.jpg[/IMG]]("http://i52.tinypic.com/16l9sih.png")

To archive it, simply install Docky (from the Ubuntu Software Center) and the icons pack.
Go to themes, customize.
[LIST]
[*]On controls, set Dust Sand
[*]On colors, leave default
[*]On Window Borders, select New Wave
[*]On Icons, select Faenza (Not Faenza dark, it is for dark themes, with clear icons)
[*]On pointer, set DMZ (Black)
[/LIST]

That's it! Now just set the wallpaper to the Edward Hopper's painting and have fun!
The square icons on docky complete the good looking. :popcorn:
