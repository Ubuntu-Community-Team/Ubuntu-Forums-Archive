### Post by whiteraven on 2008-01-14
Here's how:

-Using the Ellipse Select tool, create a circular selection. A true circle is made by holding down the Shift or Shift+Ctl keys while dragging out the selection

-Save the selection to path by selecting the menu item Select -> To Path

The rest of the process is explained here: [Text to Path Tutorial]("http://gimp-tutorials.net/node/114")

You'll need to play a bit with the text and circle size to get the look you want. I've used this technique and it works surprisingly well
