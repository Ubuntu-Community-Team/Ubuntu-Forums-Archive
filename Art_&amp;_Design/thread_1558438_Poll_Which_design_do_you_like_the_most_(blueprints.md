### Post by Nonno Bassotto on 2010-08-22
> **Yuzem said:**
> I think so, if I find a developer willing to do so, maybe it isn't very hard to port it from firefox to chrome I don't know, maybe Andrea (the addon's developer) can answer this.

I'm not really sure, since I have never written a Chrome extension. I think that most of the code would be the same, but one has to write the menu.

Indeed Firefox extensions use XUL, a language which describes widgets. Google extensions don't, so every widget has to be created with standard web technologies (HTML+CSS+Javascript) and will not have native apperance. In this case the only widget is the menu, so I think it will be easy to make a port. (Not that this is on my plans for now.)

> **Yuzem said:**
> Yes, my main problem with that is how to clearly represent 4 different states in a 16px icon.

I think we can find a reasonable way. I'm not a designer, but if anyone here has an idea, please post it. A possibility would be a animated icon (say half time figuritas logo and half time play). I'm not sure I like it, but it is something we could consider.
