### Post by hellocatfood on 2010-10-29
Put simply there isn't anything available for Ubuntu that can do that (easily). 

You may have to learn how to use Blender and just animate it from there. Otherwise, if you're into programming, Processing with the [gemomerative](http://www.ricardmarxer.com/geomerative/) library allows you to animate text and [Nodebox](http://beta.nodebox.net/) does too.

In either instance, even if you decide to go back to using Adobe After Effects, it's something that takes a long time to do, so tread with caution
