### Post by SeijiSensei on 2012-03-30
In Kubuntu, I'd use kcharselect to copy the bullet character (U+2022) to the clipboard then paste it into the text box.  I presume there's an equivalent utility to kcharselect in other Ubuntu flavors.

### Post by winh8r on 2012-03-30
Try using the "n" key on the keyboard and select the "Webdings" font in the text layer dialog.

It is best to do one line at a time if you are doing it this way, however it would be much easier to make a list in Open Office and use a screenshot of it which you can edit in GIMP.

Hope this helps.
