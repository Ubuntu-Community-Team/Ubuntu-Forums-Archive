### Post by mystics on 2015-11-13
Wine is generally the go to. I'm not sure how good CrossOver is, but it will cost you unlike Wine.

As for whether or not it runs, I would say to assume not. Most of what I've read about Microsoft Office on Wine tends to get no better than "limited functionality". There is Microsoft Office Online, but that is itself a stripped-down version of the full product.

If using Microsoft Office is important to you, then you should avoid making a complete switch to Ubuntu. Either dual boot or run Ubuntu as a virtual machine inside Windows. That will help you maintain access to Microsoft Office while still being able to use Ubuntu for (most) everything else, and it is the safest way of making sure you can use a program that you need.
