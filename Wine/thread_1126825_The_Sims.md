### Post by scheuri on 2009-04-17
> **asdfoo said:**
> VirtualBox has support for 3d graphics, check the latest version from their website

Please be reminded that this support is still considered experimental and is not enabled by default.
Right now (and I personally would predict the next two years) playing 3d-Games using a virtual enviroment (virtualbox, xen, vmware) will not be a real alternative.

As I say so often:
Linux is NOT a particular gaming enviroment, even if I personally would love it to be.
There are some games (opensource and non-opensource) which can be easily installed. There are some vendors (such as id) which allow installation of their games natively with an appropriate installer on linux.
But after all, windows is just THE gaming platform when it comes to pc gaming...:)

Have fun
