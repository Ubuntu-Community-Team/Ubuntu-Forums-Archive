### Post by Andrew_Gilbert on 2014-12-03
Master of Orion 1 & 2 (GOG download/wine/DOSBox) 
Essentially, I'm using the DOSBox version of the game, using wine to launch it.  the default settings are fine in the conf file.  

It launches from this:
```

#!/bin/bash
cd ~/.wine/drive_c/MOO12/DOSBOX
wine DOSBox.exe -conf moo2.conf

```
 Obviously I changed the directory names to be more friendly to Linux, I hate escaping spaces.

The moo2.conf file looks like this.  I left out comments, and things you shouldn't touch.
```

[sdl] 
fullscreen=false 
fulldouble=false 
fullresolution=original 
windowresolution=original 
output=surface 
autolock=true 
sensitivity=100 
waitonerror=true 
priority=higher,normal 
mapperfile=mapper-0.74.map 
usescancodes=true 

[autoexec] 
mount c c:\MOO12 
c: 
cd \MOO2 
orion2.exe 
exit 

```
The autoexec section runs automatically, and it calls from wine's drive c, not a linux directory, because it is the windows version of DOSBox.  
The only thing on here you might consider changing is fullscreen to true.  Other tweaks as neccessary, but you will know your system.  Definetely don't play with CPU.  Yeah, I know you got 8 cores, and setting cycles to max seems like a great idea, but the more precision is applied, the worse the quality gets.  I think that is true for just about all DOS games that have graphics.
