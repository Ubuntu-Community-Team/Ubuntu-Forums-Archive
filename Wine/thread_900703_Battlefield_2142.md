### Post by Anonymous111 on 2008-12-23
srry for bringing up an old post but yes, battlefield 2142 will install and run with Wine.  It works perfectly except for two issues: 

1)  The mouse is invisible while in the menu (i think there is a patch for this but i havnt tried it)  

and

 2)   You cannot play on PunkBuster enabled servers unless you can somehow get an exception from being banned for the following error - "dissallowed program/driver"  
This is because PB detects Wine as a game hook (the worst thing you can do on a PB server second only to hacking PB itself)
