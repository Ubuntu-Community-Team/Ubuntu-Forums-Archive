### Post by yetiman64 on 2012-01-22
SC2 is pretty highly rated on winehq.org, with apparently nothing faulty, [[COLOR=Blue]--LINK--[/COLOR]]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=20882")

With that the case I would suggest you check out the playonlinux package for installing such games. It is a package that allows the installation of selected Windows software and downloads the appropriate version of Wine to run them with. Good luck.
