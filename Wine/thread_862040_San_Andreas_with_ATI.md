### Post by SoberWarlock on 2008-07-17
I'm still on the learning curve and I still experimenting on some stuff to know it better. I decided to install GTA San Andreas. I did some research before I installed it and found out that the game runs on Nvidia cards only so I gave it a shot anyways. I got it installed without any problems and launched the game using WINEDEBUG=-all wine gta_sa.exe. 

The game launched but there were graphic issues and it must be the drivers. I was running the game with an ATI 2600Pro. The game itself works including the loadings and intro cut scene without any crash. When your in-game its very choppy even when Vertext Shader Support is disable and unchecked Allow Pixel Shader.
