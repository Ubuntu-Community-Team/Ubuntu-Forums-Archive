### Post by beeldings on 2007-12-01
When you installed WINE, did you configure it prior to installing CS? How did you install WINE and what version are you using?

Also, did you install the Microsoft fonts package?

If you didn't, install it with:
```

sudo apt-get install msttcorefonts

```

I would suggest that you reboot your PC once the new fonts are installed and then try playing the game again. Post back with your results. On the other hand, if you want good, out-of-the-box support for CS and Condition Zero, I highly recommend CrossOver Office from CodeWeavers. I use it to play the CS Anthology in addition to the Half-Life Anthology, and it works very well. The Steam GUI also works very good through CrossOver.
