### Post by cwwilson721 on 2011-08-16
From the looks of it, you're not running 'ddostandard.exe' from within the 'C:' drive.

Try copying all the files you've accumulated to install that program to a folder inside the 'C:' drive, something like ```
/home/<USER>/.wine/drive_c/WHATEVER 
```
Substitute '<USER> with the username, and WHATEVER with whatever you want ("setupDDO" should work.

Then, run the setup from that folder.
