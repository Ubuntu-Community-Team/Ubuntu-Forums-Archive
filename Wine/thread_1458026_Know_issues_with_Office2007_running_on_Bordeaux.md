### Post by cogadh on 2010-04-19
Bordeaux is just a font end for Wine, so if you want info on whether or not an app should work, you should check Wine's website:
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=31](http://appdb.winehq.org/objectManager.php?sClass=application&iId=31)

As for the "preliminary" support, that usually means it isn't ready for every day use, but at the very least a placeholder for that functionality is already there.
