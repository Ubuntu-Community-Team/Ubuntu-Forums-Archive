### Post by aaronb on 2009-01-18
1. Run steam
2. Go to your games menu
3. Right click on the game you want to change
4. Click on properties
5. Click on Set launch options
6. Type in -dxlevel 70
7. Click on OK.

If that does not work it might be because the game requires -dxlevel 80 or more.
