### Post by Holger_Gehrke on 2021-04-03
The extension .scr can mean a lot of things. But if you're talking about Windows screen savers, than .scr files are actually executables (which is why back in the day downloaded screen savers were a vector for malware ...). It is possible to set up a development environment on Linux to write Windows programs and use wine for testing, but if all you want is a slide show that runs as a screensaver on windows, than irfanview is a good idea. According to the [wine application database]("https://appdb.winehq.org/objectManager.php?sClass=version&iId=7834") irfanview runs very well with wine.
