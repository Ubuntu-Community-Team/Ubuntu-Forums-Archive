### Post by cogadh on 2007-07-01
> **wingnux said:**
> How did you manage to run KOTOR 2 on wine? Whenever I try to run it I'm thrown back to my desktop with the resolution set to 640x480 =/ It runs fine with cedega tough.
That's because the opening cinematics are in 640x480 resolution and for some reason Wine doesn't handle major resolution changes from within your desktop manager very well. Installing and running in a Wine "virtual desktop" is the easiest way to compensate for that. If you want to run in full screen, then use a launch script to launch it in it's own X session. I do the same thing with KOTOR 1. You can't use the automatic update features through Wine (at least I've never got it to work), so you have to download an manually apply the patches. Updating KOTOR 1 introduces a copy protection problem to the game, requiring the use of a cracked .exe, so I don't bother updating it. Not sure if KOTOR 2 has the same problem though (been a while since I played).

Oh, also make sure you are using the latest version of Wine (currently 0.9.40).
