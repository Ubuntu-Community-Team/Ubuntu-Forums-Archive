### Post by oedipuss on 2009-05-26
One more thing to consider is what mods you'll be using.
Most have no problem whatsoever, but there are some that need OBSE (oblivion script expander or something to that effect), which as far as I know doesn't work in wine. 
Also, even if a mod doesn't require anything special, it might cause a disproportionate slowdown in wine, than it would cause in windows. 

When I recently tried it again, the game was playable at an acceptable framerate, without music. There were however various utilities such as an fps optimizer (or rather stabilizer) that wouldn't run in wine, which gave windows a large advantage in fps and smoothness.
