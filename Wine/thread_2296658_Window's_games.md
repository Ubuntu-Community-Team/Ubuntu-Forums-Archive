### Post by ian-weisser on 2015-09-27
Well, you [install wine]("https://help.ubuntu.com/community/Wine").
Then you run your windows software using wine. Sometime it works, sometimes not.

Wine is a volunteer project to duplicate a set of Windows system function in Linux. That's really hard to do since they don't have the Windows source code to work from, many of those functions are not at all obvious, and the number of skilled volunteers willing to do the work is fairly small.

Wine maintains a [searchable database of applications]("https://appdb.winehq.org/") that people have tested...and the results of that testing.
The [Sims 4 might work, might not]("https://appdb.winehq.org/objectManager.php?sClass=application&iId=16273").
