### Post by cogadh on 2012-04-18
If you are still using the same version of Wine, you need to upgrade it. Current stable version is 1.4 and development version is 1.5.2, though I think there haven't been any 1.5.2 packages made for your version of Ubuntu. Follow the instructions [here]("http://www.winehq.org/download/ubuntu") to get a more up to date version of Wine. If you follow the Wine 1.3 installation ink, it should actually install 1.4.
