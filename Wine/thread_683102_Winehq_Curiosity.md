### Post by happyhamster on 2008-01-30
1. Either way is possible. You can download an installer (a .deb file) here:
[http://wine.budgetdedicated.com/archive/index.html](http://wine.budgetdedicated.com/archive/index.html)

Just make sure you match the package with your ubuntu version (latest = gutsy gibbon, i386 = default). When the download is done, you can install by double-clicking the .deb file.

If you want ubuntu to take care of things, you'll have to do some preparing first. See:
[http://www.winehq.org/site/download-deb](http://www.winehq.org/site/download-deb)

(If you install wine without adding the extra repository, then you'll get an older version, wine 0.9.46  I believe.)
Advantage(?) of this method is that you won't have to check for updates manually.

2. It's about 10 MB.

Good luck, and dond't expect too much: it usually takes quite a bit of trial and error to get applications running using wine (if at all).

[http://www.winehq.org/](http://www.winehq.org/)
