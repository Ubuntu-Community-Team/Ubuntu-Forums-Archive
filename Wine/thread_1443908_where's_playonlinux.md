### Post by xhalarin on 2010-03-31
I've found that it's helpful after you've installed something and can't find it to go into Synaptic Package Manager.  Search for the program you installed and select it.

In the tabs below, there is a tab for Installed Files.  Click on that and search through the file listing looking for any files in /usr/bin or any of the other bin directories.  Those will give you the commands to execute.  If needed, you can then Edit Menus and add that command.
