### Post by Cheesemill on 2012-03-09
> **daslinkard said:**
> Wine is a Windows Emulator

**W**ine **I**s **N**ot an **E**mulator

It's an abstraction layer that lets you run certain Windows executables directly on Linux by translating Windows API calls.

Check out the Wine app database to see if your game will work or not:

[http://appdb.winehq.org/](http://appdb.winehq.org/)
