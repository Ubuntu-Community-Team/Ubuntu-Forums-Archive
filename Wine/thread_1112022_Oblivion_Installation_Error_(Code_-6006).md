### Post by polo12 on 2009-03-31
I'm trying to install Oblivion on my machine running Ubuntu 8.10, and WINE 1.1.18. I've followed the Oblivion:Linux wiki guide step for step, however when I go to actually install the game I run into a problem. 

After typing ```
wine setup.exe
``` into the console, InstallShield pops up with an error immediately. 

Error code: -6006

It suggests closing any running programs, emptying my temporary folder, and checking my internet connection.

I can't figure this one out. Any help would be appreciated greatly, thanks.

*********** Fixed, Solution Under ***********

If anyone has a similar problem, check your drive settings in WINE. I had two drives accessing the DVD at once.
