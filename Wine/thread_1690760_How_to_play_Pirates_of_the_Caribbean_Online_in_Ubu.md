### Post by imnotjack on 2011-02-18
So how do you play Pirates of the Caribbean online while in Ubuntu Linux? Here's how. 
(1) Install Playonlinux from the Ubuntu Repositories.
(2) Install updates for Playonlinux by opening the program and let it do its thing.
(3) Click install on the playonlinux default window. 
(4) Click the category called " internet "
(5) Click on Internet explorer 7. (Will also install adobe Flash Player. This will automatically download and install everything needed for running ie7 including the browser itself. it will then activate the installer and you will see the ie7 install screen.)
(6) Go through the install that it gives you. 
(7) Download the PotCO installer using this address.

[http://download.piratesonline.com/english/currentVersion/PotC-setup.exe](http://download.piratesonline.com/english/currentVersion/PotC-setup.exe)

(8) Save the file.
(9) Right click on the file after it is downloaded. go to Properties,then the permissions tab. Check the box at the bottom that says "enable executing file as program"
(10)Go back to playonlinux found in your games section under applications menu
(11)Go to "install" again and click on the link at the bottom left of the window that says " install .pol or unsuported application."
(12)In the installation wizard select manual installation.
(13)Click Forward
(14)select " install a program in a new prefix "
(15)Name the New Prefix " PotCO "
(16)select " Configure Wine "
(17)In the Wine window, select "Add Application" and navigate to the file and double click on it.
(18)Set Wine to treat it as a Windows XP Program.
(19)Close that wine config window and use playonlinux to navigate to the potc installation file.
(20)run the install from there as you would from windows.
(21)When that is done, go to the graphics option at the bottom of the window that pops up and select opengl and login. it should act normal from there. it did for me. YAY!!!!!!!!!!! FINALLY!!
Enjoy
