### Post by wildmanne39 on 2017-02-06
If you are running windows and xubuntu is in a virtual machine just run it on windows.

If you have windows in a virtual machine just install it like you normally would in windows.

If you want to run it on xubuntu you would have to install wine, you can check the following link to see if it will work and how well  it will work in wine.
[https://appdb.winehq.org/](https://appdb.winehq.org/)
