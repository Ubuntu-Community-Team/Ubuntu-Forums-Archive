### Post by Mark Phelps on 2013-05-16
Wine does not run all Window apps, in fact, it only runs some apps OK.

You need to go to the WineHQ website and search the Application Compatibility tool for the app and version you're trying to use.

If it's not listed, or if the rating is less than Gold, trying to use that app is largely going to be a waste of time.
