### Post by cogadh on 2009-02-08
Check Wine's [Application Database]("http://appdb.winehq.org/") to see if your application actually works with Wine, as well as what you might need to do to get it running.

As for uninstalling things from Wine, you just need to either run "uninstaller" in a terminal or use the Applications menu shortcut to run Wine's uninstaller application.
