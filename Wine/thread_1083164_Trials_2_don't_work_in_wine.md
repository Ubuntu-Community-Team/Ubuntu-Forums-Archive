### Post by MarblePanther on 2009-02-28
> **hikaricore said:**
> In the future you should probably check the appdb thoroughly before purchasing a game for an operating system it isn't intended to run on.

Right.  Don't go buying new Windows games and expect them to work with Ubuntu.  WINE is just a helpful application that may or may not work with a particular windows app.  Windows games are not intended to work on anything but Windows.  The fact that some do work with WINE is the result of the dev's hard work.
I suggest looking at sites like the one in my sig for native Ubuntu games or making sure you check to make sure your game will work with WINE before you buy it.
