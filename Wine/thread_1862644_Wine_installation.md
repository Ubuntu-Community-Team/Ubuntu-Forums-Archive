### Post by Perfect Storm on 2011-10-17
You properly wants the latest version of Wine to run these games: [http://www.winehq.org/download/](http://www.winehq.org/download/)


Open the terminal:

```
sudo add-apt-repository ppa:ubuntu-wine/ppa
sudo apt-get update && sudo apt-get install wine1.3
```

Now wine repository are added and the wine is installed, and you'll receive updates automatically.


If you find wine to difficult and want something easier (and support the wine project financially for better support), try Crossover Games: [http://www.codeweavers.com/products/cxgames/](http://www.codeweavers.com/products/cxgames/)

For what you are describing, the correct terminology is command line or command line interface (CLI), as opposed to graphical user interface (GUI).  Code and coding is what programs themselves are made from.

Command line and GUI each have their place.  If you want to learn more, search for "linux command line" or "unix syntax".  Linux was derived from unix so many unix commands transfer over to linux.
