### Post by cogadh on 2011-03-23
Wine is not designed to work with an existing Windows installation and in fact, much of it is designed to prevent you from doing that. In most cases, the least headache-inducing method is to actually install the game; that way Wine has all the necessary registry information it and the game might need. Before you try anything, you should check Wine's [Application Database]("http://appdb.winehq.org/") for any details on the software you want to try to run in Wine.
