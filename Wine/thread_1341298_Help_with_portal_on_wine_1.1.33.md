### Post by Dorrax on 2009-12-14
Ok, so I got past the black screen issue by upgrading to 9.10 and using the latest version of wine, not the one in the repositories. The WineHQ website has instructions on getting the latest wine. You'll need to install directx 9 into your wine install, and go into steam, my games tab, right click on your game, select properties and type -dxlevel 81 into the box. 

After doing this I can start the game and get to the menu but it seems to freeze when I try to join or create a server. Also generally slow. I'm using a laptop with one of those motherboard based gpus. So i think I had enough power to play TF2 when using XP on this computer, but not enough to make it work with wine, because wine will need more processor power to make the game run at normal windows speed.
