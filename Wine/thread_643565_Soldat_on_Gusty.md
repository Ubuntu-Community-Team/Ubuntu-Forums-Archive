### Post by cogadh on 2007-12-18
I'm guessing that the problem is due to your video hardware. For one, the ATI driver in Linux is still developing and does not run at its peak yet. Secondly, the Xpress 200M chipset is no powerhouse, not to mention it is an integrated graphics solution and is likely sharing your system RAM instead of having its own dedicated VRAM. Integrated graphics processor are really not known for their gaming performance, no matter what brand it is.

### Post by cogadh on 2007-12-18
If you are talking about running any of those games in Windows (since C&C: Generals doesn't actually work in Wine, I assume you are), then there is no real comparison. Yes the hardware is the same, but the drivers, OS and other support software are not. The fact that a game can run in Windows fine on that hardware is absolutely no indication of how it will work in Linux through Wine using the Linux ATI drivers.
