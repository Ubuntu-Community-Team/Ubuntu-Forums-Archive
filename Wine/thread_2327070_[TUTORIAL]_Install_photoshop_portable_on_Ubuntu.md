### Post by H72david on 2016-06-07
Hello i found a way how to install Photoshop Porable on Ubuntu 14.04 (Try on more Ubuntu versions and let me know if it works), i thought maybe more people want to know how to install it.


[SIZE=7]TUTORIAL[/SIZE]

1. Open your terminal
2. Type in terminal [HTML]winecfg[/HTML] ( a window should pop up)
3. Down you should see a Operating System name (mine is Windows XP)
[IMG]http://i.imgur.com/E8nglPt.png[/IMG]

4. Press on it and then scroll up until you see Windows 7 
[IMG]http://i.imgur.com/4872Zfi.png[/IMG]
5. Press on "Windows 7"
6. After you pressed it, you need to press "OK"
7. Open PhotoShop Portable, and it should work

[SIZE=7]
BUGS
[SIZE=2]1. Most of the fonts are missing ( I am trying to fix it)
2. Most of the time the brush tool does not work
(Maybe more bugs that i don't know, let me know if you find more)

INFO ABOUT THE TUTORIAL
[SIZE=1]
