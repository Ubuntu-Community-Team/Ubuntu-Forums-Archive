### Post by Tweak42 on 2012-10-23
That doesn't sound like wine but the HUD from the Unity window manager. 
You can disable that by installing the CompizConfig Settings Manager (CCSM for short) and in the Ubuntu Unity Plugin settings disable the "Key to show the HUD".

Mind that was how I disabled it, there maybe a easier simpler way in a different tool.
