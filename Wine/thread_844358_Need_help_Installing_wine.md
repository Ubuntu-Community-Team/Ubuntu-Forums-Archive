### Post by krendar on 2008-06-29
Download the appropriate .DEB file from here with an OS with working internet:

[http://wine.budgetdedicated.com/](http://wine.budgetdedicated.com/)

Then install the deb file on your Ubuntu system using:

dpkg -i file.deb

Replace "file.deb" with the name of the wine file.

There might be some dependencies though, I am not sure what else it wants sorry.
