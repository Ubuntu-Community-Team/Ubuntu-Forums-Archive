### Post by jacksaff on 2010-03-27
You may have some luck running programs off of your windows drive but it's not recommended.
Put the disc in and click the installer. Wine should fire up automatically and if the program works well under wine installation should work much the same as under windows. Wine should add entries to your menus for most programs too these days.
Wine on it's own doesn't do anything so there is no point in having a menu entry for it. It takes the program you want to run and translates all of its communication with the system into calls that linux can understand.
If you want to 'see' wine running, start a program from the terminal with  'wine /path/to/program.exe'
