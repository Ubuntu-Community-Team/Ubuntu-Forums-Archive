### Post by EuclideanCoffee on 2014-09-06
Hello fellow Czech friend. ):P

Can I recommend a different program?

PlayOnLinux uses multiple versions of Wine, but it is much easier to manage and configure. So if something doesn't work with one version of Wine, you can install a new Wine version and configure it for that program that might not work with whatever version without needing to reconfigure any other Wine for any other program.

It also provides a tool to close any Wine programs that might be running in the background. If that doesn't do the trick, I recommend rebooting.
