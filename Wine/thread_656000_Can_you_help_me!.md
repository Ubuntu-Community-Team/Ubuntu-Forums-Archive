### Post by yabbadabbadont on 2008-01-02
System->Administration->Synaptic Package Manager

Search for wine and install it.  You may have to go to the Settings->Repositories entry to enable the extra repositories before it will show up.

If you want to install the latest and greatest version of WINE, use the directions provided at the WINE website for Ubuntu.

[http://www.winehq.org/site/download-deb](http://www.winehq.org/site/download-deb)
