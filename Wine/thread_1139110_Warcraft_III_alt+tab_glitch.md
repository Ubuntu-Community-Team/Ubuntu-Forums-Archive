### Post by FrozenFox on 2009-04-26
The winehq app database lists warcraft 3 here..

[http://appdb.winehq.org/objectManager.php?sClass=version&iId=3126](http://appdb.winehq.org/objectManager.php?sClass=version&iId=3126)

and near the top, it has a bug list like every other app, including the bug in question, which links to..

[http://bugs.winehq.org/show_bug.cgi?id=13547](http://bugs.winehq.org/show_bug.cgi?id=13547)

Post 53 there gives a workaround, and there is also a patch in there if you want to recompile wine with it. There are lots of tutorials around for how to do such things if you don't know how and want to do that instead.

Whether or not this works I do not know, as I've not played wc3 in some time.

You could also probably run the game in another X session and do ctrl+alt+f7/f8 to switch between desktop and game and likely not have this issue.
