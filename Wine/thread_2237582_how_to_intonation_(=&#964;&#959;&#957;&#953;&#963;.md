### Post by Claus7 on 2014-08-02
Hello,

I wanted to use wine/crossover office with MS office, and I faced a problem with the Hellenic (Greek) language:
I was not able to use intonation: e.g. é or &#940; e.t.c..
ubu version: 14.04

In order to solve it go to: Applications -> System Tools -> System Settings and under the tab Personal choose "Language Support". Then change the -*Keyboard input method system*- from IBUS to **none**.

It would be nice to restart your ubu box.

Regards!
