### Post by Boab1993 on 2012-04-03
To resolve this issue, download wine tricks,

Select either 'Select default wineprefix' or 'Select steam(Steam)'

Then go to 'Install Windows a DLL or component' and Scroll down the list until you come across 'dotnet11, dotnet11sp1 ...etc'

Tick the box for the one applicable to your computer (i found that MS .NET 1.1 worked best)

Click Ok and an install wizard should start

However there is an issue with running the program itself on wine and ive yet to find a fix for yet

Hope this helps
