### Post by linooxlee on 2014-03-01
linux unified kernel, or longene is a modified wine, which wineserve has been moved into linux kernel as a kernel module to to improve the operating efficiency of Wine. now, some windows apps such as ms office, firefox run on longene 1.0 very well.

Linux unified kernel 2.0 will support the windows apps run on ARM machine. the source codes:[http://www.longene.org/fileDownload.php?id=73&page=download](http://www.longene.org/fileDownload.php?id=73&page=download)
