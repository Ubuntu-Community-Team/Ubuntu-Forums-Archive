### Post by forrestcupp on 2012-05-25
I'm not sure about that, but you can get some debugging information by running it from the terminal. Just run the command with the name of what you're trying to run, which is the name that appears in your PlayOnLinux windows. So if you wanted to run Microsoft Word 2007 from the terminal, you would type
```
playonlinux --run "Microsoft Word 2007"
```The output might give you clues about what is going on.
