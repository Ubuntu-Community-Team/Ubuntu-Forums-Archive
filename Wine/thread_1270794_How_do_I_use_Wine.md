### Post by del_diablo on 2009-09-20
You got 2 ways:
```
sudo apt-get install wine
```
this in a terminal.

OR:
Find synaptic package manager somewhere under the admin menu and find wine there and select to install.

In most cases .exe and .msi would automaticly be detected as wine files, so it is just to doubleclick and install or play(alot is playable on different particpartions to, like if you dual boot).
