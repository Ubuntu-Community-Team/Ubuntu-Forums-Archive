### Post by rai4shu2 on 2012-09-27
Wine rarely works seamlessly with applications. You need to check out the notes for each particular app you want to debug.

[http://appdb.winehq.org/objectManager.php?sClass=version&iId=1177](http://appdb.winehq.org/objectManager.php?sClass=version&iId=1177)

> Additional Comments

I'll recommend to do following:
*rename Movies folder to disable cinematics
*run WarCraft in OpenGL (with -opengl parameter or registry tweak)
*on Unity DE run with Virtual Desktop
And, i am wondered, but it runs on intel gma 3150!!! (on this graphics, game doesn't run on windows!)
