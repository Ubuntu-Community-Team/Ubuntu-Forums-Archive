### Post by beastrace91 on 2009-11-20
You install them in the same way you install them in Windows - only you place them in the fake C drive Wine creates in your **~/.wine** directory.

That being said more than a couple NOCD patches are known to cause stability issues on Wine (how ever others work perfectly fine, I for example use one with my Morrowind install so I do not have to drag my CDs around with me)

Also depending on what application you plan to try and run I would also recommend [upgrading your Wine](www.winehq.org/download/deb) to the latest version.
