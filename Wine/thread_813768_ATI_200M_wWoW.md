### Post by jbaerbock on 2008-05-31
Ok so I have been wanting to ditch windows for a loooooong time however i love WoW. But with my card WoW was impossible to get working well. However I now got it working perfectly with no FPS loss to speak of by adding the following line to my config.wtf

SET M2UseShaders "0"

I also did the general regedit and etc...but in the end this added to the usual stuff turned out making WoW run like a champ :D.
