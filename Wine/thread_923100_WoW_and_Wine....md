### Post by aoanla on 2008-09-18
Upgrading Wine doesn't delete or otherwise affect the contents of your .wine folder - all your games will be fine, regardless of how many times you fiddle with the version of Wine you have installed.

### Post by aoanla on 2008-09-18
If you've added the repository at wine.budgetdedicated.com, then you can just run Synaptic again and you'll get Wine 1.1.4 installed.

Otherwise, you can download the .deb file from the same site, and double-clicking it will probably install it (I tend to do things from the commandline, but that should work).
