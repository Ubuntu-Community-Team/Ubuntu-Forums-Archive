### Post by deadflowr on 2017-01-27
Depends.
Best thing to do is look through the wine database for basic reviews
[https://appdb.winehq.org/]("https://appdb.winehq.org/")

You might also find that while a program runs perfect under one version of wine, it might break completely under a different version.
A nifty solution is to run the wine program playonlinux, which will allow you to run different versions of wine.
So you can have some apps that work in one version and other apps that work in other versions.
