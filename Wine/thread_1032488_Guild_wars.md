### Post by donkyhotay on 2009-01-06
This should really be posted in the wine forum but... I've used guildwars with wine. It runs pretty well, some minor graphic glitches and stuff but nothing that prevented me from playing the game. I found guild wars to be easier to get going with wine then some other games I've played (such as starcraft for example) but be aware that wine is a compatibility layer so it might not run flawlessly by double-clicking on it like windows. I would recommend just trying it, you should be able to get it going relatively easily and be good practice for when you want to get a less compatible program to try to run with wine.
