Check in the Wine forum and do a search for World of Warcraft -- many threads should come up, including a how-to on getting it installed and configured.

Short answer -- it should work fine using Wine as long as you have a decent graphics card (nvidia or ATi).

### Post by Yuki_Nagato on 2008-07-07
An excellent point.

Ubuntu must recognize that you have a 3D graphics card.  Otherwise, WINE will flip when trying to run.

At the worst, you can set aside a small partition for a Windows XP dual boot.
