### Post by tyle on 2009-02-04
[http://packages.ubuntu.com/intrepid/wine](http://packages.ubuntu.com/intrepid/wine)

The default package is wine 1.0.1, so you would get that one in intrepid, unless you have added the wine repositories in which case you will receive the latest 1.1.x release instead.

Version 1.0 and 1.0.1 should be almost identical, as some minor bugfixes and added translation was the only changes that it contained.
