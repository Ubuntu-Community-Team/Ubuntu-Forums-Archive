To use Photofiltre in Ubuntu you have to install Wine first. If you have installed Wine, you can download Photofiltre from the Photofiltre website and if you double click the .exe-file Wine will be started and the Install program of Photofultre will be started. In the Application menu an item will be added to start Photofiltre.

I love to work with Photofiltre because it is free available and it is easy to work with. Photofiltre can be used without any problem in Ubuntu.
