### Post by buntunub on 2008-03-20
Get PlayOnLinux and it will do everything for you. You can use that tool to manage your wine versions as well for each game/program you install on it, as well as create seperately managed wineprefixes. You can also install the latest Wine and try to hack it yourself like many of us do, but PlayOnLinux is a GPL'd frontend to Wine which was built to make using Wine easy.
