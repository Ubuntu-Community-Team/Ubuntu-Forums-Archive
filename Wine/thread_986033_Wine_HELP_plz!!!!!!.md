### Post by DieB on 2008-11-18
well there are three ways:

1) open a Terminal (in Gnome: Applications-Accesoires-Terminal) and type > sudo apt-get install wine (it will then prompt for your password)

2) use "add/remove" you can find it in the Applications-Menu search for wine, mark it and apply changes (it will ask you for your password than)

3) use synaptic (System-Administration-PackageManager) it will prompt you for your password and afterwards you can browse all avaible software, mark it for installation and apply it.


hope it helps
