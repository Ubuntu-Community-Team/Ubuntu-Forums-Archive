### Post by jken146 on 2007-12-28
Well, it depends on which programs you're talking about.  After all, Windows programs are meant for Windows, not Linux.

First, look for a native Linux version or alternative.  If it's games you want then you'll find quite a few first person shooters and not a lot else I'm afraid.

Second, try Wine.  Go to [www.winehq.org](www.winehq.org) and see if your program has been run successfully before.

Then there is Cedega, which is much like wine, but is geared a bit more towards gamers, but you have to pay for it.

You could also install Windows in a VM, or dual boot.
