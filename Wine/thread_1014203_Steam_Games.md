### Post by Interpreter on 2008-12-19
Be advised though, don't kill your native Windows partition just yet. Some few steam games work fine, the new ones usually require major compromises, eg LOW fps, graphical glitches, ....even with decent machines you probably will end up playing source games in DirectX 7 mode and such things.....

It's basically fine if you like starting up old Hl1 mods every now and then. Newer games suffer from issues not to be underestimated.
Especially when it comes to "competitive playing" ...it's honestly not enjoyable...

Steam per se works fine enough to start games, not fine enough to use the community feature or to chat with friends.
PlayOnLinux, as said by a nice person above, is a small tool that uses alot of scripts to make installations of games easier.

For example without POL you'd have to install steam, apply a special patch, install a game, manually tell it to only start in Direct X 7 mode, then apply another patch so that your mouse is working the way it's supposed to be and whatnot. POL does all that for you, GIVEN that they actually support your game.
See their website for a list of funny fun funs.
Cheers FunnyMan.
