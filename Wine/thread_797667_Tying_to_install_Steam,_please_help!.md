### Post by Rhubarb on 2008-05-17
I have steam working fine here (Ubuntu 8.04), I'm using the latest version of wine to do this.

You can use winehq's repository to always have the latest version of wine - get it here:
[http://www.winehq.org/site/download-deb](http://www.winehq.org/site/download-deb)

If using the latest version of wine doesn't help you to install steam, have a look here:
[http://appdb.winehq.org/appview.php?versionId=1554](http://appdb.winehq.org/appview.php?versionId=1554)

If you're unsure if steam has locked up, go to:  System --> Administration --> System Monitor
Then click on the Resources tab, and check to see if your PC is receiving lots of data (it could be steam trying to update itself).
