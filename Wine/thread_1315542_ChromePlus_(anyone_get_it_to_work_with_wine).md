### Post by grizzlylock on 2009-12-01
For anyone interested in the ie-tab pluggin on chrome which chromeplus supplies in windows there is a way to mask chromium as ie.  I'm not sure if it works the same as an extension but it does allow you to view more ie sites even if some of the apps on the pages don't load completely right.

Call your chromium as such:
chromium-browser --user-agent="Mozilla/4.0 (compatible; MSIE 7.0; Windows NT 6.0)"

This tells the website that it is ie7.
