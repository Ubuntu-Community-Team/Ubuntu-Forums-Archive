### Post by cloudyscreen on 2008-10-26
this is the solution I've got. Fairly awkward but it works, 
theres probaly an easier way to do it but.......
heres a solution I got from web eventually,

this is with wine installed 


I copied mathcad folder onto my desktop, 
then everytime I want to open mathcad,
click on mathcad folder,
click on mathcad folder again,
click on program files,
click on mathsoft,
click on mathcad 2002i professional,
click on regtool.exe,
register components, then close , 
open mathcad.exe

"no default printer" error comes up, just close that. 

mathcad opens.

the problem I had before was that the function manager wouldn't work, which made mathcad redundent.
So far with this method everything works ok. I haven't gone through all the functions yet, but the basic ones seem to work. if anyones got an easier way to open mathcad please share:)
