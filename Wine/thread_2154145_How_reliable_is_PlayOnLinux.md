### Post by Toz on 2013-06-13
Moving thread to the ***Wine*** subforum.

Playonlinux (POL) is simply a front-end to Wine. So to answer your question about how gaming is, its pretty much the same. POL does offer some neat features for the wine experience (aside from the nicer front-end):

- multi-version wine support - it will install and manage multiple versions of wine
- pre-defined scripts - some games/apps come with predefined scripts that assist with the installation of the game. These scripts include the use of the best version of wine for the game (see previous point) plus the automated installation and configuration of necessary other components.
