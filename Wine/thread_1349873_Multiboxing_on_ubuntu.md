Well there should be a way, problem is that multi-boxing like that requires a third party program to interact with the windows unless you get really down and dirty with the needed parts in the kernel etc and third party software like that goes against the license agreement since that is more or less the same way that bots work and you might risk a bann. 

Easiest way really to multi-box is to do it "right" 
[LIST]
[*] buy 4 really low end computers, as cheap as you can while still meeting the WoW req. (no screens)
[*] buy/get and old screen
[*] a screen, mouse and keyboard switcher, pref one that can handle 1 Input USB and digitally select 1-5 outputs (this is probably the hardest one to find but I've seen them, there are alternatives to this of course or you could build one yourself ;))
[/LIST]

that should do it, one main screen, one secondary screen for the 4 extra accounts and one keyboard and mouse to rule them all ;P 
this setup is best for grinding / questing, if you are going for PvP you might want more keyboards and really learn to use them all.

there is some cost to it ofc but not that much since the 4 extra computers doesn't really need to be that good and WoW now runs on fairly old hardware.
Did a fast check on cost, I would say about $1000 US for everything, maybe even less if you find stuff the companies want to get rid of because it's so old :P

### Post by hikaricore on 2009-12-09
No you can't run mac software on linux.
There may be tons of programs out there, but they're all against the TOS.
Any software which removes or automates the process that normally requires user interaction is forbidden.
Even if you find one that will work on Linux you're likely to get banned from WoW.

If you really want to dual box you should learn to react quickly and either use alt-tab or virtual desktops.
