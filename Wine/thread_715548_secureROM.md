### Post by Sugi on 2008-03-06
UFO: Afterlight,
Open up the terminal and input this:
> $ mkdir /home/USER_NAME/.wine/drive_c/Program\ Files/UFO/backup
$ mv /home/USER_NAME/.wine/drive_c/Program\ Files/UFO/UFO.exe /home/USER_NAME/.wine/drive_c/Program\ Files/UFO/backup/UFO.exeNow, download the attachment I uploaded to the Desktop (UFO.zip).  Extracted the zip using the gui to the Desktop as well. Now, back to the terminal.
> $ mv /home/USER_NAME/Desktop/UFO.exe /home/USER_NAME/.wine/drive_c/Program\ Files/UFO/UFO.exe
$ wine /home/USER_NAME/.wine/drive/Program\ Files/UFO/UFO.exe
or in other words, Enjoy

***NOTE: 
"/.wine/drive_c/Program Files/UFO/"
(99% sure that is the correct directory for wine.  Also, I have never played or install UFO before, so make sure that's correct directory for the game.)
