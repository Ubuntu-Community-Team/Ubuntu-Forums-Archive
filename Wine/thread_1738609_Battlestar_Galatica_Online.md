### Post by cogadh on 2011-04-25
Install an internet browser in Wine (Firefox would be best) then install the Unity Player in Wine and play the game via the Wine-installed browser. There are absolutely no guarantees that this will work, but if it is going to, this is really the only way to do it (other than running Windows via dual-boot or a virtual machine).
