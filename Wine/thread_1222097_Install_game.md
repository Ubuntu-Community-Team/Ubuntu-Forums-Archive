### Post by Simian Man on 2009-07-24
The easiest way would be to copy your ~/.wine directory to their home directories.  This would have to be redone each time you install games though.

The best way would be to configure wine to use a globally accessible directory such as /usr/share/wine where all users have read/write permissions or something.
