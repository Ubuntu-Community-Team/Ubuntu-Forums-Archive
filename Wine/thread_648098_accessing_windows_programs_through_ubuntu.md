### Post by hikaricore on 2007-12-23
You would be better off in many cases reinstalling the applications that you may want to use through WINE.

Be aware that not ever piece of ***dows software works properly or at all through WINE's API conversion, however progress is still continuing every day and WINE is updated roughly every two weeks to include these changes.  ^_^

Be sure to check the WINE application database: [http://appdb.winehq.org](http://appdb.winehq.org)
To see how well your software titles run before installing them.  :)

Best of luck,

--Aaron
