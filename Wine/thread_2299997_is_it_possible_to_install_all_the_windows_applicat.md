### Post by oldfred on 2015-10-23
Short answer No.

Ubuntu is not Windows and wine only partially works to run Windows applications. If you must have Windows applications dual boot so you have both Windows & Ubuntu. Or convert to the Linux applications. In some cases the Linux applications are better, some about equal but learning new software required, and some applications in Linux are not as good a the Windows application.
