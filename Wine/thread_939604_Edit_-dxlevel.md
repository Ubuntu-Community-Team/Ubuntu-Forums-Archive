### Post by Dark9204 on 2008-10-06
Drag the .exe file to terminal and then type -dxlevel 80.

Or create a sohortcut to the file with th -dxlevel tag.
like this: env WINEPREFIX="/home/erik/.wine" wine "C:\Program Files\Steam\steam.exe" -applaunch 240 -dxlevel 81

That was my shortcut to Counter Strike: Source running at dxlevel 81.
