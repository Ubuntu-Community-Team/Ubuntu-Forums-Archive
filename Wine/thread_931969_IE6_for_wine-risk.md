Every program you run is a security risk to some extent, including Firefox.

If the sites in question detect non-IE browsers using the version string, you can get a firefox plugin called browserswitcher which will let you pretend to be IE.  But if they rely on some specific IE technology then you'll just have to use IE I guess.

If you did indeed dump Windows after a virus attack, keep in mind that viruses are windows programs too and Wine is capable of running them, so you may want to install clam-av.
