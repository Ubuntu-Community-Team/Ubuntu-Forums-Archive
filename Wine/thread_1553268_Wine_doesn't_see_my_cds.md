### Post by cogadh on 2010-08-14
Chances are, it can't really be fixed. Wine doesn't support all copy protection schemes used by Windows PC games and likely never will (it would require Wine to license the copy protection software and possibly add closed source code to Wine). 

That lack of support usually presents itself as a "please insert the disk" message from the game. If that is the case, then the only way to get around it is to use a "cracked" game executable (an executable that has had the copy protection disabled). Don't ask where to get a crack or how to use one on these forums. Since the act of cracking and/or using a crack may be illegal (it varies by country), discussion of the details of them is not allowed here. Remember, Google is your friend.
