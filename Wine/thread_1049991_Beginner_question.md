### Post by sanderj on 2009-01-25
Yes, it seems you've done the right thing.

General tip regarding Wine: first check whether there's a native Linux application that does what you the Windows app to do. If there's no native Linux app, only then try wine.

Example:
If want Microsoft Office functionality, first look for a native Linux office suite... which is OpenOffice. Check out that first.
