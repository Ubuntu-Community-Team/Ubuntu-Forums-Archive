### Post by Akingboy on 2008-07-19
Do you have same problem as I have that it freezes after 1min of play and you got ATI card?

I think that OSS sound can be run by activating it only from winecfg and maybe go "System->Settings->Sound" and put OSS everywhere.
And I prefer you to shut mp3 player or any other program that might use your sound because it works only in one program at time.
