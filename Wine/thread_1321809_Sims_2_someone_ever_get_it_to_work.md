### Post by Mr. Picklesworth on 2009-11-10
For what it's worth (since you are "up for just about anything"), The Sims 3 works flawlessly ;)

(Better than it did on Windows, in my case).

The best place to look for help on this stuff is [winehq.org](winehq.org). In particular, its AppDB section.
Here is the page on [The Sims 2](http://appdb.winehq.org/objectManager.php?sClass=version&iId=2633).
The page on [The Sims 3](http://appdb.winehq.org/objectManager.php?sClass=version&iId=16664)

There is a script called WineTricks which you may need to use to download stuff like the MS VC++ libraries. (PlayOnLinux should do it automatically, though, if you want to use that).

Unfortunately, judging by Sims 2's WineHQ page (and we can tell it was recently tested by the version of Wine being tested with), we won't be getting anywhere with it :(

Still, keep an eye on that and something may change. Things are being fixed in Wine All The Time! If you create an account on that web site, you can click a button to monitor changes in the test results for a particular product.
(And there is a repository you can add, for Ubuntu, to always get the latest version of Wine - they mention it in the Download section of WineHQ).
