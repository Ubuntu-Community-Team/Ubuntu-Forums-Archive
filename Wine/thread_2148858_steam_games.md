### Post by CatKiller on 2013-05-26
You can only play Linux games in Linux.

You can play some Windows games with Wine, but you'll need to check each of them against WineHQ's AppDB to know if they are likely to work.

If you want to play Windows Steam games you'll need to run Steam in Wine. PlayOnLinux makes that fairly straightforward.
