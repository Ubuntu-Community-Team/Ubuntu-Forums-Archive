### Post by cogadh on 2009-01-30
The only way to run any Windows software on Linux is through Wine or one of its derivatives. Unfortunately, Game Guard will never work with Wine since it uses a driver to accomplish its task and Windows drivers don't work with Wine at all. You might be able to use [VirtualBox]("http://www.virtualbox.org/") or some other virtualization software to run a Windows installation within Linux and run the game through that, but if the game requires 3D graphics supposrt, you might be out of luck. Most virtualization products only have experimental 3D support that doesn't always work well.
