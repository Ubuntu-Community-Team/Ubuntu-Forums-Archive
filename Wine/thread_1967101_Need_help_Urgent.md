### Post by forrestcupp on 2012-04-27
Lol. This is urgent, huh? :)

First, if you're using Unity or Gnome Shell, open up the Dash and start typing the name of the game, and hopefully, it will just come up in the search.

If not, you can find whatever you installed in the .wine folder. Wine creates a virtual Windows C: drive, and that's where it installs everything. Open up the Nautilus file browser, and if the hidden files aren't showing, press Ctrl+H to show them. Then navigate in your /home folder to "/.wine/drive_c/Program Files", and you should find whatever you installed with Wine.

Hope it works for you.
