### Post by VietCanada on 2011-12-02
You have two possible solutions available.

Install Winetricks and use that to install d3dx9 which is not directx9 and is recommended by Wine.

Go to the Wine Appsdb (apps database) and search for your software for advice on how to install it properly with wine.
