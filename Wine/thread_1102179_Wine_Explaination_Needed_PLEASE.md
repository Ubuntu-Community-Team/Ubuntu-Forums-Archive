### Post by jnewl on 2009-03-21
I believe Wine is installed by default with Ubuntu. If not, use Add/Remove Applications or Synaptic Package Manager to install it. You can [go here]("http://appdb.winehq.org/") to check how to get various games to run with Wine (or whether they'll even run at all). 

It looks as if you're in luck with COD4. Users are giving it all gold and platinum ratings, which means you should be able to run it with little fuss. (Ignore all the stuff about patching Wine you read in the database entry for it; that only applied to earlier Wine versions. I'm assuming you have a newer version than 1.1.1)

Hopefully installation should be as easy as inserting the game disc, navigating to the setup.exe file, right-clicking it, and choosing "Open with Wine Windows Program Loader." Then run the game from the desktop icon, if one is created, or the entry in the Wine menu (under Applications).

### Post by jnewl on 2009-03-21
Go to Add/Remove Programs, type "wine" in the search box and wait a few seconds. You should see "Wine Microsoft Windows Compatibility Layer" show up. Tick the box and hit "Apply Changes."

(This is the general procedure to follow when looking for and installing new software. You don't need to know any command line stuff to use the package managers, as they will install everything correctly on their own. Just do a search for likely keywords based upon what you're looking for and see what comes up.)
