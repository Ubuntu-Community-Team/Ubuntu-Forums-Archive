### Post by desktorp on 2011-06-07
I realize you are talking about San Andreas, but I would like to share a similar problem that I just figured out, in GTA3.  I'm now using the Steam release, though I'm pretty sure I had the same problem with my retails discs.  Basically, when I launch GTA3 for the first time, the only thing that happens is that the resolution changes to whatever GTA likes as default.. something low.

I can still see the desktop, but if I hit enter/return twice, it's like skipping the publisher splash screen, which I think is causing launch hangups.  So at very least, try that.. maybe it's the exact same problem, two games later.

Launch > allow resolution to do its thing > hit enter a couple of times

Sorry if this is totally unrelated, but it's how I ended up actually getting GTA3 to load.  After that, I changed the settings to be appropriate for my actual resolution and the game now functions better than it ever has.
