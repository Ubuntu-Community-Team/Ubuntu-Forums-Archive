### Post by Gcentrex on 2008-09-26
Most graphic issues are ATI card related the drivers provided by ATI do not include all the features that are available under a Windows based system.

The fonts part is easy since you have a Windows drive copy the Fonts from C:\windows\fonts into Wine's fonts folder easy fix for that.

If you want to move to Linux you might consider getting an Nvidia card.
