### Post by cogadh on 2008-07-06
ALT+TAB should do it, as long as you have something else open (like a terminal window) when you launch your fullscreen game. Your other option would be to run the game in a Wine "virtual desktop" (translation: windowed mode). That way the desktop is accessible as if you are not running a full screen app.

---

### Post by lukjad on 2008-07-07
Also, if you have multiple desktops Ctrl+Alt+Right (or Left) will get you to another desktop. I'm sure you can also set up a key combo to let you minimize everything.
