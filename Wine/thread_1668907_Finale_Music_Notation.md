### Post by cogadh on 2011-01-16
Wine does not really change with Ubuntu version, every currently supported version of Ubuntu has the same version(s) of Wine available for it, currently 1.2.2 (stable) and 1.3.11 (development). For example, if an application does not work in Wine 1.2.x, it doesn't matter if you are running Wine 1.2.x on Ubuntu 9.10, 10.04, 10.10 or any other version of Ubuntu, its still not going to work.
