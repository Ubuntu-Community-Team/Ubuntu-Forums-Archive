### Post by chriswyatt on 2009-11-15
A couple of other things you could try.

Installing XP as a virtual machine (if you save machine state it's actually quite quick to get into XP from Ubuntu).  If anything goes wrong (viruses etc.) you could restore the virtual machine from a backup.  Depending on how much she needs to use IE this could be a bit annoying.

There's also IES4Linux which is a project to get Internet Explorer running under WINE, not ideal seeing as she wants to stay clear from IE but I thought I'd mention it in case you hadn't heard of it.  ( Also WINE can sometimes execute viruses as well as it can execute Windows programs ;) ).

If the User Agent Switcher works of course that's the best solution :)
