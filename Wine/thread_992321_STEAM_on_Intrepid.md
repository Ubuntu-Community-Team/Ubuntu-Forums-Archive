### Post by Swarms on 2008-11-24
> **mag1strate said:**
> Is there a way to install steam and play online on intrepid? I have heard of Cadega and WINE, but from my research you have to pay for Cadega.:confused:

Yeah there is :)

Install newest version of wine by following instructions on this webpage: [http://www.winehq.org/site/download-deb](http://www.winehq.org/site/download-deb)

Then you download the Steaminstaller from [www.steampowered.com](www.steampowered.com) and run it through the terminal with the command ```
wine start SteamInstall.msi
```

Then it will install and you can launch it from your Applications menu.
