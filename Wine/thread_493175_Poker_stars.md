### Post by drifter0000 on 2008-04-20
i use pokerstars with wine on several different distros.

The set up I use now is hardy 8.04 64bit
Wine 9.54
Wine config to XP or 2000
To resize windows the easiest is to use wine virutual desktop.

The best for me to make it work most like XP and resize windows is to configure wine not to integrate with desktop. Now you can resize windows all you want.  Only problem is no other programs can run on top of the windows.  I just minimize poker window to get to my linux windows.  Also if you config wine this way your poker tables will follow you from desktop to desktop.  Or if you are waiting to get into a game I use another desktop to use linux and watch the lobby in pokerstars on my second monitor its great.
Hope this helps.

PS newer versions of wine dont work as well with pokerstars but they do work
