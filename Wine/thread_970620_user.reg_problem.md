### Post by cogadh on 2008-11-04
Your user.reg contains some system specific settings that are different for everyone, so if you have messed it up, it is usually best to just delete your .wine directory and re-create it, rather than use someone else's settings.

EDIT - I forgot to mention, if you do end up deleting the .wine directory, you will lose anything you have already installed with Wine.

EDIT2 - If you want to try to save what you have installed, you might also try re-naming your .wine directory, then creating a new one, copy the user.reg from the new .wine to the re-named one, delete the new .wine and name the old one back to .wine.
