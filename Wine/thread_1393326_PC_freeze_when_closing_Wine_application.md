### Post by gradinaruvasile on 2010-01-29
The lates wine is 1.1.37. Update it:

[https://launchpad.net/~ubuntu-wine/+archive/ppa](https://launchpad.net/~ubuntu-wine/+archive/ppa)

Also, remove pulseaudio if you want stability for wine and other games. But that will remove you volume control applet too.
And install the latest graphics drivers.

[http://www.nvidia.com/Download/Find.aspx?lang=en-us](http://www.nvidia.com/Download/Find.aspx?lang=en-us)
