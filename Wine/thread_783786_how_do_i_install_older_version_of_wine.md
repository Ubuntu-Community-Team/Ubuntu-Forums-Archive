### Post by evozniak on 2008-05-06
Hello BBB, when my app not work in newer version of wine i install packages from this site [http://wine.budgetdedicated.com/archive/index.html](http://wine.budgetdedicated.com/archive/index.html)

to install download this package to any dir goto terminal and put this
sudo apt-get remove wine (to remove instaled version)
sudo dpkg -i wine_0.9.XX~winehq0~ubuntu~X.XX-1_iXX.deb (installs manualy the .deb package)
