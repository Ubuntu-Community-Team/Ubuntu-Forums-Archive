Hi, i've been trying to get Sonic Adventure DX installed though wine. It has worked before for me with the current development version 1.3 and the stable 1.2 with Debian. but I have decided to switch back to Ubuntu. The game is a two disc set and when I try to switch to the second cd to finish up the install, the installer doesn't detect drive D so I can't finish the install.

Any Suggestions I have tried the install in KDE and Xfce.

### Post by Topazkitty on 2011-05-15
I have solved this problem. First I went to winecfg and created 2 fake cd drives. One with disc 1 in it and one with disc 2 in it. After I was done with this it was able to install the game using both discs.

If anyone is going to do this make sure there are two seperate drives with both directories for each.

I'm marking this thread as solved.
