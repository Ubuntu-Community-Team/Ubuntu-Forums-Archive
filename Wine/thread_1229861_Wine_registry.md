### Post by Inferiority Complex on 2009-08-02
You can use the "Configure Wine" app to customise specific wine settings for specific EXE files.  There is also a global setting which is applied by default if there are no custom settings for an EXE.

For example, my global wine settings are the defaults but the MYST.EXE file (for playing Myst: Masterpiece Edition) is set for Windows 98 and also has a surrounding 800x600 desktop to stop it taking over my full screen.

Does this answer your question or are you looking for something like REGEDIT.EXE from Windows?

You could always try Registrar Lite: [http://www.resplendence.com/reglite](http://www.resplendence.com/reglite)
