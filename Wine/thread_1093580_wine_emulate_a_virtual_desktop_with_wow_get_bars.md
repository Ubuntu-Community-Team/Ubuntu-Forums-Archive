### Post by cogadh on 2009-03-11
Well, since by emulating a virtual desktop you are essentially running the game in a window, you should still see the panels, like you would with any other application. If you were trying to run the game in full screen without the panels, then you might need to turn off the "Allow the window manager to control the windows" option in winecfg.
