### Post by Paqman on 2010-06-21
Wine is a compatibility layer that allows some software written for Windows to run on Linux. Note that I said "some". You can check the [Wine App DB]("http://appdb.winehq.org/") to see how well individual titles work.

Wine can be fiddly to use. I'd recommend installing [Play on Linux](apt:playonlinux), which is front end and set of tweaks for Wine that has a good success rate. Installing Play on Linux will install Wine.
