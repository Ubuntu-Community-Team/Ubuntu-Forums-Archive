### Post by cogadh on 2010-06-22
Wine is installed, the installer you tried to run is just not marked as executable. All you need to do is right click on it, select Properties, then mark it as executable.

However, the particular game you are trying to run does have its problems running on Wine:
[http://appdb.winehq.org/objectManager.php?sClass=version&iId=2839](http://appdb.winehq.org/objectManager.php?sClass=version&iId=2839)
