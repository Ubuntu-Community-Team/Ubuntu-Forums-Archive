### Post by Frem on 2008-07-28
Yes. Running it from the menu is horribly jerky and slow, but if you use the following commands, it runs like a dream.

```
cd ~/.wine/drive_c/Program\ Files/Warcraft\ III/

wine Frozen\ Throne.exe -opengl
```
Edit: Oh, yeah. The actual game runs, but all the pre-rendered cinematics do not; there's sound but no video. You can watch them other ways, or through YouTube, though. ;-)

By the way, you should learn about the Wine AppDB. If you ask any question about Wine gaming, I guarantee a lot of the people here are learned what they know there. Plus, all the info there is archived and usually quite through for popular games. Here is the entry on [The Frozen Throne]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=3126").
