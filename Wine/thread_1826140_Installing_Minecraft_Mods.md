### Post by Kduke on 2011-08-16
I'm not familiar with the Builder mod, but I have installed various mods on it myself.  You need to locate the .minecraft folder.  Open the "home folder" then hit ctrl-h.  You should then be able to see the .minecraft folder.  Again, I'm not sure where that mod needs to be placed, but this is where I place all my mods/texture packs.

I also think some mods go directly in the .jar file.  So you may need to open that directly as well.  Hope that helps.
