### Post by efikkan on 2010-07-10
PlayOnLinux is a front end for Wine wich helps you to install games etc.

CrossOver Games is mostly based on the same codebase as Wine but with some extra features. Cedega was based on a very old version of Wine, but it very different from Wine today.

I've never tried Steam in either one of these, but generally Wine and CrossOver Games has better support for games. I would recommend trying CrossOver Games.

### Post by cogadh on 2010-07-11
POL is a nice addition to regular Wine, though not really necessary if you have a little patience and aren't afraid of using the terminal.

CrossOver Games is a great product, but it is still basically the latest version of Wine with a GUI on top of it. The plus for CXG is that when you buy it, you are directly supporting further Wine development (CodeWeavers is Wine's primary financial backer).

Cedega is a scam. They basically take functionality from Wine without contributing anything of significance back to the code base, then charge users a ridiculous amount of money for the privilege of using "their" product. About the only advantage to Cedega is that they do support more copy protection schemes than Wine does on its own.
