### Post by mastablasta on 2020-07-02
yes, but you would have to tell it to use wine syrinx.exe to open it.

might be easier to open the software and then load the file.

edit - have you looked at possible alternatives for the software?: [https://en.wikipedia.org/wiki/List_of_Bioacoustics_Software](https://en.wikipedia.org/wiki/List_of_Bioacoustics_Software)

sometimes windows based software doesn't have a good one but sometimes there are better ones out there. even for specialised use. for example a guy made fencing tournament software. something that became quite expensive for clubs was suddenly available as open source. and it was much better and easier to use than the proprietary version. so, while i don't know much about echo location and bats, it might still be a good idea to explore the software options further. 

Opensource has some really good software for science.
