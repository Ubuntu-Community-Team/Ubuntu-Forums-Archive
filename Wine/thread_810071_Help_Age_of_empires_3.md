### Post by BlackKnight7891 on 2008-05-29
Download and install wine [http://www.winehq.org/site/download](http://www.winehq.org/site/download)

cedega is designed on the work originally done by wine and wine is FREE :)

also have this command ready if you need to swap cd's during install
 wine eject %cd drive%

where %cd drive% is your cdrom's label if you try ejecting without this you will most likely run into a "cannot unmount drive busy" error.
