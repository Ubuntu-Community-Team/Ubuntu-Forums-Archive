### Post by kc1di on 2018-08-14
you may want to give playonlinux a try it's a nice front-end for wine, and allows you to install multiple version of wine. 
It works well for me.
you can install it from a terminal with this command ```
sudo apt install playonlinux
```
You can install it alongside the wine you already have and it will not affect steam, playonlinux install programs in it's own virtual files. 

Good luck.
