### Post by lite1979 on 2012-10-20
Sounds a lot like a problem that came up a few years ago regarding gameoverlayrenderer.dll. In winecfg, go to libraries, type that in, add it, then disable it when you edit. Let us know if that works for you. If I recall correctly, the problem only affected Goldsource and Source-based games (like half-life 1, 2, CS:S, et cetera). I don't think this game falls under that category, but the symptoms are the same.
