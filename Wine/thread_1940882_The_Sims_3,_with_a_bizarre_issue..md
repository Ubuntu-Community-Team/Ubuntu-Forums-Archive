### Post by melissanikol on 2012-03-23
I had the same issue and drove myself crazy trying to fix it.  Turns out all I had to do was install **driconf**, open, select the 'Image Quality' tab, and then set 'Enable S3TC texture compression even if software support is not available' to 'Yes'.
