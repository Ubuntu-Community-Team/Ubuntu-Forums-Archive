### Post by gupti on 2010-10-29
So far, I'm not sure if itunes 10.1 runs on ubuntu 10.10. Not all .exe programs run with wine. 
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=1347](http://appdb.winehq.org/objectManager.php?sClass=application&iId=1347)
The rating system in wine is:

Platinum: Runs like on any Windows machine.
Gold: Runs like on any Windows machine with a little tweaking.
Silver: Runs on ubuntu but with minor bugs that don't affect overall performance.
Bronze: I believe it has a few major bugs.
Garbage: Won't run at all.

The steps to install or run Windows programs (on my own experience) are the following:

1. Install Wine.

2. Download the program.

3. Run the .exe setup. If it gives you an error message something like "This program is not marked as executable.", right click the .exe>Properties>Permissions and then check the box "Allow executing file as program".

4. In setup, I usually place the program files in a convenient folder where I run the executable program (again, marked as "Allow executing file as program"). 

I hope this helps!:)
