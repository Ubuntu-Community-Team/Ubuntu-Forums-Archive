### Post by cogadh on 2008-05-03
Delete your .wine directory (hidden directory in your home directory), update to a more recent version of Wine (use the instructions [here](http://www.winehq.org/site/download-deb), just replace all the "hardy" references with "gutsy"), then recreate the .wine directory and try running the installs again.
