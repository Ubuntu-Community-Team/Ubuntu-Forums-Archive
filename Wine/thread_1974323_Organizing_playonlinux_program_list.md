### Post by forrestcupp on 2012-05-08
The only way I've found is to change the name. Click on one of the programs, then click the Configure button. Then in the General tab, you can change the name listed. I installed Microsoft Office 2007 in PoL. "Microsoft Word 2007" is the last Office app listed, alphabetically. When I changed the name to "1 Microsoft Word 2007", it moved it to the top of the list. So you could probably figure out how to organize things using a numbering system like that.

It may not be the perfect solution, but it works. Just be aware that if you have manually created any desktop launchers, doing this may cause them to not work until you change the name in that, too.
