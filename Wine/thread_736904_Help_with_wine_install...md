### Post by Sef on 2008-03-24
> i downloaded "wine" from: packages.ubuntu.com, but do we have to download all the dependencies seperately?? or does the final package includes all of them??

Why did you download from packages.ubuntu.com?   Easier would be to use synaptic.  System > Administration > Synaptic Package Manager

If you download from Synaptic, then all the dependencies will be automatically downloaded.  If you download from packages.ubuntu.com, then you will have to make sure that the dependencies are downloaded and installed.
