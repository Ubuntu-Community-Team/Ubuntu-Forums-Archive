### Post by ajgreeny on 2008-07-11
It functions exactly the same on my machine.  Don't forget the ~/.wine folder is hidden by default, like all those with names starting with a . (period, full stop, whatever you call it), and you will not be able to see it unless you set nautilus to show hidden files and folders, either with Ctrl+H or using the preferences and setting to show them by default.
