### Post by soekarmana on 2011-09-14
try to copy the cd contents to your home folder then right click on the .exe file (usually setup.exe / install.exe) -> Properties -> go to Permission tab -> tick "Allow Executing file as program"
now you can double click the .exe file to install it

hope this helps
