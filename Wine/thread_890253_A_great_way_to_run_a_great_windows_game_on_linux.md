### Post by speedy55555 on 2008-08-14
This is how to run a game called Blockland(it only runs on windows frown:: and mac):
You download wine at [http://www.wine.org/]("http://www.winehq.org/"), get it working. Then you download Blocklandv9 at [http://www.blockland.us/index.asp]("http://www.blockland.us/index.asp") for windows. Next, you
type *wine <the location of the exe file>*. It should open a window if wine is working. Once that's done you type *wine <the location of the game exe file>*. That should start blockland.
(note: I recommend that you don't activate full screen mode because it doesn't run so I had to reinstall the game to fix it)
