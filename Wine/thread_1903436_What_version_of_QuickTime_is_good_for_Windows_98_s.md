### Post by Riothamus on 2012-01-02
I've tried a few different versions, including QuickTime 3.0 and 5.0, and everything seems to crash or cause the screen to go black. When I Googled this, it seems like everyone is using QuickTime 7, but that requires at least Windows 2000. I specifically need to simulate Windows 98. Since this is one of Wine's features, I would think there would be some sort of QuickTime compatibility list, but I can't find any. Does anyone know?

---

### Post by Riothamus on 2012-01-03
And I solved the problem.

In case anyone else has this and finds this thread on Google:

The wrong version of Wine was installed. I installed the one that says Wine Program Loader, and not Wine Compatibility Layer. With the Compatibility Layer one installed, QuickTime 3.0 works fine.
