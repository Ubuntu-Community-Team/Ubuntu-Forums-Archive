### Post by kc1di on 2017-06-15
You may also want to install playonlinux which is a GUI frontend for wine.  It allows you to install different versions of wine and also install each program in a separate virtual file.
allowing you to use different wine versions with each program.  It works quite well. 
Remember not all windows programs will run in wine.  You can sort of find out what works and what doesn't at [wine HQ ]("https://appdb.winehq.org")
you can install playonlinux in a terminal with this command:
```
sudo apt install playonliux
```

You can find out more about Playonlinux [here ]("https://help.ubuntu.com/community/PlayOnLinux")
good luck and happy learning ubuntu :)
