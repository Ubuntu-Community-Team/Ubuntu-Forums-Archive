### Post by happyhamster on 2008-05-24
> **hazman said:**
> How do you configure wine i want to use it to flash my mobile phone if i can
Perhaps you can, but likely you shouldn't. Only if you're 100% sure that you can recover from a firmware-upgrade-gone-horribly-wrong, should you give it a try. Testing wine is all about trial-and-error, and that's usually no big deal, because you can just re-install wine and try again. But in this case, an error could cause real damage.
