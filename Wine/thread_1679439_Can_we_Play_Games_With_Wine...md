### Post by cogadh on 2011-02-01
There's really nothing specific to tell; install and configure Wine, then run the game installer just like you would on Windows. In the most basic sense, that's all you really need to do. Certain games may have specific additional requirements and sometimes you may need to use the terminal to run executables, but those specifics can usually be found on the individual game entries in [Wine's Application Database]("http://appdb.winehq.org/").
