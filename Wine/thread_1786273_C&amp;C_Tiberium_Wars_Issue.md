### Post by desktorp on 2011-06-19
Have you tried installing the program without Playonlinux?  Ultimately, with no error messages and no terminal output, it will be difficult or impossible to diagnose the problem.  The folks at Playonlinux's forums may be able to help you out.  If you want to try regular ol' Wine - first just try installing it as you normally would and if you encounter the same stalling issue, try running the installer from the terminal, so that you can get some error messages.  Put generically:
```
cd [path]
wine [filename] 
```
