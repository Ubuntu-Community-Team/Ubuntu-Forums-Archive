### Post by MattThLinuxNewb on 2009-02-04
I've recently switched to KDE and noticed that wine applications - Warcraft 3 in my case - can't use fullscreen when emulating a virtual desktop.

I can disable the virtual desktop to run it in fullscreen but then switching to another window ruins the graphics. (On Gnome I could alt-tab between the fullscreen war3 in the virtual desktop and other applications while keeping the graphics intact).

Running warcraft in windowed mode is not an option as the mouse can leave the window, making in-game scrolling very difficult.

Do any Kubuntu/KDE users know of a workaround/fix for this? Starting a new Gnome session just to run warcraft isn't much of a solution, especially since one of the reasons I switched to KDE was for increased performance in games :).

Cheers, Matt

### Post by MattThLinuxNewb on 2009-02-04
Ok I've worked it out. Right click on the title bar (or press Alt+F3) for the virtual desktop window and choose Advanced > Fullscreen.
Duh. ](*,)

There's also Advanced > Special Window/Application Settings which has a lot of other window options.

The only problem is the mouse pointer doesn't seem to go right to the edges of the screen. I haven't tried playing yet to see if it affects gameplay.
