### Post by syltty on 2007-12-12
Had the same problem. My problem was that I was missing loopback device from my network config. 

Try if you can ping 127.0.0.1

If you don't get reply, you probably have the same problem.

Solution:

Add these lines to /etc/network/interfaces
# The loopback network interface
auto lo
iface lo inet loopback

then
/etc/init.d/networking restart
