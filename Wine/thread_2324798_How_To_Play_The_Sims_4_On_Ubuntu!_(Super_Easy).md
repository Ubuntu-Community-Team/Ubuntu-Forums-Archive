### Post by brian62 on 2016-05-16
Believe it or not, the Sims 4 runs pretty darn well on Ubuntu (via Wine). The biggest problem is installing the darn game. Origin has been a huge pain to install and run on Linux since it was first supported by Wine.

I decided to give CrossOver a try last week and found out that not only do they have an app entry for Origin, the whole install is automated and works right out of the box!

By the way, CodeWeavers, the developers of CrossOver, are the largest contributors to the Wine project so if you use CrossOver, you are directly supporting the Wine project as a whole.

Anyway, I made an instruction video on installing the Sims 4 using CrossOver. If you don't want to use CrossOver, PlayOnLinux or Wine + WineTricks would probably work fine too.

[https://www.youtube.com/watch?v=_9qvwf0w8p0](https://www.youtube.com/watch?v=_9qvwf0w8p0)
