It sounds pretty straightforward to me. Just do exactly what each step tells you to do. Don't panic!
1. Get wine if you don't have it. Use the repos (Synaptic, etc) or

>>sudo apt-get install wine

Then the instructions will, perhaps, make more sense.

wine is a program which allows many (but not all) Windows programs to run.

1. Put the install CD in the drive and open a terminal, then open another terminal. Type:

cd /media/cdrom0

2. type wine setup.exe and follow the original instructions. Use the second terminal to type

wine eject

Then, if you want, get the crack mentioned, place it in ~/.wine/drive_c/Program Files/Rockstar Games/Grand Theft Auto Vice City and you should be set.

Good luck.
