### Post by itendo on 2009-08-31
In light of you being new to ubuntu, you may not be familiar with vm wares. There are a handful of virtual machine software choices that will allow you to create a virtual windows (or mac machine) if you have the software for the OS (ie, a reinstallation disk).

My old computer has a faulty component that MS stopped recognizing, but ubuntu was happy with so i re-formatted as an entire ubuntu machine. despite being a 1.7ghz pentium with only 3/4 gig ram, and a 128 meg ati radeon from 5 years ago, PS CS3 runs sprightly enough in a slimmed down version of windows xp.

recommendation bullet points:
[LIST]
[*]install virtual box (or another vm solution)
[*]use win xp for the virtual machine
[*]install bare minimum of components in windows and vb to make it useable
[*]install photoshop cs 3 (inkscape doesnt get 1/10 the harsh gIMP does, so i'd run inkscape native)
[/LIST]

also: theres a tool called nLite that you can use to make the xp more anemic ([go to http://www.nliteos.com/nlite.html]("http://www.nliteos.com/nlite.html"))
