### Post by marl30 on 2011-02-01
You need Wine. However, only attempt to install exe files in cases where no native Linux software is available. Exe was made to run on Windows, for this reason not all Windows programs will work properly using Wine.

It depends on what the .exe file is supposed to do.
Wine should work in most cases.
