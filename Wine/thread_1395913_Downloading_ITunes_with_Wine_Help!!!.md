### Post by jgoguen on 2010-02-01
> **asbjrp07 said:**
> I have linux and I need help with how to download Itunes then using wine. Can someone please give me a step by step on how to do it? An when you say terminal, what does that mean? thanks

The "terminal" is where you would go to enter commands.  It's accessible in Ubuntu by going to the Applications menu, then the Accessories menu, and clicking the Terminal item.  However, I'll save you some time, iTunes doesn't work in Wine currently.  If you absolutely need iTunes, you'll need to find a Windows installation.  If all you need iTunes for is to sync your iPod though, you can do that quite nicely in Ubuntu using Rhythmbox, or using the gtkpod application.  Rhythmbox is installed by default, it's the default music player, and you can install gtkpod by going to Synaptic (System -> Administration -> Synaptic Package Manager) and installing the "gtkpod" package.
