### Post by dino99 on 2011-07-14
D2GS is a Windows Only Binary and is not part of the PvPGN Project

[http://pvpgn.wikia.com/wiki/D2GS_Server](http://pvpgn.wikia.com/wiki/D2GS_Server)

on the other hand, you should install and use the latest wine:
[https://launchpad.net/~ubuntu-wine/+archive/ppa](https://launchpad.net/~ubuntu-wine/+archive/ppa)

the only error is:  err:ntdll:RtlpWaitForCriticalSection 
all the fixme are not errors
