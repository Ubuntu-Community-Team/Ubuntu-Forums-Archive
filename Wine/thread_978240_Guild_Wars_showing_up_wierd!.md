### Post by aceplayer97 on 2009-01-18
I found the solution (at least for me)! Go to [http://tombuntu.com/index.php/2007/04/01/guild-wars-on-ubuntu-with-wine/]("http://tombuntu.com/index.php/2007/04/01/guild-wars-on-ubuntu-with-wine/")and there you will find a great guide for setting everything up properly. What I only had to do was first open up Terminal, then I typed in winecfg, when it loaded up I clicked the Applications tab, then at the bottom there is a drop-down menu which lets you pick a Windows version. I picked Windows Version 98 and now Guild Wars works perfectly! I think picking Windows 98 version is recommended by most people. I hope this helps!:)
