### Post by samlf3rd on 2008-04-12
Hello I have found the Pokerstars solution for Linux users. If you want to play for real money you have to download the POKERSTARS.COM version not ".NET" version. Recently I put Ubuntu Gutsy Gibbon on my computer, and let me tell you it was rough for the first week. I had no clue what "Wine" was and what it does, Terminal? It looks like something off a movie, and everything you have to type in order to get something accomplished is jibberish. Never fear windows users, if want to run a program that you enjoyed (or need) that you were running on Windows simply (if you haven't already) download Wine. Any program your looking for check to see if a Linux version is available, if not, download the program and when prompted where to save make sure it says Wine. 
  On my Pokerstars mission I've been trying for weeks to configure a port or mess with network connectivity. If you are on Pokerstars click on LOBBY and then LOGIN now click on "HELP" and click NETWORK STATUS,,,,,,,,,,,,,,,,,,
    all connections are bad right? That's because your running the .net version so delete it (if it will, mine wont uninstall?) and now download the .com version. You'll see all the buttons that were missing and you can play for real money now!
 I thought the problem was in my computer and I still had to set something up on my new computer nope it all configures itself. I think you will notice the more you get acquainted with Linux the happier you get.
                         Samlf3rd
