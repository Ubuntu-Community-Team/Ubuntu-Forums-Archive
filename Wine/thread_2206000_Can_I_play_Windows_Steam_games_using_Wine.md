### Post by kostkon on 2014-02-17
It does support Wine. 

Make sure that you have the[ latest version of Wine]("http://www.winehq.org/download/ubuntu"). Also, make sure you install Steam using winetricks (search for "winetricks" in the dash).

It seems that 8BitMMO is a Java game so you might need to install the windows version of Java in wine (check if it is available to install in winetricks first).
