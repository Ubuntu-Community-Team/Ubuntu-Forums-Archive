### Post by TheFu on 2013-12-21
Not all Windows programs work under WINE.

The best place to check whether a specific program works at all is the winehq.com appdb website. [http://appdb.winehq.org/](http://appdb.winehq.org/)
Often, very specific setup is needed for any specific app using something called "winetricks"  These can be really simple point-n-click things or highly complex lots-o-typing command things.

To get any real help, you'll have to say the program causing issues, but starting at the appdb website is the first thing.  A specific title with "Running/Installing WINE {insert-program-name-with-version-here} would likely get other users to notice too.

As an example, I used to have the best instructions to get Quicken 2010 running under WINE on the internet. It was amazing all the help from others that came in. These people were crazy testers trying all sorts of things to get it working across almost every Linux system.  Eventually, something changed and I didn't have time to get a newer release working - switched back to using virtualization. It was much easier and I'm still doing that today. 

WINE is pretty great and I wish you luck.
