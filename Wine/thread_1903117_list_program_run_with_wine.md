Don't need a thread ... the WineHQ site already has such a list -- and since the relative success of running a particular application dependends not only on the application and its specific version, but also on the Wine version, you need a BIG database to contain all the info.

If we create a thread here, not only would we be duplicating effort, but we would also NOT be getting that kind of useful info into the WineHQ site.
