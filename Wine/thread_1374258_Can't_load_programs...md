### Post by pedro_orange on 2010-01-07
You need to explain your problem a little better I'm afraid.

Wine will save your stuff in ~/<username>/.wine/ by default unless you specify another location on the application installation.
If you need to configure wine you simply run "winecfg" in the terminal. 

If you're still running Windows applications through Wine then you have Wine installed.Simple as.
