### Post by DJYoshaBYD on 2012-11-24
You can install MONO if they are pure .net or mono applications. If they are made for windows, then you can use WINE, and install MONO for Windows (use winetricks or PlayOnLinux) instead of the actual .NET 3.5 or whatever, and you should have some varying level of success. I actually find that MONO is pretty good, and starting to get fairly mature. On their website, from what I saw, it should be pretty much compatible with .net 3.5..
