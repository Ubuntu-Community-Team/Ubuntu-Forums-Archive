### Post by OMJD on 2008-11-18
I can only comment for World of Warcraft. 

Firstly, you will need Wine. This is because WoW does not have native support for Linux. This is where Wine comes in to play. 

The install process is simpler if you download the client from Blizzard's website. It's not hard to install it from DVDs, but there are extra steps which you need to make. (instructions are available - see below)

WoW used to run fine for me, but recently and since I started playing WoTLK, I've been getting around 15FPS on the lowest settings. However, I got over 70FPS on Windows Vista (max settings). This problem never used to be apparent though, and I used to be extremely happy with World of Warcraft's performance on Ubuntu/wine. - Now I'm left looking for my Windows Vista installation disk once again. 

If you want to install WoW on Ubuntu, I suggest following this simple guide, it makes the process very easy.
[https://help.ubuntu.com/community/WorldofWarcraft](https://help.ubuntu.com/community/WorldofWarcraft)

Hope that helps.

- Rich
