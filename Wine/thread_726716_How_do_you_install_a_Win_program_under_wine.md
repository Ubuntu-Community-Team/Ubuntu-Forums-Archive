### Post by uberlube on 2008-03-17
If you go to [www.winehq.org/](www.winehq.org/) there is an app database in the side panel. Look through there to find what program you are trying to install and you will find lots of helpful info on how to do it. Or it will tell you that your program isn't supported yet. Hope this helps. :)

### Post by giuspen on 2008-03-17
If u use ubuntu and u have a recent version of wine, to install windows software on ubuntu u JUST DOUBLE CLICK ON THE EXE FILE exactely how u would do on windows.
U will find the link to the application in the menu APPLICATIONS--WINE...
Hope was useful.

### Post by LoneWolfJack on 2008-03-17
Or you could just open a terminal window and enter something like... 

> wine /cdrom/autostart.exe

some say cd'ing into the directory that holds your exe will avoid directory problems you may encounter when just doubleclicking the exe, but personally I never had any problems with this
