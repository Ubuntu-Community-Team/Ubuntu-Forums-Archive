### Post by Toadinator on 2008-04-12
You know that the latest iTunes doesn't exactly install right on WINE, right? Well, if you go here,

[http://www.apple.com/support/downloads/itunes732forwindows2000.html]("http://www.apple.com/support/downloads/itunes732forwindows2000.html")

then you'll find a working Windows 2000 version. I don't know if this exactly belongs in a support forum, but it'll help people who are going to ask about installing iTunes. You must be running Ubuntu 7.10/8.04 or be using the latest version of WINE for this to work, though it may work on earlier versions...
