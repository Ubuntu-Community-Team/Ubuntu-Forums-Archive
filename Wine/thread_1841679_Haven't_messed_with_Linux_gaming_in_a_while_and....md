### Post by cerda on 2011-09-10
Wine is getting better by the day. You can install from the repositories or search for some guide to install the latest build (currently 1.3.27).

There's also Playonlinux, it's a program who manages scripts to install games through wine and keep it kinda organized. I would install it first and try your game on it, if no success go for pure wine and search the wine app database to see if the game you want runs garbage, silver, gold or platinum on the current wine release.

And any wine related questions should be asked at the wine sub-forum!! :)
