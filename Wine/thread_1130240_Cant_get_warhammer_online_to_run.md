### Post by NightMKoder on 2009-04-20
Crossover is a commercial version of wine. Wine is an open-source project and Codeweavers (makers of crossover) provide a large amount of code for wine. They are also more willing to accept patches that wine would never accept. The result is that many more apps work in Crossover. Try it if you have the money, it's actually much easier than wine (comes with a GUI).

The appdb entry for warhammer: [http://appdb.winehq.org/objectManager.php?sClass=version&iId=13139](http://appdb.winehq.org/objectManager.php?sClass=version&iId=13139) . It says to play around with registry entries/install directx. Try the registry entries first, and if that doesn't work, try the winetricks.
