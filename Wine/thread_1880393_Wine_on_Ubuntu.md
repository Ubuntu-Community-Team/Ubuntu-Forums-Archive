### Post by Mark Phelps on 2011-11-14
You're mixing two very different issues -- Wine and Wubi.

Wine is a layer of software that provides the ability to run SOME Windows apps by substituting Linux system calls for Windows system calls.

Wubi is a way of installing Ubuntu from inside MS Windows into a single file (root.disk) that exists on an MS Windows NTFS partition.

When you install using Wubi, it automounts your Window filesystem -- and does that under the /host directory.

Wine has nothing whatsoever to do with that.

So, if you have installed Ubuntu using a separate partition, you have to click on the Home icon and look through the list of filesystems.  One of them is probably your Windows filesystem.  When you double-click the proper filesystem icon, it will mount that partition -- giving you read/write access to the files there.
