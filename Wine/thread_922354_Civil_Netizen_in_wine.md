### Post by KtuluDarklord on 2008-09-17
I wanted my first post to be of use to others as reading this forum has helped me many times.

I have got Civil Netizen running in wine, its a free app for sending files directly to each other that is 128bit AES encrypted end to end with a 4gb size limit.

Just send the other person a pick up slip and they can download the file directly from your system.

I have a standard install of wine from the add/remove menu (plus updates) and installed Civil Netizen version 8 beta, using wine's version setting it runs under ME or 98 but NOT XP.

It takes a while to start about 35 seconds and the drag and drop function does not work but the blue add files or folders buttons do work as does everything else i have tried. Speeds are down on compared running native in windows but there is a native linux version on the way.

I have emailed Civil Netizen support and they have shown interest in adding this info to their website, this gives linux users an option of trying their app now and until the native linux version is complete. 

I have Ubuntu 8.04 64bit Hardy Heron installed and it is running fine on that. I presume 32 bit would also run fine but do not have a 32 bit install to test.

I did not put any links or screen shots as this is my first post but can provide them if needed.

To all those who have helped other to solve problems thank you as many times you have helped me to without knowing it.
