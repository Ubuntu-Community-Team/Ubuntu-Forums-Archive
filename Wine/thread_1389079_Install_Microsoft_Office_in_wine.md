### Post by beastrace91 on 2010-01-24
Try moving the setup.exe over to your Ubuntu (extX) partition and then try it. 

Also what Wine version are you using? For the best results be sure to be using one of the newer beta versions and check the [AppDB](http://appdb.winehq.org/) for more information on installing your applications.

Lastly - if you are new to Ubuntu and seriously need MS Office (and not open office) for whatever reason (I understand - I use Office 07 on my netbook for school) I'd recommend picking up [CrossOver Office](http://www.codeweavers.com/products/cxlinux/). It makes installing/using MS Office painless for the average and experienced Linux user alike.
