### Post by Alex Carroll on 2008-05-30
Create a file called "autoexec.cfg" in the "./drive_c/Program Files/Valve/Steam/SteamApps/YOUR ACCOUNT NAME/counter-strike source/cstrike/cfg/" folder. Edit it with your favourite text editor and put the console commands that you want on separate lines.
