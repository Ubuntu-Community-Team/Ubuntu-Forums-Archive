### Post by Cilph on 2010-08-15
Wine implements the Windows API and translates DirectX calls to OpenGL. It is far from 100% compatibility and usually does not work with graphically intensive or unpopular applications/games.The MMO I don't know about as there is no entry in the Wine AppDB for it, but for Bad Company 2 I can say it will not be enjoyable while DMC4 seems to have solid performance. For Monster Forest Online it just seems to be incompatible with Wine (for now).

Furthermore, please don't use Ubuntu simply because of 'Winblows'. Be aware of the incompatibility with Windows executables and popular games/applications you will not be able to run in Linux.
