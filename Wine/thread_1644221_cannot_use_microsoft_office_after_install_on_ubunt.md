### Post by marl30 on 2010-12-13
I realize that it's better to install Ms Office in PlayonLinux than in Wine. Why is that? Well, it's not that Ms Office won't install and run great under Wine, it's just that if it develops any sort of problems and needs to be reinstalled under Wine it may not run after reinstalling. But because PlayonLinux uses different environments (bottles), you can always isolate each installation, so if office needs to be reinstalled it won't be that big of an issue. I have Office 2007 running in PlayonLinux, while Office 2003 on Wine (I've gotten Access 2003 to work with Wine).
