### Post by ZugZugTheOrc on 2007-11-05
Okay, so I spent a week or so trying to figure out why in the world I could not install Oblivion. I kept getting strange errors..

Well it all boils down to this, when configuring the drives, DO NOT change the drive_c path.. leave it as ../drive_c

I kept changing it to have drive C on my larger partition, when I did that I got errors all over the place.

You could equate this to installing Windows on drive C and then changing the drive letter to something other than C.. it's obvious that Windows will crash.. so will wine.

If you want to use a larger partition as a place to install files, add a new drive and point it to the larger partition. Dont pull out hair like I did.
