### Post by tm-hart on 2015-06-04
I wanted to install and run EZNEC ver. 5.0 or 6.0 (antenna modelling software for WINDOWS) on a 64-bit Ubuntu ver. 15.04 computer.  
After some trial and error, I found that WINE version 1.7 allowed installation and operation of both.  For some reason, an earlier test 
using WINE ver. 1.6 was unsuccessful.  Both initial installation and subsequent upgrade from EZNEC ver. 5.0 to 6.0 were successful with 
WINE ver. 1.7.  Two computers were tested; both loaded and ran the software.
