### Post by galgalesh on 2013-05-22
You can play a lot of high end games in Linux with Wine. The easiest way to do that is to use "playonlinux". You can see the supported apps here: [http://www.playonlinux.com/en/supported_apps-1-0.html](http://www.playonlinux.com/en/supported_apps-1-0.html)

Some games need you to configure some things in order to get it to work completely. Playonlinux does that for you.
