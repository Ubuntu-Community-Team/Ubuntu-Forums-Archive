### Post by cogadh on 2009-05-02
AFAIK, the only way that could be there is if you added it yourself. By default, it only has the Canonical partner archives and if you added the WineHQ repository correctly, then it should only show "WineHQ - Ubuntu 9.04 Jaunty Jackalope". As long as you aren't compiling Wine from source, it really shouldn't matter anyway and you can actually remove that source repository.
