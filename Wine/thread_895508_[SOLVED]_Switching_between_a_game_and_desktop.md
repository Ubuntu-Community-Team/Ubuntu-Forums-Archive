### Post by k1ngb0b0 on 2008-08-20

You can try what I do.

Make sure to NOT run WoW in window mode.  When you load up the game go into video options and then: check vertical sync, check windowed mode, then check maximize and also check disable resize.

Tell me if it works.

Edit: 

I usually move to another workspace to open up thottbot or wowhead.  But if you're still trying to alt + tab then open up firefox BEFORE you start WoW.  Then you'll be able to move to the net to get quest help etc.
