### Post by The Cog on 2016-07-27
I'm pretty sure you just look for the file in your home folder in a hidden folder called .wine: ~/.wine/drive_c
Alternatively, wine by default maps drive Z: to the linux filesystem root so you can move the file from within a windows program if you want.
