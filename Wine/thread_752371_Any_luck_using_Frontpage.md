### Post by michaeljking on 2008-04-11
When I changed over myself I still used frontpage for a while, but I then found a way to make it work for me. 
  I use free open source css templates which I modify slightly using Bluefish editor, (and Kompozer), and I add upload the webpage and modify/update things in Kompozer then I drag and paste the revised files into the folders online using ftp in Konqueror web browser.

My website is [www.michaeljking.com]("http://www.michaeljking.com"),  If you want you can use the internet wayback machine to see what it looked like under Frontpage and how it improved under linux and Kompozer.   

 I suppose for me I wanted a Linux solution for my business website so I made it work for me.
