### Post by barkerson on 2011-03-07
It should work if you install playonlinux(it's in ubuntu software center). Install it, then go to application, games, then playonlinux. Update it, when it's done, press the install button(beside the remove button), then go to games and search for steam.
After you have installed steam click two times on the text where it says Steam, update and your ready to go!(and hopfully play:D)
