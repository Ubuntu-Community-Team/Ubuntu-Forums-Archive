After spending a few hours yesterday trying to install some software from a Windows CD-rom and getting the "Access Denied" error, I figured out a solution:

1)  Insert CD-Rom and open it using "Brasero".   Select Copy to ISO file and save it to your folder.

2)  Right click on the ISO file and extract the contents to another folder.

3)  Open the folder and run the Autorun or Installer!
