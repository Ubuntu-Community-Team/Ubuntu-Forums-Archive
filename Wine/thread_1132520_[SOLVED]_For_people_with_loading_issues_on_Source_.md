### Post by Noise... on 2009-04-21
Ok, after messing around with these for over a week now, and getting NOWHERE with these games in Wine.

Tonight, however, I just tested Half-Life 2 with the launch option of;

```
-dxlevel 70
```

This let it load up perfectly, albeit in very low resolution. A couple quick clicks and it was set back up to my full 1280x800. 

This is with an Intel GMA 965/x3100, which many seem to have had trouble with.

Default/recommended settings were set to medium across the board. I have yet to test them at higher settings, but at the medium settings it ran nicely. Certainly nice enough to fully enjoy the game. 

I figured I'd post this, as I've seen others with the same issue, and hopefully this will help some users here. :D

I'll post results for HL2: Deathmatch and Garry's Mod in a short while.
