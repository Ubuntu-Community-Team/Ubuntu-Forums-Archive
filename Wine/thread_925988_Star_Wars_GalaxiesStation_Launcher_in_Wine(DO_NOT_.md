### Post by zami on 2008-09-22
I don't know if the cause is the same as yours (because I don't know what the cause was in my case!) but I had a similar problem with the game refusing to update.

I sorta gorilla fixed it by installing and updating SWG to a Windows partition, and then copying the directories "Sony" and "StarWarsGalaxies" from the Windows partition to their respective Ubuntu partition .wine location.

You should already have these directories in .wine from your install, they are just outdated so say "yes" to any file replacements.

If you don't have a Windows partition, maybe you could install it to a Virtual Machine environment of Windows - the game won't run in a VM (no 3d graphics in VMs), but it should install and update just fine.  Or if you have an external drive, the two directories total about 6Gigs so maybe you could copy them from a friends machine.

Good luck!
-zami
