### Post by Sugi on 2010-12-06
Let's say you could install these drivers this way, then what's the next step? Using wine to run window's version of skype and any other program to use the printer?  This sounds like a huge workaround and a bigger headache then needed.  

The best option for you, would try to get drivers/applications and/or test the default settings for your printer and webcam within ubuntu. Nine times out of ten, you will be able to find some applications to run your hardware in Linux.  The hard part is just finding it. :D  For example, My n52te gamepad doesn't include linux drivers and the available linux drivers are crap.  Thus I found pystromo application and I have been using it for +3 years now, which by the way is ten times better then the default drivers for Windows.

Hopefully this is useful to you and may provide some insight to an alternative,
Sugi
