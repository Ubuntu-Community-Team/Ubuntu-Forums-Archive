### Post by Perfect Storm on 2011-06-30
Check which stuff .Tie wants to install. then add the required stuff to steam bottle. - test the game - if it needs tweak check the games, the tweaks are usually written in the games tips and trick.

Rome: Total War: [http://www.codeweavers.com/compatibility/browse/name/?app_id=3651](http://www.codeweavers.com/compatibility/browse/name/?app_id=3651) looks like an out of the box.

Then simple check up on each games, eventually tweak it.
You can install multiply steam bottles if some game tweaks interfere with another game.
