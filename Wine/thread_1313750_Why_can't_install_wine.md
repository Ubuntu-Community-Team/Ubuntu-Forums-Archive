### Post by ad_267 on 2009-11-04
You can download the .deb files from [http://packages.ubuntu.com](http://packages.ubuntu.com). Eg, [http://packages.ubuntu.com/karmic/wine](http://packages.ubuntu.com/karmic/wine)

The problem with this is that there may be dependencies that need to be installed too, so you'd have to then go and download these too.

This might help explain some stuff: [http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)
