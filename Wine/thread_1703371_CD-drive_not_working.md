### Post by cogadh on 2011-03-09
As long as the CD drive is mounted correctly in Ubuntu and is detected/configured in Wine (you can check that in winecfg), it should "just work". You will probably need to use the terminal to run the installer; Ubuntu's security measures prevent executables on CD/DVD from running normally.
