### Post by rgb1701 on 2008-01-21
You shouldn't be using WIne for any Mame front end- there are plenty of native Linux equivalents to Mame32.

Try
[http://sourceforge.net/projects/kxmame/](http://sourceforge.net/projects/kxmame/)

[http://gxmame.sourceforge.net/](http://gxmame.sourceforge.net/)
