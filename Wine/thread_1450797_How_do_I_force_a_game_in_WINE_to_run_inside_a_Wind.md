I have installed Half Life in WINE via PlayOnLinux. How do I force it to not run in full screen? For example I'd like to run it in a window of 800x600 or something similar like 832x632.

---

### Post by hikaricore on 2010-04-09
> wine explorer /desktop=800x600 program.exe

Be aware that the program may affect the window size on its own, but it will run in a window.
Turn off emulate virtual desktop as this may interfere with my example.
