### Post by r00st3r32 on 2016-05-28
I love Linux and especially Ubuntu. I recently upgraded to the latest long term support OS of Ubuntu and wanted to share how I installed Hearthstone and got it to run superbly on it. 


1. Install Wine 1.9.10 Staging from the wine website. A google search for "Wine 1.9.10 Staging" will get you there to add the PPA's and then install using sudo apt-get.


2. Run winecfg in a terminal screen and it will prompt you to install Mono and Gecko packages. Download them and have them install. BUT DON'T CHANGE ANYTHING. Leave everything as default.


3. Download the Hearthstone installer and go through the installation of Battle.net. The only issue I ran into was the drop down menu's which when you click on them they don't show but if you move the mouse pointer down they show up and you can select your region.


4. Once Battle.net is installed just choose Hearthstone and download it. Once it is done the play button will appear.


5. Enjoy.


Hope this helps someone wanting to play this awesome game on the newest version of Ubuntu. It runs perfectly on my 2010 laptop with Ubuntu 16.02 LTS.
