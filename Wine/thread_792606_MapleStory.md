### Post by cogadh on 2008-05-17
Odds are, Maplestory will never work in Wine, as long as it is using GameGuard. GameGuard relies on running a virtual driver in Windows to accomplish its anti-cheat functions. Since Windows drivers do not work in Wine, GameGuard, and by extension Maplestory, will never work.
