### Post by Dlambert on 2011-12-10
Check List: Install WINE, PlayOnLinux, and your graphics drivers.
 
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=12117](http://appdb.winehq.org/objectManager.php?sClass=application&iId=12117)
 
The game is rated platinum, so it defiantly works in Ubuntu. Just install wine.
 
> HOWTO(This assumes you are trying to install Civ V to clean prefix, for example with env WINEPREFIX="games/civv" wine Steam)
When mounting the DVD, and launching the Setup.exe, it was not able to launch Steam installation.
Installing the Steam with supplied installer solved the problem.
Also remember to disable in-game community from Steam.
DirectX 11 mode does not work, use DirectX 9 mode instead.

 
 
TO INSTALL WINE: [http://www.winehq.org/download/ubuntu](http://www.winehq.org/download/ubuntu)
