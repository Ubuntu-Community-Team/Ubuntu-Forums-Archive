### Post by hikaricore on 2008-01-15
The lastest version of WINE 0.9.53 has added support for the startup entries in the Run and RunOnce registry keys.
This has caused troubles for a great number of people as applications are seeming randomly starting along with WINE and programs you normally run.


You can find all the info as well as options for removing items from Run/RunOnce here: [http://www.winehq.org/?issue=339#AutoRun/RunOnce%20support](http://www.winehq.org/?issue=339#AutoRun/RunOnce%20support)
The option presented of using MLIN's StartupCPL works only if you use the stand alone executable version.


I have seen numerous complaints about this new feature so I thought I'd throw up a sticky for ya folks.  ^_^

Best of luck,

--Aaron
