### Post by Fincer on 2015-02-19
Finally some progress with Adobe Premiere Pro on Linux. CS3 version runs  with the latest Wine 1.7.36, though you can't install it without applying [a special patch]("https://bugs.winehq.org/show_bug.cgi?id=18070#c73").

Adobe Premiere Pro CS3 startup process is still bit unstable with native gdiplus library (winecfg) but once you get the program running, you're pretty much good to go with your video edits. I've just tested the program so long-term stability and usage is still an unclear issue.

For more information, see [Wine Application Database]("https://appdb.winehq.org/objectManager.php?sClass=version&iId=10023")
