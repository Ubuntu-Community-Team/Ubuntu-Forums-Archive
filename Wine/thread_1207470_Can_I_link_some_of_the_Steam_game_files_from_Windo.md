### Post by Th3_uN1Qu3 on 2009-07-08
Yes, it should work. From that point on all operations on the link will affect the original file. Thus, updates done under Wine will stay there when you reboot to Windows.

### Post by cogadh on 2009-07-08
Be very careful with doing things like that. Wine is not designed to work with an existing Windows installation and depending on the app, it can and will completely hose you Windows install. What you have done, if you did it in the way I expect you did, shouldn't be an issue, but that is not the case for everything.
