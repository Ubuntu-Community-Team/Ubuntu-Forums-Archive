### Post by The Thunder Chimp on 2010-04-22
Installs of big programs (just as small programs) work sometimes. Tests about nearly all Windows programs have been made, the results being shown on WineHQ.org.

For example, WineHQ's report on Visual Studio would be:

Version 6.0 from 1998 works best, although Version 8.0 from 2005 also works.
[http://appdb.winehq.org/objectManager.php?sClass=version&iId=4647](http://appdb.winehq.org/objectManager.php?sClass=version&iId=4647)

Encyclopedia Britannica would be:

2009 edition has been reported to work on level "Silver", which is decently good.
[http://appdb.winehq.org/objectManager.php?sClass=version&iId=18052](http://appdb.winehq.org/objectManager.php?sClass=version&iId=18052)


To search for a program's compatibility on Wine, simply make a Google Search under this format:

[Program Name] site:WineHQ.org


That's all there is to it.
