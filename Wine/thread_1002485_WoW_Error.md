### Post by Nubsauce on 2008-12-05

While I've never had that problem, you should be able to put this at the top of your Config.wtf file.

SET readTOS "1"
SET readEULA "1"

The file is located in your "World of Warcraft/WTF/" directory.

Mine is located at: /home/user/Games/World\ of\ Warcraft/WTF/Config.wtf
