### Post by JiuJitsu500 on 2013-01-01
I'll second fdrake here.

WINE is great and works like a champ, even for things like autocad and photoshop, but productivity apps like that are seriously best left to native OS's. I don't know why really because more CPU, GPU, RAM, and HDD access-intensive programs like a new-ish video game run very well on WINE... photoshop and autocad are nowhere near as intensive, but for some reason just plain work better in a native environment in my experience.
