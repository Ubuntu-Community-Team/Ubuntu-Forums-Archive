### Post by veggen on 2011-03-07
It's usually enough to just double click a Windows exe to start it in Wine, but many applications will require special configs and fixes in order to install and run. Look for every app you want to try in [Wine's AppDB](http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true) to see if it has been tested and with what results. Also, if additional steps were needed, you'll find them there too. If you find nothing, it means the app has not been tested yet.
You can also try winetricks. It's a collection of scripts that can automatically install some popular apps and libraries.
One more thing worth to note is that if something is impossible to run under Wine, you have the option of running Windows inside a virtual machine (like VirtualBox) so no need for restarting.

[EDIT]
I now see there's a huge discussion on installing Photoshop CS5 but it seems possible to get it working. [AppDB link](http://appdb.winehq.org/objectManager.php?sClass=version&iId=20158).

Sorry for the mini-rant above. I've been in many arguments over the whole Photoshop v.s. Gimp thing with people who never gave Gimp a chance and I really wish more people would.
