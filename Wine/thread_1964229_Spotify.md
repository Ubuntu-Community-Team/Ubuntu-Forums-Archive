### Post by cogadh on 2012-04-24
Try running it in a Wine virtual desktop. Some people have reported problems running Spotify the first time due to problems displaying the EULA you need to accept and the emulated virtual desktop seems to fix that. 

Alternatively, have you tried Spotify's Linux client? I don't use Spotify myself so I can't comment on it's functionality or anything, but more often than not, using a native app is better than trying push something through Wine.
[http://www.spotify.com/us/download/previews/](http://www.spotify.com/us/download/previews/)
