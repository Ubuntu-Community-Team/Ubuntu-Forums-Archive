### Post by Mark Phelps on 2013-02-08
> **matimont said:**
> Hi,

What's the best tool to run Office 2010 and Iexplorer 7?

PlayonLinux, Wine, Crossover?

Thanks!

Sorry for the obvious answer --- MS Windows!

Some components of Office 2010 work (Word, Excel) reasonably well using any of these, but .docx files aren't handled well -- and those are the default file extensions for MS Word 2010.

Other MS Office components don't work at all or very poorly.

IE7 is so old that it is basically useless; nothing newer than that will install in Linux.

For specifics, visit the WineHQ site, the application compatibility database page, and look up the Office components you want to use to see their ratings.

If you're going to need to use Office components and/or IE on a daily basis, you're in the wrong OS for doing that.
