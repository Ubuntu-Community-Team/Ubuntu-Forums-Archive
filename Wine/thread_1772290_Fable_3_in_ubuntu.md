### Post by bobwyajnr on 2011-05-31
Probably not looking at: [WineHQ: Fable 3]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=23521"). Copy protection is a long-standing issue with Wine.

POL simply uses Wine versions that are considered 'optimal' for a particular game. Unfortunately it's not officially supported by the Wine Development team - since POL uses patched versions of Wine - to workaround particular issues. So filing bugs against a particular Windows application, on WineHQ Bugzilla, isn't possible. 

NB POL will not circumvent copy protection. Generally the only way to do this is with No-CD/DVD patches. :(
