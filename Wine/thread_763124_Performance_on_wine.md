### Post by Malvolio on 2008-04-22
It can.  Wine runs like it's essentially two different programs, Wine and the program it's running.  Wine translates the Windows cues into Linux cues so that the program Wine is running can interface with the hardware of your system.  A fine example is Dungeons & Dragons Online, boy is that a load on my laptop...  web pages literally load a few seconds slower than they would normally if I'm using Firefox at the same time.  Not all programs are resource hogs, though, so it's a mixed bag.
