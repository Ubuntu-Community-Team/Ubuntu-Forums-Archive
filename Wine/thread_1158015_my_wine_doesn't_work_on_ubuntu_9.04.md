### Post by kostkon on 2009-05-14
You can install it using *Synaptic*. Open *Synaptic* (*System &#8594; Administration &#8594; Synaptic Package Manager*), search for it (use the regular search, not the quick-search) and install it.

If you want to install its latest version and plus get automatic updates every time there is new version of it, then check [these instructions here]("http://www.winehq.org/download/deb") on how to add its repository to your software sources list.

ADDED: and here's [a nice guide on how to install software in Ubuntu]("http://www.psychocats.net/ubuntu/installingsoftware"). Have a look, if you want.
