Sims 2 is a Microsoft Windows program and Ubuntu is not microsfot, so the game cannot play on Ubuntu. In order to make it play, click applications >> add/remove . Then, click the thing that says "all supported software" and change the menu to "all open source software " and then search for wine . After you find wine, check it, and click apply. After this, log out and back in. Then try the install again. Good luck!:p
