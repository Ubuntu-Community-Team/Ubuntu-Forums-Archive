### Post by tinglew on 2007-12-24
I too am having trouble getting Wine to work, and it appears as though Wine does not offer support yet for USB attached devices. I have a digital photo keyring that has the software used to upload and download pictures right on the device as an *.exe file. I installed Wine and got the program to run but it would never recognize the USB connected keyring. My macMini didn't even recognize the device period but gues what, my Windoze machine saw it, immediately ran the *.exe file and saw the device right away I was uploading and downloading pictures in less than a minute (alright, after the typical reboot, but it worked).

I guess that an Apple Expert could get it to run on an Apple machine and a Linux Expert (with a minor in Wine) could get it to run on a Linux machine, but most users don't have the patience for stuff like that, they want it to run with little or no effort, what a lazy bunch of users Windows has made us.

What would have been even better would have been the manufacturer of the Keyring making file access more like a typical USB hardrive, then I'm sure all 3 machines would have had no trouble whatsoever........

Anyway, it seems to me that all these operating systems have their pros and con's and we will never really be rid of any of them to live with just one....:(
