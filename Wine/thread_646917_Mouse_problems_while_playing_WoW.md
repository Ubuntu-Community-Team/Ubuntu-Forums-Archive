### Post by AndrewRiedi on 2007-12-22
Wine now has support for D3D hardware cursors.  Try running WoW in D3D mode and enable HW cursor support.  (A check box in the options)

If it is a grey box, either you have a Wine that is 6 months old, or you are running in OGL mode.
