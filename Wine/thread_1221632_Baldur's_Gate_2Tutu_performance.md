### Post by thezood on 2009-07-24
Hello folks!

I'm hoping there's some old-school RPG fans here. I'm having some performance issues with Baldur's Gate Tutu (Baldur's Gate with the Baldur's Gate 2 engine). It lags and stutters from time to time. It's really unpredictable as well, going from unplayable to normal speeds in seconds. One would think it happened when there's a lot going on in the area as well but nope, there doesn't seem to be any logic to it. Vanilla BG1 works fine but BG2 is also very slow.

My machine is kind of low end (900Mhz netbook with hard drive) but it should be able to run this without a problem. For instance, the Fallout games runs perfectly. I've tried turning off hardware acceleration (after which the game wouldn't even start but crash before the intro screen), different detail levels, different Wine Windows versions without result.

I've also tried "biffing" the game files which should increase performance since it packs several thousand files into one file. This, however, just doesn't seem to work with Linux or EasyTutu.

Ideas, anyone?

Joy! I fixed it! After following these instructions: [http://ubuntuforums.org/showthread.php?t=1130582](http://ubuntuforums.org/showthread.php?t=1130582) and installing Linux 2.6.31-rc4, it works perfectly. I had actually followed the instructions but just now discovered this new kernel.
