### Post by ian-weisser on 2017-01-30
Wine is indeed in the Ubuntu Software Center. The package name is 'wine1.6' (even though newer releases of Ubuntu have 1.8 under the same package name)

Plex is not in the Software Center simply because no community member has packaged it for Debian nor Ubuntu. Anyone can do so - the license is compatible. If there is a plex package in Mint, do please encourage the maintainer to push that package upstream so everyone can benefit.

We gently suggest to new Windows Power-Users that they [read the most common pitfalls encountered]("https://wiki.debian.org/DontBreakDebian") when trying Debian/Ubuntu/Mint for the first time. Saves them a lot of frustration.

>   it would be helpful for you to post a summary (thumbnail or pic-link) of your "sources" (aka repo & repo directories).
