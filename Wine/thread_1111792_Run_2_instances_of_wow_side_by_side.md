### Post by airtonix on 2009-04-01
What have you tried doing?

Each time you run WoW.exe it just spawns a new instance of the program.

You need a different account for each toon you want to login with.

(ctrl + alt + shift + left/right arrow) will move the current program to the next workspace.
(ctrl + alt + left/right arrow) moves your view from each workspace.

change your video settings in wow to operate in windowed mode(full maximised)
