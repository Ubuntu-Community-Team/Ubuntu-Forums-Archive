### Post by Bucky Ball on 2017-05-06
Welcome. Perfect World International or Perfect World? For International, [see here]("https://appdb.winehq.org/objectManager.php?sClass=version&iId=9923"). Check WineHQ for any app prior to trying to install it with WINE. Not all programs will install with WINE. Some work well, some a bit, some not at all. Don't assume WINE is a tool to allow running *_all_* Windows programs in Ubuntu. It's not. 

Good luck.
