### Post by aceplayer97 on 2009-01-18
Hi, I recently installed WINE on my computer. I checked if Guild Wars is compatible with WINE and it said it would work flawlessly. I installed Guild Wars through WINE then ran the game. When I launched the game all you can see is a deformed version of the game with horizontal bars flashing everywhere. I even installed my graphics card through the Hardware Drivers section in Ubuntu.Does anyone know what could the problem be or the correct procedure to install Guild Wars and run it? If it helps, my graphics card is a ATI Mobility Radeon X600 and in the Hardware Drivers section it's apparently called ATI/AMD proprietary FGLRX graphics driver. By the way, this game worked perfectly in Windows XP for me.

EDIT: Nevermind, I solved the issue by changing Visual Preferences to "Extra" and changing the Wine Configuration so it runs on the base of Windows 98.
