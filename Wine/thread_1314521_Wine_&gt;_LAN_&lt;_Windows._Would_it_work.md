### Post by Vakman on 2009-11-05
It should work for networking without a problem.
I go to LAN parties all of the time but I usually run Windows 7 anyway so I can't really say. 
But I would not chance it, if you can dual boot with Windows 7 or something, then I would say do this. 
I think your biggest problem is going to be speed of the games running, meaning your FPS might be low. You can check all these games on [http://appdb.winehq.org/objectManager.php?sClass=category&iId=&sAction=view&sTitle=Browse+Applications](http://appdb.winehq.org/objectManager.php?sClass=category&iId=&sAction=view&sTitle=Browse+Applications)
Just search for the game you want and you will get a rating for how well it works.
The least likely game to work is Rome: Total War since it is the newest WC3 and CSS should work without fail.
Good luck.
