### Post by cogadh on 2007-11-02
Um, how active do you think these forums are? Sometimes it can take a day or more before you get an answer, it just depends on whether or not someone who comes here has actually tried the game you are running. It also might be helpful if you would provide some detailed information, such as the steps you have taken and the actual error/problem you have encountered.

As for a step by step, its really quite simple:[LIST=1]
[*]Put the game CD in the drive
[*]Open a terminal
[*]Type "wine /media/cdrom/setup.exe" and hit enter (change the path and executable name to match the actual game you are installing)
[*]Follow the prompts just like you would in Windows
[/LIST]

Once installed, to launch the game you need to do this:
[LIST=1]
[*]Open a terminal (if you haven't already)
[*]enter "cd ~/.wine/drive_c/Program\ Files/<game install directory>" (change the path to match the game's actual install directory)
[*]Launch the game by entering "wine <game.exe>" (obviously change the name to match the actual executable for the game)
[/LIST]
Once you have done that, if you receive any errors in the terminal, post the errors here and maybe we can troubleshoot the problem further.
