### Post by JustinR on 2011-07-20
Hi,

If your using Unity (Ubuntu 11.04 +), then go to the Unity dock, then click the Applications button, and type 'Wine'. Under the 'Install' area there should be an application called 'Uninstall Wine Software' Click that.

If your using an older version of Ubuntu click Applications > Wine > Uninstall Wine Software.

Sometimes the 'Uninstall Wine Software' application doesn't always detect installed programs - if so then it can be removed manually through /home/USERNAME/.wine/drive_c/Program Files.
