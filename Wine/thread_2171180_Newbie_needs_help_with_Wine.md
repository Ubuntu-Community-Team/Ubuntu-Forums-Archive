### Post by Mark Phelps on 2013-08-29
> I am determined to make Ubuntu my only OS.

Then, starting out by installing Wine is not a good thing to do.  Wine is a hack that, at best, allows some MS apps to work OK. It is NOT a Windows Emulator that will run anything that Windows runs.

Trying to run all your old MS Windows apps in Wine is going to lead to frustration and disappointment.  Ubuntu is a free Alternative to Windows, not a free Version of Windows.

You would do better ignoring Wine and learning about Linux alternative apps.
