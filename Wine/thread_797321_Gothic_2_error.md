### Post by happyhamster on 2008-05-17
> **filip102 said:**
> Copy-protection? U mean my ORIGINAL CD is burned? :popcorn:
wine-0.9.59What I meant is that not all copy protection is fully supported yet under wine. In other words, when you try to run such a protected program, it tries to determine if there's anything wrong with your windows installation, if there are debuggers present, etc. If wine's unable to translate those requests properly, the program will think there's something wrong, and abort.

What you could test is applying a nocd crack and see if the program works. Depending on where you live that may be or not be legal though.

Also, are you using a 64 bits variant of ubuntu perhaps? (I ask because copy protection seems to be broken for some applications on 64 bits at the moment.)
