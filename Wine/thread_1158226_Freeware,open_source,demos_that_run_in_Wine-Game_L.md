### Post by Duskao on 2009-05-13
Honestly, probably the best place to look for that is at winehq.org But if I were you I would just give them a try if you know of any. Steam is a great utility to try out demo's, but you will need to download a windows version of Firefox or IE so that you can give them a try. Clicking the "demo" buttons in steam doesn't work because there is nothing to transfer the URL to natively. I found that using wine firefox and searching through the steam store works rather well. Hope thats a bit of information.
