### Post by diggedy on 2009-08-24
either download it using the add/remove function to get the latest stable version or follow [these]("http://www.winehq.org/download/deb") instructions to get the latest version. Once installed your exe files should automatically run using it as they would in windows.

check the database [here]("http://appdb.winehq.org/") to see if your game is supported, which version of wine works best and if there are known bugs and workarounds

some games will need fonts installing or other items which you need a program called [winetricks]("http://wiki.winehq.org/winetricks") for

If you find you need different versions of wine for each game it might be worth using [playonlinux]("http://www.playonlinux.com/en") which helps newer people but this can also lead to other problems so only use it if you need it really

There is also a paid for version called [Crossover]("http://www.codeweavers.com/products/") which supports the development of wine

### Post by Copernicus1234 on 2009-08-24
I would start with checking if your specific game is supposed to work with Wine before you start trying to install it. Search their database. If your game has a good rating (Silver, Gold, Platinium) its likely to work fine. :)

[http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true](http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true)

For example, Rally Championship Xtreme will not work: [http://appdb.winehq.org/objectManager.php?sClass=application&iId=3054](http://appdb.winehq.org/objectManager.php?sClass=application&iId=3054)

The others look like they will.
