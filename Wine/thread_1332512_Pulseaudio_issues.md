### Post by kostkon on 2009-11-20
Wine can work just fine with PulseAudio. If you are using 9.10, then try installing the wine1.2 package instead of the wine one. Or you could add its ppa to get the latest version.

The wine package in 9.10 installs the 1.0.x version of Wine, which in the majority of cases, has problems with PulseAudio.
