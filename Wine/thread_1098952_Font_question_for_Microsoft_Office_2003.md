### Post by the8thstar on 2009-03-17
Go to Applications > Add/Remove and install the package **Microsoft Core Fonts**. This should solve your problem.

Fonts are located in two places:

**/Home/<yourusername>/.fonts** - easy to manage

and

**/usr/share/fonts** - you'll need root access

Hope this helps.
