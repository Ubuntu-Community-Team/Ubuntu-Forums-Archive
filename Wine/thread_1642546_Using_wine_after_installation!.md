### Post by Tweak42 on 2010-12-10
Before replying try reading the sticky at the top. [http://ubuntuforums.org/showthread.php?t=885111](http://ubuntuforums.org/showthread.php?t=885111)

In most cases you will NOT be able to run windows applications installed on a existing windows partition.  You must install them using wine from the original CD, or install files.

I normally run the "setup.exe" windows installer in the gui file manager, but you can enter "wine" and a space before the setup.exe at the command line, i.e. >$ wine setup.exe
