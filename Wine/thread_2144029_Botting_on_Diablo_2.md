### Post by blind3d on 2013-05-12
If you are talking about running D2 in WINE. 
I have made custom scripts for AHK that ran perfectly fine in WINE. But I'm not sure what you could use natively. 

All I could find for ubuntu automation, were a few clicking scripts and one program that seemed really confusing.
In AHK running wine, I was able to map my MIDI controller to specific mouse coordinates, to help speed up going through presets in FLstudio, versus clicking next preset every time. 

I would set up AHK or autoit in wine, and have it use pixel detection bitmaps for enemy colors and check the minimap for location. Log out then log back in. Repeat. 

Personally I stopped playing diablo a long time ago, and it would be a fun project if you were just learning scripting, but the scope of things you can do with AHK are pretty limited
mostly to games, it's not a very good programming skill to have, study html and jquery instead IMO.

The only problem with AHK in wine is that you need to put your paths absolutley versus relativley, otherwise they will not work.
