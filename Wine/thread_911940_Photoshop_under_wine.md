### Post by Lord C on 2008-09-06
Sorry, no idea about Elements (isn't that the cut down freebie version?)

But this will allow you to install the full Photoshop CS2.
You don't need a copy of it, it will download for you. There's a 30day license, you'll need to aquire a key after that.

Install wine (*sudo apt-get install wine*)
Run winecfg
Install wine-doors (*sudo apt-get install wine-doors*)
Open WineDoors from Menu (_Applications > Wine > Wine Doors_)
Install Photoshop CS2
