### Post by TidusBlade on 2007-10-27
You can try editing the config.wtf in WTF in the WoW directory, I think it would be
 SET gxResolution "1280x1024"
OR if that dosen't work try deleting the file and editing the video settings yourself using [this](http://www.wowinterface.com/downloads/info5202-ApplyToForehead.html) addon, and it will be changeable, hope it works for you ;)
