### Post by ronnielsen1 on 2010-12-17
AA 3 is rated silver but there's some tweaks
>  **What works**

 Assuming mouse patch is used:                                   

   
[LIST]
[*]A number of training levels.
[*]Most aspects of multiplayer on non-PunkBuster servers (at least  works by filtering out PunkBuster servers beforehand, noting the IP, and  joining the server via console).
[*]"Explore" mode seems to work fine on all five maps.
[/LIST]
 
**What does not**


[LIST]
[*]PunkBuster
[*]Graphical bugs: no display of video in medical training; flickering graphics on walls; map (F4) is blank; etc.
[*]No HUD => no easy way to send messages other than "enemy spotted".
[*]Haven't got text chat to work.
[*]Securing main objectives, e.g. shutting valve on Pipeline.
[*]Healing teammates (works sometimes, but can bug out nastily...)
[*]Crashes and other bugs.
[/LIST]
 
**What was not tested**


[LIST]
[*]Voice chat.
[*]Trying different video settings (used 1440x900 with most settings on low or medium).
[/LIST]
 
**Additional Comments**

Using AA 3.0.5 with GF8600GT.

After one has figured out how to install it via Steam, the game installation seems to work fine. However, some packages (e.g. VC2008 runtime) may need to be installed manually.


[http://appdb.winehq.org/objectManager.php?sClass=version&iId=17275](http://appdb.winehq.org/objectManager.php?sClass=version&iId=17275)
