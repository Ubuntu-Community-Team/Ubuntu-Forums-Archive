### Post by zami on 2009-02-10
Weird!!

It looks like a system fonts problem... maybe intrepid removed/altered previous fonts you were using?

Did you try

Menu>System>Preferences>Appearance>Fonts
and see if all those are good?  

Even if they look like they should, I'd change and apply them all anyhow and see if it just forces something to reset.  (I know I know, I'm not very specific.)

Also try restarting the X-server after changing the fonts. 
(ctrl + alt + backspace).

Also, that looks like a system problem and not a Wine problem, so I'd try asking in General Help if you don't get if figured out.

Good luck!
