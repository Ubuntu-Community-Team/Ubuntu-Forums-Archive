### Post by cogadh on 2007-09-20
There seem to be a few performance issues related to the latest Wine update. I have some games that won't run at all now, when they ran perfect (well, nearly perfect) before the update. You can downgrade back to 0.9.44 by downloading the .deb package from here:
[http://wine.budgetdedicated.com/archive/index.html](http://wine.budgetdedicated.com/archive/index.html)

You will need to uninstall Wine first, install the 0.9.44 package, then force that version by following the instructions here:
[https://help.ubuntu.com/community/SynapticHowto#head-ec06ac55f7b20957887c4b9cfdea6efd07727415](https://help.ubuntu.com/community/SynapticHowto#head-ec06ac55f7b20957887c4b9cfdea6efd07727415)
