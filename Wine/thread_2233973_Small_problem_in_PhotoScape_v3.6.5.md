### Post by Mike_Walsh on 2014-07-11
Evening, everybody.

I'm an ex-XP user, who switched to Trusty Tahr, and by and large am absolutely delighted with it.

My passions are photography and graphic design. I can thoroughly recommend darktable, and LibreCad...awesome apps. And having used Adobe's CS2 PhotoShop for years, it's taken very little doing to get used to the GIMP; another brilliant program.

However, as you all know, occasionally you just can't find an exact equivalent in the Software Centre...so some of us turn to Wine.

I used to use a program in Windows called PhotoScape; a lightweight photo editor, with which you can perform some amazing tricks. I browsed the Wine AppDB, and determined that the default version of Wine in the Software Centre wouldn't run PhotoScape without quite a few bugs; so I located the version I needed (Wine 1.7.21) which runs it as 'Platinum' (everything works) and installed it via Synaptic. I then ran the .exe file I have for PhotoScape on my external hard drive ( a hangover from my XP days, in case I needed to re-install XP...which I did; MANY times..!), and launched it.

It works perfectly, and I've used it to create some rather neat desktop backgrounds, which I used to use it for in XP, too.

However, I've got one small 'bug' (?) When ever I use the 'draw a straight line' function, it always disappears when I save the file I've created. Never anything else; everything else is saved, except for this one function. Curious...!

I just wondered whether anybody else had used PhotoScape (it's V3.6.5), and come across this bug? And if so, do you know of a 'fix' for it?

I can certainly live without it; it's nothing to get upset about, but fixing it would just be the icing on the cake, as this is the ONLY WINE application I can see me wanting to use, given my particular hobby.

Regards, 

Mike.
