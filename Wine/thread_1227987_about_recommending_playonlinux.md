### Post by marieeeuh on 2009-07-31
PlayOnLinux's a front end to Wine.
written in Python, it works with bash scripts that install applications (mostly games though) the best way we know. that is that if we know that such game works better with such version of wine, the script will install this version :) 
PoL is meant to be helpful to people who have trouble configuring Wine... we try to make the installation process easier... :p
