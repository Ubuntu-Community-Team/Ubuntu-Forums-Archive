### Post by itix on 2008-12-29
I don't quite see what your issue is.

Wine works like a kind of communicator between a space that pretends to be windows and your x-server (the graphical environment on your computer).

All you need to do to use a windows executable (.exe) in wine is to download it and double click it.

I think that the back up folder that you are talking about (that I suspect is from a windows partition originally) won't work in wine. I don't know the structure of WoW, so I don't know how it works but I suspect that it is easier to just install it anew and reconfigure it on place than fiddling around with file copying backwards and forwards.

Just donwload the installer-executable and double click it (assuming that you have wine installed on your ubuntu computer/partion) and it should install ok.
