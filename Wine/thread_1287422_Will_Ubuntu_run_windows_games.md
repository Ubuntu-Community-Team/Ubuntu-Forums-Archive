### Post by myromance123 on 2009-10-10
Yes Ubuntu can run Windows Games but not all, through a software called Wine.

If you're really new at this, then first you should go [here]("http://www.winehq.org/download/deb") and follow the instructions on how to set up Wine (for the latest development releases) or if you don't want to try the latest and rather the stable one then open up Synaptic Package Manager from System->Administration and type wine to check if it is already installed ( version 1.0 should already be installed), if it is not then tick it and click apply and input your password, after its done close it.

 Now you could go ahead and install the Game you want to play right off the bat BUT might I suggest using [PlayOnLinux]("http://www.playonlinux.com/en/") ? Normally if you install a game straight through Wine, there may be missing software pieces and its up to you to find it.
 Here is where (if the game is listed) PlayOnLinux will help you out, it will find those missing items for you and help make installing the game the same just like when you install it in Windows.

Currently PlayOnLinux can help you install COD 2 and 4, red alert 3, NFS 2,3, Most Wanted, UnderCover, underground and high stakes road challenge although I can't find Fifa mmm. :D
 Hope this helps, ask if you're not sure ;)

P.S: sometimes games don't work, not because of Wine but because you don't have any drivers installed for your graphics card!
