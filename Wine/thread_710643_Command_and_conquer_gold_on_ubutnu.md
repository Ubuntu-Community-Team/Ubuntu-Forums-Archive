### Post by Thelasko on 2008-11-17
> **ryu kun said:**
> I believe it isn't possible as C&C Gold edition and Red Alert are Win games.

Red Alert is not a Windows game!  I was doing some hacking (if you can call it that) and I discovered that Red Alert is actually a DOS game.  Westwood ported the game to Windows by including a fancy installer that installed DOSBox onto the system.  You can install DOSBox in Ubuntu and install Red Alert in it and it will run better than the original.  It's fantastic!

I must recommend against running Red Alert under Wine as you will be essentially running DOSbox under Wine on Ubuntu. I suspect the rerelease of Red Alert simply updates the DOSBox version that's included.

I don't know if this applies to other members of the C&C franchise.
