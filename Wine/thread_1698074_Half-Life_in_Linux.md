### Post by GWBouge on 2011-03-01
I've run Half-Life, Half-Life 2, and a couple other Source titles via Steam through WINE with very minimal effort.  About the only thing I really did was add the repo for the latest wine, and installed wine1.3

[http://www.winehq.org/download/deb]("http://www.winehq.org/download/deb")

Download and install the Steam client, and get the games through there (I also opted to turn off some stuff, such as friend notifications).  Steam itself seems a little twitchy at times, but the games run flawlessly for me.  The only thing I can think of off-hand that might give you problems would be issues with a graphics driver, such as an ATI issue or older nVidia, but I don't claim to be an expert  XoD
