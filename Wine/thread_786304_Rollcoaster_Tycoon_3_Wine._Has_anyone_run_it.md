### Post by cogadh on 2008-05-08
Rollercoaster Tycoon 3 (any version) does not really work in Wine, regardless of the Linux distribution you run it on. It has been tested on multiple distros (including Ubuntu) with multiple Wine versions and nearly all of them produced "Garbage" results:
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=2600](http://appdb.winehq.org/objectManager.php?sClass=application&iId=2600)

That one Mandriva success is very interesting, considering the poster gave it a "Platinum" rating (a bit high, considering the extra stuff he had to do to get it work). He did provide a decent list of the steps he had to take to get it functioning, you could try following his process to see if you can get similar results. The fact that he was using Mandriva doesn't really change anything that you might need to do. Generally speaking, Linux is Linux. The only real differences in most distros are looks, philosphy and package management.
