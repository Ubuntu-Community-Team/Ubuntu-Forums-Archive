### Post by Cup of Squirrels on 2008-06-23
Are you specifically using wine to run Ragnarok?

If not, try using wine to run it instead of opening the file from your file browser. Either open up your startmenu, and go to wine --> applications --> Ragnarok (Or whatever folder it is in there), or open up your terminal and type

```
wine Ragnarok.exe
```

Ragnarok.exe being the location and name of your Ragnarok.exe file.
