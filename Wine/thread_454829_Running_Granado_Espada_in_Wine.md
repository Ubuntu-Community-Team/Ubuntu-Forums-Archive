### Post by Cloud Xtrife on 2009-06-01
Im aware that this thread is now about 2 years old but its one of the first answers i get when i google "granado espada wine" so I'll just post here anyway.

I got the flash player error solved simply by downloading the windows version of it from the adobe page and installing it. Then the updater would fire up and say "Complete" but the start/config/exit buttons (and the rest of the buttons on the screen) would do nothing. So eventually i found a way to try to start the game. Just click on "Start automatically after patch", you just have to click on it while the patchers still initialising. You wont see the check mark until its done loading/patching, but it'll be there. 

Then the game will try to load and you'll get a white screen and the xtrap little window which updates ok. BUT, at the end, you get the following error : 

Themida
"Sorry, this application cannot run under a Virtual Machine"

"
