### Post by cogadh on 2010-06-11
You should be aware that there will often be some performance loss when running Windows games via Wine and there really isn't much you can do about it. 

What window environment you installed the game in doesn't actually matter to Wine; if the game ran in Gnome, it will (should) run in any other environment you choose to use. If it is not, then it might help us if you provided some more details, like hardware specs, Wine version and Wine's terminal output.
