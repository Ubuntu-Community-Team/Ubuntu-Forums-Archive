well, the simplest way to run widows applications on ubuntu is to use WINE . you can download that through the synaptic package manager, the software center, or via terminal using:
```
sudo apt-get install wine
```

at the wine site they have a complete DB of software that works or does not work. prince of Persia is there as Silver - meaning it installs and works but with certain glitches. you can look at it here:
[http://appdb.winehq.org/objectManager.php?sClass=version&iId=5612](http://appdb.winehq.org/objectManager.php?sClass=version&iId=5612)
