### Post by beastrace91 on 2010-02-08
First off - upgrade your Wine version. The latest beta is 1.1.38 as of posting this. Most Source games run better under the beta version than the "stable"

Second - are you installing the source mods in the same bottle/cellar/whatPOLCalls their wine prefixes as Steam is installed in? If so restarting Steam (as it does on Windows) should cause any mods installed to appear in your Steam games list. I've done this with both vanilla wine and crossover and it works fine in both cases.
