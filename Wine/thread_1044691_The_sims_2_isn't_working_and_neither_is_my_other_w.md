### Post by thezood on 2009-01-19
I'm assuming you're using Wine to run Windows programs. However, Wine is far from complete. Some programs just won't run and rarely with the same performance as in Windows. You shouldn't rely on Wine *always* working. Check [http://appdb.winehq.org/](http://appdb.winehq.org/) - there's information of how well different Windows programs work with Wine. As you can see there, neither The Sims 2 or iTunes work very well. Some have had luck with iTunes but it can be quite difficult. It's probably easier to use the native Linux versions instead.

As for your encrypted iTunes files: Google is your friend. As it's potentially illegal, I'm sure the forum managers doesn't want those discussions here (correct me if I'm wrong).
