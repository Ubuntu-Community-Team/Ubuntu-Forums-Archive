IE6 comes "bundled" with Wine, and I've read of folks being successful with IE7.  But these days, sites that require IE expect to see IE9.

So, if you rely on using IE to get stuff done on a daily basis, then you need to stick with MS Windows.

Note, when Win8 comes out this Fall, it will bring IE10. So, you'll be even farther behind if you're stuck using IE7.
