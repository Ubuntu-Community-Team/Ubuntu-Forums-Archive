### Post by naz37 on 2008-03-01
To get 3D apps running in fullscreen with compiz-fusion use the [workarounds]("http://wiki.compiz-fusion.org/Plugins/Workarounds") plugin.

Open  CompizConfig settings manager enable the workarounds plugin. select the plugin and enable "Legecy Fulscreen Support" option.

Hope this helps.
