### Post by codenine75a on 2013-12-10
I do not even think a person needs wine v1.7.8.  
I installed xubuntu v13.10 64-bit
**sudo apt-get install wine**
I think it is the installer that is the problem so I ran Wow-64.exe that was in my back up folder and it built the whole game from a single file download.
**wine64 Wow-64.exe**

I will share that 64bit file because I think it is interesting to build a whole game up from one little 64bit executable.
