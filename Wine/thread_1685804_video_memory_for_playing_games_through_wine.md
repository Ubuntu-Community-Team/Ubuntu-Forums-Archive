### Post by P4man on 2011-02-11
Nothing to do with videomemory. But you may have to install a bit more than just wine, like DirectX, vbrunxxx and whatever other libraries the game needs. Just like you need to install them on a fresh windows. Check out PlayOnLinux for a nice GUI to manage all that for your windows games (its in the software center I believe), or "winetricks" for a simple way to install such windows 'dependencies'.
