### Post by QIII on 2013-08-03
Hello!

If you go to WineHQ, there is a database that lists what software has been found to work (or not) with Wine.

I don't think any recent IE works well, but some older versions do.  The problem is that the ActiveX components may not match well.

To be perfectly honest, if this is for school you would be much better served by using Windows if that is what your courses require.  Use what works.  Be pragmatic.  Starry-eyed idealism is often not realistic.  ;)

You can dual boot or use a virtualization environment to run Windows.  Depending on the software, however, a virtual environment may not produce the best results.

Best wishes.
