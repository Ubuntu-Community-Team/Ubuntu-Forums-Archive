### Post by cogadh on 2008-11-09
SDLMame doesn't have a GUI, you run it from the terminal. You can install a separate GUI front end, like XMAMEGUI, GNOME Video Arcade, Loemu, etc., if you need GUI functionality. Most compatible front ends also have Ubuntu packages available for them, some may even be in the official repositories.
