### Post by almigi on 2008-11-29
I've managed to get Final Fantasy XI working under Wine a few days ago and now will probably wipe Windows off my computer and go full time Linux.

Piece of advice... I've noticed a slight improvement in performance in FFXI under Wine after I downloaded Winetricks and used that script to install Directx9 on my system. Also, after I used winetricks to install Directx9, I was able to view the intro movie again, something that never worked before under Wine.

Also, every guide I could find online regarding installing FFXI under Wine has steps regarding importing a dxdiagn.dll file from a working version of XP, and useing winecfg to associate that file with pol.exe (The PlayOnline Viewer).  However, I believe if you use winetricks to install DirectX9 under Wine first, then you can skip those steps.  This is because dixdiagn.dll is part of the Directx9 install and winetricks automatically sets it up so by default that file (and a dozen other .dlls) are set up so that Wine uses the Windows native version of the file.

- Alan
