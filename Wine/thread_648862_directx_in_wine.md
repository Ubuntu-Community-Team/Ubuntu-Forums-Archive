### Post by Matrix101 on 2007-12-24
To be honest your question left me confused too ;)

You simply have to download directx, extract it and then run wine DXSETUP.exe in the directory where you have extracted the directx-zip.

PS: You have to run the setup twice, else there will be some problems installing some dlls.
