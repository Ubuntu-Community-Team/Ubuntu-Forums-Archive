### Post by sohot on 2010-02-01
Is anybody else having a problem with programs that worked perfectly well on Wine on Jaunty, now not working on Wine on Karmic?

I have lots of old Windows games from Humongous Entertainment, most of which were written to run on Windows 98 or earlier (for my 3 year old son). These all ran perfectly on Wine on Jaunty, but freeze up all the time on Wine on Karmic, or will not run at all. 

I have tried them on 32 bit & 64 bit installs of Karmic, both upgraded from Jaunty & also fresh installs, on 3 different computers in the house, & the result is the same, they freeze up or will not run. Yet they all ran perfectly on all 3 computers when Jaunty was installed on the computers. I had to remove Karmic from one computer & re-install Jaunty just to get them to run properly.

On my own computer I did a fresh install of Karmic, & now Macromedia Dreamweaver 8 which ran perfectly on Wine on Jaunty, refuses to run on Wine on Karmic. However on another computer which I upgraded from Jaunty to Karmic, & which already had Dreamweaver 8 installed, Dreamweaver still runs OK.

From my experience, if you have Windows programs that you need to run I would recommend not upgrading or installing Karmic Koala, but sticking with Jaunty Jackelope.
