The default Wine package included doesn't has patches to run CnC3 properly.

[http://appdb.winehq.org/objectManager.php?sClass=version&iId=7440](http://appdb.winehq.org/objectManager.php?sClass=version&iId=7440)

The game should then work. Be sure to install the .DEB package given there.
