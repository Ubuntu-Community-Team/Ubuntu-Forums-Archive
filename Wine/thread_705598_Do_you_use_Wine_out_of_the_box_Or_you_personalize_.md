### Post by karth on 2008-02-23
I personally make one Wine directory per app installed (except when I want to install libraries as in .NET)

I almost only install games. Some of them tend to require some tweaks that would break other games that are working.
Therefore, having one Wine environment for each app allows me to apply whatever hack is required. More, I can get rid of an app by simply deleting the wine directory, without harming the other apps.
