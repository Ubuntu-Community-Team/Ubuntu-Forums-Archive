### Post by ehasbrouck on 2010-08-18
I want to print to PDF from an application running under Wine.  I have PDF printing working form Ubuntu (Lucid) applications, but not form Wine applications.  I'm not sure if this is a Wine issue or an Ubuntu issue.  But I didn't have this issue in Hardy, so I *suspect* that it may be an issue with how the default PDF printer in Lucid is configured. 

With Hardy, I manually installed a PDF printer (using "cups", I forget exactly how) in Ubuntu.  Once I did that, the PDF printer showed up in the Ubuntu printer menu *and* in the Wine printer menu.

With Lucid, PDF printing is enabled by default, and a PDF printer shows up in printer choices within each Ubuntu application. But there is no icon for the PDF printer in the System -> Administration -> Printing configuration window, and no option to select the PDF printer in the print options of any Wine application.

Have others experienced this?  Has anyone been able to configure the default Lucid PDF printer to appear in Wine print options?
