I gather that superAntiSpyware is Windows based product since you want to run it in WINE, but as far as my knowledge goes there is no point in doing this.

AntiVirus/Spyware/Malware etc. have definitions that cater for programs written for Windows based systems, programs that if were downloaded onto Linux would not be able to do anything as they would not understand the OS, this is not true for all Spyware etc but most of them.

You would probably be better off using a Linux propitiatory program for this purpose as another thing these "antiware" products do is try to detect malicious activity without the use of definitions, this feature would be useless because WINE is kind of a layer between the program and the OS.
