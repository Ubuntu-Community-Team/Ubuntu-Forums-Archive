### Post by cogadh on 2008-05-29
UFO Afterlight doesn't really work in Wine:
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=4959](http://appdb.winehq.org/objectManager.php?sClass=application&iId=4959)

However, you don't need autoplay to make a game work, that's just a "convenience" feature of Windows. All you need to do is either launch the executable on the game disk (from the terminal or file browser), use the menu shortcut that Wine creates when you install the software or launch the executable found in the game's install directory (again, from the terminal or file browser).
