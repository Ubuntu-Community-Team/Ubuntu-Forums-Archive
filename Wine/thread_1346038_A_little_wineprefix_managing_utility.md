### Post by oedipuss on 2009-12-04
Here's a little utility I made for managing wine prefixes or 'bottles', using python and quickly, under the rather unimaginative name 'gwine'.
It's not a substitute for playonlinux or anything like that, as the only thing that it does is manage prefixes and custom launchers in a GUI, and show some console output in a textbox. 
It won't automatically keep track of installed applications, only save launchers you create manually, but does provide an easy way to run wine and a number of wine utilities (winecfg, winetricks) in any prefix, from a gui.

Basically it's how _I_ would like a wine utility to be. There are better alternatives which I didn't like for one reason or the other, so I don't think it might be very useful to anyone else, but here it is, in case it is :P

I'm using it for some time now without any weird issues. If in doubt, just don't delete any wineprefix dirs from inside the utility, I think that's the only thing that could be possibly dangerous (if for instance there's a weird folder name), as it deletes an entire folder.
