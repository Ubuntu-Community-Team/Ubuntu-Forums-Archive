### Post by cogadh on 2007-09-03
> **Fir3chi3f said:**
> Or, when executing from a terminal add

```
-windowed
```

To the end of your command
Actually, that is not a Wine command line option, but is an option for certain game executables that allow a windowed argument. It won't work for everything, it just works for those games that allow that command line argument. 

Its like running Half-Life 2 with the "-dxlevel 90" argument; the variable is not a Wine variable, but is a hl2.exe variable and it won't work for other games.
