### Post by gsaggior on 2012-09-05
Using Wine on Precise 12.04 32 bit OS based PC, eMule 0.50a works splendidly. Obvious 12.04 up-to-date to the latest available.

Installing UPNP and GUPNP dependencies allows also to use the automatic eMule UPNP port mapping on a Belkin ADSL+ Wireless Router (with UPNP turned on).

I know aMule is available for Linux, however the Winzoz application has more parameters to control and handles the MP3 encoding detail on the search Tabs, last one option made me take the decision to use eMule.

Hope this helps.
