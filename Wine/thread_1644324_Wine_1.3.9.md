### Post by TenPlus1 on 2010-12-13
Go into Applications -> Accessories -> Terminal and type:

sudo add-apt-repository ppa:ubuntu-wine/ppa

...this adds the Wine repository so you can install the latest release...

Now type:

sudo apt-get update

...to update the repository list and finally:

sudo apt-get install wine1.3

...to install the latest version of wine...  Right-click on a windows .exe and select Wine Windows Program Loader to run but be warned, not all windows programs work yet, check [www.winehq.com](www.winehq.com) for information...
