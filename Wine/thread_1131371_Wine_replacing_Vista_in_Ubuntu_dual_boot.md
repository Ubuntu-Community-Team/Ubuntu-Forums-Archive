### Post by Bakon Jarser on 2009-04-20
It looks like steam runs ok in wine but as far as games you will have to look at each one individually in wine's appdb to see if it will run in wine.

[http://appdb.winehq.org/objectManager.php?sClass=version&iId=1554](http://appdb.winehq.org/objectManager.php?sClass=version&iId=1554)

ATI proprietary drivers suck in linux and the open source version doesn't have 3d acceleration.  NVIDIA is the way to go if you're going to be doing gaming in linux.  Don't delete your windows partition yet.  If you find out you don't need it then you can get rid of it later.
