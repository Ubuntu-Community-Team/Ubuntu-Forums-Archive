### Post by Anuovis on 2011-06-15
You don't need Wine to use flash, it works well enough with Firefox and other browsers. If you need a standalone application, there are those too, take a look at Ubuntu Software Center.

Wine is for Windows apps, don't bother with it unless something doesn't run natively on Linux. 
Flash usage certainly won't be better since Wine, besides running on top of Linux anyway, has its own quirks and annoyances.
