### Post by Praadur on 2008-01-03
I know this is a silly thing to ask but I'm just covering all bases...

Are you running a completely clean install?

The problem might not be wine at all, but if you're running any addons, they might be causing your problems.  I've seen badly put together addons bring the framerate of Warcraft down to single numbers in the past, and even cause the client to completely freeze up.

i admit, my perspective might be a little old as I only played up until version 1.8, and visited for 2.0, so things might be better now than they were.  Still, if you have any addons there, you might want to try cleaning out your Interface and WTF directories and playing the game clean for a while to see if it runs any better.

If you're running the game clean though, I apologise.

The only reason I mention this to be honest is because the terminal is giving no errors, and if that's the case it's more likely that the problem is within the game itself...
