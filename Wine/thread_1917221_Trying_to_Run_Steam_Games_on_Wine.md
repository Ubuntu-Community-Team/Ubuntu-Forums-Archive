### Post by Toz on 2012-01-29
You might want to give PlayOnLinux (POL) a try. Its a front-end to wine that simplifies the installation and management of games (and apps) using pre-designed scripts. There are scripts available for all 3 of those games (CD/DVD & Steam versions).

Although POL is available in the repositories, the version is out of date. To get the latest version, you should install the version from their repositories. Instructions are located in the Ubuntu section of this page: [http://www.playonlinux.com/en/download.html]("http://www.playonlinux.com/en/download.html").
