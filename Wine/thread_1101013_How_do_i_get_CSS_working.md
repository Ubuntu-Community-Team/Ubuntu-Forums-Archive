### Post by cogadh on 2009-03-19
Since you are so new to Ubuntu, perhaps you should take some time to learn the basics of Linux and Wine before you try tackling CS:S. I only say this because at this point, the steps to install and run CS:S with Steam are extremely straightforward as Wine/Linux instructions go.

Ubuntu does provide some very good beginner documentation on their site and there are numerous other sources online, like [Just Linux]("http://www.justlinux.com/nhf/"). For Wine in particular, the [WineHQ]("http://www.winehq.org/") website is the best place to start.
