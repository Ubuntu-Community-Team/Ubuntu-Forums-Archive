### Post by asdfoo on 2009-04-21
> **JakeHinojosa said:**
> since wine runs windows to open files for windows, if you download something that has a virus on it, would affect wine?

just wondering

Yes, if you run bad stuff, bad things can happen.  Wine provides no security, anything you decide to run through Wine has the same privileges as your user account.  Reading, deleting, modifying files, access to the internet, access to what's on your screen etc.

So you need to be a smart user just like you have to be on Windows, don't run anything you don't trust.

The only safe way to run suspect programs through Wine is by using Ubuntu installed a Virtual Machine like VirtualBox.
