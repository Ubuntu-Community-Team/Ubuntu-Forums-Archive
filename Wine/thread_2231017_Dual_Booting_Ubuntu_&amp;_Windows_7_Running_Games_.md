### Post by Mark Phelps on 2014-06-24
>  I would like to be able to run my games on either OS using the same install, sharing saved games & settings across both. 

This can NOT be done.  You actually have to reinstall the games using Wine (or one of its companions, PlayOnLinux and/or WineTricks) before you can run it in Linux.
