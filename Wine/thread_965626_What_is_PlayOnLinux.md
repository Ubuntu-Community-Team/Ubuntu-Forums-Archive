### Post by Bios Element on 2008-10-31
PlayOnLinux is 'basically' a Wine frontend. You select a program/game(Mostly has games) And it runs a script which installs and configures it for best performance. I personally don't use it because I like to do that kinda thing by myself but it seems to have promise.
