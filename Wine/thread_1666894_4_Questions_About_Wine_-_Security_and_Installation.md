### Post by Flood-X on 2011-01-14
Hi everyone,

I have a few questions about Wine:

1. After installing Wine, does it run as a background process? Or does it only run when a Windows program is executed?

2. Does installing Wine make it possible for Windows Trojans to be spread on one's file system?

3. Does installing Wine create any security issues with the system such a backdoors and security holes like those that exist with Windows XP?

4. If I want to run a Windows program with menus in another language like Turkish or Hindi, does Wine include the language support needed to install the Windows program, or are there additional steps involved?

---

### Post by cogadh on 2011-01-14
1. Only when you execute a program.

2. Potentially yes, but only if you don't take the time to securely configure Wine. Even if an infection did get in, odds are the worst it could do is screw up the Wine "fake Windows" directory, which is very easily fixed by simply deleting it and letting Wine create a new one.

3. No, the only (known) backdoor or security hole with Wine is the user.

4. Good question. AFAIK, the language support on Wine is not fully complete (still needs work with many of the "special character" languages like Chinese), but as long as your app, Wine and the OS support a particular language, then it should simply work.
