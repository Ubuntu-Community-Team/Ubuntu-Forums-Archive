### Post by cogadh on 2009-03-18
First install Steam. A how-to can be found here:
[http://appdb.winehq.org/objectManager.php?sClass=version&iId=1554](http://appdb.winehq.org/objectManager.php?sClass=version&iId=1554)

If you already have CS:S as part of your Steam account, then all you have to do is let Steam download and install it. If you don't already have CS:S as part of you Steam account, then all you have to do is activate it through Steam, then let Steam download and install it. There are some additional configuration steps you might need to take to get the best performance out of it, they are detailed here:
[http://appdb.winehq.org/objectManager.php?sClass=version&iId=3731](http://appdb.winehq.org/objectManager.php?sClass=version&iId=3731)
