### Post by neonprophet on 2008-02-20
I don't play TF2, but this newest update allows you to play [Perfect World]("http://www.perfectworld.com.my/") on the default settings straight from the windows directory, no copying or re-installing required.
This is a HUGE step actually, since this is the first oriental MMORPG that even runs on wine that I have found, they are historically buggy even on windows. PW runs perfect except for a small issue with the item icons color being blurred. I think this is a color depth setting on my end, I"ll update if I figure it out.
[Silkroad]("http://www.silkroadonline.net/") possibly runs also. I have to update my windows install to the new version before I can say it does for sure, the launcher runs perfect so far.
Any MMORPG's that we can get on linux will only make the OS more attractive to the windows gamers crowd.

:guitar:

I am working on [RF-Online]("http://www.rf-onlinegame.com/") as well, this is the big one since my guild is in this one hardcore. If anyone has any suggestions toss them my way but so far it doesn't even try to run, not even a error msg...

P.S If you have XGL/AIXGL installed forget about any games running in wine, *XGL breaks opengl. In order for these work work you have to uninstall *XGL, no other way around it that I've found.
