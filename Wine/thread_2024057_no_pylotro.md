### Post by sabersong on 2012-10-19
You can get Pylotro here:

```
https://sites.google.com/site/leeslinuxblog/PyLotRO-0.1.15.zip
```

or from the ppa:

```
sudo add-apt-repository ppa:ajackson-bcs/ppa
```

This information came from the application database page at Winehq, which is always the very best place to start looking for information on running an application through Wine.

```
http://appdb.winehq.org/objectManager.phpsClass=application&iId=2910
```
