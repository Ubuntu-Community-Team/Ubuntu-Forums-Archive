### Post by Muffinabus on 2008-12-10
Steam is a tad bit buggy when you're running it through Wine.  Messaging friends is useless as you've said, it doesn't display any text for whatever reason.

Also, I've found that you need to disable the in game community to have the games run properly, though I've really only tried Counter Strike Source, not 1.6.

I didn't have to transfer any files from Windows, but just a tip, you can transfer the .gcf file from the Windows drive to the same directory in your Wine C drive and Steam won't have to download the whole game.

For 1.6 it would be counter-strike.gcf located in /Program Files/Steam/steamapps
