### Post by cwwilson721 on 2011-11-08
Which of the 95 thousand "guides" did you use?

As a rule of thumb, the odds of you getting WoW to run when still d/ling it are slim to none, so just wait.

The things that will help you out are:
[LIST]
[*]Run in opengl mode (I just add '-opengl' to the end of the launch command)
[*]Turn off Desktop Effects (look for the option in the login screen)
[*]Use latest Proprietary Drivers for your vidcard/chip (Search this forum for "wow intel" for how to attempt to deal with Intel. Nvdia and ATi/AMD are much easier)
[*]Run WoW in XP mode (Use 'winecfg' to set the mode).
[*]Some have issues with sound. Try turning it off in BIOS to narrow things down.
[/LIST]

Search this forum! It has TONS of info on WoW.
