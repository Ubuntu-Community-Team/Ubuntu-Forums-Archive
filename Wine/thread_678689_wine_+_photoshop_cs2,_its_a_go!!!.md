### Post by | MM | on 2008-01-26
Personally i think this is great news!!

As of wine-0.9.54, Photoshop CS2 now runs really really well.  I have not seen any crashes yet, phone registration works.  Performance seems comparable to PS on WinXP.
But as always there are some known issues (see the wiki).
The remaining obvious glitches for me so far seem to be window manager issues.  

Wine dev's ask for people to test PS CS2 and submit bugs to the wine developers.

Here's what the wine dev had to say:
> Photoshop CS/CS2 should now work, please help us testing it.  See [http://wiki.winehq.org/AdobePhotoshop](http://wiki.winehq.org/AdobePhotoshop) for details.

So if you would like to see a really functional Photoshop CS2 via wine start finding issues.  One less reason to dual boot!
