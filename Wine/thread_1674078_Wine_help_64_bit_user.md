### Post by thenellt on 2011-01-23
I think you may have another problem here. If the driver is an exe, that means it was built for windows and will not work inside ubuntu regardless of whether or not you have wine installed. Ubuntu needs a linux driver, not a windows one, and even if you did manage to install it with wine, I highly doubt that your system would be able to use it. What you need to find is the linux driver which can usually be found by just searching google for ubuntu 10.10 (your networking hardware here).
