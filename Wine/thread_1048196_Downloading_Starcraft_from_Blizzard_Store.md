### Post by Circus-Killer on 2009-01-23
well, i dont know specifics, but you must remember that WINE cannot run everything. i always find that many applications that try go onto the net fail, especially those who use custom-made methods for communicating over the net.

a perfect example is GameGuard which is used on many online games including GunBound. i've been praying for GameGuard to eventually start working so that i could play gunbound, but unfortunately it hasn't come around despite years waiting for it.

some things might not run now, but will be able to in future versions as development gets further. some things that used to run will not run on later versions due to changes that break some old-compatibilities. and there are some apps that just will never run in WINE, not because they couldnt if they tried, but because the WINE devs believe there are more important things to focus on before focusing on small problems. also, they are probably trying to fix all generic problems first before moving onto application specific problems, which i dont think they will ever be able to do.
