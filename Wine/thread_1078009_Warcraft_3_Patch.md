### Post by balagosa on 2009-03-06
By what you are saying, I assume you have already downloaded that 1.22 patch which is also an *.exe file.

What you need to do is:
1) Copy/Cut that *.exe file and paste it in your Warcraft3 Folder
example W3 folder: /home/daniel/.wine/drive_c/Program Files/Warcraft III

2) Run the patch with wine. If you do not know how to do this, just right click the *.exe and select "Open with Wine Windows Launcher"
