### Post by ajgreeny on 2010-02-03
It may have a windows executable and similar files.

With the CD in the tray open it up with nautilus and see what is actually on it.  If there are .exe files, you may be able to either install or run them with wine, but it is difficult to know for sure without knowing more about the CD and files/programs on it.

You might, however, look to see if there is a file called **index.html** or similar which will start the whole thing running if you double click on it.
