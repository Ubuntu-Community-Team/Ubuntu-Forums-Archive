### Post by cwwilson721 on 2012-05-22
If you search this forum for "WoW+wine" there are MANY guides that, even if a bit 'dated' still work perfect.

The gist of it?
[LIST]
[*]Install "Additional Drivers'. If you have Intel Graphics, search this forum for "wine+intel", and good luck
[*]Install the latest wine from the PPA (Instructions on how are in the stickies at the top of this forum)
[*]To make it "most compatible", run wine as Windows XP (Can help w/sound)
[*]Install WoW or copy it. (Many guides in here if you need help, or if it gets "stuck")
[*]Either edit config.wtf, or append "-opengl" to end of launcher command
[/LIST]

Doesn't really get any easier, does it?
