### Post by QIII on 2018-07-24
Moved to the Wine sub-forum.

MS Office will not run natively on Linux.  It is Windows software.

Depending on the version, you may be able to run Access by using [Wine]("https://www.winehq.org/").

Not all Windows software runs equally well under Wine, so it would be best to check your Access version in the [AppDb]("https://appdb.winehq.org/objectManager.php?sClass=application&iId=12").
