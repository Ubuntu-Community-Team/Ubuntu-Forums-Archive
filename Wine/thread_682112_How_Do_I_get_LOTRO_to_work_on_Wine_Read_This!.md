### Post by lunadog on 2008-01-30
Follow the instructions here:

[http://www.lotrolinux.com/how-to.php](http://www.lotrolinux.com/how-to.php)

EXACTLY

Note you need to download the full 8GB client (not use CD), or copy the files across from a fully up to date patched version for windows.

If you update from linux from the 8GB client, you need to run the launcher with

bash lotrolauncher.script --HighResOutOfDate

otherwise just bash lotrolauncher.script

that's all.
