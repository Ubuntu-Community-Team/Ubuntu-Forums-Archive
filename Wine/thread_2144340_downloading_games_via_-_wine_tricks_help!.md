### Post by Mark Phelps on 2013-05-12
You should first check the WineHQ website for the game and version that you want to run.  It rates apps and games from GARBAGE (useless) through Platinum (everything works).  For gaming, any rating less than GOLD is going to be a waste of your time installing because key features will not work.
