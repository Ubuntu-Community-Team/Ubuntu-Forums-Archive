### Post by hikaricore on 2009-07-30
Don't get me wrong I think PlayOnLinux is great, but it's just a frontend for WINE.

Because of this, the problems that exist for the poster in WINE may very well be present/existent in POL also. This includes cases from graphic issues to crashes in the application.

For this reason, POL should not be taken as the "end all be all" solution to every single WINE problem on the forum.

Simply recommending it without bothering to address the issue of the poster is misleading and preventing the real problem from being found/rectified.

So next time you read a WINE problem thread, please stop and consider whether this is a configuration issue/an issue with multiple software installed in the same ~/.wine prefix, or whether the problem has got something more to do with WINE and the backend running the application itself.

If the problem most likely falls under the latter - which is expected with most cases - then you shouldn't really be recommending POL, as not only does it waste time, you are making yourselves and the community look bad.

Thank you.
