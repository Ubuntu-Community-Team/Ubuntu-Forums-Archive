Hi,

Yes, there are a number of sites that have Java and Flash poker games. I have played them for about a year now. There are a few problems that Linux users will run into, though.

The Java poker client used by most sites works fine for awhile, but tends to 'bog down' after an hour or so of play, especially in tournaments. The graphics will slow down, then the browser--I use Firefox--will lock up. The only solution then is to kill the browser process and restart. I [think] the issue is the Java 1.5 plugin.

Several sites, notably BravoTV, uses Flash, but they recently upgraded to Flash 8, which does not have a version for Linux.

[http://www.digg.com/linux_unix/_Flash_Player_8_for_Linux_*news_from_macromedia_adobe*]("http://www.digg.com/linux_unix/_Flash_Player_8_for_Linux_*news_from_macromedia_adobe*") has a story on a possible new Linux-compatible version.

Good Luck!

### Post by kc8hr on 2006-08-21
No problem!

Just download the installer, open a terminal window and type:

wine <name of installer><enter>

The program will install under ~/.wine/drive_c/Program Files/Party*, and you will even have an entry for it under "Wine" in your start menu.

I didn't care for the Party Poker interface, so I uninstalled it. I like Doyle's Room and PokerStars.

GL!
Tim
