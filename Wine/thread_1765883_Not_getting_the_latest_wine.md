### Post by Tweak42 on 2011-05-25
> **Gamer456 said:**
> I currently have ubuntu 10.04 and I am trying to install the latest wine which is 1.3.20 but when I installed wine it is version 1.3.19. How do I install the newer version?

According to the Ubuntu Wine Team PPA page 1.3.19 is the latest available for 10.04 (Lucid).  Unless you want to compile it yourself, you'll have to either wait till they push out 1.3.20 or upgrade your distro.

[https://launchpad.net/~ubuntu-wine/+archive/ppa?field.series_filter=lucid](https://launchpad.net/~ubuntu-wine/+archive/ppa?field.series_filter=lucid)

Is there any particular reason you need 1.3.20?  The reason I ask is a few things broke in .20 so you maybe better off staying on .19 if it works.
