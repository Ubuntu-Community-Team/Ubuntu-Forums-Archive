### Post by kc1di on 2018-11-12
Hello denizozc  and welcome to ubuntu forums,

You can install wine from the software center or terminal.  In the terminal type this ```
sudo apt install wine-stable wine32 
```

If you are unfamiliar with wine, it may be easier to install playonlinux it is a nice gui front end for wine and allows you to install many versions of wine.
```
sudo apt install playonlinux
``` after it's installed you can find it in the games section of the menu. 
you can learn more about[ playonlinux here:]("http://wiki.playonlinux.com/index.php/Home")
