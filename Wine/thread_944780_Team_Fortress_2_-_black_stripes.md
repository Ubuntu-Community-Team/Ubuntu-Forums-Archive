I installed Steam and Team Fortress 2 a few weeks ago (wine). After sorting out a few problems (Tahoma font, etc), TF2 ran fine (and even faster than on Vista). However, I started encountering problems today. There are large, horizontal stripes going across the screen (starting at the menu). The game does not freeze and I can even connect to a server and play. However, not being able to see half the screen really takes away from the game.

I have had this problem before, but a restart fixed it. Nothing seems to be working now.

### Post by Technophobia on 2008-10-11
You have to run the game in directx 8 mode. Set the launch options to "-dxlevel 81".

---

### Post by nzrock95 on 2008-10-11
Thanks for the reply.

I already have -dxlevel 81 in launch options. Like I said, it has been working for a few weeks. I'm not sure what happened today.
