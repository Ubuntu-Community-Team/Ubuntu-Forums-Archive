### Post by Tweak42 on 2015-02-28
> **mandarpowale said:**

I run both the native linux steam (install from the software center) and the windows version via wine (install using winetricks).  Check the [wine appdb]("https://appdb.winehq.org/") and you can find a steam entry with configure instructions for some games.  Generally if the game can run with wine installed the disc way, it should be able to installed and run via steam as well.
