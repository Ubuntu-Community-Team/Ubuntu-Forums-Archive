### Post by Toz on 2012-08-30
Hello Fishski and welcome to the forums.

I have moved your post to a new thread because the one you posted to was resolved.

As for your questions, have a look at the software package "Pinta" in the software centre. Its very much like paint.net. Paint.net itself won't work well in wine (see: [http://appdb.winehq.org/objectManager.php?sClass=application&iId=6551]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=6551"))

As for photoshop, have a look here ([http://appdb.winehq.org/appview.php?appId=17]("http://appdb.winehq.org/appview.php?appId=17")) for information and instructions on how to install it. You might also want to try using PlayOnLinux (look for it in the software centre) as it is a easy-to-use front end for wine that includes scripts to install Photoshop.
