### Post by kaoskorruption on 2008-12-12
I'm using the Rush Hour version. It actually works fine, I just had to configure wine. Go To Applications -> Wine -> Configure Wine -> Drives, and click "Autodetect." It should work from here. If it is really slow, go to the graphics settings in the game and change it from rendering the graphics with Hardware to Software. I think I had to change it for mine because I have integrated graphics. Hope this helps.
