### Post by Schotty on 2010-01-03
Actually, I used to play WoW (moved to DDO), and any of them should work fine.  My best results were with either Cedega or Vanilla Wine/Crossover depending on the day of the week and lunar alignment and favor with Nerull.

My recomendation:
Install wine (latest vanilla)
Install WoW thru that
Setup WoW to launch in OpenGL mode
Disable HW cursor since I think that STILL causes issues.

See if it works fine or not.
If not, check the appdb.winehq.com page on it and see if there are any new hacks that are necessary.  If it gets too complicated you can try Crossover or Cedega and go that route.  But those solutions are NOT free, although not that pricey, in this day and age I can understand being cheap.
