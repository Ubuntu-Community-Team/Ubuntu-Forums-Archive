### Post by Mark Phelps on 2014-11-20
Unfortunately, Wine is not a Windows Emulator that runs anything that works in Windows; instead, it is a "hack" that tends to run really old Windows apps the best.

I found no mention of the Library Explorer in the Wine HQ database, so there's no history of it ever being installed or tested.

From the little I read, it is used to find DLLs -- which is a waste of your time in Linux, because Linux doesn't use DLLs.
