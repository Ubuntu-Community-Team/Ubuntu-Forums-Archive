### Post by murderslastcrow on 2010-03-01
[http://wine.budgetdedicated.com/archive/index.html](http://wine.budgetdedicated.com/archive/index.html)

You are advised to install the 9.04 package. The error you have means that, for some reason, Wine requires libaudio2 as a dependency.

You can download the deb for it here. [http://packages.ubuntu.com/hardy/i386/libaudio2/download](http://packages.ubuntu.com/hardy/i386/libaudio2/download)

For 64 bit go here. [http://packages.ubuntu.com/intrepid/amd64/libaudio2/download](http://packages.ubuntu.com/intrepid/amd64/libaudio2/download)

Make sure to install that deb before the 9.04 one, and you should be fine.
