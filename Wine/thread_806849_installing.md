### Post by cogadh on 2008-05-25
If you are talking about installing Windows games, then you can use Wine, but it won't necessarily be able to run every game. Check Wine's [Application Database](http://appdb.winehq.org/) to see if the game you want to run will work with Wine and what you might need to configure to get it running properly.

Some games (very few) do have Linux compatible binaries available that will allow you to install the game natively in Linux while using the data files from a Windows installation disk. A few examples are Neverwinter Nights, Enemy Territory: Quake Wars, Unreal Tournament, Quake 4 and Doom 3.
