### Post by endotherm on 2010-11-21
I've run a few pre-NT games in wine. you sometimes have to configure the application profile for an older os, in wine settings. i'd send you a screen cap if i currently had wine installed.

check the appdb at wine HQ and see if your game is listed.

other than wine, or dual booting (often the best bet), the only other option is virtualization. VMWare Player and workstation can provide limited 3d accelleration, so for some games I have run a virtualized XP desktop on my ubuntu host. kinda annoying, but it got me through galciv2
