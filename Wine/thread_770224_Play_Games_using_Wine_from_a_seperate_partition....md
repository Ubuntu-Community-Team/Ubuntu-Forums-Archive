### Post by janfsd on 2008-04-28
Yes, you could run them, just add the vista's registry entry of the game to your wine one. 

If you don't know, in cmd, type regedit. Find the entry, right click and export, save it somewhere. After in Linux, just in the terminal run regedit again, and you shoud be able to import that file.
