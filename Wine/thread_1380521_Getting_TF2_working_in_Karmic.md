I was recently trying to get TF2 working in Karmic. I just couldn't get it to work right and all the usual tricks weren't working (I was having network problems, sound issues and the game kept crashing). Somehow or other I discovered there was a wine1.2 package in Synaptic. I installed that and now it's all working perfectly. It is a beta of wine, so it might not work perfectly for everything, but if you're having issues getting TF2 running it might be worth a go.

Hopefully this will help someone and save them a bunch of time.
