### Post by FreewheelinFrank on 2007-12-13
I've just had the same problem.

I was prompted to install Gecko during the Steam install but the installation failed.

I then went to Applications>Wine>Programs>Steam and Clicked on Steam Support Center. I got the same dialogue box about installing Gecko, accepted and Gecko was downloaded and installed without a problem.

I could then enter my Steam username and password without problem in Steam.
