### Post by cogadh on 2007-11-12
The mouse issue is a problem with Wine's implementation of DirectInput. The bug still exists in the latest version of Wine (0.9.49). It has nothing to do with the "Allow DirectX Apps..." setting which has worked for quite a while now.

All of the Orange box games work in Wine, most of them quite well. It does require some effort to get them installed and running, but as long as your system specs are OK they should run just as good as, if not better than, they do on a Windows machine. 

VTM:B does not work at all in Wine (just tried with the latest Wine yesterday) and AFAIK, HDR rendering won't work either. GalCiv 2 does work in Wine, though not perfectly.
