### Post by marl30 on 2011-01-09
> **{davros} said:**
> yes i know this its installed but has yet to be able to open anything or show uo in applications with a gui ..... i have it but its not woking

That's because it's integrated in archive manager. If you noticed that you're not able to extract or zip rar and other format using the archive manager, you may need to install Ubuntu-restricted-extras from the Software Centre.

If you want to unzip a file, just right click on the file and select extract here. To zip in different formats, highlight the files you want to zip, then right click on the highlighted files and select compress. 

If you rather to use a GUI zip program, try Peazip. It is in the Software Centre, and has the same functions as 7-zip.
