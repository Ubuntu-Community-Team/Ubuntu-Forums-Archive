### Post by hikaricore on 2008-09-13
From what I understand you are not going to be able to run Spore on linux unless you have a pirated or cracked copy of the game (if WINE even can run it), we will not discuss that here.

Battlenet may not work on Linux without some special configurations, also if you're using a pirated or cracked copy of the game it may not work at all, and we can not help you.

Please check the WINE application database for specifics on your games:  [http://appdb.winehq.org](http://appdb.winehq.org)

WINE does not use anything special for internet connection, just your default network setup.
If you are able to connect the the internet normally and not through these games, I suspect your network connection is working just fine and the problem lies within these games instead.
