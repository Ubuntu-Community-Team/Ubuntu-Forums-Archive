### Post by yther on 2009-02-15
> **Aflack said:**
> i wanted to play gunz a long time ago on linux til i learned you couldnt. but, will there ever be any chance gameguard will work on linux? not with just wine but anything?

Possibly; "ever" leaves a lot of room for speculation, you know.  ;)

However, I don't think WINE is going to support Windows *device drivers* any time soon, so emulation is the only real choice you have.  Further, only VMWare seems to support DirectX, which many Windows games need, so although you *can* run DirectX in WINE (I've tested this), if your game requires both that *and* GameGuard, you have to use VMWare.

Check the stickies over in the Virtualization forum for some guides on that; there's also a short one in the Ubuntu help docs.
