### Post by cogadh on 2008-07-03
Starcraft can be run in Linux using Wine, though it doesn't run perfectly:
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=72](http://appdb.winehq.org/objectManager.php?sClass=application&iId=72)

As for Starcraft 2, it won't work natively in Linux and AFAIK, there are no plans for a Linux native release. It may work in Wine, but newly released games often do not work well, if at all, in Wine.
