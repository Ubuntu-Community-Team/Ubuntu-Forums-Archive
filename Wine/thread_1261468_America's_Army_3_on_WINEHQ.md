### Post by xenogia on 2009-09-09
Hey Snoopy,

According the Wine AppDB this is what it had to say about America's Army 3

```

 **What works**

-Single player Training               

   -Lan  Multiplayer               

   -Account logging (must create a new 3.0 account and you'll be asked later for your old 2xx account)
    -In Game PunkBuster Update.                

   -Sound and Graphics ok (needs a BIG machine to get fine fps)               

 
**What does not**

-Installing game from steam hyperlink (check additional comments how to fix).               

   -Internet Multiplayer      (game crash after loading the map, Punkbuster???)          

   -Mouse bug like in older versions, (check additional comments how to fix).               

 
**What was not tested**
-Internet Multiplayer over NO PUNKBUSTER servers. 

**Additional Comments**

VERY IMPORTANT:
===============

STEAM KNOWN BUG OF HYPERLINKS DISABLES BUYING GAMES FROM STEAM:

To get America's Army 3 within Steam follow this instructions:


http://www.leenukes.co.uk/?p=109



 FIXING MOUSE BUG
==================

To get mouse pointer working - similar problem has been discussed:
appdb.winehq.org/objectManager.php?sClass=version&iId=5444

To make it short - you have to replace one particular wine file which u get there - fxlegrand.free.fr/index.php?page=patches


```
