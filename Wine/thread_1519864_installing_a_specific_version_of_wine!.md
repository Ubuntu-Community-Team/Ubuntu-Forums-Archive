### Post by cogadh on 2010-06-28
I'm not sure I understand what you are saying, but I think you are trying to install an old version of Wine. First question I would ask is why do you need to do that? Generally speaking if the stable 1.0.1 version doesn't work for you, the latest and greatest is the better choice. 

As for how you might get an older version, since packages for older versions are no longer made for 10.04, you will probably have to download the source for an older version and compile it yourself:
[http://sourceforge.net/projects/wine/files/Source/](http://sourceforge.net/projects/wine/files/Source/)
