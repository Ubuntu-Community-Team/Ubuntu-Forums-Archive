### Post by NickHansonNet on 2010-08-14
When ever you want to run a EXE file in ubuntu, you will have to install WINE. I know you already have that. So, if you get this error
```
EXE/Program is not marked as Executable
```
Heres a fix.

On the exe you downloaded, Right click and go to Permissions or something like that. Then Click Mark as Executable.

That will fix your problem.
Reply me back if you have any questions.
