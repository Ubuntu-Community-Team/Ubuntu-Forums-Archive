### Post by NightMKoder on 2009-03-23
I just saw a similar problem that seems to never have been resolved.
[https://answers.launchpad.net/ubuntu/+question/17611](https://answers.launchpad.net/ubuntu/+question/17611)

I think you're missing the 32-bit wine deps (or something went really bad with them). Try:
```

sudo apt-get --purge --reinstall install ia32-libs

```

I don't actually run 64-bit, but I got this from the official wine "compiling on 64 bit" page
[http://wiki.winehq.org/WineOn64bit](http://wiki.winehq.org/WineOn64bit) 
[http://winezeug.googlecode.com/svn/trunk/install-wine-deps.sh](http://winezeug.googlecode.com/svn/trunk/install-wine-deps.sh)
