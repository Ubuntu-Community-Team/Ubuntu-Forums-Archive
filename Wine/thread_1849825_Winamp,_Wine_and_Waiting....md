### Post by Tweak42 on 2011-09-27
[Wine is not a virtual machine.]("http://wiki.winehq.org/Debunking_Wine_Myths#head-00823154c670185f03a0840cc23f96d2ab070bc9")

That said according to the [Winamp appdb]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=2723") entry it seems it run fairly well.  There are a couple of tips there you might want to try on your winamp install.

I don't know how it compares to Rhythmbox but I use Banshee for tagging.  Just make sure you enable "Write metadata to files" in the preferences.
