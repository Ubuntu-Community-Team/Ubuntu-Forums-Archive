### Post by matt-fender on 2010-07-04
WINE: Wine is a compatibility layer package which allows the user to run SOME windows applications natively on the linux machine, programs such as, WINRAR, NOTEPAD etc etc.

DUAL BOOT: Dual booting is the act of running 2 operating systems on one machine, YOU MUST reboot the machine to access the other operating system.

VIRTUAL MACHINES: Virtual machines allow the user to run 2 or more operating systems AT THE SAME TIME, using a program such as VMware or VirtualBox.
