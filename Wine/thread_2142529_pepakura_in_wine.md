Anytime you're going to run ANYTHING in Wine, you should first go to the WineHQ site to see how well that app works with Wine.

The page linked below is from that site and, as you can see, has very low ratings.  You should click on each of the ratings and read the details -- to see what, if anything, works and what they did to get it working:
