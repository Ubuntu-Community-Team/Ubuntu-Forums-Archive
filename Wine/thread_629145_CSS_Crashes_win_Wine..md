### Post by cogadh on 2007-12-02
I think that is relelated to the Steam registry release problem. It happens when Steam is not exited out of cleanly. To correct it, all you should need to do is start Steam, let it load, then close out of it cleanly (use the File > Exit option in the Steam browser). Once you have a clean exit, then you can re start Steam and try running the game again.
