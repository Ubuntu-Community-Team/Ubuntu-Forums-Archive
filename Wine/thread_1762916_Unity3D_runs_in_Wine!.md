### Post by skullmunky on 2011-05-19
Unity3D is a full-featured game engine and visual development environment, with modern features like physics, networking, modern shaders, lightmapping with Beast, and the ability to target Windows, OSX, a web plugin, iOS, Android, Wii, and soon PS3 and XBox360.  It's priced to be affordable for small, indie studios, and there's also a $0 version with a few of the fancier features removed.  

The editor runs in OSX and Windows (it was originally OSX only, strangely enough).  A Linux port is not impossible, though probably not likely in the near future - but in the meantime, I've just discovered that the Unity3D Editor AND the games it makes run quite well in Wine!  

There are, of course, plenty of great FOSS game engines with good linux support, such as Ogre3D, Panda3D, and Irrlicht Engine, plus high-end commercial ones like Unigine - but this is a cool option for the Linux game development community.

http:/www.unity3d.com
