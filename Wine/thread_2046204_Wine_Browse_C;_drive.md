### Post by Butthead on 2012-08-22
You can edit application paths in the Application Launcher by RIGHT clicking on it (right mouse button) and choosing the "Edit Applications" menu item that then is presented to you.  Navigate down to the Wine sub menu.

My "Browse C: drive" doesn't work either (even when using their examples).  Just use your File Manager app to edit stuff in the .wine folder.  Open up the File Manager and hit "Alt + ." keys to show the hidden .wine folder.
