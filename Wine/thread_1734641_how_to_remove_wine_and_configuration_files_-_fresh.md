Hello,

since I wanted to get rid of old programs and wine old configurations, I wanted to make a fresh installation on my ubu box.

So in order to make things crystal new do:
1)open a terminal and wipe out:
rm -rf .wine
2)then open synaptic and choose for removal the wine package (only this would suffice)
3)right click on applications tab and choose:
Edit Menus 
a new window will appear and there under the window "show item" remove the folder named Wine.

Just go to synaptic, install again the version of wine you had, and install anew the programs you wish.
