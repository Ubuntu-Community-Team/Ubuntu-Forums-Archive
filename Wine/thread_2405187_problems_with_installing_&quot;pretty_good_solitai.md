### Post by Holger_Gehrke on 2018-11-02
If you don't have your heart set on this specific program but just want a solitaire program that knows all your favourite games, you might give pysolfc a try. It knows about 1060 variants, ranging from the mundane to the positively exotic. The version in the repositories is quite old, but the project has been dormant for almost 8 years and has only started publishing new versions again in November 2017. You can install it with 'sudo apt install pysolfc' in a terminal.
