### Post by Impavidus on 2013-10-01
You could try wine, which is a compatibility layer allowing you to run Windows executables on Linux. It doesn't always work. Try [http://appdb.winehq.org/](http://appdb.winehq.org/) to find out whether other people had success installing your app.

Else, consider using a virtual machine or continue dual booting for a while.

### Post by Mark Phelps on 2013-10-01
The linked page from the Wine HQ website shows the ratings for different versions of VectorWorks:  [http://appdb.winehq.org/objectManager.php?sClass=application&iId=488]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=488")

CLick on the version you intend to install to get more details.

But ... I will tell you in advance, a Silver rating means that many of the features don't work -- so even if you do get it installed, it may not end up being worth the effort.
