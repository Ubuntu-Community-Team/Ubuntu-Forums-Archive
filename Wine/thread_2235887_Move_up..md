### Post by grahammechanical on 2014-07-23
I do not have any experience with Morrowind. I do run one Windows application under Wine and when I installed using it Wine it puts an icon on the desktop and in the Ubuntu Dash. Did this not happen with you? How did you install Morrowind?

The simple, but not obvious, method is after installing Wine to open from the Dash Uninstall Wine Software. That will open a Windows type Add/Remove dialog with a button labelled Install. Click that and a Windows type explorer dialog will open and we can browse to the folder that contains the application's setup.exe or install.exe file. Clicking on that will run Intellishield and we install the application as we would when using Windows. This has been my experience with just one Windows application.
