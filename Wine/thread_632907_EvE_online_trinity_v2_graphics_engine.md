### Post by buntunub on 2007-12-16
Here is where its at for wine. Reposted from [here]("http://myeve.eve-online.com/ingameboard.asp?a=topic&threadID=30457&page=59")

> Basically you have to do this:
-Download the wine-git version, see here [http://wiki.winehq.org/GitWine](http://wiki.winehq.org/GitWine)
-Download the elf's patch ( [http://elfe.mine.nu/eve/linux/trinity.diff](http://elfe.mine.nu/eve/linux/trinity.diff) ) in the wine git directory and apply it with patch -p1 < trinity.diff
-Compile wine git
-Install the premium client of eve
-Install DX9 with the file in the bin directory of eve (it's named redistsomething.exe). If this doesn't work, try with this: [http://www.microsoft.com/downloads/details.aspx?familyid=9226a611-62fe-4f61-aba1-914185249413&displaylang=en](http://www.microsoft.com/downloads/details.aspx?familyid=9226a611-62fe-4f61-aba1-914185249413&displaylang=en)
If some dll is missing download it and paste in the system32 directory of wine
-Copy arial.ttf in the windows/font directory of wine (search it in your computer if you've installed ms fonts)
-Paste this line in the /etc/hosts file: 87.237.39.199 localhost and restart the network
-Launch eve, press esc and uncheck the premium client option

Hope I didn't forgot anything

Even with this, its still extremely unstable, so be warned.
