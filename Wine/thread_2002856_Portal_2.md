### Post by chase321 on 2012-06-13
For anyone in my situation, i am trying to install portal 2 on ubuntu 12.04 using wine. the game installs perfectly and starts perfectly, however when you start playing all you get is a black screen, the correct audio and crosshairs. there is also mouse movement because when you first start the game, they ask you to look up and down, and when you move the mouse up and down it registers the movement. to solve this, all you have to do is go into the ubuntu software center, and install DRI 3d accleration. and then configure it to accept graphic compression. the go in game and if it still isn't working go into options, keyboard options and allow the management console. the go into single player and start a game, press escape and then press the button to the left of the 1 which is ~ <-- button. this opens up the console which you then type "mat_postprocess_enable 0" into the console and press enter. restart the game and it will work
