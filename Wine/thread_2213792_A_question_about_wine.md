### Post by Mark Phelps on 2014-03-28
> I have to use origin for my favorite game's

Sorry, having trouble understanding what it is you are asking.

Windows games generally work poorly, if at all, in Wine.  To find out about the ones you want to run, go to the link, enter the game name and version you want to run, and see what rating it has.  Anything not found, or rated lower than Gold, is not going to work for you: [http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true")

### Post by oldrocker99 on 2014-03-28
The only way to find if Origin will work with wine is to doenload the Origin installer and try running it with wine. [http://appdb.winehq.org/objectManager.php?sClass=application&iId=13176](http://appdb.winehq.org/objectManager.php?sClass=application&iId=13176) says that it has a Silver rating, meaning it works well but not perfectly. You'd then have to check out on the same site which games you have and how well they run with wine.
