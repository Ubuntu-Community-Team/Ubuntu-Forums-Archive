### Post by jwaclawsky on 2010-12-11
Using the terminal I am trying to install wine on Ununtu 10.10 using the three commands given here: [http://www.multimediaboom.com/install-wine-1-3-8-in-ubuntu-10-10-maverick-ppa/](http://www.multimediaboom.com/install-wine-1-3-8-in-ubuntu-10-10-maverick-ppa/) They are:
 
sudo add-apt-repository ppa:ubuntu-wine/ppa
sudo apt-get update
sudo apt-get install wine1.3

Everything is fine with no errors until I do the install command. Again no errors but eventually the process puts a EULA up in the terminal window with the title "Configuring ttf-mscorefonts-installer" and I can't seem to figure out how to reply to the <ok> in the screen presented. I have included a screen shot of the terminal window where I am stuck. The EULA fills the terminal window. The <ok> is not clickable and there is no place for any user input or commands. I can't finish the install. And when I try to exit the terminal it tells me a process is still running so I know it is waiting for input from me? Can anyone tell how to respond to the EULA in the terminal window or tell me what I am doing wrong. Thanks!
