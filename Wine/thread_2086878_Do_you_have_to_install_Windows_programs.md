### Post by MikeCyber on 2012-11-22
Generally all apps that make registry entries need to be installed.

For ex. I can start FS9 but needed to reinstall many addons. Now this is also 
handy as I link .wine from 12.10 and also 10.04 and have all apps on a special partition, hence the paths are the same.

I also fire up the very same FS9 from Windows8 but this needs another addons reinstall for the registry entries. (simply overwrite as only the registry is different)

Very good...you save any .wine so can go back later...without the usual windows fuzz.
