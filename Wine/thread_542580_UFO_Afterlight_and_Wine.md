### Post by cogadh on 2007-09-05
That wouldn't be a graphics option, it would more likely be a control or general game option. It is also possible that there is no option for it in the game itself, but you might be able to edit one of the game configuration files to turn off the hardware mouse. I can't say what file that would be or what you would need to change, I don't have the game. However, some games don't have the option to turn off the hardware mouse at all so you might be stuck.
