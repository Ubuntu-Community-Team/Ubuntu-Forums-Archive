### Post by bobwyajnr on 2010-11-27
Yes it works fine (refer to the [Firefox AppDB page]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=18256")). In fact some folk argued the Windows version got more love and ran faster in Wine than the native Firefox!

Personally I find native Firefox + omnibar + decent theme (Oxygen KDE is just fine now for me) looks OK. I like the fact the browser is so similar on a different OS (without running it in Wine). The only weak point I can see is the fonts are much more scrappy on the Linux version (that could just be Gnome actually). But I think running the Windows version through Wine would have the same short comings.

Is Google Chrome any good for your banking problem (since that is cross-platform now as well)?

Personally I think you would be better off shooting for a native solution - like with media players. I reserve Wine for the really tough chestnuts like DirectX games that have no native equivalent.

### Post by lovinglinux on 2010-11-27
Have you tried to change the user agent for the bank site?

Opera allows to change user agent on a site basis. Right-click on the bank page, select "Edit Site Preferences", click "Network" tab, change the value of "Browser Identification" to "Mask as Internet Explorer".
