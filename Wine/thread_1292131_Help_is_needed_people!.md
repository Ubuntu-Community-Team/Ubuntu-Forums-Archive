Hi Dreamvirus.

Well, what i did is first install WOW. Copy ALL your WOW game cd&#347; onto your hard drive into a single folder on your desktop.

right click on "Install wow" and you should see "Open with Wine Windows Program Loader" and the install should start and proceed in the normal manner.

Patch the game.

When done you can click on the WOW symbol on your desktop and start the game

All this takes a while but it worked for me.
