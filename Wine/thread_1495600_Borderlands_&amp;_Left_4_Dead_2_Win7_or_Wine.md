### Post by frostschutz on 2010-05-28
Can't speak for L4D2, but HL2/TF2:

Got it to work in Wine, but it's not anywhere near as fun an experience than playing directly in Windows. Wine sometimes tend to crash and take your whole X server with it. The performance is worse than Windows. You don't have all the graphics capabilities like in Windows so expect to lower the graphics quite a bit.

I eventually stopped using Wine altogether - and dual-boot Windows 7 for Windows games. Linux games (like the recent Humble Indie Pack) are always welcome though, and old stuff also runs well in Linux (dosbox, scummvm).

Usually you'd expect that older Windows games should run fine in Wine today, but unfortunately it seems that game programming for Windows isn't exactly science either, so there are a lot of dirty hacks that make things break in wine. Take Diablo for example, stone age game but a classic, and it does not run in (unpatched) wine since there is some kind of black screen problem.

So in the end you still need to run Windows, although for older games, a Windows VM may suffice, which isn't nearly as bad as having to leave Linux entirely...
