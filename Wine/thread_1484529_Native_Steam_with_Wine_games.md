### Post by cogadh on 2010-05-17
Phoronix: the *least* reliable source since Wikipedia. 

Finding references to Linux in the code of the Mac Steam client is not at all surprising; there are references to Linux all over the Mac OS itself, since it is basically a second cousin of Linux. Besides, that could just be a leftover piece of development code they are seeing, we really don't know what systems Valve uses for all its devel needs.

As for the question at hand, if a Linux Steam client comes, it will only be for native Linux games and it will not interact with Win games through Wine at all.
