### Post by NightMKoder on 2009-02-25
If you're running WoW, you're running it under wine. Nothing else runs windows applications (except wine derivatives: codeweavers, cedega, etc.).

Wine's AppDB lists one of the fixes without recompiling wine being running the game in windowed mode. Other than that, you should use that patch.
