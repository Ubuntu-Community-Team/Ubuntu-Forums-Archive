### Post by soxs on 2008-04-30
A. run wc3 via for RoC

```
cd /home/YOUR_USERNAME/.wine/drive_c/YOUR_WC3_FOLDER
wine ./Frozen\ Throne.exe -opengl

```
&
```
cd /home/YOUR_USERNAME/.wine/drive_c/YOUR_WC3_FOLDER
wine ./Warcraft\ III.exe -opengl
```

Note:
Easiest way to keep trak of that permanently is to define a manual comand:
```
sudo nano /usr/bin/wc3roc
```
You should get a new empty text file.
Write this into the text file (for RoC):
```
#!/bin/bash
cd /home/YOUR_USER_NAME/.wine/drive_c/YOUR_WC§_FOLDER/wc3
wine ./Warcraft\ III.exe -opengl
```
Now make it executable via:
```
sudo chmod +x /usr/bin/wc3roc
```
From now on, to run RoC simply type ```
wc3roc
``` into a terminal.

In order to play via b.net, you need to compile wine yourself with a petch.

### Post by nille on 2008-06-15
> **wh00mp said:**
> My Rezolution is 1280x800 ... anybody know how to make that in WarCraft ?

Start **regedit** from a terminal. Locate the following:
HKEY_CURRENT_USER\Software\Blizzard Entertainment\Warcraft III\Video

There you will have two keys called **reswidth** and **resheight**. Set them to the appropriate values (1280 and 800). Just remember: decimal, not hexadecimal!

### Post by soxs on 2008-10-25
@whoever icq_mailed me: Open a terminal (Main Menu -> Accessoiries -> Terminal): and type:  ```
cd /home/YOUR_USERNAME/.wine/drive_c/Program\ Files/Warcraft\ III
