### Post by Dark Hornet on 2007-12-13
I have found that the best way to install WoW or any other game for that matter is to copy the discs to the hard drive first, then run the installer.  I have never used CrossOver, but I do use WINE, and it runs WoW great.  There are many great threads following this topic--try this one  (WINE, WoW):

[http://ubuntuforums.org/showthread.php?t=579378](http://ubuntuforums.org/showthread.php?t=579378)

This thread is a bit long, but it covers a ton of really good information....Good Luck!

**EDIT**  or if you happen to have WoW already installed on a windows box with all of the updates, and BC--simply copy your WoW directory/folder from your windows box over to your ubuntu box.  This is a great way to do it, as there is no update needed!
