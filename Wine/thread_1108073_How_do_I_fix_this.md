### Post by ajgreeny on 2009-03-27
If you are using compiz on your desktop, ie the 3D desktop effects, try running without by using ```
metacity --replace
``` and try wine again.  Compiz can sometimes mess up the graphics in certain programs.  To get compiz back use ```
compiz --replace
```
