### Post by Baldrick_NZ on 2013-05-01
Do you know what the programs on your disc are called? If you do, you could always insert the disc into the drive and browse the drive using files (nautilus), and install from there.

With regard to Acrobat, you'd probably need to download and install the Windows version for a Windows disc to work with WINE. Ubuntu already has an Acrobat alternative installed by default.

WINE isn't perfect, so something's don't work as you'd expect - if at all. 

Seems odd to have a Linux training CD that doesn't have a Linux version! lol.

### Post by Mark Phelps on 2013-05-02
Instead of double-clicking on chapter headings, try right clicking and then point to the adobe app to open it.  

Be advised though, even if that works, you will still have problems if they've embedded Windows stuff in the training, like a PowerPoint slide using OLE to embed a video.
