### Post by cwwilson721 on 2011-05-12
I made this post in another thread to try to help another fellow wine/WoW player from a common mistake:

[Other wine programs messing up your previously running Windows programs.]("http://ubuntuforums.org/showthread.php?p=10807046#post10807046")

Maybe it can help with other wine programs, too.

This will only work if you don't need another program to interact with whatever Windows app you have running at the time.

Example: If you need to run Vent at the same time as WoW, they must both be running in the same 'env WINEPREFIX='. Otherwise, press to talk won't work. Of course, you should use a Linux Native client like Mangler, anyway. It works better.
