### Post by cogadh on 2011-05-01
Generally speaking, you either can't or shouldn't do that. There are of course certain games that are exceptions to this, but Wine is not really designed to interact with an existing Windows installation or software installed in it. The correct/best way to use Wine is to install the games within Linux via Wine, that way you ensure that all of the appropriate registry entries and configurations that Wine and the game will need are present when you try to run the game.
