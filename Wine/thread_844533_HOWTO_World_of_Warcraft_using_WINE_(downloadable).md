### Post by aliayaz on 2008-06-29
This is a howto install warcraft THE DOWNLOADABLE version.
you will need:to have wine (sudo apt-get install wine)

Assuming you already downloaded it just run it using wine.
it should download about 3.16gb.

Installation should go smoothly.

now here comes the part where you actually do something

Go to the game directory and run WoW.exe

Login (trial account if you dont have one) and it will prompt you to update just cancel and exit.

Go to the WTF folder and open Config.wtf it should open a text file

Now add these for better gameplay (OpenGL and sound)

SET SoundOutputSystem &#8220;1&#8243;
SET SoundBufferSize &#8220;100&#8243;
SET gxApi &#8220;OpenGL&#8221;

save and exit and run the game if it's still slow try changing the resolution and better if you run it in a window (alsow changable in game)

And yes Private servers work perfectly i even tested them.
