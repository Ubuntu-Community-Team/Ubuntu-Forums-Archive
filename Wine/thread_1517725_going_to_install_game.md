### Post by cogadh on 2010-06-28
The procedures for installing a game are exactly the same as they are for installing anything in Wine, so if you are really asking "how do I use Wine", then you should probably spend a little time reading the documentation on the [Wine website]("http://www.winehq.org/"). Also, spend a little time Googling, there are dozens of other sites out there that offer info, both general and specific to certain applications, about how to use Wine.
