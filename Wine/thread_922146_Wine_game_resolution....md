### Post by hikaricore on 2008-09-17
If you're looking for fullscreen size this can be a bit cumbersome, however if you
don't mind a desktop window, the following command can make one of any size.

I've used part of my launcher for flatout2 as an example:

> wine explorer /desktop=1024x768 flatout2.exe

Where the desktop size is you can replace this with any standard or non-standard size that the application is capable of.
Such as 800x600 or 1280x1024, depending on the application and how you want it to be sized.

Hopefully someone else can provide further info if this does not fit your needs.
