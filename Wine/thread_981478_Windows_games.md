### Post by cogadh on 2008-11-14
You should check WineHQ's [Application Database](http://appdb.winehq.org/) for what games work with Wine, as well as what you might need to do to make them work. Unfortunately, I can tell you for a fact that RF Online does not work with Wine (yet). There is a [known bug](http://bugs.winehq.org/show_bug.cgi?id=10930) in Wine that currently prevents it from working.

It also may not work with Cedega. The installer for the game is known to work, but there has been [no testing data submitted](http://www.cedega.com/gamesdb/games/view.mhtml?game_id=4187) for gameplay, which is usually not a good sign.

As for other apps to use for gaming, your only options for running Windows games in Linux are Wine, Cedega and CrossOver Games. If the game cannot be run in any of those, you're out of luck.
