### Post by christopher.wortman on 2011-10-15
[http://frankscorner.org/index.php?p=mid](http://frankscorner.org/index.php?p=mid)
as of 11.10 you have to go into terminal and sudo apt-get install timidity
You can find this in the Software Center I used the software driven one, you can use timidity++, but you are stuck setting up fluid yourself
also follow [http://wiki.winehq.org/MIDI](http://wiki.winehq.org/MIDI) if you are unable to get timidity to work
install Ubuntu Restricted Extras packages to play .wav files
I used Fluid U3 GM
install as you would on Windows using the setup file
activate it
It takes a little fenangeling but with a little effort it does work well.
every time you want to run it on a fresh boot hit alt+f2 and run the command [FONT=verdana][SIZE=2][B][SIZE=1]timidity -iA
[/SIZE][/B][/SIZE][/FONT]
