### Post by Zzl1xndd on 2010-10-29
Select Folder > Install > If its in the game Database it should pick up the CD (if it doesn't select the game from the drop down). 

If its a game in the Data base it should input the best selections for you. 

If it is not in the Database give the folder a name and navigate to the .exe using the browse option. then click Install. 

Once its completed you should see icons in the shortcut section of Cedega > Select the game you want and click play.
