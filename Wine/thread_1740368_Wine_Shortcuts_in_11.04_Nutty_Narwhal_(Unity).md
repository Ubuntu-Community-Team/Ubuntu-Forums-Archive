### Post by Metaljaz on 2011-04-29
Well i found them. If you look in the Applications Icon in the Unity side bar you should select installed and show all installed applications. Once you find the icon and then open the program the icon will appear in the sidebar. There you can right click on the icon and select to add to launcher. Much easier now.

### Post by John Coulthard on 2011-06-07
The following procedure works for me. 

When the Windows Installer is running it will typically have an option to "make a shortcut on the desktop", which I select. (If it doesn't I can re-log in under classic mode and go to the Wine menu to select the program and make a copy on the desktop.  Then I re-log in under Unity mode.) These shortcuts have always used the program icons, not the wine cup icon.

I created a folder called "Wine Programs" and moved the desktop shortcuts over to there. Then I drag them to the launcher and the job is done. I have the Wine program shortcuts on my launcher and not cluttering up the desktop.  The icons use the proper program icons.
