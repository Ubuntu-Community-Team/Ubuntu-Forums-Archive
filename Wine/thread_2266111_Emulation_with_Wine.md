### Post by martytm42 on 2015-03-01
I think you'd be best off using something like Retroarch, since that has Ubuntu native emulator cores for pretty much every major system. If you want to emulate Gamecube there's Dolphin which has a native linux version as well. The only system I can think of that you might want to run a windows emulator for is n64, as Project64 under Windows is still infinitely more usable than mupen64plus in its current state. Though for performance reasons you'd still probably be better off using the Retroarch mupen64plus core.
