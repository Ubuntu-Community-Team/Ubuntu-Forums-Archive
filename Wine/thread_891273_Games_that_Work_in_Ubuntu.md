### Post by daengbo on 2008-08-15
I'm not much of a gamer, but my gal running through a bunch of downloadable games rather hit-and-miss has prompted me to start a blog detailing which of these games work on Lenny either natively, with the publisher's Wine wrapper, or with the current Wine and no special configuration. These games should all be stupidly simple to install.

Each entry includes:

1. A link to the game's download page
2. A screenshot
3. The publisher's description
4. System requirements (often this part is difficult to surmise)
5. A gameplay video if one is available; and
6. Instructions for installation (just simple instructions like "Unpack and run install.sh as root" or "Install the .exe using Wine")

Because of the backlog of games my gal has gotten into over the last few months, I've put up 30 entries today, but I'm almost out of material now. I could really use a teammate on this project, especially one running the popular Ubuntu (I'm on Debian Lenny).

Is anyone here interested in helping out? There's a post template already in place, so entries only take about five minutes or less each.

Hit me up if you're interested.

[Games That Work]("http://gamesthatwork.blogspot.com/")

Very few of these games are Free, or they'd be in the repos, but I want to help overcome that "No games" stigma.
