### Post by andrea000 on 2009-09-14
I run the same version of Rosetta stone but i use
wine 1.1.26 if you installed the default wine in
synaptic then it's 1.0.1.Ok go to uninstall wine
software at the top there is a install software
tab click that at the bottom set the file type to
.exe and find your file most likely it will be on
your desktop it should then install.I would recommend
installing 1.1.26 i think it runs Rosetta stone better.
You can install software by right clicking the desktop
icon but i find that when you do it that way sometimes
it doesn't install very good or it will not uninstall
if you would like to try a newer version of wine go to
synaptic and install gkdebconfig then go to [here]("http://wine.budgetdedicated.com/archive/index.html") and 
download a deb package then just double it and it will
install
