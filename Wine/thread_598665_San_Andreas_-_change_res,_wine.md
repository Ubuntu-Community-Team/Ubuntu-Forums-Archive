### Post by hikaricore on 2007-11-02
You should try launching GTA:SA in a "virtual desktop" like so:

> wine explorer /desktop=1400x1050 gta_sa.exe

I've found this is the best way to run it personally, and it allowed me to change the screen resolution setting.
Adjust the desktop size setting as needed, but you probably won't need to as "virtual desktops" launched in this way for the most part are dynamic and will resize accordingly with full screen applications.

Best of luck.
