### Post by forrestcupp on 2012-04-22
It might be easier to just open Nautilus, the file browser, and navigate to your CD drive in that, and find install.exe. Then you should be able to right-click on install.exe and choose Open with Wine Windows Program Loader to load it with Wine.

Doing it that way would save you from having to worry about what name your CD drive is mounted as.
