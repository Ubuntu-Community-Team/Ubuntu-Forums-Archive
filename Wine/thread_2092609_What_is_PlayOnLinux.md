### Post by TheFu on 2012-12-08
I've never used PlayOnLinux, but I think it is a front-end for WINE with specific settings for specific programs, mainly games.  I doubt it will help you play *"all your Windows games"*, but it should make playing a few of them easier.

When I need to be shown how to use a program, I usually search youtube for demonstration videos.  Of course, the first place I look is on the website for the program [http://www.playonlinux.com](http://www.playonlinux.com) in this case. On the main page there are easy to understand links - one of those is labeled - **I'd like to learn how to use PlayOnLinux. **That link seems like something you'd want to check out.** **There is also a** "Documentation" **link at the top of the page.[B]
