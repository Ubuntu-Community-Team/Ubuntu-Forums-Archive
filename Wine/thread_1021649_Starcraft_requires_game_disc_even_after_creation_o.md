### Post by Ahadiel on 2008-12-26
I believe once you install the latest offical patch, you just need to copy Install.exe from both CD's and rename them to StarCraft.mpq Broodwar.mpq respectively.

But if you want to mount the ISOs instead, type "winecfg" in a terminal, go to the "Drives" tab, and add your ISO's mount point (or modify the existing cd-rom's entry).
