### Post by cogadh on 2008-07-01
On my system, which keeps the home directories on a separate partition from the rest of the system, the entire OS and all installed system software takes up about 3.2GB. I don't really have a lot of extra software installed in the system (I often install into my home directory instead of the system), but I do have two different kernels right now, which can take up quite a bit of space. If your system is similar, plus you have the home directories on the same partition, I can see it easily using up the full 4GB of that drive. 

Considering that 4GB of disk space is the bare minimum requirement for installing Ubuntu, I honestly don't think that Ubuntu is the right OS for that system. You would probably be better off using Xubuntu, which has a minimum install requirement of 1.5GB.
[http://www.xubuntu.org/](http://www.xubuntu.org/)
