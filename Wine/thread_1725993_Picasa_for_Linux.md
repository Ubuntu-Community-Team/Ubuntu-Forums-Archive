### Post by oldfred on 2011-04-10
I install 3.8 windows version in wine and it runs without issue.  Ok, one setting I had to find to get it to upload photos to Kodak.

this is from my install script for a clean install, some of the commands I run outside script, so I commented to to help me remember to run them:
# Wine
wget [http://winezeug.googlecode.com/svn/trunk/winetricks](http://winezeug.googlecode.com/svn/trunk/winetricks)
# sudo add-apt-repository ppa:ubuntu-wine/ppa 
chmod 777 winetricks
apt-get install cabextract wine wine-gecko
winetricks allfonts fontsmooth-rgb gecko allfonts
#Then download picasa for windows 38 newest
#([http://dl.google.com/picasa/picasa36-setup.exe](http://dl.google.com/picasa/picasa36-setup.exe))
#wine picasa38-setup.exe
#winecfg
apt-get install msttcorefonts
