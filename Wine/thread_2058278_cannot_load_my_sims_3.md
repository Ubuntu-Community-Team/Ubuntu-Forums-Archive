What do you mean by "won't load up"? If the disc isn't being detected, then you've probably got a faulty DVD drive. Open the file manager at the top of the unity bar and try and open the disc from there.
If you CAN see the disc, then you can try running the setup executable through wine. If you haven't done so already, open the software center and install the program called "wine". Once you've done that, go to the file manager and right click the setup, open with - wine windows program loader. 
Please keep in mind that sims 3 does not officially support linux. You should always check the compatibility of programs you want to use.
