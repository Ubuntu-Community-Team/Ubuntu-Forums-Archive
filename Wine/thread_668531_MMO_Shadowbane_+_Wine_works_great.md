### Post by coloured on 2008-01-15
Anybody out there looking for a MMO to play on linux, shadowbane works great with the latest version of wine!
Its a PVP orientated game, the best PVP MMO out there for the last 5 or more years.
Download the installer (its free, create an account free).
Install it first on windows (the installer has some java stuff and i cant get it working under linux/wine)
Copy the shadowbane folder to your wine drive, start sb.exe or shadowbane.exe, (make sure you start it in a terminal), note down the  dll's its missing, copy them from the c:\windows\system32 folder on your windows system to the wine drive.
start shadowbane and you're away!

Have fun, sorry it seems so complicated, its really not.
Oh btw, make your character on the server called MOURNING, it has the highest population, they are going to be releasing a new server shortly. This game owns
