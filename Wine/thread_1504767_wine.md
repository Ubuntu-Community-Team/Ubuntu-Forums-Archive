### Post by cogadh on 2010-06-09
Well, you can stop right there, [iTunes doesn't really work in Wine]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=1347").

As for whether or not Wine is "always running", Wine is not an application, it is a compatibility library that allows Windows executables to be run on Linux. It is available when you need it and only "runs" when a Windows application is run.
