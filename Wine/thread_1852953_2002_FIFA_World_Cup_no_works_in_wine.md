### Post by Toz on 2011-10-07
According to the winehq appdb entry ([http://appdb.winehq.org/objectManager.php?sClass=application&iId=3506]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=3506")) this game is rated as garbage. 

There was a bug report some time ago ([http://bugs.winehq.org/show_bug.cgi?id=6606]("http://bugs.winehq.org/show_bug.cgi?id=6606")) and apparently everything is fixed in version 1.3.7. 

If you are still having the problem and you're using a version greater than 1.3.7, then perhaps it may be best to create a bug report over at winehq.
