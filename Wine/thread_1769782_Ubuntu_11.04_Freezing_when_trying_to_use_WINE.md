### Post by bobwyajnr on 2011-05-30
Can't afford these new fangled smartphones myself... :)

But I can tell you that if you need any kind of USB support in Wine, to access the phone, you'll have to wait quite a while for the Wine **1.4.x** release (i.e. the next major release cycle).

An alternative, for the die hard Linux user, is the USB-passthrough support (proprietary) addon for VirtualBox. Unfortunately you need a full Windows license for this of course! I can update my Ipod Touch 3G firmware and contents using iTunes - with the help of VirtualBox and a installed Windows XP guest OS.
