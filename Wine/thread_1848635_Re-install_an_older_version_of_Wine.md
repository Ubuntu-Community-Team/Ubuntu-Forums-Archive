### Post by Toz on 2011-09-23
Try using PlayOnLinux. It will manage wine versions for you. As of version 4 of PlayOnLinux, Google Sketchup is supported with an official install script. You'll need to add the PlayOnLinux PPA in order to get version 4 on your system. Instructions are located at: [http://www.playonlinux.com/en/download.html]("http://www.playonlinux.com/en/download.html") (look at the Ubuntu section - Lucid entry).
