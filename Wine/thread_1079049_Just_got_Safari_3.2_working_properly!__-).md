### Post by BEND IT 7 on 2009-02-24
Hi, everyone.

I just wanted to let you all know that I just got Safari 3.2 working with the latest updated Wine (in Windows XP mode) in Ubuntu 8.10.

It works close to (if not the same) speed as it would on a native Windows installation.  I know there is not a lot of practical use of Safari instead of Firefox, but it is something fun to toy around with and a good move to make for people using a Mac4Linux theme (like I used to...I might switch back now that I have Safari working--between that and AWN it would look almost like a native Mac OS X installation).

Note:  The only extension I have tested is Adobe Flash, but that even works to play Youtube videos!  Just be sure to save the file onto your real Ubuntu desktop and then run the .exe with Wine.  It loads in about 3 seconds and then works.  I was very surprised.  :-)

Note 2:  Downloading is not fully functional.  The Adobe Flash download works automatically, but even downloading items from Apple's own website does not seem to work.
