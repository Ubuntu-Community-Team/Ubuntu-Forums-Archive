As for preformance in Wine? It runs pretty darn good, but of course nowhere near as perfect as the Windows version as I've always had to lower some graphics settings to constantly achieve 60 frames per second. I run nVidia GTX 480 sli x 2 cards... Sadly in Linux sli is still in its early phases compared to Windows driver support... I usually end up getting slower preformance having sli enabled with anything Wine compared to even a single card. I'm hoping for better sli support in future nVidia drivers for Linux since of course SteamOS is a good boost for Linux popularity. ;)

The launcher menu is kind of buggy... so don't drag it around and move it across your screen.

I recommend using PlayOnLinux to install if trying to manually do it all on Wine becomes frustrating. The guided installation with POL works great. :)
