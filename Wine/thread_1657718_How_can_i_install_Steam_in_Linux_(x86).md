### Post by corrytonapple on 2011-01-01
Well, for installing, you would download wine from the software center.  Go to the Ubuntu Software Center by clicking Applications>Ubuntu Software Center (At the bottom).  Do as search for Wine, and it will be the one with the red glass of wine. ;) Install it, then close the Ubuntu Software Center.  Then download the Windows Steam.  When done, go to Applications>Wine>Uninstall Wine Software. When that window pops up, click "Install", and browse to where you downloaded Steam.  Go through the installer like normal, and then it will be done.  You can launch Steam by going to Applications>Wine>Installed Software>Steam>Steam.
Also the instructions should be the same with 10.10 as 10.04.
