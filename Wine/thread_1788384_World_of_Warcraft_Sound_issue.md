### Post by cwwilson721 on 2011-06-23
Mangler is easy. Download the .deb file, and install it. As far as what settings are needed, ask guildies for specific servers, etc. FORGET VENT. The hassles of installging it in wine are NOT worth the effort, when Mangler works just fine (Mangler, however, does not have as many server admin preferences/options, so if you are the Vent server admin, it won't do as well)

As far as sound goes, try setting the mode that WoW runs to Windows XP. After that, if you still have issues, you may have to get the Pulse-enabled version of wine running on your box (Do a Google search. It's easy)
