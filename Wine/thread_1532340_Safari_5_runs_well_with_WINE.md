### Post by Rallg on 2010-07-16
Ubuntu 10.04, WINE: Safari 5 (for Windows) runs well. I haven't tried it with media files. But as a browser, it runs better than I had imagined. My setup also has .NET 1.1, but I don't think Safari uses it.

Why use Safari? I discovered that some XML/XSL files are not correctly interpreted by Firefox (on any platform), but IE and Safari and Opera can read them properly. It's a known Firefox bug (or feature).
