### Post by korab on 2017-01-03
[LIST]
[*][Wine]("http://en.wikipedia.org/wiki/Wine_%28software%29") is a free and open source software application that aims to allow computer programs written for Microsoft Windows to run on Unix-like operating systems. Wine also provides a software library, known as Winelib, against which developers can compile Windows applications to help port them to Unix-like systems.
[/LIST]
