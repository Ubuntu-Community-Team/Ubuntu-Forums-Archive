### Post by Cannaregio on 2007-12-19
You can install and run the fine FRITZ7 using wine.
It's a 2001 software edition that still kicks.
It has a connection to playchess com and everything a good chess program should have. And it works flawlessly in ubuntu.

You can download a Free ten-day trial version of Fritz7 PlayChess client at [http://www.chessbase.com/newsdetail.asp?newsid=237]("http://www.chessbase.com/newsdetail.asp?newsid=237") or easily find some abandonedware copy on the web. Just google for it  adding [-demo and +plus07]("http://www.google.com/search?hl=en&client=opera&rls=en&hs=gLv&q=fritz+7+-demo+%2Bplus07&btnG=Search") and/or use directly your deluge.

Instructions [here]("http://www.braillechess.org.uk/games/fritz7/fritz7.htm").
