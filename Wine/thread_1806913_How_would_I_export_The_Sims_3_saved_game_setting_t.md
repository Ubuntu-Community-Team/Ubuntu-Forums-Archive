### Post by JustinR on 2011-07-18
Hi,

First you have to install the original game on the Linux computer with all of the expansion packs.

If you've already done that, copy the whole folder entitled 'Electronic Arts' from the Windows computer from the 'My Documents' / 'Documents' folder to the Linux computer's (with WINE installed) home folder (equivalent of My Documents) it should be '/home/USERNAME/'

When you run the Sims 3 it should act just like it did on the Windows computer.

I'm not sure if it's the same for PlayOnLinux - but if it doesn't work we can go from there.
