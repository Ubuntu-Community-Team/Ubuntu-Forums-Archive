### Post by aoanla on 2008-09-02
Dracula: Origin, as a quick check of the AppDB shows, is not rated Gold - only the demo is.
The full game is rated Bronze, with a suggestion that it may become Silver or better with Wine 1.1.3.

If you check the AppDB entry, there is a suggestion as to how to improve some rendering issues there. (Essentially, using fbo as your OffscreenRenderingMode.)

Sound issues are possibly problems with your sound configuration. Please run winecfg and play with the sound settings there.
