I am looking for brave testers to test the installation instructions I wrote for **Runes of Magic** and **World of Warcraft**.  You know, it is one thing that it works on your own machine, but another on someone else's box.  So it really is hard to say that it does work out in the wild.

My box:
> 
Processor: Intel Core2Duo
OS: Ubuntu 9.10 "Karmic Koala" **64-bit**
RAM: 2gb
GFX: nVIDIA 7300LE
Driver: nVIDIA 185
WINE: 1.1.32
POL: 3.7 (with winetricks)
My instructions are for installing games via **PlayOnLinux** which minimizes the need to edit-this-and-edit-that when switching between games, and the target audience here is those who just want to make these games run (in other words, creating scripts to automate game-switching is not an option and besides, I don't want that either :p ).

**Who am I looking for as testers?**
- If you are having problems running your game and you previously ran other games before, the problem may be with the WINE configuration and registry edits you did before, so I want to ask you as well to give this a try.

- If this is your first time to try it or maybe you already did before but it failed, it is also possible that you forgot to set something.  My How-To guides include the settings that works for me, give it a try as well.

- If you are a multi-booter and you also want to be able to play using via Ubuntu Linux, then give this a try.  It includes instructions for multi-booters (in fact, it is geared towards multi-booters because I am a multi-booter myself).

*** Note:** Not because the guides were written for PlayOnLinux users does it mean it will not work for non-POL users.  It will work, you just have to skip the POL steps and do it directly with your WINE, and then do the other instructions as well.
