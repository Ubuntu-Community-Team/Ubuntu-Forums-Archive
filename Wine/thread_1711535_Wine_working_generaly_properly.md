None are really recommended. You should only add or replace DLLs in Wine if you encounter a problem with a particular application that requires you to do that. Otherwise, you should always stick with whatever came with Wine.

As for how to do it, the simplest way for most instances is going to be winetricks. You don't need to download anything extra, just run winetricks (instructions [here]("http://wiki.winehq.org/winetricks")) and it will download whatever you need for you.
