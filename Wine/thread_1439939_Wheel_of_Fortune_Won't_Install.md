### Post by Willy Gatti on 2010-03-26
Whenever I try to install Wheel of Fortune Deluxe with Wine in the Lucid beta I get this error message:

Error Message: 

The file '/media/Wheel of Fortune/Wheel_of_Fortune_Deluxe_Setup.exe' is not marked as executable.  If this was downloaded or copied form an untrusted source, it may be dangerous to run.  For more details, read about the executable bit.

       I never had this problem in any previous version of Ubuntu including previous beta versions.  How do I get to install Wheel of Fortune Deluxe in the Lucid beta?

### Post by phatoush on 2010-04-04
Hey, I just ran into the same problem. 

The fix is pretty easy...right click on the exe file, click on properties. Then click on the permissions tab. Check 'allow executing file as program'. Click close, and run the program!
