### Post by cogadh on 2009-01-22
IE6 barely works in Wine and IE7 shouldn't work at all, when you install them normally:
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=25](http://appdb.winehq.org/objectManager.php?sClass=application&iId=25)

If you want to use IE in Wine, you need to use [IEs4Linux]("http://www.tatanka.com.br/ies4linux/page/Main_Page") (up to IE6, IE7 support is still in beta), but unless you need IE for some compatibility purposes, it is best to just use one of the Linux native browsers, like Firefox.
