### Post by epicoder on 2010-06-11
America's Army is a game which runs on steam, which means it's for Windows. You will need to get [Wine]("http://www.winehq.org/site/about") ([Click here to install]("apt://wine1.2")) and install [Steam]("http://store.steampowered.com").

Wine is a Windows program loader for linux; it will let you run most Windows programs.

Wine is experimental at best; this is not the developer's fault as Windows is closed-source and extremely hard to replicate. There are no guarantees that this will work.
