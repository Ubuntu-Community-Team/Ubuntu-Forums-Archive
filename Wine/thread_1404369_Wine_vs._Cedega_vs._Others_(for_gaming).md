### Post by zami on 2010-02-12
I had very bad customer service through Cedega and their payment processors (at the time) World Pay.  I have *nothing* nice to say on that subject.

PlayOnLinux I've only used a few times.  When it works as it should, it should be great!  I haven't had much luck with it myself yet, but the potential is all there.  It installed Steam very easily, but the one game I installed through POL, while the game installed fully and correctly, POL thought the game did *not* install, so the game (and it's prefix) do not show up in the POL menu.  So it's just the same as if I'd hand-built the prefix.

I've had a lot of luck with Crossover Games (and CrossoverOffice before that).  Great customer support, fairly helpful forums, and a nice "compatibility" database.  Usually between Wine's App database and Crossover's compatibility lists, I can get games figured out - or learn enough to know I'm fighting a losing battle and save myself some time.  ;)

That said though, in my experience POL and Crossover are both just very handy managers for Wine prefixes.  They setup the prefix (or "bottle" as Crossover calls them), and if it's a supported game they'll set-up the Windows version, .dll files, and so forth.  Nothing you can't do yourself with the terminal and Cogadh's "winestuff" page 
([http://gwos.org/doku.php/wine:winestuff](http://gwos.org/doku.php/wine:winestuff) ) and a little investigating, but either program could be extremely helpful if you are new to Wine.

Crossover Games has a 7 day trial that might be worth checking out.  And POL is free.  Try both!

-zami
