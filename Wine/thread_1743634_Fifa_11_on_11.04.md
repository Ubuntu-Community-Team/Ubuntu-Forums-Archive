### Post by cogadh on 2011-04-29
The change in Ubuntu version does not change the application's compatibility with Wine. If the game works in Wine already, it should still work in Wine on Ubuntu 11.04; if it doesn't work in Wine, it doesn't matter what Ubuntu you are running, it still won't work. Looking at the game's AppDB entry, it looks like it has very inconsistent performance in Wine, so I wouldn't expect it to run all that well.
