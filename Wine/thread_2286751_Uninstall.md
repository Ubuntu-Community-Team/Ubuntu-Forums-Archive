### Post by mystics on 2015-07-16
Search for "Uninstall Wine Software". From there, you should be able to go through a list of applications you've installed in Wine. You may even be able to find a one-click way to uninstall by just searching for "Uninstall".

If that doesn't work, which may happen if the game threw a bunch of errors during install, you can go into the ~/.wine folder, search for the game folder (somewhere in drive_c), and remove the folder. This might leave a few things behind, such as a perpetual option to uninstall the game, but the actual files themselves should be gone.
