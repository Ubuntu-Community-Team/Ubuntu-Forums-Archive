### Post by freelian on 2016-10-29
I've been playing wizard101 for more than year on an intel box using wine.  My computer is currently running Ubuntu 16.04.  Last night, the game worked fine.  I get on the computer today and I am told that there is a system update available for Ubuntu 16.04.  I install the update and I am told to reboot.  After I reboot, I can get into Wizard101, but I can no longer interact with anything.  Using another account on a different computer, I can see that my character is online, but the character is not able to move.  Eventually, I get a disconnect from the server and I am forced to quit the game.

I don't have any backups and I do not know how to roll back the current set of updates.  I am going to try uninstalling Wizard101 and reinstalling it to see if that helps.  Rebooting does not help with this issue.  I am told by Ubuntu Software that I have all the latest updates for 16.04 installed.

If there is something wrong with the latest Ubuntu update, I want someone to know about this regression so there is a chance that this issue will get fixed in the future.

Thank you.

### Post by freelian on 2016-10-29
Hi,

I just reinstalled the game using the wine uninstaller to first remove the game and then I right clicked on the executable I had downloaded from Kingsisle and told it to run the executable with the Wine Windows Program Loader.  I have to redownload everything, but I am able to interact with the game again.  So, I guess if rebooting does not fix Wizard101, try reinstalling it.

Thank you.
