### Post by Alan James on 2010-04-03
You could probably just save your HTML files then open them in Firefox/Konqueror/Chrome/etc.
 

 However, I think that the current view is that you should use only wine if there is not Linux alternative to your application, such as in the case of a video game or very specialized program. There are lots of great HTML editors on Linux that will run nativity without the need to use wine. I have heard lots of great things about [Kompozer]("http://www.kompozer.net/"), which is cross-platform and has lots of great features. 



If you are 100 percent committed to writing your HTML using Linux maybe you should try it instead. I hope this helps.
