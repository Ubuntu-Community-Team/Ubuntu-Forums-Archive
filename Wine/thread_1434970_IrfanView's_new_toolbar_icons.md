[IMG]http://ubuntuforums.org/%3Ca%20href=%22http://es.tinypic.com%22%20target=%22_blank%22%3E%3Cimg%20src=%22http://i43.tinypic.com/28jwv4l.png%22%20border=%220%22%20alt=%22Image%20and%20video%20hosting%20by%20TinyPic%22%3E%3C/a%3E[/IMG]IrfanView 4.25 has a new toolbar icon set, but when installing this application the way they show on [http://appdb.winehq.org/objectManager.php?sClass=version&iId=7834](http://appdb.winehq.org/objectManager.php?sClass=version&iId=7834), I had the old, not-so-cool icons.

I got IrfanView to display the new icons by following those instructions, and then resizing grosberg_24.png in .wine>...>irfanview>toolbars to 16 pixels high (using GIMP) and renaming it Samuel_16.png (I deleted the original Samuel_16.png file). Make sure to delete i_view32.ini, because the application keeps creating this file.
