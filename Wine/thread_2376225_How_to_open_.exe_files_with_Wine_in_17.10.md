### Post by deadflowr on 2017-11-03
Right click the .exe file and look and the "open with" settings.
See if an option to set A wine application is there and set it as default.
(Also could be called something like Wine Windows Program Loader)

Once use set the default for "open with" on any given file extension, it'll run all other files with that extension.
