### Post by Metallion on 2009-07-30
I've got the wine version that is in the repos: 1.1.26. I tried using the proprietary driver but it made X freeze on startup. Afterwards I read  that it's not supported in Jaunty so I went back to the open source one.

Edit: For now I solved it by adding this to by trying with playonlinux and adding this to my xorg.conf:

```
Section "Extensions"
        Option "Composite" "Enable"
EndSection
```

So far CS is playable but the performance is notably weaker than in Windows and the textures aren't quite as nicely filtered either. I'll mark the thread as solved it CS keeps working for a day or 2. Last time it worked, it stopped working soon afterwards for no apparent reason.
