### Post by jacksaff on 2008-05-06
Most of the apps you are trying to install with wine are 32bit apps and you have a 64 bit system. This is a problem whether they are windows exes or linux bins. If you install ia32-libs then your system should be able to deal with this as it allows 64 bit systems to access 32 bit libraries when necessary. Wine should work fine too, once this is installed. Fixed all my problems anyway.
