### Post by oedipuss on 2008-10-31
The most known would be rosegarden, but threre are some more. Just search the repos for midi or sequencer. "Best" is kind of relative in these cases.
I hear hydrogen is a good drum machine, and if you don't mind closed source and paying a relatively small fee, there's energyXT.
Ardour is said to be pretty good too, and though it doesn't currently support midi tracks, it is planned for the immediate future. (or maybe devel versions do, I'm not sure)

### Post by floatingpoint on 2009-01-26
If you want to muck around with some of the most popular, you could download and install the Ubuntu Studio audio packages. There are DAWs, sequencers, etc.

 If you open synaptic and type 'ubuntustudio' in the search bar, it should come up with lots. Get ubuntustudio-audio and ubuntustudio-plugins. It should alert you to all of their dependencies too.
