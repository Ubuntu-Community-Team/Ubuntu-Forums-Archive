### Post by rocknrolf77 on 2007-12-14
Install wine from add/remove or from a terminal

```
sudo aptitude install wine
```

Then from a terminal do a 

```
winecfg
```

Then you can rightclick the install.exe or what it's called and choose to open it with wine. 

Don't know if it requires any tweaking too work tough.
