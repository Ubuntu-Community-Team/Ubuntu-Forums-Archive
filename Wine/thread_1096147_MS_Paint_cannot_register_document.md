### Post by Dan_Dranath999 on 2009-03-14
um, you could use a native linux image editor closer to MS Paint

there's several...

gpaint: (Available in Ubuntu repositories) -gnome
kolourpaint: (Available in Ubuntu Repositories) -KDE app
mtpaint: (Available in Ubuntu Repositories)

All these are simple and similar to MS Paint, but offer extra capabilities. You can install them via Synaptic or with add/remove programs...
