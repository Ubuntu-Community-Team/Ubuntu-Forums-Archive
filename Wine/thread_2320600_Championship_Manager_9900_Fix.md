### Post by English_Invader on 2016-04-15
I experienced the problem in this thread (program fails to recognise the hard drive):
[http://ubuntuforums.org/showthread.php?t=1812544](http://ubuntuforums.org/showthread.php?t=1812544)

I managed to get round it by running Wine in Windows 98 mode.  Database loads into hard drive and game loads and saves without issue.

I'm using the original retail release of this game (big box version) and I'm running it on Linux Mint 17.2 through Play on Linux.  I've also successfully run the Sold Out budget releases of CM3 and 01/02 in XP mode with the same set-up and have no reason to doubt that the budget release of 99/00 would work in exactly the same way as they all share the same game engine and these versions appear to have been optimised for XP (the original retail release of CM3 is incompatible with XP).
