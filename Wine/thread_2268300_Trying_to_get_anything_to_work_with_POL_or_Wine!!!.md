### Post by Mark Phelps on 2015-03-09
Look at the SIMS2 ratings on the linked WineHQ webpage:  [https://appdb.winehq.org/objectManager.php?sClass=application&iId=1942]("https://appdb.winehq.org/objectManager.php?sClass=application&iId=1942")

They are all "garbage" -- meaning it's NOT going to work no matter what you do.

Windows gaming on Linux using Wine is a very hit-or-miss proposition.  One game might work fine; another might not even install.

Look at the WineQ webpage for SIMS2: [https://appdb.winehq.org/objectManager.php?sClass=version&iId=16664]("https://appdb.winehq.org/objectManager.php?sClass=version&iId=16664")

When you find this page from the listing of SIMS versions, it rates as GOLD.  But, when you open the page, you'll see the Ubuntu rating is Bronze -- which a game gets if it only installs and nothing else works.  Trying to run any game rated lower than Gold is an exercise in frustration.

If you really want to do Window gaming, the only reliable solution is using Windows.
