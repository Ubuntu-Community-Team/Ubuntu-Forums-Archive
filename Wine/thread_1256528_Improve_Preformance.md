### Post by Progressive on 2009-09-03
Not knowing your system specs, there are some DLL files which improve WINE performance, though which ones those are change from version to version.

If the how-to instructions for whichever specific games you are using mention dll files, be sure to get those.

Also be sure to tweak the game settings and registry files according to the how-to guides.

Besides that, you should look into alternate graphics drivers and settings for your graphics drivers.

---

### Post by Simian Man on 2009-09-03
Also play with the graphics settings for the game itself.  Turning down screen resolution can make a huge difference.
