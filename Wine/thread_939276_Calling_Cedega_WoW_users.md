Cedega's based on an old version of wine, so it shouldn't make that big of a difference.  There are some games cedega does better than wine, but WoW is not one of them, as it can run in opengl mode.  My advice is to stick with wine and try any tweaks you can find for it.  You should get decent fps with that video card by ensuring you have the latest drivers installed. Make sure you've edited your config.wtf to disable full-screen glow.  If you're still getting bad fps in WoW after this, try lowering its resolution.
