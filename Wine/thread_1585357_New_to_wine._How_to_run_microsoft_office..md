### Post by FahadMKS on 2010-09-30
To be easier, after installing Wine >> you may also install the application Play On Linux.
Once you have installed it, you will find it in the Games Menu.
There you may select the option as Office and then you can select install.
Once that is done, you will be asked to put on the CD of the MS office.
Insert the CD and it must install the MS office for you.
