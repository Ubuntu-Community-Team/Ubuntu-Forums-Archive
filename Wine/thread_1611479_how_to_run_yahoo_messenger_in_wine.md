### Post by cogadh on 2010-11-02
Yahoo! IM doesn't work in Wine: [http://appdb.winehq.org/objectManager.php?sClass=application&iId=29](http://appdb.winehq.org/objectManager.php?sClass=application&iId=29)

and Google Talk barely works in Wine: [http://appdb.winehq.org/objectManager.php?sClass=application&iId=2398](http://appdb.winehq.org/objectManager.php?sClass=application&iId=2398)

You'd probably be better off using one of the Linux native IM clients that supports both Yahoo! and Google, like Pidgin.
