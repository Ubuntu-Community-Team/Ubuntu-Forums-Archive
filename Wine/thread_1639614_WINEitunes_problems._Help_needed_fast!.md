### Post by khelben1979 on 2010-12-08
To get the best results from using Wine, you can always download the source code and compile it yourself. I would recommend the latest version from 1.3.x. To compile it you only need to go through 3 steps:

1. **./configure**
Configures Wine against your system libraries and searches if your Ubuntu system has what is required before you go on to the next step.

2. **make**
This compiles the source code using the information it got from the ./configure script and the information it was able to gather from your Ubuntu system.

3. **make install** (as root)
Installs the Wine application in your Ubuntu system. Usually placed in /usr/bin.

Some versions of iTunes works better than others, so you can check this link out: [WineHQ - iTunes]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=1347").
