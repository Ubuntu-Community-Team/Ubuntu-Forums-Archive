### Post by ubu-for on 2008-01-05
I play CSS with Wine 0.9.44 and use the Windows XP mode for "steam.exe", the Windows 98 mode for "hl2.exe" and ALSA only for audio.

You also should know that Compiz or Beryl in the background could be a problem.

Follow [this HOWTO](http://www.linux-gamers.net/modules/wiwimod/index.php?page=HOWTO+Steam&back=HOWTO+INDEX+Wine+Games) and read [this thread](http://ubuntuforums.org/showthread.php?t=503613).

Have fun!
