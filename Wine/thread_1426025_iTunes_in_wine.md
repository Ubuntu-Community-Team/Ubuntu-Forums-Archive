### Post by a.walker on 2010-03-09
The last time I check iTunes only worked marginally in wine and was not able to synch with ipods/iphones. You should look through the documentation [here]("https://help.ubuntu.com/community/PortableDevices/iPhone") for information on synching iphones. I personally think that the most trouble-free way to synch it will be through a Windows XP virtual machine in Virtualbox. I know it's a pain, but that's unfortunately one of the downsides to using apple products.

You should also check out iFuse [here]("http://matt.colyer.name/projects/iphone-linux/?title=Main_Page").
