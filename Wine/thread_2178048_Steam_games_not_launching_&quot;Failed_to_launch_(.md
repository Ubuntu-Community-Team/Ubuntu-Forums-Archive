### Post by oldrocker99 on 2013-10-05
Some games have done this for me; Dwarfs?! did, until an update, and a couple of others, including War For The Underworld, which is a matter of the devs not having nailed down the Linux version (promised when I Kickstarted it) quite yet. 

Have you tried entering the game's directory and starting in the terminal? War For The Underworld says "missing executable" when I try to open it from within Steam. If I enter the directory and enter ./WFTO, it does run: It open the help menu and immediately exits. As I said, the Linux version is not yet nailed down. To put it mildly.

For Windows games using Steam with wine, trying deleting the game content and reinstalling through Steam is all I can suggest.
