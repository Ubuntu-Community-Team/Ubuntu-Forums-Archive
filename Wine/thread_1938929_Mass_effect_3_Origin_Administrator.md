### Post by prshah on 2012-10-03
Yup, I know it's late, but just thought I'd post here rather than create a new thread...

I had the same problem in WINE. So I downloaded and installed the "Beta" origin client, which then asked me to update origin; that worked fine, and then the client opened up and I could login normally.

I re-ran the install from the disc, because otherwise, the client was offering a download only. Re-running the install from the disk took me straight to the origin client, and started the install.

Beta download link (As on date): [https://download.dm.origin.com/origin/beta/OriginUpdate_8_6_3_49.zip](https://download.dm.origin.com/origin/beta/OriginUpdate_8_6_3_49.zip)

Not necromancy, just FYI.
