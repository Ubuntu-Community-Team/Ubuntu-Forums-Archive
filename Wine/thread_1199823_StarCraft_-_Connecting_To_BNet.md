### Post by donkyhotay on 2009-07-04
I've never had issues playing starcraft on b.net with a legit key. However do *not* use b.net to update starcraft automatically, it fails quite often. Download the updater from b.net manually and install that way before trying to connect. Also make certain that your wine configuration for starcraft is set for fullscreen & allow the windows manager to control the window. It makes b.net look really ugly and can make it confusing when joining and creating games (especially if you're not used to it) however the buttons *do* exist and you can click on them. If you don't do it that way you can see the buttons but they're 'underneath' and you can't click on them to make any selections.
