### Post by Bios Element on 2008-09-01
> **Saikarthik said:**
> now aftr entering the user name and password its waiting for a while n closing..

According to the Wine AppDB ( [http://appdb.winehq.org/objectManager.php?sClass=application&iId=29](http://appdb.winehq.org/objectManager.php?sClass=application&iId=29) ) Only a couple versions of the Yahoo messanger actually even work. So Your best off installing Pidgin as I don't think your going to have very much luck with Wine. You can do so by going to *Applications -> Add/Remove* and typing in a search for *"Pidgin"*. Then install it and you'll have a launcher for it under your *Applications -> Internet* menu.
