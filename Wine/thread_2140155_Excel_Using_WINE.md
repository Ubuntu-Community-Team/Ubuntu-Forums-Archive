### Post by Mark Phelps on 2013-04-30
BEFORE you do this, read through the details of the link Toz gave you, for the version of Excel you're planning to use.

A Silver rating is quite poor -- meaning, a variety of features are NOT going to work -- for which you will be able to do nothing to make them work.

I generally recommend folks only use Wine when the app and version they want to use has a Gold rating or better; that way, they get most if not all the features they need to use.
