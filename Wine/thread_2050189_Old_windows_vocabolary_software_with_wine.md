### Post by mikechant on 2012-08-30
Wine only runs *some* windows software, there is no indication that anyone has got this particular program to run with Wine.

Your best approach is probably to install Windows in a VM (virtual machine) and install your vocabulary program in that VM.

However, you need a reasonable amount of RAM to run a VM.

How much RAM does your PC have, and do you have your Windows installation discs? If so, what version of Windows is it?
