### Post by Kojak Peg on 2013-05-12
Hi Everyone 
I got it to work. I already had the patch installed and the game unzipped on C: driver. Then I installed vineyard (not sure if I had to but I did anyway), 

I opened **windows applications**. And in the** General** section, I changes the windows version, to 98(its a very old game so figure it would run better in windows 95, or 98). 

Then I went to the **install** section, and installed direct x 9. 

And finally I went to the **tools** section, and looked in the "**run executable**" menu. I found the file **JAUTO.EXE**, opened it. So **JAUTO.EXE** was now in the "**run executable**" line and then clicked "**run**". 

I got an error message, however the game began running before I could read the error message. It took a few seconds but it worked fine it was a little slow to respond in game play and the cut scenes where jerky, but it was okay. And I think I'll set it to windows xp in the general section next time see how that runs
