### Post by DJ_Peng on 2008-11-27
I usually remove them with the alacarte Main Menu app (*System > Preferences > Main Menu*). Simply cselect the item you want to remove and click Delete, or if you're using an older version of Ubuntu (Gutsy or Hardy) you can right click the item and select Delete.
