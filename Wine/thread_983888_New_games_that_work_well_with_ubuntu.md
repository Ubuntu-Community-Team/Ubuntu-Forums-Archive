### Post by Phil Urich on 2008-11-25
Quake Wars doesn't "work" in the sense that the others here work.  As in, [it's actually Linux-native!]("http://zerowing.idsoftware.com/linux/etqw/")  No need for Wine or any of that nonsense (as in, no need for this section of the forum at all, heh).   I haven't played it in a bit for some reason, but it's a damn fun shooter, fast-paced but strategic. 

Also, if you never got around to playing Prey, [it too now has a Linux port]("http://icculus.org/prey/"). Not terribly "new" per se, but hey, still pretty nice and scalable enough that it pays off to have a modern computer that can push it to the max. To involve Wine, well, Steam works great under Wine and you can buy Prey via Steam then use that copy to install it.

The same guy who ported Prey has been working on the UnrealTournament3 Linux port, so that's another native game, but unfortunately legal issues (damn middleware) have been holding it up for a looooong time now.  Theoretically the UnrealEngine3 is platform-agnostic, but industry politics have kept us from getting any of those games yet (I mean hell, Gears of War PC came out for OSX, it could easily have run on Linux, but of course Linux is a far more serious competitor to Microsoft than Apple is).

Again going a bit back, but nearly any ID Tech 4 based game has a Linux port that you can play using the data from a Windows copy of the games (Doom 3, Quake 4, etc).  Also, going back to Ryan Gordon (aka icculus), the old [Postal 2 multiplayer portion of the game is available free]("http://icculus.org/news/news.php?id=4419").

A game that's old-but-new, in the very best of the open source sense, is Freespace 2.  If you're looking for a spaceflight combat game I'm not sure a better one was ever made, and since its been opensourced for so long, these days it actually looks pretty damn snazzy!  There's also tons and tons of mods and addon campaigns for it, many of which you can auto-download when you [install the full game with the java-based installer]("http://www.fsoinstaller.com/").  Speaking of, if you like Battlestar Galactica there's [a demo of a total conversion of Fs2]("http://www.game-warden.com/bsg/demo.html"), although development on it seems to have fallen to in-team fighting as far as one can tell.  Still, a fun and fancy-looking demo (at the retail store I used to work at until recently, it brought even the most expensive computers to their knees at max settings!).

So there's a few new-ish games you can try out, many free, without having to run things through Wine :)  Long live Linux gaming!
