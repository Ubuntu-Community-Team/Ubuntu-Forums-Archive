### Post by cogadh on 2010-05-21
Wine is a compatibility layer that basically fools Windows applications into thinking they are running on Windows while they are actually being run on Linux. The same applies to mods for games. As long as the game itself actually works in Wine, mods for it will too. However, before you get your hopes too high, you may want to check [Wine's Application Database]("http://appdb.winehq.org/") to see if your games of choice actually work in Wine.

### Post by darklegion on 2010-05-22
> **cogadh said:**
> As long as the game itself actually works in Wine, mods for it will too.

This isn't always true. If the mod uses additional executables, and those executables don't work, then the mod will not run. For example Oblivion Script Extender doesn't work with wine, so mods that use it won't work.
