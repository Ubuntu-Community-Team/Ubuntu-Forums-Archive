### Post by Johnsie on 2008-06-27
HoverRace is an online racing game that was written for windows but recently made open source. 

At the moment you need to have wine to play the game on Ubuntu but I have made a .deb installer:

[http://hoverrace.co.uk/files/linux/hrplus-0.0.1.i586.deb](http://hoverrace.co.uk/files/linux/hrplus-0.0.1.i586.deb)


Also, yhere is now a launchpad page for this project and anyone is welcome to contribute code or ideas:

[http://launchpad.net/hoverrace](http://launchpad.net/hoverrace)


Hopefully the game will become more Linux friendly with time.
