### Post by pytheas22 on 2008-04-12
Install wine, run the Windows installer for Titan Poker, click on Applications>Wine>Programs to find the launcher (this may be in a different location; look through the other parts of the Application menu if it's not here), and play.  According to [http://appdb.winehq.org/objectManager.php?sClass=version&iId=4037](http://appdb.winehq.org/objectManager.php?sClass=version&iId=4037) the program runs pretty well under wine, although there might be a few issues.
