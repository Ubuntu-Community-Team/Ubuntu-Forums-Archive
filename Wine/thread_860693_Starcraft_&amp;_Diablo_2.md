### Post by jimv on 2008-07-15
Install XP in VirtualBox.  This is what I do with Diablo 2.  It runs just fine because A) Virtual Box is fast and B) Starcraft/Diablo 2 are older games.

You can full screen Virtual Box, but I prefer playing with VirtualBox in a window so I can see my Ubuntu stuff.

Oh yeah, if you get a Direct3D error when you try to play your game, you need to change a registry key.  I think it was HKEY_Current_User/Blizzard Entertainment/Diablo II/Video and set Render to 0.
