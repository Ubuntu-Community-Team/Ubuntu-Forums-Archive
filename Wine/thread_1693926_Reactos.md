### Post by cogadh on 2011-02-24
No, at least not yet. ReactOS is actually based on Wine, but where Wine is simply a compatibility layer designed to allow Windows software to run on other OSes, ReactOS is an entire operating system designed to be compatible with Windows software. It is currently in the alpha stages of development, so it is not ready for any kind of regular use and it does not support much yet, most especially hardware drivers. Once it has matured (a lot) more, it may be a viable alternative to Windows or Wine on *nix, but at the moment, it is not much more than a curiosity.
