### Post by cogadh on 2007-11-30
You can change that in winecfg, on the Desktop Integration tab. 

Just out of curiousity, why do you want to change that? The whole point behind setting up Wine like the default is so that Windows apps can interact seamlessly with the Linux environment (i.e. the home folder is the Linux equivalent of "Documents and Settings" in Windows).
