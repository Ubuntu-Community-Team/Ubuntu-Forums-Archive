### Post by 3Miro on 2012-07-14
Most windows viruses will no work through wine. Of those that work, most would do very limited damage to the wine environment, but not the rest. Theoretically, it is possible for a windows virus to hurt Ubuntu through wine.

If you want to play 3D games via wine, it is best to use Nvidia graphics card. Some games work, some games don't. Google "GTA 4 winehq" and see what the official wine database has to say.
