### Post by sikander3786 on 2011-05-28
9.04 isn't supported any longer. You might not be able to access the repositories thats why you can't install wine or any other software.

See this.

[https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases)

So, either upgrade to a newer version or do a clean install with 10.04 or 11.04.

You an access the repositories for 9.04 as mentioned here but there wouldn't be any security updates any longer so it is dangerous to use that.
