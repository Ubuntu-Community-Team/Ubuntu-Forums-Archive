### Post by Maythe on 2010-12-01
Heya starsong,

I as well have been lurking these forums for about a year, and I as well have had the same trouble with Atlantica Online, so I think it's creepily appropriate that I register and help out for once :)

I had several other problems getting Atlantica Online to work, but you seem to be past those already, so I'll jump to the chase.  When you try to log in, does the client ask you to agree with the license agreement each time and then get stuck 'requesting authentication from the login server'?

If this is the case, then try logging in on the AO website.  You can agree to the license agreement via AO's site so that when you go in to the game, you no longer have to do so.  This fixed the problem for me.

Let me know if it doesn't work; I'd be happy to troubleshoot further.
