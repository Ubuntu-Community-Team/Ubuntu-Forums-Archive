### Post by zerubbabel on 2015-11-08
I just installed BibleWorks 9 (rev 4 DVDs) on Ubuntu 15.10 Mate Edition, using a trial version of Crossover 14.1.11, but when I run BibleWorks it appears to hang shortly after the BibleWorks splash window appears. However, if I logout, then momentarily I can see that there was a hidden window asking to register BibleWorks, which, if I could have brought it to the top, I could have responded to it and I suspect BibleWorks would have run just fine. So is there any way I can bring that hidden registration window to the top? There is no title bar on the initial BibleWorks splash window with which to move it out of the way.

### Post by ajgreeny on 2015-11-08

### Post by zerubbabel on 2015-11-08

### Post by zerubbabel on 2015-11-08
Figured it out. Alt-SpaceBar brings up the window menu, followed by Alt-F7 to move the hidden window so that it is visible.
