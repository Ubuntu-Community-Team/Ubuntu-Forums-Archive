Wine itself doesn't display powerpoint slides.

But wine can run various versions of Powerpoint Viewer and the full versions of Powerpoint.  However there are many issues using Powerpoint with wine.  See the Powerpoint [appdb entry]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=13") for your version.  Or the viewer [appdb entry]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=724").
