### Post by groeswenphil on 2009-02-17
Hi group,

I'm a junior school teacher. For years, I've used crossword building software as a way to make my children's lives just a little less dull.
It started years ago when I was still using Windows 97....Using Expert Software Crosswords and More, Version 1 1994.
When I upgraded to XP, this software would no longer work and so I started using Crauswords....a better, but far more complicated program.

Then the other day I found my original Crosswords disk and wondered, "would it work in WINE?"......I'd recently had success with Microsoft Publisher 97.
It does.

For some reason though, the CD installation wouldn't open in Ubuntu.
I copied all of the files into a folder in XP, then copied them all into my Ubuntu partition....then it installed.....not flawlessly....I had to poke it a few times by pressing ENTER.....but now it works.

Ubuntu users......you might want to consider looking through your old 97 disks. There might be some hidden gems there.
Phil
