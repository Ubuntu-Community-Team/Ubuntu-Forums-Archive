Unfortunately, knowing which cursor I'm using is kind of a big part of the game, so it's basically unplayable if it's going to go stay as the default arrow like that. Also, I'm not sure if this is related, but I've noticed that when hovering the mouse over things that should normally be highlighted, they don't highlight.

I was just wondering if anyone else has encountered this problem with other games, and if they've found a solution. I'm using the standard Wine configuration, except that I've set it to emulate Windows 98, and I've unchecked the Virtual Desktop option on the Graphics tab so that it will display on a full screen. I did install Quicktime 3.0 when the game's installer prompted me to.
