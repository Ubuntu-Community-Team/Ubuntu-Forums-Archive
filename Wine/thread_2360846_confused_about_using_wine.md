### Post by mastablasta on 2017-05-09
no you do not need Windows disk. think of Wine as linux pretending to be windows to the software.

the bigger issue is if you could actually run the garmin program in wine. it doesnt' look good with silver rating: [https://appdb.winehq.org/objectManager.php?sClass=application&iId=4871](https://appdb.winehq.org/objectManager.php?sClass=application&iId=4871)

but this one has gold: [https://appdb.winehq.org/objectManager.php?sClass=application&iId=227](https://appdb.winehq.org/objectManager.php?sClass=application&iId=227)

so i guess depends on the application you will be using. check it out on wine appdb before you lose too much time.

if it doesn't work in wine, you need virtualbox. in case of virtualbox or other virtual machine software you do need windows cd and you basically perform a full install on a virtual PC.
