### Post by mikedep333 on 2008-08-16
Hey,

Just like Mac OS X, Ubuntu is not capable of directly running Windows Programs, such as games.

Using Wine, or commercial versions of it such as crossover games, Ubuntu can run many windows programs, including 3D games.

Steam is well supported by wine. In fact, it is probably a good thing that you used steam because the copy protection (complete access restrictions) on the DVD version of Medieval 2 would be a huge obstacle.

However, according to wine's application compatibility database, medieval 2 does not run well on wine. This is because wine is an incomplete implementation of the constantly changing Windows environment. Most likely in a year or two, medieval 2 will run better on wine.

You can see the Wine AppDB entry here:
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=4136](http://appdb.winehq.org/objectManager.php?sClass=application&iId=4136)

In another note, welcome.
Also, if you would like to try Ubuntu out without complete jumping ship, or simply want to easily be able to switch back and forth between ubuntu and windows so you can play Medieval 2 when you want, I would recommend Wubi, the ubuntu installer for windows.
[http://wubi-installer.org/](http://wubi-installer.org/)
