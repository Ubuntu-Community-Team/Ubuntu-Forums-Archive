### Post by 5dolla on 2009-12-14
Heyo im writing this guide For people who want to play MW2 in Ubuntu.
first my rig. 
AMD Phenomn II 720
Nividia 9800gt 512
DDR 3 1600 4gb
Ubuntu Karmic 9.10

1. I used play on linux. The wine version I chose in POL is 
1.1.34. This is important. If you chose 1.1.33  you will encounter
a steam error. you may still use 1.1.33 if you have a patch(please
see the wine appdb ) note* I used 1.1.34 for both steam and mw2.
You may choose to run mw2 to from steam. I am leaning against this 
because a direct link to the MW2 mp exe allows for editing that specific \
wine registry in POL.

2. disable all the special setting in game like shadows and AA. You may need 
to play with the GFX settintgs to avoid weird graphic distortions. i current have 
all the extras turned off and getting 50-60fps with no  graphical errors.

3. disable compiz. this will give you a substantial fps boost .

so far this is what worked for me. if you have any questions please post 
back. I didnt have to do to much to get this running which is a victory for linux 
gamming. I didnt even have edit the wine registry. This is also a concern of
mine. So if any one wants to give me any pointes on wine registry tweeks 
for this game please let us know by posting.

Have fun guys!
