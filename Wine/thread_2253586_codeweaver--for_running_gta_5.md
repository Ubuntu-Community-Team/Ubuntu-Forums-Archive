### Post by Mark Phelps on 2014-11-21
The problem with graphics and Windows games is that, in many cases, the Linux video drivers simply aren't up to the performance level of the Windows video drivers.  Case in point is that AMD dropped Linux support for its HD 2x/3x/4x-series cards last Summer but still supports Windows drivers for the same cards.

Poor frame rates are often the result of software rendering and for modern Windows games, you need hardware accelerated rendering.

Basically, the experience of trying to do Windows gaming in Linux is generally disappointing.
