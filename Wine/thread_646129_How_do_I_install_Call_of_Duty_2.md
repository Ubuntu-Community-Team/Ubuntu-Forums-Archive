Wine supports directX up through 8.1, and which supports source games rather well. I don't know about the CoD2 engine specifically but it probably has DX8.1 support which means it should run under wine. Cedega is based off of wine, at one point in time, a couple years back, cedega had better DX support, but it's pretty much just wine rebranded these days, but their marketing department has done a good job of keeping up the appearance that cedega is better than wine for gaming.

As for installing CoD try this, stick the CD in. Pop open the run dialog with ALT+F2 and enter winefile to open the wine file browser. Open the CD and try double clicking on the installer.
