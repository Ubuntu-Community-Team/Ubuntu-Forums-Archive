### Post by Moloch85 on 2011-02-01
I suggest you to get it from the ubuntu software center, or from synaptic, or using apt-get (open a terminal and digit *sudo apt-get install wine*)

I usually prefer this way to install all the software, because the main advantage of a package management system like apt (on which are based also the graphical frontends like the ubuntu software center or synaptic) is a feature usually called "automatic dependency resolution"
