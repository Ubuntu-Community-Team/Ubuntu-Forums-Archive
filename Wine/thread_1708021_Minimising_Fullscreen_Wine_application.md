### Post by RDT1 on 2011-03-16
Well i thought id pass on the easiest way to minimise a full screen program in WINE when some do not allow with the default Alt+F9 combo. 

I'm running Wine 1.3.15 at present should work with the stable 1.2.2 found in the software centre.  or [http://www.winehq.org/](http://www.winehq.org/)

Go to Applictaions > Wine > Configure Wine > Graphics

X Allow DirectX apps to stop the mouse leaving their window
X Emulate a Virtual Desktop
 ** Enter your screen size here.

Make sure the other two boxes are un-ticked. 

Now boot up your wine program (for me it was Warcraft III) that was previously running full screen and now you can Alt+F9 to minimise! 

Also to change this default shortcut go to System > Preferences > Keyboard Shortcuts
and look under Window management. 

Hope that helps someone as i couldnt find a solution when i had this problem.
