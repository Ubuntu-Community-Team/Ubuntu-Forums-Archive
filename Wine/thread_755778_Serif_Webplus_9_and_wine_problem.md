### Post by MonkeyPants on 2010-06-16
No problem :).

I found a tutorial on YouTube for setting Windows up with Virtual Box (I find videos easier to follow than text): [http://www.youtube.com/watch?v=e4EYYosQo_I](http://www.youtube.com/watch?v=e4EYYosQo_I).

I think the basic jist is: -

1. Download/install VBox
2. Create a Windows Virtual Machine.
3. Put XP disc in tray and let it boot newly created VM. Then allow Windows to boot into installer on the VM and treat it as you would do a normal Windows installer.
4. Have fun :).

I haven't tested this myself, so I can't really give you much insight as to what kind of problems you might encounter, but there is a whole forum dedicated to the subject: [http://ubuntuforums.org/forumdisplay.php?f=308](http://ubuntuforums.org/forumdisplay.php?f=308)
