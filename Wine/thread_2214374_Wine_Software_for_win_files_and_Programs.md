### Post by Mark Phelps on 2014-04-01
Wine has nothing to do with the Sony.  You can not install Windows drivers with Wine, so the Sony will need to use Linux drivers -- and that is the same situation with or without Wine installed.

As to the program, go to the link and do a search for that program.  If it is not found, or if the rating is lower than Gold, you're wasting your time installing Wine: [http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true")
