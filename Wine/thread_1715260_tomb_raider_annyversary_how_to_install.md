### Post by cogadh on 2011-03-26
No, you usually just install games in Linux using Wine. Wine is specifically designed to allow Windows executables to run on Linux, including installer executables. Just find the installation executable, usually called something simple like "setup.exe", then open a terminal and run it with Wine:
```
cd /path/to/setup/executable
wine setup.exe
```
