### Post by Sugi on 2009-06-27
First off, I would like to say I have only tried FF7 on Windows platform.  FF7 is not supported with Windows XP natively, but there are a lot of hacks out there to get XP to work with FF7.  So keep this is mind while trying to get FF7 to work with Wine.  Maybe trying to set wine to run on Windows ME or 98 or something down those lines, but keep this is mind as well, just because the game only works on IE: Windows 98 doesn't mean it will work if you set wine to run in that environment of Windows 98.  Try a No-CD patch as well this could fix some issues with game.

I hope this information is a little bit helpful,
Sugi

PS: if you haven't already check out the appdb's website for FF7. [http://appdb.winehq.org/objectManager.php?sClass=version&iId=183](http://appdb.winehq.org/objectManager.php?sClass=version&iId=183)
