### Post by jacksaff on 2008-09-24
Wine doesn't support the checking software that the game uses to see if the disk is present.
You need to download the nocd hacked versions of the .exe files. Wine appdb explains what to do with them and you will find them easy enough online.
It's possibly illegal (as far as I'm aware its never been tested in court) to do this though as you have a valid, paid-for license I doubt any court in the world would convict you for it. As a bonus you will be able to run the game without the disk in the drive which saves the hassle of having to dig it out when you want to play...
