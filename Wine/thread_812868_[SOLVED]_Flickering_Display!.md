### Post by talking Llama on 2008-05-30
This seemed like the most relevant place to post this so: Whenever I run Wine with a game running, such as WoW, or if I try to run the MilkDrop plugin thing Project M I always get the same display where the screen displays the game or visualization but flickers a bit then if I run any other program it goes mental and starts to flicker more. Not sure why it does this, any help correcting this would be great. I'm using the latest releases of both programs.

---

### Post by Absurdiverse on 2008-05-31
Make sure you have Compiz turned off.
System > Preferences > Appearance > Visual Effects.
