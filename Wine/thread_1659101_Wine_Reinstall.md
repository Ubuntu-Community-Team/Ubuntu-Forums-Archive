### Post by Tweak42 on 2011-01-04
Have you run winecfg from the command line or "Configure Wine" from the gui menu?  If .wine directory doesn't exist it should create a new one with a base set of files.

---

### Post by foxmulder881 on 2011-01-04
As above, when configuring for the first time WINE will (re)create the .wine directory. ;-)
