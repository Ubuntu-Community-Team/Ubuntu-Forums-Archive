### Post by bobwyajnr on 2011-05-31
Imagine you've got a SatNav that tells you to drive off a cliff every so often... Wine is like that!! :popcorn:

Seriously though Wine is good at running applications that are relatively standalone (not too many hooks into the Windows API or dotnet evilness etc.) You are best advised to browse on the [Wine AppDB]("http://appdb.winehq.org/") for the Windows application you are trying to run. That will save you repeating someone else's heartache! :P

I would definitely try the native Linux Skype client before resorting to running the Windows client via Wine.
