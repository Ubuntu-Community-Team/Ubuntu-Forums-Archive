### Post by jespdj on 2009-01-07
Adobe CS4 does not (yet) work on Wine, as far as I know. It has taken a long time to [get CS3 to work](http://www.jonramvi.com/wine-status-report-photoshop-cs3/) on Wine.

See the [Wine Application Database](http://appdb.winehq.org/) for information about which Windows applications work on Wine.

*edit*: Looks like [Photoshop CS4 should work](http://appdb.winehq.org/objectManager.php?sClass=version&iId=14318) but you might need the newest version of Wine.
