### Post by Astinsan on 2009-06-01
If you have issues selecting stuff on the screen turn on wines desktop mode

goto the wine menu item  in your gnome menu




Click configure wine in that menu



Default settings should be selected


Click the graphics tab

check emulate virtual desktop

800x600 or what ever standard size will be fine for this


click ok


launch your installer again.


After you get this all done undo the settings you just set in wine.


You will find that some software will work better in this virtual desktop. You have the option of setting it by default for any application. 


Hope this helps

### Post by Terrain on 2009-06-01
Another option is to just copy the entire World of Warcraft folder from someone's Windows install to the Program files directory in Wine. WoW doesn't actually make any notable use of registry entries, so you can take that folder anywhere and play the game. 

The folder itself is a ~17GB monstrosity, but if you have a friend that can bring over their computer you can network them together and just grab the thing, and it will only take about half an hour depending on how you have them networked. Additional perk is that with this method, the game should already have been fully patched so no waiting on downloads.
