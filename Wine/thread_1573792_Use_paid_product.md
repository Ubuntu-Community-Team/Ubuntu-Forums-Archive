### Post by beastrace91 on 2010-09-13
> **atriks said:**
> Ok thanks I will try the free product first, but I want to know what is the actully diffrence between paid and free?

Ok let me take a moment to give you some information from someone who knows what they are talking about. The first difference you will notice between vanilla Wine and Paid for application like Cedega and Crossover is the fact that the paid for applications provide a GUI for installing/configuring your Windows applications.

Beyond this Cedega uses it's own Wine core, meaning it does run a handful of applications that fail to run under Wine (See: Crysis) and it runs other applications that run under Wine at a much higher FPS (See: UT3 engine applications and Bioshock). The downfall to Cedega is that at this point it is basically a dead product, they only update to keep things working that already work (AKA Steam) and they haven't see a new release in some time that adds anything.

Crossover on the other hand uses the same Wine core, but adds a couple closed source things of their own (such as their HTML engine) that helps things like Steam, Firefox, and IE run much better under Wine. Beyond this, the codeweavers devs are the main contributor to Wine, meaning when you buy crossover you are supporting Wine development.

@The comment regarding buying software for a free OS - Unless you are running FOSS applications under Wine (in which case what is the point?) Odds are you bought the Windows application - so if you are going to support development on a platform you no longer use, might as well support it on your platform of choice as well right?

### Post by basotl on 2010-09-13
As an FYI **Cedega** has had a UI since when it was called WineX and you could still get the source. If I remember correctly that was 2004. The UI has been redesigned a few times since then.

For quite some time **Cedega** had several little things it could do better than Wine but now that has been for the most part eliminated.

**CrossOver** is a pretty cool product in that they contribute back to Wine. While they have a gaming product the area area you will see the most polish is office applications. I found several applications I was required to use for work... worked much better with **CrossOver**.

I have used all versions of CrossOver, Cedega, PlayOn and Wine recently. I would actually advise staying away from Cedega unless you research that a game you like still has the best performance on Cedega. I suggest CrossOver if you have some Windows Office apps you are required to use and don't want to use a VM. Wine is pretty good to try anything on first.
