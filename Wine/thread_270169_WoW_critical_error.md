### Post by alfie93 on 2007-11-20
I had this same problem when i done a system restore. What i did as i couldn't just reinstall it was to go to C:\Program Files\WorldOfWarcraft (or whatever the folder path) and i went in the 'data folder and changed the PATCH.MPQ to PATCH.MPY (or anything similar) then run repair.exe and it will say there is a critical error. But then when you load up WoW the error 121 message will not appear and you can login. However you may have download quite a few patches before you can play it again.(but the are already saved).
