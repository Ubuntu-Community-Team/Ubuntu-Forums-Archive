### Post by rocky5051 on 2009-11-04
The message is irrelevant to the crash you experienced with the Office 07 installation due to the fact the crash that killed the application had already occurred before that.

However, it shouldn't crash like it did. The appDB says it installs without a hitch on wine versions greater than 1.1.23.

My suggestion: make sure you have the most recent wine release (1.1.32) installed (open a terminal and type in "wine --version" without the quotes to determine what version you have). If you do not, follow the instructions here to install Wine:

[http://www.winehq.org/download/deb](http://www.winehq.org/download/deb)

If you do have the most recent version installed and it still crashes, delete and recreate your wineprefix:

```
rm -rf $HOME/.wine && wineprefixcreate && wineboot -esr
```

After that, install Microsoft Office 07 again and it should install without issue.
