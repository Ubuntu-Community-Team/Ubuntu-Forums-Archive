### Post by hikaricore on 2008-01-20
The terminal method is probably the best choice for new users as this will give you additional info if something doesn't go properly.
This method also cuts down on pointless messages such as "y0 my gam3 dont werk!!"  ^_^  :p  (we see these way too often)

Be sure to check the [WINE Application Database]("http://appdb.winehq.org") to be sure if your title runs at all under WINE, and for special installation instructions.
