### Post by cogadh on 2007-11-29
MapleStory doesn't work with Wine at all. It's not a fault in Wine, it is something about the MapleStory client. Your only options for playing the game are probably either dual booting Windows and Linux or trying to run the game using VirtualBox, VMWare or some other virtual machine system. However, if the game uses any 3D accelerated graphics, even a virtual machine won't work (virtual machines only do basic VGA/VESA graphics), so you will be stuck with dual booting as your only option.

If you want to try the virtual machine route, I personally recommend VirtualBox. I have found it very easy to work with and has some really good documentation and community support:
[http://www.virtualbox.org/](http://www.virtualbox.org/)
