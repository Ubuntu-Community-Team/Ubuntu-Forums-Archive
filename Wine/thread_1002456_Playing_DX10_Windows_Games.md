### Post by Skripka on 2008-12-08
If you want to play Crysis, you:

a)  Need ALL the computer power you can get.  Powerful CPU, and a nuclear-powered GPU.

and

b) Need a windows install.



There really aren't any ways around either.  Use Linux for getting work done--and WinXP for wasting time.  Crysis in DirectX 9 mode looks great under XP on my rig (see below)--there are only a few settings I cannot set to "high" or "enthusiast" (namely antialiasing) at 1680X1050.


DirectX 10 is a component of Windows Vista (not a separate software) and as such it is unlikely it will come to Wine anytime soon.
