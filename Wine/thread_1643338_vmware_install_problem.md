### Post by cwwilson721 on 2010-12-11
You don't run VMware in wine. It is a separate, true "virtual machine", not a Windowsa app.

Make sure it is executable, and from the command prompt, run it. It will install, then you can start it from "Applications > System Tools". Choose "Create a New Virtual Machine", choose the options you want, and start the Operating System install (for example, put your Windows XP cd in your drive, and install XP just like normal)

Have fun
