### Post by NightMKoder on 2009-07-06
Not the best idea - not a lot of people working on wine care about remove working (for reasons that, well, we want install working).

If you dont want any windows programs, you can delete .wine in your home directory. ctrl+h unhides it in the file manager, or ```
rm -rf ~/.wine
``` does it from console.
