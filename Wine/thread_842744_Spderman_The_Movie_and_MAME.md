I wouldn't bother with PlayOnLinux, I've honestly never had it work any better than just using Wine on its own (in fact, some games that worked fine with plain Wine did not work at all with POL). Installing apps with Wine is as simple as putting the install disk in the drive, opening a terminal and typing "wine /media/cdrom/setup.exe" or whatever the install executable is named. After that, you just need to follow the prompts like you would in Windows.

Unfortunately, I don't know the first thing about GXMame, so I can't really help with that.
