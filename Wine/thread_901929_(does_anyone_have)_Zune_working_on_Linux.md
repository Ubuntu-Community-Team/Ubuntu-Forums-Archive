### Post by deputy on 2008-12-17
I have been able to successfully install and use the Zune software using a virtual machine with Xp and VMWare Server 2.0. I use an 80gb Zune and am using Zune Software 3.0 and everything works. I can sync, get updated music, vids, pics, use marketplace etc. The only thing I haven't tested is the wireless sync. This is the only way I have been able to use a Zune on linux. I've been testing this out for about 2 weeks now and so far so good!
