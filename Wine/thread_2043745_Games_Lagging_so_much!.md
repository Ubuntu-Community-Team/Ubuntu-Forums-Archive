### Post by synaptix on 2012-08-19
As stated, Wine is a hit and miss type deal. The development team tries its best to not break previously working applications from version to version.

You may consider trying PlayOnLinux, which provides a simple and organized frontend GUI for Wine, but it also selects the best Wine version for whatever you are installing provided by tests submitted by users to WineHQ AppDB. Not only that, but it will install any additional software that is needed to run the application.

[http://www.playonlinux.com/en/](http://www.playonlinux.com/en/)
