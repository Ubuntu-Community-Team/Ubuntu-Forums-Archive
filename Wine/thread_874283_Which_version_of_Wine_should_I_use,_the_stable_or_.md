### Post by ajackson on 2008-07-30
The development strand gets updated every fortnight and regressions are a possibility. The stable strand gets update when they feel the need but only secure and proven stable updates go into it so the chance of regression is a lot lower (possible zero but you can never truly know).

### Post by jacksaff on 2008-07-30
As with most developing projects the dev version is better 99% of the time - fewer bugs, greater functionality, better optimised and faster. Unfortunately however, the 1% can be painful enough to for many to keep to the stable versions.
I run wine dev versions and if there's a regression for some app I use I either wait two weeks when it's nearly always fixed or downgrade if the app is one I can't do without.
