### Post by lykeion on 2010-11-14
WineHQ is the place to get info on running windows applications with wine. Here's the entry for Cubase 5.1: [http://appdb.winehq.org/objectManager.php?sClass=version&iId=21634](http://appdb.winehq.org/objectManager.php?sClass=version&iId=21634)

This version of Cubase doesn't run very good in wine. I'd suggest you to run Cubase in Windows. It's what its written for.

But you should definately also look up some native linux alternatives to Cubase, like [Ardour](http://www.ardour.org), [Rosegarden](http://www.rosegardenmusic.com), [LMMS](http://lmms.sourceforge.net)
