### Post by Rhubarb on 2008-08-11
There are many free native linux FPS games available to choose from.

**Sauerbraten**
(in the repos, a newer version is available [here]("http://www.sauerbraten.org/"))

**Urban Terror**
Get it [here]("http://www.urbanterror.net/news.php")

**Assault Cube**
Get it [here]("http://assault.cubers.net/")

**World of Padman**
Get it [here]("http://www.worldofpadman.com/")

**Quake 3**
If you have quake3 on CD, it can run in Ubuntu with the help of ioquake3 (a native q3 engine for Linux), get it [here]("http://ioquake3.org/")

**Postal 2**
Free multiplayer online / LAN game, get it [here]("http://icculus.org/news/news.php?id=4419")

**Nexuiz** (see post above)

There are also native Linux clients for Quake 4, Quake Wars, Unreal Tournament 2004, Doom 3, Quake 2, Quake 1, Doom 1 / 2.
Halflife 2 will run in wine.
