You're really pushing you luck to do any gaming on that laptop (since it is quite underspec'd).

Does it feature Intel integrated graphics per chance (which are not great to start with and are also badly supported under Linux)?

512Mb of RAM is also a _very big ask_ to run any big programs (like OpenGL/DirectX games for instance).

If I was you I would check out that native Linux OpenGL games will actually run first (e.g. Alien Arena, World of Padman)... I think you might be disappointed...

You have to remember Wine is just a *compatibility layer* - it doesn't magic better hardware resources or graphics drivers out of thin air!
