### Post by 0ddity on 2010-11-27
Just wanted to write a blurb about my experience with running PKR, the 3D online poker game with Wine. Install/updates works flawless. If you run in windowed mode, then the button hotspots are about an inch below the actual buttons, but when run in full screen, the game is gold. Works absolutely perfect, now I'm one step closer to totally ditching Windows 7. If you like poker, this is definitely the one to get, on Windows, Mac OR Linux. :KS
