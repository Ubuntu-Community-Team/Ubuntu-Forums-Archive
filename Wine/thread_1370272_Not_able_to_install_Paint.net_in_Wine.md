### Post by Mark Phelps on 2010-01-02
Wine is not the "magic cure" that many claim it to be.  The .Net installs are infamous for nothing working since v3.  Anytime you're going to use Wine, you need to check the WineHQ site to see the ratings first.

The page below is for .Net:

[http://http://appdb.winehq.org/objectManager.php?sClass=application&iId=2586]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=2586") 

My own advice is, if you're going to rely on MS Windows, do a dual-boot and use the Real MS Windows, instead of trying to kludge results via Wine.
