### Post by bobince on 2009-08-11
Well, like the error says, the game has copy protection. This:

- typically does low-level hardware access that virtualisation/emulation software can't support
- often deliberately sabotages the game when it detects anything unusual about the runtime environment, in an attempt to defeat debuggers that also often defeats WINE

Of course it's completely pointless since the game itself is still cracked and uploaded everywhere, but try telling game publishers that...

If you want it to run, you'll need to find a way to get rid of the copy-protection. This may or may not be legal depending on where you live.
