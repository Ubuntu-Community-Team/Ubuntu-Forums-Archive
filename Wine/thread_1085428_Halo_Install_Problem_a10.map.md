### Post by cogadh on 2009-03-03
Have you already followed the installation how-to on Wine's AppDB page for the game:
[http://appdb.winehq.org/objectManager.php?sClass=version&iId=2720](http://appdb.winehq.org/objectManager.php?sClass=version&iId=2720)

If you are already following that, are you getting any error messages on that file, such as "CRC check failed" or anything like that? If so, it could indicate a problem with the disk itself and not Wine or Ubuntu. 

If you are getting no error messages at all and you are following that how-to, then it might help to run the install through the terminal and post Wine's terminal output. It may contain some error messages from Wine itself that can explain the problem.
