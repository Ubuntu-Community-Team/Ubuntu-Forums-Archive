### Post by Mark Phelps on 2014-02-03
Wine is a "hack" in which some Windows DLL files (Dynaminc Link Libraries) were rewritten to replace Windows OS system  calls with Linux OS system calls.  To the degree that a Window app/game relies only on those DLLs, Wine works well. When the Windows game/app uses other middleware like .Net and/or Silverlight, the game/app works poorly or not at all.

BEFORE you install Wine, it's best to go to their website, use the application compatibility tool they provide, and look up the name and version of the app/game you want to use.  Anything that is not listed, or rated lower than Gold, is going to be a waste of your time trying to get it working in Wine.
