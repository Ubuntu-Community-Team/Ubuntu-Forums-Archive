### Post by myromance123 on 2013-05-06
I have no idea about SimCity 5, but I can personally play SimCity 4 Deluxe and 3000 using Wine.

The question is, is SimCity 5 DirectX11 only? Or does it also support DirectX9? If it supports DX9, then chances are you'll get it up and running no problem in Wine.

Do take note, EA's Origin really hates Wine. They've put code into Origin so that if you're not running on Windows, then they'll warn you saying they don't support this OS. I know because I have BF3 and have been trying to get it working for a while. Origin works in the latest Wine (1.5.29 at the time of this writing), so at least you're good there.

WineHQ's rating is pretty bad though, they give it Garbage rating: [http://appdb.winehq.org/objectManager.php?sClass=version&iId=27821]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=27821")
