### Post by Claus7 on 2011-04-22
Hello,

back then the things might have been more hard to install, yet today things have advanced a little. 

Since this thread is popping up a lot about how someone can install and play colin mcrae rally on ubuntu, I would give my thoughts.

There are successful attempts both on wine and crossover linux. I guess that the best would be the crossover one, as it can simulate windows reboot, which is needed for a successful installation of the game. Also windows media player 9 is needed as well for that game.

Since some jerky graphics and deterioration of your screen might arise, the option of resizing windows via wine (inside crossover 9.x) might solve any remaining problems which impediment your viewing of the game on your ubu box, which is mentioned here:
[http://www.codeweavers.com/support/tickets/browse/?ticket_id=828249;list=6;ticket_level=0](http://www.codeweavers.com/support/tickets/browse/?ticket_id=828249;list=6;ticket_level=0)
