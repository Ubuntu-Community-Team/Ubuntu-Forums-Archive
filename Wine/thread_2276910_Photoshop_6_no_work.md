### Post by Bucky Ball on 2015-05-07
... and from the link v3.xx posted:
> 
 I have tested Photoshop 6.0 on many versions of Wine (from 1.2 to 1.7) and this problem manifest in all Wine-versions. Although it is possible to use this application under Wine for minor and quick editing, _**eventually the application will always stop functioning properly. Any unsaved work will be lost. This problem renders Photoshop 6.0 under Wine pretty much useless for serious users.**_ 

If you need to use Photoshop everyday I'd suggest sticking with Windows, dual-booting or running Windows in a VM (not sure how well PS would go in the latter configuration, though).

... or use Gimp as an alternative.
