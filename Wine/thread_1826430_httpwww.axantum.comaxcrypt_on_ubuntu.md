### Post by ubto66 on 2012-01-01
I found this solution:

A install wine on ubuntu
B download axcrypt2go.exe from [http://www.axantum.com/axcrypt/Downloads.html](http://www.axantum.com/axcrypt/Downloads.html)
C right click file, and make it executable 
D open file with wine

Axcrypt2go supports both encrypting and decrypting.
