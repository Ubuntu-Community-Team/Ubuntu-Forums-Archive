### Post by cogadh on 2007-11-17
Did you follow the how-to steps on that AppDB page? They are specific to Wine 0.9.47, but all those steps should still apply to 0.9.49.

Also, you never need to install DirectX in Wine, it already has its own DX implementation and trying to install the real DX will just break it so that no DX games will run in Wine.
