### Post by cogadh on 2008-05-24
Those test results are for a very old version of Wine. A lot has changed since then, so it is really not surprising that it doesn't work any more. I would suggest you sign up for AppDB and submit your test results and a bug report, otherwise, if this is a regression problem, it will never get fixed.

In the meantime, run the install from the terminal and post back with any error messages that Wine outputs to the terminal. Those error messages may explain what is going on.
