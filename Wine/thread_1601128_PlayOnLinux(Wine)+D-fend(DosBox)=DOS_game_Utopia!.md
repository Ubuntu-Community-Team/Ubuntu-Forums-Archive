### Post by sgtslwilson on 2010-10-19
Some of you may think this is crazy but here is what I did. I dual boot XP and Ubuntu. The only reason I have XP is for a program I have to use at work and for games. I have been turned on about DosBox since I discovered it on [www.abandonia.com](www.abandonia.com). They have every old DOS game I ever played and most of it is for free. I could successfully load some games on Linux and XP using Dosbox. Wine has always seemed hard for me to configure. I don't know enough about operating systems and how they work to successfully do this on most programs.

So, I booted into Ubuntu and installed PlayOnLinux, a wine frontend, and installed D-Fend Reloaded, which is a Windows DOS emulator, using PlayonLinux. When that was installed I used it to install all the old DOS games I play and love. Everything worked like a charm! I could probably figure out all the presets for every DOS game I have individually but D-Fend reloaded does this automatically for me. Literally 3 clicks to go from a zipped DOS game to playing it.

For those of you who aren't geeky enough to figure out how to make Wine OR DOSbox work on every old DOS program you want to use, take it from me, this works!

Here is the link to D-Fend reloaded:

[http://dfendreloaded.sourceforge.net/Download.html](http://dfendreloaded.sourceforge.net/Download.html)

You can find PlayOnLinux in the Ubuntu repo.

Keep in mind that it uses the WINDOWS verson of DosBox .74. It works great even in emulation. Also, don't forget to set up the sound driver in Wine using the PlayOnLinux "configurator" GUI. You only have to do this once and it's set for every game. 

You do not have to download Dosbox or Wine seperately. Wine is embedded in PlayonLinux and DOSbox is embeded in D-Fend Reloaded. 

I hope this helped someone. I'm going back to my Sierra game/Duke Nukem 3D paradise now.
