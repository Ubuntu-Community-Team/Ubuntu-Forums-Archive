### Post by kahlil88 on 2008-08-10
First install it (**sudo apt-get install wine**) and then any Windows executable (.exe file) should be able to run just by double-clicking. Certain programs will run poorly or not at all (like anything Apple or Microsoft), but for the most part it seems to work really nicely.
