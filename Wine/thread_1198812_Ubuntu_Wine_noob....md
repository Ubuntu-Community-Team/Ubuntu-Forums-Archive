### Post by ad_267 on 2009-06-28
According to the Wine application database MyspaceIM doesn't work well with Wine: [http://appdb.winehq.org/objectManager.php?sClass=version&iId=5007](http://appdb.winehq.org/objectManager.php?sClass=version&iId=5007)

But from this page: [http://developer.pidgin.im/wiki/MySpaceIM](http://developer.pidgin.im/wiki/MySpaceIM), it looks like Pidgin should be able to support the MyspaceIM protocol by default in Ubuntu, the Pidgin version in Ubuntu 9.04 is 2.5.5.

Edit: Yup just checked and Pidgin has the option to add a MySpace account, so there's need to install MyspaceIM through Wine.
