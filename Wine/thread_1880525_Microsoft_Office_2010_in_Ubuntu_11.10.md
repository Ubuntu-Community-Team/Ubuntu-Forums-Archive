### Post by Tweak42 on 2011-11-14
Microsoft Office is a suite of individual applications, so each program will react differently to wine.  I would start at the [Microsoft Office wine appdb]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=31") entry and look up the specific office program you are trying to run to see if there are any tips on getting it working.
