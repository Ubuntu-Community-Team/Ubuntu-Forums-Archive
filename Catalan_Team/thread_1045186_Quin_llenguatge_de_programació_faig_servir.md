### Post by lpargemi on 2009-01-21
Hola

Si vols un entorn gràfic relativament fàcil, es pot fer servir l'entorn apache - mysql - php

Amb això podràs consultar i modificar les dades des d'un navegador d'internet, si tens una petita xarxa tenir-ho centralitzat en un servidor, o si vols penjar-ho d'internet i fer-ho servir des de qualsevol banda (amb les precaucions i seguretats d'accés necessàries es clar).

El llenguatge de programació php segur que et serà molt senzill de fer anar coneixent C, i els codis html (i css per les pàgines d'estil) estan molt documentats per arreu. (per exemple a [http://www.w3schools.com]("http://www.w3schools.com")) Mysql es treballa amb sentències standard SQL, que son un pèl diferents de l'access (de fet per variar és access que és un pèl diferent del standard...)
