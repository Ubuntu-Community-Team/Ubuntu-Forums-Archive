Per a fer pràctiques amb el Windows 2003 server (coses de classe), voldria fer una xarxa interna....Ho he intentat ja un parell de vegades fent coses estranyes, però he vist que amb les opcions del Virtualbox podria fer-se.....

He [intentat fer açò]("http://blog.unlugarenelmundo.es/2008/11/22/hacer-visible-en-nuestra-red-una-maquina-virtual-con-virtualbox-o-comunicar-entre-si-dos-de-estas/comment-page-1/"), també [URL="http://samiux.wordpress.com/2007/07/11/bridge-network-interface-on-virtualbox/"]vaig provar de fer açò altre,

Tinc un domini creat al Windows 2003 (sóc controlador de domini i eixes coses) puc fer una xarxa amb el Virtualbox amb eixe domini? Es complicat, la veritat...

A les configuracions del Virtualbox, en la xarxa, el que estava fent es deixant un adaptador de xarxa amb connexió a internet, i afegint altre amb "Xarxa Interna", i posant-li un nom a eixa xarxa (el mateix que el nom del domini) però la veritat....no se com fer-ho! es una mica complicat!

Si algú te una xarxa en mode virtual i em pot explicar com l'ha creada (i si pot ser amb un Windows 2003 server actuant com a Controlador de domini)
seria per a mi una gran ajuda!!! :lolflag:
