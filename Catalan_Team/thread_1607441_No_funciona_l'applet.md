### Post by wgarcia on 2010-11-01
Si entres en una terminal (Aplicacions -> Terminal) :

sudo update-java-alternatives -l

Et mostrarà quines versions de java tens instal·lades. Engantxa aquí el resultat a veure si algú et pot ajudar.
