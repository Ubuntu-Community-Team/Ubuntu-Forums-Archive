### Post by marcoil on 2007-07-25
Jo per convertir entre diferents formats de so empr el SoundConverter, una eina gràfica que s'integra molt bé en l'escriptori d'Ubuntu. En general, si pots escoltar els arxius te'ls podrà convertir. Per instal·lar-la basta

```
sudo apt-get install soundconverter
```

Esper que te sigui d'ajut.
