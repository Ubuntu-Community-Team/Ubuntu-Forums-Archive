### Post by Xispa on 2009-05-01
Per a l'intercanviador ràpid d'usuaris, jo hi afegiria la instrucció:
```
gdmflexiserver
```

i per al Tomboy, si vols obrir-lo
```
tomboy --search
```
i si vols crear una nota nova
```
tomboy --new-note
```
