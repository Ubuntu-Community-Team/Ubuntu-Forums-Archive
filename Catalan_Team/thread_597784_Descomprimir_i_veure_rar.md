Amb Linux tens eines per buscar coses com ara fitxers o directoris.

"locate" (es al paquet findutils), aixo et fa una base de dades de tota la teva maquina i s'actualitza amb el cron cada dia
Exemple:
#locate DEADJOE (et busca a tota la teva maquina aquest fitxer)
(ull que aixo busca a una base de dades i aquesta s'actualitza per defecte, una vegada al dia, sino has de fer updatedb per actualitzar-la)

"find" una eina tremendament potentisima, per fer cerques
Exemple1:
# find -name "*.doc" (et buscara els fitxers .doc, si fas servir comodins has de posar el string entre cometes)
Exemple2: (en combinacio amb exec)
# find -type d -exec chmod 755 {} \; (aixo et busca nomes directoris i els hi cambia els permisos a 755)

Desde les X, amb el konqueror per exemple, tens una icona d'una carpeta amb una lupa que es per buscar fitxers.
Si fas servir el nautilus, tambe tens la opcio de buscar.

Per desar un fitxer, amb el firefox, pots ajustar com vols que es comporti. amb les Preferencies, a la pestanya "main" hi ha la configuracio dels "Downloads" o descarregues, pots seleccionar que et pregunti on deixar els fitxers.
Tambe pots anar a Tools --> Downloads (o fer CTRL+Y) i s'et obrira una finestra amb l'estat de les descarregues, aqui tambe podras veure on les deixa.
