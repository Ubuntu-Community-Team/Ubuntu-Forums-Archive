Pots provar un programa anomenat kdocker. Funciona tant en gnome, kde o Xfce.

Passos a seguir:

1.- Instales el programa: apt-get install kdocker
2.- A les Sesions: La manera de fer el "script" és: kdocker programa

La web del programa es: [http://kdocker.sourceforge.net/](http://kdocker.sourceforge.net/)

També n'hi ha un altre anomenat alltray, és una altra alternativa.
