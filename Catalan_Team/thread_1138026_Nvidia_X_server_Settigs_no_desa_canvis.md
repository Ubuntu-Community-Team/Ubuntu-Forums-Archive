### Post by jerec on 2009-04-26
Una vegada fetes les configuracions, les has de guardar al xorg.conf.
Prem el boto "Save to X Configuration File" desde el menu "X Server display Configuration"

En cas que no et deixi per problemes de permisos, ejecuta el programa des de una consola amb "sudo"
# sudo nvidia-settings

el fitxer .nvidia-settings.rc son paràmetres del programa de configuració, el que fa a ultima instància es escriure el "xorg.conf"
