### Post by pespin on 2007-09-09
Tens Ubuntu instal·lat ja?

Llavors has d'instal·lar els paquets de traducció al català:

```
sudo aptitude install language-pack-ca language-pack-ca-base language-pack-gnome-ca language-pack-gnome-ca-base mozilla-firefox-locale-ca
```

També pots configurar-ho a la secció d'idiomes del menú de l'escriptori (no el tinc localitzat ara al gnome, però porta unes banderes al costat).

Des de el web d'ubuntu descarregat la darrera versió: Feisty Fawn (7.04)
