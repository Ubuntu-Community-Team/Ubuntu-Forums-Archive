### Post by papapep on 2009-05-24
Podries provar primer a llistar les regles de l'iptables:

```
sudo iptables -L
```

Enganxa-les aquí, ni que sigui per curiositat per veure què et passa i, després, fes una neteja de regles:

```
sudo iptables -F
```

(és el mateix que *sudo iptables --flush*)

ara caldria tornar a fer un:

```
sudo iptables -L
```

per veure que realment t'ha netejat les taules de regles i, seguidament, tornar a provar amb el programa aquest que esmentes si es connecta correctament, o no.
