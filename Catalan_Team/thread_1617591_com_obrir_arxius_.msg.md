### Post by wgarcia on 2010-11-10
Una altra possibilitat és fer les següents instal·lacions des d'una terminal:

sudo apt-get install kmail
sudo apt-get install kmailcvt 

En principi el kmail amb aquesta extensió pot importar formats d'Outlook, no sé si específicament els que dius però. L'hauries d'obrir, escollir importar i buscar els fitxers en qüestió.
