### Post by akyra on 2010-08-13
Vols dir pasar o compartir?

Si vols pasar d'una carpeta de una partició d'ubuntu a una de windows o a l'inreves, amb el Nautilus o pots fer directament.

Si el que vols és compartir fitxers, podries compartir una carpeta de windows i ubuntu i podria accedir.

L'altra opció que tens és crear una partició que windows i ubuntu puguin llegir, per exemple que fos fat32 o ntfs, i deixar allà els fitxers que vols compartir.

No sé si és això ben bé el que preguntaves.

Salutacions
