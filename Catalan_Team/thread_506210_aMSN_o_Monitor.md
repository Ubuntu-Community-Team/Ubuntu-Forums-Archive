### Post by papapep on 2007-07-22
L'xorg.conf és dins a /etc/X11/.

Per altres ocasions, comentar-te que existeix un paquet que es diu "slocate" el qual permet localitzar qualsevol fitxer del sistema amb la simple ordre, per exemple:

```
sudo locate xorg.conf
```

L'únic que s'ha de fer de tant en quant és actualitzar la base de dades de paquets executant:

```
sudo updatedb
```

El qual fa una búsqueda per tot el sistema i "indexa" tots els fitxers i directoris per quan ho demanem amb "locate" poder-ho respondre ràpid.
