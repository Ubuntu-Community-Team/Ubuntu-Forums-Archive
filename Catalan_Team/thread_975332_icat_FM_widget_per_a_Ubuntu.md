### Post by jvmonjo on 2008-11-18
Jo l'he provat i funciona perfectament (tal i com comente al meu [blog]("http://www.monjo.cat/blog/2008/11/13/una-altra-manera-descoltar-catalunya-radio/")).
De fet no cal instal·lar primer l'Adobe Air perquè el propi widget te l'instal·la. Quan t'apareix el missate de s'està instal·lant al cap d'uns 30 segons se t'obrirà una finestra on hauràs de confirmar que vols instal·lar-ho. Comprovat a Ubuntu 8.10. Si ixí i tot no et funciona tal volta siga perquè no tingues instal·lat prèviament l'Adobe flash player o alguna altra dependència (t'aconselle instal·lar el paquet **restricted extras**:
```
sudo apt-get install ubuntu-restricted-extras
```
Salut
