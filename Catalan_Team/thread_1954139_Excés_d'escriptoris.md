### Post by wgarcia on 2012-04-09
Però, quin fas servir? Per desinstal·lar, per exemple, xfce, es pot fer 

```
sudo apt-get remove xubuntu-desktop
```

però per si un cas després faria 

```
sudo apt-get install ubuntu-desktop
```

no sigui cas que també elimini algun paquet de l'escriptori normal i després no et doni cap sessió d'escriptori (cosa que es podria arreglar entrant en "consola" i reinstal·lant el que faci falta, però sense interfície gràfica tot es torna una mica més difícil).
