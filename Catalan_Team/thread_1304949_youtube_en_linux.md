### Post by jdk9 on 2009-10-29
Has d'instal·lar el paquet ubuntu restricted extras. Com que veig que deus ser nou tens dos maneres:
-Usant l'instal·lador de programes (aplicacions/centre de programari de l'ubuntu si uses la 9.10 o si uses la 9.04 a aplicacions afegeix/elimina) i busques un programa que es diu "Programari restrictiu de l'Ubuntu"
-Per consola: Obres un terminal (aplicacions/accesoris/terminal) i escrius:
[FONT=Courier New]sudo apt-get install ubuntu-restricted-extras[/FONT]
Amb el comando sudo dónes "poders" de superusuari a laplicació.
Amb l'apt-get és, juntament amb l'aptitude, un dels dos instal·ladors per consola que duu per defecte Ubuntu,
Amb el comando install li dius que vols instal·lar.
I ubuntu-restricted-extras és el paquet que vols instal·lar.
Espero que gaudeixis de l'Ubuntu! Apa sort!
