### Post by Syzygia on 2007-05-07
Bones.
Jo faig servir el Seahorse per generar claus, publicar-les i obtenir claus d'altre gent.

Amb aquest programa, per generar una clau només cal fer Clau -> Create New Key i anar seguint els passos.
Si es vol publicar les claus en un servidor (perquè altre gent pugui fer servir la teva clau pública) s'ha de fer Edita -> Preferències -> Servidor de Claus i posar un servidor de claus adequat (jo faig servir el del mit, hkp://pgp.mit.edu:11371). Després, s'ha de fer Remot -> Sincronitza i publica claus.

Baixa't el programa i mira't com va, és bastant intuïtiu. Si tens algun dubte més, pregunta'l aquí.

Que vagi bé.
