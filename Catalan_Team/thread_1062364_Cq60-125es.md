### Post by papapep on 2009-02-08
Per a instal·lar el wicd (en el mateix procés, es desinstal·larà el gnome-network-manager, meravelles del sistema de paquets de Debian...) et cal seguir les indicacions de la pàgina de [wicd]("http://wicd.sourceforge.net/download.php").

Edites el fitxer /etc/apt/sources.list:

```
sudo nano /etc/apt/sources.list
```

i afegeixes la següent línia (sempre que facis servir la 8.10), per exemple al final, la posició no importa:

```
##Wicd
deb http://apt.wicd.net intrepid extras
```

Deses sortint amb Ctrl+x i executes la següent ordre:

```
wget -q http://apt.wicd.net/wicd.gpg -O- | sudo apt-key add -
```

Ara fas un:

```
sudo aptitude update && sudo aptitude safe-upgrade
```

i, seguidament, un :

```
sudo aptitude install wicd
```

veuràs que t'avisa de que desinstal·larà el network-manager i el gnome-network-manager, li dius que sí i endavant.

Quan acabi, hauries de tenir el Wicd al submenú "Internet" del menú principal, per a configurar-hi les teves connexions.
