### Post by LitusMayol on 2008-12-21
El programa es diu **UFRaw**.

És fantàstic. Està en els repositoris oficials (*ufraw*). A més disposa d'un paquet que posa l'**UFRaw** a disposició del **Gimp**. El paquet es diu *gimp-ufraw*. 

És genialment fàcil d'usar i instal·lar. Jo ho he fet mitjançant el **Synaptic**, però suposo que això també funcionarà:

```

sudo apt-get install ufraw
sudo apt-get install gimp-ufraw

```
