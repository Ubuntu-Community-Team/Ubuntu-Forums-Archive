### Post by Miquel Ubuntero on 2021-01-11
Hi ha una altre manera de gestionar les impressores amb Linux. Pots mirar a veure si la veus o la pots reinstal·lar d'aquesta altre manera? Has d'obrir un navegador web i anar a [http://localhost:631/](http://localhost:631/)
Has de tindre instal·lat el paquet CUPS, si no el tens obre un Terminal i executa Sudo apt-get install cups
CUPS (common unix printing system) és un gestor d'impressió unix que funciona tant en Linux com en Mac.
Sort!
