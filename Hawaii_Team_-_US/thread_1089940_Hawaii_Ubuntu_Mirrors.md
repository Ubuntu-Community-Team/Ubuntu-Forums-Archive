### Post by Marlonsm on 2009-04-19
Actually the list in [http://www.ubuntu.com/getubuntu/downloadmirrors](http://www.ubuntu.com/getubuntu/downloadmirrors) does not show all servers, it only shows the ones you can get the .iso from.

For example, the site only shows 3 servers in Brazil, but when I go to System > Administration > Software Sources and see the complete list, there are actually 6 servers in Brazil, but only 3 of them seem to have the .iso for downloads.

Maybe you can find a server in Hawaii.
