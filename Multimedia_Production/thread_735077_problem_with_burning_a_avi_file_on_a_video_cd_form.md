### Post by unoodles on 2008-03-25
The easiest way to make a dvd is to install the devede package from the repository.

(sudo apt-get install devede)

then go to your Sound And Video menu and click it. It is really user friendly.
The down-side is that there is (yet) no support for menus. The video file you choose will just automatically play when inserted.

If you want to try a more advanced program install the qdvdauthor package or kmediafactory.

Good luck.
