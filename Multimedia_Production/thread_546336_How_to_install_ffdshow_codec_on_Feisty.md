ffdshow is a subset of MPlayer's functionality, repackaged as a DirectShow codec so that crappy Windows Media Players can benefit from it. There's absolutely no point in running ffdshow on Linux since all of its features have been available anyway for a long time.

If you think VLC is a "basic POS" then you haven't used it. VLC is much more than just a media player. If all of its features were exposed through the graphical interface, the average Windows guy would be hopelessly confused and intimidated by them.
