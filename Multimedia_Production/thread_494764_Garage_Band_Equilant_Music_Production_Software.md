### Post by kayosiii on 2007-07-22
To play back from keyboard you should install

vkeybd
& patchage

Make sure that jack is running.
now you should be able use patchage to hook any program that excepts midi input up with vkeybd which will let you play notes with your pc keyboard. Just about every softsynth for linux will do this as well most midi sequencing packages. you can then typically route that output (again via patchage to a jack enabled app and record it.)

### Post by d_in_Conduct on 2008-04-20
Linux DAWs are going through a period of rapid development right now.  That's a good thing, but you have to keep up.  There is also the Jack factor.  This is a different way of working from PC and Mac music production.  It's a great way to work, but requires a slight 'mindset adjustment'.

There is a steep learning curve with any DAW, so don't expect to switch to Linux and jump right into making music.  I'm a long-time Cakewalk SONAR user and I set a goal for myself to record a complete song in Linux.  I found that nothing was easy.  I switched back and forth between Ubuntu Studio and 64Studio a couple of times and finally settled on Ubuntu Studio.  I found Rosegarden to be more MIDI-centric, and audio recording was a bit finicky.  But Rosegarden has the best documentation.  Ardour is good for audio and is bringing in MIDI.  I haven't checked in the past couple of months, so I don't know where it is with MIDI at the moment.  The documentation is incomplete and it seems as if the users have to write their own in a wiki.  I wasn't able to get much use out of Muse.  I ended up recording MIDI parts in Rosegarden,  then recording them in Ardour through Jack.

I found Jammin to be a great program for mastering the final mix, even with just the default settings.

If you are willing to spend the time, I think you can get pretty proficient in music production in Linux.

Some VSTs are usable in Linux if you are running a 32-bit system.  (this may have changed since I last looked)  Almost anything you can do with a VST is available as a native Linux plug-in with LADSPA, in fact, there are so many LADSPA plug-ins, you could spend all your time just looking for the one you want to use.

The only way to tell if you can leave Windows behind is to hunker down with a Linux system and start working with it.
