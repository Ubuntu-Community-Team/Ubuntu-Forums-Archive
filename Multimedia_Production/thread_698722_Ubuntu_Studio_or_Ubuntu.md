### Post by Stochastic on 2008-02-16
The main difference is the Real Time kernel that ships with Ubuntu Studio.  This allows for much lower latency times on your soundcard and can have it's priority in the stack adjusted.  If you're serious about working with audio in Ubuntu this kernel is mandatory.  However if you're new to ubuntu, the studio version does not ship with quite as many pieces of software for everyday things, so you will need to do a bit more installing of apps after install, which can be a pain for newbies.  It's possible to install the Realtime kernel on a standard ubie install and this might be the best route to try at first.
