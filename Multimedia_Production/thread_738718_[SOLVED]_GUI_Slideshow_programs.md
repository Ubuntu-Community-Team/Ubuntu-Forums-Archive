### Post by markekeller on 2008-03-29
And as for recording live video and editing it in a professional level program - try [Cinelerra]("http://cv.cinelerra.org/").  While I haven't tried recording live video with it, I'm almost certain it can do that - and it has an enormous amount of features; more "professional-level" than any other Linux video editor.

But, if you're looking for something a little simpler, try [Kdenlive]("http://www.kdenlive.org/").  It has a lot of the same features, but is more beginner-friendly (and easier to install).

### Post by cotcot on 2008-03-29
Same advise. Cinelerra or Blender or professional level, Kdenlive for common use.
In cinelerra or blender you need to know how to resize the pictures. But both have advanced features such as keyframes (this allows you to change an effect in time such as fading in and out). Both take some time to learn, but it is worth to do. Both have good manuals.

Kdenlive has a titler, fading options, multitrack to add audio, transitions, picture in picture. It is still in heavy development, so might encounter a bug. I was able to find a work around when I saw some. Easy to learn.
