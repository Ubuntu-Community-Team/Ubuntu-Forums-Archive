In case anyone is interested in making DVD-A discs...

My car (an Acura) has a DVD-Audio player.  To make my own DVD-A discs, I was using a Windows batchfile utility I got from this thread in an Acura forum:

[http://tl.acurazine.com/forums/showthread.php?t=118963]("http://tl.acurazine.com/forums/showthread.php?t=118963")

But I really was not happy with being stuck in Windows.  So I installed the dvda-author package and wrote my own shell scripts to do it in Linux, and posted it in the same forum for public download:

[http://tl.acurazine.com/forums/showthread.php?t=159101]("http://tl.acurazine.com/forums/showthread.php?t=159101")

If you're not interested in the history and just want the scripts and .deb, here's the link:

[http://www.dwasifar.com/DVD-A_Linux.tar.gz]("http://www.dwasifar.com/DVD-A_Linux.tar.gz")

Please don't laugh at my scripts.  It's not my native language.  :)
