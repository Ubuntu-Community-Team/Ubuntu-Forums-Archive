### Post by JGJones on 2007-12-02
Kdenlive - this is a KDE video editor - a quite simple one that does the job well.

It's not that stable though so I highly recommend saving very often - providing you do this, you can do quite a lot with it. It is quite similar to Windows Movie Maker although I think they aim to be more like iMovie.

As far as I know it cannot read the file format of other editors.

As for other editors in Ubuntu - there are some, but they all are big heavyweight applications with a steep learning curve and if you're after something simple then you want Kdenlive (PiTiVi is another one to watch, but it's in *very* early development so it doesn't do very much as can be seen here - [http://www.pitivi.org/wiki/Status](http://www.pitivi.org/wiki/Status) - however they plan to do a new release every 3 weeks regradless of code state)

Best bet now:

Use Kdenlive (just save very often (use Ctrl-S shortcut, it'll become normal)

EDIT - Forgot to mention - Kdenlive is a KDE application, but it will run fine on Gnome (I am using Gnome myself) - it is available in Synaptics (or Add/Remove Programs)


Hope this helps
