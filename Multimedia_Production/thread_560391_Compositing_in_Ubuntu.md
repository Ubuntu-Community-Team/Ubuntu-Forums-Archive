### Post by david3d on 2007-10-02
As a professional 3d artist I find that open source projects that tackle large complex tasks such as 3d animation, image manipulation, compositing, and NLE lack the stability, scalability, and the production workflows that the commercial offerings have.  So I don't know of any that I would really consider workable in a real production situation.

Currently even in the commercial space Linux lags behind when it comes to 2d image manipulation, compositing, and NLE.  On the 3d side of things you at least have Maya, XSI, and Houdini to choose from.

For compositing Nuke is a good choice but it depends on what type of compositing you're looking to do.  It is primarily a high end film compositing program but it does not have as nice of a workflow for motion graphics as AfterEffects.  Avoid Shake as it's dead.  Apple has already EOL'ed it and has sold it's source code to all the large studio's at $50k a pop.

So here's my advice which I'm sure most Linux users won't like.  You need to use the best tool for the job and unfortunately those tools do not run on Linux.  So if you really want to do these type of things and get away from Microsoft buy a Mac.  Get FinalCut Studio Pro and the Adobe Creative Suite CS3 bundle that fits your needs the best.  At least there you'll have more production worthy choices.  You'll also be working with industry leading and standard tools which will make you marketable in the job market and I bet more productive to boot.
--
David

### Post by fxgogo on 2007-10-05
David hit the nail on the head. Unless you are going to pay for a commercial package, the open source alternatives are still quite a way off the production needs of professionals. Jahshaka seems to be a dead project. There is precious little going on in theie forums. From my investigations, the Blender projects has the best chance of creating a really good compositing solution. The basics are all there. It does focus on effects type work over motion gfx, but I think with some linking up to the non linear editor and a faster preview feedback loop it will be good for that too.

Open source has done some great work in getting the creative apps production ready, and there is still some way to go still. The last big link in the process is a killer open source compositing app and an editing app. The Peach Project from Blender should push the former ahead quite a bit.
