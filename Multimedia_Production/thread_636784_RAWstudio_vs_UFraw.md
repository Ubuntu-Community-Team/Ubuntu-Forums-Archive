### Post by Magnes on 2007-12-10
There are also:
- Raw Therapee (very good)
- s7raw (in WINE, works very well, designed for Fuji Super CCD cameras)
- LightZone for Linux (free, but only some old version)

Some other programs (like F-Spot, Picasa 2.7 beta, Krita etc) have some options to convert RAW files.

Actually, UFRaw is the good one. RAWstudio is the one with the management features and v.1. finally added JPEG support. dcraw is the basic CLI from which the others borrow code. And the plugins for Gimp and the gang come from one of these three. I have used Bibble Pro quite well also, even the Windows version on Wine so that's good clean code, only that it isn't free.
