### Post by jexworks on 2008-02-18
I've been using Growisofs for some time yet a recent update has caused a silly issue. Previously the parameter -dvd-compat was all that was needed to burn a compatible disk but now I have to specify -dvd-video as a genisoimage parameter for the burned disk to work in a standalone player. 
Strange. Caused a few coasters and a couple of hours of annoyance.

Versions Growisofs 7.0.1, Genisoimage 1.1.6

Hope this saves someone else the time and trouble solving it.
