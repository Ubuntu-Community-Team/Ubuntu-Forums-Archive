### Post by christian.convey on 2008-01-07
I'm about to digitize my family video tapes and photos.

I'd like to be able to tag each object (photo or video) with some structured and unstructured data.  For example, "date scanned" is structured.  "comments" are unstructured.

But I'd like "people in photo/video", "location", and "date of photo/video" to be structured and fuzzy.  This is so I can model uncertainty about when/where something took place.  And so, for instance, if I want to search on things with Aunt Martha in them, my search results can display the difference between photos that *definitely* have my dear old aunt in them, and those that maybe don't.

This fuzzy-tagging feature might be overkill.  Free-text commentary, as long as I can search on it, might be enough.

Any suggestions?
