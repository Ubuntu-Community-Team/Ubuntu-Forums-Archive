### Post by DMK62 on 2008-03-19
You can install it using Synaptic package manager . Synaptic is located under System > Administration. If you have never used Synaptic hit the reload button. Just search for amarok and then right click amarok and mark for install and then click on the Apply button. It will most likely want to install several other packages ( dependancies ) and that is normal as it needs those to work.
