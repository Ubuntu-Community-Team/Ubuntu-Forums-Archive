### Post by Stochastic on 2008-02-18
Within the LADSPA plugin set there's the CAPS Audio Plugin Suite that contains some amp emulation plugins.  See [http://quitte.de/dsp/caps.html](http://quitte.de/dsp/caps.html)

To get these as realtime effects just download JackRack and add your desired plugin to the realtime effects rack.
