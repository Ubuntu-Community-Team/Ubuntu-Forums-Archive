### Post by jondecker76 on 2007-08-31
Just wanted to show off the latest state of my studio setup. I was a long time Windows user and switched to Ubuntu about a year ago. 

In Windows, I used FL Studio Producer's Edition with an Emu 1820m sound card. I was able to do anything I wanted in this setup, but after some consideration I decided to migrate to Linux... So I sold my 1820m and bought a M-Audio Delta 1010 (and couldn't be happier!)

My current setup includes:
UbuntuStudio 7.04
Dual widescreen LCD's
Furman M-8 Power Conditioner
Behringer BTR2000 Rack Tuner
M-Audio Delta 1010 audio Interface
Behringer Powerplay Pro-8 Headphone Amp
Behringer Ultrapatch Pro PX3000
Nady PRA-8 Mic Preamp
MAudio Keystation 49e keyboard
Akai MPD16 Drumpad
AudioBuddy 2 channel guitar preamp
Marshal MX990 Condenser mic
Marshal MX991 Condenser Mic
Carvin CM68 Dynamic Mic
Carvin AG100D 3 channel amp
Samson CH700 headphones

My current computer is a Pentium 4 2.8ghz, 640 GB HDs, 2 GB RAM

I will soon start building a dedicated rackmount computer (4U) with a dual core processor of some sort, 4GB RAM, SolidState operating system partition and a software RAID3 1TB HD array. I am also working on a MidiBox LC with motorfaders to interface with Ardour.

I'm mostly interested in live recording/mixing and would like to start recording some acoustic pieces soon. Just trying to learn the ropes right now.

### Post by jondecker76 on 2007-09-01
I used to do some (minor) recording in Windows - a lot of what I have now is from my old Windows box setup. For the most part, I record some friends who play but I also want to produce a few of my own. For me its basically a hobby.

I've also been looking for good linux alternatives to programs I used to use in Windows. What I have found so far is pretty promising. I'm starting to learn Ardour pretty well - its only good for Audio work right now but midi support is right around the corner. Personally I don't really like rosegarden so this will be a great addition. Another thing I really miss is a good sampler. I have also messed with LMMS (an FL Studio - like DAW) but it is a bit too young and I'm not crazy about the Interface.

Jack is probably the best thing thats ever happened to digital audio in general. Once you get it down, the possibilities are really endless. 

I don't have any experience with video editing, and from what I've heard UbuntuStudio really lacks in video tools. However, its been said that the great people working on UbuntuStudio have a big surprise for the next release and I'm guessing its in this department.

All in all, I'm very happy with making the move to Linux - it just takes time to learn the new software.

### Post by xhilyn on 2007-09-03
Hi All.
I have used Cubase for years, first on an Atari, then a Mac and finally on a Windows PC but I have now got fed up with Windows and MacOS as they are both way to proprietary and just lock you in to closed file formats all the time, with windows WGA being the final straw as far as I'm concerned. I will not be using Vista any time soon, though at the moment I still use Cubase SE under Windows XP for my main studio.

So with the idea of changing over to Linux completely sometime in the future I instaled UbuntuStudio 7.04 on an old Pentium IV 2.2 ghz PC  that I bought specially for the job. I added a tiny Tapco Mix100 mixer which is attached to a really cheap sound card that also gives me a MIDI in/out as well, I want to get an M-Audio 2496 card when I can afford one.  For MIDI I/O I use an old Novation BassStation and I have a truly ancient Roland D110 Multitimbral Sound Module attached as well.

I have started to use Rosegarden at the moment as I have always worked with mainly MIDI tracks and just a few audio tracks for Vocals and Guitars etc. I will have a good look at Ardour when they add the MIDI.

So far I have only done one small song using Rosegarden as I find that the combination of Rosegarden running on UbuntuStudio is nowhere near as efficient as Cubase running on Windows XP. But I'm sure it is early days for Audio Production on Linux and so am looking forward to using it more as it all matures. Exciting times ahead I'm sure.
