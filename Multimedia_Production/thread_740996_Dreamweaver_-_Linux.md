### Post by Stochastic on 2008-03-31
I've never been a Dreamweaver user and prefer to hand code, but here are some native WYSIWYG html editors for linux:

-Kompozer [http://www.kompozer.net/](http://www.kompozer.net/)
-Bluefish [http://bluefish.openoffice.nl/](http://bluefish.openoffice.nl/)
-Aptana [http://www.aptana.com/](http://www.aptana.com/)
-Sea-Monkey [http://www.seamonkey-project.org/](http://www.seamonkey-project.org/)
-Quatana [http://quanta.kdewebdev.org/](http://quanta.kdewebdev.org/)

if you try these out and realize they just don't measure up to dreamweaver (hard to believe, but possible) then I'd check [http://appdb.winehq.org/](http://appdb.winehq.org/) to see if it will run through Wine.
