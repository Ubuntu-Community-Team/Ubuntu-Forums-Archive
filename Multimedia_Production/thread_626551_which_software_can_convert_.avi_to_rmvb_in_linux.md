### Post by sonofusion82 on 2007-11-29
unfortunately i don't think there are any open source encoder for rmvb since it is a closed format.
if there any reason why you want to convert to rmvb? i personally will not recommend rmvb. storing videos in rmvb is not exactly a good idea as if ever Real Networks goes bankrupt, future computers will never be able to playback rmvb.

mpeg4/divx avi are great enough for most purpose and plays on many DVD players that supports divx.
if you need better compression, how about considering x264?
