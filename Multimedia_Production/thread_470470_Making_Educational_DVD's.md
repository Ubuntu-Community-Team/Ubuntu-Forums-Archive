I have seen posts about QDVDAuthor, and I tried it out with the slide show feature.  The problem is that I need to be able to control the slides and information that is displayed using the remote control and the shows created seem to require a timer.

### Post by trippinnik on 2007-06-11
You can use flash then export to .avi or some other format.  A lot of DVD menu development is being done this way from what I've heard.  Unfortunately there isn't much in the way of flash development tools for linux.  This will probably give you the flexibility you are looking for.

### Post by knorr.orange on 2007-06-18
For anyone following this thread, I have settled on using DVDStyler.  It is pretty simple, and I am able to make simple DVD's with just menu's with backgrounds, which works like a controllable slide show/presentation.

It was a mess to get installed by the way, and eventually I found another thread here referencing some old dapper packages to use.
