### Post by prshah on 2008-04-07
Maybe you need to enable more software sources? In the Add/Remove programs, you have a button "Preferences" in the lower right hand corner. Click this, then on the "Ubuntu Software" tab, check all the options except source code (main, universe, restricted, multiverse), then click Close


Then try again... did it help?
