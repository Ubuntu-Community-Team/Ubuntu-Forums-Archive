### Post by jbaerbock on 2007-10-18
Trying to fit a .avi that is 703MB on a 700MB CD-R. I used the default Kubuntu CD writer before for such a problem and it forced it to work. However I have not found an option for the default Ubuntu cd burner in which you can force it to continue. It just gives me an insufficuent space error. Any ideas how to fix this?

---

### Post by logos34 on 2007-10-18
go to the kde desktop config editor and enable 'overburn' in cd writer app

if you're using linuxmint with gnome:

sudo gconf-editor

>applications>nautilus cd burner>overburn
