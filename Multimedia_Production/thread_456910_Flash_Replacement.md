### Post by sirgazil on 2007-05-28
I'm a Flash MX Professional user too and I'm trying to migrate to free or open source software.

This is what I'm learning to create RIAs (Rich Internet Applications):

[[COLOR="DarkOrange"]http://www.openlaszlo.org/[/COLOR]]("http://www.openlaszlo.org")

You will need to learn XML and JavaScript (instead of Actionscript) because these are the basis to create your applications in OpenLaszlo.

For 2D animation (vector based) I haven't found a fully functional application yet. But you can check this:

[[COLOR="DarkOrange"]http://ktoon.toonka.com/[/COLOR]]("http://ktoon.toonka.com/")

I hope you find this useful!
