### Post by kapello on 2008-03-11
You could try the program VLC - [http://www.videolan.org/vlc/download-ubuntu.html](http://www.videolan.org/vlc/download-ubuntu.html)

Once installed, choose File > Open Capture device. I *think* this allows you to capture from a source (eg a mic I guess) and then stream the output to a network. Streams can also be directed to a file for saving.

I've never tried it myself but the functionality seems to be there, and I've heard other people recommend it for streaming solutions.
