Hello,

As you may know, the MainActor video editing software for Linux has been [COLOR="Red"]**discontinued**[/COLOR]. 

And now there is a petition going on to ask MainConcept to release the Linux version of MainActor as Open Source software: 
**Read and sign the petition:  [http://www.ipetitions.com/petition/mainactor?e](http://www.ipetitions.com/petition/mainactor?e)**

The best idea is to buy the software free (as they did with the [Blender 3D suite...]("http://www.blender.org/") for some years ago), but first we need to make MainConcept known that a lot of people is (are) caring about this matter.

Read also: 
[http://www.ibiblio.org/joey/videolinux/petittion-to-release-mainactor-source-code/](http://www.ibiblio.org/joey/videolinux/petittion-to-release-mainactor-source-code/)

Of course it is possible that Linux does not need MainActor because it already has Kino, Cinelerra and Lives.
