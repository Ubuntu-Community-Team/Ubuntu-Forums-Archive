### Post by FakeOutdoorsman on 2008-02-07
Try out pdfjam.  It's available in the Universe repository:
```
sudo aptitude install pdfjam
```

pdfjam is a collection of a few pdf related tools and includes pdfjoin.  Example:
```
pdfjoin file1.pdf file2.pdf file3.pdf --fitpaper true
```

More info: [PDFjam readme]("http://www2.warwick.ac.uk/fac/sci/statistics/staff/academic/firth/software/pdfjam").
