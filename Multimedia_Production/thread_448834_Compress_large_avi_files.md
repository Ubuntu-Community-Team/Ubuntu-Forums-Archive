### Post by rsambuca on 2007-05-19
I would recommend using something like Avidemux (some like to use kino) for a video editor/coverter.  Both are available in the repositories and you can install them easily using the Synaptic Package Manager.  With Avidemux you just open your file(s) and you can edit them, process them, cut stuff out, etc, and then save them to whatever format you wish.

Probably the most popular formats right now are xvid/divx.   They create quite small files without degrading the quality significantly.

If you really want to get fancy with your video editing, you can try cinelerra, but it is much more difficult to use (and install - not in the repositories).
