### Post by jamielc on 2007-10-08
I have a GNX4 and it has always worked with different versions of GNU/Linux that I've tried.  It does seem to setup as a sound card.  You'll really probably want to use JACK and a Real Time kernel however.  Distributions designed for audio work like 64 Studio do seem to work better.  I tried UbuntuStudio and wasn't overly impressed when compared to something like 64 Studio and plain Ubuntu with the default kernel was horrible.  I returned to Windows XP & Sonar for my audio recording work.  Ardour 2 is outstanding as a recording program but the effects plugins weren't even in the same league as something like Sonar.  My $0.02 anyway.  Not sure about your USB problem, sorry.
