### Post by fredj on 2007-08-05
There is a manual for Gimp you need to download it and read it. There are lots
of other programs for manipulating video such as Kino and Stop Motion. The
ultimate program for doing all of these things is Blender but you have to learn 
how to use it (and that will make your brain hurt). There are no easy answers
to all of these things which don't involve hard work.

If making good art was easy everybody would do it an no one would be 
interested in it.

### Post by eye208 on 2007-08-08
Check out Synfig Studio. It's a tool to create 2D animations/cartoons using vector graphics and imported bitmaps. Think Flash movies, but without the interactive stuff.

You can find the program in Ubuntu's "universe" repository. The project website is [http://www.synfig.com/](http://www.synfig.com/)
