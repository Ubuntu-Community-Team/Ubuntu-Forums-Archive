### Post by barna on 2007-11-05
well, it was a very stupid question.... in kmediafactory (video-->properties) one can set the aspect ratio. this was set to widescreen (16:9). I did not recognize the presence of this option (and the fact that this has been changed) because it was always automatically deduced from the video file itself. but now I overwrote the input video file (it had widescreen before, but the new file had 4:3 ratio) without kmediafactory being notified about this, so the (originally automatically set) aspect ratio remained there as 16:9
