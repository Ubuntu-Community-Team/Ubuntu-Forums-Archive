ho, I´m an adobe user, which uses linux blender/cienelerra/Openmovieeditor too, and I´ve created a project to bring some HolyywoodFX-like  3d transition to the GNU apps. 
Actually, in hollywood FX the users always complained that the 3D transitions can´t be modifies in a good 3D editor (3d max), but here in the GNU world, that´s possible, because blender can be used extenally for a NLE app, and that´s what we want to do.

[http://gvfx.blogspot.com/](http://gvfx.blogspot.com/)

the idea behind this is that the USER has the control of how many transitions can have, and our project wil lreceive user-made blendfiles, and will compile them into a single blend to be distributed and used by the GNU tools editors.

I think ubuntu studio can offer everything that the editor may need, and our tools culd help them to achieve more in less time, because this transitions can be used as effects, or change it to create 3D DVD menues.
