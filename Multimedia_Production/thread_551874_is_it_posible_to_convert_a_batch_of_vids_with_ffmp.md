### Post by LKRaider on 2007-09-15
You could use ffmpeg, but I would suggest you to use the vixynet flv2mpeg4 converter: it is a special flv converter that will keep the original quality of the video and is very fast on converting.

[http://sourceforge.net/projects/vixynet/](http://sourceforge.net/projects/vixynet/)
( they host an online service aswell: [http://vixy.net/](http://vixy.net/) )

I have compiled it some time ago, you can get the binary here:
[http://lkraider.eipper.com.br/files/share/flv2mpeg4.7z](http://lkraider.eipper.com.br/files/share/flv2mpeg4.7z)

You'll need 7zip to extract the archive, so apt-get install p7zip f you haven't already. :)
