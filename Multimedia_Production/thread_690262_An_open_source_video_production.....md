### Post by philc on 2008-02-07
Well, hardly a production, more like a short 4 minute compilation created with Open Source tools. 

[http://kapitalmototv.co.uk/play-183-0.html](http://kapitalmototv.co.uk/play-183-0.html)

**WARNING: Contains Nudity!**

It's something a little fun I put together using PAL DV footage edited in Open Movie Editor, using FFmpeg and mjpeg tools for some clip transformations.

Not yet nearly as slick and easy as commercial products, I can say that with only a little patience, some video knowledge and the will, editing video with open source tools is certainly possible.

A long way still to go for this software, but progress is good.
