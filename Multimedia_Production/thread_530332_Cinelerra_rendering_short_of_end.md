### Post by dad311 on 2007-08-20
Has anyone had an issue with Cinelerra not rendering an entire project?

I created a slideshow with dvd-slideshow and imported the VOB into Cinelerra.  After rendering the project, about 5 minutes of material is missing from the end of the 15 minute movie.

I tried converting the VOB to MPEG, but had the same result.  I tried rendering the "project", "selection" and "in/out"  makes no difference.

The render successfully finishes without any errors.

Any ideas?

I used dvd-slideshow to create my slideshow then imported the .vob file into Cinelerra.  It seems the Cinelerra has some issues with .vob files and editing.  to resolve this issue, I used the -noclean flag with dvd-slideshow and imported the .mpeg file from the dvd-slideshow tmp directory into Cinelerra.
