### Post by jnbek on 2008-04-02
You could install XP inside Virtual Box, thus negating the need to dual boot, or run the Adobe software under Wine. It's quite stable and will run quite stable as well. If you need USB Support get Virtual Box from [http://www.virtualbox.org/]("http://www.virtualbox.org/") otherwise you can install the Open Source Edition by entering: 
```
sudo apt-get install virtualbox-ose
``` I have written an article on setting it up on Ubuntu 7.10  I am not sure if posting the link here is allowed so PM me, or tell me if it's cool to display a link to my own site.
