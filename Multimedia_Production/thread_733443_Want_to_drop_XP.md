### Post by robeast on 2008-03-24
Hey, glad to see you're making the switch. Your MOTU interface is not officially supported and I've heard nothing but disdain for their products matched with linux. 

You're in luck, though, because I've spent the last week researching all linux compatible audio hardware and listing them in one place so users like you won't have to deal with the headache of figuring out what hardware is going to work.

[http://www.linuxstudiopro.com](http://www.linuxstudiopro.com)

This site is still expanding, but I think you will be surprised at the selection. As far as I know, most interfaces should work in most linux distros provided you have either the FreeBoB or ALSA drivers installed. Depending on what type of interface you want (usb, firewire, pci) there are other miscellaneous system components that need to be present, but any full-featured linux distro should have them. 

I use a FirePod in Ubuntu Studio, and it required no configuration or installation whatsoever--aside from installing Ubuntu Studio. Everything was set up and ready to go, just had to select the freebob driver in JACK.

Let me know if my site is helpful. Since I've had a good experience with the FirePod I recommend it as a replacement for your MOTU 828mk2. It has a VERY similar set of I/Os.

### Post by robeast on 2008-03-24
FFADO is not released yet, and I wouldn't use it unless you want to actively participate in testing or developing it yourself. You could wait until FFADO is ready for release, but from what I can tell that time is anyones guess. They are working very hard to support as much as possible and make the first release stable. For now your best bet is to get something supported by FreeBoB or ALSA. The Saffire Pro is supported by FreeBoB, but the software that comes bundled with it will not work with Linux.
