### Post by kayosiii on 2007-08-30
> **chrome307 said:**
> Unfortunately the answer is NO

AVI files and other multimedia files have already been compressed and do not benefit from further compression unless you want to lower bitrates/resolution or convert from one format to another.

This is not at all accurate. AVI files are nothing but a container format. They can be totally uncompressed or compressed by a number of schemes (a compression scheme is essentially a plugin for an AVI file and referred to as a Codec).

Most Compression schemes are lossy (They approximate your original rather than being the exact thing. Transcoding from one lossy format to another will generally result in worse quality. 
There are two senarios where this can be useful. You need a particular codec to play the file somewhere you need to. Or you want to make the file a *LOT* smaller. 

Heres a quick list of CODECS that I know are commonly used.
Divx, Xvid are two common codecs for content found on the internet you will want to use this to get your content to youtube for example.
H264 is commonly used to compress high def content. I am not sure whether AVI can be used for a container format (Quicktime is more common).
MiniDV video cameras shoot content in DV Codec.
Most older analog capture boards create motion jpeg files.

Hope that helps...

### Post by eye208 on 2007-10-02
> **Legos said:**
> I open an avi file, but I can't see any "compress" button.
Compression is done when you save the file. But first you have to specify the way it should be compressed, i.e. stream formats, scaling, cropping etc. This can be done using the drop down boxes on the left side. Their default setting is "copy", i.e. the video/audio stream is not changed. Of course this is not what you want, so you have to change these settings before saving the video.
