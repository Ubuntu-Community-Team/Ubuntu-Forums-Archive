Easiest way? Install Audacity and the lame mp3 encoder, open the file in Audacity (it can open mp3 files), use the amplify plugin until the volume is what you're after, then export back into mp3. Audacity will ask you to locate the lame encoder, I can't remember exactly where it installs to, but a quick file search should help you out. I think it may be in /usr/lib
