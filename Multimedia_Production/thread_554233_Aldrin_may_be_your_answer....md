### Post by hseiken on 2007-09-18
Those of you looking for a standalone application for making electronic music (it's not quite up to the task for arranging live instruments yet), should take a look at aldrin.  It's based on the zzub library, a new variation of the buzz system for which BUZE for windows is based on, but working in a completely different fashion (which may turn off some users, but I love it)..

Basically, Aldrin is striving to be a tracker version of Reason.  In other words, EVERYTHING you need to make a killer track is included within.  Synths, effects, sequencers, etc.  The current 'official' release is .11, but if you like, there's the Aldrin Nightly Builds repo that's already far advancing past .11's capabilities.  I've been using it relentlessly since I've found it and, coming from windows use to all kinds of bells and whistles, this damn program puts to shame pretty much everything I've tried in Linux as far as just doing everything I want, which is the following:

A.  BE STABLE
B.  HAVE GOOD SYNTHS
C.  HAVE CLEAR INTERFACE
D.  NOT NEED JACK
E.  SOUND GOOD
F.  BE EASY ON MY CPU
G.  BE FUN

Buzz users should feel pretty at home with it, and those new to the 'style' of usage shouldn't have too hard a time of figuring it out.  It does support LADSPA plugins, but they're definitely not useful in this environment.  For those that are hobbiest like myself, you can be rest assured, you don't need the newest processors to go crazy with this thing.  I've got a measly 1.45Ghz machine with 1G of RAM and I'm able to run over 2 dozen machines of varying useages (I've got a multiband compressor setup for my 'default' song which applies 3 parametric EQ's and 4 Compressor/Limiters and it tops out at 6% CPU when used...that's 7 machines!) without even hitting a significant CPU ceiling.  

It's HIGHLY recommended for those coming from windows used to programs like ReNoise, Reason, Orion, Fruity Loops, etc. where one program is all you need to make some nasty music.  

The official page is here:
[http://trac.zeitherrschaft.org/aldrin/wiki](http://trac.zeitherrschaft.org/aldrin/wiki)

The unofficial nightly ubuntu builds (I highly recommend these to see where this incredible program is going) are here:
deb [http://www.pohunek.free.fr/aldrin-nightly-builds/ubuntu/feisty/](http://www.pohunek.free.fr/aldrin-nightly-builds/ubuntu/feisty/) binary/ (add that to your sources then select ALDRIN from the synaptic list)

For some examples of what it does all on it's own, here's two tunes I started this week (probably not everyone's cup o tea, but hopefully, you can get an idea of how it sounds.  All synth sounds are realtime).

[http://hakyoku.googlepages.com/hs-ripoffwip1.mp3](http://hakyoku.googlepages.com/hs-ripoffwip1.mp3)
[http://hakyoku.googlepages.com/agf-lovesunshinehs-hcoremix.mp3](http://hakyoku.googlepages.com/agf-lovesunshinehs-hcoremix.mp3)

These are nothing but aldrin.  No jack linkage to anything else.

I hope you enjoy this program as much as I do...it's going to be BIG when it reaches 1.0.  Lookout...
