### Post by Mblackwell on 2007-12-27
I'm going to point you to this thread since I already made some posts on the subject: [http://ubuntuforums.org/showthread.php?t=650873](http://ubuntuforums.org/showthread.php?t=650873)

As long as you can get the hardware running, you can set up as many inputs and outputs as you want for any track using JACK (and Ardour).

You can  even use other JACK aware apps as inputs (although I'm not very knowledgable about that aspect as of yet). 

I also am/was an Acid user, and I've had really nice results with Ardour, although I've done multi-take recording as opposed to multi-track, I can't imagine it being a bad experience once everything is set up.
