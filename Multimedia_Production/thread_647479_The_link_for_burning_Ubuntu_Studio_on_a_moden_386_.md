### Post by Pocadotty on 2007-12-25
[Ubuntu Studio i386]("http://cdimage.ubuntu.com/ubuntustudio/releases/gutsy/release/ubuntustudio-7.10-alternate-i386.iso")

That is the link for the Canonical Hosted DVD image.  You should note that all the Studio Images (at least the ones I have found) are Alternates, so they do not have the pretty live CD installation interface.  So you might want to read up on installing with the Alternate CD before you install.
