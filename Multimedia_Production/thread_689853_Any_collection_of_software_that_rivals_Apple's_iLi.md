### Post by agim on 2008-02-13
There is nothing wrong with it. The only thing you will run into is that it will take longer for them to load. When you run a Gnome app in Gnome, the libraries are already loaded into memory, when you run a KDE program, all of the libraries have to be loaded. If you have ample memory (512MB) you won't even notice, other than a second or two lag when opening a program.
