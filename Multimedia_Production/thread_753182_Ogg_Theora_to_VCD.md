### Post by indiecast on 2008-04-21
try bringing the ogg file into Kino
then exporting it in VCD quality MPEG into 'Q' DVD Author from Kino.
Then use K3b to burn your files to a CD in VCD format

I haven't actually done this and all i worry about is if Kino can recognized OGG files.  i think it prolly does.  but yeah ive done a lot of DVD burning lately and some VCD stuff in the past, so i think i have the process right

this is the fun stuff!

indiecast


ps.  make sure you have all ur mpeg codecs installed

applications > add/remove...  then run a search for "Gstreamer"

install all the codecs u think you need

if stuff still isnt working, try searching for "codec" and you should find some more stuff

also, if you arent familiar with the terminology, codec are software that decode video files like wmv, mpeg, avi, vob, etc.
