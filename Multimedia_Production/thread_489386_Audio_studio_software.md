### Post by c_martini on 2007-07-03
> **newtimes said:**
> I too am looking for a good but simple recording program that saves to wav file.

The *.wav file format is a Microsoft proprietary audio format so I doubt you'll see this as a save option in any Linux based DAW applications besides something like 'Sound Recorder'. There are open source alternative formats that Linux based apps use when saving the output files (such as .ogg and .flac). However, if it must be saved as a wav file, there is a decent guide here on Linux audio formats: [http://www.topology.org/linux/audio.html]("http://www.topology.org/linux/audio.html") which may help...
