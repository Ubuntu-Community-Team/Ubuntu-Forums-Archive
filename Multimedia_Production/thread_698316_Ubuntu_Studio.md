### Post by Stochastic on 2008-02-16
There are thousands (maybe millions) of programs that are in the repositories online that just can't fit onto the DVD install of ubuntustudio.  These are all optional extra pieces of software, but many are great.  See [http://packages.ubuntu.com/](http://packages.ubuntu.com/) for more info.

Using ubuntu studio as a standalone, unnetworked, box is totally fine.  The updates that come through the net do a couple things: fix functionality bugs, fix security bugs (less of an issue for you if you're not networked), and add functionality to programs (newer versions).  It's totally fine to go without these updates, just be aware that you're missing out on these three areas.
