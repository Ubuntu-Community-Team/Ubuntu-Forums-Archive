### Post by starfry on 2007-07-25
Hi bluezdood,

I'd say that, given that this is your livelihood and there are a good few apps in you list, I'd say this isn't going to work for you.

However, I would recommend you try using VirtualBox to create a VM - install XP on that and then install your Adobe applications on that.  There is a separate thread on here somewhere that explains how you can run a "seamless" desktop combining the native Ubuntu and the XP VM. I run this way but it is not the machine I earn my living from.

Perhaps not what you want to hear, but I hope this helps :)
