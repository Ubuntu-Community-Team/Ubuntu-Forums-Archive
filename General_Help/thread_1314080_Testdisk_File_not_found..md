### Post by drs305 on 2009-11-04
You have to install testdisk, it's in the "universe: repository (Synaptic or Software Sources> Settings, Repositories, Ubuntu Software).

Install testdisk, then run it from the command line with:
```
sudo testdisk
```

Here are a couple of good howtos:
[http://www.cgsecurity.org/wiki/TestDisk_Step_By_Step#Current_partition_table_status]("http://www.cgsecurity.org/wiki/TestDisk_Step_By_Step#Current_partition_table_status")
