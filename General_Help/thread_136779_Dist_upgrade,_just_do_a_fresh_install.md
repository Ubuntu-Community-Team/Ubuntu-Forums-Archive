### Post by Xian on 2006-02-26
Your personal configs (themes, bookmarks, userprefs) in the /home directory should not be affected. If you have made modifications to system configs, and they are candidates to be replaced, you will generally get a prompt asking you which to keep. That would require some research on your part to review the diffs and decide on a proper course of action. Any other system changes would be necessary in order to boot and run the system as intended.
