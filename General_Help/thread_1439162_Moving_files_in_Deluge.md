### Post by Cas07 on 2010-03-27
Move Storage is to move the files to some other location on your file system for the files associated with that single torrent and does not change the system wide default torrent download directory.

My torrents default download directory is: /torrentdl
Once a torrent has finished i choose Move Storage to manually move the files to /var/media/Videos for example. 

This can all be done automatically by choosing to move storage upon completion or using the labels plugin.
