### Post by fluffman86 on 2009-11-28
startx should work, or at least used to.  I think now, though, karmic automatically starts the desktop stuff when it starts X.

Be warned, though, that just X is pretty useless.  You'll be lucky to have a console to type commands into and have to run everything from there.

If you want something a little more useful, try installing icewm, blackbox, or fluxbox, and running that.  You can also search for "widow manager" in synaptic to get more options.
