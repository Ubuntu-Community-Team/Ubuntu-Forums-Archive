### Post by wausaubill on 2008-11-18
Yes, it is possible, though not working for the moment for me.  As I understand it, WUBI makes a file your Linux "drive" so that you don't have to partition your existing hard drive.  From this I think it is safe to say that when you install WUBI you should select the largest (30G) installation, so that you have 30 gigs to play with.

I would also say, that if you are like me and have an entire external drive you would like to give over to Ubuntu that maybe WUBI is not the way to go, but rather a full blown dual-boot install, with the entire external drive used as your Linux "partition."

On the other hand, using WUBI first so you can see if Ubuntu works well on your machine and that you like is a safer and supposedly easier alternative.
