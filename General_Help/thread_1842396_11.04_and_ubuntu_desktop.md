### Post by _d_ on 2011-09-11
You can try logging out and selecting Ubuntu Classic (no effects) from the bottom menu bar on the login screen and see how that works for you. :)

Also, if you are using ATI Catalyst, there is a bug with Sync to VBLANK that causes Unity to slow down ATI cards severely. If this is the case install the Compiz Config Settings Manager from software center, Synaptic or via terminal with apt-get, then select the OpenGL plugin and disable Sync to VBLANK.
