### Post by v3xtra on 2009-06-27
They absolutely work in Jaunty, because most likely, the version of CONKY has not changed, so all of the configuration settings will be the same.  Conky is in the Jaunty repos, so you should be able to install it, create a new .conkyrc file in your home directory, and put a configuration in there and start conky!

I suggest going to the conky website and looking at the conky config variables:

[Variables for Output](http://conky.sourceforge.net/variables.html)

[Config Variables for Settings](http://conky.sourceforge.net/config_settings.html)

Hope that helps!
