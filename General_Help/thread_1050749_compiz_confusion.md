### Post by phantom3113 on 2009-01-26
For the cube, the following conditions must be met:
-Desktop cube and rotate cube plugins activated
-"Extra" selected under system>>preferences>>visual effects
-must have horizontal virtual desktop set to 4

To set the dodge effect, under focus animations there will/should be a white box with some numbers and an animation name in it. From here, you will either need to click on the "change" or "edit" button or else right click on the animation name (I'm not sure which, I'm on a school computer right now) Hope this clears things up for you.
