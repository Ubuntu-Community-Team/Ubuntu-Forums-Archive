### Post by coffeecat on 2009-03-26
What sort of modem are you using? Is it a usb modem? If so, many of these are very difficult to get to work in Linux because they are really only designed for use with Windows. If it is a USB modem, post details and someone may be able to help you.

If you have a proper modem-router, all you need do is connect the router to the computer with an ethernet cable and most times it just works.

If you have a USB modem, you would be wise to consider buying a modem-router. Better security, faster speeds, ability to use more than one computer at the same time. Much better.
