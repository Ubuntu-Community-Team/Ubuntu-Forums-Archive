```
  	Option		"EdgeMotionMinSpeed"	"15"
  	Option		"EdgeMotionMaxSpeed"	"15"
  	Option		"UpDownScrolling"	"1"
  	Option		"CircularScrolling"	"0"
  	Option		"CircScrollDelta"	"0.1"
  	Option		"CircScrollTrigger"	"2"
	Option		"SHMConfig"		"on"
EndSection
``` 

With [FONT=Courier New]Option "SHMConfig" "on"[/FONT] you can tweek the settings realtime to find your optimal settings with the "synclient" command.

You need the xorg or xfree86 synaptics driver to use this.
```
apt-get install xorg-driver-synaptics
```
or
```
apt-get install xfree86-driver-synaptics
```

Sorry for the long wait but I've moved and been without internet for a while.

You still have problems?

If you didn't back up your X config file you can recreate it with:
1. $ sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
2. $ sudo dpkg-reconfigure xserver-xorg

Where you put the section doesn't matter. It's a stand alone section wich can be placed anywhere thats not inside another section.

The config values for my touchpad is for Alps touchpad. I believe the values doesn't match a Synaptic touchpad but it shouldn't crash your X.
