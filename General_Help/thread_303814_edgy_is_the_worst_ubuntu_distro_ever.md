### Post by orb9220 on 2006-11-20
> so i had to use official upgrade method (that went almost ok, though).

Well people do have things break updating from dapper. But updating from Breezy would likely even break more.

> install cd did not even boot (checksum and verification done), so i had to use official upgrade method 

Just means you have a bad CD and have to reburn iso at a slower speed until you get a good burn.

I have been using edgy as well as many others with little problems.

For what it is worth I would not complain about a distro being bad just because I couldn't install it the way it is supposed to be installed from a CD. It not the distro's fault that you don't have a valid burned iso to install from.

Sorry couldn't be more help. But if you want to resolve these issue's  then you need to do it a step at a time. Like first getting a valid CD that will boot to live first then doing a clean install. Major Upgrades are to me anyway a cross-fingers affair.
