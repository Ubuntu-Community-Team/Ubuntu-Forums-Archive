### Post by Pettman on 2006-11-01
In earlier versions of Ubuntu you could press ctrl shift and then input a hexadecimal code with the 0 to 9 and a to f keys (while still pressing ctrl and shift) and it would output the the sign that had that Unicode number. It was in my opinion really handy. Does anyone know how to get it back?

Edit: It seems you now have to press ctrl+shift+u+<hex code> to achieve the same effect.

### Post by Rebeca on 2006-11-03
That's right, but that makes it all a bit more complicated (unless you like stretching your fingers). I was ok with the Ctrl + Shift + <hex code>, but I've found it easier to assign a Compose key.

System > Preferences > Keyboard > Layout Options > Compose Key Position

In my case, I chose the *right windows key*. So, now I just type *that* and:

a ` = à
a ' = á

[etc](http://en.wikipedia.org/wiki/Compose_key).

### Post by thepizzaking on 2006-11-03
You don't have to hold down Ctrl+Shift+u+<hex code>.  You press Ctrl+Shift+u, then you can release the 'u' key and enter the rest as you did in earlier versions.
