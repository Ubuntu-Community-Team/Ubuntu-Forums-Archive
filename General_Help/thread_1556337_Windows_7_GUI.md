### Post by Mark Phelps on 2010-08-19
The Gnome-look site is a good one for locating and downloading Themes that will give you a "look-and-feel" SIMILAR to Win7.

I emphasized SIMILAR, because there is literally no way to give Ubuntu a look-and-feel that is identical to Win7, simply because for that to be true, Ubuntu would have to be able to provide ALL the GUI features provided by Win7 -- and it does not do that.

For example, Aero Peek and Aero Snap, two features built into Win7, are not available through simple theme changes.  To get these, you would have to search separately for them, and from what I've read, a LOT of customizing work is involved.

### Post by bergfly on 2010-08-20
Default KDE looks far more like default windows 7 than gnome, so you would definitely want to start there. The reason we say this, is KDE has the bar at the bottom, a more similar system tray and a big shiney button for the menu. 

Adding wallpapers and icons in KDE is easy. You just click on system settings in the menu and choose appearance under look and feel. The advice listed above to look for "aero" is good. So making it look windows like is easy.

As mentioned many times already, after initial appearances, there is a much steeper learning curve to get fully "literate'. Having gone through all the flavours of Windows, OSX and more Linux distros than you can put in a toaster, I can honestly say that there is no painless way to do this, and that just because it seems hard at first does not mean it is hard. The thinking behind each is simply different. 

When I first abandoned the world of MS, I spent about 3 months cursing how hard it was to do the simplest things in Linux (and this is back when some things were hard!) Within 6 months, I found myself occasionally sitting in front of a windows box, and wondering why it was so hard to do the simplest things in Windows.!

They are all just different. Stick with it, read lots, Google questions and post here if you can't find the answer, and good luck
