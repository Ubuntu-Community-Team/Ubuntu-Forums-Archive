### Post by rihikz on 2013-11-03
Hello. Since you installed ubuntu over windows, there is no other way than to just install windows again normally. Simply boot your windows installation cd or USB drive and install windows, formatting your drive. This will completely remove ubuntu and all files. If you would want to keep your files, create a backup of your files on a  external drive or a cloud.
