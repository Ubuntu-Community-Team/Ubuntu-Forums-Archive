### Post by t0p on 2009-07-10
> **kumarsasi420 said:**
> Is there any software or commend to recover the files

There are quite a few.  Search synaptic for something like "file recovery", you'll see a bunch of results.  But you need to be aware that these utilities do not guarantee anything.  They can work really well sometimes, at other times they may recover just fragment, maybe nothing.

Also, if you are using the computer in question (ie for posting here), you are increasing the chances that the lost files will be overwritten.  If that happens, that's it - they'll be gone for ever.

The important thing to remember is when you use Shift+Del to delete a file, you are permanently deleting it - that's why ubuntu pops up a warning asking if you're sure you want to do that.  The smart thing to do if you're browsing files with the gui file manager is to delete files by right-clicking them and selecting **Move to the Deleting items folder**.  That pops them in the wastebasket, and you can get rid of them later.  Using Shift+Del is not a good idea if you are not entirely sure you want to get rid of the file.
