### Post by seren6ipity on 2008-11-29
What is the error it gives. Try removing by 
sudo apt-get remove avant-window-navigator 

For making the top panel go to bottom, right click any where on panel -> customize panel. Under fixed position, there is a layout which currently has top highlighted. Click on bottom middle rectangle to move panel to bottom.

Better solution would be to remove top panel completely by using customize panel option and add all the options you need to bottom panel. Click bottom panel > Add items.
