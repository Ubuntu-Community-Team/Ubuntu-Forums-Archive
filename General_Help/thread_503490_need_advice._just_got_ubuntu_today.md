### Post by Scheater5 on 2007-07-18
Nowadays, you *have* to do basically nothing on the terminal.  Use Synaptic to install programs, and Gnome has utilities to edit just about everything but Xorg graphically (and that's coming soon), KDE has Kcontrol and System Settings to do the same.  The terminal is powerful (sudo apt-get install *program name* is your best friend), but it is rarely necessary in Ubuntu.

### Post by Scheater5 on 2007-07-18
In response to your question about games - natively, almost no games you're likely to be familiar with are compatible with Linux (there are a few exceptions I've heard about, but I don't know specifics).  Many work with Wine, a compatibility layer that allows many windows based programs to run in Linux, and many more with a commercial version of wine made just for games called Cedaga.
