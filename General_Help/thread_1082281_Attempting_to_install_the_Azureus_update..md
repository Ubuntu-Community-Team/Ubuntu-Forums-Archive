### Post by taurus on 2009-02-27
There is nothing to compile for vuze.  You just save the file, Vuze_4.1.0.4_linux.tar.bz2, to wherever you want to unpack it.  Then, unpack it there and it will create a new directory, vuze, where you need to change to that new directory to run it.

```
tar xvjf Vuze_4.1.0.4_linux.tar.bz2
cd vuze
./vuze
-or-
./azureus
-or-
./updateAzureus
```
