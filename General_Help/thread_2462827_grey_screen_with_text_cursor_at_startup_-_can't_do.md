### Post by dragonfly41 on 2021-05-28
As another Dell user (3268) my first thought was to try booting up a LiveUSB to test if Ubuntu runs in "Try Ubuntu" mode.
But if Ubuntu came preinstalled I'm guessing that you have never used a LiveUSB.

The "side software" reference offers a clue but no idea what caused the error.

If you have access to another PC/laptop I would start by creating a LiveUSB.  
If you only have access to Windows then Rufus will do that.

Then you at least have an emergency toolbox.
