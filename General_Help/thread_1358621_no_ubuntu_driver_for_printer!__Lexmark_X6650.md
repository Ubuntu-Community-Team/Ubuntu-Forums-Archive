### Post by cgb on 2009-12-18
I would try a driver for a similar Lexmark printer that is available if possible.  You may not get all the same functions as a proper driver but its highly likely it will print.  So if there is any Lexmark X6*** series printer driver available, try that.  If not maybe even go to another X series printer driver.  Chances are one of them will work since all printers use similar PCL commands.
