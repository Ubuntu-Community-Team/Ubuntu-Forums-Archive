### Post by Zeroangel on 2006-03-18
The package manager is called Synaptic. Its in the main menu. System > Administration > Synaptic Package Manager.

Just press the search button and type in mysql and type in mysql and you will be able to see all installable items that relate to MySQL.

The package you're looking for will be something like "mysql-server-5.0"
