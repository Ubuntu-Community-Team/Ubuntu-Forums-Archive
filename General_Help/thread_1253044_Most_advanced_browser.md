### Post by cloudscream on 2009-08-29
Epiphany, Gnome's browser. 2.26 (stable) is based on Gecko. 2.27 (development) is based on Webkit. Epiphany 2.26 has "Smart" bookmarks, the Awesome bar, tabbed browsing and Adblock Plus. It is a native Gnome app so it integrates seamlessly into Ubuntu (unlike Firefox). I believe it is much faster than Google Chrome, and uses less memory than Firefox.

Probably with version 2.28, a stable release of Epiphany based on Webkit will be realized. Some people see that this browser has a lot of potential. If you want a plain, secure and fast web browser and not a box full of add-ons, then you might find Epiphany quite convenient to use.
