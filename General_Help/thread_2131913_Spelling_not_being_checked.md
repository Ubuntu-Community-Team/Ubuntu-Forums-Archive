### Post by slickymaster on 2013-04-03
[https://addons.mozilla.org/firefox/addon/united-states-english-spellche/](https://addons.mozilla.org/firefox/addon/united-states-english-spellche/)

You can see which dictionary is selected if you right-click in a text area and open the Languages submenu. Note that at least one dictionary should be installed and have a check mark to indicate that it is selected.
Open the "Add Dictionaries" link to install a dictionary if you do not have one and make sure that "Check Spelling" in the right-click context menu is check-marked.
You can enable or disable spell checking globally: Edit > Preferences > Advanced : General: Browsing: "Check my spelling as I type"
