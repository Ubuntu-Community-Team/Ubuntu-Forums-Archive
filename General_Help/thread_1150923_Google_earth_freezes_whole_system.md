### Post by soro2005 on 2009-05-06
Make sure that you use the restricted nvidia driver (in the repositories, and enable it in System --> Administration --> Hardware Drivers). You may also try whether you have more luck with the newest betas which you can install from the ppa here: [https://launchpad.net/~thefirstm/+archive/ppa](https://launchpad.net/~thefirstm/+archive/ppa)
Warning: This may cause your graphics server to fail on startup, leaving you with the command prompt or in very low graphics mode.
