### Post by psycho5 on 2010-08-10
Try a different USB port on your computer. If that doesn't work, go to Control Panel > Administrative Tools > Computer Management > Disk Management

If your thumdrive is not listed there, then there's a problem with the port. If it is, then format it. If it still doesn't work, then maybe your thumbdrive is corrupt. Try switching it.

### Post by garner_nc on 2010-08-10
You need to make SURE the bios in your computer is set to boot from either a USB drive and/or a USB flash disk. Go into your bios and check the boot device order.

I found this out the hard way. Finally got it to work though.

Your comment about having used this drive to boot a different computer made me think about this. If it was bootable there, it should boot this machine also.
