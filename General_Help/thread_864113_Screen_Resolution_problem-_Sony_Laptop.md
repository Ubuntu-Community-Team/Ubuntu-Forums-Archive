### Post by daimaru on 2008-07-19
you could run
```
sudo dpkg-reconfigure xserver-xorg
```
but I'm not sure if it will help. Your other option is to include the right resolution in your xorg.conf file.

```
Section "Screen"
    Identifier     "Screen0"
    Device         "Videocard0"
    Monitor        "Monitor0"
    DefaultDepth    24
    Option         "TwinView" "0"
    Option         "metamodes" "1440x900_60 +0+0"
EndSection
```
change the modes to your desired resolution eg. 1600x1200_75 (_75 = 75 herz)
restart xserver (ctrl + alt + backspace)
