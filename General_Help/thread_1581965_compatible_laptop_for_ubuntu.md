### Post by SeijiSensei on 2010-09-25
I'd suggest an [Intel]("http://intellinuxwireless.org/") wireless card myself.  Intel released all its drivers under the GPL, and they are now fully merged into the stock Linux kernels.  I've never had compatibility problems with my Dell 640m in which I had the optional Intel 3945 card installed.  It's run Windows XP and 7, Fedora 12 and 13, and Ubuntu versions since 8.04 without a hitch.

As for graphics adapters, despite their being proprietary, I strongly suggest using an NVIDIA card.  Modern NVIDIA adapters include [H.264 decoding in hardware]("http://en.wikipedia.org/wiki/VDPAU"), which makes a big difference if you watch video.  I sent my daughter off to college with an [Asus EEE 1201n]("http://www.amazon.com/Seashell-1201N-PU17-BK-12-1-Inch-Black-Netbook/dp/B002ZLOR56") netbook in part because it uses the NVIDIA ION platform.  (There's a [newer model]("http://www.amazon.com/gp/product/B003QR1TAO/ref=pd_lpo_k2_dp_sr_2?pf_rd_p=486539851&pf_rd_s=lpo-top-stripe-1&pf_rd_t=201&pf_rd_i=B002ZLOR56&pf_rd_m=ATVPDKIKX0DER&pf_rd_r=1ZTEK060QEYRSDJAND6P") that uses ION2, but it has a slower processor, and there weren't any obvious advantages to ION2.)
