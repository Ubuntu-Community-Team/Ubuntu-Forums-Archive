### Post by mc4man on 2008-06-20
if you want to add a category to the applications menu it needs to have at least 1 app enabled first in that category. For example system tools. You have to enable an app. in it first then you can add system tools as a category. If you try to add system tools first it will uncheck itself after a few seconds.
