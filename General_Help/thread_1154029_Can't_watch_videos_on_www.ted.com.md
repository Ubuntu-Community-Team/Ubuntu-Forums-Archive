### Post by regor210 on 2009-05-09
ted.com is  using Flash. So the same flash player you are using to play YouTube video should work. My AMD 64bit X2 computer seemed to have trouble with the Flash player that came with  Ubuntu restricted extras. I kept getting those blank screens. Sometimes I reloaded the page 6 or 7 times to get it to load. So to solve the problem I started using the new Adobe Beta Flash Player found here.
 
[http://ubuntuforums.org/showthread.php?t=772490](http://ubuntuforums.org/showthread.php?t=772490)  

If you need to use the new Flask Player do not forget to un-install your old one.
