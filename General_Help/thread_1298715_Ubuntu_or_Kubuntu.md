### Post by lovinglinux on 2009-10-23
I have been slowly switching from Gnome to KDE applications for a while and recently fully switched to KDE once I installed Karmic, after about a year using Gnome. I'm loving it. My performance is now better than Jaunty with Gnome, some KDE applications are much better then Gnome IMO (Kate, Kwrite, Dolphin, Ktorrent, K3b, Ark...) but I still use some gnome apps (Firefox, Meld, Gnome-mplayer, Synaptic...). Additionally, KDE configuration center is awesome.

I didn't installed the full KDE environment, but kde-minimal and kdeplasma-addons over a command-line only installation from the alternate Ubuntu CD. I didn't like the result of installing the full desktop from Kubuntu CD.

Plasma crashes sometimes, but I'm not sure if it's because some issues with the Beta release of Karmic, specially related to nvidia driver or compiz. But in overall, stability is very good and when plasma crashes, it reloads itself pretty fast. So no worries.
