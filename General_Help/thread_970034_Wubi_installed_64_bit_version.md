### Post by ago on 2008-11-04
Yes you probably have a 64bit machine and hence you got 64bit ubuntu. This means that binary blobs (such as flash) compiled only for 32bit architecture will not work, BUT there are several workarounds that make it possible to use 32bit binary packages, a simple search for "ubuntu flash 64-bit" should lead you to what you need.
