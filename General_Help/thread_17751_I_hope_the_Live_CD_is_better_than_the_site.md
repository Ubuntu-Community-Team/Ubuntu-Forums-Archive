### Post by psychic on 2005-03-03
**Problem 1:** USB automounter-support

***STEP 1:*** /dev
Add your USB-mass storage-device in your USB-port
Check for a sda(0,1,2,3) device being added to your /dev

***STEP 2:*** checking mount
[I]mkdir -p /media/usb
mount /dev/sda* /media/usb[/I]
# does this work? -- proceed to STEP 3

***STEP 3:*** adding rule to /etc/fstab
*echo "/dev/sda1             /media/usb       auto           rw,user,auto      0       0" >> /etc/fstab*
or...
*echo "/dev/sda1             /media/usb       auto           rw,user,noauto  0       0" >> /etc/fstab*

first rule should automount the usb-device when you insert it, second needs to be mounted manually.

***STEP 4:*** Adding mount-applet to gnome panel.
I do not have the english gnome-language active atm, so i cannot walk you through this but you can figure it out 
(right-click add-to-panel and add it from the menu that shows up, thats as far as i can help there)


**PROBLEM 2:** ZIP support
Installing jazip from the ubuntu package-list should solve that, i think that should be enough to work with but if it doesn't. feel free to ask :)

For the rest you need to narrow it down to the 'ubuntu' problems... saves me the work of helping with a problem you don't have anymore. :)
