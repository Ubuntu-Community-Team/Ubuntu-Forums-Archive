### Post by Horbo on 2013-02-02
Install Hamachi from [http://secure.logmein.com/us/labs](http://secure.logmein.com/us/labs)

Then install Haguichi (a very nice GUI for Hamachi) here: [http://www.haguichi.net/download/](http://www.haguichi.net/download/)

Then just run Haguichi - if you can work Hamachi in Windows you can work Haguichi.
