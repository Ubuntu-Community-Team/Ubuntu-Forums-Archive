### Post by jimrz on 2007-08-05

> I have a dual core processor, and i was wondering if i had to compile my own kernel to get smp support? or is the default kernel support multiprocessing?

thanks,bj

if using ubuntu 6.10 edgy or later on intel or amd procs the default kernel will handle it just fine (does so on my core2duo E6600 and 2 other boxes using dual PIII's). if using dapper 6.06 LTS or earlier you will need an smp kernel based on your proc mfg, intel and amd versions are both available for install via Synaptic package manager, so no DYI compiling would be required.
