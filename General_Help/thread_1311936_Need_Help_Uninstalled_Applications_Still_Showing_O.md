### Post by Richardarkless on 2009-11-02
Hi, by the sounds of things you are new to how to the gnome interface.

If you right click on the menu (where you have access to your programs), there is an option for edit menu's

from there you can use the left column to navigate the menus and the right column to edit any applications of menus inside it, from there you should be able to delete, rename, change icon, add separators, etc

Welcome to Ubuntu and hope you enjoy the experience

oh and btw this should apply to any other distribution that uses the gnome window manager
