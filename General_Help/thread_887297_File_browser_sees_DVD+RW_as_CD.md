### Post by Taxman415a on 2008-08-12
You just need to use another program to burn data to it as far as I know. The file browser sees it as full just sort of because that's how cd's and dvd's are seen. But if you open Brasero or install K3b (be prepared if you don't have any of the KDE libraries installed yet, this will download a fair amount) you will be able to overwrite whatever is on the disc, or if it has been set up properly, add to it.

Also note, dvd rw are different from cd rw in that they should not be erased. Simply over write them. Erasing them shortens their lifespan apparently. I've had better luck with k3b than brasero, especially dual layer burning, but for dvd rw you should be fine with either.
