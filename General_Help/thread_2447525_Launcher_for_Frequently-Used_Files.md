### Post by 2CV67 on 2020-07-20
Since 2014, I have been using "Drawers" to open my frequently-used files.
Support stopped years ago, but it still worked fine in Ubuntu 19.10.
I just upgraded to 20.04 & Drawers has finally vanished without trace...

Is there any way to get it back in 20.04? (I don't expect so)

Is there any other way to get the same functionality?
Drawers allowed me to pin my chosen files (about 15) in my chosen fixed positions in a window which opened with one click on the Applications Launcher (Dash to Panel) & closed automatically after selection.

Thanks for any suggestions!

---

### Post by 2CV67 on 2020-07-22
Thanks to Dennis N in another thread, I found that Gnome Shell Extension "Files View" seems to do just what I wanted!

I can pin any file or folder in the "Favorites" area.
I can access Files View Favorites in one click from the panel.
I can launch any Favorite in one simple click.
Files View closes automatically afterwards.

So that answers all my requirements.

Consider this SOLVED.
