Does anyone know how I can delete a read only file if I'm the admin, but I I am not the owner?

---

### Post by TheFu on 2018-08-09
If you are the admin, just remove it.
Use the shell and the **rm** command.

If you are the admin, learning basic shell commands is mandatory.  A free, no-hassle book, [http://linuxcommand.org/tlcl.php](http://linuxcommand.org/tlcl.php)  You'll also need a basic understanding of users, groups, and file permissions.
