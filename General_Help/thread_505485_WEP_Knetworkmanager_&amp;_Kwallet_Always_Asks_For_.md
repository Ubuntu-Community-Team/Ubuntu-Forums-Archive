### Post by Arisna on 2007-07-20
If you blank the password for KWallet, it will not prompt you for it anymore, but be aware that this will be the case for all passwords you have stored in there.  I also think the wallet is encrypted with the password as the key.  Not sure if there are better solutions for this.  I think it is possible to create multiple wallets, but I have never done so.

### Post by goodtimetribe on 2007-12-30
