### Post by Quarkrad on 2017-03-22
The version you have downloaded is intended to be installed on your pc as an independent operating system.  When you boot  your pc hit F12 during the boot part and you should get an option to boot to the memory stick - this will boot to Ubuntu on the stick.   At this point you can use Ubuntu and experience it.  You can shut down Ubuntu take out the stick and boot to Windows 10 as normal.   If you like Ubuntu you can use it via the stick, but to get the full benefit you need to decide whether you want to install it permanently on your laptop. You have two choices, wipe windows off your laptop and only have Ubuntu as the only operating system installed or, what I suggest most people here would advise, create a separate partition(s) and have what we call a dual boot system where you have both Windows 10 and Ubuntu installed and can choose between them (I like many people dual boot windows 10 and Ubuntu).

### Post by Bucky Ball on 2017-03-22
Welcome. As above from Quarkrad. Ubuntu doesn't 'open on Windows'. You need to install it to a USB or hard drive. Sounds like you might have created the Ubuntu installer with Rufus. Well done getting that far. When you boot to the USB by hitting F12 and selecting it you should see some options. Choose 'Try Ubuntu' and enjoy for now. That will NOT install anything to your hard drive. :)

Have a look around and see how you like it then decide whether you want to install.

PS: There are various 'flavours' which you may like better and they can be tried the same way. Xubuntu and Lubuntu look different and use less resources, just to name a couple.
