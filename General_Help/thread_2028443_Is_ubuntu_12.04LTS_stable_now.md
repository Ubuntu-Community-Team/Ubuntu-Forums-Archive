### Post by Cheesemill on 2012-07-18
You are probably best to hold off until 12.04.1 is released, this way you will be able to get an installation disc with all of the packages that have been upgraded since the initial release. This saves you having to do a big update straight after you have installed.

Whenever I roll out a new Ubuntu release to Servers or Desktops at work I always wait for the LTS .1 release so that the majority of serious bugs have been worked out.

12.04.1 is due to be released some time between the 23rd and 29th of August.

The best way of making a full system backup is to use imaging software such as [Clonezilla]("http://clonezilla.org/"). This way you can easily return your system to the exact state that it was in.
