### Post by Tamlynmac on 2011-03-23
To convert a djvu file to pdf.

1. Install djvu4 from the Synaptic Package Manager
2. Open the djvu file in djvu4
3. In the menu select export as .pdf

IMHO it's the easiest way to convert a djvu to pdf.

Good Luck & hope this helps
