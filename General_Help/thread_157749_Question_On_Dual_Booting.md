### Post by Ramses de Norre on 2006-04-09
The recovery mode will give you a root shell, which can be very usefull when things are messed up.

```

Ubuntu, kernel 2.6.12-**10**-386
Ubuntu, kernel 2.6.12-**9**-386

```
The first one is a higher patch version of the kernel, always use that one when possible.
