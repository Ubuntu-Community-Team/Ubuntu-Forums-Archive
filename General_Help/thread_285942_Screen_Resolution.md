Firstly, you need to make sure that you have the correct rates for your monitor in Xorg, which for your monitor according to the link you gave would be;

```

Section "Monitor"
    Identifier     "Generic Monitor"
    HorizSync       24.0 - 80.0
    VertRefresh     49.0 - 75.0
    Option         "DPMS"
EndSection

```

and then you need to make the right resolutions available as well in the Section "Screen". Default depth should be 24, so you can just edit this one;

```

SubSection     "Display"
        Depth       24
        Modes      "1280x1024" "1152x864" "1024x768" "832x624" "800x600" "640x480" "1x1"
EndSubSection

```

You should then be able to select this in the screen resolution dialog in Preferences and choose the highest refresh rate you can. I don't have a LCD Monitor, but this is how I get 1152x864@75Hz on my 17" CRT.
