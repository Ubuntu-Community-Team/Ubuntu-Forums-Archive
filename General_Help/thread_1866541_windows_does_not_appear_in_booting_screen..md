### Post by varunendra on 2011-10-21
As a quickshot, open a terminal in Ubuntu, and enter this command:
```
sudo update-grub
``` Reboot to see if Windows appears. If it still doesn't, please download boot info script, extract and run it as described on the download page ([here]("http://bootinfoscript.sourceforge.net/")) and post back the contents of **RESULT.txt** created by it.
