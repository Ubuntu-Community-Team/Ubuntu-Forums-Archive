### Post by Basher101 on 2012-01-07
install synaptic package manager. Open it and look if the packages "nvidia-current" | "nvivida-settings" | and "nvidia-common". Installed packages have a green square to the left. If one of those three is not installed, you can mark them for installation by rightlicking them and choosing "mark for installation". To install lick the small arrow at the top to apply.

Reboot and see if it is fixed.
