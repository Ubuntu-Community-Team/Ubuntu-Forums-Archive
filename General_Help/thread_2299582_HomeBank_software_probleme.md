### Post by ajgreeny on 2015-10-19
I can't help with homebank as I have not used it but I do use kmymoney2 which I find faultless; I've been using it for years with no problems other than my own occasional mistake in an entry which can then be hard to find in reconciliation.

I don't know if it is possible to move from homebank to kmymoney2 by exporting the files in any way, but I know it's possible to import quicken files (qif) into kmymoney2, so if you can export homebank data as quicken files you may be able to try it out and see what you think.

It is a qt/kde application so will pull in quite a few library packages if you don't either use kde(kubuntu) or already have another kde application installed, but as I am fairly eclectic in my package choice I already had k3b and a few other kde packages around.
