### Post by drpaul on 2007-03-27
I don't know if my experience will be useful to you [I'm in US], but I recently did the new notebook thing. The guidance I gave myself was to stick to Intel parts where possible [I have the impression that Linux software seems to run more smoothly initially with Intel] and to maximize RAM [2G for me].

I ended up getting a HP dv6125us. This doesn't have the highest end graphics, but that's not a biggy for me. All of the basic things including wireless networking were fairly easy to get going. Sound has been on again, off again but mostly OK. Builtin webcam works with ekiga, but I haven't found anything else that talks to it. I haven't really tested battery life yet. The Core 2 Duo processor hasn't been overly stressed yet, movies play fine, compiles don't take too long.

All in all, I'm satisfied, and I don't have to run Win MCE [it's on a hard disk in my desk drawer].

HTH

Paul

### Post by jtibau on 2007-03-27
As some people already said, stick with Intel... The whole centrino deal. And if you really need to go cheap, try to get nothing lower than a Pentium M. A Core Duo would be highly advice able since there's a really notable difference in performance.

I'd say something like a:
Pentium M -> Core Duo
512 MB -> 1 GB RAM (in most cases it is really easy to upgrade, so if you are a DIY'er buy extra sticks separately/later if you need more)
you don't need much disk space... I don't think you can even find anything lower than 60 GB anyways, and thats plenty.
DVD-RW is pretty standard yeah, don't miss it if you can.

I think 700 euros is around $1000 right? I'd say you can get a decent deal with that much.
