### Post by cbowman57 on 2011-11-18
Find the .desktop file that launches wireshark open it with an editor & put gksu in front of whatever command it currently launches with.

---

### Post by rajohns08 on 2011-11-18
thanks here was my solution:

1. open terminal (ctrl + alt + t)
2. issue command "sudo nautilus"
3. click "File System" in left pane
4. files are stored in "usr/share/applications" so navigate there
5. right click icon you want to change and click properties
6. add "gksu" to beginning of "Command" field
