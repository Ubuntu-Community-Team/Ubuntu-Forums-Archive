Edit: I have fixed this problem. This was solved by turning off the computer. Then unplugging the harddrive (with Ubuntu and Grub). Later I plugged the harddrive in and then everything worked as it should. The screen goes on early and I can read the bios posts. Later the Grub multiboot loader loads so that I can choose an operating system.
