If you are going nuts because of the address bar of firefox 3, here is the solution


 goto:
> About:config 


Set:
 

> browser.urlbar.matchOnlyTyped == true
 



no more bookmarks in the addressbar.




if you don't like the way the new address bar LOOKS, there is an extension called oldbar that you should add
