### Post by rahilm on 2010-11-30
Well .. i don't know of any file managers...
nor do i know of any add-on..
but what u can do is:

when u try to save a file... you have basically three frames.. one is the file name frame..
the other two are below it.
the one on the left displays places like, home, downloads, videos, music, etc etc... (bookmarks)
the one on the right displays what the current directory contains (the list of files)


click anywhere on the list of files.. and start typing the name you want to save.. as u type.. it is displayed in a new box somewhere... nautilus will jump to the file matching to what u have typed , if it exists..

this way . you can know if a file exists before u save it.. this may mean to type the name twice.. but u can copy paste whatever u have typed in the box...
