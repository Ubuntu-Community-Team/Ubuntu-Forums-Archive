### Post by oldfred on 2018-07-01
Did you update UEFI from Acer?
Or run the UEFI reset to defaults function?
Those reset many UEFI settings. I keep a list of all the settings changes I make in UEFI, so I can redo them.

Then you may have lost the "trust" setting in UEFI for Ubuntu as now it shows unknown which is typical for systems missing trust settings.
Boot0000* Unknown Device: 	HD(1,GPT,d5e93d70-4186-4ecf-9f7b-6e62561e8ee1,0x800,0x32000)/File(EFIubuntushimx64.efi)RC

       Acer Trust Settings - details, some now report that then secure boot has to be on to set trust:
