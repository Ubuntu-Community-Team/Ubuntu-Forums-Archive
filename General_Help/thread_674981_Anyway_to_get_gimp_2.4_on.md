### Post by djrobthaman on 2008-01-22
When I was using feisty I had gimp 2.4 from a repository, which is way better than just downloading a .deb because then you get automatic updates.  I'm not 100% sure which one it was though, but I believe it was this one:

[trevino's repo]("http://3v1n0.tuxfamily.org/apt-repository/dists/feisty/3v1n0/")

Here's the deal though.  Technically, it's not in that repo.  To get it, you must install the 3v1n0 repository package, which adds tons of 3rd party repos to your list, which is great for the amount of programmes it gives you access to.  But, it doesn't authenticate all of them.  I tried once to find all the gpg keys, but could never get them all.

So you'll get a bunch of annoying messages whenever you update or install certain packages complaining about them not being authenticated.

But, a long story short, I believe that doing this will get you a repository with gimp 2.4 in it.
