### Post by HermanAB on 2007-08-08
If you want to play games, then you have to dual boot so that Windows can run natively.  To do that, you have to install Windows first, then partition the disk and install Linux.  Linux will detect that Windows is already there and will add it to the boot menu.  If you install Windows second, it will overwrite the boot loader and then Linux won't work anymore.

Some Windows games can be made to run on wine, but it isn't really worth the effort in my opinion.

### Post by hikaricore on 2007-08-08
Since no one has mentioned it yet, the best way to see if your applications run properly/at all under wine is to check:  [The WINE Application Database]("http://appdb.winehq.org")
There is no reason anyone (Herman) should be telling you not to bother.

Take a look around that site for your Apps/Games and then try them yourself, and decide if they can run well enough for your comfort.
WINE has come a long way so many software titles will run nearly flawlessly, but on another note I agree that if you want to run the latest and greatest games that have just come out, you're mostly out of luck.

WINE can run many popular titles (on decent hardware) such as Diablo, Halo, World of Warcraft.  Some versions of MS Office products run,
as well as image creation software such as Photoshop and Corel Draw (not the latest versions) if the tools provided in GIMP are not enough.
