### Post by linuxyogi on 2010-03-12

I don't know what kind of internet connection you are availing.
 
 Go to ---- system ---- preferences---network connections ----highlight the wired connection (example : Auto eth0)---click edit ---- ipv4 settings ----

Now, at the "DNS Servers" field type 

              4.2.2.1, 4.2.2.2  

Restart your computer

##### If you connect to the internet via an ADSL modem which uses DHCP there are some additional steps you need to take before implementing the above mentioned procedure. #########

While connected to the internet open terminal & type :-

ifconfig

note the ip address, subnet mask, gateway (eth0)

A typical ip configuration provided by an ADSL modem --------

ip address : 192.168.1.2 
subnet mask : 255.255.255.0
Default Gateway : 192.168.1.1

Now, 

Go to ---- system ---- preferences---network connections ----highlight the wired connection (example : Auto eth0)---click edit ---- ipv4 settings ---- CHANGE THE **"METHOD"** YO **"MANUAL"** 

THEN at the addresses field :-

ADD ----type the ip address which you have previously noted then press ENTER (Dont forget to press ENTER after each entry). Similarly type subnet, default gateway & then finally at the "DNS Servers" field type 

              4.2.2.1, 4.2.2.2 

**REMEMBER THERE IS A COMMA AFTER 4.2.2.1 FOLLOWED BY A SPACE.**

RESTART YOU PC.
