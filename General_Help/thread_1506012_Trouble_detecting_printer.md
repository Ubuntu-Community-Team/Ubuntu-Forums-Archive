### Post by Mark Phelps on 2010-06-09
Open a browser and enter "localhost:631"

That will open a windows to the CUPS configuration system.  In there, you will see an entry for Manager Printers, and also one for adding a new printer.  Use the pages to enter the information needed to set up your printer.

If you're talking about using the scanner, that's an entirely different problem than printing.
