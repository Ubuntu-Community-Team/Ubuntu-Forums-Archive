### Post by MegaJim on 2008-09-18
Check out Netbeans 6.1

There is a version of netbeans in the repository, but I find the more up to date version available for download at their website ([http://www.netbeans.org/](http://www.netbeans.org/)) to be much more reliable.

Make sure you also install the official sun JVM and JDK:

```
sudo apt-get install sun-java6-* 
```

### Post by rlzyoner on 2008-09-18
Sometimes IDE's such as Netbeans and Eclipse can be quite intimidating for new programmers.
Its often handiest to use gedit or scite or some other editor and then to compile and run the programs from the command like.
eg
javac MyProgram.java
java MyProgram

First command compiles it
Second command runs it
