### Post by deadflowr on 2014-09-14
At login, you click on the icon next to the login username/password box. Then select gnome.
In the newer versions, you also get gnome-classic, which is a classic-like interface built on extensions for gnome-shell.

I would also recommend installing gnome-tweak-tool and browsing the selection of extensions at [https://extensions.gnome.org/](https://extensions.gnome.org/)
The best part about the extensions page is that as soon as you toggle an extension to install it starts functioning on your system, immediately; most of the time.
Which is great if you simply want to look at an extension, without having to hop all over the place trying to figure out how you enable it, or disable it if it stinks.

### Post by grahammechanical on 2014-09-14
You can not get rid of Unity by installing Gnome 3 shell on Ubuntu. Unity would still be there. But Unity is not present on Ubuntu Gnome because Ubuntu Gnome uses Gnome 3 shell instead of Unity. Ubuntu Gnome is what Ubuntu would look like if the developers had just accepted everything coming out of Gnome organisation instead of developing Unity.

Unity uses APIs of Gnome shell. You will see similarities between Gnome 3 shell and Unity. There is a similar kind of launcher on the left side. A similar kind of Dash. Unity would not work the way it does if the foundation was not there already in Gnome 3 DE.

Ubuntu Gnome is modified by installing Extensions. I have never done that except with installing Gnome Classic just to see what it looked like. Otherwise I cannot say how and to what extent Gnome 3 shell can be modified.

Why don't you download a Ubuntu Gnome ISO image and run a live session. then you will see for yourself. 

Regards.
