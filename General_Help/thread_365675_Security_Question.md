### Post by dcstar on 2007-02-20
> **skoalman88 said:**

Use HTTPS for as many web sites as you can, then all traffic passing through the router is encrypted and useless to anyone collecting it.

There are Anonymous web browsing services that you can sign up for so all your traffic goes to them using HTTPS, and then the "normal" HTTP traffic from their server uses this secure transport to you.
