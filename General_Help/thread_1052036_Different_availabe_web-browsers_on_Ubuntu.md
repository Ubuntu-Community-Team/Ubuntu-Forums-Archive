### Post by mb_webguy on 2009-01-27
Here's the link for the [User Agent Switcher]("https://addons.mozilla.org/en-US/firefox/addon/59") extension for Firefox.  However, just because the website now thinks you're using IE, it doesn't necessarily mean you'll be able to use the site.  A few silly websites stubbornly refuse to follow standards and use ActiveX controls to access their site content.  Only IE can access websites that use ActiveX controls, so on Linux we're pretty much limited to running IE under Wine for those sites.  [IEs4Linux]("http://www.tatanka.com.br/ies4linux/page/Main_Page") is the easiest way to go.
