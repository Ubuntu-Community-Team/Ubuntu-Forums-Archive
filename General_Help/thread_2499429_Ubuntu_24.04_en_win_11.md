### Post by yancek on 2024-07-27
When you booted windows 11, were any updates done?  Some updates will set windows to first boot in the BIOS so have you verified which OS is set to first boot?  If that is not the problem, you might try running boot repair to get information to post here for help.  Use the 2nd option explained at the site below and also select to Create BootInfo Summary and do NOT try to make any repairs but post the link you get here.

 [https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)
