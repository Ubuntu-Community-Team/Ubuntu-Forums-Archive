### Post by aquavitae on 2007-08-22
Ubuntu is a linux distribution.  There is an operating system called linux that some people have packaged, put on nice cds, etc and called ubuntu.  There are many others (e.g. mandriva, slackware, debian) that are basically the same, just packaged differently and by different people.  To make things easy - for "ubuntu" read "linux", and vice versa.  Installing ubuntu is really easy, much, much easier than windows.  Just put the live cd in ad install.  You can even use it while it is installing!  Then once you have the system running try installing ut04.  Btw, i understand you have two computers, one running windows, and a p4 you want to use for games?  Why not use the windows one for games and the p4 for your work?  A lot of games only run in windows and its not always easy to set them up in linux.  On the other hand if you need normal desktop apps (e.g. web browser, email, etc) then linux is the way to go.  Just a suggestion.

Its not that scary!  You can download a disk from the ubuntu website ([http://www.ubuntu.com/getubuntu]("http://www.ubuntu.com/getubuntu")).  Then burn it.  Alternately, you can ask the ubuntu people to mail you one free of charge.  When you get the disk, boot off it.  It will load a complete, running ubuntu system straight off the disk, without touching your hard drive, so you can have a look and decide if you want it before installing. Theres an "install" icon on the desktop you double click to install.  During the installation it will ask how you want to "partition the drive", i.e. what you want to do with windows.  You have the option of either using the whole drive (i.e. formatting it and installing ubuntu over whatever was there before) or installing next to windows.  The entire install takes about 30min, then reboot, and the new ubuntu will load.

There are ways of installing it in a virtual machine within windows, but I don't see the point of that if you want to use it as a game server.  Virtual machines are generally considerably slower than directly installing it.

Drives in linux are not the same as in windows.  Most of them should be installed when you install ubuntu.  The only ones I know of that aren't automatically installed are drivers for ati and nvidia graphics cards, but there are a couple of really simple guides on the ubuntu wiki on how to install them.

> god my p4 might never work right again..What precisely are you worried about?  Linux cannot damage your hardware, so even if it doesnt work, you can wipe ubuntu and reinstall xp.

I hope this helps!  If you have any specific concerns I'll do what I can to help.
