### Post by Dejai on 2009-04-03
SSID is short for service set identifier, it can be thought of in simple terms as your networks name because essentially it is a name that identifies a wireless network. (sourced from [http://www.webopedia.com/TERM/S/SSID.html](http://www.webopedia.com/TERM/S/SSID.html))

MAC Address: Short for Media Access Control address, a hardware address that uniquely identifies each node(desktop, pda, laptop etc) of a network.(sourced from [http://www.webopedia.com/TERM/M/MAC_address.html](http://www.webopedia.com/TERM/M/MAC_address.html)) 

As for a wireless network check the following link:
[https://help.ubuntu.com/community/WifiDocs/WiFiHowTo](https://help.ubuntu.com/community/WifiDocs/WiFiHowTo)

Hope this helps
