### Post by kamstrup on 2005-03-14
As far as I know you have two choices. Namely

[http://aria-rpm.sourceforge.net/](http://aria-rpm.sourceforge.net/)

[http://gnome.org/projects/gwget/](http://gnome.org/projects/gwget/)

Aria doesn't seem to be developed anymore, but should be quite stable as my Googling could tell. It is in Hoary Universe.

GWGet is also in the repos but it is an old version you'll find there... You'd probably be better of by building the latest release yourself.

If you ask me I'd definitely go for gwget!

Cheers!
