### Post by Elfy on 2014-09-29
assuming that you have your irc nick sorted and all passworded up 

start pidgin - Ctrl+a to manage accounts

nick in username box
server in server box - irc.freenode.net
password in password box
remember password if you want
save

If you want to use an irc client - I use either hexchat (got 2 bugs that annoy *me* at the moment) or xchat

[https://help.ubuntu.com/community/XChatHowto](https://help.ubuntu.com/community/XChatHowto)
