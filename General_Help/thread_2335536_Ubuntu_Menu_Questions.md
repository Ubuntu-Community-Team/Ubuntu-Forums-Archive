### Post by DogMatix on 2016-08-29
If you are talking about the Unity Launcher on the left of the desktop it can be made to auto-hide by opening System Settings > Appearance > then choose the 'Behaviour' tab and checking the 'Auto-hide the launcher' option. The size of the Launcher icons can also be adjusted in the 'Look' tab of the Appearance settings. When a program is launched it's icon appears in the launcher and by right clicking on the icon it can be locked to the launcher so it can be launched from there after it is closed.

### Post by DogMatix on 2016-08-29
In the Launcher click the top icon then in the search box that opens type Gedit then when the icon appears in the search results click it to open Gedit (NB: Gedit's icon will show labelled as 'Text Editor'.) Gedits icon should now appear in the Launcher. Right-click on the icon and select 'Lock to launcher'.
