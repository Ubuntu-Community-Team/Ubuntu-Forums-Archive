### Post by Beamvr6 on 2007-08-31
Some additional info I just found out. [:)]

In Firefox / Swiftfox you can set the app to start with the filetype.

If you can start the app from the menu (Applications > Internet > Azureus as an example) you can find both the path and the name of the app by going into the menu editor (right click Applications > Edit Menus > select Internet > right click Azureus > select Properties).

Back in Firefox / Swiftfox browse through the path and select the app. Easy.

This is for Gnome btw.
