### Post by u-slayer on 2008-03-26
I have some scripts saved as text files that I like to edit with gedit. However, whenever I click on the them, I get this annoying message:

```

Do you want to run "script", or display its contents?
```

I always want to display it in gedit. How do I make this the default?

---

### Post by munkyeetr on 2008-03-26
In Nautilus do the following:

1) **Edit** -> **Preferences**
2) On the **Behavior** tab, under *Executable Text Files* select **View executable text files when they are opened**
