### Post by mittwit on 2012-10-23
When you have a terminal window open and you move your mouse up to the top left of the screen do the menu items appear? If so then click *terminal* and if you see options like 80x24 try selecting one of these.
It may also help to select *help > about* to check which terminal program you are running.

### Post by 3rdalbum on 2012-10-23
> **fireball81577 said:**
> mouse is not active when i haver the terminal open.

Instead of pressing Control-Alt-F1 to open a terminal, press Control-Alt-T (or run the Terminal program from the Dash).

Once you have one terminal window on the screen, you can middle-click the Terminal icon again to bring up more.
