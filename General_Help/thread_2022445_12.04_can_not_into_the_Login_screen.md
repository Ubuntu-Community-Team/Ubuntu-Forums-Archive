### Post by Redblade20XX on 2012-07-11
Try and see if you can get into a tty. 

On the blank screen, press Ctrl + Alt + F1 keys and you should get into a terminal.

If you get into the tty, login with your user name and password. Then type in the terminal, 

```
cat /var/log/Xorg.0.log | grep EE
```

and post it so we can see.

- Red
