### Post by Jucato on 2006-12-06
In Edgy: K Menu -> System Settings -> Keyboard & Mouse -> Keyboard Shortcuts
In Dapper: K Menu -> System Settings -> Regional & Accessibility -> Keyboard Shortcuts

Shortcut Schemes tab -> Global Shortcuts sub-tab -> Desktop Switching group
