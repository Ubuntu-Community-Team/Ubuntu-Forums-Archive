### Post by hawthornso23 on 2012-03-04
I have all my important research in an svn repository. That way I can access and work on it from home - at work - on my laptop - or indeed from any other computer I might find myself using anywhere in the world. Not sure how you'd go about getting that sort of thing set up as a student though. As an academic I just asked computer support and they set one up for me.
I only use linux machines, but with this setup you can synch to any machine. You just need a subversion client to run on the machine in question. It is also the BEST way to do joint research - no more problems with trying to resolve incompatible versions between coauthors.
