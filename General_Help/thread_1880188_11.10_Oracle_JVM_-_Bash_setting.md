### Post by lechien73 on 2011-11-13
I would have thought that the command you're running sets the environment variable that tells Java where its home folder is, rather than tells Ubuntu where to find your Java binaries.

I think you need to add the /usr/java/jre1.6.0_29/bin directory to your path, so try adding the following line to .bashrc

```
export PATH=${PATH}:/usr/java/jre1.6.0_29/bin
```

You could try running it in a terminal window first, but Ubuntu should then find your Java binaries.
