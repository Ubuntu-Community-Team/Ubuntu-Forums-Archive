### Post by Rallg on 2009-12-19
When Firefox loads in karmic, it shows a Google custom search page.

I search for my own name (easy to find) and the custom search does not show either my web site, nor the Google site page where I keep my E-mail address! Yet these are the top 2 results using ordinary Google search. I am referring to my true name, not the handle I use here.

With the custom search, the top ten pages that refer to me actually link to other places where my name happens to be mentioned on the page. Some of them are very obscure, so it is not as if the custom search is only picking out highly ranked sites.

So, it appears that the custom search is far out of date. How can this be? Or, does the custom search add extra (hidden) terms that distort the results?

EDIT: Suppose I do the custom search, getting outdated results. Then, I manually edit the URL query string (which now appears in the browser location bar), and remove the reference to "partner-pub" in the search string. When I reload with the changed query string, I get the expected (up to date) results. Something in the "partner-pub" causes Google to return the outdated results.
