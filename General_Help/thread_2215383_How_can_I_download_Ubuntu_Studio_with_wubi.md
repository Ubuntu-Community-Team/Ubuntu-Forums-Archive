### Post by Impavidus on 2014-04-06
There was a time when I didn't know how to partition. The next day I installed Ubuntu. It isn't too hard.

The basic idea is to use Windows tools to create unallocated space on your hard drive, boot from the live disk and choose to install Ubuntu alongside Windows. The installer handles the remaining partitioning. The only caveat is that many Win7 systems use all 4 primary partitions available on MBR partitioned drives, so you'd have to delete one partition that doesn't contain anything useful or convert one partition into a logical partition. There's plenty of documentation online. If you've any question, feel free to ask.
