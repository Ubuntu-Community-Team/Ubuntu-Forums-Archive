### Post by Hugh971 on 2011-03-13
Hi,

If you have a limited number of resolutions available within System > Preferences > Monitors you probably don't have a graphics card driver enabled.

Check under System > Administration > Additional Drivers and see if there's anything that can be enabled. If not you'll need to install the driver manually.
