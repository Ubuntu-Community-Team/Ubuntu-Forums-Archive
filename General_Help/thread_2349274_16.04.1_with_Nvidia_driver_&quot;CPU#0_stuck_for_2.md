### Post by Kris_M on 2017-01-12
solved. I added "nomodeset" to the UBUNTU item using Grub Customizer.  I tested it first by doing a temp edit at the grub menu.
I dual boot Ubuntu(16.04.1) and win7.
So far so good but I will test using the TV tomorrow.  At least I got rid of the delay!  Using 367 I think but it would probably work for any of them.
Stolen from someone but it's on my win7 login FF so will try to check it and give credit.
credit:  [https://ubuntuforums.org/showthread.php?t=1466577](https://ubuntuforums.org/showthread.php?t=1466577)  post #9  Do not "press enter and then b" - do a Ctrl-x

code for the first menu item "UBUNTU" :
```
recordfail
load_video
gfxmode $linux_gfx_mode
insmod gzio
if [ x$grub_platform = xxen ]; then insmod xzio; insmod lzopio; fi
insmod part_msdos
insmod ext2
set root='hd0,msdos7'
if [ x$feature_platform_search_hint = xy ]; then
  search --no-floppy --fs-uuid --set=root --hint-bios=hd0,msdos7 --hint-efi=hd0,msdos7 --hint-baremetal=ahci0,msdos7  14f8bc13-3090-4bbf-96c0-1f30ea3ec99a
else
  search --no-floppy --fs-uuid --set=root 14f8bc13-3090-4bbf-96c0-1f30ea3ec99a
fi
linux    /boot/vmlinuz-4.4.0-59-generic root=UUID=14f8bc13-3090-4bbf-96c0-1f30ea3ec99a ro  quiet splash $vt_handoff nomodeset
initrd    /boot/initrd.img-4.4.0-59-generic

```
