### Post by kurt18947 on 2013-08-21
You can do a 'real' install to a USB flash drive just like you do to a hard drive.  It won't be as fast as a hard drive of course but for simple stuff like web browsing, it works reasonably well.  I used a store brand USB drive, I expect a better quality drive with stated read/write speeds would be better.  The advantage of a 'real' install as opposed to a live install is a 'real' install can be updated just like a hard drive.  A live install cannot in my experience.  One or two updates might not cause problems but  a live install will die from updates eventually.  Here are a couple  thread with useful links:

[http://ubuntuforums.org/showthread.php?t=2133067](http://ubuntuforums.org/showthread.php?t=2133067)

[http://ubuntuforums.org/showthread.php?t=2164440](http://ubuntuforums.org/showthread.php?t=2164440)
