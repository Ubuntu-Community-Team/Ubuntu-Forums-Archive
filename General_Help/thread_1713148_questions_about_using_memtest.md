### Post by ajgreeny on 2011-03-23
Not that I'm aware of, but why do you need it?  Just run memtest overnight and then in the morning look at the bottom line to see if there are any error numbers noted.  If there are you have a problem with one of your memory modules; it's as simple as that.

PS:  You must run memtest for an extended period, hence my overnight suggestion, in order to get a meaningful report from it.
