### Post by mchatel on 2005-09-21
To run CLAM AV...
From a terminal window, use the command CLAMSCAN.  There are various options of course, you can use with the command.  
ex.  to scan whole filesystem, one could use...

clamscan -r /          
-r recursive scan of subdirectories...  in this case, / (root) and under...

[www.ubuntuguide.org](www.ubuntuguide.org) has some info on various tasks with CLAM AV.  You might want to check there (however, today as I type this, the site seems to be down - hopefully it will be back soon).
