### Post by JBA2337 on 2009-03-12
You could also do a complete backup of MS Windows, using the utility that is part of Windows: Automated System Recovery. To run this program in Windows, click on the icon for the Windows backup program (ntbackup.exe), or you could run ntbackup.exe from a command prompt.
When the program starts, choose Automatic System Recovery. To use this program you will need 1 floppy disk, and another drive to back up to. The compressed backup will take up about 18-20 GB, depending on the size of your Windows installation.

If you don't like that program, you could do a complete image of your hard drive using a drive imaging program such as Norton Ghost. Their are also several free imaging programs that you could use. Google "free imaging".

JBA
