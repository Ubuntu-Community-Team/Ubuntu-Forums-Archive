### Post by ELD on 2012-01-19
In gnome when you double click an executable .sh file it will ask if you want to run, run in terminal or edit while KDE will just run it.

Is there a way to get it to run it rather than ask on double click?

---

### Post by mcduck on 2012-01-19
Sure. Open the file manager, go to Edit->Preferences, and on the Behavior-tab under "Executable Text Files"-section set the option to "Run executable text files when opened".
