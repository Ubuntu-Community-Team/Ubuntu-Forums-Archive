### Post by mcduck on 2011-02-04
You can make Conky transparent directly in Conky's own settings.

What comes to the transparency settings for any window in Compiz, you need to use the "*Opacity, Brightness and Saturation"-plugin* for that. But keep in mind that it affects the whole window, including all contents. So you can't use Compiz to make only background of something transparent.
