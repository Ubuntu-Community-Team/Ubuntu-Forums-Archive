### Post by VCoolio on 2010-01-04
Three options I can think of:
1. You haven't set your default browser properly; check in system > preferences > default/preferred applications 
2. The app you use doesn't check default ubuntu browser ("gnome-open") but default x-www-browser ("xdg-open"); install galternatives and use that to set your x-www-browser
3. The app uses own settings to open links; for pidgin e.g. I recall you can set it in preferences somewhere; the same for some terminals
