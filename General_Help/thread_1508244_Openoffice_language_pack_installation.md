### Post by Frogs Hair on 2010-06-13
Tools> language > you will see the words ( for selection ) > then it will say  (English) > drop down to ( More ) and click , a window will open . In the center there is a language selection menu , select the one you need and click ok on the bottom of the window . I think you may have to change the language support settings in the control center as well . If you don't have the control center installed click system on the main menu and check the control center box.

### Post by Hagar Delest on 2010-06-17
> **drpjkurian said:**
> Now you can change the language option in Open office 
Tools--> language
No, avoid that because this is a direct formatting, not robust. It's the same as using the switch in the status bar.

Set the language in the styles, starting with the Default paragraph style. Or in Tools>Options>Language settings>Languages and default language.
