### Post by nerdtron on 2015-07-12
None. Since you wiped everything, just boot into the Live CD/USB of Xubuntu and start testing your computer devices, wifi, ethernet, video, etc. If everything works fine, start the Xubuntu installer. You should be presented with the screen to which hard drive to install Xubuntu. The installer should detect the hard drive and create partitions automatically. 
If you have other problems on the installation just ask the forums.
