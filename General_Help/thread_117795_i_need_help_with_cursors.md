### Post by Sutekh on 2006-01-19
To install a particular cursor themes, download the theme as a .tar.gz file.  A good place to look is

[GNOME-Look.org - CursorThemes]("http://www.gnome-look.org/index.php?xcontentmode=36")

Then open up System->Preferences->Theme, choose 'Install Theme' and find the cursor theme tar.gz file, and open it.  

It will tell you that the icon theme was installed, but this is really it was the cursor theme that was installed. 

Then close the theme selector (don't click "Theme Details" and pick your cursor set, it won't work).

To select the cursor theme that you isntalled, open up System->Preferences->Mouse and choose the one you want from there.
