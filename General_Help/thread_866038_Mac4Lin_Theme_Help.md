### Post by DJ_Peng on 2008-07-21
Have you checked the [PDF documentation]("http://downloads.sourceforge.net/mac4lin/Mac4Lin_Documentation_2.pdf?modtime=1198008186&big_mirror=0") on the [Mac4Lin project website]("http://sourceforge.net/projects/mac4lin/")? You should have a theme labeled *Custom...* that should contain the theme components, just waiting to be saved. The fact that it doesn't save with a theme name is a known issue and should be resolved when Mac4Lin 1.0 comes out later this year.

You can also get support in the [Mac4Lin topic]("http://ubuntuforums.org/showthread.php?t=555373") on these forums.
