It sounds to me you chose the wrong keyboard lay-out at re-install.
Go to:
System - Preferences - Keyboard
Choose Tab "Layouts"
Does it say: USA? If so, do you have more choices?
If you have more choices make sure USA is the Default one by clicking the small circle at the end of the line.
If you don't have USA listed, click on Add and add it.
You can throw all others away (if not needed of course). I only have the USA one listed, without the default circle being chosen. It's the only choice I have so it is the default one.
