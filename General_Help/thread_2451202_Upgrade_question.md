### Post by TheFu on 2020-09-29
A320M is a 2-3 generations older than that CPU, so you'll definitely need firmware updates, probably before it will even boot. This may require getting an A-series CPU first.  That was an issue with prior generations.  I have a 2600 and paired it with a Asus B450.  

Do you really want to pair a $55 motherboard with a $200+ CPU?  For that MB, I'd probably pair an older Ryzen 3 CPU, sub-$100. 

The RAM speed matters greatly for Ryzen performance.  There are articles all over the internet about how important that is.  I'd want at least 3000 Mhz RAM. Being stuck with slow RAM is one of those things that average people don't notice, but if the system performance is 30% less, you'll become unhappy in a year or two.

Of course, all of this depends on the price paid.  If that machine is $250, great!  Much over that and I'd pass.

Would the Ryzen 2400g be worth considering? You'd need a CPU cooler, but would save a GPU and have better GPU driver support thanks to AMD. I've had nvidia drop driver support for older GPUs after 16.04. It is really frustrating when that happens for a working GPU.  AMD has been working with F/LOSS developers so those cards are supported well over 10 yrs.
