### Post by BillyBoa on 2011-11-04
Looks like you have a connection problem. Check your software sources (from the upper left corner push the icon and then write soft.. and choose software sources). From the tab Ubuntu software go to Download from and choose -other-. Then choose -select server- and wait till the best server is selected. After that try again to install programs from the center.
