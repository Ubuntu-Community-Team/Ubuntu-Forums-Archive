I love Alltray.  It is a fabulous little app and my computer would be clogged and cluttered with out it.

My question is:

Is there a way to start applications at the computer's startup, minimized to the tray with Alltray via a single command that i can enter in the sessions window?

I appreciate any response.

Using Gutsy Tribe 3

---

### Post by walkerk on 2007-07-23
```
alltray /location/to/application
```
or
```
alltray firefox
```

just add the above System>Applications>Sessions and you're set :)

this works.. i just tested it
