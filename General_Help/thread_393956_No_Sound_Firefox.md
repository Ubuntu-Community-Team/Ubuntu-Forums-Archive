### Post by ArieT on 2007-03-26
I don't know why no one has answered your first post about Flash 9. It's easy to [find](http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_install_Flash_Player_.28Macromedia_Flash.29_Plug-in_for_Mozilla_Firefox)

```

Note: if sound doesn't work in Flash Player (for example on YouTube):

sudo aptitude install alsa-oss
gksudo gedit /etc/firefox/firefoxrc

Change:

FIREFOX_DSP=""

To:

FIREFOX_DSP="aoss"

    * Restart Mozilla Firefox. Now sound should work in Flash Player. 


```

### Post by aysiu on 2007-03-26
No worries.

Can you try something else, since none of those suggestions seem to work?

1. Close Firefox
2. Back up your old Firefox profile ```
mv ~/.mozilla ~/.mozilla.preferred
```
3. Launch Firefox again. See if sound works.
4. If not, bring your old Firefox profile back ```
rm -rf ~/.mozilla
mv ~/.mozilla.preferred ~/.mozilla
```
