### Post by marshalium on 2008-10-18
You can run Windows in a virtual machine on Linux.

Two programs that I have used are:

VMWare Player: [http://www.vmware.com/products/player/](http://www.vmware.com/products/player/)
VirtualBox: [http://www.virtualbox.org/](http://www.virtualbox.org/)


VirtualBox is available in the Ubuntu repositories. You can install it with:
```
sudo apt-get install virtualbox-ose
```
