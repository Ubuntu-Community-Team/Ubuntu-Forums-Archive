### Post by ad_267 on 2008-08-12
Edit: I assumed you installed using Wubi, it will be a bit different if you used a virtual machine.

So do you want to keep windows?

You will want to back up all of your data from in Ubuntu that you want to keep. You can back it up onto the Windows partition if you're keeping but because you're changing partitions around it's a good idea to back your valuable data up on something else.

From what I understand, you can uninstall Ubuntu using Add/Remove Applications in Windows.

Then download the Ubuntu 64bit cd and burn the image to a cd using Nero or whatever you have.

Defragment the hard drive from windows if you want to keep Windows.

Then restart the computer with the cd in the drive and choose the option to try Ubuntu, then you can run the installer from the shortcut on the desktop. If you want to get rid of Windows you can choose install and take up the entire hard drive. If you want to keep windows you can choose the option to resize the main partition to make room for Ubuntu.
