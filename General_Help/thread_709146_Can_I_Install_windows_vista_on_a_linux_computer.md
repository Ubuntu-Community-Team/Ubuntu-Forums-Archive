### Post by anaconda on 2008-02-27
You can also install Vista or XP to a virtual machine under ubuntu..

No need to reboot to get to Vista.

The only drawback is that 3D acceleration wont work in virtual machines, so most 3D games wont work in virtual vista... "normal" programs will work.
