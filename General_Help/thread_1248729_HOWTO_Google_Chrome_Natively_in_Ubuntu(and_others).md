### Post by CRAY-4 on 2009-08-24
[FONT="Arial"]Google started developing google chrome 2.0 a few month ago, and they decided to make a verison for mac and linux as well, right now the browser is still in development, but is very fast and stable(compared to ffox), and even has snappy flash support

anyways, this is how to get, 

for 32-bit systems: [http://www.google.com/chrome/intl/en/eula_dev.html?dl=unstable_i386_deb](http://www.google.com/chrome/intl/en/eula_dev.html?dl=unstable_i386_deb)

for 64-bit systems: [http://www.google.com/chrome/intl/en/eula_dev.html?dl=unstable_amd64_deb](http://www.google.com/chrome/intl/en/eula_dev.html?dl=unstable_amd64_deb)[/FONT]

also, to get flash support create a launcher with the following command "google-chrome --enable-plugins"

installing will also add repo so you can always update
