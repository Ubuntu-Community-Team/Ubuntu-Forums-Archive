### Post by stinkeye on 2010-12-15
There is a package in the default repos
```
dict-moby-thesaurus
```
Don't know if this applies to stardict though.

There is also **Artha** which is an off-line English thesaurus.
I use and like this one because it has a shortcut key to show highlighted word meanings in the notification bubble.
