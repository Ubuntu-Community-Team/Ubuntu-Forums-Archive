### Post by por100pre1 on 2007-09-26
In KDE go to:

```
kcontrol
```

Select **Appearance and Themes> GTK Styles and Fonts> GTK Styles**

Select **Use another style** and then select the same theme you use in GNOME.

Right below select the same font you use in GNOME.
