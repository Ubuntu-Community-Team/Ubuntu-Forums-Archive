### Post by Lhademmor on 2007-10-09
Hi all
I'm trying to make a list of features that would be neat to have in Exaile, and thought that I'd look here for any opinions.

Trying to imitate Amarok is serious business, but I think Exaile Team is doing a great job, and an Amarok-awesome player on GNOME would be a tremendous boost for all GNOME-users :)

I'll start out myself: I think it would be great with
I) A colums or something with a nice %-wise "similar artists" with pictures like the one used in Banshee. (The one in Amarok is text-only)
II) Make the rating-ability easier - like Amarok's
III) Artist/Album-tabs should unfold when single-clicking on them. As it is now, you have to double-click on artist-tabs, but if you double-click on album-tabs, you add the whole album to the current playlist. Consistency would be nice, either way.
IV) The Album Art manager should be more like Amarok's: Ability to only scan for a single album's cover, ability to remove a cover, if a wrong one was found, ability to manually search using an alternate albumtitle for the same cover.
V) More eye-candy :P - I'm bananas about eye-candy Examples could be:
V-I) Enlarge the gaim-buddy-like icons representing artists and instead have them be artist pictures, maybe fetched from last.fm
V-II) Same with small album-icons --> covers. (Amarok already has this feature).
VI) Magnatune support perhaps?

Now you...
