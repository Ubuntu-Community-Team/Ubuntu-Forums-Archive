### Post by NERDMAN! on 2011-06-09
ive heard of hard disk partitioning, but RAM? interesting idea but as i recall random access memory is dumped after every shutdown, if you were to partition the RAM, logically it would only stay partitioned for that session, if your having issues with minecraft fps rates i suggest the optimine mod instead. it helps alot on machines that dont have quite enough resources, i know my laptop cant play the game without it.
you can find details on it here: [http://www.minecraftforum.net/topic/249637-166-optifog-and-optimine-fps-boost/page__hl__optimine+166](http://www.minecraftforum.net/topic/249637-166-optifog-and-optimine-fps-boost/page__hl__optimine+166)
