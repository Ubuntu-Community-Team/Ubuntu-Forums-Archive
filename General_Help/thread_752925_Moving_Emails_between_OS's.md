### Post by Koybe on 2008-04-12
Outlook does not exist in Ubuntu. It depends if you need the calendar/tasks things. If it's just emailing and adress book, I would suggest.
- Install Mozilla Thunderbird in Windows, at start, it will ask you to import everything from Outlook. Easy step.
- Launch Thundebird use Address Book -> Tools -> export, save it to a file
- Install Mozilla Thunderbird in Ubuntu
- Install ImportExportTools Extension  [http://nic-nac-project.de/~kaosmos/mboximport-en.html]("http://nic-nac-project.de/%7Ekaosmos/mboximport-en.html"))
- use Tools -> import/export in mbox/eml format -> import mbox, then point to your Windows Thunderbird Profile (Doc&Settings\youruser\application data\mozilla\thuderbird\...)
- Import your address book from the file you saved before.

A link in the forum : [http://ubuntuforums.org/showthread.php?t=91687](http://ubuntuforums.org/showthread.php?t=91687)
