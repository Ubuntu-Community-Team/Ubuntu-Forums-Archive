### Post by pellgarlic on 2010-12-17
hi. if you're using gnome window manager (the default on ubuntu, kubuntu = kde, xubuntu = xfce, and there are some others too) there is an icon in the taskbar for the "network manager". looks like window's "network connections" icon - two monitors side-by-side. this icon has both a left-click and a right-click menu. if you dig about in there, you should be able to find what you need. ask back if you need more info.
