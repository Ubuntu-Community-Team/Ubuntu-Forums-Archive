### Post by Arby on 2006-11-01
If there are bugs then they will be addressed at some point. But first they have to be reported. If you are interested in the process then it is all managed from here [https://launchpad.net/distros/ubuntu]("https://launchpad.net/distros/ubuntu") If you have discovered a genuine bug for your AMD64 system report go report it on the edgy bugs page that is linked from above (Edgy->bugs).

The general idea then is that if someone else can replicate the same bug they 'confirm' that bug. This increases the likelihood the bug will be addressed soon. I'm not sure what the process is for deciding the priority of bugs but if it gets reported then confirmed there's a good chance it will be dealt with sooner or later.

So step 1 would be, go report your bug:) There's nothing relating to it on the Edgy bugs page and if it ain't there it won't get addressed. There's a rough guide here[http://ubuntuforums.org/showthread.php?t=284595&highlight=bug+reports]("http://ubuntuforums.org/showthread.php?t=284595&highlight=bug+reports")on filing bugs
