### Post by Goober on 2007-01-28
I apologize if I am pointing something out which was already tried, but ... 

From my experience (with an HP), you shouldn't need OpenPrinting.org.  If you go to SYstem -> Administration -> Printing -> Add New Printer, and follow the steps, that should install your printer.  

With mine, I had to fiddle with the Printer settings to get it to work in Grayscale only.  You can do that by right-clicking on the Printer, and, well, fiddling with the settings if it doesn't work.
