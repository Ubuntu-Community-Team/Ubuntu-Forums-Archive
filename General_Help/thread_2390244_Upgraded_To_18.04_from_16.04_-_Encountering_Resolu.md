### Post by ajgreeny on 2018-04-27
Did you remove the proprietary nvidia driver before upgrading the system to 18.04?
You should have done so to avoid this sort of problem.

If you did not you will now probably need to completely remove the version you were using, and then install whatever version is now recommended by the Additional Drivers utility.
