### Post by gobbles414 on 2007-08-12
Hi b3n87,

There are three programs that appear when using the search term "stocks" in the Synapric repositories (System --> Administration). The one that I liked the best in testing was libfinance-yahooquote-perl. It will place a launcher called "Invest Chart" in the Office menu. The user interface is very simple. Not being a financial enthusiast myself, I'm not sure if it is a powerful enough tool for your grandfather or not.

But the second option, phpgroupware-stocks, installs a lot of extra stuff and it seems like it is a PAIN to configure. The other result is called qtstalker. It is actually designed for Ubuntu's cousin, Kubuntu. For me, the interface is very disorganized. Nevertheless, it appears to be a more powerful application that yahooquote.

Please try some or all of these apps and let me know what you think of them. If they don't work for your grandfather, we can expand our search to the internet.
