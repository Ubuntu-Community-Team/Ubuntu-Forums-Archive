### Post by Lars Noodén on 2012-08-23
MS-DOS and PC-DOS are based on (near rip-off) of CP/M not Unix, so there will be virtually no crossover between that and what you find on Linux.  You will find, as you progress, a lot of overlap between Linux and the other Unix-like systems.  And between the various Linuxes, there are very few differences.

That said, think of it as shell scripting and it might be easier to approach.  Learn to look up the manual pages for programs using [man](http://manpages.ubuntu.com/manpages/precise/en/man1/man.1posix.html)  For example,

```

man ls

```

One place to get started would be [Wooledge's Bash Guide](http://mywiki.wooledge.org/BashGuide),  Bash being the default shell for Ubuntu and many other systems.  There is also some Ubuntu community documentation: [https://help.ubuntu.com/community/Beginners/BashScripting](https://help.ubuntu.com/community/Beginners/BashScripting)

Don't expect to remember everything, don't be afraid to look things up or to ask questions.
