### Post by squirrel_chaser on 2006-03-17
I started using Dapper at work this week for my main desktop.  Last week I wrote some LDAP scripts in Python on the Dapper box.  So far no problems, but I keep my data on the server and have backup machines running Breezy and Windows.

I have Xgl running on the main box at work and Dapper seems a lot quicker even with Xgl.

Dapper seems stable enough but the greatest risk is that X breaks.  If you feel comfortable enough to boot into the system without X and get your data,  the risk is manageable.
