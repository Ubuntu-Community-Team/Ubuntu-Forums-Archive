### Post by Jimboted on 2010-12-17
For anyone experiencing the same problem (for example, it occurred when setting up evolution or easystroke - so I couldn't see the forward buttons etc) I have now solved the issue.

Click on the affected window press alt-F7, you can then use the mouse or arrow keys to move the window about the screen.

Jim
