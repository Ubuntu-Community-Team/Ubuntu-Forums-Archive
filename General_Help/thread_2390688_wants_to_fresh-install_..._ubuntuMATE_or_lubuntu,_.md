### Post by Rex Bouwense on 2018-05-01
Both have the Ubuntu core and use the same repositories.  They use different desktop environments and have different default applications.  Of all the Ubuntu flavors Lubuntu is the lightest, designed to use applications that are not resource intensive and have fewer dependencies.  An excellent choice for older or less powerful computers.  However, it functions with amazing speed on newer machines.  It is not as configurable as the other Ubuntu flavors and does not come with the bells and whistles that the others have.  So it depends upon what you want to do with your machine.  Any response that you receive here would be based upon the preference of the responder.  Try them both in a live environment and see which you prefer.
