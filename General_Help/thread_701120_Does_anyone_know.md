### Post by erginemr on 2008-02-19
Firefox already remembers the passwords, but you might have disabled this functionality by mistake. You should check that it is enabled inside 
**Firefox Menu -> Edit -> Preferences -> Security**
where you can also set a master password for further security.

As an alternative to the default Firefox, Opera ([www.opera.com](www.opera.com)), another popular and fast browser, also remembers your password via a technology called "wand". You can install it with:
```
sudo aptitude install opera
```

### Post by Herman on 2008-02-19
I let Firefox remember passwords for me too, but I have a habit of going 'Edit'-->'Preferences', and clicking 'Privacy' every now and again.
I click 'Show cookies', and see a huge list of cookies, some are from sites I approve of and others are cookies I don't want. 
I delete all the cookies for sites I don't recognize. 
Unfortunately, if I make a mistake while doing this and delete a cookie I meant to keep, I often have to retype a password to log into a website.

I use '[GPass: GNOME Password Manager]("http://www.google.com.au/url?sa=t&ct=res&cd=2&url=http%3A%2F%2Fprojects.netlab.jp%2Fgpass%2F&ei=trm6R8OAPKnSpgTW1-CmDQ&usg=AFQjCNFDsYPGOJVHWzd0RJhRNiPIhVUGaQ&sig2=rlF225iDDH_rBzkB_CfayA")' for long term password storage. With GPass my passwords are stored encrypted, but I can access them all in a few seconds as long as I know my GPass password. 
I have close to forty passwords stored in GPass, and some of those are for sites I don't visit very often, so they may not have been visited since the last time I installed Ubuntu. I include my .gpass directory in backups and carry it forward and paste it in my latest Ubuntu installation.  It's nice to be able to remember passwords for sites I haven't logged into for ages when the need suddenly arises.

GPass is easily installable in Ubuntu by going 'Applications'-->'Add/Remove Programs', and typing 'GPass' into the search bar.
GPass does not directly help you log into websites, but it does give you somewhere to look up any password if you have forgotten it.

Regards, Herman :)
