### Post by yager on 2007-01-08
In Ubuntu, you would enter ```
sudo ..... 
```

then you enter the password of the first user that was added during installation.  If you are the only user, the password is your password.

Give this a try.  If you still have issues installing a program, check the Wiki tab located in the upper left corner and search for what you would like to install for detailed walkthroughs on some of the more complicated programs.

For most applications, you can also go to System>Administration>Synaptic Package Manager.  Use the search tool to locate the program in question.  If it is not available, go to the System>repositories menu in the Package Manager and check the multiverse and restricted repositories and try your search again after Package Manager has finished adding the available programs to the list.

If neither of these have worked for you, tell us what program you are trying to install and then someone can point you to a place that has the exact instructions or may whip up a set just for you.
