### Post by Stupendoussteve on 2009-04-10
You can just run:
> sudo apt-get install xnest
gdmthemetester console Human

Then take a screenshot of that window the normal way.

The syntax is gdmthemetester <environment> <theme>. Just run gdmthemetester with no arguments to see an example.
