### Post by DuckHook on 2017-11-27
Google Docs is a different app from LibreOffice Calc and when translating between apps, it is quite common to lose formatting, especially if that formatting is unusual or complex. Since there are an infinite number of ways to format documents and we don't know how you've formatted yours, it's impossible to help you with any specific document. However, I would point out that you need not translate your document into a Google Docs document. Keep it on Google Drive and it will remain a LibreOffice Calc document. You will still have your backups—just not the Google Docs translation.
