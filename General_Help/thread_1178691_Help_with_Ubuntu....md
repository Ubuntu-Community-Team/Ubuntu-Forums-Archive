### Post by lindsay7 on 2009-06-04
You should be able to use your wireless card with the live cd to see if it all is going to work. Another way  to do this is to install Ubuntu with Wubi which you can download and it will install under windows.  You can also look at the install disk using windows and there should be a Wubi install there too. If it works under windows using Wubi, you can delete it in windows using the control panel remove programs and then do the full dual boot install with the live cd. This way you will know for sure if it will work with your wireless card.
