### Post by Wyrmboy on 2008-04-05
Go the the Add/Remove button under Applications tab 

click preferences and go to updates check the box under unsupported updates and then type this in the terminal  

 sudo apt-get install avant-window-navigator awn-manager

Make sure you click reload after checking the box
