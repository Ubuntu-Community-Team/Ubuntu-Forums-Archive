### Post by Xenix on 2006-11-25
OK, I have solved this issue.  There were some extra packages that I needed to download, in order for it to compile correctly.
EDIT - Just for reference, the extra packages needed are:- fontforge libgl1-mesa-dev libgtk1.2 xorg-dev   ..... So in order to compile Wine correctly, one will need these all up:- build-essential bison flex fontforge libgl1-mesa-dev libgtk1.2 xorg-dev
