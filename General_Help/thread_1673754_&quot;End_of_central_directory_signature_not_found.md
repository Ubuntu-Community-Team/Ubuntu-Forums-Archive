### Post by LewRockwellFAN on 2011-01-23
Try going into bios settings (usually accessed by holding some magic key like F2 or Esc at just the right time during the boot process. For my IBM mobo it is best to press it repeatedly while the blue IBM logo is up but this will vary with the bios and it's present settings. Watch for a message flashing on the screen during boot. It may not appear if the bios has been set to a faster booting procedure but even so, holding the right key at the right time will get you there. May take experimenting). Anyway, somewhere in the bios settings should be an option to let you change the boot order and let you put the CD drive first in the list. Unless it is a reeeeally old mobo. Do that. Then reboot. Good luck. If it were me, if you aren't short on hd space, I'd partition with the Windows disk and leave space for another partition for another OS and a third for data. Then if you want to try out other OSs you can do so without uninstalling windows.
