### Post by tamoneya on 2008-04-05
Gusty:
Visit this link: [http://www.mozilla.com/en-US/firefox/all-beta.html](http://www.mozilla.com/en-US/firefox/all-beta.html) and download the bz2 file
cd into the directory.
```
./firefox-bin
```

Hardy:
```
sudo apt-get update
sudo apt-get upgrade
```
