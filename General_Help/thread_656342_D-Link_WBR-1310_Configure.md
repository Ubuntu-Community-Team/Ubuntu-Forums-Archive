### Post by ARhere on 2008-01-02
The default password should be "admin", not the user name.

It does not matter what user name you put in as that is just used to save password information.

If all else fails, there should be a small, hard to see, reset button.  Using a ball point pen, _gently_ push in the button and hold for 5 seconds, or until the LED's reset.  That should reset all settings to hardware default.

-AR

### Post by Tainted42 on 2008-01-06
I have the same router and the OP is correct about the password. Your supposed to set the user name to admin with the password left blank.

I've googled around, because I have the same problem from time to time. It rejects the default user name and password (or if you've changed the user name and password, it will reject that.) There's something wrong with this entire line of routers. There are tons of other people that have the same problem. Every so often, the router needs to be reset in order to gain access to the control panel, it will reject every user name and password combination. I guess D-link pretty much refuses to admit that there's a problem with the router, because they act like nothing is wrong when you call in about it. 

Resetting the router is the only fix for this and your going to need to do it again. If you mess with the settings a lot and it would be a pain to have to change all the settings to your preferred liking, you may want to consider getting a new router, because this will happen a lot.
