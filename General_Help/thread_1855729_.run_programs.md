### Post by Frogs Hair on 2011-10-06
The easiest way to get the latest Nvidia drivers for Ubuntu is with this PPA. Run the following commands in the terminal . The drivers from the website are not tested with Ubuntu , but there are plenty of guides if you really want to install them .```
sudo add-apt-repository ppa:ubuntu-x-swat/x-updates
``` ```
sudo apt-get update
``` ```
sudo apt-get install nvidia-current nvidia-current-modaliases nvidia-settings
```
