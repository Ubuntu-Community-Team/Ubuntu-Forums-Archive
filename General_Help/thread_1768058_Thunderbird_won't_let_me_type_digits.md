### Post by bovender on 2011-05-26
Hi all,

I am experiencing a strange, but annoying problem with Thunderbird. When composing a message, it sometimes refuses to let me type digits. I can copy and paste numbers, but entering digits using the keyboards just no longer works. 

This behavior never occurs right after starting Thunderbird. Usually everything is fine, but when I start to compose the n-th e-mail, suddenly it refuses the digits!

No other software exhibits this behavior.

My system is a Lenovo Thinkpad T61. The bug occurred in all versions of Ubuntu that I have used so far -- currently I'm running a fresh install of 11.04 Natty.

Any help would be appreciated.

### Post by bovender on 2013-08-10
To finally answer my own question -- and to help anybody who might experience the same problem:

The problem occurs when switching back and forth between the Thunderbird compose window and other windows by pressing ALT+TAB.

If Thunderbird won't accept any digits that you type, simply press ALT once in the compose window, and it will work again.
