### Post by PhilB on 2005-08-31
I had this problem , but it affected Konqerer too.
About two weeks back I changed from Mandriva to Ubuntu (having been using Mandrake/Mandriva and prior to that Suse for a couple of years). 
Anyway, my ubuntu was a new installation and I had installed the usual browser stuff-Flash, Java runtime etc, and noticed a variety of websites freezing. I thought it was a Java problem, but everything checked out ok. The annoying thing is, I use Crossover Office for various applications and Internet Explorer worked on these websites!.
Anyway being a new install, I decided the best thing would be to  reinstall Ubuntu and keep a note of any extra applications as they get downloaded from the repositories. So far so good, the websites are now working, I have Flash and J2re installed but so far nothing else that may affect the browser (ie no multimedia plugins).
If anything does get installed which affects these websites i will report back, but so far all I can suggest is starting afresh, but that may not be the best option for everyone.
