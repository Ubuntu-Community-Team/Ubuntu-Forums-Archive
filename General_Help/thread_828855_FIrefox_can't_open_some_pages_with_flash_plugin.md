### Post by y-lee on 2008-06-14
Sadly the video or game on [www.contra.gr/vReplays](www.contra.gr/vReplays) is a x-director file  (Macromedia shockwave), and as far as I can tell there is  no support for x-director on Linux. (Somebody correct me if I am wrong) And since Macromedia "owns" the format somebody would have to reverse engineer it to write one, since Macromedia is not supplying one or cooperating with the Linux community :(

My advice is to avoid sites like that and perhaps send them an e-mail explaining the disservice to the linux community they are doing by using technology like that on their web sites.

Other than that the only way to get it to work on linux is to use the crossover plugin. (It may be possible to get it to work with wine.) you can get it from [codeweavers ]("http://www.codeweavers.com/home/")
