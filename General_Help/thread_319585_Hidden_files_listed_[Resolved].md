### Post by clucas on 2006-12-15
Using a program such as The Gimp or OpenOffice, when I select "File" then "Open", hidden files (and folders) are shown along with the non-hidden ones. Why is this? In Nautilus I don't have "Show hidden files" active.

### Post by ciscosurfer on 2006-12-15
> **clucas said:**
> Using a program such as The Gimp or OpenOffice, when I select "File" then "Open", hidden files (and folders) are shown along with the non-hidden ones. Why is this? In Nautilus I don't have "Show hidden files" active.Within the GIMP open file dialog, you can right-click and choose Hide Hidden Files (or Show Hidden Files, to make them appear again.
