### Post by rene32 on 2006-10-29
My system freezes for 2(3) reasons here:

1) Startup: Blacklistes prism2_pci. Somehow that makes the system freeze at startup the first try. Then it works fine.

2) Network Manager tries to connect and freezes. Keyboard is frozen, mouse still working but not all applications remain usable. Solution: Uninstall NW or just de-activate the wifi.

3) In Dapper Skybe made the system freeze when any sound was played (incoming chat message e.g.). Solution: De-activate all Skype sounds. Seems to be fixed in Edgy, but who knows if this is the case for all systems.
