### Post by ewanm89 on 2008-03-11
To search in pidgin by just typing, the box comes up as you press the first key.

Status on contacts in kopete, Settings -> Appearance -> Contact List -> Detailed view.

1) Unfortunately the only way I know to sort groups in pidgin is by clicking and dragging them around, fortunately it remembers the order once you have done this task.

2) I don't think anyone has managed to reverse engineer the file transfer in the yahoo protocol yet, I would suggest using open protocols for this reason (Jabber/XMPP).

EDIT: Checked with Pidgin documentation and found this for file transfers: : > Yahoo when not using an HTTP proxy and when the remote end uses a libpurple client or Yahoo Messenger 6 for Windows (but sending is limited to an unknown file size)
