•sudo apt-get remove --purge bcmwl-kernel-source

•sudo apt-get install b43-fwcutter

•sudo apt-get install firmware-b43-lpphy-installer(this actually errored saying something about how it's dependencies b43 wasn't installed nor some other modules)

•sudo apt-get install --reinstall bcmwl

•echo "blacklist dcma" | sudo tee -a /etc/modprobe.d/blacklist.conf
