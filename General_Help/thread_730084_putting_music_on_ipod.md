### Post by gobbles414 on 2008-03-23

> amarok doesnt support mp3

Click [here]("https://help.ubuntu.com/community/RestrictedFormats") and [here]("https://help.ubuntu.com/community/RestrictedFormats/MP3") for more information about getting MP3 files to work correctly in Ubuntu.

You shouldn't have encountered any problems finding and installing Banshee. There are two places in Ubuntu where you can easily access the Banshee installer. The first is *Applications => Add/Remove*. If you can't find Banshee in Add/Remove, enable the viewing of all available programs using the drop-down menu in the upper right of the Add/Remove window. You can also install Banshee in Ubuntu by going *System => Administration => Synaptic Package Manager*.

Does this help?
