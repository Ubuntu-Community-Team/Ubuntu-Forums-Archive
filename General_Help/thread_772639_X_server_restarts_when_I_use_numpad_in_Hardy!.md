### Post by FIyero on 2008-04-28
I just upgraded to Ubuntu 8.04.  So far it has been smooth excepting one thing:  If I even touch the numpad on my keyboard, the X-server restarts.

This is very annoying as I use my numpad by habit quite a lot.  Has anyone else encountered this?  More importantly, can someone tell me how to fix this?

Thanks.

---

### Post by FIyero on 2008-04-28
Fixed the problem.  During the upgrade, MouseKeys had become enabled.  Disabling it stopped the restart from happening.
