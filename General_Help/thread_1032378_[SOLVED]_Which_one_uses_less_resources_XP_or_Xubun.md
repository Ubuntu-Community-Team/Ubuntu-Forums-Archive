### Post by northern lights on 2009-01-06
First off, you can't really compare two OSs without considering the type of applications/processes in the background. You won't ever run a clean XP or a clean Linux for a long time (i.e. the way it was right after installation) without adding additional software, some of which you autostart.

In general though, Xubuntu (i.e. a Linux with XFCE as its Desktop Environment) will be more resource efficient that XP. Especially a not recently installed XP.

### Post by HermanAB on 2009-01-06
Linux will subjectively feel faster than XP, for two reasons:
1. The Linux scheduler and memory manager are better, leading to smoother operation.
2. The Linux file systems are faster than NTFS, leading to faster program load times.

The actual throughput on processing intensive tasks will be the same though.

Herman
