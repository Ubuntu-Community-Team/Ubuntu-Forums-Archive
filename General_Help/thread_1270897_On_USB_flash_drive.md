### Post by P4man on 2009-09-20
If you want a "live usb" stick, then sure.  just go system > adminstration > USB startup disk creator. Point it to the ISO, and off you go.

If you want a less typical full/regular install on a USB drive, then, well, you'd need 2 USB sticks. Make one a live USB stick with the above program, boot it, and use it to do a full install on a second stick. It seems you can't do a regular install from an installed ubuntu, even after installing ubiquiti (the installations program). Not sure why that doesn't work.
