### Post by taurus on 2006-12-07
So, you don't want to use the "1920x1440" at all!  You can remove the "1920x1440" from all those line near the end of it.  From a terminal, run

```
sudo cp /etc/X11/xorg.conf  /etc/X11/xorg.conf.bak <-- make a backup...
gksudo gedit /etc/X11/xorg.conf
```
Scroll down near the end and remove the "1920x1440" from those lines.  Save it and either start X again with <Ctrl><Alt>Backspace or reboot!

### Post by taurus on 2006-12-07
Perhaps you need to use the 915resolution as the driver instead of the i810!!!

To install 915resolution, you need to enable both universe & multiverse in /etc/apt/sources.list by removing all the lines with universe & multiverse at the end...

```
gksudo gedit /etc/apt/sources.list
```
Save it and run

```
sudo aptitude update
sudo aptitude install 915resolution
```
Then, change the line that reads
```
       Driver          "i810"
```
to 
```
       Driver          "915resolution"
```
Then, <Ctrl><Alt>Backspace to see what happens...

p.s.  Are you sure the refreshing rates are right for your monitor?

```

Section "Monitor"
        Identifier      "Generic Monitor"
        Option          "DPMS"
        HorizSync       28-96
        VertRefresh     43-60
EndSection

```
