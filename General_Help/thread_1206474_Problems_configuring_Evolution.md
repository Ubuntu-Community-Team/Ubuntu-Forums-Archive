### Post by 101011010010 on 2009-07-07
Hi there, I had exactly the same problem with Hotmail, I hadn' t set up my firewall to allow incoming connections.Hotmail couldn't connect for me to enter my password even though the password prompt(window) kept appearing.If you're using Firestarter, go to the policy tab, click on the space under the Allow service heading, click on Add Rule, if you type in the required port number, it usually fills the service name in itself. Then select the source as IP, host or network and enter the server number. Click on add, then apply policy. Once done open Evolution and everything should work.I hope this helps.
