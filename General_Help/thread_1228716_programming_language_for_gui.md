### Post by 3rdalbum on 2009-08-01
If you want to create a GUI using a GUI and program a backend, then you have many options.

Gambas is similar to Visual Basic (not a clone), and it seems to be in reasonably-wide usage. You have to program in Basic, though.

I believe MonoDevelop has a GUI designer, if you want to program in C#.

Pythoncard is no longer maintained (forked, apparently, to a new project called Picard?) but it's what I use - very easy-to-use GUI designer, with the accessibility of the Python language. Winning combo.

If you don't want to be locked to a language, there's a GUI designer for GTK called Glade. If you want something a bit more high-level than GTK, you could use WxGlade, which uses the Wx toolkit instead.
