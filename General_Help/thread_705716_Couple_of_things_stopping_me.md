### Post by cdaringe on 2008-02-23
As far as getting use out of your screen, you should be able to get waaaay more use out of your screen on a linux box.  Pretty much everything you see on your screen is customizable.  I would list the specifics, but that would flood my post.  Some things you can look into for getting better use out of your screen may be:
*window manager customizations and themes
  --Explorer would be MSwindows' window manager.  As explorer is to Windows, Metacity is to ubuntu (by default...and can be changed to many!)
*icon sets
*workspaces
*customizing your panel layouts, options, and plugins(gnome,kde,xfce,etc)

If you haven't made the move yet, I can promise you there are way more options for getting better screen use.  There are more features for window handling/movement as welll.  Easy shading, sticking, and grabbing of windows make it relatively easy [just to name some basic ones].
If you are looking for a pretty setup with excellent window configuration and screen use, many people look to a window manager called 'Compiz' which i believe comes built in in Ubuntu 7.10.  I personally use XFCE, and am throwin windows around much quicker than I was on my windows box.

That's that in a nutshell.  What I'm getting at is don't let that be a deciding feature for you, because window management/desktop utilization can be changed!
