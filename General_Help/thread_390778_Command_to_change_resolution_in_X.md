You could also edit the xorg.conf
> sudo nano -B /etc/X11/xorg.conf

### Post by taurus on 2007-03-22
Edit /etc/X11/xorg.conf

```
nano -Bw /etc/X11/xorg.conf
```
and change this line

```
DefaultDepth   16
```
to this.

```
DefaultDepth   24
```
Save it with <Ctrl>o and <Ctrl>x and reboot.
