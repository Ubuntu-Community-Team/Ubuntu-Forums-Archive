Sliding sideways a little, but have you looked at [RemixOS]("http://www.jide.com/remixos-for-pc")? Android for the computer. You may be wanting to run Android apps in Ubuntu specifically, but if not or failing that, you could install Remix on another partition or as a virtual machine.
