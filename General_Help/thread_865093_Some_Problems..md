Sorry I don't speak any Lithuanian. 
I think that basically the problem is not everything has been translated into Lithuanian yet. The places to start looking for how much of Gnome is in Lithuanian are 
[http://www.gnome.org/i18n/](http://www.gnome.org/i18n/)
[http://developer.gnome.org/projects/gtp/](http://developer.gnome.org/projects/gtp/)
[http://l10n.gnome.org/teams/lt](http://l10n.gnome.org/teams/lt)

For non-Gnome applications they have individual translation efforts so their translations will vary and usually not be as good.

For the pppoeconf I don't know what to tell you, but you are likely to be better to start a new question with a descriptive topic like 'automatic network connection doesn't work', instead of 'some problems'
