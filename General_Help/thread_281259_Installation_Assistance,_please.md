### Post by PriceChild on 2006-10-20
When your computer starts up it will say something like "press delete to enter setup" press it!

It could be delete, Esc, F1, F8 or many others... depends.

Once there, you need to find the boot order, and make it boot from cdrom before the hard drive.

Then save and exit and it should boot into live fine :)

(Remember to change it back when finished)
