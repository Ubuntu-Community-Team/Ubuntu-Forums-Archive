### Post by Frogs Hair on 2018-07-05
Hello and Welcome

You can install the Ubuntu Studio desktop though they plan a complete redesign for 18.10 _so it may be best to wait and perform a clean installation._ _All of the Ubuntu Studio applications are available in the Ubuntu repositories without the need to install another desktop environment._ Installing additional desktop environments leaves one with duplicate applications, clutter,and _possible breakage_. 

If really want to install the extra desktop. ```
sudo apt install ubuntustudio-desktop
```
 
Plans for 18.10:
[https://www.omgubuntu.co.uk/2018/04/ubuntu-studio-plans-to-reboot](https://www.omgubuntu.co.uk/2018/04/ubuntu-studio-plans-to-reboot)
