### Post by gmargo on 2010-02-03
> **nickbourcier said:**
> The only speeds are Maximum and 10x. Is there another program that goes slower?

Assuming you had it on Maximum the first time around, try it at 10x.  That's the easiest first step.  It is likely that 10x is as slow as the CD media itself can handle.  (The HP CD media I have lying around has an 8x minimum according to K3b.)

As far as another program to recommend: I don't know since I do all my burning on Linux :p (Google turns up a bunch of possibilities, but try 10x first.)
