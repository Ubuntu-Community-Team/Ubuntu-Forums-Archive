### Post by lisati on 2008-09-24
Common problem. 
From the terminal (It's on the Applications->accessories->terminal menu) type in ```
sudo dpkg --configure -a
```

You will be asked for your password. Type in the same password as the one you use to login with. Don't worry if it doesn't show - just type as usual. This is a normal Ubuntu security precaution.
