### Post by Giblet5 on 2009-10-12
Do you already know any other programming languages?

If not, you'll need a VERY basic book on boolean logic and how computers work. I'd start with a book on HTML. Then you can proceed with learning **A** language eg, Java (although java sucks for anything more complicated than fancy menus). Master that, then add another. And another. And another.

[This guy]("http://www.robson.org/gary/writing/becomeahacker.html") can tell you exactly what to do.
