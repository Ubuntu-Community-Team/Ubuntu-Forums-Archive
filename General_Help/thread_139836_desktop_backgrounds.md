### Post by akiro.yamamoto on 2006-03-04
You can also install some additional background images from the repos. Use synaptic to install gnome-backgrounds.
Also there is an app called gnome-art.
This app will download backgrounds, GDM themes, Spalsh screens, Icons and
Metacity / Gtk2 themes for you, from the Gnome-art.org web-site. As well as install some of then to your system.
I think the GDM themes are the only ones it cannot install automatically for you.
It provides thumbnail images of the various themes, so you know before you download exactly what your getting.
Yup this is one coooooll! app ;)
