### Post by cdenley on 2007-07-09
I'm trying to burn a CD that contains a mac application. Although the file's checksum matches the original, the one I burn in Ubuntu will not execute. Is there some kind of file system flag that gets lost when I copy the file from the CD in Ubuntu?

---

### Post by cdenley on 2007-07-23
The problem was that Mac cd's have an extra flag to mark file type, which is lost once the file is on a Linux machine. The CD had to be burned as a hybrid on a Mac using Toast.
