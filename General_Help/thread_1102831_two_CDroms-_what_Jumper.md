### Post by CatKiller on 2009-03-22
Largely it depends on the age and quality of the drives. On some drives, cable select is crap. On others it does exactly what it's supposed to.

Leave them on Cable Select to start with, and if there are problems (you'll see pretty quickly if there are problems) switch them to Master/Slave.

### Post by PenguinsFan on 2009-03-23
Leave them set to CS (Cable Select) - This works 99% of the time. If you start having problems with discs reading or burning, you can try Master / Slave settings - but realistically, you'll never need to worry about it.
