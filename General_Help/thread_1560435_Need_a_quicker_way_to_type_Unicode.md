### Post by Vaphell on 2010-08-24
there is a method similar to that with alt
CTRL+SHIFT+u (_u_ shows up), hexadecimal code of the character, space to finish

also you can use a compose key to produce characters (i think you can enable it in keyboard preferences, under layouts), not that it's an actual combo but imagine holding it and then pressing o and : and these two magically create o umlaut

edit: i enabled mine and compose+(s,s) = ß :-)
compose+(o,^) = ô
compose+(Y,=) = ¥
compose+(o,") = ö

figure out proper combinations and typing will be a breeze
