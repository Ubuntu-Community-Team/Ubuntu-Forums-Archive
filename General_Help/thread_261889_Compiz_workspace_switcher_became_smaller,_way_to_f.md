### Post by ayoli on 2006-09-21
you may want to update your compiz. i had this problem, but there has been updates that have fixed this.
btw, you can try:
- run *csm*
- in "General Options", pick "Numeric Values" tab.
- go to the last slider: "Vertical Virtual Size" put it to 4, then put it back to 1.
should fix for the session (mean you have to do that after each logon)

### Post by labinnsw on 2008-03-22
Hopefully, you would have solved this problem by now but this might just help someone esle. I encountered this problem when I started using desktop effects. Since it is not an essential feature I simply stopped using it.

1. Right click on Desktop
2. Select Change Desktop Background
3. Select the visual effects tab
4. Select the none radio button
5. Close the window to save.
