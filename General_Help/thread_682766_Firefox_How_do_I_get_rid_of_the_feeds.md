> **newnet said:**
> In the options/preferences, under Feeds, I have "No Application Selected" selected, which I thought would not fetch any more feeds.  But appearently  not.  I tried about:config using "feeds" as the keyword, but it too failed.

You can't turn off the Feeds Reader in Firefox, it is built into the browser. If you want to delete all the feeds you are presently subscribed to then go to in Firefox, *Bookmarks --> Organize Bookmarks* and simply right-click and delete the ones you do not want. By default, I think there is only one feed subscription anyway which is a BBC News Feed.
