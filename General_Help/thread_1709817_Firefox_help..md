### Post by lithopsian on 2011-03-19
That PPA offers 3.6.16, the latest official release.  4.0 is very close to release (next week?) and release candidates are offered direct [from Mozillla](http://www.mozilla.com/en-US/firefox/RC/)   The PPA also offers a 4.0b13 which would be the "beta 13pre" which is now the trunk development for 4.1 (or 5.0 or whatever it is called).  I would suggest trying the 4.0RC first since it is the most stable and up to date.  Or wait a few days for the official 4.0 release.  I don't know how long it will take for that to hit the Ubuntu repository.

Game lockups might be related to Flash (Flash game, I assume?) so 4.0 might be much better for you.  It offers better hardware acceleration support.  Also make sure you have the latest Flash and if that is no good then try and older version of Flash.
