That's the program that you'd use on Ubuntu to see other people's desktops. You need to go to System > Preferences > Remote Desktop to set up your own desktop as being able to be viewed and controlled remotely, and also use "Configure network automatically" if you are behind a firewall.

Make sure you set a good, strong password.

Then you can use the VNC viewer on your iPhone or another computer to see and interact with your computer remotely.
