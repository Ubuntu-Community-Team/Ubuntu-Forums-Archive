### Post by kokkus on 2008-09-24
So you wish to add ubuntu as well as ubuntu studio that's already installed?
No the live CD won't mess it up as long as you don't choose to install it on the whole disk.
Create a new partition from the installation program on your live cd will work just fine.

You can use the regular hardy live CD. You could even use Feisty live CD, or Kubuntu hardy live CD. Doesn't matter as long as it has the partition editing program: gparted.

Of course, don't install from this live CD. Just edit the partitions and reboot into your regular hard disk installation.
