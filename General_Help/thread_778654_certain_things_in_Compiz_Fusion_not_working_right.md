There are a couple things you need to be sure of.  One: you have to have the restricted drivers enabled on a vid card that will handle compiz.  Most modern cards are sufficient. To do this, go to SYSTEM>PREFERENCES>VISUAL EFFECTS> and click on EXTRA.  You should get a message that asks if you want to enable restricted drivers.  Go ahead and do this. Two:  You will need to install Compiz Configuration Settings Manager through Synaptic. Go to SYSTEM>ADMINISTRATION>SYNAPTIC PACKAGE MANAGER> then click on search and type COMPIZ.  You will find Compiz Configuration Settings Manager as one of the options.  Three:  Once installed, you have to open the compiz config settings mgr by going to SYSTEM>PREFERENCES>ADVANCED DESKTOP SETTINGS then click on the GENERAL OPTIONS>DESKTOP SIZE  and set the top button to "4".  Click the BACK button, then check the little box by DESKTOP CUBE and another little box by ROTATE CUBE.  Close the window.  I think you should be good to go.
