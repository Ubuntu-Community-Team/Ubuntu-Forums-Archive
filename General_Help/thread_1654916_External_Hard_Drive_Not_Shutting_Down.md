### Post by noah++ on 2010-12-29
> **Metungkp said:**
> I switched to Ubunto a couple of days ago.  I noticed today by external hard drive stays on when I shutdown.  It was shutting down yesterday.  Any help would be appreciated.
I can only tell you that it's unlikely anything that happens after shutdown is related to Ubuntu or Windows or whatever operating system you are running. If the problem persists, and it bothers you enough (and nobody here provides an answer), maybe Google search for the problem and your hard drive model.
> Second question:  I completely dumped Windows 7 and switched to Ubuntu.  I thought I had backed up all of my docs, is there a way to recover the files?If you formatted the Windows partition -- maybe you can recover your files, but not too easily. If you are serious about it, you should stop using your Ubuntu installation immediately and work from a live CD so that the disk sectors that may still contain your Windows data don't get overwritten.

If Windows is still lurking there on its own partition -- then that's easier.

In any case, you ought to open a new forum thread for this separate problem so it gets the attention it needs, and so others with the same problem might find the solution more easily in the future.
