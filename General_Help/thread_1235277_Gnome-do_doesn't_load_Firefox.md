### Post by Genius314 on 2009-08-08
Hmm... weird.

In Gnome-Do, start to type "firefox," but then press the down key. If there's more than one entry for Firefox, try using another one. Maybe Gnome-Do is pointing to something other than the program launcher?

EDIT: You could also try closing Do, and launching it from a terminal. Then you can see if any errors pop up when you try running Firefox.
