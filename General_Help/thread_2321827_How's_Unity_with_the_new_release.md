### Post by grahammechanical on 2016-04-24
Unity is the user interface. You can break it but you cannot hide it or hide from it. Ubuntu 16.04 has Unity 7 just like Ubuntu 15.10 and Ubuntu 14.04. It is an upgraded version of Unity 7 to be sure but Unity is Unity after all.

If you are inquiring about the Launcher then the new thing about the launcher is this

[http://www.omgubuntu.co.uk/2016/04/unity-tweak-tool-0-7-move-unity-launcher](http://www.omgubuntu.co.uk/2016/04/unity-tweak-tool-0-7-move-unity-launcher)

Unity Tweak tool is not installed by default. Autohiding the launcher has been with us from the beginning. Resizing the Launcher icons has been around for a long time. An option to set application menus to appear in the top panel or the application title bar & to appear with mouse over or permanently appear has been with us since 15.10. I think.
