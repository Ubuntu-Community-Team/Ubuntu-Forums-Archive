I am using my old system (AMD Sempron) as a 24/7 torrenting machine.I was facing one problem with downloading, that was queuing of downloads, and all downloads ran simultaneously. Searched everywhere but could find and solution for a very long time.

**Finally found the solution and would like to share it with everyone.**

Download and install [Transmission Remote GUI Software]("http://code.google.com/p/transmisson-remote-gui/downloads/list") (windows/linux/mac *source code also available)
after installing provide the software with link to Transmission Web UI..... DONE..!!

You can now use all features of a torrent client software.. including queuing..
