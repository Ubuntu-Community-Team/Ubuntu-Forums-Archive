### Post by HermanAB on 2021-03-24
In general, yes.  Linux has all the software required to do advanced routing.  In practice, unless you install a dedicated router distribution such as OpenWRT which has a special GUI, then you need to set the addressing, routing and firewall rules up yourself, which will take you a while to figure out.
