### Post by Vivaldi Gloria on 2008-06-25
Squid is generally used as a web cache. My university uses squid to store frequently visited web pages locally.

Say, a user in my university, Joe, visits a web page. Then another user, Jane, visits the same page. Since squid kept a local copy of the page when Joe visited the page, Jane will get the local copy and she will see the page much more quickly.

But of course you can use squid for much more:

[http://en.wikipedia.org/wiki/Squid_cache](http://en.wikipedia.org/wiki/Squid_cache)
