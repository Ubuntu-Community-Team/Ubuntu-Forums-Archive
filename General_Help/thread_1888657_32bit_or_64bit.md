### Post by IWantFroyo on 2011-11-29
In the name of the download, it should say either amd64 or i386.

If you bought it on a disk without knowing beforehand, it's probably 32bit.

---

### Post by ExileAmongYou on 2011-11-29
Type ```
uname -p
``` in a terminal. If you're using 64-bit, you'll get "x86_64", if you're using 32-bit, you'll bet something like "i686".
