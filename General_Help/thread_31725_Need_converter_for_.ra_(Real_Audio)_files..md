### Post by mduduzi on 2005-05-08
Proprietory formats are problamatic. If you cannot find a convertor, and the content of you audio files is that important to you, I suggest you try the manual solution:
1. Create a playlist of all the *.ra (Real Audio) files.
2. Set the recording source to 'Wave'
3. Use an audio recording applications to record while you play you play list.
4. Depending on how much space you have, record into a *.wav format.
5. Use a Wave File Editor to crop out the individual tracks.
6. Save each track as an MP3 or whatever you choose.

If you want to do it this way, it would be advisable to turn off all system and program sounds.

mduduzi.
