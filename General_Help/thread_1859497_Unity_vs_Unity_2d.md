### Post by Prendi on 2011-11-08
Click on your desktop. Then move your mouse cursor to the menu bar.
- if you are using Unity, the menu bar will display a menu similar to that of Nautilus
- if you are using Unity-2D, the menu bar will stay empty

Also, on alt-tab, Unity-2D shows only small icons similar to Windows XP (and prior), whereas Unity will display icons as big as those in the Dash Home (the windows that opens when you hit the Windows key once)
