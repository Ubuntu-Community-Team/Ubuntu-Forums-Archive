### Post by capink on 2007-09-13
You can convert them to iso, and then mount the iso as follows:

Install bchunk

```
sudo apt-get install bchunk
```

Convert them to iso

```
cd /path/to/image.bin
bchunk image.bin image.cue image.iso
```

Mount the iso

```
sudo mount -o loop -t iso9660 /path/to/the/iso /mnt
```
