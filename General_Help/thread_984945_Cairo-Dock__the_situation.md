### Post by fabounet on 2008-11-17
=== Stable version ===
the last version of Cairo-Dock is 1.6.3.1, please don't use the Ubuntu package.
32 bits packages are available for Hardy and Intrepid, and 64bits for Hardy. A source tarball is also available on Berlios.
To install it, follow the easy way :
[http://www.cairo-dock.org/ww_page.php?p=From%20the%20repository&lang=en]("http://www.cairo-dock.org/ww_page.php?p=From%20the%20repository&lang=en")
before you do that, don't forget to remove all cairo-dock-* packages from your system if you have installed it from the Ubuntu's repository (or made the upgrade with Intrepid)

Here is a short demo of Cairo-Dock and its integration with Cairo-Desklet
[http://www.youtube.com/watch?v=AXYsX_qvPS8]("http://www.youtube.com/watch?v=AXYsX_qvPS8")

=== Development version ===
the SVN is very instable at the moment, avoid using it.
