### Post by cariboo on 2016-11-28
If you use Windows software to backup a Linux partition you will lose file permissions and other attributes, Windows doesn't understand Linux permissions. It's far better to use Linux software to do the job.

Have a look [here]("http://www.linuxlinks.com/article/20160424021828491/BackupTools.html"), to see if you can find something that suits you.
