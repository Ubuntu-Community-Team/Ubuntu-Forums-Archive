### Post by raggamuffin on 2006-02-03
Kmenu -> Run Command -> type "kcontrol" (without quotes) -> KDE Components -> Session Manager....

what i reccomend is setting it to "Restore Manually Saved Session"...then get your desktop to exactly how you want it at login...and K Menu -> "Save Session"...
