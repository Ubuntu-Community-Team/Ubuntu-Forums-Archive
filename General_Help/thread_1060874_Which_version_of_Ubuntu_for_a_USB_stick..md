### Post by snowpine on 2009-02-05
Did you install to the USB stick using the USB Creator or the regular installer? Anyway, 8.10 has the best support for USB installs, so I would not recommend an older version of Ubuntu. Xubuntu 8.10 might be a tiny bit quicker, and Crunchbang is also very nice.

Flash drives are always going to be much slower than an internal hard drive. It is the nature of the technology. A lot of users have good luck with Puppy or SliTaz on flash drives, as they load completely into ram at boot time, instead of constantly accessing the drive.
