### Post by Dusk2k2 on 2006-06-08

### Post by qamelian on 2006-06-08
[quote=Dusk2k2]I have Dapper installed on my parent's computer, but they're not very inclined to click the little update icon when updates are available.  Is there anyway to make Ubuntu simply update automatically in the background without having to enter a password and go to the update manager?[/quote]

Go to System > Administration > Software Properties. Go to the Internet Updates tab and put a check mark beside **Install security updates without confirmation**.

That should do it.

### Post by cstudent on 2006-06-08
In Synaptic under the Repositories menu you can set it up to install security updates automatically and download other updates without confirmation. The other updates don't look like they will install, just download.
