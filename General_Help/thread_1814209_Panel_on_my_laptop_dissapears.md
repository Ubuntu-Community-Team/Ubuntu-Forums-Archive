### Post by Wayne_V on 2011-07-29
check to see if gnome-panel is running.

if it is, kill it (pkill gnome-panel).  It should kick off again in just a few seconds - I have had problems occasionally where it disappeared or was mangled and a restart (pkill gnome-panel) cleared it up.

if it is not running, try to start it.
