### Post by grahammechanical on 2014-08-09
I can confirm that the Music scope shows what songs/albums have been played and not the files in the Music folder. Likewise, the Files scope will show the files and folders that have recently been accessed. Dash scopes are not there to replace the File Manager. Not yet at least.

Edit: In the Dash Home we can use the filter options to expand or limit the search sources. If we have imported music files into Rhythmbox we can select Rhythmbox as a search source. Also make sure that Files & Folders is selected as a search source. Then when we open the Music scope we can search for songs and albums by name and we will get a result.

The Dash is basically a search tool. It is not a file manager tool.
