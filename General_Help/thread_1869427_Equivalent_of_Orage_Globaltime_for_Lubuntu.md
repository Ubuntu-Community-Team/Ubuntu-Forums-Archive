### Post by JC Cheloven on 2011-10-25
You can simply install orage in LUbuntu. It should work just fine. It will appear in the office section of the lxpanel menu.

Just grab it via synaptic or with this command at terminal

```
sudo apt-get install orage
```

Enjoy!

EDIT: oops, gsmanners is right about the applet. It won't show up in lxpanel. Otherwise orage works fine (just tried myself).

EDIT 2: There is also a program called gworldclock you could try. It's also in the repos.
