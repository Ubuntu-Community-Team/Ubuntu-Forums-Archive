### Post by ibuclaw on 2009-05-17
They are all mostly in the directory
```
/usr/lib/xscreensaver/
```
But you will also need the corresponding desktop file too in
```
/usr/share/applications/screensavers/
```

What OS does your friend have? The screensavers will only be compatible with other Linux OS's
