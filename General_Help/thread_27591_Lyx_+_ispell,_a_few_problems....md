 for hebrew there is hspell
to configure spell checker with lyx you should either
1. download the dictionaries to be used with ispell for they are not downloaded by default
2. use aspell instead of ispell as follow in the prefrences section in lyx select > language setting> spell checker> aspell as your spell agent (hspell) for hebrew
3. run reconfigure
4. start using the spell checker
