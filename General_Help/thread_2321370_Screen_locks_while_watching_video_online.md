While watching videos online (players such as: youtube, dailymotion, vidfile, jwplayer etc.) my screen goes black and then locks. Is there any way to keep functionality of screen locking and disable it only while watching videos online? I'm using Ubuntu 16.04 and Google Chrome browser.

---

### Post by kostkon on 2016-04-22
Yes, the screensaver and power management inhibitor app called *caffeine*. Either use the software centre to install it or just do:
```
sudo apt-get install caffeine
```


Hope that helps.
