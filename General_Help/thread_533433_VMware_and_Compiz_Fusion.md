### Post by arbulus on 2007-08-23
I have Compiz Fusion running on Ubuntu Feisty.  I just got VMware Server installed and have Vista running as a virtual machine.  I do have one hitch though.  When I hit "full screen" in the WMware Server Console and Vista goes full screen, it's semi-transparent.  I figured that it had to do with Compiz Fusion, so I turned it off and relaunched vmware, and sure enough, the full screen Vista was not at all transparent.

So my question is: is there a way to tell Compiz Fusion to use no transparency on full screen in VMware Server?

Thanks in advance!

---

### Post by dragomirov on 2007-08-24
To change transparency under compiz press ALT key (and keep it pressed :) ) and use the mouse scroll wheel to change.
