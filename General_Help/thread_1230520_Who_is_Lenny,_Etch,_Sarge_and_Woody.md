### Post by dreamsR4living on 2009-08-04
> what is up with those (dare I say) stupid non-descriptive names.

These are indeed the Debian code names. Every time a new Debian version is released, they choose a name from a Toy Story character. This is what wikipedia says.

> The code names of Debian releases are names of characters from the film Toy Story. The unstable, development distribution is permanently nicknamed sid, after the emotionally unstable next-door neighbor boy who regularly destroyed toys.

I don't know what you intent to download, but Debian is a complete OS just like Ubuntu, but it requires more knowledge and patience to get everything to work compared to Ubuntu.

I am sorry, probably you already know this, but if you want to install software for Ubuntu use Add/Remove or Synaptic package manager from the system menu.
