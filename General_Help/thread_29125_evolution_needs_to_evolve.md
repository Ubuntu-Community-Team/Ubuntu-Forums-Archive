### Post by mdbarton on 2005-05-12
My workaround:
Install JPilot
Install Thunderbird
Syncronize with JPIlot and export the the addresses in LDIF format
Import the LDIF file to Thunderbird.
Use JPilot for PIM, Thunderbird for e-mail.
