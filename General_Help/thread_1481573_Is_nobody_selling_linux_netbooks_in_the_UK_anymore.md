Download [http://www.ubuntu.com/getubuntu/download-netbook](http://www.ubuntu.com/getubuntu/download-netbook) (I have not actually downloaded this version but I presume it gives you the same options as Ubuntu desktop)!
 
Create flash drive!!
 
Run the live flash drive option! (Gives you the option to try but not install)!
 
Check everything is working correctly (audio, video, wireless, etc)!
 
Then be bold an install Ubuntu!
 
There is other stuff to consider such as what type of partitions you wish to create!
 
Or you can go with the auto option and use the whole hard disk!
 
Not an expert but if your netbook is compatible it shouldn't be a problem!
