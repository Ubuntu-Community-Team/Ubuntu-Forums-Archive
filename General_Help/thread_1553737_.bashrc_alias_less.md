### Post by l3ecl on 2010-08-15
i have my .bashrc alias set up like this:
alias ls='ls -a | less'

my only problem is the " | less" part which prevents me from adding other options like -b -c -c -d ect...

for example:
if i type 'ls -l' , it should print it
1. the hidden files ( -a from the alias) 
2. issue the command 'less' (from the alias)
3. AND print it out in long format


but for some reason it does not do the last part.
