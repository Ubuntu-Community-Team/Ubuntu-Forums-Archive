### Post by SteveDee on 2009-11-23
I assume you are running 8.04 from your user information, so your sluggish graphics could be due to driver problems.

I recently did a fresh install of 9.10 on my humble Dell 500 and its surprisingly snappy, but I don't run any games. I'd recommend a fresh install rather than an update, as my upgraded desktop is not that quick.

I doubt theres much performance difference between Ubuntu & Kubuntu. However, I also run Xubuntu 9.04 on an old desktop used to play videos, as I previously experienced dropped frames on Ubuntu.

But if you want a distro that is very light & snappy, try Puppy Linux.
