### Post by Lord Illidan on 2006-07-20
You seem to be in the right forum.

Just type in a terminal.

```
sudo apt-get install build-essential
```

This will download a bunch of programs from the internet, compilers and stuff. Among them, there will be gcc, the GNU C compiler.
