### Post by SeijiSensei on 2017-10-27
For Prime, I use Chrome from the Google repository as described here: [https://www.ubuntuupdates.org/ppa/google_chrome?dist=stable](https://www.ubuntuupdates.org/ppa/google_chrome?dist=stable)

Note that the package you want is "google-chrome-stable".

If you install Chrome via the repository, then it will be updated automatically just like all the other software on your system.
