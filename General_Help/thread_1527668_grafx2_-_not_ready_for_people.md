### Post by A_Nonny_Moose on 2010-07-09
I need a program that can do small bit maps to produce a .bmp file on a pixel by pixel basis.  So I looked in synaptic with a key of bitmaps and found grafx2, standing alone with no supporting links.

I read the description and it sounded like the millennium had arrived.  So I downloaded and installed it.  Much to my surprise when I opened it, was a set of tools that seemed to have no open or save capability, that could only work in screen sizes, and that had no standard features for control that were recognizable by any standard of which I am aware.

So I forced it to close, then opened it with one of the files I am working on.  What a travesty!  I could not create the arbitrary bit map I need, nor could I edit one.  This program is not for me, and I really question the wisdom of its availability, at all.
