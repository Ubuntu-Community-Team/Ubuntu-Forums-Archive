### Post by sayakb on 2008-10-04
Install ccsm: At a terminal (Applications->Accessories->Terminal), type in:
```
sudo apt-get install compizonfig-settings-manager
ccsm
```

In the window, click on General Options->Desktop Size and set Horizontal virtual size to 4
Now in CCSM index, enable the Desktop Cube, Rotate Cube, Cube Reflection Plugins and Cube Caps plugins.
Press Ctrl+Alt+MouseButton1 to rotate
