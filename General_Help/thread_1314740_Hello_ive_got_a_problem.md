### Post by scouser73 on 2009-11-04
Hi Alan, and welcome to the Ubuntu forums.  To enable the visual effects you will first need to install your graphics card driver.

Go to **System** > **Administration** > **Hardware Drivers**, once this is up and running you will be recommended a driver to install.

You need to select the recommended driver and install it, at this point it should tell you that you need to restart your PC.

Once you have restarted the PC, then go to **System** > **Preferences** > **Appearance** & click the **Visual Effects** tab.

You've mentioned that you are also having problems with your hard drives, can you go into some more details please.

As for Superkaramba I've not used it but if you use the search function in the forums then someone with a better understanding may possibly be able to help you.
