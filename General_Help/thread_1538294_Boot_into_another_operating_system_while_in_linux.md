### Post by Mark Phelps on 2010-07-24
You can only run one operating system at a time. Period.

You can install multiple OSs (in what's known as a multi-boot system), but then, you boot into one at a time.

You can install virtualbox, in which you install an OS into a virtual environment.  But even then, when you boot into the virtual environment, you're running that OS.
