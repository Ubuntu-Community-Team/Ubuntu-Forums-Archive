### Post by Damon on 2004-10-16
Keep in mind, that everything gui in Linux  can be started from the command line. If you just want to see xmms run, go to Applications =&gt; Run Application and type in "xmms".
 According to the [Multimedia HOW-TO](http://www.ubuntuforums.org/viewtopic.php?t=112), you may need to log out and back in to your desktop to see the changes.

I know it works because I'm groving to some monkey radio right now ([http://www.monkeyradio.org/](http://www.monkeyradio.org/)).
