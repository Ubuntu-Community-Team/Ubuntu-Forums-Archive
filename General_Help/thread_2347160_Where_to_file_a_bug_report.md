### Post by howefield on 2016-12-22
You could try from a terminal..

```
ubuntu-bug ttf-mscorefonts-installer
```

For more information on filing bugs see : [https://help.ubuntu.com/community/ReportingBugs](https://help.ubuntu.com/community/ReportingBugs)

There already is a bug(s) filed against this package which most likely covers your issue, and if you post the error we can most likely help you get a round it.
