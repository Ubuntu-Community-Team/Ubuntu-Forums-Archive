### Post by timgood on 2015-05-20
Here's how I solved this:

1. Boot from 15.04 i386 media. When purple screen starts, hit any key.
2. Press F6 and select 'nomodeset'
3. Install 15.04 and reboot
4. Will boot in low graphics mode. Install kernel 4.04 from [http://kernel.ubuntu.com/~kernel-ppa/mainline/](http://kernel.ubuntu.com/~kernel-ppa/mainline/) (make sure you have correct versions for your architecture).
5. Reboot 
6. Edit /etc/default/grub and remove 'nomodeset'.
7. Reboot. Enjoy!
