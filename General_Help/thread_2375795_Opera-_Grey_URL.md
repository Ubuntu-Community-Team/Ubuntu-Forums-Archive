### Post by coffeecat on 2017-10-27
Are you aware that domain highlighting - making the actual domain darker than the rest of the URL - is there to help you, so that it is easier to distinguish some dodgy sites from the genuine sites they seek to mimic? 

If you really want to disable domain highlighting, this page may help:

[http://www.opera.com/help/tutorials/security/guide/](http://www.opera.com/help/tutorials/security/guide/)

For myself, I prefer to stick with domain highlighting. You get that by default in Firefox and Chrome as well.
