### Post by kurt18947 on 2016-07-15
This is not an optimum solution but if no better idea presents itself, VueScan might be worth a shot. It can be run like an old time DOS program, just click on the executable. The download will put a watermark on images but if it works as you wish, purchasing a key will remove the watermark. We have a Canon scanner that works natively but the lighted lid does not. Transparencies (slides) require the lighted lid.
