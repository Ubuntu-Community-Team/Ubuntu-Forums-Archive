### Post by rotave on 2010-09-07
I'm not sure why it isn't working in Firefox, but you could try and install the FLASH-AID extension for firefox 
[https://addons.mozilla.org/en-US/firefox/addon/161939/](https://addons.mozilla.org/en-US/firefox/addon/161939/) 
The extension removes conflicting flash plugins from Ubuntu Linux systems and installs the appropriate version according to system architecture.
