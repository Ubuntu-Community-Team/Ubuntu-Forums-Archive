### Post by kc8hr on 2006-11-07
Not that I know of, but you could always use fetchmail.

Fetchmail is an old Unix mail utility that downloads mail on a schedule that you choose, to the directory that you choose. It's not easy to set up, but there is a gui config creator to go with it.

sudo apt-get install fetchmail
sudo apt-get install fetchmailconf

Good luck!
