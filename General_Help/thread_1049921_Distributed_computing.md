### Post by mb_webguy on 2009-01-25
That wouldn't work the way you think it will, unless you're working on some really processing-intensive problem that can be broken down into discrete chunks.  You're certainly not going to run regular programs faster with any kind of distributed computing setup.  Distributed systems are perfect for things like gene sequencing or protein folding, but not for desktop computing.  

Those older computers can be put to good use, though.  You could set one up as a file server, for instance, since processor speed isn't a high priority for a file server.  I set up my old laptop as a dedicated web browser, so my family can use it for browsing while I use mine for actual work.  This sort of thing is also good for older computers, since it's generally not processor intensive (except possibly for playing streaming video), and doesn't even require much (if any) storage space.
