### Post by mb_webguy on 2008-09-18
According to the [Wine AppDB website]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=5954"), Dazzle 2005 will work moderately well under Wine.  Apparently you have to enter your passphrase every time you run the program rather than just the first time, but otherwise it seems to work well.  However, that entry says that it's possibly out-of-date, so it may not reflect the current compatibility of Dazzle on Wine. It may work better now.

You can go ahead and install it and see for yourself.  If it doesn't work, you haven't really lost anything but a few minutes of time.  Make sure you're using the latest version of Wine from the [WineHQ]("http://www.winehq.org/") website.
