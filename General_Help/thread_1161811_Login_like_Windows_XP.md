### Post by kpkeerthi on 2009-05-17
System -> Admin -> Login Window -> Security tab -> Enable automatic login -> Enter your user Id.

You should enter the UserId of the user you want automatically logged in. Password is not required.

In ```
System - Administration - Login Window
``` you can set automatic login. This is done on TAB Security (see attachment)
When done you are never asked again for username nor password, infact the login window does not appear at all.
If you have one account this is the easiest way to go, if you are not afraid others will use your computer on your account.
