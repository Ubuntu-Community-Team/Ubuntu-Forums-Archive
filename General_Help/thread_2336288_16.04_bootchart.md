### Post by dn1987p on 2016-09-09
I think regular bootchart is not supported with systemd anymore (which is used since Ubuntu 15.04)

Instead you might just try

```
systemd-analyze plot > bootchart.svg 
```

since systemd should analyze the boot process by default anyways.

See [http://askubuntu.com/questions/710127/how-can-i-set-up-bootchart-in-ubuntu-15-10](http://askubuntu.com/questions/710127/how-can-i-set-up-bootchart-in-ubuntu-15-10) for further information.
