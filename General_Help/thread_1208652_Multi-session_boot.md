### Post by t4thfavor on 2009-07-10
Boot the live cd , and with a 1+GB usb stick go to system administration Create USB boot disk. this will also allow you to specify the aditional 200MB of the flash drive as persistant userspace. It will help you save your data/files/settings onto the USB stick. 

It will be slow like the CD though as the image is still compressed. You can instal Ubuntu onto a flash drive 4+GB in size and have it function like an actual install. Just rememver to install it onto the USB drive when there are no other disks plugged in (harddisks included) as we don't want it messing your bootloader up.
