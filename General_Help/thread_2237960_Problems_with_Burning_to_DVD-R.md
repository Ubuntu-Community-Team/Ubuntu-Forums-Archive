### Post by Mark Phelps on 2014-08-05
When you "burn" a bunch of files and folders to a DVD, you are creating what is known as a "session" on the DVD.  Using a rewritable DVD, you can add more sessions -- but the problem is, not all apps, and not all DVD readers, can handle a mutli-session DVD.  Some see only the last session, some don't see anything.

Also, since this forum has different sections for different questions, you would do better posting each question to the proper section, rather than just having a bunch of unrelated questions in a single thread.
