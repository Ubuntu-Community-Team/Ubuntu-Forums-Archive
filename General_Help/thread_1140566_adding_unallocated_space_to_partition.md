I have a partition made on my hd where I had windows 7 installed and now that I don't use windows 7 anymore, I want to add that space back to my windows vista partition. Is this possible or will I have to reformat and reinstall vista?

---

### Post by cwbar_1 on 2009-04-27
It is totally possible.  (I hope these directions are correct, because I no longer use Vista) Go to Start > Control Panel.  Find the disk management section and delete the windows 7 partition.  Then you can right click on the Vista partition and tell it to expand the Vista partition.  (You cannot absorb the Windows 7 partition as long as it is a partition.)
