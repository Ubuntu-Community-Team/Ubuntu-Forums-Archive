### Post by Duckslammer on 2009-05-09
My system crashed, and on restart the panels are gone from my desktop, and of course along with them all the menus.  Which X config file do I edit to put them back?

---

### Post by fooman on 2009-05-09
when your at the desktop,  try pressing the alt-f2 keys....see if that brings up a run dialog box

if it does, try typing the following into it,  then press "run":

```
xfce4-panel &
```hope that helps.
