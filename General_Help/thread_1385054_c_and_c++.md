In Ubuntu/Linux we use gcc (C compiler), g++ (C++ compiler). You should spend little time to understand those: how to use, how to compile, ...

Anyway, you need those packages by using command:

```
sudo apt-get install build-essential
```

After that, you can test gcc:

```
gcc -v
```

You can also follow [this tutorial]("http://users.actcom.co.il/~choo/lupg/tutorials/c-on-unix/c-on-unix.html") to understand how to compile sources in Linux.

Enjoy.
