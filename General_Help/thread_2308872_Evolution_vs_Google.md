### Post by amanchesterman on 2016-01-06
I use Thunderbird as my email client because it comes with the distro I use.
Like you I use Google Calendar: I have various calendars (different types of events/appointments) and tasks/to-dos as well. There are add-ons for TBird that synchronise with Google Calendar and in my experience they work perfectly, I haven't had any problems with it for several years.
I don't use Google Contacts. However I see that there is an add-on for TBird which promises to synchronise with Contacts: it says "It can synchronize the My Contacts group, all contacts, or all contacts and groups (with mailing lists)." I don't know if something like that might meet your needs? Unless anyone else has a better suggestion, you might want to give TBird a try. Make sure you back up everything beforehand, then you've got nothing to lose but the time you spend setting it up.

### Post by Jimbotronics on 2016-12-14
Evolution DOES work with Google calendar, contacts, and gmail on Ubuntu 16.04 as of 12/14/2016.  You need to enable your Google account in the Gnome/Unity settings app.  System Settings -> Online Accounts -> Google.  (Click the add account button if Google does not appear as a choice and add it.)  You should see a message that Evolution Data Server has access to Google Calendar, Google Contacts, and Gmail, at least.  Don't leave the online accounts setup until you see those Evolution Data Server specs.  Then reboot and open Evolution.  If Google Contacts and Calendar entries show up, you're done.  If not, make sure you have a Gmail account entered under Edit -> Preferences -> Email in Evolution.  If you don't, set one up.  You may have to restart Evolution or even reboot Ubuntu to finish this process, I can't remember.  When you have your gmail account selected in the Email tab of Evolution Preferences, you should see the message, "This account was created through the Online Accounts service." 

I have set this process up on two different computers, both running Ubuntu 16.04.  Some trial and error seems to be required.  I had to delete and reinstall my Google account once or twice on both computers, for some reason.  Good luck.  If you succeed, it all works pretty well.  It even works keeping the high security settings on at Google.

If someone can offer better instructions than mine, please do so.  I'm sure there is a clearer way.
