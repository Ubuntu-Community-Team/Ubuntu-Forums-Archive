### Post by oldos2er on 2007-08-14
Under the 'File' menu in Nautilus, you'll see 'Connect to Server....' Type in the name of the ftp server you want to connect to. Nautilus will open a connection, and you'll see an icon on your desktop that should have the server name as its title. Click on this, and you can drag and drop files from there into another instance of Nautilus, for example into your Home folder (or wherever).
