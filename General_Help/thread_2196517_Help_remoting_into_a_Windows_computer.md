### Post by Mark Phelps on 2013-12-30
> would like to be able to remote into a Windows 8 computer desktop so that I have full access to the PC, but the person sitting in front of the Windows 8 pc can also use the mouse and keyboard (Desktop sharing)

Been a while since I did this, but from what I recall, you can not have two users working mice and keyboards at the same time.  When I remoted in to a Windows desktop, a popup appeared on the desktop asking permission, and when OK was selected, the remote user took full control of the PC.

Maybe there are products today that don't have this limitation, but you need to realize that Windows 8 is not designed to be used in a concurrent-multi-user mode.
