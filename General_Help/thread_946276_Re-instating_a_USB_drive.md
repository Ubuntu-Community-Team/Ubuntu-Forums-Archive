### Post by AndyCooll on 2008-10-13
As mentioned above.

GParted is part of the Ubuntu Live CD (System > Administration > Partition Manager IIRC) so you could reboot your pc and boot into a Live Ubuntu session.

Alternatively if you are working in an Ubuntu environment already you'll need to install GParted first before you can use it!

GParted is a graphical partition editor and is pretty intuitive to use.
