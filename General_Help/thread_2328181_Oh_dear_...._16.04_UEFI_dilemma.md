### Post by RobGoss on 2016-06-18
Hello and welcome to the forum, If this machine has Windows on it and you want to wipe it clean and install **Ubuntu** it should be very simple just run the live USB or DVD, choose **Wipe entire drive, **this will install Ubuntu and you won't have to deal with dual booting options such as UEFI or LEGACY. This guide will help you [http://www.ubuntu.com/download/desktop/install-ubuntu-desktop](http://www.ubuntu.com/download/desktop/install-ubuntu-desktop). If you would like to read up on how to dual boot and UEFI, this might help as well [https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI)

Your very welcome, Yes the new machines these days all have **UEFI **and can be a bit confusing. I do recommend if you have now interest in Windows then you should be good to go but if you would like to keep Windows them make sure you have a recovery disk for Windows just in case anything goes wrong. 

The most important thing you should know about **UEFI **is what ever bios settings Windows is installed Ubuntu must be installed in the same order, example 

If Windows is installed in **Legacy **them **Ubuntu **needs to be installed in the same manner, if Windows is installed in **UEFI** then Ubuntu needs to be installed in the same way. If you just want Ubuntu installed on this machine it should be straight forward

best of luck with your installation
