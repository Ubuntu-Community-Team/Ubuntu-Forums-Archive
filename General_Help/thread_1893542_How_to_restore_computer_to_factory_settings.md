### Post by sibleytr on 2011-12-10
I'm sorry to say that all depends on how you installed the O/S, if the system had a factory restore option, and if you can still access the restore function.

It would be best to Google your computer make and model for the restore process:

"How do I restore my <make> <model> back to factory settings".


Hope fully you didn't take out the hidden partition on your hard drive during the installation.


Just for future reference, if you do get your system reset there are other options for trying out Ubuntu without doing a full install.

### Post by pretty_whistle on 2011-12-10
Ubuntu can be reinstalled if you want just by using the same method you used to install it in the 1st place.

Windows 7 can be reinstalled either by Windows 7 recovery disks you made (IF you made any) or the install CD you bought (IF you bought one).

Windows 7 will not be reinstalled by reinstalling Ubuntu however.[SIZE=3]
