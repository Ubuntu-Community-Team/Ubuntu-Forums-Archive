### Post by Brinstar on 2010-08-27
had exact same prob as u and fixed it just now

1. ctrl-alt-f1 to login prompt
2. sudo nano /etc/X11/xorg.conf
3. add this to  

Section "Device"
    Option   "UseDisplayDevice"  "DFP" 
EndSection

Section "Screen"
    Option         "AddARGBGLXVisuals" "true"
EndSection

in addition to what is already in those sections, then 
4. sudo /etc/init.d/gdm start

should get a working system now
