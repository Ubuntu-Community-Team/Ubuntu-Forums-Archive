### Post by cleary on 2005-05-09
From what I can see, the current developer version of Gnoppix (0.9.99b1) **is** the Ubuntu 5.04 livecd. 

Without examining the isos in any great detail they have the same startup interface (the Morphix boot screen etc in the gnoppix stable release is gone) and you have to go through the ubuntu livecd setup dialog choosing the locale info.

I'm currently in the process of trying to create a dirty hacked up frankenstein of a livecd with Morphix basemodule 0.5pre5 and the mainmodule from the gnoppix 0.9.99b1 (which i'm pretty sure is ubuntu hoary livecd). Probably won't work (or may only work a bit) but hey, frankly the ubuntu/kubuntu desktop is my favourite, and the modular morphix livecd architecture is excellent so if I can combine the two in a way that preserves the best of both distros then I reckon it'll be a great disk :)

### Post by Stroller on 2005-05-11
I too have tried both the Ubuntu 5.04 and Gnoppix 0.8.2.2 Live CD's
Sadly, Ubuntu did not reach my expectations. I've tried it on two computers, an old Intel 333, and an AMD 1700. Booting on each went without a hitch, with both machines recognising the  mouse and sound card. On finalisation a CD icon was displayed on the desktop, along with a small (128 Mb) removeable drive already plugged into a USB slot. 

But regardless of what I tried, Ubuntu simply refused to mount the hard drive. Before anyone mutters 'newbie', might I add that I had no problems with Ubuntu's nearest 'cousin' Gnoppix, which easily mounted the hard drive. The same can also be said for both Knoppix and Mepis.

It was totally frustrating. After a protracted session, I simply gave up. This is not the best way to attract more people to the Linux fold. 

Might I suggest that in future upgrades Ubuntu follows the lead of Gnoppix. Newcomers would surely appreciate a clean bright desktop, with all hard/local drives already mounted and awaiting their use. Likewise, icons of the most often used programs already displayed in position across the desktop ready for use. For any first-time user new to a system, they will want something that is easy, familiar and simple to use, without any undue frustrations.

Sorry Ubuntu, but for the simple reason of being unable to mount the hard drive, I find it difficult to recommend the  live CD. By comparison, Gnoppix 0.8.2.2 is a street ahead.
