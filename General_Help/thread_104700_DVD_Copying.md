### Post by Sutekh on 2005-12-18
Did you let it finish?  How long did it take in the end?  
The time would depend on your hardware.

It takes my computer about 1~2 hours (if memory serves) with an AMD X2 3800 and 16X DVD.  That's using DVD Decrypter on Windows.

You can install DVD decrypter in wine on Ubuntu.  I'm planning to do this in the next couple of days.

Check out these HOWTOs if you'd like to give it a try.
[ General - Howto DVD Decrypter in Breezy]("http://www.ubuntuforums.org/showthread.php?t=96763&highlight=dvd+decrypter+wine") - for copying

[ Gnome - HOW-TO: Install DVD Shrink with Wine in Breezy]("http://www.ubuntuforums.org/showthread.php?t=78611&highlight=dvd+decrypter+wine") - for resizing-burning
