### Post by gmargo on 2011-03-02
> **geoffmcc said:**
> 
It used to show username@host now it just shows username ~


See the **bash(1)** man page, under the PROMPTING section.  This lists all of the escape sequences you may use. 

"\u" = username
"\h" = hostname
"\w" = current working directory

So change your "\u" to "\u@\h" to get username@host.
Remove the "\w" part if you don't want the directory.


I find color-specific escape sequences are too hard to read, so I use some handy variables:
```

# Ansi terminal foreground colors
cRED="\[\e[31m\]"     # Red
bRED="\[\e[1;31m\]"   # Red (bold)
cGRN="\[\e[32m\]"     # Green
bGRN="\[\e[1;32m\]"   # Green (bold)
cYEL="\[\e[33m\]"     # Yellow
bYEL="\[\e[1;33m\]"   # Yellow (bold)
cBLU="\[\e[34m\]"     # Blue
bBLU="\[\e[1;34m\]"   # Blue (bold)
cMAG="\[\e[35m\]"     # Magenta
bMAG="\[\e[1;35m\]"   # Magenta (bold)
cCYN="\[\e[36m\]"     # Cyan
bCYN="\[\e[1;36m\]"   # Cyan (bold)
cWHT="\[\e[37m\]"     # White
bWHT="\[\e[1;37m\]"   # White (bold)
cNOM="\[\e[0m\]"      # default

# prompt using variables:
PS1="${debian_chroot:+($debian_chroot)}${cGRN}\u ${bBLU}\w ${bGRN}\$ ${bWHT}"

# or a direct translation using unneeded nominals:
# PS1="${debian_chroot:+($debian_chroot)}${cGRN}\u${cNOM} ${bBLU}\w${cNOM} ${bGRN}\$${cNOM} ${bWHT}"

# original version:
# PS1='${debian_chroot:+($debian_chroot)}\[\e[0;32m\]\u\[\e[m\] \[\e[1;34m\]\w\[\e[m\] \[\e[1;32m\]\$\[\e[m\] \[\e[1;37m\]'

```

Just for fun I made one that showcases all the colors:
```

PS1="Norm: ${cRED}Red ${cGRN}Green ${cYEL}Yellow ${cBLU}Blue ${cMAG}Magenta ${cCYN}Cyan ${cWHT}White ${cNOM}\nBold: ${bRED}Red ${bGRN}Green ${bYEL}Yellow ${bBLU}Blue ${bMAG}Magenta ${bCYN}Cyan ${bWHT}White ${cNOM}\u@\h \!\$ "

```
