### Post by hikaricore on 2009-01-30
Little tip if you want your settings to stick.
After changing any setting or resizing the window you need to exit the program.
Once you exit the settings should save.  They will not save when the program crashes.

I have several older systems on my home network just for when others come over and want to use the internet,
Midori is nearly perfect for these systems where Firefox is very slow.  If only stability would increase..

As far as bookmarks I've never even looked at that functionality.
