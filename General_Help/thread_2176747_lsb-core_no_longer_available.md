### Post by oldos2er on 2013-09-26
The lsb-core package is available for 12.04: [http://packages.ubuntu.com/search?keywords=lsb-core&searchon=names&suite=precise&section=all](http://packages.ubuntu.com/search?keywords=lsb-core&searchon=names&suite=precise&section=all)

I don't think there's a replacement repository for Medibuntu, but if you need libdvdcss2, see [http://blogs.kde.org/2013/09/11/medibuntu-disappear-libdvdcss-now-direct-videolan](http://blogs.kde.org/2013/09/11/medibuntu-disappear-libdvdcss-now-direct-videolan)
