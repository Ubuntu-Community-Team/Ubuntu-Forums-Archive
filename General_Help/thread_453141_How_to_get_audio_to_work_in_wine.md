### Post by Famicommie on 2007-05-24
I am no expert, but I have found that preceeding the command with "aoss" can help make sound work in otherwise unruly apps. For instance, I have a game called "Grid Wars" that doesn't play sound when I use the command ```
gridwars
``` but works when I use the command ```
aoss gridwars
``` Give it a shot and lemme know if it works.
