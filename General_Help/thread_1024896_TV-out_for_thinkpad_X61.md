### Post by jimmy the saint on 2008-12-29
What video card do you have?  It sounds like you would definitely need that converter.  I have a thinkpad and it doesn't have svideo out either.  What you would need to do is find out which video card you have and then find the solution specific to that card.  For example, I have an nvidia card, and I can set up a second monitor via the "nvidia-setting" app and I think there is even a hotkey to cycle through a few setups, but I haven't tried.  There is another option for intel.

I believe that the new x-rander system allows you to manage monitor setups like you want via the system>administration menu, but I don't have intrepid so I cannot comment on this specifically.
