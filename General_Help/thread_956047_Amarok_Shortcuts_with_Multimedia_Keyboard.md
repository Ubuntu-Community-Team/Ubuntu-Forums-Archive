### Post by tCarls on 2008-10-23
Try going to Tools -> Script Manager -> Get New Scripts in Amarok. Download, install, and run Gnome Multimedia Keys. You may need to configure System -> Preferences -> Keyboard Shortcuts.

If that doesn't work then you may need to use xev. It's a little tricky but usually works after a bit of effort.
