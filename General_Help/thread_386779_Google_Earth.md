### Post by taurus on 2007-03-17
Assuming you've downloaded GoogleEarthLinux.bin to Desktop, open a terminal and do

Applications -> Accessories -> Terminal
```
cd ~/Desktop
chmod 755 GoogleEarthLinux.bin
./GoogleEarthLinux.bin
```
