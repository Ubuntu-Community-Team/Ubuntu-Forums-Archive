### Post by WRDN on 2008-09-23
When you use Open Office, save the file with the Office formats (e.g. .doc) so they can be opened in Microsoft Office. Open Office supports all Microsoft Office formats (including the Office 2007 formats).
If you are writing simple text documents with no images, I would save the file in the .txt format, as this allows a most, if not all, computers to read and write to the file.

### Post by DrOlaf on 2008-09-23
You should be fine. Openoffice apps can save documents in MS Office format. In more detail, OpenOffice 2.4 can save in Office XP/2003 format (doc, xls, ppt etc), but not in Office 2007 format (docx, xlsx, etc). If you keep to the older format - which is readable by way more computers than the new one at the moment anyway - you shouldn't have problems.
