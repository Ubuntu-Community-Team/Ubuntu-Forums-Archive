### Post by cmont899 on 2012-05-04
I notice in my install of Firefox I have many more "Type" options under the "Page Setup" tab in File -> Print. Also look for any options named "Continuous Narrow" or "Continuous Wide" and make sure that "Ignore scaling" is checked. 


You also may want to make sure orientation is set to Portrait and not landscape.
