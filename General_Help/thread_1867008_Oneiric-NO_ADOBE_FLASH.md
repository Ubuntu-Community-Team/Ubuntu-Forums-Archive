### Post by mglennpooh on 2011-10-22
Three items -

#1) bring up Firefox; type & go to adobe.com; click on downloads, then, flashplayer, then, download, then,APT for Ubuntu,  then, wait for Ubuntu Download menu to appear, then, select ok for Software Center, then, wait for Software Center to appear, then, type adobe, then select Adobe Flash Plugin Installer(new) vs 10(old). At this point you should close Firefox before proceeding to install Adobe or else it'll take three times as long to install, then, click install.

#2) Chrome & Chromium already have Flash & Update built into their software - no need to do anything. 

#3)  Click the silver Ubuntu button top left, then, type s, then select Software Sources which is the simplified 11.10 replacement for Synaptic Packages, then, click Other, then check Partners & Independent (but not the source code for these unless you know what to do with it),then, close here, then, bring up Update Manager, then click on CHECK to refresh the search cache to include the Partners & Independents. 

Some of the above is specific to Ubuntu 11.10 but much the same also applies to earlier versions.
