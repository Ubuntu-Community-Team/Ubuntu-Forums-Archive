### Post by linuxinstalledfromhdd on 2011-05-31
It should always be made very clear to all users that the latest version of Ubuntu is in a constant state of development, and they are wilfully participating in in the latest release's development by it's very use. The older releases have already gone through most of the debugging process for those particular releases.  And if I knew someone that was completely new to linux, or wanted to give it a try, they should try the last previous release from the current release on their system from a live disc, and really give it a good test to see if their system can handle it or not, before committing to a hard drive installation.  I love using live sticks of Ubuntu on USB to test overnight hardware to see if it is going to work or not too.  Just a suggestion to anyone that is interested.:D
