### Post by RAOF on 2006-12-05
My favourite feature of Ubuntu over Windows?  Try:
```
sudo aptitude install python-pyparsing
``` from a terminal.  You've just installed the pyparsing Python library.  It's installed, byte-compiled, and in your python path, just waiting for you to *import pyparsing*

Probably the best place to start is with something that you want to **do**.  Operating systems are, after all, tools to get actual tasks done.

So, what do you want to do?  We can show you the Ubuntu way to get it done.
