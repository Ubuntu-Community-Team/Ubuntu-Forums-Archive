This is now solved, I found out how (using Heliode's tutorial [http://ubuntuforums.org/showthread.php?t=35087](http://ubuntuforums.org/showthread.php?t=35087) to compress the files then using lxsplit [http://lxsplit.sourceforge.net/](http://lxsplit.sourceforge.net/) to break it into CD sized chunks).
