### Post by dcstar on 2007-10-18
> **kelbizzle said:**
> I have the .evolution folder from my feisty install on the desktop of the gutsy install. What should my next course of action be?

Firstly, go into Edit-Preferences-Mail Preferences-Junk and then specifically set whatever Default Junk Plugin you want to use.

The Gutsy version of Evolution does not automatically select this when you upgrade, so I had to set it to use Bogofilter to get my junk mail filtering working again!
