### Post by dazift on 2010-10-27
I burned the iso to a cd and used that same cd for both the wubi and the complete install. Both were 20G installs. For the complete install, I partitioned the drive to have 20G partition at the end and installed Ubuntu using the automated install which partitioned about 800Mb for the swap. The biggest time difference I found to be the time after the login screen. you can hear the HD running for a while before everything starts working. also opening programs takes a little bit longer in the complete install, but it's barely noticeable. the major thing for me is the slow and choppy flash. In the wubi install it worked perfectly and I was entirely impressed with it's performance.
