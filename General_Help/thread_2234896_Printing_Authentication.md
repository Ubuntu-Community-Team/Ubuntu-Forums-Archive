My Ubuntu 14.04 OS laptop is wireless connected to a Canon MG3100 printer via a Windows Homegroup. The printer is wired to the host, Window 7 OS, PC which is my wife's.
My problem is that, every time I want to print I have to authenticate the process by entering my wife's username and password. I assume it is her password that is required because, silly as it may seem, we both use the same password.
Is there any way I can make the authentication automatic?

---

### Post by cariboo on 2014-07-18
Open System Settings->Printers, right click the printer and select Properties. In the Properties window select Access Control and make sure Allow printing for everyone except these users is selected.
