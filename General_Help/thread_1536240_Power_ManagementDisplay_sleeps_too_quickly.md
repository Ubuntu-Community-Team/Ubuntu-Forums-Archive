### Post by watupgroupie on 2010-07-21
There's two different options under power management. Under actions set the sleep time and then under display you need to set a sleep time as well.

---

### Post by mc4man on 2010-07-21
Check in Preferences - Screensaver and make sure you're not getting a blank screensaver and locked screen

(if the display is actually going to sleep before the power management setting then set 'regard ... idle' time *to less than* the time the display is going to sleep in.
