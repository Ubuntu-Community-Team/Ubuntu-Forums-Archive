Hi,

just in case anyone's interested. 
If you do want to use Lyx(qt) in Gnome and want to look it more beautiful, you'll have to install many kde-things (to get kcontrol and some themes). Yet this causes a problem with german umlauts in menus. 
This is a bug that has been fixed in Lyx 1.3.5. As there's no package for debian at the moment, you'll have to install e.g. the Fedora package (I used alien to convert and dpkg-i to install. Make sure you've previously uninstalled ubuntu packages)
