### Post by os56k on 2009-07-17
If you are going to run GParted when you dont have ubuntu installed you can boot from the ubuntu CD and choose the first option to test ubuntu without altering your computer and after booting to ubuntu you can do partitioning.

You must note that for ubuntu 9.04 there is not individual install and live CDs. both of them are on one CD.

### Post by devosion on 2009-07-17
To install g-parted just run this command from terminal...

> 
sudo apt-get install gparted


It will be listed as partition editor under your system menu after the download is complete.
