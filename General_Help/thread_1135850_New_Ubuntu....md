### Post by jsowoc on 2009-04-24
One thing that you probably should know with numbering, is that Ubuntu is different than most programs. You'd expect a program to go 1.0 -> 1.1 -> 2.0 -> 3.0 -> 3.1 -> 3.2 etc. Ubuntu does not.

The version "9.04" which is current simply means that it came out in 2009 in the 4th month (i.e. April). An upgrade from 8.04 -> 8.10 is the same size jump as 8.10 -> 9.04.

Depending on how often you want to upgrade the software on your computer (how long have people been running Windows XP?), you can either upgrade every 6 months or every 2 years. Your version of 8.04 will have a direct upgrade path to 10.04 when it comes out in April 2010. If you want to upgrade sooner, you have to upgrade 8.04 -> 8.10 -> 9.04 -> ...

As the poster mentioned above, you can always download a new 9.04 CD and overwrite your previous install, losing any custom packages you may have added/removed since you installed.
