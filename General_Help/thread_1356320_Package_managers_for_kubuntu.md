### Post by GameManK on 2009-12-16
The default one is actually KPackageKit.  Adept is available from the repositories, but it is not really maintained.

If KPackageKit doesn't work for you in some way, Synaptic is also available from the repositories, as well as Ubuntu's Software Center.
