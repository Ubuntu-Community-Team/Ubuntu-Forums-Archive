Is it possible to display global download&upload speed information on the panel?

---

### Post by Intrepid Ibex on 2009-01-01
Right click panel -> Add to Panel -> System Monitor -> + Add -> Right click System Monitor -> Preferences -> Network -> X Close

There you (should) go!
