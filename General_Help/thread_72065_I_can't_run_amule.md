### Post by oliwally on 2005-10-05
> I installed amule and I can run it ony once. Then it was impossible to run. I clik on aplication icon and nothing ocurs.

Ok, I'm no expert, but when that happens to my applications I use the PerformanceMonitor to 'kill' the application. Usually it's already running (in some crooked way) and all instances of it need to be 'killed'. Then try to start the amule again. Btw, on my computer the Performance Monitor (or Process Table part of it) is brought up by pressing ctrl+esc. I'm using Kubuntu though...

> How can I totaly remove amule from my ubuntu, and install again? 

In Synaptic choose amule, right click on it and then choose either 'Mark for removal' or 'Mark for complete removal'. I'd try the first one first. Then click on 'Apply' at the top and follow the process through. If the first one doesn't work, try 'Mark for complete removal'.

Worked for me on other applications and in Kubuntu, but I'm no expert.
