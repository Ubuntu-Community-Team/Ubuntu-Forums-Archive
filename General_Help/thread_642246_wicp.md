### Post by Bigbird999 on 2007-12-16
Is your router set to broadcast its SSID?  Maybe that is the issue?  

There is a hidden network tab in the wicd panel that allows you to type in your network name if it is not broadcast.  Then it will find it.  Once you can see it, you can connect and/or set it up to automatically connect.  But it is easier just to enable broadcasting of the ssid in the router config.

BTW it is wicd not wicp.
