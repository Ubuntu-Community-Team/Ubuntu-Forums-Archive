### Post by Ryan450 on 2008-08-14
Hey there,

I've been using my machine with a dual-boot setup, ubuntu/windows to play games. I use Ubuntu far more then I do windows even for a gaming machine, only a few titles I own stay in the windows partitions.

Free Games that run in linux here's a quick list:

Second Life (Online, but not so much a game, more a 3D chat room)
Freeciv (online)
Nexuiz (online)
OpenArena (online)
Frozen Bubble (direct IP)
SuperTux (offline)
ZSNES, DGEN, VISUALBOY ADVANCE, (Emulators)

Commercial Games I've gotten to run in Linux:

Enemy Territory Quake Wars
Quake 1, 2, 3
Unreal Tournament 99/2004
Warcraft III/Frozen Throne (Using Wine)
World of Warcraft (Using Wine)

The above list is my personal favorites that I run in ubuntu. I only keep 2 or 3 games that I've not gotten to run smoothly in Ubuntu yet played solely in my windows partitions.

Hope that helps
