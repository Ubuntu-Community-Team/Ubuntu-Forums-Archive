### Post by Krytarik on 2011-01-04
Check in "System -> Preferences -> CompizConfig Settings Manager -> Window Decoration" if the value for "Shadow windows" is "any", the default (assuming you have CCSM installed).

Why "gksudo nautilus", does Alt-F2 "killall nautilus" bring back the shadow also?
