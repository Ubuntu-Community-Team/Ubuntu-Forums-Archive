### Post by schmolch on 2008-09-19
From the xournal forums:
```

And yes, 0.4.1 definitely doesn't work well in Hardy. At some point there
was talk
of the maintainer incorporating a patch to 0.4.1 into the Hardy package,
but I guess it hasn't happened...

```

```
If you want to upgrade to the current version of xournal (which has great
new features) you dont have to compile it yourself.
You simply can go to packages.ubuntu.com, change the Distro-Selection to
Intrepid and enter xournal into the searchfield.
Then you have to choose betwenn a i386 and amd64 (64bit systems) -package
and can download and install the latest version of xournal.
It will not pull in any other dependencies from Intrepid, just xournal
```
