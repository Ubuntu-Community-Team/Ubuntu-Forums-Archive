### Post by shane19174 on 2008-11-02
You can install virtualbox through synaptic (open source version) or download it from Sun (not completely open source, but with extra support for USB, etc.). It will run fine on Intrepid. And I think you should be able to run any Windows version EXCEPT Vista with your RAM. You have to allocate a certain amount (say 512 MB) to Windows; that amount is no longer available to Ubuntu while running your virtual Windows. You'll have to see which settings are best for your needs (depends on whether your Windows or Linux apps are RAM-heavier).
