### Post by william_nbg on 2006-03-21
I'd have to know how you installed it??

I set up .blackbox/styles/ in my home directory with all my config files, including styles, in it.

I think a default installation puts it in your /usr/share/blackbox/styles/

you have to put it into the styles directory and it will show up on your styles menu.

You can set the location of the styles folder in your blackboxrc file which is the main conf file for blackbox.

Good luck
