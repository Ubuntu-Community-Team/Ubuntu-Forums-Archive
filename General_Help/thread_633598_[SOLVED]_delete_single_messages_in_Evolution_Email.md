### Post by rmockler on 2007-12-06
How I delete message from the trash folder in Evolution, is to right click on trash and then click on Empty Trash.  I guess the theory behind that is if you sent it to trash, you want to delete it.  I don't believe there is a way to delete a single message from trash.  If you aren't sure you want to delete it, DON'T send it the trash folder, either by moving it there or clicking on delete.  Satisfies me!
