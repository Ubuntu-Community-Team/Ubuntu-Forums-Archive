### Post by grahammechanical on 2011-07-08
May I make a couple of points?

1) Both of the companies that you link to are providing a service which  you must pay for. What is the cost? How long do you keep paying for the  license to use the software? Do you have to buy the hardware from them  as well? These are things a person must think carefully about before  making a decision.

2) How well this will work will depend on how powerful the host CPU is  and how much RAM there is. Do not try to do this too cheaply.

3) There seems to be two ways of doing this. (a) connecting USB mouse  and keyboard through a USB type of hub and then connecting the monitor  directly to the video card. This requires more than one video card or  adapter. (b) connecting mouse, keyboard and monitor to a special hub  that then connects to the computer using (I think) a USB cable. The  number of stations is dependent on the number of USB ports on the  computer. Thinking about this is part of the purchasing decision.

3) this is not a cheap way to have a network but it is cheaper and if  you accept the limitations that the stations must surely have (not doing  processor or video intensive stuff) then it looks like an economical  way of using resources.

4) Look up the advertisements for computer servers, especially  expandable systems and you will see that they are running CPUs and  memory that many people now have as standard in their desktop machines.  Why not turn a desktop machine into a type of server?

Regards.
