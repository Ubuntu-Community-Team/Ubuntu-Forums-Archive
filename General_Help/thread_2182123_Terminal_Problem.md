### Post by medior on 2013-10-20
Hi,
you probably have the option enabled to allow different keyboards for each window. Go to 'System Settings' -> 'Text Entry' and select 'Use the same source for all Windows'. Make sure that in the left part of the window you only have the input sources listed that you need, and that the default source is at the top of the list.
