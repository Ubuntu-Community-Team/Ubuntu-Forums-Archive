### Post by j-strap2 on 2006-08-10
In synaptic, just make sure the Tor box is checked and click 'Apply '. While you are at it, make sure the Privoxy package is installed.

More importantly, don't use the default repository version of Tor - it is old and deprecated. Update your repositories as detailed here and use the Update function in synaptic to update to the latest stable Tor package.

[http://wiki.noreply.org/noreply/TheOnionRouter/TorOnDebian]("http://wiki.noreply.org/noreply/TheOnionRouter/TorOnDebian")
