### Post by banjobacon on 2009-06-26
When using Firefox, flash sometimes works and sometimes doesn't, depending on what site I'm visiting. Youtube works consistently. Myspace does not; it instead ask me to install Flash. Another site <[http://www.humpdayfilm.com/](http://www.humpdayfilm.com/)> says," You need to have javascript enabled, and the flash player installed to view this website." Javascript is enabled.

So far, all these sites work well when viewed in Epiphany. This has been my workaround for a while, but I finally decided to fix the problem. Any clues as to what's causing this?

I currently have the Flash package from the repository installed. I have in the past installed Flash manually from Adobe's site. I thought this might be the problem, some sort of conflict between a new install and a previous manual install, but if that were the case, wouldn't Flash always not work?

### Post by banjobacon on 2009-06-26
I would delete this thread if I could. I replaced Adblock with Adblock Plus, something I had wanted to do for a while. That flixed the problem. I don't know if the extension was to blame, or it was filtering something it shouldn't have. Either way, all is good now.
