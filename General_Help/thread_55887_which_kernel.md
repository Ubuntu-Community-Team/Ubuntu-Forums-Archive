### Post by varunus on 2005-08-10
Quick Legend:
686 - Intel Processor
k7 - AMD Processor

So you want the linux-k7 package from synaptic.  Its optimized especially for AMD processors.  The k7 kernel has all of the advantages the 686 kernel does, along with AMD specific ones.
