It sounds like you could have a hardware issue, possibly your HD.

If you have any critical data on your PC back it up asap.

There are lots of Hardware testing tools available to download, and a memory testing utility is included with the Ubuntu install CD.

Check your memory with the Ubuntu Cd and I'd recommend downloading something like Hiren Boot CD to test your HD.

### Post by Warren Watts on 2007-09-25
In my 20 or so years of experience with hard drives, attempts to isolate bad sectors in hard drives have almost always ended in failure.  Over time, more and more bad sectors appear and just cause more grief.  My advice would be to backup what you need to save and replace the drive.
