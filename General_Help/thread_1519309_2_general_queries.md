You do not need to delete Windows to try out Ubuntu. However, I suggest you first try a LiveCD before installing to see if everything works and if you like it.

Download the .ISO, burn it to CD at the lowest speed, boot the computer making sure it is set in BIOS to boot from the CD and check it out.

If you do decide to install Ubuntu here are some links that will help you:

[https://help.ubuntu.com/community/DualBoot/WindowsFirst](https://help.ubuntu.com/community/DualBoot/WindowsFirst)

[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)

[http://sites.google.com/site/easylinuxtipsproject/Home](http://sites.google.com/site/easylinuxtipsproject/Home)

The recommended method is to install Ubuntu after Windows is installed.

As far as connecting to the wireless network at your university is concerned, in _most_ cases Ubuntu will recognize a network and you just need to enter a password (if relevant) to connect.

If you have problems, I would think the IT department would be willing to help or you can post back here to the forums.

Good luck!
