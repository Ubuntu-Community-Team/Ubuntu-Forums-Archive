### Post by Captain Giraffe on 2008-09-17
```
sudo apt-get install mysql-server
sudo apt-get install mysql-client
sudo apt-get install mysql-admin
``` should get you the necessary software.
If you already has access to a mysql server you dont need the first package.

Or go to System / Adminsitration / Synaptic package manager
do a search for mysql and check the stuff that seems necessary.
