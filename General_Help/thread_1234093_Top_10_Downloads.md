### Post by fennec_fox on 2009-08-07
graphics drivers - from the restricted drivers tool

**all of the following you can get from add/remove**

SMPlayer - I prefer it over vlc for playing videos

compiz fusion (if your comp can handle it) - for eye candy, usability and special OS features.

ntfs config tool - This may already be included in the newest ubuntu versions but, it lets you control read/write access to ntfs drives

frozen bubble - it's my favorite desktop game

flash - you can get this from the adobe site or from add/remove i believe

open office suite - I'd get the entire suite from add/remove if you want *everything*

winff/ffmpeg - for all your video/audio converting needs

wine - for running any windows software you would like (I know some users who are against this idea but, I like to keep the tool around)

ubuntu restricted extras - gstreamer media codecs, MS fonts, java, etc. This gives you a lot of functionality immediately. 
**(I'm not sure this is available in every country, I could be wrong though)** 

easytag - a great tool for editing mp3 tags (really helpful if you have an mp3 player)

dia - if you need to make diagrams


Those should get you out of the box and cover basic needs. Firefox, pidgin, evolution, zip/rar tools and GIMP should already be installed.
