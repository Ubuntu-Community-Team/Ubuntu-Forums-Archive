### Post by antaios256 on 2011-05-09
While i dont use powermizer my self the difference is simply Maximum Performance runs your hardware at its designed levels for example i have an i7 with turbo boost. typiccaly each core is able to go up to 1.6 but usually runs at 933 to save power and only increases when performance is needed. (thats adaptive) where max performance would run all cores at 1.6 all the time. uses more power creates more heat but computer would be more responsive for cpu/gpu intensive programs
