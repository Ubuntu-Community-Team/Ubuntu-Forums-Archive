### Post by KdotJ on 2010-05-11
One main difference, Ubuntu Server has no desktop... its just command line. Just something to bear in mind if you're going to run this site from your normal computer and not a separate box lol.

I use Ubuntu Server on my home server, which hosts my website as well as being a file server. I run it headless, so the the box just sits quietly in my spare bedroom. All admin work I need to do on it is done via SSH fro my main computer (or from anywhere if needed). Very handy indeed...

As for "inconveniences", with Ubuntu Server, you will have to know at least some command line stuff, for dealing with files and of course configuration (of apache for example). But this is easily learnt and plenty of help here and else where on the net.
