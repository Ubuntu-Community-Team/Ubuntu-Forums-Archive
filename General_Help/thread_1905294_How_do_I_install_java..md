### Post by Basher101 on 2012-01-06
open your software center and search for "Ubuntu restricted extras"

it is a bundle of things that are not allowed to be pre-included in ubuntu, such as java, flash, mp3 codecs and others. It is basically one of the first things you should install after you installed Ubuntu on your computer.
