Okay, I see the problem. You should install mdbtools & mdbtools-doc from the ubuntu repositories either using the Ubuntu Software Center or:

in a terminal...

sudo apt-get install mdbtools
sudo apt-get install mdbtools-doc


Once you've installed them from the repositories you will find the html documentation in:

/usr/share/doc/mdbtools-doc/html
