### Post by AlphaLexman on 2010-12-04
Not all .pdf's can be written to, some have encrypted passwords, and some have non-encrypted form-fields you can fill out, and some are not encrypted at all, but may be non-writable.

Check out 'man pdfinfo' for more details (It may not be installed by default.)

### Post by wannacme16 on 2010-12-07
PDF Editor - open program and find the open file tab on the left side of the screen under file tab. Navigate to your file and open, your file should be open and displayed within the PDF Editor program. To add text (such as name, address, telephone, etc. ) go to the page tab and down to the add text and click on it. Now whenever you click in the document will open a window to allow text to be added to the document at that particular location. When finished save the document and reopen to check for accuracy. Hope this helps.
