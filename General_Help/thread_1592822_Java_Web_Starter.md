### Post by luigi_mb on 2010-10-11
First of all, did you install the Sun Java packages? If not, uninstall the openJDK and related files, and install Sun Java (jdk, jre, plugin, etc.). Java Web Starter is "javaws" in /usr/bin .

/luigi
