### Post by MARP1961 on 2011-06-23
If you wish to return to earlier versions, the easiest way is to reinstall with a CD of the desired version. If you back up all your personal files on flash drives, CDs or an external HDD first you can then just let the installer wipe everything clean. Obviously don't do this if there are partitions you want to keep (dare I mention Windows)? If you want help with partitions you can set them manually and we can give advice on this option if you want.
