### Post by grahammechanical on 2011-10-21
From the beginning the under-lying desktop environment for Ubuntu was Gnome. It was developed over the years up to version 2. Now the Gnome people have developed something slightly (or not) different that they call Gnome 3.

Ubuntu 11.10 is based on Gnome 3. System Monitor tells me that I am on Release 11.10 (Oneiric), Kernel Linux 3.0.0-10-generic, Gnome 3.2.0. I am using Unity 3D.

There is such a thing called Gnome shell which is a user interface like Unity and it has a similar look to Unity. It can be downloaded from the Software Centre.

So, you already, if you are on 11.10, have Gnome 3. If you have installed gnome shell then I would guess that the option of Gnome in the login menu is referring to Gnome shell.

See the link:

[http://www.omgubuntu.co.uk/2011/10/gnome-shell-ubuntu-11-10-guide/]("http://www.omgubuntu.co.uk/2011/10/gnome-shell-ubuntu-11-10-guide/")

Does the desktop look anything like those images? Notice the bottom panel that would not be there in Ubuntu (Unity).
