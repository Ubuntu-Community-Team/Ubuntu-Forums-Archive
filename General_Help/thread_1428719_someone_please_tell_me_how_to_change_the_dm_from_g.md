### Post by mcduck on 2010-03-13
How to change your default desktop environment depends on which Ubuntu version you are running. in older versions you simply select the one you want to use in GDM, and you'll be asked if you want to set it as default as well. In later versions go to System/Administration/Login Window and you'll and an option for setting the default session.
