### Post by virsto on 2010-03-05
I have a dual boot system (Windows Vista and Ubuntu 9.10).
Both systems use the same email server. 

I download my email (Thunderbird) into folders on my Windows Vista system, and this I would like to keep unchanged.  However, I would like to **not download my emails on the Ubuntu system** where I am using Evolution 2.28.1. I only wish to be able to read them with Evolution; but, not download them. How can a stop email downloads with Evolution while still being able to read them?

I hope that I have made my request for help clear.

--V

### Post by plucky on 2010-03-05
Assume you are using POP server

**Edit > Preferences > select email Account > Edit > Receiving Options** and select "leave messages on server" option.This will allow you to read the messages that haven't been downloaded to the Vista account and leave them on the server.


Or you can use IMAP server if you have the option with your email provider.

Good Luck
