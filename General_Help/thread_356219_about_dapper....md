### Post by tkjacobsen on 2007-02-08
Dapper or actually Dapper Drake is just a version of ubuntu like xp is a version of windows. Dapper is not the newest version however it has long term support, that means that the system is officially supported for longer time, but the software is not as up to date as in the newest release the Edgy Eft. Both of them can be downloaded from [www.ubuntu.com](www.ubuntu.com).

If you just want to try it out download the "desktop" cd, which is able to run ubuntu entirely form the cdrom without touching your harddisks and other Operating Systems
