### Post by TBABill on 2010-09-20
The first problem sounds like just the screensaver password needs to be removed. Click settings>administration>screensaver (I think....going from memory here) and uncheck the box that states "require password" or "lock screen" or something to that effect.
 
To dual boot does not require separate physical hard drives. The LiveCD can partition a single hard drive to meet your requirements.
 
The software from Windows issue is complex. Wine can run some Windows programs directly, but there is likely an equivalent on Ubuntu you can find and install for free from the Ubuntu Software Center or Synpatic.
 
EDIT: Yes you can place them on separate drives. I have not done so therefore I cannot provide any guidance on it, but search the forums and there are lots of folks who have.
