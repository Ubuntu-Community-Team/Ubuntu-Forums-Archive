### Post by torfig on 2011-05-28
Found the solution, it is really simple.
First, open your file browser.
You need to have saved a html file there.
right click the html file and choose Open with > Other, choose Firefox or whatever browser you use.
Mark the "Remember application association for this type of file"
That's it. 
Restart evolution and now it should be able to open html links:p
I imagine this could be done with all other file types you might have trouble opening in Evolution.
