### Post by mcorr on 2011-05-02
Hi saresu.

I did exactly the same thing and had the same problem.  Here's how i fixed it.  If you open a terminal (ctrl+alt+t) then simply type 'ccsm' and hit enter, it should open compizconfig, from where you can re-check Enable ubuntu unity plugin.  

Hope it works for you.
