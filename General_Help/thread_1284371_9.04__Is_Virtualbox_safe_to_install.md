### Post by lovinglinux on 2009-10-06
> **thelugnut said:**
> One of the reasons I departed from MS Windows was because of the constant warning about viruses, worms, and so forth.

I would like to install Virtualbox  and for a couple of programs I need, I want to install Windows XP on the Virtualbox.

You are safe. A virtual machine is a self-contained environment. Besides, Windows virus won't run on Linux, even if they were able to escape the virtual machine.

Nevertheless, you should also take the necessary security measures if you use the network in the VM. I'm not sure if an intruder could get access to unauthorized stuff on Ubuntu through a security flaw in the VM. Probably not, but I'm not an expert to be sure.
