### Post by Nikayah on 2008-06-11
I want to know if (or how) to re-open all the windows that were open on shutdown, on startup. I think I got this working before, but then openarena messed up my computer so i had to reinstall ubuntu. So it would be greatly appreciated if you could help me out.

---

### Post by haldean on 2008-06-11
You can have Gnome reopen the windows when you last logged off by going to System -> Preferences -> Sessions -> Session Options then check the box next to "Automatically remember running applications when logging out."
