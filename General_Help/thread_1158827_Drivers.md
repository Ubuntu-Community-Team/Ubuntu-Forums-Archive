### Post by mb_webguy on 2009-05-14
A router is an external networking device.  You don't need drivers for it.  You computer communicates with it using standard networking protocols.  

If you're trying to connect to the router wirelessly, you may need a proprietary driver for your computer's wireless networking card, or perhaps need to use the Windows driver through ndiswrapper, but you certainly don't need a driver for the router itself.
