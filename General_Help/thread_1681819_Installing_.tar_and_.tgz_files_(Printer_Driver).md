### Post by Rhizoid on 2011-02-04
They're both archive files - the default action for both should be to open them with the system archive utility, much like opening a zip file with winzip or winrar in Windows.

Once they're open you'll be able to view the contents easily enough - there's a guidemp210series-pd-2.80-1.tar.tar file in the printer driver, open it and it's like opening another zip file.  Open the next and you'll get a MP210series folder... open that and double click on the guide index.html and you'll get a browser window with instructions ;)

Sing out if you get any more strife ;)
