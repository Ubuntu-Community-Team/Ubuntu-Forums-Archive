### Post by FeraTech on 2008-08-08
While I enjoy how in many aspects Ubuntu is much more powerful than Windows. The area it is most lacking in is quite easily it's networking and internet connection sharing.

In comparison with a Windows machine you can set it to  be a router within seconds and a couple of clicks. 

While this would seem to be the perfect job for an Ubuntu machine it is a major hassle to get DHCP and internet connection sharing properly set up.


You use to be able to use "Firestarter" and a lot of this could easily be configured. However with the release of Gutsy Firestarter's "DHCP" package is no longer even in the repos.

The last resort is to search the forums for a HowTo which in many cases does not work and of course is a hit or miss operation.

If anybody has any easy howto's or a package that does this I would be forever grateful. I have spent quit a bit of time trying to get DHCP to work and after 5 machines have given up on it. Another problem is that once DHCP is configured getting rid of it is very difficult and it's hard to reconfigure the cards back to how they were before.

In summary, at least in my experience this aspect of Ubuntu needs a major revamp unless I have overlooked something.
