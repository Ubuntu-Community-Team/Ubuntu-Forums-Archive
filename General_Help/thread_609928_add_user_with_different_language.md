### Post by mts-fi on 2007-11-15
Ok, try this!

Restart computer (or log out). 
When you are in login window, press F10 and you'll get menu where is a line change language.
Choose that and you get a list of languages installed in your computer, select one you want to use with your second user.
Next you'll be asked if you want to change the language of your login window. Answer no if you want to keep your login window like it is now.
Now select the user of which language you want to change and log in. You will be asked if you want to use selected language (for this user) always or just this one time.
When logged in you must answer if you want to change users foldernames too.

With these instructions you should have two users with different languages in you computer.

I hope this works for you too.
