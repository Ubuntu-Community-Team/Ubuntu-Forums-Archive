### Post by michy99 on 2009-04-29
in your xorg.conf file change the monitor section like this, but use the correct figures for HorizSync and VertRefresh.

```
Section "Monitor"
	Identifier	"Configured Monitor"
	HorizSync       30-85
        VertRefresh     50-160

EndSection
```

Reboot and see if it helps.
