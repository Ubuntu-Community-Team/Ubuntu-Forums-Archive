### Post by digithal on 2011-12-08
The easiest solution is to open terminal and type:
```
gconftool --recursive-unset /apps/panel && killall gnome-panel
```
Otherwise, add new panel by following [**this guide**](http://askubuntu.com/questions/45426/how-to-restore-deleted-panel-on-desktop).
