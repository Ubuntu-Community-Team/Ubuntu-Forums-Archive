### Post by maheshasolkar on 2009-06-12
I found Chromium and Chrome for Linux to be roughly of same quality. You can get Google Chrome for Linux from (Look under 'For Linux' section):

  [http://dev.chromium.org/getting-involved/dev-channel](http://dev.chromium.org/getting-involved/dev-channel)

Please read this before you install Google Chrome:

  [http://blog.chromium.org/2009/06/danger-mac-and-linux-builds-available.html](http://blog.chromium.org/2009/06/danger-mac-and-linux-builds-available.html)

It is not even an alpha yet. It is recommended that you don't use it as your primary browser.
