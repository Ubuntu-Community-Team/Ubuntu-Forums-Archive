### Post by magnusbb on 2006-04-23
I did a little research on that some time ago, and I didn't find it out. I would guess that it is not possible at this time. One thing you can try out, is Brightside. It doesn't exactly do what you want (I know that scrolling fx. is possible in KDE), but it does offer some nice options to it, and is really configurable.

[http://catmur.co.uk/~ed/main/brightside/](http://catmur.co.uk/~ed/main/brightside/) for more information about it.

And a simple:
```
sudo apt-get install brightside
```

to install it.

Good luck.
