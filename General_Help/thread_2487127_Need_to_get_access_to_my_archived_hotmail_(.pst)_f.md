### Post by Holger_Gehrke on 2023-05-24
AFAIK there's no way to directly import .pst to Thunderbird. You can supposedly convert the .pst to mbox using some tools and then import that but I haven't tried that because I don't have any pst-files and I don't use Thunderbird. What I do use is Evolution and that does have a function to import pst-files (File->Import->Single File).

Holger
