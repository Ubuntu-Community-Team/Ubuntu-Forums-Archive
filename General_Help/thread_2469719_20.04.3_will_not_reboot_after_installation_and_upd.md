### Post by yancek on 2021-12-08
In the boot repair output, line 30 shows the current boot is 0001 and line 34 shows that to be the Linpus OS.  THis is a Linux derivative which has been preinstalled on some Acer computers for years with a not very good reputation.  Use the BIOS firmware or efibootmgr to change the EFI boot priority to Boot0000 which is Ubuntu.  Ubuntu is an EFI install so if you have an option to disable Legacy/CSM boot, disable it.

I see that you

- run in UEFI mode with secure boot disabled

- run Ubuntu live (not Linpus, although the BIOS thinks so)

- boot Ubuntu 20.04.3 LTS amd64 from drive sda
