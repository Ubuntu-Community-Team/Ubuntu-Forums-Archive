### Post by Vaphell on 2011-01-12
themes are about colors, icon sets and such, they don't modify your desktop layout
Assuming you use gnome environment, it's possible to remove the panels entirely or modify them to suit your needs (right click on them to access their properties) or replace them with something else. It's all up to you.
There are also other desktop environments like KDE (kubuntu), Xfce (xubuntu), LXDE (lubuntu), each follows different philosophy in usability, bling and system requirements - you can install them and switch at the login screen (combo box at the bottom bar).

and if you want dock for more eyecandy (that thing at the bottom of that screenshot in post #4) install Gnome-Do/Docky or Avant Window Navigator

### Post by snowpine on 2011-01-23
Fluxbox is a very nice, fast, and lightweight windows manager. You can easily install Fluxbox in your existing Ubuntu install using the Software Center or the terminal commands:

```
sudo apt-get install fluxbox
```

Then, log out, and from the login screen, click Sessions and select Fluxbox instead of Gnome. (You can choose Gnome to get back to "regular" Ubuntu.)

Keep in mind Fluxbox is not as full-featured and "user friendly" as Gnome, so you might need some help to get started. Here is a page to get you started: [https://help.ubuntu.com/community/Fluxbox](https://help.ubuntu.com/community/Fluxbox)
