### Post by coffeecat on 2008-07-15
You have 320 Mb RAM. I should think this is not enough for the live CD to run in graphical mode. I've looked on the Ubuntu site but I can't find the system requirements - they used to be easy to find.

The Xfce desktop needs less RAM than Gnome which is why you can run Xubuntu OK. Also, the live CD needs much more RAM than Ubuntu/Gnome running from the hard-drive. I don't know whether Gnome would run in 320MB, but if it did it would probably be slow.

One other thought. You'd be very unlucky to get 4 faulty CDs, but it might be worth trying them out in a machine with more memory just to make sure they are OK.
