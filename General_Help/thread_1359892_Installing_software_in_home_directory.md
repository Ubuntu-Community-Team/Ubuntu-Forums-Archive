### Post by dcstar on 2009-12-20
> **peoro said:**
> Hello!
Is there any easy way to install some applications as user (in my home directory)?


No.

The Unix/Linux filesystem structure is quite specific as to where applications are installed, and this is done for very good reasons.

Do not try and circumvent the way things are supposed to be, all you end up doing is wasting your own time and then other people's time as the posts of "My system is broken, please help me" invariably appear.
