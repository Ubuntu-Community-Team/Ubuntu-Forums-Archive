### Post by sylvester_0 on 2010-05-07
If it's a fresh install and you're not worried about losing any of your settings do the following:

1. Open Applications -> Accessories -> Terminal
2. Run this code:
```
rm -rf ~/.gnome2
```
3. Log out and back in
