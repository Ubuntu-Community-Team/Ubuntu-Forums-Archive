### Post by coffeecat on 2011-05-08
I presume by old theme you mean classic gnome. I've not tried this because I prefer the new Unity desktop, so I don't know whether it will work, but give this a try. Boot into the live CD/USB Unity desktop. Now click on the close icon top right and choose "Log out". At the login screen choose "ubuntu" as the username. Once you've entered the username and before you enter a password you can choose "Ubuntu Classic" (which is traditional gnome) at the bottom of the screen. The password is blank - that is, simply press enter. Hopefully, you will now be taken to the classic gnome desktop.

I believe the global menu (application menus in the top panel) is enabled by default in classic gnome in Natty. If you don't like that, simply right-click on it and choose "remove from panel". The global menu in classic gnome is simply a panel applet.

If you are using a live USB with persistence, your preference for classic gnome *may* be remembered when you next boot up. I can't be sure - I haven't tried it - but this is the behaviour in permanent installations.

Sorry I can't help you with the problem with Acer laptops. I have an older, 18 month old Acer which behaves impeccably with Natty/Unity.
