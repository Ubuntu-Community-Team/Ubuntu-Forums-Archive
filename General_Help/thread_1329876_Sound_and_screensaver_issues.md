### Post by bededog on 2009-11-17
For the screen going blank problem: you should be able to disable using the Power Management Preferences window. Go to System > Preferences > Power Management. There should be a option that says "Put display to sleep when inactive for." The default value is 10 minutes. You can change it to something longer or disable it altogether.
