### Post by Frogs Hair on 2011-10-22
Install the Gnome Tweak Tool / Advanced Settings  from the software center.

Download , extract the archive , and check all folders to make sure everything is extracted.

There are two places to install themes or icons , one is home / hidden folders and the other is File System usr / share / themes or icons .

If you are a single user open the home folder use Crtl + H to access hidden folders . Create a folder called .themes or .icons and and move the theme/icon folders into them .

If you want to share with all users , use gksudo nautilus and move folders to File System /usr/share/themes or icons .

Open the Advanced Settings > Theme and make your selections there. Themes for the shell only will not work in Unity .
