I have a UK keyboard (Microsoft Comfort Curve 2000) where there is one additional character next to the left shift key. This key has 
> |
\

Written on it. Under Windows I can just use the USA layout and this key works as it should, but under Ubuntu 10.04 this key acts as "<", which I don't know what's the point of.

I would like to use the keyboard just like an absolutely standard US keyboard, except that this key should work as it's written on it. I've tried a few different layouts, but none worked so far. The UK one recognises this key but I would pull out my hair if I had to use the UK layout. 

Is there a standard layout which works just like the standard USA one works under Windows? If not, what's the best way to create a custom one?

### Post by cogier on 2011-02-20
If it is a UK keyboard it will also have a "£" on the "3" key and the "@" and the """ are reversed (I think). Other than that they are very similar. If you want to type what is on the keys use the UK keyboard layout. 

If you go to **System/Preferences/Keyboard** you can add the UK keyboard and if you find it not to your liking change it back to the US version.
