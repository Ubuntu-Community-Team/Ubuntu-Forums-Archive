### Post by walt.smith1960 on 2011-05-19
What Processor do you have? If you have a dual core processor, you can run Ubuntu fine. My previous machine was an AthlonXP 2600 single core and it worked fine for web browsing and office-type work.  Where it failed was certain sites' flash content.  For whatever reason one site would play flash fine, another site would not. It would also depend on whether you have onboard or PCIe graphics--onboard graphics will take a chunk of RAM for itself.  If you have 1 GB. and video takes 512 for itself you'd likely have a problem.  If you have a separate video card and all the RAM is available to to the OS, you should be fine.  If your processor is adequate, you could look into adding RAM but don't throw too much money into a marginal system. Some older RAM packages can be pretty expensive and the price difference between them and newer RAM can be half the cost of a new motherboard excluding processor.
