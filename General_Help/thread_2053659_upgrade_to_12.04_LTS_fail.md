### Post by Hwalker1 on 2012-09-05
> **microcore said:**
> Did you install software from non-official ubuntu repo? (like launchpad etc)

I think, that the best way is "clean" installation.

I just installed over my previous Ubuntu system with few problems. The install process often came up with a page that was obviously a form needing an input. Unfortunately there was no font loaded, just a set of rectangles where the letters should have been.
Click the set on the bottom right each time and it will work OK.
If you cannot connect to the net, the help file gives some assistance, but I could not get it to work by following it. There is no help on the interface, so you have to make a judgment on what is the notification area or the menu bar. To get the network icon to show, you have to use the terminal and type in nm-applet. If you then find the network icon and go to the security section and enter your WEP key, the connection then works. 
Like all new systems, they take some getting used to, but the only negative comment so far is the lack of a suitable setup for the view. If you leave it to the default setting, you get gray text on a black background, and I cannot find a way to change it. On my machine, a Satellite Pro, it is unreadable. 
Otherwise, its thumbs up for a good update.
