### Post by sleepingdragon on 2009-01-16
Shoutcast is probably the best site for stream links. You should be able to drag radio links from there into Rhythmbox. Personally, I find Amarok better for radio links, but Rhythmbox has no problem adding new stations.

Try this: where it says "Radio" on the left, right click this icon and you'll be given the option to add a new station. From there, either copy&paste the link from each yellow *Tune In!* from Shoutcast.com, or you can drag the *Tune In!* icon into the "New Internet Radio Station" box from Rhythmbox.
