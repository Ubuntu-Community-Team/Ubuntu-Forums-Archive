My entertainment PC is an Ubuntu 8.04 based machine with a dual core AMD Athlon 64 3800, 1 gig of RAM, and a 250 gig hard drive.  I've got it hooked up to my TV via an S-video cable and a sound to RCA cable.  It works great with Hulu.  I'm enjoying a bunch of free TV.  

However, I'm not having the same luck with some other free TV sources.  Joost is the main competitor to Hulu.  It works on my Windows XP machine (which is not hooked up to my TV), but not on my Ubuntu PC.  It's the same thing with the Discovery Channel's web page and those of PBS and A&E.  There's free TV I could be enjoying if my entertainment PC could play video from those web pages.  For a while I've planned to add another hard drive to the computer and to install a copy of Windows XP Pro that I'm not using.  

However, I won't have to if later versions of Ubuntu play video okay from these web sites.  If that's the case, the thing to do is to just upgrade my OS.  

Could someone with a later version of Ubuntu check out these sites and see if they're able to play the videos?  It would be useful to know both if the videos play and if they can be played maximized to fill the screen.  If they do play, that will be reason enough for me to upgrade.

PBS worked as well...AE did not, but there is a new version of native 64 bit Flash for Linux out now:
