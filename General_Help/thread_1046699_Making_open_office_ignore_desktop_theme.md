### Post by VeeDubb on 2009-01-21
I'm not sure if this is more of a desktop customization question, or more of an OpenOffice question, but I thought I'd start here first.

I've found myself a really nice dark but neutral theme,that is great for 99% of what I do on my computer, especially image editing since there is a lot of 50% gray.


The one problem I have with it is that when I try to write up a document in open office, the color theme is applied to the working space, so the paper is 50% gray, and the text is 5-10% gray.

When it prints, it will actually print correctly, with white paper and black text, but it's really annoying for trying to design letterhead, or put together an invoice, etc.

The only solution I've found is to specify a white background and black text, rather than leaving them set to 'auto' but that still leaves the margins 50% gray, which is distracting.

So, is there a way to get Open Office to ignore desktop theme settings completely, or better yet, to keep them for the window border and menus, but ignore them for the working space?

---

### Post by VeeDubb on 2009-01-22
I've finally found the VERY simple solution to this issue.

Within openoffice, go to Tools>Options...>OpenOffice.org>Appearance



Withing that window, change "Page Background" from Automatic to White, and change "Text Color" from Automatic to Black.
