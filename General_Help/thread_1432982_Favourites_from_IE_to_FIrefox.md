### Post by lovinglinux on 2010-03-18
> **kokkus said:**
> Hey. How do I import my favourites from my IE in windows to firefox here in ubuntu? I do have access to the windows files here but can I just copy all the url files and paste them into firefox's bookmark folder or how can I solve this problem?

Firefox doesn't store bookmarks this way, but on a database file. You will need to import the html files using the Bookmark manager. Open &#8220;Bookmarks >> Organize Bookmarks&#8221; then select one of the options of the &#8220;Import and Backup&#8221; drop-down menu.

What do you mean by "all the url files"? Do you have a single file for each link or do you have a big html file with all the links.
