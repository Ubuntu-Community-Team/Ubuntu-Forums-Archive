### Post by efflandt on 2012-02-22
S-Video or composite NTSC is 480 lines interlaced, which never ever was as sharp as an old fashioned 640x480 VGA monitor.  So non-HD TV never has been useful as a computer monitor, maybe only for watching DVD's.

On the other hand widescreen HDTV can make a great monitor.  I am using a 32" 1080p HDTV as a monitor.  It can directly use VGA, DVI, or HDMI and at 1920x1080 is almost like having (2) 1280x1024 monitors.

> **JayUK20 said:**
> I have a PC2TV box which connect from VGA >> PC2TV Box then PC2TV Box To TV via S-Video. In XP the fonts are fuzzy and the resolution is 640x480 and the dialog boxes are so big they take up all the screen, if I change resolution it makes the screen shake and wondered if I am going to have the same issues with Ubuntu?

It's not an operating system issue. It is simply because S-Video only transmits 640x480 resolution. If your TV has a VGA, DVI or HDMI port try using one of those and you will get your TVs native resolution, which may be a bit higher. On Ubuntu you can Alt-drag a window to move it, so if it is bigger than your TV you can at least see and use it all.
