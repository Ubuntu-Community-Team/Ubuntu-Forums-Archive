### Post by KdotJ on 2010-08-28
Have you created a VM for Windows XP yet? If so, before you run it for the first time, click on the VM entry in the left-hand menu and click "Settings"

Then select "Storage" from the left-hand menu in the Settings. In the main pane, see it says "IDE Controller" (I'm assuming it is the same as mine of you have just used defaults), there should be a node coming off of this which looks like a CD, that will say "Empty". Click it, and on the right, you can mount the ISO of your choice either from the drop down box (if you have loaded your ISO in the method mentioned above) or load it using the little folder looking button. Click OK, then start your VM
