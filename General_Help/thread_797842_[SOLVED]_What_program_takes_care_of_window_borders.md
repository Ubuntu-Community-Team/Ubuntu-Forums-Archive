### Post by Vadi on 2008-05-17
If you have metacity enabled, then it takes care of the window borders. Standard themes off gnome-look.org work.

If you have compiz enabled, then it uses "gtk-window-decorator", which essentually emulates metacity for the window borders - so themes off gnome-look.org work.

So, either way, standard metacity themes should work.
