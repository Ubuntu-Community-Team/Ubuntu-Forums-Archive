### Post by ed-koala on 2007-12-23
Just from reading the forums, and my experience, I'd go with Nvidia graphics and not ATI, I use Nvidia and have not had a single problem.   I see a lot of ATI questions, just seems to be a bit buggier.  Also, do not get a lexmark printer.  Lexmark just doesn't support Linux well at all.

As far as apps, I installed Photoshop under Wine and it ran fine ... not sure if any uses of it cause problems tho, I just booted it to see if it would run, and it did. 

There are web sites that keep a list of compatible hardware for Linux, Google it and read before buying hardware, and look up any hardware you want to buy first and you'll avoid a lot of headaches.  :)

### Post by fragos on 2007-12-24
Nvidia cards have TV-out.  For a TV tuner I recommend Hauppauge since it's well supported in the Linux world.  Some Windows software will run on Linux using a package called Wine.  I've found good Linux alternatives for everything you mention with the exception of creating Flash applications.  I have however converted to and from flash videos.  Both Intel and AMD CPUs are well supported.  Harware vendors will frequently not admit to Linux support but searching the forums will help you figure that few things that aren't well supported.  I've never had any trouble with boxes I hand assembled and Linux.
