### Post by mstephens on 2008-03-08
AS400s are still widely used but are have been called i-Series for a few years - they are about to be renamed again ! If the box has the IBM licensed software there are loads of things you can do with it. The OS is built around a very efficient integrated relational DBMS and iSeries boxes can be used for business web applications - provided you have the right software. They are programmed using various languages including COBOL,C and RPG which is only really found on the AS400 and its predecessors. You can stick Linux on an i-Series - see 

[http://www-03.ibm.com/systems/i/os/linux/](http://www-03.ibm.com/systems/i/os/linux/)

just as I believe you can on a z-Series alias the IBM mainframe  should anyone happen to give you one...
