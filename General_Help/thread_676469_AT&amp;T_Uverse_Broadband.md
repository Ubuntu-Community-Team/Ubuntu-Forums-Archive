### Post by Warren Watts on 2008-01-23
Uverse is Fiber Optic right?

As long as the modem provides DHCP services (meaning it assigns local IP addresses to the devices attached to it) , I don't see why it wouldn't work with Linux.

The software provided by AT&T is HIGHLY UNLIKELY to support anything but Windows and Mac, so you might have to manually configure the modem if AT&T uses PPPoE protocol, but that can usually be accomplished using the browser to configure the modem.
