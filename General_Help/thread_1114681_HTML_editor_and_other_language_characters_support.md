### Post by Menschenfresser on 2009-04-03
Not sure about the copy/paste, I would have thought that is not a problem but I have not used Screem much.

About the dictionaries for spell checking, just install the additional language support and it will install dictionaries and language packs for almost all applications. System > Administration > Language Support (from here: [SwitchingLanguages]("https://help.ubuntu.com/community/SwitchingLanguages"))
