### Post by 1fallen on 2024-11-02
Ha I just finished this elsewhere (another forum)
 Essentially, you need to do three things:

[LIST]
[*]Goto AdBlock settings, and add all non-regional lists (EasyPrivacy, NoCoin List, Anti-Adblock Killer, Peter Lowe's List etc). 
[*]Go to Settings -> Extensions, and check the box for GreaseMonkey. 
[*]Go to [https://greasyfork.org/en/scripts/459541-youtube&#21435;&#24191;&#21578;-youtube-ad-blocker](https://greasyfork.org/en/scripts/459541-youtube&#21435;&#24191;&#21578;-youtube-ad-blocker), and click Install this script, then click install anyway. This is the Youtube Adblocker. 
[/LIST]


Now your Falkon should be able to block all the ads. Happy AD-blocking ;)
