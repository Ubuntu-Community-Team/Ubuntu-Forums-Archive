### Post by quanumphaze on 2008-08-27
Try this.

In Gnome Keyboard Shortcuts set the fullscreen binding to something like Crtl+F11 (easy to remember and doesn't interfere with default F11 fullscreen) and press that next time to make it remember the setting hopefully.

This is pretty handy Gnome feature for all programs as a way to have fullscreen without loosing the menu bars or for programs without a built in fullscreen option.
