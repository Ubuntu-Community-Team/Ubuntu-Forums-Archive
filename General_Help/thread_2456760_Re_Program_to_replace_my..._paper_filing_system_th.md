### Post by dragonfly41 on 2021-01-19
Although you might have one solution if you are researching "100's of sites" then the tool for this is [Zotero]("https://www.zotero.org/").
Two items are needed.
First a standalone app which is installed in your desktop. Mine is installed in /opt/zotero.
Second you need a zotero extension (connector) in your browser. In Chromium open extensions and search for zotero.
You can attach notes to each site link. Beware attaching passwords if you intend to publish your collection to share with others.  A separate desktop password keeper KeePassX serves that purpose.

### Post by oldfred on 2021-01-19
If you want a password manager you can use keepassXC or similar managers.
Many use the same underlying encrypted file which has your login, passwork & Internet address. You can add notes also.
I copy encrypted file to my iPhone to use it there also. But program is different in iPhone, but same database.

[https://keepassxc.org/docs/KeePassXC_GettingStarted.html](https://keepassxc.org/docs/KeePassXC_GettingStarted.html)
