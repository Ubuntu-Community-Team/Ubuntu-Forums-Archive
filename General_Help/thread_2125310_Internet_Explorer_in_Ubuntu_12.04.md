### Post by Mark Phelps on 2013-03-20
Sorry, but having to use IE in Linux is a really bad place to be.  The version that POL installs is V6 -- and MS has made it clear that it's their mission to eliminate the use of IE V6.  So, you're most likely going to get that prompt every time you run IE in Linux.

As you can see in the linked page from the WineHQ website, very few versions of IE actually work well in Linux:

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=25]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=25")

You could try upgrading to IE V7 -- if you're running 32-bit.
