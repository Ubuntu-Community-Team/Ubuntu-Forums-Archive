### Post by Alwimo on 2011-08-10
LTS versions don't have the most up to date software. LTS versions are supported with updates for longer than regular versions (3 years instead of 18 months for desktop releases), but you won't get necessarily get state of the art software. You will get software that has been tested a lot.
 
Last time I checked, 10.04 still has Firefox 3.X, but I'm not sure about Virtualbox.
