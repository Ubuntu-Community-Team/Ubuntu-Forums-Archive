### Post by hw-tph on 2006-03-22
This is a security feature to make sure the sender of the (mass) email doesn't use image access logging to track what users read their emails and so on. It is easy to enable the images: Edit --> Preferences --> Mail preferences --> HTML Mail tab, check the "Always load images off the Internet" radio button.
