### Post by x C0MMAND0 x on 2009-09-08
Is there a way to make openoffice always save as .doc, .ppt and .xls, respectively?  As opposed to the openoffice formats?  I have to submit a lot of files every day, and sometimes I forget to save as a MS format and the receiver has to respond and ask me to do so.

---

### Post by Onesimus on 2009-09-08
Yes
Tools->Options->Load/Save->General

Change the 'Always save as' 
to whatever you want.

Done

---

### Post by amedac on 2009-09-08
Yes, there is a way:
In OpenOffice: ```
Tools-> Options->  Load/Save-> General-> Document type-> Always Save as
```

### Post by Hagar Delest on 2009-09-08
Beware that it may be dangerous. OOo had serious troubles with .doc file containing tables (from 3.0.1 to 3.1.0). I were you, I would keep working in native ODF format and export when needed only.
