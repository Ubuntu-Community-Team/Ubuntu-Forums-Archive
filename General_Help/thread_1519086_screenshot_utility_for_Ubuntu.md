### Post by zacharyrs on 2010-06-27
In my past Mac life, I used Jing as a great tool to take screens and allow me to write text on the screenshot just taken, as well as put arrows on it. A great resource when managing developers for a company remotely. 

I know of Shutter which is a great screenshot capture tool, but doesn't have the editing effects such as writing text or using arrows to point things out in the screenshot taken. 

The best part was I could then upload my edited screenshot to a hosted server and be able to copy a link to the screenshot in minutes.

Anyone know of any good alternatives that would meet my needs?

### Post by iMisspell on 2010-06-27
Ive done little image editing with Ubuntu, but heres what ive done.

- Applications -> Accessories -> Take Screenshot
- Selected the options i wanted then 'Take Screenshot'
- (next screen) Clicked 'Copy To Clipboard'
- Opened up image editor and "pasted"

GIMP is an image editing program with alot of options.
Dont think its installed with Lucid, so you will have to install it yourself.
