### Post by whitetimer on 2010-01-23
Hi All

After a frustrating few days trying to connect to a Windows 7 Desktop from Ubuntu, i found the problem.  I was able to connect from Windows to Ubuntu, but not the other way around.

It turns out that Windows Live Sign-In Assistant blocks the connection.  So having uninstalled that application and Office Live Sign-In, it all works :)

Just thought i would let others know in case of problems 
:popcorn:
