### Post by mkmatis on 2010-10-11

Has anyone encountered similair problem? What might be the cause?

### Post by omniuni on 2010-10-21
Hi,

Please try this:

1. Open a terminal. If you are not already in your home folder:

```
cd ~
```

2. Run the following to remove KWin configuration:

```
rm .kde/share/config/kwin*
```

3. Exit the terminal. Open KRunner with Alt+F2, or whatever shortcut you have it set to. Execute:

```
kwin --replace
```


Does that get them back? They should work fine with the AMD Proprietary driver. (They crash with the open driver right now)
