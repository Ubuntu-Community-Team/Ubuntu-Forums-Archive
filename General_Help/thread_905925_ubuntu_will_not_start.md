### Post by Mister.Vash on 2008-08-30
Check the MD5 Sum of the file that you downloaded and burned.  The following page has links on where to get the MD5 file and how to test.
[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

If that doesn't match, download the image again.  If that does match, try burning the CD again, maybe try burning it at a slower speed.  If the CD is working you should at least be able to run the CD Integrity Check.
[https://help.ubuntu.com/community/Installation/CDIntegrityCheck](https://help.ubuntu.com/community/Installation/CDIntegrityCheck)

It that doesn't work, post your hardware specifications, you may need to try the alternative cd or provide some boot parameters on startup.
