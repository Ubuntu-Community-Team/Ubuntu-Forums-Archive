Looking for a ftp (gftp kebear) or koffice application (kword, kspread) ?

 apt-get install not finding it?

Then uncomment the line ending 'universe' in /etc/apt/sources.list, do 'apt-get update' then 'apt-get install yourapplicationwish'.
