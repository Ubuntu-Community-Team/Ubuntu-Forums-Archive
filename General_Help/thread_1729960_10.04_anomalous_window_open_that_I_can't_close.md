### Post by Frogs Hair on 2011-04-15
If it is an application running right click the top panel > select add to panel end activate the force quit applet .

If it is a window anomaly , try restarting the window manager . Depending on what you are using it may be as follows.
```
metacity --replace
``````
compiz --replace
```
