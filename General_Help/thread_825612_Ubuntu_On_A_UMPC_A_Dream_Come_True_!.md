### Post by ecosprog on 2008-06-15
I'm writing this on an eeePC out on my patio with a wifi connection and it works without a problem. This is a stock 701 with 512MB of RAM and it handles everything I've thrown at it without a problem.

There are some install issues with 8.04 but these are easily overcmoe by following the instructions here [http://eee.ricey.co.uk./](http://eee.ricey.co.uk./) and using the install script to add the missing items for wifi etc.

If you need something now then I can recommend the Asus eeePC. There are other UMPC notebooks in the pipeline but it''s anyones guess when they will arrive.

As for boot time I find mine boots in about the same tine as my desktop running OpenSUSE 10.3. It's not as fast as the stock OS that ships with the eeePC but is isn't any longer than a standard PC.
