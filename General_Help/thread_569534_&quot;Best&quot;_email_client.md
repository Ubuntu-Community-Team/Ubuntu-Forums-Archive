### Post by rules on 2007-10-08
Thanks for all the help, I couldn't have gotten it to work any better :)

I installed TB on XP and imported all my mails from Outlook. Then I installed TB on linux and created my account there. I then went into account settings and specified a different location for Local Folders to be stored ... I pointed it at the XP profile. So now they both use the same profile, so I can download mail in either OS and it will show in either/both.

The only thing it does not share is the address book, will fiddle with that one later.
