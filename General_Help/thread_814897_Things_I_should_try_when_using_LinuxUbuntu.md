-> Synaptic Package Manager (great application to download software instead of google them out)

**Synaptic will also help you update the [COLOR="Blue"]entire system[/COLOR] at one time instead of updating only the OS then each application individually.**

Typically, can you guys recommend any great software to use for the purpose as below?

-> Graphic Editing/Design
Windows: Adobe Photoshop CS3
Linux: No idea

**GIMP**

-> Picture Organizer
Windows: Google Picasa 2
Linux: No idea

**F-Spot**

-> Multimedia Player
Windows: foobar2k (great music player on planet!)
Linux: Amarok? (haven't try it out, comment needed)

**Audacious, Rhythmbox**

-> P2P application
Windows: Bitcomet/Thunderbird
Linux: Azureus? 

[B]I may get in trouble for this, but Limewire is a P2P application. 

[COLOR="Red"]All peer to peer applications are susceptible to abuse. Please don't do anything illegal when utilizing P2P applications whether on Linux, Windows or Mac.[/COLOR] [/B]



If you have a web cam and want to talk to folks online, try Skype.

For general Instant Messaging, use Pidgen.

For mice, use Logitech.

To kill time while at work/school/home try Mahjongg.

For productivity, use Open Office. This should already be installed.

HTML editing, use Bluefish or Nvu.

Burning CD's or DVD's, use K3b.

Screenshots, use Ksnapshot.

To listen to streaming music, use Streamtuner. Last time I looked there were over 100,000 individual streams available to the end user. 

Watching videos, use VLC.

Sharing files on your network, use Samba.

For sweet desktop effects, use Compiz-Fusion.
