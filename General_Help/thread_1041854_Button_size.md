### Post by blackened on 2009-01-17
1. If the theme supports it, then change the titlebar font size in System -> Preferences -> Appearance -> Fonts Tab

or

2. Use a different theme. Themes can be had by visiting, among others, [http://gnome-look.org]("http://gnome-look.org").

### Post by blackened on 2009-01-17
Many themes don't base their button sizes on the titlebar font, which was the reason for the "or" in my previous post. Try another theme, or if your current theme is pixmap based, then you could manually edit the graphic files and change the settings in gtkrc.
