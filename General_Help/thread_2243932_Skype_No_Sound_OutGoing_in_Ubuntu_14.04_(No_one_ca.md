### Post by swarup on 2014-09-12
I have Skype version 4.3.0.37 installed in my Ubuntu 14.04 (unity). For months it was working normally; then a couple weeks ago -- I think perhaps after a software updater update -- the outbound sound stopped. Means, no one can hear me but I hear them perfectly. As far as I can tell nothing is set to mute-- but perhaps there is some inner setting that needs to be unmuted? I tried googling this without success. Solution will be greatly appreciated :-)

### Post by swarup on 2014-09-12
I've got it solved, in a way. I went into the drop-down window for sound i.e. sound settings. There for "input", the setting was set to "Digital Input S/PDIF". On that setting, no one can hear me when I speak. I changed it to the other option "internal microphone"-- and now it works perfectly. But the only problem is that when I close that sound settings window, the setting goes back to "Digital Input S/PDIF"! That means every time I want to use skype I will have to open the sound settings window first and reset the mic to "internal microphone". How can I get the setting to stay where I put it, so I won't have to reset it every time?
