### Post by hikaricore on 2008-12-13
Could be that your keyboard is damaged, I once owned one that would send the wrong key signals whenever I used the right half of home row.
If possible do you have another keyboard you can try out for a bit to be sure this isn't a hardware issue?

Aside from that perhaps go into the shortcut preferences and bind the screenshot function to a more elaborate key combination to see if it still occurs.

Another slightly less likely scenario is that someone setup a cronjob to run the screenshot application randomly. ^_^

### Post by hikaricore on 2008-12-13
Well your keyboard may work well but that doesn't mean it isn't damaged.
If you don't have another keyboard and can't test it well then that suggestion is out, but be aware if it's a hardware issue it may never go away.

I don't really know what you mean diable via keyboard.

Go into your keybindings (keyboard shortcuts or whatever it's called) and change the key combination for print screen to something like Alt+PrintScreen and see it it still happens.  This way you'll still be able to take screenshots but it may not pop up "on it's own".
