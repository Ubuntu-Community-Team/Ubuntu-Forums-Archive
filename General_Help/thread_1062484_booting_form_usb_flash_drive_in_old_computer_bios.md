### Post by andrewc6l on 2009-02-07
If you have a floppy, then you might be able to use Smart BootManager - I used that when I had an old BIOS that didn't know about USB CD-ROM drives and needed to boot. You have to build a floppy that says where you want to boot, and then boot from it, which will then have the smarts your BIOS doesn't have to boot from USB.

[http://sourceforge.net/projects/btmgr/](http://sourceforge.net/projects/btmgr/)
