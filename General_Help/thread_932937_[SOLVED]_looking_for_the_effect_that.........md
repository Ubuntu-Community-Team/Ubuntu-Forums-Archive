### Post by BGrigg on 2008-09-28
That's the "Dodge" animation in Compiz. 

Open: System > Preferences > Advanced Desktop Effects Settings > Effects Category > Animation > Focus Animation > Select "Fade" and edit the setting to Dodge. Then hit Back and Close and have fun watching your windows move about.

### Post by jualin on 2008-09-28
First you need to install compizconfig-settings-manager package using synaptic or the terminal. Then you can go to System, Preferences, Advanced Desktop Effects, General Options, Focus & Raise Behaviour, unselect "click to focus" and select "Auto-raise". Hope this helps!
