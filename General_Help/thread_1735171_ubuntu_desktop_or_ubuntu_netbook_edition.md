### Post by poodoopealeoaph on 2011-04-21
if you have a laptop that is anything bigger than 15 inches then you should use Ubuntu 10.10 regular desktop edition. If you have anything smaller then you should use the netbook edition....

don't get too comfortable though, they are getting rid of the netbook versions for Ubuntu. Since they are using Unity (the netbook interface) for the main desktop in 11.04 and on, there will not be a netbook edition....

I think this is a great step though because unity is awesome and as soon as the video card drivers work for my laptop I am upgrading and with great joy!! Everything about the new Ubuntu is great and I can't wait to try it out... that is after the new bug is worked out with the new linux kernel not wanting to work with most computers' video cards... 

if you get the netbook edition to work though, it would be a good thing. You would get a good feel for how the Unity desktop works so when you upgrade to 11.04 you will not notice a difference really. You will just notice that the buttons are smaller for the dock that is on the left and that it auto hides it's self if over lapped.

### Post by poodoopealeoaph on 2011-04-21
Well I wouldn't completely bank on the new 11.04 version because I have had bad video experiences with every distro so far.. I would advise to try out the netbook version and see if you like it. If so, then, if you have the time, try out the 11.04 RC that should be dropping this week.. if it hasn't dropped already and try to see if it gives you video. I'm trying very hard to not be biased even though I haven't had a good experience with any of the betas..

If things don't start up for you, or it gives you a "blank" screen, you should shine a flashlight on your screen and see if you can faintly see things. If so, then you have the same thing that I run into every time I upgrade. At this point you will want to restart and hit F6 when you get to the startup (I'm not sure if it will give you the option in regular ubuntu but it will in Ku and Xu). Then you should select "nomodeset". This will give you backlighting so you can install... 

sadly.. I haven't ventured further than that because I've chickened out (I tried these methods in openSUSE). If you search google, you should be able to find out how to set up gimp for an automatic setting for "nomodeset" so you have backlights when you start up each time... as far as I know, you only have to enter this option into gimp one time and then you are set.. As far as I know, you need to hold shift after you get past your bios screen to get gimp to let you enter such settings...

sorry I couldn't be of more help
