### Post by jaime256 on 2009-06-09
Hi. I'v started using the copy feature on the xsane but I have found that it cuts a bit of the top of the image even with the document aligned correctly. Here's what I found.

1. When I changed to copy, I always got that green line before the edge of the document area. First picture on the bottom.

2. I found out that this was due to the millimeter setting and not the inches setting for the US. So change this to get the correct paper size. This is under view - length unit - millimeters/centimeters/inches.

3. Under Preferences - Setup - copy, the page size is wrong for the US settings, so change that to 8.5 by 11 inches. I also changed the left and bottom offset to zero. I don't know what that's supposed to be, but that helped a bit.

Unfortunately, the image still gets cut off a bit on the top when I make a copy. Any ideas on what may be causing this and how to fix it? And feel free to correct me if I'm wrong. 

Oh yes, even with these new settings the copies still have a border, meaning the print is still not completely on the edge of the paper. I don't mind that for now, but I'm assuming taking out the offset should clear that border. Unless that's something else that I'm not aware off. The version of xsane is 0.996 and I'm on Ubuntu 9.04. Thanks.
