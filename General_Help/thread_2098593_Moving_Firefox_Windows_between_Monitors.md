### Post by Jaylor on 2012-12-27
So, I've kinda 'solved' this issue thanks to another post about a different problem.   (I really think it is a bug in Firefox and who knows when they will fix it).

If you're using multiple monitors and are NOT able to move Firefox Windows between them using Click-n-Drag just hold down the "Alt" key and then click-n-drag the window to the other monitor.

Note: This was even more strange that it let me drag it from an external monitor to my Laptop monitor.  *shrugs*

Alt works until FF fixes it.
