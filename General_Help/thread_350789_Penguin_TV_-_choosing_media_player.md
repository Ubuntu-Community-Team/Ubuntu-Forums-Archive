Any clues?

EDIT - The problem is not with Penguin TV (although its own 'choose player' feature would be useful). Instead, I just set vlc as the default app for each file type I want it to play by using the 'Open with...' dialogue in the file browser right-click menu (I use Thunar in Xfce but the same feature exists in all the major file browsers/desktop environments). Penguin TV recognised the change and now opens vlc like I want it to.
