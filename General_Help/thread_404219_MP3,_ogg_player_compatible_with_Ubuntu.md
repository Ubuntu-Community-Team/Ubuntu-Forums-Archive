### Post by laidback on 2007-04-08
You can burn mp3 with Linux, so if you can drop the other 2 I guess any player would do. You'll need the appropriate codecs installed to deal with mp3, ogg is there by default as it's open source.
I created an mp3 cd for my daughter as her new cd player accepts these (unlike the older ones), get loads on a single cd. Info on codecs is also available via the support area, I'll post a ref when I find it again.


Restricted formats here
[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)
