Insufficient data - *which* file was the error in?  If it's in a critical file (like the kernel image), it will likely result in your not being able to install from the CD.  If it's in an optional package that you won't be installing anyway, then you won't even notice the problem during the install.

As for the best way to burn - I always go to the lowest speed my drive will allow for the burn to reduce the chance of an error.
