### Post by grahammechanical on 2023-07-30
The number pad keys have two functions. They can be used as number keys or arrow keys. The delete key can be used to delete or print a dot. The Num Lk key toggles those functions. Press the Num Lk key. to switch the purpose of the number pad keys.
