### Post by Henrikzhura on 2009-07-02
First:

Make the downloaded binary installation file executable:

[COLOR="Green"]cd /directory_where_downloaded
chmod +x PlaneShift-v.0.42-x64.bin[/COLOR]

Second:

Then run the executable binary as root (wouldn't recommend messing with the sudo command if you are a beginner): 

[COLOR="Green"]sudo ./PlaneShift-v.0.42-x64.bin[/COLOR]

Third:

Follow the instructions to install, when asked to manually set permissions, select "no". Install the game in your /home directory.

Fourth:

Download any patches from the Planeshift site.

Fifth:

Register for an account and play!
