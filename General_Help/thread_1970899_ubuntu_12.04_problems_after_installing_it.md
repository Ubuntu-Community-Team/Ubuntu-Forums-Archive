Can you create another user account or already have one? Log in and see if you get the global menu (is that what you mean by lateral bar) and keyboard working. It might be best to try a new user account because of former settings from earlier Ubuntu version in user folder. It is these that may be stopping things working correctly. If you find everything working in new user account then log in to the previous or default account, delete all settings folders and files (not Documents, Downloads or Videos of course). You log out and in again all your settings will be at default but the keyboard and global menu should be back. It might be best to do a complete reboot.

Be careful when deleting things, like email account settings. You should not need to delete them because they are not a directly part of the unity desktop settings.
You can get a temporary virtual screen keyboard from System Settings, Universal Access, to make the user name and password for the new user account.
