### Post by coldraven on 2011-07-11
If you like Firefox then use the Flash-Aid add-on to install the latest version of flash. That might help especially if you are using the 64 bit version of Ubuntu.
[https://addons.mozilla.org/en-US/firefox/addon/flash-aid/](https://addons.mozilla.org/en-US/firefox/addon/flash-aid/)

Or try installing Chromium from the Software Centre. Or Chrome which is the Google version of Chromium. From here: [http://www.google.com/chrome/intl/en-GB/landing_tv.html](http://www.google.com/chrome/intl/en-GB/landing_tv.html)
