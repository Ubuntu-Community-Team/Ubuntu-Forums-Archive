### Post by SeijiSensei on 2012-04-08
If you need to ask this question, you're probably better off using the version in the Ubuntu repositories by running "sudo apt-get install thunderbird".  If you want a bleeding-edge version, you can install [this PPA](https://launchpad.net/~ubuntu-mozilla-daily/+archive/ppa), or [this one](https://launchpad.net/~mozillateam/+archive/thunderbird-next) which is a bit more stable.
