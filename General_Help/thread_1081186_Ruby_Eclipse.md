### Post by FluffyElmo on 2009-02-26
Netbeans is great and highly recommended. Last time I did the comparison it was ahead of Eclipse in Ruby support. The one thing to watch is that it uses JRuby by default but you can use your systems Ruby if you prefer. [http://www.netbeans.org/downloads/index.html](http://www.netbeans.org/downloads/index.html)

For Eclipse, you could try Easy Eclipse which has pre-configured Eclipse distributions for a variety of languages. [http://www.easyeclipse.org/site/distributions/index.html](http://www.easyeclipse.org/site/distributions/index.html)
