### Post by GrayBear on 2009-09-30
I am using Gnome Commander as my default file manager. When I push F3 to view a file, it opens it in a small window - 304 x 261 pixels. I maximize the viewer window, but when I come back to see the same file, or when I view another file, it will keep opening with a small window.

How can I convince it to start maximized ?

### Post by GrayBear on 2009-10-01
I found it. Maximize the viewer window, then:
   Go to Settings -> Save Current Settings
   or simply push **Ctrl+S**

Next time you open the Gnome Commander viewer, it will open it maximized
