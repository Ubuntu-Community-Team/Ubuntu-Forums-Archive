### Post by hartz on 2007-05-25
Agreed, the stuff in your home directory pertains only to "your" use of the applications.  Such as preferences, files currently being downloaded (by you), etc.

If you delete the .programname directory or file, and then later run the program/application again, it will start up fresh with all default settings again.
