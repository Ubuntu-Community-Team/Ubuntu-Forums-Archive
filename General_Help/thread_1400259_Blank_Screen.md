### Post by Nicolas.Perrault on 2010-02-06
You must reboot your computer before installing Ubuntu. It will run on your CD. Before doing that, you might need to change the boot order by opening the bios (the bios can be opened by pressing one of the following keys (depends on manufacturer): escape, F1 to F12 and delete).

     After you've done that, exit the bios, and a black screen with a big [SIZE="3"]Ubuntu[/SIZE] should appear. Choose install Ubuntu. 

     It should guide you properly through the installation. However if you have already done what I have told you and it still doesn't work, then I'm afraid I can't help you, I'm a noob too ;)
