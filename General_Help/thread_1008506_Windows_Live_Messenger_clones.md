### Post by nzadLithium on 2008-12-11
Get the svn version of aMSN :p It's supposed to have code in place for voice and video chat :D (It's probably buggy but I wana try it sometime :) )
And as for liking your chats in seperate windows, in pidgin right click on the persons name, and go 'Detach this tab'. To make it so that it never tabs any chats

from [http://nbcs.rutgers.edu/newdocs/jabber/pidgin001.php:](http://nbcs.rutgers.edu/newdocs/jabber/pidgin001.php:)
> Tabbed Chatting

By default, tabbed chatting is enabled in pidgin. If you'd like to disable it, the option to Show IMs and chats in tabbed windows can be found by going to the Tools menu, selecting Preferences, and going to the Interface tab 

In that tab is the option 'show im's and chats in tabbed windows' un-check that one :D
