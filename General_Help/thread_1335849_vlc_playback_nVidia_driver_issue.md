### Post by jebsector on 2009-11-23
Just a forward, i recently upgraded to ubuntu 9.10, to find my favorite! bleach episodes playing back with screwed up colors in vlc.  This can happen in different media players as well and may not be specific to ubuntu 9.10.

The problem is with the nVidia driver.  To resolve, i simply reverted back to the earlier nvidia driver version (System->Administration->Hardware Drivers->[mine was version 173])  This is easy with the later versions of ubuntu since there is now support for automatically handling your nVidia drivers, if you use the new graphics cards like i do. as always, back up your files first before messing with the drivers - 

Other forums list this as an issue in ubuntu, and apparently nVidia has been notified of the issue.

Just posting in case any windows-switchers out there have this problem and need a quick fix.  I do not know if an update is out there for this issue,
