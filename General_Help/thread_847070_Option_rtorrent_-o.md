### Post by K.Mandla on 2008-07-02
> -o key1=opt1,..., -O key=opt

    Set any number of options, see the SETTINGS section. The options given here override the resource files. Use capital -O to allow comma in the option. 
[http://libtorrent.rakshasa.no/rtorrent/rtorrent.1.html](http://libtorrent.rakshasa.no/rtorrent/rtorrent.1.html)

After that, I guess you could contact the developers via the rtorrent wiki, and ask for more details.
