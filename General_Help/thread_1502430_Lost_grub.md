### Post by ajgreeny on 2010-06-05
How to restore the Ubuntu grub bootloader (9.10 and beyond)
You must use a 10.04 live CD to do this for a 10.04 installation, as grub versions have changed. 9.10 might work, but grub has gone from 1.97 beta to 1.98, so I'm not certain.

Ubuntu 9.10 and 10.04 use Grub 2, so you need to find out what your drives are using live CD:
    ```
sudo fdisk -l
```
From that you need to find the device name of your Ubuntu drive, something like “/dev/sda5&#8243;.
Still in the terminal, type:
    ```
sudo mkdir /media/sda5
    sudo mount /dev/sda5 /media/sda5
```
To reinstall grub:
    ```
sudo grub-install --root-directory=/media/sda5 /dev/sda
```
Push enter and you’re done!

Of course you need to replace “/dev/sda5&#8243; and “/dev/sda” with what you found in the fdisk output.
