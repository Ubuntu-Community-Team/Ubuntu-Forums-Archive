### Post by chewearn on 2008-06-06
I see you have nvidia.  I find the easiest way to configure nvidia dual screens is using nvidia-settings.

```
gksudo nvidia-settings
```

You need to choose either twinview, or separate X screens (both have advantages and disadvantages).

Once you are satisfied with you configuration, remember to save it by clicking
"Save to X Configuration File".  You might need to restart X for the configuration to take effect.
