### Post by open2linux on 2007-05-02
I am a windows user ( sorry for that:(  ) waiting with great expectancy for the final Wubi version, so I can start using Ubuntu. I am new to Linux and I have a laptop with a broken CDRom, so for the time being Wubi is my only chance. (Of course the other way would be to get a [**[COLOR="Blue"]Dell[/COLOR]**]("http://direct2dell.com/one2one/archive/2007/05/01/13147.aspx") ;) ... but I can´t afford that at this time).

I saw in one of the sticky topics, a discussion with an official Ubuntu developer, about possible ways to include Wubi in the next release of Ubuntu. Of course I don´t understand much of the technical details of that discussion, so I am not posting there what could be out of place. But I want to contribute something from the perspective of a Windows user, new with the technical aspects of GNU/Linux but familiar with all the philosophy behind it. So for that I am creating this new topic.

I wanted to help testing, so some time ago I tried a previos version of Wubi. With help finally I saw Ubuntu on my computer. I like it, but have dificulties printing, going online and rebooting, so I decided to wait for the stable version.

For me this would be the safest, most comfortable and assuring path to Ubuntu:

1. Install it from and within Windows, I think loopmounted you call it. I want to be able to use Ubuntu, while being able at the same time to keep using Windows, with no change in Windows, no partitions (not yet...).
And be able to easily uninstall if necessary. Even if I am not planning to do that, it gives extra peace of mind to know that I could do it.

2. Use Ubuntu. Get familiar with it (I have started to study aready), see if I am able to print, learn to setup my USB modem to go online from Ubuntu, install and unisnstall things...

3. After step 2 is acomplished I will be ready to install to a real partition. For the the ideal way would be to be able to move all setting to a partition and keep dualbooting, but now with a real Ubuntu installation to a real parttion. I saw that plans for that in a future version. I will wait for that.

4. Well after I can do in Ubuntu everything I do now in Windows, it will be nice to be able, from Ubuntu, to totally remove Windows and that Ubuntu get all the space of my hardisk. I think that there is also plans for that .

My goal is to reach step 4. But I need to start with step 1.

Please take into consideration that even after installing Ubuntu, it is not possible to go online right away from wthin Ubuntu. So a netinstall is not possible having access to Internet only from Windows.

Please keep in mind that many of the potenial future users of Wubi think like me.

Thanks.

PS. By the way, I am not planning  just using Ubuntu. I want to help in any way I can with all things a non programer could do: propagating, translating (into Spanish). I have started learning to use Rosetta, so I can help in translating Wubi when needed. And I have joined the [**[COLOR="Red"]Spanish[/COLOR]**]("http://www.ubuntu-es.org/") forum, so I will tell people about how I joined Ubuntu with Wubi when that happens. But I am still using Windows. And getting sick of it.
