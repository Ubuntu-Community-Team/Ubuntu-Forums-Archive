### Post by gandaran on 2011-06-07
> **lakshmen said:**
> I downloaded the C Primer Plus, Fifth Edition.chm file. It is in bin format.. How to open it?
bin files are for installing not extracting, right click » properties » permissions » mark allow executing file then double click the bin file and choose run to install, if it wont work this way then open terminal, cd to folder where the bin file is and type the run command
```
sudo ./your_file_to_install.bin
```
