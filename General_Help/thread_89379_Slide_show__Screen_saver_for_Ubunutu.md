An xscreensaver that exactly matches your request is GLSlideshow. It does a nice pan and scan for each photo it shows. Remember to set a default folder from which to grab photos under: 

System > Preferences > Screensaver > Advanced (tab)

Under the image manipulation box select: Choose Random Image and give
type in the root folder below which your photo collection is in.

Once you've set that, xscreensaver will grab photos from here instead of screenshots of you desktop for many of the fancy screensavers.
