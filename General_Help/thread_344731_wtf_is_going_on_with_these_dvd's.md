### Post by allwin on 2007-01-23
Offhand, it looks like the wrong character set is being used to interpret the file names on the DVD.  I know there is a setting in the MOUNT command for what character set to use when mounting devices (typically I've seen UTF-8).  Only, I'm too much of a noob to tell you exactly what character set you require there!  Well, it looks like the FILES are present on the DVD, so you could just copy and rename them maybe.  As an example, maybe the DVD was written in the Chinese character set, while your desktop is in US ENGLISH...see...so you have to figure out how to mount it to get the characters in the file name to display properly, if even that can be achieved.
