### Post by coffeecat on 2014-08-05
It's not really clear what you are asking, but assuming you have removed Kubuntu and want to re-install the Windows 7 mbr, what you need is here:

[https://help.ubuntu.com/community/RestoreUbuntu/XP/Vista/7Bootloader](https://help.ubuntu.com/community/RestoreUbuntu/XP/Vista/7Bootloader)

Instructions include how to use the Windows 7 installation DVD or Windows 7 repair disc, how to get a copy of the Windows 7 repair disc if you don't have one or, if all else fails, how to do this with an Ubuntu live CD. The Ubuntu CD instructions will probably work with a Kubuntu CD.
