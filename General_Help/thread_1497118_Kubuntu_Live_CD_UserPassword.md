### Post by ajgreeny on 2010-05-30
The live CD should not need this, so something must be wrong.

Boot again but when the CD starts just hit any key to get a menu that will allow you to check the CD.  I suspect that the CD will fail, and you will need to burn it again.

If it passes, all I can suggest is thet you try the username **kubuntu** and for the password type nothing at all, just hit return.  Tht is what happens if you logout from a live ubuntu CD, though in that case the username is just **ubuntu**, not **kubuntu**.
