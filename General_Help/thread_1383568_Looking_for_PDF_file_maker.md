### Post by lyall on 2010-01-17
if you are scanning try ***gscan2pdf***
it is in the Synaptic Package Manager

If you open your file/s you can also print to pdf using ***cups-pdf*** it is in the Synaptic Package Manager
how to use, send the file to your printer select ***PDF***. it will create a pdf file instead of printing.

You can also use OpenOffice Writer.:KS
It will scan your document into Writer.  If you are scanning a lot of documents, just keep using your scanner in OpenOffice
You will have to import them in by clicking on 
Insert > Picture > Scan > select Source
you can also insert your jpegs and tiff files
I do not know if you can insert raw files into OpenOffice

to comvert jpeg to pdf you can try this from the command window
convert image.jpg -compress JPEG image.pdf
it should reduce the size for the file in pdf forumat

I hope this help along your way

good luck and have fun learning
