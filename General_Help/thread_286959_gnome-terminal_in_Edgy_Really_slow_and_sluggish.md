### Post by jeblinux on 2006-11-02
gnome-terminal has been much slower for me too since I did the upgrade a couple days ago. However, I found that turning off transparency made everything go back to normal. I don't remember turning it one but it was on. I don't use that fancy stuff :) 

In gnome-terminal just go to Edit -> Current Profile ... and then the Effects tab. Change it to none.

If you have been using transparency before and never had a problem I don't know what to tell you. Something else must be wrong.
