### Post by lykwydchykyn on 2008-12-27
Ubuntu does not run software designed for Windows.  It can attempt to, via an compatibility program called Wine, but doing so tends to be unreliable.  It does, however, run many thousands of programs designed for Linux.

You don't need Windows to install ubuntu.  Just boot to the CD and install.

### Post by lykwydchykyn on 2008-12-27
> **rivalgen said:**
> But are programs being made to be compatibly with linux?

That depends on what you mean by "programs".  If you mean "I go to the local computer store and buy a CD in a box for $50 and take it home and install it", then generally no.  If by "programs" you mean running any of the thousands of free, open-source software packages people are writing for Linux, then yes.

There is commercial software for Linux as well, if you like paying money for things.  But you aren't likely to find it in stores.

What kinds of programs do you want to run?
