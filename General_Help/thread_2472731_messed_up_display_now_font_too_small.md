### Post by ajgreeny on 2022-03-10
If you are using Ubuntu-Studio I believe it uses the Xfce4 DE, similar to Xubuntu, so first thing to try is to open Settings -> Appearance Fonts tab and make changes there to the font used and size of that font.

If that makes no difference try increasing the DPI setting in the same font tab.
