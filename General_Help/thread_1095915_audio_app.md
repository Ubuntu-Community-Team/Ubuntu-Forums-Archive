### Post by ajgreeny on 2009-03-14
You can also use mp3splt (not a typo, there is no i in the name).  It is a simple but powerful terminal utility which I use a lot after recording old vinyl LPs.  It is also worth getting mp3wrap and mp3gain which allow you to join mp3s and change volume (gain).  All 3 programs work without decoding and re-encoding the original so there is no quality loss at all.  All are in the repos.
