### Post by quixote on 2009-05-19
Yeah.  That's one of the maddening things about gnome.  The philosophy of not burdening us with options we don't need should NOT extend to options everybody needs.  Like arranging your own workspace!

It's one of my pet peeves.  And the only thing I've found is a package called devilspie, and its GUI frontend, gdevilspie.  Both are in synaptic.  I think you have to have the "universe" repository enabled.  For more info, this is the package homepage: [http://www.burtonini.com/blog/computers/devilspie](http://www.burtonini.com/blog/computers/devilspie) .  And this is an ubuntu tutorial on it: [http://ubuntu-tutorials.com/2007/07/25/how-to-set-default-workspace-size-and-window-effects-in-gnome/](http://ubuntu-tutorials.com/2007/07/25/how-to-set-default-workspace-size-and-window-effects-in-gnome/)
