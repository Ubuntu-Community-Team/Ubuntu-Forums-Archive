### Post by Hayksk on 2010-02-11
How can I change the tab (<tab>) size in Vim editor?

---

### Post by gmargo on 2010-02-11
To set the tab size to four:
```

:set tabstop=4

```Use ":help" to get the embedded help
Google finds all sorts of good tutorials: [http://www.google.com/search?q=vim+tutorial](http://www.google.com/search?q=vim+tutorial)
