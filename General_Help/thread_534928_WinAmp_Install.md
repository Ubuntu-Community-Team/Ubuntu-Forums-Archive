I installed Winamp and an error message popped up saying that I need at the very least one windows true type font installed in c/windows/font. I tried to copy a font from my windows font directory (I dual boot), but I could not get it to copy. Can anyone tell me how to do this???

### Post by kaylus on 2007-08-25
I will give you the easiest ways I know how to do this:

To easily install true type fonts:
Install Automatix from [http://www.getautomatix.com/]("http://www.getautomatix.com/")

To easily install WinAmp under wine without worrying about the configuration nonsense yourself, install Wine-doors from [www.getdeb.org](www.getdeb.org)

[Wine Doors Download Page]("http://www.getdeb.net/search.php?keywords=wine-doors")

Hope that helps you out =)
