### Post by Jolly-Swagman on 2007-10-14
Yes I too have the same problem and found that linux uses the the graphics adapters primary and secondary the opposite to windows drivers, thus what was the primary on your windows OS is now secondary on linux I have changed them several times but as soon as you get an update to kernel they change back again it is annoying at times.
You can set which monitor is for each in the preferences tab in the display management
 
Hope this helps
