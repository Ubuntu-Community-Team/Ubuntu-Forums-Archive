### Post by fbuchele on 2008-05-13
Just had this problem, seems to be a bit of an issue updating to hardy and then switching kernels. The solution is to manually install the open office style packages you want to use, e.g. ```
sudo apt-get install openoffice.org-style-andromeda(or any other style you want)
```
