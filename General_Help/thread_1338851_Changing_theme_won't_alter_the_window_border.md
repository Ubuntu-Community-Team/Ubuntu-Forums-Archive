### Post by blazemore on 2009-11-27
it seems you're using the "Emerald" window decorator.
Open a terminal and type
```
sudo apt-get install fusion-icon
```
Press Enter and wait for the package to install.

Then hit Alt + F2 and type
```
fusion-icon
```
press Enter

You'll see a new, blue icon in your notification area. You need to right-click this, and from there you have different window options. You can select your Window Decorator between Emerald and Metacity this way.
