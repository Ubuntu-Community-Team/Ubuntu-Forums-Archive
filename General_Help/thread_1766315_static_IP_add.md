### Post by Topsiho on 2011-05-24
What you describe is the correct way to go, in the GUI. Another way is to rightclick on the network icon in the upper panel, somewhere on the right.

You don't tell what Ubuntu you use. You can only do this as a super user (root), and somewhere somehow you should be given the opportunity to authenticate yourself as root, e.g. with an unlock button or a popup screen asking you for your password. (My) Ubuntu 10.04 doesn't offer this opportunity, rendering this network tool completely useless, in 10.04. In other Ubuntu's however, this is a very easy way to configure a static IP address.

I guess this answers your question, as actually editing the connection is straight forward.
