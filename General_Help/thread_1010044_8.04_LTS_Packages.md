### Post by sdennie on 2008-12-13
Generally, after an Ubuntu version is released, packages only get bug fixes and security updates.  However, there are things like launchpad PPAs and getdeb.net that often provide more up to date packages for an LTS release like Hardy.  You should always be careful when getting packages outside of the official repos but, it's probably viable to keep a Hardy machine fairly cutting edge if you want to.
