### Post by fennec_fox on 2009-08-06
It shouldn't have any affect on booting into a live cd. It only encrypted the hard drive. 

If it's not working though, you might want to try booting from USB instead of cd in case the problem was with the cd drive. That might have been what happened the first time. I would try that.
