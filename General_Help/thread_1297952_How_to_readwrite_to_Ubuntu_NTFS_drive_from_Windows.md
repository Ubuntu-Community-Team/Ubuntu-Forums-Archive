### Post by P4man on 2009-10-22
open a terminal and type

```
sudo smbpasswd -a [COLOR="DarkRed"]username[/COLOR]
```

replace username with the username that will be used to connect. Set your password. then try logging in again from windows. You may have to restart samba after setting the pw, I think not but to be sure:

```
 sudo /etc/init.d/samba restart
```
