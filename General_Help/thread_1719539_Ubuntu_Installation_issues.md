### Post by Hedgehog1 on 2011-04-02
Hey There!  This catches a lot of folks:

You need to 'eject' the install ISO from the Virtual CD:

[IMG]http://img812.imageshack.us/img812/1877/ejectvitualcd.png[/IMG]

 or

[IMG]http://img864.imageshack.us/img864/6620/vboxremovediskv.png[/IMG]

With the .iso no longer a boot CD, your install will run normally.
