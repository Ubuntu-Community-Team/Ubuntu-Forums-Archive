### Post by ajgreeny on 2010-07-04
You must have chosen both a username and then typed a password (twice) when you installed.

If you didn't do that, you can not have actually installed the system, so I don't know what on earth is happening.

If you have simply forgotten what you chose, or someone installed for you, it is possible to get the system running quite easily, once you have your keyboard working, by following this info:  [http://www.psychocats.net/ubuntu/resetpassword](http://www.psychocats.net/ubuntu/resetpassword)

For the keyboard, try enabling legacy usb in the BIOS, assuming it is a usb keyboard.
