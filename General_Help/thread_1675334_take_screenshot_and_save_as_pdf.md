### Post by vanadium on 2011-01-25
These are two things: 1) taking a screen shot 2) converting the graphic to a PDF.

1) you probably know.
2) can be done using convert:
```

convert image.jpg document.pdf

```

If you specify multiple graphics, you will get a multipage PDF.
