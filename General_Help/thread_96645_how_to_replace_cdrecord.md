### Post by Remmelas on 2005-11-29
I'm not sure what you are asking, so, a couple of answers for you.

If you are looking to use a different cd burning engine, I think there is a Nero Burning Rom package for linux.  

If you are just trying to not have to interface with cdrecord on the command line, there are some GUI front ends available that work very nicely.  I use K3b on my install(it's kde, so it will also pull in some kde libs, but that doesn't hurt your gnome install).  There is also Gnome-Toaster, Xcdroast, and i'm sure a few others I can't remember.
