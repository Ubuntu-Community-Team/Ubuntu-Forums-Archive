### Post by snowpine on 2008-10-22
One important point: Your computer must be able to boot from USB. For most newer computers, this is not a problem. However, older computers can't boot from USB--the bios is not set up that way.

### Post by C.S.Cameron on 2008-10-22
Unibootin and usb-creator will let you make Live USB flash drives from the downloaded iso file
You can install from these just like from the Live CD.
Unibootin is simple to use and works in Windows or Ubuntu.
No CD drive is required in this process.
Liveusb only works from the Live CD.
