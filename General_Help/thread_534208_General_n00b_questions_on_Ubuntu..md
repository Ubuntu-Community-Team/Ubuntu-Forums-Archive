### Post by livingtarget on 2007-08-24
2. I dont run a firewall I know that's pretty ignorant but it works for me. Virus scanners are really not required. Some people insist on one, I don't find it worth the trouble.

3. C sounds easiest, but A may work pretty well. They're all valid methods in my opinion.

4. It's starting to get really easy to install drivers. Most drivers are built-in the kernel so really your hardware will either be supported or not. Sometimes there are drivers that you have to download, but that's pretty rare for me.

6. Oh yes, this is fairly easy. Just select a language and it will automatically download it. Better than windows last time I checked.

Another tip would be to burn a ubuntu cd and just load it in the cdrom drive and use the live-cd to browse around. This means ubuntu loads entirely from the cdrom and you can use it and explore a bit. Obviously it's gonna be slow loading everything off the cdrom but give it a go.

If you do give it a go, remember be patient and good luck!

Also back up your sensitive data if you do install ubuntu. Accidents do happen.

Well, the "human" theme is generally accepted as being ugly, but you can install new themes from [gnome-look]("http://gnome-look.org") and similar sites. All you need to do is download them to somewhere handy (I generally put them on my desktop and move them afterwards), open the Themes manager (System, Preferences, Theme) and then fiddle around with the colors and fonts and whatever else you want. My Ubuntu is very, very pretty.

For those of you who are sick, twisted people, there are even themes that make it look a lot like Windows Vista. 

That's a real bonus about Ubuntu - it's so customizable!
