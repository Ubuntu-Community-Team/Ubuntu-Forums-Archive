### Post by oldfred on 2018-02-17
Your sda1 is your ESP - efi system partition.
And the ESP normally has these files (and maybe a few others).

       
```
 Boot files:      /EFI/ubuntu/grub.cfg  
                                /efi/ubuntu/grubx64.efi  
                               /efi/ubuntu/shimx64.efi 

```

You do not show any UEFI boot files and have what looks like a lot of system files from /boot and /usr.
I believe there are some advanced configurations of UEFI where you can have boot files in the ESP. And some other distributions do something like that, but that is not normal with Ubuntu.

Do you know how you got those files in the ESP?

And if you have not done backups, you need to do that using live installer, before doing anything else.
