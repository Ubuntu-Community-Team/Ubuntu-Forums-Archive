### Post by nikonian on 2012-08-05
Add this line to /etc/default/grub:

# Uncomment to get a beep at grub start
GRUB_INIT_TUNE="1000 334 1 334 1 0 1 334 1 0 1 261 1 334 1 0 1 392 2 0 4 196 2"

then ```
sudo update-grub
```
