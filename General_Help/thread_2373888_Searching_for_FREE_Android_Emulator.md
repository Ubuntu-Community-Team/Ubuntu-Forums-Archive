### Post by ajgreeny on 2017-10-10
If your machine is of high enough spec try using **android-x86-7.1-rc2.iso** as a VM then you can run the apps as if using android on a tablet.

Previous versions of the x86 android images have not run at all well, but this 7.1 version is very much better.  I have installed the rc1 which has now been upgraded to rc2 as you will get from the link below.
[http://www.android-x86.org/releases/releasenote-7-1-rc2](http://www.android-x86.org/releases/releasenote-7-1-rc2)
