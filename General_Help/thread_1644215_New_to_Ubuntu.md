### Post by Nexusx6 on 2010-12-13
I'm assuming you have an nVidia card and that you've install the drivers. If thats the case, getting dual monitors working is pretty easy.

First pull up the NVIDIA X Server Settings. To do you this you can navigate to the shortcut through your top bar probably under settings somewhere (sorry, I don't use Gnome), but pressing ATL+F2 and typing out NVIDIA X Server Settings should net you the same result.

Once that is up click on X Server Display Configuration and you should see two boxes on the right hand window that represent your monitors - one will undoubtedly say "disabled" on it. Click on it, then click "Configure" and choose "TwinView", this will give you the one big monitor effect.
