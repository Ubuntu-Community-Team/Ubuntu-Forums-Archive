### Post by James78 on 2010-11-08
Get 10.10, put on disk/USB, start the install, when selecting where to install to, select the partition or disk (ONLY select the disk if Ubuntu is installed on the entire disk!!) where Ubuntu is installed to, start the install, and it'll automatically wipe it and install 10.10.

### Post by karthick87 on 2010-11-08
If it is a partioned dual boot..The easiest way is to logon to your xp and then delete your ubuntu partition.

To delete the ubuntu partition from windows:
Right click on My Computer-->Manage--->Disk Management

There you will see all partitions including ubuntu.Right click the ubuntu partition and delete it.After deleting it you get unallocated free space..Now reboot your windows and boot your system with ubuntu live cd.During installation Select free space for installing ubuntu.Thats it..
