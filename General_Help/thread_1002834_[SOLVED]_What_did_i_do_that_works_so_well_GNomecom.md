### Post by Therion on 2008-12-05
Here's the deal...  Compiz allows you to choose a preselected set of Compiz options by using the menu buttons for "None" "Normal" or "Extra" (or whatever they're labeled, I forget).  Makes things nice and easy and you don't have to mess around with the Settings Mananger for Compiz this way.  It's like Add/Remove vs Synaptic.

Say you select "Normal" for example.  Poof!  A nice little preselected set of options is turned "ON" for you.  Suppose you then go make some *additional* changes via Compiz Config Settings Manager (CCSM).  AH HA!  In that instance, the highlight for the radio button you just chose (in this case the one for "Normal") disappears, since it's no longer just the "Normal" settings anymore; it's something in between the preselected settings -- so when that happens NO button is highlighted.

In short: 

Button ON:  Preselected set of Compiz Options.

No Button ON:  You've made up your own "customized" set of Compiz options that don't match a button.


Make sense?

### Post by SamNSane on 2008-12-05
I think that what you're seeing merely signifies that you've gone into ccsm and changed some settings from the "normal" or "extra" defaults. Mine shows the same thing. There are, of course, a gazillion settings that we can monkey with in ccsm. And even going through ccsm on your friend's machine and setting every last setting the same as it is on your system does not guarantee that the results will be the same -- at least not if you have different hardware.
