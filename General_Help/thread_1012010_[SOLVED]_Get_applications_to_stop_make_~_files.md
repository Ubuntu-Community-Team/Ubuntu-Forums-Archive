### Post by unoodles on 2008-12-15
Is it possible to have applications to stop creating files that end with the ~ char?

Example:
create a new document called x.txt
open in gedit and add some text and save it.
next to x.txt will be x.txt~

nautilus will hide these by default, but I want to remove all of them forever.

---

### Post by oldos2er on 2008-12-15
The backup files are created for your protection. But if you don't want them created, open gedit (Applications, Accessories, Text Editor), click Edit, Preferences; Editor tab, uncheck the box in front of 'Create a backup copy of files before saving'.

---

### Post by albandy on 2008-12-15
These files are a backup of the original file. In gedit go to edit --> preferences and in editor tab unselect the backup option.
