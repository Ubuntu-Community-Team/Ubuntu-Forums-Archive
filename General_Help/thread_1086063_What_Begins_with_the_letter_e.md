Recently I had some woes when X windows decided to commit suicide.
Then I remembered this neat little console based web browser from back in the day called elinks.

This meant that when I booted into a terminal I could type elinks and then browse the web from the terminal...awesome eh?. You don't get images because it's text based but it flys 'like a bat out of hell' (Meatloaf).

And you can download stuff from there.
So that's what I like to think of as a tip.
Save it for a rainy day.

Type elinks in the console (It's in Synaptic) and try it if you are bored.
