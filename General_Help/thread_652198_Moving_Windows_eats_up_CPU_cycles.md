### Post by randavance on 2007-12-28
Hmm, I'm not too surprised that it takes up CPU to move windows on a low end system, but there might be something else going on here.

What you could try is this, open up a terminal (applications > accessories > terminal) and type...

```
gconf-editor
```

This will open a new window with a file structure in the right panel.

Got to apps > metacity >general

and in the right panel click the check box next to "reduced resources"

Unless you already have this activated, it should cut back on CPU considerably, it makes the windows wire frames when you move them.
