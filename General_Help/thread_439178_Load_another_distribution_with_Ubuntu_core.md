### Post by ago on 2007-05-10
It is quite simple:

from wubi devel branch, get the wubi/src/inspector/IsoList.ini and send us a section for linuxmint, it will be available in wubi from next version.

Note: an URL pointing to an alternate ISO with same kernel as official ubuntu is required.
