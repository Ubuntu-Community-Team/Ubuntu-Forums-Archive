### Post by mb_webguy on 2009-02-21
Drag-select, Shift-click, and Ctrl-click may all be used to select multiple files, just as in Windows -- assuming that the particular application or dialog you're currently using allows you to select multiple files.

### Post by Agent ME on 2009-02-21
gFTP might not support selecting more than one file at a time. I haven't tried using that program.

If you need an FTP client, I use _[FileZilla](apt://filezilla)_ (clicking the link will prompt you to install it from your default software repositories), which can work with multiple files selected. It queues up all file transfers as needed.
