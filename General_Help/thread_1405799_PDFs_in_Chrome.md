### Post by fragos on 2010-02-13
If you look in Chrome Options "Under the Hood" tab you'll see some Download parameters. The one that's greyed out likely related integrating Adobe reader. Looks like we'll have to wait for a while for it to come. For now you need to download the PDF to use any viewer other than Google Docs.

### Post by andypi on 2010-02-14
Actually I did get it to work in the end. Spent quite a while going through forums and gave up. Some of them said that if I deleted the firefox extension from certain places then Chrome wouldn't try to use it, which would result in PDFs being opened by acrobat externally. In fact, oddly enough, I found that after deleting those files, PDFs now open fine within Chrome!

Also, I was surprised to see that in my Chrome settings it says "Under the Bonnet" rather than "Under the Hood"! Looks like they're making an effort with British English!
