---

### Post by Mark Phelps on 2011-07-27
Open a terminal and enter "sudo update-grub".  That will regenerate the GRUB menu.  Watch it while it's running to confirm you see something line "Windows 7 (Loader) on ..." being enumerated in the list.

If you don't see that, when it's done, take a look at /boot/grub/grub.cfg -- and see if the lines for Windows 7 are there in the OS Prober section.

If those lines aren't there, your Win7 boot loader files are gone.

You boot from a Win7 install DVD, or a Win7 Repair CD, and run Startup Repair three times.

If you don't have an install DVD, and if you didn't take the time to create and burn a Win7 Repair CD (back when Win7 worked), you have a real problem ...
