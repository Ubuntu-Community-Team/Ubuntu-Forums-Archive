### Post by Nullack on 2008-06-13
If you install CSSM, you will find the ability to edit the default Ubuntu effects. CSSM is a simple menu editor for compiz fusion. I think the setting your looking for is in desktop wall :)

### Post by Nullack on 2008-06-13
You can use synatic, command line or the add remove main menu item. If you use add / remove just search for compiz in the repos and select install. Then, find the desktop wall plugin nd change it to your liking, or turn it off alltogether. You could just set it to keybaord and not mouse based to remove the problem.
