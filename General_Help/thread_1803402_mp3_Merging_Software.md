Excellent link, but one problem is that if you simply merge several mp3s with mp3wrap the vbr header gives the time duration of the first mp3, not the merged mp3. This confuses some media players which simply stop when they get to the time of the end of the first mp3.

@mdmorash, if that's a problem for you, have a look at this post:

[http://ubuntuforums.org/showpost.php?p=10868154&postcount=4](http://ubuntuforums.org/showpost.php?p=10868154&postcount=4)

That was about concatenating the separate movements of a classical work to get one long mp3, but the principle is the same.
