### Post by Jucato on 2007-04-20
> **Gavin77 said:**
> I just did a clean install of Kubuntu Feisty and I notice something weird.
When viewing the root folder using Konqueror it shows all folders (boot, etc, sys, var...) when they should be hidden.  View - Show Hidden Files is unchecked.  The icons are not greyed either.

They should **not** be hidden. That was the (mis)feature that they introduced into Edgy, which many have called a bug. That is why they removed it in Feisty.

[https://wiki.kubuntu.org/FeistyFawn/Herd1/Kubuntu#head-c47f8a02b501783f8ab9e866bcf31b45874788b0](https://wiki.kubuntu.org/FeistyFawn/Herd1/Kubuntu#head-c47f8a02b501783f8ab9e866bcf31b45874788b0)
