### Post by foolsh on 2008-06-27
Firstly make sure your running the latest version [here]("http://www.winehq.org/site/download-deb")

Secondly make sure you run winecfg before running win32 programs the first time.

Thirdly you can check your programs compatibility with wine [here]("http://appdb.winehq.org/") 

Fourthly Good luck! and don't be afraid to ask more detailed questions,  some people actually prefer more details like the name of the program your trying to run and such.
