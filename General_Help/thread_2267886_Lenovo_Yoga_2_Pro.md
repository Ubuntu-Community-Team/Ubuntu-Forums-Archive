### Post by faure212 on 2015-03-04
I have been using Ubuntu since version 6 (Edgy something), and for a while enjoyed it. 
Once I discovered (the lightweight) Lubuntu, which ran VERY well on my old hardware, I was happy with it.  I tried and hated Unity, and found gnome also to be somewhat quirky. 
But recently I bought a Lenovo Yoga 2 Pro, with a gorgeous 3200X1800 screen (AKA HiDPI).  Windows 8.1, which came with it, works OK on it-- EVERYTHING works, although not terribly fast, and I hate the weird Metro UI, but I *can* use it.
Ubuntu on the Yoga machine with the HiDPI display is a disaster.  I tried everything-- Lubuntu, KDE, gnome, Ubuntu/Unity-- nothing works correctly under all circumstances.  In addition, the sound works only SOMEtimes, the function keys only work correctly in Ubuntu/unity and gnome, but not in Lubuntu or KDE.  The auto-rotation does not work, and the touchpad is a disaster.  All these things do work correctly in Windows 8.1 on the very same hardware. 

So I might have to throw in the towel, and stick with Windows, which I detest as much as any of you. 
BTW-- I followed to the letter all the arcane instructions on installing Ubuntu on this machine.
