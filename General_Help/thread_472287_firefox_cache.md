Hello atlfalcons866!


I think that it is not very useful to have a very big Firefox cache. This cache is only used to cache the data that Firefox downloads from the internet, such as advertisements, pictures, texts, etc. This means that Firefox will summon more and more trash if you allow it to do so.

I think that the 50 MB - the standard setting - is a fair compromise.


Cheers!
