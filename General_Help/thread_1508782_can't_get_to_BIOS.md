The BIOS screen is accessible during boot up and on most systems I've used it's **F2** and you must press it before the OS boots up. My computer is a gateway so when I see that logo I press F2 and it opens up BIOS. This method also works on my sis' laptops. You want to make sure that you press it before it accesses the hard drives and loads the os, otherwise it won't work. Hope this helps!
