### Post by Existentialist on 2008-03-14
A 64-bit OS will only work on a 64-bit CPU, and the 64-bit X-fi drivers will only install on a 64-bit ubuntu install.  There is a version of OSS that supports 32-bit installs and also has some support for the X-fi cards.  Read more about installing it in this thread: 

[http://ubuntuforums.org/showthread.php?t=571656](http://ubuntuforums.org/showthread.php?t=571656)
