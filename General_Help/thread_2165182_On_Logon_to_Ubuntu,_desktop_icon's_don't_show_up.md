### Post by Vr_Rm on 2013-10-05
Hi jervin2, sorry this response came so late, hopefully you were able to  solve this issue.  I was running into this problem running the Cinnemon desktop.  What seemed to fix it for me was to turn on some of the default desktop widgets.  Presumably, when these get added it causes something to get kicked off that shows the desktop.

Menu | System Settings (Control Center -- there are two you want the Cinnemon settings) | Desktop | Show a computer icon + Show a home icon + Show the trash

(maybe other combinations of the above work too.)  Hope that helps someone.  (Including myself, after I've forgotten how to do this and have to Google for it again.)
