### Post by mjitkop on 2010-01-18
Hi atravnic,

here is something to try but I can't guarantee that it will work because I have never done it from one operating system to another.

Start your Vista account and open Firefox. In Firefox, click on "Bookmarks" then "Organize Bookmarks..." In that window, click on "Import and Backup" and choose "Backup..." This will create a file with the extension .json. Save this file in a location that can be accessed from your Ubuntu account.

Start your Ubuntu then in Firefox, open "Organize Bookmarks..." This time, in "Import and Backup", choose "Restore..." and select the file you saved previously. Note that you should be able to get to your Windows partition from Ubuntu by default.

This should restore your bookmarks in the same order they were.

Good luck! :D
