### Post by raja.genupula on 2013-04-13
I too have experienced the same thing with the new version of firefox. but actually if you still wants the old version behaviour then you can get it by installing it.

all you need to do is simply 

> sudo apt-get install firefox=<here mention the version name you want>

some information for you : [http://support.mozilla.org/en-US/kb/install-older-version-of-firefox](http://support.mozilla.org/en-US/kb/install-older-version-of-firefox)
