### Post by Thomas Jensen on 2008-10-31
I'm going to go with the easier way of installing Scribus:

When in Ubuntu, from the main menu in the top left corner of the screen, select **Applications > Add/Remove...**

This will present you with a lot af software that Ubuntu can download and install for you automatically. You will find Scribus in the graphics section:

[[IMG]http://img185.imageshack.us/img185/3053/scribushd3.th.png[/IMG]](http://img185.imageshack.us/img185/3053/scribushd3.png)[[IMG]http://img185.imageshack.us/images/thpix.gif[/IMG]](http://g.imageshack.us/thpix.php)

Check the box next to the icon and hit **Apply Changes**.

I, too, would rank page layout and design software InDesign > Scribus > Publisher.
