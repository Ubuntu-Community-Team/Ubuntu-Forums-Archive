### Post by 3rdalbum on 2009-04-22
What wireless chipset is in use in this laptop?

If you don't know, then go to the terminal window (Applications > Accessories > Terminal) and type in this command, and then copy and paste the result into this thread.

```
lspci
```

A lot of wireless chips are automatically detected and usable out-of-the-box; can the network manager applet on the top panel see any networks?
