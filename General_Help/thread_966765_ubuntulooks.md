### Post by fragos on 2008-11-01
Ubuntu-Desktop is only a pointer to the many Gnome packages. If it's removed the packages won't be. The only potential problem will come when you upgrade to the next release. For clean installs from a CD this won't matter since everything is replaced anyway.
