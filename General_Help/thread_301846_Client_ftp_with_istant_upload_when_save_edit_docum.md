### Post by koenn on 2006-11-18
For small online editing, I open the file wfrom nautilus.

This is how it works :
Places -> Connect to server -> create a connection to your ftp server
Open the connection (this usesNautilus file browser)
Browse to a file, rightclick and open with text editor
Edit file
The 'Save' button is disabled, but of you close the file, gedit text editor asks if you want to save changes. Click Yes. 

That's all. Works great for small editing jobs.
