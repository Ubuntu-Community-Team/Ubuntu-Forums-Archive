### Post by jeffjj on 2005-04-30
The best open source editor for Java is Eclipse...eclipse.org. Quite a few of the older C/C++ programmers will try to make you believe that a full featured IDE is something that is not important. I always hear the "learn the language, not the IDE excuse." I've come to the conclusion that its just that. A full featured IDE is amazing to work with. The ability to do code completion, refactoring, etc. The learning curve is tough, but you can learn new things as you have time. I can floor the development time of any person that thinks a glorified text editor is good enough.

For C I have been using Anjuta. Its a fine editor. Not even close to Eclipse on features, but seems to be the closest to a real IDE in the GNOME world.

I am hoping that now that Novell joined Eclipse that they will put some time into puttting some plugins for Eclipse for Mono.
