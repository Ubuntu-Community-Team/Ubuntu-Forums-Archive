### Post by nymphaeles on 2006-02-10
In Windows, you need a SSH client such as Putty.  I happen to use a GUI client called SSH Secure Shell version 3.0 from [http://www.ssh.com](http://www.ssh.com).  Download the current version for Windows (exe extension) and you're good to go.  Connect to your ssh server either by IP or host name.
