### Post by nickleboyblue on 2011-08-26
Ubuntu handles this differently.  Double-click the url box and it highlights the entire address.  Single clicks only place a cursor there.  I know of no way to change this behavior, but I wouldn't be surprised if there was a way.  Ubuntu is a different operating system from windows; it will have many more small things that, once you get used to them, you will begin to miss when using other operating systems.
