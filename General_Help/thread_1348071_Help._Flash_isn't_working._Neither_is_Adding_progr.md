### Post by MooPi on 2009-12-06
First download flash from Adobe
[http://get.adobe.com/flashplayer/thankyou/?installer=Flash_Player_10_for_Linux_%28.tar.gz%29](http://get.adobe.com/flashplayer/thankyou/?installer=Flash_Player_10_for_Linux_%28.tar.gz%29)
Second extract the files 
Third navigate to your home folder and key in > Ctrl + hthen navigate to .mozilla and create a folder named "plugins".
Last copy and paste libflashplayer.so from the extracted files to ~/.mozilla/plugins
Restart Firefox and flash should work.
