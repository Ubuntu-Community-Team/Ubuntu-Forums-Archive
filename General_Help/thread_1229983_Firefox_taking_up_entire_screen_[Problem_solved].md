### Post by thegreatdestroyer on 2009-08-02
I use Ubuntu 8.04 LTS, Linux Kernel 2.6.24-24-generic, Firefox 3.0.12 & GNOME 2.22.3.

The problem I was having was that after going to online manga sites and accidentally running into other sites with heinous advertising Firefox upon reopening would go full screen covering both my top and bottom panels and stay that way. When this happened only the File Edit etc menu was available. There was no title bar and thus no close, max unmax or minimize buttons. The only thing that seemed to help at the time was a complete shutdown. Two months later even that stopped working.

After that I went into Ubuntu IRC channel and posted pics of what Firefox looked like when it did this. I got a few suggestions and experimented with all of them. The one that works every time is to press F11, press the max unmax button, press the max unmax button again, resize Firefox's window and close it. When it opens again it's fine, but opens at the same size as when I closed and I just maximize it.

That's the solution for me. I thought perhaps posting this would help others having this problem. Should I post this on Firefox's forums as well?
