### Post by vidak on 2007-03-18
Did you delete those files in konqueror or something like that? Because in this case those files was moved to Trash instead of being deleted. Check and delete the Trash. To avoid such problems, Shift+Del is usually useful - in this case the files are deleted immediately. Another way is to delete the files from console using rm, or using Midnight Commander.
