### Post by aysiu on 2011-05-30
There are several things involved. One to check is that you're part of the *admin* group. Apparently, you are. The second is to check that the /etc/sudoers file contains the correct parameters. The third is to check that the /etc/sudoers file has the proper ownership and permissions.

Apparently, your installation got its file ownership screwed up. So while you're booted into recovery mode, do ```
chown root:root /etc/sudoers
``` Then do ```
cp /etc/sudoers /etc/sudoers.backup
nano /etc/sudoers
```
