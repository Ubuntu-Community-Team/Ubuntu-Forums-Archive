### Post by 3246251196 on 2011-06-27
If I have a hard-disk with Ubuntu fully installed on it and I want to - all of a sudden - use Windows XP, is it mandatory that I firstly format the hard-disk first?

So far, I have used two versions of Windows XP; one of them is from a few years before the other. Both copies of Windows XP when in start-up from booting from CD are causing errors. One of them is a BSOD error (0x0000007B) and the next Windows CD is stopping and giving me an error with setupdd.sys (error code 4).

Is it required that the hard-disk is formatted before you even put a Windows boot CD in?

---

### Post by mocoloco on 2011-06-27
Answers: yes, and then no.
Yes, the hard drive will have to be formatted for Windows XP to be installed.  And no, you don't need to format it before putting a Windows disc in, the XP installer will walk you through formatting the drive, just follow the instructions to format as NTFS.
