### Post by ieee488 on 2011-03-22
Are you reading a book on HTML and CSS?

A few remarks:

1.) You have two <body> tags. Use a HTML validator.

2.) For <div class="h1"><H1><b>The world of linux</H1></b></div>, there is no need to use <H1> and <b>. That should be taken care of by the CSS

3.) <img> is not a block tag; it is an inline tag; it should be placed inside a block tag

4.) {display:intline;}   has a spelling error; use a CSS validator


My suggestion to use is, forget about the CSS for a moment, and concentrate on the HTML.
