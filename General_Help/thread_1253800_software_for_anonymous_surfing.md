### Post by fluffman86 on 2009-08-30
That's just the way Tor works.  Instead of making an almost instant connection to a web server, you go through multiple other computers first to anonymize your data.  

Also, there are only so many computers out there running Tor, so please don't use it for just randomly downloading large amounts of material, especially if you have other legal options available where Tor isn't necessary.

That being said, you can try using a web proxy server.  There are lots available on the web for free.  Again, they're usually a bit slower than your standard internet connection.  It's the price you pay for anonymity.

### Post by t0p on 2009-08-30
For anonymity I use **Your Freedom** - available from [www.your-freedom.net]("http://ubuntuforums.org/http//www.your-freedom.net").  It's a java program so you'll need java installed.  You also need to register at the site before you can download the program, but that's no great hardship.

*Your Freedom* is free to use if you're just browsing the internet/using IM clients/IRC/that kind of thing.  If you need any ports forwarding, you have to pay a fee.  But as you only mentioned surfing and downloading, it won't cost you a thing (though like all software developers they welcome contributions!).

Note: of course a connection through *Your Freedom* will be slower than browsing without a proxy.   But it should be faster than Tor as you'll be using just 1 proxy server, whereas Tor uses several for each connection.
