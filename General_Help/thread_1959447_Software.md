### Post by 3Miro on 2012-04-15
Some windows programs can be installed via wine and if they work, they work rather well. A quick Google search for "Microsoft Streets & Trips" suggests that it is not one of the lucky programs and in fact it requires IE so it doesn't work under Ubuntu.

You can try to find an alternative to this program or see if it works under VirtualBox (which is away to run a copy of one OS within another). If you only need one application and you don't use it all the time, VirtualBox is a good way to go.

[https://help.ubuntu.com/community/VirtualBox](https://help.ubuntu.com/community/VirtualBox)
