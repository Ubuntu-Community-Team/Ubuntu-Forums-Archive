### Post by cariboo on 2008-11-15
A windows installation needs its boot files on the first partiton of the first hard drive, or it will not boot. If you want to install Ubuntu first, create and empty partion at the start of the hard drive for windows, Then install Unbuntu on the remaining space. You will have to reinstall grub after installing Windows.
