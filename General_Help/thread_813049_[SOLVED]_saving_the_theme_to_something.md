### Post by paulphillips on 2008-05-31
Yes there is!

Easiest way is probably to backup (copy) your entire home directory somewhere (another harddrive, USB Key etc).  Install Hardy, then copy over your home directory again.

Alternatively, you could research all the config files that setup your theme and back them up only.

Hope this helps!

P.S. For this very reason, I keep my user account directories on a separate harddrive.
