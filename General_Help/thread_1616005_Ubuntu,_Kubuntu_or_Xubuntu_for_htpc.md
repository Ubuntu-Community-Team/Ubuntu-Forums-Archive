### Post by papibe on 2010-11-07
Even though I don't have a good sense of how fast is that CPU (actually any AMD CPU). My impression is you won't be able to play 1080p on that machine as is. Maybe 720p, but I'm not sure.

Luckily, you can install a budget Nvidia card (like a Geforce 210 for about $40). That would give you hardware video acceleration, which is almost a must if you want to play HD movies. I wouldn't go for an ATI card since Nvidia is THE best supported brand on Linux.

As for the connection to the network: if the movies are not going to be in the same machine, I'd recommend using a wired connection to the media's repository.

As for distro.. take a look at [XBMCbuntu]("http://wiki.xbmc.org/?title=XBMCbuntu"). It is not exactly a distro, but an special installation. XBMCbuntu is basically the media center software XBMC over a minimal ubuntu installation.

Good Luck
