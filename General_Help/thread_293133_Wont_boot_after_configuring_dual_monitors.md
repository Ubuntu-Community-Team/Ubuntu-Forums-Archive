### Post by LenzM on 2006-11-05
> **bsalmon said:**
> 
	Screen		"Main Screen"
	Screen      1   "Second Screen" RightOf "Main Screen"


I don't know if this is the reason it's not all working, but I think you have to either number all of the screens or none of them, for simplicity's sake I think it's easier to not number them and let it be done automatically.  So replace the above two lines with:
```

	Screen		"Main Screen"
	Screen          "Second Screen" RightOf "Main Screen"

```
