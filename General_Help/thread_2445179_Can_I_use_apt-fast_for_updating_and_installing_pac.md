### Post by crip720 on 2020-06-10
From a security point it is not good to run a third party program with root control.  For speeding up updates might be better to go into software and test for best mirror.  As long as you don't have a full packed drive, doubt that you would save much time.  Downloading updates is controlled by your internet speed and if you do it at least weekly, usually does not take that long.
