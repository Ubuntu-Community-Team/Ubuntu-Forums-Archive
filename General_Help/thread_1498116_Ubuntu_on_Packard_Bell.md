### Post by WinRiddance on 2010-05-31
Did you make sure that you burned an ISO file correctly onto a regular CD with an ISO burning program? I'm asking because I've seen problems with people using DVD disks and then trying those disks on older machines that didn't have DVD compatible CD drives or computers with a DVD boot option. I've also seen some cases where people burned ISO images with standard burning software which is impossible. It has to be software for burning ISO images.

Also, is your computer bios set up to read the LiveCD first? You really haven't provided a lot of information here. There's a little more to installing Ubuntu then just having a working hard disk and better help can be provided when more details and steps you've taken are provided by you.
