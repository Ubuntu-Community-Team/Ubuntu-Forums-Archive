### Post by Sanjima on 2011-04-19
Right-click on any blank space on the panel where it used to be, click Add to Panel, and add the applet called Indicator Applet

The volume control panel icon is part of this Indicator Applet and this should solve the problem.
I had this same problem earlier today actually xD Hope this helps.
