### Post by Paqman on 2012-04-23
Not exactly the same as the OS X behaviour, but installing the package [preload]("apt:preload") will allow the system to silently load you most commonly used applications into RAM before you need them, so they'll open much quicker.

In general Linux doesn't drop things out of RAM just because you've closed them. Stuff stays cached until the RAM is actually needed for something else.
