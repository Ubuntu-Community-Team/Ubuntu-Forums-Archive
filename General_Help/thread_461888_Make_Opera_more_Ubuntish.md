### Post by JohanSwe on 2007-06-02
**I hope this thread could be used to share tweaks to give Opera a more Ubuntuish look.**

[COLOR="Gray"]Opera as software is (unfortunately) not open source, but they are leading when it comes to support in their browser for open standards on the internet such as HTML, CSS and SVG. Very important to me, and for now I am running Opera :)[/COLOR]

When installing Opera it isn't that Ubuntu-ish and would need some tweaking.

Tweaks:
[LIST]
[*]In the skin dialog (right click somewhere on the toolbars and choose Customize..) edit colorscheme for the standard theme to "gold".
[*]The font for menus, File/Preferences/Fonts/Browser menus, to Batang (I don't know the name of the Ubuntu standard menu font, or if Batang comes with Ubuntu, if you can't find it, choose another one ;)
[/LIST]

Wish list:
[LIST]
[*]Drop down menus not ugly grey.
[*]Standard Gnome open file dialog.
[*]Transparency for Opera desktop widgets.
[/LIST]
