### Post by sudodus on 2015-05-21
I would suggest that you try ***Lubuntu, Ubuntu Mate and Xubuntu***. With one 1GB RAM, I think Xubuntu and Ubuntu Mate will work well, but Lubuntu is lighter, which makes a difference if you want to play video.

Even if the processor runs with 64-bits, ***I suggest that you use a 32-bit version*** because it needs less RAM for the same tasks.

If you have a good internet connection I suggest that you download several iso files.

Start with the ***long time support version 14.04.1 LTS***.

[Try Ubuntu (Kubuntu, Lubuntu, Xubuntu, ...) before installing it]("http://ubuntuforums.org/showthread.php?t=2230389")
