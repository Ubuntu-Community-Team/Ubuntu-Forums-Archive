### Post by dimoftheyard on 2010-03-27
If you know what driver you need for internet you can get it from [packages.ubuntu.com]("packages.ubuntu.com") . You can download .deb files from there and install them with
```
sudo dpkg -i files.deb
```

### Post by colobix on 2010-03-27
You can install ubuntu from windows as an under OS which means it will be installed as an application on windows but you can start it from the boot loader as if it was a 2nd operating system. The ubuntu installer for windows is called Wubi.
This is a perfect way to try it out before you install it for real. Here's the link [http://wubi-installer.org/](http://wubi-installer.org/)

A 2nd solution is USB stick if you have one.
