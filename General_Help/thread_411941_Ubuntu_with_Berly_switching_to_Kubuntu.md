### Post by H.E. Pennypacker on 2007-04-17
Beryl is its own window manager, and it should function properly using either KDE or Gnome.  Note that is is not a desktop environment, which KDE and Gnome are.

If you do have any problems with Beryl, make sure to check out Beryl's own forums, which provide support dedicated to Beryl.
