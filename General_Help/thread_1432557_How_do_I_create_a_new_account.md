### Post by silvervein on 2010-03-17
The path that was stated is right, but you have to go to the advanced tab.

K -> System Settings ->

Once that opens you will have 2 tabs **"General"** and **"Advanced"**. Select **"Advanced"**

The last icon in the bottom right should be **"User Management"**

You will be asked for a pwd next (it's just your user pwd) then user manager opens up and you can create a new account
