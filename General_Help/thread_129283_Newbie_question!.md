### Post by gingermark on 2006-02-13
The .exe files are indeed for Windows. The DVD features the combined install and live CD, that latter of which has some windows programs on it (kind of a stripped down version if the OpenCD).

Anyways, to install firefox using Adept, run it, type in your password, and then type 'firefox' in the "Quick Filter" box.. The 1.07 version should be available in a list. Right-click on its entry, and select "install". Then click "Commit Changes".

OR, run "Konsole" and type ```
 sudo apt-get install firefox
```

If you want version 1.5 have a look at [https://wiki.ubuntu.com/FirefoxNewVersion](https://wiki.ubuntu.com/FirefoxNewVersion) - it's a bit more complicated as Firefox affects other programs.
