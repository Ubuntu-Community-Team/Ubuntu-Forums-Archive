### Post by apparle on 2009-07-30
I will explain you somethings. Ubuntu uses OpenOffice, WinXP uses Office 2003, Vista uses Office 2007 usually.
 
1. Microsoft Office 2003/XP saves files like .ppt for powerpoint, .doc for word etc. Office XP can open only these files.

2. Microsoft Office 2007 saves files like .pptx for powerpoint, .docx for word etc. Office 2007 can also open 'Office XP' files. If you have installed the service pack2 for office 2007 then it can also open openoffice files.

3. Open Office can use all the files, but the files are usually saved in the format .odt

If you want your files to open in windows then follow this guide
[http://www.winona.edu/library/ig/help/openoffice_writer_save.htm](http://www.winona.edu/library/ig/help/openoffice_writer_save.htm)
The guide is for writer only but you can do similar steps for presentation also.

Don't blame good softwares for your ignorance.

If you had bought a costly version with vista then also your files would not have opened in Windows XP and you would have to follow similar steps as above, so don't judge things particularly softwares by there cost.

### Post by chaanakya_chiraag on 2010-01-18
The main thing is to realize that OpenOffice, **by default** will use the open document format: .odt for text documents and .odp for presentations. However, since MS Office cannot open these tupes of documents (except for the latest version, which you have installed on your vista pc) you must do File->Save As... every single time. In order to change which file format is used by default, go to Tools->Options and click on Load/Save.
