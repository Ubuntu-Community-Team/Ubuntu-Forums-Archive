### Post by C.B. on 2005-07-17
NoteCase is a hierarchical note manager (aka. outliner). It is free and open source for following platforms:
    * Linux/Unix (with GTK+ 2.x  installed)
    * Windows 9x/2000/XP
and is available at [http://notecase.sourceforge.net/index.html](http://notecase.sourceforge.net/index.html) .

Unfortunately you'll need to compile from source as it's only specific to RedHat Fedora Core 3.

### Post by RikoW on 2005-07-19
Hi :)

I like tuxcards a lot:

[http://www.tuxcards.de](http://www.tuxcards.de)

It gives you the tree structure of your notes in the left pane and the notes in the right.

Also kdissert might be worth looking into:

[http://freshmeat.net/redir/kdissert/42508/url_homepage/show.php](http://freshmeat.net/redir/kdissert/42508/url_homepage/show.php)

it combines the mind mapping functionality with the tree like structure of the notes.

I personally use tuxcards a lot to archive all sorts of notes in the hierarchical way.  
I plan to use kdissert instead of freemind in the future because if the better pane views which are available. But can't tell you much about the daily usage yet, since, except importing my freemind maps, I haven't really used it yet.

Hope that helped a bit!
