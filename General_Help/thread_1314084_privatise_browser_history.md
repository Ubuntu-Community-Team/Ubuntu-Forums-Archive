### Post by undecim on 2009-11-04
Definitely install the NoScript FireFox plugin. When you use it, a page can display static content and do nothing else until you allow it to.

BTW: If you are using the plugin and a site looks wrong, it's probably because the site requires javascript to display its content (because the web dev has never heard of <noscript> tags.)
