### Post by mechro on 2010-01-11
> **Hunter2379 said:**
>  
What is it that is causing al these errors?
Is the original XP cd broken? 


I haven't found a definitive answer but from my own experience of a non-booting XP CD I'd say the CD bootloader will look for any trace of a working or broken XP HD installation before moving on to the normal menu. In effect, a broken XP HD installation can then "break" the CD... Catch 22. The only way to get the CD working normally is to nuke/format the XP HD installation.
