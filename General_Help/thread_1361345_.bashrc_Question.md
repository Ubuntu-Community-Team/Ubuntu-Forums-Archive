### Post by beastrace91 on 2009-12-21
Howdy All,

So I currently have the following in my .bashrc file: ```
alias = wine='wine-1.1.35'
```

Essentially I have multiple Wine versions setup that all work (wine-1.1.35, wine-1.1.34 ect.) and I want to alias it so the latest version is linked to the most up to date Wine version I have installed. Any ideas why my above .bashrc does not work?

Thanks in advance,

### Post by falconindy on 2009-12-21
The first = is erroneous. You may also want to consider giving your alias a name that won't conflict with the name of a bin in your path, either. So something like:
```
alias lwine='wine-1.1.35'
```

### Post by PeEllAvaj on 2009-12-21
The other way that ubuntu does this for a lot of programs is by putting a sym link in /usr/bin and linking to the correct binary in /etc/alternatives/.

There is even a program for managing this called "update-alternatives".

Try out ls -la /usr/bin/ to see how this is being used already on your system.
