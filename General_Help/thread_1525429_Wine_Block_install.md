### Post by Sammi on 2010-07-08
In Linux all executable files (this means programs or applications in lay-mans terms) need to have permission to be executed (or run as you would call it in lay-mans language).

You can give this permission by right clicking of the file and choosing properties. You'll find the option to toggle "executable" right there somewhere. Executable needs to be on for a executable file like install.exe to be able to execute.
