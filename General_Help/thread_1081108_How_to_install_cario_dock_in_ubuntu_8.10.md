### Post by rhosigma on 2009-02-26
you can also get Ubuntu Tweak from the [GetDeb]("http://www.getdeb.net") website and install it that way. I would recommend using Ubuntu Tweak because you also have the ability to add the Cario dock repository to your list which will help with the latest updates.


On a side note:
To auto-load cairo-dock at login, goto System>Preferences>Sessions.

You’ll be shown a new window listing programs that auto run at login. All you need to do is add cairo-dock to the list by clicking the “Add” button.

Once you click add, name your item “Cairo Dock” and in the command box type “cairo-dock” without the quotes. You can leave the comments field blank. Then click OK. That’s it! Now when you login in, Cairo Dock should autorun for you
