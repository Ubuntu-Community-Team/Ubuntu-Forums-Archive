### Post by jerenept on 2010-08-01
Partitioning is only necessary if you are going to be installing from the Live CD. Wubi does not require any partitions, it simply creates a disk image file on Windows which is used for Ubuntu.

If you choose to partition your Hard Disk (With the CD/DVD install) it is recommended to de-fragment your hard drive before installing, and, you can set your Ubuntu partition to whatever size you like.
