Did you enable swap in /etc/fstab

add a line in the file with

```
UUID=<swapuuid> swap swap defaults 0 0 
```to find the swap uuid. run ```
blkid
``````
#
# /etc/fstab
# Created by anaconda on Fri Apr 20 14:51:50 2012
#
# Accessible filesystems, by reference, are maintained under '/dev/disk'
# See man pages fstab(5), findfs(8), mount(8) and/or blkid(8) for more info
#
UUID=25eeb02b-8dab-45aa-bd5b-b3294789f042 /       ext4    defaults,noatime        1 1
UUID=6b08137f-0320-4e40-aa7f-865a9aafc6f8 swap    swap    defaults                0 0
```Sample fstab

when fstab is working

run to enable
```
sudo swapon -a
```run ```
swap -s 
```[to check]("https://help.ubuntu.com/community/Fstab")

[https://help.ubuntu.com/community/Fstab](https://help.ubuntu.com/community/Fstab)
