### Post by TmP on 2006-03-29
I'm not shure about your problem, but I used DVD RIP and a typical set of codecs and libs installed with the automatix (type it into search on the Ubuntu forums). It lets you RIP, and encode into Xvid.

If you want it to run in Windows, download and install ffdshow, 
or a grat freeware player: Allplayer. It has an update option that checks what codecs you have and downloads the missing ones. 

It's available here:
[http://www.programs.pl/pobierz,139.html](http://www.programs.pl/pobierz,139.html)
and here's the website of the author:
 [http://www.marbit.com.pl/?newlang=eng](http://www.marbit.com.pl/?newlang=eng)
