### Post by Yownanymous on 2008-11-19
I've done it before, so I'll tell you how to do it.
Go into System -> Administration -> Synaptic Package Manager
Now, if you haven't updated the repo information, then I suggest you do so by pressing the Reload button. After that, do a search for ubuntustudio (all one word), and tick the option ubuntustudio-desktop, then click Apply. Synaptic will do the rest. For best results, give your machine a restart and you'll have a fully functioning Ubuntu Studio.
