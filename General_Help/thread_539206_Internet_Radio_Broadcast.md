### Post by arsya on 2007-08-31
What you can find in ubuntu repository, related with audio streaming server is:
icecast-server, [http://www.icecast.org/](http://www.icecast.org/)

But of course you need a computer that can be reached in the internet (public ip).

or perhaps another tool would be
darkice - Live audio streamer
DarkIce is an IceCast, IceCast2 and ShoutCast live audio streamer. It takes audio input from a sound card, encodes it into mp3 and/or Ogg Vorbis, and sends the mp3 stream to one or more IceCast and/or ShoutCast servers, the Ogg Vorbis stream to one or more IceCast2 servers

In this case you send the mp3 stream to already publicly available shoutcast/icecast server.
