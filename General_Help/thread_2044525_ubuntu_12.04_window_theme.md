### Post by Jimmyfj on 2012-08-19
Download and extract the theme you want to install in your home folder.

The open up a terminal and enter:

```
gksu nautilus
```

This will open Nautilus in root mode and give you access to the entire file system, so please be carefull.

Then you find the themes folder:

**file system/usr/share/themes**

Open a regular version of Nautilis and copy your theme folder into the theme folder. And you should be good and ready to use the theme.
