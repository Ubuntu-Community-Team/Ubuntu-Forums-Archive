### Post by donkyhotay on 2007-01-03
I have used both yakuake and tilda both and I have found yakuake to be much better. I prefer gnome to KDE and personally find it frustrating that I need the K libs just for yakuake but for me it's worth it to have multiple terminals within one drop-down. The website says it all but basically what both programs do is create a "quake style" drop down command line. It's great for quickly inputting commands at the push of a button rather then loading up a new terminal each time.
