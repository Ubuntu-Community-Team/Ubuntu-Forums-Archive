### Post by aysiu on 2005-12-07
I can't imagine cookies would have anything to do with it.

The only way to figure out the problem is to run experiments, trying different things to isolate the circumstances under which crashes occur.

Try these:

1. Creating a new user. If no crashes occur, something's wrong with your current user profile.

2. Using a different desktop environment (KDE or XFCE). If the crashes don't occur there, it's not the applications--it's the environment.

3. Thinking about what you've done recently. Did you install some program that may have borked your system? Did you make any major changes to your configuration?
