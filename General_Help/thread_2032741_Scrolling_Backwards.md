### Post by PhantomTurtle on 2012-07-24
Do you mean Inverted scrolling? [http://www.omgubuntu.co.uk/2011/07/reverse-natural-scrolling-ubuntu-os-lion]("http://www.omgubuntu.co.uk/2011/07/reverse-natural-scrolling-ubuntu-os-lion").
If you want to install that run in terminal,
```
sudo add-apt-repository ppa:zedtux/naturalscrolling
sudo apt-get update
sudo apt-get install naturalscrolling
```
