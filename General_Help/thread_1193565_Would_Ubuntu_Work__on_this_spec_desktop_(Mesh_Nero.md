### Post by Jimbley on 2009-07-20
I know this thread's a bit old, but I'm trying to use a Belkin F5d7000uk card with Ubuntu (Intrepid and Jaunty) with little success so far!!

There are three known versions of this card, each with a different chipset:

[LIST=1]
[*]Broadcom BCM4306
[*]Realtek RTL2500
[*]Realtek RTL8185L (ver. 7200)
[/LIST]

I can't comment on the suitability of the first two versions, but I've been unable to get the third version working and connected to a wireless network.

The ndiswrapper wiki page for this card is [_here_]("https://sourceforge.net/apps/mediawiki/ndiswrapper/index.php?title=Belkin_F5D7000UK").
