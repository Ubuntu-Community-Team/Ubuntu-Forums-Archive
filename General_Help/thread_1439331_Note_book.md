### Post by the yawner on 2010-03-26
If it runs Vista fairly well it should have no problems with the latest version 9.10 or even the upcoming 10.04. Although there might be a bit of an issue with wifi on certain wireless network cards.

### Post by darolu on 2010-03-26
Install the latest STABLE release, which as today (March 26th, 2010) is Ubuntu 9.10 "Karmic Koala"; I think the Celeron processor in your computer (Celeron M 530) is a 32-bit one, I'm not completely sure though ([Wikipedia article](http://en.wikipedia.org/wiki/Merom_%28microprocessor%29)), so play it safe and install the 32-bit version.

[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

As the yawner said, try the LiveCD first, to test if your Wireless Card works properly, etc. Remember it will run rather slow in the Live session, once you install it, it will be much faster.

On April 29, 2010, a new version will come out, Ubuntu 10.04 LTS "Lucid Lynx", you will be able to upgrade via Network (recommended) or Alternate CD. Your hardware should run it flawlessly.
