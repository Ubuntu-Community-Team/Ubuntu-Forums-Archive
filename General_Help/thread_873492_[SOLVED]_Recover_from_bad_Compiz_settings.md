### Post by ericesque on 2008-07-29
I have observed that the CompizConfig Settings Manager will allow users to save settings which render their session useless.

I've read about people setting all windows to be completely transparent.  I personally have set all windows to fixed positions on viewports that do not exist using the Place Windows plugin.  The following should get you out of a jam if you know which setting you goofed:
[B]
Logout:[/B]  since some of us will have no gui elements to do so, use
CTRL+ALT+Backspace

**At Login prompt:** 
Click 'Options' in the lower left hand corner
Click 'Select Session...'
Select 'Failsafe Gnome'
Click  'Change Session'
Enter your usual login credentials.

**In Gnome Failsafe session:**
Fix your goof!!
