### Post by soldier1st on 2012-07-21
As Mark Phelps said: Wine HQ is a website. you don't install wine through there but from the repositories. you can use that site to see how good a windows app will run in wine. Also, using Wubi is safer(the bootloader will not get changed) but your Ubuntu install will be slower as it relies on Winblowz to boot. If it goes, so does your ubuntu install. If you want to play games but dual boot Windowz+Ubuntu. use Winblowz for games but ubuntu the rest.
