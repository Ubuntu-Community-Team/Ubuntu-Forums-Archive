### Post by clevertomato on 2009-12-04
What's the difference? Why does some software show up in Synaptic Package Manager and other software shows up in Add Remove Software? Just curious.

---

### Post by mikewhatever on 2009-12-04
Add/Remove is a simplified version of Synaptic, with mostly gui based apps listed. Synaptic, on the other hand, lists all packages, including libraries, devs, etc.

---

### Post by Sam on 2009-12-04
They're both front-ends of the same underlying package manager, but Synaptic is more complete.
