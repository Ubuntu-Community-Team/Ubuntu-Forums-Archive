There are still three big problems with the Nautilus manual sorting though.

2. I cannot properly change the order. If I drag the last icon in a window to the beginning, instead of making place for it by pushing all other icons one step back, the icons are placed on top of one another (which sort of makes sense because they are not aligned to a grid).
3. Image viewer programs like Eye of GNOME and Gwenview still show the files in alphabetic order. Again, this is to be expected as with a "icons can go anywhere" manual sorting the program can't tell which item is supposed to be first.
Hence the last two problems are consequences of the first.

The only topic I found about this is [http://ubuntuforums.org/showthread.php?p=5473275](http://ubuntuforums.org/showthread.php?p=5473275) which describes the same problem.
