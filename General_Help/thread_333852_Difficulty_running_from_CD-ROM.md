### Post by HereInOz on 2007-01-08
Sorry I forgot the second part of your question

You can click the install icon, but you will then need to configure a dual boot situation, which can go awry sometimes.  Always back up all your data before you install Ubuntu as a dual boot with WinXP.  Ubuntu is smart enough to set up a dual boot, but as I said, it can go wrong.  Or more to the point, you can get it wrong :-)

The thing is, though, that if it doesn't run properly, albeit a little slowly, off the CD, it may not run when you install it either - it is, after all, running on the same hardware.
