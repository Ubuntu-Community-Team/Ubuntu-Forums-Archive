### Post by WorMzy on 2010-06-25
Go to System -> Preferences -> Appearance -> Theme, click Customise, then change either your Icons or Controls. I forget which one it is.

There is another way of getting rid of the solid colour, but it escapes me at the moment. It involves copying the theme in question to your home folder and editing a text file somewhere. Maybe someone else will be able to elaborate on that if you don't want to use a different Icon/Control set.
