### Post by yjwong on 2008-09-02
Why not switch to AIGLX when there's already decent support for it in the latest proprietary drivers? (Catalyst 8.8) As of November 2007 (Catalyst 7.11), there has been AIGLX support in every release of the ATI drivers.

The only change that you have to make is in the Compiz wrapper script. Make sure that Composite and AIGLX are enabled in /etc/X11/xorg.conf, then execute Compiz via the command line using "SKIP_CHECKS=true compiz --replace". If it works, modify your session startup via System -> Preferences -> Sessions.
