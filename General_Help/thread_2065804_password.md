### Post by Stonecold1995 on 2012-10-02
Unfortunately, that would completely defeat the purpose of using a secure password.  And no, the screen lock password is the same as your normal user password.

You could encrypt your /home directory or your whole drive with a long password, but use a short user password.  That way you would need to know the long password only when you power on the machine, but while its already on you only need to enter a short user password.

You can never be 100% secure with a user password though.  It's trivial to bypass a user password simply by booting into recovery mode and changing it.  Physical access=root access.

### Post by 2F4U on 2012-10-03
If you are just interested in turning off the screen without locking it, you can turn off the lock feature. The screen then goes black and doesn't ask for a password on resume. However, then everyone can access the machine when you are away.
