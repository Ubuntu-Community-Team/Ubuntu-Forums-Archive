### Post by Impavidus on 2014-03-19
You can use something like US international layout, which is qwerty with additional characters like accented characters accessible with AltGr+character, or you could use something with dead keys. Hitting a dead key followed by a character will put the accent associated with the dead key on the character. Finally you could use the compose key, but that requires three key presses for each accented character, so it may be less convenient if you have a lot of them.
