### Post by crichard on 2010-03-05
```
uname -a
```It'll show your kernel information and also your 32 bit architecture as i686.

one more way to find your architecture, try 
 
```
sudo lshw -C cpu | grep width 
```

To find processor architecture, Simply type

```
arch
```
