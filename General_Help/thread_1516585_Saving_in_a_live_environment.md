### Post by Revolutionary101 on 2010-06-23
There is no way to do it if you are running off of a CD, but you should be able to if you run it off of a USB flash drive.

When you create a bootable flash drive with Startup Disk Creator, you can select to store your documents and files in reserved space. Although I have never tried to save my documents in that reserve space.
