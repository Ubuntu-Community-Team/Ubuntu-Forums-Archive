### Post by davetv on 2012-07-23
I had this on a Windows machine recently. Running "memtest" from a live CD gave me the answer, faulty ram stick!. Even though BIOS showed all memory present, replacing the sticks (had 4 X 512MB in it) one at a time solved the problem.

Also maybe check for a ram seating problem in its socket.
