### Post by mb_webguy on 2008-09-17
I have a Dell Inspiron 1720, and had no problems installing Ubuntu except for the usual minor issue of using ndiswrapper for the wireless card, although that was my fault for "upgrading" to the Wireless-N card from the one I'd originally picked out.

I would assume that the Dell computers that come with Ubuntu pre-installed would have Linux-friendly hardware, but you never know.  Also, Dell is only pre-installing Ubuntu on certain models.  While I wasn't very impressed with them, I had pretty high standards since I was looking for a desktop replacement.  

If you decide to buy a Dell or some other OEM computer that comes with Windows pre-installed, you can get some of your money back if you call the manufacturer before you boot the computer for the first time and tell them that you do not accept the terms of the EULA, and intend to install a different operating system.  I did this when I purchased my Dell, and got a $159 refund on the operating system.  

Here's the relevant bit from the Windows EULA:> By using the software, you accept these terms. If you do not accept them, do not use the software. Instead, return it to the retailer for a refund or credit. If you cannot obtain a refund there, contact Microsoft or the Microsoft affiliate serving your country for information about Microsoft's refund policies. See [www.microsoft.com/worldwide](www.microsoft.com/worldwide). In the United States and Canada, call (800) MICROSOFT or see [www.microsoft.com/info/nareturns.htm](www.microsoft.com/info/nareturns.htm).
