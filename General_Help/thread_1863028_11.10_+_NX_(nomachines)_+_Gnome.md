### Post by ericsonp on 2012-01-04
The problem is that by default nxserver starts gnome is a away that triggers Gnome to start in Unity. To bypass Unity entirely simply edit:

/usr/NX/etc/node.cfg

Comment out this line:
CommandStartGnome = "/etc/X11/Xsession gnome-session"

And replace it with this line:
CommandStartGnome = "/usr/bin/gnome-session --session=gnome-classic"

[http://www.nomachine.com/news-read.php?idnews=355](http://www.nomachine.com/news-read.php?idnews=355)

### Post by Cyber Source on 2012-01-19
Change the /usr/NX/etc/node.cfg file 

from

CommandStartGnome = "/etc/X11/Xsession gnome-session"

to

CommandStartGnome = "/etc/X11/Xsession mate-session"

Now the nx client can just use Gnome like it normally did but it will login to the mate session.
