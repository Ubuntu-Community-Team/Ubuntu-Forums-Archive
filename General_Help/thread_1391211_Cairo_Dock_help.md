### Post by howefield on 2010-01-26
Try right clicking the desktop and select "Create Launcher"

If you use chrome and want it to open chessbase.com eg, fill in the fields with the command looking like this

```
opt/google/chrome/google-chrome %U http://www.chessbase.com
```

Choose your icon.

Then add to Cairo Dock.

Think that would do it.
