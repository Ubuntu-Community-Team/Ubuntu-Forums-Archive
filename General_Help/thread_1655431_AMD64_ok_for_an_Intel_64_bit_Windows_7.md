### Post by BurningSludge on 2010-12-29
It is good for both 32 bit and 64 bit Windows 7. AMD64 is basically the name for a i686 chip that has long mode. The extension was originally made by AMD while Intel was trying to push Itaniums and that is why the name of the architecture is AMD64.
