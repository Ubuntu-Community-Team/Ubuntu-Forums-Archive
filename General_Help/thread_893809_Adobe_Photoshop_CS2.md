### Post by Loaded.len on 2008-08-18
Vbox works. and if you enable bridging to your host nic it's really easy to work between host and guest. I use it all the time, (not the OSS one, though...I use the full one for personal use) 

The only thing wrong with the current version (1.6.4) is that someone at Sun forgot to turn off their default debugging feature and as a result if you don't set VBOX_LOG_DEST=nofile in your environment it will very rapidly consume your HDD. (easy fix for that, though).

As far as VM's go - Vbox is a really good one as far as I'm concerned.
