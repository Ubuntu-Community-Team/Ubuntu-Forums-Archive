### Post by kdwillia on 2008-01-04
Have you tried running the LiveCD from another computer?   Many times the large size download can become corrupted and it will seems as though the CD is a "good" CD, but when it tries to load, things will not work.

If you have access to another machine, test it with the LiveCD and if it does not work as well, then you probably want to try and download the ISO again.
