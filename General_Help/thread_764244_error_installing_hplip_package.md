### Post by ryanhaigh on 2008-04-24
You don't need to download from HPs site, ubuntu provides the drivers in their own repoistories which is ALWAYS preferable as it means the system can update when needed and that any dependancies are installed as well.

To install hplip just go to system>admin>synaptic package manager and search for hplip, on my system there are also documentation and gui packages available. There are many other ways to install this software, this link may be useful: 
[https://wiki.ubuntu.com/InstallingSoftware](https://wiki.ubuntu.com/InstallingSoftware)
