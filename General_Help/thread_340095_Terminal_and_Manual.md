### Post by zerhacke on 2007-01-16
> **guguma said:**
> Hello everybody,
Slackware provided a guide book (manual) on simple terminal useage
You're using the same shell on Ubuntu that you used on Slackware.  All the commands that Slackware instructed you on EXCEPT FOR PACKAGE MANAGEMENT work the same way on Ubuntu.

> Can I find a good and detailed manual on packages and their usages
If you are in Gnome, click on the SYSTEM bar at the top left of your desktop.  Mouseover Administration, which opens a menu.  On that menu click Synaptic Package Manager.

This is your package management tool.  There's a command line version of it too.  All you have to do to find more software for your system and manage packages that are already there is search for what you want.  Once you have that mastered you can move on to adding new repositories with even more software you want... all for free.

Last I checked there's something like 15000 packages to be had from the basic repositories.

> and their dependencies
Here's where you're going to LOVE Ubuntu if you liked Slack.  No more dependancy hell!  Synaptic automatically solves dependency problems for you by finding everything a package needs, downloading it along with the package and installing it all in the proper order for you.

>  and many other things I have yet to learn about LINUX.
This forum is easily the single best resource on the net for running a Ubuntu system.  Ask and ye shall receive.
