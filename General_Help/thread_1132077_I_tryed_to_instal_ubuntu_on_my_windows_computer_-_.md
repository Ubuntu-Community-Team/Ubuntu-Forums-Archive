### Post by mwacky on 2009-04-21
It looks like you need to enable the CD drive to be the startup disk, Windows is trying to start from your hard drive.  The screenshot you gave is from Windows.

Enable CD as start up drive: by pressing F12 to bring up the boot menu right after startup and selecting CD driver.

OR

F2 to enter the system setup and move the CD Drive to the top of the boot order.
