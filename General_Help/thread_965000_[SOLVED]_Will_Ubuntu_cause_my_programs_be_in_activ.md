### Post by drs305 on 2008-10-31
Most of you windows programs will not run on linux out of the box but there are several solutions. There is an app called 'Wine' which allows some, but not all, windows programs to run from within linux. There are also 'virtual machines' such as VMWare and VirtualBox which will let you run windows, and its associated programs, from within linux.

For users who must retain access to windows applications, the easiest method is to 'dual boot'. Most linux distributions will allow you to set up your machine with the ability to start with either windows or linux. Of course, you will have to install the linux system on your machine to do this.

You can also try out ubuntu from within windows. It's called 'wubi' and will allow you to create a file inside windows and run ubuntu as you would any other windows application.

Linux has a large base of applications which do the same things as their windows counterparts, for free and often better. Here is a site which shows many of them:
[Linux Alternative Project]("http://www.linuxalt.com/")
