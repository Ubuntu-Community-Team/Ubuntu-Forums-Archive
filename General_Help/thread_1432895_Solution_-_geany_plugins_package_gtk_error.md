### Post by g3c0 on 2010-03-18
Just wanted to drop by and write solution for error given when installing Geany Plugins package 0.18, more specific, when using command "./configure"...

**ERROR:**
Package gtk+-2.0 was not found in the pkg-config search path.

**SOLUTION:**
 Open Synaptic Package Manager
 Install "libgtk2.0-dev", and with it, all its dependancies.

Probably some other solutions would work aswell, but this one worked fine for me :)
