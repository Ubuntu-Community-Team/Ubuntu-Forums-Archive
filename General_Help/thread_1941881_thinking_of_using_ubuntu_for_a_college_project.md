### Post by winh8r on 2012-03-16
Speak to the IT dept at your college, they might let you use a machine that is already there. No point in going out and spending your money on a machine for a 10 minute talk! Or use a friends laptop, if you are just doing a "no install" demo

It would probably be easier to show people how to boot into Ubuntu using a Live CD , as it is much simpler to download and burn an iso image than it is to do the USB boot.

Also make sure you test it on the machine you are going to use before the big day, you dont want to run into graphics driver issues when everyone is sitting waiting on your demo!

Good Luck.
