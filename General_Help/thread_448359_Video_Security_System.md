### Post by russell.h on 2007-05-19
Check out a program called motion. It runs in the background, is highly configurable (if you have a little time and don't mind messing with some config files), records motion, can host a web server so you can view your cameras over a network (or internet if you set your router up right), and all sorts of good stuff.

Only thing it doesn't have is much of a GUI. Of course you will have to see if it works with your cameras, but I've had good luck with mine.

[http://www.lavrsen.dk/twiki/bin/view/Motion/WebHome](http://www.lavrsen.dk/twiki/bin/view/Motion/WebHome)
