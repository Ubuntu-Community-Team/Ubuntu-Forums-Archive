### Post by efflandt on 2017-05-18
Your problem might be due to doing: sudo nvidia-xconfig
If you do that, it may not properly boot with the default Intel graphics using an incompatible xorg.conf, before you use NVIDIA X Server Settings to switch it to Nvidia (in which case you may need to reboot at that point). Normally there is no need for xorg.conf unless doing something special (like coolbits for gpu overclocking), but I am not sure how xorg.conf would need to be massaged to work for hybrid graphics.
