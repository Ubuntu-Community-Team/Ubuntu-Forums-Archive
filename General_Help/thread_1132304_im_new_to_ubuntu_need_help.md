### Post by lisati on 2009-04-21
I'm not familiar with the game Rohan - the first step is to see if it comes in a Linux-friendly version. Failing that, *some* (but not all) games intended for use with Windows can be run using a program called "[Wine]("https://help.ubuntu.com/community/Wine")"

As a rule, programs intended for use with Windows don't work on Ubuntu without help - it can sometimes be easier to look for Linux-friendly alternatives.
