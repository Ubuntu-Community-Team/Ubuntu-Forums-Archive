### Post by seawolf167 on 2011-02-21
Three options:

[LIST=1]
[*]Software center, then search for WINE and click install
[*]Open a terminal, type "sudo apt-get install wine"
[*]Go to [www.winehq.com](www.winehq.com) and download their .deb file, then run that from your machine
[/LIST]
