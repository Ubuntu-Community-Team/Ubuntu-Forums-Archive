### Post by boddhisatva on 2008-01-26
I've finally found it! System > Preferences > Window behaviour (or whatever it's called, I've got a Polish version). At the bottom you configure the key for moving windows. I changed it from Alt to Super (Win), as Ctrl might also interfere with Blender. Now, while it works immediately in Ubuntu, you need to restart Blender.
Now it's fun time again :)
