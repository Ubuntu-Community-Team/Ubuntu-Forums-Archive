### Post by hvbakel on 2007-05-04
I'm not sure about your situation, but if you connect to the internet in a home setting you probably can't get around setting up the router (modem) that was issued to you by your network provider for port forwarding. If you check the type of modem that you got, you can find detailed instructions on how to set this up on [http://www.portforward.com/](http://www.portforward.com/). I'm using kopete for webcam chats and it works pretty decent (though it doesn't have support for audio yet). Hope this helps.
