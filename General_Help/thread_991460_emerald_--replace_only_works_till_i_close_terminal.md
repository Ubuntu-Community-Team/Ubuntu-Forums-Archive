### Post by Izek on 2008-11-23
In sessions under System > Administration (or Preferences) add a new startup with the command **emerald --replace** and name it Emerald Theme Manager.

Reboot or log out and log back in. Emerald should be taking over now. If compiz is in sessions, remove it, don't need both starting up.
