### Post by jualin on 2008-08-18
For the "minimize/maximize bar" problem you probably had a compiz problem. Press ALT-F2 and run > emerald --replace, if this doesn't work try > metacity --replace. Now for the "Super" key shorcut try Keyboard shorcuts under System, Preferences, and check the command for "Run a Terminal". Good luck!
