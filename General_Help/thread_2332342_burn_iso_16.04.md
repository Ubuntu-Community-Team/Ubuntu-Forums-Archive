### Post by Keith_Helms on 2016-07-30
Are you wanting to create an iso file from files or directories on your system?  k3b and, I believe, brasero will do that as well as standalone tools like genisoimage.

It's a little less than obvious how to do it in k3b.  Create a data project, drag all the files and directories you want to the pane at the bottom right, then click on burn.  When the burn settings window opens, select the checkbox for "create image", then click on the image tab to set the output iso filename and location.
