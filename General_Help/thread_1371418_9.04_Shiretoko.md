### Post by etnlIcarus on 2010-01-21
Adding:
```
.menu-iconic-icon
{
        width: 0px !important;
        height: 0px !important;
}

```

Or:

```
.menu-iconic-icon
{
        display: none !important;
}

```

To your /home/<username>/.mozilla/firefox/<something>.default/chrome/userChrome.css file should do the trick. Just restart firefox.
