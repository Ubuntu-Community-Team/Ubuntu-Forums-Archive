### Post by crazyredwizard on 2011-05-14
I'm running digiKam on gnome. At times I need to open a folder in a file manager. The problem is the default file manager for digiKam is Dolphin. I am used to Nautilus and don't really want to use Dolphin just for this one application. I found a solution and wanted to share it for anyone else with the same issue.

In searching for a way to change which file manager opens from digiKam I came across [http://www.dwasifar.com/?p=841](http://www.dwasifar.com/?p=841). 
-After installing the *systemsettings* package and opening it from the System menu, choose Default Applications> File Manager.
-There is a radio button list. Choose Other and click the ... button to the right of it. 
-Here you will see two *Add* buttons on the right. Click the lower one. 
-I had to browse to where Nautilus was in /usr/bin. 
-Click Ok and then click the Apply button. 

This should do the trick. I hope this helps someone else.
