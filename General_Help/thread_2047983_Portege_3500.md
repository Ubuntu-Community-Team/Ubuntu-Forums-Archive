### Post by cariboo on 2012-08-26
The best way to check if all your hardware will work is to download the latest released version, and try it on your laptop before comitting to an installation. You can get the latest iso [here]("http://cdimage.ubuntu.com/releases/12.04/release/"), then use a program like [imgburn]("http://www.imgburn.com/") to create a bootable Live CD.

Once you have created the Live CD, just boot from it, and try everything out, to make sure it's all fully functional.
