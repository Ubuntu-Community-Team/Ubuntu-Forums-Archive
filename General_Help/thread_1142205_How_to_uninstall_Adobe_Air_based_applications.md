### Post by kostkon on 2009-04-29
You'll need to use *Synaptic* (*System &#8594; Administration &#8594; Synaptic Package Manager*). Use the regular search, *not* the quick-search option to search for it.

The package name will be something like this:
application name, then a dot (.) and then a long string of letters and numbers, e.g.
```
tweetr.546546ioj4564sd56d5654sdfd6456456
```

Hope that helps.
