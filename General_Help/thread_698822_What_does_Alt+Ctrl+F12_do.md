### Post by s3gfault on 2008-02-16
ah, this is a new screen that you may want to run a different x server on.  You can press ctrl-alt-F7 to get back to your original one next time.  Ctrl-alt-F1-F6 would give you a console login prompt.  F7 and up are for X screens
