### Post by codemaniac on 2013-05-02
You can invoke the python interpreter by just opening up a terminal and firing the below command.

```
python
```

If you launch the python interpreter with a script parameter, it begins the execution of the script.

```
python hello.py
```

In order to view the source code of a python script all you need is a editior, there are many graphical editors available as well as the terminal ones. 

```
vim script.py
```

### Post by oldfred on 2013-05-02
You can also install python3 from repository and invoke it with python3 or its specific version.

But do not uninstall current version of python as Ubuntu depends on it.
Ubuntu is converting internals from python2 to python3, but even 13.04 still has some python2 code.

I like using an editor that runs the code also. Then I can easily copy code into it, save it with .py extension and run it.  I use both geany and spyder, both in repositories. You do want to change geany default of tabs to 4 spaces.

Too many choices but many are free so you can try them
