Hi. I recently installed Lubuntu (version 14.04) and have noticed what appears to be a bug.

Sometimes the mouse cursor glitches: it turns into multiple cursors. Essentially, there are four icons, arranged like a square: the top two are smaller versions of a regular cursor, the bottom-left cursor is a hand (such as when you click on a window to drag it), and in the bottom-right there is a small icon of some sort. These four icons all move together as you move the mouse. I've determined that the system still responds to the top-left cursor as usual, but it's pretty distracting having all of these icons floating around.

Here is how I can reproduce this bug on my machine. I open a program (ex.: GIMP), and (this must be done quickly!) before the program has a chance to open, I click on an icon on the desktop and try to move it. When the program (GIMP) finally opens, the cursor glitches in the manner described above.

This cursor glitch can be resolved by logging out and logging back in, or (of course) by restarting.

Does anyone else get this glitch? If so, I can try to create a bug report for it.
