### Post by mark bower on 2009-12-28
o.k. What I was not familiar with, and now I am, is use of Nautilus file manager.  By opening Nautilus on top of IrfanView, a very large number of files in a directory can be seen, especially if font size is reduced in Nautilus.  And then the Nautilus window can be minimized/maximized at will to work in IrfanView, the window can be stretched for convenience, and photo files can easily be dragged into IrfanView.  This all seems to work better than IrfanView alone.
