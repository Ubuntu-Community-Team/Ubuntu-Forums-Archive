### Post by plucky on 2010-04-14
> **Tazz4768 said:**
> i wanted to get away from microsoft for a long time and finally done so I have installed ubuntu os into my computer and I am having the most difficult time getting my printer to work with it the printer i am trying to install is a brother DCP-110c can anybody help

Open **Synaptic Package Manager** and search for **dcp-110c** and you should find ```
brother-cups-wrapper-extra
brother-lpr-drivers-extra
```

Select both and install,will install the drivers for your printer.

For the scanner Go [Here](http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/index.html) and find your scanner driver .deb file and download.Follow the install instructions on the website.

Good luck
