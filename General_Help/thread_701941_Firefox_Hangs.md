### Post by rivera151 on 2008-04-10
The problem with firefox hanging with javascript has been going on for a while.  I was never able to find a solution to this besides disabling java, which seems to interfere with javascript on those pages.  However, as a test, I downloaded Opera, and it seems to handle java and javascript quite gracefully.  I have not had it freeze up on me with java or javascript, and I actually use it more than firefox now, especially since it handles sizing of pages (eg. trying to view a page made for 640x480 at a 1440x900 resolution) much better.  I would ditch firefox alltogether if it wasn't for the Google Toolbar (the Opera Google widget is a pos) and the "search as you type" functionality of the search box, and other little things.

Try Opera, I think you'll be pleased.
