### Post by gadolinio on 2010-04-18
> **chisle said:**
> i installed the demo to try and play, as it turns out my integrated graphics cant play the gamer. i removed all the files for the game using the terminal "rm -r ~/Penumbra* ~/.frictionalgames/" but the game is still listed in the game menu, how do i remove it completely?

Errr you should try to remove it from synaptic, if possible. System--> administration--> synaptic package manager. Type the program's name in the search box; if it appears in the list, right-click, choose uninstall completely, then click apply.

As for the menu, press Alt+F2, type "alacarte", press enter. Go to "games", click the one you want to remove, then "quit". But this won't remove the game itself; only the menu item.
