Hello all,

I've installed lubuntu 12.10 on an ancient laptop, a Toshiba Satellite 1695 CDT, 600mhz Celeron with 192MB RAM and a 4MB ATI RAGE LT PRO AGP 2x.

I was forced to use the alternate installer because the normal install CD hung while booting either the installer or the liveCD. After installing, the system hangs as it did with the LiveCD. I suspect it has something to do with the old graphics chip in it.

The system hangs on the boot screen and switches indefinitely between a black screen, and "Lubuntu 12.10" with four dots below it, as if it were attempting to switch resolution, but failing.

Help?
