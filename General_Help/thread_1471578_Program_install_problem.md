### Post by sir_nasty on 2010-05-03
Here's a Few options the GUI one:
System>Preferences>Main Menu 
Look through those options and often times you can find a program that just didn't get added to the main menu.  From there you can add it.

Option 2: Manual:

Open a terminal type in "which PROGRAMNAME"
Example: which firefox
/usr/bin/firefox

now right click on the desktop, create launcher and put that info into the command field, and give it a name. then it should work.
