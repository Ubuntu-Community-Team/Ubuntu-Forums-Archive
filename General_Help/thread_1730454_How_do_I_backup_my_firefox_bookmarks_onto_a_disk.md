### Post by Rondonjin on 2011-04-16

Any browser will be able to import that

### Post by andrew.46 on 2011-04-16
The quickest way is probably to simply export the bookmarks as an html file:

Bookmarks --> Show All Bookmarks --> Import and Backup --> Export HTML

**Edit:** D'oh!! Beaten by seconds :)

### Post by WorMzy on 2011-04-16
+1 for XMarks.

However, if you want to copy the bookmarks and store the copy locally, just copy places.sqlite from your profile's folder (~/.mozilla/firefox/[COLOR="Red"]<profile name>[/COLOR]/ by default) to wherever you want to keep the backup.

### Post by Nytram on 2011-04-16
In the Firefox Booksmark menu there is a "Manage Bookmarks" or "Show all bookmarks" (depending on version) item, they do the same thing. From there you can import/export your bookmarks. If you choose HTML format it should be compatible with other browsers and versions of Firefox.
