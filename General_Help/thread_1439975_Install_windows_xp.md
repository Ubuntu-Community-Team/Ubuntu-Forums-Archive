### Post by Mark Phelps on 2010-03-27
If you're asking about using Wine to run this in Ubuntu, you're basically out of luck. According to the WineHQ Application Compatibility page below, TaxAct has major problems with the newer versions:

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=1265]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=1265")

Essentially, you'll be wasting your time installing Wine and trying to get this program to work.

I presume you know that you would have to install XP to the empty drive before you could then run TaxAct, right?
