There are some decent Linux books out there for projects and learning in general, but right now desktop environments are changes so abruptly I can't suggest a book covering them.

### Post by doas777 on 2011-07-12
well there are some books on ubuntu itself, that explain the options within the UI itself. I've never tried one however.

personally, I recommend learning the command line and shell scripting initially. that will familiarize you with the surface area under the UI, and let you understand the scripts you see. you will also learn about permissions, linking, and maintenance tasks. 
for that, I recommend these:
[http://www.amazon.com/Harley-Hahns-Student-Guide-Unix/dp/0070254923](http://www.amazon.com/Harley-Hahns-Student-Guide-Unix/dp/0070254923)
[http://www.amazon.com/Essential-System-Administration-Third-Frisch/dp/0596003439/ref=sr_1_1?s=books&ie=UTF8&qid=1310520074&sr=1-1](http://www.amazon.com/Essential-System-Administration-Third-Frisch/dp/0596003439/ref=sr_1_1?s=books&ie=UTF8&qid=1310520074&sr=1-1)

I'm not sure where you are, in terms of technical background, and abiity to self-teach, so I can't gaurentee that these are right for you. they're just the ones I used. be sure to check the reviews and hunt for a sample chapter online.
