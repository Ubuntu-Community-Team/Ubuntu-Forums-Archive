### Post by ankspo71 on 2011-10-24
Hi,
Is this a question about thumbnail/preview images for files?

Thumbnail images can be displayed in Thunar by installing the package called 'tumbler' (and tumbler-plugins-extra for additional thumbnails). My Xubuntu (11.10) wasn't displaying thumbnails in thunar, and now it does after installing 'tumbler' (thumbnails are still not being displayed on my desktop though).

Hope this helps.

PS. I don't think Document Viewer (Evince) can read PNGs, but image viewing Apps like Ristretto and Gthumb can though.
