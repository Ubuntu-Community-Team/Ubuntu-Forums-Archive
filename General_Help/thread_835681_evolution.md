### Post by Golem XIV on 2008-06-20
Mark his messages as junk.  The spam filter is adaptive. i.e. it "learns" that you consider all mails from that address as junk.

The other option is to set up a filter (Edit menu -> Message filters, Incoming, Add) then set it up so that all mails with his e-mail address to move to trash, delete or similar.

### Post by ddales on 2008-06-22
You can't stop someone from sending you mail no matter how much you don't want it without a professional software package in front of your mail server.  

Since I assume you're using Evolution for your personal mail, this is not possible.  However, as Golem suggested, you can take the steps to make sure you don't have to see the mail after it arrives.
