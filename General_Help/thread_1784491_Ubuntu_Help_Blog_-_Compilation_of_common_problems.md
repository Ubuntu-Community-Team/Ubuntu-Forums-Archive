### Post by gpost3 on 2011-06-17
For all Ubuntu and Linux lovers, I have started a project to create a blog and put commonly used troubleshooting entries. When we look for support answers, generally it requires scrouching through the forum such as this one or googling and reading multiple pages. I found very few websites that make a nice compilations of commonly recurring problems that face everyday ubuntuers. 

Here is a blog link:
[http://ubuntuhelpportal.blogspot.com/](http://ubuntuhelpportal.blogspot.com/)

I would also not mind getting some volunteers to help me make a nice compilation on that blog. The idea is to make concise blog entries that takes the user step by step to run the commands or do certain actions to get the job done. Screen shots will be nice. Each bullet point starts with the verb as you will notice. Later on, I will move the content to a custom domain wiki. But let's see if the idea works. If the content gets moved to a custom domain wiki, then each entry will be followed by number of voters for a) works b) doesn't work. This will save time for newbies who think the "how-to" doesn't work simply because they might be missing some steps.
