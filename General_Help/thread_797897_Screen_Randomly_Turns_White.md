### Post by ibuclaw on 2008-05-17
Firstly, you can remove the first two images, there is nothing of interest in them.

Keep the Xorg.0.log picture up, that is the one we should be looking at.

Could you post that file in your next post as an attachment for us?
```
cat /var/log/Xorg.0.log > ~/Desktop/xorg.log
```
A file will be produced on your desktop with the info on.

Also, can you wait around for the bug to happen again?
When the screen goes white, hit "**Ctrl+Alt+Backspace**" to reset X and return to the login screen.

Take note of the time is says on the gdm clock (ie: 22:25).
When you log back in, open up a terminal and type in:
```
cat /var/log/* | grep "22:25" | sort >~/Desktop/crash.log
```
Change the time for the time the crash happened, and post that file up too.

Regards
Iain
