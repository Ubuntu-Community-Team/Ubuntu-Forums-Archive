### Post by dox_drum on 2010-12-13
If you're using Kubuntu (or don't mind to install a lot of libraries), I'd suggest Kile.

If you mind to install KDE libraries, Texmaker is wonderful as well.

If you prefer WYSIWYG, try to install the new Lyx, not the one at the repositories, just the downloadable source. [Here]("ftp://ftp.lyx.org/pub/lyx/devel/lyx-2.0/beta2")
