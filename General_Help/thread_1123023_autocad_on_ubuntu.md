### Post by tommcd on 2009-04-11
This thread has multiple suggestion for autocad in linux:
[http://ubuntuforums.org/showthread.php?t=228890](http://ubuntuforums.org/showthread.php?t=228890)
I don't know squat about autocad though; so I won't be able to help you with any of those. I just know that autocad options exist for linux users.

First, try searching the Ubuntu repos for any of those programs:
> aptitude search program_name

or
> aptitude search program_name?
The ? after the program name performs a wild card search. Without the ? it searches for the exact name. You can search the same way with **apt-cache**. Or use synaptic if you prefer. 
A quick search reveals that **pythoncad** is in the Intrepid repos, for example.
