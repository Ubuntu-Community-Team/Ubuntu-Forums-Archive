### Post by guiverc on 2023-08-04
You can as I understand it, but in time you won't need to need worry about it, as it'll not be counted as an *active* use.

Refer [https://discourse.ubuntu.com/t/ubuntu-pro-faq/34042/28](https://discourse.ubuntu.com/t/ubuntu-pro-faq/34042/28)

and the reply

> you don&#8217;t have to worry about detaching. if you don&#8217;t use a machine anymore it won&#8217;t show up as active
