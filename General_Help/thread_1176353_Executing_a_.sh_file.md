### Post by MichaelSammels on 2009-06-02
Right click it, Properties and make it sure it is set to be run as an executable file. If it is, open terminal and browse to the location of your .sh file. We're going to assume it's in /home/cosmicflux:

```
cosmicflux@ubuntu:~$ ./file.sh
```

Try that code :)
