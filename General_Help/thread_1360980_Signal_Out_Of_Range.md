### Post by cariboo on 2009-12-22
Open an terminal and type:

```
sudo nvidia-xconfig
```

then reboot to see if that solves the problem. If that does work for you then set the resolution to what you want by pressing Alt-F2 and typing:

```
gksudo nvidia-settings
```

to make the settings permanent.
