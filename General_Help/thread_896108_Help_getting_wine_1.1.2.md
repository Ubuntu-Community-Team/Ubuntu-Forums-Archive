### Post by mb_webguy on 2008-08-20
Don't compile source code unless you absolutely have to.

You can get the most recent version of Wine by adding the Wine HQ repositories, as described on the [Wine HQ website]("http://www.winehq.org/site/download-deb").

Be aware that this is the development version.  The latest stable version is 1.0.  Development versions may suffer from regressions (which is when new versions won't run programs that worked fine on earlier versions of Wine).
