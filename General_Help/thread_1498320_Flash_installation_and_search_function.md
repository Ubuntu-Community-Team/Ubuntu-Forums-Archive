### Post by lovinglinux on 2010-05-31
For search use Google site search like this:

[flash install site:ubuntuforums.org](https://www.google.com/search?hl=en&source=hp&q=flash+install+site%3Aubuntuforums.org)

To solve your flash issue, install [FLASH-AID](http://flash-aid-extension.blogspot.com) extension for Firefox. It will remove conflicting plugins and install the proper flash version according to your browser architecture. 

If you don't use Firefox or prefer to fix the problem from command-line, then see the [[COLOR="Sienna"]**Flash Optimization**[/COLOR]](http://firefox-tutorials.blogspot.com/2010/05/flash-optimization.html) section of [Firefox optimization and troubleshooting thread](http://ubuntuforums.org/showthread.php?t=1193567).
