### Post by sekinto on 2008-04-14
Go System>Administration>Synaptic Package Manager
Click search and type myspell, tell it to only search by name.
You can see all the packages there.

Alternatively you could install wajig "sudo apt-get install wajig" and do it from command line:
wajig list-all|grep myspell
