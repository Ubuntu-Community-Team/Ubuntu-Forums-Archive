### Post by ffxr on 2007-04-19
*a very basic guide*

you can extract the tar archive by right clicking on it and selecting extract..

or from the command line ```
 tar -xvf archive_name.tar.gz 
```

then in the directory of extracted files, you'll find a README and/or INSTALL file, open it up and have a look at it.. and look for install instructions...

i think the vmware install routine is script based, something like ```
 ./vmware-install 
``` is run.. (from off the top of my head) 

[most times installing from a tar file, or building from source to give it  a more meaningful name, involves steps like ./configure , make, make install ]

i am only giving you a brief overview here, building vmware-server from source can become a little involved , so have a look around for more complete instruction either on the vmware site, or on the ubuntuforums/ community docs.


good luck..
