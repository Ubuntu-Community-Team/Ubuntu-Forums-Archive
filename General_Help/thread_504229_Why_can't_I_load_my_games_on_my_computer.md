### Post by ramjet_1953 on 2007-07-18
If you are trying to load Windows .exe programs, of course they won't load.

Linux is not Windows and natively, Windows programs will not run on Linux.

Would you expect Macintosh software to run on Windows?

You can possibly get your program to run under Wine, or use a visualization engine like VirtualBox.
