### Post by corncob on 2010-09-11
I'd try to troubleshoot the problem by disconnecting the peripherals if any.  Like if you're using an external mouse, etc, disconnect them and see if the problem persists.  Changing USB hubs fixed a similar problem for me.  Incidentally, at the time, I found that switching terminals (CTRL+ALT+F1), then back (CTRL+ALT+F7) worked faster than CTRL+ALT+BKSP at restoring X.
