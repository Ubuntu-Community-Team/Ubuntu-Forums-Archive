### Post by nothingspecial on 2011-11-18
The download page doesn't know what system you are running. That page has always recommended 32bit because 32bit will work on 64bit but not the other way around.

This is set to change with the next release.

---

### Post by kaldor on 2011-11-18
This is because 64 bit has been a source of issues in the past, which are no longer true. Unless you have specific needs for a 32 bit OS, don't worry about it. It also has a better chance of working on a new user's computer if they are unsure.

Starting on the next release of Ubuntu (12.04 LTS) the 64 bit version will be default and recommended. I have been using 64 bit Linux for a long while now and have noticed absolutely no difference.

Go with 64 bit :)
