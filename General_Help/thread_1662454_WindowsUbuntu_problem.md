### Post by binoplaza on 2011-01-08
Create an entry in your bootloader (grub) that points to sda5. (the partition where your windows is located). 

Would look something like this:
```

title Microsoft Windows Whatever
root (hd0,4)
savedefault
makeactive
chainloader +1

```

If I remember correctly by hard you refer to sda5 by (hd0,4), 0 representst the 'a' and '4' parititon five (starts counting from zero)

### Post by Rubi1200 on 2011-01-08
Okay, so here is what we need to do.

Boot with a LiveCD, choosing to try not install.

Go to Applications > Accessories > Terminal on the live desktop and run the following commands (if you encounter errors report back before proceeding). This also assumes an Internet connection (which you need to do this).

Follow the method to use a chroot in this fantastic guide by drs305:
[http://ubuntuforums.org/showthread.php?t=1581099](http://ubuntuforums.org/showthread.php?t=1581099)

In your case, mount sda1 since this is the Ubuntu partition. You do not have a separate boot partition, so no need to run that command.

If anything is unclear, ask first and I will walk you through it.
