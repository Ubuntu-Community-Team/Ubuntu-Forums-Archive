### Post by col48 on 2022-10-12
I know this isn't really a Ubuntu problem, but this is a forum of experts across the piece, so...

Thunderbird seems to think the Browser is a black hole, not Firefox.  How can I set it straight?

Background:
I have dual boot, currently 22.04 with 16.04.
All was well in my 16.04 setup.  It still is.
I installed 22.04 from scratch overwriting what was previously on that partition.
I copied everything in the 16.04 Thunderbird profile into the 22.04 profile (which of course has a different random-looking name).

Everything works, except that **if I get an email with a link in it, the link no longer works**.
I can choose 'Open the link in browser' and nothing happens.  But I can copy the link (from the same popup menu) and paste it into the Firefox search box and it works fine.  Just a bit annoying to have to do that every time.

I've searched Thunderbird's settings and ....

Having typed that, I've just found the solution!
The solution lies is in *Firefox*'s settings: for some reason - doubtless in the small print of the installation process - it was not set as the default browser (nor was any other browser).  
**As soon as Firefox was set to be the default browser, the problem went away**.

Having got this far, I may as well submit the post - it may save some other hapless idiot some time.
