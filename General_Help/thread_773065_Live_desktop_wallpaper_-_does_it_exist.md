### Post by LeoSolaris on 2008-04-28
Enlightenment has something a lot like that. I have used [OpenGEU]("http://opengeu.intilinux.com/Home.html") and with Enlightenment, they have a few really slick 2D effects, like wallpaper that changes, and it doesn't need a 3D card to do anything but DVDs! It has a rain drop effect that is pretty cool, sorta looks like your monitor is being rained on.

Leo

(Just a note Enlightenment is not out of testing yet, so don't use it for production machines. Meaning if you can't afford to lose it to a crash, don't use Enlightenment yet. That said, Enlightenment is fairly stable now, just not as solid as the other windows managers.)
