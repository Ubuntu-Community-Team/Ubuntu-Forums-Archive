### Post by QIII on 2012-07-04
If you are talking about the "Indicator Plugin", which has the volume control, mail icon and ethernet manager, then:

Right click on the panel.

Panel -> Add New Items 

In the search bar, type "Indicator Plugin"

Select the Indicator Plugin and click Add.
