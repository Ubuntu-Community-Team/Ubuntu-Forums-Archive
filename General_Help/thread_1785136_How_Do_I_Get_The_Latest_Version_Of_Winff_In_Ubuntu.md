### Post by dwigyit on 2011-06-17
Yes. You can either (in order easiest to hardest):

*Find a PPA for the software, which adds a developer-maintained and up-to-date repository to Software Centre and Synaptic. Enter it into the software sources setting window. (Not all software has a PPA though)

*Download a precompiled executable installer from the software's webside, or a compiled zip file which you can then extract to wherever you want (Not all software has one of these though)

*Compile from source code, which is provided for all open-source software, but which can require a fair bit of work and is all done from the terminal.
