Sure. Open a terminal and post this 


```
gksudo gedit /etc/apt/sources.list
```

Then you can edit the file lines that contain 'dapper'

you want comment out by inserting a # sign at the beginning of the line or remove the line completely. Save the file then update.

```
sudo apt-get update && sudo apt-get upgrade 

```
