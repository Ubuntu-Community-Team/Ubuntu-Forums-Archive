### Post by ozorba on 2008-04-26
It is highly likely that either downloaded the Edubuntu .iso file or you have selected to install edubuntu-desktop. The latter one replaces ubuntu settings with edubuntu one. To get your ubuntu settings back execute the following command

sudo apt-get install ubuntu-desktop
