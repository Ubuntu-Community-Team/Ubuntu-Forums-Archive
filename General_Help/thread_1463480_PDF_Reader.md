### Post by claracc on 2010-04-27
In firefox, to use foxit reader as default pdf reader you have to do:
Edit --> preferences --> programs, there you select foxit reader to read pdf documents.

And you would create a launcher in desktop to put in system tray or directly on desktop:

Right click in empty part of desktop, select create a launcher and in this window you fill spaces with the name of foxit reader placed in /usr/bin (I think it would be the dafault directory for the running program). Then every time you want to read a pdf you run foxit reader from the launcher.
