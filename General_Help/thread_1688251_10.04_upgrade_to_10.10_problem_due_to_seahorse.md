### Post by searchfgold6789 on 2011-02-15
ubuntu-desktop is the name of a collection of packages and configuration settings that is compiled into one big package, or meta-package.
And it can easily be installed using Synaptic Package manager (Applications>System). Just search for ubuntu-desktop, mark it for installation, and click apply.
I don't know if ubuntu-desktop is required to upgrade using the upgrade manager, but I know that if you do a clean install it will definitely work.
