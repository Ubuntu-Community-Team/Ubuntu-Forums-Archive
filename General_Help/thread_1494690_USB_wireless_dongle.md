### Post by blueturtl on 2010-06-02
Be a little more spesific... is this a mobile broadband like 3G or something? If that's the case, what you want is a router that accepts 3G dongles and shares their connection either wirelessly or via cables.

If it's a regular WLAN stick that you use to connect, the only way to share that connection is to use the computer with the stick as a hub for the other computers.

### Post by McRat on 2010-06-02
If it's a 3G USB modem, they make 3G hotspots that allow you to plug the dongle into the hotspot, then share the connection via 802.11b/n/n wireless LAN.

Be aware that not all 3G Hotspot routers work with all dongles.  Verify before purchase.
