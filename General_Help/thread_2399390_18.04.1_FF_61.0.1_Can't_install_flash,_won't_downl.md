### Post by Holger_Gehrke on 2018-08-24
There's a package named 'flashplugin-installer' in the multiverse part of the standard repositories. When you install this it gets called as part of it's post-installation-script and downloads and installs the latest flash. If a new version of flash is published, the script gets updated soon after and updates flash automagically.```
sudo apt install flashplugin-installer
```

Holger
