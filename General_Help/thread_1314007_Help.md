### Post by khelben1979 on 2009-11-04
Use:
```
df -h
```
to show your available disk space.

You can use [xterm]("http://en.wikipedia.org/wiki/Xterm") where you insert this code. To free up disk space you'll simply either uninstall applications from the system or manually remove files from your /home/ partition.
