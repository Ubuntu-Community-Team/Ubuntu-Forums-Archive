### Post by bryonak on 2008-10-30
I'm right in the process of upgrading, so I can't judge this yet.

The logout/shutdown applet issue lies with GNOME, not Ubuntu. There have been discussions on the Ubuntu developer mailing lists about whether to write an own unified applet, since many people aren't satisfied with the current one... but then they'd have to maintain and replace it every time GNOME gets upgraded, and distribution specific quirks are often considered objectionable.

AFAIK this is some kind of transition stage, so we'll get a better applet with the next GNOME version. This will probably have to wait until Ubuntu 9.04, though it's quite possible that there will be tools to have a single logout/shutdown menu with 8.10.
