### Post by Hedgehog1 on 2011-05-14
Welcome to the forums!

A '.torrent' file is the little bit of data any bit torrent client needs to begin the downloading of a file.

Normally you will download the the small '.torrent' file using your web browser, then 'open' the '.torrent' file to your bit torrent client (Transmission, Deluge, and so no).

I personally find the download process is easy to watch and understand using the 'Deluge' Bit Torrent client. It is also faster then Transmission.

You can install Deluge (and uninstall Transmission if you like) using the Ubuntu Software Center.
