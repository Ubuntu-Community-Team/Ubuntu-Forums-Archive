### Post by 3Miro on 2010-12-22
Unlike some other operating systems, Ubuntu (and Linux in general) doesn't waste your RAM. The 2GB that you see free are used as cache for the HDD (basically what you are trying to do is already done automatically).

I think Hulu mostly hogs CPU, so RAM will not help.
