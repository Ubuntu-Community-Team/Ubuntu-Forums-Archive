### Post by michy99 on 2009-08-05
You can use a program named LVPM. The webpage is at:

[http://lubi.sourceforge.net/lvpm.html](http://lubi.sourceforge.net/lvpm.html)

Instructions for resizing a virtual disk is near the bottom of the page.

You can also use this program to move your Wubi install to a true separate dual-boot system. You might want to consider doing this.
