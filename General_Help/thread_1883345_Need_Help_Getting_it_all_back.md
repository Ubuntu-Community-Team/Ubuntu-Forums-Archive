### Post by Mark Phelps on 2011-11-19
To install any other OS, the process is basically the same for all:
1) Boot from an Ubuntu Destkop CD
2) Use the Try Ubuntu option
3) When you get to a desktop, use the Disk Utility to remove all the partitions
4) Shutdown the PC
5) Insert the install media for the other OS, boot from that, and install it.

Before you ask, as some folks do, there is not a way to install another OS to the entire drive from inside a running Linux install.
