### Post by az on 2006-12-13
Do you mean that you installed ubuntu and now when you reboot, you are expecting a graphical screen but are being greeted by a text-only login and password prompt?

If this is the case, log in using the name and password you used during the install process and then install the desktop packages.

sudo apt-get install linux-386 ubuntu-desktop

The next time you reboot, you should now have a complete desktop.
