### Post by yancek on 2014-09-01
You will need to provide more clarification.  One doesn't normally "install" a Live CD, it is generally burned as an image to a CD/DVD and then used to either run the system or install the system to a hard drive.  A user can put the iso on a hard drive as a Live CD (generally referred to as a frugal install) and boot it but this will be a read-only filesystem which means no changes can be made to it.  You will need to explain in more detail what you did and how you did it, what software you used, etc.
