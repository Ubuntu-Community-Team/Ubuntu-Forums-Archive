### Post by Aearenda on 2009-06-22
As far as I know you can use DRBL to run diskless Linux PCs, but not Windows. Windows simply isn't designed to be able to run diskless. DRBL with Clonezilla can clone Windows machines, but they have to have a local disk drive.
