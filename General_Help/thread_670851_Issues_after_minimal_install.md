### Post by HermanAB on 2008-01-17
I had a machine like that - got stolen - worked fine with IceWM.  You really should add more RAM though.

I used the VESA display driver and it worked well.

You should have terminals on ctrl-alt-f1 till ctrl-alt-f6 with the graphical interface on ctrl-alt-f7.

To move the icons on the Rox panel, you need to right click on them and unlock them first.

Hope that helps!

Herman
