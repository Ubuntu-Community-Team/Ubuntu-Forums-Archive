### Post by Jlb181 on 2010-05-24
I found a [link]("http://www.facebook.com/sitetour/chat.php") that pointed me to facebook, these instructions are from facebook.  I don't really know where Domain goes, but the rest is pretty simple.  I did put domain in the resource space and I used the jabber protocol since xmpp isn't listed in Empathy.  But this worked great. You need to have an facebook username.  Mine is firstname.lastname.  If you don't know your username you'll need to get it from face book

1. Start your instant message program and locate the menu where you can add a new account (usually in preferences or settings).
2. Create a new account by entering the following info:
Protocol: XMPP or Jabber
Username: user.name
Domain: chat.facebook.com
Jabber ID: [email]user.name@chat.facebook.com[/email]
Password: <your Facebook password>
3. Enter the following server info, as requested:
Port: 5222
Server: chat.facebook.com
Use SSL/TLS: no
Allow Plaintext Authentication: no


[http://www.facebook.com/sitetour/chat.php]("http://www.facebook.com/sitetour/chat.php")     
Once you get there click on _other_ if your not using a program that is named in the list.
