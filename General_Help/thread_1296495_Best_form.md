### Post by t0p on 2009-10-20
Ubuntu and Kubuntu are the main versions.  Ubuntu uses the Gnome desktop environment, Kubuntu uses KDE.  Which one is "best" is a matter of personal preference.

Then there's Xubuntu, which uses the XFCE desktop environment.  It's a little lighter than the main two versions, so might be better on a less capable computer.

There are others too: like Linux Mint, which is Ubuntu with restricted license software included; and many others too numerous for me to remember right now.

Unless you have specific hardware requirements, Ubuntu or Kubuntu are probably the best to start off with.
