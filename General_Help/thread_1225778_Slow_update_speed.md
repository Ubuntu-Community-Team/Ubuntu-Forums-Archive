### Post by Finalfantasykid on 2009-07-28
You can change what server you are downloading from.  For me, the main Ubuntu Server is considerably slower compared to the more local Canada Server.  Also if there was a major software update, it might go a little slower(today, for me at least there was a kernel update, so it might have caused more traffic for the server).

But I would just try a different server, which you can do in the GUI update manager > Settings
