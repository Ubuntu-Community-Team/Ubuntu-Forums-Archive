### Post by xarmian on 2007-07-07
> **bananabob said:**
> It is not that I want the latest software, it is just that this message box keeps coming up every time I start Gaim and it is annoying.

I was getting it too and it was annoying.. I think it's because they converted everything for pidgin so the update checker gets a page it doesn't expect (a 404) and it causes it to give you a popup box.. You can stop it from automatically checking for updates by going to Tools->Plugins, find Guifications->Configure plugin, go to the Advanced tab and uncheck "Check for new releases"..
