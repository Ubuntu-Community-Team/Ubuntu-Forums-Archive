### Post by CatKiller on 2008-11-12
> **JLaw said:**
> What little I know about this machine is that it is a Gateway running Windows ME, has a Intel Pentium 4 CPU processor, **254 MB RAM**, and a 9.3 GB hard drive.

That's not enough to run the live cd. The minimum requirement is 384 MB. You can install with 256 MB from the text-based installer, but performance probably won't be great once it's installed.

I'd suggest trying Xubuntu or installing more memory in the machine. Some more hard drive space might be useful too.
