### Post by coffeecat on 2010-10-10
> **Cammigirl said:**
> Can anybody tell me how to download and install a new theme?

I'm new to ubuntu!

If you download a theme or icon set which is in *tar.gz format, simply open System > Preferences > Appearance, and drag and drop the tar.gz package onto the Appearance window. The theme will be autoinstalled.

But first, have a look at the Bisigi project  website:

[http://www.bisigi-project.org/?lang=en](http://www.bisigi-project.org/?lang=en)

They have a well-designed set of themes with complementary icon sets and wallpapers. There are instructions there for adding Bisigi as a repository so that you can use the package managers to install. I don't know whether they're ready for Maverick yet - Maverick was only released today. Which version of Ubuntu are you running?

By the way - it's best to start your own thread rather then resuscitate a moribund one. The last post here was over a year ago. It's frowned on in this forum because it can perpetuate now-obsolete information.
