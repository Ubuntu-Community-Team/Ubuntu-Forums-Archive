### Post by kpel on 2007-05-05
As a proficient Windows user myself, I can say that being familiar with Windows **will** make things harder.  It takes time to 'unlearn' the Windows stuff.

Without knowing exactly what you plan on doing it hard to recommend things.  I'll start by recommending a few things to make the Linux transition easier.  First, search is your friend.  You'd be amazed at how helpful the community is and at the sheer volume of help material out there.  The Ubuntu forums or [www.google.com/linux](www.google.com/linux) are great places to search from.

Learn the basics of the terminal.  It is very powerful and is often how people give advice, instead of having to describe menu destinations.  Some helpful links for that:

[http://www.linuxcommand.org/learning_the_shell.php](http://www.linuxcommand.org/learning_the_shell.php)
Edit: Found it.  Took me long enough. [http://ubuntuforums.org/showthread.php?t=73885](http://ubuntuforums.org/showthread.php?t=73885)

There's a great guide to the terminal as well on these forums, but I forgot to mark it and now, for the life of me, I can't find it. :oops:   Other than learning the terminal, learn the Linux basics.  The best way to do this is to read the forums, I know I've picked up a **lot** just by reading posts requesting help.  Now back to your original inquiry.


Most of the programs I use came with the Ubuntu distribution.  Here are the things I've added:

[LIST]
[*]Beryl for effects.  Window animation, the desktop cube, etc.
[*]Emerald for window border changes.  This program goes hand-in-hand with Beryl.
[*]Avant Window Manager as a pop-up taskbar.  I removed the lower panel and moved the workspace and trash items to the upper bar.
[*]gnome-look.org.  I've changed the look of Ubuntu significantly with the things I've found there.  You can customize basically every aspect of the look of Ubuntu and that website is a great resource to do it.
[*]Opera web browser.  If you want the Opera vs Firefox debate, do a search.  There's plenty of material out there. :wink: 
[*]Amarok for sound playback.  I plan on looking into Songbird and BMP as well.
[*]K3B for burning.
[*]VLC media player for all my video.  It doesn't play very well with Opera, but I also don't often watch video online that isn't in a flash or java format (youtube, etc).
[/LIST]


Be careful if you start messing around with how Ubuntu looks.  I've missed a great deal of sleep in the last couple of weeks because I keep playing with how it looks and 'feels'. :eek: 

A few more general help links:
[http://doc.gwos.org/index.php/Main_Page](http://doc.gwos.org/index.php/Main_Page)
[http://ubuntuguide.org/wiki/Main_Page](http://ubuntuguide.org/wiki/Main_Page)
[http://ubuntuforums.org/search.php](http://ubuntuforums.org/search.php)   ;) :biggrin:
