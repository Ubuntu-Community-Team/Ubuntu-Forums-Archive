### Post by SJLGamer on 2008-05-12
Hey Guys, so I found a fix for this.
I am using a Nvidia GeFroce 5200 Graphics card, and to fix the resolution issues this is what I did (Should Work With Most, If Not All Nvidia Cards):

1.) Install "NVIDIA X SERVER SETTINGS" From Add/Remove Apps
2.) In the console type ```
sam@ubuntu:~$ sudo nvidia-xconfig

Using X configuration file: "/etc/X11/xorg.conf".
Backed up file '/etc/X11/xorg.conf' as '/etc/X11/xorg.conf.backup'
New X configuration file written to '/etc/X11/xorg.conf'

```
3.) Change Screen Resolution, the full list of resolutions should now be available.
