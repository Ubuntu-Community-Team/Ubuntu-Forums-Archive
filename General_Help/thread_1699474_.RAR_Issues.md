### Post by davidmohammed on 2011-03-05
for example

if you downloaded the file called myrarfilename.rar into your Downloads folder you would use

rar -x ~/Downloads/myrarfilename.rar

or possibly

unrar x ~/Downloads/myrarfilename.rar
