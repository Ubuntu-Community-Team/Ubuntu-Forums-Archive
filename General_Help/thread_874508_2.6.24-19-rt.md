### Post by sdennie on 2008-07-30
It's the realtime kernel, yes.  You don't really need to remove it to use the default kernel because you can just select the -generic kernel via grub at boot time.  However, you can probably remove it with:

```

sudo apt-get remove linux-headers-2.6.24-19-rt linux-headers-rt linux-image-2.6.24-19-rt linux-image-rt linux-ubuntu-modules-2.6.24-19-rt

```

Make sure to see what packages are going to be removed before agreeing to remove them in this case.

### Post by sdennie on 2008-07-30
When you reboot the machine, there should be a short countdown and a message about grub.  Hit Escape when it comes up and choose the -generic kernel instead of the -rt kernel.
