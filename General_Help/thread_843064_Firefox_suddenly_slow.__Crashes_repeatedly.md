### Post by NT4usB on 2008-06-28
Empty the cache.
Have any plug-ins installed? Add new ones recently? Disable them and try it.
Corrupt profile. Create a new profile and try it.```
firefox -p
```

If that works, you can migrate over your bookmarks, etc., until it breaks. then you'll know what's the culprit (or leave it and start fresh.)
