### Post by ian-weisser on 2013-12-29
Try Triaging a few bugs on Launchpad.

Join the team
Read the [Triage Guide]("https://wiki.ubuntu.com/BugSquad")
Clean up your favorite package's queue: Merge duplicates, redirect the misdirected, answer questions, ask for the detailed information needed to duplicate and diagnose the issue, pore through the source code.


Or help test the next release of Ubuntu with your specific hardware and software combination
Bug squad experience will help you to file great bugs that developers can fix quickly


Or help update the documentation on [http://help.ubuntu.com](http://help.ubuntu.com)
Your forum experience will help a lot there.
