The ca is most likely the Canadian version, and by looking at your info. I would say that's the case. My Fn+F7 dims the display and Fn+F8 brightens it. I don't know if regular ubuntu has it or not since it's been a long time since I have used it, since I use Ultimate Edition, but I can disable the touchpad by going to
System > Preferences > Mouse > Touchpad
Mine is set to disable it while typing, enable mouse clicks and edge scrolling are on as well. If you don't have it search Synaptic for gnome-mouse-properties or run
```
sudo apt-get install gnome-mouse-properties
```

Also I can go to
System > Preferences > Keyboard > Layout > Options
and set various keyboard options there.
OR
System > Preferences > Keyboard Shortcuts
and change settings there as well.
OR
Applications > System Tools > Ubuntu Tweak > Shortcut Commands

The last one I'm sure you don't have in regular ubuntu but you can get it here:
[Ubuntu Tweak]("http://forumubuntusoftware.info/viewtopic.php?f=68&t=4587")

The first 2 may, or may not, be in regular ubuntu, or in the ubuntu repos.
gnome-keyboard-properties
AND
gnome-keybinding-properties

IF they are in the ubuntu repos. they can be installed with
```
sudo apt-get install gnome-keyboard-properties gnome-keybinding-properties
```
Like I said I run Ultimate Edition, 2.7 right now so I don't know what is available in ubuntu. Good luck.
