### Post by PaulW2U on 2012-09-04
Lubuntu 12.04 was not an LTS, see [https://wiki.ubuntu.com/Lubuntu](https://wiki.ubuntu.com/Lubuntu). There won't be any updated ISOs released alongside the Ubuntu, Kubuntu  and Xubuntu releases.

However, if you type **lsb_release -a** in  a terminal window I think it should still report 12.04.1. If so, you're fully updated. If not update your system and try again. The 12.04.1 refers to the underlying operating system rather than the Lubuntu desktop.
