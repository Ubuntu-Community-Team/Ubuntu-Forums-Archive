Is there a media player that I can get that will play anything and everything, I mean everything.:)[/QUOTE]
Yes, you can use automatix to install all the needed codecs and over 7 media players.

In terminal put this in. One command at a time. Copy and paste.

wget [http://beerorkid.com/automatix/automatix_5.7-3_i386.deb](http://beerorkid.com/automatix/automatix_5.7-3_i386.deb)
sudo dpkg -i automatix_5.7-3_i386.deb

Once installed, start it from Applications>System Tools>Automatix. Select everything that you want/need. In your case it would be: Media Players, Commonly needed Codecs, and Non-free Aud-DVD Codecs-but also anything else that intrests you.
