### Post by trikster_x on 2008-10-14
First thing to try if you aren't familiar with the Ubuntu OS is using ADD/REMOVE in your applications menu.  If you are familiar with distros similar to Ubuntu, go to system--administration--synaptic package manager.  From there you can remove just about any packages that are installed on your computer.  But you should only attempt these if you are trying to fully remove the package from your computer.  Otherwise, simply go to system--preferences--main menu and you can uncheck the box next to Debian to simply remove it from your menu.  If I were you I would simply remove it from the menu, as Ubuntu actually uses Debian to run some applications.
