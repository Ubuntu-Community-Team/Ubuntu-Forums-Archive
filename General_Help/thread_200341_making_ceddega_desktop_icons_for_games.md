### Post by Lunar_Lamp on 2006-06-20
Well, I don't use cedega, but I have used a bit of wine - so this may help:

right click on desktop and click "create launcher".  Fill in name and info as you want, and select the icon you want.  In 'command' to use a wine application I can just enter 'wine /the/wine/path/to/the/exe'.  For example:

```
wine "C:\Program Files\Mozilla Firefox\firefox.exe"
```
