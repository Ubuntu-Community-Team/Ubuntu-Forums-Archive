### Post by drmrgd on 2011-12-23
If the router's default username and password has been changed, and you don't know it, you might be able to do a hardware reset to put it back to the defaults.  Look around on it for a recessed button that you can press with a paperclip or something like that.  Usually if you press and hold that for a few seconds, the router will reset to factory defaults and you'll be able reset the admin username and password.  

Doing a quick Google search, too, might help you guess at the default admin account credentials.
