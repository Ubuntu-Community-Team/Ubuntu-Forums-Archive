### Post by Titan8990 on 2009-01-07
If anything, Ubuntu is holding you back from learning everything about Linux. I like Ubuntu a lot as a desktop OS and was a good stepping stone on to learning other distros. I had no idea how much Ubuntu sheltered me until that first install of slackware.


Personally, I recommend trying a distro that absolutly forces you to learn the inner-workings of Linux.


Something like [Gentoo](http://www.gentoo.org/) is excellent.

It will:

A) forces you to learn more or less vanilla kernel compilation
B) forces you to learn more about gcc and compilation
C) forces you to optimize your OS for YOUR hardware

I wouldn't recommend wiping Ubuntu right away. Install it on a machine you don't use much because it will not be done and installed in 45min like Ubuntu or similar distros....
