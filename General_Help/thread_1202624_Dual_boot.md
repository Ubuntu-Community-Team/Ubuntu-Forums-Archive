### Post by TeoBigusGeekus on 2009-07-02
Boot up with the live cd and open gnome-partition-manager.
Create a new partition from your free space and then install anything you want to it.
If you install windows, it will kill your grub. Here's a link on how to restore it afterwards.
[https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows]("https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows")
