I am used to right clicking links etc. and using the keyboard shortcut "T", to open the link in a new Tab. This habit led me to press "T" after right-clicking when trying to open bookmarks in a new Tab. The keyboard shortcut for a new Tab in the bookmarks menu is "W". 
On pressing "T" the selected Bookmark disappears completely, nowhere to be found, as if deleted. But the delete shortcut is "D".

### Post by gradysghost on 2010-09-20
The problem is that the shortcut 'T' is for the Cut feature.  You can undo these accidents by right-clicking the bookmarks folder where you accidentally removed a bookmark from and clicking "Paste".  This should restore the bookmark.

In addition to this, there is actually an easier way to open a bookmark in a new tab.  From your Bookmarks menu (or bar), you can middle-click the item to open it in a new tab.  This also goes for any link you see in a web page.

Hope this helps you out.
