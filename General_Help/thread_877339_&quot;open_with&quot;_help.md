### Post by gobbles414 on 2008-08-01
In Firefox, try looking in *Other*. Then in the new window that appears, click on *File System* and then navigate to */usr/bin*. The Azureus launcher is probably in there. If there is more than one program called Azureus, I recommend that you choose the one which has a blue diamond icon.

But IMHO, the easiest and most reliable way in any OS to open a file is to actually save it to your computer's hard drive. Ubuntu is typically rather good about listing the correct program when you right-click on the file and choose *Open With*. If the program you want is not in the list, select *Open with Other Application*. If the program that you want is not in the list, you can use the *Use a custom command* section. To find the necessary command go *System => Preferences => Main Menu*, navigate to the correct launcher, and then right-click on it and choose *Properties*. Paste whatever is in the *Command* field into the *Use a custom command* section; chances are good that this is the correct command.
