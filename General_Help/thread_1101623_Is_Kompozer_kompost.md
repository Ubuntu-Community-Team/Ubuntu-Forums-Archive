### Post by Herman on 2009-03-20
I have the same problem. 
Kompozer is working well for me in my Ubuntu Hardy Heron installation, but  I can't use Kompozer in Intrepid Ibex or Jaunty Jackalope, it crashes as soon as I try to do anything with tables.

 I'm still using Kompozer 0.7.10, which was released way back in September 2007, almost two years ago.  That seems to be the most recent version of Kompozer we have available in the Ubuntu repositories at this time.

Take a look here, this explains our problem,  [WYSIFA Forum Forum Index]("http://wysifauthoring.informe.com/forum/index.php") ->   -> [KompoZer Development]("http://wysifauthoring.informe.com/forum/kompozer-development-df11.html") ->-> [KompoZer 0.8 progress]("http://wysifauthoring.informe.com/forum/kompozer-0-8-progress-dt1940.html").

:D If you want, you can:

[LIST=1]
[*]Download Kompozer 0.8 alpha version 20090206, the file is called kompozer-20090206 .tar.gz
[*]If it landed in your desktop, copy and paste it to some other directory where you like.
[*]to back up you old kompozer settings, run in terminal: $ cp -R .kompozer .kompozer.bak
[*]Right-click on kompozer-20090206 .tar.gz and click 'Extract Here'. A directory called kompozer should appear.
[*]Open the directory named kompozer and look for a script inside it which is also called kompozer.
[*]Right-click on the kompozer script.
[*]Under 'Do you want to run "kompozer" or display its contents? , click on the first button 'Run in Terminal
[*]Kompozer 0.8 alpha version 20090206 should open, try it out.
[/LIST]
Does it work for you?
Do you like it? 
Does it do everything you need?

If you like it, you can add it to your 'Applications' menu.

[LIST=1]
[*]Right-click on 'Applications' and click 'Edit Menus'
[*]When the 'Main Menu' opens, click 'Internet' in the left-hand pane
[*]To the right of the right-hand pane, click the '+ New Item' button
[*]'Create Launcher' should apear, in the name feild: Kompozer 0.8 Alpha or whatever you like
[*]Hit the 'Browse' button and browse to your kompozer script (the one you started kompozer with earlier)
[*]Click Okay
[/LIST]
According to the latest post in the thread [KompoZer 0.8 progress]("http://wysifauthoring.informe.com/forum/kompozer-0-8-progress-dt1940.html"), an alpha2 version is close.

Regards, Herman :D

NOTE: This works for me in i386 but does not seem to work in amd64 computers at all
