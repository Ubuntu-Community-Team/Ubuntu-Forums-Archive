### Post by editrix on 2008-11-06
baobab?

I found it by using the "apropos" command, i.e. 

```
$ apropos usage

baobab (1)           - A graphical tool to analyse disk usage
df (1)               - report file system disk space usage
du (1)               - estimate file space usage
fsview (1)           - a filesystem viewer that graphically shows disk usage
pod2usage (1)        - print usage messages from embedded pod docs in files
time (1)             - run programs and summarize system resource usage
wwwoffle-audit-usage (8) - displays information about wwwoffle's state

```
