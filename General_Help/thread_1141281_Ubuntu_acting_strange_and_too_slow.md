### Post by ingeva on 2009-04-28
I have recently discovered two cases where Ubuntu became extremely slow, or even froze completely.

1: I copied files from an NTFS file system where some or all files were compressed, Windows style. Ubuntu 9.04 becomes very slow or even nearly halts, while I don't think 8.10 did, and 8.04 definitely did not.

2: I have two routers (and temporarily two internet connections). On one computer the network cable was connected to one, and the wireless interface to the other. The two routers have the same gateway address, and this may have created problems for the software, because all internet activity was extremely slow, causing many timeouts. As soon as I disabled the wireless, the problem disappeared.
