### Post by stchman on 2011-02-10
Hello all.

I installed MM on my Toshiba laptop last night and discovered the new partitioning options during install on the LiveCD.

Apparently Canonical decided to get rid of the install on the largest continuous free space option!!!  This is not good.

No fear, you use the alternate CD from here:

[http://releases.ubuntu.com/maverick/](http://releases.ubuntu.com/maverick/)

There is an option in the install to use largest continuous free space.  The only drawback from the alternate CD is it takes FOREVER to install the OS.  I don't know why Ubuntu removed this feature from the installer.

### Post by Mark Phelps on 2011-02-11
> **stchman said:**
> ...I don't know why Ubuntu removed this feature from the installer.

Join an ever-growing bunch of us that wonder the same thing!

This was probably the most-used installation option by folks coming over from MS Windows who wanted to experiment with Ubuntu in a traditional dual-boot environment.

And NOW, not only is that option removed, when they have to resort to manual partitioning, unless they're very careful, they will accidentally wipe out their MS Windows install!

Personally, I think this was a very BAD decision -- and it goes to point out that the decision makers have no understanding of the reality of the dual-boot world.
