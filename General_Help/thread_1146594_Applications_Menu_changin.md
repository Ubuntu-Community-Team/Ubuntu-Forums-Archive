### Post by thefun25 on 2009-05-02
How do I go about changing where the folders in my Applications menus are.

For example I installed a game called regnum which created its own whole new folder in Applications. How do I make it just a link in the Games menu?

---

### Post by VCoolio on 2009-05-02
Right click on application > edit menus. There you can switch off or even delete the showing of the folder, and add the game to games. Make sure to use the right command there. You can right click on the current launcher and add to desktop, then right click that launcher on your desktop to check the command that is used and that you have to fill in when you add it to the games folder.
