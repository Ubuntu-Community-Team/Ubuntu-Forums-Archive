### Post by Capta!nCrude on 2011-01-20
I'm using Evolution with multiple Gmail accounts, and it's working flawlessly. However it can be a bit tricky to set up.

This is how I have mine set-up.

Recieving

Servertype: IMAP
Server: imap.gmail.com
[email]yourusername@gmail.com[/email]
Use secure connection: SSL

Send emails

Servertype: SMTP
Server: smtp.gmail.com
Use secure connection: TLS
Username: [email]yourusername@gmail.com[/email]

Hope it helps

Cheers!

### Post by dodo3773 on 2011-01-20
Here are the settings that I use / what it looks like:


Server: smtp.gmail.com
Server requires authentication     box is checked
Use Secure Connection: SSL encryption
Type: PLAIN          Check for Supported types
Username: [email]emailaddress@gmail.com[/email]
Remember Password       box is checked



I have never run into any problems. I hope that helps.
