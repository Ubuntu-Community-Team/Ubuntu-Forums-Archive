### Post by veddox on 2015-12-05
Hi tripper, a few things for you to think about:

a) Lubuntu 10.04 is no longer supported. Have you considered upgrading to a newer version? If newer versions are too heavy for your machine, you could set up an Ubuntu minimal install.

b) Running old browsers is inherently dangerous due to unpatched security holes that can easily be exploited by malicious websites. If you need a light modern browser, check out Midori or look for other options (if you want ultralight, go for w3m or lynx - browsing the web in a terminal ;-) ).

### Post by Bucky Ball on 2015-12-06
> **tripper said:**
> Rather fun running vintage computers!!

If you do it safely, sure. If not ...

You are using an obsolete version of Ubuntu that can't be updated and hasn't had security updates for months. On top of that, you are attempting to install an obsolete web browser. Frankly, that's called a sitting duck by anyone that has the inclination to smack your system.

Switch off wireless, unplug your ethernet cable, and have fun! Which is not much of an answer to your question of how to use an obsolete web browser. The simple answer to that is don't, unless you're not bothered about the security consequences of your private data (and I mean email address and address book, personal files, bank account details, the list goes on) being shared 'in the wild'. 

Dumping a keystroke logger on a vulnerable machine can be a key to a goldmine, perhaps literally, of personal information about you for the hacker.

PS: Yes, the latest version of Firefox is not designed to run on something that is no longer used. It is expected that it would run poorly. Try and install a supported release, Lubuntu 14.04 or a minimal install, as suggested. We can't give you much/any support with end-of-life software.

### Post by Sweet_Baby_Jamie on 2015-12-06
Alot of the "lightweight" distros use [Seamonkey]("http://seamonkeyproject.org") in place of Firefox and Thunderbird.  Seamonkey is a Mozilla project and looks and acts like older Firefox and Thunderbird, and is alot less resource-hungry than FF and TB.  It is not in the repositories but look at the Ubuntuzilla section here on the forums for how to install it.  It is the default internet suite in [LXLE]("http://lxle.net"), which is a cool Lubuntu respin for older computers.
