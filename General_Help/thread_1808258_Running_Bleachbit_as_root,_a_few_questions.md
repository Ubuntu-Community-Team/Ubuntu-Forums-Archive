### Post by Jouke74 on 2011-07-20
You must see the root and your account as separate entities. There could be situations where root has no access to your home folder and you for certain have no access to the root folders.

As a user Bleachbit will clean your home directory of temporary data, depending on what you check for cleaning.

As root it will clean the root and system directories, including synaptic cache, languages you don't use etc. You can select different options for root and user.

End yes, if used in a wrong way it can delete data that has value to you. E.g. files with the extension .bak.
