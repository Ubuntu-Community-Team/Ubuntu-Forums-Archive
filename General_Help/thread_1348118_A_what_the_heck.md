### Post by MooPi on 2009-12-06
This is a hardware issue. You can't install to one system and then pull the hard drive out and plug into a totally different system and expect things to work. Differing chipsets , graphics, memory speed and so forth. If systems were identical then there shouldn't be a problem. Do yourself a favor and re-install on the faster system and see the boot time diminish.
