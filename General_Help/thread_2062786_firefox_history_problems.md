### Post by Frogs Hair on 2012-09-25
If there is a problem with the profile deleting .mozilla in home hidden folders will take care of it but export the bookmarks first for later use. 

A new folder will be created when you start Firefox and you will need to re-install your extensions again if using any.

Re-installing won't help because the problem is likely with profile folder which is not replaced during the re-installation.
