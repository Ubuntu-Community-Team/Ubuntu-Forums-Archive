### Post by mb_webguy on 2008-08-20
Actually, I think the casual user can transition more easily from Windows to Linux than a power user, because we're more likely to have software we absolutely must have, which won't necessarily run under Linux.  A casual user, on the other hand, primarily uses the standard software like a web browser, word processor, etc.  Also, since Ubuntu automatically organizes the Applications menu by function (rather than as in Windows, where programs are sort of haphazardly tossed into the Programs menu, usually by the developer's name), users can find what they're looking for that much more easily, which makes the transition even easier.

My suggestion is to make a list of all of the Windows software that you and your wife use, and then write out to the side the Linux equivalents.  If you're left with any without easy Linux equivalents, check to see if they'll run under Wine.  If so, then you can probably do away with Windows completely.  If not, you probably want to set up a dual-boot system, at least until you find a suitable replacement for that necessary software.

Now that you have a list of software equivalents, you can keep that next to the computer as a reference, so you know that if you (or more likely your wife) want to do something you'd normally do with MS Office, you'd use OpenOffice.org, etc.  To make things even more painless, you could install some Windows software using Wine to act as a sort of security blanket.  MS Office 2003 will work just fine under the more recent versions of Wine.
