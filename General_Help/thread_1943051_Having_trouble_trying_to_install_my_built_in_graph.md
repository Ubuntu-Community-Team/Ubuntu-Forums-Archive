### Post by Mark Phelps on 2012-03-19
Linux is not Windows, meaning, you don't start hunting around for drivers soon after you install. Linux does a hardware scan during install and loads the proper drivers for you.  If you're seeing a desktop, not a command line, then you already have the proper default graphics drivers installed.
