### Post by SweetAurora on 2013-08-21
No, Ipod 6th Gen's are not Linux compatible. As far as I'm aware, the hashes used to encrypt Ipod's post 4th Gen haven't been cracked yet so only Itunes can be used to sync them as it is the only player availible that holds the hash key. Sorry. BTW, Itunes will not run in WINE either.
