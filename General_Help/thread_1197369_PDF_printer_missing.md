### Post by Stainesy on 2009-06-26
Select print, a window displays the list of printers, in my case 'MP240series' and 'Print to file', select 'Print to file', the Output format defaults to Postscript but you can select PDF.

### Post by ad_267 on 2009-06-26
Install the cups-pdf package. It's no longer included by default. If you're using OpenOffice you can go to File - Export PDF.
