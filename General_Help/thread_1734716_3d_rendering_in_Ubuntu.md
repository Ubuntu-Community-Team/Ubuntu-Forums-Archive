### Post by ragtag on 2011-04-20
If you're looking for any kind of scene description language supporting rendering, and not just Pov-Ray compatible, I believe there are several.

Aqsis - is based on Pixar's RenderMan, and uses rib's to describe the scene. 
YafaRay - uses XML based scene description files.
LuxRender - uses a format derived from RenderMan, but with a few differences. 

There's probably some more options out there. :)
