### Post by grahammechanical on 2011-05-31
If you know what language the document was written in then you can install that language. There is a utility called Language Support. On 11.04 you find it under System Settings. On 10.10 you find it under System>Preferences or System>Administration.

In Openoffice you can go to Tools>Options>Language Settings where you can change the default language and set it for the current document only.

Also in OpenOffice under Tools>Language>For Selection>More you can change the language for the selection.
