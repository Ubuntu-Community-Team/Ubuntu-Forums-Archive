Hey.. thank you... That didn't work for me, but by following other bugs in the same "line" and such, I managed to find the fix.  

Thank you very much for the lead.

For others with the same problem, this is where I found the fix:
[https://bugs.launchpad.net/ubuntu/+source/linux/+bug/264789]("https://bugs.launchpad.net/ubuntu/+source/linux/+bug/264789")

This is the quote from the posting:

"...according to Grzegorz in [https://bugs.launchpad.net/ubuntu/+source/linux/+bug/264789/comments/58](https://bugs.launchpad.net/ubuntu/+source/linux/+bug/264789/comments/58) all that is needed is one tiny change, viz:
edit /etc/udev/rules.d/60-persistent-storage.rules
Change line 59
from
IMPORT{program}="vol_id --export $tempnode"
to
ENV{DEVTYPE}=="partition", IMPORT{program}="vol_id --export $tempnode"
i.e. add
ENV{DEVTYPE}=="partition",<space>
to the start of the line. That's all..."

After several months of trying to find the problem, this finally worked for me.
