### Post by mgmiller on 2009-03-20
There is a very high probability that you won't need to install any drivers to get everything on your mobo working.  The vast majority of this type of hardware is supported natively by the Linux kernel.  I have done over a dozen Ubuntu installs on different mobos and I have never needed to install a driver to get things working.  

If the board is very new, it might need to wait for the next version of Ubuntu.  I did have a minor sound issue with an Asus M3N78 PRO mobo that had a high definition sound codec built into it that did not work correctly in 8.04, but has worked perfectly since the upgrade to 8.10.

So, as the last poster suggested, try booting it off the Live CD and see how it goes.
