### Post by zippy028 on 2007-01-18
The first thing I would try is going to ---> main menu/accessories/alacarte-menu- editor/   Select games and see if the games you installed are their but unchecked. If they are not their you can add a new entry ----> taskbar /file/new entry. Fill in the dialog box for the prog name and launch command and you should have a new menu entry.
