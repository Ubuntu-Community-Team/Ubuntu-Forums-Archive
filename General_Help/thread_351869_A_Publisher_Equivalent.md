### Post by golem3 on 2007-02-02
I run Ubuntu, and I need to make business cards and print them out on Avery preformatted/perforated paper. I have done a lot of googling, but could not find any program for Linux that would do this; basically, I am looking for an MS Publisher type program.

Any ideas?

### Post by Marsolin on 2007-02-02
[gLabels]("http://linuxappfinder.com/package/glabels"), [KBarcode]("http://linuxappfinder.com/package/kbarcode"), [Scribus]("http://linuxappfinder.com/package/scribus"), and [VivaDesigner]("http://linuxappfinder.com/package/vivadesigner") are all options.

[PageStream]("http://linuxappfinder.com/package/pagestream") is a non-free option.

Scribus is one of the premier open source desktop publishing programs.

---

### Post by WW on 2007-02-02
It looks like **glabels** is exactly what you need. It has a large set of predefined Avery label and business card formats.

It is in the ubuntu repository.  You can install it with Synaptic, or in a terminal:
>  $ sudo apt-get install glabels
