### Post by manfer on 2008-07-02
An NFO file is not more that an ASCII text file. If you only need to read the information on it you can open it with any text editor.

If you want to see the artistic view of the NFO file then it is easier installing a viewer for those files. Here is the link for one called NFO Viewer:

[http://home.gna.org/nfoview/](http://home.gna.org/nfoview/)

It is not packaged for ubuntu so you have to read instructions to install it.
