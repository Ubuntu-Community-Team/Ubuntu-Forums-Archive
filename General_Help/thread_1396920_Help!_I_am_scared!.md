### Post by oshunluvr on 2010-02-02
I have been linux only for 10 years - do it and don't look back! The only reason to stay with windows is if you are required to use a particular ap that will only run in windows and there really aren't a whole lot of those. I use winXP in a virtualbox machine for the two things that I need win for.

My advice:
1) Be patient! - you are moving to an OS that RESPECTS and REQUIRES your input, unlike MS products that pretend they should have more control over your machine than you. This means you will have a lot of learning to do but in the end you will have a system that does what YOU want the way YOU want it. This includes choosing which distro, which desktop manager, and many other aspects of the system.

2) Start by reading some beginner forums and helpful articles for noob's (slang for new/beginner). In fact I would suggest you might start with PCLINUXOS rather than Ubuntu because it's geared toward simple and stable. They have great forums over there also, but a lot of rules you need to follow (in a good way IMO). 

Also IMO: Gnome seems easier at the start than KDE but a lot of us like the more configurable aspects of KDE. There are other desktop managers but Gnome and KDE are probably 90% of users so you'll find more help and information.

3) Take some time to learn about GRUB and how to use and fix it. GRUB is the most commonly used boot loader for linux (and can even boot your windows too). Messing up your boot loader is one of the most common early mistakes causing #4 below.

4) Don't be surprised if you bork (make unbootable) your first or even fifth install. Many distros have bootable CD's so you can try them out and test your hardware without worry. These run slow so don't let the speed turn you away.

My suggestions:
1) If you have a clean hard disk - let the installer use it's default settings rather than getting complicated. Mess with it until you break it (hopefully not!) and then try again. Linux only takes a few minutes to install, not half a day like windows.

If you need to preserve data on your disk: Start by making 1,2, or 3 partitions for linux installs. Unlike windows, linux can install and boot from a logical partition anywhere on your hard drive. I've never had a linux install take more than 12-14 gigabytes for EVERYTHING so I'm sure you have enough hard drive space. If you need help learning about partitioning look up GPARTED on the net and read.

2) Then install 1 or 2 distros to play with. Be sure you install to the new partitions to save your other data. This is where learning about GRUB will come in because GRUB will let you select which OS to boot so you can play around.

3) Once you've decided which distro and desktop to stick with for awhile, SLOWLY modify things and take some time to read about each facet of what you're attempting before you dive too far in. Keeping a basic linux install on another partition or a bootable CD handy will allow you to repair your mistakes and get on the internet to get help without wiping everything out and reinstalling.

Some other tips:
1) First order of business once you've booted up is to get your hardware all working properly. Do this before you start installing a ton of apps and playing with settings. This is especially true of your video card. A lot will depend on having these things right - video, audio, mouse/ keyboard, hard drive access and so on.

2) As soon as you are comfortable enough to do it - move your "home" to a separate partition from your install. Some distros will do this on their own some won't. This will preserve your personal files and settings in the event of a total crash or re-install.

3) I said it before but: DO a LOT of research about your issues before you post on forums. Most of us would like to know that you've at least looked into a problem before you ask for help. There are thousands of other linux users with the same issue you're having and you might find your solution waiting for you.

Most of all: Have fun - getting a linux system flying with cool graphics and no virus bugs and no blue screen of death can be very rewarding AND you can gloat about how you are free from all the MS costs and other problems. Freedom (in speech and beer) is good!
