### Post by Suzan on 2007-01-29
Nothing. Envy does install the nvidia official driver.

From the [envy website](http://albertomilone.com/projects.html):

> It is a command line application (written in Python) which automates the installation of both the ATI and the Nvidia proprietary driver on Ubuntu (Dapper and Edgy). Envy downloads the driver which your graphic card model needs, installs all the compilers, etc., sets up the Xserver configuration file for you and makes sure that the driver integrates nicely with Ubuntu's restricted modules.

So envy is downloading the official nvidia-driver an installs it for you.
