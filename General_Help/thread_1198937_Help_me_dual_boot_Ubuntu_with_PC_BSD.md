### Post by jchiar on 2009-08-14
I had a similar error, I found that GRUB really did not work. Then I used a guide from PC-3SD and used a walk-through there.  It is do-able, it is not real easy, but is is possible. First rule like always, back up any and all data. 
That is like the 2nd rule of Computing. 
Load it on, then 'slice' it in. You can use GRUB for that or the Lilo. GRUB is simplified and better at cross platform systems. It seems.
A few questions prevail, is this a Linux/BSD slice/image or is Windows included also, that makes a rather large difference in the Boot Options.
I am still playing with PC-3SD and fine tuning it. The boot thing was a work around, 50 lines of code or so.
Most all this info can be found on the PCBSD forums.
With Galleon it is easier than with 7.0.
