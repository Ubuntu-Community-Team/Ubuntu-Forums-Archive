When I first switch to ubuntu at that time I use to store file, movies and other things on desktop, now I have deleted those files but still when go through command line to desktop and "ls" it shows me almost all those files with ~ sign at the end.
Are those files are still in my system? 
if not why it is still showing it? 
and how to delete those files?

### Post by _khAttAm_ on 2009-11-09
Yes they are the files in your system.
They are generally used by editors to store backups of edited files incase you need them. Open up Desktop (Places > Desktop) or any directory you want to delete those files from and then view>show hidden files. you will be able to see them and hence delete them.
