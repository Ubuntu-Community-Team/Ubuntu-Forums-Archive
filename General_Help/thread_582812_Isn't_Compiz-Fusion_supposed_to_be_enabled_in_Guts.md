### Post by Page on 2007-10-20
It's in Preferences > Appearance > Visual Effects.

At first boot, compiz seems to be disabled by default; you have to enable it. (good thing, too... it's been giving me a lot of trouble) From then on, it should start automatically. 


For custom settings, install compizconfig-settings-manager.
