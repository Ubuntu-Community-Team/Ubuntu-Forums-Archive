### Post by Olav on 2017-10-10
> **jsjpandey said:**
> My computer is very slow due to low ram, and poor processor. Thus I am looking for lightweight PHP editor, which can help in editing file. I have tried many editors like atom and some other (most of their name I cannot remember).

It depends on personal preferences and how much you expect your editor to do for you. For most editing tasks, even on my relatively fast desktop and laptop, I like Geany. It is a capable editor with a plugin system and it has syntax highlighting and code completion for PHP, but not much else. On servers without a GUI I often use the editor that is built in to Midnight Commander.

I do most of my "serious" work in Eclipse. I wouldn't recommend it in your situation.

### Post by SeijiSensei on 2017-10-10
I write all my code in an ordinary text editor, the Emacs clone called [jed]("http://www.jedsoft.org/jed/").  It has excellent syntax highlighting which is all I really need.  I know the major Emacs keyboard shortcuts by heart, but there is also a menu system for the uninitiated.

I started programming by writing FORTRAN code and punching commands into Hollerith cards.  Every time I look at those IDE's I wonder why I would need all that overhead.
