If you are using Compiz install the Compiz Fusion Icon (It's in the software center) and first use it to reload your window manger and if that doesn't work try switching back to Metacity. 

Once the Icon is installed open it 
Applications > System Tools > Compiz Fusion Icon
It should then be in the Notification Area then use it by right clicking on it.

### Post by Enigmapond on 2011-04-15
I don't think this is Compiz related if I understand properly however, installing the icon is not a bad thing. Do you mean the Application menu or the Window list? If so both od these can be added back to the panel by right-clicking the panel and choosing "Add to Panel". Then just scroll down to Window List and add it back.
