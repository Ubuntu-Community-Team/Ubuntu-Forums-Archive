### Post by &amp;KyT$0P# on 2016-07-13
May or may not be related but "Hola" was also the name of a botnet IIRC, so make sure your machine isn't infected with malware?  If it's still basically brand new install personally I'd just re-format the whole HDD and re-install the system, but this time don't add any 3rd-party software or software sources..
(Obviously, **make sure you have all important data backed up** before proceeding!)


EDIT

Optionally can reboot between software installations if there was a reboot anywhere before.  This could help isolate the exact software causing this problem.
