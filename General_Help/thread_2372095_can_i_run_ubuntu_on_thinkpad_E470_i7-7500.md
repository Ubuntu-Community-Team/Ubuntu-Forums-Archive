### Post by Bucky Ball on 2017-09-21
Welcome. I see absolutely no reason why not. Maybe someone else can. That video card is known to work with Ubuntu. Your other main consideration is the wireless card and you don't mention what that is. Chances are that should work, too.

Hard drives, RAM, CPUs, batteries and and all that are non-issues. Generic.

Tip: pays to do some digging, not just for hardware officially certified by Ubuntu/Canonical (as those lists are often old and outdated it seems), but for user experiences before posting. [The last post here]("https://askubuntu.com/questions/923646/ubuntu-16-04-on-lenovo-thinkpad-e470") tells you pretty much all you need to know about Ubuntu on that machine. 

[Ubuntu 16.04.3 LTS]("https://www.ubuntu.com/download/desktop"), the release the poster in the forum link is using, is the latest LTS release so install it and see what happens for you. Seems like it all worked out of the box for that user except for mouse scrolling. They have provided a link to fix that in their post.
