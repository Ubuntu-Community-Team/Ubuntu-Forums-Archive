### Post by Thanoulis on 2008-03-15
To install KDE, you must install the whole package
In a terminal, write:
```
sudo apt-get install kubuntu-desktop
```

Then reboot or press Ctrl+Alt+Backspace. In the Log in screen, open the menu and select a different session. There MUST be KDE within the choices.
