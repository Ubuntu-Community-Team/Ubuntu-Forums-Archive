### Post by mutantapple on 2006-11-18
First to answer your question what you need to do is, goto:

STEP 1. **System -> Administration -> Synaptic Package Manager**

[B][I]then once the application loads, hit

STEP 2. **Settings -> Repositories**

then

STEP 3. *Check off the unmarked check boxes for*:

       *Community maintained Open Source Software (universe)*
and
       *Software restricted by copyright or legal issues (multiverse)*

check boxes. This will enable the Synaptic Package Manager to not only search the default server for that specific application but also other available file "repositories" for other available Linux based apps.

**THINGS TO CONSIDER:**

Due to the fact that Linux is an opensource development project that anyone can theoretically contribute to, there are many files and apps available for everyone to try and use. One major thing to look into for newbies (like me) is to review basic terminal command functions.

Unlike Windows where you need to manually search the web though a web browser like Internet Explorer for a particular program, then download it, and then install it, Linux if I'm not mistaken, you can just type into the console aka. the terminal, aka. the command line located here:

**Applications -> Accessories -> Terminal**

you can just type the application retrieval command:

***sudo apt-get install*** (then whatever the name of the application is in here)

Please correct me if I have used the wrong command; and it should automatically search for and install that specific application, or if you're new, I'd suggest using the Synaptic Package Manager (aka. SPM) to search for and install your various apps.

**Other similar easy program/package installer application include:**

Easy Ubuntu[/I][/B] (i think) and the recent ***Automatix2*** (search and install from SPM) which will get you your basic drivers, codecs, handy-dandy applications and the like to get you kick started.

Learning the fundamentals of Linux is fairly easy, but one needs to understand that the concept and functionality of the system is different from Windows. For example instead of a "Program Files" in your C:/ drive directory, the so-called "root" directory in Linux is just called "/home" or "ubuntu" if you've still havn't installed the OS onto your hard disk yet. Remember that "Applications" is like "Start menu, Programs to view you're currently installed programs (aka. apps), "Places" would be like the "My Computer" tab in "Start menu, My Computer," and "System" is like your control panel, like in Windows. Once you get the overall structure and how the Linux OS flows, it's very easy (not to mention more logically efficient) than Windows.

Try "googling" "10 Top Linux Apps" and see what you find. And feel free to post and ask questions on forums and the like.



Anyways, I hope this helps. :KS
