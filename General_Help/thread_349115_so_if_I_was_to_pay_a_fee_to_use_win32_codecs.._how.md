### Post by majoridiot on 2007-01-29
although it's sketchy (due to the abstract and draconian MS EULA), there is a valid argument
that if you own a legitimate copy of XP, then you have the right to use the win32 codecs, as
well as any windows .dll files, any way you like- as long as is does not entail redistributing it
or otherwise profiting from it.  personal use is perfectly legal.

as far as individual licenses to use win32 codecs or and dvdcss type software, you would need
to contact the property owners themselves, describe your desired use and inquire on license 
fees.  you can likely find a contact on MS website for the win32 codecs.  i *think* the logical
first point of contact on the dvd encryption would be the MPAA.

if you do make the inquiries, please follow up here with any response you might get. :D
