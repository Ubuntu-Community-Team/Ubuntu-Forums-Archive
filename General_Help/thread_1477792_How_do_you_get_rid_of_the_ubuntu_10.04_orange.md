### Post by WorMzy on 2010-05-09
Try installing [gnome-color-chooser]("apt://gnome-color-chooser"), it lets you change all the colours that gnome uses.

Edit: After you've installed it, run it from System -> Preferences, the colour you want to change is Entry Fields -> Selected.

### Post by Endomancer on 2010-05-09
Another way to change it, is to go to **System/Preferences/Appearance/Theme** then hit the **Customise** button at the bottom. Then go to the **Colours** tab and click on the **Background** box of **Selected Items**.

 This will give you the **Pick A Colour** tool which will allow you to change it to any colour using the graphical tool on the left or using the eye dropper under that to use any colour on the screen.
