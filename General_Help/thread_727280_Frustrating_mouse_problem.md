### Post by Sockerdrickan on 2008-03-17
When some apps crash the mouse gets locked up. What do I restart to get it working? I don't want to restart X!

---

### Post by DoctorMO on 2008-03-17
If it's a usb mouse you can unplug and plug it back in. If it's PS2 or a touchpad then you need to restart X; although you should monitor dmesg and x11 logs to find out the real problem.
