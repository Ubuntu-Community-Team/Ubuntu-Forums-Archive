### Post by Denis on 2006-05-29
You can just add the entry in the menu manually if that's the only problem. 

Make sure you have smeg installed, that way you can configure the menu with a nice graphical program. 

Select it under the System Tools menu (App. Menu Editor). Chose Internet -> New Entry and insert Name, Command etc.
