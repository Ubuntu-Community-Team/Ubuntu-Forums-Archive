### Post by Herman on 2010-11-24
Here's another idea. 
You can set an appointment in evolution calendar and get it to run a program for you.
An idea would be to get gthumb to open an image from openclipart on your desktop, for example a birthday cake if the reminder is for someone's birthday. Or you could get totem to open an audio file for you and play you a song maybe.
These can be set to run a set time before the appointment and can also be set to repeat.

### Post by Herman on 2010-11-24
:D Yes, they will.

... unless something has changed since last time I was playing around with that.

The very first time you try it you can expect a little teething problem related to Ubuntu's security awareness, and Ubuntu will need to ask your permission for Evolution to run whatever program(s) you specified. That only happens the very first time for each program you want evolution to run, but it means a test run or two are needed before relying on the alarm.

It's easy to set alarms in Evolution though, and with a little imagination we can set ourselves some quite impressive alarms.

I wrote about configuring evolution here, [Configure Evolution Email]("http://members.iinet.net.au/%7Eherman546/p5.html#Configure_Evolution_Email"), and in particular, about Memos, Tasks and Calendar Appointments here, __[GO]("http://members.iinet.net.au/%7Eherman546/p5.html#Memos_Tasks_and_Calendar_Appointments").

You can also get your Ubuntu computer to [boil the water for your coffee]("http://picasaweb.google.com/herman543/HowMyComputerMakesCoffee#5065830874883664210") before it wakes you up in the morning too if you want, I used crontab for that, but I could have used evolution just as easily or perhaps easier. :)
