### Post by black veils on 2012-08-03
you should be able to:

**update manager** >> **settings** >> **updates** tab >> **notify me of a new ubuntu version **(choose **never**).


i understand what you are saying, but you need to be on a more recent version of ubuntu for security reasons, because 11.04 stops being supported in october. you want a gnome-like desktop, i would recommend the Xubuntu operating system, it is essentially ubuntu, with the xfce desktop environment (instead of gnome or unity etc).


if you install a fresh operating system, whether that be ubuntu or xubuntu etc, know that the support times vary, xubuntu 12.04 has 3 years, ubuntu 12.04 has 5 years.

A "Partial Upgrade" is just a bundle of updates for **your current version**.

It does not "upgrade" you Ubuntu version to any newer version. Any version upgrade is offered as exactly that.

Millions of people run Ubuntu Partial Upgrades without any impact at all.
