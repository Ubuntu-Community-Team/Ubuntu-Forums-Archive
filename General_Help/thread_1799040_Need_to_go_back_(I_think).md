### Post by linuxman94 on 2011-07-07
If you don't like the new unity interface, you can use the classic session by choosing Ubuntu Classic in the session menu at the bottom of the login screen after you have selected your username.

Kubuntu is still based on ubuntu, it is an official derivative with the KDE desktop environment.  There is also the Xubuntu derivative (XFCE desktop) and Lubuntu which uses the LXDE dekstop environment and is useful for computers with low memory.

If you don't like any of the official derivatives, you could check out Linux Mint ([www.linuxmint.com]("www.linuxmint.com")), which is based on ubuntu.

Oh and you could always use the 10.04LTS version instead of the latest version
