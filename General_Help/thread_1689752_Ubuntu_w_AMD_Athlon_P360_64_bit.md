### Post by God Of Atheism on 2011-02-17
I have no experience with the specific processor you mention, but my experience with Ubuntu 64-bit edition on another 64-bit AMD cpu is fine (I'm using an athlon x2 245). Note that the 32-bit version should also run fine, if you intend to put only 2 or 3 gb ram in the computer then the 32-bit version should be enough, if you intend to put in more, you might well be better off with the 64-bit version.
