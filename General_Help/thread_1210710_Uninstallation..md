### Post by twright on 2009-07-13
Hi,
I suggest that you try the alternative install CD as it may work where the live CD does not. If the wubi uninstaller will not work you may be able to delete the ubuntu installation manually if you are short on space; it should be located either in c:\ubuntu or in program files. To recover your files you could try starting ubuntu in recovery mode from the menu at bootup, from there try the fix x, fix broken packages and fsck options.
