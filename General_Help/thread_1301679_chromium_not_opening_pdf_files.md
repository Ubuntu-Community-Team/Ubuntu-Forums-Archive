### Post by fragos on 2009-10-26
For now you will get a blank tab but the PDF should have been downloaded. If you haven't changed your download target it will be in your home folder.  In my case I changed my download target to the Desktop and a click on that file will display it. Chromium is still in development. No doubt it will work the way you expect once the package is released.

### Post by luigi_mb on 2009-10-26
> **vidak said:**
> Is there any way to make chromium open pdf files using the acroread plugin, or any other way? According to about:plugins Adobe Reader 9.1 plugin is installed, and working. When I open a pdf file, a gray screen appears, without loading the document :(

One way is to install the Google Docs Viewer extension. With it, all links to pdf (and a few others) files are automatically converted. When you click on such links, the Google Docs Viewer will display the documents.
