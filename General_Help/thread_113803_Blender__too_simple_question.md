### Post by Azriphale on 2006-01-07
By default, in Linux, Blender opens without borders, or in a fullscreen mode. This is good for me because is makes spanning two monitors a whole lot easier. I do not know how to minimise it, so if I need to get at something behind Blender (which is seldom), I save and quit.

If you find it less convenient to run in full screen mode all the time, start blender with the following command:

```
blender -w
```

You can also make a Blender launcher on your desktop or the panel to launch Blender like that.

### Post by Mr Frosti on 2006-01-07
"Ctl" + "Alt" + "D" is the default keyboard shortcut to display the desktop. This will minimize all windows on the workspace you are on.
