### Post by papibe on 2012-11-12
Hi Rocket J Squirrel.

Strictly speaking the answer is no, but in practice the answer is a **big yes**.

Lubuntu uses the LXDE desktop environment. You can add either just the desktop environment, or a meta packages with all its goodies.

Take a look at this [tutorial]("http://www.howtogeek.com/107368/how-to-install-the-lightweight-lxde-desktop-on-ubuntu/") that describes both options.

Hope it helps, and let us know how it goes.
Regards.

### Post by Lyfang on 2012-11-29
You could also try the small and fast Openbox as well and use Ubuntu applications. Be aware that Openbox uses a "right click menu". Lubuntu meta-package would install Lubuntu desktop components, stuff that you might not really need when you already have Ubuntu apps installed.
