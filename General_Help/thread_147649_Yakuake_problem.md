### Post by KubuntuKilledMe on 2006-03-20
the file is:

~/.kde/share/config/yakuakerc

here's mine:

[General]
ShowAlternativeShortcutConfig=false

[Global Shortcuts]
AccessKey=Alt+Backslash

[Options]
background=false
focus=true
height=50
location=50
screen=0
steps=20
tabs=true
width=100
