### Post by jamesharper-caesar on 2013-12-08
I'd suggest Gedit, it's my favourite text editor for working with everything from bash and C to Javascript and PHP. A lot of the preferences are off by default but there are some that I find pretty useful and really make Gedit great.

If you go to edit -> preferences you can change many options. Some that I find useful are: displaying line numbers, setting the tab width (I find the default of 8 a bit too much), performing automatic indentation, turning off the automatic backup files, and there are many useful plugins that can be activated. The themes are pretty nice too.

I'd definitely recommend Gedit, it is far superior to Notepad++ in my opinion.
