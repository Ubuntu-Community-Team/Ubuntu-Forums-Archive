### Post by lovinglinux on 2011-05-06
Try to disable global menu integration in the add-ons manager. See if that helps. If not, start Firefox in safe mode. If yes, then is probably an extension messing with the menus.

See [http://www.webgapps.org/firefox/general-troubleshooting](http://www.webgapps.org/firefox/general-troubleshooting)
