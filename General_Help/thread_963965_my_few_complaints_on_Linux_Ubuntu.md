### Post by CatKiller on 2008-10-30
> **ieBrazil said:**
> Why installing etc. things is not easy like on Windows?

Installing things is significantly easier than on Windows. Have a look at Synaptic Package Manager and the Add/Remove... tools.

It is not necessary to use the command line that often. However, to give instructions in a "click here, click on that thing, select the other" would be needlessly tiresome and inefficient for the people that give help here in their spare time, and would be specific to the desktop environment - that is, different instructions would need to be given for Gnome, KDE, XFCE, and so on, and some people would not get the benefit of the solution. Giving a simple, unambiguous command that can be simply copied-and-pasted into the Terminal is much quicker for both the helper and the helpee, and gives instant feedback with error messages if something is wrong. Hence, most assistance is given in this form.

GIMP's a pretty big program, but here's how to do some of those things:

Cropping: use the Rectangle Select Tool (keyboard shortcut: R).  Drag a rectangle around the area you want to keep.  Go to the "Image" menu, choose "Crop to Selection."

Resizing: go to the Image menu, choose "Scale Image."  Enter new dimensions in pixels; or, use the pop-up menu to enter to scale by percent, etc.  The little chain icon in between width and height means that they are linked, so if you change the width the height changes automatically.  Click that to turn it off if you want to resize width and height independently.

I use KDE and have an action add-on called "Kim" installed, so I can just select a bunch of images in a folder, Right click on them, and choose options to batch resize, rotate, convert to jpg, etc.
