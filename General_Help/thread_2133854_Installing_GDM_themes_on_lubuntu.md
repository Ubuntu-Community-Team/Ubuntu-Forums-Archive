### Post by neruson on 2013-04-09
Ubuntu and all it's derivatives, with the one exception of the Gnome edition, uses LightDM not GDM. You cannot install a GDM theme with LightDM. They are completely incompatible. You could install GDM with Lubuntu but it would drag most of the Gnome DE with it as dependencies. If you're going to do that you might as well run Gnome instead of LXDE because your lightweight Lubuntu install that's easy on resources would become bloated as all hell.
