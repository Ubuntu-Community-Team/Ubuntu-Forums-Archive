### Post by kd4hey on 2011-01-24
OK,  I installed a file utility called "4Pane" but it changed the way the "Places" folders, (documents, videos, etc) works.  Now instead of opening a folder to bring up the contents, it opens 4Pane.  

The real problem with that is the associated program do not work in 4Pane (I have to right mouse then select the program to open the file, everytime), so I tried un-installing it.  Now I get an error message when I try to bring up anything under the Places menu.

I even tried deleting & re-creating the "bookmarks" in the Places menu to no avail.  

So the question is, how do I re-associate the "bookmarks" under "Places" with opening a folder instead of this program 4Pane?

Thanks!

---

### Post by Frogs Hair on 2011-01-24
If the program is removed try right clicking the desk top > create folder > right click > open with another application > select file browser from the menu > make sure remember this application is checked.

You may remove the folder when finished , this is just to reset the default application for opening folders to the file browser.
