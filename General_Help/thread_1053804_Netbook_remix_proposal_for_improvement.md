### Post by mikasjoman on 2009-01-29
Hi guys

I got the netbook remix, and the first thing I thought was; **** this is nice - but it really could need a bit of "piffing" it up.

Just a little - to be a killer.

So I made a try.

The fist thing I noticed was a basic design rule; don´t add borders when you don´t need them. So I took them away.

Second, I took away the huge blob called "home" on the top. The user, as on a iPhone, already knows he/she is home. No need to tell what is obvious.

Third I took away the huge brown blob with the ubuntu logo. The reson for this is we have so small screens, so why show the ubuntu logo twice? There is already one in the corner, and I don´t really think you need to tell a ubuntu user that they are using ubuntu more than during startup.

I wonder who I should talk with if I would like to get this implemented in a try?

Hope you like my proposals for improvement for a cleaner netbook remix.

Sincerely yours
