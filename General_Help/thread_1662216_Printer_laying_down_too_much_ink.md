### Post by Maximus559 on 2011-01-07
I'm using Ubuntu 10.04, and I've got a little problem with my printer. It's an HP Photosmart C6280 all-in-one, and it works *almost* perfectly. I have no trouble setting up the printer and getting it to print - it's just that it's not quite printing correctly. Specifically, the printer uses too much ink.

This problem is manifested in the form of rough edges on the final printed documented. This is most noticeable when a piece of text is highlighted. While regular text is simply a little rough, highlighted text exhibits significant color bleeding. If I print the same document on Ubuntu and on Windows, both times using "normal" quality, the one printed with Ubuntu not only has these roughness issues but is also considerably darker. This is what leads me to believe it's an ink volume issue.

Interestingly enough, if I turn on double-sided printing on Ubuntu, this problem goes away. However, I don't usually want to use double-sided printing, so this isn't really a solution. Is there any way to control ink volume in the printer preferences? Note: setting the printer to "draft" mode apparently reduces ink volume, but also reduces image quality overall. Again, not a solution.
