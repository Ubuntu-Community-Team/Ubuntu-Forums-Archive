### Post by oldrocker99 on 2009-04-06
While the upgrade was going on, I was pleased to see that games, etc. were upgraded along with system files.

After rebooting, I did not see my USB printer or my two external HDDs. I looked up the bug and got this as a fix:

oldrocker99@oldrocker99-laptop:~$ sudo modprobe -r ehci_hcd
[sudo] password for oldrocker99: 
FATAL: Module ehci_hcd not found.


So...I unplugged the USB cord and plugged it back in. Voila!

No other issues so far. The new printer driver worked.
