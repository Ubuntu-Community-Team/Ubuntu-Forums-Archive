### Post by indranama on 2009-12-27
Sorry for bothering guys, I found it myself.

If you are using Mac4Lin 1.0, the latest version,
Gnome Menu icon is

```
/home/username/.icons/Mac4Lin_Icons_v1.0/scalable/places/gnome-main-menu.png
```

You may replace it with a png image of same dimensions.
