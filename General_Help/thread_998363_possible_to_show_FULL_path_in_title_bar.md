### Post by fragos on 2008-11-30
I'm assuming we're talking Nautilus File Browser. To the left of the location bar is an icon. Click it to change the display format. For removeable media the name will be /media/{volume lable}. When devices/volumes are mounted they are assigned folder names to use for access.

### Post by fragos on 2008-12-01
The alternate location display does give you the full path. The configuration editor doesn't seem to have a parameter for window title either. I've not been able to find a way to change the Nautilus title bar without changing the code. You could request the feature in launch pad. Some Gnome applications use the title bar that way.
