### Post by dcstar on 2006-01-09
[QUOTE=Y-town]I installed cpanel on my other computer, (im on my dads laptop right now) and it's says to login to your account with your root username and password. I don't remember making a root username, but i have a root password. How can i find out my root username?[/QUOTE]
"root" is the root (Superuser) username.

The root account is disabled by default in Ubuntu (for security reasons), but you access it by typing "sudo" before any command that needs to run as root (in a terminal, "gksudo" for graphical stuff).
