### Post by freelinuxhelp on 2008-02-20
Just append :PORT_NUMBER to the IP address or hostname of your server.
Example:
servername:3389 is implied by default
servername:3300 if TS is running on a different port

Screenshots are here: [http://freelinuxhelp.com/index.php?view=article&catid=45%3Atips-a-tricks&id=177%3Aconnecting-to-a-terminal-server-on-a-different-port&option=com_content&Itemid=59](http://freelinuxhelp.com/index.php?view=article&catid=45%3Atips-a-tricks&id=177%3Aconnecting-to-a-terminal-server-on-a-different-port&option=com_content&Itemid=59)
