### Post by cong06 on 2010-04-05
It seems 9.10 UNR has sporadic problems with items loading. I often get the message "An item has failed to load: "Glipper" Delete it? [Delete] [Don't Delete]" (or some such)
If you accidentally delete, then it's gone.

TO get it back is quite tricky, and involves pixel hunting. I think if you right click right near edge of the "title bar" where the X of that window is, you'll find it.
Does that make sense?
For me that's right before the "system tray"
So you could say the pixel between the Application Tray and the System Tray is the place you can right click to add items to the panel.

### Post by cong06 on 2010-04-05
> **leekaiwei said:**
> ok now how do i move it to the all the way to the right in the corner?

The hard part is over.
Now each "item" has a "lock" option. Right click around till you get a menu that looks like:
```

Help
About
------
Remove from Panel
[COLOR="Silver"]Move[/COLOR]
------
Lock to Panel

```

Unlock each item. Place them how you want (locking as you get them in place) and then make sure everything is locked in the end.
