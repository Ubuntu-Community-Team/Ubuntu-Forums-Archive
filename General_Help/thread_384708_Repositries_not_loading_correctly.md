### Post by Kateikyoushi on 2007-03-14
This will solve solve the problem with the automatix repo. Copy paste to terminal.

```
gpg --keyserver subkeys.pgp.net --recv 18B52FE3521A9C7C
gpg --export --armor 18B52FE3521A9C7C | sudo apt-key add -
```

Then try to update again.
