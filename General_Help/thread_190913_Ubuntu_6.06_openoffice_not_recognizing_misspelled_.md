### Post by wragge on 2006-06-07
Do you have the appropriate dictionary installed for your default language? You can tell by going to Format > Character > Font and looking to see if there's a tick next to the selected language.

The same thing happened to me because I'd set the default language to English (Australian) without actually installing the Australian dictionary.

To install a new dictionary go to File > Wizards > Install new dictionaries.
