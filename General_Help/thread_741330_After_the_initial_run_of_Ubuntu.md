### Post by pytheas22 on 2008-03-31
All the software you need to make a proper boot disk should be readily and freely available on Windows, as long as you can burn a CD...just install iso recorder from [http://isorecorder.alexfeinman.com/isorecorder.htm](http://isorecorder.alexfeinman.com/isorecorder.htm) , then download the Ubuntu iso, right-click on its icon and select "burn to iso."  And you have a boot disk.

Unfortunately I don't know how to solve your wubi problem, but I hope that the above will help you.
