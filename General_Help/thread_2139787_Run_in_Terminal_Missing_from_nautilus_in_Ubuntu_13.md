### Post by mc4man on 2013-04-27
You can choose from 3 options for this in nautilus, (ask, run, display), the new default is display
To change go nautilus > File menu > Preferences > Behavior > see screen

If you need the script/executable/whatever to run in a terminal then just run it from a terminal..
