### Post by nad on 2005-08-25
The program launcher 'Archive Manager' should appear in your drop down menu Applications -> System Tools. Is it there?

If it is, you may need to reassociate archive/and or compressed file types with Archive Manager (File Roller). Why the association may have become lost I do not know.

If the launcher is missing, make a new one. If the app is missing, something more serious has happened.
