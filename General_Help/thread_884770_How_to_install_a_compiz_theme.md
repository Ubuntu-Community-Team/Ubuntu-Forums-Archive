### Post by Krydahl on 2008-08-09
If you open emerald and can see the theme then you ought to be able to just click on it to activate it. Try changing between two different themes and see if you can see the change.

The thing to realise is that an emerald theme only controls windows decorations - that is the outside of the windows - the buttons and title bar and bits like that. All the rest, the main colour of your windows, the backgrounds and all that, is still controlled by the main gnome theme (accessed from system->preferences->appearance). So, if you're expecting using that theme to make everything go black and red like in the screenshot, you'll be disappointed.

Often emerald themes recommend a standard theme to use with them.
