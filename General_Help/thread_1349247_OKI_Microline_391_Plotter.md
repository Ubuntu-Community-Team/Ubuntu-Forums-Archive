### Post by axyelp on 2012-10-27
Excuse me if it feels necroposting or whatever some might call it, but having found a solution nowhere else available, I feel it an obligation to post.

For OP's printer, this package suffices:
[ftp://ftp2.okidata.com/pub/drivers/linux/SFP/dotmatrix/dotmatrix.tar](ftp://ftp2.okidata.com/pub/drivers/linux/SFP/dotmatrix/dotmatrix.tar)
it runs flawlessly, and has got a really good web interface to manage it, even on networks.

It worked on:
Linux AcerPower-Series 3.0.0-12-generic #20-Ubuntu SMP Fri Oct 7 14:50:42 UTC 2011 i686 i686 i386 GNU/Linux


One can find a ton of other printer drivers from OKI here:
[http://www.okidata.com/mkt/html/nf/Drivers.html](http://www.okidata.com/mkt/html/nf/Drivers.html)
