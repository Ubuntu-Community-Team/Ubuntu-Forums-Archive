### Post by Ryadt on 2008-07-27
You can use either the OSE version or the one you have in the repos. Choosing the OSE gives you the latest edition but you will have to install it manually. The one in the repos can be install simply by running in terminal```
sudo apt-get install virtualbox-ose
```
Hope this helped.:)

### Post by kernelhaxor on 2008-07-27
I use Ubuntu as host and Vista as the guest OS.  I installed the open source edition at first but then later switched to the "full" version (you have to download from virtualbox.org and install yourself) due to lack of USB, RDP and other features in the open source version.

So, if you wanna use any of those features, I would recommend you to download and install the "full" version.
