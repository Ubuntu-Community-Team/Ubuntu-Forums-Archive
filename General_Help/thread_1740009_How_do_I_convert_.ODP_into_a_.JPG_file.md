### Post by kaldor on 2011-04-26
You don't want to export it as a JPG. Save it as a PDF :)

File > Export as PDF

Then check out the PDF. The PDF you see should be identical to what everyone else will see.

Edit: I am attaching a Zip with two files; an ODP and PDF. I made the .odp in LibreOffice and then exported to PDF via the Method I used above. Take a look to see how well it worked.

### Post by kaldor on 2011-04-26
Read my above edit in my previous post for attachment to demonstrate.

A PDF file should look the same on any PDF viewer; it should even save fonts that are not installed on other computers.

If it works on Ubuntu, it's 95% likely it'll work on Windows. If it works on your dad's PC, then you're all set to go.

You can also use Adobe's Official (they created the PDF format) PDF reader which is in the Ubuntu repos.

Edit: If it's nothing personal, you can attach the PDF/ODP file here so we can take a look to make sure it looks fine if it makes you feel better :)
