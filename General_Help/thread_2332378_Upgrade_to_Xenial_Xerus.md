### Post by howefield on 2016-07-31
Well, it is actually the first point release of 16.04 that you would be upgrading to, ie, 16.04.1.

16.04 has been out  for three months and the point release incorporates all the fixes that have been added since release back in April. Read the [release notes]("https://wiki.ubuntu.com/XenialXerus/ReleaseNotes") and if there is nothing there to discourage you, I'd suggest creating a Live media of 16.04 and testing in Live mode to make sure all seems well.

What was it about the 15.? releases that didn't work the way you expected ?

Of course, 14.04.5 is supported till 2019 and if it is working fine for you there is probably no compelling reason for you to upgrade. At least not any time soon.

### Post by grahammechanical on 2016-07-31
A new version of Ubuntu is released every six months at the end of April and October. They do not all have the same length of support. Every 2 years at the end of April a version is released that has 5 years support. These versions are called Long Term Support (LTS). Both Ubuntu 14.04 & Ubuntu 16.04 are LTS releases.

The versions of Ubuntu released at 6 monthly intervals in the interim only have support for 9 months. So, those who installed Ubuntu 15.04 would have needed to upgrade to 15.10 and now need to upgrade to 16.04. Both Ubuntu 15.04 & 15.10 have reached end of support life.

We all need to decide on an upgrade pattern. (a) install the latest Ubuntu and upgrade every 6 - 7 months to the next latest. (b) Install an LTS version and upgrade to the next LTS about 3 - 6 months after the next LTS is released. (c) Stay on the LTS version that we have installed until the last year of its support period and then up-grade to the newest LTS version. (d) Upgrade from LTS to LTS as the impulse takes us.

What pattern do you choose? If we buy new hardware and it is the very latest hardware then we may need the very latest version of Ubuntu because the latest version of Ubuntu comes with the latest Linux kernel that has support for newer hardware.

If an LTS version is working fine on your hardware and you are not going to buy a new machine then you may not need the latest LTS version until your existing version has become a bit older. If you buy a new machine then you will need to install either the latest point release of the LTS version you are using or the latest LTS release of Ubuntu. The original ISO image that was used to install Ubuntu may not install driver modules for the newer hardware.
