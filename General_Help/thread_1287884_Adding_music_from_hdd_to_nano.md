### Post by tgalati4 on 2009-10-11
floola is a java-based application that sits in the root directory of the nano.  You run it by clicking on it when the file manager pops up when you plug in your nano.  It will not read iTunes 9 databases.  So only format your nano with iTunes 8.

With floola, you can drag-and-drop music from the nano to the pc or from the pc to the nano.  It runs on mac, windows, and linux (32-bit only).  You simply download it and put a single floola file in the root directory of the ipod.

It's really an elegant solution to Apple's brain-dead iTunes database scheme.
