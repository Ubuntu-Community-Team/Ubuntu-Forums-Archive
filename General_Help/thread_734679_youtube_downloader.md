### Post by pytheas22 on 2008-03-25
Places>Search for Files.

By default, if you just opened a terminal and ran the script without changing directories first, it should have saved it to your home directory (Places>Home Folder).  Also, make sure that the script actually downloaded the file.  It should have taken a few minutes; if you really can't find the file, run the script again and make sure you see it downloading (it displays its progression as a percentage of the file downloaded); otherwise, check your syntax to make sure you're running the script correctly.
