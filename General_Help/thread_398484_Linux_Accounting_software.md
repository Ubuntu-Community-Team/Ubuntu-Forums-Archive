### Post by FluffyElmo on 2007-04-09
No problem, you're question has actually inspired me to start organizing my finances and try the programs myself. So far for me it looks like it will come down to a choice between online bill-payment (Moneydance) or invoicing (Gnucash). Maybe I can use both...

A Moneydance Tip: If you install the current 1.6 JVM, go to *File->Preferences* and select *GTK+* as your Look and Feel. Next time Moneydance starts it will use your Gnome theme settings. This works with pretty much any Java program...I first used it in Netbeans and love the new native look.
