### Post by fantan on 2006-08-16
Hi,

Just download the "xbindkeys" and "xbindkeys_config" files, and you can setup any keyboard shortcuts as you like.

Before configuring the xbindkeys application, make:

touch .xbindkeysrc in your home directory.

If you want the xbindkeys to run with the X start just put the "xbindkeys" command in the System -> Setup -> Sessions -> Automatic start.

fantan
