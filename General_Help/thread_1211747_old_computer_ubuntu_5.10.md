### Post by Possum Films on 2009-07-13
I would definitely advise against installing a recent release of Ubuntu on a computer that old. I have tried and the speed that programs run at is almost too slow to be usable. Puppy Linux is a very good operating system for older computers. If you don't want to use Puppy Linux, Debian LXDE is another good lightweight operating system.
