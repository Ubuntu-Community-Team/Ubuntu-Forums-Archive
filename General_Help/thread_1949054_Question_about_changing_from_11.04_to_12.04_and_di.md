### Post by ccrs8 on 2012-03-29
Yes, it is possible to add versions of programs that are newer than the version that came with a particular Xubuntu release.  Firefox, LibreOffice, etc. all have their own PPAs that you can add and then get newer versions.  It is possible to apply a newer kernel than the one that came with the release as well.  Especially for newer versions of programs, this can be useful - for instance if you love the current state of your system and everything is perfect except for one tiny thing that a newer version of a specific program would bring.

However, you are talking about installing newer versions of programs (plural) and possibly the kernel.  I guess I'm not sure why doing this piece-by-piece is preferable to just upgrading to a newer Xubuntu release, since the work is already done?  I do think that if you do an upgrade, you'll have to stay at 32-bit.  I don't think it is possible to upgrade from 32 to 64 - you'll have to do a fresh install.
