### Post by procras on 2006-02-25
The airport extreme Broadcomm chipset is not supported in the default distribution.

There is a project to reverse engineer drivers and apparantly it is working!

But you would need to do alot of hacking and setting up to get it to work at the moment.

If you are in no rush it will probably be supported better in the next release Dapper Drake or two:cry:
