### Post by mano cazalet on 2008-05-07
Hello Morten Linux

welcome to the ubuntu world

the main reason you cannot install that theme is because you are using gnome (ubuntu) and not kde (kubuntu). These are different desktop environments and theme for one cannot be installed on the other and vice-versa. For gnome you can find interesting themes in [www.gnome-look.org](www.gnome-look.org).

Hope that helps and that you can find a theme that pleases you. Or maybe you can switch from gnome to kde. Take a look at these great tutorials and explanations: [http://www.psychocats.net/ubuntu/kdegnome](http://www.psychocats.net/ubuntu/kdegnome)
