### Post by spikoley on 2011-04-22
It has been a while since I have run gFTP, but this should work.  Try right clicking on the file and there should be a place to change permissions.

Another option is to use the built in FTP feature in Nautilus.  Click on places then Connect To Server.  Once you have connected then right click on the file and change the permissions.
