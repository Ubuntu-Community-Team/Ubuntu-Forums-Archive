Hey Ubu4moi

please try this :

Download Adobe Air For Linux:
user@yourpc:~/Desktop$ wget “[http://get.adobe.com/air/thankyou/?installer=Adobe_AIR_1.5.2_for_Linux”](http://get.adobe.com/air/thankyou/?installer=Adobe_AIR_1.5.2_for_Linux”)
 

user@yourpc:~/Desktop$  ls -l



user@yourpc:~/Desktop$ -rw-r–r–  1 user user 13745012 2009-10-04 16:00 **AdobeAIRInstaller.bin**
 

user@yourpc:~/Desktop$ **chmod a+x** AdobeAIRInstaller.bin  

 user@yourpc:~/Desktop$ **./**AdobeAIRInstaller.bin
