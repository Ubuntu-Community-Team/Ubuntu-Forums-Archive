I just made an script to install a HP Deskjet 710 C printer. 
  
This can maybe help some of you. Just follow the indications.
  
To install the Deskjet Printer, just type :
```
sudo bash
cd /root
wget -N http://patrick295767.sitesled.com/miniram/easyinstall_patrick_hpdeskjet710c.sh
sh easyinstall_patrick_hpdeskjet710c.sh
exit
```
([ script code]("http://patrick295767.sitesled.com/miniram/easyinstall_patrick_hpdeskjet710c.sh"))
