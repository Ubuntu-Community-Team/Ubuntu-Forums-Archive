### Post by 3rdalbum on 2011-08-01
When Wayland matures and distributions start shipping it as the default, Nvidia and AMD will make drivers for it.

Even if they don't, Wayland can also run on top of X - which I imagine might remove some of the advantages of Wayland, but you'd be able to use 3D. Vice-versa should be possible too - X running on Wayland was one of the first demonstrations around of Wayland.
