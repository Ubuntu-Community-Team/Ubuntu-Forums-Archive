### Post by RubenK on 2009-01-11
After another hour of searching I solved it. System > Preferences > Keyboard > Layouts > Add > USA International (with dead keys)
Set that one as default and you can additionally remove all other entries. This way you will also have the Alt-GR keys with which you can i.e. make the euro-sign: €
