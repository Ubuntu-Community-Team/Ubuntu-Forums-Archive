### Post by donsy on 2009-10-28
I recently installed Chromium and imported the bookmarks from Firefox. Trouble is, I had been using Shiretoko for the past week and the FF bookmarks were not current. How can I get Chromium to import and REPLACE the current set of bookmarks from a .json file? Currently I can only import from Firefox, and it merely adds a new set of bookmarks to the end of the ones previously imported.

---

### Post by fluffman86 on 2009-10-28
delete all of your chromium bookmarks, close chromium, open shiretoko, backup your bookmarks (as an html file I think?), and re-import them from the bookmarks menu of chromium.

---

### Post by donsy on 2009-10-28
Thanks. I wasn't able to save to an .html file in Shiretoko, but after deleting all bookmarks in Chromium I exported to a .json file and imported it into Firefox. Then in Chromium under Options > Personal Stuff I imported from Firefox. Now they're all in sync.
