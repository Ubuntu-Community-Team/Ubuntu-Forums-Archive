> My question is if I do a manual install through BIOS and set up the partition amount for the Hardy install will my computer auto rewrite my partitions and keep my Gutsy install.

are you confusing the ubuntu install partitioner with BIOS? this is different.

anyway, if you set up your partitions manually via the installer partitioner, as long as you don't 'tick' the box on any partition you don't want formatted, then the partition should be safe. the only partitions requiring formatting are the root and swap partition.
