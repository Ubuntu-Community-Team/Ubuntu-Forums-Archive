or type **apropos *subject*** in the terminal to list related man pages

```

cp (1)               - copy files and directories
cpio (1)             - copy files to and from archives
install (1)          - copy files and set attributes
mirror (1)           - copy file trees from FTP server to client
```

### Post by thayerw on 2007-05-05
Ubuntu's "console" is actually the standard GNOME terminal.  There are two book that I would personally recommend:

**Linux Pocket Guide**
2004 O'Reilly Media 
ISBN # 0-596-00628-4
This is a great little book that covers a lot of essential commands for the shell (terminal).  It is written with Fedora Linux in mind, but since they both use GNOME it's virtually the same.

**Running Linux (5th Edition)**
2006 O'Reilly Media
ISBN # 0-596-00760-4
While not specifically about the shell, this is a great reference book to have for Linux (and Ubuntu) in general.  It covers everything from desktop environments and gaming, to disk maintenance and compiling--it'll help with the terminology too.

Good luck!
