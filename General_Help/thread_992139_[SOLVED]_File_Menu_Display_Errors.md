### Post by ajgreeny on 2008-11-24
Compiz configuration changed?  This can cause problems with displays like yours.  Try running without compiz for a while and see if it helps.  Use ```
metacity --replace
``` in the Alt+F2 dialog box.

### Post by Feezus on 2008-11-24
Disabling Compiz fixed my issue.  Maybe it's time to put away my fancy rotating cube.   Thanks for the advice.
