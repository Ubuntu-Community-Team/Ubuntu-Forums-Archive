### Post by CatKiller on 2009-06-29
It's theme-dependent. You can use your own icon by placing it in the appropriate place for the theme that you're using. Icon themes are stored at either /usr/share/icons/<ThemeName> for system-wide themes or ~/.icons/<ThemeName> for user-specific themes. The place for the PNG icon is <ThemeName>/<icon size>/mimetypes/gnome-mime-image-png.
