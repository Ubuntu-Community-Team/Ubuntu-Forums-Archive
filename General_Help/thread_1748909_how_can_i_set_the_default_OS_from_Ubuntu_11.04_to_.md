### Post by grahammechanical on 2011-05-04
How many kernel boot options do you get on the Grub menu? Six? No, five. You cannot boot into a kernel called Previous Linux Options because it does not exist.  From the little list that you have typed i would say that 1 = Ubuntu. 2 = Ubuntu. 3 = memory test. 4 = memory test. 5= Windows 7.

You have posted the wrong section of the Grub configuration file (grub.cfg). The section you want is the part that begins with

> ###Begin /etc/grub_d/10_linux ###Count the lines marked Menu entry.
