### Post by lisati on 2009-07-18
If you want Ubuntu on a bootable USB, it can be done. Here's a quick summary of one way of doing it with Ubuntu 9.04
[LIST=1]
[*]Boot from the CD
[*]Select the "Try Ubunutu without installing anything"
[*]Go to the menu System->Administration->USB Startup Disk Creater
[/LIST]
The rest should be fairly self-explanatory.
