### Post by Krydahl on 2008-07-17
Yes - at least I think so. When you say menu do you mean panel - the things that run along the top and bottom of the screen and look like the windows taskbar?

The thing showing your wireless strength is actually an icon running in the same applet as the other icons you're talking about so you only need to move one thing.

If so, if you right click on the area that displays these icons and select "remove from panel" it will vanish. You can then go to the bottom panel and right click and select add to panel. That will give you a long list of applets you can add. The one you want is called the "notification area".

You may find that your network manager (wireless strength) doesn't appear in there when you add it, but it should be there next time you reboot.
