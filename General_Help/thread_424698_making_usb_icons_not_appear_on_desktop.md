### Post by m.musashi on 2007-04-27
Use configuration editor. It's under apps -> system tools. If it's not there you can add it by right clicking the apps menu and choosing edit menus.

Once in the configuration editor, go to apps -> nautilus -> desktop and uncheck the volumes visible option. That should get rid of the icons. However, it will also get rid of all other drive icons (if you have any). If you want internal partitons to show but not removable then I'm not sure how. If you want to get rid of them all then this works.
