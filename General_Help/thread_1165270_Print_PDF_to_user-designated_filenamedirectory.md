### Post by mcduck on 2009-05-20
Should work by default. Just click Print-button in the program you are trying to print from, select "Print to file" as the printer, set the file type to PDF and choose the file name & directory you want.

(Although, if you really are using Ubuntu 6.06 like your profile says this feature isn't available. You can install cups-pdf for similar functionality but you'd probably better off by upgrading to more up-to-date Ubuntu version.)
