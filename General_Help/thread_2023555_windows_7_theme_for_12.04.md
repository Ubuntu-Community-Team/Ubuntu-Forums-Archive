### Post by PhantomTurtle on 2012-07-16
> **linux junkie said:**
> I'll look into it. Is it a distro, or a shell?

It is a shell based off Gnome 3. It can be installed with a few simple terminal commands. Open Terminal and type in,
```
sudo apt-add-repository ppa:gwendal-lebihan-dev/cinnamon-stable
sudo apt-get update
sudo apt-get install cinnamon
```

Then just log out, press the little ubuntu logo next to your name and choose Cinnamon and log in using your password. Here is a screenshot of Cinnamon: [http://linuxmint.com/pictures/screenshots/maya/cinnamon.png]("http://linuxmint.com/pictures/screenshots/maya/cinnamon.png"). It is very customizable and is very easy to customize using the Cinnamon settings to make it feel more Windows-like.
