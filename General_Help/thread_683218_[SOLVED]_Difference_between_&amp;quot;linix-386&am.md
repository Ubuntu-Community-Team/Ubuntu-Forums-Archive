A "386" kernel will work on ancient processors (386 class processors were created in the late 1980s) as well as newer ones using very basic CPU capabilities, a 686 kernel (like the "generic" one) will use newer processor features and deliver better performance.

The Live "Intel" CD has a 386 kernel so it will work on the widest possible range of hardware.
