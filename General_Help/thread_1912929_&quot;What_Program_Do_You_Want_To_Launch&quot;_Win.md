### Post by stevedude on 2012-02-17
Sadly it took me a couple months to get an answer though no one attempted to answer my post <sigh>

The solution lies in the Ubuntu "Power Cog" icon (the icon in the upper right of your default desktop). The icon where you typically choose to log off/shut down. There is a menu choice labeled "System Settings...". Choose this menu item and a window with various system settings that can be modified is displayed. Choose "Removable Media". The window that opens has a checkbox on the bottom that states: "Never prompt or start programs on media insertion".

Uncheck this box. 

I have no idea how my system modified this because I never changed the setting. Anyway, CDs/DVDs now prompt me and ask what to do when I insert media (CD/DVD/USB)

Hope that helps someone out there!
