### Post by hyperdude111 on 2009-08-27
Other media alternatives are Songbird. It has no video or podcast but it is from mozilla (like firefox), and it supports many addons(like firefox.

Rythmbox - The default ubuntu music player. No video player but built in podcast feed support

Banshee - Find in the repositories, built in video and podcast support.


Just a tip, you mentiond getting video working well there is an easy command that installs all codecs such as; flash, java, divx, mp4, mp3 etc...
That command is
```
sudo apt-get install ubuntu-restricted-extras
```
