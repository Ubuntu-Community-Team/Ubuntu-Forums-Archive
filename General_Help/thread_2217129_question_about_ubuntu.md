### Post by grahammechanical on 2014-04-15
The only way to find out is too download an ISO image and burn it to a DVD disk and run a Ubuntu Live session.

[http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop)

In a few days a new version of Ubuntu will be released (14.04). You might want to wait until then. How to install.

[http://www.ubuntu.com/download/desktop/install-desktop-latest](http://www.ubuntu.com/download/desktop/install-desktop-latest)

Any new computer operating system would seem hard to use if a person was not familiar with it. Last year I installed a pre-view version of Windows 8. I found it hard to use. That was because I did not know how to use it.

Do not expect any Linux distribution to be like a Microsoft Windows operating system because the developers have their own ideas and are not trying to produce a cheap copy of Windows.

These are the system requirements

[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

You might want to look through the online version of the Desktop guide. It will give you an idewa of what Ubuntu now looks like.

[https://help.ubuntu.com/14.04/ubuntu-help/index.html](https://help.ubuntu.com/14.04/ubuntu-help/index.html)

Regards.
