### Post by mustang on 2005-11-07
Just install Ubuntu on the empty drive. It will install GRUB (a boot manager) that will do everything for you. Let us know if you experience any problems.

EDIT: Oh, be sure to install GRUB on the drive that has your MBR (most often the primary drive which is probably the drive windows is on I'm guessing)
