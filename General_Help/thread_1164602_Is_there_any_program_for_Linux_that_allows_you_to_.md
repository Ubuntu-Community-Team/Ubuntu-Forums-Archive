### Post by whitethorn on 2009-05-19
You might want to give Rainlendar a try.

[http://www.rainlendar.net/cms/index.php](http://www.rainlendar.net/cms/index.php)

It's not in the repositories, but you can download the lite version for free.  It's even got a .deb you can download, so no need to compile it yourself.  It has different skins, and it supports a Todo list and shows calendar events. I use it.
