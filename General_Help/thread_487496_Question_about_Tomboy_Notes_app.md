Hello.  Tomboy does not currently have support for this sort of linkage (we try to keep it very simple, what with the auto-linking between notes, and we want it to be clear when a link is internal vs external).

It is possible for plugins to do this sort of thing (the bugzilla plugin makes it so a bug number links to a bugzilla URL, for example).  But there are not currently any plugins that provide this feature.

If you've got any more cool ideas for Tomboy, or need help with anything, make sure to check out the Tomboy mailing list:

[http://lists.beatniksoftware.com/listinfo.cgi/tomboy-list-beatniksoftware.com](http://lists.beatniksoftware.com/listinfo.cgi/tomboy-list-beatniksoftware.com)
