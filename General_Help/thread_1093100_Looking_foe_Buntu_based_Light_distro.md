### Post by kevdog on 2009-03-11
You can get openbox (which is a display manager), on any regular ubuntu installation also, and it will make your system run a lot lighter (without re-installation).  In addition to openbox, there is fluxbox, and enlightenment (among others).  You might want to look at some screen shots on wikipedia to see what you can do with the various window managers.  Im using e17 (enlightenment) and I love it for older hardware.
