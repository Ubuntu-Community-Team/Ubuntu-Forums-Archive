### Post by fluffman86 on 2011-01-11
You are certainly welcome to install KDE apps and kubuntu-desktop. Just search for it in the Software Center. Then when you log in you'll select it under "Sessions" on the GDM login page. 

If you want the KDE login page and ONLY KDE apps and the full KDE experience, then you should install [http://kubuntu.org](http://kubuntu.org)
