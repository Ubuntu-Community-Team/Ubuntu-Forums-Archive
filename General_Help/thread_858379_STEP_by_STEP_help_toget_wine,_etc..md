### Post by werries on 2008-07-13
Linux is NOT windows. wine can't run everything. but to get wine.
go to Applications > Accessories > Terminal.
type in
```
sudo apt-get install wine
```
and hit enter. then type in your password (it won't show as you type it), then hit enter. it will start installation. there ye go.
