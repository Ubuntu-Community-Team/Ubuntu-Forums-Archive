### Post by themusicalduck on 2009-05-28
I think the best way to get flash is download and use the .deb from [http://get.adobe.com/flashplayer/](http://get.adobe.com/flashplayer/)

Or you can just get it from the repository by typing into a terminal:

```
sudo apt-get install flashplugin-nonfree
```

But make sure there aren't any other versions of flash installed before installing it. If there are then you should uninstall them first.
