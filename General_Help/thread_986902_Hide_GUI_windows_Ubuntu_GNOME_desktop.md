### Post by fuzzybear3965 on 2008-11-19
This is a thread inspired by a p2p experience where I was told I was either to stop running my app through the day or I was to have it turned off.  Since I use my pc for ssh, ftp and vnc stopping it was not an option.  Working it at night was not an option (can't sleep with it on) and I love running torrents.  So I needed to hide my apps running.

To do this I navigated to the bar at the bottom of the Ubuntu desktop (in a default configuration).  Then, I clicked the three dashes to the right of all my windows tabs at the bottom and clicked "remove from panel".  No longer could my p2p app be seen running.  

*** If you have a launcher icon display in the top right of that program you can do the same thing (remove the app from the panel).  Now you'll be good to go!

Finally, To re-add right click bottom panel click "add to panel" then scroll down to "windows list" and click "add"

### Post by patrickballeux on 2008-11-19
I would be easier to login a second user, do what ever you want to be on in this desktop, then go back to your first user using "ALT-F7"

When you close your session, you have "Change User", this will start a whole new desktop in the graphic terminal "ALT-F8".  Login, do what you want.  Then you can switch between the two sessions using "ALT-F7" and "ALT-F8" (Could be also "ALT-F9"...).


Anotherway would be also to use a command line version of your software, then go in a TEXT terminal (ALT-(F1,F2,F3,F4,F5,F6)) and run it from there.

And you could also use "screen" to start any command line software in the background)

And you would not loose your taskbar...
