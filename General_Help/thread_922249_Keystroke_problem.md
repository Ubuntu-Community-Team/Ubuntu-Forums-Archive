### Post by snibbets on 2008-09-17
Hi there,
I'm having occasional problems with keystrokes. This sometimes happens when I fire up Ubuntu. If I start typing there is a delay before characters appear and then they may appear repeated even though I only typed them once. This is particularly frustrating if I try to type my password in the console and can't even see if a character has been repeated.  I have found that if I delay for about a second in between typing each character I get what I wanted to appear.  This problem occurs with two different keyboards (one USB, the other PS/2).  When I started my system a short time ago I was having this problem, but it seems to have disappeared after a few minutes.

Probably, this isn't a big deal, since it only happens from time to time, but I think annoying things like this add up and could put people off adopting Ubuntu.
