I never used vbox from the repos but always used the version direct from Oracle at [https://download.virtualbox.org/virtualbox/7.0.12/virtualbox-7.0_7.0.12-159484~Ubuntu~jammy_amd64.deb](https://download.virtualbox.org/virtualbox/7.0.12/virtualbox-7.0_7.0.12-159484~Ubuntu~jammy_amd64.deb) then also get the Extension Pack from [https://download.virtualbox.org/virtualbox/7.0.12/Oracle_VM_VirtualBox_Extension_Pack-7.0.12.vbox-extpack](https://download.virtualbox.org/virtualbox/7.0.12/Oracle_VM_VirtualBox_Extension_Pack-7.0.12.vbox-extpack)

Assuming it downloads to your Downloads folder in your home you should first run command ```
sudo apt remove virtualbox* 
``` to get rid of the 6.1 version then type command ```
cd Downloads && sudo apt install ./virtualbox
``` and press Tab not Enter to allow the terminal to autocomplete the package name then immediately press Enter and it should install.

Once the virtualbox package is installed run the Extension Pack simply by double clicking it in your Downloads folder.
