### Post by lmarmisa on 2011-08-09
I agree with haqking. But I would like to add a comment that could be useful if you continue having problems with the mouse and keyboard integration.

If you need to release the cursor that is in use by the virtual machine, press the Control key located at the right side of your keyboard.

Once you have installed Ubuntu and its updates, I recommend to install the VirtualBox Guest Additions. The current focus problem will be solved then.
