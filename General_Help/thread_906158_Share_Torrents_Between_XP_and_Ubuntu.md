### Post by Tindytim on 2008-08-31
I'm currently using both XP and Ubuntu on the same machine (different drives). I would like to know if I could somehow set up Ubuntu and/or XP to not only continue with the torrents that have been downloaded while I was using the other OS, but continue downloading torrents that weren't originally downloaded with that OS. I know I can do it by hand, but I was wondering if they was a utility or Torrent program that had the capability right of the bat.

---

### Post by mb_webguy on 2008-08-31
The only way I can think to do it is if you have a shared data partition (which you really should, if you put any thought into your partitioning scheme when you decided to set up a dual-boot system).  

You would simply point your torrent client in each OS to store the torrents in a folder on the shared partition (and of course do the same for the folder where the files are downloaded).  If your torrent client(s) don't give you the option to specify where the torrents are kept, you should be able to use links.
