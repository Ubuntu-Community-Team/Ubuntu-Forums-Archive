### Post by varunendra on 2011-03-19
I doubt low memory or abnormal usage of processing and/or memory by some application.

Although turd's system seems to have 1GB of RAM (found specs on net) so low memory isn't the issue there.

Live info about CPU & RAM usage can be obtained by typing the following command in terminal:
```
top
```

@ annieyarr,
I'd recommend you to try the above command after opening a few tabs in firefox. Also, as mmsmc asked, what are your computer specs? (Especially ram).

As for swap space, often 1-2 GB is ideal swap-partition size to go with 1GB RAM or less.
