### Post by Brucevdk on 2009-04-25
> **StormRoBoT2 said:**
> is this a good C++ compiler?

Geany is not a compiler, it's a text-editor (or lightweight IDE if you prefer), but I assume you already know that. It does have the ability to call an actual compiler though, e.g. g++ (potentially through a Makefile). For a short review of Geany from a Python perspective see [my answer at StackOverflow](http://stackoverflow.com/questions/81584/what-ide-to-use-for-python/729965#729965). Otherwise you might want to look at Eclipse which is probably the closest to an actual IDE.

You can get the v0.16 version of Geany from [Getdeb](http://www.getdeb.net/release/3812) (this version is already available from the Jaunty repositories though).
