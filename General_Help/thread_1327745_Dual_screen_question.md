### Post by scouser73 on 2009-11-17
You'll need to do it as Root for the changes to be permanent so go to Terminal and enter **gksudo nvidia-settings**

Under the nVidia X Server Settings, click on the monitor that you want to be the main one, then click the box that says **Make this the primary display for the X screen**
