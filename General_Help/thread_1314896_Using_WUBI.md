### Post by mikewhatever on 2009-11-05
Edubuntu is not a separate distro, but rather is Ubuntu with a bunch of editions. Thus it can be added to either Ubuntu or Kubuntu by installing edubuntu-desktop or edubuntu-desktop-kde metapackages respectively.
In other words, use Wubi to install Ubuntu, then install edubuntu-desktop metapackage, and you should have Edubuntu in place.
