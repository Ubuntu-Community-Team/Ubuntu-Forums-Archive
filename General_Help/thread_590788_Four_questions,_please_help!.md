### Post by rich.bradshaw on 2007-10-25
Beryl has been replaced by Compiz Fusion, and is already installed by default on Gutsy.

Just go to the system menu, then preferences, then apperance and go to the last tab to change settings.

Add/Remove programs in Applications is what you need to add software - I have used linux for around 5 years, and have never compiled anything.
