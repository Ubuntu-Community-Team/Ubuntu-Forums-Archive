### Post by Adrian on 2006-03-21
[QUOTE=rhomsy]I just installed breezy badger, got the internet working, and clicked on update when I was notified.  However, my software is old, including firefox version 1.07!  When I go to Synaptic, it says that it is the most up-to-date version.  
[/QUOTE]

The quick answer is that Firefox 1.07 was the most up-to-date version when Breezy was released in October. After a new Ubuntu version has been released, only bug and security fixes are added to the main repositories.

There is a repository called the **backports** repository where updates for the most popular programs can be found. Firefox 1.07 is an exception, though, because it turned out to be quite difficult to backport properly.

If you really want to install FF1.5, Automatix is an option:
[http://www.ubuntuforums.org/showthread.php?t=138405](http://www.ubuntuforums.org/showthread.php?t=138405)
