### Post by pointone on 2008-05-27
Wine does not work with every single application. According to [the Wine AppDB entry for Endorphin]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=10754"), it is known NOT to work flawlessly with several recent versions of Wine.

One user posted this comment:

> To use the program, you must first copy MSVCP60.dll to windows/system32 folder. When the application is started, it is minimized, but do *not* try to resize it, it will result in a crash. Rather, click the window and select unminimize.
