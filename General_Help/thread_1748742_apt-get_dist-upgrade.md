### Post by satish_j on 2011-05-04
Command 'apt-get dist-upgrade' basically upgrades all the installed packages on system AND also upgrades the distribution to its latest version,if any.
For eg: Lucid has foll versions
Lucid 10.04
Lucid 10.04.1
Lucid 10.04.2
If a system running 10.04 uses the above command,it gets upgraded to 10.04.1
