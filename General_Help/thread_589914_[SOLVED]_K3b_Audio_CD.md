### Post by forestpixie on 2007-10-24
hi - that reminded me to check mine after I reinstalled - try installing this and having another go

```
sudo apt-get install libk3b2-mp3
```

I just done it and checked and all is fine for me now :)

make sure k3b is closed just in case
