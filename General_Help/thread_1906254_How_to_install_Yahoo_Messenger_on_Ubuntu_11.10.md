### Post by mastablasta on 2012-01-09
> **BNick said:**
> What is Lubuntu and does Ubuntu need codecs for video?
 
 
Lubuntu is Ubuntu that uses LXDE desktop environment as opposed to default GNOME.
 
Yes it does need the video codecs. you will get them if you install restricted extrass package from the software center.
 
you install programmes via software center.
 
If they aree not there then you need a PPA (personal package archive) or a .deb file (kind of like exe file) or you need to complie it yourself using instructions found in the programme code. usually it involves copying a few commands to terminal.
 
Regarding YahooIM, you can't install it in linux. Pidgin though good is poor substitute. there is a project aimed at bringing yahoo IM functionality to linux. it's now called gyache improved: [http://sourceforge.net/projects/gyachi/](http://sourceforge.net/projects/gyachi/)
 
this is the most compatible with YahooIM and at the same time best alternative to yahooIM. instructions on installation procedure are found inside the .tar file.
 
games that are linux based work in Ubuntu. Some windows games work with various degree of success using wine (or play on linux or crossover - all WINE frontends): [http://appdb.winehq.org/](http://appdb.winehq.org/)
 
however most modern windows games won't work or will work with limited graphics. Linux is still not a good gaming platform due to poor support of software makers and also OpenGL is not as good as DirectX at the moment. for modern windows games using windows is bets option. or console if they have a console version of the game.
