### Post by lykeion on 2010-12-16
Please don't double post. You can bump it after an hour if no one answers. FYI your question is very strange and maybe that's why you don't get any responses.

Short answer: Extract the zip archive and use jar command to create the jar file.

But (and here's why your question is strange) why would you want to convert a normal archive (ZIP) to a Java archive (JAR)? They have completely different uses. 
ZIP is used to compress files of any kind into an archive. JAR is used to archive a Java application so that you can run it easily with java -jar app_name.jar.

You might want to read about these file formats on Wikipedia:
[http://en.wikipedia.org/wiki/JAR_(file_format](http://en.wikipedia.org/wiki/JAR_(file_format))
[http://en.wikipedia.org/wiki/ZIP_(file_format](http://en.wikipedia.org/wiki/ZIP_(file_format))
