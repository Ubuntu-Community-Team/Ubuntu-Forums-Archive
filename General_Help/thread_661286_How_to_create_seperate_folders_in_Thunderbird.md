Hi.

I have Thunderbird installed and I would like you to set it up so I can use two different email addresses with Thunderbird. 
I have setup both accounts and they work OK, but they both seem to share the same inbox and sent folders, etc. I would like them to be completely separate.
I have looked through the options and can't seem to do it. On the left column I see "Local folders" which is what both accounts seem to be sharing.

Any idea how I make separate inboxes, etc for my accounts?

---

### Post by ebe326 on 2008-01-07
go to edit>account settings then click server settings on the left side. click advanced under server settings. I think it needs to be set to 'inbox for this servers account'. after doing this you might have to move your existing emails into the appropriate folders. Also, you should do this for each account.

david
