### Post by Dkkline on 2010-05-18
here is what you could do

  1.) make a .sh file (not 100% percent sure how you do :( )

  2.) or you could go "System -> Preferences -> Startup Applications" then click "Add" under name you type "Skype Autostart" and under command you type: "skype" and that's it! close logout and it should work (you might want to go skype -> options -> General: and check "start Skype in system tray" and of course check the remember me at skype login screen
