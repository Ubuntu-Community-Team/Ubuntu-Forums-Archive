### Post by itsdarklikehell on 2012-01-09
try wallch.

Wallch is a General purpose Wallpaper Changer for the
GNOME desktop which can change the desktop background
image after an adjustable time. The aim of Wallch is
to make it easier for you to keep your desktop "fresh"
and new!

Some random features:
 - Features JPG, PNG, GIF and BMP support
 - Random image changing
 - Random time changing
 - Desktop/Sound notification
 - History logging
 - Live Earth Wallpaper
 - Taking Screenshot
 - Taking Webcam Image

source: synaptic
[http://sourceforge.net/projects/wall-changer/](http://sourceforge.net/projects/wall-changer/)
