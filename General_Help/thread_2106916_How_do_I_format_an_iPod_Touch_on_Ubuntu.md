### Post by scbingham on 2013-01-21
To use an iPod on Ubuntu programs you need to set it up with iTunes & copy at least one song to it.

Start here: [http://support.apple.com/kb/ht1339](http://support.apple.com/kb/ht1339)

This will restore factory defaults, then run iTunes.

After that you'll have a good chance; it can be a minefield.

Good luck
