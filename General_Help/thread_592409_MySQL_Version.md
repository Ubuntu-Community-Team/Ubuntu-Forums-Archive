---

### Post by bukwirm on 2007-10-26
Look in aptitude for the version numbers attached to the packages you have installed.
Or try tying something like "*programname* --version" or *programname* -v" in a terminal, ie "apache2 --version" should give you the version of Apache2 that is running.
Error webpages usually contain the server version, also - try going to a webpage on your server that doesn't exist.
