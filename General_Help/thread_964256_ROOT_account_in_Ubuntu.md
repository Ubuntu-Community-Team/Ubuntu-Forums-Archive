### Post by Titan8990 on 2008-10-30
> Who can tell me how to use root account in Ubuntu 8.04.1 LTS Desktop Edition? Thanks in advance. 


Prefix any command needing root with "sudo". 

[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)

example:

sudo fdisk -l

or to use a root shell in a more traditional method:

sudo -i
