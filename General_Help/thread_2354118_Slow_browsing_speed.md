### Post by dragonfly41 on 2017-02-28
I usually see that popup when there are many tabs in use and perhaps one is stuck loading a script.

The problem is finding which tab is the culprit.   Usually a video still playing.

I have not used Firefox for a while but you can try starting up in safe mode or installing a Firefox tab manager.
I had good results from earlier version of TGM (Tab Groups Manager) which allows tabs to be put to sleep.

[https://addons.mozilla.org/en-Gb/firefox/addon/tabgroups-manager-revived/](https://addons.mozilla.org/en-Gb/firefox/addon/tabgroups-manager-revived/)
