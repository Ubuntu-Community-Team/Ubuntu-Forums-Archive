### Post by taurus on 2007-01-25
At the GUI login screen, look at the bottom left corner for the Options.  In there, pick KDE and of course, you need to install KDE on your machine first before you can use it.

```
sudo aptitude update
sudo aptitude install kubuntu-desktop
```
