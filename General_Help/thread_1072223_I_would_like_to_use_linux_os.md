### Post by mixmaster on 2009-02-17
I would avoid xubuntu.  It is great for older and/or resource constrained machines and, tbh, I rather like it.  But there are niggling inconsistencies which are likely to trip up a newbie who takes advice from mainstream ubuntu sites.

The difference between ubuntu and kubuntu basically comes down to which interface/application set you prefer.  Personally I dislike the kubuntu interface (KDE) for a variety of mainly aesthetic reasons but discussions of the relative merits tend to come end up in religions warfare so pick one and try it for a while.  You can install the other over the top later and flip-flop between them without any real hastle except for having too many apps in the menus.
