Hi!
I guess you are using Compiz. Have you installed **compizconfig-settings-manager**? If so, open it, go to **Window Decoration** and in the **command** line enter:
```
emerald --replace
```
Then restart Compiz. That should get Emerald running.
If you want some more themes, you can download some from [http://www.gnome-look.org/]("http://www.gnome-look.org/")
Check under the **Compiz** and **Beryl** sections. To install them, open the Emerald manager, click on *Import...* and select the file you just downloaded.
Cheers!
