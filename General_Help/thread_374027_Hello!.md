### Post by Maestro23 on 2007-03-02
> **mike-db said:**
> I have one hard drive 229Gb I'm willing to give as much as it needs to run smoothly.
I'm on a Linksys router.

Cool. First thing you should do, is download one of the Live CD's.  Burn the image as an .iso with your favorite burning program(slow speed..4x).  Then make sure your BIOS is set to boot the CD/DVD drive first.  Pop the CD in.  

You can then run the OS off the CD and see how you like things, if it detects/likes all your hardware etc...  If you like what you see, then you can install to your hard drive.

Here is a guide: [http://www.psychocats.net/ubuntu/index](http://www.psychocats.net/ubuntu/index)

*Beryl, the pretty eye-candy that you saw, is still in the early stages.  For some it works, and for others...crashes their systems.  If I were you, I would get my system up, running, and stable before you even tried messing with that.  Just a suggestion.

Good luck.:)
