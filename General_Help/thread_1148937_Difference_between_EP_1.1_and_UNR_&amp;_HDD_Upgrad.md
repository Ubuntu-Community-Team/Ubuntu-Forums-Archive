### Post by ubun_two on 2009-05-19
I can think of few differences:

1, Easy Peasy/eeebuntu, etc. installs netbook optimized kernel by default.  You can still install these under UNR by additing additional repository.   More info on htt://array.org/.

2, Default softwares are different.  Easy Peasy installs Thunderbird and Skype by default and installs no games, for example.

3, The icons/branding.

4, Generally, Easy Peasy already are configured so that extra buttons (turning on/off wireless, etc) work out of the box.  This may need manual configuration on UNR.

I installed Jaunty UNR and optimized kernel on my eeePC 1000h, and did few other configurations, and I think it works rather well.  

Jaunty boots up way faster than Intrepid/Easy Peasy 1.1.  I am sure Easy Peasy will catch up, but will take a little time, but when it does, I suspect that it would require less tweaking to work optimally on a netbook.
