### Post by mb_webguy on 2008-09-14
Check out the [Wikipedia entry]("http://en.wikipedia.org/wiki/Ubuntu_Ultimate").  It's a sort of non-official remastered version of Ubuntu with some extra pre-installed software and it's own repository containing packages that aren't found in the Ubuntu repositories (but generally can be found elsewhere, such as the Medibuntu repository).  Basically a guy made a LiveCD of his version of Ubuntu and his PPA.

### Post by iaculallad on 2008-09-14
Actually, UU is based off on Ubuntu. It's not maintained by the official crews who maintains Ubuntu, it's maintained by an individual trying to re-create it so it would be pre-built with the restricted codecs, eye candies, and other applications that are not included on the official Ubuntu copy. I'd tried using UU once and I'd say it's good (and I'm happy for the creativeness /artistry of the author), but I still prefer using the official Ubuntu copy and manually install the applications/codecs I needed.
