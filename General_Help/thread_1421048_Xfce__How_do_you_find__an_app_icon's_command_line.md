### Post by Kurtosis on 2010-03-03
I'm setting up Mythbuntu both as desktop workstation and PVR/TV.  I'd like to be able to see where icons in the Applications menu point to, what their command line is, so I can recreate some of the elsewhere on the taskbar or desktop.

However, right-clicking and selecting Properties doesn't seem to show this information.  Is there a way to find it?

Thanks!

---

### Post by rbc on 2010-03-03
Right-click anywhere on the Desktop and choose Create Launcher. In the Name field, start typing in the name of the application. Xfce will start suggesting names of applications. Choose the appropriate one and this will automatically fill in the rest of the information for you. I assume this will work for Mythbuntu. It does work using Xfce/Xubuntu
