### Post by knorr.orange on 2006-04-15
Sorry, I should have included more details.  Here are the steps:

Download latest 915resolution from here:
[http://www.geocities.com/stomljen/download.html](http://www.geocities.com/stomljen/download.html)

untar with this command:
tar -xzf

copy the file 915resolution to your path using this command:
sudo cp 915resolution /usr/bin

modify start up script so this program runs each boot:
sudo vi /etc/init.d/rc.local
(this is using vi, so if you are new to this, use a different editor you are familiar with, so just substitute gedit or something for the vi in that command as long as you have graphics working so far)
It will ask you for a password.  Enter your user password.

On the first line in the editor it will say #1 /bin/sh
Add a new line after that and enter this:
915resolution 5a 1440 900 24

save the file and reboot.


What we just did there is to add your resolution to the list provided by the hardware.  When the machine boots up the i810 driver asks what resolutions are available.  Yours was not on the list, so you got bad stuff.  What this does is modifies the in-memory copy of that list so when the driver asks if 1440x900 is available, it is!

I hope this works well for you.  Please post your results for the benefit of others reading the thread.
