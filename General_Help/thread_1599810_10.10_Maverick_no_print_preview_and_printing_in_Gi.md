### Post by roadrash on 2010-10-18
I had this problem and only found this bug report  after a lot of searching.
[https://bugs.launchpad.net/ubuntu/+source/gimp/+bug/636329]("https://bugs.launchpad.net/ubuntu/+source/gimp/+bug/636329")
 
So I am posting this fix here so anyone else may find it easier when searching or google'ing for it.

Open a terminal and enter the following into it:

> sudo add-apt-repository ppa:ubuntu-treblig/gimpfixes
sudo apt-get update

now run update manager from system/administration menu & install any updates. There will be a Gimp update there that will fix this.
hope this helps.
