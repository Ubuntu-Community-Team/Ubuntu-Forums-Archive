### Post by dino99 on 2010-11-07
you need to learn about formatting: use partedmagic to prepare your hdd:
- 15 Gb for root with ext4 format (/) and bootable
- 2 Gb for swap
- no limit for your data: ext4 (/home)

then install with the "alternate" iso

*** to restore /dev/sda1 use tool from [http://www.cgsecurity.org/](http://www.cgsecurity.org/)

### Post by garvinrick4 on 2010-11-07
You can try but it doesn't look good at all, but here is to mounting file system if anything is left to mount. Use live cd open terminal:
```
sudo mkdir /media/root
```
```
sudo mount /dev/sda1 /media/root
```
If anything mounts on desktop open and grab your document and drag to desktop.
