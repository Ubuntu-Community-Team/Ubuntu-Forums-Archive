### Post by Quackers on 2011-01-10
I would set the bios to boot from the Ubuntu hard drive and then try re-installing grub to sdb with the following commands from a terminal in Ubuntu
```
sudo mount /dev/sdb1 /mnt  
sudo grub-install --root-directory=/mnt /dev/sdb
``` then if that reports no errors you could run ```
sudo update-grub
``` to make sure that the Windows Loader is picked up, then try booting XP from grub.

### Post by Quackers on 2011-01-10
It seems that a space may have been left out of one of the commands.
From the live cd desktop please copy/paste these commands in to the terminal one at a time, pressing enter after each line.
```
sudo mount /dev/sdb1 /mnt  
sudo grub-install --root-directory=/mnt /dev/sdb
```
Then, if no errors are reported please run
```
sudo update-grub
``` to make sure the Windows loader is picked up. If all is ok try rebooting and selecting Windows from the grub menu.
