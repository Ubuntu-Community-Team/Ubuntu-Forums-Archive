### Post by The Cog on 2010-10-07
Well, I would normally use the Enter key for that, but you can also use the following three key sequence:

Ctrl-Shift-U
A 
Enter

The A is the hex code (000A) for newlline in Unicode. You can do any character you want with Ctrl-Shift-U then the hex code then Enter.
