### Post by yetiman64 on 2012-01-22
You will be able to mount the NTFS or FAT partitions in Ubuntu (support for both included) and directly access any Window partitions files.

If you are doing a Wubi install of Ubuntu check [[COLOR=Blue]--here--[/COLOR]]("https://wiki.ubuntu.com/WubiGuide#How_do_I_access_the_Windows_drives.3F")

As you mention installing Ubuntu to the full drive, you can use a live cd and access the Windows partitions and if necessary transfer  or copy files to either a usb stick or some sort of external drive caddy. Then do your Ubuntu install to the full HDD.
