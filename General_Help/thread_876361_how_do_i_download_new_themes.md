### Post by Archimedes0212 on 2008-08-01
You can unarchive the theme that you downloaded - so it's just a folder, not a zip or tar or whatnot and then manually toss it into .Themes in your home folder.

.Themes is hidden but if you show hidden folders you'll obviously be able to see it.  That's what I've done with some themes that I can't install through Appearances.
