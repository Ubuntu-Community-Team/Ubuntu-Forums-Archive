### Post by bakegoodz on 2010-01-31
To be honest I haven't used Scalpel and Foremost, just read about them. They seem to be pretty sophisticated forensic tools. I have used photorec to recover files before. It is pretty easy to use with its text based menu interface. I use it for badly damaged filesystems, because it will extract all files at the binary level of the hard drive, deleted or not. It usually doesn't recover file names. Sorry if my ramblings is not what you wanted, but I thought I would share what I know.
