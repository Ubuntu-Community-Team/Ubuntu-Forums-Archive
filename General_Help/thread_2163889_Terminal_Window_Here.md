### Post by Lewis Balentine on 2013-07-20
Back in the days of Windows XP there was an add on (power toy) that allowed one to right click and open a MD-DOS prompt window with that folder as the current directory (pwd). In Windows7 I copy the path string from the top of the file manager window and paste that into that after typing CD.

I am running Ubuntu LTS 12.04 with Gnome fall back (no effects). 
Is there any utility, shortcut, trick to open a terminal window to a folder that is being shown in the filemanager ?

---

### Post by carl4926 on 2013-07-20
The extension:
nautilus-open-terminal

---

### Post by Lewis Balentine on 2013-07-20
sudo apt-get install nautilus-open-terminal
nautilus -q


that did it. many thanks!
