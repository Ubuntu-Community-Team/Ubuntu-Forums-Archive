### Post by ibutho on 2009-02-23
Once a specific Ubuntu version has been released, most packages will remain at the same version. This is to keep the release stable and to cut down on introducing bugs from new packages or features. Its not that difficult to install seamonkey from the tarballs, so start a new thread if you need help with this.
