### Post by HellionDarkLord on 2010-01-17
OKay, I don't really know what I'm doing, but...

I decided I wanted to reprofile my boot up on my desktop so that I could get boot faster. [This Thread]("http://ubuntuforums.org/showthread.php?t=254263") Is where I got the idea.  But following the instructions doesn't work in 9.10.

I did just have to type "e" with the first line highlighted to get to an editor, but instead of hitting "e" again to edit the lines, Grub2 simply allows you to edit the lines as if it were an editor like kate or gedit.  I didn't know which kernel line to add the word "profile" to, so I added it to both the lines that said "kernel" in them.  (the last two)  

I think It took close to a hour for the drive to stop working completely.  It barely lit the LED indicator every few seconds, just enough to look like a tiny almost invisible spark inside the LED, I got bored and started reading a book while waiting for it to stop. I only noticed it again when the HDD started really doing something, and then it went totally quiet.  

The next boot was so fast that the bar I'd normally see with the fuzzy light spot moving back and forth didn't even appear before the username and password dialogue box popped up!  It's crazy fast boot now, but was it worth sitting for over an hour waiting?  Maybe. Maybe not. funny how the whole computer seems faster now too.
