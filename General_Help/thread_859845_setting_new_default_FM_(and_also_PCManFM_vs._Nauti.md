### Post by tramir on 2008-07-15
Presumably, PCManFM is faster and less bloated than Nautilus. But then again, it lacks a lot of the functions of Nautilus (e.g., not network support, no CD burning etc.) It really depends on what you want to do and how you use that file manager. I personally chose pcmanfm over Nautilus, but as it was said before, it's purely a personal choice. There are a lot of posts on how to replace Nautilus with pcmanfm as the default file manager, like [this one]("http://www.ubuntugeek.com/how-to-replace-nautilus-with-pcman-file-manager-in-ubuntu.html").

One other file manager that's really good, with a ton of functionality (but unfortunately relies on KDE libraries) is Krusader. It's in the repositories. And there are really a lot of file managers, each one with their pros and cons: xfe, emelfm2, gnome-commander and so on and so forth. If you google for "linux file manager", you'll have an idea :)
