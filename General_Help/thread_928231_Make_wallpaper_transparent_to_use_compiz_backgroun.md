### Post by uBeer on 2008-09-24
About the musicplayer: if you use Rhythmbox now, you should see an icon of a box in the systray. If you click on that your window should close (and the player still in the tray). If you close Rhythmbox by clicking the X you'll close the program so it quits.
There's also a key combo to 'minimize' Rhythmbox to tray. I believe it is something like Ctrl-W or Ctrl-Q. You can find it by clicking File (left-top corner) and looking in the menu what the combo is. Right now I'm at work so I can't check it for you.
Good luck with the rest of your problems, I can't help you with that.
