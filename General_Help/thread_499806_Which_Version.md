### Post by jespdj on 2007-07-13
The difference between Ubuntu, Kubuntu and Xubuntu is the desktop environment (GNOME, KDE or XFCE (not sure about the last one)).

Which one you want depends on which one you like best, and we can't tell you what you like best - you will have to find that out for yourself.

I tried Ubuntu (GNOME) and Kubuntu (KDE) myself and I like KDE better, so I'm staying with Kubuntu.

Just start with the latest version, which is 7.04 (Feisty Fawn).

### Post by rillip on 2007-07-13
I recommend you avoid Edgy Eft.  It seems to have been the redheaded step child of versions.  I've seen a lot of posts where "thing x worked in dapper, but doesn't in edgy??" and then the thread gets marked as resolved by them upgrading to Fiesty.

I agree with jespdj, I'm using Kubuntu myself, but to play around with it, I installed the xubuntu-desktop and ubuntu-desktop packages later, just to see what I was missing; you don't have to do a reinstall to get the items unique to that version, you can install it from Synaptic, Adept, whatever package manager you're using.

I would say try out a live CD of Fiesty and see if you like it/it works.  If you run into issues, I'd try Dapper.
