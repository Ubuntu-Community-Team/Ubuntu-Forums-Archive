### Post by hawthornso23 on 2011-07-05
Since unity is built on top of gnome I assure you that there are no plans to remove gnome. The problem is that supporting gnome2 desktop on top of the gnome3 toolkit is going to soon be impossible. Gnome created this problem by embarking on a radical rewrite and redesign - never a good idea in a free software project as the free software model works best via continual incremental improvement. 

The problem that gnome created is hitting ubuntu first as ubuntu has a faster  development cycle than most other distros. But all gnome based distros are going to face having to migrate away from the gnome2 desktop eventually as it is no longer being maintained. The only choices in the future are going to be gnome shell or unity.
[COLOR=Red]
**I thank Canonical for making sure that we will at least have a choice.**[/COLOR]

While I like neither in the state they are right now, unity appears to me to have a lot more potential than gnome shell. It has some quite cool features and it also works with compiz which offers the potential for creative synergy in future.
