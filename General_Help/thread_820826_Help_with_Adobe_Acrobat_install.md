### Post by mike2357 on 2008-06-06
There's an Adobe plugin for Firefox on Mozilla's web site.  Here's the link:

[https://addons.mozilla.org/en-US/firefox/browse/type:7]("https://addons.mozilla.org/en-US/firefox/browse/type:7")

You'll see several plugins; the first one is for Adobe Reader.

Before downloading anything, I suggest you make sure it's not already available through Ubuntu.  I don't think that the Adobe plugin is.
