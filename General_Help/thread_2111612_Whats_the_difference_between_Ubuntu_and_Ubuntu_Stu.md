The Main difference to my knowledge between the 2 is that Ubuntu Studio is aimed at Photography/Video Editing and has software pre-loaded to help with this.

I have never used Ubuntu Studio so not sure what the layout is, but as Linux has a lot more customize able options than windows it should be possible, but I will leave it to someone else with a bit more knowledge on the two to help with how you would do this.

As for Repositories, Ubunutu has one of the larges Repositories available which is partly from it being Debian based and having inherited a lot of its packages form this.

Generally you should be able to find most packages in the Repository and fairly often if you can't you can either download the .deb file and install it yourself or what you are after has a Debian Repo anyway which you can add.
