### Post by grahammechanical on 2011-07-15
I am confused as to what you want to do. If you want to install Ubuntu using text mode, then get an alternative CD download. They do not have a Graphical User Interface for the install process.

If you want a version of Linux without a GUI and only the command line, then you are wanting something different. Rescue mode would give you the same thing. Or you could start your experiment by un-installing the various parts that make up a desktop until all you are left with is the Command Line Interface.

There a some rescue downloads available. I am guessing that they give you LInux without a GUI and just the CLI.
