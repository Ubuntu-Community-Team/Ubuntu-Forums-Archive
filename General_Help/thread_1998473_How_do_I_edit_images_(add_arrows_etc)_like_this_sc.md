### Post by wvonstockhausen on 2012-06-06
I had the same need and needed to draw an arrow between  Las Vegas and my town on a screen shot of Google map with the driving time on it. I did use GIMP and scrabble a line with the drawing tool and wrote in cursive on it. Of course you can do it neater with a straight line and text tool.
Of course GIMP has far more possibilities (comparable to Photoshop) but you can learn it. I did. Now I fix old damaged photos for friends with the clone tool but still have to find out how to replace faces with that from another picture. I did that in Photoshop and will figure that out in GIMP eventually. When making screen shots in Ubuntu, save it as a jpg not png file.

Drawing with a mouse is of course not desirable  but with a stylus pad you can make nice curves. You can make small corrections with the clone tool.
