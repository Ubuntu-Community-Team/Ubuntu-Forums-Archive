### Post by jrib on 2006-04-04
If you use esd (check with system > preferences > multimedia systems selector) install vlc-plugin-esd and then choose esd from the preferences in vlc.  

Since I just noticed you are using kubuntu, what I said above probably won't work... you are probably using arts, so install vlc-plugin-arts instead and choose arts from the preferences in vlc.
