### Post by nikhilk on 2006-11-05
I dont think there is a direct way to do this, but you can try doing it the indirect way. I havn't tried it but hopefully it should work.

First follow the instructions given [here]("http://www.openoffice.org/FAQs/ms-access/ms-access.html")
Yes this has to be done from Window$.

Then save the Access database in OpenOffice format. Copy over the database file generated by OOffice to a Ubuntu partition.

Finally boot into Ubuntu and start OOffice Base and open the database file you just copied over.
