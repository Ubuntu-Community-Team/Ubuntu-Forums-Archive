### Post by Mark Phelps on 2011-11-29
> **nesh90ful said:**
> [FONT=Comic Sans MS]Can anybody please guide me on how you go about sharing files between windows os and ubuntu:confused:[/FONT]

HOW you do it depends critically on how you have them both installed.  If you installed Ubuntu after Windows, and did it using Wubi, accessing your Windows files from Ubuntu is very different than if you installed into separate partitions.

Also, if you have separate partitions, and you want to mount your Windows OS partition read-write to make changes to files there, I would advise strongly AGAINST that, as that is an easy way to corrupt the Windows OS filesystem.
