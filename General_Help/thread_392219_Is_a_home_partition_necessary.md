### Post by jeffathehutt on 2007-03-24
The main reason people have a seperate home is so they can reinstall the OS without having to wipe all their files from the drive.  Since you don't have many files, and since you have a backup of them, I would say you don't need a seperate home.  If you ever have to reinstall, you should be easily able to replace the files from your backup.

### Post by richmooremi on 2007-03-24
I don't know about most people, but the main reason I use a home partition is so I can create a ghost image that includes all my data and configuration settings in one drive. If you're happy backing up your files by copying them to your USB stick and have only made a few simple configuration changes, then you really don't need to create a home partition.
