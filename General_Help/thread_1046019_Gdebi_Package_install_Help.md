### Post by ajcham on 2009-01-21
What do you mean by 'all' of them?  Do you mean any .deb that you download in your browser?  If yes, and you are using Firefox:

Edit » Preferences » Applications tab
Scroll down to Debian package, and click where it says 'Always ask'.  Select the option to use GDebi, then close the preferences dialog. That should do it.

### Post by cariboo on 2009-01-21
You can't set gdebi to install .deb packages automagically. You have to double click the .deb, or right click and select Open With Gdebi Package Installer.

Edit: When  you download a .deb you can set firefox to allow gdebi to automagically install the file also.

### Post by mcduck on 2009-01-21
..actually you can set Firefox to automatically open .deb files with Gdebi, it still won't *install* them automatically. You need to click the "Install"-button and give your password to confirm before the package is installed.
