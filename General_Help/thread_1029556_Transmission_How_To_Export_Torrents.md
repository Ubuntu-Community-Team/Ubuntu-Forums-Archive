### Post by gammaGT on 2011-06-05
Hi, I wish to migrate torrents from Transmission as well, but I will move them to Deluge running on a headless server (debian).

looking for the transmission directory was a good idea, it's in ~/.config/transmission
there's a torrents directory here which contains the .torrent files for every torrent still running or suspended.

so, if migrating from Transmission to Transmission you can copy the whole directory and expect it to work right away, but if moving to a different client or even another OS, worst case you can still add back every torrent manually ;)
