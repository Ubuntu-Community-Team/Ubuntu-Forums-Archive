;    browsable=yes

### Post by bab1 on 2013-01-22
> **giantjoebot said:**
> I know that when a line starts with this # in front it means its a comment, but what does it mean when I line has ; in front of it?  

Example

;    browsable=yes

Do they do the same thing?

Yes.  But the style is: Use a # if your are commenting and use a ;  if you are disabling a line.  The both do the same this but the style (what it looks like) is different.
