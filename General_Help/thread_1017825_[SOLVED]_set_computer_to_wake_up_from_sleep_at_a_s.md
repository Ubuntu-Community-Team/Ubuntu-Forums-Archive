If you could find the command that wakes up the system from sleep mode, then you can try putting that in a cron job and see if it works.

I'm not sure it would work as I understand the wake up signals are generated by the underlying hardware, as the host O/S is essentially suspended and waiting for the hardware to wake it up.

### Post by Fox5 on 2008-12-22
Ubuntu does not have built in support to wake up the computer, no.

It can only be done through the bios, which Ubuntu may be able to modify. Currently no one has written an app to automate the process.
