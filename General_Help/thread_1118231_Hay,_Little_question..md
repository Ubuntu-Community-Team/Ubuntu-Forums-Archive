### Post by mb_webguy on 2009-04-07
It sounds like you're talking about simply remastering Ubuntu to put your own look and feel to it, but have it still essentially remain the Ubuntu distribution -- something more like Super Ubuntu than Linux Mint.

You can certainly do this, though since Ubuntu is licensed under the GPL, you have to provide the source code for all software you distribute, even that which you didn't modify, and you have to provide that source code free of charge, even if you sell the actual software.  This only applies to what you specifically distribute, and not what users may install using your product.  So you don't have to provide source for the entire contents of the Ubuntu repositories, just for the packages included on your ISO.

Several remastering tools are available, including [Remastersys]("http://www.geekconnection.org/remastersys/remastersystool.html"), [Ubuntu Customization Kit]("http://uck.sourceforge.net/"), [Reconstructor]("http://reconstructor.aperantis.com/"), [Builder]("http://www.gnewsense.org/Builder/Builder"), [ulc-livecd-editor]("http://code.google.com/p/ulc-livecd-editor/"), and [Disc Remastering Utility]("http://linux.dell.com/wiki/index.php/DRU_Disc_Remastering_Utility").
