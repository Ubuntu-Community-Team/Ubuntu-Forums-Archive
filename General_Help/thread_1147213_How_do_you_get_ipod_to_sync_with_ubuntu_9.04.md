### Post by delcypher on 2009-05-03
There are two applications I know of

[songbird + iPod add-on]("http://www.getsongbird.com/") - This can sync with its music library. I use this program daily and it is fantastic. Songbird is not in the repositories, you will need to [download it.]("http://www.getdeb.net/app/Songbird"). To use it with an iPod you will need to install the [iPod add-on.]("http://addons.songbirdnest.com/addon/12") Songbird is based off mozilla code so you add add-ons like you do in firefox.

gtkpod is another application you can use, haven't really used it myself though. If you want to install it type the following into a terminal ```
sudo apt-get install gtkpod
```
