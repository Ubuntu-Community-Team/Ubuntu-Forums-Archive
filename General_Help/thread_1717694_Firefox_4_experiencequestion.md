### Post by lithopsian on 2011-03-30
> In my humble opinion, Firefox4 is as good on Linux as it is on Windows. It was not rushedI guess that some people are just so blinkered that even facts pass them by in a haze of Ubuntu beans ...

There are numerous features of Firefox 4.0 that were explicitly dropped from Linux versions for a variety of reasons.  Want tabs in the title bar?  Tricky.  Want all the hardware acceleration features?  Maybe in 5.0.  Proper Firefox button?  No idea why not, but it isn't there.  The list is endless, but the average user would never notice unless they had both versions running side by side or were looking for some very specific feature.  The reasons have been described in various blogs but basically Linux is not the most important target for this release so anything difficult tended to get pushed into a later release.

Aero came up because it provides a great many visual tweaks such as the OP describes wanting.  If so then that isn't going to happen.  Linux Firefox is based on Gtk and you'll have to get your visual tweaks that way.  There are several themes offering imitations of other OSes, and possibly one of them will work for you.
