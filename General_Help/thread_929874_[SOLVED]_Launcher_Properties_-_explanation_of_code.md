### Post by ryanhaigh on 2008-09-25
The URL in this case probably doesn't refer to an address on the net but perhaps a path to a document. When you right click on Open Office Word Processor in the menu there are options to add the launcher to the panel/desktop. If you do that you can then drag a file from your home directory (or anywhere else) onto the launcher and it will open that file in OO Writer. You can probably also drag it to the laucher in the menu but it is easier to demonstrate if it is on the desktop or panel.
