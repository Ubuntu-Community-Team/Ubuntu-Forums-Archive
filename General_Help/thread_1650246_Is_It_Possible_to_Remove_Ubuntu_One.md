### Post by Rubi1200 on 2010-12-21
Did you use the service at all? 

In other words, are you likely to have data or passwords stored?

I have never used the service, but removed it using Synaptic without any issues whatsoever.

This is what I uninstalled using the option to completely remove:

> python-ubuntuone-client
python-ubuntuone-storageprotocol
ubuntuone-client
ubuntuone-client-gnome
libubuntuone-1.0-1
python-ubuntuone
rhythmbox-ubuntuone-music-store

As I said, have had no issues as a result.

Whether you remove the other stuff is up to you or you can wait for some more opinions.

Hope this helps.
