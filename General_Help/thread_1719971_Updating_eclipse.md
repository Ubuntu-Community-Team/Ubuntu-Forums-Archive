### Post by GregBrannon on 2011-04-02
Download the distribution from the Eclipse web site, unpack it into a directory that you have control of, usually a directory off your home folder is best, and build a shortcut to the Eclipse executable file in the resulting Eclipse folder.  When you run it the first time, you can specify that it use the workspace you already have created for the previous version of Eclipse you've been using, or you can create a new workspace.  Your old and new version can share workspaces (one at a time) just fine, if you decide to keep your old version. It'll work great with no muss or fuss, but come back if you need help.
