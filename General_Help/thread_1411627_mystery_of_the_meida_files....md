Philosophical differences between Microsoft and *nix...

Microsoft design philosophy was based on the one-thing-at-a-time idea...

one person
one program
one operating system
one machine

(or put it another way....
´it`s my ball, you can´t use it, you can´t look at it, you can´t touch it`)

*nix was based on the one-or-more, let`s-all-share idea

one (or more) person
one (or more) program
one (or more) operating system
one (or more) machine

(or put it another way...
`it´s my ball, but I´m sharing, sou you can use it, look at it, and play with it´)

It´s all about sharing.... :D
