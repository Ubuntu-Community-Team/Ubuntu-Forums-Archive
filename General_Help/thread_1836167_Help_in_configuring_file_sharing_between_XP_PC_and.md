### Post by Morbius1 on 2011-08-30
If you absolutely positively need to transfer files the fastest setup is Dukto because there isn't any:

Go to: [http://code.google.com/p/dukto/downloads/list](http://code.google.com/p/dukto/downloads/list)

* Download the DuktoR4-Linux.tar.gz to Linux
* Download the DuktoR4-Windows.exe to Windows

* On Linux just double click the *.gz file and have it extract to your desktop and double click the Dukto icon.

* On Windows double click the DuktoR4-Windows.exe icon.

One should see the other in their respective application. From Linux just right click the Windows entry and select "Send" - you can then select what you want to send to WinXP.

[COLOR=Blue]**EDIT**[/COLOR]: I would suggest Samba for a more permanent setup but it sounds like time is of the essence here and Samba can take a bit if it doesn't work out of the box.
