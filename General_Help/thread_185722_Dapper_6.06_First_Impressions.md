### Post by Spacecaptain on 2006-06-01
I have been using Dapper on my home computer since several months (flight 5 & 6). I did a fresh install, just to clean up on all the orphaned libraries that i had on the drive since i first intalled Ubuntu back in the ages.
The install went flawless. I used the Install Iso, not the LiveCD. Then i used BUMP to get the multimedia stuff up and running as fast as i could. (Automatix for Dapper wasn't released yet)
The only problem i had is that it detected my cable-modem, ethernet and wireless in random order at every startup, wich was annoying as i only have connection on the cable-modem. Eventually i just removed the ethernet and wireless card.

One other point that annoys me is that i cannot get my apache to serve php pages. My browser keeps offering to download the pages as files. I have been searching and googling to find a solution, but no luck yet...

appart from that, Dapper is very smooth, has a lot of effective and fun applications. I like OpenOffice better as days pass by... yesterday i found Rezound in the repos (great audio editor)... in a nutshell i like it!

...have to get apache running properly!
