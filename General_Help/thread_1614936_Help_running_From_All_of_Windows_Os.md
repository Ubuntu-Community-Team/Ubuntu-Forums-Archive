### Post by sikander3786 on 2010-11-06
Hi and Welcome to the forums :popcorn:

You want to install Ubuntu to your 7110GX laptop?

The best way of testing the hardware would be to pop in a Live CD and select try without making any changes. See if all your hardware is working. If this laptop has wifi, the drivers might show up in System > Administration > Hardware Drivers/Additional Drivers once you connect internet via an ethernet cable.

Look for any issue you might get, if everything goes well we will be happy to advise about installation.

Regarding VLC and open source, you already got the point :-) Open source is way better but it actually depends on your needs. The one that satisfies you is the best.
