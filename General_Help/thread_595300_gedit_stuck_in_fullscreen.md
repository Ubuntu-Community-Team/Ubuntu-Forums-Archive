### Post by ed-koala on 2007-10-28

Show panel menu - Alt + F1

Unmaximize window - Alt + F5

Resize window - Alt + F8

Toggle fullscreen mode isn't mapped in my shortcuts by default, but that would be another good one to set up.

Hope it helps!
