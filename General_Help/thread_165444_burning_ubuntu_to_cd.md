### Post by localzuk on 2006-04-24
The speed at which it copies at is normally determined by a number of issues:

1. CD Reader speed - not really that important now-days
2. Hard disk speed - a laptop HDD will be a lot slower than a desktop hdd
3. CPU speed
4. Amount of available RAM.

Copying from the HDD would likely be the same speed.
