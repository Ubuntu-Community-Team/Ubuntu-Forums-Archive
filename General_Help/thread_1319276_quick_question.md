### Post by GO%)$@*1 on 2009-11-08
Seishuku is right. If you have a program like WinRAR installed, you may see an WinRAR icon on it which basically means you can use that type of rile extension with it. To be sure in Vista go to Folder Options (which is somewhere in Control Panel). Click on the 2nd tab of the Folder Options window. Scroll down and deselect "hide file extensions for known file types". Click OK and then check the supposedly RAR/ISO you downloaded.
