> **Yaser. said:**
> thank you
but I think in system tray is better because the icon will disappear after exit from the program

I'm still not sure from your statement how you want this to work. When you launch an application, it will appear in the Task Bar area of the panel as either an icon or an icon+text depending how you have configured your panel.

The System Tray normally includes applications that are running all the time (like the network manager) so you can click on these to change settings.

Some applications (like VLC Media Player) seem to load in the Task Bar AND in the System Tray. I don't know how to do this, and I'm not sure there is any benefit.
