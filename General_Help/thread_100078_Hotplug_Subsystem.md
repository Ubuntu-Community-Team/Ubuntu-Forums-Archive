### Post by purdy hate machine on 2005-12-07
This does seem to be a common problem on a lot of laptops and I have not seen a fix for it yet. This happens to me in about one in every ten boots. You can halt the process during startup by pressing Crtl+C when you see Starting Hotplug Subsystems, you should then be able to boot but this will most likely disable something that was trying to start up (in my case the sound)
I was hoping that this problem would be fixed in Breezy but it seems not :(

### Post by eivind on 2005-12-08
This is a common problem, which regularly occurs on my machine (a Dell Latitude LS laptop) too. I, and others, have posted bugs about this, but nothing has happened so far. I guess it's hard to pinpoint, and I don't know exactly what causes it. What I *do* know, is that the problem can be remedied on my machine (sometimes, but not always) by plugging out the wireless network card before turning on the computer. 

In other words: 
- Try to unplug something.
- File another bug, or confirm those that are already there
- Wait and hope...
