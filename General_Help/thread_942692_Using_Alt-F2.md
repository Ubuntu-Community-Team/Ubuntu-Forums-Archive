### Post by Denestria on 2008-10-09
You can set it up in System Settings > Keyboard & Mouse > Keyboard Shortcuts
Under Global Shortcuts scroll down to Desktop and remove the alt-f2 short cut from Run Command.  Then click on the Command Shortcuts tab, open either system or utilities to find Konsole and then set the shortcut.
