### Post by delcypher on 2009-06-18
As plucky explained you can change the theme used in 
System > Administration > Login Window and then select the 'local' tag.

If you fancy trying some new themes pop over to [http://gnome-look.org/index.php?xcontentmode=150]("http://gnome-look.org/index.php?xcontentmode=150").

There are some very nice gdm themes (login themes) available.

Good luck.
