### Post by CatKiller on 2009-05-27
There is a kubuntu-experimental PPA repository that gets updated with newer versions, but when I tried it I found the newer beta version to be significantly worse than the version that came with Jaunty.

Normally packages are only updated with security fixes. Feature releases aren't generally included until the next Ubuntu release. There have been a couple of exceptions in the past, and some packages make it into the backports repository.
