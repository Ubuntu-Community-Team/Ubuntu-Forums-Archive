### Post by Nepherte on 2009-03-08
According the aMSN's wiki: [http://www.amsn-project.net/wiki/Installing_Plugins_and_Skins](http://www.amsn-project.net/wiki/Installing_Plugins_and_Skins), you have to copy the plugins to one of these directories:
```
/usr/share/amsn/plugins
/home/yourusername/.amsn/plugins
```
For skins it's:
```
/usr/share/amsn/skins
$HOME/.amsn/skins
```

Bittorrent is a command line program. If you want a gui for it, install bittorrent-gui. Other torrent programs are: deluge, transmission, vuze, ... Pick one.
