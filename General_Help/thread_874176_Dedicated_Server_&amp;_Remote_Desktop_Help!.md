### Post by ibutho on 2008-07-29
The server version does not have a GUI by default (unless you already installed one yourself or the people who sold you the server did it for you). You can install an application like [webmin]("http://www.webmin.com") to allow you to configure the server from a remote location using a web browser. It would also be a good idea to get to grips with the command line if you are going to be administering Linux or Unix servers
