### Post by jamesdodd on 2006-03-25
As well as being able to change the theme:
system >> preferences >> theme


You can also change the logon screen which should get rid off all the brown
system >> administration >> Login Screen Setup


There are several nice places on the net to get gnome themes including:
[art.gnome.com]("http://art.gnome.org/"), [http://gnome-look.org/]("http://gnome-look.org/")

Have fun playing
