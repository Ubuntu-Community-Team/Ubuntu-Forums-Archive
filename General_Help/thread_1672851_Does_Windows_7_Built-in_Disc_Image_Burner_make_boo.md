### Post by 3Miro on 2011-01-21
I tried it only once, but the Windows 7 inbuilt burner made a fine disk for Ubuntu (then I killed Windows 7, you had to see the grin on the penguin's face).

I don't think the bootable disk thing is a feature of the software used to make the burn or the .iso image. I think that any burner can make a bootable CD so long as it is given a bootable .iso. You may run into trouble if you want to make a bootable disk without an .iso.
