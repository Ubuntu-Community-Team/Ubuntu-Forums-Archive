### Post by LowSky on 2010-05-24
When you set up an account on Evolution, Thunderbird, Outlook, etc., Make sure to check the server options, there is an option for leaving the messages on the server, forever or for a set time limit. I set my messages to auto expire after 90 days, as I never have anything that important. You can set it to whatever. For people who have to have their messages always available, don't use a client, just use the website. Set up a desktop icon that goes directly to their inbox. I did that for my mother and she thinks its just her email program (my mom isn't so PC friendly). Why add another program to the mix if you really don't have to?
