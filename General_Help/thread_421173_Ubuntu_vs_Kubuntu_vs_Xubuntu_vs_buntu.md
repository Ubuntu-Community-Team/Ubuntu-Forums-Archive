### Post by soulfly7x on 2007-04-25
It can be difficult to explain the differences. They use different desktop environments. Ubuntu is GNOME. Kubuntu is KDE and Xubuntu is XFCE. Edubuntu is GNOME with a KDE education software package. So what's the difference between GNOME, KDE, and XFCE? Well, that's the hard to explain part. 

What I suggest is to try them all out on a live CD and see how it interfaces. How to change your theme. How to change system settings and defaults. There's no commitment with a live CD other than the time investment. Also, make sure to look at what the default programs are in each. If there are ones you particularly like that aren't default, you can still get them, but you might want to ask about how well they work. How well does GIMP work in KDE? How does Amarok work in GNOME? More specific questions like that will get better responses.

I also saw it mentioned that you can always download a new Desktop environment. This is true too. I have several installations. On this PC I dual boot Kubuntu Feisty and Ubuntu Edgy. On another I have Ubuntu Feisty with the full KDE package installed. At the login screen you select Sessions and switch it to the one you want, then it will ask if it's just for that session or if you want to set it as the new default. 

At first I liked GNOME better, but now I'm starting to like KDE better. It seems to have fewer problems with my printer and Bluetooth seems to work better, but that's just my experience. Just experiment and don't save important things on your installation until you make a set decision on what exactly it is that you are comfortable with. I've been with Ubuntu 2 or 3 months now and I'm still doing a lot of experimenting.
