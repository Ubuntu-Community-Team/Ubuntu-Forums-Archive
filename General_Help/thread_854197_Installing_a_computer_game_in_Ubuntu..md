### Post by pedro_orange on 2008-07-09
.exe files are native Windows executables.

You cannot run .exe files natively in Linux.

However! You can install something called wine, which will run Windows applications for you.
If you look in the wine section of these forums there is a how to that tells you how to get the latest wine. 
(I would recommend not just doing a sudo apt-get install wine as you get an older version)

[www.winehq.org](www.winehq.org) is the main site for the wine project. Go into the appDB and check out what it says about Orange Box. From what I hear it runs on Wine very well.

Good luck in your endeavors.
