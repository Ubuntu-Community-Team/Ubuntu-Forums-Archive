### Post by diablo75 on 2009-10-28
I would shoot for Edubuntu, since it's intended to be used by kids. This will install some educational games (math, reading, etc) that they might like.

I don't believe any one distro is "more" stable than another, but then again, KDE and GNOME are two different animals.  GNOME is the more stable of the two, so I would lean towards Ubuntu, Edubuntu.  Kubuntu uses KDE.  Xubuntu uses XFCE (a lightweight desktop that is intended for older systems).

As far as Flash/Java goes, I've never had a problem with either so long as they were installed properly. There is a free version of Java that is not as solid as the proprietary version released by Sun.  As long as you have the restricted repositories enabled, you can install both flash and java with Synaptic or from the command line.
