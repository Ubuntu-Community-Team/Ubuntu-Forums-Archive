### Post by cmost on 2007-02-05
Your best bet is to do an export from MS Publisher to EPS, PDF or SVG format.  From there, something like Scribus should be able to open it.  You should be advised that there is no known way to work with MS Publisher files natively in Linux.  If you plan on using them a lot, then I recommend you purchase a copy of Publisher and run it within a copy of Windows that you run in VMWare, Parallels, QEMU, or Xen.  This is how I run proprietary software and it works out just fine.  Scribus is VERY good.  It can do anything Publisher can do and so much more.  Try weening yourself off of Publisher and breaking free of MS for good.  Good luck.
