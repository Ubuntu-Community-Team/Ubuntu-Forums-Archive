### Post by 3rdalbum on 2011-04-20
You generally only really need your data backed up if you are doing any partitioning (which Wubi doesn't do), but wouldn't you kick yourself if something went wrong and you didn't have a backup?

I've never heard of Wubi causing data loss, but if you're worried you should back up your important data anyway.

---

### Post by Rubi1200 on 2011-04-21
As both wilee-nilee and 3rdalbum have said, you should make backups anyway.

I have never heard of Wubi causing data loss in Windows.

What you should be more concerned about is Windows causing problems for Wubi.

To help prevent this use an external medium such as a USB stick to make a backup of the Wubi root.disk once it is installed.
