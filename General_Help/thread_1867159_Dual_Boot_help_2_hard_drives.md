I have 2 SATA hard drives, one with Windows Vista, and another with my "Program Files" folder on it (160gb and 300gb). So I just want to know: Can I install ubuntu or any other linux distro to the hard drive that doesnt have the windows boot information on it, and then manually choose my boot device on startup?

---

### Post by raja.genupula on 2011-10-22
yes first install the Ubuntu in which drive you want and then in your BIOS settings set first boot priority to the hard disk in which you have installed Ubuntu.

Then every time from the GRUB menu you can choose Ubuntu and windows to boot.

All the best.
