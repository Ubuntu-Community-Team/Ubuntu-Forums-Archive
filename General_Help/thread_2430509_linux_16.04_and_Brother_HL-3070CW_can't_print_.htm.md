### Post by cmcanulty on 2019-11-04
If you are using firefox you can just go to Tools-AddOns and then search for print to pdf or print friendly and add one . Then there is an icon in the address bar and click on it and print. You can also choose File-Print-print to file and name the file location where you want it printed. Theprint to PDF addon is super easy to use.
