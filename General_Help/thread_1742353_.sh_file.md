### Post by doas777 on 2011-04-28
I haven't heard of your game. be sure that the content is legitimate/legal before you run any scripts from the web. 

.sh files are called "shell scripts" and contain commands like those you would run in a terminal.

to run them, they must be first marked as executable:
```
sudo chmod +x <path to file>
```

to run the script, open a terminal (Applications -> Accessories -> Terminal)
and use cd to navigate to the location of the file:
```
cd <path to file>
```

then enter:
```
./<filename.sh>
```
