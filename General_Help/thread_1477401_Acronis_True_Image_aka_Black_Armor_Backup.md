### Post by clappr on 2010-05-08
If anyone has trouble backing up a Linux EXT3 or Swap partition to their Seagate NAS drive, I discovered that Acronis True Image does not work well with an EXT3 file system.  What worked for me was doing a **[COLOR="Black"]sector by sector[/COLOR]** backup of those partitions.  

I found this answer by going to the operating guide .pdf file downloaded from the acronis website.  FYI Acronis is the Licenser of Seagate Black Armor software.
