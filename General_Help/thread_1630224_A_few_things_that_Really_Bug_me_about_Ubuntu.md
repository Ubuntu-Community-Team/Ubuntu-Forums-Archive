> since we're in b#tch mode this really bugs me: 

its much harder to resize windows by dragging borders in ubuntu than it is in windows or mac. in windows you have to place the cursor in GENERAL AREA of the border and it understand that you want to move the border and changes the cursor accordingly. in ubuntu you have to land the cursor in between a 0.000000000000001 mm area before it understands your request.

Many GUI windows have an area (dimples - dots) (lower RH corner) which is easy to grab with a mouse and **_R_**esize.

Another super fast easy way to **_R_**esize is with Alt+Spacebar then R for _R_esize, then use an arrow key or move mouse to _R_esize window.
