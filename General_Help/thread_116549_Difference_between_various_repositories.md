The first one is "main." These are officially supported by Ubuntu, and are usually very high quality.

The second one is the "multiverse" (as you may have guessed ;)). That one is unsupported, and contains stuff that doesn't have a "free" license. It's usually okay to use this stuff, just don't install things randomly! :)

The last one is "universe" (I must be a genius, how'd I guess? :D), and that's community supported, and it has a "free" (open) license.

They're all "official" repositories, just not necessarily supported by Ubuntu (like, if something blows up, Ubuntu/Canonical won't help you. But don't worry. People never have those problems ;)). You can use them to install all kinds of stuff (free!). If you have all the repositories enabled (multiverse, universe, main), then you can go to "Add Applications" (Applications-->Add Applications) and install stuff really easily. You don't need to worry about where it came from if it is in there, because it's generally good stuff.
