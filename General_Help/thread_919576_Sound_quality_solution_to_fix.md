Hello, i found that if you set PCM db gain to greater than 0 .. it'll create distortion and lower sound quality when listening to song with good bass .. to fix it:
1. Open a terminal
2. alsamixer
3. lower PCM db gain to 0.00, 0.00

hope it fix some problem with users :)
