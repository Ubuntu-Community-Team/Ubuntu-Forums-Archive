### Post by kebes on 2007-02-20
If you want to erase Windows and only have Ubuntu, then just run the Ubuntu installer and select "Format entire disk and Install Ubuntu" when you reach the partitioning step. The Ubuntu installer will do everything for you.


Note, however, that a 486 computer will run Ubuntu rather slowly. You may want to consider Xubuntu instead (which uses Xfce as a desktop, so it runs faster).
