### Post by pjbgravely on 2006-06-15
Boot into the failsafe mode in grub. Get to a console and type:

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.config.crash

Then:

Sudo nano /etc/X11/xorg.conf

In this section:

Section "Device"
Identifier "NVIDIA Corporation NV4 [RIVA TNT]"
Driver "nvidia"
EndSection

Change it to this:

Section "Device"
Identifier "NVIDIA Corporation NV4 [RIVA TNT]"
Driver "nv"
EndSection

You should then be able to reboot and have a GUI. If that doesn't work then maybe your video card has gone bad.

Paul
