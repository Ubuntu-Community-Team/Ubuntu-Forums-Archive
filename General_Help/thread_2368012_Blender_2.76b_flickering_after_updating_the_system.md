### Post by dmbomer on 2017-08-05
Thought I would share this work around I found.  

Issue I encountered after updating Ubuntu Studio 16.04

Blender 2.76b GUI would flickering and distort

[https://developer.blender.org/T49537](https://developer.blender.org/T49537)
Second last post

File//User preferences//last tab System
Middle of the dialog box look for Heading "Window Draw Method" from the drop down select Triple Buffer
Save the changes 

For me the flicker gui stopped hopefully this will help others.  

Good luck
