### Post by prodigy_ on 2010-07-28
You can select the whole drive. Just be careful - DON'T select the wrong drive (the one where you Windows is installed). If you'll make this mistake, your Windows installation will be overwritten.

For more info, read [this HOWTO](https://help.ubuntu.com/community/GraphicalInstall).

### Post by prodigy_ on 2010-07-28
You have two 500GB drives and 1TB, correct? The one that is 1TB should be easy to recognize, because it has different size. But since you're installing on a 500GB one, it's possible to make a mistake.

So **make sure that you have chosen the right (empty) drive** before you proceed. And when you're sure, then yes, you can tell the installer to use all of it.
