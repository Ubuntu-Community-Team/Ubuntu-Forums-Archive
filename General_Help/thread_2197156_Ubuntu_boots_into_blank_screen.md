### Post by grahammechanical on 2014-01-02
This is what you can try.

1) at the first purple screen press Enter
2) at the language screen select the language (the default is English) and press Enter
3) at the Try Ubuntu - Install Ubuntu screen press F6
4) look for a small menu at screen bottom right and select nomodeset and press Enter
5) press Esc to get back to the Ty Ubuntu - Install Ubuntu
6) select Try Ubuntu and press Enter.

See what happens. You may need to try a combination of those F6 options. That is sometimes necessary.

Regards.
