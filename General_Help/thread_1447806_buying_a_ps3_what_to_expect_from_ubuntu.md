### Post by 3rdalbum on 2010-04-06
> **jrguillett2 said:**
> thank you agent me but i thought all linux arctectures run the same software guess not...

Different instruction set. PowerPC processors can't understand x86 instructions, no matter the operating system.

If the source code was available, you'd be able to compile it on PowerPC, but the PS3 is a very limited computing platform. (256 MiB of RAM and no access to the GPU). No Wine, no Flash, no Skype, no proprietary software, no Compiz, no OpenGL at all.

And Sony's latest firmware doesn't allow Other OS as well (don't be too hard on Sony; in the end it was games developers who threatened a boycott of the PS3 unless Other OS was disabled, because the professional pirates had started to use it to work on copied games), so your best bet is to build a computer from parts, or get a cheap computer from a shop.
