### Post by KillerSiafu on 2010-09-21
I recently did a clean install on my media PC to bring it up to 10.04. After the installation, it asked me if I'd like to install proprietary Nvidia drivers, which I did. I used the latest nvidia_current.

When I went to the Nvidia settings manager, it displayed:

> You do not appear to be using the nvidia x driver. Please edit your x config file (just run 'nvidia-xconfig' as root), and restart the X server.

I of course tried that but still received the same error when I tried to open nvidia's settings. I also tried to launch my media center software, XBMC, and received the following error:

> XBMC needs hardware accelerated OpenGL rendering. Install an appropriate graphics driver.

Before this clean installation, my computer was running 8.04 LTS, using Nvidia drivers and XBMC without any issues.

I've looked around through all the forums and tried many solutions. I've disabled and reinstalled the drivers, reverted to an older set of drivers, and tried running with the default, non-nvidia drivers... all to no avail.

I also tried simply removing my xorg.conf file rebooting.

I've also tried the following:

```
sudo apt-get install linux-headers-$(uname -r)
sudo apt-get install nvidia-current
sudo update-alternatives --config gl_conf
sudo ldconfig
sudo update-initramfs -u
sudo nvidia-xconfig
sudo reboot
```

After I ran that, I then went in an activated the nvidia_current driver in System -> Administrator -> Hardware Drivers and rebooted. It's now saying that the Driver is activated but not in use.

Finally, I tried the following with no luck:

```
sudo dpkg-reconfigure -phigh xserver-xorg
```

Anyone have any ideas / suggestions?
