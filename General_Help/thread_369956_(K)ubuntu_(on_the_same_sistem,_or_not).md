### Post by raja on 2007-02-25
Kubuntu and Ubuntu are the same base system with the difference only being KDE vs Gnome.  You dont have to install the full Kubuntu system if you have Ubuntu. You can just install 'kubuntu-desktop' with aptitude and then KDE will be available as another session when you log in.
