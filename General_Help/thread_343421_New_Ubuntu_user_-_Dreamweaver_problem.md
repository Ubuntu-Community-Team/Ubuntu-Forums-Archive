### Post by scrook on 2007-01-21
Vmware creates a virtual PC for you to install windows in and you can then run windows (with dreamweaver installed) as a window in Ubuntu.

To get started with this, the easiest thing to do is to go to vmware's website and download the trial of their vmware workstation software. While this is for pay software, you'll only need it for about an hour. All you need to do is set up an image to install windows in and go through the install. 

Once that is complete, go back to vmware's website and download their free vmware player. This is what you will use to launch windows from Ubuntu from here on out.

Also, ubuntu has vmware player in the repos. The last time i tried to use it (back before edgy was released) it didn't work for me. I don't know if this package is still broken as ever since i've been going straight to vmware's website for this.
