### Post by CSEatOSU on 2008-05-01
If you download the iso for Ubuntu 7.10, and place it in the same directory as the Wubi installer, when you run the Wubi installer it will automatically detect the 7.10 iso and will install that version.

Don't be bothered that the Wubi page says you're downloading Wubi 8.04, it is only an installer and will use your 7.10 iso for Ubuntu installation.

I have a 64-bit machine but wanted to install the 32-bit version of 8.04 (Oracle doesn't run (very well) on 64-bit).  When I first installed 8.04 via Wubi, it auto-detected my 64-bit machine.  I had to redo the installation with the 32-bit iso.
