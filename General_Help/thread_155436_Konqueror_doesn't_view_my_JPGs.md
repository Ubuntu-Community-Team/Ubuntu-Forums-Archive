### Post by magnusbb on 2006-04-05
I've encountered a very strange problem in Konqueror; whenever I step into a directory with a JPG-file, 

1) Konqueror has no association with it (meaning that when double-clicking the file it doesn't know what to do), 

2) and Konqueror wouldn't even show a thumbnail of the file (which it normally does).

I am using the very latest KDE 3.5.2 from the Kubuntu packages for Breezy.

**SOLVED:** It was really easy to fix. Just had to enter the .kde-directory in my home folder and delete the mimelnk-folder.
