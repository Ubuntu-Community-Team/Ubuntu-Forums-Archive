### Post by Dennis N on 2016-06-15
> **J_Harp said:**
> My install of gthumb shows the oldest catalogs and images first. Can it be set to show the newest first?

There is a "sort by" option in the menu. Many ways to sort shown there, including date taken or file modified date.

And to point out the obvious: Check the "Inverse order" box to change the order.
