### Post by coldcritter64 on 2014-12-08
What sort of hardware specs do you have ? What brand/model of printer ?

With XP being unsupported and your need only for it to run a printer, it may pay for you to consider running XP in a virtual machine (like virtualbox). That way you can use the virtualbox settings to keep XP offline internet wise for security (you can temporarily enable for any updating etc) but you should then be able to use the printer from it.
>   Can I re load Windows from the disc and have it go dual boot???
Using a VM saves you having to repartition for dual booting. Though your hardware does need to have sufficient resources and BIOS support for such virtual machine usage. If you post back hardware specs someone will let you know if a virtual machine use is possible.

Edit: an info link on virtualbox : [https://help.ubuntu.com/community/VirtualBox](https://help.ubuntu.com/community/VirtualBox)
