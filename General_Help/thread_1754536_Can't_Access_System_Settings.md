### Post by grahammechanical on 2011-05-10
There is a second way to access System Settings and that is from the launcher panel on the left. Click on the applications icon (the + sign in a magnifying glass). Click on the word Installed and all the installed programs and utilities will appear as icons. System Setting is in there. It has an icon of a computer screen with a mauve background and a spanner with a red handle lying across the screen.

Regards.

### Post by Rasa1111 on 2011-05-10
well I don't know if this will work for you...
But it worked for me when I messed up my 11.04 desktop , So I will share, Just in case.

Do you have Ubuntu tweak installed? 

If not, install it. 

Once installed, Open it up, 
and look in the left side panel for "Desktop"

Then in that list, you'll see "Desktop Recovery" 
Like this~ [ATTACH]191778[/ATTACH]
Once you click on that,  you'll see 3 buttons at the bottom. 

Recover, Reset, and Backup. 

When I first did this, I had no "backup" made, 
But I clicked the "Recover" button anyway, Just to see...
and to my surprise.. all the fudge-ups i had made, reverted back to a pristine working 11.04 desk.  And Everything was back to normal. 

So then i DID make a backup, for the next time, just in case. 

 I think,
 that when I first did it, it reverted me back to the default settings of 11.04. 

So be prepared to *possibly* lose any custom settings you may have made. 

But it should set things back to working order again. 

Good luck.
