### Post by Vadi on 2008-06-15
I'd recommend webilder, it fetches nicer wallpapers. Get it from here ([click]("http://www.getdeb.net/search.php?keywords=webilder")), then right-click on your panel, and add the webilder applet.

### Post by cariboo on 2008-06-26
It might be easier if you used Desktop Drapes, it doesn't download a new wallpaper from the internet every day, but I have a wallpaper directory with over 500 wallpapers. you can set the interval that a change takes place. Drapes is available in the repositories and you can install it using synaptic.
