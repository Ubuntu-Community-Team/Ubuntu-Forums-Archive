### Post by t0p on 2011-04-11
Open a terminal.  Click on **Edit >  Profile Preferences**. Untick the box by where it says "Use the system fixed width font".  Now click on the name of the font that was greyed-out before (mine says "Monospace 12".  A window will appear called "Choose a terminal font", and lots of fonts in a list.  Choose whichever font you like.
