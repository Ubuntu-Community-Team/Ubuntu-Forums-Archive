### Post by MarjaE on 2012-07-05
You should have a set of LibreOffice preferences, as a hidden directory in your user directory. Using the software center to delete and reinstall something won't delete the preferences. But deleting the preferences, or moving the preferences into another directory, should prompt LibreOffice to create a new preferences [or .libreoffice] folder with the defaults.

I hope this helps.
