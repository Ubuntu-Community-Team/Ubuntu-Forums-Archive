### Post by tim1 on 2005-02-26
The error you got seems to be a problem with the livecd. But you still won't be ableto watch dvds with a Ubuntu standard install, because some components needed fo DVD playback are not included because auf licensing issues. If you install totem-xine and libdvdcss2, everything should work without hassle.
