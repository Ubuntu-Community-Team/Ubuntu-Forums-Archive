EDIT: you can also use split: ```
 split -b 500M myfile.rar

### Post by mssever on 2008-08-24
split will split any file. See man split for configuration options. You can use ```
cat segment1 segment2 > output.7z
``` to reassemble the files. If you have to reassemble them on Windows, I don't know how to do that, unless you install Cygwin and use cat from there.
