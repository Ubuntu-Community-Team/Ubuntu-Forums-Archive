### Post by HermanAB on 2018-06-21
If you simply booted the installation ISO file, then your document went to the big bit bucket in the sky - may its soul rest in peace...

The installation ISO file provides a test and evaluation system which is not persistent.  To make a proper usable system, you need to actually do the install process.
