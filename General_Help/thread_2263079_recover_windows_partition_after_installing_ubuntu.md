### Post by Mark Phelps on 2015-01-29
There are a couple of possibilities -- recovering files and folders, or recovering the former partitions.

For the first, you'll need access to a Windows PC, be able to connect your drive to that PC, and then download and install "recuva" -- to see if it can find any files and folders.  IF that doesn't work well, you could try "RecoverMyFiles" -- but that is only free in a trial version.  You will be able to see what it finds, but you have to purchase a license to do the actual recovery.

For the second, you'll need to download and burn a CD from the Minitool Partition Wizard Boot CD (Windows partitioning tool), boot from that CD, and see if it offers you the option of restoring the former partitions.
