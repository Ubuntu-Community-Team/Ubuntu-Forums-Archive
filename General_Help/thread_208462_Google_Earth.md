### Post by DirtDart on 2006-07-03
You get that message (dunno if I could call it an error), because you are using software rendering to use Google Earth.  

You've problably noticed that it is slow as death.  That's because the program has to do all of the OpenGL stuff in software, as compared to hardware.  Hardware is always faster than software.

If you have a Nvidia or ATI card, then I'd recommend installing the drivers for it, as that will allow you to do the rendering in hardware.
