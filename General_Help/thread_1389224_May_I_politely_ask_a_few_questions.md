### Post by amsterdamharu on 2010-01-24
Sorry I can't answer all your questions but here are some answers:

1)... I don't want to return back, (Windows)
Careful, you might find yourself needing Windows at some point because of software that doesn't run on Linux or hardware that is not fully supported

9) Set your Ubuntu to use the swap partition in /etc/fstab file open console and issue the following commands

```

ls -l /dev/disk/by-uuid/ #you can see the UUID of the partition you made for swap
gksudo gedit #this will open fstab in gedit as root
 
```
edit the fstab to contain the following line:

```

UUID=our_UUID_IN_FIRST_COMMAND none            swap    sw  0       0

```

Save the fstab and reboot.

Check if swap is used with the following command:
top

### Post by amsterdamharu on 2010-01-25
If sda2 is your swap partition (make sure it is, open it in a file browser and see if it's empty) then add this in your /etc/fstab:


UUID=190fd4d7-1ab0-463a-8ba4-98780b0c922c none            swap    sw  0       0

Then reboot.

our_UUID_IN_FIRST_COMMAND should be your_UUID_IN_FIRST_COMMAND
The first command shows the UUID of the partitions that way when you change your partition the right one will still be used for swap (unless windows install messes with the UUID)

### Post by nothingspecial on 2010-01-26
Well to start with, can I suggest mpd - music player daemon

The great thing about mpd is it is a daemon that uses a number of front ends. So, when you`ve got plenty of ram free you can use sonata, a gui front end. And when you need a little bit more for other things you can switch to ncmpc.

```
sudo apt-get install mpd sonata ncmpc
```

mpd will start when you log in. [Here]("http://mpd.wikia.com/wiki/Configuration") is how to configure it.

There`s quite a bit of information there but all you need to do to start with is tell it where your music is.

Then launch sonata and away you go.

When you want to try the terminal front end, type ncmpc. If you press 1 it will tell you how to use it. 2 will show you your playlist. 3 will let you browse your collection. It`s pretty straight forward. Have a go if you fancy it.

Any problems let me know.
