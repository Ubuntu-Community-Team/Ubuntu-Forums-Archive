### Post by Smart Viking on 2010-06-02
Disconnect the hard drive, and then boot from the CD, if that works, next time you should have the hard drive disconnected and the live CD inside, and at once you see the the machine is booting from the CD, plug in the hard drive, and from there you should be able to install ubuntu again.

(i have had to do that sometimes, like the hard drive wont allow booting from CD regardless of whats set up in BIOS)

Also, what is the error it gives you when you enter the CD? That is important.
