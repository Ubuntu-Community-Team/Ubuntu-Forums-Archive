### Post by Chronon on 2009-09-03
That sounds a lot like a font problem.  This happens quite often even passing files between Windows machines using the same version of Office.  The missing font will be substituted and you end up with all kinds of odd character substitutions (or a box sometimes).

It's kind of poor form, in my opinion, to embed formulas at all because of this quite common problem.  A workaround is to render the formulas locally and convert them to images that can be embedded in the presentation.  If you're in contact with the author you could explain this problem to them and have them replace the formulas with images instead.
