You can get an extension from the GNOME Shell Extensions website to display either "VPN is up!" or "VPN is down!". No icon so the extension is rather basic.

[https://extensions.gnome.org/extension/1134/vpn-indicator/](https://extensions.gnome.org/extension/1134/vpn-indicator/)
