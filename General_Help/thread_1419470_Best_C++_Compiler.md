### Post by falconindy on 2010-03-02
> **FinalShot said:**
> Ok thanks, does Vim have syntax highlighting and auto indentation?

I'm serious when I say Vim does everything. It will do every last thing you could possibly want from a text editor -- syntax highlighting, auto indenting (of multiple varieties), line numbering, macros, tabs, split windows, code folding, global and local search/replace, filetype specific settings, multiple copy buffers, saved sessions, built-in compiler/debugger support... It's got its on API for scripting as well so the sky's the limit.

I happily write Bash, Java, C/C++, Python and Ruby in Vim.

I'm also serious when I say that the learning curve is vertical. With that in mind, it seems Code::Blocks is a little more popular and no doubt a little more user friendly.
