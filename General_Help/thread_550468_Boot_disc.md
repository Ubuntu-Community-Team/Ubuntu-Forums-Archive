### Post by expatCM on 2007-09-14
Yes it is ... that is what the Live CD is for.  But ... please remember that normally when you power off the machine, all the settings you have made in that session are lost.

If you wish to boot from a USB device you first need to check that your BIOS will allow this.  Older BIOS generally does not.

If you are wanting to run Linux from say a USB device and have a portable system then perhaps you may want to take a look at DSL which may be run from a USB Thumbdrive 
[http://www.damnsmalllinux.org/](http://www.damnsmalllinux.org/)
