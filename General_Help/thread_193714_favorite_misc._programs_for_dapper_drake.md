### Post by TeeAhr1 on 2006-06-10
I can think of several, but really, as you'll see in my descriptions, I think they're indispensible for *me,* because they do perfectly what I use my computer to do.  YMMV.

Tunes - [gmusicbrowser]("http://squentin.free.fr/gmusicbrowser/gmusicbrowser.html"): I have hundreds of hours of music, and I like search filters as opposed to playlists (I want them all listed, but with the ability to filter down if I want a paticular song).  This program does that perfectly.  It's the first thing I reinstalled after installing Dapper.

Torrents - Bittornado (no link included because it's in the repositories): Cuz I can't stand gnome-bittorrent, and don't have the system speed lying around to waste on Azureus.

Burning - Gnomebaker (repos): It's a great interface, very intuitive.  I have no particular beef with the burning software included by default, I just really like Gnomebaker.

Ripping - SoundJuicer (included): My needs are simple, so is the interface.  We work well together.

Game - Nethack (also in the repos): Once I got into Nethack, I realized that I'd probably never need to play another computer game again.  I still boot to XP to play Civ 4, but not very often anymore.

Office - Abiword/Gnumeric (both in the repos): OpenOffice is a pig.
---
I could include way more.  Really, it comes down to: What do you use your computer for, and in what way do you like to do those tasks?  Some stuff (SoundJuicer) I like simple as can be, some stuff (gmusicbrowser) I need the configurability and power of a more complex application.

Having babbled now for some little while, maybe this thread should be in Community Cafe?

### Post by akniss on 2006-06-10
digiKam - I like it slightly better than picasa for managing digital photos.  The programs are almost identical in functionality, but digiKam runs faster on my system, probably because the picasa is actually the windows version running on wine.

Scribus - for desktop publishing.  Clean, simple interface that does everything I need

Gnumeric - Spreadsheet program.  I like it MUCH better than OpenOffice.org Calc.

Bibus - bibliographic database manager.  Only useful if you write a lot of research papers.

Kmymoney - similar to Quicken or M$ Money, but I actually like it better.  Definately better than gnucash in my opinion.

Nvu - HTML editor for linux, Mac, and Windows.

For music, I'm torn between Beep media player, Listen, and amaroK.  I've been using Listen lately, since I installed the newest version with automatix.  The version in the repos does not allow me to minimize to the system tray, which is a fatal flaw for me.

R - for statistical analysis; another one that is pretty useless to the average user.

### Post by Skye on 2006-06-10
Hrm, let's see... (All of these should be in the repositories)

Music: Far and away, it's amarok for me.  I love amarok's ability to quickly manage large music collections, (3500+ songs) and I really really like the way the interface is set up.

Office Apps: Abiword- it's alot quicker than OO.org writer, but lacks of some of the features- I do like the way its set up, though, and it's perfect for most common writing tasks.

Misc: stellarium is a planetarium program that run on openGL.  If you have 3d acceleration, this program is worth downloading just to sit there and drool at
