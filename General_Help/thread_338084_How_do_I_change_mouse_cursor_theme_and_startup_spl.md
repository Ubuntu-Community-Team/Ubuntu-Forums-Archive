### Post by NeoLithium on 2007-01-13
For the mouse cursor theme, download a package you like ([http://www.gnome-look.org](http://www.gnome-look.org) has nice ones). Store them in your /home/username/.icons/ 
Then go into system -> preferences -> mouse, you should be able to select your new ones.

As for the startup splash, search through synaptic for gnome-splash, or something like that, if you mean the start splash for when you log into gnome; there's a nice program that sets what you like for that.
