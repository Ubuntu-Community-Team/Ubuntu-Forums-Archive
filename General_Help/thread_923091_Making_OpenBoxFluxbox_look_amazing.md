### Post by urukrama on 2008-09-18
If you want shadows and fading menus, you can use xcompmgr. See my Openbox guide (link in signature) for more info on how to set this up. If you use xcompmgr, you can also use a dock like Avant-window-navigator.

That guide also contains a lot of info on panels, Gtk themes, and ways to customize Openbox more.

You can find good Fluxbox and Openbox themes on box-look.org.
