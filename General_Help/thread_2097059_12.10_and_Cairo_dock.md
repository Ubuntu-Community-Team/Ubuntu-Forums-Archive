### Post by mc4man on 2012-12-22
> **gsak1100sc said:**
> How do I get the latest Cairo Dock to auto start with ubuntu 12.10 please ?
Peter in Gladstone SA

Open a terminal & 
```
mkdir -p .config/autostart && gedit .config/autostart/cairo-dock.desktop

```
copy & paste this to use the OpenGl version, then save
```
[Desktop Entry]
Type=Application
Exec=cairo-dock
Hidden=false
NoDisplay=false
X-GNOME-Autostart-enabled=true
X-GNOME-Autostart-Delay=4
Name=cairo-dock
```
The "X-GNOME-Autostart-Delay=4" is optional & can be set as seen fit, I use a 4 sec delay here with plank

To use the non gl version then make the Exec= line
```
Exec=cairo-dock -c
```

log out/in to test
