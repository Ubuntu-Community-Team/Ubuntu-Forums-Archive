### Post by ingramproductions on 2011-06-06
Dual X server mode will make 2 separate desktops, I believe. So that should let you have 2 desktop backgrounds, but also 2 full desktops, so that's probably not what you need.

---

### Post by seawolf167 on 2011-06-06
Enable your monitors and set then in System -> Preferences -> Monitors, then create a wallpaper that is the correct resolution to span both monitors at your set resolution then select to "tile" the wallpaper when setting it as the desktop background.
