### Post by coffeecat on 2009-11-01
If you're worried about losing Windows the way you have it and it's impractical to be able to reinstall Windows and all your apps, then be sure to get a disk image program - and use it.

A good paid-for one is Acronis, but if you want a free one, I can recommend from personal experience [Macrium Reflect]("http://www.macrium.com/reflectfree.asp"). It's a commercial product, but free as in beer, not free as in open-source.

With Macrium, you create images which you write to an external HD or DVDs. You also make yourself a bootable CD for restoring the image if you need to. Ironically, this is Linux powered. :)

Having said all that, if you are unable to reinstall Windows and all your paid for apps - for whatever reason - you are in a vulnerable position.
