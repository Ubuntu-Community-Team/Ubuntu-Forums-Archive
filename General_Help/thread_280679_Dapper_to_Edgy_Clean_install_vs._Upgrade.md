### Post by srunni on 2006-10-20
I know that Gentoo saves bandwidth by releasing 50 MB CD's that are used for a internet install. Basically, the CD starts the install but does not even contain the Linux kernel; it and any additional packages are downloaded as necessary after the install. However, Gentoo has a completely different user base from Ubuntu: most of the people using Ubuntu want it to "just work," and have all their applications ready-to-use, with everything working. Since different people use different applications, Ubuntu doesn't really have a choice when it comes to minimizing the size of the install CD. Unfortunately, the lack of technical understanding of many Ubuntu users means that they also lack the ability to configure BitTorrent, which would be another way of saving bandwidth.
