### Post by Darkwing-Duck on 2009-09-23
These mirrors are to be used in your apt sources for updates and installing programs via apt.

If you want to add in and use these mirrors you will need to change the /ect/apt/sources.list and include those mirrors.

Hope this helps you with it. They are not for downloading an ISO. If you want a list of where the ISOs can be downloaded please go to [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) and then choose the location you want to download from.

I hope this helps you.
