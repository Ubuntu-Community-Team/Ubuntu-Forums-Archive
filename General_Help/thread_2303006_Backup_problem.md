### Post by TheFu on 2015-11-15
Welcome to the forums!

Don't know what "system restore" is in ubuntu. On windows it is just a snapshot of the registry, drivers, etc,  but not any data. Definitely not a VM.

Restore the system backup that you made prior to the issue.  That's what I'd do.
Don't think there is a better answer, unless you did some fairly advanced file system things.  Sorry.

BTW, it isn't clear to me what OS the host machine is running. Any data recovery will need to use tools for that OS.
