### Post by crazymonkey on 2006-09-10
I have followed [this guide]("http://www.ubuntuforums.org/showthread.php?t=202605") for my samba shares and its working like a charm.

I'm using the following to define my shares : 

```

[MyFiles]
    path = /media/samba/
    browseable = yes
    read only = no
    guest ok = no
    create mask = 0644
    directory mask = 0755
    force user = YOUR_USERNAME
    force group = YOUR_USERGROUP

```

You might want to try the "force user" part and check your firewall to see if your ports 137, 139 and 445 are open and accessible from your network.
