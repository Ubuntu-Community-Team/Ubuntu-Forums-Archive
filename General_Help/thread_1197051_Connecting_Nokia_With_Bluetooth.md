### Post by homeblend on 2009-06-25
Believe I have a potential solution for you. 

Connect your phone to your ubuntu machine via bluetooth. Making sure they're both connected and both trust each other. Set your phone to always allow incoming requests from this machine.

Go to synaptic package manager and install gnome-phone-manager. 

Go to applications > system tools > phone manager. It will appear in your notification area. Click on the icon. It should connect automatically - if not click edit preferences. 

You should then be able to send and receive SMS right through GNOME.

Hope this helps. :popcorn:
