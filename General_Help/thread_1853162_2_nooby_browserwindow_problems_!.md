### Post by seawolf167 on 2011-10-01
1. You want to associate the .htm[l] files with your web browser? Just right click the file, select properties -> open with -> other application -> then select the default application you want to use.
2. Are you talking about something like window grouping by application on the panels? What version of Ubuntu are you using (as I'm only really familiar with the panels in version <11). What about using [ALT][TAB] to switch between windows?
