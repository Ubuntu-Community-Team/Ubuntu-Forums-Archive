### Post by distortedecho on 2009-08-07
If you're using an earlier version of Gnome-Do, you may need to add the software sources.

For hardy these are:

deb-src [http://ppa.launchpad.net/do-core/ubuntu](http://ppa.launchpad.net/do-core/ubuntu) hardy main
deb [http://ppa.launchpad.net/do-core/ubuntu](http://ppa.launchpad.net/do-core/ubuntu) hardy main

To add sources summon do (Super)SpaceBar and type Software Sources (or part of it and it will find it, which is why I love Gnome-Do).

Then just "sudo apt-get update" in the terminal.
