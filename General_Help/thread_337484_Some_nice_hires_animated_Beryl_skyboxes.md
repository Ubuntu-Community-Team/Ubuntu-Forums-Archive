### Post by floke on 2007-01-15
Beryl will load png and svg but not jpeg - you have to enable the plugin in Beryl Manager to tell it to accept png files (beryl settings manager/ image format).
Maybe the png file didn't convert properly? A quick way to do it is to open the file (eg jpg) in image viewer, change the extension to .png, and save it.

Hope this helps (though occasionally, on a different PC I have the same issue!)
