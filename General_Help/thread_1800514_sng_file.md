### Post by mcduck on 2011-07-12
As far as I know, that's Roland's manufacturer-specific file format so no, you can't play it on Ubuntu, or really anything else than Roland's own hardware & software.

Roland should have a software for converting it into MIDI file. And a quick search with Google shows some third-party (Windows) programs for the task as well.

(It being basically a MIDI file of course mean it's not going to sound the same when played back on any other than the original device, so keep that in mind. :)
