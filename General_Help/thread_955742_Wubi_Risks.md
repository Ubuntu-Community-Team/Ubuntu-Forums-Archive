### Post by jmszr on 2008-10-22
If you follow the protocol here: [https://wiki.ubuntu.com/WubiGuide](https://wiki.ubuntu.com/WubiGuide) there should be no problems wih the installation. Defrag the drive that you are going to be using for Wubi before installation. The Windows defragmentation utility will do fine for this task.

Edit: You certainly could use JKDefrag for defragging the drive,also.

### Post by jmszr on 2008-10-24
Shortylonglegs,
                You need to defrag before installation as that allows for the largest contiguous space. This should allow for an uninterrupted install. I've tried it both ways and defrag first is a really good idea. You might wish to type install and defrag into the search box at the top of this page - the Ubuntu Guru's can certainly explain it better than I can.
