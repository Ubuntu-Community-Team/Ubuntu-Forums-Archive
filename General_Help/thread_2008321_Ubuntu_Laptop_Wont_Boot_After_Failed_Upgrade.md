### Post by gordintoronto on 2012-06-22
I've found that Unetbootin is the most reliable program for creating flash drives. Is the Mac running Ubuntu?

System 76 has a forum here. If you write a message there, give a complete identification of the laptop.

If you can boot with the flash drive, make another backup of your files, perhaps over your network to another computer. You might need Samba to do that, and I don't think it is included in Ubuntu 12.04. It *is* included in Linux Mint 13, or several distros which focus on "recovery."
