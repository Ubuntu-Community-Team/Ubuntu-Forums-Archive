emerald --replace
``` to apply the settings I can not close the terminal window that is running in otherwise I lose my themeing. That terminal window is also taking up 50 megs of ram (because of emerald?) and thats starting to slow down my system. Thanks for help in being able to shut down the terminal window.

### Post by tuxxy on 2009-01-08
Press ALT + F2 and enter the command in box that appears this will keep running until you logout.

```
emerald --replace
```

Your best off opening system > prefs > sessions and adding emerald as a new session using that command above, this way you dont have to run the command every boot it will auto run.
