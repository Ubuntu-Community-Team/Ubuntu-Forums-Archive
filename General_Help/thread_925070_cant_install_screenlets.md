### Post by sayakb on 2008-09-20
Download and install it from: [http://getdeb.net/app/Screenlets](http://getdeb.net/app/Screenlets)
Also, at a terminal, type in:
```
sudo apt-get install <package>
```Where <package> is whatever it says is missing. For example, in this case:
```
sudo apt-get install python-central
```EDIT: Since you have figured it out, please mark the thread as solved (Thread tools->Mark thread as solved) :)
