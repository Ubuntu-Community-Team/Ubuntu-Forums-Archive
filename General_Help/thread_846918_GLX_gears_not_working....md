### Post by ajmorris on 2008-07-02
under your device section, change the driver from nv to nvidia, (if it is not installed install via sudo apt-get install nvidia-glx-new)
Add a new section that looks like this: (only really need this for if you want to use compiz or another composite manager)

   Section "Extensions"
       Option "Composite" "true"
   EndSection

And also under your device section where you just changed the driver from nv to nvidia, add a new line like this:
Option "AddARGBGLXVisuals"
Restart X and enjoy, your glx server should be happy, and as an added bonus, you're now ready to run compiz ;)

AJ

### Post by forger on 2008-07-02
I would do the following (Applications > Accessories > Terminal):
```

sudo aptitude purge nvidia-glx-legacy nvidia-glx nvidia-glx-new
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
sudo dpkg-reconfigure -phigh xserver-xorg
```
Press "y" and enter if it asks to purge the packages.

Now reboot your computer or just log out and log back in.
**In case something goes wrong**, restore the backup file:
```
sudo cp /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
```

nvidia 6800 supports nvidia-glx-new, so we'll install that:
```

sudo apt-get install nvidia-glx-new nvidia-settings
gksudo nvidia-settings

```

Then head to "X server display configuration" > Detect displays > Select your resolution and refresh rate (or just put Auto in refresh rate) > Press "Save to X configuration file" > Quit
After that reboot your computer or log out.

**In case something goes wrong**, restore the backup file:
```
sudo cp /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
```
