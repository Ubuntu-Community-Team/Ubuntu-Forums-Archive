### Post by snowpine on 2009-04-30
Cmost is correct; Ubuntu's development model is stable release, in other words, they never go back and add new features once a version is released. If you want a constant stream of "exciting new features" you should use Karmic Koala (or something other than Ubuntu). Once a version has been released, they only updates that come through are to make the system more stable and secure. Most Ubuntu users appreciate having a stable install without constantly worrying that the latest upgrades will break their system.

I would argue that they "hype" of a new stable release every 6 months is brilliant marketing and Ubuntu's biggest contribution to the Linux world.
