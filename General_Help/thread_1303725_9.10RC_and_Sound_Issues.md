### Post by Buschbarber on 2009-10-28
I have been using the Release Candidate for 9.10 since it was available. I dual boot with Windows 7 and Ubuntu Ultimate 2.3, as well.

Unlike Ubuntu 9.04, 9.10 does not seem to have an Audio Mixer installed by default.

I have the Asus P5Q mobo which comes with the Realtech Audio Adapter and I have it set up to use my 5.1 Surround Sound system. I downloaded an Alsa Mixer and it gives me some more options as far as Mixing goes.

I found that MythTV 022 had no Audio Controls and the output of MythTV was very soft. After the Alsa Mixer was installed, I was able to adjust the sound to an acceptable level, without everything else being too loud, however, each time I shut down, and relaunch MythTV, I have to readjust the Alsa Mixer.

I did not have this issue with 9.04, but perhaps, when 9.10 is released, this will be resolved.

UE2.3 is based upon 9.04, but perhaps there was more Audio support in UE and in the basic Ubuntu 9.04.
