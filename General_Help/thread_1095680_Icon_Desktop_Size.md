### Post by ramjet_1953 on 2009-03-14
Hi, Iarp!

There are a couple of ways to adjust icon size on your desktop.

An easy way is to just [COLOR="Blue"]right-click on an icon[/COLOR], and when the menu opens, just choose the [COLOR="Blue"]Stretch Icon option[/COLOR].

This will allow you to change the size of a particular icon.

If you want to make a global change for all icons on your desktop, you can use the [COLOR="Blue"]Configuration Editor[/COLOR] to make the change.

By default, configuration Editor does not appear in your Applications>System Tools menu.

To Enable it, follow these steps:

1. Navigate to [COLOR="Blue"]System>Preferences>Main Menu[/COLOR]
2. When the window opens, click on [COLOR="Blue"]System Tools[/COLOR] in the left pane.
3. Now in the right pane, put a tick in [COLOR="Blue"]Configuration Editor[/COLOR].
4. Close Main Menu.

Now, when you navigate to Applications>System Tools, Configuration Editor should be listed.

Now, before we go any further, I need to warn you that Configuration Editor is a very powerful tool that has many options and I urge you not to fiddle. You can very easily destroy your desktop by fiddling with things in here.

That aside, [COLOR="Blue"]open Configuration editor[/COLOR] and using the left pane navigate to [COLOR="Blue"]apps>nautilus>icon view[/COLOR].

In the right pane, you will see an item [COLOR="Blue"]default_zoom_level[/COLOR]

[COLOR="Blue"]Right click on this item[/COLOR] and when the menu opens [COLOR="Blue"]left click on edit key[/COLOR].

When the window opens, change the value from [COLOR="Blue"]standard[/COLOR] to [COLOR="Blue"]large[/COLOR].
As far as I know, there are no other options available.
[COLOR="Blue"]Click on OK[/COLOR]. and [COLOR="Blue"]close Configuration Editor[/COLOR].

Hopefully, all desktop icons should now be larger.

Regards,
Roger :D
