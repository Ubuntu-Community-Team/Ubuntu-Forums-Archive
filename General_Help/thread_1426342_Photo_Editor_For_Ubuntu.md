### Post by t0p on 2010-03-10
+1 for Gimp.  It's *similar* to Photoshop - meaning it has similar functionality, not that it's a Photoshop clone (it isn't).  The interface may take a little getting used to, but it can do most of what Photoshop can.  I don't know about Fireworks as I've never used it.

Gimp is installed on Ubuntu by default - **Applications > Graphics > Gimp image editor**.  Well, it used to be anyway.  If not, get it through Synaptic or terminal command

```
sudo apt-get install gimp
```

### Post by t0p on 2010-03-10
> **NoFriends said:**
> Will have a look for it now, would i be able to download and install Macromedia fireworks though ? or will it only work on windows.

If Macromedia Fireworks is a Windows program, it will work only in Windows.  Linux is not Windows.  You need to use Linux programs on Linux.

There is a *possibility* of using Fireworks through Wine, the Windows "emulator". But that is far from certain.  If you want to try, install Wine through Synaptic.
