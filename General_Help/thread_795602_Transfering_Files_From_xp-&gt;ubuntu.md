### Post by pointone on 2008-05-15
This likely will not work. When applications are installed in Windows, some files are often installed to locations other than "Program Files", and then, of course, there are changes made to the registry which cannot easily be transferred.

You'll need to install WoW in Wine, too, which further complicates the process. While it might be possible to transfer the files from one XP install to another, I think you'd be better off reinstalling from scratch in this case.

Of course, I've never installed/played WoW, so I could be wrong.
