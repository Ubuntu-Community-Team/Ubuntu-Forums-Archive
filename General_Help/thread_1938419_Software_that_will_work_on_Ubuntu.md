### Post by PhilGil on 2012-03-09
VLC is multi-platform; there should be an Ubuntu package. Some Windows software will work (to some extent) using Wine.  

However, I would strongly discourage you from switching to Ubuntu unless you're willing to try Linux alternatives to Windows software. Ubuntu is not for everyone and if those programs are vital then you're only going to get frustrated and be disappointed.

### Post by forrestcupp on 2012-03-09
As far as DVDFab 9 goes, [here is an article](http://davestechsupport.com/blog/2009/02/04/how-to-backup-a-dvd-with-ubuntu-linux/) about how to install libdvdcss2 and an app called k9copy. Those two things should allow you to do anything you could do in DVDFab, only they are native Linux apps for free.

Just don't forget that those are for backup purposes only, and not for copyright infringement.
