### Post by lithopsian on 2011-03-09
Crunchbang is a very nice lightweight distribution.  9.04 was basically a themed Ubuntu with Openbox, but the latest release is Debian plus either Openbox or Xfce.  Lubuntu is more or less Ubuntu with LXDE.  I'm not sure Arch is all that user friendly, although it is well supported and documented and you can pretty much cut it down as much as you like.  You can drastically cut down Ubuntu's impact on your machine by changing the desktop.  Gnome is heavy, Metacity not the lightest, and Unity is a real dog.  Get rid of those and your machine will fly.

As for apps, pretty much all distros will run them.  Gimp is hardly lightweight, but there aren't a lot of alternatives if you need all the functions it provides.  Your machine will run it without too much trouble.

Get as much memory as you can, especially since your video card uses some of it.
