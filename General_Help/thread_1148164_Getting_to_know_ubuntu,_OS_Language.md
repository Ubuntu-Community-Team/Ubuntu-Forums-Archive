### Post by mb_webguy on 2009-05-04
Actually, no...  C++ is used for some application development, but other software including the Linux kernel itself is written in C.  Many applications are written in Python, though many other languages, such as Java, are used as well.  It really depends on what specific software you're talking about.  And if you're developing your own software, the choice of programming language is entirely up to you.
