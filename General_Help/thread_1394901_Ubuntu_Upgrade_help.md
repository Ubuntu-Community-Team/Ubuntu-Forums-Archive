Theoretically, anything is possible. Always backup before doing anything major. Which is pretty much the default response to questions like yours. If you do encounter problems it's likely to be with updated packages not working correctly. Not random pictures or music getting overwritten by the updates.

I've done an upgrade install once and never had any problems. Although, most people will tell you that your much likely to encounter problems by doing an upgrade install than a fresh install. Considering that 10.04 is coming out near the end of April, maybe you should consider holding off to upgrading until then. 10.04 is going to be an LTS (Long Terms Service) release, which means that in theory it will be much more stable than the last several releases.
