I just hooked up a new wifi printer that has a static IP address on my home network.  I can connect/print just fine from my Windows PC, but not Ubuntu.  How do I add a printer based on IP address?  I have the outgoing firewall rule set to Deny on my Ubuntu laptop. Could that be the problem?  What port would I need to open to allow printing?

UPDATE: I finally resolved this issue.  I just added the IP address of my wifi printer to the AppSocket/HP Jetdirect option.  This seemed odd at first, since my printer is a Brother MFC.  The whole App/HP thing made me think that it was proprietary to Apple and HP.  Oh well, it works now :)
