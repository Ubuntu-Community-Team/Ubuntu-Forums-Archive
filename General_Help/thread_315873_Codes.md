### Post by aysiu on 2006-12-09
Rhythmbox is already included with Ubuntu.

If you're using Kubuntu, however, you can install Rhythmbox by pasting this command into the terminal: ```
sudo aptitude update && sudo aptitude install rhythmbox
``` Where's the terminal?
[http://psychocats.net/ubuntu/terminal](http://psychocats.net/ubuntu/terminal)

Read more about installing software here:
[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)

If you're using Ubuntu (and not Kubuntu), you may find this tutorial a bit more friendly:
[http://www.monkeyblog.org/ubuntu/installing](http://www.monkeyblog.org/ubuntu/installing)
