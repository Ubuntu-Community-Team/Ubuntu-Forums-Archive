### Post by sikander3786 on 2010-09-14
With 1GB RAM and a smaller display, I will recommend that you install the Netbook Edition. You can later on add anything onto it. Here is a good link for customization of Netbook Edition.

[http://maketecheasier.com/13-ways-to-customize-ubuntu-netbook-remix-for-better-usability/2010/02/07](http://maketecheasier.com/13-ways-to-customize-ubuntu-netbook-remix-for-better-usability/2010/02/07)

You can also install the Desktop Edition, try it and if it doesn't work well for you, install the ubuntu-netbook package from Synaptic and you'll get the desktop as well as the netbook in a single install.

netbook has both gnome & netbook launcher, so it's identical to the normal ubuntu desktop. they both run the same since there is no real difference underneath. so install the normal ubuntu if you want, your not missing anything any which way.
