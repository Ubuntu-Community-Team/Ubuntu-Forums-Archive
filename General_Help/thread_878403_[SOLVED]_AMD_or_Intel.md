### Post by cogitordi on 2008-08-13
> **thegodfaza said:**
> I prefer AMD since it's better at multitasking than Intel.

A multi-core CPU is useless unless you have applications and an OS that can make use of multiple cores in a CPU. No significant number of desktop or notebook users has any such software. Do multi-core CPUs make any difference for most computers users? NO. GPUs make more of a difference.

Intel only succeeded in making a 64 bit CPU with the Core 2 Duo line. AMD was successful with 64-bit CPUs long before. Do most computer users care about 64-bit computing? They did not before, but they are starting to now because today's mainboards can address 4GB and more of RAM. AMD was there first. 

Intel's Core Solo, Core Duo, and and Core 2 Duo CPUs are low-wattage. They are somewhat better than AMD's best in this regard. Will users ever really notice this difference? NO. There are many other components in a computer that affect the electricity bill more significantly.

In short, I think you can only answer this question based on a mixture of nostalgia and prejudice. 

AMD did 64-bit first and they did it well. They gave value to consumers by letting them overclock. Their chips were priced well. ATI, which they bought, supports open-source. I'll support the company that gives me more value and I think I get more real value from AMD. That said, Intel's current CPU line is strong. But look beyond the marketing to see the real value proposition.

### Post by Linux&amp;Gsus on 2008-08-19
Sort of tough question. What exactly means "better"?
From what I understand the Intel are performing better at the same clock speed. Means, from my understanding, if I have a 2.4GHz per core then a single thread app would perform better on an 2.4 Intel than on an 2.4 AMD core. Most common are the Dual Cores, I guess. I find Intel Core 2 Duo 2.4GHz for about 150AU$ while I also can get an AMD Phenom X3 (Triple Core) 2.3GHz for about 170AU$. So, for 20AU$ more I get a slightly slower clock speed per core but I get one more core.
Pumping up Intel to the same (theoretical) speed of the AMD 2.3GHz*3=6.9GHz would be 2*3.45GHz. The closest I found is a Core 2 Duo 3.33GHz for 333AU$.
Now, which processor will perform better is just a part of the question. It's largely about what you need the processor for. Therefor I personally voted for AMD since I have many single threaded app running and I believe that more cores have greater benefit for me. Means, as far as I can see it AMD offers better value for me.
