### Post by timzak on 2010-06-28
Maybe too late for you, but here's a possible solution for you:

In Firefox, go to View menu, select Page Style, and make sure "Basic Page Style" is selected, NOT "No Style".

In Thunderbird, you'd go to View menu, select Message Body As and make sure "Original HTML" is selected.

These are changes Enigmail ask you if you want it to make during the Setup Wizard process.
