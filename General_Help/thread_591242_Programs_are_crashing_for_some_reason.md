[solution]

I noticed that the search desk let was a source of a lot of my problems and figured it was something it was doing in the background messing things up. I think I was right because things are running stable again. 

For some reason when you first install Ubuntu, the indexing is set to the highest setting for the best performance. Now unless you have a machine that has 4GB of memory, it's most likely not the best setting. So just go to System --> Preferences --> Indexing Preferences and go to Performance. Set the less memory setting and low down the indexing and your system works a lot better. You seach desklet will even work! :)

Hope this helps someone
