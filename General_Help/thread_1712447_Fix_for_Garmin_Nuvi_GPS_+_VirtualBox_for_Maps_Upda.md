### Post by RT236 on 2011-03-22
I've just gotten a Garmin Nuvi 255 GPS unit to update flawlessly for Garmin maps updates from their website.  I struggled with this for sometime as I've seen many do, so thought I would pass along the info that helped me.

I had the problem with USB grayed out in VirtualBox and I could not select it. I had set the filter for the Garmin and lsusb also showed it. After much hair pulling I found a tutorial on How to Fix VirtualBox USB Support. I was not a member of the vboxusers group and hence could not access USB in my case from VirtualBox. Apparently this is a common problem sometimes for no USB access with VirtualBox. I'm listing this in case someone else has a similar problem [http://news.softpedia.com/news/How-to-Fix-VirtualBox-USB-Support-111715.shtml](http://news.softpedia.com/news/How-to-Fix-VirtualBox-USB-Support-111715.shtml)  After doing this one needs to log out and back in to update the changes.

Garmin Nuvi 255 GPS unit
VirtualBox 4.0.4
Ubuntu 10.04
Windows XP SP3

I just did a 1 hr. update of the maps and it went flawlessly.

Anyway, I hope this might help someone if stuck with a similar problem.
