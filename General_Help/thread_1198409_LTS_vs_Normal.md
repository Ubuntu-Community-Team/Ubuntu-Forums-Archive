### Post by snowpine on 2009-06-27
LTS releases receive updates and support for 3 years; regular releases only 18 months. Therefore 8.04 will be supported through April 2011. The LTS releases are popular for servers, business environments, etc. where it is more important to have stability than the latest versions of applications. Hope that answers the question.

### Post by Therion on 2009-06-27
LTS versions get Security Update support for three years for the desktop edition and five years of Security Update support for the Server version.
