### Post by schnauzer93 on 2008-05-31
Bump again. Please help, I really want to ditch Windoze :(

EDIT: I've found the solution for anyone having this problem. It turns out I was using smart playlists to oranize my collection. I edited my smart playlist, and turned on the "Limit to" feature. I set it to limit the songs to the soze of my ipod (80GB), which effectively deactivated the limitations. I then changed the sort order to "artist" and now it works like a charm.
