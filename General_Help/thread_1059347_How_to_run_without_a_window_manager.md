### Post by timeOday on 2009-02-03
Upon login, I would like a graphical (X11) application to load, full-screened, with no other windows or toolbars.  (The app also has no popup dialogs etc).

I think the best way to do this is running without a window manager.  I think I can do this by hacking .xinitrc, but is there a better way in ubuntu?

Thanks.

---

### Post by jimv on 2009-02-03
Go to System > Preferences > Session and uncheck Window Manager.

Then just remove your panels, and figure out how to set your app to be fullscreen.
