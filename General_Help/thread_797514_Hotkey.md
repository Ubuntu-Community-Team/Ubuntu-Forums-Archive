### Post by nic2 on 2008-05-17
I suggest that you use the Force Quit button to force an application to quit when it becomes unresponsive.  You can add the Force Quit button by right clicking on the panel and selecting 'Add to panel ...', then select 'Force Quit'.

Alternatively you can use the System Monitor to end process - on the Main Menu go to System > Application > System Monitor (the System Monitor can also be added to panel).

Should you need to restart the Window Manager you have to use Ctrl + Alt + Backspace.

I hope this answers your question
