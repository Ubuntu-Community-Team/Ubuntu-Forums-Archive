### Post by ago on 2007-09-27
In theory yes, in practice no, since the minimum size is hardcoded, so you'd need to change the code and recompile...

But...

The new wubi allows to install in 1GB even though such installation is read-only (exactly like booting the livecd, but faster since everything is on HD). And you get Ubuntu 7.10.

[http://wubi-installer.org/devel/minefields](http://wubi-installer.org/devel/minefields)
