When installing a program that brings up a GUI interface, I'm suggested to install into a specific directory in the file system.  How do I give myself permission to authorize that during the installation?

---

### Post by c1981 on 2009-02-11
If you have root-access;
try "System -> Administration -> Manage Users -> yourname" and change
user privileges to administrator.

---

### Post by MarblePanther on 2009-02-11
If you are installing from a .bin, .run, or .sh file, install it from terminal with the sudo command.
