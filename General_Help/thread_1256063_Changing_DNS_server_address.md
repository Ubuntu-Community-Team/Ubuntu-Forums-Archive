### Post by mimor on 2009-09-02
You can change it in your router ;)
Pc's going to get ip from your router and will use it as a dns.
Unless you don't have a router.

sudo gedit /etc/resolv.conf
nameserver <DNS1>
nameserver <DNS2>
and so on

but if you're using a router... change it there!
