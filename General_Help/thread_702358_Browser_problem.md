### Post by danwood76 on 2008-02-20
What it might be is it checking the user agent string (which identifies browser type)
You could install user agent switcher plugin for firefox ([http://chrispederick.com/work/user-agent-switcher/](http://chrispederick.com/work/user-agent-switcher/))
This can then spoof the websites you visit that you are using IE.

This normally works unless the site requires active x.
