### Post by jamesmillere on 2023-01-31
Hi,

You can run multiple nodes on a VPS by assigning multiple IP addresses to it. Here's a basic outline of the steps to set this up:


Assign the second IP address to the VPS: You can do this through the control panel provided by your VPS provider, or by using the command line.


Install the required software for the second node: Make sure to install the necessary software and dependencies for the second node.


Configure the second node to use the second IP address: This will typically involve editing the node's configuration file and specifying the IP address to use.


Start the second node: Once the configuration is complete, you can start the second node and verify that it's running and accessible through the second IP address.


Note: The specific steps to set this up may vary depending on the type of node you are running and the VPS provider you are using. You may need to consult the node's documentation or seek assistance from the VPS provider's support team to complete these steps.
