Yikes.. where are you? If you are in the US and you use the ubuntu guide to install windows media it **could** present legal problems for your organization. Playing the older (and fairly "standardized") XVID/MPEG4 type stuff isn't so much an issue, but if you are going for support of encrypted WMA or any WMV9 stuff it *could* be a problem.
