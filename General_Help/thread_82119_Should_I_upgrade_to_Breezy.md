### Post by funkydan2 on 2005-10-25
Yes - Synaptic cares which repository you're using, so if you are using Hoary, then you'll get the Hoary apps with security patches for the next 12 months.  If you upgrade to Breezy, you'll get newer versions of the software (e.g. Gnome 2.12 and OpenOffice.org2) with security support for the next 18 months.

If you're happy with everything now, then there's no need to upgrade.  If you want some newer toys to play with, then upgrade.

Upgrading is simple - [https://wiki.ubuntu.com/BreezyUpgradeNotes](https://wiki.ubuntu.com/BreezyUpgradeNotes)

### Post by sailor420 on 2005-10-27
Yeah, I'd upgrade. Breezy feels a bit more polished to me, and I've noticed many more things "just working" now than before. That's always nice. I haven't experienced any of the problems some people have complained about.

I'd make sure you have another computer handy if possible to help you though. Sign into the Ubuntu IRC channel; that way if anything goes wrong when you are dist-upgrading, someone there can help you. I had a couple of (small) problems, and immediately got some help in IRC to get them fixed. It made the upgrade much less hairy for me.

Make sure to back up your stuff before you upgrade though. And, if the dist-upgrade totally breaks your system, you're backed up, so just reinstall from scratch with a Breezy install CD.

Good luck :)
