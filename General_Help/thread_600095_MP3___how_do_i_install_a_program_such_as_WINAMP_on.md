### Post by xyphur on 2007-11-02
I use XMMS, and sometimes wish I had a port of it for the Windows machines I'm forced to use. Winamp is so bloated these days... even the lite version

XMMS uses (classic) Winamp skins, has plugin support (for FLAC playback, for example), and it's fast and stable.

Synaptic -> search 'xmms' -> check the box next to the xmms package -> Apply, and you're done.
