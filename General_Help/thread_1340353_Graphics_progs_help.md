### Post by peterballantyne on 2009-11-30
Further discovery might be worth noting for anyone else trying to get Irfanview up and running in Ubuntu. The moment I set any options an ini file is created in the Irfanview folder, which a normal. However the existence of that ini file seems to stop Irfanview dead in its tracks. Delete the ini file and its running just fine once more, albeit with the defaut settings. If I had more time I might be tempted to explore what the ini file is referring to that Irfanview doesn't like, but this issue is thoroughly repeatable and I have just settled for the defaults now. Makes Irfanview a little more awkward to use, but I love it so much I am just happy to have it going at all.

Hope this might me of some help to someone in the future. :D
