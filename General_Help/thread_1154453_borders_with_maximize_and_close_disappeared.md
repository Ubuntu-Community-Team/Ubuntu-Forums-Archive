### Post by gettinoriginal on 2009-05-10
Hit F11 twice, then manually minimize the window, close the window, then maximize it with the max button. that should stop it from happening again.  but if it does, F11 twice will return it to normal.  You should also check under view to make sure that full screen is not checked.  :P
