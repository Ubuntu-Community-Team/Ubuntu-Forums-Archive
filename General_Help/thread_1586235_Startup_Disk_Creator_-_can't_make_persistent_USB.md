### Post by Orographic on 2010-10-01
Hi there,

I have enjoyed setting up a live USB stick to boot Ubuntu from and it works very well but I can't make my settings persistent. 

The option to do that in Startup Disk Creator is greyed out, the Stored in Reserved Extra Space is just not available.



How do I do this?

---

### Post by sikander3786 on 2010-10-01
If you've got some additional installation media i.e, another USB or CD, you can boot from it and install Ubuntu on the target USB drive as you'd do with a normal install.

---

### Post by Orographic on 2010-10-01
Thanks for that, I appreciate you reply.

Yes, that was my next approach.

I think I've worked it out. I was actually using the Mint 9 .iso when trying to make a USB startup disk (it takes up a bit less room on the stick with Flash already installed etc) and that works fine but the persistence option is greyed out until you select the Lucid .iso then all the options become available.

If you select the lucid .iso option first, then select the Mint .iso, you can then install Mint as persistent and all of your changes will remain on the USB stick.

Lucid is my main OS but I am using Mint (with an upgraded kernel) until Maverick comes out as my computer system is very new and the hardware is better supported in the latest stable kernel.
