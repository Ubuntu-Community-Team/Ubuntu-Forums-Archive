### Post by MoLE on 2007-02-03
This should be pretty simple.  Is the DVD a ghost-bootable one?  If so, then just create a new virtual machine configured for the target OS, boot the VM from the Ghost bootable DVD and away you go.

You might have more luck with this query at the [VMware forums]("http://www.vmware.com/community/index.jspa") as this is more of a VMware query than an Ubuntu one.
