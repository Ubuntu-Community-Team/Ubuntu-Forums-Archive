### Post by 3rdalbum on 2011-06-11
Well, the reason why the Unity interface disappeared is because you unticked the box that allows it to run.

The user interface that appears to be running on your computer at the moment is the old Gnome Classic interface, for people who don't have 3D accelerated graphics card drivers. If you're sure you have the right drivers working, then you can log out, click on your name, and then at the bottom of the screen under the Session popup menu choose "Ubuntu" to get the Unity interface.

If you ever unticked the Unity plugin enabling box again, you can just follow the steps on the login screen to go into the "Ubuntu Classic" session, and then retick the box in Compiz Config Settings Manager.

Be careful in CCSM as there are options in there (like Desktop Cube) that can stop Unity from working. You should probably be able to reverse these options again using CCSM in the Ubuntu Classic session.
