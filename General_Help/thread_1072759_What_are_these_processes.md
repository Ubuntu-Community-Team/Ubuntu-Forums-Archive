### Post by mttman on 2009-02-17
I'm pretty sure these processes are used to access gnome libraries for running gnome applications in other environments. ex. running gimp under kde. if you have any gnome application installed on your machine it will need these processes to run.

i checked the first one to confirm my suspicion and was correct. the other ones probably follow
