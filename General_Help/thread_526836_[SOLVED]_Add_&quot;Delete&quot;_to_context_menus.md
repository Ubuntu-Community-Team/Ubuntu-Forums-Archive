### Post by jamvegan on 2007-08-15
Hopefully this is a very simple question.  I recently did a clean install of Feisty, because I wanted to see what the install process is like these days (very easy, and went without a hitch--I'm now even more likely to recommend Ubuntu to users new to GNU/Linux).  The last several releases I had upgraded.  Anyway, at some point in the past (don't remember which version I was running at the time) I had added "Delete" to the file context menu.  I could swear that I did so via a System->Preferences option called "File..." something.  But I find no such option in Feisty's System->Preferences menu.

Can anyone tell me how to add "Delete" (i.e., eliminate the file immediately--no moving to trash first) to the right-click context menu?  Thanks!

---

### Post by aysiu on 2007-08-15
In Nautilus (the file browser), go to Edit > Preferences > Behavior > Include a Delete command that bypasses Trash
