In my Application->System Tools  menu I have "NVIDIA Setings" appearing twice. Once with the appropriate icon, and once with the default icon. They both start the same task, and I wanted to remove one of them. How can I do it?

### Post by Zalbor on 2005-12-18
Try using SMEG (Applications->System Tools->Applications Menu Editor) to uncheck one of the two or delete it, if possible. The other option would be to go to /usr/share/applications, find which file produces the icon and delete it.
