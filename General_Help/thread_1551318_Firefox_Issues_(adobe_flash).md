### Post by dino99 on 2010-08-12
remove/purge everything about flash into synaptic, then install flashplugin-installer (canonical partner repo need to be activated to get it) and ubuntu-restricted-extras too

### Post by howefield on 2010-08-12
There is a nice add-on for Firefox which removes conflicting flash versions from your system and then installs the appropriate version.

[https://addons.mozilla.org/en-US/firefox/addon/161939/](https://addons.mozilla.org/en-US/firefox/addon/161939/)
