### Post by Paul S on 2007-07-30
In my experience, slowness results whenever the cpu goes close to 100%.  Maybe the problem had cleared by the time you got the system monitor open.  I always keep gkrellm running and visible on my desktop so I can see what's happening with the cpu whenever anything gets slow.  It is far superior to the system monitor and also allows for temperature, and network readings all on a slim tower.  Suggest you try this to help troubleshoot your problem.
