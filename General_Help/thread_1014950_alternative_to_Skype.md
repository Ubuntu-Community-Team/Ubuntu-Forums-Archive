### Post by jespdj on 2008-12-18
Skype uses a proprietary, encrypted protocol. There is no software other than Skype itself that works with the Skype protocol and your Skype account.

The easiest way to install 32-bit Skype on a 64-bit system is by installing it from the [Medibuntu](http://www.medibuntu.org/) repository.

Follow [the instructions](https://help.ubuntu.com/community/Medibuntu) to enable the Medibuntu repository on your system and then install Skype:
```
sudo apt-get install skype
```
