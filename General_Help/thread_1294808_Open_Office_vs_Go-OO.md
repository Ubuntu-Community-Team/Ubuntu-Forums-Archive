### Post by Kapitän Rotbart on 2009-10-19
You can check out the description in the Ubuntu Apps Center or the Add/Remove Programs description for OpenOffice.org:
> Homepage: [http://www.go-oo.org]("http://www.go-oo.org")

You're using Go-oo unless you remove it, then download Sun's DEB installer and install it.

Go-oo is better anyway, so no need to downgrade. One of the conveniences for using a mainstream Linux distro :)
