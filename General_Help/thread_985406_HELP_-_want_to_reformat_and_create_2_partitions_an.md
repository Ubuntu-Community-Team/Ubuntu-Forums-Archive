### Post by alzie on 2008-11-17
If it was my computer I'd backup my files and reformat, installing Windows first.  Since you're planning on gaming installing Windows first will give it faster access to the hard drive.

Your Windows disk may not be able to format the disk but your Ubuntu disk will.  I'd do about 40Gig for Windows (NTFS or Fat32) and then do an Ubuntu install for the rest.

The other reason for doing it in this order is that Ubuntu has no issues with dual booting, Windows likes to think its the only one.

I'd also download an antivirus and firewall for Windows before the format,  that way they can be installed before the computer sees the internet.

I hope this helps.
