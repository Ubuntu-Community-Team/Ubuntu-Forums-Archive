```

cat /sys/devices/system/clocksource/clocksource0/available_clocksource
``` and

```
cat /sys/devices/system/clocksource/clocksource0/current_clocksource 
```

You dont have to type every letter, you can use the tab key to complete

A live cd also uses your ram very differently, so perhaps you got lucky there. Bad ram memory, thats what my money is on...
