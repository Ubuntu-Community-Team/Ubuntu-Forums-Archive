### Post by bryncoles on 2008-12-30
i think they're mounted in the /media folder arent they?

id be lazy, and open nautilus (assuming you are using the standard ubuntu and gnome environment). clisk the little folder icon (i think it is) underneath the navigation buttons, and to the left the icons which say 'home', 'desktop', 'username'. this will turn into a written filepath, in the form home/username/desktop. this will be the filepath to your flashdrive, and if you type it into a terminal with the ol' cd, it should take you there.
