### Post by deanjm1963 on 2007-05-15
you're actually missing out by NOT using the generic kernel.

the 386 is for lower end machines, e.g. 386/486/pentium I.

Anything above that should use the generic kernel, especially machines that have dual core/hyperthreading chips as it supports SMP.
