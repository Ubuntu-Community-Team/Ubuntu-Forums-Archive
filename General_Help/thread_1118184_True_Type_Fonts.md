### Post by kanikilu on 2009-04-06
Are you talking about fonts you already downloaded or have? If so, you can either add them to /usr/share/fonts/truetype (you will need sudo to do this), or add them to a ".fonts" directory in your home directory (just create it if it's not already there).

The first option will make them available to anyone that logs in. In the second option, the fonts will only be available to you.
