If you changed the outgoing server, that is most likely why you can't send mail; the server must usually be the one that the ISP gives you originally with the email address, and sending mail, for example, from a tiscali account (a UK ISP) using an aol server or any other will usually not be possible.

There are exceptions to that rule, I think, such as the gmail server which I use for everything, but generally that is not the case.  If you reset to the original smtp server, you may be OK again.
