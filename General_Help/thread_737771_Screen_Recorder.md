### Post by terto on 2008-03-28
most packages when downloaded correctly through synaptic will create icons in the Applications menu under the right categories.

In the rare cases it doesnt you can still run them through the command line, if you do not know the exact command try ```
apropos
``` followed by the name of the app

like so: ```
apropos istanbul
``` that tells me the program is installed and the exact command name

btw istanbul is the screen recorder I use.

Hope it helps.

Cheers.
