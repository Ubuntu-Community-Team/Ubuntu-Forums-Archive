### Post by tjwoosta on 2008-05-12
1.the proper command is emerald --replace

2. dont run the command in the terminal (press alt-f2 and enter the comand here)

3. you can add the command to the startup programs so it will automatically run when you log in

to do this go to System-Preferences-Sessions click add and put the command in the box that says command
