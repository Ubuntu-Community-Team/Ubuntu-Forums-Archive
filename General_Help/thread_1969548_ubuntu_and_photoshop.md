### Post by Aerivan on 2012-04-30
I can't really say anything specific about photoshop but one thing I wasn't expecting when I first started using wine was that there are several versions available in the repository, not only the latest. The latest one  is wine1.4 so make sure that is the version you have installed. In my experience, support generally improves dramatically with each version.

Also, to help you get set up you should try installing photoshop with winetricks which comes bundled with wine, or playOnLinux, which is available in the repo but needs to be installed seperately. They will help you set up an optimal configuration for the application automagically.

### Post by davidvandoren on 2012-04-30
Hi!
I have adobe cs5, adobe elements, Adobe 5.0 LE, Lightroom 3.5, Topaz Denoise and Adobe DNG converter running in wine almost perfectly. 

There are several tutorials on the net on how to install Photo-shop CS5 . 

I compressed my wine installation and am using it now for the 3rd version of Ubuntu so far.

Yesterday I found out that it works in Ubuntu 12.04 64 bit with the wine from the software center perfectly. 
Even the single error message I got when starting CS5 is gone. 

Install wine from the software center and try installing Adobe CS% there. You can always create more user accounts for trying out and you can delete the hidden wine folder in your home directory. 

Once you got it working make a backup of that wine install.
