### Post by VMC on 2022-02-22
For passwords, backup '**key4.db**' & '**logins.json**'

[s]For bookmarks save to an 'html' file, then to restore you import from same 'html' file.
Located under Library 'Ctrl+Shift+o' "Import and Backup"[/s]
Use Backup & Restore which will create a 'json' file, instead of 'html' files. Its a better option.
