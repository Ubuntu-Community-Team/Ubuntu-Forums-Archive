### Post by 5-HT on 2006-05-19
[quote=mcduck]I like Conky most, but it's harder to configure and doesn't provide any flashy graphics, only text and some simple graphs.[/quote] Conky's great!

The documentation for conky details all aspects of configuration, so as long as you're willing to read a little you can easily set it up exactly as you want (provided it's supported).

The other nice thing about conky is that it's pretty lightweight.

Check out the homepage: [http://conky.sourceforge.net/]("http://conky.sourceforge.net/")

To install, either:

    * ```
 sudo apt-get install conky 
``` 
    * For the latest version, compile from source (really easy, can grab source from conky's homepage).

For a walkthrough, see: [http://ubuntuforums.org/showthread.php?t=139262]("http://ubuntuforums.org/showthread.php?t=139262")
