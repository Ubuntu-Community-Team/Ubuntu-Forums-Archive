### Post by mb_webguy on 2008-08-30
Most of the following may be found in the repositories.  I have provided links for those which are not, or for those which have newer versions not available in the repositories.
[LIST]
[*]Firefox, with the following extensions and themes:
[LIST]
[*]Adblock Plus
[*]Colorful Tabs
[*]DownloadHelper
[*]DownThemAll!
[*]Extended Statusbar
[*]Fierr
[*]Fission
[*]Flagfox
[*]Googlepedia
[*]Image Zoom
[*]ImageTweak
[*]New Tab Homepage
[*]No Squint (for hi-res screens)
[*]Stop-or-Reload Button
[*]URL Fixer
[/LIST]
[*]Glipper clipboard applet -- A clipboard applet to replace the less-than-wonderful Gnome clipboard
[*]Amarok -- IMO the best music player for Linux
[*]VLC -- The media player that plays almost any format without requiring external codecs, and has tons of extra features
[*]OGMrip -- A DVD encoder that makes ripping a DVD into various file formats (including mkv files using x264 video, complete with multiple mp3 audio tracks and subtitles *drool*) extremely simple
[*]DeVeDe -- An all-in-one tool for making simple multi-track DVDs from video files
[*]GPRename -- A very good batch file renaming tool
[*]KDirStat -- The best graphical disk usage analyzer available
[*][GridWars]("http://www.getdeb.net/app/GridWars+2") -- An addictive Asteroid-type shooter game with amazing graphics
[*]Scorched 3D -- An artillery game, and a great stress reliever
[*]Frozen Bubble -- The best Puzzle Bobble clone ever
[*][Deluge]("http://deluge-torrent.org/") -- An excellent full-featured bittorrent client
[*]Pidgin -- The only IM client you'll ever need
[*]Audacity -- A very nice audio editor
[*]Avidemux -- A simple video editor with some very useful features
[*]Gmount-iso -- A simple GUI that simplifies the process of mounting iso files
[*][Wine]("http://www.winehq.com/") -- The best way to run those pesky Windows programs in Linux
[*]Nautilus Actions Configuration -- Allows you to add actions to the Nautilus context menu
[*]Advanced Desktop Effects Settings (compizconfig-settings-manager) -- Lets you enable the nifty effects for Gnome seen in the YouTube videos
[/LIST]
