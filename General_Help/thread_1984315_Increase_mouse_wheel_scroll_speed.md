I don't know how to do that in Gnome. As a partial solution until someone who knows gets here, you can set the scroll step in **Firefox**. (I use Xfce and only felt I need to change it in Firefox). Go to address **about:config** and search for mousewheel. Set variable **mousewheel.withnokey.sysnumlines** to **false** and **mousewheel.withnokey.numlines** to a number greater than about 3, for example 9.
