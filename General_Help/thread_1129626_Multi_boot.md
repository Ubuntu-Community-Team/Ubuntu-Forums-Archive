The only way AFAIK to have more than one OS running at the same time is to install a virtualization program (VirtualBox, VMware, etc) on the HOST operating system and then using said app to install another operating system as a virtual machine GUEST.

### Post by mikewhatever on 2009-04-18
First, MS Windows has nothing whatsoever to do with booting either Ubuntu or Suse. Regardless of the multiboot term, I think the only way to have two OSs running is to use a virtualization software. Is that what you have in mind? The term dual boot commonly refers to having two OSs installed on separate partitions and the ability two choose either to run at boot. It doesn't mean two OSs run at the same time, nor does it deploy virtualazation.
