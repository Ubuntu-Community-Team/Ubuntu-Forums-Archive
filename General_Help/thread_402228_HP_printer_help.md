### Post by DoctorMO on 2007-04-05
postscript is a format for telling the printer how to print a document; it's an open format with different versions. heavly related to pdf such that some pdf documents can be sent to postscript printers directly.

The idea is that so long as the printer accepts a standard serial connection the printer will print basic documents in postscript v3 including graphics without any need for drivers at all.
