### Post by LenzM on 2006-11-02
If there is nothing you want to keep on the 10G drive then it's very easy.

Take out the 10G drive
Replace it with the new 40G drive
Install Ubuntu on the new drive.  
This will automatically detect the copy of windows that's on the other drive, and it won't matter which is master or slave.
