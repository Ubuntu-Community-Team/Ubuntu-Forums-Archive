### Post by linuxuser12345 on 2011-08-10
Hi, I had to disable then re-enable window compositing in Ubuntu 11.04, and now the Super + W expose-like command on the keyboard doesn't work anymore! :(
Is there any way I can re-enable it?

---

### Post by CatKiller on 2011-08-10
Install compizconfig-settings-manager.

CCSM -> Window Management -> Scale -> Bindings -> Initiate Window Picker For All Windows.
