### Post by sstusick on 2008-06-08
You don't have to extract anything. The theme manager will take care of it for you in the tar.gz format. Just go to System > Preferences > Appearance > Install and then locate the theme and it should install for you.

Not too sure about GDM themes though, haven't played with them.
