### Post by oldfred on 2022-04-28
Typically when you install, you include the /home as one of the partitions. And if reusing one with data you must NOT click on the format box, so data is retained.

You show no mount of /home in your fstab.
You do have /home mounted in other installs but using bind which I am not familar with.

While you do not need to move to a new partition, so you can skip all the first steps copying data, this shows how to mount your /home in fstab.
To move /home uses rsync- Be sure to use parameters to preserve ownership & permissions 
[https://help.ubuntu.com/community/Partitioning/Home/Moving](https://help.ubuntu.com/community/Partitioning/Home/Moving)

If using multiple installs, better to have a shared data partition and keep /home inside / since with no data, it is tiny.
