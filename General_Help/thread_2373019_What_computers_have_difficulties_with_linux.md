### Post by ajgreeny on 2017-10-02
There are several manufacturers who make it difficult to install or boot anything except Windows when using UEFI, which is contrary to the accepted UEFI standards, but there are lots of workarounds to allow users to overcome these limitations.

To read a lot more about this see the long thread at [https://ubuntuforums.org/showthread.php?t=2147295](https://ubuntuforums.org/showthread.php?t=2147295) started in 2013 by our colleague oldfred who is the fount of knowledge about this problem, and who has mentioned just about all these difficulties in edits to that thread.
Scroll down and look for the manufacturer's name and you may well find lots of links to other information telling yu what to do.

Good luck!

### Post by ian-weisser on 2017-10-02
The site tells you what *compatible hardware *is in that particular model, providing you an idea of how easy (or hard) installing yourself will be.
There are a few OEMs that ship Ubuntu pre-installed. Depends upon where on Earth you are located - only Dells ships globally. 

Difficulties vary from model to model: You had problems with Acer, but my last Acer laptop installed Ubuntu quickly and flawlessly.

You seem to be assuming that All Vendor X hardware is compatible/incompatible, and that's generally not valid. You can even install Ubuntu --with a bit of effort-- onto fairly locked-down hardware like some Macs and Chromebooks. That's what the LiveUSB/LiveCD is for - testing the system before installing, so you can return it if need be.

### Post by Bucky Ball on 2017-10-03
> **mastablasta said:**
> the certified list is mostly filled with old models.

+1. As are most of the 'Certified Hardware' lists. Largely redundant.

The best way to find out how the computer is going to run with Ubuntu is to download the ISO, create install media and 'Try Ubuntu' without installing or changing anything on your drive, as suggested. In the absence of the *actual* machine in question, the best way is to do a search for other people's experiences with Ubuntu on that same machine. For my current laptop, I would search for something like

> Toshiba Satellite Pro L50 Ubuntu

That would pop up with a bunch of links which would give clues about what problems, if any, others have been having with that machine and Ubuntu. You can get more specific and 

> install ubuntu Toshiba Satellite Pro L50

Or even

> install ubuntu 16 Toshiba Satellite Pro L50

I don't remember the last time I referred to the official 'Certified Hardware' pages. The few times I did in the dark, distant past they were so out of date as to be not relevant. Still, ok if you're looking for details for old hardware I guess.

Good luck.
