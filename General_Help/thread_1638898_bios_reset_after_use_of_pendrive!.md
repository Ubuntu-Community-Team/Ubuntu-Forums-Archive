### Post by dandnsmith on 2010-12-06
A thought, engendered by the description of the machine as *old*: older machines don't always have the ability to boot from a USB stick.

If I remember correctly, the pendrivelinux site gives a method for testin for this possibility, and also a way to set up a boot CD to bypass the problem. (It may also be possible to fix the boot setup on the HDD to boot and transfer the loading process to a USB stick, but this requires rather more effort)
