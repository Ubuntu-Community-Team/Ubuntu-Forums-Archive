### Post by Krytarik on 2011-01-18
Sorry that you didn't get any response during the last days! 

Since you obviously have transparency on your desktop, Compiz will be running in some way.

Try resetting all Compiz settings by removing/renaming both directories, "~/.compiz" and "~/.gconf/apps/compiz".

Start metacity before!:
```
metacity --replace
```

Your xorg.conf looks ok so far.

### Post by stinkeye on 2011-01-18
Have you by any chance enabled the compiz ppa.

This will show compiz packages you have installed.
```
sudo dpkg -l | grep compiz
```


Compare with mine
```
glen@MavMusic:~$ sudo dpkg -l | grep compiz

ii  compiz-plugins                            1:0.8.6-0ubuntu9.1                                OpenGL window and compositing manager - plugins
ii  compizconfig-backend-gconf                0.8.4-1ubuntu5                                    Compiz Fusion configuration system - gconf backend

```



If you have more recent versions install ppa-purge
```
sudo apt-get install ppa-purge
```

Then run
```
sudo ppa-purge ppa:compiz/ppa
```
to disable the compiz ppa and downgrade the packages you have obtained from 
this repository to the packages in the Ubuntu repositories.
