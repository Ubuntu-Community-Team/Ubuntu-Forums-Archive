### Post by ubunturules on 2006-07-05
Download the .deb file from:
[http://www.yourfilelink.com/get.php?fid=130941](http://www.yourfilelink.com/get.php?fid=130941)

install with

sudo dpkg -i gnucash-1.9.8_1.9.8-1_i386.deb

It installs alongside the usual dapper gnucash and can be launched with

/usr/local/bin/gnucash
