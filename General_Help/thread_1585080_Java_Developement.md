### Post by GregBrannon on 2010-09-30
Software to do what?  I don't know what you mean by "a bunch of junk comes."  Install the JDK and an editor that you know how to use or can easily learn.  Then find a tutorial you can understand and start programming.

Other considerations:
Integrated Development Environments (IDEs), Netbeans, Eclipse, KDevelop, etc., are described, discussed, and debated to death in the Programm Talk forum.

Programming editors (VI, VIM, Emacs, etc.) are similarly detailed.

What really gets the debaters going is if you ask whether you should start your programming voyage with an IDE or a simpler editor.

If you'll ask more specific questions, we can give better answers.
