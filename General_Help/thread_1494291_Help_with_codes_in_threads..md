### Post by drs305 on 2010-05-26
Are you using the "code tag" shortcut - the # icon in the post's menu? After entering the command you want inside the code tags, simply position the cursor after the [noparse][/code][/noparse] entry and continue typing or ENTER to move to the next line.

Is that what you are asking? And no need to feel stupid - we appreciate users who take the time to use the applicable code tags to allow easier reading!  :-)

Example added:
> 
You can insert "code" tags by clicking on the # icon in the post's menu. Type your command inside the tags. You can also just type the command, highlight it, then press the # icon to wrap the content in the code tags. This method works with other tags such as the QUOTE tag and the URL tag.
[noparse]```
sudo update-grub
```[/noparse]
This is how you update the Grub 2 menuentry file.
