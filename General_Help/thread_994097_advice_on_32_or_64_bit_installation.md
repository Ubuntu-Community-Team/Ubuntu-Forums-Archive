### Post by bodhi.zazen on 2008-11-26
To be honest, for most day-to-day applications / use there is not a huge performance difference between 32 bit and 64 bit. Sure if you crunch numbers and such there will be, but most people do not do this.

Second, unless you are using some RAM intense applications (such as virtualization), a default installation of Ubuntu does not use anywhere near 4 Gb RAM, let alone 2 Gb, so, unless you are performing RAM intense apps, you will not see any difference in performance between say 3 Gb and 4 Gb.

Third, 32 bit arch can use up to 64 Gb RAM.

With all that said, if you have a 64 bit processor I would use a 64 bit OS.
