### Post by Ng Oon-Ee on 2009-02-11
If you have a cube, you're not using KWin. You're using Compiz within KDE. Use the normal methods for compiz-settings, install compizconfig-settings-manager using:-

```
sudo apt-get install compizconfigs-settings-manager
```

And you'll have the settings manager in your menu somewhere.
