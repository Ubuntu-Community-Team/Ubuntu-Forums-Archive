### Post by drs305 on 2008-08-15
The linux app is called gparted. If you want to work with the root partition you might want the gparted livecd. You can use gparted on unmounted partitions from within ubuntu but will need to use the livecd or gparted livecd to work on the root partition, which would otherwise be mounted during normal use.

[http://gparted.sourceforge.net/index.php]("http://gparted.sourceforge.net/index.php")
