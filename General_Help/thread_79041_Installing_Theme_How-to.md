### Post by unexpected on 2005-10-19
The easiest way:

Download the them you want. It should come out as a tarball file. Don't unzip it.

Go to system->preferences>theme, the theme manager pops up.

Drag the tarball file onto the theme manager.

Your theme is now installed!

I asked a similiar question a couple days ago. If you need more help, or it's a more complicated theme try searching "Gnome Themes" and you should hit my thread.
