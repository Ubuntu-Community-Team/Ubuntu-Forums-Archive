### Post by happyhamster on 2010-07-03
Don't know about dolphin, but a nice standalone rename utility is pyrenamer. And in case you're using the thunar filemanager: it has a similar plugin (part of the package thunar-media-tags-plugin). It may take a few tries to get things right (because of the multitude of options), but both allow you to rename in almost any way conceivable.

There are several others: just go to synaptic (System-->Administration-->Synaptic Package Manager and do a search for "rename"). Good luck.
