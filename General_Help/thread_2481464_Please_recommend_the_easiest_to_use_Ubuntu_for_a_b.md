### Post by MAFoElffen on 2022-11-30
I volunteered at a Computer Recylcer that offered training. We pushed selling reconditioned laptops that we didn't have Winodws Licenses for, so were installed with Ubuntu Linux... Those were also machines that we would donate to Seniors and Non-profits.

What we saw, is that if a person didn't know much about computers itself (previously), that any flavor of Ubuntu worked just as well for them, as what they usually did was emails, social media, some document writing and filling out PDF forms. They learned how to get to what they wanted to use, and how to use it.

Where training got more intense and detailed was if people know more about computers, then they had to relearn things. Most of those Users found that KDE was simpler for them. It is somewhat similar to Windows, so wasn't as much of a learning curve to relearn things.

For some people, having to relearn "something" is a jump.

### Post by grahammechanical on 2022-12-01
@kuen2

Your hardware is more than capable for running any of the Ubuntu family of desktop operating systems. The main ones are Ubuntu, Kubuntu, Lubuntu, Xubuntu, Ubuntu Mate, Ubuntu Budgie. They all require a video adapter capable of outputting a resolution of 1024x768. The minimum RAM ranges from 1GB to 4GB. This depends upon the desktop environment used. A Dual Core processor is also required with a minimum speed ranging from 1.2 GHz to 2 GHz.

The problems come if a person wants to dual boot with Windows. A person needs to know what they are doing. We can give advice on that. Other problems arise if the hardware is the very latest. Linux comes with hardware drivers but it takes the Linux developers a little while to produce drivers for the very latest hardware. Some hardware suppliers cooperate with the Linux developers more than others.

Nvidia does provide Linux video drivers for their GTX 950 card and it can be installed by the Ubuntu installer. There is also open source video driver for Nvidia cards that works well.
