### Post by michy99 on 2009-07-17
According to the page you linked to, 1.1.26 is only available in source code at the moment, so if you want it you will have to complie it yourself. If you can make do with 1.1.25 for a few days until packages are ready for 1.1.26, you can add this PPA to your sources and update thorough Update Manager:

[https://launchpad.net/~ubuntu-wine/+archive/ppa](https://launchpad.net/~ubuntu-wine/+archive/ppa)

For help in using PPAs, look here:

[https://help.launchpad.net/Packaging/PPA](https://help.launchpad.net/Packaging/PPA)
