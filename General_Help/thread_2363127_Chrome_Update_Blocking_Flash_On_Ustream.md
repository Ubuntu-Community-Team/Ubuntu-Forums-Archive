I got the update to Chrome this morning, & now it's blocking flash on Ustream. I can see/hear the video, but the chat room is blocked. I get a notification that let's me enable it, but only for the session. I have chrome set to allow flash, but it doesn't seem to work as it should. I even added the url to the whitelist, but it didn't help. How can I make this work without having to re-enable it every time? Thanx.

### Post by sasafrass452 on 2017-06-06
Ok, looks like I found a solution! Searching google, I was led to [FONT=Roboto]chrome://flags/#prefer-html-[/FONT][FONT=Roboto]over-flash & set "[/FONT][COLOR=#000000][FONT=Ubuntu]Run all Flash content when Flash setting is set to "allow"" to "enabled". Disabling "prefer html over flash" works as well. I hope this helps anyone else with this issue![/FONT][/COLOR]
