### Post by jpkotta on 2006-03-27
There is a DVD available via BitTorrent that has everything from main, restricted, universe, and multiverse.  The .torrent is available from ubuntulinux.org.  You don't need to burn it if you don't have a DVD burner, you just need to get it to your Ubuntu computer and use mount -o loop.  If the Ubuntu machine has networking, you can share the connection with the windows machine.  Then there is this project: [https://wiki.ubuntu.com/UnofficialUbuntu510AddOnCD](https://wiki.ubuntu.com/UnofficialUbuntu510AddOnCD)
