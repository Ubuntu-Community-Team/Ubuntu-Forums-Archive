[http://torrent.ubuntu.com/xubuntu/releases/hardy/release/](http://torrent.ubuntu.com/xubuntu/releases/hardy/release/)

... and give Xubuntu a go, desktop or alternate (try desktop first). I have 3 machines and occasionally install for friends so I have quite the assortment of install cds! On the admin machine, I installed Xubuntu successfully, then downloaded:

```
ubuntu-desktop
```... and installed Ubuntu that way. You can sort of mix and match and make blends.

It is enigmatic but some iso's seem to work in some circumstances and others not. My theory is has something to do with the combo of iso and machine hardware configuration, sometimes regardless of recommended RAM sizes, although this could be superstition!! Xubuntu is designed and known to work on lower spec machines, even though your RAM is officially more than adequate for a Ubuntu install.
