### Post by arloth on 2011-01-06
This is the grub boot selector screen.  You usually only see it if you hold down shift while starting up, at least with the newer versions of grub.  

When you see this, just press enter on the first selection.  That's what it usually boots too.   Now if it crashes when you select the first option, then you've got some issues.
