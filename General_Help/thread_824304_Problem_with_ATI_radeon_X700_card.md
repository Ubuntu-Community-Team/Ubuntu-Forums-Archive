### Post by iaculallad on 2008-06-10
Instead of commenting out "Extensions" on your xorg.conf file, try disabling "Composite":

From this:
```

Section "Extensions"
        #Option "Composite" "Enable"
EndSection
```

To this:

```
Section "Extensions"
        Option "Composite" "Disable"
EndSection
```

On your terminal:

```
sudo aticonfig --initial
sudo aticonfig --overlay-type=Xv
sudo shutdown -r now
```
