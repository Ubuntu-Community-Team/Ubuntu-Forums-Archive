### Post by dresnu on 2011-10-26
> **CaptSaltyJack said:**
> I searched and it seems everyone wanted it to show minimized windows, so it does. But really this should've been an option in the CCSM settings. How do I make it so minimized windows are not included in the scale/window picker?

CompizConfig Settings Manager > Desktop > Ubuntu Unity Plugin > Switcher and uncheck "Show minimized windows in switcher"

Pretty unnatural behaviour, I have disabled it myself too.
