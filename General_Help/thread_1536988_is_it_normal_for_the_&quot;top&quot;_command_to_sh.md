### Post by Rubi1200 on 2010-07-23
Yes, it is normal.

Run the command ```
w
``` in the terminal and you will see the same thing.

Both users are you; unless you see the name of a user that is different from your name you have nothing to be concerned about.

Linux is just telling you one user is doing this, the _same_ user is doing something else.
