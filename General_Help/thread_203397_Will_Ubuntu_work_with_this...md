### Post by gingermark on 2006-06-25
Should run ok.

But if you find it too slow you could try Xubuntu, which uses Xfce instead of GNOME, which is lighter.

Or you could install Window Maker, or Fluxbox, or IceWM, or a number of other Window Managers.

The main thing is to use lightweight programs. OpenOffice for example will run reasonably slow no matter what Window Manger you use.

If you care, I managed to get by for a long time with Kubuntu on a 128MB RAM 800MHz desktop.
