### Post by kevinatkins on 2010-07-07
Hmm, this might sound drastic but you might need to draw a line under everything and start again, installing Windows first, then Ubuntu. I have had really strange issues with the Windows installer in the past, which I have only sorted after *completely* wiping the drive.

So what I'd suggest is you grab a copy of Darik's Boot and Nuke ([http://www.dban.org/]("http://www.dban.org/")) and completely wipe your hard drive with that. Then try installing Windows again. If you still run into problems, then there must be a hardware problem with your computer, at which point you might need to talk to Windows forums.
