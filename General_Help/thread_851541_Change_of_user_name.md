### Post by ryanhaigh on 2008-07-06
It's a little unclear but if the OP is talking about the username on their Ubuntu installation I think the easiest thing to do would be to create a new user with your desired username and password ensuring that you give that user admin rights. Copy over your files/settings from /home/oldname and change ownership to the new user. Once you are happy with the setup you can remove the old user and their files.

If SoloSalsa is correct and you want to change your forum username I believe you have to have a very good reason for doing so (harrasment etc.)
