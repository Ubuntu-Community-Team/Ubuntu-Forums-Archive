### Post by merlinus on 2007-08-04
Try adding this near the end of the ntfs lines:

auto,umask=0 0 0

Then restart.

If that does not work, then open Konqueror as root and you can manually change the permissions.

-merlin

### Post by TBerben on 2007-08-04
Add your user and group id to the options in the ntfs lines of your fstab, like so:

```
UUID=C65C670A5C66F519 /media/sda1 ntfs-3g defaults,locale=en_US.UTF-8,**uid=<your userID>**,**gid=<your groupID>** 0 1
```

You can find your user and group id by running ```
id
``` from a terminal (if you're the first user created they'll both probably be 1000)
