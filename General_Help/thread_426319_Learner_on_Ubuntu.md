### Post by raja on 2007-04-28
wine is in the Ubuntu universe repositories. Unless you have a compelling reason, use the version in the repositories - easier to install and more likely to work. So all you need to do is search for wine in Synaptic and install it or type ```
sudo aptitude install wine
``` in a terminal. You have to have universe enabled in the repositories list. Look at [this page]("https://help.ubuntu.com/community/Wine") for help.
