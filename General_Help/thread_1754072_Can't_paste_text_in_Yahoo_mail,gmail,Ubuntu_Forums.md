### Post by aaronchall on 2011-05-09
Pasting into the text body field in Yahoo mail, gmail, Ubuntu Forums forms, or Zotero notes in firefox doesn't work whether I use ctrl-v or right click, paste. It does work when I paste in a subject field, or a search box.

I'm running 10.10, 64 bit on a Dell Studio.

I have a feeling all of these issues with these sites are connected.

Please help?

Aaron

Self-Solved: Holding down the Super-key enables the functioning of ctrl-v for at least Zotero and Yahoo. The Super-key is the Windows key for the uninitiated. I also installed a clipboard manager called Parcellite from the software center, but I don't think it affected the workaround, but I wanted to include all possible factors in my solution.
