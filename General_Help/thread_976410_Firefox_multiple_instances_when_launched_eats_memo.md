### Post by akerman on 2008-11-23
Running FF 3.0.4 and Ubuntu 8.0.4 I've had this for a while and actually upgraded my RAM in order to try improve things. (Needed to upgrade anyway).

Still FF eats memory resources. Having for example three FF browser windows open for a while sometimes ends me up with utilizing 70-80% of memory (1Gb). Not before the last open FF browser window is closed, memory drops back to normal level and a new FF browser can be opened. This new open window only increases memory usage with a few percent, so obviously this is something that happens over time (i.e. a certain amount of time or a certain amount of browser windows need to be opened).

A bit amazed to see so little reaction in the community about this, so hopefully it is a very 'local' incident / situation.
