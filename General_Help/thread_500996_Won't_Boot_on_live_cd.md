### Post by strabes on 2007-07-14
Use the alternate CD or switch to a virtual terminal with CTRL+ALT+F1 and follow [this guide](https://help.ubuntu.com/community/BinaryDriverHowto/ATI#head-d8c6fd05bce340dfc3ad483abf0e18997868540b). I know it's for edgy but it's what you need to do if you for some reason want to use the live CD to install.

I don't bother with the live CD anymore because it just takes more time to load up and I already know that all of my hardware is supported out of the box anyway.
