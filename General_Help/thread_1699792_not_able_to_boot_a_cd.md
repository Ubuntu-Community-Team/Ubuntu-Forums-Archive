### Post by grahammechanical on 2011-03-04
I am not an expert and I am guessing. If I am wrong then someone may correct me but I ask this:

What type of Windows CD do you have? Is it a Windows Install CD that will let you install Windows on a blank hard disc? Or, is it a Windows Recovery CD? What does a Windows recovery disc do? Does recover a corrupted Windows installation that is already installed on the hard disc? Can it be used to install a fresh Windows operating system?

Another point, it is always best to install a Windows operating system before installing another type of operating system. This is because Microsoft does not design its operating systems to cooperate with other operating systems.

I wonder, when you installed Open Solaris did you decide to remove an already existing Windows XP operating system? If the Windows CD needs a Windows operating system on the hard disc so that it can boot and you do not have a Windows operating system on your hard disc, then this could be the reason the Windows CD does not boot.
