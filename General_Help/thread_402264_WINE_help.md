### Post by davidgarcin on 2007-04-05
at the command line type:

```
cd /path/to/install/
wine install.exe
```

That will run the installer just as it would in Windows.  You should be aware, however, that wine is not perfect, and it may require more tweaking to get it to run properly.  Try that first though.
