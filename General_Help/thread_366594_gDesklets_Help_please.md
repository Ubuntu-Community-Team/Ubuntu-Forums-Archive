### Post by louis_nichols on 2007-02-21
The easiest way is to open synaptic package manager, run a search for package gdesklets and mark it for installation. Then just click APPLY on the synaptic main interface.

Then, you will find the program starter in Applications>Accessories. The first time you start it, you'll be shown a window that lets you select desklets to activate.
