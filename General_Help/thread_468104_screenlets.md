### Post by arvevans on 2007-06-13
By "screenlets" I assume you mean workspaces?

Right-click on an existing workspace box (lower right on Ubuntu screen).  Select "Preferences, and add as many workspaces as you desire.

Background Colors:  Right-click on any workspace (out in the middle, not on an application button or icon) and select "Change Desktop Background".  There you can change background images and/or colors.
