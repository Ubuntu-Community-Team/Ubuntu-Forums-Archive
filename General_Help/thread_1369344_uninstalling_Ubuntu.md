### Post by 3rdalbum on 2009-12-31
Ubuntu doesn't come preconfigured with codecs for the most common sorts of video and audio files, due to licensing restrictions in some parts of the world.

That's right - you have to pay to use MP3. How sucks is that?!

It's possible (and easy when you know how) to enable those sorts of things on Ubuntu, but if you don't want to do this you can install Linux Mint instead of Ubuntu.

Or, you can follow these directions:

1. Connect to the Internet
2. Go to System > Administration > Software Sources and tick the first four checkboxes (they enable extra repositories)
3. Close the program - it may ask you if you want to reload the package list, at which point tell it to.
4. Go to System > Administration > Synaptic Package Manager. If you didn't reload the package list in Software Sources, then click the Reload button in Synaptic now. Go to the "quick search" box in Synaptic and type "ubuntu-restricted-extras". Install this package.

### Post by Ordes on 2009-12-31
.. Might give ubu / linux a try... It can be difficult in the beginning specially when u know nothing else than a Windows-GUI OS. Meaning everything u did before on your machine was Mouse-Menu based... But i think ubuntu makes a good fusion...

I also come like other people here from a C64 time, so converting to linux was never a big deal....

Anyways...
The Synaptic Package Manager is one of the most important things to learn in the beginning. It will let you install software with using a GUI. Simply search what u need (etc a codec) mark it and hit install..
Also the add/remove or ubu-software-centre [karmic] can do the job but its choices are limited, though you could find some codecs in there...

For the uninstalling:
Did you 
1. install your ubu over your windows (meaning deleting windows), or
2. did you install it besides windows (dual-boot)?

..1 No matter which OS, when its the only OS in your system and you want to get rid of it, just boot from the install disc of the new OS, format your harddrive/ partition, and install. Might be good to have a backup of your important files. Or having them on a different partition than the OS itself...
Basicly there is no uninstall. Just format and installing a new one.

..2 If you have a dual, multi boot system... The idea is the same. You format the partition where the OS was on. And if you dont want to install another OS, you can just merge this partition with your existing e.g file partition. But you might need to repair your boot loader after formating. I cannot be more specific about it at this point, as this depends on your system layout, which OS and which boot loader you are using.. ;)
