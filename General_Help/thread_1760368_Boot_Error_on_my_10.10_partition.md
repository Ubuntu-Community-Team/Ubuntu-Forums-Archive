### Post by Hedgehog1 on 2011-05-17
Well, the UUID of /dev/sda3 is:

**09cf01b0-451c-64d4-bf32-4792ca1dd890**

But grub is looking for a UUID of:

**bd6c212b-75f5-47fe-b88a-811e3ba5df47**

Also, your /etc/fstab is also looking for:

#Entry for /dev/sda3 :
UUID=**bd6c212b-75f5-47fe-b88a-811e3ba5df47**    /    ext4    errors=remount-ro    0    1

To fix this and be booting Ubuntu again, there are 3 steps you need to do from your 10.10 LiveCD/liveUSB:

```
sudo mount /dev/sda3 /mnt
```

```
gksudo gedit /mnt/etc/fstab
```

[COLOR="Red"]and change:[/COLOR]
UUID=**bd6c212b-75f5-47fe-b88a-811e3ba5df47**    /    ext4    errors=remount-ro    0    1
[COLOR="Red"]to:[/COLOR]
UUID=**09cf01b0-451c-64d4-bf32-4792ca1dd890**    /    ext4    errors=remount-ro    0    1

save and exit the text file (ignore the errors)

```
sudo grub-install --root-directory=/mnt /dev/sda
```

Now you can reboot and you should be good to go!
