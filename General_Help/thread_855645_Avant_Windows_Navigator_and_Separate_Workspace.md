I'm a novice user of Ubuntu.

I have Ubuntu 8.04 installed and am using Avant Windows Navigator (AWN). When i choose a different workspace (have 4 in total), the AWN icons follow no matter which workspace i choose. Then when i click on an icon (open application) via AWN, it jumps to the workspace that originally opened it.

What I wanted was to have a fresh AWN taskbar for each workspace. I believe this is how the original/default Ubuntu taskbar works.

### Post by OutOfReach on 2008-07-10
Goto System>Preferences>Awn Manager

There should be a tickbox called: Show windows from all viewports
Uncheck it, and exit.

Problem should be sovled :) .
