### Post by apoth on 2012-11-25
I tried installing a package (apacheds). It failed, I no longer want this software.

Yet every time I try and install something now it tries to reconfigure apacheds and fails.

### Post by apoth on 2012-11-25
sudo apt-get autoremove

That did it. Streuth, took me ages to find such a simple command!
