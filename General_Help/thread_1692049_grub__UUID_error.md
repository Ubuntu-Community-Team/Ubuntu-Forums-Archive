### Post by An Sanct on 2011-02-21
Boot into a live USB and do this

```
sudo mount /dev/sda1 /mnt 
sudo grub-install --root-directory=/mnt /dev/sda
```PS. this is a common thing, you may also [look at this post]("http://www1.ubuntuforums.org/showthread.php?p=10440484") (where I had the same problem), Grub sometimes messes up ...
or, search for "grub reinstall", don't worry, the windows installation is fine with grub.
