### Post by TheFu on 2018-12-03
14.04 Server support will be ending in the next few months.  Very few people will be running it still on non-server situations.  And on servers, I have only 1 14,04 remaining. All the others were migrated to 16.04 last Feb.

[https://ubuntuforums.org/showthread.php?t=2368114](https://ubuntuforums.org/showthread.php?t=2368114) shows someone simply following the instruction and loading it on 17.10. Support for 17.10 ended in July, so that release shouldn't be used, but it does mean that 18.04 will probably work fine.  Support for 16.04 ends in 2021 and for 18.04, in 2023 ...
