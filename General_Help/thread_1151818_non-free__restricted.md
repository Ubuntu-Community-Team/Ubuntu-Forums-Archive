### Post by aysiu on 2009-05-07
*non-free* and *restricted* are pretty much the same thing: proprietary, not freely licensed.

It has nothing to do with cost (all the non-free packages are also cost-free). It has completely to do with licensing restrictions.

Ubuntu is open source software and wants to promote open source software. Open source software allows people to freely distribute the software, or modify the source of the software and redistribute the software.

Software with restricted licenses can be distributed only under certain restrictions, and the code for such software is often closed off and unmodifiable.
