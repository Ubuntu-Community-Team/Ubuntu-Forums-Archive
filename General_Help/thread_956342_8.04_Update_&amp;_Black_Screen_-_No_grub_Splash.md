### Post by nikgare on 2008-10-23
WHen the machine has booted up and the flashing cursor is the, press the key combination of "Tab F1" or Tab F2 etc
This should get you to a normal login promt - if not, you may need to try "CTRL ALT F1" etc

Login using your user name and password, and then enter the command:
sudo startx

This will try to start the X window system, and if this fails, should give you some indication of the problem.
