### Post by CookieCutter on 2010-04-18
Hello so I am very new to Ubuntu. I recently installed Ubuntu 9.10 on my PS3. I accidentally deleted the preferred applications icon under Preferences and I cannot for the life of me figure out how to get it back. Any help would be greatly appreciated. Thanks

---

### Post by wojox on 2010-04-19
You need to go into System > Admin > Main Menu > Preferences. On the right side select +New Item. Fill in:

Name: Preferred Applications

Command: gnome-default-applications-properties

Comment: Select your default applications
