> **DougieFresh4U said:**

As 900i said, a serial external modem will "just work" in that you won't have any driver issues. However, dialing into a ISP and connecting isn't so simple. You will need to use some sort of PPP application for this. Its a *long* time since I had to do this so I can't offer much advice, but as you are using Ubuntu, I am sure there will be a GUI application you can use without too much trouble. I can see GnomePPP in Automatix, that would probably be a good place to start.

When I first started using Linux, broadband was very rare and so dial up was the only way to get on the net. Luckily I was working at a computer recycling depot at the time and I used a whole bunch of external modems, can't remember any not working.

### Post by odin1965 on 2006-11-22
Yes get a USR sportster external. Thay are all over Ebay, look her; [http://computers.listings.ebay.com/External_56K_W0QQfclZ3QQfromZR11QQsacatZ14920QQsocmdZListingItemList](http://computers.listings.ebay.com/External_56K_W0QQfclZ3QQfromZR11QQsacatZ14920QQsocmdZListingItemList)

Best way I found was to use gnomePPP, as someone already said you can install it with Automatix or from Synaptic. One thing to watch out for is that I had to have the modem actually hooked to a phone line before Ubuntu would let me set it as the default route to the internet. It would appear to activate the interface, but would not until I had a dialtone.

Go to System => Administration => Networking to activate the modem connection.
