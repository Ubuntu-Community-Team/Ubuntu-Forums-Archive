### Post by ryanhaigh on 2008-04-23
I personally use jabber through google talk (but obviously use pidgin as my client) although I don't use file transfers so I can't say whether they work or not. I know that there is only a limited set of emoticons but I have never had problems with message delivery. Avatars are supported and pidgin at least has the other features you want although there are also other clients available in the repos. 

I have never used it but the Pidgin developers seem to like AIM (at least it gets worked on) much more than msn. I have pidgin setup for both jabber and msn (just a note I have had much more success with using http protocol option for msn on pidgin) but try as much as possible to talk to my friends using jabber. This setup is what I would recommend for you as a way of slowly migrating to a different protcol while still being able (ussually) to talk to msn contacts, pidgins multiple protocol support is the best thing it has going for it.
