### Post by raafe on 2009-11-07
Hi again, managed to sort this one myself - anyone using a Dell Dimension E521 will probably have this issue, it can be resolved by either upgrading the bios via the Dell website (can only be done if you have windows installed on the machine) or by plugging your keyboard and mouse into a powered usb hub (i used my monitors built in hub)
