That's how it works. But you need to remember that there's always a chance that some other application actually has a required package that also has "zeitgeist" in it's name. Or that depends on such package.

For example I bet quite many applications included in the default desktop have some level of built-in support for Zeitgeist if it's installed, which more than likely comes in a package with "zeitgeist" in it's name and which should be installed even if you don't have the actual Zeitgeist itself around.

Pretty much any time you uninstall something that might potentially connect to other applications, it's important to check what would happen before doing the actual uninstall. Apt has an option to simulate the operation, and Synaptic is also great for this ind of tasks...
