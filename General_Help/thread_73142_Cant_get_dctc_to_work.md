### Post by Ricco on 2006-01-29
if you use dctc i think MicroDC, a text-based interface ,CLI, is the best (only) alternative these days.
[]("http://www.nongnu.org/microdc/")

MicroDC works really easy and tabcompletion for commands works just like in the shell. And just by creating a file named config in ~/.microdc you can type everthing you want done when you start the program.
Like set nick, share, active and so on.

however if you want a GUI there is a easy guide here:
[http://easylinux.info/wiki/Ubuntu#How_to_install_File_share_utility_.28DC.2B.2B.29]("http://easylinux.info/wiki/Ubuntu#How_to_install_File_share_utility_.28DC.2B.2B.29")
