### Post by rajhanschinmay on 2012-12-29
Try these:
sudo add-apt-repository ppa:jconti/gnome3
sudo apt-get update
sudo apt-get install indicator-applet-complete
sudo apt-get install gnome-sound-applet
sudo reboot
Or
sudo apt-get reinstall indicator-applet-complete 

Most of the things u have told are related to indicator applet only.
Do let us know if it works.

Thank you.
