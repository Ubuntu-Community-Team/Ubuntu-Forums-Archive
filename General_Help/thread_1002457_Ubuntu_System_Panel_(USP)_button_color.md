### Post by eternalnewbee on 2008-12-05
```
sudo apt-get install gnome-color-chooser
```
After finishing installation, you can find it under: **Main Menu > System > Preferences > GNOME Color Chooser**.
Not sure it will help, but this is the best program I could think of, short of making your own theme.
