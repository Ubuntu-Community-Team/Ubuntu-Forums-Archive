### Post by fickenbaisage on 2008-06-13
Press ALT+F2 and type "emerald --replace" without the quotes. This will turn on emerald. Go to System -> Preferences -> Sessions and add the same command under system startup.

### Post by Happy_Man on 2008-06-13
If emerald doesn't work, then you have to press alt-f2 again and type ```
metacity --replace
``` to get your old window border back. 

Also, you need to have Compiz installed and enabled in order for Emerald to work.
