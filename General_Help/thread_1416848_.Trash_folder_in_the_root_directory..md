### Post by dabl on 2010-02-26
There's a hidden directory in /root named ".local".  Inside that is a directory named "share" and in that is the "Trash" directory.  In a console window you can
```

sudo su
``` to become root
```

cd /root/.local/share/Trash
```
```

pwd
``` to confirm you are where you should be.  In that folder are two folders, "files" and "info". So, ```
cd files
``` to enter the files directory, then

```

rm *
```

to delete the contents.  Then exit root mode -- it is not recommended for normal work.
