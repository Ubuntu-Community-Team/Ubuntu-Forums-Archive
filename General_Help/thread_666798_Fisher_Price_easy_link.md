### Post by sumguy231 on 2008-01-13
A quick Google search reveals that it requires their Windows-only software, so you could try installing [Wine]("https://help.ubuntu.com/community/Wine") and then installing the software.

P.S. On that page where it tells you to install Wine, despite the fact that it says "Newer Version (Not Recommended)" I actually would recommend that you install the newest version as it is probably more likely to run the software, and I've never had problems with it.
