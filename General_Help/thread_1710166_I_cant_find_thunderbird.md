### Post by WorMzy on 2011-03-19
Thunderbird should be under Applications -> Internet -> Thunderbird
Firefox should be under Applications -> Internet -> Firefox

Alternatively, you can press Alt+F2, then run "thunderbird" or "firefox".

If you want to create a launcher on your desktop or panel for either application, right-click the desktop (or panel), choose "Create Launcher" (Or "Add to Panel"->"Custom Application Launcher"), then type in "/usr/bin/thunderbird" or "/usr/bin/firefox" into the command box. Fill in the other fields with whatever you want, then click OK.
