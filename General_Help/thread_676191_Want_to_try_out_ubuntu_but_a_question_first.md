### Post by manuelb on 2008-01-23
I think you could be able to easily export your messages to a Unix mailbox with some free apps: i can't remember how i did this some time ago, but my switch was OE -> Outlook -> TheBat -> Thunderbird in a 2 year span.
I think you could be able to fire up Thunderbird and then import your PSTs from there, then track down the created MBOXes in your "Document and Settings" (if you are on Vista go in /Users/<user>/appdata/roaming/thunderbird/profiles/<profile guid>/mail) and just backup that: in Ubuntu you can use Evolution and import mbox (don't know if it can import PSTs too..) or install Thunderbird and copy/reimport your mail.
