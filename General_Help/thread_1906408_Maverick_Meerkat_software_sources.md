### Post by TBABill on 2012-01-09
I would look for some PPA's that have specific apps you want to update, if possible. Using another versions entire repository would likely bork the system unless you never did any upgrades besides specific app upgrades one by one. I'd think you would have some libs and other OS related upgrades available when you add new repos, including kernel and other critical files. With Unity in the mix in 11.10 and not in Maverick you could end up with an unsightly mess, but if you have the time and inclination it could be fun to find out.

### Post by TBABill on 2012-01-09
It is a very bad idea to mix repositories from distro versions. You can certainly do it, but the results will probably break your system...things like Xorg, Gnome versions and other critical pieces will be incompatible with older libs.
