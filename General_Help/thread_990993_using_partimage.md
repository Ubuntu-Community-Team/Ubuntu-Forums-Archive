### Post by ajgreeny on 2008-11-23
If the harddrive or partition is the installed ubuntu you are using at the time, you can not unmount it.  You should use a live CD if that is the case.  You can, however, use partimage from your ubuntu install to make an image of another partition, even one on the same harddrive.
