### Post by jeffdano on 2005-11-02
Are you downloading the 32-bit client or the AMD64 (make sure you have the right download)? Just drop the dnetc file (the correct file as there are a few) into the terminal and hit enter, should give you the config screen (the 1-4 options). Set everything up, then close that terminal, open another and put the same dnetc file into the terminal and you should have happy crunching.:D
