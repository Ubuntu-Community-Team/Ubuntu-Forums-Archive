### Post by Dragonbite on 2012-02-13
You definitely should be able to so long as the system will boot.

I've done this with my work laptop to just see if Ubuntu (or whatever distro I was trying out) would see the video and wireless card, but did not want ANY risk to the hard drive.

So I removed it and booted intl Linux with no problem.

I'm not sure how it is done when running from a CD, but if you have the image on a USB you can set a portion of it to be persistent so that changes and files are saved between boots.  My usual USB cannot handle this for whatever reason, but I do have one now that it does seem to work.
