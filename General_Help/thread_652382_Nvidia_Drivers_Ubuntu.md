### Post by overdrank on 2007-12-28
Ok lets try this and backup your xorg
To copy Xorg
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf-backup
```
To restore backup if it fails to load desktop from the command line
```
sudo cp /etc/X11/xorg.conf.backup /etc/X11/xorg.conf
```
Then I would try and reconfigure your xorg with the command 
```
sudo dpkg-reconfigure -phigh xserver-xorg
``` and hopefully this will help. Good luck!

### Post by NineseveN on 2008-01-10
Make sure you're not running XGL (you don't need it with the Nvidia card):
> apt-cache policy xserver-xgl (this will tell you if you're running XGL)

If you are, remove it (it causes the very issues being talked about in this thread):
> sudo apt-get remove --purge xserver-xgl

Restart your X (Ctrl+Alt+Backspace) or reboot if you need to. Then run nividia-settings.
