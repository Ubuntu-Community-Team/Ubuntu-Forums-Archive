### Post by Shazaam on 2008-01-29
A good rule of thumb is to wait a few days on major things like kernel and X associated updates. Search around to see if anyone else is having problems and if you don't find anything go ahead and update. As far as anything else if it is something you can live without for a while if it breaks try the update and see what happens.
