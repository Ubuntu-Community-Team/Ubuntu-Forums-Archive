### Post by tgalati4 on 2014-01-21
How many processes did you have running under Windows?  Open the task manager and count them.  Do the same in Ubuntu.

Benchmarks are processes that compete for the CPU as any other process.  So if you had a bunch of crap running in the background in Windows, you will get slower results.
