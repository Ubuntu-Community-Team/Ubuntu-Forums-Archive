### Post by Kelvin Gardiner on 2009-11-11
There is a package called netbook-launcher which provides the UNR interface only.

There is a second package ubuntu-netbook-remix which is meta package of everything for UNR.

I've not tried what you want to do, but I think the first package is what you need. The second package provide the whole UNR install.
