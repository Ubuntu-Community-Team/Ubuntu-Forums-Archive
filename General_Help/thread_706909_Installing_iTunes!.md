[COLOR="Red"][FONT="Comic Sans MS"][B][SIZE="3"]Installing iTunes in Wine!
[/SIZE][/B][/FONT][/COLOR]
All you need is 3 things!
1)  Wine version: 0.9.19 to Wine 0.9.55 - [Click]("http://wine.budgetdedicated.com/archive/index.html")
2) iTunes 6.0.5 or older - [Click]("http://www.filehippo.com/download_itunes/")
3) Wininet.dll - [Click]("http://www.dll-files.com/dllindex/dll-files.shtml?wininet")

 I am using Wine 0.9.53 but i say it will work in 0.9.55 :)

If you are using one of the old versions of Wine have a look at this page for help!
[http://www.frankscorner.org/index.php?p=itunes6](http://www.frankscorner.org/index.php?p=itunes6)

[SIZE="3"]Installing iTunes from this post is up to you and is not up to me! So if anything do not work at the end of it all. It's not me![/SIZE]

OK yet get started with it all :lolflag:
Install iTunes like what you do with any Windows .exe file but in Wine.
By right clicking on the file and go to install with Wine.

When you get to the end of the install, You need to put wininet.dll in (wine/drive_c/windows/system32) Then go to your Winecfg, In libraries! Set wininet.dll to &#8220;native&#8221; And click appy and close.

That's it all done.

But i say you will be best to have a go to update your Apple QuickTime to 7.1.3
As the new QuickTime will not work in Wine!
Same go to the new iTunes. It needs GuicktTime to play, But can not see it.

Any help on all this have a look at the two sites.
[http://www.frankscorner.org/index.php?p=itunes6](http://www.frankscorner.org/index.php?p=itunes6)
[http://pimpyourlinux.com/linux-feature-review/itunes-in-linux-not-any-time-soon/](http://pimpyourlinux.com/linux-feature-review/itunes-in-linux-not-any-time-soon/)
