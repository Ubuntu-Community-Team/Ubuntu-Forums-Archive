There is no single answer to dual-boot. It has worked well for many people, but others have had significant issues. Dual-boot is often used by gamers, since there are some games that will only play in Windows. Given your request to play WoW, I suspect that you will fall into that category. I have heard that WoW can be played using WINE (a Windows emulator that runs in Linux), but the performance can be an issue.

You may also want to consider installing Ubuntu as WUBI, in which case it will install as a program under Windows. This is definitely a 'try before you buy' option (although ubuntu is free, so there is no buy).

Hope this helps,
