### Post by TeoBigusGeekus on 2012-02-11
Hold Ctrl+Shift+U, type the _hexadecimal_ code of the character and press enter.
This is for unicode, but it should work with ascii as well.

EDIT-BONUS: Triforcing in linux
ctrl+shift+U+A0, ctrl+shift+U+A0, ctrl+shift+U+25b2
ctrl+shift+U+25b2, ctrl+shift+U+A0, ctrl+shift+U+25b2

Can't make it work in ubuntuforums, but it works in /b/...
