### Post by Vermind on 2008-04-29
Hello,
Emerald Theme Manager is located in System - Preferences - Emerald Theme Manager.
Installing an emerald theme in a .emerald file is done from the theme manager. You click import and select the .emerald file. Your theme is then added to the theme list below. Then click the theme You want.

To see emerald themes, You need to have both Compiz and emerald running. to run emerald, type ```
emerald --replace &
``` in a console. To run compiz, type ```
compiz --replace &
```. To make emerald Your default window decorator, go to compiz configuration, (ccsm in the console) and look under Window decoration. There is a command to use if no decorator is running. You may change this to ```
emerald --replace
```.
