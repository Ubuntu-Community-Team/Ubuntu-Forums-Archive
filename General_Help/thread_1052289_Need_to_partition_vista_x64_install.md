Assuming the Vista partition can be shrunk without destroying it, you are on the right track.  If you boot a live cd and go to 

System - Administration - Partition Editor

You'll be running the Gnome partition editor.  I haven't seen any howtos on that but given what you've been playing with on the windows side I think you'll find it self explanatory.

Note that you work out all the changes you want to make and then set it going, so if you make a mistake you can quit without applying them.
