### Post by wog on 2011-01-09
Question: How do I add items under the Applications menu?

Explanation: Most items seems to appear there after installing, but others don't. If I put something in place in Wine instead of going through an actual installation process there's no way to start the application other than going to the directory and starting it directly.

### Post by s.fox on 2011-01-09
Hello.

Right click your applications menu and hit &#8220;Edit Menus&#8221;. 

The Menu editor will appear . In the left panel select the category like accessibility, Debian, accessories etc. The right panel will update with the entries in that category.

After you have selected the category, click on the New Item Button.

Another dialog will appear, thats where you will make the actual entry.

I hope this helps you.

-s.fox
