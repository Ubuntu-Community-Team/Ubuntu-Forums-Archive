### Post by BertN45 on 2011-05-11
With software you can do it using VNC or RDP, but it is probably too late.
If you enabled remote desktop in the past you could do it. If you could borrow another monitor for an hour you could do the same. You have two options:
- Go to Remote Desktop and enable it. Install a VNC client in Windows Vista to control the Ubuntu machine.. 
- Or install xrdp in Ubuntu using the software center and use the built in Vista remote desktop accessory  to control your Ubuntu machine..
