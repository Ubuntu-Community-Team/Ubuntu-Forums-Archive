### Post by edm1 on 2011-05-04
What make/model is your computer? How far do you get when it locks up? Do things appear to be working normally before the lock up? Are there any error messages?

You could try [changing the boot options]("https://help.ubuntu.com/community/BootOptions#Changing the Boot Option Configuration Line") so that it says 'nosplash' rather than 'splash'. This could allow you to see what is cause the system to lock during boot.
