### Post by sam81 on 2006-12-14
I've been able to receive and send email with a yahoo account from Thunderbird. If you go to your yahoo mail webpage, click on options (it's on the top right part of the page), then "pop access and forwarding" and then "pop settings" you should get all the info you need to set up Thunderbird properly. Hope this helps, though you've probably done it all already. If you still have problems, try to give some more info on your setup and the type of errors you get.
