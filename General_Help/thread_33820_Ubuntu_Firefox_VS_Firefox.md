### Post by qstone on 2005-05-12
Ubuntu uses a 6 month development cycle. Between 2 releases we only have security updates and minor updates. Warty had firefox 0.9.3, Hoary has 1.0.2. That's it.
Really ? No !!! Have a look at Ubuntu backports project ( [http://backports.ubuntuforums.org/](http://backports.ubuntuforums.org/) ). These guys constantly port newer versions from unstable (breezy at the moment). See the site on how to get these updates.
