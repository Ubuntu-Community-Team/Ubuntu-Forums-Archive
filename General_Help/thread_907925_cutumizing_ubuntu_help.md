### Post by matdombrock on 2008-09-02
Ok so first off. In ubuntu 8.04 Compiz come pre installed. To make sure it works go to System>Preferences>Appearance, then click the tab visual effects on the top of the window. Try setting it to normal or extra. If it works you will not get an error and you should see some basic effects. 

If that works you need to install advanced desktop effects settings:
use:
sudo apt-get install compizconfig-settings-manager

Now it should be under System>Preferences>Advanced Desktop Effects Settings.

There you go, hope it helped. If not feel free to PM me.

also you might want to check out:
[http://tombuntu.com/index.php/2008/03/27/custom-compiz-effects-in-ubuntu-804/]("http://tombuntu.com/index.php/2008/03/27/custom-compiz-effects-in-ubuntu-804/")

As for the alarm clock: Kalarm might work.
