### Post by grahammechanical on 2022-08-28
> IMO these aren't very complex  softwares to build.

This is the advantage of using Free and Open Source Software (FOSS). We can download the source code and modify the code so that the application performs to our satisfaction. Then there is the situation where what you like someone else will hate. So, the developers do what they want to do.

Ubuntu comes with Gnome Evince (called Document Reader). It does two finger spread (zoom in) and two finger pinch (zoom out). Each document opens in its own window. You will not like that but will like it. Keep in mind that Gnome developers consider Evince to be a simple document reader.
