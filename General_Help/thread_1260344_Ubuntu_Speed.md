### Post by QIII on 2009-09-07
7.04 has moved beyond update support, so I would not use that.

Your system meets the published minimum requirements for Jaunty, but you might still not be happy with the speed if you use one of the heavier interfaces like Gnome or KDE (Ubuntu and Kubuntu, respectively)

You did not mention if your machine has Intel graphics, which may adversely affect your experience.

[http://www.ubuntu.com/getubuntu/releasenotes/904](http://www.ubuntu.com/getubuntu/releasenotes/904)

See sytem requirements and the section about regressions with some Intel graphics chipsets.

I would recommend a lighter weight version like Xubuntu (with the XFCE interface).  I have not  used the LXDE interface (unofficially, people are calling it Lubuntu, but I don't think it is officially a release yet.)

If you don't like the performance you get on your machine, you may be interested in a lighter weight distribution like CrunchBang, which is based on Ubuntu.

There are also a large number of light weight, non-Ubuntu distributions out there.

Don't be wedded to Ubuntu if it doesn't work well for you.  Use what works.
