### Post by wpwood3 on 2006-11-04
Try changing this:
Modes      "1024x768" "800x600" "640x480" "1600x1200"

to this:
Modes      "1600x1200" "1024x768" "800x600" "640x480" 

The first valid mode in this list will be the default display mode for startup.

### Post by tenshi-no-shi on 2006-12-26
This may sound strange but try removing all but the 1280x1024 resolution, that forces X to use that resolution and only that resolution. If you are using the fglrx driver I noticed that it likes using it;s own resolution and not the one you want, but doing that fixed it.
