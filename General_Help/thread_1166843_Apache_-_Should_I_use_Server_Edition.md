### Post by dustman on 2009-05-22
In have both Ubuntu Server (on my server, D'OH) and Ubuntu Desktop (on my laptop). You can always get your Server to become a Desktop, and vice-versa. As I saw it, Ubuntu Server just has LAMP and MySQL installed and configured by default. If you got them working on your Desktop, that's just great. On the other hand, the Server does not have a graphical interface, just command line, but you can always install it:
