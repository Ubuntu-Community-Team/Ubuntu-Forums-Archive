### Post by MoLE on 2007-05-23
In Ubuntu, the DOM inspector comes as a separate package - you can install it from synaptic or via the commandline thus:
```

$ sudo aptitude install firefox-dom-inspector

```

You should then have access to the DOM inspector via firefox.
