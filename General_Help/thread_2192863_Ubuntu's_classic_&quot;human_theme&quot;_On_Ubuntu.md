### Post by Impavidus on 2013-12-10
Several xfce window styles come quite close. If not close enough, you can modify the style to be even closer. Modifying xfce window styles is actually quite easy.

Any image file on your computer can be set as wallpaper, so if you have the original 8.04 wallpaper file you can just set it as wallpaper on a new Ubuntu.

The look of the window contents is determined by the gtk theme (in Ubuntu and Xubuntu at least). I never tried modifying any of those, but some come quite close.

I preferred the brownish background over the purple one too. Nowadays I have live weather satellite images as wallpaper.

### Post by buzzingrobot on 2013-12-10
The MATE desktop is the way to go to revive the old Gnome 2 look. It's the old Gnome 2 codebase rebuilt, plus revisions to accomodate itself to the fact that it is no longer 2010.

Existing GTK2 themes *should* work on MATE. GTK3 apps, though, really need GTK3 themes, so to have a consistent look you need a theme that has both GTK2and GTK3 versions.  In any case, it's not difficult to just install a backgorund image and tweak colors of panels and other components.

### Post by Impavidus on 2013-12-10
The bar on the top of the window, together with the close and maximise buttons, belongs to the window manager. The brownish bars you get when selecting something in a list, and the shape and colour of the buttons and background of the window itself belong to the gtk theme. I think that in 8.04 they where coupled together. On my Xubuntu I can choose them independently.

GNOME 2 is gone. There is a fork that tries to keep the original look and feel of it, named MATE. I think it's the default in Linux Mint, which is based on Ubuntu. Else, xfce, used in Xubuntu, can be configured to look very similar.
