### Post by Midwest-Linux on 2008-05-10
If anyone is interested... I tried installing Ubuntu 8.04 on my Presario F762NR Vista machine. While Wubi was able to download the Ubuntu , upon trying to install the screen went to issues with the video driver.

 Reading some posts in various forums, some say that Vista doesn't get along that great with Linux..yet others say its no problem. I also read that Vista has a great feature to shrink the hard drive and then open up a free partition...this is what i did.

 I shrunk the Vista C drive using Vista's shrink partition utility. And ended up with a 12 GB free partition. Then I used the ALTERNATE Ubuntu 8.04 version. I simply installed Ubuntu on the now freed portion with no issues.
