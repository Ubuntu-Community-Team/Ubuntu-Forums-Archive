### Post by CatKiller on 2019-10-28
If you're using Gnome (the default Ubuntu desktop environment) they've taken away the ability to have launchers on the Desktop.

In other desktop environments you can generally just drag and drop launchers from the menu to the Desktop, or the panel, or wherever.

They're just text files, with a .desktop extension, that specify the command that will be run, and the icon to be displayed, and so on.
