### Post by migs73 on 2010-09-13
When looking for any software the easiest place to look is in the Ubuntu Software Centre (It is in the Applications menu on Ubuntu, I assume it is the same in Xubuntu).
Type a keyword in the search box, find the software you would like in the list below and click install. You should then find it somewhere in you applications menu (unless it is a Non GUI program, then you will need to run it from terminal).
Open Office is in the Software Centre (again I am assuming it is there for Xubuntu, but I would think so).
