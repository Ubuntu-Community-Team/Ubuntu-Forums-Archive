It seems that your X is running just fine, this is why running startx gives you that output.
If you want to start|restart|stop X, use eg. the following command

```
sudo invoke-rc.d gdm start
```

The alteratives are, as I just stated: start, restart and stop.

[servers]
# These are the standard servers.  You can add as many you want here and they
# will always be started.  Each line must start with a unique number and that
# will be the display number of that server.  Usually just the 0 server is
# used.
0=Standard
#1=Standard


If that is ok, check the servers section in /etc/gdm/gdm.conf-custom.  Mine is empty.
