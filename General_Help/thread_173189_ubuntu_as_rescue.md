### Post by aysiu on 2006-05-09
Drives can be mounted using the Ubuntu live CD, but they don't automatically appear or get automatically mounted as they do in Knoppix.

Knoppix is designed to be a live CD, and it's also designed for recovery.

Ubuntu is designed to be a test drive of the installer CD. If you want to use Ubuntu as a recovery CD, you can, but you have to learn a little bit about mounting from the command-line.

### Post by Herman on 2006-05-09
You can use Ubuntu Live CD for file rescues, but your have to [manually mount the partitions]("http://users.bigpond.net.au/hermanzone/p10.htm") on the hard-disks by typing commands into a terminal. It just takes a minutes or so. 
After that you can make the same network connections as Knoppix. I use [SSH Networking.]("http://users.bigpond.net.au/hermanzone/p11.htm")
Compared with Knoppix, which also based on Debian and is designed for that type of work, Ubuntu Live CD is maybe just a little more work to accomplish the same goal, but not very much. 
I prefer to use Ubuntu because I'm more familiar with it. I feel more at home. I used to use Knoppix too, I guess it's just what you get used to. 
:D (Herman)
