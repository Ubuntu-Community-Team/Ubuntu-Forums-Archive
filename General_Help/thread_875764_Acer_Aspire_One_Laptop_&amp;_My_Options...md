### Post by OzCDN on 2008-07-31
I bought one for myself and the first thing I did was install Ubuntu on it. I believe that you have to install 8.04.1 on it to get support for the Intel Atom chip ... 8.04 fails during install apparently.

Personally I really like it ... only complaints so far are the touch pad buttons located on the sides of the pad are awkward and the battery life for the 3-cell is too short.

The Ubuntu install is pretty easy - I found instructions on the web to get the wireless working. There are a few bugs still outstanding but I expect they will get fixed soon. The wireless light doesn't work and there are issues with the sound from speakers vs. headphone jack.

I opened it up and threw another 1GB sodimm in the empty slot to bring it up to the max RAM of 1.5 GB ... but it ran fine on the 512 MB. This does involve taking it right apart so it is not for the feint of heart or the warranty conscious. 

I am concerned that you are proposing using it as a primary computer. This unit falls into the 'netbook' category which are units that are designed to be ultra portable while providing the power to do the simple jobs that you do 'on the road'. They are not really designed to be your primary work-horse.

### Post by Piraga on 2008-08-05
I got one of these last Friday (and within an hour installed Ubuntu). No complaints so far however it does hang up a bit sometimes (probably because of the atom). 

There are a few things it takes getting used to, but aside from some annoyances it runs pretty well. My suggestion is to get a big SDHC card with it. The Ubuntu install + swap partition left me with 3.5GB on the SSD. 

I'm also pretty worried about the life of the SSD. The Sound DOES give hiccups but its pretty easy once you get used to (having to mute the front speaker before switching to headphones), and I've had to reinstall the wireless twice.

An issue I've also had a few times was it "not shutting down correctly"(when it seems to shut down perfectly) and Ubuntu having to run the disk check program on boot. If anything crazy happens I'll let everyone know.

Aside from that it is alright. I haven't tried AWN or desktop cube yet but I really want to now (for curiosity sake) and it seems to run everything well if you open them in order and not all at once. The RAM seems to be an issue since I am only running 512, especially when opening multiple 10 megapixel photos, however for $400 you can't beat that value.
