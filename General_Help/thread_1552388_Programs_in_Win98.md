### Post by snowpine on 2010-08-13
This is the disadvantage of Windows vs. Ubuntu.

The Ubuntu repositories are full of thousands of applications that are 100% guaranteed to work with your particular Ubuntu release. Installing and updating them can be done with a single click using the Ubuntu Software Center and Update Manager.

In Windows, each individual application has its own system requirements, installer, and update procedure. You will need to research each individual application to answer your question. Microsoft does not maintain a repository or central list of tested and trusted applications for each Windows release.

Furthermore (and this is my own personal opinion) Windows 98 has reached its "end of life" and is no longer supported or maintained in any way, therefore it should not be used by anybody under any situation. The security and stability risks are just too huge. 

If Windows applications are important to your work or studies, I strongly encourage you to spend a few dollars and upgrade to the current version of Windows.

Alternately, you can check if the programs are supported by Wine and run them within Ubuntu.
