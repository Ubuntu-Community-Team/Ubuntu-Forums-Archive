### Post by t0p on 2011-01-17
First of all, download an Ubuntu install .iso, which you can find [here]("http://www.ubuntu.com/desktop/get-ubuntu/download").  Burn yourself a live CD, and stck it in the CD drive and reboot.

From the options list, choose to try Ubuntu without installing.  That way you'll get a better idea of any hardware issues etc.  If you decide to install Ubuntu, click on the Install icon.  The installer wizard will start: select to install to the whole disk, and to format the disk to ext4.  That will erase all the cruft from the hard drive and give you a nice new shiny Ubuntu OS.

Welcome to Ubuntu!
