### Post by Venerence on 2008-05-18
So I am currently in the process of transferring all my mail from my evolution client on Ubuntu onto a flash drive. Unfortunately I want to be able to read this from windows, and the best client that can be self-contained on a flash drive would be thunderbird.

So my problem is getting all of my saved mail from my evolution client to my thunderbird client. Settings and contacts I don't really have a problem with, I can set those up manually fairly easily. It's more of an issue of transferring the thousands of e-mails I have stored on my computer.

If anybody knows an easy way to transfer the files from Evolution to Thunderbird, that would be great, because I haven't found much information in my googling.

**EDIT**: Of course I happen to find the answer 10 mins after posting the thread. Apparently the save as function can save multiple messages in an mbox format. Of course that's what an export is, not what a save is, and I wish evolution was a bit more clear on what the save as function does. But oh well, at least that's one problem solved.

Found the answer at [http://email.about.com/cs/evolutiontips/qt/et110103.htm](http://email.about.com/cs/evolutiontips/qt/et110103.htm)

EDIT: I just noticed you found another answer, I hope it doesn't involve too much manual intervention. I'll just post my response anyways.

Make sure you backup before attempting any of this and initially use copy instead of cut (or move) when moving files around. I haven't actually tried this myself since I just use IMAP, for which Thunderbird seems to be using a different cache format but here's a handy guide I found using the search query [*'migrate evolution thunderbird'*](http://www.google.com/search?q=migrate+evolution+thunderbird):

[LIST]
[*][Linux.com :: How to migrate from Ximian Evolution to Mozilla Thunderbird](http://www.linux.com/articles/113798)
*Both Evolution and Thunderbird store email in the mbox format, so with a little trickery you can copy your messages from your old Evolution folder to your new Thunderbird profile and Thunderbird will recognize them automatically.*
[/LIST]
[Also see the previous search query limited to the Ubuntu Forums.](http://www.google.com/search?q=migrate evolution thunderbird site:ubuntuforums.org)
