### Post by cavalier911 on 2011-01-19
The system which provides the boot splash on ubuntu/lubuntu as of 10.04 is called plymouth.
Search synaptic package manager for **plymouth-theme **there are a few packaged themes that aren't distro specific.. 
Do a web search on **plymouth theme** for information on installing unpackaged themes and making your own.
