not precisely the same problem, but this thread came closest, so, for all it's worth, I'm just sharing the experience... 

My problem was with Skype, i believe, installed from synaptic package manager in ubuntu 10.04. From past experience on various paltforms skype is highly unreliable, but it reached the top then. 

As usual, some chats would just fail to transmit. (Skype wouldn't just crash systematically when just opening a chat, though.) I have no hope of skype being less buggy on that count. The problem was that when this happened, it was impossible to quit skype cleanly. Skype would hang until I shut down and the shutdown process would do a force quit. 

I did as suggested following the link provided by LilYoda above. I.e. 
1) remove skype (using Software Center) 
2) download the .deb package from skype website 
3) click and let it install. 

I'm sure chats will occasionally continue to be lost, but at least, for now, I can cleanly sign out and quit skype, and start it again.
