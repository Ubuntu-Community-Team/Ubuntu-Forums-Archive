### Post by grahammechanical on 2013-03-13
I would like to point out that you have posted this question in a support section of the forum. Yet you invite opinions. You may not know it but Ubuntu is based upon Debian and as Xubuntu is based upon Ubuntu it also is based in some way upon Debian. The main difference between Xubuntu and Ubuntu is the desktop environment.

> [COLOR=#000000]Is it necessary to reinstall Ubuntu periodically in order to stay current?[/COLOR]

Of course it is. But we do not need to re-install we can Upgrade. We even get an invitation to upgrade when a new release comes out. Ubuntu at present does follow a rolling release model. What Ubuntu has, and some of the flavours of Ubuntu follow the same pattern, is stable releases and not experimental (rolling) releases. At present there is a stable release every six months. Every two years at the end of April Ubuntu is released with 5 years support for security fixes. That kind of Ubuntu release is called an LTS (Long Term Support) release. In between these 2 yearly LTS releases we have interim releases every six months and at present they are supported for 18 months.

That is the system at present. It has not stopped some of us from running the next release of Ubuntu while it is under development. I have been running the development branch of Ubuntu for almost two years. And by updating it daily it is not much different from a rolling release model. A lot of people want stability, especially business people and Ubuntu provides that. Some people want what you call the "latest." The latest often means unstable.

There has been some discussion lately about Ubuntu going to a rolling release model. I am sure that you have heard about it. It may happen. It may not happen. I doubt very much that Ubuntu will follow the Debian rolling release model. From what I understand about what is being proposed, Ubuntu will not move away from having an LTS release every two years. It is possible that there will be a development branch of Ubuntu that a person can install and which they can update daily. This, as I understand it, will not be much different from what I am doing at the moment. There may be ISO images from that development branch released at certain times of the year.

There really is no need to debate this in a help section of the forum. Just read the officially released information.

[http://www.markshuttleworth.com/](http://www.markshuttleworth.com/)
