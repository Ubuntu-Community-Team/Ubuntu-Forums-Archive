### Post by DaveDH on 2007-12-13
This seems to be something new with Gutsy, but I have no idea what may be the cause, or where to look.  (I've checked Keyboard Shortcuts.)

The key combination <ctrl><alt>S 'rolls-up' an active window so that it exists as a title-bar only.

How/where can I stop this happening?   

(I have the ctrl+alt+s key combo assigned to another task, but the window rollup is taking priority.)

Thanks.

---

### Post by CatKiller on 2007-12-13
If you have Desktop Effects turned on, you can use CompzConfig Settings Manager, under the General Options -> Actions tab to change the "Toggle Window Shaded" action.

---

### Post by daengbo on 2007-12-13
System -> Preferences -> Keyboard Shortcuts
Go down to the "Window management" section and change the keys for "Toggle shaded state."

Cheers
