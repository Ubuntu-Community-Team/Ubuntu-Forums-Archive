VMware is very much available for linux, but virtualbox is free and an excellent program and can be installed with Synaptic Package Manager,

### Post by yknivag on 2009-10-24
VMWare is available but isn't free.  You can install Virtualbox from the repositories.

```
sudo apt-get install virtualbox-ose
```

You can get more info in the [WIKI](https://help.ubuntu.com/community/VirtualBox)
