### Post by sanderd17 on 2011-05-24
In almost all cases (except for games), alternative software can be found in the ubuntu software center (USC). Software from the USC has the advantage to be written for linux and compiled for ubuntu, it will update itself with bugfixes and you're able to maintain it better.

You should only install an exe via wine if it's a game that you really want to play (but most people don't play games on Linux, that's why most keep a dual boot with Windows) or in some cases an application that you can't live without and that only runs on Windows (typically something that you have to install for your work/school).

Tell us what app you want to install and we could maybe give you hints about good alternatives in the USC.

### Post by Mr. Shannon on 2011-05-24
If you are new to Linux then I would suggest using Play On Linux (a front end to Wine).  It makes installation of Windows software very easy.

However, as mentioned above, it is best to avoid Windows software on Linux if possible since it is often buggy when run in Wine.

There are games on Linux, just not the same ones as Windows and there are not as many of them.
