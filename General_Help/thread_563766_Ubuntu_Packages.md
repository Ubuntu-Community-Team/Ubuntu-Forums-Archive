### Post by Jussi Kukkonen on 2007-09-30
There's no need to deal with .tar.gz-files when you're installing standard ubuntu software: just use *Add/Remove software* or *Synaptic* to install packages. More info:[https://help.ubuntu.com/community/InstallingSoftware](https://help.ubuntu.com/community/InstallingSoftware) 

.tar.gz is just a compressed file archive. Source code is often distributed in gzipped tar-files -- if you're a new user, I'd suggest avoiding that and sticking to installing packages. I don't know if that's possible in your case as I don't know what you mean with "ubuntu packages page".
