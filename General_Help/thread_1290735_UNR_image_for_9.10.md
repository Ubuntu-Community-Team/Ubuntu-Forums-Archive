### Post by beastrace91 on 2009-10-13
Use [unetbootin](http://unetbootin.sourceforge.net/) to use the standard .iso file to install to a flash drive (this is how I got Karmic on my netbook) To install it on another Ubuntu system just use ```
sudo apt-get install unetbootin
``` or on Windows just download the .exe installer from the above link.

If you really want the netbook launcher on there as well you can install that from Synaptic after installing normal 32bit Ubuntu.

~Jeff
