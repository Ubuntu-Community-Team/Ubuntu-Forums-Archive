### Post by nick_h on 2008-04-06
You need to restore the Windows MBR. Normally the easiest way would be to use a Windows boot disk / CD.

You can download a boot disk from [bootdisk.com](http://www.bootdisk.com/) or use a [Super Grub Disk]( http://supergrub.forjamari.linex.org/).
