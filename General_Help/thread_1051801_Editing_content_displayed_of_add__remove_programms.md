### Post by mb_webguy on 2009-01-27
Add/Remove Programs pulls its information from the repositories, just like Synaptic.  So the only way to limit the programs that appear there is to change the repositories in your sources (such as only using the main repository, and not universe, multiverse, or third-party repos).  

But basically, if a user has privileges to install software, they will see everything available.
