### Post by ibuclaw on 2009-06-07
The background, I don't think so, but it should be possible to change the icons.

The icons you see in Ubuntu are located here: /usr/share/icons/Human/scalable/status

There are some other icons for the notification system here: /usr/share/notify-osd/icons/hicolor/scalable/status
Which are there as a fallback icon set.

One way you could switch iconsets is to change your icon theme to one that doesn't inherit icons from Human.
