### Post by KingCharles on 2007-03-22
Are you using the beta drivers from the nvidia website, or the open-source glx drivers?

If you are using the nvidia beta drivers, compiled and installed by their own script, it most probably is related to the driver having problems reading a crappy written EDID from the monitor.

All monitors have an [EDID]("http://en.wikipedia.org/wiki/EDID"). It is a piece of software inside themselves which tells the video card about their max refresh rate, max resolution, etc.

Check [this]("http://www.nvnews.net/vbulletin/showthread.php?t=81635&highlight=HDTV") link out which can help you fix the problem if it is the case.

If you find that this is your problem, you can acquire the EDID information and output it into a file using **nvidia-settings** (in the monitor section - some drivers do NOT have this option), or with [get-edid]("http://john.fremlin.de/programs/linux/read-edid/") which you can later on edit directly with **ghex** (available in the repositories).
