### Post by ajgreeny on 2013-11-09
Your comparison of flash in windows vs ubuntu is not valid, I'm afraid.

Flash is no longer being made available for linux and only security patches of the current flash version are going to be available, unless you use google-chrome as browser; that uses a much later version of pepperflash and it will work in linux, though many users still prefer open source browsers like firefox and chromium.  It is possible to get tye later version of pepperflash into chromium by using a ppa repository, which you could try [Pepper Flash For Chromium]("http://www.webupd8.org/2013/04/install-pepper-flash-player-for.html") but I have no experience of doing that so can say no more.

What hardware do you actually have; cpu, graphic card, ram, etc etc, as that may be your main difficulty.
