### Post by blazemore on 2009-04-20
I'd recommend removing and reinstalling the graphics card driver.

Failing that, try
```
sudo dpkg-reconfigure -phigh xserver-xorg
```
to go back to your original, maybe the login screen problem can be solved another way.
