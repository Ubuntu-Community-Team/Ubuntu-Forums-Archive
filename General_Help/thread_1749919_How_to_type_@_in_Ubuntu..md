### Post by yetiman64 on 2011-05-05
Sounds like you have found a workaround. 

However if you want to set it to normal use, go to System > Preferences > Keyboard > Layouts tab and check what it is set at there. In my case I use USA and shift + 2 gives a @ symbol. 

To change it click add, select the Country and Variant applicable to you, as you change these settings the picture of the keyboard changes as an indicator for you. Click the Add button when you are happy with the choice and if it works OK click "Apply System Wide".

Edit: note I'm on 10.04, check akand074's post regarding 11.04.
