### Post by tgalati4 on 2013-07-02
Zim uses a wiki format, so it has some formatting, but it is not the best tool for web-clipping.  For that zotero Firefox plug-in is perhaps the best tool for capturing web notes.  I find zim is best for computer/installation/troubleshooting notes and journaling.  For cut and paste of multiple data types, using OpenOffice or LibreOffice or even Google Docs might be better.
