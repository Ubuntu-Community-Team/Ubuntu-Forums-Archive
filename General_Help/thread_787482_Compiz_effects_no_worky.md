### Post by can8dnSix on 2008-05-09
Even if you check all the boxes you are going to need to configure things. Also, check the key bindings because a lot of them are "enabled" by pressing a key combo. Further, a lot of what you are talking about like the "fire" is in the Animations tab. When opened you need to select which plug-in you want to use for things like minimize windows and so forth. Also, to select a custom background with the "cube" you can do that in Skydome under "Desktop Cube" under Appearance. Further, to configure the "transparent" in the cube, you need to tell it to be transparent in the section for Rotate Cube. 

Wouldn't be boring if you couldn't configure it to what you liked? Instead just getting that one setup you saw on Youtube? Certainly nice but nonetheless, I like being able to configure everything. My windows, when I close them vanish into "blackness" (beam w/ black), my minimized windows go up in flames, my opened applications fly into the desktop (via airplane). Anyway, I hope the above helps you.
