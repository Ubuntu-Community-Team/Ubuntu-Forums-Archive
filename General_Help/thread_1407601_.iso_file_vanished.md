### Post by ironclaw on 2010-02-16
It sounds like the iso file was mounted by the 'Archive Mounter' utility in Ubuntu, and it might have been saved only as a temporary file if you did that directly from Firefox or another browser. Make sure that you select 'Save File' from the browser and not 'Open with *' to save the iso file.
