### Post by 3rdalbum on 2009-10-04
You've got your wires crossed a bit.

Shiretoko is the code-name for Firefox 3.5. When you install the firefox-3.5 package on Ubuntu 9.04, it installs under the Shiretoko name to signify that it's not the recommended version of Firefox for that Ubuntu version.

Regular Firefox will happily build and install on 64-bit platforms. x86_64 is a supported architecture for Mozilla.
