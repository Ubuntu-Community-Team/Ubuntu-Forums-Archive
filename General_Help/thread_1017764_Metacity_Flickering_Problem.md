### Post by IceReaver75 on 2008-12-21
I assuming you are using a nvidia card since thats what causes most of the title bar flicker problems.  There is couple of things you can do

1. Install emerald and use the emerald themes instead of the metacity window manager.

2. If you are using a nvidia card ..... goto the nvidia website and try the beta 180.11 or beta 180.16 driver.  They are beta drivers but I have a Geforce 6200 card and I'm using the 180.11 beta driver and I haven't had any problems at all yet and it took all flickering problems away.

Right now thats about the only 2 options you have while using a nvidia card.
