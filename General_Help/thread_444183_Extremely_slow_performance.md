### Post by MiCovran on 2007-05-15
If you want Ubuntu, "get more ram" is a good idea.
But hey, there are other distros for computers with less ram (you could try Xubuntu, for instance).

One more thing: check how much swap space you have: type 'sudo fdisk -l' in terminal (it's lower case L, not upper case i after a minus sign).

It is, but you have to compare oranges to oranges. Compare the memory and CPU usage of the OS Microsoft released in 2007 with the OS that Canonical released in 2007. You'll find that Ubuntu comes out favourably.
