### Post by roger_1960 on 2009-05-04
Hi

Suggest you read this: [http://www.psychocats.net/ubuntu/whichbuntu](http://www.psychocats.net/ubuntu/whichbuntu)

and follow the instructions from psychocats site to create a "Live" CD.  Try this in your laptop without installing to check it all works before installing to hard drive (should all work)  I suggest you try  Ubuntu 8.04LTS version as this has most of the bugs sorted out - you can upgrade later of course if you wish.

The one thing that may well cause a problem, if you use it, is the dial up modem.  These are sometimes possible to configure, but rarely easy.

NB If you have an external USB drive or stick available you may be able to extract any data you want from your XP machine using the live CD to copy from the internal hard drive to the external drive.

NB2 Its easiest to do a full clean install once you are happy it works - particularly if you have a 40GB hard drive which does not leave much room for dual booting 2 OSs. 

Good luck!!

### Post by roger_1960 on 2009-05-04
Hi again

When you have downloaded the .iso, "burn" it to disk (not just a copy) as described by psychocats.  Once done and checked (md5sum) you turn off your laptop with the disk in the CD drive and turn it on again and it should boot to the CD.  If it doesn't, you may have to enter the BIOS settings of the laptop (by pressing F2 or F8 immediately on powering up - there is usually a mesage on screen) and setting the boot order so that the CD drive comes before the hard drive.  Normally this is the default anyway.

Note that running from the "live" CD it will go pretty slowly - its just to test compatibility before installing. You should get a boot options menu when its starts up with the CD in the drive.  I have found that Ubuntu has given new life to my old laptops which really struggled to run with XP SP3!
