### Post by rbmorse on 2008-04-29
Either the wrong printer driver is installed, or the correct printer driver was selected but did not install correctly. 

system > administration > printing 

find the line that reads "make and model" and make sure that is correct, or click on the "change" button and select the correct printer driver if it is not.
