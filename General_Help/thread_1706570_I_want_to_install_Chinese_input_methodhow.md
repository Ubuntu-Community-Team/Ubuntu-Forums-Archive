### Post by Legeril on 2011-03-14
I've managed to get some Chinese input for Open Office like this:

System > Preferences > iBus Preferences > Input Method > Select Input Method

Log out and back in again, you should now see a friendly little keyboard on your top panel. You can select Chinese and/or Pinyin when using Office software.
