### Post by 3rdalbum on 2009-03-15
I think you should read the documentation for the program. I imagine not a lot of people here even know what "linear programming" is. I don't.

The message that the program is giving you is that it's telling you how to use it. It's saying that it requires you to specify a file - an "mps" file, I guess.

You would type '/usr/bin/PCx ' and then the path to the file, and then press Enter.

As such, this looks like a command-line program, which is why it didn't appear in your Applications menu.
