### Post by dagnabit dang doohickey on 2009-02-01
Firefox has a built-in popup blocker. Look under the Content tab in the preferences.

You can also use the [/etc/hosts]("http://linuxhelp.blogspot.com/2007/08/tweak-your-host-file-to-block-ad.html") file to block sites.

### Post by mb_webguy on 2009-02-01
Adblock Plus is definitely the way to go.  While it's designed to block ads, you can block any content from any website, or even entire websites if you want.  You can also install the Adblock Plus Element Hiding Helper to make it easier to block items that aren't already blocked by default.
