### Post by hwttdz on 2011-11-20
I think all but the illumination bar are compiz plugins.  So take a look at "compiz config settings manager" (command line "ccsm"), and the alt+tab behavior falls under window management, any of the various "switchers".  I think that window resizing behavior is the scale addon.

The default settings differing from your preferred settings is not necessarily something "wrong" with the distribution.  I also tend to have specific needs which differ from those of the average user, so I always start with the minimal install, and install exactly those packages which I need and set everything up just the way I like it.
