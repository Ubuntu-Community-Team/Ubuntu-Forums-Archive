### Post by gzarkadas on 2010-08-08
You don't need to build it. It is already built. Just unpack the archive in a directory, go with Nautilus to the bin subdirectory, select all files in there (there are shell scripts) and make them executable (right click|Properties|Rights tab|check the Execute box). Then run whichever you want.
