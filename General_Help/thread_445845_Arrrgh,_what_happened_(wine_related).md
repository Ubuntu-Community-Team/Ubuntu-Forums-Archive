### Post by Turellius on 2007-05-16
I don't know if this will help, but I was trying to downgrade Beryl from a version I had, and it wouldn't let me do it either.  I found out I needed to edit the sources list.  Once I did that I was able to install the older version.  So maybe look around on the net for the version you want to install and find out what the repository is for that and add it to the sources list and remove the entry you already have for WINE.  Just a thought, it  might not apply to you.
