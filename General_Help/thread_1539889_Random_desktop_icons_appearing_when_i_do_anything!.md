### Post by ajgreeny on 2010-07-27
How are you downloading and updating?  If you are using your web-browser (firefox) it is probably set to download to the desktop.

However, you should be updating using the system's own updater application, **System->Administration ->Update Manager**, and your straightforward downloads through firefox can be put to any folder you like by using the **firefox ->edit ->preferences** menu item.
