### Post by tacutu on 2008-04-30
you probably already have scim installed on your computer. to select the desired input method you can use im-switch from the terminal and then restart X. Try im-switch --help

---

### Post by tacutu on 2008-04-30
also, after using im-switch to select your input method (should be scim or scim-bridge -- apparenlty the second one works better), go to System->Preferences->SCIM input method setup to set your preferences
