### Post by louis_nichols on 2007-02-24
You need to start Ubuntu in recovery mode then. When the command prompt appears, issue the command:

passwd *username*

This will ask you to type a new password twice and will change it. Then type *reboot* to restart computer normally and you will be able to login. Make sure you don't forget the password again, though. :p
