### Post by Yellow Pasque on 2009-05-08
> **Roasted said:**
> Okay, now how do I get rid of this? Now that I installed this every time I go to myspace, firefox crashes. Every... single... time... even if YouTube isn't open.
Close any Firefox sessions, delete the file you installed ( ~/.mozilla/libflashplayer.so ) and reinstall whatever you were using before. Make sure about:plugins reports the version of Flash you want.

EDIT: I'd also suggest using Flashblock so that only the Flash content you want will load. It might help minimize conflicts for you.
