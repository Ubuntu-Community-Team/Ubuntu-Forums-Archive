### Post by Sweet on 2006-04-03
Not all windows programs can be run under wine at the moment (maybe some day).

Try installing winetools, it is very very useful and can set you up with a working wine installation easily. It can also be used to automatically install applications, windows runtimes and other necessary components.

Simply type: 
```
**winetools**
``` at the terminal prompt.


If it asks you to run winetools as root then you can type:

```
**sudo winetools**
``` instead.
