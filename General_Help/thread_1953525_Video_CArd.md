I think nothing is wrong. When I suggest that you try Lubuntu or Xubuntu, it is two flavours of Ubuntu: the same kernel and programs, but two desktop environments, that are a bit lighter than 'vanilla' Ubuntu and Kubuntu. This means that the graphics will be faster (easier to get trouble-free playing of video clips and streaming video). And more memory will be available for application programs. It will be a nicer experience to have 1.75 GB instead of 0.75 GB. Several applications will work better, bigger files can be handled without swapping etc. But still, I would recommend Xubuntu or Lubuntu.

You need *not* select the flavour blindly from advice. If you have an internet connection with reasonable speed, please download several iso files and test them. They are free :KS

- 'vanilla' Ubuntu with Unity and gnome-shell
- Xubuntu with XFCE (light desktop environment)
- Lubuntu with LXDE (ultra-light desktop environment)

If you are new to linux, select either of the following versions

- 10.04 LTS very stable, but a bit old, support until April 2013
- 11.10, the current version

Or if you have some experience, consider the following version
- 12.04 beta, the coming LTS version with support for 5 years, not yet released, but already very good (I am running it successfully).

If you have a slow internet connection, you can download only one flavour and version, or even buy an Ubuntu CD [[COLOR="Red"]_http://www.ubuntu.com/download/ubuntu/cds_[/COLOR]]("http://www.ubuntu.com/download/ubuntu/cds")

After installation, you can add the other desktop environments with simple commands and select at the log in screen, which desktop to run. For example: if you download Xubuntu, *or buy an Ubuntu CD*, you can add Lubuntu with the following command
```
sudo apt-get install lubuntu-desktop
``` in a terminal window. (You need an internet connection to get the software to install).

When I write 'test them', I don't mean that you have to install them. Instead make install CD or USB drives (on an old PC it might be better to make CDs). Boot from the CDs and test them 'live'. During testing you can run any program, even 'install' a proprietary driver for your nvidia card. And after that it will be easy to choose what flavour and version, that is the best.
