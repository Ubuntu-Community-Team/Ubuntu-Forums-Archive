### Post by bleedingsamurai on 2011-03-20
If you feel comfortable with Unix then why not. While DesktopBSD is geared at the average user, *BSD is a different culture then GNU/Linux. There are some small but important differences, for example the default shell is the Korn Shell(ksh) instead of the Bourne Again Shell(bash) 
Because *BSD people prefer a "sophisticated-engineering" approach to the "dirty-nasty-hack" typical of GNU/Linux there is usually less hardware support so make sure you look into that first.

To do the install, it is acutally quite easy as I  recall. DesktopBSD has a nice GUI-ish installer.
here is a guide for you:
[http://www.desktopbsd.net/wiki/doku.php?id=doc:installation](http://www.desktopbsd.net/wiki/doku.php?id=doc:installation)
To simplify things for you, you might want to go ahead and using a tool like gparted make a partition for the install befor booting off the install disk.
