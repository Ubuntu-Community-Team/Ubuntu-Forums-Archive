### Post by Victormd on 2008-07-28
You need to set the number of workspaces to 4. Open the Settings Manager, go to General Options and under the "Desktop Size" tab, set the horizontal virtual size to 4 (it seems like you currently have it set to 1).

### Post by RuleMaker on 2008-07-28
Ok, you have enabled everything you need for cube rotating.
You can rotate your cube by holding down ctrl+alt+left mouse button and drag your mouse.
If you have 2 workspaces you can add them by clicking on any of them on your panel, then select "Preferences" and set column number to 4.
