How can I tell Ubuntu to start File Browser with a certain directory (e.g.
/home/myname
as first directory?
 
How can I tell the File Browser to display files as List by default?
 
Peter

---

### Post by kaibob on 2009-10-21
The following will start nautilus in the /home/myname directory:

```
nautilus /home/myname
```

You can set list-view as the default in edit > preferences > views.

---

### Post by Just Pete on 2009-10-21
Also: to add a location to your "places" menu, simply bookmark them in Nautilus. Browse to the folder, click bookmark -> add bookmark and it'll show up in your places menu.
