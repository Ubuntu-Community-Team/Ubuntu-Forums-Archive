### Post by drs305 on 2008-05-07
Sometimes specific servers slow down. You can open synaptic and select Settings, Repositories, Other, Select Best Server to perhaps find a faster server. I don't know if gutsy servers are independent of the hardy ones, but generally the main servers are hit hard after a major release - you might find a faster server by searching for a less-used one.
