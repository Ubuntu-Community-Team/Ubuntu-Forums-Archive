### Post by matt_symes on 2016-01-03
Hi

Edit the file

```
/etc/dhcp/dhclient.conf
```

Look for the line that says

```
#prepend domain-name-servers 127.0.0.1;
```

Remove the # character and add your nameservers there.

Not tried this method with a VPN but it works in other cases so please post back on success or failure of this method.

BTW: What VPN ? Also are you leaking DNS requests by using 192.168.x.x ?

### Post by SeijiSensei on 2016-01-04
Edit the file /etc/resolvconf/resolv.conf.d/head and place your nameserver declarations there.  Resolvconf manages resolv.conf in 12.04 and later.  The contents of the "head" file will be placed above any dynamically-created entries.
