### Post by Vaphell on 2010-10-24
Every ide is slightly different but most use the concept of project with its own separate dir where all stuff is kept so once you've seen one, you have a general idea what you deal with.
Simply run it, create few simple projects including obligatory 'hello world' and see what it does and how.

Python is ok even with good ol' gedit, C/C++ is harder to configure your project correctly by hand so you get a bigger boost from using an IDE which handles most of the hard stuff for you.
For more detailed info, goto Programming talk subforum (inside the Development and Programing forum)
