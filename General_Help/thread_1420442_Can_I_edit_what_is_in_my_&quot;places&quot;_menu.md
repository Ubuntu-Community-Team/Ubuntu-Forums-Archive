### Post by derrick81787 on 2010-03-03
You can also use a text editor to edit the file .gtk-bookmarks that is in your home directory.  It is hidden, so you won't see it in Nautilus.  You can add or subtract whatever you want.  Someone on these forums even put Windows shared network folders in there and it worked.  I think he added the line "network:///" but how to do regular directories should be pretty obvious once you have the file open because you can just basically copy what is already there.
