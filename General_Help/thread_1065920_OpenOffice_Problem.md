### Post by blueturtl on 2009-02-10
The problem apparently lies with the new nVidia driver and Compiz.
Try disabling Desktop Effects and see if that helps (found in System -> Preferences, set the effects to None).

I had a client with a GeForce MX 440 and he had the same problem.
The solution above fixed it for him.
