Does your computer prompt you with a reason for not opening the folder?  Does your windows manager display the properties or contents of the folder?  Can you access it in terminal?  Need some more info to help you.

There is no limit of how much data can be in a folder.  The only limit I know of dealing with file size is that FAT32 formats can not handle FILES (not folders) larger then 4GB.
