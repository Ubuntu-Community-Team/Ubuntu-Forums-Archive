### Post by undecim on 2009-11-14
Compiz and Emerald still exist. Compiz is already installed on Ubuntu by default and can be configured by the Advanced Desktop Effects Settings (ccsm) program, which you can install from the Ubuntu Software Center.

While you're at the Software Center, you can install Emerald Theme Manager to get Emerald. Emerald allows you to make custom window borders and decorations.

I also recommend installing the Compiz Fusion Icon, which will allow you to switch between Compiz and Metacity (it's a good idea to switch to metacity when using fullscreen apps) and also gives you quick access to the configuration managers for each of these.
