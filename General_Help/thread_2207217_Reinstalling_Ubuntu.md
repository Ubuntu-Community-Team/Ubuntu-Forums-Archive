### Post by Mark Phelps on 2014-02-22
> while all the files are saved on the external hard drive.

The way you have stated it -- no -- because some files will be saved on the internal drive.

However, you can go into each app you use to download files and set the preferences to point to a folder on an external drive.  But, as far as I know, you have to do that one app at a time.
