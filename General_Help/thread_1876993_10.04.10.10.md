### Post by ajgreeny on 2011-11-07
If you are really asking if it is worth moving from 10.04 to 10.10, I would say don't bother.

10.04 still has 18 months further support, but 10.10 only six months left, and the differences once installed are minimal.  You can install the ubuntu font family in 10.04 now from synaptic, so that minor advantage is no more.  If you search for PPA repos it is also possible to update many packages to the same versions as 10.10, so there is no real difference there either

If 10.04 works for you leave it be!
