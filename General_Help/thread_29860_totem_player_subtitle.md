### Post by yanik on 2005-04-27
take a look at the file ~/.gnome2/totem_config

I use it to enlarge subtitle size, see:
```
# subtitle size
# { tiny  small  normal  large  very large  huge }, default: 1
subtitles.separate.subtitle_size:large
```

But there's plenty of cool stuff in this file, have fun :D

Yanik
