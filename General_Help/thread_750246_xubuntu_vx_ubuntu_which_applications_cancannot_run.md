### Post by ibutho on 2008-04-09
All the applications in Ubuntu can also be installed in Xubuntu. The problem is that if you are a bit limited in terms of ram, some apps will be dog slow which will make the system sluggish. XFCE uses an Ubuntu base but instead of using GNOME, it uses XFCE and includes default apps which are a bit lighter on system resources. It will certainly be helpful if you can get a bit more ram (384 MB seems to be the recommended for Ubuntu).
