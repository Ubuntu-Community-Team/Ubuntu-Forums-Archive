### Post by isecore on 2010-07-11
If I understand you correctly, you're asking if Ubuntu provides some sort of software to show wether the components in your computer will work with Ubuntu.

No, they don't. But it's not really needed either, since very few things are incompatible with Ubuntu. I'd suggest downloading the Live-CD and booting from it. If everything seems to work while using the Live-CD then it will work just fine after an installation.

I would be very surprised if there are problems, but give it a go.

I agree, this is what you are looking for. To do this, you boot the computer with the CD in the drive, after making sure that the computer is set to boot from the CD (or DVD) drive (this is in BIOS, or available by pressing some key during boot up).
A menu will appear, and you choose the option "try Ubuntu without changing the computer". Ubuntu will be started from the CD, and you can use it and see if every thing works properly on the computer (which it probably will).

### Post by efflandt on 2010-07-11
If you have any trouble booting the live CD, see the release notes to work around issues [https://wiki.ubuntu.com/LucidLynx/ReleaseNotes](https://wiki.ubuntu.com/LucidLynx/ReleaseNotes)

If you have to do do anything special for the live/install CD you can hit any key when you see initial graphics at screen bottom.

For example my older desktop and one laptop do not like the new kernel mode setting (kms) video driver.  The desktop booted fine, but its video was slow and it would not suspend or hibernate.  The laptop would sometimes hang during boot with a video glitch.  The **nomodeset** kernel boot option resolved all of those issues.

As with all systems, a better graphics card will give you better graphics, but you will only notice the difference with very complicated games or maybe videos.
Ubuntu will run with the graphics card you have. I believe that it will not have any problems (reduction in resolution or whatever).

If your laptop is running to warm, the first thing to do is open it up and make sure all the fans and cooling vents are not filled with dust.
