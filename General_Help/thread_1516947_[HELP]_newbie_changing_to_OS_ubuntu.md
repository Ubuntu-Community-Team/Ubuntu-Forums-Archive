### Post by WorMzy on 2010-06-24
Linux will work fine on ~99% of hardware, but the best thing to do is download a Live CD image (available [here]("http://www.ubuntu.com/")), burn it to a CD, then boot into it and try it out. If it works off a Live CD, then it'll work once it's installed on a harddrive. Just make sure you test your internet connection while using the Live CD; Linux often has problems when trying to use some Wireless cards.

### Post by snowpine on 2010-06-24
Hi Zagvot, welcome to the forums!

Ubuntu System Requirements are published here:
[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

Your hardware is below the Recommended Minimum, but many users have had success with only 512mb of RAM. If you can increase your RAM to 1gb, it would help a lot.
