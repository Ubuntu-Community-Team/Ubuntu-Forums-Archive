### Post by Autodave on 2022-02-03
Sounds to me like you are having issues with either you power supply or possibly a USB hub.  USB hubs and especially *powered* USB hubs can cause problems like that.  If you are using any USB hubs, disconnect them.  If problem still persists, try another mouse and keyboard.  If problem is still there look at the power supply.
