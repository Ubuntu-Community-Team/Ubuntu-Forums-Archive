### Post by dandnsmith on 2012-06-08
Interesting problem!
The first thing that springs to mind is using a voice-only version of Skype with a suitable plug-in Skype phone.
Alternatively, using an ATM converter to connect a standard phone to your router, if you have one and it handles VOIP.

Given a few details of the overall satellite link setup, a few more possibilities might crop up - it's all a bit nebulous at the moment.
