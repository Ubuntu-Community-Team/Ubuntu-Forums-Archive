### Post by Mark Phelps on 2012-03-29
IF you're so insistent about being a noob, then you should have posted your question in the Absolute Beginner Topics forum, not here.

That said, you should not have to resort to hand-editing anything to simply change your boot default, especially not the GRUB.CFG file which clearly contains a warning about NOT changing it!

Read the link below, install Grub-Customizer, and you can use it to set the default OS:

[http://ubuntuforums.org/showthread.php?t=1664134&highlight=Grub+Customizer]("http://ubuntuforums.org/showthread.php?t=1664134&highlight=Grub+Customizer")
