### Post by The Cog on 2012-08-20
A wubi install is installed inside of microsoft's installation. 
You will have to back up all your data that you want to save, and re-install Ubuntu. If you choose to use the whole disk for the installation, then windows will be overwritten.

### Post by lukeiamyourfather on 2012-08-20
If you want to use only Ubuntu then boot from the disc and do a normal install. When the installer gets to the partitioning section there's an option to erase everything and let Ubuntu use the entire hard disk. Be sure to backup anything you want to keep before doing this.
