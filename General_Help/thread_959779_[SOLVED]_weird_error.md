### Post by ryanhaigh on 2008-10-26
That error was probably from AWN. AWN requires a composite window manager for the transparency and other effects. It sounds as though compiz might have crashed for whatever reason. If it happens again you can simply press alt-f2 to bring up the run dialog and then run the command: compiz --replace

That will restart compiz and your AWN bar should work again.
