### Post by cdillard-hsp on 2014-09-01
This is an odd one... On three of our *buntu PCs (Ubuntu 14.04, Ubuntu 13.10 and Xubuntu 13.10), MP3 files downloaded using a browser (Chrome and FF) show up in Downloads but some of the files mysteriously vanish.  The behavior is the same on both computers.  You can open the Downloads folder and watch the contents update as the MP3 files are downloaded.  While looking at the directory contents in Nautilus you can see some of the MP3 files disappear.  I have no idea where they are going or why.

### Post by amanchesterman on 2014-09-01
Some obvious things to try are:
- check that the files aren't being hidden because of unusual characters in the file names
- check that you have ownership of the files 
- check that Nautilus is refreshing the file list (close Nautilus and restart it, reboot your machine etc.)
