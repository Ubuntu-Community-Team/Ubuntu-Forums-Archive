### Post by dragonfly41 on 2016-08-04
You do not say what version of lubuntu you have installed.
On my Ubuntu 14.04 (an old laptop) I have Konqueror installed but I never use it so I can't comment on its suitability.
It is found under Applications > Internet > Konqueror
If you do not have it installed go to Applications > Ubuntu Software Centre
and search for konqueror to install.

Or try ```
sudo apt-get install konqueror
```
