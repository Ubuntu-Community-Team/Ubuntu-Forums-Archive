### Post by loklaan on 2012-05-17
So installing the latest drivers made things worse? That is sad to hear.  Get rid of everything Nvidia for a moment to clarify if your graphics card is the problem. And also, remove and reinstall Xorg.
```
sudo apt-get purge nvidia*
sudo rm /etc/X11/xorg.conf
sudo apt-get install --reinstall xserver-xorg-core libgl1-mesa-glx:i386 libgl1-mesa-dri:i386 libgl1-mesa-glx:amd64 libgl1-mesa-dri:amd64
sudo dpkg-reconfigure xserver-xorg
sudo reboot

```

### Post by popebrak on 2013-01-09
Greetings! 

I know it's been a while, but I've run into a very similar problem.

I'm running a fresh-install of Ubuntu 12.04 Desktop x86_64 with an NVIDIA Corporation G96 [GeForce 9500 GS] (rev a1), using the latest 'experimental' NVIDIA driver (because it's required for Steam which, incidentally, is a real pain-in-the-kneck on a 64bit system). 

I also run the "Gnome Classic" (gnome-session-fallback) interface with some compiz customizations. Every once in a while, when making a change within the compizconfig-settings-manager. . . the compiz-decorator crashes, and as the OP describes, my window title-bars disappear, and I can no longer move anything around on the desktop.

My "fix" for this has been to mercilessly delete .config/ and .gnome2/ files until the system rebuilds the necessary bits from default settings, and my window-manager goes back to behaving normally. I then go back to _carefully_ re-activating my compiz tweaks. You can actually export and import your compiz settings from the "Preferences" submenu of the compizconfig-settings-manager, so that may help the recovery process a bit less painful.

I know that's not really much of a fix, but I'm afraid it's what I have to offer. My impression is that if you want to run gnome-classic with compiz, you can simply expect some difficulty.

Which is a shame, because this all worked very smoothly under Ubuntu 10.04. I understand that there's market-value in the child-friendly Unity interface, but the ability to have scores of selectively-transparent windows situated on a selectively-transparent rotating desktop cube is of tremendous productivity value. Once you get it working properly on multiple monitors, it's quite literally the most impressive workstation interface most users have ever seen.
