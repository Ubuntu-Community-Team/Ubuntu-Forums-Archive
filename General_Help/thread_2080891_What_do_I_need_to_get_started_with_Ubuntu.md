### Post by CrusaderAD on 2012-11-05
Hello! You can install Ubuntu and once you're all setup, you can right click any folder and click share folder. This will share the folder on the network. You can give it guest access too in case you don't want to bother with a username and password. You can do this for any number of folders. It uses the Samba share system to establish this.

### Post by newb85 on 2012-11-05
I believe the only thing you'll need to install once you have Ubuntu installed is Samba.

More information on Samba:
[https://help.ubuntu.com/community/Samba](https://help.ubuntu.com/community/Samba)
[https://help.ubuntu.com/community/Samba/SambaServerGuide](https://help.ubuntu.com/community/Samba/SambaServerGuide)

If you're using an older machine with low specs (WinXP seems to indicate you are), you might need to use something lighter-weight, like Xubuntu or Lubuntu.  (They're basically the same thing as Ubuntu, but with lighter Desktop Environments/user interfaces.)  Or, if you're comfortable doing all the server administration from the Command Line Interface and you don't need to use the machine as a workstation, you could install Ubuntu Server Edition, which doesn't include any graphical Desktop Environment.

What are the specs for the machine in question?
