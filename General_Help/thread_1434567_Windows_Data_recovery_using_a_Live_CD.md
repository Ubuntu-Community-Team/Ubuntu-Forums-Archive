### Post by DawieS on 2010-03-20
Yes, all Ubuntu Live CD's also support NTFS. Just set the BIOS to boot from CD, insert the CD and boot. At the first menu choose "Try without making changes" and then recover the data to an external drive.

Alternatively, you can boot from a USB flash-drive, on which Ubuntu was pre-installed.:grin:

EDIT: I am not sure about earlier versions, but 9.04 and 9.10 definitely supports NTFS.
