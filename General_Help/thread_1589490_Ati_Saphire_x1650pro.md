> So i have a Ati saphire x1650 pro iv just install ubuntu 10.04. the problem is that i cant find any driver for it ...

That's because their aren't any.

AMD (formerly, ATI) dropped proprietary driver support for your card long ago.  The current drivers will not work with your card and the older drivers that WILL, can not be used with Ubuntu versions newer than 8.10.

The open-source drivers are the only ones available now and they are installed by default during initial setup.
