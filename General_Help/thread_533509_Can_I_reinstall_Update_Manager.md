### Post by jamvegan on 2007-08-24
I don't think that uninstalling and reinstalling those packages should cause you any problems.  Although ubuntu-desktop sounds critical, in fact all that it is is a list of dependencies.  Installing it helps to ensure that you have all the desktop components installed that you need, but removing it does not actually remove any software from your computer.
