Just another disconnect between Gnome & Ubuntu, Ubuntu should ditch Gnome altogether.

For 16.04 this will work, if it has any side effects don't know. (edit - see below for one

Install xserver-xorg-input-libinput 
Open the config file as such
```
sudo -H gedit  /usr/share/X11/xorg.conf.d/90-libinput.conf
```

Add the line in blue to the 'touchpad catchall' section, save, log out. Should give about a 2 sec timeout while typing
```
Section "InputClass"
        Identifier "libinput touchpad catchall"
        MatchIsTouchpad "on"
        MatchDevicePath "/dev/input/event*"
        Driver "libinput"
	[COLOR="#0000CD"]Option "DisableWhileTyping" "true" [/COLOR]
EndSection
```
One side effect of installing that package is one would lose any of the few previous touchpad settings in System Settings so basically 16.04 becomes a matter of trade offs.
