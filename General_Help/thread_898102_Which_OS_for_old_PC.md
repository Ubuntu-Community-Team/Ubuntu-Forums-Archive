if she is just doing basic stuff, internet, office, edit photos, then puppy linux is a great choice. it is so fast you can't believe it.  it looks really nice and it is a piece of cake to use.

### Post by amitkher on 2008-08-23
As you have enough hard disk space, distribution does not matter. You can use Ubuntu , but use the alternate install CD as the normal one will refuse to install.

Once installed, just switch to a lighter window manager. You can try openbox, fluxbox, fvwm, icewm (in that order :) ). You just have to install these, and when logging in, choose the window manager. All of these run great in 128 MB RAM. 256 is royal. 

Even the default environmnet of Ubuntu (gnome) will start up but will be slow.
