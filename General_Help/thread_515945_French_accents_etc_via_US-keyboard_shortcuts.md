### Post by Jim Connachan on 2007-08-02
The following works on a UK keyboard it may work on the US keyboard.
To get accents on your letters hold down the Windows key type ' then followed by letter e.
e.g. é acute 
e.g Windows key type `  followed by e --> è e grave
e.g Windows key type ~ followed by n --> ñ tilde
e.g Windows key type , followed by c --> ç cedilla
e.g Windows key type " followed by a --> ä umlaut
e.g Windows key type ^ followed by a,e,i,o,u --> circumflex

Hope this helps

I don't think there is another solution on a US keyboard. Once you have your "compose" key, you punch in the accent and then the letter. Also works in reverse order. It beats the Alt + number codes in Windows. If you type a lot of french letters, your best bet is to use the Canadian English keyboard but then a lot of keys are different. You might know someone in Quebec who is throwing out an old computer and get the french keyboard - at least you would see which key is which. Just some extra ideas.
