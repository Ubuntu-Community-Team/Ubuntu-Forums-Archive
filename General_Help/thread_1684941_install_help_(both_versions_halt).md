### Post by 3rdalbum on 2011-02-09
The netbook version is optimised for netbooks and is actually a little heavier than the desktop version.

How much RAM does this computer have? Does it satisfy Ubuntu's minimum requirements? If not, then that's your likely problem.

How far does it get before it stops working? If you can get to the screen which shows a picture of a man and a keyboard, press the space bar there and use some of the alternate boot options listed on the screen that appears.
