### Post by moma on 2007-05-20
Install the "imagemagick" package. It comes with a display command. Eg.
$ [COLOR="SeaGreen"]display  picture.jpg[/COLOR]
or
$ [COLOR="SeaGreen"]display  *png[/COLOR]

Study: [http://www.imagemagick.org/script/command-line-tools.php](http://www.imagemagick.org/script/command-line-tools.php)
Imagemagick can do anything with images. 

My favourite guides are these
[Guide_1...]("http://www-128.ibm.com/developerworks/library/l-graf2/?ca=dgr-lnxw11GraphicsLine%3Cbr")
[Guide_2...]("http://www.imagemagick.org/Usage/")
[Guide_3...]("http://applications.linux.com/applications/05/03/29/1525217.shtml?tid=39&tid=49&tid=47")

Well, [graphicsmagick]("http://www.graphicsmagick.org/") is a good alternative.
