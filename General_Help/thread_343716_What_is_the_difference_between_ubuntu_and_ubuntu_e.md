### Post by aysiu on 2007-01-21
What "Enterprise Edition"?

There are only five **releases** of Ubuntu:
6.10 Edgy Eft (October 2006)
6.06 Dapper Drake (June 2006)
5.10 Breezy Badger (October 2005)
5.04 Hoary Hedgehog (April 2005)
4.10 Warty Warthog (October 2004)

Within each release, there are several different **versions** (which are really just different default packages and artwork):
Ubuntu (featuring regular desktop applications and the Gnome desktop environment)
Kubuntu (featuring regular desktop applications and the KDE desktop environment--just something else)
Xubuntu (featuring regular desktop applications and the XFCE desktop environment--for lower-end systems)
Edubuntu (featuring education-related applications and the Gnome desktop environment)

There are also three types of **disks** you can use to install Ubuntu (or Kubuntu or Xubuntu or Edubuntu): a Desktop CD (runs a live session that gives you a preview of Ubuntu without installing it, but it also allows you to install Ubuntu to your computer), the Alternate CD (provides you various specialized ways to install Ubuntu), and the Server CD (for installing a server, as opposed to a desktop).

None is the "Enterprise Edition." All are the regular, free and fully supported edition.

If you are a regular desktop user who has 512 MB of RAM or more, I would advise using the 6.10 *release* of the Desktop CD *disk* of Ubuntu (*version*) as a starter.
