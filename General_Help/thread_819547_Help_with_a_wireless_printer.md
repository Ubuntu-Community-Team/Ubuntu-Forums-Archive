I just set up a wireless network for a family member using an Apple AirPort Extreme as the wireless router because of it's ability to share a printer wirelessly. The printer is a Dell All-in-one 922 and I am trying to connect it to their Ubuntu 8.04 laptop. Any tips?

### Post by cprofitt on 2008-06-06
You should be able to 'browse' to the router (I hope Apple allows that) and get a list of attached devices. After that you should know the ip address of the printer... you can then setup printing using that IP address... unless the AP Extreme is actually using a USB connection to the printer... in which case you would have to determine what kind of sharing Apple uses -- you would have to browse to that printer share.
