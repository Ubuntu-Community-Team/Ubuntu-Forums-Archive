If I can find the original disks, I may just reinstall Office.  I don't have Word on my other Windows machine, so I can't check if it's just that laptop.  As far as I can tell, the print job isn't even getting to the Xubuntu server.

### Post by Jmccaffrey on 2007-03-10
What driver are you using on the windows computer? I would suggest using a postscript driver for the print clients.  It has alot less error rate in my mind than using real print drivers on the clients.

---

### Post by angrytrousers on 2007-03-13
Great suggestion!  I reinstalled the printer on the Windows machine using a generic Microsoft driver and it worked like a charm.  One of those easy solutions that makes me feel stoopid.  I was too focused on the Xubuntu machine.  Of course it was a Windows problem :)

I love my new print server!  I feel so cool printing wirelessly.
