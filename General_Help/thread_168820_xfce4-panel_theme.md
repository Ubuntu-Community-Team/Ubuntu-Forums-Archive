### Post by hollywoodb on 2006-05-01
Well, the panel buttons can be modified... the color should be set by your GTK2 (button/app style, not window border) theme... the icons should be set by your chosen icon theme.  Beyond that there's not really any such thing as an xfce4 panel theme.
