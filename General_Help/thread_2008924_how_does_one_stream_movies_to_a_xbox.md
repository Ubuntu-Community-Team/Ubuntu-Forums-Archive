### Post by GWBouge on 2012-06-23
You can check out something like XBMC or MediaTomb.  They're both in the Software Center.

XBMC is a full media center, such as for an HTPC.  So, to really get the most out of it, it helps to have it on a dedicated box.  But, you could throw it on an HTPC in the living room and stream movies to an X-Box in a bedroom, for example.

MediaTomb is just a media server that runs in the background, which can stream to any UPnP enabled device.  Configuration is done via web browser.  I'm not sure what media types an X-Box can play, but I remember having to mess with some on-the-fly-transcoding scripts to get .mkv's to stream to a PS3.  So, it helps to have a compatible library  =o)
