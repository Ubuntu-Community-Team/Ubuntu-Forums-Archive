Hi, I'm a Mac user who needs to switch to using ubuntu for work. What's making the switch difficult is that there are some shortcuts which I heavily depend on and that's making my switch to Ubuntu difficult. 

On the Mac, I use Command + ` (the tilda) to cycle through windows in the current application (eg terminal or chrome). It's like alt + tab except only for the current application's windows. I also don't want any flashy graphics like exposee or anything. Is there anyway to do this on Ubuntu? Thanks!

### Post by jerome1232 on 2011-06-23
Yes and no.

That functionality exists, it's just not turned on by default.

If you goto your keyboard shortcuts (gnome classic that's.... System-Prefrences-Keyboard Shortcuts, in Unity it's Shutdown Button-System Settings-Keyboard Shortcuts)

Scroll through the list until you get to the window managment section, find "Move between windows of an applications, using a popup window" and define your own shortcut for it.
