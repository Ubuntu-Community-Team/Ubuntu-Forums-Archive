### Post by bumanie on 2008-12-31
You could try installing clamav and the GUI clamgtk in ubuntu and then scanning you windows drive with that - if it is a virus, clam will most likely identify it and isolate it. In the clam GUI go to file and choose recursive scan. Double click windows partition and choose which folders to scan - Clam will be under Applications --> System Tools --> Virus scanner. There is a tick box under options that allows you to choose 'no maximum', without having that checked, clam will not scan large files.
