### Post by aysiu on 2006-06-11
The desktop environment is the main difference... and programs.

For example, Ubuntu uses Synaptic. Kubuntu uses Adept.
Ubuntu uses Gedit. Kubuntu uses Kate.
Ubuntu uses Firefox. Kubuntu uses Konqueror.
