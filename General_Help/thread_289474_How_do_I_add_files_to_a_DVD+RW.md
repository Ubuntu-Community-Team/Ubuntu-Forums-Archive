### Post by dannyboy79 on 2006-10-31
i would have to disagree, a multi-session dvd+rw is a multi-session dvd whether it was written in winxp or linux. there is a great website for this kind of a question, check it out for yourself: [http://fy.chalmers.se/~appro/linux/DVD+RW/#compat](http://fy.chalmers.se/~appro/linux/DVD+RW/#compat) and [http://fy.chalmers.se/~appro/linux/DVD+RW/](http://fy.chalmers.se/~appro/linux/DVD+RW/) but obviously to be on the safe side I would just write the contents to your linux box and then when you try to write the new files to the disk it won't matter if you're old contents get overwritten because you already saved them to your linux box, this would be the best way to find out if this'll work. let us know when you're done what the results are.
