### Post by Monotoko on 2009-04-03
There are 3 desktop managers you can use, xfce, KDE or Gnome

To get KDE you need to use
```
sudo apt-get install kubuntu-kde4-desktop
```

so you can use ubuntu, kubuntu and xfce desktop all in the same OS ;)
