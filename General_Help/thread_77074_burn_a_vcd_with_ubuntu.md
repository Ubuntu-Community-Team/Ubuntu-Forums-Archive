### Post by kabus on 2005-10-16
To do it from the command line :

1. Install cdrdao and vcdimager

```
sudo apt-get install cdrdao vcdimager
```

2. Create a VCD Image from your MPEG file

```
vcdimager yourvcd.mpg
```
(replace "yourvcd.mpg" with the actual filename)

3. Burn to cd

```
cdrdao write --device /dev/hdx videocd.cue
```
(replace "hdx" with your cd burner. the videocd.cue was generated by vcdimager in the previous step)

Hope this helps.

### Post by mr_manny on 2005-10-16
If you are looking to make a backup copy of an existing VCD, performing the following:
 
load a VCD into your CDROM and perform the following to create an image on your HD:
(define VCDimage name)                                                                                                                                                                                                                          
cdrdao copy --keepimage --device 1,3,0 --driver generic-mmc --multi --datafile VCD1.img
 
you will be prompted to load a blank cd to continue (w/the write step)...you want to perform this step later, you can CTL^C to exit.

To write any images you have on your HD, perform the following:
(.toc file associated w/VCD1.img from the copy step)
cdrdao write --device 1,3,0 --driver generic-mmc cd9961.toc

your device target values will varry, perform a "cdrecord --scanbus" to idenfity yours. 

hope this helps
