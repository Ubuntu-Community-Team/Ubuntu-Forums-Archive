### Post by Soupbean on 2010-12-05
most programs are for Chinese-->English (eg **DimSum**).

**Wenlin** is a windows program but it runs under wine and will do english to chinese with pinyin. (from memory it doesn't have a pop up for the definition, but the definition appears at the bottom of the screen when you hover the mouse over the word so pretty much the same thing) 

not sure if there are any **offline** dictionaries besides **stardict** that do pop-up translations

for **online** study, it might be worth checking out some of the **firefox** or **chrome** **plugins**, they might offer some of the functionality that you're looking for.
