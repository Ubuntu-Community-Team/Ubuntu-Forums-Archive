### Post by kd5mkv on 2011-11-06
Fldigi has rig control, and adds weekly maintance to the software. try this in terminal:
$ sudo add-apt-repository ppa:kamalmostafa/fldigi
$ sudo apt-get update
$ sudo apt-get install fldigi

Not only will this install on your system your regular fldigi updates with system updates!
