### Post by ubuntu27 on 2011-10-17
Natty Narwhal is the code name for Ubuntu 11.04.

**Officially**, a Ubuntu version is not called by the codename, instead by its version number. The code name is only used *officially* for a ubuntu version that is under development. But, we are all used to call them by its codename. Its easier to remember for many of us. :popcorn:

Here is the [official explanation of Ubuntu Codenames]("https://wiki.kubuntu.org/DevelopmentCodeNames")

Here is a [list of Ubuntu Code names]("https://jaqoup.wordpress.com/2009/08/12/ubuntu-code-names/") with pictures.
