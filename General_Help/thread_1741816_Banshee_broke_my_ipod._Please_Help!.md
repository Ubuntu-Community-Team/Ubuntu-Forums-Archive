Luckily, I have an XP virtual box, so I was able to "restore" the iPod to factory settings from iTunes, which obviously sorted the problem. 

I'm almost afraid to use Banshee with my iPod again, but it would be great if someone knew why this was happening; had any suggested workarounds. I'm very much loving Banshee, and would be happy to get rid of Rhythmbox if I didn't need to hold onto it just for loading up my iPod.

For anyone else landing on this post: I'm not sure if there's a Linux native solution to recover your iPod from recovery mode (maybe someone else can make a suggestion), but if not, then a reset from within iTunes on Windows will "fix" it. Be warned, though, this solution is a factory reset, so you will loose all music, files and settings on your iPod.
