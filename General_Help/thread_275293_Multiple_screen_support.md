### Post by DADDA on 2006-10-11
Most Video cards do have double Video out put connections, the standard VGA(CRT Monitors)- and DVI-connection(TFT Monitors). If you run one monitor useing the DVI-connection and another running the old analog-VGA-connection it would not be to difficult to get a dual-monitor system up and running...

But useing double video cards and running just one X Server can be difficult. But with Linux nothing is impossible...

If you whant to run just one desktop, the first answear should be enough. But if you would like to have dual desktops (One desktop to every monitor) it can be hard to configure.

But don't take everything I say 100% serius, Iam still just a noob...
