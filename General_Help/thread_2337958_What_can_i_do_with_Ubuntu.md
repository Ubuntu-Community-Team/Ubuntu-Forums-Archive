### Post by Keith_Helms on 2016-09-23
Ubuntu is a Linux distribution and a large number of servers worldwide  run Linux, although probably more use Redhat than Ubuntu.  You can use  it for a web server, a mail server, an application server (J2EE is one  possibility),  or a DNS server.  You can stream multimedia files with it  or host an internet radio station.

If you want to start out with  something that's not likely to interfere with anything else in the  domain, tinkering with the Apache web server might be a good start.
