### Post by coldraven on 2011-01-23
Yes, Artha is what I use, it is in the Software centre.
Once installed you can find it under Applications>Accessories.
When you run it an icon will appear on the taskbar or if you highlight a word and press Ctrl+Alt+W it will pop up with the definition.
If you install the "sense index" you can solve crossword puzzles using wildcards.
So searching for c?t will find cat and cot. The "sense index" might be installed by default these days, if not look in the Synaptic Package Manager.
If you want it to run every boot-up the add Artha to your System>Preferences>Start-up Applications.
