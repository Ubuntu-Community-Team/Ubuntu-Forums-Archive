### Post by mc4man on 2010-10-27
pick any folder on your desktop, (if you don't have one create one w/ r. click -> create folder
Right click on the folder -> Open With -> Other Application. Scroll down to 'Open Folder', choose it and click open. This will set nautilus back to the default folder handler instead of mplayer

Edit:
After doing the above once (returning nautilus to the default), then **in the future** - if r.clicking on a dir. and using the 'Other Application' make sure the "Remember this application for "folder" files **is unchecked**

Otherwise the app chosen will become the new default -( long standing bug that the maintainers refuse to address

### Post by Frogs Hair on 2010-10-27
Open the file browser and right click the folders in home one by and select open with another application. Select file browser from the menu for each folder and make sure remember this application is checked. I hope this helps.
