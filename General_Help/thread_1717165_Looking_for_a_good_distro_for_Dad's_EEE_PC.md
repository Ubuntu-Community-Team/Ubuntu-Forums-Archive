### Post by oneleaftea on 2011-03-29
My Dad has an Asus 1005HA EEE PC that runs XP. He has a very bad problem of getting spyware and viruses and we are thinking of installing Linux for him. 

He only needs a browser and Skype so we need to make sure the webcam and wireless network works, as well as the trackpad. 

I tried a couple of lightweight distros like Slitaz and Puppeee. I loved Puppeee but the main drawback is that it seems like applications need to be updated manually. I love how Ubuntu (I use Linux Mint) updates everything so easily.

So for my Dad, the most important things are:
- Fairly lightweight
- Easy to get important security updates to the OS and applications
- Wifi and webcam works
- Would LOVE to be able to make it so the desktop environment cannot be tweaked. I know that one day he will lose an icon or a panel and have no idea what to do. He lives far away from me so I cannot help him at a moment's notice. 
- He probably does not even need a panel/taskbar. He just needs an icon for Chrome, Firefox, Skype, and network settings, and a shutdown button. Even sleep does not need to work if it is lightweight enough that it boots and shuts down fast.

I wonder if Anti-X or one of Mint's Debian builds (XFCE or future fluxbox release) will be good? Are those easy to update?

Thanks for any advice.

### Post by PhilGil on 2011-03-29
I'm using a bog-standard install of Debian Stable (Squeeze) with Gnome  desktop on my 1000h. Everything except wireless worked out of the box  (including the mike and webcam). Wireless was easily fixed by installing the  deb containing the kernel drivers.

EEPC's are well-supported by debian: [http://wiki.debian.org/DebianEeePC/Model/1005HA](http://wiki.debian.org/DebianEeePC/Model/1005HA)

Debian Squeeze is lightweight (mine idles at a bit over 100mb of RAM) and works well. Software isn't too old (roughly equivalent to 10.04 LTS) and will be supported for at least three more years. There is an update manager similar to Ubuntu's installed by default. You can be the administrator and fine-tune the privileges you'd like to give your father.
