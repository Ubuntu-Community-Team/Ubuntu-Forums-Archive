### Post by Dennis N on 2015-10-31
The same thing used to happen with Audacious (Places > clicking Home Folder opened Audacious instead). This is how I fixed it:

Open file manager, then using any folder in the file manager window: 

Right click on the folder, select Open With > Other Application > Caja (you will possibly see 2 different Caja icons - choose the one that looks like a file cabinet).

Selecting Places > Home Folder should now work as expected.
