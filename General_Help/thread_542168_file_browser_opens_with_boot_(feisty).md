### Post by rhodry on 2007-09-03
> **svtfmook said:**
> whenever i restart or boot, file browser opens.

not sure why.

anyone got a fix for this?

You may have saved a session with the program open.

Close all running programs. Then go to menu-
System > Preferences > Sessions

The third tab should be "Session Options"

At this stage, untick (if ticked) the Automatic save option; tick 'Save the current session'. You will get a message telling you session saved. 

Log out and log back in. The browser should be gone.

Rhodry.
