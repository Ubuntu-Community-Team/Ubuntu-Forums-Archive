### Post by gaebriel on 2006-12-16
You should be able to install directly over the top. The only issue I've run into before doing a linux install on top of a >= windows 2000 was if the disk was configured as a dynamic disk in windows. So, just in case, you may want to go into Disk Management (diskmgmt.msc from the Run menu) and convert the disk back to basic. When I ran into this issue it was several years ago, so I wouldn't be surprised if it's a non-issue anymore. 

Regarding your initial question, I recommend running linux as a stand-alone OS. That way you're kind of forced to learn how to use it and can't chicken out by going back to windows. Just make sure you've got internet access configuration down before you make the switch so you can access the forums and whatnot. Of course, to each their own. :) I don't play any of the games that require Windows, so I don't have to worry about that. 

FWIW
