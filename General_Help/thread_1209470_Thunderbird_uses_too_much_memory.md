### Post by nr0mx on 2009-07-10
Have you installed any Thunderbird extensions ? Problems I had with Thunderbird went away to a large extent when I got rid of some of the extensions. It's worth a try if nothing else works.

Currently I have these extensions enabled:
QuickFolders

.. and these disabled:
Adblock Plus
Contacts Sidebar
Gmail IMAP Account Setup
GMailUI
Nostalgy
Provider for Google Calendar: Allows bidirectional access to Google Calendar.
QuickFolders
QuickText
QuoteCollapse
ThreadVis
Lightning
Timezone Definitions for Mozilla Calendar
Zindus: Google and Zimbra contact sync for Thunderbird

Some of these are quite useful, but you can disable them after making use of them for the sync/setup. Also, I went from using the Lightning extension within Thunderbird to a standalone Sunbird install, which helped quite a bit.
