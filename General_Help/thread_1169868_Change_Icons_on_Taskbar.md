### Post by cmost on 2009-05-25
When you say "on the taskbar" can you clarify?  Are you speaking of the launchers that reside on the panel(s)?  If so, those icons can be changed the same way as with a desktop launcher (right-click them and select properties.)  If you're referring to the icons in the 'Applications' menu, then you need to use Alacarte to change the icons there.  Finally, if you're referring to the task tray (adjacent to the clock,) I have no idea how to change the icons that appear for certain applications there.  Good luck!
