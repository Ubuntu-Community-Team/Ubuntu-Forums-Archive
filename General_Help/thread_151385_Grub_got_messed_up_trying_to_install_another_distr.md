insert the ubuntu install cd, at the cd boot prompt type ```
rescue
```follow the instructions, and it will take you into a terminal, now enter ```
grub-install /dev/hda
```then enter ```
reboot
``` to restart your computer, this should restore grub. after doing this you still might have to add your windows and fedora core entry to the /boot/grub/menu.lst file though.

to add fedora and windows to the grub menu run
```
sudo gedit /boot/grub/menu.lst
```and add this to the bottom of the file.
```

title Fedora Core (2.6.15-1.2054_FC5)
	root (hd0,2)
	kernel /vmlinuz-2.6.15-1.2054_FC5 ro root=/dev/VolGroup00/LogVol00 rhgb quiet
	initrd /initrd-2.6.15-1.2054_FC5.img

title Windows XP
	rootnoverify (hd0,0)
	chainloader +1

```
and save the file.

[INDENT]ubuntu and fedora both install grub into /boot but unlike ubuntu, fedora uses LVM groups and has a seperate partition for /boot that it mounted at startup, in ubuntu all files are installed into one partition / but in seperate directorys, but fedora should work just fine, if you added the entry in the /boot/grub/menu.lst file for it. 

just a note though if you upgrade your kernel in fedora, you'll need to boot into ubuntu then manually add the entry in /boot/grub/menu.lst for the new kernel.[/INDENT]
