### Post by cavalier911 on 2010-09-21
The behaviour you request of the desktop (drag/drop,arrange icons,/media mounted drive icon autocreation)is provided in linux by gnome or kde. 
Lubuntu is a stripped down ubuntu version without gnome or kde.
Lubuntu's desktop is provided by it's filemanager pcmanfm.
All it does is mirror what is contained in ~/Desktop
