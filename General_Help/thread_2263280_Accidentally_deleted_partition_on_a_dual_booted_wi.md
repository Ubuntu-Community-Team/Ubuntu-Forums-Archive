### Post by Bruce_Gibbs on 2015-01-30
I assume you want to save valuable files and may be haven't backed them up recently? If you can run one or other of the operating systems then hopefully you can use it to find important files and shift them to a safe place. If your files are your main concern the next step would be to remove the hard drive and place it in an enclosure and view it from another computer. Then find the files and shift them to a safe place before using disk manager to repair the drive.

If you just want to fix the drive and restore your pc the next advice might suit you better:

If you can boot Windows you may have a reasonable chance of rectifying matters. You need to use the help menu to ask about format a partition. Open Computer Management; Disk Management; right click on the affected partition and create a new simple volume. You may need to run extend partition first and then create the new simple volume. You will loose data in this solution so another option would be to run a windows emergency disk if you have one and use the tools included to find and save files; affect disk repair; run system restore etc..
