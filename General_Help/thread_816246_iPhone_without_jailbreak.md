### Post by Zorael on 2008-06-02
At present, no. There is no way to connect it via USB for anything more than recharging batteries. It *must* have the SSH server installed and it *must* be done wirelessly.

[https://help.ubuntu.com/community/PortableDevices/iPhone](https://help.ubuntu.com/community/PortableDevices/iPhone)
