### Post by yabbadabbadont on 2007-01-30
Try using a different client and see if it works.  If it does, then file a bug against BTuga.  If it doesn't, then it probably means that you have a firewall that is stopping your client from opening the port it needs to allow incoming connections.  Without allowing incoming connections, your client will eventually get ignored by the others in the cloud or swarm or whatever it is called.
