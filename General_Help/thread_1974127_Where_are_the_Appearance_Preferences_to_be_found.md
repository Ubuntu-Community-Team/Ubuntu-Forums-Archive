You need to use utilities such as Ubuntu Tweak, My Unity and Advanced Preferences which are for everyone, or modify theme configuration files if you really know what you are doing. You will have to check the utilities out to see what can be done. My Unity and Advanced Preferences are in the Ubuntu Software Center. Ubuntu Tweak has a PPA.
