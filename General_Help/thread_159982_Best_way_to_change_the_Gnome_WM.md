### Post by drummer on 2006-04-13
Well, you can put in a launcher for the sawfish config utility (whatever it's called, I've never used it) in the preferences menu with Alacarte menu editor, if it's not installed already, get it from synaptic. And I think a better way of running a different WM is by using the --replace argument. ie ```
sawfish --replace
```
