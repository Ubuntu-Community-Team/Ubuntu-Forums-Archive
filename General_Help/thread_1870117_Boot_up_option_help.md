Welcome to Ubuntu and the Ubuntu forums. :-)

Please boot the LiveCD and then download/run the boot info script. Post the contents of RESULTS.txt and we can check the status of your boot files. You can get to the script's download page by clicking the "BIS" link in my signature line.

If you happen to have installed Ubuntu on a second partition, you might check the BIOS boot priority and see if it is trying to boot the Windows drive first. If it is, change it to boot the Ubuntu drive first. Other than that, we'll wait to see the RESULTS.txt
