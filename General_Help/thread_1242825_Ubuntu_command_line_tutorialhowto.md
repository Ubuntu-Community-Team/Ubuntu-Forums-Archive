### Post by marshag63 on 2009-08-17
There is also a live CD that is CLI only distro with tutorials - its a good way to get your feet wet without messing up anything.  Its setup with some really cool stuff - package management, file manipulation, etc.

I was blown away that I could watch a little movie in framebuffer and read my gmail and its attachments (pictures) via mutt.

The most useful commands:

apt-cache search appname

apt-cache show appname
 (where appname is the application name or kind of application)

[http://inx.maincontent.net/](http://inx.maincontent.net/)

You can experiment all you want and not hurt anything - and you can even install it to a USB flash drive, even with persistence :)

Makes a fun addition to the links listed above.

Let us know how you like it.

### Post by marshag63 on 2009-08-19
user123,

Give the INX live CD a test drive - it has a text as well as graphical (Framebuffer) web browswer.  Screen is already setup several ways depending on what your focus will be (web, office, file management). It has built-in wifi manager.  Irssi is available for chatting.  Mutt is configured to set itself up to access gmail.  You can listen to internet radio, your own mp3s or a CD.  There are four applications available for file management.  In addition to screen, you can use dvtm to manage your screens.  There are three network monitoring applications.  You can even watch a movie called "new computer".  

There is a little bit of everything, plus tutorials.  Try out all the applications without messing your own computer up and see what ones you like and don't like.  

INX live CD is like instant "emersion" into the command line, without "breaking" anything.

You will be amazed at what you can learn to do at the command line  :)

MarshaG.

[http://inx.maincontent.net/](http://inx.maincontent.net/)
