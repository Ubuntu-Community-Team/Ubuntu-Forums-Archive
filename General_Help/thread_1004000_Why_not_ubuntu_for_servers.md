### Post by Sorivenul on 2008-12-06
Stability is a big reason.  

CentOS, being based on Red Hat, uses older, more tested, more stable software.  Between the two, Ubuntu is more "bleeding edge", even with the LTS releases.  If a person or company want performance and stability, CentOS will probably get the nod between the two.  

My personal servers run on FreeBSD, but that's for another thread.
