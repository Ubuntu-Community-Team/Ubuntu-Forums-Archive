### Post by dannytatom on 2008-12-11
> **Th3Professor said:**
> How does one use the command line interface (non-graphic user interface, text only) to chat on aim, msn, yahoo, etc.?

Are there any tutorials or how-to type posts floating around on this?

```

sudo apt-get install finch

```

then:

```

finch

```

It uses your Pidgin accounts, so if you're already using Pidgin, you won't have to set anything up.
