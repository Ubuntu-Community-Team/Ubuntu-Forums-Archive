### Post by Zorael on 2008-05-01
I'm not confident this'll work in Xfce, but give it a go.

Install the **fusion-icon** package, you'll get an app that puts an icon in your systray, with which you can select Compiz as your window manager. You may want to make sure you have the **emerald** package installed as well (to get titlebars and borders), since I'm not sure Compiz packs a window decorator that draws Xfce themes.

Then just set Xfce up to launch fusion-icon upon startup.
