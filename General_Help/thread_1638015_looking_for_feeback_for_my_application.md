Hi
 This is my first day on this forum and I am not sure where I should be posting this but i hope i am posting at the right place.
I wrote an application called Daskalos which is an interactive help  application which is aimed at ubuntu beginners who find it hard to do  everyday stuff in linux and don't know where what is.
We believed that rather than telling them how to do things we actually should show the beginners how to 
reach a place and do stuff.

So I used something called dogtail<  [https://fedorahosted.org/dogtail/](https://fedorahosted.org/dogtail/)   > with which i could make tutorials and then for example suppose a  user wants to change his password ..... then he just types password and  then out of the given tutorials available selects one and dogtail takes  him to the place where he can change his password. It clicks on  System->administration->users and groups.

This is the url for my project on github [https://github.com/dev344/Daskalos](https://github.com/dev344/Daskalos)
And this is a demonstration video for my application  [http://www.youtube.com/watch?v=XozDfNYrPy0](http://www.youtube.com/watch?v=XozDfNYrPy0)
