What is this "best" you speak of?

Xubuntu is just Ubuntu with a different desktop environment, xfce.  It uses less resources, and looks a bit different, but it is not better or worse than any of the others.  It is all about preference and choice.

If you are running Ubuntu on a desktop now, try installing kubuntu-desktop and xubuntu-desktop, and you can change which one you are using at the login screen.  Then you can make your own decision...   (But if you decide any of them are better than Gnome, you're just plain wrong...   ;)   )
