### Post by drs305 on 2011-10-14
Nocturnis,

Welcome to the Ubuntu Forums.  :-)

Thanks for posting the RESULTS.txt - it shows what the problem is.

You have Grub legacy installed to the MBR but 11.10 uses Grub 2. Perhaps you used a very old Ubuntu CD to reinstall Grub.

Boot a 11.10 Ubuntu Installation CD to the Desktop (Try without installing) and then run the following commands. The first will mount your Ubuntu partition. The second will install Grub 2 files and write to the MBR. You must use a 11.04 or 11.10 CD, as your installation uses Grub 2 version 1.99.

```
sudo mount /dev/sda1 /mnt
sudo grub-install --root-directory=/mnt /dev/sda
```
Do not add the partition number in the second command.

Reboot and Grub 2 should work. It has not yet found your Windows OS, so after booting if it isn't in the menu run this command:
```
sudo update-grub
```
