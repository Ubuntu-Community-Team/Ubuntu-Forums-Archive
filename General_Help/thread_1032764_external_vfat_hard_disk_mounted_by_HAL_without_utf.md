### Post by dcstar on 2009-01-06
> **mikegr said:**
> Oh, a restart did the trick.

It is always good policy to reboot after any upgrade as major changes have been made to a system, and there is usually (if not always) a message telling you to do so.

Version upgrades by their nature install new kernels and other binaries that are different from what is already in system memory, so a reboot is necessary to have all the new components used.
