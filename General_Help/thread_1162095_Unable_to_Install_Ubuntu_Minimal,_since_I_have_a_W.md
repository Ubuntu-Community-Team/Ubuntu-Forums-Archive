### Post by snowpine on 2009-05-17
Use the "Alternate Install CD" rather than the "Minimal" CD. Press F4 and choose "Command Line Install". This will install the same thing as the Minimal CD. Then, you can install packages from the Alternate CD without an internet connection. This should be enough to get your wireless working. Good luck!
