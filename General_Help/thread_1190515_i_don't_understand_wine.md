### Post by stevo1977 on 2009-06-18
Most of the time you can simply double-click on the .exe file, and WINE will automatically load up and run the program for you.  Alternatively you can type "wine /wherever/your/windows/stuff/is" in the terminal.  It doesn't work for absolutely everything, but it handles the basics okay.

Hi There!!!
If you have really installed wine on your system correctly then you will find an entry in Application group under start menu. Now, to install windows executable programs on linux using wine, just launch the shell and type "wine filename.exe", this will install the program on your system making an entry for it in wine under Application group of start menu from where you can launch the progarm. Please note that not all windows applications are compatible with wine.
