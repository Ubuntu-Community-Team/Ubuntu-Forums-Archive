### Post by SoFl W on 2010-07-03
From the [youtube website]("http://www.youtube.com/html5").

> **Supported Browsers**

             We support browsers that support both the video tag in HTML5 and  either the h.264 video codec or the [WebM]("http://webmproject.org/")  format (with VP8 codec). These include:
             
[LIST]
[*]Firefox (WebM enabled version  available [here]("http://nightly.mozilla.org/webm/"))
[*]Google Chrome (h.264 supported now,  WebM enabled version available via [Early  Release Channel]("http://www.chromium.org/getting-involved/dev-channel"))
[*]Opera (WebM enabled version available [here]("http://labs.opera.com/news/2010/05/19/"))
[*]Microsoft Internet Explorer 9 (h.264, [Platform Preview 3]("http://ie.microsoft.com/testdrive/"))
[*]Apple Safari (h.264, version 4+)
[*]Microsoft Internet Explorer with  Google Chrome Frame installed ([Get Google Chrome  Frame]("http://code.google.com/chrome/chromeframe/"))
[/LIST]
