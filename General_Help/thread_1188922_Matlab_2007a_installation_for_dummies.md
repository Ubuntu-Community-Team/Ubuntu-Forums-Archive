Ok, I didn't pick up on the subtlety of this initially; and it looks like I am not the only one. 

If you have a Windows copy of Matlab that you wish to install on Ubuntu, then you have to use WINE. Apparently it is compatible: [http://appdb.winehq.org/objectManager.php?sClass=application&iId=49](http://appdb.winehq.org/objectManager.php?sClass=application&iId=49)

The alternative is to acquire the Unix version of Matlab and install that instead. Then just follow these steps:
[https://help.ubuntu.com/community/MATLAB](https://help.ubuntu.com/community/MATLAB)
