### Post by veggen on 2011-01-10
You get everything you mentioned (except for an antivirus which you don't need but can easily get one if so desired) out of the box, but as wilee-nilee said, you reeeeaaaally should start with dual boot or VirtualBox.
And no, Ubuntu is not similar to Xp, or any other Windows for that matter (well, you can make it *look* like Windows, but it certainly won't *behave* like Windows). And no, there are no substitutes for everything between any two operating systems.
