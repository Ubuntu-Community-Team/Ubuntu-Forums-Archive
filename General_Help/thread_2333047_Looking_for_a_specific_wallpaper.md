### Post by CatKiller on 2016-08-06
All of the released wallpapers can be downloaded from packages.ubuntu.com by finding the ubuntu-wallpapers package for the release you're interested in. Once you've downloaded the package, you can either install it or just extract the files you're interested in.

Obviously this works for other packages too.
