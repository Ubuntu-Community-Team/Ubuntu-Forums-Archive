### Post by el3ktro on 2006-03-05
Firefox uses a lot of memory sometimes, I'd guess that you have some memory instability problems. When you boot Ubuntu, you can press "esc" at the very beginning when it says that Grub is loading, and you can select to boot Ubuntu or to run a memory test. Run this memory test, perhaps it find's any errors with your RAM ..
