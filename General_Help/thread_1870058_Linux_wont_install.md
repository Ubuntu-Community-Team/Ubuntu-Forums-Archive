Instead of using a disc, I have found that on my computer it will actually install from a USB Thumb Drive.  Download a fresh ISO of Ubuntu, and look on the Ubuntu web site for how to create a Live Stick of Ubuntu on USB Flash Drive to install it. Flash Drives these days only cost around 10 (us) dollars from most office supply stores. It is faster and I can use it to migrate my data easier in case of a crash.
