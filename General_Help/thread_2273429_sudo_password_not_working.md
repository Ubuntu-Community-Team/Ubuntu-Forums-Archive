### Post by Impavidus on 2015-04-13
When you type your password in the terminal, there is no visual feedback. This is normal. Just type the password and hit enter.

If you're positive you type the correct password, we have to investigate further. It could be the result from a keyboard layout problem, for example. Whether and how you can move the close/minimise/maximise buttons depends on your window manager.

### Post by Vladlenin5000 on 2015-04-13
Yes, when you type the password nothing is shown... Just type it as usual and enter at the end.
If it complains about a wrong password then a) it's probably wrong or b) you're using a wrong keyboard layout (affects password with special characters because they are in different locations depending on a specif layout).

Yes, you can also change the position but why? The move to the left makes more sense for Unity desktop. Developers changed that because it made sense, not on a whim. Get used to it instead of persisting in old ways from other OSs...
