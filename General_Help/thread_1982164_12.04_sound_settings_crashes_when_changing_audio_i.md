### Post by chronos00 on 2012-05-29
I found a bug report about this issue:
[https://bugs.launchpad.net/ubuntu/+source/pulseaudio/+bug/987076]("https://bugs.launchpad.net/ubuntu/+source/pulseaudio/+bug/987076")

Please click in the "Does this bug affect you?", so that devs can see many people are affected by this.

There is a suggested work-around: using "Pulse Audio Control" (pavucontrol) instead of system's default sound settings window.
See link bug for more details.
