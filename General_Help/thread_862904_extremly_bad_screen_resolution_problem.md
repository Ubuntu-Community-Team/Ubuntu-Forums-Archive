### Post by smo0th on 2008-07-18
hello, you can reset your screen resolution settings by typing:

```
sudo dpkg-reconfigure -phigh xserver-xorg
```

after that, save all your files and restart the X server with [COLOR="DarkOrange"]control+alt+backspace[/COLOR]

also, you should take a look into your [COLOR="DarkOrange"]/etc/X11/xorg.conf[/COLOR] file, it stores your screen resolution and video settings.

### Post by thedevnull on 2008-07-18
It was probably the Nvidia drivers from the REPO's that caused your problem.  Since I disabled the "proprietary drivers" and uninstalled it I actually installed the one direct from Nvidia and it works great.  

If you need 3d Acceleration for games and stuff you might want this "proprietary" driver.  So lookup your video card and check this out...  

See the following posts relating to the problem:

[https://bugs.launchpad.net/ubuntu/+source/linux-restricted-modules-2.6.24/+bug/222407](https://bugs.launchpad.net/ubuntu/+source/linux-restricted-modules-2.6.24/+bug/222407)
[http://www.nvnews.net/vbulletin/showthread.php?t=72490](http://www.nvnews.net/vbulletin/showthread.php?t=72490)
[http://ubuntuforums.org/showthread.php?t=819043&highlight=Nvidia+8800+GT+804+low+graphics](http://ubuntuforums.org/showthread.php?t=819043&highlight=Nvidia+8800+GT+804+low+graphics)
[http://www.nvidia.com/object/linux_display_amd64_173.14.09.html](http://www.nvidia.com/object/linux_display_amd64_173.14.09.html)

in terminal window
Code:
sudo apt-get install build-essential
Then :
Code:
sudo apt-get remove nvidia*

[this should get rid of all unwanted NVidia drivers. please check /lib/linux-restricted-modules/2.6.24.xx-generic/video and make sure there's no nvidia directory in there. There shouldn't be.] [xx = kernel version]


Step 2
on your keyboard

ctrl+alt+backspace

Wait for the drop out of X11.

Step 3

Login into terminal as yourself. Go to the directory where you downloaded the newest NVidia driver. 

type :
Code:
chmod a+x NVIDIA-Linux-x86-173.14.05-pkg1.run
NVIDIA-Linux-x86-173.14.05-pkg1.run = name of the file you downloaded. If it's different, then make corrections.


Step 4

type in :
Code:
sudo sh ./NVIDIA-Linux-x86-173.14.05-pkg1.run

Follow directions on the screen.

Step 5

Reboot, and you should be good as new. I don't use Envyng, or Compiz. I like a nice clean install of my devices.
