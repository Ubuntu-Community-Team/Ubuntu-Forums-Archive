### Post by tylerspaska on 2008-01-02
wubi installs just like any other program (and can be uninstalled). during install you can choose linux partition size (4-30GB), desktop environment, user name, and password. when you restart and choose wubi from the boot screen it starts a normal ubuntu install. 

i have found only a few limitations that effect me. the main one being that you cannot suspend or hibernate the system. the next being that as of right now, you cannot upgrade to 7.10 within 7.04 wubi.
