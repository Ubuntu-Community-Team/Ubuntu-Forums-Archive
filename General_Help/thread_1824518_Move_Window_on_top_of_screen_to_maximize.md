### Post by tokken on 2011-09-14
I use Ubuntu 11.04, and it does maximize windows when you move them to the top of the screen -- and it's driving me nuts! Is there any way to turn this off?

UPDATE: The solution is to install the CompizConfig Settings Manager (compizconfig-settings-manager package), then run it and deselect Window Management -> Grid.
@Juice0123: the answer to your question may be to upgrade to 11.04, or maybe the Grid plugin is available for 10.04 as well and you just need to install and/or enable it.
