Lucid (10.04) is a Long Term Support (LTS) release and, as such, the desktop version is supported until April 2013 and the server until April 2015.  This is a longer support period than that for the "normal" releases, such as Maverick (10.10).

There is no requirement to upgrade now unless you *need* to.

I did an upgrade to 10.10 on my Eee. Like you, wireless completely disappeared! Could not figure out for the life of me. Then it hit me - BIOS. Sure enough, I rebooted, jumped into my BIOS and wifi was turned off. Wild, but I changed this, rebooted again and wifi was back in black. 


To reaffirm, my ath9 wireless was literally missing in network manager, checking BIOS fixed this afterward.

Now if the issues in indeed, network manager...ethernet up to something and install Wicd. I have had to do this with machines I needed to have more control over when using multiple wifi devices with. Should help avoid any experienced bugs with network manager.

**Wifi is working but slow?** Final advice, if you upgraded AND opted to not to remove any of the earlier kernels during the upgrade (like I often do). Reboot, holding SHift and select an earlier kernel to see if there is something with the new kernel that is giving NM a problem. Just for giggles. I had to do this as ath9 runs like snot in the given kernel that came with 10.10. Faster may eye, it shows none of the unknown speed like before...but the latency when loading a new website is awful. Using [earlier kernels]("http://www.mail-archive.com/fedora-list@redhat.com/msg59513.html") solved this instantly.
