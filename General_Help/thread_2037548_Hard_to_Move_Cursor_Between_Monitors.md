### Post by QIII on 2012-08-04
Go to the "Displays" settings.

There should be a "Sticky edges" setting.  Turn it off.

If you have the launcher also on the right screen, you may want to set it so it only shows on the left screen, particularly if you have the launcher set to hide, because of the "force" needed to unhide the launcher.
