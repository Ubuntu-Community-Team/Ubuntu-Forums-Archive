### Post by C-- on 2009-12-08
> **pqwoerituytrueiwoq said:**
> 1. is there a way to save my theme to a file for sharing  (system -> preferences -> appearance)
2. is there a way i can insert a tab character in an OS title in the grub mot menu ( i tried \t)


1. Answer to Question 1

a. Under system -> preferences -> appearance, in the theme tab, click "save as" and save your current theme (give it some name).

b. Go to ~/.themes. The theme you saved will in this directory. The "." denotes that it's a hidden directory and so it will not be visible in nautilus, use ctrl-h to reveal it, or use the command line.

c. Put the theme you created (it will be a folder) into the ~/.themes directory on another account. In the appearances customization window, that theme will show up.
