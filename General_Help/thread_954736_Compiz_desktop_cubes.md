### Post by ChildOfMana on 2008-10-21
You have to set the number of desktops to 4 or more for the cube to work.

System > Preferences > Advanced Desktop Effects Settings

Click 'General Options' and navigate to the 'Desktop Size' tab. Now use the slider for 'Horizontal Vertical Size' to set it to 4 (or more).

Should do the trick.

### Post by ChildOfMana on 2008-10-21
Just realised I may have got the wrong end of the proverbial stick there.

If you already have your Horizontal Vertical Size set to 4 and still can't see the rotating cube then you need to go to the 'Desktop' tab on the left hand side of 'Advanced Desktop Effects Settings' and make sure the 'Rotate Cube' effect is enabled.

Also, it sounds as though you expect the windows to be on a cube...

What the desktop cube does is allow you to see a cube when you switch between differing desktops.

By default, Ctrl+Alt+ Mouse drag starts the free rotation. You can also see it if you click on a different desktop (right side of the bottom panel by default), although it goes pretty fast

### Post by ChildOfMana on 2008-10-21
I'm not sure if there's a 'quick and easy' way to restore it to default (there probably is if you ask around) but you can right-click on your panel items (e.g. clock, session manager etc), de-select 'Lock To Panel' (if there's no tick next to it then it's already de-selected ) and then click 'Move' and manually shift things around to your liking. Select 'Lock To Panel' again when it's in the desired position.

To add something to the panel you can either right-click in an empty space and select 'Add To Panel...' and choose what you want from the list (or create a custom launcher), or, if what you want isn't in the list, you can find it in your 'Applications' or 'System' menu, right-click on it and select 'Add this launcher to panel'.

Hope this helps.
