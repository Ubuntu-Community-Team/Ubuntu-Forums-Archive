### Post by walts48 on 2017-03-11
Thunderbird still gets an update approximately every six weeks, just like Firefox and Firefox ESR.

Thunderbird 45.8.0 only has security updates, 6 Critical among the ten security fixes. More of an incentive to get users updated ASAP.

[Security vulnerabilities fixed in Thunderbird 45.8 — Mozilla]("https://www.mozilla.org/en-US/security/advisories/mfsa2017-07/")

Mozilla still provides infrastructure, hosts the Official support forum, the mailing lists and newsgroups, IRC chat and more for Thunderbird, Lightning and SeaMonkey. Changes are in the works.
