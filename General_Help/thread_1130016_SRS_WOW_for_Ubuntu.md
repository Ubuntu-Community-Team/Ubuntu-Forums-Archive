### Post by dotnetnightmare on 2010-05-24
Mplayer has "extrastereo" option in audio that is VERY close to SRS. It enhances clarity and adds the spacial or widening effect. I use SRS Audio Sandbox on W7 & SMPlayer with Extrastereo on PCLOS. Normally you must select extrastereo from the drop down menus and it  resets every time you open a new file so you have to select it again and  again! Smplayer has an option in the mplayer section to add additional audio filter commands, type in "extrastereo" and it will be on always. PCLOS also uses SMPlayer with CoreAVC available in the Performance menu!

### Post by lilmagnus on 2012-04-26
You may have pretty good luck with this...[WebUpd8]("https://launchpad.net/~nilarimogard/+archive/webupd8") contains a **system-wide** equalizer for Pulseaudio **though they currently do not have it yet compiled for 12.04**. Add it to your sources and keep checking for it. It will be labeled as...**> pulseaudio-equalizer**
```
deb http://ppa.launchpad.net/nilarimogard/webupd8/ubuntu precise main 
deb-src http://ppa.launchpad.net/nilarimogard/webupd8/ubuntu precise main
```
