### Post by anaconda on 2007-03-28
You dont have the printing rights for that pdf-file.

Maybe you can save it as text and print that? or open it with kword. it imports pdf-files and then you have all rights, because it only reads the original wont modify it. (KWord's pdf import isn't perfect, but worth trying..)

When creating a pdf-file it is possible to password protect it. At the same time it is possible to limit what users can do with it. Printing can be disabled..as it is in your case.
