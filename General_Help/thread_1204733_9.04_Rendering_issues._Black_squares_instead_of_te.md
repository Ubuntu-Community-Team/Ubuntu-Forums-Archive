### Post by Peter K Nicol on 2009-07-08
The solution appears to be here

[http://ubuntuforums.org/showthread.php?t=1168874&highlight=icons+vanish](http://ubuntuforums.org/showthread.php?t=1168874&highlight=icons+vanish)

Open a terminal and type 
sudo gedit /etc/X11/xorg.conf  (not as shown)
and then find Section "Device"
edit the entry so that it looks the same as shown except that there are tabs rather than spaces.

seems to work fine
