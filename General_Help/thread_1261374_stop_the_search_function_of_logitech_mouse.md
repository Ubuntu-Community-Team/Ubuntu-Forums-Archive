### Post by LifeEnemy on 2009-09-08
I have a logitech VX Revolution, which has a search button behind the mouse wheel (I think the MX version also has this, and probably works the same). Whenever I press that button it opens a search window, even without any kind of button event manager installed (like btnx). Is there anyway I can stop the search window from popping up, but use the button for something else? I can't use the button for any programs because the search function screws it up.

---

### Post by P4man on 2009-09-08
Simple and logical *cough*: mouse button bindings can be configured in the "*keyboard* shortcuts applet:

System > preferences > keyboard shortcuts
Desktop > search

Remove the XF86Search entry there, and/or feel free to assign any other function to that button.
