### Post by Impavidus on 2014-09-13
There must be a setting somewhere (sorry, I forgot where it is exactly on Ubuntu 12.04, but it must be somewhere in the settings for updates and upgrades) where you can specify whether you want to be informed of all release upgrades, only LTS release upgrades or no release upgrades at all. If you now run 12.04, the option for all release upgrades would lead to 12.10, which is unsupported. Check that setting and make sure it is set to LTS releases only. That should offer you an upgrade to 14.04.
