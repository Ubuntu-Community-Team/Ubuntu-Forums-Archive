### Post by damis648 on 2008-05-31
Install startup-manager. To find startup and login screens, go here: [http://gnome-look.org/content/search.php](http://gnome-look.org/content/search.php) and search "upslash" for boot themes and "login theme" for login themes. To install a boot theme, run startup-manager and go to "Appearance" and down to uplash theme. For the login, go to System>Administration>Login window and install the theme.
