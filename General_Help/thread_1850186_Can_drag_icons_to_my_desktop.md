### Post by ankspo71 on 2011-09-27
Hi,
If you like using the "desktop view" layout, you can also add application shortcuts to the desktop by unlocking your widgets first (right click on the desktop to unlock the widgets), then go to the start menu and right click on any application and choose "add to desktop". Doing this will create a desktop widget containing the shortcut. This can be done with the folder view layout too, which is kind of nice because these widget shortcuts can be resized, and even locked in place after you lock your widgets again.
