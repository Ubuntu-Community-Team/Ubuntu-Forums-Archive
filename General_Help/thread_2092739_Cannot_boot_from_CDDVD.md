### Post by Mark Phelps on 2012-12-08
First of all, you can NOT install Ubuntu to a CD/DVD.  You can install FROM a CD or DVD.  You can burn an ISO image TO a CD or DVD.  But, you can't INSTALL to the CD or DVD.

Second, presuming this is an install CD or DVD, when you boot from it, there should be a line on the screen prompting to press a key to boot from it.  IF you're not seeing that line, the CD/DVD is not bootable.

Need to know HOW you burned the bootable Ubuntu CD or DVD.
