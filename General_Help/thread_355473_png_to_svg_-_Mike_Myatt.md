### Post by 23meg on 2007-02-07
SVG is a vectoral format whereas PNG is raster, so you'll need to trace it in Inkscape to a vectoral object. This will essentially alter the content of the image. 

To do it, import the PNG image by dragging and dropping it on the Inkscape document, select it and click Path / Trace Bitmap. A window will come up with many different tracing options; try them all to see which fits your purpose better.

### Post by 23meg on 2007-02-07
Actually, depending entirely on what you want to do with the image, you can embed a raster image into the SVG file as well. To do this, you'd just put the image in your Inkscape document, and save the document as an SVG file.
