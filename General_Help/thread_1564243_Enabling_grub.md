If you already have Ubuntu installed and you're going to install Windows, then I recommend you follow this guide:

[https://help.ubuntu.com/community/WindowsDualBoot#Installing Windows After Ubuntu](https://help.ubuntu.com/community/WindowsDualBoot#Installing Windows After Ubuntu)

As long as you follow the instructions there, GRUB will show up when you boot up. If you're just looking for general information about how to change GRUB settings, then try one of these two links:

[http://www.dedoimedo.com/computers/grub.html](http://www.dedoimedo.com/computers/grub.html) -- GRUB 1, the old default for Ubuntu 9.10 or earlier
[http://ubuntuforums.org/showthread.php?t=1195275](http://ubuntuforums.org/showthread.php?t=1195275) -- GRUB 2, the default for Ubuntu 10.04 or later
