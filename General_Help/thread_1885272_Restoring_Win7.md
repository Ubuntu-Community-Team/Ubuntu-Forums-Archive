### Post by Mark Phelps on 2011-11-23
Lets make sure that we are discussing the same thing ...

Restore generally means returning the PC to its original state, which wipes all files, data, and settings, and puts the PC back to where it was when you first used it.

Repair Reinstall means that you reinstall Windows over itself -- using the installation DVD.  This is useful when system files get corrupt and you want to return it to a working state.  It leaves the applications and data intact.

Reinstall involves erasing the contents of the partitions and installing again, from scratch.  This is essentially the same as the Restore.

Which of these do you want to do? OR, is there a fourth option that I've missed -- which is really what you want?
