 'Could not change local directory to /home/media/DATA: Permission denied'


Andy,

Download PCMan  file manager.   :)

Start by executing  PCMan  , then "tools" , then "terminal" and dropping into the current subdir as root 
As 'root' you should be able to change dir into the system folder of choice.  

Sometimes, the system folders are locked because they are in use.  No cure for this. 
You Do Not want to change owner on a system folder since this affects the security of the OS.  :(

PCMan will  allow you to change to the system folder, and then "tool" "terminal" drop into the folder. 
You Do Not want to muck with the system files with your eyes closed !  :(
