### Post by louieb on 2009-08-28
> **pagaldilz said:**
> I can't load my windows xp using grub menu.   When i try to load it all i get is a black screen and blinking pointer in the upper left corner of the screen.  I have windows 7 installed and it loads without any issue.

Welcome to the funky world of trying to get GRUB to choose between two different installs of windows.  Most of the time you can't.  Thats because when you installed the 2nd version of windows it will only keep 1 boot loader for GRUB to chainload too. 

When you boot windows you should be getting the windows boot menu where you select which version of win to boot. Do you?

If you really want GRUB to boot both then you will have to make sure each win install has its own boot loader.

If you want to see for yourself download the [Boot Info Script - SourceForge.net]("http://sourceforge.net/projects/bootinfoscript/")  it will list your boot loader files and where there at (along with a lot of other useful stuff. 

to run open a terminal window - cd to your download directory.
```
sudo bash boot_info_script032.sh  
``` 

see the info in a file called results.txt.

[Boot Info Script Instructions]("http://ubuntuforums.org/showpost.php?p=6725571&postcount=3")
