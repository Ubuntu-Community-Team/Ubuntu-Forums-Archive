### Post by ryanhaigh on 2008-04-13
Installing and managing your software in Ubuntu is quite different than in windows.
This page outlines how software can be installed and the basics of package management: [https://help.ubuntu.com/community/InstallingSoftware](https://help.ubuntu.com/community/InstallingSoftware)
While this page list SOME of the packages available in ubuntu: [https://help.ubuntu.com/community/Software](https://help.ubuntu.com/community/Software)

For a more exhaustive list look at add/remove programs in your applications menu or System>Administration>Synaptic Package Manger.

### Post by sekinto on 2008-04-13
WinZip > Ubuntu can already handle zip, tar, gz, and a lot more of the bat. You can install aditional support for things like rar.

Windows Media Player > VLC plays almost every format you would probably come across, Totem can to with some codecs installed.

CD/DVD burner & Nero > There are a few different programs that can do this. Brasero and K3b just to name a couple.
