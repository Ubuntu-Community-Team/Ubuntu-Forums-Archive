### Post by jamesisin on 2011-07-25
You don't actually download a repository.  It exists on a server somewhere for you to make use of should you require software or updates from it.  You merely load some location and authentication information about it.

I would always recommend getting to know any repo before you add it.  Research.  Mostly you'll probably be just fine without adding anything extra.  You'll see plenty of good advice on threads around here, and you can always ask about a specific repo/PPA.
