### Post by stuporglue on 2005-10-27
Any program that can be compiled for Fedora can be compiled for Ubuntu. Most, if not all, of the packages FC4 has Ubuntu will also have. Ubuntu is based on, and is mostly compatible with, Debian. 

I currently have 18313 packages available through apt-get. I don't have any Debian repositories in my list, which would add even more options (at the risk of possible incompatabilities). 

I don't know any exact numbers for performance, but Ubuntu does very well for me. I have a 2GHz AMD and don't notice any issues.
