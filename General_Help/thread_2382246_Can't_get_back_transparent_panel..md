### Post by Frogs Hair on 2018-01-13
What you are seeing is a feature of the Ubuntu session . Unity is no longer default on 17.10 and there is a gnome shell extension that can address this in the Ubuntu session though I don't  think it will have any effect in the Unity session. If you upgraded and  haven't looked, you should have three sessions to select at login to including Unity, Ubuntu, and Ubuntu on Xorg.
