### Post by wdaniels on 2008-06-06
You should be able to right click on a blank space in the other panel (at the bottom) and select "Add New Panel", but you'd have to set it all up again...the menu bars and stuff are all just panel applets that you can find with "Add To Panel".

Unless somebody else knows a shortcut to get it back as default?

I guess you should be able to just reset all your Gnome settings back to defaults (delete/rename the hidden folder .gconf in your home directory) but that would set *everything* in the desktop back to default.
