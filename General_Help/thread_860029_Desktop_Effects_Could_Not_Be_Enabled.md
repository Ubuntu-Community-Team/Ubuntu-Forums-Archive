### Post by Kevbert on 2008-07-15
I assume you're talking about the desktop cube and compiz-fusion.  Unfortunately compiz is not currently compatible with Via display cards.
Things you could use are screenlets (small applets that show various things like weather) and desktop drapes (which switch backgrounds).  Both can be found with Synaptic Package Manager.  The other item is conky which shows PC resources (there's loads of posts on this forum).
