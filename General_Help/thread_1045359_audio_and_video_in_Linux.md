### Post by PriceChild on 2009-01-20
Ubuntu should be able to play any media that you throw at it, just like Windows. Just like Windows however, not all formats are supported "out of the box". If you come across something that it thinks it can't play, then a really easy codec instillation wizard will pop up and help  you install the necessary software.
