### Post by spur on 2011-05-14
I have a similar set up with A wubi install on my Acer One and a win7 64bit on my main computer (home built).
I got the same message when trying to use wubi on my main computer. I used instead the live cd install(dual boot). This worked fine and I had no problems. As wubi installs Ubuntu into windows you may be able to remove it through your windows control panel as a 'normal' program would. I haven't tried that, but it could be worth a try.
One suggestion with a cd install is too set up your free space(to install Ubuntu on) before you start the install.
