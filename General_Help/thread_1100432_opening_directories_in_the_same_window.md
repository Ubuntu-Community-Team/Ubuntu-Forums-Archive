### Post by mcduck on 2009-03-19
What desktop and file manager are you using?

If it's Gnome (and Nautilus), go to Edit/Preferences, and on the Behavior-tab mark "Always open in browser windows".

Only browser mode allows opening different directories in the same window, spatial mode won't do that since it would be against the main idea of spatial file management.

Also, an nice trick in the spatial mode is to use middle click instead of left click. This will open the item you clicked and close the current window. Prevents your screen from filling with windows you don't need to keep open.
