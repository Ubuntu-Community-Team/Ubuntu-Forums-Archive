### Post by santaslittlehelper on 2008-01-14
Hi

Try the following,

System > Preferences > Keyboard > Layout Options > Compose key position (I use "Left Win-Key is Compose, depends on you're keyboard).

You might have to restart X before it takes effect.

Then you should be able to use the compose key to make special characters. 

Hit "Compose Key" then "deadkey" then "u", to do a Umlaut. You really need to hit each key one at a time. :)

Hope it helps.
