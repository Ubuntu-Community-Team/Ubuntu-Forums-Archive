### Post by tiger.woods on 2011-12-31
In case anyone runs into this in the future, first download the headers:

sudo apt-get install linux-headers-`uname -r`

then download the driver from Nvidia 

[ftp://download.nvidia.com/XFree86/Linux-x86/96.43.20/](ftp://download.nvidia.com/XFree86/Linux-x86/96.43.20/)

or check the forum for the latest information on the drivers: 

[http://www.nvnews.net/vbulletin/showthread.php?t=122606](http://www.nvnews.net/vbulletin/showthread.php?t=122606)

Once downloaded simply run the script:

#sh NVIDIA-Linux-x86-96.43.20-pkg1.run
