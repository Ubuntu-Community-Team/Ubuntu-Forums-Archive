### Post by DeMus on 2008-09-27
Hi,
In Windows it is easy to write German characters like ö or ü using the Alt-codes 148 and 129. In Linux I find it hard to do using the program Character Map. Is there any other way to insert these special characters?

Thanks, DeMus

### Post by der_joachim on 2008-09-28
You can either use deadkeys or compose keys. You need to decide for yourself which one you like best. 

Deadkeys are the most common. The dutch version of XP has a similar system. We hates it. If you need an ö, you first need to type a double quote and then an o. You will automagically get an ö. A big disadvantage is that when you just need the double quote, you'll have to type an extra space, which is hell for fast typers. The compose key is somewhat easier to use (IMHO). If you need an ö, just use Compose+"+o. 

As for configuring you can either edit /etc/X11/xorg.conf or you can configure your keyboard applet. 

When you decided which one you like best, just do a forum search on either deadkeys or compose keys for instructions on how to configuring.
