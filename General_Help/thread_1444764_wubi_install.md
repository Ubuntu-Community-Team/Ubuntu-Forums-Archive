You do not install in a partition with WUBI it will put itself in a folder in your C: drive right next to Users. It will use  attach itself to windows bootmanager. Just put cd in tray
and choose Wubi tell it how big you want it, your password and click go. That is it. 
You do not boot off of it, just stick it in tray. You have to boot off of it to do a partition install or use it as a live cd.
