### Post by PocketDog on 2008-10-03
Depends what network.

On Windows with my W880i on O2 I have to install PC Suite, install the phone driver, and because O2 uses a proxy to connect I have to download and install Proxifier, and enter the proxy settings into that, go to network connections and connect. A month later I have to pay for Proxifier or it stops working.

On Linux I plug my phone in, enter the proxy into Preferences > Network Proxy and it connects automatically.

I've done a little searching (not much, I don't get paid for this :P) and this ** [http://bitubique.com/tutorials/setting-up-nokia-phone-as-modem](http://bitubique.com/tutorials/setting-up-nokia-phone-as-modem) ** appears to be the sort of help you're after. It's for Dapper, which is probably waaaaaaaay out of date, but a little refining of a "N95 as modem +ubuntu" search string should help.
