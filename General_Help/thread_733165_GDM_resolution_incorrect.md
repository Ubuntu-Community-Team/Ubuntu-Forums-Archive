### Post by liquid circuit on 2008-03-25
I have solved my issue!  It seems all I had to do was set the PreferredMode option in the Monitor section.  Obvious Option is obvious.  At least I figured out I don't need that ugly modeline in the process!!! =)  Thanks for the help guys!

This fixed it.
```
Section "Monitor"
	...
	Option		"PreferredMode" "1440x900"
	...
EndSection
```
