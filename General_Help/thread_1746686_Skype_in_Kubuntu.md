### Post by bergfly on 2011-05-03
Skype has a Linux client. Go here

[http://www.skype.com/intl/en-us/get-skype/on-your-computer/linux/]("http://www.skype.com/intl/en-us/get-skype/on-your-computer/linux/")

Download the needed version  (32 or 64 bit). Once it is downloaded, click on it and it should install with your package manager. Generally a pretty painless process.
