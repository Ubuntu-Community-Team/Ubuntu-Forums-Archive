Check if these are checked....

Edit Preferences --> Content --> Load Images 

Edit Preferences --> Content --> Load Images for the originating site only.

The first should definatly be checked the second one probably should be checked and if it isnt i might be the cause of your problem. I unchecked the later last week and wondered why some pictures were not displaying!! :)
