### Post by undoIT on 2008-09-19
I've been using Thunderbird for years now. I have around 14 different email addresses. It was so easy to backup my entire Thunderbird account from the Windoze days and move it to my shiny new Ubuntu install. And, every time I've done a clean install it is really easy to backup and restore.

Now I'm checking out Evolution. Right off the bat there are a couple of things I didn't like. All of my email accounts are using IMAP. One of the things I like in Thunderbird is that I can collapse down the account so that only Inbox is showing. That way, I can view all the accounts at once and there is visual indication for each account if it has a new message. In Evolution, a default Junk and Trash folder are created in addition to the IMAP folders for each account. Those folders cannot be collapsed. Is there a way to get rid of those? They are just wasting space.

Also, I never use the global folders. I know I can collapse down the "On This Computer" area. But is there a way to get rid of it completely?

Has anyone migrated from Thunderbird to Evolution? If so, how do you like it? Other than a few minor gripes, the only downside I can see so far is that Evolution is not really cross-platform compatible. The upside is that there is more functionality with Calendars etc.
