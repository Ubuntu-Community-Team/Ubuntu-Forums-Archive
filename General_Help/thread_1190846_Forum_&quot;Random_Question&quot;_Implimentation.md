### Post by diesch on 2009-09-28
This type of challenge-responsetests are in general called *captcha*, specifically *text captcha* if they use this type of questions. 

For some forums software there are ready-made capcha implementations, look at the software's web site or ask your favourite search engine. There are also some free and non-free captcha implementations in most programming languages used for web programming.
