### Post by Howlin' Hobbit on 2006-10-10
In Windows when you right click on a file you get a choice on the resultant popup menu called "send to". You can easily add programs to that menu by simply going to the send to folder and putting a shortcut there. That allows you to easily open a file with one of the programs on that menu.

I use this feature for things like plain text data files that I've given odd extension names to, included files for my web sites (same thing... extension trouble) etc. when I just want to send them to a particular editor. It's also good for easily opening graphics files in something besides their associated viewers.

Is there some way to do this in Ubuntu? There's a "send to" choice on the popup but all it has is the default email program. Can I add to this submenu?

### Post by fragos on 2006-10-10
It seems that with Gnome file associations are by mime-type.  I used to use SuSE KDE and could assign file extensions to mime-types.  You can in Gnome assign multiple applications to a mime-type with one of those the default.  Right click will select from the others assigned.
