### Post by HermanAB on 2020-04-30
There certainly is no known malware in the Ubuntu repos.

In general, Linux, BSD and Apple systems are very secure.  I mention all three, since there is significant overlap.

The reason is because there are many system administrators, developers and users, using these systems for high security (financial, medical and military) applications and therefore the software gets a lot of scrutiny and many penetration tests.  When a security problem is discovered, it is usually fixed within a few hours.
