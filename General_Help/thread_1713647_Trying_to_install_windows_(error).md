### Post by ~Plue on 2011-03-24
you could make use of a bootable thumb drive, if your motherboard supports booting off usb devices (boards manufactured in the last ~5 years usually do)
> 
*install and open *gparted*
*format the usb drive as ntfs
*right click the partition > manage flags > check 'boot'
*open the partition from the file manager (from the 'places' menu)
*copy-paste everything off the dvd / extract the .iso to the newly created partition
*reboot your computer from the usb drive
