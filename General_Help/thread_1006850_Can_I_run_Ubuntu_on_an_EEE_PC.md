### Post by snowpine on 2008-12-10
Hi Tamalin, 

Ubuntu runs well on the eee for the most part, with the biggest exception being wireless. 

A regular contributer at [http://forum.eeeuser.com](http://forum.eeeuser.com) by the name of Adamm has developed an unofficial kernel that supports all of the eee hardware. It's available at [http://www.array.org/ubuntu](http://www.array.org/ubuntu) and can be installed into any "flavor" of Ubuntu Hardy or Intrepid.

You can also try Ubuntu eee 8.04.1 (not to be confused with eeeubuntu, which is a different project). It's Ubuntu + the array.org kernel + a "netbook remix" interface that is optimized for the small screen. Ubuntu eee is not an "official" Ubuntu project, so they are changing the name for the upcoming 8.10 release.

If your friend wants the easiest possible solution, my suggestion is ubuntu eee 8.04.1. 

Hope that clears up some of the confusion.

### Post by silkstone on 2008-12-10
As snowpine says, Ubuntu-eee 8.04.1 makes life easier by including the Array kernel. Most things work out of the box, although there are a couple of minor bugs.

New versions of Ubuntu-eee (although it won't be called that) and eeebuntu, both based on Intrepid, are due out very soon. These should provide even better support for all models of eee PC, and the aim is to extend support to other netbooks.

These projects are not directly associated with Ubuntu, which is good and bad. It's a pity in some ways that there is parallel-tracking on the issue of netbook support, with different projects (including Jaunty) aiming for similar things. However, Ubuntu-eee and eeebuntu are not bound by Canonical's adherence to Open Source, and so can include proprietary drivers and codecs.
