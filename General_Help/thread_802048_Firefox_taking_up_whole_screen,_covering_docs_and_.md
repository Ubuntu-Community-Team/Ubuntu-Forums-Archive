### Post by John_T on 2009-01-09
> **djackson75 said:**
> Pressing F11 fixed it, thank you... I feel stupid, but it's worth it.:)
I've had similar issues. 

It seems to be that sometimes Firefox jumps in to full screen mode without warning.  The thing is, it doesn't seem to be "true" full screen, it's just that firefox expands to cover the top and bottom Ubuntu bars.  The firefox menus and icon bars are still visible.

I'm not yet certain whether it's firefox itself or only certain sites that do it.

FWIW I'm using Firefox 3.05 on Ubuntu 8.04.

Edit: After a bit more digging around, it seems that this really is a bug.  See [here]("http://support.mozilla.com/tiki-view_forum_thread.php?locale=el&comments_parentId=213393&forumId=1") for one example.
