### Post by LaRoza on 2007-08-28
You can get both at once, by installing either one and using this command:

If you want Kubuntu:
```

sudo aptitude install kubuntu-desktop

```

If you want Ubuntu:
```

sudo aptitude install ubuntu-desktop

```

You will get all of the apps, and both DE's. You can switch your DE when you log on, under "Session".
