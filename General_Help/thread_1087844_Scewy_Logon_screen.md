### Post by unutbu on 2009-03-06
This line in your /etc/X11/xorg.conf
```

Virtual  2048 768
```
	
controls the resolution of your login screen. Is your single monitor 1024x768? If so,
you should change the virtual line to 
```
 
Virtual  1024 768
```
