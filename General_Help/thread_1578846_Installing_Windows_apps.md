### Post by Mark Phelps on 2010-09-21
Wine is not a miracle cure -- it only runs SOME MS Windows apps.

If you're going to use Wine, you need to go to the WineHQ site and look up the compatibility ratings for the apps you want to run.

The page below is from that site, for Adobe After Effects.  You'll see that some of the ratings are Garbages -- meaning, that version is NOT going to work, no matter what you do:

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=648]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=648")
