### Post by noriek on 2008-09-27
Hi All,

Using ubuntu/kubuntu PPC on iMac and Xubuntu on an old Ipaq desktop. Both systems being less than up-to-date I'm required to use alternate (x/k/)ubuntu. 

So my question: Are the contents of the "alternate" installation the same as the installation from a "live" edition?

I assumed they would be but I ask for this reason:

Xubuntu site ([http://xubuntu.com/get](http://xubuntu.com/get)) claims 128MB ram to run the live version. Then 192MB ram to install it. Then only 64MB for the Alternate install. 

But then ... 

"Once installed, Xubuntu can run with 192 MB RAM, but it is strongly recommended to have at least 256 MB RAM."

This suggests that all xubuntu versions need 192MB to run so presuming and assuming there's not a version which requires 64mb to install but requires 192 to run then can I assume the contents are different?

Hope this doesn't seem too pedantic! I just want to know what xubuntu goodness I might be missing out on!!

Thanks

---

### Post by Patrick793 on 2008-09-27
Both install the same things. The Live CD gives you a graphical installer, and allows you to test it out before you install, and the Alternate installer just gives you a text based installer.
