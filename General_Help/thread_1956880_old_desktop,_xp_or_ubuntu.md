### Post by critin on 2012-04-11
> 512mb DDR ram

I would advise against Wubi with this amount of ram, or it will run as slow as you say XP is.  Well almost anyway.   If there's room on the disk consider installing on its own partition and both os's should run fine.   Unity won't run probably.  To address the slowness issue, remember to clean windows with something like CCleaner regularly and defrag, keep updates current and XP will just keep on turning--it's a very good system in my experience and worth the time to care for.    

Adding another 512 stick of ram will do wonders for any OS.  One stick is cheap, try ebay.

There are lighter versions of ubuntu,  suggestions have already been given.  You're going to enjoy playing with any ubuntu...

### Post by mastablasta on 2012-04-12
increasing RAM for 1GB should make even WindowsXP fly.
 
Xubuntu is the first to try on the current configuration. You can use various browsers, Firefox is default but you also have Chromium (Chrome-google ad stuff), Opera, Midori, Seamonkey and various others. You own't be able to use Internet explorer. Not even in Wine.
