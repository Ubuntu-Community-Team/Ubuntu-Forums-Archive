### Post by bashiergui on 2014-11-23
Ubuntu routinely checks the disk upon reboot every 30th reboot. Sounds like this was a routine check and it successfully cleaned up a few issues it found.

You could run a disk check from gparted using a live cd to determine if the hard drive is indeed failing. 

If you only got those messages once then I would be less concerned about drive failure. 

Either way, this is a good time to set up routine backups of your important data to avoid any future catastrophe.
