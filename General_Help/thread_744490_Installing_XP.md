### Post by bodhi.zazen on 2008-04-03
You buy a windows install CD from Microsoft. You then put it in your CD-ROM drive, reboot and follow the installation instructions.

You then try to find the drivers you need, not included on the CD, for you hardware and install them.

You then update and install security software.

The entire process can take several hours or days.

Windows iso are illegal and unreliable, how do you know they do not contain malignant code ?
