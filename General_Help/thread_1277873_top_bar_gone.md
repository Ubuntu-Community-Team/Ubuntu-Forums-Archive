### Post by tuxxy on 2009-09-28
Are you using Desktop Effects? if you are then press ALT + F2 and enter this command;
```

emerald --replace
```Now your titlebars should be back and you can edit them or change themes with Emerald Theme Manager, you may need to install this so search for emerald using Synaptic or enter this in a terminal

```
sudo apt-get install emerald
```
