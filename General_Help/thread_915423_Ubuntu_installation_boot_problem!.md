### Post by schauerlich on 2008-09-09
The alternate CD is a text-only installer for Ubuntu. Basically, it's a way to work around computers that can't handle running a live session (low RAM) or don't like the default drivers (which may be your problem). Also, the alternate CD has a few more options for installing that are not present in the LiveCD installer. I use the Alternate CD whenever I install just because I find it faster, but that's subjective.

To download the alternate CD, just check the box under the green Start Download button on the [download page](http://www.ubuntu.com/getubuntu/download).
