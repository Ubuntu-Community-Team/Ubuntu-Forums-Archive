### Post by gdonwallace on 2009-10-14
I checked the knowledge base for Firefox, and seems that if you enable FIPS in firefox (open the browser Click Tools, then Options, then (on the top row) Advanced, then Security Devices, then enable FIPS) this should do it.  However; the draw back will be that for every tab that you open it will also ask for a password.

Don't know if this is what you are wanting, but it may do it.

I don't think firefox can be configured in this way.  I believe the closest thing would be setting the "master password" (Edit/Preferences/Security from the firefox menu).  However, I think it will only prompt when you visit sites for which you've elected to save passwords (such as an email or social networking sight).  

If you need to password protect the firefox executable for some reason, I think you'll have to use a third-party program or script to do it.
