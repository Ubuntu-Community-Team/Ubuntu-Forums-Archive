### Post by hoagie on 2006-11-29
Simply using synaptic.
Go to System - Administration - Synaptic Package Manager
Click Search - type Glade - click Ok
Then a list of packages will appear. Simply scroll down until you find glade. Then click on glade and select mark for installation. Click Apply...
Or if you feel more confident with the terminal. Open the comand line from Applications - Terminal. And type:
```
sudo apt-get install glade
```
