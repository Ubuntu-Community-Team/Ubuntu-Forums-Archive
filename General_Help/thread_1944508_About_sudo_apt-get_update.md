### Post by oldos2er on 2012-03-21
Fix the NO_PUBKEY error with ```
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys E9C74FEEA2098A6E
```

To disable [http://ppa.launchpad.net/ubuntu-ppa/backports/ubuntu/dists/maverick/main/source/](http://ppa.launchpad.net/ubuntu-ppa/backports/ubuntu/dists/maverick/main/source/) run ```
gksu software-properties-gtk
``` and untick its box, which is probably under the 'Other Software' tab.
