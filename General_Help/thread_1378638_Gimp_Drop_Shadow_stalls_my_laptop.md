My first post in this forum. 
Since my upgrade to 9.10, I have a problem with certain filters in GIMP, especially Drop Shadow (in menu Filter-Light and Shadow). Some seconds after applying this filter, the screen of the Gimp becomes dark grey (non-responsive), but at the same time key presses and the mouse no longer function. Only solution is a hard powerdown of the laptop. 
I have opened System Monitor to see if there is a memory or CPU problem. Although mouse and keyboard no longer work, System Monitor does keep running and there is no raise in CPU load or memory. Could be an XORG related issue maybe?

_My question_: do more people have this behaviour? I have found a similar defect in launchpad [https://bugs.launchpad.net/ubuntu/+source/gimp/+bug/483957]("https://bugs.launchpad.net/ubuntu/+source/gimp/+bug/483957") , but there is not much reaction on it. 

My Setup:
- Ubuntu 9.10 32 bits with regular packages
- Gimp 2.6.7
- Laptop HP Pavillion DV 7 1090ed with 4G memory and Nvidia GeForce 9600M GT GPU.
