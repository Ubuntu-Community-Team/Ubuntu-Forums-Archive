Assuming a LiIO batt I might susggest that it needs recharging. If not the proper care of such a batt. requires a charge/discharge cycle between 70% and 15%. If you do otherwise it will degrade sooner than expected. Also, if you do not discharge it and only run on line power the batt.; because it has not been cycled properly; may fail sooner than expected.


Also, some non OEM batts. are very poor at accepting and retaining a charge.

If your batt. has really degraded by 2/3rds in only "a few months" I would try and return it to the seller; or, unfortunately, replace it.
