### Post by Locutus_of_Borg on 2008-07-17
enlightenment is a desktop environment, like GNOME or KDE, not a window manager like compiz is

so yes, you can install enlightenment and not do anything to compiz

compiz currently does not work with enlightenment, but you can get something called ecomorph that is the same thing.

though you have to be careful that you dont mess up compiz when installing ecomorph


also, enlightenment is very unstable, so be warned
