### Post by beastrace91 on 2010-06-05
On slightly less developed desktop environments such as XFCE or LXDE you will need to use a tool such as **xbindkeys** to create keyboard short cuts.

This post is old, but it is still applicable [http://ubuntuforums.org/showthread.php?t=79560&page=1](http://ubuntuforums.org/showthread.php?t=79560&page=1)

EDIT: There is also an **xfce-keyboard-shortcuts** package in the repositories that may be what you are looking for. Install it from synaptic if it is not already on your system.
