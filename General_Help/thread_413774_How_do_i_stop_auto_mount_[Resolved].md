### Post by aysiu on 2007-04-19
> **darksong said:**
> thanks you very much aysiu

it worked perfectly :D
I'm sorry. I just re-read your original post. You don't want it to automount. In that case, we will have to edit your /etc/fstab file. The command I gave you before just unmounts it now. It doesn't stop the partition from being automounted at boot time.

So do this.

```
sudo nano -B /etc/fstab
``` This will open the /etc/fstab file in the Nano text editor. Then put a # sign in front of the Windows partition. It should then look like this (pay special attention to the line in bold: ```
# /etc/fstab: static file system information.
#
# -- This file has been automaticly generated by ntfs-config --
#
# <file system> <mount point> <type> <options> <dump> <pass>

proc /proc proc defaults 0 0
# Entry for /dev/hda6 :
UUID=39bddc5b-f2d4-4dfa-aecd-24644682bd72 / ext3 defaults,errors=remount-ro 0 1
# Entry for /dev/hda1 :
**#UUID=0C68626768624F8E /media/hda1 ntfs umask=222,utf8 0 1**
# Entry for /dev/hda5 :
UUID=821089c1-b9ba-4b50-aa53-3f9795f36f65 none swap sw 0 0
/dev/hdc /media/cdrom0 udf,iso9660 user,noauto 0 0
/dev/hdd /media/cdrom1 udf,iso9660 user,noauto 0 0
/dev/fd0 /media/floppy0 auto rw,user,noauto 0 0
``` Then save (Control-X, Y, Enter). Now it should no longer mount at boot time.
