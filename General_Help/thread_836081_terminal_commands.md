### Post by manfer on 2008-06-21
The default shell on most Linux distros is bash. This is a good manual to learn bash:

[Bash-Beginners-Guide.pdf]("http://www.tldp.org/LDP/Bash-Beginners-Guide/Bash-Beginners-Guide.pdf")

There is a lot of documentation just installed in your computer too. You can see a short description of a command with command man. E.g.:

prompt:~$ **man ls**

There's a more detailed info for a lot of commands too with the command info.

prompt:~$ **info tar**

When you know how a command works but you need a reminder for what some option does, you type, command --help. E.g.

prompt:~$ **ls --help**

If you prefer to see documentation with a gui in ubuntu theres a lot in System->Help and Support. In the section Advance topics you can see documentation about terminal usage.

On the net the free documentation on linux is huge. A good place for some of those is The Linux documentation project.
