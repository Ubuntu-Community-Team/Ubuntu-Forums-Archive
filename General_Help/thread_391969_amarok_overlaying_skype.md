### Post by Mega_Fauna on 2007-03-26
I had a similar problem to this and got around it by changing the setting in Skype itself, I'm sure you've done this by now but if anyone else finds this post...

Tools --> Options --> Sound Devices --> 

Audio System to use: --> ALSA - 2 device(s) found

Audio In   --> Logitech USB Headset [my setup $40 + 2 hrs free skype time at local discount comp. store, not BB which sucks]

Audio Out --> Logitech USB Headset

Ringing --> Dell Sound Blaster Live! [my soundcard]

That way Amarok came out my speakers along with the telephone ringing and the conversations were done via the headset.

/Hope that helps
