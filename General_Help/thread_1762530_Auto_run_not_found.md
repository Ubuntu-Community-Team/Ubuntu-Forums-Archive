### Post by DanneStrat on 2011-05-19
Hi!

> So im in a predicament i need the drivers installed to be ensured my
Motherboard is running to potential etc 

When you install ubuntu, your hardware is detected and an 

appropriate driver module is loaded.

If everything works as expected, you don't need to do anything.

Bear in mind that, by default ubuntu is delivered only with free, 

open-source software, so for example, if you have a graphics card 

from nvidia and want to make sure you get full functionality, you 

can install the proprietary nvidia driver (property of nvidia and 

can not be shipped with ubuntu).

How to install proprietary drivers(works for all recent ubuntu 

versions):

[http://www.howtogeek.com/howto/17495/how-to-add-proprietary-drivers-to-ubuntu-10.04/](http://www.howtogeek.com/howto/17495/how-to-add-proprietary-drivers-to-ubuntu-10.04/)   

> Ive read about wine? but im not sure this will fix the problem?

You will not be able to use windows-drivers for your hardware via 

WINE. WINE is a windows compatibility-layer for running certain 

windows applications.

Wine compatibility list:

[http://appdb.winehq.org/](http://appdb.winehq.org/)

Let me know if you have any more questions!
