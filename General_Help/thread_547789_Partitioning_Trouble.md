### Post by por100pre1 on 2007-09-10
Try using Windows Disk Cleanup and remove all Restore Points except the last one. Use Disk Defragmenter one or two times. Then try using Gparted again. Don't forget to backup your Windows files! Any resizing task is risky, so be prepared!
