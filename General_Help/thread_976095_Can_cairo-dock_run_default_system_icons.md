### Post by grimdeath on 2008-11-09
I am trying out cairo-dock and really like it but I hate how it uses custom icons for programs like firefox, transmission, folders, etc. Right now it uses these cartoony blue icons for everything that I dont like at all.

I noticed if I browse to the themes file the icons are present there but I dont know if I should delete those or if I can toggle them off within the configuration options. Here's where I found the icons:
.cairo-dock/themes/myusernamehere

Certain programs that I have installed separately like Songbird seem to keep their own icon so I figure this must be possible.

*edit*
wow, figures I would find this option right after I make a post! to anyone else looking for this, here's where the options is:

Open the configuration options for cairo-dock, go to the Taskbar section, toggle off the "overwrite X icons with launchers"
