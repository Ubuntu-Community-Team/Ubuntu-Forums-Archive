### Post by taurus on 2009-03-21
1.  Run a checksum to check the integrity of the ISO image.

2.  Burn it at a slow speed, 4x.

3.  At the initial screen (booting from the CD), run the check cd for defects option to make sure the CD is good before continue with it.
