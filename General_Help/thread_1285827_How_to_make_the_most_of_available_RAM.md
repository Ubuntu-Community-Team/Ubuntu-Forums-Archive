### Post by snowpine on 2009-10-08
Hi JakP, I just googled your computer's specs... it has a pretty good CPU, but 256mb is pitiful for running Ubuntu+Gnome. :( However, you can get an inexpensive 1gb ram upgrade and that computer would be fantastic for Ubuntu!

> Recommended minimum requirements

Ubuntu should run reasonably well on a computer with the following minimum hardware specification. However, features such as visual effects may not run smoothly.

    * 700 MHz x86 processor
    * 384 MB of system memory (RAM)
    * 8 GB of disk space
    * Graphics card capable of 1024x768 resolution
    * Sound card
    * A network or Internet connection

If you are stuck with 256mb for whatever reason, I highly recommend CrunchBang linux... it is a lightweight Ubuntu "remix" using Openbox that I personally use on my old Dell laptop (600mhz Pentium 3 with 384mb ram). You could achieve a similar end by apt-getting openbox onto your existing Ubuntu install (though it wouldn't be as pretty out-of-the-box).

Good luck!

### Post by Mighty_Joe on 2009-10-08
> **snowpine said:**
>  256mb is pitiful for running Ubuntu+Gnome. :( However, you can get an inexpensive 1gb ram upgrade and that computer would be fantastic for Ubuntu!


What he said.  Ubuntu is designed to run on current computer systems.  Walk into a computer store and ask for a machine with 256Mb RAM and you'll get laughed at.  
Upgrading your ram will be much more efficient/effective than trying to tweak your way to a good user experience.
Crunchbang and other "lighter" Ubuntu spins are still going to be fairly heavy.  If you can't or won't upgrade your RAM, consider a Linux distro that's tailored for weak hardware, like [Puppy Linux]("www.puppylinux.org/")
