### Post by Dry Lips on 2011-10-17
```
sudo apt-get install gnome-session-fallback
```Select it on log-on. But I warn you, this isn't exactly the same
as the Gnome2 desktop, and it also looks horrible. So I recommend 
that you also:
```
sudo apt-get install gnome-themes-standard
```Right click on the desktop, select "change background" (or something)
and then you can select the theme you just installed by
clicking themes>adwaita.
