### Post by Darkade on 2008-06-24
Since you say you have enabled the 3D cube effect, I'll assume that you've installed CCSM (compizconfig-settings-manager). If you haven't isntall it.There are, at least, two ways (Second I think seems easyer)
1)Open a synaptic (System>administration>Synaptic Package Manager) ; then search for ccsm, select it and click on apply
2)In Applications menu, at the bottom there's Add/Remove Software; open it, search for ccsm, and install it.

Having CCSM installed, open it (System>Preferences>Advance Desktop Effects)
The thing is that you probbably still have just 2 workspaces. So look for general settings, and then the Desktop size tab. There select the horizontal size to 4. If you enabled desktop cube it will work. just press CRT+ALT+RIGHTArrow or LEFTArrow
