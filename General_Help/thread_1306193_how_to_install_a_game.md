### Post by Toadinator on 2009-10-30
Click Applications, Accessories, then Terminal. Once you're there, type:

```
sudo apt-get install wine
```
When you do this, your password *will not* appear as you type it for security reasons; don't worry about it.

OR, go to Add/Remove Programs (Or Ubuntu Software Center) and look for Wine. Then install it from there.

After this, pop in your CD and launch the setup program. It should install/run as normal.
