### Post by jpeddicord on 2008-05-19

My second suspicion is that gnome-panel is freezing, but if it is the menus shouldn't work. Another potential issue could be the window manager. If you want, try disabling desktop effects from System > Preferences > Appearance; Effects tab and see if that helps with the problem.
