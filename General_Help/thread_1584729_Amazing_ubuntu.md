You can contribute to Ubuntu in several ways without needing any technical expertise:

1. If you find a bug, report it: [https://help.ubuntu.com/community/ReportingBugs](https://help.ubuntu.com/community/ReportingBugs)

2. Of course you can donate money to the project: [http://www.ubuntu.com/community/get-involved/donate](http://www.ubuntu.com/community/get-involved/donate)

3. Or you can contribute artwork or help translate Ubuntu into a different language: [http://www.ubuntu.com/community/get-involved/non-technical-users](http://www.ubuntu.com/community/get-involved/non-technical-users)
