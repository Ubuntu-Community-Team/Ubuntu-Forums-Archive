### Post by mcduck on 2012-02-05
That's most likely because you are using Gnome Classic, which doesn't use Compiz by default...

Make sure you are actually running Compiz at the moment, press Alt-F2 and run "compiz --replace". If that solves the problem you should make sure Compiz is started on login.
