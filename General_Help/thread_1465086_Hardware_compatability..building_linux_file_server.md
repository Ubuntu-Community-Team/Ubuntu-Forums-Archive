Mythbuntu? For a file server? Are you sure about that?

I'd suggest installing regular Ubuntu - the latest version, 10.04 (it's in release candidate stage at the moment, but you can download it and then run Update Manager to turn it into the final release version).

As far as hardware, I don't know much about that motherboard, but otherwise it all seems fine. The best answer I can give you is "Try it"; Ubuntu can run from CD as a way of trying it out.

The 320GiB hard drive might be a bit small for a file server. My home server has a 1TiB hard drive that is rapidly filling up. I use mine for video though.
