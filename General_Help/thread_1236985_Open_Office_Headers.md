### Post by measekite on 2009-08-10
In Word there is an option for a first page header that can be different from the other headers.  You can also specify no first page header and then choose another header.

This can buy you page numbering on each and every page except for the first page.  You can also number in roman numerals as well like for a table of contents.

How can you do these  things in Open Office?

---

### Post by starcraft.man on 2009-08-10
What you want is Format Menu > Styles and Formating... At the top there are different buttons, click fourth one for page styles. There is one called First Page, the first page will be independent of all the rest. A good time to note that there are many other styles there, and you can customize them by right clicking and selecting modify. You can even create new ones off the default molds if you like. Have a look around.

### Post by jmszr on 2009-08-11
measekite,

For page numbering: [http://extensions.services.openoffice.org/project](http://extensions.services.openoffice.org/project) has "pagination" and a beta improvement called "pager" . I don't know if either is a total replacement. 

Here is a link to the OO.o User Guides: [http://documentation.openoffice.org/manuals/userguide3/](http://documentation.openoffice.org/manuals/userguide3/) , page 142 of the Writer Guide starts the header and footer section.    Page 221 starts the section that relates to starcraft.man's post.

Prior to adding extensions,you should install:```
sudo apt-get install openoffice.org-java-common
```

Hope this helps.
