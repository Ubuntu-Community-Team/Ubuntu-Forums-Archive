### Post by ZeldaFan on 2008-05-10
If you want to browse packages offered in the Ubuntu repos, you can check out the Synaptics Package Manager in System -> Administration -> Synaptic Package Manager. It sorts out all the packages available by the repositories you have selected within a GUI similar to a catalog. Is that what you were looking for?
