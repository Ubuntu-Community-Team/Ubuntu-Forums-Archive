If I create an OpenOffice document (Writer, Calc, Base etc.) can I open it in (exact binary file, without any conversion)
[1] OpenOffice Windows version?

### Post by zoiks on 2005-12-31
Yes.

Yes.

No conversion will be necessary, if you save files in ODF format.  (.ods, .odp, odt, etc.)  It's not OS dependent.  If you save in .doc, .xls, .ppt, etc., then saving the file uses conversion, and then opening it in OpenOffice will convert it back.  AFAIK, ODF isn't a binary format per se, it's a zipped xml format.
