Wine ([www.winehq.org](www.winehq.org)) is a Windows compatibility layer, that means that you an use windows prorams with it (and its 100% free). However, its not perfect, and it has some (many) incompatibility with programs (SPECIALLY GAMES). Anyway, WoW will work but Bioshock won'tk. You can check on [http://appdb.winehq.org/](http://appdb.winehq.org/) if a game works or not. ### Post by Chemroydal Tissue on 2008-07-11
We all switch/start using for different reasons, none any more lame than another, so don't worry about why you're using Ubuntu, it's important just that you're using it!

To your questions:

1 - WINE can be very useful, and can be easy to use. Not knowing anything about your experience with computing in general, I can't exactly say how it will be for you. When I started using WINE, I only had about a year's worth of experience using a Linux-based OS. WINE was fairly rough around the edges, then, and it was at times tricky to get some things working. It's loads better now - even though I don't use it personally, I've had to set up a few things, games for example, for family and friends. I would say that for a complete beginner, it's a bit tricky, but nothing about it should turn you away upon your first try. Just keep reading.

2 & 3 - You should have no problems with your Zune, but I'd check the forums for issues. The worst you may need is a different media player - I don't know whether or not Rhythmbox works with Zune.

Good luck, and welcome to the community.
