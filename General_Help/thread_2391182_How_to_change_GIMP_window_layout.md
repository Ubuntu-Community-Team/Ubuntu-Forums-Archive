### Post by waltd on 2018-05-06
Hi all, I just installed GIMP 2.10 via a PPA. I created a single-window mode with Windows -> Single-Window Mode. The tool bar is a column on the right side with small icons. However, on the Internet I see a GIMP window layout with the tool bar in the upper left side with larger icons, a top menu bar with "File", "Edit", "Select", etc, and also additional mini window functions on the left and right side within the main window.  How can I create this GIMP window layout that places the tool bar in upper left side, has a top menu bar and includes mini windows within the main window?  Also, how can I change GIMP's theme colors of light, gray, and dark. Thank you.

---

### Post by waltd on 2018-05-07
I found the solution. To create the mini windows, go to windows -> dockable dialogs and choose the mini window from a list.
