I'd suggest you download an Ubuntu 64-bit LiveCD and give that a go.  With a LiveCD you can give Ubuntu a "test drive" without making any permanent changes to your existing system.  Bear in mind though, that with a LiveCD you will not be able to install Flash, Java or play files that require proprietary codecs (like .mp3).  That functionality comes with a full install (this is due to some legal wrangling). 

As for your particulars...  

I can't address synching your mobile phone.  That gets a big "I don't know" from me.

iPod synching is no problem.

Burning and making backups of DVD's is no problem.

Open Office will, most likely, handle all your document and spreadsheet needs unless you need some of the truly hardcore functions of Excel (e.g. Pivot Tables, graphing functions).

Also, if there are some applications that run in a Windows environment that you simply can't find an open source solution for, you can always fall back on running Windows in a virtual machine.  There are a few applications that allow you to do this.
