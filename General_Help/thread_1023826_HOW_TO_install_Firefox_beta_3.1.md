### Post by SonnHalter on 2008-12-28
Quick guide if you want to test out the beta. 

[SIZE="6"]**step 1**[/SIZE]


Go to system>administration>software sources
click on 3rd party software and then click add

[IMG]http://i41.tinypic.com/281wl6w.jpg[/IMG] 


Once you clicked "add" type this:

```
deb http://ppa.launchpad.net/fta/ubuntu intrepid main
```
**MAKE SURE IT IS CHECKMARKED ON THE SCREEN ALSO!**


now it should say it wants to reload packages. let it. then go to synaptic manager and search for firefox. the package should be firefox 3.1 or just look on the right side of the panel for 3.1 instead of 3.0.5.
