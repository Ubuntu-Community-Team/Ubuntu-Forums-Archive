### Post by talisfang on 2008-05-03
I installed ubuntu 8.04 by wubi. I select ubuntu on the boot screen and it waits 10 secs to start. I want to reach grub settings and remove the time. Help pls

---

### Post by TeoBigusGeekus on 2008-05-03
Go to control_panel > system > advanced > startup_and_recovery. 
You can edit there the default os as well as the timeout.

---

### Post by Big-Wayne on 2008-05-03
You could also install qgrubed, great for editing you boot list. You can choose your default os and take unwanted kernel boot options. Just install it from synaptic, or sudo apt-get install qgrubed.
Not sure if will work as you installed with wubi, but worth a try.
