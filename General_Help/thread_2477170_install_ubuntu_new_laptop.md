### Post by TheFu on 2022-07-16
I have an Asus core i5 (8th gen) version of this laptop.  I suspect the newer core i3 is about the same performance level.  You can run any OS you like, but if there isn't a separate discrete GPU, then you'll be happier if you avoid Gnome and KDE. Any of the other DEs will be fine.

As to which distro, any should be fine. Ubuntu, Mint, Fedora, SuSE, whatever.  Find one that makes you comfortable with their popularity, support channels, updates, risk mitigation against bugs and trustworthiness.  This being the Ubuntu forums, we're gonna push you towards Ubuntu-based OSes with something like XFCE or LXQt as the DE, but Mint is a fine distro for people who want to avoid some of the new complexities that Canonical is thrusting on most of us by using 3 different package management tools.  

Mint still uses 1, apt, which makes for a cleaner, no-hassle, experience, while Canonical is still beta testing some of their newer Snap packaged software which is causing issues with integrated uses and not being allowed to access parts of the OS we previously had no issues accessing.

But if you want to jump into the deep end of the pool, go with Ubuntu Gnome and see if you don't like it. Many people do (not me).  I'm odd.

Oh ... 20.04 LTS (with no DE) is what I'd install today. 22.04 needs more seasoning. Perhaps by 2023, it will be ready for end-users. Who knows?
