### Post by aysiu on 2007-05-26
As far as I know, Beryl and Metacity are mutually exclusive.

You're either running Beryl... or running Metacity. You're not running both at once.

If you're not running Beryl, go to System > Preferences > Themes. Drag and drop the Metacity theme .tar.gz to the theme manager window.

(sorry for my bad english) So most Metacity themes are one theme packaged as a .tar.gz you can drag and drop to theme manager window. This one isn't a theme but a theme *pack*, meaning it's a .tar.gz full of other themes. So double-click the .tar.gz and extract it. If you see a bunch of folders, copy them to /home/*username*/.themes. If you see a bunch of other .tar.gz files, drag those to the theme manager window.
