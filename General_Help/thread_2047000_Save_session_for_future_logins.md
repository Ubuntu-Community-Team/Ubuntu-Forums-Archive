### Post by Toz on 2012-08-25
@vasa1, 
from what I understand, saving a session will save the "open" programs and restart them automatically for the next login. So the advantage would be that you wouldn't need to re-start them - especially if you tend to run the same programs every time you log in. Where this gets confusing for me is when you start using saved sessions with autostart. I find that if you have an application in autostart (e.g. conky), saving the session will actually start 2 instances of conky (or any application that allows multiple instances). One from the saved session and one from the autostart. Personally I find this confusing and tend to not use saved sessions but rather autostart.
