### Post by xeyeile on 2008-05-13
> **mn102193 said:**
> i need help installing an emerald theme. i have the theme manager, but when i click the theme i want, nothing happens.


I had a problem with this also, this is a work around that I figured out that seems to work. (Assuming you are using Compiz)

Go to system->preferences->advanced desktop settings

Once in the Compiz Settings Manager, scroll down until you see window decoration, enable it, then click on it and it will bring up options.

Under the General tab, there is a section that says "Command"

Replace the command here with "emerald --replace"

Refresh the theme in emerald and this should work, you might need a reboot.

Hope this helps!
