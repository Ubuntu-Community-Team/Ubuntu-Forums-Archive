### Post by markbuntu on 2008-08-20
Different users get different sessions. The easiest way would be to setup different users the way you want and use System/Preferences/Session to save them as user defaults that would start on login. 

Once you are logged in as a specific user, you can use Preferred Applications and the desktop settings etc, to set things up and Sessions to save the setup. Your regular user can have compiz, etc. and you music engineer user would not.
