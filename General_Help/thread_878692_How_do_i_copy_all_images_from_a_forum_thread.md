### Post by billgoldberg on 2008-08-03
The firefox add-on downloadthemall! can do it.

[https://addons.mozilla.org/en-US/firefox/addon/201](https://addons.mozilla.org/en-US/firefox/addon/201)

Go to "tools -> downloadthemall tools -> downloadthemall"

There should be a "image" tab.

You can also set what image types you would like to download "png, ...".

Note that this will also download smilies and stuff like that.

### Post by w0ng on 2008-08-03
some forums have printable versions that allow you to view all posts in a thread.
that particular forum limits it to 40 posts per page max:
[http://www.xboard.us/printthread.php?t=214795&pp=40](http://www.xboard.us/printthread.php?t=214795&pp=40)
use DownloadThemAll and you only need to use it on 3 pages as opposed to 10
