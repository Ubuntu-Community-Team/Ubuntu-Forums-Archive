### Post by sudodus on 2016-04-04
Ubuntu 15.04 was supported for nine months (until January). So you can no longer expect that it is supported. If you can start it again, please upgrade it to 15.10 or make a fresh installation of 14.04.1 LTS or 14.04.4. Before the end of this month 16.04 LTS will be released, but I suggest that you wait for a few weeks for the worst bugs to be squashed before you install it.

-o-

Generally, I suggest that you stay with the _l_ong _t_ime _s_upport versions, LTS (14.04 and 16.04). They are released in April (month 4) of even years. Standard Ubuntu of these versions are supported for five years (while the intermediate versions are only supported for nine months).
