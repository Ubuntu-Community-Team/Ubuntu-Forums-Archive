### Post by carmenat250 on 2011-07-09
I downloaded VLC through the Synaptic Packet Manager.

When I double click on a movie file, Ubuntu still tries to open up Movie Player.  How do I set VLC as the default movie player for all video files?

---

### Post by matt_symes on 2011-07-09
Hi

Open Nautilus file browser. 

Right click one of the movie files and select the open with tab. 

Select VLC and then close the tabs.  Doubling clicking should now open in VLC.

This works in 10.04.
