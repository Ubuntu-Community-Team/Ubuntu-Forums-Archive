### Post by avtolle on 2008-06-25
You've installed using Wubi. This is not a true "dual boot"; rather, the choice to boot Windows or Ubuntu will appear in the Windows bootloader. You will be able to choose, and boot, either. I've been running 8.04 AMD-64 installed via Wubi in Vista with no problems since 8.04 was in beta (I've kept it totally updated, etc.). Your experience may be the same, or it may vary. See the Wubi Forum here for some FAQs, threads, etc., which may help if you run into any problems.
