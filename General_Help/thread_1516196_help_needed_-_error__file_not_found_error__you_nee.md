### Post by Noz3001 on 2010-06-23
Well, do this on the LiveCD:

[B]mkdir /mnt/ubuntu
mount /dev/sda6 /mnt/ubuntu[/B]

Then do
**ls /boot**

Write down the file name of the vmlinuz- file and then

**cp /boot/[COLOR=Red]<put the filename here>[/COLOR] /mnt/ubuntu/boot**

then

**nano /mnt/ubuntu/boot/grub/grub.cfg**

Look for 'menuentry Ubuntu, with Linux.....'

and then change the beginning of the "linux" line a few lines under to

[B]linux /boot/[COLOR=Red]<put the filename here>

BUT DON'T CHANGE THE PARAMETERS AFTER THIS 

[/COLOR][/B][COLOR=Red][COLOR=Black]Then  Ctrl+O, Ctrl+X and reboot[/COLOR]
[/COLOR]

### Post by Noz3001 on 2010-06-24
Okay, try this.

_64bit_
```
wget http://kernel.ubuntu.com/~kernel-ppa/mainline/v2.6.34-lucid/linux-image-2.6.34-020634-generic_2.6.34-020634_amd64.deb

sudo file-roller linux-image-2.6.34-020634-generic_2.6.34-020634_amd64.deb

```
_32bit_
```
wget http://kernel.ubuntu.com/~kernel-ppa/mainline/v2.6.34-lucid/linux-image-2.6.34-020634-generic_2.6.34-020634_i386.deb

sudo file-roller linux-image-2.6.34-020634-generic_2.6.34-020634_i386.deb

```

Open data.tar.bz2, then open the . directory and extract the contents of the boot directory to you Ubuntu installation's boot directory. (Don't forget to mount it)
