### Post by warp99 on 2008-05-25
When you don't shut down properly the drives are checked when you reboot and if there is any corruption they are repaired or placed into read only mode. If you need to troubleshoot while booting your can remove the usplash screen temporarily so you can diagnosis any error message.

During the boot sequence when the words "Grub loading" appear on the screen quickly press 'escape' to enter the boot menu and the first line should be highlighted. Press 'e' to enter edit mode then highlight the line marked 'kernel' and press 'e' again. Delete the parameters 'quiet' and 'splash' from the kernel string then press enter and then 'b' to boot. Now when you boot you can see if there are any error messages in the boot sequence.
