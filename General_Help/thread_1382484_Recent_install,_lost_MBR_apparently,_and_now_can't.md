### Post by Haitoyou on 2010-01-16
I think that Windows 7, with or without updates, scans the boot sector in search of the MBR. When it detects Grub/Grub2 installed instead (non MBR), it breaks itself, refusing to work until the MBR is reinstalled.

I am still not sure if Windows 7 breaks itself before or after updates, but it definitely happens due to it not detecting the MBR.

You are getting those errors because you are trying to install Grub 2 to a partition instead of a device. Try installing it to the device (sdb) and not the specific partition (sdb2).
