### Post by abyrne on 2011-03-20
You could try [Copyfind]("http://plagiarism.phys.virginia.edu/software.html"), but that does not check against resources on the web, only against local files that you specify. I don't think that's what you're looking for.

If you really need something that is software-based, you could just try using one of those Windows plagiarism checkers through [WINE]("https://help.ubuntu.com/community/Wine").
