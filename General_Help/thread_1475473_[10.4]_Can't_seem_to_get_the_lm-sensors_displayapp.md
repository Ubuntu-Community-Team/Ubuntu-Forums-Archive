### Post by Ctrl+C on 2010-09-12
I did that.

[IMG]http://shareimage.org/images/avgvsk2fnyz0yl70aq0i.png[/IMG]

You need to change 2 system files:
1) sudo vim /var/lib/gconf/une.mandatory/%gconf-tree.xml

Add the sections marked as italic and follow _[  inline comments  ]_
```

<?xml version="1.0"?>
<gconf>
    <dir name="apps">
[...]
        <dir name="panel">
[...]
            <dir name="general">
[...]
                <entry name="applet_id_list" mtime="1272540841" type="list" ltype="string">
[...]
[U][   In this section find available applet_N which is not listed   ]
[   here and add it (copy-paste other and change its number)   ]
[   In my case it was applet_4   ][/U]
                    [I]<li type="string">
                        <stringvalue>[/I]**applet_4**[I]</stringvalue>
                    </li>[/I]
[...]
                </entry>
[...]
            </dir>
            <dir name="applets">
[...]
                <dir name="applet_2">
                    <entry name="bonobo_iid" mtime="1272540841" type="string">
                        <stringvalue>OAFIID:GNOME_IndicatorApplet</stringvalue>
                    </entry>
[...]
_[   Change its position (was 3)   ]_
                    <entry name="position" mtime="1272540841" type="int" **value="4"**/>
[...]
                </dir>
                <dir name="applet_3">
                    <entry name="bonobo_iid" mtime="1272540841" type="string">
                        <stringvalue>OAFIID:GNOME_NotificationAreaApplet</stringvalue>
                    </entry>
_[   This too (was 4)   ]_
                    <entry name="position" mtime="1272540841" type="int" **value="5"**/>
[...]
                </dir>
_[   Add your applet_N description (copy-paste other and change appropriate values)   ]_
                *<dir name="***applet_4**[I]">
                    <entry name="bonobo_iid" mtime="1272540841" type="string">
                        <stringvalue>[/I]**OAFIID:SensorsApplet**[I]</stringvalue>
                    </entry>
                    <entry name="position" mtime="1272540841" type="int" [/I]**value="3"**[I]/>
                    <entry name="panel_right_stick" mtime="1272540841" type="bool" value="true"/>
                    <entry name="locked" mtime="1272540841" type="bool" value="true"/>
                    <entry name="object_type" mtime="1272540841" type="string">
                        <stringvalue>bonobo-applet</stringvalue>
                    </entry>
                    <entry name="toplevel_id" mtime="1272540841" type="string">
                        <stringvalue>top_panel</stringvalue>
                    </entry>
                </dir>[/I]
[...]
            </dir>
        </dir>
    </dir>
</gconf>


```
2) sudo vim /usr/share/gconf/une/mandatory/20_une-gconf-mandatory

The same thing but in other format:
```

[...]
/apps/panel/general/applet_id_list [applet_0,applet_1,applet_2,applet_3**,applet_4**,applet_5,applet_6]
[...]
/apps/panel/applets/applet_2/position **4**
[...]
/apps/panel/applets/applet_3/position **5**
[...]
/apps/panel/applets/applet_4/bonobo_iid **OAFIID:SensorsApplet**
/apps/panel/applets/applet_4/locked true
/apps/panel/applets/applet_4/position **3**
/apps/panel/applets/applet_4/object_type bonobo-applet
/apps/panel/applets/applet_4/panel_right_stick true
/apps/panel/applets/applet_4/toplevel_id top_panel
[...]

```

Of course you need the sensors-applet installed :)
