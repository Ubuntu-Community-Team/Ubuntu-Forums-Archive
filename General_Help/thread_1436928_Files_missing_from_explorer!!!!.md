### Post by bird1992 on 2010-03-23
I figured it out.  I feel rather dumb admitting what was wrong, but keep in mind i'm a newbie.  I made changes to my disk while I was in ubuntu, but windows xp was still hibernating so it couldn't register the changes so when i restarted the file indexing was erased, so i just ran chkdsk in windows xp boot and all my files are there now.  So it turns out that its a bad idea to use ubuntu while windows is hibernating.
