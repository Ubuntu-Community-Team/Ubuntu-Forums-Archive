### Post by BertN45 on 2012-12-29
If you did not yet put much effort in the installation, I would consider to install Lubuntu 12.10 and add the programs you need and like from Ubuntu.

Otherwise you will have some work to do, because you have lot of functions double, 
- like nautilus and pcmanfm
- abiword and libre-office writer
- rhythmbox and audacious
and many more.

You could go to the ubuntu software center, choose installed software and look for each category, like "sound & video", what is double functionality and choose the one you like to keep and remove. In making the choices you have to remember that many of the Ubuntu programs give richer functionality and will need more CPU and memory if used.

This action will free some disk space, but it will not really improve your memory occupation or CPU loading. I expect that your memory occupation does not exceed 128 MB after you logged into LXDE, which would mean that almost nothing of the Unity functions has been loaded. If it is more, it is wise to use task manager or system monitor to check which proces(ses) use most memory.

Afterwards I would install Ubuntu-tweak and use the janitor tab to remove all remainders of packages and old Linux kernels, that will save you another number of 100 MBs on disk.

It also is useful for helping you, if you report, memory and disk sizes and the CPU type and clock speed.
