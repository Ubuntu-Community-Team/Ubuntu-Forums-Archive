### Post by robobot on 2009-07-15
A website I need to access for work on a grant proposal only allows clients to log in using Internet Explorer or Safari. However, since I use Ubuntu, I have access to neither of those browsers. Is there a way to make Firefox lie to the server and say that it is Safari?

---

### Post by jerome1232 on 2009-07-15
Open Firefox

Tools - Addons - Search for User Agent Switcher and install it, you can select a few different browsers on a few different OS's.
