### Post by Serpher on 2010-03-16
Knowing just these few commands alone isn't going to do anything, I suggest you look up a guide on bash (the shell used in most Linux distros these days). The command 'cd Desktop' will only work when you're in your home directory. The home directory is located at '/home/<yourUserName>/' so assuming my username is  'matt' my Desktop is '/home/matt/Desktop/'. You can use the 'cd' command to go to a directory in your current directory (ls) or by typing in the full one like so: 'cd /home/matt/Desktop/'
