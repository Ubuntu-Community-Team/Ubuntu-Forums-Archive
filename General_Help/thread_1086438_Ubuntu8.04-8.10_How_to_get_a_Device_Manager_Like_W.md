### Post by BigBubbaT on 2009-03-04
Go into your Synaptic Package Manager and Search "gnome-device-manager" select it (it will need to install another file) hit apply. Once done you can find it in Applications> System Tools. Under view check box "device properties". There you go, enjoy your windows like device manager!:D
