### Post by saltmarshlamb on 2011-09-06
I don't use gnome so I'm not able to answer the unity questions. 

I assume by shortcuts you mean those available when using unity - [http://www.ubuntugeek.com/list-of-ubuntu-unity-keyboard-shortcuts.html](http://www.ubuntugeek.com/list-of-ubuntu-unity-keyboard-shortcuts.html)

As far as automounting partitions, they need to be added to fstab.

You can do so manually - [https://help.ubuntu.com/community/Fstab](https://help.ubuntu.com/community/Fstab)

You can use gui tools to do so - [http://ubuntuforums.org/showthread.php?t=872197](http://ubuntuforums.org/showthread.php?t=872197)

For the terminal operations - look for bash tutorials
