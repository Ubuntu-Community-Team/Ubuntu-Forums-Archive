### Post by tjwoosta on 2009-04-13
it looks to me like you have duplicate entries in your xorg.conf for sections "monitor" "device" and "screen"


i suggest removing the first entry for each of those sections (but first make a backup just incase, save backup as Xorg.conf.backup)

so after saving a backup remove one entry that looks like this

Section "Monitor"
Identifier "aticonfig-Monitor[0]-0"
Option "VendorName" "ATI Proprietary Driver"
Option "ModelName" "Generic Autodetecting Monitor"
Option "DPMS" "true"
EndSection

and the one that looks like this  

Section "Device"
Identifier "aticonfig-Device[0]-0"
Driver "ati"
BusID "PCI:1:0:0"
EndSection
(**make sure to keep the one that says fglrx**)

and remove one that looks like this

Section "Screen"
Identifier "aticonfig-Screen[0]-0"
Device "aticonfig-Device[0]-0"
Monitor "aticonfig-Monitor[0]-0"
DefaultDepth 24
SubSection "Display"
Viewport 0 0
Depth 24
EndSubSection
EndSection


then just save as Xorg.conf and reboot
