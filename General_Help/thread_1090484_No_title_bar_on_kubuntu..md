### Post by HavocXphere on 2009-03-08
> Well, the title is pretty much self explanatory.
Not really no.

The titlebar(s) is the thing at the top of each window that has the name of the window in it and the three buttons to minimize, maximize and close the window. If this is missing then its in all likelihood related to a graphics/desktop effects issue. Possibly also a font issue if you've set a non-standard font in the window appearances. e.g. MS fonts.

Only the titlebar(s) don't load on on KDE start...there has to be a window. And they aren't used to navigate per se either.

So I'm guess you either mean the bottom panel itself, or the task manager widget which lists the titlenames.:confused:

I'd suggest switching off desktop effects and changing to a standard desktop theme.

You can also try *temporarily* moving the hidden .kde folder in your home directory to see if this helps. Note that the move should be a temporarily troubleshooting exercise so don't delete the folder. KMail stores email there etc.

It would also be good if you could post a screenshot for the next person who comes along and reads this thread.;)
