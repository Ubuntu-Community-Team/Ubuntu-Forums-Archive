### Post by lovinglinux on 2009-04-22
You don't need to copy, but you could make a backup just in case.

I don't get two windows like you described. I just click the Greasemonkey in the status bar with the right mouse button and select "Manage User Scripts", then select the script I want to edit and click edit. The script will be opened with the text editor. If you haven't selected a default text editor, then it will show you the file browser, so you can pick it up. Al you have to do is edit the file and save it.
