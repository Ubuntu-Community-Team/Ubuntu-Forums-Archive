### Post by Dreamy1 on 2007-10-23
Mind you though Firefox does leak memory, so in a short period of time Firefox can crash because it has trouble allocating only enough memory it should use, this problem is unfortunately going to deaf ears to the Firefox developer community. Memory leaks are the result of improper coding of an application or failing to add certain variables in the code that tells the application to stop eating further memory. 

In Windows if a user uses Firefox heavily after a while it will use around 70-90% of the CPU and cause Windows to get unstable, maybe even crash totally, but fortunately with Ubuntu it just forces the application to quit regardless. That could be the problem, but Firefox has suffered this for years though.
