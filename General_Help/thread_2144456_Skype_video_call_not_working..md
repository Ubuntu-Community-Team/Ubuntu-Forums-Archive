### Post by vandamme on 2013-05-14
If you're desperate: dig around in the cellar for an old version of Linux, running kernel 2.6, and install that and see if it works. 

I'm using a video camera and tuner card (common bt878 chipset) which worked fine in older Mint versions (kernel 2.6). Since kernel 3, I haven't been able to get video into Skype. Now I'm on Ubuntu 13.04, same problem in Mint 14. On the Options->Video Devices, the TV card listed and /dev/video0 is accurate, but I get just a black screen. Cheese and TVTime work fine. There's no "Test" button in the screen, just black. It runs fine on my HP laptop, though.
