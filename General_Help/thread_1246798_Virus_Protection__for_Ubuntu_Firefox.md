### Post by sasho_zl on 2009-08-22
The only protection for Firefox that you need is the NoScript plugin. It will prevent websites you don't trust to execute malicious code in your system.
You can read this thread for more information about Ubuntu Security - [http://ubuntuforums.org/showthread.php?t=510812](http://ubuntuforums.org/showthread.php?t=510812)
About the printer question - it is very unlikely that you are going to experience any problem because the linux kernel has build in drivers for almost everything.
