### Post by foresto on 2008-04-04
In Xubuntu Gutsy, I can do it by starting Geany from my Applications menu, and then starting it again.

You might also try launching a second instance by typing 'geany' in a terminal window, or in a command line launcher like the one that shows up with Alt+F2 on some desktops.
