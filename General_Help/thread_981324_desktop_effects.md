### Post by quimico69 on 2008-11-13
You need to set up compiz.  You can do that by right-clicking on the background and choosing the "visual effects" flap.  There you can select "extra".

Not all graphics cards can support all the effects.

I would also recommend to install "compizconfig-settings-manager". Try

sudo apt-get install compizconfig-settings-manager

it will ask you for the root password before installing it. You will find the installed manager under Preferences and there you can customize to your heart's content.

### Post by waffleturd on 2008-11-13
you can get compiz by applications>add/remove and then search it.
Then get ccsm or Advanced Desktop Effects settings.
Then get the simple compizconfigure setting manager.
Then you got to system>preference> appearence
click the visual effect tab and then scroll down and there should be a custom option.
Click that and you should be good to go!:)
