### Post by lobo235 on 2008-11-01
I just upgraded my laptop to Intrepid and I had some trouble installing Opera. Using apt-get does not work for installing Opera in Ibex but I believe this worked in 7.10 and possibly other Ubuntu releases. Going to the Opera website reveals that they have install packages for most versions of Ubuntu but they are lacking an install for Ubuntu 8.10 Intrepid Ibex.

I ended up installing the package they have for Ubuntu 8.04 Hardy Heron and that seemed to work fine. Even flash is working in Opera. Until Opera releases a package specifically for Intrepid the one for Hardy works. I just wanted to let you all know in case someone else needed help with this.
