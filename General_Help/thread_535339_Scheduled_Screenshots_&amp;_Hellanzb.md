### Post by fradav4 on 2007-08-26
I'm trying to accomplish a couple of things.  I'll begin with my current setup...

I have 5 computers on my home network, each with a different user and an XBOX running XBMC.  On each computer I have created two shortcuts, one labeled 'Queue' the other 'Done'.   I have taught each user how to drop an nzb file into the queue folder where my computer has Hellanzb running and processes the nzbs.  I tell them to wait about 5 minutes for every 300 MBs they download.  (I capped the connection to 1MB/s).  The problem is I have them coming into my room asking me the status of their download all the time.  What I would like to do is create a script that takes a screen shot of only my terminal window running Hellanzb every 30 seconds and have it drop the image in the done folder (replacing each previous image).  This way every 30 seconds they could see the status of their downloads.

If others have found a solution to my problem, I'm very interested in knowing about them.  Thanks for the help

### Post by fradav4 on 2007-08-27
For anyone else that may have had this problem, I found a solution.  There is a cool web app called HellaWorld that gives you more control over Hellanzb and allows users to monitor hellanzb from other computers.  You have to have a web server running though.  The whole setup took me less than 5 minutes and I'm new to linux.
