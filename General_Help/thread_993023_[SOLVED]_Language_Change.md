---

### Post by CatKiller on 2008-11-25
System -> Administration -> Language Support. Put a tick in the box with the language you want to use. You can change the default language for new users here, if you wish.

Log out.

On the login screen, select Options -> Language. You'll be asked whether you want it to be a permanent change.
