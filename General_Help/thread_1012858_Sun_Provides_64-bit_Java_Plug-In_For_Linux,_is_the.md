It won't be packaged for the repositories until after Sun ships the final build through the jdk-distros project.

[https://jdk-distros.dev.java.net/](https://jdk-distros.dev.java.net/)

Sun has made some noise that the final build of java 6u12 might be in February.  In the past jdk-distros lagged significantly from the mainstream release, but for java 6u10 and 6u11 the releases were done simultaneously.
