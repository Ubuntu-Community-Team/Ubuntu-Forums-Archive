### Post by nsivin on 2008-05-13
I need to park a utility (Glipper) in the startup directory so that it will load when Ubuntu starts up. There is nothing in the help system about where it is. Can anyone tell me?

---

### Post by TeoBigusGeekus on 2008-05-13
You need to go to System->Preferences->Sessions. On the startup programs tab click add. Give a name, any name, and then the command that starts your piece of software. Click ok and you are done...

### Post by Xgen on 2008-05-13
With some programs yes, but with Glipper simply right click on the panel menu--->Add to Panel--->Clipboard Manager.
