### Post by ajgreeny on 2008-07-27
Assuming you are using gnome, go to System>>Preferences>>Preferred Applications and set Thunderbird as your preferred email program.  I think Thunderbird will then open when you use the "Send by mail" option in f-spot File menu.  It certainly works in mine.
