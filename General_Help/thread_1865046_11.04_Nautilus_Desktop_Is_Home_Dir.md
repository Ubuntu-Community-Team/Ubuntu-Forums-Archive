### Post by rmlrml on 2011-10-19
I'm stumped.  I want to stop showing my home folder contents as icons on my desktop.  I would like to only see mounted volumes.  I've tried three routes:

In gconf, "desktop_is_home_dir" is unchecked

In Ubuntu Tweak, "Show desktop icons" and "Show mounted volumes on desktop" are checked while "Show contents of Home Folder on desktop (Logout for settings to take effect)" is UNchecked.

In gTweakUI - Nautilus, "Use nautilus to draw desktop" is checked, and "Use home folder as desktop" is UNchecked.

So it all seems to be exactly the way I want it, as far as I can tell, and I have logged out and logged back in, and restarted, but still all my Home Folder's subfolders are drawn all over my desktop.

Thanks in advance

### Post by duke.tim on 2011-10-24
Open the file ~/duke/.config/user-dirs.dirs

Find the line that says something like
> XDG_DESKTOP_DIR="[somethingpointingtodesktop]"

Change it to an empty directory (or your desktop directory ~/Desktop)
