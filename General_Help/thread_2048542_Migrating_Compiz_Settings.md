### Post by Frogs Hair on 2012-08-26
Importing and exporting the profile can be done from preferences in Compiz. It is a good idea to save a copy of the default profile in a folder for backup before importing the new one.

When the new profile is imported(import as) the new file it needs to be named unity if that is what is in use. Saving the profile on usb or attaching it to an email will work for moving it to the other computer.
