### Post by meng on 2006-05-23
The metafile is the .torrent file which describes the file/files to be downloaded. There is more information about bittorrents [here]("http://en.wikipedia.org/wiki/Bittorrent").

Regarding your problem, what is it that you are starting up? A bittorrent client? If so which one? You can configure your browser to associate .torrent files with a default bittorrent client, and automatically download that way.

### Post by it.henrik on 2006-05-23
you usually find those .torrent-files at search engines .. yeah I know it sucks, but this was intentional by the guy who came up with bit-torrent (Bram Cohen). 

A great plugin for firefox is "sendmefile Torrent toolbar", with this toolbar you can search for torrents at all of the most popular search engines for torrents.

Bittorrent is a really cool way of distributing files .. you can upload a 100MB file that 500 others are downloading at the same time. Everyone will get the file and you might end up uploading something just above 100MB.
