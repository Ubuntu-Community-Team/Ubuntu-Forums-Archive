### Post by jasay on 2006-03-22
[QUOTE=thunderfox933]how do i install applications that are linux[/QUOTE]Easiest way:
System menu > administration > synaptic package manager.  Search for whatever you want, click on a package > mark for installation, and hit apply on the top.  It should automatically download, install and make a menu entry for the new program.
