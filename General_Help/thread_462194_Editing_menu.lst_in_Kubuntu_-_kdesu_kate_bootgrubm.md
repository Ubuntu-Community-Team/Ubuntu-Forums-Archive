### Post by mijj on 2007-06-02
kdesu is the su instruction for a graphical interface ... kate is the editor .. maybe he doesnt have kate installed - kwrite would do.

And .. if you're using Kubuntu .. you dont need to go to the command line ... you can drill down via mouse into the /boot/grub directory .. then right click on menu.lst and select "Actions > Edit as root".
