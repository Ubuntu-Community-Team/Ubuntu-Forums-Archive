### Post by AceRoom on 2010-07-13
Try using something like Reconstructor or UCK. That's probably contained in a particular package. 

There's also another way. If you're doing this on a pendrive, you can use MultibootISO. It allows you to boot from different ISOs on a single pendrive. For example, you can have the ISOs of Ubuntu, Mint, Gparted, DSl all on the same pen drive and run them all (One at a time of course) from it.

Check it out. When I'm booting Ubuntu or even Mint from that, it doesn't show me that menu. Although it does show me a menu to choose which ISO to use...
