### Post by DaBluesman on 2014-12-09
I am using weather watcher live for zararadio.  In my new Ubuntu 14.10 pc I can't get weather watcher to run at all.  so i did this....

I am using a dedicated pc to run Ubuntu and Zararadio.  The file zararadio reads to get current weather information is on the Ubuntu pc.    

I run Weather watcher on a Windows pc.  Then in the weather watcher setup I set the update file location to a map drive on the Ubuntu machine.  Of course the Ubuntu drive is shared and his appropriate permissions set.
