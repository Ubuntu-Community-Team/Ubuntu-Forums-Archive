### Post by raequin on 2017-11-02
I've added a few keyboard shortcuts, some via the Keyboard application and some via compizconfig-settings-manager.  When I power on the computer they do not work.  Logging out and logging back in is required to get these keyboard shortcuts to operate.  Any idea what I can do to have them active at startup?  This behavior occurs on both laptops I use running 16.04.  One is a System76 and one is a Dell.

---

### Post by raequin on 2017-11-13
This problem has the following acceptable solution: if I disable "Automatic Login," so that my password is required at startup, there is no longer a problem with my keyboard shortcuts being active!  It would be nice if I could have both, auto-login and working shortcuts, but this situation is better than having to logout every time I start the computer.
