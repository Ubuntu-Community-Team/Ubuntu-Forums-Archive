### Post by webjames on 2007-03-14
Hi I have been using writer for a little bit, but after coming from MS Word I have found that it doesn't capatilise an 'i' to an 'I'. For example:
in open office this is how it would look:
> ...and i said to my friend...
but in word, as I type it replaces 'i' with 'I' like so:
> ...and I said to my friend...

i have added this to auto correct:
correct:
```
i<SPACE>
```
with:
```
I<SPACE>
```
but it doesn't auto correct.

is there another was, I have tried searching but it is difficult as I is a very common letter.

Thank you,

James :-)

---

### Post by webjames on 2007-03-14
I have found a solution:

instead of putting a space after the 'i' just put 'i' and replace with 'I'.

I don't know why I didn't think of this before. I guess I thought it would correct all I's:

lIke thIs

but it doesn't!

Thanks!
