### Post by bigboy_pdb on 2007-10-15
You can use synaptic package manager (System -> Administration -> Synaptic Package Manager) to install everything that you'd like to install.

To get the flash plugin, video codecs, and MS fonts, install the packages:
flasplugin-nonfree
gstreamer0.10-ffmpeg
gstreamer0.10-plugins-base
gstreamer0.10-plugins-bad
gstreamer0.10-plugins-bad-multiverse
msttcorefonts

I don't have Beryl so someone else might need to help you with that. Although it might be a good idea to just wait for Ubuntu 7.10 to be released because Beryl is packaged with it and it should be installed when you upgrade.

### Post by reza81 on 2007-10-15
there are two way's to install flash (mozilla firefox)

```
System-->Administration-->Synaptic Package Manager-->Search-->flashplugin-nonfree-->Mark for installation
```

and

```
sudo apt-get install flashplugin-nonfree
```
