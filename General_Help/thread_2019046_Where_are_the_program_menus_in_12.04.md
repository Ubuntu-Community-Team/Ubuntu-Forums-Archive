### Post by timbo59 on 2012-07-08
I read the post you linked to and found it interesting. That said, there's a couple of points I'd make regarding the supposedly expedient method of 'simply' typing in the query for the program needed. 

Firstly, to my way of thinking a couple of simple mouse clicks on a menu to access a program is simpler and easier than popping open a query box and keying in the program required. But then, that's just old-fashioned me.

Secondly, and the biggy for me, is that there are innumerable instances when I simply can't remember all the programs I have on board or what they necessarily do. The days are long gone when I used to greedily grab for every useful looking or novel app. out there, half of which I'd never use, but the simple fact is that I still load up my computer with a great many programs that get used very infrequently, as most people do. Now do I remember if it got loaded up under the developer's name, or the actual name of the ap - if I can remember it at all after 6 months or a year! If I need a specific program for a certain job, I'll typically go online and research what others have to say, grab and use it, then set it aside for future use. Odds are that down the track I'll forget which program it was that I used, and experience has shown that I'll end up having to go round clicking on everything vague in my menu to see if I can find or remember it. 8 - 10 years ago I finally came to the conclusion that the handiest thing to do was keep a very systematic and orderly Start menu on Windows, with all the folders well labelled, where I would store all the program icons. It took a lot of work to set up, and required housekeeping, but it made life infinitely easier when it came time to use the more obscure programs on board. If I had a specific program on board simply because it converted one odd little file type, that my more commonly used programs wouldn't, I'd provide a label for the sub-folder so that I'd remember. Now I can't do that with the latest versions of Ubuntu - unless I want to dial it back in style, from what I read.

I've since come up with another idea to take the place of the menus, which is to simply create a text document for Ubuntu, listing all the programs on board in the kind of order and hierarchies I'm used to, then simply keep it on the desktop for easy access. That way if I'm in doubt or need to prompt my memory, I can readily look up the program/function list, then go to Dash and punch the name of the relevant program in.

Thanks again...........Tim
