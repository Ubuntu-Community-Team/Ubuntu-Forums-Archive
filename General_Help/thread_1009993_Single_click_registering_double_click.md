### Post by JBu92 on 2008-12-13
When I click on any icon in the file browser, it opens it, as if I had double-clicked it. I don't know if it's actually registering two clicks or if it's just set to open stuff on a single click. How do I remedy this problem?

---

### Post by CatKiller on 2008-12-13
You can check to see if its registering a double-click or not with the animation in System -> Preferences -> Mouse. If it's registering the clicks properly, you can open a Nautilus window (Places -> Home Folder), go to Edit -> Preferences -> Behaviour and select "Double click to open items."
