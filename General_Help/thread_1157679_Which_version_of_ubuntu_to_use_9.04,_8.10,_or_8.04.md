### Post by Chrissss on 2009-05-13
Why install a older version and upgrade immediately to a newer version, which should soon be upgraded to a even newer one?

You should ask yourself two simple questions

 * Do you want to install Ubuntu and stick with a version for a couple of years?
 * Do you want to install Ubuntu and stay up to date with the latest version?

If you say yes to the first question, then install Ubuntu Hardy Heron and use it until April 2010 (or longer until the support for Hardy ends in April 2011).

If you say yes to the second question, then install Ubuntu Jaunty and keep up to date by upgrading every six months from ubuntu version to ubuntu version.
