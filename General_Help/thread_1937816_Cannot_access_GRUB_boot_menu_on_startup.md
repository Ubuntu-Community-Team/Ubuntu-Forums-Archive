[http://sourceforge.net/projects/bootinfoscript/](http://sourceforge.net/projects/bootinfoscript/)     

```
sudo bash ~/Desktop/bootinfoscript
```The graphic option here is nomodeset to get in if grub is broken and we fix it.

[http://ubuntuforums.org/showthread.php?t=1613132](http://ubuntuforums.org/showthread.php?t=1613132)

**So have you customized grub to have a specific OS at the top?** If you have installed the startup manager, it is not supported and does not move the boot with a kernel upgrade. The grub customizer is a gui tool for this and works but must be carefully used.

### Post by Lyfang on 2012-07-02
1. Create a persistent Linux Live USB flash drive
[http://www.pendrivelinux.com/what-is-persistent-linux/](http://www.pendrivelinux.com/what-is-persistent-linux/)

2. Boot the persistent Linux Live USB flash drive

3. Run from Terminal:
sudo update-grub

4. Reboot & select Ubuntu where you "try to access the grub"

5. Install GRUB 2 from a package manager or install from Terminal (where you "try to access the grub" drive):
sudo apt-get install grub-pc

6. Unmount & remove the persistent Linux Live USB flash drive

7. Reboot (to GRUB 2)
