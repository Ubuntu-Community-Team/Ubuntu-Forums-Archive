### Post by luismgl on 2010-06-13
I've downloaded from the Ubuntu software center a touch-pad application that allows me to set some preferences to my touch-pad. So since I rather use a mouse, I've disabled it, the problem is, it only keeps disabled for a few minutes, later it works again. So I want to keep disabled until I enable it again, any suggestions?

---

### Post by Dave_gb on 2010-06-13
For mine I downloaded gsynaptic via the package manager.

Once installed start it from the system/preferences/touchpad menu and tick the disable touchpad box.

Then make sure in the mouse settings (system/preferences/mouse) that the *disable whilst typing* is NOT checked.

Works a treat on mine.

Dave
