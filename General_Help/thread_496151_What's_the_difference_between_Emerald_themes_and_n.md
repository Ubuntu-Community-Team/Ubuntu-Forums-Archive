### Post by Vajra Vrtti on 2007-07-08
Emerald is the window decorator used by Beryl. You can only use Emerald themes if you have Beryl installed.

Yes. Even using Emerald, some elements are still inherited from Metacity (Gnome window decorator). For instance, the icon set to be used. Is that what you mean?

### Post by Vajra Vrtti on 2007-07-08
I was not correct when I said Metacity is a window decorator. Metacity is a window manager, that uses the GTK window decorator. Heliodor "is a gnome based window decorator that can be used with Beryl", according to its own description. I think with Heliodor Beryl can use the same themes as Metacity, but I don't have it. Emerald themes have more sophisticated visual effects, like translucent borders.
