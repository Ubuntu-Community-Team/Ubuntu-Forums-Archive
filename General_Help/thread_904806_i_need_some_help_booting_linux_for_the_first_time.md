The only difference is the Graphical User Interface. The Server edition by default doesn't have the GUI installed, because administration is usually done at the command line. If you installed the server edition than the terminal console (black screen) looks about right. 
The Desktop Edition will come with a GUI where you can log into a graphical environment with a start menu, applications, and all good stuff. 
Check the ISO you downloaded and if it's the server edition and not what you wanted than try the desktop edition.
