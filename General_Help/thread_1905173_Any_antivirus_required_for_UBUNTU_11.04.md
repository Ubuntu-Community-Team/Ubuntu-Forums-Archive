### Post by Grenage on 2012-01-06
> **turbo71113 said:**
> 1.dont we have to install any antivirus in linux?
Please suggest a good one if so.

Any AV products you might use will likely just be checking for Windows viruses.  There aren't any Linux viruses in the wild, so you don't really need to bother.

What you do still need to bother about is general good practice.  For example, using secure passwords, trusting programs/code before you run it, and only using root rights where necessary.
