### Post by holiday on 2010-10-17
After the install CD loads, you are given the choice between installing or trying Ubuntu. Take the Try Ubuntu option. It will be slow loading applications because they have to load from the CD, but otherwise it will give you a taste of Ubuntu. 

And if you do install it, it will run much faster than 7. Don't judge Ubuntu by the performance of the Try Ubuntu option.

To lose Ubuntu, just remove the CD and reboot your machine and your Windows system won't be any the wiser.

### Post by oldos2er on 2010-10-17
I'm not a programmer, and I enjoy using Ubuntu, but I didn't come from a Windows background, so I had no real preconceived ideas about Linux' "user-friendliness" and other imponderables. Maybe give [http://linux.oneandoneis2.org/LNW.htm](http://linux.oneandoneis2.org/LNW.htm) a read and see if any of that might apply to you.

Windows programs do not run natively on Linux; however there is some software called Wine that allows a few Win* apps to run under Linux. Check [http://appdb.winehq.org/](http://appdb.winehq.org/) to see what will and what won't work.
