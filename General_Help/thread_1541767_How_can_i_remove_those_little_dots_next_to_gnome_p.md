### Post by WorMzy on 2010-07-29
You may need to be a bit more specific; perhaps a screenshot with the dots circled would help.


I think I know what you mean though. There's some little dots next to some applets which are used to access the generic applet actions, since the right-click context menu for the applet is used for more specific actions (For example, the Window List applet has these dots, since right-clicking on the actual content of the Window List brings up the Minimize-Maximise-Move-etc options for the window in question)


If that's what you mean, then there's no way to get rid of them. You can make them invisible, but they'll still take up the same amount of space, and it's a little confusing how you achieve that result.

[http://pclinuxos2007.blogspot.com/2009/06/how-do-i-change-gnome-panel-handle.html](http://pclinuxos2007.blogspot.com/2009/06/how-do-i-change-gnome-panel-handle.html)

### Post by Frogs Hair on 2010-07-29
The notification area is marked by dots , if you want to remove it right click directly on the dots and select remove from panel . If want put it back right click the panel > add to panel add notification area.
