### Post by Warren Hill on 2013-02-27
Ubuntu 12.04 is faster than 12.10 on a lot of machines.  Partly because Unity 2D has is no longer available.

One workaround is to use the Gnome Classic desktop instead of unity.  It is similar to the desktop as it was in 10.04 though not identical.  If you want to try it then in a terminal enter

```
sudo apt-get install gnome-panel
```

Log out then back in selecting "Gnome Classic" or "Gnome Classic (No Effects)".

If you don't like it you can go back to Unity any time.
