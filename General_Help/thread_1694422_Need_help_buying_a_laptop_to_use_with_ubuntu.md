### Post by fragos on 2011-02-24
Laptops with Nvida or Intel Graphics work well with Ubuntu. WiFi can be a problem to get running depending on the chipset in the WiFi interface. Intel WiFi works out of the box. SD card slots can be hard to get working because laptops frequently use proprietary chipsets instead of the USB interface found on Desktop internal readers. A while back I purchased a Dell 1420n which came with Ubuntu installed and all interfaces working.
