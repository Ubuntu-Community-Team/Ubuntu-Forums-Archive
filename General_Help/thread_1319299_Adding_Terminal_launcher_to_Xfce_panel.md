### Post by epp on 2009-11-08
Is there a way of adding the Terminal launcher to the top panel?  I was able to do this with a different distro in the past that also included Xfce, but with xubuntu 9.10, the Terminal item is not listed under Add New Items when the panel is right-clicked.  Could there be an additional package that must be installed?

Thank you in advance for any info.

### Post by winotree on 2009-11-08
You don't need to add any other programs -- try this, it'll be worth it!  :)
[I]
right-click[/I] Panel | *click* Add New Items | *click* Launcher | *click* Add | *name* Launcher | *click *Icon | *choose and add *Icon [or *click* Menu Icons | *click* All Icons | *choose and add more appropriate *Icon]

You should be returned to the Launcher pop-up so

*click* Folder Icon [to right of Command -- unless you know command, then insert in space | *click* OK] and you're done, otherwise

*click* usr | *click* Open | *click* bin | *click* Open | *scroll down to and click* Xfce4-terminal | *click* Open | *click* OK

And that should be that -- I don't think I've missed any steps but if I did please tell me.  ;)

### Post by epp on 2009-11-08
Launcher was already listed after selecting Add New Items, so I did not need to click Add after this.  

Instead of naming it *Launcher*, I re-named it *Terminal*, selected the Terminal icon (which was shown for Application Icons) and the application (which was in /usr/bin/) and it worked.  

By default, it added the icon to the far right on the panel next to the network icon, I simply right-clicked, selected Move and moved it over to the left next to the blue Help icon.

Thanks.  :D
