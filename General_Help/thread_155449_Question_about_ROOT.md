### Post by localzuk on 2006-04-05
I do not quite understand your question. Do you wish to login to a graphical interface as root? If so, this is inadvisable.

By default the root account in Ubuntu is disabled and everything is done via sudo by users who are a part of the admin group.

If you really want to enable the root account, take a look at [https://wiki.ubuntu.com/RootSudo](https://wiki.ubuntu.com/RootSudo) for some information.
