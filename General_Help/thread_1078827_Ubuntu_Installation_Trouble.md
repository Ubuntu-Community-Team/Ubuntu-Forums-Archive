### Post by mb_webguy on 2009-02-23
It shouldn't take 3 days to get your wireless card working.  It might take 3 hours, but if you follow [these instructions]("https://help.ubuntu.com/community/WifiDocs/Driver/Ndiswrapper") and [this troubleshooting guide]("http://ubuntuforums.org/showthread.php?t=885847"), then it's more like 30 minutes or less.  And that's only if your wireless card isn't supported out of the box.

As for the rest, I'm not sure what you're talking about with sliders.  Unless I've missed something the half-dozen times I've installed Ubuntu on my several computers, if you select a manual install (which you have to do for a dual boot), then you have to select to edit a partition (or create a new one) and enter the size in a text box with increment/decrement buttons on the side.
