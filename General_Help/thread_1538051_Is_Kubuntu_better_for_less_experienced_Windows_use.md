### Post by Zorgoth on 2010-07-24
I think that KDE4 right now is pretty buggy personally.  I tried to switch to Kubuntu recently because I want to find an environment that is going to keep supporting compiz after GNOME 3.0 comes out, but I didn't do well and switched back quickly.  However, I was trying t ouse compiz and had some weird system issues like kwin not recognizing my graphics card.

I would recommend installing kubuntu and trying it out if I were you.  It definitely has its good features that I wish GNOME had as well.  I would definitely put GNOME first for an advanced user, and for a low-end user it probably depends on the user.  I would consider them both pretty intuitive, but KDE does have a few advantages, for example the System Settings will be an easier migration from people familiar with a Windows control panel and the KDE applications menu is a big improvement over the Windows start menu without being altogether dissimilar.
