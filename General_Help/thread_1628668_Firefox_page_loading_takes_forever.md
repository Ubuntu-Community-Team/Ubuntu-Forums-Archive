### Post by lithopsian on 2011-03-14
You might be better off asking questions about Firefox at a Firefox forum.

Since this problem is not widespread among Firefox users, I have two suggestions.

1. Start in safe mode and disable any extensions.  90% of "weird" issues with Firefox are due to buggy or clashing extensions.

2. Try Firefox 4.  Most things are faster than Firefox 3.6, although if you are having issues with something like DNS it might not make any difference.
