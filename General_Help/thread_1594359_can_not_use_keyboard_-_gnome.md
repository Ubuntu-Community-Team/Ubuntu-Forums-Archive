### Post by 3Miro on 2010-10-12
You mouse and keyboard can get stuck inside virtual box. Try hitting the right control key (Ctrl on the right side). It should change the focus back to Ubuntu.

If this doesn't work:

try going to System -> Admin or Preferences -> Session or Startup apps (I can never remember which one it was). Then deactivate everything "keyring" or "security" or "encryption" related. Then reboot the computer.
