```

```[B]

Sample output:[/B]
```

karthick@Ubuntu-desktop:~/Desktop/PERL$ lsb_release -a
No LSB modules are available.
Distributor ID:    Ubuntu
Description:    Ubuntu 10.04.1 LTS
Release:    10.04
Codename:    lucid

```
