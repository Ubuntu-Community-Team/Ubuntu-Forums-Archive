### Post by frisket on 2011-07-02
Don't use EndNote under WINE, it breaks (and EndNote is huge overkill for most authorial purposes anyway).

Use JabRef instead: it's designed to work with LaTeX and BIBTeX, and it can import and export files in RIS format that you can use with EndNote on Windows systems.
