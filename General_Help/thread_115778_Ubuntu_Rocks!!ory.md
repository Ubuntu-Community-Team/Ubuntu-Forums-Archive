### Post by Azriphale on 2006-01-11
About you synaptic touchpad problem, have a look at these resources:

[https://wiki.ubuntu.com/SynapticsTouchpadHowto](https://wiki.ubuntu.com/SynapticsTouchpadHowto)
[http://wiki.debian.org/SynapticsTouchpad](http://wiki.debian.org/SynapticsTouchpad)
[http://tuxmobil.org/touchpad_driver.html](http://tuxmobil.org/touchpad_driver.html)

the "XF86config-4" file that they refer to is now "/etc/X11/xorg.conf", as Ubuntu changed from xfree86 X server to the Xorg X server in Hoary.

If you are unsure about what exactly you should do, or you are worried about breaking something, post back here and i'll give you a hand.
