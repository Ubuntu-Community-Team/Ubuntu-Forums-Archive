### Post by spirospr on 2004-11-17
In gnome go to networking and then in general i think you have to enable windows networking. If there is a pop up message telling you that there is a probem then you have not installed samba. If so go to synaptic search for samba and install it. After try again and you will not have a problem.

In case that you have allready done all these then there is problem with your settings. Take after that the settings in windows networking is the same as in windows. For examble you should have the same domain name as windows etc.
