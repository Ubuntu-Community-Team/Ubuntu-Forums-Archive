### Post by Maheriano on 2009-07-09
[www.mp3car.com](www.mp3car.com) - forum for people putting computers in cars, boats, everything

[www.openice.org](www.openice.org) - A project by a guy named Kevin that provides a front end running on Linux (works in Ubuntu) with all media buttons mapped on the backend to their applications. It runs videos, music, GPS, third party applications and everything else you'd want in a mobile computing solution.

navit.sourceforge.net - Open source Linux GPS mapping application which can be embedded into nGhost (link above). Or it can run on its own.

I currently run a computer in my car to a 7 inch touchscreen in the dash and use Road Runner as a front end with Windows XP on the backend and iGuidance for GPS but I'll be switching over to nGhost and Navit as soon as I get them compiled and running on my machine at home first.
