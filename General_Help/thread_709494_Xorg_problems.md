### Post by Sam Lars on 2008-02-27

        Identifier      "nVidia Corporation NV34 [GeForce FX 5200]"

        Identifier      "Default Screen"

        Option          "metamodes"  "1280x1024"

### Post by Sam Lars on 2008-02-27
Identifier doesn't matter.  I guess they should be the same, but I don't think it matters.
For the ID:
```
cat /var/log/Xorg.0.log | grep PCI:
```
You'll find a line like this:
```
(--) PCI:*(0:9:0) nVidia Corporation NV34 [GeForce FX 5200] rev 161, Mem @ 0xdf000000/24, 0xc0000000/28, BIOS @ 0xdefe0000/17
```
That's the 0:9:0, and the * means it's been selected for use.  Find the one that you want and use that ID.
