### Post by Tim Sharitt on 2008-07-21
To set Firefox as the default browser got System > Preferences > Preferred Applications from the menu bar. Firefox should be on the drop-down list or if it's no there is a place to enter the command manually (usually required if you have downloaded firefox directly from mozilla instead of the repos).
