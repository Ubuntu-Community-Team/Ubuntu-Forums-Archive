### Post by alastair on 2005-03-11
Ubuntu's designed for a 2.6 kernel - so lots of the base system setup is designed for 2.6. Not worth trying - unless you really know what you're doing - even then, I'd forget it. 

It /is/ possible to dual boot 2.4 and 2.6 - but you'll need to know what you're doing. If you need 2.4, best to use a distribution that goes with 2.4.
