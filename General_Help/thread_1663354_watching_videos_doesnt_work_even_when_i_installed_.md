### Post by lovinglinux on 2011-01-09
First, get [Flash-Aid]("https://addons.mozilla.org/en-US/firefox/addon/161939/") and run it. It will detect installed flash plugins, remove them and install the best option according to your system architecture and version.

If that doesn't help, delete Firefox cookies, cache and the ~/.macromedia folder.

If that doesn't help either, go to Flash-Aid Help tab, click "Generate Report" and send me the results.
