### Post by ivarn on 2010-11-03
Sure, system > pref > keyboard shortcuts.
You will find programs and functions you can add shortcuts to, and if you need to add more shortcuts, click on the add button and add the software's name, command and your favorite shortcut for the program.
EDIT: Or do you mean program launcher?
Right click > create launcher (or whatever it says).
