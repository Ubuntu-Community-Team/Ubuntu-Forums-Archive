### Post by Basher101 on 2011-08-10
If you can hear it and it doesn't show up, then the process is running but not the icon on the panel. Just open your system monitor (or type gnome-system-monitor in terminal), go to proccesses and kill skype proccess. Then restart it by launching it from applications.
