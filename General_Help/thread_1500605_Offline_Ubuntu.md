### Post by tommcd on 2010-06-03
> **pk_m said:**
> I bought new PC, I installed Ubuntu 10.04 then installed updates.

Will there be any problem for not installing regular updates? Because I will not using internet.
Since most of the updates for Ubuntu are security updates, especially for an LTS (long term support) version of Ubuntu like 10.04, there is no risk to not updating Ubuntu if you are not connected to the internet.
Some of the updates will be bug fixes also, which you may need or want.
> **pk_m said:**
> 
Can I update Ubuntu offline?
Yes you can!
You can use **keryx**:
[http://keryxproject.org/](http://keryxproject.org/)
[https://help.ubuntu.com/community/InstallingSoftware#Installing%20packages%20without%20an%20Internet%20connection](https://help.ubuntu.com/community/InstallingSoftware#Installing%20packages%20without%20an%20Internet%20connection)
And there is also **APTonCD**:
[http://aptoncd.sourceforge.net/](http://aptoncd.sourceforge.net/)
I have not actually used either of those, since I am on the internet. From what I have read though, I would try keryx first, since it has been reported to work well.

Yes Startpage is an alternate to Google *if* you value your increasingly threatened right to privacy. There is a firefox addon for Startpage also.

[http://startpage.com/eng/protect-privacy.html](http://startpage.com/eng/protect-privacy.html)
