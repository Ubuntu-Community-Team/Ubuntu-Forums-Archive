### Post by Devport on 2008-07-07
I think there is nothing to do this on linux natively - it may be possible by using wine but probably not. If you have a Windows license it is possible to do it using virtualisation ( Virtualbox / VMWare ). If your phone has an built in update service ( e.g. my K800i has this ) you could update the firmware from within the phone itself ( never tried that so I dont know if that does the update itself or not ). Using bluez you can do it over the broadband connection of your PC via bluetooth.

These are only some hints what you may want to look for.
