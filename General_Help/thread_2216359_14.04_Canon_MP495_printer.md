### Post by pdc on 2014-04-11
Canon supply both a printer driver and a scanner driver; if I point you to both directly, and offer suggestions as to how to install?

If you go here [http://support-asia.canon-asia.com/contents/ASIA/EN/0100301502.html](http://support-asia.canon-asia.com/contents/ASIA/EN/0100301502.html) you will download [COLOR="#008000"]cnijfilter-mp495series-3.40-1-deb.tar.gz[/COLOR]  and if you choose the SAVE option and your default save place should be the Downloads directory

If you are able to open a terminal; paste in commands (I suggest using the mouse and top line of the terminal.....menu at top left........go along to Edit and then down to paste ..and hit enter after paste)

the commands would be

> [COLOR="#FF0000"]cd Downloads[/COLOR]

> [COLOR="#FF0000"]tar -zxvf cnijfilter-mp495series-3.40-1-deb.tar.gz[/COLOR]

> [COLOR="#FF0000"]cd cnijfilter-mp495series-3.40-1-deb[/COLOR]

> [COLOR="#FF0000"]./install.sh[/COLOR]

_____________

that should get the printer going; Canon provide [COLOR="#0000FF"]ScanGearMP[/COLOR] to run the scanner; get it from here [http://support-asia.canon-asia.com/contents/ASIA/EN/0100302802.html](http://support-asia.canon-asia.com/contents/ASIA/EN/0100302802.html) and it comes down as [COLOR="#008000"]scangearmp-mp495series-1.60-1-deb.tar.gz[/COLOR]

close the terminal; open it anew; commands are:

> [COLOR="#FF0000"]cd Downloads[/COLOR]

> [COLOR="#FF0000"]tar -zxvf scangearmp-mp495series-1.60-1-deb.tar.gz[/COLOR]

> [COLOR="#FF0000"]cd scangearmp-mp495series-1.60-1-deb[/COLOR]

> [COLOR="#FF0000"]./install.sh[/COLOR]

to run [COLOR="#0000FF"]ScanGearMP[/COLOR], the command from the terminal is > [COLOR="#FF0000"]scangearmp[/COLOR] and if you create a launcher, you can launch it each time: 

I hope you don't mind me saying best to follow the commands exactly as pasted above; I hope this all goes well
