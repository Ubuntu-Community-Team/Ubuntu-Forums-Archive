### Post by coffeecat on 2010-09-08
> **vorbb said:**
> I'm in university and really don't wanna have to check all my documents I make under ubuntu in Office for formatting errors, etc.

One thing to beware of - page breaks. If you save a .doc.file in one word-processor and open it in another, you often find the page breaks come at different places. A couple of years ago I found that a document I opened in OpenOffice in Ubuntu, Gentoo and another Linux distro showed up with different page breaks. And this was on the same (multibooting) machine with the same version of OO on each distro with each distro set up for the same printer. Really frustrating. I had the same problem with the same version of Microsoft Word on different machines - but that was some years ago.

That's why, if you're submitting a document which is not going to be further edited, it's a good idea to export to PDF. At least they see what you see.

### Post by rjs987 on 2011-01-02
Just a thought after reading the op again...

Vorbb, are you looking for an Office replacement/equivalent for your ANDROID system or for your Linux system, or both?

I know that Android is a Java framework running on a Linux foundation. But that doesn't mean that any Linux program will work on it.

I've been running only Linux on my main computer at home now for almost 3 years using OpenOffice.org for all my text and spreadsheet document needs. Works very well. Found that OOo works with all my files from work and my wife's files from her work just fine. At my work it is MS Office 2007 or MS Office 2010 on Windows 7 and my wife has MS Office 2008(?) on a MacBook Pro with osX. We take our files back and forth all the time with no problems. The only issue I've found is that OOo has a tab (or worksheet) limit of 256 worksheets in a spreadsheet document and MS Office Excel is only limited by memory available. We have ONE spreadsheet file at work with 260 worksteets and my OOo just truncates those last 4 but otherwise opens the file just fine.

All this on my Linux system. I also just picked up an Android tablet (Archos 70) that I really like but I am still looking for a good text and spreadsheet viewer for Android. An editor would be best. OOo does not work on Android. There is one Open Document Spreadsheet format program that will allow you to edit ods files but not text documents in odt format, don't remember the name and it has some issues anyway. There is Documents To Go that will work with any MS Office file and may work for you on your Android. This is not a free program and since I use primarily ODT and ODS formats this won't work for me anyway.

So you may find that OpenOffice.org for your Linux machine and Documents To Go for your Android device may be the way to go if you stick to the Doc and Docx formats for text documents, and xls and xlsx formats for your spreadsheet documents.
