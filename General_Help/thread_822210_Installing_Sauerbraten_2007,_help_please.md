### Post by cariboo on 2008-06-08
Sauerbraten is in the repositories, you can use synaptic to install it.
Go to System-->Administration-->Synaptic Package Manager. Click the search button and enter sauerbraten. There is also a free map file called sauerbraten-wake6 mark it for installation too. To mark the files for installation right click on the filename. Synaptic will also put it in the games menu for you.
