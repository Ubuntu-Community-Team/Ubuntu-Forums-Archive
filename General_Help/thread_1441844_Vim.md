### Post by Simian Man on 2010-03-29
To learn vim, open up a terminal and enter:
```

vimtutor

```

This will give you a tutorial on using it.  It is very in depth, so don't feel you have to know everything in there to start using vim.  Nobody (well maybe Bram Moolenaar) knows *all* of vim.  Most users just know different subsets and add to what they know over time, so don't get discouraged.

As for your actual question, I have heard of this editor called "Cream" which is based on vim, but configured to be more what most users expect from a text editor (like Ctrl-C to copy and such).  I really don't know if it will help you get used to vim or just teach you bad habits that will be difficult to unlearn if you try to learn the more efficient, standard version of vim.
