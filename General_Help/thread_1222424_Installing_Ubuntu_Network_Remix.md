### Post by mikewhatever on 2009-07-25
You can do either or, plus the usb. What's important is that you'll probably need a usb stick for the installation, since most netbooks don't have cdroms. You don't have to install to usb, but, simply put, 'place' the image file there and boot from it. Once booted, you get the live cd environment, and it's your choice to install as a dual boot or a single OS.
