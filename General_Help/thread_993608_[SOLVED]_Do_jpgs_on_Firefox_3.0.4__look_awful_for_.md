### Post by jespdj on 2008-11-28
They look normal here.

Did you maybe zoom in? In Firefox 3, when you make the text larger, it not only makes the font larger, but also scales up images and everything else on the page. It might look bad if you zoom in images like that.

Try setting the zoom factor back to normal: View / Zoom / Reset (or press Ctrl + 0).
