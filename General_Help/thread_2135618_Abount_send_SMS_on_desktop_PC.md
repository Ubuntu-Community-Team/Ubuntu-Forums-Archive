I use Skype to send a big SMS (or the copy and paste from the computer). It is not free, but costs about the same as to send a single SMS via the mobile phone.

You can install Skype easily, when you activate the Ubuntu Partner repository
```
sudo apt-get install skype
```
Or from the ***Software Center***
