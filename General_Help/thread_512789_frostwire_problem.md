### Post by rovernaut on 2007-08-01
I had the same problem a while back when I installed frostwire with automatix, the problem was with Java.
If I remember correctly I uninstalled frostwire and Java, then installed the latest Java release, then reinstalled Frostwire.
I think my problem was that I installed Frostwire first then Java instead of Java first then frostwire.
