> **borabora1 said:**
> Also when trying to get java is not possible to go through but in software center, openJDK java webstart, and 6 runtime is there.

Try enabling the Partner repository in Synaptic Package Manager, press the reload button, and then install sun-java6-jre and sun-java6-plugin.

Then try installing Frostwire.
