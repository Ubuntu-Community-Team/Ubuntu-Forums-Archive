### Post by Ramses de Norre on 2007-01-06
Ubuntu uses gnome by default, kubuntu kde and xubuntu xfce (those are all desktop environments). Edubuntu is a ubuntu version especially for educational use.

There is no difference between all those except for which packages are installed by default, and you can turn each of them in another one by just installing and uninstalling some packages.

If you've got kubuntu, you can get for example the whole xubuntu setup by installing xubuntu-desktop.

The one that's best is up to you to decide, try them out and choose which one you like the most.

From what I understand:

Ubuntu is the Ubuntu distro using the gnome desktop and gnome/GTK apps.
KUbuntu is the Ubuntu distro using the KDE desktop and KDE apps.
XUbuntu is the Ubuntu distro using the XFCE desktop and XFCE apps.
I'm not sure what Edubuntu is.

Of course, you can install and use gnome/KDE/XFCE apps inside the gnome desktop environment, and vice-versa. The only difference will be that gnome will have to load KDE libs for KDE apps while those KDE apps run in gnome.. same goes for XFCE.

### Post by Henry Rayker on 2007-01-06
Ubuntu is the OS with Gnome.
Kubuntu is the OS with KDE.
Xubuntu is the OS with XFCE.
Fluxbuntu is the OS with Fluxbox.
Ebuntu (in progress) is the OS with E17.

Edubuntu is Ubuntu (Gnome) with some educational software installed by default.
Ubuntu CE is Ubuntu with some christian related software installed by default.
A couple others...usually based on Ubuntu but with software geared toward a certain group installed.

Basically, the only difference is the desktop environment. The first group I posted were all DE differences...the second group are specialized (with extra apps installed or some removed) for certain groups. The support for Kubuntu is, as far as I've seen, about the same as for Ubuntu...most help is given for the terminal and, as such, the help will work regardless of DE. The only time this differs is when help with GUI frontends is requested.

You can installed the Kubuntu desktop package onto Ubuntu (or vice versa...or any DE/Wm (window manager) on any of the installations) so that you have a choice at log-in. Currently, I have Ubuntu installed with E17, E16, Fluxbox, XFCE and the default Gnome installed. Given that you're coming from Mandriva (They use the KDE, if I remember correctly) you should probably stick to Kubuntu, if you like that DE. You could always install the others (to see what suits you best) so that you can switch at log-in, though.
