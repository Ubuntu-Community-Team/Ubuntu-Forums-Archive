### Post by darolu on 2010-05-14
There are some differences, the main one is the way it looks, KDE is more Windows-like in my opinion; you'll get the idea if you search for screenshots of it, the default configuration includes a thick panel on the bottom only, icons are larger and is blue-ish.

It includes different default applications too, the files manager is different, it doesn't use Nautilus but Dolphin (in the newer versions), it used to use Konqueror which is also the default web-browser, both are fine file managers, they are more "graphical", by default they work with one-click only to open files, etc... Konqueror is a nice browser and it was the base for Webkit, the engine that renders the web in browsers like Safari, Chrome and Epiphany. Other examples are the text editor, KDE uses Kate instead of gEdit, Okular replaces evince, Amarok instead of Rhythmbox, etc...

Another important component are the QT libraries that (at certain level) work as GTK in KDE.

You can of course use KDE apps in GNOME and viceversa, they are after all mere desktop environments.

But you won't know until you try it for yoursefl, install "kubuntu-desktop" package so you can feel the whole KDE experience, you select which one (gnome or kde) you want to use when you log in.
