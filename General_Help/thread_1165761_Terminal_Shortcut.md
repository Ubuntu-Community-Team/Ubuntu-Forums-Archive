### Post by issih on 2009-05-21
Actually, you can set that kind of thing.

Assuming you are running compiz the best way I found was to configure it in there (although I think you can do the equivalent for metacity in gconf)

Anyway, open compiz config settings manager, go to the general plugin, then the commands tab.

On that tab:

Expand the commands section and enter the terminal command you want to run in one of the 11 available slots, then expand the key bindings section and set the shortcut for the same slot as your command.

Done.

Hope that helps
