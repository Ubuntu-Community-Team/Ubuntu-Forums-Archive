### Post by TBABill on 2010-11-15
I agree, pretty rough in Linux versus Windows. It's because there is no hardware acceleration in Linux so it just heats everything up making the CPU do most of the work. I have noticed marked improvement with Flash 10.1 on my 64 bit install (and 64 bit flash).
 
One issue that gets lots of people is ubuntu-restricted-extras. You end up with open source java and 32 bit flash. So if you don't take steps to change those two you end up with sub par java performance and 32 bit flash in a 64 bit install if you are not on 32 bit Ubuntu. The 64 bit flash just runs so much better for me.
 
I do agree...Flash is just a poor Linux performer but improving rapidly lately.
