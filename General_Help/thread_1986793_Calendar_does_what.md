### Post by ubuntu27 on 2012-05-25
I don't know about Ubuntu 12.04, but in previous version you could schedule in Evolution mail client, and it will show up on the Calendar.


I like to use [Nitro Task]("http://nitrotasks.com/") for personal appointments and reminders. Give it a chance ;-)

To install Nitro:

```
sudo add-apt-repository ppa:cooperjona/nitrotasks
```
```
sudo apt-get update && sudo apt-get install nitrotask
```
