**A Wubi install does not require this.**



If for any reason you do need to (at some time in the future) need to burn an Ubuntu .iso to disc the image is designed to fit on a CD - i.e under 700mb. If you for some reason decide to burn Ubuntu Ultimate Edition to disc you'll need a DVD as it's almost 4 gb.

### Post by Rubi1200 on 2011-03-19

>  **A Wubi install does not require this.**Not entirely true. Although not a requirement, there are times when installing Wubi from the CD is recommended or even necessary. For example, the only way right now to get the 10.04.2 version is on the Ubuntu Desktop CD ISO or via this link:
 [http://people.canonical.com/~evand/w.../wubi-r191.exe]("http://people.canonical.com/%7Eevand/wubi/lucid/wubi-r191.exe")

Another very good reason to have the ISO burned to CD has to do with the fact that GRUB updates have been known to break Wubi installs. Currently, the only way to fix this is from a LiveCD.

And, just to clarify something else. I was very careful in my choice of words. I did not say that installing Wubi from a CD is a requirement. I said that IF one chooses to do so, one needs to burn the image to CD not DVD.
