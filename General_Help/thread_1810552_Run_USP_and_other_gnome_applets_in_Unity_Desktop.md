### Post by Malac on 2011-07-23
A few people have asked me if it is possible to run USP under Unity.
I have found this method by Alaukik on askubuntu.com and hope they don't mind me reproducing it here.

Login to Unity Desktop

then 

[LIST=1]
[*]Run gnome-panel in the terminal (Don't close the terminal until finished with the instructions)
[*]A GNOME Classic desktop will appear over unity
[*]Remove the top panel so we can use the Unity panel
[*]Unlock all the applets from the bottom panel (so you can remove any  things you don't need 'cause the unity launcher already do some of  these things).
[*]Remove everything you don't need from the panel.
[*]With the bottom gnome-panel empty add your applets. e.g USP
[*]If you like to have the bottom bar extended over the launcher, you can stop here. If you don't continue the steps.
[*]Right click over the gnome-panel then choose properties.
[*]Uncheck the 'Expand' box
[*]Add gnome-panel to startup applications.
[/LIST]

Hope this helps.
