### Post by jocko on 2010-05-30
> **formaldehyde_spoon said:**
> ... but when a partition is mounted with the 'user' option, every file and directory on that partition is listed as belonging to the current user.
The "user" or "nouser" options in fstab does NOT change ownership of files in file systems that support real file permissions (such as ext3, ext4 and any other linux file system). In file systems that does **not** support permissions (fat, ntfs), ownership is set at mount time according to options in fstab and permissions on the mount point.
