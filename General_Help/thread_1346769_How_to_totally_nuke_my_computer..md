### Post by bunk3rk1ng on 2009-12-05

Thanks for the help.
 
First though you have to tell the BIOS to boot from the CD drive.
 
Restart you computer and it will usually say, "Press [key] to enter setup". Usually the key will be F5 or del.
 
In the setup you will want to look for boot options or boot priority. Usually the computer will boot from the hard drive first. Change the boot priority so that it boots from the CD rom drive first.
 
Now when you reboot your computer with the CD already in the drive, instead of starting your OS it will ask you to press any key to boot from the CD. Hit any key and follow the instructions to install XP. It will give you the option to delete partitions etc.
 
Good luck.

### Post by oldsoundguy on 2009-12-05
Boot to the Windows cd.  You can't do it from Linux .. now, mind you, all you will have on the drive when you are done is Windows.

For the average Windows user, here are the instructions:
Put your Windows CD in the drive (not a recovery CD)
Turn off the computer
Turn the computer back on.
it SHOULD boot to the CD
IF it does not, shut down again.
Turn the computer back on and get into what is called BIOS .. This is done by USUALLY hitting the del key several times during boot up on MOST computers, but yours may be different so you will have to find out which key to use .. that is usually in the user's manual .. that thing you lost months ago or you used to line the bird cage.
Search in bios for BOOT ORDER
using page up and page down keys, alter the sequence until it reads: CD FIRST.
hit f10 and say yes and let it boot.
Windows will take you by the hand and let you install on the drive but it will stop several times to ask some really stupid questions that it can not figure out itself .. and then go on line and download and install the several hundred patches and updates and patches on the updates. (pack your lunch, it will take a lot of time.)
Then go get all your updated hardware drivers from the various manufacturer's sites since the drivers  that come with Windows are never the latest drivers.
Install all of your Anti-Virus, cleanup and spyware protection programs and update all of them.
Then install all of your programs that do not come WITH Windows that you need in order to function.
Update all of your programs.

Then run a cleaner program to remove all the excess files left behind from installing.

Then run scandisk and then de-fragment the drive as it is really messed up from installing stuff.
