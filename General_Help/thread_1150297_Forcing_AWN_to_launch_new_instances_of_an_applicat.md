### Post by Trekky0623 on 2009-05-06
Is there any way to make the Avant Window Navigator always create a new instance for an application?  Currently, as I have heard before, when I use a launcher with the same name as the window that comes up, that launcher becomes the instance down in the dock.  However, I don't want that.  I want it so that every time I open up an application, a new instance is put onto the dock.  Is there anyway to make this happen?

---

### Post by CatKiller on 2009-05-06
If you've already launched an application from AWN, middle-clicking on the launcher will open a new instance.
