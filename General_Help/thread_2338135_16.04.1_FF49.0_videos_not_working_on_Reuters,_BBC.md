### Post by Kris_M on 2016-09-26
yes!
On a clean system (no pipelight) from
[http://www.ubuntumaniac.com/2016/09/how-install-adobe-flash-player-23-beta.html](http://www.ubuntumaniac.com/2016/09/how-install-adobe-flash-player-23-beta.html)
I did 
```
For Ubuntu 64bit

    wget https://fpdownload.macromedia.com/pub/labs/flashruntimes/flashplayer/linux64/libflashplayer.so

    sudo sed -i "/^# deb .*partner/ s/^# //" /etc/apt/sources.list

    sudo apt-get update

    sudo apt install adobe-flashplugin

    sudo mv libflashplayer.so /usr/lib/adobe-flashplugin/

    sudo chown root:root /usr/lib/adobe-flashplugin/libflashplayer.so


After installation has completed, Restart browser For Refresh ..
```

This gets rid of 11 and installs 23, which works fine.  These instructions were what I was looking for as I had no idea how to correctly move that file into place.
