### Post by Jerry.S on 2009-08-21
Gettin a OSX look dash board is quite simple. Just do the flowing (p.s.no CL)

Use Synaptic to search for and install the  [FONT="Arial"]awn-manager[/FONT] package. (system > administration > synaptic package manager). Using the synaptic package manager will not only install the package it will also install the dependency along with it.

Once it is installed, you can start the Avant-Window-Manager by clicking its entry on the Application > Accessories menu.

To configure AWM click system > preferences > Awn Manager

To make AWM auto start each time you log in, put a check mark in the Startup behaviour box

Any Q? just ask.

JerryS
