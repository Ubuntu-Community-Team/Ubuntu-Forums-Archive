### Post by davidoloan on 2012-02-12
Here is an update to my HP CP1025 problem which I posted here :
[http://ubuntuforums.org/showthread.php?p=11683332#post11683332](http://ubuntuforums.org/showthread.php?p=11683332#post11683332)

I tried to install my HP CP1025 color again.

The HPLIP install worked, so it must have been updated.

But, black text would come out blue, and the printer would make loud grinding sounds as the mechanism worked. Text printed like bold but imperfectly. It often won't respond to the print instruction especially from other computers on the network.

So, I thought something must have gone wrong with the printer in the 3 months it was disconnected.

I reinstalled it, all same symptoms and it is just out warranty so I decided to take it to the recycling centre and buy a new one.

I reconnected my old dell, which now prints multi page documents with a gap between pages as the HP started to do after an update roughly 5 months ago. 

So I decided to get an old Windows XP computer out and install the HP CP1025 color software on it. It worked.

But here is the thing :

The printer works perfectly on Windows. It does not grind. It prints black. The print quality is good. The printer does not take a pause between every page. It can print a multi page PDF without a huge pause between every page. 

I have a been a big fan of Ubuntu up until this version. I have put up with a lot of external monitor issues before. But this time I nearly took a piece of equipment that cost over £250 with the ink to the recycling centre.

I have swapped the printer back and forth and Ubuntu is the problem.

I would no longer recommend Ubuntu to ordinary people which I would have before. The HPLIP project is one that Mark Shuttleworth uses as flagship reason for why Ubuntu is for everyone. 

This kind of problem is really basic and should not be allowed to happen. I am not talking about the technical solution.

My Dell printer works, but not well with pauses between pages. My other HP, a 4650 DN has regular wake up issues which I can most simply solve by deleting and reinstalling it. So I have 3 printers, 2 of which sort of work and one which does not work at all.

I am slowly moving to everything Apple.
