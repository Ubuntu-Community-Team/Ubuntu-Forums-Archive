### Post by QIII on 2016-06-16
The easiest method I have found by far is [here]("http://www.webupd8.org/2012/09/install-oracle-java-8-in-ubuntu-via-ppa.html").

Andrei has created what is basically a set of scripts to automate the download and install.  You just run three commands.  The beauty is that Andrei also keeps this updated so that Java is automatically updated when Oracle releases a new version.

Webupd8 is a one-man operation.  If we want to keep having access to Andrei's fine work, it might be worth dropping a few coins in his cup.
