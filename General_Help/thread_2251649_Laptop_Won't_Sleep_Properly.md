### Post by ringstaart on 2014-11-06
If the computer doesn't actually sleep, it will continue doing things while the screen is off.

To test this, try watching a video, and put it in sleep mode. If the video continues where you left off, it went in sleep mode; if the video is a lot further than it should have been, it kept playing, and only the screen was disabled.
