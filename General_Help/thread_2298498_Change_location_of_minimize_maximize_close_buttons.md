 Yeah general window decorations are built into the unity-compiz plugin nowadays.
When the compiz window decoration plugin ran things you could toggle those button at will.
(You could also run with the emerald window decorations, but that's another story; you still can run emerald in gnome flashback, fwiw)

But as for autohiding menus see:
