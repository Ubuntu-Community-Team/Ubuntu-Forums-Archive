### Post by gimcrack on 2009-11-04
If you want to get rid of Ubuntu off the computer you are selling. And have Windows XP on it instead.

Go ahead and do that. Stick the Windows XP full version installation disk inside the Ubuntu PC.

Boot of the Windows XP disk. Delete all partitions, Create partitions, Reformat the hard drive. Windows will start installing on your fresh formated drive.

It's like when you stuck Ubuntu on it. You are just reversing the roles.
