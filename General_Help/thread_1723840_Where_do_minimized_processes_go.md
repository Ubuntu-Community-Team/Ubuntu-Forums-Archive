### Post by 3Miro on 2011-04-07
A minimized window no loner appears on the screen and goes into a list of "minimized windows" that is kept by the Windows Manager.

If you cannot see your minimized windows, this means that you deleted the "window list". Right-click on the bottom panel and select "add to panel", then select "windows list". This should return things back to normal.

Note that just as easily, you can add the windows list to the top panel or any other panel.
