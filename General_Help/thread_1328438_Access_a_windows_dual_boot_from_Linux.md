### Post by Giblet5 on 2009-11-16
No.

You can try to fool virtual machine engines like vmware into booting a real Windows partition, but it will likely not work, plus, you will have to re-activate every time you reboot one way, then the other. Microsoft says that violates their EULA.

So, the short answer is no.

Most of us keep a Windows VM around to run stuff we refuse to do without, like iTunes. And we reboot into Windows to play games.
