### Post by Krysalan on 2007-11-13
It works fine with Ubuntu 7.10 there just isn't any support software for it (eg - the desktop divider, surround gaming utility etc).

I've only found this with a fresh install however.  When the Live CD boots with the TH2G connected, the screen automatically covers the left and middle screens.  I usually change the resolution to 1024x768 so that the desktop is only on the middle monitor for ease of installation.
Once the fresh installation is complete - the screen will be divided between left and middle monitors again.  It is important that the system updates are downloaded and applied before applying any restricted drivers (I use a nVidia 7900gtx).  
Once the restricted driver is activated you should be able to set the screen resolution to the ones supported by the Matrox TH2G unit.

I've managed to do this with Kubuntu too following the same procedure but haven't been able to successfully use the Matrox unit with any other Linux distros yet.  Once any of the other live cd's start they don't seem to recognise the multiple monitors and try to output the large resolution just to the middle monitor (giving an "out of range" error).

So up to now - 

Ubuntu is compatible

Kubuntu is compatible

I haven't been able to get the unit to work with...

OpenSuSe or Mandriva
