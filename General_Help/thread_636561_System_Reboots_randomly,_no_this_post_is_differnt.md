### Post by imtheguru on 2007-12-14
*Its not a heat problem, all 5 fans all working fine, and if it was a heat problem it would restart in Vista to, and it doesn't,*

Some motherboards fail to start the processor fan with some linux kernels. Since you've mentioned that the fans are running fine, then this possibility is removed.

If you can determine a pattern of the restarts (events or timelines) then one can assist with investigating it further. Till then, sorry, one cannot debug byzantine errors.
