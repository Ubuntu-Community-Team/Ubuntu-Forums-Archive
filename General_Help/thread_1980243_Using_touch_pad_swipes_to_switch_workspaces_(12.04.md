### Post by coldjeanz on 2012-05-14
I found a semi solution to this in case anyone is wondering. You have to change some settings in Compiz first in order to enable switching of workspaces through regular scrolling (will provide details if anyone is interested). Then you have to enable "Two Finger Scrolling" instead of "edge scrolling". Then if you place the cursor on the desktop you can use a two-finger swipe to switch between workspaces

Flaws:

Only works if your cursor is on the desktop, so if you have an app opened in full screen it won't work.

I prefer edge scrolling instead of two finger scrolling so that's a bummer

Still hoping for a better solution
