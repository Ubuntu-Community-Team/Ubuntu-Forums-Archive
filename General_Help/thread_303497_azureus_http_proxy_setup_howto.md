### Post by Marquis_de_Carabas on 2006-12-09
To tell Azureus to use a proxy you go to Tools->Options->Connection->Proxy Options and then input the details of the proxy server you're using. There's also some info on the [Azureus Wiki]("http://www.azureuswiki.com/index.php/UG_Options#Connection_options")
