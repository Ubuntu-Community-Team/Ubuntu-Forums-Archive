### Post by HiImTye on 2009-08-28
[LIST=1]
[*]open a video file
[*]on the left side for video, click where it says 'copy' and choose another format (I use Xvid4)
[*]do the same for audio (I use MP3)
[*]make sure output format is AVI
[*]save as..
[*]rename it if you want or just save it (it should say .avi)
[*]let it do it's magic
[/LIST]
note: if youre converting from H264 it will complain about something, hit Cancel unless you get a freeze or you get poor audio/video sync (do a test convert, let it get to about 10 minutes in and hit pause, then watch the video, if all is well, then continue, otherwise, hit cancel and start it over but don't hit cancel when it yells at you.)
