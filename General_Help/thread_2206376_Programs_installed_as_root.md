### Post by deadflowr on 2014-02-18
Don't understand what you mean, but most installation of programs are done as root.
Every time you install something from the software center, synaptic, or the command line, it asks for an admin password to install.
That gives you the right to install as root.

Now on the other foot, actually using the programs as root can be considered unadvisable.
(Except, of course those programs that are, by design, done that way)
