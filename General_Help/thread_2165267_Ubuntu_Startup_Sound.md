### Post by bapoumba on 2013-08-04
Fresh 13.04 install here (I know I'm slow), was looking for the same thing.
Please try first this works```
canberra-gtk-play --file /usr/share/sounds/ubuntu/stereo/desktop-login.ogg
```

Then paste the line in a new item in Dash > Startup Applications. I just named it Login Sound, guess you can choose what you want.
