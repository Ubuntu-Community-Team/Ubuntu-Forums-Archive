More specifically, how can I crop and save a photo in this new GIMP version.  Thanks.

### Post by Dennis N on 2018-04-17
Crop and other tools are in the toolbox window. If you don't see it, use Windows > Toolbox in the menu of the main Gimp window to bring it into focus. When you first use Gimp, the toolbox may be too small or narrow, so drag it out at the corners to a size where you can read things in the Tool Options. Size of the box is retained after that. To crop, select a rectangular area with the crop tool. You can adjust position and borders. Click in the area to complete the crop.

### Post by SeijiSensei on 2018-04-17
You can bring up the toolbox with Ctrl+B. If you double-click an icon in the toolbox you'll get its properties dialog. 

Right-clicking on the image brings up the main menu just like the menubar at the top.
