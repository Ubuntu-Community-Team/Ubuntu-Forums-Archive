### Post by BuddyThirteen on 2010-11-17
I'm a recent convert to full Ubuntu after having dual-booted for years. The only thing I really miss (other than games) is the Windows Photo Gallery. Not that it was super great at organizing, but the thing it did well was tagging. I could easily open up a photo and apply numerous tags just by typing them in.

Shotwell and F-Spot seem kinda... I don't know, clunky? Both require actual file import (as opposed to just watching a folder for pics, as I just copy straight from my camera cards) and F-Spot has a silly database that it has to build (seems like unnecessary space wasted), and Shotwell has a strange folder organization. I'd also like to be able to browse my photos without opening the managing software, and clicking through three-levels-deep date-based folders is a bit tedious if I'm just skimming through.

I got rid of F-Spot as soon as I saw that database thing (which I had to delete manually afterward), so I didn't get deep enough into the program to see about tagging.

Shotwell does have tag support, but not in an easy-to-use per-photo basis. There's a silly drag'n'drop tagging method, but that seems like more work than it's worth. I add lots of tags to each photo, so dragging every photo to its dozen-or-so tags one at a time is a pain. I'd rather just type the tags, either one at a time or comma/semi-colon separated, and apply them quickly.

Here's what I'd ideally like:
*Plug in my SD card and offload the pics (doesn't have to be automated like Windows, I'm perfectly capable of creating a folder)
*Open up the photo management software, it sees the new pics and adds them to the set (without touching the folder structure)
*I can then go through each photo and add tags in a simple and intuitive way

Is there any software that can do this? Or at least get close to this?
