I am using Ubuntu 14.04 Unity 64bit. My problem with nautilus:

I can open only one window with root rights, doing <Alt>+<F2> "gksu nautilus". If I do the same again, no new window appears. With earlier versions of Ubuntu I could open two such windows. How can I do it with Ubuntu 14.04?

### Post by steeldriver on 2015-01-03
Have you tried just doing 'File --> New window' (or Ctrl-N) from the existing instance?

---

### Post by dragonfly41 on 2015-01-03
Instead of Nautilus you could install from Software Centre a twin pane file manager such as Krusader.

Krusader > Tools > Start Root Mode Krusader .. gives root rights.
