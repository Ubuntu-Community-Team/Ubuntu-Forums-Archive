### Post by sdennie on 2008-05-15
It will depend largely on what format you save the open office file in.  Sun writes a plugin for Word that allows it to read/write ODF files ([http://www.sun.com/software/star/odf_plugin/get.jsp](http://www.sun.com/software/star/odf_plugin/get.jsp)) but, many of the other file formats that you can save as via File->Save will be compatible with Word without any other trouble.
