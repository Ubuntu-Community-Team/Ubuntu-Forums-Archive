### Post by deadflowr on 2014-07-08
[https://launchpad.net/~danielrichter2007/+archive/grub-customizer](https://launchpad.net/~danielrichter2007/+archive/grub-customizer)

ninja'd

It's still supported for 13.10, at this point.
But to be sure it's added run the
```
sudo apt-get update
```
command to make sure that it isn't causing you any errors.
After that you can either install it through the software center or through apt-get.
