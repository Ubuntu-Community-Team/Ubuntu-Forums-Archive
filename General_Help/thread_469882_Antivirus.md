### Post by tpg on 2007-06-10
I don't think an anti-virus program is necessary. I don't use one, but you might want to see what other people say about that.

You might want a firewall, although in a default Ubuntu installation, no ports are left open, so a firewall is not necessary. (If you install something like a webserver, though, and are opening up ports, you will want a firewall). The Firestarter firewall is included in the Ubuntu repositories if you want to have a look at it.

### Post by NeoLithium on 2007-06-10
Well, AntiVirus would be useful if you plan to scan emails, or files that you'll be sending to Windows Computers a great deal. Really there's nothing to worry about on the virus end with linux.  Should you want such a utility, check out ClamAV :)  

It can be installed with:  (Ensure you have all the repositories enabled, as laid out in [http://www.ubuntuguide.org](http://www.ubuntuguide.org))
```

sudo aptitude install clamav

```
