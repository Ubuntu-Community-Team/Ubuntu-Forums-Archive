### Post by nemilar on 2011-01-22
I just want to expand upon what the other posters have mentioned.

You downloaded a version that is in still in "alpha," which means it is not intended to be used by general users.  It is still being worked on by developers, and it is expected to have many, many problems (bugs) while it is being written.

Eventually there will be a "beta" release which is when users should feel free to download and *test* the software, but it is still expected to have problems.  Where as alpha software is still undergoing heavy development and will have many broken features from day-to-day, beta software is largely feature-complete, although it will certainly still have bugs.

Neither beta nor alpha software is intended to be used on a production machine.
