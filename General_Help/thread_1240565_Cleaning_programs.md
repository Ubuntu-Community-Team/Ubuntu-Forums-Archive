### Post by soapBAR2 on 2009-08-14
> **Neliel said:**
> Anybody know of any cleaning programs? In windows I used to use things to get rid of adware  and little crap that cluttered up my computer.

Generally Linux doesn't have much adware targeted at it, because its userbase is so small and setup is so diverse. Don't feel obligated to run a security program - as long as you practice common sense, you probably won't get infected (eg, running javascript from an untrusted source = bad idea). If you have programs that you installed that you don't want anymore, just

```
sudo apt-get remove yourprogramname
```

Or remove it using your GUI package manager of choice (synaptic/aptitude/whatever). Otherwise, Avira AntiVir I guess.
