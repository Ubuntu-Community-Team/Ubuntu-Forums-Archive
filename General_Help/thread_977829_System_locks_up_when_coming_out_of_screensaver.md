### Post by brown705 on 2008-11-17
> **ggtsu said:**
> I've got intel video hardware on mine. I've been leaving my screensaver off. Do lcd's even need screensavers?

*Need*?  No, I don't think LCDs or even CRTs have actually *needed* screensavers for many years.  They are more about entertainment than anything else.  If anything, you should set the display to go inactive after a set amount of time to save power, not really about the old "burn-in" effect.
