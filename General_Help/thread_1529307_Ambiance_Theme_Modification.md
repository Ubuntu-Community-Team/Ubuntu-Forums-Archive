### Post by inameiname on 2010-07-12
...And if you want to really tweak it, the settings for the buttons size, the layout, the colors, and all sorts of things are found in the 'gtkrc' file in the 'gtk-2.0' and for the window's borders, 'metacity-theme-1.xml' in the 'metacity-1' folder.

Also, if you would like to check out tons of already user-tweaked Ambiance themes, check out this site:
