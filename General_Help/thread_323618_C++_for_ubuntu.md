### Post by JayTee on 2006-12-22
Here is a good tutorial for C++. [http://www.cplusplus.com/doc/tutorial/](http://www.cplusplus.com/doc/tutorial/)
To better learn how to use G++ you can also search Google for G++ and you can type man g++ at the command line. Since G++ is a command line compiler and linker it doesn't come with an IDE by default. You can use Glade to program GUI for C++ or Python if you want to write GUI programs for the Gnome desktop. 
Anjunta is an IDE for multiple languages you can download from the repositories. 
Gedit in Gnome will add color to the source code when you save it for the first time to highlight keywords, etc. so it works fairly well as an editor. Lots of choices out there in the repos and on other sites like Sourceforge.
