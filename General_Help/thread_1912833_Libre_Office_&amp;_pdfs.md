 Yes, I've had the same problems.  The only thing so far that I've found to work is to:


[LIST=1]
[*]Open the PDF with LO, which will open it in LO Draw.
[*]Then select all and copy.
[*]Open a blank Writer file and paste.  This will give you the document as both a single large image, and individual text boxes within the image which you can edit if need be. (the same thing happens in Draw when you open the file originally, but this is the only way to get it into Writer)
[*]Then you can save the document as any format available to Writer. (Draw can only save Draw formats)
[/LIST]
Note, this will not give you a "normal" Writer document as if you had created it from scratch.  It will be more of a page-layout arrangement with everything in text boxes.
