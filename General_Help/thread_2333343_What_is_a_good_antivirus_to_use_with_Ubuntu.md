### Post by kurt18947 on 2016-08-09
If you haven't read this, it might be worth your while:

[https://help.ubuntu.com/community/Antivirus](https://help.ubuntu.com/community/Antivirus)

Notice all the references to **Windows** viruses? The vast majority of Windows malware doesn't affect *nix machines. Practice smart computing - 
[INDENT]Don't download software from the first site that comes up on a google search. Use Ubuntu's repositories or reputable 3rd party sources.

Think before granting sudo (administrative) privileges to any process requesting it.

If you must venture into the dicy corners of the internet, consider doing so with a disposable install. Live DVD/USB, Virtual Machine, separate O.S. partition. That should help to isolate your 'daily driver' from any nasties. If using a live DVD or USB without persistence, malware cannot write to the partition so a reboot should make any nasties disappear.
[/INDENT]

I keep an install that I use **only** for sensitive things like banking, health insurance, the sorts of places that are rich pickings for identity theft. No Flash, no Java here. Run an ad blocker & NoScript.  I feel the likelihood of picking up malware from social media and fluff sites is far more likely than from institutions that at least SHOULD be security conscious. One of the nice things about penguin world is that there are no restrictions on how many installs you run and they don't take a lot of room - I find a 20 GB partition is adequate unless I'm storing a lot of data. A basic functional *buntu install can be done in less than half an hour. If you want it all tweaked and pretty, that takes longer:).
