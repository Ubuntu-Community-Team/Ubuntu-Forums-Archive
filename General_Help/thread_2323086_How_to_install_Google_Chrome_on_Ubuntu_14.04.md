### Post by Habitual on 2016-05-03
> **ChristmasPi said:**
> When I go to the Google website ( [https://www.google.com/chrome/browser/desktop/](https://www.google.com/chrome/browser/desktop/) ) and download Chrome, it downloads the file google-chrome-stable_current_amd64.deb.  When I run it, I get a message saying wrong architecture.

You cannot run 64 bit software on an i386 architecture.
You'll have to find a 32 bit archive.
Since 32 bit is not supported any more, you have few options.
Continue using Google Chrome 32, or
Find another browser.
Install a 64bit Operating system that will support 64bit archives like
google-chrome-stable_current_amd64.deb
