### Post by jespdj on 2009-01-21
Ubuntu is not a free version of Windows. It is a different operating system, and it was not designed to run Windows programs. If you absolutely want to run Windows software, then use Windows instead of Ubuntu.

As the other people have already said, there is other software available on Ubuntu, such as OpenOffice, which can do the same things as Microsoft Office on Windows. Look on a website such as [osalt.com](http://www.osalt.com/) to learn which alternatives there are for well-known commercial software.

Using [WINE](http://www.winehq.org/), you can run many Windows programs on Ubuntu. But programs that are natively written for Ubuntu generally work better.

### Post by mb_webguy on 2009-01-26
Office 2003 runs well under Wine -- I have it installed on one of my computers for my mother who refuses to try anything else -- though I agree OpenOffice.org does everything that Office can do and more, and it's free.

As for other Windows software, I've found in most cases that the equivalent software available for Linux is just as good if not better, and is also almost always free.  There are a few specific applications (such as OneNote, apparently) that either don't have a Linux alternative or else those alternatives are not as good as the Windows applications, but that's why we have Wine.  Not all Windows applications will work with Wine, but enough do that it can make the transition from Windows to Linux much easier.
