### Post by hwttdz on 2011-03-21
Overclocking generally happens at the bios level and before any operating system.  So yes you can, ubuntu is overclocking agnostic to the best of my knowledge (i.e. if the hardware can handle it, it doesn't care).

If ubuntu doesn't start it likely that any other operating system would also fail to start or at least be unstable.  And you probably need to dial it back a bit.
