### Post by 3rdalbum on 2010-05-08
Any computer that you can build from new parts today will be able to satisfy Ubuntu's minimum requirements*. Just check on hardware compatibility before buying anything.

*I'm sure if you really tried, you could build something from new parts that won't run Ubuntu - for instance, doing crazy things such as using a small SD card as the main storage or obtaining a Pegasos motherboard with PowerPC CPU :-)
