### Post by Copernicus1234 on 2009-08-30
Try getting a command prompt and type:

```
gnome-panel
```

Just to see if the panel shows up and if there is a error message.

---

### Post by tpk on 2009-08-30
Thank you so much. I logged directly into the command line and ran the command. It said the gnome panel was not installed. So I installed it and now have a complete desktop. I have no idea how it would be uninstalled, but your fix worked. thanks again.
