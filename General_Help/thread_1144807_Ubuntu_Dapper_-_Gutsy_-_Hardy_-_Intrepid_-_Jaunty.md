Its because newer versions of ubuntu have some newer libraries which might be a problem if you install a mismatched version of some software.

For example, firefox 3.0 uses gtk 2.0 (or something like that) but dapper was using older version of that library and many others ,so it wont install on dapper .Since the older kernels are compiled with these older libraries .

You can safely use software for hardy 8.04 in your system .

The difference is about the same as with the different Windows versions:
Windows3.11 Windows'95  Windows'98  Windows2000  Windows XP  Windows Vista  Windows 7.
Dapper came before Gusty, which came before Hardy, which came before Intrepid and now we have Jaunty.
They are newer versions with newer programs and libraries, more possibilities, better working (we hope).
As Rajeev also wrote for certain programs to function well you need the right set of libraries. An old Dos program will not work in Vista and MS Office 2007 won't work on Windows3.11.
