### Post by 3rdalbum on 2010-05-10
> **noelc said:**
> I,m running ubuntu 10.04 and prior to that have ran a Linux op system for about a year now. I,m thinking its time to clean up amd tweak my PC. Any ideas how I can do this? In the other operating a defrag and :c drive "Clean Up" seemed to worked.

There's no defragmentation on Linux; usually no need for it.

You may benefit somewhat from running:

```
sudo apt-get autoremove
sudo apt-get autoclean
```

But be careful with that first command - make sure you review what will be deleted before you give it the goahead.
