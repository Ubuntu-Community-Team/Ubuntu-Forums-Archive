### Post by jdong on 2005-01-11
Follow the given and linked information on adding the "Warty Backports" repository to /etc/apt/sources.list... It'll walk you through the simple, graphical procedure.


Then, use the "Synaptic" program (that you used for setup), "Refresh list", "Upgrade (smart upgrade)", then use CTRL+F, find thunderbird. Install it.
