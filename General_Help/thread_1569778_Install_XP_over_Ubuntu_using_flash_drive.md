### Post by an0dos on 2010-09-07
Here are a few tips:
1) I'm pretty sure that you can only use a windows machine to do what you want to do. You may have to ask a friend to let you use his/her computer to create a bootable USB drive. That or you can borrow a USB DVD drive and install directly from the cd.

2) Windows XP can be a giant pain in the rear when trying to install on a modern computer. Lots of components will not work immediately after the install and you will have to search for drivers. You may want to download these before trying to install XP. At the bare minimum verify that XP drivers exist for your computer.

3) The problems you were having with Windows 7 were probably driver related. If you still have the Windows 7 installation media, you should put that on the computer and download the proper drivers from the computer manufacturer's website. Make the world a better place and help kill off Windows XP.
