### Post by alladnsane on 2009-02-05
If anyone else gets stuck then if you search at the following site you should be able to find your solution.  It is cheating so don't do it except in desperation.  

[http://www.xdr.com/dash/scavenger.html](http://www.xdr.com/dash/scavenger.html)

This is Dave Ashley's own website to his scavenger game and on this site you can find some hints as well as another link to "movie" which will provide you with a file called levels.scl.  copy this file into your /home/YOURNAME/.scavenger folder (after making sure you backup your own levels.scl file or else you will lose all your progress!!).  

After this file is copied then start scavenger and play the demos to the level you are stuck on.  The demo will now show Dave's solution.  Copy your original levels.scl file back in and away you go.
