### Post by grahammechanical on 2011-09-06
How are you trying to install this program through wine?

I use the Uninstall Wine Software utility. It has an install button that lets you browse to the location of the setup.exe file (or something like that) of the program you wish to install.

You might need to use the Configure Wine utility to set wine for the version of Windows that your program runs under. 

There is in the software Centre something called Winetricks. It can also be used to install applications as well as other things. I have never used it but it might allow you to set various options that will help this program to run.
