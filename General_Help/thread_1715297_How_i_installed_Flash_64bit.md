### Post by elliotbeken on 2011-03-26
Hi all i was having a look in to installing Flash player 64bit and i came across the 64bit test that Adobe have released. ([http://labs.adobe.com/downloads/flashplayer10_square.html](http://labs.adobe.com/downloads/flashplayer10_square.html))

1. download the flash file (from above link).

2. extract the libflashplayer.so

3. copy  libflashplayer.so to /home/USER/.mozilla/plugins (if the plugins folder isnt there simply create it!)

4. restart/close firefox and flash should work !!

hope this helps
