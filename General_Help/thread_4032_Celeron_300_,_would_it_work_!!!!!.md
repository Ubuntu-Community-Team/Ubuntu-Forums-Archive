### Post by az on 2004-11-11
KDE 3.2 seemed to include a lot of qt3 performance enhancements which made it much snappier.

I had always found gnome on debian to crawl.  I almost never ran gnome or gnome apps.  

When I installed Ubuntu, I was pleasantly surprised to find that it is even more pleasant (see faster) to run than KDE.  I do not know if it is because Ubuntu has fewer useless bells and whistles.  Maybe someone can mention if there are some performance enhancements in gnome 2.8...

I had it running on a 167 MHz processor and it was, well, slow, but not impossible.  Running on a 535 MHz processor is pleasant.  So perhaps a 300 MHz machine would not be so bad.

It doesn't compare to running Mandrake or Red Hat, which almost require 512 Megs or ram and 1 gigahertz of speed to be useable...
