### Post by WorMzy on 2010-12-26
Boot into the Ubuntu LiveCD you used to install Ubuntu, open up the partition manager (System -> Administration -> GParted Partition Manager), and delete the Ubuntu partition and the swap partition (if any). You may need to right-click the swap partition and choose "Swap off" before you are able to delete it.
