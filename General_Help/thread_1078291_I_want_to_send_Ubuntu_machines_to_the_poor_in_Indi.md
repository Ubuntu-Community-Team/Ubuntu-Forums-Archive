### Post by mb_webguy on 2009-02-23
1) I'm not sure.  I don't have any experience using a mobile phone as a modem.  I know some other forum members do, so perhaps they can provide an answer.

2) You should have no problem connecting the computers to the Internet using Ethernet cables.  Some wireless cards are not well supported on Linux, and require some extra work to get them going, but a regular network card with an Ethernet port will just work.

3)  Ubuntu would be a decent choice, but given the specific environment and use for these computers, you might want to look into [Edubuntu]("http://edubuntu.org/"), a variant of Ubuntu designed specifically for the educational environment.
