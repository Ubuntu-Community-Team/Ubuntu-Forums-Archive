### Post by Rubi1200 on 2023-08-05
Hi and welcome to the forums :-)

If this is your first time installing Ubuntu, I would recommend going with the defaults. The installer will erase and use the entire disk, setting up the relevant filesystems.

I usually choose to do a Minimal install but Normal gives you a full range of different applications. I do not normally select to add any third-party stuff.

Finally, you should install the LTS (Long Term Support) releases for more stability.

Good luck and feel free to ask as many questions as you need to get comfortable with the new system.
