But the kernel fails to mount my root filesystem. How come?
I use kernel-source-2.6.12. from the ubuntu mirrors.

### Post by hollywoodb on 2006-04-30
Did you compile in the proper IDE/ATA drivers into the kernel as well?

Note that without an initrd anything your system needs to boot must be compiled into the kernel, not as modules.  

Also, is the "root=" set correctly in bootloader config?
