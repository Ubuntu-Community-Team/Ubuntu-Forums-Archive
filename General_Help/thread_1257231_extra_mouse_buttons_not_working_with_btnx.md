### Post by PeterP24 on 2009-09-03
Hi! I bought today a mouse (producer A4tech, model x5-57D). I've plugged it on my Ubuntu 8.04 (amd64) box and I've experienced some trouble with it. The three extra mouse buttons won't work in Ubuntu.
The mouse has six buttons plus the wheel. It has the classical click left and click right buttons, and a double click equivalent button (click once instead of twice says A4tech). These work like their suppose to do. 
The other three buttons don't work on Ubuntu. One of them is placed on top of the mouse (just behind the wheel) - let's name it Upper - and, on Windows OS should bring, when pressed, an Office related menu. The other two buttons (I named them Lat1 and Lat2) are placed on the lateral (left) and are programable.
I've installed btnx. It detected these three buttons. I've configured these three buttons to do Copy (Lat1), Cut (Upper) and Paste (Lat2) through key combination (Ctrl C, Ctrl X and Ctrl V). 
Only Lat1 works like it supposed to, but not in Firefox. Lat2 doesn't appear to do anything while Upper works like click left + click right in the same time (console paste - if some text was selected previously). By the way it seems that the Upper buttons totally replaced the click left/ click right combination since this doesn't work.
In Firefox, Lat1 and Lat2 behave like Back and Forward functions while Upper keeps behaving like before.
My problem is that I don't know what am I doing wrong. I didn't configured these extra buttons to have those function in Firefox. And I don't know why Lat2 and Upper won't work in the file manager or in a document. Please help!


Peter

### Post by VipX1 on 2009-09-03
Mouse buttons are like that. If go into Compiz ( Install Compiz using the Synaptic application which is in the 'Administration' menu) and then go into the Desktop menu in Compiz. There you'll find 'Rotate Cube'. In the Bindings page,  you can initiate the 'Rotate Cube' effect with any one of 9 mouse buttons and you choose the button. So pick each button and then one at a time. After that you'll know 2 x things. Which buttons on your mouse work with Ubuntu and what number Ubuntu identifies the button as.
