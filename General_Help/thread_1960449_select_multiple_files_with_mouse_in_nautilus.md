### Post by kurja on 2012-04-17
On my desktop I can click, move the mouse to draw a rectangle, and when I release the mouse button everything that was inside the rectangle is now selected. I can't do that in Nautilus (3.4.1), can I enable it?

---

### Post by Dave_L on 2012-04-17
I think that rectangular selection is possible only in the Icon View.

But in the other views, you can use the Control or Shift key in conjunction with the left mouse button to make a multiple selection.

---

### Post by Vaphell on 2012-04-17
you can select range by shift-clicking and you can add 1-by-1 to the selection with ctrl.
