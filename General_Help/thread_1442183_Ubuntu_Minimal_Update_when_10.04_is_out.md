### Post by mkvnmtr on 2010-03-30
I quite often use a minimal install xfce. Really like it. If it is 9.10 you can upgrade to 10.04 by typing sudo do-release-upgrade -d. I have done it and you should not have any problems. Like is always said back up.
On virtual box I could not find a 10.04 nminimal install disk for 10.04  so used one I have for 9.04 and before I installed anything I upgraded first to 9.10 and then 10.04 and then installed xfce4 and my other stuff. Makes a nice system that uses few resources.
