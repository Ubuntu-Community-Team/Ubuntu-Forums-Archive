### Post by tredegar on 2011-04-28
I think that if you compile stuff for linux, using a version of gcc that differs from the version your running kernel was compiled by, it won't run. So it may be best to use the version that synaptic offers you, but I might be wrong here, or maybe you want to cross-compile something.

So if you *really* want the latest version, take a look [here]("http://gcc.gnu.org/")
