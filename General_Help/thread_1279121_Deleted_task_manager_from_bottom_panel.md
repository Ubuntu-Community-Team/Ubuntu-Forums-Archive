### Post by webpirate on 2009-09-30
Update: I figured it out. When you go to add widgets to the panel it will say that task manager is already there and installed. How I got it back was to "Remove" the "so-called" installed taskmanager and then add it again and then *POOF* it showed up. I hope I helped anyone who did the same thing I did.
