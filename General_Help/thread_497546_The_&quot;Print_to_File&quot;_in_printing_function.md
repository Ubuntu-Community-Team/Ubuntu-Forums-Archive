### Post by ryantmer on 2007-07-10
This is from which program? If you're talking about gedit, then I believe you can choose to print to a .pdf file or a PostScript file. On Windows XP, you could use [Foxit Reader]("http://www.foxitsoftware.com/pdf/rd_intro.php") for the .pdf and Adobe PostScript reader for the postscript file.
