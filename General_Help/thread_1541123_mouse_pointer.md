### Post by quixote on 2010-07-31
If you right-click anywhere on the desktop, and select Change Desktop Background, a window comes up.  Select the Themes tab.  The theme you're currently using will already be selected.  Push the Customize button.  One of the tabs is then Pointer, and one of the choices is a big red pointer ... and a bunch of quieter ones. :D
