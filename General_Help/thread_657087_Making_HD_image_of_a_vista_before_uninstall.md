### Post by stump138 on 2008-01-03
You shouldn't need to pull the drive to create a backup image.  Programs like Acronis/ghost etc.. run off of a bootable cd/disk.  I tend to use Acronis(have to buy) for windows images. I know you can use partimage to do this as well, but I have never tested it.  Psychocats has a tutorial [here]("http://www.psychocats.net/ubuntu/partimage") for partimage (might be a place to start).

If you want you can just hook up an external USB drive to pull the image with.  There are alot of diff ways you can do it.
