### Post by sanderd17 on 2012-01-25
Gnome classic uses compiz as window manager. Compiz asks a better supported graphical card, but gives you some fancy effects and more configuration in return.

Gnome without effects uses metacity (I'm not entirely sure) as window manager. Metacity should even work good without graphical card. So you have less configuration options, less effects, but it works on more hardware, and it's possible it uses less power too.
