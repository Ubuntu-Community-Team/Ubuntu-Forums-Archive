### Post by coffeecat on 2011-04-30
> **lotharmat said:**
> Has anyone managed to get the Thesaurus working in Libreoffice Writer?

Yes. First a mild rant and then I'll tell you how.

The non-functional UK thesaurus is a long-standing bug in the Go-OO version of Open Office, which is what Ubuntu had been using before it changed to LibreOffice for Natty. The Ubuntu people fixed this in Maverick (possibly in Lucid too) so I'm fairly annoyed that we Brits have been forgotten - yet again! - in the move to LO. I believe LO is based on Go-OO too, so there's really no excuse. 

The fix. Have a look here:

[http://www.weeklywhinge.com/?p=69](http://www.weeklywhinge.com/?p=69)

The download is in the "here it is" link and the method for installing it was added as an edit. Don't forget to restart LO for the thesaurus to start working.
