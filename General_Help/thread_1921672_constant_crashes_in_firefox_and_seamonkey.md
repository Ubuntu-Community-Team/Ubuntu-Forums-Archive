### Post by lovinglinux on 2012-02-07
Sometimes there are some incompatibilities or corrupted data in the Firefox profile after an upgrade, especially qhen the new version has many core changes.

Try to create a new profile and check if the problem persists.

The command below will launch the profile manager.

```
firefox -P
```
