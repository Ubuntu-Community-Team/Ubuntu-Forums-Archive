### Post by deadflowr on 2016-02-06
You might try moving the folder /var/lib/apt/lists to somewhere else temporarily,
and then re-run the update command.
It should generate a clean list of files, including those Release named files.

I'd run
```
sudo mv /var/lib/apt/lists ~/Documents
```
which will place that folder in your home documents folder.
If it fails to fix the issue, you can move it back, and start from scratch.
