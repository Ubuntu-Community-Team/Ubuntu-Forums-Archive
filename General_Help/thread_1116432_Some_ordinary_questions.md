### Post by ad_267 on 2009-04-04
1. Go to System - Preferences - Appearance. You can download new themes from [http://gnome-look.org](http://gnome-look.org)

2. File - Print - Options and tick "Reversed". You can get the same options from Tools - Options - OpenOffice.org Writer - Print.

### Post by chessnerd on 2009-04-04
