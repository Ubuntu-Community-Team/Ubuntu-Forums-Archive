### Post by slakkie on 2009-10-14
The easy way to give back to the community is by:

* Helping others on IRC and/or the forums

Other ways of helping

* You could translate documentation
* You could file bug reports 
* Fix minor bugs if you can

Most of these actions, if not all will also help yourself, since you will be reading much documentation in order to see how an app works. So you learn something from it.

I have no clear answer on what you should do, I think you just need to pick an area you like and try to help out where possible. 

I used to help out on #ubuntu, but I found out helping others (especially newer users) can be painfull, where as helping new users on forums is more fun to do. You can put more information in a post then in a single line on irc. So I give back by helping others on the forums. 

Once you have done something for a while you might want to do something else, and you will probably have gained some additional skills so you pick something else you want to do... 

It all depends on what you want to learn and your personal preferences.

### Post by t0p on 2009-10-14
+1 for helping in these forums.  I'm no guru, but I help where I can.

Another good way to contribute is by donating money.  It doesn't matter if you can't give a lot: small donations are as gratefully received as large ones.  Most developer's web pages will have something to say how to donate.  A lot accept donations through PayPal.

As for learning more about Linux: there are plenty of online courses in all aspects of computing; Google is your friend.  And there are even more resources for those who prefer to learn without a structured course.  At the moment I'm learning how to write shell scripts - a useful skill for anyone who wants to get into the nitty-gritty of Linux system administration - and there's a very useful guide available from the repositories, The Advanced Bash Scripting Guide.  It comes in HTML format, and you can get it by typing this command into a terminal:

```
sudo apt-get install abs-guide
```

Using perl and mysql are also useful skills.  And there are many more.
