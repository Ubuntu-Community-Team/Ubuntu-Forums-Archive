### Post by Anduu on 2006-06-27
I don't think it is broken like you think...I am running the latest fglrx and XGL works fine.

What typically happens when fglrx gets updated is the driver itself doesn't reinstall correctly...type fglrxinfo in a terminal...I'll bet dollars to doughnuts it says the "Mesa' driver is being used.

If so simply enter in a terminal:
```

sudo aticonfig --initial
```

Check your /etc/X11/xorg.conf to make sure the fglrx driver is listed.

### Post by lamego on 2006-06-28
Please note that installing drivers manually means you will not get the updates from apt and may get you into troubles later with kernel updates.
The latest xorg-driver-fglx will work just fine if you make sure  X is properly setup to use it, just edit:
/etc/X11/xorg.conf
