### Post by Elfy on 2012-08-01
> **robire said:**
>  Are there changes in update procedure ? Regards robire

Yes.

If the update requires the installation of NEW packages it will ask for your password.

For example a kernel update will require password.

[https://wiki.ubuntu.com/SecurityTeam/FAQ#Update_Manager_doesn.27t_prompt_for_security_updates](https://wiki.ubuntu.com/SecurityTeam/FAQ#Update_Manager_doesn.27t_prompt_for_security_updates)
