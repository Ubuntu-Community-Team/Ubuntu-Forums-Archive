### Post by kvonb on 2007-04-16
VMware player is the easiest and it works prefectly (no 3d games though).  My webcam doesn't work in Ubuntu, but it works perfectly in a VMware Windows session.

The only problem is that the player can't create new VMs, but that can be navigated around by creating a blank VM on this site:

[http://www.easyvmx.com/supersimple.shtml](http://www.easyvmx.com/supersimple.shtml)

It works really well.

Or you can download and compile VMware workstation Beta free from [www.vmware.com](www.vmware.com), which will do the lot, if you can handle compiling!  Although I don't know how long it will be available for.

### Post by talcite on 2007-04-16
Although VMware Player is the "easiest", it's also the most limited. For example, you can't exceed 1024x768 I believe. 

I think you'd be much better off with VMware Server. I'm using it, and really it's not that hard. 

Anyways, as for games, I think you're pretty much out of luck. The VMware graphics drivers aren't exactly DirectX compliant =P. 

If you want to play XP games, try Wine or cedega. Any simple 2D game should work in XP on the VMware server.
