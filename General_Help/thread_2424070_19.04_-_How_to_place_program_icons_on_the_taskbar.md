### Post by guiverc on 2019-08-02
I had trouble the first time I did this too.  The red 'block' (white X in red circle) icon disappears when you're positioned over the *Quick Launch* widget, becoming an arrow with black "+" inside green circle. You then drop the icon on the *Quick Launch* widget and fine-tune as the next step using the move left-right available on right-click (of the item).

For me with multiple panels on my screen I learnt you must copy from a menu on the same panel where the *Quick Launch* widget is located; as I didn't want a menu on that panel, I added it for the setup then removed it later.
