You are quite right.  A "workaround" is useful but a permanent fix is even better. :-)
> **ddjolley said:**
> ...Why is it so difficult to file a bug report?...Why can't I just send an email?
It really isn't *that* difficult.  You have to appreciate that it is necessary to have a robust bug-tracking system to control the work done on thousands of different bugs.  This requires a certain level of data-validation that is simply not possible when reporting bugs by email.  Of course it is also, unfortunately, necessary to defend the bug tracking system against spammers who would welcome flooding the system with emails.
> **ddjolley said:**
> ...I got an email but it talked about a forgotten password and did not contain any 6 digit code.  So, I couldn't complete the registration because I didn't have the 6 digit code.
It is so long since I registered with Launchpad that I cannot recall anything about a "6 digit code"!  However, just to make sure you are on the right track I attach three links below that may clarify the process:
[LIST]
[*][ReportingBugs]("https://help.ubuntu.com/community/ReportingBugs")
[*][Creating an account]("https://help.launchpad.net/YourAccount/NewAccount#Creating_an_account")
[*][launchpad help]("https://help.launchpad.net/Legal#Contact_Information")
[/LIST]
You will see that a contact email address is given at the end which can be used if you are still having problems with your Launchpad account.

Hope this helps.
