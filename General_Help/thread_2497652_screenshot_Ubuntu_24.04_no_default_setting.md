### Post by Dennis N on 2024-05-13
You can use Shift+PrintScreen to do a whole-screen screenshot. It's automatically saved to ~/Pictures/Screenshots. No dialog pops up.
There are other keyboard shortcuts set in Settings > Keyboard > Keyboard Shortcuts. I didn't find Alt-Printscreen to work at all. 
You can also redefine the shortcuts there.

### Post by deadflowr on 2024-05-13
For Plasma you can change the shortcuts in the Spectacle app.
Open Spectacle and then click on the configure button. and then select shortcuts.

Just change "Capture Entire Desktop" from SHIFT+PRINT to PRINT.
It automatically saves to the Pictures folder. No additional actions required.

You can also change the save location in the same configure tool under Save
