### Post by jasmuz on 2007-11-11
1- Burn the ISO image

2- Boot Ubuntu GNU/Linux

3- Select your external drive to be the install destination

4- Select you external drive to have the GRuB bootloader installed

5- Reboot when things are done

6- Select your external drive as a normal drive to boot in from in the BIOS options

Good Luck!
