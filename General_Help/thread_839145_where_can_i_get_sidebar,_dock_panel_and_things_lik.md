### Post by jualin on 2008-06-24
After you download those suggested you might like to check out [http://gnome-look.org]("http://gnome-look.org") since they have more variation and more themes. If you would like to change the way that your panels look, you would need to install a "GTK 2x" theme. For the login window you need to install a "GDM" theme. Hope this helps!
