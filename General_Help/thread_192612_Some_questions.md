### Post by adwait on 2006-06-09
1)To set a static IP, find the network settings in your Menu. Not really sure where you'll find it, since it varies with Kubuntu/Ubuntu...mus be under System>Admin>Networking I think... Anyway, once there, select eth0 and you'll be given to assign it a static IP.

2)NTFS suppport is pretty much experimental under Linux and whatever is there, it allows only reading and not writing to an NTFS partition. As for Windows, to the best of my knowledge there are no programs under windows that can read an ext3 partition. An option would be to create a FAT32 partition which can be read/written by both Ubuntu and Windows and exchange data via that partition.

3,4)Dunno

5) You would be better off using SSH, but yeah, the telnet client comes with Ubuntu. In a terminal just type telnet..
