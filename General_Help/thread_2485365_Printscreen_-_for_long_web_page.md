### Post by Holger_Gehrke on 2023-03-28
If you're using Firefox, that has a screenshot tool built in. You can use 'Hamburger Menu'->'More Tools'->'Customize Toolbars'. You will find an icon for the "Screenshot" tool among the non-assigned icons and can move that to either a toolbar or the overflow menu.

Alternatively there are several add-ons which offer this functionality; I've used "Nimbus Capture" in the past and still use it sometimes because it offers some additional functions for editing the image before saving.

As a third option you could print into a PDF file.

Holger
