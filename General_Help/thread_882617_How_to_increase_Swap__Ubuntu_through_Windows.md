### Post by issih on 2008-08-07
I believe that hibernate is one of the casualties if you choose to use wubi to install ubuntu.

Hibernate usually works fine in a full blown install of ubuntu to its own dedicated partition (some hardware is pickier than others, but it usually works), if you use wubi to install it within windows however i isn't achievable at present.

Sorry, you found the flaw...

Consider doing a proper dual boot install as you seem to like ubuntu so much.
