### Post by snowpine on 2010-11-18
You are correct that 7.04 is well past its "expiration date" and it's time to move on. Xubuntu and Lubuntu are recommended for older hardware. The lightest/fastest possible Ubuntu install is to start with the Minimal CD and add a lightweight windows manager plus your choice of apps: [http://www.psychocats.net/ubuntu/minimal](http://www.psychocats.net/ubuntu/minimal)

Also, since you are already familiar with Red Hat, I'll put in a good word for CentOS (a free Red Hat derivative). I have CentOS 5.5 on an older pentium 3 with 256mb of RAM, and it runs very well (plus it will be supported for years--you don't seem like the type who always wants the latest "stuff"). Other good distros for older hardware include (but are not limited to) Puppy, SliTaz, AntiX, etc.
