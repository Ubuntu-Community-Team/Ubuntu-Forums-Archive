### Post by Frogs Hair on 2011-11-19
11.10 requires GTK 3 themes and some metacity window borders will work as well . [http://gnome-look.org/index.php?xcontentmode=167](http://gnome-look.org/index.php?xcontentmode=167)

The Equinox themes have not been ported to GTK 3 as far as I know. See below if you need help installing themes . 

(Installing Downloaded Themes / Icons)

Install the Gnome Tweak Tool / advanced settings from the software center for making theme selections.  

(Single User) Create a .themes and .icons folder in home / hidden folders . Use Ctrl + H to open the hidden folder option in Nautilus .

(All Users) Use gksudo nautilus > File System / usr / share / themes or icons .

Download , extract the packages , and place the folders in either location making theme selections using Advanced Settings . Logout - in may be required .

### Post by Frogs Hair on 2011-11-19
> **layers said:**
> I'm using Unity. Why do I need to get the Gnome Tweak tool?

Unity runs on top of Gnome 3 in 11.10 . You won't be able to change themes other than the default Ambiance and Radiance themes without it . It's also needed in the Classic Gnome or Gnome Shell session should you choose to install them .
