### Post by undecim on 2009-10-01
There are hundreds of games in the ubuntu repositories that you can install from the "Add/Remove" app.

Also, if you install the "ubuntu-restricted-extras" package, you can play any of the online flash games you play on windows.

With WINE, you can run (almost) any software that will run on windows. It may be a little bit slower, so that will only be an issue if you are a hardcore gamer (like those people who spend thousands on their computer to get them playing the newest games at the highest resolutions)

But maybe you like playing 3d games with high resolution? You can keep your windows installation when you install Ubuntu, and when you boot your computer, you will have an option of which system to load. That way, you can always use it when you feel the need to frag in 1600x1200.

That would be [WINE](http://wiki.winehq.org/AboutWine). [http://wiki.winehq.org/AboutWine](http://wiki.winehq.org/AboutWine)

**W**ine **I**s **N**ot an **E**mulator.   ;)

### Post by sedawk on 2009-10-02
It depends on your games, see the Application Database
mentioned above.
Some of my favourite games have some minor issues with
fonts or shadows and this can be so annoying that it
wastes all the fun although the game runs fine.

What is nice about WINE is that every game (or application)
can be installed in its own environment without wasting
disk space. So removing stuff and getting a clear environment
is much easier with WINE.

For very older games check out "dosbox" - works great!
For old Windows games you can also try to use virtual box
or one of the free VMware products (running a virtual PC
from Ubuntu). There seem to be some improvements so that
even DirectX (or OpenGL) games work better inside a
virtual PC.

### Post by 3rdalbum on 2009-10-02
Ubuntu can play all current Linux-based games, as long as you have accelerated graphics drivers.

If you want to play your current catalog of Windows games, and any new ones that come out, then you should keep Windows available as a dual-boot system (Windows and Ubuntu sharing one disk).
