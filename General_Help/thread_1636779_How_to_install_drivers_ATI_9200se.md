### Post by Mark Phelps on 2010-12-03
Sorry ... but there are no ATI Radeon proprietary drivers that will work with your card and recent versions of Ubuntu.  ATI dropped driver support for you card prior to Ubuntu v9.10.

You will have to use the open-source drivers -- which are installed by support in the Desktop Ubuntu versions.

If you're using a Server version, can't help you there as I don't use it.
