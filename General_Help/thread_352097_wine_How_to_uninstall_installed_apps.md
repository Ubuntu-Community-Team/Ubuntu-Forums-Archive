### Post by TheRingmaster on 2007-02-02
open a terminal (Applications-->accessories-->terminal) and type in "wine uninstaller"

This will open up the uninstaller for wine applications.

### Post by atact88 on 2007-02-03
I don't know if you keep a windows install on another partition or not, but I do, and I've found it extremely convenient to install Windows programs (in Windows) to a FAT32 volume on my hard drive rather than the primary Windows partition. This way it's already installed and when I get to linux I just mount the partition, tell Wine to run the program executable from there...works very well so far for me (albeit I haven't tried tons of apps yet)
