### Post by sayakb on 2008-09-20
You can either download GTK or emerald themes.
If it is a GTK theme that you are downloading, and most of the time, it is a .tar.gz file. Goto System->Preferences->Appearance and drag n drop the theme to the Appearance window. It should notify that it has been installed. Now simply click on the theme to apply it.

And if it is an emerald theme, my tutorial shows how to install emerald themes.
