### Post by Impavidus on 2017-07-10
But the partition isn't really part of the path, unlike in Windows. The beginning of the path, which is the root directory, is always on the root partition, but different directories in the path could be on different partitions. Or you could have a file on a partition, that is in reality a disk image stored on a different partition.
