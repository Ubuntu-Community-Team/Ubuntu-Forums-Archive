### Post by bukwirm on 2007-05-28
If it's a Debian package (ends in .deb), you can use "dpkg -i *packagename*" to install it.
If it's a compressed tar archive (ends in .tar.gz), you'll need to extract it with "tar -xzvf *filename*" and look for a README file.
