### Post by linuxgeek77 on 2010-09-03
Hello there My fellow Ubuntu / Kubuntu / Xubuntu / Mint / Debian Users.. Today I solved my friend's Issue with her Intel 945GL intergrated graphics card. I am here to share exactly what i did.

We all know that since ubuntu 9.10 xorg.conf is NONE existent and for all of us with complicated intel and ati drivers are left out with little hope. Even after I tried running the sudo Xorg -configure command in the terminal after killing Xorg it would will refuse to create my Xorg.conf.new as I have seen all over the posts. So I decided to do it teh manual way and write my xorg.conf from scratch and save it on my desktop then simply move it to /etc/X11. Here is what I did. DO THIS IF IT IS YOUR LAST RESORT.

1) open gedit  (mousepad on xubuntu , kate on kubuntu)

copy and paste the following xorg.conf configuration "I added few extra resolution mods- it wont hurt you, you will have teh extra resolution options"

```
# xorg.conf.failsafe (X.Org X Window System server configuration file)
#
# This file was generated by dexconf, the Debian X Configuration tool, using
# values from the debconf database.
#
# Edit this file with caution, and see the xorg.conf manual page.
# (Type "man xorg.conf" at the shell prompt.)
#
# This file is automatically updated on xserver-xorg package upgrades *only*
# if it has not been modified since the last upgrade of the xserver-xorg
# package.
#
# If you have edited this file but would like it to be automatically updated
# again, run the following command:
#   sudo dpkg-reconfigure -phigh xserver-xorg
 
Section "Device"
        Identifier      "Configured Video Device"
        Driver          "intel"
        Option          "UseFBDev"              "true"
EndSection
 
Section "Monitor"
        Identifier      "Configured Monitor"
        Option          "DPMS"
        HorizSync       30-71
        VertRefresh     50-160
Modeline "1400x1050_60.00"  121.75  1400 1488 1632 1864  1050 1053 1057 1089 -hsync +vsync
Modeline "1280x1024_60.00"  109.00  1280 1368 1496 1712  1024 1027 1034 1063 -hsync +vsync
Modeline "1152x864_60.00"   81.75  1152 1216 1336 1520  864 867 871 897 -hsync +vsync
 
EndSection
 
Section "Screen"
        Identifier "Default Screen"
        Device "Configured Video Device"
        Monitor "Configured Monitor"
        DefaultDepth 24
        SubSection "Display"
        Depth 24
        Modes "1400x1050" "1280x1024" "1152x864" "1024x768"
        EndSubSection
EndSection
```2) save the file as xorg.conf on your Desktop
3) open the terminal and navigate to you file on the Desktop and move it to /etc/X11/xorg.conf with this command:

 ```
sudo mv /home/(user name)/Desktop/xorg.conf /etc/X11/xorg.conf
```++note insert the (user name) with your login name IE login name++ for example my user name / login name is raj this is what it would look like:
**EXAMPLE ONLY - DO NOT USE IT- CHANGE USER NAME TO YOUR OWN**
```
sudo mv /home/raj/Desktop/xorg.conf /etc/X11/xorg.conf
```4) After you move it, via terminal, run the terminal again and type this command to reconfigure xorg.conf / xserver:

```
sudo dpkg-reconfigure -phigh xserver-xorg
```5) after that simply restart, and you should have Compiz working , Along with better Frames per second in games.

You can thank me Later. 

NOTE: THIS APPLIES for INTEL 945GL and if you have a Dell 6000 series then your in luck/ if not then try it anyway it might work 

NOTE: I did not make any backups for xorg.conf as i did not need any. I CREATED the xorg.conf myself and saved it then moved it.

NOTE: this is not for computers with CRT monitors. IF you have a CRT monitor REMOVE the additional mods:
Modeline "1400x1050_60.00"  121.75  1400 1488 1632 1864  1050 1053 1057 1089 -hsync +vsync
Modeline "1280x1024_60.00"  109.00  1280 1368 1496 1712  1024 1027 1034 1063 -hsync +vsync
Modeline "1152x864_60.00"   81.75  1152 1216 1336 1520  864 867 871 897 -hsync +vsync
