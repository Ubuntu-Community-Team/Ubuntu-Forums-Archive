### Post by deadflowr on 2013-07-06
No clue as to how to change cinnamon's default players, but if you right-click on the file and select properties and goto 'open with' you can reset the defaults per file type.
For instance, if you want mp3 files to open in VLC then select vlc as the default, but if you want another file type to open with rhythmbox then set that file-type for rhythmbox.
