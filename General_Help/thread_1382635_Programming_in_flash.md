### Post by Marlonsm on 2010-01-16
You'll need Adobe Flash for it, I don't know any other program that can do it.
It's not difficult, it's the one I know better, actually. Its language is called Actionscript, some say it looks like Javascript.
It can also be used for animations without any coding.

Flash 8 runs perfectly in Wine (see attachment) although it's from 2005, it has all the tools you need today.

If you look around the web, there will be many tutorials out there. This is the site I use, but it's in Portuguese, [http://imasters.uol.com.br/](http://imasters.uol.com.br/) I'm sure you can find sites like that in any language.

If you have any questions, fell free to ask.
