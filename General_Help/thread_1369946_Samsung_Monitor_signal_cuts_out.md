### Post by gordintoronto on 2010-01-01
Nice monitor!  Have you played with screensaver/power management settings?  Does your video card really support 2048 x 1152?  What video card?

From sysinfo I can go to "NVIDIA X Server Settings," where X Server Display Configuration has a box labelled "detect displays."  If your computer still thinks it has the previous display, there are probably several bad things which could happen.
