I am using Linux Mint 17 on a late 2013 MacBook Pro with Retina display. The desktop scales very nicely, but many programs don't. GTK2 elements are microscopic, for instance. It would probably pay you to google the name of the various programs you want to use along with the phrase "retina support", to check.
