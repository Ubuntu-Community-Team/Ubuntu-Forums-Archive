To all Ubuntu users,
SANDISK - recording music MP3.
1.) set USB  in SANDISK player  to MSC
2.) Insert into USB 
2.)Open in Linux desktop the sandisk folder
3.) Move any recorded music folder  with drag and copy into "Sandisk" desktop  Music 

No resync required !
It works much better  than using
Rhythmbox in Linux
