I often find myself pasting text from the web into my OpenOffice.Org Writer documents.  OpenOffice likes to keep any type of formatting that may have come with the text from the web page, so I have to go to Edit > Paste Special... > Unformatted Text > OK just to get the text to match my document's formatting.  Is there a way that I can change paste's default action to paste unformatted?  This way, I can do the same thing just by pressing Ctrl+v or by clicking the toolbar clipboard icon.  It'd really be a time-saver.

### Post by Mander on 2009-02-12
Don't know if you found this already, but ctrl+shift+v will bring up the paste special dialogue.  I think you can probably fiddle around with the settings (tools--customize--keyboard) to change the default behavior, too.
