### Post by Finalfantasykid on 2010-11-13
You can install it using this command:

```

sudo apt-get install vim

```

No need to download a deb package.

### Post by Charlie91 on 2010-11-13
The *Terminal* is a good place to start--issuing text commands, etc. If you like GUI, try *Synaptic Package Manager*: click *System*, then *Administration*, then *Synaptic*...  Just search for *vim* and install it. Check your settings, repositories, etc (*Help* menu if needed).  I only use *dpkg* if the program is not available there. Enjoy *Ubuntu*!
