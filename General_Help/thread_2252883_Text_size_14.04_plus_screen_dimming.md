### Post by JazzPotato on 2014-11-15
I didn't think anyone else would have the same problem that I had, but here you are! My laptop is a HP Pavillion G6, and every time I would log in the text would get progressively smaller and eventually become microscopic.
When I reinstalled ubuntu the problem persisted.
The bad news is, I never figured out how to fix it. The guys on the irc channel were similarly stumped. I eventually installed Xubuntu 14.04 which does not have this problem.
The only way to get around the problem that I found is to install the package "unity-tweak-tool"
```

sudo apt-get install unity-tweak-tool

```
Or if you are happy using a different desktop environment then you could look into changing as I did.
