### Post by snowpine on 2010-04-23
Try a Live CD or Live USB of each and see which you prefer. They have very different user interfaces (Gnome Desktop vs. Netbook Launcher) but are the same core system underneath. A lot of people like Netbook for small screens because it has big icons. :)

You might also consider using the forum Search feature to read the experiences of other users with the same hardware...

If you can be satisfied with the look of UNR, then that would surely give you the best performance.
