Try 
```

```

Meaning of the color

green-executables (programs).
blue - links to files (probably programs).
red-broken links (what they point to doesn't exist any more.

### Post by Dennis N on 2014-06-21
From within the terminal itself you can:

type **help** for the list of internally defined commands.
type **info** for a list other commands and briefly what they are for - note especially the individual utilities section.
type **man** followed by the command name to get specifics on a particular command and its usage and options. Example: **man cp** to get information on the cp command.
