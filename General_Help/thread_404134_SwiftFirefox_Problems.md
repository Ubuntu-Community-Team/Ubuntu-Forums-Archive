### Post by Neobuntu on 2007-04-08
You should see the "scanner" adjust (for the other) every time you run the other. Perhaps that feature is now broken. Don't run them at the same time.

Export your bookmarks out to the desktop for a backup.

Remove both and reinstall one. (With the package managers you installed them with.)

Not that they conflict but just to get you going.

If you installed manually, remove it manually. (If you tried the one right from Mozilla.)

If that doesn't work try purging them (but backup what ever you want first).

If you stick with *ubuntu Firefox, it will handle future upgrades summarily. Swiftfox is not *ubuntu official.

I recommend going with the *ubuntu matched and updated Firefox. KISS.

Hope that helps.
