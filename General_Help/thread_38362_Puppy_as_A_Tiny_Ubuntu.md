### Post by jtt on 2005-05-31
I found a terrific new Puppy Linux with an ISO of about 60 MB which reminds me of UBUNTU, except that it is tiny, but functional like UBUNTU, yet immensely user friendly.  Puppy can be found at

[http://distrowatch.com/](http://distrowatch.com/)

[ftp://ibiblio.org/pub/linux/distributions/puppylinux/](ftp://ibiblio.org/pub/linux/distributions/puppylinux/)

[ftp://ftp.nluug.nl/pub/metalab/distributions/puppylinux/](ftp://ftp.nluug.nl/pub/metalab/distributions/puppylinux/)

Puppy runs as a live CD, but also runs on flash, on a zip disk, on DOS / MS Windows, on various network emulators, and on the several Linux filesystems available on hard disk.  Everything runs in memory, so Puppy is blazingly fast.
