Try changing the code to
```
select, input {
  height: unset !important;
}
```

But, given that [FONT=Courier New]<input>[/FONT] elements can be many different things, that code may have undesirable side-effects.  A solution involving setting [FONT=Courier New]min-height[/FONT] to some number of [FONT=Courier New]px[/FONT] maybe better in that respect.  Adjust to your liking -
```
select, input {
  min-height: 30px !important;
}
```
