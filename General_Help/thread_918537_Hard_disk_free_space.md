### Post by mb_webguy on 2008-09-13
Just to clarify, you have Vista, and you have removed an Ubuntu installation and now want to reclaim that drive space for Vista, correct?

If so, you should be able to expand your Vista partition using Vista's partition manager.  Right-click on Computer and select Manage.  Select Disk Management from the left pane.  This will show you your drives and partitions.  If there's nothing on the drive except for the Vista partition and some unused space, you should be able to right-click on the Vista partition, select Extend Volume, and then change the settings so it uses the rest of the unused space.
