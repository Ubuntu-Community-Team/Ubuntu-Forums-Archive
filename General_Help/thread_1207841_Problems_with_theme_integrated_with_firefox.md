### Post by darolu on 2009-07-08
I may not got your post right, but it seems the problem is not Ubuntu or Firefox per se; it seems like website's styles are set that way, probably the website or some of its elements (like a div) has a white background and the web designer forgot to include text colour asuming "everybody" uses a dark font colour (like black), if this is the case it is a big design error from the web designer, not yours; you can try to solve it by temporarily changing your browser preferences or enabling a custom stylesheet (under preferences).

### Post by darolu on 2009-07-09
No, you don't need a plug-in to do that, go to Edit-Preferences-Content, there you can change the font colour.

I forgot to mention you can also try installing a theme to fix the problem, go to [http://addons.mozilla.org](http://addons.mozilla.org) and browse the Appearance category.
