### Post by 3rdalbum on 2012-06-05
Ubuntu 9.10 was only supported until mid-2011. The "repositories" which hold the software for installation, are only kept alive until the end of the support period, which is why Software Center can't actually find the software in the repository. The repository no longer exists.

The latest version of Ubuntu is 12.04, so you should download that and install it. Ubuntu 12.04 is what they call a "Long Term Support" version, so it will be supported until 2017; although you'll probably upgrade it long before then :-)  All other versions that are not Long Term Support are only supported for 18 months.
