### Post by basementjack on 2006-07-14
Well.. I found an "easy" way to get PHP 5.1.4 with PDO, so for anyone wanting the same thing right now.

I expect the packages to included in backports, so try them first, if they aren't there try "the other way".

**Checking the backports**
_1. First add the backports, if you haven't already added them._
```
sudo gedit /ect/apt/sources.list
```
Add the following linies
```
## The backport packages
deb http://archive.ubuntu.com/ubuntu dapper-backports main restricted universe multiverse
deb-src http://archive.ubuntu.com/ubuntu/ dapper-backports main restricted universe multiverse

```
_2. Upgrade_
```
sudo apt-get update
sudo apt-get upgrade
```
_3. Test_
[PHP]## We want these to return true
echo extension_loaded("pdo");
echo extension_loaded("pdo_mysql"); # change to what ever database you need[/PHP]

**The other way**
Note: I don't know whether or not this causes any problems, but I will return to report problems, if they arise.
_1. Add Dotdeb to repositories_
```
sudo gedit /ect/apt/sources.list
```
Add the following linies
```
## Dotdeb packages
deb http://packages.dotdeb.org stable all
deb-src http://packages.dotdeb.org stable all
```
_2. Upgrade the php5 and php-pdo_
Open System -> Administration -> Synaptic.
Click reload.
Click to Upgradeable (upstream) see the new packages.
Mark the php5 and php-pdo for upgrade and press apply.

_3. Disable Dotdeb again_
```
sudo gedit /ect/apt/sources.list
```
Edit the linies to
```
## Dotdeb packages
#deb http://packages.dotdeb.org stable all
#deb-src http://packages.dotdeb.org stable all
```
_4. Test_
[PHP]## We want these to return true
echo extension_loaded("pdo");
echo extension_loaded("pdo_mysql"); # change to what ever database you need[/PHP]

**Troubleshooting**
_The test returns one true and one false_
Maybe for both ways, but at least for "the other way", it is nessecary check the php.ini
```
sudo gedit /etc/php5/apache2/php.ini
```
Make sure that these lines are present and uncommented
```
;Enable pdo
extension=pdo.so
;Enable pdo_mysql ot what ever database you need
extension=pdo_mysql.so
```
