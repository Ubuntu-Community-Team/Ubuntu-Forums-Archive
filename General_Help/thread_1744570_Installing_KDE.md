### Post by Lateralis on 2011-04-30
It shouldn't break Unity.  If it does then I would consider that to be a big with Unity. 

As for removing KDE afterwards, I don't know if of a simple elegant way.  I would have thought removing kubuntu-desktop would remove all of the KDE packages.  You could always copy/paste the list of packages the kubuntu-desktop installs, so you know which packages have been installed.  And then copy/paste that same list of packages into the command line when doing aptitude purge <packages>.
