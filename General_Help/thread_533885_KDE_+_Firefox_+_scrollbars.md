### Post by sumguy231 on 2007-08-27
This isn't *exactly* a fix, but it looks like you're using Polyester, so if you didn't know you can change polyester to have Windows-style scrollbar buttons (one up, one down) in the Polyester options in System Settings -> Appearance -> Style -> Polyester -> Configure -> General -> Scrollbars. That's what I did. I gave up on making Firefox behave like KDE and made KDE behave like Firefox. :)
