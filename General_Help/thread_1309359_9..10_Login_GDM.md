### Post by ranch hand on 2009-11-01
As far as the login box itself, you are pretty much stuck with what you have.

9.10 is not a LTS.  10.04 will be.

9.10 has a lot of changes in it so that they can be used in the LTS.  Some of the implementation is not there yet.

You can change the background image.  They live at /usr/share/images/xsplash.  There are 7 "bg" images there and you need to run your chosen image through gimp to match the size and resolution of the ones there.   The GDM uses the "bg_2560x1600.jpg" image.
