### Post by PriceChild on 2006-10-02
```
sudo rm sudo gedit /etc/gdm/gdm.conf-custom
sudo apt-get remove --purge  xserver-xgl libgl1-mesa libglitz-glx1 beryl beryl-core beryl-manager beryl-plugins beryl-plugins-data beryl-settings emerald emerald-themes
gksudo gedit /etc/apt/sources.list
```and remove
```
deb http://www.beerorkid.com/compiz/ dapper main
deb http://xgl.compiz.info/ dapper main
deb-src http://xgl.compiz.info/ dapper main
```Ctrl+alt+backspace
and all should be fine.
