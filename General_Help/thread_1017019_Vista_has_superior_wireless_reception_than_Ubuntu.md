### Post by jimmy the saint on 2008-12-20
My initial guess would be that it is a driver issue.  Often, especially with newer wireless cards, the linux drivers are not quite as good as the windows drivers because the manufacturers may not make linux drivers themselves.  The result is that linux drivers are created through reverse engineering.  You could try to use ndiswrapper, which enables you to use windows wireless drivers.  I would suggest getting a repeater or buying a second hand wireless router for <10 bucks that supports bridge mode.  If you can find a v1 or v2 linksys wrt54g, you can install some pretty good firmware for the task.
