If you're using Compiz, you'll need to tell it to use the cursor as well. Open CCSM, go to "General Options", and in the first tab where it says "Cursor theme" write the name of the cursor theme (note, it must be the exact name as it appears when you selected it in Appearances). This should change your cursor in most instances, however you need to reboot to get the cursor working everywhere.
