### Post by lovinglinux on 2009-10-18
Yes, install [mozilla-plugin-vlc](apt:mozilla-plugin-vlc). You might need to remove other similar plugins, like totem. Check the [Comprehensive Multimedia & Video Howto](http://ubuntuforums.org/showthread.php?t=766683).

I personally recommend [gecko-mediaplayer](apt:gecko-mediaplayer) instead of mozilla-plugin-vlc, because the last one lacks controls (as far as I know).
