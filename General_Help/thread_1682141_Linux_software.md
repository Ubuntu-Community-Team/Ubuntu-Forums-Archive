### Post by Cheesehead on 2011-02-05
Since I like my system unborked, I only install bleeding-edge stuff I really care about (and only for good reason, like to test a new feature I *really* want).

If I install bleeding edge, I recognize that I'm an unpaid tester. Breakage is expected, and I have the skills to troubleshoot and recover. Bugs are expected, and I have the skills and responsibility to report them.
