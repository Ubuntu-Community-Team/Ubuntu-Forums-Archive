### Post by Is Mise on 2010-01-13
Yes, you're right. I guess it's been removed from the latest version of UbuntuTweak. Here's a way to install it from the terminal:
```
sudo add-apt-repository ppa:globalmenu-team/ppa

sudo apt-get update

sudo apt-get install gnome-globalmenu
```

You might have to restart your computer before it shows up in the Add To Panel dialog.
