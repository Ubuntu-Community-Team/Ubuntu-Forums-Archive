### Post by M4he on 2010-08-19
Had the same issue while trying out xubuntu on my netbook recently. As far as I remember I got it fixed like that:

open up your personal folder ( /home/<username> ) and press Ctrl + H (to show hidden files & folders) then open the '.config' folder. Delete the 'softwarecenter' folder inside it.

Then try running the center again.

Good luck!
