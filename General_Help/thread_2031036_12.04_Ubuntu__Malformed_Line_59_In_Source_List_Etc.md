### Post by howefield on 2012-07-21
Try opening a terminal and type the following command,

```
gksudo gedit /etc/apt/sources.list
```

Place a # sign at the beginning of line 59 and save out, then try again to complete your apt-get task.

### Post by drs305 on 2012-07-21
Try opening Software Center, then Edit, Software Sources. Go through each tab. Untick, wait a second, then retick any enabled repository. This will cause them to reload and probably fix the 'Merge list' error. (You probably only need to do this with the Extras or Partners repository, but accomplishing it on all won't hurt).

### Post by drs305 on 2012-07-21
Try removing all the lists from the terminal. Be very careful with this command as if mistyped you could erase important system files:
```
sudo rm -vf /var/lib/apt/lists/* && sudo apt-get clean  && sudo apt-get update
```

If your system still is broken please paste the existing error messages.
