### Post by mcduck on 2009-08-31
Most likely you actually removed the Notification Area-applet from the panel, since Network Manager isn't an applet. It's just a program that runs in background and places an icon in the notification area..

Just add the Notification Area-applet back to your panel and you should see the Network Manager's icon (and other notification icons you might have..) appear. :)
