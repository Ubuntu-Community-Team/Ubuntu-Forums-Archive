### Post by emachnic on 2009-11-26
The best way would be to install iTunes on Windows in a virtual machine. If you're doing this to try and sync you're iPod in Linux, I've had the best luck with Amarok and there's actually a YouTube video that discusses that. I think you're going to run into a lot more problems trying to run iTunes on wine than is worth it.
