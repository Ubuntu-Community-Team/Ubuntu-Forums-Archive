### Post by CharlesA on 2010-09-26
PGP is used for encrypting documents and files.

Check out some docs [here]("https://help.ubuntu.com/community/GnuPrivacyGuardHowto").

Assuming you are using Ubuntu Desktop, you can use the "Passwords and Encryption Keys" program.

When it is generating the key, do other things on the machine, else it'll take a long, long time to generate the key.
