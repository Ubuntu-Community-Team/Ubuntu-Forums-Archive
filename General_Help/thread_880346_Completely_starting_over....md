### Post by Taxman415a on 2008-08-04
Just reinstall over the top of it. There's no need to uninstall since if you choose to do so the install process can reformat the entire hard drive or whatever partition you wanted to uninstall.

### Post by mb_webguy on 2008-08-04
When you reinstall Ubuntu, you might want to consider setting up a separate partition for the /home directory so that should you ever need to reinstall (or if you want to do a clean install of Intrepid Ibex when it comes out) you won't lose any of your settings or data.  Your root partition shouldn't need to be but 15GB or 20GB, leaving the rest of the disk for your /home partition.  I have quite a bit of software on my computer, and I'm currently using 16GB of a 20GB root partition.

### Post by mb_webguy on 2008-08-04
Just boot off of the installation disc like you did when you installed it the first time, and do the same thing you did before (except for possibly choosing a different partitioning scheme).  The installer will format your root partition, overwriting your previous installation.
