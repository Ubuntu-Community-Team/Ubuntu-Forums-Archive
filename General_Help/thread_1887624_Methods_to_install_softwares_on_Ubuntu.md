### Post by Lars Noodén on 2011-11-27
The way to avoid trouble is to use the package manager.  There are three common front-ends to the package manager: the Softeware Center, Synaptic and [apt-get](http://manpages.ubuntu.com/manpages/precise/en/man8/apt-get.8.html).  Which one you use is a matter of preference.  The advantage to apt-get is that it works on other Debian-based distros as well as on remote shell access.
