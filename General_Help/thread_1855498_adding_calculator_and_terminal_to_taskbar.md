### Post by Manyette on 2011-10-06
Bring up the function you wish to add by selecting it from Applications, but instead of clicking on it to start it, right click on it and select "Add to panel". Then move it on the panel to where you want it, and right click on it and select "Lock to panel".

### Post by Toz on 2011-10-06
In xubuntu, you would right-click the panel (taskbar) that you wish to add the programs to and select Panel->Add New Items. Click on "Launcher" then "Add". You will get a generic icon on the right-hand side of the panel (taskbar). Close the "Add New Items" dialog.

Right-click the new generic icon and choose "Properties". Click the "+" to add an entry and select the application that you want to add. Close the "Launcher" dialog. The generic icon will change to display the icon of the application you selected. Right-click this icon and select "Move" and place it where you would like it.

Repeat for second application.
