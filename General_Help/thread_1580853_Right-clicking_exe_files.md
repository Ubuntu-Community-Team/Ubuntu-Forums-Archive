### Post by dirghrabadia on 2010-09-24
You might have to change permissions of the exe file to executable.
Here is how you can do it. Go to the terminal and type:
```

chmod 700 *location_of_your_exe_file*

```
Read [http://ss64.com/bash/chmod.html](http://ss64.com/bash/chmod.html) to know what the above lines does :)
