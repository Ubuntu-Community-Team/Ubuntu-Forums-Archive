### Post by TheFu on 2019-05-16
Well, there is no better way to test whether backups are useful than to try to restore them onto a new HDD/system. If it doesn't work, then you know it is time to rethink the backups.

Even if you used exactly the same tools that I use for backups, there is no way to be certain your techniques are correct for both backup AND restore to work.
