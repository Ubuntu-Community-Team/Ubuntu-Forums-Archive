### Post by deadflowr on 2014-05-02
Click on the top icon in the left side panel.
When it opens, type "details", and press enter.

Or run this from a terminal 
Open a terminal with ctrl + alt + T
and type
```
lsb_release -a
```

The first option implies you are running Ubuntu, and not one of the other flavours.
(Such as Kubuntu, Lubuntu, Xubuntu)

The second option should run, regardless of flavour.
