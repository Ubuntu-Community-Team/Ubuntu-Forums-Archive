### Post by cherideng on 2012-03-22
Go to "Device Manager" (from Control Panel-->System-->"Hardware Tab"-->Device Manager) and see if the battery shows up there. And, if it does, check all the properties and if there is a yellow exclamation point by it. You might need to update the drivers.
