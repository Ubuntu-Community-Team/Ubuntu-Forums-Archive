### Post by sudodus on 2014-06-11
Often when a USB pendrive or HDD shows up [COLOR=#000000]in the boot menu as a 'Hard Disk', you can boot, but you need to change the boot order among the [/COLOR][COLOR=#000000]'Hard Disks': put the[/COLOR] USB pendrive at the top of the list :-)

See this link

[https://help.ubuntu.com/community/Installation/FromUSBStick#Booting_the_Computer_from_USB](https://help.ubuntu.com/community/Installation/FromUSBStick#Booting_the_Computer_from_USB)
