### Post by Megaptera on 2011-04-16
Computer Janitor has some pretty powerful default settings so I'd suggest steering clear of it unless you really know what it's going to do.

Also, Ubuntu doesn't need to type of "cleaning" etc that you're used to in Windows - and defragging (in normal usage) is not necessary.

Some simple safe steps explained here: [http://www.stchman.com/cleanup.html](http://www.stchman.com/cleanup.html)
