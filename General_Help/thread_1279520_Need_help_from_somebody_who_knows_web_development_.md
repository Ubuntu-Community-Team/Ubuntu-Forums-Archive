### Post by akakingess on 2009-09-30
You need to be sure and have a seperate folder called something like "images" and/or "media" and put the pictures in the "images" directory and the music in the "media" directory when you actually build the page. so that in the html it will actually say /images/nameofpicture, and /media/nameofsong, then you need to upload the page and the 2 directories. This is the best/cleanest way to go about building it, and there is also (I believe) an option in Dreamweaver to use relative or exact locations, and I would use relative as long as they are all going to be in the same directory on the server (i.e. public_html). That's all I have for now...Please let me know if this does not answer your question.
