### Post by satanselbow on 2011-11-10
There are no native Linux apps to run iTunes.

There are several ways you can continue to use iTunes whilst running linux however.

There is an excellent tutorial how to get iTunes running under wine here: [http://www.ubuntuka.com/itunes-ubuntu-linux/](http://www.ubuntuka.com/itunes-ubuntu-linux/)

Alternatives you may also wish to explore are dualbooting or running a virtual installation of windows on Virtualbox ;)
