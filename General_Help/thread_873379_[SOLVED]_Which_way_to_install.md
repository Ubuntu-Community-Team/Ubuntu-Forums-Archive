### Post by eric3_14159 on 2008-07-28
It is likely that when you begin to install normally, the choices you are given will include using the free space on each disk, which probably will be called sda, sdb, and sdc. The disks should be easy to tell apart, because each will say how full they are. Chances are, the empty one you want to install to will be sdc. Manual partitioning hopefully won't be necessary from there.

Yes, it should set up GRUB to have a duel boot menu.

Hope this helps.
