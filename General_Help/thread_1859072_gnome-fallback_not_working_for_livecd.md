### Post by mc4man on 2011-10-13
What you may have to do to try the classic session on live cd - 
Install as before, the open System Settings > user accounts
Create a new user, _activate that user_, then log out &  log in to that newly created user choosing one of the classic sessions

(to shut down or restart you may need to log out & back into the ubuntu user
