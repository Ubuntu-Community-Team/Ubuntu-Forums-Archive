### Post by louishr on 2008-05-01
once using the installer i used compiz fusion icon to select emerald as my default window decorator. Go into emerald system > preferences > emerald and select the theme it should be in there from the installer. then go System > Preferences > Appearance > Customize in there you can find a theme related option. grab the wallpaper from here [http://interfacelift.com/wallpaper/details.php?id=1145]("http://interfacelift.com/wallpaper/details.php?id=1145") and go System -> administration -> Login Window and select the theme related option under the local tab. that's everything i've done so far
