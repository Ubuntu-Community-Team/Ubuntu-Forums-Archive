### Post by noksagt on 2008-05-06
It is possible, but it still takes some work (you may have to reset the indexing status and/or restart trackerd in order for it to discover the new metadata).  You must first install an extension in Thunderbird.  See: [http://blogs.sun.com/migi/entry/call_for_test_metatracker]("http://blogs.sun.com/migi/entry/call_for_test_metatracker") and also search the forums here & refer to the tracker mailing list for additional comments.
