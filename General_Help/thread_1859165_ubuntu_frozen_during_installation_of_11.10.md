### Post by effenberg0x0 on 2011-10-13
First, make sure your install media is OK. Use Ubuntu instructions to check md5.
[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

Sometimes there are issues with drives pre-partitioned. In other machines it is necessary to use kernel parameters for acpi, limit ram, etc.

You might have to read and see where your setup fits into this.
There's a good description here:
[https://help.ubuntu.com/11.04/installation-guide/i386/boot-troubleshooting.html](https://help.ubuntu.com/11.04/installation-guide/i386/boot-troubleshooting.html)
