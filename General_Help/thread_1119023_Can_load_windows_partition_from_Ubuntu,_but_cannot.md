### Post by coffeecat on 2009-04-08
Try googling 'C:/WINDOWS/SYSTEM32/CONFIG/SYSTEM is corrupt'. That's why I asked for the precise error message. Googling an error message from any operating system is helpful.

Anyway, you need to boot into a recovery console, and for that you need a genuine XP installation disc, not a manufacturer's recovery disc. I know you don't have one, but do you know anyone who could lend you one? Failing that, if your system isn't too badly damaged, you can see if you can boot into a recovery console by pressing F8 at bootup. Or you could try that first. [This was the first link]("http://www.help2go.com/Tutorials/Windows/C:%5Cwindows%5Csystem32%5Cconfig%5Csystem_missing_or_corrupt.html") that came up when I googled the error. It gives instructions for using the recovery console from both the installation CD, and after pressing F8.
