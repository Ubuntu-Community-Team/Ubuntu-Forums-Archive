### Post by anaconda on 2007-09-28
usually the booting order can be set from BIOS. 
If you are a fast reader, the instructions how to get to bios are usually displayed during the boot-process.  Usually you need to press Delete of F2 during the boot to get there. The exact button(s) vary with different manufacturers.

If the machine is really old, then booting from CD may not be possible. For that we used to have boot floppies, which would boot from CD...
