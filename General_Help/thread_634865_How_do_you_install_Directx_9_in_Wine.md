### Post by Jadd on 2007-12-08
Wine already has DirectX 9 "installed". You should never try to install DirectX under wine. See this [FAQ](http://wiki.winehq.org/FAQ#head-fbaa851e07d7484640cc10b6d0c48abc741260b2)
You're out of luck for Casino Inc. The wine application database on this game only records one person who couldn't get it to work:
