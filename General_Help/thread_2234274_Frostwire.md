### Post by gubatron on 2014-07-16
> **lunaservant said:**
> when I left click on a magnet link it automatically opens.  when I right click on it there is no properties button.

I believe this is really an issue of your browser not FrostWire.

In Firefox:
Go to Preferences > "Applications" tab > and you should see "magnet" on the list. Click on "Always ask".

If you're using Chrome, you can change how it handles the different protocol handlers like this
[https://support.google.com/chrome/answer/1382847?hl=en](https://support.google.com/chrome/answer/1382847?hl=en)

(In Chrome)
Settings > Show Advanced Settings > Handlers > 

in there you should see a handler for magnet.

Cheers from FrostWire Developer Team.
