Hi,
I like using the firefox addon called "Open With". It allows me to easily manage opening web pages and documents in as many different applications I want. That addon is very handy because I can right click on a link or on the webpage itself and open it in another application.
[https://addons.mozilla.org/firefox/addon/open-with/](https://addons.mozilla.org/firefox/addon/open-with/)

Xubuntu 11.10 will be using Evince as the PDF reader (also called "Document Viewer" in the menu).The path to open PDFs with Evince will be /usr/bin/evince . I'm not sure what PDF viewer Xubuntu 11.04 is using though, but it might be Evince too.
