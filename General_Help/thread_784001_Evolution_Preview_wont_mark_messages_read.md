### Post by lotsofish on 2008-05-06
This is kind of a weird problem I think. It just started happening the other day. I use the preview feature in Evolution to read my mail. It used to mark the messages as read if you left it on there for 3 seconds or more. Now it doesn't mark them as read at all and I can't find any settings to get it back to how it was. It's annoying to have to manually mark each message as read. I'd rather just have it mark automatically when I am reading them.

### Post by ilrudie on 2008-05-06
In Evolution goto Edit->Preferences.  It will open a new window.  In that window along the left hand side there should be a list of of categories.  Click on the one called "Mail Preferences."  Now to the right of that about a quarter of the way down the window there is a check box labeled "Mark messages as read after" and then a text box.  Make sure that this option is checked and that a reasonable number of seconds is in the text box.  Mine is set to .5 seconds meaning about as soon as Evolution can render the message it marks it as read.
