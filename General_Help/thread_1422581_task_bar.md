### Post by mathay on 2010-03-05
Hi, Dave Otter.

If your top panel still remains, do these steps:

[LIST=1]
[*]Right click and select 'New Panel'. This should create a new panel somewhere on your screen. If it does not start on the bottom, then you can click 'Properties' and select where you want it to be oriented.
[*]After this, I assume you will want there to be a window list. You can do this by right clicking on the panel and selecting 'Add to Panel...' If you scroll-down, look for the option that says 'Window List'. Click on it and select 'Add'. This should add it to the panel.
[*]You can repeat step 2 multiple times to add and remove things from your new panel.
[/LIST]
Hopefully this helped. Let me know how things go.
