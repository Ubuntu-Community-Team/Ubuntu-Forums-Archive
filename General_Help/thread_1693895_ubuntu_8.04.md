### Post by sikander3786 on 2011-02-23
Welcome to the forums :-)

You can upgrade your 8.04 to 10.04 using Update Manager under System > Administraion sub-menu.

You can install 10.04 in dual-boot to 8.04.

You can install 10.04 inside 8.04 using a Virtual Machine Server like VMWare or VirtualBox.

You can't install any Operating System using Wine or any compatibility layer.

You need to tell us the specs of your older machine in order to let us see the real picture of it.
