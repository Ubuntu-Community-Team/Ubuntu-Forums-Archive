### Post by monkeybrain20122 on 2014-06-16
For the switcher to show you have to first enable workplaces (since 13.04 Ubuntu defaults to only one virtual desktop for some strange reason). Click the gear icon on the top right corner, choose System Settings in the drop down, go to Appearance > Behaviour and check the box Enable workspaces then the icon will show up.
