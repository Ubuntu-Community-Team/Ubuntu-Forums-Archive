### Post by blueridgedog on 2010-10-31
> **craig7 said:**
> my netbook has a multi touch trackpad so i can use 2 fingers to scroll(even though it has a scroll thing at the side of the trackpad) and i can pinch my fingers on the track pad to zoom out?

Yes, install xserver-xorg-input-synaptics from the Synaptic Package Manager application...just use the search tool and type in synaptics.

For more information:

[http://www.synaptics.com/solutions/technology/gestures/touchpad-linux](http://www.synaptics.com/solutions/technology/gestures/touchpad-linux)
