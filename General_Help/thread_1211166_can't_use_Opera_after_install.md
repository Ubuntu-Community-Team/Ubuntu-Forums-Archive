### Post by GCoffee on 2009-07-12
Hi Texel,

If Opera runs fine when running on your root account (which you shouldnt do.) Then the solution should be simple.

If you go to:

Applications > Accessories > Terminal

then type in:

```
sudo opera
``` (Someone correct me if i'm wrong in this command, i dont use opera.)

type in your password and the program will open. you can create a desktop shortcut to it as well.

As for ease of use, Ubuntu tries to combine security with user-friendliness. Its almost impossible for linux distros to get viruses and malware. I belive the "barrier" you are talking about is the fact you need to type in your password to launch certain applications & install stuff.
