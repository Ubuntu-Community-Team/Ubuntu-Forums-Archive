### Post by DaithiF on 2009-07-27
as well as a spreadsheet application, Open Office also has a database application which would be a close-ish equivalent to Microsoft Access, which you mentioned you have used.  The database app. is not installed by default on jaunty (at least it wasn't on mine), but the package is openoffice.org-base -- either install it via Synaptic or on the command line with 

```
sudo apt-get install openoffice.org-base
```
