### Post by lykwydchykyn on 2011-11-30
Honestly, if you really  like Kubuntu 8.04 and don't want to upgrade, I don't think there's a real risk in sticking with it.

It's true that there are known, published security flaws in some older versions of the software included in ubuntu/kubuntu.  But a flaw does not equal an exploit.  If the number of people running Ubuntu is not significant enough to warrant malware attacks, the number running a 2008 version of Kubuntu is even less so.

So yes, in theory you're more secure if you upgrade to a supported version.  In reality, I doubt it will make a meaningful difference.
