Hi Sike0Path, I had the same problem trying to install from an ISO written disk. The way I found to install is to use the instructions to put the ISO on a USB stick and install from there. Here is the Link [http://www.ylmf.org/en/how_to_install_ylmf_os.html](http://www.ylmf.org/en/how_to_install_ylmf_os.html).

It is not difficult. You download the Ylmf ISO from the site. Once it is on your desktop you then download a program called Unetbootin, here is link; [http://lubi.sourceforge.net/unetbootin.html](http://lubi.sourceforge.net/unetbootin.html).

After you have installed the above program, you will see at the bottom of the screen a tick box for ISO image, just point the program to where you have the ISO and it will install it to your USB. after that you reboot ensuring that your system boots to your USB (you may need to set that up in the BIOS), although many modern computers give you the option at startup. Then you choose to run it live or Install.

I hope this is helpful.
