about sound:

short version:

but then:

[http://ubuntuforums.org/showthread.php?t=789578](http://ubuntuforums.org/showthread.php?t=789578)

about partitioning:
boot into the ubuntu live cd (or download the gparted live cd)
run gparted, and you can resize partitions as much as you want


ps: resizing partitions can take a while, as you ahve to move a lot of data around, for example, i had to move a 160gb partition and it took 3 days!....

but for 7gb partitions, not too long, but expect a couple of hours
