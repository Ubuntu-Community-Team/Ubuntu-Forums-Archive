### Post by #Reistlehr- on 2007-10-30
Already included.

Just run

sudo apt-get install compizconfig [HIT TAB] for autocompletion.

Then go to System -> Preferences -> Appearance, all the way to the right tab, click it, hit custom, then go to preferences, and configure as you would like.
