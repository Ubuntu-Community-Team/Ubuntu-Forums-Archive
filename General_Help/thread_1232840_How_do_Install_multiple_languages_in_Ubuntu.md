### Post by prshah on 2009-08-06
> **pgtl10 said:**
> I've installed the english version. Now I want to install the Arabic version of Ubuntu. How do I do that?

Use System-Administration-Language support to install additional languages. Then install the relevant keyboard layout from System-Preferences-Keyboard layouts (if not already installed).

You can have it use another language on a per-person basis or a system-wide basis.

### Post by Anxious Nut on 2009-08-06
go to system --> preferences --> keyboard
In layout tab, choose the '+' button to add a language
after you're done choose the language from the list and click on the 'other options' button
from the open window choose 'layout switching', a list will slide don, then choose the keys to switch

look at the screen shot to understand better:
