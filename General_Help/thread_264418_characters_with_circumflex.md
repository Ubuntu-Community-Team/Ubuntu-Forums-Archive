### Post by ryu kun on 2006-09-26
I solved this problem with the help of [this page]("http://www.ubuntu-tr.com/v6/node/390#comment-1430").

Solution:

1. Select a key in **System - Preferences - Keyboard - Compose Key Position**.
2. Hold your compose key while using the dead keys.

So simple. :)
