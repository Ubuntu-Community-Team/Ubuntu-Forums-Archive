### Post by ajgreeny on 2014-03-20
So go to [https://www.virtualbox.org/wiki/Linux_Downloads](https://www.virtualbox.org/wiki/Linux_Downloads), download the correct architecture version for Ubuntu 13.10 from there (top in the list) and double click the .deb file to install it.

Then get the guest additions from [http://download.virtualbox.org/virtualbox/4.3.8/Oracle_VM_VirtualBox_Extension_Pack-4.3.8-92456.vbox-extpack](http://download.virtualbox.org/virtualbox/4.3.8/Oracle_VM_VirtualBox_Extension_Pack-4.3.8-92456.vbox-extpack), double click on that downloaded file and it will open and install itself in VirtualBox.
