### Post by conradin on 2010-04-25
You might have some idea where your free space is going if you're saving a lot of large files. 
```

df-a

```
or
```

du -h

```
can tell you a bit more about where that space is being used up.
Sometimes a log file will become corrupted and take up a lot of disk space.
