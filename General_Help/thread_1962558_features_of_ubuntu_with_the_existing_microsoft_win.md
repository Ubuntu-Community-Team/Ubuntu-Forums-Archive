> **Tewma said:**
> hi,

Can someone explain to me the interoperability features of ubuntu with the existing microsoft windows workstations?

Ubuntu can read/write NTFS filesystems -- the one used by recent Windows versions.

Using Wine, and its variants, it can run SOME MS Windows apps, but they have to be reinstalled in the Ubuntu partition, they can't be run from windows.

That's about it.
