### Post by KingTermite on 2008-06-11
Probably not, because you are only copying "files".

You would need to (at a minimum) copy the MBR too. You can even do better doing a disk duplicate perhaps.

Try this website/info:
[http://www.debianhelp.co.uk/ddcommand.htm](http://www.debianhelp.co.uk/ddcommand.htm)

edit: 
Of course....once you copy, don't delete files from current drive. Test that you can boot up and all is fine, THEN do whatever you want with current/smaller drive.
