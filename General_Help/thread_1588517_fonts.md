### Post by mahmoodkamal on 2010-10-05
I want to read websites in languages other than English, especially, languages such as Urdu and Arabic. However, currently text is not visible correctly in these languages. Can anyone tell me how can i install fonts for these languages?

---

### Post by fragos on 2010-10-05
Run the Synaptic Package Manager and enter ttf- in the Quick search box. This will display a list of all True Type fonts in the repositories. Scan through for the ones you want. Note that clicking a font will display a description including language name. I believe the one for Urdu is ttf-paktype and ttf-arabeyes is one of the choices for Arabic.
