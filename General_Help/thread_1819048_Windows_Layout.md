### Post by syntr on 2011-08-05
First off, none of the 9.x versions are even supported anymore, so that's not a very logical answer to this problem.

As for your issue, open gconf-editor, apps, metacity, general and change the value for button-layout to

```
menu:maximize,minimize,close
```
