### Post by SeijiSensei on 2016-02-01
Double-click on the text tool in the Toolbox to open the settings dialog for the tool.  One of the options is justification.  You can set it before creating the text box.

To align existing text, double-click the text box, highlight the text you want to align, and click a justification button.

Most tools have configuration dialogs that appear when you double-click the tool's icon.  The rectangular select option has very valuable options like size and aspect ratio.
