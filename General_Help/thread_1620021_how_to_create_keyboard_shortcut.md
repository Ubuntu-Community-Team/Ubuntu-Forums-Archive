### Post by sisco311 on 2010-11-12
Ctrl+Alt+L is the default one. You can change it in gnome-keybinding-properties (System -> Preferences -> Keyboard Shortcuts).

---

### Post by deserthowler on 2010-11-12
In 10.10, if you go to **System>Preferences>keyboardshortcuts** there is a listing and a keyboard shortcut for this very thing.  You can use this shortcut or make your own.
