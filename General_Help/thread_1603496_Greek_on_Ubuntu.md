### Post by thunk77 on 2010-10-22
A lot of fonts work with Greek characters. Droid Sans, Deja Vu Sans etc

Basically what you have to do is:

Go to System -> Administration -> Languages

On the "Language" tab, hit "Install/Remove Languages"

Scroll down to "Greek, Modern", and tick the box next to it. Install

Then go to System -> Preferences -> Keyboard

Click on "Layouts" tab and then click "Add"

Find Greek keyboard layout and activate it.

If you want to change the way the system switches inputs, hit "options" button and expand "key(s) to change layout"

Personally I have it like the old Windows default which is Alt+Shift but you can choose what you like from there.

You'll notice that now you have a new indicator on your indicator applet which shows the current layout.

When you wanna write in Greek just switch it and fire away!

This applies system wide, so not only Firefox or OpenOffice works but pretty much everything else (gedit, Nautilus etc)

The default font in Ubuntu has support for Greek (whether it is DejaVu Sans in Ubuntu 10.04 or earlier or Ubuntu Font in Ubuntu 10.10), therefore there is no need to install special fonts.

The best practices for your needs,
1. Install the Greek language support, from System » Administration » Language Support. Click OK to install the addition language support packages. These include a few additional fonts, but most importantly spelling support and program translations.
2. You can add the Greek keyboard layout from System » Preferences » Keyboard » Layouts. See [http://simos.info/blog/archives/888](http://simos.info/blog/archives/888) for more details.
3. Since you are learning Greek, you might want to use Ubuntu in Greek. Since you installed the Greek language support, you can switch between English<->Greek by logging out, selecting language and logging in. This helps when you learn a new language.
4. It becomes messy if you use Synaptic to install packages by hand. You do not need to do such things anymore, and using Synaptic/apt-get may mess up your system.
5. On your question on non-roman fonts; the default font in Ubuntu has been supporting Latin/Greek/Cyrillic for at least the last four years. 

Good Luck!:guitar:
