### Post by TheIdiotThatIsMe on 2006-02-17
Gaim is in the repos for Ubuntu, so all you have to do to download Gaim is:

1) Open the program Synaptic and enter YOUR password when it asks for sudo password.

2) Click search and type in gaim, then click install.

Alternatively:

You could open up a terminal and enter:

```
sudo apt-get install gaim
```
