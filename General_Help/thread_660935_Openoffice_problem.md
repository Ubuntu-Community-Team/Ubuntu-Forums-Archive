### Post by artesian_spring on 2008-01-07
Try looking at my post about the "compose" key on this thread:
[http://ubuntuforums.org/showthread.php?t=636828](http://ubuntuforums.org/showthread.php?t=636828)

Here it is:

> Re: Character insertion 

--------------------------------------------------------------------------------

You might find the "compose key" useful. Go to system->preferences->keyboard and look at the "layout options" tab for the compose key option. I set the right alt key as the compose key on my keyboard; if you have a windows key that you're not using that's even better, imo.
This thread gives more info: [http://ubuntuforums.org/showthread.php?t=480849](http://ubuntuforums.org/showthread.php?t=480849)

With the compose key you can type 
Quote:
<compose> + ' + e  

to get é and 
Quote:
<compose> + ~ + n  

to get ñ, for two examples.
