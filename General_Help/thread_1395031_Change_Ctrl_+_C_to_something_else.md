### Post by Brandon Williams on 2010-01-31
You can do this in such a way that it applies to all gtk applications by creating your own keyboard "theme".

[Here](http://library.gnome.org/devel/gtk/2.13/gtk-Bindings.html) is a page from the API documentation that includes some information about how the keybindings rc file works. You can also look at /usr/share/themes/Emacs/gtk-2.0-key/gtkrc to get some examples of how it works.

Once you've looked this over, post followup questions to get more specific information.
