### Post by bikeboy on 2007-10-05
Install the libdvdread3 and an install script will be downloaded, called "/usr/share/doc/libdvdread3/install-css.sh". Run this script and you should be right.

Alternatively, there is more information in the [Ubuntu Wiki](https://help.ubuntu.com/community/RestrictedFormats)
