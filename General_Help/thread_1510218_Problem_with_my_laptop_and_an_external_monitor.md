### Post by minillinim on 2010-08-24
I know this is an older post but it's the one which comes up in my Google searches so I thought I'd post here.

Had this same problem. I tried many things to fix it none seemed to work. When I set the refresh rate to 75HZ it was better but still not perfect. Finally I started playing with the settings on the monitor itself. Mine is a BENQ and amongst the settings for brightness/contrast etc. there is a "phase" control. After three days of crying over my monitor I fixed the problem in about 15 seconds. Not sure if all monitors have this adjustment but I'm glad mine does.

Hope this helps any future searchers...
