### Post by damis648 on 2009-03-28
Ubuntu? What's it like? Well I would say it's:

Fast (great boot times and very snappy)
Stable (hardly ever crashes, unlike windows (for me))
Clean & Efficient (clean UI, clean system, etc)
Easy to use (from installing the OS to installing software)
Customizable and very open and free (do whatever you want with it)

What it's NOT:

Windows (it can not run Windows software natively, there is an emulator project called WINE, but don't rely on it. You could try a virtual machine though, I recommend virtualbox)
Perfect (don't expect it to be)

So, just try out the LiveCD and play around before you install, see what hardware works OOTB and what doesn't, etc.:popcorn:
