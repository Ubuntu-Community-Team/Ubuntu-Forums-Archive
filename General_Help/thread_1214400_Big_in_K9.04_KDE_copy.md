### Post by krazyd on 2009-07-25
This is the auto-hide feature, which can be disabled by right-clicking the system tray, selecting System Tray Settings > Auto Hide, and then unchecking the 'Automatically hide...' checkbox. You can show the popup again after it disappears by simply clicking the animated icon in the system tray.

Bugs can be reported to bugs.kde.org. Search first though - in case it's already been reported.
