### Post by latna on 2008-05-07
I don't know any gui utilities off hand but there are several command line apps which do the job, eg: html2text and pdftotext.

The naming pattern kinda sucks. Some have 2text, some have totext. You should be able to find more by typing the name of the file format and hitting the tab key and let bash find the commands that start with those first letters.
