### Post by bumanie on 2009-12-11
If you are using grub2 - I don't think you can reinstall grub2 from a live cd as old as 8.10 as it only has grub-legacy on it. Unless anyone else disagrees, you will probably need to burn a 9.10 cd - if you are still running 8.10 or 9.04 and haven't upgraded to grub2, the 8.10 live cd will work.
