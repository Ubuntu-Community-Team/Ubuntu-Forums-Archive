### Post by arapa on 2009-03-13
I found this post through google. For the benefit of those who also find stumble upon this thread through google:

Search for & install gtk2-engines-ubuntulooks in Synaptic Package Manager.

or at the terminal:

sudo apt-get install gtk2-engines-ubuntulooks
