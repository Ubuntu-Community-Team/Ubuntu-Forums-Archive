### Post by unutbu on 2008-08-04
If you are using a banking program called XYZ, it is probably saving the banking certificate in a hidden directory in your home account, possibly under a name like .xyzrc. To look around for so-called hidden directories, open Nautilus, the file browser. If you don't see any files that begin with a period (.) then type Ctrl-H. This will reveal the hidden dot directories and files.

If this does not help, perhaps give the name of the banking program and then we may know or be able to find out.
