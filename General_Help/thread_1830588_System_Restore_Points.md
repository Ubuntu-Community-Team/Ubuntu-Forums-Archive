### Post by Mark Phelps on 2011-08-22
> **spec36 said:**
> Does anyone have a good suggestion for software that will make system restore points of Ubuntu, similar to Windows restore points?

Thanks,

A common misconception in Windows is that Restore Points actually save complete pictures of the state of the PC such that it works much like a "time machine" -- and can be used to restore your PC to an earlier date.

This is not the case.

What Windows really does is make copies of system files and the registry when it does Updates.  When you run System Restore, all you're really doing is replacing some system files and some registry entries with older files and entries.

So basicall, it's really GOOD that Linux does not have an equivalent of System Restore -- because it wouldn't do what most folks would want it to do.
