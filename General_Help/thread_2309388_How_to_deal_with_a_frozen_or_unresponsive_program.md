### Post by Oinos on 2016-01-10
Every now and then a program becomes too slow to respond and next to impossible to close by conventional means. How can I close it in such case? On win xp it was as simple as pressing ctrl alt del, then force closing it, is there any ubuntu equivalent to this?
Another annoying thing is that if a program freezes all computer input becomes VERY slowly responsive, cursor movement included. This makes everything all the more difficult.
Up until now if a program froze I couldn't do anything, but shut down the computer from the turn on button, and I've read this is a no-no.

---

### Post by philinux on 2016-01-10
Open a terminal. Ctrl alt t then type xkill and enter. The cursor will change to an X. Move cursor to frozen app and click left mouse. Done.

### Post by ajgreeny on 2016-01-10
I have a keyboard shortcut to xkill (winkey+Del) which I have not used for a long time, but set up when I was having a problem with something I installed to try out.  It was very useful as the program would freeze totally and nothing in the GUI would shift it except xkill.
