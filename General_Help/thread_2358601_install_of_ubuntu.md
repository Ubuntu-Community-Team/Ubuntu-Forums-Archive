### Post by Bucky Ball on 2017-04-15
Am running [Xubuntu-core]("https://unit193.net/xubuntu/") 16.04 LTS on a HP Stream 11. Beautiful. :)

You can install whatever you need once you have Xubuntu-core installed. I generally start by opening a terminal and installing Synaptic Package Manager (sudo apt install synaptic) then take it from there. Use Synaptic to install whatever else you want/need. This will keep your machine fairly 'bloat-less'. 

(Note: Installing '*buntu-desktop' of any flavour, including Xubuntu, in Xubuntu-core will defeat the purpose of the core install. Installing the desktop package is essentially the same as installing the entire OS, defaults and all; you don't want that. For instance, installing ubuntu-desktop in a core install will give you Ubuntu, more or less.)

A regular Xubuntu would give you a smooth ride (and faster than Ubuntu on that little thing). Have fun.
