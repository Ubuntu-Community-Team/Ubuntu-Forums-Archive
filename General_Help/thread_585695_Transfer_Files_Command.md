### Post by soluphobe on 2007-10-21
I use ```
cp <file> <destination>
```
although most of the time I add 'sudo' to the beginning because I need root privilege.  For example:
```
sudo cp usb8023.sys /etc/ndiswrapper/wusb54gsc
```
is one I've gotten used to typing.
