### Post by angry_johnnie on 2008-06-02
Nautilus is more than just a file manager.  It also handles the desktop.  If you want to just open nautilus you could, from a command prompt, do this:

```
nautilus --no-desktop
```

It will open nautilus as a file manager only, without having it mess with the desktop itself.
