### Post by PhantomTurtle on 2012-05-26
Run the following command in terminal,
```
sudo apt-get remove cinnamon
```

If there are any dependancies left over just run,
```
sudo apt-get autoremove
```
(You can just run it even if your not sure, it won't damage your system or remove anything that is needed)

EDIT: Oh if you are using Linux Mint you first need to run the following in terminal FIRST,
```
sudo apt-get install unity
```
(DO THIS FIRST IF YOU ARE USING MINT)
Disregard this part if you are running Ubuntu.

### Post by PhantomTurtle on 2012-05-26
To disable auto login in a virtual console, you first press Ctrl+Alt+F2. This will take you to a virtual console. Log in to it. Next you want to disable auto login in the lightdm configuration file. So type in the following,
```
sudo nano /etc/lightdm/lightdm.conf
```
Press down to move down the document. You should see something like this,
```
[SeatDefaults]
autologin-guest=false 
autologin-user=username
autologin-user-timeout=0
autologin-session=lightdm-autologin
greeter-session=unity-greeter
user-session=ubuntu

```

Put a # next to each one that says autologin.
Next press Ctrl+X to exit nano and when it asks to save press S. Now type in,
```
sudo shutdown -r now
```
This will restart and get you to the lightdm login screen.
