### Post by lisati on 2009-11-26
The Live CD shouldn't need a password. As for the "root" password, look here: [https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)
As has already been pointed out, you should have provided a user name and password as part of the installation process. Remember, it is case sensitive, so "user" is not the same as "User".
