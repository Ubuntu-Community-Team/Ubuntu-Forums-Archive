### Post by Butthead on 2011-05-28
The "apropos" command helps you search manual pages by keyword too.

For instance if you have a question about formatting a disk, typing "apropos formatting" in the terminal will bring up a list of revelant MAN pages that you then can choose (hopefully the most appropriate one) from.

### Post by WorMzy on 2011-05-29
```
man -k <search term>
```

Works the same way as apropos (and is much easier to remember imo).
