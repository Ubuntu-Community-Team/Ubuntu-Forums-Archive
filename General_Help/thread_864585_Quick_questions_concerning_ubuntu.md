### Post by jonlemur on 2008-07-19
While knowledge of coding and computing would certainly help run any OS, I think that Ubuntu can be very easy for someone to pick up who doesn't have much experience with either, especially for everyday tasks like email, web browsing, writing documents, listening to music, watching videos, etc.

As for games, I know of a lot of people who run modern games like Guild Wars and WOW with very little difficulty by using a program called wine.  Getting wine configured properly can require a bit more understanding of coding and computing, but I think there are some very good HOWTOs on this forum that would be pretty easy to follow, especially for popular games.  I'd recommend searching this forum for the game you have, or doing a google search for it with Linux, and see if people have it working in Linux.
