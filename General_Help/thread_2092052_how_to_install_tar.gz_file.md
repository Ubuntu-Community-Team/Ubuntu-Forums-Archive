### Post by picaflor on 2012-12-06
Hi,

  On the command line do this:

tar -zxvf file.tar.gz

It's a good idea to do this inside of a tmp directory, because if the tar file is not set up properly it can spew tons of stuff into your home directory which can be very troublesome to clean up. Then cd into the directory created by the tar command, and find the readme.
