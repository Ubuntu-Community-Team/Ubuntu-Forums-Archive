### Post by sanderd17 on 2011-10-12
Well, there are lots of different desktop environments.

The default for Ubuntu 11.04 is called Unity. And what you're using now is Gnome (version 2).

But the gnome team worked on a new version (the past years), and Gnome 3 (with Gnome-Shell on top of it) is the result: [http://images.google.com/search?q=gnome%20shell&tbs=imgo:1&biw=1600&bih=776&tbm=isch#hl=en&tbs=imgo:1&tbm=isch&sa=1&q=gnome+shell+3.2&pbx=1&oq=gnome+shell+3.2&aq=f&aqi=g1g-S4&aql=&gs_sm=e&gs_upl=7242l7939l0l8190l4l4l0l0l0l0l203l556l1.2.1l4l0&bav=on.2,or.r_gc.r_pw.r_cp.,cf.osb&fp=9bacd570c59b7c06&biw=1600&bih=776](http://images.google.com/search?q=gnome%20shell&tbs=imgo:1&biw=1600&bih=776&tbm=isch#hl=en&tbs=imgo:1&tbm=isch&sa=1&q=gnome+shell+3.2&pbx=1&oq=gnome+shell+3.2&aq=f&aqi=g1g-S4&aql=&gs_sm=e&gs_upl=7242l7939l0l8190l4l4l0l0l0l0l203l556l1.2.1l4l0&bav=on.2,or.r_gc.r_pw.r_cp.,cf.osb&fp=9bacd570c59b7c06&biw=1600&bih=776)

As you see, this is completely different. Gnome 2 won't be available anymore in Ubuntu 11.10. The gnome team also made a fallback mode, which looks a bit like gnome 2. I don't use fallback, but I've seen people that could just get the same features as they got on Gnome 2 (task bar, menu ...)

Now, that was all the gnome side (Unity also uses Gnome as a base). But there are other ones.

KDE is a big competitor for Gnome. KDE has a lot more bling (so heavier for your computer), and tries to follow more of the Windows rules (a start menu, a task bar that looks like in Windows ...

Then, two other well known are XFCE and LXDE. XFCE started as a light desktop environment, but currently it's about as heavy as Gnome 2. But it also has most features from Gnome 2. So if you look for a replacement of Gnome 2 in Ubuntu 11.10, XFCE comes very close. Finally LXDE is a young but very light desktop environment (in all ways: memory usage, processor usage, battery usage ...). I use it when I'm on battery. Because LXDE is so light, it does have less configuration options.

Apart from those, there are also a lot of small window managers and desktop environments, like openbox, flukebox, herfstluftwm ...
