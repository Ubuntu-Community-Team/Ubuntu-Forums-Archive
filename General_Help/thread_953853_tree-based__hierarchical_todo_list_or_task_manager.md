### Post by davescafe on 2008-10-20
Does anyone have any recommendations for a tree-based / hierarchical todo list or task manager? I know there are a lot of to do list programs out there (like Mozilla Sunbird and the one that is included with Evolution), but I am looking for something that will allow me to set up precedence or dependencies. Unfortunately, I have not been able to find a program that does this for Linux. 

For example, instead of a task list that reads like this:
[INDENT]1. Back-up computer
2. Install Ubuntu
3. Write thesis[/INDENT]

It might look more like this:
[INDENT]1. Back-up computer
[INDENT]1.1. Install Ubuntu
[INDENT]1.2. Write thesis [/INDENT] [/INDENT] [/INDENT]




There is an example of one here:
[INDENT][http://tobedone.sourceforge.net/]("http://tobedone.sourceforge.net/") [/INDENT]
Unfortunately, this app requires a server-based setup. I am looking for something that I can run stand-alone, on a desktop or a laptop.

Another excellent-looking program is described here:
[INDENT][http://solomonson.com/portfolio/todo]("http://solomonson.com/portfolio/todo")[/INDENT] but this program does not appear to be available.

If anyone has any suggestions, I would be grateful to hear them,

Dave

I didn't end up using Diligent. Although it technically fit my requirements I was not satisfied with using it.

When I started my search, I was looking for one tool that I could use for both my personal tasks and my work projects. In the end, I couldn't find one tool that was suited to both situations. For work projects, I seriously explored Gnome Planner and OpenProj. Both seem to be excellent project management tools. OpenProj is more of an MS Project clone, with more features than Gnome Planner but the interface is less attractive. I ended up deciding on Gnome Planner because it had all that I needed and it seemed to be easier to use.

For my personal tasks, I ended up going with the online (web-based) tool Remember The Milk : [http://www.rememberthemilk.com]("http://www.rememberthemilk.com"). This tool is free of charge and seems to be well-supported.
