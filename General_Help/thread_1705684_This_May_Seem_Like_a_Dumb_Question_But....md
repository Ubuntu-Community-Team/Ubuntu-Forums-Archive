### Post by lithopsian on 2011-03-12
Yeah, it's dumb :P  Sorry, couldn't resist!  Load what you like and keep your fingers crossed because netbooks often have weird graphics cards.

What is really dumb though is using Unity desktop for the netbook edition of Ubuntu, because it uses massive icons all down the side (and top) of the (tiny netbook) screen and Unity itself is a real dog while netbook CPUs are slower than an old Celeron.

So there's my review.  Only install the netbook edition if you have brand new top of the line hardware.  Otherwise install the desktop edition, or maybe even go for a lightweight distro.
