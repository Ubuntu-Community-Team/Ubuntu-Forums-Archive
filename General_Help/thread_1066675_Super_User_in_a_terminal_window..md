### Post by Koil_1 on 2009-02-11
This post is for those of you who are fighting with su in a terminal window. There is an easy solution to this problem. For those of us who use Ubuntu we can simply go into the system menu, then preferences and select "Main Menu." From there we can enable the "Root Terminal" in our Applications menu. With a root terminal open it's not necessary to use the sudo command because we already have super user permissions. This makes aptitude commands like apt-get a lot easier to manage.

 I struggled with this myself for quite a while. I know many of us have dealt with our passwords not being recognized in a terminal when we try to do something that requires it. This was something I stumbled upon while trying to get another program to appear in my main applications menu. Because I know how much it helped me I thought it only appropriate to post it so others could benefit from it as well.
