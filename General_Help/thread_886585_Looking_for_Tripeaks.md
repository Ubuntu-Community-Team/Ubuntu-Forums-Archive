### Post by mb_webguy on 2008-08-12
> **wedoo2 said:**
> adewale

How does that work?  I assume that wine is some type of Windows emulation?

Without going into the technical stuff, Wine is a sort of compatibility layer for Windows programs on Linux.  It's not an emulator or virtual machine, so it won't slow down your computer any.  In fact, I've known some programs to run faster on Linux using Wine than on Windows.

Wine is in the regular repositories, but you can add the Wine HQ repository and get the most up-to-date version by following the instructions on the [Wine HQ website]("http://www.winehq.org/site/download-deb").

Once you have Wine installed, you can install most Windows software just as you would on Windows (although .exe installers seem to work better than .msi installers).
