In Firefox go to:

Tools--->Add-ons then click Extensions and highlight the Ubuntu Forums Menu.  Click Options and check Add menu to toolbar.

You will then have a menu item called Ubuntu Forums.

Alternately you can access the Ubuntu Forums menu by right clicking in Firefox.

Hope this helps.
