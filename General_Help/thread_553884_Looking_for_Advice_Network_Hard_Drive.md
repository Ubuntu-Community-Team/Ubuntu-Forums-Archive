### Post by reiffert on 2007-09-18
I'm using a Maxtor Shared Storage Drive.  Mine is a 200Gb and it looks like this is their newer version   ([http://www.maxstore.com/product.asp?sku=3338658](http://www.maxstore.com/product.asp?sku=3338658))  now up to a 1T.

My basic install (7.04) sees it through Network and moving things is pretty fast.

I did the Open MSS software install and have installed Twonky server for music broadcasting over network (Netgear and LinkSys players) so has been running basically constantly - and has been for a year or more - other than shutting down whole network for hottest days or moving/cleaning room, etc.   There is a bug in the Open MSS software that makes the fan run continuously, but I've had no problem with the fan.

HTH,
Robert
