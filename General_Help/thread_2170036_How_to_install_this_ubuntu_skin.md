### Post by SweetAurora on 2013-08-24
That theme is outdated and is no longer compatible with any modern GTK based desktop. The theme only has files for GTK2 and no GTK3 files. It can technically be installed, but only apps still using GTK2 will be themed. The GTK3 apps will look like Windows 95.

### Post by SweetAurora on 2013-08-24
It's not a good idea to install. GTK3 apps outnumber GTK2 apps on Ubuntu 12.04. So only a few apps like Firefox would be themed. While apps like Unity, Nautilus (file manager), Rythembox, system menus, ect... will look like crap. So in reality you would seldom even see the theme.

The closest thing to Windows XP that you might wat to check out is Linux Mint. It is very XP esque in design. Suggestion for themes. For the Cinnamon Shell (desktop) Baldr, GTK theme Orion, Icon theme Faenza. It will look sleek, modern, but refined and simple.

Edit: Also to note, if you do finally pick  a GTK3 theme, be sure it is the right version! Every time Gnome updated GTK3, they broke themes. So only themes written for a certain version will work for that version. So in 12.04, you wil have to pick a GTK 3.2 theme to have it apply properly.
