### Post by autonomy on 2009-11-16
looks like this is going to work for me. I was doing it wrong. I tried before, but tonight, I started by trying to install the package. Later, I undid everything I had done. The best way to install it looks to be going to the electricsheep.org site and downloading the makesheep.sh to your home directory like it says and then running . makesheep.sh, entering your administrator password and saying y three times to the automatic installations. I ran electricsheep from the run command line instead of the terminal, and it took so long to do something that I assumed it wasn't going to do anything. But a little preview finally popped up and impressed the pants off me. The electricsheep-preferences didn't save, so I opened preferences.xml in a text editor, put my desired nickname in between the double quotation marks and saved it that way. I'll leave it running overnight like the FAQ says with my power management turned off so the monitor stays on and see if the electric sheep fill the screen in the morning. \\:D/
