### Post by Forlong on 2008-05-28
Run this command to reset your xorg conf:
```
sudo dpkg-reconfigure -phigh xserver-xorg
```
You can do a backup of your current one before if you like:
```
cp /etc/X11/xorg.conf ~/xorg.conf.bak
```
Afterwards, restart X and run this: [http://ubuntuforums.org/showthread.php?t=799070](http://ubuntuforums.org/showthread.php?t=799070)
