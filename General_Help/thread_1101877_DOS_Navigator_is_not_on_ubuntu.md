### Post by kanikilu on 2009-03-20
If you don't like nautilus, the default file manager with Gnome, you might want to try some alternatives. Just search for Ubuntu (or Linux) file managers. One cross-platform alternative I've personally tried and like is called muCommander:

[http://www.mucommander.com/](http://www.mucommander.com/)

For antivirus, you're about to get a bunch of people in here telling you that you don't need it, but, ***opinion*** aside, if you want a virus scanner, you can use Avast! Linux Home Edition:

[http://www.avast.com/eng/avast-for-linux-workstation.html](http://www.avast.com/eng/avast-for-linux-workstation.html)

Or, an opensource alternative is ClamAV:

[https://help.ubuntu.com/community/ClamAV](https://help.ubuntu.com/community/ClamAV)

### Post by mb_webguy on 2009-03-20
You don't need antivirus software unless you're dual-booting or running some sort of email or file server and want to avoid passing viruses on to Windows computers.  Linux doesn't really get viruses.  Check the link in my signature or [this thread]("http://ubuntuforums.org/showthread.php?t=510812") for more information.

That said, your best bet would be to install the clamtk package.  This is a nice GUI front-end for ClamAV, which is probably the best Linux-native antivirus available.
