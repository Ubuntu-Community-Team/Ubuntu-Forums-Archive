### Post by mb_webguy on 2009-03-14
If you install the p7zip-rar package instead of the unrar package, it will provide support for several other archive types in addition to rar archives.

The rar and p7zip-rar packages work as plugins for File Roller, the default Archive Manager for Ubuntu.
