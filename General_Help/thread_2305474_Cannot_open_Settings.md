### Post by grahammechanical on 2015-12-06
What derivative of Ubuntu are you installing? If this was Ubuntu then there would be an icon for System Settings in the Launcher that is on the left side of the screen. And then, also, System Settings can be accessed from the Power/cog drop down menu that we use to shut down Ubuntu.

In Ubuntu, if we right click the background wallpaper and select Change desktop background, that will open System Settings at the Appearance tab. And then by clicking the All Settings button we get to the System Settings front page window.

We can also search for System Settings in the Dash.

Derivatives of Ubuntu that have different User Interfaces do things differently. And your talk of menus has got me wondering if this is Ubuntu because Ubuntu does not use menus as much as it once did.
