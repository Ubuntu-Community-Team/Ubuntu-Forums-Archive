### Post by thelinuxguy on 2007-05-17
You can use ASP.NET Enterprise Manager. Its not really a *Ubuntu* tool since you will need to install it on the database server. It provides a web based interface similar to the Enterprise Manager that comes with SQL Server 2000. So the client OS doesn't matter.

[http://sourceforge.net/projects/asp-ent-man/](http://sourceforge.net/projects/asp-ent-man/)

Hope this helps.
