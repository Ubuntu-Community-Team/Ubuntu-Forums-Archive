### Post by Verbeck on 2010-10-31
i'll guess that the first one is nautilus- elementary
to get that, run the following in a terminal :

sudo add-apt-repository ppa:am-monkeyd/nautilus-elementary-ppa
sudo apt-get update

then open up update manager then select the updates from the nautilus-elementary-ppa

hope this helps
