I'm sorry you're having problems.  Ubuntu is a friendly operating system, but if you're having significant difficulty with it you might want to try something else.

If you want to reinstall the operating system from scratch, go to:

[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

you will download a large ISO file and burn it onto a cd rom.  Then you'll put the CD into your computer, restart and boot from the CD-rom.  You'll reformat your hard drive and start from scratch.  EVERYTHING will be gone, including all of your personal files, settings, bookmarks,  passwords, you name it.  There will be much more detailed instructions on the download site.
