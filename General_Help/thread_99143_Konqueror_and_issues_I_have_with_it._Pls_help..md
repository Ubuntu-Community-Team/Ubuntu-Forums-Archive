I don't think you can hide bookmarks on demand but as you have bookmarked these web sites I assume you want to use them at some point. What you can do is add another bookmark as "sites I don't use often"and within this create sub folders for the sites you want to remove from your tool bar. Thus for all these unwanted bookmarks are all in one folder ready to use when you want.
