Do you know if Ubuntu 10.10 supports S3 sleep state?
***[SIZE=2][FONT=Times New Roman]S3 Sleeping State[/FONT][/SIZE]***
[FONT=Times New Roman][SIZE=2]The S3 sleeping state is a low wake latency sleeping state where all system context is lost except system memory. CPU, cache, and chip set context are lost in this state. Hardware maintains memory context and restores some CPU and L2 configuration context. Control starts from the processor’s reset vector after the wake event.[/SIZE][/FONT]
