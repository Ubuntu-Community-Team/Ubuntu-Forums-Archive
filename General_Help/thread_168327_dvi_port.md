### Post by brihas on 2006-06-07
In your xorg.conf file listed below, the line that enable DVI is commented out:
#       Option "connectedmonitor" "dfp"
Also per ([http://www.linuxforums.org/forum/redhat-fedora-linux-help/20704-xorg-conf-help-nvidia-dual-head-dvi-vga-2-lcds.html](http://www.linuxforums.org/forum/redhat-fedora-linux-help/20704-xorg-conf-help-nvidia-dual-head-dvi-vga-2-lcds.html)) you need to put this line in the Section "Device" not in the Section "Screen" as you have done. 

I have done this and now my DVI out works, but as mentioned in the above link the VGA is now off. No big deal for me as I don't need both working at the same time.

Also there is no need to comment out the Load "dri" in the Section "Module". I tried with it on and off and both work, so I figure why not leave it on :-)
