### Post by Leppie on 2009-12-04
If you download the iso of the latest version (9.10) and you install it, that would be an upgrade.
Updates are just corrections on the version you are using (9.04 as you stated). Installing updates is always recommended.

Upgrading depends on if you want to use the latest features not available in the version your using.
