### Post by oldos2er on 2012-01-04
This is a known bug ([https://bugs.launchpad.net/ubuntu/+source/qapt/+bug/833058](https://bugs.launchpad.net/ubuntu/+source/qapt/+bug/833058)), but if you start muon with ```
kdesudo muon
``` it should work.
