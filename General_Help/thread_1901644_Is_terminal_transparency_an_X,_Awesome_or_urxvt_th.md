### Post by 2F4U on 2011-12-29
If you are using Gentoo, you should know how to do this. There are two kinds of transparency. True transparency, where the window manager supports compositing or there is a separate compositor and native transparency or fake transparency.

[http://ctkarch.org/documentation/tutorials/tuto.php?page=urxvt-transpa.xml](http://ctkarch.org/documentation/tutorials/tuto.php?page=urxvt-transpa.xml)
