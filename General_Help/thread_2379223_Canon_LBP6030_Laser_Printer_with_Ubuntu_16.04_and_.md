### Post by davidsrsb on 2017-12-02
I followed the advice in [https://ubuntuforums.org/showthread.php?t=2365535&highlight=Canon+LBP6030+Laser+Printer+16.04](https://ubuntuforums.org/showthread.php?t=2365535&highlight=Canon+LBP6030+Laser+Printer+16.04) and failed to install with many dependency errors.
Looking at the text generated by install.sh, I saw many rpm related errors. Reading install.sh showed that the Canon driver install script was assuming that a machine with rpm installed was running Fedora.

So uninstall rpm, alien etc and the Canon script then detects Ubuntu as a Debian based distribution and installs.
The rpm and alien were there from my previous HP Laser driver.
