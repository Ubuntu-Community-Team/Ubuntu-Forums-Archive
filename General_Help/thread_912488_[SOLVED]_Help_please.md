### Post by scouser73 on 2008-09-06
Go to the Adobe site [http://www.adobe.com/products/acrobat/readstep2_allversions.html]("http://www.adobe.com/products/acrobat/readstep2_allversions.html")  in the Select an Installer drop-down menu, select  Linux - x86 (.deb)

Save somewhere on your pc, once the download has completed, go to where you saved it, then click on the .deb file and it will start installing itself.
