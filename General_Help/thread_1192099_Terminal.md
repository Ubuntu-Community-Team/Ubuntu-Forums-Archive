### Post by rampageoberon on 2009-06-19
Yes you can browse via Terminal.

You need to use browsers such as links, links2, elinks, lynx etc. They are all available from the repositories so

```
sudo aptitude install elinks
```

I personally prefer elinks. Hope that helps

### Post by m_duck on 2009-06-19
There is a cli web browser called links which I think is installed by default. Read the man page for info. You can launch it with a website as an argument too, e.g. **links [http://www.google.com](http://www.google.com)**. There are a host of others too with more/different features like, links2, elinks, lynx to name a few.
