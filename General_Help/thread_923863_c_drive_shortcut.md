### Post by Pro-reason on 2008-09-18
What are you referring to exactly?

Have you created a link to a partition (called “C:” by Windows) on your Ubuntu desktop, only to find that your Ubuntu desktop is reset to default when you reboot?

If so, that's a strange problem.  Perhaps you didn't actually create a link, but you're talking about the icon which automatically displays on the desktop whenever extra partitions are mounted.  If that's the case, then simply remount the partition in “Computer”.

This could also happen if you are running from the live CD rather than an actual Ubuntu installation.  In that case, install Ubuntu.
