after lot of trying i found out now how to:

-get fresh install on other partition
-copy grub.cfg from there
-sudo update-grub
and then add:

```
menuentry "Win7" {
    insmod ntfs
    set root=' (hd0,1)'
    search --no-floppy --fs-uuid --set 366A50DF6A509E05 
    chainloader +1
}
```

replacing (hd0,1) for where your partition is and 366A50DF6A509E05 with the name of yours.
