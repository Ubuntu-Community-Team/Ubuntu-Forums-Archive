### Post by fragos on 2008-07-04
I don't have experience with USB WiFi but do with the PCI builtin variety. In that case the most important thing to check is which WiFi chip set is used. The Intel chip set in paraticular works without any trouble at all. Others like broadcom may require a bit of work to get going. Do a search by model name in this forum to see if there are any comments +/- on each of those units.
