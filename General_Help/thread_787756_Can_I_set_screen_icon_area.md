### Post by I3aH on 2008-05-10
I don't think you can. Never seen anyone do it before. Usually i drag my icons in certain places and they stay there until i move them again.

fballem, His was probably done the same way I did mine.

First go to [http://www.gnome-look.org/](http://www.gnome-look.org/)

I chose my icon that you want. Once you have an icon, you can go to System > Preferences > Appearance. Drag the icons that you have downloaded into there. If they don't automatically change you can click on Customize and go to icons and select the one you downloaded. The rest is done the same way, like Window Border, pointers etc...

Next just download AWN (Avant window navigator). There is a lot of tutorials for that in here.

His though has "screenlets" i think? You can 
"sudo apt-get install screenlets"
but there is a lot of tutorials out there you can use. I don't use screenlets since i play games and like to keep my processor and memory low.

Here is a screenshot of mine. I used "Glossy P" for the Controls. "Alphacube 0.9b Metacity Color" for the window border. "black-white 2 gloss" for Icons. All of them were from [http://www.gnome-look.org/](http://www.gnome-look.org/) even the blackground. Oh and yes, it's sorta a mac look because i have used a mac for so long i'm just use to it. So, about 90% is just point and click so i'm sure you could do it :)
