### Post by ajgreeny on 2010-07-06
Which flash plugin are you using?

There are a number of different options for installing flash, three of which end up with the Adobe version, but others which will give you an open source version called gnash or swfdec-mozilla.

If you have either of those two mentioned last, I suggest you uninstall them and install the **flashplugin-installer**, or if you have the partner repos enabled install **adobe-flashplugin**.
