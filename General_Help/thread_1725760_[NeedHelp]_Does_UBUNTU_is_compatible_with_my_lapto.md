### Post by dragonfly41 on 2011-04-10
You can follow the LiveCD option (in demo mode but _don't_ go for a full installation).

And you can consider several other approaches ..

You have three USB 2.0 ports so you could install ubuntu on an external bootable USB drive and boot to USB drive with ubuntu in F12.   This avoids changes to your laptop.

Or you could decide to install internally as dual boot .. but in this case it might be safer if you first reduce the size of your Windows 7 partition to leave some unallocated space for a new partition for ubuntu.

After you check that this works and Windows 7 reboots o.k.

Now create a new extended partition (not primary partition) and in the extended partition create

an ext4 partition for ubuntu
an ntfs partition for ubuntu data (which allows ubuntu data files to be seen by Windows)

You can resize your windows 7 partition and add an extended partition using Parted Magic burned in a bootable CD.

[http://partedmagic.com/doku.php?id=downloads](http://partedmagic.com/doku.php?id=downloads)
