### Post by Catsworth on 2007-03-28
Nothing to do with their web site.  All of the software they provide on disk as part of your courses will be Microsoft/Mac compatible.

I'm doing an OU course at the moment and I'm using Ubuntu exclusively. 

The only two bits of software I've had to install have been 1) BlueJ (a Java IDE), I oculdn't install from the OU disc so I had to download from the developers' web site and then play to get it working, and 2) FirstClass (a conferencing, email, and chat client used for communicating with your tutor and tutor group), for which there is a Linux version so no problems there.

---

### Post by Catsworth on 2007-03-28
Regarding your post about 'support'.....

Within FirstClass you have access to a number of different OU user groups, one of which is a Linx based group and there's a lot of support there, there are also a number (quite a large number) of people on other courses who are using Linux and are usually more than happy to help.

Before you undertake a course it's probably worth finding out what the course software is and then checking that you can get it working in Linux before you stump up any money.
