### Post by cariboo on 2009-10-22
You can remove the vga=769 fromthe kernel lines in /boot/grub/menu.lst. To dso this you need to edit menu.lst as root. Press Alt-F2 and type:

```
gksu gedit /boot/grub/menu.lst
```

I would suggest creating a baackup before doing any editing. Open a terminal and type:

```
sudo cp /boot/grub/menu.lst /boot/grub/menu.lst.bak
```

The above command will create a backup called menu.lst.bak
