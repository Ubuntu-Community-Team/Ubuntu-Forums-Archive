### Post by Vunutus on 2009-06-08
It doesn't lose everything, but just seemingly a few settings at random from each widget. Some are perfect and keep all settings between logins, some lose position/stickyness, some lose color settings, and others completely reset themselves. How can I fix this? It's very annoying to have to set things up again every time I log in or reboot (even though it isn't a huge problem since I usually go weeks or months at a time without logging out).

---

### Post by lovinglinux on 2009-06-08
Try closing the Screenlets Manager after setting each screenlet properties (configure the screenlet, close the manager, open it again and move to the next screenlet).

This solved the same problem with my sensor screenlets.
