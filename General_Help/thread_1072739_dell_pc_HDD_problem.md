### Post by barfy on 2009-02-18
having done this several times before perhaps I may be of some assistance.  first off the dell cabling is very sensitive so insure that your primary drive is connected to the HDD1 end of the cable and set the drive as a master with slave, then place second drive on connection for HHD2 and set jumper for cable select. Most people use the slave setting but I found that it causes the problem that you have been having. hope this is of some help.
