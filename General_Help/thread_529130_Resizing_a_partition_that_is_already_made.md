### Post by merlinus on 2007-08-18
Before attempting ubuntu install, delete temp and other unneeded files, backup data, set virtual paging to zero (set it back after install), and defrag several times.

Then whilst installing, you will be asked at the partitioning menu what to do.  Do not select to use entire disk but to resize the windows partition and use the freed-up space.

-merlin
