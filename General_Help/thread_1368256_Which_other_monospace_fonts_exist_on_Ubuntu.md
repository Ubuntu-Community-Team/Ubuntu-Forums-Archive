### Post by Brandon Williams on 2009-12-30
Many GUI applications allow you to select fonts. You can browse fonts in the dialog used to select a font. For example, open gnome-terminal, go to Edit -> Preferences -> Appearance, and then open the font selector dialog. In the 'family' box, look through the listing for any font with the word 'mono' in the name. There will likely be a few that are missed this way, like Courier, but you'll get a mostly complete list of monospace fonts that way.
