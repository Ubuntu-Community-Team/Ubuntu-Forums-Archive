Open synaptic and download/install the EnvyNG package.  It will install a menu entry for itself under applications > system tools 

Once started, select ATI on the left panel and then check the "install ATI Driver (automatic...) radio button, then click on the "apply" button. 

It's the easiest and most reliable way I've seen to install the ATI driver package.

### Post by z@ch on 2008-08-12
Hi

I've got also the Ati HD3450 and am running Ubuntu 8.0.4.1 (64bit)
I had the same problem like you but I could solve it to see the ubuntu desktop again :-)
You just have to press ESC when during the reboot as the GRUB message is shown. Then you select "ubuntu kernel (blablabla) recovery mode". and in the recovery mode you can select the option "fix x server". this solved it for me.
