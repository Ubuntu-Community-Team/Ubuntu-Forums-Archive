### Post by o_fortuna on 2006-02-08
[QUOTE=HokeyFry]Hi,

When I try to download packages using firefox, it either tries to display it as a text file (debs) or tries to play it with the mplayer plugin (rpms). does anyone have a solution to this?[/QUOTE]
Click Edit-->Preferences, then click on the downloads tab, then click "View and Edit Actions." Make sure that .deb and .rpm aren't listed to be automatically opened.

If that doesn't work, you can always right click on the file and select "Save Link As" and save it to the desktop. Then, right click the file and select properties. Then select the open with tab and make sure text editor/mplayer isn't one of the things listed. Even if that doesn't stop firefox from opening the files, you can still save them to the desktop.
