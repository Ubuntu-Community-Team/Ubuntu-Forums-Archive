### Post by CatKiller on 2008-11-16
Have you got Desktop Effects enabled? I came across a problem with one of the plugins that when the "busy" animation should have been displayed, the cursor disappeared instead.

The setting was either the "Scale the mouse pointer" or "Dynamic mouse pointer scale" (I can't remember which at the moment, I'm afraid) of the Enhanced Zoom Desktop plugin. You can change this value in CompizConfig Settings Manager. You can install this program if it isn't already with ```
sudo aptitude install compizconfig-settings-manager
```

Hope this helps.
