### Post by ajgreeny on 2014-02-03
If you choose to use Fedora in dual boot with Ubuntu, make sure that you install into normal partitions, not LVM which I think it uses by default, unless you're prepared to get all the libraries and packages needed for Ubuntu to be able to mount LVM.

Apart from that, I have used both at various times, just to look-see how they are, and I don't see much difference between them either.  They are both RPM distros, each with their own package manager, neither of which in my opinion is as good as the many apt front-ends, such as synaptic or USC,  but that is the only real criticism I have, and no doubt I would get used to that very quickly if I had to.
