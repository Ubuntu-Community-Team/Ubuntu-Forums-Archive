Hi,
I had a problem with Evolution email client application few days ago. Evolution couldn't sync data with my Inbox at local file system. There is a pop-up window which informs you about problem and only what you can do is to click on button OK.

To solve this issue with Evolution email client, you can do next: - From top menu in application choose Backup settings. - Backup settings and restore them at once, when you asked for. - From that moment application starts working without a problem.
