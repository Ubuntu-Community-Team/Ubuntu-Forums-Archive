You can use Acronis True Image. This software is soooo nice, but you will run into troubles when trying to use both hard disks (old and new) in the new system. Not Acronis' fault, but have a look at the thread [http://ubuntuforums.org/showthread.php?p=4208550](http://ubuntuforums.org/showthread.php?p=4208550) for more information!
