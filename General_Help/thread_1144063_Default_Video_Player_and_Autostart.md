### Post by mschraier on 2009-04-30
I am using Kaffeine in 9.04. HOw do I go about making it my default video player instead of Dragon Player? Also, how do I get DVD's to auto start?

---

### Post by Monsieur Gonzalez on 2009-04-30
If you don't use it, you might uninstall Dragon Player. Thus, every video type would be associated with the 2nd video player, Kaffeine. Or, you might go to System Settings, Advanced Tab, File Associations and change the video types default player to kaffeine.

As for DVD autostart, I don't really know if it's an option with KDE4.2. Maybe in 4.3.
