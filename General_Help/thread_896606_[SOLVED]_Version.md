### Post by SuperSonic4 on 2008-08-21
8.04.1 is the latest stable version. I think it's kde3.x only though

edit

[http://cdimage.ubuntu.com/kubuntu-kde4/releases/hardy/release/](http://cdimage.ubuntu.com/kubuntu-kde4/releases/hardy/release/) <- KDE4 live CDs

Choosing between kde4 and kde3.x is usually down to personal choice, personally I prefer KDE3.x mainly because of problems with the taskbar (kicker) in 4.1 although the two can run together on the same system

```
sudo aptitude install kde4-core
``` will install the core components while ```
sudo aptitude install kubuntu-kde4-desktop
``` will install the whole desktop and you choose which one you want at the login screen

### Post by Ayuthia on 2008-08-21
> **Yezinki said:**
> Isn't there a difference between KD 4 & KD 4.1??

If you are referring to kde 4.0 and kde 4.1, yes.  There is a lot of differences between the two.  They have made 4.1 more stable.  I have not tried out 4.1 on Kubuntu yet I don't think, but I am using it in Arch and it works well.  There are still a lot of KDE 4 applications that are not ready yet which does leave some people frustrated, but for the most part it works well for me.  The panel has become more useful, but it still lacks the auto-hide feature of kicker.  You can now resize the panel and relocate it.

You can't really go wrong with installing Kubuntu 8.04.1 or the 8.04.1 KDE4 remix.  You can always install the other version an try them both out.
