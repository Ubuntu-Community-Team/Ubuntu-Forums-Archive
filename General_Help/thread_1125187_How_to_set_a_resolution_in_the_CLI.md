### Post by Crafty Kisses on 2009-04-18
I get the problem, but not sure what you're trying to do. If you want to like reconfigure your X and set it to like 1280x1024, it would go something like this I think:
```
cp /etc/X11/xorg.conf /etc/X11/backup.xorg.conf
system-config-display --reconfigue --set-resolution=1280x1024 --set-depth=**
```
Remember substitute your own information, that's just an example. Where set depth is, obviously pick your depth. See if that gets you going, also remember before doing anything with your X server, please back it up, so if you have any issues later on you can obviously use the backup.
