### Post by fragos on 2008-03-10
Assume Ubuntu Gnome: 

1. find any file of that type on the Desktop or using Nautilus.
2. right click the icon and select "Properties"
3. click "Open With" tab
4. you may now select the default application -- if it's not listed, add it

this changes the association for all files of that type
