### Post by Mike_Walsh on 2014-06-26
Basically, you can't.

The Live Session is only intended for trying out the O/S, to see whther or not you like it; it is not intended as a permanent installation alongside whatever other O/S you're using. 

You cannot run a permanent O/S from a flashdrive, anyway.....it's just not practical, and it's also VERY slow compared to the full install.

I WAS running Windows XP on a 10-yr old desktop that would have happily run Windows 7; but I refuse to give Microsoft my hard-earned cash. Under Ubuntu, it simply FLIES!!!

You'll find that Lubuntu is a very good O/S (especially for older hardware); but to get the full benefits, it DOES need to be fully installed.
