### Post by 3rdalbum on 2011-10-31
1. Ubuntu 11.10 and Lubuntu 11.10 are the same underneath, but the user interface and default installed programs are different. Lubuntu is basically for very old or very slow computers, or if you don't have enough RAM to run regular Ubuntu.

2. If you boot from the CD and use the 'Upgrade' option in the installer, or do an in-place upgrade using Update Manager, your programs and personal files will not be touched. If you boot from the CD and do an "Erase this partition" install, then your files and programs will be erased.

If you don't already have a recent backup, it might be a good idea to take this time to make one. If you're making large changes to your disk, it's a good idea to make a backup. If only I wasn't so lazy to forget to backup...
