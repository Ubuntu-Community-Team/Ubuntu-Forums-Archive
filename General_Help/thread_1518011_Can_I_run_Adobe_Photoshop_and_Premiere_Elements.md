### Post by Revolutionary101 on 2010-06-25
You will not be able to run them on Ubuntu natively. Think of it this way, you wouldn't be able to run a Windows program on Mac OS X. That type of thinking must be used when you are switching to Ubuntu. However, you may be able to run Photoshop and Premiere Elements via Wine.

Wine is an application that helps Windows programs run on Ubuntu. For information on how well these programs run with Wine look here:

Photoshop:
[http://appdb.winehq.org/appview.php?appId=17](http://appdb.winehq.org/appview.php?appId=17)

Premiere Elements:
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=8673](http://appdb.winehq.org/objectManager.php?sClass=application&iId=8673)

It looks like certain versions of Photoshop can run well but Premiere Elements may not work at all. Although don't worry about possibly not being able to run these applications with Ubuntu.

You can:

1. Do a Wubi install of Ubuntu. This installs Ubuntu in Windows. It is similar to a dual boot install but without partitioning the HDD.
2. Run Windows in a virtual machine inside of Ubuntu. This will run Windows as a program inside Ubuntu and you will be able to run the Windows programs you need inside Windows
3. Do a dual boot installation of Ubuntu. This will let you install Ubuntu and Windows, side by side, on your HDD. Thus, your HDD will have two OSs to choose between when you start your computer.
