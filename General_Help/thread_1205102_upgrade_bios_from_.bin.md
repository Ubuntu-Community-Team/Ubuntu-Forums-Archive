[LEFT]Stozi,
My advice, is that unless you know that the new BIOS will solve a particular bug, do not install it; regardless of whether or not you are running Ubunu, XP, Vista, or even Solaris for that matter.

Upgrading a BIOS is a touchy affair not to be taken lightly.  ONLY perform it if you have a specific bug, that is confirmed to be solved by the new BIOS upgrade.

I looked at the link you sent.  There is no information as to what this upgrade fixes.  Additionally, it appears to be addressing something with Win Home Server.

In all of the computer's I've built ( a bunch), only once has a bios upgrade ever solved a problem for me.
