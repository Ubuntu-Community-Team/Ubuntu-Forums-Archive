### Post by sandy8925 on 2009-12-27
Check to make sure that your webcam is detected by Ubuntu first. 

If it's a USB webcam then plug it in and open a terminal (Applications->Accessories->Terminal) and type lsusb and press Enter. This will show you the list of usb devices connected to your computer. If the webcam is listed then it will probably work. If not you need to install your webcam drivers and then try using it.
