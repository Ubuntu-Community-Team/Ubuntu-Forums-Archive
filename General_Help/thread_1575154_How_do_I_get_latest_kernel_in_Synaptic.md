### Post by snowpine on 2010-09-15
2.6.32 is the one and only supported kernel for Ubuntu 10.04, and will be throughout the 5 years that 10.04 is supported.

If your system is running fine with the supported kernel, there is no reason to switch to a newer, unsupported kernel.

However, if you wish, you can install any kernel you like from an outside source such as the [Ubuntu kernel PPA]("http://kernel.ubuntu.com/~kernel-ppa/mainline/").

If you are not pleased with a custom kernel you installed yourself, you can always boot back to the supported kernel from the GRUB menu. :)
