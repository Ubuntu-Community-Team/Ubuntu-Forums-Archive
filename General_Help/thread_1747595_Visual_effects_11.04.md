### Post by lakerssuperman on 2011-05-02
If you want a simple Gnome desktop without Compiz effects, logout and select Ubuntu Classic(No Effects) for the session selection menu down the bottom.  Log back in and you should find a Gnome 2 desktop without any Compiz effects.

---

### Post by lakerssuperman on 2011-05-02
If you like Unity and want something like it without all the fancy 3D, you can install the Unity 2D environment for comparable look and functionality without the heavy system resources.
