### Post by lovinglinux on 2012-06-29
> **tobythedog said:**
> well if flash will never work again on linux i cannot see how one can use it to browse the web anymore,as so many web sites use it.how is one supposed to browse the web with ubuntu if it wont work anymore with flash ,most of utube wont work , bbc iplayer and manny more regularly used sites , is this the end of linux as a viable alternative to windows.

Flash is no longer supported by Adobe, who signed a deal with Google to develop PepperFlash for Windows, Mac and Linux.

Google has just released Chrome 20 with the new PepperFlash plugin. Install it and you will get the most recent flash version.

Unfortunately, if you want to use a different browser, then you will have to use the outdated flash version.

Try [Flash-Aid](https://addons.mozilla.org/en-US/firefox/addon/flash-aid/) for Firefox. After instaling the extension and running the Wizard, Opera will also use the plugin.

BTW, Opera works fine with flash here. Sometimes deleting the prefs file from your profile folder solves a few crash problems.
