### Post by 3Miro on 2011-01-19
The Encryption preferences are in System -> Administration -> Encrypt...

From there, you can change the password to an empty password and you will not have to type it every time.

WARNING: the keyring is an important security feature. If you disable the password you will be creating a huge security vulnerability in your system and people could do nasty thing to you. I hate carrying keys in my pocket, but I will never leave my house/car/office unlocked.
