### Post by s3a on 2008-07-07
If you have Ubuntu 8.04 LTS, enter "sudo init 1" in the terminal and select to repair brocken packages (you will need a high-speed internet connection). That will download all packages and re-install them. That's as close as you can get to a repair function to my knowledge.

Canonical should really add some sort of easy to use "repair your computer" feature to disc when you load it as a live cd for users with slow internet or without internet but that's another story.

Hope that helps!
