### Post by kapello on 2007-11-21
It's a special linux kernel which is better if you want to run audio-studio type packages. In other words, music-making software for musicians. 

It's the default kernel with the Ubuntu Studio distro [http://ubuntustudio.org/](http://ubuntustudio.org/)

It doesn't have any advantage for other applications like video, internet, word processing. It also has no extra advantage for normal music playing, like listening to your CDs or MP3s. 

So, only an advantage if you want to run music-making software (ie, the Linux equivalent of software like Cubase, Garageband, etc).
