### Post by dandnsmith on 2011-11-16
OK, so it's the only monitor, and it's plugged in before bootup:

- the 'no signal' is typical of no signal from the video out of the PC, and can be given if there is either no proper connection, or the monitor got fed up waiting (mine wil do it in very few seconds)

- the out-of-range suggests that there is a video signal, but the monitor usable range has been exceeded or somthing.

First, inspect the pins in the cable, monitor and PC.
Second, try a different cable (if you haven't already done so)
Third, if possible try a different monitor.

HTH
