### Post by hacksonfive on 2014-05-30
A few questions:
Which OS was installed first on the SSD (if both operating systems are on /dev/sda)?
If they are on the same disk, are you certain that Windows 8.1 is still there? (I have inadvertantly blown away Windows 8.1 with Xubuntu without realizing it until much, much later on more than one occasion). 
Did you install Ubuntu in such a way that it is compatible with the whole secure boot/UEFI thing that Windows 8.1 has going on (as in [here](https://help.ubuntu.com/community/UEFI))?
