I've seen this description before and I don't think it's fair.  Though what you stated is not incorrect, I find it a bit slanted.  My guess is you use Ubuntu?  It doesn't matter as I think they are all well done.  I just find it a bit of an insult to limit Xubuntu to older machines as well as to simplify KDE to point and click configuration.  I will be a little more verbose but still try to be fair.  The difference as stated above is mostly look and feel based on the default desktop environment (GUI) provided by each.

Ubuntu - Based on Gnome, designed for ease of use and consistent look and feel.  Probably easiest for new users.
Kubuntu - Based on KDE, very configurable, lots of eye candy and applications.  Still a consistent desktop but a bit more of a learning curve for new users.
Xubuntu - Based on XFCE, built more with performance in mind, ballance between look and resource usage.  
Edubuntu - Ubuntu reworked for educational use.

It should be noted that any application from one can be used on one of the others though you loose performance and consistency by doing so.  A google search on any of the above desktops (Gnome, KDE, XFCE) will get you loads of info and screenshots etc.  I also intentionally reordered the list to put Xubuntu above Edubuntu as Edubuntu uses the same desktop as Ubuntu so I thought in terms of listing differences it made more sense.  It really comes down to personal preference and the abilities of the end user to choose.  I have used all three desktops and they all have their merits.

Now to be fair I will admit I am a Xubuntu user (not on old hardware).  I have a 2G P4 processor with a GB of ram and fairly new hardware elsewhere too.  I choose to use it because I like the responsiveness and because I can make it behave like the Unix desktops I grew to love a long time ago.  Feel free to revise my descriptions and point it out if I didn't keep it fair either.  ;)
