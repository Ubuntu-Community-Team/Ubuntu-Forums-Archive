Forgive me for being blunt but Kubuntu doesn't get the man power it needs to make a very complete KDE 4 desktop, plus KDE 4 is still pretty new so a few parts are missing.  If you want to try a more complete KDE you could look at Mandriva or Chakra.

As for the network problem, I suspect it is because a Network Manager plugin for KDE 4 is still in the works.  As for Compiz, personally I'd disable it.  KDE 4 can use Compiz but I think the Compiz in Ubuntu-repositories is more geared toward Gnome.  KDE 4 has it's own desktop effects built in.
