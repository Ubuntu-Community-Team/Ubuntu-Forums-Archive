Not really.

I think it's the same as it's been for a while. Nvidia's proprietary drivers are better than Ati's proprietary drivers. Ati's open-source drivers are better than Nvidia's open-source drivers. Intel's drivers are open source so they work out of the box, and they give you the performance that you'd expect from an Intel graphics part.

Occasionally some version or another will have some problem that brings out the squeaky wheels, but it doesn't tend to last too long.

I've not had any problems running anything graphical on my GTX 560 Ti...
