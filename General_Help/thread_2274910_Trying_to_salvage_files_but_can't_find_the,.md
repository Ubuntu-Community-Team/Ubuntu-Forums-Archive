### Post by dragonfly41 on 2015-04-23
From my experience in data recovery in Windows I have had to use Windows recovery programs rather than Ubuntu.

This entails removing the faulty drive, installing it into an external USB container with its own power supply, and then connecting the unmounted USB drive to a working windows PC via USB port.

One package I used was RecoverMyFiles .. first use in evaluation mode to check that files could be recovered then purchase a licence.

[http://www.recovermyfiles.com/](http://www.recovermyfiles.com/)

but you can try testdisk in Windows.

You could first try boot repair in case it is just a boot problem.
