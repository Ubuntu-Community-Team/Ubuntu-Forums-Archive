### Post by 0mcg0 on 2008-01-25
A midi file contains no sound, it is simply a sequence of note events that triggers samples from a midi instrument of some kind.  You can record the output of your sound card when you play back a midi file through that instrument.  There is a linux program called rosegarden which might be able to do this for you.   I can't provide more details on this because I haven't done any of this kind of stuff with linux yet, only on windoze.
