### Post by drbongo on 2008-08-11
Ubuntu has a full screen magnifier and screen-reader built in! All you have to do is open the system menu, choose preferences and then Assistive Technologies. The name of the screen reader is Orca, this will allow a blind user to navigate to desktop and use the main apps like open office, evolution and Firefox with speech, the full screen magnifier will magnify the screen and invert the colours for someone who is partially sighted. Orca has a list of customisable keystrokes to help you navigate the computer without the mouse. 

drbongo
