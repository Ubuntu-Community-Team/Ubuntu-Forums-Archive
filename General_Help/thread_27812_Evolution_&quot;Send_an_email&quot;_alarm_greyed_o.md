### Post by blueplazma on 2005-04-19
If I remember properly, you can tell evolution to run a command as the alarm too... 

I think if you put "echo" as the command and "<your message> | mail -s Alarm [email]your@email.com[/email]" it will send you an email.  It's not terribly elegant, which is why I don't use evolution anymore.  You can try upgrading to the newest apt-get or source version, but I'm not sure if that functionality is there at all in those either.
