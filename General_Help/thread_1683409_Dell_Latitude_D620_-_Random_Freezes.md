### Post by mikewhatever on 2011-02-07
Not sure what you mean by 'doesn't play well'. Nvidia makes very good drivers for Linux.
As for the freezes, run memtest to check the RAM, check memory usage, disk usage and health, make sure the computer does not overheat because of accumulated dust. Install htop and look at the running processes, check the logs and xsession-errors.
If you want help, start by posting the hardware specs.
