### Post by fragos on 2007-04-23
The Ubuntu LiveCD includeds the partitioning software and will offer to create a dual boot for you.  I recommend that before the Ubuntu install you run a Windows defragmentation utility.  One should be part of the Windows install you already have.  The live CD will given you an option to check its media and I also recommend that.  You're in for a pleasnt surprise when you see all that's included with Ubuntu for free.
