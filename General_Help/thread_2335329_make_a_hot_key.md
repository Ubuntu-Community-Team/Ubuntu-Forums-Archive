### Post by yetimon_64 on 2016-08-27
> **myrjon1 said:**
> made a bunch of hot keys with compiz . ( alt + key ) to open file have a lot of combos that work , i made a home -downloads key and so on ,but cannot get 

nautilus /home/chris/Downloads/test.txt


to open it just highlights   text.txt 
how do i make it open up the file with alt combo.. thanz

Thats exactly what should happen if the hot key command uses nautilus (the file manager). 
If you want the file to open in an editor (eg. gedit) replace "nautilus" in the hot key command with "gedit" (or whatever other text editor you wish to open it with.) 

For example make the hot key command and try using it with ...
```
gedit /home/chris/Downloads/test.txt
```
Cheers, yeti.
