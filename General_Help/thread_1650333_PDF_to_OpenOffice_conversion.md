### Post by hawthornso23 on 2010-12-26
> just wanted to know if there is a pdf converter that can automatically  convert the pdf to its parent format (say .doc / .odf / .xls etc)

The answer I believe is NO. A lot of information in the original document is lost when exporting to pdf. You can't just undo the process as you cannot retrieve this lost information. There are applications to extract text and suchlike from a pdf which you can insert into a new document and try to recreate the original. But usually that is a lot of work.

### Post by Chronon on 2010-12-28
There's an Open Office extension that you can install.  I'm at work on a Windows computer right now, so I can't check the repositories for the exact name (but I'm pretty sure it's packaged there).  Here's a link to Oracle's page for it:
[http://extensions.services.openoffice.org/project/pdfimport](http://extensions.services.openoffice.org/project/pdfimport)

In general, converting a source document to a PDF is not a perfectly reversible process.  There are many possible paths to produce a given PDF, so the creation of a PDF does not strictly have an inverse (in a mathematical sense the original process is not one-to-one).  So, while plugins like this one can give you an editable document to work from, they can't really hope to give you the same parent document that was used to create the PDF.
