### Post by jobezone on 2004-11-12
In Nautilus, go to Places->Templates. (which is actually /home/user/.Templates)
Open any program, and save a blank file inside this directory, give it an informational name, like for example OpenOffice Word.sxw, and it will show up as "OpenOffice Word" in the menu.
