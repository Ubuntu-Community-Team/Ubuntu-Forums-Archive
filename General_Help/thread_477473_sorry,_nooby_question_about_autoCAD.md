### Post by bukwirm on 2007-06-18
To use Windows programs on Linux you need to install [Wine]("http://www.winehq.org/") (which you can do though Synaptic or aptitude). Unfortunately, [AutoCAD]("http://appdb.winehq.org/appview.php?iAppId=86") appears to have a rating of "Garbage", which means it doesn't work at all.

[Here's]("http://www.tech-edv.co.at/lunix/CADlinks.html") a list of CAD applications for Linux.

### Post by kseise on 2007-07-05
