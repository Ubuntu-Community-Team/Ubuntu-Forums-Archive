### Post by ajgreeny on 2012-03-27
There is quite a discussion of this subject in [http://ubuntuforums.org/showthread.php?t=1381198](http://ubuntuforums.org/showthread.php?t=1381198) with some possible ideas and maybe a solution on the last post.  I have not tried it so far as I already manage with a plugin from [http://code.google.com/p/albumartsearch/](http://code.google.com/p/albumartsearch/) which seems to work well on my 10.04 system.

My plugin version shows many possible images, which includes the CD cover if there is one available, but also many other images that are appropriate for the music played, you can then drag the one you want across to the cover art pane, see my two screenshots, one before choosing, one after choosing.

The plugin folder you extract from the tar archive needs to go in ~/.gnome2/rhythmbox/plugins on my 10.04 system, but I have no idea where it would be in 11.10 with gnome3.
