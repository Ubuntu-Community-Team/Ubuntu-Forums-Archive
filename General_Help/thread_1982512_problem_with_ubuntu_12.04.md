### Post by PhantomTurtle on 2012-05-18
To get the old classic gnome desktop back, install gnome-panel. To do this open terminal and type in,
```
sudo apt-get install gnome-panel
```
Or you can do this by install gnome-panel in the software center. Log out, next to your user name there is a ubuntu logo. Click on it and select Gnome Classic. Then log in. It is not as customizable but will give you a more traditional desktop environment. If you want more customizability try XFCE. To get that open terminal and type in,
```
sudo apt-get install xubuntu-desktop
```
If you still want to stick with gnome but have a bit more customizability try a gnome 3 fork, Cinnamon(I use it and it is really good). To get it open terminal and type in,
```
sudo add-apt-repository ppa:gwendal-lebihan-dev/cinnamon-stable
sudo apt-get update
sudo apt-get install cinnamon
```

I recommend you try Cinnamon(it is my favourite out of the three, since it has many different themes, applets and extensions available on their website.

EDIT: Oh and the cinnamon website is [http://cinnamon.linuxmint.com/](http://cinnamon.linuxmint.com/).

### Post by grahammechanical on 2012-05-18
1) The panel on the left is called the Launcher. Click on the top icon (Ubuntu icon). That will open the Dash. Type help in the search panel and you will get access to the Ubuntu Desktop Guide.

or, just tap the Super key

2) Hold down the super key (windows key) and you will get an overlay with keyboard short cuts.

3) Press Super+A to reveal the Dash showing your applications in the form of icons.

4) Press Super+F to reveal the Dash and Recently used files and folders.

4) Press Super+M to reveal the Dash and recently used music files.

5) Press Super+V to reveal the Dash and recently used video files.

That should be enough for you to play around with for now. Unity is just a different way of doing things. We can all learn new things.

One day we may be able to buy Ubuntu PCs, Tablets, and Mobile phones. They will all have the same Unity user interface.
