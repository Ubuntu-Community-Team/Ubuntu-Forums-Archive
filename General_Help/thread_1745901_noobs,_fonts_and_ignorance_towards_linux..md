### Post by collisionystm on 2011-05-01
I was just reading a recent rant on a closed thread about Ubuntu vs Windows 7 and one particular thing I found interesting was when a user noted how complicated it was to install fonts in ubuntu vs. The simplistic way to install fonts in windows. I was blown away. Ubuntu is just as easy.

Download a font from the web (same as windows) open the font and hit the INSTALL button that magically appears. Same as windows.

Or put the fonts in your home directory under the hidden folder called .fonts
If its not there, create it.

In windows you go to window, system32, fonts

So as you can see, ignorance is the driving force for how users perceive linux vs. Windows. Linux is better for a billion reasons, and yet the main argument you see is, "well it doest get viruses". If you actually understood windows and did more than surf the web, you would see how great and magical linux really is. It deserves alot more respect than what some people give it.
