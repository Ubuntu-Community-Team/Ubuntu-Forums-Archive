### Post by yaknowwat on 2008-10-11
There are setting files for wine users in /home/[user]/.wine/

the setting is probably in one of the three .reg files.
just open them with a text editor and check it out.

If worse comes to worse you can reset the wine profile by renaming the .wine directory or renaming the reg files.

files/folder with a " . " in front of them are counted as hidden in linux so you'll need to show hidden files.
