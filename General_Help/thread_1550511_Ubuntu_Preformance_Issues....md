### Post by MatthewAdams on 2010-08-11
There should be a program called "System Monitor" which will let you see all running processes, plus how much memory and CPU usage they take up, PLUS a combined total of CPU usage and RAM usage on the system. 

On Ubuntu, "System Monitor" should be under System's "Administration" or "Preferences" menu.
