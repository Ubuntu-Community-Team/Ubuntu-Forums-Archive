### Post by ragflan on 2008-08-10
Yeap that looks decent enough. And yes, Ubuntu can automatically detect all peripheral devices as far as I know. I never had any trouble with any USB devices. 

I'm using a Logitech MX Revolution mouse on Ubuntu. It's a multi-button mouse with 13 different mouse functions and I can customise the functions with a program that's less than 1mb on Ubuntu called **BTNX**.

With Windows, I had to download a 55mb Logitech program and it offered nowhere near as much customisability as the 1mb application on Ubuntu did.

I don't see why you'd have any problems with any regular mice that don't even require customisation.

### Post by issih on 2008-08-10
Unless it uses bluetooth, then as far as the OS is concerned there is no difference between a wireless mouse and a wired one. In essence its a just seen as a basic human interface device, Ubuntu doesn't care how the signal gets to its usb/ps2 port :)
