### Post by Herman on 2010-02-10
If TestDisk doesn't help you, try PhotoRec instead.
PhotoRec is a program that is automatically installed when you install TestDisk.
TestDisk is for making a new partition table, PhotoRec is for recovering files (even without any partition table).
