### Post by Imaginary on 2008-08-30
> **princerameses said:**
> I need to take text from windows to ubuntu via flash drive. How can I do this. NOTE: I cannot connect to the internet on ubuntu. (that's what I need the word processor for)

Best bet for formatted text is going to be OpenOffice, which is probably installed already on your Ubuntu system.  You can save the files as either Microsoft Word .DOC files, or just get OpenOffice on the Windows machine too.

If you just need unformatted text you can use GEdit (Applications->Accessories->Text Editor), but if you don't have anything smarter than Notepad on the other end, you may run into problems with hard returns not working quite right; Linux and Windows have different opinions on what makes a line break.
