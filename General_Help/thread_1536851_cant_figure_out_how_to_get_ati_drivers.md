### Post by spydeyrch on 2010-07-22
go to system -> admin -> hardware.

it will scan your machine for hardware with available drivers. It should find the proprietary ATI driver and then allow you to activate (install) it.

the only thing that I can think of that would create an issue is if ATI hasn't updated their drivers but I don't think that would be the case.
