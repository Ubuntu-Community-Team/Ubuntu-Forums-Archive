### Post by skymt on 2006-10-09
KPilot is a little better than Gnome-Pilot, but doesn't sync with Evolution (just Kontact) or live in the Gnome panel. Or there's J-Pilot, which doesn't support Evolution or Kontact, but has basic PIM features built in.

For a Gnome user, Gnome-Pilot is the best choice. It seems to be maintained, there was a release last month.

### Post by handy on 2006-10-09
The GP2x, is a powerful & well priced alternative that runs natively on linux.  There are 2 PIM software packages in developement that look great.  One is based on Gnome & the other on the KDE engine.  I have only briefly looked at the Gnome one, which looks great, but is a little slow.  In fairness, I had it installed in a fashion that was not optimal...

As a PDA you may find it just a *little* too big in comparison to other hand helds?

The GP2x plays movies, oggs, mp3s & more. It plays games, I have quake running really well, simple install.

It will use what ever capacity SD card you throw at it, & links via supplied USB cable to Ubuntu beautifully.

The best thing is the great community, there is a huge amount of support, & it is snow balling.  These are early days, it is only about 1 year old.

### Post by Dingo_aus on 2006-10-20
Yeah, the GP2X is a great machine, I've got one myself and it rocks.

The only issue is that it isn't a PDA it is a personal media player. So it handles movies, music etc great.

It doesn't have a battery-backup Real Time Clock (which I think was a major oversight) so I doubt it will ever really used as a PDA (even with hardware hacks).

Apart from that, it is a fully-fledge Linux kernel in your hand that you can telnet into over the standard USB connection and run a bash shell on.  It is a 200Mhz Dual core ARM 9 with 64 Megs of RAM, accepts SD cards.
