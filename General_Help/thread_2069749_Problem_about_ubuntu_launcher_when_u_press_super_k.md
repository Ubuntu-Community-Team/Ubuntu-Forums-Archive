### Post by grahammechanical on 2012-10-11
You must mean the Dash and not the Launcher. Right? I also assume that you are using 12.04. If so, then hold down the Super key and you will get an overlay giving you the keyboard shortcuts.

For example

Press Super + a and the Dash opens in the applications lens - now you can type the name of the application that you are looking for.

Super + f = opens files and folders lens

Super + m = opens music lens

Super + v = opens video lens

Of course, if we are in the file and folders lens when the Dash opens and we search for an application then we will not find it because we are not searching the application lens.

Regards.
