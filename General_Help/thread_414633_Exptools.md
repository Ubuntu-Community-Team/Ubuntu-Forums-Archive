### Post by reclusivemonkey on 2007-04-20
You can download the source here;

[http://www.bell-labs.com/project/wwexptools/packages.html](http://www.bell-labs.com/project/wwexptools/packages.html)

Then just set up your install to build;

```

sudo apt-get install build-essential

```

and then you *should* be able to compile the exptools packages. HTH.
