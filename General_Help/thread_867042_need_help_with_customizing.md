### Post by RuleMaker on 2008-07-22
Go to synaptic, search for compiz (by name) and install compizconfig-settings-manager, then go to Syste-->Preference-->Advanced Desktop Effects Settings and enable whatever you want.
If you enable the desktop cube and Cube rotation you can activate it by Ctrl+Alt and hold down the left mouse button and drag.
Enjoy!

Edit:
If you want it to be a cube you must have 4 workspaces, you can add them by clicking on any of them and then Preferences-->Columns and se it to 4.

### Post by jkyahoo101 on 2008-07-22
To install that theme right click on your desktop and hit change desktop background. Then choose the themes tab. Select install and then choose the file that you downloaded. It should ask you if you want to use the installed theme. If not find it in the list. And print screen takes a picture and a dialog pops up and asks you where you want it to save to. Hope this helps. :)
