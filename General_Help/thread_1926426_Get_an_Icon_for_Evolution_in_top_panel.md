### Post by Frogs Hair on 2012-02-16
The Gnome Desktop includes applications that the Gnome Shell does not , one of which is Evolution . I have no mail icon in the shell , only a link the calendar via the clock applet  . If you are using Classic Gnome you can use Alt+Right Click  to access the panel menu and see what's available .

Edit.

Gnome Classic allows for custom application launchers in the add to panel menu .
