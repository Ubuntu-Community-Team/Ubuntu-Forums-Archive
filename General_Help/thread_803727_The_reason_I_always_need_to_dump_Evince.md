### Post by sub2007 on 2008-05-22
I think you should try a different PDF reader. You're not alone, I don't know what it is but there's something about evince that doesn't like my system. Other people seem to be able to use it without problems but for me, anytime I open anything with Evince it crashes my system and I have to restart X. 

There's lots of other PDF readers. You can try KPDF, XPDF or epdfview (which is very nice if you can handle the fact that it doesn't print hard copies, still a very nice light program for just viewing PDFs). Adobe also make a version of Adobe Reader for Linux if you prefer.
