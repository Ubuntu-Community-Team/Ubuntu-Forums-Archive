### Post by mb_webguy on 2009-02-26
You can make OpenOffice save files in MS formats by default by going to Tools->Options->Load/Save->General.  At the bottom right, select the document type on the left, and then set the "Always save as" box on the right.  Do this for each document type for which you want OpenOffice to use MS formats, then click Ok.
