### Post by HermanAB on 2017-03-11
WinXP is only secure if you unplug the network cable.  In a virtual machine, you can disable the XP network connection and enable file sharing with the host.

### Post by howefield on 2017-03-11
You seem to be well aware of the risks in using XP, so your machines = your choices. I did read somewhere about an updated XP, service pack 4 if you will but can't remember how good it seemed or even if it was desirable, I'll try and find the link and post it, assuming it was legitimate.

Virtualising has the useful snapshot feature meaning it is trivial to obliterate your browsing session and revert back to a pristine copy of XP. I wouldn't recommend end of life operating systems but for those who use them it might be useful.

### Post by ajgreeny on 2017-03-11
I still use XP as a VM in order to update a GPS SatNav machine which can not be done using Linux.

That is the only use to which I put the VM and I have, as howefield says, got a perfect (in so far as XP was ever perfect) snapshot of the VM so if anything did ever corrupt my connected VM I can quickly boot to the good snapshot and I'm back where I started.
