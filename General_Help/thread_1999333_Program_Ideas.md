### Post by LewisTM on 2012-06-15
Now that file notes have been abandoned by Nautilus and the rest of Ubuntu, users have no way to add comments and descriptions to their files. An easy, built-in way to add such info to files and directories is with user extended attributes, accessible via CLI or with the nice eiciel GUI utility.

Problem is, there is no way to search, filter and categorize these attributes and the files attached to them. A graphical tool to do that would be awesome!
