Hope you enjoy learning.

One specific question: other drives do not get given "drive letters" in unix. The different partitions are grafted onto the single filesystem hierarchy that you see. When you "mount" a partition, the contents of that partition take the place of a directory in the tree. Frinstance, the directory /home is often kept on a different partition to the main system. The main system has an empty /home folder, and the home partition containing all the user data gets "mounted" over the top of it during system startup. This is useful because you can then use the same /home partition with two different linux systems if you want, e.g. a working and an experimental version. It's not much different to grafting drives onto My Computer\C and My Computer\D except the mount point directory must exist before it can be mounted onto. Hope that makes sense.
