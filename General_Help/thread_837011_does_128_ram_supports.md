### Post by p_quarles on 2008-06-22
You don't have enough RAM to run the live CD, so you'd need to download and burn the alternate install (text-based) disk.

Beyond that, yes it will run Ubuntu, but it will require some tweaking to get it running fast. If you decide to install, be prepared to spend some time doing this.
