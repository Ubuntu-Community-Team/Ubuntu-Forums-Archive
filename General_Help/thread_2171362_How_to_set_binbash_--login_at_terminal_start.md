### Post by bkline on 2013-08-30
You don't say which terminal emulator you're running, but most of them have an option in the preferences dialog box for that.  For example, in xfce4-terminal, from the Edit menu select "Preferences" and on the General tab you'll see a checkbox labeled "Run command as login shell."

Good luck.
