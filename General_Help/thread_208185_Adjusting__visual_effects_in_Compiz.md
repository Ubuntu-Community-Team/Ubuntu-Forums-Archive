### Post by mcduck on 2006-07-03
other way is to use the Configuration Editor in Gnome. If it's not in your menu you can start it by running 'gconf-editor' in terminal.. Then browse to apps/compiz and there you have all available options :)

It's the 'direct' way to edit Compiz settings, and often includes settings that aren't yet implemented in gset-compiz..
