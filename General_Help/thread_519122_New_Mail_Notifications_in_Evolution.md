Yes, there is a program called mail-notification.  It sit in the notification area and checks your email.  It will then pop up a bubble window when new mail has arrived.  There is some setup required.  Make sure you have it start in the Sessions portion.

[http://www.stchman.com/tools/mail-notification_4.1-1~getdeb1_i386.deb](http://www.stchman.com/tools/mail-notification_4.1-1~getdeb1_i386.deb)

Version 4.0 is in the repositories but I have V4.1 here.
