### Post by ajgreeny on 2017-01-18
You can change the settings for a distro version update to whatever you want (Any new versions, Long-term support versions, or if you always clean install, Never) in the updates tab of Software Sources.

If you choose **Never** you will not see any notification of as new version of the distro, only package updates will be notified to you.
