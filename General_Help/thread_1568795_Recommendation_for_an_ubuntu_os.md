### Post by JohnElway on 2010-09-05
If you're really interested in audio/video editing, just be warned that Linux hasn't quite caught up to other commercial OSes (particularly for video editing). Some people have done this on Ubuntu without issue, but I've heard of many others encountering problems. Although since were talking Linux, it won't cost you anything to test out a few distributions.

For audio/video editing I would try Ubuntu Studio and, if you're willing to try something non-Ubuntu, 64 Studio (which is still based on Debian). I don't know a whole lot about them but I would make a couple Live CDs (or DVDs in this case) and see how they run on your older system. That's really the only way you'll know for sure if they can run well.

EDIT: Just saw the specs for iMac G3s and it looks like audio/video editing might be out. For a *buntu distro, you might want to check out Lubuntu (Ubuntu with LXDE environment).
