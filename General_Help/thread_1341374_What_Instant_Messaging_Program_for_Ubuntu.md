### Post by BenAshton24 on 2009-11-29
The best MSN instant messenger for Linux is "emesene." you need to have a hotmail or live account to use it. 

Then there is skype, which in  my opinion is probably the best way of communicating, you can send messages and even do video calls.

For everything else there is pidgin which supports all of the following protocols.

    * AIM
    * Bonjour
    * Gadu-Gadu
    * Google Talk
    * Groupwise
    * ICQ
    * IRC
    * MSN
    * MySpaceIM
    * QQ
    * SILC
    * SIMPLE
    * Sametime
    * XMPP
    * Yahoo!
    * Zephyr

Pidgin and emesene are both available in the repos. skype you can download from [www.skype.com](www.skype.com)

Hope this helps
Ben.

### Post by ElevenWarrior on 2009-11-29
its fairly simple.
Pidgin should come built into Ubuntu by default, as well as empathy.
I personally like them both.
anyway, with pidgin you are going to do the following.

applications>internet>Pidgin Instant Messaging 
after that opens up you should be able to add accounts by going to,
accounts>Manage accounts
from there you can click the "add" button to add accounts
Which account to add depends entirely on you :)
If you have a gmail account then click on "protocol"
from the drop down list select "google-talk"
then fill in your username but NOT the "@gmail.com"
ie, if yours was [email]ubuntulover@gmail.com[/email] you would put in,
ubuntulover
after that type in your password,
then you can select whether or not to add email notifications at the bottom
(that part has been a little buggy for me)
You probably want to have it remember the password so click the "remember password box"
 hope this helps.
