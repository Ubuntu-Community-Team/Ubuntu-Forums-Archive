### Post by Mighty_Joe on 2010-01-26
A full install of Ubuntu is a little more than 2GB.  You could use the [Live USB Creator]("http://en.wikipedia.org/wiki/Ubuntu_Live_USB_creator") to create a bootable USB drive that would leave you with around 1.2Gb.  It will be a little slower than a full install.
Another option is [Masonux]("http://ubuntuforums.org/showthread.php?t=1248322"). It's a Ubuntu minimal install using LXDE for a window manager. I installed it to a 1Gb USB flash drive and had space left over.
