### Post by relay_man on 2011-07-31
It would be very rare to find a usb mouse that is not 
known/compatible with Ubuntu (or most other GNU/Linux
distributions)

To weed out any hardware/connection problems, I need to ask you
a few questions.

Approximately how old is your desktop computer?

Did you build it or is it factory assembled?

Did this machine run MS Windows prior to your installation of Ubuntu?

If so, did Windows have any problem finding the mouse?

Are you connecting the mouse to a usb port on the front or rear of the desktop unit,?

Have you moved the unit, or been inside of it recently doing maintenance or a adding/removing hardware?
