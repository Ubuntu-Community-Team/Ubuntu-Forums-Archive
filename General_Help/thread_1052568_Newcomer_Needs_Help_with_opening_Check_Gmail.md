Hi Cazzooo,

First off: Welcome to Ubuntu!
Ubuntu is great and this community is what makes it special.
I think that you'll find these forums an invaluable lifeline for your Linux related problems. I'm still learning but this forum is my saviour!

I'll start off by saying that Ubuntu is _Really_ easy to use once you get the hang of it.

Ok, the problem.
I have not used checkgmail personally but I can help you get it installed and ready to use.
First you must use the package manager "Synaptic" to get and install the program. This is really easy.
Click on the taskbar (up the top) "System"->"Administration"->"Synaptic Package Manager"
It will ask you for your password. Enter your password and then wait for it to load.

Hit the "search" button and type "checkgmail"
There in the main window a few unckecked boxes will appear with a description beside it. All you have to do is check the appropriate box and click "Apply" (It will tell you that it needs to install several other packages as well, just click "OK")

Once it has finished installing look in the main menu for something like "Applications"->"Internet"->"CheckGmail - Mail notifier", (or something along those lines)

I really hope that helps, please let me know

P.S.
None of your old Windows programs will work in Ubuntu unless you have "wine" installed. But let's not give you more than you can chew!
