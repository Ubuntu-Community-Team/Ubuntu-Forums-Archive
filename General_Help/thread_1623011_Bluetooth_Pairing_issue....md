### Post by pricetech on 2010-11-16
A lot depends upon what you're using to configure bluetooth on your computer.  I use bluetooth, install via Synaptic and have had no problems pairing devices with my Linux box.  Adapter isn't even seen by windows by the way. (stupid windows)

Look for a setting that allows you to set the passcode for syncing devices.  That might be it.  Also make sure the device is discoverable and is seeking a connection when  you try to set it up in Linux.

Just a couple of thoughts in case you overlooked them.
