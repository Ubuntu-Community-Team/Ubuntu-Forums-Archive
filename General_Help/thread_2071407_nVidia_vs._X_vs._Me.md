### Post by haresear on 2012-10-15
When you install the nVidia drivers from the nVidia website, you will have to rebuild the drivers every time Ubuntu updates the kernel. I think the easiest way to rebuild the drivers is to reinstall the nVidia drivers using the same procedure that you used to install them initially.

PS -- You always have the option to select the pre-update kernel when you boot. The boot menu should provide an option to select the previous kernel.
