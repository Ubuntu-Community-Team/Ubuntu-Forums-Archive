### Post by bthoward on 2008-09-02
What are you trying to get?  Mp3's or an ISO of the disc?  If you want Mp3's try using audio cd extractor which comes by default in Ubuntu.  If you want to make an image of the CD for making copies later then Brasero is the tool that Ubuntu provides out of the box.

There are other tools that some like more than others but these are the defaults.  You can find these in Applications > Sound & Video.
