### Post by mcduck on 2011-10-16
> **vigneron said:**
> What is the purpose of "Main Menu"?  I can not launch applications from it.  All I get is a list of applications - some with a check mark - I assume ones that are installed but clicking on the name does not launch the application.  Also, how does one change the generic icons in the launcher?  Also, where is the restart option?

It's the menu editor, used to create launchers for applications, manage in which category an application goes to, and to hide applications you don't want to be visible.

It's mainly intended for the menus used on Gnome 2, but still works in case you need to create a launcher for some third-party application you have installed that doesn't automatically appear in the Dash.

To change an icon just click the icon image in a launcher's properties, or drag&drop a new image into it.

The restart option appears when you select the option to shut down the system.
