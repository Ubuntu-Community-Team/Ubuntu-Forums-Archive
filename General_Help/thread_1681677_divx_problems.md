### Post by cogier on 2011-02-04
Welcome to the Ubuntu Forum.

I am not sure what "divx" is but if you want/need the codecs to run movies etc. I suggest you take a look here. [https://help.ubuntu.com/community/Medibuntu]("https://help.ubuntu.com/community/Medibuntu").

Follow the instructions and hopefully all will be OK. As this is your first venture on to the forum you may not understand it all but you need to open Terminal either using the keyboard **[Ctrl] + [ALT] + T** or with the menus **Applications>Accessories>Terminal**.

When asked to input your **[Sudo] password** it is the same password as you set the system up with and/or you logged into Ubuntu with. Note that there will be no indication that you are entering anything but it will work.

I hope this helps.
