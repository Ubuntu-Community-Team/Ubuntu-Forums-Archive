### Post by mc4man on 2009-01-30
If you want to do this I'd do it manually, and comment out those 2 intrepid entries in sources.

You'd need the intrepid version of libsamplerate0 first. That I know is possible to install on hardy, I needed it to compile and run the latest lmms build.

To do so go here and do a direct dl and install thru Gdebi (don't remove current version
[http://packages.ubuntu.com/intrepid/libsamplerate0](http://packages.ubuntu.com/intrepid/libsamplerate0)

After that you can install the intrepid libasound2 and possibly after installing it's plugin package. (intrepid version, same method, direct dl and install, leave current installed
Note that the Intrepid libasound will install with the hardy libasound-plugins installed.

Also note that if you have or need the -dev package of libasound things get a bit more complicated.

As to the advisability of what you want I'm not sure. (other than after upgrading libsamplerate0 you can upgrade your libasound2, ect. ect.
