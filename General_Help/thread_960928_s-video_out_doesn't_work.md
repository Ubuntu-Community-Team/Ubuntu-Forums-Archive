### Post by drelyn86 on 2008-10-28
You'll need the amdcccle package. This is the ATI Catalyst Control Center. 

```
sudo apt-get install amdcccle
```

Then simply run it with:

```
amdcccle
```
It also shows up somewhere in your Applications menu...

Should look something like this:

[IMG]http://www.michaellarabel.com/external/catalyst-linux.jpg[/IMG]

It's pretty much like any other display manager if you've ever messed with one. If your screen messes up after you hit apply, make sure you still accept the video changes (hit enter if you can't see the popup), and do a Ctrl+Alt+Backspace to restart the X server, and you should be good to go.

Kind of a tedious process to substitute a simple button, I know... I had to deal with this on my wife's laptop.
