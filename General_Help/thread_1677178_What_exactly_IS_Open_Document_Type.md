### Post by rpared01 on 2011-02-01
its all about money.
example: for the longest time Microsoft used ".doc" but then came ".docx" If you lock the user into using only your software, then you make more money as upgrades and newer versions come along. But if everyone supports the same ".doc" file format, then you risk having your user go to another office suite that might be free or a fraction of the cost of your software.
