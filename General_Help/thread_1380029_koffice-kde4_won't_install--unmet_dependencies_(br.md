### Post by nortexoid on 2010-01-13
In Karmic when I attempt to install koffice-kde4 (via "sudo apt-get install koffice-kde4") I get a slew of unmet dependencies for all the office components (kword, kpresenter, etc.). I'm using the Kubuntu backports repository.

I'm pretty sure I installed it just the other week on my old Ubuntu Karmic install with kubuntu-desktop installed. I'm now using a clean Kubuntu install.

Anyway to bypass these unmet dependences that does not involve building from source (which scares me)?

UPDATE: I had to check the "unsupported updates" in Software Sources and then do an update and THEN install koffice-kde4. It wasn't enough to just have the backports repository enabled.
