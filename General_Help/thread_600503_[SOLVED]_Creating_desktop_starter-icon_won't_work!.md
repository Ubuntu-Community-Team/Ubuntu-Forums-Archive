### Post by perixx on 2007-11-02
I got a problem:

I cannot invoke the 'Create Starter' dialogue via rightclick on the desktop anymore... 
I only get the 'Applications Menu' by doing so -- this might be some relic from my crashed 'xfce4-panel' taskbar-manager issue some time ago...

Creating starters in the taskbar works, though.

Update: solved :D
Applications > Preferences > Settings Manager > 'Workspace'-Manager > [Tab] Preferences > uncheck 'Show Workplace on rightclick'
