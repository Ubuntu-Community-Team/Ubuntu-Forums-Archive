### Post by deadflowr on 2015-01-19
Have looked at the launcher for any new icons that have been added?
Or searched the dash for the name of whatever web-site you installed it for?

Web-apps doesn't add anything to firefox, but will actually install a quick launcher to the unity launchbar, for easy access.
It also has nothing to do with firefox, when used.
When used it'll open a specialized ubuntu-built browser window called ubuntu-browser(or simply Browser).

But, of course, I have no idea if you knew this already or not...
