### Post by unplugged23 on 2010-12-19
Sounds to me like you may be having a problem with the refresh rate. For some reason I had major problems getting 10.04 working because the refresh rate was outside of the monitors range, and it would appear to be in sleep mode. What I did to get around this was use a different monitor for the installation, then install the proper graphics drivers. The I was able to go back the old monitor.

Good luck!
