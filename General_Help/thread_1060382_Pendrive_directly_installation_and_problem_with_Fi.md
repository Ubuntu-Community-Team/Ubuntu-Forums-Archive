### Post by blazemore on 2009-02-04
It's simply because Ubuntu isn't designed to run like that from a direct install to flash memory. Not to mention the fact you're SERIOUSLY shortening the life of the flash drive.

It's best to use the Live USB creator tool, and set some space for persistant changes.

And it should not really be used as a primary OS in this way, unless you're happy with frequent slowdowns and lockups.
