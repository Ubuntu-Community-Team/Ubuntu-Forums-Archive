Please read this: [http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)

To install scilab you can go to Applications - Add/Remove, search for scilab and install it. Alternatively you can install from System - Administration - Synaptic package manager.
