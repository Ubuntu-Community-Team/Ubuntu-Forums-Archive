### Post by danrche on 2010-07-20
dfd001 

I use wine and have used codewavers Crossover software. Crossover is easier to use, and has better support then wine. But in the end, it's still just a prettier wine. I've used it for many games but have had marginal success in it. Wine it's self requires a lot of additonal tuning to get apps to work right, whereas Crossover is pretty good about installing the extra's for you. If you require specific software i'd recommend checking out WineHQ's software list. They rate it on levels of success and which distro's it was attempted on. That list is for the most part the same for crossover only you don't have to install all the wine tricks and such for apps to work. 

I will say that outlook 2007 and exchange 2007 in Wine doesn't work for RPC/Https. :( I can attach to exchange, but not with rpc/https
