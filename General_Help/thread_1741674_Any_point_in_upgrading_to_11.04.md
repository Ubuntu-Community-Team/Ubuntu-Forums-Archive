### Post by walt.smith1960 on 2011-04-28
Built-in support for newer WiFi adapters for one thing.  If everything is working now I don't know of a compelling reason to upgrade. IMO the conversation regarding Unity can get a little heated.  As long as the classic desktop is available with a logout or reboot it's not the end of Ubuntu as we know it. :cool:  I've used both and still find the classic desktop more efficient but I've used GUIs since Windows 3.1 so am used to old school functioning.  I did see a demo of Gnome3 in Fedora which looks interesting.  The guy doing the demo addressed one concern with Gnome3, the missing "minimize" button. Going into a configuration editor and editing one line resurrects the minimize button. I'm sure there'll be more customization how-to's for both Unity & Gnome3.  The sky isn't falling and the tinfoil hat is still on the shelf.
