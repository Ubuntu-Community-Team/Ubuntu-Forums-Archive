The kernel is the operating system code.  Ubuntu 9.04 is probably as different from Ubuntu 10.04 as Windows 95 was different from Windows 98.

There are code rewrites, added features and deleted features from one version of the OS to another.

Edit:  The newer code may have more memory requirements than the old code.  I just recently successfully got a 10.04 LiveCD to boot to show a Desktop, and it can take an incredibly long time to run on my computer, whereas my last successful LiveCD for 6.06 did not take up as many resources.  Once I got it installed and had a swap file, it does run more quickly.

Second edit:  Xubuntu, I understand is less memory-intensive than standard Ubuntu, same version for each.  Xubuntu may not have the graphical user interface, and would be a command-line-driven version.  (I only know for sure that Xubuntu takes less resources, but have never seen it in action.)
