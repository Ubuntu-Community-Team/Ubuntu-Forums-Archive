### Post by jkyahoo101 on 2008-07-25
After you install emerald go into System>Preferences>Advanced Desktop Effects Settings. Then go to Window decorations and delete where it says /usr/bin/compiz-decorator type emerald --replace instead. Then to install themes download them, open emerald, click add new theme or something like that, and then find you theme and click on it. That should be it. Hope I was clear enough. Also just a side note when I used Emerald on my pc it slowed it down a lot.

To download themes go to gnome-look.com and to use the default theme manager again go to window decorations in ccsm and click the reset to default by the command box.
