### Post by wavery on 2009-11-07
It's my understanding that the Swap partition size should be approximately twice the size of memory.  I recently installed 9.10 and let the installer configure the drive automatically.  I'm now adding memory to the computer (doubling it).  Can the Swap partition by re-sized without installing again?  Thank you.

---

### Post by scragar on 2009-11-07
It can be adjusted, but unless you use hibernate keeping the same swap size should be perfectly fine and I would recommend not attempting to adjust it(since it's long, and annoying).

### Post by wavery on 2009-11-07

### Post by jollysnowman on 2009-11-07
To ease your mind, unless you have a lot less than 1gB of memory, having the same amount of swap as memory is usually just fine. You can get away with a lot less if you don't run a resource-hungry system.
