### Post by vigleik on 2006-07-27
You can start kword by typing "kword" in a terminal window. If kword didn't create a menu entry (maybe because kword is made primarily for kde and you are using gnome?) you have to add it yourself by using the menu-edit tool in gnome.

You cannot use acrobat reader to write pdfs, only to read. And fill out forms (but not save them) if you have a very special pdf file. kword can import from pdf, but it does a bad job at all but the simplest pdfs.

Vigleik
