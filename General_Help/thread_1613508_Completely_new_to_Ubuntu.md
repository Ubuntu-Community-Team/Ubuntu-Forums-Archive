### Post by Roasted on 2010-11-04
You're probably thinking about running it virtually. Yes, you can. It'd probably be best for you to look into Virtualbox, which is free and available at Virtualbox.org I believe. Within there you can set up an Ubuntu install, where it's a real live working install that you can use, but within a window. I do the same thing but the exact opposite here at work - I run Ubuntu, and within that I run XP Pro virtually. The only thing is, your computer doesn't exactly scream high performance machine, and running a system virtually WILL eat additional resources. I have 1gb of RAM alone dedicated to my XP instance, and it runs just great. I've ran as low as 350mb of RAM to it and it ran okay, so you could probably do that and have some luck.

### Post by 3Miro on 2010-11-04
Ubuntu is an operating system just like Windows. You can use it to replace windows, you can partition your harddrive into two and install it next to windows (and you will be given the option to select between the two when you start the computer) or you can make a special installation from within windows that you can more easily remove later, but will not let you get the full potential of the system.

You can download a Ubuntu .iso file, then either write it to a CD or put it on a USB drive and then you can start Ubuntu from there to give it a test (USB may not work on a computer that is that old). When running form the CD or USB it will be very slow, but you can see what it looks like and whether or not it works on your computer.
