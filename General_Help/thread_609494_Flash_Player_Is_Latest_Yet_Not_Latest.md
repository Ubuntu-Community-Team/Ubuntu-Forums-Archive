### Post by sultanoswing on 2007-11-11
What version does [THIS SITE]("http://kb.adobe.com/selfservice/viewContent.do?externalId=tn_15507&sliceId=1") say you have?

If it's 9.0.31.0, then that's the latest 'official' version available for Ubuntu Linux. The latest Windows version is 9.0.64.0.

You can, however, grab a [beta version from the Adobe labs]("http://labs.adobe.com/downloads/flashplayer9.html"), which is also 9.0.64.0 for linux.

If this doesn't work (once properly installed!), then it's Amway's website's problem, not yours.
