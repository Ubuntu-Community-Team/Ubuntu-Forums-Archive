### Post by amites on 2012-04-15
are you using proprietary video drivers?

have you recently run any upgrades?

likely the fast solution will be to switch to a shell console

Ctrl + Alt + F1

and re-install your video drivers through the console -- if none of the above apply to you then please describe your setup and config

as a note I've run into a number of issues when running updates and using proprietary drivers -- normally re-installing the drivers fixes everything right up though it took me a while to figure out that was one of the default issues

run into this with AMD/ATI and Nvidia
