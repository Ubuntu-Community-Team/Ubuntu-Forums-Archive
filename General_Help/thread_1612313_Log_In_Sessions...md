### Post by P4man on 2010-11-03
gnome and xfce are 2 different desktop environments (think of it as the graphical user interface). Gnome is a bit resource heavier but has a nicer interface, in my opinion. Gnome is the default DE of ubuntu, xfce is the default DE of xubuntu. Feel free to try either and decide which you like best. There are other popular desktop environments like KDE (used in Kubuntu). Google them for screenshots.

xterm is a terminal, so no graphical desktop environment, just a command prompt.
