### Post by ryanhaigh on 2008-04-21
Just to be clear if you open the run dialog by pressing Alt-F2 and enter gnome-session-properties the application to configure startup programs will come up and you can add skype to that. Press add, call it whatever you want and use /usr/bin/skype for the command.

Im curious as to why you don't have that option, are you using a different menu than the default? If not you may want to try right-clicking on the menu and select edit menu, hopefully you will be able to re-enable the entry there.
