### Post by bsussman on 2006-10-31
I have tried them all though - I like quanta best.

I recommend you subscribe to their email list - it is a good supportive community.  They have developers who consider bugs to be personal and fix them quickly!

quanta is very customizable.  Any keystroke can trigger any kind of action you can think of and/or program.

The project management is very good, integrated management of your code repository, ability to access/edit any file anywhere, integrated tools for debugging.

However it is a kde app so you pay for it's beauty and capability with more resource usage.

be sure to explore some of the special views such as the document structure and attribute editor.  they make certain editing tasks very nice.

Also - install the web developer extension for firefox - it is the best tool I have.  It allows you to play with your css, inspect and classify your document structure.

Consider implementing and developing in your own apache/mysql/php environment, on your own machine - it makes developemnt faster and easier and allows you get to working code faster that if you do not have it.
