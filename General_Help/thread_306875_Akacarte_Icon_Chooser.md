### Post by wacole on 2006-11-25
In using Alacarte I find that when using the browse to choose an icon function [access this by clicking on the Icon area in the Entry Editor which opens the Icon Selector pane, then click on Browse in that pane] that while the browser that Alacarte opens finds all the subdirectories, it does not display any files inside a given subdirectory.  I double checked what appear to be empty subdirectories by opening Nautilus and pointing to the same subdirectory.  Nautilus found files where the Alacarte browser did not.  [I was looking in both /usr/share/icon and /usr/share/pixmap.]

Anyone have a clue as to what might be going on, please?

Thanks very much,

### Post by CatKiller on 2006-11-25
You need to select the directory with the icons in with the Browse window, and then they show up in the Icon Selector window. I don't know why they don't show up in the Browse window, they just don't.
