### Post by grahammechanical on 2015-09-09
weird question = weird answer. :)

What you are seeing is nothing to do with the RAM but a lot to do with the bootloader that starts the live session. It is called, I believe, syslinux and what you are seeing is the copyright details. We do not always see that message but sometimes we do.

In the past the Ubuntu install media used a text based installer. The text based menu is still there. It is usually hidden by the first purple screen but if we press Enter when that first purple screen appears we will see the text based menu. When we are at that text based menu we can press F6 and add some special boot parameters. We sometimes need to do this.

Regards
