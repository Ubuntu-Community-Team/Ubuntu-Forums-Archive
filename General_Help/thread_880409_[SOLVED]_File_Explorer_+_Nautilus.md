### Post by prabath_fun on 2008-08-05
Hi,
   Is there way to configure nautilus like file explorer like in windows?
   That is, in windows, I can press windows key + 'E' which opens file explorer, where I can just by clicking + mark, I can see the tree structure of the folders.
   Is it possible like this in nautilus?

Using Ubuntu 8.04.

Prabath

### Post by DaymItzJack on 2008-08-05
Open up CompizConfig Settings Manager and in the "Generals" plugin, go to the "Commands" tab. You can assign "nautilus" as command 0 and then select a key combo for "command 0" in the keybindings right under that.

As for the 'list', you can't do it by hotkey that I know of, but you can go into nautilus->Edit->Preferences-> and set "View new folders using: List View"
