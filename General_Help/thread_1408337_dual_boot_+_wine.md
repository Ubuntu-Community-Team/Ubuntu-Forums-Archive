> **WarPony said:**
> I was wondering though, if it’s possible to keep the current disk running with Vista, and use the second to run with Linux? Yes, Install Ubuntu selecting the manual partition option and the drive you want to install it on. This will preserve Windows and allow you to dual boot.

> **WarPony said:**

> **WarPony said:**
> I use Office 2007 and don’t want to swap this for OOo - but would wine or some virtualisation software allow me to work in Office 2007 from Ubuntu? For virtualisation, you can use Qemu or Virtualbox. I prefer virtual box. [The latest version of wine]("http://www.winehq.org/download/deb") will run Office 2007 Word, Excel and Publisher. I am not sure what else works.
