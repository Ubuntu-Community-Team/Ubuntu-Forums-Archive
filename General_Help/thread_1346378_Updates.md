Ubuntu 8.04 came in April of last year. In 1-1/2 year there have been many updates. When you install it now you will need the updates to make your system up-to-date. You can ignore them but will face programs with errors in them, programs which do not hold the latest possibilities in them, security issues, etc.
So, do yourself a favour and install all of the updates.
