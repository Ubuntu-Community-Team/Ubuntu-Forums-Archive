### Post by AlphaLexman on 2011-07-30
In my experience, gnome-office (abi-word, etc) is lighter weight in resources, but is not as compatible with Libre/OpenOffice as well as some MS Office products...

If you have the disk space, and the memory resources, and need compatibility options, go with the Libre/OpenOffice option.

### Post by ajgreeny on 2011-07-30
Like AlphaLexman I find that Abiword, in particular is not very good with the open format files from OOo, or Libreoffice, nor MS .doc files.  If I save an .odt file in abiword after editing anything in it, It is corrupted and has to be recovered the next time it is opened in OOo.  Gnumeric is fine and I often use that, Abiword is the weak point of the two;  it will not even open any graphics in MS Ofice .doc files, for some reason, when OOo does so with no problem.  In fact the only graphics I can ever see in .abw files are those I have put in the files when writing them.

If you have the space available, and want the best MS Office and open doc format compatibility, go for OOo or LibreOffice.
