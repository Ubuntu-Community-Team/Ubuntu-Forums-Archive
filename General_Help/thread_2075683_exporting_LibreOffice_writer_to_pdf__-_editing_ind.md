### Post by danielbauwens on 2012-10-24
When you are in a libreoffice page, go to:

```
File > Export to PDF
```

Then there comes a page with a couple of tabs.
Click the "security" tab.

There is an option for changes in the PDF.
I don't know if that also allows changing/removing the index of you PDF though.
