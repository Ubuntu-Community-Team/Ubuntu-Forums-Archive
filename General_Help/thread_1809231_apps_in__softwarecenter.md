### Post by mcduck on 2011-07-21
> **bobk_nyc said:**
> when I install,some show in applications,and some don't...sofwarecenter says they are installed. how do I exe the ones that don't show, and better how do I get them to show in the aplications menu

Most of the applications that don't show in the menu automatically are either command-line applications (so  having them in a menu wouldn't even make any sense, as they are used form terminal instead) or libraries or other types of software that you'd never actually run, but that are used by other applications instead.

Anyway, if you *do* have an application that you now is a graphical  one, and it still doesn't appear in the menu, you can use the Main Menu tool to create menu entries for them. It's accessible by right-clicking the Applications menu (in Classic gnome) or system settings (in Unity).
