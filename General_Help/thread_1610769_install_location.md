### Post by Elfy on 2010-11-01
First - is this a real dual boot or wubi? Installed so you can see it from a windows add/remove thing is wubi.

Yes you can install to specific partitions - though I doubt if windows is going to be making a linux partition - which is what you'll need.

You 'can' see linux files inside windows if you use a driver to read them - I would not do too much with them as windows and linux deal with permissions in different ways.
