### Post by Bucky Ball on 2015-02-06
Welcome. Install flashplugin-installer from Software Centre then update. You may need to reboot. Adobe stopped supporting Linux with Flash at version 11.2, although they still provide security updates. The latest Flash can be used with Chromium or Chrome. You will need to install Pepperflash (pepperflashplugin-nonfree in SCentre). 

Good luck. ;)
