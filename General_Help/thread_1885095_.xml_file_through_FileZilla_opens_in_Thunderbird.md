### Post by Paddy Landau on 2011-11-29
It sounds as though XML files are set (bizarrely) to open with Thunderbird.

Try the following.

[LIST=1]
[*]Find (or create) any .xml file on your computer.
[*]Right-click > Properties > Open With > choose Text Editor. If Text Editor is not shown, press Add and find it.
[/LIST]
Ensure the following is set in FileZilla:

[LIST]
[*]Edit > Settings > File editing > Use filetype associations if available
[*]Edit > Settings > File editing >Filetype associations > Inherit system's filetype associations (ticked)
[/LIST]
