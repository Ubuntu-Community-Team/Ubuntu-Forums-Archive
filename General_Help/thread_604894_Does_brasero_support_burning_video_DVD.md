From [http://www.gnomefiles.org/app.php/Brasero](http://www.gnomefiles.org/app.php/Brasero) :

> Video DVD:
- works if you provide AUDIO_TS/VIDEO_TS (and all appropriate files) in a data project
- can be copied/written to disc if libdvdcss is installed
