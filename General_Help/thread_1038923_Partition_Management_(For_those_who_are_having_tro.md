### Post by masterblah777 on 2009-01-14
For those who are having trouble finding an easy workaround for partition management in Ubuntu I recently found a fairly easy way using the LIVE CD for Intrepid Ibex (Though I imagine Hardy Heron would work nicely as well.) 

1. Go to the add/remove programs and search partition. The first result should be a "Gnome" partition manager with a five star rating.
2. It will be under System - Administration - Partition Editor. And you will notice you cannot modify the main partition because it is in use. This is not so in the LIVE CD.
3. Put in the LIVE CD (CD or USB) and boot off of it persistently.
4. For me the partition manager was still present under the aforementioned area. (See Above).
5. The partition manager should pop up and Volia you are able to manage and edit your existing Ubuntu partition. 

For those that try this please post any success. I currently was able to size the main partition by a very little and the "other" partitions very easily. I was going to attempt resizing the main partition later. I'll post with results.
