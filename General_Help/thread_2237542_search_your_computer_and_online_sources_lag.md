### Post by veddox on 2014-08-02
Welcome to the Forums, first of all :-)

The lag you described is a fairly common problem, we hear quite a few complaints about Unity being slow on machines that are not quite top-notch. Unfortunately, I don't think there is an easy way to "stop the animations". AFAIK, you'd probably need to go right back to the source code to change that. As for stopping the lag, I have two suggestions:

1. Upgrade the RAM on your machine. On my laptop, just sitting idle in Ubuntu with Unity takes ~750MB memory - that's almost all you have on yours. Since you have a desktop PC, buying and inserting more should not be too difficult.

2. If you don't want to spend the extra money: you don't have to use Unity on Ubuntu 14.04. Please note that I am NOT referring you to Lubuntu, etc. However, when you log in you will see a little Ubuntu logo (the Circle of Friends) next to your name. If you click this, a menu will pop up allowing you to choose between whatever desktop environments you have installed. By default this is Unity, but you can change it to "GNOME Flashback". Admittedly, this doesn't look half as pretty, but it is a bit slimmer, which could make all the difference for you, without the need to reinstall your system.

 Hope that helps.
