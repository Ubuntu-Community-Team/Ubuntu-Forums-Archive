### Post by jsmidt on 2006-05-24
I downloaded vim 7.0.  How do I get the runtime spellcheck to work?  Do I alter .vimrc?  If so how?

---

### Post by jsmidt on 2006-05-24
I found out I needed to add this to .vimrc:

```
set spell
```

That's it.  Spell check works great with vim 7.0.  It even reminds you to capitalise words that need to be like at the beginning of a sentance.
                 
                  Happy vimming!
