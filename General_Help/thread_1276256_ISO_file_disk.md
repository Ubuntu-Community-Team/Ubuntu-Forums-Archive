### Post by doas777 on 2009-09-26
truecrypt is a tool to create and use encrypted "volumes" (you can think of them as folders or archives).

you won't find it in the repos (the are legal issues with exporting encryption systems), but you can get it from:
[http://www.truecrypt.org/](http://www.truecrypt.org/)
and install it via this guide:
[http://www.howtoforge.com/truecrypt-with-gui-on-ubuntu-7.10](http://www.howtoforge.com/truecrypt-with-gui-on-ubuntu-7.10)

the howto is a little old but shoudl get you through.
