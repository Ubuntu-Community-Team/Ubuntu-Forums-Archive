### Post by bashologist on 2007-05-07
Maybe the most useful program is "man". My personal favorite is "find". For text formatting "printf" is really good. For displaying partition size "df -h", or "sudo fdisk -l". For image manipulation the "imagemagick" package is amazing. To display your files "ls -lha" might be useful. For text manipulation "sed".
