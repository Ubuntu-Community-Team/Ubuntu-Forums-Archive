### Post by Lord Illidan on 2008-01-24
Have you installed ubuntu server, and now you want a gui?

In that case, if you want the fullblown ubuntu desktop, type :

```
sudo apt-get install ubuntu-desktop
```

otherwise :

```
sudo apt-get install gnome-core
```
