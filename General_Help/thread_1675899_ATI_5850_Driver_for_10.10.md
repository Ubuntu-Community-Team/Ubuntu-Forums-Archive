### Post by Mark Phelps on 2011-01-26
When I had an ATI X1650, I was running the default open-source driver in 10.10 and didn't have any problems.  But, I wasn't doing any 3D gaming, so I did not need hardware acceleration.

You should be offered the opportunity (in hardware drivers) to install the restricted drivers.  I got that when I upgraded to an AMD HD42xx card.
