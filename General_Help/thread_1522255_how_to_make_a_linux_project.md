### Post by QIII on 2010-07-02
This sounds like a homework question, which will not get any detailed answers.

The best I can do for you is to suggest that you install an Integrated Development Environment (IDE) such as NetBeans or Eclipse and develop something with those tools.  Languages like C++ and Java (and many others) are generally platform-independent, so "linux project" is somewhat meaningless.

If you want to do something simple, you can use GNU Emacs (which is a text editor), write your code in C++ and use a compiler like the GNU Compiler Collection (gcc) to generate an executable file.
