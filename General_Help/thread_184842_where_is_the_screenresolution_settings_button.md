### Post by Jucato on 2006-05-30
Kubuntu using KDE 3.5.1 below:
- you can right click on the desktop, choose Configure Desktop Display tab/group.
- you can also see it in KControl > Peripherals > Display

starting KDE 3.5.2 it can't be found by right-clicking the desktop and choosing Configure Desktop (unless you make some configuration tweaks).
