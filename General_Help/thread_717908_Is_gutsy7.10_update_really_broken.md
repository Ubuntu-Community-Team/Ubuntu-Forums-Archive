I must admit I'm a bit confused about what exactly you're trying to do:

-You have installed 7.04 (Feisty Fawn) and want to upgrade to 7.10?

7.10 is the latest stable version, if you have downloaded that one and installed it it should be fine. 

Also, I think you are a bit confused about up**date** and up**grade**:

update: bug fixes and security updates
upgrade: Moving from one version of Ubuntu to another (as in from 7.04 to 7.10 for example)

The repositories are fine by the way, go to System --> Administration --> Software Sources to see which repositories are enabled.
Edit: If you're on Kubuntu, open Adept Manager and check Software Sources there.
