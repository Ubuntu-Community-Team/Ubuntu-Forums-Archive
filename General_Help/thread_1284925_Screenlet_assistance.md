### Post by joey-elijah on 2009-10-07
> **windowsfree said:**
> [FONT="Arial Narrow"]I am using Ubuntu 9.04 and the Gnome desktop enviroment.  I need to know how to get other screenlets downloaded and properly installed so I can choose them from screenlet daemon.  Thanks.[/FONT]

1. Find the screenlet you want to download
2. Download it.
3. Open the screenlet manager and either drag n' drop the downloaded package onto the manager window or click "install" and navigate to it.
4. Launch the screenlet either by double clicking on it's icon in the manager window, selecting it and then clicking 'start' or right clicking the tray daemon and selecting it from the list.
