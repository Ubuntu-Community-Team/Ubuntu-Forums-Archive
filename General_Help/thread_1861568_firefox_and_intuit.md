### Post by lovinglinux on 2011-10-15
Install [User Agent Switcher](https://addons.mozilla.org/en-US/firefox/addon/59/). After installation, right-click on the toolbar, select "Customize", then drag the *User Agent Switcher* button to the toolbar. Close the customization dialog. Click the *User agent Switcher* button, select "Internet Explorer >>> Internet Explorer 8". You won't get the browser error again.

Disable the switcher when viewing other web sites.
