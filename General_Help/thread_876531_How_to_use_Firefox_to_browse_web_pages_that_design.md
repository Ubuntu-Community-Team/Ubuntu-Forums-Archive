### Post by tuxxy on 2008-07-31
Download this plugin,

[https://addons.mozilla.org/en-US/firefox/addon/59](https://addons.mozilla.org/en-US/firefox/addon/59)

It will allow you to change your browsers identity, from your firefox tools menu select which you would like to use.  You can select from Opera firefox or IE7
