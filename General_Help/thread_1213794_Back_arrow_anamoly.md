### Post by Oldhacker on 2009-07-18
Today, I found a "partial" solution to the NYT anomaly. If one holds down the Alt and back arrow keys for a few seconds, it is possible to return to the main page. Normally, just holding down the Alt plus back arrow for a second suffices. My conjecture is that on the NYT site, they are going through access to several of their links before a return to the previous page can be completed.
