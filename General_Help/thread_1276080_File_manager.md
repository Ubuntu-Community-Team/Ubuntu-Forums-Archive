### Post by oldos2er on 2009-09-26
> **simartem said:**
> Do i have to uninstall Nautilus for installing gnome commander and how to make "gnome commander" or "nautilus" the default one, if both are installed on the OS ?

 No need to uninstall Nautilus. Nautilus is tightly integrated with Gnome; it actually provides the desktop you see, so while uninstalling it (Nautilus) may be possible, it's not something I could recommend trying. If you're determined, see [https://help.ubuntu.com/community/DefaultFileManager](https://help.ubuntu.com/community/DefaultFileManager) and [https://wiki.ubuntu.com/CustomizableFileManagerSpec](https://wiki.ubuntu.com/CustomizableFileManagerSpec)

### Post by bruce2000 on 2009-09-26
> **simartem said:**
> Do i have to uninstall Nautilus for installing gnome commander and how to make "gnome commander" or "nautilus" the default one, if both are installed on the OS ?

Like oldos2er says, it's not necessary to remove nautilus in order to install GC. It's just a regular application which is run from the Accessories menu, but nautilus is still the default file manager (for example, if you click on an entry under the Places menu nautilus will open.)
