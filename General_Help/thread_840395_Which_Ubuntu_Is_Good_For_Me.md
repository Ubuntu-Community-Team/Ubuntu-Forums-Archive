### Post by Kevbert on 2008-06-26
You can check how much memory you have as follows:
1) Go to Applications - Accessories -Terminal
2) In the window that appears enter
```
free -m
```
The first figure is the amount of memory available to programs.
3) Close the window with
```
exit
```

### Post by mikjp on 2008-06-26
That hardware should run Ubuntu with no problems -- you certainly don't need a low specs distro or installation. Open a terminal window and enter the command:

top

Here you can see which processes take all the processing time. Maybe Ubuntu is indexing files? In that case you might turn the indexing off, unless you really want to use Beagle for searching.
