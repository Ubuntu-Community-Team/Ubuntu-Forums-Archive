### Post by adam704 on 2010-09-30
Problem solved. I used sendEmail with -s smtp.gmail.com:587 which is what I was trying all along but i needed to install some perl modules I didnt have previously. aptitude install libio-socket-ssl-perl libnet-ssleay-perl perl is the code to install them. works like a charm!
