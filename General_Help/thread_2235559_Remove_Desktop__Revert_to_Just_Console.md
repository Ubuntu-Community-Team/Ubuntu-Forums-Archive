### Post by grahammechanical on 2014-07-22
For those who wish a minimal installation there is the mini ISO image also called the netboot ISO image. It is 31MB in size. All the installation is done over the internet and it gives many options. We can install the standard Ubuntu and its flavours or just a command line Linux and then from there install what packages we require and not have any of the packages we do not want.

[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

Otherwise you have a lot of stuff to remove and you have to do it on a package by package basis. How clean will that be?
