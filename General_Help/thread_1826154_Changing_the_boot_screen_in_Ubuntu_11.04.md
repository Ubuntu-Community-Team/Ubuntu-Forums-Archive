### Post by moorhead98 on 2011-08-16
There is a program called "Plymouth Manager" that allows you to install new splash screens and change ones that you have installed. Its by far the best to download new screens, but enabling it doesnt work all the time. What I do is download the splash screen on Plymouth Manager, and enable it in zorin.
Here is the link:
[http://sourceforge.net/projects/plymouthmanager/](http://sourceforge.net/projects/plymouthmanager/)
Just download and install the .deb and enjoy!
Also, try looking around a ubuntu blog like omg!ubuntu! for a mirror link to the zorin splash screen manager.
If you have any questions, just ask
