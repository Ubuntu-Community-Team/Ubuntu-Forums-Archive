### Post by azulcm on 2008-05-13
Hi, there is a plugin called [gkrelltop]("http://sourceforge.net/projects/gkrelltop/")

In ubuntu, you just type:
[FONT="Courier New"]sudo apt-get install gkrelltop[/FONT]

Restart gkrellm and enable the plugin.
