### Post by marshag63 on 2011-03-05
Two things to check:  Right click network icon and look at connection information - it will tell you what interface it is using/trying to use, ip address, driver, connection speed, etc. You can check this with the ethernet cable plugged in and not plugged in and see what you get.


In a terminal, you can run the command "ifconfig" (without quotes) to see what interfaces are already defined.

We cam work from there.
