### Post by An Sanct on 2011-03-29
Welcome to the forums!

Well, I cannot help you with the sound problem ... if reinstalling is an option, that could end up in the same problem. Normally, take reinstalling as the really _last_ option.

If you want to install Winxp64 along side Ubuntu, I advice you to install windows first and then Ubuntu along-side, to keep away from boot problems. Windows is known to mess up other OS installs - if that happens, you have to boot from a Live media (USB preferable) and reinstall GRUB.
