Open Applications - Accessories - Character Map.
Here you find the characters you want.

Click one and write down the number at the bottom of the screen, do this for all the characters you need.

Then when inserting them into text you type:
ctrl+shift+u (first ctrl key, while holding it add the shift key and while still holding both add the u key)

While still holding the ctrl+shift keys type the number you wrote down .
for the character.
Then let go of the ctrl+shift keys.

### Post by Agent ME on 2009-06-10
Here's a really simple way: Go to System -> Preferences -> Keyboard Preferences, Layouts tab. Click the "+ Add..." button and pick "USA International (AltGr dead keys)". Remove the other layout, and optionally apply it system wide.

Now pressing the right alt key + n makes 챰, or plus any vowel puts an accent over it. With other letters it seems to do other random letter symbols.
There are even some other odd multi-step combos, like pressing right alt + ', then r makes &#341;. Experiment a bit.
