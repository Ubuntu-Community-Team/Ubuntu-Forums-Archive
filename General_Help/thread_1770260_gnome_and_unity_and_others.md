### Post by linuxinstalledfromhdd on 2011-05-29
Close.

Gnome and Unity are environments. 

And the others are windows managers and special effects.

As soon as Gnome 3 has completed development, (fingers crossed)
they will release Gnome 3 with the next release of Ubuntu. 

You can switch back to Classic Gnome in Unity by doing this:

Click on top left corner of Unity. Type in the search box for Login.  Unlock. And select from the dropdown menu Ubuntu Classic. And exit to  save. Reboot to make sure the changes have taken effect.
