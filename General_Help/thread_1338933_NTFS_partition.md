### Post by khelben1979 on 2009-11-27
Well.. the EXT3 or EXT4 cannot be accessed by Windows a.f.a.i.k. which means that you need an NTFS partition in order for both operating systems to be able to access the files.

If you only use Linux and not using dual boot, then it's pointless in my opinion.

I myself have 2 harddrives in my box which is formatted by NTFS and they are easily accessed from within Linux which is installed on the third harddrive.

I can only think of one reason for it.
You can fill it with ISOs of boot disks, and recovery information. If the ext3 partition becomes unbootable, you can read the information by plugging the drive into a windows computer, and hopefully get linux running again.
