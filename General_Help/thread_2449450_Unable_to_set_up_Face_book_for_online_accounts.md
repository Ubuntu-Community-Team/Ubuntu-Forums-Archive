### Post by sdsurfer on 2020-08-27
So you are interfacing with a Facebook app from the desktop? This is not an Ubuntu issue. An appl is not needed for Facebook access from the desktop.

 At any rate,if you are trying to set up an app, you have to set up the app on developer.facebook.com and set it to live, then allow the apps access from within Facebook. Here's a start where it's pretty well discussed:
[https://stackoverflow.com/questions/30085246/app-not-setup-this-app-is-still-in-development-mode](https://stackoverflow.com/questions/30085246/app-not-setup-this-app-is-still-in-development-mode)

Then there's always the docs:
[https://developers.facebook.com/docs/apps/](https://developers.facebook.com/docs/apps/)
