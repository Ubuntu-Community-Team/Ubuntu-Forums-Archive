That isn't really wrong, it just refers to an image on the webserver the page is on which is in the 'img' directory. You can't rely on a temporary image host like Photobucket anyway. The problem isn't that it won't show up, it's the positioning of the image.

By the way, you might find it easier to use CSS to do your positioning and such.
