### Post by RelativelyQuantum on 2008-04-29
Just letting everyone know, I was able to resolve this issue by logging in under the failsafe gnome session - the screen is still inverted, but fortunately enough you can still set the inversion to "normal" under system>>preferences>>screen resolution. Thereafter one can return to the standard gnome session.
