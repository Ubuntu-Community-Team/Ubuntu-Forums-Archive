### Post by LowSky on 2008-09-17
keyring is supposed to be a application that saves all you normal passwords for all application so you only have to use one pasword while the keyring remembers the rest... I hate and you dont need to use it.

go to your home folder, type Ctrl+h to show hidden folders and delete the .keyring folder

now the next time you go to connect your internet (wifi i'm assuming) when it asks for keyring password, just hit enter, dont type anything and it will never show up again
