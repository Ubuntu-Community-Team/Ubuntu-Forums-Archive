### Post by Canaryguy on 2009-08-25
Hello again,

     I have good news for all the people that has the same problem as me. Well, if you have problems to use some chats as I had for example in Livemocha or Babbel because you are using Ubuntu x64 you need to install Flash Player 10. But if you try to install in from the Adobe site you will get a message saying that your computer is not compatible with the version of the Flash Player you are trying to download or something of that kind.

     I tried this solution from one site and it worked for me. Now I don't have problems opening sites.

     Solution that may work for you too:
1º Close all web browsers.
2º Go to the Terminal and introduce the next lines:

     wget [http://queleimporta.com/downloads/flash10_en.sh](http://queleimporta.com/downloads/flash10_en.sh)

     sudo bash ./flash10_en.sh

That's all. Good luck.
