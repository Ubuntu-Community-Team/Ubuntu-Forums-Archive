### Post by danwood76 on 2009-04-18
Ext4 is a lot faster, you will see faster boot times etc with it.
I personally have had no issues with it and prefer it to ext3.

Some people early in jauntys development reported data loss but I think the bugs were fixed.

The only downside to it is that if you ever want to use an older version of ubuntu with the ext4 drive you will be unable to.
