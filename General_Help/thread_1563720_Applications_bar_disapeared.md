### Post by Brandon Williams on 2010-08-31
Right-click on the panel and select "Add new items ..." from the pop-up. In the dialog, select "Xfce Menu" and click "Add". Change the button title to "Applications" and select "xubuntu-logo.png" as the button icon. You might need to Right-click on the new menu and select "Move" in order to put the new menu button in the position on the panel where you want it.
