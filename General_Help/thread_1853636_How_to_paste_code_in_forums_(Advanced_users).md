To paste a code is simple (i found it out 2 seconds ago)

All one has to do is to gently correse the # symbol just above where one writes a comment. 

After one hour of corresing and wooing the # symbol the # will moan and convulse and out will come a little sentence that will let you know that you have satisfied the '#' completely.

The sentence will look like '''[Coode][/Code]"      (i had to put 2 o's in coz this was quiet a loose # symbol plus it would disappear if i didn't)

Finally, one only has to paste his code between the two brackets like so....

                                 [Coode] _**enter the code here**_ [/Code]That is how one of average standard manages to paste his code in a # symbol.
