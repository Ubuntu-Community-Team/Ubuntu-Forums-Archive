### Post by Robster2 on 2010-01-09
The file is called Score with a capital S

In the terminal Applications>Accessories>Terminal

$ locate Score 

Will find it.  

In the current version of pan it is in the /home ~/.pan2 directory.

Via the GUI file manager (Nautilus)  Places>Home Press Ctrl-h to show hidden directories/files and click on the .pan2 directory.
