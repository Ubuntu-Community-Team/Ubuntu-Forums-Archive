### Post by RichardG891 on 2009-04-26
It wouldn't hurt to put the image on a USB stick and try it out - but it expects to find an intel graphics chipset - I think your thinkpad will have ATi. You can always load the proper driver after install as remix moves VERY slowly without openGL. It is a cut down version, but I've run a recent beta of remix on an ASUS barebones (intel set) desktop PC and it's fine (java, flash, etc), and looks good on 4:3 as well as widescreen.... so it doesn't need to be on a netbook as such... good luck!
