### Post by PhantomTurtle on 2012-04-06
Download the file "pidgin-guiops-0.5.1.tar.gz" from the website downloads page. Extract it, somewhere. There is a script called "cross_compile.sh". You want to just run that and it will compile for you. Just so you know this is also in the wiki page for the plugin.

Source: [http://code.google.com/p/pidgin-guiops/wiki/HowToCompile]("http://code.google.com/p/pidgin-guiops/wiki/HowToCompile")
