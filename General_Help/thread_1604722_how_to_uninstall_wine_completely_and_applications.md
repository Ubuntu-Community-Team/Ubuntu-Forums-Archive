### Post by Irony on 2010-10-24
Once you have uninstalled wine via synaptic then delete the folder
```
~/.wine
```
and delete
```
~/.winetrickscache
```
Note: to see these folders go to your home folder and press Ctrl H

Then if the entries are still in the menu go to;

**System > Preferences > Main Menu**

and delete the items from there.
