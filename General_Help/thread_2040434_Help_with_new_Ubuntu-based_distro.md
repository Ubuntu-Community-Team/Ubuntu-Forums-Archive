### Post by mastablasta on 2012-08-11
it would be best to ask Cannonical about the legals stuff (these are community forums).

however there are plenty of Ubuntu based distros. I think that if you modified it you can't really call it and distribute it as Ubuntu but you could call it Honbuntu for example. As you can see there are plenty of Ubuntu based ditros already (check out the distrowatch website to find plenty of them). On of more popular is Linux Mint, but there are also some others like Zorin OS, PearOS, Ultimate Edition, Pinguy OS or more specialist ones such as Clonezilla or Redo backup. None of them really use Ubuntu branding (logos, names) but all of them are quite legal and each has it's own programmes/applications added.
