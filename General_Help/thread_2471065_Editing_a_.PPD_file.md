### Post by schragge on 2022-01-20
The **/etc/cups/ppd** is contolled by **cupsd**. Anything in there is generated by CUPS and will be overwritten. The PPDs there have a comment
```
# Printer configuration file for CUPS v2.3.1
# Written by cupsd
# DO NOT EDIT THIS FILE WHEN CUPSD IS RUNNING
```

[lpadmin]("https://www.cups.org/doc/man-lpadmin.html") has option **-P** to specify a PPD file for printer, but that option as well as using PPD files at large are [deprecated]("https://www.cups.org/blog/2018-06-06-demystifying-cups-development.html"). Instead, Apple suggests [ippeveprinter]("https://www.cups.org/doc/man-ippeveprinter.html") as an alternative to PPDs. See [CUPSDriverlessPrinting]("https://wiki.debian.org/CUPSDriverlessPrinting") @Debian Wiki.
