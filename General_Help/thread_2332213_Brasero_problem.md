### Post by Dennis N on 2016-07-29
> I wonder why there is a conversion to wav files ?

Because audio CDs were invented for standard 1980s vintage CD players, which only read uncompressed data, so you need the conversion of the compressed mp3 to uncompressed .wav files before burning. 

Even so, data is not stored as separate files on the disk, but as a continuous track of digital bits plus a table of contents to define where in that track each song begins.
