[SIZE=4]If you are a gmail user, you can add 

[/SIZE][SIZE=4][COLOR=Blue]ubuntu@partychapp.appspotchat.com[/COLOR][/SIZE][SIZE=4] 

to you gmail chat contact,

so that you can chat with other ubuntu users all around the word.

To change your nickname, type the following into you chat frame:
[/SIZE]```
/alias yournickname
```[SIZE=4]
To see who are in the chatroom, type:
[/SIZE]```
/list
```[SIZE=4]
To leave the chatroom, type:[/SIZE]
```
/leave
```[SIZE=4]
To get help:[/SIZE]
```
/help
```
