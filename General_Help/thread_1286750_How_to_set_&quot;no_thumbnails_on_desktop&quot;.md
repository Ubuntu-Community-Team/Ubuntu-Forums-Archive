### Post by GrayBear on 2009-10-09
I need my Ubuntu **NOT** to show thumbnails for the images that are on the desktop. I searched in the settings of Nautilus and "System -> Preferences -> Appearance" but I could not find such option. I'm using Ubuntu 9.04

Any way to set such an option?
thanks

---

### Post by Tamlynmac on 2009-10-09
Nautilus/edit/preferences/preview tab/Other Previewable Files/Show thumbnails

Change this setting to "Never".

Hope this helps.
