### Post by cubeist on 2008-05-06
nice theme find, might even use it myself.  Your right, it is a gtk theme.  GTK is just a system library that allows graphical elements...sort of like emerald (you can use one or the other...)

To install, download but don't unzip it!

Open system/prefs/appearance

click the little install button and select the downloaded file.

thats it.. that is all you need to do to install a gtk theme.

Lastly, in compiz settings, select gtk as the window manager (instead of emerald...can't use both at the same time) and then you can customize via the appearance prefs.
