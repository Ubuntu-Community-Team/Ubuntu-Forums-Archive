### Post by SadaraX on 2005-11-13
Does anyone know how to assign a hotkey (such as Windows-Key + L) to the "Lock Session" option in the kmenu in KDE?

---

### Post by kaarlov on 2005-11-13
It can be done in Control Center. Regional & Accessibility -> Keyboard Shortcuts. In Global Shortcuts section "Desktop" there's Lock Session which defaults to Alt+Ctrl+L but you can change it to whatever you want.
