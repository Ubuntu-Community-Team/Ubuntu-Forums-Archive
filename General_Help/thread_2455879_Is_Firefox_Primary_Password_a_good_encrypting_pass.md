### Post by dragonfly41 on 2020-12-31
I have so many passwords in use (one per account) that I find it easier to use the "Forgot Password" feature which returns to my email account a link to inject a new password. Then I run **pwgen** in terminal and choose a fresh string to update password.
I do keep a local record of these strings but not in a wallet. The above workflow is quite fast. And it can be automated.

There is a useful discussion on secure passwords [found here.]("https://www.grc.com/passwords.htm")   And [this is another important read.]("https://www.grc.com/fingerprints.htm")
