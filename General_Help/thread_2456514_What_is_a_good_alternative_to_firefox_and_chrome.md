### Post by TheFu on 2021-01-13
Or run firefox in a constrained environment.
```
alias fireff='firejail firefox '
alias firepff='firejail --private firefox '
```

The issue with using alternate browsers is that 1% edge cases that don't work.  I do so much filtering already that breaking for blocked subnets, blocked URLs and not allowing javascrpt are already common.  For really sketchy sites, I use Dillo
```
firejail --private dlllo
```
