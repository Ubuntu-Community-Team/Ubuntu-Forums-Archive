### Post by jpeddicord on 2008-06-21
It's been a while since I've used a Mac, so this might not be the exact effect.

If you're using Compiz:
[LIST=1]
[*]Install the [compizconfig-settings-manager]("apt:compizconfig-settings-manager") package if you don't have CompizConfig Settings Manager.
[*]Open CompizConfig from System > Preferences
[*]Find the Animations plugin, enable it, and click it to open its settings.
[*]On the Minimize Animation tab, double-click the Zoom entry in the list box.
[*]Change the effect to Magic Lamp.
[*]Close the window and test it out. If it isn't the right effect, open it again and experiment with others.
[/LIST]
