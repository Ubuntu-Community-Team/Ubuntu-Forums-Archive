### Post by IMSargon on 2010-10-24
According to the FAQ, no, you can't run OS 9. At least, nobody's done so successfully.

Anyway, the command to run PearPC, is "ppc [config]" where "[config]" is the configuration file that you've written as per the documentation I linked earlier. 

I've had good luck with Darwin and it's free, so that's a good place to start if you don't have another PPC operating system available.
