How can I make Pidgin run automatically when I login to gnome?

### Post by ThrobbingBrain66 on 2007-11-04
System > Preferences > Sessions

Add pidgin to the list.

---

### Post by unabatedshagie on 2007-11-04
Add it to the session?

Click on System > Preferences > Sessions then click on add type Pidgin for the name and pidgin for the command and that should do it.
