> **L1GTN1NG said:**

I use Compiz's ADD (Attention Deficit Disorder) plugin to make all except the active window dull and semi-transparent.

Also, if you're using compiz, holding down Alt and scrolling with the mouse wheel will make any item the mouse is on transparent; and this setting is "remembered" for the future.

### Post by prshah on 2008-11-22
> **L1GTN1NG said:**
> How would I enable the theme that you have there?

System-Preferences-CompizConfig Settings Manager-ADD helper-Enable ADD Helper, Misc Options-Enable ADD helper on start.

The theme i'm using is in the repos, it's called "New Wave". 

Alt-"down scroll" the mouse on any window / element (such as taskbar) to increase transparency. Alt-"Up scroll" to increase opacity.

I've just now realised that transparency settings using Alt+scroll is no longer permanent.

Instead, you have to use the "Opacity, Brightness and Saturation" option in CompizConfig Settings Manager. You can set the Opacity, brightness and saturation for all or particular types or names of windows etc. Enjoy!
