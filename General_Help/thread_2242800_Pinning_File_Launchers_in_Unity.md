### Post by 2CV67 on 2014-09-04
When Unity arrived on my very old PC, there were several negatives which stopped me using it, so I switched to Xfce instead.

Now with a new PC it seemed time to try again.

Lots of things are better now, but one remaining annoyance was I couldn't see any way of pinning one-click launchers for frequently-used files (other desktops allow launchers on the panels).

Eventually, I found "Drawers" in Synapt.
This provides a tile in the Main Launcher which, when clicked, opens a window where you can pin file launchers.
So I now have a 2-click launch procedure, which is not as good as Xfce, but is acceptable.

I would have thought that function should be present as standard in Launcher, or better still in Dash...

### Post by ian-weisser on 2014-09-04
Unity's workflow doesn't work that way. You must unlearn a few habits if you want to use Unity.

I have several very-frequently-used files, too. Happily, I find two-click access to be not onerous.

You can place (or link) your files on the Desktop for one-click access.
Or you can create .desktop file for your desired file for one-click access.
Or frequently-used-files should show up near the top of the Dash for two-click access.
