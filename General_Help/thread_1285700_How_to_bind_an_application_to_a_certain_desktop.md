### Post by prshah on 2009-10-08
> **yeeeev said:**
> I would like to bind a certain application to a certain desktop

Yes, this can be done since you are using compiz. Open the CompizConfig settings manager from System-Preferences, then, in the "Windows Management" category, look for the "Place windows" add-on. In this addon, open the "Fixed Window Placement" tab.

Open this, set it up as you want, then enable it. Settings will persist across reboots.

If you want more details on how to set it up, please post back.
