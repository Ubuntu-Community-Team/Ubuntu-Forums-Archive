### Post by Ramses de Norre on 2006-08-26
You should be aware of the fact that flash memory has a limited read/write count, which means that you can only write or read data from a certain location in the memory for a couple of thousand times or something like that.
This is no problem when you use it for file transfers or things like that but when you run an OS from it which has a pretty high I/O you could be facing bad blocks very soon.. So you can try to do so but I don't think your flash memory will last much longer than a year or maybe two.
