This is a web service.  I am interested in a utility that I can run at my laptop. Most of what i find treats each PDF page as an image.

One can open a PDF file using OpenOffice and have some ability to manipulate the file contents.

There is the **pdfedit** package that is pretty powerful in its abilities to manipulate PDF files. Remember, though, that PDF was designed to be a way to distribute electronic documents that were effectively "locked down" like printed documents. A great deal of what you are able to manipulate depends on how the original page author set the options of the original document.

One reason that folks want PDF-to-DOC conversion involves their desire to alter the text content of the pages or to add content for filling a form or similar. Most of the commercial tools for this are very proprietary and very expensive.
