### Post by ju2wheels on 2009-04-23
1. System->Administration->Software Sources . Select the "Download From" drop box and select "Other" and press "Select Best Server" or choose a server which you know is geographically close to you. This should help with your download problems since everyone is overloading the main ubuntu.com servers.

2. Try System->Administration->Hardware Drivers first. You are most likely missing your graphics card drivers.
