Navigating files... To navigate your way around the files you need to know only a few basic commands...

[COLOR="SeaGreen"]cd FOLDER[/COLOR] this lets you move to the chosen folder

[COLOR="SeaGreen"]cd .. [/COLOR]this moves you back up a directory

[COLOR="SeaGreen"]nano FILENAME[/COLOR] this opens up a text file in an, "in terminal" text editor

[COLOR="SeaGreen"]gedit FILENAME[/COLOR] a more user friendly "out of terminal" text editor
