### Post by ep2011 on 2007-05-28
I would start with the normal Ubuntu 7.04 (ubuntu.com). Ubuntu Ultimate edition only has more programs and a new theme, there is no reason to install that. You can get all the programs you need from Synaptic anyways. And yes, Ubuntu can use Hebrew (The livecd will not, but you can enable it while installing). I prefer the real deal, and then customizing it to my needs, instead of downloading a pre-customized version for THEIR needs, which wastes tons of harddrive space.
