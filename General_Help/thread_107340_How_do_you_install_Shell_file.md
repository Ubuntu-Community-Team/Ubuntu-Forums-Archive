### Post by ninotob on 2005-12-22
I installed cedega a week ago to try it out.  Had no problems.

I assume you downloaded the "*.deb" package.  If not, and you are using ubuntu, that is the one you want to get.  All you have to do is open your terminal, cd to the directory to which you downloaded cedega, and type:

sudo dpkg -i cedega_______.deb

Note, don't actually type the "_______" -- fill in with the version information.  The easiest way is type this much:

**sudo dpkg -i cede** and then **press tab** -- that'll fill in the rest of the name automatically if the first few characters are unique.
