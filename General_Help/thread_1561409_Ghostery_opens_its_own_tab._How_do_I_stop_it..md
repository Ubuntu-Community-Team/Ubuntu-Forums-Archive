### Post by lovinglinux on 2010-08-26
If you don't want the wizard or if it pops up again, then type **about[noparse]:[/noparse]config** and the address bar, then type **extensions.ghostery.tutorial** in the filter and double-click the result to toggle it's state. If this property is set as **false**, it will trigger the wizard every time you open a new window.
