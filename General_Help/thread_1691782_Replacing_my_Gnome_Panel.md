### Post by manoriax on 2011-03-20
> **todd1049 said:**

Easy way: Right click -> Add applet (or whatever it is called these days) and select the right applet, there. If that does not work, reset the panel configuration.

This command **resets** the panel configuration to **default**:
```
gconftool-2 --recursive-unset /apps/panel 
```
To make the change visible, restart your panel:
```
pkill gnome-panel 
```
