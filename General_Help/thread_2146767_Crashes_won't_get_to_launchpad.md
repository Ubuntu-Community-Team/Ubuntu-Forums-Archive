### Post by checoimg on 2013-05-31
I solved this by going to /var/crash with nautilus and right click the .crash file with correct name and clicking "open with Report a problem".
To know which file to use when you get the first report click "view details" before clicking "continue" if it doesn't upload then proceed with the previous steps.
