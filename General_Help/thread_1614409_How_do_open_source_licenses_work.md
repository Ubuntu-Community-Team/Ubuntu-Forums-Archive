How do open source licenses work and what ones are recommended for program/application development? ### Post by efflandt on 2010-11-05
The basics are that if you modify open source code or use parts of that in your own code, you should include that license and make your code freely available.  The specifics would be based on the license of the code you use.

If you develop something completely on your own, you are free to use whatever copyright or license you want to.
