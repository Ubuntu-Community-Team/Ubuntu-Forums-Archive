### Post by stinger30au on 2008-05-25
i have a directory called downloads as well but not on my main hdd. there i store all my downloads i get while using firefox.


for any programs i can get that i would like in the standard ubuntu repos i go and visit 
[www.getdeb.net](www.getdeb.net)

here there is plenty of files that i can add to my pc.

i save these to the download directory i created and then i can double click them and install them. it brings up a package manager and installs the software no worries.

at this point i have not found any software i need that i have to build from the source code.

you will find also that a lot of programs if you look at their homepage have got a downloads page and if you look there it will tell you how to add the program yourself to the repos on your pc so it will automatically check when a new version comes out and download and install it for you.
