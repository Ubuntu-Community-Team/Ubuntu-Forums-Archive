### Post by rainwalker on 2007-02-08
I'm almost finished with a big english project for school I made using Scribus, but I need to add the final touches and I have a question: At the bottom of each left page (I used a template to make this project) there's the page number on one corner and the text "newsletter header" in the other. The problem is, that "newsletter header" text never changes, and I don't want it there if I'm not using it. Does anyone know how to get rid of it?

---

### Post by emchai on 2008-03-15
Hi,

Go to Edit/Master Pages....
From there, click Inner Page Right or Inner Page Left, depending on which page is the "Newsletter Header."  Right click on the item box to unlock it.  Then, go to Item/Delete to get rid of it.

You're probably done your project by now, but hopefully helps someone else with the same question.
