### Post by NightwishFan on 2008-03-03
Are you using Ubuntu 7.10? If so then you should have compiz-fusion installed by default. All you need to do is install compizconfig-settings-manager and emerald.

Compiz-fusion is the merge between Compiz and Beryl.
