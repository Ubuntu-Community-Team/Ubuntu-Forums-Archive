### Post by plucky on 2011-09-15
> i still have ubuntu 10.something cd cuz my internet is so slow.

From a terminal ```
lsb_release -a
``` will tell you what version you have.

If it is 10.04,you will have to upgrade to 10.10 first and then onto 11.04.If you are on 10.10,then it will upgrade to 11.04.

If your internet is really slow,I would go down the route of downloading the "Alternate" install CD and upgrading using that,so if anything goes wrong with the upgrade,you can do a fresh install with the new "Alternate" CD.

Good luck
