### Post by ram2500 on 2010-03-12
I use my "server"--if you would call it that, as it is more of an external hard drive--for keeping my files synchronized with those files that are on my laptop. My server **does** get turned off when not in use, and in all actuality, it is my desktop machine, just with file server abilities. 

It's nothing to write home about:p, just a standard desktop machine. I use it when I don't use my laptop. However, I have two IBM Intellistation workstations in my classroom that are my own machines for the classroom network (there are 30 computers in my computer room, and six in my regular classroom). These stand between the school network servers and stores student assignments on them in my Web design class (I also share space with the Graphic Design class down the hall). These are very robust machines, having an Opteron processor and 2 GB of RAM, plus a RAID array that I have added. 

A regular PC with at least a Pentium 4 or Athlon 64 processor with at least 512 MB of RAM should suffice as a server, although your mileage will vary depending on what you are using it for. The IBM Intellistations that I have in my classroom are probably more than I need (One's a web server and the other is a file server), but I bought these from a corporate surplus sale because they were really inexpensive, costing only $150 apiece (but without a hard drive and only a CD-ROM drive--not a big concern). Old Pentium II/III PCs can even be used, just don't expect them to be useful as a multimedia/Web server.
