### Post by tonybaqain on 2008-10-05
if you didn't create any of the desired targets use ```
sudo
``` under bash (shell or terminal), or press ALT+F2 and type ```
gksudo nautilus
```, either ways you have to put your password, then you can navigate where you want, **but please make sure that you are now a super user, you can harm your system, don't do anything you don't know, and read this announcement [http://ubuntuforums.org/announcement.php?f=331]("http://ubuntuforums.org/announcement.php?f=331")**.

have fun :)

### Post by Tomatz on 2008-10-05
The only folder that you have read/write permissions is your home folder. If you want to access the root filesystem then type the following into a terminal:

sudo nautilus

Then enter your password (no chars will show)

**Be careful**
