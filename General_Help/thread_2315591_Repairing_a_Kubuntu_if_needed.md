### Post by deadflowr on 2016-02-29
> **dave157 said:**
> Thanks for the reply
Reinstall without formatting have to look into that, did not know that was possible.

I've done it before.
Note that it does take longer.

But it does so because the installer is confused by the existence of files already in the Home directory.
So every time the installer comes across a folder or file that it  usually would add to the system that is already in the system, it pauses  to note it
with a warning.

Or at least that is how I remember it; it's been perhaps 2 years.

If you use the regular Ubuntu installer, during the actual installation  time, post adding your user info/password page, there is a Details or  Terminal dropdown right below the progress bar. If you expand that you  can see the installation process and what is happening as it goes, which  will include warnings and errors.
