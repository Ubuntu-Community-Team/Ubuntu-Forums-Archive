### Post by ghpowers on 2008-04-10
I have a weird problem with Ubuntu.  Seemingly randomly, my computer will switch back to a previous state.  Its like there are two different systems installed on my computer at the same time and the computer randomly chooses which one to boot.  I realize this isn't actually what is happening, but this is what it seems like.

Basically whenever I log on to Ubuntu, I will get one of two configurations.  The first one is the "old" configuration.  It is a configuration from a few months ago, but any changes I make, such as installing new programs or changing the wallpaper, are saved the next time I get this configuration when I log on.  The other configuration is the "recent" configuration.  Just like the old one, any changes I make to are saved and show up when i log on and get this configuration.

The only way I know to switch between the two is to reboot a few times and usually the other will show up.  This is probably one of the weirder computer problems that I have ever seen and any help is appreciated.
