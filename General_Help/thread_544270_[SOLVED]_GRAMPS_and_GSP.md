### Post by artinla on 2007-09-06
Probably the best way would be to use the old family tree maker to export a .ged file and then you can easily import that into GRAMPS.

If you don't have access to the family tree maker anymore, you may have to get someone to convert the file for you.  You could post the request on any number of ancestry message boards and I am sure someone would help you.

### Post by artinla on 2007-09-06
The option in FTM may say something like "Export _Gedcom_ File".  That is the proper name and it is a standard export/import format among many different family tree programs.  I am not sure, but I think it is something like a comma seperated values file that can be imported easily into other family tree software such as PAF or even into a spreadsheet.
