It sounds like something went wrong.  I am running Ubuntu perfectly on an EA160-199 (which I think is almost identical) and everything works perfectly.

The best way to run it alongside (without any hassles) is to install XP, leaving e.g. half of th drive free, then install Ubuntu.  Ubuntu will put on GRUB (a boot loader) which will let you choose when you boot which operating system you'd like to start up with.

You can get Crossover to run stuff like MS Office, or run a virtual machine like VMware or Parallels.  Then again, I don't really use much Windows-only software these days.
