### Post by angry_johnnie on 2008-05-22
6.06 is pretty old :-), but it should be like this, sort of...

1. To change themes go to System > Preferences > Appearance (or "Theme" maybe)

2. If there are any updates available, you'll see an icon in your system tray telling you about it.

3.  There is an ubuntu version of limewire.  It should end with a .deb (as opposed to windows's .exe).  It is available at limewire's site.

4.  Depends on what you want.  Most programs --if that's what you're looking for, anyway-- are available for download through **Applications > Add/Remove**.  There's lots of stuff in there.

For anything else, there's plenty of help in the forums.  If you have any doubts, try the search function.  If that fails, you could always google for something like "ubuntuforums my problem" and you'd probably get an answer.

Cheers.
