### Post by requeth on 2011-08-15
When you connect the ethernet cable does your network symbol in the upper right show anything? The symbol is the 4 lines that get bigger the higher they go. When you connect they should go solid. If you click on the network symbol it should show Eth0   Connected. If not, are you connecting directly to you modem? Or to a router in between. Have you checked to make sure the cable is plugged in at both ends, and that the router/modem is on and working?

If you have a wireless router, does the network show up in the list if you click the Network symbol? If you have security enabled you'll need to click the name of your network the first time to give it the password before it'll connect you each time.

If any of these show as connected, try opening a command shell and typing ping google.com. Do any packets come back?

If you type iwconfig does an IP address show up? If no IP then your not connecting to your modem/router properly somehow either hardware or software.
