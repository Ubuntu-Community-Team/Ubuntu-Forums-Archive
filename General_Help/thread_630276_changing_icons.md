### Post by NilsE on 2007-12-03
Most icons are in folders under usr/share/icons depending on which set of icons you are using in appearances.  So look in appearances and see which one is selected and browse through that folder looking for the current one. If you don't find it it means it is provided by the application and you can look within it's folders.

May help just to browse to the usr/share/icons folder and search for the word pdf
