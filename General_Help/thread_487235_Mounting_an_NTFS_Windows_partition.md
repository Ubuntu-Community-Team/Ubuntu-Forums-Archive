### Post by merlinus on 2007-06-28
```

sudo mkdir /media/sda1

```

then:

```

gksudo gedit /etc/fstab

```

and add this line to the end:

/dev/sda1 /media/sda1 ntfs-3g defaults,locale=en_US.UTF-8 0 1

Reboot, and see what happens....

-merlin
