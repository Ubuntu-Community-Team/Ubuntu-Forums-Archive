### Post by coffeecat on 2009-11-19
Double-check that you haven't got caps lock on. I've never heard of an upgrade corrupting the username and password.

In an ordinary installation (i.e. not Wubi) there's a way of resetting the password, but you have to know the username. It involves booting up in recovery mode. I'm not too familiar with Wubi - is there an option to boot up in recovery mode? If so, and if you are sure of the username, I'll give you more details so that you can reset the password.
