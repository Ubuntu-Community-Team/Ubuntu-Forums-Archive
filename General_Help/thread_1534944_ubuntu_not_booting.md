### Post by varunendra on 2010-07-21
Try booting into recovery mode (press and hold 'Shift' key while booting). There, choose 'FailSafe Graphics' option.
It should allow you to logon in low graphics mode. Then try rolling back the updated graphics driver and replacing it with a proprietary one for your graphic card. Or you can try using the default one (older one).
