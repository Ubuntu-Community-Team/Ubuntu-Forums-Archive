### Post by dabl on 2011-04-27
A couple of things could cause that.  Can you boot a Live CD?  If so, try letting it run with a Live CD -- use the browser and play around with it.  If it does not reboot running a Live CD, that would be a useful thing to know.

If the rebooting problem is about the same whether it is booting the installed OS or booting a Live CD, then that means it is purely a hardware problem with the computer, and there's nothing you can do with software to fix it.
