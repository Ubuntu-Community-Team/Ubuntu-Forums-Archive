### Post by ramjet_1953 on 2007-07-29
Some questions.

1.Did you burn the image as an iso9660 image?
Just burning it as a data CD won't work.
Most burning software has that option in one of the drop-down menus.

2. What speed did you burn the image at?
If you burn ANY iso image faster than 4 X you will have problems.

3. Are you using high quality media?
Any iso requires good media. Budget stuff doesn't cut it with iso images.

4. Did you do an MD5 checksum check on the disk?
