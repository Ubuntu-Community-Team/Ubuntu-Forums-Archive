Then I suggest that you run ```
sudo lshw > lshw.txt
``` and attach [FONT="Courier New"][SIZE="3"]lshw.txt[/SIZE][/FONT] to a reply, and post the output of the command ```
glxinfo|grep -i opengl
``` to the same reply.

If you like, you can also install hardinfo, which is a nice GUI application to view information about the hardware.
```
sudo apt-get install hardinfo
```

Anyway with a working Ubuntu 12.04, you can install the light-weight desktop environments into that instead of downloading the separate flavours Lubuntu and Xubuntu iso files.

```
sudo apt-get install lubuntu-desktop
```
```
sudo apt-get install xubuntu-desktop
```
