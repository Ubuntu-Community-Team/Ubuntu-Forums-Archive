### Post by erflol on 2009-01-06
Are the hard drives removable? If so do you want to be able to access them on any other computer? Do you want to provide the password once? or every time you want to use the drive?

You can always encrypt the entire hard drive which will provide security and you will require a password to use the drive.

Check out the following thread:
[http://ubuntuforums.org/showthread.php?t=572365](http://ubuntuforums.org/showthread.php?t=572365)

I would like to make the same recommendation of Dayvy and wheredidrealitygo and say use Cryptkeeper and password protect a folder on the drive. (I'd imagine you'd be able to password protect the folder the drive is mounted as, effectively encrypting the entire drive, but I haven't tried that.)

Cheers
eRflol

### Post by erflol on 2009-01-07
You can back up a drive onto itself, although if you only have a single partition this isn't all that useful for this scenario. If you have multiple partitions and you only wish to encrypt one partition that would work.

Ideally you would back up onto a different drive. It is not required that you back up at all, but as mentioned before it is certainly recommended. 

Check the size of the data you want to back up, perhaps burning it to a cd or dvd would be appropriate?

Cheers
eRflol
