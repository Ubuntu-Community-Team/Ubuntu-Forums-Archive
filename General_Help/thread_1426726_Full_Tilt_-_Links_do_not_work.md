### Post by Rob0147 on 2010-03-11
After a bit of playing about I found a workaround.

I installed Virtualbox OSE from the applications - Ubuntu software centre where I was able to install a Virtual Windows Vista PC.

Then I installed FTP and pokerstars software where I have full functionality from there if I need it.

To run virtualbox properly I had to go to a terminal and type:

sudo virtualbox

HTH if anyone needs to know.
