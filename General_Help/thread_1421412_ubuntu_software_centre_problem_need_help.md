### Post by plucky on 2010-03-04
Check your **Software Sources** are enabled.

**System > Administration > Software Sources** and make sure the boxes are ticked (See screenshots)

Then select **Reload** to download the index files.

Now see if you can install from the Software Centre.

Good Luck
