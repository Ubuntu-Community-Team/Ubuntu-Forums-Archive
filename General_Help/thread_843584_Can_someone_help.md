### Post by Elfy on 2008-06-28
You need to -

enable graphics driver - check system >admin >hardware drivers first

go to appearances in system > prefereences and then visual effects and enable custom

install compizconfig-settings-manager - either use synaptic and search for it or open aterminal and use this command

```
sudo apt-get install compizconfig-settings-manager
```
password is your password, it will not be visible

Edgy and feisty are old versions, hardy is the latest.
