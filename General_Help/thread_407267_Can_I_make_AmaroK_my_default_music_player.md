### Post by nike984 on 2007-04-11
yes, it's quite easy.
open a nautilus and go to any mp3 file or music file that you want to open
with Amarok.
right click on the media file and click 'property'
Then go to the 'open with' tab.
In there, you can choose the application which you want to be a default player for
the files with the same format.
So, choose 'Amarok' and close the property window.
Done :o
