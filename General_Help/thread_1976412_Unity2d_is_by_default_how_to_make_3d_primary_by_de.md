### Post by Henne91 on 2012-05-08
Is there an error message? Which version of Ubuntu are you using?

If you choose "Unity" in the menu right after booting your system and log in it should automatically become default and start everytime you boot.

Otherwise, your graphics card might not be able to run Unity-3D for example because of a driver not working or something else.

What graphics card are you using?
