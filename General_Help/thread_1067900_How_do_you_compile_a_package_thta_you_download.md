### Post by howlingmadhowie on 2009-02-12
usually you don't.

software is best installed over System->Administration->Synaptic package manager

if you don't find the piece of software you're looking for, ask again with the specific name.

software packages can often be compiled with:
./configure
make
sudo make install

in the untarred directory with the software in. but this can get complicated very quickly.
