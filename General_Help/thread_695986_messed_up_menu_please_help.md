### Post by ola on 2008-02-13
If nothing else works you can move / delete all your gnome configuration files for that user. They are located in your home folder.

I'm not really sure on what folders you need to rename / delete but a good start should be the ones named .gnome and maybe the ones named .gconf.

You can see a list of them here: [http://www.yolinux.com/TUTORIALS/GNOME.html#CONFIGFILES](http://www.yolinux.com/TUTORIALS/GNOME.html#CONFIGFILES) (not sure if they all are named exactly like mentioned at that site..)

Hope it was a step in the right direction.

You can always try and create a new user to make sure it's a user specific problem (instead of a problem with your installation).

Best of luck!
