Is it possible to set Terminal to open up in a specific dimension? By default, it opens up in 80x24. Is it possible to set it to default open up in, say, 55x17?

### Post by Darkshade on 2009-04-24
Change your launcher from "gnome-terminal" to "gnome terminal --geometry=55x17" (or whatever size you want). Without the quotes of course :P
