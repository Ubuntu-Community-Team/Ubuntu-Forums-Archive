First timer on this side of the system. Running a Compaq Presario F700, not sure of all the specs, know 1 gig RAM, GeForce 7000, etc. Restricted driver download after install recognized display, allowed me to set the screen resolution, no problems. I don't have a wireless network, so cannot comment on the wireless card.

Just wanted to say thank you and give one minor success.
