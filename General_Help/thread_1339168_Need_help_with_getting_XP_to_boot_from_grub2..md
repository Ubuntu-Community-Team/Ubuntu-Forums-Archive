### Post by oldfred on 2009-12-01
I would copy your entry into 40_custom and modify (I think either will work but you can put both in and see):

menuentry "Microsoft Windows XP Home Edition (on /dev/sdb1)" {
    insmod ntfs
    set root=(hd1,1)
    search --no-floppy --fs-uuid --set 9af49ee4f49ec1c5
    drivemap -s (hd[COLOR=Red]1[/COLOR]) ${root}
    chainloader +1
}

menuentry "Microsoft Windows XP Home Edition (on /dev/sdb1)" {
    insmod ntfs
    set root=(hd1,1)
    search --no-floppy --fs-uuid --set 9af49ee4f49ec1c5
    drivemap -s (hd1) ([COLOR=Red]hd0[/COLOR])
    chainloader +1
}

If it works then to eliminate the duplicate entry:
gksudo gedit /etc/default/grub
GRUB_DISABLE_OS_PROBER="true"

IF you every reconfigure you can turn the prober on to find new systems and then turn it off again after you copy entries to 40_custom.
