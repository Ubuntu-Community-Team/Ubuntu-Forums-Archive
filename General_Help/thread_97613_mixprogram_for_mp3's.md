### Post by Bandit on 2005-12-01
Rhythmbox for Gnome, amaroK for KDE.
Rhythmbox-0.9.2 is very stable, but I dont recall it having cross fading.
amaroK is 99% stable when not using cross fading, but with crossfading (you have to use the xine engine for Xfading) it is only 90% stable. Unless you are changing songs very very rapidly there is normally not a issue with the xine engine crashing.
They both can shuffle, mixup with genre's, play inet radio stations and use IPODs.
Cheers,
Joey
