### Post by bailout on 2006-11-26
I would probably advise switching to a native linux program. I haven't tried picassa but I understand it is a bit of a hack on linux. As a kubuntu user I use Digikam and would definately recomend that you try it. It has quite good basic editing inc 16-bit support and will handle most raw formats. Your Canon definately should be ok, not sure about the Samsung. It does have an export to Flickr plugin and an 'export to remote gallery' plugin but I don't know whether this can handle picassa directly. If you try Digikam make sure you install digikam plugins and kipi plugins. Digikam is in very active development and version 0.9 is well advanced. Hopefully there will be k/ubuntu packages realeased when it goes final.

Don't know much about f-spot. The best way would be to install both and see which fits your needs best.
