### Post by papibe on 2012-03-15
Hi l1ghtchaos. Welcome to the forums!

If I understand correctly you want to connect to the server just like you did from Windows using putty, but now from another Linux/Ubuntu machine.Is that correct?

If so, it is very simple since the ssh client is already installed by default. Open a terminal and run:
```
ssh DebianServer
```
Change 'DebianServer' with the proper name of your server.

I hope that helps, and tell us if you need more help.
