### Post by orbital on 2009-07-15
I installed Firefox 3.5, and would like all programs that open links to Firefox to use 3.5 as a default browser. The problem is now all links (Pidgin etc.) open the links to an earlier version of Firefox. How to solve this problem?

---

### Post by SeanBlader on 2009-07-15
System -> Preferences -> Preferred Applications 

Change the Web Browser to "Custom" and put this in the command line: 

firefox-3.5 %s

All set.

---

### Post by orbital on 2009-07-15
Thanks, everything works now properly.
