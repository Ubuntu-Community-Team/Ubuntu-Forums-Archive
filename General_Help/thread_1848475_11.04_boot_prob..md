### Post by garvinrick4 on 2011-09-23
Put a install cd in of ubuntu and boot off of it and use Try Ubuntu.
Open a terminal ctrl/alt/t
```
sudo fdisk -l
``` (lower case L)

Pick out your ubuntu install sda#
I will use sda5 for this, use your own
Run these 4 pieces of code one at a time.
```
sudo mount /dev/sda5 /mnt
sudo grub-install --root-directory=/mnt /dev/sda
sudo umount /mnt
sudo reboot
```

### Post by garvinrick4 on 2011-09-23
We are going to go right into the / itself and install from within its own file system.
When you run update-grub below you should see the installs. Make sure you plug your usb ubuntu drive in and then plug the flash drive in and boot into flash drive. In that order.
Copy and paste one at a time.
```
sudo mount /dev/sdb1 /mnt
``````
sudo mount -o bind /dev /mnt/dev; sudo mount -o bind /dev/pts /mnt/dev/pts; sudo mount -o bind /proc /mnt/proc; sudo mount -o bind /sys /mnt/sys
``````
sudo chroot /mnt
``````
grub-install /dev/sdb
``````
update-grub
``````
exit
``````
sudo umount /mnt/dev/pts && sudo umount /mnt/dev && sudo umount /mnt/proc && sudo umount /mnt/sys; sudo umount /mnt
``````
sudo reboot
```Done. You choose Ubuntu drive sdb to boot into now.
