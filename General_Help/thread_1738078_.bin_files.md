### Post by Vaphell on 2011-04-24
in properties (rightclick) mark it as an executable or in terminal (assuming you are in right directory)
```
chmod +x file.bin
```

if it's gui installer clicking it should work, if not then in terminal
```
./file.bin
```

### Post by Vaphell on 2011-04-24
there is no space between ./ and maple13linux32installer.bin

where is that file exactly? In Downloads? Desktop?

assuming Desktop
```
cd ~/Desktop
chmod +x maple13linux32installer.bin
./maple13linux32installer.bin

```
if it complains about permissions
```
sudo ./maple13linux32installer.bin
```
