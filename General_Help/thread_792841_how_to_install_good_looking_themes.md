Hey there and welcome to the fold...

First up answering your question - you don't "need" to install emerald.  You need emerald installed only if you want to use emerald themes.

There so much you can customise with linux it can be mind boggling but dont be put off.  Its a good thing really.  :)

You are not just limited to changing a theme, a font, or some colour scheme.   You can even change window managers, decoration managers, and such to get even more range of tinkering available to you.

If you are running ubuntu v8.04 (Hardy) -  it seems to use GTK2 as decorator (if thats the right term) and so you can rock along to a site like gnome-look.org and look for GTK2.x themes and they will install easily for you.  (for kubuntu there is kde-look.org)

If you spot an emerald theme you like so much then you need to install emerald.

Then once you are finished toying with desktop themes you might want to look at grub-splash themes, usplash themes (what you see as your PC is booting), GDM themes (the login screen), and so on..  

From there you can look at rearranging the desktop totally, maybe throw out the panels and replace with "docks" or splash some screenlets around.

You can customise what your PC looks like pretty much from when the bios has finished its thing at startup... til the shutdown screen winks out.

Hope that gets you started.

Actually that one has to be unzipped before it will install.  (I tried it).

You downloaded 54573-Murrina-Aero.zip
Right click on it and click "extract here"
Now you will have a folder in the same folder/desktop called Murrina-Aero

System>Preferences>Appearance ...   now drag and drop the Murrina-Aero folder onto the now opened themes tab

Magic.  :)

If the archive was a tar.gz then you can just drop the whole archive on to that themes tab inside System>Preferences>Appearance.

### Post by tommyhakinen on 2008-05-14
Get a nice theme from [www.gnome-look.org](www.gnome-look.org).

Here is one of the nice one:

> [http://www.gnome-look.org/content/show.php/SlicknesS?content=71993](http://www.gnome-look.org/content/show.php/SlicknesS?content=71993)

1.Download the file to your desktop
2.Right click the file and select extract here
3.open a terminal and run these commands:

> sudo cp -r $HOME/Desktop/<extracted folder> /usr/share/themes

After that, go to System > Preferences > Appearance
The new theme should be availabe for you.
By doing this, the theme will also be applied to all the user including root.
