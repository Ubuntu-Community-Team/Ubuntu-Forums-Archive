### Post by azadpanchi on 2008-03-07
I use Clearlooks and just customize it for my needs.

Additionally you can install the package 'gnome-art' I find it much better then downloading from [http://art.gnome.org](http://art.gnome.org)

Here is a good link with great pointers on making Gnome look better:
[http://polarbeardk.blogspot.com/2006/11/gnome-eyecandy.html](http://polarbeardk.blogspot.com/2006/11/gnome-eyecandy.html)

Also you can install the package 'emerald'  a very cool tool to customize many aspects of the desktop look.
