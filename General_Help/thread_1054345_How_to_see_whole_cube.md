### Post by gjoellee on 2009-01-29
So you have compiz installed right? If so open the Compiz settings manager from the menu.

(It can be launched from terminal with this command:
```
ccsm
```
)

Then search for "cube". In one of you results you should fine a "Zoom" button. Zoom out and find the zoom that fits you nicely.

### Post by Bigneil on 2009-01-30
open ccsm  (advanced desktop effects setting)  and make sure you have the cube enabled and also the rotate cube plug-in enabled. You must also go to 'general options' then desktop size and move the slider for 'Horizontal virtual size' up to 4.  

You should now have cube working when you push [ctrl] [alt] [mouse button 1]

you can now adjust the amount by which you would like to zoom out on to your 3D cube by clicking on the 'rotate cube' and moving the slider for the zoom in and out.


Hope this is helpful;)
