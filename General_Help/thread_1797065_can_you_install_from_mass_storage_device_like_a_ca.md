FYI latest thing I tried was my nokia nuron and set the phone's memory to automatically connect via usb mass storage device


Works like a charm

Thing is on the phoenix bios I was confused because when you set the boot order the only selection you get is fdd usb, zip usb, cd usb etc. 

But in actually this has nothing to do with it, there is another option that you don't see and have to select hard drive priority and the usb devices will be listed there instead.

Just select usb hard drive or the usb devices that you have connected they will show in the list and my phone shows there so I just select boot to nokia usb.

Actually quit simple once I figured out that setting.
Just wanted to post an update on this.

That it will appear to work with any usb device mass storage as long as you can select drive priorities or select usb device boot first or something to that nature in the bios level.
