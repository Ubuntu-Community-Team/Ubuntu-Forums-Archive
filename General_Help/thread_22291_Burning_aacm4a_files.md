### Post by kevinatkins on 2005-05-08
One way I've found to do this is to grab the MP4 plugin for XMMS, then select the 'disk writer' output plugin. Then XMMS will convert your MP4's to WAV format, ready for burning in K3B.

Not very elegant perhaps, but it works. I'm searching to see if there's a K3B plugin available for MP4 files - would be nicer really.
