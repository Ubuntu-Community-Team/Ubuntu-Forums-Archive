### Post by canadianwriterman on 2008-06-05
I've always had issues with Ubuntu version of OpenOffice.org 2.x eating up a lot of CPU (at least 50%), especially after it has been running for a while. I use it to do presentations while I teach, and the longer Impress is running during the class, the slower it gets, especially with slide transitions.

So, I D/L OpenOffice.org 3.0 Beta and installed it parallel to the Ubuntu version. What a difference! The new OpenOffice.org consumes a lot less CPU. Although it is beta and there are still some features not working, I urge anyone who uses OOo regularly to give it a try.
