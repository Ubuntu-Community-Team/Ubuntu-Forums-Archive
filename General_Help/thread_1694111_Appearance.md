### Post by towheedm on 2011-02-23
To install a theme from the Appearance window:
1 - D/L the theme file, usually in a .tar.gz archive.  Save it whereever you want, most likely in your ~/Download folder.
2.  Open the Appearance window.  On the 'Theme' tab, click the "Install..." button.  Now browse to the folder that the downloaded theme was saved to in Step 1.
3.  Select the desired theme.  Once it's installed properly, a message to the effect: "Theme xxx installed without errors" should appear.
4.  The theme should now be shown in the theme preview box.

After the theme installs, a .theme folder will be created in your home folder.  A theme installed this way, is available only to you.

To make the theme system-wide extract the theme archive D/L'ed in Step 1 to the /usr/share/themes directory.

Hope this helps.
