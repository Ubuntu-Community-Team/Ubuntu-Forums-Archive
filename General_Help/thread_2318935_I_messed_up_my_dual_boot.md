### Post by yancek on 2016-03-30
> Can someone tell me what the commands are to get it working again?         

Not until you let us know which version of windows you are using and whether you are using UEFI or MBR booting.  Do you still have the Ubuntu installation DVD or flash drive?  If you do, you can get the boot repair software and put it on the installation media of Ubuntu or you can boot windows and download the iso file for boot repair and burn it as a bootable image from windows, then reboot the computer with the boot repair CD in the drive and after it boots, select the option to Create BootInfo Summary.  You can then post a link to the output which will give useful information and someone should be able to help.

The link below is to the boot repair site.  Select either download it on the Ubuntu medium or to download the iso and burn it to a CD.  Both methods are explained at the site.

[https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)
