### Post by mrhealthpatriot on 2008-09-05
I solved setting swiftweasel as default by using the command swiftweasel3 %s
under preferred applications / other

Also, none of the swiftweasel icons showed up, so I had to find the swiftweasel icons folder: /usr/local/swiftweasel3/icons

hope that helps!
