### Post by hemimaniac on 2010-02-12
instead of that try 

sudo apt-get install devede 

avi needs to be burnt as a data project in k3b (hopefully your dvdplayer will play avi)

whereas devede will convert them to a dvd image file that can be burnt straight to disc
