Noob here with a question.

At Login if I click on session options there are several. It's evident what some of the options do. But two of them I can not figure out. ### Post by ajgreeny on 2009-12-16
They are basically the same.  The xclient script is set in GDM I think and if you have only gnome installed, that is what it will start.  If you have more than one desktop environment, eg KDE as well as gnome, you can choose either of the named DEs, or still choose the xclient script, but in GDM, or KDM if that is what you now use the xclient script might then not start gnome any more.

I hope that helps.  If it is confusing, don't worry; either choice in a gnome-only system will lead you to the same point.
