### Post by mcduck on 2009-02-22
> **chinmaya_n said:**
> can we disable that fullscreen mode feature.. rather than using F11 everytime ???

There doesn't seem to be any option in Firefox to do this, not even under about:config, so it doesn't seem to be possible.

However some people who have had the trouble with Firefox defaulting to full-screen mode have reported that disabling "Legacy Fullscreen Support" in Compiz settings would solve the problem.

edit: also simply moving out of the full-screen mode and then restarting Firefox *should* set Firefox back to using normal mode by default.
