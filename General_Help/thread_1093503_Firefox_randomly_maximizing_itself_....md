### Post by yankeeDDL on 2009-03-11
I'm taking a wild guess here, but the maximization could be due to Java code in one or more of the pages you are visiting. Some website do and I dislike that very much.
In that case the only option you have is to disable Javascript. Rather than killing Java all together, you could leave it enabled in Firefox and then use "noScript" Add-on which allows you to selectively enable Javascript only on selected websites.
I used it a lot on windows as added security for viruses. In Ubuntu the risk of virus infection is practically non existent, so I don't have it. But you could give it a try.
