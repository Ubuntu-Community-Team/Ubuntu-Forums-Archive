### Post by SteveDee on 2011-11-30
> **rng said:**
> ...do I need to start it separately?

Yes you do. Normally you would install gnome-search-tool on an LXDE system (e.g. Lubuntu, Peppermint) and this would create a menu entry for you. But in your case, as its already installed as part of Ubuntu, it does not get picked up by the LXDE menus.

You could try un-installing and re-installing gnome-search-tool (within LXDE). But if you are just experimenting with LXDE you can open a terminal and type: gnome-search-tool
