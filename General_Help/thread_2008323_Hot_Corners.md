### Post by gr8 on 2012-06-22
I've got my hot corners setup that mimic the Mac OS X expose and show desktop. I did it through Compiz and it works great.

There is a windows 7 feature to set a hot corner as "last application". It's similar to the "last channel" feature on remote controls. It just cycles between the current application open and the last application open. It's real handy in my workflow.

Is there anyway to get this "last application" feature with Ubuntu hot corners?

### Post by LewisTM on 2012-06-22
I'm not too familiar with this feature but it sounds like a classical Alt-tab that switches between two apps.
So to trigger Alt-Tab on a corner simply use this command
```
xdotool key Alt+Tab
```
Xdotool has to be installed first.

Cheers!
