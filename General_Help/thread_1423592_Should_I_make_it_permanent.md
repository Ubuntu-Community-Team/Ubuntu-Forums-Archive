### Post by flippo on 2010-03-06
When I dumped windows I had installed ubuntu and been running it for a while.  I had not used windows in such a long time that the disk had corrupted itself because of so many improper poweroff and startups (you know, they happen over time...) to the drive.  I decided that since I hadn't noticed it was completely dead, and unrecoverable that I didn't need it, and it was an excellent decision.

I also had mac on my computer, I decided to wipe that one too.  I later regretted wiping mac because I needed it for firmware updates...  I had to wipe my whole drive and reinstall everything nice and cleanly (minimal 10G mac, all else to linux).  

The moral of my story is:
If you don't use windows for anything, at all, there is no point keeping it around. On the other hand you might be surprised what you need it for (like my mac/firmware issue).  

If I did it all again, I would make the windows install minimum (10-15Gig depending on your HD size) and give everything else to ubuntu.  After 6(ish) months I would just wipe windows and be done with it.

Just my 2 cents.

### Post by gordintoronto on 2010-03-07
I have had Ubuntu as the only operating system on my primary machine for more than two years.  I do a little web site development, all the Internet stuff, and most recently some video editing. I'm delighted!

However... I bought a Brother networked laser printer, and Ubuntu really wants it to have a static IP address. Brother provides a utility to set the printer to use a static IP address, and that utility runs under Windows. I installed it on my wife's computer, ran it once, and the printer works great from all our computers. (I also have a son and a stepson, who each have a laptop and a desktop.)

The moral: keep a small Windows partition, you might need it someday.  It's too bad there is no Windows LiveCD for cases like this.
