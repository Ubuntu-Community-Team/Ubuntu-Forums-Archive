### Post by retrow on 2008-04-29
There is a problem with the system-monitor itself. When you open the resources tab, it spikes the cpu usage. Install 'htop' which does the similar job as system-monitor. The fire up system-monitor, go to the resources tab and see how much CPU gets consumed by it. The problem isn't in Hardy, its with system-monitor.
