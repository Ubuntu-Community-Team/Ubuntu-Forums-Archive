### Post by Shazaam on 2008-10-25
After reading your other post try this...
System>Administration>Login Window, go to Default Session (on first tab). If it is set to "Gnome" change it to "Run Xclient script" (or the reverse) and see if that helps.
Or you can change it when Ubuntu gets to the login screen. Look for "Session".
