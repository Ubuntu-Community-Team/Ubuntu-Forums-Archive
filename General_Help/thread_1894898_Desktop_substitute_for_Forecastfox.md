### Post by hopefulkayaker on 2011-12-13
Before I was recently informed as to how speedy Chrome had become, and how sad Firefox had become, I used the Forecastfox plugin on Firefox, and I loved it!

What set it apart from other weather widgets/plugins for me was how it has the option to directly display a weather forecast (in such areas of the browser as the Bookmarks Bar or the nofitication bar [not sure on the proper name of that element; it's on the bottom]). This allowed me to effortlessly keep up with my local weather, not just with current conditions.

I have now made the switch to Chrome and am otherwise very happy about that switch. BUT, I have not found a Chrome weather plugin which displays a weather forecast. There is even a Forecastfox for Chrome, but it lacks the display options that it has in FF. It just displays the current conditions, forcing you to click the icon to see the forecast. This may sound like a trivial difference, but I can tell you that I have been much less aware of the weather since using the Chrome version of Forecastfox.

I then tried to find a dekstop widget (for Ubuntu 11.10 Unity2D) I tried installing Conky, which was an unmitigated disaster which I gave up on (It took a lot of work to figure out how to set up and get working. I couldn't figure out how to configure it how I wanted to, and then discovered that Conky's primary purpose is system monitoring, so I just deleted the damned thing).

After giving up on Conky I searched for "weather" in the Ubuntu Software Center, and based on the strength of ratings only one option stood out - Weather Monitor. It's slightly better than Chrome's Forecastfox as it lives up in the panel and thus runs when I do not have a browser open (which does happen once in a while, haha). But it has no option to display a forecast without clicking on the icon. This makes it pretty useless to me.

Is there anyone else out there who cares about having a weather forecast displayed automatically (whether it be on the desktop/panel or in a browser, the former being preferable)? What plugin/widget are you using?

I'll be pretty disappointed if this is a random thing which Firefox excels in, but happens to be missing from both Chrome and Ubuntu. :-(
