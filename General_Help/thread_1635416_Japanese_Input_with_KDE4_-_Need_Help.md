### Post by bcschmerker on 2010-12-02
**Good luck on this project!**  I've been using GNOME Character Map a lot for handling specialized characters that, in most versions of Microsoft® Windows®, I can access through the Alt-keypad sequence.

Canonical® has been hard at work on providing Ubuntu® with suitable resources to handle Japanese, and it has been a struggle with both GNOME and KDE window managers.  Microsoft has much of the commercial world outside of Japan well-equipped for Japanese through an integrated Input Method Editor in recent versions of Windows®.  But direct Japanese input is one area where nobody has caught up to [Personal Media]("http://www.personal-media.co.jp/") [Ch&#333;kanji®](http://www.chokanji.com/)---a descendant of the [TRON Project](http://www.tron.jp/) that, unfortunately, is almost impossible to obtain outside of Japan.

I myself have been stumped on finding the right hardware for a dual-language English/Japanese computer project.  The keyboard is the worst point:  International Business Machines Corporation never produced a full-size Model M with all the necessary keys in hardware (their 122-key versions are short provisions for the Reverse-Translate, Translate and Kana/Nihongo keys at the spacebar row), subcontracting instead to Brother Industries for a downsize version for Japan that will not take standard IBM 1392-xxx keycaps.  Even on eBay® I run across Japanese-language keyboards but rarely.
