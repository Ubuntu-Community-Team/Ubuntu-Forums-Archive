### Post by cilynx on 2009-10-24
All of your day-to-day progs will be compiled for 64-bit if that's the system you're running.

From a practical point of view, however, you probably won't notice the difference unless you're running heavy math applications like very large spreadsheets, dynamic CAD, physics simulations, etc..

By running 32-bit you get the benefit that most every application and plugin in existence is still developed 32-bit and later ported to 64.  Things tend to Just Work more often in the 32-bin world.  This of course will change as some point as 64-bit catches on; however, that point has not yet arrived.

You want the really simple answer:

My 64-bit machine is running amd64
My wife's 64-bit machine is running i386
