### Post by Cheesemill on 2013-06-15
Support for 10.04 on the desktop ran out in April but it's still supported on servers for another 2 years.

Because both versions use the same repositories the packages that you get on a server install are still being updated, this also includes some things that you would have installed on a desktop as well like the command line tools. It's these packages that you are still getting updates for. This doesn't mean you should keep using 10.04 on the desktop for another 2 years though as none of the desktop packages will be getting security updates or bug fixes any more.
