### Post by Possum Films on 2009-12-07
Here are my personal preferences for each of those programs:

To download files from Internet: DownThemAll
DownThemAll is a Firefox addon so you won't find it in the Ubuntu Software Center. You can get it here: [https://addons.mozilla.org/en-US/firefox/addon/201](https://addons.mozilla.org/en-US/firefox/addon/201)

To convert mp3 files to .aac: VLC Media Player
Install VLC from the Software Center and then click on "Convert/Save" from the "Media" munu. 

To edit pictures: GIMP
GIMP is included in Ubuntu. 

To edit videos from my handy cam and to make DVD's: If you want something similar to Windows Movie Maker you should try Kdenlive (can be found in the software center). If you are more serious and want a program with the same level of features as, say Adobe Premiere, you should try Cinelerra ([http://www.cinelerra.org/](http://www.cinelerra.org/)). 


To download torrents: Transmission
Transmission is included in Ubuntu. 

To play music & videos: VLC media player
Note that if you want to play DVDs you will need to install libdvdcss, which you can get here: [http://packages.medibuntu.org/karmic/libdvdcss2.html](http://packages.medibuntu.org/karmic/libdvdcss2.html)
