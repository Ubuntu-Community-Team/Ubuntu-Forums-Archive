### Post by p_quarles on 2008-08-04

Well, essentially, yes. The AMD64 architecture is fully compatible with the x86 architecture that has been the standard for so long. If you install the x86 operating system on the AMD64 hardware, the hardware from that point forward acts exactly as though it were the 32-bit hardware.
