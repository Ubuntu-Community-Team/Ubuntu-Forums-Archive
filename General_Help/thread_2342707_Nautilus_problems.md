### Post by howefield on 2016-11-09
> **habana said:**
> The File System button has disappeared from the left hand pane. For example,To get into var or etc I now have to edit the location bar. Not a show stopper but really annoying

Use the *Other Locations* link in the left hand pane, then select *Computer* from the right hand pane.

> The Recent button produces only an error message "Oops! something went wrong. Unhandled error message. Message recipient disconnected from message bus without replying"

Not sure of the fix but can replicate that behaviour, most likely a reported bug... this one : [https://bugs.launchpad.net/ubuntu/+source/nautilus/+bug/1633162](https://bugs.launchpad.net/ubuntu/+source/nautilus/+bug/1633162)

You could add some heat to the bug by clicking *affects me too*.
