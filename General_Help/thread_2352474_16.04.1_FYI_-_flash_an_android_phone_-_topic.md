Moto G3(2015)

  edit I just checked and there is a guide on xda to do this and it looks like it's all fastboot so Linux would be fine. YAY!

In Motorola's site prep to doing bootloader unlock
[https://motorola-global-portal.custhelp.com/app/standalone/bootloader/unlock-your-device-b](https://motorola-global-portal.custhelp.com/app/standalone/bootloader/unlock-your-device-b)
says install android SDK and points to android-studio-ide-145.3537739-linux.zip . <--- no don't need all of SDK.

did:
```
kris@kris-Z97X-UD3H:~$ fastboot devices
The program 'fastboot' can be found in the following packages:
 * android-tools-fastboot
 * fastboot
Try: sudo apt install <selected package>
kris@kris-Z97X-UD3H:~$ sudo apt install android-tools-fastboot
  ...
kris@kris-Z97X-UD3H:~$
```

NOTE: there is also an android-tools-adb but so far I don't see a need to install it.

power phone off and put into fastboot (hold down PWR, ALSO hold down volDN for 3 secs or so) and plug USB phone to computer, then:

```
kris@kris-Z97X-UD3H:~$ fastboot devices
no permissions    fastboot
kris@kris-Z97X-UD3H:~$ sudo $(which fastboot) devices
[sudo] password for kris: 
ZY(fake)SLM    fastboot
kris@kris-Z97X-UD3H:~$
```

```
sudo $(which fastboot) oem get_unlock_data
```
get key from Motorola, then
```
sudo $(which fastboot) oem unlock your-key-goes-here
```

voila, "unlocked" per motorola.  I will continue this.
