### Post by Ertai87 on 2007-12-30
Until a couple days ago, when I started Pidgin, it always popped up with the buddy list window.  A couple days ago, this stopped happening; starting Pidgin merely put an icon in the taskbar. ### Post by Ertai87 on 2007-12-30
Nevermind.  For anyone else having the same problem, I didn't notice that Pidgin had updated.  If anyone has the same problem, close the buddy list, right-click on the icon, and check off "show buddy list".  Restart Pidgin and the problem should be solved.
