### Post by Bucky Ball on 2011-11-29
I was almost there. This is pretty grubby and not the way to do it I wouldn't imagine, but it works and that's all I care about right now.

* Click on the page you don't want a page number on;
* Fomat>Paragraph>Text Flow;
* Check 'Insert' and set to 'Page';
* Check 'With Page Style', set to 'Default' and set page numbering to 0.

That page style will then continue until you stop it which means getting to the appropriate page and starting the numbering. That's kinda it but it was a bit of a fluke really.
