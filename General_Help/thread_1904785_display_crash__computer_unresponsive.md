### Post by JohnDillinger43 on 2012-01-05
I typically leave my computer on overnight, just turning off the peripherals (monitor, speakers, etc.).  When I went to turn on the monitor this morning, the monitor turned on, but was blank, i.e., power was on, but it was acting like it had no input from the computer.  I tried moving the mouse and using the keyboard -- nothing.  I checked the VGA cable, but that wasn't a problem either.  The computer was still running, so I used my laptop to check the shared folders on it, and couldn't connect to them.  After that I figured it must have partially shut down or something, so I used the magic sysrq key to reboot.  After that everything was fine, but I was wondering if anyone has ideas as to what happened?  Any other strategies I should have tried rather than just rebooting?  I figure it's not a display issue since I couldn't connect to the shared folders, but I'm running a Windows network that's not always reliable, so it's also possible the computer was actually running and connected to the network and my Windows laptop was having problems connecting, so I'm still not sure whether it was merely a display issue or if the whole system was shut down but the computer was still running.  Any thoughts or advice on how to deal with the issue if it happens again would be appreciated.

### Post by 2F4U on 2012-01-05
Did you try to ping the computer? This is usually more reliable than trying some more advanced network sharing stuff?
Did you hear some noise from the machine or where some lights on? If not, the computer may have gone in sleep mode. Since this usually shuts down the peripherals, it is not easy to wake up a desktop computer. On a laptop, you would open the lid, but that doesn't work on a desktop machine. I would check the power management settings and make sure that it won't go into suspend or hibernate.

### Post by alphaamanitin on 2012-01-05
I have had something like this happen before, but I discovered that while the computer apparently didn't come back from sleep, the screen was black and never changed, it actually had.  Typing in my password and hitting enter got me to the desktop and my display was suddenly working again.  For most desktops you wake it from hibernation by hitting the power button.
