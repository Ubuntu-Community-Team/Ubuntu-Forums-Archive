### Post by marin123 on 2010-12-08
yes, but you have to install everything manually (the theme, cursor, awn theme, login screen etc.). if you run installer you can do everything in few steps, and of course there's uninstaller, and we know what that does :D

if you want to install everything manually, uncompress the mac4lin.tar.gz file, open the folder where you unzipped it and then emerald folder. you will find your themes there, just drag and drop them into the window of appearance preferences (themes) and the theme will be installed. you need to install the icons like this, which are in icons folder...

you also have macbuntu, which is similar to mac4lin, maybe you could try that out too and see how it suits you.
