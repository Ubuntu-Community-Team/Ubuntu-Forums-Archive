### Post by waynefoutz on 2011-07-09
If 10.04 works better on that older PC, then continue using that. 10.04 has a longer support life anyway. Sometimes old hardware is better off staying with an older OS. Ubuntu comes out with a new release every six months. If you have current or cutting edge hardware, that's great. But if you're running an older system, find what works and stick with it. There is no need to constantly upgrade to the latest version.

10.04 LTS will actually go end of life, which means Canonical will no longer provide updates, it will still work, the same time 11.10 will, and 11.10 hasn't even been released yet. It still has a LOT of life left in it.
