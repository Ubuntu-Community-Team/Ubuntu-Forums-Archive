### Post by EarloftheWest on 2008-06-03
You can find the answer [here]("http://www.oooninja.com/2008/04/galaxy-icon-set-theme-look.html").

Copy the images file to your desktop. Open a terminal
change directories to the Desktop.
enter this code:
sudo cp filename.zip /usr/lib/openoffice/share/config

For the above to work, I had to open Synaptic and install all of the other openoffice themes. Search for openoffice.org-style and you'll find about 5 of them.

Good luck.
