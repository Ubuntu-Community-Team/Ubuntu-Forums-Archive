### Post by doclivingston on 2006-02-19
[quote=GameGod]I've just posted a Rhythmbox vs. Banshee comparison on my site... Both are cool music players, but you definitely need to try the latest versions...[/quote]

Nice writeup. The newest versions of both are definitely better than what is in Breezy.


[QUOTE=aysiu]but I do know that Rhythmbox isn't able to edit ID3 tags, and Banshee is.[/QUOTE]

Actually, Banshee can't edit tags - it just changes it's database and doesn't change the file. If you try to change one, and then load the file with id3tool, Easytag or whatever, you'll find it hasn't updated.

Rhythmbox's tag-editing isn't great either: it only works for MP3 and FLAC file (not Ogg Vorbis or anything else), and only with GStreamer 0.8 (so the version in Dapper doesn't).


So for what you'll get in Dapper, neither can actually edit tags.
