### Post by DJ_Peng on 2008-06-30
Actually I think the OP is referring to the extra large dropdown below the Smart Location Bar (a.k.a. the supposedly "Awesome Bar"). You can use the oldbar extension to return to the old way of displaying possible matches. There are also some about:config hacks to change how much information is displayed but I don't have links to them offhand. 

As far as the new Bookmark window goes the only way I know of to get rid of it is to use Firefox 2 rather than Firefox 3, but there may have been an extension created that I simply haven't heard about yet.
