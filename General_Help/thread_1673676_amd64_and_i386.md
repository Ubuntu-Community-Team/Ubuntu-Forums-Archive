### Post by Vaphell on 2011-01-22
amd64 = 64bit version of x86
i386 = 32bit version of x86

pretty much any mainstream processor made in last few years is 64bit compatible. 64bit is a bit faster but uses slightly more memory and sometimes can be a pain to set up thanks to few respectable companies half-assing their software (proprietary drivers, flash, skype, ...)

### Post by Cavsfan on 2011-01-23
AMDd64 doesn't mean AMD processor, it simply means 64 bit processor as it applies to Intel, etc. as well. And 64 bit processors are 
the best! You'd be hard pressed to find a PC that wasn't 64 bit. 
Not sure about notebooks or netbooks.
I have been running a 64 bit quad core processor with Ubuntu 64 bit and windows 64 bit for nearly two years without any issues what so ever.

And it is ultra fast compared to my old Dell Pentium 4!

Adobe dropped support for 64 bit flash on Ubuntu, no big deal as the 32 bit works just as well.

Bottom line: if there isn't a 64 bit app, there is a 32 bit app. and there will eventually be more and more 64 bit apps. as all 
the PCs built are 64 bit. And there are no issues with video drivers, etc.
