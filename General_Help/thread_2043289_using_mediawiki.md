### Post by Habitual on 2012-08-16
yes, start apache[2]/http and navigate to [http://localhost](http://localhost)

Check /var/www/something for directory of MW and of course, if /var/www/[COLOR=Red]mycoolwiki[/COLOR]/ is populated, then navigate to http:/localhost/mycoolwiki/

mycoolwiki is merely an example. :)

See [http://newbiedoc.sourceforge.net/tutorials/apt-get-intro/info.html.en](http://newbiedoc.sourceforge.net/tutorials/apt-get-intro/info.html.en)
