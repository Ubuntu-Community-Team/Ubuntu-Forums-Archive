### Post by PmDematagoda on 2007-11-24
The reason Firefox 2.0.0.9 is not there for Gutsy may very usually be because there are no fixes (security or otherwise) concerning Linux, so because of this there may not be an update to Firefox 2.0.0.9 on the repos and which means that Firefox will remain 2.0.0.8 until an update for Firefox  arrives which fixes certain bugs or problems on Linux.

And on GNOME (Ubuntu), Synaptic is not the update manager, there is a seperate one which, incidentally, is named Update Manager:), also you will be notified of any updates for Ubuntu via a notification, so you do not need to worry overmuch about updates in Ubuntu.

### Post by PmDematagoda on 2007-11-24
The same concept applies for KDE, but there are a few differences.

In KDE both updates and package management is handled by an application called Adept, concerning updates, a notification from Adept Notifier appears on the menu bar, when you click on that, you can start the Adept Manager which you can use to perform the updates.
