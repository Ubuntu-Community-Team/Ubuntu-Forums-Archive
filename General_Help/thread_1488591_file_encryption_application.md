### Post by cryptotheslow on 2010-05-20
+1 for TrueCrypt.

Use the basic (default) option to make an encrypted drive in a file. Then mount the file in TrueCrypt and it can be used in Nautilus just like any other drive. Once you are done copying files to it, just unmount it in TrueCrypt and upload the encrypted file wherever you like.

I use this method for my remote backup storage (on humyo.com) - as the free accounts there do not offer https transfer and I do not want my files stored or transferred in the clear.

Nice thing about TrueCrypt is that it is cross-platform so you should be able to grab that encrypted file to a Windoze PC and mount it using the Win version of TrueCrypt to mount it if necessary.

Needless to say - use a non-trivial passphrase whatever method you end up going with.
