### Post by m0o on 2009-12-31
Firefox's behavior will always save in the specified directory or ask you to save it. In order to change that behavior you can use the [Automatic Save Folder]("http://ubuntuforums.org/Automatic%20Save%20Folder") extension, it allows you to save files in specific folders according to it's file type.
