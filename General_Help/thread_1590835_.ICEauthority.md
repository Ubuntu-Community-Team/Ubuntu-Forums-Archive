### Post by drs305 on 2010-10-08
.ICEauthority isn't even showing up. Normally when the file is missing or deleted it is merely recreated on the next boot. Something must be preventing it.

Take a look at the permissions for your $HOME folder and make sure things aren't owned by root.

You can try to manually create the file with:
```
touch $HOME/.ICEauthority
chmod 600 $HOME/.ICEauthority
```

Added: Per your last post, the file may now exist. Check with a file browser to see if it is now in your home folder, and that the permissions show that you can read/write but others cannot.
