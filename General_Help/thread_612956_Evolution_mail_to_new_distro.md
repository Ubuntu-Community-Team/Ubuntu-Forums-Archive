### Post by mvanstone88 on 2007-11-14
Yes Evolution can be used in KDE not just GNOME.

Backing up your Evolution data would probably be easier on a USB drive or something like that but as long as whatever Distro you install can read your windows partition, you should be able to copy the export files back to your linux partition and import into Evolution.
