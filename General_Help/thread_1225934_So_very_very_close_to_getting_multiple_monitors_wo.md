### Post by ask@refcapgroup.com on 2009-10-16
If it is any help. I have Three ATI Radeon HD 3600 series graphic cards running a total of Six Monitors. Each independently displayed. 

All I did was download and installed the Ati Catalyst software (the latest I think it is 9.4).

Then I simply issued the following command.

aticonfig --initial=dual-head --adapter=all

After that I fired up the ati catalyst and could from there enable xinerama as well as the set the resolutions of each screen, re-arrange from one to the other and all was done.

The only thing that doesnt work is compiz and that is to be expected as Compiz and xinerama are incompatabile. 

Hope that will be resolved down the line. .
