### Post by grahammechanical on 2015-01-29
All of the applications in Ubuntu Studio are in the Ubuntu Software Centre. The advantage of Ubuntu Studio over Ubuntu is a desktop user interface that uses less system resources than Unity. So, that allows more system resources for applications.  And Ubuntu Studio comes with sets of applications for special purposes as part of the install process of Ubuntu Studio.

Right now I have Ubuntu Software Centre open in Ubuntu and I am looking at something called "Ubuntu Studio Photography Package - A collection of applications forming a photograph touchup and editing suite." That package may contain the Ubuntu Studio applications that you presently use.
