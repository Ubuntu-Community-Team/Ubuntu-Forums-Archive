### Post by Yashiro on 2008-10-25
You can't install Windows games for normal use on Linux.
It's a totally different system.

You can install a program called Wine that is effectively a Windows emulator. But really it's subpar for proper game performance.

If you want to play this new game, install and play it in Windows.
