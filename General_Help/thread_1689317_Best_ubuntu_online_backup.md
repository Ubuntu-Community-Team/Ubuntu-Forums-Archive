### Post by Megaptera on 2011-02-17
Have you looked at DropBox?
Dropbox ([www.dropbox.com](www.dropbox.com)) offers 2GB free storage to start, multiple paid options also available. Dropbox offers file synching between the PC / Mac / Linux and the cloud. Load the desktop client or upload through the web interface and begin adding folders to the Dropbox folder. Once in the folder, Dropbox synchs the file between the PC and the cloud and any other devices that you have enabled with Dropbox including mobile and tablets. Quick, light and in the background. From here [http://www.techsupportalert.com/content/best-free-online-backup-sites.htm](http://www.techsupportalert.com/content/best-free-online-backup-sites.htm) (Windows freeware but DropBox = multi o/s).

### Post by NFblaze on 2011-02-17
Google has a cloud service to store files. I've been using it for quite sometime. It's rather cheap like 5 bucks a year and I got 20GBs. It goes up as you see.

 Though, I dont have an automated system or anything and it doesnt support FTP, though I am aware that GoogleCL is a suite of commandline tools to allow working with their cloud services using CLI, so I suppose if you are scripting this would be useful.
