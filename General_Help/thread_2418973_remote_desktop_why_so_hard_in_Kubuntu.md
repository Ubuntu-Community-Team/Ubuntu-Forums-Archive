### Post by HermanAB on 2019-05-14
Note that Windows natively supports Anonymous FTP, which is the easiest server to set up and you can use that to transfer files between Windows and Linux.  Windows also supports CIFS/Samba, which can be hard to set up and you can get a NFS client from MS Technet.

So if you have a problem with transferring files using rdesktop or some such, there are various other options.
