### Post by jwbrase on 2011-09-07
> **Alastair Rae said:**
> Sorry but I can't figure that out. All I get when I click on the launcher icon is the same set of apps. What I'm trying to do is get to the old System/Administration/Network/General tab.

For that you have to run gnome-panel in your startup applications (you'll have to drag the top panel somewhere else, as the Unity panel is always at the top and can't be deleted), or else use classic GNOME (GNOME 2) as your desktop environment (If the GDM layout is the same for 11.04 as my 10.04 install, there should be a list labled "Session" at the bottom of the screen once you've selected your username). Note that in later versions (11.10 and beyond) you'll actually have to install GNOME 2 to use it. It comes with 11.04 though.

Frankly, when I upgrade beyond 10.04, I plan on using GNOME 2 and forgetting that Unity even exists.
