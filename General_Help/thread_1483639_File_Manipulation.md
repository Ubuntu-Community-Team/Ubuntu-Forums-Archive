### Post by bredman on 2010-05-14
I recommend that you search the web for tutorials for linux command line.

To get you started, you could also search for these commands

ls - list files
cd - change directory
cp - copy a file
mv - move (rename) a file
rm - remove a file
mkdir - make a directory
rmdir - remove a directory

Be very careful with the commands rm, mv, and rmdir, they can really trash your computer if you don't know what you are doing.
