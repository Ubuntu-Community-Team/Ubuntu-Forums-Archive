### Post by sloggerkhan on 2009-10-20
You probably have a buggy program that goes into an endless loop that keep allocating memory to something. When it starts happening, open up system monitor, find out what process is eating up all your memory and cpu, nuke it, and see if you can find a fix or update or report a bug.
