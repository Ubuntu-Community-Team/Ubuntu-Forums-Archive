### Post by VMC on 2013-09-05
Here's from the FAQ of keypassx:
> [COLOR=#272727][FONT=Verdana]No, KeePassX does not support the KeePass 2.x (.kdbx) password database format currently.[/FONT][/COLOR]
[COLOR=#272727][FONT=Verdana]However, you can create an export in KeePass 1.x database format (.kdb) from KeePass 2.x,[/FONT][/COLOR]
[COLOR=#272727][FONT=Verdana]which KeePassX can read (and use as the native password database).[/FONT][/COLOR][COLOR=#272727]

[FONT=Verdana]That's the way I did it. [/FONT]

[FONT=Verdana]edit: If I'm reading what your wanting to accomplish. keyypass2 > keypassx [/FONT][/COLOR]
