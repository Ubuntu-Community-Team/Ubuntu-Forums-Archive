### Post by vav on 2007-04-05
It would be really great if the restart button had a little drop down list (like the back/forward buttons), that lets you choose which OS to restart into from grub's menu.lst list.

This frees up time during when the computer is restarting (you dont have to wait around for the grub menu), and also prevents accidental restarts into full mode when recovery mode is intended etc.

Has someone done this already? One easy option is a script to change grub's default option, and make the script run from an app-launcher button, but integrating with the restart button would be elegant. Of course once the OS changes there needs to be an easy way to undo the change too...
Maybe make grub restart into the other OS only for once, and on the next restart default back to it's original default (sorry for the confusing language)

What do you guys think? I for one have banged my head too many times for restarting into the wrong OS.
