### Post by abovett on 2007-10-07
> **jovannicabanag said:**
> Please help me with my concerns using Ubuntu:

1. Can I update and install additional programs without using the console, just clicks like in Windows?

Yes - use the "Add/remove" option on the applications menu for common programs, or for more choce and control over exactly what's installed, use the Synaptic package manager (System menu, Administration sub-menu)
> 
2. What webcam (brand/model) which is compatible with Ubuntu?

Not sure - don't use one.
> 
3. In installing Ubuntu, do I need to install the drivers for the video card and sound card?

Only if you have something obscure or want to use proprietary drivers for some reason. Ubuntu includes drivers for the vast majority of common sound and graphics hardware. If in doubt about your hardware, try booting the live CD version which will allow you to test your hardware for compatibility without installing anything.

Andy B
