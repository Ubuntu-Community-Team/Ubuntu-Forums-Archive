### Post by TheFu on 2016-09-30
You can manually create a .desktop file which contains information about the program.  Most programs come with that file as part of the installation. There are probably thousands of examples on every Linux desktop system, so just look around for files ending in** .desktop** for examples. [https://developer.gnome.org/integration-guide/stable/desktop-files.html.en](https://developer.gnome.org/integration-guide/stable/desktop-files.html.en) is the docs for it.
