### Post by 3rdalbum on 2011-10-19

The cost difference between 4gb & 8gb is only about $20 so the real question is whether I'll get any benefit over 4gb. My other question is will two 4gb sticks of ram use be any more of a drain on the battery than two 2gb sticks?

It depends on what you're doing.

Copying multi-gigabyte files, multiple times? You'll see a benefit. Using some very big programs, video editing, 3D animation? You'll see a benefit.

Since you're talking about a laptop, I'm assuming you'll be doing none of these things; if you were, then you'd probably prefer to use a desktop, it'll be faster. So the extra four gigabytes might just be a drain on battery.

It probably won't be a noticeable drain, so if you really want to do it you might as well.

### Post by erind on 2011-10-19
> **garyed said:**
> I'm running Ubuntu 11.10 & planning on upping the ram to my laptop.
The cost difference between 4gb & 8gb is only about $20 so the real question is whether I'll get any benefit over 4gb. My other question is will two 4gb sticks of ram use be any more of a drain on the battery than two 2gb sticks?

For general use 4GB is plenty of RAM, but I'd go for 8GB given the* _small price difference_*. Besides the known advantages of having more RAM, another one is that if you're going to use only 4GB in your laptop, the power consumption of one 4GB stick is generally less than two 2GB sticks used together, (even though small). I use virtual machines a lot, and there have been times when I've maxed out my 4 GB RAM. And probably the power usage of 2x4GB vs 2x2GB might be trivial, as discussed here:

[http://forum.notebookreview.com/lenovo-ibm/572634-4-8gb-ram-6.html](http://forum.notebookreview.com/lenovo-ibm/572634-4-8gb-ram-6.html)
