### Post by TeeAhr1 on 2008-01-08
If you're asking about file formats (mp3 and the like), go into Synaptic (System > Administration > Synaptic) and install ubuntu-restricted-extras.  This will cover pretty much all your bases.  For more information, see [here]("https://help.ubuntu.com/community/RestrictedFormats#head-ade7b6cc5a280ee943fd7884cf7dc49ebe7e22ca").  Automatix (mentioned above) is really unnecessary (ubuntu-restricted-extras covers pretty much everything, like I said) and can hose your system.  If you're just getting started, don't push your luck.

If you're asking about functionality and applications, Ubuntu is already light-years ahead of Windows out of the box.  The default installation should cover all your day-to-day needs quite nicely.  Is there something in particular you feel you're lacking?
