### Post by danwood76 on 2008-05-21
Generally to have an optimised kernel you have to build it yourself.
Installing through synaptic wont break your system and the normal kernel will still be there.

There is no 686/K7 packages anymore, unless you are using an old version of ubuntu.
But I doubt you would see any noticable difference anyway.
