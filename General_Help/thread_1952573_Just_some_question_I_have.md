### Post by solocommand on 2012-04-04
Welcome to the community!

If you are installing Ubuntu in a dual-boot setup, you will not need to back up your files (though as a general rule it's recommended, just in case!). If you plan to use Ubuntu as your only operating system, you will most definitely need to back up your files to an external drive, flash drive, or disc.

If you choose to dual-boot Windows and Ubuntu, you should be able to access your files by selecting the Windows partition from the file manager, and browsing out to your files.

Gaming on Linux is a widely debated topic, but the gist of it is this: Your gaming experience will not be the same as it is in Windows. This is due in part to different levels of driver support from both the manufacturer and the linux community. Before you make the switch, you should research and make sure that your graphics card is supported, and find out what drawbacks to expect (some cards/linux drivers do not support all the options that the main windows driver do, and cause decreased performance in-game.).

The other issue is lack of support from the game developer. For many games there is no linux client at all, and you will be forced to use something like Wine or Cedega/Crossover to run the game under Linux. You can find out more information about what games are supported and how well they run on the WineHQ and Cedega websites.

The level of support varies widely based on the game developer. Minecraft is java-based, and I believe it runs very well under linux, provided you have the correct graphics driver. Some online games such as World of Warcraft have a large following, and work fairly well. Many other games do not.

### Post by papibe on 2012-04-04
Hi jacob70. Welcome to the forums!

> **jacob70 said:**
> will i have to back up what ever files i have then, or are they safe? Like if i don't back them up and i install unbuntu, will they be there?
The installation is a fairly safe process, however, it is always a good practice to backup your personal files before an OS installation.

Considering you want to keep Windows, there are 2 installation options: dual boot with proper install or Wubi. If you decide to do an proper install. I would recommend to start making space for Ubuntu using Windows 7's partition manager.

In the very unlikely case something fail, you would temporally loose the ability to boot into Windows, bit your files would be safe. If you have your Windows recovery disks, it relatively a simple process to get it back.

> **jacob70 said:**
> Secondly, I play a lot of games. Minecraft, Some Online Games, and a Lot of Steam games. Are there any problems in this area? Will Unbuntu support games that, say, a windows 7 OS would?
Recently released Windows games won't play on Linux at all. But if you wait a little (3-6 months) there will be some new tricks to make them work (at least the most popular ones).

Nevertheless, note that it's not the same experience. Usually lower graphics, and some minor features may not work.

If you enjoy PC gaming, you need to keep Windows around.

Hope that helps. Keep reading and researching so you can take the best solution for you.
