### Post by garvinrick4 on 2010-11-18
The only thing I could actually tell you to get your Windows install to boot would be with this fix for XP in this thread below.
[Grub/XP/Vista Bootloader - Ubuntu Forums]("http://ubuntuforums.org/showthread.php?t=1014708") 

That would boot your Windows install to boot and you would have to reinstall grub2 to make them both bootable if all is well with your XP. Did you defrag windows a few times before installing linux?
 Here is code in a live cd to install grub2.
```
sudo mkdir /media/root
``````
sudo mount /dev/sda5 /media/root
``````
sudo grub-install --recheck --root-directory=/media/root /dev/sda
``````
sudo umount /media/root
```
Do make sure you backup your /home before doing anything with drive.
