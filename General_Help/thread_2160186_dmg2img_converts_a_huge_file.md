### Post by tgalati4 on 2013-07-13
A dmg file is a compressed OS X file that contains the folders and directories of a Mac program.  The image file is the uncompressed disk-mountable ISO file--like what you would find on a CD or DVD.  Typically the dmg is uncompressed into an image file (on the hard drive) that is then mounted and accessable on the desktop.
