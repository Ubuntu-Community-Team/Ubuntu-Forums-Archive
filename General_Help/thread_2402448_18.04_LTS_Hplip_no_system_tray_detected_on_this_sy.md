With 18.04, I stopped using Hplip. But when I did use it, I always downloaded from HP directly to get latest version.

18.04 Bionic Beaver

I have this in my old notes:
No system tray error:
open up the "startup applications" editor from the admin menu.
add a new program, for the command put:
sleep 10;/usr/bin/hp-systray
