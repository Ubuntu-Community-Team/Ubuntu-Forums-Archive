### Post by Old_Grey_Wolf on 2010-06-06
Try right click on panel, then select "Add to Panel...". You should get a window with a list of applets you can add to the panel. Select "Clock". If that works then you can delete the non-functioning one by right click on it and select "Remove From Panel".
