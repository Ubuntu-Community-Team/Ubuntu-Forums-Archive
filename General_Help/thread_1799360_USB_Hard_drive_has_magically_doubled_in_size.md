### Post by zero2xiii on 2011-07-07
Could be the controler. Open up the case and remove the drive. Put it in the pc as a normal HDD and see if anything changes. If linux is able to detect and mount it then, you know it's the usb/sata converter, just replace it.. If however it still fails to mount (most common is an error message about a read-only file system or such) then you know it's the drive self, and needs replacing.
