### Post by decrepit on 2011-03-12
I've fallen into this trap myself a few times.
Every time you delete something, a trash box is created, (if it's not there already, ie if you delete it, it'll just create a new one), and whatever it is just gets put into it. 
As you say it's hidden, so there's no indication that you haven't increased spare memory.

Solution is easy, just "empty trash" before you unmount the drive.

You don't have to go looking for it, it's linked to the main trash can.

It doesn't help when ubuntu doesn't have a desktop trash icon.

I've added "garbage bin" to my top panel, trouble with this is it doesn't change when it's empty, so you're not reminded that it's got something in it.
