### Post by Rocket2DMn on 2008-06-09
You can try installing with the alternate install cd, available at [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) by checking the box at the bottom.  This uses a text based installer rather than a live session, but it is still intuitive and easy to use.  This solves a lot of problems associated with LiveCD installations.
