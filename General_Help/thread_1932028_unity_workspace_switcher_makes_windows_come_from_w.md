this is kind of a silly problem, but it's vexing  me.  when switching between workspaces in unity (11.10), my windows go in the wrong directions.  I have four workspaces set up in one horizontal row, and when I go to the workspace to the left of my current one, my windows in the current one move left, while the windows in the new workspace come from the right.  I can't seem to find the proper setting in compiz to change this, but I'm guessing the solution is ridiculously simple.

### Post by 3Miro on 2012-02-26
Are you using keyboard or mouse. The setting is in CCSM, for mouse use Viewport Switcher and for keyboard check the Desktop Wall -> Binding -> Move within Wall.

You need to change the left/right direction.
