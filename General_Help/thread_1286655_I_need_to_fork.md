### Post by 3rdalbum on 2009-10-09
The thing about "you need to fork" is some advice to the developers of whatever package you were updating. I haven't used actual Kubuntu so I haven't come across this error message before, but it sounds like the update stalled and the PackageKit system thought "Hey, this package is taking a long time to install, there might be something wrong".
