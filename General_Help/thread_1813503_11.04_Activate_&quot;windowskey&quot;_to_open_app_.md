### Post by Toz on 2011-07-27
Goto Settings Manager -> Keyboard -> Application Shortcuts. Here you can define whatever you want the windows key to do. To popup the applications menu, do the following:

1. Click on the add button

2. For the Command, enter **xfce4-popup-applicationsmenu**

3. Click OK

4. When prompted for the key combination, press the Windows Key.

Now the windows key will popup the applications menu.
