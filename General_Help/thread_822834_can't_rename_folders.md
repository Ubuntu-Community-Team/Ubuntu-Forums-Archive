### Post by mike2357 on 2008-06-08
Assuming you have permission, this should work from the command line:
mv "Untitled 1" New_Name

If you are using Nautilus, you should be able to right-click on the folder and select Rename.  Again, you have to have permission.

If the above doesn't work, try from the command line and post the error message you are getting.
