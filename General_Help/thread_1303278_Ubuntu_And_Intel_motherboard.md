### Post by Bucky Ball on 2009-10-28
Are you buying one to build a machine or you have one in a machine already?

If you are running that motherboard already, download Ubuntu ISO, burn a disk and try it. When you boot from the disk there is an option to 'Try Ubuntu without making any changes to your system'. 

There are extremely rare instances of Ubuntu not working with a particular board (VERY unusual) and Intel is very common so you should have no problems. I would be extremely surprised if you did.

Motherboards are generally not an issue. Other hardware can be. Another good reason to try Ubuntu from the installer disk to start with.
