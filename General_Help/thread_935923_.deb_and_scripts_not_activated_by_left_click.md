### Post by drs305 on 2008-10-02
If you aren't installing deb files via synaptic, they are usually installed via command line with the "dpkg -i packagename.deb" command or with an app called "gdebi". 

If gdebi isn't installed, install it and you should get an option to "Open with Gdebi Package Manager" by right clicking on the deb file in nautilus or by double clicking the deb file.
