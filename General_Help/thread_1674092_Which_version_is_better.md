### Post by akand074 on 2011-01-23
> **ajcurran94 said:**
> I switched from Windows to Ubuntu last night, so I guess you could say that I haven't had much time to look around.  I was just wondering if anyone could explain to me what the Terminal is used for.

I greatly appreciate the help.

Terminal is the command line of Ubuntu. Basically it's the brain of the OS. You throw it a command, and it does what you tell it. It supports a large number of commands, and you can also run applications from there as well. Also there are some applications that only run through the command line. But basically, everything you do is just commands going through the Terminal, you just don't see it because you have what they call a GUI (Graphic User Interface) front end. [Here]("https://help.ubuntu.com/community/UsingTheTerminal") is an overview of it.
