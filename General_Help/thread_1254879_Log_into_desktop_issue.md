### Post by mantis8 on 2009-08-31
I just got 8.10 installed, but when I upgraded to 9.04, I have to type in my user name & password to be able to use it. There is no need for this in my situation, so how can I have it log in automatically at every boot up?  I understand I don't need/want to log in as admin every time; just as user.

---

### Post by scouser73 on 2009-08-31
Hi mantis8, to automatically log in go to **System > Administration > Login Window**, enter your password when prompted, then click the **Security tab** & make sure that **Enable Automatic Login** is checked & in the drop down menu make sure that your username is showing then press **Close**
