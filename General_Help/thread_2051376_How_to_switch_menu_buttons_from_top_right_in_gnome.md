### Post by JRV on 2012-09-01
Install ubuntu-tweak with the following commands.

```

sudo add-apt-repository ppa:tualatrix/ppa
sudo apt-get update
sudo apt-get install ubuntu-tweak

```

Run ubuntu-tweak, select Tweaks=>Window.
