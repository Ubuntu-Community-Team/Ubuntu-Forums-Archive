### Post by MG&amp;TL on 2011-10-01
[https://wiki.ubuntu.com/PackagingGuide]("https://wiki.ubuntu.com/PackagingGuide") Link to get you started with packaging

[https://launchpad.net/]("https://launchpad.net/") Open-source project hosting.

[http://mrbook.org/tutorials/make/]("http://mrbook.org/tutorials/make/") Makefiles.

[http://linuxcommand.org/]("http://linuxcommand.org/") BASH scripting.

[http://www.debian.org/doc/manuals/maint-guide/]("http://www.debian.org/doc/manuals/maint-guide/") Bit of a monster, but sometimes worth reading if you're confused about syntax for example. Best read in small bits, after the others.

If you can code and package, you are ready to contribute to Ubuntu
