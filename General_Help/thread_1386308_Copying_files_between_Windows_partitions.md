### Post by mk1w86 on 2010-01-20
> There we go, that's the kinda thing I needed to know  Are Windows files even hidden in Linux then?


No, hidden files in Windows are different to Linux.  On Linux you add a period before the file or folder name to make it hidden.  For example if you mount a Windows partition on Linux you should see all the files that were hidden in Windows and if you view an NTFS or FAT32 partition on Windows any files that were hidden in Linux will be displayed. ;)
