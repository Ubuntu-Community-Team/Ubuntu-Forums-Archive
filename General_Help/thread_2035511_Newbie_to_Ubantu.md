### Post by Sandertje on 2012-07-31
Linux can be seen as a "family" of operating systems, who all use some version of the Linux kernel. The kernel is the core of the operating system. 

Ubuntu is just one of many operating systems that use the linux kernel. So: Ubuntu is by a Linux system. But "Linux" does not necessarily mean Ubuntu (just like an apple is a fruit, but "fruit" does not have to be an apple).

### Post by OldHarbottle on 2012-08-02
I had a similar problem with my old Dell Optiplex. I would suggest you try to boot from the live DVD first, just to check that your hardware is fine.

In my case it was a dodgy BIOS, it supported USB boot about 1 try in 10. You could try updating your BIOS (that worked for me) but only if you have confirmed that its not the USB stick or any other hardware issue, i.e. video cards, sounds cards etc.

### Post by JosephBeck on 2012-08-02
To explain it more simply, Ubuntu is to Linux what OSX Lion is to Mac in a way. Ubuntu is a version of Linux operating systems, the most common at the moment I believe. There is no "Linux" OS, but OS based on Linux properties like Debian, Ubuntu Variations, OpenSUSE, etc. Those new to Linux are often recommended to start with Ubuntu variants. 



If you are sure you mounted the file correctly to the USB do the following:

1)Turn of Computer

2)Insert USB

3)Turn on USB

4)Press The Key For Your Boot Menu When Your Computer Starts (The Screen Will Show Your Laptop/Desktop Brand And Have The Whole "Hit ESC For ____" screen. Mines F10 I believe)

5) You'll get An Image Like This, Arrow Down And Hit Enter On Your USB:
[http://www.howtogeek.com/wp-content/uploads/2010/05/sshot22.png](http://www.howtogeek.com/wp-content/uploads/2010/05/sshot22.png)

6) It should load it up, and from the Ubuntu LIVE menu just hit install. If you want to keep Windows 7, choose the option to install along side it.



Last time I checked there was no boot from USB on the Ubuntu menu, getting to the Ubuntu menu means you already are booting from the USB.
