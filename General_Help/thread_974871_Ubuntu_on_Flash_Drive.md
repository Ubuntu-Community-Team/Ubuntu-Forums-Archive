### Post by C.S.Cameron on 2008-11-08
If you are running 8.10 there is a tool in Administration to Create a USB startup disk.
If you are running windows, download Unetbootin it will install the iso to USB, also works in 8.04.
If you are running 8.04 you can install usb-creator, which will also install the iso to bootable flashdrive.
