### Post by xerman on 2008-11-06
> **jang said:**
> Any soft keyboard displaying the corresponding characters for every english key?  Is there a table I can print and attach to laptop?

You do not need to know characters, at least in chinese/japanese, or russian, as far as i know.

For Chinese use pinyin so the keyboard layout matches the pinyin, so you type "wo shi" and will come out "&#25105;&#26159;". Japanese seems to be the same. 

Russian I just use Yawerty keyboard which matches phonetic so to type "ya goboriu po ruski" come out "&#1103; &#1075;&#1086;&#1074;&#1086;&#1088;&#1102; &#1087;&#1086;-&#1088;&#1091;&#1089;&#1089;&#1082;&#1080;&#1081;". For russian you need to find some special letters who do not match latin characters, but it is easy to handle in Yawerty, unless you are a &#1056;&#1091;&#1089;&#1089;&#1082;&#1080;&#1081;, and you feel more comfortable with another keyboard layout.
