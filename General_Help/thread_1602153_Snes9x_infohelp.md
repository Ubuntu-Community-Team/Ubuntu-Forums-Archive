### Post by Darth Penguin on 2010-10-21
The reason that ZSNES is better than SNES9X is that SNES9X is written in C++ and ZSNES is written in Assembly. Which means ZSNES runs as if it were native to the OS running it. ZSNES for Linux does not work on AMD64, unfortunately.
