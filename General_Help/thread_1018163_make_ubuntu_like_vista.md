### Post by .arean on 2008-12-24
Starting from a fresh download of the aero-clone theme from gnome-look:

Extract the contents of 57352-aero.tar.gz, then extract aero-clone.tar.gz

Copy the aero-clone folder to your .themes folder (press ctrl+h in your home folder to show hidden files). The aero-clone folder is the GTK theme.

To install the beryl (emerald) theme you'll need to have emerald installed. Open synaptic and search for and install emerald. Then go to System > Preferences > Emerald Theme Manager and import the emerald theme.

To install the orb icon open a terminal window and change directory to the aero directory (cd /where/aero/is/located) and run the shell script with sh place-the-icon.sh and follow the instructions.

After that just follow the rest of the directions on blog.
