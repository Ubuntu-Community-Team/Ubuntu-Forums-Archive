### Post by talraja on 2010-01-01
I was trying to remove some bookmarks in the places menu myself and figured it out so I thought I would share the information.

In the users home directory there is a hidden file .gtk-bookmarks that you can edit with your favorite editor to remove any unnecessary entries.
