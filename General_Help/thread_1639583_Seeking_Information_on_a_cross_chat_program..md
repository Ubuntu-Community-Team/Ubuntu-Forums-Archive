### Post by tgalati4 on 2010-12-06
It's not open source, but skype works pretty well.  You can have multiple skype chats going between windows and linux.  For instance you and a buddy both have a linux machine and a windows machine and you are 3000 miles apart.  You have skype installed on all 4 machines.  You can open up chat windows and see the chat session on all 4 machines at once.  This is helpful when troubleshooting and you need to send files from Windows to linux or the reverse.
