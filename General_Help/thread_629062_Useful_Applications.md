### Post by prodigalson666 on 2007-12-02
compiz fusion is already installed. Upgrade your system first,  install the restricted drivers for your video card. Go to synaptic and install compiz fusion setting manager. When thats done go to system-preferences-appearance. Hit the visual effects tab, choose custom and start having fun.
In terminal you can also type ccsm.
