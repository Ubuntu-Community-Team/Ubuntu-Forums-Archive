### Post by schizomasochizt on 2009-07-14
Hi guys. I admit I'm not that good in using Ubuntu but this time I think I have proven mysef as one. Hehe.

I booted up my desktop one day just to find that my system is corrupted. It gave me error messages stating that my system has unattached inode and I have to run Fsck manually. It also stated that the file system with error (i.e. /dev/sda1) is already unmounted. As far as I know Fsck must be run on an unmounted file system so I just typed in Fsck /dev/sda1 and waited for around 2 minutes. After that it prompted me to choose if I will fix some errors then reboot the system. Now Its working again. Thanks guys.

Ubuntu 9.04 (Jaunty Jackalope)
