### Post by Joshua on 2007-12-19
I guess I could also mention that if you want some really detailed info about running a web server (aside from Googling for web server tutorials) the [Apache]("http://www.apache.org/") website has some really good documentation about configuring Apache to do some more complicated things than just host a single web page.  Also, I'm not sure how much you already know about this kind of thing, but if this is a home setup and you have a router, you will probably have to configure the router to allow access from other networks.
