No you need the Alternate CD, not the Live CD. From the next version you will be able to use the live CD. As for the download you can download the Alternate-Desktop-ISO using bittorrent or any download manager. It's a 700MB file, and most servers are very fast, so if you have an ADSL connection it should not  be that bad. The downloader built-in within Wubi is not very smart when it comes to choosing a mirror, so sometimes it is fast, sometimes it is very slow, hence we encourage people to download separately (using the download manager of their choice) and place the ISO in the same folder that contains the wubi executable.
