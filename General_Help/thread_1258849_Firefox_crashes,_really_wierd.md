### Post by lovinglinux on 2009-09-05
> **munchi3s said:**
> ...Then later on you can go back and be just fine. It does not make any sense to me but if anyone knows whats going on I would sure appericate any help.

That's probably an add with bad javascript screwing the page. It doesn't happen all the time, because the adds are rotated.

Install [NoScript](https://addons.mozilla.org/en-US/firefox/addon/722) + [AdBlock Plus](https://addons.mozilla.org/en-US/firefox/addon/1865) extensions and you should be fine.

Also check the [**[COLOR="DarkRed"]Firefox optimization and troubleshooting thread[/COLOR]**](http://ubuntuforums.org/showthread.php?t=1193567)

BTW, Opera sucks :)
