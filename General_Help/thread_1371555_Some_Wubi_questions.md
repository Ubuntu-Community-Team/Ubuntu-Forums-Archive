### Post by sdowney717 on 2010-01-03
you really should consider just using the regular ubuntu installer to shrink the win7 partition. running wubi is fine but likely to be slightly slower than a full ubuntu install. then you get the benefits of using ext4 file system instead of ntfs. I think of wubi like an install of a program to try out ubuntu in windows. if you do a regular install read up on this first before doing it as you can wipe out your windows if you goof it up.
