### Post by SeijiSensei on 2013-12-16
I suspect you mean you want your university to become an Iranian "mirror" site for Ubuntu.  Ubuntu Forums cannot help with this.  You need to follow the procedures for registering a mirror at [https://wiki.ubuntu.com/Mirrors](https://wiki.ubuntu.com/Mirrors).

However there is already an Iranian mirror at [https://launchpad.net/ubuntu/+mirror/yazd-university](https://launchpad.net/ubuntu/+mirror/yazd-university).  You're right though that it doesn't appear in the list of repositories when I use Synaptic.  It looks to me like they only archive the CD images and do not provide a repository for updates, which is why it doesn't appear in the repo list.  If you visit [http://mirrors.us.kernel.org/ubuntu/](http://mirrors.us.kernel.org/ubuntu/) you'll see an entirely different directory tree.  The URL [http://mirrors.us.kernel.org/ubuntu/dists/](http://mirrors.us.kernel.org/ubuntu/dists/) shows all the repositories.  There is no equivalent set of files on the Yazd University site.
