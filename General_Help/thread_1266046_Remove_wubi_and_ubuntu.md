### Post by lisati on 2009-09-14
First, make sure that Ubuntu isn't showing up in the Windows control panel "Add/remove programs" menu - if it is, uninstall Ubuntu like any other piece of software.

If after uninstalling Ubuntu it still shows up, you might need to edit the Windows "boot.ini" file. I can't remember exactly where to find it at the moment (it can depend on the version of Windows) and care is needed to make sure you delete the correct entry.
