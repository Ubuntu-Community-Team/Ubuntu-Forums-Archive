### Post by howefield on 2010-02-20
> **VTPoet said:**
> Yeah... and that's the weird thing. I don't get that dialog box.

I thought there would be more to it, seemed too simple :)

Try opening File Manager and open the Edit > Preferences menu and from the Media tab, make your selection in the Media Handling > DVD Video section.

If it is Thunar you are using, you might need the thunar-volman plugin.

> The Thunar Volume Manager is an extension for the Thunar file manager, which
enables automatic management of removable drives and media.
