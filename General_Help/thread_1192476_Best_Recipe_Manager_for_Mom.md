### Post by ugm6hr on 2009-06-20
OO Base is a bit confusing if you are not familiar with database frontends.

It is easy to add recipes, but searching through them requires the use of "Form-based Filters" which are not intuitive.

This might help: [http://welcometoubuntu.blogspot.com/2008/02/howto-install-gourmet-recipe-software.html](http://welcometoubuntu.blogspot.com/2008/02/howto-install-gourmet-recipe-software.html)

Note it is in the repos for 9.04, so consider installing 9.04NBR.
