### Post by Wtower on 2011-02-11
Welcome. With all respect, I believe you should devote some time to read some guide for installing Ubuntu before doing something that you are not aware of. A helpful guide is:

[https://help.ubuntu.com/community/DualBoot/Windows](https://help.ubuntu.com/community/DualBoot/Windows)

Remove the CD from the drive and boot (basically turn on the computer). Tell me what happens. 

Then try to boot from the liveCD. Get into the BIOS screen (for most computers you can do that by pressing DEL key during the very first screens you look at when you turn on the computer). Somewhere there there is a boot sequence option that tells the computer in which order to llok for a OS from a choice of all hard and removable media. Make the CD a primary boot option and then the hard disk. Then insert the LiveCD, save all changes and let it reboot. See if the OS can indeed boot from the CD and let us know.
