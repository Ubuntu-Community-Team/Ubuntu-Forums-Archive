### Post by jonian_g on 2008-09-29
Compiz is installed by default on ubuntu 8.04. The only thing you need to install is "Advanced Desktop Effects Settings (compiz-config settings manager)". Then go to System>Settings>Advanced Desktop Effects Settings and enable the cube. Then ctrl+alt+left or ctrl+alt+right.
