### Post by codeslicer on 2008-11-07
Well first you're going to need a theme file, which is a .emerald file for the Emerald Window Manager. Next, you should go to the main menu at the top of your screen and go to System->Preferences->Emerald. From there there should be an "Import" button, or you can drag that .emerald file into the list of themes.

Good luck!
