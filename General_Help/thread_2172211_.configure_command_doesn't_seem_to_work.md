### Post by CatKiller on 2013-09-04
Is there an executable script in that directory called configure?

If there is, then the error message could be referring to something that the script needs to function; there's probably a list of prerequisites somewhere for that software. * build-essential* and dev versions of particular libraries are fairly common necessities when compiling software.

If there isn't, that's probably the problem.
