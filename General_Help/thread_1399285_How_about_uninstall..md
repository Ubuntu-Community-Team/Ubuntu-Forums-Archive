### Post by agnes on 2010-02-05
Why delete it? I assume you want to delete it to put another OS on your PC afterwards.
Simplest solution, therefore, is to plainly install the "new"  OS over the entire disk.
(Assuming you had (a) linux distro(s) on you entire disk.)
E.g. if you install Windows there will be an option to use the entire disk, then use that option. It will erase&replace everything on the disk so also Linux.
Before doing this backup files you want to keep on USB or whatever.
