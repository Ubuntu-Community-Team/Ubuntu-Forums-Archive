### Post by OutOfReach on 2008-09-13
Those are names of the Ubuntu releases Hardy Heron is 8.04 (current version), Gutsy Gibbon was 7.10, and so on... They're just release names
You can find out which one you have by going to System>Administration>System Monitor in the System tab.

### Post by northern lights on 2008-09-13

### Post by Elfy on 2008-09-13
They are the names given to different versions of ubuntu - [http://en.wikipedia.org/wiki/History_of_Ubuntu_releases](http://en.wikipedia.org/wiki/History_of_Ubuntu_releases)

You can find out which you have with ```
lsb_release -a
``` from a terminal, or System > About UBuntu
