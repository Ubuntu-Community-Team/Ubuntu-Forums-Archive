### Post by grahammechanical on 2020-07-03
I think that you need to open the Settings utility. Not knowing what version of Ubuntu you are using my directions may not match your system. In Settings>Language and Region you should see a list of Input sources and a button or icon to click to add input sources. You should get a dialog called Add an Input source. You should see  English (United States). Click on it and a large list of keyboard layouts will appear. Select the ones you want to add.
