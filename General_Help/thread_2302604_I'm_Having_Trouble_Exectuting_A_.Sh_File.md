### Post by yancek on 2015-11-11
So you have downloaded the JD2Setup file to your Downloads directory and cannot open it by clicking it, is that correct?  Open it in a terminal by changing to the Downloads directory or wherever you downloaded it and run it; ./JD2Setup_x86.sh  should do the job.  Change it if you have the 64bit.  I don't know why you don't get the option so if you insist on trying to do it that way, good luck.
