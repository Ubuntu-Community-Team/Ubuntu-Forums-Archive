### Post by oldfred on 2013-08-14
One of the install choices is to totally reformat drive. And drive is the Linux definition of physical device not Windows definition of "drive" which really is a partition. Some users think they will keep their d: "drive" when installing and lose data.

If you have any thought at all of wanting Windows best to make a full backup. Some users overwrite entire drive including recovery partition, then later say they want Windows for that one app or game that does not work in Linux. Once erased there is no going back.

       Backup windows before install - post by Mark Phelps
[http://ubuntuforums.org/showthread.php?t=2137439&p=12611710#post12611710](http://ubuntuforums.org/showthread.php?t=2137439&p=12611710#post12611710)
