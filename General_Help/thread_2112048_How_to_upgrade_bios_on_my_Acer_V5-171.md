### Post by Mark Phelps on 2013-02-03
The file from Acer is almost certainly a self-extracting archive.

If you run this on a Windows PC, it will likely expand into a folder containing a bunch of files, including the BIOS and (maybe) a BIOS updating utility.

There may be  a way to update the BIOS using these files (on a CD or USB stick) if there is a way to boot into a BIOS-updating utility.

You should check your PC manual as they often have this information.

HOWEVER -- word of warning (from experience): UNLESS you also have a way to (1) save off the current BIOS and (2) restore that saved BIOS if anything goes wrong, I would advise strongly AGAINST flashing the BIOS as any error would result in your PC being turned into an "electric brick"!
