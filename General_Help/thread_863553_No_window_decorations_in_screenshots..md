### Post by danhm on 2008-07-18
Whenever I use alt+print screen to take a screen shot of the current active window, the window decorations are not included. I use Compiz as my window manager and GTK and my window decorator. I've included as an example. Is it possible to use Compiz+GTK and get window decorations to show up in screenshots without having to manual crop them?

---

### Post by Vadi on 2008-07-18
Enable the Screenshot plugin in Compiz, and then you can use Windows key + left-click drag to make a screenshot.
