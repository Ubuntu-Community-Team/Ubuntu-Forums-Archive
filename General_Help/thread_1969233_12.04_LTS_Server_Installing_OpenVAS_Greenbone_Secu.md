### Post by thewanderer on 2012-07-12
Since you have noticed the missing packages in the 12.04 directory for OpenVAS 5, I will assume you are talking about OpenVAS 5.

Many people have been having problems with this version as 12.04 is using a newer version of libgnutls26 that seems to break the communication between different OpenVAS components.

You can use the 11.10 packages, but you will need to downgrade libgnutls26 as well.

I put together a guide for [installing openvas 5 on Ubuntu 12.04]("http://hackertarget.com/install-openvas-5-in-ubuntu-12-04/").

Warning - **Not sure if downgrading will break other things.**
