### Post by snowpine on 2010-04-22
64mb is not enough for a satisfying everday computer, in my experience. Add more ram if at all possible. :)

Good distros for older hardware include: Puppy, SliTaz, TinyCore, AntiX. But 64mb is borderline. If you have any hardware budget at all, recycle/donate it and buy something a bit newer. Depending on where you live of course... here in New York, you can get a used Pentium 4 with 512mb+ ram for under $100 on Craigslist; that should run Ubuntu just fine.

### Post by The Real Dave on 2010-04-22
64Mb can be enough for a desktop environment, you just need to keep it very very tidy. Check out [KMandla's blog]("http://kmandla.wordpress.com/") for proof ;) 128Mb will be plenty if you do a minimum install, add Openbox, and use Dillo for web browsing. Replace heavy GUI apps with terminal ones (rTorrent instead of Transmission for example) and your good to go :) A base install of Ubuntu 9.10 boots at about 20Mb, Openbox and Xorg, could be anoth 10 or so, so ya 64 can be enough, if your careful.

128Mb and you flying though :)
