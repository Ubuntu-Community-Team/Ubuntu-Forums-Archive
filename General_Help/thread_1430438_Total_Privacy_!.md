### Post by noway2 on 2010-03-15
Try this: [https://help.ubuntu.com/community/OpenVPN](https://help.ubuntu.com/community/OpenVPN)
If you decide to go with OpenVPN, you should also search for a utility called gopen, which will allow you to add a little button to your launcher bar to connect to the VPN.

I personally use OpenVPN and have been very happy with it.  You can find out more about it by searching in the server forum.
