### Post by bobnutfield on 2009-05-12
As I remember Zone Alarm, it is simply a sophisticated firewall setup.  I have only used the free version the past.  But, as a firewall, it is only needed if you are connected to the net as firewalls are used for filtering what traffic gets to your machine.

So, if you are NOT connected to the net with Windows, a firewall is not necessary.  I would still keep a virus scanner installed though for whatever you might install from other sources (CD's, USB's, etc.  Keep in mind, though, if windows is connected to a home network with other computers that are connected, you might still be vulnerable.
