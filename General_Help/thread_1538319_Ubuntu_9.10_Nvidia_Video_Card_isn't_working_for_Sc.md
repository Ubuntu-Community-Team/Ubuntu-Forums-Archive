### Post by 4Orbs on 2010-07-28
Go to the main menu System>> Administration>> Hardware Drivers and make sure the recommended proprietary driver is activated. It seems yours is already activated, judging from your xorg.conf file.

Open the nvidia settings manager as root user by entering in a terminal:
```
gksudo nvidia-settings
```
Change the resolution to the desired, click apply. Then click the button at the bottom of the settings mgr to "Save Configuration" to /etc/X11/xorg.conf and select the option to merge with the existing file.

Reboot and your resolution settings should now stick.

If the nvidia settings manager won't open because "You Don't Appear to Have a Driver Installed" (or something similar) run this command in the terminal:```
sudo nvidia-xconfig
```
Then you should be able to open the settings mgr as root by using the first command.
