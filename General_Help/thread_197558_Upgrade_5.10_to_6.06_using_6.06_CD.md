### Post by bluevoodoo1 on 2006-06-15
Yes it should be possible. 

In Synaptic, if you go settings > repositories > add cdrom you should be able to add it there. Then select the cdrom and deselect the breezy repositories. Hit reload, then mark all upgrades. 

The packages won't be as up-to-date as they would be if you had downloaded them all, but it will be a very good start for Dapper.
