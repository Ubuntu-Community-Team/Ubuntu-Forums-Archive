### Post by RikoW on 2006-03-31
It's very easy to upgrade to the latest version of openoffice. I did this already a few time by downloading the rpm archive from the OOo web page, convert rpm to deb and install.

See my post here for example:
[http://ubuntuforums.org/showthread.php?t=106271]("http://ubuntuforums.org/showthread.php?t=106271")

If you have the 1.9.x installed, you might want to remove it first via synaptic for example, but maybe that's not necessary. After that, whenever there is a new version you want to install, just do again what is described above.

good luck, :)

           Riko
