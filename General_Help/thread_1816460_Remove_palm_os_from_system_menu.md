### Post by 3rdalbum on 2011-08-02
It depends how you remove it.

You should just be able to remove it from the menu itself by right-clicking on the Applications menu and choosing "Edit Menus". A menu editor will appear and allow you to change the contents of your Applications and System menus.

If you're talking about actually removing it from your computer entirely, the package is called gnome-pilot. Removing it won't affect stability, because without a PalmOS device it's never actually running. I doubt it will affect the updates process either, in fact Ubuntu 11.04 doesn't even come with it.

Removing a menu item isn't what I'd call "fun", but it's your system and in this case the modification isn't extreme enough to cause any concern. Have fun.
