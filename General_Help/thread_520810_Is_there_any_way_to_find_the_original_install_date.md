### Post by apetresc on 2007-08-08
I don't know of any official way :( But if you go into /sbin and do an ls -al, those files shouldn't really be touched once they've been installed, so whatever the dates on all those files are (it should be the same),  that's *probably* the date you installed your system, unless you did some weird tweaking.

Not foolproof!
