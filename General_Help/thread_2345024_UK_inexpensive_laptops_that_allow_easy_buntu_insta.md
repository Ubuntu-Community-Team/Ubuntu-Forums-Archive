### Post by sudodus on 2016-11-30
0. I know that some new HPs do not play well with Ubuntu. Do you know if your HP 15 can boot in BIOS mode? If that is the case, you can unplug the internal drive with Windows and replace it with a fresh drive, and install Ubuntu in BIOS mode. If for some reason you need Windows once or twice per year, you can simply swap the drives and run Windows again.

1. I have good experiences of professional class HP computers, that can be bought as 'refurbished' or 'second hand'. I don't know in your country, but in Sweden it is possible to buy for example HP 8470p or HP 8560p Elitebooks and some HP Probooks refurbished for a reasonable price. Ubuntu works well in those computers.

The company (that refurbished and sold those computers to me and family) installs Windows 10 in BIOS mode, which will make it easy to install Ubuntu, but they sell the computers without Windows too (there is an extra fee for getting Windows installed). I would think that there are similar companies in your country, and I think a refurbished professional class computer gives you 'more computer for the same money' compared to a brand new consumer class computer.
