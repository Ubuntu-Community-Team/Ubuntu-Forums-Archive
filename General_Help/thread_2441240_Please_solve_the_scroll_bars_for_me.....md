### Post by guiverc on 2020-04-21
You haven't mentioned your release (your tag line mentioned 16.04, so do you mean `nautilus` under Unity 7 (default desktop for 16.04)?  You didn't specify, but I tried on my much later release.

I loaded `nautilus` (or Files 3.36.1 for me) on my system, and whilst I can scroll in multiple ways on my generic *dell* mouse, the 'maximum' effort scroll-wheel does a full screen in `nautilus`.  If I type "f" I also get to view all files in my current directory that start with "F".  Yes the "list" view I prefer changed to "thumbnail', but it was easier to view the reduced "f" list that way even if not my preference.

If looking for a specific file with "blah" in it's name, I only need to type "blah" anyway and any files with that it in it's name appears.

 When I entered "f" following your example, I noted files which had "filter" in their name in my shortened list, so I added "ilt" (so the search was now "filt") an my list became very small. I really don't recognize many of those files, but it worked for me pretty easily.

`nautilus` isn't my favorite file manager anyway, so if unhappy you could try another.
