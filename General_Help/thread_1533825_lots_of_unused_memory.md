### Post by AlphaLexman on 2010-07-18
On any system, memory is something you want a lot of, on a *nix sytem, memory and swap are something that makes the system run faster, therefore the more memory and swap that are used, the faster your system will run.
(This is just a generalization, assuming you run the same applications all the time.)
Large amounts of free memory will NOT make your system faster, to the contrary, reading data from memory vs. your hard drive is faster from RAM.
