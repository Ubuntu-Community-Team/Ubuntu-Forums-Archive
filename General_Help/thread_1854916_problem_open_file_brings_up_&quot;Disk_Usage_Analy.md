### Post by Frogs Hair on 2011-10-05
You can change the default application for opening a folder with a right click . Select open with another application , choose file browser from the menu , select remember this application .

I use the Pastie PPA in 11.04 for my clipboard , but I don't if it will be updated for 11.10 until the final release . This may be the case for other PPA'S as well . Parcellite has a PPA also but I see nothing for 11.10 yet .
