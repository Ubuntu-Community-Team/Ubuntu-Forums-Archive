### Post by conjur3r on 2007-06-11
Pretty messy instructions hey.

Download the second file as RPMs are not designed for Ubuntu.  There are tools to convert it but let's not go there... The seconf file looks like a compressed archive with all the files it needs to start the game.  Lucky for you, it doesn't look like there is any compiling necessary!

Ok, so download the file, double click on it then uncompress to somewhere in your home directory (I like to use a /home/username/games) so that you do not need to fool around with permissions.  Then navigate to the folder and double click on AAStart.  Sometimes it is better to try running it from the terminal so that you can see any error messages it may spit out.
