### Post by SoFl W on 2010-09-01
When creating a message put long text like yours above inside the "#" tags.

To answer the question, scanning the "/sys" directory will produce those errors.

Try 
```
clamscan -r --bell --exclude-dir=/sys  --exclude-dir=/dev --exclude-dir=/proc --quiet /DIRECTORY/YOU/WISH/TO/SCAN 
```
