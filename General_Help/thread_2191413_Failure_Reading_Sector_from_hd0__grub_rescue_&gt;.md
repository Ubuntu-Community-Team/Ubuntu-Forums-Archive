### Post by sudodus on 2013-12-02
Boot the computer from another drive, for example the boot-repair live disk or the lubuntu live CD/DVD/USB drive, and try to mount the internal drive!

If that works, you can copy the valuable files (personal data) to an external drive, and start again from the beginning. It is probably easier than to start trouble-shooting.
