### Post by c2tarun on 2013-03-18
> **Tim036 said:**
> 
That means I can't install any applications !

Loosing Software center doesn't mean that you cannot install any applications. There is a command line way to install an app or package.

```
sudo apt-get install package-name
```

If you prefer GUI approach then install synaptic by above method and then you can install applications. Also as mentioned in one thread above, rebooting should fix your problem.
