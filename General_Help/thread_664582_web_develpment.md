### Post by djrobthaman on 2008-01-11
I'm not sure about one unified suite, but I use bluefish to code web sites (it's good but not great... the highlighting doesn't always work and there are a few other gripes I have with it, but overall it's a pretty okay program).

And to view your sites in development, you can install a server on your pc.  Just open up synaptic and install apache, mysql (if you want a database) and php5 (if you want to be able to test php).  You can also install phpmyadmin (for easy databse editing... to use it, once it's installed, just point your browser to [http://localhost/phpmyadmin/](http://localhost/phpmyadmin/)).

Once you have those installed you just save your site in the root folder of your server (/var/www/ by default, but that can be changed) and point whichever browser you want to test to [http://localhost/](http://localhost/)
