### Post by jgoguen on 2008-12-01
You have to download and install Canadian English dictionaries for spell check to work.  Go to File -> Wizards -> Install new dictionaries... A new window appears, click English, click the button, then follow the wizard from there.  Restart OpenOffice and Canadian English spell check should work fine.
