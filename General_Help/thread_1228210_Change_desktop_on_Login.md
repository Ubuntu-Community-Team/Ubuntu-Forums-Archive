### Post by ajgreeny on 2009-07-31
I think that if you have different usernames, each user can use a different DE.  I'm not certain about this, however, personally only using gnome.

However, if it is the cosmetic things like the two taskbars or panels that worry her, you can certainly set up gnome to look much more like windows for her login, with a simpler gnome main menu, or even Linux Mint menu, and a single panel, with all the things normally found on the two gnome panels.  In fact that is the way I have my system set up.  It works for me, perhaps for her too.  See screenshot attached.  You can still keep your own look as it will still be gnome, and all those differences will be in each users own ~/.gconf folder of configuration files.
