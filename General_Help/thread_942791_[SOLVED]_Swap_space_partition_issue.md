### Post by Sam on 2008-10-10
Replace in /etc/fstab
```
UUID=1f147c3d-0f25-4e98-8b5d-441d43535d9b none swap sw 0 0
```
with
```
/dev/sda2 none swap sw 0 0
```
Then run
```
sudo swapon -a
```
