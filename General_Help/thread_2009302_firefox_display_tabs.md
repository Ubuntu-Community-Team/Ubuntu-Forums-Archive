### Post by sgage on 2012-06-24
> **sgage said:**
> Go to about:config, set browser.newtabpage.enabled to false.

enter "about:config" where you would type in a URL

search for browser.newtabpage.enabled

double click on it to toggle between true and false.

setting it "false" removes the behavior that you want to get rid of.
