### Post by FluffyElmo on 2009-02-25
I just noticed you also asked the JVM IDE question earlier, I don't use Kubuntu but the basic steps are the same... 

First you need to install Java. Instructions are here, though if you search the forums here you may find Kubuntu specific instructions. [http://www.javadesign.info/SystemsHardware/OS/Ubuntu/install-java-on-ubuntu](http://www.javadesign.info/SystemsHardware/OS/Ubuntu/install-java-on-ubuntu)

Then you should be able to download the Easy Eclipse distro for Python ([http://sourceforge.net/project/downloading.php?group_id=131065&filename=easyeclipse-python-1.3.1.tar.gz](http://sourceforge.net/project/downloading.php?group_id=131065&filename=easyeclipse-python-1.3.1.tar.gz)), uncompress it and then double click the file to run the installer.

Hopefully you can get up and running, or maybe someone else will have an easier way to accomplish this. For instance I know there is a copy of Eclipse in the repositories but I'm not certain how to get it configured for Python so the Easy Eclipse package is probably the easiest way to go.
