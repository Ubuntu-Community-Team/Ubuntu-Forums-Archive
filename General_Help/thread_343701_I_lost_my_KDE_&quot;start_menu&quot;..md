### Post by Jucato on 2007-01-21
> **snakyjake said:**
> I lost my KDE "start menu".  I also notice the Taskbar appears  without virtual desktops and a few other applets.  However, I can see the Taskbar icons for Firefox, Desktop Access, Terminal, and Network Folders.

Right-click on the panel, select Add Applet to Panel. Look for K Menu (the "start menu"), Desktop Preview and Pager, and, if you're also missing it, the System Tray. Hope that helps.
