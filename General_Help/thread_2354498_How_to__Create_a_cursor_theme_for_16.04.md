### Post by Dennis N on 2017-04-02
If you just want to make a change in the existing cursors, I know you can open a cursor file in Gimp and edit it. Then export them as type "x11 mouse cursor (.xcm)" back to the cursor folder. Then, if I remember correctly, you need to remove the .xcm extension. For example you can change the fill color. See attached screenshot of Gimp where color of Paper cursor theme's left pointer was changed from black fill to gold fill.

Sounds easy, but there are a lot of cursors! This one (Paper) has three sizes, and that triples the work. Quite possibly there are special tools to edit these, but I have not looked for them.

Have fun.
