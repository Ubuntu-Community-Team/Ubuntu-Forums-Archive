### Post by 3Miro on 2009-04-07
Depending on the game, the installation process may vary. The standard way is to use:
```

wine /medial/cdrom/whatever_the_set_up_file_is_called.exe

```

However, often there are settings to be enabled in wine config manager and perhaps files to be edited.

Go to winehq and search their app database for your games. That is the correct source for wine games.
