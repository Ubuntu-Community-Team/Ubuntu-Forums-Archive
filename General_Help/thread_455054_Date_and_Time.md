### Post by pmg on 2007-05-27
I would expect the desktop clock to be controlled by the **locale** settings in effect when it's started... if it respects them.  Here's a page that talks about [localeconf]("https://help.ubuntu.com/community/LocaleConf?highlight=%28locale%29") that might be helpful.

If you can't figure it out, you could always install/run another clock applicaiton, such as dclock which allows it's output format to be highly customized.
