### Post by piquat on 2010-12-08
> **soryu said:**
> a data cd?


if i save mp'3 on a cd  will they still be mp3 when i (guess)extract them from a cd.

better if i compress and save? like a tar. or rar, zip.

Someone correct me if I'm wrong here but mp3 is already a compressed format.  You're not going to save much space by compressing them.  If most of this data is mp3's, you need a bigger thumb drive or burn them to a DVD.

And you won't "extract" them from a CD because there is no need to tar them in the first place.  They'll look like mp3's on any other file system, except they'll be on a disc.
