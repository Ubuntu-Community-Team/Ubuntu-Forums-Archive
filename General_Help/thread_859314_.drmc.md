I was messing around with the permissions, and now when I log on it gives me an error message, saying something about $HOME/.drmc not having proper permissions, and it should have 644 permissions and be available to no other users.  How do I fix this?  I've found the file they're talking about, but have no idea what 644 permissions are.

---

### Post by ibuclaw on 2008-07-14
644 are read/write, read, read permissions

permission numbers are defined by their binary equivalent.

ie:
```

rw- r-x r--
110 101 100
6   5   4

```
To fix it, use the chmod and chown commands.

```
chmod 644 ~/.dmrc
```
```
chown $USER:$USER ~/.dmrc
```
