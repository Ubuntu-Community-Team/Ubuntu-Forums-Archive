### Post by Navin_Johnson on 2005-07-14
I found a workaround if not a solution to the "Open File" dialogue of DVD Shrink not working (i.e. DVD Shrink through Wine -- thanks to Mr. Bass for his guide and postings on getting it working in wine!)

Open DVD Shrink and press the "Re-author" button without having selected "Open Disc" first.  At least on my system that opens up an area from where you can select a file or directory.

Now that I've discovered this I'm less frustrated at being unable to get transcode/dvdrip/lxdvrip installed!

Hope this helps
