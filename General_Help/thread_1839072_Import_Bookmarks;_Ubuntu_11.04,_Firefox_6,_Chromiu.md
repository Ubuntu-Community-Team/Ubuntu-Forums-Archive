### Post by kyrealtor on 2011-09-06
Mystery 1 Solved! Firefox 6 & up may require you to set up sync before creating the Import and Backup menu.
Once you set up sync, you can sync from another computer, or import a .json file via Import and Backup, Restore, file..
 
Then you can import settings, including bookmarks into other browsers such as Chromium that allow import from Firefox.
