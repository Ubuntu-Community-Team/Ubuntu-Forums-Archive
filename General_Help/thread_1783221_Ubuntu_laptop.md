### Post by seawolf167 on 2011-06-15
Just about any hardware will work - before you purchase it, do a little research on the main parts of the laptop (like graphics cards, webcams, wireless cards, etc.) to ensure there are no major issues with them. Though nowadays, most hardware you find out in the market will work just fine out of the box.

The issue with ordering laptops is paying fees for things you don't want on your laptop like Windows, Microsoft Office, etc. If you can, try to find one that doesn't come preloaded with an operating system (since you will be installing Ubuntu anyways).

Example places to look: [newegg]("http://www.newegg.com"), [linuxcertified]("http://www.linuxcertified.com/linux_laptops.html"), [system76]("http://www.system76.com/index.php?cPath=28"), [dell]("http://www.ubuntu.com/dell"), [tigerdirect]("http://www.tigerdirect.com"), [amazon]("http://www.amazon.com"), etc.

### Post by Jouke74 on 2011-06-15
There are hardware compatibility lists for Ubuntu versions X.X which you can check. Try to find laptops that use hardware which have open source drivers available (e.g. Intel, Atheros). Problematic hardware is often the wireless card and sometimes also the video card. Do not forget to check things like working function keys and the touchpad. 

I suggest you find a few laptops you like and then check the compatibility.

Also take your time to install linux as your main OS, on a new laptop that might take some time to get everything going. 

Before you start you might want to make an image backup of everything which is already on the harddisk in case things fail, you can at least make it running with Windows using the 30 days grace period (and have guarantee).

Finally a second computer during install to find things on the internet (in these forums) might be handy.
