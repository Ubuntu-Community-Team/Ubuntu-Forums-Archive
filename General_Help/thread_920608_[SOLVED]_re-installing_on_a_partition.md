### Post by vanadium on 2008-09-15
The easiest way to reinstall Ubuntu will be to delete your current Ubuntu partitions, leave the space open, then instruct the install program to use the "largest contiguous free space". The installation program will automatically recreate the needed partitions and (re)install the grub boot menu.
