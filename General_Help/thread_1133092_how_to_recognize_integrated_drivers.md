### Post by chomptown on 2009-04-22
Ok, so now go back to your File Browser, and go to the folder where you xorg.conf file is.  Right-click within the folder and choose the "Open as Root" option.  This will open a new window with a red background.  Open the xorg.conf file and past this over the last section.  If you aren't sure where to paste it, just ask, and I'll describe it better.  Next, you will reboot your computer.  Be sure to have a LiveCD to boot off of, just in case we have to change your xorg.conf file back to the way it was.

> Section "Module"
	Load 	"bitmap"
	Load 	"ddc"
	Load 	"dbe"
	Load 	"dri"
	Load 	"extmod"
	Load 	"freetype"
	Load 	"glx"
	Load	"GLcore"
	Load 	"int10"
	Load 	"type1"
	Load	"v4l"
	Load 	"vbe"
EndSection


Section "Device"
	Identifier 	"intel"
        Driver     	"intel"
	Option          "AccelMethod"   "XAA"
	Option 		"AddARGBGLXVisuals" "On"
	Option 		"XAANoOffscreenPixmaps"		"true"
	Vendorname	"Intel"
EndSection

Section "DRI"
	Mode 	0666
EndSection

Section "Extensions"
	Option 		"Composite" 		"true"
EndSection

Section "Monitor"
	Identifier	"Configured Monitor"

EndSection

Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
EndSection

### Post by chomptown on 2009-04-22
Yes, the Ubuntu disk.  If you boot up the computer and the video is all screwed up, you can use the Ubuntu disk to boot off of and fix your xorg.conf file.


Did you open the xorg.conf file as a root?  If you did, then take the part of your xorg.conf file that says:
> 
Section "Device"
Identifier "Configured Video Device"
EndSection

Section "Monitor"
Identifier "Configured Monitor"
EndSection

Section "Screen"
Identifier "Default Screen"
Monitor "Configured Monitor"
Device "Configured Video Device"
EndSection

and replace it with:
> Section "Module"
Load "bitmap"
Load "ddc"
Load "dbe"
Load "dri"
Load "extmod"
Load "freetype"
Load "glx"
Load "GLcore"
Load "int10"
Load "type1"
Load "v4l"
Load "vbe"
EndSection


Section "Device"
Identifier "intel"
Driver "intel"
Option "AccelMethod" "XAA"
Option "AddARGBGLXVisuals" "On"
Option "XAANoOffscreenPixmaps" "true"
Vendorname "Intel"
EndSection

Section "DRI"
Mode 0666
EndSection

Section "Extensions"
Option "Composite" "true"
EndSection

Section "Monitor"
Identifier "Configured Monitor"

EndSection

Section "Screen"
Identifier "Default Screen"
Monitor "Configured Monitor"
Device "Configured Video Device"
EndSection 

Then Click "Save"
