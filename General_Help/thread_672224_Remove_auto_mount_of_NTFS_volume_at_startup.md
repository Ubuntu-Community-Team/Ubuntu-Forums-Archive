### Post by forestpixie on 2008-01-19
edit fstab first

gksudo gedit /etc/fstab

put a # in front of /dev/sda1

then you can do your partition work, you can use gparted to add it to the sda2 - you need to move sda2 into the empty space first and then resize it

or format it to ext3 or whatever and have it as a seperate partition

you will then need to add it to fstab in it's own right for it to mount

### Post by vanadium on 2008-01-19
Do not hesitate to post in the beginners forum for rather basic issues. Place a comment sign before the line for /dev/sda1 (or delete the line altogether). Better yet, add the noauto option. This makes manual mounting easy as you do not have to type the options anymore.

### Post by MightyMe on 2008-01-20
Thanks for the help. It all went well. This is roughly what I did:

* Comment line in fstab to stop auto mount - reboot to test -OK 
* Started GParted and removed the NTFS partition - reboot to test - OK
* Start with LiveCD and run GParted - move sda 2 to the beginning, no resize - OK 
* Reboot without LiveCD  - OK
* Reboot again with Live CD - and run GParted to resize sda2 - OK

Done.
