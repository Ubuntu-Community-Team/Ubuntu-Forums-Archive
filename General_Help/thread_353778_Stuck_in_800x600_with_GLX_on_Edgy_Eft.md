### Post by renzokuken on 2007-02-05
could be wrong but you should try changing the Depth 24 modes to the same as the other "modes" entries for resolution.

i.e.

SubSection "Display"
Depth 24
Modes "1152x864" "1024x768"

probably ought to be

SubSection "Display"
Depth 24
Modes "1024x768" "800x600" "640x480"


either that or run 

```
sudo dpkg-reconfigure xserver-xorg
```

and select the appropriate resolutions that way

### Post by MiKeLEtE on 2007-02-06
Solved!

I've added this line ```
Option "UseEDID" "False"
``` to the Sectión Device this way:

```
Section "Device"
Identifier "Tarjeta de vídeo genérica"
Driver "nvidia"
Option "UseEDID" "False"
EndSection
```

I've killed the X, and then it's worked! :)

Thanks for the help.
