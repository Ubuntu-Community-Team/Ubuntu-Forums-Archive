### Post by lykeion on 2011-02-18
It would be simpler if you provided a link for the program you talk about. You can't assume people are familiar with a program that organise and manage your pigeons. ;)

I googled and downloaded it myself from [http://www.pigeonplanner.com](http://www.pigeonplanner.com) and after extracting the archive I looked through the README and INSTALL files. 
It instructed me to "run pigeonplanner.py to start Pigeon Planner". You'll find pigeonplanner (no .py extension) in the pigeonplanner-1.0.0/bin/ directory. Just double-click it to start the program.

The homepage also has a forum: [http://forum.pigeonplanner.com](http://forum.pigeonplanner.com) and I'm sure the author would like feedback and is happy to answer any questions you have about the program.
