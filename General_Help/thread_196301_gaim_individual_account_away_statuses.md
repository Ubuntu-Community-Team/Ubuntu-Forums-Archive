Well, I hope this is the right forum to post this on and if it's not please move it to the correct one. I can't find any info anywhere on the forums or even Google so I hope someone can help me out. As the title suggests, I'm having a problem with gaim and away statuses. I used to use it only for AIM, and was using aMSN for MSN chat because I didn't like gaim's MSN  interface. Well I recently decided to try it again because I'm now using gaim 2 beta 3, and now I'm having this problem. According to all the documentation I've seen (at least on gaim's sourceforge site for 1.5 - the beta 3 documentation didn't specifically mention this), there should be an option under the "Tools" menu to set individual away messages for all the accounts (Tools --> Away). As it stands now, I don't have that option and can only do global away messages for all the accounts at one time - e.g. I can't be away on AIM and available on MSN at the same time, it's all or none. I've looked through every single option available and found no such setting, even with a plugin. If it helps any, I recompiled gaim 2.0 beta 3 from source because of a guifications installation problem. So my question is, is that feature not built into beta 3, or is it something with my installation? Can someone with gaim 2.0 beta 3 from the repos confirm that they have / don't have this option?

EDIT: Well, I kinda found a way around the problem but it's a PITA and should be a little easier to do. I had to click on the "Away" box at the bottom of the Buddy List, click on "New", then click on the little twisty for "Use different status for some accounts". I can also modify my saved away messages this way, and choose individual statuses for all the accounts. There has to be an easier way to set individual away statuses for different accounts using the main window. I'm still waiting for anybody to tell me if they have different options in the repo version.
