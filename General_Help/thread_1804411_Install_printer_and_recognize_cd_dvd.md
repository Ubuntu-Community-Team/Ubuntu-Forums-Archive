### Post by CatKiller on 2011-07-14
> **mrag said:**
> I know in Windows, I go to a freeware site, find a high rated burner software like CDburnerXP, download it, click the exe file, install it and it is fairly intuitive to use. I've no idea what to do next with Ubuntu 11.04.

Generally, that approach is bad. There's a whole bunch of stuff already installed (including a reasonable cd burning application). If there's something you want that isn't already installed, you'd install a package through a package manager. Synaptic if you already have an idea of what the program's called, Software Centre if you're just having a browse. Find what you want, select Mark For Installation and hit Apply. Everything else gets handled for you.

Things from the repositories won't contain spyware, or any of the other fun extras that you get from trawling random sites for software.

Welcome to the forums, and good luck with your exploration of Ubuntu!
