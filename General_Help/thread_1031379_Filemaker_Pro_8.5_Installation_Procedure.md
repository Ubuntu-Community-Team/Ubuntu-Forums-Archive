### Post by NCFC on 2009-01-05
This is a c/p from the Linux Mint forum on installing Filemaker Pro 8.5. 

I've been struggling to get Filemaker Pro 8.5 installed in Wine 1.1.12 using the purchased CD.  The CD installation would always require root privileges and the installation never actually worked out.  There seems to be a total lack of a how-to instructions on this particular piece of software on the internet, that being that it's a Windows/Mac application.  For me however, it's the database software I prefer/use and the long journey of full installation on Linux Mint 6 has finally been fulfilled.  Anyhow, to make a long story short, here's how I got Filemaker Pro 8.5 installed in Wine 1.1.12 using the purchased CD, without root and the terminal.

1. Create .iso image of Filemaker Pro 8.5 CD using Brasero Disk Burning software. 
2. Open .iso image with Archive Manager.
3. Create new folder  in** Desktop** and extract contacts of .iso image into this folder.
4. Open **Files** folder. 
5. Run **Setup.exe**

The above process worked perfectly and installed into my .wine/drive_c/Program Files folder.  There were no hitches and the software works just as it does in Windows.  By keeping my database files located in my Dropbox[(http://www.getdropbox.com)]((http://www.getdropbox.com)) folder, I can now work on either my Desktop or Notebook without any worry.
