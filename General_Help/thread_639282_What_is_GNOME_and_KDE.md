### Post by LaRoza on 2007-12-13
They are [desktop environments]("http://en.wikipedia.org/wiki/Desktop_environments")

You can have both at the same time, however, they both can't run at the same time. Install KDE on Ubuntu with:

```

sudo aptitude install kde-core

```

When you log on, select "KDE" from Session.
