### Post by msammels on 2012-07-03
It's because you're using Microsoft >:D

No, but seriously, it could be a driver issue. How did you install Skype? The Ubuntu Software Center for some reason seems to hold Version 2.2 (Beta), for me anyway, whereas the actual Skype website has 4.0 - ignore the fact that it says Ubuntu 10.10 on there, it works fine.

All I'm saying is at this point, if you installed Skype via the Software Center, check the version, if it's 2.2 (Beta) uninstall it, purge it and replace with the one from [www.skype.com](www.skype.com) and see if that works.

If it doesn't, or you're already using 4.0, let us know.
