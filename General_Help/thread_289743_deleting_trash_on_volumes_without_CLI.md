### Post by hotani on 2006-10-31
When something on a volume such as a usb drive gets moved to the trash, it doesn't go to normal trash, it goes to a hidden trash folder you can't empty via the GUI. So the space has not been freed up. The only way I have found to delete items is to go in with terminal and do it manually. 

Alternatively, I could add a 'delete' option to my right-clicker but I tend to avoid that since it defeats the purpose of the trash system. Plus, this seems like something that would bite a beginner pretty hard: "I deleted everything off my thumb-drive but it still says it is full, WTF?"

This seems pretty basic, so I must be overlooking something. Any ideas?

### Post by mozetti on 2006-10-31
