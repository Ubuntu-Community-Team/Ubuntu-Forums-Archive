### Post by scrime on 2008-06-13
The best tool I've found to do this is Network Notepad [http://www.networknotepad.com/]("http://www.networknotepad.com/"). Unfortunately it's a Windows  program but still works well under Wine.  Another problem when using this with Wine is that the extra icon pack you download has all the files in WMF format, which seems to be not be supported under Ubuntu.  If you could convert these to ICO files and do a find and replace inside the corresponding PLF files this should solve the problem.  I'm currently trying to figure out how to convert these.

The program is worth taking a look at anyway, I wish the author would release a native Linux version.  Maybe even open source it.
