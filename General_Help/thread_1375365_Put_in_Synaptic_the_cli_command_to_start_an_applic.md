### Post by howefield on 2010-01-07
If the application has an icon on the menu, drag it to the desktop where it will create a shortcut.

Right clicking the icon and selecting properties will show you the command invoked when starting.

If it is a terminal application, the name of the app normally starts it, for the others, hmmmm... ;)

### Post by mechro on 2010-01-07
Synaptic can help you find the command for the application. In Synaptic go to **Settings > Preferences > General** - check the box for "Show package properties in main window".

Synaptic will then show a tabbed list including "Installed Files" for your installed applications. You should then be able to figure out the command binary as it's usually installed in /usr/bin or /usr/sbin.
