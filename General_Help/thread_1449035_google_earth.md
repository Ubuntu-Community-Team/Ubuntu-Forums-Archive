### Post by gadolinio on 2010-04-07
Go to the folder you downloaded the .bin file to. Right-click that file, choose properties. Go to the permissions tab, and tick the box "allow execution of the file as a program" or stg like that. Then click close to that window.
Next, go to applications--> accesories--> terminal. Type "cd /PATH/TO/WHERE/YOU/HAVE/THE/FILE/DOWNLOADED" (for example, "cd /home/YOUR_USER_NAME/Downloads). Hit enter. Then type "sudo", a space, and the file's name ("GoogleEarthLinux.bin") and hit enter.
Then follow the installer's instructions.
