### Post by soxs on 2008-06-03
There are always ways to get your data, especially if you manually open ports (by default all ports are closed) or send some one your bankcard and PIN or give someone your root passwd. or use a one letter root passwd, which can be cracked down in a few seconds...
So it depends on a bunch of things how secure your data actually is, especially on you.
Note: Ubuntu has a built in firewall
Note: [http://ubuntuforums.org/showthread.php?t=815111&highlight=anti+virus](http://ubuntuforums.org/showthread.php?t=815111&highlight=anti+virus) > **LinuxIsInnovation said:**
> Generally, are very, very few viruses for Ubuntu. So there's virtually no chances of infection. Plus, the security holes are fixes on daily basis through update. Hence you don't need an AntiVirus.
Though, you might try [Avast]("http://ubuntuforums.org/www.avast.com/eng/avast-for-linux-workstation.htm") Antivirus. Invoke it by using 
```
avastgui
``` from a terminal.
