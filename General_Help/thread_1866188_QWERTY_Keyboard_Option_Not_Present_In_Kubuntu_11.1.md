### Post by henshu70 on 2011-10-21
Recently I have upgraded from Kubuntu 11.04 Natty Narwhal to Kubuntu 11.10 Oneiric Ocelot (clean install). Overall a nice experience, however, there is one issue that makes my working on the computer a hell more difficult. I need on my system Slovak and Czech language keyboard inputs. No problem except there is no option to choose QWERTY keyboard layout. I would very much appreciate any help with this problem as I am getting desperate being used to the QWERTY layout. Thanks in advance to anyone with a possible solution.

### Post by henshu70 on 2011-10-21
QWERTY is not default but thanks to kubicle from Kubuntu Forums I have already solved the issue. I am quoting his reply:

'For some reason, the add layout option in SystemSettings>InputDevices>Keyboard>Layouts only allows adding the "default" layout sub-type (for czech and slovak). However, once you've added the layout you can change the layout sub-type (to "czech (qwerty)", for example) by double-clicking on the correct spot on the available layouts list.'
