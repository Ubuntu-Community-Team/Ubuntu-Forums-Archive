### Post by vambo on 2009-01-10
Try:

Alt-F2, then run ```
xfce4-panel
```, this should give you your panels back.
Make sure you have "Save session on logout enabled (which you seem to have). Also, in "Desktop" make sure you have "Allow xfce to manage the desktop" enabled.

When you next logout, then back in, hopefully the panels will still be there.
