### Post by PyroSamurai on 2012-05-05
Okay, the other post say it is impossible without having Yahoo mail plus. This is not true. To set up a @ymail account in Evolution all you have to do is set it up the same way as a @yahoo account except for one key difference: you add @ymail.com to both user name entries. That's it, and it will work. Your Welcome. I seem to be the first one to figure that out lol. 

Edit: And when I mean the same way as the regular @yahoo account setup I mean this: [http://askubuntu.com/questions/20075/connecting-yahoo-mail-with-evolution](http://askubuntu.com/questions/20075/connecting-yahoo-mail-with-evolution)
