### Post by eggdeng on 2008-02-15
fstab and GRUB use different notations for describing disks and partitions.
fstab will usually refer to the first hard disk as sda, the first partition on this disk is sda1, the second is sda2 etc.
GRUB starts counting from 0, so the first hard disk is hd0, first partition hd0,0, then hd0,1 & so on.
So, for example fstab's sdb3 will be GRUB's hd1,2. Easy, right  :)
