### Post by meijer.o on 2019-04-18
Importing a .gpx file to the 66s is no problem with Linux as it supports mass storage. It can be mounted as an external drive.  You can copy the file to a folder /Garmin/gpx on either the sd-card or on the internal memory.

### Post by RegUB on 2019-04-18
I have a Garmin Etrex. I'm not interested in recording or creating tracks, as I only use it if I need to confirm where I am on the map. However, according to its manual you can download a Garmin application called BaseCamp which allows you to "create and manage your adventures". But it seems there are only Windows and Mac versions of this.

Garmin records tracks in a binary format (.fit) but you can convert these to gpx using a program called fit2gpx. There is mapping software called gpsprune which runs on Linux and can display tracks from gpx files.
