### Post by howefield on 2010-01-07
When you install Chrome via the .deb package from the google website, it will automatically add the Chrome repository to your sources.list, meaning that you will get the updates as and when they are available. So in that respect, your browser will be managed for you.

You won't have to worry about using an out of date Chrome any more than say, Firefox ect.
