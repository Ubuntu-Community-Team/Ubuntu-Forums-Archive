### Post by howefield on 2010-08-13
Might be worth checking that your desktop is set to show files,... (it isn't the asterix that is hiding the file)

Press the Alt + F2 keys together and type 

```
gconf-editor
```

and press run.

Navigate to apps > nautilus > preferences > show desktop
