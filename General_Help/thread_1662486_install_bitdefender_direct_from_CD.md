### Post by howefield on 2011-01-09
First off, do you have a reason for installing anti-virus software ?

[https://help.ubuntu.com/community/Antivirus](https://help.ubuntu.com/community/Antivirus) 

Is your Bitdefender CD for linux or windows ?

You can download a .deb package from the Bitdefender website which would be easier to install, simply double click the .deb file to start the install process.

To install Google Earth and have downloaded the .bin file from the Google website, open a terminal and type

```
sudo sh ./GoogleEarthLinux.bin
```

assuming that's the name of the file.
