### Post by FaceToChair on 2011-12-13
So I have wubi and have been using it for almost a year. I have too many files to put on a flash drive and don't want to move the files into windows. Also don't want to move wubi to a new partition as I have so much random stuff installed. 

Can I install Ubuntu to get the files and make sure everything is working the same before I get rid of wubi?

---

### Post by bcbc on 2011-12-13
Yes. Install your new dual boot. It won't notice the wubi install.

Then you can either boot the wubi and mount the new partition, copying data across, or loop mount the wubi install from the new one.

It's a good idea to backup your data first (even though you said you don't want to)
