### Post by DeMus on 2010-01-03
> **Creative Lad said:**
> i want to back-up program

I use grsync which is in the universe repository. Install it with Synaptic and read as much as possible about it, [here]("http://www.opbyte.it/grsync/") for example.
Grsync is the graphical frontend of a command line program called rsync and simply does what it needs to do: make backups the way you want to and also is able to put it back when necessary.
