### Post by sdennie on 2009-02-22
It should be safe to install the 32-bit compatibility stuff.  It just means that 32-bit apps running on a 64 bit machine can access OpenGL.  If you don't use any 32-bit apps, at worst, you've lost a very small amount of disk space.
