### Post by avtolle on 2008-06-19
Xubuntu will run; you will need the alternate install iso burned to CD, however, because as I recall, to use the graphical installer under Xubuntu, 192mb ram is the minimum needed. You will have a learning curve for the XFCE desktop environment, as it is different from Gnome, and there are some differences in menu items, etc. to become accustomed to. "Under the hood", the basic O/S is the same; again, the differences are w/ the desktop, Thunar (not Nautilus), etc.
