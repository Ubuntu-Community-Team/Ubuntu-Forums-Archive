### Post by rraj.be on 2008-06-06
Gammu and wamu can help you a lot in this regard.


to install wammu and gammu try the following in terminal.

```

sudo apt-get install wammu

sudo apt-get install gammu
```

You can use ```
[WINE]("www.winehq.org/")
```

to install some windows applications in ubuntu platform.

To install wine type the following in terminal

```
sudo apt-get install wine
```


NOTE: not all windows applications can be installed using wine.

You can use your favourite phone manager like myphoneexplorer or xplorer through this wine.

For more help regarding installing through wine just see the home page or private mesage me or post a new thread.
