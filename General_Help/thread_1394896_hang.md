### Post by NightwishFan on 2010-01-31
Make sure to run a check on your CD for errors, though the issue may just be with Ubuntu itself. To check a CD, place it in the drive, reboot and choose "Check CD for Errors". If you find any, I would advise you burn another CD using the slowest speed possible, to prevent errors.

If the CD check passes, we can troubleshoot the problem then.
