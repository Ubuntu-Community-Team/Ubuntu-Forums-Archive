Is there an app that can this?

You might want to try my media center extension for Firefox - [FoxMediaCenter]("http://fmc.isgreat.org/").

It has an EPG with xmltv support, TV recorder with ability to schedule recordings (including weekly series), a media playlist manager with tagging support and several other features. It's a lightweight media center alternative.

I wrote this extension because I wanted a simple TV recorder and programming guide. MythTV is great, but it has too many features for me.

I'm not sure if it will work with your TV card, because the extension relies on TVTime to change channels before every recording and I only tested it with a analog tv card. Anyways, you will only know if you try it.

Download, screen shots, user guide and support at [http://fmc.isgreat.org](http://fmc.isgreat.org)

Please read the "Important Notes" and "Requirements" sections of the web site before installing it.

I hope you like it.
