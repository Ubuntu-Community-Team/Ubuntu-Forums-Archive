### Post by mlomker on 2005-09-22
> I would prefer to have all the above info in one app though (maybe I'm stuck in my Windows-centric ways!!).


Evolution has most of the functionality because it has an Exchange server plug-in.  Setting it up isn't always painless and I found that it crashed way too much to actually use.  A part of my problem is that I'm an Exchange server administrator!  lol.

One possibility for you is to buy a copy of VMWare and run Windows within a window on Ubuntu.  This requires a machine with 1 Gig of RAM, realistically, and buying the software is about $190.  It keeps you from having to reboot during the day and it *is* Windows, so it'll run almost any Windows app that you need.  They have a [30-day demo](http://www.vmware.com/products/desktop/ws_features.html)  on their website if you want to kick the tires.
