### Post by zootreeves on 2006-11-10
My university requires that all Internet connections go through their proxy. For windows I use a program called proxifier which automatically sets all programs to access the internet through the proxy. Is there a similiar program for linux? Is it possible to just set a global proxy? Thanks

---

### Post by Dr. Nick on 2006-11-10
you can set a global proxy in gnome, go to system - prefrences - network proxy.
 
That will take care of proxy for some of the native gnome apps like epiphany and possibly gaim.
 
The rest you may have to setup by hand
