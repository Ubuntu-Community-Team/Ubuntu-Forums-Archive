### Post by Rhubarb on 2008-06-29
You could try sauerbraten.  It's an open source FPS game (works with linux / windows / mac) with very very nice graphics.
[http://www.sauerbraten.org/](http://www.sauerbraten.org/)
Once installed, tweak up all the extra graphics options, and load up a pretty map / go online.
Some maps a very nicely made, others are ok.

Sauerbraten won't stress your CPU or RAM though, and probably won't stress your video card that much.  (Turn on bloom and everything else though).
