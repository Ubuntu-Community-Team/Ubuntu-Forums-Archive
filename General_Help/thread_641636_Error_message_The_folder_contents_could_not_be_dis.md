### Post by gabuleon on 2007-12-18
I did a search on google and it appears that it is a known bug.  The workaround that some have come up with was from a terminal type $ pkill nautilus.  The default windows in ubuntu is gnome and nautilus is the file browser.  It worked for me but others have said that specifically after browsing thru music files especially it messes up again and again even after doing that.  Hope this helps you as it did me.
