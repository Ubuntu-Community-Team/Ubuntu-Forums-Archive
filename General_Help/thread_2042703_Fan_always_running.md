### Post by Mark Phelps on 2012-08-15
There is no general solution for this as it is a known bug with recent Linux kernels and, so far, has only been fixed for some Sandy Bridge processors.

What you can try is installing Jupiter to see if you can throttle back your processor:

[http://ubuntuforums.org/showthread.php?t=1854019]("http://ubuntuforums.org/showthread.php?t=1854019")
