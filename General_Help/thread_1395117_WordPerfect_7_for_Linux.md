### Post by srs5694 on 2010-10-05
First, I recommend WP8 rather than WP7 on Linux, although somewhat tentatively. Your father's preference for WP7 over WP8 is almost certainly based on the Windows software, not the Linux software. Although they probably have some code in common, the two are different enough that what was more stable on one platform may be less stable on the other. My memory is that WP8 was the superior product on Linux, although my memory about this is rather imperfect -- it *was* over a decade ago that I used WP7!

I have heard of people running WP8 on fairly recent versions of Linux. In fact, until about a year or two ago, I did so; however, it was rather flaky on that last installation of mine. The trouble is that it was written a decade ago and so relies upon libraries and other configurations from that time. You'll have to install a bunch of very old compatibility libraries, some of which don't even ship with modern versions of Ubuntu. There are sites out there with instructions, such as the one to which Nicolas.Perrault linked; however, I note that that site is itself five years old!

Another comment: You mentioned fonts. IIRC, WP7/8 for Linux will not *exactly* match the WP7/8 for Windows fonts. There will be small changes in kerning, size, etc. I've got a very elderly (last updated in 2002) Web page on the subject of fonts and printers for WP for Linux [here.](http://nessus.rodsbooks.com/wpfonts/index.html)

As to obtaining it, try eBay. Also, look for the books *Special Edition Using Corel WordPerfect 8 for Linux* (which I wrote -- Roderick W. Smith) and *WordPerfect for Linux Bible* (by Stephen E. Harris and Erwin Zijleman). Both come with WP8 for Linux, although my book only comes with the reduced freely-redistributable version; *WP for Linux Bible* comes with the full version. Amazon lists several [used copies](http://www.amazon.com/gp/offer-listing/0764533746/ref=dp_olp_0?ie=UTF8&redirect=true&qid=1286300745&sr=8-1&condition=all) of the Harris book. Just be sure you get the CD-ROM with the book and you should be set. (Both titles are long out of print.)

The idea of running WP in a virtual machine may also be worth exploring; but if the main use of the computer is word processing, running it in a virtual machine seems like a needless complication to me. IMHO, it's better to just run Windows in that case, unless you've got some compelling reason not to.
