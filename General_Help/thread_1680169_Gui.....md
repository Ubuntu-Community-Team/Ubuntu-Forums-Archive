A regular installation of Ubuntu server is text only. If you want a GUI interface (which some tools may or may not have a GUI tool you can use) you can install Gnome, KDE, XFCE, LXDE, etc. from the terminal via:

sudo apt-get install <packagename>

Where <packagename> denotes what package you want to install. It varies depending on what you need and what packages you want.
