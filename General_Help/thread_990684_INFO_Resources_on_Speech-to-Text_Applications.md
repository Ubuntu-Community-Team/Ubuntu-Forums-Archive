### Post by Th3Professor on 2008-11-23
This is by no means a definitive source of information on speech-to-text applications. However, it is a plain and simple copy/paste of links found pertaining to these apps. If I get a chance I'll update/revise the links, though you can also just check any replies that people leave for more info.

Note: This applies to applications for **speech-to-text**. This does *not* apply to applications for text-to-speech.

Disclaimer: If there is redundancy of resources within the links, we'll see if we can clean up the list later (or just reply with updates). Similarly, if I accidentally slip in some text-to-speech sources instead of speech-to-text, let us know. The emphasis here is speech-to-text only, which can include dictation, commands, or anything else that is initiated with the voice. Some sources may be out of date though their basic information could still apply in cases where you replace old versions with more recent versions of apps. There will be some links to apps that assist with speech-to-text apps. These sources are presently in no particular order.
