### Post by WorMzy on 2011-03-22
Press alt+F2 and run the following:
```
gnome-appearance-properties
```

Next, make sure you're on the theme tab, then click "Customise".

Click the window border tab, and pick one that suits you.


Alternatively, if you have Compiz installed, you can use the resize plugin to be able to resize windows without needing to grab the window's border. Just hold alt down, and middle-click near the edge of the window you want to resize. Or press Alt+F8.
