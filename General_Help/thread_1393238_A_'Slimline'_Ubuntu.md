### Post by cascade9 on 2010-01-29
Running from a USB flash drive would probably be a fair bit faster than a liveCD. If you did a minimal install to a USB flash drive, it would be faster still. This would only be true for USB 2.0, USB 1.x would probably be slower than liveCD.

Only problem is you would need 2USB ports for this to work, as you want to backup to USB. But that shouldnt be a problem, almost anything that has USB 2.0 has more than 2 ports.
