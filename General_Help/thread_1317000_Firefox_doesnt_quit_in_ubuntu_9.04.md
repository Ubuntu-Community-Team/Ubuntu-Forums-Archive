### Post by jbrown96 on 2009-11-06
This can happen if your profile is locked. Firefox locks the profile when it opens, but if it crashed and didn't unlock it, then this can happen. 

Launch it from a terminal and post the output here. Apps-->Accessories-->terminal, then ```
firefox
```
