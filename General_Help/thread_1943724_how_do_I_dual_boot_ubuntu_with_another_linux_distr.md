### Post by Bucky Ball on 2012-03-20
Yes, no problem. You need to make free space on your hard drive to put them. To shrink/expand/delete/create partitions boot from the LiveCD and use Gparted (partition manager).

Bout as simple as that. Install the next OS. With Ubuntu, choose 'Something Else' when it gets to partitioning and you will see the existing install there. Just don't go near it and untick 'Format' if it is ticked.

If you have a separate /home partition already your new (Ubuntu and a couple of others) install can use that also. Untick the format box there, too. Post back anymore questions and don't forget to have a search first.
