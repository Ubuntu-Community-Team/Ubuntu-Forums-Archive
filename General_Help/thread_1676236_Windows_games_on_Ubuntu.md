### Post by Mark Phelps on 2011-01-27
I have read that, once you have installed Wine, you can then use it to run Windows games installed inside your Windows partition.  But, have not confirmed that actually works.

First thing you should do is go to the WineHQ site and look up the info for each game you want to run:

[http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true")

Unless you get a rating of Gold or better, I would not waste your time trying to run it.
