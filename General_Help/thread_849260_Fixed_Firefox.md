### Post by steveneddy on 2008-07-04
I was a little frustrated with how slow my browser was, especially after a fresh install. I thought FF3 was going to be better.

I found that installing 

```
ubuntu-restricted-extras
```

that I got some more java codecs, even after I had installed iced tea, and went to a java test page on the internet and then was told to install a GJC java variant to get the java applets, and now the browser is zippy and fast.

```
sudo apt-get install ubuntu-restricted-extras
```

Then by going to this website:

[http://www.bodo.com/javame.htm](http://www.bodo.com/javame.htm)

Firefox detected I didn't have and downloaded the GLC plugin for java applets.

Now with all of this said and done, I do block java with the FF3 plugin, but I am only saying that the browser works better for me and is noticeably faster between pages after this round of software installs.

You mileage may vary, but it's working for me.

Have a great day!
