### Post by p_quarles on 2007-12-08
The first step in debugging a problem like this is to run the app from the terminal. Hit alt-F1 and type:
```
exaile
```
By running it this way, you will be able to see any error feedback that comes up.

Now, I wasn't sure whether you meant that Exaile itself is freezing, or if the entire system is freezing. If it's the latter, you should be prepared to write down any errors you get in the terminal. Otherwise, it's easy enough to copy-and-paste them to a text file.
