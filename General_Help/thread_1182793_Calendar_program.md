### Post by Zill on 2009-06-09
jmichael:  If your work schedules are on a web page then you should be able to use a web query to import the data to an OpenOffice Calc spreadsheet:

[http://www.openofficetips.com/blog/archives/2004/11/webquery_scrapi.html]("http://www.openofficetips.com/blog/archives/2004/11/webquery_scrapi.html")
[http://plan-b-for-openoffice.org/ooo-help/r2.1/en-US/MAC/calc/guide/webquery]("http://plan-b-for-openoffice.org/ooo-help/r2.1/en-US/MAC/calc/guide/webquery")

Once you have your data on a spreadsheet then you *may* be able to import it to a Calendar application, such as Evolution.  You will probably have to try saving it in different formats, such as csv etc.  Getting the field names to correlate with the data may take a bit of tinkering as well. ;-)

Good luck. :-)
