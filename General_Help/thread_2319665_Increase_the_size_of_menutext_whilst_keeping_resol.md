### Post by Dave_Berry on 2016-04-06
In windows there is a setting "change the size of text, apps and other items" which will increase/decrease the size of the UI of the operating system whilst keeping the same resolution. Just installed Xubuntu on my laptop and the resolution is 3840 x 2160 which makes the icons/menu ridiculously small. Is there a setting in Xubuntu/Linux that I can change that will increase this for me?

---

### Post by Impavidus on 2016-04-06
In Xubuntu, go to settings&#8594;appearance&#8594;fonts. Use settings&#8594;desktop&#8594;icons fo icons on the desktop. Have a look around for other settings you want to change. For many applications you'll have to change things in the application itself.
