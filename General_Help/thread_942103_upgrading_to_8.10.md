> If I'm forced to do a clean install would it be safe to just take my current home folder and replace the one that gets installed or should I just transfer each folder I want individually,

If you have a seaparate home partition, then the data should be safe.   If not, then the data will be erased.

**Back up** your data always before upgrading or reinstalling.   There are no 100% guarantees that all will go well.
