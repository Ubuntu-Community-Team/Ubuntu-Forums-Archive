### Post by greenco on 2008-08-31
I am backing up some of my files onto CD Rewritable disks and I need a CD burner that will let me add files to the disk, once it has been ejected. I am using the K3b burner right now and I have to format the disk (remove the files that are already on it), before it will let me add additional files. ### Post by mrtomservo on 2008-08-31
If i'm understanding you correctly, I think what you want is a Multi-session disc.  K3B will do this just fine.  When you go to burn your data and the dialog comes up, it should be under "Misc", and then "Multi Session Mode".  It should be self-explanatory from there.  I think you can open an existing one by going to File, New Project, Continue Multisession disc.  Good luck!
