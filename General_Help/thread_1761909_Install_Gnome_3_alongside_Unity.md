### Post by ivanovnegro on 2011-05-18
You could but is not really recommended at the moment because Unity uses Gnome 2 and not 3 for Unity anf if you install Gnome 3 from the PPA or from the PPA from the [Ubuntu Gnome Remix ]("http://ugr.teampr0xy.net/")project it could break your system, but I heard the UGR project is at the moment the preferred way.
So you cannot install Gnome 3 like the other metapackages of desktops into your Unity install because its not yet in the official Ubuntu repos.
Gnome 3 will be in the repos with the 11.10 release.
But there are people they did it with the PPAs and some of them installed it on top of Xubuntu 11.04. 
If you want a stable system, a really stable system without to experiment too much I wont do it but hey, you have the possibility. :D
