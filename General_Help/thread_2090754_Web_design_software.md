### Post by lildigiman on 2012-12-04
I second the use of Aptana Studio 3 even though it doesn't have WYSIWYG. I've used that ever since it came out and I'm very pleased with it aside from being a bit slow (thanks Java...) sometimes. If you're using any sort of framework you can set it up with auto-complete and such. The FTP access is also nice as well as GIT support.

I personally don't see a reasonable benefit with WYSIWYG; your HTML and CSS should be separate anyway. Once you've got the basic HTML setup go ahead and pop in any CSS you'd like.

But that's just me.
