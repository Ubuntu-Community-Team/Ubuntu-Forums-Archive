### Post by mb_webguy on 2008-08-09
> **PsychedelicWonders said:**
> so from my understanding, GNOME is basically a program that allows you to customize ubuntu with any idea you have?

Um, not exactly.  Gnome is the [desktop environment]("http://en.wikipedia.org/wiki/Desktop_environment").  Unlike Windows, which only has one desktop environment that is integrated into the operating system, Linux can have any of several, or none at all.  Without a desktop environment, Linux is nothing but a command-line system, like what you get when you run the terminal.  A desktop environment gives you... well, a desktop.  Icons, windows, menus, and that sort of thing.

Several desktop environments are available for Linux, including Gnome (used by Ubuntu), KDE (used by Kubuntu), and Xcfe (used by Xubuntu).  Each of these desktop environments does basically the same thing, but each has its [advantages and disadvantages compared with the others]("http://en.wikipedia.org/wiki/Comparison_of_X_Window_System_desktop_environments").  Xcfe, for example, is smaller and uses less resources than Gnome or KDE, and is therefore better for older computers, but consequently doesn't have some of the advanced features of Gnome or KDE.

Actually, Both Gnome & KDE are "Window Managers" for Linux, just as Windows is a "Window Manager" for Microsoft's Operating System.
That is, they do all the "on-screen" stuff. Maybe this will help (extremely simplified):-

Microsoft OS -> Windows -> You
Linux OS -> Gnome -> You
Linux OS -> KDE -> You

Whether you choose Gnome or KDE (or anything else - others are available) is entirely your choice.
For myself, I use Gnome, but with several KDE programs as they have the features I need.

### Post by sub2007 on 2008-08-09
Both are free, download them both (in fact you already have Firefox in Ubuntu) and try them both out. I actually put both on every install I do, for some reason Opera renders some pages better than Firefox.

Firefox is open source and very extendible with extensions. It also handles Flash better on Ubuntu. It has full GTK integration (so it blends with GNOME/Xfce, Opera is Qt based) although in my experience it runs up more RAM than Opera. I recommend on Linux trying out something like Swiftfox or (my personal favourite) Swiftweasel which I find are more stable and run faster (although these won't run on Windows, they are essentially Firefox).

Opera is less extendable but it comes with shedloads out of the box. As well as full web browser with things like mouse gestures it also packs an email client, IRC client and torrent program (as well as a really good download manager) out of the box. It does all this without feeling bloated (runs up lower RAM than Firefox/Swiftweasel in my opinion and loads faster) and many say it loads web pages faster than Firefox or Internet Explorer. It also works very well on Windows (as does Firefox) and is functionally identical. What's very impressive is that they are very proactive in keeping their Linux version up to date.

If I was asked the best I couldn't call it. I know for me Swiftweasel is the one I always use because I love my extensions but I also love Opera and could really very happily use it.
