### Post by spikoley on 2011-05-07
> **hamolton said:**

Can you boot into recovery mode?  If not, boot into the live CD.  Next edit xorg.conf to change the driver to "vesa".  Then reboot and log in under Ubuntu Classic mode.  On the log in screen click on the user name then at the bottom you can switch to Ubuntu Classic mode.
