### Post by audax321 on 2005-03-31
Not entirely sure if this is what your looking for, but I programmed a renamer a long time ago under windows. You will need to run it with wine though. Its called X-Rename and is available here if you want to try it (just a single .exe file):

[http://www.dieburnbot.com/X-Rename.exe](http://www.dieburnbot.com/X-Rename.exe)

It'll has the the following modes:

Mass Rename: Renames multiple files one after the other.
Append: Allows the user to append the same prefix or suffix to multiple filenames.
Rename: Renames all selected files with different extensions to the same name. (useful for a rar archive with multiple files and the filenames are screwed up)
