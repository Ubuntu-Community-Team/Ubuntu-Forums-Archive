### Post by mimada on 2008-04-29
I had some problems with FF3 so I uninstalled it and reinstalled v2. I can wait for the official release. I also noticed that the Hardy upgrade on my machine replaced my original Java installation (Sun 1.6.0_02) with the open source version. I fixed my Java issues by uninstalling all versions of Java and reinstalling just the latest Sun version from the repositories.

P.S. backup your Firefox bookmarks before uninstalling. Also, close off Firefox before messing with Java, just to be on the safe side.
