You can actually export a slide as a jpeg instead of doing a screen capture. Impress is probably better at adding text than Gimp, but I don't think you can control the resolution. If you're working with a high res picture and you use the Impress method, you'll end up with a low res picture with nice text on it. If you need to keep it high res, you'll have to use an image editing program.
