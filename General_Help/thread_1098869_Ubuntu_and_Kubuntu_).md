### Post by mb_webguy on 2009-03-17
As lykwydchykyn said, the primary difference is that Ubuntu uses the Gnome desktop environment, and Kubuntu uses KDE.  This means that the desktop will look different and be organized a bit differently.

Because some programs use the GTK+ widget library and others use the Qt library, the former integrate better with Gnome (which uses GTK+) and the latter integrate better with KDE (which uses Qt).  The Ubuntu team therefore uses different programs in Ubuntu than in Kubuntu.

For example, Ubuntu uses Brasero as the default disc burning application, while the default in Kubuntu is k3b.  The default music player in Ubuntu is Rhythmbox, while the default in Kubuntu is Amarok.  Of course, because they're all still Linux programs, you can run Amarok in Ubuntu or Brasero in Kubuntu if you want.  The applications that don't match your desktop environment will simply look a bit different from those that do. 

Really, it boils down to personal preference.  KDE4 is pretty, but IMO isn't quite as stable as Gnome, so I use Ubuntu.  (Though recently I've been using Openbox instead of Gnome, and I like Amarok and k3b better than Rhythmbox and Brasero.  So while I'm using the Ubuntu distribution, I'm not using the Ubuntu desktop.  Linux is flexible like that.)

Because Ubuntu, Kubuntu, and Xubuntu are just different flavors of the same distribution, you can try a different one without reinstalling whichever you're currently using.  If you want to try Ubuntu (and you're not currently using it) then install the ubuntu-desktop package, if you want try Kubuntu, install the kubuntu-desktop package, and if you want to try Xubuntu, install the xubuntu-desktop package.  Then when you login, go to Options and select the session you want to use for that login.
