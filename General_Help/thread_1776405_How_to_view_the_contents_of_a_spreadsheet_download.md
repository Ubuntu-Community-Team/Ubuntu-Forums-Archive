### Post by seawolf167 on 2011-06-06
Just as an FYI, in the new versions of office (2007, 2010), the extensions xls**x**, doc**x**, ppt**x**, are only there so the correct program opens the files. These are actually zip files containing all relevant information (which is why it saved as a .zip file and not a .xlsx file)
