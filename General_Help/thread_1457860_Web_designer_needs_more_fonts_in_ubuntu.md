### Post by Kai Summers on 2010-04-19
Just find ttf fonts that you like, there are many sites that provide them, download the ttf file to your computer, then open it on your local machine. When open there is a button on the bottom right of the font window that can be clicked to install the font.

### Post by fragos on 2010-04-19
I'd caution you about using special fonts in web designs. Your site visitor may not have the font you used and the browser will default to a basic serif or sans-serif that's available. You can however use special fonts to create image files for site headers and the like but again there is a downside in that the text on images can't be indexed by Google and others for search results. there's a lot of information on designing sites on my FragosTech site.
