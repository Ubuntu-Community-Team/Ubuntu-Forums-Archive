### Post by kaldor on 2011-05-16
Copy the name of the theme from Mint's /usr/share/icons directory if you already have it installed elsewhere.

Personally, I dislike Mint's default icons; they blend two themes together and make some stuff look too out of place.

The square icon set is a popular theme called Faenza. Here:

[http://tiheum.deviantart.com/art/Faenza-Icons-173323228](http://tiheum.deviantart.com/art/Faenza-Icons-173323228)
