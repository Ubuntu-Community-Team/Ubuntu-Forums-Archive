### Post by Mark Phelps on 2013-06-28
> **AbhimanyuAryan said:**
> bf3, skyrim, nfs shift, gta4, and other graphic intensive games

Not a gamer, so I have to ask -- are these native Linux games? Or, are they Windows games?

Because if the latter, sorry to tell you, but there's no such thing as "full Ubuntu support"!  Why? Because you'll have to run them in Ubuntu using Wine and its companions -- and there's no guarantee that any games will work well in that situation.  If you check the WineHQ website application compatibility tool for the games and versions you want to run, unless you see a Platinum rating for every game, you are going to be disappointed in the gaming experience.

Sorry, but the unfortunate fact is that the only "full support" platform for running Windows games is -- Windows.
