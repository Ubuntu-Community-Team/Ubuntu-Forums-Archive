### Post by martinbc on 2008-11-12
No problem.

Just go to:

1. System -> Administration -> Synaptic Package Manager 
2. Enter your password
3. Search for eclipse, mark for install and press apply!

And there you go, eclipse on Ubuntu!

enjoy :)
