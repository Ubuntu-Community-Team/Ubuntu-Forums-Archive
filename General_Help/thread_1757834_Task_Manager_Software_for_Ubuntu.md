### Post by nickleboyblue on 2011-05-13
I use Zim.  It's a desktop wiki that, with plugins, can render mathematical equations, do spell checking, and various other things.  It is not a website, but it does have an interface much like a web browser.

I use it to keep track of daily tasks because it has support for checklists, and I use it for taking notes in college classes.  I also use it to keep track of any project I am working on.  In short, if you keep it organised enough using sub-pages, you'll be able to keep track of virtually any information you need to.

Outside of that, there are a few simple programs for to-do lists, though I prefer a lot more functionality than just that.

Also, if you want other people to see your notes, plans, etc., you might consider hosting a copy of wikimedia on your computer.  You first need to install a LAMP server, and set yourself up with port forwarding and dyndns services so that the outside world can have access to your site if they know the address.  Then you need to download wikimedia from synaptic.  A quick search online will show you how to set up a basic installation.  Once finished, you and your friends can see your work.  It's not nearly as user friendly as Zim, but others can see and edit your wiki with wikimedia if you want to let them do so.
