### Post by Crash Dummy Dave on 2010-07-23
Thanks for the helpful links. The Runescape game was using the Java plugin, and the sound was not working in Firefox, but was working for Chrome. Firefox was still the problem, not Java. However the sound in these games is now restored by the recent (today) update of Firefox to 3.6.7; so that again proves it was Firefox's fault and not Java, since I had made no other changes at all.

Anyway if anyone loses sound in Runescape and other similar sites in the future using Ubuntu and Firefox, it is often simply Firefox that is the problem. This has happened before in at least one previous Firefox version. For these reasons the Runescape Linux forum prefers Google Chrome.

I also was having a video issue with a gif image not being fully loaded by javascript (not Java) on another web site, and that also was corrected by installing the new Firefox release.
