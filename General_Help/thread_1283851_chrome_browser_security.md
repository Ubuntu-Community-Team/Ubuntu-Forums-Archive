No. It's running natively. Google Chrome is based on the open source [Chromium](http://code.google.com/chromium/), which has a Linux version under development.


> **ub9876 said:**
> Anyway...since its a Linux code ,is it safe like Fox in Linux..?

No. Firefox is a matured browser while Chrome/Chromium for Linux is not even beta yet, so is not intended for production environment and might have bugs, including security related. Well, any software has security flaws, including Firefox, but most of the time they are patched as soon as they are discovered. 

It seems the concept behind Chrome is more secure than Firefox, but I wouldn't trust an application under development.

> **ub9876 said:**
> Anyone know or comments..??

Make a search for Chromium in the forums and you will get tons of threads with all sorts of opinions. Nevertheless, most users agree it is among the fastest browsers on the market. Fortunately, Mozilla is doing a great job recently. Firefox 3.5 already provided a huge performance improvement and 3.6.1a is almost as fast as Chromium.
