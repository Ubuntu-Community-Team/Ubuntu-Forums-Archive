### Post by ubuntu27 on 2008-03-17
In order to use a second language in KDE, you have to install the kde-i18n packages. For example, if you want Spanish support, you have to install kde-i18n-es
if you want Japanese support, you have to install kde-i18n-ja 

Look for your desired language with Synaptic or Adept.

After you installed the international language package, you have to go to COntrol Center, and then languages. Select the language that you want. That's it :)
