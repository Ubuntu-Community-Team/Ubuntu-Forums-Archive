### Post by HermanAB on 2009-03-24
Hmm, to actually answer your questions:

Linux Game Emulator
This is Linux, you don't need a Linux emulator.

[] CDCheck

### Post by pbpersson on 2009-03-24
For a scanner GUI front-end you can either use XSane or Kooka.

To connect to a network printer you should just be able to add printer and tell it to add a network printer, assuming it is being shared either from a Windows machine or a Linux machine using Samba.
