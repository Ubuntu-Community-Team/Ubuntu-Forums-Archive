### Post by mc4man on 2011-07-23
What version of ubuntu are you using?
The default in the current releases is to disable the touchpad for 0.5 sec.'s after each key press (normal key press), which is pretty much unnoticeable

In the current ubuntu dev version (11.10 - onieric) this has been increased to 2 sec's which can be quite an annoyance depending on use.

In any event you can turn off the touchpad disable in your mouse preferences, how you open depends on what you're using (system setting, gnome-control-center, ect.
