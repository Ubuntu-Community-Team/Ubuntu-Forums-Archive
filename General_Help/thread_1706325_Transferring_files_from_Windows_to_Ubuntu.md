### Post by lithopsian on 2011-03-13
Just mount the external drive in Ubuntu and read any files you want.  You'll have to save files in some proprietary formats into a format that is readable in Linux, but most common files can just be opened in Linux applications.

If you have sufficient disk space you can just hive off a partition for Ubuntu, mount the remaining Windows partition and read the files directly.  No matter if you've already copied them.
