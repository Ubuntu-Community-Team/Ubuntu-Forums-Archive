I'm not an UbuntuKylin user but I was curious about this towards the end of the 18.04 develoment cycle so I decided to install UbuntuKylin in Virtualbox. The set-up process can give you an English installation just like any other Ubuntu flavour but some applications haven't been translated so are in Chinese only. Obviously once set up to your liking everything can be switched to Chinese, there'll be a language setting somewhere.

Beware UbuntuKylin looks nothing like any other flavour of Ubuntu. The latest release uses the [UKUI]("http://www.ukui.org/") desktop so you may have your work cut out to find all of the places where you can change settings. It was an interesting experiment.
