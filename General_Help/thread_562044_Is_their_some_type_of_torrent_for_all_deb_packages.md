### Post by Insightfill on 2007-09-28
Found a good link [here]("http://www.howtoforge.com/dvd_images_of_ubuntu_repositories").  In particular, keep note of "debmirror" to get it local, and then the "debpartial" command to split the results.

It's not bittorrent however, but I think it uses standard rsync under the hood to get the contents, so it's resumable.

Some of the contents and recommendations are particularly good.  I especially like the use of the "popularity contest" info to decide what goes on the first CD, etc.  This is close to the trick used for the Debian downloads - there are a bunch of CDs, but you only really need the first one (or two).
