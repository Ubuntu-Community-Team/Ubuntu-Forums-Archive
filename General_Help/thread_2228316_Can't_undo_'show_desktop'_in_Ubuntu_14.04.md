### Post by 3rdalbum on 2014-06-06
I don't use Show Desktop, sorry, so I didn't even notice a difference.

However, maybe you should use Workspaces instead? You get four "workspaces" or "virtual desktops", a bit like monitors you can switch between, and each can have different windows or no windows at all. If you need to see the desktop for any reason, just switch to a different workspace with no windows.

Look on Google for how to enable workspaces in Unity.
