I have an odd problem setting up my Epson Stylus Photo 890 for remote printing from Ubuntu 10.04 through an SFR Neufbox...

I've moved into a rented house which has a Neufbox like this:
[IMG]http://www.le-test-adsl.fr/wp-content/uploads/2008/12/neufbox.jpg[/IMG]

I point Firefox at 192.168.1.1 and press the "service" button on the Neufbox to get into the administration screen, and then I follow the instructions [here]("http://doc.ubuntu-fr.org/serveur_impression_neufbox") to set up the printer. It takes a long, long time to detect and to list the available PPDs, but in the end it seems to work.

I tried printing the test page, and saw that the page got 18% into the job, but then it seemed to stall.

The ink levels reported in the admin web interface were at 97% for each ink, but then after sending the test page the ink levels just disappeared, although the printer is still listed as attached to the Neufbox.
