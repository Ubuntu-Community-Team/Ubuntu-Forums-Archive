### Post by drs305 on 2011-10-02
Fletch89,

Welcome to the Ubuntu forums.

Computers don't just come with Windows and Ubuntu. Windows would have had to be installed at some point, or come preinstalled. While Ubuntu is free, Windows is not and can only be installed legally in compliance with its EULA. Since you weren't the purchaser and the OS was removed, legally I don't believe you would have no right to use the software even if you could recover it.

You can use a Windows CD to install it in compliance with the EULA. Trying to restore deleted partitions most likely would violate the Microsoft terms and the Ubuntu Forums do not allow discussions about operations which can potentially violate EULA's.

If you have a legal copy of Windows to install on the machine, it can live happily along side Ubuntu. And if you have questions about using Ubuntu, we are always here to help.

### Post by akand074 on 2011-10-02
Check for a recovery partition (it may show an option at boot). Some recovery partitions are intentionally hidden (Acer's for example). I believe it's alt+F12 or something on boot to run it on acers(you can do a quick search). HP always has a recovery partition and is never hidden as far as I know. You can usually select it on boot. If you have a recovery partition you can just boot into it and it'll automatically wipe your entire system and return it to it's factory default settings. You might not if the original user wiped it as well. Otherwise, if you can get your hand on a Windows Vista disc (or whatever came with your system) you can install Windows that way and use the license at the bottom of your laptop.

### Post by DaneAU on 2011-10-03
As mentioned usually you will required a specific disk to revert back to vista, during the installation process usually it asks for how much space you wish to use or whether or not to recover to the entire drive.

If you don't have the recovery disk but would like windows installed also, you will have to edit your partitions and ensure you leave both untouched. I do recall previously dual booting and encountering some issues at times on my laptop and decided to stick with one operating system and a Virtual Machine inside that operating system. Worked much better.

So saying that i think for you the best option is to either install one operating system or use something like VirtualBox to run XP inside your ubuntu install. Unless you are gaming you shouldn't have any issues at all.
