### Post by cor2y on 2008-12-01
Test the microphone is working via gstreamer's multimedia properties.
you can reach it via System->Preferences->Multimedia System Selector 
or Alt+F2 then type gstreamer-properties
or type gstreamer-properties in the terminal.
Check what device is set for audio capturing and test if it works.
