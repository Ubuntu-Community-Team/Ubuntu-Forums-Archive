### Post by TheFu on 2015-09-02
For what purpose. Workload matters greatly.

Start by listing all the applications to be used by each seat concurrently. Typical disk and network I/O used per person. Whether the seats will use different versions of these programs - different versions of firefox will use more RAM than everyone running the same version.

If CAD software is being used, that is very different than someone using ssh or running g++ compiles or surfing the web.

Workload matters.
