I think you've got your wires crossed a bit.

The thread you've linked to is from 2007 and it concerns a regular PC with a touchscreen, in tablet form-factor.

Fxy is asking about whether Ubuntu can be loaded onto an Android-based tablet. This isn't a regular x86 PC, it's an ARM-based "appliance". More like a smartphone than a computer.

The process of installing Ubuntu to one of these "smartphone-like" tablets would differ from machine to machine; they certainly aren't designed to be able to boot up anything other than Android, and it would need to be hacked to run Ubuntu. Also, you'd need an ARM version of Ubuntu that corresponds with the internal hardware of the tablet.

And it would need to have all the drivers already - not likely.

The best thing to do is to either stick with Android, maybe find a third-party firmware like Cyanogen that gives you a later version of Android. Or, do a Google search for that particular model of tablet and the word "ubuntu" to see if anyone has found a procedure and written some drivers.

The TL;DR version: Probably not possible and definitely not easy; unless you instead bought a tablet that has an x86 processor and is able to run Windows 7.
