### Post by nichipet on 2007-07-14
I don't know if I can really answer your questions, except maybe #2, but I do know this: Ubuntu Studio is a repackaging of Ubuntu with extra software installed from the beginning. That's about it. As far as I know, anything Ubuntu can do, Ubuntu Studio can also do. I installed Ubuntu 7.04 on a friend's laptop and then upgraded it to Ubuntu Studio with no problems. I am trying to make dual monitors work (it works in Vista on the same machine), and I installed wine already. Ubuntu Studio does not come with wine installed, but it's easy to add.
