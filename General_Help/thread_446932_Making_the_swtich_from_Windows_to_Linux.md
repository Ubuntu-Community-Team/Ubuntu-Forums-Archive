### Post by starcraft.man on 2007-05-17
Your problem with installing the 3d managers can have several reasons for failing. I can only imagine the most obvious ones being you failed to install your 3d drivers for your card (ati or nvidia). If thats the case the easy fix for that is to use the Envy Script, [here.]("http://www.albertomilone.com/nvidia_scripts1.html") Download it and install the debain by double clicking. Then go to applications > System Tools > Envy, open it up select the automatic installer of whichever card you have and answer yes to any prompt, when you reboot youll have drivers.

If you had drivers, I can only imagine it was a bad install, refer to my sig ubuntuguide and go to the section on eyecandy, follow the install for your card. 

As for Edgy, you didn't give much info about the connection. If its a wired ethernet line and you have DHCP on you should just plug it in and be connected. If you done this and it doesn't work you need to give us more info, your modem maker, your router if you have one and any other relevant info.

Oh and you don't need to use bold or caps, we read plain text just fine.

Edit: Ya, I somewhat agree with hammedhaaret, you might want to just install Feisty the latest and see if that fixes your connection trouble.
