### Post by aysiu on 2007-10-30
> **HelixAgent said:**
> Because I am not a very experienced user in Ubuntu and I am trying to get used to it AND there are a few softwares that make their installationg easier like some codecs and plugins I cant remember the name.
Go to System > Administration > Software Sources

Check all the boxes.

Then go to Add/Remove, show All Applications, and install *ubuntu-restricted-extras*

That should take care of the most common codecs and such. Better than being stuck in dependency hell.

Or, better yet, use Linux Mint instead of Ubuntu. It comes with all the proprietary codecs any Automatix user would want.
