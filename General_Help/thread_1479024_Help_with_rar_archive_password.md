### Post by ryan858 on 2010-05-13
Try typing in **CTRL+SHIFT+U** and you should should see a _u_ appear, then type in **08**. Do not press enter until it looks like this:
_u08_

This is basically like a windows "alt-code", but it's Unicode instead of ASCII.
