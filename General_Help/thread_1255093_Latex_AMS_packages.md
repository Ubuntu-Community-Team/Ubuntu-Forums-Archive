### Post by rafita on 2009-09-01
amsmath is included in [http://packages.ubuntu.com/jaunty/texlive-latex-base](http://packages.ubuntu.com/jaunty/texlive-latex-base), so you probably
already have it. If you are starting with LaTeX, probably you should install texlive-doc-en 
([http://packages.ubuntu.com/jaunty/texlive-doc-en](http://packages.ubuntu.com/jaunty/texlive-doc-en)), and do some reading, like say

texdoc lshort

and then 

texdoc amsldoc
