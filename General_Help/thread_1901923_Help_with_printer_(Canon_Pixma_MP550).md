### Post by groundhog136 on 2011-12-30
PIXMA Linux Compatibility Page states MP550 driver available autumn 2009:

[http://www.canon-europe.com/Support/Software/Linux/PIXMA/index.asp](http://www.canon-europe.com/Support/Software/Linux/PIXMA/index.asp)

Managed to find a driver here:

[http://software.canon-europe.com/products/0010755.asp](http://software.canon-europe.com/products/0010755.asp)

Canon state tested with 9.04, I'm using 11.04. Initially would print test page (better than nothing?) but restart allowed print of documents.
