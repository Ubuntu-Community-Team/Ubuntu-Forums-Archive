### Post by nereid on 2007-02-16
Hi, it depends on the program you're using. It is the same as in Windows. Some programs reside in the tray and some do not. In the normal case you quit the program by hitting the 'x' on the window border. Some applications like Skype or Gaim will continue to run in the background and you can get you window back by usually clicking on the tray icon of the respective application.

Hope I could help you.

**Edit**:
If you come from the Mac OS X background, then it is understandable that you're confused. As everything will reside in the dock. On Windows/Linux the concept is a little different.
