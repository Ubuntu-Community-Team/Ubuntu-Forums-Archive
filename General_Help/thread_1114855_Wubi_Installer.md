### Post by philinux on 2009-04-03
Yes it install like a .exe file would. It creates it's own folder. Uninstalling is via add/remove software just like a regular .exe programme.

It does download about 700 meg of OS though so be aware of that.

[http://wubi-installer.org/](http://wubi-installer.org/)

Click the faq's link top right.

Like any major event, backup anything important first.

### Post by clawlor on 2009-04-03
