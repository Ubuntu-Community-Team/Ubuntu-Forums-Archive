### Post by hxcobd on 2010-05-15
Eshad, for what it's worth, Ubuntu has been the only linux distro that has worked 100% for me out of the box without any (considerable) tweaking whatsoever.

KDE is pretty horrendous, especially if you use an external audio card/interface. XFCE is pretty bad about handling dual-monitor setups, in my experience.

Gnome is really just the most solid desktop environment. On top of that, themes in Gnome/Metacity are MUCH easier to change/tweak than themes in KDE or XFCE.

I'd try Ubuntu first and foremost. If that doesn't quite hit the spot, Kubuntu. If that's still not preferrable, Xubuntu.

After those, I'd try one based on Enlightenment, like Elive.
