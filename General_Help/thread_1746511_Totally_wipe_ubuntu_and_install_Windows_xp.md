### Post by Mark Phelps on 2011-05-03
If you're trying to do the resize while running Ubuntu from your PC -- that won't work.  You can't do work on partitions when they are mounted, and they have to be mounted in order to run from them.

You will have to boot from an Ubuntu LiveCD and do the partition work from there.
