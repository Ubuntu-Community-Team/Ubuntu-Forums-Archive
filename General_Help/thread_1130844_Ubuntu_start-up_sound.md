### Post by kpkeerthi on 2009-04-20
To customize the sounds, look in:
System -> Preferences -> Sound -> Sounds
System -> Admin -> Login Window -> Accessibility tab -> Sounds

To edit sound, try [http://audacity.sourceforge.net/]("http://audacity.sourceforge.net/")
