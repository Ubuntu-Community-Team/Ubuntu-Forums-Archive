### Post by lovinglinux on 2009-06-07
Open CompizConfig Settings Manager, then click the "Widget" plugin then click the "Behavior" tab and untick the option "End Widget Mode on Click".

If the problem occurs when you open the screenlets properties, then I'm afraid I can't help, because it also happens to me. It opens the screenlets properties window outside the widget layer, so the layer is closed. If I want to modify  the screelets preferences while viewing it, then I need to hit the shortcut key twice to display the properties window along with the screenlets layer.
