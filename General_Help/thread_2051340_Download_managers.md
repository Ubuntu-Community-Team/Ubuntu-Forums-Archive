### Post by petersonlevi88 on 2013-01-27
If you want a downloader for say, movies that is pretty universal and is fairly automated, try jdownloader. Open your terminal and put these commands in the order I post them. Works great.

Code: 
sudo add-apt-repository ppa:jd-team/jdownloader

Code: 
sudo apt-get update

Code: 
sudo apt-get install jdownloader


Then open up dash home icon on ubuntu style desktop, click on the jdownloader icon and it will start updating package. This takes a while. I did this with no experience on terminal. Just copy and paste 1 command at a time. Easy as pie my friends. I like to use a site for movies and much more called warez-bb.org. Just thought I would put it out there since I had a rough time finding help on this.

P.S. Do not copy and paste the word "code:" Start with the word sudo and is put in exactly as you see it even spaces.

### Post by slickymaster on 2013-01-27
Give Steadyflow a test. It's a simple download manager, written in GTK. It has a lot of good features without any unnecessary complexity. Download process is quite simple – just add the URL, choose destination (you can also go for default) and it can easily handle parallel downloads.

To install it just type at terminal:
```
sudo apt-get install steadyflow
```
