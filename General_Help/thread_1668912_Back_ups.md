### Post by RedSingularity on 2011-01-17
You can make a clone of your HDD and update it from time to time.  I use Clonezilla to clone my HDD for that purpose.  If you mess up the system clonezilla will help you to return the system to the working state you cloned to the other HDD.  Google it and see if it fits your needs :)
