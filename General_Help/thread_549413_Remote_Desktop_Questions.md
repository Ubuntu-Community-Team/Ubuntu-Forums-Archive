### Post by orangey on 2007-09-12
Go to System->Preferences->Remote Desktop and enable it there.  Its not unlike windows XP.  Windows uses rdp, but ubuntu uses vnc.  You can get the vnc viewer client [here]("http://www.realvnc.com/download.html").  Install the client on your winxp system and connect to your ubuntu box.

To remote desktop to a windows xp box, just go to Applications->Internet->Terminal Server Client.
