### Post by Zeroangel on 2006-03-24
But don't dogs run extremely fast? :p

If you have an AMD processor, you should give [Swiftfox]("http://ubuntuforums.org/showthread.php?t=142798") a try. There are tweaks on the internet that can be used to make FF 1.5 much faster and less memory intensive. You could, for one, do a search on this board for 'Firefox Pango' to run Firefox without the pango internationalization feature which makes FF *crawl*. You could also try a search on the internet for "Firefox bfcache" to reduce the amount of caching the Linux versions of Firefox do (default FF 1.5 can eat up to 200MB of memory with only a few tabs open).

Swiftfox already has most of these customizations done on it already.
