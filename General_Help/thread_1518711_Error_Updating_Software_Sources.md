### Post by stderr on 2010-06-27
'Could not resolve .....' means that your PC couldn't successfully discover the IP address of the host from your DHCP server (or hosts file, etc). The fact that it isn't able to determine the IPs for any of them indicates loss of Internet connectivity, or possibly a DHCP server failure.

Can you actually browse the Internet (with a browser with no special proxy configuration settings) when you are unable to update?
