### Post by desire.linux on 2011-09-14
I'm on 10.04 LTS, and I'll sooner or later need to upgrade to 12.04 LTS.

---

### Post by Quackers on 2011-09-14
I would back up everything I didn't want to lose and then upgrade.
However, many people here recommend a fresh install in these circumstances, as a lot can change in 4 releases.
You could try the upgrade first and if that doesn't go well do a fresh install from the cd or usb.

---

### Post by seawolf167 on 2011-09-14
I'm on the side that will recommend a fresh install from 10.04 -> 12.04. You are skipping 4 releases ahead, IMO there are things bound to cause headaches. 

Quakers advice is sound though - make a backup, then try an upgrade to see if it works, and if it doesn't, you can always do a clean install after.
