### Post by lmarmisa on 2011-04-11
I think your hard disk is not healthy. If this hard disk is new and it is under warranty, I recommend to change it for a new one.

Try to repeat the repair procedures of the thread [http://ubuntuforums.org/showthread.php?t=1699286](http://ubuntuforums.org/showthread.php?t=1699286). Then, if you are able to boot into Ubuntu, open System -> Administration -> Disk Utility and check the SMART status and data of your hard drive.

I believe you have a hardware problem not related to Ubuntu.
