### Post by merlinus on 2007-05-23
You can download the .iso disk image file from ubuntu.com.  Be sure to burn it as a disk image and NOT as a data disk.

Then after installing XP, boot from the ubuntu disk.  When it is up-and-running, click the Install icon and follow the partitioning instructions at 

[http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)

Good luck!
