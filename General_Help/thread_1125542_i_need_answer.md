### Post by mb_webguy on 2009-04-14
IPv6 doesn't really have anything to do with memory.  It's a protocol used by network devices.

And yes, there are a few servers and clients using IPv6, though not many.  This number will likely grow rapidly in the next few years as Internet usage in places like China increase and the number of IPv4 addresses begin to run out.

Some people claim that Internet access is slower when IPv6 is enabled, but many others claim there's no perceptible difference -- which there shouldn't be.
