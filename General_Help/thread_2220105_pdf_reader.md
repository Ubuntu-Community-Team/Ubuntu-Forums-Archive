### Post by m_duck on 2014-04-26
The default in Ubuntu is Evince. As already said by [**[COLOR=#000000]Paulgirardin[/COLOR]**]("http://ubuntuforums.org/member.php?u=952449") and [ 						 							]("http://ubuntuforums.org/member.php?u=208550")[**[COLOR=#000000]andrew.46[/COLOR]**]("http://ubuntuforums.org/member.php?u=208550"), Okular is really rather good. My goto reader is usually epdfview, though it appears no longer to be in the Ubuntu repos...

You can set the default viewer for PDFs by right clicking a PDF file and choosing **Open With Other Application...** and choosing a program from the list, ensuring **Use as default for this kind of file** is checked before clicking **Open**. Alternatively, you can use XFCE Menu -> Settings Manager -> MIME Type Editor and reset the file type on PDF files.
