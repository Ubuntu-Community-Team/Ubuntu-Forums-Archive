### Post by phen on 2005-06-11
Hi!

the easiest way to install applications is to download .deb files. ubuntuguide shows how to install them:

[http://www.ubuntuguide.org/#installuninstalldebfiles](http://www.ubuntuguide.org/#installuninstalldebfiles)

note: sometimes you will have to download multiple packages in order to resolve all dependencies. the program will inform you about.

you can find a large collection of software for ubuntu here:

[http://archive.ubuntu.com/ubuntu/pool/](http://archive.ubuntu.com/ubuntu/pool/)
