### Post by timcredible on 2009-06-25
1.2 is out at the songbird website.  just download the file, double-click on it, extract to your home folder, and then double-click on the file named 'songbird' to run it.  it's a little slow to start, but very nice. i suggest installing all the recommended plugins, plust install the mediaflow plugin, which you can do when it first starts up, it opens a browser tab of their website and just click on 'mediaflow'.
