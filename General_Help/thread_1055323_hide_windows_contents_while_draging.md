### Post by sedawk on 2009-01-30
Depends on your window manager, e.g. "enlightenment" has a very
"scientific" wireframe mode when dragging windows (configurable).

With default Ubuntu look at
System->Preferences->Appearance->Visual Effects.

If I remember right there is some "Preferences button" in that
windows only active when you have installed some extra compiz
configuration tool.

Listing available packages:
```

aptitude search compiz |grep config

```
