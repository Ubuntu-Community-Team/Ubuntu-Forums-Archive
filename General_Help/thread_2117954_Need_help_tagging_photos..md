### Post by LewisTM on 2013-02-20
+1 for gThumb.
Get the latest (3.1.2) from the Webupd8 PPA.
[https://launchpad.net/~webupd8team/+archive/gthumb](https://launchpad.net/~webupd8team/+archive/gthumb)

The file properties dialog also lets you comment the file and edit the tags.
The Tags floating window is modeless so you can select multiple files, tag them, press Alt-s to save the changes, then select other files and repeat without closing the tag window. Pretty much like a dashboard.
Once you have tagged files, you can click 'Organize' (top right in the folder view) to create catalogs based on tags. The Find feature also supports tags.

Cheers!
