### Post by yosiska on 2006-10-20
Hi everyone below is the link to get flash player 9 beta from ADOBE.

[URL="http://www.adobe.com/go/fp9_update_b1_installer_linuxplugin"]
Click Here to download[/URL]


Intructions to install: (make sure all the browsers are closed)

1. untar the file

2. Click terminal and type "sudo nautilus" (no quote). Enter password if prompted.

3. navigate the un-tared folder.

4. Copy the **libflashplayer.so** from the folder.

5. Navigate the /usr/lib/<browser name>/plugins

6. Find the **libflashplayer.so** rename the file to **libflashplayer.so123** (Just in case you wanna go back to version 7).

7. Paste the new libflashplayer.so

8. You are done with the upgrade!!
