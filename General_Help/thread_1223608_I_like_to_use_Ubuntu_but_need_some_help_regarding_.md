### Post by vegetarianshrimp on 2009-07-26
Just to clarify what linux is: [http://www.linux.org/info/](http://www.linux.org/info/)
Ubuntu is a OS which uses the linux kernel, it is called a linux distribution, or flavor.
 
First of all, you can't run windows programs in linux, unless there is a linux version, or that particular program works with Wine (winehq.org). Wine lets you run windows programs in linux, but usually doesn't work.
 
Second, there are advantages and disidvantages of linux, which you should go over before making it the primary OS for your computer. ([http://packratstudios.com/index.php/2008/04/06/the-pros-and-cons-of-linux-windows-and-osx/](http://packratstudios.com/index.php/2008/04/06/the-pros-and-cons-of-linux-windows-and-osx/))
 
Third, there is a way to have both windows and Ubuntu on one computer, it's called dual boot, when you turn on your computer, you choose which OS to boot into. Its much easier to install windows, then Ubuntu, not vice versa. ([https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot))
 
Hope that helps, :)
vegetarianshrimp
 
P.S. About the partitions, when you boot into the LiveCD and double click install, there is a little "wizard" that guides you through installing Ubuntu, which you can install on a seperate partition than windows, to get a dual boot system.
