### Post by utnubuuser on 2009-01-11
Hi -- My personal experience has been that different versions of Ubuntu react and function differently on various hardware.  For example, I have an X31 Thinkpad, which worked flawlessly running Feisty Fawn, but poorly with Gutsy Gibbon.  I'm currently using Intrepid Ibex, which works like a charm on my machines, whereas the original release of Hardy Heron was choppy.
I've read similar threads about such version regressions.  If you like Ubuntu, use it.  If not...(see signature). 
Also, it's probably a good idea to give people a bit of an idea which hardware you're referring to. - sometimes a simple problem can effect your entire system.

### Post by blazemore on 2009-01-11
I'm sorry to hear things haven't been working out the way you'd hoped.
I would suggest burning the CD again, perhaps at a lower speed.
Check the disk using the in-built checker (On the installation boot menu)

And reinstall if it passes the check.

Failing that, Ubuntu may not be the distro for you. A couple of other Distros to try are Fedora (There's a great review in this month's Linux Format) and OpenSuse, along with more "advanced" distros such as Slackware and Gentoo.
