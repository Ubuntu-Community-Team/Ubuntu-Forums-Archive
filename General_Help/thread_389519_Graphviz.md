### Post by gerbman on 2007-03-20
As far as I know, GraphViz is really a set of utilities for producing graphs of all sorts. The most popular utility (in my experience) is `dot', which produces the nice graphs you've probably seen. Install GraphViz again, and run```
man dot
``` from the command line for information on using Dot.

I'm not sure how useful a menu item for Dot would be. Under normal operation, you have to supply a few command line arguments to the program for it to run. This would make a menu item nearly useless.
