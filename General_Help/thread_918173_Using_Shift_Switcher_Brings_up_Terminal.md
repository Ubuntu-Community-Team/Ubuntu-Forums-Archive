### Post by daemon3 on 2008-09-12
> **elyena said:**
> So, relatively simple to explain problem.
I try and use the Shift Switcher in compiz, and every time I open it up, a Terminal window also pops up, unrequested.  How do I fix this?

You probably have the terminal shortcut enabled.  In the main gnome menu, go to System -> Preferences -> Keyboard Shortcuts.  Included in the list is "Launch Terminal."  I'm sure you'll be fine from there. :)

If you didn't think you'd need "Keyboard Shortcuts" and removed it from the menu, the application can also be in gnome-control-center.
