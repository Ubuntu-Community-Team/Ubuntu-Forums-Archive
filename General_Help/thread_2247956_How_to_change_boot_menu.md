### Post by Dennis N on 2014-10-11
> **huxley2 said:**
> Hi,

I'm dual booting with Ubuntu 14.04 and OpenSuse 13.1, but I'm having too many wireless issue's with OpenSuse, so want to get rid of it. Because I installed OpenSuse last, I automatically go to it's boot menu on start up. How can I switch back to Ubuntu's boot menu, before deleting OpenSuse? 

Thanks.
This is a common method for a BIOS system:

1) Boot into Ubuntu
2) Run these commands in order in the terminal (assuming your HD is /dev/sda)

```
sudo grub-install /dev/sda
sudo update-grub

```

Reboot, and you should reach a boot menu generated by Ubuntu.
