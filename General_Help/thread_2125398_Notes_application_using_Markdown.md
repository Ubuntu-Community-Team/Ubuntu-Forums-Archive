### Post by niranjan_rao on 2013-03-13
Hi there,

I apologize if this is wrong forum for this kind of problem. I wrote yet another notes application and wanted to get general feedback for the application as this is my first python application and first application written for GTK platform. You can treat this application as pre-alpha as there are lot features and possibally issues in the application.

I know there exists plenty of application for notes taking on ubuntu but they were lacking something and I was not happy with any of those applications at least the one I tried. Finally I decided to do something about it and wrote my own application. The application allows you to save notes in markdown format.

The application uses Webkit as a primary UI device. Currently it uses python CherryPy as backend, but I am planning to remove it as soon as possible.

You can find the application at [https://github.com/nhrdl/notesMD](https://github.com/nhrdl/notesMD). I have developed/tested this on Ubuntu 12.10.

Clone it from gihub. 
Make sure dependencies are met - python-cherrypy3 and peewee for python. I installed cherrypy using apt-get and peeweek using pip.

Since there are lot of firsts, this application may not be conformant to python or GTK style of applications. If that is the case, please be gentle with me.

All comments/issues/features wishlist welcome.
