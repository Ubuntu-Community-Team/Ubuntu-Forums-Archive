### Post by wgscott on 2007-02-13
You may wind up having to install ubuntu to get xubuntu (I did).  But I prefer Xubuntu as it has a very clean, streamlined interface. I use it on a machine with a gig of memory and two 3.2 GHz processors, and it is definitely snappier than Gnome or KDE.  The nice thing is it is fairly easy to switch between desktops, so you don't have to make a choice ahead of time. Just see which you wind up going back to.

### Post by ehowell on 2007-02-14
You do have a couple of choices, you can install ubuntu and then install the xubuntu desktop later (so you have a choice).

Or, just install the server version which does not have a GUI DE. This is what I did and then I just installed the xubuntu desktop with aptitude later. It works great on a ~1998 Gateway 500 Mhz with 384 MB RAM and a 8 0GB HD.

Either way, the choice is yours. The server route would give you the default LAMP stack and you wouldn't have to take up disk space with Gnome.

But then you couldn't compare the two either :)
