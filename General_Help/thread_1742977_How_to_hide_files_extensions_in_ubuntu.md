### Post by matt_symes on 2011-04-29
Hi

As far as i am aware there is no simple way to hide a file extension because, as you pointed out, linux can identify the file without using an extension.

Therefore, in linux, the extension is not so much of an extension (file identification) as with microsoft, but actually part of the entire filename. (if you see what i mean). It' not something special to be hidden.
