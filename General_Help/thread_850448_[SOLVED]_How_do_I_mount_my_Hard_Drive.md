### Post by sisco311 on 2008-07-05

>  /dev/sdb1 /media/DATA ntfs-3g defaults,locale=en_GB.UTF-8 0 0to
>  /dev/sdb1 /media/DATA ext3 defaults 0 2save the file and exit.

```
sudo mount -a
```To get write permission:
```
sudo chown -R $USERNAME: /media/DATA
```

### Post by drs305 on 2008-07-05
> **ludatha said:**
> Thanks, but I cannot seem to save my fstab... is there a way to disable the whole permissions thing forever?

You need to open the fstab editor with administrator privileges. If you are running gedit (substitute nano, mousepad, etc):
```
gksu gedit /etc/fstab
```

There are ways to extend the time between required password entries as well as other workarounds. They are controlled by the /etc/sudoers file. Do a search for 'sudoers' and 'visudo'.
