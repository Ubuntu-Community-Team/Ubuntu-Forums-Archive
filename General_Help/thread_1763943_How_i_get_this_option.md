### Post by prshah on 2011-05-21
> **MMALLOW said:**
> PDF FORMAT INCLUDING WEPSITE CAN YOU 

In Ubuntu, creating a PDF from a webpage means you print to a (PDF) file.

Accordingly, you can setup your page to include the web address. Click File-Print-Options and in the "Header and Footer" section, select "URL" from the top right box. This will ensure that the URL is included in the PDF page when you Print (To File, PDF)

Please post back (in lowercase) if you need more information.
