### Post by ellisf2 on 2014-08-21
**Note: although I can now add the search engine I wanted, I still don't know why Firefox isn't being properly recognized. Any ideas?**

I just installed Ubuntu 14.04, which has Firefox as the default browser. When I try to add additional search engines to Firefox, I get an error message that says:

"Sorry, you need a Mozilla-based browser (such as Firefox) to install a search plug-in."

I don't even know where to begin to try to remedy this problem, since I'm using Firefox 31. I have Classic Theme Restorer installed, but I tried disabling that and even removing it and it made no difference. I currently have other Linux distros installed (Ubuntu forks) and this is not a problem in them.

Any suggestions?

[SOLVED] For anyone who has the same problem and finds (or stumbles upon)  this thread, all you do is go to the site in question. Once there,  click on the search engine menu "arrow" and you should (we hope) see  "Add the (search engine in question) near the bottom of the list. Click  on it and the search engine will be added. One caveat -- I've only tried  it with those two search engines, so far. For example, if you want to add the IMDb search engine, simply go to [www.imdb.com]("http://www.imdb.com"). Then, click on the search engine bar's down arrow to look at the list of installed search engines. "Add (the search engine in question)" should appear at the end of the list of installed search engines. Clicking on that will install the search engine.
