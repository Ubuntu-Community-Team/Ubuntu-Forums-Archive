### Post by uncaspi on 2008-11-17
I assume you have a wubi installation?? In that case ubuntu is a file in your windows partition. So you must increase the windows partition  i.e using partition magic.exe or delete some of your windows files. If you selected to little space when you installed ubuntu and have a lot of space for your windows installation I don't know why ubuntu should do this automatically. Probably it had to do with the formatted file system in ubuntu which is ntfs-3g and probably you have to reinstall and choose more space. Maybe other on the forum has the solution.
