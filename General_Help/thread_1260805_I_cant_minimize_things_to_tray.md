### Post by ankspo71 on 2009-09-08
Hi,

Before I got used to using multiple workspaces, I used a program called alltray. If you run that program, you will be able to send any other program into the tray. I don't use this program anymore though, because now I prefer multiple the workspaces instead (for me it's one less program running).

to install from the terminal:
```
sudo apt-get install alltray
```
It can easily be found in Synaptic Package Manager too.
