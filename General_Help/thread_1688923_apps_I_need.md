> **usagiakumu said:**
> 
  1.       Complete Microsoft PowerPoint 2010 replacement
a. video background with word overlay

c. Live editing, where I can edit on the first screen should I notice an error on the second and simply hitting a hotkey and it auto-fixes the slide and everyone being none the wiser

I don't know anything about this sort of functionality. Ubuntu ships with Openoffice.org (10.10) or LibreOffice (11.04) which contains a presentation show program. Pretty sure it has multi-monitor support.


> 2.       Android contacts manager and editor

I don't believe anything exists for Linux to do this. I just use my phone, or my Google account in my web browser.

> 4.       A complete Microsoft Publisher replacement… too much to list as I generally use it all…

Scribus. You'll find it's more like QuarkXPress than Publisher, but then there's a reason why typesetters don't use Publisher.

> 5.       Graphical FTP client (any will do)

Ubuntu's built-in file manager can connect to FTP servers. If you desire a standalone client, there's Filezilla.

> 6.       GUI application to replace winrar that is as simple to use as winrar without configuring a thing

When you install the "unrar" package, the default File Roller archive manager can deal with RAR archives.

> 8.       Something like Acronis True Image (as stable as Linux may be my 1tb hdd just died and hard disks are being made cheaper and cheaper, it’s hard to find a good hard drive these days)

Clonezilla comes highly recommended.


> 1.       Media Center replacement, I know there are many but I want something that works as smoothly with as little input as possible from me that works with all the codecs I have on my computer.

Boxee is multiple shades of awesome. There's also XBMC and Moovida.

> 3.       iTunes-like application that can sync with Android devices (I don’t have one in Windows but this would make me smile)

Banshee would be the closest - it will come preinstalled in Ubuntu 11.04, or can be installed in earlier versions of Ubuntu. I don't know how many Android handsets can be music-synced with Banshee, but these things can be loaded with music via drag 'n' drop anyway.

> 6.       DVD auth tool… something like iDVD or Windows DVD maker but way more powerful, I know nothing as good as sony vegas tools exists so I won’t ask for too much. DeVeDe kinda sucks.

ManDVD or Tovid.

> 7.       Video editing tool as awesome as Windows Live Movie Editor, this one borders on knocking sony in the teeth for video editing and simplicity.

Kdenlive or Openshot.

> I have been dualbooting for a while now and I will leave my application chart to this very minimal list.

Wow, I'd hate to see the full list :-P
