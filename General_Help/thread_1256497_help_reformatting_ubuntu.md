### Post by pastalavista on 2009-09-02
I would suggest that before trying to dual boot, you first complete all updates of Windows and anti-virus programs etc. and then defrag your hard disk a couple of times. Also be sure there aren't any programs hanging when you shut it down. Windows can 'lock' the HD when it's 'in use'.

Then, when you boot the Ubuntu disk, boot it to the "try without changes" desktop, go to System > Administration > Partition Editor and shrink the Windows partition. Then install Ubuntu from the icon on the desktop.

edit: 

If you aren't using Windows or want to get rid of it, right-click on the partition and delete it.
