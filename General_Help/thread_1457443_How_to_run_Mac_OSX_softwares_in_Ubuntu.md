[COLOR="DarkOrange"][FONT="Trebuchet MS"]FYI: a more common ground would be to [FONT="Comic Sans MS"][COLOR="Sienna"]use a Java APPLICATION[/COLOR][/FONT] which can run on either OS without any special setup, other than having Java already installed.

### Post by srs5694 on 2010-04-18
Mac-on-Linux is intended for PowerPC CPUs only; it won't work on x86-based computers. That said, some people have had success running OS X inside emulators such as VMware, and it's possible to run OS X in a dual-boot configuration on commodity PC hardware. (Try Googling "Hackintosh" or "OSx86" for the various sites that cover this topic.)

Also, OS X is a Unix-like OS, just with its own unique GUI and a few unusual twists in configuration, layout, etc. Chances are you're looking to run GUI programs, but text-mode OS X programs are mostly just normal Unix programs, most of which compile and run fine in Linux. You'd need to post details about what programs you want to run to get feedback about whether they might fall into this category.
