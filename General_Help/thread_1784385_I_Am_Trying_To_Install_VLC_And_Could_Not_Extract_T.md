### Post by snowpine on 2011-06-16
The easiest way to download and install VLC is using the Software Center or the following terminal commands:

```
sudo apt-get update
sudo apt-get install vlc
```

Downloading .zip installers is the "Windows way"; we don't need to do that in Linux (thank goodness)!
