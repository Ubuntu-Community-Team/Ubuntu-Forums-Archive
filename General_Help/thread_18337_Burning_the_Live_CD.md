### Post by ioliver on 2005-03-06
No, you don't want to burn the ISO file to the CD in the same way that you'd put any other file on a CD.

You need to use a CD burning application that supports burning an ISO file to CD. Most of them do if you hunt through the menus/modes.

The ISO is a pre-built CD image with everything in it that the burning software needs to make the disc.

Ian
