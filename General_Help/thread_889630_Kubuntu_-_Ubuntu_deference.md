### Post by theyranos on 2008-08-14
We'd have to see the commands to be sure, but the answer is very probably yes. Primarily, the difference between kubuntu and ubuntu is that the former uses KDE as its main desktop environment, and the latter uses Gnome. The base system is more or less the same. So, unless your commands depend on something gnome-specific, you'll be fine.

### Post by theyranos on 2008-08-15
Most of these will work exactly the way you expect them to in Kubuntu. Some of the menus are in different places (for example, the settings for allowing root login, enabling autologin, and autorunning programs are all going to be someplace different in KDE's systemsettings application).

Your MSN Messenger program will still work, but you might consider giving Kopete a try, as it integrates nicely with KDE.

All of the others look okay.

Also, just in case you don't already know, you can have Kubuntu and Ubuntu on the same system if you're just interested in trying it:```
sudo apt-get install kubuntu-desktop
```

**Edit:** *Actually, unless you're short on hard drive space or bandwidth for updates, you can keep both ubuntu-desktop and kubuntu-desktop installed (you can even throw in xubuntu-desktop and fluxbox if you want) without causing any problems.*
