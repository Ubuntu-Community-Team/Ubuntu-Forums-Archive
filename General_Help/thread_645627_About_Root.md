### Post by rsambuca on 2007-12-20
If you want to clear your Firefox cache, open your /home folder in a file browser.  Click on "View - Show Hidden Files".  Go to your .mozilla folder, then firefox folder.  Inside there is a random folder for your profile (it will be a bunch of random letters and/or numbers).  Inside there is another folder called cache.  You can delete those files if you wish.
