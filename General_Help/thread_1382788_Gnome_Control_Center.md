### Post by ajgreeny on 2010-01-16
Try the command ```
gnome-control-center
``` in the run box (Alt+F2) or in terminal.  It is certainly there in jaunty, and I think is still available in karmic, though I am on jaunty at the moment so can't check it for you.

I think from memory it is not shown in System menus by default, but you can always enable it to show by right clicking in the menu bar and editing the menu.

### Post by Tnoty on 2010-01-16
Control Center is a part of       Gnome desktop environment; it is included in       control-center package. You may start the Control Center 	one of three ways. 
2. Type the 	command **gnomecc** at the command line 	prompt. (Try this first)
2. To launch the Control 	Center without any particular active capplet 	select the **Control Center** menu item 	from the **Settings** submenu in the 	**Main Menu** or 

       	3. If you know which capplet you would like to edit you may start 	that capplet by selecting the correct menu item in the 	**Settings** menu.
