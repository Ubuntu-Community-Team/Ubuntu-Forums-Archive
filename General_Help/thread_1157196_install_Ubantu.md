### Post by jimbob on 2009-05-12
Use the Vista partitioner to do the partitioning.  There have been reports of problems with doing it in Ubuntu.  Ubuntu will run in 5Gb but I recommend at least 10Gb.

How much space required is dependent on the operator.  Usually, 10-20GB is ok to run and grow the system.  However, this does not include files (media, pics, etc) that the operator puts into the system.  A good idea is to run the liveCD to test out ubuntu vis-a-vis your specs and then dual boot as you learn ubuntu.
