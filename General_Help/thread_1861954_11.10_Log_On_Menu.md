### Post by BlinkinCat on 2011-10-22
Hi all,

I have found the answer following a question on Stack Exchange.

If you don't have "Recovery Console" or "User Defined Session" on your logon menu you can install them by loading the "gdm" package from the Software Center and then selecting "lightdm" as the default login manager. 

Maybe this will be of help to others,
Cheers - :)
