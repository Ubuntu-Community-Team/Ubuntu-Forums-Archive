### Post by meilin on 2013-08-16
When you install an app from Ubuntu Software Center, it downloads the DEB packages and dependencies and caches them in **/var/cache/apt/archives. **So I think you can backup these DEB packages and later install them offline. Via: [http://ubuntuhandbook.org/index.php/2013/07/free-up-disk-space-by-removing-apt-cache/](http://ubuntuhandbook.org/index.php/2013/07/free-up-disk-space-by-removing-apt-cache/)
