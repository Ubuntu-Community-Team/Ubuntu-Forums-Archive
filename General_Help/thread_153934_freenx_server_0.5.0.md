```
Can I connect to the same session from two different machines, without killing running programs (e.g. to use the same programs from home and work)?

Yes. Just log in using NX on *both* machines (the local and the remote machine). When you switch between machines, just take over the session. (You will need at least an 0.5.0 snapshot of FreeNX server for this to work.)

Note that you cannot currently be connected to the same session from both machines simultaneously.

Is support planned for NX clients to connect to already-running local desktop sessions, or to have two NX clients connected to the same session simultaneously, for remote support purposes?

Yes.
```
