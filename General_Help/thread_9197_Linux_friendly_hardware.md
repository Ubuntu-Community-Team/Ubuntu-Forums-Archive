### Post by Wertigon on 2004-12-26
Video: Generally speaking, nVidia is better, although some cards from ATi (like Radeon 9200) has the advantage of Open Source drivers, which makes it much easier to work with. Especially if you want the latest and greatest; sometimes ATi/nVidia aren't quick enough to get their drivers out. ;)

Network: Well, if I put it like this; I have yet to see a network card that doesn't have Linux support.

Sound: Anything that works with ALSA.

Modem (dial-up): External ones. ;)

Printer: HP. Definitely HP.

### Post by LB06 on 2004-12-27
A note on wireless LAN, if applicable: I suggest you get a wlan card with a prism chipset, because they have open source drivers in the kernel and therefore will probably work right out of the box. 

As a graphics card, I would probably buy an nVidia. Their Linux/BSD driver support in still way ahead of ATI's (although ATI's catching up). A sidenote on video cards: If you want some serious 3D performance you're probably going to need 3rd party drivers, but I think this won't be a problem in Ubuntu, because Ubuntu has packges for those drivers, as do most distro's these days.
