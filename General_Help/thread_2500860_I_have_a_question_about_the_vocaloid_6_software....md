### Post by Holger_Gehrke on 2024-09-14
Since Vocaloid 6 is a program for Windows and Mac but not for Linux, and Wine - according to the Wine appdb - can't run it, you're entirely in the wrong place to ask that question.

And from reading the specs and capabilities of the program on it's homepage I'd say there's no translation feature included; there are voice banks included which can be used with multiple languages, but any translation has to be done by yourself. Vocaloid always has been a singing Voice Synthesizer - you provide a text and a melody, the program produces a voice singing that text to that melody - and the 'AI' features of version 6 are aimed at producing a more natural sounding voice.
