### Post by Atamisk on 2011-10-18
Did you try giving it what it wants?

The keyring is an application that saves your stored passwords from apps like Firefox, Empathy, etc. Usually, the password the keyring is asking for is your login password. However, if this was the case, i don't think it'd bug you for a password, so the keyring password might be different for some reason. Did you set a password when you installed Ubuntu?
