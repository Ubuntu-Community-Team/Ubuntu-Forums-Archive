### Post by BigCityCat on 2009-01-15
Let me give you some real help with themes from a guy that knew very little not too long ago and I will tell you. You can't get help like this very often.

Yes, go into add remove or synaptic and search for gtk 2. Once installed go to this website 

[http://www.gnome-look.org/index.php?xsortmode=high&logpage=0&xcontentmode=170x171x172x173x174x175x176x177x178x179&page=16](http://www.gnome-look.org/index.php?xsortmode=high&logpage=0&xcontentmode=170x171x172x173x174x175x176x177x178x179&page=16) 

It's the best one. On the left side of the screen you will see gtk 2 themes. Do highest rated. Find the ones you like. Download the packages to your desktop. Right click on packages and do extract here. Hit your start menu and open appearance settings. Drag and drop the files you extracted on your desktop into the appearance manager. Sometimes the files have multiple themes and you have to open them and then drag and drop, and sometimes some themes just will not work, but most will. Slickness is a real good one. After you have draged and dropped. Hit start and go to system tools and select gtk manager. Select the theme you want to use and you will see all your controls and panel change. Then go back to appearance and hit customize. Scroll around and find the borders and controls that match the theme you selected in gtk. 

Drag and drop for icons as well.

If you have compiz installed you need the compiz icon. If you don't have it, go into synaptic and type in ccsm. Install everything connected with it. After it is installed hit start, go to system tools and select the compiz icon. It should appear on your panel somewhere. Right click on it. go to select window decorator. Select gtk. Then right click on it again select window manager. Select compiz. Select it again. Go to the top and select settings manager. This is how you set up the cube.

This website is the best for describing how to set up the cube. [http://forlong.blogage.de/entries/2007/8/29/How-to-set-up-Compiz-Fusion](http://forlong.blogage.de/entries/2007/8/29/How-to-set-up-Compiz-Fusion)

Good luck. Once you get it you will love it.

### Post by gettinoriginal on 2009-01-15
I am going to try to simplify the instructions from bigcity:

CCSM = CompizConfig Settings Manager (Synaptic) Allows Compiz Settings and after installation will appear in System > Preferences

Compiz Fusion Icon (Synaptic) is for setting your window decorator ie: gtk or emerald, and window manager ie: compiz or metacity. After installation, it shows up in Applications > System Tools, click on it and it puts a blue laucher in your notification area, right click and make your selections.

While you are in Synaptic, you might as well install Simple CompizConfig Settings Manager which allows general settings of animations, screen edges, desktop (workspaces), ect.

### Post by BigCityCat on 2009-01-16
well you would have to install some other type of theme engine and then you should be able to use the compiz icon to select your default theme engine by right clicking on the icon and selecting window decorator and then selecting the one you want to use as default. They generaly all work together and you just have to manipulate the different aspects of the themes until you get something you like.

Like right now. I have blue joy theme selected in GTK. Then in my appearance I have blue space controls with blended crystal icons. The border doesn't matter because I am using emerald theme manager for my borders with vrunner and you can manipulate the colors and transparency with emerald.[IMG]http://i276.photobucket.com/albums/kk14/bigcitycat/13.png[/IMG] 

[IMG]http://i276.photobucket.com/albums/kk14/bigcitycat/mine.png[/IMG]

If you don't have emerald it is part of compiz. Just search for it and install it in synaptic.

As far as the GTK engine message. I think it just gives you that message in your appearance manager if you have a theme selected in your appearnce manager that is a gtk theme(it will still work), and the appearance manager does not recognize if you have GTK installed, so it just assumes that you don't even if you do. If you have GTK installed and you have your themes selected and you see that it makes the changes when you select your themes in gtk. You can be assured that it is working.

Another thing you should do if you don't already know. You can go into firefox and select tools, select add-ons, then select get add-ons, then select browse all add ons. Then in the firefox search box on the popup web page type linux and then with the box to the right of that select themes and hit enter. That will give you about 50 or so browser themes for fire fox. Just look for the ones you like and install.

One other thing is. Transparency, you were saying for your dock and menus were not working. The way to fix that is, go into compiz settings manager. Select the opacity brightness and saturation tab. Under opacity tab select add new. Write in dock, select your transparency ratio and close. Do the same for "popupmenu" and "dropdownmenu" select your values and you will have transparency without having to use it for the entire browser.
