### Post by lisati on 2011-02-10
For most people, particularly those with dynamic IP addresses, this kind of message usually indicates that another user of the same ISP as you has been suspected of getting up to mischief.

If you're on a static IP address and have been for a while, it might be a good idea to check what the users of your local network have been up to.

There are a number of places where you can check the chances of your IP address being blocked: [blacklistalert.org]("http://blacklistalert.org") [whatismyipaddress]("http://whatismyipaddress.com") and [debouncer.com]("http://debouncer.com") to name a couple. Good sites provide links to help you troubleshoot any problems detected.

I'd be remiss if I didn't mention my own contribution: [http://lisati.homelinux.com/dnsbl-check](http://lisati.homelinux.com/dnsbl-check)
