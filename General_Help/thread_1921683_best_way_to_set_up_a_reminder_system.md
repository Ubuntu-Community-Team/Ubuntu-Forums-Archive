### Post by Quarkrad on 2012-02-07
I found using Thunderbird as my email client useful - you can set up events within it (using the calendar function) and tell it what notice to give you.  So you could set up a birthday at some time and tell it to remind you so many days before hand.  Providing you check your mail daily this is quite good.
