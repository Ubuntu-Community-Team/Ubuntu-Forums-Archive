### Post by Bigtime_Scrub on 2008-05-08
I dont know if this should go here or not. Its more of a trick/tip then anything else.

Ever want Ubuntu but don't have a CD on hand? Ever trying to save a broken system only you left all your Linux Cd's at home? Well you're in luck.

If you have an Ipod, a computer, and, the IPod USB cable you can install Ubuntu.

I. Make an Ubuntu install .ISO image. You can d/l from a torrent site or rip it off the CD you used when you first installed Ubuntu.

II. I use amaroK. Install it. You will need it. Create a new folder on your desktop and name it Ubuntu (or whatever you want to name it). Move the iso image of ubuntu into that folder. 

III. From amaroK, select Music->Import File. Hit ok and sit back. This should take several min. 

IV. You can disconnect the ipod now and take it to whatever computer you want to try this on. Connect the Ipod to the USB port and reboot the computer. You will need to select the boot order from the BIOS to say boot from "Ipod" or boot from "USB" or something to that nature. 

V. If done correctly your Ipod should say "Please Restore via ITunes. when you turn it on. 

*****CAVEAT******

Back up all your data from the Ipod before trying this. It erased everything after I tried this including the Ubuntu iso, so its a one hit quit. YOu will have to start all over again when you do this.
