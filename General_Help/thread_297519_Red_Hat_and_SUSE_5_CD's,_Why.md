### Post by taurus on 2006-11-11
Fedora Core IS RedHat.  Redhat Linux went up to version 9 and they decided to turn the whole project over to Linux community and the name then became Fedora Core.  Fedora Core 6 came out a couple days ahead of Ubuntu Edgy Eft.  RedHat is concentrated in the enterprise, server (RedHat Linux Enterprise), now.

So if you want to install RedHat, install Fedora Core.  And there is a forum for that if you want more info.
