### Post by themusicalduck on 2010-06-12
Did you install it with a .deb file? Open Synaptic (System > Administration > Synaptic) and select Local on the list on the left side. Search for the application, right click on it and select Mark for removal (or complete removal if you want to remove any configuration files for the program as well).

Then click Apply.

You could also search for it under Installed in the software centre I think.
