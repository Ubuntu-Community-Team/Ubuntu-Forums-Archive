### Post by munkymonkjr on 2005-10-24
So yesterday I erased the windows partition and reinstalled Ubuntu Breezy. To get all the basics down I ran automatix which saves so much time on configuration. Then I went ahead and removed restricted modules, which disabled my wireless (atheros...needs madwifi). Next thing is I rebuilt a custom kernel that supports Suspend2, then I made fglrx installed. The problem with FGLRX is that i need the latest version 8.6.20(?) because I have a samsung LCD that doesn't work right with the older drivers (resolution issues). That said, I moved on and tried to build madwifi for the new kernel, but it kept failing, so i said screw it and went with ndiswrapper which got my wireless up in no time. So now my lappy is complete. Hell, for the gaming points of things I even installed Americas Army and will install cadega from cvs a bit later. 

Can't say its not problematic, in fact I think most of you had the ati alarm go off in your head. That is the problem, fglrx does not support suspend so i can only do 1 of 2 things but not both at the same time. i have to choose if i want FGLRX OR if i want to SUSPEND

regardless, doing good. 

In fact, I now have no windows in my home at all. Except for my laptop, there is also a desktop that has been linux-only for almost a year now (first Vida 1.1, and now Mepis 2005.1)
