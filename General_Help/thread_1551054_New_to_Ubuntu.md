### Post by linux18 on 2010-08-11
OK, one problem at a time

-learn the basics here: [http://ubuntu-manual.org/](http://ubuntu-manual.org/)

try not to read too much at once linux can be very addictive

-as for the terminal, it has its own place: [https://help.ubuntu.com/community/UsingTheTerminal](https://help.ubuntu.com/community/UsingTheTerminal)

that is a brief overview, bash the terminal command parser has its own scripting language which is also very addictive

-as for security, as long as you only install packages from the repositories, you will never get a virus ever no matter what you download

also forget about defragging, drivers and driver updates (excluding special drivers), rebooting, and AOL/yahoo toolbars

one last thing, prepare to spend a lot of time voluntarily tweaking ubuntu, as I have said, it is very addictive and has unlimited customization, just running simple cache dropping commands will be fun:

```
 
sudo -s
sync && echo 3 | sudo tee /proc/sys/vm/drop_caches
exit 
```

if the output of the second command is 3 you just dropped your RAM cache, YAY!
