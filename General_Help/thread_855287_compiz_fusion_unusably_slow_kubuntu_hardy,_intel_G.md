### Post by sayakb on 2008-07-10
Open CCSM (System->Preferences->Advanced Desktop Effects settings) and **disable** the **Bicubic Filter** plugin. Then restart X (Ctrl+Alt+Backspace)
If you dont have CCSM:
```
sudo apt-get install compizconfig-settings-manager
```
