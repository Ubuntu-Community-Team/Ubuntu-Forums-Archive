### Post by GrumpySmurf on 2005-06-11
You'll need to 'mount' the Windows drives, or 'filesystems' under Ubuntu.  The default filesystem type on WIndows XP is NTFS, which Ubuntu supports by default.  There's an excellent guide on how to do this at the Unofficial Ubuntu Guide site:

[http://ubuntuguide.org/#windows](http://ubuntuguide.org/#windows)

I suggest reading the entire guide, or at least the parts that interest you, from the top:

[http://ubuntuguide.org/](http://ubuntuguide.org/)
