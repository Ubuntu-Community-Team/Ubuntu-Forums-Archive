### Post by HPD2 on 2010-08-16
It wont cause any problems, I have it set up the exact same way. My primary HDD is windows, for gaming, and my secondary HDD is Ubuntu. Each drive has it's own MBR so they are independent of each other. A failure of one wont cause issues for the other one.

When I want to load into Ubuntu during post I tell the bios that I want to select what boots.
The the bios then gives me the option of selecting my 1st HDD, CD/DVD rom, 2nd HDD, and USB.
