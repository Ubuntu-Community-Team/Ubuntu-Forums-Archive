### Post by rainwalker on 2008-07-01
On Linux, you usually install programs with packags (end with ".deb"). With FrostWire, you should have been able to get the .deb straight from their site and installed it with no problem.
Otherwise, you install stuff from repositories, which are basically big storage centers for tons of stuff. For example, if I didn't have Firefox installed I could go to System > Administration > Synaptic Package Manager and search for "firefox" and install the package. And boom, it would handle all of the dependencies and everything :)
A lot of programs have their own repositories that they update, even if the program is in the Ubuntu ones (usually because the versions in the Ubuntu repos can sometimes be a little behind), so if you ever have to add their repository it can be done in the System > Administration > Software Sources window.

To remove/uninstall stuff you just remove the package, and that's it
