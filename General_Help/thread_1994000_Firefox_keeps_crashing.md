### Post by Paddy Landau on 2012-06-03
[LIST]
[*]Leave Firefox closed.
[*] Install [ubuntu-restricted-extras]("apt:ubuntu-restricted-extras") again.
[*] Open Firefox.
[*] Go to Add-ons and install [Flash-Aid]("https://addons.mozilla.org/firefox/addon/flash-aid/") (by our own [lovinglinux]("http://ubuntuforums.org/member.php?u=649167")).
[*]Restart Firefox.
[*]Tools > Flash-Aid > Wizard Mode.
[*]You may need to restart Firefox.
[/LIST]
Let us know if this helps.
