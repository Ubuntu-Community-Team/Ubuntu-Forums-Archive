### Post by aysiu on 2007-09-02
If you install *kubuntu-desktop* on an existing Ubuntu installation, you'll have both Ubuntu and Kubuntu installed. If you install only Kubuntu, you'll have only Kubuntu (not Ubuntu packages).

### Post by aysiu on 2007-09-02
You *could* dual boot Kubuntu and Ubuntu on the same PC, but you can also have both available on the same (single-boot) installation.

**Advantages of Single-Booting**
[list]
[*]Save on hard drive space
[*]No need to reboot your computer to switch desktop environments
[*]Easier to remove KDE or Gnome later, should you wish to do so
[/list]

**Advantage of Dual-Booting**
[list]
[*]"Clean" KDE and Gnome. No mixing of menus. No themes "taking over" other thremes. No stray .desktop file in Gnome after using KDE.
[/list]

By the way, if you dual-boot and use the same /home partition and user account for Ubuntu and Kubuntu, the advantage of dual-booting will pretty much be lost. In fact, not only will things be mixed, but you also have a lot of empty menu entries that appear or settings that don't work correctly.
