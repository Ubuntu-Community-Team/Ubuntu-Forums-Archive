### Post by 454redhawk on 2007-03-11
As you can see from your screen shot it has 4 connections open and its doing exactly what its supposed to.

1.It first sees what parts are available
2. It then finds the PAR2 files and skips them.
3. it downloads all the proper parts.
4. It then verifys the files and downloads the PAR2 files if repair is needed.
5. then it UNrar or unzips the files and deletes the parts when finished.
6. A fully assembled and complete file is stored in the NZB processed directory ready for burning.
