### Post by Jay_Bowles on 2017-04-20
Have you thought about running windows in a virtual environment within *buntu?? Vitualbox or equivalent will let you do this. This way you will be able to run any apps that are only provided for that OS. Alternatively you could dual boot (which is what I do) Windows and Linux so that you can run either natively and take your time to acclimatise to the new operating system.

 Having said this Linux does provide many very good alternatives to windows apps and extra gaming can come from steam. It's really all about what you want to do and how much you are willing to learn (it really isn't that hard) about your new shiny Linux OS. Remember Linux isn't Windows and although you can tweak it to look similar (or completely different) the two do not work the same so most windows apps will not work.

Good luck with your adventures.
