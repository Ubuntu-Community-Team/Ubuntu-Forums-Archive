### Post by carusoswi on 2009-09-07
Getting this scanner to work in Ubuntu is, perhaps, the last of my Windows --> Ubuntu conquests.  Nothing I've tried previously has allowed it to work in Ubuntu (I'm currently running 9.04).

This morning, as has been my habit over the years, I googled my scanner and linux, and today was my lucky day.  I came across a site that offered to allow me to download a driver that worked.  I completed the download, extracted the file, but it turned out to be an html which took me to another site (or another section of the original site, I don't know which).

At this location was an actual driver posted for Ubuntu 8.04 and later.

I downloaded that file which brought up the package installer, installed the file which placed a new application on my system, Image Scan, which, if I recall, is a Linux version of the Windows program that shipped with my scanner (way back when).

Image Scan in Ubuntu seems to work just as it did in Windows, and supports all the functions of my scanner (settings to select positive or negative film, set the source for the flatbed or film holders, etc.).

Best of all, in really works.

If, like me, you've been forced to switch back to Windows whenever you needed to scan something on your Epson Perfection 2400, check out the following site:

[http://www.avasys.jp/lx-bin2/linux_e/scan/DL2.do](http://www.avasys.jp/lx-bin2/linux_e/scan/DL2.do)


If this post is a duplication or I've placed it in the wrong area of the forum, please accept my apologies.  Lord knows I've been searching the forum for a long time without finding a clean solution such as can be obtained at the referenced site.

Thanks, and good luck.

Caruso

### Post by carusoswi on 2009-09-07
This driver isn't quite as slick in Linux as it is in Windows.  For instance, in Windows, you get a thumbnail view of four slides when you preview.  You can rotate them, select all or any combination of the four for scanning.

The linux version only gives you a view of the four slides (the preview is more like the preview of the entire underside of the slide tray).  You have to manually select the area you want to scan, which means you have to select each slide separately and scan it one at a time.

I suppose you could scan the entire strip and then crop afterward, but my guess is that you would be compromising resolution in that case - perhaps not.

You cannot rotate ahead of time,either.

But those are minor inconveniences compared with having to switch into Windows to scan photos.

Caruso
