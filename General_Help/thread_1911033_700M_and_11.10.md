### Post by 3rdalbum on 2012-01-18
Onboard Intel graphics do usually support Compiz, which means that they support Unity 3D.

If your graphics isn't able to run Unity 3D, Ubuntu automatically switches to Unity 2D. It's very, very similar to Unity 3D and it doesn't require any video acceleration at all.

So, in short: The answer is "You can run Ubuntu 11.10".
