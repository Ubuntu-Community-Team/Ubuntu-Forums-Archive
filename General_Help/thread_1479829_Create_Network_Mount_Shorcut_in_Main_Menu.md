Right, again I post the solution to my own problem. In the File Browser click on the folder once -> choose Bookmarks on the menu above -> add bookmark! The folder will now appear in the Files & Folders menu in the main menu. As far as I can tell in is **not** possible to add a network folder to the favorites menu. This is possible with regular folders using drag and drop (drag and drop the folder in the file browser to the Ubuntu symbol in the top left corner of the screen).
