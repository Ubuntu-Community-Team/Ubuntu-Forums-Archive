### Post by Slorg on 2011-04-20
I had the exact same problem, you're probably using 64-bit, aren't you?
In that case, here's the solution:

Download the 64-bit flash, here's a link with how-to:
[https://help.ubuntu.com/community/AMD64/FirefoxAndPlugins](https://help.ubuntu.com/community/AMD64/FirefoxAndPlugins)

It fixes the whole problem, it kinda drove me nuts when I used Firefox 4. (I even started using Chromium)
