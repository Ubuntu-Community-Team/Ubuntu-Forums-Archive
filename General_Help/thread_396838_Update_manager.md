### Post by dpar on 2007-03-29
When the update manager notifies me of updates, I sometimes uncheck some of the updates that I don't want. The problem is that update manager keeps informing me that there are updates unless I download the ones that I don't want. Is there any way to tell it that I REALLY don't want that update???

---

### Post by the.phantom on 2007-03-30
you can use synaptic to do that

just click on the item, package and select "lock version"

it will now show up on the left side as "penned" and you can change the status if you wanted to
