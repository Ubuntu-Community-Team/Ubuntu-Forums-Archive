### Post by tgalati4 on 2013-06-24
Look at how the other lightweight distros handle graphics:  slitaz, tinycore, puppy linux.  There are several tricks that are used, but these tricks can severely limit the graphics driver ability to work with certain resolutions or certain graphics cards.  So if you find something that works, great.  But if you get low resolution (800 by 600) or flickering then you might not be able to use the lightweight graphics server.
