### Post by mister_playboy on 2010-11-09
unrar interfaces with both file-roller and ark so that you can almost always open .rars from the GUI if you wish.  I say almost because it is my experience that some passworded .rars cannot be opened in the GUIs, and then you have to use the CLI.
