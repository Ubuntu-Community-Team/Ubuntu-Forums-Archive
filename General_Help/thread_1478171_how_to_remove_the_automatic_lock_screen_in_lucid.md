### Post by cogier on 2010-05-09
If you go to **System>Preferences>Screensaver** and uncheck the "Lock screensaver when computer is idle" box that should sort it.

Go to "System --> Preferences --> Screensaver". Then, uncheck "Lock screen when screensaver is active".

Not sure if there is a way to extend the time before it locks other then extending how long the computer has to be idle for the screensaver to kick on.
