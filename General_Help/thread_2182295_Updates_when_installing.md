### Post by Bashing-om on 2013-10-20
brachra2055; Hi Welcome to the forum.  
I suggest that you do these updates from the command line, in the event there are any errors or complications these will be generated verbosely on the terminal.
from the desk top the key combo ctl+alt+t will yield a terminal, input terminal codes:
```

sudo apt-get update
sudo apt-get upgrade
sudo apt-get install ubuntu-restricted-extras

```
that will complete the basic install, and may be all that you will ever need or want.
