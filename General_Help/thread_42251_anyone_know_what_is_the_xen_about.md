### Post by Xian on 2005-06-16
The weblink to the Xen homepage is below and there are further links at that page which will describe the project, how it is installed, and what needs to be downloaded. It basically allows you to run multiple nix OS's within a single environment. In the fall it is planned to release a version which will also support Windows just like VMware and Win4Lin do at present, but with a somewhat different technology and implementation.

[Xen Virtual Monitor](http://www.cl.cam.ac.uk/Research/SRG/netos/xen/)

Edit: I just found another great resource for Xen: [Xen Wiki](http://wiki.xensource.com/xenwiki/)
