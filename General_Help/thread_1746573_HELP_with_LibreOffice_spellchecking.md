### Post by lec0rsaire on 2011-05-04
Opening a terminal screen and type "sudo apt-get install language-support-en" without the quotes.That should fix your problem. My problem is not that spell check isn't working. It's that grammar checking is nonexistent. I purposely wrote run-on sentences, subject-verb disagreements and it picked up nothing. This makes it unusable for me unless there is a solution. In Word 2007, the checker picks up nearly everything except when I'm working in German. It's sad because I would like to use Linux full-time but these little things are forcing me to continue to dual-boot.
