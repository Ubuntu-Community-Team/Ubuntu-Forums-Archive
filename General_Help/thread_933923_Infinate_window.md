### Post by compnut41 on 2008-09-29
I just happened to stumble on one of the coolest Compiz Fusion effects ever.

What you do is set up 3D windows, and the desktop cube. Then rotate the cube so the windows come up all the way. Then switch to the unfold cube effect. I think the default is Ctrl Alt down key. 

The result is one of your windows extending infinitely into the background.
