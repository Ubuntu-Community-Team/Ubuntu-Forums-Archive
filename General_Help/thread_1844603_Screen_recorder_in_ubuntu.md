### Post by JoshuaMiller0 on 2011-10-06
> **Denver Dave said:**
> 
Maybe I'm doing something wrong, but I've never been able to copy anything to the terminal screen.

In terminal the default shortcut key to copy and to paste is:

For copy (one of the two):
crtl+shift+C
crtl+alt+C

For paste (one of the two):
crtl+shift+V
crtl+alt+V

You can change this in preferences if you like or you can use these commands or you can right-click and click copy/ paste.

Hope I helped! :)
