### Post by drs305 on 2010-11-14
If you are seeing a black screen with an "Busy Box" and  "(initramfs)" prompt it means your boot failed. There are a variety of reasons, many associated with Grub not passing the correct parameters.

If you can boot a LiveCD, go to the following site and download and run the boot info script. Post the contents of the generated RESULTS.txt file and we may be able to see what is causing the failure.
[http://bootinfoscript.sourceforge.net]("http://bootinfoscript.sourceforge.net")
