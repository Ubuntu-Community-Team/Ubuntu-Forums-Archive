### Post by Original Brownster on 2005-11-27
Hi,
Things to check:

In your bios you can choose the boot order, make sure the cd-rom is the _first_ device to boot from.

Check the md5 sums for the images you have downloaded to ensure they are not corrupted in anyway.

Make sure that when you burn the disk, you do not just drop the iso image onto the disk as a file as this will not create a bootable disk.
You must choose to create a disk from an iso image in your particular software. Sorry if you have done this but I know it's possible to make this mistake.

HTH
