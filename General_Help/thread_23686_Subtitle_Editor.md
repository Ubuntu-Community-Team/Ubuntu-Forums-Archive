### Post by empinchen on 2005-04-22
I successfully got ksubtile [http://ksubtile.sourceforge.net/](http://ksubtile.sourceforge.net/) to compile and run under Hoary.

To compile, you'll need the usual development packages plus libqt3-headers and libqt3-compat-headers

(I'm running kubuntu, if you don't,  you might need some KDE packages too)


It uses mplayer to show vids, so you might want to apt-get that too. (Or even build it yourself)
