### Post by SuperFreak on 2014-08-20
I thought this might be useful for those new to Ubuntu. Granted it is personal preference and there are lists like this on the web

**VLC** - Best all arround video, music player [http://www.videolan.org/vlc]("http://www.videolan.org/vlc")
**K3b** - Best all round software for ripping , burning movies, data, music [http://www.k3b.org/]("http://www.k3b.org/")
**Systemback** - great for system restores and for creating a LIVE Disc or USB that is unique to you [http://sourceforge.net/projects/systemback/]("http://sourceforge.net/projects/systemback/")
**GParted** - GUI Partition Editor -in Software Centre
**Grsync** - GUI Back up Utility - in Software Centre
**GIMP** - Graphic editor - in Software Centre
**X -Sane** - Scanner Utility - in Software Centre
**WINE** - Windows Compatibility layer - in Software Centre
**Boot Repair** - Repairs boot problems for UEFI, BIOS, Diual Boot, Single boot and other situations [https://help.ubuntu.com/community/Boot-Repair]("https://help.ubuntu.com/community/Boot-Repair")
**SAMBA** - Connect PCs over a network and share files & Printers - in Software Centre
