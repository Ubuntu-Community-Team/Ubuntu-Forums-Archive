### Post by zero2xiii on 2013-03-18
Hay,

+1 for the "translate to English please".

However, my (Spanish? I think??) Is not that great.. However, that screen is remarkably similar to the disk check screen during boot.
This however is not an ERROR, but rather a build in program to check the drives for errors. Similar to windoz's "chkdisk".

Just allow it to complete (do not press C to skip) and it should be fine. Some times it reboots after it finishes. This happens every so often, after every 30th restart it is done automatically,
as well as after unclean shut downs.

To see the reason, press F1 once the screen comes on, and read the terminal output.

It will generally give an idea such as:
"Drive sda1 has been mounted 30 times, check forced"
or some other reason.

Cheers.
