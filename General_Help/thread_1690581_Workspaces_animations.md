### Post by seawolf167 on 2011-02-21
Everything you are looking for should be in compizconfig-settings-manager

System -> Preferences -> ConpizConfig

It should be installed by default, but if it isn't you can run this from a terminal (Applications -> Accessories -> Terminal) to install it:

```
sudo apt-get install compizconfig-settings-manager
```

I believe the item you want is either workspace preview or cube switcher (not at the correct machine to check it out atm)
