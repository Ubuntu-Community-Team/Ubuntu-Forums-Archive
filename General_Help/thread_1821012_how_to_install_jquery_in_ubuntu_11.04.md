### Post by ~!geek!~ on 2011-08-08
Jquery is not something which need to be installed. Its just a file (1.4 kB I hope) consisting of scripts which needs to be included in a javascript or php script file using the way we generally include files.
To start with jquery, you may download jquery file from internet. Put it in the directory with other php and other script files. Include this jquery file in other script files and you would be able to use the jquery api (its functions etc.) 
These steps should be sufficient for you. To learn jquery you may start with w3schools etc.
