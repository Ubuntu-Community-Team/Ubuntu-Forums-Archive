### Post by Gen Vert on 2011-04-10
For some unknown reason, the Skype login Window has started coming up after desktop (10.04).
I looked in **Startup Applications** but found no reference to Skype.

How can I stop this happening?

---

### Post by RedSingularity on 2011-04-10
A friend of mine has the same issue which had me wondering if it was a bug after a recent package update.  Try looking into the skype settings and make sure that the option to "start with windows" is NOT checked.

---

### Post by Gen Vert on 2011-05-24
I solved the problem by changing the **Login Screen Settings **to **Failsafe GNOME **(This session logs you into GNOME without user applications).
