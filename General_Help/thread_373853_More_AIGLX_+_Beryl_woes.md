### Post by mac.ryan on 2007-03-02
I solved the problem of the window frames and blank Terminal by changing the DefaultDepth to 24 in the screen section of the xorg.conf.
Also add this to options
Option "AddARGBGLXVisuals" "True"
Option "TripleBuffer" "true"
to the device section
Reply With Quote

More info in this thread:
[http://ubuntuforums.org/showthread.php?t=353488](http://ubuntuforums.org/showthread.php?t=353488)

Let us know if it worked for you.

I've never used Ubuntu Ultimate so I don't know about that question.  As for beryl, try launching beryl with this command: 'beryl-manager && emerald --replace'.  Either that or download a newer version of beryl.  That was a common bug with the earlier beryl versions, as emerald didn't properly replace the metacity window decorator, but I haven't heard about it in a while, so I would assume you have an old beryl build (0.1 or 0.2)?
