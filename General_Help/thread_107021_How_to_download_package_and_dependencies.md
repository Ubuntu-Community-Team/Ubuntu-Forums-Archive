### Post by briancurtin on 2005-12-22
if the other computer has ubuntu as well, you could try and use the internet connected ubuntu box to download something, which would get all of its dependencies if you do it in synaptic, but then you have to track it all down. you could, in theory, do this, but then the differences between the internet connected ubuntu box and the non-internet one could throw off the dependencies you think are in the non-internet one. 

all-in-all, i would say no. your best bet is to set up a little network with a router to split that connection off so that your ubuntu box can have the internet and get the proper files and dependencies. this can be done with a crossover cable and such, but i dont know how to do that, never tried.
