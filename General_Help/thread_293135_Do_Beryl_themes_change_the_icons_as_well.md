### Post by awatts on 2006-11-04
you can change the icons by using the theme manager (System->Preferences->Themes).  However make sure to have "gnome-settings-daemon" in your startup programs (System->Preferences->Sessions) or the icons will not change when logged on to your "Xgl" session.

### Post by NeoChaosX on 2006-11-04
No. Beryl is a window manager. Thus, any themes to Beryl will only affect the window borders. if you want to change the widgets or icons theme, you have to use the GNOME theme settings applet.
