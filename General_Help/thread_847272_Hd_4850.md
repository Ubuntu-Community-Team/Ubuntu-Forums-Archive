### Post by phoronix on 2008-07-02
The supportive Catalyst Linux driver can be downloaded @ [https://a248.e.akamai.net/f/674/9206/0/www2.ati.com/drivers/linux/ati-driver-installer-8-6-x86.x86_64.run](https://a248.e.akamai.net/f/674/9206/0/www2.ati.com/drivers/linux/ati-driver-installer-8-6-x86.x86_64.run)

If you want to use the open-source driver, to address those package problems you'll need the X server development packages. Here is an install guide that can be roughly followed when replacing -radeonhd with -ati @ [http://www.phoronix.com/vr.php?view=11067](http://www.phoronix.com/vr.php?view=11067)
