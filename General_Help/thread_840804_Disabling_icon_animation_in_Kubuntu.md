### Post by Zardus on 2008-06-25
When you click on a KDE icon in Kubuntu, the icon sort-of zooms in real quick in a little launch effect. This might look nice on a screen but is a pain in the *** over a slow network connection, even with NX. It wasn't this way in Dapper, but I'm not exactly sure which non-LTS release it started in since I just jumped straight to Hardy.

Is there any way to turn this off? I've turned off every effect option I could find to no effect.

---

### Post by diaa on 2008-06-25
Yes I didn't like it too and it wasn't easy to find in the control center, actually it's in a strange location.

*Peripherals* > *Mouse* > *Visual feedback on activation*

although it happens when using the keyboard too :)

### Post by Zardus on 2008-06-27
Thank you so much! Turning that dang thing off has done wonders for my sanity.
