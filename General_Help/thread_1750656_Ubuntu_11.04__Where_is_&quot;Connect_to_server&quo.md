### Post by Morbius1 on 2011-05-27
Old way: Places > Connect to server

New way: Tap the empty desktop to ensure focus. Hover over the top taskbar to reveal the items. Then select File > Connect to server.

What ironically will become the "New New Way" to explain this in the forums because the "New Way" is too long to write:

Open a Terminal and type: [COLOR=Black]nautilus-connect-server[/COLOR]
