### Post by Dennis N on 2014-10-02
> Since the computer can't read usbs as a boot option, I found a pack of 700mb disks.

Here is a tip to be able to boot from USB on your 'antique' machine.

I have a Dell 2300 (bought in 2002) P4, 1.8 ghz that also will not boot from a USB. There is a small program called Plop Boot Manager that is burned onto CD and after booting it from CD, allows you to boot from the USB. I have recently tried Plop with a live USB of Xubuntu 14.04 on that old machine, and it boots up Xubuntu 14.04, seems normal, and it is usable. I haven't done a permanent install, however - not yet.

So, you are not really limited by the size of the install media if you use Plop. Google for where to download if you are interested.
