### Post by claydoh on 2005-06-04
Qdvdauthor and [dvdstyler]("http://dvdstyler.sourceforge.net/") are 2 that come to mind as well as one called [tovid.]("http://tovid.sourceforge.net/")

[Qdvdauthor]("http://qdvdauthor.sourceforge.net/orig_index.html") has some decent how-tos on their site, but you need to have the video in the correct format (mpeg2) for either Qdvdauthor or dvdstyler to create a dvd. 

Check out[ kde-apps.org's video app section]("http://www.kde-apps.org/index.php?xcontentmode=221") for some others. 

I can't remember which program I used to convert some avi files to a dvd, but once that was done, Qdvdstyler worked well creating the menus and file structure, and k3b was used to burn it all to a dvd. Qdvdauthor is available via apt, may need to add backports to your apt sources.lst, DVDstyler has a deb somewhere, others may have deb files available or need to be compiled.
