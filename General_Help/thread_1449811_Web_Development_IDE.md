[LIST]

### Post by jeremysawesome on 2010-04-28
For Linux - Geany all of the way([http://www.geany.org/](http://www.geany.org/)). It's lightweight, has code completion and syntax highlighting. You can customize the syntax highlighting here: [http://geany.pdarko.co.cc/filetypes/index.php](http://geany.pdarko.co.cc/filetypes/index.php)

You can also bring up a window from within the editor to run command line arguments.

I use it in conjunction with Filezilla if I want to use FTP. I don't need any "project management" assistance, but it does have a File Browser that you can use to browse your project directories. Sometimes I will use SSHFS to mount a file system from my server and edit the files directly with Geany. (Rather than using FTP or ssh and vi)
