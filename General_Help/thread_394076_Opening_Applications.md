### Post by Ramses de Norre on 2007-03-26
1) Graphical programs: should be in the gnome menu, try restarting the menu if doesn't appear instantly (killall gnome-panel)

2) commandline or non-appearing graphical apps: mostly the command to start them is just their name in lower case. Otherwise you'll need to be inventive, some people give their executables weird names... Nothing to do about. Use locate to search for the files.
