### Post by meng on 2006-10-10
No. Ubuntu-desktop is a meta-package (like a dummy package), so don't worry about the fact that it is being uninstalled. Nothing useful is being removed except what you are asking for.

### Post by aysiu on 2006-10-10
No.

*ubuntu-desktop* by definition is the set of default packages. If you remove the default packages, you remove *ubuntu-desktop*.

It's like calling yourself a couple after a divorce. You're no longer a couple any more.

*ubuntu-desktop* is just a label. You can safely uninstall it.

You may want to reinstall it, though, before you upgrade to Edgy.

### Post by Jussi Kukkonen on 2006-10-10
No. The whole point of ubuntu-desktop is that it ensures you have those installed.

There's no harm in removing ubuntu-desktop though (except maybe in the case of upgrading the Operating System, when you won't get new applications that have been added to ubuntu-desktop).
