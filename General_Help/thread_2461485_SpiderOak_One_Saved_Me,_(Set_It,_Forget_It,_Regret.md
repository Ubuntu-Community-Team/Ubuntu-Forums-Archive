### Post by webmanoffesto on 2021-05-01
If you use SpiderOak One make sure that you don't end up with multiple conflicting "devices". About 3 months ago I noticed that SpiderOak One was not sync'ing correctly. I was getting "Backend Worker Process Has Exited" error messages. Fortunately I contacted SpiderOak technical support. They helped me to realize that I needed to delete "devices". I have one main laptop which I sync on SpiderOak. But each time I had reinstalled Ubuntu I got a new "device" on SpiderOak. That created problems. So I deleted all the old "devices" and kept only my current Ubuntu install as a "device". I recently had computer problems and had to do a clean install. That wiped all my data. Fortunately I was able to use SpiderOak One to get my data back.
