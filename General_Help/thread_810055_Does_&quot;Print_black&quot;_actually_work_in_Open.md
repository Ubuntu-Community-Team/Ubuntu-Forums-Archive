Seems like this is a bug in Ubuntu Hardy, and I'm guessing it's also in Kubuntu Hardy. The only fix so far is just to type everything in black and white, and if you have images, go here- 
[http://www.online-image-editor.com/](http://www.online-image-editor.com/)
And upload your image. Go to "Edit" then "Colors" then "Black and White" then save the image, and import it into Open Office. Print normally. Sorry, that bug is quite annoying... :mad:
