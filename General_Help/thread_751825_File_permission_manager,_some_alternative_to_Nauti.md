### Post by pbhj on 2008-04-10
Have you tried Konqueror (it's a file manager as well as being a browser) or it's replacement "dolphin"?

Select files, then alt-enter, alt-p then "advanced permissions" and you can choose the permissions to set in the way you describe, etc..

Also in Konqueror you can do ctrl-e and type a command line for chmod / chown.

HTH
