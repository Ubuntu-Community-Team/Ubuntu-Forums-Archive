### Post by SuperSonic4 on 2009-11-15
No worries

0. Download the Script but **do not** extract it. Leave it as is

1. In **Amarok** go to **Tools -> Scripts**

[IMG]http://img.photobucket.com/albums/v365/SuperSonic4/Screenshots/Tutorial%20pics/Amarok-Script/1.png[/IMG]


2. Then go to **Install Script** and click on it

[IMG]http://img.photobucket.com/albums/v365/SuperSonic4/Screenshots/Tutorial%20pics/Amarok-Script/2.png[/IMG]

3. Navigate to wherever you downloaded the tarball to (for me it's ~/Downloads). Single/Double click to install it

[IMG]http://img.photobucket.com/albums/v365/SuperSonic4/Screenshots/Tutorial%20pics/Amarok-Script/3.png[/IMG]

4. Restart Amarok from the system tray

[IMG]http://img.photobucket.com/albums/v365/SuperSonic4/Screenshots/Tutorial%20pics/Amarok-Script/4.png[/IMG]

5. Repeat step 1 and scroll to Radio Espanola

[IMG]http://img.photobucket.com/albums/v365/SuperSonic4/Screenshots/Tutorial%20pics/Amarok-Script/5.png[/IMG]

(et Voila - to find it)

6. Home icon -> Internet

[IMG]http://img.photobucket.com/albums/v365/SuperSonic4/Screenshots/Tutorial%20pics/Amarok-Script/6.png[/IMG]

7. Select **Radio Espanola** and pick the station to listen to as normal

[IMG]http://img.photobucket.com/albums/v365/SuperSonic4/Screenshots/Tutorial%20pics/Amarok-Script/7.png[/IMG]
