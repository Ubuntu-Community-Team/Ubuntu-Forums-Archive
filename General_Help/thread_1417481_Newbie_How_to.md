I have installed and using 9.10 for a few days. I would like to know how I can add or change the USER using the computer. I would like to have one user using with one language (English) and the other with another language and key board (German)

### Post by warp99 on 2010-02-27
This is very simple. When logged in under the German users account in System > Administration  > Language you can set the language just for that user. In System > Preferences > Keyboard you can set the keyboard for just that user. Please see my signature for some helpful sites to visit for questions like this.
