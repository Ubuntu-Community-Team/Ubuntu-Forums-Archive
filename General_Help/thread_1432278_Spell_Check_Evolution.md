### Post by blackmail on 2010-03-17
If you pay a visit to:
System>Administration>Synaptic Package Manager
Click search and type myspell, tell it to only search by name.
You can see all the packages there.

This solution is credited to [Sekinto]("http://ubuntuforums.org/member.php?u=399472")
