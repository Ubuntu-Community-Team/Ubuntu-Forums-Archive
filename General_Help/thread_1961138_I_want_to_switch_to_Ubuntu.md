### Post by HiImTye on 2012-04-19
> **mt605504 said:**
> Microsoft Word, Powerpoint, Excel and Publisher
LibreOffice is a replacement for Microsoft Office family of programs. there may appear to be 'formatting' problems but that is only because the formats you are using are not portable formats (meaning they will appear differently on different computers using Microsoft Office also)
> **mt605504 said:**
> Itunes
Banshee, Rhythmbox, MPD (my personal favorite), Exaile, etc..
> **mt605504 said:**
>  Firefox and Chrome
Firefox is the default browser and Chromium is a replacement for Chrome
> **mt605504 said:**
>  Dropbox
dropbox works on Linux. there is also UbuntuOne which comes pre-installed
> **mt605504 said:**
>  VLC
VLC has a linux client. there are better replacements such as FFMPEG for streaming and (S)MPlayer for watching
> **mt605504 said:**
>  Photoshop and Cinema 4D (Like Once a Month)
GIMP comes close to replacing Photoshop but there are certain things it does not handle well (such as palette-based conversions). these are few and far between, however, and for the average user they will be negligible. there is also Inkscape and Blender for working with graphics. for more info on the types of programs to use I would check the UbuntuStudio list
> **mt605504 said:**
>  Ive heard of wine and dont know anything about it!
WINE is hit or miss (or sometimes both) when it comes to programs that work with it. in most cases there are replacements for those programs that work natively, or, you can always use VirtualBox+Windows if you are lost without said program (for instance, accessing the iTunes store). there are always the forums here to get answers if you are unsure and WINE has a [database]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true") of programs that work and don't work (or work with some modification, or work sans some features, etc).
