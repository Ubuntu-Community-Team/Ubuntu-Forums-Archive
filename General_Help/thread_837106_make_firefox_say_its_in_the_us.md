### Post by Nepherte on 2008-06-22
I think you can do it with the User Agent Switcher plugin for firefox. Just change the App version value to something else.
My app version: 5.0 (X11; nl-BE) -> 5.0 (X11; en-US) for example

### Post by Nepherte on 2008-06-22
If you installed the User Agent Plugin, go to extra > user agent switcher > options > options. Then choose the user agents tab. You can't change the default user agent, but you can view it. Copy those values and create a new one with the same values. Then edit the App version value.
