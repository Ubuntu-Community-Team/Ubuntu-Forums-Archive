you don't need to remove previous themes using ppa, what you have to remove is any old version of the Orta theme if you have it installed before installing the new Orta one.
to remove a ppa theme look it up in synaptic package manager and mark for complete removal then apply.
to remove the ppa repository go to software sources and delete the ppa then update the package list.
and removing the ppa doesn't remove the installed themes, the themes you have to remove them one by one if you wish.
