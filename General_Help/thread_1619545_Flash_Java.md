### Post by efflandt on 2010-11-11
Use System > Administration > Synaptic, refresh the sources if you have not done that (swirling arrow icon), and when if finishes reindexing Quick Search type "restricted" there (without quotes).  Look for **ubuntu-restricted-extras**.  That installs flash, a java substitute and codecs.  Read the description of that package to see what else you need to do if you want to play DVD's.
