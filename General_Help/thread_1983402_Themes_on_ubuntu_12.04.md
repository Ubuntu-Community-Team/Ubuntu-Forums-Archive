### Post by Frogs Hair on 2012-05-20
Choice is fun !:) If you are using 12.04 Unity 3D or the Gnome Shell _use Gnome 3.4 themes only_ . When you use the link select GTK 3 themes and in the theme description it will indicate if the theme is 3.4 compatible. I currently have 38 Gnome 3.4 compatible themes because some packages include many themes. [http://gnome-look.org/](http://gnome-look.org/) 

All my installed themes work in both Unity and the Gnome Shell.

### Post by Purplerob on 2012-05-20
To get what you want install gnome3 classic using this guide:
[http://www.omgubuntu.co.uk/2012/03/gnome-classic-in-ubuntu-12-04-its-like-nothing-ever-changed/ ]("http://www.omgubuntu.co.uk/2012/03/gnome-classic-in-ubuntu-12-04-its-like-nothing-ever-changed/")
Than using alt-right click to customize the panels remove the bottom panel, and replace with cairo-dock. Then customize the top panel and add Indicator Applet to add the global menu.
