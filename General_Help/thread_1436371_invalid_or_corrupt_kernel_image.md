### Post by quixote on 2010-03-23
Is this an installed version of Ubuntu?  Or have you been trying it just from the LiveCD?  If the latter, try downloading and burning another CD, just as you said.  Burn it at the slowest speed you can, just to be on the safer side.  

I don't think it would complain about invalid kernel images because of a problem with your computer.  There are some very unusual configurations that might require kernel customizations, but an HP with a Celeron CPU doesn't sound weird :).  Ubuntu needs a minimum of 384MB system memory to run (Xubuntu and minimal installs fit in smaller spaces), so 512MB should work.
