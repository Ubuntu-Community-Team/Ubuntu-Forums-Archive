### Post by new_tolinux on 2010-05-20
Grenage is probably right.
Also depends on what he's doing with his machine.

If buying RAM keep in mind that there's a limit of about ~3.25GB that can be used by 32-bit operating systems. You can put in more (4GB) but the remaining memory just will stay unused.
No issues like that on a 64-bit OS, the motherboard is probably the first limitation there when you're adding memory.
