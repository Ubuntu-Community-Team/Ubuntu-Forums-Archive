### Post by gwpritch on 2009-10-14
I had a similar proble although it was the other way, accessing my HP printer on ubuntu from windows. 
Instead of browsing for the printer, try accessing it directly by typing \\xxx.xxx.xxx.xxx\name_of_printer in the search box, of course replacing the above with the ip address of the windows box and actuall name of the printer on the system.
