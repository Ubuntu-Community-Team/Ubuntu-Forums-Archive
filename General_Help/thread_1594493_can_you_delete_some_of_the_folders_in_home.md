You can safely delete all three of those folders as long as you aren't using them for their intended purpose:

[LIST]
[*]The **Templates** folder is for storing "Create new..." templates. For example, you could put an empty OpenOffice document in there, call it "Text Document", and then when you right click in a folder you'll be able to select "Create new Text Document". You get the idea.
[*]The **Ubuntu One** folder is used for syncing your files to the Ubuntu One cloud service. If you aren't using that service, then the folder is meaningless.
[*]The **Examples** folder is just a bunch of stuff that explains how to use various parts of the system and programs. It's outdated and frankly I don't know why they still include it.
[/LIST]
