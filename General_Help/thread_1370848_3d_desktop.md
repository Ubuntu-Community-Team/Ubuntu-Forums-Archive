### Post by Saiko Berry on 2010-01-04
Run the command ccsm & in terminal or Alt+F2.
You can also run an easy noob friendly version called simple-ccsm.

Remember, if you're using a terminal window to open it use & to detach dependancy.. although I recommend Alt+F2 and running it through there.

You can also (obviously) make a launch and throw ccsm or simple-ccsm into the command box.

Although they should be there under preferences. Try right clicking system and editing the menu to see if its just disabled.
