### Post by grahammechanical on 2014-11-10
Lightscribe is not in the Ubuntu Software Centre and the Lightscribe web sight does not offer a version packaged for Ubuntu. We can download a compressed or archived version from here.

[http://sourceforge.net/projects/qlscribe/](http://sourceforge.net/projects/qlscribe/)

Installing it is another matter. The download is a tar file. This will give some help with tar files

[https://help.ubuntu.com/community/FileCompression](https://help.ubuntu.com/community/FileCompression)

[https://help.ubuntu.com/community/File%20Roller](https://help.ubuntu.com/community/File%20Roller)

You will need to extract what is in the archive. And that is as far as I can help.

Regards.
