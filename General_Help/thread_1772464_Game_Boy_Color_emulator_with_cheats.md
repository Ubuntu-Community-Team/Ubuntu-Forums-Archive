### Post by termin on 2011-05-31
Try VBA-M. It's like the windows version of VBA download it here:
[http://sourceforge.net/projects/vbam/files/VBA-M%20GTK%2B%20svn%20r1001/](http://sourceforge.net/projects/vbam/files/VBA-M%20GTK%2B%20svn%20r1001/)
Don't download the tar.gz file, just download the .deb GTk and Sdl file. Then install both of them. Also, if the Application won't start then just download getlibs from:
[http://frozenfox.freehostia.com/cappy/](http://frozenfox.freehostia.com/cappy/)

then open up a terminal, navigate to /usr/bin by typing:
cd /usr/bin
then just type in:
sudo getlibs gvbam


enjoy your games.
