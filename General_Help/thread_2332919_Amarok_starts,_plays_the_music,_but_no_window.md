### Post by ajgreeny on 2016-08-05
It is years since I used amarok, but check that it is not running in the background with an icon in the system-tray.  There are several music players that are able to play with no window showing; I think rhythmbox does, or can do that as well, but I don't use that either, so you may need to check further.

If you open amarok from the menu do you get a window in that instance?  If so, look in the settings menu to see if there is one for "Minimise to icon in status-bar" or similar.
