### Post by Idefix82 on 2008-10-25
You don't need an anti virus for Ubuntu since viruses can do almost no harm on Linux. The only thing they can do is get passed on to some Windows machine.
A firewall is built into Ubuntu and unless you have very specific needs, you don't really need to change the settings.

### Post by sethvath on 2008-10-25
As mentioned by Idefix82 you do not necessarily need an AV program like in windows, no harm if you really want it for peace of mind. 

Firewall: [GUFW]("http://gufw.tuxfamily.org/index.html") or [Firestarter]("http://www.fs-security.com/download.php")

Anti-virus: [ClamAV]("https://help.ubuntu.com/community/ClamAV")
