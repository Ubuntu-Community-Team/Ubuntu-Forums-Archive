### Post by and.hunt on 2009-04-18
I would watch out when using a normal usb-stick since flash memory can die after a while. In the past I've tried using Linux on a number of usb-sticks (I actually used Knoppix back then), but they all died quite quickly. However the last time I did this was two years ago, and flash could now be more reliable. A better idea is to use a pocket sized usb hard-drive though, since they are better suited to all the write-access you'll be getting.

### Post by HeirPixel on 2009-04-18
Another option for you would be just to install Ubuntu on both machines and use your portable drive as a sort of common Home directory. This would greatly alleviate wear on your portable drive and and free up system memory for actual tasks (RAM is already a precious commodity on netbooks without using it to host your OS as well). You could still set it up as a LiveUSB just to boot on other computers with your same Home directory on the move!

Not sure if that would really fit your needs, but it could be worth considering :)
