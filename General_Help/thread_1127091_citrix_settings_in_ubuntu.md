I have found a solution (or rather a workaround)


1. Installed Citrix ICA client under WINE (I have used old version found via google, new one didn't want to install and freezed on checking system during install)
2. I had ica files saved, I started them with WINE and everything works like a charm :)

link to old Citrix ICA: [http://www.landonline.govt.nz/software/ica32.exe](http://www.landonline.govt.nz/software/ica32.exe)
