I was wondering if xubuntu will run on my old box. I has 128MB of ram and 1.4GHz pentium 4 processor. I have tried to find information on this but I have not been having much luck on finding an answer.
This page: [http://www.ubuntu.com/products/whatisubuntu/xubuntu](http://www.ubuntu.com/products/whatisubuntu/xubuntu)
says that you need 192MB of ram to install which has me worried.
If I had a swap partition set up on my hard drive at the time of install would the installer use it to take some load off the ram? Or is there a not so ram intensive installer I could use?

Any help is much appreciated?

Thanks!

### Post by Woody1987 on 2009-02-28
yes it will run albeit a bit slower than you might expect but it will run none the less. With so little ram you should definitely use SWAP. The installer itself i assume wont actually use any SWAP but the OS will when installed.

### Post by RedSingularity on 2009-02-28
