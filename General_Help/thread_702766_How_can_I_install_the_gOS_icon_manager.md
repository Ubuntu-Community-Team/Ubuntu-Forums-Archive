### Post by y-lee on 2008-02-20
> I was wondering if anyone know what the icon manager at the bottom was called and how I would be able to install it? 

It seems that is called the gOS iBar. and is a maybe modified (?) part of the [Enlightenment window manager.]("http://en.wikipedia.org/wiki/Enlightenment_(window_manager)").

you can install[ Enlightenment on ubuntu]("http://www.howtogeek.com/howto/ubuntu/install-enlightenment-on-ubuntu-linux/")

```
sudo apt-get install enlightenment
```

and if you really wish you can [install the gOS desktop]("http://zebardast.wordpress.com/2007/12/18/how-to-install-gos-on-ubuntu-gutsy-gibbon/") :)

Myself I wouldn't bother I would use something else. But to each his or her own, linux is about freedom and choice.
