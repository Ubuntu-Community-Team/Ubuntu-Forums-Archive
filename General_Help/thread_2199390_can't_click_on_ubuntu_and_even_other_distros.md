Are these problems only related to using the mouse?

Does the keyboard work properly? Almost everything you can do with the mouse has a keyboard equivalent. I'm not saying that you shouldn't use the mouse, but if the functions work ok with the keyboard, that helps isolate the problem.

Do you have any other pointing devices to try? A touchpad, trackball, or another mouse?

Do you have a second USB port to try?

Have you tried changing the mouse settings? There's should be an area for that in the system settings area.
