### Post by Mistiq Dragon on 2010-12-05
ummm, that reminds me of something I've heard about.  I would suggest using CCSM  to control what applications open in which window (can't remember the name of the plugin that does that.)  Once you get that set up the way you like, I would go save and export that profile in CCSM.  

Then once you have that set up the way you like, you can set up a profile the same way that fits your work habits.  Once you have both profiles in CCSM saved and created, I would think all you have to do is load the profiles whenever you need one.  maybe about 5 or 10 min to set up initially and then not more than a minute to switch between them once they are set up.
