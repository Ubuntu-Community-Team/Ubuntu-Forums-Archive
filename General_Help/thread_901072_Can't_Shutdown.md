### Post by iaculallad on 2008-08-26
> **starving030 said:**
> I have hibernate, suspend, log out, switch user and lock screen. How do I get shutdown or restart? It was there when I first installed but is gone after running all the updates. Running Ubuntu 8.04

Try this:

System->Administration->Login Window

In the "Local" tab of "Login Window Preferences", make sure that "Menu Bar: Show Actions menu" is checked.
