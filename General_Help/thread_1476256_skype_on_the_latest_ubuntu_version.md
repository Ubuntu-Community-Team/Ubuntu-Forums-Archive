### Post by GSF1200S on 2010-05-07
Skype is working fine for me (10.04 Xubuntu 64bit). I just opened a chat message from someone else, and attempted to make one with no crashes. A good troubleshooting technique is to open the application from a terminal. In skype's case, simply run:
```
skype
```
in a terminal. Leave the terminal where you can see it, and then open a chat message while you look for terminal output. Then post the output here so we can see what error messages it spits out.
