### Post by michy99 on 2010-01-12
Open your sources.list for editing.```
gksudo gedit /etc/apt/sources.list
```Replace ```
us.archive.ubuntu.com
``` or ```
archive.ubuntu.com
``` with ```
old-releases.ubuntu.com
```Save and exit and try updating.

But the best thing will be installing a new release.
