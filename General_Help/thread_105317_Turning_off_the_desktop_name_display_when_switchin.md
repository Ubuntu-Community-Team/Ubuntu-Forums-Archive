### Post by One Quick Question on 2005-12-21
I guess I'll post this in case anyone else was silently wondering the same thing.   I found the option.  For some odd reason it's located under the Focus settings of Window Behavior.  "Popup desktop name on desktop switch"
Seems like a pretty random place to put such a thing.
