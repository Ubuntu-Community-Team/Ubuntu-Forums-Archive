### Post by Razmear on 2011-10-18
I'm still using Ubuntu 10.10 mainly because I have no interest in the Unity experience. 
If I install the kbuntu desktop, can I then upgrade to kbuntu 11.04 and then 11.10 to get the new kernel without messing up all my other installed apps? 
Yes I know a clean install would be 'cleaner', but I'd rather do upgrades and keep my current installed apps and games than have to rebuild everything.

eb

---

### Post by bodhi.zazen on 2011-10-18
Yes you can do that, probably easier to upgrade first, then apt-get install kubuntu-desktop second.

As you say, a clean install would probably be less hassle, otherwise there will be a number of duplicate applications in your menu.
