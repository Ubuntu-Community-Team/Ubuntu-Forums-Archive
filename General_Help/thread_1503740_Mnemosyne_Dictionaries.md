### Post by StuartN on 2010-06-07
The European Union produces a multilingual translation dictionary (heavily biased to business and governance language) at [http://europa.eu/eurovoc/sg/sga_doc/eurovoc_dif!SERVEUR/menu!prod!MENU?langue=ES](http://europa.eu/eurovoc/sg/sga_doc/eurovoc_dif!SERVEUR/menu!prod!MENU?langue=ES)

You can download the whole database, any language pair, and some subsets of the dictionary. It is easy to read in a CSV etc in Mnemosyne / Parley.

### Post by StuartN on 2010-06-07
> **JugglinPhil said:**
> Don't think I can import .kvtlm files, and if so I do not know how to.

The .kvtlm is just an XML file. The easiest way to handle it is probably to import with something like Parley and export the two columns you want as CSV, unless there is a more vocabulary aware format that both Parley and Mnemosyne have in common.

I guess that Parley is a very light install if you already have Mnemosyne.
