The "Connect to Server..." under menu Places is very useful under the desktop. But how to realize its function under command line environment?

For example, I can use "Connect to Server..." with ftp(with login) option to mount a drive remotely, and browse this remote drive under desktop. So which command can I use to realize this in terminal? I think this is the same as the map drive in Windows.

---

### Post by nsn3 on 2007-03-24
"pon dsl-provider" to connect

"poff dsl-provider" to disconnect
