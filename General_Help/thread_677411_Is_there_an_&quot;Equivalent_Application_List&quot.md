### Post by Zeroangel on 2008-01-24
For your media needs, Xine will play DVD movies and such (I use Kaffeine since i'm on Kubuntu), anything which uses the Xine backend will also be able to do so, provided you have the appropriate codecs installed (libdvdcss and ffmpeg).

There are lots of media programs that will allow you to manage what's on your IPOD, I think Ubuntu comes default with one of them. No itunes shop support though!

GIMP is not a suitable photoshop replacement for graphics professionals, IMO (being a hardcore photoshop user), but some people beg to differ. Photoshop 7 runs fantastically using WINE, I havent tried any of the later CS versions, but I know that the people at WINE have worked very hard on getting PS7 to run and they have done a great job. CS to CS3 support should be coming someday though. If vector graphics are your thing, Inkscape does a great job (lots of people use it for linux icons) but from my experience theres a bit of a learning curve.

kinfocenter is a replacement for cpu-z and AIDA32, but it doesnt do as great a job as they do.

k3b is a GREAT replacement for Nero Burning ROM.

For Web Development, I *highly* recommend Quanta. Its got project management (FTP synch and two click uploads!), session saving, browser preview including the ability to preview PHP files on the local server. In addition to syntax highlight and autocomplete. Its WYSIWYG leaves much to be desired, but if your hardcore enough of a web developer chances are you dont even use WYSIWYG and just hand code things like a real man. :KS

There is a version of Filezilla for Linux -- but also, Nautilus comes with FTP support! Figure out how to use it because it works great (can even be set to show file previews), and if you want you can have the FTP site show up in your 'places' menu. 

For torrents, there is a version of azureus for Linux. I personally dont like it because I have bad experiences with Java programs under linux (they seem so slow and crashy). Your experience might be different.

Another list is here: [http://www.linuxalt.com/](http://www.linuxalt.com/)
