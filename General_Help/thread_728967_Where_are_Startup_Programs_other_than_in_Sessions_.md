### Post by OzzyFrank on 2008-05-05
OK, nobody seemed to know this, from the lack of replies, so here is the answer. Now, in **Sessions** it is pretty obvious what the **Startup Programs** tab is for, but if an app isn't on the list, then that doesn't help. You would be forgiven for thinking the **Current Session** tab just is for what is running - like System Monitor - but delete something from there, and it will no longer run on startup. Seems a bit silly to call it Current Session, but there you go...
