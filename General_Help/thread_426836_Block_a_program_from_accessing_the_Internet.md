### Post by honeydew on 2007-04-28
If you know what ports the program is running on you could use a firewall.. a really good gnome firewall is firestarter. It is pretty easy to setup..

in the terminal you can type
```
sudo apt-get install firestarter
```

then it will be under system>administration>firestarter
