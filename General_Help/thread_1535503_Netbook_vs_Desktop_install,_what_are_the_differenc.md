### Post by MikeyDL on 2010-07-21
You actually install regular gnome with the netbook flavor version.  The netbook version gets rid of the bottom bar and merges the top part of the application into the regular top bar.  

It basically maximizes space for the netbook which is nice given the average vert resolution is usually 600.  

However, that GUI does take away virtual desktop switching, adds a funky large icon menu on the side and there are a couple of other minor annoyances (for me) such as no "connect to server" item which made it a little bit painful get into network shares.

However, you can just log out and then go into the regular gnome so if you have a netbook it's probably best to install the netbook version to get both choices.

Can't think of any other differences.
