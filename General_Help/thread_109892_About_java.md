### Post by kaamos on 2005-12-30
If you have both installed, type this in a terminal and then choose sun
```
sudo update-alternatives --config java
```
More info here: [https://wiki.ubuntu.com/RestrictedFormats#head-e2ebd70ede0e3eb2117ffbd618d2295dd1540dca](https://wiki.ubuntu.com/RestrictedFormats#head-e2ebd70ede0e3eb2117ffbd618d2295dd1540dca)
