### Post by LadFromWales85 on 2008-05-28
I have two panels, one at top and one at bottom, and programs never overlap the panel, unless I click the hide button on the right of either.  Upon clicking them again, program windows will move back up to where they were before I hid the panels.

I have the "Hide only when a panel-hiding button is clicked" option selected.
If I change to "Allow other windows to cover this panel", then maximised windows will overlap the panel, but only when this is selected and applied.

When the former option is applied, the panel area is a 'no fly zone' for application windows.  I've not come across any applications that can overlap the panel, though applications can become full screen fine, such as Firefox.

Hope this helps :)
