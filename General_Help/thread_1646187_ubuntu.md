### Post by coffeecat on 2010-12-15
Yes. The ISO is only referred to as AMD64 for historical reasons. If you have an Intel 64-bit processor, it will run the 64-bit version of Ubuntu from the AMD64 CD OK.

### Post by WorMzy on 2010-12-15
AMD64 is named after the company that created it, AMD. Modern Intel processors can run AMD64-based operating systems, just like modern AMD processors can run Intel 8086-based (32-bit) Operating Systems.

If you can run 64-bit Windows, you can run 64-bit Linux.

The reason why the 32-bit OS is "recommended", is because the 64-bit variety isn't supported by older CPUs (e.g. most CPUs released before 2003). There are also a few teething issues with 64-bit Flash support, though this is Adobe's problem rather than Ubuntu's. Other than that, 64-bit is on par with 32-bit, it really comes down to whether you have more than 3GB of RAM, or do a lot of multimedia work (e.g. video editing), if you don't, then using 64-bit probably won't benefit you.
