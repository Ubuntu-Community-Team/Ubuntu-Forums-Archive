### Post by NoaHall on 2009-08-29
Is compiz installed? Are they running as widgets? If they are, you'll need to enable a "widget layer" in compiz, and then use the command chosen to show them. Also, set them to keep always above.

---

### Post by CatKiller on 2009-08-29
If you're using Compiz, I believe that you can change the behaviour of the Show Desktop plugin to ignore windows of a class that you specify (although I don't use that plugin, so I couldn't confirm that it works). The easiest way to get to a clean desktop with your screenlets intact is just to switch to a different workspace, though.
