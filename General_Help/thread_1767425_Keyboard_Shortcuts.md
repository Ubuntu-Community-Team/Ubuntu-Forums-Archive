### Post by zasderf on 2011-05-25
There is usually (limited) shortcuts via "System" --> "Preferences" --> "Keyboard Shortcuts"

And/or look into something called "Gnome Do" which you can get through the usual Software Centre. It's a program which effectively launches other programs. Say I wanted to run firefox, I can run Gnome Do by a very simple shortcut (customisable) then type 'fire' and firefox should come up, enter and you're done. Literally running firefox or most other daily programs within 3seconds.
