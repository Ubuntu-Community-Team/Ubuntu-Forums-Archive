### Post by Woody1987 on 2008-10-29
The kernel update shouldnt affect the driver installed with envy, i used to use envy and i recall it working fine after every kernel update. If you do have to reinstall it then yes, remove the old one first. Hardy wont get a newer driver than 169.12, but if i were you, i would wait till tomorrow, update to intrepid ibex which DOES include the latest nvidia driver. If you choose to update to intrepid you MUST REMOVE the driver installed by envy otherwise you will have problems. But as i say i recommend waiting till tomorrow for intrepid ibex
