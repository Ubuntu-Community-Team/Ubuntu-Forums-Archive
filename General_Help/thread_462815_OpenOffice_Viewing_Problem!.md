### Post by Hagar Delest on 2007-06-05
I've seen this reported already on French and English OOo web forums. It's linked to Compiz. You could try the ALT+Space keys to reset the maximization. If the setting is not kept, a user has found a rather strange fix : close your session without closing OOo. Next log in led to a correctly wrapped window. But no information about the stability of this fix.
