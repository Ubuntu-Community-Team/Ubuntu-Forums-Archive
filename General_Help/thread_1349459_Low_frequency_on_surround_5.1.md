### Post by ForgivenByJC on 2009-12-09
Perhaps, in daemon.conf uncomment
```
; enable-lfe-remixing = yes
```
to look like this 
```
enable-lfe-remixing = yes
```
And restart pulseaudio.
```
killall pulseaudio
```
Did that help/work?
