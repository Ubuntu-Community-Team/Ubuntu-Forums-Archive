### Post by skippy81 on 2006-05-10
Control - Shift - V :p 

You have to add the shift key to make normal cut/paste operations work in terminal mode.  This is because the standard combinations are used by the shell for 'break' commands etc.

Nice tip with the middle mouse button by the way :) I will use that from now on.

### Post by Life On Mars on 2008-07-12
If you want to use Ctrl-C and Ctrl-V for cut and paste in Terminal you can use Alt-F2 > gconf-editor > apps > gnome-terminal > keybindings > look for copy and paste > right-click > edit key > delete <Shift>
