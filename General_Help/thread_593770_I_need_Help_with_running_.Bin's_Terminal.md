### Post by FuturePilot on 2007-10-27
If you want to use the terminal put this command in
```
sudo apt-get install sun-java6-bin sun-java6-jre sun-java6-plugin sun-java6-fonts
```
If you'd rather do it graphically
System>Administration>Synaptic Package Manager
search for sun-java6
and install these packages
sun-java6-bin
sun-java6-jre
sun-java6-plugin
sun-java6-fonts

I'm not sure what RoR is.

### Post by seshomaru samma on 2007-10-27
Maybe you mean RAR?

in that case put this command in the terminal
```
sudo apt-get install unrar
```

if you get some kind of error when you do that ,then it means you haven't enabled the multiverse repo. If you have no idea what that is , read [here]("http://www.psychocats.net/ubuntu/sources")
