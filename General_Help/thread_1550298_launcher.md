### Post by worksofcraft on 2010-08-10
I have a bunch of different projects I'm working on and each one is in a folder on my desktop.

Now working through the GUI desktop is great for lots of things but quite often I just want to open a command prompt with that specific project folder as current directory.

What I want to do is click on the folder and select "open in terminal window", or perhaps drag the folder and drop it on a launcher on my panel. However my understanding of creating panel launchers is just not good enough... so I was wondering if someone else already solved this?

---

### Post by ubunterooster on 2010-08-10
If you run ```
 sudo apt-get install nautilus-open-terminal
```then you can right-click a folder and select "open in terminal".

---

### Post by Vaphell on 2010-08-10
well, nautilus has the 'open in terminal' option in context menu with stuff mentioned above
in list view it works only with listed dirs, with the other views you can also rightclick on the background to get the current dir (in other words list view has no background to click, all space is used)
also you can install nautilus-actions and customize the context menu even more.
