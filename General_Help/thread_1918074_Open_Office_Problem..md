### Post by SeijiSensei on 2012-02-07
Create a style and apply it to each question.  That's tedious, of course To speed things up you could construct a simple macro to perform the steps required to change the style.  If you set it up to work from the beginning of a question, you can just put the cursor before each question and hit the key to which you assigned the macro. 

Are the questions in the same style as the surrounding text?  Maybe there already is a question style that you can modify.
