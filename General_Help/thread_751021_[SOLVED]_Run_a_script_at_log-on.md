### Post by whitefort on 2008-04-10
Hi.

I have a nice little script that uses xplanet to create a wallpaper of the earth centred on my home location, and updating every 20 minutes to show day and night moving across the planet.

I use 'cron' to do the 20-minute update.

My problem is that when I turn on the PC, it starts off with no wallpaper at all until I reach one of the 20-minute intervals when the script runs.

So, my question:  how do I set the script to auto-run when I first log-on?

Thanks!

---

### Post by kpkeerthi on 2008-04-10
Add the script with full path to System -> Preference -> Session -> Startup Programs -> Add -> Command
