### Post by Daremo_06 on 2009-04-05
Fairly new user to Ubuntu 8.10 x64.  Something I have noticed while using Firefox over IE, is that when I click on the address bar to paste in a new address, it does not "autoselect" the whole address and thats a feature I am used to coming from IE.  It also doesnt do that in the google search window I have set as the default search engine.  

Is there a setting for this that highlights the entire text in address box or search bar so that you can just simply paste your new text in without having to delete 1st or highlight it manually?

Thanks!

### Post by Vague180 on 2009-04-05
In the address bar, type about:config

Scroll down to browser.urlbar.clickSelectsAll and double-click it to set it to "true"

### Post by Vaphell on 2009-04-05
few hints:

if you are into keyboard shortcuts:
- ctrl+L selects whole address so you can replace by pasting in new one
- F6 too, but it's not as reliable (sometimes doesn't work)

if you aren't and have url in plain text:
- drag the url and drop it onto unused space in tab area, new tab with given url will spawn

keyboard option is faster: if you want new tab, you can do fast CTRL+C, CTRL+T, CTRL+V in 1sec
