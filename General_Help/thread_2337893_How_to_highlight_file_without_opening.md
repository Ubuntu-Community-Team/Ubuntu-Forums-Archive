### Post by bswarm2 on 2016-09-22
I'm using 16.04, but it should be the same. If you have Nautilus or Nemo set to open files with a single click, you can highlight the file without opening it by holding the left mouse button down longer. Otherwise you can change back to double click in Nautilus or Nemo preferences.
