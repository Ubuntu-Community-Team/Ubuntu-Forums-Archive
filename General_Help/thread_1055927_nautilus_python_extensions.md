If anybody interested, I created an application that handles the extensions to the file manager with some extensions included and possibility to create new ones with little studying.
To see screenshots/download the debian/ubuntu package click [HERE]("http://open.vitaminap.it/en/linux_developments.htm").

P.S. A little more detailed description:

A graphical handler, written in python and pygtk, of the nautilus file manager python extensions.
 
Included python extensions :

open-terminal-geometry.py allows to open the terminal window on the selected folder/current directory, with predefined geometry (edit the parameter GEOMETRY to change it), with the right-click;

copy-to-browsed-place.py allows to create a copy of all the selected files/folders to a browsed destination with the right-click;

set-as-desktop-background.py allows to set as desktop background the selected image file with the right-click;

add-to-audacious-playlist.py allows to add all the selected audio files to the audacious playlist with the right-click.
