### Post by sekinto on 2008-04-13
Why don't you just use Synaptic Package Manager to install Nexuiz, it is in the Ubuntu repositories.

System>Administration>Synaptic Package Manager
Click "search"
Type "nexuiz"
Right click it and click "install"
Click "apply"
