Hi all!  I would like to hide the Grub menu, but still display a splash screen, and be able to blindly select the items.  Is this possible?

### Post by DOSIX on 2011-01-21
Umm I wouldn't recommend doing so unless you are just going straight to Ubuntu. What you could do is download a program called Start Up Manager from the Ubuntu Software Center. Then just go to System->Administration->Start Up Manager and set the total time that GRUB displays to 1 second. You can also select it to show bootsplash from there I think, but I'm not entirely sure. If you ever need to load another OS, then just be quick and press an arrow key as soon as GRUB pops up. That will disable the timer.
