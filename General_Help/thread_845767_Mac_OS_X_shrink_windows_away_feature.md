### Post by Doji on 2008-07-01
Ubuntu tweak looks neat, but I would recommend using CCSM for the extra configurability it provides. So, if you haven't already, install CCSM (sudo apt-get install compizconfig-settings-manager).

Go to System > Preferences > Advanced Desktop Effects Settings.

Click on General Options, and click the button across from "Show Desktop". An image of a computer monitor with red things around the edges. Just click the red thing in the corner where you want to move your mouse and hit ok. Now go back.

Find the "Show Desktop" plugin. Make sure it's enabled and go to the 'Misc. Options" tab. Across from "Movement Direction" you can choose the animation you want. I don't know what OSX uses, but I like "to corners" :D.
