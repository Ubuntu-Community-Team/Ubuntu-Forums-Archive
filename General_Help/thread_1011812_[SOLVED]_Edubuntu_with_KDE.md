### Post by TeXtonyx on 2008-12-15
> **theozzlives said:**
> Edubuntu is an add-on to Ubuntu. I was just wondering if it can be appied to Kubuntu

In order to install Edubuntu 8.10 Desktop permanently on a computer, you will need the following two CDs

   1. Get the Ubuntu 8.10 Desktop CD - You are free to choose 
the Alternate CD, if you prefer a text-based installer. 
   2. Get the Ubuntu 8.10 educational add-on CD

--------------------------------

[http://edubuntu.org/Screenshots](http://edubuntu.org/Screenshots)

"The KDE Edutainment Suite

KDE is another graphical environment, and also serves as an 
application framework. The KDE project has a sub-project called 
the KDE Edutainment Suite, which is a collection of fun 
educational software that makes use of the KDE framework. KDE 
applications runs fine under GNOME. We have included the very 
latest versions of the KDE Edutainment software in Edubuntu."

-------------------------------------

You can add the KDE-desktop to Ubuntu, and boot to either the
Gnome or KDE desktop. Edubuntu comes on two cds that total over
800mb, it is an operating system in itself which uses Ubuntu
as a foundation. I think the choice of desktop will be presented
by the Edubuntu install cds. I don't know if Edubuntu works with
Kubuntu, but I doubt you can force a KDE-desktop by using Kubuntu
rather than using Ubuntu with the KDE-desktop package installed. 

If the KDE files are on the Ubuntu partition, I think Edubuntu
will offer KDE as a desktop, or Gnome as a desktop (both) or
provide no choice and install only KDE or only Gnome. Since
the KDE Edutainment Suite is included with Edubuntu, then the
KDE framework will be available. Either it will come with the
KDE files, or it will utilize the (added) Ubuntu KDE files.
It is quite unlikely that Edubuntu works without Gnome files 
since that is the Ubuntu default and it says Ubuntu is needed.
Thus if you used the Kubuntu install cd as a basis, I think it
is most likely you would have to add Gnome files to it, and 
again it would come down to what Edubuntu offers as install
choices, or it will not offer a choice and probably just use 
Gnome, less likely, Kubuntu, although it supports KDE Edutainment
or just use its own Edu desktop with associated icons and menus.
