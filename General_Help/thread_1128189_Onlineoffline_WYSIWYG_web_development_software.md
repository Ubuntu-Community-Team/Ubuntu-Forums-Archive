### Post by aaronp on 2009-04-17
At the risk of unpopularity I'd say start looking at some html/css tutorials and learn to code it by hand. It may seem like a chore now but it really won't take long and is worth it.

This is where I learnt a lot and it is really quick and easy to go through: 

[www.w3schools.com](www.w3schools.com)

It's also good as a reference point when you get stuck on something.

Once you have created something using WYSIWYG the underlying code will be something that you have little chance of easily modifying by hand so you will forever be reliant on the app you used.

And if you get really stuck on something, there are plenty of people on this forum who will no doubt assist...
