### Post by sisco311 on 2009-11-09
The default shortcut is Ctrl+Alt+d.

To change it, go to
Xfce Menu -> Settings -> Window Manager -> Keyboard tab

### Post by refp16 on 2010-04-05
Hello again,

I just figured it out (I'm new to linux).

So the windows manager, i believe, only lets one modify the shortcuts for windows manipulation. Once i double clicked in one of the entries, one actually has to press the desired (new) shortcut combination as to replace the original one. But i think you can only change the key bindings of the existing windows commands.

On the other hand, if one wants to add a shortcut, for lets say, launching an application, one can use the xfce menu button->settings->keybord (application shortcuts tab). Again, one has to actually press the desired (new) shortcut combination as to create (or replace) it. This is done right after you enter the new command to launch the application.

Hope this helps.
