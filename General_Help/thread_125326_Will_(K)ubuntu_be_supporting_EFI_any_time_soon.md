### Post by nrwilk on 2006-02-03
Intel's newest boot system; designed to replace BIOS.

[QUOTE=http://en.wikipedia.org/wiki/Extensible_Firmware_Interface]
Extensible Firmware Interface (EFI) is the name for a new system developed by Intel that is designed to replace the aging BIOS system used by personal computers. It is responsible for the power-on self-test (POST) process, bootstrapping the operating system, and providing an interface between the operating system and the physical hardware.
[/QUOTE]

To greater define my question, you should know this:
Apple's new intel Mac models use Intel's new EFI system instead of BIOS, or Open Firmware (which is what they previously used).  I want to know when I'll be able to run a X86 version of ubuntu on a new mac.

Further down on the above linked wikipedia page, is this quote:

[QUOTE=http://en.wikipedia.org/wiki/Extensible_Firmware_Interface]
Linux systems have been able to use EFI at boot time since early 2004, using the elilo EFI boot loader.
[/QUOTE]

Is elilo available for ubuntu now?  Does anyone use it?  How well does it work?


While researching this, I've come across this link, which says that Red Hat is already working on it:
[http://www.engadget.com/2006/01/25/red-hat-to-bring-linux-to-intel-macs/](http://www.engadget.com/2006/01/25/red-hat-to-bring-linux-to-intel-macs/)


Here are the facts (which explain the dilemma in better words than I can) explained by macrumors.com:
[QUOTE=http://guides.macrumors.com/Mac_OS_X_on_Intel_FAQ#Will_I_be_able_to_run_Windows.2FLinux.2Fother_x86_OS_on_my_Intel_Mac.3F]
While many of the core components of new Intel Macs are now able to run x86 OS's natively, some design aspects of how Apple implemented its low-level motherboard firmware looks to prevent extremely easy running of other x86 OS's on Intel Macs. Notably, Apple has decided to use EFI instead of BIOS. EFI is currently not supported by 32-bit versions of Windows XP and also may cause headaches for many popular Linux distributions. However, it does not appear that Apple is intentionally being difficult when it comes to motherboard design, but is rather simply going with a more recent Intel standard. In addition, Windows Vista will support EFI, so Windows compatability is not far off.

"That doesn't preclude someone from running [Windows] on a Mac. They probably will. We won't do anything to preclude that." - Phil Schiller, when asked about the possibility of running Windows on Intel-based Macs. [1]

It is also worth noting that the Wine Project, created to run Windows applications unmodified on Linux, is being developed for Mac OS X on Intel, possibly negating any reason to install a full Windows operating system alongside Mac OS X. 
[/QUOTE]


I'm going to love being able to dual-boot x86 Kubuntu with OS X on an intel mac laptop.:D
