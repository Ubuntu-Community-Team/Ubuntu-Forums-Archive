### Post by snowpine on 2010-09-25
To be blunt, Ubuntu is not Slackware. :)

Most applications in Ubuntu can be installed using the Ubuntu Software Center, Synaptic Pagckage Manager, or terminal commands such as "apt-get", "aptitude", or "dpkg".

I would suggest either installing Slackware (if that's what you want to learn) or reading up on the Ubuntu basics: 

[https://help.ubuntu.com/community/InstallingSoftware](https://help.ubuntu.com/community/InstallingSoftware)
[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)

Also, when asking for help on these forums, it is good etiquette to tell us which application you are trying to install. This makes it easier to give you specific instructions rather than giving generalized advice to cover all possible scenarios. :)
