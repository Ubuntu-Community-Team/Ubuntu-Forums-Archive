### Post by schauerlich on 2008-10-07
You've got a couple of options. You could do a normal Ubuntu install and trim the fat afterwards, you could use the Ubuntu server CD to get a solid base to build off of, or you could try out Arch and build from the ground up with precompiled packages. As far a localization, Ubuntu will probably be the best for you.
