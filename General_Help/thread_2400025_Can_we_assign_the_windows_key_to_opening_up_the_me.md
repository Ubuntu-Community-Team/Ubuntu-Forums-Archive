### Post by vanadium on 2018-09-01
Use the utility ksuperkey. With it, you can have a keypress on the superkey trigger any key combination, in your case the key combination used to open your menu. By default, ksuperkey will trigger Alt+F1 if you press super, which in many desktop environments, and perhaps also in lubuntu, opens the menu.It is not available in the repositories. It is hosted on hithub: [https://github.com/hanschen/ksuperkey](https://github.com/hanschen/ksuperkey). You can install it from a user repository ([https://store.kde.org/p/1081256/](https://store.kde.org/p/1081256/)).
