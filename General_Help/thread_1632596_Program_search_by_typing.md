### Post by krige on 2010-11-28
One thing which I find very useful in Windows 7 is the possibility to run a program by pressing the [super|windows] key and then start typing the name of the program. Windows shows the programs in the Start Menu by sorting them by use frequency, so that most of the times you only need to type only one letter to access the desired program (e.g. super+m+enter to access Mozilla Firefox, super+w+enter to access Word, etc).

Is there a similar functionality already present in Ubuntu or is there a program we can install to have it?

### Post by coffeecat on 2010-11-28
Press Alt+F2 and begin to type in the app name. For most you only need 2-3 letters before you are prompted for the nearest match. Then press enter.

One disadvantage is that some apps have  "real" names a little different from what you might expect. For example, 'alacarte' for the main menu editor, 'palimpsest' for the Disk Utility. But most are what you expect: firefox, gedit, and so on.

**EDIT:** after all these years I've only just noticed. :| There's a 'show known applications' drop-down list as well. Which is useful.
