### Post by guiverc on 2023-03-03
I started using redshift in 2015 when I discovered it, and I've been using it ever since  (*I was pleased to discover it existed in trusty/14.04's repositories so I installed it on a 14.04 machine too*).

Sure my GNOME desktop comes with [Night Light]("https://help.gnome.org/users/gnome-help/stable/display-night-light.html.en") but that only works on GNOME, and I use multiple desktops and want to the same setup for all; thus I disabled *Night Light* and still use redshift using that *same *config file I setup back in 2015. 

 It works really well, is highly configurable, and works the same if I login to a Lubuntu, Xubuntu, or Ubuntu Desktop session on this machine (*my box is a multi-desktop* *install*), having the ~same control on the panel on my desktops.  I also use it on other machines where I'm using other desktops too.

### Post by DuckHook on 2023-03-04
I used to use redshift and will also attest to its effectiveness.

But just a (mild) word of caution: if you are running vanilla Ubuntu and only vanilla Ubuntu, then there is no need to add a fifth wheel&#8212;you will already have nightlight in your settings. Just turn it on and let it do its job. It's best to keep your install as close to default as possible. This forestalls a lot of pain and anguish on your next release upgrade.

If you are running another flavour, then you will likely need it.

@DuckHook, thanks for that - I've learned something new and useful. However, I got lost in a cul de sac somewhere between steps 2 and 3 in the [official 22.04 help.ubuntu.com page for this]("https://help.ubuntu.com/lts/ubuntu-help/display-night-light.html.en"). That might be me having a senior moment, or it might be an erroneous holdover from an earlier version of gnome, but to help others similarly challenged here's my (simpler) navigation for getting to night light from the Ubuntu 22.04 desktop: 

[list=1][*]Click on the power icon at the extreme right of the top bar, and thence Settings in the drop-down menu.
[*]Select Screen Display in the left pane.
[*]Click on the Night Light tab in the top bar of the settings window.
[*]The rest should be straightforward. :)
[/list]
