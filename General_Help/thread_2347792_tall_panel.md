### Post by yetimon_64 on 2016-12-29
By right clicking on the panel > Panel > Panel Preferences you can bring up a window for the panel which includes a couple of "sliders" for setting the panel sizes. 

Check you only have one row active, a second one could possibly double the panel size if accidentally changed to "2" rows. Or the Row Size (in pixels) may possibly have been moved.

As per the attached screenshot under the "Measurements" section, check out how many rows you have set as active (normally would only be one) and check the size in pixels set for the panel by the "Row Size" slider.
