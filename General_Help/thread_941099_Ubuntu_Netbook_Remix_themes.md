### Post by jjpcexpert on 2010-07-27
Basically, I've fiddled around with the Human Netbook gtkrc, the Panel Tabs, the Panel Background, and have come out with a theme that will install in the way that XPGnome will, since this one is made with the Phranks Theme Saver

Attached is the Seriously Netbook Theme.
I've left you with a potentially rock-based background, and a blue theme for the maximized-window tabs and the GTK. If Ubuntu/Canonical can step in to tell me if this is legal, that is the best way to go.
The launcher is always dark, to save power on netbooks (I'm running the UNR desktop from a mid-range laptop for god's sake!) but it should be OK if you choose a light background. This theme will only work on a system with UNR packages installed and WILL transform your desktop! Requires 10.04 for best effect, however the 2D version of 10.10 will work also.

It is on an external site, [here (hover to find that this is a link)]("http://freewebs.com/theinfositebyjackj/SeriouslyUNR.tar") (right click to save to location other than Downloads or Desktop). This is not a tarbomb, it is about 19MB extracted, and is not compressed. It is suitable for any system with the package named (ubuntu-netbook-remix) or (ubuntu-netbook) (be sure it's 2D in 10.10).

Also, I mashed-up two themes (Shiki-Brave and Seriously8.10-HumanNetbook) to create [Shiki-Seriously (tape archive .tar, 5.5 MiB long)]("http://freewebs.com/theinfositebyjackj/ShikiSeriously.tar")

Look in the attachments for a Unity-like GNOME Panel setup. I call it U2D.
Prerequisites:
DockBarX (just Google it)
Global Menu (just Google it)
Cardapio (ppa:cardapio-team/unstable)
Gnome Window Applets (just Google it)
It was created with Theme Saver, and no cheese flan. It just restores to my panel positions. It is only tested on 1280x800 screen resolution. Make sure that the screen has a horizontal resolution of 1280 pixels (1280 vertical scanlines for CRTs). 1280x600 is not recommended, because I don't want people having 22:6 screens.

When I say "just Google it", I mean search it using your fav. search engine.

Hope you have a good day, and enjoy the theme!
