### Post by ryanhaigh on 2008-05-22
I have issues like these very rarely on my gutsy system and I have worked out that, for me at least, the culprit was compiz/desktop effects. To test this yourself next time it happens press alt-f2 to bring up the run dialog then enter metacity --replace, this will turn of desktop effects. If the problem persists then effects aren't the issue and you can restart them using compiz --replace to start them again. If you determie they are causing teh problem turn them off permanently in system>preferences>appearance.
