### Post by bromix on 2007-10-17
I guess it's been awhile since this request has been made, so I'll courteously and humbly do it again. :)  Just a friendly reminder that when posting a question or problem, once you have an answer or workable solution, please remember to mark that thread as solved.  You can do this by navigating to that particular thread, and choosing "Thread Tools" and selecting the option to "Mark This Thread As Solved".   This has a two-fold purpose.  The first being that users actively searching the forum in hopes of solving a problem can see that this post contains at least a hope of a solution if they have the same question.  The other being that when users are browsing to see if there are people that they might be able to assist, they don't have to spend the time to read through to find out help is no longer needed.  Just something that takes a couple of seconds and really helps out the whole community.  Thanks :)
