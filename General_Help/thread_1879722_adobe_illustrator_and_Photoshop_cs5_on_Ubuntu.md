### Post by 3rdalbum on 2011-11-12
Installing Windows as a guest OS using Virtualbox is certainly one option.

If you can install Photoshop on 10.04 or 10.10 then it should be the same procedure or easier to install under 11.10. In both instances it will use Wine. I believe that, these days, no version of Wine is released unless it works well with Photoshop.

Why not just try it though? You already have Ubuntu on your notebook, so you should just be able to install Photoshop CS5 and Illustrator on there as a dry-run.
