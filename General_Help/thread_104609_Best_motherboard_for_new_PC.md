### Post by BathroomNinja on 2005-12-16
I'm using an ASUS made Nforce4 motherboard.  Socket 939.  If you go with an AMD 64, I highly recomend getting the 939 socket Processor and Motherboard since it has a few instructions that the slightly older 754 socket doesnt have that are pretty sweet.

So pretty much, get a MB and proc that are 939 socket, and I recomend NForce4 chipset and of course, SATA.

I'm using an ABIT, an ASUS and a Dell made motherboard, and all work flawlessly.
