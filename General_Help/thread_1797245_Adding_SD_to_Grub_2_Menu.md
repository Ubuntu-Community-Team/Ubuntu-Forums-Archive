### Post by An Sanct on 2011-07-05
For SD, you need to load some things first ... more about this issue in [this forum thread]("https://bbs.archlinux.org/viewtopic.php?id=97207").

The alternative is to use [plop boot manager]("http://www.google.com/url?sa=t&source=web&cd=1&sqi=2&ved=0CCAQFjAA&url=http%3A%2F%2Fwww.plop.at%2Fen%2Fbootmanager.html&rct=j&q=plop%20boot%20manager&ei=G68STqX6F8_3sgaxhOnmDg&usg=AFQjCNHJvJJto1AIrG64vCU4Dolys0UbnQ&cad=rja"), it can also be used from a CD, so no actual install is needed.
