### Post by rsambuca on 2008-01-16
If you have a Windows License and installation CD's, you can use either [VirtualBox ]("http://www.virtualbox.org/")or VMWare.  Both of these will allow you to run a virtual OS inside of Ubuntu at the same time.  Keep in mind that you need a decent computer to do this since both systems will be running at the same time.  Another alternative if it is just a couple of programs that you need from Windows, is to use Wine, which allows some Windows programs to be run directly in linux.  You can check out the [Wine website]("http://www.winehq.org/") to see if your programs are compatible or not.
