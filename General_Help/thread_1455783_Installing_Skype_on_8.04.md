[CENTER][SIZE=4]**[COLOR=RoyalBlue]Installing Skype on Ubuntu 8.04 Hardy Heron[/COLOR]**[/SIZE]

[LEFT][SIZE=2]I noticed that many people, as well as me, want to install skype on 8.04. I found many tutorials with lots of dependences and terminal commands. So i found a better solution how to install.

1. **Go** to Skype's download site: [Latest 2.1 version - click here]("http://www.skype.com/intl/en/download/skype/linux/choose/")
2. **Download** Static (5th type, little skype cube), i didn't test any others so maybe some of them work
3. **Save** .tar.bz2 (26,3 MB)
4. **Extract**
5. **Open** the folder and **run** Skype.exe
6. Done, just type your username and password

I'm using this version, i had no problems with chat and calling. Also options and pop-ups work. **Post any questions, i'll try to help**
