Firstly, Ubuntu 11.10 will run fine on those specifications. I have a similar machine (slightly weaker CPU, even) with 11.10 and 12.04 beta.

Secondly, installing an older Ubuntu is rarely a good idea. There's only 18 months of support for most versions of Ubuntu, and six months in between versions. If you tried installing Ubuntu 10.10, for instance, you'd find yourself running out of support in just a few months. Unlike Windows, new versions of Ubuntu rarely bump up the system requirements. Stick with 11.10, or wait a month for 12.04.

On a computer like yours which seems a little behind the technological curve, you'll probably find everything will work perfectly well out-of-the-box without you even having to do anything.

Did you know that Ubuntu can be run from CD? If you insert the Ubuntu Desktop CD into your computer and boot your computer from it, you'll be able to run Ubuntu and see if your hardware is supported. When you're satisfied that your hardware works, you can double-click the Install icon on the desktop to install Ubuntu. If the wireless works when running Ubuntu from CD, it'll work in the installed Ubuntu too. If the wireless doesn't work from the CD, you may still be able to install a driver for it in the installed system.

You'll find that proprietary media files such as MP3, AVI and encrypted DVDs won't play, but it's fairly easy to add this support after installing Ubuntu.

Good luck, let us know how it goes. If you have any problems with anything just post a new thread and we'll see if we can help.
