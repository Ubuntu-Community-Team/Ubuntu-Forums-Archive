### Post by jjex22 on 2011-11-30
I'd suggest using the live CD, run in trial mode, mount your Ubuntu partition with

sudo mount /dev/sdax /mnt

(where x is your ubuntu partition number)

now you need to change the .conf back how it was, but remember the partition's now mounted at /mnt not /, so the path is now

sudo nano -w /mnt/etc/lightdm/lightdm.conf

and change the value to =false save with ctrl+o, exit with ctrl+x

then unmount ubuntu

sudo umount /dev/sdax

and restart - should now work. If you're running off a live cd all the sudo's shouldn't be necessary as I think you start as a root user, but as i always fix from a different partition, I've not checked this and it won't hurt - If you do this from another partition - say you're dual booting linux you will need the sudo's! 

Hope it helps! :)
