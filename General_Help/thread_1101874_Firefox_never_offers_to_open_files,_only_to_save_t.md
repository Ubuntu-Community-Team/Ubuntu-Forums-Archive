### Post by kanikilu on 2009-03-20
I would check your Firefox applications preferences:

Edit > Preferences > Applications

See if, for example, PDF is listed, and if so, see what the default action is set to. I think you can change it or at least remove it so next time you click on a PDF it should ask you what you want to do with it.
