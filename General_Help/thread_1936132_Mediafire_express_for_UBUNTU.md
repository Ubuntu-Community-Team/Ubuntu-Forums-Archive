### Post by ebozzz on 2012-04-14
Ok, to any who are following this thread there has been some improvement. Since my last post there have been two updates to MediaFire Express Beta. The DropZone still does not work like it does in Windows but there is now a way to drag & drop files for upload without accessing your account online. See below...

[LIST=1]
[*]Launch the MediaFire Express dashboard.
[*]Drag & drop the selected file onto the Share section of the dashboard.
[*]Upload should now complete successfully.
[/LIST]

While I am now able to upload using the above method, it does not provide an option to copy the URL assigned to the uploaded file to your clipboard. So, if you were intending to share that file with someone you would still need access your account online.
