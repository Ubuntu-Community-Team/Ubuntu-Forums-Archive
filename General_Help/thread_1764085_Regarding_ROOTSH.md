### Post by Karlchen on 2011-05-21
Hello, prasad2k5.

You can get **rootsh** here: [**rootsh**]("http://sourceforge.net/projects/rootsh/").
Yet, beware: to the best of my knowledge, **no .deb package** is available for Debian or Ubuntu, only .rpm packages which cannot be used to install on Ubuntu directly. On Ubuntu you will have to download the source code package, i.e. the tarball file [**rootsh-1.5.3.tar.gz**]("http://sourceforge.net/projects/rootsh/files/rootsh/1.5.3/rootsh-1.5.3.tar.gz/download") and install it following the instruction given in the file **install**. This file can be found inside the **rootsh-1.5.3** folder of the tarball file.

This article on using **sudo** and securing its usage with the help of **rootsh** on Debian (Ubuntu is Debian based) might be helpful: **[Providing root privileges for users Using SUDO]("http://www.debianadmin.com/providing-root-privileges-for-users-using-sudo.html")**.
