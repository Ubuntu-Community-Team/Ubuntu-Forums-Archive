### Post by TBABill on 2011-02-14
I'm not sure if you need Times New Roman in Ubuntu in order for it to work, but there is a file in Synaptic you can install for non-free fonts. I think you can search *msfonts *in Synaptic for the real name, but it installs the Microsoft fonts in Ubuntu.
