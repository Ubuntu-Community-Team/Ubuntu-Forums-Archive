### Post by krzysztofdanielewicz on 2010-01-06
I hope this short insturction helps somebody installing LDOCE 5 on Ubuntu 9.10 64 bits.

1. Install package 'ia32-libs' using apt-get or synaptic.
2. Start installation script in console using command ```
linux32 setup.sh
``` from Longman DVD.

That's all. LDOCE 5 works without any problem.

I post this because the Pearson Support Team claims that they are not going to support 64 bits distribution. There is an intresting Dictionaries Compatibility Table. [http://pearsonsupport.helpserve.com/index.php?_m=knowledgebase&_a=viewarticle&kbarticleid=115](http://pearsonsupport.helpserve.com/index.php?_m=knowledgebase&_a=viewarticle&kbarticleid=115)
