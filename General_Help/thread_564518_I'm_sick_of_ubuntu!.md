### Post by iposner on 2007-10-01
You do realise that burning an iso is not like burning a file to a disk:

If you've got a windows machine, you can download cdburn.exe which is contained within the windows server resource kit. This will burn an ISO image file correctly to a blank CD. (FYI, there is also a program called dvdburn.exe which does a similar thing for DVDs).

Use that to burn the image to disk from the command line.
