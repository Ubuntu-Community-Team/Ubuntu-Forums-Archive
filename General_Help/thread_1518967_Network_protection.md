### Post by The Cog on 2010-06-27
I don't know what you mean by being able to "see" other computers, or "access" them.

But anyway, a wireless network is no different to any other network in that all the pcs can communicate with each other. Ubuntu doesn't behave any different just because the network is wireless. 

A default Ubuntu install doesn't have any services running that other computers can connect to, so it cannot be "accessed" unless you install extra services like web server or file sharing services. But this is normal configuration, it's nothing to do with being on a wireless network.
