### Post by NilsE on 2007-10-22
Try looking here: [http://gnome-look.org](http://gnome-look.org)

You are looking for Metacity (frames and frame buttons), GTK (window buttons, panels etc.) and Icon themes.

The themes get installed in /usr/share/themes and the icons go in /usr/share/icons

You can also install Art Manager from the Applications/Add/Remove tool  to download and install some automatically from the  Gnome Art website.
