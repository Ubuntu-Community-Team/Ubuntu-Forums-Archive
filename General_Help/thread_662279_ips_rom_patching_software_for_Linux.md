### Post by dlegend on 2008-01-08
I found two solutions:

**a) JIPS** - [http://www.zophar.net/utilities/download/jips30.zip](http://www.zophar.net/utilities/download/jips30.zip)
>  JIPS in an IPS patcher which was written written in Java so it can run on Linux, Solaris, Windows and MacOS.

**b) IPS.pl** - [http://www.zophar.net/utilities/download/ips.txt](http://www.zophar.net/utilities/download/ips.txt)
>  This is a Perl script which is designed to apply an IPS patch to a ROM. It is a quick hack according to the author, but it seems to work OK for me. It is released under the terms of the GNU GPL. Shift-click on the link to save it, and then rename to end in .pl again, as it currently ends in .txt instead to avoid running the script ;).

One uses Java, the other uses Perl. You'll have to figure out how to get them to work as I've never used them before. I found them at this site: [http://www.zophar.net/utilities/patchutil.html](http://www.zophar.net/utilities/patchutil.html)

Another solution is to run the windows ips rom patching software through Wine. I read this application ( [http://www.romhacking.net/utils/240/](http://www.romhacking.net/utils/240/) ) called Lunar IPS will work in Wine.
