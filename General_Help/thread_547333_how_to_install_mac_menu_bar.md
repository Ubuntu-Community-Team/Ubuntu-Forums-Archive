### Post by shirilover on 2007-09-11
If you are running Feisty, the easiest method is to use the .deb files from the archive available here -> [http://rapidshare.com/files/27341099/libgtk2.0-0_2.10.11-0ubuntu3_macmenu1.0.11_1_.tar.gz](http://rapidshare.com/files/27341099/libgtk2.0-0_2.10.11-0ubuntu3_macmenu1.0.11_1_.tar.gz)

Keep in mind that this is a hack of libgtk and you would need to reapply the patches from the thread you referenced in the OP whenever a new version of libgtk comes out.
