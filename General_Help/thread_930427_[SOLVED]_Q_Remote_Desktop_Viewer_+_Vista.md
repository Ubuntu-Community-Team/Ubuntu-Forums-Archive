### Post by NoReflex on 2008-09-26
If you want to use Ubuntu's Remote Desktop Viewer(Vinagre) you must install a VNC server on your Windows machine. It's very easy to install and configure (I used TightVNC and UltraVNC - I prefer TightVNC).
You can also use Ubuntu's Terminal Server Client to connect to your Vista box. On your vista machine all you need to do is to enable Remote Desktop (I think you can find it in system properties).
Hope this helps you.

Best wishes,
NoReflex
