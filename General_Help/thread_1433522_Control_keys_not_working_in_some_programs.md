### Post by mrwawa on 2010-05-13
Hi all,

I finally found out that the problem was in VirtualBox.  Actually, I was the problem.  I have a Toshiba laptop which doesn't have a right Ctrl, so I set the host key to Ctrl (the only one) on my laptop.  Thus, VirtualBox keyboard shortcuts trumped the Linux shortcuts.
