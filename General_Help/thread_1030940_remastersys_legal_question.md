### Post by The Cog on 2009-02-04
As far as I can tell from a quick google, remastersys helps create Ubuntu derivatives. If he can make a derivative that looks like Vista only copying freely licenced stuff (themes etc.) then I think that would be fully legal. He should probably provide access to the source code to meet GPL license requirements, somehow.

If he wants to copy MS copyrighted code, and in making a copy of Vista itself, that would not be legal. But I don't think remastersys can do that.
