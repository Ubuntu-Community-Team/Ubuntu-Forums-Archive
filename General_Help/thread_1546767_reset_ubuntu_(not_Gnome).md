### Post by Revolutionary101 on 2010-08-05
Synaptic Package Manager (System>Administration>Synaptic Package Manager) does have a history of all the apps that you have installed. To see the history open it up and go to File>History. That should show you all the applications that you have installed, however uninstalling them all would be very time consuming.

Other than that I don't think there is a way you can reset Ubuntu. These are the times that I wish Ubuntu had a System Restore like Windows.
