How can a folder be made hidden in both Ubuntu as well as Windows XP?

---

### Post by zvacet on 2010-02-24
I you want to hide existing folder then for example let say folder name is **folder.** Type in terminal

```
mv folder .folder
```

 If you want to create hidden folder then

```
mkdir .folder
```
