### Post by whitesmith on 2013-09-28
Check Tools->Addons->Extensions. A lot of FF issues come from poorly written extensions. You can disabled them in this screen one by one to find the offender. As for Flash, it's notoriously slolw and buggy. I refuse to use it. You should first do this: select Help->RestartWithAdd-onsDisabled. If there is no speed improvement, look for problems elsewhere--cache that needs clearing, low RAM, etc.
