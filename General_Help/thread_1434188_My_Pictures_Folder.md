### Post by iflema on 2010-03-19
Generally the installers are safe, either wubi or a native install..... I mean unless you select the wrong partition to format there should be no issues.

For wubi install:
Im not familiar with the windows install option (wubi), but there should be no surprises. Just Carefully step through the installer and read all prompts. 

For a native install:
Of course backup data when partitioning, particularly when resizing partitions.

Carefully read the options available during the install., in particular the partitioning section of the installer.

You need to identify what your windows install partition(s) are and ensure you do not wipe (format) them.

Native install would be the best way to go..... all thats needed can be accomplished under the livecd environment, boot to the livecd desktop and if all is working well, simply run installer from the livecd desktop.
