### Post by Locutus_of_Borg on 2009-03-13
are you sure you do not have one?

i am pretty sure linux will not boot without it

you can make one though
```
sudo -i
nano /etc/fstab

```

the context is:

/dev/sda1		/		ext3		noatime		0 1
/dev/sda3               /home           ext3            noatime         0 0
/dev/sda2               none            swap            defaults        0 0

change to match what devices are mounted at / and /home (if any) and where your swap is
