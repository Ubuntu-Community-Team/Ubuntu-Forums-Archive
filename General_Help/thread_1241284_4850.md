### Post by automaton26 on 2009-08-17
I use this card and it works fine under my dual-boot kubuntu and vista. Putting it under heavy load gets the GPU temperature up to 80C, but that's what it is designed to cope with.

If you have a large, well-ventilated case with decent fans it should be fine. You could try adding extra PCI slot fans which might help.

Personally, I've underclocked both of the clocks on mine, using the "ATI Overdrive" mechanism, so it runs cooler than that now. I can still get the FPS that I need, for the games I want to play.

In Vista you can do this in the "ATI Catalyst Control Center" application, but in ubuntu you just use "aticonfig" commands in the Terminal window.

Driver installation is easy too: [http://ubuntuforums.org/showthread.php?t=1205418#7]("http://ubuntuforums.org/showthread.php?t=1205418#7")
