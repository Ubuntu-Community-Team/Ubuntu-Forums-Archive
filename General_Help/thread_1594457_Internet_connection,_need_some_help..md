### Post by JohnElway on 2010-10-12
The wireless in Ubuntu is just screwy. If my connection isn't working I usually right-click the network manager applet and uncheck Enable Wireless, wait a couple seconds, then re-enable it. Doing this once or twice usually fixes any problems I have.
