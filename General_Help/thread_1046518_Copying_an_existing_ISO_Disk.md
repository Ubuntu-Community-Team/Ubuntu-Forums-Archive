### Post by jerome1232 on 2009-01-21
Yes it can work, You can save an image to your hard disk, then remove the cd, insert a blank one, and then it'll burn that image back to another cd.
If you look at my screen shot see how I told it to burn to image, and it's saving the image at /home/jeremy/brasero.iso.


When it's done you would select burn image to disc from the main menu, select that brasero.iso file and you will get an exact copy of the cd.

--edit-- you could also copy the image file to a flash drive but I don't see why you would when you can just burn it straight from Ubuntu.
