### Post by daengbo on 2007-11-12
Pull out that Samsung and try it again with Rhythmbox. The MTP protocol (which WMP uses) is not a plugin for Rhythmbox. You used to be able to sync on the command line, but it was a PITA. Now, you can drag and drop to MTP-compliant players through Rhytmbox. Go to Edit -> Plugins and check the "Portable Players - MTP" box. No other configuration should be necessary. After that, simply plug your player in and it should be recognized.

I understand that Amarok can do it to, but I'm not sure how to set it up.

If the player still doesn't work, I suggest the iRiver Clix. It works with the MTP plugin and has a [movie converter]("apt:iriverter"). Click the link to install it.

Best of luck.
