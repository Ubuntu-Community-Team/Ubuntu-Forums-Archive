### Post by Praill on 2007-05-29
Atari 2600 - stella. Its in the repos, has a gui.

NES - tuxnes.. no gui but theres a post in these forums somewhere that shows you how to make a script that uses a gui file chooser to run with the emulator. It's written for fceu but can easily be substituted to work with tuxnes (tuxnes actually has sound.. fceu doesnt)

SNES - snes9x (backend) gsnes9x (frontend gui)  -- both in the repos.

N64 - mupen64 - not in the repos but easy to find

Gameboy Advance - vgbaexpress (i think) is in the repos. But I use NO$GBA windows client through wine because it works better.

Nintendo DS - Theres an emulator called iDeaS that is the only one available for commercial roms. It works OK, but Ive only been able to get it working in windows (wont work correctly with wine). DeSmuME is another emulator that has a linux client but it only works with the homebrew roms right now.

PlayStation 1 - psemu --repos (might be a command line emulator... if it is theres a gui for it in the repos too)


As you can tell Im a nintendo fan :)
Thats all I can remember right now. If i wasnt at work Id look into it deeper for you.

Good Luck.
