### Post by jlh on 2007-12-30
Solving my own question, 

Open a document and go to Format>AutoFormat, and make sure "While typing" is checked.

What I'm not sure I've figured out is whether that check is saved as the default for the next time the App is launched.  If it is not, there must be some other way to do so.
