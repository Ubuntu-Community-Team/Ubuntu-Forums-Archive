### Post by Nano_ext3 on 2009-03-07
Hi!

I suggest you visit [http://www.gnome-look.org/](http://www.gnome-look.org/) and go to the "GTK 2.X" section on the left hand side.  Download any theme from there you wish, some require additional configurations that the uploader will describe for you if needed, but all you need to do is load up the "Appearance" app in your preferences panel and navigate to the screen for the window appearance.  Click the button that says install, and choose the archive you just downloaded.  It should then appear as an option.  If there is multiple skins for that theme, select the theme and hit customize.  From there you can change the different options for that theme.  

Note: you should not have to install any GTK engines, just make sure Ubuntu is updated fully by doing a "sudo apt-get update" and "sudo apt-get upgrade" in the Terminal application.



Hope this helps!

Cheers,

-nano
