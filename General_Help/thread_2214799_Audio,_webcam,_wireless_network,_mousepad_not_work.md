### Post by deadflowr on 2014-04-03
I think you/we should determine if it is a software issue or a hardware issue.

If you have the Lubuntu installation disk still, try booting it, and when selecting whether to install or try, click on the try option.
This will boot into what is known as a live session.
If Everything works(or at least those things that should work without needed extra stuff; the keyboard,mouse) then it's probably a software issue.
If those things that should work don't even work, then it's probably a hardware issue.
We can probably help resolve a software issue, but a hardware issue will be a little bit more difficult.
