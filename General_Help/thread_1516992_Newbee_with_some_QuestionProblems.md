### Post by Pollox on 2010-06-25
On a couple of the questions (perhaps someone else could help with the rest)

1) Gnome and KDE are desktop environments.  Ubuntu is an operating system that uses Gnome.  Kubuntu is just Ubuntu using KDE instead of Gnome.  They come with different programs installed by default, but they're the same OS and you can install the same programs on each.  I would not recommend installing Kubuntu if you want to use Gnome and not KDE, although you can install gnome and remove KDE on an existing installation.

3) The extra applets are right where you are looking, it's probably just not apparent which ones are newly added.
