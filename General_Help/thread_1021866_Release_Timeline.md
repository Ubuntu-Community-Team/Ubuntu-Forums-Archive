Ubuntu version have very interesting codes eg, Hardy Heron 8.04, Intrepid Ibex 8.10 etc. Here the code represents the year/month of release. eg Hardy was released on April (04) 2008 (8 ) Similarly Intrepid on October 2008. Ubuntu follows a six month cycle so the release is slated on April 2009 and its code is 9.04 (now u know why).

However, Hardy is a LTS, which it will be supported for a longer time than the other versions. The current release of Hardy is 8.04.1 and the next is 8.04.2 which is slated to release on 1/1/2009.
