### Post by Tim Sharitt on 2008-07-21
Boot into the LiveCd. Select System > Administration > Partition Editor from the menu bar. You should be able to delete the windows partition and resize the Ubuntu partition to fill the free space. If the windows partition is first on the drive, you may have to move the ubuntu partition over to it's place before resizing.
