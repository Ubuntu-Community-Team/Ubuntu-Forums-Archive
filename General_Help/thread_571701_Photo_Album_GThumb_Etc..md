Hi all

Just wanted to express my experience with the photo album programs

The best thing I've found is when using GThumb and importing images the following seems best for me. the import to destination feature does not appear me to import to a destination on my server, so I basically select the images and select move to and then select my location and that works well.

Delete images from camera, then unmount and disconnect to save battery life.

For getting the images you wish to import/edit back into Gthumb once you've moved them from the camera to your desired folder.
Go to your folder with the images And then right click and select your images. You can hold the ctrl button down to select multiple images. then right click then select (open with gthumb)
Now the images are in the Gthumb, From there you can crop using the alt-c button, and crop, then save, then hit next to continue with editing the next image.
Then, select all images again, but this time within the gthumb and import to a destination/folder or you can create web album and select designation etc.

Anyhow that is what is working for me for my ebay listings etc. and working very well.

Another thing to try is Select import and for destination select (other) from the pulldown menu.
Then browse to your destination and even create a folder along the way.

Same thing when creating of the web albums, just select (other) for the destination again (from the pulldown menu) and browse to your location in which you want to save the web album. 

Anyhow thats all I know about it so far.

I hope this helps anyone who may have been wondering about this
