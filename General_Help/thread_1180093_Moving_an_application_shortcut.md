### Post by yonyz on 2009-06-06
I've installed 'Personal Brain' and it created a new category under the 'Applications' menu, called 'Other'.

How can I move that shortcut to another category and delete the 'Other' category?

---

### Post by Legace on 2009-06-06
Right-click on "Applications" and select Edit menu.
Select **Others** and drag **Personal Brain** it to Accessories (or where ever you want to put it into).
Then go back to others, and make sure its empty. If there are any entries there (including Personal Brain) make sure to uncheck them :)
