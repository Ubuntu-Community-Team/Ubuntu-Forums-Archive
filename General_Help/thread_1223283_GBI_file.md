### Post by darolu on 2009-07-26
There are double layer DVD's that are 7,92GiB (8,5GB) [Read it!]("http://en.wikipedia.org/wiki/DVD%2BR_DL"), you of course need a double layer capable burner and media.
Brasero (default Ubuntu dvd/cd recording program) should work fine to burn GBI files. Another option is GnomeBaker.
