I believe the only way short of editing code  would be to lose the shadow on the panel. Then it may not be as apparent though would still likely be there to some extent.
 This can be achieved by lowering the panel opacity in ccsm > Ubuntu Unity Plugin though the panel opacity is also affected by the Background color opacity which by default is at 0
So getting both the launcher & panel matched up can be a bit of a chore, slightly easier when using black as the Background color with some level of opacity.
