### Post by Quicky on 2008-07-21
Hi Shadowjester - I haven't requested permission from the designer of the icons to repackage them in a gnome-friendly iconset, so I'm reluctant to upload them as yet. Its actually very simple to create a theme however, all that needs to be done is for the icons to be renamed, and for a small amount of modification to be made to any other theme's icon definition file (I forget the name of the file uncfortunately and I'm using Windows as I write this...) There will be plenty of tutorials on Google regarding the creation of gnome iconsets.

I will say that because AiOlorWile's PNGs are such large images, that all you would need to do is rename them and place them in your new icon theme's 'scalable' folder - there is no need to create different sized icons, as I learnt after wasting a couple of hours one Sunday...

As for the link above, I'm a big fan of Preyor's desktop. That would look great with those British Library icons.
