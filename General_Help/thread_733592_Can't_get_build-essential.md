### Post by aysiu on 2008-03-24
It is not a standard part of Ubuntu.

If you're connected to the internet, go to System > Administration > Software Sources and uncheck the CD repository and check the rest of the repositories. When prompted to Reload, reload, and then go to System > Administration > Synaptic Package Manager and try to install *build-essential* again.
