### Post by openfly on 2009-09-22
Honestly you'll probably want to reinstall.

/ is the root directory in linux.

All other directory paths are attached to it, and while you can make it small you don't want it to be so small that it is constraining to you... especially as a new user.

Is this a dual boot system?

My suggestion is that you play close attention to the various options and interfaces when doing the disk partitioning during the install process.  

Also consider doing a single large partition on /.  With maybe a small swap partition... and not much else... at least to start.
