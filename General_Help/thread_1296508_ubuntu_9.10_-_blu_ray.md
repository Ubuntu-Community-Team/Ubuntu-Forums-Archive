### Post by Bachstelze on 2009-10-20
> **franar said:**
> hi all,
a simple question, does new ubuntu version supports blu ray playing, which is the reason for current ubuntu does not blu ray?

It doesn't work because it would be illegal in some jurisdictions. In order to legally support Blu-ray playback, you have to pay a license fee and probably sign a bunch of NDAs, which is not compatible with the Free Software spirit.

In other words, we would need to have a software company developing a Blu-ray playing application for Linux. I can't see that happening any time soon.
