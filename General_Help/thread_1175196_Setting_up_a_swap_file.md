### Post by guntherdouglas on 2009-05-31
Is there any way to set up a swap file in order to give more virtual ram to video display so that it will run smoother. I currently only have 64 megs of ram for my video display card I would like to make it think it has twice that. My system has 2 gigs of ram and my hard drive has 100 gigs. Any recommendations for a swap and how to set it up.

---

### Post by mikezila on 2009-05-31
There's no way to add video swap, only general use swap.

You may be able to dedicate some of your general memory for use by the video card, which may or may not help your performance.  The option to change that would be in your computer's BIOS.  Check your computer's manual or motherboard manual to see what options you have.
