### Post by scouser73 on 2010-10-21
Download the package from here; [http://sourceforge.net/projects/cupsdriverkodak/](http://sourceforge.net/projects/cupsdriverkodak/)

Save it to your desktop then extract it & set up the printer by choosing to install the ppd file which can be found inside the folder **usr/share/ppd/c2esp/Kodak_ESP_5xxx_Series.ppd**

Although the file states CUPS driver for Kodak ESP 5xxx AiO, it works on the ESP 3xxx perfectly.

I have installed this on my brothers Kodak ESP 3250, so can verify it.
