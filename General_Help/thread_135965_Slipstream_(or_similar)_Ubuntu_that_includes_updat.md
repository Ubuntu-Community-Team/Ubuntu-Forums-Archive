### Post by narnian99 on 2006-02-24
I was just wondering whether there was a project or somesuch to create for Ubuntu what in the Windows world would be a slipstreamed version of the OS. In Windows you can take a standard XP installation CD, copy it to a directory, and then apply service packs against that directory (using "update -s"). You then create a new bootable ISO based on that service-packed directory. (See [http://www.winsupersite.com/showcase/windowsxp_sp2_slipstream.asp]("http://www.winsupersite.com/showcase/windowsxp_sp2_slipstream.asp") if you are interested in seeing how this is done)

What I would want to do is take a standard stable Ubuntu installation, say Breezy, and copy to it all the updated .debs (with correct dependencies of course) and then create a new ISO. This would obviously best be distributed as a .jigdo much like say the current Dapper dailies - [http://cdimage.ubuntulinux.org/daily/current/]("http://cdimage.ubuntulinux.org/daily/current/") )

Anyone heard of this, or how one would do it? 

The advantage of course is that you could give a friend who didn't have a high-speed 'net connection a install disk that also had the current updates.
