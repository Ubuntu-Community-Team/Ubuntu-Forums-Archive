Hi

Skype automatically loads when I boot into Ubuntu 9.04.  I have checked that it is not listed under System>Preferences>Startup Applications, and I can't find any setting in Skype>Options to change whether Skype loads automatically or not.  I am running Skype 2.1 Beta.

Any suggestions?

Thanks
Wayne

### Post by running_rabbit07 on 2009-12-14
Go in your menus to System> Preferences> Startup Applications and find Skype, then uncheck the box.

---

### Post by howefield on 2009-12-14
> **Wtwine said:**
> Any suggestions?

Try this,...

Close all running applications then go to System > Preferences > Startup Applications, and click on the Options tab, check "Automatically remember running applications when logging out".

Close and reboot.

Do it again, only uncheck the box this time.
