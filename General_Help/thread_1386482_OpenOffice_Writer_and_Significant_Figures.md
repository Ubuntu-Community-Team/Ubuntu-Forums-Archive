### Post by JackRock on 2010-01-20
The FIRST Problem is that you allowed very little time for somebody to respond.

The second thing is you refer to "Writer" in the title, but then mention "cell" in the body.  Are you creating a table in Writer, or are you using Calc?

If inserting a table into Writer, highlight the cell, Right-Click on it, and uncheck "Number Recognition".

If using Spreadsheet/Calc, you can right-click on the cell, choose Format Cells, and the default tab is "Number".  Adjust the decimal places in that page.
