### Post by MoLE on 2007-06-07
There is a fairly simple way to do this - use nautilus (file manager) to navigate to a file type that you want to launch automatically with VLC.  Then right-click and go to the open-with tab.  You will be given at least two options - totem movie player and vlc.  Just click the radio button next to VLC, then click OK.
Next time you open a file of that type - nautilus will launch vlc instead of totem.
