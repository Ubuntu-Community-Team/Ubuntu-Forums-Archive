### Post by ferminnajar on 2010-06-05
I only want to post for express some of the (personal) frustration with open source.
I start installing ubuntu 9.04 who was dificult to tune. At the after some searches in forums for problems, it work good but the flash video doesnt work very well.

I try to upgrade to 9.10 from 9.04. Was a total mess. The computer doesnt boot well,ubuntu stayed in a black screen and i had to boot several times to enter to the system. I finally finish to erase the updated system and re-installing from scratch.

After several weeks, some patience, upgrades and search in forums i found the way of correct problems, and i re-install the graphics drivers ( seems like ubuntu have several problems with ATI cards ) and tuning. At least, i achieve to fix all my problems and had a functional ubuntu with amazing compiz effects and running even microsoft office 2007.
That was great !

Three days ago, i decide give a chance to the Lucyd Linx. I was insecure about the upgrade, but the main page claim to be a great upgrade, Ubuntu 10.04 runs faster in new computers bla bla bla.

Ok, i push the upgrade button. As the message says, litteraly, they took HOURS to upgrade, ( about 6 ), i feel more slow to start and, one more time, my graphics card didnt work. I said to my self "i knew i knew i knew !!!, why i did that !!!"   I try to solved the problem searching in forums. I try one, and the system crash.

I reinstall Karmic Koala, install the upgrades, install the ATI driver, and now everything is ok.

How i cant trust open source ? How can you say many things about Windows when open source is a mess and is not for simple users !!!

Sure, all the guys how think windows is a **** are geek-experts who have the time to spend solving problems of the OS, but i dont have that time. Windows has his own problems, but linux is years of distance for been to easy get all your system working. I think ubuntu is great, but not reliable enuf to uninstall my windows.

Why i still have Ubuntu ? Because im curios and i think is not so bad. If we compare to others linux, i think is a great distribution, more easy to use, install, more intuitive, etc.
But if we compare to windows ( not mention Mac ), there is  a lot of things to do (and fix)

In the company i work, we program in java and we use open source software to make products ( netbeans, etc ) but, the contradictory is, we use Windows XP as operative system, office 2003, exchange, etc.

Good software is not about been open/private, is about been made well (look firefox joomla eclipse etc )

Lesson learned: wait months before using a brand new release of software, specially if it is linux.
