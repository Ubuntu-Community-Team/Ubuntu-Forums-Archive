### Post by Mark Phelps on 2011-08-24
Try seeing if you can get a GRUB menu to display.  Hold down one of the SHIFT keys while booting. If that does eventually display a GRUB menu, try selecting the OTHER Win7 option and seeing if it still boots.

If it does, count yourself lucky ... and use the Win7 Backup feature to create and burn a Win7 Repair CD.  You're likely to need that later.
