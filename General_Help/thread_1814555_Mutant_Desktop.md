### Post by Theredbaron1834 on 2011-07-29
Doubtful. 90% of laptops have their cpu's soldered on. They don't use "standard" socket's, so you can't do it.

Now, what you might be able to do is to get a new CPU. Now they are really quite cheap. I bought me a dual core 2.9ghz am3 cpu for 60$ last year. The only problem is you would have to buy it used. I would say ebay would have it, if you care to look.


Also, you could buy a new mobo/cpu/mem combo pretty cheap. Since you already have a case/harddrive/psu, you can get some nice parts for under 100$ The cpu I was talking about, I got with a mobo and 2gb of ram for 130$. My computer now can run PS2 emu's at full speed. I also didn't get the cheapest. I went the cheapest for the newest at the time stuff. IE I had am3, the newest cpu slot for amd's, DDR3, and 3 pci-e slots. Then I bought the cheapest that would work for that.

If you want help with that, I would be glad to walk you through not only buying good cheap stuff (I like Neweeg), but also putting it all together. It really isn't hard at all. Just scary sounding.

### Post by Theredbaron1834 on 2011-07-30
Well, I don't know about old lappies, but I know that modern lappies use smaller versions of ram, thus it won't fit. Also, it might be soldered as well. Hard drives will fit, if you get an adaptor. That is all you can bring over as far as I know.

As for slow distro's. Well, the fastest I have seen is*_ [TinyCoreLinux]("http://ubuntuforums.org/tinycorelinux.com/")_*, but really it sucks. Then there is _*[DSL]("http://ubuntuforums.org/www.damnsmalllinux.org/")*_, my favorite tiny distro. If you want a full modern distro, then *_[Lubuntu]("http://ubuntuforums.org/lubuntu.net/")_* should work. Won't be as fast as the other 2, but it will be eaiser to use I think.

I would say you should try Lubuntu first. It is a ubuntu base, with a _*[lxde]("http://ubuntuforums.org/lxde.org/")*_ desktop. I have been thinking of moving to that myself, but it doesn't have as good a bluetooth ad2p support by default. I doubt that affects much people though. If you have low ram, ie less then 1gb, make your swap space at least 1gb, more if you can. Especially if you want to use Firefox. I love the fox, but it is a ram suck.
If you do go with Lubuntu and find you are always fully used ram, you could try changing the swappiness. You can find how to do that *_[Here]("https://help.ubuntu.com/community/SwapFaq")_*. It changes how fast the kernel swaps the mem to the swap space.

All that being said, I run Ubuntu, with gnome 2, on my eeepc 900, that has a 900mhz cpu, a bit less then yours. Though I do have 1gb of ram. Depending on how much ram you have, and how big a swap partition you can use, you might be able to run ubuntu, if you get an older one. The new one uses Unity as it's default, and it sucks/uses way to much cpu.


Ha, lots of links here, hope they help you out some.
