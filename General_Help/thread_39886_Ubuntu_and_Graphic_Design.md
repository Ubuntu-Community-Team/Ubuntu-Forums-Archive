### Post by Bob D. on 2005-06-06
None of these are, of course, direct one-for-one replacements but:

For Photoshop, try The Gimp; for Illustrator, try Inkscape or Sodipodi; for DreamWeaver, try NVU; for Quarkxpress, try Scribus

There are other alternatives for most of the ones I list above, but these should be good first places to look for F/OSS replacement apps for the ones you listed.

Good luck!

Bob

It's file format is the standard SVG,
and Adobe Illustrator can open SVG files if you ever have to provide files in Illustrator format.

Something simillar happened similar to me. A graphic shop wanted the file in FreeHand, so I opened with Illustrator Inkscape's SVG, saved in Illustrator's format, and opened in FreeHand, saving it again. In this process the file was allways correctly converted.
