### Post by sunny_nwho on 2005-11-30
Hi friends....I really appriciate the level of support that Ubuntu has in its repositiries......I have solved my issue of dual monitors just now.......

First of all u have to add all ur universe and multiverse repos...(check help on how to do this)

Start synaptics and search for i810...the results shud return with a switching program called i810switch just apply the changes 

now u can go to the terminal and 

type i810switch crt on lcd on        ---> for both crt and lcd to display

type i810switch crt on                ----> only for a crt display

type i810switch lcd on               -----> only for lcd display

a word of caution might be to check ur chipset before installing as far as i knw it supports 855GM so i guess many laptops have this chipset

but on the other hand I cannot switch directly using my FN + F5 key if any ones find out how to use that then please do tell me.......i have not even tries extending my display on other screen as it is not my requirement but I would be interested to know if someone wants to share...hope this is useful...
