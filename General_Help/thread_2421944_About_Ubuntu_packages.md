### Post by deadflowr on 2019-06-29
Repositories are a place that holds packages.
archive.ubuntu is one.

To create ppas you need to setup a launchpad account and follow the instructions on how to setup a ppa archive.


More on repositories:
[https://help.ubuntu.com/community/Repositories/Ubuntu]("https://help.ubuntu.com/community/Repositories/Ubuntu")

More on PPAs:
[https://help.launchpad.net/Packaging/PPA]("https://help.launchpad.net/Packaging/PPA")

Ubuntu software sources (repositories) contains 10s of thousands of packages.  All repositories use the .deb file format.  You can view all of these packages by using Synaptic Package Manager.
