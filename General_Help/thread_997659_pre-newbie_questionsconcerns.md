### Post by SPr on 2008-11-30
1. Dual boot Ubuntu and Windows: [https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot) Searching this forum, [http://help.ubuntu.com/community](http://help.ubuntu.com/community), [http://www.psychocats.net/ubuntu/](http://www.psychocats.net/ubuntu/) and Google will explain how to set up Windows and Ubuntu.

2. Both OSs need security updates so keeping them up to date is essential. Only one OS will be active at any one time so having two OSs (or three or four...) wont make the PC less secure.

3. Dual booting can be harder but the main worry if you only have one partition will be shrinking the Vista partition to make room for a partition for Ubuntu without messing up the Windows partition. Perhaps Wubi ([http://wubi-installer.org/](http://wubi-installer.org/)) will be better while you decide whether or not you want to keep Ubuntu.

### Post by Wight_Rhino on 2008-11-30
Hi Joe,

1)It depends on you. How you choose to install Ubuntu. You ***can*** install Ubuntu to your entire Hard Drive if you want, or you can keep Windows on a separate "partition". If you choose to keep Windows, you'll end up with a "Grub" screen on startup which basically allows you to select to boot into Windows or Ubuntu.

2)If you boot into Windows, any security problems you now have will still be there. If you choose to boot into Ubuntu then those Windows vulnerabilities will no longer be a factor.

3)You may want to do searches for "Dual Boot". Here's a good Ubuntu document on the subject; [[COLOR="Orange"]Windows Dual Boot[/COLOR]]("https://help.ubuntu.com/community/WindowsDualBoot") but there are also some good "independent" sites with walkthroughs on how to do it. Just google "Dual boot Windows Ubuntu" that should get you started!

4) (I know there wasn't a four). Just dive in Joe, remember we were all where you are at one point. Oh, and good luck!
