### Post by bgerlich on 2009-01-25
It's Netbook remix launcher, written bu Cannonical's Niel Patel using gtk and libclutter. It is a menu of sorts that takes the whole screen. In netbook remix almost all apps are maximized, so is the menu. The window list is represented by icons in the upper left corner, the first icon is the launcher that you can see on the desktop.

**EDIT:**
Almost forgot, the project was abandoned, Niel Patel is working on a new interface we may see in next Ubuntus.

The repos for hardy and intrepid can be found here:
[https://code.launchpad.net/netbook-remix](https://code.launchpad.net/netbook-remix)
