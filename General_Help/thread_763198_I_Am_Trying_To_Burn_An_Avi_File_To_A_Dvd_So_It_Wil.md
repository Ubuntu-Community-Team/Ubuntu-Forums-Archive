### Post by warp99 on 2008-04-23
> **adn258 said:**
> ANy ideas?  What should I use?

Well you have to use an authoring program in order to play the file in any DVD player. The file has to be convert to a mpeg, then burned to the DVD with a menu structure even if no menu is present. I would suggest using 'Q'DVD-Author which is a GUI front-end for DVD-Author

You could also use KDVDCreator which is also a GUI front-end for DVD-Author, but it depends on KDE4 or KDE3.5 with the KDE4 libraries installed:

[http://kde-apps.org/content/show.php/KDVDCreator?content=73014](http://kde-apps.org/content/show.php/KDVDCreator?content=73014)

KDVDCreator needs to be downloaded and compiled while 'Q' DVD-Author is available in the multiverse repository.
