### Post by lovinglinux on 2010-05-29
Check if you have libmoon installed and remove it. It is causing such problems in Firefox. If you don't have libmoon, then see General troubleshooting section of the tutorial mentioned above, then post your results.

Also check which version of Flash you have. Install [FLASH-AID](http://flash-aid-extension.blogspot.com) extension for Firefox. It will remove conflicting plugins and install the proper flash version according to your browser architecture.
