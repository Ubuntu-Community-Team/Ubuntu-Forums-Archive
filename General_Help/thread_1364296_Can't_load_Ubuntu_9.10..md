Looks like you just had a bad install disc.
You could install whatever you want on a computer without an operating system! (Not to point fingers, but how do you think they get one on there in the first place! :D )

If you have access to another working computer, try and burn the .iso to the cd again. When you've finished that, pop the disc into your computer and turn it on (you might have to tell it to boot to the cd, a google search for your manufacturer's boot options/boot menu will easily help you there). You might want to test the cd for defects before you go any further. If i remember correctly, its the fourth option on the list you see when you boot up.

Once you get the cd loaded you can either run Ubuntu from the cd without changing anything on your system or you could install it.

You had said that you wanted to wipe the drive and use it full Ubuntu, so when you install, just select that option, and away you go!

Hope this helps! Merry Christmas!
