### Post by Mark Phelps on 2011-08-24
First of all, this is a FORUM.  Problems and answers are shared with the community -- so others can benefit from that.  So, we discourage folks from using email.  You should edit your original post to remove the email address -- to save you from being attached by SPAMBots.

Second, Linux is not Windows, in that, you don't start loading up drivers manually as soon as you install the OS.  The Ubuntu installer does a hardware detection pass, and using the repositories, downloads and installs the necessary drivers.

It would be better for you to tell us WHY you want to load drivers, since there's likely to be an underlying problem you're trying to fix by loading drivers.

### Post by Mark Phelps on 2011-08-24
Drivers are installed automatically as a byproduct of installing ubuntu.

If a device is not working, then post back here with detail information about the device -- make, model, etc.  Perhaps someone will be able to help with that.

If Wifi isn't working, again, you need to know the make and model of the Wifi device that doesn't work.  Once you know that, you should go to the Networking forum here and search using that information.

To find out details about your devices, open a terminal and enter "lspci".
