### Post by SoftPops on 2009-05-12
I get the same from time to time (it was all the time but seems to have settled a bit).
 
I don't know a permanent fix but if you press "F11" to go into fullscreen mode and then "F11" again the problem goes away for the Firefox session.
 
Hope this helps.
