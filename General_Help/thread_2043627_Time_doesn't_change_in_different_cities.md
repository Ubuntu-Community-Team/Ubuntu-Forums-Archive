### Post by efflandt on 2012-08-17
I have never noticed Windows automatically changing time for different timezones.  How would it have any clue what timezone you are in if you do not manually change it?

Windows cannot even tell if another operating system (Linux) already updated the CMOS clock/calender for daylight savings time if Windows was not running during the time shift.  If you were running Linux during the time shift and then boot Windows, you have to adjust the clock by an hour to get it back on track.
