### Post by mk1w86 on 2009-06-24
Is the printer already being shared out by XP?

If so then go to System >> Administration >> Printing and click on New to add a new printer.  Under Network Printer select Windows Printer via SAMBA and enter the details of the Windows share on the right.  After this select the printer make and model. :)
