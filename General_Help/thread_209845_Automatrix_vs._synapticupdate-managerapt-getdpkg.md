---

### Post by aysiu on 2006-07-05
Because oftentimes, for certain situations, Automatix is faster.

Also, there are a few things that Automatix installs that cannot be found in Synaptic Package Manager.

For some new users, it's actually easier to install Automatix than to add extra repositories.
