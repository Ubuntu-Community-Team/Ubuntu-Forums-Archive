### Post by Frogs Hair on 2013-06-14
After two upgrades the .mozilla may be corrupted. You could export your bookmarks and create a new folder. To create a new folder open home, use Ctrl + H to show hidden folders, locate and delete the .mozilla folder, copy-paste and run the following command in the terminal. ```
sudo apt-get install --reinstall firefox 
```
