### Post by luisromangz on 2008-04-03
Mono and VS2008 are different things. Mono is a runtime, and a set of tools, like the .Net framework plus the .Net SDK. VS2008 is an IDE. 

Monodevelop is the IDE for Mono in Linux, and unless it is under heavy development, it can't compare itself to VS. It is usable tough, in fact I am using it (version 1.0) for my last year's project for university, so you might find it useful.
