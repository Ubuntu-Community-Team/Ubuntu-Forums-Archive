### Post by Gerardeux on 2011-11-16
Hola, Getting the Sun Secure Desktop to work under Ubuntu is fairly easy, it has nothing to do with support or not. Just install Java in your browser and your done. I'm a beginner under Ubuntu but this one worked right away. Actually the Secure desktop works better and is faster under Ubuntu then it is under Windows. What I did was did a Google on "install java in firefox ubuntu" and chose a method that seemed easy to me.. In this case try for instance the 6 step method on eHOW. But I'm sure there is also plenty of documentation in the official Ubuntu documentation. Follow the steps, test Java in your browser and enter your address for the SSGD in the browser and it will start up.
