### Post by tfwo on 2004-11-07
You installed it as a regular user in your home directory, right? Then it should reside in /home/user/StarOffice5.2 or something like that. Go to that folder and run "soffice". You can create a shortcut for that script on either desktop or the panel. If you first installed it as root to /usr/local/StarOffice (AFAIK it's the default installation path), you should first run "setup" from there for the per-user installation and run as above.
I'm wondering why you might need StarOffice when there's OpenOffice built upon the same codebase?
