### Post by ShakaDoodle on 2011-03-06
My guess would be that the site is trying to run a script on Firefox that has been accidentally blocked somehow.  Perhaps it's trying to install a cookie that helps the site run on Firefox.  Check those settings & do a comparison with settings on Firefox on similar system that is working to check for differences.  If it uses flash perhaps 3rd party cookies are being blocked & are necessary for site to run - check settings on flash website.  Just some ideas to work with that have helped me to get some stubborn sites working.  Wishing you success for your Dad.
