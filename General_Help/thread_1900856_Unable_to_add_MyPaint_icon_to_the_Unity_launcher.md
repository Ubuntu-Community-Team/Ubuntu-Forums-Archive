Whenever I click on Dash home > More Apps > MyPaint, the program starts and runs normally. 

Also, I can drag and drop the MyPaint icon from Dash onto the Desktop, however I am unable to drag and drop the icon into the Unity launcher without leaving a copy on the Desktop.  If I move the icon from the Desktop to the Rubbish Bin then the icon disappears from the launcher.

How can I add the icon to the launcher, without leaving a copy on the Desktop?

Update

### Post by Bobhuber on 2011-12-27
Open up the program. This will create an icon in the launcher bar. Right click on the icon in the launcher bar and select keep in launcher.
