### Post by kernalkorn on 2013-03-21
How can I cause all file views,  including USB storage and devices,  to default to detailed lists rather than Icons?  Getting tired of doing the ctrl-2 key reset every time I plug in a thumb drive, etc.  Ubuntu 12.10 Unity  Thanks

---

### Post by solarghost on 2013-03-21
In Nautilus:

Click Edit -> Preferences -> View Tab

The "View new folders using: " option is what you are looking for.
