### Post by TheFu on 2017-04-03
There are 10+ ways to solve this. The easiest is to create a new userid and use that.  I'm only suggesting it because it is a way that I'm positive will work.

Keep the old one. You will need it for sudo stuff.

If you feel adventurous, you can change the new userid into the old one, but this is a non-trivial and will require using alternate boot media.
