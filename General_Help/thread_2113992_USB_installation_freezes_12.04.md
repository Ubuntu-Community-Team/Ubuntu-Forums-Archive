### Post by jannk on 2013-02-08
Try format the USB stick and download the 12.04 again. Repeat, and set you're USB stick as number one prior in bios. 

If not that works, maybe try another USB stick.

Another option could be to try and format the sectors in the usb (format 16) with BOOTICE. Had this problem myself and got it fixed with it.

Other than that maybe try some other USB ports on the laptop/pc.

*PS, i assume you are using the Universal usb installer if you're doing this on win.

If you're doing it in linux read [this.]("https://help.ubuntu.com/community/Installation/FromUSBStick")
