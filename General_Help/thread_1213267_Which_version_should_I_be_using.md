### Post by dark.generic on 2009-07-15
Hey!

I would suggest install the compiz-setings-manager to get started with a bit of eye candy!

Open Applications > Accessories > Terminal  and type the following

```
sudo apt-get install update
```
```
sudo apt-get install compizconfig-settings-manager
```

As for music player i prefer amarok
```
sudo apt-get install amarok
```

And other media such as videos ect Video lan player
```
sudo apt-get install vlc
```

### Post by 3rdalbum on 2009-07-15
> **MegaToTheMax said:**
> I'm using an AMD64 processor and running 9.04.  Would I be better off using an earlier version? It seems like there are still some kinks with 9.04 and AMD64.

What issues are you having? Earlier versions of Ubuntu were not as good with 64-bit; and of course if you use an earlier Ubuntu you're stuck with older software and a lower degree of hardware compatibility (if you install 8.04 or earlier, you can kiss goodbye the idea of mobile broadband)

Rhythmbox, Banshee and Songbird are good music managers for Linux. If you don't like them... well hell, there's about a hundred more you can try :-)

I'll add to what an earlier poster said about Wine: It's an amazing feat of engineering that Wine is as good as it is, but the majority of Windows programs do not run on it. Switching to Linux and then trying to use Windows programs for everything is like the tourist who went to Japan and spent all his time at Western hotels and ate all his meals at McDonalds: He didn't get much of a Japanese experience.
