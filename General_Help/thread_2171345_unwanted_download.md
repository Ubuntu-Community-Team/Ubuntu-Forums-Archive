### Post by dhunt84971 on 2013-08-30
I am going to ask the obvious question, but I am assuming that you have already rebooted the computer since the first time you saw the dialog box?

Another question would be, if you start your computer and don't use any programs does it come back and how quickly?  If so, use top and see if you see any strange applications running.

If you definitely suspect your web browser, try using a different one and see if the problem goes away.  If it does check the installed extensions in your web browser or maybe turn them off.

Most of these suggestions are intended to identify the source of the problem.  I wish I could help in a more direct manner, but I have never seen this before.  At least not in Linux. :)

Good luck.
