### Post by Zimmer on 2009-04-09
Probably best to send the invite as an attachment (pdf file, maybe.)

You can create what you want using OpenOffice Presentation (Impress) .

Format your page to A5 and insert images, create text boxes etc. and export the resulting slide as a pdf file to attach to an email.
