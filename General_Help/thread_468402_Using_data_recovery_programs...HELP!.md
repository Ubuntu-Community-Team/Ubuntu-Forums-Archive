### Post by ramjet_1953 on 2007-06-08
At its best, data recovery is a "Black Art"

However, there a several data recovery software packages available for download on the Internet.

Most data recovery is done by a stand-alone disk that then accesses the sick hard drive.

Most of them are DOS-Based.

I have used a package called SpinRite for years and have had good success with it.

The author of the package is quite a character and has attracted quite a bit of flack from the more conservative elements within computing.

Nevertheless, SpinRite works well and will recover lost data on an accidentally formatted HDD, as long as it hasn't been subsequently over-written. 

It is also good at recovering corrupted data on a HDD that hasn't been formatted.

The download is a Windows exe, that runs fine under wine, which creates an iso image. You then burn this iso image to a CD or a floppy.

The demo will show you what can be recovered, but if you actually want to recover the data, you will need to fork over a few dollars.

Here is a link to the Gibson Research website:

[http://www.grc.com/spinrite.htm](http://www.grc.com/spinrite.htm)
