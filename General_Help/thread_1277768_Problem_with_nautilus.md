### Post by fooman on 2009-09-28
try the gnome configuration editor...press the alt-f2 keys.  when the run dialog box opens,  type "gconf-editor" (no quotes) into the box and click run.

when the editor opens,  go to apps > nautilus > preferences

scroll down on the right side and make sure that "start with location bar" is checked.

open another instance of nautilus and see if the location bar is back.
