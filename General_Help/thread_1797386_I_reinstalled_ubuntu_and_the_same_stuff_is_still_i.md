### Post by An Sanct on 2011-07-05
In Linux, multiple drives are seen as one space (unlike windows C:, D:, ...) So if you had several partitions, the trash on other partitions is still present and visible.

Also you can install Linux, with no need to format, if the drive is already formatted. You can also format the drive (one of the installation steps prompts you what to do about formatting.)
