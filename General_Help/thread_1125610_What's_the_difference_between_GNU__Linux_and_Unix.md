### Post by 3Miro on 2009-04-14
Unix is an operating system created in late 60s early 70s, or more specifically the kernel (core) of an operating system.

Linux was created in early 90s, deriving from Unix, but rewritten. There are many modernizations and improvements in Linux and some traditionally Unix systems (like BSD) are now taking ideas from Linux. Linux is actually just the core of the system.

GNU is the license that everything in GNU/Linux is created under. Look up Richard Stallman.

GNU/Linux refers to the entire system: The Linux kernel, the Xorg server, the desktop (Gnome/KDE/Xfe), all the apps that come with it.
