### Post by freqhack on 2008-10-25
I'm assuming that you have a bottom panel showing.

If so, right-click on it and select 'Add to panel'

Then select 'Window List'. It should be about 3rd from the bottom.

This will display all the programs you have open.

hope this helps.

No worries, I do the same sort of thing all the time.

It kills all the desktop and appearance settings (and possibly some more I don't realise) and put them all it back to their default. That way it looks like a fresh install without having to reinstall any apps.
