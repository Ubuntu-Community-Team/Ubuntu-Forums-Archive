### Post by z_diver on 2006-04-21
[QUOTE=ampop]What's the better way to do a Ubuntu System Backup?
There is any program to do a disk image? I allready tried with Partition Magic 8, but isn't possible.[/QUOTE]
I like [G4L]("http://freshmeat.net/projects/g4l/") which stands for Ghost 4 Linux.  It's a bootable CD that can copy ANY type of filesystem(yes ntfs is included) either locally or to/from a network location.

This is REALLY handy for backing up laptops.  Start the laptop and another machine with 2 of these CDs and you can ghost the laptop's disk right over the network, no screwdrivers needed.

Developement looks pretty active.  It's at version .19 as of March 06.
