### Post by jmszr on 2010-01-17
Sapien_Geeek,

Right click on an empty space on top panel > click on Add to Panel > scroll down to "System Monitor" > Add or drag & drop to top panel.
I think that's the one you're looking for.

Edit: there is also Htop:
```
sudo apt-get install htop
```

Htop can be quite useful.
