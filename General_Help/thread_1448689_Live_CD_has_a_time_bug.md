### Post by KMilburn on 2010-04-06
When using an Ubuntu disk in the live mode it changes the computer clock to Universal Time.  This has been observed on all Ubuntu variations.  A large number of people do not use GMT as their setting.

Since new comers expect the usage of a live disk NOT to make changes to their system, this may cause a bad experience.
