### Post by Quarkrad on 2022-08-16
If I enter **caja trash://** as a terminal command I open the trash folder; if I click on the trash icon in the caja file manager window the trash folder opens - as per normal operation. However, when I click on the (bottom) panel trash icon I open VLC.  If I remove the trash icon from the (bottom) panel and then replace it with a new on via 'add to panel' - the same thing happens.  I launch VLC.  Any guidance on how to correct this will be much appreciated.

---

### Post by Quarkrad on 2022-08-17
I have been looking for a *command line/under the hood* solution but in the end it was a GUI that saved me.  Like all things so simple in the end .....

Open *Control Center* and then click on *Preferred Applications*.  As you can see on the attached - my File Manager was associated with VLC Media Player (no idea how or when this happened?).  I changed the association to my file manager to Caja and now it works.  Launching my gui trash can now opens the trash folder, not VLC.
