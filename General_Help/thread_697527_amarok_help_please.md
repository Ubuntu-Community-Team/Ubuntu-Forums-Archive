### Post by logos34 on 2008-02-15
use the [xine engine]("https://help.ubuntu.com/community/Amarok")--plays just about everything

sudo apt-get install amarok amarok-xine

for mp3 you'll need libxine-extracodecs or libxine1-ffmpeg - read this:
[https://help.ubuntu.com/community/RestrictedFormats/MP3#head-eea6f341d3c611167270106a187907c2a23e08e2](https://help.ubuntu.com/community/RestrictedFormats/MP3#head-eea6f341d3c611167270106a187907c2a23e08e2)
