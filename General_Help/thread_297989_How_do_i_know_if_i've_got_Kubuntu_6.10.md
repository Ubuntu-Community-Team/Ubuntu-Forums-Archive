### Post by ciscosurfer on 2006-11-12
If you were using 6.10 with Gnome and install kubuntu-desktop (to get KDE) then you are running 6.10 when you switch to KDE.  Be sure to upgrade to be sure you have all the necessary packages.  But other than that, you're fine and are running 6.10 when using KDE.
