### Post by vis.15 on 2010-10-26
Hi 10Zav01, Konversation does use scripts. The best place to put scripts is where Konversation puts it scripts, which is: /usr/share/kde4/apps/konversation/scripts/

Konversation does not have a script editor. This is because you can use almost any scripting language for scripting. Also, you can use GEdit.

I hope this helps you.
