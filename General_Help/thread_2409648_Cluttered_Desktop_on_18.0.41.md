### Post by QIII on 2019-01-05
Hello!

If I recall correctly, in 18.04 you can click the "down arrow" in the far upper right corner of the screen, then in the drop-down, click the "tools" icon to get to Settings.

That should show "DOCK" in the list to the left. From there you should be able to set Auto-Hide and icon size.

I use Kubuntu generally, but I did fiddle around with vanilla Ubuntu when 18.04 was released.

### Post by HermanAB on 2019-01-05
As the two posters above alluded, the default Ubuntu desktop system is Gnome and it is generally a huge waste of computing resources.  Gnome seems to be the preferred desktop system of marketing droids - Ooooh Shiny! - while anything else is better for real users.

There are many other desktop systems and all of them will work far better than Gnome on a netbook.  My personal favourite lightweight desktop system is XFCE (Xubuntu), while LXDE (Lubuntu) also has a good following.  I even use XFCE on OpenBSD on a netbook.

You can add XFCE to your current system and select it when you log out/in:

$ sudo apt install xfce4

With XFCE, your netbook will be a hundred times faster and easily configurable.
