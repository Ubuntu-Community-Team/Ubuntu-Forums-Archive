### Post by jcoles on 2013-03-06
Ubuntu One has suddenly dropped support for Tomboy Notes synchronization. 
Here's a quick guide to using Dropbox for this purpose. 

[LIST=1]
[*]Create a Notes folder in your Dropbox folder.
[*]In Tomboy Preferences, go to the Add-ins tab. Under Synchronization, make sure Local Directory Sync Service is enabled. Disable Web Sync Service.
[*]Go to the Synchronization tab. Click Clear. In Service, select Local Folder. In Folder Path, select the Notes folder. Select Save. Your notes will sync to that folder and Dropbox takes care of the rest. 
[*]Repeat the steps 2 and 3 on your other PCs (assuming they already have Dropbox installed).
[/LIST]
This is probably obvious to some of you. But it took me about 20 minutes to work this out and I know there must be others out there who need some help.
