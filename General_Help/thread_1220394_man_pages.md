### Post by michy99 on 2009-07-22
You should already have the man page for anything you have installed which has a man page. You can read it by entering this in a terminal:```
man command
```Substitute the command you want for the word command. For example:```
man evolution
``` will display the man page for evolution.

To exit a man page, press Q.

### Post by y-lee on 2009-07-22
michy99 is right the linux man pages are on your machine and can be accessed in a readable form from the command line or terminal. But if you really want a more useable  perhaps ya might want to look at the [Linux Documentation Project]("http://tldp.org/docs.html").

> A tarball containing an HTML-based compilation consisting of core man-pages, and several other sources, can be downloaded from the [man-html directory]("http://tldp.org/manpages/man-html").
