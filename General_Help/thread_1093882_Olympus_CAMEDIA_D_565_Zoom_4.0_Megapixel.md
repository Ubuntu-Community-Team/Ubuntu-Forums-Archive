### Post by Spaceman9 on 2009-05-16
I'm updating this thread so if some one else has the same question they'll know the answer. 

Linux can read any digital camera that can be pluged into a usb port. KDE, Gnome and XFce can see and mount any digital camera plugged into a usb port.

To copy the photos to a hard drive, floppy drive or usb stick you can use a file browser or Gwenview or Gthumb or ShowPhoto or FSpot or Digicam and there could be others I don't know about. 

You can also buy a "card reader" that plugs into a usb port if your computer doesn't come with one. With a card reader it doesn't matter what camera you buy or how old it is as long as it has a memory card there's a card reader somewhere out there that will read it's memory card. There are lots of cheap memory card readers on the big auction sites.
