### Post by roblloyd on 2008-05-17
The best download manager ive found for rapidshare with a premium account is modrapi. 

[Heres the link.]("http://m0ds-ubuntu.blogspot.com/2008/01/en-rapidsharecom-download-manager-for.html")

Just follow the instructions and it works like a charm. The way i use it is to use the same file every time for the links. onece you have a load of files you want to download, just make an alias to send all of the commands, pretty sweet really.
