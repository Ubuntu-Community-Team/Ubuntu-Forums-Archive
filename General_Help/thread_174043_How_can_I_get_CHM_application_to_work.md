### Post by matthew on 2006-05-11
I use xchm and it works quite well. It's available in the Ubuntu repositories.

(Just in case you or someone else reading this needs further instruction)
Since you seem to be using Gnome, using the menu select Applications->Accessories->Terminal
At the command line that comes up type```
sudo apt-get install xchm
```
Enter your password when asked (it won't appear on the screen).

Then it will be installed. You can use it by clicking on any chm file in your file browser (Nautilus) or by typing xchm at the command prompt in the terminal.

Enjoy!
