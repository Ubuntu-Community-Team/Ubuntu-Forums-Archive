### Post by DuckHook on 2019-04-27
A KDE/Kubuntu expert might be of more help to you, but AFAIK, there is a significant difference between the Qt environment of Dolphin versus the Gtk environment of Nautilus. You are trying to run a Kubuntu app in a non-Kubuntu environment. Since you are missing the underlying infrastructure, I am not surprised that certain displays and behaviours are wonky.

I see three options:

[LIST=1]
[*]You continue trying to fit the square peg that is Dolphin into the round hole that is vanilla Ubuntu (with enough fiddling, I suppose anything is possible, but I can't be of any help to you there),
[*]You install and run Kubuntu in lieu of Ubuntu,
[*]If you really can't stand Nautilus, you can try an entirely different file manager that is still Gtk-based like Nemo, PCManFM, or Thunar.
[/LIST]
