### Post by Dylan1473 on 2012-08-02
Backtrack is a pretty advanced distro and it's unlikely you can install it within Windows easily unless you actually use a virtual machine. It might be possible through more complex methods but I wouldn't know about that.

A quick Google search tells of something called mint4win for Linux Mint, I'm not sure if it's still applicable. I'd suggest searching around and seeing if you can find it/if it's for you. It's also worth noting that you can also use Wubi to try out other Ubuntu flavours or, since you presumably already have it installed, just download the other major desktop environments (XFCE used in Xubuntu, KDE used in Kubuntu, GNOME, etc - I'm personally using the _[Enlightenment]("http://www.enlightenment.org/")_ window manager and really liking it).

EDIT: I forgot to include _[this link]("http://usbtux.hostzi.com/mint4win")_ earlier, it's a guide to installing Linux Mint using mint4win, plus a link to the actual installer. I know it's not exactly what you were looking for, but if you're interested. Additionally, if you want to try Backtrack on Windows and don't mind a slightly less elegant solution you can use a virtual machine through a program like the free and open source _[VirtualBox]("https://www.virtualbox.org/")_. You'll just need to _[download an ISO for BackTrack]("http://www.backtrack-linux.org/downloads/")_ (a torrent will probably be faster and less stressful on there servers, they'll give you a link) and let VirtualBox walk you through installing everything, it's not too hard.
