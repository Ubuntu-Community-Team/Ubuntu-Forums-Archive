### Post by CaptainMark on 2013-03-06
Just hover the mouse over a plugin in the list to get a tooltip about it, if you hover over the magic 8 ball plugin you'll see it says to type in command /8ball into the text box. Some plugins have their own options, if they do then the "Configure Plugin" button will light up when you single click the plugin name
