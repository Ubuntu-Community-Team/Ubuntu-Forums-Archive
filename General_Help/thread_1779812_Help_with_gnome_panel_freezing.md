If I needed to run that many windows applications, it would be better probably to install VMware and run them from inside of there instead.  Don't use Wine if you need to use them all-the-time. Wine is great for occasional application usage -imo.   Installing VMware is really easy too:

[http://cinderbox.net/2011/06/08/how-to-run-windows-in-ubuntu-with-vmware-player/](http://cinderbox.net/2011/06/08/how-to-run-windows-in-ubuntu-with-vmware-player/)

Just my suggestion. :)
