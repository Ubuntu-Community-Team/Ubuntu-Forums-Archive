### Post by cariboo on 2010-06-19
The default behavior has changed in Lucid, If you go to System->Preferences->Passwords & Encryption Keys, You'll notice by default the password is set to login, which is why it didn't ask for a password when you logged in. If you must use auto login, just create a blank password for the keyring and you won't be asked for a password again.
