### Post by clevertomato on 2010-01-26
If it happens in both OSes... hmmm... that would make me begin to wonder if it's a hardware problem. Have you run the Dell Diagnostics that come with the PC to check all aspects of the hardware? If you call tech support that's the first thing they'd probably have you do. If you no longer have the Dell Diagnostics on your drive, you should be able to boot from the Dell Diagnostics CD and run them from there.

With the PC off, hold down the Fn key while pressing the power button to turn on the PC. That should start it up and make it go right into the Dell Diagnostics... then grab a chair or get some coffee.
