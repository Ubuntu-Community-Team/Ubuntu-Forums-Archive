### Post by 505 on 2008-08-24
1. You don't need to delete Ubuntu. When you reinstall from the live CD, you have to option to choose the partition you want to install it to. I think you have to choose the advanced option in the installer part to get there. Just choose the correct partition.

3. You need to enable the visual effects. Go to System->Preferences->Appearance->Visual effects and set it to full. If you want to customise it (e.g. get the flames), install "compizconfig-settings-manager" and set your preferences under System->Preferences->Advanced desktop effect settings.

### Post by kpkeerthi on 2008-08-24
Enable multiverse under System -> Admin -> Software Sources

To install, run this command in a terminal window:
```
sudo apt-get install compizconfig-settings-manager
```
