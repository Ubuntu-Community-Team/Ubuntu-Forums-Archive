### Post by traditionalist on 2012-06-05
> **jinjin said:**
> i have gnome tweak tool and gnome color choser.  you can make the scrollbar bigger but there is not way in either of them to make the minimize, maxmise and close buttons bigger on windows....how do i do that?

Some of these things depend a lot on the basic theme you use if you want to customise them easily. I am not an expert on this, I have only been playing with this stuff a couple of weeks. However  I have found that you can influence the size of the buttons in various themes. I like fairly large clear buttons.  This is how I did it on this theme;

[[IMG]http://img99.imageshack.us/img99/5302/ubuntutweak001.th.png[/IMG]]("http://img99.imageshack.us/i/ubuntutweak001.png/")

the size of the buttons in this theme changes with the size of the top panel;

[[IMG]http://img651.imageshack.us/img651/4428/selection001iq.th.png[/IMG]]("http://img651.imageshack.us/i/selection001iq.png/")

If you change the size of the text in the top panel, here set to 30;

[[IMG]http://img43.imageshack.us/img43/692/selection001ga.th.png[/IMG]]("http://img43.imageshack.us/i/selection001ga.png/")

the control buttons change size to fit.

here set to 14;

[[IMG]http://img692.imageshack.us/img692/5302/ubuntutweak001.th.png[/IMG]]("http://img692.imageshack.us/i/ubuntutweak001.png/")

that is the easiest way to do it.

It doesn't matter what program you use to change font sizes in this case.  I have used "Ubuntu Tweak" for this.  You can get it here;

[http://ubuntu-tweak.com/](http://ubuntu-tweak.com/)
