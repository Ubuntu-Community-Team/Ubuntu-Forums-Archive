### Post by watchitman on 2008-05-15
You need to install compizconfig-settings-manager, which isn't installed by default, with Synaptic. Then simply click checkbox next to the Expo plugin and you're good to go. :)

### Post by bobblehat on 2008-05-15
You can install a compiz config tool with

sudo apt-get install compizconfig-settings-manager

(or search in synaptic).

You can then find the options for scale under Preferences -> Advanced Desktop Effects -> Window Management

(Crossed in the post :) )

### Post by adult swim on 2009-03-15
to get an effect similar to mac's expose, youll need to enable the scale effect.  it is in compizconfig-settings-manger and then go down to the window management section.  look for scale and check the box.  you may want to configure it, you can do that by click on scale and then choose "initiate window picker" by the mouse, if you want to do it with your mouse, and then make a mouse key combo to start it.
