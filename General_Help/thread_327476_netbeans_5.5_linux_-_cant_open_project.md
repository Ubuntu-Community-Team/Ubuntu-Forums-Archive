### Post by belda on 2006-12-29
i had a problem, i was installing netbeans 5.5 from the official installer run via sudo. Than when i ran the ide as normal user, i was unable to open any project, or even create new project. When i selected new project it created the folder, but opening it just clicked through. Just can't open project folder. also there was too few options in the new project dialog.

i tried installing it as normal user in my home dir, everything was working fine, so i unintalled it, installed it again and 
```
sudo chmod +r -R /opt/netbeans-5.5/
```
and everything works fine now
