### Post by louieb on 2009-09-26
The password is the one you used to set up your user at time of install. 

To change the password boot into recovery - command line.

```
passwd  <loginid>
```

Greater detail here [http://www.psychocats.net/ubuntu/resetpassword](http://www.psychocats.net/ubuntu/resetpassword)
