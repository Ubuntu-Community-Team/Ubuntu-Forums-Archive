### Post by Cheesehead on 2011-01-25
No, Brainstorm mods will likely reject that idea as a bug or a duplicate.

Generally, if you have a problem with existing code, it's considered a bug -including usability bugs and wishlist items-, and should be reported to the bug tracker at Launchpad.net

What you have described so far is a classic usability bug.

If you have a new idea or concept for a feature, system, app, new ways for packages to work together, etc...those belong on Brainstorm.
