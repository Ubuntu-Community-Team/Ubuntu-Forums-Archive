### Post by mörgæs on 2010-02-21
I don't think that you will get much help regarding Vista here, but if you want to save your documents from the sick Windows you can boot with an Ubuntu live CD and copy them to a USB stick. 

With the live CD you can still use the computer with a full Ubuntu, but you will not make more damage to the hard drive.

When you get Windows up and running again you can install Ubuntu next to it, so you can boot into Windows or Ubuntu as you please. That way you will always have a working system, also when Windows gets hit by a virus.
