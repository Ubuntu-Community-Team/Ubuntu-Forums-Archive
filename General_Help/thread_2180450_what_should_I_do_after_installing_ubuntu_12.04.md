### Post by grahammechanical on 2013-10-13
It used to be said that Ubuntu is for humans. We do not say it any more but the sentiment is still there. Some users of Linux like to do things the complicated way. Ubuntu is very much a distribution for ordinary users. And things have become simpler. When I started using Ubuntu five years a go there seemed to be 3 or 4 different utilities for doing the same thing. And the utilities were hard to understand and so to use, if we were a new user. 

You need to decide whether you want to continue using 12.04 until the next Long Term Support release comes out in April 2014 or upgrade to the latest release which is 13.10, to be released in a few days. Your system is set to notify you of the release of the next LTS. So, you will not receive notification of the release of 13.10. You will have to change a setting to be notified of a new six monthly release and with that notification an option to easily upgrade.

We can upgrade from 12.04 to 14.04 (when it comes out). But to upgrade from 12.04 to 13.10 we first need to upgrade to 13.04 and then upgrade again to 13.10. A little bit more complicated. This is how things work. We are advised to remain on 12.04 LTS if we want a stable OS. We should consider the six monthly releases as stable testing releases. Their life support is only 9 months. So, if you go on to 13.10 you will have to upgrade to 14.04 anyway. Best to wait another 7 months.

You can also adjust the time intervals that Ubuntu uses to check for updates. In 12:04 I think you do this in System Settings>Software Sources. I cannot say for sure as I have been using a development version of 13.10 for several months and the names of certain utilities have been changed. Software Sources is now called Software and Updates. An example of making things more understandable to new users

Regards.

Thanks for this reply.

In addition to this I want to mention two things, which I don't remember were there in dos:

1. Tab completion: You almost never have to type the full name or names of files and paths, and so on, in the command line. After entering a few letters most times the full name is suggested whith a press on the TAB-key, anyways until the next /.
2. the commands are remembered. A previous command can be re-entered by means of pressing the up-arrow, until you find it. By means of both the up- and down-arrows you can navigate through previous commands. Quite often you need the same command time and again, which is easy using the up-arrow.

These two points make using the command line very much easier than dos, and spare you a lot of time and frustration, and of course typing mistakes.
