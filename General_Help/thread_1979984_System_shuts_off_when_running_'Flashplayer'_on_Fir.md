### Post by Carborundum on 2012-05-14
If it only happens several minutes into the game, it doesn't sound like the problem is directly related to flash. Is the Lenovo ThinkCentre a laptop? If so, it could be overheating. Flash is known to be fantastically inefficient and CPU intensive.

As for ideas, my first suggestion would be to either upgrade or downgrade to a supported version of Ubuntu.

Well... Colour me dumb :oops:
