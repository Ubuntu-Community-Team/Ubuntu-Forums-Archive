### Post by ivze on 2008-05-23
The 64 bit distribution of Ubuntu is that for x86_64 processor architecture.
The difference lies in how low-level processor instructions look like. 64 bit processors are generally more capable (e.g. memory>4Gib), despite it is not critical yet. 
Any basic 64 bit processor supports 32 bit mode - means that can execute 32 bit instructions, what means that a 32 bit programms, OS distributions will work fine on 64 bit systems.
To see if your processor is 64 bit capable, see your processor manufacturer web pages.
