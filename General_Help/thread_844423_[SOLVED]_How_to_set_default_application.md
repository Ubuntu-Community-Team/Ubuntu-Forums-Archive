### Post by PoopyTheJ on 2008-06-29
I have deluge downloaded and I prefer to use that for torrents rather than the default bittorrent application that comes with Ubuntu. How do I set Deluge as the default app so that it always opens .torrent files when I click on them rather than the other bittorrent app?

---

### Post by manfer on 2008-06-29
Right click on a .torrent file and on menu click on properties.

Click on the Open With tab. A list of applications appears.

Select deluge as default application you want for torrent file type. If deluge is not on the list, use the Add button to add it to the list.

That could be applied to any file type.
