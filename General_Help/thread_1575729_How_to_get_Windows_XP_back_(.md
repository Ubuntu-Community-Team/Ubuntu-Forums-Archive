> Ok the problem is i do not have the discs. Is there any way of getting them? Dont worry my PC is 100% legal, i just do not know where the discs are.
Are they dear to buy?

Windows XP install discs are not cheap and I don't know whether you can still get them. Have a look on Ebay, but beware that you are not being sold pirated or counterfeit copies.

However, there is a potential problem. If your computer is a big-name manufacturer one, then the discs that came with it (if discs did come with it) would have been OEM image restore discs. These are designed to restore the system to its factory state with Windows pre-configured and drivers installed for your particular machine. If you reinstall from a Microsoft install disc, you will get a vanilla Windows and you will have to spend a frustrating hour or so hunting down all the drivers you need for your motherboard components and graphics card before it works properly.

One thing to check. Machines these days usually come with a restore partition from which you can reinstall Windows. In XP days restore discs were sometimes used instead. Are you sure you don't have a restore partition, or was it overwritten when you installed Ubuntu? Another question: when you installed Ubuntu, did you install it to the whole drive? Are you sure Windows is not there?

What is the make and model of your computer? Is there a (Microsoft) sticker of authenticity on the back with a 25-digit number?

### Post by coffeecat on 2010-09-18
> **Hawhaw1 said:**
> You know when you say 'recovery' disks? IS that the same as installation disks? It doesnt sound to me like it is. What will the recovery disk do if my Windows is completely wiped?

A manufacturer's recovery disc **is** an installation disc for that particular machine. Properly speaking, we should be saying 'restore disc'. A manufacturer like HP provides self-booting restore discs which restore the OS plus all needed drivers + the free software that came with the machine (which includes some bloatware) and all configured the way the manufacturer likes it - their own wallpaper, etc. The advantage over a Microsoft Windows installation CD is that once you're reinstalled the system from the restore discs, your machine will be as it was when it left the factory. Just a bit older.

But you **must, must, must ** use the restore discs specific to your particular machine. Any old HP discs may not do.
