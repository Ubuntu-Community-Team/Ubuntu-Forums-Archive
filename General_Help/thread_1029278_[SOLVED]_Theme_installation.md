### Post by Ivo Moelans on 2009-01-03
Unpack the file (select, right click, Extract Here). This will give you a directory *Macchiato Theme Pack*. Open an drag *Macchiato.tar.gz* to the *Appearance Preferences* window (System&#8594;Preferences&#8594;Appearance)
