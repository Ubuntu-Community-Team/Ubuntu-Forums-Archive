### Post by fridge2020 on 2020-06-28
Im new to Ubuntu (first setup day) and so far i think i have managed to get almost everything how i want it, except for this highly annoying detail.

I use a multi monitor setup as well as play games, which makes "Borderless full screen" in games the obvious option to use. I tried full screen but when minimizing the full screen application and trying to maximize it, it turns into a regular window with no apparent way back. Borderless fullscreen appears to work except that my bottom bar is placed on top.

No worries, i can use intellihide and problem solved.

Now comes the problem. With intellihide activated and even when bottom panel is visible, the bar itself becomes borderless making any full screen window use the entire height of the screen with the bar overlapping. Without intellihide this is not an issue and full screen windows (such as the browser) stretches to the bar and stops there.

See images below for explanation. Without intellihide, the bar places itself over the full screen application like below. With intellihide, the behaviour is solved in game, but instead is present when not in game for example using a browser.
