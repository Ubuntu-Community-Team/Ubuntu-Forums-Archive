### Post by northrup on 2010-01-01
Ultimately, it depends on the iPod.  Most of the new ones require iTunes, which doesn't have a Linux port.  These iPods have specific database formats and other oddities that only iTunes can successfully implement as of now.

If the iPod in question isn't latest-generation, you can look at the [Comparison of iPod managers]("http://en.wikipedia.org/wiki/Comparison_of_iPod_managers") Wikipedia page.

### Post by rudeboyskunk on 2010-01-01
Yes, there are several programs you can use.  If you just Google "ipod ubuntu" or "ipod linux" you should find tons.

I use Songbird, even though it is very slow and kinda bulky, but it's the only program I've found in which the "sync" command actually WORKS.  Exaile is good, Amarok (KDE) is good; alternatively you can use gtkpod.  I think Rhythmbox has ipod support, but I'm not 100% sure.

If you decide to use Songbird, you need to download the Songbird addon "ipod device support," or something along those lines.

Good luck!
