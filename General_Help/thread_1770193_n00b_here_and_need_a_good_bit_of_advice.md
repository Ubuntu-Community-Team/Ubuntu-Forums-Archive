### Post by Zenrex on 2011-05-28
To answer your question:

- Download the .torrent file like you normally would.
- Goto the menu 'Applications > Internet' and run "Transmission BitTorrent Client".
- Drag the torrent file into the main window.
- Toggle Ok.

The torrent will download ASAP.
