### Post by lindsay7 on 2009-06-11
From the Moneydance web site,   [http://moneydance.com/features](http://moneydance.com/features)


> will then install Moneydance.

Linux (x86 processor, does not include Java) This download assumes you already have Java installed. After the download is finished, double-click the moneydance_linux_x86.tar.gz file and extract its contents or run "tar xzf moneydance_linux_x86.tar.gz" from the command line, in the same directory as the downloaded file. You can then run Moneydance by double-clicking the "Moneydance" icon in the newly created folder.

Linux (x86 processor, includes Java) After the download is finished, double-click the moneydance_linux_x86wj.tar.gz file and extract its contents or run "tar xzf moneydance_linux_x86wj.tar.gz" from the command line, in the same directory as the downloaded file. You can then run Moneydance by double-clicking the "Moneydance" icon in the newly created folder.

Linux installer (x86 processor, includes Java) After the download is finished, double-click the moneydance_linux_x86wj.sh file or run "sh moneydance_linux_x86wj.sh" from the command line, in the same directory as the downloaded file.
