### Post by Aearenda on 2009-10-05
No, because the CPU is off in suspend, even if some other parts of the machine are still alive, and most of it is off in hibernate. You would need a network card able to talk directly to the disk drive, and if you're leaving the disk drive on, you might as well leave the computer on!
