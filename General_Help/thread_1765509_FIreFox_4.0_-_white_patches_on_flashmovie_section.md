### Post by lovinglinux on 2011-05-23
Get [Flash-Aid]("https://addons.mozilla.org/en-US/firefox/addon/flash-aid/versions/") 2.1.1 and run it. It will detect installed flash plugins, remove them and install the best option according to your system architecture and version. Additionally, it will apply some tweaks that should improve performance and fix some issues.
