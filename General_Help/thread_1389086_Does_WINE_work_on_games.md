### Post by flabdablet on 2010-01-23
There are as many answers to that question as there are games, unfortunately. Some work really well; some work a little bit; some fail horribly. Best place for specific answers would be the [Wine application database.]("http://appdb.winehq.org/") The [Codeweavers site]("http://www.codeweavers.com/compatibility/") is probably worth checking as well (Codeweavers makes Crossover, which is a commercial product based on Wine and offers better compatibility for some apps).
