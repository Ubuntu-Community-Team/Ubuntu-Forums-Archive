### Post by AlexDudko on 2011-08-10
No, you don't. You can freely do usual things while update process is running. But you have to restart processes, which were updated and of course you've got to restart computer to run new kernel if it was updated.

---

### Post by papibe on 2011-08-10
Not really. The only exception I would say is if you are updating an open GUI program like Firefox or Thunderbird. In that case I would recommend closing those applications.

Most underlying services are restarted as a part of the update (upgrade really) process.

Regards.
