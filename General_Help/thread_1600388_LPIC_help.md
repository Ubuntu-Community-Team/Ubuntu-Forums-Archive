### Post by amauk on 2010-10-19
I've got LPIC-1

To be honest, you don't need any specific books or anything, just a decent working knowledge of Linux
(being a Linux admin certification, it's all about the CLI - so nothing GUI related)

It's mostly multiple choice, and if you have a decent working knowledge, the questions are quite easy to answer

You'll need to know about both DEB & RPM package managers (not from a technical POV, but from an admin POV - installing, removing & querying packages via the CLI tools)

You'll need to know about Unix job control
inc. moving jobs from foreground to background and back again, plus other job operations, use of nohup for daemonising processes, etc.

You'll need to know about user management
Adding, modifying & deleting user accounts
Preventing users from logging into a running server

You'll need to know about general server maintenance
inc. log file management and common locations for different file types (executables, libraries, config files, etc.)

You'll need to know about the common CLI utilities> Which of these programs would you use to perform a "find & replace" operation on a text file?

a) grep
b) ping
c) sed
d) ldd
