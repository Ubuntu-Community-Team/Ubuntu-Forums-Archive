### Post by dabl on 2011-02-06
I think that computer is going to struggle to run a reasonable KDE desktop, and also a VM at the same time.  But I think you can do it, if you don't mind things running a little slow.

Win 98 is not a big memory hog, compared to the later versions. Try giving it 256M of memory.  If it works OK, and your Linux desktop is working, then power off the VM, and try setting the memory down to 228M, and run the VM again.
