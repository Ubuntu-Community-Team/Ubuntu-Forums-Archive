### Post by vasa1 on 2014-04-12
I poked around a bit and if a person is willing to sign into Google, it's possible to create one's own search engine to search sites of one's choice.

No need to know scripting or html :) just the links you wish to add.

This is the starting point: [http://www.google.com/cse/create/new](http://www.google.com/cse/create/new). And then you can choose to be provided with an url to your custom search engine.

I got this: [https://www.google.com/cse/publicurl?cx=000411765324082433246:98dbgx-jqz8](https://www.google.com/cse/publicurl?cx=000411765324082433246:98dbgx-jqz8)
which can then be used from any browser; I tested it on Xombrero where I'm not signed into Google and so I'm assuming anyone can use it.
So far, I added just a few sites and I don't know if there's an upper limit.

This is a screenshot of a search of "Openbox".
