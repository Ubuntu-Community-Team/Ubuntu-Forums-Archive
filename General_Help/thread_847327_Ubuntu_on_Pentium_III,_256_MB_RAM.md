### Post by avtolle on 2008-07-02
With 256mb RAM, that computer will never be a speed demon; while Ubuntu will run with 256mb RAM, you will need to use the alternate install disk (text based installer, but gives you a fully functional o/s with a GUI after it completes installation). For a bit better performance, Xubuntu, with the lighter DE will be a better choice for that computer, IMO.
