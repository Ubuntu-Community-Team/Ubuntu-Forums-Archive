### Post by Bapabooiee on 2009-03-13
Almost everyone here will probably definitely recommend using "Pidgin" - it's a client that supports pretty much every Messenger client out there. Be it MSN, AIM, Yahoo Messenger, and a whole crap-load of other ones I've never heard of.

But, I pretty much only use MSN. I've tried-out a few MSN clients, and the best one, hands-down, is "Emesene". It's a really nice, robust, and extensible client. It's also the most aetshtetically pleasing for me as well.

Summary: I'd recommend Emesene if you're only gonna use MSN. Otherwise, use Pidgin.
