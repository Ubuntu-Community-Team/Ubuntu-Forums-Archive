Your windows, ntfs partition, is /dev/sda1, not /dev/sda2 since /dev/sda2 is an extended partition.

I assume we are talking about the second harddrive, /dev/sdb1.

```
sudo mkdir /media/sdb1
sudo mount -t ntfs-3g /dev/sdb1 /media/sdb1
df -h
```
