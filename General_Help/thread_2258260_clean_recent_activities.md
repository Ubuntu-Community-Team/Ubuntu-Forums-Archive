### Post by slickymaster on 2014-12-26
> **sarower2 said:**
> thanks , i want to clean recently used files and application

[LIST=1]
[*]Click on System settings;
[*]Click the Security & Privacy icon under the 'Personal' section;
[*]Select the File & Applications tab;
[*]Click the 'Clear Usage Data...' button.
[/LIST]

Use the 'Record file and application usage' button switch if you want to limit which items are recorded, if any.
