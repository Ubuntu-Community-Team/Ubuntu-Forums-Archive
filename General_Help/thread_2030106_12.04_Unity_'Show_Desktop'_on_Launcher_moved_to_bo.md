### Post by mc4man on 2012-07-24
If you get tired of the keyboard method you could make your own show desktop icon & move as desired or add as a quicklist to an existing icon you keep at the top or so.

Moveable icons represent .desktops so there would be the limitation that you could only use once per 5 sec.'s or so

If so something like this
```
sudo apt-get install -y xdotool
```

```
gedit ~/.local/share/applications/showd.desktop
```
Name= & Icon= can be as desired, for many icons best to use full path/fullname
```
[Desktop Entry]
Version=1.0
Type=Application
Icon=user-desktop
Name=Show Desktop
Exec=xdotool key ctrl+super+d
```
Then browse to ~/.local/share/applications/ & drag the showd.desktop file onto launcher

Quicklist options aren't limited to once every 5 sec.'s or so though obviously not quite as handy
(though if desired the use of quicklists can reduce the number of launcher icons, I use a couple of Category based icons with 8 to 10 quicklists, like 'Utilities', 'Multimedia', ect.
