### Post by GhX6GZMB on 2022-04-30
You need to go to the software center in Ubuntu and look for "b43-fwcutter". Install it and run an update/upgrade (you may need a reboot). Then it should work.

Apart from that: move to 20.04 LTS ASAP. 16.04 is far too old.
And as you're talking older laptops: go for Lubuntu or Xubuntu instead. Much lighter and faster. Ubuntu is too heavy.
