### Post by oldtimer7777 on 2011-10-25
I'm just going to say what most old Ubuntu users are going to say about Wubi, and it is nothing against Wubi in general. 

But....

Wubi is like a bicycle with training wheels. Eventually they need to come off. 

Wubi isn't meant for really long-term usage.  Think of it more like a test-drive in a car. It would be wise at this point, if you want to keep using Ubuntu long-term to actually install Ubuntu to your harddrive.  First, make sure to backup everything you want to keep somewhere else.  And here is a guide to get you started.  Hopefully you have a relatively new USB Flash Thumb Drive that you can use to install Ubuntu 11.10:

[https://debianhelp.wordpress.com/2011/09/12/to-do-list-after-installing-ubuntu-11-10-aka-oneiric-ocelot/](https://debianhelp.wordpress.com/2011/09/12/to-do-list-after-installing-ubuntu-11-10-aka-oneiric-ocelot/)

Download a copy of Ubuntu 11.10 and use Ubuntu's Startup Disc Creator to format your thumb drive and copy the ISO file onto that Flash USB Stick.
