### Post by rraj.be on 2008-06-16
Just open synaptic package manager

System -> administration --> synaptic package manager

Select ```
Custom filters 
```in left pan

Now at the top of leeft pan you get a name called```
 BROKEN.
```

Just click it.

in right pan it will show you all broken packages and now you have to completely remove them all.
