### Post by zero2xiii on 2012-05-06
Hay,

Sure try this:

"alt + sysRq + S" then
"alt + sysRq + B"

And bob's your uncle.

For more information on what this does (aswell as some other usefull stuff) see [here]("http://en.wikipedia.org/wiki/Magic_SysRq_key")
