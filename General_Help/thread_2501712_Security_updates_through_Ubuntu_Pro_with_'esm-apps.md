### Post by grahammechanical on 2024-10-18
Think of that message as either an advert or as an invitation to get extra security updates. Extended Security Maintenance = ESM. Ubuntu Long Term Support (LTS) versions get five years support. With a Ubuntu pro subscription we get an additional five years support. Not only that these security fixes also apply to software applications that are in addition to the standard software applications that are normally supported. These are those packages that are in what is called the Universe repository.

How much does all this extra support cost? It is free for the user with up to five machines. If you want more than five machines covered you have to pay and the amount increases according to the increase in the number of machines you want covered.

Normally, we upgrade from one LTS release to the next every four to five years in order have a a fully security supported operating system. This offer postpones that upgrade for five more years. This can be useful for business users.

Read the full advert.

I think that it is also worth keeping in mind that although we are using a version of Ubuntu that is still within its five year support period, there are applications ( in Universe repository) that are not getting any security fixes. Ubuntu Pro takes care of that situation. That is why the advert is there in the terminal.
