### Post by Vakman on 2009-07-13
Itunes does not run under Ubuntu, you could try running it under WINE but that does not usually work well. So the short answer is no. If you want to run Windows in a virtual machine for Itunes then you could. Also in Vista's Control Panel under Regional & Keyboard settings or something similar to that you should be able to add and remove the keyboards you do not want or want.
