### Post by 3rdalbum on 2009-11-28
Ubuntu 9.04 can have different login window themes (GDM themes). It's done exactly the same way as in Ubuntu 8.04/8.10 (you don't specify which version you were previously using).

Ubuntu 9.10 uses a completely rewritten GDM so it's not possible to theme it yet without recompiling.
