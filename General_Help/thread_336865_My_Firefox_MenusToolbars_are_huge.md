This is a bug that only affects firefox.  [This link explains what's going on]("http://ubuntuforums.org/showthread.php?t=272139") and provides a .css file that you need to put into your firefox directory.  I've done the fix and it works perfectly.

Reducing the system font size does not help the problem (since you have to reduce the size to the point where fonts are fine in firefox, but unreadable in the rest of your programs).
