### Post by sandyd on 2016-01-01

1) Binary packages are listed here ->[https://launchpad.net/ubuntu/+source/broadcom-sta/6.30.223.141-1](https://launchpad.net/ubuntu/+source/broadcom-sta/6.30.223.141-1). DKMS is needed unless you want to be compiling broadcom-sta manually each time you install a new kernel. 

Also, I am assuming you are actually using broadcom-sta, not bcmwl-kernel-source. Differences can be seen here [https://help.ubuntu.com/community/WifiDocs/Driver/bcm43xx#Broadcom_STA_Wireless_driver_.28Proprietary.29](https://help.ubuntu.com/community/WifiDocs/Driver/bcm43xx#Broadcom_STA_Wireless_driver_.28Proprietary.29)
2) ```
apt-get install software-properties-common
```, add-apt-repository will be available after that
3) ```
apt-get install xorg
``` should install xorg without network manager, and just a minimal xorg.  Xorg does not install the DE so use something like openbox

```
