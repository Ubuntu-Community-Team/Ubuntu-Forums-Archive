### Post by Stefan_K on 2009-09-28
Hi,

if you don't want to use Synaptic or apt-get you could use the [MiKTeX package manager]("http://texblog.net/latex-archive/linux/mpm-miktex-package-manager/"). Alternatively you could install TeX Live 2008 or the upcoming [TeX Live 2009]("http://texblog.net/latex-archive/news/tex-live-2009-pretest/") with its package manager tlmgr.
If you want to know more about fulfilling dependencies nevertheless have a look at [Kile and TeX Live 2008 on Ubuntu Linux]("http://texblog.net/latex-archive/linux/kile-texlive-2008-equivs/").
