### Post by steveneddy on 2009-01-18
Open CCSM

System --> Preferences --> advanced Desktop Effects Settings

Find the section titled [B]Desktop
[/B]
select **Desktop Wall**

Select **Viewport Switch Preview**

**Show Viewport Switcher Preview** - uncheck
