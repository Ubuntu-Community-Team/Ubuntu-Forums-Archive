### Post by Laobi on 2012-03-25
> **mrag said:**
> And in general, is there a source to go to when looking for "similar" Win to Ubuntu software? That is, is there a place like Snapfiles.com but only for Ubuntu apps?


You can try these links for Ubuntu software, some of them list alternative Linux applications to common Win software:

[LIST]
[*][[COLOR="Blue"]https://apps.ubuntu.com/cat/[/COLOR]]("https://apps.ubuntu.com/cat/")  - Ubuntu Apps Directory, browse applications by category
[*][[COLOR="Blue"]http://www.linuxalt.com/[/COLOR]]("http://www.linuxalt.com/")  - The Linux Alternative Project, database with Windows software and the Linux equivalents and alternatives
[*][[COLOR="Blue"]http://portablelinuxapps.org/[/COLOR]]("http://portablelinuxapps.org/")  - Portable Linux Applications, just download, make executable and run
[*][[COLOR="Blue"]http://www.videohelp.com/tools/sections/linux-video-tools[/COLOR]]("http://www.videohelp.com/tools/sections/linux-video-tools")  - Linux video tools
[*][[COLOR="Blue"]http://www.makeuseof.com/pages/best-linux-software[/COLOR]]("http://www.makeuseof.com/pages/best-linux-software")  - Linux apps sorted by category
[*][[COLOR="Blue"]http://alternativeto.net/software/?profile=linux[/COLOR]]("http://alternativeto.net/software/?profile=linux")  - find alternative software
[*][[COLOR="Blue"]http://ubuntuguide.org/wiki/Alternatives[/COLOR]]("http://ubuntuguide.org/wiki/Alternatives")  - List of applications sorted by category
[*][[COLOR="Blue"]http://www.ubuntulinuxhelp.com/top-100-of-the-best-useful-opensource-applications/[/COLOR]]("http://www.ubuntulinuxhelp.com/top-100-of-the-best-useful-opensource-applications/")
[*][[COLOR="Blue"]http://www.osalt.com/[/COLOR]]("http://www.osalt.com/")  - Find open source software alternatives to well-known commercial software
[*][[COLOR="Blue"]http://www.linuxsoft.cz/en/[/COLOR]]("http://www.linuxsoft.cz/en/")  - Linux Software directory, a comprehensive list divided into categories
[*][[COLOR="Blue"]http://gnomefiles.org/[/COLOR]]("http://gnomefiles.org/")  - GnomeFiles, GTK+ software repository
[*][[COLOR="Blue"]http://freecode.com/[/COLOR]]("http://freecode.com/")  - Linux, Unix and cross-platform software
[/LIST]

But you should use official Ubuntu software repository whenever you can, because it is safer, easier to install and the software there is more adapted to Ubuntu.  I usually use these lists just to find an application and then install it from repositories using Synaptic package manager.
