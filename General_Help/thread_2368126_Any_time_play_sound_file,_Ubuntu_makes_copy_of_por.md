### Post by swarup on 2017-08-07
This is a bizarre problem! Around two months ago I started finding sound files in my home folder which I hadn't put there. Now I have realized just what is happening: Any time I a play sound file, Ubuntu makes a duplicate copy of the portion I played, and puts it in my home folder. If for example it is a one hour sound file and I play the first 20 seconds of it, then an mp3 file containing those 20 seconds will be created and placed in my home folder; this new file has the same name as the original sound file. I am running Ubuntu 14.04 and never had this issue until around two months ago. Any ideas how to stop this from happening?

### Post by swarup on 2017-08-07
You seem to be right on target! My default player is Audacious, and now I see that under the "output" options, the "stream" facility is selected and check-marked. I have unchecked that just now, and it seems that no more duplicate sound files are getting created. Great going! I will go on testing further, but as of now it appears you have given the solution. Will go ahead and mark this thread as solved  :-).
