### Post by DJ_Peng on 2008-09-26
While I can't speak specifically for a Palm Centrino I add files to my Palm OS4 devices by dropping the files on the gnome-pilot applet on my panel. Then when I sync up my PDA again the files are transferred over. You probably need to do something very similar with the Centrino.

If you're using J-Pilot rather than the panel applet to update your Centrino you add files via **File > Install**.
