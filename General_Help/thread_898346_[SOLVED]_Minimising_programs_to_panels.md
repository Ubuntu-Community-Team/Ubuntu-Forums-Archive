### Post by sameerds on 2008-08-23
> **PIHKAL said:**
> I've been messing around with the top panel and I've deleted the bit where programs like Amarok used to minimise to (the top right bit), so now when I minimise Amarok it isn't there but is still running. 
Does anyone know how to fix this?

The bit that you are looking for is an applet called the "Notification Area". To install that, right click on your panel, and select "Add to panel". In the list of applets shown, select "Notification Area" and click Add. This will add the area, but it is likely that applications that are already running will not show up in it. If that happens, quitting each application and starting it again might work.
