### Post by amcavoy on 2009-09-30
This is probably a stupid question, but how do you change the default programs used to open certain files?  For instance, whenever I click on a PDF in Ubuntu it automatically opens with Document Viewer; I would like to have it open automatically with Adobe (which is already installed, of course).

---

### Post by coldReactive on 2009-09-30
Right-click the file, go to the Open With Other Application... right-click menu option (if it's there.) Otherwise, go to the open with (or was it Applications) tab after clicking properties on the file (in Xubuntu, it's on the general tab as a drop down.)
