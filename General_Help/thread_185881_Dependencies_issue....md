### Post by Dr. Nick on 2006-06-01
If you are using ubuntu I dont know if I would get the debian versions. It may work in some cases but I pesonally would rater keep it a "clean" Ubuntu. You may encounter problems with conflicting versions by mixing them, also some of the "core" libs must be ubuntu else alot of stuff wont work. What dependencies do you need? Also check out [http://packages.ubuntu.com/dapper/]("http://packages.ubuntu.com/dapper/") and it will give you the dependencies for a program and let you download them.
