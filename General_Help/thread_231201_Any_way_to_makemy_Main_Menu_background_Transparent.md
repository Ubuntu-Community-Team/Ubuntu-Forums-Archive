### Post by mority on 2006-08-07
For true transparency and more eye candy you'll have to install one of the new 3D hardware accelerated graphics servers like xgl. But xgl is not part of the official Dapper Drake release, so you have too choices: Install it manually in your Dapper Drake installation or wait for Edgy Eft, the next Ubuntu release, which possibly will contain it prepackaged.

If you want to install it manually just search the forum for "xgl" and you will find plenty of information how to do it.
