### Post by invalid on 2006-06-30
[QUOTE=msg]I only use this comp for rendering and downloading... I still want to keep it installed so I can show off to my friends when they come over, so is there like a command thing that can disable/enable it without installing/uninstalling? Thanks[/QUOTE]
```
metacity --replace
compiz --replace gconf
```
Disable, and enable, in order.
