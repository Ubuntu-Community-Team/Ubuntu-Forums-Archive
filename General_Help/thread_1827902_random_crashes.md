### Post by HereInOz on 2011-08-18
Looks a bit like a hardware issue, but very hard to be definitive.  Possible video card problem but I can't guarantee that.  I would put a new video card in there, then my next choice would be RAM, and if that doesn't fix it, you are in to serious head scratching territory.

If you have another machine available, you could take the HDD out of the offending machine and fire it up in the other computer.  If all is well, then you have certainly confirmed that it is a hardware issue, however, if the fault transfers to the second machine, then it is either HDD or a software problem.
