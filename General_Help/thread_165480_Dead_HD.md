### Post by garner_nc on 2006-04-24
From my limited knowledge, do a search for "thin client" on Google.

 Thin clients run a small Xserver simliar to VNC to allow you to use a "main" computer from another machine. You can connect many thin clients to a server. This has been used  to allow older machines access to new iron with more power. All that gets transferred from the main computer is the display graphics via ethernet.

Hope this gives you some direction.
