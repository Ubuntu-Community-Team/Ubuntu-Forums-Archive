### Post by jgordon510 on 2006-12-12
> **electrodad said:**
> One of the problems I'm having so far is I cannot run my video clip attachments from my emails. 

A good way to get video set up is to get the Automatix package.  For most users, it's the easiest way to get a bunch of things working without delving too deeply.  You mentioned that you're using 6.0+; you need to know whether you're using Ubuntu Dapper or Edgy and pick the right Automatix package to download.  Just follow the instructions here:
[Automatix Installation Page]("http://getautomatix.com/wiki/index.php?title=Installation") 

> I enjoy playing chess but my chess package within Ubuntu keeps closing on me when I playing. And I'm still confused on how to install downloaded programs into the system. Well that's all for now, any help would be appreciated.

In your system menu, there's a program called the "Synaptic Package Manager."  If you run that, you can search for all sort of pre-compiled packages.  There are a bunch of chess packages.  This is how you install software.  Many of the chess games are designed to interface with online chess servers.  Other users can probably help you get one of them set up easilly.
I would recommend playing chess online games.yahoo.com has a good flash chess board.  After you run automatix, you'll be able to use that.
