### Post by slooksterpsv on 2011-02-21
Mint is nice if you want things installed quickly without having to find all the codecs, plugins, etc. afterwards. - I like it, but it's not my fave, I still love Ubuntu best.

Fedora is good, more security, had issues running apps cause of the level of security on it. VMs ran really well on Fedora.

Ubuntu - perfect for everything IMO.

Others:

OpenSuSE - Awesome for easy LDAP Server setup.

Debian - What Ubuntu is based off of.

Sabayon - Based off of Gentoo (to my knowledge).

Jolicloud - Cloud based Linux but still able to install and run local apps. The look and feel is more like Ubuntu Netbook Remix.

PCLinuxOS, Gentoo, CentOS, etc. etc. etc.
