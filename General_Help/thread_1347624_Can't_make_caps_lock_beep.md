### Post by Sam on 2009-12-06
Run this in a terminal:```
echo -e "\a"
```
If you do not hear anything, system beep is disabled.

If you use Ubuntu <= 9.04, enable it in System > Preferences > Sound > Enable System Beep

In Ubuntu 9.10, I don't know...... #-o
