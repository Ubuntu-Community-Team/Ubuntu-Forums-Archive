### Post by Average Joe on 2008-07-19
I am using Ubuntu 8.04 and Pidgin 2.4.3 and I had the same problem. Installing the Pidgin-plugin-pack (2.0.0) from the repositories didn't solve it. Apparently the Switch Spell plugin is not in that version of the plugin pack.

Instead, I downloaded a copy of the Pidgin-plugin-pack [here]("http://download.ubuntu.pl/_Feisty_Fawn/pidgin/2.1.1/pidgin-plugin-pack_2.1.1-1_i386.deb"). That's version 2.1.1 and works fine for me.

I can easily switch language when using Pidgin now.
