### Post by MariusSilverwolf on 2007-10-30
Brotell,

If you're looking to dual-boot, installing from the LiveCD has the options to re-partition the drive so that Ubuntu and Windows can co-exist.  It if HIGHLY recommended that you back up any important data to an external source before beginning the installation.  Most times, Ubuntu can redraw the partition tables and install in dual-boot mode without affecting the Windows data, but in the off-chance you have a problem, you don't want to lose anything critical.

Most likely, judging from the experiences other users here have had, you will encounter some type of difficulty getting your wireless network card to connect properly initially after installing Ubuntu.  This is common, and we can help you solve that problem.

If you're looking to not be completely reliant on waiting for others to answer your questions, make prodigious use of the search feature here within the forums, and read through the tutorials offered in the Community Support pages as well.  Most every topic is covered between the two.

Before beginning with any installation, though, I second Occas' suggestion.  Run off the LiveCD, see if you like it before committing to an installation.  It's the "try before you buy" option that hooked me to Ubuntu in the first place.
