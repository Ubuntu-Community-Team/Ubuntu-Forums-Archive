### Post by Wim69 on 2007-12-29
Windows -> Ubuntu: use VNC client (TightVNC or RealVNC) on Windows and configure System, Preferences, Remote Desktop on Ubuntu.
Ubuntu -> Windows: use the Terminal Server Client on Ubuntu and install VNC server on Windows (TightVNC or RealVNC).
