### Post by Sam on 2008-10-16
> **smooth_dudes said:**
> I know what a normal partition is so this might be a stupid question, but what exactly is a home partition?

You can have multiple partitions, and each of these can be mounted to a specific point in the filesystem. The most common scenario is to have a root partition ("/") containing the system and a home partition ("/home") which contains users' documents and settings.
