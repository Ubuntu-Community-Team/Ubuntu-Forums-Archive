### Post by TheFu on 2021-07-24
Does the BIOS still see the HDD?  Some ideas:
[LIST]
[*]dead HDD
[*]bad cable(s) (power/data)
[*]loose cable(s)
[/LIST]

Did the cables get moved? Depending on how the OS sees the HDD connection, this could break any automatic mounting, but if the BIOS doesn't see the drive, there isn't much we can do.

### Post by TheFu on 2021-07-24
If BIOS doesn't see the disk, it is a bad disk.  No OS can do anything about that.  

If this was business critical data, use your backups. Lacking backups, open your wallet and spend the $3K for a proper data recovery team to work the problem.

If this was "acquired internet media" .... I don't have any good answer.
