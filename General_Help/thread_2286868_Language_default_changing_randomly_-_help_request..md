### Post by grahammechanical on 2015-07-15
There are two keyboard shorts cuts that affect the keyboard layout.

Super + Space = Switch to next source
Shift + Super + Space  = Switch to previous source.

In my case my clumsy typing style often mis-hits keys and and sometimes the effect is to change the keyboard layout. Do you have any logical reason for wanting to retain the US layout? If not remove it. Keyboard app indicator icon>Text Entry Settings>Select English (US)>Click the minus ( - ) button.

We can also get to the Text Entry Settings through System Settings.

Regards.
