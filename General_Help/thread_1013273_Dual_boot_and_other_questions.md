### Post by TeXtonyx on 2008-12-16
I think you should dual boot. Things which run inside of another
slow down the machine. Wubi will slow XP and Wine will slow Ubuntu.

Mostly I see users who like to play games keeping a copy of Windows.
It's almost universal that people have more trouble getting their
high-end video card drivers (games, dual monitors) working with
Linux than they do with Windows. Maybe you are a geek and like
persisting to a Ubuntu solution. Because Windows has a much greater
market share, one will find special purpose programs for the more
expensive professional programs available, whereas the analog of
the program for Linux hasn't been written yet. But one will find
free programming suites in Linux that they charge you for in Windows.

Some of the games work ok with Wine, or Virtualbox, but others
work better with crossover which costs, so it doesn't pay to
use that if you have Windows. So Windows is better for games at
this point and some specialized professional software functionality.

For instance I have a Microsoft Intellipoint Office keyboard and 
the Function keys toggle between MS Office apps and functions. 
And it has a roller ball that lets me scan through many pages of
a pdf paper. It's much easier than using the down arrow key or
clicking the mouse, and more controlled than the slider. I don't
think there is ever going to be a port of the driver that would
let me map to OpenOffice programs, Firefox and Thunderbird, and
most importantly my scroller. I use it with my mouse, so getting
a mouse with a scroller doesn't work (the roller is on the left).

I do have a little app from SourceForge which let's me map a few
function keys to Firefox and Thunderbird on an ordinary keyboard.
But I've seen a Windows utility that will let you map 352 functions
to different keyboard combinations for $30. I've never seen that
for Linux though one can do miracles with Emacs. I have just been
trying to illustrate the point that for some people there are a
few treasured Windows thingies that they don't want to give up.
I'm not sure that describes you since you didn't mention games.
