If you mean how to change between Windows & Linux, then it depends how you installed it. If you installed it straight off the LiveCD, then you should get GRUB coming up when you boot your PC, with the choice to boot to Ubuntu, or whatever other OS you have. If you have installed using Wubi, then you will be presented with the Windows Boot Manager to choose what OS to boot.
