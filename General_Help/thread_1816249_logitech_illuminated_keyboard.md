It's just a keyboard. It's quite silent compared to my previous one, and I bought it for the backlight, which has 3 intensity levels (+off), and which is great. Other than that, I always use the F-keys for what they are, I'm not that much into hotkeys.

You can use the F-keys also as hotkeys to
- Open browser (F1);
- Open mail (F2);
- Search (F3);
- Open IM (F4);
...
- Eject (F8 );
- Open media player (F9);
- Prev track (F10 );
- Play/Pause (F11 );
- Next track (F12 ).

In Linux, you need FN+F8 to send the special keycode otherwise the system will see F8. In Windows it's configurable whether you send the special keycode or F*n* by default. The one annoying thing -- if you use the keyboard a lot -- is that the 'context menu' key (don't know how else to call it) which is usually close to the right Windows key is not there. Instead you need FN+PrintScreen (this is indicated on the keyboard). Also, scroll lock is FN+Pause/Break.

The shortcut keys function in Ubuntu (GNOME, Classic, no Unity) can be found under System>Preferences>Keyboard Shortcuts.
