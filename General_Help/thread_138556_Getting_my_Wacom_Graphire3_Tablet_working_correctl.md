### Post by SteveF on 2006-03-03
I got my Wacom working in Breezy.  I haven't played around with it a lot but here are some things I did.  I apologize if the text is choppy or non-intuitive.  These are notes to myself and I haven't had a chance to go inand refine them.  Also, I do not have a USB mouse so my results may differ.  There's one area I mention looking at the mouse events.  Using that you may be able isolate which mouse event to use for Wacom to avoid conflicts with your USB mouse.  Also, I found that I need to bootup, then plugin Wacom, then login.  If plugging in Wacom after login, then I need to logout and then login to get the tablet to work correctly.:

Setup Wacom Tablet
Determine tablet event (3 worked last)
	cd /dev/input
	ls 					(shows events)
	sudo cat event0		(replace 0 with each number until found)
	press stylus on tablet, if garbage appears, then found
	
	check mice and mouse0 as well	(may help if USB mouse is causing conflicts)
	
Modify /etc/X11/xorg.conf
	not sure if needed, but if failing, try changing mice to mouse0
	in input device section for mouse
	
	Add lines (input device should come prior to serverlayout stuff):
		Section "InputDevice"
			Identifier "cursor"
			Driver "wacom"
			Option "Device" "/dev/input/event3"
			Option "Type" "cursor"
			Option "USB" "on"
		EndSection
	
		Section "InputDevice"
			Identifier "stylus"
			Driver "wacom"
			Option "Device" "/dev/input/event3"
			Option "Type" "stylus"
			Option "USB" "on"
		EndSection
	
		Section "InputDevice"
			Identifier "eraser"
			Driver "wacom"
			Option "Device" "/dev/input/event3"
			Option "Type" "eraser"
			Option "USB" "on"
		EndSection
	
		In ServerLayout Section, add:
			InputDevice "cursor" "SendCoreEvents"
			InputDevice "stylus" "SendCoreEvents"
			InputDevice "eraser" "SendCoreEvents"
	
Prior to use:
	Make sure new settings take affect
	logout and do CTRL-ALT-BACKSPACE
	restart with sudo /etc/init.d/gdm restart
	
seems to work if plugged in after restarting computer,
	but need to plug in prior to logging in (or plugin, logout and log back in)
	
In GIMP
	File->Preferences->Input Devices->"Configure Extended Input Devices"
	Under "Device" there will be 3 settings (cursor, stylus, eraser)
	change each from "Disabled" to "Screen"
	
	Stylus and eraser remember the last tool they used
	so to get sylus/eraser to work like pencil:
		select the eraser from the toolbox
		use eraser side of stylus
		move off image and flip stylus
		select tool desired
		(eraser should stay on eraser)



Hope some of this helps.  Feel free to ask for any clarifications on what I did.

Steve
