### Post by DGortze380 on 2009-04-23
Yes, it's possible, takes some work...

[http://pcquest.ciol.com/content/enterprise/2005/105070101.asp](http://pcquest.ciol.com/content/enterprise/2005/105070101.asp)
[http://www.ultimatebootcd.com/](http://www.ultimatebootcd.com/)

I would advise against putting Ubuntu and Kubuntu on 1 DVD, because ultimately, it's still a waste of space. Ubuntu and Kubuntu are identical except for the Desktop environment which can easily be added to either distro, you can select between them at log-in.

sudo apt-get install ubuntu-desktop
sudo apt-get install kubuntu-desktop
