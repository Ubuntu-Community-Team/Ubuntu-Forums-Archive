### Post by Frogs Hair on 2017-10-30
Select the activities corner and move your cursor to the right side of the screen and there will be a workspace panel. There is also a workspace indicator extension in the repository.  The following package includes some standard extensions that will appear in gnome tweak after reboot.

```
sudo apt install gnome-shell-extensions
```

Add the gnome extension "WorkspaceBar" to have a workspace switcher on the panel.
In addition, if you want a static number of workspaces rather than dynamic, you can configure that option in Tweaks > Workspaces.
