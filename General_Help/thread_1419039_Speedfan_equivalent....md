### Post by Mark Phelps on 2010-03-01
If you're looking for a simple one-stop solution that is similar to SpeedFan, look into GKRellm. It will install a single app that allows you to select the sensors you want to monitor.

You will need to install lm-sensors as well, as that provides the modules that do the sensor monitoring.

A more complex, and vastly more customizable solution is using conky -- but that can involve a LOT more work and is quite daunting at first.
