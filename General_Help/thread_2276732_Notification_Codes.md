### Post by tgalati4 on 2015-05-05
There are many notification frameworks in linux.  *notify-send* is one that you may want to study:

```
man notify-send
```

An old tutorial:  [http://www.thegeekstuff.com/2010/12/ubuntu-notify-send/](http://www.thegeekstuff.com/2010/12/ubuntu-notify-send/)

*Notify-send* is located in /usr/bin.  You can find the source code in the standard Ubuntu source code repositories.
