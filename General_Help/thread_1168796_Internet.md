### Post by dhughes on 2009-05-24
> **nigel_salvador said:**
> I currently use a DI-524 Wireless Router however, I am a bit unfamiliar with setting a password with WPA/WPA 2 encryption using Ubuntu 8.04

This seems to be a good explanation of how to set up your router: [http://www.howtodothings.com/computers-internet/how-to-secure-a-d-link-di-524-router](http://www.howtodothings.com/computers-internet/how-to-secure-a-d-link-di-524-router)

 As for your wireless network interface I think Ubuntu 8.04 is similar to Ubuntu 9.04, use System>Preferences>Network Connections select the 'Wireless' tab, select it, choose Edit and then Wireless Security then enter the same WPA password as what is used for your router.
