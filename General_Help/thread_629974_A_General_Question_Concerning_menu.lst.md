### Post by drs305 on 2007-12-02
A good protion of the grub menu.lst is generated by grub itself. I believe grub updates itself whenever a kernel is updated. It probably does the same when grub itself updates. Since it's autogenerated, it probably recognized your new setup.

In the past there have been problems after updates which forced the user to go back in and edit grub in order to get the system to boot. I don't know if those problems still exist.

### Post by louieb on 2007-12-03
Check the automagic section. In your case the **groot= **statment. Your menu entries will be adjusted with that values found here when update-grub is run.
