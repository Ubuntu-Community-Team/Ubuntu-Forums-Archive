### Post by utkarsh009 on 2011-05-03
@somapusher: you select the safe mode session at login screen. (look at the bottom, there are 3 dropdown menus, select session from the middle one) now move to system >preferences >startup apps and click on add button. now in the command field type unity and in the name field again type unity. click on ok. now logout and select back the Ubuntu session and login.
