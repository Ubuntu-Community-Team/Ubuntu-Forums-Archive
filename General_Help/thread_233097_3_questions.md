### Post by Tomosaur on 2006-08-09
Not sure about the firefox one, but the command you're after is 'ls', like this:

```

ls /home/me/

```

or ```
ls /home/me/ *.mp3
``` to find all files ending with .mp3 in the directory /home/me/
