### Post by Robban59 on 2005-04-11
Hi everybody,

just thought it worth sharing with all of you, sure you all know it already but there may be some newbie just like me who might find this interesting; 

Archive Manager , aka File Roller, doesn't support decompressing the RAR archive type natively.

To add UNRAR support just :

1) Open Synaptic
2) Search "unrar-nonfree"   -> Unarchiver for .rar files (non-free version)
3) Mark for Installation -> Apply

It is in : Utilities - Multiverse

Happy UNRARing... =;
