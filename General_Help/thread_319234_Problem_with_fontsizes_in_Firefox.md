### Post by Linuturk on 2006-12-17
I found a nice fix for this problem. 

Use the font settings in Firefox to handle the "larger" fonts.

Edit > Preferences > Content Tab 

There you can define the default that Firefox uses. My web browsing has been much better since adjusting this preference.
