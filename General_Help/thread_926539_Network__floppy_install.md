### Post by deviations on 2008-09-22
My laptop has a broken cd drive and can't boot from usb drives. The only way I could install would be with the floppy drive or a network. My other computers have Windows running on them. How could I install Ubuntu Hardy on my laptop?

---

### Post by Taxman415a on 2008-09-22
There are a few ways to do this, the easiest of which is to use [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/) download the Ubuntu LiveCD/Install iso and point unetbootin to that iso and you should be all set. This assumes you are keeping or shrinking the Windows partition you have, since you can't install over the partition that the install iso is running from. A real safe bet is to make an extra 750MB partition and put the Ubuntu image there.

There are other manual ways as well:
[https://help.ubuntu.com/community/Installation/FromWindows](https://help.ubuntu.com/community/Installation/FromWindows)
and even more listed here:
[https://help.ubuntu.com/community/Installation](https://help.ubuntu.com/community/Installation)
