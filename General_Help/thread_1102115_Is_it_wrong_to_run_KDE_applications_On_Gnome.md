### Post by mb_webguy on 2009-03-22
No (and I personally prefer Amarok and k3b to Rhythmbox and Brasero), but there is a downside other than binbash's aesthetic displeasure.  

To run a KDE app on Gnome requires the installation of the Qt libraries, which will take a bit of additional space on your hard drive.  There will also be a very slight performance hit when running KDE apps, since the Qt library has to be loaded.  This is only really noticeable, however, on older computers with very limited storage and memory, however.
