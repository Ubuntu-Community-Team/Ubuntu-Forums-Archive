### Post by n3had on 2010-04-10
Step 1: Install Ubuntu on a Vmware Player 

Step 2: Install all sharing via network related packages on Ubuntu
sudo apt-get install samba 

Step 3: Add a Physical Harddrive to the virtual Machine 

Step 4: Automount it by adding the entries to the fstab

Step 5: Share the necessary partitions. 

Step 6: Access it via Windows Network.
