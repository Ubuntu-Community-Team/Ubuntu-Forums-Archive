### Post by elvis6 on 2016-01-13
I have a really old and slow Desktop PC, that currently is running Ubuntu 12.04.4 LTS Precise Pangolin.
It is asking me if I want to upgrade to version 14.04 Trusty Tahr.
My question is whether my old dinosaur PC can handle the upgrade, or if it will make it even slower.
I am confused because reviews I have read on the newer version state that it is a much faster OS.
On the other hand, the newer version uses more RAM than the old one, so I can see how it may actually slow down a PC such as mine.
The specs on my PC are a Intel Celeron CPU 2.8 Ghz, with 479 MB of RAM.
So, in my case, which factor will over-rule, when it comes to the speed of the new system ?
Will it be even slower, due to the increased RAM, or will it be faster, due to the new version's features ? Or will the 2 factors offset, and be about the same speed as before, if I upgrade ?
