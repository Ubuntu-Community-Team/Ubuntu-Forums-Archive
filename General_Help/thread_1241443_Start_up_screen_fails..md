### Post by Bucky Ball on 2009-08-16
> **volksgewer said:**
> ... I updated from 8.10 to 9.04.

In that case I would download the 9.04 ISO and reinstall with it. Upgrading through an already running Ubuntu install is time consuming and, as you have discovered,  often problematic.

If everything was working fine in 8.10 there has been a screw up with the upgrade. Beats me it doesn't work from CD though. Best and safest is to get the ISO through a torrent. If you download the 9.04 ISO, see if you can run it from CD first (after checking for defects). 9.04 may not be ready for you yet (your hardware at least) and whilst there may be workarounds etc, might be easier to use 8.10 for a month or two and try again later. :)

Main question: Did 8.10 run fine before the upgrade? Do realise that *8.04 *LTS is the most stable and supported for a few years yet so is still very much current whilst Karmic Koala is totally testing (not for production machine). 9.04, while it works great and out of the box on some machines, does not on all, and is somewhere in between and quite young (released April). They'll be some tweaking to do yet.
