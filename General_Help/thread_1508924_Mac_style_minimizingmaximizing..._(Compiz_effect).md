### Post by kerry_s on 2010-06-13
i think thats "magic lamp" animation, but it looks to me he's using random.

so just open preferences-> compizconfig
select-> close animation
double click on the top 1 or select edit
select-> random
adjust the time higher if you want the effect slower
in the bottom select the random effects you want.
