### Post by victor.zamanian on 2008-06-11
I'm afraid it seems like the Run Dialog isn't an application itself, but is a module of the gnome-panel application. And I don't think there's a way to start it other than through metacity (Alt+F2) or through gnome-panel (adding a button on the panel then pressing on it). If you add a button on the gnome-panel to open the run dialog, you'll notice that if you right-click it, it won't have a menu item called "Properties" either, like a separate application item would have.

Sorry to have to break it to you, guy. /-:
