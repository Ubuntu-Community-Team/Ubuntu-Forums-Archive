### Post by tootlet on 2007-09-07
I have a Bytecc Hotdrive 2.5 that I use with Ubuntu

([http://www.newegg.com/Product/Product.aspx?Item=N82E16817145329](http://www.newegg.com/Product/Product.aspx?Item=N82E16817145329))

 and also a Vantec 3.5" that I use with Ubuntu and OpenSUSE with no problems.
[http://www.newegg.com/Product/Product.aspx?Item=N82E16817145133](http://www.newegg.com/Product/Product.aspx?Item=N82E16817145133)

 Can't put too large a hard drive in them as I usually format with Fat32 so I can use them with Windows. Never had a problem with either. Of course, if you go strictly Ubuntu then I would suggest a different file system. 

I use a Hitachi in the 2.5 and a WD in the 3.5 for what it's worth.
