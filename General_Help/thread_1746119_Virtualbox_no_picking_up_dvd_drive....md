### Post by howefield on 2011-05-01
Hi and welcome to the forum,

If you are trying to write to your DVD drive, I don't think that is possible via VirtualBox. Or if you have installed guest additions perhaps you need to mount the DVD drive again rather than guest additions iso.

For the most part, I'd recommend using Linux applications in a Linux operating system. 

Eg, instead of convertxtodvd within a virtual machine, you might use Devede in Ubuntu.
