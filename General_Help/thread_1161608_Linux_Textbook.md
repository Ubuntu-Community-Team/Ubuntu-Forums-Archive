### Post by Vunutus on 2009-05-16
The very nature of Linux means that it is impossible to put anything into print that is not outdated.

That said, I had a Linux sys admin class a few semesters ago and the book we used was "The Complete Guide to Linux System Administration" by Nicholas Wells. I still keep the book around as a reference and have learned much from it.
