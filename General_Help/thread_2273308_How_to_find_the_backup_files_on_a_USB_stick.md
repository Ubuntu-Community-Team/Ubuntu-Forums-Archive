### Post by Holger_Gehrke on 2015-04-12
> **george110 said:**
> I can list the file contents on Ubuntu but there don't seem to be any options to select files by names. How exactly would you go about getting to where your advice bears fruit ?

If you're using standard Ubuntu and it's file manager 'nautilus', then you can just start typing and the highlight will move to the first file that matches the name you're typing. If you have nautilus showing the list-view (ctrl-2), you can order the list by clicking on the headers of the tabular view.
