### Post by ajgreeny on 2014-03-31
Right click in the toolbar and select customise; you can then see "Visible buttons" or "Customise toolbar" which alowsyou to add or remove buttons/icons entirely.  I agree with you, it is very annoying when you use that icon the first time and "Bang!" it's printing with no way to configure the print properties or anything else.

### Post by frenchian on 2014-03-31
Well, I found the answer.

When modifying the toolbar (in this case "standard"), there is an option (on the right) to add commands, etc. Choose that, choose category "documents" and, lo and behold, there is the "print" button I was looking for (icon is a plain printer, no flash or eye superimposed).

Cheers
