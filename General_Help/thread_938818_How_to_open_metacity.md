### Post by f1r31c3r on 2008-10-05
metacity is the standard gnome window manager.


change all the stuff from the appearance tab under system > preferences or just type metacity in a terminal window might need to type

sudo metacity and enter your login password.

if it says you already have a window manager it's probably already running or you have compiz running etc 

install fusion-icon you can flick between metacity manager and compiz manager via the icon in the task bar you can install fusion icon using the synaptic package manager
