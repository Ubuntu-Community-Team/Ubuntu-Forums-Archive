### Post by Zorael on 2008-05-14
Was there such an extension, or did you change settings in [FONT="Courier New"]about:config[/FONT]?
[quote=lifehacker.com][list]
[*]Type [FONT="Courier New"]about:config[/FONT] into Firefox's address bar and click the "I'll be careful, I promise!" button.
[*]Right-click anywhere. Choose New>Boolean. Make the name of your new config value [FONT="Courier New"]extensions.checkCompatibility[/FONT] and set it to false.
[*]Make another new boolean pair called [FONT="Courier New"]extensions.checkUpdateSecurity[/FONT] and set the value to false.
[*]Restart Firefox.[/list][/quote]
