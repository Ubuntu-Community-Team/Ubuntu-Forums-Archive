### Post by Schrute Farms on 2010-10-26
Live CD will not touch your Windows install as long as you select the Try Ubuntu option when it boots and not Install.

Put the CD into your drive and restart your computer.  If it goes into Windows, then you have to make a change in your BIOS setup.  Restart your computer, and when the first screen comes up, press DEL or F12 (There should be instructions on the screen).  Then, somewhere in the menus there should be boot order.  Make your CD drive boot first, then save changes & exit.  If there is no option to make CD boot first, your computer may be too old, though most made within the past 10 years will boot from CD.

I'm not 100% sure if you can install apps, but they definitely be gone when you're done with the Live CD.

As long as you use it as a Live CD, you can go back to Windows with no changes.  If you make a dual boot, and you don't like it, you can erase Ubuntu and go back to Windows.  I would worry about that when you get there.

If you do decide to actually install Ubuntu, even with a dual boot, BACKUP everything important first!!!!  BACKUP EVERYTHING IMPORTANT!  That can't be said enough.  You never know when something will go wrong.  I've read many a story of people losing important stuff because they barreled into an install.
