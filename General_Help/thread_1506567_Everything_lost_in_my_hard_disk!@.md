### Post by oldfred on 2010-06-10
I prefer to partition first, then use manual install. You can do partitions as part of the install.

Dual boot 2 drives
[http://members.iinet.net.au/~herman546/p24.html](http://members.iinet.net.au/~herman546/p24.html)
With 2 drives you want to be sure to install grub to the same drive but then have to use the advanced button on the final partition screen.
