### Post by Impavidus on 2020-08-31
I can think of three things that could cause something like this:
- The applications were upgraded and the new version is incompatible with the settings for the old version, so it changed back to defaults. Not uncommon for themes, but shouldn't happen with bookmarks in a web browser.
- Some rogue software has decided to tease you and deletes random files. Sounds unlikely.
- Your hard drive is dying.
