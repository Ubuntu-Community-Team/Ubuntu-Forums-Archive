### Post by JordanL on 2009-02-12
I'm trying to figure out how I can change the properties for an Item/Launcher in the panel's main menu on ubuntu studio 8.10. I want to be able to add this 
```
env LD_PRELOAD=/usr/lib/libv4l/v4l1compat.so skype
``` (this fixes webcam problems) to the skype launcher under Internet> but i cant find a way to do it. I made a launcher on the desktop and had that as the command and it works perfectly well. Help is much appreciated :D

---

### Post by plucky on 2009-02-12
Right click on **Applications** and select **Edit Menus**

Left click **Internet** and then **Skype** and then right click and select **Properties**.

Change the command window to whatever you want.


Good Luck
