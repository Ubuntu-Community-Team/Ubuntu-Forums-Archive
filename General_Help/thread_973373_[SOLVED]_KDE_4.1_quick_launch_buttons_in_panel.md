### Post by oliwally on 2008-11-06
I'm trying to get used to working with KDE 4.1 (although I'm very much missing the functionality of 3.5) and want to create some quick launch buttons in the panel/task manager for firefox, thunderbird and other applications I use all the time, rather than having to launch them through kmenu or kickoff menu.

Is this possible? How is it done?

---

### Post by MickS on 2008-11-06
Open the Kickoff menu and right click on the application you want, drop down the menu and click on add to panel, this doesn't work if you have it set for the classic view.

Mick

### Post by oliwally on 2008-11-07
