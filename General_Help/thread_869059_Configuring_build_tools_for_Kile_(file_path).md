### Post by dvase on 2008-08-26
By now, you have probably found the solutions to your questions.  However in case you haven't...

Most of these tools require an additional program to be installed via synaptic or apt-get (e.g., '$ apt-get install lilypond').  Installing the program via synaptic should make it immediately accessible and messing around with the path shouldn't be necessary.  Additionally, double check your spelling, 'lilipond' should be 'lilypond'.
