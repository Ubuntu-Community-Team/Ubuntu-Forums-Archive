### Post by Tlem on 2010-06-08
Gnome and KDE are just desktop environments.  Basic Ubuntu install uses Gnome, but the others are available to install.  Use whichever you like the best.  Applications for one work fine in the other.  I myself prefer gnome.  I also prefer 2 KDE apps over the gnome equivalents, K3B for burning, and Amarok for my music player.  

For mounting ISO files.  The easiest way would probably just to install Gmountiso.

To mount your media automatically to a specific location you can do this by editing your fstab.

### Post by 3Miro on 2010-06-09
Ubuntu uses Gnome so if you have a KDE and a Gnome programs that do the same, you should probably use the Gnome one. On the other hand, KDE programs work just fine under Gnome (maybe a bit slow to start and maybe using somewhat more RAM).

Other than Rhythmbox you can also try Exaile, Audacious (not Audacity), Kaffeine and Amarok (the last two are KDE apps, but Kaffeine is probably the one that will work best for you). There are other players of course.

Deamon Tools, i.e. "How to mount an .iso file in Linux?"

Applications -> Accessories -> Terminal, then you should type something along the lines of:
```
sudo mount -o loop -t iso9660 image_file.iso /some/target/folder
```

You should probably make a new folder in /media (i.e. ```
sudo mkdir /media/iso 
```
) and then use /media/iso as the target.

If you have .mdf files (or some other strange format) search the forums on how to convert them to .iso (there are multiple programs that do that).
