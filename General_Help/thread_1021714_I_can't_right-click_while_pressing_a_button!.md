### Post by fragos on 2009-01-05
Right click is context sensitive with where the cursor focus is. On the desktop for example pressing the "f" will change focus to an icon starting with "f" if it exists. Holding the key down while also changing focus with the mouse by right clicking has no intuitive result. It's unclear what you're trying to accomplish. If you change focus with the keboard the menu key will give the same context sensitive result as right clicking the keyboard selected icon.
