### Post by victorgreen on 2008-05-09
There is a settings importer in Hardy's installer. So far Ive only had it work if I was installing over an existing windows or Ubuntu install on the same hard drive, but this installer might. Also, because you still have the other hdd plugged in, even if it doesnt import stuff, you can easily copy paste stuff over from the other hdd or just tell music players, etc to use libraries you already have set up on the other disk.
