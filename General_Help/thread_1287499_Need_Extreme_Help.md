### Post by drs305 on 2009-10-10
even.raghu,

Welcome to Ubuntu and the Ubuntu forums.

Installing 9.10 Karmic Beta would not be my choice. I have been testing Karmic for quite a while and really like it. However, it is still BETA, and is still being tested. As such, I would not recommend it as a new user's choice.

If you install 8.04, you will be getting a very stable release. The drawback is that if you install it you won't be able to directly upgrade to Karmic in a few weeks without either a clean install or mulitiple upgrades.

I think your options are to 
1) wait about 3 weeks until Karmic is officially released and it's been out a week or so
2) Install 8.04 Hardy and stay there while you learn Ubuntu.
3) Install Jaunty, then directly upgrade to Karmic at some later time.

Note there is a bug in the Jaunty install if you choose "side by side" with Windows. Ask about it if you decide to install Jaunty with this option.
