Well, if you do not want open source applications perhaps Linux is not for you ;)

Linux is not a "drop in " replacement for windows and my guess is that you are unlikely to get your windows applications running on Linux.

### Post by paultag on 2008-02-19
> **erickster said:**
> It's very vague.  I'd like to continue using the same software I've been using all along.  (Nuendo)  I'm not sure if I'm comfortable with using an opensource recording program when I have professional gear that cost me a ton of money.  What do you think?

If you look at some of the FOSS apps that were put out, you will find that they may be higher quality then some of the tools you already use. Take for example Jack Audio server. It is a low latency audio server that lets you connect applications ( read connecting Reason to Pro Tools to a Mixer )

Other tools like Hydrogen and Rosegarden are almost so advanced its not funny. 

To put it another way, Audiophiles and professional recording studios should look not at the fact that they appear too complex, but the fact that they can have software that is updated more often, by more people, and for free,

I would suggest installing it on an older box, and trying to get used to the setup. I suggest using Patchage, and Jackrack with Jack Meterbridge to boot. After you see this software in use, you will laugh off the clowns at Guitar Center trying to sell you Reason.

Cheers, and best of luck,

### Post by robeast on 2008-02-23
Sell your copy of Nuendo, Ardour is just as good! It also gets along with other programs, unlike proprietary junk like Nuendo. I've used Nuendo, Cubase and ProTools for extended periods of time and Ardour is just as good for recording, editing and mixing. If you find it lacks a feature, well, that feature is probably covered by another program that you can easily install for free in a matter of minutes! 

I've been using Ubuntu Studio for the past few months and can't be happier that I don't have to deal with proprietary software anymore. I use a Presonus FirePod (worked out of the box!) so your FireBox should work well with Ubuntu. My computer is also three years old, so you shouldn't worry about upgrading your computer hardware.

I just wrote an article that intends to "sell" new users on Ubuntu Studio applications and decrease the learning curve a bit. Read through it and you should be able to determine if Ubuntu Studio is right for you: [http://hubpages.com/hub/Recording-in-Linux-aka-Free-and-Open-Source-Digital-Audio-Workstation](http://hubpages.com/hub/Recording-in-Linux-aka-Free-and-Open-Source-Digital-Audio-Workstation)

I hope this info helps!
