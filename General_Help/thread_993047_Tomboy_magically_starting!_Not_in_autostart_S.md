Do you use the Deskbar search applet? If so, check to see if Tomboy note-searching is enabled in the Deskbar prefs because it will cause Tomboy to start up automatically in the system tray.

*EDIT* Come to think of it, Tracker may cause something similar (although I don't use it, so couldn't say for sure), so check that if you use it.
