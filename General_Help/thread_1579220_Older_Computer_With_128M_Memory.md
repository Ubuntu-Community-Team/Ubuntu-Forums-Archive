### Post by dougalkerr on 2010-09-21
Looks like Crunchbang will run Orca, the text-to-speech program for visually impaired.
Crunchbang is a fast and light Ubuntu off-shoot and should run on the old gal (the PC that is).
Don't know how much Orca might slow things down.
Download Crunchbang and burn it to a cdrom and try it out Live before installing. If everything appears to work then, hey presto.

Crunchbang has a lot of command line needs though but, installing programs is just the same as standard Ubuntu and Orca may be there already.

It is worth a shot... if more ram could be installed then all the better (for any system).

You could dual boot as well but, it would mean having to use a partition editor to make another partition of you don't have two already plus you need a small one for the Swapfile.

Good luck.

### Post by snowpine on 2010-09-23
You can read the Ubuntu system requirements here:

[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

128mb is the bare minimum for a command-line-only (no GUI) Ubuntu Server install. So you are out of luck with the 'buntu familiy unless you can add more RAM.

Puppy is a great choice for older hardware, very user friendly too. :) Not sure if it comes set up for the visually impaired out-of-the-box, but it can be customized.
