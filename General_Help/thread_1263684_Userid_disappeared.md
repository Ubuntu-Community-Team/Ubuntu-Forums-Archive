### Post by Gianl09 on 2009-09-11
I was creating an iso image of my system using Remastersys when the process died and the computer froze. When I restarted Ubuntu (9.04) my user id was not there any more. The home directory is there with all my files. What can I do to go back to my old account and all my customizations?

---

### Post by eric3_14159 on 2009-10-08
Create a "new" user with your old username using the adduser program. It won't delete your old home directory, and so will leave the configuration files alone.

I'd back up your home directory to another disk anyway, just to be safe, though.
