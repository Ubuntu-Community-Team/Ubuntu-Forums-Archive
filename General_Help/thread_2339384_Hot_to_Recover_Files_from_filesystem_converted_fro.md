### Post by sisco311 on 2016-10-08
It's never late to make a backup. If you have the possibility (an external storage with the same size of the partition) create a backup of the corrupted partition.

Once you have it you can try out various recovery tools without worsening the situation.

I would start with TestDisk:
   [http://www.cgsecurity.org/wiki/TestDisk](http://www.cgsecurity.org/wiki/TestDisk)
   [http://www.cgsecurity.org/wiki/Running_TestDisk](http://www.cgsecurity.org/wiki/Running_TestDisk)
