### Post by sayakb on 2008-09-21
You may use screenlets as an alternative. Download it from [http://www.getdeb.net/](http://www.getdeb.net/)
Install it and start the **screenlets-manager**
Click on the Clock screenlet and set it to autostart (check the autostart on login checkbox).
Also for the Clock screenlet, open Properties. Set them to cairo-clock, and under the tab Options->Clock, set timezone as **localtime**
