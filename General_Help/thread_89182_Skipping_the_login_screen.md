### Post by jeremy on 2005-11-12
ALT + F2 and type 'kcontrol' then System Manager -> Login Manager.
Click 'Administrator Mode' and give your password, then click the 'Convenience' tab, and there select 'Enable Auto-Login' and make sure that the correct user shows just below that, now select 'Enable Password-less Login' and also select your username in the list below that.
'Apply' 'Ok?.

Tip, if you have a problem going into 'Administrator Mode', stert again, this time using ALT+ F2 and type 'sudo kcontrol'.
