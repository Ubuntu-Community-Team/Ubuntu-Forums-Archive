### Post by funkydan2 on 2008-07-05
All you should need to do to type with a different keyboard is to go to System->Preferences->Keyboard then click on the Layout tab.  In the layout tab click "Add" and choose the 'Germany' layout with whichever variant you might like to use.

It'd probably also be helpful to add the "Keyboard Indicator" applet to a gnome panel so that you can both choose the keyboard you want as well as see which keyboard is currently being used.

### Post by lightstream on 2008-07-05
Why not keep your English keyboard layout, and just use the standard key-sequences for accented characters?

Basically you hold down the Alt-Gr key while pressing a key for the accent you want, then release and press the letter that should be accented.

So if you want an umlaut on an 'o' for instance, press Alt-Gr and [ together, then press 'o'. You should get ö

If you want one of those funny double-s 'B' things, just press Alt-Gr and 'S' together: ß

That's pretty much all you need for German! But you can do the same whatever language you need to use. [See this page for full info]("http://wiki.linuxquestions.org/wiki/Accented_Characters")
