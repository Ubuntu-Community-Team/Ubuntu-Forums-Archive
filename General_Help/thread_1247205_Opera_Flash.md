### Post by Ellaine on 2009-08-22
OK: I found a real easy way. It's easy to install flash in firefox. Find the libflashplayer.so in the usr/lib/firefox/plugins folder. Now open terminal and run "sudo nautilus". Find the usr/lib/opera/plugins folder. Now drag the libflashplayer.so from the firefox folder into the opera folder in nautilus and drop it there. Now flash works in both firefox and opera.

PS: Copy & Paste will NOT work.
