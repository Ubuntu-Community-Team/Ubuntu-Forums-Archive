### Post by ubun_two on 2009-12-21
One thing I found strange is that there are two Anthy you can select in iBus.  One is with the crown icon, and the other with "A" hiragana icon.

If you choose the one with the crown icon, you can use function keys to do the Katakana/Hiragana conversions (F7 for Katakana, if I remember correctly).
