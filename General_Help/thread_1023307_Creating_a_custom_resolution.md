### Post by SuperSonic4 on 2008-12-27
What happens if you try "auto" and saving that?

Should that fail can you post the results of ```
cat /etc/X11/xorg.conf
```

It seems the best way is to do it manually:

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak
```
```
sudo nano /etc/X11/xorg.conf
```
now change where it says 800x600 to 1280x720 and press ctrl+x to exit and y to save and return to confirm. 
Restart X by pressing ctrl+alt+f1 to drop to a terminal and press *Ctrl + Alt + Backspace*

If all goes to plan you'll have a shiny new resolution, if not then go back to a terminal session and type ```
sudo mv /etc/X11/xorg.conf.bak /etc/X11/xorg.conf
```

the last code restores your old xorg.conf which was backed up in the first code
