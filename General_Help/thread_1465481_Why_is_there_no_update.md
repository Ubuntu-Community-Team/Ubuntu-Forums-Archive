### Post by jocko on 2010-04-30
> **schlep3 said:**
> 10.04 Release Candidate.

That was the last update I had.
And how many updates do you expect? How did you expect those new packages to have been tested if they were not released to the repos **before** the final release?

Since ubuntu development is completely open, updates come **between** the milestone (alpha, beta, RC, final) releases, to allow proper testing by the tester community (=us).
The updates you think is the "rc update" actually were updates that came quickly **after** the rc were released, to fix some bugs that were found by the testers (again, that's us).
Those same updates actually made your install reach the same state as you would get from a clean install from a final 10.04 cd.

So relax. If you have installed all updates until those that came a few days ago, you have the final version already, and you had it a few days before 10.04 was officially released.
You have not missed any super-secret untested updates.
