### Post by LeifAndersen on 2009-12-29
Yes you can.  Open up the gnome-terminal, and type in:

```
sudo apt-get install kubuntu-desktop
```  voala, you have kubuntu.  Although, don't do this unless you're certain you don't mind ubuntu, and kubuntu, sharing the same desktop.

### Post by sanderd17 on 2009-12-29
It's less likely to mess up your system if you use virtualisation but installing virtualisation software isn't that simple. 

If you install KDE packages from synaptic, you'll be able to login into Kubuntu but it's normal that kubuntu will be uglier than on a clean install. Attention: if you install kubuntu via synaptic, you'll have all programs twice, one program for KDE and another for Gnome. But they'll both work in KDE and Gnome.

As a final option you cann install Kubuntu with a life CD next to ubuntu. I think this is easy but you'll have to reboot to use an other OS.
