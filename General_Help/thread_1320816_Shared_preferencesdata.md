### Post by Giblet5 on 2009-11-09
Ubuntu can do this pretty easily with any other flavor of Linux, and most flavors of Unix because applications (of similar version) running under those OSes use the same mechanism for saving preferences and configuration.

Windows uses the registry, and how would one map that filesystem nomenclature? What does one do under linux with "C:\Windows" as a pathname? Sure, it can be done, but a user would need a post-grad degree to maintain the configs...

Early AIX had a mechanism for storing configuration that could be shared with OS/2. Pretty cool, but nobody used it. There just isn't enough interest to make the effort worthwhile.
