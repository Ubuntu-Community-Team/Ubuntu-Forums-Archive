### Post by kellemes on 2008-07-10
These repositories provide packages for the different branches of Debian, not Ubuntu. The fact it works in most cases means it won't in all cases, it really depends on your running Ubuntu and the current state of the Debian branch who's repository you're using.
So issues like this you'll likely run in to when mixing repositories for different kind of os's.
You better use Ubuntu repositories as much as possible.

Edit: Or use Debian ;-)
