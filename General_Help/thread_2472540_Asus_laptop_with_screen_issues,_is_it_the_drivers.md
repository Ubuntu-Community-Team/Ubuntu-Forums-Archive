### Post by grahammechanical on 2022-03-02
Having recently purchased a laptop with Ubuntu 20.04 pre-installed I have noticed that unlike a desktop machine with Ubuntu installed the OS has built in functions to prevent the OS from using up all the power before it can suspend.

You do not tell us if it is possible to resume the OS. As this fault happened when using Windows 10 I would suspect a problem with hardware. Does the battery hold its charge? You, of course, used the Try Ubuntu option before installing did you not? Were there any screen issues?

In Ubuntu settings>Power you can set screen blanking to never. But it is not possible to stop the OS from suspending when on battery power. The UEFI settings utility might give some information.
