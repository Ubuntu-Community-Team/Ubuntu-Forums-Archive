The trash icon is set fixed in place on the unity desktop only, you can install other desktop environments beside unity, as noted by vasa1; for example I have unity installed here but am currently booted into xfce. I also have cairo-dock, gnome, gnome-classic, mate and openbox DE's available. This is all on the one ubuntu install and the different options are available from the log in screen.

If you do choose to use mint you won't be getting unity as the DE but likely "cinnamon". Cinnamon can be installed on your current install of Ubuntu with the use of a PPA. Some info on how to do so is [[COLOR=#0000ff]--here--[/COLOR]]("http://www.omgubuntu.co.uk/2017/05/install-cinnamon-3-4-ubuntu-ppa"). Basically you can have the mint desktop on your current install of Ubuntu with 2 commands and download the packages; or you can do a complete distro reinstall and replace Ubuntu with mint, much more work that way, but that is your choice. 

The developers set the trash icon position fixed in unity, not much any of us users can do about that but to add another DE if we don't like it. At least with Ubuntu, if we don't like a feature, we can add other desktop environments (while still maintaining the standard DE installed as well).

 Cheers and good luck to you whichever option you choose, yeti.
