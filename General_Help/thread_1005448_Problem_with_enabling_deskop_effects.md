By using system > preferences > appearance from the menu, I try to enable normal visual effects on the "visual effects" tab.

Then it says "NVIDEA accelerated graphics driver (version 173)" 

Then I click "enable", and than all it says is:

"Desktop effects could not be enabled"

Is there a way to find out why it couldn't enable the effects, or is the solution already known to someone here?

### Post by eternalnewbee on 2008-12-08
If you have CompizConfig Manager installed, press **ALT-F2** and enter ```
compiz --replace
```

---

### Post by Bas1234567 on 2008-12-08
Thanks! After typing that, it downloaded the new driver, so this is solved.
