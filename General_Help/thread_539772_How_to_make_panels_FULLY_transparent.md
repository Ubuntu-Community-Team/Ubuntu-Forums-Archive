### Post by emerbomb on 2008-07-20
I know this thread is old but i just figured out how to make the panels transparent 
1.Make sure you are using the default theme 
2.Right click on either of the panels 
3.Go to properties 
4.THen click on the Background tab 
5.Select solid color and set the color to white if you have changed it 
6.And move the sliderbar to adjust the opacity. 

Hope that helps
