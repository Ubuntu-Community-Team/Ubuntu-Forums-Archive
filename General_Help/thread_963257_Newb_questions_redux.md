### Post by ad_267 on 2008-10-30
1. Neither really (My opinion anyways) but each can be made to look like a mac pretty easily.

2. Yes

3. Yes/No. You can't install a KDE icon theme easily in the same way you can a Gnome theme if you're on Gnome. But the themes are all basically just sets of pngs or svgs. So if you've got a bit of spare time you can rename them to the correct thing.

4. No it's not fixed and neither is the number of panels. You can have zero to as many as you want. Don't know for certain about KDE but I assume it's the same.

5. Sort of. There's [gloobus]("https://launchpad.net/gloobus") but it hasn't been around that long and isn't very mature. I think that's supposed to be like Preview but correct me if I'm wrong. There's also Gnome do which is similar to Mac's quicksilver.
