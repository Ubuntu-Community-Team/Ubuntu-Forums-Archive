### Post by rmccutchan on 2009-09-27
I was in the same situation and tried several raw converters and settled on Rawtherapee.  It does not have the same file management functions as Lightroom, which I didn't use anyway, but it seems to do a fantastic job of converting images and probably has as much, if not more, ways to edit photographs than lightroom.

From Rawtherapee I save the images as 8bit PNG files and open them in GIMP to finish.  8bit PNG files seem to have a huge latitude while editing in GIMP so there is still room for error.

Both of these will take a little getting used to, but I know the effort will be worth it.  It was for me!

BTW-I heard through the grapevine that GIMP has some surprises coming!
