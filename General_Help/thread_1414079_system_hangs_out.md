### Post by MooPi on 2010-02-23
If your system is completely frozen you can use  ```
Alt+SysRq+r-s-i-e-u-b
```This should reboot the system safely. If you just want to drop out of X use to drop you to command line```
ctrl+alt+F1
```  > alt+F2 should also give you a run dialog

### Post by Enigmapond on 2010-02-23
Also you can add the "Force Quit" applet to one of your panels. I don't know how good this works with a "completely" frozen system but it works well with individual programmes.

go to the panel ->right-click->add  to panel
