### Post by milandiebel on 2009-03-20
I have a rather simple question. I used to run my server on Fedora and would like to switch to Ubuntu. Now I found that there is a desktop and server edition. I am using the machine essentially in both ways: normal desktop tasks as well as a web server plus mysql (LAMP type of setup). Which way is more preferable? 

1. Install server edition and add ubuntu-desktop with apt 

or 

2. Install desktop edition and add all the servers needed

I have the feeling that option 1 is probably easier. Maybe someone here can give me some advice.

Since I am using an Intel Quad core CPU I probably should use the 64bit version although I have only 4GB RAM installed at the moment. My thinking is that it will make it easier if I want to stick in more in the future (preventing a new install).

### Post by mb_webguy on 2009-03-20
