### Post by cariboo on 2009-04-23
Most of the default applications will do everything you need. Ubuntu isn't like Windows where you only get the OS and a couple of minor application.

In a default installation you don't need an anti-virus application. there is a fire wall installed, but it is not active as there are no ports listening in a default installation. If you feel the need to enable the firewall the preffered gui front-end is ufw which is installed by default, or if you aren't comfortable with the command line gufw is available.

Almost all the programs you will ever need are in the repositories and you can use either Applications-->Add/Remove Programs or System-->Administration-->Synaptic Package Manager.

As a new user you may find Add/Remove Programs the easiest way to install programs. In the Show dropdown make sure all available applications is selected.
