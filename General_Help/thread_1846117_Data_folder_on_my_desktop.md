### Post by bcbc on 2011-09-18
Those files come from the Wubi source i.e. if you download the wubi sourcecode from launchpad.net, there is a data folder and it contains those files. It shouldn't ever be present on an installed version, and certainly not on the windows desktop. So... I can't explain how it got there.

But there should be no problem deleting it. Send it to the recycle bin, and then delete them if no issues later if you are uncertain, but they are not needed or used to boot the Wubi install for sure.
