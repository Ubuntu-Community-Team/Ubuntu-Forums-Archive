### Post by Zorgoth on 2010-08-10
If you are doing computer programming there are a lot of major benefits to developing in Ubuntu aas opposed to Windows, first and foremost of which is a far superior command line interface.  Also the workspace functionality becomes extremely valuable when trying to deal with a lot of different source files at the same time and when trying to multitask in general; 4xscreen space is good!  

Also, if you're going into CS, Ubuntu should be great because you can learn a lot about computers from using it; it isn't much harder to use than Windows imo, but at the same time it is very easy to get "under the hood" and see how the OS works.  Using Ubuntu will also help if you want to join any free software projects.

In addition, Ubuntu has a highly customizable UI which you can basically optimize for any purpose/style you want (depending on your taste, you can have whatever combinations of panels, docks, etc. that you want (for docks look up cairo-dock and avant-window-navigator)).  Also look up gnome-do; that is awesome.

Ubuntu will automatically deal with keeping your software up to date more easily than Windows *without* forcing you to update anything.

You don't really have to worry about viruses on Ubuntu as a Linux virus has not been seen "in the wild" for a long time.

The repository makes it very easy to search for free software, which is often of similar quality to products you would otherwise have to pay for and which, while they may exist for Windows, might be hard to find; it might not even occur to you to look.

Window management is generally pretty superior to Windows, if you configure compiz/kwin correctly.  (compiz >>> kwin, and is probably what you're using, but if you ever wanted to try out KDE compiz is buggy with it).  For example, to move a window, just hit alt and drag.

Middle click to paste last selected text - I love this feature.

And most importantly...  WOBBLY WINDOWS!!!

### Post by pastalavista on 2010-08-10
The main advantage of Linux over Windows is security. Linux won't run Direct-X so your gaming will suffer but there are many games for Linux almost as good as Windows games, running on open GPL. Wimdows has too many security vulnerabilities, (including Direct-X) plus Linux has a flexibility advantage in that there are many user interfaces available (Gnome, KDE, XFCE, LXDE etc) not to mention Compiz. But the best part is, you can do anything, without any restriction, to modify Linux to better suit your needs.

Linux doesn't get much attention merely because it is free and, in a business sense, valuless. But once you use it a while you realize, proprietary software is too costly for what it does and too restrictive to be of that much value.
