### Post by kvmapr on 2008-09-29
I came across the thread, in archives, of the same name ( libmjpegtools0 problem, broken) because I too have stumbled upon the same problem while trying to install Cinelerra.

I manage to get a successful Cinelerra install by first going to this site: 
[http://debian-multimedia.org/dists/experimental/main/binary-i386/package/libmjpegtools0.php](http://debian-multimedia.org/dists/experimental/main/binary-i386/package/libmjpegtools0.php)

And clicking on the link at the bottom of the page:
Direct package download: libmjpegtools0

This opened the libmjpegtools0 deb file in gdebi and then successfully installed. After that it was back to Synaptic to install Cinelerra. All happy now.
