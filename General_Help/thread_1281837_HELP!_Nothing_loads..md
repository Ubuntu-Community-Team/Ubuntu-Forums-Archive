### Post by kayvortex on 2009-10-03
Nothing appears on the screen at all? Not one thing? No POST (that writing that comes on at the very start of the boot sequence) messages at all?

The only thing that comes to the top of my head is leaving the computer off for a while before switching it back on again (to let the circuit discharge completely). Maybe also try pressing some buttons as soon as you start the system and press them persistently to see if you can catch some config state (try the delete button, F2, F10, F12, F8 -- all of which I've come across before).

But, if nothing appears at all, then that's very strange: a problem with my system hardware clock prevented me from accessing the BIOS once, but even then it spat out an error message about it. I'd try to find out your motherboard make and model and google up problems about it not booting. One other thing you can try is detaching hardware that's not essential to the system (the CD drive for example).
