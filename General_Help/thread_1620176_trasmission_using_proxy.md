### Post by hawthornso23 on 2010-11-12
Tor would hide your IP. But it couldn't handle torrents. It may or may not be possible, but it isn't a good idea. The bandwidth would destroy the utility of tor which people in repressive parts of the world depend on. If you got it to work by some miracle it would be extremely slow.

Privoxy is a filtering proxy you run on your own machine. It is intended for filtering web pages to remove ads and scripts and annoying cruft you don't want. It does a similar job to adblock or noscript. You can't put torrents through it and in any case as it runs on your own machine so it would have the same IP.

Don't know about the other one. 

As far as I know there is no defense against IP logging on torrents, except filtering IP addresses.
