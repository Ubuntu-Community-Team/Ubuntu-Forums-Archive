### Post by johnuk on 2012-06-22
Flash is terrible on mine. I've spent a year or more messing around with it trying to get it to calm down, but it'll always be using vast amounts of CPU even if I just boot into the OS and open one flash video. I was never able to solve the problem, despite trying a lot of options.

To start ruling things out, try getting Chrome and using it to browse videos on youtube. Youtube supports HTML5, but you need to enroll here; [http://www.youtube.com/html5](http://www.youtube.com/html5)

After enrolling and switching to browsers that support HTML5, that really helped with CPU on mine, the fan no longer sat whistling away. Chrome supports HTML5 natively, and HTML5 gets round the flash requirement.

HTML5 is only supported by a few sites at the moment though, so it won't work for sites that still rely on flash.
