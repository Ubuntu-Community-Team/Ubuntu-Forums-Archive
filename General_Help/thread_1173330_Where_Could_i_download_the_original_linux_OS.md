### Post by tyroeternal on 2009-05-29
As the others have added, linux is not an operating system, linux is technically a kernel.  The kernel is essentially the base that everything is built upon.

Distributions like Ubuntu are simply collections of software that are meant to work together and run on top of the linux kernel.  Also, as a few users have pointed out, you can get the linux kernel from [http://kernel.org](http://kernel.org).

Ultimately there is no "original linux os" because the system does not work that way.  There is no essence of linux other than the kernel itself.  Through kernel.org you may be able to find the original (as in first) kernel developed, but that does not seem to be what you are looking for.

If you are interested in trying out other distributions you might like to look at [http://distrowatch.com](http://distrowatch.com)  Also, reading the wikipedia entry for linux may be a good idea too: [http://en.wikipedia.org/wiki/Linux](http://en.wikipedia.org/wiki/Linux)
