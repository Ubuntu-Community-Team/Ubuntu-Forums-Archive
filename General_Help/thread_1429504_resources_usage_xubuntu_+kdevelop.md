### Post by phillw on 2010-03-14
Hi and welcome to the forums

Xubuntu 

> Recommended minimum requirements
700 MHz processor
256 MB of system memory (RAM)
6 GB of disk space
Graphics card capable of 800x600 resolution


Kdevelop

> KDevelop was tested with SuSE Linux 5.2 on an AMD K6 200, 64MB RAM and FreeBSD 3.0-Release as well as SuSE Linux 6.0 on an Intel 200 MMX, 128MB RAM.



As xubuntu does not require 256MB for itself (the recommended includes being able to run FireFox etc) I reckon your specification should be able to handle it.
