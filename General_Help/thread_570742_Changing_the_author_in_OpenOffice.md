### Post by Man_Beach on 2007-10-08
Is there any way to change the author information in OpenOffice for a particular document?

I regularly update one of my documents which I then save in pdf format and link to from my website - but I'd prefer it if it didn't have my real name as author.

### Post by thirddeep on 2007-10-08
Openoffice files are zipped up of smaller files.  Go ahead, unzip one of you docs.  Now, one of the files are : meta.xml

Inside meta.xml is your author information.  You could create a small script to unzip, find / replace, zip up the file.
