This is an unusual request, that I'm fairly sure nobody asked before... maybe.

I am a volunteer in Malawi, and we are setting up a computer training centre running Ubuntu Linux. We need to put together some training material, and the documentation page for Ubuntu is quite good (we are just teaching basic skills) but it is all in seperate web pages. Is there a single document with all this info available somewhere, or is there a way to download the whole lot in one go without having to do it page by page, which can be slow and messy?

### Post by ajgreeny on 2007-06-27
Have a look at this big big page which is a downloadable html doc, but all in one page not linked from page to page.
[http://ubuntuguide.org/wiki/Ubuntu:Feisty](http://ubuntuguide.org/wiki/Ubuntu:Feisty)

If you go to File>Save Page as in Firefox you will be able to save the file and not need to be online to read it, you can just open it with Firefox as a local file.
