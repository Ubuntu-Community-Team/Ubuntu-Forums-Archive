### Post by jdong on 2004-12-04
ugh, Nautilus-cd-burner just plain sucks for anything more complicated than just burning an ISO or a single set of files.


When you choose Write To CD, there should've been a Multisession checkbox , which had to be checked the first time you created the CD.



I personally recommend that you install K3b from Universe, a far superior burning program. Only disadvantage: it's a KDE app, not a GNOME app.

Actually, Ubuntu's light brown theme blends in quite nicely with KDE's Keramik, dark-gold theme! what a coincidence!
