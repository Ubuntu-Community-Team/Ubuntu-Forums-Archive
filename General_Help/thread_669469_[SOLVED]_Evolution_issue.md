### Post by Ertai87 on 2008-01-16
Hey guys!  I have a question about Evolution.  I'm using it to read and write various newsgroups, but I find that when I write to newsgroups, my replies aren't threaded properly.  This is a bit annoying.  Anyone know why this is happening and how to fix it?  It's not a big deal, but I'd like to fix it if possible.

---

### Post by Ertai87 on 2008-01-17
For anyone having the same problem, the solution was to go to edit -> preferences -> mail preferences and select the checkbox "Fall Back to Threading Messages by Subject".
