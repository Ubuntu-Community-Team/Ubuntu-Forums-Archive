### Post by Elfy on 2011-11-11
It is entirely possible that you are confused between memory/RAM and disk space.

If you want to install more RAM you need to ensure that the machine is firstly capable of using more than is currently installed, if it can then you need to buy more and fit it.

If you are talking about disk space - then this is NOT memory.

If that is the case then ity might be possible to increase what is available to wubi.
