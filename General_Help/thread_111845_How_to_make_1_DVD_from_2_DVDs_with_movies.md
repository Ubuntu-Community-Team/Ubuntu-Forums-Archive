### Post by Kethinov on 2006-01-03
Personally, I would use DVD Shrink for this. It's a Windows program, but it will run in WINE with the proper WINE config. I have it running perfectly in Ubuntu with WINE.

You can use DVD Shrink to remove the menus and extras, then make a DVD ISO from the movie which you can burn to any DVD-R.

There are probably other more Linux-centric ways to do this, but I couldn't find any when I did my research. Thankfully, WINE works wonders sometimes.
