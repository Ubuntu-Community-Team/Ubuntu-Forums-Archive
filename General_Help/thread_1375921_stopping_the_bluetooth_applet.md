### Post by Handssolow on 2010-01-08
Can I remove the applet? I'm not using bluetooth with my desktop PC and I have a wired connection to the router. There's no wireless card or motherboard based bluetooth device yet System Monitor shows I'm running the bluetooth applet.

Surely this is unnecessary, can I can remove or disable this process and easily re-enable things if I should buy a wireless keyboard etc.

---

### Post by Tim Sharitt on 2010-01-08
Goto System > Preferences > Startup Applications and un check the box beside Bluetooth Manager.

That should keep it from starting and make it easy to re-enable in the future.

### Post by ajgreeny on 2010-01-08
If you really don't want it you can uninstall bluetooth and several of the other bluez packages with synaptic.  Just watch out as some of them will also remove some needed gnome packages, so you will need to keep a few that are necessary for gnome.
