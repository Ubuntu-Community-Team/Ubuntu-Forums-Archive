### Post by seawolf167 on 2011-09-23
> **vanhenryjr said:**
> The question I guess is do all open software wordprocessors use the same file extension?

No.

Example: LibreOffice/OpenOffice use their ODT formats, Microsoft Office uses their own proprietary formats (the difference between the new and old formats is that the new .doc***x*** files are archives where the old .doc files are not).

Microsoft doesn't play nice with other products, but Libre/Open office does, so you can open MS documents in Libre/Open office, but not vice versa. Even then though, not all internal document formatting will be the same. 

I suggest whenever possible, use the same software to edit your documents as the creation software.

LibreOffice is basically a version of OpenOffice (not exactly, but it's a derivative), so they share file formats. The file formats that they use are also open document formats.
