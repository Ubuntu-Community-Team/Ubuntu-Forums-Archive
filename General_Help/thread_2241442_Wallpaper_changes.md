### Post by rickm1945 on 2014-08-26
I set a wallpaper on my Unity Desktop ( Ubuntu 14.04 LTS) and it keeps changing like it was a slide show. How can I lock the wallpaper so only one shows all the time?

---

### Post by qamelian on 2014-08-26
If the wallpaper you selected has a little clock icon overlaying it in the wallpaper chooser, then you have selected a slide show rather than an individual wallpaper. You will need to go back to the chooser and select the single wallpaper rather that the slide show that includes it.
