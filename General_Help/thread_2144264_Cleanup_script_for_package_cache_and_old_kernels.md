### Post by mperrin on 2013-05-11
Cleaning up old kernels to regain disk space has been addressed in several previous threads. I have three machines with small solid state drives so it is particularly important for me. Doing it step by step became tedious so I created the attached script. It initially autoremoves and cleans, then optionally displays resident kernel image and header packages and provides a convenient way to remove older packages that are no longer needed. It has to be run as root (sudo) obviously. Hopefully you will find it useful. No guarantees of course but I use it on several Ubuntu and Linux Mint machines and haven't bricked one yet.
