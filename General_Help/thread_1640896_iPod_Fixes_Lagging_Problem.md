There are other issues, but you get the point. I was not sure what caused it, but I decided that with that going on, I would update to 10.10 and completely get rid of the Windows partition. I have tested this theory a few times and it is completely sound. My computer only quits lagging if my iPod is plugged in. I have tried plugging other things in and it does not help, only the iPod does. Does anyone have any idea what's wrong with it?
