### Post by guine on 2007-04-18
When I go to some websites(Ive mainly noticed this problem on flickr) in opera I have trouble with opera loading the page how it looked the previous time I opened it regardless of changes that have occured.(such as new pictures or comments not showing up until I refresh the page after opening it once)  This problem has just started occuring recently on some but not all sites and opera had worked fine at loading the proper version of the page before.  Anyone have any ideas on whats going on and how to fix it?  Thanks.

---

### Post by Marsolin on 2007-04-18
One potential is to select Preferences from the Tools menu and then History from the Advanced tab. Changing how long Opera will use a cached version of the site may help.

### Post by guine on 2007-04-19
