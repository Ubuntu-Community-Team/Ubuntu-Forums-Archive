### Post by DeMus on 2009-11-12
Since yesterday I get this message in Firefox 3.5 Shiretoko:
```
Firefox prevented this page from automatically reloading.
```
It's written in a yellow bar on top of the page. On the right side is an Allow button and a Red Cross button.
It's not on all pages, this page for example does not get it, but I had it on more than 1 page.
Any idea how I can get rid of this bar?

---

### Post by scouser73 on 2009-11-12
Go to **Edit > Preferences > Advanced** and uncheck the box **Warn Me When Websites Try To Redirect Or Reload The Page**.
