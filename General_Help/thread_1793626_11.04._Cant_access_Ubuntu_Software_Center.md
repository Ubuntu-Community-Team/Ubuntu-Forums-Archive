### Post by cariboo on 2011-06-29
It sounds like the software center is broken, you can re-install it using synaptic package manager. Click the application icon in the Launcher and type syn in the search bar. Click the icon to open synaptic.

Enter your password when asked, and once synaptic is open, enter software center in the quick search box. Once the software center has be found, right-click on it, and select mark for Reinstallation, then click the apply button.
