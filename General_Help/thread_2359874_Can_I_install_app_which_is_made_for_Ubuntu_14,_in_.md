### Post by kc1di on 2017-04-29
This will be totally dependent upon what Libs and dependencies are needed to run the program. 
16's Libs have all been updated quite a bit and if the program has never been updated to use them then there will be problems trying to down grad them to older versions.
Not that it can not be done, but it may be a bit involved.  I would contact AIR Infotech (Lesearch) tech support if they have one and ask them if Ubuntu 16 is supported - they may have a patch or an updated verison. The problem is this is not a ubuntu program and appears to be propriety offering so it will be up to the vendor to update their software not ubuntu.
