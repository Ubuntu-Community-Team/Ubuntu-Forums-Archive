### Post by TheFu on 2017-09-28
TOR is not 100%, especially now.  There have been multiple nation-states that have found attack techniques to de-anonymize traffic.

The way that the professionals do it is to have a different PC, never used online, ever, and use TOR from it.  Never use any websites that you've used without TOR through TOR.  Never mix account.  Never mix hardware.

Even with that level of paranoia, it is possible to be traced, if they really want to find you.

OTOH, if you are trying to hide from normal people on the internet, then not mixing online accounts is sufficient and be careful what you write and how much you write.  Smart people learn to recognize the writing styles for people they know.  We tend to write in specific ways that can be used to find us.

You can use a second login if you like, but I don't think that helps any more than using a wiped profile will.  Perhaps running a TAILS distro inside a VM would be smarter?  Just be certain you keep it patched.

And there are viruses for Linux.
