### Post by Leppie on 2010-01-20
If you like the windows live messenger look and feel, try emesene which you can install from the repos:
```
sudo apt-get install emesene
```
to use hotmail in thunderbird, you need to download and install the webmail plugin for thunderbird first. you then can download and install the hotmail, or yahoo, or other providers' plugins.

open office supports ms office formats very nicely. just the layout may be slightly off here and there and be careful with macros.
