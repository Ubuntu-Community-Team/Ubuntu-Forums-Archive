### Post by Therion on 2009-03-16
Download the theme or font that you want to install and save it to your Desktop.

Go to System/Prefernces/Appearance.
This is the Theme Manager and you should be on the Themes tab.  Stay there.  
Drag and drop the theme, or icon, file that you saved to your Desktop ONTO the Theme Mananager.
You should get a prompt that new Theme has been installed prompted to use it if all goes well.
You can customize further by clicking on the "Customize" button down at the bottom of the Theme Manager.


EDIT: Forgot to mention...  If you get the "**xyz** does not appear to be a valid theme" message you'll need (most likely) to right-click on the file you downloaded, click on "extract here" and then drag the resultant folder onto the Theme Manager.
