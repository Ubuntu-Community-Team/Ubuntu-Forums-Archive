### Post by Morbius1 on 2010-04-07
Open **Terminal**
Type **sudo mkdir [COLOR=Blue]/media/Suse[/COLOR]**
Type [B]gksu gedit /etc/fstab
[/B]
And put a # sign in front of this line:
>  /dev/sda1 /media/sda1 ntfs-3g defaults,locale=en_US.UTF-8 0 0so that it looks like this:
> #/dev/sda1 /media/sda1 ntfs-3g defaults,locale=en_US.UTF-8 0 0Then add the following line:
```
/dev/sda1 [COLOR=Blue]/media/Suse[/COLOR]         ext3    relatime        0       2
```Save fstab, exit gedit, and back in the Terminal type:
**sudo mount -a**
