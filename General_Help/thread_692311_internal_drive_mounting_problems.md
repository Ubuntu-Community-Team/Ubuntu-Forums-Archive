### Post by pdub on 2008-02-09
I would suggest that you create a mount point for the drive such as /mnt/data or somewhere in your home folder depending on the type of data you are storing and whether more than one user will need access.


```
sudo mkdir /mnt/data
sudo chown your_username /mnt/data -R
```

or 

```
mkdir /home/your_username/data
```



Then edit your fstab file as follows.

```
/dev/sdc /mnt/data ntfs-3g -o silent,umask=0,locale=en_US.utf8
```

This will give full access to all users. The drive will mount each time you reboot. You can manually mount with:

```
sudo mount -a
```
