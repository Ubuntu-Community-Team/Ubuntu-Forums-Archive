### Post by Haegin on 2006-12-20
Firstly - you shouldn't start a new thread - just bump the old one so people notice it. Also please don't swear in the forum - some people are offended by that kind of language and won't help you.

What is the URL of your other thread and I will help you there.

I recommend you learn how to use the command line when you have some time as it is so useful and powerful in Linux and lots of people will give answers in the forums that involve console commands purely because it is easier than saying "click on this and then right click on that and drag that icon onto that brown box (which isn't brown because you changed your theme)" etc...

As one last but important point - you ***_NEVER***_ need to login as root. EVER. Full stop. Everything you need to do can be done as root using the sudo command or - in the unlikely event that sudo won't work you can use sudo su to run as root. The GUI equivalent of that in gnome is gksudo and works by preceding the program you wish to run with gksudo. For example to run gedit as root use "gksudo gedit".

I hope this helps - if you need more help please link to your other thread and I will try and help you there.
