### Post by sudodus on 2011-11-27
Ubuntu 9.10 is old, and the repositories are not supported, maybe with an exception for the netbook version. Is that what you are running? Otherwise I strong recommend that you upgrade in order to get the current security updates.

I think that updating software will be much easier if you switch to a newer version, either 10.04 LTS with support until April 2013 or 11.04 or 11.10. But there may be issues with some of your hardware, so please*** test the new versions live from a CD or USB drive*** before installing or upgrading to the new version!
