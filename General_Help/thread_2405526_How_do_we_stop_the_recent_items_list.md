### Post by howefield on 2018-11-07
Which version of Ubuntu are you using ?

If a later such as 18.04 or 18.10 go into the settings menu > Privacy > Usage and History and switch off "Recently Used".

If you are using an earlier version of Ubuntu there will be a similar setting.
