### Post by jespdj on 2009-01-22
As far as I know, there is no Sun Java 6 update 11 available for Hardy. If you really need update 11, you can uninstall the current Sun Java on your system and download Java directly from [Sun's Java website](http://java.sun.com/javase/downloads/?intcmp=1281) and run the installation program.
