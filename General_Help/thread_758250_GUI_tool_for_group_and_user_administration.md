### Post by pytheas22 on 2008-04-18
You could try just installing the user-administration utility itself, not all of Gnome.  It will probably also require some Gnome libraries, but that's all; you won't need to install all of Gnome, and you should still be able to run it from within Xfce by typing "sudo users-admin" in a terminal.  This is the command to install:
```

sudo apt-get install users-admin
```
