### Post by ajgreeny on 2008-06-12
If you want to use something else, just go to System>>Preferences>>Preferred Applications and chose what you want there, eg thunderbird.  Or if you want it to be your default, you will need to set up the mail accounts, just as you would in any mail application.  Without that information none will work.
