I think the only thing that you could do (which probably only partly covers what you want) is to have both computers sharing the same home directory.

This would mean that your personal files and settings files would be accessed on the same drive.

However, this isn't ideal, because:
[LIST]
[*]If you have different versions of programs on each computer, they may need a different settings file format.
[*]It would be a bit risky to log on to both machines at once.
[*]Global settings (as opposed to per user settings) couldn't easily be synced.
[/LIST]

Also, I guess you could sync your .mozilla folder with Dropbox so all your firefox settings and bookmarks would be synced.
