I know nothing about this subject but in System Settings there is a utility called Remote Desktop. This allows me to set up this machine to allow remote viewing and control of the machine. So, I guess that the first thing that is needed is to set up each machine to allow remote viewing and control. It would help if these machines were using Ubuntu But perhaps this is not the case. You do not say what OS is being used on the network machines and on the administrator machine.

If you go into to the software centre and search for remote desktop you will get a list of the type of programs that you are looking for.
