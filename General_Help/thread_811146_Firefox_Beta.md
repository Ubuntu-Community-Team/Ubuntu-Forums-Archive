### Post by bingoUV on 2008-05-28
[https://bugzilla.mozilla.org/enter_bug.cgi](https://bugzilla.mozilla.org/enter_bug.cgi)

If the bug is in Ubuntu's build of firefox, Ubuntu should be the first point of contact. So file a bug on Ubuntu.

If you have verified with mozilla's build of firefox and can reproduce the bug there, go ahead and file a bug directly on mozilla.
