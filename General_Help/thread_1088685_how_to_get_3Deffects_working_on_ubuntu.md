### Post by Kevbert on 2009-03-06
Welcome to Ubuntu.
If you have an ATI, Intel or Nvidia graphics adapter this should work.
You probably need to install Compiz Configuration Settings Manager. Go to Applications-Add/Remove and search for ccsm. Install the package by ticking the box and Apply Changes. You'll find the program installed under System-Preferences-Advanced Desktop Effect Settings.
