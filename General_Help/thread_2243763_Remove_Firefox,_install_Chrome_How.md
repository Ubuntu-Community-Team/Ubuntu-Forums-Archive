### Post by uRock on 2014-09-11
You can remove Firefox via the Ubuntu Software Center. Just open USC, then search for Firefox, then click on it, then click **Remove**. Don't uninstall it before getting Chrome.

Here's a link to a vid I made showing how to download and install Google Chrome. [http://youtu.be/gmzmNriRVqs](http://youtu.be/gmzmNriRVqs)
