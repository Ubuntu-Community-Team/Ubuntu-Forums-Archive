### Post by Bucky Ball on 2017-07-24
Try installing the one in the repos. Use whatever package manager you're using (Software?) or a terminal. There is no need to install from a third-party. Prior to installing, do an update.

```
sudo apt update
sudo apt full-upgrade
```

This will NOT upgrade your 16.04 to a newer release, just upgrade the software you have installed.

The version in the repsitory is 5.1.6 from memory.

The best version for your release of any app is in that releases official software repositories. Best to check there for apps before searching and installing from other sources.
