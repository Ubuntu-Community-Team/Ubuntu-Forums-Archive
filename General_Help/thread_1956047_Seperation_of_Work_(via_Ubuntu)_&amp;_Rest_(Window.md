### Post by Mark Phelps on 2012-04-10
From my own experience (I multi-boot a multi-core processor desktop PC between Win7 and Ubunu 11.10), I have seen no measurable performance difference between the two.  Sure, is some things, one might be faster than the other, but I've not seen any major difference.

Thus, your fear of adding things to Win7 would slow it down are (IMHO) unfounded.

Don't let the OS force you into making the decision -- make it yourself based on what works best for you. If SW development works better in Ubuntu, and you like the tools there more, then use it. IF the opposite is true, but with Win7, then use it, instead.

After all, ANY PC is nothing more than a high-tech tool.  You be the driver of the tool; don't let it drive you.

As to resetting Win7, agree that a total restore would be a major pain -- and something you should NEVER have to do.  Instead, build yourself an easy restore solution by doing the following:
1) Download and install the FREE version of Macrium Reflect (MR)
2) Use the MR option to image your Win7 install to an external drive
3) Use the MR option to create an burn a Linux Boot CD.

NOW, if you need to "reset" your Win7 down the road, you can do that in only a few minutes by connecting up your external drive, booting from the Linux Boot CD, and using the MR option to Restore from your Win7 backup.  Instead of hours (or days) or work, you're only looking at a few minutes.
