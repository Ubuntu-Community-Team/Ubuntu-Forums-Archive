### Post by Gotterdammerung on 2007-05-22
This was my quest too. 

These are my test results:
- Azureus: good interface, but unstable.
- Deluge: good interface, but too damn unstable.
- Bittorrent-GUI, Bittornado: only one download a time per window, does not allow the user to set global speeds.
- Freeloader: extremely simple, does not allow the user to set global speeds.
- **kTorrent**: very good interface, stable, allows the user to set global speeds. **IMHO, the best choice!**
