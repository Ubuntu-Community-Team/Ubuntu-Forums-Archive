### Post by justleen on 2008-07-29
for softmodding you will need either an already softmodded box for hotplugging, or a gamehack on a memory stick. for hardmodding you need hardware..

there is a bootable linux cd that will allow your disk to mount as a normal drive, but you cant actually mod it with that, just unlock it..

I've spend hours and hours, and ended up buying splintercell and a mem card for a few euro. With that i was able to unluck a 80gb drive, install it as "orginal" then mod it with a game hack..

Trust mem the game hacks are BY FAR the easiest way around it! once modded, installing linux is a breeze...
