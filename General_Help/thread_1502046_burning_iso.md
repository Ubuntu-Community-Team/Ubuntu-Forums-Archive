### Post by 3rdalbum on 2010-06-05
Things to check:

1. Burn at 8x or as close as possible

2. You need to use the "burn image to disc" function of your burning software. (it might be known as "Burn ISO image"). If you don't have such a function in your burning software, then try a different program. InfraRecorder is a commonly-suggested program for burning.

3. DON'T use any function labelled "make bootable" or "Create a bootable disc".

4. Use good quality CD or DVD media; looks like you're already doing this.
