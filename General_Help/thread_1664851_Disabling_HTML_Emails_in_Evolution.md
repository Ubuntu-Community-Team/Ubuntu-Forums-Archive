### Post by lasnaghi on 2011-01-11
Hello fellow forum members;

I'm trying to disable Evolution from sending out emails in HTML. While I can every single time click on "Format" and select "Plain", I'd like it to be Plain all the time without the extra steps. So far, no luck.

Under Evolution's options "Composer Preferences" there is no option to disable sending out HTML emails. I'm pretty sure that's where it is on my regular desktop.

Using gconf-edit, I went under "Apps" and then "Evolution" and saw that under "Composer" the "Send HTML" tab is already unchecked. So great.

But Evolution still insists on sending out HTML emails.

What gives? I searched on the internet but it seems that I am the only sucker having this problem -- or are people just not using Evolution on their little Netbooks, what?

As I find myself offline for some periods of time, it is convenient for me to have Evolution so that I can write emails. Is there some sort of way to solve this?

Thank you.
lasnaghi

### Post by lasnaghi on 2011-01-11
I "fixed" the problem by disabling the Netbook Ubuntu Remix desktop and going back to classic. After reconfiguring the menu bar on top (almost everything was gone), Evolution behaved as expected.

Not exactly the solution I had in mind, but it also solves the real-estate space being taken up by the Netbook edition side menu thingy.
