### Post by themusicalduck on 2010-12-02
Just to check as it doesn't say in your post. Can you actually still boot into Ubuntu? Or do you just get that error message and it stops the whole booting process? If it is just that the splash screen isn't displaying -

What resolution did you put in with Plymouth Manager? If you did put in the right resolution originally, then something else is wrong. If not then it might be worth running it again and putting the right resolution in.

If that doesn't work, then download and run this script - [http://sourceforge.net/projects/bootinfoscript/](http://sourceforge.net/projects/bootinfoscript/)

Save it to your desktop, open a terminal and and type ```
cd Desktop
```then type ```
sudo bash boot_info_script*
``` and copy the results from the RESULTS file into code tags on here. That'll let us know a bit more.
