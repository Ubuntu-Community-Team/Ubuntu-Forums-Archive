### Post by hakermania on 2011-07-29
~ is the path /home/your_username
For example, if your username in your Ubuntu PC is [beyondlimit]("http://ubuntuforums.org/member.php?u=1377541"), then the path
```

~/Pictures

```is actually
```

/home/[beyondlimit]("http://ubuntuforums.org/member.php?u=1377541")/Pictures

```It is just a shortcut

### Post by JC Cheloven on 2011-07-29
A file starting by dot is a hidden file. It shows up in the file browser hitting ctrl-h.
The dot also means "current directory" when used as first character of a path. So ./kompozer calls for the execution of the file kompozer which is in your current directory.

Please search a little, there are a lot of tutorials over there:
[https://help.ubuntu.com/community/UsingTheTerminal](https://help.ubuntu.com/community/UsingTheTerminal)
