### Post by stinkeye on 2011-11-20
ctrl+alt+s maybe?

You can also set the mouse wheel when positioned over the titlebar, to shade a window.
```
gconf-editor /apps/gwd/mouse_wheel_action
```
and change the value to **shade**

The actual animation settings are in the ccsm animations plug in.
