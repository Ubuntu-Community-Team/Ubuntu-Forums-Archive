No, you don't need any special set up. When picking the software app from Synaptic or apt-get or aptitude it will resolve any dependencies you may need and prompt you to install those as well. KDE based applications use a different set of libraries. But it is no big deal to install those. The kernel doesn't care what you want to run, just as long as it knows where to reference libraries.
