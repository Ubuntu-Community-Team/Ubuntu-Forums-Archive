### Post by Richard1979 on 2010-02-09
Wine has issues with sound and will typically use software emulation unless told not to.
I've had the same problem in games and basically it sounds like they are running at half bitrate.

The solution to that is here:
[http://wiki.winehq.org/UsefulRegistryKeys](http://wiki.winehq.org/UsefulRegistryKeys)

```

UseDirectHW
      |      [When set to y, direct hardware access is used

```

Be aware that some apps might not like it but those that do will sound the same as on Windows.
