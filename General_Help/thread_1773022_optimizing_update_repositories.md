### Post by plucky on 2011-06-01
> Is there a way to optimize repositories?


There is a way to choose the Best Download server.

Open Software Sources through the Synaptic Package Manager and on the first tab, select **Other** in the "Download From" window.
In the window that opens ,choose "**Select Best Server**"

Good Luck
