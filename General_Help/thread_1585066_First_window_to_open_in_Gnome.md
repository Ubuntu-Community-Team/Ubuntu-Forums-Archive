### Post by prshah on 2010-09-30
> **waxranger said:**
> (only the first window, the consecutive windows shall align in reference to the first)

Hi If you are using Compiz (ie, Visual Effects), you can do this with the CompizConfig Settings Manager: System-Preferences-CompizConfig Settings Manager-Place Windows-General (Tab)-Placement mode-Cascade.

Don't forget to enable the "Place Windows" plugin after choosing!
