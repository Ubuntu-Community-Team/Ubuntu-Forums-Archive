### Post by Impavidus on 2017-01-20
By default you already have two: the one with ctrl-c and ctrl-v (shift-ctrl-c and shift-ctrl-v when in a terminal) and the one with click the middle mouse button to paste the currently selected text.

Some applications have their own mechanism for copy/paste buffers. In vim you can select a buffer (register in vim parlance) with "<any letter> and use that for the next cut/copy/paste (delete/yank/put in vim parlance).
