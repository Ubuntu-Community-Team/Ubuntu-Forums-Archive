### Post by e_james on 2009-03-21
You still didn't say but I assume it's the Live CD (the alternate install CD is less obvious on the site). If you boot from that CD and everthing seems to work, it's a good assumption that your hardware is generally compatible. Some items like the wifi adaptor may require additional driver downloads. The standard technique is to search the forums for that specific model and read through the suggestions until you find one which suits your requirements. Much of the more popular hardware is also described here [https://help.ubuntu.com/](https://help.ubuntu.com/)

An idea which you might like to consider is to install ubuntu on an external hard drive or pen drive (recommend 8GB) without disturbing your existing installation until you are ready. I expect that your bios will be able to boot from usb devices. You might need to change some settings.

I expect that your bios will be able to boot from usb devices. You might need to change some settings.You can do this via usb-creator which is bundled with the live CD, just so you know.

Yes, you can have both Linux and Windows on the same computer.  It's called dual-booting.  You will have to create separate partitions for Linux and Windows, though, and that could mean resizing your existing Windows partition - which can be a problem since modern Vista partition setups are pretty messy (I've seen a computer with 4 partitions when only 1, maybe 2 for the recovery were needed).

However, if you're just looking for a new OS, I would be careful.  Ubuntu isn't always for the casual user - no Linux distro is.  You have to be willing to get your hands dirty from time to time.
