### Post by nzjethro on 2011-07-05
The latest stable Ubuntu version is 11.04 Natty Narwhal, while the latest development alpha (which isn't stable and thus has lots of crashes) is 11.10 Oneiric Ocelot.
What you are referring to is probably 11.04, which can be downloaded from the Ubuntu homepage, or torrented from [http://torrent.ubuntu.com/simple/natty/desktop/](http://torrent.ubuntu.com/simple/natty/desktop/).

My suggestion if you are interested in trying it out is to download the .iso, and create a Live CD, which you can run without installing it over your 10.04 install. The reason for this is that Natty is a big change from 10.04, as it is the first release to use the Unity UI, which is quite different to GNOME 9which 10.04 uses). Furthermore, if your PC is older, or lower specs, it may not run 11.04 as well as 10.04. Finally, 10.04 is a long term support (LTS) release, meaning it has had all the bugs worked out, while 11.04 is "rolling" which means that there are occasional hiccups. If you try 11.04 Live for a week, you will see whether it is to your liking, and if your hardware supports it before deciding whether or not to install it.

My 2c say don't upgrade. 10.04 LTS is a solid release which shouldn't give you any issues, while 11.04 has had a bit of bad cred on these forums with a lot of issues. At the very least, try it Live for at least a wook or so to see whether upgrading from LTS is what you really want to do. :D

Do you mean that you restarted your computer with the 11.04 CD in it, and it didn't start the installation procedure?  If that is the case, you may need to press F12 or the key it says to on the first screen when you first turn it on in order to say that you want to 'boot' from something different and then choose the CD.

EDIT: Oh, okay. I think the message was probably telling you that something went wrong in the installation. It could be that the file you downloaded was slightly corrupted along the way from the server to your CD somewhere.

I suggest downloading it again and giving it another shot. 

Save the ubuntu 11.04 iso you downloaded in the home folder of your computer. Open up the terminal and copy/paste "md5sum ubuntu-11.04-desktop-amd64.iso" (or whatever your filename is) without the quotation marks into the  terminal (Ctrl+Shift+v is how to paste in the terminal, not Ctrl+v).
 
If it says anything other than what it says it should here: [https://help.ubuntu.com/community/UbuntuHashes](https://help.ubuntu.com/community/UbuntuHashes)
