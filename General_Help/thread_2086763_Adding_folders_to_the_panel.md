### Post by LewisTM on 2012-11-21
The Directory Menu panel item lets you do that. You can set it to display any base directory in a tree. By default it will show only subdirectories but you can set it do show all files (*) or a subset (*.doc;*.pdf;*.odt).

Cheers!
