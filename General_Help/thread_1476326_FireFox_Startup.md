### Post by lovinglinux on 2010-05-07
If that page is opening when you start, like dracayr suggested, then it might be a problem with Ubuntu Firefox Modifications extension, unless you accidentally added it to your startup preferences. That "page" is in fact a localization (translation) file of  ubufox extension and in normal conditions it should be opened by the browser like that.

If that is the case, then re-install ubufox.

```
sudo apt-get install --reinstall ubufox
```

Anyway, please be more descriptive when posting help requests.
