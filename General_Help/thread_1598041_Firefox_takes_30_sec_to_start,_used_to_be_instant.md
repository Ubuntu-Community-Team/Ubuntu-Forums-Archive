### Post by HermanAB on 2010-10-16
Howdy,

You probably have a lame DNS in resolv.conf.

Check your DNS with 'dig'.

$ cat /etc/resolv.conf
$ dig @ dns.ip.add.ess [www.google.com](www.google.com)

If a DNS is lame, remove it.

### Post by lovinglinux on 2010-10-17
Most likely is not a DNS issue, which affects page loading only.

Startup time is mostly affected by extensions and databases. Optimize your databases and check if there is any extension slowing down the startup process.

See: [http://ubuntuforums.org/showpost.php?p=9904339&postcount=3](http://ubuntuforums.org/showpost.php?p=9904339&postcount=3)

If that doesn't help, then try a clean profile.

[Fixing a problematic or corrupted profile](http://firefox-tutorials.blogspot.com/2010/05/fixing-problematic-or-corrupted-profile.html)
