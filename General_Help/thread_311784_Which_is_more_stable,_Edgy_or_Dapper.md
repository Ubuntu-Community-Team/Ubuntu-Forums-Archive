### Post by johnnymac on 2006-12-03
They are both stable.  Biggest difference is that Dapper is LTS (Long-Term-Support) and Edgy will be replaced in a few months with Feisty Fawn.  So if you need something that you don't want to change a bunch - go with Dapper until the next LTS.  If you don't mind re-installing or updating a bunch of files later - go with Edgy.

Either way they are both good.  I run Dapper and virtualize Edgy for development purposes.
