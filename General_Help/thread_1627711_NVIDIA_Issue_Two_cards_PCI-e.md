### Post by papibe on 2010-11-21
It may me possible that nvidia-settings would be able to recognize your cards. Run:
```
# nvidia-settings
```
you can even save the settings you do with the tool to your xorg.conf, but you need to be root:
```
$ cp /etc/xorg.conf /etc/xorg.conf      # backing up your file.
$ sudo nvidia-settings
```
then when you find your settings acceptable, go to X Server Display Configuration and select Save to X Configuration File.

I hope it helps.
