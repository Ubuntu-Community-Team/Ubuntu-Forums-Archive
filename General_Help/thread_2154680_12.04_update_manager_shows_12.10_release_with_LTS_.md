On two other machins this is not happening. I have checked /etc/update-managet/release-upgrades and it contains "prompt=lts".  Is anone else seeing this and is there a fix?

### Post by rsteinmetz70112 on 2013-06-17
Checking further it seems the Launcher Icon on one machine starts Update-manager with the -d switch. This causes it to display the new release. Then next question is where is it getting that from? I can't seem to find a fils with that option specificed.


I discovered tha the was a file 
~/.local/share/applications/update-manager.desktop 
which didi include the -d command line switch. Editing the file changed the behavior back to what was expected. 

I must have made that change myself and forgotten all about it, probably when I was waiting fro 12.04.1 to be released.
