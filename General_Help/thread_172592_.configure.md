### Post by blackjack6.21.91 on 2006-05-09
you are supposed to use the cd command to enter the directory with the program you want to install

for example, if the folder's on your desktop

cd ~/Desktop/foldername
(remember it's case sensitive)

then just
./configure
make
make install
and you should be ok
