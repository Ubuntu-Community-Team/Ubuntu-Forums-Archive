### Post by NeoLithium on 2007-06-01
Chances are that your CPU has the governor enabled. Which is nothing to worry about; it will keep your processor at half speed until the computer realizes that it needs more power, and will kick it up to a full clock speed.  If you really feel like hating it, check out [http://www.ubuntuguide.org](http://www.ubuntuguide.org)  There is a section on CPU frequency scaling that can help you set it up to maintain it at full throttle, if you think you prefer it that way.
