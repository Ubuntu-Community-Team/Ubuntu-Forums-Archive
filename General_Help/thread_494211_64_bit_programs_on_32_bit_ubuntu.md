### Post by Ayuthia on 2007-07-06
You can't run 64-bit on 32-bit.  It can only be done the other way around because 64-bit addressing will not fit into a 32-bit address.  It is kinda like trying to pour a quart of beer into a pint glass.  The quart of beer looks like, but it will not fit into the pint glass...
