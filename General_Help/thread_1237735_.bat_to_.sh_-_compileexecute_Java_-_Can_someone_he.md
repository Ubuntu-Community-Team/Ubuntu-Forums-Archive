### Post by geirha on 2009-08-11
Not sure I fully understand what that script does, but I believe an equivalent will be something like this.
```

#!/bin/sh
echo "Compiling..."
if ! type javac >/dev/null 2>&1; then 
  echo >&2 "Sun Java(TM) Development Kit (JDK) is not installed."
  echo >&2 "Install it with 'sudo apt-get install sun-java6-jdk'."
  exit 1
fi
javac -cp . *.java && echo "Finished successfully!"

```

Though, isn't it easier to just run javac *.java manually instead?
