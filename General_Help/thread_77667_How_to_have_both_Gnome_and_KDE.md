### Post by Pablo_Escobar on 2005-10-17
Yes it is possible.
If You want to have kde enable in synaptic install package called "kde-desktop". 
It'll install all the neccesary packages. Then logout and under seesions You can choose which DE you can use. Most of the installed apps will be available in both DE.
