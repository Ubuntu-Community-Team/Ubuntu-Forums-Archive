### Post by drs305 on 2008-03-10
In a terminal, you can type
```
xev
```
which opens a window. When you type any key, it will then return the name and values of the key. This is useful information if you are trying to switch key results.  On my keyboard, I have two Window keys, Super_L and Super-R.
