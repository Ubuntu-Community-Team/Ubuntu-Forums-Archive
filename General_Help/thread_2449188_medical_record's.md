### Post by HermanAB on 2020-08-22
Howdy,

I have encountered a medical X-ray picture collection once before.  What it amounted to, was a vast collection of jpeg files, with an index file and special Windows software that could read the index and allow the doctor to quickly scan back and forth in the sequence of pictures.  

On a Linux system, one can simply ignore the index file and use the file browser to click anywhere in the sequence on a file to view it.  I don't think the filenames had a .jpg extension, which will confuse Windows machines, but a Linux machine doesn't use the file name to know that it is a jpeg.

If your data is similar, then simply copying the whole collection to a DVDRW should be all you need to do to make it good for Windows.
