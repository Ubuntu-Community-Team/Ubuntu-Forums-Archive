### Post by bobleny on 2010-01-23
Generally, my laptop runs hotter under Kubuntu than vista. However, it runs hotter while playing video games under vista than it has ever ran under kubuntu.

My battery lasts longer while running Kubuntu by about one half of an hour.

At idle, with no programs running, Kubuntu uses about 300MB of memory where as windows uses about 750MB of memory.

Provided that the CPU load given by the KDE system guard is measured in megahertz, it runs, again at idle, at about 8 megahertz verses the 176 megahertz used by vista. I should note however, that the calculation done for vista is based on the percentage of the processors maximum frequency. This percentage bounces around a lot, but seems to average around 10%. This is for the most part a default vista install, I have only disabled a few system services, and got ride of all the background application possible.

I will also note, that from a users stand point, Kubuntu runs much faster than windows...

I am dual booting on an acer laptop between windows vista ultimate and Kubuntu 8.04, running KDE3. It has an Intel Celeron clocked at 1.73Gh, 2GB of DDR2 memory, and 252MB Intel mobile graphics.

I hope this helps.
