### Post by Bungler on 2010-04-19
You can probably remove it using the synaptic package manager. Without removing it you can prevent it from starting up using: System -> preferences -> Startup application

---

### Post by Rubi1200 on 2010-04-19
If you removed Evolution completely, go to System -> preferences -> Startup application as Bungler suggested and remove the entry there (use the remove option not just untick). It seems that some programs leave a startup entry even after removal.
Hope this helps.
