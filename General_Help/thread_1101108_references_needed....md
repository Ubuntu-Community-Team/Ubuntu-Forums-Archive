### Post by mb_webguy on 2009-03-20
No.  Linux doesn't use anything like a registry.  The [filesystem]("http://en.wikipedia.org/wiki/Filesystem_Hierarchy_Standard") is structured so that applications and associated files go in specific places, and the system doesn't need anything extra to keep track of them.

Here's a few good links to get you started with Linux.
[The Ultimate Linux Newbie Guide]("http://www.linuxnewbieguide.org/")
[Linux is not Windows]("http://linux.oneandoneis2.org/LNW.htm")
[The Ubuntu Guide]("http://ubuntuguide.org")
