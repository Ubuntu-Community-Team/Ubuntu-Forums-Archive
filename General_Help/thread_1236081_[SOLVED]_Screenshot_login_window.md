### Post by Muscovy on 2009-08-09
How can I screenshot my new login window for the gdm preview?

---

### Post by aysiu on 2009-08-10
Install *xnest*: ```
sudo apt-get update && sudo apt-get install xnest
```

Then launch the command ```
gdmflexiserver --xnest
``` and then ```
gnome-screenshot
```
