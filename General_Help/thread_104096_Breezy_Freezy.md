### Post by Zelut on 2005-12-15
You mentioned you were going to make it a web server?  Why would you have the gnome environment installed on a machine that wont need it?

Unless you've made a lot of progress I might suggest reinstalling using the 'server' option & only installing the base system.  This will cut out a lot of possible bugs that could be connected with gnome (or related packages) and also give your server more resources and drive space.

I have a web server setup using Breezy 5.10.  If you need some tips or help on reinstalling for server let me know.  I'm glad to help.
