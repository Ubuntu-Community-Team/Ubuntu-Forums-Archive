### Post by Sorivenul on 2008-08-29
Are you looking to encrypt the whole drive, a specific folder or set of folders, specific data?  There are different methods to use here.  Full hard drive encryption from my understanding has to be done during install using an encrypted LVM setup.  I personally use [TrueCrypt]("http://www.truecrypt.org/") to encrypt my data.  You would have to download the package from the website to install as it is not in the repositories - however there is an easy to use GUI frontend called EasyCrypt in the repositories if you have problems with the traditional EasyCrypt GUI.
