That depends.

But if chrome's installation failed and update manager, is not working, then open the update manager go to the section in the left hand bottom marked "settings", open it and then when the new window pops open go to "other software" and see if any entries for google are listed, if so then uncheck them, close the window and hit check in the main update manager window..
