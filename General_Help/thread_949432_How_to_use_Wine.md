### Post by ronnielsen1 on 2008-10-16
Assuming you have wine installled you can right click on the exe and select open with wine. You can file associate to do this automatically but I don't do this because older exe's I use with dosbox. 
After you install wine, open a terminal and type in winecfg. This will open a wine config program. Select drives tab and autodetect. You can also tweak wine here.
