### Post by apetresc on 2007-08-09
> **EdUbun said:**
> I want to install something like source safe on windows.
What can i use for this?
There's quite a lot of version control systems for Linux. One of the most popular (and the best, in my opinion), is Subversion. It's open source, free, powerful, and quite ubiquitous. You can install both the server and the client simply by doing ```
sudo apt-get install subversion
``` And you can get some good documentation on using the system from this (free) book right here: [http://svnbook.red-bean.com/](http://svnbook.red-bean.com/)

Enjoy! Feel free to ask any questions here, a lot of us use Subversion on a very regular basis.
