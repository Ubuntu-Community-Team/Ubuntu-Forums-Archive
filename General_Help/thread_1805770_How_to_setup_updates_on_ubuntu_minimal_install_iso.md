### Post by Frogs Hair on 2011-07-16
Use , Update Manger > Settings to change update settings . You should receive the updates the minimal installation requires and if you add officially supported packages they should be updated also .

### Post by snowpine on 2011-07-16
The Minimal Install downloads everything from the internet, therefore it is normal for your system to be up-to-date immediately following a Minimal Install. Once some updates become available (give it a few days/weeks) then you can apply them. :)
