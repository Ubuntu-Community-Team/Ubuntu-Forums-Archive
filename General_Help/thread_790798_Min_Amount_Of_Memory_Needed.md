### Post by ibuclaw on 2008-05-11
256MB Sounds right to be the minumum to me.

Let me just get out my Memory Footprint Comparison sheet...

Yep. Gnome Default Desktop (Complete Desktop) takes up  around 480MB of Main Memory space (KDE 4.0 only required 390MB).
So there will be a lot of activity between the RAM and Swap devices.

If you want a better equipped OS for your low RAM PC, try out Fluxbuntu or Xubuntu.
They are light desktop managers (requiring only around 64-128MB), so your PC will run more efficiently.

Although, just a small note. I suppose almost all modern PCs of today would be expected to have of the very minimum 2GHz Single Core Processor and 512MB RAM. So just making a small upgrade won't hurt.
