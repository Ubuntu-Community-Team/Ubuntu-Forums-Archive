### Post by bodhi.zazen on 2012-08-28
If you are serious about web design ...

Use any text editor (vim, gedit, bluefish) and use as many browsers as you can to preview what you write. I advise bluefish.

WYSIWYG editors write sloppy code and what browsers display may or may not correlate to what you see with the editor.
