### Post by Idefix82 on 2008-09-19
I think, Ubuntu is a far superior operating system compared to Windows, but to make use of it you need to be prepared to learn new stuff. A lot of people try to use Ubuntu like another more stable windows and of course they come to the conclusion that Ubuntu is not good at that and sometimes switch back without ever experiencing the power of Ubuntu.
What I am saying is that if you decide to switch, I would recommend you to try and learn as much Linux native software as possible instead of just porting everything from windows and running it in the emmulator. I found that most Ubuntu alternatives are much more flexible, user friendly and powerful, provided one is prepared to learn how to use them properly.
