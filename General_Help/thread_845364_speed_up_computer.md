### Post by finer recliner on 2008-06-30
next time you see the HDD light going crazy, open up a terminal and type the command
```
free
```
this will show you how much memory you are using. ideally you should be using zero swap (virtual memory on the HDD). if you are using alot of swap, then you should  upgrade your memory. it's probably the cheapest way to upgrade the speed of your computer anyways.
