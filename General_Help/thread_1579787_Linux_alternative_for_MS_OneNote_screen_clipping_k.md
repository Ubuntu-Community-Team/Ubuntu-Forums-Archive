### Post by JRBye on 2010-09-22
If you just tap the Print Screen key you will get a dialogue box that will contain the entire screen. You can copy that to the clipboard and paste it into a program if you like or you can save it. If you are talking about taking screen shots of just certain parts of the screen then I would recommend Shutter. It is a free program. You can get it from the Software Center.
