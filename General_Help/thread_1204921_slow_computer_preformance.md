### Post by Waappu on 2009-07-05
Hi

Go Process tab and then View->All Process

also if you type in terminal
```
top
```
you can see all processes.

One helpful app is also htop.
you can install that with command```
sudo aptitude install htop
```

and then start it by typing in terminal```
htop
```
