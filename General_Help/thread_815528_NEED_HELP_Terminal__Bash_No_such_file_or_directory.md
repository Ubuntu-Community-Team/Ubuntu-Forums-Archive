### Post by phidia on 2008-06-01
It will help you if you type > ls and press the <enter> key in the directory you are working in because then you will see how the files are actually labeled. I think desktop is actually Desktop it makes a difference in linux if you are typing upper or lowercase. Also commandline completion is a real saver-just type a few letters of the filename and press the tab key if bash "sees" the file it will complete the name for you.
Hope that helps.
