### Post by zerubbabel on 2012-09-06
I have a large archive of fonts (about 16,000 fonts) that I need convenient access to, without actually installing them all and bogging down my system. 

I tried Font Manager (from the repositories), but when I add the folder that contains this archive, it effectively installs all the fonts, and then programs like LibreOffice take forever to load, because apparently they have to provide menu access to all those fonts. 

So I'm looking for a tool that I can use to search my archive without actually installing all the fonts. Then once I find the particular font I need, I can install just that particular font or font family. 

Does anyone know of such a tool?

### Post by zerubbabel on 2012-09-18
Found a solution: the Opcion Font Viewer (opcion.sourceforge.net)
