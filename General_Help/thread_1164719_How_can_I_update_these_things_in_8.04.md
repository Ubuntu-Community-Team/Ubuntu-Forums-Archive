### Post by pytheas22 on 2009-05-19
Upgrading some applications to the Jaunty versions is going to be really difficult.  There are instructions [here]("http://ubuntuforums.org/showthread.php?t=797059") for updating NetworkManager using a custom repository on launchpad.  I don't think you'd be able to upgrade Gnome to 2.26 without breaking a lot of other stuff.

Your best bet is probably just to use Jaunty if you want the updated applications.
