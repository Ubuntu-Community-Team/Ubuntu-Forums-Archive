### Post by Dinodoc on 2009-11-28
Okay I have no experience on 9.04, but I can see how to do this on 9.10 so hopefully its the same.

Go to the System Menu > Administration > Login Screen

In there, there is an option to login as ... automatically, you can even have a delay beforehand so if at any time you want to login as another user you can before it automatically logs in.

Hope this helps.
