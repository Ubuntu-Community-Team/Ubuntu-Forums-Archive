### Post by amadeus266 on 2007-10-13
I believe the problem is with the amount of free space. I have had this happen in windows as well. If I'm not mistaken, when moving a file the system actually copies the file to the new location and then deletes the original. You need to have enough space for twice the size of the file you are moving and then some more for the operation to complete successfully. This is also why, in windows, you cannot run a defrag unless you have a certain percentage of free space. As far as recovering the lost files, you may need to consult a professional data recovery service and still there is no guarantee that they can get all of it back. Good luck.
