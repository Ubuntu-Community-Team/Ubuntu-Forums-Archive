### Post by jjex22 on 2011-11-26
I wouldn't have thought that it'll be an issue for a lot of users, but it is worth highlighting that the hardware support is still for two years - after that it's maintenance - bug and security patches etc - this is good as it allows that focus to switch over to what will then be 14.04.

As we all know 2 years is a massive time frame in computer development, and if recent paterns are anything to go by linux even more so (makes sense - it's the opensource model - exponential growth) So I'd imagine that even with this 5 year release relatively few box's will be still on 12.04 in 5 years, but it's really nice to have that option - especially for the older hardware - as new features make the OS heavier it does give them a few more years!

And let's be honest, unless you're a gamer, or use some other high resource program, for the average home user, basic computers are currently far exceeding their requirements - a quick look on pcworld - £300 for a 4gig 1TB dual core - so 64bit ready, and it's going to store holiday snaps and keep an eye on facebook. I think this is the geniousness of the 5 year plan - not so long ago the budget pc's were so out of date they could barely cope with the version of windows they shipped with, but combining this 5 year model with the fact many of these machines are so overpowered, they'll last a lot longer for those that need them to - I'd be very surprised if in two years Ubuntu was unable to run on such a machine - giving it at least a 7 year life - for £300
