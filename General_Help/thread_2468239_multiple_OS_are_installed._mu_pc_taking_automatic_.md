### Post by yancek on 2021-10-22
If I understand your situation, you have Ubuntu and windows currently installed.  Is your computer now automatically booting to windows?  Since installing Ubuntu, have you successfully booted Ubuntu?  If so,  what changes have you made that might have affected booting?  If Ubuntu booted earlier, it is not likely to stop booting without some change to the boot process or some hardware failure.  Are windows and Ubuntu on the same drive?  Do you still have your Ubuntu install USB?  If so, boot it and go to the site below to download and run boot repair.  Select the 2nd option described on the page and run it with the Create BootInfo Summary option.  When it finishes, it will show a link which you can post here to get help as what you have currently posted is not enough to get any help.

[https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)
