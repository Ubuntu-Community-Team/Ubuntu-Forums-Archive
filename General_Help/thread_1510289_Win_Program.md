### Post by Mark Phelps on 2010-06-15
Don't understand your question.  Are you asking how to install and play MS WIndows games in Ubuntu?

If so, you need to do the following:
1) go to the WineHQ site link below and do a search on the game and version you want to run.  Unless it's rated Gold or better, you'll be wasting your time:

[http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true")

2) Install Wine in order to install the game later.  For more details, go to the Wine subforum.  When there, ask for info on,or look around for posts on, PlayOnLinux and WineTricks.  Both are extras that will make installing games easier.
