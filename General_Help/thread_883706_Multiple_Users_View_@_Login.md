### Post by DJ_Peng on 2008-08-08
You can go to System > Administration > Login Window (your admin password will be needed) > Local and select any Login screen with a face browser. The face browser is the listing of accounts so you can just click the account and put in the password.

### Post by ad_267 on 2008-08-08
It will work. :)

Just make sure that the "Style" drop down box is set to "Themed with face browser" then select Human List for the theme. Make sure you click on the little circle to the left to select it.
