### Post by oldfred on 2015-04-08
If system has issues you have to be able to boot another repair CD/DVD/flash drive. If that is not possible then you have to remove hard drive and install it in another system to fix it.

If you only have Lubuntu, then you should not get grub menu. You have to hold shift key down to get grub menu. Grub is both the boot manager (menu) and the boot loader of Linux systems.
Can you boot second line in grub menu or recovery mode to command line?

What brand/model system? 
What video card/chip does it have?
