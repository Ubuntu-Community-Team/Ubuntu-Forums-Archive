### Post by DawieS on 2011-06-12
My previous computer had the same behaviour with a USB mouse, even with Win XP. Also, whenever there was a power failure, or the power was disconnected from the wall socket, the mouse had to be re-plugged first to get it working.
I eventually solved the problem by using one of these USB-PS2 adapter plugs, and concluded that the motherboard BIOS USB interface was somewhat buggy.
