### Post by 2handband on 2008-12-22

### Post by sdennie on 2008-12-22
If the software fundamentally won't run in a 64bit environment, you could always use a 32-bit virtual machine to run the software.  This may not be the answer you are looking for but, it will work.  I recommend VirtualBox: [http://www.virtualbox.org/](http://www.virtualbox.org/)

---

### Post by beno1990 on 2008-12-22
Unpack the software from a .deb file using another archive manager (Several regular Linux package managers will unpack .deb files) and run it from your home directory, as long as other users don't need to use the software too.
