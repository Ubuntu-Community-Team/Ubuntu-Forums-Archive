### Post by Dennis N on 2020-02-27
You can do it from the gnome shell extensions web page:
[https://extensions.gnome.org/](https://extensions.gnome.org/)
If you haven't set up browser support, do that first (directions will appear at the top of the page), then click on the "installed extensions" tab at the top. The green gear icons will update an extension.
