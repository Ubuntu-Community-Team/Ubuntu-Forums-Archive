He is asking for something completely unrelated :-)

I've never used Scim to type Bengali characters, but if you might have to use a *virama* or *hasant* (unicode character [U+09cd]("http://www.fileformat.info/info/unicode/char/09cd/index.htm")) to indicate no vowel follows the consonant. At least, that is what you need to do when you use [Gucharmap]("http://gucharmap.sourceforge.net/") in Ubuntu and other software I've used in Windows. If Scim acts similarly, it will transform a consonant+virama+consonant in the proper ligature or *yuktaksara*.

I hope this helps.
