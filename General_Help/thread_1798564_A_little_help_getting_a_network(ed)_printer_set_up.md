### Post by Topsiho on 2011-07-18
I don't know about the HP Officejet 5600 MFD.
But HP printers often are extremely easy to set up.
You need to install hplip first, and then often you'll find (maybe after some time, to let the system(s) find it) there it is, and can be used. But next may be necessary first:

You can use it on other systems on the network if you do this:

On the server (the computer that the printer is connected to with a USB-cable): select System>Management>Printing>Server>Set up and tick Publish shared printers connected to this system.
I retranslated this from Dutch, so the English wording maybe a little different.

On the other computers you do exactly the same, except that in the last step you tick Show printers shared by other systems.

I wish you good luck :)

Topsiho

[http://www.techrepublic.com/article/control-printers-in-linux-from-the-command-line/5055067](http://www.techrepublic.com/article/control-printers-in-linux-from-the-command-line/5055067)

sudo lpadmin
lpstat -t

Command to print:
lp -h 192.xxx.x.x:631/printers/deskjet-5600 /home/yourhomedirectory/mytest

response to request:
request id is deskjet-5600-305 (1 file(s))

^mytest = test file I made in vim

Once it is all working you can make an alias to put the first part of command in and can type the directory and file name, if you want.
