### Post by akoskm on 2011-08-19
The easiest way to get an accurate answer about a specific game is to visit:
[http://appdb.winehq.org/]("http://appdb.winehq.org/").
This is a well maintained database about applications ever tested/reported on wine, with test results, workaround and such as.
