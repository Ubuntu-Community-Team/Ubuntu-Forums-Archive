### Post by 2F4U on 2012-01-01
Yes, if the Additional Drivers utility doesn't suggest other graphics drivers, you are using the open source ATI drivers. In that case you could try to add nomodeset to the grub kernel boot parameters. It helped on one of my laptops to get suspend/resume working.
Some information about the whole process of suspend/resume is contained in log files
- /var/log/pm-powersave.log
- /var/log/pm-suspend.log

### Post by plucky on 2012-01-03
> How would I go about adding nomodeset to the grub kernel boot parameters?


Edit the file /etc/default/grub using ```
gksudo gedit /etc/default/grub
``` and put **nomodeset** in the line as shown ```
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"
GRUB_CMDLINE_LINUX="nomodeset"
```

Then run ```
sudo update-grub
``` to get the line into grub.cfg so it is there at boot time.

Good Luck
