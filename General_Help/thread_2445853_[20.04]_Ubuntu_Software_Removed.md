For Flatpaks, I look at [flathub.org]("https://flathub.org/home"). When I find what I want, I copy the install command it gives on the application's page, and paste it into my terminal to install it.

I like these always up-to-date packages (Snap and Flatpak) for some of the most-used applications in the LTS release. It keeps them current. On balance, I have more Flatpaks than Snaps. There is no general rule that one system is better than the other. I often test both and use whichever seems best.
