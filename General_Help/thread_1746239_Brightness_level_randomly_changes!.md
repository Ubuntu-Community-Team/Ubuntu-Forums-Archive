### Post by ReMO451 on 2011-05-01
Hi there,

I recently did a fresh install of Natty 64bit. Everything seems to be working fine except my brightness will randomly lower. The notification bubble in the top right will show up and display the animation saying that the brightness is decreasing, and the display will get dimmer. This happens as if I'm manually changing it, but I am not. 

This happens every ~5 minutes or so, on AC or battery power. The power management settings are currently at default. Adjusting via the Fn keys or the brightness applet works fine. This keeps happening on Unity, Ubuntu Classic, and Gnome-shell. I did another fresh install last night and the problem keeps happening.

This happens fairly randomly - can't trace it down to a specific program or action. 

The laptop is a Lenovo Y460. It has a sensor to dim the screen when the outside light level is high, but that only works on Windows. I've never had this problem with 10.04 or 10.10, along with Fedora 14.

Anyone have a clue as to how to fix this?

Thanks
