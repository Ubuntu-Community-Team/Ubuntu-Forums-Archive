### Post by skymera on 2010-02-20
I suggest a dual boot.

This way you can have Ubuntu and Windows on the same computer. I suggest dual boot becasue i see you play games.
Ubuntu can't run Windows games natively. WINE can allow .exe files to be run.. but it's flaky and a last resort.

> Wireless? It's generally pretty good and there's always a way get it working?

> Video drivers? Intel drivers are built in and Nvidia/ATi can be easily installed.

Run the Ubuntu LiveCD. This can give you a basic feel of Ubuntu without the hassle of installing. If yuo like, you can then install! :)
