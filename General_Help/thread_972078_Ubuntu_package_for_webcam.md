### Post by chronographer on 2008-11-05
Cheese is rather good. Install it with "sudo apt-get install cheese" and then run it from command line with "cheese".

It is a webcam viewer, recorder and has some filters and effects too! If you webcam is supported then it will work with no input at all, if it is not supported then you will need to do some digging... try 'lsusb' to find your webcam type and then search google "webcam ubuntu <insert webcam type here>"

Hope it works!
