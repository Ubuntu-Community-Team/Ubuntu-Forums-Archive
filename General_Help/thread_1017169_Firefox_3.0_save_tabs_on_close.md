### Post by alphaniner on 2008-12-20
The out-of-the-box behaviour of FF3 when closed with multiple tabs open is to ask if you want them to be open again when FF is restarted.  I made the mistake of checking the "Don't ask me again" box when I clicked "Yes."  My thinking was that - like every other piece of software I have ever used - it would save my selection as default.  Whoopsie!  Now when I close FF with multiple tabs it just asks me if I'm sure I want to close it.  How do I get it to save open tabs by default or, barring that, at least get back the dialog that asks me if i want to?  I didn't see anything in Preferences or about:config.

---

### Post by edd07 on 2008-12-20
Go to Edit>Preferences and choose "Show my windows and tabs from last time" in the "When Firefox stars" option.

Is this what you needed?

Hope this helps

### Post by trbaugher on 2008-12-29
I had this problem too, ff 3.0.5, Ubuntu 8.10. I discovered that clicking File->Quit (Ctl+q) lets ff save the tabs, while clicking the x (Alt+F4) does not.
