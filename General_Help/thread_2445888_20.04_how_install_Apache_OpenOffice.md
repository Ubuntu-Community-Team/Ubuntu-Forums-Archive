### Post by deadflowr on 2020-06-21
You used to just need to remove LibreOffice and install OpenOffice.

Then if you want LibreOffice,
you can either reverse it, or look into alternate packagings of it like flatpak or snap.

AFAICT both use the /usr/bin/soffice file which is really just a link.
So in theory you could just unlink it and relink it to which ever suite you want to use.
But any updates will break that,
or worst case, break the update system.

So usually easier to just remove one entirely.
