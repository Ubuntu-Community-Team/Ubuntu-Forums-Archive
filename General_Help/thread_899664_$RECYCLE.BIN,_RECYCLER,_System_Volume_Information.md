### Post by victor.zamanian on 2008-08-24
Hey sstecken!

I have the same problem. I just delete them at random and don't experience any personal inconveniences with my drives (they are FAT32, by the way).

If someone urges you not to delete them if you access them with Windows as well, then maybe you can create a hidden folder (one with a "." as the name's initial character, for instance: ".windows") and put the folders in that while you use Ubuntu, then before you use Windows, copy them out again. Possibly add that to some sort of script that you run as you log in/out, or something.

Good luck!
