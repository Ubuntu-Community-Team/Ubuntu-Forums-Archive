Would it be possible to sync a Zune while running Ubuntu? Would I have to use a virtual machine or dual boot?

---

### Post by mb_webguy on 2009-02-01
Since Zune is a Microsoft product, they of course made it difficult to use with anything but Windows.  AFAIK, you can't currently sync with a Zune using Linux, so you'd have to dual-boot or run a VM.  Work is being done in the areas of allowing Linux apps to sync with a Zune and to install Linux on a Zune, but I don't think there's been much progress from an end-user's perspective.
