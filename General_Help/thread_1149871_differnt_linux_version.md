### Post by maheshasolkar on 2009-05-05
Most Ubuntu versions differ based on the desktop environment they use. Ubuntu uses Gnome, Kubuntu uses KDE, Xubuntu uses XFCE, etc. Its a matter of personal choice which desktop environment one likes. Although, there are sometimes technical reasons too - for instance, Xubuntu is a better option on old/less powerful hardware, since XFCE is a lighter desktop enviornment.

If you already have Ubuntu installed, I would suggest you install VirtualBox and try all the flavors of Ubuntu in VirtualBox. That way you can decide which desktop environment you like most, without having to lose the existing Ubuntu installation.

Other way is to use Live CDs or USB to try out different versions of Ubuntu. In both cases you will need to download the .iso files from:

  [http://releases.ubuntu.com/jaunty/](http://releases.ubuntu.com/jaunty/)

If you wish to use USB boot disk, you can use the USB Creator tool available in System -> Administration, and boot off the USB disk.

If you wish to use Live CD, just burn the .iso into a CD using, say, Brasero.
