### Post by mcduck on 2009-06-22
How did you install it? From repositories, using a .deb package or from source code? Or some other way?

If you used repositories or .deb package it can be removed using "sudo apt-get remove *packagename*", or with Synaptic.

For source code, the source package should have uninstaller or instructions how to manually uninstall the program. I recommend using Checkinstall instead of "make install" to install programs from source, as it will create .deb packages for the program and use them so the program can be uninstalled easily with apt-get/Synaptic.


### Post by merciadriluca on 2009-11-07
The good thing about xflux (aka F.lux) is that it is a stand-alone program. You just execute the binary, and it works. A really nice thing. Note that it is, to my knowledge, not in the repositories, and that you are thus neither able to install it using apt-get, nor to remove it with apt-get too.
