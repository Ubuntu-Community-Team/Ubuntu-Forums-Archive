### Post by Giblet5 on 2009-11-18
Desktop will work fine as a web server.

I have a desktop edition box running a web server, a samba server, an NFS server, a DNS server, a media server, pop3 and imap servers, and it acts as an smtp relay.

Pentium III laptop with 256MB of RAM. Works great, sits mostly idle.

There isn't a lot of difference between the server or desktop installs... I think server installs skip some stuff like OpenOffice, and others.
