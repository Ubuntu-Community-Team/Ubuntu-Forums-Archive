### Post by R3N3G4D3 on 2008-09-16
I took the g15macro code written by Mike Lampard, and modified it to simulate G7-G18 keys on G15 version 2 keyboard. I'm attaching the source for those who want this functionality. I've also submitted the code to Mike, so he can include it in next g15macro version if he chooses to. I used version 1.0.3 of g15macro as the base.

Here is how this version of g15macro works:
-If you press M1 once, you access M1:G1-G6 keys
-If you press M1 again (while in M1:G1-G6 mode), you now control M1:G7-G12 (M1 light starts blinking so that the user can identify the mode)
-If you press M1 a third time, you access M1:G13-G18 (M1 light starts blinking faster)
-If you press it a fourth time, you go back to M1:G1-G6 mode
-If you press M2 while in M1:G7-G12 or M1:G13-G18 mode, you go to M2:G1-G6 mode (same applies to M3)
-All of the above also apply for M2 and M3
-While this mod is useless for G15v1 users, it does not break any G15v1 functionality (pressing the same M key on G15v1 will rotate the keys, G1 would become G7, while G18 would become G6)

I hope those with G15v2 find this feature useful.
