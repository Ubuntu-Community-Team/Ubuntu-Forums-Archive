### Post by misGnomer on 2005-01-06
Please note that *the latest Gnoppix beta, hoary_0.9.2b1-i386.iso, appears to be an early beta of Hoary **install CD** and not a live CD* as expected.

Under "expert" mode (install) boot there is an option called "start scripts for the liveCD" but even that failed here. People interested in trying an actual disk install of Ubuntu (Hoary) might be better off trying [Ubuntu's own daily builds](ftp://cdimage.ubuntulinux.org/cdimage/daily/).

Guess we need to wait for a real Hoary live CD beta for a while yet.

Meanwhile there's another new ISO at [MDZ's ubuntu-live directory](http://people.no-name-yet.com/~mdz/ubuntu-live/)  titled "hoary-live-i386-**alpha1**.iso". That one's dated 06 Jan 2005 and it's 583MB in size, compared to the non-liveCD Gnoppix's 852MB. I haven't tried that one yet.

[Alextreme](http://www.alextreme.org/drupal/) of Morphix fame (another Ubuntu liveCD collaborator) has also released an [updated Morphix "Light-GUI" beta](http://www.morphix.org/modules/news/) version featuring the GTK-based Xfce desktop (270MB) but there's no updated Gnome build yet.

### Post by misGnomer on 2005-01-25
**Amu**, thanks for correcting my mistaken first impression of the newly-designed boot method that looks like install CD but halfway through kickstarts a live CD session. Since an actual new install may potentially blow away user's boot sector, it might be useful to add a brief notice at the start of the boot advising the user that the first screens do resemble an install. OTOH some day somewhere an unsuspecting user with both an Ubuntu install CD and live CD at hand may inadvertedly insert the install CD thinking it'll just come up with a live session and POOF...  ;-)

**Jerry**, Gnoppix has adopted the new-fangled Ubuntu way of creating a live CD session, instead of the previous method based on Knoppix/Morphix. In fact, the contact information at the "official Gnoppix site" has now (today?) been changed from Credativ (the original Gnoppix company) to Canonical (the Ubuntu company).

I don't know if the Gnoppix branding will be kept in the long run, but both Gnoppix and Ubuntu teams appear to work towards the same goal of making kick-ass Ubuntu live CD based on the install version of Ubuntu. Maybe the developers can explain the changes and future direction better but that is my understanding of the events.
