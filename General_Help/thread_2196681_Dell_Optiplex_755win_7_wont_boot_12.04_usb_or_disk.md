### Post by dsc1596 on 2013-12-30
Make sure you are setting the disc/usb drive as the first boot device in the BIOS or alternatively using the one-time boot menu. BIOS can be entered with the F2 key, and the one-time boot menu is F12 if memory serves. Also, if you do use the BIOS, verify that the usb disc is recognized there. If it is not, try a different USB port as well, sometimes the ones in the back of the computer are more reliable than the port(s) in the front. 
Also, which program did you use to create the USB disc? I have used LiLi in the past myself and not had any trouble.
