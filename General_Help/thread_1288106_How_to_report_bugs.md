### Post by yabbadabbadont on 2009-10-11
First, search bugs.launchpad.net/ubuntu to see if there are open bugs for your issues.  If so, add your information to the existing bugs.  If not, open new ones.

For example, here is a list of the open bugs for the fast user switch applet:

[https://bugs.launchpad.net/ubuntu/+source/fast-user-switch-applet](https://bugs.launchpad.net/ubuntu/+source/fast-user-switch-applet)

And here is the list for boinc:

[https://bugs.launchpad.net/ubuntu/+source/boinc](https://bugs.launchpad.net/ubuntu/+source/boinc)
