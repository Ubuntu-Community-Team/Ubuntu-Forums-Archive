### Post by bettlebrox on 2006-10-16
Post your xorg.conf.

Is this a laptop/desktop? What kind of mouse do you have?

Maybe one of the apps your running has a memory leak? I'd suggest running top a few times a days (press M to sort by memory usage) and see if you see something using up lots of memory).

Also, look at the last few lines of "sar -r" (if you have sar installed), it will show what the memory has been and maybe we can see if your running out of memory.
