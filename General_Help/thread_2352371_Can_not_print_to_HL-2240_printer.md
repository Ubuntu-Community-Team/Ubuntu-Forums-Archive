### Post by Bucky Ball on 2017-02-11
Have you installed the drivers for it and then opened 'Printers' and 'Add Printer'???

[Try here]("http://support.brother.com/g/b/downloadtop.aspx?c=us&lang=en&prod=hl2240_us_eu") and you want the .deb file for your Ubuntu. When it's downloaded, double click and it will install. Go to printers. Delete whatever printer you've added in there already. 'Add Printer' and when you are asked to select a driver the correct one should be in the list. Choose it.

Don't know about the 2240, but easiest to set them up using a USB cable plugged directly into the computer (I have the 2270DW which is a wireless model and same for that).
