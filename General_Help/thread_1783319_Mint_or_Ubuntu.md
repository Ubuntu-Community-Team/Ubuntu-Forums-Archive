### Post by FormatSeize on 2011-06-15
It depends on what you want to do. If you are doing normal, everyday things, like word processing, data analysis, surfing the web, research, and the like, they are basically the same thing. On the other hand, if you are an oddball like myself that gets into your computer's insides to the point where you need to start to do wierd things, then you should go with Ubuntu.
 
Mint is a lighter distro as well, so how much pressure your system can handle may also need to be considered. I too have an older machine, and I had to make significant upgrades to the base system to get it to run without lags or hangs (but not so much that it can handle KDE4).
