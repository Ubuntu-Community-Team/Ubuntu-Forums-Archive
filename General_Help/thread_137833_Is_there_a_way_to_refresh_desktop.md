### Post by varunus on 2006-02-28
It looks like nautilus (the gnome desktop manager) isn't loading.  Try opening the run dialog box with ALT-F2, and typing "nautilus".  Does that work?

If your desktop doesn't appear, try opening a terminal, and typing "nautilus" (no quotes).  Do you get any errors?

Hopefully this isn't permanent; if you still can't get it to open, or if you have a blank desktop the next time you reboot, post back here, and we'll make it permanent.
