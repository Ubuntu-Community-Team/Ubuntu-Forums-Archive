### Post by doublemeat on 2010-02-22
rishabh_destiny gave terrible advice.  Your BIOS has nothing to do with it, and it *is* a problem with Ubuntu.  There are several bugs for several different versions of Ubuntu filed for it.

Fortunately the fix is easy.  Go here:

[AppleKeyboard]("https://help.ubuntu.com/community/AppleKeyboard#Ubuntu 9.04 (Jaunty Jakalope)").  

That link jumps to the solution for 9.04, which works on 9.10.  I use the local.rc method, but they all basically do the same thing.
