### Post by Rhubarb on 2007-06-02
The keyring manager can remember any ssh sessions that you make with nautilus.
Nautilus will actually prompt you if you want it to add it to your keyring.
I haven't tried storing any other passwords with the keyring manager yet.
I don't think gaim / pidgin uses the keyring manager.
