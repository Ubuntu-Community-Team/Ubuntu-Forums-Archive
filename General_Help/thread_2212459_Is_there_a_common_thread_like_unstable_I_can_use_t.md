### Post by gifford on 2014-03-21
You can use the testing unstable version which is currently 14.04. There is a forum for it which is Ubuntu +1.
However, if this is going to be used on your day to day machine, perhaps you should reconsider. You can always get new additions to the current stable release by going to your software sources and under updates, check to install pre-released updates. Again, sometimes this may make your system unstable.
