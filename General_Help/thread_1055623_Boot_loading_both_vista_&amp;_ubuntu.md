### Post by doas777 on 2009-01-30
I usually use [Startup manager]("http://packages.ubuntu.com/intrepid/startupmanager") to do this (if i bother.). On the Advanced Tab, check "Limit Number of Kernels to Keep", and set the combobox to '1'.

personally i always keep 2 kernels, just in case a kernel update goes awry, and i need to roll it back.
