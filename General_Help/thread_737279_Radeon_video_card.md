### Post by UTF16 on 2008-03-27
I've got a Radeon 9600 video card. It works absolutely fine on Windows.

I've recently installed Ubuntu, and was stunned how well it performed complicated tasks (like video and midi playback, Windows application work faster than on the original system etc.) and how terrible it did the most simple things, like web browsing. Scrolling in any application (like Firefox or file manager) is unsupportably slow. I don't even talk about Flash animations, because I don't call "playback" 10 frames per second full screen (and this, of course, if Firefox doesn't crash). ePSXe, the PlayStation emulator, doesn't work even with software plugins. And, for some unexplained reason, video playback in any player is perfect in every aspect.

Having googled some information I found out I was using open source drivers and that I might want to try something ATI itself developed. I've installed those drivers via the proprietary driver installation manager. This solved the slow scrolling and flash animation problem. But it did something with all my players – any video in full screen contains squares and the playback is extremely low quality (produces artifacts in motion). ePSXe does work with hardware plugins, but is unsupportably slow. Software plugins work, but they are as slow as hardware ones. All my cool Ubuntu desktop window animations were gone and replaced by very slow Windows-like animations.

This is very odd... I just used a Knoppix CD and tried all those tasks I was complaining about using only the default configuration. Everything worked fine! ePSXe didn't work in hardware mode, but everything else was absolutely fine.

There is probably something I missed, isn't there?
