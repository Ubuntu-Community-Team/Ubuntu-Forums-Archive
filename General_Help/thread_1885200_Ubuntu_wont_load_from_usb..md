### Post by jjex22 on 2011-11-22
Sounds like copying to the USB went wrong - give it another go - if from windows, make sure you use Universal USB installer, if linux, unetbootin - unless you already have an ubuntu, in which case you can use startup disk creator.

If you get the same problem - make sure you format the drive between 'burning' the iso to usb, and also try re-downloading the cd.
