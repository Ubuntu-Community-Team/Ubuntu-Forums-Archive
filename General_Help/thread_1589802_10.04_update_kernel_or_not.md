### Post by NightwishFan on 2010-10-07
Yes there is! :D Add the kernel ppa.
[https://launchpad.net/~kernel-ppa/+archive/ppa](https://launchpad.net/~kernel-ppa/+archive/ppa)

Using this command:
```
sudo apt-add-repository ppa:kernel-ppa/ppa && sudo apt-get update
```

Then simply run:
```
sudo apt-get install linux-generic-lts-backport-maverick linux-headers-generic-lts-backport-maverick
```

Reboot and you will be using a backported 2.6.35 kernel.
