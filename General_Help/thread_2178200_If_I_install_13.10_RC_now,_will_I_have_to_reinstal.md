### Post by grahammechanical on 2013-10-02
It is not a pre-release version. What you install will need to be updated just like any installation of Ubuntu. Whether you do it daily, weekly or even monthly is your decision. And what you will be using is and will be 13.10.

I am running Saucy Salamander now. I have been using it since development started. I do not expect to see much difference between what I am using now and what will be released in a few days time. Some bugs will be fixed that is all. No new features will be added. I update daily but you can wait until after release day and when you update you will have a fully updated 13.10 in the same way as you can now have a fully updated 13.04.

Any one who installs 13.10 after release day will be installing the code that was frozen at release day. They will need to update to bring their install up to date. We can even tick for the OS to be updated as part of the install.
