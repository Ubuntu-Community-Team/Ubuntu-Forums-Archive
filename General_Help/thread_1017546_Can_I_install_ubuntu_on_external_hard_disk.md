### Post by northern lights on 2008-12-21
As for part A, sure you can. Once correctly mounted your Linux doensn't care what device hosts your data. The filesystem might be of slight importance, but not the interface (USB, IDE, ATA, SATA, etc).

As for B, it depends whether your motherboard/BIOS supports booting from USB. If you can boot from a flashdrive/stick you can boot from an external HDD also.
