### Post by jpeddicord on 2010-08-06
Try a memtest from the GRUB menu for a while and see if there are any memory issues. Intel GMAs use system RAM, so if something goes wrong there then it will cause display issues.

I had total lockups in lucid, but that was using the NVIDIA driver so it's likely unrelated. I don't see any problems on Maverick, however. If you've got some free time and free disk space, why not install Maverick on a separate partition and see if you experience the same issues?

In the event that the GMA *has* failed, there aren't a lot of options other than replacing the motherboard, since they're soldered on. But I doubt this is the case; it's likely bad RAM or driver issues.
