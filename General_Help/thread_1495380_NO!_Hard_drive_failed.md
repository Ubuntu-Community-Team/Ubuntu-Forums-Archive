### Post by dtfinch on 2010-05-28
If you're lucky, putting the drive in a ziplock bag in the freezer for an hour or two might work if you have a specific thing that you want to copy off within a couple minutes of plugging it back in and mounting. You probably wouldn't want to boot off of it.

Edit: If the hard drive hasn't really failed, but just isn't booting and Ubuntu is refusing to mount because it's unclean, you might be able to run ntfsfix on it, then try mounting it again.
