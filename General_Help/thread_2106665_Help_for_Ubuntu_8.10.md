### Post by sudodus on 2013-01-19
Ubuntu 8.10 is hopelessly beyond end of life. The oldest desktop version 'alive' is 10.04 LTS, and it has only 3 months to go.

It is OK to run 8.10 in an isolated computer, but if you connect it to the internet, you will be wide open to attacks, because there are no security updates.

Instead we can discuss the alternatives. There are several versions and flavours of Ubuntu that might work well with your computer. There are also other linux distros, that are extremely light-weight, and that work well with old computers.

Please specify your hardware, at least

- computer brand (or motherboard)
- cpu
- ram (size)
- graphics card or chip

The first alternative among the current Ubuntu flavours would be Lubuntu 12.04, but that is a crude guess without knowing anything about your computer.
