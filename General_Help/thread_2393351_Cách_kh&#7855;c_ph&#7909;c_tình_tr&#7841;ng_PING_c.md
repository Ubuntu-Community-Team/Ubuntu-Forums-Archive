### Post by PaulW2U on 2018-06-02
This question was first asked back in 2011 when we saw the first release of Unity and it has been repeatedly asked ever since. It was a design decision that annoyed many users but they were always intended to be on the left. I seem to remember that it was possible to move the window buttons to the right on earlier versions of Unity but it certainly isn't in Ubuntu 16.04.

Other desktops such as Xfce, LXDE and GNOME Shell do of course have window buttons on the right by default. The various [flavours]("https://wiki.ubuntu.com/UbuntuFlavors") of Ubuntu and of course later Ubuntu releases use those other desktop environments.

You do have some choices but not on which side your Unity window buttons are placed. :)
