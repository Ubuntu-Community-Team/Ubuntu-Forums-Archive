I have been using Ubuntu for a couple of years now and I recently decided to try out Lenny, due to its new GUI.
One thing puzzles me in Lenny, when opening my home folder, every time I open another folder within the home folder, I get a new window, instead of moving on within the home folder,as per Ubuntu.I have read the howto for Lenny by raiden, but this does not seem to cover my issue.
Any ideas please as how to overcome this?

### Post by Alterax on 2009-04-21
Well, I'd love to help you out on this, but before I can make a go of it, I need to know which desktop you're using.  Gnome, KDE, Xfce, some other one that I can't think of?

If it's gnome you're dealing with, then your file manager is likely to be Nautilus--here are the instructions you're looking for if this is the case:

EDIT > PREFERENCES > BEHAVIOR > check the box marked "Always Open in Browser Windows"

close out the file manager, open a new one, and voila!
