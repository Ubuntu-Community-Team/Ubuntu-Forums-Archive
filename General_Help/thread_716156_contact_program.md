### Post by fragos on 2008-03-07
You don't have to use Evolution to run Contacts.  One hates duplicating data but at least Contacts is simple and fast.  It even has versions than run on handhelds like the Nokia N800.  I'm a Gmail user.  In that contacts I only keep name and email address.  Contacts is my main contact list with more indepth information.  Not perfect but workable.  In this family there's also a "Dates" application which is simple and also uses an Evolution data base.  Data can be entered in Evolution or these packages and is availble to all.

What you want doesn't exist yet.  Lightening is in development to run as a Thunderbird extension.  Calendar is done with Contacts planed. [http://www.mozilla.org/projects/calendar/lightning/index.html](http://www.mozilla.org/projects/calendar/lightning/index.html)
