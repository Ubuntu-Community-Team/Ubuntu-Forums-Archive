### Post by Robor on 2006-01-25
[QUOTE=Svip]Is it possible, with any Linux mail client to open the format that Microsoft Outlook Express saves in (I am saying Express just in case the two versions saved in different formats)?[/QUOTE]

Microsoft Outlook (the Email client that comes with Microsoft Office) stores it's Email in a single file called outlook.pst by default.

Microsoft Outlook Express (the free version included with Windows) stores it's Email in .dbx files.  Just search your system and you'll find them.

Also, not sure you need it but Outlook Express can export it's Email to a single .pst file as long as you have Microsoft Office installed.  It's a nice way to backup your current Outlook Express store to a single file.  :)
