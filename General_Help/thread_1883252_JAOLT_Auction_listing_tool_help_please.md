### Post by Luxx on 2011-11-18
OK... I found a help page somewhere in German that I got a bit of a translation on and played around with things until I figured out there are 3 things going on with the user account. 

1. You need to set one up for the program, which saves to a file on your computer automatically, but you can change the filepath if you want to. 
2. This user account will be connected to your eBay dev account that you have to sign up for online where you can get your keys and click on a link to get the token that will make the program work with eBay.
3. This will bring you to a page where you will be asked to link your dev account with your eBay account before you can get a token to make the program work. 

Well, now I know. That was a lot to go through online just to get what was supposed to be a simple app. It all seems to work, except I thought there were templates, or a way to create templates. Clicking on Select goes to the /home/user directory, where there is nothing.

I was hoping for templates. :(
