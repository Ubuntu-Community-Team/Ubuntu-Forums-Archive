### Post by rsambuca on 2008-11-13
Normally you can shrink an ntfs partition, add a new ext3 partition, and install a linux distro.  HOWEVER, based on your description, I would recommend strongly against it.  XP needs some free space on the drive in order to run properly, so you will probably run into problems.  Also,modifying a drive with only 10% free space is pretty risky, so it may not work either.
