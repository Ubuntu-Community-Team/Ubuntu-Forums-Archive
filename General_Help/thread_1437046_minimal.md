### Post by TitanusEramius on 2010-03-23
I guess you have two ways to go:

Install Ubuntu Server, it's installed without a GUI so you are free to choose from the lightweight editions like Xfce (not so light) or maybe LXDE (more light, but less functionality).

The other way would be to made a "real" light installation using this image:
[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)
And then only install those packages you need, a suited GUI and so on.

My Server with LXDE is currently using 1950mb on the disk and is running just about the same stuff that you needs, and it is using ~250mb ram and ~5-8% cpu-power when torrenting.

Good luck :)
