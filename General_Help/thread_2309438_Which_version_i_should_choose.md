### Post by ajgreeny on 2016-01-10
You should definitely use the AMD64 version with that machine.
Don't worry about the AMD name of the 64 bit ISO file you download; it is for all 64 bit machines, whether Intel or AMD.

Support for 32 bit systems is just beginning to run out according to some people, though I've never seen anything to confirm that, and my 32 bit netbook with an Intel Atom CPU still runs fine and is fully supported with the 32 bit versions of either Xubuntu or Lubuntu.

Your machine should run either Ubuntu with the unity DE or Xubuntu with XFCE; it's your choice so use the DE you prefer.  I personally use Xubuntu even though I have a powerful machine that does easily run Ubuntu; I prefer XFCE to unity.

I have no idea what wifi chipset is used so you may possibly have to figure that out after trying the live system, but it is not a brand new hardware generally so will probably find that it connects without a problem.  If you do have difficulties come back and ask and we will do our best to help you.
