### Post by C.S.Cameron on 2011-02-10
MS has put in a lot of effort keeping 7 from being able to be installed on flash drive, limit of only one machine per purchase.

If you just want to install 7 from USB, format the drive as NTFS and set the boot flag with gparted, then copy the contents of the install DVD or extract the ISO to it.
