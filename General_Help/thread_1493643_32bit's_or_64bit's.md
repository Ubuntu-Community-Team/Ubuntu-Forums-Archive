### Post by 3rdalbum on 2010-05-26
> **azmo35 said:**
> Hi, to all of you, this is a small question that is on my head for some time, switch from 32bit's to 64bit's but i think sofware and drivers have a good suport in 32bit's system but not in 64bit's, is that true?

It's true on Windows, but not true on Linux.*

The only real downside to 64-bit on Linux is that there's a bug in Flash Player that sometimes causes clicks inside a Flash movie to not actually register. I'm hoping this will get fixed some time this DECADE please Adobe!

* Linux is open-source; almost anything can be compiled for 64-bit (and is). Linux developers write programs with the assumption that they will be compiled and running on up to eleven different CPU architectures, whereas Windows developers wrote their programs with the assumption that you're using a 32-bit i386 CPU, and a lot of them can't be bothered changing their code and releasing a 64-bit binary.
