### Post by darolu on 2010-04-09
Chromium is the open source project, Chrome is the browser from Google.

Google Chrome draws its code from Chromium.

Chromium was born when Google made a big part of Chrome open source though.

[http://en.wikipedia.org/wiki/Chromium_(web_browser](http://en.wikipedia.org/wiki/Chromium_(web_browser))

[http://en.wikipedia.org/wiki/Google_Chrome](http://en.wikipedia.org/wiki/Google_Chrome)

I used to use Chromium, but I found some bugs that made it no longer usable for my needs, so I switched to Google Chrome which is more stable (even when it says "beta"). With Chromium you get the newest features, but are not always stable.

### Post by fragos on 2010-04-10
Chromium is a pure GPL product. Chrome allows Google to integrate non-GPL code like codecs. I've even heard they may be bundling Flash in Chrome. GPL would never allow them to do that. Personally I like the speed lack of application wide crashes that come with the Chromium/Chrome architecture. I run Chrome unstable as my default browser. Installing Ubuntu Tweak will give you easy access to repositories for Chromium, Chrome stable/beta and Chrome unstable. When developing I like to use Epiphany because of integration with Gedit. I only turn to Firefox for final testing of my websites and if I wish to see if the Gecko rendering is different than Webkit. Chrome now has some great extensions.
