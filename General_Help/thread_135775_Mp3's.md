### Post by Brunellus on 2006-02-24
the mp3 codec is not a given;  it is patent-encumbered and its distribution and use is legally questionable in the United States.  

Consult the RestrictedFormats page on the wiki, or search these forums for Automatix, which is an install script which should help out on most initial-setup things like this.

The free software alternative is ogg vorbis, which is technically superior anyway.  I have the mp3 codec for backwards compatibility/interoperability reasons;  anything I encode for my own use is ogg.

The poster above is correct, and Automatix is a very easy way to get a lot of the basic stuff you need.  For mp3, you can look for the LAME codec.  It's in the repos.  As far as having "all the features already installed," there's over 17,000 packages that are in the repos, so it'd be pretty difficult to have all that suff on one DVD.  To burn CD's I use Serpentine, which is included in Ubuntu.  There's others that are better that other folks could point you to.
