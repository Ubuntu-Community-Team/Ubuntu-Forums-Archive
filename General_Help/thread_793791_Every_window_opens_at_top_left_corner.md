### Post by chrisccoulson on 2008-05-14
Are you using Desktop Effects? if so, try installing compizconfig-settings-manager from the repositories, then go to System -> Preferences -> Advanced Desktop Effects Settings, make sure the 'Place' plugin is enabled, and try changing the settings in it (sorry, I can't remember exactly what settings there are as I'm at work)
