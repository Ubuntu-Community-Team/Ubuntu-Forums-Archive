I assume you want to mount /dev/sda1, /dev/sda5, & /dev/sda7 (since /dev/sda6 is already mounted to /media/Studies).

Install ntfs-config and use it to config those ntfs partitions.

```
sudo apt-get update
sudo apt-get install ntfs-config
gksudo ntfs-config
```

### Post by taurus on 2009-02-21
Edit /etc/fstab

```
gksudo gedit /etc/fstab
```
and add these two lines to the end of it.

```

/dev/sda6 /media/Studies  ntfs-3g  defaults,locale=en_IN  0  0
/dev/sda7 /media/Entertainment  ntfs-3g  defaults,locale=en_IN  0  0 

```
Save it and close down gedit editing windows.  Now, make sure you have those two mount points in /media or they won't be able to mount.

```
sudo mkdir /media/Studies /media/Entertainment
sudo mount -a 
df -h
```
