### Post by sgosnell on 2010-10-14
Gpsbabel should load the coordinates, in either direction.  It's installed by default, but only the command line version.  You'll need to get the GUI if you want to use that.  I'm not aware of many geocaching programs for Linux, since the market for making money is Windows.  Google might find something that works for you, though.  I abandoned geocaching a few years ago, so I'm not up on the most recent developments.  I just lost interest over time.
