### Post by oldfred on 2013-06-27
If you disable UEFI, you then are using BIOS/CSM/Legacy mode. Grub does not correctly install in BIOS mode unless you also have a bios_grub partition. It has to be unformatted and only needs to be 1MB anywhere on drive with bios_grub flag.
If booting with UEFI then you need the efi partition which is FAT32, should be first and has the boot flag.

Did you turn off secure boot & fast boot?
