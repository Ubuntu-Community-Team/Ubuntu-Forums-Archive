### Post by fragos on 2008-04-20
Cards using the bttv driver are automatically detected and run well after a TV viewing package like tvtime is installed. I have no experience with your tuner card. 

In a terminal window run "lsmod |grep bttv". If it's there, all you need is the viewer. Tvtime is my favorite.
