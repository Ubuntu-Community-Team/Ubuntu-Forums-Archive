### Post by ankspo71 on 2012-04-19
Application launchers (application shortcuts) on Lubuntu's panel (taskbar) are inside an "application launch bar" panel applet. So you will need to right click on the application launcher that you would like to remove and select "application launch bar settings". Now a configuration window will open that will let you add and remove application launchers.
