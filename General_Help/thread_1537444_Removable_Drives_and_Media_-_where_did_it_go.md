### Post by jondecker76 on 2010-07-23
I'm running a fresh 64-bit install of Ubuntu 10.04.  I no longer have a System->Preferences->Removable Drives and Media option in my menus, and need to re-configure it (I had set it to "Do Nothing" and "Always do this action" sometime in the past, but now I want to change it)
I have right-clicked my menu, went to edit, and made sure it wasn't un-checked)

Has the "Removable Drives and Media" option been moved somewhere else in 10.04?

---

### Post by jondecker76 on 2010-07-25
Ok, found my solution after tons and tons of searching

This functionality has now moved (there is no longer a Removable Drives and Media section under preferences)

It is now located in Nautilus's (the file browsers) preferences under the Media tab - in case anyone else has the same problem.
