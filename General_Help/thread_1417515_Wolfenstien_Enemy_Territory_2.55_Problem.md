### Post by 1337 on 2010-02-27
I think you have ran game as a root/administrator (probably with sudo) and therefor you messed your access rights to the game config folder (/home/user/.etwolf). Solution login as root and delete the game config folder. Then run game again using normal user account (without sudo). For the record latest version of Enemy Territory is 2.60b, on 2.55 you will hardly find populated servers. Good luck!
