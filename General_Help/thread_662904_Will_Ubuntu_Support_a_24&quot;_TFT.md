### Post by rune0077 on 2008-01-09
Yeah, standard plug n' play in Gutsy can run 1920x1200. No need to wait, you can just change the resolution.

If you have/plan to get a monitor with some fancy features that requires a driver, then you don't want to run as a standard plug n' play, in which case you may be better off waiting.
