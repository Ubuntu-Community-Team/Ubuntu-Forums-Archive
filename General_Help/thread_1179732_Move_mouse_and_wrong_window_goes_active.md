### Post by jammet on 2009-06-06
If I have a few different programs running all in the same work space and I click on one on the window list panel; then move my mouse upward in to the window another one from behind goes active.  It does this several times to the same window and then if I stop and give it a minute it will work fine again.  Could this be a hardware issue or has anybody else experienced this as some software configuration/bug issue?

---

### Post by jenkinbr on 2009-06-06
Open System -> preferences -> Windows

Uncheck 'Select windows when the mouse move over them'
