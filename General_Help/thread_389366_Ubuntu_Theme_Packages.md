Now an then you will see some artists make a "full" theme (gtk2, metacity, icons, gdm, xmms, gkrellm, etc) and post them all to the same place, but gnome themes don't work that way. You have to install theme components seperately - as bloodniece said.

Some folks are good with xmms themes but don't know how to create GTK2 themes, the same with other components.

I try to post as many components as I can when creating a theme and then post new components as I learn how to make them. See my [Bluecurve how-to]("http://ubuntuforums.org/showthread.php?t=342060").

It would be nice if gnome had "styles" functionality where you can open one app and choose all possible theme components (wallpaper, GTK2, metacity, xmms, gkrellm, etc) and make a style out of that and then changing a style would be a single click on the style. KDE has such functionality, but gnome doesn't yet.
