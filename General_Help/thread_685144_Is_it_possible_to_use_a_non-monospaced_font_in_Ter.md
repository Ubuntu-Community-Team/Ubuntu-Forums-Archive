### Post by LodeRunner on 2008-02-01
I can change fonts just fine, but all fonts are monospaced (including traditionally non-monospaced fonts such as, say, Arial.) "Used system fixed width font" is unchecked.

Is there any way to use a non-monospaced font?  This really sucks for reading large amounts of (non-code) text.

---

### Post by jpeddicord on 2008-02-01
Terminals must have monospaced font; a lot of terminal apps depend on it to display properly. If you tried to use Aptitude with a proportional font, none of the menus or any of the interface would line up. It's just the way many terminals work.

As an alternative, you could shrink the font size for the code displays if there is a lot going on.
