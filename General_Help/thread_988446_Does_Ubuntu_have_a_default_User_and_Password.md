### Post by philinux on 2008-11-20
Seen this before and the solution was that the livecd was a bad burn. Pop it in boot it and choose check cd for defects. If it shows errors you need to burn another no faster than 4x speed and do the check disk again.

### Post by niyonk on 2008-11-20
Hi, 
There was a phase in the installation where it actually asked you your Full Name,

It then used the name you typed in first before the space. So if you typed jphpe dell, then your username is jphpe.

As for the password did you leave it empty? 

check psychocats' [screenshot]("http://www.psychocats.net/ubuntu/images/installinghardyplus14.png")

I think you will have to reinstall if that doesn't work out :(
