### Post by BeRA on 2010-06-11
If you want the latest stable version of Chromium (and to get fresh updates through Update Manager) then add PPA for your ubuntu version from: 

[https://launchpad.net/~chromium-daily/+archive/ppa]("https://launchpad.net/~chromium-daily/+archive/ppa")
*(procedure is explained on this page)*

and then use Synaptic's search to find chromium-browser (then click on checkbox on the left side of chromium-browser and select the only available option... and than apply button above... oh, and Synaptic will ask to approve dependencies-accept that)...

or you can use code provided in the post above from KeyserSoze93 :)
