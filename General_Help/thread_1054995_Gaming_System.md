### Post by sedawk on 2009-01-30
Yes there is a difference.

All network traffic for/of the virtual machine passes a software
layer of the VMware server running on windows. This might
slow down things or cause instable network under heavy load.
Also 3D graphics in a virtual machine are slow because fast
3D hardware cannot be used (until proper drivers exist).
