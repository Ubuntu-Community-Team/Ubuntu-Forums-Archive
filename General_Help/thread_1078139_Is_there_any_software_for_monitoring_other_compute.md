### Post by Peter09 on 2009-02-23
There is plenty of software which will allow this, it depends exactly what you want to do.

VNC is installed by default and will allow monitoring of Linux->Linux desktops.

ssh is capable of remotely running tasks on a remote machine, including a desktop GUI.

Finally RDP is supported in Linux.
