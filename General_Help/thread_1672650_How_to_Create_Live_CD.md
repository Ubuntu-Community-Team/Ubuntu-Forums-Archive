### Post by mastablasta on 2011-01-21
you dowload the ISO file and burn it to a CD. but you need to burn as image not just burn data to CD.
 
Another option would be to make a bootable USB key. you can use one of the freely availble programms to do that. for example: [http://www.linuxliveusb.com/](http://www.linuxliveusb.com/)
when new verison arrives you just overwrite the USBdisk. plus i think computers read faster from USB disk than CD.
 
if you want to create your own distribution then you can use remastersys [http://remastersys.sourceforge.net/tips.html](http://remastersys.sourceforge.net/tips.html)

### Post by BruisedQuasar07 on 2011-01-21
You want to install Ubuntu to a USB Flash stick drive. To get around the
live CD issue you mention (losing your setings, etc each time), you need 
a persistent install. This can be easily done by using a utility called
Universal Installer. Just search the forum for "persistent install" 
(do not include the " marks in your search) to get more detail. The
result is referred to as "live USB". You should do a "live USB" search
as well

--Bruised
