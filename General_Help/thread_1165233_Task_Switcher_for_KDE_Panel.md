In Gnome I can place an applet in the taskbar that allows me to switch tasks from a list. Is there a way to do this in KDE?

### Post by nortexoid on 2010-07-28
This is really old, but I stumbled upon this by searching, so someone else may.

The answer is yes (in KDE 4.4.5 and probably much later)--add the Window List widget/plasmoid to your panel. The widget package is installed by default in the base KDE install.
