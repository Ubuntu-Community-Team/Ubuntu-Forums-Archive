### Post by kenny99 on 2009-09-15
Hi, I recently installed Jaunty on my work PC and found it ran much more slowly than Intrepid did on my own laptop. My work PC has 4GB of RAM (32-bit ubuntu sees 3.2 of it) and my laptop has 3GB - both machines have IntelCore2Duo processors. It looked like the issue was specific to Jaunty and the Intel graphics card issue, but I've now installed Intrepid on my work PC (dual boot with Jaunty on another partition) and the performance is just as poor, if not worse. There seems to be little difference between the spec on my laptop and my work PC, apart from the laptop has the Intel Corporation Mobile GM965/GL960 Integrated Graphics Controller, while my work PC has the Intel Corporation 82G33/G31 Express Integrated Graphics Controller (rev 02) - could it just be the graphics card that is the issue? If so, I don't mind buying an nVidia card if i know it's going to improve performance significantly.

Many thanks for any help!

### Post by uylug on 2009-09-15
Your need to run a 64 bit ubuntu. My computer specs are similar to the ones you mentioned and I've had quite an important performance boost when i switched to 64bit.

Also, Ubuntu not seeing 4GB is because it is a 32 bit distro. You can still use 4GB by enabling Physical Address Extension.

I think it could be done by running:
```
sudo apt-get install linux-kernel-headers
```
