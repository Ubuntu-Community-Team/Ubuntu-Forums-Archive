### Post by Warlon on 2010-05-03
There has been a bug report of this for ages.

[https://bugs.launchpad.net/ubuntu/+source/libwnck/+bug/150690](https://bugs.launchpad.net/ubuntu/+source/libwnck/+bug/150690)

I used to use this feature a lot, but now it has been broken for so long that I've given up hope. If I remember correctly, it works if you disable the desktop effects completely, so it has something to do with compiz.

One "show-off" way of moving windows between workspaces is using shortcut super+e (super meaning the windows-key). This shows all your workspaces and allows you to drag and drop windows.
