### Post by ajgreeny on 2010-05-03
[http://packages.ubuntu.com/](http://packages.ubuntu.com/) will have everything that is avaiolable for all the versions of ubuntu you could possibly want.  You can download these packages on a windows machine if needed and then use ```
sudo dpkg -i packagename.deb
``` to install them.  Just make sure you get all the dependencies for everything or you will still end up having problems.
