I'm afraid your question is pretty generic in nature, but rather verbose in terms of the response(s).

 As the previous poster suggested, you need to install LAMP.

 That isn't too bad, but understanding how a web server works can be a daunting task.

 Then you can create your actual website, and that again in itself is a huge undertaking and something I don't think we can teach you in these forums, but we can help you install the programs to make it work (LAMP!)

 From there you need to decide if you want to host your web site on your home computer and have the outside world see it. To do that, you'll likely want to use the services of something like [www.dyndns.org](www.dyndns.org) that allows you to create your own "domain name" and point it to your home computer... assuming you're on an always connected connection that is able to pass through port 80.

 I'm sure the forums are full of tutorials on setting up LAMP and using DynDns.org. Have a look around, do some reading, and like magic you'll get there! 

 It's fun and exciting, and you'll learn a lot too.
