### Post by grahammechanical on 2011-08-03
If you are using 11.04 then you first need to install something called Compiz Config Settings Manager. Open up the Ubuntu software Centre and search for ccsm and select the first item in the list for installation.

Then open System Settings from the menu that appears by clicking on the power icon. And there you will find the Utility to set things like the desktop cube.

Be aware that these effects work well when in Ubuntu Classic mode but when using the Unity desktop (Ubuntu in the login menu) some of these effects will mess up the desktop. You will have to disable the effect and reboot to get the Unity desktop back.
