### Post by ajgreeny on 2015-02-25
I have been using Hide-Caption-Titlebar-Plus in FF for some time now and found it very helpful in increasing useful screen area for content instead of the toolbars and titlebar.  Today with the update of FF to 36 it stopped working completely.

I can report, however, that there is a development version 2.9.3rc that you can get from the very bottom of the add-ons home page which does work very well, so far at least.
[https://addons.mozilla.org/firefox/downloads/file/299128/hide_caption_titlebar_plus-2.9.3rc-fx-linux.xpi?src=ss](https://addons.mozilla.org/firefox/downloads/file/299128/hide_caption_titlebar_plus-2.9.3rc-fx-linux.xpi?src=ss)
