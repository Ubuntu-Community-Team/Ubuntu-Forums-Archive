### Post by little_penguin on 2009-02-28
dvd-slideshow is run on the command line, which is why you there isn't a menu entry. For a graphical slideshow software you can try ManDVD, which can be used to create photo slideshows or video DVDs. You can get a deb binary from debnet. Select your specific ubuntu version and then enter ManDVD in the search field at the top right.

[http://www.getdeb.net/distro_select.php](http://www.getdeb.net/distro_select.php)
