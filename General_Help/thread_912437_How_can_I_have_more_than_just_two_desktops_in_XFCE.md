### Post by Genius314 on 2008-09-06
This might be a little different, since I'm using Gnome, not XFCE, but it should basically be the same.

1. Install compizconfig-settings-manager, if you haven't already:
> sudo apt-get install compizconfig-settings-manager

2. Launch "ccsm."

3. Go to "General Options."

4. Click on the Desktop Size tab.

5. Change "Horizontal Virtual Size" to 4.
