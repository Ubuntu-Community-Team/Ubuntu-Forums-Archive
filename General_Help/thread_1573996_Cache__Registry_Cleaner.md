### Post by 3rdalbum on 2010-09-14
You should not be using Ubuntu 8.10. It is no longer supported. 

Caches on Ubuntu keep themselves fairly clean, but you could run sudo apt-get clean to remove the package cache if you are running low on disk space.

There is no registry on Linux.

Note that clearing caches does not make your computer faster. If anything, slower for a while.
