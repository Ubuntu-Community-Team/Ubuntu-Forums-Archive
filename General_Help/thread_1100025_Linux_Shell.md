### Post by mb_webguy on 2009-03-18
Writing a shell is a non-trivial task.  If you don't have strong programming skills, you might want to wait a bit.  Besides, there are already plenty of shells available.

If you're really intent on writing your own shell, it would probably be useful to look at the [source code of other shells]("http://www.ibiblio.org/pub/Linux/system/shells/!INDEX.short.html").  The msh shell is probably the smallest and simplest you'll find.
