### Post by LordSmight on 2008-12-29
Try downloading the file and extracting it to your desktop. Then go to: 
System->Preferences->Appearence. Then at the "Theme" menu drag the extracted cursor theme file from the desktop into the "Theme" box.

Now select the GTK theme you're using from the "Theme" menu in Appearence and click "Customize"->Pointer and the cursor theme should be shown. Select it then log out and back in to enable it.

### Post by Ratscallion on 2008-12-29
I've got a couple of methods for you to try, other than these, I'm afraid I cannot help you. 
First of all, make sure you have Visual Effects on at least normal to show you have a compositing manager installed. 
Now, only put the EXTRACTED versions of the cursors in the ~/.icons/ folder.
Make sure the cursors have no visible file extension, it should be FILENAME only. Check the properties of the cursors, if they are Windows Cursors, they are unusable in Ubuntu.
Finally, do what I do, open up the Appearance Window and then copy the .tar.gz archive to your desktop. Drag and drop the archive into the main window from there.
