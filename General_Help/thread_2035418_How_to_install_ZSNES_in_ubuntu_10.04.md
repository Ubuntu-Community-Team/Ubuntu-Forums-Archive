### Post by Ep1c0wl on 2012-07-30
This is a tutuorial on installing ZSNES in Ubuntu 10.04. The first thing you need to do is open Terminal (*Applications->Accessories->Terminal*) and type this code:

```
sudo apt-get install zsnes
```

You can find ZSNES in *Applications->Games->ZSNES Emulator*. All you have to do is download ROMS and run them as you usually would.

Additional information at:   [http://ubuntudan.blogspot.com/2010/04/how-to-install-emulators-on-ubuntu.html](http://ubuntudan.blogspot.com/2010/04/how-to-install-emulators-on-ubuntu.html)
