### Post by amoore on 2007-01-17
**This is not meant to start a flame war for Skype vs other SIP clients.  **

I know the advantages and disadvantages of OSS vs. Closed software and network protocols.

Never the less, I use Skype for my VIOP needs. Recently I have grown discontent with Skype for Linux because, compared to the windows version Skype there are many features lacking, such as: SMS messaging (text messaging to cellphones), Video chat, Firefox plug-ins, ect.

I have written several request to have these features implemented in the Linux version of Skype.  Below is an email that I received from the Skype support staff commenting on SMS, video chat, and Firefox plug ins. 

> 

Thank you for contacting Skype Support.

I've read your request several times and gave you appropriate answer. New versions for Skype for Linux will be in a public release and a lot of new features will be added. SMS features and video calls will be available in July. As for plug-in for Firefox, I've already forwarded your desire to our development team and they took it into consideration.

We hope you will continue using Skype. Please keep your patience and stay with us.

Best regards,

Olga Z.
Skype Support
[email]support@skype.net[/email]

### Post by gdi2k on 2007-01-26
As a start, you can make skype links work in firefox using [**this**]("http://share.skype.com/sites/linux/2006/08/making_skype_links_work.html") guide. Where they use /usr/local/bin/skype-action-handler as a key, you'll want to use: ```
/usr/bin/skype-action-handler
```

That will at least allow you to call other skype users directly by clicking on their web link such as the ones used [**here**]("http://forum.skype.com/index.php?showtopic=63549").
