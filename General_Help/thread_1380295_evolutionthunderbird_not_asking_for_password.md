### Post by ajgreeny on 2010-01-13
Normally when you try to send or receive emails without having set a password, a dialog box should appear asking you to enter one.  Does this not happen, or are you just wondering why this is not in the account setup dialog?

If you have entered all the info needed to set up the email accounts, ie, the username, the server names, etc etc, but not the password, which is never asked for at this point, try receiving and/or sending and see if an error message appears and asks you to enter a password at that point.

The only other thing I can think of is that there is already an account set up but with an incorrect password, though again I think that would immediately throw up an error message telling you so, and asking for the correct password.
