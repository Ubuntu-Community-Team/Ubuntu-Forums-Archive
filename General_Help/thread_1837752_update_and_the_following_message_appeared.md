Lets just get it working.

Copy your sources.list file to a backup. Enter this command ```
sudo cp /etc/apt/sources.list /etc/apt/sources.list.backup
```

Then use this command to edit the file. ```
gksudo /etc/apt/sources.list
```

Delete everything in the file and paste the following into the file. ```
## Ubuntu Main Repos
deb http://us.archive.ubuntu.com/ubuntu/ natty main restricted universe multiverse 

## Ubuntu Update Repos
deb http://us.archive.ubuntu.com/ubuntu/ natty-security main restricted universe multiverse 
deb http://us.archive.ubuntu.com/ubuntu/ natty-updates main restricted universe multiverse 
```

Save the file.

Try to update.
