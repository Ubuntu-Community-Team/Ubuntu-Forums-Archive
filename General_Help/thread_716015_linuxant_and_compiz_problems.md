### Post by boeing777 on 2008-03-05
To Solve Your Compiz Problem:

1.  Goto System --> Administration --> Software Sources

Under "Ubuntu Software" tab, check all the checkboxes except "Source Code"

Under "Updates" tab, check all the checkboxes.

2. Goto System --> Preferences --> Appearance 

Under "Visual Effect" tab, enable compiz and it should install all the files automatically for you.
