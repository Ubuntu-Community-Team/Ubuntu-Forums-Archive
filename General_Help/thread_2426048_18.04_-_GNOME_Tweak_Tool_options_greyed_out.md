### Post by vanadium on 2019-09-04
To get rid of the "Your window is ready notification", you need to install a Gnome Extension, appropriately called "Noannoyance". There is an officially supported Ubuntu package that installs the extension system wide: "sudo apt get install gnome-shell-extension-no-annoyance".
