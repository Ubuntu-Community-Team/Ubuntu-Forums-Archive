### Post by 25tom on 2013-06-03
Gespeaker using espeak (the default) sounds stylised, so it's not great for pronunciation. However, you can install mbrola voices which will work with Gespeaker and they sound a bit more natural. I installed the mbrola voices I wanted from Synaptic, so I'm not sure how the install process for them works if you want to use the Software Center, but once they're installed, you can use them from within Gespeaker. You may need to tweak the Gespeaker preferences a bit to get the voices going.

Hope this helps :)
