### Post by pastalavista on 2009-05-01
You look to be alright for a Gutsy (7.10) install but I'm wondering about devs/ sda3 & sda4. You didn't say how you upgraded and I'm guessing you didn't use the update manager. All I can figure is the disk you used wasn't a Jaunty disk but Gutsy. 

All I can suggest now is backup your home folder and all your personal data and download a 9.04 iso for yourself and reinstall. Then copy your old home folder into the new /home (make sure you use the same user name) It is recommended, when you install, to create a separate partition and label it /home. Make it large enough to hold all your saved files. Then next time you need to reinstall, you can just reformat the / drive and leave the /home in place.
