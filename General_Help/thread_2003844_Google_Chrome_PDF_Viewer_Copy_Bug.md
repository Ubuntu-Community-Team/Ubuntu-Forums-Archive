### Post by Guilden_NL on 2012-06-14
Chrome version: 12.0.742.100
Ubuntu Linux 12.04 also confirmed with Lubuntu 12.04

Using native Chrome PDF viewer, when copying a range of characters (bank account number in our case) the viewer drops the last character when posting to the clipboard and then pasting elsewhere.  

Have confirmed that this is not a bug in the Windows version.

Cost me loads of fees at a university!

Reported to Google.

Best workaround: install Adobe Reader and configure as the default PDF viewer through about: plugins, then disable the native PDF viewer.
