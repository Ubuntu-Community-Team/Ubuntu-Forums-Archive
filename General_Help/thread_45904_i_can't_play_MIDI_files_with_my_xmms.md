### Post by Limulus on 2005-08-14
If it isn't already, install "freepats" from Synaptic.
Then go to Terminal and run *sudo cp /etc/timidity/timidity.cfg /etc/timidity.cfg*

That should clear up your problem :)

For a complete walkthrough on getting MIDIs to play in XMMS in Hoary, see [this thread](http://ubuntuforums.org/showthread.php?p=300724).
