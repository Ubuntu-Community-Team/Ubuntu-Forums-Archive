### Post by deadflowr on 2013-08-22
Well, try any other distro, like Linux Mint, or Opensuse, or Fedora, or Crunchbang.

I would recommend running whatever you decide to go with from a liveCD first.
If the problem still arises, then the issue is most likely a hardware failure, than an issue with kernels or drivers.

If it doesn't, then try installing which ever distro you choose and see if the problems occur again.
If they don't, then it truly is an Ubuntu problem.
If they do, then it's almost a certain hardware problem.
