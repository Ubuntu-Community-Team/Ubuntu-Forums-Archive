### Post by mcduck on 2014-03-27
Sorry, but the dialog you are looking for is not part of Nautilus so switching your file manager won't change anything.

That's the GTK file chooser dialog, a ready-made dialog from the GTK toolkit, which *does* support preview of files if the developers of each program only wish to use the feature. Firefox developers have decided to not use the preview so there's nothing you can do about it.
