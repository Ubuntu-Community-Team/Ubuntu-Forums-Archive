### Post by bobonthenet on 2006-04-13
I'm used to typing my numbers with the number pad and in order to do that I have to push the num lock key.  Its a minor nuisance that I have to do it every time I restart my computer.  How can I change it so that the num lock is on by default when ubuntu starts?

---

### Post by JoshHendo on 2006-04-13
Look in synaptic for a program called numlockx.
That will allow you to have it either on or off on boot.

### Post by z_diver on 2006-04-14
Automatix is great if you are starting a new machine and want to use it as your default desktop.  It has the abilty to install a months worth of apps in 20 minutes or less.  But if all you need to do is turn on numlock, look here:
[http://easylinux.info/wiki/Ubuntu#How_to_turn_on_Num_Lock_on_GNOME_startup](http://easylinux.info/wiki/Ubuntu#How_to_turn_on_Num_Lock_on_GNOME_startup)

You should also be careful with numlockx.  If you have a laptop it can create situations where you cannot log in.

BIOS & OS numlock settings are independent - each sets its own. So turning it on in the BIOS (like I do) doesn't keep it on in Ubuntu.
