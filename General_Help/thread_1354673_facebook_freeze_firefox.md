### Post by lovinglinux on 2009-12-14
I don't use Facebook, but whenever Firefox exhibits high CPU usage is probably flash or javascript. I would bet on javascript on this case.

You could try to use [AdBlock Plus](https://addons.mozilla.org/en-US/firefox/addon/1865), since there are several javascript adds that can cause this issue. You could also use [Flashblock,](https://addons.mozilla.org/en-US/firefox/addon/433) so you would be required to manually allow any flash content.

There is also NoScript, which is great, but probably would render the site unusable.
