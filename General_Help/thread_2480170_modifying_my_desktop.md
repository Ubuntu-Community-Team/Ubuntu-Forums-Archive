### Post by TheFu on 2022-10-21
This 100% depends on the window manager and desktop environment used.  Most DEs have a User's Guide document. Have you checked the one for your DE?

Gnome v3 and later have moved away from folders for organization for the GUI.  This is part of the reason that other DEs are still popular.  For example, I don't use any DE and don't have any folders for organization on the desktop, but the top 10 things I do are all available using keyboard accel keys to launch specific programs.  My WM doesn't have much of a menu system, but that's my choice to keep clutter down.  Different people like to work differently.  I suspect you might be happier with Mate or XFCE DEs.
[https://guide.ubuntu-mate.org/](https://guide.ubuntu-mate.org/)
[https://docs.xubuntu.org/latest/user/C/xubuntu-documentation-A4.pdf](https://docs.xubuntu.org/latest/user/C/xubuntu-documentation-A4.pdf)
[https://help.gnome.org/users/gnome-help/stable/](https://help.gnome.org/users/gnome-help/stable/)

When it comes to GUI stuff, seeing how someone else organizes in a video is very useful.  Might be worth watching a few DE youtube videos.

Anyway, to get GUI help, please share:
a) which ubuntu version is being used
b) which DE is being used
c) the exact issue

Today, there are 2 or 3 supported releases for the DEs that Canonical supports. There are vast differences in each, even the same DE from different releases can be a different version. For example, 22.04 Gnome used v4.2 of Gnome while 20.04 Gnome uses v3 of Gnome. That's a bunch of differences.

There might be some Gnome extensions that bring back the folder organization. IDK. [https://extensions.gnome.org/extension/1217/appfolders-manager/](https://extensions.gnome.org/extension/1217/appfolders-manager/)  seems like an option.
