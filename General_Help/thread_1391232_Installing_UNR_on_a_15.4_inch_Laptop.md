The only difference is the user interface. UNR's "launcher" is designed for small screens. It has big icons like an iPhone. If that's what you think you want, go for it! ### Post by snowpine on 2010-01-26
You are confusing three separate Ubuntu projects:

Ubuntu Nebook Remix
Ubuntu Moblin Remix 
Ubuntu LPIA (Low Power Intel Architecture) Port

As I said above, Ubuntu Netbook Remix is nothing more than Ubuntu with a "launcher" interface (no Atom required). You can install it over a "regular" Ubuntu install by installing the ubuntu-netbook-remix package. I would not recommend the other two (Moblin Remix or LPIA Port) as they are not well supported.
