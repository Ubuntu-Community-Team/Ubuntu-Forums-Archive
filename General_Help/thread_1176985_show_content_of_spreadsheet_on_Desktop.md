### Post by Yumi on 2009-06-03
To answer my own question:
I opted for an entry in System - Preferences - Startup Applications  Command = ooffice -calc -o Desktop/weight.ods

Reduced the calc to small size, removed toolbars and saved it. Now on startup it launches the table on the desktop with the last used cells visible
