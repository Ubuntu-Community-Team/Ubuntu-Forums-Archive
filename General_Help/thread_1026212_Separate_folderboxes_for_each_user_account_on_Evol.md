> When you setup additional e-mail accounts/users in the Evolution e-mail program, does it give you separate boxes/folders for each e-mail user account ?

Thanks.

You don't set up separate **user** accounts in Evolution, you set up separate Linux users and they have their own Evolution setups.

When you add multiple e-mail accounts in Evolution, all POP accounts go to the same place, all IMAP accounts have their own Inbox tree.
