### Post by jasper28 on 2009-04-02
I currently run Ubuntu 8.10 on a decent computer.  I use the "normal" level of desktop effects, but I find some of the enhancements annoying. Is there any way to turn on and off specific aspects (like application start effects, minimize animation, etc...) of the desktop effects?

---

### Post by Simian Man on 2009-04-02
Yes.  Installing the package 'simple-ccsm' will give you a configuration editor for compiz (the desktop affects program) in which you can fine tune your settings.
