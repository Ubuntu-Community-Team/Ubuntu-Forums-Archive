### Post by mcduck on 2009-05-21
> **brandoncolorado said:**
> If you turn compiz off, you can hold alt and drag the window up.  This is the only way I have found to access the lower buttons.

That should work just fine without turning Compiz off. :D

It's a basic feature available in pretty much every Linux window manager, and Comiz definitely does that by default. The only reason why it wouldn't work with Compiz is if you have yourself made some changes to Compiz configuration and those changes conflict with this feature.
