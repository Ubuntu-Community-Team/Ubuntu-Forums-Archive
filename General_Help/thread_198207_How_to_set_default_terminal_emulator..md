Hi everyone, 

I have a brief question on how to set the default terminal emulator in ubuntu 6.06 (gnome). 

Let's say I have a file called: somefile (executable bash script)

```

# !/bin/bash

echo "Hello, Linux!"



```


I want that to open in Konsole, and not the gnome terminal. When I double click it, it has an option to run in terminal. When I click run in terminal, it opens up gnome terminal. How do I set it to open konsole instead of gnome terminal?

---

### Post by bluevoodoo1 on 2006-06-16
Hmm perhaps the following will help. 

System > preferences > preferred applications

Then under the system tab try 'custom' then as the command 'konsole'
