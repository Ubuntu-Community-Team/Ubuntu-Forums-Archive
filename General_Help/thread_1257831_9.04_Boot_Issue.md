### Post by lobhob on 2009-09-04
Yesterday I upgraded my laptop to 9.04. My system now freezes midway through the boot process when I use the default kernel 2.6.28-15. If I select an older kernel like 2.6.27-14, I don't have any problems. Are there any drawbacks to making the older kernel the default?

---

### Post by bumanie on 2009-09-04
No not at all. Most seasoned linux users keep an older kernel that they know works around for the exact problem you describe. As unfortunate as it is, the occasional kernel will cause idiosyncratic problems with a few computers and work with the majority of computers. Booting with an older kernel is better than having no ability to boot. Make the older kernel 'default' - it's fine.
