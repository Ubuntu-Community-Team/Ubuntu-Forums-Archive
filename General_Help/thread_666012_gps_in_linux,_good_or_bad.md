### Post by erpo on 2008-01-13
If you're using a GPS device with Linux, you're using a program called gpsd. You can find a hardware compatibility list for gpsd on its web site, which you can find by searching on google.

I wouldn't go with bluetooth anything with Linux. It's just one more thing that can go wrong.

If you want a cheap GPS device that works well with Linux, check out [http://www.geeks.com/details.asp?InvtId=UT-41](http://www.geeks.com/details.asp?InvtId=UT-41) which is the one I use. It works very well with gpsd.
