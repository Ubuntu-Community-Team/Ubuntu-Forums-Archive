### Post by jdb2 on 2010-11-30

I've set up my environment like so : 

( from 'env' )

```
CFLAGS=-m32 --sysroot=/home/jdb2/sysroots
CXXFLAGS=-m32 --sysroot=/home/jdb2/sysroots
```
