### Post by polardude1983 on 2010-08-31
Or you can also hit create launcher on desktop and do something like this in the command line

for me to open [www.yahoo.com](www.yahoo.com) on chrome i type this in command

/opt/google/chrome/google-chrome %U [www.yahoo.com](www.yahoo.com)

for firefox its

firefox [www.yahoo.com](www.yahoo.com)

Just name it whatever you want
