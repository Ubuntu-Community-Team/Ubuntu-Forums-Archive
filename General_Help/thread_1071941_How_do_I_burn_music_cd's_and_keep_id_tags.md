### Post by caljohnsmith on 2009-02-17
Unfortunately the original audio CD format standard that is still in use today does not suport ID2/ID3 tags; the music on the CD is basically raw 16-bit PCM (Pulse Code Modulation) data. ID tags only come with music formats like mp3. So if your music player can instead use mp3 or some other modern music format, you're in luck, but otherwise a standard audio CD will not retain any ID tags on the source music when you burn the CD.
