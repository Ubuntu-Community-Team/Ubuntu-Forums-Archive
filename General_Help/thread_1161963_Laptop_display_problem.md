### Post by Legace on 2009-05-17
NOTE! I assume the resolution you want is 1280x800. IF IT IS NOT THE RESOLUTION YOU WANT, THEN DON'T FOLLOW THE INSTRUCTIONS. Tell me what's the resolution you want if it's not 1280x800.

--

Make monitor look like:
```
Section "Monitor"
	Identifier	"Configured Monitor"
	ModeLine	"1280x800_60.00"  83.46  1280 1344 1480 1680  800 801 804 828  -HSync +Vsync
        Option "PreferredMode" "1280x800"
EndSection
```

And screen look like:
```

Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
	SubSection "Display"
		Modes      "1280x800"
	EndSubSection
EndSection
```

See if that helps.

Note if you fail to boot after doing this, boot into recovery mode, select root shell, and type in **sudo nano /etc/X11/xorg.conf** and remove the modeline and preferred mode thing by adding a #..

So it should look like:
```
Section "Monitor"
	Identifier	"Configured Monitor"
       #ModeLine	"1280x800_60.00"  83.46  1280 1344 1480 1680  800 801 804 828  -HSync +Vsync
       #Option "PreferredMode" "1280x800"
EndSection
```
