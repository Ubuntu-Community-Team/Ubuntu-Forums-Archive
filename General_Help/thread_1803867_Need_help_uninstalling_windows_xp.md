### Post by Alwimo on 2011-07-14
The easiest way is to install Ubuntu again, and choose to delete everything when you are choosing how to install it.
 
You should back up all the files you want on an external drive beforehand, of course.

### Post by Mark Phelps on 2011-07-14
The low space warning comes up most often when folks install using Wubi and allow it to default to the smallest space allocation.

If you installed using Wubi, then Ubuntu resides INSIDE Windows in a large file.  In that case, removing Windows removes Ubuntu.
