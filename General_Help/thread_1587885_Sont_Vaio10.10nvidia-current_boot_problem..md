### Post by rkues on 2010-10-21
I don't know if this is a bug or what :confused: I have a Sony VPCF113FX and went through this very same problem using Ubuntu 10.04. In that case, I had to use an user compiled version of the nvidia drivers, which can be really annoying being ubuntu the "unnoficial" test base of every new version of the linux kernel. Believe me, don't go that way with ubuntu.

This problem is specially related to the new LED lids from Sony.

Anyway, today it runs with the nvidia-current driver (10.10), but you still have to add two new options to your xorg.conf file, device section:

```
Option "ConnectedMonitor" "DFP-0"
Option "CustomEDID" "DFP-0: /proc/acpi/video/NGFX/LCD/EDID"
```


You can read more about it on: [http://code.google.com/p/vaio-f11-linux/wiki/NVIDIASetup](http://code.google.com/p/vaio-f11-linux/wiki/NVIDIASetup)

Kind regards,
Richard.
