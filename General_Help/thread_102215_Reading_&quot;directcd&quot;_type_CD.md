### Post by Kevinator on 2005-12-11
I believe DirectCD is a packet-writing program. It allows your CD drive to write (and read) UDF file systems on CDs. I've never tried it, but I assume Ubuntu can handle this. The man page for mount shows UDF as a supported file system.

I can't tell you exactly how to do it, but it looks like it should be possible. Maybe this helps clarify the question for others who know how to use UDF, but don't know what DirectCD is.
