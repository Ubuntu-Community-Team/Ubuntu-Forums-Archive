### Post by synapsys on 2009-07-03
I would have to recommend 9.04 (Jaunty Jackalope) because it has the most hardware support 'right out of the box' Definitely get the 32 bit version (default download), as the 64 bit version issues are a little more complicated to fix. As far as I know, it's pretty damned stable. 

Also be mindful of where you install it, as to not remove any windows partitions you may have.

If you've never done this before, this guide can help you.

### Post by QIII on 2009-07-03
Installation is about the same for all of them.  Point and shoot.

As for which version is the more stable, expect to get a lot of opinions.

8.04 (Hardy) is mature.  The LTS is mature and has support through April 2010.

8.10 (Intrepid) is the distribution prior to the current one.  It has a lot of the kinks worked out.

9.04 (Jaunty) is the current distribution, and some complain about problems.  I've had not except for some issues with graphics card support, which I was able to fix.

Don't use 9.10 (Karmic) since it is still in testing.
