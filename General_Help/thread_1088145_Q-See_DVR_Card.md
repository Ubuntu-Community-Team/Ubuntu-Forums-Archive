### Post by scarface_willie on 2011-01-11
Hi, I have the Q-See       QSU2DVR04 USB DVR with 4 (wired) VideoSecu 480 line res. cams with IR and mic.  I can say firsthand that this setup is better than most convenience store's security cams in image clarity.  Remote viewing works great, even on my HTC HD2 Win phone.

I too want to install it on Ubuntu 10.10.  I tried loading it using WineHQ Windows emulator and it gave back a driver fail error notice.  This is not to say that it can't be done, but I'm no programmer and I don't know where to start on how to configure Wine.  What I am able to do, though not ideal is run it on another Windows machine and VNC to it from Ubuntu.  This actually gives a clearer image than the remote viewer that comes with it plus you get 100% of playback function and admin control, unlike the remote apps they provide.  

I am looking into the possibility of trying another DVR app to use it on Ubuntu, though I do not have high hopes.  Go to your Ubuntu Software Center and search for DVR.  There is an app that I am experimenting with. 

Slightly off topic; Another option for you is to look into Foscam PTZ IP cams. Amazon has them for around $100 and they work flawlessly with full pan and tilt and 2 way sound and motion activated and manual DVR functions. Just make sure get get outdoor rated cams if putting outside as the standard versions will fail instantly if wet, no brainer.
