### Post by conchobar on 2004-11-14
I'm going to introduce you to Linux's trump card: Synaptic. Roar, Synaptic, roar. Roar. Oook, with that said, open Synaptic by this succession of menus: Computer > System Configuration > Synaptic Package Manager. Load the list. (you might have to add some things to the repository list; search these boards for that into) Now, click Search and enter "window maker" making sure to click the box below it to select Description and Name. The search should nab quite a few related packages, but the one package you want is wmaker. Magic, eh?
