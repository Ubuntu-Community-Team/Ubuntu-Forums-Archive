### Post by Dave_L on 2011-05-14
Here's one way:

Press Alt+F2 and then enter the command "xkill".

Position the resulting X cursor over the application's window and left-click it.

### Post by Bonster on 2011-05-14
I usually add my own hotkey, in system > preferences > keyboard shortcuts

Add a custom hotkey command 'xkill' then bind it to any key combo you want
