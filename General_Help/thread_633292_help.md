### Post by jespdj on 2007-12-06
Try selecting a mirror near you instead of using Canonical's servers. That will probably make downloading software much faster.

Select System / Administration / Software Sources. On the "Ubuntu Software" tab, click on the "Download from" select box. Select "Other...". In the list, select a server close to you.
