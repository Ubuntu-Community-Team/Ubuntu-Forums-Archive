### Post by INH on 2010-08-08
What you described is perfectly normal.  The menu you were presented with is called GRUB and is the bootloader that Ubuntu uses.  From it you can choose whether you want to boot into Windows or Ubuntu.  Just use the arrow keys to select the option you want and hit Enter.  If you want to boot into Ubuntu, select the topmost Ubuntu option - the others are not important.
