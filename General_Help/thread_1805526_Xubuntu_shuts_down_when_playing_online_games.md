### Post by jramshu on 2011-07-16
Are they flash based games? If so, install flashaid in firefox and verify you have the correct flash for your setup. Flash uses a lot of resources and can 'bottleneck' the cpu sometimes and cause system freezes. Check the logs for errors.

### Post by mendhak on 2011-07-16
I'm wondering if this is a hardware problem rather than a software problem - you say it shuts down, does it shutdown/reboot immediately?  That can usually indicate a power or heat problem with the CPU.  Do you have an application like PSensor which can tell you the CPU temperatures?  Try monitoring it while playing.
