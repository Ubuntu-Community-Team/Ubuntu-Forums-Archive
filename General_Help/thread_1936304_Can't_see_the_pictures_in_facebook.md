### Post by rhss6-2011 on 2012-04-14
I had the same issue at different times with different Linux versions so I'll give all the solutions that worked for me...

1. (Half arsed solution) Click at the end of the URL and press *ENTER*. Reloads your FB page and images can be viewed

2. ```
sudo apt-get remove -purge chromium-browser
``` and then install the actual google-chrome either by using a deb file + gdebi or finding out how to add the repositories and installing it *properly*

3. (If you use advanced popup blocker app) click on it and allow for Facebook...

Those were the only things I had to do (with different versions of linux) so hopefully that works for you...
