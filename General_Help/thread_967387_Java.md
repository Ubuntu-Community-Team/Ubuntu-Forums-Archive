### Post by ad_267 on 2008-11-02
You don't need to be smart to use Ubuntu, you just have to realise things are done a bit differently to Windows.

To install java go to System - Administration - Synaptic Package Manager and then search for sun-java6-plugin package and install it. 

If you install the ubuntu-restricted-extras package it will install java as well as mp3 and other codecs.

It would be a good idea to read this about installing software in Ubuntu: [http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)
