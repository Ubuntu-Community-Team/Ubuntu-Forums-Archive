### Post by 3Miro on 2011-01-18
7zip is a much better program for handling rar files. You should also be able to right-click and select "extract".

For viewing .pdf, I would avoid Adobe as much as possible. Ubuntu comes by default with evince (listed as "Document Viewer"). Evince is a much lighter and faster viewer (only a few .pdf files have features that require Adobe).
