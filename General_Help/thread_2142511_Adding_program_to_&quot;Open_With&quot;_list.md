### Post by kostkon on 2013-05-05
Create a dekstop file for it, i.e. a launcher, using, for example [*MenuLibre*]("http://www.iloveubuntu.net/menulibre-1304-released-refinements") (a very good app) and then right click on a .sunvox file, select *Properties*, click on the *Open With* (or something similarly titled) tab, then click *Add* and select Sunvox, click OK, and finally make sure that it is set as the default app for this type of file in the list.

Hope that helps.
