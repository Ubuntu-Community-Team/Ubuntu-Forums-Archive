### Post by amaurynieto on 2007-05-29
Hello all, I installed a few things, specifically a few windows programs
in Wine which i've since uninstalled, as well as VMWARE player and workstation. These items were installed via automatix and are no longer isntalled, however they still show up in my applications list.

How can I edit the items which show up in the applications menu so I can erase these useless "goes nowhere does nothing when clicked" links?

Thanks very much for your help! A.

---

### Post by Medieval_Creations on 2007-05-29
Right click on the the menu bar and select Edit Menus.  You can then Uncheck the items not needed or just plain delete the item.
