### Post by Mark Phelps on 2010-09-24
> **warakawa said:**
> 

3. Since my laptop is a tablet, how much support does ubuntu/linux have. 
Very little -- and it varies a lot by individual make and model machine.  I have a tablet PC -- a  Fujitsi Lifebook -- and it worked great under Ubuntu 8.04.  But I need to use Tablet mode, which means, the bezel keys HAVE to work.  No Ubuntu version newer than 8.04 has supported the bezel keys. Period. Other Ubuntu shortcomings sent me back to MS Windows. Have Win7 on the machine and EVERYTHING works.

>  Does ubuntu have handwriting recognition? 
One of the major things that sent me back to MS Windows -- since the answer then (and I think it's still the same today)  was NO.

> 4. Some of my homework are done is windows journal, can they be transformed into xournal?
Used Xournal, but it's not a replacement for MS Windows Journal because it doesn't have handwriting recognition (at least it didn't when I last used it). In fact, to quote from the Wiki article:

> Unlike Windows Journal, Xournal lacks the capability to automatically perform OCR on handwritten text, thus precluding the existence of features such as searching a handwritten document for text.

Don't know if the .jnt files can be imported, but given MS's reluctance to provide info on their proprietary file formats, I would be very surprised if you CAN import them.

I use Windows Journal on the Tablet every day, and the handwriting recognition is the major reason for keeping it.  Win7 improved that tremendously!

> 5. How do I install lenovo drivers for my x200t?
Unlike with MS Windows, you do not just arbitrarily install drivers in Linux. Drivers are installed as part of the initial OS setup.  IF there are hardware drivers available, you will be offered them.

> 7. Will I need to learn command? 

No, but you will find as you come to use Ubuntu more and more, that some things are just easier using the command line.  But in nearly all cases, you should be able to use the machine without having to resort to that.

### Post by Mark Phelps on 2010-09-24
> **warakawa said:**
> will office 2010 work in ubuntu?

Don't know for certain, but if I had to guess, I would say NO.

It took CodeWeavers (the Wine folks) a LONG time to get Office 2007 working in Wine, and even then, I believe you need to use Winetricks to get it to work, and still today, some of the Office 2007 components (notably Access and Outlook) don't work.

If you're as tied to MS Office apps as you're implying by your questions, switching to Linux at this time is a bad idea.  Perhaps when Office 2010 is up and working in Wine would be a better time.

Also, I'd like to point out that you're getting positive responses from folks using laptops -- and as you and I both know, tablet PCs are a special version of laptops.  Just because a laptop works is no indication that the full features of a tablet PC will also work.

### Post by Favux on 2010-09-25
Hi warakawa,

The Lenovo X200t has a Wacom digitizer with an internal serial connection.  The default Lucid Wacom X driver xf86-input-wacom 0.10.5 doesn't work quite right with it.  It's an early version of the driver and a lot of fixes and improvements have been made.  So you'd want to update it to 0.10.8 or better.  Then it works well.  There are a couple of HOW TO's for that, about 5 to 10 minutes.  Maverick already has 0.10.8 as the default.

As far as I'm aware almost everything works.  A lot of developers have Lenovo tablets so they tend to have among the best support.  Either threads on the forum or HOW TO's at the thinkwiki pretty much tell you how to do anything that needs doing.

There are a couple of open source OCR projects now, so handwriting recognition isn't out of the question anymore.  Currently CellWriter, which has letter/number recognition and is also an onscreen keyboard, is available.
