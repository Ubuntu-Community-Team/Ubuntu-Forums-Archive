### Post by Sunforge on 2007-10-22
Books about any kind of software tend to go out of date but I can recommend:
 
"Beginning Ubuntu Linux (from novice to professional)" by K Thomas published by Apress.
 
It's a good solid starter guide.
 
Another nice book is O'Reilly's "Ubuntu Hacks"
 
BTW Ubuntu is a Debian based distribution and installs .deb binaries. You can install RPM's using alien.
