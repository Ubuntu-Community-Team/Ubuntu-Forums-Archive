### Post by oldfred on 2012-03-14
A liveCD is a bootable CD or USB install that you can use to make repairs. You should have one that is the same version as the current install.

But if installing with RAID you need the alternative installer which is not a liveCD to have the extra drivers for RAID. If making repairs with a liveCD you have to download the extra drivers every time you boot with the liveCD.
