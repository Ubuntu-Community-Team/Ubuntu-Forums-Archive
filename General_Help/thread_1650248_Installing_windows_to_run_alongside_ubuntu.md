### Post by theasprint on 2010-12-21
Hi,

If you are quite sure that Windows XP cannot use your HDD, then I suggest that you use the Ubuntu disc to format it.

I think you boot the LiveCD, use Try Ubuntu without any Changes.
And then open the disc utility, and try to format your drive to NTFS type if possible. 

If not you can use GParted to do the job. But report back first.
