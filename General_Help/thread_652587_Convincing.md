### Post by p_quarles on 2007-12-28
If she doesn't see any reason to switch, then don't press the issue. Ubuntu isn't for everyone, and switching operating systems is a big undertaking. 

As for Wine, OpenOffice.org, and all the other great things: those aren't good reasons to switch. Wine just runs Windows programs, OOo is available for Windows, and virtually everything else in Linux either works natively or has some kind of equivalent in Windows.

### Post by p_quarles on 2007-12-28
Oh, I see, you're trying to convince her to let *you *use it. That's different.

Obviously, she's the boss in this situation, so you'll have to respect her decision. You are correct, though: installing a dual-boot system will have absolutely no impact on the speed of either operating system. 

One thing you could do is get yourself an external hard drive, and install Ubuntu on that. You should be able to get a small one for a good price, particularly on Ebay. She may be more willing to let you install it if it won't effect the hard drive that's already on the system.

### Post by billgoldberg on 2007-12-29
Dual boot is ideal.

If windows is already on the pc, just install ubuntu an a spare partion and grub will be able to boot into windows and ubuntu.

This has NO effect on the speed of either operating systems. There are no negative effects having two operating systems on a pc, unless you are short on harddrive space.

If you do this, make sure you have a super grub disk live cd, so you can repair either grub or vista mbr if you choose to uninstall an os.
