### Post by UbuntuNerd on 2009-05-30
I know you can check for duplicates files with "fslint" in terminal type:
```
sudo apt-get install fslint
```
after you installed it you can find it under Applications-> System Tools-> FSlint. 
Add the file path that you want to search. On the left, click on the Duplicate tab and click Find at the bottom.
