### Post by lovinglinux on 2009-05-15
Some extensions can make it slow down considerably. For example, I have the TagSiffer extension, which is great and I can't live without it. But this extension slows down the addons administration page. I guess it is because it loads stuff from the sqlite file responsible for storing the tags. Sqlite is not a fast database, so while the extension is reading it, the addons page becomes unresponsive.

You could try to disable all extensions, then enable one by one to find out which one is slowing you down.
