Basically when I try to type something in these fields the first letter I type will appear in the field but then nothing after. If I hold down a key, eventually the letter I am holding down will appear, but that's a considerable amount of work just to type anything. This continues after the second letter, the third and so on. If I have never typed a specific word into the input field (so it therefore does not appear  in a that input fields drop down list of past words i have typed), there's no problem; I can type freely. 

Maybe not related, but I am using UIM in order to type Tibetan and wonder if there might be some sort of conflict with Firefox and uim. I could uninstall uim to check, but would prefer not to if I can avoid it. Has anyone else encountered problems inputting in Firefox?
