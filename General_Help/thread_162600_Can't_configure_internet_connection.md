### Post by m.musashi on 2006-04-19
Have you tried activating the connection?

system -> administration -> networking

Click on ethernet connection and then properties. Choose "enable this connection" and then set configuration to "DHCP" unless for some reason you have a static IP address and then fill in accordingly.

Click OK and then click "activate". You should be good to go unless there are other issues in which case please give as much detail as possible.

(Is there a CLI way to do this? Seems like a lot of steps.)
