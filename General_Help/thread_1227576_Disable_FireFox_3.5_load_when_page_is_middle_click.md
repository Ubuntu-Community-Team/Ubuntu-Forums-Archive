### Post by SoftwareExplorer on 2009-07-30
I successfully installed Firefox 3.5. I like it, but the one annoying thing is that it tries to load whatever middle click would paste if I accidentally middle click on a web page. This happens especially a lot on my laptop that supports two finger scrolling.

---

### Post by Vaphell on 2009-07-30
had that problem and it went away when i set middlemouse.contentLoadURL to false in about:config
