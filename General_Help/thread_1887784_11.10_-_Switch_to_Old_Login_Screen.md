### Post by SnowyCheetah on 2011-11-27
In my previous Ubuntu installation (10.04), I had the login screen such that you actually had to type the username and password, there was no list of the accounts on the machine. Can anyone tell me how to get that on 11.10?

---

### Post by gman16000 on 2011-11-27
Ubuntu 11.10 uses lightDM by default so I'll assume that you are using that.

Edit your lightdm.conf:
```
sudo vi /etc/lightdm/lightdm.conf
```

And then make sure you have the following lines in the [SeatDefaults] section:
```

greeter-hide-users=true
allow-guest=false

```
