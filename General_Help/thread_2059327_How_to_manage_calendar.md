### Post by daslinkard on 2012-09-17
Just from doing a general Google search I found how you can add a calendar indicator to display Google Calendar events on the Ubuntu /Unity panel in 12.04.

If interested the steps to do so are as follows:

```
sudo add-apt-repository ppa:atareao/atareao
``````
sudo apt-get update
``````
sudo apt-get install calendar-indicator
```
