### Post by kokkus on 2008-10-04

Nvm.
You can do this in many ways.
The program I use is Remastersys.
[http://scumbox.com/ubuntu/remastersys_2.0-5_all.deb](http://scumbox.com/ubuntu/remastersys_2.0-5_all.deb)
With this program you can create distro iso and an iso with your private stuff as well. (Both as live cd with installation)
So you can use this program to create Ubuntu installation CD's for your friends or for your own use.
When you've installed the program, you'll find the launcher under System > Administrator. Start the program. Now you can play around with the program and create a dvd or cd image.
