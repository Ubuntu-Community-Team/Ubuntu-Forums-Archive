### Post by Tomosaur on 2007-05-02
You need software capable of burning an iso. The one I used on Windows is [ISO Recorder](http://isorecorder.alexfeinman.com/v2.htm). Install that, then just right click the .iso and select the burn to cd option.

You don't extract the .iso file to burn it, you just burn the .iso as an image.
