I am using kate in GNOME (installed kde-base). It works great, but I have a problem I never had before.

Every file I open kate opens in new window, now a 'tab'. What could be a reason for this?

---

### Post by rCXer on 2009-11-07
[This answer](https://answers.launchpad.net/ubuntu/+question/83102) may help.  Basically you should right-click on any text file and select "Properties". Click on the "Open With" tab and press "Add".  At the bottom click "Use Custom Command" and type "kate -u" and press "Add".  If you want kate to be the default program for text documents select it in the "Open With" tab before closing.
