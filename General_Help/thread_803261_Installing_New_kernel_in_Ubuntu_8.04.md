### Post by Mighty_Joe on 2008-05-22
There are several alternative kernels in the Ubuntu repositories.  See [this comment]("https://bugs.edge.launchpad.net/ubuntu/+source/linux/+bug/204996/comments/192") for instructions on how to switch.  
The advantage to using the repository rather than compiling yourself is that it's easy to get your system back to the default.  
As Nepherte says, messing with the kernel is not for the faint of heart.  
And if your knowledge is limited, how did you arrive at the conclusion that the kernel is a problem?
