[http://sylpheed.sraoss.jp/en/news.html](http://sylpheed.sraoss.jp/en/news.html) has download links. Of course, these later versions may not have been tested with your particular OS version.
Generally, software updates that are **not** security-related are done at the time a new version of Ubuntu is released if a new software update is available in time. In other words, every six months.
