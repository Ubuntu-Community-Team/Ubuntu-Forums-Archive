### Post by Perfect Storm on 2017-10-22
I guess you have done
```
sudo apt install libdvd-pkg
```

Have you tried with VLC player?

also: [https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs](https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs)
> There's a chance that even after installing libdvdcss, some of your DVD's still won't play. One solution is to play the DVD with a media player that allows you to disable DVD menus. In VLC Media Player for example, when you choose to open a disc, there's an option "No DVD menus". You just have to enable that option before you press Play. Media players that lack support for DVD menus should work as well (e.g. mplayer). Also there's a chance that dvdcss keys generated for the movie are corrupted, the solution is to delete the folder related to the movie that you are trying to play inside the ~./dvdcss folder.

### Post by Tadaen_Sylvermane on 2017-10-22
installing libdvd-pkg is the first step. You need to either install other packages separately or run ```
sudo dpkg-reconfigure libdvd-pkg
``` to get it installed fully. libdvd-pkg is just an installer script and for whatever reason (no idea) it can't run on the initial installation of itself.

### Post by Perfect Storm on 2017-10-22
Try with:
```
sudo apt install -f libdvd-pkg
```

or:
```
sudo rm /var/lib/dpkg/info/libdvd-pkg
sudo apt-get purge libdvd-pkg
sudo apt install libdvd-pkg
sudo apt-get install -f

```
