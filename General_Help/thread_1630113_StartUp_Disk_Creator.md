### Post by C.S.Cameron on 2010-11-24
So what is wrong with Unetbootin? there are both Windows and Linux versions. It can be made persistent if you like.

Universal USB installer from PendriveLinux.com will install to USB from a Windows machine.

MultiBootUSB is a nice GRUB2 iso booter but I don't think it works with Arch:

[http://ubuntuforums.org/showthread.php?t=1518273](http://ubuntuforums.org/showthread.php?t=1518273)

The Arch website also has a few recommendations:

[https://wiki.archlinux.org/index.php/Putting_installation_media_on_a_USB_key](https://wiki.archlinux.org/index.php/Putting_installation_media_on_a_USB_key)
