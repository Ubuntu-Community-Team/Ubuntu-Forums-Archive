I have a dual moniter display, and in order to get rdesktop to take up one, but not both moniters, I run it from terminal with the command 
```
rdesktop -g 1024x768 -D 192.168.36.2
```

I am unsure about a command to also attach my local disk.  I can do it in the gui, just tick attach my local disk in the Local Resources tab.  My question is, how can I see what command is generated by the application?

### Post by iponeverything on 2010-05-21
```
ps auxwwww |grep rdesktop
```

will show options the process is running with.
