### Post by squakie on 2014-05-21
I thought you had to have the screen set to some turn off value other than never before lock makes any difference.  I *think* what happens is the screen blanks after "x" time, then the lock is applied at "x" + what the lock-after value is.  So when you have the screen set to never turn off, I don't think it would ever lock.  If you want to test this, set the turn screen off when inactive timer to it's lowest value.  Set the lock on and set the lock after to it's lowest value.  Then doing nothing for 5 minutes.  I believe you'll come back and find it locked and you have to provide the password.  Now set the turn screen off when inactive timer to never and I think you'll find you never get a lock.

I *think* this is how it works.
