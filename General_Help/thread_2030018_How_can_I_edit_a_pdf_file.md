### Post by thatguruguy on 2012-07-20
It depends on whether the pdf is a scan of a document, or an exported text document.

I don't remember if 10.04 uses OpenOffice or LibreOffice, but do a search for "pdfimport" [without the quotation marks, of course] in the software manager of your choice (I'd recommend either synaptic or the Ubuntu Software Center) to find and install the pdf importer for the office suite you use. Then, you will be able to import and edit the pdf like any other document.

NOTE: this only works if the document was exported from a word processing program, not if it is a scanned document.
