### Post by renzokuken on 2008-03-07
it shouldnt matter whether you use 32/64bit between machines. the file sharing side of it is handled by SAMBA and since samba is available for both 32 and 64 bit ubuntu it doesnt matter.

there might be a small increase in speed with 64bit ubuntu but i doubt you'll notice it since i guess the main limiting factor is your network speed, not your CPU

a desktop version would be easier to manage than a server version if your not used to it. you could always have the desktop for configuration, then stop the X server while not in use to save on resources
