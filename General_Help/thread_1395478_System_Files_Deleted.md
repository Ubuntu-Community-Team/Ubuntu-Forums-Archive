### Post by quixote on 2010-02-01
Just guessing by the thunking sound, I'd guess that your HD failed.  If it was a matter of system files being deleted, a liveCD should have still been able to mount the drive.  You don't need system files on a drive to mount it.  The fact that it can't mount suggests a hardware fault. Unfortunately, if there were files on it you absolutely have to have, I think your only option is one of those (hideously expensive) professional data recovery services.  Using photorec or testdisk can only recover data off a corrupted but functional drive.

Maybe somebody else here has some better ideas?
