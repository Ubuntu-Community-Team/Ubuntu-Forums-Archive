### Post by wallyhall on 2008-06-19
Sorry you've waited so long without answer.

Mondoarchive will burn to anything supported by the underlying software (Mondoarchive doesn't actually burn, it uses some other tools for that).  As I recall, all the tools/software needed to burn DVD disks are included in Ubuntu by default, and so it works "out of the box".  However I may be wrong, in which case let me know!

When you've ran Mondoarchive, you'll be presented with a choice of mediums (places) to back up to.  Just select DVD/R and choose the default size (unless you know otherwise).  Mondoarchive will call some other tools to create an ISO image and burn it!  Simple as.
