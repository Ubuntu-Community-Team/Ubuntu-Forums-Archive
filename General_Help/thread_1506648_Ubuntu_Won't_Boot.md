### Post by wilee-nilee on 2010-06-10
Installing when the cd doesn't even work is not a good idea.

Check the cd by booting it hit the shift and then check cd, also do a md5 sum.
[MD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

Since this a fresh install it can be removed easily, make sure you have a bootable cd try it out for a while if it works consider dual booting, if you have a install disc for the stock os so you can recover the mbr to it if needed.

It may be able to be fixed as is, but it may be easier to just get a working cd and using it to install.
