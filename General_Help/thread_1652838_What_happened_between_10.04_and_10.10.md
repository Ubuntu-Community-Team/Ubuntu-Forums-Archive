10.04 was specifically a release based on stable versions of apps and not the developmental versions that **all** other Ubuntu releases get.

It is a very simple choice, if you want stability choose the current LTS (10.04), if you want the latest versions of apps (and all the problems that come with them) then use the latest release.

No one is forced to use the latest version, everyone chooses to use whatever version is available.
