TeXlive is not an editor, it's the LaTeX system (like MikTeX on Windows). So you'll feed it tex files, and it will answer pdf or dvi files.

If you want an editor, there are a bunch you can try.

Kile is very good, but a bit slow on a Gnome environment.

Texmaker is a very complete editor, but it uses qt, and that means it has different theming than other gnome apps.

LaTeXila is a very complete gnome editor. I use LaTeXila.

I personally use LaTeXila.
