### Post by VCoolio on 2009-05-01
[Here you go]("http://ubuntuforums.org/showthread.php?t=809695&highlight=compiz+visual+effects+cube").
in short:
set your visual effects to extra (system > preferences > appearance)
install compizconfig-settings-manager
open system > preferences > compiz settings manager
under General Options > Desktop Size > Horizontal Virtual Size: set to 4
enable cube and rotate cube plugins

### Post by fballem on 2009-05-01
> **BeeDeeGee said:**
> Hey everyone, I just downloaded and installed unbuntu. I am a complete noob to Linux and I saw someone using 4 desktops on a tutorial on youtube, but mine only has two. How do I see the 4 desktops like a cube?

1/ Don't worry about Compiz - it allows a lot of different effects, which many consider eye-candy, but is not required to setup the Desktops, or Workspaces, in ubuntu. The fact that you have two workspaces indicates that your system is setup with the needed programs.

2/ To change the number of workspaces, right-click on one of the workspace panels (right hand side of bottom panel, next to the trash bin).

3/ You should see a panel similar to the attached screenshot. Change the number of and rows so that the number is the number of workspaces. For example, you could set columns to 2 and rows to 2, which will give you 4 workspaces. You could set columns to 4 and rows to 1, which will also give you 4. If you set columns to 4 and rows to 2, then you will get 8 workspaces.

Hope this helps,
