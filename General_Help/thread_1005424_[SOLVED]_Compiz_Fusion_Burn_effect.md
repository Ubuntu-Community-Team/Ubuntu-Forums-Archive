### Post by Kaneda187 on 2008-12-08
It sure is possible go to 
System>Preferences>CompizConfig Settings Manager(if you dont have this get it from the repository)
now go to 
Effects>Animations Add-On,
now find the burn header and hit the arrow to get the drop down menu now you should see a color box hit it and you can change to colour to what ever you want! 

I hope this helps if so hit the gold medel on the bottom right of this post! thanks
