It is a fine line, nitpicking ...

"gnome" is a large project and includes things such as GDM and nautilus ...

so ...

libs == "gnome 3"
The GUI == gnome-shell

So Oneiric will be using the gnome 3 libs + Unity rather then gnome 3 + gnome shell.
