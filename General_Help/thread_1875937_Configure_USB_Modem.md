### Post by gandaran on 2011-11-05
I once had a modem that could either connect with usb or ethernet but could not use the two at the same time, this was the modem problem not ubuntu, try the ubuntu PC with USB cable, remove the ethernet cable from the modem just to see it USB works, if it does then maybe the solution is to manually set a static IP for each USB and ethernet connections.
or buy spare ethernet cables, they aren't expensive.
