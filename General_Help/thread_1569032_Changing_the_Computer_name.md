### Post by coffeecat on 2010-09-06
mastablasta makes a good point. If you want to be sure that your files are completely unrecoverable even with special recovery software, you might want to think about securely overwriting the whole drive with the dban CD or with 'shred' from a live Ubuntu CD. Then reformat and reinstall. A bonus is that your friend gets a clean install of the OS.

If you want to know more about shred, just post back.
