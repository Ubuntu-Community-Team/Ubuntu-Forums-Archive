### Post by gaussian on 2008-08-05
Sorry to be the bearer of bad news:
I have zero experience with this particular Lexmark model, but it seems that Lexmark printers often have zero linux support. Just google "Lexmark x3550 CUPS" or "Lexmark x3550 Linux". 

One half-cooked idea: OS X (MAC) uses CUPS (the "Linux Printing System"). Actually CUPS is now developed/owned by Apple or something like that. This means that the PPD file is somewhere buried in the Mac package. This might not be easy to extract, since Mac installation stuff often comes as "disk-images". But maybe with some extra Googleing you could find out how to extract those. Worst case: find a Mac, install the drivers, find the .PPD file and copy it over. No guarantees obviously.
