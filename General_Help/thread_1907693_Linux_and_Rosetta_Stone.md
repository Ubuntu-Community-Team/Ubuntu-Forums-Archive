### Post by Mark Phelps on 2012-01-12
If Windows XP is working fine, you might want to rethink moving to Linux...

Linux does NOT offer "amazing" hardware support, just the opposite.  The best hardware support historically has been with Windows XP -- because every hardware vendor wrote drivers for it and it has been around a LONG time.  Win7 also has very good hardware support for new products and can often use XP drivers in compatibility mode.  

If Linux hardware support was truly "amazing", I wouldn't advise folks to try the LiveCD mode on new PCs -- because the drivers would just work.

As to Rosetta Stone, any time someone depends on a specific Windows app for using their PC, it is a BAD idea to migrate them to another OS that does not support that app.

We have folks coming here all the time, demanding to know how to run IE 9 or Office 2010 -- and are very disappointed when we tell them they can't.

Your realtime performance using Ubuntu on that PC is likely to be no better than using Windows XP -- and you already suspect that a MAJOR reason for using the PC is not going to work under Ubuntu.

So, leave the PC with XP.  Your mother "doesn't know much about computers", so I'm sure she doesn't care which OS it is using, what she does care about is being able to continue to use Rosetta Stone.  So, if you can't get that fully working, you're only hindering her by switching over to Linux.
