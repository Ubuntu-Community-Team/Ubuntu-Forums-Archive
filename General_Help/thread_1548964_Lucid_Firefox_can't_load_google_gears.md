### Post by lovinglinux on 2010-08-09
User Agent switcher won't help you with that. If you want to force the installation of an extension, you can [Add-on Compatibility Reporter]("https://addons.mozilla.org/en-US/firefox/addon/15003/"), but I do not recommend doing so, because this method is used only when an extension is incompatible with the Firefox version and the method not always work. In your case, the extension has a different OS target.

For a solution see [http://ubuntuforums.org/showthread.php?t=1502104](http://ubuntuforums.org/showthread.php?t=1502104)
