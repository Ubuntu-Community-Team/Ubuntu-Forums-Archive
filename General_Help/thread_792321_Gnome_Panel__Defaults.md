### Post by Xgen on 2008-05-13
Perhaps I misunderstood, I thought that you wanted the Start Menu menus back to default. If you just want to edit the panel menu, then right click on it and either create a new one or add/remove the applets. If you want to default to the original panels (not menus) then rename 'panel' in '.gconf--->apps' which is hidden in your home folder and restart (not tty) the computer.
