### Post by howefield on 2013-03-06
You can find the file in /usr/share/example-content/Ubuntu_Free_Culture_Showcase/Josh Woodward - Swansong.ogg

To delete it, open a terminal and type the following command.

```
sudo rm /usr/share/example-content/Ubuntu_Free_Culture_Showcase/Josh\ Woodward\ -\ Swansong.ogg
```
