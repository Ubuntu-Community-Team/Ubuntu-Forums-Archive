### Post by twipley on 2012-10-10
One way to break the system, including existing partitions, is to fiddle using the XP tools. Therefore, the best way to use XP is, under most set of circumstances, through a virtual machine (VirtualBox is good).

Probably the partition are touched (or if not, then the MBR).

The best thing to do for now is not to touch anything and wait for the help reinforcements as to the methods of backuping the current data.
