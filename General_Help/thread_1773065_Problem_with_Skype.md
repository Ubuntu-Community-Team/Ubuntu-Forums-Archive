### Post by linuxinstalledfromhdd on 2011-06-01
It is possible that the network you are running has restricted the use of Skype on their networks.  For example, Skype will not work on mobile Dubia networks.  It was banned.  Check with whatever ISP you are using while traveling first, and start a trouble ticket with them about using Skype with their service. If they provide you with a workaround, come back and let us know. :))
