### Post by BertN45 on 2012-11-09
The USB is like the CD/DVD it is nice to try, but if you want to keep the settings, you have to install the OS. If you have free space or a not used partition on the disk you can safely install the system to the disk. See the next explanation: 

[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot) 

You could install it on your USB stick/drive, but then you need a second USB stick or an installation DVD.
