### Post by C.S.Cameron on 2008-10-22
Unibootin and usb-creator will let you make Live USB flash drives from the downloaded iso file
You can install from these just like from the Live CD.
Unibootin is simple to use and works in Windows or Ubuntu.
No need to have a CD drive, and the USB doesn't scratch.
