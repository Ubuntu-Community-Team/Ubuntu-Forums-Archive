### Post by PaulW2U on 2018-06-21
> **wheelsmith said:**
> I've set my default font as 'URW BOOKMAN L' and that is not what I see.
The default font that you can set is used when the web page doesn't specifically set the font either in the HTML or CSS. I have a number of browsers installed and changing the default font has no effect on most of the websites that I visit. However, one site that I use regularly doesn't specifically set the font so I do see whichever font that I choose to set as the default. 

You will need to use a browser extension to override any font that the web page has set. Such extensions are definitely available for Chrome, Chromium and Firefox but Palemoon's options seem to be quite limited.
