### Post by ibuclaw on 2009-03-30
To restore the configuration of an application to its defaults when you initially installed it?

If you just want to reset your user settings. Then open up **Places->Home Folder** and press **Ctrl+H**.

Then locate the **.folder_name** folder that corresponds with the application you want the settings to reset for, and delete it.

ie:
For the program **pidgin**, you delete the **.purple** folder.
