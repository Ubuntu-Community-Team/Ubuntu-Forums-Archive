### Post by mechdave on 2010-06-13
First of all can you make sure your system is fully updated via command line. Hit ctrl+alt+f1 and login usin your login name and password. Then type in ```
 sudo apt-get update
```

then after that completes type ```
 sudo apt-get upgrade 
```

See if that resolves the issue, if not we will have to look further into it :)
