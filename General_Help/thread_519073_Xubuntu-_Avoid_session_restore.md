### Post by strabes on 2007-08-06
Alt+F2, type in "xfce4-panel" to start the xfce4 panels. Then go to the log out menu where you can select log out, restart, shut down, etc, and uncheck the box that says something like "Save Session for future use." I'm not sure what it says exactly. Then log out and try logging in again.
