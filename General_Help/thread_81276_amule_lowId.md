### Post by Pablo_Escobar on 2005-10-24
My advice.
1. Install firestarter (front-end for iptables - default (K)Ubuntus firewall)
2. In the last tab when You start it there will be a chance to add allowed ports. You'll have to look in emule which ports are these (probably the server ****.***.***.****(ip):**********(port)**. Then You allow that port to be open in Firestarter, apply policy and You'll have highid. Works like a charm for me :)
