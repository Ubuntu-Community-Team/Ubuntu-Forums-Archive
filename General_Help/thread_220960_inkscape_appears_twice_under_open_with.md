### Post by viciouslime on 2006-07-22
If I right click on an SVG file then inkscape is in the openwith menu twice. I have reinstalled ubuntu (for other reasons) and it is still there, so I guess it is something in my home directory causing it, anyone know what I need to edit?

### Post by Arisna on 2006-07-22
See if you can right click on an SVG file, go to Properties and then the "Open With" tab, highlight one of the entries for Inkscape, and delete it.  One will not be removable because it is set by the system, but the other should be.
