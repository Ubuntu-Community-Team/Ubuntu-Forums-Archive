### Post by taurus on 2008-12-28
Currently, your large harddrive, 1TB, is mounted to /media/disk.

```
/dev/sda1             932G  123G  810G  14% /media/disk
```
If you want to add an entry to /etc/fstab so it would mount automatically each time you boot Ubuntu, you can install ntfs-config and use it to configure your ntfs drive/partition.

```
sudo umount /dev/sda1
sudo apt-get update
sudo apt-get install ntfs-config
gksudo ntfs-config
```

### Post by taurus on 2008-12-28
You can use /media/sda1 as your mount point and the program will add an entry in /etc/fstab for you so you don't have to worry about it.  But if you want to do it by hand, then edit /etc/fstab

```
gksudo gedit /etc/fstab
```
and add this line to the end of it.

```
/dev/sda1   /media/sda1   ntfs-3g   defaults,locale=en_US.UTF-8   0    0
```
Save it.  Then, create a new mount point for it.

```
sudo mkdir /media/sda1
```
Reboot and you can access it through /media/sda1.
