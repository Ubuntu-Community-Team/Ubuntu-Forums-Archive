### Post by it.henrik on 2006-02-26
I reccomend you download the azureus client from their webpage and run that one. It will give you nice information about the port handling stuff. It also includes a NET test which you can use to check if you can through on the specific ports you want to use. 

I would guess that if it is only bittorrent specificly that does not work then your ISP blocked it (and prob other p2p protocols). Azureus includes a nice feature that encryps the traffic so traffic shapers cant recognise it.

Azureus have some nice status indicators in the bottom left of its window. If you have green light on all of them and the user count lies around 900 000 then it should work perfectly.
