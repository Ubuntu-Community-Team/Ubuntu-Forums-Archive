### Post by mb_webguy on 2009-01-24
If you mean that you want Firefox to open with the same tabs and pages you had open when you closed it, go to Edit->Preferences, and on the Main tab, the Startup section, change the drop-down select labeled "When Firefox starts:" to "Show my windows and tabs from last time".
