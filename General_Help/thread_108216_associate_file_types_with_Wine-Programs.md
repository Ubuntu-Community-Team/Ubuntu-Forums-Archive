### Post by dcstar on 2005-12-25
[QUOTE=The Hedgehog]Is it possible to associate files with a program you've installed under wine(e.g. when you double-click on an MP3 it will open in WinAmp).[/QUOTE]

Right-click the file in Nautilus, got to Properties, and put in the command to launch the program you want in "Open with" (use the Custom command area).

To use wine, it would be something like: "wine /path-to-wherever-this-is/winamp.exe" or somesuch.
