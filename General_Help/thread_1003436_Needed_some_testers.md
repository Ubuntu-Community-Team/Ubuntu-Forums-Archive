  I am maintaining a project called [Gnome-EasyBackup]("https://wiki.ubuntu.com/GnomeEasyBackup") and looking for someone to help give a feed back. The project is on LaunchPAD so you can look at the code [here]("https://launchpad.net/gnomeeasybackup"). The basic testing should not takes much time (see [here]("https://launchpad.net/gnomeeasybackup/+announcement/1574") (quite a detail instruction of how the test may be done)). [Gnome-EasyBackup]("https://wiki.ubuntu.com/GnomeEasyBackup") is backup solution using rdiff-backup as back-end. **Its utilize existing Gnome features such as Nautilus emblems, Nautilus script and Zenity as front end.** The test should never give you any risk or make you loose data unless you:
[LIST]
[*]have files in folder /opt/EasyBackup
[*]have files in ~/.EasyBackup
[*]or, have custom nautilus emblems name "To Backup", "To Not Backup" or "System Backup"
[/LIST]
NOTE: All these conditions are very rare.
