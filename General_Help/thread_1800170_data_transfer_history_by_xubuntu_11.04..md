### Post by Toz on 2011-07-09
If fedora is using the gnome-system-monitor, you can install it if you need it. Look for it in the Software Centre. Also, **bmon** is a good curses-based bandwidth monitoring tool.

To disable automatic updates, from your Main Menu, select System then Update Manager. Click on the Settings button (enter your password) and configure as needed in the "Automatic Updates" section.
