### Post by oldfred on 2023-09-27
SSDs are just drives. Fast drives.

Typically best to use Windows tools for Windows & Linux tools for Linux.
Is new drive also NVMe?

I have never moved Windows to larger drive and have not installed it since XP in 2006.

But built system in 2016 with M.2 port, but NVMe was so much more expensive I used a smaller M.2 SSD.
Updated M.2 SSD to larger M.2 NVMe drive as NVMe drives are now a lot lower in cost. I did new install & restored data.

I ended up putting M.2 SSD into M.2 to USB adapter. External drive was almost as fast as external drive as it was as internal drive. Do not now plan on buying any more flash drives as they are  now so slow comparted to SSD. Ended up having to use it with very old 2006 laptop and made that laptop at least functional for emergency use.
