### Post by P4man on 2010-05-01
Welcome to ubuntu :)

umount is a command to unmount a filesystem. You probably dont need it, in the file manager just click the "eject" icon next to the drive.

if you do want to run it, add "sudo" in front of the command. I suggest some reading here:
[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)

More help on mounting usb here:
[https://help.ubuntu.com/community/Mount/USB](https://help.ubuntu.com/community/Mount/USB)

Feel free to ask any specific questions!
