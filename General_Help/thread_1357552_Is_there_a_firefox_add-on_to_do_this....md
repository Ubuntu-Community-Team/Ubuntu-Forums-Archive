### Post by Darael on 2009-12-17
> **emigrant said:**
> by the way, why does it says, 'external ip' rather than current ip or like that?

Because if you're behind a firewall or router, you've also got an internal IP which is only available from inside the network you're on. Your external IP is the IP of the computer/router/whatever that is visible to the outside world, but when you're using Tor, it will appear to be the external IP of the last computer in the onion-routing chain, ie the last hop of your torified connection.
