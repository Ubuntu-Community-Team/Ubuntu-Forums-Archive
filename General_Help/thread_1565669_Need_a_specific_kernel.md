### Post by TBABill on 2010-09-01
Go to [www.distrowatch.com]("http://www.distrowatch.com") and click the search function up top. Around the middle is a drop down of different apps you can search by version number. Enter 2.6.15 in the box and in the drop down select linux. Click search and any distro using that kernel will show up. You will have to manually enter each subsequent earlier kernel to get .14, .13, etc. If you want Ubuntu specifically, that kernel was used on 6.06 LTS.
