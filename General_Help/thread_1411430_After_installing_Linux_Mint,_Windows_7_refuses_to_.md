### Post by Mark Phelps on 2010-02-20
Since it's an OEM installation, I'm presuming that you do NOT have a Win7 Install DCD, right?

And, even though you intended to mess with a machine for which you did not have a repair disk, did you at least have the foresight to use the builtin Win7 function to create and burn a Win7 Repair CD?

I'm guessing NO on both accounts ...

So, go to the NeoSmart Technology forums, download their Win7 Repair CD image (you'll have to find out ahead of time whether you need 32-bit or 64-bit), burn that to a CD, and use that to boot their machine and run Startup Repair.  You may have to do that three time to fix all the boot issues.
