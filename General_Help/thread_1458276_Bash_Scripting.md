### Post by Miyavix3 on 2010-04-19
Think of it as a bunch of terminal commands stored in a text file mixed with programming principals. When you run the file, it just runs all the terminal commands.

It can be as simple as this:
```

cd ~/Desktop
mkdir New_Folder
cd New_Folder
touch New_File

```
All this does is go to your desktop make New_Folder then go in that folder, and make New_File.

For this, you could run by double clicking, after you give it executable privs of course. You could also go in the terminal and run it from there.
