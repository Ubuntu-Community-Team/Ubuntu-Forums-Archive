### Post by wadesmart on 2005-10-24

Do you have System Monitor on your top panel? If not, right click and choose Add to Panel and then add it. Then Open FireFox and surf for a while. That app shows memory and cpu usage. If the cpu usage tops out or starts to top out, click once one the app and find FF and then kill the process. 

I have had this same problem as well. It isnt occuring as much as yours is but when it does happen, that is about the only way to regain control of my computer.
