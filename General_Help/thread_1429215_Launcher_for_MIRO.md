### Post by 3rdalbum on 2010-03-13
Miro should install a launcher in your Applications menu (or in your case, the netbook interface) - are you sure you didn't just install the "miro-data" package?

Assuming you have the 'miro' package installed, you can hit Alt-F2 and type 'miro' to run the program. You can also edit your menu and add a launcher that runs the 'miro' command.
