### Post by Therion on 2009-01-20
You'll need to install CCSM (CompizConfig Settings Manager):```
sudo aptitude install compizconfig-settings-manager
```

More information here:  [http://wiki.compiz-fusion.org/CCSM](http://wiki.compiz-fusion.org/CCSM)  and here:  [http://forum.compiz-fusion.org/](http://forum.compiz-fusion.org/)
