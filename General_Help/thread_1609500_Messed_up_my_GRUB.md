### Post by oldfred on 2010-11-02
The link had info on useing grub to show what modes are valid, not all modes may be available in the VESA driver mode. This is because it is before the video driver has been loaded.

from grub manual:
[http://www.gnu.org/software/grub/manual/grub.html](http://www.gnu.org/software/grub/manual/grub.html)
‘GRUB_GFXMODE’Set the resolution used on the ‘gfxterm’ graphical terminal.  Note that you can only use modes which your graphics card supports via VESA BIOS Extensions (VBE), so for example native LCD panel resolutions may not be available.  The default is ‘640x480’.
