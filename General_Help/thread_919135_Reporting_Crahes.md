The first step is generally to come here so that we can see if we can suggest a solution.  If it still doesn't work, go and [file an Ubuntu bug]("https://launchpad.net/ubuntu/+filebug").

Sometimes it is appropriate to file a bug directly with [GNOME]("https://launchpad.net/ubuntu/+filebug"), [KDE]("http://bugs.kde.org/"), or other project.
