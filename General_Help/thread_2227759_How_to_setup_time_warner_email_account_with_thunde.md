Ok guys wanted to post this to help anyone behind me. so this is how you will setup your account.
                   
incoming = IMAP | mail.twc.com | 143 | none                  | autodetect
outgoing = SMTP | mail.twc.com | 587 | STARTTLS      | No authentication

as long as your user name and password are correct this will work. Hope this helps someone and saves them time.):P
