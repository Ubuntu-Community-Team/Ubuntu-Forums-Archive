### Post by searchfgold6789 on 2010-09-30
Try running gshutdown as root, from the terminal.
You could also try the main linux shutdown command in the terminal which works like this:
```
sudo shutdown <no. of minutes>
```
For example, to shutdown in half an hour,
```
 sudo shutdown 30 
```
Or to shutdown now
```
sudo shutdown 0 
```
