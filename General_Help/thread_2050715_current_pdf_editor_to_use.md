### Post by zerubbabel on 2012-10-24
I use the [pdf-xchange]("http://www.tracker-software.com/product/pdf-xchange-viewer") viewer for annotating pdfs. It's a Windows program, but it runs perfectly under Wine. The free version allows markups and comments to be added to existing pdfs, and the "Pro" version ($37.50) adds many more editing abilities. I've used it for years, and highly recommend it.

I have not found any open-source native Linux product that allows standard annotations to be added, imported, and exported. If I have overlooked any such product, I'd be glad to know of it. I don't think even Xournal supports annotations that can be exported and sent to another party such that they can be imported by Adobe Acrobat. I find this ability to be essential when collaborating on publications with others who work on other platforms.
