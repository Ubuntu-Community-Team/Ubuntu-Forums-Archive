### Post by Rubi1200 on 2011-12-25
Hi,
you need to install GRUB to the MBR of the drive rather than the partition where Ubuntu is installed.

From the LiveCD, run the commands again but this time:

```
sudo mount /dev/sda3 /mnt
sudo grub-install --[COLOR=Red]boot[/COLOR]-directory=/mnt /dev/sd[COLOR=Red]a[/COLOR]
```
The parts highlighted red are the important changes in the commands.

Let us know if this works for you.
