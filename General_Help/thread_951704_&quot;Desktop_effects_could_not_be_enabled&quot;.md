### Post by fikelfikel on 2008-10-18
> **Ryadt said:**
> Try enabling your graphic card. It's in System > Administration > Hardware Drivers.

(G2bandrew) That solution should work, but in some cases, you'll have to be:

1) Connected to the net in Ubuntu
2) Google "Envy Nvidia"
3) Get the program and
4) You should be able to enable the desktop effects.

It also works for ATI Graphics cards as well.

If you are having some trouble installing or getting Envy, use the 1# solution, but update the driver after that.

Also, if that dosn't work, go into Add/Remove and search for Nvidia. My ATI graphics worked with the Ubuntu version of Catalyst Control Center, so there should be a program available for your Nvidia graphics.
