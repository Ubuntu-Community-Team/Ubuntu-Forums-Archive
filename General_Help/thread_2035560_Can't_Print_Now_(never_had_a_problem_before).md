### Post by johnsmoke on 2012-08-04
Figured it out!! I had to get some things printed, so I drug out the old dusty laptop that had windows vista on it, and hooked up the printer to it... tried to print and a message comes up saying that I need a color catridge in it in order to print, then it gave the option to just use the black cartridge..... I guess the message just didn't come up in Ubuntu and it just didn't print and gave the error. Anyway, luckily I already had a replacement color cartridge here already, I installed it and hooked the printer back up to my Ubuntu PC and it printed perfectly :)  Ahhh, made my weekend. I'm happy... and I didn't want to have to hook this old windows laptop up every time I wanted to print something. Made my weekend!
