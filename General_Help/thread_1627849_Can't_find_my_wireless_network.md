### Post by papibe on 2010-11-22
There's a security option that disable the broadcast of the name of the network. Check your router settings and loook for an option called "Enable SSID Broadcast" or similar.

Nevertheless, If you know the name of the network you can create a connection manually by going to System -> Preferences -> Network Connections. Go to the wireless tab and click add. Use the SSID field for the name of your wireless network.

I hope it helps.
