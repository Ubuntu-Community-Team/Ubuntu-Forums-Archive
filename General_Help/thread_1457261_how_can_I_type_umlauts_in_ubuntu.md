### Post by drs305 on 2010-04-20
> Is it possible to set my keyboard up to be the same as a German keyboard?


System, Preferences, Keyboard. Layouts tab.

Add the German keyboard, then move it to the top and select "Apply System Wide". This will provide the German layout. When you are selecting the keyboard, pressing "Add" displays the key mapping. (Not tested on Hardy).

### Post by mcduck on 2010-04-20
To use the german keyboard layout, you'd actually need a keyboard that has enough keys. US keyboard doesn't have that many...

Anyway, all those letter _definitely_ are in the Character Map, you'd better look again if you couldn't find them If it helps anything, they will be under the "Latin" script. (If you still can't find them there, make sure you aren't using some strange font that only has a few basic letters. Any of the Ubuntu's default fonts will include all the letters you requested, and many more..)

What comes to typing those letters directly on US keyboard, I can't really say how to do it as I use a different keyboard layout and have the same amount of keys the German keyboard has (and things work quite differently on Finnish layout anyway, as I have the AltGr key and thus no need for a compose key). But I think I have some old US keyboard laying around somewhere, I'll try to find it and test how to do this (unless somebody typing in German with US keyboard comes here and gives you a solution before that...)

edit: Here's one way to do it: Set your keyboard layout to "USA International (AltGr dead keys)". You'll now find all those letters on your keyboard, and the right Alt key functions as AltGr, allowing you to access the extra letters. For example ß can be typed using AltGr+S, and ü using AltGr+Y. (Add the shift key to get capital letters, as usually).

Somebody with more experience on using compose key instead of AltGr will have to help you with that way of typing letters if this way doesn't solve your problem...
