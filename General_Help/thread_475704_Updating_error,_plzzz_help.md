### Post by OutCell on 2007-06-16
Ok i was able to get rid of the error msg but i am not sure if it's the correct way or not..

I went to System --> Administration --> Synaptic Package Manager
Then Settings --> Repositories --> Download from --> choose others and let the program choose the best server.. I live in Qatar, but the program choose the best server for me in Finland (And it was very fast).. So now i don't get the error anymore and i get + install all the latest updates..


:D


I hope this is OK!! :)
