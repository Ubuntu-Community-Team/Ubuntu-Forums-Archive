### Post by Zorael on 2008-05-05
Did you use any extra third-party repositories or scripts to get a newer Compiz version in Gutsy than what was available from the official repos? Upgrading doesn't always work when the versions installed don't match with what's expected, see.

Try reinstalling compiz completely by searching for "compiz" in Synaptic and tagging all installed matches for **complete removal**. Grab "emerald" while you're at it. Then reinstall the **compiz**, **compizconfig-settings-manager** and **emerald** packages.

Terminal way.
```
$ sudo aptitude purge ~ncompiz ~nberyl ~nemerald ~nlibdeco
$ sudo aptitude update
$ sudo aptitude clean
$ sudo aptitude install compiz compizconfig-settings-manager emerald
```

Lastly, are you using some form of script to start Compiz? You shouldn't need to use SKIP_CHECKS to get it going with Nvidia cards.

Your xorg.conf is in a bit of a mess but hopefully Compiz alone is at fault.
