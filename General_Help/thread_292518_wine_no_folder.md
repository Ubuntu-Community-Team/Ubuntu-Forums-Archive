### Post by CatKiller on 2006-11-03
Files that start with a . are hidden. Ctrl-H or "Show Hidden Files" from the View menu will show you. I believe that the .wine file is created when you run **winecfg** for the first time.

If you installed Wine through Synaptic, you should be able to uninstall it the same way. If you installed it by double-clicking a .deb file, you should be able to uninstall it with Synaptic still. If you compiled it from source, I have no idea.
