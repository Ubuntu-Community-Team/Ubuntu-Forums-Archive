### Post by ruibernardo on 2007-09-18
Linux is a Unix-like operating system. 

It was developed to be like Unix. The Linux kernel developed by Linus Trovald (and others) back in 1991/2. It is not a copy of unix, but "acts" like one, and the code was all developed apart from the Unix one.

That's why the code is different.
