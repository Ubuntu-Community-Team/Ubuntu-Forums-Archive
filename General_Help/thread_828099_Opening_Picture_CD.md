### Post by indytim on 2008-06-13
Does the CD have some variant of a Windows executable embedded in it for viewing?  If so, your options are maybe Wine or....

If it just pictures as Indytim said, then the default browser will open; to view the pictures, if this is the case, right click on the file, and there will be a default app to use to view the picture. Click on this, and you should be able to view it.
