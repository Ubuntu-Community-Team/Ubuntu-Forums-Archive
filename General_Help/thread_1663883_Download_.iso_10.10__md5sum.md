### Post by ankspo71 on 2011-01-12
Hi,
If you are still having trouble downloading the ubuntu server iso from the main site, then you might want click on the link below to find a new download location:
[http://www.google.com/search?q=index.of+ubuntu+iso+10.10+server](http://www.google.com/search?q=index.of+ubuntu+iso+10.10+server)
(the "MD5SUMS" file is included on the sites you will be seeing)

Lucky for us, Ubuntu has 100's of official mirrors located around the world to download iso's from. Oddly enough, I don't see them listed on the Ubuntu site.

An example to check an md5sum (in a linux terminal) would be:
```
md5sum /path/to/ubuntu-server.iso
```
