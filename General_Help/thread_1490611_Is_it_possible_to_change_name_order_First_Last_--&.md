### Post by sgosnell on 2010-05-23
This isn't optimal, but you can go to each contact, open it for editing, and there is an option on the Contact tab, for File Under.  You then get a dropdown with both options.  You can also select this each time you add a contact.  It would be convenient if there were a global option for this, which would let you select this for the entire database, but I don't know of one.
