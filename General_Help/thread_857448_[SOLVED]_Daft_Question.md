### Post by ajgreeny on 2008-07-12
In nautilus Preferences go to Display tab and make sure that None shows in the drop down boxes if that is how you want it.  That certainly does the trick on my machine.  I don't use icon view except for photo folders, so it is not something that I had even thought about.
