### Post by westie457 on 2017-03-24
There is no need to delete any partitions.
Run the installer and when you get to the 'How do you want to install?' window select the Something Else' option and prepare your partitions.
See here for more guidance. [http://askubuntu.com/questions/343268/how-to-use-manual-partitioning-during-installation](http://askubuntu.com/questions/343268/how-to-use-manual-partitioning-during-installation)
