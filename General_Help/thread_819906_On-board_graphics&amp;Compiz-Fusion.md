### Post by jpeddicord on 2008-06-05
I'm not sure if that chipset will work with Compiz or not, but it's worth a try!

Go to System > Preferences > Appearance, and then select the Visual Effects tab. Choose the Normal option and see if anything changes. If you get a dialog that asks if you would like to keep your changes, then Compiz is up and running. If you get an error message, then most likely your VIA chipset will not run Compiz.
