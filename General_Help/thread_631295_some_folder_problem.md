### Post by bulletxt on 2007-12-04
what do you mean for system files? *nix has nothing to do with Windows&#8482;.

those  folder &/or files are hidden because they have an "." in front of the name.
for istance if you call a folder  " .blabla "  it will become hidden. take away that "." so it becomes " blabla " and it won't be hidden anymore. those folders are hidden by default because they are the folders that hold configuration files of the software you use. there is generally no need to view those files unless you have to do something specific. 
if for example you want to reset firefox simply delete .mozilla folder.
