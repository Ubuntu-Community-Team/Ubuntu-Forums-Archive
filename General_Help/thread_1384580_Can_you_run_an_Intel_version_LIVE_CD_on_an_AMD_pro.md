### Post by trpnblies7 on 2010-01-18
A friend is trying to rescue some files from his computer because Windows won't boot, so he's making a Live CD. However, he realized that he accidentally downloaded the Intel version of Ubuntu, but he has an AMD processor. Does this matter for a Live session if he's not actually installing anything?

---

### Post by audiomick on 2010-01-18
It doesn't matter at all, even for an install.
Intel, or x86, has become the generic term for 32 bit systems. AMD64 has become the generic term for 64bit systems. The actual manufacturer is not what is being referred to. The only thing to watch out for is that a 32bit system will run on a 64bit processor, but not the other way around.
