I'm not really sure what that is but I imagine you could configure [conky]("http://conky.sourceforge.net/") to look like that if you played with it enough. (Conky is available in the repositories for installation via synaptic, apt-get, or aptitude)

I actually really like [htop]("http://htop.sourceforge.net/") for general system stats and some administration tasks (it needs to run in its own terminal window so if you don't have the screen space to spare it might not be something you want to run, but just thought i would throw it out there)
