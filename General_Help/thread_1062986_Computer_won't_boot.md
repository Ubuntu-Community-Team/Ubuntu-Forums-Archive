### Post by kestrel1 on 2009-02-07
It is possible that something is not seated correctly inside the case. Are you confident working inside the computer?
First unplug everything, ideally you want to be wearing a anti-static earthing strap:
[http://www.geeks.com/details.asp?invtid=WRISTSTRAP](http://www.geeks.com/details.asp?invtid=WRISTSTRAP)
but you might be OK.
Remove the graphics card & then place it back in it's slot, unless you have onboard graphics of course.
Do the same with the memory modules & any other cards connected to the motherboard.
I have had machines that go like this & it is as simple as the CMOS battery have discharged too much. To replace this you need to check the size of the battery, normally a button type (CR2032)
I have also had a strange problem with my machine that it will not boot & I get nothing on the screen, but after removing the power completely (unplugging) It decides to start again.
Anyway, try the above & let us know.

### Post by Capt. Trouserblast on 2009-02-07
I had a similar problem...sounds like a BIOS problem. Certainly removing the CMOS battery will wipe the BIOS and then you will have to add the settings back in. I usually go right down to basics when this happens...take out the HDD and see if the BIOS will load without any HDD after removing the battery for about 15 minutes or so. Then if it will...you know it is working. If it is not you might need to re-install the BIOS program which shuld be available from the manufacturer.Also see if the HDD will work on another machine as an external drive. Somewhere along the line this way you will root out the problem.

Cheers
