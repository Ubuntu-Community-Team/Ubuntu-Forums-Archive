### Post by wilee-nilee on 2012-05-22
I use clonezilla for images/clones, remastersys is good but it is limited to the size of a dvd.  Clonezilla will save everything and the mbr to.  The save is roughly the size of what is being saved as far as data goes though, and has to be put back in a partition equal or bigger in size of the original. If the partition numbers are different this can be dealt with as well.

[http://clonezilla.org/](http://clonezilla.org/)
