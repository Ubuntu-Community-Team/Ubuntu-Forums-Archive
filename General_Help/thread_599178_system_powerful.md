### Post by daengbo on 2007-11-01
Can you give specifics about your system? I might be able to offer advice.

Ultimately, Ubuntu is much more stable over long periods than Windows is. You will likely never need a reboot unless there is a kernel update which requires one. Mine stay running and logged in for months at a time.

Firefox sometimes has memory leaks, especially with regard to Flash, and you should close FF every couple of days, at least. Running out of memory from leaks will make you start to use the disk swap space, significantly slowing your system down.

If you have three users with Firefox running, then the memory leak is increased by threefold.

You may also have a bad plugin or applet.

Let me know the hardware you're using and the programs you're running, and we can speed up your system.
