### Post by blueridgedog on 2011-09-19
Oldos2er's links are a great start.  I would add that Linux is essentially a kernel and a set of utilities.  You can run Linux without a desktop environment or a GUI.  You are embarking on a path of learning the core of Linux.  All graphical environments are placed on top of this core of the OS - and there are many to select from.  Knowing the structure and methods associated with the OS, independent of all graphical methods, will enable you to use most any graphical environment.
