### Post by mike2357 on 2010-04-24
There's a way to use Cairo and not get the annoying black box.  You have to enable a "compositing" manager, such as Compiz or Metacity.  An easy way to do it is to run your Configuration Editor, then navigate to Apps, Metacity and General.  Then check the box that says "compositing manager", then quit the Configuration Editor.  You may have to log out and back in.

Post back if you have any problems.
