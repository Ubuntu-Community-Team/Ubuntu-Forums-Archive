### Post by sisco311 on 2010-07-07
The default theme in Ubuntu 10.04 (and higher) is Ambiance. Ambiance and Radiance are provided by the light-themes package. The gnome meta package does not depend and doesn't recommend the light-themes package, so if you install it and want to use the default Ubuntu theme you will have to install light-themes.
