### Post by kanikilu on 2009-03-11
There are a lot of options, but I've used [EASUS Disk Copy](http://www.easeus.com/disk-copy/) to take a full disk image from a clean install (after updates and customizations), and then I just backup my home directory with rsync every week - I'd do it daily if I had more important data on this machine ;)
