Among several systems I'm running with Ubuntu, I have a system with an older version of Ubuntu -- Breezy and I detected a bug in gThumb. I plan to upgrade. However, this question is geared for a user with the latest _stable_ version of Ubuntu. If you have the latest stable version of Ubuntu, can you try this test if you don't mind?

1. Before you go to bed at night, find an image on your computer. In my case I had a PNG file on my desktop.

2. Open the image with gThumb Image Viewer. It's the default image viewer.

3. Press F11 to go full screen.

4. Let the image sit there all night long (turn off your monitor if you want).

5. In the morning, turn on your monitor, shake your mouse, press Esc and ALT+F4 to close Image Viewer, and see if your mouse pointer (aka cursor) is working again. In my case, it is not working. However, the keyboard is working just fine. I had to do CTRL+ALT+BACKSPACE to kill GNOME and login again. Then, my mouse was restored.

Anyone having this problem on the latest stable version of Ubuntu?
