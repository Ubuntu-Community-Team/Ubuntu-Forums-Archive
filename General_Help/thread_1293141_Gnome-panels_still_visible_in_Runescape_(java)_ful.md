### Post by magnavoid on 2009-10-19
(NOTE: this doesn't fix the in game full-screen option, it just makes the game full-screen no matter what... even free worlds.)

The only way that i have figured this problem out is to use add-ons on Firefox.

such as Full Full screen which can be found here: 
[https://addons.mozilla.org/en-US/firefox/addon/1568](https://addons.mozilla.org/en-US/firefox/addon/1568)
(note: the only way you can exit after enabling full screen with this add-on is to use alt+f4, it isn't really a hassle though)

And also use a Grease Monkey script...

Grease monkey is also a add on for Firefox, and it can be found here: 
[https://addons.mozilla.org/en-US/firefox/addon/748](https://addons.mozilla.org/en-US/firefox/addon/748)

I would also suggest getting ad-block another great add-on for Firefox (gets rid of some of the more annoying ads), which can be found here: 
[https://addons.mozilla.org/en-US/firefox/addon/1865](https://addons.mozilla.org/en-US/firefox/addon/1865)

the grease monkey script you will be wanting to make it appear more full-screenish is found here (it removes the menu bar at the top and most ads):
[http://userscripts.org/scripts/show/31197](http://userscripts.org/scripts/show/31197) (to install it click the green "install" button)

or a more direct link is found here:
[http://userscripts.org/scripts/source/31197.user.js](http://userscripts.org/scripts/source/31197.user.js)

After youre done installing all that just restart Firefox and load up Runescape, enter the "Graphics Options" menu and select the "Resizeable" option.
Then the View > fullscreen or F11
(Note: its a good idea to have extra tab open to a fan site or something else like google.)

More than likely your brother plays Runescape a lot, same as i do. i have been using this setup for months now without a problem.

Enjoy and Happy Runescaping :P
