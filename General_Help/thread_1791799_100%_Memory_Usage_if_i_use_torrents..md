### Post by DoubleMacchiato on 2011-06-27
What symptoms are there when your PC dies? 

I see large memory use also when using these types of programs, but it's mostly in the form of cache. Saidar doesn't seem to report the difference. In any case, cache isn't really committed memory, and your system should free up that space when you open a new process. 

If you're in gnome 2.x, try adding the system monitor applet to your gnome taskbar. Right-click it and go to preferences and check the Memory checkbox. This should display how system memory is currently being used.
