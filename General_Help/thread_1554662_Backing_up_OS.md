### Post by kiridude on 2010-08-17
I'm assuming you want to back up your windows OS?

Do you want to install a dual boot?

Its not really necessary to back up your OS as you must already have it on a cd from when you installed it, or a backup came with your computer if it came with the machine.

I would say just back up the files and folders that are important to you. If you want to save time in the face of disaster, however, you could simply take an image of your hard drive as is right now and use that to reinstall.

### Post by Jett Deion on 2010-08-17
START > All Programs >Accesories > System Tools > Backup
You can use the wizard or select manually the files you want to back up.
