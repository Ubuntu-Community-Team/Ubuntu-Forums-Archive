### Post by deadflowr on 2016-01-16
A looong while ago, I had something like that happen.
Simple solution I did was to simply create a new thunderbird profile and log into it, then recreate my accounts.

```
thunderbird --ProfileManager
```

kill any running thunderbirds first, though.

This will also start you totally blank, no add-ons or other tweaks you may have setup.
