### Post by deputy on 2008-12-15
I was able to get a Zune 80gb working using Vmware Server 2.0. Under vmware I installed XP with SP2, and downloaded the latest Zune software. I connected my Zune, enabled it in virtual XP and the Zune software picked it up. I am able to sync music, pictures, videos. I can connect to Marketplace and download stuff. Basically everything works as if it were on a real XP machine. The reason I used Vmware Server 2.0 is because it supports USB 2.0.
