### Post by jmfal on 2010-08-19
Hi Sirguy

I had the same problems with audacity not cooperating

try adding those app to your user/group

system>administration>user and groups
    click>manage groups>add>type in app>close that window>scroll through the group settings list >click on that app>click properties>put check in user name box

worked for me, just be careful, as adding a app that uses the internet  can open up a way for a hacker/virus. when I use audacity I DISABLE THE NET, so I`m safe.

Hope this helps out
