### Post by TBABill on 2012-05-10
Ubuntu 11.04 was built on Gnome 2 while 11.10 and 12.04 are built on Gnome 3. You will not have "classic" as you know it today on either of the newer versions. There is a fallback session, but not the classic version you are accustomed to having available in 11.04.

```

```


Logout then the option will be avbl from the cog wheel.

This is not exactly the same as classic but near enough.
