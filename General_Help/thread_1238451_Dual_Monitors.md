### Post by fluffman86 on 2009-08-12
Ok, so if you're currently using a VGA monitor, then your new one should be DVI.  If you already use DVI, then your new monitor will need a DVI-VGA converter.  I'm sure you already have your nVidia drivers installed, and you will also want to install the nvidia-settings package with apt-get or synaptic.  How you set it up from there will be up to personal preference, but it really helps if the monitors are the same size/resolution.
