### Post by dataw0lf on 2005-04-13
Since it's not in Ubuntu's official repos, you might want to 1) install it from a debian package or 2) compile and install it yourself.  

[http://packages.debian.org/stable/web/php4-ming](http://packages.debian.org/stable/web/php4-ming) for stable Debian pkg
[http://ming.sourceforge.net](http://ming.sourceforge.net) If you wish to install it yourself

Be aware of libswf as well (although I've been told Ming is much better).
