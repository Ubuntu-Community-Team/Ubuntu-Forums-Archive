### Post by fragos on 2008-09-21
Many things effect battery life. Backlight brigthness for example. CPU useage also impacts battery life so background tasks can have an effect. Disk useage also has an impact. Install the package "powertop" which will identify configuration changes relating to background tasks you can make to improve battery life.
