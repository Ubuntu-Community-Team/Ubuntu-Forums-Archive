### Post by Ramses de Norre on 2006-06-10
I hope you are aware that a flash card wont live long when doing things like running an os from it. Flash memory has a limited amount of read/write cycles which isn't a problem when you use it for storage but an operating system causes many read and writes and you're flash card will get a lot of broken adresses very soon. Meaning the working memory space will shrink fast.
