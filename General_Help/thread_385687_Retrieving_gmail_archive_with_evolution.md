Suggestions?

---

### Post by floke on 2007-03-16
I think you need to change the gmail 'pop and forwarding' settings (assuming that gmail is already pumping mail to evolution), since when you pull mail from it, it updates the settings so that you don't automatically pull all mail down every time you check in. So change the date in pop back to all mail and that might do it.
