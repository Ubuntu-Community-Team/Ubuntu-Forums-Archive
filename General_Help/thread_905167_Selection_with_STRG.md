### Post by bthoward on 2008-08-30
Are you running compiz?  Perhaps you've set compiz to attach to ctrl for some reason.  Try pressing ALT+F2 and in that box type "metacity --replace" then see if that fixes your problem.  If it does then we need to start going through your compiz settings looking for the culprit.

Edit:  Also try holding ctrl and clicking and dragging on a window...  If it moves the window then we can fix that pretty easy.
