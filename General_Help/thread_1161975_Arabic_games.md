### Post by geirha on 2009-05-17
Most games are translatable, so if you find a game that you'd like to have in arabic, you can translate it yourself. For instance, here's an overview of the translations for tuxmath in Ubuntu Hardy: [https://translations.launchpad.net/ubuntu/hardy/+source/tuxmath/+pots/tuxmath](https://translations.launchpad.net/ubuntu/hardy/+source/tuxmath/+pots/tuxmath)

There's no arabic translation yet, but if you create a user on launchpad, you can go through the 84 sentences and translate them to arabic. The translations you do though, will only be suggestions, so once you're done, you'll need to contact the Arabic translation group and ask them to review and approve your translations. The translations will eventually end up in updates on your Ubuntu system, but unfortunately, such updates doesn't happen very often.

A better way may be to download the translation file (.po) and edit it in a regular editor, or by using specialized translation software. Once you're done, send the .po file to the arabic translation group for review, so others can benefit from your tranlations, and also, you can convert the .po file to a .mo file, and copy it to your system (/usr/share/locale/ar/LC_MESSAGES/tuxmath.mo). After that, tuxmath will appear in arabic instead of english.

See this page for a full documentation on how to translate: [https://wiki.ubuntu.com/TranslatingUbuntu](https://wiki.ubuntu.com/TranslatingUbuntu)
