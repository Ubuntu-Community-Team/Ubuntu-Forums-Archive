I have several bookmarked ftp shares under Places->Bookmarks.

If I want to edit the properties of one of these shares, which file should I edit?

---

### Post by mciverza on 2011-01-21
Open your file browser.
Select the bookmarks menu, and the click on Edit Bookmarks.

A new window will open up. Find the bookmark in the left pane that you want to edit.
click on it, and its properties will be displayed on the right side of the window

Modify the "location" text area as required, then click 'close'
