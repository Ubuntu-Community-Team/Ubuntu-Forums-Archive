### Post by branau on 2015-06-24
Why would you want to block him from using the terminal? If it's to prevent him from changing any settings or deleting any important files, you can always get him set up in the guest session so any changes he makes won't be saved and he won't have access to anything important unless he knows your password.

### Post by ajgreeny on 2015-06-24
Or if your brother wants a named account of his own, set him up as another new user from the "Users and groups" but give that user whatever limited rights you want him to have.

The default is for the first user only, ie you, to have admin rights, so this way he will be unable to make any changes to the system, install or remove applications, and if you change your own home's permissions to 700 rather than the standard 755, he will not even be able to see any of your personal files.
