### Post by FesterHead on 2005-06-27
Ripped files are set in:
Config tab -> Rip Tab -> Ripper tab -> Rip file format textbox

Encoded files are set in:
Config tab -> Encode tab -> Encoder tab -> Encode file format textbox

Can't help with the mp3 encoding, though.
I use Grip to extract to flac (search Synaptic for 'flac') using command line options '-5 --verify --replay-gain %w', copy the flacs to a remote Windows machine via samba, rdesktop in for mp3 conversion using dbPowerAMP and flac/mp3 tagging with The Godfather.
