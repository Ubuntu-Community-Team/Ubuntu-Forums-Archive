### Post by STaRDoGGCHaMP on 2011-08-13
Sorry guys for bumping this pretty old thread. But i think it is not the fact that ubuntu is responding too slow. I experienced this as well and i had the same idea of a slow "reacting" ubuntu. But in my opinion it is something else: Under ubuntu caps lock does not work **while** the key is pressed, at least for me.

You can try this yourself. Press the caps lock key and keep it pressed, then press any letter key. You will notice, that the letters will not change until you release the key (or it will cause another strange behavior). On a windows machine you should achieve the opposite effect. I did not find a fix for this till this day.
