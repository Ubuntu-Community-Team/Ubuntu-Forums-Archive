### Post by Ayuthia on 2008-05-01
If you install Ubuntu inside of Windows, you will lose some performance.  I have installed Hardy on my Dell laptop that is running XP.  It is using wubi (the Windows install method).  It works fine, but it is slower than I am used to.  The nice thing about it is that if you don't like it, you can remove it.  Another issue is that you cannot suspend Ubuntu when you install it this way.

Like ghstridr said, take the live CD for a spin.  If you like it, you can take the next step of installing it in Windows.  If you really like it, you can then partition a space for it and gain some performance.  If you find out that you don't need Windows anymore, you can just do the full install!!!
