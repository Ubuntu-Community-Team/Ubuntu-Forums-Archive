### Post by cymbaline42 on 2011-05-10
I'm sure there are many ways to do this, here's one solution I can think of (and what I have done in similar situations):

Install Ubuntu Tweak ([http://ubuntu-tweak.com/](http://ubuntu-tweak.com/))

Open it, on the left column, scroll down to System - File Type.  On the right column, under Categories, choose All.  Scroll down to HTML document, click Edit and choose the software you want to open it with.
