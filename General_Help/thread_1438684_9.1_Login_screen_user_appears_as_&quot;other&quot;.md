### Post by aimGuy on 2010-03-25
Hello all, I just started using ubuntu desktop 9.1 and think its great!

I am having a problem with my login screen. The user "testUser" appears as Other or Null. So it looks like there are 2 "other" accounts when I go to log in. When I mouse over the top "other" it does say "log in as testUser", but only if I mouseover.

Has anyone else had this problem, or know how to fix it?

Thanks guys.

---

### Post by cdenley on 2010-03-25
It is probably listing the user's full name, not the username, and you probably don't have a full name for the user.
```

getent passwd testUser
sudo chfn testUser
getent passwd testUser

```

---

### Post by aimGuy on 2010-03-25
Thanks for the response. The full name had been set. I did find out the problem though... I had an ampersand in the name willy & bob, type thing, and apparently ubuntu didn't like it, and failed to tell me. I removed the ampersand and bingo!
