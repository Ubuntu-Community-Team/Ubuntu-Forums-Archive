### Post by matthew on 2005-09-01
Fabulous idea!

First step, search the repositories using Synaptic. Make sure you search both name and description for things like "kids" or "children" or "education."

Some to look at: gCompris, and there are a lot of k"insert name here" educational apps that are intended for KDE but most work well under Gnome as well.

i have used others, but my kids are 2 and 3 and what I can think of right now is a bit too young...

Oh, yeah, there is a Debian jr package that has good stuff in it as well.

Happy searching, and thanks for thinking of others!! If you find a good list and you have a minute we would love a follow up post.
