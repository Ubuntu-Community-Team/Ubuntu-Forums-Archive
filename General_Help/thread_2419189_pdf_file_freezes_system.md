### Post by DuckHook on 2019-05-17
[list=1]
[*]Adobe support for Linux is pathetic—almost criminal. You state that you have Adobe Reader. How did you install it? Adobe stopped issuing a Reader for Linux about five versions ago and it hasn't been available in the repos for years.
[*]The manual you are trying to access is either corrupted, uses some really oddball Adobe calls, or is so GPU-intensive that it crashes some display routine (unlikely).
[*]There are very few occasions where a PDF file simply refuses to work on Linux-based readers, but they do exist. I recently ran across one on a government website (of all places). When I run across such oddballs, I do two things:
[list]
[*]Open them in a VM running Windows using the latest Adobe Reader,
[*]E-mail the webmaster/mistress of the site asking them to stop restricting access to only lousy, proprietary bloatware.
[/list][/list]
This last is probably pointless, but unless we make ourselves heard, Linux users will perpetually be relegated to the backwaters of the dispensable.
