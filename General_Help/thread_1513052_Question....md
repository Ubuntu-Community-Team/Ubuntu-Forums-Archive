### Post by Tim Sharitt on 2010-06-18
You'll need to edit the /etc/default/grub file as root. From a terminal enter:
```
gksu gedit /etc/default/grub
```

GRUB_DEFAULT is the option you want to change. It can be a number to set the default entry by menu position starting with 0, so if Windows is the third entry you would make this GRUB_DEFAULT=2.

You can also set it by name, so it would be something like GRUB_DEFAULT="Windows XP". You need the exact name of the menu entry, which you can find in the /boot/grub/grub.cfg file (DO NOT edit this file).

Lastly you can set it to GRUB_DEFAULT=saved which will set the default to whatever was last booted.

See [https://wiki.ubuntu.com/Grub2](https://wiki.ubuntu.com/Grub2) for more information about Grub.

After the changes are made you will need to run
```
sudo update-grub
```
from a terminal.
