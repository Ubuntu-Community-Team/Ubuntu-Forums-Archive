### Post by bodhi.zazen on 2008-03-12
Wubi is still dual booting.

If you want to run an OS without dual booting you need to run virtualization, ie Virtual Box, VMWare, qemu, etc.

Wubi is still in Beta and if you would like assistance you may want to try their forums.
