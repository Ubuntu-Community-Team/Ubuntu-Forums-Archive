### Post by MarjaE on 2012-03-03
Right-Click or Command-Click on the top panel, where the Applications, Places, and System menus were. Select 'Add to Panel.' You should get a list of things you can add to the top panel, and you should find the Gnome Main Menu in the list.

Hope this helps!
