### Post by bodhi.zazen on 2009-08-26
You can install Ubuntu and leave the recovery partition alone.

If you no longer wish to use Windows at all, simply install Ubuntu and use the entire hard drive.

If you wish to keep windows, keep the recovery partition, resize your Windows partition, and make an extended partition to house your Ubutnu partitions (root and swap at least).

[HowTo: Partitioning Basics - Ubuntu Forums]("http://ubuntuforums.org/showthread.php?t=282018&highlight=partitioning")
