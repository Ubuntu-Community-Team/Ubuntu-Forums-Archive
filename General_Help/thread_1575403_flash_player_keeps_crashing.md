### Post by TwelveGauge on 2010-09-15
go to synaptic and search 'flash' 
remove the packages. 

then run in the terminal:

```
sudo apt-get install flashplugin-nonfree
```

I know you said that you redid flash, but it's picky about how you do it so it's best to try a clean install and see what happens. Also you should ignore anything that pops up in firefox as that just causes problems.
