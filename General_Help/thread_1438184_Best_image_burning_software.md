### Post by quadproc on 2010-03-24
I couldn't say which is best, but I use Brasero, a part of the standard Ubuntu distribution.

When doing ISO files, please be sure to choose the correct selection in the burner software so that you get the proper collection of files burned, rather than just the single ISO file.  This is "burn image" in Brasero.

Also, it is wise to use the slowest burning speed available, for best readability of the finished CD/DVD.

quadproc
