### Post by Upstart Blogger on 2008-03-31
I've recently installed Ubuntu 7.10 on a Sony Vaio with a broken CD drive.  I did it by removing the hard drive, putting it in a USB caddy, and installing via another machine.

When I try to add more applications I get a prompt which asks for the Ubuntu 7.10 cd to be in the drive.  I have the CD in an external USB CD drive, which is being recognized since the CD mounts on the desktop, but the add applications software still demands that a CD be put in the main drive.

Since the main drive is recognized but broken, how do I get round this?  Is there a way to disable the main CD internal drive or do I have to tell Ubuntu to use the external drive in preference?

---

### Post by PmDematagoda on 2008-03-31
If you want all the packages to pulled from the repositories without the use of a CD-ROM then go to Software Sources in System>Administration>Software Sources and deselect the option for the Ubuntu installation CD in the Ubuntu Software tab, that should fix the problem.
