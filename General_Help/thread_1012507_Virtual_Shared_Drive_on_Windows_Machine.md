### Post by nielz on 2008-12-16
It's basically called a windows network and you can achieve something that is more or less compatible with it using (a program named) samba. There's lot's of info about it on these forums and on the internet as a whole, so I'll leave the searching to you :). Depending on how much integration with Windows you want (just a shared drive as opposed to building a domain controller) it can be all the way from very easy to very hard to setup.
