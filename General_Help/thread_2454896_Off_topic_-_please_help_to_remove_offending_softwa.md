### Post by dragonfly41 on 2020-12-08
Open Firefox
Click on Open Menu ("hamburger" icon top right).
Select Addons
or you can just type "about:addons" in URL field
Select Extensions
Remove unwanted Extension


For more information select "Add-ons Support" in left menu panel

P.S. I don't use Firefox much so you might have to tweak these steps  - I use Chromium.
