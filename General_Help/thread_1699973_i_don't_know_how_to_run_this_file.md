### Post by Flyers2391 on 2011-03-04
You should be able to open it in "archive manager" to extract it.  After that, according to [http://tkchat.tcl.tk/features.html:](http://tkchat.tcl.tk/features.html:)

> tkchat is distributed as a single file executable or as a starkit. Starkits are a virtual filesystem in a single file. This means that there is no worrying about where to install dependent libraries and packages - nor will the packages in tkchat affect any other Tcl or Tk applications already installed. Everything needed is in one single file and the only installation required is to download the file and run it.
So you should be able to right click on it and select "Run" I guess

### Post by mcduck on 2011-03-04
> **jolian ramos said:**
> i did that i extract the file sir then i double click on it to run it then ubuntu gives me error message saying "select an application to run it "

Did you download the the .tar.gz package ot the .kit one?

Get the .tar.gz, extract it, then right-click the contained file and select Properties. On the Permissions-tab mark the "Allow executing file as a program"-checkbox. After that you should be able to click the file to run it.
