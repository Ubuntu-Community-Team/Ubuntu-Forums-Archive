### Post by vasa1 on 2015-09-09
This seems to work for me.

Right-click the image.
Click on "view image information".
Near the right-bottom of that screen, click "save as". Specify a destination.
Subsequent images will save to the specified destination.

[s]I think your preference is stored in ~/.mozilla/firefox/...../content-prefs.sqlite. That file changed when I specified a destination for saving images. I'll peer into that file later to confirm.[/s]
