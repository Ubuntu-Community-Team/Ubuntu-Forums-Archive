### Post by ACGarland on 2008-04-30
I'm running kubuntu and I have a battery icon down in the system tray which, when I click on it, brings up configuration for the power manager.  The power manager configuration dialog includes a radio-button control to select what action should be associated with closing the laptop lid (e.g., suspend, hibernate, nothing).
