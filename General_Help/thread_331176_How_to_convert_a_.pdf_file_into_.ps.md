### Post by ctcarton on 2007-01-04
Searching synaptic for ghostscript should have returned a lot of hits.  The package name is gs-esp.  After installing that you should have a program called 'pdf2ps' that you can use to convert the file.

### Post by mexlinux on 2007-01-04
It comes by default in KDE (via kprinter).
Simply open it whit any compatible (kpdf or kghostview in this case), print it choosing the ps pseudo printer, ready.
The same applies for any kind of file to be coverted to ps or pdf,
