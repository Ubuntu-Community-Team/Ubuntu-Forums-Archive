### Post by tbodine on 2007-03-03
In my eyes, a real _full_ switch to Ubuntu GNU/Linux would mean all free software, and nothing proprietary -- which is how I run my computer. But alas, most people who use Linux don't go the full nine yards, they usually still use proprietary software, either Windows stuff (via Wine, etc.) or stuff built for Linux that isn't free.
So if you want to be a true free software advocate and relish in the true _spirit_ of GNU/Linux, forget all of your proprietary software!
Anyways, even if you do go to only Ubuntu GNU/Linux, you can still easily use your Windows apps through Wine. But I still suggest against doing so.

Also -- spacegypsy, your first reason you might not be able to work around if it doesn't work in Wine under Linux, but your second "reason" is no real reason at all. It would be much easier to fix your system from a GNU/Linux(most likely an Ubuntu) LiveCD than to try to fix it from a Windows install.
