### Post by robsoles on 2010-07-12
In 10.04 LTS you can open "Applications"->"Accessories"->"Passwords and Encryption Keys" and find references to passwords the system has stored for you.

I haven't set up chat (am not very sociable but for forum) on this machine so I can't confirm whether or not you will definitely find the password in there but I expect you will. Delete the reference in there and then next time you use Empathy, and enter your password, don't select "Remember Password Forever" in the next dialog box but instead select "Remember Password for this Session".

Exact terminology in 'remember password' dialog may be different but basic idea will be same.
