### Post by Cheesemill on 2012-10-06
If you stay with 12.04 you will only receive security updates for your applications, they wont get upgraded to new versions unless there is a major security issue that requires an upgrade.

The 12.04.1 release is identical to the original 12.04 release except all of the updated packages are on the CD, if you install 12.04 and then update your system you will end up with the same versions of software that you get with 12.04.1.

The point releases just mean that you don't have to download as many updates straight after installation than you would do with the original release.
