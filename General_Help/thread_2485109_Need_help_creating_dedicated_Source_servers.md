### Post by TheFu on 2023-03-20
Ubuntu Server doesn't have any GUI. That's a very steep learning curve for someone without any Linux experience.  Typically, someone new to Linux needs at least 6 months of daily use with a desktop before they should even consider attempting to run a server without any GUI.
[http://linuxcommand.org/tlcl.php](http://linuxcommand.org/tlcl.php)

Look of the section about "job control" and daemons.  I hate to say this, but you are so new and probably expect someone to say exactly what to type.  You'll be wasting so much time because Unix skills build on prior skills that must be learned.  You aren't ready to "run", since you haven't learned to crawl, stand, walk, jog first.  Follow that link, work through the first 12 (or so) chapters to get the basic knowledge needed.

BTW, I have no idea what a "Source" server is. I'm assuming it isn't a code source server, running git, accessed using ssh and git commands.  By just seeing the subject, that's what I expected.
