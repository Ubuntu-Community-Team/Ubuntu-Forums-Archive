### Post by Vaphell on 2011-04-14
when i type valid commands, default action is 'run' and these commands in fact do run.
also do you have terminal plugin for gnome-do enabled? It should allow to chose 'run in terminal' if you want to see the standard output.

[http://do.davebsd.com/wiki/GNOMETerminal_Plugin](http://do.davebsd.com/wiki/GNOMETerminal_Plugin)
