### Post by Mark Phelps on 2011-07-27
IF you really have separate partitions, just make sure that you ONLY use the Vista Disk Management utility to shrink the Vista OS partition.  Do NOT use GParted to do this, as it can sometimes result in filesystem corruption when used against Vista/Win7 OS partitions.

And, yeah, this same warning is probably in the linked thread -- but just mentioning this now in case you jump in without reading the whole thread.
