### Post by scooper on 2006-11-04
A feed reader like liferea can give you some offline capability.  You only get the blurbs and titles in the reader, but still have to get the article online (the web page can usually be read in the reader).  All the Ubuntu forums have an RSS link that gives you the URL you need to provide the feed reader.  This is the icon you may want to look for.  [IMG]http://ubuntuforums.org/images/misc/feed_icon.png[/IMG]

There are probably proxies you can install to cache the web pages themselves locally for a completely offline experience.

Hope that's helpful
