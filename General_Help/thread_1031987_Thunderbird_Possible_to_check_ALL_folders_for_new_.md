### Post by digitalPhonix on 2009-01-05
I use IMAP (GMX is the provider) with Thunderbird, and I have set up a number of filters, so the mail gets sorted depending on who it is addressed to.

The problem is that Thunderbird only seems to notice new mail in the root Inbox folder, however, as GMX has already sorted the mail into the subfolders, Thunderbird misses them.

In Thunderbird, if I go into one of the subfolders, it will 'see" the new mail, but it wont give any notification of new mail.

I know Thunderbird can do the sorting itself (I used to have it set up like that), however I access my mail from a number of locations, so I would prefer if GMX could do the sorting.

Is there anyway to get Thunderbird to "check" each of the subfolders, instead of only checking the root Inbox folder?

### Post by zman0900 on 2009-10-20
I too was wondering this and I just found the answer.

From [http://www.mozilla.org/support/thunderbird/tips:](http://www.mozilla.org/support/thunderbird/tips:)
> Check all IMAP folders for new mail

    Thunderbird can download mail from all accounts when you start the program. Just open the Config Editor, search for the preference mail.check_all_imap_folders_for_new, and change its value to true.
