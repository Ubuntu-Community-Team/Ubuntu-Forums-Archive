I use timer-applet which is an applet for the gnome panel.
It has a countdown,flashes when time is up, sends a message to the notification area 
and you can set a sound file to play when time is up.
I think it's in the universe repo or you could compile from here:
[[COLOR="DarkRed"]_timer-applet_[/COLOR]]("http://timerapplet.sourceforge.net/")

I don't use it for anything as worthwhile as making tutorials.
For me, it just stops me from burning dinner but I'm sure it will suit your purpose.
