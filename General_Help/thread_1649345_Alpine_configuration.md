### Post by xtheunknown0 on 2011-01-11
I'd like to thank (again) Lucas Levrel in [http://groups.google.com/group/comp.mail.pine/msg/73d4e5030f06e556](http://groups.google.com/group/comp.mail.pine/msg/73d4e5030f06e556) for his reply.

Note from the help text that the range you set applies for all groups you choose to subscribe to (and view). For more control, I (now) recommend slrn - when you view a group you can specify how many headers to look at. It took me a while before I decided to have a go at setting up a domain name (this is part of the recommended way of setting slrn up [https://help.ubuntu.com/community/slrn](https://help.ubuntu.com/community/slrn)). This, for me, was the main hurdle - afterwards, learning the navigation keys is a similar process to learning them in Alpine.
