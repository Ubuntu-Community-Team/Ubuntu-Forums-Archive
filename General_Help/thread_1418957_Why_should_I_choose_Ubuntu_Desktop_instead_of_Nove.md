They're both equally good, when you pick the Ubuntu LTS (in april the new Ubuntu LTS will arrive, 10.04). It's like comparing Mercedes and BMW. And you can buy professional support for your Ubuntu, too: [http://www.ubuntulinux.org/support/services](http://www.ubuntulinux.org/support/services)

Only disavantage of SLED versus Ubuntu LTS: SLED has older application versions than Ubuntu (older Firefox, older Open Office). When you want new application versions in SUSE, you have to use the community distro OpenSUSE, which isn't professionally supported by Novell.
