### Post by claracc on 2012-03-20
You can use this greasymonkey script: viewtube [http://userscripts.org/scripts/show/87011](http://userscripts.org/scripts/show/87011), it doesn't use adobe flash to watch videos, instead you can use classic video plugin.

In a pentium III pc, 1GHZ CPU, 512 MB ram from which 64 MB go for sis video card works very well using the viewtube script with epiphany 3.30.6.
