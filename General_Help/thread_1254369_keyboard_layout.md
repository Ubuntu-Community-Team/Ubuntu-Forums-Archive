### Post by jesuisbenjamin on 2009-08-31
The easiest way to add new keyboard layouts is to go to System > Preferences > Keyboard.
Pick up the 'layout' tab and add a language of your choice.
Later on you can chose a keyboard shortcut to switch from one layout to another, i personally use the Win-key.
