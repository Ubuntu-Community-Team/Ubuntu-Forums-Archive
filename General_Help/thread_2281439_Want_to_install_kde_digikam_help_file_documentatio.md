### Post by v3.xx on 2015-06-07
Just install digikam-doc.
```
sudo apt-get install digikam-doc
```
[http://packages.ubuntu.com/trusty/digikam-doc](http://packages.ubuntu.com/trusty/digikam-doc)

[https://help.ubuntu.com/community/UsingTheTerminal#Starting_a_Terminal](https://help.ubuntu.com/community/UsingTheTerminal#Starting_a_Terminal)
