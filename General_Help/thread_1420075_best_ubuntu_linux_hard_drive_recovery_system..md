### Post by mechro on 2010-03-02
**Testdisk/Photorec**. Run the LiveCD and install Testdisk via synaptic. Start Testdisk or Photorec with **sudo testdisk** or **sudo photorec**

Testdisk is generally for recovering partitions and Photorec is for recovering many types of files.

[http://www.cgsecurity.org/wiki/TestDisk](http://www.cgsecurity.org/wiki/TestDisk)
