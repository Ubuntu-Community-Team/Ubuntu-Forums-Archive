### Post by vanadium on 2010-05-05
Install the package "r-base-core" using Synaptic (System - Administration - Synaptic Package Manager; type "r-core" in quick-search, ccheck the box on the left of "r-base-core" and hit "Apply".

Alternatively, you can install using the command terminal
```

sudo apt-get install r-base-core

```
The program is started from the command terminal with the command
```

R

```
