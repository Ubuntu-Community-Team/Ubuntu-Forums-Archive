### Post by philippe_carlo on 2006-07-25
It's gonna be hard for you to choose from:

aria - Tool to download files from the Internet via HTTP or FTP
d4x - graphical download manager
d4x-common - graphical download manager - common files
freeloader - A nice GNOME download manager supporting torrents
gwget - GNOME front-end for wget
jigdo - GTK+ download manager
kget - download manager for KDE
qtstalker - Commodity and stock market charting and technical analysis
wmget - Background download manager in a Window Maker dock app

And many more too. AND all free! And the only you have to do for getting them is typing 'sudo apt-get install <name>'. How cool is that?
