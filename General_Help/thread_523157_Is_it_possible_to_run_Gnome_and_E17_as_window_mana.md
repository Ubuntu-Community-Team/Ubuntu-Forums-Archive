### Post by aysiu on 2007-08-11
Sure.

After all, Gnome uses the window manager Metacity. You can (if you have E17 installed) probably just do Alt-F2 ```
enlightenment --replace
```

If you want things to be more complicated, try this:
[Enlightened Gnome: The Ultimate in Ubuntu Eye Candy](http://ubuntuforums.org/showthread.php?t=53528)
