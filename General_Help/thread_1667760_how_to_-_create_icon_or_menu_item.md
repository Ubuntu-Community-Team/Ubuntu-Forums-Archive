### Post by cscj01 on 2011-01-15
Choose System>Preferences>Main Menu.  From there you can add a new Menu or a new item within a menu.  When you add a new item, a launcher properties window opens.  In the upper left hand corner of the window is a graphic.  If you click on that, you can search for any appropriate graphic in your file system, be it one you created with something such as Gimp or Inkscape or one that comes with the system and apps you have installed.

The link to which you refer is only for adding an icon (launcher) to a panel, not the Applications menu.
