### Post by Big Lizard on 2011-10-21
Disk Utility is the easiest. It's in Synaptic Package Manager and Software Center.
Or in the terminal enter: sudo apt-get install gnome-disk-utility

Watch this how-to: [http://www.youtube.com/watch?v=glzmDtJWzCA](http://www.youtube.com/watch?v=glzmDtJWzCA)
