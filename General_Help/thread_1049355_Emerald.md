How can i configure the emerald themes to take over at start-up?  As of now i set my theme in emerald but when i restart it just goes back to Metacity.  And also, how can i edit the panel at the top?  The one with Applications, Places, and system on it.  I want to change the font color.

### Post by Wartender on 2009-01-24
ok for emerald go to system->preferences->sessions, click add, and type emerald --replace
for the font try downloading gnome color chooser from synaptic

---

### Post by blackened on 2009-01-24
In Compizconfig Settings Manager, in the "Window Decorations" plugin, select the "Command" box. In that box, put emerald --replace. Make sure the "Window Decorations" plugin is active.
