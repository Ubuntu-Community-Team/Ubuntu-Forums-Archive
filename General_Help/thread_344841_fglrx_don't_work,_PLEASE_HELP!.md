Find:
> 
	Driver      "vesa"


Replace With:
> 
	Driver      "fglrx"


Then restart the X server (CTRL + ALT + BKSPACE).

### Post by jocko on 2007-01-24
Try running:
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak #in case you need a backup
sudo dpkg-reconfigure -phigh xserver-xorg
```It should take care of any errors in xorg.conf (I don't see any obvious ones, but it's worth a shot. Don't run the aticonfig tool with the "--initial" parameter, it usually mess things up for me.).
