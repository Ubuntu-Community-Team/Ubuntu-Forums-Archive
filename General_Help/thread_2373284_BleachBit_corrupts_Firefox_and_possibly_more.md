### Post by deadflowr on 2017-10-05
You might try simply creating a new (another) firefox profile
Done two ways, 

Either open firefox and in the menu go to Help > Troubleshooting Information > scroll down to Profiles click on *about:profiles* 
(or simply type *about:profiles* in the address bar)
Then in Profiles, click on Create New Profile and follow the directions; you might consider naming it something like Test profile.

or

open a terminal and type
```
firefox --ProfileManager
```
and create the new profile here.

Then set it to start with the new profile and see if that has the same issues or not.
If it does have the same issues then you can't really blame bleachbit. 
And if it works like it should  then lesson learned about what not to do with bleachbit.
(Not to say you've done wrong, but to say that bleachbit can have some wrinkles, if you're not aware of them.
And now you'd be aware)
