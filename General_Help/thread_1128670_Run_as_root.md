### Post by doas777 on 2009-04-17
use the sudo command:
```

sudo <command>

```
[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)

that'll get you started. it is strongly recomended that you do NOT enable your root account (in fact there is a forum ban on pasting links to tutorials on the topic).
