### Post by 3Miro on 2011-11-01
You thread is tagged as Wubi, if you are using Wubi than you cannot remove Windows without also removing Ubuntu. If you are currently using wubi and you want to move to a Ubuntu only setup, then you need to backup all your files to an external HDD and use the Ubuntu CD to make a clean install completely overwriting anything currently installed.

If this is not your setup or what you want to do, then you may have to do something else.
