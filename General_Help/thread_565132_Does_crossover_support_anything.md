### Post by machoo02 on 2007-10-01
Crossover aims for the the more business-oriented side of things, so they are focused on making sure things like Microsoft Office are working.  There is a list of the officially supported applications [here]("http://www.codeweavers.com/products/cxoffice/").  That said, people do report success in installing numerous other applications, and you can search through that database 
[here]("http://www.codeweavers.com/compatibility/browse/cat").

Applications like the desktop search engine that you mentioned are heavily tied into the Windows API, and you might be better suited to install one of the native desktop search engines like Beagle or Strigi.  You could also try looking over at the [Wine Application Database]("http://appdb.winehq.org") to see if anyone has gotten your software working in regular Wine.

### Post by machoo02 on 2007-10-02
> **uxe1 said:**
> but good luck withit, if you find some way to get it to do wha tit claims plz let me know.Crossover does exactly what it claims: they guarantee that a limited number of programs (mainly MS Office) will install and run.  They do not claim that you will be able to install and run every single piece of Windows software under the sun.  For a lot of programs, one will have more luck and support trying to get things running through regular Wine, but Crossover is designed for those who ***need*** MS Office to install and run.
