### Post by Vaphell on 2010-04-14
there is another thing - since 9.10 old way of doing login screen skins was dropped and currently login screen is much less flexible than it used to be. For example you can't really install a skin that allows user to type in login and password without touching the mouse or disables user list without much hassle. Some things are doable by hacking some config files, but not all features of old GDMs are possible at the moment.
