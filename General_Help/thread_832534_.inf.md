### Post by ibuclaw on 2008-06-17
Its
```
sudo ndiswrapper -i <file>.inf
```
To install the driver. (NOTE: replace **<file>** with the actually name of the file. Also note that the linux terminal is "CaSe SeNsItIvE".)

Then run
```
sudo modprobe ndiswrapper
```
And to load the modules for it (If you wireless has a light, it should turn on immediately after issuing this command).
