### Post by papibe on 2011-09-20
You can use the Minimal CDs (read about and get them [here]("https://help.ubuntu.com/community/Installation/MinimalCD")). Just choose the 'command-line install' option when installing. Here's a very good [guide]("http://www.psychocats.net/ubuntu/minimal") with pictures.

Be aware that you may encounter a little bug on some of these version in which right after you boot for the first time, you boot into a black screen. You just press Ctrl+Alt+F1 to get into the text mode login screen. If you are not going to install a graphical environment, read [this ]("http://ubuntuforums.org/showthread.php?t=1836985&highlight=minimal")to fix that bug.

Regards.
