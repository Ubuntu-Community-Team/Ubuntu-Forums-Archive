### Post by cariboo on 2008-11-15
I personally use Deluge, but Transmission is installed by default. Go to Applications-->Internet-->Transmission. For more P2P programs go to System-->Administration-->Synaptic Package Manager and click the Search button and enter p2p and it will show you some other choices.
