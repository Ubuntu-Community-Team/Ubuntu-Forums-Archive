### Post by FakeOutdoorsman on 2006-12-29
I'm running a command-line installation of Ubuntu Edgy with openbox as my windows manager.  I don't have Gnome or a login manager installed.  When I start XFE it appears partially outside of my screen and I can't move it since the top is not visable or accessible.  XFE is the only program that does this.  How can I move the window or make it start in a different position?

Duh...

As I was typing this I found the answer.  I'll explain for future reference:

1. exit XFE
2. edit the XFE config file at ~/.foxrc/XFileExplorer/Xfe
3. change xpos and ypos values
4. save Xfe config
4. start XFE in it's new, less stress inducing position!
