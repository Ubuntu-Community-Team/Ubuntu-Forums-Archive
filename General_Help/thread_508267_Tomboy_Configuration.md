### Post by zanglang on 2007-07-24
I think you've accidentally removed the Tomboy applet and instead added a launch shortcut to Tomboy. That's the kind of behaviour you get if Tomboy detects an instance running, it just pops up the search box.

Try quitting Tomboy, remove it from your panel, and then add it as an applet (Right-click panel > 'Add to Panel' > select 'Tomboy Notes' > 'Add').
