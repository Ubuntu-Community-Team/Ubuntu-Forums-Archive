### Post by Mark Phelps on 2011-04-18
> **dabyv said:**
> ... i have to intall any other softwares for use windows games on linux?

In a word -- NO.  

There is winetricks and Play on Linux -- but both are add-ons that attempt to improve the Wine experience by making it easier to install and configure certain apps and games.

As already mentioned, you should check out the WineHQ app database for the games and versions you want to run to see their ratings.  Low ratings (or no ratings) means you are wasting your time trying to run that app or game.

### Post by mendhak on 2011-04-19
World of Warcraft does work on Wine and here are instructions on how to set it up:
[https://help.ubuntu.com/community/WorldofWarcraft](https://help.ubuntu.com/community/WorldofWarcraft)

However, I recommend you install PlayOnLinux and have it do the wine prefixing and configuration for you.  It comes with a list of supported games such as Portal, Half Life, Civ, Assassin's Creed and so on which it installs isolated from other games. It also can set up and configure WoW for you.
