### Post by 3Miro on 2012-06-17
To my knowledge, currently there are no tablets that come with Ubuntu pre-installed. Of the ones that exist, theoretically Ubuntu should run on most Windows tablets (x86 or amd64/emt64 architecture), however, considering how heavily modified tablet hardware and firmware is, my guess is that Ubuntu will not run on many of the Windows tablets.

Personally, I would wait until someone like System76 comes with a tablet with pre-installed Ubuntu rather than gamble with other OS.

If you find a Ubuntu tablet, I would like to know about it.
