### Post by JC Cheloven on 2012-03-17
> **komax2000 said:**
> I can open, the usb, but when I search for the file, I cant find it. I created the file wile using ubuntu on the usb, saved at Documents in home folder. Sorry if I'm not so cear, but my English is pretty bad.

At the time of creating the live usb, you were given the option of "saving documents and configurations in an extra space of the pendrive". 

Scenario1:
If you didn't choose that, your file is lost for sure.

Scenario 2:
On the other hand, if you choosed the right option, the file should be over there, although perhaps in some compressed way. If you can't boot from the pendrive, and you don't find your file by searching from an external operating system, I could suggest to run a live session in Virtual Box, with that same pendrive. This would be pretty equivalent to actually booting a real machine in live.
