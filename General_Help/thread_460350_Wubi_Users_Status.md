### Post by minhmeoke on 2007-06-01
I used Lubi; no version number was listed. The installation went fine until I had to reboot; then on the first reboot I encountered minor issues with an Xserver crash (black screen with lines) when the splash screen was supposed to appear. It took me some time to figure out that this was due to my laptop's widescreen display; but it seem like this is a general Ubuntu issue. I solved the problem by changing the entries in the xorg.conf file, as described in this post: [http://ubuntuforums.org/showthread.php?t=457109]("http://ubuntuforums.org/showthread.php?t=457109")

Also, the window borders are missing every time I boot in to Ubuntu; I have to go into terminal and type:
```
metacity &
``` in a terminal to get them back.

Other than that, everything is working fine. Thanks to the developers for creating this great way to try Ubuntu!
