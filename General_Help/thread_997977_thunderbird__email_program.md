### Post by aysiu on 2008-11-30
> **OrangeCrate said:**
> Point of order...

As a long time TBird user, I'm curious as to why you would think that that would make a difference?
I'm also a long-time Thunderbird user, and it definitely makes a difference.

If you're using IMAP, sometimes if your email server is buggy, Thunderbird can have trouble copying the message to the sent message folder even if the actual message was sent.

If you're using POP, there shouldn't be a problem, since the sent message is saved locally on your hard drive. It's also possible that the OP is looking on the mail server for the sent message when it has been copied locally.
