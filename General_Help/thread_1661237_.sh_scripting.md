### Post by JCM_Pico on 2011-01-06
Hello there,

I've been trying to create a script to run some tasks, for example:
I have this script receita.sh, that needs the input of a time data to run. like:
*receita.sh <yyyymmdd>*

and I would like to get it running every 5 am with the computer time date...
I have found the command *date*, that retrieves the system time date like:
*Thu Jan  6 17:39:39 WET 2011*

Is there any way of getting the time date, and make it work in the script

Cheers

---

### Post by sisco311 on 2011-01-06
Try:
```
receita.sh $(date +'%Y%m%d')
```
