A shortterm fix is to install and use the flashblock addon. I have a couple of sites that do just what you describe (I'm not using FF 3.05b). When necessary I just enable flashblock when I visit that site. That way the flash doesn't start and I can see the menu. When I'm done with that page I turn flashblock off.  You can put a shortcut on the browser's menu bar.

I would consider this a VERY imperfect solution but there were times I had to use it. I do hope someone comes up with a good solution - for both of us.
