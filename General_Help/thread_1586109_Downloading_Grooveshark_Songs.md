I am looking for a Grooveshark downloader too.

Orbit downloader requires Windows.  Apparently it runs under wine but I have not tried it.  Although it works it is pretty much useless as you have run the Windows version of Firefox in wine to be able to use it, which defeats the purpose.

There is also SciLor's grooveshark™.com Downloader, which is open source.  But unfortunately it only works in Windows too.  I reckon if this works under wine it would be better as you don't have to use Firefox in wine.
