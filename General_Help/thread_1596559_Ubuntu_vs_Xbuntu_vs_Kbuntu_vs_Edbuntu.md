### Post by snowpine on 2010-10-14
The different members of the 'Buntu family have the same "core" system under the hood, and differ mainly in two ways: 1) The "Desktop Environment" or user interface (Gnome, KDE, Xfce, etc.); 2) The default applications that come pre-installed (but you can install any apps you like; you aren't necessarily stuck with the defaults).

I would suggest visiting the web page for each of the "flavors" where you will find descriptions and screenshots. Pick one that looks nice and give it a test drive! :)
