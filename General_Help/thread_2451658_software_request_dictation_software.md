### Post by SeijiSensei on 2020-10-08
Sometimes there is Windows software that you just can't abandon.  In those cases you can run Windows in a virtual machine. Install VirtualBox on the Linux desktop, then install Windows in a VM. Make sure to install the "Expansion Pack" as well.  You can install VB from the Ubuntu repositories, but I prefer to use the one at Oracle as described here: [https://www.virtualbox.org/wiki/Linux_Downloads#Debian-basedLinuxdistributions](https://www.virtualbox.org/wiki/Linux_Downloads#Debian-basedLinuxdistributions)

I'm pretty attached to my tax software and occasionally use Microsoft Access to work with databases.  I use them in the virtual machine.
