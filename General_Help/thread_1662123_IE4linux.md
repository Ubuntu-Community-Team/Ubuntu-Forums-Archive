### Post by JC Cheloven on 2011-01-07
I recently had to install IE for a friend that has to use one of these shameful "ie-only" web pages. I found that the ie4linux project seems to be somehow outdated in general. 
Also, support for ie7 is said to be experimental, and ie6 (the most recent one said "supported") may not be reent enough for someone's needs.

But: I succeded to install ie7 with no particular hassle using playonlinux (it's in the repos). I'd go for that.

BTW, I wasn't succesful at installing ie8 following some [instructions]("http://wine-review.blogspot.com/2009/11/internet-explorer-8-on-linux-with-wine.html") out there. If you have better luck, please let us know ;)
