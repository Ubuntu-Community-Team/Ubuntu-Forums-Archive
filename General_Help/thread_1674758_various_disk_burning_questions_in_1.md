### Post by HappyLinux on 2011-01-30
I've decided to stick with Windows Live Movie Maker. It has basic controls to create various menus and automatically assigns chapters (to a certain extent).

I've created 3+1 DL-DVDs so far through WLMM. The +1 DVD is one that got turned into a coaster. Probably because I forgot to set the burn speed from fastest to slowest. Unfortunately WLMM doesn't have much in burn speed selection other than Slowest, Medium and Fastest.

I've learned that no matter what software, burner and disk you're using, that the slower the burn speed you use, the better the chance of the disk not being turned into a coaster.

So in all, I will mark this thread as solved, but it will still be open in case anyone has anything to mention on the subject. Cheers anyway folks.
