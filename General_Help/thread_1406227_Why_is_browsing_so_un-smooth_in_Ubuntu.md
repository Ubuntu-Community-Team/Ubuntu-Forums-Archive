### Post by NightwishFan on 2010-02-13
Please define smooth. Also, Linux is just fine as a desktop OS. I have used it for years.

Ubuntu Firefox has "smooth scrolling" disabled by default. You can find it in Edit -> Preferences -> Advanced(Tab) -> General(Tab) -> Use Smooth Scrolling.

If you mean page loading, I really never see any difference between the two, unless you are using a poorly supported wireless device.
