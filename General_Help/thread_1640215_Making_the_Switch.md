### Post by pizza-is-good on 2010-12-07
Well, welcome to the forums!

As far as your question goes. Is it worth it? Yes. There is a learning curve, but you seem like a techy person, so I think you would have fun learning Ubuntu.

Now, my question to you, is it practical? In short, I will answer no.
Wine is a good program, but it has its limitations, which are many. Chances are you won't be able to play high end games using wine, but you can check for yourself [here]("http://appdb.winehq.org/"). 
As far as XAMPP, there is a tutorial on how to get it running in Ubuntu [here]("http://www.devshed.com/c/a/Administration/How-to-Install-XAMPP-on-Ubuntu-Linux/"). 

Word and Excel are replaceable with Open Office, which comes by default in Ubuntu. I don't know which version of MS Office you have, but Word and Excel are simple enough programs that they will work in Wine. Be sure to check first though.

As a recommendation. You do not have to completely remove Windows to use Ubuntu. You can do what is called a dual boot. This way, you can choose which OS you want to use when you turn on your computer. Myself and many users here do this. It is very useful for when you are learning, and invaluable for when you have to use software that only runs in Windows.

I myself use Windows for some email, photoshop, flight sim, and AOE, and Ubuntu for everything else. It works out. Eventually I found that I like Ubuntu more than Windows and find myself using it more and more. Now, I really need to want to play games for me to use Windows.

Just FYI, I have been looking at xPlane, which has been reviewed as better than MS Flight sim, and it works in Ubuntu.

Hope the comments were helpful,

~pizza
