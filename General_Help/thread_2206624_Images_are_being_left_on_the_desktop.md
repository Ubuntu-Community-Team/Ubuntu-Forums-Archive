### Post by coldraven on 2014-02-20
I don't know your model but it might be a problem with the Compiz window animation effects.
If you have selected "Require login at startup" you could try switching Ubuntu into 2D mode by clicking on the cog icon next to the username input field.
It will remember your choice but you can always switch back to 3D mode if it does not fix the problem.
You can enable/disable "Automatic login" by going to "User Accounts". Open the dash (Winkey + A) and start typing "user"
