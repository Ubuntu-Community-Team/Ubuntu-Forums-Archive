### Post by mb_webguy on 2008-08-27
Try [this one]("http://http://isohunt.com/torrent_details/41131981/kubuntu?tab=summary"), but make sure you check the md5 hash to be sure the file is good.  To check the md5 hash, open the terminal and change to the directory where you downloaded the iso, and type "md5sum <filename>", where <filename> is the name of the iso file you downloaded.  This will provide a string of characters that is the md5 hash.  Make sure it is identical to the one listed on [this page]("https://help.ubuntu.com/community/UbuntuHashes") for the file you downloaded.
