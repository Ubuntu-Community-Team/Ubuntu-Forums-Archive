### Post by Sin@Sin-Sacrifice on 2009-11-25
It doesn't have to be so. I ran into this issue and it was the ram but re-seating the ram actually solved the problem. You have 2 gig? A single chip or more than one? Have you tried booting with only one of the chips? If it still happens with the one try the other. Then if it happens... it might be time to think about an RMA. Try out different configurations. Does your motherboard have more than one dimm? If so... also try each chip in a different dimm to see if it happens. RMAs are a pain especially when you get new hardware, it doesn't work so you have to wait yet again for the system you wanted. So hope you can avoid it.
