### Post by aperomsik on 2011-11-30
The Thunderbird 8 release notes link to this extension:

[https://addons.mozilla.org/en-US/thunderbird/addon/folder-pane-view-switcher/](https://addons.mozilla.org/en-US/thunderbird/addon/folder-pane-view-switcher/)

However, if you don't switch that often, it might be just as easy to use the menu: View / Folders / all, favorite, etc.
