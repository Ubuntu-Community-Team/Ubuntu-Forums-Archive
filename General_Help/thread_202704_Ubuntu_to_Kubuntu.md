### Post by vigleik on 2006-06-23
sudo apt-get install kde
sudo apt-get remove --purge gnome

I've executed the first command several times, not the second one. I would consider the first command safe (as long as it doesn't want to remove programs) while I don't know about the second one.

Unless you're short on space, keeping gnome on your hard drive won't affect you adversely. In fact, it might come in handy if you screw up kde.
