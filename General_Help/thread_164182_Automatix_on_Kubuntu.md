### Post by gingermark on 2006-04-22
Automatix now supposedly works for Kubuntu too.

I personally have had some bad experiences with it, but it seems like a lot of work has gone into it, so you could give it a go.

I prefer just to bite the bullet and set things up manually, and to be honest it doesn't take to long assuming you have a decent internet connection.

Installing synaptic is easy enough, (sudo apt-get install synaptic) and it looks ok once you've set up the GTK syles and fonts.

And [https://wiki.kubuntu.org/RestrictedFormats](https://wiki.kubuntu.org/RestrictedFormats) covers most of what you need as far as media goes.

As far as amaroK goes, it works fine for me. Maybe if it's broken then that's a topic for another thread.
