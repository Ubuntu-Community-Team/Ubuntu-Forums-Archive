### Post by Toz on 2012-03-11
This might also help: [http://blog.tersmitten.nl/archives/1082]("http://blog.tersmitten.nl/archives/1082").

Open a terminal window, and type in the following command:
```
sudo gedit /etc/modprobe.d/alsa-base
```
...type in your password, and add the following to the end of the file that displays:
```
options snd-hda-intel model=auto
```
...save the file and reboot.
