### Post by drs305 on 2010-10-10
If you can access the Grub2 menu during boot (hold down the SHIFT key if it doesn't display) try booting to an older kernel.

If this isn't an option, boot to the LiveCD and we will chroot into your Ubuntu install and update the initrd file to see if that fixes things. From the LiveCD Desktop:

```
sudo mkdir /mnt/temp
sudo mount /dev/sda1 /mnt/temp
for i in /dev /dev/pts /proc /sys; do sudo mount -B $i /mnt/temp$i;  done
sudo chroot /mnt/temp
```

You should now be at a root prompt. Update the initrd image and grub:
```
update-initramfs -u
update-grub
```
Exit *chroot*, unmount.
```

exit
for i in /dev/pts /dev /proc /sys / ; do sudo umount /mnt/temp$i ; done

```
Reboot and see if that corrects things.
