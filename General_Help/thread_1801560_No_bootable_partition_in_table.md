### Post by Jugg on 2011-07-10
I have Ubuntu 10.4. I recently changed my desktop theme and now when I start my computer it does not load Ubuntu and instead says:
```
No bootable partition in table.
```
I can get around this by pressing F12. This takes me to the Boot Device Menu. Then I can boot from Primary Master Drive and it seems to boot into Ubuntu, although some of the icons I had recently moved from before the problem do not show up in the correct places. I would prefer to not have to go through this method every time I turn on my computer however and would like it to boot normally. Any help in accomplishing this would be appreciated.

### Post by drs305 on 2011-07-10
Do you have a USB drive connected that isn't normally, or a card in a slot that you have forgotten about? These are common causes for the message.

You could enter BIOS during startup and make sure it's attempting to boot from the proper device.
