### Post by gotix on 2008-11-07
Is it possible to disable some graphics effects (for example windows fading, growing when restoring, etc) in order for the interface to run faster?

---

### Post by ajgreeny on 2008-11-07
*System->Preferences->Appearance* or if you have **compizconfig-setting-manager** installed you can deal with each plugin separately with *System*->*Preferences*->*Advanced Desktop Effects*.
