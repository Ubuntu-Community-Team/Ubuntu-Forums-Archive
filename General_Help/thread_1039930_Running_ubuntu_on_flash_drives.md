### Post by blackened on 2009-01-15
The first problem will be whether or not the BIOS is even able to boot from USB. On a machine that old I would venture a guess and say it won't.

The second problem is that, if it *is* able to boot from USB, then I hope you have a USB 2.0 pci add-on card, otherwise running (or at least booting from) USB 1.1 is going to be pretty slow.

If none of what I've just stated applies, then yes, 8 Gb should be plenty as long as you don't store tons of extra data. For example, I have about 30 Gb alotted to my 8.10 partition, of which I'm only using 5 Gb and this includes two 900 Mb isos in my user directory plus countless updates (been using 8.10 since early betas).
