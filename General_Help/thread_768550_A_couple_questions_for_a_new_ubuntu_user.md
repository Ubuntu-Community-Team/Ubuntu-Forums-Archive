### Post by crane on 2008-04-26
1. Look in your root folder for a file called Host. /host

2. Use synaptic to install compiz (if your video card supports it) also install compizconfig-settings-manager so you can tweek it.

3. how do you mean skin? go to System>Preferences>Appearance Maybe that is what your looking for.

4.System>Administration>Printing  you should be able to add new printer there. You should also be able to just browse to locate when your adding it there.

5. I am not to sure on. Does the system you are trying to access have the same user name and password as the Ubuntu system? Maybe go to synaptic and see if samba is installed. but I would do I little more searching on this as I believe it is a pretty common error.
