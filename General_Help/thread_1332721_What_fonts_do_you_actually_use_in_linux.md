### Post by fragos on 2009-11-21
You can delete fonts with Synaptic. Search for "ttf" and you'll get them. There are a number you can uninstall. If you shouldn't delete a particular font you'll get a warning. As to what I use on the desktop, it's the entire DejaVu family. I like the condensed versions very much. If I'm not mistaken DejaVu is based on Bitstream.
