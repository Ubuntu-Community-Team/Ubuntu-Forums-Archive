### Post by oldos2er on 2014-04-12
It doesn't help with Ubuntu at all, it allows you to install and run a limited amount of Windows applications. See [https://help.ubuntu.com/community/Wine](https://help.ubuntu.com/community/Wine)

You should check [http://appdb.winehq.org/](http://appdb.winehq.org/) first to see if it's possible to run the Windows program you want. Anything rated less than "gold" probably isn't worth bothering with.

### Post by meistersreign on 2014-04-12
I'm not sure as to why or for what reason you were suggested to use WINE, but it is mostly used  for running Windows games that are not available native for Linux. It can also run Windows software for the same reason. As suggested, [http://appdb.winehq.org/](http://appdb.winehq.org/) is your friend in this situation, and search for whatever it is you are trying to run. Most pages will have detailed explanations on how to get it running properly.
