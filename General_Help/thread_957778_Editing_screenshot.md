### Post by Gallo_Pinto on 2008-10-24
A program called "the GIMP" is the most popular for that type of thing. You may have it installed already in your applications menu under "graphics". You can install or upgrade it using "add/remove programs". You'll find it quite easy to figure out for basic stuff like resizing, saving in a specific format, but you can also find lots of help online. [http://www.gimptalk.com](http://www.gimptalk.com) is a pretty good site.

Because the GIMP uses several windows, you'll find it easiest to manage if you open it up on a separate desktop (access them with the four little numbered squares on the taskbar, bottom right).

### Post by scandido on 2008-10-24
If you are comfortable using the command line, an even simpler strategy would be to open it and type:

convert Screenshot.png Screenshot.jpg

This will convert your original PNG into a JPEG file. However, this precludes any editing. 

Also, in Gimp there is a screenshot option which allows you to capture only a single window. This might be useful.
