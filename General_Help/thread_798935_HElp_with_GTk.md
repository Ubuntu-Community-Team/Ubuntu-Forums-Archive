### Post by sistarbliss on 2008-05-18
Go to system>preferences>appearance>theme tab


Drag your tarball to that window and the gtk theme should install.  
(you can drag icon sets to the theme tab too)

You can further tweak things out by going to the customize button, there you will find controls, colors, window border, etc.  Once you get it all tweaked out the way you want, go back to the theme tab, and where it says "custom"....save as whatever you want to call your theme.  You can apply specific backgrounds with it too.  This will save the whole shebang, that way you don't have to go in and manually tweak things every time.  

There is an option to reset themes back to default colors too...that way if a skin you were using in one color gets altered you can turn it back to the way it was when you originally installed it.  

Have fun!  :)
