### Post by mike2357 on 2008-12-19
There seems to be an incompatibility between compiz (i.e. Visual Effects), Openoffice menus and older Nvidia drivers.  I think you have two choices:

1.  Change Visual Effects to None.  (System -> Preferences -> Appearance -> Visual Effects -> None).

2.  Change one of your Font settings.  (System -> Preferences -> Appearance -> Fonts -> Subpixel smoothing (LCDs) )

I've been using Option #2.
