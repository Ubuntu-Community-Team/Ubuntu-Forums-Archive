### Post by The Real Dave on 2010-04-30
Ubuntu will work fine with 1GB of RAM, though, depending on what music editing you'll be doing, you may need more, I do not know exactly how much. 1GB is more than plenty for average use, browsing the web, office etc.

You do not need to be able to code to use Linux. You do not even need to be able to use the command line (though that's where the real power is). Ubuntu is much simpilier to set up and use then Windows, but just remember that there will be differences. You can install just about any program you need through the Software Centre, and it'll all be kept up to date automatically.

Quite a few Windows apps will run in Ubuntu through a program called Wine, though, I don't know if that specific program does. I'm currently running Office 07 in Wine for example.

I'd advise that you dual boot, considering that you have certain Windows apps, or hardware that may be Windows only. In that case, you can have the speed and security of Ubuntu, and easily switch to Windows to do what you need to do.

EDIT: It would appear that the latest version of Ableton works in Wine
