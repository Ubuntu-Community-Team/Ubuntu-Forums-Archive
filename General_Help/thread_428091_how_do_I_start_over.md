### Post by H.E. Pennypacker on 2007-05-01
> **swatF1RESTORM said:**
> I'm not sure how to do that? I know that in windows there was a screen that flashed that said "Hit F2"  or something like that but I didn't see that when restarting Ubuntu. I was able to get to the GRUB but not my BIOS.

The BIOS has nothing to do with any operating system.  When you start your computer, the BIOS can be started using whatever keys that show up on the screen.  Start your computer, and press those keys.

If you are not entirely sure, try F2.  It is impossible to not be able to get to the BIOS.

I don't see how you've been able to install Ubuntu at least once, and not be able to install it again.  It works the exact same way as the first time.  Nothing changed.  You still have to insert the CD, start up the LiveCD, and install Ubuntu.

If you are able to get to GRUB, that means you've passed the point of entering BIOS options.  You'll be able to enter the BIOS options before that point.  It should be a dark screen that tells you the keys.
