### Post by daweefolk on 2009-11-10
My fiance and I both use my computer for games, and I'm wondering if there is any way to set it up with another monitor, mouse, and keyboard, and have one monitor display desktop 1 and the other display desktop 2. Then have each keyboard/mouse pair work with one of the two desktops. Like keyboard and mouse A work only on desktop 1 and Keyboard and mouse B on desktop 2. Like so we could have two different games running from the same computer and we could play them both at the same time on different monitors

### Post by fluffman86 on 2009-11-10
You'll probably need something like a [Reverse KVM](http://www.google.com/search?sourceid=chrome&ie=UTF-8&q=reverse+kvm) to handle the hardware implementation.

After that, you could set up multiple accounts and run multiple X sessions on Ubuntu at once, and each keyboard would control 1 X session.

More on KVMs: [http://en.wikipedia.org/wiki/KVM_switch](http://en.wikipedia.org/wiki/KVM_switch)
