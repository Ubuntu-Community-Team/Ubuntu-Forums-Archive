### Post by ankspo71 on 2009-12-25
Hi,
The only way I know how is to create a new layer (preferably transparent).
Then use the 'rectangular select tool' to select a straight line shape on the new layer.
Then paint in your color. (don't deselect the line yet)
Then use the 'rotate tool' to rotate the line you created. (it will allow you to choose a specific angle too)
Then you can anchor the layer when needed.

Hope this helps.
