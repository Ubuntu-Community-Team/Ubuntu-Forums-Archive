Looks to me like whoever made that is using a combination of desktop opacity (or transparency) settings and the 3D Windows plugin.

In **Compiz Config Settings Manager**, first go to the *Desktop Cube* plugin, and from there to the *Transparent Cube* tab. Set the *Opacity During Rotation* to 0. (zero - slide it all the way to the left.)

Second, enable the *3D Windows plugin*, and use whatever settings for* Window Depth* that you find appealing.

I use the *Cylinder* option in *Cube Reflection and Deformation* plugin, but I switched to a cube to show the effect that you're after. I kind of like the look and just might leave it that way for a while.
