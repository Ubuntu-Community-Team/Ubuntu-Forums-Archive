### Post by CantankRus on 2016-11-28
Because of a [**[COLOR="#B22222"]bug[/COLOR]**]("http://crbug.com/660005"), chromium based browsers need to access the login.keyring to install a dummy entry.
If you use autologin the login.keyring is not unlocked when you login to your session.

To get rid of the "Unlock Keyring" popup you can turn off autologin or change the login.keyring so as not to use a password.
See [[COLOR="#B22222"]**HERE**[/COLOR]]("http://askubuntu.com/questions/867/how-can-i-stop-being-prompted-to-unlock-the-default-keyring-on-boot") to set the login.keyring password to blank. ****Note the warnings****
