### Post by happyhamster on 2010-10-20
How old is that computer? Anyway, try booting the livecd with the "noapic" parameter. On the default livecd you can set that option by pressing F6 from the boot menu (directly after choosing the language). If you don't even get that far, try booting from the alternate cd (it will have a text-mode installer).

[http://www.ubuntu.com/desktop/get-ubuntu/alternative-download](http://www.ubuntu.com/desktop/get-ubuntu/alternative-download)

Good luck.
