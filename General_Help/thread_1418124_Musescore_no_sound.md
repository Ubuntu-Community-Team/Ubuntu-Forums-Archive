### Post by howlingmadhowie on 2010-03-08
musescore connects to a running jack server. the best way to do this is to install the qjackctl package and start the server over the item in the menu. (applications->sound and video->jack control).  

on systems without a realtime kernel you will have to change the preferences before jack starts. Click on settings and remove the tick mark next to "real time".
