### Post by Ambimom on 2006-12-31
Azureus
1. File > New Torrent (or CTRL + N)
2. Tick &#8220;use an external tracker&#8221;.
And again, This is probably the hard part for most people. But it&#8217;s pretty easy, just put in one of the popular public trackers. Here are some of the most popular trackers at the moment:
[http://tracker.prq.to/announce](http://tracker.prq.to/announce)
[http://inferno.demonoid.com:3389/announce](http://inferno.demonoid.com:3389/announce)
[http://tracker.bt-chat.com/announce](http://tracker.bt-chat.com/announce)
[http://tracker.zerotracker.com:2710/announce](http://tracker.zerotracker.com:2710/announce)
Put one of these in the tracker box 
3. Select single file or dicectory, click NEXT and point to the file or directory you want to share, and click NEXT
4. Do NOT tick &#8220;private torrent&#8221;
5. Do tick &#8220;allow decentralized tracking&#8221;
6. Save the torrent and send it to your friends
