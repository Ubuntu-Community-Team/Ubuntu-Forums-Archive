### Post by geirha on 2009-08-30
I believe you need to choose manual partitioning to be able to use ext4. Do note however, that there is a bug regarding ext4 in Jaunty that hasn't been resolved yet, so you may want to use ext3 for the time being. [http://www.ubuntu.com/getubuntu/releasenotes/904#Lock-ups when deleting files from ext4 filesystems](http://www.ubuntu.com/getubuntu/releasenotes/904#Lock-ups when deleting files from ext4 filesystems)
