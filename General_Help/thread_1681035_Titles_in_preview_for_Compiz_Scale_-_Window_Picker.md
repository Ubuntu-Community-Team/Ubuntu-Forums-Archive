### Post by Umang on 2011-05-12
On Lucid, there's a "Scale Addons" in my CompizConfig Settings Manager. Under appearance, I've been able to edit the "Window Title Display" field and set it to "All Windows". I had to tweak it to make it look nice, but it works! :)

PS: I don't know which version of Ubuntu you are using or the version in which this was introduced.
