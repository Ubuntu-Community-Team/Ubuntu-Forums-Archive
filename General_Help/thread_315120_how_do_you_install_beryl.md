### Post by groggyboy on 2006-12-08
[The Ubuntu Beryl Wiki]("https://help.ubuntu.com/community/BerylOnEdgy")
[A 3rd Party Blog for Ubuntu Gnome]("http://lhansen.blogspot.com/2006/10/3d-desktop-beryl-and-xgl-on-ubuntu-edgy.html") - this one has pictures, and may be easier to follow, but it is written with an ATI graphics card in mind.

I know this isn't exactly what you were asking for, but with so many excellent guides out there on installing Beryl, why reinvent the wheel? ;) 

Feel free to ask back here if you have any issues.

A good place to start is by telling us what video card you have, and if it can support 3d rendering (typing "glxinfo | grep rendering" in a terminal will answer that last one).

### Post by redDEADresolve on 2006-12-08
1. Before you do anything you should get used to Ubuntu first. If you don't know what you are doing you probably shouldn't be doing it.

2. You have to have the right driver installed for you video card. Do you have an ATI, Nividea or Intel graphic card. Intel should work out of box but the other two require a little work to get the Beryl ready.

3. Make sure you card/machine is capable of running Beryl.
