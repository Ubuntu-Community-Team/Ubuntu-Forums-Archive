### Post by murderslastcrow on 2009-09-12
But hey, doesn't mean you can't play around with it with the Windows installer for Ubuntu- Wubi. [http://wubi-installer.org/](http://wubi-installer.org/)

It will allow you to install Ubuntu like a Windows program to try it out on a virtual hard drive in your system. It will somewhat reduce the speed since it'll be running from a Windows file system, but it's a good way to see if you like it.

And, if you do, you can easily dual-boot and enjoy the security, speed, stability, and the other benefits and features of Linux without having to sacrifice Windows (if you have enough space on your hard drive for this). Just make sure you uninstall Wubi from Windows first.

I hope that Adobe will release their software for Linux someday. :\ There are many people who would use Premiere and After Affects quite often, I'm sure.
