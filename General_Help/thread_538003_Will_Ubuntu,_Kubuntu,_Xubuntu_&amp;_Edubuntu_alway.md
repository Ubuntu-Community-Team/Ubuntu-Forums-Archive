### Post by ddrichardson on 2007-08-29
<tinfoilhat>

Microsoft needs people to upgrade to bigger and more powerful machines. It maintains their high volume OEM sales. With all good symbiotic relationships, OEMs like more hungry versions of Windows because people buy more hardware.

</tinfoilhat>

Ubuntu, like most Linux distributions is driven to support the latest hardware but is not driven by it. So it does tend to have a longer life cycle compared to Windows.

If you go back a few years, then comparing say Fedora to RedHat 5.1 it seems bloated and power hungry.
