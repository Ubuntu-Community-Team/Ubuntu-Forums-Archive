Ubuntu always looks like it is running most of your ram but it isn't, it caches it.  Active RAM usage is usually much lower.  Also, using more RAM is a GOOD thing, RAM is far faster than a hard drive, so swapping stuff out to the hard drive is inefficient if there is enough RAM to do the job.

512MB is more than enough.

### Post by fragos on 2007-07-15
512 should work but a little more may improve system performance.  I found that with 1024 I rarely ever use any swap space on disk.  Performance improvements are particularly noticeable when you change applications.  Run "free" in a terminal to see if you're using swap space.
