### Post by fabounet on 2013-06-25
You are talking about some alternative menus (GnoMenu Cardapio, etc).
You need to install them before you can use them in the dock.

However, from Cairo-Dock 3.3, the Applications Menu has a text entry where you can quickly search for an application by name or description :-)
