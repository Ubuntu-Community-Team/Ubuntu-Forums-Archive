The situation you describe seems to be caused by a processor which has too much to do. The active program does not get all the attention it needs from the CPU and therefore the screen becomes dark. I sometimes have the same, just for a second or so. Try opening a terminal next to your Thunderbird window and type top. This way you have both programs on screen and can see which process is eating the CPU when the screen goes dark.
