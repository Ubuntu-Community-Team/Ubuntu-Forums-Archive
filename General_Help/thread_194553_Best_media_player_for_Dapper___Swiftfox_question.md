### Post by blackjack6.21.91 on 2006-06-14
To make an icon for swiftfox, right click your panel, click add to panel and then custom application launcher.  For name and generic name just put "Swiftfox" and for command put "~/swiftfox/firefox".  That should do the trick.  Then you should be able to just remove the old firefox icon if that's what you'd like.
