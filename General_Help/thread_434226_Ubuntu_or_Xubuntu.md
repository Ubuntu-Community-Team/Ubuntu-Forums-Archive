### Post by afoglia on 2007-05-06
> **mattva01 said:**
> Quick question, is there any way to add a drag select to Xfce?

I assume you mean the desktop, since Thunar (Xfce's default file manager) already has drag-select.  I don't think so.  But you can use Nautilus to replace Xfce's default desktop, xfdesktop.  Usually Nautilus will take over the desktop when you first run it.  Just do that and save your session setup when you log out.  (If so, you may also want to replace thunar with Nautilus for consistency.)
