### Post by advoss on 2006-12-09
If the files are on the Windows XP partition you can follow the instructions in the Unofficial Ubuntu guide: [ here ]("http://ubuntuguide.org/wiki/Ubuntu_Edgy#How_to_mount.2Funmount_Windows_partitions_.28NTFS.29_manually.2C_and_allow_all_users_to_read_only")
Then you will be able to copy them over (I assumed windows XP was installed on an NTFS drive, which is default)

If the music is only on the USB drive, then you can install drivers in windows which will let you put the music on the Linux drive from within windows: see [here]("http://www.fs-driver.org/index.html")
