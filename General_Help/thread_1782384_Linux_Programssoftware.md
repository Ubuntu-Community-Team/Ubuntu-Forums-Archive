### Post by inameiname on 2011-06-15
In Ubuntu you have several options on how to install packages (ie, programs/games/whatever), but all options install the programs the same way, through code. The choice is mainly who puts the code into the terminal, you manually, or a graphic gui program. Ubuntu Software Center is a good one. Windows does this too; its just nearly everything is in a graphic gui program, not a terminal.

Now, there are also options as to where to get programs/games/whatever. You can do like you did with Windows, and search the internet for them and manually download and install an installation program for them (debs, which are like exes for Windows). OR, unlike Windows, you can install the programs (debs) through the repositories, which are just websites housing several of the programs alltogether for easy access. Anyway, using for instance, Ubuntu Software Center, found in the Main Menu (ie, Start Menu-like), you are able to visually see what is inside the available repositories you have, you can simply search for a program on it and install it. 

Once installed, you can run the program several ways. Again, it all runs with a terminal command; its just who inputs it, you or a graphic gui, such as the Main Menu shortcuts (just like those found in Windows Start Menu), icons on your desktop, or using the 'Run' command for Linux, similar to the one for Windows (just press <ALT>+<F2>.

Overall, the whole concept of the terminal and writing commands is something new for people not used to it who've migrated from Windows. Once you are able to understand its really no different than in Windows, just an extra option to access/download/install/run/delete/remove programs and files/folders which isn't there in Windows (well, to a smaller degree it is) you might even find it useful. Using the Main Menu, desktop, and 'Run' box you can do all that you could in Windows to install and run programs.
