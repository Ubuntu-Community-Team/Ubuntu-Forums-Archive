### Post by bscbrit on 2006-04-22
You will probably find the linux is not configured to read the Alt+Gr combination and is simply ignoring anything that is associated with it.  The Alt+Gr might have default settings under Windows but the keys have no special significance under linux.  If you search the forums (perhaps for 'configuring keyboard' or something similar) you might find the information that you need to make the keyboard do anything that you want it to do. :-D
