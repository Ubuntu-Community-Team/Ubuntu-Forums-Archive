### Post by iheartme on 2007-05-15
On my 80Gb drive, I have a 8Gb / partition and the rest for /home.

 / have about 2.4 gig used with all my preferred applications installed, so 8gig might a bit much but better to be on the safe side.

Simple but effective, no need to worry about personal data if an upgrade or something else goes awry.

And if you're interested, I have /home mounted as ext4 with extents and delalloc and it's rock solid and, for me, as fast or faster than reiser4.
