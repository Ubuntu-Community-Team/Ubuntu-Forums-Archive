### Post by Thirsty Ferret on 2008-05-16
Desktop effects can be enabled by choosing from the options available in System > Preferences > Appearance menu item.

If you want a bit more control, you can install the Compiz Config Settings Manager with the following command: sudo apt-get install compizconfig-settings-manager

This program is then accessible from either running "ccsm" from terminal, or (for me at least) it appears in System > Preferences > Advanced Desktop Effects Settings.

From this program you can enable / disable particular plugins. I'm afraid you may have to hunt around as I'm not sure what you're after, but you may want to try Desktop Cube (that's 3D) and the Put plugin. If you can be more specific for the effect you're looking for, its more likely we'll help you find the plugin :)
