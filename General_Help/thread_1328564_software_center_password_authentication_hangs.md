### Post by fluffman86 on 2009-11-16
Sometimes authentication dialogs hang on me if I have Visual Effects enabled.  If you've manually installed compiz, disable it from the compiz settings dialog.  If you haven't, go to System > Preferences > Appearance, click the Visual Effects tab at the top, and select "None."

Your screen may flicker, but it should be back shortly.  Try running Software Center again and report back.
