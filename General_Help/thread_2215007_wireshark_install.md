### Post by Lars Noodén on 2014-04-04
Applications will be found in the repository and installed with apt-get, Synaptic or the Software Center.  

[https://help.ubuntu.com/community/InstallingSoftware](https://help.ubuntu.com/community/InstallingSoftware)

However, Wireshark is a graphical program and the desktop version of 10.04 (Lucid Lynx) reached end of life back in May of last year.  If you are thinking of the server version of 10.04 then there is tcpdump which is text based and that should be available still since the server edition is good until 2015.  Are you using 10.04 on the desktop still?  If so, time to consider converting to a supported version.
