### Post by hwttdz on 2012-03-02
Sounds like you
1) have a windows machine
2) would like to remove windows and install ubuntu

Yes, you can do this.  
1) Back up all your data to external media (external hard drive, cloud drive accessible from both windows and ubuntu).  
2) Download and burn ubuntu iso to a dvd
3) put the disk in reboot and follow the on screen instructions.

I'd advocate during setup having a separate /home and / partition (in addition to any others you might want).
