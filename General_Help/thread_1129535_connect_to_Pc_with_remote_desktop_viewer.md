### Post by toupeiro on 2009-04-19
> **JakeHinojosa said:**
> i have been able to connect to all my computers with remote desktop viewer exept for my PC. how do i set it up so i can use it in remote desktop viewer? its running windows vista, the other computers i WERE able to connect to where macbooks, so am i able to connect to my PC even though its using a modem and not wireless? if so, how?

First, please note that I've only used Vista in Enterprise environments with different security policies applied, so this information may not apply to your machine.

You could try adding the account you are connecting with to the remote users group.  This can be done by right-clicking My computer and going to properties, then setting it up in the remote tab.

Secondly, use the "Terminal Server Client" as opposed to the remote desktop viewer.  You may have better luck with MS based remote machines.
