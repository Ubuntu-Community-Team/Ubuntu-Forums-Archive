### Post by tuxcantfly on 2007-07-20
Download this via some other computer's broadband connection:

[http://releases.ubuntu.com/feisty/ubuntu-7.04-alternate-i386.iso](http://releases.ubuntu.com/feisty/ubuntu-7.04-alternate-i386.iso)

Put it on a USB flash drive, and place the iso file in the same directory as the Wubi .exe file is in, and it will detect and use it automatically.
