### Post by OpSecShellshock on 2009-12-10
Hey there, folks. Not sure if anyone's posted this yet, but as a new version of Flash was released yesterday that addresses some of those oh-so-mysterious "unspecified" critical vulnerabilities, it might be of interest to some of you to update your plugin.

I was able to do that by taking the installation script in [this thread]("http://http://ubuntuforums.org/showthread.php?t=1259102"), opening it in a text editor, and changing all instances of "libflashplayer-10.0.32.18.linux-x86_64.so.tar.gz" with "libflashplayer-10.0.4234.linux-x86_64.so.tar.gz" and then saving the file and running the script in terminal as specified in the instructions.

It's worked fine on youtube. I haven't really checked it anywhere else. Keep a copy of the old script just in case you need to revert.
