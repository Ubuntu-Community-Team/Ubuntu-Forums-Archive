### Post by sdowney717 on 2011-11-14
solved it.
Solution was crop the image out of the scanned page into a new image which takes up less space on the edges.
Apparently the printer has borders, so it scales if it cant fit the whole image. Since the scanner scanned a full 8.5 by 11 page of paper, it wont fit when it prints.

Why it does this? Think about it, the scanned page is an image not text. Still think it should be able to ignore the border somehow. BUT did not see any options for that when in the print dialog.
So then an 8.5 by 11 scanned page then is not reproducible to the same size when printed.
