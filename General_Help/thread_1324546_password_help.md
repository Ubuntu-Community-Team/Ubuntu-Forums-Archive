### Post by scragar on 2009-11-12
Once you are logged in it's easiest to change the password by running:
```
passwd
```in a terminal(applications>accessories>terminal).

It's the easiest way to get feedback on any error should it fail for any reason.

### Post by hwttdz on 2009-11-12
