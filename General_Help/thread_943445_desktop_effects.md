### Post by loomsen on 2008-10-11
As the whole compiz API is based on plugins, that happens from time to time.
Some plugins get outsourced and get their own trunk, others get integrated into yet others. 

Not much you can do about as I never tried tbh, but I don't think exporting your profile as gconf-backend will solve this problem, tho it might. Actually thats why this option is given at all I think.

Anyway, if exporting your profile to back it up, make sure you DON'T skip default values when prompted.
