### Post by Potatoj316 on 2008-04-23
This really isnt answering your question but it might solve your problem and it works great for me.  

If you go to System -> Preferences -> Keyboard -> Layout Options (tab) you can set a keyboard led to show your alternate config.  I set the scrolllock led to show when on the other layout because scrolllock is never used anyway.
