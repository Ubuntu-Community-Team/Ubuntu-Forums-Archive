Hi all, 

I've been working on a customized live CD based on the Ubuntu 5.10 (Breezy) live CD for x86. I got sick of typing the same commands over and over, so I put together a couple of scripts that make it pretty easy to make your own CD, with custom graphics and installed programs. 

The download is a little less than 4.5Mb. You will also need a Ubuntu live CD iso file and a couple programs installed like cloop-utils and some graphics programs for it to work. 

You can get it and read more about it here: [http://stuporglue.com/content/view/29/35/](http://stuporglue.com/content/view/29/35/)
