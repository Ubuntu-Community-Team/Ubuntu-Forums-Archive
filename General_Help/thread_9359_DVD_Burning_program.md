### Post by randy on 2004-12-28
I just use growisofs from the command line.  It's not exactly the easiest program to use but it's generally faster than the gui alternatives once you get up the learning curve.  If your looking for dvd movies install wine and use dvd shrink to make the isos, then burn them.
