This problem occurs even if I boot up Ubuntu with the KVM switched to this machine - works until I switch over the XP machine, and try to switch back.

### Post by googlebytes on 2007-05-02
For those who may run into this - the switch is a D-Link KVM - standard mouse. I found I have to boot up my Ubuntu machine with the switch set to the XP machine. Once the Ubuntu machine gets to the sign in screen, I switch over to it, and sign in - then all is fine - the switch works perfectly.
