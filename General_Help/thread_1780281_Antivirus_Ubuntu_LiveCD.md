Make a "persistent" live Ubuntu USB Flash Drive.  That way it can remember any changes when you boot your computer from it.  Then you can install any antivirus application you want on it, to use later on on whatever computer you boot that USB live stick of Ubuntu from...  Here is where you can find some good antivirus apps to install on it as well:

[http://cinderbox.net/2011/04/03/to-do-list-after-installing-ubuntu-11-04-aka-natty-narwhal/](http://cinderbox.net/2011/04/03/to-do-list-after-installing-ubuntu-11-04-aka-natty-narwhal/)

To make a persistant live USB bootable flash drive of Ubuntu.... download an ISO of ubuntu and use Startup Disk Creator.
