### Post by TheFu on 2014-11-11
Connecting a PC to a TV depends on the TV and the computer working together.
First, you need to decide which connector to use between them. There are a few options. 
* VGA
* DMI
* HDMI
* DisplayPort (*)
* S-video
* RCA-video
For laptops, HDMI and VGA are the most common. For desktops, any of those are possible, though mine only have VGA, DMI, and HDMI. If you want to know more about any of them, wikipedia is there.
I think only HDMI supports audio too. All the other cables need a separate audio connection. 

I don't know anything about displayport.

Colors on cables mean nothing most of the time. The connector type needs to be known, period. Wikipedia will have diagrams and photos for each connector type.

After all that, if you want to use a different display output, you'll need to tell ubuntu to do so - often it doesn't automatically enable a connection.  xrandr, arandr, lxrandr are the tools I use for this, but I don't run stock ubuntu with unity.
