### Post by WinterWeaver on 2007-07-29
Are you running Compiz Fusion??

This happens on some pc's running compiz fusion. If this is the case, I suggest switching back to Metacity when doing any system updates etc.

```
metacity --replace
```

After doing all the things you wanted to, switch back to compiz.

Hope it helps.

WW
