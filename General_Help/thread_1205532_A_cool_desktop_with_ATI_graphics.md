### Post by imewel on 2009-07-06
Hello everyone,

The bottom line is that I would like to know if with my ATI graphics I can have full desktop effects and an Apple-like desktop including something like Avant navigator. Before asking this question in a different way I'd like to give some background and feedback.

Recently I got a HP Pavilion DV-5 1140ed laptop (marketed as such but at the bottom it says DV-6). It has a AMD Turion(tm) X2 Dual-Core Mobile RM-72 processor, and an ATI Mobility Radeon HD 3400 series graphics card. 

Sadly, Windows Vista was included in the price, as well as some rather annoying HP and other software. Like many others, I had all sorts of problems. After a lot of trouble shooting, I still have unsolvable issues that make the use of Vista a source of annoyance.

So, I thought this would be a good opportunity to try Linux, and for some reason Ubuntu seemed to be the best pick of the choice. Basic installation was a breeze, even with a dual boot next to Vista. My first impression was that I needed to tune all sorts of settings to make myself at home in the desktop, but I've always done that with Windows too. So far, so good.

Unfortunately I soon discovered that I had problems with sound and video. It turns out that my laptop is not the easiest hardware to install Ubuntu on, so perhaps Ubuntu is not to blame. On the other hand, I come from the plug & play world and left this sort of issues behind me when I stopped using Windows 98.  Ubuntu is free, so please take is as feedback, not criticism. I would like to stress this, because as unhappy as i am at the moment, I do not want to appear ungrateful. Still, if I express my research time in hourly rates I paid dearly for trying gratis software. Also, like with Vista, so far I cannot fully use normal functionality without crashing the computer.

I solved the sound issues, and tried to solve the graphics issues as well. Unfortunately I forgot exactly how I did this, and lost the browser session with the pages I used. At the moment the status is that as soon as I play some video and anything is changed in the window in which the video is played, the computer hangs. This is a marginal improvement over the black crash screen I got before disabling desktop effects.

As said, I forgot what I did exactly, but I remember I had to use a proprietary driver for my graphics card. I also remember I had to install compiz in order to use Avant Window Navigator. 

I would like to create a desktop which has nice effects and is a functional mix between Ubuntu, Windows and Apple - aside from Evolution mail this is one of the main Linux attractors for me. Perhaps that is impossible with my graphics card on Linux, in which case I think I will just go with the boring but familiar and stable alternative of Windows XP.

If as such it should be possible to have fully functional graphics in Ubuntu on this laptop then it would be great to get pointers to solutions which a newbie can understand and implement (I'd also pay for someone to solve it if the cost would be roughly comparable to an OS licence fee).

If you need any further information from me I would be happy to give it if I can find it.

Thanks in advance for any suggestions and help! And please do not let my discouraged mood discourage your response ;)

I'm not a big fan if HP, especially because of all the unecesary bloatware.
I use a Dell laptop, and hadn't had many hardware issues, so I can't help you there.
 
Regarding the 'cool desktop'.
If you have compiz, do you have the Compiz Fusion Icon? If not, install it. (It's under Add/Remove Programs).
After that, go to System, Preferences, Startup Applications, and add /usr/bin/fusion-icon
Then load the fusion icon, and set your preferences. You can seriously go wild with your desktop if you have good hardare.

How many drivers are shown for your graphics card?
If more than 1, like I have, you need to have the one with the highest version #, and it should say "[recommended]"
How much memory does your Graphics card have? and how old is it?

Now, you said that your system crashed every-time you played a video. What videos are you trying to play, and what video player are you using?
Also, if it gives you any type of error, could you let me know what they say?

Also, if you install the Compiz Fusion Icon, you can change your window manager, and try metacity, and try to play videos there, if it still happens, you'll know that its not your Window Manager, and if it doesn't, then you'll know that it is Compiz thats causing the crashes, and we should move to fixing Compiz. (After you install Fusion, you need to go to right click on it, Select Window Manager >> Metacity, then right click on it again, and Reload the Window Manager)

Try all that, and let me know what happens.
