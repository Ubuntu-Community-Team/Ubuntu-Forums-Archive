### Post by #&amp;thj^% on 2017-11-27
Please Note: Compacting it an important maintenance task. Despite its poor choice of naming, it removes messages that have been deleted. When you select a message and hit delete it only marks for deletion and hides it from view. Compacting removes it and frees the disk space. 
It has nothing to do with compressing file size despite its poorly chosen name.
See this for more Information on maintenance: [http://kb.mozillazine.org/Keep_it_working_%28Thunderbird%29](http://kb.mozillazine.org/Keep_it_working_%28Thunderbird%29)

Now if you still want to disable that message, Been a while since I last used T-bird but this should get you there.

On the toolbar > Options> Advanced > network and Disk space. and turn off Automatic compact.
