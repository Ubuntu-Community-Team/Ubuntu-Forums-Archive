### Post by bcbc on 2011-10-17
Wubi is purposefully hiding Windows from the grub menu due to the problem some people have: they set Ubuntu as the default in the windows boot manager, thinking they can boot Windows from Wubi's grub menu, but grub just transfers control back to the windows boot manager, and straight back to grub. Hence, no ability to boot windows.

Which makes me wonder... how is it that you can boot windows now?
