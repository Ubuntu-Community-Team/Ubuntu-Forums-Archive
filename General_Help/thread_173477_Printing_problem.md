### Post by Zdravko on 2006-05-10
Hi, I have a 500KB jpg image, which I want to print on A4. Unfortunately I get an error message from the printer, saying that memory is not sufficient. Is there a way to reduce the size of the image, so that the printer can handle it?

---

### Post by PhrankDaChickenGeek on 2006-05-10
You can resize JPG files by opening them in gThumb: "Applications -> Graphics -> gThumb image Viewer"

Select the image you want to resize, and select the 'Tools -> Scale Images' menu item.
