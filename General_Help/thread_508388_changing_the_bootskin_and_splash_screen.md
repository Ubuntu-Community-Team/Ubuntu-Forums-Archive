### Post by testube_babies on 2007-07-24
Bootskin (good luck making one and getting it to work; it's difficult):
[https://help.ubuntu.com/community/USplashCustomizationHowto](https://help.ubuntu.com/community/USplashCustomizationHowto)

Login screen:
System -> Administration -> Login Screen
Download GDM themes from gnome-look.org and drag here.

Splash screen:
```
sudo apt-get install gnome-splashscreen-manager
```
Find a splash screen and set it with gnome-splashscreen-manager.
