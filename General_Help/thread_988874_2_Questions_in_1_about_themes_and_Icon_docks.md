### Post by IceReaver75 on 2008-11-21
to answer your questions

1) emerald is a windows manager.  you have to install emerald from synaptic package manager. just search for emerald.  then go to places like gnome-look.org and download some compiz themes.  they should be .emerald type files.  install them through emerald settings under system-prefences.  

2) there is also cairo dock which can be placed on the top, bottom, or either side of the screen.
