### Post by qpieus on 2008-07-07
Hi and welcome. I can answer a few of these....

> am i fully going to have to convert the entire mp3 collection into aac? or has a clever puppy managed to bypass this?

No need to convert anything. Install a package called "ubuntu-restricted-extras" using the synaptic package manager. That package will install mp3 codecs so you can play the mp3s.

> my music/pictures/samples/movies live partly on the ntfs partitions, and partly on the fat32... can ubuntu use these drives when booted up, or will i effectively have to burn to dvd / usb and recopy over?

ubuntu can read and write from/to fat32 and ntfs partitions so there's no need to burn to external media and copy back. To access these partitions you have to mount them. If they weren't mounted for you during the install process, you'll need to do a little work getting the automounting set up, but once that's done those partitions will appear as just another directory with your music in them.

> 
can i do any of this without an internet connection at home?

The installation cd has quite a bit of software on it. I'm not sure if it has the ubuntu-restricted-extras package though. In the synaptic package manager there should be a menu selection for managing your repositories. In there you should see that the install cd is one of the available sources. If it's not you should be able to enable it to be used by synaptic. Then you should be able to install any software that is on the cd. (sorry for being a little vague here, but I don't use synaptic, so I don't know the exact menu items for you to check through).

Good luck and have fun. I think you'll enjoy using ubuntu. Having such a quality product for free still amazes me!
