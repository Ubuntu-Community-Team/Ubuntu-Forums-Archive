### Post by simosx on 2010-07-06
> **diego_pmc said:**
> I added a Romanian keyboard to the system. When I check the layout I see some keys have four characters associated with them. For example the key highlighted in the screenshot I attached has four characters: &#537;, &#536;,  ;, :. The first two can be used by pressing the key with or without Shift, but how about the other two?


For the other two you press the AltGr key (it's the Right Alt key).
That is the default key (configurable).

So,
1. press key as is
2. press key with Shift
3. press key with AltGr
4. press key with AltGr and Shift
