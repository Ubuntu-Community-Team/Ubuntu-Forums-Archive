### Post by lovinglinux on 2011-07-21
See [Firefox 4, 5 & Beyond Mega Thread]("http://ubuntuforums.org/showthread.php?t=1712247")

They will do what they did when Mozilla retired Firefox 3. Upgrade all supported Ubuntu releases to the latest version of Firefox. BTW, Firefox 3.6 won't last for long. Mozilla is already recommending Firefox 3.6 users to upgrade to FF 5. Firefox 4 reached end-of-life with the release of FF 5 and will no longer be updated independently. Firefox 5 will reach end-of-life on august 16, when they release Firefox 6. 

Essentially, Firefox has become a rolling release with every new major version replacing the previous one. Once Firefox 3.6 is retired, there will be only one stable Firefox branch.
