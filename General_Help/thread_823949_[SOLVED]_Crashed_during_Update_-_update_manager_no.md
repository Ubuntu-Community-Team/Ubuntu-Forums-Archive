### Post by prophet665 on 2008-06-09
Ok...fixed the problem.  I tried to uninstall the update manager through the add/remove panel and it said I should do this through the synaptic package manager.  

I fired that sucker up and noticed it had an option for reinstall.  Reinstalled the two packages and viola!  I am now using the official Firefox 3 release and everything seems to be running smoothly.
