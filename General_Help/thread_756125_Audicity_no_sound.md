### Post by dasunst3r on 2008-04-15
If you are using Hardy Heron, you will need to run Audacity by issuing the command *padsp audacity*.  *padsp* is an OSS wrapper that opens a PulseAudio stream for a program that does not support PulseAudio.  I saw this coming from one year ago...
