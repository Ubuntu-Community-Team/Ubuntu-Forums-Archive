### Post by doas777 on 2009-09-28
looks like the reporter has indicated that it is an upstream bug for the kernel itself.
ubuntu cannot directly route kernel bugs to the kernel team, unless they are tested with the vanilla kernel (not the ubuntu refined one) so look around for advice on testing with it. if the problem persists then it can be sent to the Linux Foundation for the kernel team.

convenient that i just attended a talk on exactly this at linuxfest this last weekend.
