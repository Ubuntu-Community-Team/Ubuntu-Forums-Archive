### Post by ralanyo on 2009-06-28
I use Joomla extensively. It is a great program if you want to build a website. If you want to install it on an Ubuntu server, you will have to install SQL, PHP5 and apache. 

Once installed If you are not sure of how to use the command lines that well in Ubuntu, I would install something like phpmyadmin to administer your SQL Server, and simple control panel to create the dns entries and such.

This site might also help once you get the Joomla application installed.
[Joomla Training]("http://www.joomlatrainingonline.com")
