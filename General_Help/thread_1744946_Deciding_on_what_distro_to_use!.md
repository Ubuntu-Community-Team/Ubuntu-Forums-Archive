### Post by Lateralis on 2011-04-30
The nuts and bolts of Mint are similar to Ubuntu.  If you get on with Ubuntu then you'll be fine with Mint.  But Mint doesn't have a different desktop environment, hence you won't find a mint-desktop in the repos.  Instead, Mint uses Xfce, LXDE, and a handfull of others.  Like Ubuntu, you choose which desktop environment you wish to use.

### Post by 3Miro on 2011-04-30
Xubuntu, Kubuntu and Lubuntu provide Ubuntu with different DE. 

If you learn the DE, it is the same on all distributions.

Other distributions provide different programs, different management tools and mostly different philosophy. The DE you get there, are just the same.

Since X/K/L/Ubuntu are so similar, you can switch between them with apt-get install kubuntu-desktop. To get another distribution (such as Mint), you have to completely reinstall.

### Post by WorMzy on 2011-04-30
Different distros use different package managers and different default desktop environments. There are other differences, such as the system and service manager (e.g. init, systemd, upstart), but these make litte difference to the average user.

Ubuntu and co. (e.g. Kubuntu, Xubuntu, etc) are all members of the same distro family, which is based on Debian, so you'll find each of these relavtively familiar.
