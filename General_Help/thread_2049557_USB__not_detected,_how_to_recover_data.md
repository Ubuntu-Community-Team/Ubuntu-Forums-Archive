### Post by Mark Phelps on 2012-08-29
I have had this happen more times than I care to admit, and the ONLY solution I have found to work reliably was using MS Windows apps to recover the data.  

The best app that has worked for me is RecoveryMyFiles from runtime software.  You can download their trial version from their website, install it on a Windows PC and run it.  You can then see how well it finds the files -- but be SURE to check the filesizes of the files.

I had my smartphone recently trash a micro-SD card, and even though their app found all the files, a LOT of them were zero filesize.

If the files appear to be OK, you can go online and purchase a license, download the key, and activate the product without exiting it.  You can then save off the recovered files to another partition or drive.
