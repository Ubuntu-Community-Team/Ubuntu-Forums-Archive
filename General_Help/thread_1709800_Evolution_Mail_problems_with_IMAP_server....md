### Post by Raybo on 2011-04-13
I don't know what version you are using.  I have gmail imap working fine on 2.28 and 2.32.  My config:
Server type: IMAP+
Server: imap.gmail.com
Username: [email]username@gmail.com[/email]
SSL Encryption
Authentication Type: Password

SMTP
Server Type: SMTP
Server: smtp.gmail.com
SSL Encryption
Authentication Type: Login
Username: [email]username@gmail.com[/email]
