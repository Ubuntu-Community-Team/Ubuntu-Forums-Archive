### Post by The Cog on 2015-04-19
Linux has supported multi-CPU architecture for many years - all of this century I'm sure. But all OSs have to be written for the processor it runs on. 

Windows and mac programs will not run directly on linux (or on each other). But Linux has a compatibility layer called wine which is able to run windows programs (some better than others - it's not perfect yet).
