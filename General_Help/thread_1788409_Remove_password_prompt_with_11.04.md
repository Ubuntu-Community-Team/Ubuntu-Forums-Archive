### Post by An Sanct on 2011-06-22
Welcome to the forums!

The wireless shouldn't ask you for the password every time (assuming, you man WPA/WPA2 password), but for running administrative tasks you should always be asked for the password (or the behavior becomes like Windows XP)

For the wireless, just make sure, that the "available for all users" checkbox (bottom left in the WiFi settings tab) is checked.

### Post by aysiu on 2011-06-22
If you go to the Network Manager applet and edit connections, you can make your preferred wireless network available to all users. You'll have to password authenticate *one time* to authorize that change, but after that, you should never again be prompted for a password to connect to your wireless network.

In terms of installing software, the password authentication is there for security purposes.
