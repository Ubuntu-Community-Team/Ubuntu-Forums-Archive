### Post by gunksta on 2009-07-09
I'm not sure this is the best forum for this question (General Help might elicit more responses) but you could go through and group the programs you want the user to have access in a single group. Then group the other apps into another group. Add the user to the first group and restrict all access to the second.

That should do it, although you will need to deny read access to the .desktop files too or the user will be able to see everything they aren't allowed to use.

There may be a better way. I'm sure this will work, but it will be a fairly involved process.
