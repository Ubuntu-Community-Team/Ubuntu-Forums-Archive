### Post by Giblet5 on 2009-10-26
If only there were a lesson in there somewhere... ;)

Ubuntu 9.10, which is not even released yet, is a beta product.

UbuntuStudio, which is based on Ubuntu 9.10, is slightly MORE beta.

The right thing to do is wait for UbuntuStudio 9.10 to release, and THEN try doing this with the released Live image.

Or, you can mount the Live (.iso) image, fix the install script, unmount it, burn the boot stick, and try again. With the beta².
