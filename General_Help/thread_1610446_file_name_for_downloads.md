### Post by sohlinux on 2010-10-31
Every time I download a file with firefox it names everything export.php whatever the file type be it zip or exe or sql etc and I cannot even get an option to change the name before I save it.

anyone know a fix for this?

---

### Post by bobcollard on 2010-10-31
In Firefox open Edit>Preferences>General Tab: under Downloads "Save File to"  Choose a file in your Home Directory.  You can either change the name as you are saving or afterwards in your saved directory.
