That seems to indicate serious problems with your hard drive - it could be totally unrecoverable.

Listen carefully to the hard drive as it is being accessed, if you hear a repetitive pattern then it will be trying (in vain) to access damaged areas.

Use the Disk Utility from the Live boot to run a SMART test on the drive.
