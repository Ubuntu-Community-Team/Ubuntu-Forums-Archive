### Post by Slingshot on 2008-03-05
```
2D CAD? No 3D?
```

2D only. DXF file support only. Try here [http://www.ribbonsoft.com/qcad_doc_animations.html]("http://www.ribbonsoft.com/qcad_doc_animations.html") for some tutorials.

> Then what do you consider Blender to be? Let's say I want to design something simple like... a computer case. What would be the best tool to use?

For something like a computer case you want a parametric modeler program like Solidworks, Alibre or Pro/Engineer. They allow you to develop a sheet metal part then produce drawings of it in "flat pattern" that you can cutout and fold. Tedious to do in a programs like AutoCAD or QCad, but possible.

Blender is a 3D nurbs modeler that allows the user more free form development and really should not be compared to the likes of AutoCAD as the wikipedia states. You might be able to design a computer case with it, but as  **Whiffle** pointed out, you could not dimension it and print out a drawing for some to make the part. It can be used for rendering output from CAD programs and there was a plug in to make it a parametric 3D modeler in development called BlenderCAD.
