### Post by drs305 on 2008-12-25
If you want to put the icon in one of the system folders, open nautilus with admin rights:
```
gksudo nautilus
```

Find your icon and move it wherever you wish

If you are having problems running a system command in your script or are having other problems, it would probably be best to post the commands/script.

If you just needed to move the icon and can now do so, please mark the thread 'Solved'.
