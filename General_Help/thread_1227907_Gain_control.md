### Post by t0p on 2009-07-31
> **nipunreddevil said:**
> 
Why is it said that Linux is much better than any other platform for networking?

Linux is a Unix-like operating system.  And, historically-speaking, Unix is the operating system that the internet ran on.  TCP/IP was written *for* Unix.  Other operating systems (like the older Windows versions) were not created with networking in mind - networking had to be shoehorned into those OSes, and it didn't always fit too well. Whereas the networking protocols are an integral part of Unix (and Linux).

Nowadays, the above arguments are not so valid.  Vista was written with networking in mind, so it's not such an odd fit.  And, while it's true that a lot of internet servers run on LAMP (Linux-Apache-MySQL-Perl/PHP/Python, there are also plenty of Windows-running servers.

But there is still a really good reason to use Linux for network stuff rather than Windows.  When a computer is connected to the internet, any security failings become very very relevant.  And Linux has an excellent, built-in security model.  A properly-configured Windows box also has excellent security.  But Linux is, I think, better.
