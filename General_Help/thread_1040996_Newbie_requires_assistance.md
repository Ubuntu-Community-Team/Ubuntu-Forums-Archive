### Post by blazemore on 2009-01-16
I'm glad you have decided to use Ubuntu!

To allow Ubuntu to take over the whole drive when installing, you can select the "Guided: Use Entire Disk" option when installing. If you have more than one hard-drive, they will appear as options, allowing you to select to which one you want to install.

If guided partitioning fails, I can walk you through doing a manual partitioning. Don't worry it's very straightforward, it's just that some of the choices you need to make aren't very obvious.
