### Post by Jesus-Ninja on 2008-06-27
Well, I fixed it!

Although I'm not sure if I did it the right way....

I've comment out a line from a block that i copied and pasted from the ati setup pages:

```
Section "ServerLayout"
        Option          "AIGLX"         "true"
        Identifier      "Default Layout"
        Screen          "Default Screen"
        InputDevice     "Generic Keyboard"
***#        InputDevice     "Configured Mouse"***
EndSection

```

And the problem has gone.

Thanks though :)
