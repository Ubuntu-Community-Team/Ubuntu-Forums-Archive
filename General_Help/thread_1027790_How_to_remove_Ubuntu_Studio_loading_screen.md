### Post by zeronothing on 2009-01-01
on suggestion is you could use the program "startupmanager" to change the the boot screen to back to the original. you can install it using synaptic or you can open up terminal and do:

sudo apt-get install startupmanager

after you install it, it will be located under System -> Administration -> Startup-Manager. then once you have the program up the option to change the boot up screen is under the tab Appearance called Usplash theme. hope this helps...
