### Post by snowpine on 2011-12-25
Lubuntu is simply Ubuntu with the LXDE desktop environment instead of Unity. Your core Ubuntu system (kernel, library, applications, etc) will receive bug fixes and security patches for the full 5 year support period. Only a small number of Lubuntu-specific packages are not considered LTS.
