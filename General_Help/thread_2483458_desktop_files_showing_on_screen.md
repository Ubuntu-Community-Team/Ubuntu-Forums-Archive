### Post by Impavidus on 2023-01-31
What you see on the desktop is normally the contents of your Desktop directory, plus maybe some other stuff, depending on your desktop environment. The exact name varies with localisation. The red dots with a cross suggest you have no write permission on those files. Maybe some installer script running as root put them there? I thought the desktop environment was supposed to be smart enough not to display the name of the file in the case of desktop files, but the name of the program instead. Maybe it isn't properly included in the file? I keep my own desktop always clear. It's my weather monitor. The colours look a bit weird.
