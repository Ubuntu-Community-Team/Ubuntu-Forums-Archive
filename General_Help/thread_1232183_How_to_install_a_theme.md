### Post by Andreas1 on 2009-08-05
themes go into .themes/ in your home folder (hidden, press ctrl+H to show hidden files) or into /usr/share/themes (need admin rights to do that).

but if you drag a .tar.gz file that contains a theme into the appearance settings window it should be installed automatically.
