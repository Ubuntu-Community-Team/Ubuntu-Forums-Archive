### Post by ajgreeny on 2011-11-29
As it appears to download as a binary file in a .gz file, you should be able to run it direct from your home folder.   Make a folder called ~/.bin in your home and put the executable binary file in there and you should be able to run it simply with the appropriate command provided you make sure the binary file is executable.
