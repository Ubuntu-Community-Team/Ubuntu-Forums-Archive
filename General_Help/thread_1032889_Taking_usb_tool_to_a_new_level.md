### Post by CoffeyW on 2009-01-07
You can actually install ubuntu directly to a usb disk as you would with a hard drive if your computer supports booting from usb. However if you do it this way you will want to disable the swap partition. Other issues also arise from moving between computers such as network interfaces building up.
