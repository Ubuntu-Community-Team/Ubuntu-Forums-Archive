### Post by Snow986 on 2008-12-17
You go to ubuntu.com and click desktop edition.  Then click download.  It will download an .iso file.  Then you should be able to double click the file downloaded and your cd creator program should know to create a disk, but if it doesn't then you should open your cd creator program and should find an option that says create iso disc or create disc from iso file.  Then once it has burned you boot to your new disc on the computer you want to install and follow the instructions from there. Hope that helps some.
