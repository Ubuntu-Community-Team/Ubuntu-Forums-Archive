The trouble I have is that the Gnome panel doesn't show any icons of applications in the system tray - e.g., i run my Pidgin, it works correct but there is no icon of it in the system tray.

---

### Post by Tim Sharitt on 2009-01-16
Make sure you didn't inadvertently remove the notification area applet. 

Right-click on the panel, select Add to Panel...

Select Notification Area and click add.
