### Post by Evil Wayz on 2015-03-14
Originally when I clicked on a torrent icon to download it, a message box would pop up and ask me if I wanted to open it with transmission or save it.  Well I was doing something with transmission and  my computer was running slow so when a really delayed message window popped up I saved it.  Now I don't get a choice it automatically saves it.  Can someone tell me how to make it go back to asking?

### Post by pmi2 on 2015-03-14
In Transmission, Edit/Preferences, Downloading tab, make sure the box called "Show the Torrent Options Dialog" is checked.  (When the box is cleared, any torrent you click on will open automagically.)

After the Torrent Options dialog box pops up, and IF you want to start the download later, clear the box called "Start when Torrent is added".  This way the torrent is added but paused, and later you can enable it, or enable only the torrent or torrents that you want to run.  This is typically more convenient than saving and opening the torrent files later from the Download folder and such.

Or,

The other option will be in your browser.  In Firefox, you can go into Preferences, Applications, and find the action for "magnet" links in the list on the left, and select the action you want on the column on the right.  In your case you would change the action to "Always ask".
