### Post by WorMzy on 2011-05-14
You can run any applications on any Desktop Environment. Just be aware that you'll need to download the KDE libraries to run KDE apps, the GTK2 libraries to run GNOME2 apps, and the GTK3 libraries to run GNOME3 apps (I don't know if these are available on Ubuntu yet, you may need to add a PPA for them). Depending on what the app does, it may not be worth downloading and installing ~200MB of libraries just to run it.
