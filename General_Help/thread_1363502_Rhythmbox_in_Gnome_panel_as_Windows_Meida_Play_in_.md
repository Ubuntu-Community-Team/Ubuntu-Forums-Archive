### Post by 3Miro on 2009-12-24
> **Burky said:**
> Do you know how you can minimize Windows Media Player or iTunes into the task bar and able to control it without opening it up? Well is it possible to do the same with Rhythmbox and gnome panel?

I think this is the default behavior. When I start Rhythmbox, I do get an icon in the panel (of both Gnome and Xfce). Then when I x-out if it, it just minimizes. Right click and one can select Next/Previous/Play/Pause. You can also configure it with shortcuts or use a media keyboard.
