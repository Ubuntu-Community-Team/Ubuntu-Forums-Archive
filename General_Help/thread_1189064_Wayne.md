First let us know what type of browser you are using.

I use seamonkey and in edit-> preferences
There is section called downloads, and in there I have it set to automatically download all my files to my dekstop.,

If you use firefox its something similar.

There is an option or preference to allow you to either have it prompt you every time it downloads something, or it downloads the file for you automatically.
Change that and save you settings and it should work.

On the second question, it all depends on what you are downloading.

The best thing to do is look for applications and programs in synaptic and it will install it for you.
system->administration->synaptic

Thats all possible and not difficult, however, again, it would be good to know the name of the program and the extension type.
It could be a .deb file, a tar.gz file and so on.
