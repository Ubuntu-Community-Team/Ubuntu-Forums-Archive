### Post by taurus on 2009-03-11
Do you have the /media/Doze mount point for it?

```
ls -la /media
```
If you don't, then you need to create it.

```
sudo mkdir /media/Doze
sudo mount -a
df -h
```

p.s.  You don't also want to mount the other two partitions:  /dev/sda1 (ext3--Kubuntu) and /dev/sda6 (vfat) or at least /dev/sda6?
