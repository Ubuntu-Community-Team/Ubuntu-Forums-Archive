Firstly, "Ctrl-S" in Evolution saves the message as a draft, it does **not** send the message. Ctrl-Enter is used to send the message.

Check the Character Encoding setting, and if it is not the problem create a fresh Evolution profile by renaming the (hidden) .evolution folder and see how that behaves.

### Post by lswb on 2010-02-28
One idea: From gnome main menu, select System/Preference/Appearance/Interface and see if the "Editable menu shortcut keys" box is checked.

When this box is selected, some gnome programs allow changing of menu shortcut keys by highlighting the menu entry and pressing the desired key. You may have inadvertently changed the shortcut to a plain 's' Set the shortcut back to your desired key combo then uncheck the box in the appearance app.
