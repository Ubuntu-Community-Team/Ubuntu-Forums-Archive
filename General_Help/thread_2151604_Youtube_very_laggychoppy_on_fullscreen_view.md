### Post by ajgreeny on 2013-06-05
This is largely a flash problem in Linux, I'm afraid, which is a lot more of a problem than it is in Windows. As you probably know, flash is not being developed any more for Linux, so other than security fixes of 11.2, there is little you can do except make sure your system is fully updated, and that you have the latest graphic drivers for your video card.  You can also opt in to the html5 system to see if that helps with some flash videos. See [http://www.youtube.com/html5?gl=GB](http://www.youtube.com/html5?gl=GB)

What hardware do you have, as flash uses a lot of computing power in Linux, and low spec computers will always find it more difficult to run flash than high spec machines.
