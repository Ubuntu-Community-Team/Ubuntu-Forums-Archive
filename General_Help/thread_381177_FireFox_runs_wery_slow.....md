### Post by liegerm on 2007-03-15
Do other browsers work normally? Is your PC slow with other applications?

You could try turning of IP6 support in Firefox. To do this, type 'about:config' in the address bar and find an entry that has 'disableIPv6' in it. Change to 'True' and restart Firefox.

Long shot, but worth a try.

### Post by fragos on 2007-03-15
Disabling IPV6 will help performance when start a session with a site a DNS lookup is required.  With IPV6 enabled the attempt will have to time out, which it will, before IPV4 is used.  Another point is that DNS performance varies greatly between ISPs.  Comcast is particularly fast.  As a user you can't impact that as it relates to their DNS network design.
