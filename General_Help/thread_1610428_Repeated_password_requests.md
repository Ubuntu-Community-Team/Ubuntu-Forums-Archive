### Post by beacon on 2010-10-31
When installing 10.10 I ticked automatic log-in, so no password is needed. However, I am continually asked for a password when working. If I leave for a few minutes I come back to see a password request on the screen. What can I do to stop these irritating requests?

### Post by coffeecat on 2010-10-31
> **beacon said:**
> When installing 10.10 I ticked automatic log-in, so no password is needed. However, I am continually asked for a password when working. If I leave for a few minutes I come back to see a password request on the screen. What can I do to stop these irritating requests?

The password request on the screen is simply the screensaver kicking in and locking the screen. You can turn that off with System > Preferences > Screensaver. Untick the 'Lock screen...' checkbox.

Other password requests are for when you want to do things that need administrator privileges, such as updates and installing software. This is no different from what you get in MacOS and no different in principle from Vista/W7 UAC confirmations. It is unwise to bypass password prompts for these.
