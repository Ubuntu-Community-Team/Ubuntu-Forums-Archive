### Post by geirha on 2008-05-31
That theme seems to be a gtk2 (only) theme, which only controls how the components inside the window looks, not the titlebar and borders. For the titlebar and borders, you'll need a metacity theme.

The description suggests
> 
The best metacity themes that match this gtk theme are LiNsta Black Flat, LiNsta Black Plastic, LiNsta Black Plastic & Alu, and LiNsta Silver Plastic.


So try one of those :)
