### Post by justin whitaker on 2007-11-21
> **theman said:**
> distro that's STABLE

That's a broad brush. Every distro has it's quirks, so it might not be stable on all machines at all times.

> doesn't update every six months

Most distros do not update every 6 months: only Ubuntu, Fedora, and other cutting edge distros do that. You can use Debian, Slackware, Gentoo, Arch, openSUSE, PCLinuxOS, and a myriad of others that update on a 9-12 month or longer basis.

>  needs to update flawlessly without clean install

Again, that's a broad brush. Not every update system works flawlessly 100% of the time. Then again, you never really need to update at all if you are comfortable running your system as is. 

>  prefer rolling   distro

That's trickier. You need a meta-distribution then, something like Gentoo, openSUSE, Fedora, or Mandriva. 

> 	based on kde (can't stand gnome's lack of apps and not being configurable)

With a meta-distribution, you can install anything you want. OpenSUSE and Mandriva do KDE really well. 

> is active and has a lot of software/ large repository

Well, that shoots PCLinuxOS in the foot. 

>  excellent hardware detection

Is this still an issue these days? 

>  BONUS: one-click install ( such as ADD/ REMOVE programs), 64-bit

Once click install? Most package managers have a front end you can use...so I guess that fits. 

> So far PC-BSD and PCLinuxOS fit the bill, but BSD’s behind hardware detection and PCLinuxOS's lack of roadmap make them less appealing. They also don’t have 64-bit versions.

PC-BSD does really well with most common desktop hardware, so don't write it off too quickly. It also has the ability to use both ports, packages, and PBIs, so it has a wide range of potential software available for it. 

And don't right off PCLinuxOS because it does not have a "roadmap". It doesn't really need one, since their feature set, and goals are pretty much set. 

If it weren't for Mandriva's flakey quality control, I would recommend that to you. Instead, take a look at openSUSE.
