### Post by ago on 2008-02-02
Do you mean you want to install using an existing ISO or do you want to access the file within an ISO from an installed system?

The answer is yes in both cases.

To install using an existing ISO put the ISO and wubi in the same folder. Note that the ISO will be ignored if it is not the correct one or if it is corrupted.
