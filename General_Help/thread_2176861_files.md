### Post by Impavidus on 2013-09-26
Yes, mount the partition and delete the file.

However, two warnings: Windows must not be hibernating and writing to the Windows C partition (and file deletion is an example of writing) has been reported as corrupting Windows. Also, the file system must be undamaged.
