### Post by drs305 on 2012-06-25
Themes can cause this issue but on my latest Firefox installation there is only the default theme.

You can also try turning off Hardware Acceleration:
Edit > Preferences > Advanced > 
General Tab: Browsing > Use Hardware Acceleration

I've had this issue but it occurred so rarely that I don't know if the above has corrected the issue or not.

If the 'clenched fist' returns, you could try disabling all your addons and then reinstall them one at a time. Again, if the problem is occurs rarely it will probably take some time to determine which addon is causing the problem.
