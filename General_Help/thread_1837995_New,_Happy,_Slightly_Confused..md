I am also not a very experienced user myself but I would strongly suggest watching a few 11.04 reviews from youtube (there are many, and they are short & not boring) then you will get familiar with things like software center and synaptic package manager, reading about linux file system hierarchy ([here]("http://tldp.org/LDP/Linux-Filesystem-Hierarchy/html/index.html")) for not quickly messing the system up, and trying to get familiar with the system and preferences.
