### Post by nerdy_kid on 2009-11-13
open a terminal, type ```

sudo apt-get install synaptic

```

there you go :)

the program 'synaptic' should be somewhere in the 'system' menu...

if you want, as the program will look different from KDE ```

sudo cp ~/.gtkrc-2.0-kde4 /root/.gtkrc-2.0

```have fun :)
