### Post by unutbu on 2008-05-27
Are you planning on installing Vista?

See [http://ubuntuforums.org/showthread.php?t=634681](http://ubuntuforums.org/showthread.php?t=634681)

I think the general consensus is that 
1) If you have a Vista partition then you should use some kind of Vista partition manager to manipulate it -- not GParted
2) If you are not using Vista, then GParted works fine.
3) If you are using Vista, some people report being able to manipulate partitions with GParted, and some people report having problems.

### Post by bodhi.zazen on 2008-05-28
Yes, most likely you can.

I just finished resizing a vista partition -> Installed Ubuntu.

Everything was just fine, no problem at all. Booted to Vista -> windows ran check disk -> vista boots.

BUT, you should be prepared to restore Vista if you need to. This includes making a back up of your data and knowing how to restore Vista. Some installations come with a restoration partition and some allow you to create a set of restore disks. Read your end user documentation or call your manufacturer for assistance.
