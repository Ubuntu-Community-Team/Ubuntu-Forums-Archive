I'm willing to bet this is related to a problem I'm having on my Debian desktop with the mouse and keyboard in some games.

Try putting this line in maybe your .bash_profile or something to get executed at login:
export SDL_VIDEO_X11_DGAMOUSE=0

It seems that the most recent Debian (and apparently Ubuntu) packaged version of Xorg's DGA support has trouble.  I'm guessing Compiz might also try to use this system - which is better, except that now it's broken apparently.
