### Post by LasloHollyfeld on 2010-05-04
Hi. I have Verizon FiOS, and there isn't any specific software that you really need to run in order to use FiOS other than a web browser like Firefox or Chrome. When they connect up FiOS for you, the Verizon installer will use his own laptop to configure the service, and he'll install an Actiontec broadband router / wireless access point. Your computer asks that broadband router for an IP address using DHCP. That works great with Ubuntu. If it's a wired network connection, it's almost completely automatic. If you use a wireless adapter instead, you will have to do a small amount of setup for the security.

You won't be able to use Verizon's $6/month "Verizon Internet Security Suite", but the good news is that most of that stuff is either unnecessary with Ubuntu, is installed already by default, or can just be installed for free from the repository.

It works great! Enjoy!!
