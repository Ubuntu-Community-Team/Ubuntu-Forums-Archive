### Post by songshu on 2009-02-16
the easiest would be to do a fresh install (format) indeed if you want your /boot and /root partition to be ext4 as well and really use its full potential.

ext4 (the stable version) is only available in the 2.6.28 kernel in Jaunty and the last time i checked you need the "alternate" install cd and select manual partitioning.

works pretty well, but i don't think its really worth the fuss to upgrade to ext4 for desktop use personally if its going to cost you a lot of effort. 
you might just want to wait for a convenient moment.
