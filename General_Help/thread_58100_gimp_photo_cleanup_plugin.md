### Post by AndyAWS on 2005-08-18
I'm not sure about a Gimp plugin but I can tell you that good photo-editing software for linux is hard to find, but seems to be starting to come around.
I don't know if you'll find what you're looking for but from what I can tell, two programs that seem to be making nice progress are F-Spot, and digiKam with Kipi plugins.

F-spot is in the repos (Backports I think)

digiKam is in Backports but there is a newer version, along with working plugins at an unofficial repo here:  [http://www.mpe.mpg.de/~ach/kubuntu/hoary/Pkgs.php](http://www.mpe.mpg.de/~ach/kubuntu/hoary/Pkgs.php)
