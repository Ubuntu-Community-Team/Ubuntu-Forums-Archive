### Post by PointSource on 2007-06-07
For long downloads a workaround:
Right click on link and select "Copy Link Location".
Then open a terminal or run dialogue and type in "wget {paste link here}".
Your download should eventually be in your home directory.

I've had this problem happen to me as well, but I was using a downloaded version of firefox and an extension called "Download Statusbar". It hasn't happened (yet?) since I upgaded to feisty and used the included firefox v2.
