### Post by fooman on 2009-02-01
are you saying that when you log in to ubuntu,  there is only the desktop wallpaper with no top/bottom panels? 

if so,  try pressing alt-f2 and see if a "run application" box pops up.  if it does, type this into it and press "run":

```
gnome-panel &
```

hope that helps.
