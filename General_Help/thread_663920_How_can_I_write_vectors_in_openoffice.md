### Post by Irony on 2008-01-10
OpenOffice already has a formula editor which can be run as standalone or with say the Word Processor.

For example;

Open the Word Processor type a few letters then go to;

```
Insert > Object > Formula
```

You can google for a Open Office formula, e.g. [http://wiki.services.openoffice.org/wiki/Documentation/OOoAuthors_User_Manual/Writer_Guide/Math_Objects](http://wiki.services.openoffice.org/wiki/Documentation/OOoAuthors_User_Manual/Writer_Guide/Math_Objects)

You will find vectors look like this;
```

vec z+ vec b
```
