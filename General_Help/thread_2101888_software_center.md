### Post by Bucky Ball on 2013-01-05
10.10 has reached end-of-life (quite some time ago!). Install 12.04 LTS, supported until 2017. LTS will upgrade to the next LTS release in a couple of clicks. Interim releases need to be upgraded from one release to the next (until you get to an LTS which can go to the next LTS).

You will not get security updates (or anyupdates) for 10.10 and this will leave your machine vulnerable.
