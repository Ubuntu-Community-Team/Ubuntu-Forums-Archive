### Post by fooman on 2008-11-02
the servers are still tied up with the new release and all...

try a different server.

go to system > administration > software sources. click on the ubuntu software tab and then click the down arrow at the end of the "download from:" button.  choose "other"

from there,  click the "select best server" button and let it find the fastest server for you.

or you can manually choose a server from there also.

hope that helps.
