### Post by niteice on 2007-12-02
So I recently purchased a large (500gb) hard drive for storage. As I dual-boot Kubuntu and Windows XP (both 64-bit), I need a way to access it from both operating systems. Am I better off formatting it as ext3 and using ext2fsd under Windows, or should I format it as NTFS and use ntfs-3g under Linux?

I will probably be using Windows mostly for programming and the occasional game/app that won't run under WINE...so I'll probably be using Kubuntu most of the time, which leads me to think that ext3 is the better choice; but I'd like to hear about both.

---

### Post by rsambuca on 2007-12-02
You answered your own question.  Format it to which ever system you will be using more.  While both methods work, accessing the foreign filesystem is a little bit slower with both methods.
