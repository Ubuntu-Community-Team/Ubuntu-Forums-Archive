### Post by howefield on 2011-10-14
In terminal use the commands..

```
cd /etc/xdg/autostart/
```

```
sudo sed --in-place 's/NoDisplay=true/NoDisplay=false/g' *.desktop
```

Startup applications should now show up as normal.

### Post by MG&amp;TL on 2011-10-14
You can press the power button on the top right hand corner, and there's an entry for 'startup applications'. You can edit and make more there.
