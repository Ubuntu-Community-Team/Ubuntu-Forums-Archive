What is UNR?  ubuntu.com tells you what's needed to run it but doesn't really say how it differs from standard Ubunu.  Is it a stripped-down Ubuntu? ---

### Post by t0p on 2009-06-22
UNR is a pared-down version of Ubuntu, optimized for netbooks.  This means there are less apps, to suit a netbook's smaller disk, and a GUI that's been changed to suit a netbook's tiny screen.  If you have a proper laptop, you'll be a lot better off installing the full desktop version.
