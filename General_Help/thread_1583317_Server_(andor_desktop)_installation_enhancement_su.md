### Post by Cool Javelin on 2010-09-27
Here is a suggestion for the Ubuntu team regarding the installation scripts, especially the servers softwares.

Add a few questions for setting up the samba or whatever software you choose for serving files.

Three questions come to mind right away, "Do you want this computer to be a wins server" (why and why not,)"Will older Windows (or other) OS's ('98, DOS, 3.11, etc..) be attempting to connect to this computer?" and "Should this computer be using user or shares access?" (give a short explanation of the differences.)

Perhaps ask for a list of potential users and what permissions you want for each. Maybe not.

Then, setup the samba smb.conf file properly.

There seems to be tons of threads of people asking for help in this area so there are still a significant number of people using the older OS's and with a tiny enhancement during the installation, a lot of frustration can be avoided.

Just my 2 cents.
Mark.
