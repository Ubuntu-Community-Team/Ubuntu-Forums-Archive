### Post by jerome1232 on 2011-10-14
Well as to your original question though yes, a 32 bit machine simply won't boot the 64 bit version.

There are *still* a few proprietary programs around that won't or are difficult to get working on a 64 bit machine I'm sure, though what these programs are I don't know, I've never ran into one.

The only disadvantage I can think of is 64 bit programs have a slightly higher memory footprint, so if your severely limited on ram (think 512 MB's of ram, maybe even 1 GB) then I wouldn't run 64 bit.
