### Post by Bucky Ball on 2012-06-18
```
sudo apt-get update
```Your error message advises you to do this to attempt to fix some issue. Then:

```
sudo apt-get upgrade
```This last command will ONLY update installed packages, NOT the release. ;)

Also:

```
W: Duplicate sources.list entry [http://ppa.launchpad.net/ubuntu-wine/ppa/ubuntu/](http://ppa.launchpad.net/ubuntu-wine/ppa/ubuntu/) lucid/main ... etc
```Do you have this entry twice in your Sources list? Try doing this in Synaptics rather than a terminal and you might save some time ... it is not in the repositories although may be in the one you are having a problem with. And if it is, then your repo issue is your problem, not your command. 

```
sudo apt-get install ubuntulooks
```... is right, providing the software actually exists. Post #2 in the link Shadius provides would be somewhere to start.
