### Post by Kevbert on 2008-07-19
As well as the standard (default) installed bluetooth programs you need gnome-obex-server which you can get via Synaptic Package Manager.
Go to System-Preferences-Bluetooth.  Check under the default tab that Visible and Connectable for other devices is selected and adapter is never invisible.  Check all services are enabled and running.  You can now close this tab.
Run up the gnome-obex-server.  This can be found under Applications-Accessories-Bluetooth File Sharing.  All the will happen is that a new icon is displayed in the taskbar top righthand corner.
Now send a picture from the phone.
