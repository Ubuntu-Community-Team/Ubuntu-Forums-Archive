### Post by tjlane on 2009-07-18
I just attempted to install 9.04 on my toshiba satellite A305D, when I encountered an error. The system hung completely, and there was nothing I could get it to do without doing a hard power-down. 

So I did it.

Now the computer, which worked flawlessly 15 minutes ago, will not boot at all, not even into the Toshiba BIOS. Instead, it goes into the InsydeH20 setup utility, which I had never seen before. 

InsydeH20 does have a 'Boot' tab, which should provide options on boot sequence, etc- but there's nothing there. 

On the general page that lists some computer specifications, it lists both Hard Disk and Optical Drive as "None".

I'm at a loss. I attempted to run both Ubuntu live discs and Windows recovery discs, but it was no dice. Any help anyone can offer will be much appreciated.

Thanks in advance!

---

### Post by tjlane on 2009-07-18
Well, problem solved- I finally tried mounting a USB flash drive on a whim and it caused my menu to reload, which did the trick- at that point I was able to access both drives, and all of my data was safe!
