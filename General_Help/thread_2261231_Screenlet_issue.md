### Post by Mike_Walsh on 2015-01-17
Hi, Terrell.

I use them myself. I think (since I like them on ALL desktops) that you need to open up the particular screenlets you want, on the desktop you want.....but then right-click, and make sure that 'Sticky' is unchecked, but 'Locked' IS checked.

You may have to use the screenlets Manager to physically 'switch off' the ones you want to confine to one desktop. Then shut it down (you might need to re-boot, I wouldn't like to say!); select the desktop you want, open up the screenlets you want on that particular desktop, and perform the checks I mentioned above.

Hope that helps; let us know how you get on.
