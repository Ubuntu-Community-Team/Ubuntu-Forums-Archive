### Post by marshag63 on 2010-11-24
Sxynerd, is everything working with your computer - saw you mention "wrappers."  I learned a lot just by solving how to fix the little things that were not working out of box - wifi, printer setup, network setup, monitor settings.

Is there something in particular you are needing to do -play music, write emails, text editing, working with photos?

The best piece of advice I have is do not expect linux to work like windows.  For example, when you find a program you want to install or are missing a helper app, you go to your distributions repository to get it and you use the package manager app - in Ubuntu it is called "Synaptic."  To use it, you have to get administrator rights, which you do by giving the administrators password. Make sure you have let your computer install all the recommended updates before you try to install anything - very important to update a newly installed Ubuntu system.

Are you using Ubuntu Gnome?  Some of the same keyboard shortcuts as in windows work in Gnome - Alt + F4 to close a window, Alt-Tab to switch windows.  You can also drag and drop files.  Ctrl + Esc is default to bring up the "Start" menu (main menu listing apps).  You will also get "right click" options.

Hope this helps a little.

MarshaG.
