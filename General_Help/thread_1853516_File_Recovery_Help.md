### Post by carranty on 2011-12-13
Recovering files is possible, there are many threads on this topic.  The most important first step is to create an image of the drive your trying to recover from, and work on that image.  That way you won't further corrupt the damaged drive.

After this use programs like photrec to try to recover files.  Note however file recovery is a long and tedious process, it would be better if we can just fix you problem.

Before you do anything else I'd recommend making an image of the drive, then we can we about getting your original install up and running again.  

After making an image, try booting into your original install  Are there any error messages?
