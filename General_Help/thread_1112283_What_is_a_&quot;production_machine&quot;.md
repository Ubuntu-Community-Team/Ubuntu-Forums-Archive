### Post by Therion on 2009-03-31
A production machine is one that supports internal business functions or customers directly. As opposed to a test, staging, or development machine.

It basically means any machine you can't live without since the software is still in a testing phase.

### Post by dcstar on 2009-03-31
The bottom-line is that it means do not use on anything you **rely** on, because it has not (yet) been proved to be a reliable system/component/whatever.

And as it also says (as it does for all Ubuntu pre-release versions), don't even think of posting problems anywhere else but the development forum. If you install this stuff before release then you are on your own and have no right to use the other forums for the many problems that will arise from Alpha/Beta releases.
