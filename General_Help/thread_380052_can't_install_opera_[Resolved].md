### Post by Sef on 2007-03-09
Opera is available in Add/Remove.  (Applications > Add/Remove.)  You need to open [Multiverse]("https://help.ubuntu.com/community/Repositories/Ubuntu"), so you can access it.  Once Multiverse is open, then just type in Opera in the search box in Add/Remove.
