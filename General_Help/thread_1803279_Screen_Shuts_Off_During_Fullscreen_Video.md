### Post by Abir33 on 2012-08-04
Quick reply for anyone who hits this thread. If you are working in 'Gnome classic' in 11.04,
install full gnome package from software center, once you have done this you will find that there are TWO screensaver options in your preferences menu. Go into the second one and disable, in the first one make the minutes 740 as is highest. 

Any advanced users, please don't get upset, I know that's a weak work around, writing this for noobs though.
