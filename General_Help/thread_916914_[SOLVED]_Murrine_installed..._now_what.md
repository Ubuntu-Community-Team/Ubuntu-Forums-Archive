### Post by sayakb on 2008-09-11
First, goto System->Preferences->Appearance and select the Murraine theme.
Not press Alt+F2 and type in **gtk-window-decorator --replace** to get the full murraine theme controls.
If you wish, you may add the gtk-window-decorator in startup (System->Preferences->Sessions->Add)

### Post by benerivo on 2008-09-11
The murrine based theme that you want does have a coloured line at the top of the active window - it only goes completely dark when it is an inactive window (check this out on the screenshots in gnome-look).

For a style without the coloured line, install the [clearlooks one]("http://gnome-look.org/content/show.php/Clearlooks-Colors?content=75417") (also made by perfectska04).

For the murrine version, make sure you have installed the murrine package that the author provides, and not the one from the ubuntu repositories.
