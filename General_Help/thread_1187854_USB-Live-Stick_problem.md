Now, what I would like to change is that the stick automatically logs into a user named "ubuntu", then I have to log out and log my own user in.

### Post by briguy47 on 2009-06-15
Absolutely.  All you have to do is go to System --> Administration --> Users and Groups.

When I did it, I chose to remove the Ubuntu user and created my own.  Then, for security purposes, just go to System --> Administration --> Login Window.  Then select the Security tab and unselect Automatic and Timed Logins.

If you have any questions, just let me know!  :D
