### Post by sumguy231 on 2007-09-15
As far as I know, it means exactly that: the number of the room the user is located in. It's probably a holdover from olden days of yore, when people used Unix Timesharing Systems and each user logged into one central UNIX box using dumb terminals. You can just leave it blank if you want to.
