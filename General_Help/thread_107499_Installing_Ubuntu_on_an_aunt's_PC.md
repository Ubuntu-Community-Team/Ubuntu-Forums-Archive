### Post by welsh_spud on 2005-12-23
Give the [LiveCD]("http://mirror.mcs.anl.gov/pub/ubuntu-iso/5.10/") a go, it will do everything the same as a normal Ubuntu installation, but will not use the hard disk. If dial up works on the live CD you can be sure that it works in Ubuntu.
