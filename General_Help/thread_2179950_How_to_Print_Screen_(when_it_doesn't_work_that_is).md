### Post by mcduck on 2013-10-10
Just use some tool that allows you to do a screenshot with a bit of a delay. If you use Unity or Gnome, you should find one simply by searching for "screenshot" in your applications. Or if you prefer, Gimp can also take one for you which can be handy if you wish to edit the image afterwards ;)

### Post by scouser73 on 2013-10-10
**1** - Click on the Ubuntu Icon in top left of screen

**2** - Type Screenshot & click on it

**3** - Set the timer function if so desired

**4** - Click on Take Screenshot

### Post by deadflowr on 2013-10-10
Yep, use the app directly, through the app menu.
Set a delay.
Or run the command through a terminal
```
gnome-screenshot -d <#>
```
<#> is the seconds for the delay. Just put the number without < > symbols.

```
man gnome-sceenshot
```
will give you the various commands for the various ways you can run screenshot.
