### Post by euler_fan on 2007-05-05
I agree with the previous poster--go ahead and run both. On occassion it is useful to have 64-bit (if you are an R user, you can work with bigger datasets using 64-bit than under 32-bit, for instance) with I suspect that is more on a case by case basis.

If you have a lappy requiring ndiswrapper to use the wireless networking, then I suggest some caution using 64-bit. I run edgy, and 64-bit edgy (last time I checked) does not work with ndiswrapper. I don't know how it does on Fiesty (I may upgrade my 64-bit install at some point just to play around with it).

Otherwise, I have seen nothing to suggest I am somehow "losing something" by not running 64-bit as my primary OS.

Good Luck!
