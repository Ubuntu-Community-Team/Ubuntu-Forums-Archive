### Post by Frogs Hair on 2013-03-01
It could be a corrupt profile or an installation problem.  You can create a new profile by opening the home folder > Ctrl + H and deleting the .thunderbird  folder . When Tbird is opened again a new folder will be created. The down side is you have re-enter all the information and install any add-ons again. If you have a lot of important information with you profile you my not want go this route.
