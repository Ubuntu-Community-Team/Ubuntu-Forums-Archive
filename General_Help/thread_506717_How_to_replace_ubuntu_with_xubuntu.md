### Post by bodhi.zazen on 2007-07-22
You can install with sudo apt-get install xubuntu-desktop

BUT to remove ubuntu (gnome) desktop, see this :

[http://www.psychocats.net/ubuntu/purexfce](http://www.psychocats.net/ubuntu/purexfce)

NOTE Copy-paste the command from the "Remove Ubuntu" (that command first removes gnome and then installs xfce)
