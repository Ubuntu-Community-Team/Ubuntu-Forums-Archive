### Post by Roger Mudd on 2006-10-28
Try signing up for a [Hotmail]("http://www.hotmail.com") email account. Once you've done that you log into MSN's network via GAIM using that email address and the password you selected. One of you will need to know the other's email address so that you can add each other to your respective buddy lists.
As an aside -- and not to complicate things -- If she is using an up-to-date Mac with iChat installed, you might try the open source chat protocol [Jabber]("http://www.jabber.org/") instead. Both GAIM and iChat support Jabber.
