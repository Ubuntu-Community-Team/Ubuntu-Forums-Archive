### Post by Cheesemill on 2009-11-23
Yep.

Ubuntu Studio is just normal Ubuntu but with a different set of pre-installed applications and a slightly different kernel.

Any applications that you can install on Ubuntu will also work on Ubuntu Studio and vice-versa, they both use the same repositories.
