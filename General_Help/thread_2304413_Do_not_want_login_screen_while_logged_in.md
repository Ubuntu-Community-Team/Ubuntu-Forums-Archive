### Post by grahammechanical on 2015-11-26
Is it 14.10 that you plan to upgrade to 15.04? We cannot get from 14.04 to 15.04 without first upgrading to 14.10. But 14.10 has reached end of life. So, upgrading to from 14.04 to 14.10 is difficult.

If it is Ubuntu + Unity that you are using, then go to System Settings>Brightness & Lock and set "Turn screen off when inactive" to "Never." And if "Lock screen After" is set to "Screen turns off" you will not return to a blacked-out screen.

### Post by deadflowr on 2015-11-27
> **Sherman_Willden said:**
> I am preparing to upgrade from 14 to 15. I do not want the login screen to reappear while I am away during the upgrade. How do I set the time to login screen to zero?

Open System Settings, go to Brightness and Lock, set *Turn Off Screen When Inactive For:* to Never.
As a secondary measure, you might also go to the section for Power and set the *Suspend when inactive for*: to Don't suspend.

This is how it should work, and does for most users.
