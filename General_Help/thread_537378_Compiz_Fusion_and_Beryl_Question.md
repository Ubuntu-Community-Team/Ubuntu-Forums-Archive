### Post by TheThinker on 2007-08-31
The Beryl program from the Ubuntu repositories works great for me! Currently, the only problem I have is that the window decorator is not working (no window borders) but other than that I've never experienced a crash. Just make sure to install the tray manager since it includes a fall-back, just in case. Were you using the TAR file from the Beryl website?

I've seen a few videos of Compiz working, and so far they seem comparable to Beryl. After all, Beryl is a fork of Compiz. Whether it's more functional is another matter. I just downloaded a Compiz TAR file to instlal so wish me luck.

Compiz-Fusion, I hear, is a mix between Beryl plugins and the Compiz core program. So far the developmental version is available although I would wait for the stable version if I were you.

Either way, you win.:)
