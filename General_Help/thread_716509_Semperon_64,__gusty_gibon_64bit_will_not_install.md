### Post by jimv on 2008-03-06
I'm going to be honest with you.  I've got a Turion 64, and the 64 bit Ubuntu is no quicker than the i386 version.  The only real advantage to having a 64bit OS is if you need to run more than 4gb of RAM.

I'd go with the 32 bit version.

### Post by mrsteveman1 on 2008-03-06

There are more advantages to using native 64bit code than just memory, on WINDOWS that may be true but on Linux there are other things that make native code worth the trouble. There are some threads about this in the 64bit forum here.
