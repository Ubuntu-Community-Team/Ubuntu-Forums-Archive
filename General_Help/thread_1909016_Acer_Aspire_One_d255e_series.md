### Post by Karlchen on 2012-01-14
Hello, dha027.

Ubuntu 10.04 was released in April 2010, therefore the version number 10.04.
Ubuntu 11.10 was released in October 2011, therefore the version number 11.10.
Quite a few changes have been applied to Ubuntu between v10.04 and v11.10. The most important changes are: The Linux kernel 2.6 has been replaced by kernel 3.0. The desktop manager Gnome 2 has been replaced by Gnome 3 and Unity.
So, no, the two versions are not really the same.
I have not verified whether the minimum system requirements which were valid for Ubuntu 10.04 still apply to Ubuntu 11.10. The specifications given here, [hardware-req.html](https://help.ubuntu.com/11.10/installation-guide/i386/hardware-req.html), in particular the minimum requirements seem unreasonably low to me. I tend to consider the recommended values to be the real minimum values.

Anyway, before trying to install, you should consider running Ubuntu 11.10 from a live USB stick on the netbook. In order to install Ubuntu on the netbook, you will very likely have to create a Ubuntu live USB stick anyway, unless you have got an external DVD drive for your netbook. If the live system runs fine, then it should be safe to install Ubuntu 11.10 on the Netbook as well.

I would not be amazed, though, if other forum users will come up and recommend to use a leaner version of Ubuntu 11.10, one which does not use Unity, but XFCE or LXMDE.

Kind regards,
Karl
