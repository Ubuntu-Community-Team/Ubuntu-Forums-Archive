### Post by TeoBigusGeekus on 2010-12-14
There will be a notification about graphics card, wireless, modem, etc. drivers. Click on it and select what you want to install.
If you mean codecs, fonts, flash, etc., then
```
sudo apt-get install ubuntu-restricted-extras
```
