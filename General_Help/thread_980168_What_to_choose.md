### Post by john183 on 2008-11-12
From what little i know, all variants of Ubuntu will do all the things that you want to do. The question is really just what look you prefer. To put it in a somewhat backwards way, Ubuntu has the Gnome desktop environment which in its base install config looks similar to Win98, Kubuntu has the KDE desktop environment which in its base install format (version 4.1) could be equated to Vista (only in looks mind you) and Xubuntu has XFCE(i think that's the right acronym) which is very simplistic and could almost be compared to win 3.1 looks-wise. You have to remember with my descriptions that i am meaning them only in looks and simplicity alone, i, in no way, shape or form, am comparing them to the listed windows OS's. The base Ubuntu system that they all run on is the same.

### Post by quimico69 on 2008-11-12
> **jumpmanjones said:**
> whats the reason...just curious :)

swap is a file where the operating system will temporarily put stuff from RAM that is not being used, to make room for new stuff that needs to use. 

Let's say you are working on a report using a word processor (big program, requires big chunk of memory), then you open an electronic sheet where you have some numbers and draw some graphs (another big chunk of memory required), if RAM is small, then the OS will probably send to swap the word processor stuff to make space for the plots and graphs active at the moment.  That's why, if you go back to the word processor, the machine will feel somewhat slow: it is recalling the word processor from swap and probably putting in there the electronic sheet.

hope this helps
