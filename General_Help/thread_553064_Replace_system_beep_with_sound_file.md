### Post by erigami on 2007-09-17
A few apps in ubuntu like to beep with the terminal bell. I'd like to replace the ugly terminal bell sound with a sound from a file of my choosing.

### Post by eggdeng on 2007-09-17
In gnome, System -> Preferences -> Sound ->Sounds. Opposite the event you want to configure, click on the up & down arrows on the sound type box and you will see, among other options, Select Sound File.
