Supposedly, Fedora does not recognize other Linux Distros, it only recognizes Windows.

Install Fedora- boot loader to the / and disable MBR. Then use already installed Ubuntu GRUB to boot into Fedora. You must update grub after installing Fedora by login back into Ubuntu.

### Post by Tykin on 2011-05-29
> **i.mehrzad said:**
> How do I login back into ubuntu? I don't have that option at startup.

I found it was easier to simply keep the Fedora boot loader and add the Ubuntu partition to it. I went through the hassle of getting back into ubuntu and reinstalling grub, but it was unable to pick up on the fedora partition. It's easier to manually edit grub with Fedora's version of Grub.

Anyway, boot up Fedora and go to the terminal. We'll load up your boot menu list. Type

```
su -
(type in your password)
gedit /boot/grub/menu.lst
```

Gedit will pop open with your boot configuration. Now you need to manually add Ubuntu. Here is what mine looks like for reference:

```

# grub.conf generated by anaconda
#
# Note that you do not have to rerun grub after making changes to this file
# NOTICE:  You have a /boot partition.  This means that
#          all kernel and initrd paths are relative to /boot/, eg.
#          root (hd0,6)
#          kernel /vmlinuz-version ro root=/dev/mapper/vg_fedorapc-lv_root
#          initrd /initrd-[generic-]version.img
#boot=/dev/sda
default=0
timeout=8
splashimage=(hd0,6)/grub/lovelock.png
hiddenmenu
title Fedora
	root (hd0,6)
	kernel /vmlinuz-2.6.38.6-27.fc15.i686.PAE ro root=/dev/mapper/vg_fedorapc-lv_root rd_LVM_LV=vg_fedorapc/lv_root rd_LVM_LV=vg_fedorapc/lv_swap rd_NO_LUKS rd_NO_MD rd_NO_DM LANG=en_US.UTF-8 SYSFONT=latarcyrheb-sun16 KEYTABLE=us rhgb quiet
	initrd /initramfs-2.6.38.6-27.fc15.i686.PAE.img
title	Ubuntu
	root (hd0,4)
	kernel /boot/grub/core.img
title Windows 7
	rootnoverify (hd0,0)
	chainloader +1

```

mine is root (hd0, 4) because my Ubuntu partition is installed on sda5. Similarly, root (hd0, 6) takes me to sda7 where my Fedora 15 partition is located.

The default=0 is because I want Fedora to boot by default. If I wanted Ubuntu, I would make it default=1, Windows default=2.

Good luck! Hope this helped!
