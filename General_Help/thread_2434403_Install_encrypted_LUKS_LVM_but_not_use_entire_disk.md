Is there a way?  Yes.  It is cumbersome and complex.  Paddy has posted a guide in these forums.  There is also a guide on help.ubuntu.com.

Why doesn't the installer have this built-in?  Only Canonical can answer that.  Only a tiny number of Ubuntu installations use encryption, so the reason could be simply because it isn't popular enough.  Most people who would want to do partial encryption probably want a dual, tri, quad boot setup, which would make things much more complicated, especially with Windows involved.  Or it could simply be because Canonical is concentrating on paid server deployments and consulting to create a viable business.

We certainly don't know the actual answer.
