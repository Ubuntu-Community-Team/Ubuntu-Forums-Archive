When I resize a window an orange overlay shows me the new size of the window until I release the mouse. Can windows be resized 'live' / on the fly?

---

### Post by zombifier25 on 2012-10-12
Install ccsm first:
```
sudo apt-get install compizconfig-settings-manager
```
Then open ccsm, choose "Resize Window". On the "General" tab, choose "Normal" for "Default Resize Mode".

(also, it's a little slower than normal resizing on slightly average computers, but it works.)
