### Post by negora on 2012-05-08
Hello:

I've installed both the 32-bit and 64-bit versions of Kubuntu 12.04 LTS and then tried to run pgAdmin 3. However, whenever I open the "Properties" window and close it, it crashes.

After lots of tests, I've discovered that the origin of the problem is the "oxygen-gtk" theme, which is set by default to make the GTK applications look like the KDE counterparts.

A workaround is to set another theme, until it's fixed in next updates. I've chosen "Mist".

I hope it helps other users who are in the same situation. I've opened a bug report here: [https://bugs.kde.org/show_bug.cgi?id=299633](https://bugs.kde.org/show_bug.cgi?id=299633) .

PS: By the way, the KDE update 4.8.3 doesn't fix this issue :( .
