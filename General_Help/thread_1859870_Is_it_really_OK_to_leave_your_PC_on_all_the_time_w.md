### Post by Quatrix on 2011-10-14
> **collisionystm said:**
> That part about windows is a load of crap.

Agreed.  Hard disks (and other hardware) are designed to be used and left running.  The vast majority of disk failures happen early in the life of a drive.  If it's lasted a few months without any symptoms, it will probably last a lot longer.

That said, I put Windows 7 to sleep most nights, but more to save electricity (even if < $0.10 per night) and heat than anything.  I have no problem leaving it running for overnight TV tuner recordings or other jobs.

The problem is performance; older versions of Windows would choke themselves if they were left running for any length of time and needed a reboot to start running properly again, whereas some Linux boxes are running perfectly happy with uptimes measured in years. It's possible that newer versions of Windows are better in this regard but I've not used them so I couldn't say.
