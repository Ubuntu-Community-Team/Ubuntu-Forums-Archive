ntemis:  if the subs are a seperate file, the easiest thing to do is name it the same as the movie (not the same extension of course, although i wonder if vlc would pick up on it...).  if the movie's an .ogm or .mkv file, the subs'll likely be incorporated into the file, in which case you should go to Video in the file menu (or right click on the window), and there should be a Subtitle option under it. ### Post by capleton on 2008-05-18
This is an old thread, but I got this problem as well and thought I should post a solution.

Resize the player to 100% so the text is not distorted.  Then run Compiz and use the zoom tool to zoom in on the player and make it fullscreen; yeah it's kind of a hack.  
(also if you have a widescreen monitor, you might want to drag the bottom of the window down to create some black space above and below the video)
