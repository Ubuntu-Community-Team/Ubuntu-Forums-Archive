### Post by deadflowr on 2018-09-05
> **trickshot said:**
> so, its not like a distro?

steamos?
It's both or either.
It's like [Ubuntu flavors]("https://www.ubuntu.com/download/flavours") are.
You can install an Ubuntu flavor's desktop packages or programs on your existing system, or install the flavor's .ISO installer to the machine.

Example, you can install Xubuntu to the machine as it's own operating system to a new or separate partition, or you can also install the xubuntu-desktop package to an existing Ubuntu system, and then you will be given an option to use it at the login screen.
The xubuntu-desktop is a package within Ubuntu's software ecosystem, just like any other (like firefox, or libreoffice).

But if you have it installed on a current Ubuntu system then it's easy to say it's not installed to the machine, because steam even explains the installation would wipe out any installs
from [SteamOS download]("https://store.steampowered.com/steamos/buildyourown"):
> WARNING: Both installation methods will erase all content on the target computer
Basically SteamOS as a true install wants to use the full machine exclusively.
