### Post by tgeer43 on 2009-09-23
Since MKV is a video format and MP3 is an audio format I guess what you really meant is that you want to extract the audio portion of an MKV file to an MP3 audio file.

As Bonster stated, mkvtoolnix contains three command line tools for working with MKV files. One of these is mkvextract which will do what you want. If you are not comforatble with the command line there is a companion graphical frontend for mkvtoolnix called 'mkvtoolnix-gui', also available in synaptic.
