### Post by lmarmisa on 2010-11-20
There is an error in the file /etc/fstab. The device for swap looks incorrect. Change this line to:

```

/dev/sda3       none            swap    sw              0       0

```

Finally, delete the blank line between swap and /media/Discoteca.
