Last year I did some work building a bare-bones ubuntu with openbox using remastersys.  I had the best success making it lean by starting with a minimal install cd and building up from there.  I wound up having to install some stuff that remastersys seemed to need... I remember gdm needed to be installed, even though I didn't want it, or the live cd wouldn't play nice.  I wound up finally writing a custom script to remaster that was based on the original remastersys instructions.

I did try starting with a full install and removing things, but it's hard to know if you've gotten rid of everything that's extra.

It turned out to be a lot of work.
