### Post by cariboo on 2008-09-12
You probably need to make the file executable, to do this in a terminal type:

```
cd ~/Desktop
```

assuming your file is on the desktop, then

```
chmod +x *bin
```

then in the same terminal

```
sudo ./*.bin
```

this will only work if this is the only .bin file on your desktop.

> **cariboo907 said:**
> You probably need to make the file executable, to do this in a terminal

... To do this in a terminal, first change to the directory in which the file is located (using "cd <path>", where <path> is the directory path).  Then type "chmod +x <filename>", where <filename> is the name of the file.  Now you should be able to run the file by typing "./<filename>".
