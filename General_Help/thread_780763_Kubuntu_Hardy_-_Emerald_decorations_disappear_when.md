### Post by Zorael on 2008-05-04
I'm getting this every now and then in KDE3 as well, running the compiz and emerald builds from the official repositories.

You might want to search the forums for any available tweaks you could add to your xorg.conf (/etc/X11/xorg.conf) and see if that helps.

Further, you could give the compiz-git script a go. It has its own huge thread here in the Desktop Effects forum. It'll install a more recent though unstable build of both emerald and compiz.
