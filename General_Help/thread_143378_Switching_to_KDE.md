### Post by rubengs on 2006-03-12
You just have to run this command: 

```
sudo apt-get install kubuntu-desktop
```

You will be asked to use kdm or gdm as session manager if you plan to stick to kde use kdm.

This installs a basic KDE and some related apps, i f you want more you can search in synaptic for more KDE apps or the official KDE Desktop.
