That's really strange.  Have you tried looking for the listing in Synaptic? If not, try it. Synaptic should tell you where all the files from the package(s) is/are installed.

Right-click on the package name in Synaptic and select "Properties". Then go to the "Installed Files" tab.  That will list the path for all files installed from the package.

Now if you've done that and are still in the dark, then I can't help you. ;-)
