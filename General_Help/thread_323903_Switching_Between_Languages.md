### Post by dbbolton on 2006-12-23
go to :

system > administration > language support


i think there is also an applet for the panel that can allow you to switch keyboard layout, if that's what youre looking for. you can also change keyboard layout by going to

system > preferences > keyboard > layouts

To specify the keys to switch between languages :

system > preferences > keyboard > layouts > layout options > group shift/lock behavior
