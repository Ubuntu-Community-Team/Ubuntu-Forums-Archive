### Post by coffeecat on 2014-10-16
You don't need to use the terminal. Go here using your Ubuntu system:

[https://www.google.com/chrome/browser/](https://www.google.com/chrome/browser/)

Click on the download button and choose the 32-bit or 64-bit .deb depending on your architecture. Once the download is complete, double-click on the downloaded deb package. The Software Centre will install it for you and the Google Chrome repository will be added to your sources so that updates get applied when they become available.
