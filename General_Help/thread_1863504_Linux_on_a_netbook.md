### Post by Lars Noodén on 2011-10-18
+1 for Lubuntu

However, if you know what tools you want, then a minimal install is a good way to go.  In that way you can skip having a full desktop environment and just run with a bare window manager.   Compiz is there if you still want eye candy, or openbox or fvwm-crystal if you want much, much leaner.
