### Post by ankspo71 on 2011-09-28
Hi,
The default Xubuntu 11.04 theme is called 'greybird', and it is inside the package called 'xubuntu-artwork'. Reinstalling that package should fix it, and that can be done in Synaptic Package Manager.
Hope this helps.
