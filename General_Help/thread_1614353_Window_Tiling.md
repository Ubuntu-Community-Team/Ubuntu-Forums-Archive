### Post by Mark Phelps on 2010-11-05
The answer is yes -- it's called Aero snap (in Win7).

But ... achieving it in Ubuntu is a LOT of work because, unlike with Win7, it's not a built-in function. It involves writing and running some very complicated scripts.

If you want to take it on, suggest you do some forum searches on "Aero snap"
