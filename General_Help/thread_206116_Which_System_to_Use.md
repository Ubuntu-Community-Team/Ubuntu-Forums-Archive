### Post by Nonno Bassotto on 2006-06-29
The distribution is the same, no matter what you choose. that is, the same software will be available through the repositories. The difference is what you get installed out of the box.
Ubuntu uses Gnmoe dekstop. use this if you like simple dialogs, few options and no bloat.
Kubuntu uses KDE desktop. Use this if you prefer to have more control on your application settings.
Xubuntu uses XCFE desktop, a lighter desktop, well suited for not so new computers
Edubuntu is similar to ubuntu, but well suited for school installation, because of the choice of educational packages.

### Post by LKRaider on 2006-06-29
I'll do a quick overview of them for you. The main difference is the desktop environment they use:

Ubuntu : Uses the [Gnome](www.gnome.org) Desktop by default
[Kubuntu](https://wiki.ubuntu.com/Kubuntu) : Uses the [KDE](www.kde.org) Desktop by default ( [www.kubuntu.org](www.kubuntu.org) )
[Xubuntu](https://wiki.ubuntu.com/Xubuntu): Uses the [Xfce4](www.xfce.org) Desktop by default ( [www.xubuntu.org](www.xubuntu.org) )

Choosing between Gnome and KDE is usually a personal choice, since both are robust and equivalent choices. The Xfce is a lightweight desktop, ideal for low resource usage.

The [Edubuntu](https://wiki.ubuntu.com/EdubuntuWiki) is a distribution focused on schools and other educational environments. It uses the Gnome desktop by default, as Ubuntu does, but has packages focused on simplifying the process of adoption of linux in these institutions (like teaching and learning applications and resources, technical things like easy setup of thin-clients, and more). ( [www.edubuntu.org](www.edubuntu.org) )
