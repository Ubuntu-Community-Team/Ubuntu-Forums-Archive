### Post by zombifier25 on 2012-10-31
I don't think CrossOver is for about 90% of the users here anyway. The interface is designed so that even your grandma can use it, and the results are almost similar to what you'll get with Wine, but it comes with automated setups. If you want to take advantage of CrossOvers, use their automatic installations, manual installation of programs is like using plain Wine.
Also, it's officially supported so you can complain.

Don't get me wrong, CrossOver is good, I (and many people here) just don't need it.

### Post by Mark Phelps on 2012-10-31
BEFORE you try to use ANY Windows app in Linux, you need to go to the WineHQ application compatibility website and look up the app.

The page below lists the ratings for iTunes.  Garbage means it won't work, no matter what you do; Bronze means very little functionality will work:

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=1347]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=1347")
