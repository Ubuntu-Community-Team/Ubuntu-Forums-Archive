### Post by KBD47 on 2011-11-16
I've only seen the persistence option in Unetbootin available in Linux in the newest versions. I think it is new in the ubuntu 11.10 versions, but not in any Ubuntu's older than that. If you are trying to install Ubuntu or one of its derivatives then you can use startup disc creator in Ubuntu to make a usb with persistence.
