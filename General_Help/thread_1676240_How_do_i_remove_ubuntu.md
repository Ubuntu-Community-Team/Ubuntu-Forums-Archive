Not to worry! As long as you can boot into Ubuntu, you're safe.

Log into Ubuntu and go to Applications->Accessories->Terminal and enter:
sudo update-grub2

Then it will ask for your account's password (when you type, don't worry if no ****** appear as you type, that's normal). Press enter when you're done typeing. Tada! The next time you boot, an option to boot into Windows should be available.
