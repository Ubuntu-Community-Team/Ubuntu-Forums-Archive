Still, if you're going for stability.. think again.
From the wiki.
```
**[COLOR="Red"]Warning[/COLOR]**
E17 is Beta software, it's still in development, so it may segfault for no apparent reason 
at any time. This won't make your entire system freeze or shut down, after all.
You'll almost always be able to recover E17 (it never happened to me not to be 
able to recover it), so your work will be relatively secure and OK, but always remember 
that E17 is still a beta, so it's impossible for Geubuntu to provide a 100% stable version 
of this distro until a stable release of E17 happens.

```

Doesn't change the fact it looks damn fine!
