### Post by NilsE on 2008-07-30
Typically update manager will only try to install updates to what is already installed.  So, look in synaptic package manager and see if you have both version 2 and 3 installed.  Then remove 2 but make sure you backup your bookmarks etc. first just in case 2 and 3 are sharing them.
