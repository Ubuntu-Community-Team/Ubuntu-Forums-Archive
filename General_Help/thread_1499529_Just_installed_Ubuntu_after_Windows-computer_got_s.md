### Post by inameiname on 2010-06-02
No. You have nothing to worry about. When you installed Ubuntu it formatted your hard drive and erased everything to put Ubuntu on. Technically unless you truly erase it, there are ways of getting that stuff off there, but it'd be hard even for the government (hehe) and of course nothing you have to deal with. And even if the viruses did exist, Windows viruses simply can't do anything to Ubuntu or Linux. Everything about them is for a Windows-based computer.

### Post by WorMzy on 2010-06-02
How did you install Ubuntu? Did you install it, as inameiname presumed, by burning the iso file to a disk, restarting the computer and booting from the disk, and then installing Ubuntu over the entire disk (or just part of it); or did you install Ubuntu -inside- Windows, like an application?

If you installed it as a Windows App, then while you're booted into Windows, the files that Ubuntu depends of are just as susceptible to attack as any other; but if you booted the CD and installed as a partition, then you have (almost) nothing to fear. There aren't many wild viruses out there that target Linux operating systems, so you're almost completely safe.
