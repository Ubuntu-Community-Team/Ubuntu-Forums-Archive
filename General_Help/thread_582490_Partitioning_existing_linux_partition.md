### Post by bodhi.zazen on 2007-10-19
You need to manage your partitions from a live CD.

If you are already running a live CD, unmount the partition.

It the partition automagically remounts, 

Go to System > Preferences > Removable Drives and Media and uncheck the box(es) "Mount removable ..."
