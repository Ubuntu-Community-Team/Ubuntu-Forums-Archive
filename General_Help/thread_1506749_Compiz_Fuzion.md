### Post by charlie cat on 2010-06-10
I've been running it since Ubuntu 9.10 and I've only had one major problem that I've already filed a bug report for [here]("https://bugs.launchpad.net/ubuntu/+source/compiz/+bug/584734").  Vote for it if you can replicate it on your system.
To get it going, install compizconfig-settings-manager (sudo apt-get install compizconfig-settings-manager).  This creates a menu item, System->Preferences->CompizConfig Settings Manager that lets you configure everything.
Have fun!
