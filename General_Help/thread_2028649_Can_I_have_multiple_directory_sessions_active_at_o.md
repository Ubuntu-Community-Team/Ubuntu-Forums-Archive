### Post by niranjan_rao on 2012-07-18
Assuming you are using nautilus, it has new tab/new window menu items under file menu. Also if right click nautilus icon in launcher, you will get new window menu item.

### Post by Cheesemill on 2012-07-18
Just hit F3 while you're using Nautilus, this will give you a split-screen view where you can navigate to different directories in each half of the screen.
