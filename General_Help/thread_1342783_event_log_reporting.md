### Post by JKyleOKC on 2009-12-02
Have a look at "swatch" which is in the repositories (at least for Hardy LTS, which I have). It's a log file viewer that can pick interesting lines from various log files and send them to you via email.

I've not used it myself but came across it in a book on "Hack-Proofing Linux" that I bought several years ago, and the description there sounds like it might be what you are looking for...
