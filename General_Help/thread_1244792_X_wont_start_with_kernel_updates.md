### Post by riazrahaman on 2009-08-20
From xorg.conf it seems like you have installed the nvidia propriertary driver code and not using the driver that comes from apt-get.

You will have to check if there is a different driver for your latest kernel and download that. Rebuild and install as per the steps provided. If there isn't a new driver and rebuild the driver that was working with your previous kernel.
