### Post by amanchesterman on 2014-06-10
I looked into the possibility of editing .pub files a year ago and found that it's problematic in anything but MS Publisher: as monkeybrain20122 says, it's a format designed to lock you in to using that particular software product.

You don't say anything about the size or complexity of the labels you want to produce. If they are, say, A5 or A6 size (i.e. two or four to an A4 sheet) then you could produce them in LibreOffice Writer, using both text and images. But if they are like address or packing labels, i.e. self-adhesive labels, ten or more to an A4 sheet, only with text on them, then I recommend *glabels* which you can install from the Ubuntu Software Center. It's a lovely piece of software in my view, straightforward and adaptable: you can select from a wide range of predefined label templates, or create you own, and you can include as many fields of text (in various styles) as the label will hold.

### Post by amanchesterman on 2014-06-11
In LO Writer, if you follow the menu File --> New --> Labels, it takes you to a wizard that helps you create a whole page of labels by just typing in the text for one. I don't know how versatile it is - I don't know if it would handle borders, logos etc. as well as text - but it might be worth a look. (It has a 'database' option, for creating address labels from a file of names and addresses, which I assume you wouldn't need.) There's also a site with lots of document templates you can download for LO Writer and there may be one there with the size of labels you need.

Good luck anyway!
