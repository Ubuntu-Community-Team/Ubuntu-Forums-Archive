### Post by lukeiamyourfather on 2010-06-15
Likely there is no need to reinstall anything. Boot into Linux and share the results of these commands with us. This command will tell us what disks and partitions are in the system.

```
sudo fdisk -l
```

This will tell us what your current boot loader configuration is like. This is probably where the issue is, the option for Windows might need to be tweaked.

```
cat /boot/grub/grub.cfg
```
