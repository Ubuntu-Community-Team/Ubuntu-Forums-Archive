### Post by deadflowr on 2013-06-24
Ubuntu is released every six months, in April and October.
Every fourth release is an LTS, which comes up every April in an even number year.
The number scheme is a clue, with the year and the month, so 12.04 is 2012, and the fourth month April.

The only exception to this was dapper drake, which missed the April deadline and was release in june, so it's number was 6.06.
