### Post by t0n1 on 2008-10-23
Avant requires compiz to be running, that is the fancy visual effects.
Try turning it on: system -> preferences -> appearance -> visual effects and normal or extra.

### Post by pirattrev on 2008-10-23
this is a common problem newbies have and is also the reason many old machines can't do AWN at all, because compiz is needed. So if you actually _can't_ run compiz i'd recommend another dock program like wbar or cairo-dock. But that whole last bit was base entirely on speculation.
