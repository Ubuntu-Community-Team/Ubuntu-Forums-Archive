### Post by RikoW on 2006-02-23
Hi,

some time ago I played around with diagramming/graphing applications a bit, but since then didn't have much use for it, so my experience is about 2 years old.

But, there is a diagramming application within koffice (the office suite of KDE) call kivio, which comes with a set of stencils for different types of diagramms. Additional stencil can be purchases, but the program itself and the basic set is free. Kivio can be installed through the ubuntu repositories:

[http://www.thekompany.com/projects/kivio/]("http://www.thekompany.com/projects/kivio/")

Another program that I tried and which looked pretty good, as far as I remember is jgraph. It's java based and seems to be under rapid development/improvement. It is a commercial product, but has a free version which at least for my purposes was powerful enough:

 [http://www.jgraph.com/]("http://www.jgraph.com/")

I don't remember however, if any of them has the proper stencils for your purposes.

Hope that helps,

          Riko
