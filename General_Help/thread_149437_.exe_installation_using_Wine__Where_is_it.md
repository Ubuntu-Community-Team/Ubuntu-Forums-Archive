### Post by bluevoodoo1 on 2006-03-23
The c_drive is located in your .wine folder (which is a hidden folder in your home folder) Ctrl+h will let you see the .wine folder. From there you should see a drive_c folder. Within drive_c will be the Program Files folder.

Here's an exmaple of how to open an app with wine.

```
wine .wine/drive_c/Program\ Files/Finale\ 2003/FINALE.EXE
```

Use a \ before you have a space. So in a terminal "Program Files" should read "Program\ Files"

You can also navigate over to the .exe file with Nautilus, right-click it, and select "open with wine."
