### Post by lithopsian on 2011-03-21
Uninstall?  Not necessary.

Reinstall?  Very easy.  Burn or buy a CD, or copy an Ubuntu ISO file onto a bootable USB.  Install over the old one, or even next to it if you have spare disk space.

Keeping your data?  Tricky.  That depends where your data is.  If you have your data stored on a separate partition from Ubuntu then it should all be intact and will be there when you have installed a new version.

What's that?  Your data is not on a different partition?  Try booting from a Live CD and see if you can copy off any of the files, or fsck the dud partition and maybe get it working again.  There are more advanced data recovery methods but you'd have to be pretty desperate for those files.  Next time you'll create a separate partition, right?
