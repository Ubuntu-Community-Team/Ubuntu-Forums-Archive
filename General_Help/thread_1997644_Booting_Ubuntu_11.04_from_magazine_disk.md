### Post by Edmason on 2012-06-05
I have Ubuntu 11 .04 on a cd from The Complete Linux Manual. I have a Pentium 3 processor which has been running windows xp. I have tried to run Ubuntu from the disk but get the following message "this kernel requires an x86-64 CPU. Detected an i686. Unable to boot. Please use a kernel appropriate to your CPU". The magazine suggests it should be possible to boot from virtually any old computer. Can you help.
Ed Mason

Unless it is an Intel 64 architecture (you have a i686), you can't boot with a 64bit Ubuntu. 
You can't use Ubuntu x86_64 version, so you will have to download the 32-bit version as your hardware supports it.
Select 32bit from the drop down and hit start download .
[http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop)
