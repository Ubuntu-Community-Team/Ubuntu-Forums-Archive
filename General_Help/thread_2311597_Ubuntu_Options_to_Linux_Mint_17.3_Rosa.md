### Post by Frogs Hair on 2016-01-28
If the CPU is 32 bit then any 32 bit version of Ubuntu would work . If the Mint version installed is still supported with security updates you may want to keep it. To make sure you have 32 bit CPU run the following command. ```
uname -a
```

### Post by Dennis N on 2016-01-28
Rosa is based on Ubuntu 14.04 LTS and uses (in part) the Ubuntu repositories. Keeping it depends if you like the Desktop Environment it has - Cinnamon? MATE? XFCE? It doesn't have an official Unity Desktop option, so if that is your preference, go to Ubuntu. The support period is the same as Ubuntu 14.04 - to 2019.
