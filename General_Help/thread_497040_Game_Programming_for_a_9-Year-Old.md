### Post by Kodfish on 2007-07-09
easy dude. easy. never turn away a futre geek from something he likes. sure physical activity is required, but you assume too much. calm the **** down dude. 

OK, so game programming is pretty much either: 
a) using pre written engines, and scripting simple activities, or 
b) using a pre written api to do modeling and work and stuff.

i actually do not reccomend ubuntu for either, but i do not have much expierance. look into microsoft's XNA, i think that is an api driven foundation. it's hard, and a lot of c and c++, but using prewritten stuff is really crummy. i did it as a kid and did NOT enjoy it at all. then again, learning an entire language can be really daunting if you just want to do some "game programming", but thats what it really is. my cousin wrote a nifty little screensaver in 5 lines with XNA, if i recall correctly. learn with him if you can,at you  don't give up.

it's really not easy, but it gets better once you learn more and more. explain it to him like math. you learn how to add, then subtract, and then multiply, and divide...etc, but then after that, hard as it is, addition is easy.

above all else, keep with it.

Well if he likes to Play Windows Games, try Wine

```
sudo apt-get install wine
```

After installation refer to my Tutorial for Wine-Doors: [http://ubuntuforums.org/showthread.php?t=490382](http://ubuntuforums.org/showthread.php?t=490382)

Or for a free CVS Compiled Cedega search in the Gentoo wiki for Cedega (and you shall find).

Also theres a crapload of games  and other useful stuff on [www.getdeb.net](www.getdeb.net), espically 3-D games.

Also there is  Battle for Wesnoth (a great strategy game) open Synaptic:

and search for Wesnoth and install all the packages you see.

Your kid will love Battle for Wesnoth!

### Post by meatpan on 2007-07-12
Great Question!  Programming can be fun and exciting for children, as well as good bonding time with parents and a motivation to keep up with school work.   I have some great memories of my Mom teaching my Bro & I how to program in Basic on a TI-4a.

Code-rally and Robocode are two game platforms that children can probably learn (with some help of course).  IBM offers both, and they are open-source:
[http://www.alphaworks.ibm.com/tech/coderally](http://www.alphaworks.ibm.com/tech/coderally)
[http://www.alphaworks.ibm.com/tech/robocode](http://www.alphaworks.ibm.com/tech/robocode)

You build little cars (or robots), and program their functionality on a course.  As you get better you can add more features and eventually compete with other programs.

Lego Mindstorms aren't really a game, but a lot of children really enjoy them.  One of my coworkers has mindstorm running via ubuntu (not sure how he did it.  I'll ask him).  The tutorials are great, and the kids get instant feedback by watching the robot perform various movements and tasks.   Mindstorms are a bit expensive, and you might find it tough to share ;)  I currently use a different OS for my personal mindstorm.
