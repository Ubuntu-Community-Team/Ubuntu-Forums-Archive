### Post by Eismagier on 2011-10-16
Before upgrading to 11.10, I've always used /etc/gdm/Init/Default to run the synergy client at the login screen. Since 11.10 ditched gdm for lightdm, this method obviously does not work anymore, and I've been unable to find an equivalent for lightdm.

Does anyone have a solution for automatically running synergyc before login?

---

### Post by vincentdenancy on 2011-10-17
remove all gdm modifications and add the following line to /etc/lightdm/lightdm.conf

greeter-setup-script=/usr/bin/synergyc HOSTNAME

---

### Post by Eismagier on 2011-10-17
Perfect, thank you!

### Post by yohoho46 on 2012-08-25
Since I wanted synergyc ran as login, I added the following line to startup applications.  My computer name is different than what I used for setting so I added the -n wilson.  My host is running on a static ip so I used that rather than the computer name.

synergyc -n wilson 192.168.20.50
