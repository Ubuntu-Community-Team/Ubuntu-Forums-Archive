### Post by tgalati4 on 2013-02-07
```
man xset
man xinput
apt-cache search mouse scroll
```

The first controls the mouse in all applications.  The second can be customized per application.  Use with care and take note of your default settings.  The third brings up a few vendor-specific tools, but this might help:

gpointing-device-settings - configuration tool for pointing devices
