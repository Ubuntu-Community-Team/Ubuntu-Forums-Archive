### Post by wolfen69 on 2008-09-12
> I have dual screens. Can I use dual screens in Ubuntu like Windows xp?
yes.
> I play ALOT of games. They range from command and conquer (whole series) to Unreal tournament 3 and various steam games. Can I still use these in Ubuntu.
check [here]("http://appdb.winehq.org/") to see if wine supports those games.

GIMP is a photoshop like app. flash works fine. there are many winamp type media players. skype has a linux version. don't know what x-fire is. for instant messaging, there is pidgin, which can do all major protocols. there are daemon tools equivilants.

you should have no problems with that card.
> Also, with virtual box (or similar product, please tell) can I assign all the ram and Video ram to a windows boot?
you can't assign *all* the system's resources for windows, but depending how much ram you have, you can assign enough to do most anything.
