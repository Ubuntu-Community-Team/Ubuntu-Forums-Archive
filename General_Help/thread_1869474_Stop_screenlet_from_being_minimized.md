### Post by Sun_Master on 2011-10-25
I'm using a calendar screenlet on my desktop and frequently use the 'Hide all normal windows and set focus to the desktop' shortcut. The problem is that I want the calendar to always be visible even after hiding all other windows. Is there a way to set the screenlet to unhideable or maybe modify the shortcut to hide all windows except the calendar?

### Post by CatKiller on 2011-10-25
I think it's the Skip Pager window option that does it. One of them, anyway. You can change the window options on the screenlet itself or with Screenlet Manager. You can change window management behaviour with CCSM.

### Post by Sun_Master on 2011-10-25
CCSM does have a set of window rules including one which prevents windows from being minimized, but that's overridden by the show desktop shortcut. The only thing the window rule does is remove the minimize button, which the calendar doesn't have in the first place. Thanks anyway.
