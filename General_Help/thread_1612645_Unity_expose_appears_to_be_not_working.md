### Post by elamd on 2011-04-20
Since Google led me here, I'll chime in.

In 11.04, the keyboard shortcut for OSX Expose style is shift-alt-up_arrow.

Personally, I like to have it mapped to a hotspot on the screen( lower left corner).  You can customize the initiation behavior by installing the CompizConfig settings manager.  The plugin that mimics Expose is called Scale(Window Manager -> Scale).  It has settings for hotspots, keyboard shortcuts and all kinds of stuff.

Cheers!
