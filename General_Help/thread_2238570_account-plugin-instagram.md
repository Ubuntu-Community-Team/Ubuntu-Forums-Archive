### Post by Mr.Dunham on 2014-09-12
This post is 1 month old so I don't know if you're still around, but anyways this could be useful for others.

*account-plugin-instagram* is just a plgin for the *Online Accounts* panel, and *friends-instagram* is the integration of Instagram into the Friends-app, although I never see anything from Instagram on my Friends-app.

That's all, there's no actual desktop integration to Instagram. No Unity webapp, no Firefox integration from it.

There are a few Firefox add-ons to browse profiles and photos (like [this one, click here]("https://addons.mozilla.org/en-US/firefox/addon/instagram-for-firefox/")), even a stand-alone software known as Stogram that does the same under Linux.

But there is no way to upload photos from our computer under Linux; well, maybe if you're willing to try Windows software under Wine, there's at least one called Gramblr.

Hope this helps anyone in some way...
