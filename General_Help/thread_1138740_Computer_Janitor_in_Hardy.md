### Post by linuxuser21 on 2009-04-26
I've only seen it in Jaunty.  I don't think you can get it for any other versions because Jaunty (as do the other versions) has different packages than the others.  In other words: let's say there is a crucial file that is needed in Hardy, the Janitor might see it as a unessential file because Jaunty doesn't use it.
