### Post by ADT on 2007-07-17
There is a very good modification of wikipedia geared towards children called [wikipedia for schools ]("http://www.soschildrensvillages.org.uk/charity-news/wikipedia-for-schools.htm")

I suppose teachers want children to gain skills at researching. Wikipedia is very easy to use so it doesn't allow children to develope researching skills to the extent that locating books and journals in libraries achieves.

I find tracking down the books, etc that are referenced by wikipedia articles good resources. If a teacher wont accept work referencing wikipedia directly maybe wikipedias own references would be better.
