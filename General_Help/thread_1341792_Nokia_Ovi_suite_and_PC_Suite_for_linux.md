### Post by nuunkotad on 2010-11-22
I can confirm that series60-remote works well with nokia N82 (symb^2 v35.0.002) as well with Nokia 5230 (v.unknown)
***works over bluetooth***
***_functions:_***
*edit calendar, phonebook, sms
*export contacts, messages HTML, CSV, vCard, ldif (calendar export can't get active - *maybe thru support forum it's possible)
*import contacts and messages
*browse phone and card memory (this is also possible through bluetooth applet on GNOME))
***_issues:_***
*does not present images added to contacts (only one from many was presented - don't know if this is problem for any other user)
*some mismatch with sms presenting and statistics on PC after another device (nokia 5230) was connected

suppose that there is no option to connect with data cable :( but as far as there is option to buy bluetooth receiver on dealextreme for some 1-2$, this is not an problem any more


***_install_*** (Application version: 0.4.80 (0.5.0 - Alpha 1)
quite easy on ubuntu 10.04
