### Post by wildmanne39 on 2011-12-25
Hi, with the livecd you should copy the files to an external hard drive, or to another internal hard drive but it will not say C: drive ubuntu will show it in the file manager as media storeage or the size of the partition that is mounted, you can check with gparted the names of the partitions.

Anything you save or install while running the livecd is just stored in memory so when you reboot it will be gone.
