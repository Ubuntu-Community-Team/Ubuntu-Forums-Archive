### Post by ad_267 on 2009-10-16
It's impossible to know how a web page will look in other browsers unless you actually install that browser. The operating system a browser is installed on can also sometimes cause differences in rendering. No program is going to be able to replicate all browsers.

Yes it's a pain, and I'm sure it's mostly crappy support from browsers rather than your CSS skills that is to blame, its a problem for all web developers.

I suggest trying [http://browsershots.org/](http://browsershots.org/).

You could also install Windows using Virtualbox if you have a decent amount of RAM and a spare Windows license.
