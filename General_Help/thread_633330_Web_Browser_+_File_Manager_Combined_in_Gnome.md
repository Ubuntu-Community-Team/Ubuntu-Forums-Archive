### Post by p_quarles on 2007-12-07
Konqueror is the only program that I know of that combines web browsing with file management. If you won't install Qt libraries, then I don't see any options.

The package "pcmanfm" provides tabbed file management for Gnome. Is this what you're referring to when you say pc-file manager? If so, I think that's the best you can do. Personally, I love Konqueror, but I've heard that many Gnome users really dislike the idea of one application performing so many functions, so I doubt there's much chance that a similar app will be developed any time soon.
