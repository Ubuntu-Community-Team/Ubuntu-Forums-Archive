### Post by mechro on 2010-02-13
There isn't a "Date Created" list view in Nautilus.

You could try installing **gThumb** (available via Synaptic) which has a view "by Exif Time" option.

### Post by mechro on 2010-02-13
If you're referring to the Eye of Gnome picture viewer which opens when you "click to open" then I'm afraid it doesn't use the Nautilus view settings. Sounds silly and buggy but that's the way it is. 

Again, a solution would be to install gThumb and set Nautilus to open gThumb when you "click to open" a picture.
