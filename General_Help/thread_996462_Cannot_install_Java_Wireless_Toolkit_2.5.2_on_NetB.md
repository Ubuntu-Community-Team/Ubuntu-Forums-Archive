Gee, I don't remember exactly. It was a pretty long time ago... What I'm sure of is that I couldn't install the Wireless Toolkit by those means (plugin manager). I believe I reinstalled it a bunch of times using different methods (to no avail) and finally realized that somewhere along the installation of the full IDE package (downloadable from the net beans web page), there was an option of which packs are installed with the IDE. There you could choose to directly install the WTK with NetBeans.
I don't have that computer with me right now so I can't give you a detailed answer, but as soon as I get it back (probably within the next few days) I'll look around and surely remember more to give you better answer.
Just... don't waste your time with either the incorporated "Add/Remove Software" of Ubuntu, the Synaptic Manager or the Plugin Manager of NetBeanas. Those are dead ends... I'm almost sure the answer is somewhere in the NetBeans website (albeit obscure) and it has something to do with the installer or the package that you download to install the IDE.
