### Post by cariboo on 2008-12-20
I would suggest that you finish downloading the iso and burn it to a cd then boot your computer from the LiveCD, this is a feature that allows you to try the distribution, before you install it. It would affect your Vista installation at all. I would suggest installing it on a computer before trying to install it on your Wii. There are two methods of installation, one is called wubi that allows you to install Ubuntu inside of windows, and the other method installs on it's own partition. I would suggest running the LiveCD for a bit to see if Ubuntu is for you.
