### Post by Mark Phelps on 2010-09-07
If you ARE able to figure out HOW to do that (use ALL the space), you will break Windows and render it unbootable -- and that will also prevent you from booting into Ubuntu.

MS Windows needs free space to allocate temp files and logs.  15% of the partition size is a good rough estimate of how much space to leave unused.
