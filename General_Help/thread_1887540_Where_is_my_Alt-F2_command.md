### Post by dabl on 2011-11-28
F-lock = Function L`+
ock.  Some keyboards, like a Microsoft Natural Ergonomic (and many others) have a F-Lock key that needs to be set to activate the function keys.

Also, check your BIOS -- sometimes you have to set F-Lock as well as Num-Lock to be "on" at bootup.
