Are you talking about doing web development or developing stand alone applications? Ubuntu is not the same as Debian, although it is based on it.  But for web development, it shouldn't be a problem because everything will run through the browser or the engines that are installed on the web server.  For application development, you might run into library version problems but even they wouldn't be unsurmountable if you actually recompiled your code on the production box.
