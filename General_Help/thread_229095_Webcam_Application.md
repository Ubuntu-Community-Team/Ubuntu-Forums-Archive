Unfortunatelly this task is not as user-friendly as in Windows. But if you are familiar with the console, it should be acceptable. Review the following website to find something for you:
[http://www.linux.com/howtos/Webcam-HOWTO/framegrabbers.shtml]("http://www.linux.com/howtos/Webcam-HOWTO/framegrabbers.shtml")

"motion" is my personal favourite because of easy configuration and powerfull features (not only motion detection, but also recording costant video, broadcasting stream into the internet etc.) Howewer I'm not sure if it can capture sound, so if you need it you may need sth else.

If you need GUI application , read VLC media player website. I have some success with recording V4L wideo with VLC but it was not so reliable as "streamer" or "motion".
[http://www.videolan.org/doc/streaming-howto/en/ch02.html]("http://www.videolan.org/doc/streaming-howto/en/ch02.html")
