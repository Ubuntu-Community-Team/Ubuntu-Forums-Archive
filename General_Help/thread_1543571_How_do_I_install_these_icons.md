### Post by Smart Viking on 2010-08-01
You download the files, and then you move the folder with the stuff in it to /usr/share/icons.

then they will be available in "System->Preferences->Appearance->Costomize->Icons". Hope this helped :)

### Post by conradin on 2010-08-01
goto system --> preferences --> appearance.

then goto:
[http://www.deviantart.com/download/173323228/Faenza_Icons_by_tiheum.zip](http://www.deviantart.com/download/173323228/Faenza_Icons_by_tiheum.zip)

to download the files. Then extract them someplace.

in the appearance window, click install. then locate the extracted files. When it ask, select apply new theme.
