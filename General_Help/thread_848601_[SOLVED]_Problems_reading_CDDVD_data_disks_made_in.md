### Post by mc4man on 2008-07-03
You probably used vista's default burning mode (live system) which uses udf 2.01 and or 2.50 neither of which is supported in ubuntu. There is a way to enable udf 2.50 (patched to support 2.01), but i don't think you want to get involved in that yet.
If you have access to a Vista or Xp (for 2.01 unless 2.5 driver was installed) you could read, copy and re burn making sure if using Vista to choose 'mastering' mode. (or tranfer to a usb flash drive)
If you are/become interested in the udf 'fix'
[http://ubuntuforums.org/showthread.php?t=775775](http://ubuntuforums.org/showthread.php?t=775775)  post 10 (link and patch)
