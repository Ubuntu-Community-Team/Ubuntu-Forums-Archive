### Post by pan69 on 2010-10-01
The default desktop that comes with Ubuntu is called Gnome. Gnome can be customized greatly. Have a look here: [http://art.gnome.org/themes](http://art.gnome.org/themes)

As an alternative you can switch desktops entirely, e.g. you could switch to KDE ([http://www.kde.org](http://www.kde.org)).

Both these desktops use the underlying window manager called X (or X11 a.k.a x.org). E.g. the Apple OSX user interface is also built op top of X (OSX is a version of BSD).
