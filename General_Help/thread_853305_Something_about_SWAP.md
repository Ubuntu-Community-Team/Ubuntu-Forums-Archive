### Post by Titan8990 on 2008-07-08
Memory is by far your superiour to virtual memory on a hard drive. It will not use swap space until 100% of your RAM has been used. If it used it earlier you would see a performance hit.


Edit: RAM is dirt cheap these days. You can get 1GB for < $30.
