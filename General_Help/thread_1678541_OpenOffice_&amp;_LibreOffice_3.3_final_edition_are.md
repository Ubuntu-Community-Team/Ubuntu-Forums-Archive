### Post by mcduck on 2011-01-30
> **gdiloren said:**
> Simple question! When are they going to be added in the repository. The tar files are available for download on their official sites but I want to install it the secure way via update manager. Thanks.:lolflag:

When the next Ubuntu verisonis released.

The basic rule in Ubuntu is that after release, all package versions are frozen and packages will only be updated to provide security updates and fix serious bugs. Never just for new features or to ave the latest version of a program. (the backports repository does give you *some* updated packages, backported from the current development release, but the rules for what can be backported this way are pretty strict)

You'll probably find a third-party repository or a PPA repository with packages for latest OpenOffice/LibreOffice.
