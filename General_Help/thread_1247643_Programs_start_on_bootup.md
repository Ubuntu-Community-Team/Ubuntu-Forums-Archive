### Post by SuperSonic4 on 2009-08-23
Were those programs in use when you shut down the PC?

KMenu --> System Settings --> Advanced Tab --> Session Manager --> "Start with an Empty Session" --> Apply

That will give you only the programs specified in ~/.kde4/Autostart
