### Post by XubuRoxMySox on 2009-11-27
Open Synaptic Package Manager and find HPLIP. If it's not already installed, install it! It's a huge help.

Afterwards if you want to install your printer you can either open HPLIP or open a web browser to [http://localhost:631](http://localhost:631)

It's a graphical representation of HPLIP. Click on **Administration** (you will be asked for your computer's username and password) and see if your printer is listed. If not, select **Add Printer** and select yours from the drop-down list.
