I'm guessing that you're referring to a download of Ubuntu. You'll need to "burn" it to CD - make sure that it is "burn image", and NOT as a data disk. You'll then need to boot your computer with the CD in the disk drive.

Have a look at [https://help.ubuntu.com/community/Installation](https://help.ubuntu.com/community/Installation) for more information
