### Post by rainwalker on 2007-05-25
I've noticed (on Edgy, it didn't do this on Dapper) that if I'm using a program and I want to open a file, the configuration files (the .whatever ones that are hidden) are listed in the "Open" dialog box. This is really annoying because they're always listed before my normal files/directories, so I have to scroll through everything to find the stuff that before was just there by itself.

Is there a way to set Edgy so it doesn't show these hidden files?
I know in the file browser you have to choose View -> Show Hidden Files to see them, which is basically what I want here; for them to be hidden.

Any suggestions?

---

### Post by aysiu on 2007-05-25
Right-click inside the open dialogue box. You'll see a context menu where *Show hidden files* is checked off. Select it to uncheck it.
