### Post by mjuhasz on 2011-10-30
It's a known issue that global appmenu breaks the focus-follows-mouse feature. There has been a [long discussion]("https://bugs.launchpad.net/unity/+bug/674138") about it, even some [specification]("https://wiki.ubuntu.com/MenuBar#focus-follows-mouse") was created about how focus-follows-mouse might be catered for with the global menu bar but as far as I know nothing has been implemented for 11.10
