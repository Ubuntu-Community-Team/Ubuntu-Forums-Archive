### Post by zvacet on 2009-04-13
In system>preferences<main menu check under internet group.Maybe it is there but unchecked.If it is not there and you are sure that you installed it then make launcher.From left side select internet and on the right side select new item.That will create launcher and make it look like this

 Type Program
 
 Name Opera
 
 Command opera %u
 
 Comment Web Browser

You will find icon in /usr/share/pixmaps
