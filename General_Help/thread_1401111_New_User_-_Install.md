### Post by marshmallow1304 on 2010-02-07
Yes, you may install Ubuntu in your D: partition (will erase data in that partition), or you may use Windows' disk utility to create a new partition and install there.  The Ubuntu installer can resize and create partitions, but it seems that Windows 7 doesn't appreciate other utilities editing its partitions.

The Ubuntu installation will provide a boot loader, so that you may choose between Ubuntu and Windows when you start your computer.
