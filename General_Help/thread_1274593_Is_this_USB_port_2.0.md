How do I tell if my usb ports are operating as USB 2.0? They are supposed to be, but as I'm transferring some files to an external USB HDD, the transfer rate is only 10.5 MB/sec.

Ideas?

---

### Post by StuartN on 2009-09-24
> **knottshawk said:**
> the transfer rate is only 10.5 MB/sec

USB 1 transfers at 12 Mbit / sec compared to USB 2 at 480 Mbit / sec - these are 1.5 MByte and (theoretically) 60 MByte / sec. Getting an actual transfer rate of 10.5 MByte / sec from a spinning disc is okay, and you might get 30 MB / sec from a USB memory stick.

So yes, you have USB 2.
