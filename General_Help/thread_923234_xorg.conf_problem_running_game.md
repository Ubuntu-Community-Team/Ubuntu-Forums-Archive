### Post by Nepherte on 2008-09-18
Next time if you want to reconfigure your nvidia xorg configuration file, use:
```
sudo nvidia-xconfig
```

If you use
```
sudo dpkg-reconfigure xserver-xorg
```
it will switch to the nv driver and not nvidia.

P.S That nvidia xorg.conf file looked ok to me. xorg now hides much information, that's the reason why it looks so empty.
