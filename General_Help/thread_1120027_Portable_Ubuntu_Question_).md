### Post by mb_webguy on 2009-04-10
You can use the LiveCD to run Ubuntu on any computer by booting from the CD, but you're limited to the applications on the disc, since it's not persistent (meaning you can't save any changes).  You can remaster the LiveCD to add or remove applications to suit your specific needs, but it still won't allow you to save files without some additional storage, like a USB drive.

A LiveUSB is persistent, and therefore a better option.  You're still limited to the applications on the LiveUSB, but you can make changes to the settings and preferences that will persist from boot to boot.  You can also save files to the USB drive.  The downside is that not all computers can boot USB, and many of those that can require you edit the BIOS settings to enable it.

You can also install Ubuntu to a USB drive.  Installing to a USB external hard drive is essentially the same as installing to an internal drive, though to avoid messing up your existing installation, it's best to unplug your internal drive(s) before doing this.  Installing to a USB flash drive is a bit different, but still possible.  Again, the main downside to this is that not all computers can boot from USB, or are configured to do so.
