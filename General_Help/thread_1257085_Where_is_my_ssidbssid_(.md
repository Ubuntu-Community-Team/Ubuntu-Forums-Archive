### Post by boof1988 on 2009-09-03
> **hakermania said:**
> well...what's the command i have to write?

If you're computer is already connected to the wireless router... Then you may be able to access the router's configuration by using the following:
[LIST=1]
[*]Open up your preferred web browser.
[*]Type the following address into the browser's address block and then press enter.  (Note: You can also try clicking on the address below, if it is the correct local LAN address for your router, and it will take you to your router's configuration page)
> [http://192.168.2.1/](http://192.168.2.1/)
[*]If that doesn't work, then try looking up your router's manual, which should indicate which address should be used in the previous step.
[*]The previous steps should take your browser to the router's setup/configuration page.  You should be able to log in now and check the SSID etc.
[/LIST]

Hope this helps some...
