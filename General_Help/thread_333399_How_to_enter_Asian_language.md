### Post by paker on 2007-01-07
This is what I have done so far.
System>Administration>Language Support>Korean
System>Preferences>SCIM Input Method Setup>(clicked a few buttons)
Right clicked on top bar>Add to Panel>Utilities>Keyboard Indicator

Now I have "USA" in the top bar. When I click on it, it changes to "KOR." But typing continues to be English. Thank you.
