### Post by Krydahl on 2008-10-20
You need to be in the directory above where the hidden files are. Then right click on the directory that holds the files you're interested in. You should see an option to show hidden files. Once you've clicked it, go into the directory and you should now see your files.

This assumes you're using the gnome dialogues not the openoffice ones. It's also based on OOo 2.4, I suppose they might have changed something in 3.
