### Post by tgalati4 on 2007-08-26
Mac has Stickies,  Ubuntu has TomBoy.  Perhaps a clever person out there could write a script to translate between the two.

I looked for TomBoy on Fink Commander (a package manager that ports open source packages to the Mac) but didn't find it.

If you have iPods, you can transfer notes between machines.  They are simple text files.  The other possibility is to use a personal wiki page--gtd wiki--do a google search.  There's a lot of info at 43folders.com.

Basically you put a wiki page on a shared directory then each machine can see it and edit it.  It's designed for to-do lists using the Getting Things Done (trademark) method.  Sharing folders between Macs and Ubuntu is easy.
