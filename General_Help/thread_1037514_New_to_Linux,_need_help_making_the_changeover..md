### Post by andresmh on 2009-01-11
The LiveCD comes with an option to just try Ubuntu without installing. You should download the ISO file, burn it and then boot from it. One thing that happened to me on a Dell laptop was that the wifi card was not recognized by the LiveCD "try without installing" option but after installing Ubuntu for real it worked. It was probably just something it downloaded automatically but I was pleasantly surprised.

Another thing you can try without removing Windows is Wubi. It's Linux running on top of the Windows file system. It works great and performance is almost the same as having it fully installed. 

These forums are really helpful when you're starting. I hope your transition from Windows goes well :)
