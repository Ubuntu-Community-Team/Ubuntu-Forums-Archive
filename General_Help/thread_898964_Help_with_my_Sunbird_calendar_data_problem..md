### Post by jhaiduce on 2008-09-15
> **souljahh2050 said:**
> Can anybody help?

I had the same problem; you should be able to run Sunbird by typing "sunbird -ProfileManager" in a shell. This will allow you to create a new profile and use that new one.

Unfortunately this method involves starting over with a new profile and losing whatever calendars you already created in Sunbird. However, the old profile will still exist (it's just no longer default), so may be able to recover data from your old profile later.
