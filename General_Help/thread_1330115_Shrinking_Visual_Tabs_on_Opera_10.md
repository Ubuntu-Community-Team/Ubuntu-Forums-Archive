### Post by Infinite Indefinite on 2009-11-18
When I installed Opera 10.01.4682 on my Acer Aspire 3100 running Ubuntu 8.04, I found that expanding the visual tabs tended to shrink the tab, and only rendered a thumbnail of part of the screen.

I later discovered that this problem did not occur with the Opera Standard Skin, but did occur with the other skins.

It seems to me that the previous skins (i.e. those made for earlier versions of Opera) are not fully compatible with the new visual tab function.

Anyway, if one downloads newer skins (or updated versions), this problem does not occur.

I'm not sure if anyone else has faced this problem, but it may be of interest to all users of Opera.
