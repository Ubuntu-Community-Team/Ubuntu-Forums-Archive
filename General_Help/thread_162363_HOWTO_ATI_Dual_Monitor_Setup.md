### Post by djsroknrol on 2006-04-19
I'm using the OSS Radeon driver..If you look at the man radeon pages, you can find out more about CRT2Position...It tells X where the second screen is. All I know is that there is no need for a reference to the second monitor with CRT2Position as an option in the device section as you're telling X that there is a second monitor already. All you do is define the two devices, the two monitors and screens and then set up the "LayoutServer" and it works. Good luck with yours...:)
