### Post by mehtdosa11 on 2008-06-13
hi i loaded ubuntu onto my dell inspiron in march and have been enjoying it ever since. if you mean by the 'downloadable' ubuntu you have to burn the downloaded .iso image to a cd/dvd then boot and install from that. [if you know all this already please accept my apologies!}

ubuntu should load everything from that, it will tell you if you need extra proprietary drivers, i.e for the ati graphics card and broadcom wireless chip. but it downloads them for you and installs them itself. so it wasn't a problem.

all in all quite painless..almost.
