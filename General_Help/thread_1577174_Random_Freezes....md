### Post by HermanAB on 2010-09-18
Hmm, intermittent hardware failures can be hard to track down, unless you have a junkbox full of spares that you can swap out.

I usually start by inspecting the motherboard for bad distended/leaking capacitors and frozen cooling fans, then replace the PSU, then the ethernet card, then the RAM and try to narrow it down that way, but even a bad keyboard can cause a machine to freeze up.

If the MB has bad capacitors, then you need a new computer - sorry.  Fixing a MB is generally not worth the trouble even if you have the tools to do it.
