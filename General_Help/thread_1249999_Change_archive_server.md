My default Ubuntu download server is [http://hk.archive.ubuntu.com](http://hk.archive.ubuntu.com).  It is painfully slow.  Is there some way I can change my default server to something else?  How do I do this and is there a list of ubuntu archive servers?

### Post by hgraham on 2009-08-26
Ok I found out how to do it.  Go to System/Administration/Software Sources/Ubuntu Software/click on Download from/other/Select Best Server/Choose server   This will reassign the server for updates and package installation.
