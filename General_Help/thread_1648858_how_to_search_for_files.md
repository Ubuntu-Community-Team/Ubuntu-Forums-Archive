### Post by oldos2er on 2010-12-19
If you need a GUI, try catfish, it's a front-end for **find** and **locate**. The **find** command is very powerful, and worth the time it takes to learn. [http://en.wikipedia.org/wiki/Find](http://en.wikipedia.org/wiki/Find)

### Post by thameera on 2010-12-19
Applications -> Accessories -> Search for Files...
You have to select the drive/location you want to search

### Post by thameera on 2010-12-19
Sorry, I only saw later that you've already tried that. In case you need to search all drives, select Other... from the drives list and choose /media
