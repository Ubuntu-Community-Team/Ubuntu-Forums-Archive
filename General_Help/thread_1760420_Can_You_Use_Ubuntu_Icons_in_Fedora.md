### Post by Tamlynmac on 2011-05-16
I believe the only requirement for that [icon theme]("http://linux.softpedia.com/get/Desktop-Environment/Icons/Ubuntu-Mono-Dark-Elementary-61097.shtml") (see requirements at bottom of page) is Gnome. When one clicks on the requirement "Gnome" it directs to Gnome 3.1.1. I have no idea if it works with Gnome 3 (Fedora 14/15), but suspect it probably will.

I'd suggest you just install it and see if it works.
