Thinking to be clever, I made some changes to /etc/X11/xorg.conf and now I cannot see anything on the screen after Ubuntu boots up, just a 'mode not supported' message on the screen.

How can I log into Ubuntu in text mode in order to fix the xorg.conf problem? I hope there is a "hot key" to stop Grub right before it starts loading Ubuntu and allows me to log in into Ubuntu in text mode.

### Post by ibuclaw on 2010-01-09
For Pre-Karmic (Grub0.97), you press/tap the **Esc** key.

For Karmic and onwards (Grub2), you press/tap the **Shift** key.

Regards
Iain
