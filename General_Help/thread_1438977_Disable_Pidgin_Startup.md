### Post by nstannik on 2010-03-25
Hi all,

I'm running Karmic on a new-ish Dell laptop: 3GB RAM, plenty of HDD space, etc. My problem is this: I rarely use Pidgin and don't want it to run at startup. (At all! Not minimized, not hidden - actually not running until I click the application icon). 

This seems pretty simple; unfortunately, Pidgin doesn't have an option for this, and there's no Pidgin entry in the start-up programs list. I've tried Ubuntu Tweak to no avail. Pidgin shows up whether it was running when I shut down or not. I've checked a number of posts - the few that have this exact problem can't seem to find a solution or workaround.

I don't want to uninstall Pidgin, as it has a lot of my settings saved and I do use it occasionally. That being said, it's really annoying to have to close it 19 times out of 20 when I start up.

Thanks in advance for any help,

-Nils

### Post by howefield on 2010-03-26
Try this,...

Close all running applications then go to System > Preferences > Startup Applications, and click on the Options tab, check "Automatically remember running applications when logging out".

Close and reboot.

Do it again, only uncheck the box this time.
