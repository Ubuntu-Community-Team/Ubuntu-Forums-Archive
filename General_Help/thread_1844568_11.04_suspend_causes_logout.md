### Post by nathany on 2011-09-16
Problem fixed!  Thanks so much for all the help.

In case anyone ever happens to come across this thread in search and the previous link is dead, adding
```

Option "NvAGP" "1"

```to xorg.conf can fix X crash on resume with nvidia driver and some AGP chipsets.
