### Post by BenAshton24 on 2009-11-14
Giblet5 gives you a pretty solid method for learning a programming language so long as you are trying and failing at creating programs along the way. However it is also a quite boring method :P I started off by learning HTML and CSS to get into the coding mind set before I did any programming. After that I moved on to JavaScript and then I learnt a lot of C++ and every so often I would have a break through and suddenly understand a concept that I didn't get before. Since then I have also got good at PHP and Python, both of which are relatively easy to pick up after having done lots of C++.

I did all of the above without reading a single book. I instead learnt everything by following tutorials that I found across the internet. There are a few downfalls to this method, the main one being that you don't really get any idea of good programming practice, but it all comes eventually after you face-palm yourself a few times when someone tells you that you have been doing it wrong for years :P

When following my method you will probably feel that you are in the dark constantly and then all of a sudden you will grasp the general programming concept and everything will make sense and slot into place :D Good luck with your programming journey.

Ben.
