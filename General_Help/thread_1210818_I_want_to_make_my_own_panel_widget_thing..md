### Post by jrox717 on 2009-07-12
To add something to a panel, right-click on an empty part of the panel and select "Add to Panel." To add a new launcher, select Custom Application Launcher and give the link to the command. Alternately you can drag and drop the application from nautilus, your desktop, or the main menu to the panel.
