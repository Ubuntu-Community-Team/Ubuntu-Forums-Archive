### Post by stevepetmonkey on 2010-05-01
When I start rythymbox it only starts in the top panel in the notifications area, is there any way to have RythymBox start full screen, not a major problem but I like to have the panels hidden so its a bit of a pain to have to start the app then find it on the panel to use it.
:confused:

---

### Post by WorMzy on 2010-05-01
Change the shortcut to run 'rhythmbox-client %U'. That starts rhythmbox and the GUI simultaneously.

I've never understood what the %U does, but hey, it's there by default, so I leave it alone.

(To edit the shortcut, right click on menubar [Applications|Places|System] and click "edit menus", then navigate to Sound and Video, and double click on the rhythmbox shortcut.)
