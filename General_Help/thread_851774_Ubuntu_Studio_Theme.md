### Post by stchman on 2008-07-07
Hello all I just discovered that Hardy has the Ubuntu Studio theme in the repos.

This is one of the coolest themes for Ubuntu.  It gives Ubuntu that very expensive look.

Here is how I installed it.

```
sudo apt-get install ubuntustudio-theme
sudo apt-get install ubuntustudio-gdm-theme
sudo apt-get install ubuntustudio-icon-theme
sudo apt-get install ubuntustudio-wallpapers
```

Give it a try as it also modifies the boot splash as well.
