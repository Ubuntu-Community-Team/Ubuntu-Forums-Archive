### Post by skippymo on 2008-12-24
in evolution setting up pop email is super easy.

launch evolution
go to edit -> preferences

mail accounts button (on left) then add

it will then ask for your name and email
then select pop
put in your server info....if it is google, yahoo, hotmail, google the server address ex. outgoing.google.com

put in the type of security generally it is SSL and make sure your username is correct above it.

check remember password

set the rest of the settings plugging in the details that you got from googling the server info.....

message me if you need more assistance.
