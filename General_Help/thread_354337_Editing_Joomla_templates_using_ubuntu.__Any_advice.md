### Post by phersotty on 2007-02-06
I would recommend Nvu, Bluefish, or Quanta Plus.  Nvu has WYSIWYG capabilities.  Also Gwenview has a nice feature to make a web slide show.   AND... there is also the py-slice plug-in for Gimp that allows you to convert an image into a webpage.  Its a good start and then you can tweak the generated code as necessary.
