### Post by kokkus on 2008-09-24
I think this will work fine.

In the ubuntu installation program you can create extra partitions for a new ubuntu installation right. 
If you create a new partition and set it to let's sey 20 GB.
Now install ubuntu again on the new partition, restart and insert the xp cd.
Now overwrite the 20gb partition wiht the new xp installation.

It doesn't matter if the xp cd won't recognize the new ubuntu partition. 
You can delete the partition with the ubuntu cd again.
