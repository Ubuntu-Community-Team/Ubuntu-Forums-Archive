Is there a way in ubuntu 12.04 to encrypt single files that you need a password to view it?

### Post by papibe on 2013-04-07
Hi breezypt.

There are several alternatives.

The easiest way is to compress the file and set a password to uncompress it. To do that, right click on the file, select 'Compress...', and then expand 'Other options'.

If you require an stronger encryption, you can install an encryption package, for instance the command line tool **bcrypt**.

Hope it helps. Let us know how it goes.
Regards.
