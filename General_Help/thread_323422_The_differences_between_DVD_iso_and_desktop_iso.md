### Post by aysiu on 2006-12-22
I've moved your post to a more appropriate section.

The difference is in the amount of software. The CD has only a particular desktop environment and one set of applications (one application per task). The DVD has the entire Main and Restricted repositories.

You can read more about Main and Restricted here:
[http://www.ubuntu.com/ubuntu/components](http://www.ubuntu.com/ubuntu/components)
