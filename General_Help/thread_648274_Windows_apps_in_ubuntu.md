### Post by srunni on 2007-12-23
Counterstrike: Source will work perfectly in Wine, a compatibility layer for Windows applications. Check [http://appdb.winehq.org/objectManager.php?sClass=version&iId=3731&iTestingId=17757](http://appdb.winehq.org/objectManager.php?sClass=version&iId=3731&iTestingId=17757) for instructions. Team Fortress 2 will also work, look at [http://www.fsckin.com/2007/10/15/how-to-run-team-fortress-2-half-life-2-hl2-ep-12-in-ubuntu-using-wine/](http://www.fsckin.com/2007/10/15/how-to-run-team-fortress-2-half-life-2-hl2-ep-12-in-ubuntu-using-wine/) for info on that.

As for Adobe CS3 applications, they won't work with Wine, but if you really need them, you can try out VirtualBox, which lets you emulate an entire copy of Windows in Linux, and run the applications through that copy of Windows. This method of emulation doesn't let you run any 3D games, but all other applications work perfectly fine. Look at [http://www.fsckin.com/2007/10/29/how-to-run-microsoft-outlook-natively-on-linux-using-virtualbox/](http://www.fsckin.com/2007/10/29/how-to-run-microsoft-outlook-natively-on-linux-using-virtualbox/) for info on that. It says it's for MS Outlook, but just install Adobe CS3 instead; all other instructions will still apply.
