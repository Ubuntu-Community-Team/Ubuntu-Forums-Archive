### Post by louieb on 2006-11-17
Insert blank CD.
Click ignore when asked what do you want to do.
In nautilus open the directory with the iso file
Right click on the iso file
Choose Write to disk...
Nautilus is smart enought to burn a .iso as a cd image.
Easy as that.
