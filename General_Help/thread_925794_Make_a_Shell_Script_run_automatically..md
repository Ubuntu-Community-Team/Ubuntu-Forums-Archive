I'm guessing you mean run it by clicking on the file icon.

Create a launcher for it. On the desktop, right click and select "Create Launcher" or on a panel, right click and select "Add to Panel/Custom Application launcher"

In the dialog box that opens, enter the full path to the script in the "command" field. You can use whatever you like for the name and comment. The icon for the launcher can be changed by clicking on the default icon picture.
