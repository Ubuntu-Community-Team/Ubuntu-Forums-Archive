### Post by Clancy_s on 2009-05-14
It's probably a video driver issue, of which there are many depending on which version of ubuntu and which drivers you've chosen.  

I have a Radeon mobility 2600HD (76M) in a Toshiba Satellite A200 laptop.

For me with ubuntu 8.10 64 bit suspend didn't work with the open source but does with the proprietary drivers (installed using the restricted driver manager).

Apparently ati with ubuntu 9.04 has all sorts of problems, both with open source and proprietary drivers, I'm holding off the upgrade for the moment.

If you search the forums you'll find many active discussions on ati problems.
