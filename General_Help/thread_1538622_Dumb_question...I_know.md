### Post by selittl on 2010-07-25
How do I know if I am running 32 or 64 bit Lucid?

---

### Post by patchwork on 2010-07-25
type in a terminal ```
uname -a
```

If you are running 64 bit you will see x86_64 near the end of the output

If you are running 32 bit you will see i686 (or similar) instead
