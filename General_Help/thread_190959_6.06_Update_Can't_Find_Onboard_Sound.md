### Post by flar on 2006-06-06
type "lspci" in a terminal and see if your sound card is listed. if it's not, make sure it's enabled in the bios. 

If it is listed, type "lsmod | grep snd" in the terminal and see if your card's modules are loaded. If they are, then the problem might be alsa.  If the modules aren't loaded, they need to be loaded.
