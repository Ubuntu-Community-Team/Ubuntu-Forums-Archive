### Post by frodon on 2005-09-06
The [mount ](http://ubuntuguide.org/#mountunmountisofileswithoutburning) command allow you to mount .iso file.
For .bin, .cue file i use [cdemu](http://cdemu.sourceforge.net/) which allow you to simulate a CD with any image file like CDDaemon tools under windows.
