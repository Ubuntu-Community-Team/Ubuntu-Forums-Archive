### Post by Old_Grey_Wolf on 2012-02-26
Choose Session allows you to select the Desktop Environment (Window Manager); such as, LXDE, XFCE. It looks like they are running Lubuntu or Xubutnu; however, they are really selecting the Desktop Environment (Window Manager) to use.

You can install different distros; such as, Fedora, Arch, Debain, and so on in different partitions. You select the distro to boot by selecting the distro using Grub(2) when you boot the computer.

I'm not sure if you want to select different distros or different Desktop Environments (Window Managers).
