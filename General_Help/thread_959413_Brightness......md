### Post by ragtag on 2008-10-26
If you have an nvidia graphics card you can do this with nvidia-settings.

It's in the package repostiory, so you can simply run Synaptic Package Manager and install it. You need to run it from the command line, 'sudo nvidia-settings' (I'm not sure if you need sudo, but you would if saving xorg.confg).
