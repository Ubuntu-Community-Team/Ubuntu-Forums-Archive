### Post by mcduck on 2013-01-19
That's the Nautilus search box, same one you'd get if you started typing in a file manager window. If you have no other window selected, the the desktop itself is selected. And that's managed by Nautilus. ;)

So basically whatever you type in the box is used for a quick search in the current directory. In your case the desktop.
