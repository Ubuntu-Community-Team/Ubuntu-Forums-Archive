### Post by syntr on 2011-08-05
HTML and CSS describe the layout of a site. They can only make static webpages.

JavaScript adds most dynamic content to websites. A good JavaScript library you might want to try is jQuery.

PHP works on the server side to generate pages, usually based on input from a user.

### Post by WorMzy on 2011-08-05
(X)HTML and CSS are the two most important languages to learn, imo. HTML defines the structure and content of a webpage, and the CSS defines the appearance. Even when you move on to PHP, Django, Ruby on Rails, etc, you will still need to know about HTML and CSS, as that is what the web browser will receive from your server.

After you've got to grips with CSS, move on to Javascript (personally, I suggest you focus on AJAX - Asynchronous Javascript and XML). PHP is the next step, and will allow you to create much more flexible multi-page websites. Django is an alternative to PHP, but is very different; you may want to learn that first -- it's a lot more fiddly than PHP, but involves a lot less typing in the long run.
