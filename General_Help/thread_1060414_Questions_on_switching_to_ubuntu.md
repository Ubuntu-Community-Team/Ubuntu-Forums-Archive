### Post by linux5uper on 2009-02-04
Regarding the first three points - you might wanna keep your Vista install (dual boot) while you test Ubuntu on another partition, so that you can still do all your important tasks. 

With your PC configuration I'd guess Vista runs pretty well, and if you mainly work in Photoshop, then stay in Windows. You shouldn't have a problem with the Firefox themes though, if there is one specific one that's special to you, you can check in advance if there is a Linux version (usually the case).
