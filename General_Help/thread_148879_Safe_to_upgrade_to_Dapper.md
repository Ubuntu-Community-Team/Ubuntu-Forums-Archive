### Post by incubus on 2006-03-23
I have both installed in my computer. Especially since the last two alpha releases, Dapper has been behaving very well, quite stable.

Because I like to test the default configs as well, I usually install from scratch using the new ISOs. At least in my case I still see some things a bit unstable, but this is not general by any means. The Kubuntu team is doing a GREAT job.

But remember things can still break -- you could upgrade to a kernel that doesn't go well with your computer, as happened to some people. For a production system doing serious work this may be too risky. If you can afford to have multi-boot, I'd give it a shot.

I wish I could test synce-kde for you.
