### Post by coldReactive on 2009-08-08
It's a GTK2 theme, not an emerald theme. You install it from **System > Preferences > Appearance** click the install button and choose the tar.bz2 archive.

To apply the theme, click apply theme when the dialog comes up saying it successfully installed, or click the customize button on the appearances dialog.

GTK2 themes don't affect the window border/title bar.
