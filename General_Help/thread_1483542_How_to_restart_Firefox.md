### Post by Potters Son on 2010-05-14
If you're feeling programacidal (or Firefox reports that it's already open when you try to open it), use System > Administration > System Monitor. It's 95% the same as Task Manager in Windows; the only difference is that you just have a task list, no Applications listing (which I've found is generally the better and faster way to kill a program anyhow).
