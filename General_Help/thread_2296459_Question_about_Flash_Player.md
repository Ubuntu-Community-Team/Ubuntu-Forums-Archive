### Post by efflandt on 2015-09-26
Adobe is no longer providing newer flash versions for Linux, only security updates for the older version. You could try Google Chrome which includes its own Pepper flash which corresponds to Windows flash versions (Chrome no longer does NPAPI netscape plugins). Google Chrome works for Netflix too, but using HTML5 (instead of MS Silverlight).

When Hulu recently implemented DRM, Google Chrome stopped working (because its flash does not do DRM), and for flash in Firefox I had to add a ppa to install "hal" which is no longer in the repositories for newer Ubuntu versions which use udev.

But I do not have Xfinity, so not sure if either Chrome or hal for flash in Firefox would help.
