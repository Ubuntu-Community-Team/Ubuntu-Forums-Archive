### Post by atomkarinca on 2008-04-15
type this in the terminal:

```
sudo apt-get install gnome-dictionary dict-freedict-eng-ara
```

then open gnome dictionary:

```
gnome-dictionary
```

Edit > Preferences > Add:

hostname: localhost

then select that option and you're good to go.
