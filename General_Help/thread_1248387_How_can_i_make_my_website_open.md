### Post by exutable on 2009-08-24
I'm assuming the website is in /var/www/ if it is opening as localhost, what you need to do is create a directory called floricolor and place the html files in there.

Then you will get for example:

127.0.0.1/floricolor/index.html

to open everytime you type in localhost/floricolor
