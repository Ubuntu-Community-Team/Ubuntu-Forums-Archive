### Post by justleen on 2010-06-25
yea, there is.
check out [http://bootdisks.org/](http://bootdisks.org/)  there is a bunch of different bootdisks including ones with PXE boot support. 

I have used these in the past to pxe boot systems with only a floppy drive available and no pxe-bios support.

As to how to get them on to cd/usb, use google for some example setups like
[http://www.hps.com/~tpg/notebook/pxe.php](http://www.hps.com/~tpg/notebook/pxe.php)
