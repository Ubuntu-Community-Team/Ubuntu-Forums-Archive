### Post by u.2 on 2009-04-15
I am using a Dell Latitude D830 series laptop.

I use the FN + Up arrow to increase the brightness of the display when using it on the battery.  Every couple minutes it goes darker and I have to then manually increase it again using FN + Up several times.

Is there a way to turn that off?  I believe I have all the display settings, screensaver, and power options set so this shouldn't ever hibernate or go into standby mode, the only thing left is after 30 minutes the monitor would turn off if idle.

---

### Post by Narvinye on 2009-04-15
System >> Preferences >> Power Management

At the On Battery Power tab you can uncheck 'Reduce backlight brightness' or 'Dim display when idle'. 

(9.04 Beta)
