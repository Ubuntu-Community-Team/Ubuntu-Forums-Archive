### Post by Ramses de Norre on 2007-01-19
If you install gkrellm or conky you can see your ram usage on your desktop. If it's often full and the disk swap is often used you may benefit from more ram, if it's never full however, extra ram wont give you more performance.

### Post by Moulton on 2007-02-01
DSL is good for scoping out a machine that's in some kind of unknown state or configuration.  You can boot up DSL and cycle through the little widget that mounts all the partitions on all the hard drives and find out what you've got.  That, and fdisk on DSL suffice to map out pretty much any motley collection of drives and partitions.

Then, since you have a shell and root, you can tinker any of the partitions to repair whatever ails them. This is especially useful if you are scoping out a machine for multi-booting or for figuring out how to prepare it for use with virtualization.
