### Post by lobralleo on 2010-12-01
Installing KDE on top of Ubuntu is usually a safe procedure; the only big drawback is that you will find several redundant applications in your menu, since both KDE and GNOME come with a large array of software (text editor, media player, browser, mail client, IM, torrent client and so on). Other than that, you can proceed freely.

If you just want to try the KDE system, you can install either kde-minimal or kde-standard from the repositories; these metapackages will provide the core or main KDE components. If you feel like going the long way, you can directly install the whole of Kubuntu by installing kubuntu-desktop, which will actually turn your system into a full-fledged Kubuntu installation.

Hope this helps!
