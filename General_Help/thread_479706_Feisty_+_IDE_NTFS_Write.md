### Post by MichaelLerch on 2007-06-20
Alright, I tested it.  This is what I did.  

1. Go to Applications > System Tools > NTFS Configuration Tool
2. It **should** tell you that it has found additional NTFS partitions and it will ask you to give them a mount name [if it doesn't already have one] and if you would like to mount it.
3. Give your newly found NTFS partition a mount name if it didn't have one, then enable the mount.
4. Click APPLY and then OK.

Now, you should have your mounts located on your desktop.  If you don't see it, restart your computer.  If you didn't get a box saying it found new NTFS partitions, restart your computer and do the steps again [it worked for me].

If you don't have the NTFS Configuration Tool installed,
1. go to Applications > Add/Remove...
2. Show:  [x] All Available Applications
3. Search: NTFS
4. Install: NTFS Configuration Tool
5. Follow above steps to configure your NTFS drive(s) and/or partition(s).

I hope I've solved your problem, it's worked for me!   Thank you Taurus for providing some help with this, at least you've established some new commands for us newbies to play around with eh? :P


**NOTE**:

If you can't find the NTFS Configuration Tool in Applications > System Tools... open the Terminal [ Applications > Accessories > Terminal ] and type this

```
ntfs-config
```

It should display the NTFS Configuration Tool.  

Good Luck!
