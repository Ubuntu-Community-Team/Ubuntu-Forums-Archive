### Post by ryanhaigh on 2008-04-23
Running in a VM will always be slower but I think you will find performance more than satisfactory, upgrading the ram might be helpful if you are going to be running big apps in the vm purely so you can allocate more memory to windows.

You might also want to look at virtualbox which is an alternative to vmare server, there is a version available in the repos but the one from their site is more fully featured and still free.
