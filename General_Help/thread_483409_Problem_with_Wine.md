In Wine File, I have the following drives

Shell
C:\
D:\ (CDROM)
Z:\


It doesn't show my windows partitions. How do I enable or view them so that I can run windows apps?

---

### Post by Azakus on 2007-06-24
You have to add them manually.
First run winecfg from the terminal.
Then, click Drives -> Add and type in the path to your windows partition.
Keep in mind, most things can't run off of a Windows partition (especially games).
