### Post by Rsjc741 on 2012-07-13
Is the keyboard and mouse USB? If so, is it connected to a USB 2.0 slot?
 
If it's connected to a usb 3.0 slot or a older green/pink color connector (the name escapes me >.<), consider using a generic mouse on a standard USB 2.0 slot.
 
Also, don't use a USB hub (a single usb connection with multiple connectors)
 
Remove any unnessissary usb connections and try again
 
Hopefully this should help =)
 
If you don't have a generic mouse, you can usually find one online or in stores for under $10.
