### Post by jerome1232 on 2010-04-21
You don't lose anything by using the server version. The Desktop and the Server versions both have access to the same repositories of software.

The main differences is the server version comes with server specific software, no gui by default and the server kernel (optimized more for server type tasks than the generic kernel)

You can add anything that comes with the desktop verision to the server version (like a gui) if you wish.
