### Post by misfitpierce on 2009-10-31
If you want simple yet instant setup and sleek/nice looking... Use Docky... It comes built into Gnome-Do...

sudo apt-get install gnome-do in terminal or find in add/remove... Go to properties in the app and change theme to docky... Then dock will be at the bottom. You can get newer version from their site via their repo. But try it out from the repos in ubuntu first and see how you like it even though the newer one from site has few new features. It is nice and is also soon splitting into its own app just called Docky so keep eyes peeled for that as well :P

If Docky not your style then try Avant-window-navigator in add/remove or synaptic and install that one. Think its helped maintained by google and it works pretty good. Cairo can be tricky for some to setup etc. As for wbar... I have no idea on that one myself.
