### Post by yancek on 2016-03-30
There's nothing unusual about that.  It is expected behavior from windows that it will overwrite boot code.  To get some information so that someone can explain to you what you need to do, go to the site below and get the boot repair software and burn it as an image to a CD and reboot the computer with the CD in the drive and set the CD drive to  first boot priority.  When it boots, select the option to Create BootInfo Summary and do not try to make repairs.  Post a link to the boot repair output here and someone should be able to help.

     [https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)
