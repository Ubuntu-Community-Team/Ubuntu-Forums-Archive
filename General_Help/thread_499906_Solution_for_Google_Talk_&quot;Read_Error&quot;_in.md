### Post by vishzilla on 2007-07-13
For anyone who are getting the error message "Read Error" for the Google Talk account in Pidgin, I have found a solution for it. These are the only changes to be made from the one mentioned in Google Talk FAQ
Go to Accounts->Add/Edit
Select your Google talk account, go to the advanced tab
Under XMPP options,
Check the "Force old (port 5223) SSL" and change the Connect port to 443 with the connect server : talk.google.com
You should find it working. Cheers :)
