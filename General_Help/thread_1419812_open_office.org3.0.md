### Post by nischalinn on 2010-03-02
I've created a table in open office word processor.

I've listed some names and other information in that table.My problem is; the names are not in alphabetical order and I've to make them in alphabetical order.It is very difficult to again create the data.Is there any alternative way to arrange the data in alphabetical order,without loosing any data?

---

### Post by marshmallow1304 on 2010-03-02
Select the data that need to be sorted, then Tools->Sort.

---

### Post by zeroseven0183 on 2010-03-02
Yes, highlight the columns or the rows that need to be sorted, then click on **Table** and select **Sort...**
Then adjust the options in the box that will appear depending on the output you want.
