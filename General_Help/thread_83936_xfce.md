### Post by aysiu on 2005-10-29
Follow these directions:

[http://www.psychocats.net/linux/sources.php](http://www.psychocats.net/linux/sources.php)

Then open the gnome terminal ```
sudo apt-get update
sudo apt-get install xfce4
``` Any reason you want to do this in the terminal? In Synaptic, it's easier to browse for xfce-related packages.
