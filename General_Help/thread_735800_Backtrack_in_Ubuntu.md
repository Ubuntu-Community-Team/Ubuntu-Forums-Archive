### Post by _godbout_ on 2008-03-26
Hi!

You can install the aircrack-ng package from the package manager, the tools will then be installed on your Ubuntu.
But if you're using the ndiswrapper drivers for your wireless card, you'll not be able to switch it into monitor mode. The way to do that is to install the madwifi drivers instead, or to boot with the Backtrack live CD.
