### Post by grahammechanical on 2014-07-07
Wine is not one program that we launch. There are about 3 or 4 utilities that come as part of wine. I do not use Lubuntu so I cannot direct you but look for a menu item called Uninstall Wine Programs. Load that and a dialog will appear called Add/Remove programs and it will have a button labelled Install. Click the button and a Windows style file open dialog will appear. Use that to find the setup exe or install exe of the Windows application that you want to run under wine. And when you run that setup exe file the application will install just like in Windows. You should even get a desktop icon for launching the application.

There will also be a Configure Wine utility and a Notepad utility.
