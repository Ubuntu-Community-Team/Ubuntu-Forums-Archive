### Post by expatCM on 2007-09-25
Yes, it is possible to remove the earlier version but it is not a good idea to do so.

If you have an earlier version on your computer and there is a problem with the latest version then you can simply boot to the earlier version and have a working system.  Problems can appear, I have even experienced this myself.  If you remove the earlier version then obviously you cannot boot to that earlier version and life would be a lot more interesting ............
