### Post by Sam on 2008-11-11
I'm not sure, but try that:

1) Backup
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
```

2) Open this file for editing
```
gksudo gedit /etc/X11/xorg.conf
```

3) Find:
```
Section "Screen"
```

4) Inside this block, add:
```
    Option      "NvAGP" "0"
```

5) Close all your applications, log out then press Ctrl-Alt-Backspace.


If it doesn't work, just revert using the backup:
```
sudo mv /etc/X11/xorg.conf_backup /etc/X11/xorg.conf
```
