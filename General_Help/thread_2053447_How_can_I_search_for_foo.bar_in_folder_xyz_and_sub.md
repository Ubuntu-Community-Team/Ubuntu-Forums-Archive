### Post by sanderj on 2012-09-05
Filezilla is a FTP client, so apparently you have FTP access. Right?
I don't know Filezilla, but you have checked that Filezilla has no search option.

Nautilus: Nautilus can mount a FTP site. So do that. Then use the search in Nautilus to find your file on the remote FTP site.

HTH

### Post by hanzj on 2012-09-05
I found the file. 
In Filezilla, press **F3** or, on the toolbar, go to _Server_/_Search remote files..._.
