### Post by Ambimom on 2007-04-25
This is a problem with the last update but it is easily fixed. 

In Synaptic, mark the version you have for removal (not complete removal)  

Go to the Opera site and download and install  the Ubuntu version directly. 

It should work again....and you have the latest version which fixes a bunch of bugs associated with Linux to boot!

[http://ubuntuforums.org/showthread.php?t=413719](http://ubuntuforums.org/showthread.php?t=413719)

Good luck!
