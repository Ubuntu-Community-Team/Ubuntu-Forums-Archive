### Post by balleyne on 2007-07-04
I'm on a laptop, so I'm not sure if this is different on a desktop, but in Ubuntu you should just be able to go System -> Preferences -> Power Management, and then slide the bar all the way to the right to get the 'never' option for putting the computer to sleep.

### Post by t-dome on 2007-07-08
Hi,

it seems that the power settings are not installed by default on a desktop system. Just install the package "kpowersave" from adept or synaptic (I use Synaptic even in Kubuntu) and then the icon appears in the system tray.
