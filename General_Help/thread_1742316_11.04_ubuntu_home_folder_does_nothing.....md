### Post by GreatKeyHunter on 2011-04-28
Solved.  If anyone else is having problems with this its an easy fix.  

[https://bugs.edge.launchpad.net/do/+bug/290136/comments/48](https://bugs.edge.launchpad.net/do/+bug/290136/comments/48)

to access that file open up the terminal and type the following. 

```
gksudo gedit /usr/share/applications/nautilus-home.desktop
```

after you make the changes, logout and log back in.  problem solved.
