### Post by mcduck on 2014-06-03
The indication is there, how visible it is depends on how well the GTK theme you are using is made... (So, the only option is to switch to a theme that works well and gives visible indication. Finding one can be a bit tricky since Firefox & Thunderbird are not using GTK directly but through Mozilla's own XUL language, so while things mostly look the same as in native GTK, there are some inconsistensies...)

### Post by amanchesterman on 2014-06-05
I use Xubuntu with the 'Greybird' theme installed and I find that everything in Thunderbird is clearly legible -- both the indicator bar and the whole left panel, as well as the emails themselves of course. I don't know if you can get Greybird for Ubuntu? Anyway, as McDuck says, it's mostly down to the Ubuntu theme, so encourage your friend to try out different themes to find one that is better.

Also bear in mind that you can install different themes in Thunderbird itself: go to Tools > Add-ons > Get Add-ons and search for 'themes'.
