### Post by cAlpha on 2010-06-13
Looks like it can be accessed using super+e in 10.04.  You can change this in the Compiz 'Windows' settings, but even doing so it operates differently than it did in 9.10--hitting super+e activates the Window Picker and you need to click on a window to close it, whereas previously, simply letting go of the super+e keys (or alt+shift+up) would close the Window Picker.  

I kinda prefer it the way it previously worked, so a bit disappointing.
