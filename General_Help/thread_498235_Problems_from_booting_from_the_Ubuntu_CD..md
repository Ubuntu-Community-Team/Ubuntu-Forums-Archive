### Post by p_quarles on 2007-07-11
You'll probably get more help with this question in the "Beginners" forum, since a lot more people monitor that.

That said, the first thing I would do is try running the alternate installation CD. It sounds like you're using the live CD, and for various reasons, that doesn't work on every system. The alternate runs a more lightweight and flexible installer. 

You can get the alternate CD the same way as the live CD; just make sure the "Alternate" box is checked before you start the download.
