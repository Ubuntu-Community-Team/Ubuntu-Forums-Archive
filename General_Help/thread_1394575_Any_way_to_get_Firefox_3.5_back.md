### Post by DemonSharkKisame on 2010-01-30
Just as the title says, I'm wondering if there's any way to get Firefox 3.5 back on my machine (9.10). I installed Namoroka through the ubuntu-mozilla-daily PPA, and it overwrote the 3.5 install. Any way to get my old Firefox back? Help would be appreciated!

---

### Post by Compintuit on 2010-01-30
Remove the source ppa from your sources, uninstall it, update, and install firefox. The only version left will be 3.5.*, so that's the only one you can get.

### Post by DemonSharkKisame on 2010-01-30
Thanks! Worked just fine!
