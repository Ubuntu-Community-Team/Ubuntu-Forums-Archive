### Post by aysiu on 2008-11-24
> **Soldier4Christ said:**
> When i try to log in as root it says that i can't log on at that screen. Where do i log on at? :confused:
You don't log in as root.

If you want to perform a root-privileged task, you use *sudo* or *gksudo*.

More information on how to do so can be found [here](http://www.psychocats.net/ubuntu/permissions).

More information on why Ubuntu uses *sudo* instead of a root login can be found [here](https://help.ubuntu.com/community/RootSudp).
