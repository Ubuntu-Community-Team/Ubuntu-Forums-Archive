### Post by 3rdalbum on 2009-10-04
I don't think this Ubuntu user who gave you the advice really thought things through; Ubuntu can usually help you access the drive, but any repair efforts are your responsibility; Ubuntu is not a Windows recovery tool.

Also, if your hard disk is formatted as NTFS, I don't believe you'll be able to mount it read/write if it has not been correctly unmounted from Windows (and if your problems started from a power failure, then it hasn't been correctly unmounted).

If the hard disk is Fat32 formatted, then you might be able to recover the boot files if you know what to do. If you don't, then your best bet is to back up all the data from that disk onto a flash drive or external hard disk, and then reinstall Windows.

Or install Ubuntu instead of Windows.

Good luck.

### Post by Ichtyandr on 2009-10-04
Perhaps the recommendation to use Ubuntu as a live CD had the rationale of a rescue/backup measure, and also introducing to a superior OS :)

If your XP installation is irreversibly broken, you can use Ubuntu live CD system to access and backup your files, to a usb drive e.g. and then try to recover XP installation from XP CD or else to make a fresh installation. Ubuntu live cd is a good measure here as it is easy to use and does not make changes to your hard drive install and will not run any viruses or malware if you have some running on your xp machine.

After you are done it have another look at Ubuntu and maybe do a dual boot to compare your experiences :)
