Konqueror is pretty good about working under non-KDE environments in my experience, but Nautilus also tries to draw the gnome desktop and mucks things up. I'd get dolphin from the repos: "sudo aptitude install dolphin", it's the file manager for KDE4 (although konqueror will still be around for users who like it and as a web browser). Xfce probably has a file manager setting somewhere, but I'm not sure where.
