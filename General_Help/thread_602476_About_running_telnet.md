The SMTP server will not hae specific commands to add attachments.  Instead, attachments are encoded in the message body (MIME encoding).

When you prepare your body text, try something like:
```

$ cat message.txt <(uuencode picture1.jpg) <(uuencode picture2.jpg) > body.txt

```

This can also be broken down into multiple steps:
```

$ uuencode picture1.jpg > temp1.txt
$ uuencode picture2.jpg > temp2.txt
$ cat message.txt temp1.txt temp2.txt > body.txt

```

Most mail clients will understand that the text generated by the uuencoding process is really an attachment.
