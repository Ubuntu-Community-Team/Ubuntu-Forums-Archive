### Post by TheFu on 2018-04-16
NextCloud requires manual steps before it can be used.  Find a guide to follow. Most have 20+ steps.
Here's one from a reputable source: [https://www.linux.com/learn/how-install-nextcloud-server-ubuntu%20](https://www.linux.com/learn/how-install-nextcloud-server-ubuntu%20)
I used a "source install" with my setup.

I've never installed NC through a package manager.  The source install has build-in updates and visual reminders to administrators that updates are available, so maintaining an existing NC install isn't bad compared to most web-apps.  Usually it is point-n-click from the web interface.  NC addons are almost always 1-click installs too, though some are more complex and do require manual config file modification.
