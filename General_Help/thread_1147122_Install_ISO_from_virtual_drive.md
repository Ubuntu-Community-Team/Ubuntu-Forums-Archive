### Post by dandnsmith on 2009-05-03
I think the basic premise is wrong - you don't create a virtual drive on anything - it's virtual!

I suppose OP wants then to boot from it to run the install - no chance.

I'd go with putting the stuff on a USB stick (see pendrivelinux.com for details of methods for several OS), and booting from that to do the install.
