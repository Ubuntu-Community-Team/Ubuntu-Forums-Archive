### Post by namdung on 2009-01-09
When any application greys out, it just means that it is processing at the background and has reached CPU max limit. This usually lasts a few seconds. If a particular application is causing this issue regularly, monitor its process in System==>Admin==>System Monitor==>Processes. U may kill any unwanted applications and give more priority to the particular application by *Right click==>Change Priority*.
