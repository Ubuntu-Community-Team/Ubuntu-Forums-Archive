### Post by SpaceMaster on 2008-08-15
Yes, Ubuntu is generally compatible with iPods.  I use Rhythmbox to interface with all mp3 players.  You may have to install Gtkpod to properly configure the ipod, and then you should be able to manage it through Rhythmbox.  It seems that Hipo iPod Management is gaining popularity, so you might use that instead of Gtkpod.  Like Gtkpod, with Hipo you'll only be configuring the iPod, and you'll still have to manage the music on it through Rhythmbox.

One last precaution: Generally, support of iPods runs a generation behind the market, so if you have a brand new iPod, it may be a month or more before you can actually manage it properly.  I have no problems with older generations, though.
