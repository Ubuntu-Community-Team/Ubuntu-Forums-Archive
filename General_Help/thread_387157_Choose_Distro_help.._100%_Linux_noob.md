### Post by Sam Clemens on 2007-03-18
Assuming you still have ubuntu installed, you can probably just install  the *kubuntu-desktop* that should give you an idea of what KDE looks like. Or try a knoppix live cd if you wiped Ubuntu.

Alternatively you could go through the pain of dealing with Gnome for a few months and then you'll never want anything windowsish again.  It's like anything new, it takes time to get used to, and has it's foibles.

If you really want something that's targeted to be very Windows like, check out Linspire or Xandros but you're digging yourself into purchase and upgrade costs there.  

I know what it's like playing with an unfamiliar layout, I've been forced to use Windows recently and have found the layout, and the look and feel alarmingly different. That's because I'm used to Linux and Gnome, of course. Coming the other way is likely to be much the same but with a certain amount of settling time the transition to something with a Linux feel rather than a mock Windows feel is probably a worthwhile one to make.

If you want everything to work perfectly (if not perfectly, atleast without much incident) I would suggest reinstalling from one of the kubuntu releases.  I'm not going to tell you which one to use, but dapper is more stable all around, and edgy is faster and has newer versions of software + other new features.  It's really a personal choice.

If you want the decently easy way WITHOUT reinstalling, run this from a terminal:

```
sudo apt-get install kubuntu-desktop
```

This may be a long download depending on your connection speed (over 100mb if I remember) as it will download kde and all the other software that comes stock in kubuntu, but if you don't want to reinstall, this is THE way to do it.  :)
