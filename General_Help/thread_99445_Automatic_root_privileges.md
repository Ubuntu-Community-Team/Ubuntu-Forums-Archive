### Post by Gray. on 2005-12-05
Well what seemed to be a godsend is now turned into annoying.
I can't remember doing anything to alter my user's permissions and know when I want to do something that requires me to enter my password (firestarter, synaptic, etc), I don't even have to enter my password. Even using the nautilus script to "browse as root" (from Automatix), I don't need to enter any password.
It is, I admit, more convenient, but I don't want this to go on much longer as it is a security risk.
Any help would be greatly appreciated.

*EDIT* Wow, I found the solution already. In case someone experiences the same problem, it was because I had added my user to the sudo group in System > Administration > Users & Groups. I guess some people may find that to be a handy hint. :P
