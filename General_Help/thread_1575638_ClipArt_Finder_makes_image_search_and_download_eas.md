Hello,
The amazing ClipArt Finder version 0.2 has just been released to the public.

ClipArt Finder can search and download free clipart and other images from the internet. 
It can translate search queries (words) from almost any language to English. 
And your search result/images are saved locally for later use. 

This program has two variants: 
* clipart-finder is an ordinary window'ed GNOME-program. You can start it from the command line or Applications -> Graphics menu. 

* clipart-finder-applet is an applet that you attach and run from the GNOME's tooolbar panel.

Please watch the demonstration video and get ready-made package from: 
[https://launchpad.net/~osmoma/+archive/clipart-finder](https://launchpad.net/~osmoma/+archive/clipart-finder)

There is a package repository for Lucid only, but the very same .deb package should install and run on all recent Ubuntu-versions (Karmic, Lucid and Maverick), on both 32 and 64bit systems. Though I haven't tested all combinations yet ;-) Simply find and click the .deb package's name.

Source code:
[https://launchpad.net/clipart-finder](https://launchpad.net/clipart-finder)

Enjoy 8-)
