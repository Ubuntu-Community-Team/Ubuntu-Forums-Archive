### Post by aclaunch on 2005-12-07
OK more questions: you did install the nvidia driver, preferably via synaptic? If so, rerun synaptic and check that the nvida driver (nvidia-glx) is installed along with the  other necessary packages (linux-restricted-modules-2.6.12, nvidia-kernel-common). This presumes that you have a recent nVidia card (you didn't specify) because if it is an older card you need the "legacy" packages, both nvidia-glx-legacy and linux-restricted-modules-legacy.

Once you are sure the driver is installed, rerun "sudo dpkg-reconfigure xorg-xserver" and make sure you choose "nvidia" as the driver. Once that is done, reboot and all should be good. If not, post back.

Alan
