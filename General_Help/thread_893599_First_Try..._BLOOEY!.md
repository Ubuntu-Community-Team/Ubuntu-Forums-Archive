### Post by kirsis on 2008-08-18
It's come up a few times apparently. Some AV software will mark wubi as malicious.

It's a false alarm. If it had a trojan in it, there'd be some feedback (since everybody uses the same CD) about this on the net yet you wont find any. Every AV app makes mistakes now and then.

You can also check if the CD wasnt corrupted during download (see [https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM))
