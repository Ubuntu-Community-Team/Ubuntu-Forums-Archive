### Post by kevinlyfellow on 2007-07-23
I don't know kde very well, but having something like MediaMonkey isn't Gnome's style.  Gnome tends to have a Unix like way of dealing with apps... several of them doing something specific.

While I can't help you get a media player that does all of that, I can suggest individual programs that I find work very well at their specific task.

I personally use Audio Tag Tool for my tagging.  What is GREAT about this tool is that it will rename files according to music tags and tag files according to file names.

I use Rhythmbox for my music player (if you are adept at python, you can always add extensions to it to get the functionality you want... I'm hoping this plugin thing they have going is going to be very popular).  I don't know how to attach album art to music files, but rhythmbox will look up the album art through amazon and display it.  I think banshee also has this functionality (but banshee is still young, and one poster here had thinks that it has corrupted his/her files).  

I'm afraid I don't know if Rhythmbox is compatible with the Sansa music players, but if its just a mass storage device, you can just drag and drop music/playlists into the folder (there may be a better way of doing this, but I'm not aware).  If it has mtp, then you may be interested in this thread [http://ubuntuforums.org/showthread.php?t=478042](http://ubuntuforums.org/showthread.php?t=478042)

For converting music, I use SoundConverter.  It's very straight forward, because I don't want to have to think about something like that (gnome provides plenty of tools like that ;-)).  For making playlists I use Rhythmbox's fuctionality.  I don't know if it's good or not, I just now that its good enough from me.  Audio Tag Tool also has a playlist maker, but I don't know how to use it.  There is a really cool playlist generator called gjay.  Unfortunately, it looks like it's a dead project :-(

I'm afraid I don't know of an app that can do all of that stuff you want, but I'm sure there will be more people to make suggestions on what you can use.
