### Post by jazzmuzik on 2006-04-18
You can easily install Azureus in your home directory, FWIW. You need to have a java client installed like j2re1.4 which you can install via Synaptic. Then continue with step 2 here: [http://azureus.sourceforge.net/howto_linux.php](http://azureus.sourceforge.net/howto_linux.php) for the rest of the instructions.

I suggest installing aureus to a user 'apps' directory:

cd ~
mkdir apps
cd apps

Then unpack the azureus files there. You'll most likely want this package: Azureus_2.4.0.2_linux.tar.bz2
