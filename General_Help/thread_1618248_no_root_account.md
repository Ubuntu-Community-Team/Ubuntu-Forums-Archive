### Post by cek on 2010-11-10
You do have a root account.  However, you don't know the password.

You can escalate your privileges by running the sudo command.

### Post by uRock on 2010-11-10
The last few releases haven't had a root account. To learn more as to why and how, please read this article. [https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)

You can still install stuff with sudo.
