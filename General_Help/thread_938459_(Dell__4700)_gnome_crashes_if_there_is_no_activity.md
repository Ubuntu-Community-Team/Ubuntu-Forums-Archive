I had this exact problem before. What I found out was that my external hard drive (on which I was booting Ubuntu) went into "sleep mode" after a certain number of minutes, which would cause Ubuntu to lock up; I counteracted this problem by downloading the management system for the hard drive and changed the hard drive's settings to remove the timer.
