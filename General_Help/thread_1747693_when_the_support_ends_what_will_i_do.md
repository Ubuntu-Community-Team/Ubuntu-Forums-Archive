### Post by Leed on 2011-05-03
You can also just manually put in all the Updates you want... but if I were you, I wouldn't, it's just a lot of work. 

Just install 11.04 and first try the power users manual to unity... If you still can't live with the new interface, just log in using gnome2 (logout and upon login choose "Ubuntu Classic" at the bottom of the screen)

Even in future updates, you'll still be able to have different Desktop environments running aside Unity. All you need to do is install them and set them as default

When you're logging in, Click 'Session' and switch from Unity to Ubuntu Classic (No Effects). This is the Gnome desktop you prefer.

Because of the rather secure nature of the Debian/Ubuntu structure, you could just continue running 10.10 for the foreseeable future. It's not like it will suddenly get worse. Of course, if the Unity interface is all you're avoiding, just do the above.
