### Post by Topsiho on 2010-10-14
You could try and use kdepim, which contains kmail and korganizer (also installable separately). These are applications in the KDE desktop, all KDE applications you also can use in Gnome.
Be prepared that when you install your first KDE applications into Ubuntu you'll also have to install quite a lot of libraries and dependencies which are not present in the Gnome desktop, but that the KDE applications need.

Hope this helps,
