### Post by mikewhatever on 2008-10-19
If you know the location of the file, use alt+f2, then type 
**gksudo gedit /path-to-file**.
If you prefer navigating the system graphically, use alt+f2, then type
**gksudo nautilus** navigate to the file you need to edit and open it with a text editor.
