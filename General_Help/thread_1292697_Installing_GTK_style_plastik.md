### Post by justleen on 2009-10-16
I dont use kde, but it should be quite straight forward:
    *  Install KDM Theme Manager
    * Gentoo: emerge kde-misc/kdmtheme
    * Debian based distros: apt-get install kdmtheme
    * Other: Download from: [http://beta.smileaf.org/](http://beta.smileaf.org/)
    * Open Control Center -> System Administration -> KDM Theme Manager
    * Click Administrator Mode, enter root Password,
    * enable Themes and choose...
    * Restart KDE
