Hi,

You can able to resize the hard disk partioning using the Gparted tool.

Steps:
1. Boot through the Live-CD media 
2. Unmount the partition; 
3. Open the Gparted tool
4. Resize the partition.
5. Save the changes
6. Reboot your machine
