### Post by jasay on 2005-10-31
You might open System > Preferences > Sessions and make sure automatically save sessions is unchecked.  While there make sure there isn't anything in the startup programs tab you don't want.  Then close everything (might double check the system moniter to make sure everythingis really closed) and log out with the save session clicked on the log out box.  Other than that I'm sure there is a config file somewhere that could be deleted, but I'm not sure where.
