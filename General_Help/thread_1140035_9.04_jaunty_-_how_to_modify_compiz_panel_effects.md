### Post by Tipped OuT on 2009-04-27
If you'd like to manage all your compiz settings, go to Applications<Add/Remove Programs and make sure you have it set to show "All Open Source Applications" and type "compiz" into the search box. And download CompizConfig Settings Manager (CCSM). Have fun. :P
