### Post by scandido on 2008-12-21
You can use any file storage source that supports connection via SSH (SFTP), FTP, or WebDav just using the standard file manager. This includes a lot of web hosting companies and also many free "online storage" sites. Over all these connections, you'll be able to use files like they're on your hard drive (except it will take a little longer to open them because they'll be downloaded in the background before they open).

Check out the "Connect to Server" dialog in the Places menu.
