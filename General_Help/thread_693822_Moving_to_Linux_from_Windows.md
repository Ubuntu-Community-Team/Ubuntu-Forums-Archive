### Post by luisromangz on 2008-02-11
Fedora is good enough, but the package system is crap. RPMs are slow and cumbersome. That was what forced me to switch to Kubuntu.

About your three points:

1) I'm a web developer and work in Kubuntu, so you would be able to program, too.
2) I don't know about FLAC, but mp3 is no problem, as long as you download the proper codecs. In Ubuntu this is handled automagically.
3) I have just finished watching the latest Naruto chapter in DivX, very good framerate in my crappy portable.

In general, as loong as you don't feel the need to run the latest and greatest PC games, or other Windows specific programs with no real Linux alternative (like Autocad), Linux is for you.

### Post by Thingymebob on 2008-02-13
The following formats are proprietary and therefore don't ship with Ubuntu. It is relatively easy to get them all working, but some extra mileage is required:
MP3, AAC, WMA, WMV, AVI, MPEG-4
There is more info available in the Ubuntu documentation [https://help.ubuntu.com/7.10/musicvideophotos/C/index.html]("https://help.ubuntu.com/7.10/musicvideophotos/C/index.html")

K3b is a good cd burner with a nero feel to it. Its a KDE app but runs quite happily under Gnome.
I'll second Virtualbox for a VM.
