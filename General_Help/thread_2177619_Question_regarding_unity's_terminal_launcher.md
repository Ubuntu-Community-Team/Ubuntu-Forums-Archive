### Post by 3rdalbum on 2013-09-29
> **xeddog said:**
> I have been trying to figure out how to modify the gnome-terminal.desktop file so the default action will open a new terminal window rather than an existing window.  How do I do that?

You can't modify that. With a left-click, Unity switches to the currently-running instance. Nothing to do with the .desktop file.

Instead, just middle-click the icon and a new instance will open. Same applies for any running application, and you'll also find this works on Windows 7.
