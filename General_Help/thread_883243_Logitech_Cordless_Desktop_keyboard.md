### Post by deltaromeo on 2008-08-14
I sorted it now.

I tried running the live CD and the problem was still there.

I then unplugged the USB keyboard receiver, rebooted and reconnected it, but the problem remained.

So then I took out the batteries from the keyboard and re-established the connection with the receiving unit.  It's working fine now :)

Thanks.
