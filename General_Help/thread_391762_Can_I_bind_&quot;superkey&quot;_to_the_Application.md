### Post by Yoooder on 2007-03-23
> **Kasper1990 said:**
> As I ask in the topic: Can I bind "superkey" to the Applications menu? I don't use the button for anything, so is it possible?

Yup.  Go to System | Preferences | Keyboard Shortcuts

Find "Show the Panel Menu" (at least I think that's the entry).  Click the binding column and press the Super key, and it should be bound.

### Post by CocoAUS on 2007-03-23
You should be able to change the icon by changing the icon in /usr/share/pixmaps...I think.  It might be elsewhere.  Anyway, you can also download and install a new icon theme from gnome-look.org or similar sites.  If you're using Konqueror (KDE browser), the icon is actually replaced, at least in the taskbar, with the favicon of the site you're visiting.
