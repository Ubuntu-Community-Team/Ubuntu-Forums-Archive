[SIZE=2][FONT=Arial]Hi there,
You may want to look at [[COLOR=#800080]ABBYY OCR for Linux[/COLOR]]("http://ocr4linux.com/"). It provides CLI interface as well, but it’s really simple, here’s how it works: 
*abbyyocr [open image and analysis options] -if <image file> [export options] -of <export file>*
[FONT=Times New Roman]All options are very flexible and nicely described in documentation at ocr4linux.com. It’s not free, but it provides the best quality of OCR, you can [/FONT]refer to one of OCR engine comparison articles, for example check out [[COLOR=#800080]http://www.splitbrain.org/blog/2010-06/15-linux_ocr_software_comparison[/COLOR]]("http://www.splitbrain.org/blog/2010-06/15-linux_ocr_software_comparison"), or try it yourself, trial can be requested [[COLOR=#800080]here[/COLOR]]("http://ocr4linux.com/en:trial"). If you are concerned about OCR quality, you should try it out. I work @ ABBYY and can provide you additional info if necessary. Feel free to contact me if you have any questions.
