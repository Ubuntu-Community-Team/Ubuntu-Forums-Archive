### Post by SteveDee on 2011-10-28
> **crutch145 said:**
> ...I can't find anywhere in the visible screen where it lets me select colors to change the text...

I may not have understood you correctly, but here goes anyway...

When you start using Writer, the font colour icon (letter "a" symbol) has a clear small box and arrow next to it. (see icons1.png)

If you click on the arrow you should see a colour pallet/picker. (see ColourPicker.png)

If you select a colour, the icon changes to show the selected colour. (Icons2.png)

If the mouse pointer has now changed to a spray can (or in my case a tipped paint pot) you should be able to left-click and drag the icon across any text to change to this colour. If the icon is a standard "I" beam, you can highlight text first, then click on the text font colour icon to change to the currently selected colour, or click on the down arrow to select a colour (...this seems a lot more complicated when you write it down).

Most of the time I highlight the text first (using the standard "I" beam pointer) then select a text colour.
