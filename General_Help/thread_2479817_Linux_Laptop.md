### Post by dragonfly41 on 2022-10-08
Some points to check before you jump to the expense of buying a new laptop

Can you change internal default HDD to SSD which is much faster?

Do you have a USB 3.0 port which would allow an *external* SSD Ubuntu installation in a container. See Crucial site for examples. I use a Startech dual docking bay with power.
I run Ubuntu very nicely in external USB 3.0 SSD drive attached to host Dell Tower. But of course you can install in the internal HDD/SSD.  I leave Windows 10 to run in the internal HDD/SSD.

Have you tried LiveUSB connected via USB 3.0 to check performance in "Try Ubuntu" mode?

### Post by TheFu on 2022-10-08
There are many tuning options for Ubuntu systems, based on the workload.  Also, some less-used hardware devices don't have great Linux drivers, whereas every vendor in the world works for years to optimize their MS-Windows drivers.  

So, the choice is ours ... we can be cautious about the hardware we purchase, striving to support the companies that make the effort to have great Linux drivers or  ...er ... not and live with the less-than-great outcomes.

Buying a computer pre-installed with Linux almost always means that the vendor setup all the drivers and sourced hardware with Linux support, at that time.  In 2 yrs, who knows about the Linux support in the next distro release?  For my money, I'd rather get popular components from companies that are known to be popular. Today, these are Dell and Lenovo.  Support by all other vendors seems to come and go.  I've had some sort of driver issue with all the others, but my 5 laptops are hardly a statistical sample.

There's nothing wrong with trading money for expertise, which is what you do when you buy pre-installed Linux on a laptop.
For the price of an expensive laptop, I can have a quality home-built desktop with plenty of upgrade possibilities and a cheap laptop.  Over time, this method saves even more money as only a few core parts need to be replaced to have a new computer that is 2-4x faster than the prior one every 5-10 yrs.  I'm reusing the same cases from 2000, power supplies from 2008, HDDs have come and gone over all the decades, but some are over 10 yrs old now and in a 3rd computer after a migration last fall.  Last Feb, I replaced a $0.50 used keyboard bought in 1996 for a new mechanical keyboard.  I still miss my old keyboard, though the new one has "good feel" and is quieter, it has a terrible USB interface rather than the PS/2 port which worked much, much, much, better with my KVM switch.

The last few days, the 20 yr old mouse I've been using is starting to behave odd.  Will probably trash it next time it does something funny, and pull out a old mouse from a plastic bag somewhere. Probably have 5 old mice around, since people seem to want "new" when there's nothing wrong with the mouse they have.  I like to use things until they stop working or have a good, long, useful life - say 15+ yrs.

Anyway, there are lots of options and you'll get plenty of "free advice" from people here who will never have to use the system you choose.  That goes for my free advice too. ;)  

When buying a laptop, be certain to have a list of all the external ports you want/need.  If you want to connect to a TV or projector, you'll want HDMI.  If you are a photographer, you'll want support for whatever memory card your camera supports.  I always want a wired ethernet GigE port on my laptops, since I avoid wifi (poor security).  I want the performance of a Core i5, 8GB RAM, at least 1080p screen, but no larger than 14 inches and under 3.5 lbs with 8+ hrs of battery for $350 or less.  BTW, that can be found as refurbished or used from reputable sellers.  My last laptop purchase was $305 in 2018.  I've spend more and less on laptops over the decades.  The best laptop I've ever owned was a 13.3 inch, 2.95lbs, laptop. It lacked sufficient RAM and the keyboard needed replacement after 3 yrs, but besides that, it was totally awesome. I miss using it.

### Post by TheFu on 2022-10-09
BIOS passwords can be reset to factory by pulling the battery and crossing a few jumpers. In desktops, there is a CMOS battery (watch battery) to pull and the reset jumper is usually very close.  Some laptops have a CMOS battery and some don't. They just use the main battery, which is why we never allow the main battery to die in laptops unless we want a full factory reset.

I don't know anything about that model ... but new Core i3 CPUs are fast like 4 yr old Core i5 CPUs.  Best to always post the exact, full, CPU model when it comes to performance questions.  There are vast performance differences between different models of the same line. All Core i3 are not equal.

I have a laptop from 2015 with a Core i3-5015U CPU.  It is still fast enough to remote into my real computers. That's the reason I have a laptop - to remote into other computers or watch some videos when traveling for entertainment. When I want to do real work, that's why we have desktops on networks.
