### Post by Jammanuser on 2009-02-07
That's the whole point of Wubi, ya know...;) Wubi was designed to install Ubuntu on **top** of Windows, which implies installing it on the same partition (and hence the same drive) as Windows. If you don't like that, then simply use something else. Wubi was created to make it easier to try out Ubuntu, if you're mostly a Windows user, and to prevent having to mess with partitioning and all. It was made to install and go.
