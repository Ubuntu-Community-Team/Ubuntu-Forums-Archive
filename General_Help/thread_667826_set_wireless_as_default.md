### Post by Metaljaz on 2008-01-15
If you have the network bar in the top right hand corner, right click on the icon and put check marks in the boxes that say Enable Networking and Enable Wireless. 
You can also Enable roaming mode by clicking on the network tv's, then select configure and put a check mark in the Enable roaming mode box.
