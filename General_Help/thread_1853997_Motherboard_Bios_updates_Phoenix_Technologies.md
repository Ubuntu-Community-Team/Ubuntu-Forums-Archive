### Post by Mark Phelps on 2011-10-03
As you may already know, BIOS for laptops tend to be customized by the laptop vendors.  So, if Compaq or HP does not have an upgrade on their site for this laptop, I would be hesitant about using ANY other BIOS on it.

### Post by seawolf167 on 2011-10-03
> **Mark Phelps said:**
> As you may already know, BIOS for laptops tend to be customized by the laptop vendors.  So, if Compaq or HP does not have an upgrade on their site for this laptop, I would be hesitant about using ANY other BIOS on it.

+1 for Mark Phelps' comment. Check the manufacturer's website for the model/part number to see if they have BIOS for that particular model/part or a BIOS that works for a range of their products (inclusive of your model/part). I definitely would not suggest using any "non-recommended" BIOS as IMO you chance of bricking the motherboard increase hugely. 

Is there any particular reason you want to flash the BIOS? Any new features that you need? Else you can probably stick with the "if it ain't broke, don't fix it" motto.

### Post by Frogs Hair on 2011-10-03
Your assumption is correct , there are many outfits like this that charge for something that is free . Many people use these services because of the " free bios scan " , which is used to hook people that don't know how find the status of the bios .

As written you can find the latest bios for your system on the manufacturers website . Updating the bios should be done only if the update provides something that benefits you current system configuration . 

If the bios update includes support for hardware that you will never install don't bother .
