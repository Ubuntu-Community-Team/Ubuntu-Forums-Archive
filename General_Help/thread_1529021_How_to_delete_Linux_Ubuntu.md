### Post by ov3rcl0ck on 2010-07-11
To install xp/vista you'll need to delete the Ubuntu partition which can be done with either the Ubuntu boot disk or the Windows boot disk.

For windows 7, you can just resize the Linux partition and install on the freespace you've created. You may need to reinstall a boot loader like GRUB to be able to boot into Linux again.
