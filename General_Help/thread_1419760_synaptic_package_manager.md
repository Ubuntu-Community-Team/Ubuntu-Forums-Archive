### Post by dabl on 2010-03-02
The 28000 packages are in the Ubuntu repositories.  The CD can't hold anywhere near that many.  If the computer will be connected to the Internet, then there is no good reason to have the CD marked as a source in Synaptic.  Change it back to use the standard repositories, and not the CD.
