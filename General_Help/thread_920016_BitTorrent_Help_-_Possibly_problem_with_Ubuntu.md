### Post by mb_webguy on 2008-09-14
Torrents are downloaded in multiple chunks from multiple different sources, so if you try to play a file before it is completely downloaded, it's like watching a film with some sections cut out and spliced back together.  Some torrent clients will show you the status of individual files within the torrent, and if this shows a file at 100%, then it should be fine to open, though you shouldn't move it or the client will start downloading ot again.
