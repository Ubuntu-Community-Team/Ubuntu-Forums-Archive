### Post by wrgb2 on 2009-11-04
I believe the little orange icon beside a program in Synaptic means that it is a Canonical-maintained application.  You can run Gnome applications in KDE and vice-versa, it's just that it may for example install some KDE dependencies with the program.  I'm running Open Office in Gnome (Ubuntu) and Xfce (Xubuntu), and it has the orange icon beside it.
