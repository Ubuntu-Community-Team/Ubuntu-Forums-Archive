### Post by mc4man on 2013-02-17
To go from the xorg/mesa libs to the lts-quantal ones in a 12.04.1 install is do-able though not exactly easy.

It's best done in synaptic as you need to remove some & then alot of packages before installing the lts ones.
After that one needs to re-install what was removed & d. check that all is back **before logging out** or you'll not login again easily.
(the final d. check is to re-install ubuntu-desktop, if it & any deps are installed without removing anything then you'd likely be ok.

Laptop users with a touchpad would want to make sure the xserver synaptic package is installed, lts version (or the xserver input-all-lts.. meta package will do.

Wine is not an issue on a 32 bit install, for 64 bit an i386 package must be pre-installed, mentioned here
[http://ubuntuforums.org/showpost.php?p=12514870&postcount=104](http://ubuntuforums.org/showpost.php?p=12514870&postcount=104)

If one has ppa's installed that could make it even more difficult.

Much easier to copy out anything of value & just install 12.04.2 fresh if really wanting/needing the lts X stack

12.04.2 with lts stack also could present some build-dep issues & for the moment Steam doesn't seem possible
(may try something concerning steam later..

edit: got steam up & running, not that hard though would be better if they adjusted their package
