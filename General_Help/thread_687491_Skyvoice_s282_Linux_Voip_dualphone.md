### Post by mooscape on 2008-02-04
I would like to start a thread to put together more information on the Skyvoice S282 dual voip phone.
I have been using this phone for some months now and am generally very impressed.  In fact I bought two.  It runs on linux (don't know what version), and behaves like a mini computer (could be fun to reprogram = ubuntu?)  It has on board camera that records video and images, dictionary (english/chinese) which is very useful, especially as you can write the chinese characters on the screenwith the stylus or a fingernail.  It also has all the pda functions and java games. It comes with opera browser and can be used to watch videos or listen to music, though you would be recommended to add a micro sd card. I have set up the voice facility through freeworlddialup (greatplace for free SIP account) , which was very easy to do.  I have not yet been able to get my ekiga account connected though. Connecting to my ubuntu machine is also very easy - just tap  "connect to pc", then select "sd card" and you can connect just like to usb drive.  
Only downside is relatively short battery life in wifi mode (less than an hour), though it came with two batteries. Easy to charge through usb cable, so this will not be a problem if you use computers regularly. I keep mine plugged in when in wifi mode at work or at home.

Enjoy the pix.

 Are there any other users of this phone out there?
