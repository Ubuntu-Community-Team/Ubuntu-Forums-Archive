### Post by scouser73 on 2009-09-21
Well there are many benefits in using Ubuntu, firstly there are no viruses or spyware & if there is I've certainly not come accros them.  Secondly you can modify Ubuntu to your liking with themes, so you don't just have to use the default one.

For me it has to be these forums because as you can see there are many people asking questions and questions that have been answered, this community is excellent and I reckon that you won't see this type of involvement in helping people with their questions being answered so quickly elsewhere on the internet.

Test Ubuntu for yourself, as that is the only true way of seeing if it's right for you although I doubt you'll be disappointed.

Should you need to ask a question then you can use the search pane to see whether that has been asked previously, which is usually the case.

Ubuntu has a six month release cycle, so a new edition is out every April & October, there are also regular updates for security and programs so just because it's gratis doesn't make it any less secure.

I have been using Ubuntu for over a year now and I enjoy using the forums for getting and also giving help.

Here is a list of free e-books that you can download that will also help you: [[COLOR="Red"]**Free Ubuntu e-books**[/COLOR]]("http://www.ubuntugeek.com/free-ubuntu-linux-e-books.html") & [[COLOR="Red"]**Ubuntu Pocket Guide**[/COLOR]]("http://www.ubuntupocketguide.com/download_main.html")

You will need a PDF reader for those, so you can use the default PDF reader called Evince or you can install Adobe's PDF reader by following the instructions.

1 Go to the [[COLOR="Red"]**Adobe Reader Website**[/COLOR]]("http://get.adobe.com/uk/reader/?promoid=BUIGO")
2 Click on **Different Language Or Operating System**
3 Click on the **Select An Operating System** drop down menu
4 Select **Linux - x86 (.deb)**
5 Click **Continue**
6 Select **Adobe Reader 9.1.2** & Click **Download Now**

That will start installing Adobe PDF Reader for you, once installed you can find it in Office.

I hope that you see the benefits of Ubuntu as an operating system for yourself and realise the potential of getting things done in a secure, virus/spyware & crapware environment.

### Post by badger_fruit on 2009-09-21
> **thermopads218 said:**
> I am very new to this. Can any one tell what is the benifits to use ubuntu?? 

**As opposed to what?** Chewing spiders? Flying a 747 upside down? licking toads?

Sorry for the sarcasm but as expressed above, the benefits vary - I will presume you mean benefits over using Windows.  

Well, apart from the obvious fact that it's totally FREE (even without having to reply of a dubious cracked or pirated copy of XP), it is completely secure, has little to no virus attacks (although that's probably due to the small % market share Linux in general has) and is completely customisable from the outset ... you try to change the colour of the taskbar in Windows XP without adding 3rd party software!

I personally use a few different "flavours" of Linux, OpenSuse and Ubuntu mainly but have some virtual Fedora and Xandros installs kickig around for playing with.

The reason I chose Ubuntu was simply because I had just purchased a TV card and wanted to use Myth TV - Ubuntu has a "brother" if you like, Mythbuntu - all pre-installed and practically pre-configured.

Anyway, although the Linux desktops are doing well and in some cases, better than Windows XP/Vista or 7, the backend side still has a long way to go before it can truly be a replacement for Windows.

For example, games.  I was over the moon to discover UT2004 came with a Linux installer on the CD but dissappointed that UT3 does not.  Although WINE is making some progress, it's slow overall but in a few years I am certain the guys who maintain it will be up to speed.

Depends on what you want to use it for as well. Simple HTPC or Media centre - who really cares about the OS? As long as the GUI looks nice and is easy to use then it makes no difference to the underlying OS.

If you want a hard-core games platform, nothing (yet) beats Windows.

Office and productivity? Again, who cares what OS you use to write a letter to your mum? 

This website here is one I like to refer people to ... [http://linux.oneandoneis2.org/LNW.htm](http://linux.oneandoneis2.org/LNW.htm)

Very good read.

Anyway, welcome to the forums and Linux!

### Post by lovinglinux on 2009-09-21
- free
- open source
- no virus, so there is no need for anti-virus that slows you down
- It is incredibly stable. For instance, there is no BSOD like Windows. If something crashes really bad, which is not common, you usually don't need to restart the machine. Just need to log back in.
- no need to defrag the disk every week to avoid running like a turtle
- no Windows Registry, so you don't need to use CCleaner and registry defrag tools every week to keep it running "smoothly".
- easy to install applications from Ubuntu repositories, using [Add/Remove manager](http://www.ubuntu.com/files/u1/addopera9.png) or by simple commands like** sudo apt-get install firefox** or even from a post link like [this](apt:firefox). So there is no need to search for applications on the web and they come free of malware.
- excellent support from the community
- flexible and extremely customizable 

the list goes on..
