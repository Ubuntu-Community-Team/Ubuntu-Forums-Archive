### Post by marshag63 on 2012-06-04
timetopat, sorry to hear about the accident.

There is an abbreviation expansion and script program called autokey.  You can even assign an "expansions" and scripts to be accessed via a mouse click on a system tray popping up a menu, or a pop-up box where you starting typing the first letter or two and then pick from a resulting list.

Here's a link:  [http://code.google.com/p/autokey/](http://code.google.com/p/autokey/)

You highlight, copy and paste from previous stuff you have typed and and assign it a name and even tell it in what program it can execute.  Pretty handy stuff.

Hope this helps.
