Ubuntu has 2d graphics drivers, long as you have nvidia drivers you shouldn't have any real hardware problem installing Ubuntu. So no, you won't have to install any drivers or do anything special unless there is a problem (then you post here and we try to solve that problem separately :)) Once you install Ubuntu, you will have to install graphics drivers separately (nvidia or ATI), as well as you will have to do a bit of work if you have to use wireless access to the internet. 

The partitioner built into both the live CD and alternate cd will let you resize any partition, but like any other technology there is always possibility that data is corrupted. Back up important data please, we are not responsible if something important is lost >.>.

As for the two cds, the Live CD is the one you shall download by default. You can also download the alternative CD should you encounter serious errors, its a little check box at the bottom of the main download. Have fun with ubuntu and tell us if you have a problem.

Hope I got all your questions. [Here]("http://www.psychocats.net/ubuntu/installing") is a good site for step by step install.

### Post by Teamgeist on 2007-05-11
Yes, the installer will take care of the resizing of the partition.
The LiveCD is the regular Ubuntu CD. Just put it in your drive and boot the computer up.
It will let you try Ubuntu without installing anything on your hard disk. It runs completely from the CD.
Then you can have a look if everything works fine and if so you just click install and the installer comes up and will lead you through the installation. Your hardware should be detected just fine and drivers will be installed for you.
In case your wireless doesn't work (if you have a wireless card): This seems to be an issue with some Dells but can be resolved most of the time. Just open up a new thread here in the forums if you run into any problems.

Good Luck. ;) 

P.S.: Make sure to backup all your important data from XP before installing. Just in case...

### Post by carlosqueso on 2007-05-11
Simple question first: The Live CD IS the ubuntu CD

As for drivers, your windows drivers (or, for the most part programs) WILL NOT work on ubuntu.  Most things should work automatically, especially sound, 2D graphics, wired networking and such.   Two things could be problematic.   First, wireless networking (although it's getting better all the time), second, 3D graphics.  If you have a nvidia or ATI card, there are drivers you can install, just ask how on the forums (or search the forums).  If you have the Intel graphics card with which some dells ship, I'm afraid you don't get 3D graphics.  If you have any problems getting hardware to work, go ahead and post here and we'll be happy to help.
