### Post by GWBouge on 2012-03-03
You can add .rar support to the built-in Archive Manager by adding the 'rar' package.  Just search for 'rar' in the Software Center, or in the terminal:

```
sudo apt-get install rar
```

Or, you can use an archiving tool such as Ark (also in the Software Center).
