### Post by Gattsu on 2008-07-13
I know this is an old thread, but I was looking for the same answer and found this thread on my way to the answer. 
In case someone new stumbles here, here's the answer:

```
rm -rf ~/.amsn
```

[http://www.amsn-project.net/wiki/FAQ#How_do_I_remove_aMSN_completely.3F](http://www.amsn-project.net/wiki/FAQ#How_do_I_remove_aMSN_completely.3F)
