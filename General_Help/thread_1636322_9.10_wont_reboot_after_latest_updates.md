### Post by drs305 on 2010-12-03
If your external was connected when the update ran, it may have installed Grub to the external or pointed to a partition on the external drive.

If Grub is still not working but you are now in your working Ubuntu, unmount and disconnect the external and run the following:
```
sudo dpkg-reconfigure grub-pc
```
You will be asked where to install grub-pc. Make sure your main drive (not the partition or the external drive) have the asterisk next to it and then tab to OK.
