### Post by Satoru-san on 2010-02-10
I am not even sure what you are asking... If you want to extract a rar file you can use the archive manager that came with your computer.

You can try this.

```
sudo apt-get install unrar
```

the cd over to the .rar file and do this

```
unrar e file.rar
```

hope this helps.
