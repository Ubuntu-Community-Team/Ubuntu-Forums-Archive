### Post by exclipy on 2006-06-17
Go to Preferences > Advanced > Downloads.  Most of the entries should be set to "Open with default application", with "kfmclient exec" in the disabled text box.  Click [Handlers for saved files].  There should be two entries - files and folders - both of which should be set to "kfmclient exec".
