### Post by jerome1232 on 2009-01-31
If you type "winecfg" into a terminal, it should come up with the tool to cofigure wine, on the first screen it brings you to, click add application, browse to the executable of the first program you would like to configure, now you can configure settings to use for that specific program. Just continue doing that for each program.
