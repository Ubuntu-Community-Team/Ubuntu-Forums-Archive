### Post by Idefix82 on 2008-10-05
Do I understand correctly that you want to use Ubuntu with GNome and with KDE? In this case you don't need to install two completely different OSs. You can first install Ubuntu and then install the KDE through Synaptic. At login you will be able to choose which Desktop to use under Session in the lower left corner. This way you don't need to install two versions of almost every piece of software.
