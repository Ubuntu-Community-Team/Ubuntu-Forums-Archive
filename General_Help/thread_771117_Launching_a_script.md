### Post by ryanhaigh on 2008-04-28
If you want to do thiois only for one particular device you will need to make a udev rule. [http://reactivated.net/writing_udev_rules.html](http://reactivated.net/writing_udev_rules.html)

If you want to do it for all devices you could use removable drives and media settings in system>prefs.
