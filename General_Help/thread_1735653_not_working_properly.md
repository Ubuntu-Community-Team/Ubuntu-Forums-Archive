### Post by TenPlus1 on 2011-04-21
Personally I thought that the learning curve for ubuntu was easier than windows 7...

Using the menu's in Metacity (the file manager) or even right-clicking the empty window will let you create directories or empty files to play with, same as windows...

To install themes, icon sets, cursors download the .tar.gz file from gnome-look.org and drag & drop it into the Appearance prefences window, that will isntall it and let you select it...

Screensaver and Power Settings are two different things, once sets the screensaver time, the other tells the system to turn off the monitor after a set time...

You will only be asked for a password whenever something important is happening, like changing a system setting or updating the software etc.  This stops nasties like viruses getting onto your computer...

Are you connecting your dell monitor BEFORE you switch on the laptop ?  Connecting monitors should never be attempted when the computer is ON...
