### Post by houseworkshy on 2013-04-25
If you can't see the windows files from the linux file browser you may have accidentially chosen to use the whole drive rather than install to the prepared partition. If you can see them then it's a boot issue which is easy to fix.  As you are new rather than ploughing through command lines try this [http://sourceforge.net/p/linux-secure/wiki/Home/](http://sourceforge.net/p/linux-secure/wiki/Home/)  It can be run live and has an excellent graphic boot repair utiltiy on it.
