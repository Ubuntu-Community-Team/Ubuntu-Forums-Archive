### Post by o_fortuna on 2006-01-09
[QUOTE=powdercarrot]OK.  I've tried rhythmbox, gtkpod, beep, xmms, etc. with just about every plugin suggested via my research.  m4p files downloaded via iTunes won't import or play![/QUOTE]
Songs downloaded on iTunes have restrictive DRM (digital rights management), so I'm not sure that they can be played directly without iTunes. You could google 'mp4 drm' and see if you can find a way to remove the DRM, which would allow them to play with rhythmbox or anything else you have.

To make sure that it works, you might try to find some non-DRM mp4 (aac) audio and try playing it with rhythmbox or anything else, to make sure that's not the root of the problem.

On second thought, have you tried using VLC with the right plugins? Just try searching for VLC in Synaptic. That should be able to play DRM AAC (m4p) files.
