### Post by Nytram on 2012-03-19
There's a [Programming]("http://ubuntuforums.org/forumdisplay.php?f=39") forum on this site with stickies which will answer some of your questions.

If you just want to develop for Linux/Ubuntu then I suggest learning Python. If you want something cross-platform I'd go with Java.

For editing there are some good IDE's available, but I'd recommend starting with something simple such as Geany. That way you're not learning the language and the IDE at the same time.
