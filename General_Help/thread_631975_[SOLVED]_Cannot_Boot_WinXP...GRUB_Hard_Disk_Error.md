### Post by el escocés on 2007-12-05
> the LiveCD will say it cannot open the WinXP partition, even though it sees it and recognizes it.  It says it cannot be mountedI'm not sure you need to mount the windows partition.  Anyway, I had exactly the same problem as you and here is how I fixed it.

Booted using a liveCD
Mounted my linux partition using the results of:
```
sudo fdisk -l
```Here's my result 
```
/dev/sda1              13        5483    43945807+  83  Linux
/dev/sda2            5484        5726     1951897+  82  Linux swap / Solaris
/dev/sda3   *        5727        7296    12611025    7  HPFS/NTFS
```So I did:

```
sudo mkdir /tmp/linux
mount /dev/sda1 /tmp/linux
```Now you have your linux mounted.  You have to now install grub again.

```
sudo grub-install --root-directory=/tmp/linux /dev/sda --recheck
```Obviously you replace the 'sda' with your drive lettering.

If everything goes to plan you should then check that the grub config files are correct.  First check the device map:

```
cat /tmp/linux/boot/grub/device.map
```Here is an example of mine, yours should follow your drive letter.

```
(fd0)   /dev/fd0
(hd0)   /dev/sda
```Then check your grub menu

```
cat /tmp/linux/boot/grub/menu.lst
```Here is mine, I just copied the important bits, if your windows partition is not there try inserting yourself, look at mine and try copying it using your configuration.

```
title           Ubuntu 7.10, kernel 2.6.22-14-generic
root            (hd0,0)
kernel          /boot/vmlinuz-2.6.22-14-generic root=/dev/sda1 quiet splash
initrd          /boot/initrd.img-2.6.22-14-generic
quiet

title           Ubuntu 7.10, kernel 2.6.22-14-generic (recovery mode)
root            (hd0,0)
kernel          /boot/vmlinuz-2.6.22-14-generic root=/dev/sda1 ro single
initrd          /boot/initrd.img-2.6.22-14-generic

title           Ubuntu 7.10, kernel 2.6.20-16-generic
root            (hd0,0)
kernel          /boot/vmlinuz-2.6.20-16-generic root=/dev/sda1 ro quiet splash
initrd          /boot/initrd.img-2.6.20-16-generic
quiet

title           Ubuntu 7.10, kernel 2.6.20-16-generic (recovery mode)
root            (hd0,0)
kernel          /boot/vmlinuz-2.6.20-16-generic root=/dev/sda1 ro single
initrd          /boot/initrd.img-2.6.20-16-generic

title           Ubuntu 7.10, memtest86+
root            (hd0,0)
kernel          /boot/memtest86+.bin
quiet

```Good luck!
