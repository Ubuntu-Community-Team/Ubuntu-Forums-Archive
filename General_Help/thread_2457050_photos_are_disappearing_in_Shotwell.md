### Post by pushbike06 on 2021-04-09
In requesting assistance:
**Specify the source of the files**. Camera or other storage device, local or remote. This will indicate possible download speed and/or capability.
**Specify the destination**. Computer (processor), size of RAM. May indicate capacity to process the download.
**The size of each file or the total size of the download**. Will indicate the possible time for the download to complete. Termination during download will likely cause loss of complete download. A download of 2Gb could take several minutes.

From a camera or similar local device just use File Manager and watch for the completion of the process.
To organise the files, delete, rename etc use Thunar File Manager/ Bulk Rename application(s). To process individual pic files use Fotoxx.
From a remote source download, use if possible FileZilla or similar application. Again wait for the process to complete.
