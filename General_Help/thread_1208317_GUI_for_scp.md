### Post by boondocks on 2009-07-09
Is there a secure copy (scp) GUI application available for the Ubuntu desktop that does the same thing as WinSCP ?

I have newbie user who needs to copy a lot of files back/forth between various Ubuntu desktops.
So the commandline activity would take too long.

---

### Post by hetx on 2009-07-09
Open up the default file explorer Nautilus. Click the address bar and write sftp://USER@IP-ADDRESS

Make sure SSH port is open =)

---

### Post by XCan on 2009-07-09
More or less any FTP client worth something will work as well. I personally use gFTP for your purpose. Just select SSH2 as protocol when you connect.
