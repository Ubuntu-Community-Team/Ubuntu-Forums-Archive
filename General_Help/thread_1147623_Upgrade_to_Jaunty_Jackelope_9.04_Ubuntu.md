### Post by geirha on 2009-05-03
If you want to upgrade from 8.04 to 9.04, you have to first upgrade to 8.10, then 9.04. If you've got 8.04 working as you like, I recommend you just stick with it. It's supported until April 2011. In April or October 2010 the next LTS will be released, and if you stick with 8.04 until then, you'll be able to upgrade directly to that LTS release. Look here for more information: [https://help.ubuntu.com/community/UpgradeNotes](https://help.ubuntu.com/community/UpgradeNotes)

### Post by srdjo on 2009-05-03
Installed it on 4 PC-es so far.

1. Toshiba Qosmio F50 - all went well, no configuration at all. All was working from the start.

2. Home PC (Q6600, 4GB, ATI Radeon 3650) - Installation went well, but spent 3 days trying to get ATI card do work in Xinerama (dual-head) mode. At the end I gave up and replaced it with nVidia 9500GT that works like the charm. Tried Hamachi (works), freeNX server (works, but has issues with arrow keys on keyboard)...

3. IBM LTSP server - Install went OK, but had many problems installing network printer and printers on clients - [Thread...]("http://ubuntuforums.org/showthread.php?t=1133924&page=2") - spent whole day trying to find the solution, but didnt find any, so if you have some network printers, you should think twice....

Overall 9.04 was a great upgrade until I got stuck with network printers (imagine doing business without printers), but I hope it will be resolved soon.
