### Post by mikewhatever on 2007-06-24
Are you talking about keyboard layout? If so, I think the default Ubuntu key combination is Alt+Alt. If you want to switch the interface language, do it before you log in from the option menu in the bottom left corner. Also, keyboard layouts and and interface languages are two different things. If you use Hebrew as your interface language, you still may need to enable Hebrew keyboard layout.
