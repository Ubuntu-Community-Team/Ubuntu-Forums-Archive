### Post by grahammechanical on 2011-11-11
What machine will you use to make the presentation? If you use a Ubuntu machine to present a presentation made in Libreoffice then everything will display as you wish it to.

You will not be able to use a Windows machine to present a Libreoffice presentation as Microsoft Powerpoint may not display it correctly. Your friends could not use your Ubuntu machine to make their Powerpoint presentations for the reasons you have mentioned.

This situation exists because Microsoft insists on developing its own file formats instead of using open formats.
