### Post by jlochhead on 2009-03-19
Are you using 64 bit Ubuntu with the flash-nonfree (32 bit) from the repositories?

If so you should try the new 64 bit alpha flash.

Download:

[http://download.macromedia.com/pub/labs/flashplayer10/libflashplayer-10.0.22.87.linux-x86_64.so.tar.gz](http://download.macromedia.com/pub/labs/flashplayer10/libflashplayer-10.0.22.87.linux-x86_64.so.tar.gz)

Move it to the directory ~/.mozilla/plugins

Restart Firefox.

Also try the NoScript add-on. This stops scripts executing by default (you can whitelist sites if you want), this will speed up browsing.
