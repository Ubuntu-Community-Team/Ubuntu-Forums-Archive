### Post by iaculallad on 2008-07-25
Say we are to automount the /dev/sdb1 partition:

First step is to install ntfs-3g utility:

```
sudo apt-get install ntfs-3g
```

Secondly, create a mount point for the SDB1 partition: we name it as 
ntfs_1.

```
sudo mkdir /media/ntfs_1
```

And lastly, we open your /etc/fstab file for editing:

```
gksudo /etc/fstab
```

and insert the line of text below at the last-part of your fstab file:

> /dev/sdb1 /media/ntfs_1 ntfs-3g defaults 0 0

For ending:

```
sudo mount -a
```

### Post by Rocket2DMn on 2008-07-25
If you want to use fstab entries, they need to be like formatted like this:
```
# Entry for /dev/sdb1 :
UUID=708C3CCE8C3C9098 /media/stuff ntfs-3g defaults,umask=007,gid=46 0 0
# Entry for /dev/sdc1 :
UUID=9470CAA670CA8E84 /media/stuff2 ntfs-3g defaults,umask=007,gid=46 0 0
```
the fstab file shows you the layout:
```
<file system> <mount point> <type> <options> <dump> <pass>
```
You were missing <type> and had an extra 0 attached.
