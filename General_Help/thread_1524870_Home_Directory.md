### Post by SoFl W on 2010-07-05
Your personal home directory?
It is under "Places"  (home Folder)  on the top menu.

In the file system it is under "/home" or "/home/USERNAME" for each user.

### Post by cavalier911 on 2010-07-05
Click the folder with the cursor on it(tooltip file manager if you hover your mouse cursor over it) next to the menu button(lower left,think quick launch toolbar in winxp) to launch pcmanfm the equivalent of nautilus in gnome.PCmanfm opens with the logged in users home folder open. Another way is Menu/Accessories/File Manager
