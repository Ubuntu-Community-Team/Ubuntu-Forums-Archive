### Post by coffeecat on 2015-09-20
I've moved a couple of your posts where you refer/link to your now removed duplicate thread.

You have the install CD included in your sources and apt-get can't find it. Open the software centre &#8594; Edit menu &#8594; Software Sources &#8594; Ubuntu Software Tab. Untick the CD-ROM.

**Edit:** - that's assuming Ubuntu Mate includes the Ubuntu Software Centre, which I believe it does. If you can't find what I'm referring to, post back and someone with Ubuntu Mate experience will be able to point you to the right place.
