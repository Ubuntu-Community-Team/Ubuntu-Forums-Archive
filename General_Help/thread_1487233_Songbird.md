### Post by lrcaballero on 2010-05-18
Hey guys I was able to Forced songbird 1.4.3 to play all of my m4a files by doing the following:

you can open a terminal change to your Songbird directory, e.g.

$ cd ~/Songbird

and then type

$ export SB_GST_SYSTEM=1

and then start Songbird from within the terminal:

$ ./songbird

you can also read more about this issue from here:
[http://getsatisfaction.com/songbird/topics/playing_m4a_files_on_ubuntu](http://getsatisfaction.com/songbird/topics/playing_m4a_files_on_ubuntu)

This is an issue with Songbird NOT Ubuntu Lucid Lynx 10.04.
