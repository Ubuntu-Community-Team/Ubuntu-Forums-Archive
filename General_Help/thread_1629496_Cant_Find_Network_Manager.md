### Post by efflandt on 2010-11-23
It is usually System > Preferences > Network Connections at upper left panel.

Are you the original user that you set up when you installed the system or a different user?

If the latter maybe you need to use the original user to give your other user permission to do that.

Under **Advanced Settings** for that user, under **User Priviledges** see if "Connect to wireless or ethernet networks" is checked.
