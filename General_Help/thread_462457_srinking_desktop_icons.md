### Post by GhostZrydA on 2007-06-02

Start Nautilus. Go to Edit/Preferences. In the Views tab, under Icon View Defaults, set Default Zoom Level to suit yourself. I usually set mine to 75% (tho I always turn off all the desktop icons and run my Nautilus in List View mode).
