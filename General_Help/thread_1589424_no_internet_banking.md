### Post by efflandt on 2010-10-07
Any particular bank that gives you issues?  I have no problems using 64-bit Firefox in Lucid with Chase or Bank of America.  But the PC I have used for that is an old Athlon64 lacking lahf_lm used by 64-bit flash, so I just use flashplugin-installer (32-bit flash w/nswrapper).

For Chase before you can connect from a system/browser you have not connected with before, the website first has to e-mail you a code and you have to enter that code first (which likely plants a cookie) before you can use your normal login.  So that is only a minor issue if you install new Ubuntu versions fresh.  Not sure if it is an issue if you upgrade versions instead or use your old home directory.  No problem at all for just browser version update.

Bank of America has something so you can tell if you are connected to their site (not a phishing site) where they show a picture you selected and play a related sound.  The sound would not play in Linux, but that did not seem to be an issue.

I did have a problem last time I connected to AT&T.  I could pay my phone/internet bill, but could not "view" the detailed bill from a number of different links on different pages (Firefox kept saying connection reset).  But after going to customer service looking at some faqs and thinking about asking them about the problem, I logged in again and it worked.
