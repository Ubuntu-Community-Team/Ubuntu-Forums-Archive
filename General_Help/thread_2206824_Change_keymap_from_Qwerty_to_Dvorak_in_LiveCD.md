Is there a simple way (ie without having to edit config files) to change the keyboard setting from Qwerty to Dvorak when booted with LiveCD?

### Post by papibe on 2014-02-21
Hi gdesilva.

Open 'Keyboard Layout', then press the '+' on the bottom left to add a new keyboard layout. There are several Dvorak layouts available.

Hope it helps. Let us know how it goes.
Regards.

### Post by gdesilva on 2014-02-21
@papibe, Thanks a lot for the quick response and the solution which is what I was after.

Just for others information, there were 4 listings (English) and I could not add a new one. So I deleted one of the entries and then it let me add English Dvorak.

Thanks again.


EDIT: The above issue was found with 13.04 and there were no such issues in 13.10
