### Post by oldos2er on 2011-03-05
Isn't Winrar a Windows program? If so, it won't run natively on Linux. You can add rar/unrar capability to Archive Manager by running ```
sudo apt-get update && sudo apt-get install rar unrar
``` in a terminal.

There is a PPA for jdownloader here: [https://launchpad.net/~jd-team/+archive/jdownloader](https://launchpad.net/~jd-team/+archive/jdownloader)
