### Post by Dennis N on 2018-06-29
The extension I installed is 'Window List' by fmuellner. Try that one. 

[https://extensions.gnome.org/extension/602/window-list/](https://extensions.gnome.org/extension/602/window-list/)

I checked this extension and it works fine in my Ubuntu 18.04. It has a button for each open window. I have had it turned off since I like to use either the Alt-Tab switcher or Activities Overview screen.  (I think the best switcher is 'Alternatetab' by fmuellner. It doesn't group by application.) 

Note: I install my extensions directly from the Gnome Extensions web site.
