### Post by Simian Man on 2009-02-12
If you need to write Visual Basic programs for school, I'd really recommend doing so with Windows.  Visual Basic is not like langauges like Python where there is a cross-platform reference implementation or languages like C++ (or even C#) that are standardized and used on many systems.

No sense risking something going wrong and getting a bad grade just for using an alternative implementation.
