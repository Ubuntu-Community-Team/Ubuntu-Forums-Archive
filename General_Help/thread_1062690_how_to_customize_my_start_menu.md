### Post by konqueror7 on 2009-02-07
hello, you could try using the single main menu applet instead of the three-part main menu, just right-click you top panel, and click add to panel, look for "Main Menu"...also if you don't want it, you could try using USP (haven't tried it myself, i'm contented with my main menu;)), [USP Ubuntu Thread]("http://ubuntuforums.org/showthread.php?t=222546")...and also, why not just install kubuntu desktop, you can still login in your "default" gnome ubuntu, and also login in kde desktop...

```
sudo apt-get install kubuntu-desktop
```

hope it helps...
