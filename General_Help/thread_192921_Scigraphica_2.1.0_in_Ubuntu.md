### Post by neutro on 2006-06-20
The current version of the Debian package is also 0.8.0 in Debian unstable. A quick check on the SciGraphica homepage tels us that this version is not a "very old" version, it's simply the previous version. I think they bumped the version number from 0.8 to 2.1 when they revamped the UI to use GTK+2. Admittedly, I guess SciGraphica (i.e. its interface) would look much better in version 2.1.0, but using 0.8.0 is not really using outdated software.

Ok I checked back and 2.1.0 was realeased last year, so I guess it would be nice for the package maintainers to update the .deb... At least scigraphica is not listed in the "orphaned packages" list.
