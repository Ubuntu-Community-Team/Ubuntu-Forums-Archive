### Post by Chriske on 2010-11-05
Well, if it's of any use, I had exactly the same problem.

After trying most every suggestion I could find on the Web I finally solved it by **installing Opera**.

What I mean is I installed Opera and opened NY Times (and other) pages in Opera. Then I tested these pages again in Firefox and now it works and so does Chrome!!!

I don't know why this should be. All I know is that this installation seemed to fix the problem. I also noticed that this also fixed an issue in Firefox when other pages would load, but the progress bar still showed them as loading - as if they hadn't quite finished.

I don't need to use Opera now, but I'm going to leave it on my machine so as not to disturb any fix that has occurred!
