What exactly are you having difficulties with? As you posted is so vague that nobody can give you specific advice.

And, of course, you don't need a previous version to install a newer one as you thread's title seems to imply. You can *upgrade* from one version to the next without any installation media. However, LTS releases like 14.04, by default, do not alert the users about upgrades to interim releases (like 15.10), only to newer LTS releases (like 16.04). This setting can be changed if you want to upgrade online from let's say 14.04 to 14.10 to 15.04 then finally to 15.10... By now you certainly noticed the problem... Exactly! Had you performed such upgrades in sequence and most likely everything would have happen painlessly. If not, them now is NOT the time to consider this. If you want 15.10 an installation from scratch is advisable (backup first) -or- you can just wait until April and upgrade online.

I do not understand what you are trying to do. Do you want to upgrade 14.04.3 to 15.10?

In 14.04.3 the file manager should be able to access what is on the DVD. It does on my system. Are you unable to access the contents of the DVD? We do not normally use one version of Ubuntu to install another version. We usually boot the DVD and install Ubuntu from an Install session running off the DVD. 

Are you trying to upgrade 14.4.3 from the 15.10 DVD? In 3 months time Ubuntu 16.04 will be released and you can upgrade 14.04 directly to 16.04 which will have 5 years support for security fixes. If you upgrade to 15.10 you will have to upgrade to 16.04 by July this year because 15.10 only has 9 months support starting October 2015.
