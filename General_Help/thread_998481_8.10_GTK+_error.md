### Post by geofurtado on 2009-02-03
I was getting the same error right after an Intrepid install.  I opened Synaptic and searched on "themes" and reinstalled all of the "theme-related" packages and restarted the machine.

The problem has been solved.  Apparently, there were a couple of packages including Metacity that had a red "X" next to them.  The "X" is now gone and the themes are working.

I used the same install media on another machine and did not get the problem on that machine.
