### Post by jerome bettis on 2005-02-07
you can try swapping the following with the existing synaptics touch pad section in that file.  i'm not sure if it will work for you, but it works great on my laptop which also has a synaptics touch pad.  hope it helps

Section "InputDevice"
        Driver      "synaptics"
        Identifier  "Touchpad"
        Option      "Device" "/dev/psaux"
        Option      "Protocol" "auto-dev"
        Option      "LeftEdge" "1700"
        Option      "RightEdge" "5300"
        Option      "TopEdge" "1700"
        Option      "BottomEdge" "4200"
        Option      "FingerLow" "25"
        Option      "FingerHigh" "30"
        Option      "MaxTapTime" "180"
        Option      "MaxTapMove" "220"
        Option      "VertScrollDelta" "100"
        Option      "MinSpeed" "0.06"
        Option      "MaxSpeed" "0.10"
        Option      "AccelFactor" "0.0010"
        Option      "SHMConfig" "on"
        Option      "UpDownScrolling" "on"
        Option      "CircularScrolling" "off"
        Option      "LockedDrags" "off"
        Option      "TouchpadOff" "off"
        #Option      "Repeater" "/dev/ps2mouse"
EndSection

you might have to tweak those numbers, but hopefully that's good enough to get you started.

edit:  looking at that again, i had the same stuff in my config file right after i installed ubuntu and it performed exactly as yours is.  i couldn't scroll, or tap the pad for a single click which is really sweet.  i deleted that section and put this in there and it's great, so it's very likely it will work for you.
