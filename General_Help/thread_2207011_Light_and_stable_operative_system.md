### Post by sudodus on 2014-02-21
Please make a more descriptive title for your next thread! It helps attract those who know about that particular problem or question.

1. I think it depend very much on what you have installed in Windows, how many processes that are running in the background, and how clean you keep it.

Many people here are more concerned about comparing various flavours of the Ubuntu family and other linux distros. Usually the ***desktop environment*** makes the difference is speed and responsiveness.

Lubuntu has the ultra-light LXDE desktop environment
Xubuntu has the medium-light XFCE desktop environment
Ubuntu has the heavy but fancy Unity desktop environment
Kubuntu has the heavy but fancy KDE desktop environment

but all desktop environments can be tweaked with more or less graphical effects 'bells and whistles' compared to the standard installation.

2. We can guess from the current daily build, that standard Ubuntu might be slightly lighter, while for example Lubuntu will be slightly heavier, but we don't know until it is released. The engine 'under the hood' will be a bit heavier, but the Unity desktop will probably be lighter in 14.04 than 12.04.

Anyway, for a particular computer it is usually good to stay with the same version of Ubuntu, as long as it is supported, and to use long time support versions (LTS), because they are more polished and debugged. 12.04 is an LTS version and 14.04 will be when released in April.
