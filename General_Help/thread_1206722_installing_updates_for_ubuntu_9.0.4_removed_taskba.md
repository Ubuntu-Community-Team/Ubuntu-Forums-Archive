Had taskbar at top of screen.

home, file system, trash, and floppy drive.

How do I get the taskbar back?

### Post by dawnlove on 2009-07-07
open up run command [alt & f2} then type    xfce4-panel  ,then make sure you have [automatically save session on logout] checked in [session and startup] settings. you might intermittantly need to use this command in xubuntu
