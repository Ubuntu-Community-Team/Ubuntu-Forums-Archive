### Post by Murdoc_of_puts on 2009-09-12
Hello.
I'm having an annoying problem with Gedit.  Everytime I save an edited file it creates another hidden file with a ~ behind it.  So it's like it's copying the file every time I save it.  This is just taking up extra room on my computer.  How do I make it stop doing this?  Thanks.

EX:  thing.txt thing.txt~

---

### Post by b0b138 on 2009-09-12
In gedit click edit - preferences - under the editor tab uncheck "create a backup copy of files before saving"
