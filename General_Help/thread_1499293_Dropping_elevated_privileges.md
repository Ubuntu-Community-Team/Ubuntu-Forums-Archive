### Post by WorMzy on 2010-06-02
This is the default behaviour in Lucid; partitions no longer require you to put a password in to mount them. If you'd like to be prompted for your password, then please read post #4 in this topic: [http://ubuntuforums.org/showthread.php?t=1467629](http://ubuntuforums.org/showthread.php?t=1467629)

As for the key appearing in the notification area; I still see it periodically, but I don't really pay attention to it, so I can't offer any advice regarding it, sorry.
