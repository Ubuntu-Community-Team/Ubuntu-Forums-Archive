> **gettinoriginal said:**
> I also understand that Turbo Tax runs in Wine.  I have found this, but have not tried it:
[https://help.ubuntu.com/community/TurboTaxOnline](https://help.ubuntu.com/community/TurboTaxOnline)
I am using Turbo Tax 2009 Deluxe by installing it with VirtualBox and XP. Just install virtualbox-3.1 and then install a copy of XP into it. Set the RAM allocation to at least 256 MB (I used 512 MB). Read the users manual to set up CDROM drive, printer, USB and a shared folder. The users manual will be found at /usr/share/doc/virtualbox-3.1. When you get XP working with your CDROM drive, printer, shared folder, and maybe a USB drive, install Turbo Tax. The Turbo Tax install seems to take an hour or so.
