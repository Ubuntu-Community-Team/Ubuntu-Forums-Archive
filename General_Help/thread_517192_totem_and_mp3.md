### Post by aysiu on 2007-08-04
**Step 1**:
[Enable extra repositories](http://www.psychocats.net/ubuntu/sources)

**Step 2**:
Paste this command in [the terminal](http://www.psychocats.net/ubuntu/terminal): ```
sudo apt-get update && sudo apt-get install ubuntu-restricted-extras
```

You may also want to check out [this thread](http://ubuntuforums.org/showthread.php?t=194114&highlight=MPEG+1+Layer+3+CBR%2527+is).
