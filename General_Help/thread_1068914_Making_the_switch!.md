### Post by avtolle on 2009-02-13
You need to work on unmounted partitions. The best way, IMO, to do this is to use a live CD (either the Ubuntu install cd with gparted on it) or the Gparted Live CD; boot your system from the Live CD, open gparted, and then select the partition upon which you want to work, and enlarge it, whatever. Good luck.
