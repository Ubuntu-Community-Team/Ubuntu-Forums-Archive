### Post by strabes on 2007-05-08
If you're just writing a paper with then you shouldn't have any problems. Just save it as a .doc in openoffice. Make sure to save your file often (and enable auto-save in the OOo options). Even if it does crash, OOo has a really good recovery system.

On another note, I was surprised to learn that my university has openoffice.org on all of its computers. They all run debian linux and vmware windows. It's crazy.
