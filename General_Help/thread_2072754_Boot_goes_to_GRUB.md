### Post by sienile on 2012-10-18
Since it's a fresh install, I'd say just reinstall. It seems that it messed up somehow and is going to GRUB recovery console.

If you know what partitions are and feel comfortable with making one for Ubuntu, I'd install from boot instead of using the Windows installer (Wubi). Wubi is for ultra-newbies that would otherwise screw up their system. You will have a lot less problems with a direct install.

If you feel that you are one of those "ultra-newbies", there is the stickied "Wubi mega-thread" which might have some info to help you.
