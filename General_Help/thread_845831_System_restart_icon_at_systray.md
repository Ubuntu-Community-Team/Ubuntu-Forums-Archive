> Hi,
  If I do a system update from the command line, how do I know if I need a system restart? How does this system restart systray icon do its job?


A restart is needed when there is a kernel update or a very important security update (like the one for ssl last week) and these are rare events. 

When you click on the systay icon a window pops up and asks you something like "a system restart is needed - do you want to restart" and you have OK, CANCEL buttons at the bottom.
