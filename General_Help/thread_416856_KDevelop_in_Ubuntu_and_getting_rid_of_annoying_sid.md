### Post by KeyserSoze93 on 2008-04-06
you need to righclick the toolbar at the bottom and select unlock. the right click and go to menu options. there should be a setting somewhere for "panel" which will fix the sidebar.

kdevelop is not actually the compiler, it is an integrated development environment. for compilation it just calls the gnu c compiler "gcc" or the c++ compiler "g++" which should actually be installed by default.

as well as kdevelop, you will also need the package "build-essential", although that might be installed automatically anyway.

you should be able to just run "sudo apt-get install kdevelop" to install it.
