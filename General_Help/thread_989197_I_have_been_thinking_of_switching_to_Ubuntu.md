### Post by aysiu on 2008-11-21
If you have 1 GB of RAM or more, I would advise [installing Ubuntu as a virtual machine inside Windows](http://www.psychocats.net/ubuntu/virtualbox). There are a few downsides to doing this (the main one being the amount of memory it uses up), but it's a pretty failsafe way to try out Ubuntu and not have to worry about hardware compatibility.
