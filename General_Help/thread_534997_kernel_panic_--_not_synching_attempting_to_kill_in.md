### Post by K.Mandla on 2007-08-26
Hi. If you load any live CD -- Ubuntu, Kubuntu or Xubuntu -- you should be able to access the files on that drive. You could copy them on to a USB drive too, if you like.

Alternatively, you can recover the Windows partition with a Windows system disc, and run fixmbr when the setup procedure begins. Search the forums for the term "fixmbr" and you'll probably get about 250 threads that describe how to use it. ;)
