### Post by deadflowr on 2017-12-01
File transfers should now show an icon in the top header bar of Files (on the right side near the Search icon)
The progress indicator looks like an analog clock timer (or egg timer) instead of a bar.
When done a popup dialog-like box should show at the top area of the main window area of Files.

I'm not sure if this needs to be set anywhere as I cannot remember ever setting such a thing.
