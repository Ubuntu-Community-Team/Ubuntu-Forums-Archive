### Post by ctg on 2009-10-14
I'm running Ubuntu 9.04 on an HP Pavillion dv9000 laptop.

I find that when I press various function keys (F5 to F8), the Rythmbox Media Player opens up. This is not something I have configured myself, so I assume this is some out-of-the-box setting that I would need to disable. Where would I look for this setting, and how do I disable it?

The odd thing is that it doesn't happen every time, nor is a set number of key presses that does it.

Any help would be appreciated - I have searched these forums and the internet, but I can't find anything to help.

### Post by prem1er on 2009-10-14
I'm assuming your using Gnome.  Go to System>Preferences>Keyboard Shortcuts. The shortcuts are listed by groups and you can add/edit/delete them at your choice.

### Post by undecim on 2009-10-14
Look in System -> Preferences -> Keyboard Shortcuts

Find the offending actions (for example, "Launch Media Player" probably launches rhythmbox) and change them to the shortcuts you prefer, or use the backspace key (as if you were trying to assign the action to the backspace key) to disable the action completely.
