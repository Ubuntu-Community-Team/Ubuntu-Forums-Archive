### Post by Frogs Hair on 2013-02-03
Try going to the settings drop-down in Chromium and in history clear all browsing data. This will clean the browser cache also. Then, go to the Chrome web store and install Ghostery or Adblock Plus and follow the setup instructions.

Restart the browser and see if the popup returns again and if it does disable any other extensions that are suspect if you have any installed.
