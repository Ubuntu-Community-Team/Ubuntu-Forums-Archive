### Post by grahammechanical on 2013-08-31
To what do you want to add new words to? Ibus is an input method.

> [COLOR=#333333][FONT=Ubuntu]You can use IBus to type in your own language in most GUI applications.[/FONT][/COLOR]

[https://help.ubuntu.com/community/ibus](https://help.ubuntu.com/community/ibus)

System Settings>Language support lets us change the input method from Default to ibus.

> [COLOR=#3C3C3C][FONT=sans-serif][SIZE=2]To type certain languages, such as Chinese, Japanese or Korean, a more complex input method than just a simple key to character mapping is required. Through the [COLOR=#6C6C6C]Keyboard input method system[/COLOR] drop-down list you can set a framework for input methods that will be started automatically at login. The recommended input method system for Ubuntu is &#8220;IBus&#8221;. If you want to use alternative systems, install the corresponding packages first and then choose the desired system from the drop-down list.[/SIZE][/FONT][/COLOR]
[COLOR=#3C3C3C][FONT=sans-serif][SIZE=2]If the &#8220;default&#8221; option is selected, Ubuntu makes a guess, and thus starts an input method system at login if the current language is Chinese, Japanese, Korean or Vietnamese.[/SIZE][/FONT][/COLOR]


That is a quotation from the Ubuntu Desktop Guide.

Regards.
