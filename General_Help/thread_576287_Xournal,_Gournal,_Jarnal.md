### Post by yosef on 2007-10-17
Well, from my experience of the last few days, I would choose jarnal over xournal.  I have not tried gournal.  I am running xubuntu on a fujitsu stylistic lt p600, using the fpit driver for the touchscreen.  Xournal seems to react strangely, changing the pen tip to an eraser occasionally when I double back on a line (crossing a 't' for example).  Jarnal does not do this.  I have not explored all of jarnal's functions, but it seems to me that it fulfills all of my notetaking needs.  I think perhaps all of them allow one too write on a pdf as a background, which is very convenient for me as several of my schoolbooks are in pdf format and I like scribbling on books.  If anyone has any other notetaking suggestions, please post.

### Post by Ferret-Simpson on 2008-06-02
Xournal (Especially the recent builds) is as stable as jarnal and much faster. Not to mention the subpixel resolution.

However, it only works on Linux. It only reads and writes it's own file formats and it's completely incompatible with all other notetaking programs.

If it gets a Mac OS X port, or Jarnal file format compatibility - or the reverse, then it will be a better choice. Unfortunately, I now have to migrate back from Xournal to Jarnal - since I've fallen in love with the machines I sell! At least the Sun/Apple VM is good.
