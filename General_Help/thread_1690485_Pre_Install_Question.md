### Post by lisati on 2011-02-18
Welcome to the forum.

Most of the applications you're accustomed to in Windows have Ubuntu-friendly equivalents, some of which are installed automatically. For example, a "standard" installation of Ubuntu can happily display PDF files without the need to install extra software.

As a rule, applications designed for Windows won't work "out of the box". *Some* (but not all) can be used with the help of programs such as "Wine" and "PlayonLinux."

### Post by sikander3786 on 2011-02-18
Every Operating System has its own pros and cons. You've to make your own choice.

Most of your Windows applications, I say most, won't work on Ubuntu or any Linux via Wine or PlayonLinux. Some may work equally well as they do in Windows and some may install correctly but act weird. So you have to make a choice if you have time to make the switch to the open source, Linux native applications. You will need some time to understand them. They function equally well as the proprietary ones but tools, options etc might not be in the same exact place.

And for the operating system itself, it is not as easy as switch from one version of Windows to another. You'll find tools, settings, configuration everything in places not the same as in Windows. Software installation is different and all that stuff.

If you think you are not ready to make a direct switch at the moment, I'd advise to install Ubuntu in a dual boot to Windows so when you've got time, you can switch on Ubuntu for fun and learn that way. Once you know the basics, you can take the plunge.

I have no hesitation to say that Linux is much more powerful OS than Windows. Once you start using and understanding it, you'll enjoy it.

Or for running Windows apps in Ubuntu, there is another choice i.e, installing Windows inside Ubuntu using some Virtual Machine Server such as VirtualBox. See the link in my signature.

Hope you enjoy your stay with Ubuntu!
