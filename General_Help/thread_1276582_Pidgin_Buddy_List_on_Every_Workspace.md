### Post by wojox on 2009-09-27
Right click the Window List 

Select Preferences

Select Show windows from current workspace.

### Post by edouardp on 2009-09-27
This is a [known bug]("https://bugs.launchpad.net/ubuntu/+source/pidgin/+bug/346840").
The best workaround I found for Jaunty is to install pidgin from [their own PPA]("https://launchpad.net/~pidgin-developers/+archive/ppa"), you will get a higher version where this bug is fixed.
