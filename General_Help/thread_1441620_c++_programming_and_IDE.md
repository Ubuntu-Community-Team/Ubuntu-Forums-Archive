### Post by john newbuntu on 2010-03-29
Typically in an IDE, you create a project and import one or more folders into it.  Once this is done, you should be able to see all your files on a panel on the screen.  You would then double click the file that needs to be edited and make your changes there.
If you need to add more files to the project, there should be an option to add more files/folders.  Use it and continue coding/making/linking etc.
