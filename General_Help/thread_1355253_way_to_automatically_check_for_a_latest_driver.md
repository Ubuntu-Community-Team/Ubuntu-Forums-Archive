### Post by synapsys on 2009-12-14
What drivers are you looking for?

Most 'drivers' are built into the kernel. Ubuntu automatically tells you when your kernel and/or packages need to be updated. As long as you are keeping up with the latest updates, you should be alright. Unless you install something from source or via svn, it will be updated for you.
