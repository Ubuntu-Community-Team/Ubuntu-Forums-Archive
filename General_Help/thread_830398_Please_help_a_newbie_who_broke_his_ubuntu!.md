### Post by drs305 on 2008-06-15
Your medibuntu sources file has been corrupted. Rename the corrupted file and then reinstall the medibuntu repository:
```
sudo mv /etc/apt/sources.list.d/medibuntu.list /etc/apt/sources.list.d/medibuntu.list-bak
```

Reinstall medibuntu repository:
```
sudo wget http://www.medibuntu.org/sources.list.d/gutsy.list -O /etc/apt/sources.list.d/medibuntu.list
sudo apt-get update && sudo apt-get install medibuntu-keyring && sudo apt-get update
```

Once you get things working again delete that -bak file as it won't do you any good in the future. ;-)

Also, medibuntu now uses its own directory and file (medibuntu.list) with sources.list. If you run an installation and get some kind of 'duplicate' error message, it means you probably have a reference to medibuntu in /etc/apt/sources.list. If so, just comment it out.
