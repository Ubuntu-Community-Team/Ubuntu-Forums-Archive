### Post by mihai.ile on 2009-05-05
the envelop thing is not the pidgin's icon.
It's an applet used to store all applications like messengers, mail notifiers, etc.
It does not appear when it does not have any application to show.

Now to enable the old pidgin icon just go to the preferences (in pidgin menu) and you have an option to display the icon (I think the default is never, so change to always)

If you don't want the redundant envelope icon just remove it by right click and remove option.
