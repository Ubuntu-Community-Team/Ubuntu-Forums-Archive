### Post by justin whitaker on 2008-02-25
> **m4mb4 24 said:**
> why?

Well, if you are still dual booting, you must have some program on Vista you want to keep running. Unless you are completely comfortable going open source and/or futzing with Wine for those applications you still need, you are going to want that Vista install back at some point. 

If you have alot of drive space, then just forget that Vista is there, and give Ubuntu 3 months. IF you find you really no longer need windows, then use Gparted to make the Vista partition into a partition for all your media stuff, or whatever.
