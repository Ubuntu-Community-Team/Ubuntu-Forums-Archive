### Post by newyorkcityjay on 2007-05-01
I take online courses and want to be able to create a quick link to my Blackboard websites.  I normally use Firefox as my default browser - but have had massive problems getting some of the Java applets to work and I don't really want to worry about it now... Epiphany works great and loads the pages quicker as well.  The problem is when I create a launcher it wants to open in Firefox because that's my default - how do I code the launcher to open in a URL in a different browser?

### Post by Ziggy72 on 2007-05-01
the default browser is set up in System > Preferences > Preferred Applications. The Internet tab will allow you to enter a command to make your favorite browser the default.

---

### Post by newyorkcityjay on 2007-05-02
Right, but that's not what I was trying to do - I didn't want to change my default browser.  I want Firefox to stay as default, but have a desktop or panel launcher to open a URL in Epiphany.  I did figure this out on my own - it's actually pretty simple.

In the command line I entered:

epiphany [http://blackboard.gwu.edu](http://blackboard.gwu.edu)

and it worked fine.
