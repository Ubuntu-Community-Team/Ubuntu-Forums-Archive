### Post by alof8k on 2005-02-11
I don't know about mplayer, but if you want to watch .mov files, you can try using xine.   it works great for me.  What i did was first I installed the win32 codecs, then the xine package (both of which can be instructed through [www.ubuntuguide.org](www.ubuntuguide.org) ).  Once I had those two, I installed gxine (  sudo apt-get install gxine )  .. make sure firefox is closed.  Once you have installed these 3 software, open firefox, goto [www.apple.com/trailers](www.apple.com/trailers)  and try watching a trailer.  if its working properly, gxine will open up and play the trailer.  Then you can just put it to fullscreen and enjoy  :-)  .
