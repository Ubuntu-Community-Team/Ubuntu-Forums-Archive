### Post by gingermark on 2006-02-17
Linux doesn't use file extensions. Instead, it reads the header of a file to determine what file type it is. Granted, file extensions are used sometimes, but they aren't necessary.

Whilst there are some Windows drivers that can be made to work in Linux (for example, ndiswrapper can make some Windows wireless drivers work) as a rule they won't.

However, there are many open-source drivers available for Linux, and some manufacturers produce their own ones. For example, nVidia supports their graphics cards in Linux.

There is the [Linx Questions' Hardware Compatibility List (HCL)](http://www.linuxquestions.org/hcl/index.php), but unfortunately they don't have your laptop listed.

However I did find [this page](http://www.geocities.com/randomnumbergenerator2001/), which suggests that everything should work reasonably except for power managrement.

EDIT: In fact, they have a section especially for [(K)ubuntu Breezy!](http://www.geocities.com/randomnumbergenerator2001/#ubuntu_breezy)
