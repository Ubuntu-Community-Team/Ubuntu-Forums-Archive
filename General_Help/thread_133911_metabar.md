I've been bothered by that same question, but only had the time to search on that now. Here's what I found:

Taken from [QUOTE=http://www.freebsdsoftware.org/deskutils/metabar.html
Metabar is a Konqueror sidebar plugin that displays contextual information and actions for selected files.[/QUOTE]

In short, if you remember the Windows XP "My Computer" Explorer, the info at the left side of the window that changes each time you select a file/files and offers different actions.
