### Post by lukeiamyourfather on 2011-02-14
Way too much in a single post, hard to wade through all of it. I'll tackle the first bits. Your motherboard and other driver discs are useless because they probably have drivers only for Windows and these will not work in Linux. Chances are the things you need drivers for are available in the repositories. The "Hardware Drivers" utility should list any available proprietary drivers that are not already installed, like for your graphics card. Just click the button to install them and restart when done.

As for the other hardware it is probably already installed and ready to go. Drivers for most hardware come with the Linux kernel or as packages in the repository. The only ones not installed for you are the ones that are proprietary. Some less common hardware might not have any Linux support though (open source or manufacturer).

**This is in the repositories and can be installed through the software center** - Audiacity (A program to edit songs)
