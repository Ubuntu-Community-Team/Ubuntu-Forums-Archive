### Post by Dennis N on 2013-07-13
The CD information comes from an internet database and is not on the audio CD itself. Has ripperX retrieved information for other CDs prior to this? Or, try it with another well known CD to be check this.

Optionally, tag editors like audio tag tool or EasyTAG allow you to enter track information yourself. Only takes a few minutes per CD. Common information (such as artist or album name) can be filled on all tracks at once from one entry.
