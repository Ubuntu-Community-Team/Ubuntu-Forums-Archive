### Post by stoneage on 2010-11-29
Wine installs applications to .wine/drive_c/Program Files

Ubuntu will install to its root file system. An easy way to find things is ```
whereis APPLICATIONNAME
``` in a terminal.

Or open Synaptic, right click on the application and choose Properties -> Installed Files.
