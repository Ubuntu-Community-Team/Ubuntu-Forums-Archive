### Post by FluffyElmo on 2007-05-04
GQView will do randomized slide-shows. After selecting Random for Slideshows in  *Edit->Preferences* right click on an image and select "Start Slideshow", you can then cycle through randomly using the space-bar and backspace for forward/back. The 'F' key toggles fullscreen.

You can install GQView via Synaptic or just *sudo apt-get install gqview* from a terminal window. There may be other programs, but GQView is the one I'm familiar with...

Good Luck!
