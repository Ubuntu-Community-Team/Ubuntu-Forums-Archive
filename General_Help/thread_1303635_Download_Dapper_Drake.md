It's not about how old version you run, it's about how lightweight desktop & programs you run.

Firefox is heavy, no matter if you use current version or a 3-year old one. Same applies to Gnome desktop.

So instead of using outdated OS and software (which is unlikely to benefit you in any way) you should use up-to-date OS and lightweight software. Xubuntu has a bit more lightweight desktop, and LXDE, IceWm & Openbox are great options as well. If the desktop is lightweight enough your machine should easily be capable of running Firefox but you might still want  to use something lighter instead, like Epiphany.

Crunchbang is Ubuntu. It only installs different desktop and set of default applications, but the OS itself is exactly the same. So you can ask in Ubuntu forums if you need any help, just mention that you are using Crunchbang (or Openbox instead of Gnome desktop, which is the actual difference that might be relevant when people are helping you).

(It even uses Ubuntu's repositories, so you have all the same software available as if you installed Ubuntu, and all updates Ubuntu gets apply to Crunchbang as well)

edit: It just came to my mind that you have Intel graphics card. If the Ubuntu version you tried was 9.04, the low performance you experienced might have simply been result from the known performance issues with the Intel graphics driver version 9.04 had. Since your hardware seems easily capable of running normal Ubuntu desktop you could also try Ubuntu 9.10 to see if it works fine for you...
