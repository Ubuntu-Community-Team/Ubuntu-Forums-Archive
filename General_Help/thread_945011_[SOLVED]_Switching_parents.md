### Post by Bucky Ball on 2008-10-13
Sounds like you're starting at square one again so let us know how it goes this time. When you boot after installing Ubuntu, hit escape at the beginning and see if that brings up the grub menu. If it boots into Ubuntu instead of Windoze, in a terminal paste this:

**sudo nano /boot/grub/menu.lst**

in there you may find a line that says:

**hiddenmenu**

Change that to

**# hiddenmenu**

... and the grub will come up. I have installed a couple of times recently and this has been on as a default for some strange reason so machine boots into one OS or the other.

Stick to the manual partitioning. Auto gives / and /swap only, no home, so very limited. :)
