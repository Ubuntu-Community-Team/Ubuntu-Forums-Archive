### Post by bscbrit on 2007-05-12
Reboot the system, but select recovery mode rather than the normal boot when the menu is displayed.  If you do not see the menu, then you need to press the 'Esc' key while the countdown is displayed on the screen.  This will boot Ubuntu in a terminal with you already logged in as the 'root' user.

Type the following command, and accept defaults for each answer:

dpkg-reconfigure xserver-xorg

That should get you back to a working GUI.

### Post by mojoman on 2007-05-12
From command line, run
```

sudo dpkg-reconfigure -phigh xserver-xorg
```

This will autogenerate a new xorg.conf. 

If you made any changes in your old xorg.conf you'll probably wan't to back it up before so you can view those changes later. Try:
```

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```

before generating a new file, this will give you a backup-file with the name xorg.conf.backup.

/mojoman

```
ls -la /etc/X11
```
