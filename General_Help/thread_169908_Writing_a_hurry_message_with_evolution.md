### Post by bbarrons on 2006-05-04
a command line option is
evolution mailto: emailaddress.com
this will open an email just for that address. If you dont know the address then just use a *.com for an address.  It will open with *.com in the send to line...
good luck
 Bill

### Post by bbarrons on 2006-05-04
correction
if you use evolution mailto:

### Post by bbarrons on 2006-05-04
make it even simpler. create a launcher. Right click on the desktop, give it a name like evol mail , insert into command line evolution mailto:
and then choose an icon. drops onto the desktop and it is now only a click away.
