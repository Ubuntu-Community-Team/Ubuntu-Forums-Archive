### Post by Dennis N on 2016-08-06
If your new data is entered in columns towards the right, you can hide the columns with the oldest data. Example: Columns A to L representing previous 12 months are set to hidden, the 12 columns M to X (current year) set to display. Hidden columns can be easily set to unhidden when needed.
