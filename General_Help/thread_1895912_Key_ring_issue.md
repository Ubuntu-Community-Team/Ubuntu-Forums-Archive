### Post by ExileAmongYou on 2011-12-15
Have you changed your settings so that your computer automatically logs in, rather than showing the screen with the list of users when you first boot up? That's a fairly common cause of this problem.

### Post by thaelim on 2011-12-16
Try changing the password for the 'login' keyring in the Passwords and Keys applet. If you change it to be the same as your login password, that should fix it.
