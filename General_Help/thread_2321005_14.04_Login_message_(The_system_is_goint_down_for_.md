### Post by QDR06VV9 on 2016-04-19
> [COLOR=#111111][FONT=Ubuntu]There was running a deferred shutdown process and suddenly power turned off.[/FONT][/COLOR]
My question..was you running a deferred shutdown?
In any case if that is what you intended(deferred shutdown)
You can disable or quit by typing this in a terminal
```
sudo shutdown -h ***But do not exit yet***
```
and then hit the keyboard keys <Ctrl+c> which should cancel the deferred shutdown
But Broken packages is a different matter to address.
