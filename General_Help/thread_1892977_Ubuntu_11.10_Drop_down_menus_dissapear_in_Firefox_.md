### Post by veroslav on 2011-12-15
I have made some new observations regarding this issue (logged on in a Unity 3D session):

- It seems that this problem mostly occurs when a new tab is opened in Firefox just after closing one or two of the already opened tabs

- When I click on the address bar arrow in order to display the drop down address menu while the issue is present, and then just leave it open for a few seconds, it closes by itself after that

- When I try to type an address of a web site I KNOW I visited previously and for which a complete url should be displayed while I type, nothing is displayed while I type. It is as I never visited the url before.

- After I finally loaded a new url in the newly opened tab, and switched a bit between opened tabs/other opened windows, the issue is gone.
