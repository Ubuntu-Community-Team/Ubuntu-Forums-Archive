### Post by yancek on 2015-04-15
Short of reinstalling from the windows media, there is really nothing anyone can reasonably suggest if your question is can you get your installed windows back and working.  Which method did you select during the install of Ubuntu?
Check to see if you have any windows partitions from a terminal in Ubuntu:  sudo fdisk -l(Lower Case Letter L in the command) should tell you.
You might be able to mount and save data from Ubuntu.
You might be able to repair the windows install from the installation media of windows.

You would need to give more info before any suggestions.
