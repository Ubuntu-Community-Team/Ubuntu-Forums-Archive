### Post by phDaemon on 2010-01-06
When i first started using linux (about 3-4 years ago! lol) I started out with mandrake (mandriva now) then moved on to OpenSuse and now ubuntu. To me, the easiest way to learn, was to get rid of windows all together which forced me to only use linux, and whenever a problem came up, i would post whatever error i got into google and figure it out that way. Granted i wanted to learn linux bad so i figured that was the quickest way for me to learn. I suggest that whatever part of linux you want to find (for example a shell is also known as a terminal which in windows would be a command prompt) you type in google and most answers are there. There is no "catch all" linux website as there are different levels of people switching over to linux.

BTW, welcome to the linux world, and we are always here to help!
