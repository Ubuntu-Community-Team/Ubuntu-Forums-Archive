### Post by ArielEnter on 2010-05-07
If you are trying to synchronize a Black berry with evolution by following a tutorial, you may encounter the following error:

Unable to instance plugin with name asd: Unable to find the plugin "evo2-sync"

If you do, all what you have to do is install opensync-plugin-evolution package.

For some reason when I try to look for it using the quick search of the Synaptic Package Manager I didn't find it, so I used the following command in terminal: 
sudo apt-get install opensync-plugin-evolution
