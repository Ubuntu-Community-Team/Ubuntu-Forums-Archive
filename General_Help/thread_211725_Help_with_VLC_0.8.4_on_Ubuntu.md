### Post by hanzomon4 on 2006-07-10
Try this... Open System>Administration>Synaptic Package Manager...
Click the settings tab and select "Repositories" 
When the window pops up click "Add" and paste this in ```
http://nightlies.videolan.org/build/dapper-i386
```
Then just go back to the main Synaptic window click update and then reinstall VLC... It should then be "VLC 8.5"
