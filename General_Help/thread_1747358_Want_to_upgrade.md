### Post by Nasair on 2011-05-02
I'm not sure how different it is with the Ubuntu live disc, but in Xubuntu it gave me an option to upgrade once I actually started the install...i.e. I had 10.10 installed and then booted to my usb > selected to install Xubuntu > Before selecting the drive I wanted to install too it gave me options to upgrade. I'd boot into the 11.04 live version again and see if you get these options, as it won't hurt anything as long as you don't go past the point in which you select a drive to install to.
