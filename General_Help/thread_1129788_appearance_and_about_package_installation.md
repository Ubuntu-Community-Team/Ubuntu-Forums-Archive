### Post by CatKiller on 2009-04-19
If you are using a 64-bit processor then you probably want to use the 64-bit version of Ubuntu.

You should be able to install graphics drivers with System -> Administration -> Hardware Drivers.

You don't need to compile things to install them. They can be installed with System -> Administration -> Synaptic Package Manager. If you **do** want to compile things for other reasons, gcc is included in the **build-essential** package, along with other things that you'll need to compile software.
