### Post by Mark Phelps on 2010-06-07
My advice is to avoid Wine, and running ANY MS Windows apps, until you become a LOT more experienced with Ubuntu -- and with getting all your configuration items set the way you want.

Wine is (at best) a kludge to allow SOME MS Windows apps to run well, others barely, and quite a few -- not at all.

You will have more than enough to deal with getting native Linux apps working the way you want.  You will be drowning yourself in problems if you take on Wine as well.

IF you're seeing the same "weird graphics" running native Linux apps, then we have something we can address.
