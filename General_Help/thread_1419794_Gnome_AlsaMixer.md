### Post by lewmur on 2010-03-02
In order to get Skype to work it is necessary to uninstall the PulseAudio Manager and install the Gnome AlsaMixer.  The only drawback to this is that there is no longer a Volume Control in the panel.  Does anyone know a way to get the master volume control for the AlsaMixer on the panel?

---

### Post by underquark on 2010-03-02
Right-click on the panel.
Add to Panel.
Scroll down to Volume Control.
Add it to the panel.

Now, if you open AlsaMixer in a Terminal note that as you change the volume via the panel control you see the Terminal AlsaMixer Master volume change with it.

### Post by Enigmapond on 2010-03-02
I had the same problem and the above instructions seemed like the obvious thing but it didn't work. Once Pulse was deleted...the volume refused to show. The way I work around it was to also install alltry. This will put any programme on the panel. Once installed, open gnome alsa-mixer and have alltray put it up there. Either that or use docky or Avant Window Navigator (AWN) and add the volume applet to one of those. For some reason this works as well
