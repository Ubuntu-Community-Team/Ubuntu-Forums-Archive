### Post by FLeiXiuS on 2005-07-22
[QUOTE=Adrenal]cd then the directory

A users home folder can also be abreviated by ~.

Example, my username is fleixius.  (/home/fleixius)

Rather then typing `cd /home/fleixius` I can type `cd ~`.
