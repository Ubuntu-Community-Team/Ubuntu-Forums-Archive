### Post by bruseleno on 2011-04-30
I had the same problem today. Open a terminal clicking on the desktop with the right button of your mouse. 
type ccsm and hit enter
Click on the Unity icon so that it is selected. 

It will tell you that there are some conflicts and ask you to solve them or ignore them. Choose to solve them. 

Then just accept the proposals on the left side (which will use the Unity plugin action instead of the Gnome compatibility action). 

As soon as I did this I got both pannels, but the top one was just black. So I restarted in the terminal typing "Sudo reboot", then my password at the prompt. 

And that's it. Everything is now working fine. 

To log in and have access to the old menu and taskbar, choose ubuntu classic in the login window, no need to install or uninstall anything.
