### Post by Mark Phelps on 2011-04-14
Actually, you DO have KDE.  

Ubuntu (at least, through 10.10) installs some components for both GNOME and KDE, but defaults to the GNOME desktop.  

If you were to choose to use the KDE desktop instead, you would see a HUGE list of additional packages that Ubuntu would ask to install.
