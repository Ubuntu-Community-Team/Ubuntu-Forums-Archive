### Post by ian-weisser on 2013-12-13
That looks like a Windows application.
Ubuntu is not Windows. Windows applications will not run in Ubuntu. Nor will Ubuntu applications run in Windows.
Like trying to run a train on the wrong gauge track. Simply the wrong environment for it to work.
...unless you do have a Linux version, and I missed it on the website?

You can try using Wine, available in the Software Center, to see if it can open the application.
You can run a licensed copy of Windows (in a Virtual Machine [VM] hosted by Ubuntu) that can open the application.
You can contact the developers of the application to see if they will port it to Linux.
