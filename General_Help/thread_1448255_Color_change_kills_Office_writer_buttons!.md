### Post by Tamlynmac on 2010-04-06
Not quite sure why it happened. Perhaps, it's a default for customization.

Check to see if "Text Only" is selected or if Icon Only is selected.

Tools/Customize/Toolbars Tab
From the selection box at the top for "Toolbar", select "Standard". Then on the right side left click on the "Toolbar" button (located under the "NEW" button"). From the dropdown list select Icons Only.

This can be done for any and all toolbars displayed. By changing the Toolbar selection, each toolbar can be set for Text Only, Icons Only or Icons & Text. All Toolbars being used are checked in the Open Office main screen/view/Toolbars (list).


Good Luck & hope this helps...
