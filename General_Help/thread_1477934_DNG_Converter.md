### Post by rabil on 2010-08-01
If you are using Ubuntu 10.04, open synaptic package manager and search on dng. You should see at least 2 pacakages (exifprobe and kipi-plugins). I installed both. Now when I open a .dng file with F-spot, the DNG converter opens. Try it and let us know.
