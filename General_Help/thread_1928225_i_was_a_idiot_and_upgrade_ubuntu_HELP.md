### Post by MARP1961 on 2012-02-19
Click the Ubuntu logo at the top of the side launcher. Once the Dash is open just type TERMINAL and the search should find it. Once you click to open Terminal it will appear on the side panel. Right clicking the icon will give you the option of keeping the application on the launcher.

If you really cannot get used to the Unity desktop (permanent left side launcher) then you can install Gnome Shell desktop (or XFCE4 desktop) via Software Centre or Terminal: 

sudo apt-get install gnome-shell
