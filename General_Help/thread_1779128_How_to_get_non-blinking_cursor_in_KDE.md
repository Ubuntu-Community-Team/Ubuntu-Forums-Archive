### Post by iX9 on 2011-06-10
**SOLVED!**

By installing qt4-qtconfig and run it, -> Interface -> CursorFlashTime -> set "No blinking".

Everything of NON-blinking cursor is here:  [http://www.jurta.org/en/prog/noblink](http://www.jurta.org/en/prog/noblink)
