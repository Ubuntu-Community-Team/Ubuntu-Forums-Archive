### Post by Portaro on 2014-03-05
You need to edit the cursor you have tweak tool that do this, but you need a cursor type file compatible to the theme, by default the cursors on Buntus have 2 paths the base (root) path is on /usr/share/icons, and in user path /home/username/.icons, in both paths you can edit and put cursor themes, and edit default cursor of users.

MAybe you can give to us where is the cursor that you want install to people see if the file type of cursor can be recognized to an active cursor theme.
