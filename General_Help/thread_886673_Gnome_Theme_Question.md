### Post by zwaardmeester on 2008-09-08
The following worked for me. 

1) -> System -> Preferences -> Appearance
2) select 'Clearlooks'
3) -> Customize
4) tab Window Border: select 'Human' or 'Inverted' or whatever you prefer
5) tab Icons: select 'Human'
6) tab Colors: select 'Selected Items' (background) to set window titlebar colour

Now you have the Clearlooks theme, but with `Human' icons and borders.
