### Post by IceReaver75 on 2009-01-06
the text colors for a theme are controlled by the theme settings themselves.  You will have to open the theme folder from where it is installed either /home/.themes(hidden folder) or /usr/share/themes and open the gtkrc file inside the gtk folder with a text editor and change the text color that way.  As for the menu color itself you have to go into the folder inside the gtk folder of the theme you are using look for the menu folder open it up and look for the menu.png item.  open that with gimp or other graphic editing program and adjust the color and save it.  Hopes that helps you some.
