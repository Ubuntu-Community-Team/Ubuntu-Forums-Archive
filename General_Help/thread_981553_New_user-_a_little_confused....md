### Post by taurus on 2008-11-13
If you want to remove windows and install Ubuntu only, then just boot from the LiveCD (desktop CD) and tell the installer to use the whole harddrive when you get to the partition screen.  That will wipe windows off the drive and will install Ubuntu on that same drive for you.  No need to format the harddrive first.

### Post by NoWayBill on 2008-11-13
When installing simply select "use entire disk".

Although I recommend dual booting for possibly the first year until you learn the ins and outs of Linux.
I'd hate to see you have network or other problems and have no way to come here and get the help you may need.

Hopefully you are downloading the Live disk so that you can boot and run from CD first to be certain of hardware compatibility.

Linux has come a long way, but there are still some hardware compatibility issues you may face.
