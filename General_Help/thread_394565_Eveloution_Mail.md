The English magazine LXF has exactly this query in the April 2007 issue no. 91. They suggest :-
To invoke First_Run Assistant you should delete the folder** .evolution** in your home directory. (Note the **dot**, it's a hidden directory, you can view them by clicking **Go>Show Hidden Files **from within the file browser). Now type** evolution** from the command line. This will make **evolution** think that it is running for the first time.
