### Post by george9233 on 2008-04-21
Hello everone.  Have you ever thought adding MORE MS fonts to Ubuntu? Here's what I did.

1. Install "msttcorefonts" in Synaptic

2. Go to "C:\WINDOWS\Fonts" folder in the Windows partition

3. Copy all TrueType fonts (all fonts end with .ttf or .TTF) in this folder

4. Go to "/usr/share/fonts/truetype/msttcorefonts" folder with root permission by typing```
gksudo nautilus /usr/share/fonts/truetype/msttcorefonts
```

5. Paste all the fonts here

6. Done! Now it's totally available. It can be used in the entire operating system

I'm sharing this experience because I found it fun and useful. Hope to be helpful.
