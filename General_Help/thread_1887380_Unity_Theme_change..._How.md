### Post by oldtimer7777 on 2011-11-26
Ubuntu 11.10 no longer uses the older themes (gtk2 in favor of gtk3) and so the selection is very very limited at this time. If you are interested in themes you should probably stick with the older version of Ubuntu that still supports the older gtk2 themes that you probably would like to be using.
