### Post by seekers on 2007-06-24
Yes there are free virtual software packages out there.    Vmware Server which is free.  Some are using virtualbox which is free also.   I am using Vmware Server which is a good product.  Their are other post on virtualbox.   In Add/Remove you will found Vmware player and Vmware Server.   If you have not wipe out your windows yet you can make a Virutal copy and just use Virtual Player once you have Ubuntu or whatever distro you are running.  You will need to register to get a serial number for Vmware Server.   [http://www.vmware.com/products/free_virtualization.html](http://www.vmware.com/products/free_virtualization.html).    You don't need to download Vmware Server because it is in Add/Remove but you will have to register got get your free Serial Number.:p
