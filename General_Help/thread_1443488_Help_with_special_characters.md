### Post by diesch on 2010-03-31
Either use a keyboard layout that has AltGR sequences for the keys you need (like US international) or use a [compose key](https://help.ubuntu.com/community/ComposeKey)

If you really want to use funny numbers you can [do that]("https://help.ubuntu.com/community/ComposeKey#Unicode%20composition"), too.
