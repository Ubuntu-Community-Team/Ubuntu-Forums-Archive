### Post by Forlong on 2008-06-22
1. Go to *System &#8594; Administration &#8594; Hardware Drivers* and **disable** the ATI driver

2. Run this in a terminal:
```
sudo dpkg-reconfigure -phigh xserver-xorg
```
3. Reboot.

4. **Enable** the ATI driver in *Hardware Drivers* again.

5. Reboot again.

6. See if it worked.


Note that this will only work if you did **not** install the fglrx driver manually (or by using Envy).
