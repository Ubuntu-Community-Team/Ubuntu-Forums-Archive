### Post by NightwishFan on 2010-01-30
On Gnome, you can use the program File-Roller (aka Archive Manager). It is installed by default on Ubuntu. If it is not configured to work by default, just go into the file's properties, and set file roller under "Open With".

On Windows perhaps you could use 7-zip to open them.
