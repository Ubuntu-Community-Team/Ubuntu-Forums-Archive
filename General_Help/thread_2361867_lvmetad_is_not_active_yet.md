### Post by Dennis N on 2017-05-21
Here are two options: 

Option 1:
Working from live media, install Boot-Repair (URL below) and choose "create a BootInfo summary". Post URL of results here. Do not do "Recommended Repair". This summary may suggest a solution.

For complete instructions, see: [https://help.ubuntu.com/community/Boot-Info](https://help.ubuntu.com/community/Boot-Info) 

Option 2:
Reinstall Ubuntu using the "Erase disk and install Ubuntu" option on the "Installation Type" Screen. Using the live media, first copy any important files to a USB for safe keeping.
