### Post by CatKiller on 2020-04-16
Depends what you're on. If you're on a supported release - 16.04, 18.04, or 19.10 you should do all your normal updates. If you're on an unsupported release you should get yourself onto a supported release.

You can help with testing the pre-release 20.04 if you feel like helping testing. It won't be automatically offered as an upgrade option to users on LTS releases till the first point release has been done, but you can manually upgrade to it at any time after release, which is still a week away.

### Post by Impavidus on 2020-04-16
You can only upgrade to the next release when you have installed all upgrades for your current release. The upgrade from 19.10 to 20.04 can be expected a few days after release, sometimes a bit more (like two weeks). The upgrade from 18.04 to 20.04 can be expected in July. Other releases offer no upgrade to 20.04. And in any case, make sure you've got backups and it's best to have a live disk ready too, so you can do a clean install in case the upgrade fails.
