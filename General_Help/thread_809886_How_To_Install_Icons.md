### Post by spupy on 2008-05-27
The icons that are installed for your account can be found in a folder named ".icons" in your home directory. You should place them there in their own sub-folder.

Easier: from the appearance dialoge where you select themes and icons. There should be a button for install. If there isn't, try dragging the archive with the icons and dropping it on the appearance dialog.
