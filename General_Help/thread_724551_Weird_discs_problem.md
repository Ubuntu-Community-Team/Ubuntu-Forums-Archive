### Post by taurus on 2008-03-14
Open a terminal and post the outputs of these commands here.

Applications -> Accessories -> Terminal
```
sudo fdisk -**[COLOR="Blue"]l[/COLOR]**

```
That is a lower case letter L in the first command, not number 1.

### Post by taurus on 2008-03-14
> **dingorunner said:**
> 
sergio@anaile:~$ cat /etc/fstab
# /etc/fstab: static file system information.
#
#  -- This file has been automaticly generated by ntfs-config -- 
#
# <file system> <mount point>   <type>  <options>       <dump>  <pass>

proc /proc proc defaults 0 0
# Entry for /dev/sda3 :
UUID=32f32f3c-e4ef-4d4c-8506-d9e32fadcb1d / ext3 defaults,errors=remount-ro 0 1
# Entry for /dev/sda1 :
UUID=9A641D0E641CEEAF /media/sda1 ntfs-3g defaults,locale=es_ES.UTF-8 0 1
# Entry for /dev/sda4 :
UUID=EA486B1C486AE733 /media/sda4 ntfs-3g defaults,locale=es_ES.UTF-8 0 1
# Entry for /dev/sda5 :
UUID=ab6f904d-71ce-457b-b7f6-c59eff7dff66 none swap sw 0 0
/dev/scd0 /media/cdrom0 udf,iso9660 user,noauto,exec 0 0
/dev/fd0 /media/floppy0 auto rw,user,noauto,exec 0 0
[B][COLOR="Blue"]/dev/sda4 /media/sda1 ntfs-3g defaults,force 0 0
/dev/sda4 /media/sda1 ntfs-3g defaults,force 0 0[/COLOR][/B]

See something interesting there!  You already mounted /dev/sda1 to /media/sda1 and /dev/sda4 to /media/sda1 a few lines above but then you mount /dev/sda4 to /media/sda1 twice at the end.  

Edit /etc/fstab

```
gksudo gedit /etc/fstab
```
and put a # in front of the last two lines to comment them out.  Save it and reboot.
