### Post by DrPotoroo on 2011-06-23
Print to file then use ps2pdf script on the resulting postscript file. This will give you a snapshot of the filled in form - but you won't be able to edit it.
 
If you want to be able to come back and edit the form later, maybe use xournal to annotate the form. This will store a file of annotations seperate to the pdf. When you are finished then you can export it to pdf to have the annotations and form together.
