### Post by abcuser on 2020-03-18
Hi,
because of coronavirus I am working at home using VPN on Ubuntu 18.04 laptop. I use Remmina remote desktop tool to connect to my Windows work PC.
I am very very very used to my Windows keyboard shortcuts on remote machine, but my Ubuntu laptop is constantly stealing keyboard shortcuts.

Is there any way when I am in Remmina full-screen mode to disable all of Ubuntu laptop shortcut and send them directly to the remote machine Windows?  
Regards

---

### Post by abcuser on 2020-03-18
Problem solved. In Remmina remote desktop tool, there is **Grab all keyboard events** icon.
Pressing on it and all keyboard shortcuts are grabbed by Remmina and so send to remote Windows desktop.
