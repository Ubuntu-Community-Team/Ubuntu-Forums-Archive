### Post by efflandt on 2010-11-20
Bootable live/install iso on USB 2.0 will boot quicker than the CD.  But USB flash or SD can widely vary in speed capabilities (somewhat depending upon price), so you may not get the same speed as someone else.

An advantage of USB (besides speed and not using up CD's) is that if you set persistent data when you installed it to USB, it can save certain settings like timezone and wireless settings.  If you later boot to the live system on USB and then install, it will carry timezone and wireless settings over to the installed system.
