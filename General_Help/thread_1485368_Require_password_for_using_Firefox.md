### Post by lovinglinux on 2010-05-16
Don't use sudo for opening the browser. You will mess your Firefox profile.

You can use [Public Fox](https://addons.mozilla.org/en-US/firefox/addon/3911), to block her from doing dangerous things within Firefox, but I think would be better to just create a new Ubuntu user for her and give it limited rights.
