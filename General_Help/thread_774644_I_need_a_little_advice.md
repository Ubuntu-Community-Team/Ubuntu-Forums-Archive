### Post by forrestcupp on 2008-05-02
for screensavers try installing these from a terminal and see if it gives you more
```
sudo apt-get install xscreensaver-data xscreensaver-data-extra xscreensaver-gl xscreensaver-gl-extra rss-glx
```
If you already have some of these installed, it will still install the ones that you don't already have.
