### Post by comprocket on 2008-05-26
Ok.  i have solved my problem after many attempts!  Everything works correctly now!

What I did was do a fresh format.  After this, I did all of the system updates that were available and then installed the restricted drivers.

Before resetting, I went into xorg.conf and added this line to "Screen":

```

SubSection	"Display"
	Modes	"1280x1024"
EndSubsection

```

The reason for this is because the default monitor size seemed to determine that my old CRT was capable of VERY large resolutions over 1900px.  So by doing this, it would limit the max resolution and make the screen work right on the first attempt.
