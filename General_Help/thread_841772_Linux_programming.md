You can install the C/C++ (as well as a few other languages) by:
sudo apt-get install build-essential

Then you can do the C/C++ stuff easy peasy.

As far as tutorials, just google....there's probably about 10 million out there. Or get a decent book which can walk you through step by step.

If you aren't set on a particular language, then as housam mentioned, Python should be an easy and fun first language.
