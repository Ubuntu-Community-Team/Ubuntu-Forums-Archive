### Post by john newbuntu on 2010-01-09
This is a great time for you to familiarize yourself with Ubuntu Linux or for that matter any other distro.  Here is a site devoted to help you get it installed.  It is for a slighthly older version of Ubuntu.  But the process for the newer version (Karmic or 9.10) is very similar.
You will need a liveCD that you can download from the internet or borrow from a friend of yours.  It needs to be burnt as an ISO and not a normal data CD.  Once you  do that, the remaining instructions are in this doc:  Enjoy Ubuntu.
[http://www.ghacks.net/2008/12/12/an-illustrated-guide-to-installing-ubuntu-desktop/](http://www.ghacks.net/2008/12/12/an-illustrated-guide-to-installing-ubuntu-desktop/)
