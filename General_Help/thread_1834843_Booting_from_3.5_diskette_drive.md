### Post by MartyBuntu on 2011-08-28
- Make sure CD-ROM is set as first boot device in the BIOS.

- Make sure you've properly burned the Ubuntu ISO as an image file.

- Make sure the CD-ROM in your older computer is capable of reading newer brands of storage media.
