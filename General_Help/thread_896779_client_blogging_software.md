### Post by pytheas22 on 2008-08-21
As long as your server supports PHP, you can use [simplephpblog]("http://www.simplephpblog.com/").  It's server-side, but it only requires PHP, so you don't need shell access to setup databases or anything.  As long as you can ftp in to upload the simplephpblog directory, it should work.

### Post by ugm6hr on 2008-08-22
Probably not an exact fit - but worth mentioning:
[http://www.tiddlywiki.com/](http://www.tiddlywiki.com/)

It's more a digital web notepad that stores everything as a single file, but has the capacity to date-stamp stuff, so it should be accessible like a blog too.

You can experiment with it in a few minutes to see if it fits the bill.
