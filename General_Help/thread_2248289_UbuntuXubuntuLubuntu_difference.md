### Post by TheFu on 2014-10-13
The DE is just a GUI to control the OS interaction.

Everything (ok, almost everything) at the CLI is the same.  Most self-contained applications will be the same too. It is only those extras (the little popups that drive me crazy) like the DBUS messages .... 

If you think there is a huge difference between Win7 and WinXP -then yes, there is a huge difference. Notice that notepad between them is the same?  Same thing on linux.

If you don't like a heavy DE, then don't run any DE - use a pure Window Manager - WM.

An easy way to see the differences with little effort is to look on youtube for reviews.

### Post by Bucky Ball on 2014-10-13
I personally go for a minimal install and use xfce4 and only the apps I want. Less moving parts means less chance of breakage.

Be aware, that if you are talking about running Ubuntu (or any *buntu) and then installing, say, xubuntu-desktop you are going to end up with ALL the default apps of Ubuntu AND the default apps of Xubuntu. This can cause conflicts and you will have crowded menus and possible duplication of apps (different apps that do the same thing). Better to just install the desktop environment, e.g. xfce4 or lxde in Ubuntu, log out, choose the DE from the Sessions menu at the login screen, log in.

In the end it's really down to personal preferences re. which DE you choose to use. Neither is 'better' than the other. See the links below re. minimal installs. Good luck. ;)

Minimal Install page:
[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

Older, but info still relevant while screenshots might be a bit different now:
[http://www.psychocats.net/ubuntu/minimal](http://www.psychocats.net/ubuntu/minimal)
