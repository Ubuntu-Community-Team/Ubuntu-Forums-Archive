### Post by mike2357 on 2008-05-18
When I've run into this problem (with either Thunderbird, Outlook or Outlook Express), the problem has turned out to be a corrupted message on my ISP's e-mail server.  Fortunately, it has a web-based interface as well.  I've been able to resolve the problem by logging in to my ISP's web-based interface and deleting the offending message, which may be the first or last one you see.

If you don't have a way to access your e-mail through the web, I don't have any ideas; maybe someone else will.

Good luck!
