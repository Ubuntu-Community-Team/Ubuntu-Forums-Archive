WINE is limited in its ability to work with Windows programs.  Some work very well, some not at all.

My suggestion would be to install Windows in a sandbox (virtual machine) such as VirtualBox -- or another recommended by someone who may post after this -- and run your software natively in an "almost really" Windows environment.
