### Post by sayakb on 2008-09-06
Open your home folder. Press Ctrl+H and open **.icons** folder. Now copy the icon folder there and now try to apply the icons. 
If the folder already exists, then press Alt+F2, type in **gksudo nautilus**, open /usr/share/icons and copy the icon folder there.
**NOTE: **It may be risky to use nautilus as root. Please be careful while handling it.

### Post by robz0rz on 2008-09-09
I just tried to install those Crystal icons. I dragged the archive into my Appearance dialogue, and they installed fine. I see what you mean though, once applied, the icon-theme seems incomplete. Most icons are just the default gnome icons.

The icon-theme in question is from 02/2003. I remember reading something about certain naming specifications for icon-themes being changed since Gnome 2.20 (or something along these lines). My best guess is just that this theme needs an update to get the names of the icons right. What's happening now is that when Gnome looks for let's say the folder icon, it can't find the correct file in the Crystal theme, so it reverts to the default icon.

I'm sorry, but the only thing you can do now is either wait for someone to update that theme or try it yourself. Take an existing icon theme that is very complete (I suggest copying the folder /usr/share/icons/default to your home and renaming it ~/.icons/my_icon_theme) and then copying and renaming the correct files from the Crystal theme into that. It's work, but it's worth it. I made my own icon-theme this way.


Hope that I was of any help
