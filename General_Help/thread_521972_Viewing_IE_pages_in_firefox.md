### Post by hedgefighter on 2007-08-10
I'm fairly certain that IEtab only works in windows. If you need IE compatibility, there's a couple options. If the site is just blocking you for not using IE, try user agent switcher [https://addons.mozilla.org/en-US/firefox/addon/59]("https://addons.mozilla.org/en-US/firefox/addon/59"). If you need IE to render you can try IEs4Linux: [http://www.tatanka.com.br/ies4linux/page/Main_Page]("http://www.tatanka.com.br/ies4linux/page/Main_Page"). It lets you run IE 5, 5.5, and 6 with WINE.
