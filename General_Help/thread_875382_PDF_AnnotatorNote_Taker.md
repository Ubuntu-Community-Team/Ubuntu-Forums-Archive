### Post by bmac on 2008-07-31
I believe PDFedit (available in the Synaptic package manager) performs this function simply by selecting "page" in the tool bar and then "selecting add text".... 

I'm not certain if you're wanting to move the existing text automatically when typing your message or if you just wish to insert a text message? I know PDFedit will allow for modification of the existing text, by don't  know if it will automatically insert a message. You can peruse their documentation here:
[http://pdfedit.petricek.net/user_doc.html](http://pdfedit.petricek.net/user_doc.html)
