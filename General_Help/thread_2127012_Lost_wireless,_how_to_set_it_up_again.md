### Post by Kopkins on 2013-03-19
> **secuono said:**
> When I press these three buttons together, command prompt shows and I can't get out of it. control alt f2 [[theres a little beacon and lines going away from it, I think thats the wireless button]]

Try pressing CTL+ ALT + F7. using CTRL + ALT and a function key (F1 F2 ... F6) brings you to a virtual terminal or console. The gui is usually on the seventh one, hence F7. 

Try using just one button at a time in combination with the wireless button. Usually there is an fn key. fn = function. Try it with that if you can find it.

Also, what are your computers hardware specifications. You should run a newer version of ubuntu. Ubuntu 8.04 is no longer supported so you will not receive any security updates. If your computer has low spec's you can usually use a different type of ubuntu. like lubuntu or xubuntu. 

Kopkins
