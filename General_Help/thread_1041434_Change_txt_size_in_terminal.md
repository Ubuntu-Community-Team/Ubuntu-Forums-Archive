### Post by 73ckn797 on 2009-01-16
> **miocene said:**
> Is there any way to reduce the size of the text in terminal as due to my old laptop not having great resolution it would help with screen real estate?

I do not use Xubuntu but in the regular Ubuntu it is done as follows:

Open terminal. 
Go to Edit/Profile Preferences.
Under the General tab: Uncheck "Use system fixed width font". Select font and font size as desired.
See what that does for you. I can get it down to a 6 size font.

In terminal you can using: ***ctrl -*** to decrease size; ***ctrl/shift +*** to increase size and ***ctrl 0*** to normalize to default. Under the View section it lists commands to increase or decrease font size but only "**ctrl/shift +**" works to increase size.
