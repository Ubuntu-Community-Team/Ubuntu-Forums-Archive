### Post by CatKiller on 2019-06-19
nvidia-xconfig is pretty terrible at generating a decent xorg.conf. You should remove xorg.conf if you've still got an automatically generated one.

Manually creating a file at /etc/X11/xorg.conf.d/20-nvidia.conf and putting 

```

Section "Device"
    Identifier "Default Device"
    Driver      "nvidia"
    Option     "CoolBits"               "21"
EndSection

Section "Screen"
        Identifier      "Default Screen"
        Device          "Default Device"

```
should be enough for what you want.

430 doesn't support your graphics card. The 418 branch is the last one that supported the GT 735M.
