A1 - Yes. Ubuntu is a stand-alone OS. Everything you need to run your computer is on the CD.

A2 - If you have certain software that you have to have, as you say, you have several options:

[LIST]
[*]See if the provider has a version ported for Linux;
[*]Use Google (or [LinuxAppFinder]("http://linuxappfinder.com/")) to find a Linux equivalent;
[*]Dual-boot your computer and keep a version of Windows on its' own little partition. XP Pro can live quite happily in a 5 or 8GB partition and not use up too much space.
[*]Use VMWare or Virtualbox to run Windows inside a virtual machine, and run your software that way.
[/LIST]
Personally, I use the last two options. While there are some, not many vendors port their applications for Linux. My computer is a Vista/Hardy dual boot, and I have an XP virtual machine I can run when I run across the oddball Internet-Explorer-only web page or a small bit of software I need to run. For bigger things and for games, I just boot into Vista.

Hope I helped.
