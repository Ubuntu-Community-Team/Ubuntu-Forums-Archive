### Post by se6astien on 2023-02-10
Yes, by default does only works if an app is in focus, not Gnome. To fix it, navigate to Settings/Keyboard/View and customise shortcuts/Custom shortcuts/Add shortcut (Name: guake, Command: guake, shortcut F12, Add)
