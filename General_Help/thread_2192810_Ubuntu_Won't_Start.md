### Post by varunendra on 2013-12-12
I think posting a screenshot would be much better than posting a video link, as not everyone has a fast internet connection (I'm on a slow gprs for example).

Pressing win key allows you to boot normally - is an interesting thing. Have you tried a different keyboard?
Can you boot normally with a Live CD/USB ?

You may try editing the boot line to try a few boot parameters, or remove some odd one (or to remove "quiet splash" to be able to see detailed background process in text form).
Common boot parameters : [https://help.ubuntu.com/community/BootOptions#Common_Kernel_Options](https://help.ubuntu.com/community/BootOptions#Common_Kernel_Options)
