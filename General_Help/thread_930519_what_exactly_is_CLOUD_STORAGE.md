> I require a database ...for large storage and someone told me about cloud storage..technology..Could anyone please explain about cloud storage and its usability?????????

as per wiki
> Cloud storage is the delivery of data storage as a service (including database-like services), often billed on a utility computing basis (eg per gigabyte per month)[40]. For example:

    * Database (Amazon SimpleDB, Google App Engine's BigTable datastore)
    * Network attached storage (MobileMe iDisk component, Nirvanix CloudNAS)
    * Synchronisation (Live Mesh Live Desktop component, MobileMe push functions)
    * Web service (Amazon Simple Storage Service, Nirvanix SDN)
