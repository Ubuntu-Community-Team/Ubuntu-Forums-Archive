### Post by TheAnachron on 2010-10-14
Well you should know that languages are based on applications, not on systems.
Just because you had this setting does not mean its supported at newer version too.

Lets face it: Several applications have several supported languages.
A system has also several supported languages.

What if you change the system to a language some apps do not support? 

To solve this issue, please always download and install the correct language for your OS. 
Changing applications language is a bad thing (right now, because it screws up many things, f.e. the symbol support (ISO >.<))).

I don't know how to help you, I can just say, next time better download the correct version.
