### Post by Het Irv on 2008-01-18
When you select the first option on the Ubuntu Live CD, let it boot up.  Select the third menu at the top (System).  Go down to the second menu (Administration) and then look for "Language Support" (it has a flag beside it).  If it asks you to install anything hit Cancel.  You should be able to find Turkish in this menu.
