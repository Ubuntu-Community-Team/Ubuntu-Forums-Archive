### Post by NikoC on 2008-06-05
As an addition to the previous post I could also recommend the following options if you want to resize your / partition:
- use an ubuntu live cd and start gparted from that cd
or
- download the gparted livecd [here]("http://gparted.sourceforge.net/livecd.php"), burn it to a disk and boot from it

or

- us other partitioning software such as partition magic
