### Post by jemaliahz on 2006-05-03
Hello!

Easiest way to do dual head in Breezy is to use Matrox's mgapdesk -application. Application can be installed thru synaptic. After you start program it will where config file is. Mine is at /etc/X11/xorg.conf.

When program starts you can choose you setup with graphical window. Save and reboot - vola.

Another way to do it is create config file by hand.
