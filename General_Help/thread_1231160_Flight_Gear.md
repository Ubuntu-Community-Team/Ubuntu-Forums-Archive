### Post by visionary on 2009-09-23
To have multiplayer enabled, you need to start it with MP options either from command prompt or from FGRUN...

I guess it will be easier if you download the lastest DEB files from [www.unitedfreeworld.com](www.unitedfreeworld.com) and install them. Now you do not have the worries about missing dependencies and etc! These Flight gear DEB files found at [www.unitedfreeworld.com](www.unitedfreeworld.com) seem to work very well with ubuntu 9.04. It gives you the latest flightgear, FGRUN for graphical launch and FGCOM for voice chat.

Once installed, you start FGRUN, select your aircraft and go to advance->network settings and input the commands for multiplay.
You can refer to [http://wiki.flightgear.org/index.php/Howto:_Multiplayer](http://wiki.flightgear.org/index.php/Howto:_Multiplayer)
