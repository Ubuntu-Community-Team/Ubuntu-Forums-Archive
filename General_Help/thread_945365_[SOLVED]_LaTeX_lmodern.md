### Post by Stefan_K on 2008-10-12
Hi achelis,

install the package lmodern by Synaptic, alternatively type at the command line:
```
sudo apt-get install lmodern
```
You can see the texlive packages installed if you start Synaptic, you will find the texlive packages in the TeX section or use the search button.

Stefan
