### Post by bryonak on 2008-10-27
That's because the game claims "full-screen on top of everything" for itself... of course, it depends on the game.

Print will save the whole screen, Alt+Print just the selected window. Have you tried the print key only?
Another possibility would be that it opens the dialog **behind** the game, so you can save it only after exiting or alt+tabbing out of it.

There is probably some way to bypass the save-as dialog and save it directly to a folder, but I don't know it off the top of my head.
