### Post by Ahmed Khattab Shams on 2010-07-26
0wner,
>if i download that 692MB package will the package inbuild with audio and video plugins as well?

No, It wouldn't contain them, but you can install them easily this way:
Go to your desired Multi-media(video or audio) file, click to play with any Player, then If your ubuntu needs some codec to play, it will prombet you, and automatically download and install them, then start playing.
they don't get installed by default with Ubuntu, because it's restricted in some countries, for some reason.

>If i reboot my system and installed fresh Ubuntu will that package 692MB
>will help me to get audio and video plugins?

No, the 692 MB iso file is just an ISO image with the latest Ubuntu Updates, which can be burned to CD and installed on any machine so after installation you won't need to update again, as if you just downloaded and installed the iso that had been realeased on 29april, ok?, and it doesn't contain the RESTRICTED Software.

>will this audio and video plugin packages work with any flavor as well or will work with Ubuntu only?

It should work with any flavor of Ubuntu you may have, else please file a Bug Report.

Hey, here is another Idea:
If you want to update your existing system, just get some internet connection and from your machine with the existing ubuntu installation, On the upper menu (GNOME) Click: Sysytem > Administration > Update manager
when it appears, click the 'Check' Button (you'll be prombeted to enter your password), when it finishes, click 'Install Updates' Now it will be Updating your system, Note; It may take some time.
