### Post by CandidMan on 2011-01-02
It sounds like you did something similar to me the other day. Here's what I recommend assuming your using Nautilus as a file manager
Alt+F2
type in 'nautilus'

You should be in your Home folder. Right click on any folder and select 'open with other application'.
Select 'Use a custom command' and type in nautilus or the path to nautilus

Don't forget to tick the Remember this application box

Hope this helps
