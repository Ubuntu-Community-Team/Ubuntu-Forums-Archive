### Post by balaknair on 2009-07-19
Just boot from the Windows 7 DVD, format the entire drive while partitioning and install Win7. This will remove both Vista and Ubuntu(including GRUB), and you'll have just Win7 on the drive.

If you want to install Ubuntu 9.10 later, I'd suggest you plan partitions accordingly now(leave some space for Ubuntu when you install Win7). It'll save you the task of having to resize your Win7 partition later on. I haven't used Win7, but if it's anywhere near as fussy about it's partitions as Vista(resized Vista partitions sometimes don't boot), it's safer not to resize it later. Additionally, if you're going to install Symantec/Norton suites on Windows 7 resizing later can become a pain in the neck.
