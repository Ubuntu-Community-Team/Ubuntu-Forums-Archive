### Post by satimis on 2021-01-20
Hi all,

Problem solved as follow:-

Right-click "Top Menu Bar"
Select "Dash to Panel Settings"

-> select Position tap
(see screenshot_position25.png)
change the position of menu-bar to Bottom/Top/Left/Right

-> select Style tap
(see screenshot_style25.png)
change "Panel Size"/"App Icon Margin" etc.

Done
