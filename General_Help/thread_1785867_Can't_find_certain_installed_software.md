### Post by prodigy_ on 2011-06-19
> **Ginzell said:**
> OK, I'm still learning so, if there's no GUI for me to get to it, are you saying that I need to run a command in Terminal?  If so, what's the command?
You can use p7zip directly from command line but Archive Manager can use it as well (to decompress 7z files). So it's up to you.

> I've got this Compiz installed - does that have no GUI either?
Compiz is a window manager. If it's running it defines the look and feel of your desktop. You can install compizconfig-settings-manager (a GUI tool) to configure Compiz options.
