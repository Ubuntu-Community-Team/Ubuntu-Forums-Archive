### Post by floke on 2006-12-16
My only experience with a straight Kubuntu build was with the Feisty Fawn release, which did hangup on close down (with a spectacular display of colour that looked like it was melting my monitor!). However, I run Ubuntu 6.10 and have the KDE core modules installed and this runs fine with no problems (other than those that seem to come with KDE anyway, such as the odd random application disappearance!) Have recently moved back to Gnome though, since KDE has too many bells 'n' whistles - though its nice to be able to switch between them when you get bored or just fancy a change (Vive la Linux!) :D 

You've nothing to lose either way really, but if the bug potential is there then I would say go with the straight Ubuntu and simply install KDE core mods on top.
