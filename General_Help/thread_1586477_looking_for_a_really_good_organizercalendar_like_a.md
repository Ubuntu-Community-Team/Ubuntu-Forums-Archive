### Post by envygeeks on 2010-10-02
Sunbird is (still being developed but) now trying to be closely integrated into Thunderbird, so it's better you download thunderbird and install lightning since we might soon see the fate of sunbird come to light. Much like evolution and outlook Thunderbird is trying to integrate calendars.  

While I prefer evolution and syncing with Google which syncs with my phone and so forth, you should be able to install "Sundbird" and "Thunderbird" as one using apt-get.

```

sudo apt-get install thunderbird xul-ext-lightning

```

You can see from [http://www.mozilla.org/projects/calendar/sunbird/](http://www.mozilla.org/projects/calendar/sunbird/) that Mozilla now says to use Lightning and Thunderbird instead of Sunbird.  
You can find out more information on Lightning at: [https://addons.mozilla.org/en-US/thunderbird/addon/2313/](https://addons.mozilla.org/en-US/thunderbird/addon/2313/)
