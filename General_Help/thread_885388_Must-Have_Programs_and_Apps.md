### Post by Unanimated on 2008-08-10
Okay, I've gotten some really useful programs over my time with Ubuntu, some that I think should be included with the OS, but aren't. I'm almost 99% positive that there is a thread exactly like this somewhere on the forums, but I searched around and didn't find one. So, sorry if there is another one, but here goes--
1. Audacity
2. Wine
3. Amarok
4. Sound Converter
5. PulseAudio
These can all be acquired from Add/Remove if you enable Universe repositories, but there is one that shouldn't be gotten from there--
Virtualbox - [http://www.virtualbox.org]("http://www.virtualbox.org")
Well, those are some programs that everyone should have, and some that I personally enjoy. However, if you play guitar, then you should also get Tuxguitar (repos) and LMMS - [http://lmms.sourceforge.net/]("http://lmms.sourceforge.net/")
LMMS is really only if you play Frets On Fire (another great one) and want to fret your own songs the good way, or if you actually want to use it for what it can do. There are some programs (Ubuntu restricted extras) that you will inevitably come across over time (we all own mp3s), and you will eventually have to download.
Some others -
EnvyNG - [http://albertomilone.com/nvidia_scripts1.html]("http://albertomilone.com/nvidia_scripts1.html")
