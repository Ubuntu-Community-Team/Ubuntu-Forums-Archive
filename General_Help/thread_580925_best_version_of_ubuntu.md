### Post by Soarer on 2007-10-21
Gutsy comes with Compiz which gives all the desktops effects you could want. It's enabled by default. As always you can try the Live CD first if you like.

You set-up is similar to mine. I have the desktop Core 2 Duo 1.8Ghz, 2GM RAM and the nVidia 8400 GS. Compiz runs great on mine so it should be fine on yours too.

After you install it will probably tell you it needs to install 'Restricted Drivers' from nVidia - when you do this it will (IIRC) reboot and then you will have your desktop effects. There are a few tools you can configure them with, Preferences>>Advanced Desktop Effects and some others you would have to download (can't remember what they are :))

There will also be an entry in Administration>> Restricted Drivers Manager which will tell you what drivers are in use for your graphics card.

Once it is working smoothly, you can install all the programs out of Ultimate that you wanted.

Are you going for 64bit or 32bit Gutsy?
