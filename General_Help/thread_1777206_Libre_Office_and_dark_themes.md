### Post by bank8705 on 2011-06-07
I use dark themes!

I found a workaround.   Although it's probably not ideal, try this:[INDENT]Open Synaptic Package Manager
Uninstall libreoffice-gtk (will also remove libreoffice-gnome)
[/INDENT]Doing this changed my libreoffice back to the default theme (removing this package simply disables gtk integration, as far as I can tell).

Hope this helps.  :)
