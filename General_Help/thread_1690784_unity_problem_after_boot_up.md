### Post by Quackers on 2011-02-19
I'm not sure exactly what the problem is, but you could go to System > Admin > Synaptic Package Manager, then enter "unity" in the search box. When it appears in the main window right-click on it and select "mark for complete removal" then click on the green tick mark in the toolbar to apply the change. 
Then reboot and see if your desktop comes back.
