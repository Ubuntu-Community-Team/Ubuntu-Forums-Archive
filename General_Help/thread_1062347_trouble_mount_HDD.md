### Post by drs305 on 2009-02-06
Yes, if you can you should mount it back in windows and do a normal shutdown/removal.

If that is not available, you can force mount the partition  with the "-o force" option, or use "ntfsfix" after installing ntfs-config. 

Use the first option if at all possible.
