### Post by TeoBigusGeekus on 2011-10-29
In small words, open a terminal and give
```
sudo apt-get install ubuntu-restricted-extras
```
This will install flash, codecs, java, microsoft fonts, etc.

### Post by linuxuser12345 on 2011-10-29
Go to the Ubuntu Software Center on the left-hand side where the launcher is. Search up "Ubuntu Restricted Extras", click on it, and click "Install." You will have to enter your password.

You should be good to go! :)
