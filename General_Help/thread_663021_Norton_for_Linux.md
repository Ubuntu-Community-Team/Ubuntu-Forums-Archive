I believe you are talking about the Wubi installer.   It installs while you are in windows, but it creates a virtual disk that is contained in a Windows Folder.   You are not running two O.S.'es at the same time,  When you boot, the boot.ini has an added option for booting Ubuntu.   If you want to uninstall Ubuntu, all you have to do is go into Windows and run the Wubi Installer again and it will erase the Virtual Disk.   

I would not consider this the preferred method for installing Ubuntu since Ubuntu is great about creating a dual boot system during its normal install procedure.

Either way, a little Ubuntu doesn't hurt.
