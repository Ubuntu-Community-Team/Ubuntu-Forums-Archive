Our team are running 14.04LTS.

The current test solution is an instance of Jessie on an old machine set up as a print server with all jobs being held until released through the CUPS web interface; the idea being that the employee will only release the job once they are next to the printer.  CUPS version 1.7.5.  If I can get this working, we'll be looking at using a Raspberry PI instead (we don't have any on hand at the moment).
