### Post by gyussz on 2010-10-11
Hey!

The easiest way to create your own web server is to install XAMPP ([http://www.apachefriends.org/en/xampp-linux.html](http://www.apachefriends.org/en/xampp-linux.html)). 

When everything is working fine, choose a dns service on the internet (I suggest dyndns.com/), make an account, and register your ip adress as a host.

If you are behind a router, make sure every needed port is forwarded to the machine that runs your web server (usually port 80 needed).

Hope I could help, have fun!
