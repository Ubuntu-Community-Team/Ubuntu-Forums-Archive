### Post by Tony_Lillie on 2014-04-13
I'm running Ubuntu 12.04 on a system with an Nvidia 9600gtx gpu with 2-dvi and 1-svideo out. I'm using this system as a simple way to view videos and streamed TV shows etc... on a tv through the S-video output. I'm using the proprietary Nvidia 304 driver, which seems to be detecting the two connected displays. One on a DVI and the other the TV on the svideo.

I want to make the appropriate configuration changes so the TV becomes the ONLY display, but in the display GUI when I make ANY changes the system panics and does a desktop reset of some kind. Black screens followed by flickering and then it comes back to mirrored displays.

I've read a ton of info about X and editing the xorg.conf file, but it seems like most of the time it's very system and gpu specific.

Can someone help me please?

EDIT: NO HELP NEEDED. AFTER DISCONNECTING THE DVI DISPLAY AND JUST LEAVING THE SVIDEO, IT HAD NO PROBLEM USING THE TV AS THE ONLY DISPLAY. THE IMAGE QUALITY IS ALSO MUCH BETTER THAN IT WAS IN MIRRORED DISPLAY MODE.
