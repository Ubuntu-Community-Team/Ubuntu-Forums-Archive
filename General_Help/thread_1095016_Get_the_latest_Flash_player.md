### Post by roccivic on 2009-03-13
Enable support for restricted stuff:
System -> Administration -> Software sources
Check the "multiverse" checkbox.
Close and click on reload when asked.

Then:
Applications -> Add/Remove
Select "all available applications" in the drop down menu,
Search for "flash".
You will have the choice of installing "flash 9" or "flash 10".

Hope this helps

Peace, Rouslan
