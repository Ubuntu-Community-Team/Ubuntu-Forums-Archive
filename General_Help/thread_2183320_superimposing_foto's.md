### Post by amanchesterman on 2013-10-24
I'm no expert with GIMP but I've used it a bit. Personally I would:
1. Make sure both the fotos are the required (same) size by opening the files and resizing if necessary.
2. Create a new blank image of the required size.
3. Create a new layer, open the first foto and copy-and-paste it into that layer.
4. Create another layer, open the second foto and copy-and-paste it into that layer.
5. Finally save and export the composite image in your desired format (jpg or whatever).
It may seem a long way round, but having the fotos on separate layers allows you to move and align them independently, so I've found that easier in the end.

### Post by coldraven on 2013-10-24
If they are the same size then just use File>Open as layers, and select them both
You probably want to play with the transparency of each layer.
Imagine that you are looking down on the list of layers. the one at the top is first. You can hide it by selecting the "eye" icon. You can also drag the layers up or down the stack.
