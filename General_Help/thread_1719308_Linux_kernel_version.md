### Post by ojdon on 2011-04-01
Type in "uname -r" without the quotes in the terminal. That will display the Kernel version.

Hope that helps!

### Post by 3Miro on 2011-04-01
Use
```
uname -r
```
or
```
uname -a
```

Also, it is not obvious that you are running Ubuntu 10.10 since you could be running 10.04 or 9.10.

On 10.10 you should be running Linux kernel 2.6.35.
