### Post by lost.sync on 2006-11-20
You can also install the Art Manager via:

```
$ sudo apt-get install gnome-art
```

It's located under System->Preferences->Art Manager after installation.  It's simply a front end to gnome-look.org that can help browse and install themes, wallpapers, and icons.  

artificial intelligence is making heavy use of gDesklets as well, which you can get from the repositories.  happy tweaking.
