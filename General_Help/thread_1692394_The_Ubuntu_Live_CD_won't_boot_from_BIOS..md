### Post by stillnotelf on 2011-02-21
I usually try hitting all the "F" keys (F1-F12) during boot - just jam on them like a rock star.  Eventually you'll find the one that lets you get into the BIOS and/or pulls up the "boot from which device" menu to boot from the CD.

Not autobooting from CD is a good thing - you don't want it trying to boot from every CD you leave in the machine, like in the bad old days of floppy disks...
