### Post by zombiepig on 2008-11-11
It's not installed by default on ubuntu, so you'll need to install it first.

Load up System->Admin->Synaptic Package Manager, then do a search for 'hamster'. You should see hamster-applet found, right click on this and select 'mark for installation'. Then press 'Apply'. This should install the applet.

Next, you need to add it to your panel. Right click on a blank area on one of your panels, and select 'add to panel'. In the list that comes up find 'time tracker', select it, and press 'add'. :D

Hope that helps!
