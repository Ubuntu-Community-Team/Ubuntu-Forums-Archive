### Post by bonzodog on 2007-07-13
There is a way of doing it, yes, as Zenwalk has a boot splash with Lilo. It seems that in our case the initrd has been built to include the splash.

If you have an older system, you might like Zenwalk Linux -- it's as easy to use as Ubuntu, uses Xfce as the default desktop, but it is a very light distro - it only has one app per purpose. 

We have a thread for it here in 'other OS talk':

[http://ubuntuforums.org/showthread.php?t=231269](http://ubuntuforums.org/showthread.php?t=231269)

It is based on Slackware, and thus is very light and fast.

The website for Zenwalk Linux: [http://www.zenwalk.org/](http://www.zenwalk.org/)
 It comes with all codecs out of the box, and GUI tools for most configurations.
