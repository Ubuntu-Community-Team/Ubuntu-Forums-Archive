### Post by garyzw on 2012-06-07
I use Configuration Editor 1.2.0 An editor for the MateConf configuration system.

Just download the 32bit or 64bit deb and install with software manager

[http://www.ubuntuupdates.org/packages/show/446078](http://www.ubuntuupdates.org/packages/show/446078)

Open the Configuration Editor and go to apps > caja > desktop then uncheck the  Computer, Home and Trash options
