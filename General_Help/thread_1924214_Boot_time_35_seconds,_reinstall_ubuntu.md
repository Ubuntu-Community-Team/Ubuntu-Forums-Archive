### Post by Thorsen V on 2012-02-12
How often do you reboot? 

Saving a few seconds each boot-time needs to be factored against how long it will take to reinstall AND install all the extra things that you've customised your system with. That a lot of system booting to get back all that time!

I don't believe linux has the same issues/criteria for doing a reinstall  as windows (specifically the danger of spyware is much lower IMO ... but one must not be complacent about this either)

But if you do decide to go ahead, make sure of your backups first!

Oh and before you do anything check out dmesg for any clues to what the problem is and/or check out System > Administration > Log File Viewer. You never know, there may be something indicated that you can just disable to to speed up the boot, or you perhaps might discover that there would be no advantage in reinstalling because the thing that's taking the time is something you can't live without...
