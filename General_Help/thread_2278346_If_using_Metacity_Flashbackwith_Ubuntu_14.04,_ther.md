### Post by oldefoxx on 2015-05-15
The title says it all, pretty much.  Every step I've found online concerning making global settings changes for either the cursor theme or size does NOT work everywhere.  The result if you try all these, is that you will have different mouse cursor themes and sizes showing up as you move the mouse pointer about.

The problem appears to be Metacity interface-related, and you won't be dealing with Metacity unless you opted for installing Flashback to get back the Classical Look of Ubuntu.

The fix is a simple workaround.  Either stick with the login Ubuntu default, or switch to the Ubuntu conpiz choice.  Cursors appear pretty stable for the most part under either of those two, but not 100% of the time.
