### Post by stuartwood89 on 2008-02-13
You might have opened up your CD burning program and just dragged the file into the burn list.  That will only put the .iso file onto the CD, not burn the disc as an actual image.  So what you need to do is this:

Open the burning program as normal, then on the top bar (depending on your program) click on 'actions' and there should be something that says 'burn image'.  Click on that and find the .iso you downloaded, then follow the instructions from there.

If you're using InfraRecorder like I did, then follow what I said above.

Then when you boot from the CD, you will see the Ubuntu install screen, then install.

Hope that helps!

### Post by thecorleonefamily on 2008-02-13
Start your computer and press DELETE (works for most computers) or F2. You will be presented with the BIOS options for your motherboard. Here navigate either to Advanced BIOS Information or Boot Device or somethign and change the boot order to CD ROM first. If you need to know which key to press to enter BIOS you need to refer to the motherboard manual.
