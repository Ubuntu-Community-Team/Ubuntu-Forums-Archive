### Post by Tim Sharitt on 2008-04-23
If I understand you correctly you would like to open a file manager to a certain folder. A launcher for your particular file manager(nautilus is the default for gnome, and I believe kde uses dolphin) followed by the folder you want to open should do the trick.
/mnt folder in nautilus:
```
nautilus /mnt
```
