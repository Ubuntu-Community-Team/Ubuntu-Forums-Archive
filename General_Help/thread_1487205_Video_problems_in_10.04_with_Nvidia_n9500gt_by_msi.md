### Post by Skywa&#8224;cher on 2010-05-18
I had a similar problem with an NVidia card when I installed Ubuntu 10 on my desktop PC. First, try going to System > Administration > Hardware Drivers to see if there is a (new) driver available for your graphics card. If there is, install it and restart your system.

If not, or else if that doesn't fix your problem, see if there is an option for the NVidia Control Panel (if memory serves me right, since I'm currently writing this from my laptop) in the System > Preferences menu. If there is, click it and have a look through the options available. I seem to recall there are quite a few handy little tweaks available, which could be of use. Unfortunately I can't remember exactly what options NVidia provide, and I don't have access to my desktop PC right now...hope this helps somewhat, though!
