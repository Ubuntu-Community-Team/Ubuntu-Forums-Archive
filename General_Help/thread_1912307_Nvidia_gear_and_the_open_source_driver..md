> If I remove the proprietary driver, does the open source one kick in automatically or do I have to re-install it or re-enable it somehow?

> Also, would the Nvidia-Settings GUI still be used, or is all monitor work (dual and triple display modes, etc) handled through the System Settings - Display menu?

The Nvidia X Server Settings are part of the Driver package , so you would be using the system default monitor settings .
