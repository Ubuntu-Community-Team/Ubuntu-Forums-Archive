### Post by Rob Sayer on 2014-08-16
You change that in Dolphin file manager, which is very, very powerful and configurable.  I use nothing else even on a non KDE install.

Right click the file, hit Properties and then file type options.  You can then set up an ordered list of applications to open the file with.

You can even do this with folders.  I set dolphin up to open folders with Audacious if I just want to play a ripped CD in a folder, or soundconverter if I want to convert those audio files to mp3.
