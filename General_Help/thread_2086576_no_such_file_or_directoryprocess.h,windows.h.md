### Post by Mark Phelps on 2012-11-21
From wikipedia:

> windows.h is a Windows-specific header file for the C programming language which contains declarations for all of the functions in the Windows API, all the common macros used by Windows programmers, and all the data types used by the various functions and subsystems.

This is Linux, not Windows.  Apps written for Windows will not run here.
