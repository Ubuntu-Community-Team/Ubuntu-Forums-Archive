### Post by SPr on 2008-04-28
I would check the read/write permissions on the file first. Right click the file and then click properties>permissions tab. If they're wrong change them to allow reading and writing.

If the permissions are correct (ie reading and writing is allowed) open a new file and copy and paste the document to the new file.

There is no guarentee that downgrading OOo will fix this problem nor introduce new problems.
