### Post by xelxel on 2009-01-15
I'm using a Ati Radeon HD 2400 XT graphic card in my laptop, and at the moment using windows XP. In XP i am currently running 1280x800 in resolution. While in Ubuntu/Xubuntu the maximum resolution i get is 1024x768. How can i up the resolution in *buntu to the same i have in windows?

---

### Post by DocHoliday52090 on 2009-01-15
First check out 'Hardware Drivers' in System -> Administration. Install the drivers through there. 

Once you got that done (restart), go to System -> Preferences -> Screen Resolution. 

Good luck
