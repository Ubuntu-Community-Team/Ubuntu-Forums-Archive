to install rssowl visit this web site , theres a link to install the repo for ubuntu

[http://packages.rssowl.org/README](http://packages.rssowl.org/README)


i have had rssowl running for a few days now and today my pc locked up hard

i had no choice but to hit the reset button

upon restart rssowl refused to start

kept giving me an error that it could not starts cos it was trying to read past end of file

lucky for me i back up my rss feed list every now and then

i went to  $ home/.rssowl2

and delete this directory fully

now when restart rssowl it start up like a new install and ask for my *.opml file

it loaded up and all good again

hope this helps others
