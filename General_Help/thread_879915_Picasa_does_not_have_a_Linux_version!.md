### Post by jimv on 2008-08-04
For all intents and purposes, it IS a Linux version.  If you have Linux, you can download the Linux version of Picasa, install it, and it will run on your Linux OS.

Linux program = program that runs in Linux

There is no special Linux code or programming language that makes a program a Linux program.  At the most you could say that Picasa is a Linux program that is dependent on WINE libraries.  But nevertheless, it is a Linux program.
