### Post by WorMzy on 2010-11-16
How can one Linux operating system be more Linux-like than another Linux operating system? :P

I'd recommend that you try out multiple distributions though, just for contrast. The primary differences between distros is their package manager, e.g. Ubuntu/Debian use DPKG, Fedora uses RPM Package Manager, Arch uses Pacman, etc. Another point of interest is the Desktop Environment that a distro uses. Ubuntu uses GNOME, Kubuntu uses KDE, Xubuntu uses XFCE, etc.

As for your original questions, I have no idea whether those books are good or not, as I've never read them. Some books I *can* recommend though are:
Krause, A., 2007. Foundations of GTK+ Development. New York, USA: Apress.
Samuel, A., Oldham, J. & Mitchell, M., 2001. Advanced Linux Programming. New Riders Publishing.
Petersen, R., 2007. Linux: The Complete Reference. 6th Ed. New York, USA: The McGraw-Hill Companies, Inc..
Dalheimer, M. & Welsh, M., 2006. Running Linux. 5th Ed. Sebastopol, California: O'Reilly Media, Inc.
