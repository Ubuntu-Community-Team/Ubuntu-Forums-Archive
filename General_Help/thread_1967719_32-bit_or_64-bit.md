### Post by Carterclan on 2012-04-28
It's recommended because some hardware cannot run 64-bit but most can run 32-bit. so for those who are not sure can get the 32-bit without issue.

### Post by IWantFroyo on 2012-04-28
It's only recommended for people who don't know their processor architecture.

If you know your processor is 64 bit, then go for it. 64 bit Linux has been stable for a long time.
