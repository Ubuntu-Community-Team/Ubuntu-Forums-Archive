### Post by mcduck on 2011-12-01
All Ubuntu versions are one and the same operating system, only the desktop environment and applications installed by default change. So there is no difference between Ubuntu, Kubuntu, Xubuntu, Lubuntu, Edubuntu, Mythbntu or the Server version. They all run same programs, use smae packages form the same repositories, and can have any desktop environment, server applications etc. you want to install.

Mint however is based on Ubuntu, but makes some changes of their own. For example your problems could be related to the fact that Mint is trying to stay with the old Gnome 2 while the latest Ubuntu version has already moved with times and runs Gnome 3 and GTK3 instead. It's hard to say anything for sure without knowing more about the actual problems you are having, and if they are related to installing or running some application.

### Post by crazy bird on 2011-12-01
Yes, software which runs on Ubuntu can run on Lubuntu, Xubuntu and Linux Mint. Firstof all, Ubuntu is Debian-based, Xubuntu and Lubuntu is just Ubuntu with another desktop environment. Xubuntu uses Xfce and Lubuntu uses LXDE as desktop environment. Linux Mint is a fork of Ubuntu.
 
But all the program's you're running on Ubuntu can also run on other Debian-based (or Ubuntu-based) operating systems.
