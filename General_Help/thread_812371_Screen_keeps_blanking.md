### Post by wpshooter on 2008-05-29
How do you have the timing of your screen saver set and how do you have the timing of your screen power off function set ?

Try setting the screen saver timing to 1 minute (this is even if you have the screen saver disabled) and then set the screen power saver to never.  Then restart the O/S/computer and see if this fixes it.

Good luck.
