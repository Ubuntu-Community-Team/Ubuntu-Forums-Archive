### Post by lovinglinux on 2011-10-10
Only Chrome 32bit comes with the plugin.

Firefox doesn't recognize Chrome's flash plugin. However you can copy the plugin to Firefox. Get [Flash-Aid](https://addons.mozilla.org/en-US/firefox/addon/flash-aid/) for Firefox. After Flash-Aid installation, instead of running the Wizard as it will suggest, select the *Advanced Mode* from Flash-Aid menu. Then select "Google, from Chrome" in the *Version and Source* drop-down menu in the *Installation Options* and execute the script. It will remove all current system flash versions and create a symbolic link to Chrome's flash plugin. Every time you update Chrome and get a new flash version, it will become available to Firefox as well.
