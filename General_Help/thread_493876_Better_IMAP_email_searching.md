### Post by p_quarles on 2007-07-06
Are you averse to storing your e-mail locally? If not, Thunderbird fully supports downloading IMAP mail for offline use. Once you do that, it becomes a lot easier to index them using whatever search tool you want. You should see if KMail supports something similar. 

That said, I think Thunderbird keeps really good track of online IMAP mail as well, so if you don't want to download everything you might give that a shot.
