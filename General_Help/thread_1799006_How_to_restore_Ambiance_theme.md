Hello everyone. I recently installed the Ambiance Evolution theme. In order to do so I had to remove the original Ambiance theme. Well... now I want the original back! Any ideas on where I can get it?

---

### Post by Alwimo on 2011-07-07
Remove the ppa (go to the software centre, then go to the edit menu, then software sources, and remove the ambiance evolution ppa/s).
 
Then open synaptic, search for light-themes, and reinstall that package.
