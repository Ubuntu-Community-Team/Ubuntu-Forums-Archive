### Post by coskierken on 2010-05-21
I have the GTS 250/1gig mem with a Zalman cooler VF1000 (great addition to a good card as it keeps it much cooler and quiet, also and yes, I know it is a 9800 with a few improvements).  It is great in Win7 and Ubuntu (dual booting).  The only thing I don't like is when kernel updates happen, sometimes, it will not boot and gets stuck with driver problems.  I keep a copy of the driver in my root directory so I can easily run the setup program and all is well in just a few minutes.  I wish they would fix this as it should not happen.  The only other thing which might lead me to ATI is the on-board sound it has on it's card.  Nvidia does not and you have to run the sound through a pass-through from a spdif header on the motherboard to the card (usually a two-wire).  My motherboard did not have a spdif header so I had to use a toslink cable from the on-board sound (not tht bad at all) to my sound system.  Just something to think about.

Thoguhts? You'll pay through the nose with Quadro cards. A FX3700 uses the same chip as a Gt8800, but its still over $750 US. FX5800 (currently over $3000) uses the same chip as you will find in a GTX285. 

You used to be able to 'softmod' the geforce cards into quardos, and while it wasnt quite as fast as a 'real' quadro, you do save a huge amount of money. The softmods always hang a generation or so back, and ou really need to check if you can softmod before you buy the card. 

If you are going to be using this under linux then I wouldnt bother- the linux geforce drivers are the same ones used for quadros ;) 

BTW, I know somebody who works with maya, etc, I'll ask him how good the ATI cards are (if hes used one)- FirePro/FireGL cards are a lot cheaper.
