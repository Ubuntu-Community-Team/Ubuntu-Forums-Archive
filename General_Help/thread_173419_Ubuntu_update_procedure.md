### Post by aysiu on 2006-05-10
You'll be hurting without an internet connection.

Honestly, you won't need security updates, since you aren't connected to the internet, but if you want software, you can download .debs and install them manually (more info [here](http://www.psychocats.net/ubuntu/installingsoftware)).

Or, if you want a whole CD repository: [https://wiki.ubuntu.com/AptMoveHowto](https://wiki.ubuntu.com/AptMoveHowto)
