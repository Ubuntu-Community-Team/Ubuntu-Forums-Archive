### Post by sikander3786 on 2011-01-26
Welcome to the forums :-)

That is not possible. Every OS runs independent of the other. Unless you install Windows inside Ubuntu or Ubuntu inside Windows as a Virtual Machine, you can run both of them simultaneously.

Yes you can run both of them on seperate hardware connected to one keyboard, mouse and monitor and swap using a KVM switch. But that is not the solution here.
