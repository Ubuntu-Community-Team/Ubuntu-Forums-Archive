### Post by dunkar70 on 2009-05-26
It sounds like your theme might be off a bit. 

Navigate to **System** > **Preferences** > **Appearance**. The **Theme** tab should be selected as should the theme you are currently using.
Select the **Customize** button.
Select the **Colors** tab.
Make sure each of the non-text boxes have enough contrast you can tell the colors apart. Make sure none of them are too close to the text as well.

When you are done, you might want to use the **Save As** button to give it a new name. This way you will still have the original if you want it.

By the way, OO has its own theme selection too. If you are only having trouble in OO, then try navigating to **Tools** > **Options** > **OpenOffice.org** > **Appearance** and play with the themes there too. Remember to save your changes as a different name so you can restore the original if you need to.
