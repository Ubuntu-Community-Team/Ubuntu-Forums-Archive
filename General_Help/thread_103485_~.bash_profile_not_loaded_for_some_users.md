### Post by mikkelwe on 2005-12-14
The reason why ~/.bash_profile is not sourced is probably that the user is not using a login shell. Depending on shortcuts and other settings, launching a terminal window may or may not cause the shell to be a login shell. In the case of a login shell ~/.bash_profile is sourced otherwise ~/.bashrc is sourced. This is the reason ~/.bash_profile usually doesn't do much more than sourcing ~/.bashrc, making sure ~/.bashrc is always sourced for interactive bash shells.
In other words put whatever you want to be sourced in ~/.bashrc and make sure ~/.bash_profile sources ~/.bashrc like this:

# ~/.bash_profile
# include .bashrc if it exists
if [ -f ~/.bashrc ]; then
    . ~/.bashrc
fi
