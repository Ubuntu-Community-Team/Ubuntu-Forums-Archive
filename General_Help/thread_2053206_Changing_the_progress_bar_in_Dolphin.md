### Post by Stonecold1995 on 2012-09-04
When I run Dolphin under my normal account, the progress bar for copying files is in a passive popup, but if I run Dolphin as root the progress bar is in it's own window.  Personally, I prefer the latter, but I haven't been able to find any configuration options that'll allow me to change it.  So my question is simply, how do I set dolphin to use the window progress bar without having to run it as root?

### Post by Cynical on 2012-09-04
Yes, right click on the "I" in your taskbar, go to Notification Settings, and uncheck "File transfers and other jobs". The change should take effect after you log out and back in.
