### Post by Petro Dawg on 2016-04-17
Unetbootin is a similar program that I use to create bootable USBs and it also requires an admin password. The nature of what it does allows it to modify essential parts of the system which will require a password. Just to be on the safe side, I would stick to using software available in the software center if it can serve the same need.
