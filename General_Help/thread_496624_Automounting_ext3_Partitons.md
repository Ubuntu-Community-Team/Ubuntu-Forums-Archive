### Post by Trebaruna on 2007-07-09
Can't you simply edit fstab to have that partition mount as /home?

Do like so:
(commands should be typed without quotes)
Open a terminal, and type "blkid"
Note the UUID of the partition that is going to be your home folder.
Open fstab in an editor with root privileges (like: "sudo gedit /etc/fstab")
Add a line in the file like this:

UUID=<the number you found> /home ext3 defaults 0 1
