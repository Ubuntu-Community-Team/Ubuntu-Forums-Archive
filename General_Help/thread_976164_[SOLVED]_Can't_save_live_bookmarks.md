### Post by PeregrinGo on 2008-11-11
Okay all, I guess we can mark this one as semi-solved. I'm not sure what actually caused the problem because I tried disabling the few extensions I had and that still didn't take care of it. Then I tried Safe Mode and that didn't do it either. So what I did was create a new Firefox Profile. I checked to see if it would allow me to add Live Bookmarks. This worked. So I added those feeds I desired to have.

Next I switched profiles in order to export my bookmarks to an HTML file. I also copied all of my original profile's files to the new profile folder, excepting those files that were already existent from the creation of the new profile. This prevented my having to re-install the aforementioned extensions.

After this, I closed Firefox and opened the new profile. From here I imported and organized the bookmarks from the HTML file I had created. Then I closed Firefox again, deleted the original profile. Renamed the new one to default and PRESTO. All is well.

The one down side is that I will have to manually enter and re-save my passwords to web sites I frequent, such as this one. But at least I can get RSS feeds now. I'm somewhat of a news junkie, so that's important.
