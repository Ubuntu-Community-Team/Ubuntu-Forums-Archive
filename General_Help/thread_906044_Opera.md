### Post by jonian_g on 2008-08-31
Got to Opera menu Tools>Preferences and on the Advanced tab on the left bar choose Downloads. Uncheck the "Hide file types opened with Opera" and search for .torrent extention. Select it and click the Edit button to set the application you want to handle torrents.
