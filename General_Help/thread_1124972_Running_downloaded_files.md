### Post by cariboo on 2009-04-14
If you are a new user, I would suggest you install the video drivers included in the repositories. Go to System-->Administration-->Hardware Drivers and select the recommend driver for installation. It may take a while as the driver has to be downloaded before it installs automagically. To install programs, use Add/Remove Programs from the Applications menu or Synaptic Package Manager From System-->Administration menu. There are over 40,000 package in the repositories. Not all of them are programs, but for the average user there is almost every program you can imagine. There is no need to download files from random web sites, like Windows.
