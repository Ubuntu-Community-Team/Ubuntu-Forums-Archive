### Post by Rubi1200 on 2010-12-14
Hi and welcome to the forums :)

Odd...because the results seem to suggest that everything is normal.

Just out of curiosity, and since sometimes the results can be misleading, let's reinstall GRUB from the LiveCD and see if it makes a difference.

```
sudo mount /dev/sda5 /mnt
sudo grub-install --root-directory=/mnt/ /dev/sda
```
If there are no errors and you can reboot into Ubuntu, run this:
```
sudo update-grub
```
