### Post by GrimmReaperVI on 2009-01-18
I bring up the terminal (The alt ctrl F1 terminal) and open (sudo) nano to edit a file (xorg.conf to be exact)
So, I edit it. Great. How do I save it? And how do I use the commands at the bottom like ^X or ^G and stuff? It's a very nooby question I know, but we've all been there.  :confused:

---

### Post by MISIIM on 2009-01-18
The commands at the bottom are used Ctrl+[letter]. To exit use Ctrl+X. When you exit it will ask you if you want to save. You can also use Ctrl+O to save the file.
