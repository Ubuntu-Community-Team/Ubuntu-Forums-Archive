### Post by Irihapeti on 2014-09-09
I uncheck all the boxes except for "automatically download new messages". Then, if I actually want Thunderbird to check regularly for new mail, I'll check the "check for new messages every .... minutes" box. If I'd rather do it manually, I leave that one unchecked as well.

"Leave messages on server" is also over to personal preference.
