### Post by jlintz on 2006-02-21
Well there is always bmpx or amarok.  bmpx has the feel of a modern xmms with a media library.  Amarok is a kde app but you can run it in gnome fine and is a full out media library.  Not a compact gui but has a lot of cool features.  The latest version supports ipod's.
