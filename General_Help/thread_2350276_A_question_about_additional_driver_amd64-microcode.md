### Post by mörgæs on 2017-01-23
So-called microcode is a closed-source unit delivered by AMD without the option to review. It might deliver (a marginal?) performance boost, it might invade your privacy. Noone knows - well, someone knows but you can't be sure that they are telling you the whole truth.

I tend to stay away from microcode unless I have a specific problem that I am trying to solve.

### Post by howefield on 2017-01-23
> **waltd said:**
> ... What is it and should I install it or not?  Will it improve my laptop in someway?

It is an update to the code already running inside your CPU, microcode updates are generally released to fix/improve issues with the original cpu firmware.

There are probably release notes online that you can search out, I don't readily have links.

I prefer to run it albeit improvements are probably imperceptible.
