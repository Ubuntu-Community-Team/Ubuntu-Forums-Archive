### Post by buzzingrobot on 2013-10-19
Regardless of the font you decide to specify, it needs to be available to a visitor before it will be displayed.  Unless you want to add code to download the font, say, from Google, on each page load, be sure your font-family spec in the CSS includes fallbacks to fonts commonly on the machines of your target audience.
