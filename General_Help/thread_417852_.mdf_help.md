### Post by EntropicRapture on 2007-05-05
Type UDF: it's the "standard" DVD filesystem type, to my knowledge; even though iso9660 will usually work, that's really more of a CD filesystem format.  For instance, this just worked fine for me (replacing `image.mdf' with a valid file, of course):
```

sudo mount -t udf -o loop ./image.mdf /mnt/cdrom

```

Enjoy.

### Post by cendant on 2007-05-12
Guys, you forgot to load loopback device into kernel

try these:

sudo mkdir /media/mymdf
sudo modprobe loop
sudo mount image.mdf /media/mymdf -t udf -o loop
