My firefox didn't want to update itself so I deleted it with Synaptic and used the download package of the firefox website. Now I have the new version of firefox but the folder is in my Desktop and I would like a shortcut to be in the Applications>Internet menu of Ubuntu as it was before. How can I do it?

### Post by uRock on 2010-03-19
After installing the new Firefox, go to System> Preferences> Main Menu, then click on Internet on the left panel, then click New Item in the upper right of the window. In the top line enter Firefox, in the command line enter Firefox-3.6, then if you want you can enter a description in the third line. Once you are done there should be a Firefox button in the Internet menu where it was with the 3.5 version.

I'm not sure that will work if the firefox you downloaded is in your home folder.

It's still easy enough to do, however, by just using the path to the newly downloaded script file called simply firefox in the new folder from the tarball extract.  For example, if the new firefox folder is on your desktop, the pathway will be /home/username/Desktop/firefox/firefox.

The only thing missing will be the icon.  Another way to do this update which will then continue to update itself is to enable and use the ubuntuzilla repos.
