Hello,

Here is Voximp, the new little one. 

Speech Recognition, intended to Desktop Control: Voximp:
pocketsphinx, speech recognition, voximp. 

One found a thing called pocketsphinx &#8211; a speaker-independent speech recognition library for linux (think can be compiled on windows too). Even better &#8211; it has python bindings. The result of this was:
Voximp

Here is the installer:
 
```
cd /tmp
wget http://yellowprotoss.ye.funpic.org/website/voicereco/voximp/voicevoximp-installer.sh
sh voicevoximp-installer.sh

```
and follow the information
