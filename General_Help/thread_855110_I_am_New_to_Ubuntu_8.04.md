### Post by Sn3ipen on 2008-07-10
Welcome onboard.

Ubuntu wont tell you if you have a cam installed. In most cases it just works. :)

About the Windows apps and games you should try [url=http://www.winehq.org/]Wine[/ul]. It will make you able to run Windows apps and games
If you want to know what Windows programs that will run under Wine you can check out [this site](http://appdb.winehq.org/). It will tell you what programs and games that will work and not.

Good luck with your new operating System. :)
