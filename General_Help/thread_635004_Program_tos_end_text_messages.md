### Post by p_quarles on 2007-12-11
You could try installing smssend:
```
sudo apt-get install smssend
```
Full package description: [http://packages.ubuntu.com/gutsy/comm/smssend](http://packages.ubuntu.com/gutsy/comm/smssend)

There's also an extension for Firefox, as well as a Google application. Those aren't CLI, though.
