### Post by TenPlus1 on 2011-05-07
Ubuntu isn't windows so it should not get slow after months of use unless you specifically install many programs that run in memory or forget to clean up your browser cache...  I would recommend using BleachBit to clean up the software you have installed and also remove any startup programs you don't need so it will make your system faster...  Also, make sure your gfx driver is active (nvidia/ati) after kernel updates as this could be the culprit of a slow setup also...

### Post by chinoto on 2011-05-16
Gnome Menu on the Gnome Panel> System> Administration> "Additional Drivers"
If you do intend to reinstall, go into Synaptic Package Manager and look through the manually installed programs for stuff you want to install after reinstalling the system and backup your home folder (use Disk Usage Analyzer to avoid backing up space hogs, eg caches). Also you may want to install Ubuntu 11.04 since its done (if you hate Unity and they don't offer an option to change it, try Linux Mint 11 at the end of May).
