### Post by taurus on 2009-04-17
The advantage of having a separate /home is that when you reinstall, you can keep all your personal settings and files in $HOME.

[http://www.psychocats.net/ubuntu/separatehome](http://www.psychocats.net/ubuntu/separatehome)

### Post by ravi_buz on 2009-04-17
does that mean all the software in install will also be in it and can use it if i upgrade or reinstall

---

### Post by taurus on 2009-04-17
No, all the install programs will go to / (mostly /usr/bin) unless you install them directly to your $HOME.
