### Post by Enigmapond on 2010-03-19
You should be able to right-click on the file and choose "Encrypt" from there, if you don't have a key set, it should open "encryption & keyring" to allow you to set a key. Once you have a key set, it should then allow you to encrypt the file....there might be an easier way but this is how I figured it out...
