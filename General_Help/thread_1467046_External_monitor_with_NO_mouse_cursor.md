I am using a Asus UL30A laptop computer. I just performed an upgrade from 9.10 to 10.04.
I remember running into this problem before, but I forgot how to fix it.

The problem is, when I plug my laptop into the external monitor with my laptop screen turned off automatically when I first log in, I do not see the mouse cursor on the external monitor! The mouse cursor is still here, although invisible - if I do a right-click, the property menu will come up.
