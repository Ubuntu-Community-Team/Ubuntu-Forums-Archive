### Post by swarup on 2007-05-19
When you have created an alternative keyboard layout (in my case Hindi language), how do you get it to be active? 

I have added Hindi language support under System -> Administration -> Language Support. Then I added the Hindi keyboard layout alongside the English one, by System-> Preferences -> Keyboard -> 'Layouts' tab -> 'add' -> select the 'India' layout. Then I set up a hotkey to move between the two keyboards, by Layout Options tab -> 'Group Shift/Lock Behavior' -> pick which hotkey you want. 

So I set up everything. But in the end it says under the Group Shift hot key setup, that the keyboard only switches "While the hotkey is pressed". So how is that any use? If you are going to type a document in OO, you can't sit there holding the alt key down the whole time. How do I get it to switch keyboards to the Hindi one, and remain like that while I am working?

### Post by Aathos on 2007-05-19
Swarup,

The first 6 say "switches group while pressed", the rest say "changes group".   If you chose one that says "switches group while pressed" you have to keep it pressed.  If you choose one that says "change group" you will not have to hold the key down. So, chose one that says "changes group".  You may have to scroll down to see the rest of the options.
