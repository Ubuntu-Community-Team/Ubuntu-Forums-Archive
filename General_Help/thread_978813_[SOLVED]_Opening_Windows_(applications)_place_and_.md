### Post by sdennie on 2008-11-11
It sounds like you are using compiz without the window placement module enabled try installing ccsm and enabling it:

```

sudo apt-get install compizconfig-settings-manager

```

Then go to System->Preferences->Advanced Desktop Effects and enable the "Place Window" module (it's close to the bottom).

Edit:
Also, please don't bump your posts this frequently.  They get read even when they make it off the first page and you are actually *more* likely to receive help here if you don't bump.
