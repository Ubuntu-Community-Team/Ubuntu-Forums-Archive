### Post by ssreddy555 on 2011-06-17
U can find HP Device Manager in Applications >> Accessories if u r using the Classical interface & i'm sure u can find it in a similar place in unity as well if u r using that interface.

U can then right click on it & click "Add this Launcher to Panel"

I find the classical interface much more user friendly than the unity interface. Very convenient to use. After using Unity for a while, I switched over to the Classical Interface.

### Post by ssreddy555 on 2011-06-19
Here is the solution:

It's actually very simple.

Right click on the upper panel >> Add to Panel

In the pop up menu, navigate to "Notification Area" & add it.

Make sure in the start up applications menu in System >> Preferences, "HP System Tray Service" is ticked.

On reboot, HP icon reappears in the system tray.
