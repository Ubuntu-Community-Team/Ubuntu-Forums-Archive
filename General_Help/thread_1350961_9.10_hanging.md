### Post by john newbuntu on 2009-12-10
It appears like you may have one or more bad sectors on your disk (i.e., a bad disk).  Check the disk utility program and see if it shows bad sectors.  
Run the command
dumpe2fs -h /dev/sda5
from a terminal and see if it show bad blocks assuming /dev/sda5 is where you have your linux root filesystem.

You may have to run the e2fsck program from a liveCD on your filesystem to correct any errors.
sudo e2fsck -f -y -v /dev/sda5
