### Post by priegog on 2008-04-25
Automatix stopped existing becuse as Ubuntu became better what it provided became redundant and irrelevant. But everything you described can be updating by doing a 
sudo apt-get install ubuntu-restricted-extras
If you need anything else, just ask.
