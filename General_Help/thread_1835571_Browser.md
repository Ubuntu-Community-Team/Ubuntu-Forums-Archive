### Post by seawolf167 on 2011-08-29
Firefox has been working just fine for me for a very long time. Have you been updating and upgrading your computer? 

You can check just Firefox with

```
sudo apt-get install firefox
```

Or the entire system

```
sudo apt-get update && sudo apt-get upgrade
```

My "fall-back" browser is Chrome (not my primary because it doesn't have all my addons available yet)

```
sudo apt-get install chromium-browser
```
