### Post by DawieS on 2011-05-29
To save flash videos that you have viewed earlier, open a file browser in "List View" mode and select "View - Show Hidden Files".

Browse to your home folder, and look in the following folder:

home/.mozilla/firefox/???????.default/Cache.

Click on the "Date Modified" tab, and determine from the type, size and date/time which file it is that you want to get, then copy this file to another location of your choice, renaming it also to your liking.

If you have your Firefox preferences set up to clear the cache on exit, you have to do this before closing Firefox.:grin:
