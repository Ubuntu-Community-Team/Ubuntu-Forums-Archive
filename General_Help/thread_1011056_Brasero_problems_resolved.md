### Post by exploder on 2008-12-14
I had been having problems with DVD burning in Hardy 32 bit. Today I found that getdeb.net has Brasero 0.8.3 packages available for Hardy. I completely removed Brasero using Synaptic and installed the new packages from getde.net. The problems with error messages burning DVD iso files and blanking DVDs are gone! One note, I do have all of the plugins turned off. Here is where you can find the updated Brasero packages.

Hardy 32 bit

[http://www.getdeb.net/search.php?keywords=brasero](http://www.getdeb.net/search.php?keywords=brasero)

Intrepid 32 bit

[http://www.getdeb.net/search.php?keywords=brasero](http://www.getdeb.net/search.php?keywords=brasero)

Intrepid 64 bit

[http://www.getdeb.net/search.php?keywords=brasero](http://www.getdeb.net/search.php?keywords=brasero)

The Hardy 64 bit package has not yet hit the servers but I would expect to see it available soon. Hopefully this will help some of you out that have been experiencing problems.
