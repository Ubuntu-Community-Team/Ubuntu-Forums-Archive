### Post by Rubi1200 on 2011-04-22
That's a good point made by Frogs Hair.

It would be quicker to just download the regular desktop ISO from the Ubuntu site and burn it to CD.

When finished, start the CD inside Windows and it will automatically let you start running wubi.exe.

The other advantage is that you then have a usable LiveCD for troubleshooting should the need ever arise.
