### Post by Xero Xenith on 2009-03-27
Not sure if this is exactly what you're looking for, but try System - Preferences - Keyboard Shortcuts. Click Add, type an approprate name for the shortcut and type the command you want it to run when the key is pressed.

Apply, go all the way to the bottom and double click next to the name under the "Shortcut" column, and hit the gamer button.

That's all I can suggest without knowing excactly what you want each button to do, but maybe someone else with more knowledge of shortcuts and the like could help you more.
