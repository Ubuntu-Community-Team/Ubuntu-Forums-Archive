### Post by Lateralis on 2011-05-24
Check out [SciDAVis]("http://scidavis.sourceforge.net/") or [QtiPlot]("http://soft.proindependent.com/qtiplot.html").  Both are very similar, so there's only really a need to check out one or the other.  Both are also in the main repos so can be installed through the software centre, synaptic or the command line, whichever is your preferred method of software installation!  

As an aside, you may also want to check out xmgrace.  I'm not sure it can plot user-given functions, but can read in ascii datasets and make graphs which look OK without too much effort.
