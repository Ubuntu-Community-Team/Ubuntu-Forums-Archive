Basically I done the following:

Installed 2 fonts using Import function in FontMatrix.
Decided I didn't like them so deleted them from my system and then re-synced font cache using FontMatrix.

And when I go to Fonts in something like OpenOffice, they still appear. Even though they're not installed on my system anymore.

I have re-logged in and also rebooted but they will not go away.

EDIT: I just solved the problem myself. I installed and run started the xfs font server and it seemed to have fixed the problem.

### Post by r00t4rd3d on 2011-01-07
Also you can install a app called Font Manager from the synaptic package manager. It puts the launcher in Applications , Graphics , Font Manager.

You can easily install / remove fonts. In Synaptic its called font-manager 0.5.6-1
