### Post by bigboy_pdb on 2007-10-10
If you're having problems with WINE, you might want to do some research on whether or not there are any virtual machines (such as VMWare Server) that will allow you to perform the tasks that you want to do. They'll allow you to install Windows and to run Windows programs on them, but some virtual machines may not allow you to interact with devices that you may need to interact with.

I tried to do some research for you, but I found LCDStudio's site uninformative. From what I saw it appears as though it allows you to write programs for LCD devices. Perhaps someone else will know more regarding this issue.
