A terminal is essentially the same thing as a command prompt in Windows, if that's something you're familiar with. 

That sudo apt-get install epiphany-browser command simply gets put into a terminal (which can be opened with Applications>Accessories>Terminal). Simply copy and paste that command into a terminal (you must right-click and select paste when in a terminal as control+v will not work), and after prompting you for a password it will install that program.

 I can't remember the procedure for setting a sudo password right now, but basically sudo gives you administrator privileges. No normal users can install programs without entering a sudo password. But if you enter that command, it prompts you for a sudo password...if you don't have one maybe someone else can help you with that since like I said I can't remember how to set it at the moment

But really, once you get used to Linux it's not that bad! I'm so used to the terminal now it's second nature to me.
