### Post by CHaoSlayeR on 2009-01-10
Hi there,

your problem reminds me of my own some time ago. The main thing that you did wrong is defining two device and screen sections for the fglrx.

You should only use one screen section and one device section. In the device section you should tell the fglrx driver that you want to use the second monitor:

```

	Option	    "EnableMonitor" "crt1,crt2" # this is just for my CRTs
	Option	    "PairModes" "1280x1024+1280x1024"

```

As for "crt1" and "crt2" you may have to use other values. Available ones are:

[LIST]
[*]none
[*]crt1
[*]crt2
[*]lvds
[*]tv
[*]tmds1
[*]tmds2
[*]auto
[/LIST]

Change the PairModes value as needed.

The only screen section will just be defined as usual for the first monitor section and the device section. Additionally you may have to add a virtual resolution to get your desired screen resolutions (default maximum virtual screen resolutions are often too small).

Additionally in the ServerLayout section get rid of the line for the second screen.
