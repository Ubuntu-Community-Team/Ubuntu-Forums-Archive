I don't think you need to reformat the disk.  QEMU usually uses a disk image as a virtual hard drive (a disk image is merely a special kind of file, that is as big or as small as you set it to be).  I would recommend reading the quick start instructions on the QEMU homepage.  note: the instructions you type in the terminal shouldn't be dependent if you run them from a windows computer or a linux computer, the main thing to look at is how to make and use a disc image.
[http://kidsquid.com/cgi-bin/moin.cgi/QuickStartGuide]("http://kidsquid.com/cgi-bin/moin.cgi/QuickStartGuide")
