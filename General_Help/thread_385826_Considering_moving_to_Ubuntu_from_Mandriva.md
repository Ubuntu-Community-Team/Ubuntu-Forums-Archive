### Post by laidback on 2007-03-16
Did exactly this move myself some months ago. I prefer Ubuntu. Firstly because all users get the same system, no power user subscription stuff. I find it smoother. Now that's a very subjective statement, but, it runs nicely, updates beautifully and altogether is a happy environment. That's not to say that Mandriva wasn't any good, it was, but I find Ubuntu better. The concept of Ubuntu doen't allow for power user distro etc, we all get the same. The software is all much the same, although no doubt versions may differ. I ran both for awhile, than found that I hadn't used Mandriva for several weeks, well that was it, off with Mandriva.
An aspect I really like is that I can buy books on Ubuntu in my local store or via the web, cannot do that so easily with Mandriva, can only buy from them.


Welcome to Ubuntu

### Post by wxnker on 2007-03-16
I'm trying out Mandriva these days and I must say that I find it very nice. I use the KDE version besides Kubuntu. A shame Metisse does not work with KDE yet. I'd like to try it.

There isn't really a big difference between the two so I'm quite sure you will like Ubuntu, Kubuntu etc. 

The biggest difference I've noticed so far:
Packages for Ubuntu are *.deb based and as far as I know *.rpm based for Mandriva. So I haven't figured out how to install from Konsole in Mandriva yet. :D

What I especially like about Kubuntu/Ubuntu is, the exact thing mentioned by "laidback". Everyone get's everything. The forum is really great too.

What I really like in Mandrive (only tried the KDE version) is:
- Nice install features (XGL & Compiz automatically installed during MAndriva installation if chosen, Networkmanager setup etc.) 
- Very clean menus and some very nice configuring options in the menu.

That orange thing and the Kmenu icon... Well I ditched those pretty quick :mrgreen: 

I bet Feisty will have most of these things within a short period though.

But I gotta say ... Mandriva 2007 is nice too.:mrgreen:

Try:
# urpmi program-name
That will then download and install all the relevant packages and dependencies for you.

Of course, you'll need to set up the repos first; use [http://easyurpmi.zarb.org](http://easyurpmi.zarb.org) for that.
