### Post by stimpsonjcat on 2006-07-04
Regarding battery life:
1. check if laptop-mode is enabled (sudo laptop_mode status).
2. check if CPU throttling is enabled. There's a CPU frequency monitor you can add to the panel.
3. disable wireless/bluetooth and other devices when not in use
