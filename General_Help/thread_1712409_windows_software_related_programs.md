### Post by Mark Phelps on 2011-03-23
You can't install Windows drivers in Ubuntu.  So, if you're trying to use a CD or other software that came with the web cam ... that simply won't work.

You can run SOME Windows programs in Ubuntu if you install Wine.  You should go to the WineHQ website and check out their compatibility section to see how well each particular app and version will work.

But, you can not use Wine to install Windows drivers.
