### Post by RoyLongbottom on 2010-11-28
[FONT=Verdana][SIZE=2]I have started porting my PC C/C++ benchmarks to run on Linux based PCs. The Windows versions can be obtained via:[/SIZE][/FONT]
 
[FONT=Verdana][[SIZE=2]http://www.roylongbottom.org.uk[/SIZE]]("http://www.roylongbottom.org.uk")[/FONT]
 
[FONT=Verdana][SIZE=2]The first ones are the Classic Benchmarks that set the standards for measuring relative performance of computers in the 1970s and 1980s. These are the Livermore Loops for supercomputers, Whetstone for general scientific systems, Dhrystone 1 and 2 for UNIX commercial applications and Linpack for workstations and, later, supercomputers. [/SIZE][/FONT]
[FONT=Verdana][SIZE=2]There are four compilations of each of the five benchmarks to cover optimised and non-optimised, 32-Bit and 64-Bit working. Shell scripts are provided to run 10 execution files under either 32-Bit or 64-Bit versions of Linux. The programs do not require installation. Download the following, extract all and run. Results are displayed and saved in .txt files. The package includes source code and compiling instructions.[/SIZE][/FONT]
 
[FONT=Verdana][[SIZE=2]http://www.roylongbottom.org.uk/classic_benchmarks.tar.gz[/SIZE]]("http://www.roylongbottom.org.uk/classic_benchmarks.tar.gz")[/FONT]
 
[FONT=Verdana][SIZE=2]Downloading the above to Windows (in my case) changes the extension to .tar.tar but the file can be renamed with .tar.gz for transferring to a Linux based PC. More detailed descriptions and results can be found in the following:[/SIZE][/FONT]
