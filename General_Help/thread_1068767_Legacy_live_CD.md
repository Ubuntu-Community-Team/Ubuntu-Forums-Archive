### Post by mb_webguy on 2009-02-13
If your goal is just to install Ubuntu, you can use the [alternate install cd]("http://www.ubuntu.com/getubuntu/downloadmirrors#alternate").  It uses a text-based installer, and will therefore run on computers when the LiveCD won't.
