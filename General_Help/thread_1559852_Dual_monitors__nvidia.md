I have noticed a bit of a strange happening in ubuntu. I'm not too worried about fixing it for myself but thought it could be useful for others. 
Just so you know where we are...
I have a Dell laptop (with a quadro 360m graphics card) with nvidia drivers. I have tried both the 256 release and now the 196(?) release - both behave the same. The OS is 64bit. 

I usually just use one screen (the laptop) but at work I plug in the external display. 
When I go to set it up, I must configure the screen twice every time to make sure that the laptop is configured as the main display, as otherwise the external display will be configured as the main display. 
As I said, I'm not too concerned about fixing the issue on my computer but if people weren't aware of the problem before then you will want to configure the set up twice. 
I use the NVIDIA settings manager to configure. 
I've tried to save the xorg.conf file but this just means that the next time you boot up without the screen attached you run into difficulties...
