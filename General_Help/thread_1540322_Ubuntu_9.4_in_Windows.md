### Post by jmszr on 2010-07-27
BSG Fan,

She can install using Wubi: [http://wubi-installer.org/faq.php](http://wubi-installer.org/faq.php) which can be downloaded from here: [http://wubi-installer.org/](http://wubi-installer.org/) .

If Word doesn't read .odt then she can install the Abiword word processor in Windows (Abiword is free): [http://www.abisource.com](http://www.abisource.com) which can read .odt and save the document which can then be printed by Windows. (I am unaware if it will print directly from Abiword.)

Wubi is removed through Windows Add/Remove Programs.

Edit: Sorry for the changes-it has been so long since I used Windows that I have a hard time remembering how things interact with it.
