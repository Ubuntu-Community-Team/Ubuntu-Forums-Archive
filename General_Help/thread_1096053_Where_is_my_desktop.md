### Post by taurus on 2009-03-14
Open a terminal and post the outputs of these commands.

```
sudo fdisk -l
sudo blkid
cat /etc/fstab
df -h
```

p.s.  If you resized your /home and you use UUID in /etc/fstab, the UUID for /home could change so you probably need to use the new one.

### Post by drs305 on 2009-03-14
If the desktop is completely blank it may be "turned off". Try running this command to "turn it back on":
```

gconftool-2 --type bool --set /apps/nautilus/preferences/show_desktop 'true'

```
