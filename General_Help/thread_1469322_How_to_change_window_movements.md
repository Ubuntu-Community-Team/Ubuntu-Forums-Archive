### Post by SnickerSnack on 2010-05-02
If you're using Gnome, then

1) Install Compiz and CompizConfig Settings Manager.
2) Then run System > Preferences > CompizConfig Settings Manager.
3) Click the checkboxes next to "Animations" and "Animations Add-On".
4) Look in the settings for "Animations".
5) Go to the "Minimize Animation" tab.

There are several animations available, but I think the one you want is called "Magic Lamp".

Otherwise, :confused:
