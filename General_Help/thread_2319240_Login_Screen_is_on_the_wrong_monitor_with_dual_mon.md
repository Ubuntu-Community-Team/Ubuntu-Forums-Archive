I was never able to fix a similar problem in both Windows and Linux on my Nvidia GTX750Ti graphics card because whatever topmost port you use on that card was always assigned as the main display, any connected below that were assigned as secondary. My workaround was installing an adapter in the topmost output, then that became my main display.
