### Post by Darkade on 2008-07-08
1) Enable restricted drivers(codecs)
2) Configuring evolution to easy access his e-mail, if he uses msn or hotmail, use webmail add-on, [here]("http://webmail.mozdev.org/")
3) Link to home folder on desktop. Encourage him to keep things ordered in the suggested way (by the default folders)
4) If he uses MSN Messenger, install aMSN.It's more familiar/friendly to new users
5) Enable DVD playing, and install VLC
6) Configure dictionaries in OpenOffice
