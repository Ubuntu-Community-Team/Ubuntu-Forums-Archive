### Post by imjafo on 2009-09-02
Try getting NVClock GTK from synaptic. I use it to set my Nvidia fan to 100% duty cycle. It doesn't change the speed, just the duration of time the fan is on. I tried NVclock QT, but it doesn't have all the controls that gtk has. You can open it from terminal by typing "nvclock_gtk", without the quote marks. It will open a gui that will allow adjustment of the fan as well as giving you the option to overclock your GPU, should you desire to do so. I don't know how to make the setting for the fan permanent, yet, so I set it to 100% after I boot the computer and then hit the apply button. There are several adjustments available on it that can enhance you viewing pleasure as well. It has made a difference on my 9400GT temps by about 5C when it is set to 100%.
 Hope this helps.
