### Post by Ratscallion on 2008-12-29
First of all, what version of Ubuntu are you using, if it's Ubuntu 8.04 and above try this:
Open Synaptic Package Manager, System-> Administration->Synaptic Package Manager.

Search 'compiz'
Check all the ones with a description along the lines of:
*Compositing Manager*
Install those packages.
When you go into System->Preferences->Appearance and go to the Visual Effects tab you should now be able to click Extra or Normal.
Any other problems, please ask.
