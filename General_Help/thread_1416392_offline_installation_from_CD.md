### Post by darolu on 2010-02-26
If you are going to install in a computer that has no internet access, I would recommend downloading the Ubuntu **DVD** it comes with all the packages, you can download it from this link: [http://www.ubuntu.com/getubuntu/downloadmirrors#dvd](http://www.ubuntu.com/getubuntu/downloadmirrors#dvd)

You install the packages normally, either with Software Centre, Synaptic or from the command-line with apt-get or aptitude, when you install you'll have to add the DVD to your software sources and that's about it.
If you need other software (i.e. proprietary software), download it and transfer it on a USB flash memory (pen-drive) or CD.
