### Post by jonian_g on 2008-08-28
I think you have to enable xinerama.

"Xinerama is an extension to the X Window System which enables multi-headed X: applications and window managers to use two or more physical displays as one large virtual display."

[http://en.wikipedia.org/wiki/Xinerama](http://en.wikipedia.org/wiki/Xinerama)

### Post by kavon89 on 2008-08-28
Seems when I rebooted Xinerama was set but not saved in the xorg.conf. I went in an enabled it and all is good now. :D thank you
