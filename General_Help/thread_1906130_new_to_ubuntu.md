### Post by Mark Phelps on 2012-01-11
Basically, you're in Linux now, not in Windows -- so you don't install from CD-ROMs.  Those contain Windows software and drivers -- and you can't use those drivers in Linux.

As to games, if you're asking about Windows games -- I should tell you NOW that, if that is a key reason for using Linux, then you need to rethink your choice.

Linux has a facility known as Wine that will run SOME Windows apps and games -- but just SOME. A lot of the newer stuff (IE 9, MS Office 2010) simply will not work.

You should go to the winehq website and look through their compatibility database for the games you want to run.  Anything not rated, or with a rating lower than Gold, is going to be a waste of your time.

Linux is not a free VERSION of Windows, it's a free ALTERNATIVE -- meaning, you will have to learn to do some things differently, and you will need to move away from Windows.
