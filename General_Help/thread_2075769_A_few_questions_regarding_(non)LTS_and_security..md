### Post by snowpine on 2012-10-24
1. "Support" means actual software support, as in, when 12.10 hits its "end of life" there will be no more security updates, no more bug fixes, and the repositories are closed so you can't install more apps. Presumably that won't matter to most 12.10 users as they will have moved on to 13.04 (and then 13.10, 14.04, etc.) by that point. LTS is for users who prefer 24-month instead of the 6-month upgrade cycle.

2. Lots of good security info here: [http://ubuntuforums.org/showthread.php?t=510812](http://ubuntuforums.org/showthread.php?t=510812)

3. Good intro to installing software in Ubuntu here: [http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)

### Post by grahammechanical on 2012-10-24
Ubuntu uses the Gnome desktop environment and that has a six months release schedule and so Ubuntu also has a six months release schedule. But businesses like something more long lasting so the Long Term Support version was invented.

[https://wiki.ubuntu.com/LTS]("https://wiki.ubuntu.com/LTS")

Ubuntu is distributed without charge. So, if you require what you call 'technical support' you can pay for it.

[http://www.ubuntu.com/business/advantage]("http://www.ubuntu.com/business/advantage")

Otherwise, its is support from forums like this one which is a users community forum.

So, the choice is to stay with 12.04 LTS for most of the next five years or move from release to release every six months or every eighteen menoths. Either way we still get the same amount of support.
