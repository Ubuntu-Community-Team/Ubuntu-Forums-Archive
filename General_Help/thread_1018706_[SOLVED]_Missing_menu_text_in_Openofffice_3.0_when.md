### Post by j.marinus.dejong on 2008-12-22
After installing Ubuntu 8.10 and Openoffice 3.0, I activated advanced desktop effects with compiz. This resulted in missing text in de menus in Openoffice. The text flashes shortly when the mouse scrolls over the menu, and then disappears, only the underscores remain. When I use Metacity instead of Compiz the problem is solved. But I don't want to miss the pretty rotating desktopcube, and other nice compiz effects. Does anyone have a solution to this conflict?

---

### Post by gettinoriginal on 2008-12-22
System > Pref > Appearance > Fonts    set rendering to "subpixel smoothing"  then go to details   set smoothing to "subpixel" and hinting to full.  :P
