### Post by deanhopkins on 2009-08-01
Click on the Down arrow left of the search box (next to the search icon). Then from the menu select Manage Search Engines. Now choose Restore Defaults, or if that is unavailable choose Get More Search Engines.

### Post by VCoolio on 2009-08-01

You can copypaste them to to any other firefox or swiftweasel or whatever firefox version you're using.
