### Post by Adam_GUI on 2020-06-13
A live DVD wasn't my first choice.
The motherboard, itself, sees the mouse and keyboard just fine.
Motherboard is a Gigabyte GA-970A-UD3.

I had attempted a USB live disk and was consistently getting a "no mountable file system found" message.
The Live DVD boots straight to desktop for "try xUbuntu."

SOLVED, but still posting in case someone comes behind me with the same issue.
In the BIOS menu, change the option from "Other OS" to Windows 8.
From here, the USB Live Disk is seen just fine and the keyboard & mouse work in the live desktop.
