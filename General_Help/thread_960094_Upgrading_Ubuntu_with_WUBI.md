### Post by lisati on 2008-10-27
I'm currently checking out the 8.10 release candidate which I installed via Wubi - this is my first time using Wubi, so I'm not clued up on all the ins and outs of using Wubi,

Having said that, I have a suggestion: If you want to update without making a fresh install, you might want to wait a few days for the "official" release. Then open up the update manager from the System->Administration menu, check for updates, and see if it offers to upgrade your system.
