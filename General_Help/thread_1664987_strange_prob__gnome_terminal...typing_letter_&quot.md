### Post by wamanning on 2011-01-11
running 10.04.  for months.  havent had any problems like this before.

started terminal, needed to do some stuff, and everytime i type the letter "k" (lowercase), terminal launches a the gnome terminal help/contents instead.  if i type a "K" (capital, via shift) it works fine.  

just the lowercase "k", each keypress launches a new window.  capslock doesnt help, it still happens.

strange.

any ideas?  ive googled to no end, and can find nothing.

### Post by features on 2011-01-11
I imagine that could be quite distracting :D

First thing to try is to see if the keyboard shortcut for the help has been changed.  If you open a terminal, then go Edit-->Keyboard Shortcuts... you will be able to view and change the shortcuts for the terminal.  If the shortcut for "Help Contents"  is "k" then you've found the culprit.

I'm not sure what the problem would be if that doesn't fix it - it's only happening in terminal, right?
