In the manual I see the setting you are after along with a stern warning:

```

3.262. sig_on_top

Type: boolean
Default: no

If set, the signature will be included before any quoted or forwarded text. 
It is strongly recommended that you do not set this variable unless you really 
know what you are doing, and are prepared to take some heat from netiquette 
guardians. 

```

Just be careful with the netiquette guardians :-)
