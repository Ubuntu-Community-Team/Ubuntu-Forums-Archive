### Post by prshah on 2011-06-20
> **tej1984 said:**
> Is there a way to set ubuntu  to swtich on and play music like an Alarm clock?

If your BIOS has RTC wake support, then you can set a time for the computer to start up.

You can then put Exaile (Music Player) in your startup apps, and use the "Alarm Clock" plugin to start music at a preset time.
