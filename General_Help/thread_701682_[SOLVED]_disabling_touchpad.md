### Post by strabes on 2008-02-19
Please paste the output of this command:```
cat /etc/X11/xorg.conf
```

If you want to disable the tapping, you can simply add the following line to the Synaptics Touchpad Section of /etc/X11/xorg.conf:```
 	Option	    "MaxTapTime" "0"
```

If you want to disable it completely you could disable it in your BIOS or add the following lines to /etc/X11/xorg.conf to set the speed to 0:```
 	Option	    "MinSpeed" "0"
	Option	    "MaxSpeed" "0"
```

### Post by erfahren on 2008-02-19
to use GSynaptics (there's also QSynaptics - you might want to try that) you just have to insert the line:
```

Option          "SHMConfig"             "on"

```
into the Synaptics section of your xorg.conf

so it looks like
```

Section "InputDevice"
	Identifier	"Synaptics Touchpad"
	Driver		"synaptics"
	Option		"SendCoreEvents"	"true"
	Option		"Device"		"/dev/psaux"
	Option		"Protocol"		"auto-dev"
	Option		"HorizScrollDelta"	"0"
	Option		"MaxTapTime"		"0"
        Option          "SHMConfig"             "on"
EndSection

```
(what it does is enables the use of shared memory for additional configuration settings for the touchpad if needed)

there's more on the [Synaptics man page](http://linux.die.net/man/5/synaptics)
