### Post by SunnyRabbiera on 2008-08-12
Well one issue I have encountered in getting it running is you need the official java an not the one pre installed with ubuntu.
Open JDK really doesnt work that well (though its still new)
I suggest removing open jdk and installing the regular sun java packages, there is an application called synaptic package manager located under system > administration > synaptic package manager.
After that opens and you enter your password and then go up to "settings" and to "repositories" and click off all the boxes in the first and second tabs except the source code ones.
After that search for Java runtime and remove open JDK, if you are unsure use this guide:
[http://monkeyblog.org/ubuntu/installing/](http://monkeyblog.org/ubuntu/installing/)
