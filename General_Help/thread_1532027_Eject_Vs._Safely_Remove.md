### Post by techunit on 2010-07-16
Eject generally defines medium that can't be written to in the same ways as other media. Thus the media can't be effected if removed prematurely. Example is a CDR

 Safely remove refers to things that can be affected if removed prematurely. Example would be a Harddrive.
