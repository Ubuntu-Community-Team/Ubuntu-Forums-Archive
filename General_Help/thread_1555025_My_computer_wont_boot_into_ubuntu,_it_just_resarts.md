### Post by apoorvumang on 2010-08-17
The exact same thing happened with me (but with a different graphics card). However, I was able to boot into Ubuntu this way.

1. Open grub menu on boot (long press shift key)
2. Select Ubuntu recovery mode
3. Select failsafeX. After that, it will probably say that X could not get graphics settings. Choose "Run Ubuntu in low graphics mode this once".

That should work (I'm a complete newbie, sorry if it doesn't)
