### Post by WorMzy on 2010-10-16
[Here]("http://archive.ubuntu.com/").

You can download source code on a per-app basis with apt-get.
```
apt-get source <appname>
```

You may need to enable source code repos under System -> Administration -> Software Sources before that command will work.
