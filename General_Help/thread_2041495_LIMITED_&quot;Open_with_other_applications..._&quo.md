### Post by ajgreeny on 2012-08-12
You're not the first to notice that limitation, but it is not actually Ubuntu's fault, but an upstream gnome 3 problem in nautilus.

See [http://ubuntuforums.org/showthread.php?t=2037497](http://ubuntuforums.org/showthread.php?t=2037497) for more discussion of this, but unfortunately not real answers other than using thunar, the XFCE file manager, which still has this ability, and runs very well on gnome 3 desktops.
