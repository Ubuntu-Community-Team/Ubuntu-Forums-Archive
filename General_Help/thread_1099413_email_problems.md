### Post by pmlxuser on 2009-03-18
Please note that the default file saving extension of Open Officece is not *.doc / *.xls its *.odt

Therefore if you wnat to send document to windows friends make sure that on saving the document you cange the file extension to appropriate *.doc / *.xls in the save as dialog box.

### Post by pmlxuser on 2009-03-18
Basically if one is using outlook, the probability is they are using MS Office, so the appropriate extensions would be *.Doc (specifially Open office specifies as word for office xp/2003) for word documents, and *xls for excel sheets

### Post by LiamWilson on 2009-03-18
Yeah, it's not that anything is broken, it's that files saved with OpenOffice save as .odt or similar. What you need to do is save it as an Microsoft office format such as .Doc, explained above, or convince the reciever of the document to download OpenOffice for Windows.
