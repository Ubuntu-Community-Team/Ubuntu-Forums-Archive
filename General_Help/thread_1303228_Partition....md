### Post by psycho5 on 2009-10-28
If you want to partition a hard drive that Ubuntu isn't running of off, then you can do it from System>Admin>Gparted .

If you want to create a partition in your hard drive that contains Ubuntu, then you have to do it by either using the Ubuntu live cd, or the Gparted live cd. You'll have to shrink your Ubuntu installation to a smaller size to create the empty space, then use that empty space to create your partition.
