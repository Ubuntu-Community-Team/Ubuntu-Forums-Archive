### Post by forlaf on 2008-05-06
This isn't a big deal, but does anyone know how to change the way gnome positions the window when an application opens?

When I open Firefox, for example, it opens in the upper left corner of the screen. When I move it to the center of the screen, where I want it, it tends to snap to either the top or bottom. Is there some configuration tweak I can make somewhere that will open application windows at screen center? Can I get rid of the snap?

---

### Post by erginemr on 2008-05-06
To disable snapping, assuming that you have enabled Compiz; you should install **compizconfig-settings-manager** from Synaptic, and then open "Advanced Desktop Effects Settings" from Main Menu -> System -> Preferences and disable the "Snapping Windows" plugin.

To pre-determine the location of windows, you can try **devilspie**:
[http://ubuntuforums.org/showpost.php?p=4668528&postcount=6](http://ubuntuforums.org/showpost.php?p=4668528&postcount=6)
