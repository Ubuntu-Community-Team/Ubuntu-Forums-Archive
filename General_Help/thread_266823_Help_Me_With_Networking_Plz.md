### Post by Jussi Kukkonen on 2006-09-28
For file sharing, you need Samba. You probably want to setup a samba server on ubuntu, and access that from windows. I believe the wiki has some instructions on this, but the simplest way to start is just to right-click on a folder in Nautilus and select "share" -- there should be some kind of wizard there to help you.
