I had setup sevreal VPN connects using the network manager and everything was working. Recently network manager no longer has a 'Disconnect VPN' under the VPN Connections menu. ### Post by bearlake on 2017-03-09
I noticed that problem myself as a few others from the same update.

If you click on the vpn with line that you used to select, it will close the vpn connection that you are currently using.

Not a fix but a work around until they fix their mistake.
