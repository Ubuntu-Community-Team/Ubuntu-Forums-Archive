### Post by credobyte on 2009-07-28
When you enter a password in terminal, it's hidden ( you don't see what you are typing ) because of the fact that someone could possibly see you typing it in. Since terminal does not have * ( or dots ) as in HTML ( www ), it simply shows nothing.
