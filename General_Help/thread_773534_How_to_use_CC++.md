### Post by amohanty on 2008-04-29
1. sudo apt-get install build-essential
2. write your code emacs, vi, anjuta, etc.
3. for c:
gcc file.c 
for c++
g++ file.cpp
g++ is just a wrapper for gcc with options to identify cpp files.
The gcc manual is here:
[http://gcc.gnu.org/onlinedocs/gcc-4.3.0/gcc/](http://gcc.gnu.org/onlinedocs/gcc-4.3.0/gcc/)
