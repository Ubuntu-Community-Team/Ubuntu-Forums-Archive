### Post by grahammechanical on 2014-11-08
Do you still have a Documents folder under /home folder? Note the capital D. If you do then copy and paste all the documents in the Downloads/documents folder into Documents and once you have confirmed that they have really transferred over then delete the documents in the Downloads/documents folder to empty the folder and then delete the folder.

An installation of Ubuntu will create the following folders or directories, if you like,

home and in that a folder with our user name (called Home in the file Manager) and in that folder there will be Documents; Downloads; Music; Pictures; Videos. This is why I ask if you still have a Documents folder. Perhaps you have Cut and Pasted the Documents folder into the Downloads folder and now you must Cut and Paste it back into what the File Manager calls Home.

But I advise that you use Copy and Paste and then empty and delete. Just to make sure you do not delete all your documents.

Regards.
