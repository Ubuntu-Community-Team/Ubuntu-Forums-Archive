### Post by IllegalCharacter on 2010-04-30
It's possible you have distribution upgrades disabled. To check:
1) Go to System->Administration->Software Sources
2) Go to the Updates tab
3) At the bottom see if the dropdown for "Release upgrade" is not set to Never.

Otherwise you can try reloading the software sources by clicking "Reload" under Synaptic.
