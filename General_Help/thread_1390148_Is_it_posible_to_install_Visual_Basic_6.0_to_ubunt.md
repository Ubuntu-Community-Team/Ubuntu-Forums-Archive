### Post by nullmind on 2010-01-25
Visual Basic is GOLD status and should work fine ([http://appdb.winehq.org/objectManager.php?sClass=version&iId=130](http://appdb.winehq.org/objectManager.php?sClass=version&iId=130)) There have been other implementations of VB6, but if you're learning VB6 for school simply learn it and move on.

While I love Wine, in this case you can't afford to waste time if something weird occurs. Install VirtualBox and run VB6 in that environment will be the most stable.

If you have room for learning, etc. give Wine a shot.
