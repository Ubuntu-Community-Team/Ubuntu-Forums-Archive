How to shutdown Skype completely?  The one way that I've found is to use "System Monitor" and "End Process;" such is quite inconvenient. ### Post by gifford on 2013-07-17
Does a Skype icon appear in the panel on the upper right?
If so, right clicking on the icon will bring down a menu and just click on quit. It will shutdown and close the application.

### Post by monkeybrain2012 on 2013-07-17
It is in 12.04. If somehow it doesn't show up, open dconf-editor (install dconf-tools if not installed already), go to desktop > unity > panel and add 'skype' to the systray-whitelist  (or simply change the entry to 'all') See screenshot.
