### Post by ubnoobie on 2006-06-02
MD5SUMS should be in every release download directory... first file listed, above the torrents, iso's, etc...

md5's for:
ubuntu: [http://releases.ubuntu.com/6.06/MD5SUMS](http://releases.ubuntu.com/6.06/MD5SUMS)
kubuntu: [http://releases.ubuntu.com/kubuntu/6.06/MD5SUMS](http://releases.ubuntu.com/kubuntu/6.06/MD5SUMS)
xubuntu: [http://cdimage.ubuntu.com/xubuntu/releases/dapper/release/MD5SUMS](http://cdimage.ubuntu.com/xubuntu/releases/dapper/release/MD5SUMS)
