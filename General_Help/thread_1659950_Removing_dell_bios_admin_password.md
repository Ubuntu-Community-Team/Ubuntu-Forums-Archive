### Post by jcolyn on 2011-01-05
> **jsvidyad said:**
> Hello, 

   I set an admin password for the bios of my dell inspiron e1505 laptop. I do know the password. Is there some way for me to delete the admin password so that I can access the BIOS settings without a password?

Enter the password go into the bios to where you set the password click disable, leave the entries blank then exit and save. There should be instructions in that part telling you how to disable.

However for security you should leave the password enabled and also enter a boot password. If your laptop gets stolen pawnshops and resale shops won't buy it if they can't get into it. In fact most pawnshops will notify the police if the seller cannot give the password. That's how I got my stolen Dell back..the next day...
