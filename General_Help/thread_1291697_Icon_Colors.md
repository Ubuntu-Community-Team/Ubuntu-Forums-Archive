### Post by leonardo_neo on 2009-10-15
You could try few things.

Well, I am not sure how to change the color of default 'human' icon theme, but I could suggest you few other ways, which could give you different color icons.

1. You could change the default icon theme.

System----> Preferences----> Appearance-----> Customize----->icons

Then select whatever you like.

2. You can install few different icon themes. Go to 

[http://www.gnome-look.org/]("http://www.gnome-look.org/")

and download whatever icon theme you like.

### Post by nvikram on 2009-10-15
If you like the icon design, but just want to change the color, you can download the Humanity icon pack (its a tar.gz file). Extract it and you should get access to all of the icons. There if you want you can use a program, like InkScape, to edit them. Of course this is a very tedious thing to do and could take a LONG time. But, its just something to keep in mind.

Hope this helps :)
