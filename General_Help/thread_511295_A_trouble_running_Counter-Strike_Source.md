### Post by rdavis on 2007-07-27
Oddly enough, mine did the same thing and this worked:

First, move the mouse cursor towards the bottom-left of the screen, where the menu should be. If you can hear the mouse cursor switching between options, press CTRL-ALT-RIGHT ARROW to move to another workspace and then move it back. For some reason this displayed the menu and the game.

On the other hand, once I try to join a server and it begins to download their sound and mod packages, the game hangs and I have to reboot. Does anyone have a fix for this, or figure what it could be?  I'm running the latest Wine and ATI drivers and, like I said, the game loads fine up to that point.
