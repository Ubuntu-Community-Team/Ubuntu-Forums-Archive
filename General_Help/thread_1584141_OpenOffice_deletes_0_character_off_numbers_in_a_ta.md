### Post by themusicalduck on 2010-09-28
OpenOffice has a rather annoying feature that deletes the 0 character, if it is the first character, when I type a number into a table on a form in Writer. This means I can't type in any phone numbers into it as the 0 just gets deleted.

It's not such a big deal, but I've had this problem a few times with forms that are sent to me in .doc format (if that makes a difference being in .doc) and it'd be nice to stop it doing it.

Anyway to turn it off? Unchecking Format > AutoCorrect > While Typing doesn't help.

---

### Post by WorMzy on 2010-09-28
Highlight the column/row/entire table, right-click, choose "number format", and in the category choose "text". You'll need to re-add any already removed zeroes, but any future numbers beginning with a 0 in that column/row/entire table should remain.
