### Post by SecretCode on 2011-09-22
One thing that might help: install (if not already) compizconfig-settings-manager (that's the package name) and run it (ccsm is the command).

Enable the "Place windows" plugin and use the "windows with fixed viewport" to get LibreOffice back onto workspace 1.

### Post by dre12b on 2011-11-05
I had been cursing this bug, but trying to ignore it for the last couple of months.  After reading this thread I found that "Place Windows" plug-in was enabled.  I disabled the plugin and the situation appears to be fixed.  Viola!
