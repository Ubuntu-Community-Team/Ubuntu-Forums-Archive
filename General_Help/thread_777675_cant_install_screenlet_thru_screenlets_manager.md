### Post by spdf on 2008-05-02
Did you install the screenlet? This involves opening the screenlet manager, pressing the Install button, and selecting the screenlet archive.

Once the screenlet package is installed, you should be able to start it from the screenlet manager and specify whether or not it starts when you login.

### Post by halln on 2008-05-02
You have to open Home/Hidden files/ then look for .screenlets folder, (if it's not there, make one then rename it with .screenlets) drop your downloaded screenlets file there, right click the file, extract file then close the folder. Open your screenlets file manager and you should find it there.
