### Post by hovzio on 2008-07-19
hi, you first need to install compizconfig-settings-manager. This program allows you to play around with all the eyecandy and so forth.
```

sudo apt-get install compizconfig-settings-manager
```

When you have it install press alt f2 and enter ccsm. This will startup the the settings manager with which you can configure compiz.
