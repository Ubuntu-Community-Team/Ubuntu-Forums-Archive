### Post by SnickerSnack on 2010-04-03
One difference is that xfce is a compositing window manager, so there are some shiny features you get with it.  Whereas gnome needs compiz in order to have some shiny things (but compiz offers much more than xfce does).

I knew I wanted to use the Avant Window Navigator, which requires compositing, so I have both gnome and xfce4 installed.  I mostly prefer gnome since compiz has several very useful features.  I'm using a tablet pc with an intel x3100, so the video power is quite limited (so I can only use some of compiz' features).

Xfce4 is made to be minimalist, so it appeals to some people who prefer a minimalist look and to those with older hardware.  Unless you have a very low end video card, you will probably be happier with gnome+compiz, but I recommend installing both gnome+compiz and xfce4 and trying them out.  It's easy to switch between the two.

Also, I found that on my hardware, many graphical games to not work well (or at all) while compiz is running.  Again, this is only a problem if you want some visual bells/whistles but have low end video hardware.
