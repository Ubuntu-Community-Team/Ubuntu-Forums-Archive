### Post by Jixxon on 2007-07-16
I have only heard of this when a laptop is overheating. A quick search of the forums produced the same info. If everything is working fine in XP it certainly doesn't seem like this would be the problem through. I also don't think Ubuntu works the CPU as hard as XP does (just a personal thought - no data to back it up). 

Just for kicks a good quick test would be to use some kind of spacers under the laptop to improve airflow and see if that does anything.

There was also some mention of closing the lid of a laptop and a shutdown occurring, but that is obviously an easy change to the power settings. If you were working on email and it just shutdown then this doesn't relate at all.

If it is not an airflow issue and if is is not a power management issue, a last ditch effort would be to reinstall Ubuntu and see if it was an installation error. Good luck.
