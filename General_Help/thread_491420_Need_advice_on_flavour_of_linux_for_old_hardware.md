### Post by MCSE_Crossover on 2007-07-03
The Xubuntu website states that it can run on 64MB of RAM at a minimum but recommends 128MB. 

Deli Linux will run on 50MB HD with 8MB of RAM - as far as I found for old systems. Deli looks like the best option in this case. Xubuntu may be a little resource intensive, even in it's dumbed down state.

### Post by donkyhotay on 2007-07-03
There's always DSL (damnsmalllinux) which can go down to 16mb (though 128mb is recommended). I've also heard good things about puppy linux for older systems which only requires 32mb of RAM.
