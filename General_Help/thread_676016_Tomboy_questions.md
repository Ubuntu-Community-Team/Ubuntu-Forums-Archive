### Post by y-lee on 2008-01-23
I use TomBoy to keep quick notes that i can  easily access. So lets see 

> [1] Is it possible to import notes into Tomboy? 

Tomboy doesn't support that as far as I know. Not the version I have anyway. However tomboy keeps all its notes in XML format in the **/home/username/.tomboy** directory. SO it would be possible if you could convert your notes to the same format. I don't know of any tools to do that tho. Probably would not be hard to write a script to do tho if your notes were in a easy to read format.

> [2] Is it possible have to multiple files of Tomboy notes? 

I'm not sure what you are asking here. I have about 160 notes now. each as a separate XML file, so yes if that is what you mean. Since Tomboy lets notes link to each other it is easy to create a simple wiki like structure that might satisfy your needs.

I however have been consider switching to online notes, such as [http://www.zotero.org/](http://www.zotero.org/) . I already use Google notes but find it too limited.
