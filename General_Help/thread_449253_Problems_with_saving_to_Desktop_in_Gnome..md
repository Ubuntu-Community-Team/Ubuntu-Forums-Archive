### Post by Kilowatt on 2007-05-20
Hello, I am pretty new to Ubuntu. I'm using Feisty 7.04. 
I'm having problems saving files to the Desktop folder in almost every program that I've used. When I click on the Desktop folder to save a file in Thunderbird, Firefox, Evolution, and others, the program hangs and I have to "force quit" to exit it. Saving to other folders like the Home folder for example, seems to be ok.

I have no idea how to fix this.
Any help with this would be greatly appreciated!

---

### Post by Kilowatt on 2007-05-20
Problem solved.
Apparently the Gnome file saving window/applet doesn't like when an icon uses images that are large. I was using a large image for my Firefox icon on the desktop. This explains the freezing. :D
