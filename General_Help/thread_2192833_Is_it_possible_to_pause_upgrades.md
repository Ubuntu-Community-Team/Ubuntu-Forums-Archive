### Post by deadflowr on 2013-12-09
I believe you can only cancel it.
Which at that early of a stage shouldn't be too bad, as the packages have yet started to be installed.
However, I don't know what effect it will have if you want to restart, lets' say, tomorrow.
Most likely, it will start all over again, hopefully the packages it has already grabbed will still be there and the packages it needs will be added to it.
I'd say worst case would be you needing to clean the cache and start all over.

But upgrades from version to version always take much longer to run.
The system is updating every single package on it, including all those you installed.
It's also important to read what it says when you start the upgrade right before the final upgrade button is pushed.
An output on the last screen before it starts gives you info on how much needs to download and based on the network speed it had just run, how long it will take to do so.
Network speeds can go up or down but the general speed it lists is pretty good overall at what the speed will be, give or take byte a second or two(or a little more) This of course is also dependent of not having networking issues in the mean time.
