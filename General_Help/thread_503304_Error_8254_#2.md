### Post by jespdj on 2007-07-17
The error message suggests what to do:

**Boot with apic = Debug then try booting with the "Noapic" option and send a report**

Are you booting from the live CD? Then do this: When you see the boot menu from the CD, press **F6** (Other Options). Type:

apic=debug

And press Enter. Linux will now boot. Maybe you'll now see more elaborate messages. If it doesn't work, do the same but now type:

noapic

And press Enter. Does it work?
