### Post by mb_webguy on 2008-08-08
GTK-Gnutella is another Gnutella client like Limewire.  You can install it by typing "sudo apt-get install gtk-gnutella" in the terminal.  I would suggest it instead of Limewire or Frostwire if only because it's not written in Java, and Java apps always seem a bit slower to me.

I also suggest Deluge for a torrent client, but I suggest getting it from the [Deluge website]("http://deluge-torrent.org/downloads.php"), since the version in the Ubuntu repositories is quite a bit out of date.
