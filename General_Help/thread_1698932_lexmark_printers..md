### Post by HermanAB on 2011-03-03
Howdy,

If your printer is not in the Openprinting database, then fuhgeddaboudit: [http://www.linuxfoundation.org/collaborate/workgroups/openprinting](http://www.linuxfoundation.org/collaborate/workgroups/openprinting)

Either sell the offending printer on Ebay and buy a HP printer, or create a Windows virtual machine and install the printer on that.
