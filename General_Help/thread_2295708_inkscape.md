### Post by Lars Noodén on 2015-09-21
In general, PDF is a terminal stage format for your data.  Once your data is in PDF, you're done with it and there is no more editing.  It either goes to the recycle bin via the printer after a wait on the desk or the bit bucket directly after a wait on the disk.   The format for a live document would be SVG.  

So if you are working with a document, see if you can track down the original that was used to create PDF and use the original.

### Post by ajgreeny on 2015-09-21
However, if you have a PDF that you want to edit in some way, or perhaps add notes to it, have a look at xournal which will open all PDF files and you can then annotate it and save again (or export, I can't rememeber which) as a PDF.

I have used it several times when there was no option but that, and it is good, as far as it goes.
