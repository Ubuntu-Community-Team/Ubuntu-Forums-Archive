### Post by El Zoido on 2010-03-02
Depending on what you are connecting to, there may be an additional advantage of a router:

Some methods of connecting to broadband seem to offer only limited IP-addresses per connection (e.g. two).
That means you can easily connect 2 different pc's over a switch, but with more you would run into problems.
A router, if I understand this correctly, has it's own IP and is only showing this one IP to the ISP.

In my case for example I use broadband provided by a tv-company over cable and using a switch I can in fact connect two different pc's with the modem. If I then swap one of the pc's for a third, the new one cannot connect to the internet without powering the modem down first to reinitalize the IP-addresses.
So I think a router would solve this problem, but as I'm a cheapskate and only need two connections most of the time this is fine for me...
