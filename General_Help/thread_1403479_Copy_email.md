### Post by hemimaniac on 2010-02-10
1- For evolution mail try > File > Backup Settings (this will include mail in the default storage location)

2- The resulting .tar.gz file can be placed anywhere, disc, extHDD, etc etc

3- try right clicking and select write to disc, you may also need to update brasero or try k3b (in data project mode) for your back-up-to-disks

It's four steps ... I'm not sure where you are having issues? 
1.) In Evolution: Click File > Back up settings...
2.) Pick the location to save to, make a name for it (or leave the default name).
3.) In Ubunut: click applications menu > Click Sound and Video > Click Brasero Disc burner
4.) In Brasero Disc Burner: Click New Data Project > Drag your tar.gz file in the Brasero window, put in a blank disc and click burn. 


If that doesn't work, let us know where it fails, post specific error messages. The forums is a great place to receive help and lots of people are willing to assist.
