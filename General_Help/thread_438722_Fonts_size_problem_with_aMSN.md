### Post by useResa on 2007-05-10
I don't know which version of aMSN you are using. I am using 0.97b.
To change the font choose Account --> Preferences.

In the Personal tab you can change the font of the messages.
In the Appearance tab you can change the font of the interface (in the top part "My aMSN Messenger look).

Good luck
