### Post by jameswallace on 2008-12-23
I was having trouble getting Evolution to work with the Exchange server at the office.  My co worker had it running with no issues.  I had all the setting in exactly like his, but Evolution would ask for my password repeatedly and then give a message stating that Exchange was offline.  I checked and re checked and everything was setup correctly.  I wiped out Evolution and all its files and did a clean re install.

About five minutes ago, my co worker threw a few ideas at me.  The last idea solved the problem.

Here is what I was going to setup Exchange in Evoltuion.

1. Ran the wizard
2. Typed in the https for the OWA
3. Typed in my user name
4. Typed in my mailbox name
5. Authenticate
6. Went on to the next step
7. Pulled out hair 

Here is what I did that worked.

1. Ran wizard
2. Typed in https for the OWA
3.Typed in the user name
**IMPORTANT**
4. Don't type in your mail box.  Click on authenticate and let Evolution create the mailbox name.
5. Went to next steps.
6. Wondered how I could be such an idiot...

This doesn't appear to be a Ubuntu issues, but I have seen countless threads online about this same issue and frustrations. It seems that Evolution is missing something if you input your mailbox manually.


I hope this information helps others who have experienced this issue.  Thanks to Kevin for all his help.
