### Post by NoaHall on 2009-09-06
Yes, you need the 64 bit to get the full potential. Personally, I think if you can run 64 bit, you should run it, because it speeds up your system a lot (and not just if you have more than 3.4 GB of RAM)

To get graphics cards driver 
System -> Admin -> Hardware Drivers

### Post by NoaHall on 2009-09-06
You have to have "Rotate Cube" enabled and then it's "ctrl + alt plus mouse drag" If you double click on the options in Compiz, it will tell you the shortcuts, and you can change them if you want.
