### Post by WorMzy on 2010-08-06
Open up a terminal (Applications -> Accessories -> Terminal) and run ```
sudo apt-get install monodevelop
```

Alternatively, open up the software centre (Applications -> Ubuntu Software Centre) or Synaptic (System -> Administration -> Synaptic Package Manager) and search for it there.
