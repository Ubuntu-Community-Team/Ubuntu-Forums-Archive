### Post by rune0077 on 2008-03-17
If it's just the menu items, this can be done by right-clicking the menu and selecting "Edit menu". Then right-click the menu item you want to edit and select "Properties". You can change the name in the window that pops up, and you can change the icon by clicking the icon preview in this window as well.
