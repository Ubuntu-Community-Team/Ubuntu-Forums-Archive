### Post by grahammechanical on 2015-09-03
@G6Fe8kn

It becomes very confusing trying to give help to different people especially when their problems are not the same. On your system Software Updater is not crashing out of existence is it?

Please read this link. If you are runniing 12.04.5 then you should also be using the kernels in 14.04 (trusty). The purpose of what are called point releases is to keep long term support versions up to date with the kernel and Linux hardware modules that are in the newer versions of Ubuntu. So, that old versions like 12.04 will still be compatible with newer hardware.

[http://www.omgubuntu.co.uk/2014/08/ubuntu-12-04-5-lts-released-ready-download-now](http://www.omgubuntu.co.uk/2014/08/ubuntu-12-04-5-lts-released-ready-download-now)
