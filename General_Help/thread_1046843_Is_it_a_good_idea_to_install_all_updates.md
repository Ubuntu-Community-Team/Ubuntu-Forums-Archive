### Post by Shazaam on 2009-01-21
+1 for what jrusso2 posted. I have only had two update hicups; an Xorg in Dapper (6.10) and a kernel update in Hardy. If you keep at least one older kernel listed in grub you can boot that one if an update borks your system.
In Software Sources>Updates make sure the "Pre-released updates-intrepid proposed" box is unchecked unless you like troubleshooting/fixing crashes.
