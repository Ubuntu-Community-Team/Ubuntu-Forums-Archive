### Post by ajgreeny on 2009-08-26
Any of the disk burning applications should give you the option to "burn disk image" to the CD or DVD.  Brasero certainly does, as does k3b, gnomebaker, and in fact every one that I can think of.  Just make sure you burn as an image, not just burn the iso file as data.
