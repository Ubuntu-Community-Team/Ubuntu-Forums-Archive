Answer to all these questions would be "yes". Perhaps you should just give it a try. :)

(just to add a bit of details, the right-click menu for removable media will show some different options depending on what type of media it is. For example CDs and DVDs get the "Eject"-option which will physically eject the media from the drive, while external hard drives get a "Safely Remove Drive"-option which unmounts the file system ad then powers down the device. You might also get the "Unmount" option in addition to those above, which wold leave the device functioning and just unmount the file system, useful with internal drives or when you need to unmount a file system for some reason or other but you want to use it again in future without having to reconnect the device)
