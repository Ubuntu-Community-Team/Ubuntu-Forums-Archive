### Post by grahammechanical on 2011-07-21
Do you want?

1) to change the language of the operating system? Go to System Settings and load Language Support. There you can Install/Remove Languages and Apply system-wide. Then you can select a language at login.

2)to type in a different language in programs like Libreoffice but keep the operating system language as it is. Go to system Settings and load Keyboard. There under the layout tab you can Add a different language keyboard layout. This will put an icon on the panel which will let you chose the language layout for using in documents. You can switch languages in the same documents.
