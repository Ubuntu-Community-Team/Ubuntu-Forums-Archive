### Post by Insane1 on 2009-02-19
Actually, if you go into synaptic and lookup nautilus, there is a package you can install that adds that option. There's another package I believe you can install to add your own, but I forget which one. There's also a gksu nautilus package similar to the open terminal one which, once installed, lets you right click a folder and open it in a root window (povided you have the right password. :P) Note, use the actual search button rather than the quick search bar, or you may miss it. (The quick search bar has gotten...rather inaccurate lately.)

If you need the exact package name, I can edit my post later. I'm not at my Linux computer right now so I can't do a synaptic search.

### Post by geirha on 2009-02-19
The two packages Insane1 is talking about is [nautilus-open-terminal](apt:nautilus-open-terminal) and [nautilus-gksu](apt:nautilus-gksu). After installing, log out and log back in for the change to take effect.
