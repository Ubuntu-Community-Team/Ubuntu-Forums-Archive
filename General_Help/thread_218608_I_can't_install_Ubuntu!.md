### Post by BuffaloX on 2006-07-18
The desktop/live (normal) CD needs 192MB to work. That's because it boots up with a completely working environment. This makes it possible to test Ubuntu, before installing.

You definately need the Alternate CD for your installation. 

The download site:
[http://www.ubuntu.com/download](http://www.ubuntu.com/download)

Choose where you wish to download from.
Scroll down till you find Alternate CD
Choose "PC intel x86 alternate install CD"
