### Post by TeamJ on 2009-07-15
In windows I would recommend EaseUs partition software, or Norton Partition Magic. Just use on of those to resize both partitions, it is self-explanatory. Or there is a partition manager in the repos, I for get it's name. I would recommend making 30GB on each, plus 70GB shared memory. This is because Ubuntu can read Windows, Linux and OS X format partitions, whereas Windows can only read Windows format partitions.
