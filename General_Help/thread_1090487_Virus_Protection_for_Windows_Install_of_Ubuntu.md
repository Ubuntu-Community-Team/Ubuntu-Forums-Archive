### Post by beno1990 on 2009-03-08
No; Ubuntu is running in a virtual machine environment which is pretty much completely independent of Windows.

Viruses aren't much of an issue on Linux operating systems. In fact, about 90% of the viruses that Linux virus scanners can detect are Windows viruses, which can't even affect Linux anyway.

### Post by damis648 on 2009-03-08
Actually Ubuntu is running natively on the system, but in a non-native way. Back to the point though, you won't get viruses in Ubuntu, so virus protection is not needed. That's not to say there aren't any viruses at all, but the chance you will contract one is so small there are actually no virus scanners. If you worried about Windows becoming infected while in Ubuntu, you can install ClamAV, which scans Windows files.
