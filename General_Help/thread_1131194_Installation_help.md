### Post by prem1er on 2009-04-20
> **MarshWarrior said:**
> I also want to know if I give all the space to Ubuntu does that mean that I have no free space to download programs and things? I want to know because I only have 1 HD.

No.  You are allocating the amount of space that Ubuntu has to work with.  So for example, if you allocated 100 gigs of space to ext3 (ubuntu) your hard drive would install ubuntu on that space, and anything remaining would be yours to use to "download programs and things".  Ubuntu only takes up about 4 gigs of space (including swap).

### Post by prem1er on 2009-04-20
If you know how much space you would like to allocate between the two OS's than you can partition it accordingly right now.  You should have 3 partitions created for Ubuntu.  A "/" partition for your system files.  A "/home" partition for your personal use. And Swap space.  The remaining space you can format as NTFS.  You can than later install Windows if you like.  Also you can remove that partition later if you decided to stick with ubuntu and add it to your "/home" partition.  There are lots of posts on this topic search for "dual boot".
