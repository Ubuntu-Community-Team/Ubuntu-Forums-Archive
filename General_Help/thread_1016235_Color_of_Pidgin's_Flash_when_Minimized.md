### Post by pmigli01 on 2009-01-26
I recently switched to a dark theme and this annoyed me as well.  You have to turn on the message notification plugin and check "Set window manager URGENT hint" under configuration.  This will have the window flash whatever color you have set for selected items in your color scheme.

Hope this helps.
