### Post by svenskmand on 2010-05-15
Regarding the clicking when the mouse is still: I may have a solution for that, it sounds like you have enabled "Dwell clicks", to disable it go to "System -> Preferences -> Mouse", then click the "Accessibility" tab and make sure there is not mark in the box saying "Initiate click when stopping pointer movement".

Hopefully that should fix the second problem :)

Regarding the forward/backwards buttons I think it is usually a pain to get it working, but I think it is possible, but I have not solution. You need to know which signals the buttons send out, and then you should attach keys to these signals, like Alt+Left to go back and Alt+Right to go forward. I have never gotten it work myself thought :S But I also have not really tried that hard.
