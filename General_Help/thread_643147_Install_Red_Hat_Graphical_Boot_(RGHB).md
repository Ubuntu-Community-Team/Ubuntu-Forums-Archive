### Post by SpookyET on 2007-12-17
Recently, I have installed Fedora 8 because its artwork captured my eyes, and yes, it's pretty. However, it is functional. 

I wonder if there is a deb or a way to port the RHGB to Ubuntu. It's more than just a splash screen. It's real. It shows a real progress bar. It also allows you to maximise a window that displays the boot log. It actually opens it automatically  if there is a failure. The reason why it's pretty is because it is running XOrg. It's a special Xorg just for it which is turned off after a successful boot.

[RHGB at Fedora 8 Artwork]("http://fedoraproject.org/wiki/Artwork/F8Themes/Infinity/Round3Final#head-fe57bb05ef4ec062d8c4da11e8bc6c2b78aa89e9").

I already stole the GDM, splash, cursor, and GRUB themes from Fedora. RGHB is a bit more complicated.

Fedora 8 has some inovative things we can copy. An iptables GUI (firewall). A nice services GUI. PulseAudio which allows you to control application specific volume (like Vista), which I could not get to work in Ubuntu. SELinux is also enabled by default, and it has graphical tools for its management. I turned it off because it was annoying. I wish it would behave like Comodo Firewall Pro on Windows, which has a "Clean PC Mode" where it auto-learns instead of blocking. It also uses LVM by default, at least the LiveCD.

However, it's not as friendly as Ubuntu overall, especially when it comes to package management, font rendering, codecs, and SLED menu, etc.
