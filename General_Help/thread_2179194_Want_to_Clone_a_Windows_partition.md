### Post by Mark Phelps on 2013-10-07
> can I clone a 600 GB Windows partition onto a 240 GB SSD if only 100GB of the windows partition is used 

Basically -- NO.  Cloning is what the word implies -- an EXACT copy of the original.  

If you shrink down the partition to fit, you can then clone it.
