### Post by ScarletWyvern on 2010-08-20
Strictly speaking, "Linux" is a kernel that many operating systems use. These operating systems are collectively referred to as Linux because they all work primarily the same.

What others mean by customizing depends on what you mean by customizing. If you're talking about customizing  in terms of changing components of your desktop (different desktop environment, different display manager, etc.) Linux has loads of a freedom in that area.

If you're talking about customizing the code, Linux is open-source, which means you're allowed to download the source code and make whatever changes you need to make it suit you.

Other than that, I'm not entirely sure what you're asking or trying to do.
