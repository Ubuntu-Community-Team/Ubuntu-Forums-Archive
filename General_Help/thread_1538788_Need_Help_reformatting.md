### Post by WorMzy on 2010-07-25
?

Are you using a LiveCD (Ubuntu) or the Windows 7 disk? Your post says you're using both..

If your using the Windows disk, you don't want to use the repair console, you want to use the installer. Repair is for repairing broken Windows installations, not for installing Windows.

Once you start the Windows installer, you'll be able to delete partitions there.

If you're using the LiveCD, then you just need to run gparted (System -> Administration -> Partition Editor)
