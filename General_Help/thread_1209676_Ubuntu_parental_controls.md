### Post by phillw on 2009-07-10
[SIZE=2][FONT=Arial]Hi,

Having put child controls onto a computer recently, can I suggest an easy way to keep web browsing / sites child safe.

Using firefox, go into tools >> Add Ons >> Get Add Ons

Type in WOT  - it will come back as Web Of Trust, install it - it will restart firefox and ask for some details. You can then go onto the settings part of WOT and select the levels of warnings you want - INCLUDING the block option. In my case any RED rated site is barred & anything less than GREEN for child Safety is also blocked.

You can read more about the project here :

[http://www.mywot.com/](http://www.mywot.com/)
