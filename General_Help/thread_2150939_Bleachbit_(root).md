### Post by Maintech on 2013-06-02
In previous versions of bleachbit you could run it root mode by typing in your password when prompted. In 13.04 it has changed from gksu and sudo to su. It will accept no password. Anyone else had this issue? What is the fix?

***_The fix is to edit the properties and change the su to gksudo. Works correctly then._***
