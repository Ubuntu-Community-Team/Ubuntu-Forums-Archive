### Post by Helios89 on 2008-09-23
You can't directly install MSN Messenger.

As you can see [here]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=127"), the latest version (partially) supported is 7.0, really old compared to "Windows Live Messenger" that you can nowadays install on your Windows PC.

But in Ubuntu there are lot of alternatives that work great: I use Kopete (which can stand lot of other protocols such as Gtalk, IRC, and much more..) and the only thing I miss from MSN are the customized emoticons, but it's not a great deal.

Open Synaptic and type "MSN".. there should be a list of the instant messaging programs that support the MSN protocol
