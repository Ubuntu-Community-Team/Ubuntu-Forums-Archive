### Post by tmatt95 on 2007-04-20
Hi cshelswell,
Here is a link to an article that you may find helpful:

[http://www.ubuntugeek.com/how-to-export-your-mails-from-evolution-to-thunderbird.html](http://www.ubuntugeek.com/how-to-export-your-mails-from-evolution-to-thunderbird.html)

I have taken the bit out of it that you might find helpful and quoted it below:

"In the other window from your home folder navigate to “.evolution/mail/local” This is where any locally stored mail from Evolution is placed. There are a similar set of files to what you see in the Thunderbird folder, Inbox, Drafts and so on."

I imagine you can copy the files out and then paste them back into the working version

Best Regards,

P.S I would try that with a Live CD first so as not to end up loosing the e-mails if it does not work.
