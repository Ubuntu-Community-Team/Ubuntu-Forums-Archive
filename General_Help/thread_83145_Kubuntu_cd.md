### Post by aysiu on 2005-10-28
Well, you have several options:

1. Go to an online store like [Cheap ISO](http://cheapiso.com/) and order it for a few dollars. I'm not sure if Cheap ISO ships internationally, but you can shop around and find something.

2. Find a friend who has a broadband connection. Have her start downloading the ISO for you overnight. It should be finished by the morning.

3. Order the Ubuntu shipit CDs. Then, after installation, open a terminal and type in ```
sudo apt-get install kubuntu-desktop
```. Now you have Kubuntu. Log out, pick session of type KDE, then log back in.
