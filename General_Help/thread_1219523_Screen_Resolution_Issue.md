### Post by synapsys on 2009-07-23
Ubuntu will install all the **open** source drivers available for your hardware. At this time nvidia drivers are **closed **source. They are not installed by default because they don't exactly follow the philosophy of open source software (ubuntu). Some people want totally clean systems with no proprietary software at all. This is why you must specifically tell ubuntu to install any proprietary drivers (nvidia). 

Glad you got it fixed. :)
