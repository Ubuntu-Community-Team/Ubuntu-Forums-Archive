### Post by snowpine on 2009-03-09
> **rexy1 said:**
> I thought I'd try out Puppy Linux, since Ubuntu (and Xubuntu) runs sluggish on my system. So I tried using Unetbootin to create an installation usb-stick, but when I reboot the usb-stick won't boot up! The usb-stick is formatted to FAT32. Help much appriciated!

First thing to check is whether or not your computer is capable of booting from USB (most older computers aren't; how old is yours?)

Enter the computer's BIOS menu (turn the computer on and press one of the function keys, depending on the computer's manufacturer), go to the Boot options, and move USB to the top of the list, if it is an option.

If not, I recommend burning a Live CD instead.
