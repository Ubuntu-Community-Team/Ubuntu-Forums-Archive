### Post by scrooge_74 on 2007-11-21
Let me put it this way:  you can get ubuntu installed in less of one hour, maybe some tweaks after but should be ready in a flash.  With Debian it is going to take you a little longer (a couple more hours), but the end result will make you happy.

You can download a net install CD for Etch or Lenny and off you go.  With Etch it is very likely things will work pretty well, a few apps will be a little old. You work around that using backports.

If you go with Lenny, things are going to be also pretty solid with a few quirks that you will have to look for the fix, but should be pretty good also.

this laptop I have has run Edgy, Feisty and then Etch and now is running Lenny.  I was very happy with Etch, but I wanted to have Pidgin instead of Gaim (plus a few other little things), but now that I am running Lenny I am in a blistfull state.
