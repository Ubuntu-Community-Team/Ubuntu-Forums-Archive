### Post by RAOF on 2006-11-23
On Feisty I'd say "AIGLX, for sure", because Feisty has the latest nVidia drivers which support the TFP extension needed for compiz/beryl.

Unless you want to install the new nVidia drivers manually (and there are some pretty good howtos around.  Try tselliot's envy script, particularly), you need to use XGL.

I'd also suggest Compiz ([forum.go-compiz.org](forum.go-compiz.org)), but that's just my KDE hating coming out :P.
