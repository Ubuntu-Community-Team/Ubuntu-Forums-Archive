### Post by raja.genupula on 2012-04-27
```
sudo gedit /etc/apt/sources.list

```
and remove all the entries in that and get a new one from [http://repogen.simplylinux.ch/](http://repogen.simplylinux.ch/)


then paste the code in the sources.list , save the file and try again . 


NOTE: take a back up of old source.list before removing the entries of it .
