### Post by MJBoa on 2010-03-02
It's virtually impossible to install Ubuntu to a mobile phone. Now there are different definitions of mobile phones, and I don't know all the devices out there so there MAY be some netbook that's able to make phone calls that can run Ubuntu, it's not impossible.

But I assume you mean the normal definition of a mobile phone.

OK, so mobile phones that "run windows" run a version of windows that barely resembles the Windows you know (XP, Vista, 7).
Same with Linux on mobile phones. There is a Linux-based operating system called Android that's pretty popular on today's smartphones, I use it myself. But it's not similar to Ubuntu at all, except at the most basic level. If you like Linux because it's free and open source then maybe you should go with Android.

Bottom line, you can't run Ubuntu on mobile phones. The Windows you see on mobile phones is a completely different version of Windows.
