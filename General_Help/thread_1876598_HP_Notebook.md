### Post by Matt__ on 2011-11-06
[ubuntu 10.04lts](http://releases.ubuntu.com/lucid/) for long term life and classic 2 panel look
or
[mint julia](http://www.linuxmint.com/edition.php?id=67) for bottom only panel that is more akin to windows and includes many codecs etc that need adding manually in ubuntu.

If she is willing to try something new go for ubuntu 11.04 with unity interface, but which still has classic fallback mode if she dosnt like it!

of course like most netbooks you can ["hackintosh"](http://www.youtube.com/watch?v=t4iv6wkd5a0) it too..that is install MAC OSX leopard which runs really well from what I've heard.

//edit: Like collisionystm said....try them as live cds/usbs so you can get a feel for what theyre like and a heads up on any hardware issues that you may encounter.


also [check the hardware support list](https://wiki.ubuntu.com/HardwareSupport/Machines/Netbooks) for the exact model to help you decide.
