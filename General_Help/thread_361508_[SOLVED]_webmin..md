### Post by Jussi Kukkonen on 2007-02-14
[http://www.webmin.com/intro.html](http://www.webmin.com/intro.html)

Webmin is a web-based interface for system administration for Unix, usually used on headless servers that have no graphical user inteface  installed

Everything webmin does, can be done through the command line, but some people like to do things through a web interface...
