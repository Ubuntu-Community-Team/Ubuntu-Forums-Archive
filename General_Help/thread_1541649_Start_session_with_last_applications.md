### Post by hhh on 2010-07-29
Try System>Preferences>Startup Applications>Options, uncheck "Automatically remember..."

If that doesn't work, go to your home folder and select View>Show Hidden Files (or just Ctrl+h) and go to .config>gnome-session>saved-session and rename the file(s) inside. If that works (it should), you can delete the renamed file if you want.
