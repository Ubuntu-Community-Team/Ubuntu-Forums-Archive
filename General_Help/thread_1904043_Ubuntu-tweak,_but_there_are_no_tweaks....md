### Post by crazy bird on 2012-01-04
I don't know what you want to do with Ubuntu Tweak, but keep in mind that Ubuntu Tweak is not a tool for starters. This tool can be very destructive if you change the wrong settings. Also Ubuntu Tweak is a security breach since Ubuntu Tweak is able to add PPA's of unchecked sources which can contain packages which can have a negative influence on your system stability. 
 
In normal situations i do not recommend Ubuntu Tweak, but as long as you know what you're doing... Go ahead i would say!
