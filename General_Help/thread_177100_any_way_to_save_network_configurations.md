### Post by DJ Scribblinni on 2006-05-15
If my mind serves me correctly, if you go into the Network Settings dialog and set a Location.  Set one to home it should automatically save it for you.  Create another location for work, change all the settings needed, it should save them by itself.  When you switch between the two select which one you need.
