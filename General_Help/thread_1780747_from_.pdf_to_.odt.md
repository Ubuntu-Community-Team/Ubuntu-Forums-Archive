### Post by amanchesterman on 2011-06-12
Hi I am fairly new myself and I'm not familiar with all the software available for Linux. But like you I am using Ubuntu 10:10, and until I retired I was a teacher.

You can open a .pdf using the default application (it's called 'Document Viewer'), choose the Edit > Select All command to highlight the text, then Edit > Copy; then simply paste that into a blank Open Office document and save it in .odt format. You lose the formatting and layout but it works with the Orca text-to-speech application which your student needs. In Document Viewer you can also enlarge the text up to 400%.

I realise of course that that is not as neat as a program that will simply convert the file and retain all the layout and formatting. The trouble is that .pdf documents often contain images etc. and I guess that it is hard to write a converter that will not produce errors. There is an add-on extension for OpenOffice that allows you to import .pdf files, but it works with OpenOffice 'Draw' and not 'Writer'.

As an alternative there are several online services that offer to convert .pdf files into .doc format (for Microsoft Word). I don't know how reliable they are, but if you could get the documents into .doc format you could then open them in OpenOffice.

Good luck
