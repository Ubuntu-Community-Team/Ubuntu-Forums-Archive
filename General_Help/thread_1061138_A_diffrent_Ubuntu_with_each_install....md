### Post by cariboo on 2009-02-05
There is nothing wrong with your installation that a little time spent looking at the problem won't solve.

When using Synaptic, I always click the Status button, see the screenshot.

If you see an error message, do what it says instead of reinstalling. You have got to get rid of the windows mindset, that to fix a problem all you have to do is reinstall. In Linux unless you completely purge a package the config file won't be deleted.
