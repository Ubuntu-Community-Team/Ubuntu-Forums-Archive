What an interesting problem.  One, which to answer I had to experience, so I unexpanded mine.

Open up the **Configuration Editor** (in the Applications - > System Tools menu) and use the tree on the left to open up apps -> panel -> toplevels -> top_panel_screen0 and check the **expand** box.

Now that you have space on the top panel, right-click on it and choose **New Panel**

Get the panel positioned where you want it and lets add those things you lsot when it got deleted.

Right-click the bottom panel and choose **Add to Panel**.  Then to get a basic looking panel (pretty default I think) add a **Show Desktop**, **Window List**, **Workspace Switcher** and **Trash**

### Post by Qrk on 2006-04-13
There should be several pixels of blank space at the very end of either side of your top panel. You can click on those, then on Properties to expand it back.

To get your bottom panel back, click on the top one and slelect "new panel" choose the bottom orientation. To that panel add the following applets (right click, Add to Panel)

 "Show Desktop"
"Window List"
"Workspace Switcher"
"Wastebasket Applet"

You should be set to go.
