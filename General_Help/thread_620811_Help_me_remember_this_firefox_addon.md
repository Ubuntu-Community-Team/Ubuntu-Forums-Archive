### Post by retrow on 2007-11-26
I'm making some assumptions here and I might be totally wrong, but I'll try anyway.

If you have upgraded to latest Ubuntu without actually removing your older version, you should be having the addon you were using under Addons in the extension section.You should be able to find that name in the list of extensions. Secondly it might be the case that the addon wasn't compatible with Firefox 2.0.0* and was disabled. There were some in the Firefox 1.5* era which didn't release updates for 2.0.0* compatibility and were unusable under the new Firefox.

As far as I know Firefox doesn't remove any addons unless a huge number of people report them as unstable. There are addons and themes compatible only with Firefox 1.x on the mozilla list of downloads. So I suppose that the addon you are looking for is still there, but it must have been buried into oblivion because of it wasn't updated to keep up with new versions of FF.

Hope this helps.
