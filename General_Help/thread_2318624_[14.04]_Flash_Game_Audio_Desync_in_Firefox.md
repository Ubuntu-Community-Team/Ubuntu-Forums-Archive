### Post by bernard-godard on 2016-03-28
Adobe Flash Player for Firefox Linux is not updated anymore. Usually I don't have a problem with FlashPlayer in Firefox, but if you are using an Intel/AMD system you may have more success with Chrome (which uses an up to date Flash Player): [https://www.google.com/chrome/browser/desktop/index.html](https://www.google.com/chrome/browser/desktop/index.html)

Now there could be many reasons for the problem you mention and you might want to give a bit more details:
- Ubuntu version and architecture: eg 16.04 x86-64
- CPU
- RAM
- sound card
- graphic card and driver.
