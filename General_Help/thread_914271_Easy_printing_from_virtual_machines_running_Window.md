### Post by squashpup on 2008-09-08
If you're using a virtual machine running Windows (VirtualBox,VMware, etc) and you have a shared folder between your guest and host operating system, you might want to try the freeware BullZip Print to PDF as an easy way to print from your guest Window OS.  There is no cost for personal use or up to 10 stations if used commercially. This program will allow you to print to a PDF file from your Windows OS directly to your shared folder where it can be easily printed from your Ubuntu host.  

I just set this up and it works great.  Probably not a good solution if you have to do a lot of printing from Windows, but for occasional printing it does the trick, and it's simple to use.  

I know this sounds like an advertisement, but it's saved me a lot of anguish and seems to work well from computer one computer to the next, and I want to save others potential headaches. 

If you like it, you can donate to them via PayPal on their website.
