### Post by strabes on 2007-01-17
You sound like a pretty serious gamer...I would recommend just dual booting windows and ubuntu, and having a 3rd ext3 partition for shared data, installing the program from fs-driver.org to read/write to the ext3 partition in windows. For converting your .wma files, try soundkonverter (sudo apt-get install soundkonverter) but I don't know if it will be able to convert them since .wma is a proprietary windows format. There must be something in windows to batch convert .wma files.

Have you tried kino for capturing video?

Some applications don't show up in the gnome applications menu, and you have to run them with a command. Generally the command to run them is the same as the package name, but not always.

For mplayer firefox stuff, do this:
```
cd /usr/lib/firefox/plugins/
sudo rm libtotem*
sudo apt-get install mplayer mozilla-mplayer
```
Then restart firefox and you should be good.

You can download the flash9 beta for linux from labs.adobe.com, and put the file in the ~/.mozilla/plugins (~/ represents your home directory by the way)

hope this helps
