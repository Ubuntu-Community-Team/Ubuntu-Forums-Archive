### Post by lovinglinux on 2011-10-16
Depends on what you have installed. When you install some applications, ither applications are installed as dependencies.

Compiz for example is required by Unity 3D. So don't remove it.

7zip cannot be removed without removing file-roller. So don't remove it.

Computer Janitor probably can be removed safely, but on 11.10 it is not installed by default, so I can't check now.

Character map is a tool for adding special characters into documents. I would leave it alone.
