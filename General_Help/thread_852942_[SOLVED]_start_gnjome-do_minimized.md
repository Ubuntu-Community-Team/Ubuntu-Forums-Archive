### Post by funkydan2 on 2008-07-09
If you're running the latest version (0.5.0) then when you see the gnome-do window, click the little down arrow in the corner and then select 'Preferences'.  In the 'General' tab, tick the 'Hide window on first launch' option.

If you're running the earlier version, check that in your 'Startup Programs' in your sessions manager than gnome-do in invoked with the --quiet option (I think).
