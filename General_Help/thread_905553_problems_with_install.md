### Post by EmanresuusernamE on 2008-08-30
Sounds like the CDROM drive isn't reading your CDs properly.  I've had that happen to me a few times, or the ISO you downloaded is bad.  Have you tried using a different CDROM drive to install Ubuntu?  If you like, you can remove the hard drive you're going to install Ubuntu on, and put it in another computer, (disable the other hard drives in that computer) and then install Ubuntu that way, and finally move your hard drive back to the first system, (re-enable those other hard drives too).  Unlike windows Ubuntu won't kill itself when it notices the hardware changes.  You'll just need to reconfigure it a little bit.
