### Post by audiomick on 2010-12-29
Whilst it is good, and in the long run perhaps unavoidable, to learn terminal commands, for practically all of "normal" day to day use, you shouldn't need to for a start.

The quickest way to learn Ubuntu is just to start using it, i.e. boot into Ubuntu to do all your stuff, and only start windows when you really can't get any further and really need to get something done in a hurry.

Oh, and read those links that have been posted... ;)
