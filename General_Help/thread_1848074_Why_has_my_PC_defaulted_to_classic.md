### Post by patryk77 on 2011-09-22
If I understand correctly, Classic mode is for the display. Instead of using Unity (new interface with fancy graphics) it uses standard Gnome.

That can be changed on the login screen, after you select your user name you can change Classic to Unity if it's available, though it may have not been installed in the first place, in which case you would have to install it.

Either way, personally I don't like the Unity layout, so I wouldn't worry about it, but there must be a way to get it running if you choose to do so.
