### Post by Portaro on 2021-03-30
This is a tool that manage errors that occur on your system and to improve and correct the problems you have the option to send the problem for the Ubuntu Team developers, with this reports they can know that the package software have problems and try to solve the problem and patch the packages and update them.

Is a software reporting tool to inform about crashes and problems with software on your SO.

I recommend to you to use it if you want to collaborate in the development of corrections of Ubuntu Team, but also you have the option to disable it on your System Settings &#8594; security and privacy &#8594; diagnostics.
