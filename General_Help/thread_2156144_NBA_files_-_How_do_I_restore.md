[LIST]
[*]The "SETUPAPI.LOG" log file: This file will list all installed devices, Windows Service Packs and Windows Hotfixes.
[*]The "TWAIN.LOG" log file: This file will list the installed devices recognized via TWAIN.
[*]The "SchedLgU.txt" log file. This file contains information on  currently scheduled tasks (e.g. the JobScheduler in Nero BackItUp)
[/LIST]
