### Post by jondodson on 2011-01-22
Ubuntu is far superior to Windows, as long as your main interest isn't playing 'big name' games (if it is, get a console).  Some specifics answers:
1. Security - virus infection is a non issue with Ubuntu
2. Freedom - ubuntu is free in two ways.  Firstly, it doesnt cost anything.  Secondly the software is intellectually free, which means that you are free to improve existing ubuntu software for the greater common good of it's users, if you so wish.
3. Power - there is a huge existing database of free software that you can download direct to your desktop.  If you can think if it, there's probably an app for it.
4. Power again - windows is split into home, professional, ultimate and most people opt for the cheapest home version, which is restricted in various ways, especially with respect to networking.  Ubuntu only has one version and i'd say it equates to windows ultimate.
5. Interface - you can customise the appearance and behaviour of absolutely everything.
6. Community - a glance around this forum should reassure you that there are tons of people willing to help if you run into problems.

Regards,
Jon.
