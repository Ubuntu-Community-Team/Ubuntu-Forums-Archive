### Post by KevMeistr on 2008-01-18
I use quite a lot .dat files and all you have to do is allow the permissions of the file by Right Clicking -> Properties -> Permissions -> Check Allow to execute...

This should allow you to double click the file and then choose whether to display the file in gedit or to run the file in the terminal...safe to display the file... if nothing appears then the file might be protected or you are not the "owner"
