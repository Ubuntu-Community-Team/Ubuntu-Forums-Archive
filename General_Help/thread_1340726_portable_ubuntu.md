### Post by fluffman86 on 2009-11-28
Um, I think you're confused as to what RAM is.  If by "RAM" you mean "DISK SPACE" then about 4gb of disk space is required to install ubuntu, and you only need about 256 or 512MB or RAM.

Personally, when creating a portable Ubuntu thumbdrive, I use a 1 or 2 GB drive and create it in a full install of Ubuntu with System > Administration > USB Startup Disk Creator.

If you need to do this in Windows, check out unetbootin (google it).
