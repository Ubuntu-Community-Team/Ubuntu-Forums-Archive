### Post by sayakb on 2008-07-05
Why dont you just burn the iso on a disk? Use the inbuilt program brasero. Goto Applications->Sound and Video -> Brasero disk burning
Click on Burn Image button. Rest should be self explanatory.
Also, this solution applies only to Ubuntu Hardy Heron 8.04
If you are using some other version, open a terminal (Applications->Accessories->Terminal) And type in:
```
sudo apt-get install brasero
```And then follow the above solution.
