### Post by RedSingularity on 2009-02-04
How do i make a link with a name?  I dont want to see [http://www.blah-blah.com........I](http://www.blah-blah.com........I) want to see the word LINK that can be clicked on.

### Post by Old *ix Geek on 2009-02-04
You mean here on the forum? Just type it like this (but REMOVE all the spaces!):
[url = "YourURL" ] HowYouWantItToLook [ /url ]

So, in your example:
[url = "http://www.blah-blah.com" ] LINK [ /url ]

and, without the spaces, it'll look like this:
[LINK]("http://www.blah-blah.com")

### Post by Iowan on 2009-02-04
On the forums, I open another tab to the site/thread I want to link, highlight the address, Edit>Copy (CTRL-C also works). Then I open a Reply window, click on the "Insert Link" icon, Edit>Paste (or CTRL-V), click OK, edit the link name (to replace link address), END key to move to end of line, and continue with other reply text. To use existing text as a link name, highlight it before clicking the "Insert Link" icon, and it will be used instead of link address.
