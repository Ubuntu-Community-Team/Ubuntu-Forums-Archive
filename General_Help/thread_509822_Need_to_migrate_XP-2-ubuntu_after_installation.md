### Post by logos34 on 2007-07-25
Here's a link that covers email, browser bookmarks, network settings:
[https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows/TransferringFilesAndSettings?action=show&redirect=MigratingFromWindows](https://help.ubuntu.com/community/SwitchingToUbuntu/FromWindows/TransferringFilesAndSettings?action=show&redirect=MigratingFromWindows)

If you use firefox in XP, a neat addon is Foxmarks to keep your browser bookmarks synchronized/up to date across platforms.

You might also want to get fs-driver for windows to add ext3 read+write capability, and get ntfs-3g and ntfs-config to enable writing to ntfs filesystem from linux
