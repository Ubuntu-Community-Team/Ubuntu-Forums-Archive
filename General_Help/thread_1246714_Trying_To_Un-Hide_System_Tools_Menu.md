### Post by Mel Thompson on 2009-08-22
That did it.  I checked Config Editor.  What was odd though, was that Virtual Box was in there, and it was checked.  But somehow having Virtual Box checked was not enough.  Somehow checking one of the other items, items that had been there before Virtual Box was checked, got the Systems Tools menu to show up.  This newer item that was checked apparently did not count as checked until I checked a seemingly more established item.  Anyway, thanks for the suggestion.  It helped, but for those who installed Virtual Box, they'll hopefully read this and now know that checking Virtual Box is not enough.  They need to check some other older item too, if they want to see their System Tools menu.
