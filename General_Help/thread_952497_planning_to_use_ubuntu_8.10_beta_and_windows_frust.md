### Post by shawnrgr on 2008-10-19
I can't answer all your questions but some....

1. planning to download ubuntu beta 8.10 but the official one will be released on oct. 30th if i used the beta should i still download the official realease when it comes?

If this is for your office stick with 8.04 or wait a few days after official release of 8.10 before you install it (waiting a few days after release lets to genpop clear out any missed bugs during testing)

3. now my laptop has 120 gb hdd and windows xp is currently installed there if i install ubuntu on top of that does the fragmented files of windows affect the speed of the ubuntu and vice versa?

Always defrag windows drives before installing along side and you should be fine.

4. im always concerned in security and researched if wine can be infected by virus but still no one answered it very clearly cause
they are divided to YES or NO. cause if its yes i think ill just
be infected cause i have windows program im planning to use with wine so whats the point to use linux i think.

First whats the program you plan on using in wine, there might be an alternative. Second, as far a I know, WINE CAN get infected but it doubt it will cause any damage, besides it will all be contained to your ~/.wine directory and nothing else. Rest assured that your entire ubuntu system will stay safe. Also most viruses are obtained when browsing the web and downloading emails, which would then be done on your ubuntu system, not wine, so i don't think you have anything to worry about.

5. planning to use VMWARE but i heard you cant play games with this is this true? and also correct me if im wrong if you run a virus on vmware it runs as normal as when youre using windows itself?

I prefer VirtualBox myself as do a lot of users. As for viruses... its a windows virtual pc, it almost no different then installing windows. So viruses, yes def possible.  As for games, most virual machines don't support advanced graphic card features. so things like OpenGL wouldn't be able to run. So depends on what type of games you want.
