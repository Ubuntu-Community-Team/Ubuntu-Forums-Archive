### Post by sikander3786 on 2010-09-13
It might be the screen saver coming up in case you leave the system idle. Then it asks for password to unlock the screen. Look in System >>> Preferences >>> Screen Saver. Adjust it accordingly in case it is the one causing problems. Otherwise, post back.
