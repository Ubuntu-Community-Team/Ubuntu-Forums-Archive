### Post by agnes on 2010-02-05
Yes with firefox it is, and quite easy.
You have to add a new "Custom Application Launcher"  to the panel.
Then the command should be: ```
firefox -new-tab www.website.com
```
[www.website.com](www.website.com) will open in a new tab in your current ff browser window, or if ff is closed, ff wil start and open that website.
