### Post by donald187 on 2023-01-14
I used Gnome Software a few years ago to do updates on Debian.  It didn't do automatic updates as such.  Rather it displayed a banner notifying you that updates were available.  If you clicked on the banner it would open Gnome Software which had a button which would install the updates if clicked.  There's a Flatpak plugin which includes Flatpak packages in Gnome Software. I think the updates are supposed to work the same way with Flatpaks but I'm not sure about that.  Here's the Flatpak site describing how to install the plugin.  I have not tried this on Ubuntu.  Just Debian.  The site says it installs a deb version of Gnome Software to work with Flatpaks.

[https://flatpak.org/setup/Ubuntu](https://flatpak.org/setup/Ubuntu)
