### Post by forrestcupp on 2008-02-08
Your problem is that you are running Xgl.  If you are using the fglrx driver in the repositories and you are running Compiz, you would have to be running Xgl to do it.  OpenGL stuff doesn't work with Xgl running.  If I were you, I would disable fglrx in the Restricted Drivers Manager, then go to Synaptic and uninstall fglrx and Xgl.  Then I would download and install [Envy](http://albertomilone.com/nvidia_scripts1.html) and use it to automatically install the very latest ATI drivers from their web site.  With the latest drivers, you don't need Xgl, but the ones in Gutsy's repos still require Xgl.

If you do this, you will be able to do OpenGL stuff, but you will need to have the Desktop Effects turned off to do it.  With Compiz running, opengl apps and videos flicker.  Honestly, ATI cards in Linux are still a pain.
