rockbox does not replace the organinal firmware.  If you install rockbox, you get a dual boot between the two OSes.  If I remember correctly with the iPod, to boot into the OF you turn the hold switch on, else it boots into rockbox.  I will also note that the OF in ipods does not shutdown, it goes into a type of sleep mode.  In order to boot into rockbox again from the OF, you will have to reboot the device using the key combination in the manual.  I *think* it is up + select.  The manual for rockbox contains all sorts of goodies, you should check it out.
