### Post by techvish81 on 2011-10-31
dont give " y" , just reboot your system and update cache using the check button in update manager and after that install synaptic from software centre.  
It solved my problem which was a bit different,  as I was trying to install synaptic from software centre and it was taking ages and failing everytime.  so, better to try this before taking any risk.
