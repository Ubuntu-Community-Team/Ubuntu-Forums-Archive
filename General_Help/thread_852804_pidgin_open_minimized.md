### Post by sdennie on 2008-07-08
One way to do this would be to install the pidgin-plugin-pack:

```

sudo apt-get install pidgin-plugin-pack

```

Then go to Tools->Plugins->Buddy List Options.  Enable it and one of the configuration options is to make the buddy list minimized by default.
