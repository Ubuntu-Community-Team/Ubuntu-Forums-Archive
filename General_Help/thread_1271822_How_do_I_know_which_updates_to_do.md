### Post by drs305 on 2009-09-21
Welcome to Ubuntu and the Ubuntu forums. 

The updates offered depend on what settings you have in the package manager (APT).

As a minimum, you want to have the security updates enabled. You find this in 
System, Administration, Synaptic Package Manager. 
Go to Settings, Repositories, Updates.
Enable 'Important Security Updates'
It is probably not a bad idea to add 'Recommended updates'.
'Proposed' and 'backports' are generally not necessary or recommended for normal use.
With these settings, you should be able to accept all updates.

Here is a link to the Ubuntu Repository wiki, which explains things further:
[https://help.ubuntu.com/community/Repositories/Ubuntu]("https://help.ubuntu.com/community/Repositories/Ubuntu")

I personally install all updates.  Working on the assumption that they are patches for things already installed (regardless of whether or not you know what they do, or what depends on them), it's rarely a bad thing.
