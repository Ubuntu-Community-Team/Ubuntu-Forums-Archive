### Post by novize on 2009-07-21
There is rapidjc. But it does not resume uploads.
There is tucan too, but it does not support rs uploads yet. They say, they work on it and there will be soon a possibility to upload files with this programm.
Untill then, you can download rapidjc and run it normaly in ubuntu, or you can download the latest rapiduploader and run it with wine (it works, I have tested it).
Or... the third way, look here: [http://ubuntuforums.org/showthread.php?t=1195389](http://ubuntuforums.org/showthread.php?t=1195389)
