There are linux equivalents of Itunes to run and manage your Ipod as well. It's not like you would have to switch/boot to XP every time you want to use Ipod, itune and transfer files from Itunes to the Ipod. That really sounds like crap and is a hardcore inconvenience.

Banshee ( [http://banshee-project.org/Main_Page](http://banshee-project.org/Main_Page) )
Listen ( [http://listengnome.free.fr/](http://listengnome.free.fr/) )
Quod Libet ( [http://www.sacredchao.net/quodlibet](http://www.sacredchao.net/quodlibet) )
SongBird ( [http://www.songbirdnest.com](http://www.songbirdnest.com) )
Exaile ( [http://www.exaile.org/](http://www.exaile.org/) )
Amarok ( [http://amarok.kde.org/](http://amarok.kde.org/) )

The free programs mentioned above are pretty similar to Itunes.  Also, if you want to use your Ipod in Linux, you may want to have a look at the following sites:
