### Post by alan404 on 2010-11-11
Hello. I have BT "Total Broadband" and it's fine. You don't need any weird windows software for set up. To configure the router you just go to a web address in a browser ([http://bthomehub/home](http://bthomehub/home)). But I didn't need to as it just worked. The router is far cleverer than me. I had trouble setting fixed IP addresses on the LAN, but I just set my mini file server to DHCP and the router found the host name and it has always worked every time by just typing the host name rather than the IP address. It also has a BTFON section where you share some of your wireless connection in return for free wireless access anywhere (this is really worth doing -- although gone are the days sadly where I simply enjoyed a coffee in Starbucks).
