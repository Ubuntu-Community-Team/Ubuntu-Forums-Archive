### Post by forrestcupp on 2008-04-01
Try opening Synaptic and go to Settings->Repositories.  In the 'Updates' tab check the boxes next to 'Pre-released updates' and 'Unsupported updates'.  Then close that window and hit the Reload button in Synaptic.  Search for flashplugin-nonfree and install it.  Then restart Firefox, and it should work.

### Post by forrestcupp on 2008-04-01
I'm pretty sure they have B4 in the proposed repos.  If so, you'd be better off installing it that way and the flash plugin will just work.  To enable those repos, open Synaptic and go to Settings->Repositories.  In the 'Updates' tab check the boxes by 'Pre-released updates' and 'Unsupported updates' then close that and click the Reload button in Synaptic.  Then search for Firefox and find FF3 to see what version it is.  If it is B4, then install it, and you should be good to go.
