### Post by zombifier25 on 2012-06-25
Try Right Clicking a tab > Undo Close Tab (or press Ctrl+Shift+T)

If that doesn't work, open about:cache?device=disk and search for your file from there (Ctrl  +F and start typing part of the name). With luck, Firefox may still be caching that file on your disk. Look for its entry, click on it and view "file on disk". The file may have duplicate entries, so check all of the entries that has the same name.
