### Post by Yashiro on 2009-04-16
Browse to the package.[http://packages.ubuntu.com/hardy/admin/acpid](http://packages.ubuntu.com/hardy/admin/acpid) for example.
There's a CHANGELOG link on the right of the page.

Or find the package using Synaptic and select Download Changelog from the menus.

Or do aptitude changelog <packagename>

Or install apt-listchanges

Or try deb-gview

Or browse directly at [http://changelogs.ubuntu.com/changelogs/pool/](http://changelogs.ubuntu.com/changelogs/pool/)
