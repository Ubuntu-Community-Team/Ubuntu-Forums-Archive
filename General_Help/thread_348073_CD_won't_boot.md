### Post by Happy_Man on 2007-01-28
Well, when your computer starts up, it first goes into BIOS. This is usually when the computer manufacturer's logo is displayed for a couple seconds. The BIOS is what initially boots everything up. It is also set to default boot from the information on the hard disk-in your case, XP. Usually somewhere on the screen when it starts, there will be a line that tells you what key you have to press to change the boot order. Most times, it's something like F12 or ESC. So all you have to do is press that key and make BIOS boot from your CD drive. Then the LiveCD will load. Hope this helps!
