Hello,
I have VLC player installed on my Ubuntu 11.10.
Default Music and Video application was Media Player.
My goal was to change default for VLC Player.
From System Settings -> System Info -> Default Application I change default to VLC Player for Music and Video. I rebooted the system but when I start any e.g. video file it still opens it with the former default app i.e. Media Player and not with VLC.
What's wrong?

### Post by holycow131415 on 2012-04-03
known bug.

work around is to just right click on a file, go to properties, and somewhere in there you can set the default app to open for that type of file. you will need to do it for all files.
