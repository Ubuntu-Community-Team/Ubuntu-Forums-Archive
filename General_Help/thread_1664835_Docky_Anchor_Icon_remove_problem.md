### Post by hhh on 2011-01-11
That key only works with Docky version 2.1.0. Go here, find the right version for your system (example, if you use Lucid you want docky - 2.1.0~bzr1761-0ubuntu1~10.04~dockycore1), click the package name to show all files and click the correct "all.deb" file to install it with Gdebi (the second from the bottom one, not the "docky-dbg" one at top)...
