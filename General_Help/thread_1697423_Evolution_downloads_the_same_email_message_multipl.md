### Post by Bitrate on 2011-02-28
Sounds like a problem with the message pointer on the Hotmail server. 

The duplicate messages should really only appear once when you do a full mail download. Use an evolution add-on (eg. [https://launchpad.net/~toobaz/+archive/toobaz/](https://launchpad.net/~toobaz/+archive/toobaz/)) to remove the duplicate messages from your inbox.
