### Post by FrostDust on 2008-05-15
How exactly don't your installs work? Is there a specific error message or other event that happens when you try to boot?

If no one takes you up on your offer, plenty of places offer pre-installed Ubuntu computers (Dell and system76 come to mind), and I would also suggest downloading the Ubuntu server installation disk, which comes with a convenient "install omni-server" option.

Edit: Also, it'd be ill-advised to do this, as the install would just install support for the hardware in the machine it was in at install time. If you sent the drive to someone and had them install it, it'd be configured for their machine, which would be useless in your machine, unless you both had the same exact hardware.
