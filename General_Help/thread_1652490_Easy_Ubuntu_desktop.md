You start your post with the words you are a complete noob. Then you continue with virtual box, which is not something fit for a noob. Terminal commands don't work properly.
I think you should start all over again with a clean install AFTER having read something about Ubuntu, so you know what it's about, how it works, at least the basics. Then when installing you will see it does work because now you know how to use it. Stay away from complicated things like Virtual Box until you really get the hang of it. Don't dive into the deep end of the pool when you didn't learn how to swim.
I know it sounds harsh, but it's just a well meant advice.
