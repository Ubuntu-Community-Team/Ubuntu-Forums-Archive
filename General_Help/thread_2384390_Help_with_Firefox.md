### Post by TheFu on 2018-02-06
Many browsers prevent local files from being opened as a security technique.  Browsers run inside a "sandbox" and access to the local file system breaks that security.

Google found this solution: [http://kb.mozillazine.org/Firefox_:_Issues_:_Links_to_Local_Pages_Don%27t_Work](http://kb.mozillazine.org/Firefox_:_Issues_:_Links_to_Local_Pages_Don%27t_Work) - I use firefox as my presentation platform, so I load local files into the browser too.
