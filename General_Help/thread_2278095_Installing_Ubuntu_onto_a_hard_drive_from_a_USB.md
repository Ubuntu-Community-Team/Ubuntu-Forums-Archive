### Post by yancek on 2015-05-13
That is surprising that you were unable to install to a computer that has no operating system on it.  Not having posted any information about the hardware you are using would make it difficult for anyone to do more than guess.  You could boot the Ubuntu installation media and go to the site below and get the boot repair software.  Select the option to Create Bootinfo summary and post that output here.  It will give details on your system and someone should be able to advise you.

[https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)
