### Post by SeijiSensei on 2011-04-11
I've never tried this, but it might be possible to use the Import tool in Thunderbird for this.  If your Windows partition is mounted in Ubuntu, you should be able to see your Thunderbird configuration in your Windows home directory.  If so, try telling the Import tool to import your settings from there.

Otherwise, you might be able to copy the appropriate files and folders from the Windows Thunderbird location into ~/.thunderbird/.

I'm guessing there may be better help for this at mozilla.org.
