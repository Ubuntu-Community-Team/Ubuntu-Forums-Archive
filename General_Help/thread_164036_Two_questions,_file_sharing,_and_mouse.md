### Post by Face1 on 2006-04-22
Do you have an optical mouse?  If so, you'd be surprized by how much your mousing surface (mousepad of lack thereof) affects your ability to use the mouse.  If you aren't using a pad, try a pad.  If you are using a pad, try using a pad with a different color/texture.  If in doubt I always test like this:  I put the mouse on my leg (so the surface is my jeans).  I've noticed that denim makes an exceptionally good mousing surface, so if it is still jumpy then you have a different problem.
