### Post by undoIT on 2012-03-02
I have a bash script I created. When I double-click it, a prompt is displayed with options for "Run in Terminal" "Display" "Cancel" "Run".

I use this script on a regular basis and for convenience sake, I'd like to set the default action to "Run". Is it possible to set the default to "Run" for this script only? If not, is it possible to set it "Run" for all bash scripts?

---

### Post by ajgreeny on 2012-03-02
You can set the default for all executable scripts in nautilus preferences.

---

### Post by stinkeye on 2012-03-02
Set a keyboard shortcut for the script.
Use the path to the script as the command.
