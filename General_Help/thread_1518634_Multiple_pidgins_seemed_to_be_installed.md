### Post by marshmallow1304 on 2010-06-26
There is one Pidgin.  When you run Pidgin with sudo, you are simply loading Pidgin with root's settings and chat logs instead of your own.  Also, it is inadvisable to run any non-administrative program as root unless necessary.
