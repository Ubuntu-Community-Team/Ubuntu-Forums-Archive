Have you tried Playonlinux?
There is a steam-wine-wizard inside..also for Halflife aso)

[http://www.playonlinux.com/en/download.html](http://www.playonlinux.com/en/download.html)

For Karmic:

[B]sudo wget [http://deb.playonlinux.com/playonlinux_karmic.list](http://deb.playonlinux.com/playonlinux_karmic.list) -O /etc/apt/sources.list.d/playonlinux.list
sudo apt-get update

[/B]
