edit: this worked for me:

 If you want to install programs via .deb files, I suggest to use Gdebi.
  It's a lightweight application that is better at installing .deb  packages, and is quicker as well. If there are dependencies, it notifies  that as well.
  In the terminal, type:
  sudo apt install gdebi
  You can right click on .deb file and select to open it with Gdebi.
  If you like Gdeb, you can make it default to have it open all the .deb files in future.
  Read this: [http://itsfoss.com/gdebi-default-ubuntu-software-center/](http://itsfoss.com/gdebi-default-ubuntu-software-center/)

### Post by howefield on 2016-04-26
Presumably you are using the file "*v11.0.57095 (deb 32-Bit / 64-Bit Multiarch)*" downloaded from the teamviewer website ?

Can't tell you what is going on with software centre, but installing gdebi might be a better idea.

```
sudo apt install gdebi
```

then right click on your downloaded .deb package, select properties then Open with tab and choose "Gdebi Package Installer" and set as default. This will use gdebi rather than the software centre for installing local debs.
