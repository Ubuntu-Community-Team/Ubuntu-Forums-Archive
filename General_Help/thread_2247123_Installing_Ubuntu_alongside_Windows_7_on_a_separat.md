### Post by yancek on 2014-10-05
If you want the installer 'forced' to do this, you can select the "Something Else" option in Installation Type which is more of a manual install and you will be able to actually make selections of partitions and see what is happening.  I've never trusted these auto/semi-auto install methods although they usually work if you have a simple layout.  You might take a look at the tutorial at the link below on installing Ubuntu 14.04.  There is a step by step guide if you scroll about halfway down the page:


[http://www.dedoimedo.com/computers/ubuntu-14-04-install-guide.html](http://www.dedoimedo.com/computers/ubuntu-14-04-install-guide.html)

The first part of the tutorial has a lot of useful information.
