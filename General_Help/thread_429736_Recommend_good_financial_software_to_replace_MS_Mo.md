### Post by rmfought on 2007-05-03
I tried both GnuCash and Moneydance, and the problem with all these programs is that they can only import from QIF files, not native MS Money files.  Each account you have has to be exported and then imported separately - I had about 17.  And the export/import don't really work that well - after I finished importing all 17 files, I started looking through my accounts and much of the data was just plain missing.  So be sure it try before you buy.

I currently run Money in a VMWare Server image of Win XP.
