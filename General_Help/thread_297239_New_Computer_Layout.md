### Post by podunk on 2006-11-10
I don't have any SCSI drives - so this is a FWIW thing here. I do know that until a few years ago most web servers had SCSI drives, and most web servers run some form of Linux. So it's an almost for sure Ubuntu will be able to use your drive.

You could try booting off the live CD. When the desktop loads click 
System>Administration>Disks
Is the drive there? Is it identified correctly? If so I would assume that the controller and disk were recognized properly.

How to partition, set up etc I really can't help with.
