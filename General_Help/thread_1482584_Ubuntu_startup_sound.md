### Post by towheedm on 2010-05-13
> **altjx said:**
> I wanna get the startup sound Backtrack plays when logging in :P. Anyone can show me how to change the system startup sound and where I may be able to download that startup sound from?

I'm assuming you mean the sound that plays when logging into the desktop.  To do this click on [COLOR=Red]System => Preferences => Startup Applications[/COLOR].  From the list, select the[COLOR=Red] GNOME Login Sound[/COLOR] and click on [COLOR=Red]Edit[/COLOR].

Now edit the [COLOR=Red]canberra-gtk-play[/COLOR] function in the [COLOR=Red]Command[/COLOR] box.  For more info using the canberra function check the link at the beginning of this thread:
[http://ubuntuforums.org/showthread.php?p=9068333#post9068333](http://ubuntuforums.org/showthread.php?p=9068333#post9068333)

Using the canberra function starts on Pg 41 of the PDF file.

As to the Backtrack sound, have no idea what that is.
