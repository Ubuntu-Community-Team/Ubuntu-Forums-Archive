### Post by beefinit on 2009-05-28
Hi,
I had the problem that my suspend/hibernate functions stopped working after I ran this: 
[http://ubuntuforums.org/showthread.php?t=1130582](http://ubuntuforums.org/showthread.php?t=1130582)
That is, I tried to make my graphics work better. This both prevented me from being able to suspend/hibernate and also prevented me from booting into the server kernel (2.6.28-11 server) to see all of my memory. My previous xorg.conf was:
```
# xorg.conf (X.Org X Window System server configuration file)
#
# This file was generated by dexconf, the Debian X Configuration tool, using.... (I deleted the rest of the comments here)
# again, run the following command:
#   sudo dpkg-reconfigure -phigh xserver-xorg

Section "Monitor"
	Identifier	"Configured Monitor"
EndSection

Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
	SubSection "Display"
		Virtual	5440 2160
	EndSubSection
EndSection

Section "Device"
	Identifier	"Configured Video Device"
	Option		"AccelMethod"			"uxa"
	Option		"EXAOptimizeMigration"		"true"
	Option		"MigrationHeuristic"		"greedy"
	Option		"Tiling"			"true" # i8xx users: set to false

EndSection

```
and it all works if my xorg.conf is:
```
# xorg.conf (X.Org X Window System server configuration file)
#
# This file was generated by dexconf, the Debian X Configuration tool, using... (I deleted the rest of the comments here)
# again, run the following command:
#   sudo dpkg-reconfigure -phigh xserver-xorg

Section "Device"
	Identifier	"Configured Video Device"
EndSection

Section "Monitor"
	Identifier	"Configured Monitor"
EndSection

Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
EndSection
```

So, I don't know if that helps anyone, but it sure helped me figure out how to navigate between the server kernel (where I can't suspend/hibernate), and the generic kernel, where I can, but lose 3d graphics abilities
