### Post by Paddy Landau on 2017-04-03
Have you set the printer's duplex status to on?
HPLIP Toolbox > Print Settings > Installable Options > Duplexer Installed

You might find extra help on [HP's Linux page for the F2400 series]("http://hplipopensource.com/hplip-web/models/deskjet_aio/deskjet_f2400_series.html").
