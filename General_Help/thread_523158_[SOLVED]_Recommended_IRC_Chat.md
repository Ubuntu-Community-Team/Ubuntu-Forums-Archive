### Post by strabes on 2007-08-11
An "all-in-one" program like gaim/pidgin is a program that is designed to connect to multiple instant messaging protocols (AIM, MSN, jabber, IRC, etc)

Stand alone just means the program is designed for one protocol, like Xchat, konversation (my favorite), lostirc, and the proprietary IM abominations like AIM triton.
