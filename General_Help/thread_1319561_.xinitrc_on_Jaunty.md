### Post by oscurochu on 2009-11-08
Try this:

```
$ chmod +x ~/.xinitrc
```

Also, the first line of your .xinitrc should usually contain this:
```
#!/bin/bash
```

So your .xinitrc should look like this:
```
#!/bin/bash
xset m 1 1
```
