### Post by aysiu on 2007-04-23
Are you using Reconstructor? Just curious.

I'd say...

**Thunderbird** instead of Evolution (I was used to using Thunderbird in Windows, and I like that it's cross-platform... can be used on Mac, too)

**Goobox** instead of Sound Juicer (no need to tweak MP3 profile if you have the proper codecs installed)

**FireFTP** extension in Firefox. Not sure if gFTP is installed by default, but it sure is recommended a lot. Coming from FileZilla in Windows (FileZilla in Ubuntu is not the same), I found gFTP's interface confusing. FireFTP made me at home.

Otherwise, I think things are pretty good. When I first started in Hoary, Rhythmbox *sucked*, but now it's my preferred music player. GIMP is pretty good. Sometimes I prefer the simplicity of Kolourpaint, but that is a QT application--not ideal for Gnome.
