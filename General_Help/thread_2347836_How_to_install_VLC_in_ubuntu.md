### Post by howefield on 2016-12-29
> **peter d said:**
> It's in the software centre.

It's slightly more confusing than that, there are two versions of VLC in the Software Centre, one the traditional deb package and the other a snap package of the daily version.

For someone like the OP who is familiar with the command line, might be easier to use the terminal with..

```
sudo apt install vlc
```
