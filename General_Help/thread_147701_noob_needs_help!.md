Until here nothing's shocking, it's just the input (kboard and mouse) and font configuration.

Now it comes the "trick"
```

Section "Device"
	Identifier	"NVIDIA Corporation NV34M [GeForce FX Go5200]"
	Driver		"nvidia"
	BusID		"PCI:1:0:0"
	Option          "NoLogo"        "true"
	Option          "TwinView"      "true"
#Xgl
#       Option          "RenderAccel"           "true"
#       Option          "AllowGLXWithComposite" "true"
#Xgl
        Option          "Metamodes"     "CRT-0: 1024x768, DFP-0: 1400x1050"
        Option          "HorizSync"     "CRT-0: 50-110;  DFP-0: 40-70"
        Option          "VertRefresh"   "CRT-0: 60-120;  DFP-0: 60"
        Option          "TwinViewOrientation"      "LeftOf"
EndSection

```
The twinview option has to set to true, then 
"Metamodes": defines the monitors you're using and their max resolution. I have one DFP (liquid crystal) and a CRT. 
"HorizSync": defines the horizontal syncro for both monitors
"VertRefresh": same as above, but for vertical refresh
"TwinViewOrientation": defines the position of the 2nd monitor respect to the first (usually a CRT is considered first. Don't ask me for is a X server issue ;))
```

Section "Monitor"
	Identifier	"Monitor Generico"
	Option		"DPMS"
	HorizSync	28-70
	VertRefresh	43-60
EndSection

Section "Screen"
	Identifier	"Default Screen"
	Device		"NVIDIA Corporation NV34M [GeForce FX Go5200]"
	Monitor		"Monitor Generico"
	DefaultDepth	16
	SubSection "Display"
		Depth		1
		Modes		"1400x1050"
	EndSubSection
	SubSection "Display"
		Depth		4
		Modes		"1400x1050"
	EndSubSection
	SubSection "Display"
		Depth		8
		Modes		"1400x1050"
	EndSubSection
	SubSection "Display"
		Depth		15
		Modes		"1400x1050"
	EndSubSection
	SubSection "Display"
		Depth		16
		Modes		"1400x1050"
	EndSubSection
	SubSection "Display"
		Depth		24
		Modes		"1400x1050"
	EndSubSection
EndSection

```
In the "Modes" line you can define as many resolution as you wish (and above all your main monitor supports)

```


Section "ServerLayout"
	Identifier	"Default Layout"
	Screen		"Default Screen"
	InputDevice	"Generic Keyboard"
	InputDevice	"Configured Mouse"
	InputDevice	"Synaptics Touchpad"
EndSection

Section "DRI"
	Mode	0666
EndSection
```

Then have a look to the [nvidia README]("http://download.nvidia.com/XFree86/Linux-x86/1.0-8178/README/index.html") it's full of good tips
