### Post by heyup on 2011-05-16
Default email application is Evolution.

From Opera -> Preferences -> Advance -> Programs -> -> mailto -> Edit 

When I select 'Open with default application' it won't open Evolution.

How do I set Opera to mailto from Evolution. I have no problem with this using Firefox.

---

### Post by TeoBigusGeekus on 2011-05-16
Give 
```
which evolution
```
to find the full path of the evolution executable, choose "Open with other application" in Opera mailto settings, paste evolution's path in there and you're done.
