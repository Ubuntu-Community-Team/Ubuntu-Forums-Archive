### Post by o_swas on 2007-11-25
OK, fixed.  Using the information found here:

[http://ubuntuforums.org/showthread.php?t=467335](http://ubuntuforums.org/showthread.php?t=467335)

At the login screen, I used Ctrl-Alt-F2 to login via the command prompts.  Then, despite my limited experience with the Linux command line (I do use the command line in Windoze, however) I was able to delete a big file.  I then rebooted, was able to login normally, and empty the Trash from there.

Thank you for your help!!
