### Post by yasir.elsharif on 2010-12-08
Hello,
It's installed in this directory
/opt/IBM/SPSS/Statistics/19/bin
and it doesn't show up in the menu but you can either run it from the command:
/opt/IBM/SPSS/Statistics/19/bin/stats
or just add a link to the menu by right clicking on the gnome menu and select edit menus and the go to either "Others" or "Science" or whatever menu group and add the link
