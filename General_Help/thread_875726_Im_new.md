### Post by rossjman1 on 2008-07-31
If you're proficient in Windows, then you shouldn't really have a problem learning Linux. Installing programs seems to be one of the biggest differences. You can install some basic stuff by the Add/Remove (Applications > Add/Remove) and once you know what you're doing you can use Synaptic (System > Administration > Synaptic). Linux doesn't need virus/malware protection and it already has a built-in firewall. If you would like to configure the firewall, you can use firestarter. Just install it using the techniques I just described. Good luck!
