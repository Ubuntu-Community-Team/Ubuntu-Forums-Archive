### Post by Dennis N on 2020-08-04
> You will notice a column on the left of the screen that has links to stuff like firefox, file manager, amazon and so on. How do I change the width and behavior of that column
Go to:

System Tools > Settings > Appearance

Under 'Dock', change the icon size to a smaller number to reduce the width.
Turn on 'Autohide' if you want it hidden when windows overlap it, like maximizing a window.

To change color, length, position and more, you need to install the gnome-shell extension dash-to-dock which will replace it.
