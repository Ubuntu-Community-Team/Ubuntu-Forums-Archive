### Post by mali2297 on 2007-12-08
If you are new to Linux, I suggest you use a more beginner-friendly editor, like **nano**. That is, change vim for nano in the instructions you have. In nano, you can save/write out a file by pressing Ctrl-o. Then you can quit with Ctrl-x.
