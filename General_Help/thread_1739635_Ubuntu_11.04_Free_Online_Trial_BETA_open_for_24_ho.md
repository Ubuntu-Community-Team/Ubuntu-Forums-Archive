### Post by twtgd09 on 2011-04-26
Hey, I don't know where to put my feedback, so i'll post it here and you can move it / delete it after you've read it. 

1. Install - I can't for the life of me figure out whats going on here, when i go with "install alongside windows"  it tells me to adjust some sliders to change the size of the partition or somethign like that, then i need to go back and do it again, or it splits it twice. Either way, i need to set the ubuntu partition size thing all the way up. I'd like to see ubuntu install simplified to "click the partition your'd like to install ubuntu on" and have it get installed there without creating virtual discs for storage or install or whatever. 

2. Interface - I don't love unity, the lack of an ability to adjust the positions of things and add icons to a "panel" is a big turn off for me. Also, the dock looks clunky. I think that the idea of storing the close minimize maximize buttons in the top "panel" along with file, menu, etc. is neat, but the little icon for the dock looks kind of ugly. Also, i dont like the "tabs" design, and would prefer if pulling up the dock (by moving to one corner of the screen) and clicking on an applicatios icon, i could switch through my maximized windows.

I figured out how to get gnome panel open, but i'd prefer if there was a more obvious way to do this than the little combo box at the bottom of the screen on login. Also, when you make a gnome panel thicker, there is a "lighting" effect that makes it so it doesnt look consistent. If you could code something in there that on a panel, if the "top" color is 000000, then when you expand the panel, the space between the previous "top" and the top of the screen is painted all black, if you follow what im saying. 

3. Software -
Installing on 11.04 blows, in 10.10, if you needed to install a package (compiz, well say) and it required extra components, ubuntu software center would just install those extra components, but that doesnt consistently work in 11.04, and you need to manually install all of the components. Also, compiz wont install at all, and while i respect the choice to switch to a lighter system, i'd appreciate it if it was *possible* to install compiz, which many people like for its appearance.

Also, installing in general is my least favorite thing about ubuntu, with windows, you download an exe, and you click through the install without opening any other applications up. Ubuntu on the other hand, has six methods to install things the last time i checked, and quite frankly, its annoying. First, ubuntu software center, i love that you can queue up multiple items to be installed, then have the computer run through all of them. BUT there's also the issue of installing .debs, if i have it on my desktop, i just want to click through the install because quite frankly, software center takes some time to boot up. Then you need to worry about ppa's. I HATE ppas, only 2/3 of the ppas ive tried to use have worked, and if something gets screwed up in the software sources list, there is no way to reset it to defaults. then there is the command line interface, which to my knowledge, has software neither software center, nor ppa, nor package manage, nor the internet has for "easy" install. And then we have installing through the package manager. Why does package manager need to be separate from software center? I'd love to be able to go into software center, click "banshee" and have a list the actual program, and all of the compatible/necessary plugins, rather than needing to download them manually (particularly the decoders). Finally, theres compiling source code, which at the very least, id like an application for, but at the very most, id like to see it "banished", its terribly confusing for noobs and it takes forever. 

A third complaint about ubuntu software, theres wayy too much preloaded on the install disc. I'd like to see a "bare bones" version of ubuntu released that minimizes install time, minimizes install time, and maximizes system performance. Seriously, it really bugs me that you guys pack things like a "help" application or a scanning program or sound recorder or even the printing thing in with ubuntu when you could use that data to add features to ubuntu or to make the system faster. 

Fourth, some of the pre installed software seems kind of crappy to me, for example, "sound recorder" which would be much better replaced by audacity, "simple scan" which is much more effectively replaced by xsane which has compatibility with gimp.

Finally, WINE sucks, i would be thrilled if you could introduce compatibility with .exe by default on the next iteration of ubuntu, rather than the clunky wine system.

Thanks
