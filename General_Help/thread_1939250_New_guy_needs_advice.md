### Post by howefield on 2012-03-11
If it were mine and in the absence of a direct question, I'd be looking at the following steps..

Backup all data that would cause sorrow if lost, run some sort of housekeeping application, even if it is just ccleaner (which is fairly mild and benign).

Defragment the Vista file system, create partitions for Ubuntu and take a disk image as "insurance" against disaster with something like Clonezilla.

Finally install Ubuntu and enjoy.

### Post by flemur13013 on 2012-03-11
> I tried Puppy Linux one time with much help from friend and bungled up my computer while doing so.Puppy's rambling, semi-coherent documentation is even worse than that of most other software, so don't feel too bad.

Try Ubuntu 10.04, it seems to be the most reliable *buntu version and works on the most systems.  Other linux versions are less like Windows in what they supply as far as software and GUI.

The normal dual-boot install causes major problems for many people (see this very forum, for example! The ubuntu installer is pretty weak, yet still better than most other linux installers, all of which are far superior to any Windows OS installer), so you might try installing on a separate disk or some USB gizmo unless you don't mind trashing Windows. (If possible, disconnect your main/windows disk while doing this...)

Edit: backup your stuff first!
