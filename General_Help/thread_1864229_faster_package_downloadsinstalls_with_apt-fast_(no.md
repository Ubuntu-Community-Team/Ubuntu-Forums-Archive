faster package downloads/installs with apt-fast (now with autocompletion, action preview and installer script):

I recently installed from scratch the minimal install which is often followed by installing a large number of packages with a large number of dependencies and the download time can sometimes be obnoxious.  The apt-fast script here seemed to be a step in the right direction, but at least the copy I got didn't do everything I wanted it to.  So now it does autocompletion of both actions (i.e. install, remove...) and packages names, it also tells you what packages would be installed/removed before taking any action and also includes a handy installer.  You can find it [here]("http://astoryworthtelling.wordpress.com/2011/10/18/apt-fast-with-preview-autocompletion-and-installer-script/") (end of the post).  I encourage you to look at the code before you run it and understand what it does, I claim no responsibility if you somehow break your system (which isn't unlikely given that it only runs with sudo permissions).
