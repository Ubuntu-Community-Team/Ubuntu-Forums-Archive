### Post by jkxx on 2010-04-17
WineHQ has you covered!

The Wine AppDB (at [http://appdb.winehq.org/]("http://appdb.winehq.org/")) will give you an idea if a particular program will run and typically how well it does.

For things like web browsers, e-mail, and office you'll be better off using the native apps that come with *Ubuntu. Wine mostly helps with 3D games - for any other app, try to find an alternative first.
