### Post by hakermania on 2011-07-17
The easiest way is to have pre-installed windows in your system and then
1) Boot from CD
2) install alongside Windows

After this you'll have options on boot where you'd like to boot (windows or ubuntu linux)
This is the easiest way.
Unfortunately the switching cannot be easier than this: Reboot to change OS...

The other solution is to install ubuntu in VirtualBox using Windows. Download VirtualBox inside windows and install there ubuntu
This requires a lot of RAM and CPU maybe and also ubuntu won't run perfectly and in full performance of course. But this way has a much easier switching.
