### Post by tombott on 2007-07-27
Are you using KDE or Gnome as your Desktop environment?

Did you install Ubuntu or Kubuntu?

If you are using Gnome then you need to do the following:

Navigate to System - Preferences - Keyboard

The following should be displayed:

[IMG]http://freetowatchtv.co.uk/images/ubuntu/KB1.png[/IMG]

Click on Layouts

[IMG]http://freetowatchtv.co.uk/images/ubuntu/KB2.png[/IMG]

As you can see mine shows 'United Kingdom'

Click Add

[IMG]http://freetowatchtv.co.uk/images/ubuntu/KB3.png[/IMG]

Then scroll down to United Kingdom and select the Keyboard that matches your layout, just highlighting United Kindom as the above image shows should be enough. Click ok then select your new keyboard and set to default.

You may need to reboot for the changes to take effect.

Hope that helps.

Tom.
