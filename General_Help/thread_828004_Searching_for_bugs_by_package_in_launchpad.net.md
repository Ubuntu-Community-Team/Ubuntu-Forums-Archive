### Post by ngolian on 2008-06-13
Is there a way to browse/search bugs in specific packages in launchpad.net? I try to check existing bugs in case of duplicates before reporting new ones, but launchpad.net doesn't seem to provide any sort of package filter. If you try searching for a package name you also get loads of irrelevant bugs. The search term seems to use OR instead of AND (even with hyphens) so the more specific you try to make the term the more general it actually becomes.

---

### Post by Rocket2DMn on 2008-06-13
If you go to [https://bugs.launchpad.net/ubuntu](https://bugs.launchpad.net/ubuntu) you can click the Tags link on the left (a gray area) and click on the appropriate package.  Otherwise click Advanced Search and add the package to the Tags option near that bottom.
