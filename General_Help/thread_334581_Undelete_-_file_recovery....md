### Post by teaker1s on 2007-01-09
the question would be-has the data on hard drive just been in effect deleted. By this i mean that the space allocated now shows 0byte but the physical data hasn't been wiped until the operating system uses the in effect free space for saving files=recoverable

if the file has been deleted and the area the operating system now considers empty has been overwritten=at least partially corrupted and most likely unrecoverable
