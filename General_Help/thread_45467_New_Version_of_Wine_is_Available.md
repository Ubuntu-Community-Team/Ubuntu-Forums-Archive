### Post by rachii on 2005-06-30
are you using winecvs or just wine?   if you are using winecvs then just run the script again and tell it to upgrade.  if not then yes you should remove the old version before upgrading, using apt or synaptic if you downloaded the package, or doing ```
make uninstall 
```if you compiled the source.
