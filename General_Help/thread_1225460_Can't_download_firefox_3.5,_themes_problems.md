### Post by speedwell68 on 2009-07-28
> **Hydrid said:**
> Wait it to be updated from your update manager! I think its a compatibility issue with 9.04

AFAIK Firefox 3.5 will not be added to the repositories until 9.10, there is no compatibility issues.  If you go into Synaptic you can install FF 3.5 as a separate package, but the default 3.0.xx version will not be replaced by 3.5.  The 3.5 package in the repositories will be branded as Shiretoko.  The only reliable way I have found of installing FF 3.5 is downloading it directly from the Mozilla website.
