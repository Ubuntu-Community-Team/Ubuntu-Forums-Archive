### Post by SPazzo on 2010-01-24
OK, Ubuntu can't run Windows programs, so you use WINE.  WINE is a program that *may* allow you to run a Windows program.  It works on some programs, but not on others. 

A few things that may help you.  [The WINE FAQ   ]("http://wiki.winehq.org/FAQ")Which, hopefully should answer some questions.  And a [tutorial on installing and running]("http://www.psychocats.net/ubuntu/wine") WINE.  Also, the [WINEHQ Application Database]("http://appdb.winehq.org/") has a list of programs that should work OK.

Last but not least, try going to *Applications* (top left hand corner), then to *Ubuntu Software Centre*.  A window will open with all sorts of software that **WILL** run on Ubuntu.  There are some games on that list.  If WINE doesn't work for your game, at least you can play others.

Hope that helps! :D
