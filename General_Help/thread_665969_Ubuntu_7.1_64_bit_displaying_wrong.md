### Post by Rhubarb on 2008-01-12
It could be that your screen resolution is set very small.
To make it bigger System --> Preferences --> Screen Resolution.

It could be that your fonts and icons are too big (I don't think this is what you're experiencing).
System --> Preferences --> Fonts.  (Or Appearances, then select the fonts tab if you're using ubuntu 7.10)
