### Post by amo-ej1 on 2008-02-11
Simple, you see that entire list of packages that it wanted to uninstall, select it all from alacarte until zenity, paste it in a textfile, call it for example /tmp/bla.txt 

then type 

```

for i in `cat bla.txt`; do sudo apt-get install $i; done 

```

this will reinstall each individual package again on the system.

What actually cause this problem ? well 'lamp-server' is a so called metapackage, is include a whole set of other packages, by uninstalling the metapackage you uninstall all package it contained, not all packages it installed 'extra'.
