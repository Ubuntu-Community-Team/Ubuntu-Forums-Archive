### Post by Metacarpal on 2007-06-29
Hi.  Also a high-school drop-out.  Currently working as a Linux systems administrator for one of the stock exchanges.  Just wanted to offer a little encouragement there.

That said, though programming is somewhat mathematically based, it's evolved to the point where it's more like learning a new written language instead.  This is great because, as with all languages, you can start out "speaking" a little and still be part of the programming community.  As you practice and learn, you'll get to coding like a native in no time.

That said, I'd say the place you'll want to start is with Linux commands in general (to know how to make it move).  I can suggest an excellent book if you like.

For the more user-oriented end of programming, I believe Python is the programming language of choice, at least for Ubuntu's Gnome desktop environment.  I've not gotten into it myself, but I've heard it's easy to learn.
