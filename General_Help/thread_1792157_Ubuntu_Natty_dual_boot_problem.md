### Post by garvinrick4 on 2011-06-27
If you can go into Ubuntu install and run this and post:
[SourceForge.net: Boot Info Script - Project Web Hosting - Open Source Software]("http://bootinfoscript.sourceforge.net/")
Download this zip file to Desktop:
right click on it open with archive manager and extract:
Now open a terminal and run this below will put a RESULTS file on Desktop copy and paste
here. Highlight whole thing and click on # sign in upper right of Message box and wil
put in a nice little box to read in.
```

sudo bash ~/Desktop/boot_info_script.sh
```
