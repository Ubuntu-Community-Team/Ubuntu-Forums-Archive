### Post by ajgreeny on 2007-11-07
Not sure about ubuntu but if you are prepared to install kcontrol from the kde desktop you can do it with that.  Go to System administration > Font installer and there you can uninstall, though you will need to click the Administrator Mode button for system-wide fonts.  Gnome really needs something similar.  You can then even uninstall kcontrol afterwards if you don't like keeping it.  It installs a lot of lib files as well, but may be worth it.  I have no idea what happens if you simply delete the font files using** gksudo nautilus** which opens nautilus as root, but others may be able to tell you.
