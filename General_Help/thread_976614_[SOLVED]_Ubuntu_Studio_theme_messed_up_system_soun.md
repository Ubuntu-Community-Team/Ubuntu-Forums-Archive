### Post by tokyo77 on 2008-11-09
I just installed the Ubuntu studio theme (the complete package with the sounds, icons, theme, usplash, etc.) and I didn't like the sounds, the usplash, and the login, so I used Synaptic to uninstall them, and reinstalled gnome-audio. However, when I go to the System Sounds, everything's set to "none" except for login/logout. The login/logout sounds are very strange and unappealing to me, but I don't know how to set everything back to the default, with the drumroll login and stuff. Help would be appreciated!

---

### Post by tokyo77 on 2008-11-10
I found out my mistake.

The drum roll was in the package ubuntu-sounds, not gnome-audio. When I installed ubuntu-sounds, the good old familiar drum roll was back. :)
