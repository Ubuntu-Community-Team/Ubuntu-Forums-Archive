### Post by Trab on 2006-02-22
obviously a larger HDD is better, but i ran ubuntu on a 10 gig for a year... heres my recommendation:
500mb-1gb swap (depending on what you do, but minimum of 500mb swap)
5 Gb for /home (i absolutely recommend seperate system and user files... should the system die, u can reinstall from Breezy disc without losing EVERYTHING as config files and everything else is saved in /home)
4 Gb for / (i ended up using wayyy more space in /home than / even with a ton of programs installed...)

i would use Automatix to install Firefox 1.5

definetly check out GAIM, unless ur sister ONLY uses MSN (aMSN might have more specific stuff for MSN, i dunno, i <3 gaim)

Rhytmbox is the BEST mp3 player/music library ive found, but i also like to have Totem-xine installed for other things... make sure to get all the codecs u need... Also get EasyTag (if ur sister or u care about properly labeling mp3s, i hate it when they are wrong)

install FrostWire (free GPL version of Limewire, im very happy with it, i found no diference between this and limewire for linux cept no adds in this one...)

use AutoMatix to install anything else u need...

if they are converts from Windows, i would also sugest some Automatix options (ctrl-alt-del, numlock, cd eject etc...) and installing Klipper (set it up in sessons to start at boot) a very useful clipboard manager (for text only) as i find the built-in gnome one... sucks...badly. 

any other questions, PM me, ive used ubuntu on a 10 gig for a year, and set it up for my 8 year old brother to use... :-D


good luck
-Trab

### Post by nerval on 2006-02-22
For Dad : 
Openoffice + Abiword
Mplayer + Codecs
Firebird + Opera

For Sisters:
Frostwire
Gaim
(Flash add-in to the browser you are using)

For Music:
Ask your sisters to burn their mp3s into a CD. Each CD can carry 120 songs or so, that'll cover their needs. 

For Powerpoints:
Koffice and OpenOffice have nice Powerpoint-like versions. Pointless is another nice tool to do such job. Yet, it is hard to do all things Powerpoint can.

Second option:
I installed Ubuntu on my mother's Sony Vaio laptop (10 gb hd as well); and later on I've also installed Kubuntu. Somehow she is much more comfortable using KDE rather than Gnome.

If you use Kubuntu; you can use Koffice instead of OpenOffice, Kopete instead of Gaim, and KMplayer instead of Mplayer. Kmail instead of Firebird, and Kopete instead of Firefox or Opera.

My mom thinks all KDE programs are much more user-friendly and better-looking. Well, KDE is more eye-candy and beginner-friendly; yet I'm quite used to Gnome.

So, good luck. ;)
