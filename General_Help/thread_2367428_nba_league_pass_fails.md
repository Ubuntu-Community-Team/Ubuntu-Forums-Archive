Try using a "user agent switcher" for your browser. Many websites check which browser and operating system you are using, in order to tailor content for you. You can fake the operating system and browser with an user agent switcher plugin. Say, if you are using Opera in Ubuntu, you can change the user agent line, and report that you are using Firefox 54 in Windows 7. Maybe then it will work.

Some sites may claim not being Linux compatible, but probably they don't have any problems with it. Netflix comes to mind. You have to change the user agent, and then it works alright. The compatibility stems from using proper web standards like HTML,CSS, Javascript, and MPEG-4. The choice of operating system should not be a problem for most websites.

The real problem is if the NBA website decides using something very Windows specific, like a particular program to handle Digital Rights Management, or proprietary video codecs, and whatnot. Then there may be no way to access their content but using Windows.
