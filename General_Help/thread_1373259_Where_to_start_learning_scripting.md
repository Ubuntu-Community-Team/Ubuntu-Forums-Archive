### Post by steviefordi on 2010-01-07
You might want to try following the tutorial on this site:

[http://linuxcommand.org/]("http://linuxcommand.org/")

It starts with just commands but progresses to shell scripting. This will give you a good starting point.

Once you know the basics you can use the abs guide ([http://tldp.org/LDP/abs/abs-guide.pdf]("http://tldp.org/LDP/abs/abs-guide.pdf")) for reference.

The main thing is to get a leg up and then write some scripts. You can read all you want, but you learn mostly by doing.

Hope this helps.
