### Post by wmblowers on 2007-02-08
> I'm curious, can Ultimate Ubuntu run as a LiveCD? Like, could I play around with all the features in a live environment before I install it?

Yes, you can.  I have installed the Ubuntu Ultimate 1.1.  It's running quite well on my system.  Be aware, however, that I ran a virus scan with Aegis after the install and the post-install updates and it showed two viruses in directories related to Wine and Picasa.  I'm sorry, but I didn't write down the virus name; I did, however write down the names of the two files affected.  One was 'appshell.dll', and the other was 'vp31vfw.dll'.  Since I don't need Wine or Picasa either one, I simply wiped these packages and my computer is now virus-free.

There was speculation that these hits may have been false alarms.  I tend to doubt that since they were actually present in Windows-type modules; however, they had no effect on my machine, other than being reported.  I still wanted them out.

This does not stop me from enthusiastically recommending the Ultimate edition, however.  A small caution:  it is based on Edgy, and there have been a few very minor glitches; on a couple of occasions a program has behaved unexpectedly, and on one occasion a program hung and forced me to kill it; IMO, these are minor inconveniences to live on the 'bleeding edge'.  So far, no show stoppers.

Also:  note that the Ultimate (Live) CD allows you to install for dual boot.  I actually did this at first with my current Edgy install.  It took what seemed a long time to perform the partition work, but then the install went the rest of the way.  After I played with it for a day I decided to take the big plunge, wipe out the drive and install Ultimate by itself.  So far, overall, I'm extremely pleased.  Along with the caveats mentioned above, I highly recommend it.

WLB  :>)

PS:  I'm not sure where it can currently be d/l from... I'd simply Google "Ubuntu Ultimate" for more info.
