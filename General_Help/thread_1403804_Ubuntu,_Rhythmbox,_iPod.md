### Post by colobix on 2010-02-10
you gonna have to jailbreak your ipod in order to use it on linux.
You must have a windows pc or a mac in order to do this.
Check out [http://blackra1n.com](http://blackra1n.com).
I did this on my ipod touch for a few months ago.
You see, microsoft made an under-the-table deal with apple to only make iTunes for windows, and if you dont jailbreak it you're gonna have to use iTunes to manage your ipod.
If you want more info on how to do this, go check out [www.youtube.com/ireviewjust4you](www.youtube.com/ireviewjust4you).

Good luck.
OH and btw, if you jailbreak it you must check the ipod's software version first. otherwise you can end up with an ipod that doesnt work.
