### Post by WorMzy on 2010-11-09
I use deluge. It has a scheduler like uTorrent, can store torrents, can download to one folder, then move the completed files to another folder, has a web interface, and several other features like a blocklist, audo-add folders, an auto-extractor, etc.

When I tried it (a few years back) uTorrent worked well with wine, but I'd still recommend using a native client.

### Post by fuzzyworbles on 2010-11-10
