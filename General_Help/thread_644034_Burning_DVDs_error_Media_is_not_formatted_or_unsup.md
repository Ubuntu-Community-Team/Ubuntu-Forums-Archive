### Post by danwood76 on 2007-12-18
all DVD writers have like librarys of codes of media types, and each disk has a small sector at the start of the disk which gives the disk information, the code is then used to calculate maximum speeds the disk can burn at.

You might need to update the firmware of your DVD writer so that it supports your new media.
