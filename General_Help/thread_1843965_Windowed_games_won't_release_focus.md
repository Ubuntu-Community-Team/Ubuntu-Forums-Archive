### Post by _d_ on 2011-09-14
Ubuntu 11.04 64bit, using latest ATI Catalyst 11.8.

When playing games in window mode, they don't seem to want to release focus when I try to navigate to something else.

For example, when playing Warsow in window mode, and I press the tilde (~) to open console to release mouse lock to navigate to Firefox the game will force focus on me and won't move behind the Firefox window or any other for that matter.

When I release mouse lock, and click the MINIMIZE button it usually ends in the game completely dying forcing me to have to kill it via terminal.

I hope this makes sense, and wondering how I can fix it or work around it.

**EDIT: Using workspaces works wonders and has solved all my problems. Switching between workspaces is definitely the way to go if anyone else happens to have this issue.**
