### Post by PeterJS on 2008-01-04
In Konqueror if you right click on a file and select properties, in the properties dialog next to the line that has the file type there's a yellow wrench icon. This will open the properties dialog for that file type, in the upper left corner there's a button that displays the icon for that file type, if you click on that you can change the icon to what ever you want.
