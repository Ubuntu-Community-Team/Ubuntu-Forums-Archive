### Post by keljaden on 2010-04-06
When a print a paper for school, my HP Deskjet 5440 prints page 1 first (then 2,3,4, etc).  I want it to print page one last (ex: pg 4, 3 ,2 ,1) so when i print a paper I do not have to reverse the order they are in. Is there a simple setting, or is my printer not able to do this?

Thanks in advance.

---

### Post by Morbius1 on 2010-04-06
Do you have **hplip-gui **installed?

If you do then under Print Settings > Basic > Output Order

You have the option of :

Print first page first
Print last page first
