Have you tried
```

sudo nano /etc/default/grub
```
add
```
radeon.audio=1
```
to the end of ```
GRUB_CMDLINE_LINUX=
``` (between the double quotes, with a space in between)

Press control+x to save
Run
```
sudo update-grub
``` and reboot to ssee if it works.

Else, see [https://bugs.launchpad.net/ubuntu/+source/linux/+bug/1169984](https://bugs.launchpad.net/ubuntu/+source/linux/+bug/1169984)
