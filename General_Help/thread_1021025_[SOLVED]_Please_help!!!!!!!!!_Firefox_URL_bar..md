### Post by LoneWolfJack on 2008-12-25
you're not giving much detail, so I assume you recently started using firefox 3 and your problem is the so-called "awesome bar". this thing is an annoyance and its a shame they didn't put an option toggle in the settings to disable it.

there is, however, a firefox addon called "oldbar" that will give you back the old firefox 2 url bar (the url bar shouldn't be a search bar, after 
all).

you can also turn it off through the "about:config" page in firefox:
set browser.urlbar.matchOnlyTyped to true
set browser.urlbar.maxRichResults to 0
