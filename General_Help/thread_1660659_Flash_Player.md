### Post by lovinglinux on 2011-01-05
> **FinalMan said:**
> The only problem is I am using Chrome...

No problem. If you still have Firefox installed, just use it to run the extension once.

If you are using 32bit, then you will need to disable the flash version shipped with Chrome. To do that, type **about[noparse]:[/noparse]plugins** in Chrome's address bar, then click the "Details" option on the top-right, then scroll down to "Shockwave Flash" section and disable the plugin that has **/opt/google/chrome/libgcflashplayer.so** as location. Restart Chrome and it will begin using the version installed by Flash-Aid.
