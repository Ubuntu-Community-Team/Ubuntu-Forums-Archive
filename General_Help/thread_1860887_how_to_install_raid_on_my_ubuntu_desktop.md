### Post by DeMus on 2011-10-15
Raid is something you setup before installing the OS. There are two sorts of RAID installations: hardware and software. The choice depends on your motherboard. If it can handle the hardware sort then choose this, if not use the software version.
There are plenty of websites about this, just google around.
For installing Ubuntu after having created the RAID, use the alternate CD in which you can really setup the RAID.
