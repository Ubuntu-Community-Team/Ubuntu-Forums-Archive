### Post by taurus on 2008-12-20
You have both feisty and gutsy repos mixing in with intrepid!  You are only asking for trouble if you do that.  Therefore, you need to edit /etc/apt/sources.list

```
gksudo gedit /etc/apt/sources.list
```
and either remove the repos that have gutsy and feisty in them or put a # in front of those lines to comment them out.  Save it and then run

```
sudo apt-get update
sudo apt-get upgrade
```
