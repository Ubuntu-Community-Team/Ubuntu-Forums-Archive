### Post by Mark Phelps on 2009-12-02
You will not be able to use the CD that came with the netgear device because the driver and associated SW are for MS Windows, not for Linux.

You generally configure network devices in Linux by plugging them into your desktop via a network cable, launching a browser, and entering the default address of the device -- typically 192.168.0.1.

You will need to check the documentation that came with the device to see what default IP address it uses.
