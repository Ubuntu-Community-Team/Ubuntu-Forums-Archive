### Post by yabbadabbadont on 2009-09-17
For something like Visual Studio, yes, you would have to install Windows inside of a virtual machine (Virtualbox, VMWare, etc) and then install VS in that.  For some software, it is possible to use WINE to run it outside of a virtual machine.

See the following for details, especially their AppDB for a list of applications and how well they run under WINE:

[http://www.winehq.org/](http://www.winehq.org/)
