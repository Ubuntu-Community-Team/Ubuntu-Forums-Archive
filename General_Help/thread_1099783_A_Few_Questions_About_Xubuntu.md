### Post by mb_webguy on 2009-03-18
The LiveCD is pretty static since it's running from a read-only device.  You can't do much as far as customization, and since it's non-persistent, any changes you do make will be lost as soon as you reboot.  If you want a persistent installation, don't want to install to your computer, and have a spare USB drive at least 4GB in size, you can create a USB boot drive using the Ubuntu Live USB creator (usb-creator).  It currently only installs Ubuntu to the drive, but since it's a persistent installation, once it's installed you should be able to remove ubuntu-desktop and install xubuntu-desktop.

### Post by mb_webguy on 2009-03-18
I think USB 1.0 has a transfer rate of about 12Mbps, while an 8x CD drive has a transfer rate of about 9.6Mbps.  So a LiveUSB connected via USB 1.0 will be very slightly faster than a LiveCD on a drive with 8x read speed.  A disc drive with a higher read speed will be slightly faster than the LiveUSB.  

USB 2.0 has a transfer rate of 480Mbps, so a LiveUSB connected via USB 2.0 will be *much* faster.
