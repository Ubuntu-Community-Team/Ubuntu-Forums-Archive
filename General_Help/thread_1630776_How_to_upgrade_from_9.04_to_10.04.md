Sorry, but you have to install 9.10 first over your 9.04. After that you can do the same with 10.04, over 9.10.
It always works one step at the time. You can not skip a version.
Still, a clean install is always better since it gives you less problems. So if you don't have too many programs installed, which you need to install again, or settings which you have changed, I would suggest to do a clean install.

You can update through the Update Manager...just go to the settings and make sure "Long Term" is checked...however...the BEST way to do this, as there will an 80% probability that you will encounter problems "upgrading", is to back up the home directory and do a fresh install...you will save yourself headaches...;)
