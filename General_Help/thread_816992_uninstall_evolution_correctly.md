### Post by me208 on 2008-06-04
if you want a GUI you can go to Applications>>Add/Remove Programs search for evolution and uncheck the checkbox then click "Apply Changes". 
If you are okay with using a terminal you can simply type 
```
sudo apt-get remove evolution
```
which is a lot faster.

Hope this helps!!!
