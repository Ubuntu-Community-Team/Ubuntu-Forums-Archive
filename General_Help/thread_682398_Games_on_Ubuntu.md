Welcome to Ubuntu. Linux is great, but dont expect it to work without some setup, games will be one area that a lot of setup will be needed.
What you are describing is running a windows application (games) under an operating system they were not designed to run on. It is possible. There are 2 options.
1. Wine - its free to install, and the games should install and work. But you are going to have to do a lot of setup. You can install wine with Synaptic. [Here is a good page describing Synaptic and its use.]("http://www.monkeyblog.org/ubuntu/installing/#installing_with_synaptic") Then you will want to search the [Wine application database ]("http://appdb.winehq.org/")to get instructions to install the game under wine. [Here is the half life 2 page.]("http://appdb.winehq.org/objectManager.php?sClass=version&iId=2890")
2. [Cedega ]("http://www.transgaming.com/") - Cedega is a commercial application. It costs money to get it. It costs $15 for a 3 month subscription in witch tinme you can download the application and updates. If you let the subscription run out Cedega will still work, but you cant get updates. Be sure to backup the install files if you do. The advantage of Cedega is its very simple to install and run games, just as easy as windows.

Transferring games from one operating system (Windows XP) to another (linux) will not work, but you can move over saved game files and levels after you get them running in either option. 
Im not fimalure with the hardware so maybe someone else can give you info on them.
