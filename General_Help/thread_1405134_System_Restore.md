### Post by Mark Phelps on 2010-02-12
Hard to answer your question because I'm not sure what you really want to do ...

But, with respect to System Restore, a common misconception is that it makes a copy of your "system" and then is able to restore that later.  In fact, all it does is make a copy of some of the MS Windows system files (including drivers) and the registry.  It does not save other settings or files.

So, it's useful for backing out driver updates and SOME Windows Updates -- but not much else.

If that's what you're looking to do, you should be OK.

But if you're looking to restore your entire "system" to a previous date, System Restore will not do that for you.

Win7 has enhanced backup/restore capabilities over Vista.  You should read up on those since, now, these capabilities are included in ALL versions, not just Business/Ultimate/Enterprise.
