### Post by kesetyan on 2019-05-09
Hi Old Nomad,

I'm not sure if this is the same for the 'Mate' desktop but on my Ubuntu 18.04 LTS version, click the 'Application Menu' icon and open 'Settings'.  If 'Devices' shows at the left end of the top bar of the 'Settings' window, click on the left pointing arrow to to change it to 'Settings'.  Click 'Language and Region' to access the related settings which include the keyboard under 'Input Sources'.  Click the '+' button to add an input source - afterwards you can use the up or down arrow to move selected input sources up or down the list.  If you want to remove unwanted input sources from the list, select them and use the '-' button.

### Post by kesetyan on 2019-05-09
> **old-nomad said:**
> During the installation when i came to select the keyboard layout i selected the English (UK) option. But when i typed out some text in the box provided for this option all it did was come back with a foreign language that i didn't recognise. On the English (UK) keyboard the '@' symbol is obtained by the 'Shift' + the single speech mark key.  If you don't want to have to select the 'Shift' key when typing a password then you will need to change your password to ensure it only contains characters obtained from key presses without the use of 'Shift'.  This, of course, will limit to a certain extent, the strength of your password.  Presumably, once you have started up the system using the existing password you can change the passwords to whatever you want.  However, you have not been entirely clear as to how these passwords were created in the first place so it is difficult to be more precise.  You may find the following link useful: [https://askubuntu.com/questions/423942/change-password-on-root-user-and-user-account](https://askubuntu.com/questions/423942/change-password-on-root-user-and-user-account) .
