### Post by Verbeck on 2010-11-24
even though wubi installs ubuntu as a program within windows, it wont be listed as in the all programs list. you should restart the pc and choose 'ubuntu' from windows loader when it prompts to do so

also, remember that wubi is just for checking out, not for long term or prolonged use

edit : i dont know about the error it gave, so someone else would probably be of more help for that
