### Post by SEMW on 2008-05-27
> **Genius314 said:**
> Sans, freesans, MS Sans... whatever. Simple, clean.

Btw, is there a place where I can easily get the Segoe UI font, or is it not legal to get it without owning Vista (or some Microsoft program of some sort)?

It's perfectly legal to get without owning Vista: lots of freeware crap from Microsoft comes with it (e.g. Windows Live Mail, Powerpoint Viewer, etc.).  If you know how to extract files from whatever archive format msi installers use, then get it from there; if not, I've just attached mine.

(Note that depending on how you've set up your font rendering, Segoe UI seems to sometimes appear thicker than it should be (and too tightly kerned) at small point sizes with the bytecode hinter; one fix if you have this problem is to switch to the autohinter.)
