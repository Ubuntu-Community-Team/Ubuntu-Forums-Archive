### Post by SeijiSensei on 2013-11-13
If you're trying to connect to recent Android device, Ubuntu versions starting with 13.04 support the Media Transfer Protocol which lets you mount the device via a USB cable.  I've also used the [Wifi File Transfer Pro]("https://play.google.com/store/apps/details?id=com.smarterdroid.wififiletransferpro&hl=en") app for Android that runs a little web server on the device.

I've never used Kies though it is on my Galaxy S3.
