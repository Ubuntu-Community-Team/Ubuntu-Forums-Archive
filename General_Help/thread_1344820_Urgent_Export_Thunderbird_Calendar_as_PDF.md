> **lightnb said:**

I don't know if this is a Thunderbird feature or not, but to get around it, you could do a screenshot.  With the Thunderbird window selected, just hit Alt+Print Screen.  That will take a screenshot of only the Thunderbird window.

I don't know if you can save directly to PDF or not.  You might be able to, but if not, you can always save it to  normal image file such as PNG and then print to PDF using Ubuntu's built in PDF printer.
