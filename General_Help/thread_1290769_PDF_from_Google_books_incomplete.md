### Post by ramjet_1953 on 2009-10-13
When you downloaded the file, did it include a separate folder that contains the graphics?

If not, someone has posted the pdf, without these files.

Another problem can be that you have the files, but the links in the document are still pointing to the web site, not your local folder.

The way to fix this is to open the file in a pdf editing program and then right click on the image boxes and enter the link to the files on your system.
