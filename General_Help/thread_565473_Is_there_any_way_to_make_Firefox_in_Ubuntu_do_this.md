### Post by InorganicMatter on 2007-10-02
Under all Windows OSes, when you click once in the address or search bar of Firefox, it highlights all the text in there.   You can then start typing and it will overwrite what's in there.

Under Ubuntu, when I click in said bars using Firefox, it instead places my cursor in the text of the box, where I clicked.

I'm sure there's a good reason for this behavior that a bunch of elitists are going to jump all over me for asking this.  Really thou, I don't care; that's what I've gotten used to after using Firefox for 4 years, and I'm not going to change any time soon.

### Post by praet on 2007-10-02
Try entering about:config in the location box, and search for "click".  Scroll to this entry and change it to "True"
```
browser.urlbar.clickSelectsAll
```

Hope that helps.
