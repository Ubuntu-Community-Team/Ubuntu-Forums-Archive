### Post by kanikilu on 2010-02-11
That's what should come up if you hit Print Screen (PrtScn).

There are two things I can think of that might be going on here:

The first may just be that the keyboard shortcut got changed. To check that, go to System > Preferences > Keyboard Shortcuts. Under the Desktop section, there should be an entry for "Take a screenshot". See what that is bound to. If it's set to "Delete", just change it to PrtScn.

The only other thing I can think of is that maybe you are using a laptop or keyboard with Function keys with the function lock turned on, and perhaps maybe the Delete key and PrtScn key share a button? Admittedly, I've never seen a setup like that, though, so it's probably not likely.
