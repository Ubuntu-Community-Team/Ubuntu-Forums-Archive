### Post by jim_deadlock on 2011-06-10
I deleted my Music folder and now I want it back with the original emblem on it. I've tried properties -> emblem but it puts it on the left instead of the right. How do I get the original back on the right of the folder?

---

### Post by jim_deadlock on 2011-06-11
Answer:

1. Delete the file "$HOME/.config/user-dirs.dirs"
2. Run "xdg-user-dirs-update"
