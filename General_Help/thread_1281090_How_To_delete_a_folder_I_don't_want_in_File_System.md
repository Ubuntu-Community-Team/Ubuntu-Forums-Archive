### Post by Vaphell on 2009-10-02
uninstalling msttcorefonts in synaptic maybe would help?
probably myspace has defined fonts in the order of: Arial/TimesNewRoman/AnyWindowSpecificFont -> something of general serif/sans-serif/monospace family

previously Ubuntu used its own fonts but since MS ones are in place now, they take precedence.
you can also configure firefox to override webpage font settings with fonts of choice - i personally set Dejavu family to do the job and define minimum size of 14 (i hate tiny text). Play with these settings a little, see if you can achieve desirable effects.
