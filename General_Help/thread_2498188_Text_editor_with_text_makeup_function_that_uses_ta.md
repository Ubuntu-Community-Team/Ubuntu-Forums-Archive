### Post by dragonfly41 on 2024-06-05
I'm not sure that I understand. The last line does retain last highlighted text so you want to reset that when switching or shutting down so that there is no history?
The place to post "issues" is here.
[https://github.com/giuspen/cherrytree/issues](https://github.com/giuspen/cherrytree/issues)

Usually I find a workaround for any issue. There are many undiscovered features in CherryTree.
I have used this tool for years and indeed it is weaved as a container into every workflow I have.
I advise to have multiple documents rather than pouring everything into one massive database. Regard each document as a "node" in your desktop and use Recoll to index these. I have hundreds scattered around my desktop and Recoll gives me a satellite view.
The query to view all indexed CherryTree documents is ext:ctd
