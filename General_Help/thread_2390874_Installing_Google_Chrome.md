### Post by deadflowr on 2018-05-03
If you already had chrome installed, at one point, then you might already have the google repositories added to your system.
If that's the case then all you would need to do is either search for google chrome in the software store or run
```
sudo apt update
sudo apt install google-chrome-stable
```
