### Post by grahammechanical on 2011-06-08
I am not familiar with the wubi set up, but did you need to assign an amount of RAM to work as video memory? If so, did you assign enough?

You say the upper panel keeps disappearing from time to time. Is this the whole panel or just the menus on the panel? You might need to give a clearer explanation of the problem.

As for the brightness thing it might be the screen saver. Click on the shut down icon and select System Settings>Screensaver and move the slider to give you more time.

By the way, remember that by using Ubuntu through wubi you are also using it through Windows. Wubi is a Windows program that lets you run Ubuntu as if you are dual booting. But you are not in fact doing that. I imagine that there will be a slowness on some systems.

Regards.
