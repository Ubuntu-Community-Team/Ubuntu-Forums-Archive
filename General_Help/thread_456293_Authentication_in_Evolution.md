Generally speaking, I use **Gmail **directly for email, but as **Gmail** is not supported (!) in **Picasa**'s **Linux** version, I opted to use **Evolution** to send some photos to friends. However, all these messages have become stuck in my outbasket ; when I attempt to send them using the Send/Receive command, I get the following message : «Authentication Required». I have been unable to find any information regarding this matter on the singularly unhelpful «Help» service provided in **Evolution**, but I should be most grateful if anyone here could help me install the authentication necessary to send email messages from my evolution account. I'm running **Ubuntu 7.04** on **AMD 64 X2**....

### Post by mhenriday on 2007-05-27
For anyone else who couldn't figure this one out on their own, here below the reply posted to me by **Markus Thielmann**, which worked like a charm :> 1) Start Evolution
2) Open Edit-Preferences
3) Choose GMail Account, click "Edit"
4) Choose tab "sending Email"
5) Click "Server requires authentification" on "Server configuration"
6) Enter your Gmail username
7) Click "Remember password" if you want evolution to do so

Now you're able to send mail through Gmails SMTP service.

So can it go !...
