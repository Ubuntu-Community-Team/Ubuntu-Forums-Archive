### Post by xtremesupremacy3 on 2010-07-14
I asked that question yesterday and got a reply which was expected.

Autorunning is probably not a good idea, you never know what you may get.

However, if you want there are 2 ways you can do it manually.

The icon on the desktop, right click it and it gives an autorun option...However, this fails a lot for me so I then do this:

I open the folder, right-click the .exe file and hit 'Open with Wine Windows Program Loader', then it starts up as normal.

I know, it seems out of the way but as far as I can tell, this is the only option there is right now
