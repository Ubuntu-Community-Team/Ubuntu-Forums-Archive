### Post by jamesisin on 2011-06-02
There some options which come to mind but it really depends on what you are trying to do *in specific*.

You could, for instance, limit each user's home directory to a small quota which would disallow too much file creation.  Some writing is necessary for much regular functioning typically, so eliminating *all* writing is maybe not a feasible option.
