### Post by yancek on 2014-10-18
The link below provides a pretty detailed explanation of networking.  What you set up depends upon what you want to do.  Will you have one computer as a central server accessible to other computers, do you want all to be able to access each other, what limitations do you want.  Network sharing with Linux systems usually uses nfs (network file system), 2nd link below:

[http://www.linuxhomenetworking.com/wiki/index.php/Quick_HOWTO_:_Ch02_:_Introduction_to_Networking](http://www.linuxhomenetworking.com/wiki/index.php/Quick_HOWTO_:_Ch02_:_Introduction_to_Networking)
