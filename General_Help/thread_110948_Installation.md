### Post by Sutekh on 2006-01-01
GDM stands for GNOME Display Manager; it is basically the program that controls your login.  

Go to System -> Administration -> Login Screen Setup
Type your password (if needed) and choose the graphical greeter tab.  Then click install theme, guide the browser to the GDM tarballs and your done.  The files get extracted to a hidden folder in your home folder.
