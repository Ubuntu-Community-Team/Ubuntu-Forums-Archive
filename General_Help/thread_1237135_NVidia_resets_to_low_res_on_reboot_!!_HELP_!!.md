### Post by joriad on 2009-08-11
Ok, here is what I found and it worked for me perfectly.

> type in a terminal:

sudo nvidia-settings

set your options

x server display configuration

select resolution, refresh rate explicitly
click apply
click save to x configuration file - "tick merge with existing configuration file"

Should stick.

The problem is that by default nvidia-settings is not launched with super user privileges and thus cannot modify the xorg.conf - resulting lost settings at each reboot.

### Post by P4man on 2009-08-11
What Joriad said, except use **gk**sudo rather than sudo. So run in a terminal:

```
gksudo nvidia-settings
```

(nvidia-settings is a graphical program, and if you launch a graphical program that needs admin rights, always use gksudo and not sudo. More info on that here:

[http://www.psychocats.net/ubuntu/graphicalsudo](http://www.psychocats.net/ubuntu/graphicalsudo)
