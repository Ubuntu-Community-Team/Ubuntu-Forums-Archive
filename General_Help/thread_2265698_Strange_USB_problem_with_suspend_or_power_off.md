Hello !
I've bought a Lenovo Thinkpad Edge E540 (20C600JH) onto which I've installed Ubuntu 12.04.1 LTS alongside Windows 7 (pre-installed).
I also have an ergonomic mouse so I use it. This mouse has a led indicating the resolution set on the mouse. (Various colors for different dpi).
Last night I was astonished to see this led lit ! Indicating the mouse was powered up (the laptop was turned off). 
As I know this device has an "always on" USB port to charge mobile devices, I triple checked that :
The mouse did not use this USB port
The BIOS settings were set to DISABLED for this function and the setting to disable it on battery was set to YES.
The machine was really powered off.
Of course when I started the machine, the battery was only 60 % charged indicating that the mouse use quite a bit of power.
So I made some tests :
I put the machine to sleep and the mouse is still powered on.
I tried to remove the mouse BEFORE going to sleep. And plugged it back in when the machine was in sleep mode. The mouse did not turn on.
I did the same test but with power off instead of sleep. Same results as above. 

Last but not least, the BIOS is 2.17 (latest one) and USB 3.0 is disabled because, otherwise, the machine cannot awake from sleep. 

Any help or advice would be highly appreciated.
