### Post by Chronon on 2009-06-12
Not at all.  I understand that people have different preferences.  I can understand why you would want this capability.  Unfortunately, I am not aware of a tool that does what you want.  I was going to suggest Inkscape too before remembering it's troubles with copy/paste.

When you paste an image into OOo Writer you can scale the image in situ, so I don't entirely understand your comment about image size limitations.  Of course if you scale it too much any bitmap image will start to look bad.  This is why it would be so much better if Inkscape supported copy/paste properly (since it's a vector graphics app).

Edit: I think crossedout's option is good too.  Since you can manipulate text boxes in Draw it might make more sense to use that for note taking since you have the capability to make diagrams and also create text boxes.
