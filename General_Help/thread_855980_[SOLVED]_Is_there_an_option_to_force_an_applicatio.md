### Post by editrix on 2008-07-15
Ah, and there was I with an answer for you if it *was* Kubuntu.

In KDE if you click on the program's icon in the top left corner, there's an option to Configure Window Behaviour. **IF** Xubuntu works the same way, you can set up the window as you like, then in the Configure Window Behaviour screen there's a "detect" button that will detect how the window's set up, and you can tell it to always use these settings.
