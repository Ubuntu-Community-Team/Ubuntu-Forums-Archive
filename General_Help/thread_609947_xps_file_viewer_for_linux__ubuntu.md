### Post by MentalNotes on 2008-04-26
Okular, the 'Document Viewer' for KDE 4, has preliminary support for XPS. If you have upgraded to Hardy it is available in the repos. Install the package okular-kde4.

I would suggest that you use PDF if you can, as it has better support under all three main platforms: Windows, Mac, and GNU/Linux. Under Ubuntu Hardy pdf printing is installed by default and Open Office exports to pdf. In Windows you can use OpenOffice as well or buy a copy of Adobe Acrobat.
