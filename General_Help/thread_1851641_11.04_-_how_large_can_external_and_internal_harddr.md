### Post by oldfred on 2011-09-28
The issues start at 2Tib, but the installer may convert partitioning scheme if over 1TB.

Computer have used MBR(msdos) partitioning since the beginning of PCs. It has a 4 partition limit, but one can be an extended partition with many logicals.

Large drives (or small drives if you want) can use gpt a newer partitioning scheme. Apple Macs use it now. Windows will only boot with gpt if using the new UEFI which is a replacement for BIOS booting. I have BIOS and gpt on a 160GB drive and even used gpt on a 16GB flash drive.
