### Post by Toz on 2011-12-08
The command to start the quickstarter and place it in your tray automatically is:
```
libreoffice --quickstart --nologo --nodefault &
```

You can run this from Alt-F2, add it to your startup applications if you want it to start automatically or create a .desktop file so that you have a menu item for it.
