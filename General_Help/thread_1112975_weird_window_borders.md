Are you using Intrepid (8.10) with an Nvidia card, and the proprietary nvidia driver? There's a bug in the proprietary nvidia driver that does this with some (but not all) nvidia chipsets. Which nvidia chipset do you have?

The bad news: I don't think there is a fix, short of changing your video card or using the open-source 'nv' driver.

The good news: I'm not getting this in Jaunty Beta.

**edit:** do you get this with the Human theme? If not, ignore what I said. What I described happens with all themes that I've tried.
