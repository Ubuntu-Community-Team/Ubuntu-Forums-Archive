### Post by john.nicholls on 2008-08-06
> **Thirdman03 said:**
> I set up the Session manager to automatically open the Evolution mail client at startup, but I was wondering if there was any way that I could specify that it open in a specific workspace every time (e.g. open the program in workspace four while Ubuntu opens to workspace one).

Thanks in advance to any answers.

Set up the windows the way you want with Evolution open on workspace four.
Then go to 
System | Preferences | Sessions | Sessions Options | check Automatically remember running applications when logging out.

Then when you log back in all windows should be on the same workspaces as they were before.
