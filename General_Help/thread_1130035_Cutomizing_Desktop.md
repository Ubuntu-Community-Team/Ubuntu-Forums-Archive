### Post by 3rdalbum on 2009-04-19
If you're talking about Gnome, you will probably find glassy panel images on [www.gnome-look.org](www.gnome-look.org) that can be applied to your panel.

If the screenshots you are talking about are KDE, then this is a specific theming function of KDE.

You can get themes that will change the desktop background, window borders, widgets (buttons, scrollbars etc), icons and colours, but these are relatively rare - a lot of people like to mix and match. You can't get themes that change the actual layout of anything - this would be very intrusive. The way I like to work is different to the way anyone else likes to work, and I'd be extremely annoyed if installing a new theme ruined my workflow.
