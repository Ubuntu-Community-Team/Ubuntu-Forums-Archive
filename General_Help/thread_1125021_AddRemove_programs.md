### Post by Vaphell on 2009-04-14
add/remove programs from app menu
In case you dont see that option: right click on applications menu, modify, mark checkbox next to add/remove
in combo you can select what apps you want to see - all/installed/etc
but i think some are missing, they exist as packages in synaptic, but don't show up in add/remove menu.

### Post by mb_webguy on 2009-04-14
Add/Remove under the Applications menu can show you all of the applications you have installed.  Synaptic Package manager under the System->Administration can show you *all* software installed, including libraries and other non-application software.
