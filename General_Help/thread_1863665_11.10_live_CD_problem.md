### Post by varunendra on 2011-10-19
It's good you tried live cd first, not directly jumped in with installation.:)

If your graphics adapter needs proprietary NVidia drivers, then that's it! You need to have it to get enhanced graphics. Your worries about the possibility of things going wrong after driver installation are reasonable, so if you want to try it safely, you may install 11.10 on a USB flash drive first to see how it goes. Of course make sure grub also gets installed on the USB itself, not on the hard drive.

However, your description suggests that you can't do anything after that message (system hangs permanently). If so, please try advance options: [https://help.ubuntu.com/community/BootOptions](https://help.ubuntu.com/community/BootOptions)

I would suggest to boot with options acpi=off, noapic, nolapic and nomodeset.
