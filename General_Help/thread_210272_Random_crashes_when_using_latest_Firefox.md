### Post by fimbulvetr on 2006-07-08
dapper seems to be very unstable in the apic/acpi area. I struggled with lockups for a long time and thought it was related to the browser because my browser was usually the primary window during the crash.

Try appending ```
acpi=off noapic
```
to your /boot/grub/menu.lst "kernel" line for the kernel you use.
