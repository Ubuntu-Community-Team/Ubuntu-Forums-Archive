### Post by |{urse on 2011-03-04
for an easy dock, Cairo-dock.

> sudo apt-get install cairo-dock

There are a bunch of other docks, docky is popular (my favorite is wbar), you'll see them on google.

If theres a black bar behind the dock, then you need to enable compositing.

Personally I like to kill xfce4-panel, and install a lightweight panel like pypanel or tint2. Then I use the desktop's right-click menu for the xcfe menu.
