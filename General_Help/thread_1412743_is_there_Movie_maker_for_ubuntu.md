### Post by Vakman on 2010-02-21
There is PiTiVi, which will be included in Ubuntu 10.04.
There is Cinelerra.
There is OpenShot Video Editor, which is really easy and pretty basic.
There is also Kino.
There are others but these should work fine for you. Your needs sound basic, try using OpenShot or PiTiVi first.

### Post by kostkon on 2010-02-21
The next version of Ubuntu (Ubuntu 10.04 aka Lucid Lynx) will have a video editor by default (*PiTiVi*).

For the time being, I would recommend you to use [*Openshot*]("http://www.openshotvideo.com/"). It's fairly advanced and easy to use.
