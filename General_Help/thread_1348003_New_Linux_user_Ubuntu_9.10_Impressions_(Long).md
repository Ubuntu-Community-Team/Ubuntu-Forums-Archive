### Post by sophicsage on 2009-12-06
[SIZE=2]I am totally new to Linux, just to get that out there.  When I say new, I mean I used it for the first time last week.  In training at work, we used a live Linux CD to examine Windows PCs for certain types of documents.  I was quite impressed with Linux's flexibility and ease of use.  When I got home, out of curiosity, I decided to try out a Linux system on my Sony Viao (VGN-NS238J).  

Installation was easy.  During the initial load process, I was given the option to partition or wipe the system.  On gut instinct (and frustration with years of Windows irritations and let-downs) I decided to wipe out Windows 7, the underlying Windows Vista install program, and install a Linux only system.  I was skittish about this, but didn't have too much fear since I have Windows 7 disk in case it didn't work out.

Initial installation was basically very fast and easy.  I was thinking I had run across a Windows-like program and didn't understand what all the rumors were about Linux being "difficult to use".  I had immediate web access, etc.  Then a few bugs cropped up.  Youtube videos didn't play, Pandora Internet Radio didn't work right, and I was unable to play encrypted DVD movies.  I got a little frustrated.  Just for the benefit of other new users, I did the following:  1) Searched in Synaptic Manager for 'flash' and load the freeinstall and non-free to solve the Youtube issue, 2) Went to System-Preference-Appearance-Visual Effects-Chose 'None'.  This solved the Pandora play issue, as well as the unusable button issue that persisted in Youtube, 3)  I don't remember exactly what I did to solve the DVD playability problem, but I remember it had something to do with installing some gnash files under the Synaptic Manager.  Movie player now works.

Every site that I usually go to works just fine, from Facebook, to CNN, Evony.com (some visual jumpiness exists in some of Java effects, but otherwise no change).  Streaming movies through Netflix does NOT work for me.  I have researched this issue and there isn't a fix as far as I can tell.  Apparently Netflix has just decided not to have any Linux support.  That's irritating, but I can live with it, since hardly ever stream movies through their site...instead of using the mailed DVDs.  Netflix needs to get a clue, seriously.

I was assisted by the OS in installing my ATI graphics drivers.  This was accomplished easily, in addition to some other file in the Synaptic Manager for ATI cards.  No prob there, but I haven't loaded any games yet.

The Evolution Mail program works excellently with my Gmail account and sends and retrieves mail without any problem whatsoever.  I'm impressed.

I've downloaded a couple of pictures, tested them by moving them around, copying, posting them to my Facebook, etc. and I've seen nothing wrong.

I was a little afraid to use the terminal, but I had to in order to activate the firewall, by using a few usf commands which turned out to be very, very simple and effective.  I'm now less afraid to use advice from good sources when it comes to this.  I've had to use it 5 times and the results have been excellent.  I may have used it once for the DVD playing issue.

Now that my system is up and running, it's smoother than Windows could ever hope to be.  I think Ubuntu could completely destroy Windows if people didn't have to do anything to get it totally functional from the start.  Most people won't go through what I did just to get everything working right.  But for me, it was worth the effort.  A free, far more secure, and lightweight OS vs. a total hack program like Windows that destroys everything?  There is no comparison.

I have only one task left:  Get my OS to meet up with my wireless HP printer.  I think I found a solution, but I'll do that some other time.

I LOVE Ubuntu and feel that it will suit my personal needs.  What I have learned about Linux is that it doesn't do like Windows and give everyone the exact same software.  It allows each user to customize the OS to your needs.  Different hardware, etc., means different needs, etc.  Little mistakes don't mean the whole OS will crash like with Windows.  You can make a mistake here and there, and still be able to repair it without destroying everything.  Try doing such things in Windows Disk Registry Editor and see what happens!  You'll be buying a new computer it's so junked up.  I thought I was taking a great risk putting a Linux system on a brand new Sony Viao.  But now I'm grateful, and the little challenges I come across are actually fun and I'm learning a lot.  I know I can make some changes and test the results more or less safely.

I can't thank the Ubuntu forums and community in general for their postings.  They helped me immensely and got me started.  I will try and start documenting my activities in order to help future users even newer than me.  I wish I had more carefully documented my DVD playing fix ,but it is what it is.

The OS boots BLAZINGLY FAST.  It puts every other system I've ever used to sad, teary-eyed shame.  My OS boots up in about 30 seconds total, and that's from hitting the power button.  OpenOffice works well in every document I tested thus far.  It also seems that my battery is lasting longer.  Maybe it's just me, but it just seems to be more efficient?

Now my next question is:  Why does Windows even exist and why is Bill Gates making any money on that load of crap? 





[/SIZE]

### Post by 3Miro on 2009-12-06
Welcome. I have been using Linux since 2000 and since Ubuntu 7.10, this is my exclusive OS.

- For the reason why flash and such doesn't work form the start:

You can read about the "free software ideology" and what is the philosophy involved. Besides, you have done some things the "hard" way. If you go to Apps -> Ubuntu Software Center, then you can install "Ubuntu Restricted Extras", this installs flash + java + mp3 and other codecs.

Visual effects would also only run properly if you have installed the proper video driver. You should be able to reenable them without trouble now (I am assuming you first turned them off and then you installed the ATI driver). You can go to synaptic and get Compiz Config Manager, then you will get a ton of effects to chose from and configure.

- On why is set up more involved than windows:

Actually set up is less involved then windows (or any windows that I have seen and worked with). Most people don't have to go through this since they get computers with windows preinstalled and somewhat set up for them. When I got a system76 laptop, I had to: turn it on, select username and password, open firefox, play youtube video.

- Ubuntu is the only distro that assumes its users know nothing at all. All other distros assume some level of experience and while they work great for their purpose/audiences, people that try them usually run into trouble.
