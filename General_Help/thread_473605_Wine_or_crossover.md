### Post by jasonlfunk on 2007-06-14
I have never used crossover so I cannot speak to the ease of installing it, but wine is quite easy to install. 
```
sudo apt-get install wine
```

Though not all windows applications run well with wine, it isn't magic like that. You could give it a shot and cross your fingers. :)

More information for wine: [https://help.ubuntu.com/community/Wine](https://help.ubuntu.com/community/Wine)

### Post by HereInOz on 2007-06-14
Personally, I reckon you will find Crossover Office a little easier to work with than Wine.  More GUI, less CLI.  However, jasonlfunk is right when he says that not all Windows apps run in wine or Crossover, so don't be disappointed if the maps don't work.  You can but try.

I used to run Crossover, but in the end I installed VMWare server, and installed Win2K in it for those times when Windows is the only option.
