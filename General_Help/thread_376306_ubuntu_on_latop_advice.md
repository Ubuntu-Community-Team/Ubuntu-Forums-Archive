### Post by strabes on 2007-03-05
I would STRONGLY advise you NOT to get a laptop with an ATI card if you're going to be using linux on it. ATI drivers are terrible for linux. If you are bent on ATI, make sure you get one that supports the open source driver. I made that mistake and I regret it terribly.
