### Post by mcduck on 2010-04-07
It's not visible in the menus by default (since you rarely need to start the app that way, as it's pretty well integrated into the Gnome desktop itself).

For example if you right-click a file/directory and select "Create Archive" ("Compress" in 10.04), or right-click some archive and select "Extract Here" you are using the Archive Manager.

If you still feel that you want to launch the app itself, just right-click the Applications-menu and select "Edit Menus" to open the menu editor. There's already  a ready entry for the Archive Manager under Applications/Accessories, so all you need to do is to mark the checkbox to make it visible in the menu.

edit: of course you can also just run "file-roller" in a terminal to start the Archive Manager.
