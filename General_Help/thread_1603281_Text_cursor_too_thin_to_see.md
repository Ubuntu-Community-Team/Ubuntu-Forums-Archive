### Post by CTBuckweed on 2010-10-22
I have had this nagging problem for months. In both Ubuntu (10.04 Lucid) and Windows XP, I have trouble seeing the text cursor. The mouse cursor is fine, but the text cursor is only 1 pixel wide, too thin to see at times on my 1680x1050 22" monitor. XP has a control to change the width of the text cursor (Control Panel -> Mouse), but I haven't seen a way to adjust Linux's cursor. In a google search, I came across an new theme based on the 'Ambiance' theme that has a wider cursor.

The site is [http://linux.softpedia.com/get/Desktop-Environment/Themes/Ambiance-Thin-57397.shtml](http://linux.softpedia.com/get/Desktop-Environment/Themes/Ambiance-Thin-57397.shtml)

There, I found a zip file for a new theme "Ambiance Thin". I downloaded it a followed the instructions. It didn't install. If an effort to make it install, I unzipped the files and then tarred them up into a 'tar.gz' archive and tried the install again. It worked! Now I have a text cursor that I can see!

It doesn't help with gmail thru Firefox; they must do their own cursor. But for Open Office and text editors it works!

Maybe I don't know how to do it, but I think that Gnome should have a way to customize the text cursors.

The theme should have been called 'Ambiance Thicker' :)
