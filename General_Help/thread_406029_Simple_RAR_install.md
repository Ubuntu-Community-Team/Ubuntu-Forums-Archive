### Post by LinXway on 2007-04-10
I have read allot of post on people having issues with rar archives...


This is how I do it (useing Nautilus):

1. goto [http://rarlab.com/download.htm](http://rarlab.com/download.htm)    <-- download the linux version...
2. decompress it into a folder (right click in file manager) select extract here...
3. copy the rar file (only one you need) into the /usr/bin  (need to be root)...

note: dont make a link to it...place a copy of the rar file in the directory

I have done this for every version of ubuntu I have installed and did it in 7.04 and it works fine. After I place it there all I need do is d-click on the file like any other compressed file or right click and the "Archive Manager" handles it. 

Hope this helps
