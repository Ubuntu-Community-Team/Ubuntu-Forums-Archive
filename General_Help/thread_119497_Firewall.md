### Post by ace2005 on 2006-01-19
In Adept (or Synaptic if you've installed it) install kde-i18n-**, e.g if you wanted french you should install kde-i18n-fr

Then go into kcontrol, open it using Alt + F2 and typing "kcontrol" and not by using the one in the start menu

Then go into Regional & Accessibility > Country/Region & Language and select the language you want in the dropdown list (you will have a choice of the installed ones)

As for a firewall i use Firestarter, it makes opening ports for stuff like bittorrent easy as well as Internet Connection Shareing, so you might want to give that a try

:)  Hope this helps
