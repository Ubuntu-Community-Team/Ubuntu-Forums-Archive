### Post by oldfred on 2011-11-04
Welcome to the forums.

I am more a fan of clean installs. You can copy /home which has all your user settings & data over. You can export a list of installed programs & reinstall. Then you have your old install in a new clean system.

My entire recovery plan is just a clean install of the operating system and backup of my data. So my backup has everything you should need. The advantage you have is, if you forget something you can go back and copy it also.

Oldfred's list of stuff to backup May 2011:
[http://ubuntuforums.org/showthread.php?t=1748541](http://ubuntuforums.org/showthread.php?t=1748541)

If you have a home network you can connect the two computers and copy data that way also if you do a clean install on the new system.
