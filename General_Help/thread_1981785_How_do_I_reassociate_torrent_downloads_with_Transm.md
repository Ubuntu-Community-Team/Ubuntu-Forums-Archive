### Post by asharpham on 2012-05-17
I downloaded a torrent downloader to see if it worked any better than Transmission. It didn't, so I decided to go back to using Transmission. However now Transmission is no longer associated with torrent downloads and I don't know the name of the file that launches it, or where it's kept. Can someone help with this?

Alan.

---

### Post by Dave_L on 2012-05-17
In Nautilus, right-click on a .torrent file, select "Properties", then "Open With".

The executable is "/usr/bin/transmission".
