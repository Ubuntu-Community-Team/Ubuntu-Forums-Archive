### Post by mb_webguy on 2009-04-29
Panel text color can be changed in Appearance by customizing your current theme.  Unfortunately, you can't currently change panel text color independently of window text color (which is actually what you'd change in the Appearance dialog).

### Post by Eneerge on 2009-04-30
You could try installing gnome-color-chooser: sudo apt-get install gnome-color-chooser 
It gives you a GUI to customize a few more things that can be helpful.  What's really neat is you can set the "Engines" for certain objects.  If you setup the panel on the "High Contrast" engine, you can enable hover effects for the Ubuntu Custom Menu.
