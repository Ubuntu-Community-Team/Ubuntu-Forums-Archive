### Post by COKEDUDE on 2010-10-06
I wanted to add a key detail that was neglected from the first article I read. This only matters if you are trying to watch videos. 

> Once finished, make sure resolution is set to the resolution of the monitors&#8211;for example, if they are both 1440 x 900 monitors, make sure it is set to that resolution. If the monitors are different resolutions, you will need it to the resolution of the smaller monitor. So, for example, if one monitor is 1440 x 900 and the other is full HD resolution, you will have to set it at 1440×900.

[http://productivelinux.com/2009/01/23/how-to-set-up-dual-monitors-on-ubuntu-810-intrepid/](http://productivelinux.com/2009/01/23/how-to-set-up-dual-monitors-on-ubuntu-810-intrepid/)
