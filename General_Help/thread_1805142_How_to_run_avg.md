### Post by howefield on 2011-07-15
As I understand it, AVG no longer comes with a GUI.

To start AVG use the command sudo avgctl &#8211;start.
To update AVG use the command sudo avgupdate
And to scan use the command avgscan on a specific file/folder.

If you want a GUI, try another product eg, Bitdefender which does come with a GUI. Many others do also.
