### Post by atomkarinca on 2008-06-14
On [Wine Application Database]("http://appdb.winehq.org/objectManager.php?bIsQueue=false&bIsRejected=false&sClass=application&sTitle=Browse+Applications&iItemsPerPage=25&iPage=1&sOrderBy=appId&bAscending=false") you can see the results you should get with a specific game. If you're having problems you can out that game's page and see what other people did to overcome those problems. If that doesn't help either you can run the game from a terminal (Applications > Accessories > Terminal) and paste the output here and probably get feedback.
