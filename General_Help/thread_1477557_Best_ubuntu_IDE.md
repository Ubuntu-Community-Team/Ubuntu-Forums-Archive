### Post by gerowen on 2010-05-08
I like IDLE.  It's basically the official python text editor with syntax highlighting and the ability to run scripts inside of it or to check your script for errors from beginning to end.  It comes with python if you've ever actually installed it on a Windows OS, but not in Linux.  To get it in 10.04 just open the software center and search for the package named "idle".
