### Post by Mike1977 on 2014-06-09
I'm trying to configure conky. Following one of the forum guides, I disabled Ubuntu Unity Plugin after selecting Window decoration in CCSM. That did not fix my conky, but worse yet, I can't access ccsm or even terminal in orderto change back.  Apparently there is a bug report for ctl+alt+t not working on 14.04. 
I have no tool bar,  window or launcher bar.
I can get command Window with ctl+alt+F1, but typing both "unity-reset" or "ccsm" will not work

EDIT: 
I never noticed this, but I guess you can't delete a post/thread. Even if no answers yet?
I tried a number of things, but what actually worked was simply:
Right clicking on desktop>create a folder>click on folder to open file manager>go to "computer" > USR>Share>Applications
then open CCSM and re-enable Unity Plug in. 

Now back to Conky :rolleyes:
