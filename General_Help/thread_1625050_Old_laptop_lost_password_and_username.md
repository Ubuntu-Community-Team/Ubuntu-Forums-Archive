I pulled out my old laptop which still has Ubuntu v.8.04 and haven't used it in 3 years approximately. I have long since forgotten the username and password as I used to change it frequently for security reasons. Is there a way to circumvent the login screen through Grub (v.1.5) and replace them?

### Post by Ancanus on 2010-11-18
Just boot in recovery mode and change the password using the passwd command. Here is a relevant [link]("http://www.psychocats.net/ubuntu/resetpassword").
