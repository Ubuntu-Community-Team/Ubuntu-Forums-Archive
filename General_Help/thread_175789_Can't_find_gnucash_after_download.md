### Post by bluevoodoo1 on 2006-05-13
if you use the menu editor you can addit yourself. Some programs don't always add themselves to the menus...

```
smeg
```
menu editor

Navigate to an area where you want a new entry. Create a new entry with the button and write Gnucash as title and 'gnucash' and use any icon you want. See if there is an icon for gnucash in the /usr/share/pixmaps folder (usual place for application icons)
