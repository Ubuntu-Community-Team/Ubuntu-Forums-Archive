### Post by confused57 on 2007-06-01
> **sibir said:**
> How did you reconfigure GRUB? Is there a GUI tool for it?
Hi sibir,  see you found your way over here...what you can try at the grub boot menu is to highlight your Ubuntu entry, press "e", change the root to (hd0,1), then press "b" to boot.  The (hd0,1) should be the second partition as rscheckler mentioned.  This is temporary, but if it works  this is  how to make it permanent:

Open a terminal(Applications---Accessories---Terminal), enter:
```
cd /boot/grub
sudo cp menu.lst menu.lst
```
this changes directory to grub & makes a backup of your menu.lst, then to edit your menu.lst:
```
gksudo gedit menu.lst
```

Dell's configuration only has modem and wifi drivers extras on the laptop, nvidia on the desktop.
