### Post by GuitarFreak on 2009-06-06
Remember: This tutorial is for GIMP ONLY. Not Photoshop or any other art program.

If you don't already know what rendering means, it means to make part of an image transparent so you can put the image as a layer on another image to make ART!

Ok, first off, get the image you want to render. Select the "Paths" tool. As you can see, it makes sort of a "connect the dots" sort of thing. What you do with the paths tool is make a connect the dots thing around the image you want to render. 

*Zooming in is HIGHLY recommended to improve accuracy. *

After you have made a path around your image, go to the "Select" button at the top and pick "From Path". This should select the path you made for your image. A sort of "walking ants" type thing should form around the path. Now, go to "Select" again and pick "Invert". After you have done that, go to the "Layer" tab and hover over "Transparency" and then click "Add Alpha Channel". After you have done all of these steps, go to the "Edit" tab and select "Clear", and you're finished! Save your rendered image and now it can be used as a layer!

[B]You MUST complete all of these steps. All of the steps before "Clear" can be performed in any order, but you MUST complete all of these steps.

[/B]Hope this helps!
