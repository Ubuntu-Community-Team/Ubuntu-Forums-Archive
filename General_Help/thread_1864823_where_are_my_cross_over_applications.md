### Post by duboisj on 2012-01-05
Hello, Reddhandd, 

  I'm very sorry the Unity environment makes it hard to find Crossover.  Unity behaves very differently from other environments in terms of showing you were applications are - it's something we will address in a future release.

  Crossover will work fine once you get the necessary icons into the Unity dock.  If you click on Unity's 'Dash' icon in the upper-left hand corner of the screen and type the word 'Manage' into the search menu, you will find a Crossover icon which represents the bottle manager.  Drag that into the dock and you will be able to click on it and work with it.

  If you want Crossover's install-helper to install new windows software, you can type the word 'Install' into the Dash search panel.  You will find a Crossover icon labeled 'Install Windows Software.'  You can drag that into the dock to use it to access the install wizard.
