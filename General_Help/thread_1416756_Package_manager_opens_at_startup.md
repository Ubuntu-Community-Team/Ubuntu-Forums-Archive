### Post by derrick81787 on 2010-02-26
If I remember correctly, Xubuntu automatically saves your session when you log out, if you don't tell it otherwise.  If the package manager was open when you logged out, and your session was saved, that could be an issue.  Next time you log out, check and see if the checkbox that says something along the lines of "Save Current Session" is checked.

I haven't used Xubuntu in a while, so sorry if this is out of date.  This has happened to me before though, except with a different program always opening and this was the culprit.
