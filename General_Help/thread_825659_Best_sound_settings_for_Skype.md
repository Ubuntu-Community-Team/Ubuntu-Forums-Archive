Hi

Anyone know how best to configure Skype for audio - the choices are absolutely bewildering. Sometimes it's OK, but of late I've been having terrible problems with stuttering sound, making the whole thing unusable. Have to say, it all works fine in Windows.

I'm running Ubuntu Hardy, and I've got a Logitech Quickcam Pro 9000 webcam - which works fine with guvcview.

Under System>Preferences>Multimedia Systems Selector, I've the Audio Default Output Plugin set of PulseAudio Sound Server, similarly the Default input Plugin is set to PulseAudio Sound Server. I've no idea if this is correct, but I found this was the only way I could get both the mike on my webcam to work, and the mic plugged into the mic socket on my sound card

In Skype Options Sound Devices, under Sound in I've selected USB Device 0x46d:0x990 (hw:U0x46d0x990), which seems sensible, as I believe this refers to the mic on my webcam.

Anyone any better ideas?
