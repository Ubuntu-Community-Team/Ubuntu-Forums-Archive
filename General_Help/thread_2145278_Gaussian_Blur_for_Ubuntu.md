### Post by CatKiller on 2013-05-15
Not any more. The Compiz team ran out of developers and couldn't keep up, so a number of plugins got dropped so that development could continue on the rest. Blur was one of them. It's a shame because it was quite useful for the task you describe.

Some of the plugins that got dropped have come back, though, so all hope is not lost.

### Post by CatKiller on 2013-05-15
It's hard to say. It's more useful than the Burn plugin, for example. The lead developer has been hired by Canonical, so effects that are used by Unity are the priority, and it seems reasonable that Unity could use some blur for context highlighting and the like. I have no idea how difficult that plugin is to maintain, though. If it would need a rewrite to function properly with future changes to Compiz it might never happen.
