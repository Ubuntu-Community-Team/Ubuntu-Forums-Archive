### Post by damis648 on 2008-12-18
If you are using compiz, make sure the compiz settings manager is installed
sudo apt-get install compizconfig-settings-manager
and then go to System>Preferences>Compizconfig. Head to General, and then the desktops tab. From there you can change the number.

### Post by akelsall on 2008-12-18
lidengdeng, if you right click over one of the "desktop icons" in the lower right-hand corner and select properties (or it might say "configure"), a window will pop-up where you can set the # of desktops to any # you choose.
