### Post by MidnightGrey on 2013-06-20
> **anon_private said:**
> I am using a livecd. Is there an editor for Ubuntu 12.04 already installed. I can't find one.

Thanks
No, ubuntu by itself does not  come with any graphics editors. If you wish to remain using a LiveCD, check out Ubuntu Studio, which is a flavor of Ubuntu prepacked with multimedia applications.
In particular you may be interested in the applications mentioned here: [http://ubuntustudio.org/tour/graphics/](http://ubuntustudio.org/tour/graphics/)
You can run Ubuntu Studio as a LiveCD just like ubuntu.
