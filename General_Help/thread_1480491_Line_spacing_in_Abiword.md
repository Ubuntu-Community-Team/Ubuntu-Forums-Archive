### Post by AlexDudko on 2010-05-11
**Format -> Paragraph -> Indents and Spacing** Then simply set desired spacing.
Or select a paragraph, rightclick and choose **Paragraph**, then see above.

P.S. Don't know about windows (just don't have one), but I think must be the same.
