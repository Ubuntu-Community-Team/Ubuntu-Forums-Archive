### Post by 3rdalbum on 2009-10-11
If you're using Gnome, when you insert the DVD it asks you if you want to run the designated Autostart script. Also, when you open a file browser to the root directory of the DVD, there will be a message saying that there's autostart content, and a button that allows you to run it.
