### Post by Dennis N on 2016-01-09
Using **View > Zoom In** in Thunar will enlarge the thumbnails without using keyboard, but this is limited. Instead, you should install a good viewer like **gthumb** and view your images with that. In gthumb, you can set the thumbnail size in its **Preferences > Browser > Thumbnail Size**. They can be set larger than Thunar can display. Browser mode views images as thumbnails, Viewer mode shows one image at varying sizes including full screen, full size, and various percentages of full size.
