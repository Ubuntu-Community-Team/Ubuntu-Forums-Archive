### Post by katykat on 2010-10-26
There is no real utorrent for Linux.

There is an alpha server whose instructions are clear as mud, which does not appear to be working right on meerkat. At least I am having some problems getting it to recognize torrent files. Tell it to load all torrents in /tmp. It does not. 

On my system uTorrent 2.0 for Windows will work fine in wine on jaunty and meerkat. 
But 2.2 will not work on either. 

For some reason the native torrent apps like Deluge and Transmission have a problem with ports, with slow downloads, while uTorrent runs at full native speeds. So its worth installing.
