### Post by rbmorse on 2007-11-25
download gparted liveCD, burn the CD from the .iso, boot from that CD and nuke the linux partition to obivilion. 

You'll then have to boot with a Windows distribution CD and do the fixmbr trick from the repair console to restore the window's boot manager/boot loader.
