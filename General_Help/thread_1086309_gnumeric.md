### Post by gradavies on 2009-03-06
Hi

I'm using Ubuntu 8.10 64 bit (Gnome). On my system the Gnumeric templates are stored in the /usr/share/gnumeric/1.8.3/templates folder.

First save the spreadsheet that you want to use as a template in an accessible location. Then open that location as root. Copy the gnumeric file and paste it into the templates folder shown above. Root access is necessary to paste into that folder.

Open Gnumeric and click on File Open. The 'templates' folder should be listed on the left side of the 'select a file' dialogue box. Alternatively, open the /usr/share/gnumeric/1.8.3/templates folder and click on the relevant template.

Hopefully this will help and it's not too different in Xubuntu.
