### Post by solar george on 2010-09-15
Do you need a specific version of jpilot or would the one in the ubuntu repos work for you (in general its best to use software from the repos)?
If you can use the ones from the repos then download the following files;

[http://gb.archive.ubuntu.com/ubuntu/pool/main/p/pilot-link/libpisock9_0.12.4-7ubuntu1_i386.deb](http://gb.archive.ubuntu.com/ubuntu/pool/main/p/pilot-link/libpisock9_0.12.4-7ubuntu1_i386.deb)
[http://gb.archive.ubuntu.com/ubuntu/pool/universe/j/jpilot/jpilot_1.6.2.9-1_i386.deb](http://gb.archive.ubuntu.com/ubuntu/pool/universe/j/jpilot/jpilot_1.6.2.9-1_i386.deb)
[http://gb.archive.ubuntu.com/ubuntu/pool/universe/j/jpilot-backup/jpilot-backup_0.60-3_i386.deb](http://gb.archive.ubuntu.com/ubuntu/pool/universe/j/jpilot-backup/jpilot-backup_0.60-3_i386.deb)
[http://gb.archive.ubuntu.com/ubuntu/pool/universe/j/jpilot/jpilot-plugins_1.6.2.9-1_i386.deb](http://gb.archive.ubuntu.com/ubuntu/pool/universe/j/jpilot/jpilot-plugins_1.6.2.9-1_i386.deb)

then copy them to your linux machine and install them (in that order) by double clicking on the package files in the file manager.
