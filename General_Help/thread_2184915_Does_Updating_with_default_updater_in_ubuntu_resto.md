### Post by Elfy on 2013-10-31
No - updating with the default updater, or indeed any other method, does not mean you have to reinstall apps you already have.

What it will do is download the newest deb available for any package and install that version. 

You can see what debs you have downloaded in /var/cache/apt/archives/
