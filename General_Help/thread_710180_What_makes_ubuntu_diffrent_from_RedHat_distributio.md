### Post by EvilMarshmallow on 2008-02-28
While I'm not a highly technical Ubuntu user, one major difference I've noticed is the way that root access is handled.  On a default RedHat box, you have to use su and the root password.  On a default Ubuntu box, the root account remains locked and you use sudo.

Also, RedHat by default installs via RPM.  Ubuntu uses .deb packages.

Again, I'm no Linux expert but these are the two major differences I've noted so far.  As far as software development, your only difference would be how you package the files (if you're one of those nice developers that doesn't make every user compile your programs every time:)).  Although... at the core they're similar enough that you can install a RPM packager / installer on Ubuntu... it just doesn't come by default.
