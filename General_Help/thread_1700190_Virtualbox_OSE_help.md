### Post by wiggly81 on 2011-03-04
when you first run the virtual machine (in this case windows xp) it will prompt you for the disc location, there is a small yellow folder i believe to the right side, if you click this it will allow you to browse your computer for a iso file.

EDIT... You can also select Device from the menu at the top and select cd/DVD device and browse for a image (choose virtual disc)
