### Post by bumanie on 2009-01-05
Ensure you burn the .iso you download as an image as suggested by oilchangeguy's link. Also you will have to ensure that your computer bios is set to boot from cd-rom, if HDD is set as the first boot device, you won't get the cd to start - cd-rom needs to be the first boot device.
