### Post by damis648 on 2008-11-25
Sure. Just download the LiveCD and boot it, then once booted select Places>[you windows partition] from the top panel to view the files. You will not be able to burn them to a CD because the Ubuntu CD will be occupying the drive but you can copy to a thumb drive or external hard drive.
