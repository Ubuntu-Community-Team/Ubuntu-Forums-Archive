### Post by rampageoberon on 2008-12-19
Hi there,

Okay regarding the PC not reading the Live CD on boot is something that concerns the boot sequence in the BIOS. So you have to manually change that by entering the BIOS setup before it loads Ubuntu or Windows.

With regards to completely removing an OS (ubuntu) you'd need the windows cd. I would delete partitions for ubuntu and then fix boot and master boot records with the recovery console using the windows cd

hope that helps
