### Post by Thelasko on 2010-06-25
No need to download a program you want to install.  just type "sudo apt-get install kstars" and Ubuntu will download and install the program automatically.

For more information, [read this.]("http://www.psychocats.net/ubuntu/installingsoftware")

Side note: The "K" in kstars indicates that this is a program designed for the KDE environment.  If you are using standard Ubuntu(Gnome) and not Kubuntu (kde), you may want to consider using an alternative.

I like stellarium personally.
