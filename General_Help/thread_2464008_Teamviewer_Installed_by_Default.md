### Post by revm on 2021-06-22
I just did a fresh install of Ubuntu 20.04 LTS and everything has been going fine so far. I noticed today (maybe 3 days in) that under the applications Teamviewer is showing up. I never installed Teamviewer and I cannot uninstall it. It does not show up as installed in Ubuntu Software, not does it show as installed when I work from the command line. Part of why I left Windows was because of things getting installed without ever asking. Can someone help me get this off my computer (Teamviewer that is) and help me to prevent this from happening in the future (if such an option exists)? I really appreciate any help or assistance!

RevM

---

### Post by revm on 2021-06-22
So I figured it out. I had Teamviewer "installed" as an app in Google Chrome (which I have installed on my Ubuntu machine) and that is where it was coming from. I removed it from Chrome and it is gone now from the Applications folder. Sorry for the frustration. Hope this helps someone else in the future.
