### Post by texpat on 2016-09-30
If printing from the screen is really the only way for you to get this, I'd suggest you make a screenshot, crop the relevant part out and paste that into a word-processor document which you then print out. That way you can resize the image to fit the paper, giving you enough control to get a printout of what you need.

I like Shutter for making screenshots, because it has a few handy built-in tools for cropping annotating and so on: [http://shutter-project.org](http://shutter-project.org). (Its in the Software Center so you can install from there directly).

On the other hand, I don't know about Ryanair, but other companies give you the boarding documents in PDF, which usually prints quite easily. If you have issues with this, you may want to check the printer settings for margins, or perhaps scale the document down a bit.

### Post by Impavidus on 2016-09-30
I rarely fly (I want to save the climate) and I've never flown Ryanair, so I don't know about their tickets. If it's pdf, then some scaling or setting the margins should work.

My local cinema sends tickets with a bar code in png format. When I print these directly from the gmail web page, parts of the bar code are cut off. But on the web page is a small button with a printer icon and the label "print all". Clicking this button renders the page in printer-friendly form, and then I can print the ticket.
