### Post by kokkus on 2008-09-28
HI and welcome to Ubuntu.

The printer part is easy.
It will go automaticly if your printer is an out-of-the-box printer or  if the driver's already installed.
If not, go to System > Administrator > Printer, add your printer from the list. You can then take a test print to see how it's going.

THe CD/Audio part is also easy as pie.
You can use Rhythmbox. Reads all kind of CD, MD, iPod, iPhones, mobiles and everything. You can also add radio streams from the web. (Pretty cool program)
If you like iTunes, you can go to [www.getdeb.net](www.getdeb.net) search for songbird and install it at once. Songbird is an iTunes clone but reads more formats like ogg and live streaming music / radio.
Another program I can suggest is Banshee. This is more a media player and works great for both music, video and stuff like that.
If you want a simple little tiny player I would go for VLC media player. Can be installed from add/remove.

ANd a must be done before you start using ubuntu is to install this from the terminal: sudo apt-get install ubuntu-restricted-extras.
THen you'll get all the plugins, addons, codecs and stuff you need to perfectly run all kind of video, music and flash files and formats.

Good luck:)

### Post by oldsoundguy on 2008-09-28
[http://ubuntuguide.org/wiki/Ubuntu:Hardy](http://ubuntuguide.org/wiki/Ubuntu:Hardy) (bookmark this!!)

Great guide and will help you with a lot of your questions and with installing.  It will also get you used to using the terminal (applications> accessories) and get you used to copy and paste.  You can open the guide and then place the terminal right on top of the guide and just copy and paste the commands into the terminal.  DO NOT LET IT SCARE YOU.  Almost everyone that uses the system eventually DOES use that terminal and you might as well learn now. (it is the same as the CMD window in Windows)

Now, you will NOT have to use terminal for everything (as a matter of fact, not for a whole heck of a lot .. as there is add/remove (also under applications) and the synaptic package manger(under System> Adminstration)
(for those you do have to set up the repositories (where the TESTED programs are stored) (sort of like taking out a library card for the book mobile.)
The guide shows you how to do that FIRST.

Printer is set up under System> administration> printing> and is self explanatory once there.

Good luck, have fun, and any more questions, remember no such thing as a stupid question.  Only UNANSWERED ONES because they were not asked.

And Google will become your right hand man on lots of items .. with many a search leading you right back to this forum!

(the fun thing .. you will soon have an operating system that will work in the background and do what YOU want it to do .. not what the Operating System designers decided that THEY wanted you to do!)
