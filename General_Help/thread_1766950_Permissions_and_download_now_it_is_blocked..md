### Post by linuxinstalledfromhdd on 2011-05-25
I googled your windows software. I think you are getting a java error with this program because it requires windows Java to be installed to run it. Its not going to use the java that Ubuntu has installed that is used by linux applications. 

So it won't install it. 

Your best bet is LibreOffice. 

Or you can install Virtualbox 4 and install Windows inside of Virtualbox to install your windows software. Then you can install windows java so you can install this product you have. 

You can find out more here about LibreOffice:

[http://cinderbox.net/2011/03/29/libreoffice-ubuntu-ppa-makes-installation-easy/](http://cinderbox.net/2011/03/29/libreoffice-ubuntu-ppa-makes-installation-easy/)

If you want to install Virtualbox 4 you will find it near the bottom of this how-to step-by-step list:

[http://cinderbox.net/2011/04/03/to-do-list-after-installing-ubuntu-11-04-aka-natty-narwhal/](http://cinderbox.net/2011/04/03/to-do-list-after-installing-ubuntu-11-04-aka-natty-narwhal/)
