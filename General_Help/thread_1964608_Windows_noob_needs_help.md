### Post by grahammechanical on 2012-04-24
There is something that you have to remember about Ubuntu. It is designed for ordinary users.

It is still possible to do everything through a Command Line Interface (called the terminal) because Ubuntu is still Linux but much effort is being put into Ubuntu so that things are done using simple, standardized utilities.

First, regarding accessing your programs, you will have noticed in 11.10 the launcher panel on the left side of the screen. The Ubuntu icon on the top of that opens the Dash. with the Dash open type help in the search panel and you will get an icon for the Ubuntu Desktop Guide. Have a look through that guide.

Second, with the Dash opened notice the icons at the bottom of the pane. The second icon from the left gives you access to all your applications. click on that second from the left icon and then click on Installed see xxx more results and you will see icons for all your applications.

Things get better in 12.04 (soon to be released). There we can press Super + A to open the applications lens, super + F to open the file lens, super + M to open the music lens and super + V to open the video lens. Super key = Windows key.

Third, we install and remove programs through the Ubuntu Software Centre. It has the icon of a shopping bag. This is the best way to install/remove programs in Ubuntu.

We can do it other ways. We can download the program as a deb file, which is a kind of archive of the program. Then we can open the file manager and when we right click  on it we can choose to open it with the Software Centre and the Software Centre will install it for us.

Ubuntu being Linux we can also install it using the command line.

Oh, one final thing. Ubuntu has moved away from menu lists and slide out menus. It does not take much to get used to this different way of working. And that is all it is, a different way of doing things that is being improved all the time.

Regards.

Generally with Ubuntu's interface, once you know what you can run you just hit the Windows key and start typing, or pin the program to the Launcher (the sidebar on the left).

Just take my word for it that it's better; it's so much better that all operating systems are moving in this direction :-)
