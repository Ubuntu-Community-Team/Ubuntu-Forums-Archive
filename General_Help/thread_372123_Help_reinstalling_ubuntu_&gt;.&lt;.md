### Post by Cobikegeek on 2007-02-27
No, you just run the cd installer again.  If you have a separate /home partition you can save that by telling the installer not to reformat /home.  Might be something to consider if you have to reinstall anyway. Putting /home on it's own partition saves your data files and settings so reinstalls and updates are easier.
