### Post by wernst on 2005-03-31
Nvu is pretty much the only game in town if you are looking for WYSIWYG HTML editing. However, it doesn't really do site management (that is, asset management and link mapping, and such), and of course, it doesn't support FrontPage Extensions.

It also has no support for Frames.

With these qualifications in mind, Nvu is pretty good for what it is: an editor that will let you work in a page as it will look when it renders. The HTML is fairly clean. Its dialog boxes are a little clunky compared to, say Dreamweaver. It can FTP files to and from your server fairly well.

Bluefish is my favorite non-WYSIWYG editor for Linux.

For complicated stuff I've made in Dreamweaver, well, I still go back to Windows from time to time... Flash too...
