### Post by SirPerigrin on 2008-06-01
Little known secret in the linux world, most binarys are compatible between distributions, especialy within the same distribution. 9/10 times a package for a previous version of ubuntu will work perfect in the latest. 

as for the archetecture, you need the 32 bit install if you used the 32 bit version of ubuntu, and the 64 if you used the 64 bit (x86_64) version of ubuntu. Odds are if you dont know your probobly installed 32 bit. Have a look at your CD, it should be obviously labled (or your ISO if you burned your own)
