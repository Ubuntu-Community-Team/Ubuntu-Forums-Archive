This is a new problem with the state-sponsored login in Denmark, and may be related to the recent discussion about security issues with Java.
I have discussed this with the Danish login team, which could not help me, as well other Danish Ubuntu users, which have had the same (or similar) problem.

I'm using Ubuntu 12.04 (the one officially supported in DK), Firefox 18.0 (it seems to be the only one available in the Software Center?), and my Java-related programs include OpenJDK 7 (with IcedTea plugin), IcedTea Web Control Panel and IcedTea Java Web Start.

My problem is, that whenever I wan't to use my login for contact the authorities, the following message appears in the lower left corner: "Start: Java Applet not initialized." The login-box is not shown.
Curiously, it works every now and then, or every now and then the login-box does blink into view, but instantly dissappears again. Also, java works fine with e-banking in Denmark. 
Other Danish user have suggested that Java isn't the problem - it's how the login-box is imbedded into the website.
