### Post by blackened on 2010-01-18
Do you mean desktop icons or icons in the file manager (nautilus)?

For the desktop icons, as suggested, right-click and select "Stretch Icon..." then manipulate the icon size using the corner grippers that appear.

Icon sizes in nautilus can be altered by changing the zoom level. The default zoom levels can be adjusted by going, within nautilus, to Edit->Preferences and selecting your choices from the dropdown menus on the "Views" tab.

### Post by Ahriman on 2010-01-19
In firefox, right click the toolbar in question and select 'Customize'. Make sure that 'small icons' is checked at the bottom (you can even set it to be text only, no icons). 

Personally, I have the navigation toolbar on the same level as my menu bar, which is again on the same level as my address bar. All are set to be as small as possible, and it gives me LOTS of screen real estate.
