### Post by prshah on 2008-08-23
> **sadhanavs@hotmail.com said:**
> 
When ever I type s like is ls os sudo a new window opens up.


You may have inadvertently setup such a keyboard shortcut: Places to check:

System-Preferences-Keyboard Shortcuts

Or if the problem occurs only in the terminal:

Open a terminal (Applications-Accessories-Terminal), and check in Edit-Keyboard Shortcuts.
