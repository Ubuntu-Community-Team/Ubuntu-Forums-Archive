### Post by ronaldprettyman on 2009-07-02
Once you've added a new repository or ppa, which from what I just read sounds like a new repository.
```
sudo apt-get update
```
and you should be ready to install software from that repository. Your also need a key but that should be provided by who ever is running the repository. I hope this helps.
