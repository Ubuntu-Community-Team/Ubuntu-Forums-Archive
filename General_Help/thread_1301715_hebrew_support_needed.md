### Post by Giblet5 on 2009-10-26
You'll need to install the Hebrew language pack ```
sudo apt-get install language-pack-gnome-he-base
```

Then you'll need to set the system language as US English (System > Administration > Language Support > Click on English > Apply)

Then you'll have to modify EVERY menu item and set the language to Hebrew before execution, just for that application.

You'll also want to keep the keyboard setup tool handy so that you can switch back and forth.....

All of that weirdness is required because you said you want the GUI in English, but all of the apps in Hebrew. It won't be fun.

Just run all-out in Hebrew!
