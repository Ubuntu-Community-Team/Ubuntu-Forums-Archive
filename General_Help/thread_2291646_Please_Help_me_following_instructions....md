### Post by cariboo on 2015-08-21
You can mount iso files to a directory, so you see the files in the directory just like you would if you had burned the iso to a cdrom.

One suggestion, because of the spaces in the name of the iso file, I would enclose it in quotes eg:

```
mount -o loop "Mathworks Matlab R2015a Linux/R2015a-glnxa64.iso" /media/cdrom
```
