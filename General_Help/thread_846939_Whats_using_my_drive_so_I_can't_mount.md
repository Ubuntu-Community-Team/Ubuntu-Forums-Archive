### Post by sdennie on 2008-07-02
Does this show anything?

```

ls /media/Hd2

```

If so and you can't see it on the desktop, hit Alt-F2 and type gconf-editor then navigate /apps/nautilus/desktop and make sure volumes_visible is selected.  You may have to login/out before that change works (if it works).
