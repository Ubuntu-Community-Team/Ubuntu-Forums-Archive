### Post by andrewmv2 on 2015-05-28
I'm using Ubuntu 12.04 for work (this is the latest corporately approved version). I prefer to work with a touchpad, but I find that this makes window resizes very difficult. The default drag handles are very small, and the default keybinding for window resizes is Alt+Middle-Drag; a gesture which is impractical to impossible to accomplish on a buttonless touchpad.

I've installed CompizConfig Settings Manager, and changed the mouse keybinding for "Initiate Window Resize" under the "Resize Window" module, which describes exactly the behavior that I want to change.

This setting is not only completely ignored, but it returns to the factory setting every time I exit and re-enter the module. The same seems to apply to the "Move Window" bindings.

Is there a way that I can quickly resize windows without invoking a middle-mouse-drag under Unity?
