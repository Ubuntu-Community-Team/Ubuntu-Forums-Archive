### Post by kanikilu on 2012-06-14
If you haven't already installed it, the easiest way is to find and install the **Ubuntu Restricted Extras** package from the Software Center. Or from a terminal: ```
sudo apt-get install ubuntu-restricted-extras
```

For more information: [https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)
