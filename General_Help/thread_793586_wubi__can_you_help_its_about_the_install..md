### Post by sunstriker on 2008-05-14
The screen went blank while downloading in windows? Or did your pc already boot into ubuntu. Anyway, the method I used already a couple of times to install wubi is as follows:

Download the full Ubuntu iso (699mb)
Mount or burn the iso in windows (I prefer using daemon tools)
An installer pops up and you can choose wubi.

this way you don't have to download anything (although the installer says so, but it's actually copying from the cd)

Hope this method works for you...
