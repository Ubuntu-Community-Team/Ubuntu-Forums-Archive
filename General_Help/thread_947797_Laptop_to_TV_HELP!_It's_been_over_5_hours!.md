### Post by fenian on 2008-10-14
> I'm not using two monitors. Its my laptop, and my LCD TV! None of it worked!

You are using two monitors the LCD on your laptop (monitor 0) and your LCD tv (monitor 1)

You need to change your xorg.conf file to allow motitor1 (your tv) to display at the resolution you want.

You can do this with the aticonfig tool

-or-

you can edit the xorg.conf file manually,this can be more difficult and any mistakes can cause the GDM (the GUI) to not work properly which appears scarier than it really is.

First backup he xorg.conf file with the command...

> sudo cp  /etc/X11/xorg.conf  /etc/X11/xorg.confbackup

Next open the xorg.conf for editing with the command...

> sudo gedit /etc/X11/xorg.conf

find this section...

Section "Screen"
	Identifier	"screen1"
	Device		"device1"
	Monitor		"monitor1"
	SubSection "Display"
		Depth	24
		Modes		"640x480@60"
	EndSubSection
	Defaultdepth	24

and change it to look like this...

Section "Screen"
	Identifier	"screen1"
	Device		"device1"
	Monitor		"monitor1"
	SubSection "Display"
		Depth	24
		Modes		"1368x768@60"
	EndSubSection
	Defaultdepth	24
EndSection

then find this section...

Section "Monitor"
	Identifier	"monitor1"
	Vendorname	"Plug 'n' Play"
	Modelname	"Plug 'n' Play"
  modeline  "640x480@60" 25.2 640 656 752 800 480 490 492 525 -vsync -hsync
	Gamma	1.0
EndSection

and change it to look like this...

Section "Monitor"
	Identifier	"monitor1"
	Vendorname	"Plug 'n' Play"
	Modelname	"Plug 'n' Play"
  modeline  "1368x768@60" 25.2 640 656 752 800 480 490 492 525 -vsync -hsync
	Gamma	1.0
EndSection


Before you go any further write down this command...

sudo dpkg-reconfigure -phigh xserver-xorg

(In the event that the computer fails to load X and sends you to a command prompt when you reboot you will need to use this to reset the xorg.conf file to defaults)

Reboot the computer.
