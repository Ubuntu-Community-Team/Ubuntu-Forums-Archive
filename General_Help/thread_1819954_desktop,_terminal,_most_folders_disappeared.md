### Post by Wim Sturkenboom on 2011-08-07
If you have know filenames that is missing, try to use *find* and see if they show in.

There are a couple of programs that attempt to recover  lost data. Run them using a live CD or usb stick. [COLOR="Red"]Don't install them[/COLOR].
Recovery might take a long time as well as it's not guaranteed that directory structures and filenames can be restored.

What stops you from putting backups back and continue working? Don't you trust the system anymore? If so, reinstall.
