### Post by bodhi.zazen on 2007-10-13
Usually this happens when there is a problem the ntfs file system.

In the past I have fixed this by booting windows and letting windows do the repairs. This is the downside of using ntfs if you do not use windows. In that case I would advise your convert to a Linux native file system.
