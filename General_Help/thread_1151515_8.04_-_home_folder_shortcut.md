### Post by frunze on 2009-05-07
Hi, I installed Ubuntu 8.04 (LTS) and there is no "Home" folder link on the Desktop. I've had some experience with other distros, and all of them had the link and I got pretty used to it.

Now, when I drag the folder to the desktop, it says "Can't create, the destination folder is inside the source folder".

Is there any way to create such a link in 8.04?

Thanks a lot!

---

### Post by BslBryan on 2009-05-07
Hello, frunze. :-)

It's pretty simple.  Go to Applications --> System Tools --> Configuration Editor, 
then 
Apps --> Nautilus --> Desktop and check the "home folder visible" box.

You can also put, say, a trash icon on your desktop from there.  :-)

I hope I've helped. Best to you, frunze, and good luck. :-)
