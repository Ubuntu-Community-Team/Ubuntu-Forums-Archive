### Post by Cheesemill on 2013-03-09
That depends.

If the disc has been finalised then no, you can't add any more data.

If it is a multi-session CD (it hasn't been finalised) then any of the standard burning apps should let you add another session.

### Post by Mark Phelps on 2013-03-09
You didn't say if the CD was CD-R or CD-RW.  IF it's the first, you can't add any more data -- it can only be written to once.  If it's the second, as Cheesemill indicated, you should be able to add more sessions.

### Post by Cheesemill on 2013-03-09
> You didn't say if the CD was CD-R or CD-RW.  If it's the first, you can't add any more data -- it can only be written to once.
You *can* add more data to CD-R's, but only if they haven't been finalised.

It's called multi-session burning.

[http://doc.opensuse.org/documentation/html/openSUSE_113/opensuse-apps/cha.gnome.burn.html#sec.brasero.multisession](http://doc.opensuse.org/documentation/html/openSUSE_113/opensuse-apps/cha.gnome.burn.html#sec.brasero.multisession)
