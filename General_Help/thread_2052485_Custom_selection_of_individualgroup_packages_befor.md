### Post by 2F4U on 2012-09-03
You cannot select individual packages/package groups with any of the Ubuntu liveCD's I know of (Ubuntu, Kubuntu, Xubuntu, Lubuntu). The server install is kind of minimal in that it comes only with the base system, but can be enhanced later (needs network connection). It is the same with the MinimalCD (you need internet):

[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)
