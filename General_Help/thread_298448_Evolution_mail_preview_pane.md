### Post by DavidS on 2006-11-12
I'm using Edgy, and Evolution 2.8.1 which comes packaged with it.

In Evolution mail, the end of any long message is usually shown in the preview pane, although it's a bit unpredictable.

How do I get the preview always to show the start of a message (and the headers), which would be far more useful?

### Post by wbxv on 2007-03-01
Hi, I had the same problem with Evolution 2.8.1 on Edgy Eft.  I tried all sorts of voodoo without success until I did the following steps.

1) Open a message by double-clicking on it.
2) Deselecting 'Caret Mode' in the 'View' menu on the newly opened window.

This fixed it for me. Later I found that deselecting 'Caret Mode' in the 'View' menu of the main Evolution window works too.  Caret Mode seems to give you a cursor at the end of a message, so this was making long messages scroll to the end.

Hope this works for you.
