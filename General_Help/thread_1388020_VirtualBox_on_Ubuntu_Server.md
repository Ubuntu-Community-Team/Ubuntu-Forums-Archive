### Post by marshmallow1304 on 2010-01-22
How did you install Virtualbox?  The version from the Ubuntu repositories is the open-source edition and does not support USB (see [here]("http://www.virtualbox.org/wiki/Editions") for editions).  The solution would be to install the closed-source edition from [here]("http://www.virtualbox.org/wiki/Linux_Downloads").  If you scroll down on that page, you'll also find a repository you can add to make sure you stay up-to-date.
