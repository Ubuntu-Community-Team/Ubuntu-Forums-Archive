### Post by moephan on 2007-07-03
I don't know of an easier way to do it in the GUI. However, if you are renaming a lot of items, you might consider using the command line. To rename you use the "mv" command:

```

$mv oldfilename newfilename

```

You can also use mv to change file locations instead, or change the location and rename at the same time.

HTH

Cheers, Rick
