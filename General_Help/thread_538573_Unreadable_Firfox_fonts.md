### Post by jim_p on 2007-08-30
The site looks ok for me in opera and in firefox.

In firefox you can go to Preferences > Content > Fonts and Colors and change the default font there.
If you look for more specific font settings click Advanced and change the fonts you dont like
My font in both is Dejavu-Sans

EDIT:
I just found that the site designer has assigned these fonts to be used in the site:
Sans, Tahoma, Helvetica

This means that the browser will search your system for a Sans-type font (like Dejavu-Sans) to use,
if it fails, it will fall back to Tahoma and if Tahoma fails too it will fall back to Helvetica
