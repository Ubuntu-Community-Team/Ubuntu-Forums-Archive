### Post by Favux on 2009-02-22
Hi daller,

The cursor jumping (mostly) to a corner, and staying there, is usually a symptom that Xserver isn't getting any input.  My first thought is your xorg.conf is configuring the "CorePointer" which should be your mouse to the touchscreen.  So let's first try changing:
```
Section "ServerLayout"
        Identifier      "Default Layout"
        Screen          "Default Screen"
        #InputDevice "WizardPen Tablet" "SendCoreEvents"
        InputDevice "touchscreen" "CorePointer"
EndSection
```
to
```
Section "ServerLayout"
        Identifier      "Default Layout"
        Screen          "Default Screen"
        #InputDevice "WizardPen Tablet" "SendCoreEvents"
        InputDevice     "touchscreen"
EndSection
```
And let's go from there.

What verison of Ubuntu are you using?
