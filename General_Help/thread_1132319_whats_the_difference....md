### Post by Yashiro on 2009-04-21
**Ubuntu** uses the Gnome desktop. Most common feature filled environment.
**Kubuntu** uses the KDE desktop. Alternative to Gnome, more cutting edge features.
**Xubuntu** uses the Xfce desktop. A less feature filled environment that is somewhat similar to Gnome.
**Mythbuntu** is a TV extender/media center.

You can use the same programs in all of them. The only real difference is the desktop experience/performance.
