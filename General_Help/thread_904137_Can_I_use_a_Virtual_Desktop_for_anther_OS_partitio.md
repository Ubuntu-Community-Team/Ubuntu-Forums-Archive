### Post by quibbler on 2008-08-29
Virtual desktop, either in windows or ubuntu are just a large file on the respective
operating system.
Any changes you make to settings in programs stay only in the virtual drive.
If you make say a doc file in word you could share it between the virtual drive in ubuntu and the true windows by saving it in a shared folder. You could do the same in Ubuntu.
You can't "quickly" switch between windows and ubuntu..you have to reboot and go into the other OS.
