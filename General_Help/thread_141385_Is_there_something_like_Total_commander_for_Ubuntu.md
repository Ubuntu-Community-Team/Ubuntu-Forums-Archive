### Post by az on 2006-03-08
Midnight commander is the only file manager one really needs....

It is in universe and has it's own text editor.

sudo apt-get install mc

Be sure to enable lynx-like motion in the configuration.  If you run it from the gnome terminal, you need to dissable gnome-terminal's use of F10 so that you can exit mc.
