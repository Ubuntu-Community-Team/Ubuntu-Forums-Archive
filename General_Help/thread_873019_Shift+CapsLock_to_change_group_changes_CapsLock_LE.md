### Post by jazzgossen on 2008-07-28
I have two keyboard layouts, and have group shifting configured so that shift+capslock changes the group. This has always worked, and still does.

However, since 8.04, pressing shift+capslock changes the group *and* toggles the capslock LED, even though the capslock state doesn't change. So the capslock LED on the keyboard doesn't actually reflect the capslock state anymore. According to [this thread]("http://www.nabble.com/capslock-led-after-upgrade-to-xorg-server-1.4.2,1-td18007012.html"), it seems to be an Xorg thing. Does anybody have any other news on this, or a workaround?
