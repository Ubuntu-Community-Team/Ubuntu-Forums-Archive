### Post by taurus on 2008-01-31
Well, you have one of those Intel onboard graphic controllers.

```

00:02.0 VGA compatible controller: Intel Corporation Mobile GM965/GL960 Integrated Graphics Controller (rev 0c)

```
Have you looked in System -> Administration -> Restricted Drivers Manager to see if there is a restricted driver for that?  

What video driver are you using right now for your card.  Look in /etc/X11/xorg.conf especially the line that says Driver.

```
cat /etc/X11/xorg.conf
```
