### Post by gingivere0 on 2010-06-12
You need Ubuntu Tweak to do this.  Type this into a terminal:

```
sudo add-apt-repository ppa:tualatrix/ubuntu-tweak
sudo apt-get update
sudo apt-get install ubuntu-tweak
```

Once all of that is finished, run Ubuntu Tweak by going to Applications>System Tools>Ubuntu Tweak.  Under **Desktop** look for GNOME Settings and click on that. Now look under menu settings on the new page and the last thing under menu settings says "Click this button to change the menu logo image."  Click the button and navigate to the picture that you want the menu button to look like.  Hope this works! Good Luck!
