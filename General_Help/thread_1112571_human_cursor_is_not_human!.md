### Post by mb_webguy on 2009-04-02
If you're running Compiz, there are actually two places to set the cursor theme.  The normal place is System->Preferences->Appearance->Theme, Customize->Pointer.  The other is System->Preferences->Advanced Desktop Effects Settings->General Options->Cursor Theme.  These should match, though the exact name needed in Compiz is the filename of the cursor theme in /usr/share/icons, rather than the name shown in the Appearance dialog.
