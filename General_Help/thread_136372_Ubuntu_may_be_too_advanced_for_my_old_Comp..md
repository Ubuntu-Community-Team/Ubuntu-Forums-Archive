### Post by az on 2006-02-25
Open up synaptic, add the universe repository and install the menu and icewm packages.  Then, log out and before you log in again, use the icewm session.  It is much faster.


Try that before reinstalling.

### Post by aysiu on 2006-02-25
You can try Damn Small Linux.

Or you can do a server install of Ubuntu, [enable extra repositories](http://www.psychocats.net/linux/sources.php), and then install Xubuntu ```
sudo apt-get update
sudo apt-get install xubuntu-desktop
```
