### Post by mckenna1977 on 2009-02-26
try doing route -n to print your routes

you should have a route which to destination 0.0.0.0 through a gateway. Maybe you've none or a wrong gateway.

do eth -a to view all connections and select the one your wireless is on then add the internet destination with your dhcp server as gateway so something like : 

sudo route add -net 0.0.0.0 default gw 192.168.10.1 dev wlan01

Hopefully that'll do the trick for you....
