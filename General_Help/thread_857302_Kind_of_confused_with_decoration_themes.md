### Post by joshdudeha on 2008-07-12
I assume you are using GNOME.

Well, metacity is the default window manager for gnome - and if you're using 8.04 then you will also be using GTK 2x

The Metacity part, does the window border and captions and stuff
The GTK 2x is the actual theme for things like widgets and so on.

So, basically - to theme your computer, download a GTK 2x theme and drop it into the Theme manager, same with metacity.
If you want to use Beryl Emerald themes, install Emerald via synaptic and use that.

Hope I clarified it for you.
