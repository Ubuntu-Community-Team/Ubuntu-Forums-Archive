### Post by ajgreeny on 2022-01-09
From the **File -> Save Page As** menu you can save the single page as an html file; just go to the bottom right of the save dialogue and choose  what you want, ***Web Page- HTML only*** or for folder with separate files choose ***Web Page Complete***.

### Post by KBar on 2022-01-09
You're supposed to end up with at least two files: the resources in one directory and the actual document itself. The resources contain all of the elements of the page, like images, scripts, style sheets and so on and so forth. It's not like any other file.
