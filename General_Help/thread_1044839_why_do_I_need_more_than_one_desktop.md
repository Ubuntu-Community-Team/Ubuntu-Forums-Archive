### Post by mcduck on 2009-01-20
Sounds like graphics card issue or something like that.

Running multiple apps on multiple desktops doesn't really use any more RAM or resources as running the same apps on one desktop would.

I've been happily running 4&#8211;6 desktops on my computers since my old 450Mhz AMD-K6-2 machine that had 128MB of RAM. And virtual desktops have been available on Linux & Unix desktop environments way longer than that.. I believe Amiga had them first in 1985..

I usually have 4 desktops, which seems to be just the right amount for building web sites: Firefox on 1st desktop (so I can see what the page looks like), text editor (Gedit) & file manager on second desktop, graphics editor (Gimp or Inkscape depending on style of the current project) on 3rd desktop and the 4th desktop is for everything else (instant messaging, IRC, music player). I never need to minimize a window, and can jump between tasks easily with keyboard shortcuts.

Once you get used to having couple of virtual desktops you'll have hard time living without them. Only thing that beats having multiple desktops is having multiple monitors (and even when I have 2 monitors I still use at least 2 virtual desktops.)

Also, for future reference, when I am in windows, I know where yo find my hardware specs.  Where do I go to find them in Ubuntu 8.10?

Open a terminal (Applications -> Accessories -> Terminal)
and post the output of:
```
lspci | grep VGA
```

To list the hardware specs in the terminal:
```
sudo lshw
```
or you can create a html file on your desktop and open it with your web browser (firefox):
```
sudo lshw -html > ~/Desktop/hardware_info.html
```
```
firefox ~/Desktop/hardware_info.html
```

Or you can try a GUI application. Search for hardinfo in Synaptic or Add/Remove and install it.
Or use the terminal to install the app:
```
sudo apt-get update
```
```
sudo apt-get install hardinfo
```
```
hardinfo
```

**EDIT: is gnome-device-manager installed by default (Application -> System Tools -> Device Manager)?**

### Post by Montblanc_Kupo on 2009-01-21
A lot of times I will have multiple items open on once screen so I can see them all at once... but then I will uses multiple desktops so I can have a couple applications open but have them using the entire screen. Since linux, like mac, and unlike windows heh... keeps things going live while you work you can check on things that are running often without fully switching to the desktop.
