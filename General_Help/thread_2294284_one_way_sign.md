### Post by grahammechanical on 2015-09-10
Go to System Settings>Software & Updates>Other Software and disable that PPA.

Personal Package Archives (PPA) are useful for installing software that is not in the official Ubuntu software archives (repositories). Some PPAs will provide updates. Others do not and sometimes the existence of an enabled PPA will provoke a warning indicator from software updater.
