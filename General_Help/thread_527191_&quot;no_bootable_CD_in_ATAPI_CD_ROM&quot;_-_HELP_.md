### Post by Yoooder on 2007-08-16
When you downloaded the CD, you downloaded an "ISO" which is a complete image of a CD, and when burned correctly the CD will show many files on it.  If you just burned the file to a CD then it will not boot.

A guide on burning Ubuntu CD's can be found at:
> [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

If you followed such a guide, or your CD shows many files on it (check it with Windows explorer) then let us know.
