### Post by AlphaLexman on 2011-04-08
Really neither is better or worse than the other. They both use different toolkits or framework to draw and render the GUI.

The toolkits are gnome/ubuntu = gtk and kde/kubuntu = qt.

You can run qt programs in a gtk environment, and likewise,
you can run gtk programs in a qt environment, the downside is,

for either to work you MUST install many libraries from the other toolkit, so it can weigh on your system.

If you have a modern, fast machine, it may not make a difference.
If you have an older machine, running extra libraries, can significantly slow down the overall speed.
