### Post by lrPrentice on 2011-07-15
Hello,

My 11.04 system chooses image viewer when I try to open a *.pdf file. But image viewer gives me  "Could not load image" error message.

If I right click, however, select either Adobe Reader or document viewer, the file opens fine.

How can assign the task of opening *.pdf files to either Adobe Reader or document viewer instead of image viewer?

Many thanks,

LRP

---

### Post by flipper T on 2011-07-15
right click on a pdf

go to "open with other app"

find the app of your choice

tick the box that says "always open with this app"

---

### Post by ~!geek!~ on 2011-07-15
To set a default application for any type of file,
Right click on file, select properties. Go to open with tab. Select appropriate application. Close the properties window and done. Now clicking the file will open (or try to open) it with the application of your choice.
