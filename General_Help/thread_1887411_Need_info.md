### Post by 3rdalbum on 2011-11-27
What exactly do you mean by "full cloud base"?

Windows is going toward a single-tasking tablet UI by default, for everyone. Ubuntu is trying to strike more of a happy medium - a UI that is tablet-friendly but is still usable on a desktop computer. And, importantly, makes it easy to multi-task, even on a tablet. Unlike Windows "Metro" you will still be able to have multiple windows open and visible simultaneously.
