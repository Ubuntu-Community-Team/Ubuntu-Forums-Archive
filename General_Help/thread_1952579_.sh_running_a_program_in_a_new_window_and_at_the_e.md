### Post by forrestcupp on 2012-04-04
Just put **gnome-terminal** at the end of your code, and it will open a terminal. The only problem is that after the code is run, it will exit, and you'll never see the terminal. So you'll have to have some kind of user input at the end or something, unless you intend to invoke it from a terminal.

Edit: I don't know how to do it. But gnome-terminal is the command for the terminal.
