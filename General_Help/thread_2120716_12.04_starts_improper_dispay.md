### Post by Mr. Shannon on 2013-02-27
Are you saying that you are trying to use gksudo from tty2?  tty2 will not be running an x session by default so the "no display available" error is expected.  On tty2 you either need to start the x server (probably not a good idea since you are having graphics issues) or use all terminal based applications (using sudo for root privileges if needed and not gksudo).

gksudo is for graphical applications.

As far as fixing your graphical environment you could try:
```
unity --reset
```
which can be found by searching for how to reset Unity.


Hopefully someone who knows more about Unity will be able to help you restore you graphical environment if that does not work.
