### Post by bytemangler on 2010-06-18
Shutter - Screenshot Tool (installable via Ubuntu Software Center) comes pretty close to jing. Not anywhere near as polished. Annotation is a little more work, but it will do the job.  Instead of having set annotation bubbles, Shutter provides a drawing tool to add annotations, lines, shapes to your screenshot.
