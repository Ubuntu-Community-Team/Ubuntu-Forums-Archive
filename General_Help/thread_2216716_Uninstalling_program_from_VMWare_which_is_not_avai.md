### Post by Double.J on 2014-04-13
Hi there, I'm a little bit confused what you are trying to remove, but if you know the name of the program you can search for it in synaptic. These will show you all packages with those names. A green box to the left means installed, if you click on the package you get a more detailed description, then right click on it and mark for removal. Click apply and you're done.

You do not need an uninstalled program like windows if that is the question?
