### Post by dayanandasaraswati on 2008-02-16
The problem of Hal.dll can be fixed by putting your XP cd and selecting repair from it. That option will reinstall your full XP without deleting a single software or data. It'll reinstall all windows files and not at all touch any user data. So that option can be used to repair this problem. 
Now for your problem 2: Ubuntu will not mount any NTFS file system unless and untill they have been cleanly unmounted. Had you hibernated windows or had you not shut windows down cleanly this error would occur. Get back to windows and  perform a fulll shutdown
