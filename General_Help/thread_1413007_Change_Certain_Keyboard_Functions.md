### Post by labinnsw on 2010-02-24
If the information in [this link]("https://help.ubuntu.com/community/MultimediaKeys") does not work, Try installing Keytouch and Keytouch Editor from the repositories. Use Keytouch Editor to change the key functions. Use Keytouch to apply the changes.
