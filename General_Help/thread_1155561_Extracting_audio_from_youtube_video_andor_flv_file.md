### Post by TenPlus1 on 2009-05-10
Download your youtube or flv file and install SoundConverter from the add/remove tool in Ubuntu...  This will let you drag the .flv file into the window and convert it to an mp3 or ogg sound file... :)

To actually download a youtube file, place 'pwn' at the start of the address:

[http://www.youtube.com/watch?v=lK7KddvdY6Y](http://www.youtube.com/watch?v=lK7KddvdY6Y)

change to

[http://www.pwnyoutube.com/watch?v=lK7KddvdY6Y](http://www.pwnyoutube.com/watch?v=lK7KddvdY6Y)

and it will let you download the movie file :)  enjoy!
