Look in System -> Preferences -> Appearance -> Interface

### Post by OrangeCrate on 2009-11-12
Gnome 2.28

> 
GNOME menus and buttons have been standardized across all applications to not display icons by default.


[http://library.gnome.org/misc/release-notes/2.28/](http://library.gnome.org/misc/release-notes/2.28/)

### Post by wojox on 2009-11-12
Yeah, like mc4man said System -> Preferences -> Appearance -> Interface

They turned them off by default.
