### Post by Vaphell on 2010-07-18
in my compiz config that option is available under Window Management stuff, something like Fast Switch/ fast selection (have no idea how it's in english, sorry). It's icon is two small windows next to each other.

by default it shows all workspace windows with WIN+W, all windows across all workspaces WIN+A. If you find it in Compiz Manager you can add trigger that reacts to mouse hitting a chosen corner to get the desired functionality (though i forgot how :D)

edit: in preferences you need to enable an appropriate trigger with monitor icon and choose which corner/edge is to react.
