### Post by jespdj on 2008-04-15
ATI's drivers for Linux just aren't very good. They don't support a number of 3D graphics features and so it can be difficult (or impossible) to get 3D graphics working properly on an ATI card. As far as I know this is the same for any ATI card, old or new. AMD/ATI promised in the summer of 2007 that they were going to improve their Linux drivers, but AFAIK there has not been a revolution in this yet.

nVidia's Linux drivers are a lot better, so your best choice for Linux is an nVidia card.
