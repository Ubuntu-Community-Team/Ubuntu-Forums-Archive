### Post by jombeewoof on 2008-07-23
You need to install KDE, then customize your main and secondary panels.

Right click the panel, choose edit or whatever. 
Set it to dock in the center, and it looks pretty close to 60% or so width. 
Then you can set transparency if you want in the same place. But first of all you need to install kubuntu-desktop.

That might be xfce, but I dont' know how to make XFCE pretty like that.

### Post by Martin Marshalek on 2009-01-02
Actually, in GNOME you can achieve this look by right-clicking on the panel, selecting Properties and then increasing the size (in pixels) of the Panel and de-selecting Expand.
