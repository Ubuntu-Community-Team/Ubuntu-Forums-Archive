### Post by ian-weisser on 2014-01-18
Unity is specifically designed to avoid the put-lots-of-stuff-on-the-desktop-so-you-can-find-it workflow.
And, yeah, it can be a hard habit to break.
But very much worthwhile.

If you decide that you really do want to keep your stuff on the desktop, and you want to live without the cool Unity features, then Ubuntu has lots of other desktop environments like XFCE and KDE that are easy to install using Software Center.

### Post by deadflowr on 2014-01-18
The workaround for this odd request(odd because it becomes redundant since the launched app will have an icon in the left-panel-like launcher) is to copy the application from
/usr/share/applications into the home folder's desktop folder.
Open the file manager and go to the side panel option "computer"
then go to the section usr
then to the section share
then applications
now find the app you want and right click it, then choose copy.
Now go the side panel option Desktop
Open the desktop folder and paste.

Sidenote: Big +1 to simply using the system's designed launcher methods.
