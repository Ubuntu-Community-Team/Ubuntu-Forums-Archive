### Post by Omnomnom on 2010-10-30
If you have any files you wish to keep, back them up by booting a live CD NOW.

It's probably caused by the download for the update. Packet loss, corrupt bits, and others. Now go ahead and download the actual liveCD for the new ubuntu, if you have a windows partition or something. If you don't, just reinstall ubuntu and get it. If you do, burn it to a disc (Durr) and reinstall over the current ubuntu partition. If all else fails, reinstall it. ;)
