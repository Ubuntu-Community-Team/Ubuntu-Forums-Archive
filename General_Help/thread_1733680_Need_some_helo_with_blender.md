### Post by themusicalduck on 2011-04-19
You can install Blender from the Software Centre. Installing it this way will install it into the proper place, add menu entries and it will keep itself updated. Don't need to download it unless you want the absolute latest version and if you do then you may be able to find a PPA to use instead. 

If you do want to run the version you downloaded, you can unpack it by right clicking the file and selecting to Extract here. Then you should be able to find the binary inside the extracted folder and run it by double clicking on it. You might need to set it as executable first, by right clicking the binary, going to Properties and looking on the permissions tab.

Since you just came from Windows, programs are installed a bit differently in Ubuntu. Generally you just install everything you need by searching through the Software Centre. If you can't find it there, then look into finding a PPA for your software, or installing using a .deb file downloaded from a trusted source.
