### Post by marshag63 on 2008-12-14
With so low ram, you should pick from the UNC and UCI programs as they are for low ram.  You can download them and save them to a cd or flash drive.   See instructions in, I believe, the Dillo info page that comes up when you start DSL for installation instructions, etc.

[http://distro.ibiblio.org/pub/linux/distributions/damnsmall/mydsl/](http://distro.ibiblio.org/pub/linux/distributions/damnsmall/mydsl/)



Something that helps with low ram systems is to make a Swap partition that is at least twice the amount of ram you have, if possible, depending on how big the harddrive is.

Regarding internet, you cannot use the built-in modem - for desktops you will need an external serial modem (not usb).  You can find instructions on setting up the modem at the forums.  For a laptop, you can use a PCMCIA modem card or even an external serial modem.


I used to run DamnSmallLinux on a 166 Mhz laptop with 32 MB ram with "X" running (but not lightening fast - for true speed try no "X" at all - did that too a few months ago because I had trouble setting up "X" on a computer with a dying harddrive - learned an incredible amount - could browse files/directories, do IRC chat, e-mail, and surf the net from the command line.

Let us know how things go for you.

MarshaG.
