+1

You can't just drag and drop an .iso file to a disk...it has to be installed from the .iso.  An .iso file will have that suffix attached to it; i.e., "<filename>.iso".

Almost every time I've burned an .iso to disk, all I had to do was double-click on the .iso file, and if I had the proper burning software installed, it would launch and install it to the disk.
