### Post by markginter24 on 2008-11-03
If you're using the fglrx driver then you have to use the aticonfig tool.  It gives a quick 'help' section after simply typing the command and hitting enter.

Otherwise -- revert to using the radeon driver with Intrepid and then you can use xrandr to setup dualhead stuff -- but you'll have to add the Virtual line to your x config file first.  Google for xrandr and virtual and you'll find quick instructions on how to do it.
