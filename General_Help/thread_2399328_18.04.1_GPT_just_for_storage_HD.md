### Post by sudodus on 2018-08-24
When the drive size exceeds 2 TB you need GPT to manage it correctly (to use the whole size), and modern operating systems can manage it.

- Ubuntu 18.04.1 LTS can manage a 3 TB drive with GPT :-)

- Windows 10 can manage it too. I am not sure about Windows 7, but I think it can manage it.

### Post by oldfred on 2018-08-24
If drive may ever be used in an UEFI system, better to use gpt now.
I started using gpt with my BIOS only system, but when Windows 8 required UEFI/gpt, I started adding both an ESP (future use) & bios_grub (BIOS boot) to every new or reformatted drive. It was several years before I then got my first UEFI system.
Only if drive will get a BIOS install of Windows would you use MBR(msdos).
