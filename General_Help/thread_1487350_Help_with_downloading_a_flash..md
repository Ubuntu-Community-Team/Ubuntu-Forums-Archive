### Post by parn on 2010-05-18
If you are trying to save a flash embedded movie or something, they are located in your /tmp folder. Dont really need any plugins, just copy and paste. They usually have the file name Flaxxxxxx

Only 1 thing, as soon as u navigate away from the page (sometimes the flash is made to do this), the file is lost - so pause the flash - when it is done loading, copy it out.

### Post by 3rdalbum on 2010-05-19
If you use the "Page Info" function of your web browser (it's usually in one of the menus) you can find the URL of the actual Flash file, then you can copy and paste that into a wget command or a download manager.

Note that the flash file I'm referring to is NOT a video file, it's the actual program that's running in your browser.
