### Post by Peter09 on 2008-11-16
Although I believe it is possible to get virtual box to run an already installed windows system its not an 'official' thing to do and is a bit tricky.

VirtualBox allows you to run other operating systems within a window in the Host system. You install the 'guest' operating system through virtualbox (although the process is similar to a normal install).

It is very easy to do - worth a try.
