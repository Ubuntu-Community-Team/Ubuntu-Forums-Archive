### Post by condorloco on 2005-06-16
After installing 'mldonkey-server' and 'mldonkey-gui' with synaptic, run 'mlnet' in a terminal (or through 'run application'). Then, while mlnet is still running (DON'T close the terminal if you started mlnet there!), run 'mlgui'. I hope this solves your problem.
