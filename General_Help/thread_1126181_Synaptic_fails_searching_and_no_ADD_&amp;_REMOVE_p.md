### Post by Bucky Ball on 2009-04-15
You will find the manual install simple, especially if you have a free hard disk to install to. The Desktop CD will automatically partition and format it for you! (although not always how you want but you can change that later).

Download and burn the Ubuntu Desktop ISO, make a cd with it and boot. It is all GUI from start to finish. Even asks you to remove the CD and reboot at the end. All you have to do is fill in some details.

It is handy if you have a working, online computer alongside. There are a heap of how-tos you could reference as you go. This one makes it easy: 

[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)

Wubi is not always a reliable permanent install - I've had problems with it myself, but might just be me.

### Post by 3rdalbum on 2009-04-15
It just sounds to me like you don't have any repositories enabled. Go to System > Administration > Software Sources and enable the main, universe, multiverse, restricted, security, updates and backports repositories. Make sure you have some sort of internet connection and then click "Close". You will be asked if you want to reload the package list; click Reload.

EDIT: Reinstalling Synaptic will do diddly-squat. Your copy of Synaptic is identical to the stock Ubuntu copy and no amount of use changes that; programs don't suffer from wear and tear as they are just pieces of code.
