Is there a way to view/edit an open office document in MS Word? and vice versa?

### Post by canadianwriterman on 2007-04-24
OpenOffice will open most documents of any format. MS Office is more restrictive, and will not open a "native" OpenOffice format document. The easiest thing to do is set OpenOffice to save documents in a format that MS Office will read.

### Post by James SoCal USA on 2007-04-24
Save in either .rtf or .doc format for the version of word you have installed, if there is one for that version.  As long as the document is fairly basic there will likely be no rapairs needed but in some cases it does not save with all the formatting as it should.  I have documents from word 95, 98 & 2000 which have been read into OO with few changes needed and have saved them out in the 98/2k/XP format which were then readabale in XP with few changes needed.  I have found the .rtf about as good with OO as it is with word - basically semi-worthless with many changes often needed.  Excel and PowerPoint files work the same way.  I do all my presentation work in OO and save one file in OO format and one in .ppt format for M$ XP at school.  Again few changes needed.  Hope this helps a bit at least.
