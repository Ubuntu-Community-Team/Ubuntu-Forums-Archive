### Post by Bucky Ball on 2014-06-14
Welcome. Presuming the file manager is PCmanFM? Go to 'View' in the top bar of the file manager window then choose the view you want. If you want to change it permanently, then Edit>Preferences>General>Default View and change it there in the drop-down list.
