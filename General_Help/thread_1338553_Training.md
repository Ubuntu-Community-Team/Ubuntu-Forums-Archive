### Post by J V on 2009-11-26
Gratz on converting a school :popcorn:

Well first I would suggest edubuntu, no games, some education software, designed for networking (Schools tend to like to watch what their students are doing on computers)


[LIST]
[*]Don't be afraid of commandline, there are very few dangerous commands (And you should look all commands up with "man *command*" first)
[*]If you learn to write a script, you can run 100 commandline tasks in a few seconds, I use it when migrating to a new version so I don't spend forever installing the software
[*]Each user has different preferences, a single user per PC might mean they all become a bit different, instead you can have all the users log in and ubuntu send the information to a server, so every user has their own preferences
[*]Don't give students administration access :P
[*]Installing ubuntu-restricted-extras will help
[/LIST]
That help a bit? :P
