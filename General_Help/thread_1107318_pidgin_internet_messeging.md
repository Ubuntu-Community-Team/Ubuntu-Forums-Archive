### Post by evilaim on 2009-03-26
Well, luckily for you, IRC is an open server based protocol.  Meaning you don't need an account to start.  Here: 

[http://thecyberian.wordpress.com/2008/05/09/irc-tutorial-how-to-set-up-pidgin-in-5-easy-steps/](http://thecyberian.wordpress.com/2008/05/09/irc-tutorial-how-to-set-up-pidgin-in-5-easy-steps/)

but instead of there servers and channels fill it in like this:

screen name: Whatever You Want, (I'm EvilAIM)
server: irc.easynews.org
password: LEAVE BLANK

and when you connect, the Channel is #ubuntu.

(the official server I think is irc.freenode.net, but they have a bunch of services which are complicated if you're not used to them.
