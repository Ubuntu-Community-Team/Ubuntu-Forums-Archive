I am running 9.10 - update manager (at the top o the window) shows me that a new release of 10.04 LTS is available - together with an **Upgrade** button. Is it possible to to have an update manager window that does not have the option to Upgrade to 10.04?

### Post by -kg- on 2010-07-15
Yes.  Now I'm using Lucid, but it should be much the same.

Launch "System/Administration/Software Source" and click on the "Updates" tab.  There you should see something like "Show New Distribution Releases" with a drop-down menu.  One of those selections should be "Never".
