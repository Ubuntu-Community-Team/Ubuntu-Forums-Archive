### Post by tvtech on 2008-11-12
server is command line only.  

server uses considerably less resources thus cheeper to run he computer and more computational power to serve up your website and any server side applications associated with it.  

if you don't care you can run apache behind a ubuntu gui,  you just loose a lot of proc power to running x server and windows manager is all.

the short of it if you have a real server system and are serving a high volume website YES IT matters.

the long of it if your doing this for fun and serving something up that maybe 20 or 30 people will visit a day, go with the gui, that way you can play with the computer while it's serving up your site.
