### Post by sikander3786 on 2011-12-22
Which version of Ubuntu are you using? If using 11.04 or older, open the screensaver properties dialog and at the bottom of the list, you'll see the option for setting your 'Pictures' folder as screensaver.

If using Ubuntu 11.10, it doesn't come with a screensaver at all. You'll need to install XScreenSaver however I am not exactly sure if it supports pictures or not. It should, though!

[http://www.tuxgarage.com/2011/10/installing-xscreensaver-in-11-10-ubuntu.html](http://www.tuxgarage.com/2011/10/installing-xscreensaver-in-11-10-ubuntu.html)

### Post by ajgreeny on 2011-12-22
xscreensaver has a GL-slideshow which you can set to do lots of different things, ie change folder used, set time of each image, set panning and zoom, etc etc.  It is much better than gnome-screensaver, in my opinion
