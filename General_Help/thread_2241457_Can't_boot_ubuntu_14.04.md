### Post by yancek on 2014-08-26
I think we would need to get more details on your drives/partition, uuids and boot files.  The simples way to do that is to use the bootinfoscript.  If you use any Linux Live CD/flash drive, open a web browser and just google bootinfoscript and go to the site and read the instructions on how to use it in the link in the Description box.  Then downloads and run it and post the output which is a results.txt file.  Someone should be able to advise you what to do.
