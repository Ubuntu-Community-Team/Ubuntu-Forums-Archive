### Post by bobcollard on 2010-12-16
Try a Live CD of both Ubuntu and Xubuntu.  The difference between Gnome and Xfce can make the speed different.  As for your memory and CPU, think about Linux and what it has in it.  First the System is smaller than Windows and second there is a Swap partition that acts like RAM if you run short.  Try it, you'll like it.
