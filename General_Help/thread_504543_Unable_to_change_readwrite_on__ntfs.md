### Post by taurus on 2007-07-19
```
sudo apt-get update
sudo apt-get install ntfs-3g
```
Then, edit /etc/fstab and add an entry for your ntfs filesystem.

```
gksudo gedit /etc/fstab
```
and add this line to the end of it.  (Also, you need to remove the second to the last line in there since it's a wrong entry, [COLOR="Red"]/dev/hdb /media/<mount point> ntfs-3g 0 0[/COLOR].)

```
/dev/hdb1   /media/hdb1   ntfs-3g   defaults,locale=en_US.utf8   0   0
```
Save it and exit the editing window.  Then, create a new mount point and mount it.

```
sudo mkdir /media/hdb1
sudo mount -a
df -h
```

### Post by taurus on 2007-07-19

```
gksudo gedit /etc/fstab
```
and replace this line

```
/dev/hdb1   /media/hdb1   ntfs-3g   defaults,locale=en_US.utf8   0   0
```
with this new one.

```
/dev/sda5   /media/sda5   ntfs-3g   defaults,locale=en_US.utf8   0   0
```

```
sudo mkdir /media/sda5
sudo mount -a
df -h
```
