### Post by prshah on 2010-08-21
> **Zylstra555 said:**
> how well PalmOS works with Ubuntu when it comes to synchronizing calendar, notes, and things like that. Much as I absolutely love Windows Mobile, 

First: PalmOS != Windows Mobile 

Though there are PalmOne phones that use Windows Mobile, they are not PalmOS, and don't sync with Palm Desktop.

PalmOS phones such as the Treo650/600 and devices such as Zire71/72 work very well with both Evolution as well as linux native programs such as jpilot. Connecting and syncing is effortless (at most requiring a "modprobe visor", if at all).

Windows mobile phones are another cup of tea. I suggest you can start with the [Community Documentation for Windows Mobile]("https://help.ubuntu.com/community/PortableDevices/WindowsMobile").

Please post back if you need more clarifications / help.
