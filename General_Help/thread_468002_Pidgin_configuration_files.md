### Post by stami on 2007-06-08
Hi,
Pidgin/Gaim configuration files reside in your $HOME/.gaim directory. This folder is automatically created the first time you  open gaim. Once you add you account you should have something similar in your .gaim directory:

```

accels
accounts.xml
blist.xml
icons
prefs.xml
smileys
status.xml

```

Icon and smileys are two directory, the other are files
