### Post by natekanguru on 2011-11-01
Hello, 
 
I suggest you open up a support ticket with Kanguru's technical support team ([http://support.kanguru.com/](http://support.kanguru.com/)).  One of our technicians should be able to work with you on the problem.  One item of note is that there have recently been some Linux kernel updates which are causing some USB enumeration issues for certain file formats.  These are largely version dependent.  
 
Also, to address the other note posted, the Kanguru Defender Elite does support cross platform use (Windows, Mac, and Linux).  All cryptographic operations are done within the onboard crypto chip.  
 
Thank you.
