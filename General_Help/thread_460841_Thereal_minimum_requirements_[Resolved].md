### Post by aysiu on 2007-06-01
256 MB of RAM is plenty for running Ubuntu once it's installed, but if you're trying to install Ubuntu using the Desktop CD, 256 MB won't be enough, as the Desktop CD runs a fully functional live session while installing.

You'll need to use the Alternate CD to install Ubuntu.

P.S. For even better performance, you might want to install Xubuntu instead of Ubuntu:
[http://www.xubuntu.org/get](http://www.xubuntu.org/get)
