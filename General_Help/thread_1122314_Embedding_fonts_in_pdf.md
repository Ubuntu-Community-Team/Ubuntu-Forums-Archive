### Post by Simian Man on 2011-07-20
This is the command I use to turn a .ps into a .pdf with fonts embedded.  It is part of my LaTeX build system, so I'm not sure if it will work for other .ps files, but it works for mine:
```

ps2pdf13 -dPDFSETTINGS=/prepress file.ps

```
