### Post by WorMzy on 2011-11-01
The best solution, in my opinion would be to use a music manager. If you want a CLI-based solution, try MPD + MPC.

```
wormzy@sakura[pts/1]:~$ mpc list album | sort
12 Gauge
2econd Skin
A'arab Zaraq - Lucid Dreaming
Abrahadabra
A Change Of Seasons
...
Wolves From The Fog
Worlds Collide
You're Better Off Dead
```
