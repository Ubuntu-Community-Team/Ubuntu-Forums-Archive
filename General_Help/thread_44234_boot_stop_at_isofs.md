### Post by swampdweller on 2006-09-15
Hello AquaTek:

I recently tried Ubuntu 5.10 Live.
I found this posting of yours by googling
ubuntu isofs 92
...which is to say, mine froze at the
exact same spot.

After several repeated attempts, I noticed
the activity light on my external usb hard drive
coming on and off.  So, I thought to disconnect
that usb cable and try again.

Voila.  Up came Ubuntu 5.10 Live.

I had hoped to post this to you from within Ubuntu,
but alas it did not seem to see my wifi card,
and since the live version apparently will not
easily retain settings, it seemed too much work
to try to get the wifi recognized and the wep code in.

However, I did get what I wanted.
I got to see Ubuntu, up and running.
It seems nice, fairly intuitive.
I'm tempted to install it, if I can keep my Windows,
but my drive is pretty full and all that partition talk
sounds spooky.  I might simply toss in an old used
hard drive, with my existing one, and see about putting
Ubuntu on there for a spin.

Anyway, I don't know if your trouble relates to an
external usb drive, but for what it's worth,
that's what worked for me.
