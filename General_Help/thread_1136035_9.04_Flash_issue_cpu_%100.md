### Post by codeseer on 2009-04-24
Try uninstalling it, then do this:

```
sudo apt-get remove -y --purge flashplugin-nonfree gnash gnash-common mozilla-plugin-gnash swfdec-mozilla libflashsupport nspluginwrapper
sudo rm -f /usr/lib/mozilla/plugins/*flash*
sudo rm -f ~/.mozilla/plugins/*flash*
sudo rm -f /usr/lib/firefox/plugins/*flash*
sudo rm -f /usr/lib/firefox-addons/plugins/*flash*
sudo rm -rfd /usr/lib/nspluginwrapper
```

...and reinstall it (the official Flash off the adobe website).

### Post by codeseer on 2009-04-25
What processor? Did you do the upgrade or fresh install? You got all your updates from Synaptic, right?

Go ahead and give this first bullet point a go, the migration heuristic.

[https://wiki.ubuntu.com/JauntyJackalope/ReleaseNotes#Other%20known%20issues]("https://wiki.ubuntu.com/JauntyJackalope/ReleaseNotes#Other%20known%20issues")

You'll have a section that looks something like this:

```

Section "Device"
	Identifier     "Configured Video Device"
	Driver	"nvidia"
	Option	"NoLogo"	"True"
EndSection

```

You'll just need to add a new line under whatever is already there, unless the MigrationHeuristic is already there and then it'll look something like this:

```

Section "Device"
	Identifier     "Configured Video Device"
	Driver	"nvidia"
	Option	"NoLogo"	"True"
       Option  "MigrationHeuristic" "greedy"
EndSection

```

Then you'll need to hit the 'Save' button and restart the system.
