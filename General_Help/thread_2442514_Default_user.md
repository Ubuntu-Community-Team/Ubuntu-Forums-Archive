### Post by deadflowr on 2020-05-04
Is this auto login, where no interaction is required?
Or no password login, where you need to hit enter to login?

If the former then you need to disable autologin for the user account for that user.
If the latter then you only need to set your user at login.
Should stays as the default after that.

Some helpful information would be useful,
like
Ubuntu release.
Ubuntu desktop. (ie: Xubuntu, Ubuntu, Kubuntu, other)
