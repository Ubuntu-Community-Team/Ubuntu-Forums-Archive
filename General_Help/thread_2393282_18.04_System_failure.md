### Post by cruzer001 on 2018-06-01
This is Apport and could be an old report that was generated during your version upgrade.  So reset apport in terminal with:
```
sudo rm /var/crash/*
```
If it is not a false report then the error message will regenerate and the troubleshooting begins :)

[https://wiki.ubuntu.com/Apport](https://wiki.ubuntu.com/Apport)

You have Midnight_Commander installed?

Those files can be removed and will regenerate next time you use mc, your custom settings for mc will be removed by doing this.

```
rm -r ~/.config/mc ~/.local/share/mc
```
