### Post by jbrown96 on 2009-12-14
There is a project that is reverse engineering the new database used in 3.x. However, it's not really stable yet, probably 6 months away. 

I've had an iphone since they came out, and it's been really frustrating to use linux. I haven't really been able to sync it at all. 

The best method that I have found is the use Virtualbox (not the OSE version) to run XP. DO NOT TRY TO UPDATE THE IPHONE SOFTWARE IN VIRTUALBOX!!!!! When the iphone reboots, it gets funky usb ids, and Virtualbox can't pass it through to XP, so the update can't finish.

If you have questions about the Virtualbox method I can help more.
