### Post by zoqaeski on 2006-12-27
Greetings everyone
As I do a lot of typing using extended Latin characters, including IPA script, I need to find a way to quickly insert the characters, as copying from the character map is far too tedious. The Alt-Gr keys work fine, but there's a limit to those, and they aren't quite as obvious as I would have expected (but understandably so, as a's with tildes are more common than those with macrons). 

I used to be able to enter characters by holding CTRL+SHIFT+ hex code point on my old Ubuntu system, but since upgrading my hardware to a new computer and OS to 6.10, I've lost the ability. Surfing through configuration dialogues hasn't found much, and I'm doubtful it's hidden away in gconf (searching has revealed nothing so far). Can anyone help me restore this much-wanted feature?

cheers
Robbie

---

### Post by shirilover on 2006-12-27
You now need to use CTRL+SHIFT+U+hex code point.
