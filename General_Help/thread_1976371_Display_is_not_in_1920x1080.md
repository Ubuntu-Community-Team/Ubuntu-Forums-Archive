### Post by davidvandoren on 2012-05-08
Ubuntu 12.04 click in the left upper corner and type hardware.

Click on the icon that says Hardware drivers. 

If you are using an NVIDIA card you should see the driver for it. Select it and activate. Restart you system and this time type NVIDIA there you'll see all options.
