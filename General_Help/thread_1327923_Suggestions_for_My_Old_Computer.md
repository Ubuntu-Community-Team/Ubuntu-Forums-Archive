### Post by marshag63 on 2009-11-16
I have a Dell Inspiron 8000 (700 MHz) with 512 MB ram running both Crunchbang 9.4 and Linux Minx 6 Gnome.  If you swap out Firefox for lighter Swiftfox (which still uses the firefox plugins) you should have decent web access.  Crunchbang is more streamlined than Gnome, but both run acceptable - just don't try to do too many things at once and use lighter apps if you can.  It depends on what the main focus of use for the PC is.

You can do a regular install from an alternate CD (it will be a text based install and will go much faster than from a live CD).

Let us know how things work out.  Good luck :)

MarshaG.
