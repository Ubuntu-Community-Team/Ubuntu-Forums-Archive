### Post by Alaric on 2008-11-01
That's alright :).  Well, if you would still like to use the terminal you could run 'apt-get install brasero', but being a newbie, why do you prefer the command line?  Is that what people have been telling you to use?  Anyways, Ubuntu is way above just using the command line (unless you prefer it).  Synaptic is a graphical version of apt-get and it comes pre-installed on Ubuntu.  Just head up to your system menu (top-right) and go to System->Administration->Synaptic Package Manager.  This will give you a "point and click" interface to install just about every program Ubuntu has to offer :)  

Good luck.
