### Post by Sutekh on 2006-03-19
[QUOTE=fannymites]
If I right-click on a panel icon and select properties then click on the icon there to change it, when browsing the folders, I am only able to click on folders, all files and icons are greyed out and this is also the case with alacarte when trying to select custom icons.[/QUOTE]

When you go to select a custom icon, you are actualy browsing for the folder not the icons.  

Choose the folder you want and click OK.  

You should return to the original dialog box, but note that the path has changed.  

Pressing enter will update the path and allow you to choose the icon you want.



Check out this post by [audax321]("http://www.ubuntuforums.org/member.php?u=11439") for a fuller explanation with pictures.  [Ubuntu Forums - Access to Icons]("http://www.ubuntuforums.org/showpost.php?p=490790&postcount=4")
