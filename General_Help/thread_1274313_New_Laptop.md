### Post by NewlyHuman on 2009-09-24
There really isn't that much choice if you want Linux pre-installed. I'd opt for a Windows laptop from a vendor that's willing to reimburse you for the unused license.

Warranty issues are unlikely - A friend of mine did warranty work for a computer company and generally hardware issues never necessitated booting the system up - It was more economical to replace the part and ship it back out than it was to spend half an hour verifying the problem on each system.
