### Post by snowpine on 2013-02-16
A few of thoughts:

1. It's useless to discuss performance without mentioning hardware. What is your CPU, RAM, video card, hard drive, etc.?

2. If you run openshot from the Terminal, then you will get detailed error messages when it crashes. You can use these to troubleshoot the issue.

3. Any time your computer is running slow, you should use the system monitor or the terminal command 'top' to see what are the top processes eating up your CPU %. Give it a try and post back with the results!
