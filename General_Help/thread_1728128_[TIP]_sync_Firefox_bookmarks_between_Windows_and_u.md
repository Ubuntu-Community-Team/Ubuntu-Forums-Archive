Tip:

*Where you have Firefox in both windows and ubuntu it can be useful to synchronise the bookmarks across both browsers.*

Instead of using Firefox default bookmarks use zotero plugin.

Create an account at [http://www.zotero.org](http://www.zotero.org)

Download and install the zotero firefox plugin in both browsers (windows and ubuntu)

Use the zotero synchronise feature to sync between windows and ubuntu browsers.

### Post by lovinglinux on 2011-04-14
Firefox 4 has an encrypted built-in sync feature, that can sync bookmarks, passwords, settings, history and tabs. I works on any OS that has Firefox or the mobile Fennec.

If you want to use the feature on Firefox 3.6, then install [Firefox Sync]("https://addons.mozilla.org/en-US/firefox/addon/firefox-sync/").

To upgrade to to Firefox 4 see [Firefox 4 Mega Thread]("http://ubuntuforums.org/showthread.php?t=1712247").

### Post by Rebelli0us on 2011-04-14
You can use the same "shared" profile in Windows AND Linux so that there is no need to synch anything. Start Firefox with "-profilemanager" option in each OS and create a shared profile by specifying the same path. Works with Thunderbird email profiles too.
