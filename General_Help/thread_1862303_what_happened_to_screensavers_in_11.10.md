### Post by galacticaboy on 2011-10-16
> **bbernstone said:**
> they seemed to have either moved or disappeared in the new release.. is there somewhere i am just not looking? or do i have to install them seperately?

They have been removed, to install them you need to install the xscreensaver. That will bring it back.

```
sudo apt-get install xscreensaver
```
