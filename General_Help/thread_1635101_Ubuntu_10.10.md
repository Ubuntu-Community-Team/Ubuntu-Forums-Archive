### Post by sikander3786 on 2010-12-01
Welcome to Ubuntu and Welcome to the forums :popcorn:

> I am a complete newby, navigation is simple enough but e.g. i can't play some video's & music unless they're mp3's or mpegs.

Yes you need some codecs and the ubuntu-restricted-extras is the complete all in one solution for nearly all the codecs, open source version of java, flash and some fonts etc.

> Is it really a internet dependable OS to make it fully functioning as a complete OS?

Correct up to some extent. The positive here is that you get all the software from same one source that is the Ubuntu Repositories using Software Center or Synaptic or apt-get (you'll discover them later :-) )

> If so, i don't have internet at home, but i do have a usb edge modem. How do i use the usb edge modem to connect to the internet?

First of all you can use some other Windows or Linux computer with internet access to download the software and transfer it via USB to your home computer and install all the software ;-) And keryxproject is the best for doing that.

[http://keryxproject.org](http://keryxproject.org)

Regarding the USB modem, I don't know much about that. Did you try that with Ubuntu? It might just work out of the box. Or it may need some work.... Once that device is plugged in, we need the output of this command from Applications > Accessories > Terminal.

```
lsusb
```

> Isn't the 10.10 the steady version of the 10.04 LTS?

Yes 10.10 is the latest steady version but 10.04 is the Long Term Support version (LTS) and that will be supported for the next 3 years on the desktop while 10.10 will be supported for 18 months.

Feel free to ask any more :P
