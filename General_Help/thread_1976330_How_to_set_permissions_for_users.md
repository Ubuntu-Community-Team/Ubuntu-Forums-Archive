### Post by phibxr on 2012-05-08
You might wish to specify which permissions you would like to change, for whom and why, for us to be able to help you as well as possible.

The easiest way to change permissions is to open up your file manager (**Files**, in **Unity Launcher**), right-click the file or folder and select the **Permissions** tab.
