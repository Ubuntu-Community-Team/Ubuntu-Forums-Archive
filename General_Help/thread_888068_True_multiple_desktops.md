### Post by john.nicholls on 2008-08-13
> My question is: is it possible somehow to get true multiple desktops, each with a different background, different folders, etc?


Different backgrounds were covered in the previous answer. Generally, desktop icons will show up on each desktop, but you can put different windows on each of the four desktops, such as a terminal on one, a calendar on another, a file manager on the third, and an internet connection on the fourth.
