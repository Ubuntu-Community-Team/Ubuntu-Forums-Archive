### Post by mikegior on 2011-11-22
Well, from what I understand, "cloud computing" in Ubuntu is done through the Ubuntu One service which isn't a mandatory service; strictly optional. With Ubuntu One, you can upload share files between Ubuntu One computers/devices and use the deja dup utility to send backups of you home directory (or other specified directories) to Canonical's Ubuntu One service.

If you're thinking this cloud computing you're reading about in relation to Ubuntu 11.10 is comparable to, say, ChromeOS/Google Chrome Books, that is not the case. You have the right and ability to keep your applications and data strictly on your machine and not the cloud.

The Ubuntu One (cloud) service is free and completely optional.
