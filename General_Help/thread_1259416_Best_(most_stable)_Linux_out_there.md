Quick way to install debian with KDE if you can boot off USB:
1) download required files and put them on memory stick
2) install syslinux on the memory stick to make it bootable, also make sure it's a FAT16 partition
3) Boot memory stick
4) install to a single partition to avoid problems, make sure it's ext2/3/4 and not XFS to avoid problems with GRUB
5) Boot into new system
6) run as root, aptitude update && aptitude install kde-full kdm
7) as root, /etc/init.d/kdm start

What could be easier?

Well, the "Best" and most stable aren't always the same thing. Of what I know, the most stable linux distros are the linux parents. Debian, Arch, ect.
