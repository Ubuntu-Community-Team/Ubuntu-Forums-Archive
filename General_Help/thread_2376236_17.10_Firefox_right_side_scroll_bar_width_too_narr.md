### Post by vasa1 on 2017-11-01
Firefox is now a gtk3 application. So the following change *may* affect all your gtk3 applications.

Create a text file called *gtk.css* in *~/.config/gtk-3.0*.

Put the following code in there and save the file.```
*{
-GtkRange-slider-width: 12;
} 

```
Change the value to whatever suits you. Now, when you restart Firefox, you may have broader scrollbars assuming gtk3 hasn't changed how scrollbars are managed between 16.04 (where this works) and 17.10.

### Post by Hugo_Dias on 2017-11-18
GTK+ 3.20 has changed CSS in many ways. In order to increase scrollbar width on Firefox Quantum with Ubuntu 17.04, this worked for me:

Edit/Create ~/.config/gtk-3.0/gtk.css and add:

.scrollbar.vertical slider,
scrollbar.vertical slider {
    min-width: 15px;
}
