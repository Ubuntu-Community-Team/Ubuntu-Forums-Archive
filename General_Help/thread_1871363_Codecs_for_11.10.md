### Post by garvinrick4 on 2011-10-28
You need gsteamer's, libdvdcss2
Install medibuntu ppa here is easiest:
[http://www.ubuntuupdates.org/ppa/medibuntu_free?dist=oneiric](http://www.ubuntuupdates.org/ppa/medibuntu_free?dist=oneiric)
Now 
sudo apt-get install libdvdcss2
Then go to UBuntu software center as in screenshot. (click on)
And install the ones checked marked.
