### Post by starcannon on 2009-06-03
I have a person I help out that uses Comcast. Her set up is:

Comcast Modem>Linksys NAT Router>Computer

We let the Linksys Router deal with the Comcast Modem and by doing the OS in use is never even a consideration for the Comcast Router. If you already own a router, you should be able to allow the Router to do your ppoe or whatever protocol they use (sorry I forget as I don't have to fiddle with it ever, set it and forget it), The Router will have a place to put your PPOE Username and Password, I left the Linksys keep alive settings default and up to this very day (I was just over there doing a tutorial) it has had no issues with connectivity or complaints about the OS.

Anyway, not sure if that anecdote helps or not, if there are any questions you have about the setup I deal with lemme know, I'd be happy to answer any questions I feel qualified for.
