### Post by ian-weisser on 2014-08-07
Some Garmins are compatible with Ubuntu.
Few TomToms are compatible with Ubuntu.

Most models of both require their own proprietary interface software to load and change maps. Some Linux applications will work with some Garmin hardware; none that I know will work with any TomTom device.

Not Ubuntu's fault. Those companies don't release Linux software.
