### Post by MAFoElffen on 2023-10-15
What about Google Chrome? I don't know if Chromium has this, but I know the full-blown Chrome version does... 

Start Google Chrome. > Be signed into your Google account. > Navigate to docs.google.com. > Open a new document. > Open Tools > Voice Type. 

Click on the icon that it adds on the left of the page and start talking into your microphone...

I know if you go to that page with another browser, like FireFox, that "Voice Type" option is not there.
