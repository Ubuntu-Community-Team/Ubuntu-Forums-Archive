### Post by grahammechanical on 2011-07-17
Hi welcome to the forums both of you.

You each have a different issue (as far as I can tell) It is not good to try to help two people at the same time. So,

@getbuck 911

Go to System Settings and load the utility called Disk Utility. In the left hand panel do you seeing anything on the list under CD/DVD? On my system I see the make and model of my DVD drive and when I select it I get some more information. Not much information but it tells me that the operating system knows that there is a DVD drive in the machine. What do you see?

You could also open up the case and make sure that the DVD drive has a power connection and that the data cable from the motherboard is also connected both at the DVD drive end and at the motherboard end.

If you put a CD or DVD with information on it, such as music, do you hear the drive start to spin?

### Post by utnubuuser on 2011-07-17
Hello Piyush9198

Is there a good reason why you've installed Ubuntu 9.10? You'd be much better off loading Ubuntu 10.04 LTS (long term support).  
Ubuntu 9.10's was not a great release, (when compared to 10.04 or later) - and the release's life-cycle has expired.
If I remember correctly, 9.10 had a bug in Nautilus, which is the file-manager, and as such, responsible for media such as CDs, DVD's, harddrives, etc,etc.

Since 9.10 has expired, you won't be able to get the system updates - including bug fixes.

The easiest way for you to try to load 10.04 would be to symply do a release-upgrade through the update manager.

Here is the link to the Community Documentation pertaining to release uprades from 9.10 - 10.04: [http://help.ubuntu.com/community/LucidUpgrades]("https://help.ubuntu.com/community/LucidUpgrades")

Also, to play DVDs you have to install the appropriate codec. Here is a link to the Media how-to: [https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs]("https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs")

and here is another relevant link: [https://help.ubuntu.com/community/RestrictedFormats]("https://help.ubuntu.com/community/RestrictedFormats")

If I may be so bold, I'd really recommend installing 10.04 or even one of the more recent releases such as 11.04 - I've heard great things about the latest release.

Another idea, sacrilege though it may be, would be to install the latest Linux Mint, which is a great Ubuntu Derivative that comes with all the necessary media codecs pre-loaded.  Linux Mint  is becoming even more popular than Ubuntu - probably because it comes pre-configured to "just work" out of the box.
