### Post by mivo on 2007-10-20
When you first use 7.10, Tracker will index the files on your disk. This can take some time and it will cause your computer to run slower. You can wait this out or disable tracker in the Services menu (or kill trackerd). Others things to try: install restricted video card drivers, disable the desktop effects, install Compiz Settings Manager if you do want the new effects so that you can fine-tune them.
