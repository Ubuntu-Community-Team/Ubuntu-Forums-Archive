### Post by the yawner on 2007-09-13
The default file system is ext3. Referencing [Wikipedia]("http://en.wikipedia.org/wiki/Ext3"):

Max file size 	16GiB &#8211; 64TiB
Max number of files 	Variable[1]
Max filename size 	255 bytes
Max volume size 	2TiB &#8211; 32TiB
