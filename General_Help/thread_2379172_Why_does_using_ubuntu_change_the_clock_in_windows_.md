### Post by DuckHook on 2017-12-02
Though I don't often just answer with a link, your question is identical to the following current thread: [https://ubuntuforums.org/showthread.php?t=2379109](https://ubuntuforums.org/showthread.php?t=2379109)

The solution to your question is contained in that thread. Please read through it and decide which of the two options you would rather use.

Personally, I prefer to set the system clock to UTC. This makes more sense to me because setting every box I have to Universal time just seems more logical and better practice. That way, different computers in different time zones can all share the same basic setup, and be set to local time by a simple offset of the time zone. It also makes travelling from zone to zone more intuitive. But it's a personal preference and you can set yours either way.
