### Post by fast-Eddie on 2009-04-19
Hi All,

I figured out how to do it. 

When I start multiple instances of the Stocks screenlet, it does copy the first one however you can just right-click on the other instances and in the properties, you can change the 3 letter stock code to any other valid code. The stock details will be retrieved and the screenlet will be updated with the new details.
