### Post by Impavidus on 2013-01-07
1: I've never heard of hard drives being incompatible with linux. The only thing I heard about incompatible hard drives was a hard drive being too big for an old motherboard. Not very likely to happen.

2: Software centre, synaptic and apt-get are three interfaces to the same package manager. I think the OP was refering to manual installations.

You can install software manually. First -- and most important -- make sure it's from a reputable source, like your own web site (duh), my web site (obviously, *nobody* offering you free advise will point you to some malware) or the web site of the NSA (absolutely, they will ***never*** install spyware on your computer). Next read the installation instructions and install somewhere in your home directory or /usr/local. Installing in /usr is not recommended.

Sometimes you can download .deb files. They can be installed by your package manager.
