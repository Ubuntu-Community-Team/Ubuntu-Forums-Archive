### Post by grashdur on 2019-05-23
**What I want to do is add my signature to a PDF file, and type text into it wherever I want so that I can sign a contract. Also on my bank statements I want to add colored checkmarks or shapes while balancing my checkbook, to indicate which transactions I've already checked.**
This should be a really simple, basic thing to do. (I used to do this with a fax program that was antiquated when I was using it at work in 1995.) But I haven't found any way to do this now. Here is what I've looked into:
1. Okular offers PDF annotation, but I can only add standard shapes to the file, and the changes are saved only in the program rather than in the actual PDF itself. If I later open the file with another program, move it to another folder, or open it on another computer, the annotations are lost. Even if I try "print to PDF," all the annotations I've added disappear.
2. Dropbox offers document-signing features, but they are only for sending a document to someone else for signature, not all the other things I want to do.
3. Adobe Reader can do it, but not in the last version that Adobe made for Linux. So this might be a workable solution for Windows or Android. (I've installed this on my phone but haven't tried it yet. I want to do this on my computer.)
4. There is HelloSign, but it's web-based and you have to upload the document and the signature. I want to just do this on my computer.
5. In the Ubuntu Software Center, there is a program Xournal. I installed it today. You're supposed to be able to easily do all the things I want to do, but unfortunately the program's tools are unable to actually add anything to a page -- this is a beta program and it's apparently broken.
6. I can open a PDF in Gimp and export it back to PDF format, but only one page of it. And Gimp is pretty complicated to use.

I still can't believe it's so difficult to find a way to do this. I suppose any kind of drawing program should be able to work with at least a *screenshot* of a PDF...

### Post by drv9977 on 2019-05-23
You are talking about Digital Signatures I think, Before signing and document legally you must sign up with Digital Signatures Authorities located in your area (They will charge you with their little fees). After that you can use your Digital Signature anywhere you want, like in Cheques, Contracts, Mails etc. This is the legal way to do what you want to perform. Using the photoshoped form of signatures on documents is not legally accepted.
