### Post by nemarasu on 2007-04-11
Last time I used DNS, and this was 5 years ago or so, you needed to register your DNS server with Network Solutions. 

Your forward and reverse IP name must match (you need a static address for DNS server)

There used to be free DNS servers, but I am not aware of anyones that are still operating.

A good place (if it is still running) for DNS questions is Ask Mr. DNS (google it). You should be able to find DNS server how-to's on [http://tldp.org/](http://tldp.org/)

I really don't recommend running your own DNS server. You can get free dns (under a domain, meaning you would get a sub-domain name) from DynDNS

[http://www.dyndns.com/](http://www.dyndns.com/)
