### Post by bumanie on 2009-12-10
Maybe you should do a dual boot install or possibly a wubi install so that he can try ubuntu before committing to it as the only OS. As long as you are capable of setting up the few things that are not installed during the system install (ie multi-media codecs etc) there should be no reason why your father cannot use ubuntu as easily as any other OS he has used. Also, ensure that the laptop meets minimum hardware needs (which are fairly low) - these can be found [here]("http://www.ubuntu.com/products/whatisubuntu/desktopedition"). There are numerous tutorials re how to set a dual boot system - have a bit of a read and then decide which way to go.
