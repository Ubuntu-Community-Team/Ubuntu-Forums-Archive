### Post by Bucky Ball on 2015-01-27
*Thread moved to **General Help**.*

Welcome. ***Ubuntu & OS chat*** is not a support area. ;)

If JPK have done their own spin, it's anyone's guess how they've tweaked and configured it, but generally, if you get to the login screen and choose Unity from the Sessions menu, that would become your default desktop environment and you'd boot to Unity anytime you restarted the machine. It would stay that way until you changed the default session from the Sessions menu to another desktop environment. If this is not happening, perhaps you could try enquiring with JPK why not, because it should be.

Is there a reason they want you to use xfce for the microscope app? Perhaps it is intended to be used only in xfce because of some feature they've written in specifically for that DE. Does it work fine in Unity? 

Good luck.
