### Post by BQAggie2006 on 2009-10-07
Yes, VirtualBox is a Sun product. I recommend downloading it and installing it from the Sun web page because the version in the Ubuntu Repositories is the free edition, which doesn't have some of the features as the non-free edition from the web page (mainly remote desktop and USB support).

All you have to do is go to [http://www.virtualbox.org](http://www.virtualbox.org), click on Downloads, then VirtualBox for Linux Hosts, and download the .deb installer file for the appropriate version of Ubuntu you are running. Once it's finished downloading, installing it is as simple as double-clicking on the .deb you downloaded.

Once you have VirtualBox set up, you can create a new Virtual Machine, and install Windows XP inside it. If you need more help, VirtualBox has a very thorough user manual on their Downloads page, and you can always post more questions here on the forum.
