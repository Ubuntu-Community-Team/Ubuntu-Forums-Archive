### Post by dbott67 on 2006-10-22
1. You may not have burned the .ISO correctly... check my signature on how to properly [burn an ISO]("http://www.ubuntuforums.org/showthread.php?p=465704#post465704").  If you browse the CD from Windows (I'm assuming you downloaded it on a Windows computer) you should see a whole bunch of files & directories... not a file called xxx-whatever-xxx.ISO.  The most common problem people have is buring an ISO as a data CD instead of burning an ISO image.

2. If you did burn the ISO as an image, you may have burned it too quickly... try slowing down the burn speed to 8x or less.

3. Your ISO may be corrupt... check the [md5 hash]("http://userpages.umbc.edu/~mabzug1/cs/md5/md5.html").  Download checker from [here ]("http://www.irnis.net/files/acsvi.exe")(for Windows) - [http://www.irnis.net/](http://www.irnis.net/)

Load the CD on a running system (Window, Linux, whatever) and click it open.
If you see a directory of files, you did a proper burn of the iso.  However if you see a single file ending in .iso, you simply copied the iso image to the CD.  In that case check with your software help for instructions on burning an iso image.
