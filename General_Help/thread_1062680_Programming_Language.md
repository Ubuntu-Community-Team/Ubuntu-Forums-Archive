Java is the de facto standard for cross-platform.  Many web apps. are Java based.

I learned it in about 3 months, somewhat proficient.  It's a good language.  The concept of library calls seems to be the most difficult, where the (.) indicates the subclass.  IE,
  system.out.time(current_time);

will call the function to get the system time.
The main library is system, then the out library is searched for the time(function).

It can get tricky, and is a very eloquent language, but it is fun.

There are many, many freely available tutors, books, and several good shelf items.

Search for 'java code' and see what you think.
