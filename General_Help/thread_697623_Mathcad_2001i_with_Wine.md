### Post by MoLE on 2008-02-24
I'd suggest having a look over at appdb.winehq.org.

There are several entries for mathcad 2001, and you may find this more helpful.  The other suggestion I would have is upgrade to the latest wine from the winehq.org site - there is a simple method of adding the repository so you always have the latest wine release available.

If it requires windows scripting host, this may need to be installed prior to installing mathcad.

If you install the latest wine-doors (at [www.wine-doors.org](www.wine-doors.org)), this program provides a simple interface to install various windows software and system packages, including windows scripting host 5.

HTH.
