Safest thing to do is make a copy of the root.disk and keep it somewhere safe like an external drive or USB stick just in case.

### Post by linuxinstalledfromhdd on 2011-06-11
The first thing that happens when you boot your computer is the BIOS screen.  There should be a key you can press to select which device you would like to boot from.  Then you will get to a menu that says, floppy/cd/hdd/network/USB..    Which one you want to boot from... etc etc.. Make sure you have Ubuntu installation disc in the drive, and it should boot from that drive as long as you select it in BIOS.
