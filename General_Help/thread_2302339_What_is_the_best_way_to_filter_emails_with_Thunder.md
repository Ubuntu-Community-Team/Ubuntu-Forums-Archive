### Post by TheFu on 2015-11-10
I do all email filtering on the server side, never on the client. That way all clients see the same views. Plus searching through all folders is performed on the server and not on some tiny smartphone.

### Post by TheFu on 2015-11-10
It is my email server. Yes, every client sees the changes when updated. 

For a client to see imap folders, the client must "subscribe" to the folder. It isn't automatic. This is an imap thing and honestly one of the reasons imap is so fantastic when we have multiple clients.

I stopped using pop3 around 1998, after I brought up my first personal email server.  I'd been running email for a few companies for a few years before that.
