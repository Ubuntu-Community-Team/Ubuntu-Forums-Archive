### Post by kanikilu on 2009-04-06
You can fill out PDF forms with Adobe Acrobat Reader. Supposedly version 8 does not allow you to save filled-out forms, only print them. So you might want to try the latest (9.1) from Adobe: [http://ardownload.adobe.com/pub/adobe/reader/unix/9.x/9.1/enu/AdbeRdr9.1.0-1_i386linux_enu.deb](http://ardownload.adobe.com/pub/adobe/reader/unix/9.x/9.1/enu/AdbeRdr9.1.0-1_i386linux_enu.deb) (this is a direct link to the .deb for x86).

### Post by spcwingo on 2009-04-06
You might want to check out flpsed or pdfedit.  I prefer flpsed to just fill out forms, but if I want to do some hard core editing I fire up pdfedit.  They're both available in the repos.

### Post by HermanAB on 2009-04-06
Use Adobe Acrobat.  It will likely not allow you to save the resultant file, so simply print to a file, which results in a postscript file, which you can convert back to a PDF file with ps2pdf.
