### Post by swisscow on 2007-06-26
Now that can depend on a few things - it just doesn't seem to be as simple with ATI cards as others.

I'll illustrate with my case - I have a laptop with an ATI mobility X300 card

If I want to use GoogleEarth I use the fglrx drivers (search in synaptic, followed by an install and a quick edit of the xorg)

If I want to use Beryl/Compiz then the drivers with Ubuntu (radeon) and the AIGLX setting works well, but no GoogleEarth

If I want to use Beryl/Compiz with XGL then the fglrx drivers again - but GoogleEarth is unusable.

For other ATI cards - I don't know, but I hope this has been a little use to you both.
