### Post by srunni on 2006-10-21
Hi,

Does anyone know how people take all those screenshots of XGL/Compiz when they're in "cube mode"? I know that I can't take any screenshots while in this mode. Is there another program which can take screenshots while Compiz is in "cube mode"?

Thanks in advance for the help!!!

### Post by ayoli on 2006-10-21
use scrot
```
sudo aptitude install scrot
```
then run:
```
scrot -d 12 -c 
```
then you got 12 sec to move the cube with ctrl+alt+left mouse button

### Post by _simon_ on 2006-10-21
You should be able to do the following:

Press CTRL ALT and use the mouse to put the cube in whatever position you want it.

Then let go of CTRL ONLY and whilst still holding ALT press print screen.

### Post by 3rdalbum on 2006-10-21
Or you can use The Gimp. Go to File > Aquire > Screenshot. You can set a delay of however many seconds to give yourself time to get the cube in the right position. Hold it there until the PC's speaker beeps.
