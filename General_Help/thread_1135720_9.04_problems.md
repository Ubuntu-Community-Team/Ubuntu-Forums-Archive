I say this with love, for what it's worth - I'd expect in using relatively new file systems like ext4 to result in problems like what you're experiencing with df. I imagine it's just a matter of time before there'll be a df update that'll fix that. All that said, in order for a file system like ext4 to really get out there, people have to use it, so I guess good on ya.

As for your other issue, I couldn't tell ya. Sounds like that could be a framebuffer issue.. Maybe check to see if any framebuffer modules are being allowed to run and/or try temporarily removing any "vga=xxx" parameters from your kernel line in grub.
