[QUOTE=eyebrowman92]alright it turns out it was gnome art i was looking for.[/QUOTE]
Cool; so next time you're looking to find what package a certain program is in, you can do a search in Synaptic with descriptions included. Hit the search button, and in the "look in" combo box select "description and name"; this will search for both package names and descriptions so you'll have a higher chance of finding a program whose name isn't the same as the package name.
