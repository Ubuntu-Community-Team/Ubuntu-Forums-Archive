### Post by NikTh on 2012-11-18
[Rar program is available in multiverse repositories](http://packages.ubuntu.com/search?keywords=rar&searchon=names&suite=precise&section=all). Are you sure that you have multiverse enabled ? Check from Software Center => Edit => Software Sources. 

Also (if I'm not wrong) rar included in ubuntu-restricted-extras. So alternatively you can search in Software Center for Ubuntu-Restricted-Extras.
