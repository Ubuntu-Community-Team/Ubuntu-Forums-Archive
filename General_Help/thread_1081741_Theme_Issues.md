### Post by T.J. on 2009-02-27
You can reset your theme under System->Preferences->Appearance.  You can install themes and icons by dragging and dropping them to the Appearance window. =)

You should read the instructions with the theme, if any?

Many themes require the gtk2-engines that you can install using Synaptic.  The engines help change they way the buttons, etc look.  Using themes can be a little annoying at first, but if you install the gtk2-engines many of your problems should go away. Theme files provide graphics and color.  Themes normally do not provide background images.  

If you have given up, and just wish to delete the theme files you aren't using you can delete the .themes folder in your home directory.  You can also delete the .icons folder there if you wish.

Please post here if you need more help.
