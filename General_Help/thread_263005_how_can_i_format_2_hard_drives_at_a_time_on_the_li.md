### Post by Dinerty on 2006-09-22
I'm not sure you can format 2 hdd at the same times, once you have Ubuntu installed you can go to 

System > Administration > Synaptic Package Manager and search for "gparted", this program will allow you to format your drives.

If however you run into difficulties, you can install the gparted LiveCD. Once burnt you can reboot with the CD/DVD in your tray and perform your format this way.

[http://gparted.sourceforge.net/livecd.php](http://gparted.sourceforge.net/livecd.php)
