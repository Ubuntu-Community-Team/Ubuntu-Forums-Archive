### Post by Joris Donders on 2011-07-19
This is a strange question...., 

It is best that you install (online) the "restricted extra's" for Xubuntu from the Softwarecenter. This package contains the Openjava JDK . 

Once installed you can work with FF4 offline with that plugin working. 

There is an option for installing SunJava JRE but than you have an EULA, that can put you in an illegal position when using professionally. And it is poorly supported with updates from the repo's. 

I hope this will help you out.
