### Post by terrrorr on 2010-04-07
Hi,

Could you please give more information about your messaging service. Are you using some specific messaging service, What is the software you are using in Ubuntu and in Windows.

It is quite hard to believe that there is some sort of conflict between your router and Ubuntu since TCP/IP Protocol has been used several decades. Only reason when you could encounter these kind of problems is if you changed TCP/IP protocols MTU size.

Please notice: If your virtual machine works (or IM software in your virtual machine) under Ubuntu, then there is no problems in your network connection as your virtual machine uses same TCP/IP stack than your host machine (Ubuntu).

I hope this gives you some sort of idea what is happening
