### Post by a-you on 2016-04-26
I just installed ubuntu studio xenial 16.4. I had been using trusty and with that I could have 2 users logged in and switch between their sessions without having to enter any passwords by typing CTRL-ALT-F8 and CTRL-ALT-F7 to go back to the first user's session. Now it leads to a "this session is locked" message which takes "a few seconds" to go not to the other user's session but to where I have to enter a password.   How can I get back the previous behavior?? I want to be able to switch between user sessions without having to enter passwords!   This is my own laptop; I sometimes use a separate user for specific tasks and being able to switch between the sessions seamlessly was VERY USEFUL. Now that's broken. Hopefully there's a way to re-enable that feature.  Thanks in advance for any insight.

### Post by a-you on 2016-04-28
Figured it out. In case it helps anybody else: under Settings Manager->Session and Startup->Application Autostart tab turn off Screen Locker (the package is called light locker I do believe).
