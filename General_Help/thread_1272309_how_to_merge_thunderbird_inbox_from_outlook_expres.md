### Post by bob-linux-user on 2009-12-23
If you dual boot with Windows you can use store the Thunderbird Profile on a partition that both operating sytems have access to. If you then use the Windows Thunderbird version (instead of Outlook Express)  and make sure both the Linux and Windows versions are sharing the same profile, you will then always have your latest emails whether you boot into Ubuntu or Windows.
