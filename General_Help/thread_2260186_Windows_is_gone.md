### Post by Mark Phelps on 2015-01-09
> replace it with Ubuntu.BAD idea -- especially to mess around with a machine that has a critical use so soon.

Unfortunately, the Ubuntu installer "replace" option automatically erases THE ENTIRE DRIVE.  It tells you that in the process, but not very clearly.

You might be able to use Testdisk to get the former Windows filesystem partitions back.
