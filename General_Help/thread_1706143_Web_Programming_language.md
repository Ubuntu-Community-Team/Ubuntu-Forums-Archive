### Post by Guenhwyvar on 2011-03-13
PHP and JavaScript (and My SQL if you want a database). Try apachefriends (LAMP) for developing php. Many books teach all 3 of these. 

Computer Science courses teach all of these as standard so they are good to know.

PHP is server-side and JavaScript is client-side. 

Also you can make java applets if you wanted full programs in browser (but I recommend javascript and php first if your focus is web development).
