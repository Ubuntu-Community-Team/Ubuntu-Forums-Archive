### Post by DuckHook on 2020-05-31
Starting with Focal, Canonical has decided to deprecate 32-bit except for bare essential 32-bit libs needed to run WINE and Steam, and even there, only because their original announcement to drop 32-bit altogether caused an unholy row so loud it threatened to bring down the roof over their head.

I wouldn't be surprised if 32-bit Java has fallen victim, though I wouldn't really know because I don't run Java.

It may be possible for you to extend 32-bit into Focal (at least for 3 more years) by using a Bionic LXD container. I haven't tried LXD on an RPi yet, so I caution you that such an approach must be considered highly experimental. You have been warned. Instructions for LXD in my sig: "Sandboxing Apps with LXD".

As an aside, did you mistype or is there now a brand new 8GB? I thought the RPi 4B+ was the latest model and maxed out at 4GB.
