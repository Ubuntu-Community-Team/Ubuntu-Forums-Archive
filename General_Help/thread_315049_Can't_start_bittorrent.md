### Post by richardhp on 2006-12-08
bittorrent doesn't work like the windows version, ie open it up and see lots of nice boxes telling you the download status etc, all it does is open a torrent file and download it for you simple as that.

all you have to do is find a torrent file, download it onto your desktop and then open it by double-clicking. It will ask you where you want it saved and bittorrent will do the rest for you. 

if you want to stop downloading and switch off your computer, simply stop the download and turn off, when you start up again just re-open the torrent and click save in same location or something and it will continue as before.

### Post by mynoks on 2007-11-05
I suppose that you would have forgotten or solved this problem already but if anyone have the same problem an easy way to solve it is (assuming you used firefox and haven't deleted cache files, and have an standard installation)

Just do :
1. Firefox -> Tools -> Downloads (the download window will appear)

2. Right-Click on properties on the desired file
3. Copy the value of the "FROM" input box.
4. Paste the URL in the browser
5. and the torrent will start again from the point you reached.
