### Post by svenole on 2011-10-19
I've been using Simple Backup and Restore for years without any problem what so ever. The main reason for using this tool is that it is easy to use and it gives a working GUI for single file / directory restore. 

After upgrading to 11.10, I'm not able to use SBackup for restore anymore. It starts, finds the backup data and gives the dates for the backups in bold in the calendar. But selecting the backup, do not bold up the snapshot management and restore management choices and it is not possible to use them. 

I'm really interested in getting this to work and wonder if someone else has seen the same issue?

---

### Post by svenole on 2011-10-23
Eventually I found a solution to this problem at last. The restore window did display the dates of the stored backups, but did not display the windows where each backup is identified and where you actually have to select the backup you want to restore from. Making the window bigger, makes this window appear and now I can select the actual backup and restore does work again.
