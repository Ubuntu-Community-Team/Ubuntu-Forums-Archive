### Post by hikaricore on 2007-10-25
Web: This is easy, you have Mozilla Firefox by default.  You can also use any other browser that is cross-platform such as Opera.  ^_^

Storing pics and videos:  Most digital cameras act like flash drives.  When you plug them in they "should" auto mount, and you will be presented with an option to open the device for file browsing or import them to the photo manager software.

CD Burning: My overall suggestion is K3B.  There are cd recording functions built-in to Gnome, but I've found them to be a bit obtuse for new users.  K3B will feel alot like the Nero style software most ***dows users have become accustomed to.

Basic photo editing: The GIMP is installed by default, as well as the FSpot photo manager (which I believe can also do editing though I don't use it so that's just my guess).  At the very least you should be able to do almost anything you'd like in GIMP.

Video: I'm not a user of such equipment but I've heard Kino is good, it's free and in the software installer.

My suggestion, is burn a Live CD: [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

You'll be able to boot up into a fully functional Ubuntu Linux desktop to test it all out without installing a single thing.  ^_^

### Post by ed-koala on 2007-10-28
Since you have the disk, just go into your BIOS setup - for me it is hitting the delete key when the computer starts, yours is probably different - and change the boot order to boot from CD first.

Then have your Ubuntu disk in the drive.  It will start Ubuntu without installing it yet, that way you can check it out first.  If you like it, there will be an install option on your desktop you can use to put it on your HD.
