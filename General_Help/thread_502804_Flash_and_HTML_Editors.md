### Post by mcduck on 2007-07-17
There really is only one decent Flash editor, Adobe Flash. And that's only for Win/Mac. But if you get it installed on any window machine you can just copy the programs's directories and registry keys to Wine and use that to run Flash on Ubuntu.

For HTML editors, there's more than one man can ever try.. I've heard a lot good about NVU so you could give it a try. Or just search for 'html editor' in Synaptic and try all that look interesting. (I rprefer to code my pages by hand so I can't really suggest any particular wysiwyg editor).

### Post by southernman on 2007-07-17
From my reading on the forums, Bluefish is the latest craze for Ubuntu (WYSIWYG editor). It's in the repos, just do:
> 
sudo aptitude update
sudo aptitude install bluefish

I was doing some research on them myself earlier and read that Nvu is not in the repos due to inactive development. Development of it has been taken over by another project but I don't recall where exactly reading this stuff.

Bluefish does look interesting, as well as full featured. Your mileage may vary... ! :)
