### Post by H.E. Pennypacker on 2006-06-14
[QUOTE=skyrender]Curious, what's the difference between ubuntu, kubuntu and xubuntu (in laymans terms)? which one is best for the average college kid?[/QUOTE]

Quote from Xubuntu.org:

> Xubuntu is a complete GNU/Linux based system with an Ubuntu base. **It's lighter, and more efficient than Ubuntu with GNOME or KDE**, since it uses the Xfce Desktop environment, which makes it **ideal for old or low-end machines, as well as thin-client networks**.

Quote from Kubuntu.org:

> **Kubuntu is a user friendly operating system based on KDE**, the K Desktop Environment. With a predictable 6 month release cycle and part of the Ubuntu project, Kubuntu is the GNU/Linux distribution for everyone.

What is Ubuntu?  Kubuntu without KDE.  Instead, it uses Gnome.

What is good for you?  Likely Ubuntu.
