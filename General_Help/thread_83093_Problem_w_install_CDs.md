### Post by Spunkyboy2812 on 2005-10-27
Hey i just put Ubuntu on my computer.  At first i had the same problem as u.  I dl and burned the boot disk but my comp would never boot from cd.  my BIOS was set correctly and everything.  What turned out to be the problem was in the actual way that I burned my cd.  I used B's recorder Gold that came with my sony cd writer.  It doesn't have the capabilities of writing a useful boot disk.  What u need to do is DL the trial of Nero and actually select creat a data (boot cd) from the wizard.  This will burn the cd correctly and u should have no problem running it.  There are also a couple other programs of which i do not kno the name that can burn boot cd's.  Try googleing it and u should be fine.
