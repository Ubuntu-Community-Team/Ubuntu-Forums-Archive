### Post by aysiu on 2008-06-11
Why not get some real security instead of just a false sense of it?

Forget anti-virus. It's useless. Really.

If you want real security, don't start any listening services (FTP server, OpenSSH, Apache, etc.), use strong passwords, use NoScript on Firefox (and whitelist only sites you trust), and install software only from the Ubuntu repositories.

### Post by Dynamo_Joe on 2008-06-12
I'll tend to agree with Wolfen69 with regard to the onslaught of "you don't need to install anti-virus on linux ...". 

However, if you are on a network with Windows machine and/or share files/communicate with Windows user, it would be a good sign of social courtesy if you scanned any files/attachments before passing them on. 

ClamAV is normally sufficient if you're only passing on emails ... however for the really paranoid, a full blown anti-virus suite such as Avast (yes, they do have a linux version but requires registration).
