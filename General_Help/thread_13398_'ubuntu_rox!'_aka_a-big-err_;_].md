### Post by alastair on 2005-01-31
Great bug report ....

If you have problems booting, the first thing to do is try various boot options. These are detailed in the "help" menu's before boot (F3,F4 etc.). Take 5 mins to read these, write down those you might thing relevant (e.g. acpi=off, noapic etc.) and try booting with different args. Perhaps you could try booting without X for instance.
