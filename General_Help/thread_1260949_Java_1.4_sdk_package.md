### Post by nikhilk on 2009-09-08
> **Zeemvel said:**
> Hi, I need the java 1.4 SDK, but can't find a package of it. How can I install this on Ubuntu 9.04?

Since JRE 1.4 has reached its End-Of-Life, it is no longer supported by Sun and its use is not recommended. Hence those packages are not present in Ubuntu repositories.

You can download it from Sun's [software archive]("http://java.sun.com/products/archive/j2se/1.4.2_19/index.html") though.
