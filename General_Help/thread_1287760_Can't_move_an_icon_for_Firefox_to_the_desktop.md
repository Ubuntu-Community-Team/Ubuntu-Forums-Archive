### Post by SonicMonkey on 2009-10-10
I had the exact same problem, but I solved it by removing Firefox from my favourites and then using the search filter in the start menu, I click-and-dragged the Firefox icon to the desktop.

I'm not sure if you can have it both on the desktop and your favourites.
**[EDIT]** Just tried it, you have to add Firefox to your favourites after getting it on the desktop.
