I'm running Maverick 10.10 and have "Main", "Universe" and "Multiverse" sources enabled. My version of Evolution is 2.30.3, yet at the the Evolution website, the current version is 2.32.1.

Why am I not getting the latest version of Evolution?  I check for updates daily and even if the latest release is delayed, I should have received some sort of update from my current version of 2.30.3 to some version between this and 2.32.1

### Post by howefield on 2011-01-10
You have the latest version available for Ubuntu 10.10. To get a more recent version, you'd need to download the source code and compile it yourself.

You are not likely to see the latest version of Evolution in Ubuntu 10.10, the only updates that you will get for it will be security ones.

Application versions tend to be set at release time, and stay the same for the length of time the release is supported. Except for security fixes.

There is the "backports" repository where newer versions of some applications can sometimes be available, but I don't think that includes Evolution.
