### Post by gonber on 2009-10-04
ok, here is the answer:

1) run sudo lazarus-ide
2) Tools > Configure "Build Lazarus"
3) Advanced Build Options > LCL interface > gtk 2
4) build
5) exit lazarus
6) run it!!
