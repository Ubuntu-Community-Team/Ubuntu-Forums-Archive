### Post by LenzM on 2008-11-02
System -> Preferences -> Keyboard
Layouts
Other Options...
Alt/Win key behavior
Check 'Super is mapped to the Win-keys'

---

### Post by dorkdork777 on 2008-11-03
Just checked that, it isn't it, but I did remember on the way that I'd changed my layout switching rules - a short check showed that I had the left Super key mapped to change keyboard layout! :oops:

After unchecking that box, Super works perfectly. I guess this could be useful to someone else down the line :P
