### Post by asp314 on 2010-12-05
The default shell in Ubuntu is bash (Bourne Again SHell).  There are also many other shells available for *nix systems, such as tcsh, and zsh.  Though they all perform the same function, each has subtle differences, usually in syntax.  Bash is the most common, however, and worth looking into for a shell beginner.

There are many bash scripting guides available online; Googling "bash" will yield some decent links.  Ubuntu also provides a beginners guide for bash and is as good a place as any to start:[https://help.ubuntu.com/community/Beginners/BashScripting](https://help.ubuntu.com/community/Beginners/BashScripting)

There are probably a few commands that will help you transition into the shell environment:

man - "man" (manual) is *THE* place to get information about commands in the shell. Typing "man *command*" will open up what is called a manpage, which contains the information on how to use the command.  Probably the first thing you should do in the shell is type "man man" to figure out how to use the manual.  You may exit a manpage by pressing "q" at any time.

ls - Prints a list of files in a directory ("folder").

cd - Change directory.

mkdir - Make directory (create a folder).

cp - Copy a file

Reading the manpages for these commands will help you understand how they work. However, if you need more general info than the manpages can provide, never forget "Google is your Friend!"  Good Luck!
