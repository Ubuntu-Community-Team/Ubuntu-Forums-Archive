### Post by Petro Dawg on 2013-08-10
Yup, it's possible.  If you are using Ubuntu12.04 you can goto System Settings -> Keyboard -> Shortcuts

Click on Custom Shortcuts and then click on the "+" 

Then enter the following...

Name:  GIMP
Command: gimp

and click on APPLY

then click on "Disabled" and press the key combination you would like to launch GIMP.  (I used Ctrl+Alt+G)

I'm sure there is a similar process for other Ubuntu versions.

### Post by deadflowr on 2013-08-10
If you're running plain Ubuntu, with the left side launcher, the first ten apps can be launched with the keyboard shortcut 
super(windows key) + number key.
Super key alone launches(opens) the dash.
Super key + 1 will open the first app below that.
And so on, down to the tenth, which opens with 0(zero).
