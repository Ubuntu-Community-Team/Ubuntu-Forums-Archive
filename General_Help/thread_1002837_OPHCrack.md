### Post by AlanRogers on 2008-12-05
I really don't know if I should be posting this because of what it is and how it can be abused but here goes anyway.

I'd set up a Windows machine for a customer some weeks ago, which they subsequently hadn't used because I hadn't told them that it was ready. In going back to it, the password wasn't what I thought I'd set it to, nor any easily guessed misspelling thereof.

Long story short, I downloaded and burned [OPHCrack]("http://ophcrack.sourceforge.net"), booted the customer's machine from it and discovered that I'd mistyped the password originally. This enabled me to get into the machine and reset the password, without having to do anything more drastic.

For me, a very very very useful tool - it saved me potentially hours. However, also downright bloody scary that such a tool can, does and should exist. In the wrong hands ...

Welcome to the real world of the security professional.
 
     It is a constant nagging fear, and the reason that every security professional I know says "The only Secure System is the one that's off and can't be turned on"  or some version alluding to the same meaning. 

    If you look to the area of helping your customer you need lots of tools to deal with the various problems that you are faced with.  All of these tools are useful to a professional in doing the Job, but these same tools in the hands of a non professional is scary beyond belief.

     A few months back there was a big stink raised because someone was helping the police forensics community by putting a bunch of useful tools onto a USB drive.  Well the press got hold of this and went nuts as usual.  What was so funning is that every one of the tools on the USB were and still are available to ANYBODY with an INTERNET connection for free, just download them.

BOTTOM LINE: What man can invent he can circumvent.
