### Post by mixim on 2010-05-02
> **oldtraveler said:**
> Is there an email client that will import ALL of my hotmail folders?  I've tried Thunderbird which works well except that it won't import some additional folders that I use in hotmail.

Hotmail does not support IMAP services, only POP. So there is no possibility to import these folders and establish any kind of synchronization. GMail does offer IMAP for it's users, you would have to swap to GMail or any other webmail supporting IMAP if you would like to be able to connect and synchronize folders or tags no matter which desktop email client you are using on Linux.
