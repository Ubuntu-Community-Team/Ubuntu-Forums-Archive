> **PcFrk256 said:**
> I was trying to enable Remote Desktop, but when I go to System-->Preferences it's not there...How do I install it or do whatever I need to do to enable it?

Odd -- it is there in all of my installs.

The application used is actually called vino. You can use Synaptic to check if it is installed by looking for vino. If not simply select it for install.

If it is then the actual command used to set the preferences is:

vino-preferences

You can type that from a command line or add a shortcut to your menu yourself.
