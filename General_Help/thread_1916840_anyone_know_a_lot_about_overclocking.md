### Post by Stonecold1995 on 2012-01-28
Turbo Boost is not overclocking.  Turbo Boost just allows a CPU to go slightly higher than it normally does for intense tasks.  Overclocking would be setting the DEFAULT speed to that above its normal speed.  For example, my computer's CPU runs at 2.0 GHz (it's an Intel Core i7-2630QM 8-core, 4-physical, 64-bit), but when I run an intensive task (I run Folding@home), it can go up to 2.9 GHz (but mostly stays around 2.4 GHz).  Overclocking would be me setting it to, say, 3.5 GHz.  You CAN adjust the clock so that it is always on Turbo Boost by changing the power settings to "performance" (as oppose to "power saving", where the preferred clock rate is around 800 MHz).

The turbo-boost is a new development on intel i5 and i7 processors. It boosts performance when you need it most (high processing needs). You can read more about it [here]("http://www.intel.com/content/www/us/en/architecture-and-technology/turbo-boost/turbo-boost-technology.html")

### Post by Tanker Bob on 2012-01-28
Turbo Boost is a good compromise for laptops. Overclocking generates significantly more heat than most laptops could dissipate, which would probably lock up the CPU for overtemps. I'm running a 2.4 GHz CPU at 3.3 GHz in a desktop, but I put a honkin' heat sink/fan on it. In a laptop, you only get the cooling that's designed into it.
