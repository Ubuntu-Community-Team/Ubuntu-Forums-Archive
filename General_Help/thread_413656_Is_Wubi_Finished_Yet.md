### Post by ago on 2007-04-19
> **nphoops said:**
> The "normal" Feisty is out, but has wubi reached it's completed stage? Or should I still wait a couple of days before downloading? :confused:

As mentioned elsewhere, the official wubi release is still the beta one. We still have some pending changes. You can expect some minefields soon that will work with Feisty-final ISO, in a few days, if there are no complaints, we will have our own "final". 

To be more precise, the backend should be finalized soon (early minefields), which means that most of the pending changes will affect the front-end, which means that if you manage to get a successful installation of Feisty final with the early minefields you will not gain much from using the later ones.

We welcome as much testing as possible for the coming minefields.
