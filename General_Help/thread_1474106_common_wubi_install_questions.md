### Post by bdig85 on 2010-05-05
When you install Wubi, for all intents and purposes you have just installed a full installation of Ubuntu. It acts almost exactly like a native installation acts. When you start the computer, you'll see an option for Ubuntu ( or whichever distro you chose) in the bootloader alongside whatever version(s) of Windows you have on your hard drive. If you want to do a real installation of Ubuntu onto your hard drive, with all the partitioning fun, you could always make an installation disc or USB from within Ubuntu using the Startup Disc Creator.
