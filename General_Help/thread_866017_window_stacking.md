It allows you to make your desktop look more interesting than it normally does.  You can set desktop effects to make your windows wobbly, or display your workspaces on a cube.  Compiz can do a lot of stuff, some is useful, some is not.  In this case scale might be useful.

### Post by Potatoj316 on 2008-07-21
To install compiz config settings manager open a terminal and type ```
sudo aptitude install compizconfig-settings-manager
```

Then in your System -> Prefrences menu there should be an entry (near the top) that says Compiz settings manager.  That will allow you to change which plugins are activated, which arent and how you control them.
