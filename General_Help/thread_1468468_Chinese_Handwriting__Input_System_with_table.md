### Post by jyaan on 2010-05-12
i found a possible solution, although it isn't very pretty. set up your tablet to be used as a normal tablet, then install cell writer. after that, start up cell writer and go to setup, and add "CJK Unified Ideographs" to the languages list. Then you'll have to write every character you want to use in order to train it.... This would probably be extremely time consuming, so obviously the program needs some work. But at least it's a step in the right direction.

Maybe there's something better out there that anyone knows of?
