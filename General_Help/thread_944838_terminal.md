Run as root, put sudo before any command.

sudo apt-get update

More tecnical way, if you are using Terminal for a length of time, and want to stay root, and use the root shell,

sudo su
Enter password:

Then just type commands away. Bearing in mind its dangerous to run anything and everything by root, always be sure of what you are doing.:)
