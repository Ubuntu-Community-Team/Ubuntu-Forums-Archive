### Post by mb_webguy on 2009-02-06
That's a normal warning whenever you try to install an xpi from an unverified source.  In Firefox, the list of trusted sites can be accessed in the Security tab of the Preferences dialog, by clicking on the Exceptions button next to "Warn me when sites try to install add-ons".  However, as long as the checkbox next to "Warn me when sites try to install add-ons" is checked, you'll always get this warning when you try to install an xpi manually.  I've never used SeaMonkey, but it probably has similar settings.

As long as you trust the site where you downloaded the xpi, go ahead and click "Install" to install the extension.  (Btw, I'm assuming you downloaded this xpi file from a website.  You can usually just click on the link to an xpi and automatically install it from the website without downloading it.)
