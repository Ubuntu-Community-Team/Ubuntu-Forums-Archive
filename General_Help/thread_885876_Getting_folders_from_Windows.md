### Post by WRDN on 2008-08-10
Although I can't answer your first query as I didn't install Ubuntu using Wubi, for the Firefox bookmarks:

- Open Firefox (in Windows)
- Select Bookmarks > Organise Bookmarks
- Click "Import or Export" button (if using latest version), or the Export option in the File menu
- In Ubuntu, select Bookmarks > Organise Bookmarks in Firefox
- Select File > Import and navigate to the file you exported in Windows
