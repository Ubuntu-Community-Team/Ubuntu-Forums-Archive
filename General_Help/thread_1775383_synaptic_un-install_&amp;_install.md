not 100% sure what your question is, but if you want to know how to search for a package in Synaptic, then simply type what you know of its name in the quick search bar. To remove a package that is already installed, right click on the package, and select remove, or completely remove. Then click apply in the toolbar. To install, select install from the right click dialogue.
