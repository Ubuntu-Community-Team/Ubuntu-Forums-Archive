### Post by gvkoeller on 2010-02-09
Hi there!

I am dealing with the same problem, and think I am on my way to solving it.

an ease way to connect to the AS/400 is via tn5250 o x5250 (which get installed at the same moment. You may find them easy [Download tn5250 0.17.3 for Linux - tn5250 is a telnet client for the IBM iSeries and AS/400 that emulates 5250 terminals and printers. - Softpedia]("http://linux.softpedia.com/get/Terminals/tn5250-32711.shtml") 

to configure it, and lear to use it: [tn5250(1) - Linux man page]("http://linux.die.net/man/1/tn5250")

it is a simple solution, that might be found too simple by the user...

However, now I am trying a very promising package, which I would recommend you to try: [http://n8.thruhere.net/export/free/ibm5250/](http://n8.thruhere.net/export/free/ibm5250/)

you will launch it with the command: ibm5250

Setup options available using command setup5250

Good Luck!:popcorn:
