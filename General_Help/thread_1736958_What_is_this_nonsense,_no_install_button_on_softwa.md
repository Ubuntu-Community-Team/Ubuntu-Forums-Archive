The Sun Java package is in the Canonical "Partner" repository. You can go to System > Administration > Software Sources > Other Software Tab, and enable it. Click the reload button on exiting to update the sources to include it.

Sun Java will then be available in USC or Synaptic or available with the apt-get command. This repo is not enabled by default.
