### Post by frafu on 2006-11-04
In a few words, you have to do the following: 

- create a new folder 
- mount the harddisk on the newly created folder

Post4 of [this]("http://ubuntuforums.org/showpost.php?p=1163487&postcount=4") thread talks about it. 

[Here]("https://help.ubuntu.com/community/MountingWindowsPartitions") you can find further information about mounting Windows partition. 

In the livecd of ubuntu 6.06, there was a control panel under the system menu, that provided a gui to mount partitions. I don't know whether it is also present in ubuntu 6.10.
