### Post by dschaller on 2006-05-12
If you don't mind using the command line, ps2pdf is a powerful application for creating PDF files from PostScript documents. I usually save as a .ps file from my word processor or another application, then run it through ps2pdf to create the PDF document. Although I do not usually create complex PDF documents, my understanding is that ps2pdf can actually do more than Adobe Acrobat Distiller can do and can create most any PDF output you would need. If you have Ghostscript installed, this little utility should be installed with it.
