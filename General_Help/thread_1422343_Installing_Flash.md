### Post by suntom on 2010-03-05
> sudo apt-get install ubuntu-restricted-extras
This will install a package with all the multimedia tools you need, like codecs - this includes the Flash Player.

> sudo apt-get install flashplugin-nonfree
This will just install the Flash Plugin.
