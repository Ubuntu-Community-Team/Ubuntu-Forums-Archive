### Post by Haoshoku on 2010-03-16
Uhm, firstly, there is NO insert key for a Macintosh keyboard/laptop. Apple - instead, replaced it with a fn key which means function.


Function keys :
[fn]+[return]= Insert
[fn]+[UPARROW]= Pg Up
[fn]+[DOWNARROW]= Pg Down

And a few more that i'm not really sure about, so I won't post it here.
