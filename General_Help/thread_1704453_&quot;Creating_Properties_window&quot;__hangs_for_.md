### Post by jaayll on 2011-03-10
I saw a couple of old threads not solved about this problem, like when trying to associate a media file with a player.

When right click on a file and selecting Properties, but "Creating Properties window, etc" goes for ever, you can go tu gnome-terminal, and do:

sudo nautilus

Enter you root password, in the nautilus window navigate for the file, right click and select Properties, and you will get Properties window with no delay.
