### Post by AfrikaDietmar on 2010-06-25
Since the debs didn't work i tried to use the windows version via wine, but there i was always getting an error message when i tried to move an item into another folder within chandler which brought me to post the bug in the wine forum where thanksfully "**dimesio**" advised me to try the compressed (tar.gz). Which you can followup here:
[http://forum.winehq.org/viewtopic.php?p=45167#45167](http://forum.winehq.org/viewtopic.php?p=45167#45167)

I followed the instructions as given here:
[http://downloads.osafoundation.org/chandler/releases/1.0.3/#enduserlinux](http://downloads.osafoundation.org/chandler/releases/1.0.3/#enduserlinux)

At the **Compressed Image (*.tar.gz)** section of that page.

Its working perfectly on ubuntu 10.04!

Give it a try.
