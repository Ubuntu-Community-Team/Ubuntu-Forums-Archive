### Post by QIII on 2020-11-11
There is no "Ubuntu Corporation".

Ubuntu is an operating system produced by Canonical, LTD.  It's a very small company with something around 600 employees.

While it is not likely that you will be able to convince anyone to do so unless they already plan to; the bug is a major one; or represents a security risk, the best place for users to get directly in touch with developers would be at [this mailing list]("https://lists.ubuntu.com/mailman/listinfo/Ubuntu-devel-discuss").

If you simply must have it right now, you can look [here]("https://www.libreoffice.org/").
