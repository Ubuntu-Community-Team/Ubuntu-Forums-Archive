I don't know of any Gnome-based distros that will run with that amount of ram, and certainly not Ubuntu (for which I recommend 512mb minimum). Is a ram upgrade a possiblity? Actually Vivaldi's link will be very helpful to you. It's the most complete thread here on the forums regarding which distros work well for low-spec computers such as yours. Read through that thread and many of your questions will be answered. :)

I would not recommend Xubuntu on 128mb. From Xubuntu.org:

> Minimum system requirements

To run the Desktop CD (LiveCD + Install CD), you need 128 MB RAM to run or 192 MB RAM to install. The Alternate Install CD only requires you to have 64 MB RAM.

To install Xubuntu, you need 1.5 GB of free space on your hard disk.

Once installed, Xubuntu can run with 192 MB RAM, but it is strongly recommended to have at least 256 MB RAM.
