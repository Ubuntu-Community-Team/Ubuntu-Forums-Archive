### Post by adit on 2010-05-05
Ctrl + L to give focus on address bar
Ctrl + K to give focus on search bar
Like these, is there a keyboard shortcut to give focus on the main content (where webpage is displayed).
By default when a new webpage loads, it automatically gets focus.  However, what to do if I am already working in the address bar and want to reach the main content area (without using mouse)?

It's been a long time since this question was posted but I just stumbled upon it searching for something else and I guess I could answer.

The [tab] button does all the magic here. When you're in an address bar then the tab button does:

[LIST]
[*]first press - go to search bar
[*]second - go to tabs bar
[*]third - **go to the content**
[/LIST]
