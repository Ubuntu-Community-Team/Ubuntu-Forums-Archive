### Post by Too Late on 2008-08-20
There should be no problem when installing a linux distro on a logical partition. (Logical partition is a partition inside the extended partition.) Some other operating systems, like BSD, are a different story. Remember that you can only have four primary partitions. One of them can be an extended partition, which can contain many logical partitions.
