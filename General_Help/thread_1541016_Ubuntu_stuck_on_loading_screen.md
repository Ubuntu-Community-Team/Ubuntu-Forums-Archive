### Post by deostroll on 2010-08-08
hi, even I am experiencing the same problem. However, I found a way you can still log-in. This is not a solution however. I suppose the problem lies with mounting the hdd. Enter the grub menu. Choose a kernel in recovery mode. YOu'll get the recovery mode menu. Choose to 'resume normal boot'. You'll be defaulted to a login shell. Type in username & password. In the shell type startx. This will start the x gui system, and gnome. Hopefully this will help for the time being.
