### Post by sayakb on 2008-07-06
Did you install awn from the repos (ie. Using Add/remove, apt-get or synaptic) Install the latest version of AWN which has a veriety of applets: [http://news.softpedia.com/news/Install-AWN-on-Hardy-Heron-82611.shtml](http://news.softpedia.com/news/Install-AWN-on-Hardy-Heron-82611.shtml)
When you start awn, it is just a small black bar at the bottom with nothing. Right click on it and click **Configure AWN **or **Dock Properties **and add applets from there.
