### Post by merlinus on 2007-07-15
If you moved partitions around, the UUIDs changed.

```

blkid

```

will show them for the various partitions listed in

```

sudo fdisk -l

```

Compare the results with those in /etc/fstab, and change where necessary.

Then reboot.

-merlin

### Post by merlinus on 2007-07-15
blkid output is fine, but /etc/fstab is the one from the live cd, not ubuntu.

You will have to mount that partition manually, and then use Nautilus to browse to its filesystem and /etc/fstab.

Here is some good advice:

[http://users.bigpond.net.au/hermanzone/p10.htm](http://users.bigpond.net.au/hermanzone/p10.htm)

Resizing and moving partitions will always change the UUIDs.
