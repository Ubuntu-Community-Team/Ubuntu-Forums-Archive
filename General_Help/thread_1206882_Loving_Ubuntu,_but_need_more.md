### Post by aysiu on 2009-07-07
FrontPage has no real equivalent, because it's a terrible application. Dreamweaver has no real equivalent, because Linux users who design websites tend to prefer handcoding to WYSIWYG. But you can try Kompozer.

OneNote has no real equivalent. You can try Basket or Zim.

The "grand puba" of software repositories is... the software repositories. They're pretty easy to search or browse through. If you don't know how to use them, read this:
[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)

### Post by pricetech on 2009-07-07
The best replacement for frontpage is to learn HTML.  There's an abundance of free material on the web.

If you're looking for an HTML editor, try Arachnophilia.  I haven't used it under Linux, but it's written in Jave so it's cross platform and the author has a tutorial on making it work under Linux.  the winders version includes an installer so you can tinker with it in that environment as well.

### Post by mwolfer1 on 2009-07-07
Kompozer is a really useful tool, if you need to quickly design something and start from scratch.  
The problem really is that if one works on existing material that has been created with MS centric tools like Frontpage, it's almost impossible to get compatible results. Even simple formatting is off, mainly due to the ridiculous overhead these tools create in the source.
I ended up having an XP environment in VirtualBox for Frontpage to maintain web sites for some non-profits I volunteer for.
