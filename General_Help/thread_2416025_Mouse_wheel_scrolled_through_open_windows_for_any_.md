### Post by swarup on 2019-04-04
The mouse wheel used to scrol through open windows for any application in the Dock in 14.04, but it doesn't do this in 18.04. Is there a setting to activate this?

For example, if TB is open and there are three emails one is in the middle of composing, then if one overs the mouse of the TB icon in the dock and scrolls using the mouse wheel, it will scroll through the open emails as well as the main application and it saves a lot of time and effort in reaching to the desired page. Whereas because that facility is not working now, one has to right click on the TB icon in the dock, request to show "All Windows", whereupon all are shown as tiles in an overview, and then one can select the desired window. It creates two extra steps. And this goes for every application in the dock which has multiple windows open.
