### Post by JRV on 2012-11-14
Try turning it off with ubuntu-tweak.
```

sudo add-apt-repository ppa:tualatrix/ppa
sudo apt-get update
sudo apt-get install ubuntu-tweak

```
Run the program, select Tweaks=>Unity.
