### Post by minisori on 2008-11-07
I have hear some acer recovery disks only work with the same partition layout they come out with, but not sure about this.

If you have ubuntu live cd try to retart the comp with it, then run gparted and remove the linux partition and make a ntfs one, maybe it works like that. :S

Anyway if you have a data partition, as many acer come with, you can still install linux on that partition and keep vista.

Also if you know anyone with the same windows version as the one you had in the comp, (probably home premium), you can install it with the key in the back of your laptop.
