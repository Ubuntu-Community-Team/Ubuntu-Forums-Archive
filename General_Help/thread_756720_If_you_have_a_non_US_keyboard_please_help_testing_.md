Wubi should detect the keyboard layout and install accordingly.
We have mixed reports at the moment.


1) Uninstall and remove any old ISO (also from ubuntu-backup)
2) install using Wubi 487 with today's daily ISO, use a dummy password (will be in the logs below hashed)
3) Before reboting back-up c:\ubuntu\install\custom-installation\preseed.cfg
4) When you reboot, after you select "Ubuntu" press "esc" and go for verbose mode.
5) When the installation is finished, see if the keyboard layout is correct (test @ " and othe keys). 

If it is report it here
If it is not please attach the preseed.cfg above (#3), moreover pls zip and post /varl/log/installer (whithin ubuntu).
