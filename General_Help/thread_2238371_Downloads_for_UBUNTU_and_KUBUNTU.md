### Post by grahammechanical on 2014-08-07
Ubuntu 12.04 was the last version to have Synaptic Package manager as part of the default installation. Kubuntu has similar utilities as Ubuntu but they have different names because they are different utilities. So, Ubuntu Software Centre is not in Kubuntu. Not only that but Kubuntu also has applications built especially for the K Desktop Environment (KDE). They usually have a name beginning with the letter K. They will run in Ubuntu but the themes might not match so well.

The Kubuntu equivalent to Ubuntu Software Centre is Muon Software Centre.

[http://kde-apps.org/content/show.php/?content=137507](http://kde-apps.org/content/show.php/?content=137507)

All the flavours of Ubuntu will have utilities and applications in keeping with the desktop environment the flavour is built upon. With Ubuntu it is Gnome DE.
