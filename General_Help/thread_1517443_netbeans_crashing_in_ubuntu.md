### Post by Pollox on 2010-06-25
What version of Ubuntu are you using?  Ubuntu 9 is not a thing.  You may mean Ubuntu 9.04 or Ubuntu 9.10.  You can find out by going to the System menu and clicking About Ubuntu.

How did you install netbeans?  From the official repositories?  That is, did you use add/remove or Synaptic, or did you download from the netbeans website?

Do you get any error message when it is crashing?  No one can help you without some information about the crashes.  Are you doing anything in particular when it crashes, does it happen right when you start netbeans, etc?  If you don't get an error message, try running it from terminal.  That is, go to the Applications menu, and under Accessories click terminal, then type netbeans in there.  It should output some text to the terminal when it crashes.
