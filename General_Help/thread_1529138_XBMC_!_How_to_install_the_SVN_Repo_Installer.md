### Post by JustinR on 2010-07-11
Hi!

I use XBMC, its nice media player.

So anyways, click Places on the top Ubuntu taskbar and click Home Folder. Then type CTRL + H to show hidden files. Find .xbmc, then navigate to Plugins\Programs. Then copy the extracted SVN repo installer folder into that and start XBMC.
