### Post by wieman01 on 2007-11-07
Frankly, 3 GB is too small.

I usually recommend 3 partitions at least:

1) SWAP = Size of your RAM
2) ROOT = 5 GB
3) HOME = 3 GB

If you have any space left, set up a separate data partition. But DSL might be an option for you:
