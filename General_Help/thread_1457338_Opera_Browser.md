### Post by cguy on 2010-04-18
You want the version of Opera which does not have GUI libraries bundled. (GUI = Graphical User Interface)
Uninstall what you already have installed. You need the "shared" version. It uses the libraries that are installed on your computer.

Grab it from here:
[ftp://ftp.opera.com/pub/opera/linux/1010/final/en/](ftp://ftp.opera.com/pub/opera/linux/1010/final/en/)
