### Post by Zularan on 2008-11-11
There are two options that I know of:

*Menu -> Applications -> Accessories -> Character Map* works like the windows one.

Alternatively, (my preference) you can install gnome-specimen
```
sudo apt-get install gnome-specimen
``` will graphically show fonts.
