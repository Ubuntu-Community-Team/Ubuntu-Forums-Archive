### Post by kerry_s on 2007-05-13
Try Xubuntu or Fluxbuntu, when you get comfortable enough with linux go custom install, nothing beats building ubuntu to your needs. Xubuntu will be easier for a newby, it's just as bloated as gnome, nut uses xfce4 which is geared more towards speed, yeah right. The net installer can install all the supported ubuntu variants, including a custom install when you get familiar enough. Fluxbuntu is a little bloated for a fluxbox install, but it should be faster and you can always add and remove what you want/don't want.

net installer-> [http://mirror.linux.org.mt/mirror/ubuntu/dists/feisty/main/installer-i386/current/images/netboot/mini.iso](http://mirror.linux.org.mt/mirror/ubuntu/dists/feisty/main/installer-i386/current/images/netboot/mini.iso)

Fluxbuntu-> [http://modzer0.cs.uaf.edu/~hardwarehank/fluxbuntu/rev2/fluxbuntu-nbuild1-rev2-desktop-i386.iso](http://modzer0.cs.uaf.edu/~hardwarehank/fluxbuntu/rev2/fluxbuntu-nbuild1-rev2-desktop-i386.iso)


There's more but your not there yet. Take some time learn, explore, get familiar.
