### Post by vandorjw on 2010-08-16
Yes you can. However, since that board only has PCI-Express 1.1 it might not be able to transfer data as fast as possible from the card to the motherboard.

However, newer cards, even though they'll be either PCI-E 2.1 or PCI-E 3, should all be backward compatible with PCI-E 1.0 and up.

Basicaly, it is not the bits you need be worried about, but the...for lack of a better word, architecture.
