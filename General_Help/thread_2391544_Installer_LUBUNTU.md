### Post by oldfred on 2018-05-10
Generally better to install Windows first. Also what version?
But is hardware UEFI or older BIOS only? 
If newer UEFI, then how you boot install media UEFI or BIOS/Legacy/CSM is then how it installs for both Windows & Ubuntu. And both systems need to be in same boot mode.

       BIOS & UEFI Windows partitions, note system has totally different format  & meaning between BIOS & UEFI
