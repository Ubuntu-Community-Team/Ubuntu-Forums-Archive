### Post by Ubutuxer on 2011-12-20
What is the correct way to enable guest auto-login when customizing the live-CD with UCK?

I have copied the user settings to /etc/skel and the customized CD works otherwise well, but it stops at the login screen. I remember editing some initrd scripts in an older Ubuntu version but I don't think this is the correct way anymore. I also don't think adding users to the /etc/passwd file on the ISO image is the right way for a guest user auto-login.

Any ideas?

---

### Post by Ubutuxer on 2011-12-21
Figured it out myself. Put this to **/etc/lightdm/lightdm.conf**:

```

[SeatDefaults]
greeter-session=unity-greeter
user-session=IceWM
autologin-user=ubuntu
autologin-user-timeout=0

```

That's the whole file. Change **user-session** to something else if you don't use IceWM. I think you can probably even leave it blank for the default session.
