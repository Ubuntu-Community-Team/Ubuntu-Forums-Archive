### Post by Verbeck on 2010-11-10
in system> preferences >main menu,check 'file management' under system-preferences

if its not there, the command is 
```
nautilus-file-management-properties
```

hope this helps

### Post by 101011010010 on 2010-11-10
Hello.
Have you tried "gconf-editor"? The option that relates to your gif image is:
> apps/nautilus/preferences/executable_text_activationThe "long description" gives the 3 command options.
I hope this helps.
