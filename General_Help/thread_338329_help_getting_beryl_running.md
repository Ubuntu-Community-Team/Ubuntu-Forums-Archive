### Post by Waappu on 2007-01-14
Hi

Follow careful those guides. Before you start beryl you may want check is window decoration plugin enabled and maybe for performance disable blur and water plugins. Go to system->preferences->beryl setting manager. You can find those plugins under visual effect and extras.

Or if you want here is script that uninstall ATI proprietary "fglrx" driver if you have installed that, install beryl and do few modification to /etc/X11/xorg.conf file.
[ati-aiglx-beryl.sh]("http://koti.mbnet.fi/waappu/download/ati-aiglx-beryl.sh")  USE AT OWN RISK!!

Save it to home folder and type
```
sudo chmod a+x ~/ati-aiglx-beryl.sh
sudo sh ~/ati-aiglx-beryl.sh
```

after script finish you still need to modify your /etc/X11/xorg.conf file Section "Device".
Here is mine for example. Don't change "Identifier" and "BusID" to matching this example. Leave those what already exists in your file.
```
Section "Device"
	Identifier	"ATI Technologies, Inc. RV280 [Radeon 9200 PRO]"
	Driver		"ati"
	BusID		"PCI:1:0:0"
	Option 		"DRI" "true"
	Option 		"ColorTiling" "on"
	Option 		"EnablePageFlip" "true"
	Option 		"RenderAccel" "true"
        Option		"AccelMethod" "XAA"
        Option		"XAANoOffscreenPixmaps" "on"
	Option 		"AGPFastWrite" "on"
        Option		"AGPMode" "4"
	Option      	"AGPSize" "32"
EndSection
```
After re-boot login to Gnome session and press Alt+F2. Type beryl-manager. If Beryl start and runs Ok you can add Beryl to startup programs. system->preferences->sessions goto to startup programs tab, press add and type beryl-manager.

I hope this helps
