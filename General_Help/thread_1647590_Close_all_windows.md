### Post by Michael_Brasher on 2010-12-17
Sometimes when I am away from the computer my cats walk on top of my keyboard and end up sitting on the print screen button. When I return, I'm greeted by a seemingly endless amount of screen shot attempts. I have to hold alt+f4 for several minutes before I can use my computer. I have no idea how many windows I'm closing in this process but I imagine it must be a few thousand.

Is there any easier way to do this? I have used google to search for some kind of shortcut to close all windows but what I have found has been useless. Is there any way to close all windows without having to hold alt+f4 for several minutes?

Help!

---

### Post by WorMzy on 2010-12-17
Assuming it's gnome-screenshot you're using, then this should work when entered in a terminal:
```
killall gnome-screenshot
```

If it's not gnome-screenshot, then substitute it with whatever you use instead.
