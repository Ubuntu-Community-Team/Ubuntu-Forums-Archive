### Post by hawthornso23 on 2010-05-18
```
 xrandr -s 1280x1024
```changes screen resolution to the indicated value.

Some applications, especially games, don't put the screen resolution back where they found it. This can be frustrating if you find yourself then unable to reach stuff with the mouse to fix the problem. I've assigned a keyboard shortcut to this command which gives me an instant fix in these situations.
