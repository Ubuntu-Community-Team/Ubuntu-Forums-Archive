1. Do you mean the Adobe Reader or Adobe Flash?
2. Type *uname -a* in the Terminal and you know if you are using 32 or 64 Bit.
3. *sudo apt-get install skype* or search for Skype in the Software Center.
4. There is no native Yahoo Messenger for Linux but you can use Pidgin. *sudo apt-get install pidgin* or search for it in the Software Center

Hi there,

For better responses to your problems try starting a thread in the Absolute Beginners Forum. It'll get more attention there.
In the meantime, to find out if you have 32 or 64bit go to System > Administration > System Monitor and click on the tab called System, if it doesn't say its 64bit then you have the 32bit Version.

When you start the new thread explain how you are trying to download the apps. Is it from the websites or via the Ubuntu Software Centre?

Ubuntu software centre is the best place for any newbie to download apps.

Find it under Applications > Ubuntu Software Centre  then just type in the name of the app your looking for and click download.
