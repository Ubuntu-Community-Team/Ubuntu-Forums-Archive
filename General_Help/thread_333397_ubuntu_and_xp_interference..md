### Post by gaebriel on 2007-01-07
There's no way that Ubuntu is causing your problem just because it's installed on the same system. The only time the ubuntu files are going to be loaded is when grub runs on bootup or when you're actually running ubuntu. If this doesn't happen when you're ubuntu, then I doubt it's a hardware issue. Unless it only happens when you're running a particular program within Windows that especially taxes a particular piece of hardware like your video card or memory or something like that. Your problem is most likely Windows doing it's common thing of corrupting parts of itself once in a while which is fixed by a reinstall of Windows. Good luck troubleshooting. :)
