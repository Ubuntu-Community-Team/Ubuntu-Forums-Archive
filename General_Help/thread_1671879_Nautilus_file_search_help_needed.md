### Post by buteobuteo on 2011-01-20
Can't access windows but need to find *.pst file. Have found nautilus will not accept *.pst as valid search input and pst files are not included in the list of extra types.  How can I find the missing pst files so I can load XP and Unbuntu on a freshly formatted drive?

### Post by abdulapopoola on 2011-01-20
.pst files are usually located in a path like this: C:\Users\username\Application Data\Microsoft\Outlook\outlook.pst

Search here and copy your .pst files to watever location you want.
