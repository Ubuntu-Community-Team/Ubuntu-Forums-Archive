In that case it seems that X can't automatically configure itself for your hardware, and a human needs to intervene.

However, if you want to disable the default gnome window manager's (metacity's) compositor (about that warning in the previous post), run
```

gconf-editor
```

and go down to apps > metacity and then click general. Untick compositing manager, then run
```

compiz --replace
```

in a terminal and paste the output.
