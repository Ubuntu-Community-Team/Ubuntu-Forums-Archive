### Post by ad_267 on 2008-10-05
You could do this using symbolic links.

To do your example make a new folder called topic and open that up. Open up a second file browser and browse to Music. Middle click and drag the file you want into your topic folder and a menu will pop up. Select "Link Here". You can make symbolic links to directories too. These aren't just shortcuts like in Windows, they're a lot more useful.
