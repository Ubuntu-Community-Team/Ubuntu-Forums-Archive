### Post by cogier on 2010-03-06
If you install Ubuntu and select the "Erase and use entire disk" option this will erase Windows. Full details are here [http://news.softpedia.com/news/Installing-Ubuntu-9-10-126370.shtml]("http://news.softpedia.com/news/Installing-Ubuntu-9-10-126370.shtml")

The viruses are probably all Windows viruses and so will not trouble Ubuntu but you can pass them on to your Windows friends. If you have installed Ubuntu over the entire disk the viruses will all be wiped out. If you have backups the viruses could still be there. So install a virus checker just in case.
