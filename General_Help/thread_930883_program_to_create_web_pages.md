### Post by basenvironment on 2008-09-27
You can also install seamonkey/mozilla it includes a web page editor tool.

Anymore I actually prefer a good editor that has code completion. I type in the start of a tag and it finishes it for me with a closing tag. I define a few of my most used tags so I can quickly and easily insert them. I find I am faster working that way than trying to tweak/change the code that a wysiwyg editor creates. I keep a browser open and switch to it and refresh it if I need to see my changes.
