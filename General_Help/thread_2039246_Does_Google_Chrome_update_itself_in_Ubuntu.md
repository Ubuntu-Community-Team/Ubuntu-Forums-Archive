### Post by adam.d.clemons on 2012-08-08
Quick answer, No. Google Chrome will not update itself, it updates the same way all the other software on ubuntu updates through the update manager. 

Yes, Adobe is killing Flash for Linux, version 11.2 is the last version to target the Linux OS, but have no fear, there's HTML5, and Chrome.
