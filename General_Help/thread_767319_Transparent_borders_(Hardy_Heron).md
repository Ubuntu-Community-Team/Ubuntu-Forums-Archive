### Post by PhoenixP3K on 2008-04-27
> **PhoenixP3K said:**
> 
Further more there used to be and additional 4th option in the Appearance settings windows that lets you personalize desktop effects. It is now gone.


I must say that I found the solution to that part of the problem. 
The blog post I found is at this url: [http://tombuntu.com/index.php/2008/03/27/custom-compiz-effects-in-ubuntu-804/](http://tombuntu.com/index.php/2008/03/27/custom-compiz-effects-in-ubuntu-804/)

Basicly you need to have Simple Compiz Config Settings Manager to enable the 4th option in the Appearance window.

```
sudo apt-get install simple-ccsm compizconfig-settings-manager
```
