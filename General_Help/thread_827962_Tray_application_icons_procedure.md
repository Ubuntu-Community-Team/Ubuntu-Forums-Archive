### Post by Zorael on 2008-06-13
As for klipper, just start it manually, then right-click the icon and pick Quit. A dialog should pop up to ask you if you want it autostarted upon next login, so just pick yes. Then start klipper again to get it back into your tray for your current session.

As for wicd, this is probably unrelated to your other tray issues; perhaps it is merely a problem with the version of wicd you are using. Make sure you can start the main window manually - else it may need to be reinstalled.
