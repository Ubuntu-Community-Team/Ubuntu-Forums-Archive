The EeePCs do have fans.  My system is very stable, cool enough, and much much faster when overclocked.  IMO, keeping it at stock speeds would be the bad idea. ;)

I overclock mine by using the eee.ko module that someone made for the Celeron EeePC's.  It allows you to use the command line to control your fsb, fan, voltage, etc.  It's fantastic but it's designed for the old Celeron EeePC's, not the Atoms.
