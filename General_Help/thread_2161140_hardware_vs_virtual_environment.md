### Post by lukeiamyourfather on 2013-07-09
The lag might be from the graphics drivers. Ubuntu will run faster on hardware natively, but you should also be able to run it in a virtual machine without discernable lag. Try installing the "extras" or "addons" in the guest for your virtual machine software if they exist.
