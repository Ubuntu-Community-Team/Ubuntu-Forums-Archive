### Post by searchfgold6789 on 2011-10-27
If you simply reformatted the disk that Windows was on *and then installed Ubuntu on the same hard disk*, especially the same area of the hard disk, there is little or no chance that you can get the files back. 
For general file recovery, however, I always recommend [TestDisk]("http://www.cgsecurity.org/wiki/TestDisk"), which as far as I am aware is the best tool out there for this purpose, hands down.
