### Post by geirha on 2009-09-24
I recommend installing the package ubuntu-restricted-extras, using the method described here:

[https://help.ubuntu.com/community/Synaptic/PackageDownloadScript](https://help.ubuntu.com/community/Synaptic/PackageDownloadScript)

The ubuntu-restricted-extras package will install codecs for most formats, and some other commonly installed packages, like unrar, flash plugin and microsoft ttf fonts.
