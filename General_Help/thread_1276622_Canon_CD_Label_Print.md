### Post by Monocerotis on 2012-09-09
Just for the record, what you need to install is a module called "Canon inkjet printer driver add-on module" in order to use the CD-LabelPrint that comes with Canon Pixma printers. I'm not sure if this is available for Linux. But at least in Windows you will need to install this one. I don't think you need it in Windows XP. But from Windows Vista and up this became a necessity. I remember I had this problem when I upgraded to Vista.

This is sort of a way for Canon to make sure that the LabelPrint software is used on a Canon brand printer. It was specially designed for Canon printers, so basically no other printer brand can even make use of it. So I don't get it why they need to make sure you use a Canon printer.

CD-LabelPrint serves its purpose well, but there are better programs available. I personally use Adobe Illustrator, but you can use any program, you just need to make sure to select the CD-R tray as paper source in the printer driver, or the print dialog.
