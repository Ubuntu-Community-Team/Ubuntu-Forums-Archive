### Post by venturosa on 2011-03-18
I'm having a display problem with both AbiWord and OpenOffice. They both display correctly at the start of a session but if I open a second window or close the program and later start it again the display is unreadable with a grid of horizontal lines over all words including those on the navigation bar. Any suggestions?

---

### Post by venturosa on 2011-03-19
I've eventually found a workaround for this:
Go to: System/Preferences/Appearance/Fonts
Choosing "Subpixel smoothing (LCD)" as font rendering option leads into the font corruption problem. Changing to "Best shapes" corrects the problem. For a better font appearance change Hinting to slight.
Both programs work perfectly now.
