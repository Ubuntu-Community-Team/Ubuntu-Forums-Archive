Ubuntu 18.04, gnome shell. ### Post by Frogs Hair on 2018-11-29
Open Tweaks > Top Bar and enable the application menus for the top panel. This takes application menu from the windows and places it on the top panel. The icon style in the window is determined by the set in use.
