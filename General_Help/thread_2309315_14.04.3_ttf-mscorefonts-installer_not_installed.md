### Post by Bashing-om on 2016-01-09
Kris_M; Hi !

To install the proprietary software, run terminal command:
```

sudo apt-get update
sudo apt-get upgrade
sudo apt-get install ttf-mscorefonts-installer

```

There will be a UELA that you must accept that is sometimes hidden behind other windows that you might have open;
When the EULA pops up use the space bar to page down, Tab to highlight "Ok", then Enter.
