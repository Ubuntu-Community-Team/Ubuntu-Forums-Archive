I installed the screenlets program that I downloaded from the compiz site and compiled (the latest version) and I have an orb screenlet that I want to use. How do I use that screenlet?

---

### Post by irjenius2100 on 2008-05-26
Save the .tar.gz file that the screenlet is in to your desktop.  Open the screenlet daemon and just drag the .tar.gz file into the window.  Once you have done this you will get a pop up asking if you would like to install the screenlet and once you click ok it will be done.  The screenlet will now appear in the list of installed screenlets.
