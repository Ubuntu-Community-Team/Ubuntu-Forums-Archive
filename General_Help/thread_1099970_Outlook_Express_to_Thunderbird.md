### Post by KeyserSoze93 on 2009-03-19
Sadly, the native Ubuntu version of Thunderbird doesn't support importing from Outlook. The Windows version, however, does. So the best thing to do might be to install Wine + Windows T-Bird, then import the outlook files with that, and copy the imported thunderbird profile in as the default profile for native thunderbird.
