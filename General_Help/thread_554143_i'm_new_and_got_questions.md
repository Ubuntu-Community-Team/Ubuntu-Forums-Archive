### Post by treb0r on 2007-09-18
Hey,

Well, you can't install Ubuntu on an ntfs drive - you will need to create an ext3 partition.

I've not had much success booting from external drives - it's usually better to shrink your windows partition and install it on the internal drive, but then I've never tried that with Vista, only XP.

The upcoming 'Gutsy Gibbon' release of Ubuntu is scheduled to include the desktop effects by default, and is due out in a few weeks. Otherwise you can install them using synaptic.

### Post by Lord Illidan on 2007-09-18
Welcome to Ubuntu!

No, you can't install on an NTFS harddrive.

However, you can resize your existing Vista partition from Vista itself and install Ubuntu in the free space.

Regarding the desktop effects, there are some included, but the really flashy desktop effects have to be installed afterwards. There is some tweaking to be done, but I assure you the end result is worth it.

While we're at it, I'd like to introduce you to some helpful links : [https://help.ubuntu.com/community/Installation](https://help.ubuntu.com/community/Installation)

Feel free to ask more questions!
