### Post by delcypher on 2009-10-11
You want something lightweight on your netbook, Don't use KDE 4, the thing's a resource hog! GNOME might be okay but, I'm a fan of XFCE ([http://www.xfce.org/]("http://www.xfce.org/")) on my netbook, nice and nippy.

If you decide to use xfce then you won't have NetworkManager by default, try installing wicd instead of installing NetworkManager (will install GNOME stuff - which is probably a waste of space)

You can install it as a package after installing ubuntu and configure or you can use xubuntu ([http://www.xubuntu.org]("http://www.xubuntu.org"))



Also read here about some other lightweight Desktop managers
[http://elevenislouder.blogspot.com/2009/10/desktop-environments.html]("http://elevenislouder.blogspot.com/2009/10/desktop-environments.html")
