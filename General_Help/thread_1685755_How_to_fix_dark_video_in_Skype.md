### Post by coldraven on 2011-02-11
The problem was very dark video in Skype on my friends Asus laptop running Ubuntu 10.04
The solution was to:
Install guvcview from the software Centre.
Make sure Skype is **not** running.
Run guvcview and increase the brightness.
Quit guvcview.
Run Skype>Options>Video>Test and see if your picture has improved. If not quit Skype and start again.

Later on you can try adjusting the gamma for better colour reproduction.
It's probably best to only change one thing at a time even though it is time consuming.
