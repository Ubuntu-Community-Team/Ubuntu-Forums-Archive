### Post by Xgen on 2008-07-23
Yes, mine does that if nvidia is enabled, but not (properly) installed. I'm surprised that you had a problem with Envy though...always worked with my GeForce 5500. Now that I'm using Intrepid I've had to get back to installing the driver manually. Perhaps you could try  (if nothing works in the repos or no one has an easier suggestion):

Download the latest driver from NVidia - 173.14.09.
Make sure build-essential is installed (Synaptic) or 'sudo apt-get install build-essential'
Ctrl | Alt | F6
Stop X - 'sudo /etc/init.d/gdm stop'
'sudo sh (path to NVidia file)'
Don't let it change the 'xorg.conf' file. I've had problems with that and had to 'sudo dpkg-reconfigure -phigh xserver-xorg' to change it back to the original.
'sudo nano /etc/X11/xorg.conf" and add  'Driver  "nvidia" ' to the 'Device' section. 
Start X - 'sudo /etc/init.d/gdm start'
So... basically, all I'm just running is the original xorg.conf with 'nvidia' added, and NVidia settings and Compiz work just fine.
