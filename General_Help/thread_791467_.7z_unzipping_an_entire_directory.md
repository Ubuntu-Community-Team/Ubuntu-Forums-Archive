I was wondering what the command would be to unzip an entire directory using .7z 

I know the command for each file is 

7z x file.7z

---

### Post by Monicker on 2008-05-12
7z e file.7z


That will unzip all the files in the archive into the current directory.  FYI - just typing 7z gives the most common options.

### Post by Monicker on 2008-05-12
Define "didnt work".

What happened?  Were there error messages? Don't skimp on the details.


Also, try this:

```
for i in *.7z; do 7z e "$i"; done;
```

Paste as is, all on one line, and hit enter.
