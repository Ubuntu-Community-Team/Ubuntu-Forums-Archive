### Post by john183 on 2008-11-18
If i am not mistaken, FireFox 3 has a "phishing" filter similar to the one in Internet Explorer 7. It uses a database that is stored on the web to warn you of malicious sites. To disable it open the Firefox preferences and on the Security "tab" uncheck "Tell me if the site I'm visiting is a suspected attack site". That should stop the blocking of sites.
