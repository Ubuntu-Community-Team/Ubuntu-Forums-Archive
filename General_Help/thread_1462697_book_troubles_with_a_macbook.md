### Post by 3rdalbum on 2010-04-26
> **wcutrombonekid said:**
> so, i told my friend to try ubuntu on his macbook.  He put  in the live cd and restarted and it didn't work.

You have to hold down the C key in order to boot up from CD.

Try posting in the Apple Users subforum?

BTW, Target Disk Mode is something completely different; it lets you access drives from one computer on another computer, via a Firewire cable. Get your friend to unplug all Firewire devices and start up. If this doesn't work, then try holding down the Option key while starting up.
