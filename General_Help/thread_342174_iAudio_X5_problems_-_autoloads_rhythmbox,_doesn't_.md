### Post by balance07 on 2007-01-19
i have an iAudio (Cowon) 60GB X5 portable music player. i love the player, it works great, but i have 2 nagging issues w/ it in Ubuntu (Edgy)

1) when i connect it via USB, an icon pops up on my desktop called "IAUDIO" with an iPod icon. it gets mounted as /media/IAUDIO. at the same time, Rhythmbox opens up and tries to scan it or something. i don't need or want rhythmbox to autoload when i plug in my X5. is there any way to keep this from happening?

2) when i am done transferring and organizing files, i right-click on the IAUDIO icon and choose "unmount volume", then the icon disappears. however, the X5 doesn't recognize that its "OK to disconnect" like it does in windows when i choose to disconnect it.  i just want to make sure i'm not gunna lose any data by prematurely disconnecting it. any fix for this?

thanks in advance for any help.

### Post by balance07 on 2007-01-20
ok for anyone with the same issues, i found the solution to my first problem:

under Preferences -> Removable Drives and Media -> "Multimedia" Tab...

there is an option for Portable Music Players. disabling "play music files when connected" did the trick.

### Post by cracker on 2007-01-20
I have the 30GB X5, and I've been using it for a couple months with edgy, and have not had the slightest hint at losing data. I never noticed that it even said anything different on Windows, to be honest. As long as you unmount the volume properly, you won't lose data, because the system is no longer able to read/write anything to the player.
