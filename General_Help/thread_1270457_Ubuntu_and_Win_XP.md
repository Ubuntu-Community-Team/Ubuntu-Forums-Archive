### Post by solidblu on 2009-09-19
You should be able to.  There are a few different virtual machine programs out there and all the ones that I have used let you specfy specific hardware as the virtual machines's hardware (cdrom).


I know some people that use itunes in a virtual XP machine to sync their iPhones so I see no reason why a scanner would be any different.
