### Post by ago on 2008-09-25
Not sure what you would gain to have wubi if you already have a dual boot setup (you can easily install other desktop environments within your existing installation...). But if you want to do so, yes you can, but note that wubi will only modify the XP boot menu. This means that you will have 2 boot menus, in the first you select WINDOWS, then in the second you will have to choose between Windows or Ubuntu (Wubi).
