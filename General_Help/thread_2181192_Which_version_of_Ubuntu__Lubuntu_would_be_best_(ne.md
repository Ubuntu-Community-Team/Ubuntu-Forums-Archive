### Post by Petro Dawg on 2013-10-17
I think Ubuntu 12.04 would be fine on nearly any dual core.  I run it without issues on machines with as little as 1GHz single core.  Also, 1G of Ram would be adequate (barely).   If the system is too light to run standard Ubuntu, it will log you in under the 2D fallback mode; which works great in most cases.  I would try it first, and if it is too slow for your taste, then try a lighter distro like Xubuntu, or Lubuntu as mentioned by others.  I also expect you will need to use the 32bit downloads, so be sure to get the right version.
