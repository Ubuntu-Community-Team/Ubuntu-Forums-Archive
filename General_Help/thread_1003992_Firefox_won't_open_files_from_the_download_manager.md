### Post by geoaraujo on 2011-11-01
> **MarcoPau said:**
> I've always had the same problem, and it's still there, firefox 7 and kubuntu 11.10. Any hint?
Yes. Install package *firefox-kde-support* and it's done! Now you can open files and the containing folder. For more details, have a look at [http://forum.kde.org/viewtopic.php?t=17786](http://forum.kde.org/viewtopic.php?t=17786)
Cheers.
