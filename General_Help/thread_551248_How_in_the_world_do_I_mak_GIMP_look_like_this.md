### Post by mojoman on 2007-09-15
I use Fluxbox as my wm and it has a lovely little tab on the window. If one middle clicks on that tab and drag the window into another window the will be in the same window but with tabs, much the same way that firefox or opera functions. This is an exellent feature when one runs applications from the terminal (i.e. keep the terminal and the application in one window for good debugging overlook and to keep from cluttering the desktop) but it is absolutly fabulos to use for Gimp. All the different windows in one single window, all neatly tabbed. To the best of my knowledge this only exist in Fluxbox, but then again, Fluxbox is simply awesome...
