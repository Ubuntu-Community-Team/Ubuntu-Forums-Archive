### Post by amsterdamharu on 2011-02-11
I had a problem like this once (nothing to do with installing an OS though). One day I try to boot up and nothing happened.

I took it back because I bought it a couple of months before and the guy from the shop took out the memory and put it back, everything worked. What bothered me was that the "beep" speaker wasn't connected so I asked him to connect.

Now when the computer starts it'll beep once for ok and have several beep signals for not ok. One way to find out if the speaker is connected is to remove the memory and boot, it should contentiously beep in some pattern. If that happens you could put it back and see if you get another beep pattern when starting up.

You've tried unplugging the video but you could try unplugging everything and see if you at least get a BIOS screen and then plug in stuff one at the time.
