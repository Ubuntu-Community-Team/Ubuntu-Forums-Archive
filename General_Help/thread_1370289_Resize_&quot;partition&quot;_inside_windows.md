### Post by nerdtron on 2010-01-02
You mean you installed ubuntu using wubi?
Sorry, I don't know to resize the 'partition' you allocated for wubi... But you can access and write to the host drive by going to Places>Computer>open the filesystem and you will see the host folder where you can save your files.
I know this is not the solution for your problem but I hope it helped something for you.

But if you really love Ubuntu why not create a dedicated partition for it?

### Post by prshah on 2010-01-02
> **linuxnovice. said:**
> my problem is that I installed ubuntu inside windows and I only have one partition

> **nerdtron said:**
> why not create a dedicated partition for it?

You can use [LVPM (Loopmounted Virtual Partition Manager)]("http://lubi.sourceforge.net/lvpm.html") to change the size of your wubi ("within windows") install and/or to transfer your existing install to it's own dedicated partition. Exhaustive details (including screenshots) are available at the link, but feel free to post back here if you have doubts or comments.
