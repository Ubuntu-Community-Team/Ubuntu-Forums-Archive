### Post by Keith Hedger on 2009-04-14
Usually if you download an image you would have to sort out where to put it yourself ( unless its a complete theme file ), for instance to change the splash image for gimp replace /usr/share/gimp/2.0/images/gimp-splash.png

Other programs would be in similar places ie: /usr/share/SOMEPROGRAM
