### Post by gpsmikey on 2008-11-16
How did you create the CD ??  
Does this CD boot in a different machine ??  
If you didn't burn the CD from the ISO correctly (as an image to the CD), but instead burned the ISO to the CD as a data file, it will not boot.  Also, make sure the image you used to burn from was a valid download (check the checksum against the download list).  When you burn the CD, make sure you enable the verify option if your burning software supports that too - cheap media often gives errors when you get over about 50%.
