### Post by Autodave on 2018-11-20
+1 with deadflowr.  If this has an easily removable battery, try powering down the unit and removing ALL cables.  While the battery is out, let's play with the keyboard.  (If you cannot remove the battery, just make sure that it is powered down.) First off, clean any potatoe chips, pretzel pieces, etc from it.  If you have a can of compressed air, blow the keyboard from every angle.  If your hand strength is good, hold the unit upside down while doing this.  Next, try every key one at a time.  Listen for a different sound or feel.  If all good so far, rapidly and repeatedly press every key at least 10-15 times.  If you removed the battery, reinstall it.  Power up.  Report back.
