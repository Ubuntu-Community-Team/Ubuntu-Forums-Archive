### Post by lloyd_b on 2009-04-22

> **** **i increase the ram size 512 mb to 1gb..how to change the swap size?... and then why we put swap=2*ram**. any one know means reply me..


Suresh.

I would leave the swap where it is, unless you are actually running out of swap space.  Many systems can run with no swap space whatsoever, and work just fine.

The "Swap should equal 2 times RAM" rule is quite old, and obsolete for most modern machines.  Swap = RAM is quite sufficient for almost all users, unless you are doing a "suspend to disk", in which case you'd need more swap space.
