### Post by Johnsie on 2007-01-10
Both have their advantages and disadvantages... If you're happy with KDE stick with Kubuntu, if you want to try something different (Gnome) try Ubuntu. You can have both on your computer at the same time too:

1. Install ubuntu
2. sudo apt-get install kubuntu-desktop

then you can choose at login which window manager you want to use.
