### Post by jjpcexpert on 2010-10-31
You should be able to do so without formatting. Back up your data, and make sure to reinstall to the same partition that's borked that much. Your data should still be there if you enter your username and password as before, because the deleting of conflicting system files does not (normally) delete /home or /home/* .
