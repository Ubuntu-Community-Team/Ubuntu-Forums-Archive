### Post by dayvy on 2007-10-19
Sounds like it's having trouble figuring out the layout of your hard drive. I suggest booting with a live cd and go to a terminal window and seeing what fdisk has to say about things. 

i.e fdisk -l

Then use fdisk to wipe out any partitions that it sees ( assuming you don't have any data that you want saved). Reboot and try the install again.

If you've never used fdisk here's a primer:

fdisk /dev/sda (or hda or sdb or whatever your hard drive is).

Press m for a list of options. Make sure to write the changes to disk before rebooting.
