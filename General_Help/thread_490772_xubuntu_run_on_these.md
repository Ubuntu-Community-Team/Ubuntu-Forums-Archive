### Post by bodycoach2 on 2007-07-02
On the AMD Althon, not only will Xubuntu work fine, but you can probably get more RAM for that one, and install Ubuntu on it. I wouldn't install Ubuntu unless it's at 512mg ram. Xubuntu will run more than fast enough if you can get 256 ram on it, but 128 will be okay.

The AMD will seem to run faster than the Celeron. In fact, you might want to consider trying the 64bit version of Xubuntu on the AMD one.

On both machines I'd focus more on trying to get more memory installed. Not sure where you're from, but you can join Freecycle.org, and put out a Wanted: computer memory post.
