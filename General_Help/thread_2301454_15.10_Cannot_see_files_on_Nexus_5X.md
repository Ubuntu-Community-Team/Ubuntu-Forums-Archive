### Post by shochatd on 2015-11-02
I just got my USB Type C cable for my Nexus 5X and tried using it to connect to my computer. A window comes up labeled "Nexus 5X" and I can also see the device in the Launcher, as if it had been mounted. However, the window is empty.
Is there a way to access the files, or is the USB useless except for charging?
I do know that the "wifi file transfer" app can be used to access the files without USB, but I figure there must be some reason for the USB port other than charging. I have seen some others asking similar questions on other forums, but no answers.

---

### Post by shochatd on 2015-11-02
No sooner had I posted this than the answer arrived: A notification showed up on the phone labeled "USB for charging". I touched this and got a menu including a file access option. The menu and notification have now disappeared and I don't know how to get either back, but this seems to solve the problem. There is still no mention of any of this under Settings->Storage & USB, which seems like the logical place to look.
