### Post by meho_r on 2010-06-06
The problem is that players write ratings in their databases, not in the file itself, so I'm not sure if there's a direct way for your request. However, some people use workarounds like creating separate playlists for every rating, so when they import these playlists into another player, it is easy to apply new ratings to all files in a playlist.
