### Post by BertN45 on 2012-03-17
Depends on which versions you are running now. If you run Xubuntu 10.04, you could stick to it for another year. Else you could upgrade to 12.04 end of April, since 12.04 is a LTS release and it is maintained for the next 5 years.

The main question is of course if you have some problems with respect to e.g, special hw, like tv-tuners, scanners etc. If so it might have been solved in the 12.04 release. If you don't have any problems and if you are happy with the system why change?

### Post by kurt18947 on 2012-03-17
IMO there are two considerations.  One is the availability of security updates.  The other is that more recent distros may have built-in support for newer hardware.  For me, 11.04 brought plug -n- play capability to two wireless dongles that previously required 3rd party fixes.  And newer apps though this may or may not matter to you.  Beyond that, why mess with what works?
