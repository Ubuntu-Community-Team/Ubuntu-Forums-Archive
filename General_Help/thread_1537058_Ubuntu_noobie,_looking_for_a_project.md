### Post by watupgroupie on 2010-07-23
Well lots of themes over at [http://gnome-look.org](http://gnome-look.org) (just be careful of .deb files from there). Installing them is as easy as loading up the Appearance menu, hitting install and selecting your theme package. 

For things to do, I'd setup a good conky script you're happy with. Lots of sites around to help you with that if you do some searching, plus conky is a great desktop customization tool. Coding in Ubuntu is quite easy, things like python are very easy to get right into with little to no setup. Plus coding is something productive to learn :P. 

Basically just go exploring. Read a Ubuntu blog and you're bound to run into something that perks your interest.
