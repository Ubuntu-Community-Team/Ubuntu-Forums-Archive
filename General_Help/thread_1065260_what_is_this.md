### Post by bu2971x on 2009-02-09
Can someone make this more understandable in relation to dvd burning...what is re authoring..??


you'd need to burn them as the image files, not as data files, onto the CDs. The resulted CD should contain a .BIN file. A common mistake is to burn the firmware files as data files by re-authoring a new CD. The resulted CD would contain a .ISO file, which cannot be recognized by the player. For more information about the firmware upgrade process and how to correctly burn the firmware CD, please refer to the firmware upgrade instructions.

### Post by photon_man62 on 2009-02-09
In this they meant that you should OPEN the .iso file in a program and burn the files which are compressed INSIDE the .iso file, not burning the .iso file itself onto the disc.

Let's say you compressed

-VIDEO1.mpg
-PICTURE1.jpg
-PICTURE2.jpg

into an ISO file. Let's call it COMPRESSED.iso

So you want to have

-VIDEO1.mpg
-PICTURE1.jpg
-PICTURE2.jpg

on the disc, not a disc with the file

-COMPRESSED.iso

Got it?
