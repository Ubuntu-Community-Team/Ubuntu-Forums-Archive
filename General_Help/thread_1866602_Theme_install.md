### Post by gsmanners on 2011-10-21
Hi. :D

Create a folder in your "home" folder called ".themes" or just do the following in a terminal:

```
mkdir ~/.themes
```

Then copy your extracted theme into that folder. Make sure you enable "Show Hidden Files" as the directory will otherwise be invisible.
