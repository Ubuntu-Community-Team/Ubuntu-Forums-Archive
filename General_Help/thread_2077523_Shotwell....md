Sorry to hear that.  The thing to understand here is that Shotwell is non-destructive; it's intended to never damage your original photos (like iPhoto, Lightroom, etc.)

If you want a copy of the photo with all the edits in place, you have to select one or more photos and do File->Export.  Shotwell will then write out the photos to the path you specify.

Note that it will *not* import these photos into the Shotwell database.  If you'd like to do this, you can then import the new files by selecting File->Import in Shotwell.
