### Post by yancek on 2018-01-21
More information is needed for someone to help so go to the site below while booted to your Ubuntu install media.  Select the 2nd option to get the boot repair software.  After it is downloaded, you need to run it and it will output a link at the end which you can post here with details needed.  Make sure you select the option to Create BootInfo Summary and do NOT try to make any repairs.
