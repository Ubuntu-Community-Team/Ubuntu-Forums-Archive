### Post by danamo on 2019-06-25
I installed Linux (Ubuntu) for the very first time.  It's on a Toshiba laptop. Everything works fine except when I connect the VGA output to my TV the TV only shows the desktop background wallpaper. No icons, favorites bar, the time, or any open windows are shown on the TV, just the desktop background picture itself.

The laptop screen shows everything normally. When this laptop had Windows 7 I viewed it on this TV with no problems.

Any suggestions about this problem?

### Post by uRock on 2019-06-25
Hello and welcome to the forums.

What you're seeing is the default way the dock is displayed, so it adds more screen for whatever windows you move to the secondary monitor.

This can be change by opening **Settings**, select **Dock**, then you'll see where you can select which or both monitors to show the dock on.
