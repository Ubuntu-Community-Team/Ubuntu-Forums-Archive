### Post by Frogs Hair on 2014-02-02
All security updates come via the update manager since the Linux version of flash is no longer being developed. There were no new versions for Flash Aid to install, so no reason to maintain the extension. 11.2xxx  with security updates is the last version. The only new versions for Linux come via Chrome which includes Pepper Flash. Pepper will be available with Chromium in 14.04 and there is an un-official installation method currently.
