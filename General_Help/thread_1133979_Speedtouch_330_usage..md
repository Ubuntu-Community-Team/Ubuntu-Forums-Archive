### Post by _zeb_ on 2009-05-05
Go here:   [http://steve-parker.org/speedtouchconf/](http://steve-parker.org/speedtouchconf/)

Use the same microcode that worked in a previous distro and, while I'm figuring out how to make the "get online at boot time" work, run "sudo speedtouchconf.sh" each time you want to get online (entering VPI/VCI, login details, etc).  It's not perfect, but it works
