When I close my laptop lid then open it after a while, over 50% of the time if I have a web page displayed when the lid is closed, a new web page will be displayed when I log back in. I removed the mouse USB dongle prior to closing the lid this last time and when I logged back in, I still had a new web page open from when I closed the lid. ### Post by yetimon_64 on 2019-03-24
> The new pages appear to be from links on the web page on the browser tab that was open when the lid was closed. Check under System Settings > Power management > both tabs for "On AC Power" and "On Battery Power", what the setting for "When laptop lid is closed" is set to.

 If it is set to "do nothing" then closing a touch screen with a browser tab open with links on it may be accidentally activating a link on the page.
Try to set the "When laptop lid is closed" setting on both tabs to "blank screen" and see if that helps your situation at all.

Regards, yeti.
