### Post by HereInOz on 2010-05-07
There is a skype version for Ubuntu - check out the Skype site

Adobe Creative Suite CS4 - you will need the latest version of Crossover to run that.  It is Windows and Mac only, and you need Crossover to have any hope of running it, provided the lads at Codeweavers support it yet.

Yahoo messenger - you can use Empathy IM client for that.

The others, I am not familiar with.

Suffice to say that programs designed for Windows do not run on Linux or Unix, and programs designed for Linux or Unix do not run on Windows.  Some programs, like GIMP, have both Windows and Linux versions, and this is a good thing, and some Windows programs can be run on Linux using Wine, or its Crossover development.

Start playing with Wine and see what you can do, but for CS4, you will need to check with Codeweavers to see if Crossover will run it.
