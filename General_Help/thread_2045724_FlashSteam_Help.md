### Post by vandorjw on 2012-08-21
Installing flash-plugin for Ubuntu will not help you.

You need to download 

[http://get.adobe.com/flashplayer/otherversions/](http://get.adobe.com/flashplayer/otherversions/)

from there.

Select windows, then pick other (not the one for IE)

Install via wine.

Cheers
