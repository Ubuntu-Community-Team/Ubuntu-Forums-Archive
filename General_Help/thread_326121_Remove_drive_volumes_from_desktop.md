### Post by Vord on 2006-12-27
So, I'm using Edgy with Gnome, and everytime any drive is mounted, it shows up on my desktop. CD drive, DVD drive, iPod, NTFS drive, they all show up on my desktop, I need to know how to configure it to where they won't

---

### Post by mcduck on 2006-12-27
Open Configuration Editor (Alt-F2 and run gconf-editor), browse to apps/nautilus/desktopand disable 'volumes_visible'. No more drive icons on desktop. :)
