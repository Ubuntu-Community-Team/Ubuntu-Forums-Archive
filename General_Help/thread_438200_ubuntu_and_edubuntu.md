### Post by dfreer on 2007-05-09
no difference whatsoever other than the packages installed by default. Any package available for edubuntu is available from the same repositories in ubuntu, and vice versa. so you can install ubuntu-desktop and you will be able to login to the exact same enviroment as from installing a ubuntu live CD.

EDIT: same goes for kubuntu, xubuntu, *ubuntu

### Post by dfreer on 2007-05-09
hmmm well cosmetic reasons aside. It should use the same repos (/etc/apt/sources.list) as ubuntu, which means that any package on the one can be installed on the other. so basically if you are wanting to install a program that says it is for ubuntu it will work in edubuntu. Any guide for ubuntu should work for edubuntu.

there is a server install available for *ubuntu as well.
