### Post by snowpine on 2009-01-21
Hi Navillus,
Technically you can't have a command-line only Xubuntu install, because the X in Xubuntu stands for the Xfce desktop environment. :)

All joking aside, what you probably want is to use the Ubuntu mini.iso as described here: [https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

Once you have the minimal system installed, you can add whichever packages you like. It's a lot of fun! Good luck.

(edit) A good tutorial, with some tips if you want to take it to the next level and add a lightweight windows manager: [http://www.psychocats.net/ubuntu/minimal](http://www.psychocats.net/ubuntu/minimal)
