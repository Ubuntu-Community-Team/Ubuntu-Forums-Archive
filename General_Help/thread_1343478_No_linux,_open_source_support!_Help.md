### Post by tgalati4 on 2009-12-01
Try to get the model number of the clearwire modem.  I bet it runs linux.

You probably only need firefox with the "User Agent" plug-in and set it to Internet Explorer 7".  Connect to your modem with a cat5 cable and go to it's address:  (probably [http://192.168.1.1](http://192.168.1.1)) and when the settings page comes up, tweak it to your needs.  Most modems need to be authenticated by typing a serial number or some other code into them to wake them up.
