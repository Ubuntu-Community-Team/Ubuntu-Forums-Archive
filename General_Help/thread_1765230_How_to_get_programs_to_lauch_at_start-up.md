### Post by hulkman on 2011-05-22
1. Open System;Preferences;Startup Applications.
2. Click "Add"
3. Type in what ever name you want in the "Name:" box. (Probably the name of the program would be nice.)
4. Click browse and navigate to the application you want to start and double click it.
5. Click "Add"  (This window should now close.)
6. Click "Close"
7. Restart and see if the program starts for you.

----------------------
[http://www.dotdeb.com](http://www.dotdeb.com) is a great place to find deb packages.

### Post by Larkspur on 2011-05-22
Programs are usually started by their names in lower case: "pidgin" will start Pidgin, for example.  To test, try typing the name into the Terminal and see if the program launches.  Otherwise, if you open Main Menu, go to the program's entry there and click Properties, you'll find the command to use.
