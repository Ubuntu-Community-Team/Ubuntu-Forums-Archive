### Post by seawolf167 on 2011-09-26
IMO the easiest way would be to make a bootable Ubuntu USB that has persistence with [UNetbootin]("http://unetbootin.sourceforge.net/"), then install any programs you want when booting into it (they'll be saved - just ensure you add enough persistence space).
