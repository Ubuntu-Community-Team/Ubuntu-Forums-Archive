### Post by Cavsfan on 2010-09-09
When you type "gconf-editor" it opens up Configuration Editor where a lot of stuff is controlled.
If you use it much or would just like to have easy access to it, you can add this to **Applications** > **System Tools**.
Just click on **System** > **Preferences** > **Main Menu** and then click on **System Tools** in the left panel and put a check mark 
by **Configuration Editor**.

I know this is not rocket science, but I couldn't remember what to type into the terminal to get it. So, I like having it there.

Just an idea. :D
