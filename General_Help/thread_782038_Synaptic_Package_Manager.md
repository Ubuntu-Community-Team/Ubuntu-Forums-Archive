### Post by chrisccoulson on 2008-05-04
And once it is in to Intrepid, you *could* try to backport it yourself (or someone else may have a go for you). Have a look at [**_Prevu_**]("https://wiki.ubuntu.com/Prevu").

Also keep an eye on [**_http://www.getdeb.net/_**]("http://www.getdeb.net/") for up-to-date Hardy packages.

It could also end up in someones PPA on Launchpad as well, so there's still hope yet [**_https://launchpad.net/ubuntu/+ppas_**]("https://launchpad.net/ubuntu/+ppas")

### Post by forrestcupp on 2008-05-04
Ubuntu is not a rolling distribution.  That means that once a version is released, nothing gets updated to newer versions except for bug fixes and security updates.

So you'll either have to wait for 8.10, or like already said, hope it gets into either the backports or proposed updates.  Or you could break down and compile.

But honestly, does it have enough added features for it to be that important?
