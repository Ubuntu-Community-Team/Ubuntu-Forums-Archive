### Post by oldfred on 2015-07-19
I have two core2 Duo systems.

Old laptop has 1.5GB of RAM and I do run 64 bit w/fallback for gui on it, just so it is same as desktop. But if I try to load two larger apps at once like Firefox & Thunderbird, or LibreOffice, screen turns grey for a couple of seconds, so I know it is using swap. My normal use with just Firefox, terminal & Zim or text editor works ok. But I shut down Firefox usually if I want to use Thunderbird or LibreOffice and then they work ok. Sometimes a bit slow loading as not cached in RAM.

Desktop with similar Intel chip as yours and 4 GB of RAM using 64 bit, and a nVidia card did run Unity ok, but I preferred fallback or the old menu system. It almost never uses swap. And after adding a SSD in 2011 made me put off new system until end of last year as it ran too well for me to justify new system to myself much less my spouse. :)
