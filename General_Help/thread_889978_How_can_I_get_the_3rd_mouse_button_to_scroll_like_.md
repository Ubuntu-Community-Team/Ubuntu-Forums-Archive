### Post by cat5e_ on 2008-08-14
Hi. In Windows when you hold the third mouse button (usually in middle), it brings up this little arrow on the screen that lets you control scrolling by just moving the mouse up and down. Well, I have an old laptop without any scroll features other than a button that you can click in. I know the button is working with Ubuntu because when I click a link with it in FireFox, it will open a new tab or if I click a tab, it will close it. But when I click in away from those things, it does nothing.

So how can I add that feature of where if I click it on a blank space that it will put the scroll cursor up? 

I appreciate any help.

---

### Post by Too Late on 2008-08-14
The feature is called autoscrolling. In Firefox, go to Edit -> Preferences -> Advanced -> General (tab), and check the "Use autoscrolling" button.

Of course, this has only effect in Firefox. As far as I know, there's no "global setting", which would allow you to use this kind of feature in any application, eg. in Nautilus.
