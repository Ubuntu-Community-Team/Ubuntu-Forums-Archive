### Post by Socialsymbol on 2010-04-11
I wasn't entirely sure where to post this, so apologies if there was a better section to go to.

Anyway, to me Rhythmbox is a great application for playing music, and it's getting even better as the iPhone/iTouch support for it grows. It's conservative on the hardware usage and well featured.

The only thing about Rhythmbox that *really* bugs me is that if I change a song's information (title, artist, album) to organize it and then close Rhythmbox later, the next time I open Rhythmbox that song's information has changed back to the previous info.
([http://boonce.org/up/7752_Screenshot.jpg](http://boonce.org/up/7752_Screenshot.jpg))

At the moment I'm not sure how to get around this, although I'm sure there's a way (I haven't looked too hard as of yet). However I still think it would be good for there to be an update where changing the information of a song actually changes the info of the actual file, seeing as how currently in Rhythmbox if you delete music from the application it deletes the song itself from your music folder, which I think is great for weeding out songs you don't want anymore.

Anyway, all I'm saying is I'm surprised that such a great application has such an obvious drawback, albeit small, yet annoying. Would make for a great update.
