### Post by dog_biscuits on 2009-12-21
I've been using the UNR 9.10 on an Asus Eee 1000HE for over a month now and find it excellent. Much quicker than my wifes running Win XP... ;)

On the wiki about UNR ([https://wiki.ubuntu.com/HardwareSupport/Machines/Netbooks#Asus%20Eee%201000HE](https://wiki.ubuntu.com/HardwareSupport/Machines/Netbooks#Asus%20Eee%201000HE)) and compatibility, these problems were for a previous release and weren't a problem when I installed 9.10.

The main difference I've found between UNR and Regular Ubuntu is the user interface.  As the UNR is designed to be used on a smaller screen it's a lot easier to use.

I have had one problem (only today) which was after applying the latest updates I couldn't connect to wireless networks that used WPA2 encryption.  However I found the fix to that here [http://ubuntuforums.org/showthread.php?p=8537237](http://ubuntuforums.org/showthread.php?p=8537237) and everything is now working fine again.

Hope this is helpful.
