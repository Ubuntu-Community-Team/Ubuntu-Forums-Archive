### Post by Reaper829 on 2009-04-28
The Stand-Alone version is for computers that aren't already running their own web server (such as  apache, tomcat).  It's an all-inclusive package that includes everything necessary to get up and running.

The WAR version is for computers that already have a web server installed.  The WAR version requires an already-working apache server, and it needs tomcat (a java-based program that runs with apache) already installed as well.  

Already having a server, I installed the WAR version following the instructions from subsonic's main site.  I didn't already have apache or tomcat running, so I set them up quick.  No harm in planning for the future!
