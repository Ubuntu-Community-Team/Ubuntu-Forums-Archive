### Post by C.S.Cameron on 2008-10-09
A full install of Xubuntu will work on a 2G flash drive, (not much room left over).
An image install of ubuntu using unetbootin, usb-creator or liveusb will take up less than 700M, but will use extra space if you require room for persistence, (home -rw, casper -rw folders or partitions).
Puppy, DSL, etc are fine for computers with low memory, but Xubuntu should work fine if you have a gig or more.

Instructions for DSL? flash install can be found here:
[http://damnsmalllinux.org/wiki/index.php/Installing_to_a_USB_Flash_Drive](http://damnsmalllinux.org/wiki/index.php/Installing_to_a_USB_Flash_Drive)
