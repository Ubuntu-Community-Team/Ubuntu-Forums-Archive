### Post by braddcadd on 2008-07-31
Well a *.bat file is an old DOS scripting file.  The only way to run that would be in a DOS emulator (I'm sure there is one in the repositories) or in actual DOS.  

If you have jave installed, run this in a terminal:
```

java -jar jarfilename.ja

```
