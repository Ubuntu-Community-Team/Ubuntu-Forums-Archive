### Post by cymbaline42 on 2011-05-01
Download Firefox 4 from the official website ([https://www.mozilla.com/en-US/firefox/fx/](https://www.mozilla.com/en-US/firefox/fx/))

Extract it to any folder you like, open Terminal, go to the installed folder and run it by typing "./firefox".  You can create a shortcut on the panel too.
