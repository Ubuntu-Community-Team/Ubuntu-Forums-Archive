### Post by schilcha on 2006-02-27
If you're aiming at an xscreensaver-screensaver, just start the screensaver from the command line, eg
```

/usr/lib/xscreensaver/fuzzyflakes

```
It'll start in a (resizeable) window. Take a screenshot from that window as you always do.

Good Luck!

### Post by aysiu on 2006-02-27
You can run the live CD using Qemu and run a screensaver there.
I think the syntax is something like: ```
qemu --cdrom /dev/cdrom
``` You can do ```
man qemu
``` to get the exact command.

Qemu's in Synaptic.
