### Post by ghostborg on 2011-06-07
I had some weirdness going on about a week ago and my system slowed down and was acting like my hard drive was going.
I would restart and or shutdown and the system would work for a while then slow down and some programs appeared to lock up.
Looking through various pages of tips I tried:
```
sudo touch /forcefsck

```
which forces a one time file system check the next time you boot.
Took a while but after running this or some other coincidence my system returned to normal. I thought for sure I was replacing my drive. For a moment I had a flashback and thought maybe I was running Windows. haha.
