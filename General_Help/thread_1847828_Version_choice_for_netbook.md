### Post by kevin2849 on 2011-09-22
I have tried several and settled on Lucid Lynx netbook addition. I removed the things I did not need and stopped any service I did not want running. I have 2 gig RAM in mine and it runs fine.

All in all it is a usable system without (at least so far) any show stopping faults.

You should have no problems with this system.
