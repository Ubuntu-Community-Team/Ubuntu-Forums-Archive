### Post by takisan on 2010-11-26
Apple tends to have a proprietary file organizing system for iPods. I know not how it works, only that it does keep original .mp3 files on it. Just putting the files in the root of the drive doesn't work to play them. There might be an i-pod syncing program in the repos, or install wine and itunes.
