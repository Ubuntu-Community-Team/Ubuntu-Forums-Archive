### Post by John Bean on 2009-10-20
I have a Advent 4211 (Wind u100 clone) that came with XP. I used Gparted on a live CD of Jaunty (Ubuntu 9.04) to shrink the XP partition and installed it. Works fine, no issues whatsoever. 

I had tried the live netbook remix version but for whatever reason it was actually much slower (and clunkier) than the full version on this particular machine.

The only required adjustment I made to the default setup was to allow windows to be moved up past the top of the screen to allow proper access to the bottom of dialogs that are longer than 600px, but other than that it "just worked" after a default installation.

Edit: I suspect the slowness of the remix version is down to the heavier demands on the not-clever Intel graphics system. Either  way the standard Ubuntu is solid as a rock and very responsive on this little machine, I'm very happy with it.

### Post by snowpine on 2009-10-21
Hi mjp29, All you need is a USB thumb drive (or SD card if the computer is capable of booting from one). You can install Ubuntu directly to the thumb drive/SD card, allowing you to boot Ubuntu on the netbook without making any permanent changes.

I would personally recommend "regular" Ubuntu 9.04 Jaunty Jackalope because 1) I don't like the netbook remix interface (it is slow and confusing, just my opinion), and 2) 9.10 Karmic Koala is still in beta, hasn't been released yet.

I've run many different Ubuntu releases (and other distros) on my two netbooks (Dell Mini 9 and Asus eee 900HA). Netbook hardware is definitely up to the challenge (so long as you don't expect great gaming/multimedia/video).

Good luck!
