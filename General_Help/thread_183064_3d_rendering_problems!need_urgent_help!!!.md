### Post by Zimmer on 2006-05-27
[https://wiki.ubuntu.com/BinaryDriverHowto/ATI](https://wiki.ubuntu.com/BinaryDriverHowto/ATI)

Did you follow the instructions and read the notes on the above wiki?

I am curious that you say that you have a 9550 yet your xorg.conf is set to identify it as a 9600... it may have no effect but it is just an observation...

Read the wiki carefully, particularly the notes ...for example...
 If you are having problems related to DRI or 3d acceleration and the following lines show up in your /var/log/Xorg.0.log

(WW) fglrx(0): Kernel Module version does *not* match driver.
(EE) fglrx(0): incompatible kernel module detected - HW accelerated OpenGL will not work

then make sure you installed either linux-$arch or ubuntu-fglrx-$arch.
