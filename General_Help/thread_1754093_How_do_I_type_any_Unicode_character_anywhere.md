### Post by rockinforgod on 2011-05-09
I frequently use the symbol for emptyset ([http://en.wikipedia.org/wiki/empty_set](http://en.wikipedia.org/wiki/empty_set)), and would like to know how to get this directly from the keyboard.

In OpenOffice Writer, I can insert it via the Insert Special Character dialog. It is there noted as U+00D8 (216), and in Windows I can get it by typing Alt+Numpad 0216. 

**Is there a similar shortcut to insert Unicode characters by their codes in Ubuntu?**

I'm using Ubuntu 10.10, with mostly deafault config settings. (I'm pretty new at this...)

Some readers might notice that the symbol I use for the empty set is very similar to the Danish O-slash, and for my purposes they are indeed interchangeable.

---

### Post by VCoolio on 2011-05-09
try ctrl+shift+u<type unicode code here>
so together ctrl+shift+u, release, then 00D8, then space or enter.

For others seeing this thread, note that it is the hex code, and not the decimal code, that is to be used. In my case, Ø is spelled [ctrl]+[shift]+[u],[d],[8] and not [ctrl]+[shift]+[u],[2],[1],[6].
