### Post by yatt on 2006-03-26
[QUOTE=Califer]Can I upgrade my Breezy install to Dapper without losing any of my files or installed programs?[/QUOTE]
Open Synaptic

Settings

Repositories

Edit each one on the list and change breezy to dapper.

Reload

Mark All Upgrades

Apply
