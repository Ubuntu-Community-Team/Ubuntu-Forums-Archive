### Post by malachi on 2008-11-21
Go to Applications > Add/Remove

Search for 'advanced compiz settings.' You'll be able to install the advance settings there. However, you'll probably need an internet connection.

For installing packages without an internet connection, see this page:

[https://help.ubuntu.com/7.10/add-applications/C/offline.html](https://help.ubuntu.com/7.10/add-applications/C/offline.html)

### Post by philinux on 2008-11-21
