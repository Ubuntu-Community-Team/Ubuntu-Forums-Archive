### Post by Mituness on 2007-04-24
Ahh, thanks so much for your reply! I was a little confused as first as I thought that the i810 drivers *were* the ones I was supposed to have, but some searching around proved that I needed to install the specific intel ones. :)

For anyone else for future reference, I just did:

```
sudo apt-get install xserver-xorg-video-intel
```

and then changed my /etc/X11/xorg.conf to list:

> 
Section "Device"
        Identifier      "Intel Corporation Mobile 945GM/GMS/940GML Express Integrated Graphics Controller"
        Driver          **"intel"**
        BusID           "PCI:0:2:0"
EndSection


instead of **"i810"**. I then just restarted X using Ctrl-Alt-Backspace and all was working!

I can now run things above 800x600 resolution. Thanks!
