### Post by seawolf167 on 2011-12-19
Start with some basics by cleaning the dust from the fans, heatsinks, vents, PCI cards and adapters. Is your computer by any heat vents, registers, etc.? If so, move it if you can.

If the computer is still hot, check your fan speed settings (either a manual switch if any of them have any) or through BIOS if you can control them there. Change the settings to high, or ensure they are already on high.

If the computer is still hot, how hot is it when idling in degC? What is it peaking at?
