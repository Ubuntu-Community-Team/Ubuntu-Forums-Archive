First do [this]("http://members.iinet.net.au/~herman546/p18.html#MbrFix.exe") and after that you can remove ubuntu partition with Ubuntu live CD or with [Gparted live CD.]("http://gparted.sourceforge.net/") Then you can add free space to Windows or make another NTFS partition.
