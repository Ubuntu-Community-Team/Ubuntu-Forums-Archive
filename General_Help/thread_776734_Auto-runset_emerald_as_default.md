### Post by VictorR on 2008-05-01
It should work if you:
1. launch CompizConfig Setting Manager
2. tick Enable Window Decorations on
3. type 'emerald --replace' (without quotes) in the Command field
4. type 'any' (without quotes) in Decoration Windows (two lines below)
This should be enough.
