### Post by blueturtl on 2007-12-17
It is possible, but for you to be able to run Ubuntu from within Windows you must install it into a virtual machine. One free solution to your problem is the [VirtualBox]("http://www.virtualbox.org/") virtual machine manager. Do note that you need more power to be able to run two operating systems at the same time. Also some of the more advanced features of Ubuntu like the 3D enhanced desktop most likely will not work inside a virtual machine.
