### Post by MJWitter on 2011-05-11
Not exactly what you are looking for, but Deluge has a label add on which can move completed torrent files to specific folders.  So when you add torrents, label them and they will be moved automatically.  Also, if certain trackers are used only for certain files, eg an iso tracker, then it can automatically label anything from that tracker for you.
