### Post by prshah on 2009-02-12
> **swordfishRU said:**
> 
It's possible to import a web-based table into a spreadsheet application (Gnumeric or OO Calc), likewise MS Excel?


Yes, of course, but it depends on the web page in question.

Usually, copy and paste works just fine. Calc is almost as good as Excel at recognizing table formatting.

Further, opening a web address in the File-Open dialog will automatically open the webpage in Writer (Word) and tables will be recognized and formatted as tables.
