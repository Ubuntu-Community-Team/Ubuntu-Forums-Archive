### Post by akkiraju on 2008-06-22
See that your compiz manager is added in your sessions.

system-> preferences -> sessions


If you dont see compiz manager there

 Click Add and make an entry for compiz

  Name: compiz

  Command: compiz --replace

save , logout and relogin into ubuntu

And see if you have total compiz packages installed.

You can do that by using system-> Adminsitration->synaptic manager and searching for compiz

Re installing compiz via synaptic might help as all dependency pkgs will be installed that way
