### Post by FluffyElmo on 2009-02-24
There are a *lot* of text editors for Ubuntu and you are sure to find something you like. I'd recommend starting with GEdit which is the default text editor, there are a lot of plug-ins available which can make it do just about anything you'd want. 

Here is a guide to configuring it for web development but it gives you an idea on how to find and install plug-ins... [http://www.micahcarrick.com/09-29-2007/gedit-html-editor.html]("http://www.micahcarrick.com/09-29-2007/gedit-html-editor.html")

I've also tried and liked Scribes, but tend to come back to GEdit. [http://scribes.sourceforge.net/]("http://scribes.sourceforge.net/")
