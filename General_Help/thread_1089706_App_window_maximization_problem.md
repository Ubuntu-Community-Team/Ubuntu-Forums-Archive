### Post by Rallg on 2009-05-17
My original question was solved by some update or other. But the first response raised a different question, for which there is a solution:

If an application launches a dialog box (window) that is too long to fit the mini 9 screen, do this: Hold down the ALT key, then left-click (and hold) somehwere in an empty area of the window. You can then move the window, and you can move it off the top of the screen, so that its bottom will appear.

Apparently, this only works if the main Ubuntu panel (the horizontal bar with the applications launcher) is at the top of the screen. If you have it at the bottom or side, then any attmpt to move a dialog box will stop at the top of the screen.
