With reference to Slashdot article [http://it.slashdot.org/it/06/03/13/0525254.shtml](http://it.slashdot.org/it/06/03/13/0525254.shtml) a bug has been found in Ubuntu Breezy Badger (5.10) which may leave the system vunerable to attack. There is more information here: [http://www.ubuntuforums.org/showthread.php?t=143334](http://www.ubuntuforums.org/showthread.php?t=143334) and here: [https://launchpad.net/distros/ubuntu/+source/shadow/+bug/34606](https://launchpad.net/distros/ubuntu/+source/shadow/+bug/34606)

I have posted to the main section rather than the security section as it is more likely that a greater number of users will see this message.
