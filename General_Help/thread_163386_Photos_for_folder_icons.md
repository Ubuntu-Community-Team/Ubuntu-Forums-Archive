### Post by bjornkri on 2006-04-23
I found a way!

Here's how it's done:
Enter a folder full of photos. 
Pick one to use as the folder icon.
Make sure the sidebar is on (F9).
Make sure the sidebar is set to 'Information'.
Now simply drag the picture to the folder icon in the sidebar.
Voilá! :)

### Post by ncappel1 on 2006-04-23
you can also change the default icons for any thing by going into /usr/share/icons/folder_of_your_icon_theme/mimetypes and replaceing icons with your own custom ones. 

be sure to backup the directory so you can change it back, and also remember to name the new icon to the **exact** name of the icon you want to replace. To see the changes you will have to restart x, either by refreshing it or logging on again.
