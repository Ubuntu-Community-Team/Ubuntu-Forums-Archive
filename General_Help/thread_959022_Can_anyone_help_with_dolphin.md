### Post by Denestria on 2008-10-26
The first option in the Settings > Configure Dolphin dialog is default view mode = icon/preview/details.  Show hidden files can be saved if you check the box to remember settings for each folder.  View > Adjust View doesn't work properly for settings hidden files in all folders, but is supposed to be fixed for Ibex being released next week.
