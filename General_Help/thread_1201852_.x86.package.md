### Post by aquavitae on 2009-07-01
If you install it from the repos, you should be able to copy the downloaded file off your computer anyway - I think they get saved somewhere under /var/cache. Alternatively, download it from [http://packages.ubuntu.com]("http://packages.ubuntu.com"), which is basically online access to the repos. That way you know you're downloading a file that has been tested with ubuntu.

EDIT: I've just done a bit of googling and it seems that the x86.package file is meant to be a sort of global installer.  However, if you have a closer look at the installer (autopackage) website, it seems it doesn't work too well with apt yet.  So unless you have a very specific reason for wanting to use it, I'd suggest you stick to the repos. Anyway, its not a particularly large file, only about 3Mb, so even with low bandwidth you should be able to download it again if you need it
