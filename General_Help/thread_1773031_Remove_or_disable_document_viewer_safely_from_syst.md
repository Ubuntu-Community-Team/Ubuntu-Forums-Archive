### Post by linuxinstalledfromhdd on 2011-06-01
It's not going to remove your entire system. I just checked it and did it on my own system. It is only going to remove ubuntu-desktop, which is just a package that tells a system to install the rest of the "default" Ubuntu desktop applications on your system. It's hard to explain, but you should be ok.  Don't worry.
