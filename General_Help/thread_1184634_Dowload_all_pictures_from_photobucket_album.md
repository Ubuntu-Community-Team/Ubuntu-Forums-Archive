### Post by Legace on 2009-06-11
Maybe you could use [https://addons.mozilla.org/en-US/firefox/addon/3404](https://addons.mozilla.org/en-US/firefox/addon/3404)
You have to download it from here for Linux: [https://addons.mozilla.org/en-US/firefox/addons/versions/3404](https://addons.mozilla.org/en-US/firefox/addons/versions/3404)

Or alternatively:
    * Press CTRL + I while in the Photobucket album
    * Click on it and you will see &#8220;Manage Information&#8221; Button popping up. Click on it
    * Next You will see another popup with lot of options. Select the Media Option.
    * In this tab you will see all the images and other media files listed which is availble in that web page.
    * Select one image and press CTRL + A ( Select All )
    * Then Click on the Save As button. This will ask you to specify a folder where it will save all the media and image files.
