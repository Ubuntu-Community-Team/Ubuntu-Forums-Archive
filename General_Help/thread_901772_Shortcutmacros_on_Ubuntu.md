### Post by todak on 2008-08-26
Got to **Sytstem> Preferences> Keyboard Shortcuts**. Under the **Desktop** heading there is an action named **Launch web browser**. Click on the **Shortcut** setting next to that and choose your desired keystrokes! :)

### Post by mssever on 2008-08-26
If you use Compiz, there's a place in the Advanced Desktop Settings window (you need to have compizcomfig-settings-manager installed). If you're using Metacity, open gconf-editor and see /apps/metacity/global_keybindings and /apps/metacity/keybinding_commands.

EDIT: You might also like Gnome Do.
