### Post by doas777 on 2010-05-11
yep, mono is part of the default install. please be aware however, mono will not allow you to run a .exe compiled in .net. to get that, you have to use wine. mono just allows C# use on linux, so if you are writing code for linux, then you are golden, but if you are trying to run a windows forms app, then you will need more than just mono.
