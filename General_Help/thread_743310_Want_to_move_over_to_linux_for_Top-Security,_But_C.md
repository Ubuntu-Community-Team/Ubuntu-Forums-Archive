### Post by aysiu on 2008-04-02
I don't think it quite works that way. If the Windows computer is connected to the internet at all, it's connected and would be vulnerable to the same exploits it would ordinarily be subject to, whether connected through Linux or directly.

If it's possible, run Windows with a limited user account instead of an administrator account. I know some Windows programs (maybe your games) require you to run as administrator. If that's the case, make sure your Firewall is up, and that you're not surfing sketchy sites or clicking links in emails.

Yes, in Linux-based operating systems (Kubuntu, for example), the drivers are all included in the operating system kernel. In rare cases for which there are not appropriate open source drivers, you have to manually install drivers (Nvidia / ATI, for example).
