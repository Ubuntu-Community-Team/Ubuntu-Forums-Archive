### Post by 0pul3nce on 2009-06-28
Here is something I've done on my computer and i suggest others use this if they are new to ubuntu. Basically this makes ubuntu help startup when users log in.


Systems > Startup Applications > Startup Programs (tab)

Select Add

In the _n_ame box i placed : Help
In the co_m_mand box i placed : yelp

Contrary to popular belief the terminal command for this program is not help, because that command refers to terminal commands, not ubuntu's in depths help
Terminal commands for programs can be discovered by using properties on the applications, for example firefox is firefox %u in the terminal...
