### Post by 3Miro on 2010-11-05
Older ATI cards are often problematic. The only advice that I can give is to try with and without the proprietary drivers (form the Hardware Center), I had an ATI 4850 that worked better with he default driver. You can also try to disable the desktop effects, you can do it either from the menu (System -> Prefs -> Appearance) or you can install the Fusion Icon applet that will stay as a tray icon and you can right-click Windows Manager -> Compiz/Metacity.
