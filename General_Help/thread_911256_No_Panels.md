### Post by gjoellee on 2008-09-05
You must have a panel (is far as i know), but you can add and remove applications from it. Simply right click on the panel/or application on the panel for more info:)


EDIT: I know one way to get a desktop without panels, but I don't know if the dock will work. But you can try....
1. Applications->Accessories->Terminal
2. Add the command ```
sudo apt-get install openbox
```
3. Log out
4. On the logout screen click on "Sessions" and click on "openbox" (Not openbox-kde or openbox/gnome)
5. Log in
6. You should now have entered openbox (NOTE: You launch apps using Right click and then use the menu that comes up.

Hope it helps=)
