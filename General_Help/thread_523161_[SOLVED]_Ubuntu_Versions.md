In theory you will not have to reinstall gutsy in a couple of months if you try gutsy now. However I strongly recommend you do reinstall gutsy when it is released since theory might not be what is actually in practice. 

As far as upgrading is concerned, the update manager will ask you if you want to upgrade to gutsy when gutsy is officially released. During that upgrade you should not lose any settings.
