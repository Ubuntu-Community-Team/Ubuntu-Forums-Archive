### Post by LtPitt on 2007-03-20
Hopefully we can do better :D

Write in the address bar about:config and then hit enter
In this page look for browser.urlbar.clickSelectsAll and set it (with a double click) to true.

Horray! Firefox *non*developer usual behavior is back =)
