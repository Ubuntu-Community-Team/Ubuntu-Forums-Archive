I've just recently installed Kubuntu Edgy.

When using Konquerer to copy a bunch of directories and files to a destination where some of the same directories and files already exist, I get a message asking me how I want to manage these existing files.  I have the option of Skip, Auto Skip, Overwrite or Overwrite All.

Now, when I see 'Auto Skip' I get all excited because I start to think that maybe, just maybe I've finally found a file manager that is intelligent enough to allow me to only overwrite files if the source file is newer than the destination one.  But after looking in the Konquerer handbook, these forums and even Google, I can't find anything that actually explains to me what Auto Skip does, let alone if I might be able to customise it.

So can someone tell me what this feature is supposed to do?

Furthermore, it's unclear to me how konqueror will handle overwriting directories.  At least in Windows I would get a message saying something like "files in the destination folders with the same name will be overwritten".  The dialog in Konquerer gives me no idea of whether it will do what windows does or if it will simply replace the existing directory and all its contents with the source directory (rather than combining the contents of both).

### Post by Lil_Eagle on 2007-02-02
Autoskip is another way of saying "Say no to all".

In otherwords, if files already exist, they will not be overwritten.

A very handy feature because it allows you do for example, copy your home directory to a usb stick, and you know only things that are not already copied will be copied.
