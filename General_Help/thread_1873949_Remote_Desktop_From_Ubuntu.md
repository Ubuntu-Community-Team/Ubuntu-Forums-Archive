### Post by Lars Noodén on 2011-11-02
There are several packages that will provide [VNC](https://help.ubuntu.com/community/VNC) for you.  tightvncserver and x11vnc are two to start with.  VNC is not so secure, so it has to be tunneled over SSL or SSH.  The latter is easier.
