### Post by dino99 on 2010-05-06
you have installed grub2 on your stick as i understand, you might install it on the ubuntu partition on hdd (not over windows bootloader).

Look and note the name of the ubuntu partition by running fdisk -l, then:

sudo grub-install --root-directory="your ubuntu partition" (ie /dev/sd?)

when done, run into console: sudo grub-mkconfig && update-grub
