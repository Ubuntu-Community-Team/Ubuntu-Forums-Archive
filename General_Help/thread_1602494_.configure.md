### Post by 3Miro on 2010-10-21
The source is for a different version of gcc, a much older version it would seem. Try to find out which version it requires and then you can install the corresponding version (i.e. g++-4.2 or g++-4.1, look it up in synaptic package manager). Then you will have to tell ./configure to use the old version, type
```
 ./configure --help 
```
for info on how to specify which version of gcc to use.

You should probably first look for a newer version of whatever you are compiling.
