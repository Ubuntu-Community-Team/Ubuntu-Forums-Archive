### Post by sisco311 on 2010-06-03
su prompts for the target user password. If no user is specified it defaults to root. By default, the root account password is locked in Ubuntu. If you want to run a command or start a shell as another user or root, then use sudo. See: [uhelp]community/RootSudo[/uhelp]
