### Post by stylishpants on 2007-05-15
Here are a few that I like to use. 
In this example code, I'm sorting a list of files generated by find.

```

# This is probably what you want, perl is available everyewhere
find . -type f | perl -ne '{push(@arr,$_)} END {print sort{rand(2)} @arr}'

# You can even alias it if you're careful with escaping.  Add this to ~/.bashrc:
alias shuffle="perl -ne '{push(@arr,\$_)} END {print sort{rand(2)} @arr}'"

# Here is a ruby solution, just for the fun of it
find . -type f | ruby -ne 'BEGIN{@a=[]}; @a.push $_; END{puts @a.sort_by{rand}}'

```
