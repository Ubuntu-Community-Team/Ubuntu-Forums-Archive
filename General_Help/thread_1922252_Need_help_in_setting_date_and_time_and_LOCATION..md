### Post by pixiq on 2012-02-08
1. There is a menu entry or GUI application called something like 'Time and Date' where you find administration tools. The gnome version is called in the terminal by ```
time-admin
``` Use this to set the time and locale.

2. The clock widget setting and the places, where you want to watch the time and weather is set separately by clicking on the widget, and as jamesisin wrote you can add several such locations and select one as your home location (shown at the lconized widget)
