### Post by bcbc on 2011-05-01
Wubi will default to 64bit if the computer architecture supports it, and wubi.exe is run standalone. If you run it from a CD it will use whatever is on the CD. If you copy the ISO to the same folder as Wubi it will use the version in the folder (provided the release matches).
