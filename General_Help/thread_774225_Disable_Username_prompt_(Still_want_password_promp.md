### Post by Monicker on 2008-04-29
System -> Administration -> Login Window -> Security -> Enable Automatic Login

I think that will disable the password prompt to log in altogether though, which may not be what you want.

---

### Post by Absorbed on 2008-04-29
Also in the Login Window Preferences, on the Local tab, you can select "Human List" which allows you to select a user from a list rather than having to type in your username.
