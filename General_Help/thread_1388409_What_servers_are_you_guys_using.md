### Post by lukeiamyourfather on 2010-01-24
There's several hundred Super Micro servers running where I work anywhere from ten years to brand new and all that I've used Linux on work just fine. I would imagine some systems from Dell, HP, and the like might not work with Linux that well because some of the firmware features are integrated only in Windows (fan control, status LCD on front of server, status lights, etc.).
