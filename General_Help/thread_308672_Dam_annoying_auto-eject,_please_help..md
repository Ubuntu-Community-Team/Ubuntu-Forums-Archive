### Post by drworm01 on 2006-12-12
System > Preference > Removable Drives and Media > Multimedia
Deselect everything.
At least, that seems to have done the trick for me. I tried removing eject via Synaptic but that would result in a lot of programs disappearing. I tried turning off autoeject for the drive via eject in the terminal, but that didn't solve the problem either.
Hope that helps.
