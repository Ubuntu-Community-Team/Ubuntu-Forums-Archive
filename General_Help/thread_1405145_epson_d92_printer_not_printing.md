### Post by InspiredIndividual on 2010-02-16
It could be a lack of ink. Did you use to see 'Out of Ink' warnings for this printer on Ubuntu? I've got the same printer myself, but I never get messages like that. Maybe you can verify this in the following way: Go to System -> Administration -> Printing, right-click the printer and choose Properties. On my system I can now switch to the tab Ink/Toner levels. There, it is reported that ... no such thing as ink levels are reported! Maybe this report is available on either your or your partners computer, in order to rule out this possible cause?

If you managed to print earlier, you should have at least some driver installed. You can check that as well, at the Settings tab. By the way, I'm just curious, how does this printer work for you? It is dreadfully slow on my Ubuntu 9.10 system (and all previous versions), and I haven't managed to improve its performance so far.
