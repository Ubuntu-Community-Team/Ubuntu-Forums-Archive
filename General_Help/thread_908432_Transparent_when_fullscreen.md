### Post by prshah on 2008-09-02
> **Mashy said:**
> Whenever I go fullscreen the window is transparent and I can still see what's below it.

Go fullscreen, hold down Alt, and scroll up; if it is Compiz related, the opacity will deepen (ie, less transparency). Corollary: Alt+Scroll down will increase transparency.

This setting will be remembered the next time the affected window is opened.
