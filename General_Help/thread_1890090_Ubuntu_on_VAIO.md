### Post by matt_symes on 2011-12-02
Hi

Yes you can. You can use the external drive as a data drive.

You may want to consider a persistent flash drive if you want to install new software and update your system.

Here is one write up.

[https://wiki.ubuntu.com/LiveUsbPendrivePersistent](https://wiki.ubuntu.com/LiveUsbPendrivePersistent)

Search for other references on the web. 

You can use unetbootin on windows to create the persistent pen drive. Again, Google it.

### Post by oldtimer7777 on 2011-12-03
And make sure you create plenty of duplicate Persistant sticks of Ubuntu to keep on-hand, since they tend to break quite easily. 

Installing a new hard drive on  a Viao desktop is pretty straight forward. Usually just remove the power and the IDE/SATA cables, and install the HDD device. You only have to worry about jumpers if you have more than one HDD.   It doesn't have to be a Sony HDD part. Any generic HDD will do.  Just make sure you remove the broken HDD and take it down to the parts supply store to match it up with a good generic equivalent, either SATA/or/UltraIDE.
