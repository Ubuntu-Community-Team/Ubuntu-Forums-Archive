### Post by murph2481 on 2006-04-24
Actually I just did this to my computer.  Using synaptic or Adept you can download the package GNOME (and its dependencies).  Then when you start up Kubuntu the first screen where you log in, just click the menu button and change session type to GNOME and you are good to go.  You can always go back by choose KDE at startup

### Post by Nano Geek on 2006-04-24
You can also install ubuntu-desktop from Synaptic. This will install the Ubuntu theme to. then log out after in finishes downloading and select gnome under **session** in the log-in window. Be careful though, it will add gnome programs to your KDE menu if you install it.:(
