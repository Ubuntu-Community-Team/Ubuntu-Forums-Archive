if you use a compositing window manager, try using AWN with the curves option for a nice dock, and screenlets/superkaramba/conky for a nice system monitor, don't enable desktop icons and choose a nice image for a wallpaper, make the taskbar as small as possible/transparent and use a nice icon package (black and white style 2 is my favorite)

e17 is a pretty desktop environment with some cool animated icons and backgrounds that looks really nice and does not use compiz as well
