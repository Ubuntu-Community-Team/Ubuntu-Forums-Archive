Hi

If to go to the "System" Menu, then "Preferences", then "Sessions". Under the "Options" tab you can set it to remember what programs you have running. This will also remember desktop apps. 

If you are only looking to make uTorrent start when the ubuntu starts, then you can add it under the "Startup Programs" tab.
