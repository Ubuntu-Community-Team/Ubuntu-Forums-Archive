### Post by macaholic on 2008-05-13
That is because emerald (sorta part of compiz fusion) is drawing it's own window borders. You can either change them to something that you would prefer within emerald (System > Preferences > Emerald Theme Manager) or you could use the gtk theme defaults by running```
gtk-window-decorator --replace
```
