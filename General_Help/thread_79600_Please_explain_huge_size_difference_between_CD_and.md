### Post by drtvasudevan on 2005-10-20
I need the i386 version of Ubuntu 5.10.

[http://ubuntu.cs.utah.edu/releases/5.10/](http://ubuntu.cs.utah.edu/releases/5.10/) contains the ubuntu-5.10-install-i386.iso which is 617 MB in size.

[http://cdimage.ubuntu.com/releases/breezy/release/](http://cdimage.ubuntu.com/releases/breezy/release/) contains the ubuntu-5.10-dvd-i386.iso which is 2.8 GB in size. 

Now even given that the DVD also contains the live CD, I would expect the DVD to be about just double the size of the CD, not four times (and more) as much. The Live CD ubuntu-5.10-live-i386.iso itself is only 627 MB.

Please explain why the DVD is so much bigger compared to the CD. Does it contain more packages?

### Post by mtron on 2005-10-20
[QUOTE=drtvasudevan]Does it contain more packages?[/QUOTE]

Yes. The DVD image contains everything from the "main" repositories, so, you most probably won't need it. To save bradwidth i suggest you download the usual x86 install image ~700 MB and get the additional packages you might need from the main and other repositories over the network.
