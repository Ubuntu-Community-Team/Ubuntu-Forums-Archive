If I choose one of the bottom desktops, the cube uses those 2 (I know this because I have 4 different wallpapers).  How can I get all 4 sides?  Feel free to leave any replies.

---

### Post by 3Miro on 2011-06-10
The virtual dimensions of your desktop are in 2D. The cube only uses the vertical dimensions of your current row. You have 2 rows and 2 columns.

To get 4 sided cube, you need to install CCSM (which you probably already have), then go to general settings and change the virtual desktop size to horizontal having at least 4. Then it should work.
