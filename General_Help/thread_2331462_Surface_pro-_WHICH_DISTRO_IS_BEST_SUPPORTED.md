### Post by grahammechanical on 2016-07-22
The manufacturer of the Surface Pro does not support Linux. And I doubt very much if the manufacturer would in anyway cooperate with the Linux developers the way they cooperate with Microsoft.

 And all Linux distributions are built around the same Linux kernels. The best we can hope for is that by running the latest version of a distribution we will get the latest Linux kernel which may or may not have improvements in the areas we desire.
