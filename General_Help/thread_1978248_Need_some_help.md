### Post by hwttdz on 2012-05-11
That looks like a proprietary windows only application.  You could try wine which would likely be buggy at best, or virtualbox, which can be a hassle, and introduces many of the annoyances that come with windows (as you still have to deal with windows).  

What I would suggest is to ask yourself what it is you want it to be able to do and setup your machine to do that using native applications.
