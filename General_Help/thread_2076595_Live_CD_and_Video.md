### Post by ajgreeny on 2012-10-26
2F4U is correct; a live CD does not have the codecs nor the flash plugin to allow you to view many commercial videos or youtube.

It is just about possible to do that using a live USB system with persistence set up when making the live USB, but even then live systems tend to run slowly.

There are some distros which come with all needed codecs and flashplayer already installed, eg the DVD version of Linux Mint, Mepis, CentOS, and several others I can't think of at the moment.
