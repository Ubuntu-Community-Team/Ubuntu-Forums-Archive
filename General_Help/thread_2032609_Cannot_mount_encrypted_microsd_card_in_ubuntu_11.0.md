### Post by Mark Phelps on 2012-07-24
Sorry, but had you NOT encrypted the card, you would have several options to recover the data -- but all of that bit the dust when you encrypted the card.  You will have to decrypt it before any data can be recovered from it.

Encryption wouldn't be worth the effort if you could simply run routine data recovery utilities and harvest the data, now would it?
