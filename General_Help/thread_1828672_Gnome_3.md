### Post by MARP1961 on 2011-08-19
The newest version is still Natty Narwhal 11.04 which runs Gnome 2. The new Unity shell desktop runs on top of this. The next version (Oneiric Ocelot 11.10) comes out half way through October and will again have the Unity desktop running on top of the new Gnome 3. However, if you wish to use Gnome's new desktop (Gnome Shell) you will need to install it separately (easily done via Software Centre).

At present, in 11.04 you can only use Unity or 'Classic' Gnome (different login options). Some people have installed Gnome Shell in Natty Narwhal; but it is a bit more complicated and involves effectively disabling or breaking the Unity option. If you want to try Gnome Shell, then my advice would be to wait until the release of Oneiric in October. If you want to see it now then get yourself a live Fedora 15 disk (which uses Gnome Shell already) and try it without installing.

### Post by MARP1961 on 2011-08-19
I did a fresh install of 11.04 and didn't like Unity very much. I read instructions for installing Gnome 3 Shell via the Gnome 3 Team ppa and went ahead with it!

If you do this, Unity will no longer work (no great loss unless you find you like Gnome Shell even less).

We have been using it on two of our laptops for several weeks with no problems. 

I would advise waiting until Oneiric comes out in October when installing Gnome Shell will be easier; but if you can't wait and don't mind a little excitement and uncertainty then try this:
