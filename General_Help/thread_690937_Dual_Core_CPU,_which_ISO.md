### Post by pointone on 2008-02-08
Core 2's support 64 bit, so you could use the 64 bit version of Ubuntu if you have 4 GB or more RAM. Otherwise, I don't recommend it. Almost all GNU/Linux software is available on x86, but not everything can be found for AMD64. Unless you know why you want to run 64 bit, you probably shouldn't. It'll just prove to be a hassle.
