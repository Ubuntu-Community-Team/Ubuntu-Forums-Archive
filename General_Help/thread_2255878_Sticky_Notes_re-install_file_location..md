### Post by techni4ar on 2015-04-30
While trying to synchronise sticky notes between two machines, I realised that they are only read once, while you are logging in. After that they are only saved periodically, but not read any more from the file.

So there are two options:

1. Copy your backup file back to where it should be while you are logged as a different user (if this would work will depend on the corresponding permissions), or using a live CD/DVD.

2. You can just copy and paste the text from the old sticky notes file into new notes - and in the next minute they will be magically auto-saved.

Needles to say, option 2 is my favourite!  :-)
