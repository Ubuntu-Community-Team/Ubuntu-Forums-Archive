### Post by pwabrahams on 2012-10-13
OK, I've been able to make the fonts in Synaptic tolerably large, though not as large as I would like.  The trick is to go into the Appearance / GTK section of System Settings and set the widget style to QTCurve.  If QTCurve isn't listed as a widget style (it wasn't for me, at first), you'll need to install it.

I still don't know how to make the fonts bigger than that, though.
