### Post by ryanhaigh on 2008-04-09
What remote desktop software are you using? If it is the built in one you configure via system>preferences>remote desktop then you need to connect to it using a vnc client as that is the protocol used. I recommend tightVNC which is available for all platforms and is free.

### Post by ryanhaigh on 2008-04-10
[This page](https://help.ubuntu.com/community/VNC) has all the info you will need to setup the connection. Basically allow remote connections in system>preferences>remote desktop, get a windows vnc client such as [tightvnc](http://www.tightvnc.com/download.html), connect via ip:0 and away you go.
