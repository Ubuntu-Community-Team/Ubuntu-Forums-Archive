### Post by loomsen on 2008-10-19
I never had them installed, but I got fed up with screenlets pretty fast. Every single one was a memory trap par excellence.

I decided to write my own, ain't too hard. And it takes about 100kb of memory.

sudo apt-get install python-pymetar

to get that one. You just gotta tell it to parse a weather report to a file you can then read out with conky or sth. 

Or relax and chose [url=http://ubuntuforums.org/showthread.php?t=869328&highlight=conky+weather+forecast]kaivalagis conky-weather-forecast.[/code] He even provides a repo, that way you'll get updates and bugfixes.
I got it installed quite some time ago, but, as using my own one, didn't give it a try yet. But I'm going to soon I think.
