### Post by nickleboyblue on 2011-04-28
If you also want a good tinkering project, look up LinuxMCE.  It's got a lot of power, but it's also going to occupy your time for the next three months.  If you want something easy and virtually infinitely customisable, I would go with just a plain vanilla install of Ubuntu, though I would personally wait for 11.04's official release, as that's going to have Unity, which will be a bit friendlier on a television screen.

As for software, you can set it up to start XBMC or Boxee directly from boot.  I use boxee, because it allows me to have access to several paid entertainment subscriptions that are not normally supported under Linux.  I haven't tried netflix in it yet, but I am planning to do so soon.

File sharing can be done through Samba, and though I do know it is possible to automatically load folders from other computers on boot, I don't know if that's possible with windows machines.

Anyway, hope that helps!
