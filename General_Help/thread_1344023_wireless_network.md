### Post by Xnst on 2009-12-02
Hi,

I think you can connect the computers to each other once they are connected to the router. You just have to give the correct rights to the directories you would like to share. Since you have windows in your private network you need to use samba. 

As for the wireless network it is configured in the router with encryption key and DHCP and so. Hook one of the computers in with the a cable and login to the routers http config site (usually something like [http://192.168.0.1](http://192.168.0.1)).

I guess it is the same procedure as when using a router with internet access.
