Need help reading German? Or you don't know how to run a java program?

My guess:

Make sure you have java >= 1.6  installed.
D/L the zip file from the link you posted.
Open it.
Copy the **JBinUp.jar** file somewhere (drag & drop).

Probably it would be sensible to copy the **Templates** directory from that zip archive to the same place as you put the **JBinUp.jar** file (drag & drop).

Read the Docs directory (also in German, and if this isn't a language you are familiar with then be aware that search engines offer a "translate" feature now).



**cd** to that directory in a terminal, then 
run it with **java -jar BinUp.jar**

See what happens.

Disclaimer: I *assume* you know what you are doing.
