### Post by DonnOMalley on 2008-03-14
I ended up figuring it out. I had to set my conduits to "Copy to PDA", run synchronize, then set all conduits back to "Synchronize".

I tried using the one-time action, this worked, but appeared to be run before the sync, which caused everything to double up for some reason. I cleaned everything up in Evolution, reset my PDA, then used the method above to get a single copy.
