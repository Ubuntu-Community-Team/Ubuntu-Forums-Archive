### Post by 3Miro on 2011-04-06
Flash does take a lot of CPU resources, I blame Adobe's crappy coding. Youtube and/or flash games would take one full core for me.

For windows, I can't be 100% sure (since I don't use windows), but it seems like you have heating issues. Install some windows program to monitor CPU temperature and keep an eye on it (GPU as well). A fried is using CPUID, but since I don't have windows, I have not used it myself.
