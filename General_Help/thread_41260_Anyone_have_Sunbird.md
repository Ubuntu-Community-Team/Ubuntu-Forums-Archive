 It's always been more convenient for me to have it as part of Thunderbird.  But then again I have my email client open all the time. Tools->Calendar is just about as fast as clicking the Sunbird Icon.

Note that if you go to the Mozilla Calendar site (google it, I'm not sure if it's different than the Sunbird site), there are two versions of the extension: one for Firefox, one for Thunderbird.  Make sure you get the correct one.  Also, extensions in Thunderbird are installed differently than those in Firefox. (Installing the extension in Firefox won't make it appear in Thunderbird btw)  In firefox you click a link and let in install, in thunderbird you download it locally and then IN THUNDERBIRD run Tools->Extenstions and look for the file you downloaded.
hope this helps
