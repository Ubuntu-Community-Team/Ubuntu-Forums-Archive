### Post by RedRat on 2010-04-25
I had a similar problem and found that if the directory to which I was saving a file had too many files in it, the saves took much longer. BTW, I save daily cartoon strips. What I finally did was to make file sub-directories and move files to them. I note that when directories contain close to 300 or more files, the save process seems to take longer. However, keep in mind that I am running 8.04 and and using the new ext4 file system.
