### Post by Zorael on 2010-05-11
On my machine, aside from spacing it looks pretty identical to how it's rendered in Chromium.

What are your font settings set to, in Settings -> Configure Konqueror -> Web Browsing: Appearance -> Fonts tab? I set mine as such;
[list][*]Standard font: Times New Roman
[*]Fixed font: Deja Vu Sans Mono
[*]Serif font: Times New Roman
[*]Sans font: Arial[/list]
Those net me the closest I can get to how it looks on a Windows environment.

Also, do you have a **~/.fonts.conf** file? (Try renaming it and restart Konqueror.)
