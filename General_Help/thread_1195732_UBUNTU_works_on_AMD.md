### Post by jward3010 on 2009-06-24
You can use either version, although I recommend AMD64 for your CPU. There are two versions of Ubuntu available:

- x86 or 32-bit 
Compatible with any AMD or Intel CPU - 32bit or even 64-bit as most 64-bit CPU's can run as 32-bit CPU's. So for example I have an AMD Athlon64 3700 CPU and I run 32-bit Windows XP on it, I used to run 32-bit Ubuntu and now have 64-bit Ubuntu on it.

- AMD64 version
Specifically for 64-bit AMD CPU's - Can be a performance boost from using this one with an actual 64-bit AMD CPU.
