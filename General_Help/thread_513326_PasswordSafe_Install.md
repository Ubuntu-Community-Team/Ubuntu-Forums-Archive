### Post by rg_stephens on 2007-08-10
Password Safe doesn't currently install on Ubuntu, but you can run it using the Windows Emulator (WINE). WINE allows you to run many Windows programs under Linux, but not every program will work well. 

The good news is that this is not necessary, because Ubuntu has a password manager called MyPasswordSafe, which will open your Password Safe files. It also looks very similar. You can find MyPasswordSafe by searching under the Synaptic Package Manager. 

Here is more information on MyPasswordSafe:
[http://www.semanticgap.com/myps/](http://www.semanticgap.com/myps/)
