### Post by jamesrl on 2008-08-21
I got the white text appearing too, but fixed it via this post here: [http://ubuntuforums.org/showpost.php?p=4522073&postcount=11]("http://ubuntuforums.org/showpost.php?p=4522073&postcount=11").

Basically just start mathematica with:
```

Mathematica -defaultvisual

```
and it works so far for me.
