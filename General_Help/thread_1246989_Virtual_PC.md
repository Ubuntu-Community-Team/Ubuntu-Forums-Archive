### Post by Hendrixski on 2009-08-22
oh ... umm, you're probably asking about how to install ubuntu in a virtual machine.

You can install Ubuntu in VirtualPC and then run it inside of windows, but since virtualPC is crap it will make your computer run REALLY slowly.  You can install it yourself in VMWare and then run it inside of windows (VMWare have two products that you'd want to try, "Player" and "server",  I recommend vmware server).


Also VMWare has a list of pre-installed images that you can download and just run them as if you had installed them yourself. I haven't used those in a long time (mainly because my job switched from Windows to Ubuntu almost 2 years ago, so I haven't used Windows in 2 years), but I remember it was as easy as pointing VMWare to that file, and then it would boot Ubuntu inside of Windows.
