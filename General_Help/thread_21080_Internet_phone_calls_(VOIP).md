### Post by Ufo on 2005-03-20
Hi

 I started using my computer for phone calls.
First I did try gnomemeeting. But my computer is behind NAT and with gnomemeeting I was only able to call to someone not behind NAT and they were not able to call me.

I found this guide in internet:

[http://www.richard.neill.hemscott.net/voip.html](http://www.richard.neill.hemscott.net/voip.html)

which helped me to start using kphone (which uses SIP protocol) for phone calls.
With SIP client there are not same problems with NAT that there were with gnomemeeting that uses H.323 protocol.

The reason why I posted this is that I could not find any discussion in these forums about VOIP calls that would be possible behind NAT, except those related to Skype.
And after all Skype is proprietary and closed-source software.
