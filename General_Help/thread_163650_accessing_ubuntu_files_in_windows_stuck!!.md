### Post by Stealth on 2006-04-21
I'm guessing what you're doing is plugging in the key, putting files in it, unplugging, rebooting into Windows, plugging back in, and you see nothing?

Make sure that before you unplug the key in Ubuntu, that you right click the icon on your desktop and click "Eject", this way the files written on it are properly saved.

### Post by kingmonkey on 2006-04-21
Can you access the files in Ubuntu again? 

The file probably did not get written to the USB stick while in Ubuntu. Sometimes added files are stored in memory and are written to the stick when you unmount it. 

Everytime you write something to a USB stick, make sure you unmount it by right clicking on the icon on the desktop, and wait to make sure all activity stops on the usb drive (flickering lights).
