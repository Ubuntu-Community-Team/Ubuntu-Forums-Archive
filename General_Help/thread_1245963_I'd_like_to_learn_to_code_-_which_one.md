### Post by bchurchill on 2009-08-21
Excellent question, and certainly not too long.  Your desire to help the community is greatly appreciated, and I'm sure you'll find many ways to do so.  If you want to give back quickly, you can learn a lot by crusing around the forums and trying to figure out how to answer others' questions.

There's certainly no language that is any more useful than the others for this purpose.  Your language of choice is going to depend on what you're trying to do.  In the end, you'll probably learn several.  Once you've learned one or two you'll find that learning the syntax of a new language isn't a big deal.

For getting started, python has very nice syntax and is a very fun language.  bash scripting is very useful to learn in general since people use shells all the time.  Perl is a very important language, but has the potential to be quite terse for a first choice.  Other options include compiled languages like C, C++ and Java.

It's really up to you.  Do whichever language seems the most appealing, and it will be useful at some point.
