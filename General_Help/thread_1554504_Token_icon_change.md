### Post by Ginsu543 on 2010-08-16
For most icon packages, you simply unpack the package into the .icons folder inside your home folder. If there isn't a .icons (remember, it's a hidden file, so you must hit Ctrl+H to see it), just create one. Once you unpack it, you should be able to see it in System --> Preferences --> Appearance --> Customize --> Icons.
