Step 1

### Post by trevelyan on 2008-06-27
add this at the end of your menu.lst file:
```

title		Windows XP
root		(hd0,0)
savedefault
makeactive
map		(hd1) (hd0)
map		(hd0) (hd1)
chainloader	+1

```

also the line that reads:
```

## hiddenmenu
# Hides the menu by default (press ESC to see the menu)
hiddenmenu

```

make it look like this:
```

## hiddenmenu
# Hides the menu by default (press ESC to see the menu)
#hiddenmenu

```


alternatively if you want.. you can get rid of the older kernels by doing the following:

go to:
System > Administration > Synaptic Package Manager

search for "linux-image" and mark for removal the ones with an older version.. they look something like. linux-image-2.6.24-16-generic
be careful what you delete though.

and to show your drives on the desktop do this:

navigate to:
/usr/share/applications

open "Configuration Editor"

inside configuration editor, make your way to Apps > nautilus > Desktop
and check the boxes for the things you want. like volumes_visible

let me know if it worked for you. ;)

EDIT:
i forgot.. to open your menu.lst file so you can edit it.. do it like this:

open a terminal
type the following:
```
sudo gedit /boot/grub/menu.lst

```
