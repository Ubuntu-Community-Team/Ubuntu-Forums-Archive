### Post by Paddy Landau on 2011-11-20
I'm not quite sure what you mean, but it looks as though you have lost your Unity launcher.


[LIST=1]
[*]Open a terminal by pressing Ctrl-Alt-T.
[*]Type the following command:```
sudo apt-get install ccsm
```
[*]Type the following command:```
ccsm
```
[*]Select *Desktop* (on the left).
[*]If *Ubuntu Unity Plugin* is ticked (see screenshot), untick it.
[*]Then tick it.
[*]If you get error messages asking to resolve conflicts, always press the right-hand button to resolve the conflicts.
[/LIST]

That should bring it back.

Then you can close the two windows.
