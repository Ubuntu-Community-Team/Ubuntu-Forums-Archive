### Post by shadoogie2960 on 2012-11-29
I solved this problem in an easy way:
I copied the files from a partition on my harddisk, containing Ubuntu 11.10, from directory /usr/share/keyrings with 'sudo nautilus' to the partition of Ubuntu 12.10.
Updates works again like it shoud !  :P
