### Post by dugh on 2008-10-31
I didn't realize the tips & tutorials forum is moderated, so I'll post this here instead, since it is about a defective java configuration.

New style java 6 update 10 applets won't work in Ubuntu because the java6 package still configures the old plugin (javaplugin.so) for Firefox and other browsers, not the new plugin (libnpjp2.so).  For example these applets won't work:
[https://scenegraph-demos.dev.java.net/demo-applets.html](https://scenegraph-demos.dev.java.net/demo-applets.html)


I posted instructions for enabling the new plugin in Firefox here:
[http://edtechdev.blogspot.com/2008/10/getting-new-java-6-update-10-plugin-to.html](http://edtechdev.blogspot.com/2008/10/getting-new-java-6-update-10-plugin-to.html)
