### Post by ReFranz on 2016-11-18
I recently attempted, unsuccessfully, to uninstall 14.04 LTS and install 16.04 LTS. I concluded that my desktop was incompatible with 16.04: see [https://ubuntuforums.org/showthread.php?t=2343426](https://ubuntuforums.org/showthread.php?t=2343426). Now, if I attempt to use the Updater, installation fails with the following message: "CD/DVD 'Ubuntu 16.04.1 LTS _Xenial Xerus_ - Release amd64 (20160719)' is required." The Software Center is also damaged: Google Chrome is not listed as a web browser, even though it's still listed in the Synaptic Package Manager, and appears in the Dash--which is also damaged! Is there some way to walk the OS back from this damage, or as I suspect, will I need to burn a 14.04 LTS ISO, and then uninstall and reinstall?

---

### Post by ReFranz on 2016-11-18
Solution (found by serendipity): "Software and Update" folder, click on "Other Software" tab. There I found a checkbox for the 16.04 CdRom. Unchecking it restore the update function to normal
