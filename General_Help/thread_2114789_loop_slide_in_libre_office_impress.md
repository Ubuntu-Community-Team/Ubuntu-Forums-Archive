### Post by blackhawkover on 2013-04-29
hey  I am still waiting for this feature to be added but what I have found works is using a seperate show and just automatically have that mini show loop  then you switch to your main show  so if I have a presentation of 25 slides but want there to loop the first 5 I copy those 5 to a new file, add the automatically advance option under transition, and in slide show settings set to auto with 0 seconds  that will loop but then you need to start your main show  BTW this does not work with the presentation mode, because with that one you can't start 2 shows at once. hope that helps until they add the feature
