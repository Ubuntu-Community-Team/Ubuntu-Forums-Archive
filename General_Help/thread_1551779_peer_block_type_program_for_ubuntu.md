### Post by jre on 2010-08-14
You can use the builtin IP blocker of your client. But this will only affect the direct traffic of your client.

If you want privacy on your whole system you need a system wide application.

If you need a GUI you can install moblock/blockcontrol/mobloquer. If you don't need a GUI you should install their successor PeerGuardian Linux (pgl). I maintain a repository with Debian packages (also for Ubuntu) with all of these at [http://moblock-deb.sourceforge.net](http://moblock-deb.sourceforge.net). There's also a wiki at [https://help.ubuntu.com/community/MoBlock](https://help.ubuntu.com/community/MoBlock).

Alternatively you can use iplist ([http://iplist.sourceforge.net/](http://iplist.sourceforge.net/)) by uljanow.
