### Post by PyrexKidd on 2011-07-18
Most likely: you do not have enough memory to perform the 'normal' install.  Try the 'alternate' install disk.

Alternatively, try burning the CD at a different (slower) speed, or create a live USB and boot from that.  (UNetBootIn is your best bet here, there are both Linux and WinBlows versions available.)Z
