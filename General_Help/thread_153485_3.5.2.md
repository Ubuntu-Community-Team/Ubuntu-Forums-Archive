### Post by njf on 2006-04-01
[QUOTE=3dmaker]I did not mean that page. I ment the deb: links... They do not work.[/QUOTE]
You don't go to the deb link from the browser...

In Konsole, type
```
wget http://people.ubuntu.com/~jriddell/kubuntu-packages-jriddell-key.gpg
sudo apt-key add kubuntu-packages-jriddell-key.gpg
sudo cp /etc/apt/sources.list /etc/apt/sources.list.backup
kdesu kate /etc/apt/sources.list
```

Add 'deb [http://kubuntu.org/packages/kde352](http://kubuntu.org/packages/kde352) breezy main' (exclude the quote) at the end of the file.

In Konsole, type
```
sudo apt-get update
sudo apt-get upgrade
```

I recommend you to logout and login again.
