Hi,

I've had the same problem, sporadically.  I had a Lucid install that could burn fine, but then the problem cropped up again in Maverick.  What works for me is to copy the ISO to a USB stick, take it downstairs with a CD, and ask my wife to burn it on her Lucid machine.  Nothing else seems to work.

It's not my hardware, as other installs on the same box work fine (it's just that right at the moment all I have on it are a bunch of Maverick beta installs.)

BTW, if you're like me, you probably set your personal details to indicate that you're running Jaunty and then just forgot to change them.  But perhaps you really are still running it.  In which case, I would suggest giving Lucid a try.  I honestly don't remember if I had this problem with Jaunty, but it was somewhere in that general time frame that this "on again and off again" burning problem began showing up.
