### Post by Daniel0108 on 2010-11-12
Hmm.. I don't know if it's pre-installed. But you can install it ;)
Alt+F2, type in: "gnome-terminal"
A window appears.
Type into the window: sudo apt-get install html2text
This also works on Live CD's :)
