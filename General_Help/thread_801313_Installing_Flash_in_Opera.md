### Post by pointone on 2008-05-20
The latest stable version of flash is not compatible with the latest stable version of Opera, IIRC. Use Opera 9.50 beta or an older version of flash (9r48).

The latest version of Opera should be [9.5 beta 2](http://www.opera.com/products/desktop/next/) and flash should work fine.

go to tools>preferences>content>plug-in options and see what is there.

```
 sudo cp /usr/lib/flashplugin-nonfree/libflashplayer.so /usr/lib/opera/plugins/
```
