### Post by mcduck on 2009-08-09
> **akrchstn said:**
> Okay I'm just going to bump this instead of a new thread. When I use nautilus (rare) when I'm in awesome it changes my wallpaper and generally makes things look bad. Are there any other alternate file browsers

Nautilus also manages the desktop in Gnome, so if you want to use Nautilus but without it taking over the desktop you need to launch it with "nautilus --no-desktop".

And yes, there are alternate file browsers. You could try PCManFM, ROX Filer or Thunar, for example.
