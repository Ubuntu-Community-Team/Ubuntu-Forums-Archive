### Post by Yellow Pasque on 2013-09-29
Ubuntuforums is the wrong place to request things to be added to the repo. In the future, please file a bug asking for it to be packaged. 

In this case, the package is already available starting with Ubuntu 13.10: [https://launchpad.net/ubuntu/+source/fonts-opendyslexic](https://launchpad.net/ubuntu/+source/fonts-opendyslexic)

It's also available in a PPA for Ubuntu Precise/12.04: [https://launchpad.net/~fmarier/+archive/ppa?field.series_filter=precise](https://launchpad.net/~fmarier/+archive/ppa?field.series_filter=precise)
