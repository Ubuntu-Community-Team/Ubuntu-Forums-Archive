### Post by LepeKaname on 2009-07-06
As far as I know there is no Microsoft-like PHP GUI in which you combine WYSIWYG with automatic PHP coding. However, you may check Eclipse or Netbeans (between others). Eclipse I believe it has a WYSIWYG GUI for HTML(? not sure as I don't use it). But in both have a nice PHP IDE. If you don't need that much, then you can use kate, kdevelop or any text editor with syntax highlight function. PHP+HTML is not that difficult, take your time to learn it and you will not need any special IDE to create any website.

Good Luck
