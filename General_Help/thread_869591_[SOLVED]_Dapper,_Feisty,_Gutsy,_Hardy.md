### Post by Oldsoldier2003 on 2008-07-25
> **mikelmahoo said:**
> What are dapper, feisty, gutsy, and hardy?
Also, what are backports?

dapper= Ubuntu 6.06 LTS
feisty = Ubuntu 7.04
gutsy = Ubuntu 7.10
hardy = Ubuntu 8.04 LTS

backports is a repository that has selected updated software for supported versions of Ubuntu. When a version of Ubuntu is released, the main repositories are frozen except for important security and stability updates.

see [https://wiki.ubuntu.com/StableReleaseUpdates](https://wiki.ubuntu.com/StableReleaseUpdates) for more information.
