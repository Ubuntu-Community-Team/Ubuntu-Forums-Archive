### Post by bolucpap on 2010-01-12
Was the other computer (the one that worked fine) using windows when you installed Ubuntu? If so, I can assume you used wubi to install. From what I can tell, this computer has trouble reading the CD (actually, the autorun.inf inside the CD) because it is not displaying the Ubuntu wubi icon correctly. If you are certain that your CD is error free, then the most probable cause is something inside windows is interfering with the autorun procedures, which unfortunately points towards a virus. In any case, instead of double clicking on the D: drive, you can right click, then select open, which will show you the contents of the CD. Can you do that and post a screenshot again?
