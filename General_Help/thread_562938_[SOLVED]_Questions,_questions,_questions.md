### Post by climatewarrior on 2007-09-29
Warning! Im am by no means an expert so I might get quite some things wrong. My Linux and Ubuntu experience is using it every day on my laptop and installing it onto my friends laptops and helping them when they need tech support. And that I am a huge Linux fan, but im not an expert. Im also a coputer engineering student.


Awnsers

1. Yes there is quite a lot of software for web desing but there arent a lot of Wysiwyg programs like dreamweaver, but they are available. I use quanta plus as my wysiwyg web desing program. As for Illustrator you can cover your vector graphics needs with Inkscape. In my opinion it isnt nearly as good as illustrator but for the average user who isnt working on stuff that is very complicated it shoud be more than enough. As for fireworks gimp should cover your image editng needs quite well. As for flash pro i really dont know about that, im sorry.

2. The Gimp GUI is quite different to photoshop but they are working on that now and it is my understanding that the next version will feature a GUI very similar to photoshop's. Also there is a project called GimpShop which is a version of gimp modified to look like photoshop. Before switching to Ubuntu as my os I used to use photoshop quite often. And when I swithched to Gimp I really had no problems. I takes a little while to get used to it but there isnt any really big learning curve. If you now your way around photoshop you wont have any major problems getting used to the gimp. I dont know if you can use your photoshop filters in gimp but I highly doubt it. But on the other hand gimp comes with a great variety of filters and you can download a lot more filters from the internet. You will probably find similar filters to the ones you use. Yes there is quite a lot of good gimp tutorial websites around. Also remmber that Gimp also works in Windows and Mac so you can try it there before swithching.

3. Yes there is quite a lot of DVD burning software. That is one of the things I can say Ubuntu does way better than Windows. It is quite simple to burn DVDs and CDs.

4. As for video editing Im sorry to say that Linux doesnt offer anything as professional as Final Cut or Adobe Premiere. But there qute a few video editing software that is quite good for non pro stuff. I use Kino and Kdenlive . As for Adobe After Effects i really dont knwo of any equivalent software although Blender which is a really powerful open source 3d suite has a compositing program integrated but I have never used it. I haveonly used the 3d modelling and animation parts.

5. Yes!  The Linux kernel is quite powerful indeed.

6. Yes! the open source community often takes advantages of this kind of things faster than windows or other commercial os's

7. Calc which is open offices excel equivalent is quite powerful and to this day I havent found anything that i need that isnt available in it. But I am no accountant or anything I only use it for simple stuff. There are other spreadsheet programs for Linux but I havent tried any of them and probalby calc is the best. So yeah I think your wife might still need Excel, But dont worry Wine runs the old versiond of Office quite well. And there is also a program called Cross Over Office who's only purpose it to make windows office apps run and install really good in linux. The only catch is that Cross Over is comeccial so will have to put around 50$ for it. Also remember that Open Office also works in Windows and Mac so she can use it for a while to see if it meets her needs. And also IBM is about to release a brand new office suite desinged to compete head to head with office and its coming to Linux soon so keep an eye on that.

8. I have managed to Install and run windows games on Linux and sometimes it can be a pain in 
the *** and other times it can be really easy. I dont know about those games specifically but you can check out the cedega website if they support those games. I have to be honest that gaming in Linux is quite a bitch but it can be done. Sometimes you will spend 2 hours intalling a game. But in the end I believe its all worth it.

[http://games.cedega.com/gamesdb/](http://games.cedega.com/gamesdb/)

There you can find out how to run an install games with cedega.
