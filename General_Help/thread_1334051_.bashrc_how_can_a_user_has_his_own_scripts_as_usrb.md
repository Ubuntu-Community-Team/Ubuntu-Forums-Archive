I have a folder with my own user scripts, than are giving more function to the machine, such as 
hcp
```
#/bin/bash
rsync -avx --progress --stats "$1" "$2"
```

I would like that this folder /home/frenchn00b/myuserscripts/
that contains all scripts are seen as command into /usr/bin

the scripst are alreasdy chmod +x ...

### Post by emigrant on 2009-11-22
open bashrc file

```
gedit ~/.bashrc
```

and add this lines:

```
PATH=$PATH:/home/frenchn00b/myuserscripts/
export PATH
```
