### Post by lovinglinux on 2012-03-31
Disable hardware acceleration in flash Settings (right-click on a video to get it).

If that doesn't work, you can get an older version at [http://archive.canonical.com/pool/partner/a/adobe-flashplugin/](http://archive.canonical.com/pool/partner/a/adobe-flashplugin/)

### Post by SUPERFITTER on 2012-03-31
I figured out how to fix this problem. Downloads from Ubuntu sometimes screw up the flash player.  

1. Uninstall Adobe Flash Player and Gnash SWF Viewer Shockwave in Ubuntu Software Center.

2. Reboot

3. Reinstall Adobe Flash Player and Gnash SWF Viewer Shockwave in Ubuntu Software Center.

4. Reboot

Now my Pandora Radio and other sites work (For Now). :lolflag:
