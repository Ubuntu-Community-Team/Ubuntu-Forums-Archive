### Post by RedSquirrel on 2007-06-19
I don't think this information has been gathered in one place. 

When I do research on programs, I look at Synaptic for some basic information, and then I just use Google (as you have done) if I want to know more.

A number of programs also have manpages that can be helpful (once you've installed the program, of course):

```
 man program_name
```I think the best way to find out the status of a program is to check the project's page and see how things are going. (You get the current and generally reliable sense of where things stand.)

If the project is in active development, you can often browse the developers's code repositories as well and obtain a recent version if you desire it.
