### Post by Sutekh on 2006-04-02
Try this link

[Herman's Dual Boot](http://users.bigpond.net.au/hermanzone/p3.htm)

It's a dual boot one, but we can modify it.

When you get to step 8, you can choose how you want to partition your hard disc.  You have the option to completely erase the hard disc and install Ubuntu on it.  This will remove Windows in the process.  

In the guide the option to **Manually edit partition table** is chosen.  For you the option to take is **Erase entire disc**.  Just make sure you choose the right one (if you have more than one disc).  

You can then follow the guide to its end.  (It's still a good install guide)
