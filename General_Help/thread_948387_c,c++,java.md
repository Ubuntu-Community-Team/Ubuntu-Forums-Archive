### Post by Sycron on 2008-10-15
For c++ the compiler is g++. For java is gcj.
Read more here: [https://help.ubuntu.com/community/InstallingCompilers](https://help.ubuntu.com/community/InstallingCompilers)

To install them type ```
sudo apt-get install g++ gcj
```
