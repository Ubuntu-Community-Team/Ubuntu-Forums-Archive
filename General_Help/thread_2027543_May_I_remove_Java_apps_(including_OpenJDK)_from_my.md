### Post by claracc on 2012-07-17
You need java in order to see applets and play games, and there are lots of applications and web pages which use it, if you uninstall, you won't see them. 

If you are using firefox you can prevent running java with the noscript add on, letting to run it only in the web pages you are confident. [http://ubuntuforums.org/showthread.php?t=1912426](http://ubuntuforums.org/showthread.php?t=1912426)
