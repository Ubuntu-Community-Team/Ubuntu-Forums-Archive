### Post by Jameshardy88 on 2009-01-15
i eventually managed to resolve this problem you need to enable the wallpaper plugin in compiz and then add 4 different wallpapers to the list, the interface does not make it clear that it will set each one sequentially as a wallpaper for each workspace but it does. After this you need to disable nautilus from drawing the desktop wallpaper which can be done in gconf-editor and navigating to nautilus's preferences
