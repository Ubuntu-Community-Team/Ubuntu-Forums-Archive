Hello,
I noticed since i've installed to 11.04 on my new laptop and using the classic ubuntu mode when i drag any app window to top right corner it auto maximize, it wasn't doing this for me in the past on my 11.04 when i was upgrading through distributions

any idea how to disable this?

Thanks

---

### Post by raja.genupula on 2011-09-06
```
sudo apt-get install compizconfig-settings-manager

```

then type ccsm in terminal in the settings 

Uncheck the "Grid" in the "Window Management" category.

This will stop the window from maximizing when dragged to the top.


all the best
