### Post by steeleyuk on 2007-06-06
Install Brasero by typing at the terminal:

sudo apt-get install brasero

In Brasero, click File > New Project > Burn an image

From the drop down menu then choose CUE file and locate the CUE file on the drive. Keep the CUE and BIN files in the same directory.
