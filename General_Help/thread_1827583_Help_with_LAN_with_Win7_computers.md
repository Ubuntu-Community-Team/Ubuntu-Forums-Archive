### Post by LowSky on 2011-08-18
Windows 7 introduces a few new security sharing protocols, you need to make sure each computer is correctly set to do so.

I'm not sure how much stuff you wish to share but if it is just little things, then using Dropbox (which gives 2GB of Free Space) is a great option. That way its all shared with no need to set up file sharing protocols.

but if you want to have real file sharing over a network see this:
[https://help.ubuntu.com/community/Samba](https://help.ubuntu.com/community/Samba)
