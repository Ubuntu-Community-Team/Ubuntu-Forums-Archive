Well, I finally *almost* have a fully functional Linux machine. That's what I did and, sure enough, there was my video! I don't really like having to use the lower resolution, but I can live with it. I'm using an external monitor connected to a laptop; haven't checked yet to see if the result is the same on the laptop display.

My wireless connection is listed in network setting along with the wired, but the wireless is disabled and I can't enable it. Any suggestions appreciated. 

(2) About 50 percent of the time, I hang on shutdown or reboot. The only way to make sure it shuts down or reboots correctly every time is do it from the command line. Not a big deal, but it would just be nice to have it work the way it's supposed to.

Despite those three problems, I'm enjoying Ubuntu (or, actually, Kbuntu). I'm not anti-Windows or Mac (I have three computers: one running Vista, one running Tiger and one running Ubuntu); I just like playing around with computers and like having Linux in the mix.
