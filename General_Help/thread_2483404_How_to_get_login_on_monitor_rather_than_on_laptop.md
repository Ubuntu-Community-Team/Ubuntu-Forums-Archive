My desk being small, the I keep the laptop underneath and set the desktop monitor to be the sole display (i.e., in Settings | Display Mode, I choose Single Display). ### Post by MAFoElffen on 2023-01-29
Some Laptops have a Primary Display > Internal/External option in the BIOS. Other than that... Before you are logged in, the primary display follows whjat is set in BIOS. If no option, the Primary Display is the Internal Display. Once you log in, then it follows what is set for the User Session.
