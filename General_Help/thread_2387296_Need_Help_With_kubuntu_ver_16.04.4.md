### Post by HereInOz on 2018-03-17
If your computer is running the Intel 945 chipset, and you are attempting to install 17.10, then be aware that there is a known bug in the 4.13 kernel which affects graphics on the 945 chipset.  The bug has been fixed in 18.04.  If all the above is indeed the case, perhaps try to install the daily build of Kubuntu 18.04
