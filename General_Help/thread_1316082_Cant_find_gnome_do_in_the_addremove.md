### Post by stderr on 2009-11-05
It's there for me under karmic, and I also know it was there under intrepid not long ago.

```
sudo apt-get install gnome-do
```

Or you can just check your downloaded package lists to see if it's listed there with:

```
apt-cache search gnome-do
```
