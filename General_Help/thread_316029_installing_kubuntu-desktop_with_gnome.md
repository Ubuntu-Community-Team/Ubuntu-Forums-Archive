### Post by ajgreeny on 2006-12-10
Yes, don't worry about this, I do it all the time and there has never been a problem with anything.

You can even, if you want to, have both desktops open at the same time; just start up your usual session and then go to "Switch user" and login to the other desktop.  The second one will run as a separate session on Ctrl+Alt+F8, with your first on Ctrl+Alt+F7.  This can be useful when you want to check differences between the two as it means you don't need to shutdown a session and start another.  You can also have two different users running together, each with a different DE.
