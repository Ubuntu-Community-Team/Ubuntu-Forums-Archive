### Post by marc321 on 2007-07-23
Well, that is definitely not what we want! :)

Let's make sure the backup file we created works.  Since you can't get into the GUI, you can login directly into the terminal.  Where it says login, type your username, hit enter, type your password, and hit enter again.

If there is no "login", try Alt-F3 to go to the third "virtual terminal".  Usually, the first terminal (F1) is the one where the GUI is launched from.  The seventh terminal (F7) is the graphical desktop, if it is actually running (in this case, it isn't).

Once logged in, type the following, and hit the Enter key where I have typed <ENTER>:

```

cd /etc/X11 <ENTER>
sudo cp xorg.conf.bak xorg.conf <ENTER>

```

Whenever prompted for your password, enter it in.

Now, type in this command:

```

startx <ENTER>

```

If it does not bring up the GUI, it will display error messages (usually prefixed with (EE)).

Please see your private messages here on the forum for additional instructions.
