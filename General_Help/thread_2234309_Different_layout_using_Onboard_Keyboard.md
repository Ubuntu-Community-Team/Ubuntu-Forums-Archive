### Post by m-0 on 2014-07-13
Hi,

I have a notebook which has English layout keyboard. By the way, I speak Turkish. I want to have a Screen Keyboard which has Turkish (Q) layout. Is it possible?

---

### Post by veddox on 2014-07-14
Yes, it is.

Go to System Settings -> Text Entry. Click on the "+" button in the lower left hand corner, then add your keyboard layout from the list. Go back to "All Settings" -> Universal Access -> Typing. Turn on the on screen keyboard.

Now you need to reboot before the keyboard shows up (logout/login might do the trick too). When you're logged in again, you should see the keyboard on screen. To change its layout, go to the top menubar, where you will find a menu to change the keyboard layout. Choose the Turkish option, and you're set! (Note: this will change both the on screen and the physical keyboard layout.)
