### Post by netJackDaw on 2009-01-30
The support period for intermediate releases is 18 months which means Intrepid will be supported until april 2010. You can keep Intrepid and get updates til the expected release of the next LTS (which is 10.04).

You can set your preferred updates to LTS only in Software Sources (System -> Administration -> Software Sources and go to updates tab).

### Post by rjklindsay on 2009-07-02
Hi mickbuntu,
Jaunty 9.04 was a poor release anyway, so stick with Intrepid 8.10.
ATI have dropped support for all but their newest GPU's. so drivers stop at Catalyst 9.3. However Jaunty's new xorg-server 1.6 requires Catalyst 9.4 or newer. So, Jaunty can only use the very slow open source drivers with little or no 3D support.

If your ATI card is more than a few months old, you lose 3D if you install Jaunty.  So stick with Intrepid, and pray that Ubuntu returns to xorg-server 1.53 in their next release, or buy NVidia.
