### Post by elicoten on 2008-01-31
I've also been trying. One option (although even that doesn't work yet) is to keep a close eye on ReactOS. ReactOS is an operating system that runs a totally Windows compatible kernel, yet is also completely open source. This hopefully means that it will, with time, run all windows programs, as well as provide a completely free 100% Windows compatible operating system.

[http://www.reactos.org/](http://www.reactos.org/). And ReactOS could be run under Ubuntu by using something like QEMU, VirtualBox, Bochs or VMWare.
