### Post by annoyingrob on 2010-10-15
The best way to start is to ask yourself "What do I want this computer to do?" IMO, the best way to learn how to use it is by accomplishing something that you want it to do.

For example: 
-set up a samba file server so that your windows machines can access files
-set up a meda server so you can watch movies on your PS3 or xbox
-set up an internal DNS so that you can call computers by names like computerx.home.int or something.
-set up a way to monitor you internet bandwidth
-set up a quake server


Once you have  a clear idea of something you want to accomplish, do some searching on ways to do it. You will become very familiar with the command line, and the filesystem after you try to accomplish a few things.

Edit: Another thing to do is figure out to accomplish things that you still rely on windows for, like "How do I run this game on Linux"

There's a new streaming standard called DLNA, supported by the PS3, Android iOS, Windows Media PLayer, even new TVs and blu-ray players with network capabilities.
