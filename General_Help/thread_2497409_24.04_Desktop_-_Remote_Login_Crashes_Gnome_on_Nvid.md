### Post by ubfan1 on 2024-05-03
The normal repositories for Ubuntu 22.04 have the 545 and 550 drivers, but the standard installation methods (ubuntu-drivers or Software&Updates/AdditionalDrivers) do not show them as candidates.  The 545 driver can be made visible to the standard install methods if you add the graphics-drivers PPA (and maybe even the 550 driver by now).  Doing a manual install of the 550 will apparently work, but while lsmod will show nvidia, nvidia-settings does not show the expected screens, and nouveau is listed in the Software&Updates Additional drivers, without 545 or 550, and all candidates greyed-out.  At this point, either wait a bit, join the launchpad bug 2037004, ([https://bugs.launchpad.net/ubuntu/+source/ubuntu-drivers-common/+bug/2037004](https://bugs.launchpad.net/ubuntu/+source/ubuntu-drivers-common/+bug/2037004)) or try the PPA for the 545 driver.
