### Post by kanikilu on 2009-04-27
[InfraRecorder]("http://infrarecorder.org/") (free, open-source) is a handy Windows utility that can (among other things) blank/erase CD-RW's.

### Post by coffeecat on 2009-04-27
The Ubuntu [Burning ISO Howto]("https://help.ubuntu.com/community/BurningIsoHowto") recommends the free [Infra Recorder]("http://infrarecorder.org/") for Windows. I've used that in Vista and XP and it works very well. Apart from all the usual features it will blank rewritable discs.
