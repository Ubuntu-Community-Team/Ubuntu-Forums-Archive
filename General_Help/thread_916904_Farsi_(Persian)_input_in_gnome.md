### Post by civillian on 2008-09-11
In order to set up standard unicode farsi, it is possible to do it easily in KDE (and therefore Kubuntu) (the guide is [here](http://www.hoomanb.com/Unicode/Linux_KDE.htm)

I appreciate that this is not quite what you were looking for so I did a bit more searching, and I came accross some packages that enable farsi truetype fonts which you can use in any application that uses truetype fonts. There is an official ubuntu package which is cales 'ttf-farsiweb' and also another called 'ttf-freefarsi' both of which you can apt-get or get from synaptic (whichever you prefer), and so for text documents you could use this font to type in farsi.

For other things (pidgin etc.) this won't be a fix, however this might be start for you. 

There is also a farsi release of ubuntu which might be someplace else to try.

Hope this helps.
