### Post by eapache on 2007-02-17
With XP already installed, you can install Ubuntu. When the PC boots, it will show a list of all your installed OSs (including Ubuntu and XP) Choose which one you want, and it will automatically boot to that OS. To change from one to the other, you will have to reboot the computer.

### Post by AndyCooll on 2007-02-17
You've got three main choices.

1. Dual-boot. See the link in my signature for an excellent howto.

2. Virtualisation, i.e. Xen or something similar. Lots of howtos on this.

3. You can install Virtualbox or VMware and run XP in that. Essentially you boot into Linux and then run another operating system inside a "window".
