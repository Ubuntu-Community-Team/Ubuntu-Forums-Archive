### Post by majoridiot on 2007-01-11
the same thing made me nuts, too when i first switched over from windoze ;)

pull up a terminal and type:

```
nautilus
```

then in the nautilus menu, Edit-->Preferences

there are the icon view defaults... i set mine at 75% default zoom and checked 
"compact layout". it gives me a much cleaner desktop that i'm more accustomed to.

there are also a lot of settings to tweak on the other tabs, depending on your
inclinations.

the top and bottom (or side) panels can be resized by right-clicking on them,
selecting preferences and adjusting the size in pixels (there is a minimum 
allowable size).

enjoy ubuntu! :D
