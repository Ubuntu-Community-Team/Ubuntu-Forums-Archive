### Post by oakgrove on 2009-12-23
make a file in your home directory called whatever.html.  Right click on it and select properties > open with and see what is listed there.  If you don't see firefox or whatever it is you prefer and firefox isn't selected as the default application, click add then select Firefox in the box that pops up.  When it takes you back to the box before that, make sure firefox is selected as the default app.  Then click close and see if when clicking on the whatever.html file you made in nautilus, firefox tries to load it or not.  

Hope I was clear.
