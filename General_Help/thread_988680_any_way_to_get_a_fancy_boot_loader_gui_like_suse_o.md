### Post by drs305 on 2008-11-20
For the very basics, StartupManager is a gui app that will allow you to use colors in the menu and select an image for the background.
```
sudo aptitude install startupmanager
```
System, Admin, StartUp-Manager

It will also allow you to tweak the menu options such as the number of kernels to display, timeout, default OS, etc. There's a link to it in my signature line for more information.
