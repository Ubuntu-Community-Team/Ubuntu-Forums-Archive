### Post by taurus on 2009-02-27
Do you get any error message when you run this command from a terminal to mount your ntfs partition, /dev/sda5?

```
sudo mount -t ntfs-3g /dev/sda5 /media/DATA
```

### Post by taurus on 2009-02-27
You need to create that mount point.

```
sudo mkdir /media/DATA
sudo mount -t ntfs-3g /dev/sda5 /media/DATA
```
