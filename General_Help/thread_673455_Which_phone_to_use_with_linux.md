### Post by Whiffle on 2008-01-24
I have a Treo 650 that works pretty well with Kontact and Kpilot, and could sync via bluetooth if I ever get around to it.  I don't sync it real often, mostly just for backup purposes actually.  You have to be careful how you set the options in kpilot or it tends to make copies and delete things and whathaveyou, but overall it works well.

As for the phone itself, I love it.  ITs not huge, but its also a PDA.  I plays music.  I can watch movies on it if I were so inclined.  There are a bazillion programs for it.  I don't have a data plan for it, but if I did I could send email, IM, surf the web..etc..its crazy.  Only real issue is that it occasionally crashes and reboots itself, but not so often that its an issue for me.  It usually only does it when I'm testing out some new software anyway.
