### Post by merlinus on 2007-06-22
Did you burn the downloaded .iso file as a data disk?  Or as a disk image?

If the former, then it will not work.

Your burner software should give you the option.

You can check by looking at the contents of the cd in windows.  If you see folders and files, then it is some other problem.

If you see one file, then go back and burn it slowly (4x max) as a disk image.
