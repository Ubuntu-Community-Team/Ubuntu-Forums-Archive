### Post by josephmills on 2011-12-14
gnash swf works for me with ubuntu extras open terminal and enter 
```
sudo apt-get install ubuntu-restricted-extras gnash
```

```
man gnash
```
then capture flash with gtk-record my desktop or xvidcap 

you might also want to look at adding some additinal repos to you ubuntu Like mediabuntu [http://medibuntu.org/repository.php](http://medibuntu.org/repository.php)
