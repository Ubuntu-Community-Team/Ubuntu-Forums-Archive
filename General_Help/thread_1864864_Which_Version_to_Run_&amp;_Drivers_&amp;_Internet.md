### Post by viperdvman on 2011-10-20
I pretty much agree with everyone here. With that setup, any *buntu should run on your computer no problem. The only drivers that might need additional installation after you install Ubuntu is your NVidia driver and probably a Wireless driver if your computer has one that isn't supported out-of-the-box. With an internet connection, Additional Drivers usually activates immediately. If not, then it would be under the system settings of whichever *buntu you decide to install.

So all the *buntus should run on your computer. That means your decision is going to boil down to which desktop environment you want to run. There's **GNOME 2.x** in Ubuntu 10.04 and 10.10 (and 11.04 in Ubuntu Classic mode), there's **Unity** in Ubuntu 11.04 & 11.10, there's **KDE Plasma** in Kubuntu. There's **Xfce** in Xubuntu, and there's **LXDE** in Lubuntu.

Unity and KDE Plasma use the most RAM, but also have plenty of eye candy if you're into that. KDE Plasma is very Windows-like by default, but can be made into pretty much anything you want. Unity is pretty fixed, has a few Mac OS X like features, and features a large dashboard instead of a main menu as well as a dock (launcher) on the left side.

Xfce and LXDE are both lightweight desktop environments that still keep a traditional desktop look unlike Enlightenment, Fluxbox, and Open box (which are also very lightweight on resources). Xfce uses a bit more memory than LXDE, but is a more visually stimulating desktop closer to GNOME 2.x. LXDE was designed to be run on legacy computers running as little as 256MB of RAM, and it has a bit of a Windows 9x look to it. Both are great if you want a lightweight and fast desktop that doesn't use much RAM.

GNOME 2.x kinda falls in between KDE Plasma and Xfce as far as the "eye candy-to-RAM usage" ratio. By default, it has a top-mounted main panel and menu bar like Mac OS, and also adds a second panel for a workspace switcher and task manager. By default, it doesn't provide much eye candy like KDE Plasma and Unity do, but with the **Compiz** window/compisiting manager installed and activated, GNOME 2.x can be very visually stimulating, very close to KDE Plasma _*(note: using Compiz will eat up a little bit more RAM)*_. In addition, the panels can be customized any way you want (even removing the bottom panel altogether) and even make it to look and act like Windows or Mac OS X. (e.g. Linux Mint 11's default setup mimics Windows)

So it's all in what desktop environment you want to run, and which fits your style best. Good luck with your decision :)
