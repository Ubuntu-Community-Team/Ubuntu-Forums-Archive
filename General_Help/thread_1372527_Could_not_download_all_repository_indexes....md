### Post by plucky on 2010-01-04
Replace all instances of ```
http://ca.archive.ubuntu.com/
```

with ```
http://old-releases.ubuntu.com/
``` 

So they look like```
deb http://old-releases.ubuntu.com/ubuntu/ feisty main restricted
deb-src http://old-releases.ubuntu.com/ubuntu/ feisty main restricted
```


Then update again

Good Luck

p.s.There are no security fixes provided for Feisty systems.
