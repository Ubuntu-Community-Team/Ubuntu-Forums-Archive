### Post by Bari on 2008-08-29
go here to read about busybox [http://www.busybox.net/about.html](http://www.busybox.net/about.html) If its appearing as soon as you start up you may be able to go to System - Adminstration - Services and untick the entry for busybox. If its there this should stop it loading on startup. Hope this helps  Or if you don't want or need it go to Applications - Add/Remove and uninstall it.
