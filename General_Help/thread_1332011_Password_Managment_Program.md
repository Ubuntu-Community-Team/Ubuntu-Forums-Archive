### Post by Cheesemill on 2009-11-19
I use KeePassX, it's in the Software Centre.
There's a windows version called KeePass which will read the same database, if you store this on a drive that both OS's can see you can access your passwords from Ubuntu and Windows.

Backing up is as simple as making a copy of the database file.
