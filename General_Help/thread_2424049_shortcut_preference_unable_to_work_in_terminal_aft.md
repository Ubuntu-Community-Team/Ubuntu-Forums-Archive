### Post by TheFu on 2019-08-02
Welcome to the forums.

If you use X/Windows, it isn't copy/paste, it is select/paste.

Select using the left mouse button and paste using the middle mouse button.  Much more efficient than using a keyboard.  X/Windows select/paste should work regardless of the program, unless the programmer is a **** and went out of their way to prevent it.  If they did nothing, it works, automatically.

Bash has nothing to do with copy/paste, but mastering this: [https://github.com/jlevy/the-art-of-command-line](https://github.com/jlevy/the-art-of-command-line) and your editor would be smart to be crazy efficient.
