### Post by howefield on 2011-05-21
Open the Ubuntu Software Centre.

Search for flash and it should be the first result in the list. CLick on it to highlight and press the Install button.

---

### Post by coffeecat on 2011-05-21
Install the package flashplugin-installer from Synaptic or Software Centre. Better still, install ubuntu-restricted-extras which includes flash and most of the multimedia stuff you'll need.
