### Post by Bucky Ball on 2015-09-28
Try this:

```
xrandr --output VGA-0 --off --output HDMI1 --mode 1920x1080 --pos 0x0 --rotate normal --output LVDS1 --mode 1366x768 --pos 1920x312 --rotate normal
```

That is not a script though. What you need to do is create a file that looks like this:

```
#!/bin/sh
xrandr --output VGA-0 --off --output HDMI1 --mode 1920x1080 --pos 0x0 --rotate normal --output LVDS1 --mode 1366x768 --pos 1920x312 --rotate normal
```

You can do this using Leafpad if you have it, a text editor or the terminal. Save it and then link to this in your Session and Startup command.
