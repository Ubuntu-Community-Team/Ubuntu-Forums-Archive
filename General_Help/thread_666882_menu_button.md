### Post by fifthwind on 2008-01-14
Make a panel image that is say 40 px high where the top 10px is transparent. Then you can make a menu button that is the full 40px making it look like it is taller than the panel. :)

You can use any image editor to make your panel graphics (I use Gimp) just make sure that you save in a format that supports transparency (png, gif) ect.

To put a panel graphic into use, just drag and drop the image onto the panel.  No need to go to config menu's or even right-click.  Just drag an image onto it. (Helps if it is the right size for the panel :) )
