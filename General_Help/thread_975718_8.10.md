### Post by ibuclaw on 2008-11-21
Upgrade XP to Vista (you should be given that option).

If it overwrites grub, just re-insert your Ubuntu LiveCD, open a terminal, and run:

```
sudo grub
```

Then locate the filesystem with Ubuntu on:
```
find /boot/grub/menu.lst
```
And use the output from that to re-setup grub.

For example:
```
root (hd0,1)
setup (hd0)
```

quit, and reboot.
