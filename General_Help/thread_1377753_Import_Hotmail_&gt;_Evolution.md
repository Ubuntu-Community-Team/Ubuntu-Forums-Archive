### Post by Tourdog on 2010-01-16
Your Hot mail account needs to be set up for SMTP  for sending mail and do the same for Evolution*..............  sending 1st through Evolution then on to Hotmail.  This is the SMTP part.   Now to receive...........  in your Hot mail account set up POP and and the same in Evolution.   So in recap you make a few key strokes in each account and they will handshake.   I can't image not.   If you use IMAP for receiving it will be harder to set up the receive.................   I never could with my GMAIL but POP works beautifully for receiving and SMTP for sending.

Evolution up and running........Edit ..Preferences... Mail Acc't  Just work your way thru these on this page.  You can catch the incoming/outgoing (receive/send) on this page.   Do the same for Hotmail.
