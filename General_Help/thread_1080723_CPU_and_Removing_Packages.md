### Post by ibuclaw on 2009-02-25
What are you using to measure the CPU usage?

The Gnome-System-Monitor (if that is what you are using) has had a longstanding bug in it that causes the application (itself) to run at an abnormally high CPU level.

Open a terminal and run:
```
top
```

for a more accurate reading.


Although, what you can do to help get a better reading off gnome-system-monitor, is to go into **Edit->Preferences**, and set all **Intervals** to 5 seconds.
