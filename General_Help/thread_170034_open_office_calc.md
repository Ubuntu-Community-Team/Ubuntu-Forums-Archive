### Post by garner_nc on 2006-05-03
You need to format the cell to display the number of decimal places you want. 

Left click on the cell to format. Right click in the cell  to get a drop down menu. Select format from the drop down menu. Select the number tab from top of format window. Select number of decimal places.

### Post by dguedry on 2008-01-27
Although Open Office can do statistics, financial, logical, etc. , the basic mathematical function QUOTIENT bizarrely only returns the integer and not the decimal.  You need to do a division to overcome this.  ex : =A1/A2 :confused:
