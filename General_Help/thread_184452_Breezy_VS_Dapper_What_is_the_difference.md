I think you misunderstand what the Ubuntu team does - importing the new versions (and fixing the interrelated bugs, etc) **is** pretty much the difference between Ubuntu versions.  The Ubuntu devs don't write (almost all of) the software that goes into Ubuntu - they (in general) take the latest version from Debian, possibly fix some bugs in it, and test that the **whole thing** works together.
