### Post by Rallg on 2011-02-09
My Asus EeePC goes into hard freeze shortly after reaching the desktop. Typically this is about the time, or shortly following, the keyring request to unlock my stored WiFi codes.

My installation began as 10.04 then was upgraded to 10.10, with all the latest updates. I have not had this problem before. The Windows partition has no problem.

I believe the problem lies with the recent -35-25 kernel, because if I boot to the older -35.24 kernel, all is OK as before.

I have noticed other posts about hard freezes lately, on various computers. If that is you, try booting to an older kernel if you still have one.

UPDATE: With time, the problem went away. Probably some update fixed the issue.
