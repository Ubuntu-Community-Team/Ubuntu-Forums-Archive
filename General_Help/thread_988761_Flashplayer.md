### Post by benny bronx on 2008-11-20
What version of Ubuntu are you using? And how did you download the plugin?  If you are running hardy or ibex, you can get flashplayer 10 by enabling the partner repository in the synaptic package manager, then search for adobe-flashplugin.

To enable the partner repo; menu-system-administration-synaptic package manager.  Then settings-repositories-third party software. Tick the box that mentions partner.  Then in the main menu of synaptic, hit the reload button.  Then do the search mentioned above.

Note: you must remove any previous flash plugin before doing this.
