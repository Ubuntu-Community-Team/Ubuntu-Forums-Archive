### Post by dalee on 2010-05-24
Hi,

Kodak doesn't support their printers for Linux in general. So they aren't really recommended. But I have a 5650 working wirelessly with my netbook and Lucid. The driver can be found here: [Here]("http://sourceforge.net/projects/cupsdriverkodak/")

I had to do sudo apt-get install libcupsimage2-dev to get it to compile. There wasn't a .deb available then. I haven't been able to get it to work on a wired network yet. But I'm not concerned as I have a perfectly good HP connected to my desktop box.
