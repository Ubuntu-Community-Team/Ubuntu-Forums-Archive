### Post by kahlil88 on 2008-07-25
The file recovery programs for GNU/Linux are pretty lame. I recommend installing [Recuva]("http://www.recuva.com/") in Windows. Maybe it will find what you lost. Repair installs don't usually erase things, but they sometimes ask you to create a new user account even if one already exists, so you might look in **C:\Documents and Settings** and see if the folder for your old account is still there.
