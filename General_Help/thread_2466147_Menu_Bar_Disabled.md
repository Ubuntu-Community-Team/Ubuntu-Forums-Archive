### Post by Dennis N on 2021-08-21
The menu bar availability is inconsistent. Gnome applications like Files (once called Nautilus) and Text Editor (once called Gedit) substitute several buttons (it varies how many) for the menu bar functionality. Gnome Terminal lacks buttons, but you can define a keyboard shortcut (in preferences) to reveal a traditional menu bar. Applications not created by Gnome developers like LibreOffice almost always have menu bars (and tool bars) by default. Firefox shows a menu bar with the Alt key if you need it.

(Ubuntu could offer Xed which is Text Editor with a menu bar (and tool bar) but for some reason they don't. Other Linux distros do.)
