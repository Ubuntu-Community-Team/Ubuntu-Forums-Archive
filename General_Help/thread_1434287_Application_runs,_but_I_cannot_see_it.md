### Post by El_Shrander on 2010-03-20
I have this game I like playing (Guild Wars, run on WINE), and I have been doing so for months now, no problem at all. Today, however, when I click on its icon, the start window shows up only for a second before vanishing in thin air.

I can hear the game running somewhere in the background, waiting for me to log in, but I cannot access it because it's not anywhere to be seen. Go to System Monitor, there it is - but I can only end the process; if I try to run the game, the whole cycle starts all over again.

As far as I can tell, only that game suffers from this kind of annoying invisibility, any other program I start up works just fine.

Any ideas? :confused:

### Post by El_Shrander on 2010-03-23
Since Guild Wars is the only .exe I run on my computer anymore, I decided to make a clean uninstall of Wine, reinstall, and update. Now I can play again.

My problem is fixed, then, but I won't mark it as solved for the time being because I found out it is related to a longstanding WINE bug, related to the switching of workspaces, as reported [here]("http://archives.free.net.ph/message/20071022.212015.d41a9cd0.ca.html"), i.e. A quick search of *wine switching workspaces* in Google will produce many hits on this topic.

I won't say it's a matter of life or death, but it would be nice if this minor yet annoying bug could be gotten rid of, eventually.
