### Post by 505 on 2007-09-02
There is not file system on that drive, therefore it does not show up. To get it back, depends on whether you want to use Ubuntu on it or just as an extra disk in Windows.
For Ubuntu, just start an installation from a live CD.
For Windows, right-click My Computer, select manage, and go to disk management. You should see you second drive. Right-click the drive and choose format. This will install an NTFS file system on the drive.
