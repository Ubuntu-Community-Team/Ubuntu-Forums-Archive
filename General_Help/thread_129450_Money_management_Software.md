### Post by beartard on 2006-02-13
I'm waiting on the gtk 2 version of GnuCash to come out.  In the meantime I'm using PLcash, a java-based checkbook manager.  It's not the most spectacular software on earth, but it can import gnucash data and export QIF (quicken) data.

I still wish there was something that worked like Quicken.  I much prefer single-entry personal accounting with categories that match my budget, not separate accounts like GnuCash does.

### Post by fangorious on 2006-02-13
I've been pretty happy with kmymoney2. I can import the Quicken 2003 files that most of my credit card issuers offer and the interface is pretty clean. I used to use Quicken back around '99/'00 and kmymoney2 seems to pretty much work the same way from what I can remember. There was a review from last fall in LWN that covered gnucash, kmymoney2, and grisbi. Conclusion was that gnucash was more feature-rich but slightly harder to use asit had a lot of features aimed at business use, so the editor recommended kmymoney2 for personal use.
