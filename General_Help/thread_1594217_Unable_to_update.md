### Post by Ahava591 on 2010-10-12
Would changing the server you get updates from possibly help?

In Lucid this can be done thusly:

System->Administration->Software Sources
there is a line in Software Sources that reads: "Download from: " with a dropdown menu of choices.
Try switching to a different server in this manner.

I don't know where you would do this in 10.10; but I don't think it would be too different.
