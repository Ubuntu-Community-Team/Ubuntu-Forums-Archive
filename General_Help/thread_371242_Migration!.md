### Post by thunderduck3141 on 2007-02-26
Hi all,
I have a Windows XP laptop (HP Omnibook 4150), and would like to banish Windows from it. Is there any way that i can remotely (as this laptop has no optical burner) copy ALL of the data on the laptop so that i can burn it to some DVD's? And would it be possible for me to make it so that i can (in the event of needing to do so) restore my XP operating system so that it is usable? Any ideas? Thanks in advance.

---

### Post by raja on 2007-02-26
If you are connected over a network to another computer, this will be the best solution. Use systemrescue cd to boot into the system and use partimage, which is present on this cd, to make an image of your windows partition on to the networked computer. If you want to restore later, you use partimage again for that.
