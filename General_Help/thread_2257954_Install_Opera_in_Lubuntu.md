### Post by nerdtron on 2014-12-23
Download the deb package from the website [http://www.opera.com/computer/linux](http://www.opera.com/computer/linux)

On your Downloads folder you should the downloaded file, maybe like opera-stable_26.0.1656.32_amd64.deb
Just double-click the file and the Software center will open, then click Install.
