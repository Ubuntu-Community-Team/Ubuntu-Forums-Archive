### Post by blazemore on 2009-11-23
Yes.
The program that provides this feature is called Maximus.
You can disable it from running automatically by turning it off in your Startup Items, under the System -> Preferences menu.

Hope this helps!

### Post by blazemore on 2009-11-23
If you uninstall maximus, a future update may reinstall it, as it is a dependancy of some very high-up metapackage (ubuntu-netbook-remix I think). That's why you should leave it installed, but just disable it.
