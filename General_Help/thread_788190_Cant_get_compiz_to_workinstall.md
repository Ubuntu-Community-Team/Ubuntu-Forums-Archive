### Post by fmartinez on 2008-05-10
If you have installed drivers directly from the ATI website instead of enabling "Restricted Drivers". You need install the XGL server. 
#sudo install xserver-xgl
This gets installed automatically when you enable the Restricted Drivers through Ubuntu and will allow compiz to work. 
For Hardy:
System => Administration => Hardware
You should see a listed of restricted devices that you can enable by checking off a box next the device.
