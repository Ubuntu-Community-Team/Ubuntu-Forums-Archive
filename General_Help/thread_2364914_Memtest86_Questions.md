### Post by ajgreeny on 2017-06-29
Memtest does not finish on its own, it just keeps repeating the tests again and again to allow a thorough test to be carried out.

It's a very ,long time since I ran such a check, as you can not do it when using UEFI which I have been doing for a few years, but I ran it overnight when I did use it, and the figure I wanted to see was the 0 errors.  I can not now remember how the test reports how far through it has managed to get, but I think it was a lot faster than 15 hours to get through a few restarts of the memtest.

Look more carefully to see if there is an error number anywhere.
There's a good but old descriptive page about it at [https://askubuntu.com/questions/591488/how-do-i-run-memtest86](https://askubuntu.com/questions/591488/how-do-i-run-memtest86) with some images of more or less what you should see.
