Yep, As Jackzor said, you do not need to create another partition, and you will be able to select which operating system to boot into when you turn on your computer.

If you decide to remove Ubuntu, you can do so through the "add/remove programs" application in Windows Control Panel (or in the case of Windows 7, I think it is called simply Programs ?).

Bear in mind, that a Wubi install is primarily intended to give you a flavour of how Ubuntu runs/performs and get you familiarised with the system, longer term usage is generally better suited to a real dual boot.

In time, that's the route you might choose to take.
