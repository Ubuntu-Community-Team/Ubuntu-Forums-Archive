### Post by raijinsetsu on 2007-09-19
You could make a ramdisk, and mount it as "/tmp".
Other than that... there's nothing you can do that would sanely make use of the memory.
Another thing: if you're running 32-bit linux and have 3gb+ of ram: get rid of your swap. It won't be used unless you recompile the kernel. :)

### Post by lolindrath on 2007-09-19
You can also turn off swapping, there's no point in the kernel copying memory to hard disk if you have that much RAM.

You can use Folding@Home, that'll eat up a couple megs, plus any spare processing power.

If you just want to see it full you can always right a quick C application make make a really really big array and run it.
