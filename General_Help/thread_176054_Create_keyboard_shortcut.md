### Post by aysiu on 2006-05-14
Go to Applications > System Tools > Configuration Editors > Apps > Metacity

Within Metacity, you'll see two more options: Global Keybindings and Keybinding Commands.

Put your keyboard shortcuts in Global Keybindings
Put your program command in Keybinding Commands
