### Post by 3Miro on 2011-04-02
If you have reinstalled windows, then Ubuntu is gone. Check to see if there are any Ubuntu partitions left, but if you told windows to take over the entire HDD, there wouldn't be any left. If there are any partitions, you can remove them.

Either way, Ubuntu is gone from your system. If you can only run windows in "safe-mode" this is a windows problem. Check with a windows forum.
