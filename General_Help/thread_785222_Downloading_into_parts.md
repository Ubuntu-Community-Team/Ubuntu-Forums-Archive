### Post by latna on 2008-05-07
Wget will let you do this. It's command line not gui. I'm not aware of an option to stop downloading after certain amount of data. But you can stop it manually by typing <ctrl> C. Then resume the download with "wget -c URL", without quotes. Replace URL with the actual url of the file you want to download.

For a gui program you could have a look at an ftp program such as Filezilla. There's a windows and a linux version.

HTH
