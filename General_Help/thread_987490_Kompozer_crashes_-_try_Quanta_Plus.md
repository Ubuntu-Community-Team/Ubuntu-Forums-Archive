It's bad because they don't generate good code.  They reinforce poor code practices and force browser developers to support bad code (*cough* IE).

If you want to learn how code a website, use an editor like bluefish or the code editor in quanta (or even gedit and kate/kwrite have great syntax highlighting), and then keep a web browser open and click "refresh" when you want to see your changes.
