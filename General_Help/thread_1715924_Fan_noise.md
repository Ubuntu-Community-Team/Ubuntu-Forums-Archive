### Post by Enigmapond on 2011-03-27
Your problem isn't with the operating system but is hardware related. Spinning down disks has nothing to do with the fan...the fan is cooling the processor.

If your laptop is running hot, maybe you should think about getting a cooling pad. I use one and it keeps the temp down to 35-38C pretty much all the time.

### Post by arvevans on 2011-03-27
Go to the Synaptic Package Manager and do a search for "Toshiba".  There are a number of BIOS and ACPI interface issues that require Toshiba-specific patches for monitoring temperature and managing the fan(s).
