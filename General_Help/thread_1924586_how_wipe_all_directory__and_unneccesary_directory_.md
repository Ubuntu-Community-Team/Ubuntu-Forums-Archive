### Post by jerrrys on 2012-02-14
Ubuntu directory commands (NOT for windows)

[http://manpages.ubuntu.com/manpages/oneiric/man1/rm.1.html](http://manpages.ubuntu.com/manpages/oneiric/man1/rm.1.html)

[https://help.ubuntu.com/8.04/basic-commands/C/files-directories-commands.html](https://help.ubuntu.com/8.04/basic-commands/C/files-directories-commands.html)

[https://help.ubuntu.com/community/Beginners/BashScripting#Deleting_files](https://help.ubuntu.com/community/Beginners/BashScripting#Deleting_files)

The "rm" command will delete a directory, regardless if it a good or bad idea and cannot be reverted.  So be very, very sure about what you delete or you will break your system and you could easily find yourself reinstalling ubuntu.
