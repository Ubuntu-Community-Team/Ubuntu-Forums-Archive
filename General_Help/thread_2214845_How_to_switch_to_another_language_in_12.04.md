### Post by kpfuser on 2014-04-03
After installing Ubuntu 12.04 recently, I downloaded the modern Greek language pack intending to use Greek as a second language. Consequently, I would like to be able to type documents in Greek since documents I get in Greek I can already read without needing to resort to the Greek language pack. From past experiences (not Ubuntu-related), once the second language pack is installed, one could switch to typing in the second language via the keyboard and even switch back and forth between languages if, say, a portion of text in English had to be entered somewhere in the body of the Greek document. Now it appears from what I found so far that such simple switching between languages by way of key strokes alone may not be possible. So I am writing in order to confirm whether or not simple and fast switching of languages before or during typing is still possible in 12.04

### Post by sudodus on 2014-04-03
You can install several languages and select which one to use. 

Select ***Language Support*** and install the relevant language. (It is a little different with different versions and desktop environments.)

You may want to set the ***Keyboard layout*** separately.

tt will work after the next reboot.

### Post by kpfuser on 2014-04-10
Thank you both for your replies.

The link provided by Warren Hill in his earlier post contains the info needed to answer my original question. For the sake of completeness I will summarize what needs to be done here.

1. For merely being able to type an occasional document or email in Greek, it is not necessary to download and install a Greek language pack from Dash/Language Support. This is needed only if one wishes to change the language in the menus that appear on screen, which was not my intention.

2. From Dash/Keyboard Layout/All Settings by clicking on the "+" sign in the lower LH corner, Greek can be added as a second language. Then continuing with Options/Key(s) to change layout, a keystroke combination (Shift + Caps Lock in my case) can be chosen to toggle between the English and the Greek keyboard layouts as needed.

It should be born in mind that the above procedure applies only to 12.04 LTS. However, Warren Hill's link details the corresponding procedures for other Ubuntu versions.
