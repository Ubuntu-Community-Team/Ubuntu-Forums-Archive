### Post by nullmind on 2008-04-24
Having troubles with Firefox 3.0 and Xorg eating your CPU? No, I don't mean a few percent here and there, I'm talking about it completely eating your CPU usage and trashing your disk. I have no idea why this happens, but I suspect it's because of the beta development.

If you wish to fix it, for the time being, you'll have to run Firefox in a different profile using this:

firefox -ProfileManager

(You can do this by pressing Alt+F2 and running the above command)

For updates on this situation and any additional information please see my blog links.
