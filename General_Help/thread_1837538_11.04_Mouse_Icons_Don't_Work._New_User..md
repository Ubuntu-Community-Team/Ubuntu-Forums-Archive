### Post by wizwaz3 on 2011-09-01
Hello, I'm new to the Ubuntu universe, so please forgive me if this is something easily fixed. I've installed Macbuntu in order to get my user interface to more closely match that of OS X, my native OS. It automatically changed the mouse cursor to the OS X cursor, but it only shows up in certain applications, such as Firefox. Otherwise, it only shows the default cursor. In addition, the only "default" cursor showing up is the busy icon even though the system is not busy. I can use the terminal if I know what needs to be entered, but I cannot free-range in the terminal. If anyone knows how to fix this problem, please help. I would greatly appreciate it.

### Post by TeoBigusGeekus on 2011-09-02
It's a compiz bug.
Find the name of your mouse theme. It is probably located at ~/home/.themes.
So, if you're using, say, the theme Macbuntu, check to see if there's a folder there called Macbuntu.
If there is, do this:
```
gksu /usr/share/icons/default/index.theme
```
Replace the line
```
Inherits=...Whatever...
```
with
```
Inherits=Macbuntu
```
where Macbuntu should be the name of your mouse cursor theme.
Save, exit, log out, log in, done.
