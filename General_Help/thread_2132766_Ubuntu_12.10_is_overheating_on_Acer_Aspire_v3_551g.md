danieltrotter; Hi !

Many time it is the graphics driver not having good control causing the machine to overheat.
To see your graphics card and info: Post the output of these terminal codes:
```

sudo lshw -C display
lspci -nnk | grep -iA3 vga
```
Then we will see about swapping out the graphic's driver.[INDENT=2]
