### Post by mcduck on 2010-02-11
Extremely customizeable, the only limitation is your own skills.. ;)

If you want to set up a system for persosn who are not really that computer savvy you can simply create a normal user account for them to use. Normal user accounts have no privileges outside of their own home directory. Of course that means that you need to enable automatic updates and have soembody else handle installing programs etc (using admin account).

If that's not enough, Gnome has pretty complete set of tools for locking down the desktop and restricting what kinds of things users are able to do. I suppose that's possible in XFCE as well, but I haven't used that enough to say how easy it might be.

If you need more information you'll need to tell in a bit more detail what you want do.

### Post by waynefoutz on 2010-02-11
You can edit the menus and remove the items from the menu. Actually, an ubuntu install is kind of hard to destroy, as long as the person stays within the confines of mouse clicking and away from the command line.  There are easier distros out there for newbies, gOS comes to mind, but you the installer is going to have to put some work in it getting all the hardware to run, it's no where as easy to set up as Ubuntu is, but it's worth a try to download the iso and try a live disk to see what hardware it recognizes on your system. gOS is an ubuntu based distro that Walmart and Sears were putting on their linux based computers a couple of years ago, because it is geared for people new to computers in general.  It's been upgraded since then, they gave it more of a Mac OS look.
