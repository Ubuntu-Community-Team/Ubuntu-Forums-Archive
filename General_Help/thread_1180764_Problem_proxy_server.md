### Post by blueridgedog on 2009-06-07
The error states that it could not connect to the server.  In an attempt to be helpful, it suggests that you check your proxy server which is a method whereby some people access the Internet.  Since you are not using a proxy server (I assume, as the are a bit outdated for home use and are nowadays only seen in corporations) you can ignore that bit.  The short answer is it could not connect to the site.
