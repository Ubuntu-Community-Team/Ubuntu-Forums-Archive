### Post by anil_robo on 2005-12-18
Install winamp through wine. It plays .nsv (NUll SOft VIdeo) file format.

If you have winamp already installed in your windows partition, mount that and run winamp.exe from program files. Load nsv file in it and done! :)

More details [here!]("http://www.ubuntuforums.org/showthread.php?t=98190")
