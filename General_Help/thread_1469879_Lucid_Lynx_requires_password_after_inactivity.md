### Post by GammaPoint on 2010-05-02
I upgraded to Lucid Lynx a couple of days ago and now when the computer has been inactive for some time it asks for a password to log back in. I wanted to remove this behavior. I went to 'User settings' and made it so that "Password: Not Asked On Login". However, this didn't fix my problem. Instead of showing that a password needs to be entered after inactivity, the screen just stays black. To get out of that black screen I have to enter my password and press enter (although there is no prompt that I should do this, I just guessed). 

How can I change the setting so that a password isn't required? Thanks for the help :)

---

### Post by lisati on 2010-05-02
From System->Preferences->Screensaver, uncheck the option to lock the system when the screensaver is active. If that doesn't help, feel free to let us know.

### Post by jocko on 2010-05-02
That's the screen saver that locks the screen and asks for the password.
Just open gnome-screensaver-preferences (System-->Settings-->Screen saver) and inactivate "Lock screen when screen saver is active".
