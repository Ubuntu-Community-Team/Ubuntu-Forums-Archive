### Post by kyphi on 2007-07-21
If you go to System on your desktop menu, then Administration, then Users and Groups you can add a user and assign privileges to that user.

All users can have their own password.

If you do not want to use a password and go to the desktop immediately when logging on, go to System, then Administration, then Login Window, then Security.  Place a tick into the box next to Enable Automatic Login.

### Post by kyphi on 2007-07-22
The only way that I know of logging in without using your password is to enable automatic login.  A password will still be needed for other tasks that require administrator privileges.

It is part of the security system for Ubuntu.  If it were not so, irreparable damage could be done to your system.

In Windows there is no need for such an elaborate security system because the source code is hidden.

Without a login name and without a password your computer will not be able to determine which preferences to load for the individual user.
