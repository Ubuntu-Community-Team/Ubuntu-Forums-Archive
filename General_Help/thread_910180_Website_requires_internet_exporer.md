### Post by ronnielsen1 on 2008-09-04
Go to Tools and click Add-Ons > User Agent Switcher

After it is installed it will be an option under Tools > User Agent Switcher to simulate windows. You could try that. 

If that doesn't work, you could try running IE. Don't know how well it works.

[http://www.ubuntugeek.com/running-internet-explorer-in-ubuntu-linux.html](http://www.ubuntugeek.com/running-internet-explorer-in-ubuntu-linux.html)
