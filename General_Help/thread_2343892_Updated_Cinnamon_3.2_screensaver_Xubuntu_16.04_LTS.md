HI, I've upgraded to Cinnamon 3.2 which is running well on Xubuntu 16.04.1 LTS.  My question is, where are the config settings to anchor the clock on the screen saver? Prior to this version, the screen saver clock was stationary, now the clock randomly repositions itself across the screen while the screen saver is active. ### Post by silverspr on 2017-03-05
HI, I was able to do this by:


You need screensaver version 3.2.8 or 3.2.7 for this method to work!


run in terminal gsettings set org.cinnamon.desktop.screensaver  floating-widgets false
