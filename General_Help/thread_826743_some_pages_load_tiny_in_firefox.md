### Post by cpetercarter on 2008-06-12
Pressing Ctrl and the +/= key simultaneously is the normal way of increasing the text size on a web page. (Similarly Ctrl and _/- will reduce text size.) I don't know why you are getting small text size on some sites and not others - possibly Firefox has cached the pages or "remembers" the reduced text size settings for them - but you could try Ctrl and +/= on those which are causing a problem and see if that works.
