### Post by matthew on 2005-12-14
In synaptic, install the package "unrar-nonfree". You will probably have to have the [extra repositories enabled]("http://www.psychocats.net/linux/sources.php").

After that, most of the time you can just click the rar archive in the file browser, but occasionally you will need to use a terminal and type```
unrar e filename.rar
```
