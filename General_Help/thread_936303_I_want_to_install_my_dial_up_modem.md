### Post by Irihapeti on 2008-10-03
Whether or not it will work depends on what modem chip it uses. If it's Conexant, then you are in with a good chance. (I know - I've used one for over a year now.)

[https://www.linuxant.com/drivers/modemident.php](https://www.linuxant.com/drivers/modemident.php) has several ways of finding out what chipset you have, including a program that runs under windows.

If you decide to use the installer program (cnxtinstall.run), be aware that it might corrupt your Firefox profile. This need not be a problem if you backup your .mozilla directory **before** you use it; you can replace the directory afterwards.
