### Post by orb9220 on 2007-06-20
> 1.how do i split my existing hardisk into 2 partition so that i can run window on 1 and ubuntu on 1?

During the install you will be able to split or resize the partition. Take note make sure your xp partition is defragged first or gpart may choke or stop unable to resize the partion.

> 2.after installing ubuntu on the other partition, will i be prompt whether which os i wan to boot in?(xp or ubunta)

Yes as long as xp is installed first then Ubuntu will see it and add an entry in the grub menu.

> 3.is an intergrated graphic card able to support beryl? mine is the intel 945gm chip up to 256mb shared.

Most likely since intel graphics are supported. Just search the forums on  [B]intel 945gm 
[/B] to verify and check for problems with this chipset and beryl.

> 4.if integrated graphic card not able to support beryl, does that means i only wont get the cube effect? and can still use the other effects like the burning up of windows when i close the folder?


No because that is what beryl is. So if no beryl support then no cube or burning windows.
