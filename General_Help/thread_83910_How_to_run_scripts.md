### Post by Arktis on 2005-10-29
sh scriptname.sh
or
./scriptname.sh
or for php scripts
php scriptname.php
etc. Sometimes shell scripts don't have the sh extension but that won't matter.  In *nix, filenames count for much less than content.

Scripts need to have executable permissions.
