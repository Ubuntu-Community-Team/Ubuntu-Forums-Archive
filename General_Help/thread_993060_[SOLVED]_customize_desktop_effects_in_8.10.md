### Post by __Ryan__ on 2008-11-25
Install the compiz composite settings manager package:

```
sudo apt-get install simple-ccsm
```

Then find it in your menu here:

System -> Preferences

It will allow you to customize almost EVERYTHING.

### Post by Izek on 2008-11-25
And if you don't like simple-ccsm (Like me) you can install the normal CompizConfig Settings Manager via Add/Remove (When All available applications is chosen in the drop-down.)
