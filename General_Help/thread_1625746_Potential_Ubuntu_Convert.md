### Post by WorMzy on 2010-11-19
The LiveCD and installation CD are one and the same. You can download both 32- and 64-bit versions here: [http://www.ubuntu.com/desktop/get-ubuntu/download](http://www.ubuntu.com/desktop/get-ubuntu/download)


Using the Live part of the CD (you will get a choice when you boot the CD) will allow you to try out the various features that Ubuntu offers without ever needing to install the OS to your hard drive. This gives you an opportunity to play around and see whether you like it or not, without ever running the risk of breaking your existing OS. If you find that you quite like the Ubuntu interface, then you can install it while booted into the Live environment.

If you'd prefer to skip the Live environment and just install straight away, there's an option for that too.

### Post by JohnElway on 2010-11-19
As easy as it is to install off a LiveCD, many new users get tripped with partitioning. For this reason I would recommend using Wubi ([http://www.ubuntu.com/desktop/get-ubuntu/windows-installer](http://www.ubuntu.com/desktop/get-ubuntu/windows-installer)), although this assumes that you're able to get into Windows to install it. If you absolutely cannot startup Windows, then installing off the LiveCD will be your only option (there are many tutorials on how to do this). But if you can, I would really use Wubi because this allows a new user to get a Linux environment up and running quickly and easily. I used a Wubi install without issue for a little over a year before I decided to try a 'real' dual boot installation. To be honest, I didn't notice any major stability or performance discrepancy between the two, but this was just my personal experience with it.
