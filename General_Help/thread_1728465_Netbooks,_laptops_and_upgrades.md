### Post by wilee-nilee on 2011-04-13
10.04 is a longterm release covered for 3 years, 10.10 is the release between the LTS 8.04 was the last LTS. All the releases between 8.04 to 10.04 not including these two are not LTS and are covered I believe for 18 months, as far as repo and security updates.

As far as unetbootin goes, you can load an already downloaded ISO's look at the GUI tick the ISO button then search for the ISO and install. So download 10.10 use the search function in Unetbootin.

If you want real confirmation that any other OS will work I would install it alongside in a dual boot then you will actually know.
