### Post by harbimilan on 2013-01-03
Hi,
I recently installed MATLAB's unix version on Ubuntu 12.04.
It works fine, but there is a little problem: keyboard shortcuts are all different from Windows versions.

For example (in the editor window) :

**Save** : (Ctrl+X) + (Ctrl+S) (chord)
**Copy** : Alt + W
**Paste**: Ctrl + Y

I think the problem is clear:
How do I set these shortcuts back to the proper ones? (if possible, without changing them one by one ?)

Thanks for any help.

---

### Post by harbimilan on 2013-01-04
The settings were in 
File -> Preferences -> Keyboard -> Shortcuts.
Changing "Emacs Default Set" to "Windows Default Set" worked.
