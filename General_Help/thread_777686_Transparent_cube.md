### Post by celticbhoy on 2008-05-01
First of all the background, or skydome as it is called, if you do a google image search for skydome you will get some you can download, they have to be 2048x1024 in size. Go to Desktop Cube --> Appearance Tab --> Skydome option at the bottom.
Turn on the skydome & select the image you have downloaded.

There is also a transparency tab to alter the those settings.

CubeCaps form the CompizSettings Manager will allow you to set and adjust the top and bottom of the cube.

### Post by itix on 2008-05-01
Do you have the CCSM (compizconfig-settings-manager)?
If you do, there will be an entry at the top of the *system => settings* menu called Advanced desktop effects settings.

If you don't, go to the synaptic package manager and install compizconfig-settings-manager.

From there (CCSM), select desktop cube and go to transparent cube. There you can set the transparancy.

The background is also changed in desktop cube under the appearence tab and it's called skydome.

If you want to animate the skydome, you must first find a 360 degree picture and check the box "animate skydome". Reccomended formats are 4096x1024 pixel pictures and can be found [here]("http://www.gnome-look.org/index.php?xcontentmode=102")

The cube "bottom" as you call it is called "cube caps" in CCSM and can be changed from the cube caps menu entry.
