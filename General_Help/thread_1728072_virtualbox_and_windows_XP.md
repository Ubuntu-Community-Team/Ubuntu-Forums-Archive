### Post by 3Miro on 2011-04-13
You install XP the exact same way you install it on a desktop.

First you need to make a virtual machine and give it a .vdi harddrive, then you put the CD in, tell Virtual Box to use "Host CD" then you boot the system. Then you install XP as you would normally (including swapping CD).
