### Post by Wayne_V on 2008-12-11
from 'man bash':

When an interactive shell that is not a login shell is started, bash reads
       and  executes commands from /etc/bash.bashrc and ~/.bashrc, if these files
       exist.  

So, following the recommendations in .bashrc, you should uncomment the lines regarding sourcing .bash_aliases and create that file for your own customizations.
