### Post by lovinglinux on 2009-09-16
I recommend a clean install. If you have a [separate home partition](http://www.psychocats.net/ubuntu/separatehome), then you should get most of your system to the state before 9.10 installation. If you edit configuration files outside your home, then you will have to edit them again after the clean install.

To install all applications you already have, use the Umarks extension for Firefox from my signature.
