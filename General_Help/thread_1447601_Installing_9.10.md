### Post by ajgreeny on 2010-04-05
You will not get Ubuntu to run on a machine with only 144MB of ram, I'm afraid, so eithre get some more ram installed if the motherboard will accept it, or perhaps preferably, try another distro based on Ubuntu, such as Lubuntu (ubuntu with the low resource requiring lxde desktop) or even a completely different distro, such as Puppy.
[http://puppylinux.org/main/index.php?file=Download%20Latest%20Release.htm](http://puppylinux.org/main/index.php?file=Download%20Latest%20Release.htm)
[http://puppylinux.org/main/index.php?file=How%20NOT%20to%20install%20Puppy.htm](http://puppylinux.org/main/index.php?file=How%20NOT%20to%20install%20Puppy.htm)

Puppy may actually be the best for a machine like that, and you don't even have to install it, it will run from a live CD, but then offer to save the major boot files to the hard disk when you shutdown first time, along with all your personal files and configurations, so next boot it will start up as if it was a hard disk install, not a live CD.  You can install to disk normally, if you want, but it is not really any better, so I suggest you try it as live CD, and see what you think of it.
