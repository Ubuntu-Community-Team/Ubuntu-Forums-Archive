I changed my tune after re-reading the OP and looking up this win32 disk imager...

I think you are sort of barking up the wrong tree,
Unetbootin and Disk Imager are each for their own different purposes.

Disk Imager copies an image file to a device, which is why it wants an *.img file.
Unetbootin converts an ISO for use from flash storage and writes it too.

If you really can't get Unetbootin to work, your best bet is going to be to download
a pre-made *.img such as this one for Ubuntu 9.04 Netbook Remix:
[http://releases.ubuntu.com/jaunty/ubuntu-9.04-netbook-remix-i386.img](http://releases.ubuntu.com/jaunty/ubuntu-9.04-netbook-remix-i386.img)

By the way, if you have any access to an already working Ubuntu installation,
you can just use the builtin tool at "System -> Administration -> Startup Disk Creator"
