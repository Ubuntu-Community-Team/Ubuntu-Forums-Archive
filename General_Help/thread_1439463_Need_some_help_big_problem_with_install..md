If you're serious about installing Ubuntu Server 9.* I encourage you to send an email to the [help grub mailing list]("http://savannah.gnu.org/mail/?group=grub").

If you manage to solve the problem, it would be great if you posted the solution here as it might help someone else (it would certainly enlighten me!).
