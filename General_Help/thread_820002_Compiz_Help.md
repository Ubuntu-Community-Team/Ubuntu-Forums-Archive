### Post by jpeddicord on 2008-06-05
If you are using 7.10 or above, there is no need to install Compiz, as it is available by default from System > Preferences > Appearance; Visual Effects.

If you would like to configure custom animations, install the package ccsm. (See the forum FAQ for information.) You can access it from System > Preferences > Advanced Desktop Effects Settings.
