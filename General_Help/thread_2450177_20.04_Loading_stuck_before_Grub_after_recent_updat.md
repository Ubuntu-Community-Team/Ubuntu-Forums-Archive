### Post by oldfred on 2020-09-09
Did you install the nVidia driver for your version?

> GRUB_CMDLINE_LINUX="nouveau.modeset=0"
You normally then do not use nomodeset as permanent setting. That is only needed until you install correct nVidia driver and may interfere with boot if you still add it. Normally you just manually add it when booting the first time or until you install the nVidia driver.

### Post by oldfred on 2020-09-11
If you report a bug they want lots of details.
Many bugs get auto reported on crash and then all the logs & errors are uploaded.

First check if any reported bugs are similar. Sort by number will be most recent.
[https://bugs.launchpad.net/ubuntu/+source/grub2](https://bugs.launchpad.net/ubuntu/+source/grub2)
