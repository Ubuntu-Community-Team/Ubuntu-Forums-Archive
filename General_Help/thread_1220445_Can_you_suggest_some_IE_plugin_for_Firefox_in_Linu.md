### Post by Rocket2DMn on 2009-07-22
You can't actually open a page rendered by IE in Firefox in linux because IE is not on the system.  If you just need a webpage to *think* that you are running IE, you can use the User Agent Switcher plugin.

If you actually need IE, you need to run it in Wine, or install a virtual machine with Windows in it.  ies4linux (uses IE6) - [http://www.tatanka.com.br/ies4linux/page/Installation:Ubuntu](http://www.tatanka.com.br/ies4linux/page/Installation:Ubuntu)
