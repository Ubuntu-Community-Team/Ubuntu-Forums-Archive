### Post by chinocracy on 2007-03-12
I think Stardict uses Gnome, while Stardict-GTK uses GTK runtime environment. Basically, different runtime environments. I just used Stardict. Yes, the -common is needed, as that contains the basic system files.

### Post by Marsolin on 2007-03-12
If you are using Gnome you want the stardict package. If you are not you want the stardict-gtk package.
