You dont need to create directories, in fact, im not sure that will work.

You could go about it by doing this:

System>preferences>main menu

Then go to whatever section you want to put the launcher under.

Next click "new item" and fill in the information for the launcher. you will need to select 'application' and then type the name
the command is the most important part. If Im not mistaken, MatLab tells you what the command to run it is.. if not, you could just point that command to the location of the matlab binary that you want to run. 

In my example, its Maple, but its the same deal.

You could assign an icon too, just make sure you dont lose or move the icon image. 

and youre all set!
