### Post by macaholic on 2009-10-26
I'm not sure about DMZ white specifically, but you could install some extra cursor themes:```
sudo apt-get install comixcursors gcursor
```
There are "left-handed" ones in that pack I believe. Maybe you will find one you like.
EDIT: I forgot to add, gcursors is a nice little utility that lets you select your pointer, in System>Preferences>Cursor Selection.
