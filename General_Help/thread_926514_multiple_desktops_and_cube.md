### Post by NoReflex on 2008-09-22
Please install Simple Compiz Config Settings Manager:
```
apt-get install simple-ccsm
```
And there you can change very easy the number of desktops. Go to General Options -> Desktop Size and set Horizontal Virtual Size = 4, Vertical Virtual Size = 1 and Number of Desktops = 1.
Hope this helps!

Best wishes,
NoReflex
