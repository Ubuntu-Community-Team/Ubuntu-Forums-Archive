### Post by FluffyElmo on 2009-01-07
I've installed the 32bit version of 8.10 without problems. The 64bit support is quite recent so if you don't need 64bit for some reason 32bit would be the way to go.

Also, in the past when I've had problems with the live cd installer I've downloaded the alternate installer and that worked. It used to be on the same disk but now it's a separate download. You may want to give it a try as it's text based and likely won't have the same problem. It may have *different* problems though;)
