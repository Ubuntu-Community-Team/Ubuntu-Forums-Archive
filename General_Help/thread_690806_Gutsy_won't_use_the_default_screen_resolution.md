### Post by pdub on 2008-02-07
Change the following section from:

```
Depth 24

Modes "1024x768" "800x600" "640x480"

Virtual 1280 1024

EndSubSection

EndSection
```

to

```
Depth 24

Modes **"1280x1024"** "1024x768" "800x600" "640x480"

Virtual 1280 1024

EndSubSection

EndSection
```
