### Post by scxtt on 2007-02-15
vmware {Server or Player} will allow you to run a virtualized "Windows" environment into which you can install windows-related programs - it's not like Wine where you just install programs ... vmware Player won't do you much good if you don't already have a functioning Virtual Machine, altho a site like easyvmx.com can help you create a VM to be used w/ Player --- your best bet is Server tho ...
