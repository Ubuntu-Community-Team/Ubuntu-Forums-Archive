### Post by Basher101 on 2012-01-21
you can format anything if you want. The actual question as it seems is what use will that give you...linux can read and write to NTFS partitions that Windows uses...

so all you will need to install ubuntu is format a single hard drive and make a ~ 20 GB minimum ext4 partition and a SWAP partition (you may not need SWAP if you have 4 GB+ RAM). I will gladly help you with this, and in case you cannot cope with the explanations i would also be glad to help you with remote control.
