### Post by ajgreeny on 2009-01-26
Try with compiz disabled ```
metacity --replace
```and you may find it runs with the proper window decorations again.  It could be that you need to play around with the combination of compiz plugins you use in order to get the program(s) running when compiz is also running, but that is a bit difficult, except by trial and error.  To re-enable compiz it will be ```
compiz --replace
```
PS  Both these commands can be run in the Alt+F2 dialog box as well as the terminal.
