### Post by Hagar Delest on 2009-11-20
If tar doesn't find the file, it just means that the name of the file is not correct. Have you used the Tab key to autocomplete the filename? That's the better solution to avoid typing mistakes. The filenames given are for an old version so don't copy/paste the complete command line, just the beginning and change the filename.

Note that if you want to install the Sun version, you have to remove the Ubuntu one.

And don't worry, I also run xubuntu and never had any problem to install the Sun version.
