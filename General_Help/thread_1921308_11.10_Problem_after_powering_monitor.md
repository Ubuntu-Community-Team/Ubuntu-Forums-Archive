Hi 

First of all lets find out if it the PC box or the monitor Or even a combination of them both

All you have to do is take the monitor from the faulty set up and swap it with one from a working set up 

If both now work its a combination fault, something borderline about the pair, it works so don't worry 

If the fault moves with the original monitor and the new monitor works on the pc that did not work in the first place, it's the monitor that's faulty. Replace it 

If the fault stays with the original PC box and the new monitor then you know it's the PC hardware in that original box or a software problem 

Please post the results
