### Post by s.fox on 2010-01-05
Hello,

I would try running it through WINE.  Some versions of the software appear to be compatible.  Information can be found [here]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=3467"). 

A video editor that I like to use is kino.  To install it open a terminal and enter this command:
```

sudo aptitude install kino
```

-Silver Fox
