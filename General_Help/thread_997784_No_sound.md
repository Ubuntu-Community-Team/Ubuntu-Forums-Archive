### Post by khelben1979 on 2008-11-30
I downloaded the source from the [Alsa project homepage]("http://www.alsa-project.org/main/index.php/Main_Page") today and then I used an alsa script to get it to work over here.
You'll find the script [[COLOR="Red"]here[/COLOR]]("http://ubuntuforums.org/showthread.php?t=962695&highlight=alsa+script").

Also, make sure to use Alsaconf to set up your audio device and the different speaker applications must have a speaker meter where you are able to raise the volume on the sound.
