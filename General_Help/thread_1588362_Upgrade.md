### Post by snowpine on 2010-10-04
If you are concerned with stability, I recommend thoroughly testing 10.10 as a Live CD before even considering an upgrade.

The answer really is "more stable for some users, less stable for others, equally stable for many." There will be new bugs in the new release, but then again some old bugs will be fixed. 

Certainly don't upgrade a perfectly-working system on the 1st day of the new release. Wait a little while, hang out on these forums to read others' experience, and test test test the Live CD.
