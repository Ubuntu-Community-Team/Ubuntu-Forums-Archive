### Post by steveneddy on 2009-09-07
Undo everything you have done so far (uninstall the printer).

Install HPLIP from here:

[http://hplipopensource.com/hplip-web/gethplip.html](http://hplipopensource.com/hplip-web/gethplip.html)

Follow the install instructions on the page linked above.

Install printer from HPLIP and you should be fine.

This is software from HP and it will give you better control of your printer and help you manage your printer easily.
