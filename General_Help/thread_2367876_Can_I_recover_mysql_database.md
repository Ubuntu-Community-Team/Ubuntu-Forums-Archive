### Post by TheFu on 2017-08-04
the easy answer is yes, if you have a backup.  Running a linux install or DB on flash memory isn't exactly the safest thing.  Flash storage fails with little warning and is highly dependent on the architecture of the storage for longevity.

Backups should follow a 3-2-1 method.
