This is a relatively slow processor but I have two computers with slower and YouTube is one thing that ran fine once Pepper Flash was installed (Pepper Flash should not be needed anymore, this was before the YouTube conversion to HTML5 -- it continues to run fine afterwards for me).  _**If you haven't tried it yet go into a video, click the gear button and select 240p, that may take care of your problem.  If it does, click the blue icon at the top right on your YouTube page, click the gear icon in the box that appears, click Playback option at the left of your screen and select "slow connection" to keep the computer from playing videos at high bit rates.**_  If that works your problem is likely a slow internet connection.  From your somewhat cryptic comment it sounds like YouTube is working but not smoothly so you probably have the necessary HTML5 driver already to use that site (not having the driver would cause video to not work at all), leading me to give the above advice.  You have 2GB of RAM, that isn't enough to load some sites smoothly but IME YouTube alone does not use that much RAM so I think I can rule that issue out here.

Is it possible for you to enter the terminal and type in a command?  It would help us if you could get and post the full specs of this computer.  Without more information I am personally at a loss as to how to advise you further.  The command is below:  ```
sudo lshw
```

You can type the specs into your text box for this forum (unfortunately it is not possible to copy and paste from the terminal to this forum, I have tried to do so several times), highlight them and click the # in the second line of icons above the text box to create a box with this info inside (as I did above with the terminal commands).
