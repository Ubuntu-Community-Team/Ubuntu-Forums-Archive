### Post by huwnet on 2008-01-04
Personally I'd recommend Jabber as it is a more open protocol (the two main jabber 'providers' are Google Talk and jabber.org).

If you'd rather use AIM / ICQ then I'd recommend AIM. ICQ and AIM are both owned by AOL but ICQ has been treated like a second class citizen since the takeover.
