### Post by itang sanjana on 2009-12-17
RPM — RPM Package Manager, originally from Red Hat, is now used by many other distributors as well, like openSUSE and Mandriva Linux. [1]

deb — Debian package, originally from Debian. Used by Debian and derivatives such as Ubuntu. [2]

So Ubuntu is .deb "category"

[1] [http://en.wikipedia.org/wiki/RPM_Package_Manager](http://en.wikipedia.org/wiki/RPM_Package_Manager)
[2] [http://en.wikipedia.org/wiki/Deb_(file_format](http://en.wikipedia.org/wiki/Deb_(file_format))

### Post by pricetech on 2009-12-17
RPM originally stood for RedHat Package Manager, but when it got picked up by so many other distros and became something of a standard, it was renamed RPM Package Manager.

RPMs can be installed using alien, but the DEB packages are preferred.

The repositories are really your best source though, especially just starting out.
