### Post by Sutekh on 2006-04-27
Check out the Character Map (Applications -> Accessories menu).

Go to View and then By Unicode Block.  You should be able to find the symbol you need and the shortcut key, which is listed down the bottom.

For example,

U+00A3 - means Ctrl+Shift+A+3


You could also consider adding a Character Pallete to a panel, which can be used to select a character when you wish.  Right-click a panel and choose Add to Panel, then find the Character Palette.

To use if simply click on the character you want, and middle-click you mouse to paste it where you are working.

Hope these can help.
