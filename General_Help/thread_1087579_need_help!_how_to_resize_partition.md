### Post by skymera on 2009-03-05
Boot the Ubuntu LiveCD and then you can choose the Gnome Partition Editors (gParted)

System > Administration > Partition Manager (Or similar)

There, you can remove,resize and create new partitions.
Once it's done it should restart and you can boot Ubuntu again :)
