### Post by inameiname on 2010-05-31
Install Remastersys (you'll need the Remastersys repo first to download). It's the easiest way for me to do what you want.

And then open it, and hit "Backup", and it'll make an ISO just like that with absolutely everything that you have on your computer, including whatever "My Documents" you have too. 

You can then just insert the custom live cd/dvd, made with the ISO file, and install it on any computer, just like the original live cd ISO that you used.

### Post by ryan858 on 2010-05-31
Yes, in fact it would only work with 32 -> 64, not the other way around. Since a 64-bit processor is backwards compatible with 32-bit stuff.

There might be a few problems with hardware differences in the different comptuers, but Linux is quite robust, so it'll probably just fix itself.
