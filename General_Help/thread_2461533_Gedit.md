### Post by TheFu on 2021-05-03
In general, PDF files cannot be edited.  They can have annotations added and have pages removed, added and page order changed.

**Xournal** is a way to add annotations, then Export to a new PDF file ... which can appear to be "editing".  There are a number of other tools. The best is a commercial tool that runs $50. I can't recall the name anymore. Sorry.

**LibreOffice** can parse some existing PDF files and appear to "edit" them as well. Because PDFs can be images (jpg/png/gif/tiff) wrapped in a PDF container, editing those isn't possible. Nothing can change that. It is almost always sufficient for most needs.
