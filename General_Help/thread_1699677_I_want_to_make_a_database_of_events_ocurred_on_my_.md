### Post by nickleboyblue on 2011-03-04
If you know sql, you can use mysql, postgresql, sqlite, etc.  Those are the most powerful database applications that are available in the repositories.  If all you want to do is store information and you aren't looking to write a program, try out OpenOffice.org Database.  It has a decent gui and allows you to store all sorts of information.  It isn't installed by default, so you'll have to find it in the software center.
