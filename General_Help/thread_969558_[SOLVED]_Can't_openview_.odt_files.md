### Post by timbalisto on 2008-11-03
I use an up-to-date Hardy Heron. I can write and save an .odt document in OOo, but I can't open it to view or edit. I can edit and view the document if I rename the file suffix from .odt to .doc . If I try to open an .odt file it says within the window "gedit has not been able to detect the character coding.

I was able to edit and view these documents a few weeks ago, I just now tried to view them. It's not a major problem(yet), but it's a hassle to save only in a certain format or rename all the documents. Any help is appreciated, thanks

### Post by drs305 on 2008-11-03
Open nautilus or other file browser, right click on an .odt file, select "Open with Other Application" and select OpenOffice.

If you recently installed OpenOffice 3, the selection would be "OpenOffice.org 3.0 Writer".
