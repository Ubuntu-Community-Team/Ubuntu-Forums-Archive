### Post by deadflowr on 2017-05-31
> **Kris_M said:**
> yeah, chromium - should I be on chrome? no it looks like chromium is included in ubuntu, not chrome.

Yeah chromium is the free software version which ships with Ubuntu (through the software center, et al)
No need to install chrome if you do not want it,
What method of installing flash did you do originally?

Getting flash on chromium should be as simple as
1)open software and updates and go to other software and enable Canonical Partners.
2) install adobe-flashplugin
```
sudo apt install adobe-flashplugin
```
and it should have the most current version you can use.


Edit: you cannot install rpm's on Ubuntu without first running them through a program called alien, which can convert it to the debian package format that Ubuntu uses.
