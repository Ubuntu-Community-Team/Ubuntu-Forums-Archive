### Post by Topsiho on 2011-07-17
If you downloaded the correct file, it is a file with the extension .iso, called an image file. The folder where in it's downloaded depends on the configurations for your browser. You can also find it using locate *.iso, in a text screen (maybe you'll have to do sudo updatedb first), or by searching for it in your home folder, probably in the folder Downloads.

If you have found  it you'll have to burn it as an image, NOT as a data file, using brasero or K3B or else, or, indeed, use it for an USB-stick.

I assume here that you are using Linux, elde you should ask the question on let's say a Windows forum :)

Topsiho

### Post by Topsiho on 2011-07-17
Your remark you downloaded a zip file I don't understand. You should download from Ubuntu itself: [http://www.ubuntu.com/download/ubuntu/download](http://www.ubuntu.com/download/ubuntu/download).

Here you can select whether you want Ubuntu 11.04 (the latest one, considered buggy by some), or the LTS version, 10.04, which is far more stable, and lasts longer.

Then you have to select whether you want the 32 bits or 64 bits version. The first is always OK, if you use a PC, the latter the same, if you have a 64 bits processor, as many (most?) newer computers have.

The CDRom or USB-stick that you make from the downloaded .iso should be checked for errors (one of the menu options you get after starting the computer), to be sure everything is OK.

Topsiho
