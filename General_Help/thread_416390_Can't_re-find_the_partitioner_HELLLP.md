### Post by NikoC on 2007-04-21
Do you mean Gparted?
It should be in your main menu --> system --> administration --> gnome partition editor

Or you could open a terminal and type:
```
sudo gparted
```

A nice cd to 'rearrange partitions' is Gparted live, it's a live cd that requires 50 mb of downloading. You can find it [here]("http://gparted.sourceforge.net/livecd.php").
