What exactly is the problem you are having as I did not see your other post to my knowledge. If you are just having problems downloading the file when installing, I would suggest downloading the actual ISO from [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) for Ubuntu, [http://www.xubuntu.org/get](http://www.xubuntu.org/get) for Xubuntu and [http://www.kubuntu.org/getkubuntu](http://www.kubuntu.org/getkubuntu) for Kubuntu. When you are done getting the ISO from one of those sites then put that ISO file in the same folder as wubi.exe, then turn off your internet connection and run the wubi.exe. This should allow the installer to use the ISO you just downloaded.
