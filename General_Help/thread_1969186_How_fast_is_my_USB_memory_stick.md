### Post by pqwoerituytrueiwoq on 2012-04-29
you will have to wipe it to do a write benchmark (at least with the disk utility)
flash drives are like ssds and do have write limits so benchmarking write speed does use up some of its life

if this is a usb 3.0 be sure you are using a usb 3.0 port otherwise the speed will be caped by the usb 2.0 limit
