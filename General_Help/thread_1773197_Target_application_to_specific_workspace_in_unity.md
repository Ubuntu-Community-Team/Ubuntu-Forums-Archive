I have certain programs that I like to keep on certain workspaces.  Is there a way to edit the launcher so that the application launches to a specific workspace?

---

### Post by Larkspur on 2011-06-01
If you've got CompizConfig installed, go to Put windows and selected the Fixed Window Placement tab.  At the bottom is Windows With Fixed Viewport.  Add the programs you want to it, and you'll be set.
