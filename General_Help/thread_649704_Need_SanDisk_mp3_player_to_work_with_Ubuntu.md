### Post by dasunst3r on 2007-12-25
Even if you have it set to MTP, you *might* be able to have Rhythmbox or AmaroK detect and work with it by installing *libmtp* or something like that.  However, if you like a simple MP3 player where all you need to do is to drag-and-drop (my favorite kind), then you'll want to change the USB settings to use MSC.
