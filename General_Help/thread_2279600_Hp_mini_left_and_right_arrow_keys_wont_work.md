### Post by kerry_s on 2015-05-24
i have the hp mini 210-1010nr, the os that works best is xubuntu 15.04 32bit, all the keyboard keys work correctly out of the box.

i'm currently messing with ubuntu-mate, so far i only notice the display(f4) key opens the menu instead of display, other than that the keyboard also works fine.

if you got the 2gb upgrade & ssd drive, these little hp mini's fly.

also don't forget to install zram-config to compress the ram, it'll just run smoother.

are cpu's can run 64bit, but the 32bit runs much faster & smoother, you'll notice in how long it takes apps to open too, with 64bit it takes much longer.
