### Post by snowpine on 2010-10-01
Yes! :)

Ubuntu is a great beginner's distro, but it also has the same underlying system as any other Linux distro. So most of the command-line skills you learn in Ubuntu will also apply to other distros as well. There are a few things that are distro-specific (the "apt" package system for example) but don't let that stop you.

When you are comfortable with Ubuntu and ready for an intermediate project, you can create "virtual machines" and, for example, [create a virtual server install ]("http://www.linuxjournal.com/content/consolate-put-your-servers-virtualbox-vm")and practice networking! In other words, once you have a stable Ubuntu install, you can use it as a base to experiment with other distros (like Debian, Fedora, etc.) and expand your knowledge. Fun stuff. :)
