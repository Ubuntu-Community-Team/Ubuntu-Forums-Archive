### Post by bodhi.zazen on 2009-12-20
I will assume you are asking for the server edition with some kind of a graphical interface.

Traditionally , Linux servers run without a graphical interface.

You have two options, install the Desktop edition and than add the server you want to run on top. This will work but is a bit subopotimal (servers do not need X).

The other option is to use one of the web interfaces such as webmin. There are several depending on what you need.
