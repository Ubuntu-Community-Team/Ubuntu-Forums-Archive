### Post by ajgreeny on 2011-07-21
If you want to be really geeky, you get all the info you want with the terminal command ```
top
```Use upper case P to sort by cpu %, and upper case M to sort by memory use.

It may seem odd to you to begin with, but it is much better than System Monitor or conky, in my opinion, both of which use more of both cpu and memory than "top" will.

I do, however, use conky on my desktop, because it is useful for other information, which shows like the screenshot, and I can let you see the config file to get that if you're interested.
