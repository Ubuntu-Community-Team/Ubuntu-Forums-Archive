### Post by mb_webguy on 2009-01-28
You can change the color of the font used in the panels by going to System->Preferences->Appearance and customizing your theme.  The panel text color is controlled by Windows Text.  Unfortunately, Windows Text controls the color for all window text, not just the text of the panels.

[Here]("http://ubuntuforums.org/showthread.php?t=334570")'s a way to change just the panel font color (and lots of other things about the panel's appearance), but it requires creating a configuration file.
