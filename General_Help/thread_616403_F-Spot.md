### Post by monktbd on 2007-11-18
F-Spot user here.

i already use it for quite a while now and it constantly improved.
important is to decide for yourself first how you want to store the pictures on the harddisk(s)., like using the photos folder of f-spot or some other setup you prefer yourself before you start tagging a lot of pictures.

i have for now an avergae amount of images tagged (about 5500) and the performance is still ok although my comp is quite old. 

f-spot is open and free and native which makes it - for me - superior to picasa.
