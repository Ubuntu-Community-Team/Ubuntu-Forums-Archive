### Post by tiggerntatie on 2014-04-25
None of the configuration settings for blanking the screen or suspending after inactivity seem to have any effect.

After installing Light Locker I was able to blank the screen, finally, but I am still not able to suspend automatically.

Furthermore, configuring Light Locker to require password after screensaver activation seems to prevent the blanking function from working. 

Is there a way to figure out whether this is an idle detection issue, or a configuration issue?

EDIT: Light Locker is not a solution. The time-until-blank keeps resetting to "never" all by itself. The "turn off screen when inactive for..." setting in configuration is still set for one minute.

SOLVED, PEBCAK: Chrome "Keep Awake" extension was accidentally set to "keep awake," and was performing flawlessly.
