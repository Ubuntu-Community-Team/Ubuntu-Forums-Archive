### Post by dino99 on 2012-06-27
you can make some room by running these commands from a terminal:

sudo apt-get clean
sudo apt-get autoclean
sudo apt-get autoremove

then boot on a livecd (liveusb) to run gparted, and set your partitions close to this standard install:
[http://ubuntuforums.org/showpost.php?p=11843514&postcount=9](http://ubuntuforums.org/showpost.php?p=11843514&postcount=9)
