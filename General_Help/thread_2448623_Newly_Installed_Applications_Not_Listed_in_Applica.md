---

### Post by Dennis N on 2020-08-11
Possible reasons:

1. Application is on another page of the grid (use pager on right to switch grid pages).
2. Application is in an AppFolder tile (example: Utilities)
3. Application has no .desktop file in the searched locations.
4. You are only looking at Favorites.
