### Post by Sef on 2006-04-07
> When I try to click on any of the options in the drop down menu it doesn nothing. What should I do.

You could update from the terminal: Applications > Accessories > Terminal

then to install your updates:

sudo apt-get update

sudo apt-get upgrade

> does telling it to install those just add them into the package manager, not "really"installing/activating them, or will installing all updates alter my system and setting/configurations?

The updates should not affect your settings, but there is no 100% guarantee that no problems will arise.  Keep your system up to date.  Patches are added, and newer versions are added as well.
