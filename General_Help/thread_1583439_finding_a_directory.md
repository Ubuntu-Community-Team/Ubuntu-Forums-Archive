### Post by mjneil.81 on 2010-09-27
you can go to the place menu on your menu bar and click on computer which will bring up nautilus file browser then you can type the directory into the location bar (Ctrl l to bring it up if its not there) or you can use terminal which is found in applications>accessories>terminal and use the 'cd /your/directory/location/' note the space between cd and the / is necessary.
