### Post by starfry on 2007-07-13
Hello,

YES you can do what you want.  Virtualbox only supports virtual hard drives but if you use VMWare Player (also free) you can use real hard drives and this allows you to boot your dual-booted OS into a VM.

There is a thread specifically about this [http://ubuntuforums.org/showthread.php?t=380699](http://ubuntuforums.org/showthread.php?t=380699)

Good Luck!
