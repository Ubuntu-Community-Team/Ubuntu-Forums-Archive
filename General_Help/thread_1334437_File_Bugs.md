### Post by fluffman86 on 2009-11-22
First do a search to see if you can find someone else who has already filed your bug.  If you don't see it on the search screen, there is a "report a bug" button to the right of the search bar.

If you already know the program or project that you're filing the bug against (you *should* know that) then you can click Help > Report a Problem on a graphical program, or use "ubuntu-bug programname" from the command line.  This will automatically collect important system information about the problem you're having and send it to launchpad.
