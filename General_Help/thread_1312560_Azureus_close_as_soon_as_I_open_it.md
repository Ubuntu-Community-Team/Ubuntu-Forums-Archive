### Post by kellemes on 2009-11-03
Type ALT-F2
You'll get a commanddialog..
Now type..
```
gnome-terminal
``` and press [ENTER]

From the terminal-window start typing..
```
azureus
``` and press [ENTER]

Could very well be Azureus will throw back some error-messages you can work with..

Maybe there is some garbage in the azureus-settings, you can try the following from a terminal-window and see if this works..
Purge and reinstall Azureus..
```
sudo apt-get purge azureus
sudo apt-get install azureus
```

Good luck
