### Post by .nedberg on 2009-05-19
Are you trying to use VNC/Remote Desktop or just access some files on the Windows machine?

I connect to my parents Windows machine with VNC. They start TightVNC on their machine and I connect from my (K)Ubuntu machine. I had to configure their router to allow the connection from outside.

If you just want to access some files then you will need to run some kind of file server on the Windows machine (ftp would be one option).
