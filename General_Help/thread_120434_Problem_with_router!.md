### Post by nkhansen on 2006-01-21
If it is a ubuntu router, you should be able to restart the router and boot it in failsafe mode (I don't remember how, but you should be able to find it in these forums or the wiki). Then you should be able to change the interface to another port (i.e. 81). 

If it is a embedded router, you could try to reset it to factory settings.
