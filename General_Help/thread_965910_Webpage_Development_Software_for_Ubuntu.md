### Post by Alaric on 2008-10-31
Most people around here don't want to admit it... but there's not really any good Linux webdesign program.  Nothing stacks up to good ol' Dreamweaver.  If you're a college student, you should be able to pick up a copy on the cheap from your school.  Dreamweaver is, of course, Windows only, but I've heard of people that have been using it with Wine under Linux for a few months  (almost) legally and haven't experienced any problems.  Unfortunately, Dreamweaver's licensing program (usually) won't let you run the "normal" version of it.  You'll probably have to install it on Windows machine first, then copy it over (unless you download it from a source besides adobe...).  If you're a major developer or a studying college student, this would probably be the best solution, and I doubt you'd get in trouble for disabling the copy protection for Linux if you bought a legit license.  If you're a small time/hobbyist developer, then something like Quanta or Kompozer would probably suffice, but if you're looking for the full web development experience, nothing beats Dreamweaver.  

P.S., you could also look into Crossover Office instead of Wine; I *believe* this can run Dreamweaver 100% legal... for a price.  Both Crossover Office and Wine can run Photoshop as well.
