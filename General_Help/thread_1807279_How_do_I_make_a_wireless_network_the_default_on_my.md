How do I make a wireless network the default on my Starling netbook

I connected to the wrong neighbor, which was encrypted.  Anyway, that is a long story to explain I was not unethically trying to hack into anybody's network.  Now to my problem:

### Post by garvinrick4 on 2011-07-18
Go to Network manager applet and then to edit connections to wireless and highlight the neighbors SSID and delete it.
Make sure your SSID has the connect automatically box checked.

### Post by Cybie257 on 2011-07-18
Assuming you are using Ubuntu Desktop...

Go to : System -> Preferences -> Network Connections
(Or Click the Network icon and select "Edit Connections")

In the Wireless TAB, delete the one(s) you don't want. Or, better yet, EDIT the connection(s) you don't want and DE-Select the Connect Automatically box. That way, it won't try to connect or even rebuild that connection if you delete it.
