### Post by Cheesemill on 2013-03-09
You launch installed applications by searching for them in the dash.

To add an application to the launcher just drag it to from the dash or if it is already running you can right-click on it to lock it to the launcher.

You should have a read of the official documentation...
[https://help.ubuntu.com/12.04/ubuntu-help/index.html](https://help.ubuntu.com/12.04/ubuntu-help/index.html)

### Post by plucky on 2013-03-09
Assuming Ubuntu and not Kubuntu, the top icon on the launch bar on the left is called the **Dash**
If you select it, you should be able to navigate around to see all the installed applications.

**Ubuntu Software Centre** is used to install new applications.

Good Luck

### Post by deadflowr on 2013-03-09
Quick tip, the keyboard control 'super +A will open the dash directly in the applications section of the dash.
Simply drag and drop launchers to the sidebar, or when launched find the newly added icon and right-click it, then select lock to launcher.
Icons in the launcher(sidebar) can be rearranged.
