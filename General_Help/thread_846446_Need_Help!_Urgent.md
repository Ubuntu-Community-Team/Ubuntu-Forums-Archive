### Post by sande87 on 2008-07-06
iv dualbooted MS and Ubuntu. It worked fine. I loaded the Ubuntu LiveCD, then I launched the installer. And when I was asked to partition my drive, I chose the use avaliable free space option. it worked, but my MS partition wasnt too happy afterwards. 

if you want to dualboot MS and Ubuntu I would advice you to backup all you data, install MS first and give MS half the harddrive space, and then install Ubuntu and let ubuntu have the other half. You can partition any way you want, but I would use atleast 8 gigabytes for ubuntu. 

if you are having problems with the installer, then download the Ubuntu 8.04.1 Hardy Heron burn it and boot the LiveCD. Like I said, the easiest way to dualboot MS and Ubuntu is to install MS first, and then install Ubuntu, as Ubuntu detects the MS partition and will set up the bootloader automagically for you.

feel free to mail me if you have more questions or want more help :)
