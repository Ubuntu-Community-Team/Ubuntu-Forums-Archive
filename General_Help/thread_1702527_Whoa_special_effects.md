### Post by stinkeye on 2011-03-08
Install **compiz-fusion-plugins-extra** 
```
sudo apt-get install compiz-fusion-plugins-extra
```

then in **compizconfig-settings-manager**
enable the animations add-on plugin and configure your effects 
in the animations plugin.
