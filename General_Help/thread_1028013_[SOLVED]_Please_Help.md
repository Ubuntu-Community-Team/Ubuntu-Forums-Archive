### Post by lswb on 2009-01-02
Using the same "Add to Panel" menu, add the "Notification Area" NM Applet is not really a panel applet but a "notification applet" or system tray applet, and it needs the notification area on a panel to make itself visible. If it doesn't show up right away. try Main Menu/System/Preferences/Sessions and make sure NM applet is enabled. If it still doesn't appear, log out and back in (or reboot)
