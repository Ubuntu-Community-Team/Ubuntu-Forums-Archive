### Post by IWantFroyo on 2011-02-25
Yes, Ubuntu is good for most users. Unless you are totally tied to some Windows software, Ubuntu can work for you. Just be understanding if something goes wrong. Many of Ubuntu's packages are from Debian Sid (unstable), so some things break.
As for the software:
Photoshop works in Wine (winehq.org). I'm not sure about the others, but if they don't work with Wine, you could make a virtual machine for windows.
