### Post by snowpine on 2008-07-30
Hi there, here is a good guide to installing "the base system": [http://www.psychocats.net/ubuntu/minimal#barebones](http://www.psychocats.net/ubuntu/minimal#barebones)

You have many questions; slow down and do some research before you install anything. There is an Ubunbu Live CD that you can try without changing anything on your hard drive. That way you can learn how Ubuntu works and see if it's right for you. Good luck!

ps "root information" is not needed to install Ubuntu, not sure where you got that idea. You set your own username and password when you do the install, and if you are using the Live CD, there is no username/password.
