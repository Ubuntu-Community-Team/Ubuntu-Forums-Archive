### Post by mcduck on 2012-08-22
Perhaps give Ubuntu Studio a try?

[http://ubuntustudio.org/]("http://ubuntustudio.org/")

...although if you already ahve Ubuntu installed, just adding the applications for what you want to do would be a lot less work than making a fresh install of Ubuntu Studio or any other distribution.

If you can give more details on what exactly you want to do, people would surely be able to recommend suitable applications for those tasks. (being a game/multimedia/web designer myself, I sure could help with that if you just tell what kind of tools you are looking for...)

edit: just to list a few basic tools I use for most web desing work:
Gimp - for most image editing tasks
Inkscape -  awesome vector graphics application. I do most of the layout design and graphics with this
LAMP stack from Ubuntu repositories -  a testing/development server is of course required for any PHP stuff, and the native LAMP stack is better option than XAMPP or other third-party stuff.
Imagemagic -  command-line based image processing tool. Great when you need to do stuff with large number of image files, like thumbnailing, watermarks etc.
...and then just pick a text editor (or more advanced IDE if you prefer) you like. I use Gedit myself, enhanced with a couple of nice plugins, like the [zencoding]("https://github.com/mikecrittenden/zen-coding-gedit") one for example.
