### Post by Impavidus on 2013-04-10
Don't install 11.10. After this month it will be unsupported. Try 12.04.2 LTS or 12.10 instead. If you don't want Unity, you can opt for the more classical look of xfce. It is installed by default when you install Xubuntu instead of Ubuntu. Under the hood it's the same, with the same software available.

The only way to install Ubuntu (or Xubuntu) is by booting from the live dvd or usb (with the exception of wubi, but that is quite limited and depends on the windows file system, so it's no option if windows crashes or simply isn't present).
