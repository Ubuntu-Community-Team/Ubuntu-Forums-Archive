Note though if you manually access your drives you can delete files.

### Post by drewsus on 2010-05-09
I think we should also mention that even better than running it from CD, if your BIOS supports it (and most all modern ones do) you can put a Live CD on a USB stick and boot from that AND it allows you to save a limited amount of data on it too! 
You can achieve this with the tool found at System -> Administration -> Startup Disk Creator

Combining this with a custom .iso made with Remastersys and you are ready to go!
