### Post by Quackers on 2011-10-05
Welcome to UF :-)
First check the md5sum of the downloaded iso file

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)

If that checks out ok then check the cd/usb for errors.
At the first purple screen press any key. Vhhose language then select "check for errors" from the menu. If any errors are found then the disc/usb is no good.
Try re-doing it.
