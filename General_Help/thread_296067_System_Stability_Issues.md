### Post by Sherlock Holmboy on 2006-11-09
I've had moderate system stability issues with all releases. I've finally tracked it down to buggy/crappy memory. I run AMD Athlon 64. Even though my usual hypertransport frequency worked with WIN2K, it did not work with breezy or edgy, which would crash when the login screen came up or shortly after. I just replaced my non-matching brand, questionable quality memory with a single stick of good memory and everything is TITS!8) So if any one is having stability problems, consider the quality of your memory. Windows seems to be amazing with crappy memory, but now that I've upgraded to good memory, Ubuntu runs like a dream.:KS

P.S. If I'm not posting in the right place, please move because I have had innumerable headaches because of crappy memory. ](*,)
