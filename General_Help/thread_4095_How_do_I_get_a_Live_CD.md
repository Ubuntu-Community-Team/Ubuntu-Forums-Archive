### Post by khad on 2004-11-18
From the Pressed CDs page ([http://www.ubuntulinux.org/support/documentation/faq/shipit/](http://www.ubuntulinux.org/support/documentation/faq/shipit/)):

"For every Intel/i386 "CD" you order in Shipit, we will send you one package that contains both an Install CD and a LiveCD in a single cardboard folder. There is no need to specify one or the other as you will get both.

"There are no Live CD version of Ubuntu 4.10 for either PowerPC or ADM64. As a result, if you order AMD64 or PowerPC CDs, you will recieve a cardboard folder containing one install CD that is pressed using the image available on the web."
