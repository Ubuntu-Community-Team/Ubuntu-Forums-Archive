### Post by sudodus on 2014-12-08
Welcome to the Ubuntu Forums :-)

There can be many reasons why it does not work.

- the newest system does not work with some of your hardware (a driver does not work)

- you have some PPA or other separately installed program package, that causes problems with the new version of Ubuntu.

What is your current system (before the upgrade)?
Did you back it up before trying to upgrade?

Which version did you try to upgrade to (14.04 LTS or 14.10)?

Please specify your computer

- Brand name and model
- CPU
- RAM (size)
- graphics chip/card
- wifi chip/card

Knowing about your system makes it easier to give good advice.
