### Post by madverb on 2009-04-07
It might be possible if you haven't logged into Linux. It is possible that it has already started overwriting it.
You may be able to recover the old partition information by using Testdisk. You other option is to use testdisks file recovery. That way you can recover most files but will lose their original naming, which is better than not being able to recover them at all. Make sure you don't logged into Ubuntu until you have recovered all your files to another partition/hdd.
