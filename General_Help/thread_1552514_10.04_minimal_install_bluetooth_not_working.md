### Post by rubylaser on 2010-08-13
I have a minimal lucid install to run XBMC with only xinit x11-xserver-utils installed no GUI.  I'm trying to get my Logitech Dinovo Edge working via bluetooth.  I've install bluez-utils and can do an hcitool dev and see my bluetooth dongle, and I if I use hcitool scan, and press the connect button on my keyboard, the keyboard shows up with it mac address.  It won't connect at this point though, so I have to issue a sudo hidd --connect 00:11:22:AA:BB:CC and get it to connect.

Unfortunately, if I reboot or turn off the keyboard, it doesn't work again without SSHing in and redoing the steps above.  Does anyone have an idea to get my keyboard working at startup?

### Post by rubylaser on 2010-08-14
Okay, I solved this myself.  I uninstalled bluez-utils and bluez, and left the entry in /lib/udev/rules.d/70-hid2hci rebooted the computer and everything is working without having any of the bluetooth packages installed.  Apparently, this keyboard works without using hci.
