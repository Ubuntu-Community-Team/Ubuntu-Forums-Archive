### Post by mcduck on 2006-01-18
Have you tried disabling ipV6 DNS from firefox? In my experience, Opera was not a bit faster than FF after I did that. (and Fasterfox extension made FF even faster than Opera)

If you haven't tried it yet, type about:config in the address bar, use ipV6 as filter and you'll get a line like 'network.dns.disableIPv6'. right-click that and select 'toggle' to set it's value to 'true'. Restart Firefox and you're done :)

Also, creating line 'nglayout.initialpaint.delay' with type 'integer' and value '0' makes FF feel faster. But Fasterfox extension does that too..
