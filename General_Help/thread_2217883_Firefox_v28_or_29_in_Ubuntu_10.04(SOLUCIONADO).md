### Post by Impavidus on 2014-04-19
You're right that some things cannot be done in Firefox 20, having good security for example. Firefox 20 is unsupported and so is Ubuntu 10.04 – except for the server version, but that has no web browser. Better to move to a supported release of Ubuntu and you'll automatically get the latest Firefox (28, 29 hasn't been released yet). If you insist, you can download the latest Firefox from Mozilla and run it on your system, or at least try. I can't guarantee it will work on Ubuntu 10.04. It doesn't require installation.
