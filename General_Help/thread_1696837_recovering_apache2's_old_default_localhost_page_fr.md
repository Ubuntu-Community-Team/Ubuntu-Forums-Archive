### Post by myth1914 on 2011-02-28
You should be able to reconfigure mythweb (easily done through mythbuntu control centre) and answer no when it asks something about "will this be the only web page hosted"  If done with the only site, you will get the result you have.  Otherwise, it will configure to localhost/mythweb
