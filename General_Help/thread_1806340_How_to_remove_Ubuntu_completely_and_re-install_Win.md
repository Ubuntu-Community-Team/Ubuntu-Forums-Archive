### Post by Mark Phelps on 2011-07-17
Last time I checked, Unetbootin only allows you to make LINUX bootable USB drives, not MS Windows.  Yes, it can be run from MS Windows, but it only creates Linux boot solutions.

And, everything else I've seen requires that you have a working MS Window installation to run the associated utility.

Sorry, know of no way to make an MS Windows boot USB from Linux.
