### Post by mb_webguy on 2009-04-14
Multiple desktops allow you to manage you applications and open windows easily.  If I have several different applications open at once, I can group them according to function, and then switch back and forth between the different task-related groups.  

For example, workspace 1 is my web-browsing/general purpose workspace.  I have Firefox set to always open on workspace 1, regardless of my current workspace.

Workspace 2 is my downloading/transcoding workspace.  I have Deluge, Handbrake, and Avidemux set to always open on this workspace.

Workspace 3 is my media workspace.  Amarok and VLC are set to always open on this workspace. 

Workspace 4 is my programming/professional workspace.  I have MySQL Administrator, NetBeans, and other programming tools set to open on workspace 4.

So if I'm doing several things at once, I can switch back and forth quickly between different tasks without closing any windows.
