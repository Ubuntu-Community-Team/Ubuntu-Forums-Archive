### Post by mystics on 2015-10-05
Sorry if this sounds like a stupid question, but have you tried searching for "Task Manager" in the Unity dash? I'm not sure what comes with Unity, but I know Xubuntu does have a task manager.

If you don't have one, LXTask is in the Ubuntu Software Center and will be the first option that comes up if you search for "Task Manager" (without quotes). It's just labeled as "Task Manager" on the store page, but the description will point out it is LXTask. That one might suit you very well. It's informative, easy-to-use, and can do things like kill processes if you right-click on them.

Edit: Oh yeah, also try searching for "System Monitor" in the Unity Dash. I believe that comes with every Ubuntu flavor.
