### Post by colin.p on 2010-07-24
I bet you are looking at "printing" under administration, it has limited functionality.
You need to install hplip gui in synaptic, then run the set-up for your printer.
I have the same printer set up wirelessly and had to run the setup by its IP address. I have three ubuntu, two XP, Vista and 7 computers, and all print to the printer fine.
Once you have hplip installed all printer functions work, even colour. However, mine doesn't print in colour as all 3 of my colour cartridges are bone dry and I'm too cheap to buy new ones.
