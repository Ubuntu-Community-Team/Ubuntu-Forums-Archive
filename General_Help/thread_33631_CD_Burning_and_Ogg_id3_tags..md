### Post by lt_kije on 2005-05-11
I use gnomebaker (which can be found by searching for "CD creation", I think :-)). As for your Ogg files, they don't use IDv3 -- that's an mp3 thing. To modify your Ogg tags, install an app like Cantus ([http://www.debain.org/software/cantus/](http://www.debain.org/software/cantus/)). It'll let you edit each file, or groups of files, etc. It's quite powerful.

Good luck,

lt_kije
