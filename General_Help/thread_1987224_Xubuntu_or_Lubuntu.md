### Post by Bucky Ball on 2012-05-26
Either/or.

I would advise you try them from the LiveCD first and see which one grabs you before leaping into a full hard drive install.

Download the ISOs, burn them, boot from the CD and 'Try Lubuntu' (or 'Try Xubuntu'). This option is there on the desktop version, not the alternate. ;)

Lubuntu will be lighter 'out of the box', but if you want to bypass any of these deliberations go for a Ubuntu minimal install. It would be a learning curve but  would also be lighter than either (as long as you don't install more than you need to work). Good luck.

### Post by Peripheral Visionary on 2012-05-26
My 8-year-old hand me down Dell (Celeron, 512 RAM) runs Xubu 12.04 just fine. I have tried both on this machine and experienced no noticeable difference in speed or performance between them.

Xubu 12.04 is a 3-year LTS edition. Lubu 12.04, from what I've read here, is *not* a LTS release if that matters to you. I like the Xfce panel "goodies" (the weather applet, calendar, etc) which aren't available in the LXDE panel. But you can certainly use Conky to add those things to either desktop.

If and when Xubu becomes too much for this old hardware, I'll look at Lubu before anything else. But for now Xubu offers more features and configurability without any loss of performance on my computer.
