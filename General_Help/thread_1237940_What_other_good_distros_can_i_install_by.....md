### Post by P4man on 2009-08-12
those are not "distro's". You can only install other desktop managers that way. Gnome, KDE and XFCE (giving you Ubuntu, Kubuntu and xubuntu respectively).

Other "distro's" (=short for distributions) would be Fedora, Suse, PCLinuxOS, etc. You can't just install them with apt though, you'll have to download their cd's and install the way you installed ubuntu. If you're interested in trying one of them out, perhaps this link helps:

[http://distrowatch.com/dwres.php?resource=major](http://distrowatch.com/dwres.php?resource=major)
