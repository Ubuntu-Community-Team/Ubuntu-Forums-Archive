Not sure if this will work for you cos you might have already tried it, select "Message" in the main window click on "New message to mailing list" a new window will appear,"Composer" from this one click on "Tools" click on "Select recipients, in the new window click on the first option, next to Address Book a drop down menu will appear with more options amongst these are groups etc,add them now if they are not there already, hope this helps.
