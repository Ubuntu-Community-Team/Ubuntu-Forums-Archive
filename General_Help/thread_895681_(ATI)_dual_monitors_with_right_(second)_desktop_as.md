### Post by CowzRule on 2008-11-02
Try this
> 
Section "Device"
	Identifier	"ATI Technologies Inc RV370 5B60 [Radeon X300 (PCIE)]"
	Driver		"fglrx"[COLOR="Red"]
        Option          "DesktopSetup" "horizontal,reverse"[/COLOR]
	Busid		"PCI:1:0:0"
EndSection

The line in red is what is added.
If that doesn't work, take a look at the file I've attached. You can get this same info by opening a terminal and typing> aticonfig --help
Also, try swapping the monitor connectors on the back of the card.
