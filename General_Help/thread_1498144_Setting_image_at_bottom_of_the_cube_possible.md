### Post by WorMzy on 2010-05-31
It is. You need to install compiz-fusion-plugins-extra though (sudo apt-get install compiz-fusion-plugins-extra). Once you've done that you can enable the Cube Reflection and Deformation plugin in the CompizConfig Settings Manager, which gives you the option to change both the top and bottom cube caps.

### Post by WorMzy on 2010-05-31
There is, but you would need to change the opacity of the workspaces too. If you want the workspaces to appear transparent, then you'll need to keep the caps transparent too.

If you don't mind the workspaces being opaque, then all you need to do is open the Desktop Cube plugin's settings, switch to the Transparent Cube tab, and set the Opacity During Rotation to 100.
