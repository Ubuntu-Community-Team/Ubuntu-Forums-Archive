### Post by 25tom on 2013-06-04
If you want a Debian-based live distro, I can fully recommend Knoppix - easy to set up, automatic persistence, Debian repos - personally I've found that there's more useful software available for Knoppix than Slax, but YMMV, as they say...
If you're feeling more adventurous you could try [http://live-build-cgi.debian.net/cgi-bin/live-build](http://live-build-cgi.debian.net/cgi-bin/live-build), which will let you generate a customised Debian Live image - you choose the desktop environment you want and then type in the list of extra packages you want included in the image. Persistence is harder to get going, though...
