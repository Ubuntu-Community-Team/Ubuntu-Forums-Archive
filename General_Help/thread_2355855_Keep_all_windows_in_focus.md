### Post by dragonfly41 on 2017-03-17
I'm not sure if this helps.
but in Chromium Web Browser there is an add-on I have tried.
[Window Resizer V2 beta testing.]("https://chrome.google.com/webstore/detail/window-resizer/kkelicaakdanhinjdeammmilcgefonfh")
This allows windows to be set to preset sizes from a list.
So you can stack multiple browser windows into a grid
and in each window you can optionally use tabs.
You could probably strip top menu bars in browser windows if you need more "real estate".

### Post by dragonfly41 on 2017-03-18
On reflection there is another approach which will consume less memory.
Create a single HTML5 page containing a grid of iframes laid out using css.
Each iframe will contain a web site.
This has the advantage of only using one browser window and top tool bar.

As vasa1 writes above you will need to open up more on your exact requirements.
