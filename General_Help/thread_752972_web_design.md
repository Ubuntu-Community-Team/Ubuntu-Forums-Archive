### Post by Ioky on 2008-04-12
I don't know how much you know about web design. But what I use is Bluefish, Gimp, and inkscape. Of course millions of difference browser to test out the site. 

Bluefish are just simply a good text editor. 
Gimp, and inkscape for the graphic I need. 

if you want to have a photo gallery or blog there are also some Open Source software out there like WP and Gallery2. 

I highly recommend you learn HTML-XHTML, PHP, Javascript, and MYSQL. You can almost make anything you see on the web today by using those language. HTML-XHTML are the mark up language. and PHP is a programming language which take place at the server, and Javascript are progamming which take place at your browser. and MySQL are data base which control your data, like blog post or photo in the photo gallery. 

What I don't recommend is use WYSIWYG software like Dreamware, The reason I say that is because the time that take you to learn the software itself are longer than the time you will need to learn all HTML-XHTML, PHP, Javascript, and MySQL combine together. I mean those software are just help to the HTML or maybe PHP, Javascript. So if you already know how to do those thing yourselves you wouldn't need those software. In the long run. you will need to learn those thing anything for you to done anything good site. 

At least, welcome to the Free World.
