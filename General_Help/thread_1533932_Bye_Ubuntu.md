### Post by MattBD on 2010-07-18
I suspect much of the issue you're having is due to the lack of multimedia codecs in the default install. This is for good solid reasons - in some countries it's somewhat questionable to distribute these, due to things like software patents, so by including them the Ubuntu developers would be risking legal action. Most of these can be installed by just installing ubuntu-restricted-extras.

If you want my advice, I think going cold turkey from Windows is always a bad idea - if you do you'll probably go back to Windows quite quickly when you find something you can't do. Ease yourself into it and learn to do things before you commit yourself. A good way to do that is to run Ubuntu in Virtualbox on top of a Windows install, and use that to learn how it works. Only make the final move to Ubuntu once you're comfortable doing so.
