### Post by jerrylamos on 2007-04-29
Well, "safe graphics mode" the second choice on the CD Live boot menu usually works.  

You might get a clue by at the boot selection screen by pushing F6 and deleting the words "quiet splash" which should get you a flood of text messages as it boots up.  What are the last few lines?

If it stops with a blank screen, push Ctrl-Alt-F1 and you should get a text screen - are there any messages there?

Cheers, Jerry
