### Post by coffeecat on 2009-12-24
If you know the username, you can reset the password with the instructions here:

[http://www.psychocats.net/ubuntu/resetpassword](http://www.psychocats.net/ubuntu/resetpassword)

In fact, if you don't know the username, you can discover that with the instructions there. Please read them carefully.

If the grub menu is hidden at boot, you need to know which version of Ubuntu you are running in order to show the grub menu.
