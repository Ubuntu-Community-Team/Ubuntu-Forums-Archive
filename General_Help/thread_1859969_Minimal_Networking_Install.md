### Post by aeiah on 2011-10-14
it'd be a lot easier if you go with a wired install to start with.

if it's inconvenient to do a wired install, and you know your wireless card is supported out of the box, you could install ubuntu minimal and connect via the command line and start installing things. its probably easier to use the netinstall or ubuntu minimal over a wired network though until you've got X and can connect to wireless easier.

i think both versions already have apt-get, but the netinstall may require internet access before it has the required parts for wifi to work
