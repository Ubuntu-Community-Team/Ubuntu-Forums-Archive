### Post by merc82 on 2012-06-21
I have two Xubuntu 12.04 systems. One is i686 and has Skype 4 and a Microsoft USB camera; the other is x86_64 with an internal cam and Skype 2 from the repository. Skype 4 uses much more memory when I'm logged in but but in a conversation (115MiB v 77MiB). 

I believe the picture quality with Skype 4 has improved for my Microsoft cam. This cam was never well supported but now is usable in Skype. The in conversation interface is bulky and too busy, so it's more inline with the Windows version.

So Skype 4 helped me for one machine but I'll be keeping the old version on the other until the repositories are updated.
