### Post by oldos2er on 2008-11-30
Of course there's a 64-bit version of Ubuntu 8.10. And I've yet to find any package (except for non-free Java and Flash) that doesn't have a 64-bit version in Ubuntu's repositories. There is a 64-bit version of Flash now, but it's alpha.

 If you're determined to go back to 32-bit Ubuntu, you'd need to do a complete reinstall.
