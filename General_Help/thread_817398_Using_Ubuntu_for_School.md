### Post by Rocket2DMn on 2008-06-03
While you may be able to get by using solely Ubuntu, I recommend dual booting.  Most universities will not support your computer if you have problems on their network, and some require you to use specific software for anti-virus and firewall.  While this is not needed in Ubuntu, they may not let you use their network until you can prove that you are protected.  Also, if you ever have to use software for classes, the chances are they won't have an open source alternative for you to use.
Basically, the biggest problem is that they won't support your system on their network unless you are using a Windows or Apple computer.
Even though tools like VPN can be used fairly easily in Ubuntu, the school won't help you set it up (we can help, but no guarantees).

+1 based upon my younger daughter's experience. While she preferred to use Ubuntu, there was no support (in fact the college's IT department states very clearly that one is on one's own if using Linux), and the anti-virus, etc. apps were required. There might be another issue, depending upon which word processing app you choose to use. The requirement was MS Office (whatever) for papers, etc. If using OO Writer, for example, one must be careful to format the page with the same margins that Word uses; generally, one inch top, bottom, left and right margins. OO writer, for example, uses .99" margins, as I recall; this created a problem for her, believe it or not. She used Abi Word a lot, but there were some issues with that, too, about file types as I recall. So, a USB stick was used to take the paper from her computer to one in the computer lab, reformatted as needed, and then submitted.

Anyway, no personal experience; back "in the day" when I was in college, then law school, there weren't any PCs; and we had to use typewriters. :-)
