### Post by grahammechanical on 2013-08-24
You are trying to install software meant for Fedora on Ubuntu and that will not work because Ubuntu and Febora use different systems for packaging software.

> RPM fusion provides software that the Fedora Project and Red Hat does not want to ship

[http://rpmfusion.org/](http://rpmfusion.org/)

You do not need PRM Fusion on Ubuntu. You have the Ubuntu Software Centre.

What is you think you need that Ubuntu does not already provide?

MythTV? That is in the Ubuntu software Centre. There is even a complete version of Ubuntu with MythTV built on top of it called Mythbuntu

[http://www.mythbuntu.org/](http://www.mythbuntu.org/)

Nvidia drivers? They may have been installed when you installed Ubuntu. Did you tick to install third party software? Yes? Then you are running on a Nvidia driver, provided, of cource that you have an Nvidia graphic adapter. Look for the Additional Drivers utility.

Realcrypt? Look for Truecrypt in the Ubuntu Software Centre.
