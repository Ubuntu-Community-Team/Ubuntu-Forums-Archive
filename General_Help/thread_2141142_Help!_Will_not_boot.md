### Post by Impavidus on 2013-05-02
When you get to that menu (which is called the grub menu) you can select previous versions and try to boot the previous version (not the recovery mode). Maybe a recent kernel upgrade broke your system. That happens sometimes, especially in combination with proprietary graphics drivers.
