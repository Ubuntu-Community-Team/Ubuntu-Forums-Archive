### Post by r-senior on 2011-04-16
If you want to run a Java application that is supplied as a JAR file, it's usually a matter of:

```

$ java -jar <name-of-jar-file>

```

See [http://download.oracle.com/javase/tutorial/deployment/jar/run.html](http://download.oracle.com/javase/tutorial/deployment/jar/run.html)
