### Post by nkri on 2008-09-26
Right-click in the space where the launcher should be.  Click Remove This Launcher.  Click the green check mark and the space should disappear.

If it doesn't show "Remove This Launcher" right away, right-click around in that general area a few times...it should come up after a few tries.

Good luck!
-nkri

### Post by loomsen on 2008-09-26
quit cairo-dock, then navigate to your configurations directory with your favourite file browser, i.e. type into a terminal:
```

nautilus ~/.cairo-dock/current_theme/launchers

```

now you can edit any launcher with your favourite text editor, which should be self-explainatory i think.
