### Post by RasterBurn on 2010-03-18
hello everyone, it seems to me that while i search the forums looking for information or looking for someone to help i often see people trying to install programs that are windows based on the *buntu system, and respectively asking for help when it isn't working right through wine or crossover this post is primarily for those who partake in running windows software under wine (or a variant of wine) under *buntu please have a look at this link:

[http://wiki.linuxquestions.org/wiki/Linux_software_equivalent_to_Windows_software](http://wiki.linuxquestions.org/wiki/Linux_software_equivalent_to_Windows_software)

as the link states it is an equivalence page for windows to linux users (its not complete, but it shows the main ones) you should see in that list that there is no reason to run windows software under *buntu unless there is a specific windows program needed (for what ever reason it may be such as file compatibility, stability, or some other specialized requirement), this is not meant to be a rant or a rave about anything, its just more for sharing what i know for equivalences between windows and linux software to save everyone headaches regarding windows emulation under *buntu (i know i get them when installing stuff)

some examples of equivelents would include but not limited to:
MSN -> aMSN, PidGin, Emesene, Kopete
Microsoft Office -> OpenOffice.org
Outlook Express -> Evolution, Thunderbird, Sylpheed
Adobe Photoshop -> GIMP
3D Studio Max -> Blender
Daemon Tools -> mount
winRAR -> Karchiver, File Roller, ark

many of these tools come pre-installed in your Linux flavor so i say, give them a try before you decide that your windows software needs to be installed through an application like wine or crossover
