### Post by aaaantoine on 2008-11-09
Two bumps in three hours?  Patience, friend.

This change affects Gnome as well as Ubuntu.  If you want your old Shut Down button back, you have a few choices:

1. Roll back to Hardy.
2. Program your own applet.  See if you can pull from the Gnome 2.22 panel applets source code for time savings.
3. Wait for someone else to do it.

The System menu has all the shut down options broken down nicely, courtesy of the Gnome devs.  The FUSA applet (the thing that replaced your shut down button) is a hack from the Ubuntu devs.

The folks at Canonical want to make it so that when the FUSA applet is showing, the buttons in the System menu will disappear.  I agree that this is how it should behave.  However, I prefer the way that the System menu Shut Down button works over the FUSA "Suspend | Hibernate | Restart | Shutdown" one-click options.  Also, it bothers me that none of the FUSA options have icons.

I guess my point is, you're not alone in not liking FUSA.  But this is the way they're going with it.
