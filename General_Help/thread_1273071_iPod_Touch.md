### Post by Chronon on 2009-09-23
Well, you only need iTunes if you want to sync media to it.  You can download apps directly from the device.  IMO, it's better as a MID than a media player.  It is my least favorite music player and I find that I only really use it for WiFi connectivity and to run various apps on.  

You can install iTunes in VirtualBox, at least.  This makes the time to sync shorter because you don't need to go through a couple of reboots to sync and get back to your main system.

### Post by Chronon on 2009-09-23
I have found I can run iTunes in Vbox, at least.  This means I can mount my music library (in Ubuntu) as a shared folder and directly drag folders from the shared folder onto the device in iTunes without ever leaving Ubuntu.  I have found that songs added this way don't even get added to iTunes' library but do get synced to the device.  I do share the folder as read-only just to be sure that iTunes doesn't botch the organization of my files.  

However, if I actually want to listen to music, I would rather use my Gigabeat running Rockbox than the iPod Touch.
