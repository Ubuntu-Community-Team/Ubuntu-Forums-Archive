### Post by Hedgehog1 on 2011-05-27
There were some pretty substantial changes for Natty; and it now appears the default videos drivers the LiveCD carries for Natty are not compatible with you video card.

A (possible) solution is to create a LiveUSB of Natty with persistence, and when you boot from it load the proprietary video drivers first thing.  Then (hopefully) all future times you boot from the LiveUSB you will get a usable screen.
