### Post by Rocket2DMn on 2008-06-14
No, I don't think so.  By default Ubuntu does not install restricted/proprietary codecs or drivers.  For mp3 and dvd support (among others), you need to use the Medibuntu repository - [https://help.ubuntu.com/community/Medibuntu](https://help.ubuntu.com/community/Medibuntu)
