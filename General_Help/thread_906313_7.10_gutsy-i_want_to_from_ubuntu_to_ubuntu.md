### Post by Sycron on 2008-08-31
If the other installation is ubuntu 7.10 then you can save all the **/var/cache/apt/archives** debs, copy them on a folder, then boot ubuntu from the other installation and run ```
sudo dpkg -i *.deb
``` after you **cd**'ed to the saved debs.

With this you wont have to install again wine,etc. Your settings won't be saved (I mean the desktop background,etc)
