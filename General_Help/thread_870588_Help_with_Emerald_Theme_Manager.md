### Post by steveneddy on 2008-07-26
Emerald's only job is to change the window border.

You should try

gnome-look.org

and search for GTK 2.x themes to get you started.

There are Compiz and other Emerald themes there, also.

Don't forget to get a couple of icon themes and look at the mouse themes so you can change it up a little.

You can also change the default colors yourself, which is what I do, by

Right click Desktop

choose Change Desktop Background

Tab on top, choose Theme

Button on bottom, choose Customize

Choose Colors Tab

There are other customization options there also, so look around.

When you get to a point that you like, save it in the Themes window.
