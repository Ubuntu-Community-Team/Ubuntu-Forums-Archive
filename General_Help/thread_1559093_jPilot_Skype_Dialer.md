### Post by arm-c on 2010-08-23
A command line interface program has been released by [Cho-Guem Utilities]("https://sourceforge.net/projects/chogeum-utils/") on SourceForge. 

Deb files are available.

Depends:  skype4py

    ==========================================================================
    JPILOT-DIAL-SKYPE HELP
    --------------------------------------------------------------------------
    jpilot-dial-skype is a short python script that can be utilized 
    with the jpilot program to enable calling your synchronized contacts
    from within the jpilot program.  It can also be used from the command
    line
    
    Syntax:  jpilot-dial-skype (CallNumber|--help)"
    
    Sample:  jpilot-dial-skype --help"
             or
             jpilot-dial-skype +18005551212"
    
    J-Pilot integration is simple.  From your contacts entry, click any 'dial'"
    button.  In the resulting dialog, change the dial command from:"
        Dial Command:  jpilot-dial --lv 0 --rv 50 %n"
    
            to:"
        
            jpilot-dial-skype %n"
    
    Project site:  [https://sourceforge.net/projects/chogeum-utils/](https://sourceforge.net/projects/chogeum-utils/)
