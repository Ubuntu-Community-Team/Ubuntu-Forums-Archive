### Post by guywithcable on 2009-08-05
Here is a script to create lists of emails from system groups. The lists get placed in /etc/lists, and their filename is the same as their group name. The emails are created from the user names + what's in /etc/mailname.  You can easily use these lists as aliases in Postfix to create mailing lists. :)
