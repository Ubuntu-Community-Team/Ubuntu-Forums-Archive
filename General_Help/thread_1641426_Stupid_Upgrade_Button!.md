### Post by ChadMMc on 2010-12-09
If you are on an earlier distribution (before 10.10) and the update manager is not showing a button for upgrade, choose 'Administration -> Software Sources' and select the 'Updates' tab and make sure 'Release Upgrade' is set to 'Normal Releases'.

Should be able to upgrade to 10.10 then.
