### Post by simohell on 2006-01-20
As we can see your Xorg.0.log reports your screen to be 1280x800.

> (II) I810(0): Display Info: LFP (local flat panel): attached: TRUE, present: TRUE, size: (1280,800)

You are very likely to need a program 915resolution which should be found also in Ubuntu repositories.

Set up 915resolution by editing the file " /etc/default/915resolution " to match your desired resolution.

The file might end up looking like following:
```
#
# 915resolution default
#
# find free modes by  /usr/sbin/915resolution -l
# and set it to MODE
#
MODE=3c
#
# and set resolutions for the mode.
#
XRESO=1200
YRESO=800
#
# We can also set the pixel mode.
# Please note that this is optional,
# you can also leave this value blank.
BIT=

```


If this does not help, you might need to force several different modes to your display resolution to make it work. Also you might need a modeline, but first try different uses of 915resolution. If the default usage of 915resolution wont work I can give you more details on what to try next.
