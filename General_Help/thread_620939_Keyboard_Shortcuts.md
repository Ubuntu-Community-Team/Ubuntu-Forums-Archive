### Post by Linton on 2007-11-23
Being a long time windows user, I have gotten used to its assorted keyboard shortcuts. Although Ubuntu has the good old ctrl-alt-delete, and alt-tab, the lack of the Windows-D key has slightly hindered a very smooth transition to Ubuntu from Windows. My Question is: Is there any way to bind the show/hide desktop action to a key press? I know there is a button on the bottom panel, but it would be great to have a keyboard short cut for it anyway. I am newbie with Linux, so please tell me exactly what to do. ( I have no problems with modifying code)

### Post by luisfcup on 2007-11-23
Here is a link with same help on Ubuntu shortcuts.

[https://help.ubuntu.com/community/KeyboardShortcuts]("https://help.ubuntu.com/community/KeyboardShortcuts")

You can always configure your own shortcuts or, select windows style shortcuts.
In Kubuntu, go to K Menu -> System settings -> Keyboard & Mouse and you will see the configuration screen were you can change you shortcuts.
In Ubuntu, I don't remember exactly but, you will probably find a "Keyboard & Mouse" application in System -> (Preferences or Administration) menu.

### Post by Linton on 2007-11-23
I have fixed this problem, thank you.  It was both a combination of changing keyboard short cuts, and changing the Win key behavior to a Hyper rather than Super key behavior. This means that the windows key can be used in combination with a letter key, rather than an Alt, Ctrl, Shift key.
I hope this helps people who want to have the windows-D combination to show the desktop, just like in windows.
