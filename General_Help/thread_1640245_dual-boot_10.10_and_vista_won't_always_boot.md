### Post by argedion on 2010-12-07
You can go to the Manufacturers web site and see if there is a bios update for your particular machines. If there is you can either download the update then run it. (Advise that you know what your doing) or you can download the update then take your machines to someone who does know what their doing and have them install the updates. The best advice I can give you is to make sure you do not shut off the computer during the update. The problem could be a bad disk as well but happening on both computers at once is suspicious. First thing I would do is check the disk and make sure there is no physical damage to the disk if I find that there is none then I would look for a bios update.

NOTE: Most Manufactures will have a bios update in the Windows Format, meaning that it will be a Windows executable file and not a Linux binary file.
