### Post by steeldriver on 2015-10-17
The PATH variable is used by the shell to locate executable programs (so that you don't need to type the full pathname every time) so if you use a different variable name it won't have the desired effect.

You previous post shows mismatched quotes (double quote at the start of the string - single quote at the end): if you really typed it like that, then that's something you need to fix. Otherwise I don't see anything wrong.
