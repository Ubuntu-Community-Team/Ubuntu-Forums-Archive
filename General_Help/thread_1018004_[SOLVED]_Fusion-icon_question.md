### Post by davidself1001 on 2008-12-21
How do I get the compiz fusion-icon to show up in my taskbar at startup?

---

### Post by davec64 on 2008-12-21
Go to System >> Preferences >> Sessions 

Click on Add under The Start Up Tab and add "fusion-icon" as the Command, give it a meaningful name as well!

It will start when the system loads then.
