### Post by EgoGratis on 2011-05-27
> I'm interested in synching my android phone with thunderbird.  Have you made any progress with this?Yes i bought Samsung phone with SyncML support and then i installed Funambol server on my PC. Ubuntu (Funambol server) and mobile phone can sync without a problem over Wi-Fi router (localhost address).

Then you sync your Ubuntu client with Funambol server. For Evolution you need SyncEvolution and for Thunderbird you need extension for Thunderbird you already found.

I did some tests and at this moment SyncEvolution + Evolution does work better. Thunderbird extension in my opinion does not work as it should in this moment and a lot of problems with duplicating when syncing. In the end i am very happy with this solution but it does take a few days for average user to understand all and make it work. I don't know about Android but i think Funambol developers do provide extension for SyncML support for Android. My mobile phone does not need extensions it has SyncML support built in.
