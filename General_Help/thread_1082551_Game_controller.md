Keytouch didn't help me, but maybe you can figure it out.

Instead, I'd try mapping the features you need in Keyboard Shortcuts to a different key combination that the joystick program can handle.

For example, map Volume Up to Ctrl-Alt-PageUp. Then map button6 in your joystick program to Ctrl-Alt-PageUp.
