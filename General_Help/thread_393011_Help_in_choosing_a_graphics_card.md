### Post by jeffathehutt on 2007-03-25
I recommend nVidia cards over ATI, but technically either will work.  I have gotten beryl to run on an nVidia GeForce MX200 64 MB card.  That's a really old card, and any newer card will also be faster, so installing beryl shouldn't be too much of a problem.

As for the installation, it requires a bit more than just plugging it in.  Once you boot up the computer for the first time, you will need to install the official drivers from nVidia, but that isn't too hard of a task these days.  To make it easier you can use the envy script ([http://www.albertomilone.com/nvidia_scripts1.html](http://www.albertomilone.com/nvidia_scripts1.html))

Edit:  As for price, you can probably find a decent card for less than $100 US
