### Post by anaconda on 2007-02-22
root is the same as administrator in windows.

And you can get root rights in terminal by typing:
sudo su

and if you want your graphical file browser with root rights then just type
sudo nautilus
in terminal.. and if you need to edit a file just right-click in the root file browser and select edit..To make a new directory File>New folder..

And the sudo password it will ask is your own password.. the same that you use to log in.
