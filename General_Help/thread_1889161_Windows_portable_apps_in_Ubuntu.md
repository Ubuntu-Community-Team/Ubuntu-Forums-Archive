### Post by Mark Phelps on 2011-12-01
> **Benhermies said:**
> Some of these are Portable Apps.

The only thing different about "portable" Windows apps is that they can be run without being installed.

With regard to Wine, it doesn't make any difference -- because the portable and nonportable versions of the same app use the same DLLs.  So if one version doesn't work, the other won't either.

As mentioned, you need to check the WineHQ site for the version of each app you want to run.  Wine only runs SOME Windows apps.
