### Post by phillw on 2010-08-21
Hi,

> A Pentium II or Celeron system with 128 Mb RAM is probably a bottom-line configuration that may yield slow yet usable system with Lubuntu At 256MB RAM you *can* use the standard installation CD method, but I would recommend using the [minimal installation](https://wiki.ubuntu.com/Lubuntu/DocumentationHelp/MinimalInstall) as it is quite likely that part of that 256MB is being used as shared video memory and it could take a real long time,  Ubiquity (The standard programme that installs the ubuntu family) may well struggle. There is more information on Lubuntu my signature and for the Ubiquity issue in the above link.
