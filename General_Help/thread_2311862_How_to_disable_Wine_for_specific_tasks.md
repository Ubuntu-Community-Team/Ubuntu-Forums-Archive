### Post by syntaxerror74 on 2016-01-31
Best thing would be to find a CLEAN Windows **virtual** machine (i. e. not that friend's real one with the virus that's for sure :mrgreen:) and run *HijackThis* there to see what's going on. Virtual, because you had better NOT infect yet another **physical** machine with your (now virus-ridden) files.
