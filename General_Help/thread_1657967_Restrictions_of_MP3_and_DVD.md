I try to understand the reasons for restricting DVD and MP3.

### Post by noah++ on 2011-01-01
With DVD, I think it's only a DMCA problem, not a patent problem. It might be illegal to circumvent the encryption that "protects" almost all commercial DVD's, which is exactly what all unlicensed software DVD players do.

The MP3 restriction is a patent thing. Although the codec is cost-free to end users, there is a fee for device makers to use it in their products.

If Ubuntu included capabilities for either format by default, there might be legal *and *philosophical ramifications.
