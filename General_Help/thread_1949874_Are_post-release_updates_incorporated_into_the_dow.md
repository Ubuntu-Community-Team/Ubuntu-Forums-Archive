### Post by Cheesemill on 2012-04-06
Nope. If you interrupt the installer and reboot, the whole process has to be started again.
I think you would be better off using the normal installation CD and then applying updates after you have installed. This way if you have to reboot the machine the packages already downloaded remain available.
