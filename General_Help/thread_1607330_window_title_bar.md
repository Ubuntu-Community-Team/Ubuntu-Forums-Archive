### Post by sikander3786 on 2010-10-27
Which Window manager? Compiz or metacity?

Probably best to install the fusion-icon, let it sit in the system tray and use it to reload/switch your Window manager when you're in troubles like that.

```
sudo apt-get install fusion-icon
```

Found under Applications > System Tools > Compiz Fusion Icon.

Add it to startup from System > Preferences > Startup Applications. Click Add and type fusion-icon in command.
