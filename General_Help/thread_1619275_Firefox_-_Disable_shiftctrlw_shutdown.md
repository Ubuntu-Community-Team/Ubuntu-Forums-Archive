Thanks for responding. Keyconfig did the trick. For others with the same question, I couldn't find keyconfig in Mozilla's add-on collection, so I had to install it via the link in the first post here:

[http://forums.mozillazine.org/viewtopic.php?t=72994](http://forums.mozillazine.org/viewtopic.php?t=72994)

A key combination that shuts down the entire browser with no confirmation, and is that easy to hit accidentally, really should have an enable/disable option in the preferences. Actually a preferences tab similar to keyconfig would be a nice addition. I seem to remember Opera having something like that, though I haven't used it in a while so I could be thinking of something else.
