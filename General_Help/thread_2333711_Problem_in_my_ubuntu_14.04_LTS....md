### Post by grahammechanical on 2016-08-12
How much RAM? What video adapter and how much video memory?

When an application window darkens and freezes it is a sign that the application is busy. It could be that some other process is using system resources and forcing the application to wait or forcing the system to page data in and out of the swap partition in order to make memory available to whatever task needs the memory.

Are the tasks that you are doing video intensive or CPU intensive or both?
