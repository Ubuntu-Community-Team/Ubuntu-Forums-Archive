### Post by Bruce H on 2009-11-15
I found this solution after searching the net for a moment. Try this:
```
xrandr -s 1280x1024@75
```[Source]("http://www.linuxtutorialblog.com/post/solution-resetting-your-screen-resolution-with-xrandr")

I don't know if you need sudo or not. I also don't know if that will persist across sessions. If not, you might need to research how to do that.
