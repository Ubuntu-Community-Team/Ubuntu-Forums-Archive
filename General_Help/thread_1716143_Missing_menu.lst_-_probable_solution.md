### Post by PCNetSpec on 2011-03-28
Erm... you are not supposed to edit /boot/grub/grub.cfg manually.

Any changes are *supposed* to made in /etc/default/grub and applied with:

```
sudo update-grub
```

Read the first few lines of grub.cfg

[COLOR="DarkRed"]# DO NOT EDIT THIS FILE
#
# It is automatically generated by grub-mkconfig using templates
# from /etc/grub.d and settings from /etc/default/grub[/COLOR]

and check out **blueridgedog**'s link.
