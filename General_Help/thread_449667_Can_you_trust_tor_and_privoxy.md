### Post by rich.bradshaw on 2007-05-20
Think it's OK the gatecrasher thing is a bit iffy though - think it's a port collision though...

Remember that if you become an exit node, traffic from others will come out from your computer - i.e. if some one else is looking at illegal material, this could look as if it was coming from you -this is the whole point, traffic gets bounced around so it is impossible to see where it comes from. 

Unless you have very good reason, I'd avoid it. Think why you might need that security.

PGP signed IM's work well using Kopete/Miranda + gpg for instance, to get private instant messaging, no need for Tor...
