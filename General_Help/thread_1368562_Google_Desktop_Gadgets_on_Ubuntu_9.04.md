### Post by fireflake78 on 2009-12-30
How do I put Google Desktop Gadgets onto my desktop? I know they look like screenlets, universal applets, etc, but I don't want to use those.

---

### Post by howefield on 2009-12-31
Open a terminal (Applications > Accessories > Terminal) and type

```
sudo apt-get install google-gadgets-gtk
```

Or install with Synaptic Package Manager.

You'll then be able to run the program from Applications > Internet > Google Gadgets (GTK).
