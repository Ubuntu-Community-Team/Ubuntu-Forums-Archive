### Post by mb_webguy on 2009-05-15
The server edition is for setting up dedicated headless servers.  The primary difference between the server and desktop versions is therefore that the server edition doesn't come with a graphical desktop environment.  There may also be a few differences in the kernel, but that's really the main difference.

If you're going to use this computer for anything other than a server and aren't completely comfortable working in a text-only environment, stay with the desktop version.
