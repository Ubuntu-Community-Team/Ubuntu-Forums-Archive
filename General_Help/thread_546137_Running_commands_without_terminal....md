### Post by Luinar on 2007-09-08
Or, if you specifically want a file which will execute a command, then just create a new text file and put whatever command(s) you wish to be executed inside it. Save it, and then in a terminal do:

```
chmod +x [filename]
```

This makes it an executable file.

You should then be able to double click on the file and be given the option to run it. :)
