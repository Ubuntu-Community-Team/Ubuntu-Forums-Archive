### Post by starcraft.man on 2007-04-18
Ummmm... all that comes to mind is that Windows isn't a perfect program. I can only guess that if your getting a black screen that you may have corrupted a part of your windows partition somehow during Ubuntu installation and chkdisk is running every boot and failing to recover a portion of the data (it eventually gives up if a sector is completely unrecoverable). All one can really do in that situation is reinstall windows... windows and its many registry faults/other problems are numerous.

Oh and just a thought, but I strongly recommend with windows having two partitions first one just for xp and core firewall/virus and then another partition for all your programs/games and other such things. Hope your problem is fixed and best wishes.

Oh and maybe use the alternate install disk for installing, I found it was easier than using the gui when I set up my dualboot.
