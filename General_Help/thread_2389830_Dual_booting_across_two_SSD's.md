### Post by rbmorse on 2018-04-22
If you have a UEFI computer, after you get Windows installed and configured you can boot back into Ubuntu using the BIOS/EFI setup and install rEFInd into Ubuntu. rEFInd will give you a boot menu and it's faster and easier than going into setup on every boot. I've been using it for three or four years and it's proven effective and problem free.  Plus, the developer answers his e-mail if you have questions. 

[http://www.rodsbooks.com/refind/](http://www.rodsbooks.com/refind/)
