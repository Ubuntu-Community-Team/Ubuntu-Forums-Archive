### Post by grahammechanical on 2015-08-18
What do you mean by "it won't work?" Does Network manager not detect any wireless networks? Network Manager will remember the networks we connect to but that will not stop us from connecting to other wireless networks. Once you have established a connection to your home wireless router and authenticated the connection by putting in the passphrase (not your user password) then Network Manager will remember and automatically make the connection if we have set things up that way. And yes, We can Edit Connections and Delete connections using Network manager.

More than this I cannot say as I have not used 10.04 for 5 years the user interface changed with 11.04. So, present directions may not be applicable. I cannot even remember if 10.04 had a Network Manager icon. I seem to remember an Administration menu and a Control Centre.
