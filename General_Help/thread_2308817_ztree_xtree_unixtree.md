### Post by bab1 on 2016-01-05
There are 2 applications available.  Both are in the repos so you don't need to search for downloads.  You can install using the Software Center or the Synaptic Package Manager.

The first is [**ytree**]("http://www.han.de/~werner/ytree.html") and the second is [**Midnight Commander (MC)**]("https://en.wikipedia.org/wiki/Midnight_Commander")

Ytree is more like xtree and MC is more like Norton Commander.
