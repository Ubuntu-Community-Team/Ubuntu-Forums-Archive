### Post by runyonave on 2012-12-24
Hi, I was just wondering if there would be any problems with the laptop if I keep on suspend mode instead of shutting it down. From what I understand, suspend mode basically stores all running data on memory, so the laptop is technically still running. With hibernation (i don't have that option), everything gets saved in the swap partition. So if I always put my laptop on suspend, would that cause any problems, other than running out of battery.

---

### Post by Toz on 2012-12-30
That's correct. Assuming your computer is properly suspending, there should be no issues with suspending instead of shutting down (assuming of course that you have an adequate battery supply to maintain the suspend state). Note that since 12.04, hibernate is disabled by default. It can be enabled by following the instructions [here]("http://askubuntu.com/questions/94754/how-to-enable-hibernation").
