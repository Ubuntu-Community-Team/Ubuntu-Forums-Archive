### Post by PhilGil on 2010-02-10
One thing a lot of people forget when they compare Windows to Linux is that Windows performance degrades significantly over time.  Both OS's are speedy right after installing.  

But your Windows machine needs anti-virus software, so you take a performance hit.  Then you install a few programs (many of which insist on being memory-resident) and take another performance hit.  Then, give your registry 6 months to bulk up with useless crud.  Even if you're really conscientious about keeping your Windows installation clean, you'll still notice a drop off in performance.  Linux performance doesn't suffer nearly as much.

Of course if you have a reasonably-sized hard drive, you can aways dual boot and have the best of both worlds.

### Post by ubun_two on 2010-02-10
Well, consider what you want to do on netbook.

If Flash is really important to you, maybe you want to stick to Win7.  Flash plugin with Ubuntu is unfortunately not as efficient as one for Windows.

If you like the ability to customize for your personal taste (both interface and look), then go with Ubuntu.  You'd be pretty much stuck with the look-and-feel of what they throw at you with Win7, and you'd be forced to get used to it.

If you like to be able to use the general Windows softwares like MS Office, go with Win7.

If you like the vast availabilities of open source softwares through repository with few clicks, and if you can live with alternative softwares, then go with Ubuntu.

Both will work well for web browing, emailing, Skype, handling simple documents, etc.

### Post by Mark Phelps on 2010-02-11
What it really boils down to is simple -- which one provides you the best of what you need?

I run several OSs on a daily basis -- Linux versions and MS Windows versions -- and each provides something unique that warrants using it.

From my experience, Win7 (on ALL the machines I use) performs every bit as quickly as the Ubuntu versions -- which came as a surprise to me as some of these machines had Vista on them.

As to viruses and malware -- yeah, while it's not a surprise that a machine-base that has 90% of the market also has 90% of the malware infections, I've been running two Win7 test machines, connected to the internet, for over six months and to date, NEITHER of them has become infected with anything.  I know that because I periodically boot from several different AV disks and run AV checks just to be sure.

I'm not an OS-hater, anymore than I'm a company-hater -- I simply use what works best for me when I need it.  And, I have found Win7 to be a vast improvement over Vista in every respect.

I'm even running Win7 on a seven-year old machine that Vista wouldn't work on -- and ALL the hardware works (by default) and all the applications (carried over from XP) work without problems.

So, if you're OK with using Win7 -- then stay with it.

However, if you want to switch over to Ubuntu (or any other Limux distro), you can do that without the cost of having to spend hundreds of dollars to try out an OS product you can't later return for a refund.

So basically, do what works best for you -- and ignore the OS-haters and company-haters.
