### Post by H3g3m0n on 2007-05-11
If your talking about having a local Ubuntu desktop on one screen and a remote connection to a Windows system on another or visa versa, you can do this by using VNC and full screening it on one desktop. You need to install VNC on the windows system, otherwise there is probably something around that can use the windows equivalent, rdesktop or some such, not sure if they will full screen as well.

Theres also synergy if you want to use the same keyboard and mouse on 2 different computers each with a different monitor.
