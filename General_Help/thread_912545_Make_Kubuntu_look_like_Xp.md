### Post by perspectoff on 2008-10-22
Yours is a question that is extremely common. 

A large number of Linux zealots hate to admit that Windows has been the dominant OS for a few decades and that people are used to how it looks and feels (and also crashes, but that is a different issue!).

Kubuntu Hardy is the first release that I have been successfully been able to make look and feel like Windows XP (which, really, isn't that bad of an OS). But XP is now gone and Vista suuucks (even now in October 2008).

I have now converted 11 computers (and their users) from diehard Windows XP/2000/2003/Vista users to Kubuntu Hardy. 

6 of these computers are in a corporate environment (and we are now converting the entire organization) and 5 were home users, kind of like your pappy. 4 were laptops with wireless (including Sprint EVDO) connections.

To make everyone happy and barely notice the transition to Kubuntu, I did the following:

1) Installed Kubuntu from the Live CD. This worked in 95% of cases without any further hardware tweaking, However,

a) On one old laptop (an old Dell Inspiron), I had to adjust the monitor resolution to 1650 x 1050 manually from System Settings.

b) On one laptop I manually installed Madwifi drivers for integrated Atheros wireless cards.

c) For my Sprint EVDO card, I had to look up the KPPP instructions for using it. Mind you, this was easier than the Windows installation. 

d) Set all sound to ALSA. PulseAudio doesn't work with much of my hardware.

e) Occasionally I had to fiddle with network printer settings. But this was a task that isn't easy in Windows, either.

That was about it. I really didn't have many hardware incompatibilties.

But Windows users are all about little cosmetic differences, mostly.

So to make things look like Windows, I fiddled:

1) I used the same wallpaper from the old Windows setup. When switching from Vista, which bundles some nice wallpapers, this was particularly easy. Believe it or not, this single step almost fooled everyone into thinking they were using Windows!

2) Made the default Kubuntu icons smaller. System Settings --> Appearance --> Icons --> Advanced --> Size 32

3) Installed (from Add/Remove Packages) Microsoft Core Fonts. This includes Times New Roman, Arial, and Courier, which the world has gotten used to. I then changed the default font in Firefox to Times New Roman 16, just like in Windows.

4) Installed Kubuntu restricted extras (from Add/Remove Packages). This adds Java, Flashplayer, and the multimedia codecs that allows your internet to run seamlessly.

5) Fiddled with the menus to make them resemble Windows (classic) arrangement.

6) Installed Audacious music player in addition to Amarok. Audacious is like XMMS and WinAmp, and is great for playing streams from Shoutcast or other internet radio. It must be set to use the ALSA output plugin.

7) Placed the Dolphin File Manager icon on the desktop. Dolphin is like File Manager and is way better than Konqueror.

8) Fiddled with the Kaffeine media player to turn off the OSD (on-screen display) and to play at double height. 

Now Kubuntu acts for 90% of stuff (Quicken, TurboTax, Paperport 9, Pirates of the Caribbean Online and other games are a few exceptions) like Windows XP. It can look like Windows Vista, but why?

I have tried (K)Ubuntu Dapper, Edgy, Feisty, and Gutsy before this, but have not been successful in the past converting Windows users. Each had problems: Samba shares (i.e. Windows network) not working, difficulty with USB or Bluetooth, difficulty with wireless connections. Those mostly seem to be solved with Hardy Heron. Also, Windows users haven't taken well to the Ubuntu desktop appearance, so I would stick with the Kubuntu desktop (in the Hardy Heron version).
