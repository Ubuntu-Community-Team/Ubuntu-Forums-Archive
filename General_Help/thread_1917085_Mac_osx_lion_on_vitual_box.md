### Post by 2F4U on 2012-01-29
Technically speaking, you could install Mac OSX in VirtualBox:

[https://www.virtualbox.org/wiki/Guest_OSes](https://www.virtualbox.org/wiki/Guest_OSes)

but legally speaking you cannot, since Apple allows the installation of Mac OSX only on Apple hardware.

### Post by coffeecat on 2012-01-29
@khaj_vah, installing MacOS in non Apple hardware is not permitted by the Apple EULA. From the Apple EULA:

> you are granted a limited non-exclusive license to install, use and run one (1) copy of the Apple Software on a single Apple-branded computer at a time. You
agree not to install, use or run the Apple Software on any non-Apple-branded computer, or to enable others to do so.

And from the forum [Code of Conduct](http://ubuntuforums.org/index.php?page=policy):

>  We do not support circumventing TOS, EULA, etc here.

Therefore I have closed this thread.

You can install Linux/Ubuntu in a Macbook and there is a whole forum here devoted to using Apple hardware. See:

[http://ubuntuforums.org/forumdisplay.php?f=328](http://ubuntuforums.org/forumdisplay.php?f=328)
