### Post by p_quarles on 2008-12-04
Moved to General Help (which is for support requests, while the community area is not).

My guess is that you've opened a copy of the document that is set as read-only. There are occasionally permissions problems with moving removable media between Windows and Linux. If the document is on a CD, of course, it's read-only due to physical limitations on the hardware. 

In either case, the fact that you can read it means you can also save a file elsewhere, and you should get the appropriate permissions over that copy.
