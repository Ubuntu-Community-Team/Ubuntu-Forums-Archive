### Post by dbloom on 2011-07-11
sometimes when changing a setting in compiz, unity goes a little haywire.  if alt-F2 is working for you, enter "unity --replace" and press enter.  that should restart unity for you.  

if alt-f2 doesn't work, try Ctrl-Alt-T to bring up a terminal -- this time try "unity --replace &&".  the && will allow it to run independent of the terminal so unity will stay running when the terminal is closed.

compiz and unity work fine -- don't give up just yet.

good luck.
