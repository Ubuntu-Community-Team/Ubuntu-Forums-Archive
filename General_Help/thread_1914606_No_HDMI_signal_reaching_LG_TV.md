### Post by YfoMp5QBh2 on 2012-01-25
I had this particular problem with a cheap HD ready TV I bought while woolworths was still alive!

If you install the drivers for your graphics card from the Nvidia website it usually installs a GUI called Nvidia control panel which will help you reconfigure xserver.

After you have done that it's simply a matter of making sure both TV's / monitors are switched on, click "detect displays", select twin view, then select clones from the various drop down menu's.
