### Post by ghost_ryder35 on 2008-10-26
[http://en.wikipedia.org/wiki/Telnet](http://en.wikipedia.org/wiki/Telnet)
As the wiki page says, telnet has been almost replaced by SSH.  Unless you are running some legacy application that requires it I would not use telnet.  I would setup SSH
```

sudo apt-get install openssh-server openssh-client

```
