### Post by scooper on 2007-05-28
FWIW - If what you want is a simple Wiki for personal information I'd look at TiddlyWiki.  It lives (javascript program and your data) in a single HTML file and supports saving, etc.  It can go on a USB stick and works multi-platform.  If you need remote access there are many people with server-side solutions.  Having the whole LAMP stack and Mediawiki seems like more complexity, but I don't know your exact needs. :)
