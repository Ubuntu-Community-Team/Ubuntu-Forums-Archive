### Post by cybergalvez on 2011-06-30
Ok so after giving Natty a try I finally get it and think its great! When I first say it I was very skeptical of the new interface, and felt it was too mac like (I came to ubuntu from windows XP), but I now see how they have taken a couple of the good features from windows7 along with the document centric features of the mac and put together a very nice user experience. I can't wait to see how unity evolves and matures. I know there are still some usability issues, and I hope the Ubuntu team manages to work them out.

Now after some trial and error, including dumping my fakeraid setup for a standard setup, adding cube in-place of the wall, and adding screenlets it is finally running smooth and stable. 

Thanks for the new interface, and please keep making it  better!
