### Post by airtonix on 2008-09-02
not sure if this will help you, but if your familar with the use of the inspection tool in the firefox addon firebug then you should check out ginspector.

```
sudo apt-get install ginspector
```To get an idea of which elements you might need to look at twiddling with firefox use ginspector like so : 

```
g-inspector firefox
```When you find the g-inspector window (which is quite small & maybe underneath your target application), have a look around in the view menu at the various inspectors.
