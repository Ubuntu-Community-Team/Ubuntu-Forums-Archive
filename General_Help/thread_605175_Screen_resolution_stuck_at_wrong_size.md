### Post by jonathanmotes on 2007-11-06
You have to install the nvidia restricted driver to get Compiz to work. Compiz will only run after the initial installation if your graphics card is supported with an open source driver. So, all you have to do is to go to System --> Restricted Drivers Manager and select the driver it gives you. It will download and install it for you. That should also give you more screen resolution options.
