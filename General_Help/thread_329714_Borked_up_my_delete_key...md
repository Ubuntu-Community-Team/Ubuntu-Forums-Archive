### Post by zanglang on 2007-01-02
I had my X key accidentally mapped to a multimedia key a few times... Not fun indeed. Anyway, changing both the keyboard layout and model settings (in System -> Preferences -> Keyboard -> Layouts) to something else, deleting the default layout and then back again worked for me. :)
