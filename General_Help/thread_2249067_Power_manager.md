### Post by Dennis N on 2014-10-19
It is probably running after you clicked run - it runs in the background and is not visible to the user. To verify is running, you can look for it in the task manager, or use the **top** or **htop** command. Be sure it is included in the startup applications to have it running all the time.

### Post by Dennis N on 2014-10-19
Xubuntu uses Light Locker, which will blank and lock the screen after a certain period of time. You can adjust its settings (or disable it) in Settings > LIght Locker Settings.
