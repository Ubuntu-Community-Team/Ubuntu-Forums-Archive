### Post by gordintoronto on 2011-02-12
You've hit the nail on the head: the question becomes, what services do you want to run?

My opinion is that a *home* server should be based on the Ubuntu Desktop package, not Ubuntu Server. Then you just install the services you want, one by one. I think there are individual tutorials, how-tos, youtube videos or community docs (or all of the above) for each of the services. To pull everything together in one place would require a thick book, which would be out of date within six months.
