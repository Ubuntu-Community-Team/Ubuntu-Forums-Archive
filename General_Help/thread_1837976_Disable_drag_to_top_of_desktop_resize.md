---

### Post by ajgreeny on 2011-09-02
I agree it is extremely annoying!

Luckily it is easy to stop, though not well documented.

Install **compizconfig settings manager**, (ccsm) and in the **grid** plugin you can set the many drag options to screen edges to do nothing.
