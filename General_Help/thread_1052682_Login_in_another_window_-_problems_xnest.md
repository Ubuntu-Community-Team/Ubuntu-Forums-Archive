I was trying to login in a window, where you can login another user in a window on the current desktop. However i kept getting an error "X Server failed. not setup correctly" For anyone who might be having similar errors, install 'xnest' by typing 'sudo apt-get install xnest' then type in your password. After the package is installed it should work
