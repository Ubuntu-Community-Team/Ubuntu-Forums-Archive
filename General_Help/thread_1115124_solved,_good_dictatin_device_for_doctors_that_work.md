### Post by andylinux on 2009-04-03
Anyone looking for a dictation device that works well with Ubuntu (Linux in general I assume), I would like to recommend the Philips DPM 9370 Digital Pocket Memo.  My wife is a Doctor and I have been trying to get her to switch to Linux but the problem was her old Olympus dictation device would not work on Linux.  
Well it finally died and after considerable research we got the Philips 9370.  It mounts like a usb hard drive so you can copy the files file right off of it using any file browser.  The industry standard file format seems to be *.dss files which I haven't figured out how to play on Linux.  However, I only need to be able to download the files on Linux so they could be sent to the transcriptionist.

### Post by pmcchapman on 2009-06-03
I too have been looking at dictation devices that are useful for medical staff. I agree, the "professional" type products from Philips (9370, 9500 or 9600) work really well as they mount as a removable drive. These are the best, as unlike cheaper note-takers, they allow you to rewind/insert/overwrite files rather than having to start a new file, i.e. excellent for correct mistakes when dictating. On the downside, they use the 'dss' format which is proprietary, and I do not believe there is any native linux support for this file type. It does however seem the that Olympus DSS Player Lite 2 works OK using Wine.

Conversely, if you are just looking for a note-taker, the Olympus WS-110 and WS-210 are excellent, mount as a removable drive, and record in 'wma' (still proprietary, but can at least be played back with relative ease). These are much, much, cheaper, but cannot do the rewind/insert/overwrite etc.
