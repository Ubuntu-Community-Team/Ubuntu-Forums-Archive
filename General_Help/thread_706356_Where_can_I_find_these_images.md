### Post by seventhc on 2008-02-24
the folder would be **.themes**, it is a hidden folder. (notice the dot in front of it.)
hit <*Ctrl-h*> when in your home directory to show hidden files.
I'm not sure if that folder exists by default or if it's created after installing a new theme though.
To get more themes have a look here
[http://gnome-look.org/](http://gnome-look.org/)
On mine the ubuntu logo for the main menu has been changed to the gnome foot since that is what I like. That sites has a lot of different themes, so give it a look. :)

### Post by seventhc on 2008-02-24
i use 4 different themes together, but this is the one that will give you the foot
[http://gnome-look.org/content/show.php/Tango-noir?content=38964](http://gnome-look.org/content/show.php/Tango-noir?content=38964)
I don't particularly like this theme by itself which is why I use a combo.
heres what I use:
1: Install-
         Tango-noir-8.0 theme,
        Tropic-look 0.2 orig theme,
        and outdoors theme 0.1 orig theme,
        Chameleon-Anthracite-0.5. theme

    2: Setp-
**         Controls **= Outdoors theme,
**         Window border** = Tropic look theme,
**         Icons** = Tango-noir theme,
**         Pointer** = Chameleon-Anthracite small"""

these two can be installed with apt-get if you want to try them
```
sudo apt-get install outdoors-theme
sudo apt-get install tropic-look
```you right click on your desktop>Change Desktop Background
click on themes and just drag the tar.gz files into it.
I then select human again since it changes once you select use new theme.
Then I click on customize and make it how it is from the list above.
for the tar.gz files i just dr

### Post by seventhc on 2008-02-24
The human theme folder is the default theme being used.
So when your in the themes list by right clicking on your desktop>etc. It's in there.:)
So I keep it set to human, then customize the rest same as in this list.

**         Controls **= Outdoors theme,
**         Window border** = Tropic look theme,
**         Icons** = Tango-noir theme,
**         Pointer** = Chameleon-Anthracite small

they're are so many different ones to choose from, but this is the only example I have since I have used this set up for quite some time.
