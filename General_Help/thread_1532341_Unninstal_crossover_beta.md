### Post by gdonwallace on 2010-07-16
If you downloaded it from codeweavers website, it should have been in  a .deb file.

To remove this from your system, open Synaptic and you should be able to find it there and mark it to remove.

If it is not there, try the following:
**sudo dpkg -r [program_name]** where program_name is the name of .deb file your used to install crossover.  (i.e. crossover8.0.deb or whatever version you have installed.)
