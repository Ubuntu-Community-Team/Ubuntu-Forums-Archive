### Post by Archmage on 2008-09-16
If you are runing an Alpha, than all automatical updates will update to the next version of this release, if it is out.

So if your update an Alpha 5 when the Alpha 6 is out, than you will get automatical an Alpha 6 with the latest updates.

If the final is out than an update from Alpha 6 will bring you to the Final with the latest updates.

If you have the Final and do the updates, you will get an updated Final.

But kindly DO NOT run "update-manager -d" from the Alpha of 8.10 since this will bringt you to the LATEST devlopment-version. E.g. if 9.04 is already in the repro, than this would upgrade 8.10 to the final and than start with going to the 9.04. :lolflag:
