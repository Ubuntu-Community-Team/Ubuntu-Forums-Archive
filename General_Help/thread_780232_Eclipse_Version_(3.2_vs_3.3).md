### Post by Olav on 2008-05-03
This is a well known [problem]("https://bugs.launchpad.net/ubuntu/+source/eclipse/+bug/123064") with Ubuntu. They seem to have been trying to get Eclipse 3.3 into Ubuntu since quite a few months (read the comments on the bug page to learn something about the difficulties). Meanwhile, people are just [downloading]("http://www.eclipse.org/downloads/") and installing Eclipse 3.3 themselves. It is not hard to do.

You are correct that for very simple / straight forward Java programming, Eclipse 3.2 will do just fine. The problem is with all the fancy new plugins that people want to use that require 3.3.x. Eclipse is quite a bit more than just an editor as you know.
