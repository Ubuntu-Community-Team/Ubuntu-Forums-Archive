### Post by oldfred on 2019-02-14
If the screen listing is not full, you can can right click on the blank space below your other files & folders.
Or change to icon view and click on space between icons.

I just create a new document in /home and copy that into folder. If one level up, you can copy into folder, but if in that folder & screen is full same type of  issue as paste will not appear.

### Post by Tom_Carr on 2019-02-14
Thanks.  I also just learned another way:

In the upper right corner, click on the 3 bar icon.  It brings up a window, and in the upper left there is an icon that looks like a file folder with a plus in it.  Click on that icon to create a folder.
