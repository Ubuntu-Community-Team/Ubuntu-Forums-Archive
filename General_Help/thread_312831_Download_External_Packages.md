### Post by sam81 on 2006-12-05
do you have your ubuntu machine connected to the internet? if so have you enabled all repositories (universe, multiverse)? there are around 12.000 packages there, that's really lots of software!!

if you've not enabled them you can do it going to system>administration>software sources from the main menu. if for some reason you have problems accessing internet from your ubuntu machine you should try downloading an ubuntu dvd from another machine.
