### Post by spiderbatdad on 2009-12-17
not sure, but I can tell you I have a /root directory with permissions 700 and a /root/Desktop with 755
so you might try
```
sudo -i
cd /root
mkdir Desktop
ls -al
exit
``` ls -al will list for you, and I believe the default permissions will be 755, or drwxr-xr-x. Observe upper and lower case letters for commands and directory names.
