### Post by nancy_t on 2007-03-18
Hi
I'm not sure if you have the same problem as I did. When I first used OO there was spell checking available. But then, for the next document it was gone. But here's how I got it back:

Highlight all of your document, on menu click *format*->*character*
In the Font tab, right in the middle of the window, there's a *Language*menu with a drop down button. Go through it and find one with a blue check mark next to it and select that one. 

Hope this helps. 
N

### Post by Icebear on 2007-03-19
When I first used open office in Ubuntu the spell checker did not pick any spelling mistakes. The problem was that I had Australian as my default language but for there was no dictionary for Australian English. I then changed the default language in open office to UK which has a dictionary and it now works.

in open office go >tools>options>language settings>language>and then change the default language for documents
