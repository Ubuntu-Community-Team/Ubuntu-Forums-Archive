### Post by sedawk on 2010-05-12
You could open some terminals and run a bunch of
commands inside the terminals, e.g. one would be
"while sleep 60;do dmesg;done"
Check if there are any problems reported in dmesg after
the freeze, e.g. disconnecting USB devices.

If you use USB keyboard and USB mouse you
can try to unplug them for a few seconds and then reconnect.
If they are not USB devices you can try to connect USB keyboard
or USB mouse to see if they stop working too.
