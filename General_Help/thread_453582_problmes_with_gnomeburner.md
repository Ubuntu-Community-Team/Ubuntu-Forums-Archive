### Post by jamesjtucker on 2007-05-24
I dont know much about gnomeburner other than i tried it once and couldnt stand it. (one of the main reasons i use kde is k3b). 
 For the lockup, was it a hard lockup? IE no keyboard/mouse/etc? If so then there is not a whole lot you can do. 
If you still had some control, you can do CTRL-ALT-F4 (or f3 f5, etc) to get to a prompt and then kill the offending process or reboot the computer. If that fails you might want to try CTRL-ALT-Backspace to try restarting  X, which may not kill the process, but will allow you to log back in and fix things.
