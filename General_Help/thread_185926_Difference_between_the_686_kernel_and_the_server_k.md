### Post by jonibo on 2006-06-01
Server kernel is tuned for server workloads... this essentially boils down to it being less responsive for desktop use.  I doubt if you'd notice a difference, though... perhaps the odd delay now and then between a mouse-click and the system response.

Unless you are running a heavily loaded server, it does not really matter which kernel you pick.  But if you're not running a desktop on your P3, choose the server kernel, just for fun, if nothing else...!!!
