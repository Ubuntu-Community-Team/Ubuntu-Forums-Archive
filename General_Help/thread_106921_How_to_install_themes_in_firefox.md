If I click on "tools" and then select "themes" in the firefox browser I see "Get More Themes" and that takes me to a web page by the good folks at mozilla. ### Post by wylfing on 2005-12-21
It's just doing its job protecting you from maliciousness. Themes and extensions can be installed from lots of different web sites, not just the official one, so all installing is blocked by default.

So you need to let Firefox know what sites you trust. When that yellow bar appears on the top of the screen telling you that it blocked something, click the bar (not the X to close it -- just the yellow part). That should show you options for allowing that site to install things in Firefox.

After you've done that, try installing the theme again.
