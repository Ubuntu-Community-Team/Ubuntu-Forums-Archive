### Post by hotani on 2006-10-13
Ever since embarking on this journey that is Ubuntu, I have not yet found a way, or good resource for replacement icons. There are many an icon theme out there, but a theme replaces all or most of the icons and all I'm looking for is one single icon. 

Here are the solutions I've found so far:
1- find a site that has the one icon. Example: in the recent HOWTO for Songbird, the author posted a nice png icon for us to download, drop in /usr/shared/pixmaps and link to the shortcut. However, that is a rare case. 

2- download an icon set and hunt through the directories for one that fits. So far I haven't had much luck with this.

Just FYI, it is the standard rhythmbox icon that *must* be replaced, so if anyone knows of a good one floating around out there let me know. :)

EDIT: I found a nice one, and using [this theme](http://www.silvestre.com.ar/?p=56) I went with option 2 above. Although there was no specific rhythmbox icon, the generic audio mime icon is a nice fit. Still, this process is cumbersome - it would be nice to have a way to just download a collection of single icons. :confused:

### Post by electricshoes on 2006-10-13
You can usually change an icon without a problem by right clicking on the app and go to properties, click on the icon in the top left corner and then you can browse to that 1 icon you want.

Also if it is in the menu, right click - add to desktop, right click on it on the desktop and do the same thing, click on the icon, choose a new one, then drag it back to the menu.
