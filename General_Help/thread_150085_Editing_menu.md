### Post by naruto11111 on 2006-03-25
I'm not entirely sure if this is your question, but try right clicking on the actual main menu button ( the little ubuntu sign on your taskbar) or otherwise opening the menu editor application (you shouldn't have to install one, it comes with the distro).  If you use the right click, select Edit Menus and just unclick the application you want to remove.
