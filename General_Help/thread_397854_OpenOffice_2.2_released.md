### Post by jem7v on 2007-03-31
Basically, about a month before a release is made official they freeze the version of all the software in the repositories to make sure it'll all work for that release.  The only updates they do are usually bug fixes and security updates.  This is to make sure that you don't end up installing a newer package with dependencies that can't be filled because the packages it depends on aren't up-to-date or stable enough... It's still possible to obtain and install those packages, but it can be a pain in the butt Sometimes.  So OpenOffice won't be showing up in your repositories until you upgrade to Feisty.  BUT you could still install it manually like Maestro23 said.
