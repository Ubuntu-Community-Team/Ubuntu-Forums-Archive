### Post by TheFu on 2015-12-08
Cannot help, but 15.04 support ends in a few weeks, so after you get fully patched, it would be smart to move the system to 15.10 which also only has 9 months of support.  16.04 is what you probably want, this will be a 5 yr supported release - called LTS in Ubuntu-speak.

15.10 introduces a few new things which should impact suspend and possibly bluetooth. Move to that release and see if things are fixed. If not, working on the solution won't be useful for just a few weeks, like the 15.04 release would be.

Release Support Table: [https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases) - use this to plan.
