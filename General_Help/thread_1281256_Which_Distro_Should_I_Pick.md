Try out a couple, a LIVE CD doesnt obligate you to any thing (except media unless you burn them to USB :P) and even if you find a distro you like there are different Desktop Managers available for each distro as well.
Theres GNOME, KDE, XFCE, and many others.

Check out the different Desktop Managers to find one you like from an appearance stand point then look into what makes the distros different from the others.
Redhat based distros like Fedora or OpenSUSE uses .rpm packages where as Debian based distros like Mint or Ubuntu use .debs.

### Post by carnagex420x on 2009-10-05
1.How to install blocked updates
- Not to sure what you mean by "Blocked Updates"

2.How to know that the ram and motherboard drivers are installed.
- Theres no such thing as RAM drivers, and as far as your MoBo gos as long as all your hardware is present and functioning the drivers are already there.

3.Do Beryl and compiz start by default or you have to start them.
- Beryl and Compiz merged a while ago and are now Compiz Fusion, but it is still known as Compiz. In order to get compiz to run you need to have the right video drivers installed to allow advanced 3D acceleration. I dont use KDE but GNOME prompts for the install of them automatically when selecting the level of desktop effects you want to use.
