### Post by lisati on 2010-07-22
Ubuntu is an [operating system]("http://en.wikipedia.org/wiki/Operating_system"). This means that it's a little different to your regular program.

And yes, it comes with an option, [wubi]("https://help.ubuntu.com/community/Wubi"), that lets you set it up *as if* it was a program that can be deleted from the Windows "Add/Remove programs" menu.
