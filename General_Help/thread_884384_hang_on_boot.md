### Post by bdfoster on 2008-08-08
Reinstall?

I had that same problem. Honestly, it is probably better not to use Wubu. I would download your version of Ubuntu from Ubuntu.com and install it in a partition. You should be able to install it giving it it's own "drive" and it also should use it's own boot loader for dual-boot. 

The magic thing is that you should not have to uninstall Windows to install Ubuntu (although I would back up everything onto some DVD's)

If you have a CD burner, download the ISO image from Ubuntu's website, burn and install. If not, you can request the CD and it will get to you within 6 weeks or so.

The Ubuntu that you had previously used can be deleted by going to the "ubuntu" directory in Windows and deleting it. That's it! Just make sure you do that before you install the new OS.

I hope this answered your question. Although I have no solution, I have a workaround.
