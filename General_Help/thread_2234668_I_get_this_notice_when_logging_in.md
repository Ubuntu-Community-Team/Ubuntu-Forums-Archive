### Post by grahammechanical on 2014-07-17
On the desktop when we install Ubuntu/Kubuntu we set a language keyboard layout. The installer then installs stuff like dictionaries for the language/keyboard layout. When we add another keyboard layout we get the new layout but we have to choose to install the language packs. On my Ubuntu Desktop I have an additional Greek keyboard layout but I do not get that notification but if I go to System Settings> and run Language Support then I get the warning that language support is not installed completely and I have an opportunity to install.

I do not know how things are done in Kubuntu but it must be something similar. Look for a Language Support utility if you want to install whatever the system is offering or to click Remind Me Later. That might make this notification go away.

Regards.
