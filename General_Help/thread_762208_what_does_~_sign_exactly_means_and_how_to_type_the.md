### Post by Rocket2DMn on 2008-04-21
Files and folders that begin with a . are hidden files/folders, so to view them in nautilus, you do CTRL+H, or to see them from the terminal, you do
```
ls -a
```
or even better
```
ls -la
```

### Post by retrow on 2008-04-21
In your file manager got to [COLOR="Red"]View > Show Hidden Files[/COLOR]

That would list all the files that are stored in ~/.xxxxxxx folders.
