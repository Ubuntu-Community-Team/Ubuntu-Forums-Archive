### Post by grahammechanical on 2013-01-05
You should be able to upgrade using the Update Manager. If you change the notifications setting so that you are notified of the latest Ubuntu release, you should get an invitation to upgrade to 11.04 and then to 11.10 and then to 12.04.

If you have burned Ubuntu 12.04 to disk then you can do a fresh install using the Do Something Else option.

If you want to upgrade to 12.04 from the CD, then you must tell Synaptic Package Manager to make the CD/DVD one of the Software Sources. Synaptic should be able to read a label from the CD.

It looks to me that Synaptic is asking you to give a label of your choice. Have you tried writing in a label such as Ubuntu 12.04.1 DVD? May be that is all you need to do.

[https://help.ubuntu.com/community/UpgradeNotes]("https://help.ubuntu.com/community/UpgradeNotes")

[https://help.ubuntu.com/community/NattyUpgrades]("https://help.ubuntu.com/community/NattyUpgrades")

[https://help.ubuntu.com/community/OneiricUpgrades]("https://help.ubuntu.com/community/OneiricUpgrades")

[https://help.ubuntu.com/community/PreciseUpgrades]("https://help.ubuntu.com/community/PreciseUpgrades")

Regards.
