### Post by SunnyRabbiera on 2008-07-29
Yeh I noted this too, there is a semi easy fix.
go up to your menu and right click it and select "edit menus"
in here you have to go to said application you want to have an icon for and set up the icon manually.
Say you want to have an icon for one of the kde4 games like patience, go to where its located (games)
and double click the name of the application
Right now you will note there is no icon for it, hit the choose icon button and let the icon loader dialog and click browse and make your way to /usr/lib/kde4/share/icons/oxygen and select a size...
The oxygen theme is kind of incomplete, you may with to install more icon themes and such to make up for the lack of icons... but this can get you started, if you download a icon theme and install it with the gnome theme installer just direct your way to home/yournamehere/.icons folder and play around a bit.
