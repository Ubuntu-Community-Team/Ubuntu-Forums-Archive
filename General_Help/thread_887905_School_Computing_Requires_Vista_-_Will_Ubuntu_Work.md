Powerpoint 2004 on Intel-based Macs

Some students who try to view Powerpoint presentations under Powerpoint 2004 on their Intel-based Macs have reported several issues. The main issues are:

   1. Links embedded in the Powerpoint file do not work.
   2. Audio embedded in the Powerpoint file does not play.

Students who are experiencing the above issues are encouraged to upgrade to Powerpoint 2008 for Mac. SLIS has verified that the issues have been resolved under Powerpoint 2008.

Students can obtain Office 2008 for Mac (which includes Powerpoint 2008) at a discounted rate ($79.99) through the SJSU Spartan Bookstore or the CSUF Titan Bookstore. See also Creation Engine under the heading Microsoft Office and Academic Pricing above.

### Post by WRDN on 2008-08-12
Ubuntu works on many systems, so the hardware isn't a big worry. You may find the [Hardware Support]("https://wiki.ubuntu.com/HardwareSupport/") documentation useful.

Now for the software:

- Ubuntu includes the firewall, iptables, which can be configured through UFW (Uncomplicated Firewall)
- Regarding Anti-Virus, if you use Linux, you most likely won't need antivirus, though it can be useful if you want to send Windows users data- it may be courteous to scan the file first to ensure it is not virus infected)
- Anti- Spyware (see above)
- Ubuntu includes the "Evolution Mail" email program, which will accept programs among other features
- For a web browser, Ubuntu comes with Firefox 3.0 already
- You also should not need Adobe Acrobat Reader, because Ubuntu comes with a PDF viewer as standard
- Ubuntu also includes Open Office which includes a word processor, spreadsheet, database, presentation and drawing program. This can be used as a brilliant replacement for Microsoft Works/ Office
- Finally, realplayer, and adobe flash player can be easily installed. For the flash player, you will be prompted in Firefox to download the plugin. Easy!
