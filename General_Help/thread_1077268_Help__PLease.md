### Post by makisupa123 on 2009-02-22
Make sure the package 'pdf-viewer' is installed.  There is no Adobe licensed version of acrobat/reader -- there are only <very good> third part tools.

```
sudo apt-get install pdf-viewer
```

Here is info on the package:

[http://packages.ubuntu.com/intrepid/pdf-viewer](http://packages.ubuntu.com/intrepid/pdf-viewer)
