### Post by mrsteveman1 on 2008-02-21
You can't really "import" the torrent but all torrent clients can check existing files and continue from where you left off.

So what you want to do is just add the torrent to the new program and point it to the same directory that already holds the files, the new program will check the pieces of the files and start where you left off.

Note that you won't be keeping any statistics etc, in effect you are just getting the new program to finish the files.

### Post by disturbedite on 2008-02-21
in ktorrent, you can simply **import** (File --> Import) the data & torrent.  (just point it to the torrent & then to the data).
but ***don't open*** (File --> Open) the torrent & then point it to your data, as it will overwrite the already downloaded data.
