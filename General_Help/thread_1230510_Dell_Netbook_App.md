A little googling around and I found out the launcher you refer to is something Dell custom created for its own version of Ubuntu Netbook Remix. I really haven't found out the name of the package but I suppose one could perhaps add Dells repos and try to find the package and try to install it that way. Just be aware that from what I read online Stuff in Dells repos may not be 100% compatible with Ubuntu as Dell has modified UNR somewhat. therefore if I were to add Dells repos to install that application I would remove dells repo after I installed the needed application.

Edit: There probably are legal issues with adding  Dells repos and using software that might be and probably is copyrighted by Dell. A better solution would be to find another application launcher you like as well in Ubuntus repos.
