### Post by nikolai.m on 2005-11-12
openoffice2 (build 1.9xx) which came with kubuntu 5.1 has an issue with Kmail (or should i say that i have an issue with OOo for not recognizing Kmail). when i open OOowriter and trying to send letter that i just typed by clicking "document as email" button on main toolbar i get error message: OOo was unable to find working email configuration". well... i got working email configuration - Kmail....what's up with that? how can i fix it?
thnx

---

### Post by adwait on 2005-11-13
Try going to Tools>Options>Internet>Email and then change the program name to kmail. When you click the send as email button, it will open a new mail in kmail with the file attached.
