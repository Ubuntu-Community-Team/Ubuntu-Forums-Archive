### Post by efflandt on 2014-01-08
Alt+PrtScn captures the window with focus (just like in Windows). If Dash is open it has focus. Therefore Dash IS the window you are in. If focus is anywhere else, Dash closes.

You might want to look at the **Screenshot** app in Dash if you want to capture to a file the current window, full screen, or a selected area, with or without cursor, and possibly using a time delay to get everything like you want it before the capture.
