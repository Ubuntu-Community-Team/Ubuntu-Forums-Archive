### Post by nbkr on 2010-08-19
> **Lakeside5 said:**
> I want to use it as a server to host my website.  I used Hardy Heron before and had a gui, but my server got hacked!


Ok, on such a machine you shouldn't install a GUI, it rises the risk of being hacked. 

Also know what you are doing, running a server that has a public ip address isn't an easy thing to do. And you should remember that you are always responsible for that machine. 

If someone hacks that machine and uses it to send spam or distribute illegal material the authorities will ask you why _you_ did this. Saying that you got hacked won't help in such cases.
