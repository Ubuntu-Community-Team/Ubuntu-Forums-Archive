### Post by 3rdalbum on 2011-09-05
> **mike555 said:**
> Wine will run Microsoft office ,

Depends on whether it's Office 2010 or Office 2007 (or Office XP which some schools STILL use). I don't think Wine runs 2010, but feel free to correct me if I'm wrong.

> or if you have a XP license you can use virtualbox .

This. Virtualbox creates a "virtual machine" - it's like a virtual computer that runs within a window on your Linux desktop. Now, you don't have a CD or a USB flash drive, but that's okay. You can download Windows XP from Bittorrent, tell Virtualbox to boot its virtual machine from the CD image you download, and put in your own legitimate Windows XP license code when the Windows installer asks for it.

Just to be clear, I'm not advocating piracy. If you have a Windows XP license code but have lost your Windows CD or you don't have a CD drive to put it into, then I think it's still reasonably legal to download the Windows XP disc image from Bittorrent as long as it's not a cracked version, and you put in your own license code. I Am Not A Lawyer though, so you might want to get proper legal advice (or buy a new XP CD?).
