### Post by Rob_H on 2009-07-23
If your goal is to get a web site up as quickly as possible and you're not particularly interested in learning Linux, you might want to consider purchasing a subscription from a web hosting company. They will set you up with a basic server, usually, for a few US Dollars per month. Alternatively, your Internet Service Provider may offer some free web space on their servers if you just want to post some pages. If you're interested in staring a blog, there are services like WordPress and such.

The bottom line is that unless you're really interested in doing it yourself and taking on the responsibility of running a secure web server on the Internet, my advice would be: Don't do it. Let someone else do it because running a public web server is not easy if you're new to the concepts.

On the other hand, if you are really interested in doing it yourself, I would suggest googling **LAMP tutorial** first to get some background information, and then download [Ubuntu Server]("http://www.ubuntu.com/getubuntu/download-server") and choose the LAMP installation option. (LAMP is shorthand for a Linux computer running Apache web server software.)

### Post by Avidanis on 2009-07-23
To serve pages from your pc you need several tools. One popular setup is to have Apache,MySQL, and PHP. You also need to have these tools configured properly with one another. 
The most painless way I have found to do this is by downloading the LAMP package from [here.]("http://www.apachefriends.org/en/index.html") This will do much of the configuration for you.

If you are clueless about Apache and Mysql and PHP and Perl , then you will need to do some homework. Setting up your pc to share with others opens up your pc to all kinds of potential threats if you don't know what you are doing. 

I have many books and have researched this and spent countless hours toying around with this in the past. I still would not feel comfortable about administrating my own server even though I could.

My advice is to download the package from Apache Friends (the link above) and play around with it ) You need to educate yourself a bit about localhost , static IPs and a bunch of other stuff.

Its fun and I won't try to discourage you but I would also suggest that you shell out the money for a hosting account online until you familarize yourself with setting up your own server. Its not something for the beginner to tackle. For a few dollars a month you can get a remote hosting account online .
