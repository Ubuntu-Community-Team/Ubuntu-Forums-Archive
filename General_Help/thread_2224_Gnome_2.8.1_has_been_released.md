### Post by renato on 2004-10-26

The first point release of the stable 2.8.x series of GNOME has been released. This release includes the latest bugfixes and other improvements such as updated translations and is the first in a series of point releases.

Sources will be available from the following URLs when the mirrors have synced:

Platform:
[http://ftp.gnome.org/pub/GNOME/platform/2.8/2.8.1/sources](http://ftp.gnome.org/pub/GNOME/platform/2.8/2.8.1/sources)
Desktop:
[http://ftp.gnome.org/pub/GNOME/desktop/2.8/2.8.1/sources](http://ftp.gnome.org/pub/GNOME/desktop/2.8/2.8.1/sources)
Bindings:
[http://ftp.gnome.org/pub/GNOME/bindings/2.8/2.8.1/sources](http://ftp.gnome.org/pub/GNOME/bindings/2.8/2.8.1/sources)

Hope to see it in apt soon.
