[http://gridpt1.fe.up.pt/mlopes/blog/index.php/2005/11/19/kiso-packages-for-ubuntu-breezy/]("http://gridpt1.fe.up.pt/mlopes/blog/index.php/2005/11/19/kiso-packages-for-ubuntu-breezy/")

Install kiso:

```

cd ~
mkdir kiso
cd kiso
wget http://gridpt1.fe.up.pt/mlopes/blog/wp-content/kiso_0.8.21_i386.deb
wget http://gridpt1.fe.up.pt/mlopes/blog/wp-content/libcdio6_0.761_i386.deb
wget http://gridpt1.fe.up.pt/mlopes/blog/wp-content/libiso96604_0.761_i386.deb
sudo dpkg -i *.deb

```
