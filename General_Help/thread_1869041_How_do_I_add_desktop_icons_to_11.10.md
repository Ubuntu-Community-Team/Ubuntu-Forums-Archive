### Post by sanderd17 on 2011-10-25
I don't believe gnome-tweak-tool is installed by default on Ubuntu 11.10, so go to the software center and install that first.

Then open the tweak tool (it will probably get a name like "advanced settings"), and check the "filemanager" tab. There you can choose to let the file manager handle the desktop.

This is at least how it works on Gnome Shell. And because Unity uses Gnome 3, I believe it will also work with Unity.
