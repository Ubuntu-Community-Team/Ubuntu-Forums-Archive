### Post by Erik1984 on 2011-08-27
I'd like to know where Ubuntu stores the 'system' wallpapers. There is a 'cosmos' slideshow you can choose and I would like to set one of these images as my desktop background (and not the whole show).

---

### Post by JRV on 2011-08-27
/usr/share/backgrounds

The image you are looking for is in /usr/share/backgrounds/cosmos
