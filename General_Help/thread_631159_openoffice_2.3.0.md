### Post by Tyche on 2007-12-04
In Synaptic, look for msttcorefonts.  This will give you the basic MS fonts.  Then, unless you have and are using some non-core font on the windows machine, your document should look the same on both machines.

BTW, this is also a problem between MS machines, where one machine has a font that a different machine does not have.  The reason is that the program wil substitute a font for the missing one, but there is seldom a perfect match between the two.

Hope this helps
