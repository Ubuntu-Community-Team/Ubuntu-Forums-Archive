### Post by s3MA00RRNY on 2009-08-15
You probably had Compiz's Wobbly Windows turned on. Go to System > Preferences > Appearance > Visual Effects. Click Extra. All done. If you want extra control over the graphics features, install compizconfig-settings-manager.
