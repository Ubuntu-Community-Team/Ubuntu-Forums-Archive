### Post by hailtothethief on 2007-11-25
What you could do is find the program in Add/Remove programs, then start up the Synaptic GUI:
System > Administration > Synaptic Package Manager

Then click 'search' at the top of the screen and enter in the 'English style name'

It will return a list of packages that contain the program's name
