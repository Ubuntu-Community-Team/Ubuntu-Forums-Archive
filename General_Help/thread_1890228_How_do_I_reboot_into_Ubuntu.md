### Post by Basher101 on 2011-12-03
you still have the installation media for Ubuntu (the CD or USB you made, otherwise you could not have installed it...). You need that same installation Disc (or USB) for Ubuntu to repair the GRUB bootloader so you can get the boot options for Windows AND Ubuntu.
