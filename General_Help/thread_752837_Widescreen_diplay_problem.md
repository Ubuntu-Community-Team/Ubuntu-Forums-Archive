### Post by Rocket2DMn on 2008-04-12
Right now you are using the failsafe "vesa" driver, which is very limited in capacity.  When you do the reconfiguration, you need to make sure you select the "intel" driver.  You can also try having it autodetect everything by running the reconfiguration with the -phigh flag, but since it doesn't seem to have detected to begin with, you will probably have to do it manually.   Here's the command anyway
```
sudo dpkg-reconfigure xserver-xorg -phigh
```
Remember, when you make changes to xorg.conf (through reconfiguration), you need to restart X for the changes to take effect, so do the reconfiguration I first suggested, select the "intel" driver, then when it's done, do CTRL+ALT+BACKSPACE to restart X (the GUI).
