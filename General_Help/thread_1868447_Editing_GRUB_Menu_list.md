### Post by Mark Phelps on 2011-10-24
Don't use Startup Manager -- it doesn't work well with recent GRUB versions.

To change the default OS, select Preferences.  The left tab will have a pull-down at the top. Don't select the By Position entries, instead, select the entry with Windows 7 Loader ... in it.

Then SAVE the results.

If you just close the window, no changes will be made.

I can confirm this works as I did thy myself just yesterday.

### Post by oldfred on 2011-10-24
If your first entry is Backtrack you are booting with Backtrack. But are you using grub customizer in Ubuntu? That will not change Backtracks boot menu.

You either have to edit Backtracks menu or (preferred) install Ubuntu's grub2 boot loader to the MBR.

Boot into Ubuntu:
#reinstall from working (not liveCD) system - first find Ubuntu drive (example is drive sda but use your drive not partitions):
sudo fdisk -l
#if it's "/dev/sda"  then just run:
```
sudo grub-install /dev/sda
sudo update-grub
```Only required if more than one drive:
#to get grub to remember where to reinstall on updates:
sudo dpkg-reconfigure grub-pc
#Enter thru first pages,spacebar to choose/unchoose drive, enter to accept, do not choose partitions
#To see what drive grub2 uses see this line - grub-pc/install_devices:
sudo debconf-show grub-pc

### Post by Andrew_nuts on 2012-01-18
It can be done manually all you need is go to 
**/boot/grub/grub.cfg**.

open the this file with administrative login and change the menu list  priority.
The menu which you want on top put it in top in this file.
