### Post by cariboo on 2011-02-20
What are you using to create your usb drive? If it is an Ubuntu iso you can use System->Administration->Startup Disk Creator. If you don't have another system running Ubuntu, use [unetbootin]("http://unetbootin.sourceforge.net/"), it's available for Linux and Windows.
