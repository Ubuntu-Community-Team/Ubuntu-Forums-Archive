### Post by jazee on 2006-01-15
The Root User is your administration account user. This user has full permissions for your system were as you a normal user as limited permissions. You do not need to log into him to enter synaptic. You can enter synaptic my going to system -> Administration -> Synaptic. When it loads it will ask you for the root passsword. If did not change the password at all after install it will most likely be your password for the first account you created when you installed Ubuntu. Most likely the account your currently loged into as. Enter the password and your in to Synaptic.

On the internet problem can you search google or is it just bringing up the main page?

I had somewhat the same problem with my card were I wasn't viewing any webpages. So try this and see if it works. Goto System -> Administrator -> Network. It is going to ask for the root password. So enter the password I told you about above. When it loads up you should see your network devices. Click on the device that you are trying to connect to the internet on. If the Deactivate button lights up I have no idea whats wrong but if it doesn't the Activate Button should be lighten up. Click that. If it is not lighten up eather. Click the properties button. and check the enable this connection box and hit ok. Now the Activate button should light up. Hit it and you should be able to browes the internet if all else is fine.
