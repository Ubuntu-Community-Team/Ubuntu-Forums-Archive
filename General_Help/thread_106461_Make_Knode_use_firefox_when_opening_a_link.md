### Post by bigblop on 2005-12-20
When I click on a link in Knode Konqueror is used to open the link. How do I change this to Firefox instead?

---

### Post by One Quick Question on 2005-12-20
You can change the default browser in KControl.  Go to KDE Components > Component Chooser > Web Browser, and then choose "In the following browser" and type in firefox.  I suspect that should work!
