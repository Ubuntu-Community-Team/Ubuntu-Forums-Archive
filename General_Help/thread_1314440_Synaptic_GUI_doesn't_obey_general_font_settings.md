I'm not sure though if we're on the same page here (because I see that you've been using Ubuntu longer than I have), but I think you're talking about the difference in themes between "you" user and 'root' user (as shown by synaptic).

Here is a tutorial to make the "you" user and 'root' user themes identical: [http://tombuntu.com/index.php/2007/09/18/theme-applications-running-as-root-in-ubuntu/](http://tombuntu.com/index.php/2007/09/18/theme-applications-running-as-root-in-ubuntu/)
