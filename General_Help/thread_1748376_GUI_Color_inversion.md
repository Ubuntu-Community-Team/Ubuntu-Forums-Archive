### Post by edm1 on 2011-05-03
Lol, have you been messing round with compiz plugins? Seems like you have enabled the 'negative' compiz plugin and pressed Super+N (for desktop) or Super+M (for window). If you havent got this plugin enabled then???

To play around with compiz plugins install 'Advanced Desktop Effects Settings' from the software centre if you havent already.
