### Post by kidders on 2008-05-03
Hi there,

There are a few keyboards around that feature cosmetic changes to make them *look* like they're designed for Linux. There isn't really any such thing though ... at least not in the same way that there is for Sun or Mac systems, for example. Linux's focus has always been on allowing people to use any keyboard they choose, rather than laying down one "preferred" way of doing things.
