### Post by bfonseca on 2006-02-18
Hi, I had the same problem. It would freeze on me and then sometimes it would get past it but for some odd reason it would fail to install.  The problem I had was that I over clocked my cpu.  I am using a sda hard drive and when I took the over clock off it installed properly.

I had tried burning multiple distro cd's but ended up with about 15 cd's and none of them worked.  I went back to the original one and set the cpu to default and it worked just fine.  Since then i have been able to reinstall ubuntu with no problems.  Once it was istalled I over clocked it again and it worked fine. When I needed to reinstall ubuntu because I messed it up I played with the cpu clock.

I dont know why this was the issue with me, I am not sure why that would have anything to do with the install but it did or I am just getting lucky. But anyway I hope this helps.
