### Post by Lars Noodén on 2013-11-18
Hardy Heron was Ubuntu 8.04 from 2008.  It was a Long Term Support release so parts of it (for servers) were supported for five years, a period which ended earlier this year.  

I'm not sure about 'portable' Ubuntu but Ubuntu can be installed and run from a USB stick.
