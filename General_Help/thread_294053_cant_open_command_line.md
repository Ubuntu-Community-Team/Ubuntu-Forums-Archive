### Post by hikaricore on 2006-11-06
Are you using ubuntu edgy?  There were some posts by users with certain video cards having trouble with gnome-terminal closing instantly.  If you need terminal access you can hit ctrl+alt+F(1,2,3,4,5 or 6) to get to a login prompt and proceed from there.  To get back to your X session hit ctrl+alt+F7 at any time from the terminal.

My suggestion is if you're having trouble with gnome-terminal, install another terminal such as yakuake.

```
sudo apt-get install yakuake
```

when you run this you can hit F12 at anytime to pulldown a terminal at the top of your screen in quake style.  :)

Hopefully this helps.   And here's the trouble others were having [http://www.ubuntuforums.org/showthread.php?t=288624]("http://www.ubuntuforums.org/showthread.php?t=288624")
