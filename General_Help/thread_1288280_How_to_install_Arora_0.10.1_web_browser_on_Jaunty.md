### Post by antony.s on 2009-10-11
First try the PPA, [https://launchpad.net/~mapopa/+archive/arora-stable](https://launchpad.net/~mapopa/+archive/arora-stable)

Or installing the karmic version of the package: [http://packages.ubuntu.com/karmic/arora](http://packages.ubuntu.com/karmic/arora)

Worst case, install from source following [http://code.google.com/p/arora/wiki/source?tm=4](http://code.google.com/p/arora/wiki/source?tm=4) ?
