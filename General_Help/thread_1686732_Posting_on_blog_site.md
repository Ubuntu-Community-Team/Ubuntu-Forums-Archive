### Post by WorMzy on 2011-02-12
The only difference between using a website on Linux and using it on Windows, Mac, etc (that I know of) is a slight deviation in the user agent header that browsers send to the server. Unless a web developer deliberately sets up the page to reject connections from specific operating systems, the page should work exactly the same on any OS.

Unless the page uses ActiveX controls, in which case it's just silly and will probably only work right in Internet Explorer anyway.
