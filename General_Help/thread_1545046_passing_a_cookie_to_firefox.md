### Post by chopinhauer on 2010-08-03
> **jsstn said:**
> 
google searches and the firefox man page suggest that firefox doesn't have this functionallity, but would it in that case be possible to link the cookie into firefox's normal cookie jar?


If Firefox is not running, you can add the cookie to the cookies.sqlite database (with sqlite3 and an SQL query). However once Firefox starts the database is locked.

Have you considered writing an extension that does everything inside Firefox?
