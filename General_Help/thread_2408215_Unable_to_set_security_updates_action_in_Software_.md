### Post by poorguy on 2018-12-15
I would ***uninstall*** the*** Software and Updates*** and ***Software Updater ***and then ***reinstall*** them both using ***Synaptic Package Manager***.

Go to the ***Main Menu*** under ***System Tools*** find ***Software ***and open that then at the top find ***Installed*** and open that look for ***System Applications ***and then look for ***Software and Updates*** and ***Software Updater*** .
