### Post by piratebill on 2010-10-14
> **bac0926 said:**
> 
Bus 001 Device 007: ID 1058:1100 Western Digital Technologies, Inc. 

This is good.  The computer sees your drive, its just not mounting it (making it usable to you).  There are two things you can do here: mount it by hand (if you want to do things with it), or format it (which is what you said you intend to do).  Here is how I would format it.  Download gparted
```

sudo apt-get install gparted
```

Then under System->administration (I think) you find gparted listed under gparted, or partition editor.  Your external drive should show up there.  format to whatever you'd like and all should be well.  If you have any questions, please ask.

### Post by piratebill on 2010-10-14
```
sudo mount /dev/sdb1 /some_folder
```

sbd1 would be the second connected hardrive's first partition.  Change as needed (sdc2,sdc3,sdb2 etc).  some_folder will be the mount point.  By default ubuntu mounts everything in /media/some_folder.  When I mount by hand, I normally create a folder on my desktop and mount the drive there.

to unmount 
```
sudo umount /some_folder
```
