### Post by djf_jeff on 2006-12-07
Yeah, you are right, both install softwares. But Synaptic is aim at a more "advanced" user. It has more options, more packages, etc.

Add/Remove is more user-friendly. It has the more commons packages for the common user.


Both use apt as a backend and packages you install in one will also be installed in the other. You can use both and it will work fine.
