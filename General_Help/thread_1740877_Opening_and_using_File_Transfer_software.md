### Post by Dave_L on 2011-04-27
> **Donnie Love said:**
> Applications > Internet > FTP

1) System > Preferences > Main Menu

2) Under Applications, select Internet

3) Select FTP

4) Click the Properties button

5) In the Launcher Properties dialog, the values of the Name, Command and Comment boxes should help to identify the application.
