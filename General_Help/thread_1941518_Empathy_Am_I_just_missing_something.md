### Post by munkymac on 2012-03-15
Does Empathy play horribly with Gnome Shell? I've found it to be slow, often unresponsive, and nearly useless, save for the notifications (which worked on Pidgin until the latest Gnome Shell upgrade from Ricotz). Actually, I've used it in Unity, and i experience many of the same problems, and there it lacks the one selling point: notifications.

I'll give more details so this isn't just an "Empathy is the WORST" post (though I do feel that way):

1. I'm unable to arrange my various groups in the contact list the way I would like them. The ability to change group order seems to be non existent
2. When using the notifications in Gnome Shell, my responses are put at the bottom of the conversation, not in the order in which they were made, thus destroying all context if i need to go back and look at the logs
3. The window won't show up when I want it to.
4. The window won't close when I want it to.
5. Contact grouping no longer works with Facebook (though I believe this is a Facebook server-side problem)
6. Sometimes doubleclicking on a contact does not open a contact window. 
7. Searching a log or trying to send a new message via ctrl n forces me to choose a protocol before entering a user name. I don't have to do that with pidgin.
8. Username ordering within the list doesn't have an order by log size option. So the people I would most likely talk to aren't at the top of my list. This is annoying.

I've been able to use DeskSMS connected to my gmail account and had that work with Empathy, which has been great, but thus far that seems to be the only thing that has gone right for me.

In general, I seem to have no control over making Empathy work in my workflow, and currently I'm only using it until the pidgin notify extension for Gnome Shell gets updated. 

However, if there's some reason I should continue using Empathy, I'm all ears. I know it integrates better on the back end, but it definitely doesn't integrate with me. There don't seem to be plugins or extensions that can tweak my experience, either.
