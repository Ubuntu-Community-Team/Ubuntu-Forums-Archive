Hi Ubuntu-Users!

I just migrated from Debian Sarge to Ubuntu 5.04 and I do not regret this step at all! With one exception: when I try to use the print wizard in one of my favourite applications "digikam" I do get a print-out but with very strange colors. A test page with the same printer from CUPS web frontend instead looks perfect. On Debian-Sarge the plugin works perfectly.

The following related packages are installed on my box:

ii  digikam        0.7.2-2ubuntu1 digital photo management application for KDE
ii  digikamimagepl 0.7.2-2ubuntu1 plugins for digikams image editor
ii  digikamplugins 0.6.2-3        Digikam Plugins
ii  kipi-plugins   0.1+beta2-20ub image manipulation/handling plugins for KIPI
ii  libkipi0       0.1.1-2ubuntu1 library for apps that want to use kipi-plug

Any hint, idea or suggestion is really welcome ...
Even sharing the same bad experiences with somebody else might be nice ...
