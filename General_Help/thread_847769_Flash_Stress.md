### Post by cj100570 on 2008-07-02
Whenever I visit a page with heavy flash content I can here my cpu fan spin up. How is it that flash animation can cause enough load on a quad core pc with 2 gigs of memory that the cpu fan spins up? To me it just doesn't seem possible. I've run encoding programs on the pc that didn't even cause this behavior.

---

### Post by sdennie on 2008-07-02
This is primarily because Adobe Flash is closed source and the support is "just good enough".  Which means that it works but it is not in any way optimized.  You can expect any flash page to eat an entire CPU core and generally cause your fans to come on until Adobe puts some effort into making flash work better on linux.  It's unfortunate but, there isn't really anything that can be done (there is a open source implementation of flash called gnash but, it's still not quite usable).
