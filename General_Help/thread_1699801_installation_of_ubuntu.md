### Post by sikander3786 on 2011-03-04
What you need to do is to individually try both the current Ubuntu versions i.e, 10.04 and 10.10 from a Live CD and test all your hardware and performance. Then install the one that performs better.

10.04 is LTS i.e, will be supported for 3 years whereas 10.10 will be supported for 18 months.

For installation, due to some bugs, it is recommended to do manual partitioning while installing alongside Windows. Please go through this page.

[http://ubuntu4beginners.blogspot.com/2011/02/manual-advanced-partitioning-in-ubuntu.html](http://ubuntu4beginners.blogspot.com/2011/02/manual-advanced-partitioning-in-ubuntu.html)

Once you have an idea, you will definitely have queries and they will be easier to answer later :-)

### Post by frenziedfemale on 2011-03-04
The fabulous Ubuntu folk have made installing alongside an existing operating system so easy that even a Windows user can do it without much effort (haha). Check out the download ubuntu page and click 'Show me How' next to Install. there are some great screen-shots to give you an idea of how it works. You will need to put in some thought on how much diskspace you will require for each operation system. The install program walks you through it quite nicely! (no need to pre-make partions). As to which Ubuntu to install ...  you will need to know if you are going to run the 32 bit or 64 bit. I use Ubuntu 10.10 myself. I have an Acer Aspire one Netbook with AMD II neo processor.  Hope that helps!

### Post by YesWeCan on 2011-03-04
I would...
1) use 32 bit. There is no major performance advantage to 64 and not everything works with 64, such as Thunderbirds calendar and contacts organiser.
2) If your laptop supports it, go and buy a second HD and put Ubuntu on that. This will avoid future inconveniences due to conflicting boot systems. Furthermore I heard once that you may void your Dell support contract if you mess with the Windows installation - not sure, worth checking.
