### Post by 3Miro on 2011-05-19
"Environment" is not a very well defined term. 

Pretty much everything in Linux is compiled with gcc (GNU C compiler). This is the one you should use for Linux too.

Then comes the question of what program to use to edit the source code files (not an environment, just an editor). Eclipse is one option although I personally find it rather slow. In the past I have used Code Blocks and I definitely recommend that (it has many options and is rather fast). Currently I use Geany, it has overall fewer features, but it is very light and it is a real pleasure to work with.

I only recently heard about NetBeans, I don't know much about it, but I think it is also Java based like Eclipse.
