### Post by Devilotx on 2006-10-11
Dunno if anyone is interested, especially with Automatix and/or EasyUbuntu available to us, but I grabbed and tested Mandriva 2007 which comes with LinDVD for DVD viewing, and it seems pretty cool, I grabbed the RPM from the DVD and ran it through Alien and it installs on Dapper just fine, works pretty good too.

I remember being interested when Turbolinux offered this functionality, wondering if it would work on other distro's, so I figured I'd let people know.

so yes, running the LinDVD rpm through Alien will let it install and function in Dapper.
