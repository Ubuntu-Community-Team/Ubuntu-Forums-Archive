### Post by Nostoc on 2009-11-17
Easy peasy is basically ubuntu netbook remix, with some proprietary software preinstalled.

UNR offers a better screen real estate management than the regular Ubuntu releases, and a few tweak here and there. If you have a netbook, you should most likely be running NBR or easypeasy instead of plain old ubuntu, but it's up to you.

### Post by t0p on 2009-11-17
I've got an EeePC 701, on which I'm running Eeebuntu 3 - basically Jaunty with a custom kernel and some built-in scripts to make it compatible with all the hardware; if you run plain vanilla Jaunty, some devices won't work, eg the webcam.  Eeebuntu is available from [eeebuntu.org]("http://eeebuntu.org").

I don't know if vanilla Karmic works okay on an EeePC.  I'm planning to try it, but I haven't got round to it yet.  But it would be very easy for you to find out.  Make a Karmic live usb and boot the EeePC with it.  If all the onboard devices work, it works.  If any devices don't work, you can try Eeebuntu or Easy-peasy; or you could use Karmic and just do without the non-functioning devices.

I bought my EeePC with Xandros installed.  But that was some time ago.  I'm not sure, but I don't think Asus do EeePCs with Xandros anymore.  You'll have to check.  But I certainly do recommend running Linux on the little netbook.  I love my EeePC.
