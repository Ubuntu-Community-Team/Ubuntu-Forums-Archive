### Post by oldfred on 2018-04-27
Many want to see what differences are. 
If you have 25GB (maybe less if just a pure test install), you can install it to try it out. But keep your 16.04 as your main working install for the time being.

I have several obsolete installs, not yet overwritten and 16.04 is still my main working install.
But I have 17.10 & 18.04 in 25GB partitions. I think I still have my old main 14.04 install on one system, but have not booted it recently. I usually have another install of my main working version, so I can experiment with settings without destroying my main system.
