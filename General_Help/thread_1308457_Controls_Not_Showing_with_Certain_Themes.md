### Post by pablo180 on 2010-02-13
Fixed it, but as usual with this kind of thing I did so much trying to solve it, that I have no idea what finally did it. 

I uninstalled Aurora Engine 1.4.1 and then installed Aurora Engine 1.5.1 from [Gnome Look]("http://gnome-look.org/"), that seemed to make no difference. 

I thought that it may have something to do with the mouse, as moving the mouse over things was the only way to make things appear. I was using the Aurora Leopard BSM theme, so I clicked Customise and selected the DMZ Black pointer (which I am sure was what the theme used in 9.04?), that didn't work either. Although the mouse pointer still seemed to be the default one for everything except firefox. 

Next I tried switching Compiz from GConf to Flatfile, nothing, and so back again, nothing happened. Anyway so I switched off, turned back on again this morning and hey presto something worked!

Currently using the Aurora Leopard BSM theme, as intended and as it was on 9.04. I daren't try any of the others yet just in case it breaks, but when I do I'll post back. 

I hope this helps someone.
