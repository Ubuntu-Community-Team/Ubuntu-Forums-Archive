### Post by varunus on 2005-11-19
I'm slow at typing replies, so someone will probably beat me to it..

Due to patent issues, ubuntu won't play MP3s out of the box. That said, its very very easy to do this; since the MP3 license lets you use a codec for "personal use" free, you have to install the codec.

This post assumes you know about the "Ubuntu Way" of installing software.  A quick primer:  Instead of having to go to ppl's websites and get their software, ubuntu uses a centralized system of packages that can all install software with one click.  Places to get packages are called repositories.

Fire up your package installer at System->Administration->Synaptic.  The first thing you'll want to do is add some repositories.  Go to Settings->Repositories in Synaptic.  You'll see all of your current package sources there.  Hit Add, and check everything, including "Community Maintained" and "Non-free", and then hit ok.  This will add a repository.  You will then be asked to reload your package lists; go ahead and do this.

Once you do this, install the following packages in synaptic:
gstreamer0.8-mad <--Lets you play MP3s in rhythmbox
gstreamer0.8-faad <--Lets you play AAC files in rhythmbox

You may want to grab some other codecs while you're there.  Some codecs aren't available in the Ubuntu repositories due to their gray legal status; you can check out the Penguin Liberation Front's Ubuntu package repository (somewhere there's a link on this forum, search for it) to get every codec you could ever dream of (windows media, real audio, divx, xvid, all sorts of other video and audio things).

After you do this, you can now play mp3s in rhythmbox.

Also, your windows partitions should already show up on the Desktop (the installer found and put mine there.)  If not, try looking in the folder /media/ or in your Computer place.  If you don't see them there, post back here, and I'll show you how to add them manually.

Good luck.

### Post by varunus on 2005-11-19
To add your windows partitions (if they're NTFS)

First open up a terminal.  Type "sudo fdisk -l" (that's an L).  You'll get something like this:
```
Disk /dev/hda: 60.0 GB, 60011642880 bytes
255 heads, 63 sectors/track, 7296 cylinders
Units = cylinders of 16065 * 512 = 8225280 bytes

   Device Boot      Start         End      Blocks   Id  System
/dev/hda1   *           1        6566    52741363+   7  HPFS/NTFS
/dev/hda2            7201        7296      771120    5  Extended
/dev/hda3            6567        7200     5092605   83  Linux
/dev/hda5            7202        7296      763087+  82  Linux swap / Solaris
```

Next, open up your fstab file (sudo gedit /etc/fstab).
For each of your NTFS partitions, add a line like this (here's how mine looks):
```
/dev/hda1       /media/hda1     ntfs    defaults,user,ro,umask=0222,nls=utf8     0       0
```

Make sure its all on one line.  After you do this, make sure you have made empty folders corresponding to where you want it mounted (in this case /media/hda1 is my mountpoint, so I'd have to open a terminal and do "sudo mkdir /media/hda1").

Then, type "sudo mount -a" and your windows drives should appear on your desktop.

### Post by Bachstelze on 2005-11-19
Open a terminal, run

```
$ sudo -s
```

to have permanent root access. The first thing you need is to know which is your Windows partition. Run

```
$ fdisk -l
```

You'll see a list of all partitions existing. What interests us here is the name of your Windows partition, for the following, I'll just call it /dev/hdax, replace x with the corret number. Then you create the directory where you want to mount your windows partition, for example : 

```
$ mkdir /mnt/windows
```

Finally you have to tell your ubuntu system where to mount the partition on startup, those informations are stored in the /etc/fstab file, run :

```
$ gedit /etc/fstab
```

You'll see all your mounted partitions, add this line at the end of it :

```
/dev/hdax	/mnt/windows	ntfs	ro,user,auto,gid=100,nls=utf8,umask=002	0	0
```

Don't forget to change the x in the partition number to the one of your windows partition. Save the changes. Reboot your system. Voil&#224; :D
