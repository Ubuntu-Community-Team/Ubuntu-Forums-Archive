### Post by aaronmarsh632 on 2008-11-10
Hi, is there a way you can create your own custom commands. For example in the Terminal instead of typing 'apt-get install example' you could type 'install-app example' and so on??

Thanks

---

### Post by -Zeus- on 2008-11-10
this is how to do is: alias install-app='apt-get install'.  To get them to stay alive, you need to put them in the ~/.bash_aliases file.
