### Post by coffeecat on 2010-11-20
> **Wtower said:**
> Ethernet over mains sounds interesting, I never tried it.

There are several makes, all good I'm sure. The ones I use are Devolo which are reliable and have the bonus of including a Linux version of the configuration software. There's next to no configuration anyway. You can set up an encryption password (good idea) and monitor the transmission speed. The Linux config software is terminal only and has to be compiled but there are clear instructions for Ubuntu. Here's the Devolo site for Greece:

[http://www.devolo.gr/](http://www.devolo.gr/)

> **Wtower said:**
> I have tried different channels, which kind of works better for that moment, but it seems like after a while I get again the same results. If it is interference, then whatever is causing it is changing channels as well. I wonder, could it be another wifi router close to mine? I doubt that such a device could cause such an issue.

One thing you have to be aware of is that 802.11 channels overlap so, in effect, there are only three discrete channels: 1, 6 and 11. More here:

[http://en.wikipedia.org/wiki/802.11#Channels_and_international_compatibility](http://en.wikipedia.org/wiki/802.11#Channels_and_international_compatibility)
