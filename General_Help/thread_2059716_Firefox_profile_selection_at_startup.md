### Post by ChaosChris2009 on 2012-09-18
[LIST=1]
[*]Open Terminal
[*]Run this command ```
firefox --profilemanager
```
[*]The profile manager window will come up
[*]Create your new profile
[*]Uncheck the box "Don't ask on startup"
[/LIST]
Every time you run firefox you'll be prompted to choose which profile you want to use

Make sure to close/exit Firefox before following any of the steps
