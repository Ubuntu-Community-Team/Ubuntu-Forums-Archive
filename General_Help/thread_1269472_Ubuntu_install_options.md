First, I think you are a little bit confused, so before I answer any other questions that you have, what exactly do you mean by this statement?  You won't be running both these OS's at the same time. You will need to chose one or the other.  The main difference between these two are the fact that Ubuntu Desktop comes pre-packaged with a desktop manager, which the Ubuntu Server does not.  For your experience level, I would suggest installing the Desktop version on your computer, because the GUI will help you become more familiar with the OS (unless however the 'old computers' you have cannot handle the extra CPU/memory needs).  What are the specs on the computers you wish to run Ubuntu on?
