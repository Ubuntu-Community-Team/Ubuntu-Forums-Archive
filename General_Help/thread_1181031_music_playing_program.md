### Post by cariboo on 2009-06-07
Rhythmbox is installed by desfault, It is located at Applications-->Sound & Video-->Rhythmbox. You may need to install some codecs to be able to play audio/video files. I would suggest installing the ubuntu-restricted-extras meta package that is in the repositories. Go to System-->Administration-->Synaptic Package Manager and search for ubuntu-restricted-extras and mark it for installation. 

Personally I store music in the Music directory in my home directory.
