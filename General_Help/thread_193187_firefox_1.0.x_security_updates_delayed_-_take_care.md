### Post by nanotube on 2006-06-09
according to this security posting ([http://ubuntuforums.org/showthread.php?t=192903)](http://ubuntuforums.org/showthread.php?t=192903)), the security updates to the official breezy firefox (v1.0.8) will be delayed.
so i'd strongly recommend everyone to install the newest firefox from mozilla official release.
there is an automatic script that i have created with aysiu's help, available here:
[http://pykeylogger.sourceforge.net/wiki/index.php/Ubuntu:Chronicles#Install_Firefox_1.5](http://pykeylogger.sourceforge.net/wiki/index.php/Ubuntu:Chronicles#Install_Firefox_1.5)

or, you can just follow the manual update instructions, available here:
[https://wiki.ubuntu.com/FirefoxNewVersion](https://wiki.ubuntu.com/FirefoxNewVersion)

edit: and while you are at it, you might also want to install the newest thunderbird, which also has security issues addressed. there is an analogous automatic update script here: [http://pykeylogger.sourceforge.net/wiki/index.php/Ubuntu:Chronicles#Install_Thunderbird_1.5](http://pykeylogger.sourceforge.net/wiki/index.php/Ubuntu:Chronicles#Install_Thunderbird_1.5)
