### Post by neorou on 2007-06-11
The Endnote equivalent is still in development:
[http://bibliographic.openoffice.org/](http://bibliographic.openoffice.org/)

If you need to run Endnote, it might run version 6 for Windows, using Wine. Here is the information on Wine:
[http://winehq.org](http://winehq.org)

You can install Wine on Ubuntu using Synaptic Package Manager.

Good luck...

### Post by sybille on 2007-09-02
tjoel99,
Take a look at Zotero. Zotero is a bibliography management extention for Firefox that is kind of like EndNote, but with more features. Zotero has an extension that will let you insert your citations and make bibliographies in OpenOffice.org documents, too.
[http://www.zotero.org/](http://www.zotero.org/)
[http://www.zotero.org/videos/tour/zotero_tour.htm](http://www.zotero.org/videos/tour/zotero_tour.htm)

Zotero can import EndNote files. Instructions here:
[http://www.zotero.org/blog/making-the-switch-to-zotero/](http://www.zotero.org/blog/making-the-switch-to-zotero/)
