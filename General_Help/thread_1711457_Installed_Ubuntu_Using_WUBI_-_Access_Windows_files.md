### Post by bcbc on 2011-03-21
See the [wubi guide]("https://wiki.ubuntu.com/WubiGuide#How do I access the Windows drives?") - they're under /host

You can ALT-F2, nautilus /host 
or navigate through the Places, Computer, File system, to get to /host
