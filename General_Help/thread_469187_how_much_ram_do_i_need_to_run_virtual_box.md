### Post by orb9220 on 2007-06-09
You need at least 1gb I believe. As ubuntu will use 512 and box uses the other 512mb.

When you are running another OS you must have set aside what requirements the orig OS uses to run.
 
If XP which really needs 512mb to run well then that is what virtual box will need to run it.

So I would say in your case you need to max out ram. And you may still run into slowdowns depending on cpu, and video requirements.
