### Post by frotzed on 2011-02-27
I have a somewhat basic question regarding Ubuntu upgrades.  If a person is using 10.04, and they intend to use it for the entire 3 year life cycle, I assume that when the next LTS is released (presumably 13.04) the user will be able to upgrade directly from 10.04 to 13.04?  Or will upgrading involve a complete reinstall?

---

### Post by snowpine on 2011-02-27
The upgrade process is described in detail here:

[https://help.ubuntu.com/community/UpgradeNotes](https://help.ubuntu.com/community/UpgradeNotes)

The next LTS release will *probably* be April 2012, and upgrading from 10.04 to 12.04 will *probably* be similar to the 8.04 LTS to 10.04 LTS upgrade, [described here]("https://help.ubuntu.com/community/LucidUpgrades#Upgrade from 8.04 LTS to 10.04 LTS").
