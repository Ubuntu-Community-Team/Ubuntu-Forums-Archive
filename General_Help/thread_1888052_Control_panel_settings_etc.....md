### Post by SteveDee on 2011-11-29
> **rmcellig said:**
> I just installed Lubuntu 11.10 and was wondering how I can access  my sound settings and other settings I would see when I was using Ubuntu 10.10...?

Skinny little Lubuntu does not carry all the features of the Fat Gnome. The basic way to access sound settings is to type in a terminal:-
```

alsamixer

```
...or you can go to Synaptic and install one of the GUI sound mixer apps.

There is no menu editor for Lubuntu, so menus are generally either left alone or edited manually. A search in this forum should reveal how this is done.
