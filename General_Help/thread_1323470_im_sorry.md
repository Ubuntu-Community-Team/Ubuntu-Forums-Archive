### Post by Monsuco on 2009-11-11
You can't get free copies of XP (unless you work for Microsoft or have an MSDN subscription that is). You can sometimes find discounted legit versions on eBay or at local computer stores. Unlike Vista, 7 seems to be attracting converts. XP is in long-term support (meaning it will get security updates, but MS no longer makes sure it will work with new stuff) though so if you end up buying something take that into consideration.

If your PC came with Windows XP on it you should check to see if you have a recovery disk. If yours just had a recovery partition, you might have wiped it out when installing, in which case you can probably contact your hardware vendor to order a recovery disk.

If Kubuntu isn't working for you, you could try Ubuntu. Gnome lacks KDE's bells and whistles but Ubuntu seems to receive a tad more refinement and attention from developers than Kubuntu and seems like a better choice if you are new.

Also if Linux is just taking a lot of getting used to, try multi-booting. Just set up Windows on one partition, Linux on another (and a swap for Linux of course). The installer can resize NTFS and set it up so you get a menu when you start your PC saying to pick an OS. This is good if some of your programs or games require Windows. You can even get Windows to access your Linux partition (there are free Ext2/3 Windows drivers somewhere, but I don't think ext4 works yet and Vista/7 have occasional issues, XP is fine) and your Windows partition should automatically be accessible from Linux's partition. I still keep a partition with Vista on it for games and MS Office (OOo is good, but I have a class that needs Word 2007).
