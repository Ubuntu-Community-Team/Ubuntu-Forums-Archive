### Post by localzuk on 2006-04-01
The disc should have been a .iso file. This is a bootable cd-image that needs to be recorded as such (eg 'burn cd-image to disc' command).

In order to boot from it, you simply need to restart your computer with the disc in the drive and it should (if your CD-ROM drive is set up to be boot first) boot the installer.

You do not need to load any file by hand.
