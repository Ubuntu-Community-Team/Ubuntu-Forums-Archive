### Post by sedawk on 2009-07-15
For simple stuff on slideware level you can use
OpenOffice (impress).

A few months back I used 
[http://www.yworks.com/en/products_yed_about.html](http://www.yworks.com/en/products_yed_about.html)
which is very nice because you can feed it with
a graph (e.g. some xml-format) and then ask it
to do different layouts.
With a script to create the xml input file and
some layout magic of the graph editor I could
create much acclaimed results in nearly no time.

I have been looking for open source flowcharting programs
in the past a few times without much success.
It look like drawing flowcharts is something only done
or needed in commercial software development processes ;-)
But if you like, check out
kivio
dia
oodraw (part of openoffice, interface similar to impress)
