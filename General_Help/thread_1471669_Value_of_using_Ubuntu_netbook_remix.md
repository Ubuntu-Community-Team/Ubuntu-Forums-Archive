### Post by SallyK on 2010-05-03
If you are used to the full version, you may not find these advantages, but they can be very useful.

It's designed to maximise the useful area of the screen - just one panel at the top that is also the title bar of the active window. It's also designed to make it easy to load the apps you want most of the time - look at some screenshots and you'll see the difference.

I believe it's also optimised to run on less powerful hardware - it'll handle 512Mb of memory much better than the full version.

### Post by mje1708 on 2010-05-03
I have UNR 10.04 on my netbook (ASus EEE PC900) and standard 10.04 on my desktop. I don't know if there are any major technical differences between the two, but UNR seems to me to suit the netbook much better. Main advantage is how it displays on the screen - each application maximises automatically and the way in which you select menu options seems to suit a small screen netbook better.

However best option is to try it yourself on your netbook and see if you agree. It can be run from a USB stick and so fairly easy to try out. Some of this may be down to personal taste.
