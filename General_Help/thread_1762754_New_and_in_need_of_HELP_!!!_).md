### Post by jespdj on 2011-05-19
WINE is not a 100% perfect replacement for Windows. Remember that Windows is a proprietary operating system, Microsoft does not show the source code of Windows to anybody. Therefore it is very hard to make a software package such as WINE that will emulate Windows on your Linux system.

So, you shouldn't expect all Windows programs to run flawlessly on WINE.
