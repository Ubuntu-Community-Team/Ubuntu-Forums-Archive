You do not need any extra software t download and manipulate the pictures.  The camera will be detected and an app called f-spot will handle the download.  You can modify the files with GIMP.

The Canon software for Windows is not needed on a Windows host either.  The OS has its own detection and download app built in, but I bet you knew that already.  :D

Edit: I have a Canon A520 and I use both Linux and Windows to download  it.
