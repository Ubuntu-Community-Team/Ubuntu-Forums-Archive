### Post by Catlike on 2012-06-13
Before updating to Oneiric Ocelot, when I bookmarked a web page in Firefox, a dialog window opened and I got to choose which folder to put the bookmark into. 

Ever since updating to Oneiric, when I bookmark a page in Firefox, it simply plunks the bookmark at the end of the unsorted bookmarks; no dialog box opens. 
I don't think this is a Firefox change, because I see no conversations about this on the Firefox forums page. And I don't see a way within Firefox to modify this behavior.

Any ideas how to fix this? 

Thank you.

### Post by Seb71 on 2012-06-13
How do you bookmark the page?

If you click on the star in the address bar, then the result should be as described (add the bookmark to Unsorted Bookmarks).

If you bookmark the page with CTRL+D or from Bookmarks menu (Bookmark this page), then the dialog window with the Bookmarks folders should appear.
