### Post by t4thfavor on 2009-07-10
WINE is a reimplementation of the entire (sorta) windows API, and common libraries. It should work if it (a) does not use .net (b) not use directx > 9.0c (even that may give you trouble.


So in short, it will help you play *some* windows games, but not all of them can be run due to some essintial (to your game) part of windows not being present inside of WINE.
