AFAIK, [MSGTAG]("http://www.msgtag.com/home/") is Windows-only software. My first inclination, if I were to use it, would be in conjunction with email software that runs using [WINE]("https://help.ubuntu.com/community/Wine").

I stopped using MSGTAG on Windows about 4 or 5 years ago when I realized that it wasn't doing any good - people had wised up and weren't displaying images in their emails. I think the MSGTAG people have looked at changing the way it works.
