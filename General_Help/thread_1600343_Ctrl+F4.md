### Post by ghostborg on 2010-10-18
I'm using an Apple keyboard and I have to Press Cntrl+Fn(Function)+F4 and it works in Chromium and OO.
Possibly you have to do the same since F4 can be a dual function key. Also look to see if you have a function lock which might need to be toggled.

Under the menu System -> Preferences -> Keyboard Shortcuts for assignments.
