When an application crashes unexpectedly, one way to figure out what went wrong is to launch the application from the terminal instead of clicking on the icon. So just type "firefox" into the terminal, minimize it, and do whatever it is that you normally are doing that causes it to crash. After it's crashed see if there are any error messages.
