### Post by ctsdownloads on 2007-09-20
Found a better way:

> Create a text file. Paste the following:

#! /bin/sh -f

sudo -S rmmod psmouse

Save it as disable-tp.sh then right click on it and go to properties, select permisions and check off allow to be executed. Now do the exact same thing with this:

#! /bin/sh -f

sudo -S modprobe psmouse

…and name it enable-tp.sh .

Tired of using the touchpad and buttons on your notebook? Double click disable-tp.sh and choose run in terminal. Need that functionality back, run enable-tp.sh the same way. This allows you to use a USB mouse, even plugged in while executing these commands, without stumbling over that pesky touchpad.
