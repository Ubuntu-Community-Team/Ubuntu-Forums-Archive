### Post by gargo2000 on 2007-07-04
Took a bit to find, but here is how I found it:

Synaptic Package Manager 
Search "Splash"
Installed "gnome-splashscreen-manager 0.2-1ubuntu1"

Now you can find it in:
System --> Preferences --> Splash Screen
