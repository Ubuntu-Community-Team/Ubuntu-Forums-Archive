### Post by Fully216 on 2006-10-12
I know that word can publish a document also as a basic hmtl document.  It will not necessarily have all the bells and whistles that you can possibly create with some other html editor, but for a how-to it will suit the job just fine.

After you have created the document as you like it, instead of saving it as a word document, save it as an html.  The program will convert all the formatting, tables, pictures, etc into html code automatically and transparently, so the result should be a semi-polished version of the how-to.
