### Post by deepee111 on 2008-01-31
Gimpshop only modifies the menus, you still get the single document interface. Many will argue it's merits, but it drives me nuts. My workaround has been to use devilspie to launch Gimp in it's own workspace. That way, when you alt-tab or use multiple apps, your toolbars don't get buried in the mix.

### Post by rax_m on 2008-01-31
Another option I use is to set my main gimp window (the tools) to be always on top. You can set this in the Gimp preferences. Then make it really thin and long to resemble a built in tool bar. It hardly comes in the way. Esp if you use double-click to roll up the window.

EDIT: And I also use it on it's own workspace. Then the multiple windows don't bug me at all.
