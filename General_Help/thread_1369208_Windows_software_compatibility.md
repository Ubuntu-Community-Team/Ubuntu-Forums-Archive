### Post by TetonsGulf on 2009-12-31
There are alternatives to like it, lump it, or Wine though...

In some cases it means finding and using an open source program or combination of programs that perform the same functions. In other cases a virtual drive/OS is the way to go.

For what doesn't work in Wine the way I want, I use VirtualBox with a copy of Vista. It let's me accomplish what I want with specific Windows programs where I don't care for linux alternatives, like iTunes, Dreamweaver, Flash Professional, etc. This set up, as opposed to dual-booting allows me to work faster since I'm not shutting down and restarting and I almost never have to leave linux.

Try this as a starting point: 

[http://www.linuxalt.com/](http://www.linuxalt.com/)

If you don't see what you're looking for listed, try Google...

Hope that helps.

### Post by blueridgedog on 2009-12-31
> **jrysatx said:**
> Is there a list of software compatibility. Will programs like NetObjects Fusion install and work on Ubuntu? Will other programs that are Windows based work with Ubuntu?

There are a few ways to make things work:

1.  Attempt to run the application under wine: [http://www.winehq.org/](http://www.winehq.org/)  Check their database of applications.

2.  Ask the vendor for a Linux version.  Many of the windows programs I use now come with a Linux version.  It never hurts to add your voice to those asking.

3.  Find an alternative that performs the same function.  I like this site:  [http://linuxappfinder.com/](http://linuxappfinder.com/)
