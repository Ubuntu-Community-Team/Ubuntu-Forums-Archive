### Post by dummey on 2011-07-29
I realize that this is a fairly dated thread, but it's the top link that came up when I searched for this problem myself. 

The issue with empathy in 10.10 and 11.04 is that it comes bundled with an old version of empathy. The version in the repo is 2.34. According to [https://bugs.launchpad.net/empathy/+bug/332348/comments/17](https://bugs.launchpad.net/empathy/+bug/332348/comments/17) , the bug with the mobile icons not showing has been fixed in 3.0.

Would be great if somebody updated the repo and the ubuntu page ([https://help.ubuntu.com/community/Empathy](https://help.ubuntu.com/community/Empathy)).
