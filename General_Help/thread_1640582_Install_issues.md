### Post by sikander3786 on 2010-12-08
Welcome to the forums :-)

I suspect you are trying to install Ubuntu inside Vista using the Wubi installer. Right?

Wubi is intended for testing only. If you plan a long-term relation with Ubuntu and want to use it as a production OS, I would recommend a proper dual boot setup.

[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

You can also go with Wubi though, choice is yours but Wubi is a bit slower and a bit less stable than a proper install.

Regarding your ISO problem, did you check the image's integrity and make sure it is not corrupted.

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)
