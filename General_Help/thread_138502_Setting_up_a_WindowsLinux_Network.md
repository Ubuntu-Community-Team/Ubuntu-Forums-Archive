### Post by cronholio on 2006-03-02
You can connect to a Windows PC just by clicking Places->Connect to Server->Windows Share.

To make it work the other way around, go to Administration->Shared Folders. You will be prompted to install either Samba or NFS. Install Samba which enables Windows PCs to access your Linux machine.
