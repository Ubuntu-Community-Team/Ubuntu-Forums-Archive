### Post by logos34 on 2009-04-13
SOLVED !

[http://www.zen.org/2008/10/06/numeric-keypad-just-doesnt-work-under-ubuntu/](http://www.zen.org/2008/10/06/numeric-keypad-just-doesnt-work-under-ubuntu/)

system>prefs>keyboard>mouse keys>UNCHECK 'allow to control the pointer using keyboard'

I wasn't anywhere NEAR that config today (or recently).  How in the world did it get changed??

I'm usually so careful, and as such do not respond well the the unexpected
