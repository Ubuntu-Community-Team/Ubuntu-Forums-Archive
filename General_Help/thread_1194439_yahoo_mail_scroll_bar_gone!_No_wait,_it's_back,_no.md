### Post by mbeach on 2009-06-24
The incompatibility has to do with Yahoo providing a web application which they've not gotton to work properly with some browsers. It has nothing to do with interactions between Firefox and Ubuntu.

Try developing a "complex" web page and only test it on Firefox. Then go load it up on IE and then decide if IE is incompatible with Windows or if your application needs to be changed to work on multiple browsers.

Firefox is compatible with Ubuntu. The gmail interface works great in Firefox as do most websites I visit. I'm sure over time, Yahoo will fix the problems, and have things appearing the same in both IE and Firefox/Safari/Opera etc. In the meantime, LowSky's suggestion is available.
