Thanks alot.

### Post by JKyleOKC on 2007-10-21
I'm using Xubuntu, so this may or may not be useful. I popped open the Applications Menu, and under its "Settings" choice, clicked "Login Window." On its "Security" tab, I added my own user name to the "Automatic login" edit box, and closed the window. Now I no longer have to enter either my user name nor my password when booting; it comes up as me with no special actions.

If I switch users, then I still do have to enter the password, but at least it eliminates the need on the initial boot-up...
