### Post by DarkMageZ on 2007-10-20
amarok supports libvisual
rhythmbox & totem support libvisual via gstreamer (but it's sucky)

if you install the libvisual-plugins package and use amarok there are some okish visualizations.

there's an older version of g-force available for non-commercial use under libvisual if you are running x86 arch on feisty or gutsy.
