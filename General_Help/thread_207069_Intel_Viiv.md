### Post by gr83 on 2006-08-02
Viiv is just an Intel platform gimmick.  In order for a system be Viiv it has to be dual core CPU with X amount of cache and Intel chipset.  I believe the lowest end Pentium D 930 dual core doesn't quailfy to be Viiv.

It's much like Centrino, which all it really is an Intel Pentium M, Intel wireless card and Intel chipset.  If you don't use an Intel wifi card, such as an atheros or prism card, you no longer have Centrino platform.
