### Post by blazemore on 2008-12-19
Hit alt+F2, type in
```
sudo apt-get install ubuntu-restricted-extras -y
```

TICK THE "RUN IN TERMINAL" BOX

And then click Run (or whatever)

Type your password, and follow the onscreen instructions.

(You press Tab to access the Accept buttons for EULAs)
