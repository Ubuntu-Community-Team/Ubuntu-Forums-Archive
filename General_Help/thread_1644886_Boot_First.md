sda1/**Wubi**: 

That is a Wubi install I suspect there is a boot file in Windows that could be modified so it defaults to Ubuntu. Really if you like Ubuntu you can move that Wubi to a partition or just do a dual boot by using a Live Ubuntu Cd to install.

Wubi to partition: [http://ubuntuforums.org/showthread.php?t=1519354](http://ubuntuforums.org/showthread.php?t=1519354)

It is easier to to change the boot order in the Ubuntu setup, which isn't in control nor can be in a Wubi. Wubi has some other problems as well, no big deal most of the time, but it is designed for trying out Ubuntu not longterm use, though some do.

### Post by bcbc on 2010-12-14
For vista, you need to go to the startup & recovery dialog box (Advanced tab in the System properties box) and change the default OS. Don't change the timeout to zero or you won't be able to boot Vista anymore.

PS. Don't update packages grub-pc and grub-common for a wubi install, if the Update Manager prompts you. They are known to break wubi at the moment.
