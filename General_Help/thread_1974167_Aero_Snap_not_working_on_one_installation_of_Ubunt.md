### Post by wilee-nilee on 2012-05-13
I set up the cube in unity, so I am not sure if that is on the unity desktop, and if this desktop is what you are talking about. Gnome 3 has this option.

If unity I would try to reset the desktop, you do this with alt-f2 then type in unity --reset

If you have tweaked compiz in the unity desktop you can lose this function.
