### Post by montenik on 2013-03-04
A tool you might want to use is called "powertop" (it can be installed from the Ubuntu Software Center). The description it gives is:
> "PowerTOP is a Linux tool that finds the software component(s) that make your laptop use more power than necessary while it is idle."
You may also want to watch the video at: [http://www.youtube.com/watch?v=XloTXFgNL7s&feature=youtube_gdata](http://www.youtube.com/watch?v=XloTXFgNL7s&feature=youtube_gdata)
The episode is called "Hak5 1225.1, Linux laptop battery optimization and using encryption with dropbox"
