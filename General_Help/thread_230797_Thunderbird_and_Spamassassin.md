### Post by MarinaraOfDeath on 2006-08-06
Well, my uni set up spamassassin on all incoming accounts (without telling us -- bad IT people. BAD!). It started stamping my email, which when you're running a large student organization and thus forwarding/replying to something 5-10 times an hour can be aggravating: too many false positives. I had to harrass them twice to get it turned off, since it wasn't working for me.

That being said, they both use Bayesian statistics to analyze spam, so it can be redundant to use them together. Nevertheless, it can be done, and probably pretty successfully. Though honestly, I wonder if my impression that spamassassin is more a server-side tool than a standard user tool is at all correct.
