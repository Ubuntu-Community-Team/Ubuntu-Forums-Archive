Is there a way to disable games? Don't want the students I am setting up this server for to be playing games during class. I understand that you can play them still online and all, but can you disable them?

### Post by gsiliceo on 2007-10-12
Of course, you can uninstall them, 

In a terminal
> 
sudo aptitude remove gnome-games gnome-games-data

Or you can disable them from the menu, using the menu editor.
>
