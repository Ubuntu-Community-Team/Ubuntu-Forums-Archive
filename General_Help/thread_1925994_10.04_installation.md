### Post by Leppie on 2012-02-15
Could you please check the images you downloaded with md5sum?
You can get the checksum file here: [http://releases.ubuntu.com/10.04/](http://releases.ubuntu.com/10.04/)
Place the checksum file in the same directory as the images then issue the following command:
```
md5sum -c MD5SUMS
```
That way you'll know if the image you downloaded is corrupt or not.
