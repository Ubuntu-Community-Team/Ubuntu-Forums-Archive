### Post by schidlo on 2011-01-19
I had similar problem with my usb flashdisk. It became read-only. I repaired it in windows. Go to (I have czech version so name can be different) My Computer, right click on the partition and choose properties. Then on tab "Tools" there is Check for mistakes (or errors). I hope this will help you.
