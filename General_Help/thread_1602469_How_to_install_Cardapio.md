### Post by Refalm on 2010-10-21
Open the terminal, and do this:
```
sudo add-apt-repository ppa:cardapio-team/unstable
sudo apt-get update
sudo apt-get install cardapio
```

Then just select it from the "Add to panel" menu (and remove the original menu I guess).
