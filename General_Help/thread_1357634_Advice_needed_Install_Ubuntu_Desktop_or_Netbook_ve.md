### Post by elias4444 on 2009-12-17
I have an old, dying Sony Vaio that my wife and son use mainly to browse the web (and play Flash and Java web games), and to write documents in OpenOffice. It's had Vista on it for a while, but it can no longer sustain the heaving rhinoceros riding on its back. SO, I'm putting Linux on it for them.

Ubuntu Desktop and Linux Mint both run great on the system (it's like it has new life), but I'm interested in finding something easier for them to navigate and figure out (since it's all new to them).

Would installing the Ubuntu Netbook Remix be a good choice? Or do I lose too much functionality/hardware-compatibility with that?

Anyone have a similar experience with this? Advice?

### Post by snowpine on 2009-12-17
Functionality and hardware compatibility are identical. Netbook Remix has a different user interface (the "launcher") which users either love or hate. I don't care for it personally (I like a traditional Gnome desktop) but it might be a good choice for your family.

FYI it has slightly higher system requirements than "regular" Ubuntu and is a little slower in my experience.

---

### Post by Darael on 2009-12-17
Well, UNR should have identical hardware compatibility to Ubuntu. As for functionality, it shouldn't really be a problem, especially if it's only used for web and office. If you've got Ubuntu on it at the moment, you can give UNR a go by installing the "ubuntu-netbook-remix" package and then just removing it if it doesn't work for your purposes.
