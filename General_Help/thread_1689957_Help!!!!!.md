### Post by dontbugmee on 2011-02-17
1) first see if you have compiz effects enabled, if yes disable them, that should speed things up, 
go to system additional drivers, and see if there are any available for video, if there are, install them, if not, try to find xorg.conf tweaks for your video card

2) go to skype settings/ sound, specify your mic there, if you don't know which one it is try them all, you might need to restart skype every time you change it,
