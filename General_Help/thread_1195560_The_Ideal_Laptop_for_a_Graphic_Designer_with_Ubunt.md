### Post by gamblor01 on 2009-06-23
Make sure that you get a laptop with an nvidia graphics card!!  nvidia cards certainly have the best support under Linux.

Ubuntu should run fine on all of the modern hardware (Athlon/Core 2/i7 CPUs).  I don't think you're going to have a problem with laptops from Dell, Lenovo, HP, etc.  I would suggest that you go out and find some laptops with specifications that fit your budget as well as your desires.  Once you have narrowed it down a few different laptops then you can start trying some google searches to see if anyone has any problems running Ubuntu on them.  I seriously doubt you will find any issues.

As a side note, I would highly recommend checking out Bibble if you are a photographer shooting images in raw format.  I haven't tried it (I use Aperture 2.0 on my Mac) but it looks like it's the best thing available for Linux users at the moment.  :/
