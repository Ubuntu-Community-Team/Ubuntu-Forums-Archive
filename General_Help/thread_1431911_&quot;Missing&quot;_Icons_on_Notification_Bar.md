### Post by djchandler on 2010-03-20
Try going to System>Preferences>Appearance, choose customize and change which set of Icons you are using. Try the Gnome default set. See if the right icons display. If they do, then try switching back.

You may need to re-install the icon set (from Synaptic or with apt-get) that is misbehaving. It seems the system can't can't find the correct icon to display, so the set may have become corrupted.
