### Post by furtom on 2015-11-25
I'm sure howefield is correct. "Pin tab" is a feature in Chrome/Chromium (and I believe FF also has it). It allows you to designate a tab as an "app" (think gmail, etc). If a tab is pinned, it will resume upon restart, even if you don't have Chrome set to display tabs that were open last time.

The UI for a pinned tab is narrower.
