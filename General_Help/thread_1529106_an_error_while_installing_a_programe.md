### Post by chessnerd on 2010-07-11
This "error" is a security warning, similar to the ones that come up in Windows and Mac OS X, when trying to install a downloaded program.

Ubuntu is asking you for your permission to continue because it doesn't know whether or not you want to authorize Google Chrome to install itself. This is a good security feature that can prevent truly malicious programs from running on your computer.

libnss3-1d is a set of Network Security Service libraries that is required by a number of programs, including Google Chrome.

If you downloaded Google Chrome from Google's website ([www.google.com/chrome](www.google.com/chrome)) you should be fine and you can let it proceed.
