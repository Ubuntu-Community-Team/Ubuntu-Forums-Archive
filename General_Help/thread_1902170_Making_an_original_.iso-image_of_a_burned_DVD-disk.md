### Post by zero2xiii on 2011-12-30
Hay,

Yes (and I feel like stopping my answer here becuase I answered your question. But hay, I'm a nice guy :))

Under ubuntu you can use brasero, disc copy, and then just select an file output instead of the same drive containing the source disc.

Under kubuntu you can use k3b, hit copy medium, then on the new oped dialouge, select the option "Only create image" and then under the image tab select the path and filename for the image.
