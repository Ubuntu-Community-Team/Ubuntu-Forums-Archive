### Post by y-lee on 2008-08-11
```
sudo apt-get install p7zip
```

Type the above command in a terminal should install 7-zip for you. Note Linux doesn't yet provide a GUI that graphically exposes all of 7zip's features. This means that you'll have to type some 7z commands into a command prompt.

Hope that helps and for more help just ask.

At present, the 7zip developers offer the Linux port of 7zip as a command line tool only.  Luckily, other archive managers like File-roller and Xarchiver (which are just GUI front-ends to various archivers) are able to use 7zip, that's why it worked with Archive Manager (File-roller).
