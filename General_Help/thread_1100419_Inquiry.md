### Post by aeiah on 2009-03-19
a few music players you can install have support for ipods. i believe the default one does too. i think some people have problems with the ipod touch though.

there are several bittorrent programs you can use. i use deluge which is native to linux. transmission is another one that some people like. utorrent is also pretty popular, and runs well under wine, which is a program that can be used to run windows software (although it doesnt run all stuff well and a lot doesnt run at all. utorrent runs fine though).

there are several burning programs for ubuntu including brasero and k3b. if you make use of nero's other features like encoding avis to dvd format, shrinking dvds or converting dvds to avi, then you'll have to use other software but there are solutions available for linux.

before installing you should check to see what problems you might encounter with the installation. some printers and some wireless cards dont work out of the box with ubuntu and some printers dont have any drivers at all for linux.

### Post by lovinglinux on 2009-03-19
Your questions have been already answered, but I would like to add a thing.

Do you know you can try Ubuntu without installing it? Yes you can. Download the latest desktop release, burn it into a CD and boot from it. You will be prompted to install or run it from live CD, among other maintenance options.

Select the Live CD option and the OS will be launched. You can do pretty much everything using the Live CD, without affecting your current OS installation, including installing torrent applications ( +1 for Deluge here ), browsing the web etc. Just be aware that you will loose some things (additional software installation and configuration for example) after turning it off. See link further below to know how to make these changes persistent between Live CD sessions. Anyways, if you need to save something while using the Live CD, you can browse your hard drives with it too.

More info [https://help.ubuntu.com/community/LiveCD](https://help.ubuntu.com/community/LiveCD)
