### Post by taurus on 2009-01-19
You have a few options.

1.  Boot into windows and shut it down cleanly.

2.  Use ntfsfix to fix that error message.

```
sudo apt-get update
sudo apt-get install ntfsprogs
sudo ntfsfix /dev/sda1
```

3.  Or mount it with the force option.

```
sudo mount -t ntfs-3g /dev/sda1 /media/sda1 -o force
```
