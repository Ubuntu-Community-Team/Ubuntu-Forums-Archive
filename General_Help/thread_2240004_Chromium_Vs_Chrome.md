### Post by grahammechanical on 2014-08-17
I recently installed Chrome as a test. I found that I needed to run a command in the terminal to load Chrome. And even though I had installed it using the software centre I could not use the software centre to remove it but only re-install. I had to remove it through the terminal. I am staying with Chromium.
