### Post by westie457 on 2014-01-16
If you have a gmail/google+ account it is quite easy to set up sync to move bookmarks etc from the old to the new. Sync must be set up first.

Another way is following this link. [https://support.google.com/chrome/answer/96816?hl=en](https://support.google.com/chrome/answer/96816?hl=en)

### Post by ajgreeny on 2014-01-16
You could also simply copy the hidden **/home/*username*/.config/chromium** folder from old to new machine and you would have everything including bookmarks, extensions, etc etc.
