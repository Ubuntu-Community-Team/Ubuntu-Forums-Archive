Firstly, I can't see much point in changing from 64 bit to 32 bit - the vast majority of people who can run 64 bit are moving to it.

Secondly, all you have to do is install the ubuntu-desktop package to convert from Xubuntu to Ubuntu (and stay at 64 bit), but if you do want to install the 32 bit version of Ubuntu and keep your files then you need to set up a separate /home partition. A forum search will provide detailed instructions on this.
