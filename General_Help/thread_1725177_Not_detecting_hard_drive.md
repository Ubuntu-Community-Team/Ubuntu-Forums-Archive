About 3 hours ago, I used this command to wipe out a GPT partition tabled hard drive with an MBR one using gparted.  I knew I had this task, and I wanted to make double sure this works under normal conditions.

> I booted the computer with the windows CD and it said that the hard drive cannot be found.

I don't know if the controller or the drive is acting up.

One way to try to tell is to boot Ubuntu (from the LiveCD/LiveUSB), run the Disk Utility (menu to System >> Administration >> Disk Utility) and get the SMART status of the drive.  There is also a disk drive self test option.
