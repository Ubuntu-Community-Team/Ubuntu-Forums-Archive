### Post by Ziggy72 on 2008-04-23
I'm using Hardy and have been trying to do a simple nautilus copy/paste in my home folder. Copying seems to work ok, but when I try to paste I get the message "Error while copying. Operation not supported". Copying using the command line works without a problem so presumably it's a nautilus issue. Thoughts appreciated...

---

### Post by Ziggy72 on 2008-04-24
The problem is solved! I had DDM (Desktop Data Manager) installed. When DDM is running, it seems to upset some functions of nautilus:(.
