### Post by divergex on 2012-02-21
This worked for me:

```

sudo gedit /etc/lib/plymouth/themes/default.grub

```

Change this:

```

if background_color 44,0,30; then
  clear
fi

```to this:

```

if background_color 0,0,0; then
  clear
fi

```Next do:

```

sudo update-grub

```Note that the above edits only affect Grub. You also need to make changes to plymouth, which shows the logo and progress dots. To get the plymouth background to be black as well, do the following:

```

sudo gedit /lib/plymouth/themes/ubuntu-logo/ubuntu-logo.script

```Change this:

```

Window.SetBackgroundTopColor (0.17, 0.00, 0.12);     # Nice colour on top of the screen fading to
Window.SetBackgroundBottomColor (0.17, 0.00, 0.12);  # an equally nice colour on the bottom

```to this:

```

Window.SetBackgroundTopColor (0.0, 0.00, 0.0);     # Nice colour on top of the screen fading to
Window.SetBackgroundBottomColor (0.0, 0.00, 0.0);  # an equally nice colour on the bottom

```Finally, reboot and you should have a black background throughout the boot process.
