### Post by -kg- on 2009-05-02
The main reason to install from the repositories instead of importing from the website (besides ease of installation) is that the software is then listed with apt-get.  This enables apt-get to monitor the installation, provide you with updates when they become available, and in the event you uninstall it (or something else with the same dependencies), will uninstall unneeded dependencies and keep those that are used by something else.

As for feeling stupid; everyone here has gone through a learning curve, and in most cases, still are.  I have had to literally *battle* with losing my Windows mindset, where you download or obtain software, put it in your computer, and install it.  That still happens with Linux, but with the repositories and package managers that practice has largely been surplanted.

Enjoy! ):P
