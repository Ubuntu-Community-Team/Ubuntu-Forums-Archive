Hi there,

I've got a very annoying problem that basically renders Konqueror useless as a webbrowser. When I click any link (html, php..), Konqueror asks me whether I want to save the file, rather than just show the page.

like, 
> 
Open 'http://youtube.com/watch?v=L8QuZfT5yIE'?
Type: HTML Document

[Save As..]    [Open with 'Kate']

I've had this problem before, but only for some pages. Now it happens everywhere. I searched the forum, but that didn't return anything useful.

---

### Post by aysiu on 2007-05-04
In the Konqueror settings (or maybe in KControl--I forget which), look for KDE Components and File Associations. Make sure the HTML ones are set to Konqueror and not Kate.
