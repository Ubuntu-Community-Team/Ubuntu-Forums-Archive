### Post by aquascrotum on 2009-08-16
Hopefully simple question - I'm setting up my folks machine on Jaunty, but I know rightly I'll get a "eh what do I do now" phonecall when they get to the login screen.

Can it be configured that a list of available user accounts appears (a la Windows) at startup?

---

### Post by Brandon Williams on 2009-08-16
Go to System -> Administration -> Login Window. Select the Local tab and then select a theme that includes a user list.

---

### Post by balaknair on 2009-08-16
Right click on the 'User-Switcher'/'Log-out' applet on the panel> 'Setup Login Screen'> 'Local' tab> Choose one of the themes with a face browser, like 'Human List'.

You can also install other GDM themes with face browsers that you download, like this one below, using the 'Add' button next to the theme list on the 'Local' tab
[http://www.gnome-look.org/content/show.php/Jaunty+Human-Face?content=105416](http://www.gnome-look.org/content/show.php/Jaunty+Human-Face?content=105416)
