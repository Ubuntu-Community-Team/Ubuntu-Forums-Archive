### Post by tew88 on 2009-05-09
I'm trying to use a second monitor on my laptop (running Ubuntu 9.04). I'm experiencing a number of difficulties, but I believe the root of the problem has to do with the fact that upon opening System -> Preferences -> Display, the system starts to lag and the Display manager is just an unpopulated window. It does, however, function normally when the external monitor is not plugged in.

If I boot up the laptop with the monitor plugged in, it works fine - but runs in my laptop's resolution of 1280x800.

ATI's Catalyst Control Center app recognises the monitor's existence, and under an information tab confirms that the monitor is capable of 1680x1050. I am, however, unable to increase the resolution beyond my laptop's maximum.

The card in question is an ATI HD 2400 (M). I'm using the proprietary AMD/ATI driver.

Before trying Linux, I had the system working as desired under Windows Vista.

Hope someone's able to point me in the right direction! ;)
