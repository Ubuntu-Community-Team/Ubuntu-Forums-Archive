### Post by hivelocitydd on 2009-01-09
the DVD edition mainly contains many if not all of the packages in the main repository plus many in the restricted one,
In theory this would allow you to complete the install without accessing the net.
In practice this may not be the case as some packages in the repository could very well be updated while the ones in the DVD may be obsolete.
