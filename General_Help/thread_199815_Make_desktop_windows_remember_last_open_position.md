### Post by [S|G] on 2006-06-19
Click on the window icon on the top left corner. Then choose advanced, special application settings. Then go to the geometry tab, mark the "position" checkbox and choose "remember" from the drop-down list.

The dialog names might differ slightly, since I'm not using my kde with english regional settings, but it must be similar to that :P

### Post by bryanlking on 2006-06-19
I like my windows to open centered on the desktop.  In KDE it's very easy to set this up.  KDE will then always remember to open all windows centered.

Right-click Task bar of any opened window..
Select Configure Window Behavior..
Select Moving Tab..
about halfway down the page find..
Placement..
Choose Centered.

Click OK, and then you're done.

BK
