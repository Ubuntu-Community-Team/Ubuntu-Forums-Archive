### Post by fragos on 2008-12-28
Right cclick the bluetooth icon in the upper right aand select preferences. There should be two tabs. One will have your hostname. There's a Known Devices text block. The "+" button will help you pair with bluetooth devices. To receive bluetooth files you'll need to run "Add/Remove Application" and install "Bluetooth File Sharing". To send files, right click the bluetooth icon and select the apprpriate menu entry.
