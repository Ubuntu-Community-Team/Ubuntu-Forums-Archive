### Post by blueridgedog on 2009-11-19
In firefox, select "bookmarks" then "Organize Bookmarks" then click "import and backup" select the option to backup your bookmarks.  Create a file your desktop, then email that file to your son with the instructions to follow the same proceedure, but selecting restore.  

The entire process could be done with export and import HTML on the same menu.
