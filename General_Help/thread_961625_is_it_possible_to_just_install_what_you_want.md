### Post by jerome1232 on 2008-10-28
Well it might be more base than your thinking but you can download the alternate install cd and just install the base system (press f4 and tell it command-line install).

This will give you the bare necessities for a running command-line system and you can start installing the programs and features you want. You'll need a good idea of what you actually want though :)

### Post by snowpine on 2008-10-28
Jerome has good advice. Building up a system from a command line install is a really fun way to learn how Ubuntu works!

If you just want to prevent bluetooth from starting when you boot up, go to System->Preferences->Session and uncheck the box next to it, as well as any other services you don't need to auto-load.
