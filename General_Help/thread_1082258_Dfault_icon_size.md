### Post by mrl777 on 2009-02-27
Is there any way to change the default size of all the desktop ions?  I find them too large, and I want to change all of them.  I hate doing it manually for each one.

---

### Post by geirha on 2009-02-27
Open a nautilus window (Places -> Homefolder). Choose Edit -> Preferences, and look at Icon View Defaults on the first tab. Changing the default zoom level will change the size of all icons in both the nautilus windows and the desktop (since nautilus is responsible for drawing the icons on the desktop).
