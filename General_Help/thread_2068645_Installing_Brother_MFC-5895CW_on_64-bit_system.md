Here is a link to the Brother driver download page,

[http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/download_prn.html#MFC-5895CW](http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/download_prn.html#MFC-5895CW)

Here is a further link to the installation instructions for the MFC-5895CW driver,

[http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/instruction_prn3.html#dpkg1](http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/instruction_prn3.html#dpkg1)

I suggest following the recommended install procedure and see what results.
