### Post by drs305 on 2008-12-27
> **DeMartini said:**
> thanks so much, i was kinda hoping to have the icon on the panel as well.....

You can place just about any application on the panel:
Rt Click the panel, Add to Panel, Custom Application Launcher (if the app is not listed in Application Launcher), Add. Enter a description and command (in this case: nautilus). If you enter 'nautilus' in the command section, ubuntu will recognize it and automatically insert the correct icon. If you want a custom icon, you can click the upper left section and navigate to the icon location of your choosing.
