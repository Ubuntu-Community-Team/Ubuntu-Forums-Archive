### Post by top_bullfrog on 2008-11-01
Hi,

I figure this is a simple enough problem, but I can't find anything in the system settings to fix it, nor can I figure out how to word it in any search engine.

I'm running kubuntu 8.10 and whenever I switch to a different virtual desktop, all the applications from all my desktops are visible on the taskbar.  I'd like to just see the applications from the current desktop on the taskbar.  I was playing with KDE 4 with 8.04 and found a way to do this buried in somewhere, but I haven't been able to find it in 8.10 after I upgraded.  If anyone can tell me how to do this, I'd be grateful.

Top_bullfrog

### Post by top_bullfrog on 2008-11-03
So in case anyone runs across this thread with the same problem, I found out how to fix it.  Make sure your taskbar has some free space on it and right click on the free space.  Click "Task Manager Settings" on the menu that pops up.  Check the box marked "Only show tasks from current desktop" and hit OK.  There ya go!

### Post by ultrageeky on 2011-10-13
Thanks for this. If you don't have any free space where the running task icons show you can right click beneath the icons for the running windows to open the task manager settings menu.
