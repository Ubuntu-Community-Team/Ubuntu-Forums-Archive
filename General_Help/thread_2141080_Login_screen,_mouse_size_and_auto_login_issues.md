### Post by Dennis N on 2013-05-01
> **rmcellig said:**
> Maybe I will try LXDE. I want a light fast and easy to use DE for her.

Lubuntu (and the LXDE desktop installed in an existing Ubuntu) use the same login screen manager, so that won't change.

As to the cursors being too small, consider installing a larger size cursor theme:

I use these in Xubuntu 12.04 and 12.10:

[http://gnome-look.org/content/show.php/Flatbed+Cursors?content=52027](http://gnome-look.org/content/show.php/Flatbed+Cursors?content=52027)

There are 4 sizes. Perhaps the 'huge' size would work. Colors make them easy to see.

If you set up one of these as the cursor theme with the Alternatives Configurator tool (package name = galternatives), it will not change from place to place in Xubuntu. Mine is consistently the same.
