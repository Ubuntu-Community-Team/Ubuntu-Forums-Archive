The red arrow automatically disappears when critical Ubuntu updates had already been installed on your system.

Icon colors may vary according to importance, the red color would equal to critical updates. The other colors, I'm not sure. (Try not to quote me on this as I'm not sure with the color coding)

### Post by fragos on 2009-03-04
You can force the icon to display by installing a package. For example with synaptic. The icon will appear greyed out when to click the apply icon and turn off when the install is complete. Update Manager only turns on the icon if updates are available. It's checking interval, normaly daily, can be changed in the Updates tab of Software Sources Preferences.
