### Post by mb_webguy on 2010-05-11
Not to dampen your spirits, but have you tried the many, many other media players available for linux other than Totem (Ubuntu's default multimedia player) or Rhythmbox (Ubuntu's default music player)?

For music, I prefer Amarok to Rhythmbox.  It's a KDE app, but except for the fact that it requires a few extra packages if you're running the Gnome desktop and not KDE, it runs fine under Gnome.  There's also Banshee, Exaile, and a [couple of dozen others]("http://en.wikipedia.org/wiki/List_of_Linux_audio_software").

As for multimedia players, there are likewise a number alternatives.  I prefer VLC, but there's also MPlayer, which is [skinnable]("http://www.mplayerhq.hu/design7/dload.html#skins") and is also used and the back-end of [numerous players]("http://www.mplayerhq.hu/design7/projects.html#mplayer_frontends").

Rather than reinventing the wheel, if you simply don't like the appearance of any of the existing audio/media players, you might want to consider creating a skin for MPlayer.  If you want to add some functionality, you might want to create a GUI front-end for MPlayer.  Either would make more sense imo than writing a new media player from scratch.
