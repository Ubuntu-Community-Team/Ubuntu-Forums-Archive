### Post by Cheesemill on 2013-02-20
Preload won't ever affect the amount of RAM available to applications.

If an application asks for more memory than is currently free then preload will drop data from the cache to make memory available.

For a good background on caching and memory take a read of this...
[http://www.linuxatemyram.com/](http://www.linuxatemyram.com/)

All preload does is actively load your most used programs into cache before they are used, instead of applications only being cached after they have already been used.
