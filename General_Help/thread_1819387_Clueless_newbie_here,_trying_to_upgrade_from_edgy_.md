### Post by CatKiller on 2011-08-06
With the release that old, you'll be much better off backing-up your data and installing fresh. You can't upgrade between arbitrary releases, they have to go in order. When you get to a Long Term Support release you can miss out the intermediate releases and go LTS -> LTS, but you'd still need to do six upgrades by my reckoning, each of which takes several hours. Installing fresh will take less than an hour. There have been quite a few significant changes over the last five years, so you should probably expect some breakage with that many upgrades-in-place.

Grab the cd image and try the live environment out to make sure that everything works and then install fresh. If you create a separate partition for /home (where your settings and personal data are stored) it will be easier to install over the top in the future without losing everything.

To answer your question, you're getting the 404 errors because when a release reaches End-Of-Life, the repositories of software for that release get moved to a different server.
