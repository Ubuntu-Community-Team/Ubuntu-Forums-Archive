### Post by Holger_Gehrke on 2018-07-22
The redirection to amazon is not a bug, it's a feature ... Look in preferences->Search. It's probably set to use the adress bar for search. If you activate the separate search entry field it should stop searching for mistyped addresses. You can also set the default search engine there (It's probably set to amazon for some reason, that's why you get sent there on mistyped URLs) and set or remove search engines to use.

You can use the add-on Noscript to stop all scripts except the ones you allow. A lot of pages nowadays contain more JavaScript than actual data (some don't contain any data at all - it's loaded by the scripts ...) and become completely unusable without JS. You might also want to use an ad-blocker - a lot of advertisement contain heavy scripts.

If your machine gets completely unusable because of scripts from websites, you've either opened a lot (10 or more) tabs or one of the sites is doing something (bad and) calculation-heavy on your machine (like mining crypto-currency in your browser on behalf of the website-owner) - or your machine has such low specs that you should look for a lighter browser than Firefox or Chrome.

Oh, and 'monopoly' and 'democracy' are not opposites, they are words from completely different domains: monopoly is a economic situation where some good or service is only available from one seller, who can set price and availability at whim, while democracy is one of the three basic political systems (the other two being 'monarchy' {rule by one person} and oligarchy {rule by a (selected) few}. A monopoly can exist under a democracy - take copyright as an example, which grants a temporary monopoly to the creator of a work.

Holger
