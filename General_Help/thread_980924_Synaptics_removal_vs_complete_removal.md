---

### Post by niyonk on 2008-11-13
Complete removal will remove the application together with its configurations (example .compiz, .nautilus, .limewire etc...)
-Anything related to the application

A simple removal will just remove the program without touching the configs. So next time you decide to install the software again, it will start configured.
