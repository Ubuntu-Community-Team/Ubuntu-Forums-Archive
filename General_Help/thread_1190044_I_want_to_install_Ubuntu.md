Hello RJ12,

Let me link you the wiki (or part of it) for WUBI to help in your decision-making.

[https://wiki.ubuntu.com/WubiGuide#How%20can%20I%20access%20my%20Wubi%20install%20and%20repair%20my%20install%20if%20it%20won%27t%20boot?](https://wiki.ubuntu.com/WubiGuide#How%20can%20I%20access%20my%20Wubi%20install%20and%20repair%20my%20install%20if%20it%20won%27t%20boot?)

Now, I have limited experience in WUBI.  However, as you have tried it once before, you may know better.  We are working right now on a WUBI install with some boot issues.  Nothing difficult ... just different from a true dual-boot setup. 

If it worked with XP home, I see no reason why it should not work in pro.... unless you changed from 32 to 64bit.  Worse case, should it happen, is you can uninstall wubi as if you unsintall a window file.

Peruse the forums ... you will see some wubi-related threads that may interest you and give you a bigger picture.  I, personally, prefer to make a true 2-boot ... that way should anything happen to one OS, I could always use the other for as long as I can still boot.  Unlike in WUBI, if windows crashes, so does your Ubuntu as well (be 'unreachable) until you can fix windows.

Good luck,
