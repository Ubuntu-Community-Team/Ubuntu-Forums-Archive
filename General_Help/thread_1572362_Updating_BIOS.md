### Post by lykwydchykyn on 2010-09-11
I think you're out of luck here.  The only BIOS update file available for this system is a Windows executable.

In the past, you could usually get a DOS executable, or at least a BIOS image file that could be flashed with some 3rd party tool.  Some manufacturers like Dell actually offer BIOS images and flashing tools for Linux.

But this situation is none of those, it looks like you have to run this thing under Windows Vista or 7.

Maybe there's a way around this, but I'm not seeing it at the moment.
