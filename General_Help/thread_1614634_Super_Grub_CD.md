### Post by oldfred on 2010-11-06
I do not find it difficult to reinstall. I always like to have multiple ways to boot computer, so I have several liveCDs and several repair ISO installed on a bootable USB flash drive. I would then want both the old and new versions of supergrub.

Install from LiveCD install on sda5 and want grub2 in drive sda's MBR:
Find linux partition, change sda5 if not correct, and/or even sda if sdb wanted:
sudo fdisk -l
sudo mkdir /mnt/sda5
sudo mount /dev/sda5 /mnt
sudo grub-install --root-directory=/mnt /dev/sda
If that returns any errors run:
sudo grub-install --recheck --root-directory=/mnt /dev/sda
