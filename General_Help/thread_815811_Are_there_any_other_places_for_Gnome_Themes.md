### Post by Forlong on 2008-06-02
There are no such things as GNOME themes.

There are Metacity (that's the window manager of GNOME) themes -- those can't be used on Xfce (when using Xfwm, of course).
And there are GTK themes -- GTK is a toolkit and Xfce uses it too.
