### Post by speedsix on 2006-11-02
To get **totem-gstreamer** to play pretty much every file type make sure you've got the following packages installed (enable universe/multiverse first)

gstreamer0.10-ffmpeg
gstreamer0.10-fluendo-mpegc
gstreamer0.10-pitfdll
gstreamer0.10-plugins-bad
gstreamer0.10-plugins-bad-multiverse
gstreamer0.10-plugins-ugly
gstreamer0.10-plugins-ugly-multiverse

Then install the W32codecs if you haven't done so already
```

wget -c http://www.debian-multimedia.org/pool/main/w/w32codecs/w32codecs_20060611-0.0_i386.deb
sudo dpkg -i w32codecs_20060611-0.0_i386.deb

```


Or you can use **totem-xine** plus the libxine-extracodecs package along with the w32codecs (see above)
