### Post by squenson on 2012-01-31
Make sure that the memory is well plugged in the socket. If it doesn't give better results, try the memory one by one to see if one is a faulty one, then try the four sockets separately to see whether one is faulty.

### Post by pbrane on 2012-01-31
you should try reseating the DIMMs as **squenson** pointed out. Also some BIOS will not update the RAM unless you follow the prompts when the computer boots. I think most modern BIOS will update the RAM configuration automatically.
