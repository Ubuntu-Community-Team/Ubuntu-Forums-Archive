### Post by Herman on 2006-07-15
> "The alternate install CD allows you to perform certain specialist installations of Ubuntu. It provides for the following situations:

* creating pre-configured OEM systems;
* setting up automated deployments;
* upgrading from older installations without network access;
* LVM and/or RAID partitioning;
* installing GRUB to a location other than the Master Boot Record;
* installs on systems with less than about 192MB of RAM." (Quoted from the Official Ubuntu Wiki)

The 'Alternate' Install CD has the more traditional text based partitioner and installer.
With the partitioner being text based, it can be used to install in machines that have less RAM.

The 'Alternate' Install CD's installer offers a choice of bootloaders, GRUB or LILO.
 It also offers the full range of choices as to where you want the bootloader installed. You can install GRUB to MBR (standard), or to the Ubuntu partition, some other partition, to a floppy disk, or other custom location. Lilo can also be installed with a similar range of choices as to location.

The 'Alternate' CD has a more advanced 'Rescue a broken system' mode too. (Your system does not need to be broken to use it, it's for doing special jobs like re-installing a bootloader for example.)

The 'Alternate CD' install is the one we always used before the new graphical partitioner in the new 'Desktop' CD was developed. The problem with the traditional text based install was that some people didn't know what to do with it (how to use it). I made a website to try to help guide people through the process. Click on my 'Sig links' to see what installing with the 'Alternate' CD will be like.
Regards, Herman :D
