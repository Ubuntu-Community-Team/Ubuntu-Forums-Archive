### Post by Zill on 2013-06-24
> **huilong said:**
> ...I have deleted a user account accidentally and its data also. Do you know How I can get user home directory back after I delete a specific user account...
Simply restore your latest backup!  Presumably you *do* have backups - if you don't then your data cannot be important to you.

If, regrettably, you do not currently backup your system(s) then you might like to consider this in future. [This link]("https://help.ubuntu.com/community/BackupYourSystem") might help.  Personally, I recommend [luckyBackup]("http://luckybackup.sourceforge.net/features.html") as this is a reliable backup program that is very easy to use.
