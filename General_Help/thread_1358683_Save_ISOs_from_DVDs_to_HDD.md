### Post by howefield on 2009-12-18
Using Ubuntu, with the cd in the drive, right click the desktop icon for the cd and select "Copy Disk"

In the "Select a disc to which to write" drop down field select Image File, click on the properties button to select iso or other.

Then press copy to create the image.
