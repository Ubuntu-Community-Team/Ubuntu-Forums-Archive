### Post by amaroKer on 2008-03-01
You're not going to be able to use the same programs you used on Windows for Ubuntu.  However there are of applications that provide similar features or functionality to those you're used to on Windows.  I switched to Ubuntu about 2 months before I started at college, and the only application I need that I cannot find an equivalent for is AutoCAD.  Most desktop apps have a Linux-compatible alternative.  I'll try to find you a more comprehensive list.

Perhaps the question you should be asking yourself is what functionality do you need that you're not getting from Windows.  A change of scenery is always nice, but if you rely on proprietary software to a great degree, you may have problems working in a Linux environment.  [www.osalt.com](www.osalt.com) is a good starting point for this situation.

We should discuss what programs and tools you currently use and see if we can find a solution for you.
