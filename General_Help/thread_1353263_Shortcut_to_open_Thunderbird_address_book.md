### Post by StuartN on 2009-12-12
> **winecurmudgeon said:**
> In Windows, it's reasonably simple to set up hot keys to open applications. Is there something similar in Ubuntu? I want to be able to launch the Thunderbird address book with hot keys. Any thoughts?

You could create a launcher (right-click on the desktop), add a menu item or type directly in the terminal **thunderbird -addressbook**

Other thunderbird commandline options are here [https://developer.mozilla.org/en/Command_Line_Options](https://developer.mozilla.org/en/Command_Line_Options)
