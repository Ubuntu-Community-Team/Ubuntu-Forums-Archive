### Post by Trog on 2008-05-02
Select Animations, select the Close Animation tab. In the animation selection window, double click on the current animation and select the one you want from the drop-down list

### Post by MulletMan on 2008-05-02
You can also do this from the Simple Compiz Config Settings manager under the animations tab. It's very straight forward once you're in there.
