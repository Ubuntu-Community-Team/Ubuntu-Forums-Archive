### Post by kostkon on 2011-05-10
The microsoft fonts package contains the times new roman font and its variants:
```
Times New Roman (Bold, Italic, Bold Italic)
```

The package is called:
```
ttf-mscorefonts-installer
```
just install it using the software centre (you'll be asked to agree to its term of use and license at the start of the installation.)

Then just restart libreoffice.
