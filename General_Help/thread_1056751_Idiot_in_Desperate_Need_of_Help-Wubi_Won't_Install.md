### Post by mainnetwork on 2009-03-30
Sorry to revive an old thread.

Nada means nothing. Acer computers come with two partitions, the C: and then a D: called data, which is empty. Is wubi giving you an error? It should simply download ubuntu and put it in a folder. After that you reboot your computer, choose Ubuntu when it starts, and wait while it installs. It won't touch your windows files if that's what you're worried about.

Also, if you want to try Ubuntu, consider a Live CD or USB, you can order a free ubuntu CD off the website.
