I generally check these forums for issues before I apply any kernel update or update any mission critical applications.

If you are running an internet facing server, or are a heavy link following web surfer, the security updates are crucial so you do want to drop what you are doing and search for problem reports, but the others can always wait.

If its a desktop behind a decent firewall and you are a careful web surfer, even the security updates can wait a while for you to search the forums for problem reports.
