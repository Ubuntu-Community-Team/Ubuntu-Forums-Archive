### Post by Dreamer Fithp Apprentice on 2012-03-12
You might be more likely to get a useful reply if you give a lot more details. Like exactly what ubuntu you are running, what desktop if any, graphics card, precise circumstances when you see this behaviour, and what if anything, prevents it. Did you just install Ubuntu or have you just changed versions or what?  You might want to look at your power management settings. For me, with Oneiric with Mate, on a desktop, that is System, Preferences, Power Management, AC Power tab - under display - try setting it to never and unchecking dim.  Also check, System, Preferences, Screensaver. Try makeing sure there is NO check in the box by "Activate screensaver when computer is idle". If you want to be belt and suspenders anal about it like I am you can run up the regard as idle time as far as it will go (2 hours for me) and make sure it is set to something OTHER than blank screen so if some cursed malfunction kicks in the screensaver in spite of the other settings that should prevent it, at least it won't look like a dead monitor.  And of course you might want to see if the same thing happens with the computer attached to a different monitor or with the same monitor attached to a different computer but check the easier stuff first. You might want to make sure the power and data cables to the monitor are firmly connected at both ends too. Sorry if some of this sounds kindergarden but your post doesn't make clear you checked this stuff and even the geekiest of us have sometimes wasted time trying to figure out what was wrong with an some piece of unplugged hardware.
