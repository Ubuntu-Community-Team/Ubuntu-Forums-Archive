### Post by Elfy on 2013-10-08
I don't have problems with it remembering for me.

Maybe try either editing the config file 

.config/Mousepad/mousepadrc

Make sure that ViewWordWrap=true

or delete the mousepad folder from .config

---

### Post by motorcity909 on 2013-10-08
Thanks Elfy

The config file already was set as ViewWordWrap=true so I deleted the folder and then re-opened Mousepad and now it is indeed saving word wrap.

Great stuff - thank you!
