### Post by edm1 on 2009-04-25
Just download the new wave pack from gnome-look, extract it then drag the file named new-wave080.tar.gz (or something like that) into the Appearance settings window (System>Preferences>Appearance). There are other aspects of the theme that can be installed eg. the firefox theme but there should be instructions within the extracted folder.

### Post by W.Sorting on 2009-04-25
Hi there!

1. Head over to "www.gnome-look.org" or "art.gnome.org" (These are the best two websites for ubuntu themes, in my opinion ;) 
2. Browse the website.  To get themes for ubuntu, click on the "GTK 2.x" link on the left.  Then, when you see one that you'll want for a theme click on the title to enter its page.  
3. There will be a "Download" button if you scroll down.  Download the theme and save it to your computer.  Remember the folder you saved it in!
4. Open System -> Preferences -> Appearance.  Click on the "Theme" tab if it is not open already.
5. Click "Install", and then find the file you just saved. After clicking "open", the theme you chose will be automatically installed.

NB: For some themes, when you click "Download" it may redirect you to a third-party website.  

Hint: You can customize themes as well! Make sure your theme is selected, then click "Customize".  In there you can customize the Window Border, Icons, Pointers, and much more!

Hint 2: There are ubuntu applications that will automatically install themes online.  Then you won't have to physically go to the websites. Search for them in the repositories and they will be there.

Hint 3: If you're using KDE (Kubuntu), go to "www.kde-look.org". Process may vary.

Good luck!  

-W.Sorting
