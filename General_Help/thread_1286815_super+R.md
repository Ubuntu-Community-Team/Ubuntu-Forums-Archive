This is equivalent to super-d on a windows system.

I think that's what you're looking for?

### Post by prshah on 2009-10-09
> **fcuk112 said:**
> pressing the windows key + R zooms in on the desktop and scrolls the screen slightly when i move my mouse.  is there any way to revert to the normal desktop using a key combo?

You can set it up in System-Preferences-CompizConfig Settings Manager-Enhanced Zoom Desktop-Specific Zoom (Hint: Setting Zoom factor to 1 disables zoom).

In the meantime, you can use <Super> (Windows key) and the mouse scroll buttons (scroll down) to reset the zoom factor.

If that doesn't work, check the correct mouse and keyboard bindings in System-Preferences-CompizConfig Settings Manager-Enhanced Zoom Desktop-Zoom In/Out.
