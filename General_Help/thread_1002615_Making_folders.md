### Post by ugm6hr on 2008-12-07
System->Prefs->Main Menu

This allows you to look at the properties of all your menu items, and to add new Menus or New Items.

You can add the New Menus to create separate Photoshop / Other folders.

Then, you may be able to drag and drop.  If that doesn't work, right-click on Photoshop entry, and select properties.  Copy the command there, and create a new item with the same command wherever you want to put it.
