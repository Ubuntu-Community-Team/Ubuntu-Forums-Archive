### Post by 2F4U on 2011-09-03
There are so called archive managers that are able to handle formats such as rar, if an appropriate command line tool is installed. In the case of rar, this would be 
  - rar
  - unrar

Both are in the multiverse repository, since rar is a proprietary format. So, if for example, you use file-roller under Gnome and install the above mentioned programs, you should be able to handle rar archives.
