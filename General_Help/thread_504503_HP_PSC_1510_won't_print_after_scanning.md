My PSC 1410 doesn't have this problem with Feisty or with Edgy.  These printers  are the same vintage.  Have you tried using HPLIP Toolbox.  It's in System-> Administration.  You may need to install the printer with HPLIP Toolbox for it to see the printer.  It was well worth the small effort as I can now run tests and even see the ink levels.
