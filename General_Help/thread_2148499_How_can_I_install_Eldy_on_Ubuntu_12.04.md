### Post by ohnonot on 2013-05-25

### Post by ohnonot on 2013-05-26
only after posting i realised what eldy actually does and thought, maybe our instructions weren't helpful to the typical user of eldy.
also the link i gave in post #2 is not the newest version.
so i'm rewriting our instructions:

download [_**this**_]("http://download.eldy.net/eldy23.tar.gz").
open your file manager to where the file is now.
extract it - often your file manager has an option "extract here" when you right-click the file. another way is just doubleclicking the file and use the archiver.
you now have the archive and a folder called "eldy23". is the folder in a place where you are happy to leave it? now is the last chance to move the whole folder to a different location, but that's optional.
change into the "eldy23" folder.
find the eldy.sh file and double-click it. - eldy should start normally now.
make a mental note where the eldy.sh file is and then close the file manager.
right-click your desktop -> create new launcher.
there's an entry field that says "executable" or sth like that, with a button on the right. click that button, navigate to the folder with the eldy.sh file and double-click eldy.sh.
save the launcher.
-that should be enough, you should now have an eldy launcher on your desktop.
optionally, you can delete the archive you downloaded.

please write again if this does not give the desired results.

ps: i know nothing about the functionality of the program itself.
