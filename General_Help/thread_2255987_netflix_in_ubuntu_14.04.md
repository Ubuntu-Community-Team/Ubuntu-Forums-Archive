### Post by deadflowr on 2014-12-09
You can't use chromium, either install pipelight for firefox, or install Google Chrome.
Pipelight can be a little complicated, so I'd recommend the Google chrome method.
Once installed, and as long as your Computer is up-to-date, it'll run without any need to do anything, like using an agent switcher.

Here's a link for pipelight if you want to go that route
[http://pipelight.net/cms/installation.html](http://pipelight.net/cms/installation.html)

And a link to get the Google Chrome browser
[https://www.google.com/chrome/browser/desktop/](https://www.google.com/chrome/browser/desktop/)


Simply put chromium and Google Chrome are not the same, even though Google Chrome is built from chromium. Google Chrome contains the closed software that allows the streaming of the content from Netflix. chromium does not.

Hope this helps.
