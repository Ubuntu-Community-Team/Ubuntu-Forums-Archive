### Post by Afkpuz on 2008-11-12
Welcome to the wonderful world of linux.  First lesson: wine is not a perfect program.  Wine does amazing work, but there are countless games that will not work at all with wine.  Some games work halfway.  Some work perfectly, like starcraft.  It's not ubuntu's fault: it's microsoft's fault.  Since direct X is copywritten and since most games use direct X, you'll have some trouble.  My advice is get on winehq.com and search for which games work, rather than just trying games.  Sorry man.  Wine is always getting better, but linux is not for gaming.

### Post by -Zeus- on 2008-11-12
to install vista now, you need to boot to a live cd, go to System > Administration > partition editor.  You need to add a new partition, NTFS type, to install vista on.  Once that's done, you can install vista, then follow this guide to restore Ubuntu: [https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows#Quick%20Start](https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows#Quick%20Start)
