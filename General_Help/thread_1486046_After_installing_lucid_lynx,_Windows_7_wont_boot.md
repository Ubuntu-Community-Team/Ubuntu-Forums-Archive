### Post by Skumleren on 2010-05-17
Arh! fixed it. It was as i feared. I has overwritten the boot sector.
If anyone else ends up in the same situation:
This may be a fix for repairing the boot sector
[http://sourceforge.net/apps/mediawik...ms:Boot_Sector]("http://sourceforge.net/apps/mediawiki/bootinfoscript/index.php?title=Boot_Problems:Boot_Sector")

Otherwise open Bootrec.exe from the windows recovery cd/dvd. Repair the boot sector(Not the master boot record, if you hold your grub dear)

If your problem was a damaged boot record, this will work.
