### Post by SeijiSensei on 2012-04-10
> **cottfcfan said:**
> Even if Thunderbird deleted your profile, all your emails should still be on the server.

Not if he uses POP3 which deletes the mail from the server and stores it locally.  IMAP leaves the mail on the server and is the better protocol to use if it's available.  There is an obvious privacy issue with IMAP since your mail is stored on someone else's server.  Of course, if you can't trust the mail provider, why are you using their service?
