### Post by theicyj on 2007-05-11
You can install KDE alongside gnome in Ubuntu very easily.  Open synaptic, the under either the edit menu or the preferences menu, select "Install packages by Task" or something along those lines (I am away from my linux desktop right now).  Then check the Kubuntu Desktop box and apply changes.

### Post by gh0st on 2007-05-11
Yes you can just install the KDE window manger and choose between Gnome or KDE session when you login.

To install kubuntu desktop try this:

```

sudo apt-get install kubuntu-desktop

```

It should do the trick, you'll probably have to log out and log back in with a KDE session to try it. Good luck :)
