### Post by lovinglinux on 2009-07-23
> **TheNerdAL said:**
> What do you mean? I am new to this.

I means it is a css style problem. Basically CSS tells for example the size of the post area, the color of the heading, the color and size of the text. Each post area has several page elements like borders, titles, text area and such. CSS also controls the position of all this stuff, so when when CSS is borked, the elements of a page get misaligned and eventually overlap each other.

There are some extensions like those I mentioned that modify the css style dynamically when displaying to you. They might be messing with your page.

You could try to delete your cache and reload the page.
