### Post by HavocXphere on 2009-10-10
Universities usually don't provide pop/imap access to their email servers...only web interface.

Check web interface for an option to forward all emails.

In the unlikely case that they do provide pop/imap you'd need to know the ports they use and the authentication/encryption type to set it up. And no, there is no app to set it up for you.
