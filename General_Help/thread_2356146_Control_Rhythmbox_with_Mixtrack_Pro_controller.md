Am currently playing sound with Rhythmbox via my Mixtrack Pro II controller soundcard, but no actual physical controls.

Wondering if there is a why to write a plugin to offload the Rhythmbox controls to the sound card controller?

I found this wiki on [Rhythmbox Plugins ]("https://wiki.gnome.org/Apps/Rhythmbox/Plugins/WritingGuide"), it appears to be some kind of a guide.

Rhythmbox plugins can currently be written in two languages, C and Python

> For Python plugins, best practice is to have a __init__.py file containing your code, the .plugin file and and other files in a directory called "yourpluginname" in the per-user plugin directory
