Compiz is installed by default in 8.04 and 8.10 (I'm assuming you're using one of the two).
**Compiz Fusion Icon** isn't. But you can install it from here: **Main Menu > Add/Remove**, and enter **Compiz Fusion Icon** in the searchbar. Select to install. After installation is complete, you can find it under: **Main Menu > System Tools**. When you click on it, it will appear in your panel.
