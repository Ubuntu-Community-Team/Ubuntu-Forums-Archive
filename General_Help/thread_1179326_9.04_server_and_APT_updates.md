Here's a script I use for manual updates (with mods based on the link in the previous post). This is meant to be run manually.

```

#!/bin/bash
rebootflag=0
answer="N"
/usr/bin/apt-get -V update
/usr/bin/apt-get -V dist-upgrade
/bin/ls /var/cache/apt/archives/ | /bin/grep -i linux-image* &> /dev/null && rebootflag=1
/usr/bin/apt-get -V clean
/bin/sync

if [ ${rebootflag} -eq 1 ]; then
{
    /bin/echo
    /bin/echo -n "Reboot required - reboot now? (yes/N) "
    read answer
}
fi

if [ "$answer" == "yes" ]; then
{
    /sbin/shutdown -rv now
}
else
{
    /bin/echo
    /bin/echo "Update done!"
    /bin/echo
    exit 0
}
fi

```

After update, if the kernel was updated, the script will prompt "reboot now?". You have to type in "yes" or else it won't reboot, but just end.

Hope you can use it.
