### Post by Mark Phelps on 2012-03-04
Dual-booting does NOT slow down the operation of a PC because each OS basically runs in its own native environment as if the other OS did not even exist.

Therefore, converting a dual-boot PC to a single-boot PC is not going to make it run any faster.

Furthermore, if the old PC was VERY slow under Windows XP, loading an 11.x version of Ubuntu is not going to yield any better performance than you had under XP.
