PmDematagoda, to remove the Xorg completely (including the configuration files), you can use this command:
```

sudo apt-get remove *--purge* xserver-xorg-core

```
or
```

sudo apt-get remove *--purge* xserver-xorg

```
(I'm not sure which one, but the *--purge* option will surely clean out the installation)
but make sure you've backup your xorg.conf to another directory (just in case, it gets removed too).

Then try installing again
```

sudo apt-get install xserver-xorg-core

```
or
```

sudo apt-get install xserver-xorg

```
