### Post by mb_webguy on 2009-03-27
Apparently vdrift is distributed as an autopackage.  Download it [here]("http://downloads.sourceforge.net/vdrift/vdrift-2009-02-15-src.tar.bz2"), and extract it to a convenient location (such as your home directory).

Right-click on the autopackage, and click Properties.  Go to the Permissions tab, and make the file executable.  Close the Properties dialog.  Double-click on the file to run it, and it will install the software.

NOTE:  This is all based on the installation instructions provided on the [vdrift website]("http://wiki.vdrift.net/Installing_on_Linux"), so I make no guarantees this is correct.
