### Post by tamoneya on 2008-08-20
ther is alt-tab and ctrl-alt-tab.  The first switches between all windows on the current workspace while the second switches between all you open windows on any workspace.  Both of these can be modified with shift to go in the opposite direction.  For closing ctrl-w typically works.
