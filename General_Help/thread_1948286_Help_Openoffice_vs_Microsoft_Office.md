### Post by wyliecoyoteuk on 2012-03-28
Unfortunately, Microsoft introduced something called "The Ribbon" in Office 2007.
This does away with the traditional menus and replaces them with a Tabbed interface.
Although it works in a similar way, it is much harder to navigate if you are already used to menus, and some options are a little harder to find.
Theworst change is that the file item is replaced with a large round icon called the "orb".
I would look at some web tutorials on it. if I were you.

Edit: apparently keyboard shortcuts are still similar, that might help.
