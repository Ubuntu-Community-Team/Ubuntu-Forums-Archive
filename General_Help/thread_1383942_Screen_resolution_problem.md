### Post by lidex on 2010-01-17
What's the version number of your mga driver? You can look in synaptic under "xserver-xorg-video-mga"? (Search "xserver")

Post the contents of:
```
gedit /etc/X11/xorg.conf
```

Try this command in a terminal:
```
sudo dpkg-reconfigure -phigh xserver-xorg
```
Then **reboot**.
