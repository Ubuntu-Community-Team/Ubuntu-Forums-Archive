### Post by Frogs Hair on 2012-11-04
The top panel in the Unity desktop has no auto hide . The launcher on the left can be hidden from system setting > appearance > behavior.

If using the Gnome Fall-Back session use  Alt + right click or super + Alt + right to access panel properties and set to auto-hide.
