### Post by TheOrangePeanut on 2008-03-20
Have you considered using gedit and the java command line compiler?  gedit has really good syntax highlighting, auto-tabbing, conversion of tabs to spaces, and line numbers; I use it as my primary code editor.  Building on the command line is easy, too.

cd into the directory where your java files are and then javac hello.java and then java hello.  Simple and efficient.
