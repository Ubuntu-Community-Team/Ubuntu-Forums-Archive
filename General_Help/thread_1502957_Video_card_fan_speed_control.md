### Post by KeyserSoze93 on 2010-06-06
Install the fglrx drivers using the restricted drivers manager or envyng.

This should provide a tool called aticonfig, which, according to [http://ubuntuforums.org/showthread.php?t=1236131]("http://ubuntuforums.org/showthread.php?t=1236131"), allows you to change the fan speed.
