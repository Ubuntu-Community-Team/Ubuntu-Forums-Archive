Most installations on a thumb drive do not install it in persistent mode meaning it does not save configuration or session information.  That would be why it does not save your screen size info.

I know there are ways to make it persistent but have not tinkered with thumb drives for awhile.  You may want to search around for different installation methods that would make it persistent.
