### Post by pointone on 2008-01-26
Try Edit > Preferences > Content > Colors...

Firefox isn't fully skinned by the system theme; it tends to do its own thing a lot of the time. I usually install a separate dark Firefox theme when using a dark system theme just so everything matches better.
