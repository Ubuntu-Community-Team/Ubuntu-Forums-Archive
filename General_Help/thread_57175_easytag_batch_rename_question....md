[QUOTE=escobar]I found how to fill in multiple selected fields. I can't figure out if I remove data from a certain field, how to apply that to everything else. For instance, I want to remove the comment fields for every song. I can do it, one by one, but not for every selected file at once. I did not see that button you were referring to though.[/QUOTE]
I just ran through doing this so I could try to figure it out for you as well as post the process for anyone else who may want to accomplish the same thing.

Let's say you have 10 songs from the same cd ripped into the same folder. In the comment line of each tag you find it says "Ripped by Goobox" and you want it to say nothing. What can you do??

In Easy tag (I'm using version 1.99.6) you do this:
1. use the browser on the left side of the window to choose the folder containing the songs which have the tags you want to change.
2. In the middle pane that shows all the songs, highlight all of them by selecting any one of them and then pressing the blue icon at the top for "Select All Files"
3. Go to the pane on the right and delete the tag in the comment line. (or whatever else you wish to change)
4. Right click on the now empty comment line and select the option "Tag selected files with this field"
5. All the selected songs should now have the tag deleted. Feel free to look at several to confirm. Then with all the changed fields selected, press the Save File(s) icon, hit yes to all, and you are done.
