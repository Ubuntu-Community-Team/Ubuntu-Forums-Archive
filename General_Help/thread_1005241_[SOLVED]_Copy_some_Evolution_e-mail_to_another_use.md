I'm leaving the organisation that I'm currently working at and need to give some (a few hundred messages) of my e-mail communications to another user. These are all stored in a subfolder of my Inbox on Evolution. 
The other user users MS Outlook. 

Does anyone know the best method of sharing these e-mails without having to send each one as a message?

---

### Post by rax_m on 2008-12-08
Ok, I have a workable solution:

1. Go into a folder on Evolution, select all the messages you want to export and click File->Save Message
2. This will export all of the selected messages as an mbox file
3. I downloaded and ran this Windows freeware application (under WINE): [http://www.broobles.com/imapsize/download.php](http://www.broobles.com/imapsize/download.php)
4. Ran the Windows exe and converted the mbox file to multiple .eml files

I would have preferred and all linux solution, but I didn't have time to play around. 

Hope this helps someone.
