### Post by kaibob on 2009-04-22
I don't have an answer to your question, but since there have been no other replies, I do have a suggestion. The Firefox about:config screen has a number of entries that relate to the print-to-file PDF function including:

```
print.printer_PDF.print_paper_width

print.printer_PDF.print_paper_height

print.printer_PDF.print_paper_name
```

You may want to play around with these to see if they will do what you want. I mention this as a possible fix because I was able to change the default save path and file type in this manner:

[http://ubuntuforums.org/showthread.php?t=1043339](http://ubuntuforums.org/showthread.php?t=1043339)
