### Post by abcuser on 2009-12-07
> **Cytochromec said:**
> 1) What difference does it make what GUI the distro uses? What makes OpenSUSE a better KDE distro than Kubuntu?
It is a matter of personal opinion. OpenSuse defaults KDE as default desktop environment, Ubuntu puts Gnome to default environment. Ubuntu has idea to have non-technical user to use computer.

> **Cytochromec said:**
> 2) I was thinking of trying Fedora KDE to kill two birds with one stone (try my second distro and KDE).  Is this a newb move?
Fedora distro is a playground for Red Hat Enterprise Linux. Fedora is for users that likes to get the latest and greatest software, so you can also get some unstable software. More suitable for technical people. If you are not afraid of asking on forums, then you could try Fedora as well. Kubuntu is more for the users that likes more stable software - more non-technical user that likes to get more advanced desktop environment comparison to Ubuntu (Gnome).

Now days it is not so important which distro you select (Fedora, Kubuntu, openSuse) there are all excellent and no hacker required.
