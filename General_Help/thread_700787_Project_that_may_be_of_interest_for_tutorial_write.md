Hey ubuntu folk,
I had an idea for a project, and and I have been coding it up.  I've been posting the code here at [https://launchpad.net/climl](https://launchpad.net/climl)

The basic idea is that you can create a basic GUI that posts commands to the CLI.    One of the main goals is that if a tutorial is talking about how to do something on the CLI, then it could be embodied into this projects html format, and then a GUI can be generated off of that.  

The original posting is here: [http://ubuntuforums.org/showthread.php?p=3886241&nojs=1#goto_threadtools](http://ubuntuforums.org/showthread.php?p=3886241&nojs=1#goto_threadtools)

That forum includes directions to get it working, and a link to a video demoing the project.  The video sucks, but hopefully, you can at least get an idea of what the project does.  The video is probably the best way to understand it at this point.  

Anyway, I hope you guys will check it out, leave some feedback, and even use the project.
