### Post by loboc on 2008-11-19
The best wat to do it would be to untar both icon sets to a two folders A and B then create a folder C where you copy in all of A replace the ones you want from B 

Edit the name of the icon set (I think theres a config file in there)

then re-tar C and use the theme installler to install it.

That way you dont have to deal with using sudo to copy files around /usr/share/icons or whatever
