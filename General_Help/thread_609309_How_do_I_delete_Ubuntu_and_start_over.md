### Post by conehead77 on 2007-11-10
I know all the Firefox bookmarks are stored in the ~/.mozilla directory. Just copy it to a usb stick and after a fresh install copy it back to your home folder.

Thunderbird has its settings saved in the ~/.mozilla-thunderbird directory. It saved all newsgroups and stuff for me but i didnt have any addresses stored so i dont know about them.

edit: directories with a "." are hidden files, You can see them in the Nautilus file manager if you press Strg+h.
