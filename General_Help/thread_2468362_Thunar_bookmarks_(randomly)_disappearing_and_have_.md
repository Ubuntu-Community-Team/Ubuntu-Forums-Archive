### Post by ajgreeny on 2021-10-26
I assume they have not simply chosen to use the tree view in the left hand pane with Ctrl+E?
If so return to bookmarks view with Ctrl+B

For extra safety keep a backup of the ***/home/<user>/.config/gtk-3.0/bookmarks*** file and restore if if necessary.
