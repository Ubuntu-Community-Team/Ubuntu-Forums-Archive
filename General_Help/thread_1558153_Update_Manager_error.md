### Post by Frogs Hair on 2010-08-21
I resolved this issue by installing the Apt Key manager from the repository . Install the program then open it from system tools it will display a list of software sources , then reload the synaptic package manager to see if the key was located . I used this app on two installations and it worked both times. I found this solution in a forum archive.
