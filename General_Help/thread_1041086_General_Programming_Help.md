### Post by mixmaster on 2009-01-16
Assuming you are doing this on a linux box, you will need to be able to write device drivers and kernel extensions to do the sort of things you seem to be aiming at.  You will also need to be a better than competent C programmer.  There is a free book on kernel programming here : [http://www.linuxhq.com/guides/LKMPG/mpg.html](http://www.linuxhq.com/guides/LKMPG/mpg.html) .  I don't recommend it - I haven't read it - but it should at least give you a flavour.

Be prepared to reboot your system.  Frequently.
