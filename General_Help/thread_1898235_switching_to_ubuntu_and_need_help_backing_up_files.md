### Post by jualin on 2011-12-20
Usually you are not able to easily back up already installed programs since to install them you will need the installer (hopefully I am not confusing you more). However, installing Ubuntu doesn't generally erase your windows partition if you choose the "[Install Ubuntu side by side Windows]("https://help.ubuntu.com/community/WindowsDualBoot#Automatic_partition_resizing_.28recommended.29")" option during the Ubuntu installation.
Also, you can try installing Ubuntu within Windows, using the [Wubi installer]("http://www.ubuntu.com/download/ubuntu/windows-installer") which will install Ubuntu as a program within Windows. The only drawback is that you will sacrifice some performance.

### Post by Mark Phelps on 2011-12-21
There is no simple way to back off ANY Windows apps, games included.  The most difficult part is finding all the entries in the Registry that the apps/games use, and saving those.

Laplink makes several products that DO allow you to "migrate" apps (games included) from one PC to another -- which is essentially what you're doing if you save off the apps/games, reformat your drive, and reinstall.  But ... the apps are not free and only work in Windows.
