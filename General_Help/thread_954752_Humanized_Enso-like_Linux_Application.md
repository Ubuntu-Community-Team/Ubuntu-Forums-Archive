### Post by potherca on 2010-02-16
First of, Thanks for the link!

I've also been looking for something to replace Enso on linux.
The GnomeDo interface is actually an improvement over Enso with its use of icons and the finding/installing of extensions being so easy. It's a shame it doesn't run on Windows too! (Although there's a project porting it to windows.)

It took a bit of getting used to using the [super]+[space] key combo, but there is a fix for that if you've got a key you don't use: [http://ubuntuforums.org/showthread.php?t=490402](http://ubuntuforums.org/showthread.php?t=490402) (I highjacked the pause/break key )

Something that might be worth looking into, that works on both windows and linux is Launchy: [http://www.launchy.net/](http://www.launchy.net/) Sadly I haven't gotten round to it yet...

Finally, migrating from Windows to Ubuntu, [http://alternativeto.net/](http://alternativeto.net/) has been a great help to me for finding alternatives to windows programs in linux. Although it didn't list GnomeDo as an alternative for Enso... 

Hope this helps.
