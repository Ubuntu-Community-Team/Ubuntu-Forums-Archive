### Post by kpkeerthi on 2009-05-15
How did you install it? If you installed it using a .deb file, you should be able to uninstall it using the below command:
```
sudo apt-get purge google-desktop-linux
```

Or search for google-desktop in Applications -> Add/Remove and uninstall it. 

Or post the link of the guide you followed to install it.
