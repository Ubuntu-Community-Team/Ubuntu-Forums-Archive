### Post by dcstar on 2010-03-06
> **nadarm said:**
> No that's not it.  I tried that but the font size did not change in the Evolution New Message window.  If you are in Evolution Main, create a new email.  In that window click on Format, Font Size.  This you can click on +1 to make the font type just a little larger.  What I would like is when I create a new email message, the Font Size is always +1.

The only way to do this that I have found is to create a new HTML e-mail message with +1 set **and** at least one character in the message saved as a Template.

You can then use the template for new messages and the +1 setting should be retained.

I have just reported this requirement for an extra character as a bug:

[https://bugs.launchpad.net/evolution-templates-plugin/+bug/533539](https://bugs.launchpad.net/evolution-templates-plugin/+bug/533539)
