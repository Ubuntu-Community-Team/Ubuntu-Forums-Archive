### Post by Zorael on 2008-04-30
Well, yeah, we're hurting in the corporate-support department. Adobe is one of the bad guys.


If you're going to make the leap, you have two options, regarding those apps. Either you try and run them under Wine, but while I know Photoshop should work, I wouldn't bet much on the others. See [http://appdb.winehq.org/](http://appdb.winehq.org/). I bet the Flash program is awarded a garbage rating.

The other option would be to run Windows in a virtual box, starting it when you need to do work and closing it afterwards. All programs that do not rely on specific hardware "will" work. The virtualization programs emulate hardware, so programs run inside Windows there will not have direct access to your videocard, for instance. Sadly, this rules out iTunes as well, since USB is (peripheral) hardware, too. Though I hear some have had greater success with vmware.

Obviously, you can't escape asking yourself what the point is of running linux if you're going to virtualize Windows anyway, but consider; it would be a closed environment with optional access to the Internet. You'd still do all your other work (browsing, sending emails, deleting emails, doubleclicking, whatever it is you do) in the much more secure linux milieu. You could install an antivirus program for the nostalgia, but that'd be it.

So, not completely having your cake and eating it too, but pretty close.
