### Post by 3Miro on 2010-11-07
Logout and at the bottom of the screen you should be given an option to use either Unity or Gnome. If Unity is the only session available, then you can go to Synaptic Package Manager and install Ubuntu-desktop (I don't know how to open SPM in Unity, you can use the Software Center which will list the Ubuntu-desktop as a technical item).
