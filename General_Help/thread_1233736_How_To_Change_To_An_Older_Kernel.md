An Ubuntu version will only have a selection of current kernels (for that version) available.

You may be able to use an older Kernel from a previous Ubuntu version by downloading it (and all the dependencies) from:

[http://packages.ubuntu.com/](http://packages.ubuntu.com/)

Then you can manually install the .deb packages.
