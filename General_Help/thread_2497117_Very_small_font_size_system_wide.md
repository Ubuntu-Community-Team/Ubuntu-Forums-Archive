### Post by cbiweb on 2024-04-24
One thing I never thought to check: screen resolution.

My 17" screen resolution defaulted to 1920x1080 (16:9) when I installed 22.04.
My Windows laptop, also 17", is set to 1600x900 (16:9), the default when I installed it.
When I changed Ubuntu to that resolution, everything improved a lot, almost to what I'm used to.

It's still a little different but I chalk that up to the font faces that Linux uses, which isn't a big issue. I can change them in the browsers, and they're fine for everywhere else.
