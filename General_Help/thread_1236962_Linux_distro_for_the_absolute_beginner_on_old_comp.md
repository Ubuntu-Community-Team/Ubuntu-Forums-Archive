### Post by chessnerd on 2009-08-10
Xubuntu might work on that, but with only 128MB RAM you'll be looking at a slower computer. Xubuntu was my first Linux OS and I like it, but on my computer with 256MB RAM it regularly used over 180MB.

You could look into Puppy Linux. It would run pretty well on 128MB RAM, however, the desktop isn't as nice as Xubuntu. If you want to use XFCE (the desktop from Xubuntu) you could try Zenwalk Linux. I was able to run the Live CD of that on a 128MB computer and it seemed okay, but it still wouldn't be as fast as Puppy.

### Post by snowpine on 2009-08-10
Is that your only computer? I would recommend learning Linux with an easy distro (like Ubuntu) on a faster computer. Once you are familiar with how Linux works, you'll be up to the challenge of installing it on ancient hardware. 

If that old clunker is your only computer, then it will have to do... Puppy is generally regarded as the easiest distro for really old hardware. Good luck!
