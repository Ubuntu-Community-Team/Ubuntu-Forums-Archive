### Post by sudodus on 2022-06-05
Please give us details about the computer and USB drive. Knowing these details will make it easier for us to help.

- Brand name and model of the computer
- Brand name and model of the graphics card/chip
- Brand name, model and size of the USB drive
- Version and flavour of Ubuntu (Desktop or Server; point version, for example 20.04.4 LTS)
- Method or tool to make the USB drive bootable with Ubuntu (for example Rufus or Ubuntu Startup Disk Creator)

Did you check that the downloaded iso file matches with the sha256sum?
