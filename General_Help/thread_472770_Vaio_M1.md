### Post by trippinnik on 2007-06-13
You could give Myth Ubuntu a shot.  MythTV has been developing media center features for quite a while (longer that Media Center Windows).  You can always try a live CD and maybe even use part of you hard disk and decide based on that.  I've used a SONY Media/PC TV deal here in japan and the interface worked pretty well without much hassle, I don't know if it's the same as what you had.  Anyway you should try before you commit to using Linux solution.
