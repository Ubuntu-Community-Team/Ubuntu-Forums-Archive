### Post by ajgreeny on 2008-08-12
Pseudo dual booting can be achieved by leaving the windows MBR on the first drive with windows, putting grub on the second drive with Ubuntu, and then using BIOS to change the boot order which is often possible with a single key press on new machines and not too difficult even on old machines where you need to go to BIOS.

However, what's wrong with grub?  I find it an extremely good boot manager, and very easy to configure, in my experience.
