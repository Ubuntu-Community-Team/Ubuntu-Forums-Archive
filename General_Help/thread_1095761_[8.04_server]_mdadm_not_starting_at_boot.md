### Post by fjgaude on 2009-03-15
Okay, and sorry for taking so long to get back to you...

I can't say if you have a mountpoint for the array. The array looks solid. So, make a mountpoint:

```
sudo mkdir /home/raid
```

You can use whatever name you wish for the point. <raid> seems a good one.

Now mount the array:

```
sudo mount /dev/md0 /home/raid
```

After that, and if it works, you can add a line at the bottome of your fstab file like so:

```
/dev/md0 /home/raid   ext3   defaults    0     2
```

That should do it. Let us know how things work out.

BTW, I know nothing about using LVM. <sigh>
