### Post by beesthorpe on 2007-06-04
Hi Zaphod- I believe this problem is caused by Feisty having 3d acceleration enabled by default .  Older apps that want to use 3d won't share nicely (Stellarium has exactly the same problem). The workround seems to be to run the game in a new x session - there's a post about how to do that in this thread: [http://ubuntuforums.org/showthread.php?t=323485&highlight=game+new+xsession]("http://ubuntuforums.org/showthread.php?t=323485&highlight=game+new+xsession")
I do hope this irritation is sorted out for Gutsy
