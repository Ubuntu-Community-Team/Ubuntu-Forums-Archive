### Post by inameiname on 2010-08-26
It was stupidly removed in Lucid. You can toggle it back and forth with Control L.

If you want a button for it, among other glorious things, you should install nautilus-elementary.

[B]ppa:am-monkeyd/nautilus-elementary-ppa
sudo apt-get update
sudo apt-get upgrade
[/B]
