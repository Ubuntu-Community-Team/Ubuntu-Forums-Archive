### Post by notwen on 2008-02-14
Sounds like you may have enabled Compiz-Fusion/Emerald and you're seeing the default emerald window decoration.  Try downloading some Emerald themes from [here]("http://www.gnome-look.org/index.php?xcontentmode=102").  You would then Open up Emerald by clicking

**System**--->**Preferences**--->**Emerald**

Once Emerald opens click the Import button(top right) and locate any theme you downloaded then select it, as you select a theme you should notice your window decorations(borders) change accordingly.  Let me know if this helps or is not your actual issue. =]
