### Post by jessdog9001 on 2010-05-30
I've encountered this same exact problem. The panel and Docky disappear, although I can still click on an item on either one and it works. I've noticed this happening mostly when I try to use a Wine application. Are you using Wine? Or some other non-native applications? I think it has something to do with that.

** Addendum **

Just noticed you said you were using Google Chrome. Chrome uses non-native window decorations. Perhaps this is the cause. You could try changing that setting and see if the panel still disappears (select Options -> Personal Stuff -> Use system title bar and borders).
