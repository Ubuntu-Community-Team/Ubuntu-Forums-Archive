### Post by Cheesemill on 2013-01-22
I have a setup on my laptop where I use full disk encryption, but I have the unencrypted /boot partition on a USB stick instead of at the start of the hard drive.

This way without the USB inserted the system is unbootable and all of my data is safe, even if someone boots from a Live CD.
