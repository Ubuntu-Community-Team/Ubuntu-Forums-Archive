### Post by collinp on 2008-12-06
Run the following command:
```
sudo apt-get install compiz compiz-fusion-plugins-main compiz-core compizconfig-settings-manager compiz-dev compiz-dev fusion-icon compizconfig-backend-gconf 
```
That installs Compiz Fusion from the repositories. Once that is done, you should find a menu entry that is labled "Compiz Fusion Icon". Click it, and you are off :). It starts automatically if you reboot too, so no need to manually start it.
