### Post by FakeOutdoorsman on 2008-03-28
Just to point out a third option, I just recently switched from Thunderbird to Claws-Mail.  It is lightweight, customizable, and fast.  I like it's speed, stability, mail file format (each message has it's own file unlike Thunderbird), bogofilter spam plugin, and mailing-list support.  I don't like it's default theme, sometimes odd word wrapping and how each message must be clicked to view. The official [Claws-Mail]("http://www.claws-mail.org/") site has several nice themes, so that got fixed easily.  The developers are friendly and receptive to feature requests.  Claws-Mail is available in the Ubuntu universe repository.
[B]
Update[/B]: I should look at the [Claws-Mail User Manual]("http://www.claws-mail.org/manual/claws-mail-manual.html") more often:
> You can then select an email using the mouse, or by using the Up and Down keys to navigate through the list, and the Space bar to display and scroll emails. You can use other keys to navigate through emails, like P and N (previous and next).
