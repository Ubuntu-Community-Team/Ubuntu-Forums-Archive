### Post by seawolf167 on 2011-05-13
I know this isn't the answer you want to hear, but it would probably be easier to simply reinstall a fresh clean version of 11.04, especially if you have a separate /home partition - then the install will only take ~15 minutes. This ensures that there are no conflicts between package versions.

If you don't have a separate /home partition you can [move it to its own new partition ]("https://help.ubuntu.com/community/Partitioning/Home/Moving")and reinstall / with the LiveCD and still have everything (just specify the mount points for / and /home when installing)

### Post by ivanovnegro on 2011-05-13
All you have to do is to get somehow into the terminal and type:

```
unity --reset
```

to get to the default settings of the Unity desktop. It seems to be a common issue when playing with ccsm settings to screw up Unity and even the Classic desktop especially using the cube.
