I just installed Edgy (formerly an FC4 user) last week, and it's been going well (minor nits blogged at [http://silmaril.ie/cgi-bin/blog)](http://silmaril.ie/cgi-bin/blog)).

I managed to get CUPS to recognise my HPDJ 1220C on the parallel port, and it prints from GUI apps. But neither lp nor lpr can print (dialog below). I think FC had some kind of lp-for-cups-compat RPM...is there an equiv for Ubuntu? Is it possible to use lp/lpr or do I have to open an app to print a file like Windows users do?

### Post by frisket on 2006-11-18
Fixed (but ungainly).
Turns out that if /etc/printcap says 
hpdj|DeskJet-1220C
instead of
DeskJet-1220C|hpdj
then lpr recognises the printer (ie lpr only recognises the first token before the pipe). This looks like a bug ](*,) 
///Peter
