### Post by pbb on 2006-02-01
x-desktop does not need to be installed, you can just download the file, unzip it and open "index.html" in your browser. Or launch the demo from their own website.

Note, x-desktop is not comparable to Gnome or KDE, it is a web-based desktop system. I haven't seen applications that run on this desktop system.
