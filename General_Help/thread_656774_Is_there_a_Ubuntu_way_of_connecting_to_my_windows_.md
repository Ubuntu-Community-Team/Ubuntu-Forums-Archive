### Post by andymushu on 2008-01-04
take a look at multisync [http://multisync.sourceforge.net/news.php](http://multisync.sourceforge.net/news.php). if you do use multisync, you will need the synCE plugin for it. Note this will not work for windows mobile 6, as of yet. And activesync under wine is not the way to go. It seems to install fine, but ultimately doesn't detect the usb device and fails. so, i would recommend [[COLOR="Lime"]multisync[/COLOR]]("APT:multisync")(click this link to install), along with the synCE plugin [[COLOR="Lime"]synce multisync plugin[/COLOR]]("APT:synce-multisync-plugin")(again, click link to install).
