You can find the options for your display under System>Preferences>Display (Monitors if you happen to be using Lucid). There you can change the resolution. ### Post by 3rdalbum on 2010-04-05
There may be an Ubuntu forum available in your native language; you might get better help there.

Your screen problem: It sounds like Ubuntu is using a lower resolution than your monitor. If you have an Nvidia or ATI graphics card, have you enabled the graphics driver from the "Hardware Drivers" program in the System menu? If you don't have one of those graphics cards, try going to the Monitors panel in the System menu and making the resolution higher.
