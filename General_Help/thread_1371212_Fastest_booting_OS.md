### Post by northrup on 2010-01-03
Chrome OS is basically a modified Linux distribution.  It even has APT, if I recall correctly.  I don't know the keystrokes, but there are ways to get into a terminal, too.  From there, perhaps install Xfce?  However, that might also take longer for it to load, and Chrome OS wasn't designed to run software other than Chrome.

If you want to combine fast booting and good looks, I'd look into an ultra-small Linux distribution (like Damn Small Linux) or one that's low-power and command-line and then manually install a better desktop environment, such as Xfce or Gnome (KDE if you're willing to sacrifice your RAM), plus any other packages and drivers you need.

Basically, I would just make my own personalised distribution.  I think there is a website tailored toward doing just that.
