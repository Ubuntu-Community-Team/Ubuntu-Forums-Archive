### Post by popeyelin on 2008-04-10
Although I know how to make a pen-drive linux, I still wish Canonical can sell the pen-drive ubuntu with a 4 Giga bytes drive.
As far as I know, Mandriva is selling that the price is 59 Euros. If Canonical can have this one I will buy it if its price is under 80 USD:guitar:
The flash drive must have the following features:
I. A full featured Ubuntu 8.04 including the most recent drivers especially the graphic card drivers: Nvidia,ATI,Intel. It can automatically recognize the hardware and load the modules correctly and automatically.
2. It must have at least 3 giga bytes as a writable spaces to copy and exchange data because the most important usage of this flash drive is to exchange data and recover data from any crash. 
3. It must have the ability to read most file systems in the market including fat16,fat32,ext2(3even4),reiserfs,ntfs,jsf,xfs and so on. It can recognize the harddrives and mount them automatically.
4. It must have the ability to update itself using apt-get to keep the drive always up-to-dated.
5. Since it's writable, it can allow users to install more softwares if they want especially the w32codecs.
6. If users think necessary, it can allow user to make a exact copy of the current flash drive's status. And user may use another tool to restore the flash drive to any time status. The idea looks like Apple's Time Capsule.:)
7. Can Canonical talk to Apple to develop Ubuntu iTunes and port iPhone's SDK to Ubuntu? I really love my iphone and don't want to buy a macbook for now. If the iTunes and iPhone SDK are available,I will be even happy to buy it. Of course, if Canonical can negotiate with RIM to port all RIM's tool including the management tool and SDK to Ubuntun and included in this flash drive, it's the best thing for me.
8. User may choose to clone the flash drive to its PC when he thinks it's good enough and configured well enough. All configurations should be the same except the clone process will detect the difference of the harddrives and mount points and adjust a little.
9. Of course, the speed of the flash drive is every important. The live cd is good but is too slow. So please choose a fast flash drive provider to to this cool stuff if you think it's good to do.
