### Post by jmszr on 2009-08-15
Assim,

This should help: [https://wiki.ubuntu.com/Fonts](https://wiki.ubuntu.com/Fonts)

The package "ubuntu restricted extras" will install the Microsoft fonts and can be installed via Applications > Add/Remove > Show: All available applications > Search: ubuntu restricted extras > Click the box > Apply changes. 

Or Type (or copy and paste)in the Terminal (Applications > Accessories > Terminal):```
sudo apt-get install ubuntu-restricted-extras
```
type in your password when it asks (will not be visible-security) and press Enter.

### Post by Herman on 2009-08-15
Try this link for more, [300+ Easily Installed Free Fonts for Ubuntu]("http://ubuntu.wordpress.com/2007/05/21/300-easily-installed-free-fonts-for-ubuntu/")

Since you're interested in fonts, I also recommend you install 'specimen font viewer', that's a program for browsing all of your installed fonts so you can pick out which ones you like for whatever purpose you have in mind.

There's another program called 'font forge' in case you want to make your own fonts too.
