### Post by Teamgeist on 2006-11-15
To be honest: I have no clue where the movie files are.

But I recommend giving k9copy a shot.

```
sudo apt-get install k9copy
```

It will install a couple of kde-libs but that doesn't matter.
If it does not appear in the menu just type ```
k9copy
``` in the command line and the programm will start.

It works just like DVDshrink in Windows does.
