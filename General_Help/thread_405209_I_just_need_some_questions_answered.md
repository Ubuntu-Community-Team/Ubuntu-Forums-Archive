### Post by mmmpancakes on 2007-04-09
There a re a number of reasons to switch to Ubuntu. What do you primarily use your computer for? Odds are Linux is able to provide you with a solution. 

There a re a number of ways to install Linux. You can even install it while leaving Windows ME on the computer, and decide at startup which OS you want to log in to. This is called dual booting. 

You also have the option to wipe the drive entirely and install Ubuntu across the whole partition.
