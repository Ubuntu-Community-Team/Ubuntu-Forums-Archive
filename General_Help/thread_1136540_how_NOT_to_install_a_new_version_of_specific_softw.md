There is a program which I do not want to update. Is it possible to turn off the update for that program, while keeping the old version?

### Post by zvacet on 2009-04-25
In synaptic under package tab select **lock version**.

---

### Post by Macamba on 2009-04-25
Thanks.

(And for those stumbling upon this answer and not knowing how to progress: System > Administration > Synaptic Package Manager. Select the programs you want to lock, followed by Menu: Package -> Lock Version)
