### Post by amingv on 2011-09-17
Neither IDM nor JDownloader can accelerate the speed at which your OS downloads updates since (both in Windows and Ubuntu) updates are downloaded by dedicated update management software that does not use any sort of plugins or addons.

In the case of Ubuntu, you can choose which update server you'll download from. If you change it from the Main Server to some other mirror with a lesser load you might (or might not) get a better download speed, other than that it's all in your Internet connection.

EDIT: Oh, and if you're looking for a good download manager/accelerator (for browser downloads) I kindly suggest DownThemAll for Firefox. Best download manager ever IMHO.
