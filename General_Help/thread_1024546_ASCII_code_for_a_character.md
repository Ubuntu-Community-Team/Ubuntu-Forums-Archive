### Post by ajcham on 2008-12-29
The character you describe is not part of the ASCII set, therefore does not have an ASCII code. (ASCII only has 128 characters, 95 of which are printable)

The various Unicode references for the character are given in your OP:

> U+1D41
UTF-8: 0xE1 0xB5 0x81
UTF-16: 0x1D41
C octal escaped UTF-8: \341\265\201
