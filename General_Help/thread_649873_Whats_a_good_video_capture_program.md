### Post by eggdeng on 2007-12-25
Most people use recordmydesktop for this sort of thing - its in the repos. ```
sudo apt-get install gtk-recordmydesktop
``` You can capture directly with no need for a camera - the only snag is that videos are in .ogg so you need to convert if you want them on youtube.
If you want to use a dv camera and then capture through a firewire port, kino ```
sudo apt-get install kino
``` is excellent once you get it set up.
