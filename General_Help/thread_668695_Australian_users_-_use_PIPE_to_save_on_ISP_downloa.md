### Post by dcstar on 2008-01-15
A lot of Australian ISPs have free peering with the PIPE network, which means any data downloaded from this source can be zero cost for a lot of people.

There are mirrors of Linux repositories available on PIPE, so if you set up your Linux system correctly all software downloads and updates will not be counted on your monthly quota.

The two on PIPE that I have found are Internode and Netspace, and you can select these in Ubuntu Linux by going into Synaptic-Settings-Repositories-Dow* nload from-Other-Australia.

I have checked in my own account that the Internode settings count as free PIPE downloads, the others will be at your own risk!

[http://mirror.internode.on.net/pub/ubuntu/](http://mirror.internode.on.net/pub/ubuntu/)

Using these can save you a lot of quota - especially if you want to download an ISO.
