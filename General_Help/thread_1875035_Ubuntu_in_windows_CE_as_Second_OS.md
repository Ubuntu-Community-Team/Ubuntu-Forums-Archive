Welcome to the forums!

Unfortunately your computer falls short of the [minimum Ubuntu system requirements]("https://help.ubuntu.com/community/Installation/SystemRequirements").

There is a ton of info about these devices [in this discussion thread]("http://ubuntuforums.org/showthread.php?t=1349626"). The general consensus is that installing Linux is not a beginner project.
