### Post by Aurels on 2005-09-21
if you use gnome, run nautilus and right click an mp3 file, select "properties", go to the "open with" tab. There you can choose which app will open this file type.
You'll have to do this for each kind of audio file.

### Post by Borusa on 2005-09-21
In the file manager, there is (I believe) a Command menu. In that, is an edit mime types option. This will bring up a thingy that allows you to change which files are associated with which file types. If you switch the ones pointing at rhythmbox with xmms, it should work.

Easier than on a file by file basis.

And yes, that should work with streamed audio, assuming xmms can play the file.
