### Post by jcd29 on 2010-10-13
I'm using Parcellite right now since whenever I copy something from Firefox and close it before pasting, whatever I copied is gone. Parcellite fixes this, but I don't want it to be on my tray all the time. How can I hide it?

---

### Post by WorMzy on 2010-10-13
Right-click it and choose "Quit". Then press Alt+F2 to bring up the run dialogue, and run "parcellite -n".

You can still access all the functions of parcellite with hotkeys, whether the icon shows or not. Ctrl+Alt+P should bring up the preferences context menu. Ctrl+Alt+A should bring up the actions context menu, and Ctrl+Alt+C should bring up the clipboard context menu.
