### Post by mkvnmtr on 2010-03-14
In virtual box I have a Ubuntu minimal install of XFCE4 that I gave 300Mb of ram. It runs great and would run on 256. It is pretty simple to install. From the command line install I install xorg, xfce4 and synaptic. Then I startx and have a desktop and package manager to use to install what I want. I keep ram use down by not installing gdm to log in. I just use the command line and type startx each time.The system does everything I want and I think when I go to 10.04 I will install the same thing for my main system.
