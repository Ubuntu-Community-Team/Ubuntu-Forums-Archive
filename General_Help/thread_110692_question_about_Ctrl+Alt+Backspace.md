hi, the other day i had to reboot to get my fonts working again after uninstalling kcontrol.

is Ctrl+Alt+Backspace the same as rebooting in this instance? does everything get reloaded? thanks

---

### Post by az on 2005-12-31
CTRL-ALT-Backspace restarts the X server.  Reloading fonts is an X server function.

Booting runs you through all the init scripts which start services and initialise hardware.  Starting the X server is one of those scritps.  So restarting the Xserver is not like rebooting.  It is just restarting one of the services that is running on your machine.
