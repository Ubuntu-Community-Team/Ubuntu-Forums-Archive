### Post by willie_wang on 2007-12-20
ok. i've already had this problem and fixed it. all you need to do is add a line in your xorg.conf in your device section:

```
	Option		"XAANoOffscreenPixmaps"	"true"
```

so as an example, my device section in xorg.conf looks like this.

```
Section "Device"
	Identifier	"aticonfig-Device[0]"
	Boardname	"ati"
	Option		"XAANoOffscreenPixmaps"	"true"
	Busid		"PCI:1:0:0"
	Driver		"fglrx"
	Screen	0
EndSection
```

it fixed my problem, hope it fixes yours... btw, it took me hours and hours of searching to find this solution. i've filed a bug report too. can you comment it to confirm youit too please? [https://bugs.launchpad.net/ubuntu/+bug/177433](https://bugs.launchpad.net/ubuntu/+bug/177433)

let me know if this works for any of you!
