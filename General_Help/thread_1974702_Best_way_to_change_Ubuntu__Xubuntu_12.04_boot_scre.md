### Post by bogan on 2012-05-07
Hi!, **doctertonic**,

Go to Grub Customizer>Preferences>Appearance and browse for the background image you want for the grub menu and boot screen.

 It needs to be a .png file and there are size restrictions. Make sure you save afterwards to update the config files!
 
There is a HOW TO: Grub Customizer Thread in the Tutorials & Tips Forum, if you need further info:
[http://ubuntuforums.org/showthread.php?t=1664134&page=27](http://ubuntuforums.org/showthread.php?t=1664134&page=27)

Chao!, **bogan**.
