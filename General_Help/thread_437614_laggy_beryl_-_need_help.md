### Post by aidanr on 2007-05-09
open a terminal (applications -> accessories -> terminal)
backup /etc/X11/xorg.conf
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```
and run through 
```
sudo dpkg-reconfigure xserver-xorg
```
selecting the appropriate resolution when prompted, also go into beryl settings and turn off detect refresh rate and put the refresh rate up to 200, and then restart X (ctrl+alt+backspace)

### Post by aidanr on 2007-05-09
to view the contents of xorg.conf type in ```
gedit /etc/X11/xorg.conf
```
add .backup to the end to see the contents of the old one, and gksudo to the beginning of the command if you want to be able to save the file (gksudo gedit /etc/X.....)
