### Post by buzzingrobot on 2012-06-04
Try the browsers available on Ubuntu and decide which one you like. Firefox, Chrome and Opera are all very good browsers. Many people use a lot of browser extensions and addons.  They're fine, but none of them are essential.

If you use any Windows-only software in your business, you need to determine if Linux software is available that provides the same functionality.  If, in fact, you do use Office on Windows, LibreOffice is the Office clone on Linux. It is very compatible with Office. Whether or not it is compatible enough for you depends on how you use Office.

If you are able to set up Ubuntu as a dual-boot situation on your Windows machine (look for info on "wubi"), you could try out Ubuntu with minimal risk.
