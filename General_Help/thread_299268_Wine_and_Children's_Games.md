The problem is that most of those games use a different and proprietary way of running hardware acceleration - basically, those games are made for windows and running them through an emulator will be difficult.

If it is feasable, I think you will have much more success running windows98 on a few of those classroom boxes for the games and running Edubuntu on the others.  I would not put windows98 on the internet, though.  Use only the Edubuntu computers for internet access.

I think that you will end up frustrated if you try to migrate all the boxes to Ubuntu at once.  Even if you get most of the games to run, someone will one day bring in a game they got from a cereal box and try to get it to run and break everything.

As stated, Edubuntu ships with a great variety of wonderful games for kids.  My two girls (2 and 4 years old) have learned more about using the computer from those games than any other game we have seen that is made for windows.
