### Post by feelshift on 2009-04-10
You can't install windows programs (.exe) in Linux (Ubuntu). In Ubuntu you can use Synaptic to install programs from repositories (System -> Administration -> Synaptic Package Manager). Just search a program, select it to install and click 'Apply'.
