### Post by RAOF on 2006-03-13
It's probably the case that the software you installed depends on WMP.  Install it (through wine), see if your software works then :)

I believe you should generally be able to run the installer a second time and select "uninstall" to uninstall software through wine, but since there's no control-panel in wine there's no way to get to the standard way to remove windows programs.

Incidentally, this is one of the reasons wine is not installed by default - it's horrible to support :)
