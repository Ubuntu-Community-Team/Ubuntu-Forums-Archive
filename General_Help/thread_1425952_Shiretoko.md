### Post by lovinglinux on 2010-03-09
Disable the *ubuntu-mozilla-daily* ppa in your Software Sources, then remove Firefox and install it again. This should revert it to the default version. Then add Firefox 3.6 using one of the methods described [here](http://lovinglinux.megabyet.net/?page_id=220#Installation-Methods-2).

Your bookmarks and the rest of your profile are probably stored under ~/.mozilla/firefox-3.5. Just move the contents of this folder to ~/.mozilla/firefox. Make backups first.
