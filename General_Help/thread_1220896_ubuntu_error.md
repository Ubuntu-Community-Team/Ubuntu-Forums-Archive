i've had ubuntu on my desktop for a while and it's been working but all of a sudden this morning i boot it up and it says "Could not find kernel image:linux" and i'm at a "Boot" prompt. i've tried looking this up and most people that's had this problem has had it booting from a live cd or something like that but mine was already installed. any help would be appreciated. thanks in advance.

---

### Post by hotshot247 on 2009-07-23
i figured out this message. i had my usb flash drive plugged in and it was trying to boot from that like a HDD so i just unplugged it and it worked. if anybody else has this problem, it can also be that another hdd is set to boot first or if you try to boot from a live cd, the files could either be corrupt or missing.
