### Post by Barafu Albino Cheetah on 2010-06-08
64 bit installations of modern (spring 2010) linux distro's are all well made, except for openSUSE maybe. There ARE some minor problems, but they all have working easy-to-find fixes. 
So the only question is if you really need 64bit installation. On desktop machine for everyday use it is both slower, and consumes up to 30% more memory. If you are going to do a lot of CPU loads (math, compilation, multimedia transcoding, archive cracking/packing, or other tasks that usually show "Time to completion: 3 days 17 hours" progress bars, then you will benefit of 64 bit.
