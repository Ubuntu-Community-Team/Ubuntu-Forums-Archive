Yes, Ubuntu is an ideal OS. On one side it is Linux and has all the security things included (no viruses, so no anti-virus program needed, you using your computer as a user not as an administrator so you can not harm your computer by accident), on the other side it is friendly system (it has some nice desktop managers to choose from, most of the things you need (want) to do can be done with a gui). If you do need to do something technical there is always the terminal with the code: man <instruction> to show you the manual of how to use this instruction. And of course there is this forum where people from all over the world are trying and willing to help.

OpenOffice.org is almost 100% compatible with MS Office, even the 2007 version. Sometimes formatting texts is not done 100%, but this is easy to restore.
You can save your documents with the MS Office extensions and in those formats so others, who still use MS Office, can also read them.

Listening to music can be done in Ubuntu very well, there are a couple of mediplayers available, including those who can work with the Ipods.
IM programs are here and can join the most popular networks.

One big advantage is: the OS and many programs are free including free of charge. Being still at school this might sound interesting as well. You don't need to buy the programs you want (or need) to use, you simply download them.

When you have re-installed Windows a couple of times already you have the source to do so. You keep that on your D-drive. Make sure when installing Ubuntu this drive (or partition) is not ruined. Use your C-drive to install Ubuntu.

What would be a good idea is to try Ubuntu from the live CD you will download. Burn the downloaded ISO file to a CD and boor the computer from this CD. Make sure you burn the ISO file the right way, not just make a copy but burn it as an image so it will be extracted while burning it. This creates a bootable disc.
When booting choose for live CD and look around. This way nothing is done to your harddisk. Dis-advantage is the speed which is slow this way but that is caused by the speed of your cd player.

Have fun.

> My drive D contains all the recovery files for my windows XP, so installing Ubuntu would allow me if at anytime to do a system restore to my factory settings XP right?

Not necessarily. Not all recovery partitions work like that. Your best bet would be to back up any data you might want in the future before you install Ubuntu. One small mistake during the install process could wipe all your partitions and delete any backup files.
