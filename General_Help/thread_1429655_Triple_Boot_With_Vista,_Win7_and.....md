### Post by Mark Phelps on 2010-03-15
Just make sure when you install Ubuntu that you first use the Vista/Win7 Disk Management utility to shrink the appropriate partition to make room for Ubuntu.  Don't allow the Ubuntu installer to do the shrinking.
