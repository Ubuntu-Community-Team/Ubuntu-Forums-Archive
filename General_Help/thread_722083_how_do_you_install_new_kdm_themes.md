### Post by undecidable on 2008-03-12
1.  nb: background,color,style,icons,fonts,screensavers, panel positions are all part of theme,
so make sure you make any changes to these after you change the theme or they will be overwritten.

2.  download a theme file, which will end in .kth, from, eg, kde-look.org.

3.  Open  Settings/Appearance & Themes/Theme Manager

4.  Click "Install New Theme"

5.  then navigate to the directory in which you downloaded the .kth file,
click on it, and it will be installed.  

3,4.  Alternate:
there are some kde theme packages available with Adept, eg kde-artwork
After you select and install these, then new themes will be available in the left hand side of the Settings/Appearance & Themes/Theme Manager window.  Just click on one.  

6.  After you have made whatever changes you want to make to your theme, save it with:  
 "Create new Theme" in the Settings/Appearance & Themes/Theme Manager box.
