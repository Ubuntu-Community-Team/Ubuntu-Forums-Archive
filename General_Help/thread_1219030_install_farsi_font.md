### Post by fragos on 2009-07-21
There are a number of sources for Farsi TTF fonts. To install, create a folder in your home folder called ".fonts" and place the TTF fonts there -- example /home/user/.fonts. Next run "sudo fc-cache -fv" in a terminal window.
