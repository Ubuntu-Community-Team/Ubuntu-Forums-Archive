### Post by reckless2k2 on 2007-05-27
First of all make sure you have printer and file sharing enabled on your XP machine. Go into the firewall exceptions and make sure that box is checked. From the you'd ideally like them to be in the same WORKGROUP. 

Follow instructions here:

[https://help.ubuntu.com/community/SettingUpSamba](https://help.ubuntu.com/community/SettingUpSamba)

You can use a thing called SWAT which makes it a little easier to setup. You can apt-get or synaptic SWAT as well. Query SWAT in this forum if you'd like.
