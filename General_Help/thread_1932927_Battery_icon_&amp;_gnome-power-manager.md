Lucid 10.04.4 w/ updates, stock Gnome 2.*

After recent updates, my battery icon is no longer displayed in the upper taskbar notification area. When I go to System>Administration, gnome-power-manager is not listed on the menu, but when I enter gnome-power-manager in Terminal I get a message saying that it is already running.

I changed "Show icon only when charging or discharging" to "Always show icon". Apparently one of the updates changed my preferred preference from "Always show icon".
