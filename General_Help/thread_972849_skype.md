### Post by fsmithred on 2008-11-06
You can install software while you're running a live cd. It won't persist after a reboot, but you should be able to test it. (I'm pretty sure I've done that with Skype for a friend.)

Before you make any permanent changes, ask yourself why you want to upgrade from a long-term support version to a newer version on your work machine. If it does the job, leave it alone.
