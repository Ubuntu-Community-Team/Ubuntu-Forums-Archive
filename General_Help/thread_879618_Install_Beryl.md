### Post by armageddon08 on 2008-08-04
You don't have to do much. Goto terminal and type: 
```
sudo apt-get install compizconfig-settings-manager fusion-icon
```
Then goto System-->Preferences-->Advanced Desktop Effects Settings, configure the settings and enjoy!!
To start CF you will need to click Compiz Fusion Icon which can be found in Applications-->System Tools.
If you are using 'blacklisted' ATI/Radeon graphics cards, this thread may be useful:
[http://ubuntuforums.org/showthread.php?t=764633]("http://ubuntuforums.org/showthread.php?t=764633")
