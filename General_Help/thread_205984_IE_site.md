### Post by thasheep on 2006-06-29
try the firefox [User Agent Switcher](https://addons.mozilla.org/firefox/59/). A user agent is a short string that tells a site what browser and operating system you are using. Install the plugin, restart firefox and then go Tools>User Agent Switcher>Internet Explorer 6 (Windows XP). Now when you visit a website, it'll think you're using ie on windows xp. No wine needed.
Oh yeah, I got to the login page without any trouble doing this.
