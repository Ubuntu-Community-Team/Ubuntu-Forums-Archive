### Post by Patrick70 on 2010-10-17
For some time I've been searching for some kind of equivalent to Mailwasher for Ubuntu, but never really found anything that came close.
I did once find something called Kshowmail, but that project had been halted and it did not work anymore on Ubuntu.
In the end I installed Wine and had Mailwasher running through that. It did the job, but it was not perfect since it apparently did not notify (no sound nor any other notification) when new mail arrived, so I still had to keep checking whether there was any mail or not.

Until yesterday, when I stumbled upon a solution. I found that the Kshowmail script which I told about before, actually has a new version, albeit for KDE. However, it does work perfectly fine on Ubuntu and Gnome (I have 10.10 installed, but I read it works fine on 10.40 as well).
It's exactly what I was looking for, so I have now dumped Wine and Mailwasher again for Kshowmail.

I know that many people have inquired about this in the past, so I thought I'd share this in case other wish to use it as well.

This where I got it from: [https://launchpad.net/~ferramroberto/+archive/testing/+build/1893023]("https://launchpad.net/%7Eferramroberto/+archive/testing/+build/1893023")
