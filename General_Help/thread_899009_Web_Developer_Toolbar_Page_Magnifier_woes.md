### Post by mb_webguy on 2008-08-24
While I'm a regular user of Web Developer, I've never actually noticed the page zoom tool.  I use the Resize Window feature quite often, but for some reason have never noticed Zoom just below it.  If I want to zoom in or out of the entire page, I use FF3's built-in zoom.  Of course, this wasn't available, I don't think, in earlier versions of FF.  But if I just want to zoom in on a particular graphical element, I use the Image Zoom extension (which allows you to zoom in or out of any image using the scroll-wheel while holding right-click).
