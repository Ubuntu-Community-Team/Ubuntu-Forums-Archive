### Post by tjwoosta on 2011-02-23
Not a live cd, you could create a customized live cd of any distrobution rather easily. LFS is something completely different, you create the entire distrobution from the ground up.

LFS is a book that teaches you how to get the source files for every single piece of software involved and compile and configure everything yourself until you have a complete and custom home made linux distrobution, with no affiliation to any other distrobution.

After your done theres nothing stopping you from creating a live cd of that custom system if you wanted, but the actual creation process is done on a hard disk. In the beginning you would use a live cd environment to mount the hard disk and start the process of building the distrobution. After you get the foundatation of the system laid you boot directly into the hard disk.

Its mostly a learning tool to teach people the inner workings of linux, and all how all the individual pieces of software come together to make a distrobution.
