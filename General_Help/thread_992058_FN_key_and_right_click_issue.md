> **clysher said:**
> The first, and least annoying of the two, is that whenever I adjust brightness using the FN and arrow keys while on battery power it also turns down my volume as if I were hitting the FN and Pageup or pagedown keys.

That sounds like you've got Fn-Arrow mapped to both functions. I believe you can set shortcut combinations in System -> Preferences -> Keyboard Shortcuts. Make sure that you haven't got Fn-Arrow listed as the combination for volume up and volume down.

That's all I can think of.
