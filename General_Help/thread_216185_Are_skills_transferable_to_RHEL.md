### Post by T700 on 2006-07-15
This is just a subjective guess, but I'd say 80-90%.  Some things will be in a different place and lots of graphical items will be different, but the command line is pretty much the command line--at least as far as the Linux OS commands go.  If you've become comfortable with Ubuntu, you certainly have the skills and patience to learn the others.

### Post by T700 on 2006-07-16
> **beausimon said:**
> Thanks. So using the command-line would be more portable than using the GUI?

In my opinion, yes.  The basic command set remains the same, because at the heart of every distro is Linux.  The frontend--the GUI--is changed by most every distro.  That said, once you're familiar with the concepts, adjusting to a new GUI isn't too bad.  Sort of like buying a new car and having to relearn how to turn on the wipers.
