### Post by ajgreeny on 2009-04-11
wubi is the **windows** ubuntu installer (note "windows").  It will only install ubuntu as if it were a program within a **windows** OS, and can not put it separately on a flash drive, as far as I'm aware.  If you want ubuntu on a flash drive it can certainly be done, but not using wubi.
