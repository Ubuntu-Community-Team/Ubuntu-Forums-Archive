### Post by seawolf167 on 2011-04-12
You can find out what is taking up space with *du*

example - to see all folders & files in human-readable in your current directory (i.e. /home by default), use

```
du -h
```

there are a ton more options if you want to get more specific

```
man du
```

you can also use *df* to see the space used/available

```
man df
```
