### Post by grahammechanical on 2016-08-20
If I remember correctly we have to select an option by ticking a box. Once we select one of the five options we should be able to click Continue. Can you not do that?

If you wish to over-write the existing OS, then the option Erase disk and install Xubuntu should be sufficient. If we are using the Something Else option we get to the partition layout screen and there we need to select each partition in turn and click Change. That will open a dialog box where we select Ext4 in the "Use as" panel; we tick to format; and we select a mount point. For the OS the mount point is /. Then we can click OK and when back at the partition layout screen we can click Install.
