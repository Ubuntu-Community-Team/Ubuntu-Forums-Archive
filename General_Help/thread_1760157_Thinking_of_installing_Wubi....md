### Post by sanderd17 on 2011-05-16
Wubi is perfect for what I call "a long testing period". If you want to test Ubuntu for 1 up to 6 months, Wubi is perfect. I also used Wubi for some months.

Wubi is not perfect for a permanent install though. This is because the stability of wubi depends on windows and if you use Ubuntu too much, and don't update windows, then windows will become unstable.

If you want to try Ubuntu, try it good. There are no limits.

As a famous quote goes: Linux gives you enough rope to hang yourself. In other words: you can do everything with Linux, but it won't protect you from your own mistakes.
