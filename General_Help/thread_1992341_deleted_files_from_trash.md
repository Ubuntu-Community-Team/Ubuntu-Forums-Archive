Do not use the disk as you have rightly done. Use testdisk which is in ubuntu's repo. This can be installed in a live cd or usb session. Part of testdisk os photorec which can recover all sorts of files.

[http://www.cgsecurity.org/wiki/TestDisk](http://www.cgsecurity.org/wiki/TestDisk)

Read up before you use it. You need a target partition to recover the files to. This must not be the same partition that contains the deleted files.
