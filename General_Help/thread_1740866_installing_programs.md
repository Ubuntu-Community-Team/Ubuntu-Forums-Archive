### Post by Mark Phelps on 2011-04-28
Perhaps I'm misunderstanding what you're trying to do ... but ... Linux is NOT Windows.  You can't just stick in Windows CDs and install them.

There is Wine, a set of libraries that allows you to run SOME Windows apps in Linux.  But, you should go to the link below first and do searches on the Windows apps and versions you want to run:

[http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true")

If there is no rating, or if the rating is not at least Gold, you're wasting your time.
