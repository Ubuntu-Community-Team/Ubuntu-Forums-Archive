### Post by mac.ryan on 2007-03-13
Most of the applications are installed in ubuntu via the System / Administration / Synaptic Package Manager menu (not downloading them from the software website).

This is the case for Banshee, for example. The procedure is as follow:
[LIST=1]
[*]Click on "System --> Admin --> Synaptics"
[*]Insert your password when prompted to do so
[*]Setting --> Repositories (select the ones you are interested in, this is a one-time operation, make sure you have "universe" enabled as this is where banshee is)
[*]Click on "search" and type "banshee"
[*]Click on the square next to "banshee" in the search results, and select "mark for installation"
[*]Select "mark" to the other packages banshee need to have to properly run.
[*]Click "apply"
[*]Wait for the installation to be finished
[*]ENJOY! :)[/LIST]This procedure is the same for the very most of the programmes prepared for ubuntu. If you need codecs for MP3 and other stuff, you can get them via Automatix2 ([www.getautomatix.com)](www.getautomatix.com)).

Best luck!
