### Post by Impavidus on 2018-04-26
The only difference between Ubuntu and Lubuntu (and the other flavours) is the set of default software, most notably the desktop environment. Everything that works or can be installed on one flavour, can also be installed on the other flavours, and you'll automatically get the same version.

4GB ram should be enough to run any flavour, although a lighter flavour could help a bit.
