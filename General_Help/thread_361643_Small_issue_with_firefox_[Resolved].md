### Post by billdotson on 2007-02-14
In Windows when I use firefox if I click in the search bar or the URL bar it automatically highlights all the data in there.  Since I have a text file where I copy and paste the links to all my forum posts it wastes alot of time when I have to scroll to the end or the beginning and highlight it manually.

Is there a simple preference I change to fix this?

---

### Post by towsonu2003 on 2007-02-14
to the url bar, type about:config

than use "click" as the filter text and look for browser.urlbar.clickSelectsAll and set it to true (double click on it)

hope this helps :)
