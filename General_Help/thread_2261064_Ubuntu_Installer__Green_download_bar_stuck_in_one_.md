### Post by Gordonbp531 on 2015-01-16
WUBI is not recommended any more - it's not being supported, and if you are trying it on a Windows 8 machine then AFAIK it won't work at all.
If you *really* want to run Ubuntu within Windows then install a Virtual Machine program and run it inside that. 
Given the security status of Windows, i would either dual-boot, or run Windows within a VM in Ubuntu!

### Post by Impavidus on 2015-01-16
Don't extract the .iso file and don't run wubi. Burn the .iso file to a dvd or usb: [https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick), [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto). Then boot from it.
