### Post by rturner on 2006-12-29
I like nano for ease of use.  Vi is much more powerful, but I always forgot the commands.  Then I learned vim, which is supposed to be an improved version of vi.  I know that if you install the full version in synaptic, it comes with a great command line tutorial.  If you actually work the commands in the short tutorial, you will never forget them.  Hard to believe, but trust me, it works.   Once you have vim-full installed, just type vimtutor and it starts.
