### Post by XubuRoxMySox on 2011-07-28
Hello,

It's possible (and best) to upgrade using a LiveCD if you are going from one version to a higher one (say, from 10.04 Lucid to 11.04 Natty). A fresh install is always better than an upgrade IMO.

I think I would try out some games on Wine or PlayOnLinux first before "going back" to Windows, and if they work well, great. I'm told that Final Fantasy XI plays beautifully on Wine, for example.

If you decide you need to dual-boot, it's best to install Xubuntu *after* Windows.

Hope that helps!
