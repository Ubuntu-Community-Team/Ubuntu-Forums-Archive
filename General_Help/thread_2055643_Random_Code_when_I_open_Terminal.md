### Post by mevun on 2012-09-10
If it's consistently similar, then you probably have some script that runs when you open Terminal.

I'm too new to Ubuntu to know if there are different Terminal programs, but typically they will run a "shell" and that shell will execute one or more "dot" files.  For example, a csh (C-shell) will run a file called .cshrc in the user's home directory.  A bash shell will run .bashrc and maybe .profile .  Some shell programs also execute files that are global to all users, e.g. "/etc/profile".

The random text you show looks similar to the output of the "netstat" command, but I think there are several other commands that do similar things, so it could be another program.
