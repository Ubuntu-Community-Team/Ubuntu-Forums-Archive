### Post by mikewhatever on 2007-11-21
Ubuntu is a different operating system, unlike XP vs windows 2000. Vista programs do not work here, and exe files are useless. To get programs, go to Applications>Accessories>Add/Remove and install whatever you want. At least VLC player is there.

You do not need to install any drivers, especially none made for Vista.

Whatever you do next, please, ask first. Do not expect.

### Post by Sef on 2007-11-21
> 
1) how to i get things to work

Programs are listed under Applications.  You cannot run Window programs on Ubuntu or any GNU/Linux.   Yes, it is a learning curve, but not hard if you have patience and ask questions. 

 > 2) and once i get them to work how do i get various programs on to my applications list so i wont have to keep going to the computer window to run them from the other partition

If you want to run Window programs on Ubuntu or any GNU/Linux operating system, then you will need to install WINE or Crossover Office.   The former is no cost, while the latter does cost money, but installs and is easier to use (at least initially.)

 > 3) finally do i need to install all the driver files for my graphics card, motherboard..etc all over again?

They should all have installed up on install.   If you have a problem, then post what your problem is.   Best to do one problem per thread.
