### Post by Sutekh on 2006-05-26
Somehow that particular setup must have been saved using the Sessions manager.

Go to the System -> Administration Menu and select Sessions.

Try seeing if there is a Session saved under the Session Options tab.  You might want to remove it.  You could also uncheck Automatically save changes to session, so your setup is not saved when you quit Ubuntu.

Otherwise check that nautilus and the terminal don't appear in the Startup Programs tab.
