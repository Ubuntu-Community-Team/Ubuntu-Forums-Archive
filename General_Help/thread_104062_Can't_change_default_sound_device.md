### Post by Jormundgand on 2005-12-15
System > Preferences > Sound has a box allowing you to change the default sound device. By default on my system it's set to VIA 8235, which corresponds to my main speakers. I've been unable to get these to work with ESD, ALSA or OSS, but that's not a problem since I also have my Logitech USB headset.

So I change the setting in the Sound window and close it. However, when I open the Sound window again, the setting has not been saved and the box again lists "VIA 8235" as the default device! Can anyone assist?

### Post by hajk on 2005-12-16
Do you have a second sound card that you want to switch to? 

If you just want to switch between OSS, ALSA or ESD then you should click on the little loudspeaker on the upper right toolbar (or open the Volume Control in Applications/Sound & Video), then under File you can change the "device" to OSS or ALSA or ESD.
