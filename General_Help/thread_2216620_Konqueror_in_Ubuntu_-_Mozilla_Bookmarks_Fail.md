Question #1? Why do you have a need to use Konqueror on Ubuntu [unity assumed]? Chromium, Epiphany, and Firefox are all opensource browsers that aren't going to pull in all the KDE libraries on a non-KDE DE. Epiphany only pulled in a couple of packages on my Ubuntu [unity, 13.10] installation. Nothing wrong with Konqueror, mind you, but doesn't make sense if you aren't already running a KDE DE.

To answer your question about the conflict, you can export bookmarks from firefox into a file. Can you not then import these from Konqueror? I don't believe there is any conflict between Gnome and KDE that would cause this to fail (isn't it just a config file in a folder??). 

To answer your question about Kubuntu, if you prefer to use KDE apps, you could consider installing Ubuntu [KDE DE] aka Kubuntu.
