### Post by jasonlfunk on 2007-07-16
What do you mean that you "downloading the installer"? There is not a installer persay for Ubuntu. There is a iso that you burn onto a CD to install from. If you are downloading the ISO then no, that won't affect XP. But you are actually installing Ubuntu from the CD it certainly will delete everything on your Windows if you weren't careful in setting it up. You have to tell it where on your hard drive you want to install it, and if you choose the partition that has Windows on it, poof it's gone. 

It really isn't as scary as it sounds. Make sure you have all your data backed up (on a flash drive or dvd or something) before you attempt to install Ubuntu *just in case* something goes wrong. 

If you want to keep XP you will want to look into "Dual-Booting".
