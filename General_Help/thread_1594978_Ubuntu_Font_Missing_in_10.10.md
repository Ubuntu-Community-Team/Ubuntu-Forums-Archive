### Post by up2early on 2010-10-12
I have two machines running 10.10, one was updated from 10.04 on Sunday when 10.10 was released and going to:

System-->Preferences-->Appearance-->Font Tab

Each Font Choice pulldown menu has an Ubuntu option.

The other machine was updated from 10.04 to an earlier beta of 10.10 and has been updated constantly through the Update Manager.

When I go to:
System-->Preferences-->Appearance-->Font Tab

None of the pulldowns has an Ubuntu font choice.

Any ideas as to why this is missing the Ubuntu font option? And what I can do to fix this, as I like this new font.

---

### Post by abrianb on 2010-10-12
Just install it.  Click --- System-->Administration--->Synaptic Package Manager.....when Synaptic opens type the word Ubuntu into the search box. When the search appears scroll down to ttf-ubuntu-font-family. Click on the box in the s column next to this entry. Select mark for installation. Then click on the word "apply" at the top of the synaptic package manager. Done, easy peasey
