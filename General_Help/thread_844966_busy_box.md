### Post by rbanavara on 2008-06-30
Just make sure to note any errors that has still on screen / scrolled off screen (erm, you cant scroll up, just have to be alert while it boots!). Normally I get to busybox prompt when it fails with some filesystem verification.

You just type exit at the prompt & it continues with booting. To fix this once for all, If you have made some changes to partition, and are sure the changes are as intended, then set ubuntu to skiip the file verification part. If the reason is something else, then you have to take action based on the error...
