### Post by gregorye70 on 2010-02-17
In XP if you put an underscore at the begining of a file name, it will bring that file to the top of the list of files in any particular folder, it doesn't seem to have the same effect in ubuntu........is there a way to do the same thing to files in ubuntu?

---

### Post by jamesisin on 2010-02-17
In Ubuntu an underscore is more or less the same as a null character.  Files are alphabetized and numericized in folder views.  You can use this to your advantage by prepending 00 or somesuch (I use formats like _0_0_First Folder).

---

### Post by geirha on 2010-02-17
You could sort by emblems, and put emblems on files (right click -> properties -> emblems) you want to be listed first.
