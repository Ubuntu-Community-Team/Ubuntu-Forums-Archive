### Post by Bucky Ball on 2014-01-19
*Thread moved to **General Help**.*

Set up your Ubuntu how you like it then use Remastersys to create an ISO.

[http://remastersys.sourceforge.net/](http://remastersys.sourceforge.net/)

There is also a GUI you can use to make life easier. Remastersys creates an ISO that can then be burnt to install media like any other regular ISO ... 'cept it's your spin!

You can also boot the new ISO as a virtual machine.
