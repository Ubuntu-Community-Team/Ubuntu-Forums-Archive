### Post by hibliss on 2009-06-19
This is 2009, haven't you guys heard of lawsuits? Bittorrent is the future, limewire and Gnutella networks are the past.

If you must use Gnutella, fins a native Linux client, not a Java client which is a half attempted at designing it to be cross-platform. Limewire and Frostwire are nearly identical, both Java based and very heavy on resources.

If you want a true native Linux client without the Java, try GTK-gnutella -- [http://gtk-gnutella.sourceforge.net/en/?page=download]("http://gtk-gnutella.sourceforge.net/en/?page=download")

Simple interface and you can create a tray icon for it that does what you want.

I won't call you stupid for using the Gnutella network, but you should really look in to Bittorrent as an alternative.
