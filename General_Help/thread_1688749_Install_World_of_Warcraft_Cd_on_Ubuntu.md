You can do WoW over wine.  Wine sets up an environment that will look like windows to a piece of software.  Not all things will run on it, but I'm pretty sure WoW will.
 
Three sites:
[http://www.playonlinux.com/en/](http://www.playonlinux.com/en/)
[www.winehq.org]("http://www.winehq.org/")
[http://www.ubuntugamer.com/](http://www.ubuntugamer.com/)
 
I'd start with the first with a strong inclination to just google something like "installing WoW in wine on linux", or some variation of that.  I'd be shocked if there aren't multiple guides out there for this...
