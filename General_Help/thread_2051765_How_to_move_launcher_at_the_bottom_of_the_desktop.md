Have you tried placing mouse pointer over the launcher, then holding down the Window + alternate  then right mouse click? That will display options to move or remove from panel if it is an icon. If it is over a blank area it will display "Add to Panel, Properties, Delete this Panel and New Panel".  Assuming the keyboard is of a generic style with the Windows Logo key.
