### Post by Herman on 2008-11-10
:) Hello Skerit,
I'm not sure if I understand what you want, but I think you mean you want to know how to set 'volume labels' for your file systems?

If you have Intrepid Ibex, and if you have Gnome Partition Editor, you can easily do that by right-clicking on a partition and then selecting 'Label'. You can type in whatever you want to call it there.

This page shows an illustration of that, plus if you use Hardy Heron or an earlier Ubuntu, you can do the same thing from the command line. 
There are different commands for different file systems.
You can look that up on this page also, here is the page, [File Systems and Mounting Page]("http://users.bigpond.net.au/hermanzone/p10.htm").

I hope that's what you wanted to know?
