### Post by tamoneya on 2008-05-28
unless you have a problem with the BIOS I would just leave it.  Really the only time I update a BIOS is for a CPU which is newer than the motherboard.  When this is the case the BIOS wont be able to detect the CPU since it didnt exist when the BIOS was installed.  Other than that I typically leave it.  I think there are some open source BIOSes that actually load linux and have some basic programs(browser, text editor, command line) all in the bios and they load crazy fast.

[http://www.coreboot.org/Welcome_to_coreboot](http://www.coreboot.org/Welcome_to_coreboot)

You should note that I have never used it and you should do your research to make sure it is compatibile before you try to install it.  If you mess up with your BIOS you can permenantly brick your motherboard.
