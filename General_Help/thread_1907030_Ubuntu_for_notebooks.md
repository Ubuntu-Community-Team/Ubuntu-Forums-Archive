### Post by ajgreeny on 2012-01-10
There is no ubuntu version specifically for "netbooks" any more, and I think the last version of that was 10.10.  The desktop environment on that was almost an early version of unity, or certainly very like it, so with the release of 11.04, and unity as the default desktop, it was deemed unnecessary to continue with the separate "netbook" version.

Even when it was in existence, it was not smaller or faster than the standard ubuntu gnome desktop version, it simply had the different desktop environment, and removed some packages that were really only needed for larger screens and CD/DVD drives.  Otherwise it was the same beast.  It was possible, and in fact I did, install the UNR version and then add the ubuntu-desktop package to give you a full ubuntu system, but with the netbook desktop as well as gnome.
