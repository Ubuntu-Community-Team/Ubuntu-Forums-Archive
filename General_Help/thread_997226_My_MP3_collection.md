### Post by CatKiller on 2008-11-29
It's perfectly possible to do this from the command line, but you'll probably find it easiest to use **soundconverter**. It's got a drag-and-drop interface and you can queue up as many tracks as you like. Lots of output options. By default it will put the new versions in the same place as the old ones, but you could put all the new ones somewhere else if you wanted.
