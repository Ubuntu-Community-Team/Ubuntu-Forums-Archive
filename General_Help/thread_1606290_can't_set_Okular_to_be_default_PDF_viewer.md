### Post by vim_lover on 2010-10-28
Right click the pdf file.  You will get a menu (See menu.png)
Click properties.  You will get File Properties window (See properties.png)
In the File Properties windows click the tab "Open with"
Here you can select whatever program you want.  (and also remove whatever program you don't want).
Then close the File Properties window.  Now double click the pdf file.  It should be opened by okular by default.
