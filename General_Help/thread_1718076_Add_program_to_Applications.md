I have installed Google Earth under Ubuntu 10.10. It runs fine but I can only run it from a Terminal using the command Googleearth. How can I add it to the Applications/Internet Menu?
Thank you.

---

### Post by Tim Sharitt on 2011-03-30
Go to the System>Preferences menu and select Main Menu. From there, select the sub menu you wish the application to appear in from the list on the left (Internet in your case). Once the sub menu is selected, click the "New Item" button on the far right. In the pop-up window which appears, enter the name of the application and the command to launch it (most likely whatever you've been using to launch from the terminal) into the proper places.
