### Post by kricit on 2005-09-16
Hi, Ive just started developing with gambas and i found this tricky to.
If your installation was like mine GAMBAS was installed in the folder /opt/gambas
to begin with what i did was typed in the Konsole 'opt/gambas/gambas' and clicked return. Gambas then started but if you close Konsole the program will exit. after the first time you do that go into the KDE menu editor and add gambas under the development section and in the command to run section type 'gambas'. that worked for me, tell me how it goes!
