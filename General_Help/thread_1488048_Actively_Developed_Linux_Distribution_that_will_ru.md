Hi Will, curious what your goal for this project is? You might be able to get a basic Linux terminal running as an educational/hobby project, but you're not going to be watching Youtube, surfing the web, creating Office documents, or any number of other tasks we take for granted these days. :)

The lightest Linux I've personally tried is DeLi Linux: [http://www.delilinux.de/#about](http://www.delilinux.de/#about)

They recommend 64mb of ram, but you **might** be able to get it working with less if you create a swap partition first. Swap expands your available memory at the expense of speed (hard drive is much, much slower than RAM).
