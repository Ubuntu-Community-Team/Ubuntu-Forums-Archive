### Post by Dennis N on 2017-01-03
I still have the setup described at [https://ubuntuforums.org/showthread.php?t=2234069&p=13071667#post13071667](https://ubuntuforums.org/showthread.php?t=2234069&p=13071667#post13071667) in Lubuntu 14.04, and just copied the little empty-trash.desktop file posted there to the same location in Lubuntu 16.04, then installed **trash-cli**. I can confirm it does work in Lubuntu 16.04. 

As it says there, I didn't use a .deb file, but just wrote the file in a text editor and saved it as described.

In Lubuntu, the lack of that empty trash option with the Desktop icon is not new. Always been so.
