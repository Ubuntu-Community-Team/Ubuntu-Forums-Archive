### Post by coffeecat on 2011-01-06
> **Scunnered said:**
> I wish to install 10.04 LTS on another system and believe that there is a download which is right up to date as this would save me time.

You are probably thinking of the daily builds which are only relevant to a testing version - currently Natty/11.04. A rebuild for an LTS version is released every few months. mastblasta has given you a link to download  the desktop version of the 10.04.1 point release. If you want the alternate or server 10.04.1 images (netbook seems to be stuck at 10.04), here is another link:

[http://releases.ubuntu.com/lucid/](http://releases.ubuntu.com/lucid/)

The 10.04.2 build is scheduled to be released on 27th January.
