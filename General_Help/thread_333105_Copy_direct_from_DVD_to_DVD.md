### Post by orb9220 on 2007-01-07
To copy single layer dvd's in gnome all you have to do is right click cd on desktop and select copy disk. Then you have a choice of **copy disc** or save it as an iso to hardrive.

At the top is **copy disc to** where you select target drive or select file image to save to hardisk.

Alot of less work than running a separate application for just copying a data DVD.
