### Post by fyo on 2008-01-28
I had a similar problem back when I was installing Breezy (5.10). It turned out I had a bad DVD-RW drive AND a bad CD-RW drive. What are the odds of that?!

Anyway, like lizadove said, try burning the CD/DVD again.

As for which version, I have an X2 and am using the 64bit version of Ubuntu. Going 64bit is a much better proposition now than it was just a version ago, although there are a few caveats. The upside is you get a 15-30% typical performance boost, with many "heavy" applications much, much faster (as much as 100% or more).

You can always create a chrooted 32bit environment for any applications that don't play well in 64bit (e.g. Wine). There is a sticky in the 64bit forums for doing exactly this - and I've had great success with it.
