### Post by lintoon on 2008-09-07
The first thing you need to do is configure your router to allow wireless connections. This involves setting an SSID (the name the router broadcasts).  Remember to save your settings on the router.  Check your router manual for instructions.

You will need to set security on the router eventually but I would leave it off until you have proved you can connect.  Then when you have it working you can set the security (WEP or WPA - choose WPA).

On your pc if your wireless card is working you should be able to go to
System -> Administration -> Network and select properties for your wireless connection (you may have to click "Unlock" first. Here you can set the ssid.  Alternatively, you may have an icon at the top of your screen on the right hand side which looks like 2 pcs.  If you click on it you should see your routers ssid.

If you get connected you should now set security.  The security is basically setting the same password on both the router and pc.  Choose WPA as your security option.

### Post by lintoon on 2008-09-07
A nice tip is to reboot your router after you make any changes.  So make your changes, save (wait until it finishes), unplug and plug your router back in.

This is not always necessary but I can't tell you how many times I've set up a wireless router (mostly netgear) which didn't work until I rebooted (even though I had saved the settings).  It's best to be safe and not waste any unnecessary time.
