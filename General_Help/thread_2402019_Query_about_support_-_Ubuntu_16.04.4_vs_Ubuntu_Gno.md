### Post by oldfred on 2018-09-25
Others discussing issue.
[https://discourse.ubuntu.com/t/support-for-deprecated-flavors/6191](https://discourse.ubuntu.com/t/support-for-deprecated-flavors/6191)

Part of confusion is that Ubuntu Gnome is discontinued as a flavor as now Ubuntu uses Gnome as its standard again in 18.04. So it is not a flavor anymore, but just Ubuntu.
[https://wiki.ubuntu.com/UbuntuFlavors](https://wiki.ubuntu.com/UbuntuFlavors)

To the extent that much of system if same as Ubuntu, those bits & pieces will still be in repository. But the unique Gnome bits at versions in 16.04.5 will then not be updated after April 2019.

       [https://wiki.ubuntu.com/Kernel/RollingLTSEnablementStack](https://wiki.ubuntu.com/Kernel/RollingLTSEnablementStack)
