### Post by beast2k on 2006-02-28
I believe Azureus is what your looking for, you can install it with the automatix script (easy way). During setup choose the advanced option for user and you'll have what you need it has more options and capabiltity than bittornado or the default client. The downside is it depends on java so ensure your java is up to date and working properly.
