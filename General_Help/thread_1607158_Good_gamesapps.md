### Post by gradysghost on 2010-10-27
Depends on what you're into.  The package sgt-puzzles has some fantastic logic puzzles.  I'm addicted to the game Inertia from this package.

If you're more into first person shooters, you can try all of the following:

[LIST]
[*]Sauerbraten - Great, very fast-paced FPS built on the Cube2 engine.  Has a fantastic map editing system.
[*]Nexuiz - Fast-paced deathmatch style FPS.
[/LIST]

Battle for Wesnoth is kind of the very definition of "open source RTS", and is a lot of fun to play if you're into that kind of thing.  It's got some good single-player campaigns and a great LAN/WAN multiplayer mode.  Graphics are 2D, and if you're a hardcore gamer, that may turn you off, but it doesn't make the game any less fun.  (Think early Warcraft games.)

Another RTS that I had some fun with for a while was Warzone 2100, although I seem to remember encountering a few bugs.

There are also some really good games that aren't in the repos, aren't open source, and may cost you a few bucks from their independent producers/distributers, but are worth the cost.  The Penumbra series ([http://www.penumbragame.com/](http://www.penumbragame.com/)) are very cool horror/adventure games.  World of Goo ([http://www.worldofgoo.com/](http://www.worldofgoo.com/)) is a wonderful casual puzzle game with a native Linux port.
