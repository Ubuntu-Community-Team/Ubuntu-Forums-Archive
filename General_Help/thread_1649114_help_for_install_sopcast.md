### Post by howefield on 2010-12-19
What version of Ubuntu are you using ?

If either 10.04 or 10.10 you should be able to use this repository...

ppa:ferramroberto/sopcast


Terminal commands to add the repository and install are as follows.

```
sudo add-apt-repository ppa:ferramroberto/sopcast
```
```
sudo apt-get update
```
```
sudo apt-get install sopcast-player
```
