### Post by skiingdomo on 2008-08-19
Hi guys.

So this month I have decided to upgrade to a 3rd monitor on my work PC. 

Currently running 2x 24" LCDs on a 8800GT. My motherboard is the oldest part in my system (gigabyte board). I am running a Q6600 & 4gigs DDR2 ram.

To upgrade to a 3rd monitor I will need to purchase a second video card. I had a bit of trouble settting up dual monitors, but as with any linux install, when it works it works beautifully and never breaks down.

I have knocked it back to the following 3 options to get 3 monitors going:

1. Get a PCI video card and run the 3rd monitor off a VGA port. Possible compatibility issues & 3rd monitor will not look as clear as the 2 other DVI connected monitors.

2. Upgrade the motherboard to one with 2x PCIE16x slots and get a second 8800GT - No compatibility issues, only problem is Nvidia drivers suck for 8800GTs last time I checked and it could be a pain setting it up.

3. Upgrade the motherboard & get two ATI cards. - Better support and possibly easier to set up?

Can anyone give me some advice on what you would recommend? The monitors run at 1920x1200 each and I will run the desktop spanned accross all 3.

Thanks guys.
