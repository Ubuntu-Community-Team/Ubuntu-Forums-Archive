### Post by tom mclernon on 2007-12-27
Yes the Kubuntu KDE 4 RC2 CD is installable, if you boot and run the program from the CD.  You will find an "Install" selection on the menu, it is not on the first level menu.  I forget where to start the drill down from, but you can play with this until you find it.  It installs pretty much the same way as any Kubuntu OS. 

There will be no Adept installer option, but you can install this from the command line window. Type: sudo apt-get install adept.   You need to watch the packages, that you install, and make sure that they are all for KDE4.  Even then you may have some conflicts.
I am not a Linux expert, I just follow progress, since it is coming awfully close to a viable Windows replacement.
