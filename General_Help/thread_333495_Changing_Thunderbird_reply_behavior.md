### Post by useResa on 2007-01-07
When you have started Thunderbird, select Edit --> Preferences.

Next select the "Composition Icon"
The first option on the "General" tab will give you the possibility to set how you like to forward messages. You can choose between "Inline" and "As attachments".
Default setting is "As attachments"

Hopes this helps you (and your mother)
