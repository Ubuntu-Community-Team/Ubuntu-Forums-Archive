Is this really normal?Used to be normal (especially with Acer) before 32nm processors. But for newer models (not older than 3 years), that much heat is not normal unless the game is really keeping the cpu on toes all the time.

You did,'t mention your laptop model or its specs, nor anything about the OS and its version, so it's hard to guess the reason.

If it is Ubuntu, and you can run the game in 'windowed mode', you can run 'top' command in parallel to see how much processing power is used by what.
