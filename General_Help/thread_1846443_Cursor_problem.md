### Post by merely on 2011-09-19
Greetings,

I am having an innocuous but annoying problem with Ubuntu. Randomly, as I am typing, the cursor will flit to where the mouse-arrow is. That is to say, I might be typing in abiword, or this post, and if the arrow is elsewhere on the page, I'll occasionally be 're-loctated' thence, and begin typing there.

Might anyone know how to fix this? Thank you in advance.

---

### Post by sikander3786 on 2011-09-19
Go to System > Preference > Windows and make sure 'Select windows when the mouse moves over them' is disabled.

Only if you installed CCSM already:

Go to System > Preferences > CompizConfig Settings Manager > General Options > Focus & Raise Behaviour and disable 'Auto Raise'.

---

### Post by bupe on 2011-09-19
Also if you are using a laptop make sure that your touch pad is disabled and test typing that way - I had issues with a few laptops I've been using that while I was typing my palm accidentally hit the touch area and made the cursor jump to the mouse pointer.
