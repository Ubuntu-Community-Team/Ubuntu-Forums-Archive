### Post by Lahntaler on 2009-02-28
I managed to set up Evolution to use an Exchange server - hooray.  A major challenge was to find the Global Address List server. To find it in Outlook, do this:

Click the little phone-book icon on the Outlook tool bar to open the Address Book. 

[LIST=1]
[*]In the drop down menu select: "Global Address List".

[*]Right-Click on "Global Address List".

[*]Select "Properties".
   There you find the name of the server for the Global Address List.
[/LIST]

The Properties also have the container. The container seems to be irrelevant for Evolution.

In Evolution in the account editor on the tab "Receiving Options" enter just the name of the Global Address List server without protocol. For example:  domcontroller.mycompany.com
