### Post by bodhi.zazen on 2012-09-17
Adding them to the sudo group is easiest, otherwise you can use edit /etc/suoders.

If editing I highly suggest using visudo

```
sudo visudo
```

See [http://www.gratisoft.us/sudo/sudoers.man.html](http://www.gratisoft.us/sudo/sudoers.man.html)
