### Post by khedron on 2009-01-29
Well, I'm not sure exactly what features you want, but if you find GIMP too complicated (and a lot of people do) you could try Krita, which is a KDE version of MS Paint. It's probably not exactly what you want, but it should do the job.

For text there is usually a text tool in most image editors.
For the GIMP see this: [http://www.yourmachines.org/tutorials/gimp.html#fig_17](http://www.yourmachines.org/tutorials/gimp.html#fig_17)
