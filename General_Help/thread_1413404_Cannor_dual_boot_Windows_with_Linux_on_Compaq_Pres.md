### Post by meierfra. on 2010-02-22
[QUOTE=http://ubuntuforums.org/showpost.php?p=8848594&postcount=13]I am having the same problem on a Compaq Presario CQ60 dual booting between Karmic and Windows 7. If I reinstall Windows 7 from the upgrade CD, the system will dual boot for a few days but eventually the Win 7 boot fails with the symptoms described in this thread: The Win 7 boot starts, the Windows logo forms on the screen, then the screen blinks, goes black, the BIOS restarts, and then GRUB boots again.[/QUOTE]

From your description and  the RESULTS.txt this looks like a Windows or hardware problem. You might run "chkdsk /r C:" from the Windows command prompt to schedule a file system check on the next reboot.  You should also check out your hardware (the hard disk, memory, cmos battery,cables...), but I don't know much about hardware.
