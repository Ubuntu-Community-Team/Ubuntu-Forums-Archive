### Post by inameiname on 2011-11-01
Aside from gconf and dconf commands to add/remove them, there is Ubuntu Tweak or Gnome Tweak Tool. 

In the past, it was easy to add/remove them via Ubuntu Tweak. However, that isn't updated yet for 11.10. 

Fortunately, Gnome Tweak Tool has that option in it. Therefore, just install it, if you don't have it already (which would be my guess as to where you checked them to be on the desktop in the first place), and it should be able to remove them in 11.10.

sudo apt-get install gnome-tweak-tool
