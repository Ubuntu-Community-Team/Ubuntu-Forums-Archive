If this is the case then get the eeePC with linux pre-installed (why pay for windows and get a smaller hard drive if you just want to run Ubuntu?).

If for some reason you do want to install Ubuntu via wubi, then get the 32bit version of Ubuntu (the eeePC is not 64bit capable anyway).

I can't be sure if wubi will work for installing ubuntu, as I have no need to run windows here.  But Ubuntu should install fine by itself without wubi.

If all else fails, try installing Xubuntu - as it has less hardware speed requirements than regular Ubuntu.

You can install Wubi without a CD. Just download the installer from the Wubi site. When you run it it will pull the correct disk image down over the net and use it to do the install.

Obviously this makes the normally quick Wubi install take a lot longer.
