### Post by HermanAB on 2008-11-11
I have seen that happen - most annoying.  It is a screen driver problem.  It is probably a timing issue that causes the display chip to screw up.

Update the screen driver and hope it doesn't happen again.  When all else fails, buy a new (different!) display adaptor.
