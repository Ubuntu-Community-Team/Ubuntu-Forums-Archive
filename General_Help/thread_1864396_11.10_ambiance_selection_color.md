### Post by ReptilianShadow on 2011-10-21
I was looking for the same thing, found this thread on askubuntu:
[http://askubuntu.com/questions/63126/how-do-i-change-the-selected-items-color](http://askubuntu.com/questions/63126/how-do-i-change-the-selected-items-color)
Tested it out, works just  as you'd expect it would.
Just make sure to write down the hexidecimal color value you change it to, so you can easily find it and change it again later.

Also you need to have administrative access to edit the files, so, use in terminal:
```
sudo nano /usr/share/themes/Ambiance/gtk-3.0/gtk.css
```
...or whatever text editor you want to use, like gedit.
```
sudo gedit /usr/share/themes/Ambiance/gtk-3.0/gtk.css
```

Oh, and here's an online hexidecimal color viewer: (so that you can see the color before you change the files)
[http://drpeterjones.com/colorcalc/](http://drpeterjones.com/colorcalc/)
There's also a hexidecimal color viewer in 'Appearence' in the lower right corner (colored box)

You might need to log out and back in again for the desktop to show all the changes

Hope that works for you.
Cheers
