### Post by Ms_Angel_D on 2011-08-19
> **Harris6310 said:**
> I would go for the duel boot option. If I was to have two partitions half of the hard drive each (one for vista, one for Ubuntu.) Then say I didn't like Ubuntu, so I would delete the partition for it. Could I then extend the vista partition to fill the hard drive?

That is absolutely possible.

If you wanted to get rid of windows you could use [Gparted]("http://gparted.sourceforge.net/") which is freely available in the software center. You could also use gparted from a livecd to wipe your ubuntu partition and format it to NTFS so you could re-gain the space in windows should you choose to go that route.

Please note that the option is only available for an actual dual boot and not with a wubi install which merely puts Ubuntu in a virtual partition.

### Post by stephenhau on 2011-08-19
I'd suggest dual booting, and if you like Ubuntu enough, leave Vista there! Hard drive space is cheap, and if your space is limited, you could shrink Vista back to say 10GB.
There might just be some software that you want to use that only works on Windows, then you'll fortunately still have that option. 

If there's any data that you want accessible to both Ubuntu and Vista, make a NTFS partition for that shared data.
[This guide]("http://ubuntuforums.org/showthread.php?t=203524") is useful for having your Firefox and Thunderbird profiles available to both Windows and Ubuntu.

Good luck!
