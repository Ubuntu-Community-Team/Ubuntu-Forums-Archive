### Post by wpshooter on 2009-06-16

But if I did, I would first use [www.killdisk.com](www.killdisk.com) to WIPE the hard drive, then I would install Windows O/S first and then I would NOT install Ubuntu inside Windows but would install it as a standalone O/S in it's own partitions by installing it from the Ubuntu CD and then when I booted machine after installing both O/Ss, I would choose which of the 2 I wanted the machihne to boot to.
