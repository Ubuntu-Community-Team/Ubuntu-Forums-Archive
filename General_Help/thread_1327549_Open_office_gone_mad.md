### Post by Baelus on 2009-11-15
In your home folder there should be a hidden folder called ".openoffice.org2" or something like it. In nautilus file manager you can hold ctrl+h to see hidden files and folders.

Try moving that folder somewhere else temporarily, like to the Documents folder. Then start open office. If the problem remains check that there is no hidden folder in your home directory then try an uninstall and reinstall.

All your user settings and preferences for open office are in that folder. Maybe having the program reset it to its default state may solve the problem.

If you need your usual ".openoffice.org2" folder you can find it the Documents folder, or wherever you moved it to.

See if that helps. :)
