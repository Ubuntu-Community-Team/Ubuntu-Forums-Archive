### Post by howefield on 2016-10-18
> **rcrahul60 said:**
> anyone can tell me how to use keepassx after this

Hover the mouse over the top panel to reveal the Keepassx menus and select Database > New Database, then fill in the password that you will use to access the database, and optionally type the path to a key file if you want to use that as a secondary method to opening the database. OK your way back out and you should be able to start entering individual entries.

### Post by howefield on 2016-10-18
No problem, you're welcome, one point that might be useful if you have more than one machine is that you can put the database on a "cloud" device, so for example I put mine on an owncloud setup and sync each machine to that so one change made on one machine is synced to the others automatically.
