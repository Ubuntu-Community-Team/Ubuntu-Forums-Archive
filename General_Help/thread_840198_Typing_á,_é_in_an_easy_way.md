### Post by drs305 on 2008-06-25
Depending on your setup (I'm using US 105):
System, Preferences, Keyboard, Layouts, Layout Options, Compose Key Position (I chose R-ALT for this example).

To type á, type ALT-R then **'** then **a**
To type é, type ALT-R then **'** the **e**

You are typing each in sequence, not together. Don't delay too long between keys.

### Post by tyfius on 2008-06-25
If you are using US International you should be able to get that using a combination of the ',",` and : characters with a key.

For example, to type "é" I need to press the ' key, (nothing will show up at this point) followed by the e key.

The downside is that if you only want a ' you need to press the spacebar after the ' key.
