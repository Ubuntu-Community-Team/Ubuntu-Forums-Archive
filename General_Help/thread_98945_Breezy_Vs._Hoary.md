### Post by aysiu on 2005-12-04
The differences, as far as I can tell, are mainly cosmetic:

1. Terminal is in Accessories instead of System Tools
2. There's a "pretty" boot image instead of scrolling text
3. There's a simpler version of Synaptic Package Manager available called Add Applications

Other differences were generally instability and weird bugs (especially around release time--October 13--though, a lot of these have since been fixed).

Hoary was rock solid. I wouldn't switch if you already have Hoary installed. You may want to change up for Dapper in April, though.

### Post by az on 2005-12-04
Breezy is the newer version.  It is newer and more stuff works out-of-the-box.  You can install Hoary and dist-upgrade to Breezy.

If Hoary is working fine for you, stick with it.  It is officially supported (as in security updates and severe bug fixes) for another year.
