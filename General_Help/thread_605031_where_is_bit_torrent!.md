### Post by Ouchy on 2007-11-06
Do you have Java installed? You need to have Java in order to use Azureus.

Anyways, the default BitTorrent manager is installed, it's just not visible in the applications menu by default. If you want to use that, right click on applications and click edit menus. From there navigate to internet, and check BitTorrent. It will now be available from the menu.
