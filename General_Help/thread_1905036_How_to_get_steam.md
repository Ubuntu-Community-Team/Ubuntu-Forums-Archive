### Post by chipbuster on 2012-01-05
[Wine]("http://www.winehq.org/") is one option.

There are also (I believe) several other layers that are similar to Wine.

None of them guarantee that games such as TF2/L4D will run well--they just give you the possibility of running the games.

The only surefire way I know of to play Windows games on Linux is to virtualize Windows inside of Linux, in which case you might as well just dual-boot.

### Post by QIII on 2012-01-06
There are only a few options for playing Windows games in Linux.

You might try Wine or PlayOnLinux, but there are no guarantees any particular application will work.

Virtualizing is good for many tasks, but not so good for games.  The virtual graphics hardware is not robust.

So, I would suggest dual booting, but that is a messy and often difficult proposition with Linux installed first.

What are the specs and setup of the machine you are using?  Setting up Windows as a dual boot after Ubuntu is a snap if you have two physical hard drives.
