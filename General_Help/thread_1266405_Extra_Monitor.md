### Post by NoaHall on 2009-09-14
System -> Admin -> Hardware Drivers -> Download

If you have nvidia 

Applications -> Accessories -> terminal -> gksudo nvidia-settings -> Display configuration -> Enable the other screen -> Save to X file

If you have ATI/Intel, I don't know.
