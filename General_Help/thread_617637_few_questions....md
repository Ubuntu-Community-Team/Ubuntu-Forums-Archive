### Post by arnaudus on 2007-11-20
For your first question:

Perhaps I don't understand it, but the easiest way to reinstall Ubuntu is to reboot with the CD/DVD, and to perform the install again (without formatting the /home if you want to save it). So: the rescue partition is the CD. 

The other way would be to get the list of all packages included in the default distribution, and to remove/add additional/missing ones by hand. This will not deal with all stupid things one my do using sudo, though (such as modifying a script, installing an alpha-version driver, etc.
