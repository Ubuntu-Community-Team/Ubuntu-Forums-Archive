### Post by HereInOz on 2006-11-20
For your info, we are running Ubuntu on a 500MHz PIII with 512 MB RAM as a "file server" for a small peer to peer network, and it works fine.  Apps such as Firefox take a little bit of time to open, but all in all a useful machine.  Your RAM may be a little on the small side, and if your HDD is a bit slow, that will also cause a problem, but it should go OK.  

It will always be slower when using the Live CD, so I would install Ubuntu to the HDD first, and see how it goes, and then change to Xubuntu if it all bogs down and goes nowhere.  Kubuntu seems to run slower than Ubuntu, by the way, but that is a subjective assessment.
