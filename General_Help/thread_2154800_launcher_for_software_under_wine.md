I am running Ubuntu 12.04 (Unity)
I have programs running under wine.
Question: How to make program launcher for software running under wine?
Simple in Ubuntu 10.04

---

### Post by 3rdalbum on 2013-06-15
Simple in 12.04 as well.

Go to the Main Menu program (you can find it in the Dash) and create a new menu item. Set the command to be the word "wine" then a space, then the path of the program to run.

When you have created it, search for it in the Dash and drag its icon to the Launcher.
