### Post by WorMzy on 2010-05-08
Use synaptic to search for them. For example, to install nVidia drivers, you'd search for "nvidia". Installing nvidia-current will ensure that you always have the most up-to-date nvidia drivers, or you can install one of the specific packages, such as nvidia-glx-173.

OR

You can download the drivers from the respective websites. Mark the downloaded .run file as executable, then run it.
