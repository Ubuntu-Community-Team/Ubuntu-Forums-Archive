### Post by lovinglinux on 2009-05-16
Deluge is the best torrent client in my opinion. The current version (1.1.7) likes to eat some processor power on Jaunty, but nothing compared to java+vuze and this has been fixed for the next release.

Deluge has full encryption, blocklist plugin, UPnP, PEX, DHT, file selection and priority and other important stuff. It is extremely easy to configure when compared to Vuze, has a nice interface and can be used to connect remotely to a headless server using the gtk frontend, webui or cli.

You can get the deb file from [here](http://www.getdeb.net/app/Deluge) or setup it's own [PPA repository](https://launchpad.net/~deluge-team/+archive/ppa).

Further info at [http://dev.deluge-torrent.org](http://dev.deluge-torrent.org)
