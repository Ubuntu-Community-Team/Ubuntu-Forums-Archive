### Post by apunc1 on 2007-07-01
I think by downloading you mean installing. I'm going to assume that, anyway.

If this is your first time using Ubuntu and you would like to see if it really is for you after using the LIVE CD for awhile, then you should probably dual boot. By dual booting, you can resize your Windows XP install and partition your hard disk to make room for ubuntu. Then when you boot, you can choose from grub to boot into windows or Ubuntu.
Here is a dual boot guide [http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)

If you are sure you want only Ubuntu on your hard drive, or have a separate hard drive/computer to test Ubuntu, then yes you can format your hard drive to have only Ubuntu on it, and that will over write any data you had.

Have you tried the Live CD yet? It simulates an Ubuntu install so you can see if your hardware is supported and generally get the feel of the OS.

Hope that helps.
