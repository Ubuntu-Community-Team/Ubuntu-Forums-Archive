### Post by Topsiho on 2016-01-26
Ubuntu 14.04 will be supported for 5 years after it's initial release, in April 2014. This support ends in 2019.
14.04.3 is just a so-called point release for 14.04, so that after it's install there are not so many accumulated updates.
If you install the initial release, or a point release of Ubuntu 14.04 you are safe until April 2019. If you install the updates when they come, of course.
This you can also see for yourself in the link by howefield.
