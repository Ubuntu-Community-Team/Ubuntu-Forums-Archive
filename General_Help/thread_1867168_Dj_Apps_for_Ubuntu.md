### Post by zero2xiii on 2011-10-22
Hay,

So glad I'm not the only guy on here with these ideas.

Unfortunately ubuntu does not have that great DJ software (LMMS and Hydrogen and all that are excelent for creating beats and such...)

So this unfortunately translates to having some form of either windows or OSx to run DJ soft on. I use virtual DJ on a windows XP virtual machine with a controler simmilar to the one you have in mind, and that works pretty well.

I have tried running it in wine. It runs, but burns my CPU at near 100% on an i5 2.66 CPU.. On both 32 and 64 bit architectures.

This is the ONE area linux fall very far behind. With the prospects of lightworks running on linux in the very near future, video editing is coming to light on linux, but not yet DJ. Ardour is an excelent choice if you are famillier with the likes of protools or cubase.

Good luck with all this, really hope you get everything workin:)... If you have some other questions, just shout.

### Post by TREESofRIGHTEOUSNESS on 2011-10-24
audacity is a REALLY simple multi-track recorder that loads really fast.  Ardour takes over jack, so it is more for professional studio recordings.
you might also look at Traverso.
I am a musician and not a DJ, so I don't quite know what you'd be trying to do.
I assume you want to hook your turntables up to the computer, and possibly add effects?  There are some live performance tools you can use.  But I don't use those, so I can't add much more to the conversation, other than Audacity is very lightweight, and records very easily.  I use it quite often, and it has never cause me problems
