### Post by trinitydan on 2010-12-25
This is a question that has been coming up a lot lately so you may wat to try searching around for info about what netbooks people like and are well supported running Linux.  As far as the question of whether to run UNR or Desktop edition Ubuntu, I have to say that I REALLY don't care for UNR.  That is sugar coating it quite a bit too as I know people are working hard on this, but I don't understand how it even is fitting of the "netbook" designation (Ubuntu Netbook Remix) when it looks like something that would be better suited to an iPod/tablet type mobile device.  I question the reasoning behind this shift toward a mobile device format.  It seems like it is because of a perceived general trend toward mobile devices, but in my opinion the netbook isn't going anywhere any time soon, if only for the fact that it fills the need for a basic functioning computer on a very small budget and there is a need for this.  

There has got to be something going on behind the scenes that is driving this shift.  It doesn't seem to be driven by the wants/needs of the average current Ubuntu user.  Unless you like the idea of a super simplified Ubuntu with giant buttons and limited functionality, I'd suggest sticking with desktop edition.  Also I prefer to set my panels to autohide to maximize screenspace on my little 10" screen, something that is not yet functional with the giant panel in UNR.
