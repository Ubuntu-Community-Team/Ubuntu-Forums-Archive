### Post by Skaperen on 2019-10-07
this command makes many connections to several websites.  these warnings are saying that some of them failed.  try running the command, again, and see if it repeats.  scroll the terminal back and capture more messages from earlier if it does it again.  there may be basic problems, like failed DNS lookups, happening.
