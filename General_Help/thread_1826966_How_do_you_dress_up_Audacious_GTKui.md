[its possible to specify a particular GTK theme for a particular program]("http://urukrama.wordpress.com/2008/07/13/setting-a-custom-gtk-theme-for-specific-applications/"), but themes are really there to give a unified desktop look.

### Post by Frogs Hair on 2011-08-17
Appearance Preferences will indicate which GTK theme is in use . Right click an open area on the desktop select change background , then select the themes tab . If you then select customize you will find more options .
