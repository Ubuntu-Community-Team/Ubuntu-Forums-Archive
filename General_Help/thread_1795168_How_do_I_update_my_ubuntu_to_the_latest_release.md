### Post by Wim Sturkenboom on 2011-07-02
As you don't state if you're running a desktop version or a server version, I'll assume the former. There is an upgrade button in the update manager. Just use that.

And yes, it can go awfully wrong, so make sure your important stuff is backed up.

Alternative is to do a fresh install; if you have a seperate partition for /home, don't format it during the partitioning.
