### Post by Clipster on 2007-05-27
When I try to install a new package in Fiesty Fawn from a Terminal window, I can type in my user name, but when I try to enter my password, the typed keys do not show up and the box remains blank. The password I type is correct. Any suggestions?

### Post by earobinson on 2007-05-27
this is a security feature to hide your password as well as the number of charters in your password. You should be able to do ```
sudo apt-get install <package name>
``` then type your password (even though nothing happens) hit enter and it will install your package.
