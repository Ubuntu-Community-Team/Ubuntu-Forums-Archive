### Post by Scott Harrison on 2012-08-17
At a terminal, run the following:

```
sudo apt-get install nvidia-current
```

followed by:

```
sudo apt-get update
```

Probably best to reboot after this.

If this doesn't work, please reply and I'll try to help further.
