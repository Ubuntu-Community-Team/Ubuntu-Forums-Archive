### Post by pytheas22 on 2008-10-21
Ubuntu Studio Edition is pretty dark.  There's also Ubuntu Satanic Edition, which I've never used but which looks pretty black in screenshots.  You can install each of these things as a theme on top of your existing Ubuntu system.

Also, there's a compiz effect called 'Negative' that allows you to invert colors on your screen, so that everything white becomes black, etc.  It's a nice way to save energy by reducing screen brightness without affecting readability of text, if that's what you're trying to do.  On my computer I press [Windows Key]+M to trigger it.  I'm not sure if it comes enabled by default, but if not, open Synaptic and install 'compizconfig-settings-manager', go to System>Preferences>Advanced Desktop Effects Settings and turn on the Negative effect.
