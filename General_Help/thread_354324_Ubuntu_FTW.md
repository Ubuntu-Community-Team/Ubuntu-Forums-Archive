### Post by atact88 on 2007-02-05
I'd just like to relate a short, sweet moment I had with Ubuntu today.

Well yesterday I installed Edgy on my PC desktop (I run Dapper on my laptop). I had set up a separate partition I would use to install programs to (to share between Windows and Ubuntu via Wine) and to save data to. I forgot to format the partition into a fat32 volume before I installed Ubuntu. When I realized my mistake, I went into Windows and tried to do so, but Windows (through some fault of my own) ended up overwriting my partition table...I got scared and went to reboot the computer, and was greeted by a friendly GRUB Error 17 - partitions not found and whatnot.

So today I sucked it up, popped in the Windows install CD and thought I was going to hit R at the option screen, type fixMBR, and that would be the end of it. Nope. The Windows install must have freaked out when it saw what it had done to my hard drive - the XP setup program completely froze. Tried it three times, after a reboot, a complete power down and a reboot, etc. Nope.

I went online, where I read about some similar problems and people pointed out bad BIOS's, motherboards, dead hard drives, etc. "Maybe it's hardware failure," I thought. I decided to give it one last chance. I popped my Ubuntu cd in and booted up on it. Bingo, it worked. I set up new hard drive partitions - piece of cake. I installed the OS - Boom! I was up and running again. Ubuntu succeeded where the world's most expensive home operating system had failed. My comp is up and running like a champ with a fresh Ubuntu install now.

Thanks Ubuntu! You Rock!
