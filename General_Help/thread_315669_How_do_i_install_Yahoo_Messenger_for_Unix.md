### Post by iamhugeinjapan on 2006-12-09
The official Yahoo Messenger for Linux is extremely old and unmaintained. It is unlikely to have any features that Gaim doesn't have.

However there is a more advanced specific Yahoo implementation called GYachI [http://gyachi.sourceforge.net/](http://gyachi.sourceforge.net/)
It has an Edgy deb.
