### Post by PmDematagoda on 2008-02-14
If you mean by an indexing service then Tracker which is preloaded in Ubuntu 7.10 should help, Tracker can be found in Applications>Accessories>Tracker Search Tool, you can make changes in Tracker(add more paths to be monitored and indexed) through System>Preferences>Search and Indexing.
