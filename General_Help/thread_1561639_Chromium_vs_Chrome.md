### Post by afrowildo on 2010-08-26
Chromium is basically just Chrome minus Google's tracking software, Adobe Flash, and automatic updates. Chromium is indeed developed by Google and is intended to be the well from which Chrome draws it's source code.

Contrary to other posts, Chromium is perfectly stable and secure. I've used it for a while on Linux to check it out and never had a problem with it (I went back to Chrome because of the extra features Google have added).

As an aside, Chromium also come bundled with Google's sync software (open sourced) so you can sync your bookmarks, preferences, themes and extensions between browsers while maintaining good open source karma.
