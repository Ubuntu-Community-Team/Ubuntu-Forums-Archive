### Post by Paddy Landau on 2012-03-01
> **adithya25 said:**
> Using Ubuntu.

Not Sun Java.
Ubuntu is an operating system, for example Windows, OSX (Apple Macintosh), Android, Unix, Linux (Ubuntu is a version of Linux).

Java is a way to run some types of programs, and works equally well (or should do so) on all operating systems.

On Ubuntu, there are two primary ways of running Java. The default is called Icedtea, and the other is Sun Java.

To find out which Java you have, open Ubuntu Software Centre, select *Installed Software* (on the left), and search for "java". Do you see "Icedtea" or "Sun Java"?

You should also see "Ubuntu restricted extras". If you do not, select *Get Software* (on the left), search for *Ubuntu restricted extras*, and install it. You may have to log out and in again.
