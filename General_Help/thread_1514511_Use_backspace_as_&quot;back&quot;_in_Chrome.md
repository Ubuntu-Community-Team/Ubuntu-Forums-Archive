### Post by Rubi1200 on 2010-06-21
Apparently, this is default on Ubuntu; i.e. the backspace function.

I don't know about Chrome, because I don't use it, but you could try this:

Open a new tab and type about:config in the address bar; say yes to any warnings and then search for an entry called **browser.backspace_action**

If you find it, set it from 1 to 0 and restart; should work.

If not, then create a new integer by right clicking and entering **browser.backspace_action **
set it to 0 as above.

Hope this helps.
