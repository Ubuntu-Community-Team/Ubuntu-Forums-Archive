### Post by chrishere01 on 2007-10-22
So I have contributed my last couple of days finding the way to installing steam on to my 64 bit Gutsy Gibbon Linux Box
I tried many different methods, until, I narrowed the problem down and found the alternative source for installation.

1. Get Wine- I used synaptic suggested from a friend from the forum, it work the best. I also install the development files.

2. Get Tahoma font. Which can be found [here]("http://fonts.appliedlanguage.com/tahoma_font.shtml"). When done downloading the fonts. Place them into ~/home/home/.wine/drive_c/windows/fonts

3. Download STEAMINSTALL**.EXE** which can be found [here]("http://www.ausgamers.com/files/details/html/10694").

4. Run the .exe file using wine emulator, update, and you're good!

I hope this helps!
