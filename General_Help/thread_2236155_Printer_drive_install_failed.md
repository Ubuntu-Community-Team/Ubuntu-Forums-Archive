### Post by coldraven on 2014-07-25
I'm not if this will work but it is worth a try.
Using your browser go to [http://localhost:631/admin](http://localhost:631/admin)
Then click "Manage Printers"
If you see the printer listed then click it, then from "Administration" select "Delete Printer". That might clear up any partial install.

If that worked you could go back to the admin page and select "Add Printer".
