### Post by Mr_JMM on 2008-11-22
Another web editing thread?

If you want a WYSIWYG editor then Quant-Plus is IMO much better than Bluefish. Quanta-Plus will also allow you to work with remote and Local files with built in FTP.

GIMP is an excellent replacement for Photoshop and as has been pointed out in another thread, you can try a version of Gimp called GIMPshop that has been designed to very closely emulate Photoshop look and feel. So much so that Photoshop tutorials can be followed almost to the t.
