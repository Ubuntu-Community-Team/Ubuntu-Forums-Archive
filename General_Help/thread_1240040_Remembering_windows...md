### Post by JillSwift on 2009-08-14
You can do this if you're using Compiz.

You'll need the advanced configuration app:
Applications menu -> Add/Remove... -> Search for CCSM and install.

Once installed, open it:
System menu -> Preferences -> Advanced Desktop Effects Settings (or CompizConfig Settings Manager)

You will find there, under the header "Window Management" a plugin called "Place Windows" which you can click on to configure. Under the tab "Fixed Window Placement" you can define a new rule to place Firefox (or most any window) where you want it.

HTH, don't bump so quick ;)
