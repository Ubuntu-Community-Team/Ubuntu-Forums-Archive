### Post by 3rdalbum on 2010-05-25
> **Chrissib said:**
> Hi, I am just starting out, and working through tutorials obtained on disks. As instructed, I was told to download an ISO image of V6.10, and burn on to disk, but when visiting the web site, I was only offered V10.
 
OK so I downloaded this, assuming it's just a more up to date version. However, the size of the ISO image wont fit onto the 700MB available on my CD Rom, so I cannot continue.

Make sure you're using the "Burn Image to Disc" or "Burn ISO Image" function of your CD recording software. NOT just "data disc". If you're using the correct image burning function of your software and you get this error, then your CD burning software is brain-dead and does not know how to count. The Ubuntu ISO is 699 MiB, and CD-Rs can hold 700 MiB.

Also, if your tutorials are designed for 6.10 (which is an ancient version), then you should just discard them. Ubuntu 10.04 is to 6.10, what Windows 7 is to Windows 98.
