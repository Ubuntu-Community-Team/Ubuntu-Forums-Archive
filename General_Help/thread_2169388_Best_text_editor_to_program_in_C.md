### Post by leg on 2013-08-22
Firstly there is no such thing as "The best <enter ignition for flame here>" for anything best is usually a personal choice. However you do have some options:

[Anjuta]("https://projects.gnome.org/anjuta/") is an IDE that you can use with GCC. I haven't used it for a long time but the last time I did it was a respectable IDE.

[Code::Blocks]("http://www.codeblocks.org/") Also allows you to use GCC and has templates for C/C++ projects.

However you did ask about editors and my personal favourite is [Geany]("http://www.geany.org/") and you can use GCC as a compiler through it also. Another is [Vim]("http://www.vim.org/"). Others will soon explain your many other options.

### Post by Impavidus on 2013-08-23
I always use vim (=vi improved) nowadays. It has syntax highlighting, parenthesis matching, autoindent and several other features. It can even run make (which calls your favorite compiler) and move the cursor to the first error encountered. Previously I used gedit. It also has syntax highlighting, but less features than vim. On the other hand, it's far easier to get used to.
