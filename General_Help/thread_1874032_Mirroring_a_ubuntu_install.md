### Post by sanderd17 on 2011-11-02
The easiest thing would be to attach both hard drives to the same computer, and clone one hdd to the other with clonezilla (best done via a live CD/USB).

If you want it on USB stick, you can use tools like remastersys, but they're not meant for that purpose.
