### Post by DaymItzJack on 2008-08-20
If you use Emerald:
Go to System -> Preferences -> Emerald Theme Manager
Select the "Edit Themes" tab, select the "Frame/Shadows" tab, on the right side change "left" "down" and "right" to higher values. This might not be the most correct way, but it works.

### Post by GeorgeMurango on 2008-08-20
also, if you're not using emerald, just the regular settings -> appearance menu, then different themes have different frame widths. I had the same problem and the only fix without emerald is changing the theme.

Thanks guys.

I also found another way if you use compiz which is to use the keyboard shortcut, mine is Alt + middle mouse button but it's configurable in compizconfig settings manager -> Resize Window -> Bindings -> Initiate Window Resize.

That way any part of the window can be grabbed for resizing.
