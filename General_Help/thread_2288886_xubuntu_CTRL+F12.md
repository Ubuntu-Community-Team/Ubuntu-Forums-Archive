### Post by jeremy51 on 2015-07-30
Hello,

I have a specific application that uses CTRL+F12 to strike through a patient progress note. When using Xubuntu the keyboard does not seem to respond to or issue the char CTRL+F12. I can use SHIFT+F12 no problem. Anyone know how this could be fixed or am I SOL?

Thanks,
J

---

### Post by Toz on 2015-07-30
Hello and welcome.

If you go to Settings Manager >> Window Manager >> Keyboard tab and scroll down to the "Workspace 12" entry, you'll see that Ctrl+F12 has been pre-assigned to this action. Select the "Workspace 12" entry and press the "Clear" button. This should free the keyboard combination so it can be used in your app.
