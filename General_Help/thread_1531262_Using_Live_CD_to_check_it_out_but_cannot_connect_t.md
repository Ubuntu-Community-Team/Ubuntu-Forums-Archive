### Post by ubunterooster on 2010-07-14
Your laptop uses an intel wireless chipset that should work with Linux (WLAN: Intel PRO/Wireless 3945ABG dual-band tri-mode 802.11a/b/g)

First thing I would try to do is connect it through wired ethernet and see if there are any drivers for the wireless chipset (via the "Hardware Drivers" utility)
