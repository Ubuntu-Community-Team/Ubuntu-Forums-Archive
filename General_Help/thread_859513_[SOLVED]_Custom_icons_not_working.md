### Post by sayakb on 2008-07-14
Download custom icon packs from [http://gnome-look.org/](http://gnome-look.org/)
The file should be .tar.bz2 or .tar.gz
Easiest way is to extract the file (Right click and select extract), and copy it to **~/.icons** folder (.icons is a hidden folder in your Home folder. Press Ctrl+H at your home folder to make it visible)
After you copy it to ~/.icons, goto **System->Preferences->Appearance**, click **Customize** button and **Icons** tab and select it from there.
