### Post by ad_267 on 2008-10-26
To add a launcher to AWN you can drag and drop a launcher from the desktop or the menu onto the dock.

Applications in Linux usually have an executable file (that doesn't have any extension) in /usr/bin. Launchers in GNOME are .desktop files that say where to find the executable and what icon to use, what the application is called and a description. These are usually in /usr/share/applications
