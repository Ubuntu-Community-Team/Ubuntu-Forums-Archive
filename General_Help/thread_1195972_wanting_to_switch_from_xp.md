### Post by milton1 on 2009-06-24
Please take no offense at this, but you have a lot to learn about the nature of a different operating system before you switch. The drivers for ubuntu will be completely differnt from those on windows. Ubuntu (linux) replaces windows as the operating system (unless you dual-boot, in which case you can run only one at a time). I strongly suggest you download and burn one of the live cd's first, so you can boot from that and get a feel for ubuntu, and if you like it or not, before you try installing it.

In **very** basic terms, no software you have for Windows will work on a Linux (Ubuntu) machine. You will need to go to Linux drivers (of which there are many) and Linux software. Unless you have some really exotic hardware I wouldn't worry about driver issues. 

In slightly more complicated terms: you will need to find & install Linux versions of the software you're using. You can also use a program like wine, which will allow you to run Windows programs, but with varying degrees of success.
