### Post by grahammechanical on 2021-08-31
If you can find somewhere to download an application's APK file from, then install Google Chrome web browser and then the Chrome ArcWelder  extension. The idea behind ArcWelder is to allow developers to code and test Android apps when using the Chrome OS. But ArcWelder as a Google Chrome Web browser extension will also run an Android APK application. I know this to be true. I have done it.
