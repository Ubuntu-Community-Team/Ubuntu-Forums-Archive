Hi Floyd etc. Yes I'll keep it as simple as possible. I'm a newbie and haven't much of a clue.  I have Ubuntu 6.10 on two boxes. One is my main one, which has Java 1.0.5 on it, and Frostwire ran well straight away. OK. So I set up the other U 6.10 box for my Mum. No way could I get Frostwire to run with Java 1.0.5.(!)  In the end I went to [http://daveshuck.instantspot.com/blog/index.cfm/2007/2/7/Installing-Java-6-16-in-Linux](http://daveshuck.instantspot.com/blog/index.cfm/2007/2/7/Installing-Java-6-16-in-Linux) and followed instructions to the letter. Java 1.0.6 is a 60 meg download. It worked perfectly and now Frostwire runs well. Maybe save yourself the big download and look at Dave's instructions re checking Java version especially if you reckon you already have 1.6.0.

I'll never know. This is all a bit tricky but worth a long sigh of relief when IT WORKS (eventually)
