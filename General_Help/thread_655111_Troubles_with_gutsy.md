### Post by zipperback on 2007-12-31
> **the_sorrow said:**

There is no codec that works for "all music and video".

You have to install compatible codecs for whatever file format you want to work with and software for them.

System -> Administration -> Synaptic Package Manager

Use the search term:  codec

It will pull up several items which you are probably looking for.
