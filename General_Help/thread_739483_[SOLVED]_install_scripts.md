### Post by Claus7 on 2008-03-29
Hello,

Judging from the beans you have, it seems to me that you have some experience with ubuntu and linux in general and that you know the ways in order to install a program.

From version to version the installation of a program becomes even more easier. Anyway. The real problem lies on what you want to install. Synaptic and debian packages or even automatix make things very easy. Easier than might be a script or at least equal! So you might have to search there first.

You can create your own scripts in case you do something that is repeated all the time and you want to automate such a process either this has to do with installing some things or making your life easier with tasks you do every day. For example I have come accross amsn scripts that most of them are version specific. In my case these scripts didn't work.

I say once again that it depends on the needs you have and sometimes on the version of your operating system. For example a library that might be missing in one version, in the other might be installed by default and the script might not take into account this parameter and as a result the installation will fail.
