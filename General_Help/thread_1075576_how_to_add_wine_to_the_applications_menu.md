### Post by donkyhotay on 2009-02-20
This isn't really a wine issue and this is proper forum for this question, in order to add/remove any application to the menu system you can either go to 
preferences > main menu
or from the terminal run
```
alacarte
```
both of which run the (same) program that allows you to edit your menu system.
