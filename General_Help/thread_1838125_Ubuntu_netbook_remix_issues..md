Also, if before i do anything else, i click one of the large icons on the left, i can use the OS like normal and the netbook UI will not die if i open and close apps. Both these issues have consistently been problems across multiple computers and multiple editions of ubuntu. 

If this is not posted in the correct space, feel free to move it, and i will provide any information requested about my current install. If these bugs could be eliminated, my current ubuntu OS will basically be perfect.
