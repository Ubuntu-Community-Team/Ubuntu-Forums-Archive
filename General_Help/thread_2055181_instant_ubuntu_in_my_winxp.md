### Post by drmrgd on 2012-09-08
Depending on your computer resources (CPU and memory in particular), you could run Ubuntu in a virtual machine on top of Windows.  You might have a look at Virtual Box or VMware Player, which are the top two.  If you have an old computer with a CPU that does not support virualization, or if you don't have enough RAM, though, it's not going to run very well.  You may also notice some other lag and issues running the virtual machine.  But, assuming your computer can handle it, it's a surefire way to run Ubuntu on top of Windows.
