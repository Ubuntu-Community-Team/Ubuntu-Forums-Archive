### Post by PowerPatrick on 2010-04-15
I found out about it yesterday, and I've already submitted it to some blogs and news sites. :)
[http://lifehacker.com/5517862/teamviewer-remote-desktop-tool-available-for-linux](http://lifehacker.com/5517862/teamviewer-remote-desktop-tool-available-for-linux)

It's kinda ironic when you say "no more WINE", because this edition is actually based on Wine, but are recompiled with wrappers that provides the desktop integration and uses the screen capture functionality, which makes it able to create a Linux host for remote control. Though earlier in the Windows edition without the wrappers and stuff under Wine, you could only achieve to remote control Windows and Mac guests.
