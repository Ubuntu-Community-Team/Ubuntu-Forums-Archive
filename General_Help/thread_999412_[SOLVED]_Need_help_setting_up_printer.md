### Post by iAndrew on 2008-12-02
Alright I got it working.

I looked into this for a few minutes then I found that there was a need for a ppd file.

I found out (on windows machine) that there are different levels of postscript.

I found the one on my printer (which was level 2) then I looked into which had been drivers were available (in system > printing I think) then I checked which printer drivers had corresponding postscript levels.

I'm not 100% this works for everyone.
