### Post by PmDematagoda on 2007-09-29
Actually Synaptic would tell you that Pidgin was the latest version if the version of Pidgin found at the repositories is the same as the one you already have. So unless you add a repository with the latest version of Pidgin, or the repositories get the latest version, Synaptic would keep on telling you the same thing, that Pidgin is up-to-date.
