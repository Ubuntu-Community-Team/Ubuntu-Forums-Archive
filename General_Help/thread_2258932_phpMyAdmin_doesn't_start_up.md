### Post by mcduck on 2014-12-31
Once installed, phpmyadmin will work without you having to start it. Just open your web browser and type "localhost/phpmyadmin" in the address field.

Of course you'll still need to have installed all of the LAMP stack (Apache, PHP & MySQL)  to make use of phpmyadmin.
