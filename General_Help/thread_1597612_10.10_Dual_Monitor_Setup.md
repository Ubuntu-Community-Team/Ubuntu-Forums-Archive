### Post by cojon on 2011-01-05
I just upgraded from 10.04 myself and found the dual monitor problem a bit perplexing. I went through the same disturbances as the rest of you and possibly more as I completely lost my alternative KDE desktop which I have yet to restore. I am backed up, so I have my fingers crossed. 

Apparently I located the same bug report as others, and in case you need the fix I used, here it is. It works perfectly . . .  

I have an older Nvidia 1 Gb card (GT 220) and first I upgraded the driver by entering the following in terminal:

sudo apt-get install nvidia-current
sudo nvidia-xconfig

(installs Nvidia driver v. 260.19.29) 
		
Then you will need to Log out and back in or reboot.
-------------
Then solve the dual monitor problems everyone is having by installing xorg-edgers. The method is to Install the ppa from xorg-edgers and update the packages. The xorg-edgers ppa page is:  [https://edge.launchpad.net/~xorg-edgers/+archive/ppa](https://edge.launchpad.net/~xorg-edgers/+archive/ppa)

They suggest: "*IMPORTANT NOTICE* - If you are upgrading from one release to another and are using this PPA, be sure to install ppa-purge and use it to downgrade all of this PPA's packages before the upgrade or you will have a broken system." 

To fix the dual monitor problem (if necessary, after following the advice above) type this in terminal:
         
sudo apt-add-repository ppa:xorg-edgers/ppa
sudo apt-get update
sudo apt-get upgrade

Then logout or restart

That did it for me and from what I have learned from a bit of reading, that should fix the problem for the bulk of those with similar dual monitor problems.

Yes, that is the right file. However, having 2 separate X displays and "TwinView" are mutually exclusive options. And unless you have a very specific setup in mind, you most definitely want the TwinView option as it adds another monitor to your current desktop.

I noticed that the nvidia-settings program is attempting to bloat the configuration by adding unnecessary sections, this is my configuration for dual screen (xorg.conf):
```
Section "Screen"
        Identifier      "Default Screen"
        DefaultDepth    24
        Option         "TwinView" "1"
        Option         "metamodes" "DFP-0: nvidia-auto-select +0+0, DFP-1: nvidia-auto-select +1680+0"
EndSection

Section "Module"
        Load    "glx"
EndSection

Section "Device"
        Identifier      "Default Device"
        Driver  "nvidia"
        VendorName     "NVIDIA Corporation"
        BoardName      "GeForce GTS 250"
        Option  "NoLogo"        "True"
EndSection
```

Note that the VendorName and Boardname options are redundant, but I like pretty names in other programs and logs :)
