### Post by geirha on 2009-02-26
compiz is installed by default. What setting is it set to under System -> Preferences -> Appearance -> [Visual Effects]?

If you want the more advanced configuration tool, you'll need to install the package compizconfig-settings-manager. Read here to see how you can use synaptic to create a download script that you can copy to a computer with internet access. [https://help.ubuntu.com/community/Synaptic/PackageDownloadScript](https://help.ubuntu.com/community/Synaptic/PackageDownloadScript)
