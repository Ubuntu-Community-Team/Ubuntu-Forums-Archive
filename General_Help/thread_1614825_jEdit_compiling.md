### Post by GregBrannon on 2010-11-06
There is a Programming Talk area of the Forum, a sub-forum, that discusses these programming details.

A compiler turns source code, usually saved in a text file, into something the computer understands.  Examples are g++, javac, etc.

An Integrated Development Environment (IDE), programmer's editors, text editors, can be used to edit the source code.  Examples are Eclipse, Visual Studio (Windows only), NetBeans, Geany, Emacs, Vi, etc.

I think you're asking how to set up a programming environment, maybe including the compiler, but mostly the editing part.

To ensure you have what you need for programming in Java, install the JDK which will include the JRE.  Find detailed instruction on how to do that with a simple Google search like, "ubuntu install java" or 'jdk' instead of java.

As for which editor to use, this topic has been discussed to death in the Programming Talk sub-forum.  Search it for 'ide'.
