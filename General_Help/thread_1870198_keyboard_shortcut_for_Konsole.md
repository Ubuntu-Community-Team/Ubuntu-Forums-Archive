### Post by shva on 2011-10-26
I am using Kubuntu 11.10. As a KDE newbie, one problem I had was I couldn't set a customized keyboard shortcut. For example, in Ubuntu GNOME, I used to set Ctrl+Shift+T to open up a terminal emulator. However, I found it hard to do in Kubuntu. After a few googlings, I opened "khotkey" and add a "globe keyboard shortcut" of "Command/URL" mode. The "trigger" was set to "Ctrl+Shift+T" and "action" to "/usr/bin/konsole". But still it didn't work. Could anyone please help? Thanks.

---

### Post by dniMretsaM on 2011-10-27
I did the following and it works for me:

System Settings -> Shortcuts and Gestures -> Custom Shortcuts -> right-click -> New -> Global Shortcut -> Command/URL -> type a name (i.e. "Terminal") -> Trigger -> click on None -> press Ctrl+Shift+T -> enter "konsole" for Command/URL:
