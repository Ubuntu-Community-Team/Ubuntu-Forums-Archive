You need to have both installed. Redshift is what actually changes the screen, gtk-redshift gives you the user menu. If you don't think it's working, try running gtk-redshift in the terminal and see what it says. 

The name is misleading. Redshift makes (more or less) the same changes as f.lux. What both programs do is change the screen temperature. Depending on how low you go, your screen gets an orangey tint. If you go lower, it turns red.

### Post by olborj on 2016-01-26
I had it installed on my old netbook with Ubuntu 14.04, and when I tried to install it to my desktop computer (release 15.04), i found the same problem.
What i did to fix it: Software & Update-> Other software->Select [http://ppa.launchpad.net/kilian/f.lux/ubuntu](http://ppa.launchpad.net/kilian/f.lux/ubuntu) vivid main ->Edit-> Change "vivid" to "trusty" -> Save.

I know the software version is not meant for this ubuntu release, but i have been using f.lux for some time now without any problem.

And then installing:

sudo apt-get update

sudo apt-get install fluxgui
