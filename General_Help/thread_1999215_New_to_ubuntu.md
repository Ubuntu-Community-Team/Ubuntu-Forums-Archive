### Post by ExSuSEusr on 2012-06-07
If you are new - I would suggest you give** PlayonLinux **a try.

You can find it in your Software center.  It will insall Wine and Wine Tricks and everything you are going to need for WOW and D3.

I will tell you now, WOW downloading has some issues.  If you have the CD's you should be ok.  The downloading issue is NOT the fault of Linux - Blizzard has some problems - it is also affecting Apple users too.  

But certainly give PlayonLinux a try first.

It really does make life a LOT easier for new users.

Actually - let me give you a few tips (you may already know some of this, but anyway):

1. Go into System Settings --> Additional Drivers --> make sure that the "recommended" driver for your video card is ACTIVE.  It is not always automatic after an install.  If you don't have it active the game won't even load.
2.  Don't worry about Wine.  Just install PlayonLinux.  It will install everything you need for the game include MS Core Fonts.  
3.  If you are going to count on a digital download  (WoW) - you probably wont get very far.  For me and many other Linux / Apple users the install gets to about halfway and times out.
