### Post by Legace on 2009-06-06

> I've a few directories in my /home folder but Gnome does not show them under Places. Why ?

Add those folder to Nautilus bookmarks:

Browse to the folder you want to be listed in the Places menu with Nautilus, and select **Bookmarks -> Add Bookmark** and you're done :)
