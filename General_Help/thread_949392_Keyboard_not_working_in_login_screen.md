### Post by myxiplx on 2008-10-16
It might not be this, but only recognising some keys is often just a Ctrl key stuck.  When this happens only the keys that don't have a ctrl command mapped will type.

Try just tapping each of the ctrl, shift and alt keys on your keyboard and see if that helps.
