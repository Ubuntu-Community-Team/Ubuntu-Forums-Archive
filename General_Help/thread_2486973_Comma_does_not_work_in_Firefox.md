### Post by sudodus on 2023-05-18
You mention Ubuntu and Firefox. I guess it means that you are running Ubuntu Desktop with the Gnome desktop environment.

- What computer is it (brand name and model)?
- What keyboard is it (built into a laptop or separate; if separate, how is it connected)?
- Can you check that the keyboard *hardware* works; if yes, *did* you check it?
- Are you running Wayland or Xorg (x11)?
- Can you use comma with some other keyboard setting, for example set via the command **[FONT=Courier New]setxkbmap us[/FONT]** in a terminal window?
- Is it a fresh installation, where comma never worked, or did it work, and then stopped working?
- Does comma work in a *live* Ubuntu 22.04.2 LTS system (booted from the installation USB drive)?
