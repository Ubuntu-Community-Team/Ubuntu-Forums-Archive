### Post by andreasis on 2008-12-08
Hi T4nkFr4nk, and welcome to the forum,

There is a link on that website 
[http://art.gnome.org/download/themes/gdm_greeter/91/GDM-Cracked-Windows.tar.gz](http://art.gnome.org/download/themes/gdm_greeter/91/GDM-Cracked-Windows.tar.gz)

Just download this file to your desktop (do not extract it).

Go to the upper panel > System > Administration > Login Window

Now drag the file that you downloaded from the deskop, and drop it onto the 'Login Window Preferences' window.

That should be it... was that what you wanted?
