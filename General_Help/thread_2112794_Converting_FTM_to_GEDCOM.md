### Post by forumdoctor on 2013-02-07
I was actually able to find a way around this issue.

I learned that Genbridge was a component that could read the binary of the FTM file and convert it into another format well.

However I needed to find a program that used this proprietary code. after some research was done I found that The Master Genealogist uses Genbridge. So I downloaded the 30 day trial and installed it on my windows partition. I used it to convert my FTM file directly into Gedcom.

I hope this helps someone.
