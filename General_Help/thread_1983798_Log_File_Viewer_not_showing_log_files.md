### Post by HiImTye on 2012-05-20
Log File Viewer displays no logs by default
you will need to add them manually by opening them in /var/log/ (or the subfolders therein)
there's no danger in viewing those files as they are not owned by the user (and even if they were the biggest danger would be losing a log)
