### Post by andreanadelphon on 2009-02-02
Shift + space wasn't working for me either, but it seemed like it only started after I added multiple keyboard layouts, so I looked in the Keyboard Layout Options and found what fixes it!

System > Preferences > Keyboard > Layouts > Other Options > Using space key to input non-breakable space character

It was on "Default", but I changed it to "Space key outputs usual space at any level", which seems to work fine.  The other ones might work too; I'm not really sure what they mean.
