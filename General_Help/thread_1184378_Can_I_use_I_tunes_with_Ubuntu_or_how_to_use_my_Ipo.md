### Post by hellocatfood on 2009-06-11
I've only ever use [Songbird](http://getsongbird.com/) to sync my ipod. You can download an easy install package from [http://getdeb.net](http://getdeb.net) and search for Songbird. 

Additionally, if you go to Applications > Add/Remove you can search for "ipod" there some software listed there.

7digital is a music store integrated into Songbird, but you can also get free (CC licenced) music from [Jamendo](http://www.jamendo.com/en/)
