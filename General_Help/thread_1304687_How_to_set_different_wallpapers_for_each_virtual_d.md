### Post by hwttdz on 2009-10-29
There's a workaround that allows you to do this using the compiz wallpaper plugin and disabling desktop painting in nautilus.  The downside is that you will lose your desktop icons (and ability to right click on desktop). Disable desktop painting in gconf-editor then setup wallpaper plugin in compiz.
