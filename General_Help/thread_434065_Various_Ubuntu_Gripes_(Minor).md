### Post by iamajd on 2007-05-05
I suppose this is the correct place for this post.  I love Ubuntu (now that I got Nvidia and Feisty working together).  I've been an Ubuntu fan for nearly two years, but have only recently completely transitioned off of Windows.  I do, however, have a odd OS hobby... so I have several installed.  And having all these options makes me notice a few... minor irritations... with Ubuntu.  So, here, I list them (in no particular order)!

[LIST]
[*] Requires the "acpi=off" kernel option, although no other OS needs that (including FreeBSD).
[*] Feisty/Nvidia compatibility requries "irqfixup" though it was never needed before & isn't needed by other OSes.
[*] When rebooting Ubuntu, BIOS performs full "Memory Testing" while no other OSes do (perhaps ACPI related??).  Requires waiting for it to finish or pressing ESC.
[*] Hardware "Automagically" (and apparently randomly) stops functioning: IDE DVD/CD Drive, USB Printer, PS/2 Mouse
[*] Swap space double mounted (*FIXED* by using /dev/???? instead of UUID for swap entry in fstab)
[*] Slow boot items: "Waiting for Root Filesystem", "Reading Files Needed to Boot", HALD
[*] Why is splash text [COLOR="Blue"]Blue[/COLOR] in Verbose Mode (i.e. no "Quiet")??
[*] [File Roller now lacks Drag-n-Drop](http://ubuntuforums.org/showthread.php?t=399780)
[*] Main menu gets unnecessary scroll arrows (Up/Down) which go away on next showing (covered in many threads).
[*] Main menu sometimes requires logout/login to reflect changes made with alacarte.
[*] Run in terminal / Display prompt when opening plain text documents... just gets old.  Just display them by default, require a right click to run or something... 
[/LIST]

I know many (most? all?) of these issues aren't Ubuntu specific (Debian, Gnome, etc).  And I hope it doesn't sound like I'm being too critical.  I absolutely adore Ubuntu and prefer it to any other OS regardless of these minor quirks.  I'm just kinda posting this to share my thoughts and perhaps to see if anyone has solutions or explanations for this stuff.  I know a "1 problem per post" system is preferred, but since these are all so minor and I can live and function just fine with them, I thought none of them deserved their own thread.

Keep up the good work team!  Looking forward to Gibby or whatever its called.  :)
