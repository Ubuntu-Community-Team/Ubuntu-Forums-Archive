### Post by mcduck on 2011-02-18
Nothing weird there, Adobe updated the Flash plugin and it's now caching the flash video files in the same place where all the other cached data from the browser goes.

If you use Firefox, you'll find the video file in ~/.mozilla/firefox/*profilename*/Cache/.
