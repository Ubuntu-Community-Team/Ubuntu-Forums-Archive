### Post by Dr. Nick on 2005-11-09
I think you can use the same busid that I have as mine is agp aswell. Replace your Busid, change the driver from "ati" to "nv" and change the identifier to what ever you want.

You can do this a few ways. If you boot all the way into the Gui just open a terminal and run "sudo gedit /etc/X11/xorg.conf", If your at a command line run "sudo nano /etc/X11/xorg.conf" and then save the changes and restart. After you do this though I dont know how to use the ati card for anything else. But atleast you will be using the nvidia one.
