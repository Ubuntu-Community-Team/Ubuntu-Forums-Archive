### Post by mula on 2004-12-14
All I did was: "File -> Import -> Import a single file". Then I just browsed to the folder where Thunderbird keeps all the mail and chose "Inbox" (the one with no extension). Worked like a charm for me.
I previously used Thunderbird in Windows but I guess there shouldn't be any difference even if you import your mail from a Linux version.

### Post by adbak on 2004-12-14
The Ubuntu devs, due to time constraints I believe, shipped Evolution with different kind of spam blocker instead of SpamAssassin.  This should be rectified in Hoary.  Also, Evolution also supports Palm pdas, has a calendar, and looks more like MS Outlook.  Thunderbird slightly resembles it's cousin, Firefox, and is just an email program (whereas Evolution is an email suite).

What it boils down to is your tastes.  Try them both and see which one you like better, then just uninstall the other.
