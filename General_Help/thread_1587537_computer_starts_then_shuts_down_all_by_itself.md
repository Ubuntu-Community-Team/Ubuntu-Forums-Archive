### Post by HereInOz on 2010-10-04
I would have a go at replacing the power supply initially.  If that doesn't fix it, then it is hair pulling time.

Next step is to grab an old hard disk, if you have one, and run up a fresh install on it.  If that works, then it is a software issue.  However, if it still bombs out, then it is looking a little like a motherboard/RAM/adaptor card or similar.
