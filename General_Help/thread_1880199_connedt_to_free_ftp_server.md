### Post by Lars Noodén on 2011-11-13
If you want a graphical client, then look at FileZilla.  If you need a text client, then look at ncftp.  

If you are planning on setting up a server, remember that [FTP is insecure](http://olex.openlogic.com/wazi/2011/stop-using-ftp-how-to-transfer-files-securely/) and that you should be using SFTP instead.
