### Post by brucehohl on 2011-09-20
You could do this:

Thread Tools > Show Printable Version

If multiple pages then click:
Show XX post(s) from this thread on one page

Now print to file (PDF).

### Post by Rasa1111 on 2011-09-20
bruce gave a good idea..

or, you could simply go to your "File" menu, in your browser. 
Then "Print"
when the print window comes up, choose "Print to file", and check the "PDF" option, Name it, then click print.
 and it will save as a PDF to your computer, wherever you told it to print/save to.
