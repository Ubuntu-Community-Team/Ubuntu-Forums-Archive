### Post by mb_webguy on 2009-04-19
There are a number of games designed for Linux, btw.  Wine is only for running applications designed for Windows on Linux.  And Windows games using certain kinds of 3D graphics don't do so well in Wine.

If you like FPS and 3d action/combat games, you might want to check out [Nexuiz]("http://www.alientrap.org/nexuiz/") (FPS), [America's Army]("http://www.americasarmy.com/") (FPS developed by the U.S. Army), [Tremulous]("http://tremulous.net/") (FPS), [World of Padman]("http://www.worldofpadman.com/") (cartoon-themed FPS), [Alien Arena]("http://icculus.org/alienarena/rpa/") (FPS), [Urban Terror]("http://www.urbanterror.net/news.php") (FPS), [Warsow]("http://www.warsow.net/") (FPS), [TrueCombat: Elite]("http://www.tcelite.net/") (FPS), [Excalibur: Morgana's Revenge]("http://emr.excaliburworld.com/") (FP action/adventure), [AssaultCube]("http://assault.cubers.net/") (FPS), [Savage 2: A Tortured Soul]("http://savage2.com/en/main.php") (FPS/RTS), [Blob Wars: Blob and Conquer]("http://www.parallelrealities.co.uk/projects/blobAndConquer.php") (3D action/adventure), and [Yo Frankie!]("http://www.yofrankie.org/") (3D adventure with Pixar-quality graphics created to show off Blender's game engine).

Some of these are in the Ubuntu repositories, and you can install them by clicking the links below:
[Nexuiz]("apt://nexuiz")
[Tremulous]("apt://tremulous")
[Alien Arena]("apt://alien-arena")
[Warsow]("apt://warsow")

A number of commercial game developers release Linux versions of their games, including [Enemy Territory: Quake Wars]("http://www.enemyterritory.com/") (FPS), [Vendetta Online]("http://www.vendetta-online.com/") (3D space combat MMORPG), [Neverwinter Nights]("http://www.vendetta-online.com/") (RPG), and [Darwinia]("http://www.introversion.co.uk/darwinia/") (???).  Some companies like [Linux Game Publishing]("http://www.linuxgamepublishing.com/") port games previously released for Windows to Linux.

If you [Google "linux games"]("http://lmgtfy.com/?q=linux+games") you'll find a lot more.
