### Post by Mighty_Joe on 2009-06-11
Jython is a Java implementation of the Python language.  It is used to facilitate the use of Java packages in Python programs and to run Python scripts in Java programs.  It has nothing to do with "executables".
[Jython FAQ]("http://wiki.python.org/jython/JythonFaq/GeneralInfo")
