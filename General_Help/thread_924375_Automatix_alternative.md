### Post by mb_webguy on 2008-09-20
I agree with Nepherte.  You can install any of the programs available through Ultimatix with a single command in the terminal (at least once you enable the Medibuntu repository, which takes two commands, I think).

Also, at least some of the software Ultimatix installs are beta versions, which I don't necessarily want on my system.  For example, Ultimatix installs Amarok 2, which is currently in its first beta release.  I'd rather stick with the current stable release.  Also, Amarok 2 isn't available as a deb from the Amarok website or any of the usual places like GetDeb, so I don't know exactly where Ultimatix installs these applications from unless it uses its own repository.  I don't like not knowing where my software comes from.  Linux may not get viruses, but it's certainly capable of running malicious code.
