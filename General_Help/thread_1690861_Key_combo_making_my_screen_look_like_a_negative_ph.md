### Post by hawthornso23 on 2011-02-19
Compiz has a plugin that will do this, the `negative' plugin. The keyboard commands to toggle it by default are 

<super> m
<super> n

One toggles the screen while the other toggles a window.

If this turns out to be the problem the negative plugin can be disabled via compizconfig settings manager.
