### Post by hmw38 on 2011-08-03
I installed BasKet because it looked like  a great tool for academic work, even on Ubuntu, though I guess the KDE versions are much better.  It would be perfect for my needs except for one thing: I can't figure out how, if at all, to get files in baskets to sync with their originals in other folders (e.g. Dropbox).  Can anyone tell me how to do this, or how to figure out an alternative way to get the same functionality?  At the moment if I edit file abc.odt in Dropbox, file abc.odt in BasKet does not show the edit, and vice versa; obviously when it was imported to the basket it became a new file.  I guess I could just import everything to BasKet and keep the BasKet files themselves in Dropbox to back them up, as I've seen recommended (and it's easy to do), but I like the idea of having separate access to the files outside of BasKet, so that if I'm on a machine which doesn't run BasKet I can still access them

A bit more about how I want to use BasKet: I'm using this for writing a PhD in the humanities, so its great attraction is being able to combine different types of file in one basket which functions as a subject heading.  E.g. in a basket called "Nationalism" I could have a PDF of an article I want to read, a .odt file containing the relevant chapter of my thesis, some short text notes of rough ideas, and a bullet point list of books I want to read.  I mainly use an eeepc running Lucid (not netbook remix) for my PhD, so I don't necessarily need to sync BasKet across multiple computers (although if that's easy I would do it with my other Asus which I occasionally use to write), but it would be good to use it as my main organising/thinking/editing tool while still keeping OpenOffice files in a format in which they can easily be accessed from any machine, as I often have to print from campus computing rooms, or look things up on public terminals in libraries.  In an ideal world it would work like this: in the example above I could open the Nationalism basket, read the notes, add a paragraph about those ideas to my thesis chapter and save, then go to another thematic basket, open the same thesis chapter there (because it's about e.g. nationalism *and* education), add some more writing to do with a PDF in that basket, and save, all in BasKet, then go to a public terminal  in the computer room, log into Dropbox on the web and print out the edited thesis chapter. The way it works at the moment, the chapter in Dropbox would not contain any of the edits made in BasKet.  How do I change this?

Sorry to be so verbose, but it's usually helpful to supply more information at the start, isn't it?!  Thanks in advance for any help anyone can offer.

### Post by hmw38 on 2011-09-27
Ok, I realised that files are accessible in the baskets folder from Dropbox, so can be printed from any internet-connected computer, and keeping the baskets folder in dropbox keeps them backed up.  I deleted the originals from their old locations and keep  them only in Basket so I have the most up-to-date version in there at all times.  It's not quite perfect, but it's a workable solution.
