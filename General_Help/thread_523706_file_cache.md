### Post by prince_niceguy on 2007-08-12
As such ubuntu OS does not cache files. the we browser fire fox does use cache and you can empty the same by clicking Tools --> Clear Private Data.

You can clear recent documents by clicking clear recent documents which is the last option in the recent documents list.
