### Post by TuxLyn on 2010-10-14
I've finally found workaround for the issue with DEL key on my mini keyboard. I thought I would share it here. What I did is used System -> Preferences -> Keyboard Shortcuts and then scrolled down to "Windows Management" and selected "Hide all normal windows and set focus to the desktop" then pressed my "DEL" key on keyboard to assign it to this function.

The reason I did this is that I always had issues with my "DEL" key. Whenever I would of used it in nautilus or desktop, it would never ask for my confirmation to delete something and would just delete it to recycle bin right away. It might not seem as a problem for most, since it deletes to recycle bin. When I work with files in nautilus for example and have my folders set to "List View" I would hit "DEL" key accidental lots of times and it would delete all kinds of folders without confirmation. Now since I've assigned my "DEL" key to do something else its much safer.

I hope this helps some one who has similar issues with there "DEL" key on keyboard.
