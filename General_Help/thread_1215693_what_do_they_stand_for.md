Xfce...*looks up on wikipedia* = I guess it is not an acronym...

EDIT:  K I guess Xfce does stand for something, but apparently that is no longer what the developers intend

[quote=Wikipedia][Olivier Fourdan]("http://en.wikipedia.org/wiki/Olivier_Fourdan") started the project in 1996. The name "Xfce" originally stood for "[XForms]("http://en.wikipedia.org/wiki/XForms_%28toolkit%29") Common Environment", but since that time Xfce has been rewritten twice and no longer uses that toolkit. The name survived, but it is no longer capitalized as "XFce", but rather as "Xfce". The developers' current stance is that the [initialism]("http://en.wikipedia.org/wiki/Initialism") no longer stands for anything specific.[/quote]
