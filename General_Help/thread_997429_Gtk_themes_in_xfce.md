### Post by thefiestysoldier on 2008-11-29
How do I apply Gtk themes from gnome-look on my Xfce Desktop?

---

### Post by Denestria on 2008-11-29
Unpack the theme into your .themes directory.  Example: I have the theme called BlackFate so it is in /home/denestria/.themes/BlackFate

Then open the settings manager > user interface and select the theme from the list.
