### Post by howefield on 2011-10-20
Not sure about doing that via the Launcher, but you can Alt - Tab to cycle through your open windows.

When highlighting the relevant icon with multiple windows, eg your terminal, the open terminals will auto preview.

If you are working on the same program you'll need one click to show all windows belonging to that program.. If you are working on another program (Writer for example) and you click on the firefox icon on the launcher that'll show the topmost open window in firefox first, and with another click you get a preview of all open windows in firefox.
