### Post by sanderd17 on 2011-05-15
customisation can be done using the remastersys program. That will compile any installation to an iso file, ready to install elsewhere.

As for what apps companies need. That depends on the companies. I can imagine that most companies need an ERP system, and I heard good things about openERP.

I think it would be better to give "support" to companies, using customised distro's than to give a customised distro. You should talk to the companies to know what they need exactly. Then, if you don't fine a certain thing, you could ask it here.

You know that almost everything in the Linux world is open source, so customization can be done without limits. In the worst case, you need to adapt the source-code and recompile it. But most applications give a big amount of customisation options.
