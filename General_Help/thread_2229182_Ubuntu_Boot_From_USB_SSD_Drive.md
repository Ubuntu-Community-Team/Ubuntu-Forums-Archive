### Post by sudodus on 2014-06-12
Not too badly. I run pendrives, HDDs and SSDs via eSATA and USB 3, and the performance is quite good with USB 3. I suggest that you set up some tests relevant for your particular applications to compare the speed when it is connected internally (SATA) and externally (USB 3).

Concerning trim, I'm not sure, but I guess it it independent of the USB interface.
