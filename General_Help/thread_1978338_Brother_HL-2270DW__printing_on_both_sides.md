### Post by willejos on 2012-05-13
If you go in your cups config you can change the default print options to not use duplex printing. Try:
Go to [http://127.0.0.1:631](http://127.0.0.1:631)
Then click 'Printers' top right (or [http://127.0.0.1:631/printers/](http://127.0.0.1:631/printers/))
Click the name of your printer e.g. 'HL2270DW' (or [http://127.0.0.1:631/printers/HL2270DW](http://127.0.0.1:631/printers/HL2270DW))
Click the 'Administration' drop-down and select 'Set Default Options'
In the 'Duplex' drop-down select 'None' then click 'Set Default Options'
Does that work?
