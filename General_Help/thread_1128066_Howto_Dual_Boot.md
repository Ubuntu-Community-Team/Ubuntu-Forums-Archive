### Post by _Purple_ on 2009-04-17
If you already have Ubuntu installed in a machine, you can make the bootable USB disk using System > Administration > Create e USB Start up Disk. You only need the ISO for that.

### Post by _Purple_ on 2009-04-17

[Click here](https://help.ubuntu.com/community/Installation/FromUSBStick)
