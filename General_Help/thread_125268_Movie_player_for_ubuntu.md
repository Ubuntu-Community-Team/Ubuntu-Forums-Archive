You might also want to search the forums to see how you can address your performance issues. For example on my laptop Ubuntu was always running in powersave mode, which throttled back my cpu.

### Post by hilltop_yodeler on 2007-02-21
I have had difficulties in the past with getting mplayer and xine to work properly, not to say that those players don't work well for other people.  What I have found works best for me is the VLC media player.  It will play just about any video or audio type, and it will do it right out of the box.  You can install it easily through Synaptic (just search for "vlc").  More info on VLC can be found here:
[http://www.videolan.org/vlc/](http://www.videolan.org/vlc/)

Also, you mentioned that Ubuntu is slow for you.  All of my computers tend to be older systems because that's what I can afford, and I run Xubuntu on (most) all of them.  Plenty fast.  Xubuntu uses the XFCE desktop management system which is lighter weight than Gnome or KDE.  It's nice having a light weight distro that doesn't come packaged with a lot of extra stuff that I might not use; for the most part, it gives me the choice to install only the stuff I really want and not have extra apps sitting on my machine taking up space (my opinion on most things is that less-is-more).  You can still install Gnome and KDE libraries in case you wish to run apps that were made to run under Gnome or KDE.  Anyway, I'm sure you have found out how awesome Ubuntu is, so if you feel like it runs a bit slow, check out Xubuntu.  Out of the many different distros that I've tried out, Xubuntu is by far my very favorite.
