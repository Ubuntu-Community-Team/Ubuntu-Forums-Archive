### Post by momist on 2008-03-10
Well, I'll reply to my own post.  I have found a work around.  Using "Print with Gutenprint" from within the Gimp, I can happily print anything I want onto a CD direct.  Simply select CD as the media type, and media source as "CD tray", and hey presto, it simply works.  Why I struggled to print directly, I can't say.
