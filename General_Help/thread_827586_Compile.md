### Post by spiderbatdad on 2008-06-12
Emerald is in synaptic package manger...precompiled and ready to install.
System>>Administration>>Synaptic Package Manager.

Compiling is building a package from source...which is usually a little easier than it sounds, but requires some tools, also pre-packaged in Ubuntu...like the package "Build Essential," for example.

Many how-tos are available should you choose to get into compiling software from third part sources:[https://help.ubuntu.com/community/CompilingEasyHowTo](https://help.ubuntu.com/community/CompilingEasyHowTo)

Or Google how to install anything linux: [http://monkeyblog.org/ubuntu/installing/#source](http://monkeyblog.org/ubuntu/installing/#source)
