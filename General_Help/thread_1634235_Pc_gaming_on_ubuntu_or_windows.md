### Post by Mark Phelps on 2010-11-30
If you're running Windows games, then Windows is most probably "better" -- because Windows will come with the video drivers needed for hardware acceleration that the games demand.  And, you don't have to mess around with Wine, or Crossover Games, or other such tricks in order to get them to work.

That said, some folks have had remarkable success running Windows games using Wine and other such stuff.  And, give the more enhanced performance of Ubuntu on the same hardware, will claim that Ubuntu provides a better gaming platform than Windows.

So basically, it varies -- by the hardware and drivers involved, and by the particular games involved.
