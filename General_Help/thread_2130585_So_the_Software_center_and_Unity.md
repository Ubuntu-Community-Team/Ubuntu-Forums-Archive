### Post by ManamiVixen on 2013-03-29
No, it isn't a bug. Its a feature to help users find the newely installed program. If you want to keep it on the "Unity Launcher", you have to right-click and click lock or after reboot, search for the program in the "Unity Dashboard" by clicking the Ubuntu Icon and typing the program name, then drag the program to the Unity Launcher and it will auto lock in place for you.
