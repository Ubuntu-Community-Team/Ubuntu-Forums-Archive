### Post by taurus on 2006-02-27
1.  It's up to you if you want to use Gnome or KDE.  Since you want to make it look like Mac (!!!), then perhaps KDE is a better solution and don't forget to check out [http://www.kde-look.org/](http://www.kde-look.org/) for some themes there...

2.  nVidia is probably more Linux friendlier than the other company, ATI.  So, you shouldn't have any problem with it.

3.  I don't think cedega will let you run Windows apps though.  If you want to run some Windows apps, see if you can find a replacement for Linux and if not, try running them through wine or CrossOver.

4.  If you don't have anything out of this world, then you shouldn't have any problem with Ubuntu.  To make sure, stick in the LiveCD and boot from it.  See if Ubuntu LiveCD finds everything in your machine.

5.  Good luck.  :)
