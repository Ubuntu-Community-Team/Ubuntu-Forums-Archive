### Post by merilyn2 on 2022-10-06
Uninstalling app packages does not speed up your system, but for some that is always running in background (such as Ubuntu Software) it could free up some memory space. 

If your machine is running with power supply, you may try to enable "Performance" mode, by going to top-right system status menu -> Balanced sub-menu. You may also disable animation by going to "Settings -> Accessibility -> Seeing -> Enable Animations". Or, install and try MS Windows' default web browser 'edge' in Ubuntu:

[https://fostips.com/install-microsoft-edge-ubuntu-fedora-linux/](https://fostips.com/install-microsoft-edge-ubuntu-fedora-linux/)

Or just try a lightweight Ubuntu flavor, such as Ubuntu MATE, XUbuntu.
