### Post by rumli on 2006-05-09
I use TkCVS and find it fairly decent.  It comes with TkDiff which is useful for viewing differences in file revisions.  I've tried others like Pharmacy and gCVS and found them either overly complex or else lacking in features.  TkCVS is in the universe repository.
