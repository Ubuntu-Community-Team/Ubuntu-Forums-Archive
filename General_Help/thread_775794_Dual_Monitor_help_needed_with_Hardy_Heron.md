### Post by klange on 2008-04-30
[Here](https://wiki.ubuntu.com/LaptopTestingTeam/AcerAspire5610#head-f8f4f78b3060e46ccb01d92f75db29dc7088ec57)'s a guide I wrote on getting a second monitor working (properly) with an i945 (with DRI). You *should* be able to add more monitors (ensure that you use DVI instead of VGA for the DVI one, I'm not entirely certain it'll work with three monitors, though, depends on how the hardware functions), but if you want Compiz or other intensive DRI-related things to work properly, you need to keep everything within a 2560x2560 square.
