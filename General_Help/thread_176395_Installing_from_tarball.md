### Post by seth0x2b on 2006-05-14
Well, to see the contents of the archive you downloaded, go to the folder where you saved it and type:
```

bunzip2 xmms-scrobbler-0.3.6.tar.bz2

```
then
```

tar -xvf xmms-scrobbler-0.3.6.tar

```

This will extract the tar'ed folder. This is most likely the source of what you are trying to install you you'll have to read the documentation that the program is (probably) providing.

Cheers
