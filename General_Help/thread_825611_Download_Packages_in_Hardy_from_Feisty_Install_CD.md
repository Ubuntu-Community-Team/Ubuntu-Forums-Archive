### Post by haldean on 2008-06-11
You can get old packages from packages.ubuntu.com
The gaim ones from Gutsy (actually it was Pidgin by then) can be found here:
[http://packages.ubuntu.com/gutsy/pidgin](http://packages.ubuntu.com/gutsy/pidgin)
You'll also need to download the pidgin-data of the same version, found here:
[http://packages.ubuntu.com/gutsy/pidgin-data](http://packages.ubuntu.com/gutsy/pidgin-data)
Install pidgin-data before pidgin. You can just double-click them in the file manager to install.
