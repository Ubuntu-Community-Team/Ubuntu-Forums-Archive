The real Firefox theme is what you have, it is just different from Windows.

The theme you are trying to install is not designed for Linux. You can trick the browser in order to install it. 

To do that, use [Add-on Compatibility Reporter]("https://addons.mozilla.org/en-US/firefox/addon/15003/") extension. I have used Windows themes in the past. However, it may not look like the way you want or even have undesired side effects. So you must check if the theme generates any error in the console or interacts badly with other extensions.
