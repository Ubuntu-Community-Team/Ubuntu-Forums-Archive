### Post by lefo on 2011-04-30
Found an interesting article through the forums that made me try something that worked: I was using two monitors; couldn't get Unity to work, but Unity 2D would.  First, I changed the screen size on the second monitor and logged out.  When I logged back in for Ubunty with Unity, I did get Unity to work, but I lost my background picture.  Too many screen buggers, too.  So, I disabled the 2nd monitor, and voila!  Unity in all of its glory.  So, I guess the ATI 9550 just doesn't have the juice to run Unity on two monitors worth a darn.  I did find a GIGA-BYTE Radeon HD 4650 Graphics Card - GV-R465D2-1GI through Buy.com.  Can anyone tell me if this has the horsepower to run Unity on two monitors well?  I can only go up to AGP since I don't have PCI Express available.
