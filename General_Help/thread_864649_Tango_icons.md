### Post by jpeddicord on 2008-07-21

> tANGO ICOMS

How do you get the new Tango icons's for Ubuntu 8.04.1 64 bit and how do you install.
edwells 5 many thanks

You'll want to install the package [tango-icon-theme]("apt:tango-icon-theme"), either by clicking that link or searching for it in System > Administration > Synaptic. Then, in System > Preferences > Appearance, click the Customize button, then select the Icons tab. Tango should be at the bottom.
