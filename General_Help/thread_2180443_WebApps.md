Post reported for trolling.

When you open a webapp, it integrates the site's functions into Unity. Some sites can add items to the browser's menubar, some can draw emblems, progress bars or quicklists to its icon on the Unity launcher, some can send desktop notifications, and some have functions searchable in the HUD. A really complex webapp might do all of those or a lot of those. It's about making web sites more consistent with native programs.

No data collection, I can assure you.
