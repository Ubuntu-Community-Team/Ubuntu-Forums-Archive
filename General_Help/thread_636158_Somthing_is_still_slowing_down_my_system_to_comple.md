Does the slow down happen also when you are not running compiz?

Are you running tracker (the indexing utility)?

This error message does not tell much, try to find out what is the application giving the error message for a start. Haven't seen anything similar, so it must be a special application you have installed...

Try to determine under which conditions your system goes slow.

First try: unplug the network cable and power off the wireless connection. Is the system still sluggish? In the version 7.10 there was an issue with IPv6 being enabled, and causing slow downs. If I remember right it was noticeable if the DNS server/proxy did not support IPv6.
The fix was to disable IPv6.

Maybe your issue is other.
