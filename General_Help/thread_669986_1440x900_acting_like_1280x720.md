### Post by GoofusGreen on 2008-01-16
Hey everybody,

Virtual 1440 900

In xorg.0.log, it says the monitor is detected as a 1440x900 with dimensions 303x190mm and sets DPI to (120,120). This all makes sense, but for some reason the actual DPI my desktop ends up with something closer to (107, 96).

After much fruitless hand-editing of my xorg.conf, I solved an issue like this by simply changing the Model in System -> Administration -> Screens and Graphics

In the "Model" drop down, I chose "LCD Panel 1440 x 900 (widescreen)" without trying to select any particular model.
