### Post by wadageek on 2008-04-02
8.443 is the old naming convention ati/amd used mostly in 2007.  They've decided that a common naming convention be shared between their Windows and Linux drivers. So starting in Oct 2007 versions started with the last digit of the year and the numeric value of the month. ie driver 7.10 represents a driver released in 2007 in the month of October.

So the current version is 8.3 --> 2008 March

hope that helps.
