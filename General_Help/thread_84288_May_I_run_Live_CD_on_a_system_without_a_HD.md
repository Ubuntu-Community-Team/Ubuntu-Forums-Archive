### Post by svas on 2005-10-30
May I run Ubuntu Live CD on a computer without a harddrive, and just with enough RAM? Currently I have 256 MB RAM, witch is not very much. How much do I need if it works with no HD?

---

### Post by az on 2005-10-30
Yes, you can run it without a hard drive.  You can even run a thin-client without a hard drive.

256 Megs is enough, but the more you have, the less difference in speed you will find in comparing the live-cd with the same installed system.  You need at least 128 megs of ram to run the live cd.
