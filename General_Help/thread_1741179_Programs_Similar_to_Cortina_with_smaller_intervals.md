### Post by ~Plue on 2011-04-28
[this one]("http://gnome-look.org/content/show.php/Create+Background+Slideshow?content=125710") looks like it can set the duration to 1 min

if you want to make the time interval even shorter, you could make a xml file manually. (there should be a sample in /usr/share/backgrounds/cosmos/)
save the file with the .xml extension, then drag-and-drop it to the appearance window
