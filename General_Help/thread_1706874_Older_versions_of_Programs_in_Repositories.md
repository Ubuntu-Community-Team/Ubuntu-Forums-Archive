### Post by Frogs Hair on 2011-03-14
I don't know how often the repositories are reviewed , I would think that program compatibility would have to be checked often considering the six month release cycle.

If you down load a newer version from the web site you won't receive updates via the update manager but you may receive an update notification from the application , this has proved to be true with two of my applications.

When using the software center the more information option often has a link to the software web site and if the site is kept up to date , OS compatibility is often , but not always listed.
