### Post by EndianX on 2005-12-19
Hey everybody.  In Ubuntu, things mounted to /media/ are displayed on the desktop (CD Rom, my external hard drive, my NTFS partition).  Is there a way to turn this off?  I don't want anything on my desktop.

Thank you.

EndianX

---

### Post by Sutekh on 2005-12-19
Go to Applications -> System Tools -> Configuration Editor

Then go to Apps -> Nautilus -> Desktop

and uncheck volumes visible
