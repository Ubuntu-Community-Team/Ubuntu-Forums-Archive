### Post by mastablasta on 2011-09-21
yes, easilly. you just need to install the necessary software.
 
aside from Ubuntu server there are also other Ubuntu based distributions that focus on server with more software preinstalled. for example: [http://distrowatch.com/?newsid=06882](http://distrowatch.com/?newsid=06882)
 
Then you have specific internet cafe ubuntu based distro for example Dewalinux (indonesian, but maybe they have english) [http://www.dewalinux.web.id/](http://www.dewalinux.web.id/)
Cafebuntu: [https://wiki.ubuntu.com/Cafebuntu](https://wiki.ubuntu.com/Cafebuntu)
 
you also have non Ubuntu distributions for cyber cafe for example: [http://distrowatch.com/table.php?distribution=zencafe](http://distrowatch.com/table.php?distribution=zencafe)
