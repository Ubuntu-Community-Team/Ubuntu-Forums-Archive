### Post by winh8r on 2012-03-06
If your system locks up like this it is usually possible to restart using the following method:

press and hold down the ALT key

press the PrintScreen key and release it

type the following letters

R E I S U B 

The system should shutdown and reboot without you needing to use the power button or the plug.

A good way to remember the letters is this :

the word BUSIER backwards


As for the reason behind this, it might be due to memory and swap space being used up for buffering the content you are burning to the disk.

Therefore reducing what is available for the system to use to function normally

Probably be more apparent when burning large data files to DVD.

I know this is not a solution but it could be the cause.

Hope it helps a bit.
