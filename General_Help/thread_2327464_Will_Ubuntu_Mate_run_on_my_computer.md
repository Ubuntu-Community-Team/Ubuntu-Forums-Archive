### Post by grahammechanical on 2016-06-11
I have an Intel core 2 Duo 2.40 GHz + 1 GB RAM and Ubuntu Mate runs very well. In fact Ubuntu + Unity runs well even though some more RAM would make things a bit better. I have a video adapter with 1 GB of video memory and that does make a difference.

Ubuntu Mate will allow us to login using Compiz as the window compositor/manager or some other window manager that uses less memory than Compiz. If we choose the Compiz option then expect to use the same amount of memory as Ubuntu. This chart shows the situation.

[https://bryanquigley.com/memory-usage/ubuntu-16-04-livecd-memory-usage-compared](https://bryanquigley.com/memory-usage/ubuntu-16-04-livecd-memory-usage-compared)

But with 4 GB RAM you should not have any problems with Ubuntu Mate. The problems come when we need to dual boot with some other OS. If you need to do this we can give you advice so as to avoid problems.

Regards.
