### Post by jmorris1161 on 2008-07-18
When I started using ubuntu it showed all of my running applications in the task bar. Now it shows none and I have to alt tab between applications. How do I get it to show my running applications again? I am sure I did something but not sure what.......

---

### Post by jimv on 2008-07-18
You need to add the Window List applet to your panel.  Just right click on the panel and click Add to Panel, then find the Window List applet and click Add.
