Hello and welcome
32 or 64 bit?
You will need to enable the repo for java
Go to 'System > Administration > Souftware Sources'
Under the first tab, tick the boxes, I always tick them all and when you close it will ask you to reload and accept
Then under 'Applications > Ubuntu Software Center'
When that opens, just type sun or java into the search and you should be able to handle it from there,
