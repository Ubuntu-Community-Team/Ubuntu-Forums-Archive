### Post by dragonfly41 on 2023-08-12
My thoughts are to start by considering the performance of your laptop HDD.
If your wallet allows, you can pull out your HDD and replace with same size (or larger) SSD.
[https://uk.crucial.com/products/ssd](https://uk.crucial.com/products/ssd)

Going further you can plug in a *second SSD in container* to contain Ubuntu. But this is only good if you have a fast port (USB 3). Otherwise you can have on main SSD drive
Windows with VM Ubuntu
Ubuntu with VM Windows
Dual boot Windows/Ubuntu.

Keep your old HDD to place in a container to be a backup device.

[Here is one configuration using portable SSD.]("https://uk.crucial.com/articles/external-ssd/choosing-an-external-ssd-for-laptop")
