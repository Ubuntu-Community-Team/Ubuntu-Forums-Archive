### Post by Yellow Pasque on 2009-01-21
It's hard to say what's causing your problems without more info (especially details about any partitioning changes you made). I would definitely run an fsck from a LiveCd environment if possible and also try to capture real error output from a terminal before making a FUD thread on ubuntuforums.org.

Thinking that MS is putting time/effort into deliberately sabotaging desktop Linux installs is extremely cynical/paranoid (this comes from someone who considers himself fairly cynical/paranoid).
