### Post by daneb3 on 2010-03-01
Go to the Software manager and download Simple Compiz Config Settings Manager. Just search for simple compiz, and that should bring it up. Then go to System|Preferences|Simple Compiz Config Settings Manager and open it up. Look for a tab named Desktop, and you should see where you can add rows and/or columns to it.
