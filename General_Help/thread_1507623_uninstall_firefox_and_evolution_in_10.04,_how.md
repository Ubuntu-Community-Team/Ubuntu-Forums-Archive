### Post by labinnsw on 2010-06-13
System >> Administration >> Synaptic Package Manager

1.
Type the application name in the "Quick Search" window.
Left click the green box beside the application and select "mark for complete removal."
Follow the prompts.
When you have finished marking, click the apply button.

2.
In Software Centre
Click on Installed Software
Then click on the application (Firefox is listed as "Safe and easy web browser from Mozilla")
The remove button should then appear
If you tried this before, please excuse me posting it again.
