[QUOTE]what is easyest way to install on a sytem like the amd i described

Ubuntu is easy to install if it works on your system. That is not the answer you are looking for. Is it?

Do the research in three parts. Motherboard, graphic card, and CPU. If these are supported in Linux then they will work with Ubuntu. If you you want to use wireless networking then include that in your research. What about a TV card? These are the things that you want to know work with Linux before you buy them. You might want to check out mini-itex.com.
