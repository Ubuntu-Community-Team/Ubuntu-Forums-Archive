I think the first step to recovering might be to open up a command-line shell from the Unity desktop so I can re-install a few things. Any hints would be greatly appreciated.

### Post by plucky on 2013-07-17
Ctrl+Alt+t might open a terminal window.Depends on how badly your system is broken.

Ctrl+Alt+F1 will take you to a login prompt to open a command line window.

Good Luck

### Post by sudodus on 2013-07-17
If you still have a graphics desktop, you might be able to use ***ctrl + alt + t*** to start a terminal window. Otherwise you can use 

***ctrl + alt + F1***
...
[I][B]ctrl + alt + F6
[/B][/I]
to get text screens, where you log in and run commands.
