### Post by aysiu on 2008-07-03
I don't know if things have changed recently, but at least last year, you weren't supposed to remove the Ubuntu Firefox after having installed the Mozilla Firefox. You can find out more details about that on [https://help.ubuntu.com/community/FirefoxNewVersion](https://help.ubuntu.com/community/FirefoxNewVersion)

Of course, if you're on Ubuntu 8.04 (Hardy Heron) and using the Ubuntu Firefox 3, you can definitely remove Ubuntu's Firefox 2.
