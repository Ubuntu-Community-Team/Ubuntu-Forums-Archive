### Post by Drain on 2006-01-07
Okay, I've recently burned a number of .avi files to a DVD, that played in my stand-alone DVD player. I used the program [tovid]("http://tovid.org"), and there are pretty good step-by-step instructions on their webpage (look for the section about "using the graphical front-end").  I found [this forum thread]("http://www.ubuntuforums.org/showthread.php?t=101572") to be helpful in installing tovid, but there is a lot of threads about tovid in the forums. 

Once you have it installed, if all you have is a regular CD burner in your computer, you will have to use the "VCD" or "SVCD" settings instead of the "DVD" settings. This may entail breaking up a 2-hour long .avi into multiple cds, and I don't know exactly how tovid handles that. But you can give it a whirl. Good luck!
