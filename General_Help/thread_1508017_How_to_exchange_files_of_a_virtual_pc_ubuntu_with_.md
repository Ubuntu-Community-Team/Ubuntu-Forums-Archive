### Post by leoh on 2010-06-12
First of all, it doesn't matter if the system is physical or virtual. Each should have an IP, so from there on, everything is the same.
You should verify if they both have an IP and if that IP is accesible from the other. You should also enable the network switch on your virtualization software (VMware?). From there on, it should work exactly as if on 2 physical systems.
