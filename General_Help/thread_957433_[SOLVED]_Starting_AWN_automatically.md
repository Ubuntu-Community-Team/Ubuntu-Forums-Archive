### Post by johnhdsi on 2008-10-24
I am using Avant Window Manager and screenlets on my pc. I want to make AWN start auto every time I log in. How do I add this as a start up item? I also am having problems with screenlets savings my settings. I lock the screenlet to the desktop and turn off the stay above feature but each time I log in it returens to the default settings, any ideas?

---

### Post by perlluver on 2008-10-24
For AWN, go to System>Preferences>Sessions and add a new Entry, call it Avant, then under command type in avant-window-navigator.  That will add it to startup.  Not Sure about the screenlets problem.
