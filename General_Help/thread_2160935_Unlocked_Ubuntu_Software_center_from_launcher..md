### Post by Triblaze on 2013-07-08
I can find it by opening the dash (super/windows key) and searching software, or ubuntu, or center. From here you can drag it to the launcher in order to place it.

If you can't find it there, try opening a terminal (ctrl+alt+T) and running
```
software-center
```

This will open the software center. Once opened, it should have its icon appear on the launcher, right click and select lock to launcher.
