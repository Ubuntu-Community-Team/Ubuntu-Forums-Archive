### Post by bredman on 2010-05-12
Since you have some interest in programming, I suggest you hang around the bug tracker in Launchpad.

You can help out by confirming if a reported bug also affects your setup, this can provide valuable information for debugging.

You can also try out the proposed bug fixes and report if they work or not. You can do this blindly, but you can learn a lot by reading the code that has been patched. You will be amazed how quickly you will be sucked into the development work by doing this.
