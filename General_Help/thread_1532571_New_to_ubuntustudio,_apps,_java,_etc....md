### Post by cchhrriiss121212 on 2010-07-17
Just put the dvd in the drive and select it as a source in synaptic package manager. You should be able to then install the packages straight from the dvd as opposed to downloading them.
Or just browse the dvd to this location:
/media/cdrom1/pool/universe/u/ubuntustudio-meta
and install the deb packages you want by clicking on them.
