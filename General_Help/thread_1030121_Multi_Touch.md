With all due respect, that's like saying:  "Hey guys don't worry about using GUI for drag and drop!  All you have to do is open the terminal and use the CLI to move the file from one directory to the other!" ;)

I think it is interesting that MS and Apple are looking to find the next leap in user interface.  They think it will be "Natural User Interface".  The first implementation of this will be rolling out multi-touch.

If we want Ubuntu to remain competitive, we will take the effort to make sure that the kernel, the X windows system and the desktop environment are able to make full use of the technologies that are around the corner.  Otherwise the next questions will be: 

"Um...I think there's a bug in Ubuntu 12.10... the Log-in screen has a strange little arrow in the middle of the screen and when I touch the area on the screen with my name on it, nothing happens.  It is asking for this thing called a "mouse"...Is that like a dependency or something?"
:D

OK, this might be a little far fetched.  But at the moment, typing from my MacBook Pro with a multi-touch pad, I'm wondering why I ever thought a mouse was an intuitive way to control the computer.  This is nothing compared to what is expected to be around in the next three years or so...(see [http://natural-ui.com](http://natural-ui.com))

Interesting that natural-ui.com will make linux versions on request, but I very much doubt these will be free(dom)/open-source versions...not to mention free(as-in-I-don't-have-to-pay-for-it).  If we don't catch up a little, we'll just see some newbie linux distro take-over the lead that Ubuntu has set up (if people even bother with linux anymore...by that stage).

I think it is time for linux to be pre-emptive.  Imagine if Ubuntu had set up a partnership with some NUI company to bring out the first full-multi-touch computer (keyboard, pad, screen, etc).  Imagine if Ubuntu were an innovation leader.

Anyway, that's just the rantings of a sleep-deprived old man :D, who loves Ubuntu and doesn't want to see it go the way of MS-DOS, when something more fancy comes out.
