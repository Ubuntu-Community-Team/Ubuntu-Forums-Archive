**EDIT** Please read through this and jot down the commands youll need, or perhaps print it. You can get locked out of X at the CLI if this doesnt work the first time.

Youll need to manually download the driver. I have all Nvidia cards, but the process for installing the driver should be the same.

First, grab the driver that you need, and we'll say you put it on the desktop. Now, youll need to kill X (at least I did with nvidia):

```
sudo /etc/init.d/gdm stop
```
Hit alt+f1 or alt+f2 (cant remember which) if the GUI is still there or the screen is just black.

Then, login in at the command line, and change directories to the desktop:
```

cd ~/Desktop
```

Then, you will execute the driver. We'll say the driver is called ati.run:
```

sudo sh ati.run
```

The driver should update xorg for you (nvidia one needs to run nvidia-xconfig).

Finally, to get back to the login screen:
```

sudo /etc/init.d/gdm restart
```

Let us know if this helps..

**EDIT** If you have any problems and cannot get back to the desktop, do:

> sudo nano /etc/X11/xorg.conf

And navigate down to the driver line. Change it to "vesa", hit Ctl+O and then Ctl+X. Once again:
> 
sudo /etc/init.d/gdm restart

I apologize if this is more complicated than it needs to be for ati users- perhaps someone using ati has a better way?

### Post by khelben1979 on 2009-03-07
[The ATi Catalyst]("http://en.wikipedia.org/wiki/ATI_Catalyst") has a working GUI for Linux. See the wiki link for more information.

Also, if you need a good editor in text mode, I suggest that you use [emacs]("http://en.wikipedia.org/wiki/Emacs"). And if you don't have emacs in the system:

```
sudo apt-get install emacs
```

If you need to edit the file, try this after Emacs is installed:
```
sudo emacs /etc/X11/xorg.conf
```

```
ctrl + x + s
``` to save changes to the file and ```
ctrl + x + c
``` to quit the editor.
