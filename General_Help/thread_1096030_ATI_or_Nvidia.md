### Post by jespdj on 2009-03-14
NVIDIA has had much better drivers for Linux for a long time.

ATI is catching up, and it even has published the full specs for some of its graphics chips so that it will be possible for open source developers to make a fully open source driver with support for 3D graphics, but such a driver is not yet available.

So, currently, NVIDIA is the best choice. (Note that NVIDIA's driver is not open source, but at least it works well on Linux).
