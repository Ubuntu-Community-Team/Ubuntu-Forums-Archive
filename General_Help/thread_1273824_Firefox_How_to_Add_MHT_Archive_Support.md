In case you're like me and had to install **Opera** in order to get **[COLOR="Indigo"]MHT web archive support[/COLOR]** in your browser, [check out this blog post]("http://ubuntugenius.wordpress.com/2009/09/24/firefox-how-to-add-mht-archive-support/") I just stuck up. It has a brief outline for people who have no idea what an MHT file is, and a link to the extension that will give **Firefox the ability to save and view MHT archives**. You'll even be able to **save all open tabs as MHT files in one click**!
