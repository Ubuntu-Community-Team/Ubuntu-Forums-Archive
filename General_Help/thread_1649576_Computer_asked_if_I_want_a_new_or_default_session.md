### Post by tomsax on 2010-12-20
For some unknown reason on ubuntu start up my computer has started asking me if I want to go to the "defualt session" (telling me when I last used it) or start a new one.  Both options seems to go in okay but it just a pain having to choose which option before running.  I also have to then key in my password.  Any idea why it is doing this.

### Post by sikander3786 on 2010-12-20
Do you mean the choice offered at login screen? If so, you can check under System > Administration > Login Screen and choose a default session and also set it to Login Automatically without asking for a password.

### Post by JKyleOKC on 2010-12-20
Also check to see whether the checkbox on the log-out window to "save session" is checked. If it is, that would be why you're being asked next time whether to use the saved one (default) or start a new one. If it's not checked, you should automatically start a new session at each login.
