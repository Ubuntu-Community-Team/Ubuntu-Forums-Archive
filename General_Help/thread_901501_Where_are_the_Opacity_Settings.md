### Post by crwmike on 2008-08-26
In the CompizConfig Settings Manager, click on General Options, got to the Opacity Setings tab, under Window opacity, add **dropdownmenu** with a value of **90** to start, adjust for the desire effect. Also add **popupmenu** for context menues and **tooltip** for mouse hover tooltips.

### Post by Genius314 on 2008-08-26
You need CCSM installed, which is more complicated that Simple-CCSM. I think both can be installed at the same time, so you can use CCSM just this once.

In a terminal, type:
> sudo apt-get install compizconfig-settings-manager

Now go to System>Preferences>CompizConfig-Settings-Manager, and from there you can see the General Options.
