### Post by ursako on 2012-05-09
i seach on google... i find some results...
this is what u should do

download nokuntusp_0.6-1_all.deb (or other versions) in your home folder
then open a terminal and run this comands
```

mkdir -p nokuntu/control
cd nokuntu
ar x ../nokuntusp*.deb
cd control
tar xzvf ../control.tar.gz
rm -rf usr
tar czvf control.tar.gz *
mv control.tar.gz ..
cd ..
rm -rf control
ar -r ../nokuntusp-new.deb debian-binary control.tar.gz data.tar.gz
sudo dpkg -i ../nokuntusp-new.deb

```


this works for me
