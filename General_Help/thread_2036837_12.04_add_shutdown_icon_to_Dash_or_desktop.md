### Post by mc4man on 2012-08-03
Pretty simple, where you put the .desktop your choice.
Could go in ~/.local/share/applications/ & then should show in Dash & or be added to your dock however you do such with cairo-dock

Or just stick on the Desktop, if there you'd need to make executable after creating (right click on > properties > set as executable

So in ~/.local/share/applications/ (assumes that directory exists, if not then create it
```
gedit ~/.local/share/applications/shutdown.desktop
```

Or on the Desktop
```
gedit ~/Desktop/shutdown.desktop
```

Use this - 
```
[Desktop Entry]
Name=Shut Down
TryExec=/usr/lib/indicator-session/gtk-logout-helper
Exec=/usr/lib/indicator-session/gtk-logout-helper --shutdown
Icon=system-shutdown
Terminal=false
Type=Application
Categories=System;
Version=1.0
X-Ubuntu-Gettext-Domain=indicator-session
```
