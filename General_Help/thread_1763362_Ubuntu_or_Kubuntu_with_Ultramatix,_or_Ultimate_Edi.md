### Post by gsmanners on 2011-05-20
> **timandmonica said:**
> 1. Does Ubuntu give easy access to all those plug-ins and codecs, as well as an intuitive way to work with WINE? If it fulfills those requirements, is it also easy enough to navigate for a (novice) Windows user?

Ubuntu is pretty intuitive about most things, though wine is pretty tricky to work with, at times. It's pretty easy as long as your needs aren't too specific. For example, you can easily install flash via Software Center or through the browser's pop up dialogue. For other stuff, you might want to consider using mplayer (or a variation of mplayer) for videos and audacious for music. Those are just my suggestions, though.

> 2. Would Kubuntu be a better choice given its slightly closer resemblance to Windows?

Kubuntu is a better choice if you prefer to tweak things and get your interface looking a particular way, sure. If the user just wants things "to work," they may prefer regular Ubuntu. If they are really lazy about most things, they would probably prefer [Mint]("http://linuxmint.com/index.html"), actually.

> 3. Would Ultimate Ubuntu have everything she needs pre-installed? If so, could it be just too overwhelming for a noob? We do want Windows-style functionality built-in (web plug-ins, etc.) but we don't want to scare her with endless options and probably a slower computer b/c of bloat.

I don't recommend Ultimate anything as a matter of principle. If you try it out and they like it, that's another matter, but I wouldn't offer it sight-unseen.

> 4. Would using Ubuntu with Ultimatix be a good compromise, giving her many of the Windows-style oonveniences and make us not have to install Ultimate Edition just to get all those missing plug-ins, codecs, etc?

For compromise, I suggest using Mint.

> Oh yeah, her computer is a dual-core AMD something-or-other, about 4 years old. 2GB of RAM. No fancy hardware (other than a digital camera to plug in every once in a while!)

I have the same kind of computer here (except that I don't have a digital camera, and that might be problematic, so you'd be well-advised to look up whether it will work with Linux).

### Post by gsmanners on 2011-05-20
> **timandmonica said:**
> - So is Mplayer and Audacious kind of like a VLC player where there is codec support built-in to the actual player w/o having to install various codecs on the OS?

VLC is similar to mplayer, but it's more of an all-in-one solution. Some people prefer that. Mplayer uses a lot of external libraries, but it's a lot less annoying than Totem or whatever Ubuntu uses by default.

> - Do the various repositories for Ubuntu also work for Kubuntu? Do Ubuntu installers that are on an app's website that say "Ubuntu installer" really mean they work for Kubuntu as well?

Kubuntu (and every other Ubuntu derivative) uses the same repositories. The only thing you need to worry about is which version (date) of Ubuntu or Kubuntu you use. For example, 10.04 packages won't likely work with 10.10 or vice versa.

> - Same for Mint - I understand that it is based on Ubuntu, but can you actually use Ubuntu repositories and installers or do you have to find an actual, separate Mint version? I'm guessing there are a lot less Mint versions of things if it can't use Ubuntu's repository.

Same goes for Mint as Kubuntu (or Xubuntu/Lubuntu/Ubuntu Studio or whatever Ubuntu derivative you use).
