### Post by celldweller1591 on 2011-01-10
There are two ways to install compiz and compiz fuzion settings manager. 
First is via Software center. Search for 'Compiz' and install "Compiz", "Advanced Desktop Effects Manager" & "Emerald Theme Manager". Thats it. Configure Compiz using System>Preferences>CompizConfig Settings Manager. Then reboot and use it the way you want. 

Second is terminal install. Open the terminal and type : 
> sudo apt-get install compiz compizconfig-settings-manager
enter the password and let it install. 
Thats all. 

Edit: Any Nvidia GPU runs good with ubuntu. ATI even runs fine but there are sometimes issues with them, so if you are planning for Ubuntu, stick to Nvidia for now. Any decent card with clock of around 500Mhz and 256 MB ram would be great to run every effect of compiz with ease.
