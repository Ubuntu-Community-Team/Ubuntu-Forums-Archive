### Post by aysiu on 2005-12-11
[QUOTE=Ptero-4]Get her the iBook. Apple hw has far better compatibility with Linux than any brand of Wintel can have[/QUOTE] Actually Ubuntu (or any Linux distro) will not work with the Airport Extreme wireless card.

Also, the software available for PPC Ubuntu is a bit limited. For example (and this is something your mother may actually want to do), playing Flash in webpages for PPC is possible but a bit buggy. See [https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats) for more details.
