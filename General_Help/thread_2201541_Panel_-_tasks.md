### Post by Toz on 2014-01-24
> **makitso said:**
> Ok, xubunti, top panel, apps that at active, have an icon and text that names the app.  The question is, I only want the icon, not the text.  Can this be configured?

In the properties of the Window Buttons plugin, there is an option to "show button labels". Unchecking this will remove the text. To get to the properties of the Window Buttons plugin, either right-click to the left of the first button, or go through Settings Manager -> Panel.
