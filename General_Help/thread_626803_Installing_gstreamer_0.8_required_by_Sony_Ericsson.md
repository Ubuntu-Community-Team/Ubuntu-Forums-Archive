### Post by HakanS on 2007-11-30
Download the older packages [http://packages.ubuntu.com/feisty/libs/libgstreamer0.8-0](http://packages.ubuntu.com/feisty/libs/libgstreamer0.8-0) och [http://packages.ubuntu.com/feisty/libs/libgstreamer-plugins0.8-0](http://packages.ubuntu.com/feisty/libs/libgstreamer-plugins0.8-0)
First install libgstreamer0.8-0 and then libgstreamer-plugins0.8-0
Then Theme Breator works.
