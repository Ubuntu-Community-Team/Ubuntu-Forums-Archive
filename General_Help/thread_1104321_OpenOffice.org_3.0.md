### Post by igor. on 2009-03-23
I would suggest either waiting a month for jaunty, which includes the new version: [http://packages.ubuntu.com/jaunty/openoffice.org](http://packages.ubuntu.com/jaunty/openoffice.org)

Or getting it directly from [http://www.openoffice.org/](http://www.openoffice.org/). On the [other downloads](http://download.openoffice.org/other.html#en-US) page you can get the Linux DEB package. After extracting it run this in a terminal:
```
sudo dpkg -i OOO300_m9_native_packed-1_en-US.9358/DEBS/*.deb
```

The folder names for installation may need adjusting. For more info look here: [http://www.howtoforge.com/how-to-install-openoffice-3.0.0-on-ubuntu-8.04](http://www.howtoforge.com/how-to-install-openoffice-3.0.0-on-ubuntu-8.04)
