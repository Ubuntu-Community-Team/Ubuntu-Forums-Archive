### Post by grahammechanical on 2011-06-13
We need more information about the computer that you have, what version of Ubuntu are you using, what you have tried to do to get wireless to work and the error messages or how you know that it is not working. For now I have to ask what might be silly questions.

Do you see a Network manager icon on the top panel? Does the OS show you messages saying wireless disconnected or off line?

Are you able to connect to the modem/router using ethernet cable. If you can then please do so. Then go to Additional Drivers and activate any drivers offered. And reboot. As I do not know what version of Ubuntu that you are using I cannot tell you where to find Additional Drivers.

With laptops it is often necessary to press a keyboard combination to switch on the wireless adapter in the laptop. Make sure that the adapter is switched on. Do you have another OS installed? It is possible to use that other OS to switch off the wireless adapter. In which case it will not be easy for Ubuntu to switch the adapter back on. So, make sure that you have not switched the wireless adapter off in that other OS.

Try out these things and if they do not solve the problem post back what you have tried. We need clues as to what is preventing Ubuntu from using wireless.

Regards.
