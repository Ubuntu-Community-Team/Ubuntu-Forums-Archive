### Post by Nexusx6 on 2009-06-08
99% of the time when you need one program to launch another, you can do what was posted earlier and just type in the program name. If for some reason you need to know the bin path to program (such as, when just typing the program name in console doesn't work), you can use

```
whereis [program name]
```

to find the bin.
