### Post by howefield on 2010-03-21
> **sille777 said:**
> Will creating a Live USB Stick be the same as having a Live CD?

Yes, exactly the same, if you wish you can make and save changes to it, eg, change the wallpaper and it will reflect your changes next time you reboot. Add an application and it will be there next time you reboot with the stick. Just like a "real" install.

> So By creating a Live USB Stick and running it from the stick on a computer with out changing anything on that computer just like i can with the cd, correct?

Again, absolutely correct.

> Can I install Ubuntu on a usb drive from a liveCD or other method that will simply allow me to boot to Ubuntu on another computer without changing that computer but allow me to have all the personalized settings I have made to ubuntu on my Ubuntu USB Stick?

As above, yes. You will have the option during the creation of you USB stick to manage how much space you want to give to your disk for saving settings/apps ect.

> I would ultimately like to go the route of the second option so i can basically turn any computer I walk upto into my own with all my settings and internet bookmarks and what ever.  If this is possible how would i do that.

I think this is answered as above ?

### Post by colorlessprism on 2010-03-21
you can even use "Remastersys" to create a live CD image of your current installation (this i how i do my backups). i backup everything but music, videos, pictures, documents, .wine (i use tar for these). then i use the usb startup disc creator to convert the remastersys iso. once im loaded to my 16gb flash drive i untar my files to the proper location, and am now able to carry my "computer" in my pocket. should anything go wrong with my install i need about 5-10min to get back up and running like nothing ever happened. Remastersys is the best.
