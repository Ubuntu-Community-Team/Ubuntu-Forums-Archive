### Post by omegamike on 2005-12-22
Hiya,

Glad to see you are giving Linux a spin. There are a couple things you should know about installing programs on Linux. First of all, the old .exe files from windows don't really work unless you have a program called wine installed. You can install this program using Synaptic in your system menu. If you are looking for some good linux programs, I would also suggest downloading and following the instructions for Automatix: [http://www.ubuntuforums.org/showthread.php?t=66563&highlight=automatix](http://www.ubuntuforums.org/showthread.php?t=66563&highlight=automatix) It will install a ton of programs to get you started including XMMS which is similar to Winamp. For an antivirus program, I recommend clamav, it can be found by searching synaptic for "clamav" or clam or something of that nature. As for the defraging, the way the ext3 file system is set up, it really does not need to be done. There are programs out there that defrag, but it really isn't worth it. I hope this helps a bit, and please follow up if you have any questions.

Ohh yea, this site: [https://wiki.ubuntu.com/RestrictedFormats?highlight=%28restricted%29](https://wiki.ubuntu.com/RestrictedFormats?highlight=%28restricted%29) will tell you about how to install RealPlayer

### Post by Sutekh on 2005-12-22
[QUOTE=learning curve]I am only about 8 hours into Ubuntu and this is my first Linux experience.
How do install a program that I have downloaded from the internet ie: Winamp, Realplayer[/QUOTE]
If you are new to Linux, its best to stick to Synaptic to find programs you want.  Installing programs from the web can be difficult when you've first started.  Synaptic is available from the Sytem -> Administration Menu.  You can browse programs available and it will download and install them for you.  

Here's a link that will explain installing software using other methods, once you get familiar with Linux [http://www.psychocats.net/linux/installingsoftware.php]("http://www.psychocats.net/linux/installingsoftware.php")

[QUOTE=learning curve]and what program should I use for an Antivirus?[/QUOTE] Don't need one.  ClamAV if you really want.
[QUOTE=learning curve]Don't laugh but does Linux need to be de-fragged and How do I do a disk clean-up?  Any help for a newbie would be appreciated.  
Thanks, 
[/QUOTE] Since Linux uses a nice filesystem (ext3) no need to defrag like crappy NTFS, it will do a forced disc check after every 30 mountings (but that's automatic)
