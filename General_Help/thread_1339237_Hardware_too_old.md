### Post by audiomick on 2009-11-27
I've read opposing views on this subject. Some say that newer distros use up more resources, others that the newer (particularly) kernels and program versions are more efficient and therefore run better. Probably both opinions are true, and it is luck of the draw which way it turns out in the individual instance.

If your computer will take more RAM, it will probably run better for it.
