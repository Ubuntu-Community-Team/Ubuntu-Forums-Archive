### Post by hakimaki on 2007-09-05
I got Windows XP running in Vbox great, no problems.  However, how do I use the same Windows installation under a different user?  Ex:  My brother logs into my computer under his own name and would like to use my vbox windows install.  I tried loading the existing drive under his account but it yielded an error.  Help?

---

### Post by llamakc on 2007-09-05
You'll need to give him permissions for the *.vdi file, or a member of a group that has access to the said file.

I always keep mine in a separate directory OUTSIDE of my /home/USER directory.
