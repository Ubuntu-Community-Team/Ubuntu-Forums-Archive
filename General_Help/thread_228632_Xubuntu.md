### Post by carlosqueso on 2006-08-03
It isn't possible to obtain a copy of xubuntu from shipit, although you can get a copy of ubuntu, and then (while connected to the internet), just type ```
apt-get install xubuntu-desktop
``` and you'll be able to select xubuntu.  This download will be smaller than a full CD download. 

As far as wireless goes, it's not great, but it's getting better all the time. My suggestion would be to try the CD and see if yours is supported.

### Post by oss_monkey on 2006-08-03
It's as if this post was made for me to answer. :)

I've been using Xubuntu, more out of necessity than anything. To help you get started, I used aysiu's tutorial on installing Pure XFCE, but Ubuntu has an official wiki, so that might be a bit easier to follow. ([InstallingXubuntu]("https://help.ubuntu.com/community/InstallingXubuntu"))

In addition, wireless support is pretty good, at least in my experience, but YMMV. I use a Linksys Wireless-G card. I used a [tutorial]("http://www.ubuntuforums.org/showthread.php?p=1105101#post1105101") by nzruss, and I highly recommend using [wifi-radar]("http://wifi-radar.systemimager.org/"). 

(shameless plug) You can find a blog of my Xubuntu/wireless experience at my hp: [ossmonkey.blogspot.com]("http://ossmonkey.blogspot.com/"). :)

Good luck!
