### Post by adamitj on 2009-01-06
Hi Zaki.

Ubuntu Linux is an excellent Operational System, but you might check out if your applications exists in a Linux version. Virtually, you can switch from any Windows software to Linux, if there isn't a "native" (meaning a version compiled right for Linux) you can emulate some Windows applications with WINE.

However, even using Wine some applications aren't compatible. Take an example with MS Office 2007 - it does not run at all.
