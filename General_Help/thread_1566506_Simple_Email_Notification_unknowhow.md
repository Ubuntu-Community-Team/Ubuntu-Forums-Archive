### Post by howefield on 2010-09-02
If you are referring to the default messaging menu, it pulls the info from your Evolution setup. (Unless you changed it and I'd guess you would know if you had)

Clicking on the email envelope icon, then mail should load Evolution, then Edit > Preferences > Mail Accounts > then either add a new account or edit the server settings (Receiving E-mail/Sending E-Mail) in an existing account.

If you are referring to another email notification applet, then the above may be as useful as a chocolate fireguard ;-)
