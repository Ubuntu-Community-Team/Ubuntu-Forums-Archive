### Post by oldos2er on 2008-12-02
Install the package ubuntu-restricted-extras , which includes MP3 codecs, Flash, Java, and some other goodies.

 To play DVDs you'll need to enable the Medibuntu repository. See [http://medibuntu.org/](http://medibuntu.org/)
Once you have it enabled, install libdvdcss2.

Welcome!
2. Some programs have dependencies. For example KDE programs need a lot of KDE specific libraries. While it is perfectly possible to install any dependency you need and then to run pretty much any Linux software, you will keep your computer cleaner and things will work better together, if you try to use as many programs from the repositories as possible and if you try to only use GNOME programs rather than installing lots of KDE libraries. Hope that answers your question.

3. You should be able to see your Windows partition in the nautilus file manager and when you click on it, it should be mounted. After that you might need to click on it again and you should be able to see and modify its content.

4. Do you not have an icon in the top right corner of the screen in the notification area? Try left clicking on all the icons that you see there. One of them should show you the list of available wireless networks.
