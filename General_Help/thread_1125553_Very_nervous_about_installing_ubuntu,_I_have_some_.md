### Post by 3Miro on 2009-04-14
The DVD that you have is already a Live one. You can boot from it and select "Try Ubuntu without any changes to your system". Then it just runs from the CD and doesn't touch the HD and windows.

Wibi is safe, meaning it will not damage windows.

The danger comes if you have valuable data under windows and then try to play with partitions. That's why wibi is the right choice.

The risk of wibi is mainly damage for Ubuntu (it will be surely slower, but it may also get damaged by windows). Some people have hard time uninstalling wibi, but overall there is no risk for windows.

To use wibi,
insert the disk and follow the instructions to install it like a normal windows program.

### Post by upchucky on 2009-04-14
you can download ntfsclone, make an image of your windows and save it to a usb or dvd. then you can restore windows from the usb or dvd.

this ensures that your complete windows install complete with all the files and software you have installed can be restored to the hard drive in about 20 minutes instead of having to spend an entire weekend re-installing windows and all software.

as always, make sure you back up any important files before doing any partitioning, installations, or messing with windows settings.

windows is well known to slowly self destruct, so back ups are a good idea, and images are a life saver.
