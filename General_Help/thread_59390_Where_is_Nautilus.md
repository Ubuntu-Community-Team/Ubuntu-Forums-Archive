That will  open up nautilus for u.[/QUOTE]
Ooo not quite right.  Nautilus is not a window manager, it is a file manager.  Much like Windows Explorer, it is integrated into Gnome - when you browse for files you're usually using Nautilus.

Technically the window manager is Metacity and it is more akin to the windowing shell that "explorer.exe" seems to provide.  That is, kill "explorer.exe" as a process in Windows and it seems to restart the window manager.
