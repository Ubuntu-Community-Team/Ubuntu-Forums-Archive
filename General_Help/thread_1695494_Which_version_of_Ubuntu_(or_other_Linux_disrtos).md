### Post by SE7EN-LOCSTA on 2011-02-26
honestly, i would start with the regular version of ubuntu, and then add xubuntu and lubuntu if needed. they are moderately small in the amount of downloading needed to install them, and you can select which desktop version to load at login. 

you could just start with the xubuntu/lubuntu, but i have heard some stories of troubles trying to add ubuntu after the fact. i would probably not worry about kubuntu, as it is more resource intensive, but you can always try if you're into the eye candy. 

personally i have all of the above, switching based on my needs/mood...

as far as other linux distros, ive only used *buntu, so i cannot speak on them, and as you didn't specify your linux-knowledge level, i couldnt say anyways. ubuntu seems to be the most well rounded for beginner-expert from what i've seen.
