### Post by 3Miro on 2010-09-04
Linux comes with the best and latest of all of your drivers with only a few exceptions. You need to go to System -> Administration -> Hardware Drivers and it will give you a list of all the drivers that you need (just select "activate").

As for the video game that you want to play, in order to play windows games, you have to install wine (you can do that from Ubuntu Software Center) and the you need to check on the forums of [www.winehq.com](www.winehq.com) for specific instructions on how to install and run your game. You can use Google: "Winehq Counterstrike". Most video games require specific tweaks.

PS. When you make a new thread on the forum, try to make the title meaningful. Something like "How to install drivers" would be far more appropriate and helpful to use, then just saying "Please Help!"
