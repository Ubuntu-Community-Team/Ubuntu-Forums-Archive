### Post by snowpine on 2010-09-16
Most Windows applications will not run natively in Ubuntu. An application called WINE can be used for some, with varying degrees of success. I recommend searching the Wine Application Database to see what they have to say:

[http://appdb.winehq.org/](http://appdb.winehq.org/)

In general, I would absolutely 100% recommend dual booting. Windows and Ubuntu are both excellent operating systems and it is nice to be able to choose between the two. Also, if you are new to Linux, it can be very helpful to have Windows as a fallback, for example if you are on a tight deadline and can't figure out how to do something in Ubuntu.

### Post by colin.p on 2010-09-16
Make sure you read up on all the caveats on installing ubuntu alongside 7.
There are an infinite number of threads on some of the issues, especially on some manufacturers laptops, they have "data safe" type programs (Dell, HP are a couple) that screw with grub and cause the computer not to boot, as well as using win 7's native disk managing utility to resize the partitions... but you probably already knew that.;)

Just wanted to add, don't let that discourage you. When you get everything up and running it's a beautiful thing. You will probably find that you boot less and less into 7, until it fades completely from memory.
