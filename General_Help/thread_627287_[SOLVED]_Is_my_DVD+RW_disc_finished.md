### Post by p_quarles on 2007-11-29
Looks like it means the your system can only read 512 bytes from the disk. That's roughly the size of the file that would represent the device on your system. So, sounds to me like it's toasted. Unfortunately. :|

You could try reformatting it, but you'll probably need to write it off. Rewritable optical disks don't really live that long.

### Post by tgalati4 on 2007-11-29
Be sure to use slow burn speeds (perhaps even 1 or 2X) when formatting RW media.  Using 4X or more tends to create coasters.
