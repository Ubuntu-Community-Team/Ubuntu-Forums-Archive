### Post by mssever on 2008-08-25
I wonder if something has confused the default serif font. Try setting the system font to an actual font, such as DejaVu (what you normally see when you specify serif).


On another line, I think it's likely that one of the fonts you installed has the same name as one of the standard fonts. so when the forum asks for font x, you see the font x that's installed, instead of the one installed on the system.
