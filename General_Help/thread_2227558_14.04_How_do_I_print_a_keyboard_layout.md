### Post by shochatd on 2014-06-03
I've come up with a workaround until a real solution/fix emerges:
1. Bring up the keyboard layout chart for the current entry language (2nd to last item in menu).
2. Use the Screenshot tool in the current window mode.
3. Print the resulting png file using an image viewer.
The resulting print quality is slightly inferior, but usable. I assume that the loss of direct print functionality is a bug in 14.04
