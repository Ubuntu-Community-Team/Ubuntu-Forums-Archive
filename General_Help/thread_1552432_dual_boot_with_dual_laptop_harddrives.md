### Post by warddr on 2010-08-13
This can be done very easily, if they are on separated disks that are not in the computer at the same time ubuntu doens't even know you are also running windows on that computer and vice-versa.
You don't have to do anything special during the installation, just install ubuntu on one disk, and windows on the other. The computer itself will see which disk is in there, and boot the operating system on that disk.
