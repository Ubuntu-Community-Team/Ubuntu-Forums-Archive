### Post by angryfirelord on 2009-12-04
> will ubuntu 8.04 be stable?
Yes, as it was built to be a LTS.
> will ubuntu 8.04 have the same network manager GUI as 9.10?
For the most part. It won't have the new popups, but it still uses NetworkManager.
> will I get the latest software on 804?
Unless you compile the software, no. LTS releases focus on providing stability. Incorporating new software would bring in new problems and defeat that purpose.
> whats the disavdantage of using 804?
As stated above, you're running older software. If you have a machine that's less than a year old, 8.04 may give you some issues. Unless you really need that long term support, 9.04 would be a better choice.
