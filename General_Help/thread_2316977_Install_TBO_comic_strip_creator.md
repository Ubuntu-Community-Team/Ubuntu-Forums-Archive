### Post by grahammechanical on 2016-03-12
Do not use "software updater." It is not meant for the purpose of installing software but for updating/upgrading installed software. Use the file manager to find the .deb file that is the application and then right click the deb file & select Ubuntu Software Centre to open/install the application.

Oh, if the application is an .exe file or a .rpm file, then it will not run on Linux/Ubuntu.
