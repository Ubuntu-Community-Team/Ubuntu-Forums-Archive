### Post by Mark Phelps on 2012-02-27
Shutting off an upgrade midstream is likely to leave your package management system in an unworking state.  At this point, you may have to completely reinstall to get it working properly.

As to motherboard-based power-saving, if you can implement the features in the BIOS, then you can get it working.  But, if you have to install and run some MS Windows, apps, then no.
