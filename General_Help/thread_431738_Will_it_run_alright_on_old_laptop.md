### Post by 3rdalbum on 2007-05-03
Eudora Light is an old program indeed (I used to use it on the Mac). There is a program for Linux called Wine, which can run some Windows programs - it has more success with older programs, so you may be in luck there.

Eudora has now become open-source, so maybe a Linux port will eventuate.

May I suggest that when you install Ubuntu or Xubuntu, that you take things slowly - don't try to get absolutely everything going in the first day. It's better for you to take it easy and learn about the system before delving into things that might confuse you otherwise.
