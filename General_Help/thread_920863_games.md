### Post by jonian_g on 2008-09-15
Go to Applications>Add/Remove and in the games section you will find many.
Also some more games can be found on [http://www.getdeb.net/](http://www.getdeb.net/) .

Some well known commercial games that have a linux version are Quake series, Doom3, Unreal Tournament, Penumbra etc.

Websites about linux gaming:

[http://www.linux-gamers.net/](http://www.linux-gamers.net/)
[http://www.linuxgames.com/](http://www.linuxgames.com/)

Games list on wikipedia: [http://en.wikipedia.org/wiki/Category:Linux_games](http://en.wikipedia.org/wiki/Category:Linux_games)

Also you can play some windows games like WoW, NFS Carbon and more using cedega(Commercial - [http://www.cedega.com/](http://www.cedega.com/)) or wine(in the repos - Free, open source - [http://www.wine.hq/](http://www.wine.hq/)).
