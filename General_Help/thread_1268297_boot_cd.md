I can't get my computer to boot from the ubuntu cd.

I downloaded the ubuntu iso from the main ubuntu website, just the generic ubuntu download of the most recent stable version, and burnt it to a blank dvd, on a windows machine. Windows says the disc is fine, but when I boot up my linux machine and tell it to boot from cd, it sits there for a while then starts the normal ubuntu installation from the hard drive. ### Post by ram2500 on 2009-09-16
ISOs are universal to any and all platforms (for the most part), so an ISO burned in Windows would work on others and vice versa. I am more than likely probably wrong, but I'm guessing you were supposed to use a CD-R or RW, not a DVD (???) so there might be a little nagging intricacy there about that. I assume that your machine has enough memory to get to the Live Desktop and still be able to install, as the bare minimum for live installs is a meager 384MB. So, ruling out a defective DVD, the little intracacy of using a DVD instead of a CD, and the lack of memory, I am probably thinking that you were really supposed to use a CD. 

Wait a second, though: I have had experience with burning a Linux ISO and having it work until well into the install, where it says, "Your media is defective, blah blah", so it may very well be defective media.

Also, it takes a while for Ubuntu installs, or any for that matter, because it's detecting your hardware, so it may very well be doing its job right, albeit slowly.

A couple things I would check just to rule somethings out:

Check BIOS boot order (just double check the DVD drive is before hard drive)

Check to make sure your DVD player can read the DVD media you burned to ( DVD -R, DVD -RW, DVD +R, DVD +RW, DVD -RAM)

You might have defective media as stated already above.  Try either using a cd-r or cd-rw or make sure you are burning at a low speed to minimize errors during the burning process.
