### Post by AirRaven on 2006-08-17
I assume you're using MSN for teh file transfers?

Gaim's MSN network file transfers, to put it politely, suck. They route everything through the MSN servers before it gets to you rather than using a direct peer-to-peer connection. 

I suggest you try and find a messenger that doesn't have this problem if you want to use MSN (aMSN and Mercury might work), or use Aim file transfers if possible. 

I hope they're going to fix this problem in the future- it's a huge flaw in the messenger.
