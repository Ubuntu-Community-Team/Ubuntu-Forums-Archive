### Post by mcduck on 2009-10-11
Sure, if any system can run Apache then that's Linux. :D

The easiest way to install Apache, MySQL and PHP is opening a terminal and running the following command:

```
sudo tasksel install lamp-server
```

Alternatively you can open Synaptic (System/Administration/Synaptic Package Manager), then go to Edit/Mark Packages by Task, enable "LAMP Server" and click "Apply".

You'll find more help with Ubuntu's LAMP stack here: [https://help.ubuntu.com/community/ApacheMySQLPHP]("https://help.ubuntu.com/community/ApacheMySQLPHP")
