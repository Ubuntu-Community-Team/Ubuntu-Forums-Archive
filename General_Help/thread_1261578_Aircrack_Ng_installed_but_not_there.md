### Post by t0p on 2009-09-09
Aircrack-ng won't appear in your menus.  It isn't a pretty-gui point-and-click thing.  It is a suite of command-line utilities.  And you'll need to invest some time in learning how to use it properly.

If you want to use aircrack-ng, check out the tutorials [here]("http://www.aircrack-ng.org/doku.php?id=tutorial").  Don't be frightened off by the command line interface.  It really is an excellent set of programs.  But you need to shake off your reliance on guis and menus.
