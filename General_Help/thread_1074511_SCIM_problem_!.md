### Post by geraldm on 2009-02-19
Generally, if you want to type in Croatian language, you need to use a keyboard and keyboard layout that produces Croatian letters.  

This solution does not use the SCIM input methods.  You could select the US_International keyboard layout.  This layout has dead keys; keys that do not produce a code on the first keypress.  Generally, the letters that have different accents must have their accent  pressed before the vowel; secondly, then press the vowel key and the two kepresses will produce one character.
There is more help at 
[http://en.wikipedia.org/wiki/Keyboard_layout#Slovak](http://en.wikipedia.org/wiki/Keyboard_layout#Slovak)
You may need to figure out how to set the keyboard layout type to allow Croatian letters.  This information should be available in selecting the keyboard type, or in selecting Croatian language.  I do not know the easy way at this moment.
There should be more information in the help pages
[https://help.ubuntu.com/8.10/advanced-topics/C/index.html](https://help.ubuntu.com/8.10/advanced-topics/C/index.html)
search for Croatian.

Gerald
