### Post by Dr. Nick on 2006-05-11
Their is no easy or safe way to fully reverse a upgrade, If you find the conflicting program it is possible to reinstall the old version of it.

In theory you might be able to make it install a lot of old version but I would think that would cause even more trouble.

Try rebooting into your old kernel from grub and see if it works then
