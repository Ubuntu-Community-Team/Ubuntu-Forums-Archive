### Post by l3dx on 2008-07-23
Why not both? :)

To program you don't even need a PC. You can program on a calculator or a Commodore 64! If you program in Python you can run your code on all platforms that have a Python interpreter, meaning that your code will run at both Windows and Linux :)

Personally I'm a big fan of Python, and I believe that it is a very good starting point when it comes to programming. As you learn, you can start using libraries and frameworks to create almost everything. [Pygame ]("http://pygame.org")for games and graphics. [Turbogears]("http://turbogears.org") or [Django]("http://djangoproject.com") for web, or [PyQT]("http://wiki.python.org/moin/PyQt") for GUI applications. And when you start getting familiar with Python it's quite easy to adapt other languages.

Also check out [this thread]("http://ubuntuforums.org/showthread.php?t=796494")!

Good luck! :)
