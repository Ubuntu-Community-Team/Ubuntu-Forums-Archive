From Wikipedia:

> **Microsoft Compiled HTML Help** is a proprietary format for [online help]("http://en.wikipedia.org/wiki/Help_%28online%29") files, developed by [Microsoft]("http://en.wikipedia.org/wiki/Microsoft") and first released in [1997]("http://en.wikipedia.org/wiki/1997") as a successor to the [Microsoft WinHelp]("http://en.wikipedia.org/wiki/Microsoft_WinHelp") format. It was first introduced with the release of [Windows 98]("http://en.wikipedia.org/wiki/Windows_98"), and is still supported and distributed through [Windows XP]("http://en.wikipedia.org/wiki/Windows_XP") and [Vista]("http://en.wikipedia.org/wiki/Windows_Vista") platforms.
