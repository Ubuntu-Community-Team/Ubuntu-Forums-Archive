### Post by jerenept on 2010-04-10
you might want to try editing the sound settings. Right-click the speaker icon on the Ubuntu system tray. Select "Sound Preferences". Go to the tab marked "Input". You should uncheck "mute" if it isn't already. Select the correct microphone port and sound hardware you are using. If the sound is too quiet, you might consider raising the volume a bit. This is assumung you use 9.10 like i do
