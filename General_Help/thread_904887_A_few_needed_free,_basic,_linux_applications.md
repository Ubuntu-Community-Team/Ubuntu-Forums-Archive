### Post by mb_webguy on 2008-08-29
I think you'll find that there's a greater number and variety of applications for a given task for Linux than you'll find for Windows.  Those applications may not necessarily be as well-developed as commercial products available for Windows, but on the other hand, a few are actually better.

Here's a few WYSIWYG html editors for Linux:
[LIST]
[*]Quanta
[*]Bluefish
[*]Ginf
[*]Screem
[*]Nvu
[*]Mozilla Composer
[/LIST]

Of course, if you just want syntax highlighting, the default text editors for K/Ubuntu, Gedit and Kate, will do the job.

You might also want to check these pages for html/php IDEs:
[http://www.linuxdocs.org/HOWTOs/PHP-HOWTO-9.html]("http://www.linuxdocs.org/HOWTOs/PHP-HOWTO-9.html")
[http://linuxmafia.com/faq/Devtools/ides.html]("http://linuxmafia.com/faq/Devtools/ides.html")

As for clipboard apps, install Glipper (or Klipper if you're using Kubuntu).  It's a panel applet that does pretty much exactly what you're looking for.  Type "sudo apt-get install glipper" to install it, then right-click on a panel, select "Add to panel", and then "Clipboard Manager" to use it.
