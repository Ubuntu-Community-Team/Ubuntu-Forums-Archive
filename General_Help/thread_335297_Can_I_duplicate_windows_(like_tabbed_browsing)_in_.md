### Post by Tux Aubrey on 2007-01-10
I'm not sure I understand the question, but since I'm in the mood to answer one I'll just note that if you use [fluxbox]("http://fluxbox.sourceforge.net/") as your window manager, EVERYTHING can be "tabbed".  Effectively all open windows (and duplicates) can be a tab in single window.  I'm nor sure whether this is possible in gnome or kde or xfce.

There are several good guides to installing and configuring fluxbox around the forum.  The current version of fluxbox in the repos (ie the one that Synaptic will get for you) is pretty up to date, so don't concern yourself with a number of older guides that suggest compiling from source code (unless you want to - its pretty easy)

Good luck
