### Post by zcacogp on 2012-05-09
OK, solved it. 

For reference (if anyone else reads this), they are called 'Bookmarks'. If you 'Add a bookmark', a new "Bookmarks" section appears on the left hand side and you can then drag other things in there as well.

### Post by Lorin Ricker on 2012-05-12
> You have to be in a folder that you want to add.

Yup, and there's an easy way then to add the bookmark (shortcut):

1. Navigate (drill down) to the (sub)directory (folder) you want to create a bookmark for... then:

2. Press Ctrl+D -- the bookmark will be added to the bottom of the list in the left-panel.

3. If you want, you can drag-&-drop the bookmark itself to sort it somewhere else (other than the bottom) in that list.

4. If you want, you can right-click any bookmark to: a) Open the bookmark's directory, optionally in a new tab or new window; b) rename it; c) remove (delete) it.
