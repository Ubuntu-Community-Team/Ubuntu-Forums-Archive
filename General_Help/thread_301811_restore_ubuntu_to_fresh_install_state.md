### Post by dasunst3r on 2006-11-17
If you would like to clean up your installation, first start off by going through the menus and seeing what you don't need.  Uninstall those through Synaptic.  Be careful if you have to remove other packages as well because you might need those.

After that, check this article on Digg: [http://digg.com/linux_unix/Linux_Tricks_Keeping_your_Ubuntu_Debian_machines_clean](http://digg.com/linux_unix/Linux_Tricks_Keeping_your_Ubuntu_Debian_machines_clean)

### Post by doobit on 2006-11-18
There is a filter like that in Synaptic. The little ubuntu symbol by the package name tells you what is the official ubuntu package. The filters can be set up to tell you what's installed,. right clicking ont he package name will give you a menu with the choice of listing where all of the parts of the package went during the install.
