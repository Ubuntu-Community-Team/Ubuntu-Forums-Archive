### Post by Welly Wu on 2018-05-05
I installed Ubuntu 18.04.0 64 bit LTS on my mid-2016 Acer Predator 17X (GX-791) gaming notebook PC using a clean installation method thus erasing Microsoft Windows 10 Pro 64 bit build version 1803 overnight. For the first several hours, Mozilla Firefox Quantum loaded all websites normally and I finished installing my software applications, packages, and products. I restarted and neither Mozilla Firefox Quantum or the Brave Browser will load Facebook correctly. To be specific, I like numerous Facebook Pages and I am a member of quite a number of Facebook Groups and none of that content is loading on my news feed any longer. I only see People You May Know show up on my news feed. If I click on a specific Facebook Page or Group, then it will load the content normally. Facebook Chat functions normally as well. All other websites load up normally. I am a member of the Ubuntu Facebook Group and a few other people replied to my post stating that they experience a similar bug with Ubuntu 17.10 and 18.04.0 64 bit. I know it is early days and there are known software bugs, but I am curious to find out if anyone else has a similar problem and if there is a Launchpad bug report available with more information. If there is a workaround or fix, then please let me know.

For the record, I did everything recommended by Mozilla including clearing my cache, history, deleting my profile, uninstalling Firefox, deleting traces of the Mozilla folders in hidden folders in my /home/wellywu folder, and resetting Firefox Quantum, but the problem persists and it is replicated in the Brave Browser which I later installed using an Ubuntu Snap. This seems to be a problem with Ubuntu 17.10 and 18.04.0 64 bit because none of the recommendations offered by Mozilla fixes the problem. I also connected my gigabit Ethernet category 6 cable, used my 802.11 dual-band AC Wi-Fi, disconnected from all virtual private networks, double checked to ensure that I am using my home IP address when accessing the Facebook website, etc. Nothing solves this problem.

Please reply to this thread if you experience this problem and let me know where to get more information. Thank you.

### Post by Welly Wu on 2018-05-06
I think that I figured out how to solve this problem which may be of help to other computer users. The problem seems to occur with your specific Facebook account when you join several groups and you like many pages. Over time, I liked numerous Facebook pages and I joined many groups and it resulted in my News Feed not loading properly. I researched this problem and I reduced the number of Facebook pages and I left several groups. A day later, my Facebook News Feed loads up properly. It seems to be a bug with Facebook and it is not indicative of your web browser or desktop operating system based on my research. I am passing this information along to this community just in case someone searches for a similar problem and they find this thread of mine. Thanks.
