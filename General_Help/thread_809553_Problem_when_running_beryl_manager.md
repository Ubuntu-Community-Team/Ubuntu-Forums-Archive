### Post by Forlong on 2008-05-30
Sorry, for the late reply, have been busy.
> **BATLI said:**
> ```
deb http://ubuntu.beryl-project.org [COLOR="Red"]feisty[/COLOR] main
```
You see what's wrong here?

In case you didn't know, Feisty (also known as Ubuntu 7.04 Feisty Fawn) is *the release before the release before the release* of Ubuntu you are using right now.

You can't mix packages between releases (at least in the very most cases).
Get rid of these packages ASAP.

Then another "in case you didn't know": Beryl is a discontinued project. Most of its features have been ported to Compiz and are part of the Compiz Fusion project.

Both Compiz and Compiz Fusion are installed by default in Hardy.
Is there any particular reason you still want to use Beryl?
