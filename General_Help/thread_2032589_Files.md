### Post by Frogs Hair on 2012-07-24
To create a sub folder open the host folder(documents, music or created folder) and select File > Create New Folder . The new folder will be inside the host folder. 

I don't have any .odt location in my home folder . The .indicates it's a hidden folder and that may be why you can't find it when trying to navigate to it. 

When I send more than one item I place them in a folder , right click the folder , select compress and choose the format from the menu in compression dialog box. For Windows I would probably use  zip .
