### Post by oldtimer7777 on 2011-11-22
The main problem with new users to Ubuntu isn't that they can't use it the way Windows used to work for them, but they don't have a solid firm grasp of what they need to do with their system or a good comprehensive "walkthrough" provided officially and recommended by Ubuntu on how to install everything they want correctly.  If they knew about something like that from the beginning, much frustration that new users experience could easily be eliminated.
[B]
Everything that the above original article has a complaint about with Ubuntu Linux can easily be resolved with a complete fresh installation guide like this guide here:[/B]

[https://debianhelp.wordpress.com/2011/09/12/to-do-list-after-installing-ubuntu-11-10-aka-oneiric-ocelot/](https://debianhelp.wordpress.com/2011/09/12/to-do-list-after-installing-ubuntu-11-10-aka-oneiric-ocelot/)

With that guide new Ubuntu users would be able to do everything, and anything they wanted with their new Linux system.  I don't know why there is no official walk-through like the link above, but it should be [COLOR=Red]Mandatory [/COLOR]to use a guide like the one above if you are a new Ubuntu user. I'm tired of reading complaints from frustrated users (which is expected) but it would be nice to see some effort is placed in letting them know exactly what they need to do if they are new. Right now it is like throwing some Window user into a new system without any guide at all, and I think that is terribly inefficient considering how much effort has gone into Ubuntu to make it simple enough to use for just about everyone out there.

Just my suggestion and two cents..
