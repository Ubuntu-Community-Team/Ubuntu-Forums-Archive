### Post by Klump on 2011-02-02
I guess you're using the Unity interface now. If so, the top panel (I believe?) is not customizable. You could, however, achieve similar interface with now called classic Ubuntu desktop and AWN dock by putting AWN to the left side and forcing monitor height-panel_size through it's settings so the dock and the panel don't overlap. You won't have controls in the top panel, though, unless you install Window Applets from gnome-look.org.

### Post by kozo2505 on 2011-02-02
Hi everybody!
I'm having same problem here. I bought a new HP DV6T-SE, installed UBUNTU 10.10 amd64, everything looks great, but can't use right click on my unity mouse pad. Searched for a solution but got nothing. I NEED the secondary button, and the only way to have it back was:
System>Preferences>Mouse>Accessibility then select "Trigger secondary click by holdingdown the primary button" That's not a permanent solution, but I'm able to use the options given by the right click till someone comes with a real solution for this. ;)
