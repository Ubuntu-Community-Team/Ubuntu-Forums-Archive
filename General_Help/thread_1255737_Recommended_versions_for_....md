### Post by Topsiho on 2009-09-02
Why not put some more Ram into your box?
If you do, extend Ram to at least 256 MB, however some more is better.

Then you could run the regular Ubuntu, Kubuntu and of course Xubuntu.
But don't expect your computer will run real fast though. I have a box that has a 500 MHz processor, and it runs well enough, not more than that (with 384 MB Ram).

Note: a distribution upgrade of Ubuntu with only 256 MB Ram will probably fail. With a dist upgrade the OS uses some of the Ram, while running Ubuntu needs all 256 MB's. In a normal install the OS sits in the CD-rom.
