### Post by Tipped OuT on 2009-05-17
32 bit is for 32 bit processors and 64 bit is for 64 bit processors, but can still use 32 bit operating systems, and applications. I believe there's a command for terminal to check whether you have a 64 bit processor or a 32 bit, I'm not sure what it is though.

32 Bit can only use 32 bit applications. It is also limited to using 4 gigabytes of RAM for the operating system.

64 bit can use most 32 bit applications and 64 bit applications. It can use up to 8 gigabytes of RAM for the operating system and is generally faster, depending on your specs.
