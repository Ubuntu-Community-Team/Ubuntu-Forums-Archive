### Post by Mze on 2007-12-25
You most  probably have to reconfigure your graphics card on your new system.

First check your pci devices with the code below (in the terminal screen):

lspci

The info generated by the command above should allow you to find the name of your graphics driver and then run the code below (also in terminal window) to reconfigure your graphics card.

sudo dpkg-reconfigure xserver-xorg
