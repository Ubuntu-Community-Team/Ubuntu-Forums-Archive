### Post by QIII on 2010-06-17
This is a well-known issue.

Flash is proprietary.  Adobe has paid little attention to the Linux community, so the Linux version of Flash is poorly written and demands an inordinate amount of system resources.

The problem is not with Ubuntu or any other Linux distro -- all of which demonstrate the same behavior.

The problem is with the Linux version of Flash itself.

I use the 64 bit version of Flash, but I have a fairly state of the art machine with substantial resources.  Flash still consumes more than it should.  If your resources are limited, Flash can cause your system to crash.

You may try uninstalling Flash and using either swfdec or gnash.  Use one or the other.  Flash conflicts with both swfdec and gnash.

Neither swfdec nor gnash is universally recognized by sites with Flash content, so you may find some disappointment.

I wish I had better news.  This is something Adobe will have to fix.  We can't do much about it.
