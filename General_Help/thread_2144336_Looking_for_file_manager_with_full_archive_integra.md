### Post by thinkinhurtz on 2013-05-11
Hi everyone, I have been searching for something without any luck and I need some help. I am looking for a file manager for the Linux desktop that can open archives in the same window. Other features are welcome, extract, create archive, etc. But I want to be able to double click on an archive from inside the file manager, have it open in the same window, and perhaps cut/paste drag/drop from other file manager windows. A file manager that supports multiple tabs, or file view panels is even better. I have looked at Xfe, Thunar, Gnome-Commander, PCManFM, and some others. If I missed an option somewhere please let me know! Thanks for the help.

### Post by oldos2er on 2013-05-12
Dolphin has this ability, check the menus Control, Configure Dolphin, Navigation, tick the box next to 'Open archives as folder'
