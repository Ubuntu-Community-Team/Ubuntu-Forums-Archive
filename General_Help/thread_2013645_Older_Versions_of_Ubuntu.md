### Post by rickm1945 on 2012-07-01
I am using Ubuntu12.04 on my Dell Desktop but I can't run it on my new HP laptop. The CD won't even boot. I have an Intel integrated graphics card and it has proved to be a real pain. Is there an older version of Ubuntu that I might be able to use that would get along with the Intel graphics? If so which one would be the best?

---

### Post by sudodus on 2012-07-01
Since it is a new HP laptop, I think you have the best chances using the new(est) Ubuntu (to detect and manage new hardware). But you may need one or more boot options, for example nomodeset. See this link

[[COLOR="Red"]_https://help.ubuntu.com/community/BootOptions_[/COLOR]]("https://help.ubuntu.com/community/BootOptions")

So ***try*** running a live session from your boot CD or USB drive.
