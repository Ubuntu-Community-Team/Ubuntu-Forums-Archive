### Post by grahammechanical on 2011-05-30
May I point out that MS Windows normally cannot read and write Linux partitions. But a Linux OS such as Ubuntu will be able to read and hence copy data from both Windows and Linux drives/partitions. This is why it is being suggested that you use a Ubuntu Live CD on that windows machine and use it to copy data off of the old drive on to the new drive or even a CD/DVD.

Regards.
