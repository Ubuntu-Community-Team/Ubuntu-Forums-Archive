### Post by blazemore on 2010-05-18
Welcome to the forum.

If you are running Google Chrome on Ubuntu, and download the themes from the Google Chrome Extensions website, they will install automatically, like any other extension.

If you download them from some other source, just dragging them onto the Chrome browser window should make Chrome install them.
