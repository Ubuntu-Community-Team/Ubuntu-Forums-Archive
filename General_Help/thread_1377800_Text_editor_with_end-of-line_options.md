### Post by waterboy12 on 2010-01-10
I'm an Ubuntu newbie and I'm looking for a text editor that has options for DOS-like or Unix-like end of line characters.  I'm used to working with Notepad++ for windows that has an option for Unix/Mac/DOS end-of-line characters.

It would be great if I could find a text editor for Ubuntu that has this built in or as a plugin instead of running the file through a converter. 

Thanks for you help!

---

### Post by sgosnell on 2010-01-10
The built-in gedit does that.  To use DOS EOL, save the file as a .txt file.  To use Unix-style, save as anything else.

---

### Post by rCXer on 2010-01-10
Also consider trying kate.  End of lines can be changed by going to Tools->End Of Line. No plugins needed.
