HI!
Well, that's the "hard way". The "easy way" would be to right-click on the panel you want, select **Add New Item**, look for the **Task List** and drag it to wherever you want it. BTW, if instead of having those long buttons with the applications name you want just the icons, you can use the **Icon Box**.
Cheers!
