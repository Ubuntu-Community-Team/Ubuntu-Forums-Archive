### Post by wildmanne39 on 2011-08-09
Hi, two things, you are missing firmware for your driver, and you have the hardware switch off.

To turn the switch on it has a button on the laptop or you need to hit like fn f11 or something like that.

To install the firmware, you need to do this make sure multiverse has a checke by it in synaptic, then run this command.
```
sudo apt-get install b43-fwcutter && sudo apt-get install firmware-b43-installer
```
Also I believe you had tried other drivers, so we might need to blacklist them.

After you run that command disconnect your wired connection then restart your computer.

If it does not work post the results of this command.
```
lsmod | grep b43
```
