### Post by HermanAB on 2015-05-19
For marking up PDF files, I use Xournal.  By default, Xournal keeps the markup as an overlay file, separate from the original PDF, but you can export to another PDF or picture file.  One can also insert pictures into a PDF this way (I use that to add a signature to a PDF for example), so I suppose one could insert a photo into an empty (new) Xournal file and then add markup to it, then export the result to a picture or PDF.
