### Post by Toz on 2012-07-06
Yes, though aren't really documented (see this post from 2008: [http://ubuntuforums.org/showthread.php?t=982992]("http://ubuntuforums.org/showthread.php?t=982992")).

Basically, the RSS feed for everything is [http://ubuntuforums.org/external.php]("http://ubuntuforums.org/external.php").

The RSS feed for individual forums is in the format of **[http://ubuntuforums.org/external.php?forumids=xxx](http://ubuntuforums.org/external.php?forumids=xxx)** where xxx is the forum ID that you can get by hovering over the name (in the url of the sub-forum) on the main page. For example:
- General Help (331) = [http://ubuntuforums.org/external.php?forumids=331]("http://ubuntuforums.org/external.php?forumids=331")
- Networking & Wireless (336) = [http://ubuntuforums.org/external.php?forumids=336]("http://ubuntuforums.org/external.php?forumids=336")
- Ubuntu +1 (416) = [http://ubuntuforums.org/external.php?forumids=416]("http://ubuntuforums.org/external.php?forumids=416")
