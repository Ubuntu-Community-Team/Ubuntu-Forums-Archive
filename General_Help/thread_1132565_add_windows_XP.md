### Post by cornstalk on 2009-04-24
You don't say why virtual box didn't work.  Are you aware that you must have a copy of Windows to install on the virtual machine that virtualbox creates?  Also you need a fairly muscular computer to run a virtual machine.  You can go to the virtualbox website and find out all about this.

Your other option is a dual boot.  Under this, when you boot your machine, you are presented with a menu, and you select which system you want to boot to.  This is a big subject and you should study carefully before doing this, particularly on the subject of boot loaders (grub is the best one).  This will involve re-partioning your hard drive.  A previous post has indicated a reference that you can use to get started.
