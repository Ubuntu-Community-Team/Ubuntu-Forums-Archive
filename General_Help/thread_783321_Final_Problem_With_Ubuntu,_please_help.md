### Post by eragon100 on 2008-05-05
I had this problem with a lot of websites when I installed hardy, way to fix (and now, this doesn't make all letters to big :) ):

In firefox, go to Edit -> preferences. Then to tab "Content"

At default font, select "sans-serif", Size: 20

Problem solved :guitar:

I checked the website you named with those settings, looks fine to me :)

### Post by gs_berg on 2008-05-05
Did you try View -> Text Size -> Increase in Firefox.  You can repeat it as many times as you want until you get the font size that you like.  To go back to normal size, go to View -> Text Size -> Normal.
