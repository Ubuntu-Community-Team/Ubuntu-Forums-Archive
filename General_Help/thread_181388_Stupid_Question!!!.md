### Post by katiad on 2006-05-25
The only real difference between Kubuntu and Ubuntu is the default graphical environment. A "real" Kubuntu is really the same thing, essentially, as a "real" ubuntu. They're both equally well. They're both k/ubuntu. You can use either. It doesn't transform anything. It's not quite a theme - it's much more than that - but it doesn't change the underlying system. It only changes which environment you use to interact with it. 

The only difference between Kubuntu and Ubuntu is this: When you install Kubuntu from a kubuntu cd, it will automagically install kubuntu-desktop (kde), not ubuntu-desktop (gnome). When you install from an ubuntu cd, it automagically installs ubuntu-desktop, not kubuntu-desktop. But you can have both. The cds just don't come configured to install both from the start. They wanted to simply things and only install one, so new users didn't get all confused in the beginning. There are really no other differences between the two otherwise. 

So, to answer your question - if you install kubuntu-desktop, yes, you will be using your computer just like a "real" kubuntu. You'll be using the ubuntu linux operating system with KDE. When you switch your session at login to gnome, you'll be using the "real" ubuntu. They're really the same thing. :)
