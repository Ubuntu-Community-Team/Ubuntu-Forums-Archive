### Post by adam.ec on 2009-01-12
The only problems I had with the 64bit version were with Java (as a developer) but with the proper open source solutions available now it doesn't look like there are any issues with it. Flash works fine, wireless works out of the box for me and I have seen some little, tiny, weeny speed increases in some virtualisation products.

I don't know if it's related to the 64 bit version of Ubuntu or not but I never had problems running apps in Wine on the same machine with 32bit Ubuntu, but nothing seems to work too good now with my new 64bit install.
