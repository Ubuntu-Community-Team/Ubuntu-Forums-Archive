### Post by Badger32d on 2010-04-07
@thomashw,

Is this an old computer? Sometimes older computers do not support sleep mode in Linux. 

I've noticed with a couple of computers that I have installed various versions of Linux on that they don't come back out of sleep/standby mode.

Try disabling sleep/standby. That should help.

The fact that this is a recurring problem leads me to believe that it is a hardware issue, not a software issue.
