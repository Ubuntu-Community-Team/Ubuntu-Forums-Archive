### Post by hedgefighter on 2007-08-10
It depends. You can use either one. Generally, you want to install the Ubuntu Server. It has Apache, MySQL, PHP, and DNS included. You just tell it what you want when you install. Just remember, it's a server, so it has no GUI by default. You'll be dropped to a terminal login. If you're not familiar with this, you might try this walkthrough: 

[http://www.howtoforge.com/perfect_setup_ubuntu_6.10](http://www.howtoforge.com/perfect_setup_ubuntu_6.10)

It's for the last version version of ubuntu, but it should be basically the same. Good Luck!
