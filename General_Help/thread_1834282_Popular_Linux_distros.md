### Post by .... on 2011-08-27
- One rolling-release distro (Arch, aptosid)
- An RPM distro or two (Fedora, suse, Mageia)
- Deb distro or two (aptosid, Mint, vanilla Debian)
- A BSD if you're into server stuff.

If you're really masochistic, try LFS or Gentoo (Sabayon for easier version of Gentoo).

Personally, I like to try things in Virtualbox before I bother doing an install to the bare metal. If I don't like the package management (I'm not fond of RPM), then it's a lot easier to clean up.
