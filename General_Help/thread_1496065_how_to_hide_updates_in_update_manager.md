### Post by Dn4mycrownj on 2010-05-28
In update manager I have some updates that I do not want to download ever. Is there any way that I can make certain updates hidden on the list. Or make them not be clicked automatic. Right now I unclick them every time my lazy         wants it automatic. I am using 9.1.

---

### Post by WorMzy on 2010-05-28
Highlight the package in synaptic, and in the menu bar go Package -> Lock Version. You need to do this for every package, however you can lock multiple packages at the same time by highlighting them all first. I believe you can list all the packages with updates available if you go to Custom Filters -> Upgradable.
