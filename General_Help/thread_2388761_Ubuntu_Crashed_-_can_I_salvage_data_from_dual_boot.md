### Post by yancek on 2018-04-06
> it hangs on the login screeen asking for a password for encryption or something.

If you have an encrypted partition, you should have a password you created, use it.  What exactly do you see on screen?

It is highly unlikely you can do anything on Ubuntu/Linux from whichever windows version you are using because a standard windows system is unable to read or write to a Linux filesystem.  Better off trying to resolve the problem or use a Linux Live system on DVD or flash drive to access your files.
