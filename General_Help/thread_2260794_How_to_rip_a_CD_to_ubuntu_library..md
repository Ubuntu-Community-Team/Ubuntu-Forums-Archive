### Post by Lars Noodén on 2015-01-14
Sound Juicer worked well for me to rip to flac.  You have to go into its settings to tell it the default format for ripping, flac is one of the options there.

For player, I use Clementine with the flac files.

### Post by Dennis N on 2015-01-14
Depending on the ripper/encoder, you may need to install some additional libraries if the format you want is not available.
Examples:
**flac** for flac encoding
**vorbis-tools** for ogg encoding
**lame** for mp3 encoding
