### Post by Teh Ethan on 2005-04-30
Applications Menu > Sys. Tools > Terminal

Type these commands:
1. sudo passwd root [follow prompts to set admin password]
2. su [enter password]
3. apt-get install xmms
4. exit

Applications Menu > Sound and Video > XMMS
