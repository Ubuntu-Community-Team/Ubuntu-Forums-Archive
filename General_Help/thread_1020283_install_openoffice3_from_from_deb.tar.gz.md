### Post by fragos on 2008-12-24
I recommend you install Ubuntu Tweak -- [http://www.getdeb.net/release.php?id=3597](http://www.getdeb.net/release.php?id=3597). Run Ubuntu Tweak and select Applications-> Third-Party Sources-> check the OpenOffice.org box. The repository for OO 3.0 will be added to software sources and Update Manager will offer you the chance to upgrade from OO 2.4 to 3.0. Perhaps a low tech GUI approach but one that gets the job done and through Ubuntu Tweak offers a number of other goodies without having to go to the command line. You may also want to check Medibuntu aand Google to get those repositories as well -- all the free stuff that isn't GPL licensed.
