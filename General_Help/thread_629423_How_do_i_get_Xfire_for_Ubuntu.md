### Post by Yankees16 on 2007-12-23
Did you save it to Desktop? If you did, you would have a .deb file on your Desktop that you could have just clicked on and installed. After it is installed, It will show up in Pidgin. But before it does, quit Pidgin. Next, Go to Applications > Internet > Pidgin. Bring it back up. Hit ctrl + A. Add an account. Under account types... There should be Xfire towards the bottom. Rest is self-explanatory. 


I'm only 13, so im not a computer expert, but that's how i got it to work. :)

P.S. I used the .deb file on my 7.10 Gutsy
