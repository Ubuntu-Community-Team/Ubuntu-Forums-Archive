### Post by Vivaldi Gloria on 2008-08-11
> **Yezinki said:**
> Can I install Ubuntu first & XP/Vista later?


It's better to install XP first, ubuntu second. Because ubuntu sees XP and adds it to bootloader. So you get a menu at the boot to choose between XP and Ubuntu.

If you install first ubuntu and second XP, then XP only puts a bootloader to boot XP and you don't get that useful boot menu. But there are threads around showing how to fix that.
