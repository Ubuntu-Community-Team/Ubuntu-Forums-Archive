### Post by aysiu on 2006-12-15
*kerry*?

It's in the universe repositories.

**Step 1.** Enable extra repositories by following these directions:
[http://www.psychocats.net/ubuntu/sources](http://www.psychocats.net/ubuntu/sources)

**Step 2.** Install *kerry* by pasting this command into the terminal: ```
sudo aptitude update && sudo aptitude install kerry
```

**Step 3**. This thread might help you get up and running:
[Starting beagle deamon at Kubuntu startup](http://ubuntuforums.org/showthread.php?t=153326&highlight=kerry+beagle)
