### Post by inameiname on 2010-05-03
You are right. There are several threads with people with the same question, and asking how to put them back to the right. 

The Ubuntu folks themselves decided to apparently, as a way to limit accidentally hitting the 'X' button. Stupid to me, but whatever. It's an easy fix. Also, many themes automatically put them on the right so it doesn't matter.


And there are several ways:

Use Ubuntu Tweak, use the gconf-editor, use a script. I personally use this script I found:

[http://gnome-look.org/content/show.php/Change+Window+Buttons?content=123360](http://gnome-look.org/content/show.php/Change+Window+Buttons?content=123360)
