### Post by quicknick_26 on 2009-02-15
You should be able to hit <ctrl>+<alt>+<backspace> to restart Gnome. I have a Viewsonic VX2233wm myself. The only fix I could find was to boot into the recovery console from GRUB and select the fix X server option. This gets Gnome back up on the screen, but deactivates the NVIDIA 180 driver.
