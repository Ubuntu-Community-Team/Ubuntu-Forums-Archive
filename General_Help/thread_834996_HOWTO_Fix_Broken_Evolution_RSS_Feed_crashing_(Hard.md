### Post by TitanKing on 2008-06-20
This is a very simple solution for the much irritating bug where Evolution crashes when certain RSS feeds are added like Digg in Ubuntu Hardy 8.04.

Download this package [evolution-rss_0.0.8-2_i386.deb]("http://packages.debian.org/sid/i386/evolution-rss/download") (different then the default 0.0.8 Ubuntu version), when completed, double click on it and install it.

Restart Evolution, your problem should now be fixed.
