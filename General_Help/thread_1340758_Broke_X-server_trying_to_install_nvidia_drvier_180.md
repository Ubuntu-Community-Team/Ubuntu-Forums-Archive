### Post by blur xc on 2009-11-29
Ok- done more digging, and it turns out that it installed the 180.60 driver, but failed to properly install the 180.60 kernel module.  So, I have two options that I can figure- uninstall everything nvidia, and start over, or try and fix the kernel module?

### Post by blur xc on 2009-11-29
Ok- I guess I'm back to where I sarted (fingers crossed, knock on wood, etc..).  I ran "sudo sh NVIDIA* --uninstall" and after about 15 seconds I was back at my login screen, and after logging in all 3d performance has been restored.  Lesson learned- leave well enough alone...
