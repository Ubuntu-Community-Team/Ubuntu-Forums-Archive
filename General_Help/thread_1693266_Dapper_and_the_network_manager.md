### Post by Zill on 2011-02-22
DBsnare07:  Unless you are running a server-only system, Dapper (6.06) support ended on July 14 2009.  Even the server support for Dapper ends in June 2011.  See [Releases]("https://wiki.ubuntu.com/Releases") for full details.

If you are running Ubuntu 10.10 (Maverick) then it is unwise to mix up repositories from other releases, such as 6.06 (Dapper).  I suggest you concentrate on getting your system working properly using just the 10.10 repositories and delete any other repos in your sources list.

Have you tried connecting your router to the PC using an ethernet cable and then clicking on System, Administration, Hardware Drivers to establish if a wifi driver if available?  Please advise the results of this and, if no driver is found, advise what wifi hardware you are using.
