### Post by ABCC on 2008-07-29
Dual booting is both safe and easy. If you want to have access to both your best is to buy the pc with a windows license included. Once you receive it you can boot windows to see if everything works and to try it out. The best time to set up the dual boot is right at the beginning though, don't get carried away with installing things just yet.

Once you've seen it all working you need to do two things. The first is to pop in the Ubuntu cd and reboot. The other is to decide how little harddrive space you're going to let Windows mess around with. In the installer you're asked if it's ok to wipe the entire disk to install Ubuntu. Just say no and use the graphical tool to shrink the windows partition (as small as possible :p) et voila!

Oh, and don't forget to have fun setting it up and marvelling at how easy it is.

ABCC

### Post by amenszer on 2008-07-29
When you set up Ubuntu, you will have the option to resize the Windows partition. During the 'Setting up the disks' section of the installation, it will show a disk meter that you can simply drag back and forth to adjust the size...in turn freeing up space for your Ubuntu install.

Back in the old days, I remember using QTParted to do that, but partition resizing is built into Ubuntu's disk manager.
