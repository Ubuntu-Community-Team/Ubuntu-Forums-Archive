### Post by sanderd17 on 2011-09-16
+1

Burn Ubuntu to a CD and try it. 

Try if you can edit documents like you want, if you can get your hardware to work etc.

A live USB might be even better because you can save your configuration on it (a live CD just forgets everything when you reboot)

Use this to create a live USB: [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

### Post by JD8182 on 2011-09-16
Go to: [http://www.ubuntu.com/download/ubuntu/download]("http://www.ubuntu.com/download/ubuntu/download"), In my opinion, when choosing your version, choose 10.04 if you are new to Ubuntu since it is the longterm support version, then choose either 32 or 64 bit version. Then scroll down to section 2 where it says: Burn your CD or create a USB drive. follow the instructions and click (Show me How) for instructions in creating a live CD or USB. Hope this helps, Enjoy..Post back with any questions.
