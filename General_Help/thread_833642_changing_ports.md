### Post by plucky on 2008-06-19
> **shewy said:**
> how can i change the port numbers for incoming and outgoing mail with evolution mail to set it up with my new email?

When you set up your mail servers you put the port number at the end.

For example the server is pop.googlemail.com:465

Where 465 is the port number.

And the same for the SMTP server smtp.googlemail.com:997


In evolution,to change the account you have already set up **Edit > Preferences > Mail Accounts > Select Account > Edit** and select the receiving and sending tabs and add the port numbers to the end of the server information.


Good Luck
