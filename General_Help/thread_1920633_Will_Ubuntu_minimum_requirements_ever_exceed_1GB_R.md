### Post by aasdfasdfg on 2012-02-05
"ever" is a very strong adverb! Modern PCs are designed to be replaced periodically as the state of current technology increases. It is hard to see this trend ending anytime soon.

With that being said, the term "Ubuntu" is vague here. I am assuming you are referring to a graphical ubuntu-desktop x86 experience. While there may be a point in the future when it becomes difficult to run a full ubuntu-desktop install on 1GB of RAM, users with limited hardware capabilities always have options. For instance, the ubuntu-minimal experience involves a ~30Mb download and can run on 64Mb of RAM. From a ubuntu-minimal install, the user can enable extra features as they see fit using the tasksel utility.

So to summarize, the answer to your question is that you will always be able to run some form of Ubuntu; however, the default requirements may outstrip your hardware at some point.
