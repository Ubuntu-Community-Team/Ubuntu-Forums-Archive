### Post by Solarius on 2008-11-26
Just type "wget url" and replace url with the address of the website, like "wget www.google.com".

If you want to then use the source for something, you can pipe it into a file (like "wget [www.google.com](www.google.com) > ~/my_folder/blahblah/page.html") or into another program.
