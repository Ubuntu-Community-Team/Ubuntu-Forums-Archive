### Post by drs305 on 2009-02-18
You can change the server via Software Sources or Synaptic. Settings > Repositories - Ubuntu Software tab: Toward the bottom, Download From: Click on whatever is currently selected, select Other, then choose the one you want or let "Select Best Server" run a test and select one for you.

### Post by heindsight on 2009-02-18
Go to System>Administration>Software Sources
It should ask for your password, so enter your password and click OK

There will be an option saying 'Download From: Main Server'
Click on 'Main Server' and select 'Other...' from the drop down list. 
Now you'll be given a long list of servers to choose from as well as the option to select the best server.
