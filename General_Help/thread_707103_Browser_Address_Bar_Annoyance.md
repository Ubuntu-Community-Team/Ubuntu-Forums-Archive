### Post by deviant78 on 2008-02-25
Under windows when you click in the address bar of a browser in automatically selects the entire address.

Under Ubuntu is place the cursor where you click without selecting anything.

Is there a way to get it to behave in the windows way?

Btw, using ubuntu 7.10 64 bit and Opera.

### Post by pointone on 2008-02-25
Double click the address bar or use the "Ctrl+L" shortcut.

There's probably an option in opera:config to enable this, though. Search around in there, perhaps?

---

### Post by aysiu on 2008-02-25
Type ```
about:config
``` in the address bar

Look for browser.urlbar.clickSelectsAll and double-click it to make it true instead of false.

**Edit**: Whoops. You're using Opera and not Firefox.
