Install VMWare Server (or any other free VM system) and install as many distros as you have free disk space.

If your hardware is good enough you can run multiple OSs simultaneously as well as your host OS.

I once had XP, 32bit Ubuntu, 64 bit (beta) Ubuntu and 2 other Linux distros in VMs (although things do slow down when ALL of them are up and running....)
