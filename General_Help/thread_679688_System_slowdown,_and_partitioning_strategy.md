### Post by bobbybobington on 2008-01-27
AS far as I know, ubuntu shouldn't need anything to make it go faster. Check and see if the slowdown isn't hardware related, make sure your swap partition is big enough, and run memcheck86 (it's on the install disk under "check memory for defects" or something similar) As far as reinstalling without backup, I would adivse against it. You can make your /Home folder a partition , but you should ALWAYS back everything up in case your hard drive dies or the partitioning gets messed up.
