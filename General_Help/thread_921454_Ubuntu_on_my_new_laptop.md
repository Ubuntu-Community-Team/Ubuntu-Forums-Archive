### Post by sethvath on 2008-10-23
Do not expect everything to work out of the box, do a search for your laptop model here in the forums. It is likely someone else with the same computer has already tried to install ubuntu on it. 

As for your sound card are you able to verify the model name and number in windows? If so do a search here as well. If not you can try with the live cd, open a terminal and type in "lspci"

There are numerous tutorials on how to get audio working, please verify which model you have and do a search for it here.
