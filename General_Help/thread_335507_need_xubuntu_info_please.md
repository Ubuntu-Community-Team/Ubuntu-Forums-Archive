Xubuntu does not have gedit, although you could add it with Synaptic. Xubuntu does have nano in a terminal so you can do sudo nano /etc/apt/sources.lst
However, you can really do that in Synaptic's interface if you want to, without editing in a text editor.

The Xubuntu desktop has several different interfaces to change various parts of the desktop. Under the settings menu you will find Desktop settings, user interface settings, Window manager settings, Display settings, etc. I believe you can change the Desktop background with the Desktop settings interface. (i'm not on my Xubuntu machine right now, but I think that is the right place)
