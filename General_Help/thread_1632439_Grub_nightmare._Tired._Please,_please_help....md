### Post by lmarmisa on 2010-11-28
Boot your system into Ubuntu Live CD, open a terminal ant type:

```

sudo bash
mount /dev/sda1 /mnt
mount --bind /dev /mnt/dev
chroot /mnt
grub-install /dev/sda
exit
exit

```

I believe that your GRUB loader will be available again.

Boot normally, open a terminal and type this command:

```

sudo update-grub

```

Check if GRUB menu allows you to start Windows 7 too.
