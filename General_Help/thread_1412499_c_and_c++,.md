SciTE has a panel for showing the output. It's quite simple actually. The UI features two panes: one editor, and one command line. You can type up your file in the editor, and compile/run in the command line panel using menu options, or simple keyboard shortcuts. It's pretty simple for new users, unlike a full-fledged IDE - which would have a much steeper learning curve.
