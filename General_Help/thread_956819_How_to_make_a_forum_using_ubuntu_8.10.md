### Post by tubezninja on 2008-10-23
The first thing you're going to want to do is look at the various forum software packages out there.  Some of the more common platforms are [FluxBB]("http://fluxbb.org/") which is pretty straightforward, and [phpBB]("http://www.phpbb.com/") which is pretty widely used.  This forum is powered by vBulletin, but that's commercial software and you'd have to buy it.

There's a whole listing of different forums software packages [here]("http://en.wikipedia.org/wiki/Comparison_of_Internet_forum_software_(PHP)").  But if you're just starting out, I would stick with one of the above two.

Installing phpBB is fairly straightforward on an ubuntu server.  [This tutorial should get you going on phpBB]("https://help.ubuntu.com/community/PhpBB2").  

I couldn't find a FluxBB package in the ubuntu repositories, but you can download a copy from their site and set it up manually, if you feel able.

Not unless you want to make any changes to how the forum functions.  If you're fine with it working the way it does out of the box, then you don't need to have any programing skills.
