### Post by lisati on 2007-07-12
> **Dark Wizrd said:**
> I just installed ubuntu 7.04 and my usb mouse has been freezing randomly, as soon as I start up between 2 and 5 minutes it stops moving completely, however the keyboard works. I have tested this with a logitech and a memorex mouse, are there any fixes because I really want to stop using windows and make the full switch. Thanks in advance.

I had a similar problem on my laptop - I think it could be a usb thing, the last time it happened to me the USB keyboard I had plugged in froze about the same time the mouse froze. Some of the solutions I've seen involve tinkering with ACPI settings. The solution that worked on my laptop (derived from other places on this forum, which might not suit everyone) involved editing /boot/grub/menu.lst as follows:

1 -From a terminal, type:
```

sudo gedit /boot/grub/menu.lst

```
2. Change the line that reads:
```

# defoptions=quiet splash

```
to read
```

# defoptions=quiet splash acpi=force irqpoll

```
3 Save the file and exit GEDIt
4. At the terminal, type
```

sudo update-grub

```
5. Restart computer


p.s. I just noticed a typo in the changed line in my menu.lst which still resulted in the mouse working!....Weird, huh? One of life's mysteries!
