### Post by vasa1 on 2011-10-23
There are quite a few ways to do this for those who find it a bit difficult to resize windows on the first (or second or third) attempt.

One is via Compiz Config Settings Manager. In this, there's "Unity grab handles" which give large handles for resizing windows.

And several approaches have been discussed [here]("http://askubuntu.com/questions/4109/how-do-i-increase-the-resize-margin-on-windows"): Of those, I like the one which suggests editing an .xml file. The advice is for Ambiance which just happens to be what I'm using. I suppose the same approach may work for other themes.
