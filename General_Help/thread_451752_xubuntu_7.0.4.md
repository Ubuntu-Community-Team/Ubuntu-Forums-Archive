### Post by maxamillion on 2007-05-22
Xubuntu is a child distribution of Ubuntu who's development cycle begins with the ubuntu-base, and therefore it is 100% compatible with anything that Ubuntu is, since it technically "is ubuntu", just with a more lightweight set of default packages, a different desktop environment, and a completely community driven development cycle. Xubuntu shares the ubuntu repositories so when you look in /etc/apt/sources.list or use synaptic you will find all the same software packages.

So, no you will not have to learn anything new.
