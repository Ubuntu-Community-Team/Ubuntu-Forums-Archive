### Post by Cliff_Simonds on 2014-08-02
You haven't said which *buntu flavor(desktop) you're using. But Dennis N said in post #2.. you need to install Unity Tweak Tool, Gnome Tweak Tool(both in software center), or Ubuntu Tweak(link: [http://ubuntu-tweak.com/](http://ubuntu-tweak.com/) )  to activate it.
 Which ever tool you use, go to themes and in the dropdown menu should be your theme. These tools are also to be used when you install icon themes too. 
For example in ubuntu tweak go to Tweaks->Appearence->Theme->GTK Theme->drop down menu: find **numix bluish theme **and it will be turned on. Ubuntu tweak has alot of other tools to personalize ubuntu also. This is just an example, if your using another flavor it should have it's own "tweak tool". I'm not sure (haven't used mint in a long time) I think mint comes already with it's own tool.
