### Post by DHSdrone on 2009-05-09
After countless searches using one of the greatest tools available; Google and our very own Ubuntu forums and good 'ole dumb luck I've managed to get Gmote running. Yep, running, well mostly. I've ran into most of the Java issues that seem to be common with this app, but I don't think it was the hold-up. Here's what I've done. I downloaded the app from [http://www.gmote.org/server](http://www.gmote.org/server) and extracted the .zip in its current folder, which was Documents. It contains the 2 folders and the 2 other files that end in .sh and .sh~ . I opened the terminal and typed the following: *NOTE Folder and File Names are CASE-SENSITIVE.
cd /home
cd "your user name" without quotes
cd Documents 
cd GmoteServerLinux2.0.0
sh GmoteServer.sh

With this above I've managed to start a gmote server on my jaunty box and connected to it with my G1. Now I haven't been able to use every function that I've seen in the youtube videos but I would assume it's a lack of understanding the program completely.
I hope that my attempt to give back to the open source community will help another person who may have this same issue.
