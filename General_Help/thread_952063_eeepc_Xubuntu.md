### Post by quill3033 on 2008-10-18
I have eee pc 1000h and installed Mandriva 2009 (kde) onto an 8gig pendrive and everything pretty much worked from the start - internet, skype, mp3, flashplayer etc. Did not try wifi as I'm in a pretty remote area.
Now I've installed the "ubuntu eee 8.04.1" onto an 8gig usb drive and it's working pretty well also - Skype, internet, office everything is really good. I found the sound to be better on ubuntu than on mandriva - just louder and clearer (though this may all have to do with settings, I don't know) Don't know about Xubuntu though.
I have an external dvd burner so it was easy to just install from the live cd. 
I have kept Windows only because I have a textbook on it that uses VitalSource Bookshelf and I wasn't able to get it running with Wine on my desktop computer. 
I find that installing onto the usb drive is handy as it gives me a chance to really test the system out before touching my hard drives. I'll wait a few months before I try anything on the actual hard disk.
