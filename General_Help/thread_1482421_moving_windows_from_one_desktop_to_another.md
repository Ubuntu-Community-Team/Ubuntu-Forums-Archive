### Post by blazemore on 2010-05-13
If you're in Firefox, click [here]("apt:compizconfig-settings-manager") to install the desktop effects settings manager.
Run it from Settings -> Administration

Go to the Desktop Wall plugin and click it. 
There's an option in there *somewhere* to enable viewport switching on window move, and also on drag-and-drop, which you may also appreciate.
