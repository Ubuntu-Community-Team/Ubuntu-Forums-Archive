### Post by Mark Phelps on 2012-02-14
> **Rawrzarz said:**
> Ok this is a "Just in case" question. Is uninstallation easy? i have seen some videos that require alot of effort. Is their a easy way?

The actual answer is -- depends -- on WHAT you want to uninstall and HOW you installed it.

Easiest example is installing Ubuntu using Wubi inside MS Windows.  To uninstall, just remove it like any other Window app.

Hardest case is uninstalling an application whose executable you built from source.  Uninstalling could require tracking down every single file the app used and manually removing it.

Other examples depend on WHAT and HOW.
