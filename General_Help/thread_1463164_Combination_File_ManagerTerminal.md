### Post by antenna on 2010-04-26
There is a package in the repositories called nautilus-open-terminal, that adds an "Open in Terminal" to the nautilus right click menu so you just click that in a nautilus window and it opens a terminal in that location.  It's perhaps not quite as convenient as Dolphin (I haven't used Dolphin), but it might be useful.

I like nautilus these days, it used to be a lot worse.
