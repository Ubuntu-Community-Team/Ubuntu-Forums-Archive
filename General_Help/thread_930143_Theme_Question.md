### Post by nkri on 2008-09-25
I would get emerald theme manager:
```
sudo apt-get install emerald
```

and then use the theme I attached (once emerald is installed, go to System>Preferences>Emerald Theme Manager; there is an import button at the top right...after unzipping the attached file, select it and click Open).

The Humanoid theme is the best I've found for Metacity (the theme manager which ships with Ubuntu), but DreamAccurate OSX (_[http://gnome-look.org/content/show.php/Dream+Accurate+Tiger?content=50605](http://gnome-look.org/content/show.php/Dream+Accurate+Tiger?content=50605)_), the one attached, is far better and is as close as you can get to the real Mac OS X theme; I tweaked it a little to make it look more accurate, but it's pretty much the same as the link.

Good luck!
-nkri
