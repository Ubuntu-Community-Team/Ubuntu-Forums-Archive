### Post by bcbc on 2010-09-30
wubi.exe is version specific - if you have the latest it will only install 10.04.1 
Burn the 9.10 .iso to cdrom and run it from there or else find the old wubi.exe for 9.10 (e.g. here [http://people.canonical.com/~evand/wubi/karmic/](http://people.canonical.com/~evand/wubi/karmic/), rev170 i guess)

Also if you install 9.10 you need to replace the wubildr as the one on the release .iso is buggy. [http://sourceforge.net/apps/mediawiki/bootinfoscript/index.php?title=Boot_Problems:Wubi_9.10](http://sourceforge.net/apps/mediawiki/bootinfoscript/index.php?title=Boot_Problems:Wubi_9.10)
