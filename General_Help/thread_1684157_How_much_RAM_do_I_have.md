### Post by DanneStrat on 2011-02-08
What windows version was used? 32 bit or 64 bit?

And what version of ubuntu are you running.

32-bit operating systems can use a maximum

of 3 GB. It can be increased with the help of Physical Address

Extension.

64-bit can use much more memory.

In short the windows version may be 64-bit

and the ubuntu version 32-bit and therefore ubuntu

can't make use of all the physical ram and displays less.


EDIT: As Rhizoid said, have a look in bios setup and see how much memory you have.
