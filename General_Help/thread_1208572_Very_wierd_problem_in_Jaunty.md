### Post by DawieS on 2009-07-09
Probably a RAM problem starting to develop. I had a similar problem with one of my RAM sticks starting to fail after 2 years of use. To check it out, do the memory test at the boot menu.

There is about 8 different tests. Run each test for a minute or so and continue with the next test. Press the keys shown at the bottom of the screen to do this.

If at any time red bars with faulty addresses come up on the screen, shut down.

To identify the problem RAM, unplug the RAM sticks, leaving only one in the first slot and repeat the test.
