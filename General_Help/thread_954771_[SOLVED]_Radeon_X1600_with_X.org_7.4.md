### Post by loomsen on 2008-10-21
*edit*

```

Section "DRI" 
        Group       0 
        Mode        0666 
EndSection

```
I read this section quite often with ATi. Try appending it to the end, and you should add the modules section as well

Try adding optional modules as well:
```
Section "Module" 
   #Load   "GLcore" 
   Load   "bitmap" 
   Load   "ddc" 
   Load   "dbe" 
   Load   "dri" 
   Load   "extmod" 
   Load   "freetype" 
   Load   "glx" 
   Load   "int10" 
   Load   "type1" 
   Load   "vbe" 
EndSection
```
```

Section "Extensions" 
   Option   "Composite"   "Enable" 
EndSection 

```


deprecated options will just be passed.

Actually my xorg looks similar to the one you linked to,tho I got a nvidia.

but... what about:
(II) AIGLX: Screen 0 is not DRI capable?

This will be a problem if so...



But, what is even more important is probably 

(WW) RADEONHD(0): Option "DRI" is not used

DRI specifies direct rendering capabilities.
