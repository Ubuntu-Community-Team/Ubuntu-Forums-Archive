### Post by karatedog on 2010-02-12
Found it! 
JTwigg, do you happen to use Cairo-Dock and Clipper in it? I unchecked "enable actions" in Clipper configuration and the problem is gone. I set "Which items should be remembered?" to Clipboard (from Both) so selecting text won't be saved.
