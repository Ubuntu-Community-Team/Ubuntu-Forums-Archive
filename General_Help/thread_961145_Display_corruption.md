### Post by Olivier Lec on 2008-10-31
After more googling, 

I found this [thread]("http://forum.compiz-fusion.org/showthread.php?t=6742&page=9") on compiz-fusion forum [http://forum.compiz-fusion.org/showthread.php?t=6742&page=9](http://forum.compiz-fusion.org/showthread.php?t=6742&page=9), even if the thread and distro mentioned there seem pretty old the screenshot are really close from my problem. 

After applying the nvidia-settings ```
nvidia-settings -a InitialPixmapPlacement=2 -a GlyphCache=1 &
``` things mentioned on the last page everything seems to be ok ... :KSso far so good:KS ... So if someone else have the same problem maybe a way to go.

For the record, I have also upgraded to 8.10 this morning and before changing the settings the bug was still there.
