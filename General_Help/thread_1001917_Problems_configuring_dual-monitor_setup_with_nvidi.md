### Post by easybake on 2008-12-04
I hate editing the xorg.conf.
The way I got mine set up was fairly easy. Have you tried:

sudo nvidia-settings

*set up twinview under X Server Display Configuration*

Click "Save to X Configuration"

You should be set after a reboot.

EDIT: Your problem might stem from not having your left monitor as primary.

### Post by vizitorq on 2008-12-04
I figured it out. I had to MANUALLY edit the xorg.conf file and change Xinerama "0" to Xinerama "1"

I knew xinerama had something to do with it because as far as I know it's what allows dual monitors to work as one.
