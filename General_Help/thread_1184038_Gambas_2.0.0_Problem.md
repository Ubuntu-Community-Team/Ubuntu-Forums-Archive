### Post by kalaharix on 2009-06-11
Hi,

Gambas 2.0 is very old. What version of Ubuntu are you using?

If it is the the same as the installation of Gambas 2.8 from Ubuntu 9.04 then you must go into synaptic (system, administartion, synaptic).
Search on gambas and install the remaining components.

The problem is not with Gambas itself but with an incomplete installation package that someone has set up for Ubuntu.

Gambas is currently 2.13.1 and there is a development version of Gambas3 (currently 2.99). Both can be installed from source which is the ideal answer. I know that installation from source in Linux can be daunting but you soon get the hang of it.

The other problem is that you are not going to get quality support from the community for an old version like 2.0.
