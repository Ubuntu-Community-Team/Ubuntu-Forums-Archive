### Post by Anthony A on 2012-09-16
When I go to the extension site for Gnome Shell I get a message that I do not have an up to date version of Gnome Shell. I am running Ubuntu 12.04 and Gnome Shell version 3.4.1 which is the up to date version in Synaptic when I check. I am also using the most up to date version of Firefox when going to the site. Why is this happening?

---

### Post by Anthony A on 2012-09-16
I fixed this. Just in case anybody else has this issue what happened with me is I had plugins set to click to play in Firefox. This disables all plugins on the site you are viewing unless you click to enable them. Of course this disabled the Gnome Shell Plugin and was causing my issues. I set the plugins to run on the extension site and everything works well now.
