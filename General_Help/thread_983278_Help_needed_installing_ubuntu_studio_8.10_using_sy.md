### Post by LowSky on 2009-03-21
Ubuntu Studio is just a bunch of extra software installed to Ubuntu, not really as special as you might think

go into synaptic and type

ubuntustudio (yes all one word)

install ubuntu-studio desktop, that one is a meta package, it will install all the things needed

otherwise 

```
sudo apt-get install ubuntustudio-desktop
```
