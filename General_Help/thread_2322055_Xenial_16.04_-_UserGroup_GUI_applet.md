### Post by deadflowr on 2016-04-26
You're using it wrong.

First, to add users to groups, you need to be in the users tab.

The groups tab is for adding new groups, if you want to.

Then in the user tab, highlight a user and in the user's profile shown in the right area, the Groups will list the user's groups.
Move the mouse cursor over the listed groups and a clickable box will appear around it.

Click on the box will open a pop-up window listing every group on the machine.
Simply check which groups you want to add (or subtract) the user from and select OK.

Now all you need to do is logout and back again, to make the user gain whatever rights the new groups can give it.
As is always the case.

Hope it helps.
