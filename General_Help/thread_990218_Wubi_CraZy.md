### Post by prshah on 2008-11-22
> **yeegf said:**
> I dont have a burner.
Thats why i need Wubi

Download the Ubuntu desktop 8.10 CD image, preferably using BitTorrent. Then, mount the .iso image using virtual CD mount software for Windows (such as [Virtual CloneDrive]("http://www.softpedia.com/get/CD-DVD-Tools/Virtual-CD-DVD-Rom/Virtual-CloneDrive.shtml")), and then Autoplay the virtual drive (if it has not done so already). 

From that, choose the option to "Install within Windows".

The disk image includes both wubi and a tie-in with the CD so that you do not need to download anything.
