### Post by WorMzy on 2011-12-11
I don't understand what you mean. File extensions don't carry much weight in the Linux world, you can name a file with text in it "I_am_not_a_textfile.honest", but you'll still be able to open it with a text editor, because it's still a file with text in it. File managers generally let you associate certain extensions with different programs, so .html files open in a webbrowser, .jpg files open in an image viewer, etc., but at the end of the day, all that matters is that the program that reads the file can tell what it is; it's file extension is just a way for the user to keep track of what's what.

If you want your .xwii file to open in something other than a text editor, then right-click it, go into the properties, and change it's "Open with" value.
