### Post by cariboo on 2008-08-08
Most usb flash drives are formatted with fat16. If you want to repartition your drive you can use gparted. If you have it installed go to System-->Administration-->Partition Editor. If you haven't got it installed you can use Add/Remove PRograms, Synaptic Package Manager or the command line to install it.
