### Post by FluffyElmo on 2007-02-21
For media center use you should investigate MythTV. It basically has all the features of MS Media Center edition and then some. I've had good luck with a PVR150 capture card/remote, but if you're going HDTV you may want to examine other options.
[https://help.ubuntu.com/community/MythTV]("https://help.ubuntu.com/community/MythTV")
[http://mythtv.org/]("http://mythtv.org/")

For the mail server, Ubuntu is perfectly capable of replacing both servers. This looks like a decent guide to start with, though if you already have a how-to you prefer go with that. [http://www.howtoforge.com/virtual_postfix_mysql_quota_courier]("http://www.howtoforge.com/virtual_postfix_mysql_quota_courier")

Generally, I find Linux to be much more reliable and it has much lower resource usage than the Windows equivalent. The one downside is that most configuration is done via command line and text files. There are web based administration panels which can ease the day-to-day administration but at least someone has to become familiar with the underlying packages.
