### Post by go_beep_yourself on 2008-05-08
Yes, I've been doing a lot of research lately. I just stumbled upon a program called KTranslator and now I love it. It's the simplest easy to use one, user friendly. The gnome program is StarDict. Finding good dictionaries can be a problem though. [http://wordweb.info/free/](http://wordweb.info/free/) I read one post that this one running in Wine has good dictionaries included. [www.freetranslations.com](www.freetranslations.com) provides great translating, but you have to use the web page I think. I wish I could have their dictionaries. It tranlates a single word, a sentence, or an entire page.
