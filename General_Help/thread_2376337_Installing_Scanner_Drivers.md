### Post by phdsuppo on 2017-12-23
There is a closed thread from 2010 titled Epson Perfection V30 Scanner Installation Guide ([https://ubuntuforums.org/showthread.php?t=1447789](https://ubuntuforums.org/showthread.php?t=1447789)). there is also a 2016 update to it ([https://ubuntuforums.org/showthread.php?t=2338852](https://ubuntuforums.org/showthread.php?t=2338852)) It has a good step by step install. 
Go to epson's website [http://support.epson.net/linux/en/iscan.php?model=gt-f720&version=1.0.1](http://support.epson.net/linux/en/iscan.php?model=gt-f720&version=1.0.1) and download the deb file. Make sure you choose the correct one based on whether you have an Intel or AMD chip. For Intel the file you will download is iscan-gt-f720-bundle-1.0.1.x86.deb.tar.gz.
once downloaded you can right click on the file and say "extract here". It will create a subdirectory. Open the subdirectory and there should be a readme file which will hopefully walk you through the install. Also on the epson site where you do the download there is a link on how to install the software. Good luck!
