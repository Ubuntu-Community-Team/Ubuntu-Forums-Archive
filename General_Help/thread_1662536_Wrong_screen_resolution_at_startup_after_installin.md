### Post by tredegar on 2011-01-08
Thank you.

You need to edit this bit:```
Section "Screen"
    Identifier     "Screen0"
    Device         "Device0"
    Monitor        "Monitor0"
    DefaultDepth    24
    SubSection     "Display"
        Depth       24
        Modes      [COLOR="Red"]"1280x800"[/COLOR] "1024x768" "800x600" "640x480"
    EndSubSection
EndSection
```
Then logout and back in again to restart X.

[COLOR="YellowGreen"]**Update:**[/COLOR]
If you're having the same problem I was - the same problem the OP had, but with a flimsier looking xorg.conf - this seems to be the solution.

Start your NVIDIA Server Settings (under System>Administration>Nvidia...  or, run **nvidia-settings** )

In Nvidia Server Settings, under 
X Server Display Configuration
set your resolution to wherever you like it (and it displays properly, etc)
near the bottom, click "Save to X Configuration File" and follow the prompts.
Viola!

You should probably backup your current xorg.conf before doing this, incase it explodes.
**sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup**

and if it explodes and you need to revert to your previous xorg.conf 
**sudo cp /etc/X11/xorg.conf_backup /etc/X11/xorg.conf**
(Write these down on paper before starting.  And somene PLEASE correct me if those commands are wrong?)
