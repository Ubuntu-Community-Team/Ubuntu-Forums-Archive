I am trying to install a game that's a .run file. I opened it with sh command and went through the license agreement and all that and it goes to installation path but when I click enter it always says "no write permission to /usr/local/games/" how can I install it? I have Ubuntu 9.10

---

### Post by sakisds on 2009-11-14
Use the following command to run it when [filename] is the filename of your .run file:

```
sudo sh [filename].run
```

And for you to know, sudo gives admin(root) rights to the command you are running with it. It useful with other commands too.
