### Post by LaRoza on 2008-01-24
> **jmahan said:**
> Well I downloaded it and I was wondering if I could get some command Line code to Install the software because I'm still sort of a newb I was just wondering the command line functions that will install my software. Oh and if i were to use compiz how who would i go about configureing it?

```

sudo aptitude install compizconfig-settings-manager

```

It is already installed (Compiz) but you can get the plugins for it, and the manager to really tweak it.

It wil be in System->Preferences->Appearance and under the "Visual Effects" tab, and the setting manager will be in System->Preferences->Advanced Desktop Effects Settings
