### Post by Darael on 2009-11-24
The password will be that of the user that is sharing the file - you need to use that username as well.  However, you can reconfigure the share to allow guest access - if you do it through the Nautilus file properties box then you can enable "guest access" on the file properties tab.  I don't know what other ways there are to achieve the same thing, I'm afraid.
