### Post by Bucky Ball on 2009-04-19
One way would be to go to the gparted site and download the LiveCD version, make a CD, boot from it and wipe the Ubuntu partitions. This will leave you with only the Windoze partition but gparted will allow you to expand that or create new ones on the disk (and any other disk in the machine). :)

[http://gparted.sourceforge.net/download.php](http://gparted.sourceforge.net/download.php)

Get the 'Live' version.

### Post by cariboo on 2009-04-19
When setting a a dual boot system with Windows and Linux, I would suggest you install VIsta first, use it's partition to set up a partition for Vista. When I set up a dual boot, I usually use half the disk for Windows and and leave the other half blank. then when I install Ubuntu I install it on the blank half.

I prefer using the manual partitioning option in the Ubuntu install process
