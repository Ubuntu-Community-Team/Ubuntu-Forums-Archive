### Post by Tadaen_Sylvermane on 2017-04-11
A few differences. OMV is Debian not Ubuntu. Second, OMV has  a web interface while Ubuntu is command line access. Third, imo you can do more with Ubuntu than with debian (not trying to start a holy war).

As far as Desktop vs Server usage, don't bother with desktop if it's a server. Google can help you get set up very quickly. A gui just adds unnecessary weight to your installation. Learn to run it the right way the first time.  The command line isn't nearly as scary as it sounds.
