### Post by gordintoronto on 2011-01-18
Are the PDFs in a RAR or a Folder? (Documents is an example of a folder.) You don't need an unrar package to open a folder. I'm pretty sure Ubuntu includes a program to open RARs, right in the basic installation.

Double-click the RAR and a window should open which shows you the content. Double-click one of the files, and it should open in the appropriate application.

Note that Linux does not just use file extensions to decide what application to use, it looks at the content of the file. If it doesn't recognize it, it can't pick an application.
