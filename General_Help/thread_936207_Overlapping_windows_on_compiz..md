### Post by Therion on 2008-10-02
I *think* you could accomplish this by setting up however many desktops (you're calling them "planes") you want (2, 3, 4, 10... whatever), disabling both the **Desktop Cube** and **Rotate Cube** plug-in(s), if you have either or both enabled now (to stop the animation when switching between them) and then enabling the plug-in called **Desktop Wall** to keep an open window from crossing-over from one desktop and into another.

You would then use Ctrl+Right Arrow (or is it Alt? (stuck at work behind an XP machine at the moment)) to switch between desktops, or you could put the graphical switcher applet in one of your Gnome panels if you prefer to use your mouse for switching between desktops.
