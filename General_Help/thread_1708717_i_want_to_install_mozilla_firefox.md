### Post by lithopsian on 2011-03-17
Find your downloaded Firefox tarball in your preferred file manager.  Right click, you will see an option like "Extract to ...".  Click it and select /opt.  Firefox will be installed in /opt/firefox-something.  To execute, run /opt/firefox-something/firefox.  You can create a menu item for this.

At this point, you might want to consider installing the new Firefox 4.0 instead of 3.6  It is currently only a release candidate, but it is 99% what you will get when it is released on March 22nd (fingers crossed).  You can always downloaded a new version at any time over the top of the old one, or in a different location if you like.

And for the doubters, yes the Ubuntu repository version of Firefox is weird and modified.  Primarily it includes a plugin which does a number of bizarre things, some nice, some not so nice.  It enables you to install (some) extensions through the repositories, but it also messes with the branding, installs its own search engine (ask.com), and quite often breaks stuff.
