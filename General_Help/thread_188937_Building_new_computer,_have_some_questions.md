### Post by eqisow on 2006-06-04
You shouldn't ahve any hardware recognition problems with the hardware you've picked so far, so as long as it fits your needs, you're good to go! If you need wireless, I recomend picking up a Linksys WMP54G as I know for sure that these work out of the box. Also, if you're getting a printer, go HP. Be warned, however, that printer/scanner combos don't always work well.

Regarding the 32/64 bit question. You will still have to run flash, java, w32 codecs, etc, in a 32 bit chroot environment. If you want/need these things, I reomend sticking with 32 bit for awhile longer. Go ahead and spring for that 64bit CPU though!

P.S. - The PSU is probably a little overkill.

P.P.S. - If this computer is just for your mom, it may be overkill. What does she use her computer for?

P.P.P.S. - Newegg rocks!
