### Post by alanphil on 2006-07-15
5. Is there any way to hotkey through xfce's workspaces?

Settings > Windows Manager Settings > Keyboard tab
This configuration screen will allow you to define keyboard shortcuts to move through xfce's workspaces. NOTE: There are some defined by default, take a look at the list and modify, add new hotkey combinations to define the behavior you need.
