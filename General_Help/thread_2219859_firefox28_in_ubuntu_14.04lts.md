### Post by monkeybrain20122 on 2014-04-25
It could be caused by some addons. You can test with a new profile to see if the problem persists. Close Firefox,open the file manager, display hidden files, locate the folder .mozilla and rename it to something else like .mozilla-bak. Now open firefox to go to the problematic sites and see what happens. To revert to old profile, close FF, open file manager and display hidden files, delete .mozilla and rename .mozilla-bak back to .mozilla.
