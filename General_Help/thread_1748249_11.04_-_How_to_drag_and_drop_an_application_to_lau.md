### Post by mc4man on 2011-05-03
When you add an 'application' to the launcher it's just a shortcut to a  <whatever>.desktop.  (so they need to exist.
I'd just stick them (your new desktop(s))  in an applications folder - either /usr/local/share/applications or ~/.local/share/applications, then add to the laucher

For more than a couple of  script I'd create something like scripts.desktop, add the scripts as quicklists entries and have a scripts menu launcher

simple starting example here - 
[http://ubuntuforums.org/showthread.php?p=10741859#post10741859](http://ubuntuforums.org/showthread.php?p=10741859#post10741859)
For scripts the Exec= line would be 

Exec=/path/to/script

Edit: - if you have a number of scripts better to create a dir in home folder named bin
Then scripts added can usually just be called by scriptname (no spaces
If needing a full pathing to ~/bin then probably best to ues
/home/username/bin/scriptname
