### Post by ppirilla on 2010-02-09
First, this question would probably be more appropriate in a Pidgin support forum.

In any event, a "pounce" is an automatic action that Pidgin can take whenever your contact of choice does something.  For me, I set a pounce so that a chat window opens up whenever my girlfriend logs in.  Former roommates of mine set up what they called a pounce war, which involved sending an automated messages for every message received from the other.
