### Post by dr1094 on 2011-08-16
First, I am going to assume that 11.11 is probably a typo, since the latest release is 11.04

you can install Google Chrome by opening the ubuntu software center. Google Chrome in Ubuntu is called "Chromium." 

Assuming you don't know about the software center: If you see the "Applications" button at the top left corner, click it and then click ubuntu software center.If you don't see the applications buttons hit the 'windows key' aka super key, and just type software center and press enter to open it.


If you are unable to find chromium in the software center, you need to enable the repositories (I think all should be enabled depending on your preference). On how to enable repositories see: [https://help.ubuntu.com/community/Repositories/Ubuntu](https://help.ubuntu.com/community/Repositories/Ubuntu)
