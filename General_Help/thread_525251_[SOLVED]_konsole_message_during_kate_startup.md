### Post by forestpixie on 2007-08-14
If you don't have wacom then let's assume we're safe to proceed then.

Open a terminal 

```
sudo cp /etc/X11/xorg.xonf /etc/X11/xorg.conf.bak.wacom
```

will backup the file, lets make sure that the backup worked, in the terminal

```
ls /etc/X11/xorg.*
```

you should have at the least xorg.conf and xorg.conf.bak.wacom, probably more - if you haven't got the xorg.conf.bak.wacom - try the command again or come back here.

if after rebooting you don't get back in but only a text window you can use this file to restore xorg.conf - the command to do that would be

```
sudo mv /etc/X11/xorg.conf.bak.wacom /etc/X11/xorg.xonf
```
BUT you'll only need to do that if you have a problem.

now you can edit the file
in the terminal

```
kdesu kate /etc/X11/xorg.conf
```

should open the file in text editor so you can edit 

remove the highlighted section by deleting - and comment the 3 lines with # at the bottom
