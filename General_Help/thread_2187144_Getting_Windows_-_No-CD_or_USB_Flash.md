### Post by Mark Phelps on 2013-11-11
You can't install Windows from inside Ubuntu. The Windows seven forum (sevenforums.com) has a bunch of different installation tutorials, but all of them expect you to either have installation media (USB or DVD) or an ISO extracted to your hard drive -- and then installing that from Windows.
