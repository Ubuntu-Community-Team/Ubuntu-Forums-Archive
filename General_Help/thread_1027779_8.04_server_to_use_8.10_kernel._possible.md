### Post by sdennie on 2009-01-01
A much safer way to do this is to just add the kernel PPA.  Edit /etc/apt/sources.list and add the following to the bottom:

```

#Kernel PPA to get 2.6.27 kernel
deb http://ppa.launchpad.net/kernel-ppa/ubuntu hardy main
deb-src http://ppa.launchpad.net/kernel-ppa/ubuntu hardy main

```

Then:

```

sudo apt-get update
sudo apt-get install linux-image-2.6.27-9-server linux-headers-2.6.27-9-server

```

Reboot and choose the new kernel from grub.
