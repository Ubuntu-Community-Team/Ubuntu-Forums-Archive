### Post by audiomick on 2009-11-24
I had start problems that were a bit similar to what you describe. It sounds like _maybe_ the machine is having trouble reading the disc.
My solution was
a: to turn off all the fast start options in BIOS, to make sure the disc has time to spin up properly
b: open the box and make sure all the cables are plugged in properly.

Do the cable one first, then the other one. Do them one at a time, not both at once ( like I did ;) ) so that you know which, if any, made a difference.

My discs are Samsung, which I wouldn't recommend to anyone, even though they are cheap.
