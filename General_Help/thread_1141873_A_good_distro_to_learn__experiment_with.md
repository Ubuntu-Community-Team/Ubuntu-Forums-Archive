### Post by mb_webguy on 2009-04-28
Just create a new user account without admin privileges, and start playing with the terminal using that account.  Or boot from the LiveCD.  Either way, you won't be able to do any permanent damage.

Linux is Linux.  The only real difference between distributions is the software bundled with the kernel, and the support provided (in terms of software repositories, user base, etc.).  The only reason Ubuntu may be more "user friendly" than other distributions is because of the choice and configuration of software it comes with.  But underneath it's all the same.  You don't need to switch to some other less "user friendly" distribution just to learn Linux.  Just open up the terminal and start playing around.
