### Post by lha on 2009-03-02
In Nautilus, choose Edit->Preferences and open Media-tab. That's the place in which you can change the behavior to suit your preferences.

### Post by radar2020 on 2009-03-02
If you are using ubuntu 8.04 or later, open up Nautilus file browser EDIT > PREFERENCES > MEDIA TAB.  There you will see a listing for CD,DVD etc.  Work your way down the list choosing how you want your computer to handle the various media.  There is also towards the bottom of the window a choice to never prompt or start programs on media insertion.
