### Post by pedro_orange on 2010-06-08
You probably need to manually edit your /boot/grub/menu.lst 

You will need to add the new operating system to the bottom of the file.

```
title My OS Name
root (hd#,0)
makeactive
chainloader +1 
```

# being the location on the disk.
