### Post by zzigahertz on 2006-03-12
Vaya al siguiente URL:  _[https://shipit.ubuntu.com/](https://shipit.ubuntu.com/) _.  Se necesita entender Ingles (un poco).   De alli mandan unos juegos de discos CD, uno para Live CD Y uno para instalacion integra en cada juego.  Si la nueva distribucion viene cerca, no mandan los CDs hasta que este lista la nueva version.  Es todo gratis, y mandan cinco juegos para re-regalarlas extras.  

Espero que esto le ayude.
