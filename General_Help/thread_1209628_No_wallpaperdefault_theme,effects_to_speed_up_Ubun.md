### Post by lykwydchykyn on 2009-07-10
Wallpaper is not that big of a deal in this day and age.  Granted, putting an image on the screen does take some amount of RAM, but you'd have to have a pretty low-spec machine for it to make a significant difference.

I believe the deal with windows back in the 9x days was that it had to load an extra "active desktop" component to put bitmaps on the desktop, and that took a few extra resources.  I'm a little hazy on that though.

Disabling desktop effects makes a pretty significant difference, and if speed is still an issue investigate a lighter desktop environment.  swoll1980 suggested some good ones.
