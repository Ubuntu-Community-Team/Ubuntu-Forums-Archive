### Post by Toribor on 2008-10-27
Alright, should be an easy question, but I'm having trouble with it:

When I resume my computer from suspend my SD card I keep in my laptop for readyboost on my Vista partition mounts and displays it's file contents. It's not a problem as much as a nuisance but is there a way I can prevent linux from displaying file contents of my SD card each time I resume from suspend?

---

### Post by jerome1232 on 2008-10-27
I'm pretty sure this will do it.

Open Nautilus (Places->Computer)

Hit Edit->Preferences->Media->Uncheck Browse Media When Inserted

It's down at the bottom. Let me know if it fixes the issue.
