### Post by StuartN on 2009-04-18
> **MIH1406 said:**
> I am a C++ programmer (using VisualStudio in MS Windows). I want a similar way to program in Ubuntu linux.

Your biggest decision is probably what GUI you want to program for - the big two are probably GTK+ [http://www.gtk.org/](http://www.gtk.org/) or Qt [http://www.qtsoftware.com/products/](http://www.qtsoftware.com/products/). Anjuta and Eclipse work well with GTK+. Qt has its own SDK environment.

Personally I like Geany with GTK+, which is a really lightweight way to program - Geany will autosave, make and run the target, but you need to write your own makefiles.
