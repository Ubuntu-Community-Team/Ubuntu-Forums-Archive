### Post by kryptikos on 2009-04-02
You would have to make sure that the account set up in Evolution and your Vista mail were told to leave the email message on the server until you direct it to delete it. By default POP3 downloads the email message effectively removing it from the server. The next time you are in your Vista and check the email nothing would be on the server. This POP3 setting goes for the same in Vista. Some POP3 servers allow you to mark an option that says "leave on server" but many do not. It is something you would have to check with your ISP (if that is who you are receiving emails from).

To keep the message on the server till you are actually ready to delete it use IMAP instead of POP3. By default it is set up opposite of POP. It will leave the message on the server until expressly told by you to delete it. Both your Evolution and Vista mail clients would then be able to access the email.
