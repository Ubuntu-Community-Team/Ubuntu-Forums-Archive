### Post by Rocket2DMn on 2008-02-24
The LiveCD does not work for everybody, it's a video card driver deal.  If you still want to reinstall, try the alternate cd which is a text based installer without a live session.  It's easy to use and you can get it at [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download) by clicking the checkbox at the bottom for the alternate cd.

If you want to just reinstall GRUB, follow this guide: [https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows#head-9f28b5a7f41d3659b2ae759665f8bc89ed5b351d](https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows#head-9f28b5a7f41d3659b2ae759665f8bc89ed5b351d)
