### Post by Lunx on 2009-03-29
> **sabith said:**
> What is a good program for desktop effects or 
munipulating the desktop


I suggest getting on terms with Compiz, has most of the effects people seem to want (desktop cube, wobbly windows etc.) You should install the compizconfig settings manager

```
sudo apt-get install compizconfig-settings-manager
```The Avant Window Navigator is pretty cool too, I prefer to use it rather than have a panel at the bottom of my screen as I set it to auto-hide and gain a little more screen real-estate.

```
sudo apt-get install avant-window-navigator
```And there are also Screenlets that offer a bit of eye-candy

```
sudo apt-get install screenlets
```I'm sure others will have more preferences they can pass on (Emerald for instance). Plus there is also KDE to consider if you are using Gnome.
