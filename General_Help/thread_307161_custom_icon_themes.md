### Post by CatKiller on 2006-11-26
You might be able to hunt through the default Gnome theme's icons to see which one looks right, and then name your icon the same thing.

Otherwise, there might be indications in the release notes for the latest version of Gnome as to which icons were changed, and what they were named to.

If you do find such a list, I'm sure people would appreciate it if you posted it on this forum somewhere.

Thanks for that CatKiller. I've looked around and it seems that there is an effort to have standard names for icons in Gnome and KDE. Tango seems to be leading the charge for this and I found a sample set of icons with all the names of them here:
[http://tango.freedesktop.org/Tango_Icon_Gallery]("http://tango.freedesktop.org/Tango_Icon_Gallery")
This makes it quite easy to find the relevant icon and certainly covers the most popular.
You can also look in the icon folders of Human and Tango (found in root -> usr -> share -> icons ...) to see what they are using.
To answer my original question: the volume control has now been moved into the 'status' folder and is named 'audio-volume'.
