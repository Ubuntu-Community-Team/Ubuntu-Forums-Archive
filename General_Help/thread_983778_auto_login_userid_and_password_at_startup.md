### Post by Terrycymru on 2008-11-16
Assuming you are using Ubuntu 8.10:

If you want to autologin go to:
System>Administration>Login Window>Security
Click on Enable Automatic Login
Ensure that your username is selected
Save

The next time you reboot you should be automatically logged in. However, for security reasons you will still be required to provide your password when performing administrative functions. (That is a good feature. :) )
