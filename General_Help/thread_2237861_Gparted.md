alexius2; Hi !

Let's not make this so difficult, ok ?
GParted is installed by default in the liveDVD(USB) that you used to install ubuntu in the 1st place -
ntfs-3g is also installed by default -

When following advise from an 'old' source, keep in mind this is an evolving operating system, and it does change rapidly... what was true yesterday, may no longer apply. Take any instructions with that grain of salt.

unetbootin may also have progressed, but the instruction should still be useful .

I would suggest you boot the liveDVD, and from there format and install onto the USB device with unetbootin.
