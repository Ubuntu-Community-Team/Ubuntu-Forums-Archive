### Post by dcstar on 2008-07-16
> **enpakkam said:**
> I have installed both windows xp and ubuntu on my system.

I want to work Linux Command Line from windows xp environment.

Any tool or s/w available to do this?

You can only get a "Linux Command Line" while Linux is running, and you can only have one system or another running with dual-boot.

If you want multiple systems running simultaneously, install something like VMWare server and run the other OS in a virtual machine.

Then you can telnet from the Windows machine to the Linux one.
