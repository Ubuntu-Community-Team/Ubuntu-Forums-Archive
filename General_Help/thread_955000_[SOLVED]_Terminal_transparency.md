### Post by ddarsow on 2008-10-21
Open a terminal & then:

Edit >  Current Profile > Effects Tab

then select transparent background & move the slider for the desired effect!

### Post by gusst250 on 2008-10-21
Nice. to make 8.10 more darkroom than it already is I painted the folder backgrounds #4C4039 (the darkroom color). It's very comfortable for the eye now.

This is done by Edit -> Backgrounds and Emblems -> Colors. Make a new color: 4C4039 and drag it into the folder background
