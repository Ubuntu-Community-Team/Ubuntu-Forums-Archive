### Post by ironclaw on 2010-02-13
You can easily edit the menu and delete the blueMSX entry with 'alacarte'. It's at System -> Preferences -> Main Menu.

### Post by sdowney717 on 2010-02-13
If you look at your home/username/.local/share/applications/wine/Programs

(.local is hidden folder), I think you see your wine entries.

I think if you edit using the menu editor, it will leave the folders still in place
