### Post by Smart Viking on 2010-06-28
Hi, i suppose you are talking about the web browser iron?

If so, the only thing you have to do is to go to documents, then the directory "iron-linux-32" and then double click the executable file "iron" :)

This will then be the command to launch iron(if you need it to make shortcuts and so on):

```
~/Documents/iron-linux-32/iron
```But the command wont work if you move the program another place. I hope this helped. :)
