Hi there, so it seems that I solved this problem myself by simply selecting a different ubuntu theme.

Use the unity search to search for "Appearance" and then open it. At the bottom right corner you can select a different theme.

I assume by selecting a different theme you simply are refreshing the icon cache or something like that...I don't really understand the solution, but it works!
