### Post by Bucky Ball on 2015-06-18
Yes, please put your terminal output between code tags. See the last link in my signature. :)

Also, 13.10 is dead. It is no longer supported by Canonical and you will get little to no help with it here. It has reached end-of-life so you will get no response from the 13.10 repositories, which explains your current issue. 

Back-up important data and install a supported release. I'd go for 12.04 LTS or 14.04 LTS, both long-term support releases supported til 2017 and 2019 respectively, or the current interim release 15.04 (supported for about another seven months). 14.10 is current but has almost reached end-of-life so hardly worth it.

Good luck and don't hesitate to ask if you need help installing a supported release. See [here]("http://ubuntuforums.org/showthread.php?t=2229730") for staff recommendations.
