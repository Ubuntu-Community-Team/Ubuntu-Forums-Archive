### Post by CatKiller on 2009-05-11
It's generally a different focus. There are a handful of official variations; Ubuntu uses Gnome, Kubuntu uses KDE, Xubuntu uses XFCE (these are just user preference), Netbook Remix is for netbooks, Edubuntu has an education focus and so comes with educational software installed by default. As soon as you install any piece of software you've customised the distribution you're using; it's perfectly possible to install Edubuntu's software in Kubuntu, for example, and use Thunar as your file manager, and so on. You are then running a customised distribution yourself.

Then there are combinations that the community have put together because they scratch their particular itch. Medibuntu and Mint are both variations that have different software installed by default. Ubuntu Studio comes with audio applications installed by default, and a slightly experimental realtime kernel.

Some or all of these projects will come with different artwork and themes to distinguish themselves from the others.

Any GNU/Linux distribution consists of the Linux kernel and a whole bunch of other applications. It's the choice of these other applications that makes a given distribution different from the others.

EDIT:

> **medicalystoned said:**
> Can one variant do more than another for simple home use of a machine, i.e., e-mail, myspace occasional movie watching etc.? would xubuntu make my machine faster?

Some parts of the software stack take more resources than others. OpenOffice.org takes more memory than Abiword, for example, but arguably does more. Gnome runs more things than XFCE, but if those things are ones that you want running, that's a good thing. If not, you might prefer the stripped-down nature of XFCE. Or you might prefer the user-friendliness of Gnome's environment. There are generally a number of alternatives for any function that you might need from your computer. Because of the customisability of the whole ecosystem, you can choose the alternatives that meet your desired place on the performance/features spectrum. Or you could just go for an already-prepared distribution whose focus aligns well with yours.
