### Post by benerivo on 2007-12-31
There should be a tick box if you right click > 'open with...' > 'Other' > select ark and tick the box at the bottom. If not, you can open the kde control centre, and go to KDE Components > File Associations and type rar in the top search box. There should be at least one entry for rar files and you can manually add applications to be associated with rar files.
