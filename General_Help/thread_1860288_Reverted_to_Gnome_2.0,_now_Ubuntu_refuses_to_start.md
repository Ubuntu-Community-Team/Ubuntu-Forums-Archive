### Post by garvinrick4 on 2011-10-14
> Was afraid of that. What about old files and settings?
Put in a Live CD (install CD using Try Ubuntu) and your install will be there, just
copy /home folder onto some backup medium and done.
Your settings will be gone: 11.10 uses Unity and Gnome3 (gnome-shell) now.
There is a fallback Gnome Classic but not as a completed package as you are used to
in 11.10
