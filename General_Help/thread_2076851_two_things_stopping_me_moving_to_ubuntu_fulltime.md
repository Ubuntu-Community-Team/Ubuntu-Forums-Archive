### Post by zombifier25 on 2012-10-27
[Wine](http://www.winehq.org/) is an applications that allows you to run Windows applications on Linux. Some run well, some need special modifications, and some don't run at all. Just install the wine package in Ubuntu.

About Chessbase in particular, I looked it up in Wine's App Database and there are conflicting results as to whether Chessbase will run, so I won't count on it.
