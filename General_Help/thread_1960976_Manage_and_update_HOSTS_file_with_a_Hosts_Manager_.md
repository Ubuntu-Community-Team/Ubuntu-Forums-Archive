### Post by Sadi58 on 2012-04-18
After years of testing, I've begun using Linux in general and Ubuntu in particular fully since the beginning of 2011.

One of the few things I missed from Windows was the HostsMan: [www.abelhadigital.com/hostsman](www.abelhadigital.com/hostsman)

In time I've managed to find ways of creating a similar hosts file, but I think it shouldn't be too difficult for advanced users and developers in the Linux community to put together a utility similar to HostsMan.

In the end I've decided to make a start and put together some scripts and a couple of zenity and notify-send commands, hoping that someone will take it forward and provide a more complete GUI with perhaps better scripting.

I've tested this successfully in Ubuntu 11.10 and also in Linux Mint (LMDE) 11, which surprisingly displays zenity windows even better, whereas I have sometimes one or two zenity windows bigger than my screen in Ubuntu although all I need is to hit Enter and move on to the next step.

In case of any problem with the attachment, you can also download it from here:
[http://ubuntuone.com/0kgrkx27FYqd3a3umpMtF9](http://ubuntuone.com/0kgrkx27FYqd3a3umpMtF9)

Please extract the folder to your home directory, and read the file named "hosts-manager-readme" first.

Desperately looking for a volunteer better equipped for the task ;-)
