### Post by Bucky Ball on 2014-12-23
11.10 has not been supported for over two years so your chances of support are minimal. I suggest upgrading to a supported release (14.04 LTS is supported until April 2019). Continuing with 11.10 will probably see you encountering further issues and support dwindling further as time goes on. Also, you haven't had any updates for two years, including security updates, which can create vulnerabilities if you are online. Your software apps are very outdated which can also create vulnerabilities.

Good luck. ;)

PS: Tried taking the battery out and just using the power cable? Will it boot okay from Live media (disk or USB)?

### Post by Bucky Ball on 2014-12-23
Batteries often just die, especially if they are kept charged at 100% or close all the time and are then run completely flat. Good idea to flatten them out every few months and completely recharge them. 

What I mentioned about the unsupported release still stands though. Good idea to think about upgrading. If you have room on the drive, perhaps install 14.04 LTS to another partition and see how it goes, mark this thread as 'solved' to help others, and start a new thread regarding any issues with an upgrade. [This link]("EOL release recommendations: http://ubuntuforums.org/showthread.php?t=2229730") might get you started. You could do a net upgrade from 11.10 to 12.04 LTS at one stage (12.04 is supported until 2017) but unsure if that is still possible. [This link]("https://help.ubuntu.com/community/EOLUpgrades") might help with that.  

Good luck, either way. ;)
