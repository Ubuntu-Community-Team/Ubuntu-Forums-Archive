### Post by risegeek on 2011-05-02
Do you want a means of remote server administration or website content moderation? for administration, like the poster above me I would highly recommend the use of webmin. If you're looking for software to allow non-technically savy users to edit / post content & modify the look of a website then I can suggest several options below;

Joomla CMS ([http://www.joomla.org/](http://www.joomla.org/))
Wordpress ([http://wordpress.org/](http://wordpress.org/))
Drupal CMS ([http://drupal.org/](http://drupal.org/))

Good luck to you :D
