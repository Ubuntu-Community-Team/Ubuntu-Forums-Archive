### Post by coffeecat on 2013-11-12
As this is really for technical support, I've moved this to General Help.

I think TheFu was referring to photorec which is part of the testdisk app. Here's a useful guide. You can run photorec off your permanent installation instead of a live CD if you wish to recover files from a flash drive:

[http://www.howtogeek.com/howto/15761/recover-data-like-a-forensics-expert-using-an-ubuntu-live-cd/](http://www.howtogeek.com/howto/15761/recover-data-like-a-forensics-expert-using-an-ubuntu-live-cd/)

As you can see from the screenshots, the guide is getting a bit old now, but the instructions for photorec should still be valid.

An alternative guide:

[http://www.cgsecurity.org/wiki/PhotoRec](http://www.cgsecurity.org/wiki/PhotoRec)
