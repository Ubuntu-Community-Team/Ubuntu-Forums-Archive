### Post by deadflowr on 2023-04-26
Uninstall it as Ubuntu already comes with dash-to-dock.
It's simply renamed as ubuntu-dock in Ubuntu but it's really dash-to-dock.

If you need to make some customizations outside of the little that can be done in Settings > Ubuntu Desktop,
you can install dconf-editor and make changes in org > gnome > shell > extensions > dash-to-dock.
