### Post by skyjacker on 2010-10-10
1) Download the .iso file to a computer and put it on a usb drive with "unetbootin". You can download it here.... [http://sourceforge.net/projects/unetbootin/files/](http://sourceforge.net/projects/unetbootin/files/) 
2) Install unetbootin, and choose the usb drive and iso file to install on the usb.  
3) Set up your notebook to boot from a usb drive FIRST, and then insert the usb to your netbook,restart, and and let ubuntu load. 
4) Choose the "install option icon on the desktop, and install from usb drive (you have to choose the correct drive #).
Follow the instructions , and you should be able to install to the netbook .
