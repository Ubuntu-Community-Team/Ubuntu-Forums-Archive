### Post by fluffman86 on 2009-10-23
So when you use a program to burn the mp4 file, you are actually burning it as a DVD, right? If you put the burned DVD into your computer's DVDROM drive, can you find the .mp4 file on it?  If so, you're doing it wrong.

Also, since you have a DivX player, you can use handbrake to convert the mp4 file to a DivX AVI.  Then you can just burn that file as a normal file to a CD (if it will fit) and play it on your TV.
