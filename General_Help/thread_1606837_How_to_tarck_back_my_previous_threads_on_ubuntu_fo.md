### Post by TiBaal89 on 2010-10-27
You can have the forum automatically subscribe you to any thread you post in. Go to User CP -> Edit Options -> Default Thread Subscription Mode... I set it to "No email notification" which means it will auto-subscribe you, but not send you emails.  This is really great - any thread you participate in will show up in your User CP when it has new posts.
