### Post by warfacegod on 2009-12-28
Hp printers generally work quite well with ubuntu. The only HP computer I've put ubuntu on (my mothers') has no problems running it. Check and see if your swap partition is big enough. Gparted can show it to you and resize it if need be(be careful). I seem to recall that the swap file should be at least as big as Your RAM. 2GB RAM = 2GB swap or better.
