Hi,

before anyone else gets this insane twitching eye thingy like me, here is one thing I found to be causing Kde 4 (in my case karmic) to be REALLY slow when moving files to the trash. (Single file ~ 20-90 seconds):

* Rightclick the trashcan icon (on desktop or in panel)
* Go to the trashcan options (might be called  "wastebin options"?)
* **UN**check "Limit to maximum size".
* <<OK>>

That did it for me. Moving files to trash is instant again. :)
