### Post by uRock on 2010-08-17
I have the same problem all the time on my netbook. I wish I knew of a way to set the delay time between letters typed and when the I/O of the touchpad starts working. I guess it could be considered a bug being that the only way I have seen to disable the keyboard is temporary and only lasts a few minutes before it is reactivated.

The following command is what I use to deactivate the touchpad;
```
synclient TouchpadOff=1
```
