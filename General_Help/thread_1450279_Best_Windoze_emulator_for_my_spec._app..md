### Post by hemimaniac on 2010-04-08
I have been down this road many times as I have been turning wrenches since before the inception of OBD. The trouble is the less expensive Code readers are limited in apps that can read them, my suggestion (if you do this enough to warrant the expense) invest in a Star type scanner or aim a little higher and grab an OTC Genisis Scanner, these actually have the code and diagnostic info. on board and only an internet connection is needed for an update. 

Now that being said, I did get an earlier "simple code reader" to work with its uplink site with wine-doors and winetricks (once you figure out all the extra .dlls and such that need to go in) then install via the wine-doors GUI
