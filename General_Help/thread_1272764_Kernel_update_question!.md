### Post by snowpine on 2009-09-23
All Ubuntu releases are "frozen" meaning you never get major new functionality (a new kernel, new version of Openoffice, latest Firefox, etc) through the normal update process. Major new features are "lumped together" every six months to create a new Ubuntu release. This is what makes Ubuntu stable and distinguishes it from a "rolling release" distro like Arch or Sidux.

You can install any kernel you want in any version of Ubuntu. But you have to do it yourself; it is not officially supported. I encourage you to stick with the official Jaunty kernel unless you have a very specific reason to upgrade. Karmic is right around the corner. :)
