### Post by cortman on 2012-04-25
Either Xubuntu 12.04 or Lubuntu 12.04. I'd lean toward Lubuntu.
If you're feeling adventurous, you can try Bodhi 1.4, which is based on Ubuntu and uses Ubuntu repositories. It uses the Enlightenment DE, which is a bit different but very, very configurable and good looking.
If you want to try a different OS altogether, Crunchbang will probably run like a champ on your netbook. It's based on Debian, so it has a number of similarities to Ubuntu.

### Post by xyzzyman on 2012-04-25
You can't run 64-bit on the N270 cpu you have. That Atom processor is 32-bit only. You can easily upgrade that model to 2GB though, which even with lighter weight distro's still might not be bad if you want to multitask easier and also are like alot of us who like to open lots of tabs in Chrome or Firefox.

Unlike some early model Aspire netbooks, yours has a panel with either one or two screws on the bottom just for the RAM. It's a single slot and you just have to replace the DDR2 1GB stick with a DDR2 2GB stick.
