### Post by scottbporter on 2008-03-14
Just wanted to say how much I enjoy using Ubuntu. About a year ago I started toying with the idea of using Linux. Especially, as I am sick of windows and the advent of Vista. Over $300 for an operating system! Especially when they sell it for $3 in China like drug dealers giving product to future junkies. Every year since windows inception and V3.1 and since I started on a 386 I kept hoping that windows would get more efficient. With every new release I kept hoping it would run faster. Instead, as the hardware got better, windows grew more massive, hogging more and more resources, outpacing the development of hardware. So, just as the cost of harware and memory got cheaper, windows was using more, effectively making using it more expensive, not cheaper. And for what? DMR? A registry that sucks. I used to be able to install aprogram on windows and move the directory anywhere and the program would work. Now I have startup programs that are hidden in the registry and need special programs like startup buddy to find them. An operating system should not consume extensive resources and run in the background allowing you to run your programs at maximum efficiency. OK, I digress. 

Thus, it was Linux or Apple. Apple was to expensive for me. Plus, why trade one expensive OS for another? Especially when the number and types of Linux distros offered seeming flexability. Plus, I could try Linux for free! on the hardware I already had.

So, I jumped in starting with openSUSE. I spent a lot of time working with it. But, YAST was slow to load. Finding programs was hard, and I ran into dependency hell! everytime I tried to install a package.

I then tried simplyMEPIS. It loaded great. recognized my NVIDIA card from the start and was easy to install. I ran it for awhile. I next tried Freespire secondary to having proprietary drivers installed and their click n run installation of programs and their website to find and install programs. But, they were just not up to speed yet and their click n run and website weren't quite ready. Plus, I had a lot of display problems. I kept hearing about Ubuntu and PClinusOS and how each had their own repositories of programs. I decided to try Ubuntu based on their regular update schedule which seemed to promise constant work on the product. Plus, it came in GNOME which I prefer and PCLinus didn't at the time. Further, I wanted to try a Debian based package. (I'm a Purdue grad, Debian being developed by a Purdue student) and also I just hadn't done so yet.

I started with feisty and must admit, the transition to Gustsy has been a little disappointing (problems with 3D graphics, they won't enable). But, I figure these will eventually get fixed, probably with Hardy Heron.

I think the thing that keeps me with Ubuntu is the community. This forum is great! Everyone is helpful, nice, and like a community itself. It's better than telephone tech support. I'm having problems installing my NVIDIA card (and simplyMEPIS would probably install it right away). But, I just like the Ubuntu community.

I want to thank all the people who work for free to make such a great product. I don't think everyone appreciates that this is a free product developed by many, many persons who have dedicated untold hours to making it what it is. Linux isn't for everyone yet. I like tinkering around with my computer, and love the idea of a non-windows system! For those who just want to boot up and everything works, I say, OK spend the $ and get Apple for now. But, for thise who want independence, variety, to be able to set up their system to their own tastes and needs, then Linux is for you. Innovation comes from the constant input of the users, andhere the users can change the system if they want. Just look at firefox and how it just blows windows explorer away.  

Linux Rules.

By the way, Cafepress.com has nice Ubuntu and Linux stickers (which I proudly display on my vehicle.)
