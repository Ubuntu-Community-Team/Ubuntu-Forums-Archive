### Post by T700 on 2006-08-01
Open a terminal window and type:

```
sudo aptitude update
``` 
Press Enter

```
sudo aptitude install mozilla-thunderbird
``` 
Press Enter

-OR-

Open a terminal window and type:

```
gksudo adept
```

Press Enter and when it prompts you for your password, enter it.  Then when the application opens, search for Thunderbird and select it to be installed.  Tell Adept to apply changes.

Also, check out the link in my sig for details on how to install things.  Welcome to the forums!
