### Post by promet on 2007-01-17
Hello Mr. Wooster,

As far as an actual tutorial, I'm not sure. I don't know if you've tried the "man" (manual) page yet though, which is very helpful. 

On the command-line, within a terminal window, type:

$ man apt-get

(The "$" symbol represents the terminal line and is not part of the command ;) )

This should print to the terminal screen more than you, probably, ever wanted to know about apt-get. Hope this is of some help.

p.s. Most common Linux apps have a similar manual page you can access via the "man" command.
