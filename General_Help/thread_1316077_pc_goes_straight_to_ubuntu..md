### Post by Fc1032 on 2009-11-05
From there you just have to find startupmanager in the menu. 

I'm on windows ATM so im not too sure which section, but i think its under the admin section.

Find it then run it.

When the window pops up, the first option should be what your default OS is. It should be ubuntu by default, click it and find the OS you want to boot first.

Then further down you have more options such as how long you want grub to stay up before loading your OS.

hope that helped!
