### Post by hi970283 on 2005-03-01
I don't know where it fits in, but I noticed that many people experience "GRUB error 21" when booting the live CD. After some googling I found a workaround:

- Download Smart Boot Manager 3.6 (use google to find a download link as I don't have it handy). Download the ISO CD-image. The alternative is a floppy image you can write to a disk with dd under linux or rawrite under Windows.
- Burn the ISO image to a disk with a tool of your choice.
- Boot from the Smart Boot Manager (SBM) disk you just created
- When the menu appears, remove the SBM disk and replace it with the live CD.
- Then select the "CD" option in the menu and hit enter. The live CD should now boot.

This workaround should also work for any other bootable CD.

Hopefully this helps someone out.
