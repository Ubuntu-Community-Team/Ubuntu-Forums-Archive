### Post by ward83 on 2007-10-04
While configuring my xorg.conf file to support dual monitors, I accidentally created 2 instances of X, one on each screen. I had mouse and keyboard control over one, but no control over the other. It was just a typo on the xorg.conf file and I got it fixed, but it gave me a cool idea. Would it be possible to run 2 monitors, 2 mice, 2 keyboards, and configure xorg to act like 2 computers? For example, mouse 1, keyboard 1 work on monitor 1, and mouse 2, keyboard 2 work on monitor 2? I don't know how useful it would be, but it seems like a cool experiment :P

# if it has not been modified since the last upgrade of the xserver-xorg
# package.

# If you have edited this file but would like it to be automatically updated
# again, run the following command:
#   sudo dpkg-reconfigure -phigh xserver-xorg

	Identifier	"Terry's Custom Layout"

	Screen  1	"Screen1" Below "Screen0"

	InputDevice	"Configured Mouse"  "CorePointer"
