### Post by TheFu on 2014-10-14
Check the host file on the other system.
Also check the hostnames on all systems.
and the DNS result for a reverse lookup.
and the ssh-key name in the ~/.ssh/ directory from the client machine.

Most of the time, on home networks, this stuff is NOT dynamic and names are placed into files.
