### Post by 3rdalbum on 2010-02-06
If you want to return a particular program to its default state, removing it and reinstalling it won't do anything useful because the settings will still be present in your home directory. You need to remove the settings file, which is inside a hidden folder in your home directory (~/.gimp-2.6).
