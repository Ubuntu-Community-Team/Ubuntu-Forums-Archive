### Post by cscott0108 on 2010-02-08
There was a setting I used in Kubuntu 9.04. The setting was if you are using multiple desktops and when you open a window on a desktop it would show up in the taskbar of that desktop only, it would not show up on all of the taskbars on all desktops. I cannot remember the setting to change it, and since I upgraded to 9.10 the setting has been reversed. Any ideas?

I solved this, If you right click the Taskbar, click Task Manager Settings, and check the box's for "Only show tasks from the current [Desktop/Screen], then save it. Now I have Thunderbird loaded on  desktop one and the rest on the other and it does not show up on desktop two's taskbar. :)
