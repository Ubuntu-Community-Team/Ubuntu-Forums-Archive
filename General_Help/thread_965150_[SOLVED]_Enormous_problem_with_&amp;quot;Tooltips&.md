### Post by trapgate on 2008-11-02
I ran into this also. I had customized my gnome appearance settings. To fix it, go to System/Preferences/Appearance, click Customize to edit the current theme, then click colors, then change the tooltip color. In my case, this had gotten set to black on black.

I'm guessing that previous versions didn't have a setting for tooltip colors, so when the customized theme was opened in the latest gnome version, it was given a default value. I haven't verified this though.
