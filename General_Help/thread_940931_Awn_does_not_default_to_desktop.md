### Post by issih on 2008-10-07
Set it to be launched at boot.

To do that go to System->Preferences->Sessions and add a new item to the current session's profile.

copy the command that launches awn from the properties of the launcher on your desktop, and use that to set up your new entry.

awn should now be launched every time you log on.

Hope that helps
