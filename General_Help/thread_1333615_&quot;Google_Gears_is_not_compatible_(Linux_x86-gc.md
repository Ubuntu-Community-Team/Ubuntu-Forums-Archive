### Post by BrowneR on 2009-12-30
You are right, that link is now dead.

Reading through the google code [bug report]("http://code.google.com/p/gears/issues/detail?id=975") I found another link to a fixed up version that seems to work for me.

Try downloading the file from here: [http://drop.io/gears_linux_0_5_33_version_hack/asset/gears-linux-opt-xpi](http://drop.io/gears_linux_0_5_33_version_hack/asset/gears-linux-opt-xpi)

Then just drag it from your file manager into a firefox window and it should be installed over your existing version. (Or alternatively select file then open from the firefox toolbar and select the file you downloaded).

Solved my problems.
