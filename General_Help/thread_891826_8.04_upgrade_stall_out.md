> This is an upgrade, not a fresh install. The system is old and was done with the Alt CD. I'm running the upgrade from the buit-in upgrade manager.

Sorry I wasn't reading carefully :)

Here is my suggestion, burn an alternate install cd of 8.04.  When you get to the partitioner choose manual partitioning and make sure when assign your root partition that you choose **not** to format the drive.  The installer will give you a message that it will replace the system files and leave everything else alone.  I did this several times with Ubuntu before I separated my /home and / partitions.  It should work the same with Xubuntu.
