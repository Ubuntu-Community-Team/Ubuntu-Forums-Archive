### Post by NFblaze on 2009-06-07
Beginning Ubuntu Linux from Apress is a very good book. I'd say thats all you need to effectively get started.

This website has a list of free books to learn about Linux,

[http://www.linuxhaxor.net/2009/05/10/10-free-linux-ebooks-for-beginners](http://www.linuxhaxor.net/2009/05/10/10-free-linux-ebooks-for-beginners)

Apart from that the books I have on my computer that I have actually read or started reading when I was just starting out we're:

Linux Cookbook.pdf
Sams.Ubuntu.Unleashed.2008.Edition.Jul.2008.pdf

Ubuntu Unleashed is a little more technical if your a non-tech saavy ,but it's still good. The Linux Cookbook is also very technical, but it's sorta dry and not all examples in the book apply to Ubuntu. Though it does teach some useful stuff pertaining to the shell (aka command line)
