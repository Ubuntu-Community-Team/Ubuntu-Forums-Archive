### Post by KeLa on 2009-10-30
Hi,

I had very bad problem with my Ubuntu 9.10 and Firefox 3.5.3 here in this forum.
When you put mouse cursor over some thread name of section containing name of the thread you get a popup box that contains text from first post of that thread and
while that box is still showing you scroll down or up the place where that box was changes
to gray and stays there until you scroll so much that it vanish from screen and then scroll back then it's gone.

The fix for this that is simple. In Firefox just go to Edit -> Preferences -> Advanced
and in section Browsing check the "Use smooth scrolling" checkbox.

I don't know how common this problem is but here you have the easy fix.

Thanks.
