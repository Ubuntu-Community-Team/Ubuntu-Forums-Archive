### Post by Johnsie on 2009-11-09
I would try booting in 'SAFE MODE WITH NETWORKING' by pressing "F8" when your computer is starting up.

If you've already deleted explorer.exe then windows windows wont start up. You will need to replace it.

If it starts up that way then immediately apply  Windows XP service pack 3:

[http://update.microsoft.com](http://update.microsoft.com)

That will replace the corrupted explorer.exe with a new, more secure clean one. Most Windows machines only have service pack 2. 

Afterwards, make sure you install an antivirus like Avast that runs in memory so virus's cannot get in after that.  Personally I would look at doing a complete re-install but I know that isn't always an option.

### Post by Mark Phelps on 2009-11-09
While I would ordinarily agree with "call IT" as the first choice (since I AM in "IT"), my concern here is that, at least with all the systems WE support, several things are true:
1) AV software is installed by default and runs automatically
2) Usage is limited to "designated personnel" for "company business"
3) Allowing others to use the machine, especially for "personal business" can lead to "disciplinary actions".

In other words, once IT learns that someone else used the work machine for personal use -- and got it infected in the process -- THAT could be grounds for being fired.

So, would suggest trying to clean the machine on your own first. Google for and download either an Avira Rescue CD or Kaspersky Antivirus CD, burn it to CD, boot from that -- and let it do the cleaning on the machine.

And ... in future ... advise whomever is entrusted with the machine to NOT loan it out to others to use -- unless they want to lose their job in the process.

And hey ... I'm not OPPOSED to personal use of machines, I'm just saying that nowadays, companies will use ANY EXCUSE to get rid of people, and violating a company policy about personal use of company property ... that's as good an excuse as any.
