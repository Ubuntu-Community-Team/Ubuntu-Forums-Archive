### Post by cymbaline42 on 2011-05-01
Uninstall your current installation of Firefox and do a fresh install in your desired language by downloading it from here: ([https://www.mozilla.com/en-US/firefox/all.html](https://www.mozilla.com/en-US/firefox/all.html))

Extract the tar.gz file to wherever you want.  Then open terminal, go to the directory where it is installed and type: **./firefox** to start it.
