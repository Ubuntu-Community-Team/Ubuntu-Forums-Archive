### Post by 3rdalbum on 2009-09-19
> **bridgetothesun said:**
> I had installed 64bit Ubuntu 9.04 a while back and found it to be slow. My graphics card is the Intel 4500MHD GMA card, and it seems it was not recognized and the drivers were not installed.

Intel graphics drivers are open-source, and therefore included with Ubuntu. They are activated by default if you have an Intel graphics chipset.

There are some problems with the current Intel drivers - they are being rewritten and performance is currently not as good as it used to be. This situation will change in the future. But as a result, Compiz doesn't work with it out-of-the-box but from what I gather there are ways of making it work. Do a search online.

Finally, don't let anybody tell you that there is "not as much software on 64-bit" or "compatibility problems". All software in the Ubuntu repositories has either been recompiled for 64-bit or can be run under 64-bit without drama. The only exception is Flash Player, which works better if you use the 64-bit version from Adobe Labs.

The only time you should listen to someone about "not as much software that works on 64-bit" is when they are talking about Windows.
