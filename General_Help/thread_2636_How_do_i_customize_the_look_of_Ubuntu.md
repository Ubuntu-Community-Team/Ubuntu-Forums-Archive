### Post by cacofonix on 2004-10-30
To install icons you need to unzip them and move the folder to the .icons folder hidden in your home directory (show all files will show it) * note:this will only change the icons on programs for everything else you need to apply them yourself*

To install a custom theme:
click on Computer Menu
go to Desktop Preferences
click on Themes
and then click on the install theme button locate the zip file and click ok

To install a different manager just do a search in synaptic Im not how to enable it as default manager though sorry
