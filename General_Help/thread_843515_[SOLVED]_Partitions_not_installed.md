### Post by Victormd on 2008-06-28
When you did
```
cat /etc/fstab 
```
and
```
sudo blkid
```
you got two lists with IDs for your partitions. They have to match. If they don't, then your partition won't mount.
To edit your fstab file, run:
```
sudo gedit /etc/fstab
```
and check to see which entries don't match (in bold) with the output from
```
sudo blkid
```
and modify accordingly
> 
(...)
# Entry for /dev/sda1 :
UUID=**E268ADC068AD9437** /media/sda1 ntfs-3g defaults,locale=en_CA.UTF-8 0 1
# Entry for /dev/sda13 :
UUID=**D0D06953D06940B8** /media/sda13 ntfs-3g defaults,locale=en_CA.UTF-8 0 1
# Entry for /dev/sda5 :
UUID=**F4FCA4D2FCA49102** /media/sda5 ntfs-3g defaults,locale=en_CA.UTF-8 0 1
(...)

Let us know if you need further assistance... :)
