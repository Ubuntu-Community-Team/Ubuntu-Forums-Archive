### Post by cmoraes on 2011-01-19
I assume your XP disks are bootable.  In order to reinstall WinXP, 
1. put your Win XP disks in the CD drive
2. restart your PC, 
3. go into the BIOS settings (the start up screen has instructions on what button to press. Usually it is F12 or F2), 
4. change the boot order to make the CD drive boot before the HDD.  
5. Save the bios settings and exit.
6. The system should reboot and boot directly from the WinXP CD.

When you re-start your computer, there should be a message giving you the boot options...look for it quickly...it should be F9, F12, or Delete key depending on the manufacturers.  Once you press the appropriate button, you can select boot from CD-ROM.  Place XP CD in the tray then hit Enter.  XP installer should take over.  You can then perform a fresh install on your computer.  If XP installer does not run then there is something wrong with your XP disk.  You are out of luck.  Microsoft does not give you a free copy of XP.  You can't even buy a new version anymore.  Can you still log into XP from your computer?  If yes, you might be able to create a backup disk from there.
