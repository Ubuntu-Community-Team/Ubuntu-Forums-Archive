### Post by howefield on 2011-01-01
System > Preferences > Main Menu

Highlight Games on the left, and click the New Item button. The name and Comment fields can be whatever you want, and the Command field should be launch command which you already know. Click on the springboard icon to select another if you want and finally ok your way back out..
