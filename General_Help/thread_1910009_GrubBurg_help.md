Welcome to the Ubuntu Forums.  :-)

The easiest fix would be to boot the LiveCD and install/run the Boot Repair app. If the system isn't too badly corrupted Boot Repair can usually fix things.

If it can't, it provides an option to run the boot info script. This script generates a file called RESULTS.txt. If Boot Repair doesn't work, please post the contents of RESULTS.txt so we can see what is happening.

There is a link to "Boot Repair" in my signature line.
