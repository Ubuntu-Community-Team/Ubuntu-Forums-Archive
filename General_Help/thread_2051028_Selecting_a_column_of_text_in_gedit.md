### Post by vasa1 on 2012-09-01
Seems that one has to install **gedit-plugins** (available from the Ubuntu Software Center). This is a [collection of plug-ins]("https://live.gnome.org/GeditPlugins") and includes **Multi Edit** which allows for column selection among other things.
Marking this solved.
