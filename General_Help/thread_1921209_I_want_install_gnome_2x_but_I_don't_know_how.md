### Post by grahammechanical on 2012-02-06
Do not expect older operating systems to work on newer hardware. Did the i3 even exist when 10.04 was released?

Read the sticky at the top of the General Help section that you are posting in.

[http://ubuntuforums.org/showthread.php?t=1859961]("http://ubuntuforums.org/showthread.php?t=1859961")

But you need to understand that you can only get what looks like an older OS. The Gnome developers have discarded Gnome 2 in favour of Gnome 3 and Ubuntu and other distributions are now using Gnome 3 and it is the way forward.

I doubt very much if it is possible to remove Gnome 3 from 11.10 and replace it with Gnome 2. It would not be 11.10. Broken, that is what it would be.

There are distributions that continue to use Gnome 2. You could try one of them.

Regards.
