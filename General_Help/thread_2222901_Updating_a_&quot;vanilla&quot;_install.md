### Post by grahammechanical on 2014-05-08
Look in the Software Centre for Ubuntu Restricted Extras. Look in System Settings>software and Updates>Additional Drivers tab for the proprietary video driver that would have been installed with third party software. If you need a proprietary video driver.

Regards.

I'm guessing this is what you mean by 3rd party.

As to the Ubuntu-Restricted-extras package, during it's install from the software center a EULA will pop up.
Sometimes, though, the pop up window doesn't pop up correctly and hides behind the software center window.

If at some point during the install it seems to hang, see if the EULA window is hiding behind the main window.
Just in case.
