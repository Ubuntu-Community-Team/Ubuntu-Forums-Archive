### Post by lidex on 2010-05-08
Try this in a terminal="Applications->Accessories->Terminal"
```
sudo apt-get update
sudo apt-get upgrade
sudo update-grub
```
Then this:
```
sudo apt-get install --reinstall gdm && sudo dpkg-reconfigure gdm
```
And reboot.

### Post by lidex on 2010-05-10
Guess I misread your first post. From the grub menu select recovery option - should be second line. It will load and give you some options. If you are presented with fix/reconfigure xserver option, take that. Otherwise get to a command prompt and run this command:
```
sudo mv /etc/X11/xorg.conf xorg.conf.old
```
Reboot
