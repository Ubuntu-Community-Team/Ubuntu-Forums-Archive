### Post by kevcool on 2005-11-23
It will be recognized as an external USB flash drive when you plug it in.  It will mount to a path like /media/usbdrive.  You can then copy mp3 files into the folder named "audio" - either manually or through some app like rythmbox.  Beware copying files will be very slow because the geniuses at Motorola made the connection USB 1.1 rather than 2.0.

I connect my Mot E398 which is the same phone before Mot rebranded it as the E1 ROKR.  Fortunately they stuck to standard USB storage protocols so you shouldn't have a problem with it being recognized and mounted.
