### Post by jimv on 2008-06-04
So I've been having random lockups (no mouse movement, no keyboard, no nothing) with Hardy for the last few weeks, and it SEEMS that one of two things that I did yesterday fixed the issue.  I haven't locked up all day, and I was locking up about once per hour before.

The first thing I did was disable IPv6 because it seemed that the last thing recorded in the logs each time was the acquisition of an IPv6 address.  Which is odd, because I'm not connected to an IPv6 router.

The second thing I did was get the 2.6.24-18 kernel and install it.

Since then, I've been crash/lockup free.  Cross your fingers.
