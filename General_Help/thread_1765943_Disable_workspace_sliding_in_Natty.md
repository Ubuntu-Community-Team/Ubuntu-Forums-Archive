How to disable sliding effect when stitching workspaces? Thank you.

---

### Post by silkworm2.5 on 2011-05-23
Install Compiz Config from the software repository, then open it.
Look for the heading Desktop Wall.
Under that, go into The second tab, and change the wall slide animation time to 0.
