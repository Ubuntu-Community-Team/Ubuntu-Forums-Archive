### Post by slakkie on 2009-11-30
Take any version from 8.04 and up (these are the supported version). 8.04 is currently at 8.04.3 LTS and the rest (8.10-9.10) are the current stable releases. Everything below 8.04 is not supported anymore. Although you can always install them and upgrade to a newer version.

Grab the minimal CD, as you have more control on what to install and what not to install compared to the liveCD version. You'll need a working network though.

[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

Or use the alternate CD: [http://releases.ubuntu.com/hardy/ubuntu-8.04.3-alternate-i386.iso](http://releases.ubuntu.com/hardy/ubuntu-8.04.3-alternate-i386.iso)

For more images goto: [http://releases.ubuntu.com/](http://releases.ubuntu.com/)
For the unsupported releases: [http://old-releases.ubuntu.com](http://old-releases.ubuntu.com)

But not terrible either. From [https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)
> 
Ubuntu should run reasonably well on a computer with the following minimum hardware specification. However, features such as visual effects may not run smoothly.

    * 700 MHz x86 processor
    * 384 MB of system memory (RAM)
    * 8 GB of disk space
    * Graphics card capable of 1024x768 resolution
    * Sound card
    * A network or Internet connection 


384k ram will be your biggest bottleneck but may not be a problem depending on how you use your pc.
