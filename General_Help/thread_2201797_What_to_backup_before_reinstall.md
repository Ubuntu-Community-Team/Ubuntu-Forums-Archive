### Post by DrScum on 2014-01-26
I just did what you have in mind, moving an Ubuntu 12.04 LTS install from one system to another. I installed 12.04 LTS  on the new system then just moved the thunderbird profile and the firefox profile to the new system and thus had all e-mails and all account settings available. In firefox I had all saved bookmarks and all saved passwords available. In addition to that, I also transferred the /etc/cups folder in order to have the printer configurations available.
