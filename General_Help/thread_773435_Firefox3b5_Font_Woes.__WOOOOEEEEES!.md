### Post by yaknowwat on 2008-04-29
You can change the default font in firefox

To change the default font in Firefox 3

Edit > Preferences |> Content > Advanced Font options are there.


Though i suggest doing Ctrl++ on some sites as the font rendering smooths out when you increase the page layout.



Anyways i haven't had much of an issue with bad font rendering in Firefox 3 seems more realistic to me (printed and OS blended wise).

though my resolution is 1280x1024 and i have to zoom just about everything but, it all comes out nice looking.
