### Post by crazyness003 on 2008-11-20
yes it it possible, but not recommended since you can only use one at a time. 

To install KDE (basically all that makes kubuntu), do this:

```
sudo apt-get install kubuntu-desktop
```When i did this, it still defaulted to the GDM and in the one of the selectable options in GDM, you can choose which DM to log into.

Or just search for kubuntu in synaptic, you'll get a better, more comfortable way to install there.

But, if i were you, just pick one and stick with it. Or keep a lookout for [TWiMU]("http://ubuntuforums.org/showthread.php?t=866390").
