### Post by howefield on 2021-12-17
> **daniell59 said:**
> Thanks.

I tried to email an attached MP4 file to somebody. His server rejected it because it was too large. I was wondering whether it can be compressed even more.

Email file attachments are often limited in size to about 10Mb, some like gmail allow a bit more at 25Mb, email is not a good medium for sending video.

How big is the file that you are trying to send ?

You are likely able to reduce the file size by re-encoding with a probable loss in quality. Or you could use a file sharing service to upload the file and share a download link to it.

### Post by KBar on 2021-12-17
I agree with howefield. A file sharing service is the best option. Google Drive is the most obvious choice, though if you are not a fan of Google or simply do not have a Google account, there should be a bunch of other ones. You can also upload videos to YouTube and set them to be private.

I'd suggest against compressing an already compressed file further.
