### Post by shlim89 on 2009-02-19
This is how I did it Vista:

Downloaded and executed it. I have found a batch file called "run_portable_ubuntu.bat" in the installed folder.

(I've learned that I needed to run it as administrator rather than as my account even if my account type is administrator. One way to run it as administrator is as follows.)

Right-clicked the "Command Prompt" icon in the start menu and selected "Run as administrator." I got an "Administrator: Command Prompt" window on the screen. I ran the batch file and got the Portable Ubuntu up and running. Hope it helps. Thanks.
