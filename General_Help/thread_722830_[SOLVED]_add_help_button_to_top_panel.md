### Post by PoopSlayer on 2008-03-12
Make a file-launcher and have the command be:
firefox file:///usr/share/... (whatever your help file location is)

If you don't know that location, then it's probably in your firefox's "restoredefault" files

EDIT: here it is. I don't know if it's the same in Ubuntu as in Xubuntu:
file:///usr/share/ubuntu-artwork/home/locales/index-C.html
