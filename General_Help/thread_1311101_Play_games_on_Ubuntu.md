### Post by OMJD on 2009-11-02
> **abhishekdagr8 said:**
> I have downloaded Wine. But my windows games dont seem to work on it. 
Can i run a windows cd game through wine?...

Unfortunately Wine is not always the answer to all your problems with regards to getting Windows applications or games running on a Linux system. Often, one will be required to make many tweaks or configuration changes in order to get the chosen application or game to work properly. In some cases, the application or game may not even work at all.

As you may have guessed, compatibility depends very much on the application or game in question. I suggest checking Wine's "App database" to check to see how "compatible" a certain application or game is. "Compatible" is perhaps not the best term to use in this case, but I think it gets the point across nonetheless. 

You can check this database by clicking the link below:
[http://appdb.winehq.org/](http://appdb.winehq.org/)

If you would like further assistance in this thread, I would suggest stating which game(s) you're trying to install or run. Also, if you could state the exact problem along with any error messages, that would also help people assist you. Remember that it's difficult and often impossible for people to troubleshoot a problem if they have nothing to start with.
