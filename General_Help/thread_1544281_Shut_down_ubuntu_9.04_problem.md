### Post by stlsaint on 2010-08-02
use this command: 
```
 sudo shutdown -h <time> 
```

so for instance if you wanted your computer to shutdown in 5 minutes you would use:

```
sudo shutdown -h 5
```

If you are root on your server than there is no need to add in sudo.
