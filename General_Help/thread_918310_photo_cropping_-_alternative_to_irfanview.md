You can crop using gthumb, just open the picture, and goto Image -> Crop or hit ALT+C.  You then select the area that you want to crop.  Not as easy as irfanview, but it works for a small set of pictures

### Post by Dyonas on 2008-11-13
I feel your pain when it comes to IrfanView, it's been my image cropping / adjusting / resampling program for almost 10 years now. The only thing that's almost as quick that I've used personally is a KDE program, KSnapshot.  You can specify the size and / or area of the screen before it takes the screenshot although again it isn't quite the same as IrfanView.

### Post by alexthunder on 2010-05-07
I've used Faststone Image Viewer for cropping pictures in Windows. It was the best from what I have tried. It has the following options:

[LIST]
[*]Show hide the net for the rule of thirds
[*]Darken the area that is outside the selection
[*]Lossless cropping
[*]Printing size selection
[*]Switch between portrait and landscape mode
[*]Shortcut for opening the cropping tool
[/LIST]

Is there any program like that for Linux? I've tried many image viewers including **Gwenview**, F-Spot, Shotwell, but none of them has the features mentioned above. Gwenview is OK, but not as good as faststone image viewer. 

I've also tried to use Faststone Image Viewer under wine, but it is not working well.

p.s. the developers of **shotwell** and **gthumb** are implementing these features... so stay tuned :)
