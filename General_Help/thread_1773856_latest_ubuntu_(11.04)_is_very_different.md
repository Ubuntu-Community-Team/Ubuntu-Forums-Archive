### Post by vranjan on 2011-06-02
Hello,

I see a few annoying things in Ubuntu 11.04:

(i) The panel where all the application shortcut launcher can be glued is on the left side and cannot be moved to the top, bottom, right. 

(ii) There is no panel on the bottom of window. All the open windows of an application stays in the launcher. Now matter how you do it, all the open windows launches simultaneously if you click on the launcher. Earlier, if I had 10 windows of chrome opened, there was a shortcut for all of them in the bottom panel and I could chose which of the ten I want to use. No more. 

(iii) If I type $HOME on console and then hit tab, it becomes "/$HOME/" and then hitting tab it complaints that there is no directory "$/HOME". Earlier if you typed "$HOME" and hit tab it would become "/home/uid/" and then hitting tab gave options for file names. No more. This seems the most serious digression from long established Unix protocol. 

(iv) No shortcut can be put on the top panel. 

(v) Chrome keeps crashing because a process associated with Chrome takes a lot of memory. Killing the process makes all open chrome window blind and refreshing each one of them brings back the page.

In conclusion, the latest changes do not add any new fundamental feature, rather it is confusing and annoying.

Thanks,

Vivek:confused::(
