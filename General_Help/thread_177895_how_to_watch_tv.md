### Post by Cal Little on 2006-05-18
Check out MythTV. It is considered the best DVR program available in the Linux community. If you just want to watch TV and don't need to record, then the tvtime app in Ubuntu is more than up to the task. You will find it in the synaptic package manager. If you want to install the MythTV, you will need to make sure the repositories are available in the synaptic manager. I also recommend you look up a complete How-To on the process since it requires setting up a database backend to handle the large data files you will generate. It is a killer app but not an easy install.
