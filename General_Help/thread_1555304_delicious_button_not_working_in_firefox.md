### Post by lovinglinux on 2010-08-18
Re-installing Firefox won't help. The problem lies on the latest version of Delicious extension, which affects keyboard shortcuts too. The solution is to disable Delicious or install and old version from [https://addons.mozilla.org/en-US/firefox/addon/3615/versions/](https://addons.mozilla.org/en-US/firefox/addon/3615/versions/)

---

### Post by cjaramilu on 2010-08-18
I installed version 2.1.072 and it started working again. Thanks.
