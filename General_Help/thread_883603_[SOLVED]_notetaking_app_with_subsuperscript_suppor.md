### Post by mikjp on 2008-08-08
I suppose it can be difficult to find a combination of sub/superscripts and a lightweight solution. If you want WYSIWYG, you probably should take a  lightweight word processor (Abiword).

Or maybe some lightweight markup language + editor? You could then process the file to produce a printable PDF, LaTeX source or HTML file to be viewed in a WWW browser.

See e.g. [http://redcloth.org/textile](http://redcloth.org/textile)
