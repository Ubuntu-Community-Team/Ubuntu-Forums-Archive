### Post by theozzlives on 2009-02-10
If you used programs and properties it should have uninstalled completely. If it didn't and Vista is as close to Windows 7 as I think, go to start > run, type "msconfig", click on the boot tab, highlight Ubuntu and select delete. Sorry to see you go back to Microsnot.

### Post by Darkmage09 on 2009-02-10
I had the same problem when I uninstalled Ubuntu because my HDD was almost empty. Now I have more room and now using Ubuntu. But to fix this go to Computer>Ubuntu-backup>delete>Reboot. That should work.
