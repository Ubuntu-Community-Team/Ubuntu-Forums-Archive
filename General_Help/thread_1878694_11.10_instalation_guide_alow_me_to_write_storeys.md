### Post by grahammechanical on 2011-11-11
When you created that Live USB memory stick you used what is called an alternate ISO for the download. It uses a text based installer and not a Graphical User Interface installer. The purpose of this alternate CD ISO download is to make it possible to install Ubuntu on a machine that does not have enough memory (RAM) for a Graphical User Interface install.

I think that you have other problems as well.

I am also thinking that you might have downloaded a Server version of Ubuntu. The Server version does not have a Graphical User Interface Desktop at all.
