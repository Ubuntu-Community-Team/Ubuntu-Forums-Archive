### Post by nothingspecial on 2016-04-23
You need to do

```
cd ~/Downloads/Wordbiz
```

then

```
java -jar wordbiz.jar
```

or 
```
java -jar ~/Downloads/Wordbiz/wordbiz.jar
```

and make sure you have java installed
