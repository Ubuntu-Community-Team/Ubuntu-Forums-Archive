### Post by codemaniac on 2012-03-25
Hello Wrakage ,

Directories are containers of the files and other objects you create in any OS .In Unix like systems if you want to create a directory just open a terminal and type .```
mkdir *your_directory_name*
```.
This would create a new directory under your present working directory.
Now if you want to switch into the directory just use the cd command .
```
cd *your_directory_name*
``` .

If you want to go back to the parent directory or the old working directory (in this case) .use ```
cd ..
```
