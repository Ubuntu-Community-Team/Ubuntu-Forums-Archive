### Post by WorMzy on 2011-03-15
Not sure what you mean by the "usr1" part, but running emerald from the run dialog will only last for your current session. You need to enable the Window Decoration plugin in Compiz (use ccsm) and set it to use Emerald.

Alternatively, install the fusion-icon package, log out and back in (or manually start the fusion-icon application), then use it's panel icon to control your Window decoration.

Oh, and you can control which theme Emerald uses by running emerald-theme-manager.
