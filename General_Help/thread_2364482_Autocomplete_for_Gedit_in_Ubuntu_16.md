### Post by deadflowr on 2017-06-23
gedit plugins go in
```
~/.local/share/gedit/plugins
```
you may need to create that folder.

Also...
The plugin may not work with current versions of gedit.
I see the last autocomplete plugins were built for the older gnome2 versions of gedit.
We now use gnome3 versions.
