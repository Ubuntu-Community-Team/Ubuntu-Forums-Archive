### Post by prshah on 2008-11-24
> **rduijkeren said:**
> 
I've a problem with updating 8.10. When I start update manager I see dowloading 52 files, after that it says reading package information 

The "52" files being downloaded contain information about the latest versions available in the repositories. 

After downloading the latest information, it checks the packages on your system.

In your (this) case, all packages on your system are the same version as those on the repository, and hence you should see a message (if you're using the GUI) "Your system is up to date."

So you have nothing to update.
