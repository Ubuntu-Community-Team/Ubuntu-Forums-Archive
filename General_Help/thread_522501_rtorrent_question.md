### Post by yabbadabbadont on 2007-08-10
Use the arrow keys to navigate to the individual file display.  You can then use the up and down arrows to select individual files.  Use the space bar to change the priority of the files.  One of the available priorities is "off".

### Post by bodhi.zazen on 2007-08-10
From the command line :

rtorrent <torrent_file>

select your torrent 

Ctrl-S to start the download

Crtl D to stop a download, second time to remove it from the list

ctrl -R to restart a download 

ctrl 

See here for further info :
