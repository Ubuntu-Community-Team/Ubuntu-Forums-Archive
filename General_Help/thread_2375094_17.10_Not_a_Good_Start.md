### Post by mc4man on 2017-10-21
Wow, what a joke.
It's not that that gnome-system-monitor is corrupt, it needs another 'core' type snap to work .
(- and then it's no better than the one already installed by default.

If you must - 
```
sudo snap install gnome-3-26-1604
```
when that finishes
```
sudo snap connect gnome-system-monitor:gnome-3-26-1604 gnome-3-26-1604
```

Then you can open it - 
/snap/bin/gnome-system-monitor

Or just do this & use the one already installed in 17.10
```
sudo snap remove gnome-system-monitor
```
