### Post by conscious on 2008-04-09
Under System -> Preferences -> Keyboard -> Layout options -> Group Shift/Lock behavior, you are presented a multitude of choices. To have Alt-Shift switch the layout, select "Alt+Shift changes group."

In addition, you can set up a keyboard indicator on the top panel, if you don't have one. Right-click on the top panel, then go Add to panel... -> Utilities -> Keyboard indicator. (And it's full of other useful stuff, by the way.)
