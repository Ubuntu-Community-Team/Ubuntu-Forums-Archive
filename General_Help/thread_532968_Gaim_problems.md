Found this in another thread, it seems to fix the gaim problem for now anyway. The thread I pulled this out of implies this is a problem with MSN accounts. In gaim, do the following:
Accounts ->  select your MSN account -> Modify -> Advanced -> deselect "Use HTTP method"
