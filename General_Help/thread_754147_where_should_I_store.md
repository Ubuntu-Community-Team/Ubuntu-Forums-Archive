### Post by TeoBigusGeekus on 2008-04-13
There is no default folder for this stuff, but it would be better for you to store them in your home folder, in your Documents for example.
In this way your system will be better organised and you will know where to find everything. Especially if you have other users on your machine...

---

### Post by sekinto on 2008-04-13
There isn't a "default" place for storing backgrounds for a single user, but if you wanted to store them for use in only one account I would store them in /home/<account>/backgrounds (or .backgrounds if you want the folder to be hidden). If it is for multiple accounts there actually is a default location: /usr/share/backgrounds.
