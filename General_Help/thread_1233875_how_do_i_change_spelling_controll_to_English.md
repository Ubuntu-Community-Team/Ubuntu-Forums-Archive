### Post by furuno on 2009-08-07
Hello there!

If you can, try go the System > Administration > Language Support and simply change the drop down box to a language that you wish. If you can't get to the Language Support (due to weird menu language) simply type press ALT+F2 and type :
```
gnome-language-selector
```
After that Ubuntu will download and install (if it has not been installed yet) the language file. After installation complete try logging out and relogin. The language should have been changed by now...
