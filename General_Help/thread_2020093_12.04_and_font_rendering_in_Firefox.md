### Post by plegall on 2012-07-13
OK, I've solved this problem.

1) sudo apt-get install ttf-mscorefonts-installer ubuntu-tweak gsfonts-x11
2)  System Settings > Ubuntu Tweaks > Fonts,  hinting to "moderate" and antialiasing to "greyscale"
3) restart computer
