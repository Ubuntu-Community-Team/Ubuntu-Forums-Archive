### Post by balaknair on 2009-01-28
Most themes from Gnome-look should install without any problems. There are a few that may not be properly packaged.
The themes should be compressed into a file with the file name ending in .tar.gz 
Eg: 57063-Moomex.tar.gz

If the theme package name ends in .zip, .tar.bz2 or .rar, it will not install.
