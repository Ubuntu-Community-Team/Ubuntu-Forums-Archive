### Post by grahammechanical on 2011-05-08
Go to System Settings and look for a utility called Windows. Open it and look at the heading called Titlebar Action - double-click titlebar to perform this action. Select the action you prefer from the drop down list. It includes the action of None. If No Action can be accurately described as an action.

Regards.
