### Post by darolu on 2010-04-24
10.04 comes out in a few days, on April 29 to be exact; the release candidate is out already and it should be safe to install it, I don't think we'll see many changes between RC and final versions. If you decide to install the RC it will be updated to the final version (all can be done via cli).
Upgrading from 9.04 to 10.04 would require to upgrade to 9.10 first.
How VPS's are managed vary a lot from company to company, but I suppose you can ask your VPS provider to install 10.04 instead of CentOS (which is also good).
