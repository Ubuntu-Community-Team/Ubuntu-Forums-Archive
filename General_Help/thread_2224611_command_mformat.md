### Post by sudodus on 2014-05-17
Try with ***gparted***. It comes with the Ubuntu install iso files, and you can easily install it into an installed Ubuntu [based] system via the ***Software Center, Synaptics*** or a command line in a terminal window

```
sudo apt-get install gparted
```

Gparted has a graphical user interface and is quite intuitive and easy to use for making and changing partitions and file systems.
