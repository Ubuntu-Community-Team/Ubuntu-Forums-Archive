### Post by 3rdalbum on 2009-01-27
It's called "Notification Area", not System Tray.

If the notification area is actually there but the Network Manager applet isn't, you can restart the network manager applet by typing "nm-applet" in the terminal or in the Alt-F2 "run" box.
