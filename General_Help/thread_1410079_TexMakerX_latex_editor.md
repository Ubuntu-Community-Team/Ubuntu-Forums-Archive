### Post by Benzjaminz on 2010-02-18
After spending a long while trying different latex editors (I'm a grad student so I write a lot of documents), I've finally found a latex editor that really suits me. 

Texmakerx ([http://texmakerx.sourceforge.net/](http://texmakerx.sourceforge.net/)) is an extension of texmaker that has a whole lot more features added. 

Now I have most of my latex needs met:
inline spelling
section wrapping
cite and ref autocompletion
error highlighting

Definitely my favourite latex editor. Anyway just thought it might be useful if I shared it on ubuntu forums (I've just found the editor today and have no affiliation with the project).
