### Post by Sn00ze on 2013-12-31
Problem solved

The 13.10 Ubuntu version of uTorrent is not working, however if you download the earlier uTorrent verion for 13.04, either 32 or 64, just upack it in the download folder, also unpack the gui package same location, then just double click the utserver and the server will start in the background, then open your browser and use localhost:8080/gui/ to open the user interface and start using uTorrent.  Use admin as name and skip password.  It works fine.
