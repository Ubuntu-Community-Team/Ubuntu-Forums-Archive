I am running Ubuntu from the live cd. However, every time I turn off my computer, and turn it on, all the files that I saved are gone. Please Help.

### Post by ep2011 on 2007-05-28
This is because you are using a LiveCD. It does not save ANYTHING to your hard drive (and it is impossible to save on a already burned cd). This is normally a great thing, as you can go to a library or a public computer and use your favorite operating system, except in your case, when you need to save a file. To save you must save the files to a flash drive or external hard drive and then just plug it in next time you run the LiveCD. Alternatively, you can install it to your hard drive and everything will save and open perfectly.
