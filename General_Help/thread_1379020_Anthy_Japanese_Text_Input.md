### Post by benmoran on 2010-01-12
Ubuntu 9.10 now uses iBus by default instead of SCIM. Under your language options you can select which one to use. I recommend giving iBus a try if your system is still set to SCIM. After switching to it, you need to go to System-Preferences-iBus Preferences. From there you can add anthy.
