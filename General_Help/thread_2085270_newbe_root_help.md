### Post by lisati on 2012-11-17
On Ubuntu, the "root" user is disabled by default. The preferred method of accessing superuser privileges is via the "sudo" command. For more information, see [https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)
