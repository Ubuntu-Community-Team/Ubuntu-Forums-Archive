### Post by 3rdalbum on 2009-06-01
I installed Ubuntu 8.10 on a Foxconn 45GS motherboard (single-core Atom) and everything worked out-of-the-box. Note, this is a home server machine so I did not try sound.

I agree that the CPU isn't the compatibility sticking-point, it's the hardware on the motherboard. The ethernet card, sound, graphics, wireless if appropriate.

I'd maybe go so far as to say you're fairly safe with Atom-based desktops as they are likely to use all Intel stuff that is supported with Linux, but you'll want to check components before buying.

### Post by joker535 on 2009-07-03
I have done it. I bought one of the MSI wind nettop boxes to use as a nas and I put ubuntu on it to play with while I was waiting for my raid drives. It was slow to install on the single core atom but it went fine. I used 8.10 desktop 32 bit and was running 512M of ram. I surfed the net with it and checked my email and it ran fine. The MSI wind is an excellent box for a low end home surfer, cheap nas, or test server.
