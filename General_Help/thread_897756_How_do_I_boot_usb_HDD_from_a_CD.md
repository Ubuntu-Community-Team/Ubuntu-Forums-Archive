### Post by WRDN on 2008-08-22
If you want to boot Ubuntu at school, you could boot from a LiveCD (providing the BIOS on the PC isn't locked, and you can set the CD drive as the first boot device). If there are files on the USB HDD, just mount it from the LiveCD and access them.
