### Post by fragos on 2008-06-25
Gedit Preferences-> Plugins Tab-> Session Saver Check Box.

### Post by PenguinInside on 2010-02-21
> **fragos said:**
> Gedit Preferences-> Plugins Tab-> Session Saver Check Box.
This is only available if you have the gedit-plugins package installed:

sudo apt-get install gedit-plugins

Note: universe repository should be enabled beforehand.
