### Post by QIII on 2012-01-24
Might I suggest that you use some sort of virtualization, such as VirtualBox, and do all of your learning and practice there instead of directly in your primary machine?

If you break a virtual OS by doing something incorrectly, then you can just say "Meh.  Reinstall the virtual version."

Breaking your "real" OS can be disastrous.

### Post by bodhi.zazen on 2012-01-30
The best way to learn the command line is to use the command line.

All the guides listed here will help get you started, but when you perform sys admin, use the command line as much as possible.

It helps to learn regular expressions, sed, grep, awk, and redirects such as < , > , and |

Use tab completion

and try other shells, zsh is a nice shell as well.
