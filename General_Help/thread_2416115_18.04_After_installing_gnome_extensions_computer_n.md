### Post by huff2 on 2019-04-08
EC should be enabled to awaken from S3 (sleep) state with an event such as a key press. 

First, *sudo -i *(so you can be root, in order to edit this file)
Then, *echo "EC" > /proc/acpi/wakeup* (TYPE THIS LINE EXACTLY!)

Open that file back up and it should now say enabled next to EC. If successful, go to sleep and try to awaken with a random key press.

This process can be repeated to toggle any of those devices listed but I don't recommend playing too much with it. 

IF this doesn't work, then the problem could reside deeper. Let me know if this works.

If you want detailed info, I found a good resource for you: [https://wiki.ubuntu.com/UnderstandingSuspend](https://wiki.ubuntu.com/UnderstandingSuspend)

This details the entire call structure for sleep/awake

I tested it on my laptop just a second ago. I disabled LID (which controls what happens when you close your laptop lid, basically) and it worked fine. No issues.
