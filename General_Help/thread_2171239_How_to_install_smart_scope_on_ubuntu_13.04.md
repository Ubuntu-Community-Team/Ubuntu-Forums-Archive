### Post by lah7 on 2013-08-29
The Unity Smart Scopes will be coming in Ubuntu 13.10, so I would advise you to try a daily build of Ubuntu 13.10 if you want to test them out 'officially'.

As you already know, there's a PPA out there which allows smart scopes on Ubuntu 13.04, but as it's **still in development,** still isn't **quite as polished**, and is **unofficial**, that's all what is available for now. It's also why it was delayed for Ubuntu 13.04.

This page explains how you can install it with the PPA:
[http://www.webupd8.org/2013/04/how-to-install-unity-smart-scopes-in.html](http://www.webupd8.org/2013/04/how-to-install-unity-smart-scopes-in.html)

There's also an article on OMG! Ubuntu about its arrival:
[http://www.omgubuntu.co.uk/2013/06/ubuntu-13-10-readies-arrival-of-smarter-unity-dash](http://www.omgubuntu.co.uk/2013/06/ubuntu-13-10-readies-arrival-of-smarter-unity-dash)
