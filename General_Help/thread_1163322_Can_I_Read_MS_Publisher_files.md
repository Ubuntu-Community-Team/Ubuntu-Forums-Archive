### Post by s.fox on 2009-05-18
Hi,

You can use scribus which is a free alternative to MS Publisher

More info on scribus:

[http://www.bizhack.net/archives/73]("http://www.bizhack.net/archives/73")

[https://help.ubuntu.com/community/Scribus]("https://help.ubuntu.com/community/Scribus")

To install scribus give the command in the terminal:

```
sudo apt-get install scribus
```

Hope this helps.

-Ash R
