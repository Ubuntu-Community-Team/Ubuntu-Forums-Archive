### Post by Vince4Amy on 2009-03-04
Try the latest ATI Driver.

```
wget https://a248.e.akamai.net/f/674/9206/0/www2.ati.com/drivers/linux/64bit/ati-driver-installer-9.2-x86.x86_64.run
```

```
sudo sh ati-driver-installer-9.2-x86.x86_64.run
```

Run through all steps in Automatic. 

This will configure X with the new driver:
```
sudo aticonfig --initial -f
```

Reboot your system and everything should work.
