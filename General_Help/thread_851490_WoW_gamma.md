### Post by sayakb on 2008-07-06
There is a compiz plugin. Install ccsm (if you haven't already)
```
sudo apt-get install compizconfig-settings-manager
```
Open ccsm (*System->Preferences->Advanced desktop effects settings*)and enable the **Brightness and Saturation** plugin.
