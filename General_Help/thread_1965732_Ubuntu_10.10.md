### Post by gordintoronto on 2012-04-26
The most common cause of a laptop freezing is overheating. Over time, dust builds up, and the air circulation is not as good.

How long has this been going on? How long does it run before it freezes? Can you watch videos on Youtube, or do other CPU-intensive things? Do you hear the fans running?

I would install Ubuntu 12.04, since 10.10 is no longer supported, then install lm-sensors. In a terminal, you can run "sensors" and it will report from the available temperature and fan-speed sensors.
