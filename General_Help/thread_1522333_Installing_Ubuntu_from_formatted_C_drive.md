### Post by Elfy on 2010-07-02

You really don't want to kill the BIOS ...

Formatting the drive will not delete the BIOS. Now, if there is a BIOS update from your motherboard manufacturer, you can flash the BIOS.

Just burn an Ubuntu image to a disk and put it in. Installing it is really simple and user friendly.

Remember, drives don't have letters in Linux. So you're not installing Linux to a "formatted C: drive."
