### Post by bom28 on 2007-02-04
It seems one program is eating up all the ram. You should try to find out which one, using ksysguard for example. Of course you have to do that when the computer is still a bit responsive. If ksysguard takes too long to load, you can use a console program like top, it should be faster.
Once you know which program cause this, you can try killing it, or ask for more help.
