### Post by mb_webguy on 2009-04-15
Well, if your grandma "didn't get" Windows, I'm assuming her computer use is going to be fairly limited, mostly to two or three programs -- Firefox, a music player like Rhythmbox or Amarok, and a media player like Totem or VLC.

Personally, I think Ubuntu is already much easier to use for a user with little to no experience than Windows, partially because applications are automatically arranged by function, rather than by developer.  However, you can easily simplify things further if you want.

The simplest thing to do would simply be to add a couple of launchers to the desktop for the applications she's likely to use the most.  If the presence of the Gnome panels is a source of confusion, you can simply remove them (or at least the top one, since the bottom one is kind of useful).

To get something somewhat similar to Linpus, however, you could install the [Ubuntu Netbook Remix]("http://www.canonical.com/projects/ubuntu/unr") interface using the [Launchpad PPA]("https://launchpad.net/~netbook-remix-team/+archive/ppa").
