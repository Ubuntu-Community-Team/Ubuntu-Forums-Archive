### Post by QIII on 2014-03-28
Do you, by chance, have a dual boot with another OS that you can test with?  If so, that would be helpful.

If not, I'll go with my initial impression and say that looks like a hardware fault, especially since it just started and gets worse over time as you use the machine.  That makes me think it is a thermal fault -- it gets worse as the machine heats up.

I would try a cheap solution first:  If you can open the case, check the fan area out for obstruction with junk and blow it out with compressed air if you can.
