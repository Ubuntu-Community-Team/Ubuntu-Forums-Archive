### Post by Mark Phelps on 2013-07-06
The Mint "equivalent" of Ubuntu 13.04 is Mint 15 -- and if you installed that, you would likely see the same poor performance.

Mint 13 is more like Ubuntu 12.04 -- and if you want better runtime performance on a notebook, you should consider Lubuntu, instead of Ubuntu.

### Post by 3rdalbum on 2013-07-06
What graphics card is in your notebook? If it is an Nvidia or an ATI Radeon HD 5000 or above, you can install additional drivers to make the graphics faster. Go into the Software Sources program and click on the Hardware Drivers tab and you will be given the option to install a graphics driver if there is one available for Linux.
