### Post by kidders on 2006-09-26
Hi there,

There is almost no difference between the way Ubuntu organises things and the way other Linuxes do, so virtually any web page that deals with the conventional Linux filesystem layout can tell you far more than anyone could do here.

If you lose something, the "find" and "slocate" utilities can help, btw.

Hope that helps :-)
