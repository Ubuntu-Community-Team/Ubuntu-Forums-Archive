### Post by yancek on 2015-09-13
That's not enough information to give any suggestions.  Boot Ubuntu and go to the site below and either download per instructions to Ubuntu or download and create a bootable disk with it.  When you run it, select the option to Create BootInfo Summary and post the output here.

[https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)

Since you make no mention of any other operating system, you should have used the defaults during the installation which would have installed boot code to the MBR.  If you have a newer computer using UEFI, things are a bit different.  Posting the output from boot repair should give enough information for someone to suggest what to do.
