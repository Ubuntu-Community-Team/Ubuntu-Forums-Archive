### Post by 3rdalbum on 2011-05-22
If those folders actually have files in them, then they MUST be from mounted filesystems. Which would be normal, because mounted filesystems and their contents appear in /media.

If they are empty, then that occasionally happens (I think?) and are safe to delete if you want, or leave them there - they use no space when empty.

### Post by lmarmisa on 2011-05-22
If you have no devices mounted, I reccomend to remove these ghost folders. Maybe they are there because the external devices were not properly unplugged or because the system crashed while they were connected.

You can check the mounted devices typing this command:

```

mount

```

Remove the ghost folder typing this other command:

```

sudo rm -rf /media/*

```
