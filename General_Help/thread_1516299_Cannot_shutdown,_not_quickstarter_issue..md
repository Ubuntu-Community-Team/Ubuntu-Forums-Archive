### Post by Hen Tenis on 2010-07-17
Restart the computer, go into the bios, don't change anything, leave the bios and the issue is gone for a while. This is not a permanent solution! There must be something else wrong.

You can try to set the 'post delay' option in the bios set to 5 secs. I tried but i don't know yet if the problem is gone forever. It looks like the bios runs to fast for your hardware.
