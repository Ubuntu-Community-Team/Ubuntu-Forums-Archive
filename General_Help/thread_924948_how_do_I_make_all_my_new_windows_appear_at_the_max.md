### Post by howefield on 2008-09-21
I think it is in System > Preferences > Advanced Desktop Effects Settings

Find the "Place Windows" options and select maximise from the "Placement Mode" drop down box.

If you don't have the advanced desktop effects settings option in your preferences menu, load up Synaptic package manager and search for compiz, put a check in the box next to compizconfig-settings-manager and click apply, then apply again.
