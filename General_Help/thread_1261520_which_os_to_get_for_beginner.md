### Post by flugh on 2009-09-08
Try downloading each of the iso's and running them in live mode. No installation to your hard drive needed. Personally, I use 'unetbootin' to write iso images to my 1Gb thumbdrive then reboot the machine from the thumb into Linux. Safe, simple, let's you test drive without making any changes to the hard drive, and saves on CD media.

Personally, I'm a fan of the vanilla Ubuntu. I've always been a Gnome guy, although KDE4 looks really slick.
