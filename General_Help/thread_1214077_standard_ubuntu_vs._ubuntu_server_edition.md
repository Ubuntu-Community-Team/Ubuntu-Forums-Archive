### Post by jerome1232 on 2009-07-15
The only difference is the Server Edition has a kernel that's optimized for running server daemons, and it comes with no gui by default. If your comfortable working from a commandline I'd say go with server edition and just access it from another computer via ssh.


Even if you aren't comfortable with the command line a server doesn't need a full blown Gnome Desktop Environment with all the gadgets and whosits a desktop has, you could do the server install and add a lightweight Window Manager like xfce, fluxbox or icewm for example.
