### Post by deadflowr on 2016-11-02
You mean the /lib folder?
Then no, no one can really help.
as that folder contains lib/library files.
from libraries that come with the installation to the libraries for packages you installed yourself, ala sudo apt install.
So it is unlikely that anybody would have a lib folder that matches the one you have, exactly.
And it would have to be exactly, or else the apt package management system will go to pot.
As the package database will be all mish-mashed.

Best case, find some external media to backup what you have/can and reinstall.
