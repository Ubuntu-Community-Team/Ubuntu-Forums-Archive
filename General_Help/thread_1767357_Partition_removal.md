### Post by LordNeo on 2011-05-25
Yes, you can, you can go to Ubuntu, install gparted and then delete Mint partitions and assign it as a new partition for Ubuntu, or boot from LiveCD (or create a LiveUSB using the Boot Disk Creator) and then assign the space to one of the Ubuntu partitions
