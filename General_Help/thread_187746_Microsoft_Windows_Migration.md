### Post by rcarring on 2006-06-03
Windows users may be wanting to try out Ubuntu 6.06, but be reluctant to commit an install to disk. 

Initially, using the Live CD is the best option:

a) it's safe, as it doesn't write anything to your hard disk
b) it's feature rich -- you get the opportunity to try out some of the apps
c) it lets you see if your hardware works under Ubuntu

Now, once you are comfortable with running Ubuntu from CD, you may want to install it. There are three choices:

a) emulation under host
this way you use a product like vmware-player and a hard file downloaded from Ubuntu or from VMWare
b) dual boot
this may require you to resize the Windows partition, or edit the mbr.
c) new hard disk

Of these c) sounds difficult, but if you are happy changing the hard disk, it is the easiest way to try a hard disk install of Ubuntu without touching Windows at all, mbr or otherwise. You still have the windows drive of course, and if you want to migrate data, then you can either burn it to cd or slave the windows drive to the Ubuntu one and mount it.

a) is a great option, the one I use now, as if I muck up my installation I can restore from a backed up hard file and start again.

+++

Ubuntu provides two good programs for Office and Graphics --

a) OpenOffice.org 2.02
b) The Gimp 2.2.11

These programs are comparable to Microsoft Office and Adobe Photoshop.

There are several browsers, of which Firefox 1.5.3 is the default.

+++

For those Windows users who want to have access to Windows programs under Linux, there are three solutions AFAIK possible ---

a) WinE
b) CrossOver Office (30 day trial available -- commercial package costing $40)
c) Emulation

+++

I cannot stress how important it is to have a good backup of your Windows data if you go the dual boot option.

This is just an initial commentary, and I may add to it later.

Thanks
