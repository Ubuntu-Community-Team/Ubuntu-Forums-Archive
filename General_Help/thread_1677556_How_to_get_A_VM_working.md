### Post by daniel333 on 2011-01-28
Hey dude, 

Virtualization is a big topic. It can be weird at times. Just installing Virtual Box doesn't mean you get a free copy of Windows like you get with XP mode on Windows 7. Virtual Box just allows you to install a copy of Windows. (or any many other OSs)

To get virtual box you should just add it from the Ubuntu software center. You'll need a legal copy of Windows. So shoot down to best buy or where ever and snag one. 

There are lots of great guides on getting started with Virtual Box, but if you are looking for something that won't put you to sleep you can watch a series of videos on youtube and vtc.com.


Best of luck,

### Post by steveneddy on 2011-01-28
A Virtual Machine is **[COLOR="Red"]virtual[/COLOR]** hardware - in that you run this software and it looks like hardware to other operating systems.

If you are running Ubuntu then you must use the Ubuntu version of Virtual Box and install Windows on that.

By installing Virtual Box you are installing a **[COLOR="red"]make believe[/COLOR]** machine that only exists in software that Windows believes it is operating.

Weird at first but once you install it I think you will get the idea.
