### Post by ddales on 2008-06-22
This is normal for CPUs that use speed step.  My P4 and cpuinfo report the same way.  Mine is a 3.4 and cpuinfo shows 2800Mhz.

If you load your CPU the frequency will rise.

To see supported frequencies:
cat /sys/devices/system/cpu/cpu0/cpufreq/scaling_available_frequencies

### Post by Dhaun on 2008-06-22
As far as I know, you can turn on/off this feature in BIOS. When it's OFF  your frequency is set to constant 1,5 Ghz. When on, it will boost your frequency only when it's needed. The feature is called EIST. More info -> [http://www.intel.com/cd/channel/reseller/asmo-na/eng/203838.htm](http://www.intel.com/cd/channel/reseller/asmo-na/eng/203838.htm)

### Post by sdennie on 2008-06-22
I recommend installing cpufrequtils:

```

sudo apt-get install cpufrequtils

```

To see information about your cpu do:

```

cpufreq-info

```

If, for some reason you need to change your scaling governor (there are very, few reasons to ever do this) you can use, for example:

```

sudo cpufreq-selector -g performance

```

There is no need to use the BIOS to change this behavior however, I don't recommend changing it unless you are having problems with the default.
