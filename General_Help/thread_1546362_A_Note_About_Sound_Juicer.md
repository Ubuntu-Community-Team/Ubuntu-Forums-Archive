### Post by A_Nonny_Moose on 2010-08-05
I had occasion to extract a CD using this product, but the maker of the CD had been most punctilious in including French accents in the titles of some of the pieces.  Sound juicer shows these as a boxed X, and if you try to extract the track, crashes to the desktop.

The work-around is to replace the accented letters with unaccented ones.  I did not find out if this is a file system problem or a sound juicer problem, but felt it worth a note.
