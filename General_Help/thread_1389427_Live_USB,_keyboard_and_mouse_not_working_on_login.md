### Post by hwttdz on 2010-01-24
1st - yes it's ok to update your flash install, think of it as just a floating hard disk, and you're building a computer with that hard disk every time you sit down somewhere new.  

2nd - sounds like legacy usb is disabled in the bios of the machine on which you're trying it now (though that may not be the case).  Did you switch machines between updating it and trying it now?  if so when booting press del or f2 or f8 or.... and set legacy usb keyboard and mouse support to true.  

I was a little disappointed with a flash install when I first tried it, I was excited to just be able to reboot and have my machine, but I find that pretty often I have to flop the boot order in bios.  Nonetheless it's nice to be able to use community machines.  Last time I was working in a lab one person had to reboot windows 5 times while I was working.
