No, only the alternate iso will work on Ubuntu 7.04; you can't use the official CD, which is a liveCD (though liveCDs will work on the Ubuntu 7.10 version). However, you can always just install straight off of the CD if you don't want to download the alternate iso; just back up your important data in case you screw up, boot the CD, install, and you'll be good to go.
