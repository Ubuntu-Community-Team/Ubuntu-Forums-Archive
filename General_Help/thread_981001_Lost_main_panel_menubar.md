I have systems set up for work.  Someone has changed it so the entire bottom panel with all the menu items are gone.

When I right click on the desktop there is no option to re-add a panel (as I have seen suggested on other forum posts).  How do I get the main menubar back?  Is there some kind of kiosk option that makes the bar seem to go away?

---

### Post by Linux user 66 on 2008-11-13
You can get the menu bar back by right-clicking on a bar that already exits and selecting *New Panel*. Then you add the items, menus and features you want to it by right-clicking on it and selecting *Add to Panel...*

I hope that answered your question.
