Like you said it is simply the fact that Ubuntu can use the filename and XP cannot. There is no workaround. (check [this](http://msdn.microsoft.com/en-us/library/aa365247(v=vs.85).aspx) MSDN page for more information)

Your best bet is to rename the folder in Ubuntu without any reserved Windows characters. You should be able to open it in XP after that.
