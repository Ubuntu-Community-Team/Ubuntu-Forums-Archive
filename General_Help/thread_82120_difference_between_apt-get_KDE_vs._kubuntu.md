### Post by kairu0 on 2005-10-25
The difference, as you note, is all the peripheral applications that make up the kubuntu desktop as opposed to the plain-jane kde desktop.

If you just install KDE, you will get a basic kde setup that will function. If you install kubuntu-desktop, you'll get a bunch of extra stuff and it should ask you what to do before overwriting your configuration.

Another option is to add kubuntu-desktop, and then manually uncheck the packages you don't want to install before hitting Apply.
