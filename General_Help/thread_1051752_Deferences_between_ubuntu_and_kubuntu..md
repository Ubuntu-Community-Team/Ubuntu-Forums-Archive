### Post by mb_webguy on 2009-01-27
Kubuntu uses the KDE desktop environment, while Ubuntu uses Gnome.  Some people prefer one, some the other.  They both do the same job, which is to control the look and feel of the interface.  In my opinion, KDE feels similar to Windows XP, while Gnome feels like a cross between Windows and MacOS.  Both of them are highly customizable, and can be modified to look and act like anything you want.

Some applications are designed for use with KDE, and others for use with Gnome.  While you can run applications designed for KDE using Gnome and vice-versa, they tend to have a distinctly different look than applications designed for the desktop environment you're using.  For that reason, Kubuntu comes with a different set of applications than Ubuntu.  For example, Ubuntu uses Brasero to for disc burning, while Kubuntu uses k3b.  Ubuntu uses the Synaptic package manager, while Kubuntu uses Adept (though both use the Ubuntu repositories).

Basically, Ubuntu and Kubuntu are the same operating system, but are bundled with different sets of programs.  You can even have both installed at once, and decide when you log into your account which desktop environment you want to use.  If you use Ubuntu and want to try Kubuntu, just install the kubuntu-desktop package using either apt-get or Synaptic.  Likewise, if you're using Kubuntu, install the ubuntu-desktop package.  Then the next time you log in, click on Options and Select Session to choose which desktop environment you want to use.

And all of the above applies to Ubuntu's other sister project, Xubuntu (which uses the Xcfe desktop environment), as well.
