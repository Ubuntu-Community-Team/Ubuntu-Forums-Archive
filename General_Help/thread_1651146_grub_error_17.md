### Post by oldfred on 2010-12-22
Can you boot a USB. Create a USB live version.

If you cannot directly boot a USB from BIOS, do you have a working floppy drive? You can use plop to boot a USB live version.

If you have neither of above you have to remove hard drive from non working computer and plug it into another computer to repair it.

Buy a working CD drive, they are pretty cheap now.
