### Post by accooper on 2009-05-04
Open Open Office Writer

Go to files menu

go to save as...

the save as dialog box opens

under the list of places to save your file you will see a long narrow box with an up arrow and a down arrow.  Click on the down arrow.

There should be a list of formats you can save your file to such as Word, RTF, Text, etc.

Chose the one you want.  If you want true compatibility RTF is good, DOC is in there.

If there are no other formats available your Open Office is corrupt and needs to be reinstalled.

### Post by geirha on 2009-05-04
Does the recipient need to edit the document you send? If not, I recommend you choose File -> Save as PDF. Virtually everyone has Adobe Reader or some other PDF-viewer installed, regardless of OS. In addition, the fonts are saved inside the PDF-file, so it doesn't matter if the recipient doesn't have the same fonts as you.

Otherwise, see the screenshot on how to save it as a Microsoft Word document.
