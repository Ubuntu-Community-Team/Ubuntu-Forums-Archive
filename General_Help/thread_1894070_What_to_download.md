### Post by c.cobb on 2011-12-11
Hi and welcome to the forum.

+1 for the suggestions of Ubuntu 10.04 and Mint. I haven't tried the latest Mint, but I read good things about it. 

If your PC has decent power, maybe try installing the free [VirtualBox]("https://www.virtualbox.org/") software from Sun/Oracle. It may be a little tricky the first time you use it, but it avoids having to burn ISO files to CD, and lets you easily run Linux while you're doing other things throughout the day. 

Set up a couple of virtual machines, and plan on spending some time evaluating things. That way you can make sure that everything that you need will work, or that the alternative stuff you find is sufficient for your use. 

I spent many years doing Unux admin and programming, and it took me several months to switch. No need to rush, and have fun!

### Post by hypertyper on 2011-12-12
Long story short, on that hardware try Xubuntu and see if you like it.

Short story Long:

In terms of disk space you'll definitely be fine. The hardware however isn't fantastic. Are you currently running Win 7 on the machine?

If I were you I would boot into several different Linux distros and see which one you like. There usually are downloads of around 700mb which you can easily put onto a usb stick and boot from within minutes. That way you can quickly get a feel for different versions without doing anything permanent.

I'm a long time Windows user and have played around a lot with Linux lately. Quick summary:

Kubuntu: 
- flashy looking but will need decent hardware
- very different approach to desktop and extensions
- runs a different set of libraries for its graphics so some standard applications look pretty horrible
- I kept getting annoying errors when shutting down

Ubuntu:
- Unity leaves no room for customization which I hated
- otherwise very complete package

Mint:
- liked the 11.04 version 
- the latest version is a imho horrible crossover of gnome 2 and gnome 3 = no customization = not for me

Xubuntu:
- very fast
- has everything you need
- great for customization
- my favourite by a long way at the moment
- it should run very well on your machine

Everybody has their preferences so there isn't a "best" distro that suits everybody.
