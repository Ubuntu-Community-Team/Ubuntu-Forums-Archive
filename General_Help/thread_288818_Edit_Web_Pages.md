### Post by ejket on 2006-10-30
I've always edited html files locally and launch my editor from the browser.  This allows me to follow links and edit pages as I come to them, which I'm not sure nvu and the other html editing environments will do.  It seems more natural to edit files in context this way.

Firefox requires an extension for this (I've used launchy), which is annoying since extensions disappear after upgrades.  Konqueror does this natively -- all you have to do is add your editor to the html file type and move it down the list at least one place so its not the default.  Then on any page, you just do "open with" from the context menu and select your editor.

This is the way to go especially if you have a favorite editor; in my case I like to use gvim for everything.

Nvu does look interesting, though.
