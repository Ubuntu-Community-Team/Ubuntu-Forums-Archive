### Post by jdong on 2004-10-31
nano is a pretty nice editor. The key sequences are extremely simple to learn (i.e. ^X to quit, all listed on the bottom of the screen.)

jed is another nice editor with more features. It is simple to use, with a menu system (keyboard shortcuts are optional, you can always use ALT to navigate menus)

jed supports highlighting and other programming features.
