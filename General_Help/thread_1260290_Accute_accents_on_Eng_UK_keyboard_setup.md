### Post by CatKiller on 2009-09-07
> **Leighman said:**
>  Is there a comparable easy way to do/set this on Ubuntu?

Even easier. You use the Compose key (you can set which key this is in the Layout Options screen). Compose &#8594; ' &#8594; a gets you á. They're all quite easy to remember; Compose &#8594; ~ &#8594; n is ñ, Compose &#8594; " &#8594; o is ö, and so on. There's a fairly comprehensive list [here]("http://www.hermit.org/Linux/ComposeKeys.html").
