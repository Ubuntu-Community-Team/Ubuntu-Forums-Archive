### Post by GammaScorpii on 2012-05-03
"amd64" just means it's 64-bit. It will still run fine on an Intel system (if your CPU supports 64-bit). If you select the 32-bit version, it's named as "i386".

I think it's because back in the day the 64 bit version was only for AMD systems because obviously Intel didn't make 64-bit cpu's. Nowerdays they do, but the naming convention just stuck.

### Post by newbie-user on 2012-05-03
GammaScorpii is correct. AMD was the first one to introduce a 64-bit general-use CPU, so the name amd64 was used and is still being used today. 32-bit systems need the i386 and 64-bit systems can use either the i386 or amd64 iso.
