### Post by snowpine on 2009-04-14
The Dell Mini can't boot from an SD card, so that won't work.

Can you boot from the ipod like a USB thumb drive? If so, that could work.

Another option is to use Unetbootin in hard drive mode. This will allow you to install Ubuntu without any external devices. [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)
