I just figured a solution for running Boxee on a dual monitor set up. It seems loads of people were having this problem as well as me.

Run Boxee in windowed mode. press \
open ccsm
go to Effects
Window Decoration
Decoration windows =  "(any) & !(class=Boxee)" without the quotes
voila
Boxee corrects itself to the size of your screen as well.
