### Post by mnavasuk on 2009-08-28
You're likely to have it figured by now, but if it helps, I just found it out myself. On gnome-do website is a page with the repositories to enable. You can find the page here: [http://do.davebsd.com/wiki/Installing_Do](http://do.davebsd.com/wiki/Installing_Do)
As of now, for hardy they are :
deb [http://ppa.launchpad.net/do-core/ubuntu](http://ppa.launchpad.net/do-core/ubuntu) hardy main
deb-src [http://ppa.launchpad.net/do-core/ubuntu](http://ppa.launchpad.net/do-core/ubuntu) hardy main
