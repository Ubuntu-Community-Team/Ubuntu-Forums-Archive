### Post by JKyleOKC on 2009-07-16
Have you tried "ls Sp*" to get around the strange characters? If that works, try "mv Sp* SpinalTap" to rename it...

The wildcard capability can often solve the problem of unusual characters in file or directory names, but it's always a good idea to do a list first, just to be sure that only the one name you're interested in gets matched.
