### Post by FuturePilot on 2007-12-04
Was it halfway through downloading or installing when you lost the connection? It downloads all the updates first so if you lost the connection while they were installing you'll be fine. You can check to see if everything was installed by opening Synaptic (System>Administration>Synaptic Package Manager) and going File>History. Find today's date and it will tell you everything that was changed.
