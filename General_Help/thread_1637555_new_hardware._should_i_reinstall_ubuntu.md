### Post by cwwilson721 on 2010-12-04
Ubuntu (and most Linux Distros) couldn't care less. The kernel, when booting, will load the drivers it needs. You might have some minor issues with some hardware if it is VERY new (and no kernel modules for it), but 99.9% of the time, you should be fine.

Windows, on the other hand, will think it's in a new computer, and either BSOD or un-activate itself. Do a Google search on "windows bsod new motherboard" and see what might help.
