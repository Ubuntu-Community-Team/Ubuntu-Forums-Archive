### Post by CrusoeTP on 2009-09-13
I'm sure some of you out there have had problems printing PDFs from Okular, or any webpage from Firefox under Kubuntu. I FINALLY FIXED IT!

So I set up my network printer through the KDE System settings tool.

But whenever I loaded a PDF in Okular, and tried to print, the printer light would blink for a few seconds, and then nothing would come out. Same for Firefox!

So I went back to the Printer Configuration in the KDE System Settings tool, and went to the Options tab, and under Resolution, changed it from "Printer Default" to a specific DPI setting, I chose 300DPIi ( any supported value should work ).

Went back to Okular, and it finally started printing! It even fixed printing from Firefox for me!  ;D
