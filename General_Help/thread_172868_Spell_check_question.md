### Post by PhrankDaChickenGeek on 2006-05-10
You can add Languages in "System -> Administration -> Language Support"
I don't know about Gaim, but you can select the spell check language in Text Editor by selecting 'Tools -> Set Language'
You can also set your profile language when logging in. Before you log in, click then 'Options' menu, and you can set your default language.
