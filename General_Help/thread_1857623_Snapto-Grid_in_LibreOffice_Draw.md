### Post by MarjaE on 2011-10-10
The points are not typographic points. The points are intermediate snap-to points between the basic grid.

If you want the basic grid to be 1" intervals and the snap-to positions to be 1/10" apart, you can set the grid to 1" and 9 points. If you want the basic grid to be 1/2" intervals and the snap-to positions to be 1/10" apart, you can set the grid to 1/2" and 4 points.

So if you want to divide in 4, 3, N, you select 3, 2, N-1 points. I think 1 point, the default setting, disables snap-to-grid.
