### Post by null byte on 2008-08-12
Yes it is :)

After you install your graphic card, go to System - Prefences - Apperance - Visual Effects and select **Extra**.

After that, go to terminal:
```
[FONT=monospace]
[/FONT]sudo apt-get install compiz-extra
sudo apt-get install compizconfig-settings-manager
sudo apt-get install fusion-icon

```
