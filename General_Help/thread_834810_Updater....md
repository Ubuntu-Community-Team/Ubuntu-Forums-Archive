If I NEVER want one of the updates on the update manager that shows in system tray, how do I tag it as such?

---

### Post by iaculallad on 2008-06-19
Navigate to System->Administration->Software Sources, select the "Updates" tab and uncheck the Ubuntu Update you don't like to be installed. 

*But never the Security update*
