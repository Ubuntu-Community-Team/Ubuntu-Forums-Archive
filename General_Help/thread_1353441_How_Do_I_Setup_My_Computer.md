### Post by Rob_H on 2009-12-12
I believe you can run all three operating systems legally and easily only if you purchase a Mac and run a commercial virtualization product on top of it like VMWare Fusion. That will enable you to run Windows and Linux as guests.

If you are willing to give up the OS X requirement, you'll have a lot more flexibility. In that case, you can run either Windows or Linux as the host OS and run the opposite one as a guest OS using any of several excellent cost-free virtualization packages like VirtualBox (my personal favorite for desktop use) or VMWare or several others.

### Post by Ordes on 2009-12-13
OK... so here more detailed...
What i am not sure of is, do you also want to run MacOS virtual, or like dual boot? If virtual its basicly the same as i describe here for Win..

---
Got your new Laptop and ubuntu on a CD.
Boot the Live CD
Look for partition editor or gparted. (you actually can do this during instalation, but i prefer doing it before)

Create your partitions:
1. Active partition 10-20 gb which will be /..
   in linux this will be sda1, and this is where we install the system format to ext3, or ext4, whatever u prefer and distro u use. For Karmic ext4 is default
2. For the rest of the drive i prefer 1 extended partition, that i divide following:
   A) Extendend will be sda2
   B) 4gb; Swap-Drive, which will be sda3  > a virtual RAM
   C) Rest; format ext3 or ext4 which will be sda4 > will be /home, this is where you safe your files, configs etc

You have to remember those SDAs, as you will need them for the instalation. Where sda2, is just on big partition, with sup-partitions sda3 & sda4. Why sup-partitons? Because you only can have a limited amount of active partitions like sda1 on each harddrive. Active partitions are needed to install a OS.

Now you r ready to restart and install ubuntu. Or just hit install in the Live OS. 
When using a virtual machine you dont have to worry about different filesystems e.g windows, as the virtual software will take care of it. So, also your windows can read from your ext filesystem. No need for extra partitions etc, for your virtual machines.

When installing, ubuntu will ask, sth like how to install on the drive. You can choose between guided, auto and manual...
Choose manual, you should see your partitions
sda1 choose /   (which is the main entry for your system)
sda3 choose Linux-Swap
sda4 choose /home

finish the installation
Now you have your system and your files on 2 different partitions.
-----

Virtual Machine
e.g Vbox
1) Install Vbox
2) Create a folder; e.g V-Machines
3) Create a folder; e.g V-Docs
4) Load up Vbox, and install Windows. Vbox will ask to create a file/folder where the virtual machine will be installed. Put this into your V-Machines/WinXP. So this folder/file is your complete virtual windows machine.With everything you install, safe, etc in windows
5) Got your vir-Win running.. You have to create a shared network drive, which Vbox uses to get access to your hard drive, this folder will be; e.g V-Docs. 
6) Vir-Win can now access the folder V-Docs on your hard drive and all the files in it.
7) I prefer to just link into this folder. e.g create a link in ubuntu for your documents. Cut&Paste this link into V-Docs. Now vir-Win can also access your documents on your hard drive... and so on

---
Backup, restore, DeepFreeze
As i said, if ubuntu is your only real OS and the others are virtual i would just go with remastersys.
Remastersys makes a 1:1 clone image of your Ubuntu with all the apps u installed and so on. When u make the clone u can choose to include /home or not. Probably u going to have a lot of data in /home. So i exclude it, as i backup my files seperated. Remastersys is just a system backup for me.

It will create an iso. This u can burn on a CD or put with the USB creator (comes with ubu) on a usb stick. Now u have a live system, (like the ubuntu u downloaded before), of your system. You load it from the CD/USB, and can choose between, Start Ubuntu without changes to Computer, Install Ubuntu and so on. Difference is, all application you installed etc will be in this system. So in case of a system-crash, all you need to do is reinstall your system to sda1, and you have your running system back. Should take about 10min. It is also good to make a clone of your ubuntu to put on other machines.. Or to just carry your ubuntu around with you on a usb....
