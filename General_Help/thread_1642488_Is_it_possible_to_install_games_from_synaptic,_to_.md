### Post by Jahid65 on 2010-12-11
This can be done by manual installation. something like tar.gz or .sh file. I have installed netbeans & PDf Studio like this. Directory should not have space.

If you want deb from synaptic for later installation(offline) select the game in synaptic & mark for installation (don't press apply). Then go to file=> generate download script & save the script (give a name) in home or wherever you like. Then double click that file & select open in terminal. The deb package will be downloaded in the same folder.
