### Post by garvinrick4 on 2010-09-30
Wired or wireless. If wireless use a cat5 and plug into your router and see if wired works.
If no cable unplug the one the desktop is using and plug into laptop. Go through the procedures to what is exactly wrong. Right click on wireless applet and make sure all is enabled and so forth. Then give these folks a complete run down and some one will help you get up and running. 
```
gedit /etc/resolv.conf
```
is where network manager will put its info in when working.
