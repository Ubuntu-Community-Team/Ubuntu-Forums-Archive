### Post by AlanR8 on 2008-02-25
Evening fellow KUBUNTU user!

I've been using K now for over a year and love it. I find it stable and quick. I also like the way I can customise almost all aspects of the GUI.

Now back to your question.

Part of my install procedure is to install the application Automatix (prepared for flames!!). 

[www.getautomatix.com](www.getautomatix.com)

This is a script installer that automates the installation of several bits of software and makes things nice and easy and is graphical. The reason I use it is that one of the scripts it runs is the installation of all the codecs you need to run MP3 files, all video file formats and the codecs required to play commercial DVDs.

Personally, I don't like the native Kaffeine program for playing videos and DVDs. Not a problem. I make sure I get VLC from the repos and I install Mplayer together with its Firefox add in. (I don't use Konqueror as a web browser).

All multimedia is now covered.
