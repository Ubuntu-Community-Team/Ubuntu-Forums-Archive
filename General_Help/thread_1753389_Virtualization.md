### Post by daweefolk on 2011-05-08
AFAIK the server install is just ubuntu installed without X and a gui. What I suggest is (if you want a really light system) go with the server then apt-get install fluxbox or apt-get install ratpoison. fluxbox and ratpoison are EXTREMELY light window managers, and you'd be able to install and use virtualbox like normal.
Both are quite different than gnome and kde, so there was a bit of a learning curve when i first used them but the only way i could speed up my system more was leaving X out of the picture.
