### Post by geog_eric on 2008-01-22
I've been trying to do n-up printing (multiple pages per sheet) using the foo2zjs driver with an HP LaserJet 1022 and found a simple solution: install "psutils" via Synaptic (as it is stated within the foo2zjs description). I didn't have to do the manual install (at [http://foo2zjs.rkkda.com](http://foo2zjs.rkkda.com)) that's suggested in other forum entries.

Figured this might be useful for someone!
