### Post by fish2ways on 2007-10-21
Hi
I've tried most audio players and currently use Exaile, in synaptic package manager under Multimedia/Universe. 
If you're using KDE then you're already using the best audio player bar none - Amarok. 
If you're using Gnome then you would be better off using a different player. I know it is possible to get Amarok to run under Gnome but I've never managed to do so without problems.
Or there's VLC, in Synaptic under Graphics/Universe, which is a multimedia player that plays anything - audio or video.
Best of luck
