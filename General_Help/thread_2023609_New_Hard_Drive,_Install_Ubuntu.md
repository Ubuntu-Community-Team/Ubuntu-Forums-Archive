### Post by PhantomTurtle on 2012-07-12
> **czerdrill said:**
> Well Ubuntu was installed on the old one also.  I'm just upgrading the disk space.  Does it make a difference if Ubuntu was already installed?

Yes it will make a difference when you put in a whole new hard drive because the stuff on the old hard drive...well its on the old one. Backup all your documents and you will have to reinstall on the new hard drive. Make sure to make backups of all your config files and a list of programs you want to install.

EDIT: If this drive is being used as a secondary drive, then don't worry about reinstalling Ubuntu.
