### Post by gpilkay on 2009-04-10
I would also second VirtualBox, as I need XP for college, but I have an Ubuntu 8.10 VM on it that works well for almost all my linux stuff.  I also have a 'spare' system I rebuilt that runs Ubuntu, though rather slowly.  If you are unsure, go the VM route first, that way if you don't like it or can't get everything you need, you have not hurt anything.

The next stage up is the Wubi installer, on the Ubuntu disks recently.  This installs Ubuntu as a windows app, without partitioning.  I have NOT done this myself, so it may not be the best, but I've heard good things about it.

These both assume you don't want a duel boot, that is also an option, though partitioning spooks me.
