### Post by Mark Phelps on 2010-02-07
Overheating with something as simple as an install is a very bad sign!  Ubuntu 9.10 has seen more reports of overheating than previous versions.  Overheating can permanently damage your laptop to the point of making it useless.

Laptops are designed to be used without overheating but in many cases, that is done by having a hardware monitor running the in background that reads the CPU and/or system temps and adjusts the fan speeds and/or CPU speed to compensate for the heat.

If the Linux version you're running does not support such monitoring, the result is either fans that are running too slow or CPUs running too fast (and NOT throttling down when they get too hot) -- or both.

So, have a few questions ...

Did the earlier installation exhibit the same overheating?

IF not, is this the same version you're reinstalling? Or is this a newer version?

IF this is a newer version, and you did not have overheating before, you should seriously consider going back to the former version.
