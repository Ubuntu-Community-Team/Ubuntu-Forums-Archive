### Post by joeyknuccione on 2009-04-24
Turn the computer off, then back on.

Early in this process it should offer a way to go into BIOS. In BIOS, make sure the disk is enabled as first boot option.

edit to add:
Make sure the CD/DVD is the disk set to boot first :)

### Post by blazemore on 2009-04-27
If there is a message saying "Press F12 for Boot Menu", or similar, then repeatedly hammer the key it's telling you to. This should bring up a menu allowing you to select from which device you'd like to boot.
