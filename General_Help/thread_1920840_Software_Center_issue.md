### Post by ajgreeny on 2012-02-05
Ubuntu 9.10 is not supported any more, so updating will only be possible if you want to move to 10.04, which is supported for another 14 months, and should be available through synaptic, if not software-centre, which I never use;  it's always synaptic for me.

There are what is known as end of life, or end of support repos for unsupported versions of Ubuntu, see [https://help.ubuntu.com/community/EOLUpgrades](https://help.ubuntu.com/community/EOLUpgrades), but that is only for adding any packages you want in those old versions, and will still not give you any security or bug-fix updates.

### Post by Old_Grey_Wolf on 2012-02-05
9.10 is no longer supported, and the repositories have been moved to old-releases. 

Ubuntu does not upgrade versions of most software. Usually, Ubuntu only applies security patches and some recommended patches. 10.04 will have older versions of software than 11.04.

If you always want the newest version of software then thy a rolling release distro; such as, Arch or Gentoo.
