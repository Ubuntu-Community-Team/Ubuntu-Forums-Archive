Can i make thunderbird to start on its own as ubuntu boots in automatically?

---

### Post by anystupidname on 2008-10-20
Add Thunderbird to the list:
System>Preferences>Sessions

Or if you're using KDE:
cd /home/<user>/.kde/Autostart
ln -s /usr/bin/thunderbird
