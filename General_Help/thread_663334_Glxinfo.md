### Post by taurus on 2008-01-10
Maybe try using the ati driver instead of vesa for your graphic card.

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak
sudo dpkg-reconfigure -phigh xserver-xorg
```
One you are done with it, restart your X server with <Ctrl><Alt>Backspace.
