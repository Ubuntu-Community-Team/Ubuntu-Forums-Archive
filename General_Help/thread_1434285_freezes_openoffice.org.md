### Post by Hagar Delest on 2010-03-20
For the freezes, try first to [reset your OOo user profile](http://user.services.openoffice.org/en/forum/viewtopic.php?p=58403#p58403) but don't transfer your personal data during the welcome process (if you had 2.x before), configuration files from the former version might corrupt the new profile.

If no change, you can try to install the Sun version: [[Ubuntu] Installing OOo on Debian and Co.](http://user.services.openoffice.org/en/forum/viewtopic.php?f=16&t=68)

For the images from the web, it's a known problem, either copy the pic alone (with right click) or save it first on your HD. Note that saving first is better because when you paste directly a pic, it is often changed in .png format to avoid quality loss. But if it's a jpg, it may increase heavily the file size. Whereas using the menu Insert>Picture>From file preserves the original format.
