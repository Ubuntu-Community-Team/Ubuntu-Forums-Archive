### Post by PABlanche on 2017-10-25
Hi,

Upgrading from 17.04 to 17.10 I cannot print anymore to network printer (HP laserjet pro 1606). 
I re-install HPLIP and the driver, which did not help.

All the printers have been re-installed automatically after the 17.10 upgrade.
I can access the printer via its ip address (not a network issue)
I am seeing the printer in the device manager.
The printer sate has been enabled.

When printing, even a test page, the printer queue say the job is "stopped".

I cannot cancel the job via the new device manager GUI, I need to go to the "additional printing settings", which open the old device manager GUI, then go to the "view print queue" tab.

I had a system error (not all the time) that said:
"hpcups assert failure: *** Error in `HP_LaserJet_Professional_P1606dn': free(): invalid pointer: 0x000055d0f2a71544 ***". 
see: [https://bugs.launchpad.net/ubuntu/+source/hplip/+bug/1727455](https://bugs.launchpad.net/ubuntu/+source/hplip/+bug/1727455)

Thanks for any help
-- 
Pierre
