### Post by fragos on 2007-12-09
The most I've been able to do is convert the flash file to another format and edit that.  Check out "Lives" on [http://www.getdeb.net](http://www.getdeb.net) as a video editor.  It's the only one I've had real success with.  ffmpeg is a great command line format converter although I'd suggest you go to [http://www.gnomefiles.org/app.php/PyTube](http://www.gnomefiles.org/app.php/PyTube) and check it out as the GUI tool to convert formats.  It actually uses ffmpeg.
