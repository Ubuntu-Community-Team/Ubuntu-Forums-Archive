### Post by seatea on 2006-05-04
Download the ppc file or files you want. Open them with tar -xjvf and then read the "ReadMe" file in the folder. It advises manually moving the files to  /usr/lib/codecs or some other appropriate place. That is how it is done under Breezy. I am guessing it is the same with Dapper.
