### Post by PCaddicted on 2011-07-20
Try installing Internet Explorer using WINE or PlayOnLinux( look in the Software Center). Or use VirtualBox to create a virtual machine with Windows, on which you could run IE.
Alternatively you can install Windows as dual-boot in order to use IE, if the two options above don't work.
