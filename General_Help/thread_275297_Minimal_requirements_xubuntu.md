> **Architeuthis said:**
> What are the minimal requirements to run xubuntu? How many ram do i need and how much mhz? Will it work on a computer with 1000mhz and 128 mb ram or can it go even lower?
That machine will work perfectly for Xubuntu. That's about what I use for my Dapper machine -- the one I rely on to be working all the time.

Of course, you can use much less than that, if you're willing to trim the desktop environment, or work with a different desktop altogether. In my experience, Xubuntu installs a lot of bells and whistles that might slow down a sub-500 machine (a lot less than straight Ubuntu, of course ;) ), but works great if you leave those things out.

### Post by K.Mandla on 2006-10-20
> **s1ugh34d said:**
> Im wondering what the minimal graphics would be to set upubuntu ...
In my experience, a practical bottom line for Ubuntu is about what you would expect for a practical bottom line for XP -- around 1Ghz, with 256Mb and at least 4Gb of disk space. 

You can get away with a low-end video card, if you're willing to sacrifice some graphics processing power. I've used 32Mb GeForce2 GTS cards without issue; 8Mb cards are acceptable if you're not doing any 3D work, but 2Mb cards tend to stutter unless they're at 800x600 (or less -- of course, there are ways around that). Below that, and it gets a little clunky. :???:
