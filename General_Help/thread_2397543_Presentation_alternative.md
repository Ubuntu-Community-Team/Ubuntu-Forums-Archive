### Post by dragonfly41 on 2018-07-31
If you are making a presentation I suggest that you write it in markdown (I use Atom editor) then use the Atom Markdown Preview Enhanced package to create a reveal.js presentation.  It is quite easy to use.  Pandoc.org also allows conversion of markdown to reveal.js presentations.
