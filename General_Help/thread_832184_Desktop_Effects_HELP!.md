### Post by mempman on 2008-06-17
Your getting that error because you have generic video driver installed.  You can install proprietary drivers by:

i) Check to see if you have the video driver listed under System > Administration > Restricted drivers.  If so, you'll have to enable it.

ii) If i) fails, then you will need to get new drivers from either your package manager or ur hardware vendor's site.  I would recommend the former.
