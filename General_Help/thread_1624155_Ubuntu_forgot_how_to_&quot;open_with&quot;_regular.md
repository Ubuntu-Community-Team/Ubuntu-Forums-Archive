### Post by papibe on 2010-11-17
The "file manager" application is called Nautilus. It looks like the defaults for opening a folder got changed. Does this happen also when you go to Places and open a folder (like Home or Music)?

If it also happens in Places, you need to change a gnome (desktop) setting. If not, it's a firefox setting (Firefox -> Edit -> Preferences -> Applications).
