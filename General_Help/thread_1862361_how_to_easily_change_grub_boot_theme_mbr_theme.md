i've been doing some looking around and found this super boot manager that lets you change the boot menu options and boot screen in grub.in two simple steps here it is.

install Instructions:
Open Synaptic Package Manager and Add repository:
ppa:ingalex/super-boot-manager
open a terminal:
sudo apt-get install "super-boot-manager"

it will be under accesories.
works for ubuntu/mint 11.04 and below.
doesnt currently work with 11.10
