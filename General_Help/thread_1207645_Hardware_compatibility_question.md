### Post by amopresunto on 2009-11-29
Sorry to rehash an old thread....

I also have this same motherboard and CPU.  Was running 9.04 on an old Seagate 20GB IDE HDD.  Decided a newer HDD would be good, so I got a WDC 500GB Blue series HDD, and installed Karmic Koala (9.10).

Under 9.04, the graphics were not great.  Playing youtube clips fullscreen was sloooooooooooooww.  Other video (MPG, AVI, etc.) was also sub par.  With 9.10, the graphics are greatly improved.

9.10 is not perfect by any means.  Occasionally after coming back to the computer after the screen saver/power management has put the monitor to sleep, I cannot get the system to wake the monitor.  When this happens, I just have to hard reset the system.  Not sure if there is a fix for this, but this is my experience with 9.10 on the MSI G31M3-L V2.

I also notice that flash player is buggy.  Some flash based web pages do not work correctly.  Example_1: embedded video from youtube in a site other than youtube does not work; if you click to play or anything else, nothing happens.  Example_2: Flash based games do not respond to any clicking whatsoever.  Note that this is not from all websites.  Example_3: Viewing product images from newegg.com that use Flash to allow you to zoom in --- well, you can view the images just fine, but it always shows a message in the middle of the image stating "Please try again later. Our apologies."

While there are bugs, overall the system functions well.

[QUOTE=bw1]It failed under multiple install attempts with 8_04 & 9_04 i386 and AMD64.[/QUOTE]
Did you run a disc check on each CD to make sure the media was error free?  I have run into installation problems in the past, then when I used the media check to make sure the disc was good, the check found errors with the install CD.  After making sure that the ISO was good, I burned a new disc and the install went without a hitch.
