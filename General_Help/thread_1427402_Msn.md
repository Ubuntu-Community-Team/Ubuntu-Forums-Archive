### Post by bonethug on 2010-03-11
FahadMKS,

I'm using Kopete in Gnome with kubuntu-netbook-default-settings package on my mini book. The kubuntu-netbook-default-settings is a kubuntu package which was/is prepared for mini books. This means that this package redesigns the original Kopete (and other kde applications) to a smaller, clearer, cuter and smoother version.

To install Kopete just su to root and type: ```
apt-get install kopete
```For the additional package that makes Kopete look even better type: ```
apt-get install kubuntu-netbook-default-settings
```Hope that helps.
