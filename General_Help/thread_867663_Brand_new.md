### Post by sdennie on 2008-07-23
You can install what essentially amounts to Kubuntu into an existing Ubuntu install by typing the following in a terminal:

```

sudo apt-get install kubuntu-desktop

```

When you login, you should notice an Options button on the bottom left of the screen.  Click it and you can then select KDE as your session type.
