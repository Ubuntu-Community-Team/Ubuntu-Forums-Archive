### Post by 666f6f on 2010-10-23
I use postfix for SMTP and dovecot for POP3/IMAP in a simple home setup (and love it!). Up and running with ```
sudo tasksel install mail-server
```.

You might want to check this out: [https://help.ubuntu.com/community/PostfixBasicSetupHowto](https://help.ubuntu.com/community/PostfixBasicSetupHowto), to fine tune the postfix installation.
