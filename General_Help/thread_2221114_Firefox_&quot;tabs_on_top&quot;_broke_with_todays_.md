### Post by ajgreeny on 2014-04-30
I think this is supposed to be a feature not a bug in the new FF 29 with the australis UI which some users love, others hate.

I'm indifferent to the changes made except that I always have several add-ons and some of those stopped working in FF 29, eg Hide Caption Titlebar Plus, and DownloadHelper, and I always used small icons in the main toolbar which is not now possible unless you use the extra add-on I mention below to get back the classic theme.

There are newer versions of both of those available from the devs' sites, even if not from Mozilla-Add-ons direct, and you can get some of the other UI back with the [Classic Theme Restorer]("https://addons.mozilla.org/en-US/firefox/addon/classicthemerestorer/?src=search") add-on.  It does not, however, allow you to put the tabs back below all other toolbars as you could before.  Perhaps this will become available in future, perhaps not, but as I say, it does not bother me too much
