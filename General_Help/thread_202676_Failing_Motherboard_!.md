### Post by nixternal on 2006-06-23
What mobo is it? I know some Abit motherboards are known to break a solder trace around the memory seats. This would cause that too. If it isn't memory, then motherboard would be next. If it failed memory tests prior, and then you got new memory and it still fails memory tests, then I would say your mobo might be on its way out the door. However, if the mobo is a no brand off the wall type job, for instance PC Chips, it could be a good motherboard, just bad logic and not really compatable with much.

### Post by nixternal on 2006-06-23
You would have to remove the motherboard from the system, and flip it over. Wiggle the ram seats and look and see if any of the solder traces move with it. Don't use a lot of force, kind of light. If it moves that could be your cause, and even wiggling it could have reseated a solder trace.
