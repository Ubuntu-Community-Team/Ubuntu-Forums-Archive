### Post by brookswm on 2008-09-24
I think Pidgin comes with Ubuntu, it did with mine, it's pretty easy to set up too, however if you have long friends lists it can get cluttered.  I was using Pidgin for XP before I made the switch over and I like the program.  I have it set up for AIM, MSN and Yahoo.  When you open up Pidgin for the first time it'll prompt you to set up an account, there is a drop down with all of the supported IMs, you pick the IM you want, enter the password, and then save and it's good to go.  Once you have that first one set up  you have to click on one of the drop downs, I think there is one called accounts, and add the other IM services that you use.
