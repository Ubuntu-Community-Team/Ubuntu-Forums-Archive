### Post by macaholic on 2008-05-14
The edubuntu desktop can be installed with ```
sudo apt-get install edubuntu-desktop
```
You should be able to run Xfce with this as well. On that note, the only difference b/w ubuntu/kubuntu/xubuntu is what is called the "desktop environment" Ubuntu is GNOME Kubuntu is KDE, and Xubuntu is XFCE. They can all run most if not all linux apps, even ones designed specifically for any particular environment. Yes xubuntu can run compiz fusion.
Also note, edubuntu-desktop will install ubuntu-desktop which includes GNOME, so make sure you boot into Xfce if that is what you want.
