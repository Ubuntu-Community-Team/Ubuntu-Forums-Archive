### Post by Rubi1200 on 2011-02-09
If you have to use Windows, may I suggest the following:

use a limited account for daily usage

strong passwords

a Windows anti-virus such as Avira, Avast, or Microsoft Security Essentials
(ClamAV is well-known for false positives by the way)

firewall

and, last but not least, common sense.

There is also a great tool (writes to the registry though) from Sysinternals which you can use to limit privileges even further:
[http://technet.microsoft.com/en-us/sysinternals/bb896653](http://technet.microsoft.com/en-us/sysinternals/bb896653)

I believe the setting is somewhere under RunAs (or something similar)
