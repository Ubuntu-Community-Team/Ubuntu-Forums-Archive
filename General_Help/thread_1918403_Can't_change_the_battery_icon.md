### Post by CatKiller on 2012-02-01
You really don't need to go changing ownership of files. Pretty much ever.

Make or choose an icon in your new icon theme's directory structure with the same name as the icon that's being picked up at the moment and reload the theme.

There's information about how it all works on FreeDesktop.org, Gnome.org and other places if you're interested.
