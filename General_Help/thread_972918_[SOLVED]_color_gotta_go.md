### Post by slinkey1981 on 2008-11-06
Goto:

System>Preferences>Advanced Desktop Effects Settings

If you don't have it, try this 
```
sudo aptitude install compizconfig-settings-manager
```

Then (in Advanced Desktop Effects Settings) click on "Desktop Cube" 

The second tab (should) say "Appearance" and  there is an option for "Cube Color".

I   *think*  that that is the setting that you are looking for.

If not, try enabling the "Cube Caps" plugin. and (if you want) removing the images that it lists by clicking on them, then the trash can icon, and changing the "Cube Top Color" and "Cube Bottom Color"

I do have one question....  are you perhaps using Mint? Because my cube always defaults to the Human Orange color.
