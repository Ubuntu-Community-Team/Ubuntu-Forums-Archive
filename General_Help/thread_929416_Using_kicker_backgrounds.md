### Post by Oplix on 2008-09-25
I downloaded some kicker backgrounds from KDE-look, but I can't figure out how to install them or use them?

---

### Post by Denestria on 2008-09-25
The system keeps them in /usr/share/apps/kicker/wallpapers/ but the dialog to select a background for the kicker allows you pick an image from where ever you want so if you want to make a directory in your home folder for them you can like /home/username/.kde/apps/kicker/wallpapers/

---

### Post by Oplix on 2008-09-25
But where do I change the kicker background from? I can't find this option in the appearance settings.

---

### Post by Denestria on 2008-09-25
Just unpack whatever archive you downloaded to the directory you want and then right click on the taskbar > Configure Panel > Appearance > check Enable Background Image > click on the button with the little white thing on it and a dialog to pick the image will come up.
