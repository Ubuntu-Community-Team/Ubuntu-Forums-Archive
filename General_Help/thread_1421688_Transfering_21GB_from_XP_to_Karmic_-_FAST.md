### Post by TitanusEramius on 2010-03-04
Well, you say you are using a wireless network for the copy? Those are not very fast, they can only copy ~< 1MB/s or something like that.

Can you connect the two machines to the router by cable? If so you can probably get ~10 MB/s instead. Another option would be a external USB harddisk of some kind, if it's USB2 you can get up to 50 MB/s depending of your system, but off course you have to copy the data twice if you use an ext-hdd.
