Could something be wrong with your graphics driver? It looks to me the CPU is doing much work which is intended to be done by the GPU.

Please do the following things and report back here:

1) Open a terminal:
```
Applications > Accessories > Terminal
```
and type
```
lspci | grep VGA
```

2) Copy the contents of the file /etc/X11/xorg.conf
In Terminal type:
```
more /etc/X11/xorg.conf
```
Select the text with your left mouse button and when all is black type:
```
ctrl-shift-c
```
Here in this page you type
```
ctrl-v
```

ctrl-shift-c = hold the ctrl and the shift key while pressing the letter c
ctrl-v = hold the ctrl key while pressing the letter v
