### Post by david sousa on 2008-08-25

2 - You can get new themes [here]("http://gnome-look.org/")

3 - Go at System -> preferences -> Advanced Desktop Effects Settings -> Window decorator and at the cammand box you must change "compiz decorator" to "emerald --replace"

4 - Now enter at Emerald Theme manager at system -> preferences and install the themes wich you have downloaded

5 - enjoy it ;)

### Post by twisted_steel on 2008-08-25
I looked some more and haven't been able to find new themes other than the defaults.  I did, however, find out how you should be able to change the color of the classic theme.  Open up the System->Preferences->Appearance program and click on Customize.  On the colors tab, change the Background color for Selected Items.  The Classic theme uses a darker version of whatever you pick on there.

As far as other themes, you might try asking in the Gnome-Do IRC channel.  I searched for "gnome-do" on my computer and didn't find any theme-related files.
