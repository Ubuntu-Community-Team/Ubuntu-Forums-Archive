Hi there, Ubuntu will not run on 128mb ram unfortunately (the Live CD you are currently downloading requires 384mb, and the Alternate CD requires 256mb), but you should be able to install Xubuntu if you use the Xubuntu Alternate CD. If you want a speedier experience, I'd recommend something even lighter such as bare-bones Ubuntu with a lightweight windows manager (openbox, fluxbox, icewm, etc), Fluxbuntu, Puppy Linux, DSL, etc.

Another thing to check is whether or not your system's bios allows you to boot from the external usb cd-rom. If not, you will need a work-around since you will not be able to install using the standard CD method.

Good luck!
