### Post by Mark Phelps on 2013-06-20
You're trying to build drivers from source, apparently.  

The readme text file generally contains step-by-step instructions for doing this, and doing a "./configure" is often the first step.

Also, to build stuff from source, you need to have already installed build-essential using "sudo apt-get install build-essential"
