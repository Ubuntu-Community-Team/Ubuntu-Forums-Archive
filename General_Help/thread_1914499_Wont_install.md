### Post by ubudog on 2012-01-25
Hmm, try using an "alternate" install disc.  Download the iso here:
[http://releases.ubuntu.com/11.10/ubuntu-11.10-alternate-i386.iso.torrent](http://releases.ubuntu.com/11.10/ubuntu-11.10-alternate-i386.iso.torrent)
(you need a bittorrent client) 

Put that iso onto a flash drive or CD (with a tool like Windows disc image burner, or, in Ubuntu, Startup Disk Creator).  Boot off of that when it is done, and follow the instructions like normal (you will not get a GUI).

I hope this helps,
ubudog
