### Post by andymorton on 2011-01-24
I don't know much about Arch but here goes  - Unlike with Ubuntu, Fedora, Mint etc, once you've installed Arch you are presented with a command line rather than a GUI. You choose which packages to install rather than a number of packages being pre-installed with Ubuntu, i.e. you need to configure wireless, sound, etc youself. You also install a GUI yourself. I suppose they're the main differeces. And Arch uses it's own package managment system, called Pacman.

Whereas Ubuntu is geared towards users of all levels of experience Arch is more for the intermediate to advanced user. 

It would be a good idea to read through the wiki to find out all the steps you need to take to get in up and running. [https://wiki.archlinux.org/index.php/Main_Page](https://wiki.archlinux.org/index.php/Main_Page)
