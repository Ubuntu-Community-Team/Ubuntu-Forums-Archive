Not to beat a dead horse here, but

**[COLOR="Blue"]H: ATAPI DVD A DH20A4H[/COLOR]**

is **not** a **_file format_**.

A file format would be a file name that ended in

**[COLOR="Red"].iso - .jpg - .txt - .sys - .exe[/COLOR]**

This

H: ATAPI DVD A DH20A4H

denotes your address to the DVD burner, NOT a file format.

**H:** is the drive letter in Windows

**ATAPI DVD** is the brand name of the DVD burner manufacturer

**A DH20A4H** is the serial number if the DVD burner.

I'm just trying to help you, but IF you know what you are doing, please learn to use the correct words when you are describing what you are trying to do.
