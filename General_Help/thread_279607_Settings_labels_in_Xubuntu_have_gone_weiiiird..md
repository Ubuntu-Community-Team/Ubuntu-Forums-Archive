### Post by IMS3 on 2006-11-02
Just worked out how to fix the problem.

Simply go to System -> Language Support

Select 'English (United States of America)' as the default language.

I received the Settings label problems when I changed the default language to 'English (United Kingdom)'.

Btw, tested on 2 Xubuntu machines and the problem is sorted.

Could someone plz file a bug report to Xubuntu or detail instructions on how to.

Bug is probably something to do with Xfce 4.4 BETA 2.
