### Post by Tristan Schmelcher on 2012-06-18
I bought Bastion for Ubuntu (through HIB) and I am having trouble playing it because the keyboard and mouse input lags a lot. i.e., there's often a delay between pressing a key and it taking effect in the game--about half a second. Sometimes if I quickly tap a key or mouse button the game doesn't register it at all, or it registers that I pressed it but doesn't register that I released it. And the mouse pointer is very jerky.

It happens on both my gaming machines, both running Ubuntu 12.04 64-bit. One is a System76 Serval Pro laptop, the other a Leopard Extreme desktop. I'm pretty sure both should be good enough to run the game. The really strange thing is that the graphics are silky smooth the whole time, so it doesn't seem to be a performance issue. Also, Compiz vs. no Compiz doesn't seem to make a difference, nor does resolution.

Has anyone else had this problem and/or found a solution?
