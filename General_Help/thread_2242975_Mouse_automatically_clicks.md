### Post by boxcorner on 2014-09-05
Hello, could someone help me pleased

Recently my mouse started behaving very strangely. If I leave the pointer hovering over anything that can be clicked, then after a few seconds the thing is actioned as if I had clicked on it, but without having physically clicked the mouse. In other words the mouse clicks automatically. It's quite alarming as several time I've inadvertently launched things without intending to! 
 
What I'm wondering is whether I've inadvertently started some universal access application. If so then I don't what it is.

Thanks in advance for any help.

### Post by boxcorner on 2014-09-05
Ah, I Googled 'how to stop mouse auto-clicking in ubuntu'
found this >Disabling Auto/Dwell Click doesn't work in Ubuntu 12.04
2nd answer > Go to: System Settings > Universal Access.  Select Tab: Pointing and Clicking.  Turn Off: Hover Click
This worked for me ... problem solved :~)
