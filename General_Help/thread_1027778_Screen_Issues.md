### Post by 2hot6ft2 on 2009-01-01
You need to connect it to the internet. Then click on the menu then System>Administration>Hardware Drivers 
and select the graphics driver it recommends and apply. It will download the driver and install it. It will require a reboot after it finishes.

After the reboot you should go back to System>Administration>Hardware Drivers and make sure the driver is active.

If it is then go to screen resolution and you should have more choices.
