### Post by Ms. Daisy on 2012-02-27
I can't really tell from what you describe what your computer specs are.  Have a look at this to see if it can handle Ubuntu.
[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

If you don't know your system specs, open a terminal using ctrl + alt + t and type
```
lshw
```
compare the output with the link above.

There are lighter distros if your computer can't run Ubuntu.
