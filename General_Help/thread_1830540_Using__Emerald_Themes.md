How do I use Emerald themes instead of Metacity in Natty? Is there any way to convert .emerald to .theme, or to replace Metacity with Emerald?

---

### Post by Frogs Hair on 2011-08-21
First , you need to install the Emerald Theme Manager and then add the PPA at the first link because it includes a bug fix for 11.04 .

[http://www.webupd8.org/2011/05/get-emerald-to-work-in-ubuntu-1104.html#more](http://www.webupd8.org/2011/05/get-emerald-to-work-in-ubuntu-1104.html#more)


Second , you will need to install the Compiz CCSM and the Compiz fusion icon from the software center . The icon starts and stops Compiz and Emerald . 

See this link for the Emerald Wiki .[http://wiki.compiz.org/EmeraldThemeManager](http://wiki.compiz.org/EmeraldThemeManager)
