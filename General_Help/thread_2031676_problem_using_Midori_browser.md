### Post by Toz on 2012-07-22
See: [https://bugs.launchpad.net/ubuntu/+source/midori/+bug/950285]("https://bugs.launchpad.net/ubuntu/+source/midori/+bug/950285").

On install, the default homepage seems to be set to /usr/share/ubuntu-artwork/home/index.html but that file doesn't exist. Simply go to preferences and change the homepage to whatever you want it to be. Using the speed dial setting gives it that more recent look.
