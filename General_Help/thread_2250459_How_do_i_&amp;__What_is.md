### Post by oldos2er on 2014-10-28
A terminal is a program that provides a command line interface (CLI) to the system. There are many different terminal programs available to use; the default one in Ubuntu is gnome-terminal. See [https://help.ubuntu.com/community/UsingTheTerminal](https://help.ubuntu.com/community/UsingTheTerminal) for more info.
