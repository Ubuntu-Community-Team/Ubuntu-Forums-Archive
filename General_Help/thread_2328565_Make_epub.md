### Post by oldos2er on 2016-06-24
Use libreoffice writer (for example) to enter your text, save it as a .doc file which you can then import into calibre, edit your metadata, etc.

I just found [http://writer2epub.it/en/](http://writer2epub.it/en/) , a plugin for libreoffice that claims to be able to create an epub file too, so you might not even need calibre. I have no experience with it though.
