### Post by Samy_Merchi on 2005-10-20
Whenever I double-click on a .txt file, I get a requester:

Do you want to run "bronzeage.txt", or display its contents?

Options: Run in Terminal, Display, Cancel (default), Run

How do I make it so that the requester will not appear, and instead the double-click will automatically open the file in the text editor without intermediate stages?

I have checked the file Properties/Open With, and it's set to open with Text Editor.

---

### Post by Jens Willem on 2005-10-20
Right-click on file -> Properties -> Rights (or maybe it's "Permissions"... I use a translated version of Ubuntu) -> uncheck the "Execute" box.

### Post by Samy_Merchi on 2005-10-20
