### Post by spacesamurai on 2009-05-15
Yes, if you boot with the older kernel, there will not be support for the ext4 by default. Ext4 has been introduced from Ubuntu 9.04, so I would not except it to work. 
So it seems there are two optons.

1. If you use the older kernel, you will need to alter you fstab to so that it supports ext3 format.

2. If you boot with the kerenl included with Ubuntu 9.04, try the link that I posted previously.

Hope it works out for you.
