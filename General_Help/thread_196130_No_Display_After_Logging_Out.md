### Post by mlind on 2006-06-13
[QUOTE=ewl1217]I have it attached and compressed as .tar.gz

This forum is so picky about attachments, and it was too long to fit in a post...[/QUOTE]

It seems that /etc/X11/xorg.conf, which is X server configuration, contains wrong
values for your SyncMaster monitor. Refresh rates are probably set wrong, on 
log file there's complaint about 1280x1024 resoulution clock mode being out of range.

You must reconfigure Xserver by doing

```

sudo dpkg-reconfigure xserver-xorg

```

then open /etc/X11/xorg.conf using **nano** editor for example, and remove
all references to wacom devices, those are not needed if you haven't got tablet
pc. 

Then post your /etc/X11/xorg.conf here. You can post it inside [ code ] [/ code ] tags (without whitespaces).
