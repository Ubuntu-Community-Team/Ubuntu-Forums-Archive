### Post by Dennis N on 2014-06-29
Updating flash player plugin is and should done by the package flashplugin-installer and is done whenever you see it updated. There is nothing you need to do if flashplugin-installer is installed on your system. There are no new versions beyond 11.2 r 202 - except for some minor (security?) upgrades. Adobe is not providing newer versions for Linux.

Adobe Statement:

> NOTE: Adobe Flash Player 11.2 will be the last version to target Linux as a supported platform. Adobe will continue to provide security backports to Flash Player 11.2 for Linux.
