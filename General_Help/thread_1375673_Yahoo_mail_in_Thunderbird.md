### Post by balaknair on 2010-01-08
You'll need the webmail extension for thunderbird
[http://webmail.mozdev.org/](http://webmail.mozdev.org/)

Instructions:

Save(Right-click> save link as) the following two files on your computer, and then install them in Thunderbird with the Add-on Manager in T-bird(Tools> Add-ons--> Install--> select the .xpi files)
[http://downloads.mozdev.org/webmail/web-mail-1-3-5.xpi](http://downloads.mozdev.org/webmail/web-mail-1-3-5.xpi)
[http://downloads.mozdev.org/webmail/yahoo-1-4-6.xpi](http://downloads.mozdev.org/webmail/yahoo-1-4-6.xpi)

Restart T-bird, and now you need to configure the account. To do that I suggest you check this page, as it has screenshots and is very easy to follow. If you still have doubts, post back here and I'll try to help.

[http://webmail.mozdev.org/setup.html](http://webmail.mozdev.org/setup.html)


All the best.
