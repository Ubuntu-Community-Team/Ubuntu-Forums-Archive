### Post by dziamid on 2010-10-07
It seems that you cannot configure Midnight Commander to use TAB key for auto completion, however desperate your desire is. So, I wonder if there is any alternative file manager that makes use of TAB key for autocompletion. 
I also noticed that i quite often use TAB-TAB (double tab) to list possible options. Wouldn't it be great to have file manager that follows your path as you type?
For example, you type 
"/home/user" then you either push TAB or slash and manager lists all the contents of your home directory, so you can continue traversing your path easily..

Is there anything like that already? :confused: Thank you.
