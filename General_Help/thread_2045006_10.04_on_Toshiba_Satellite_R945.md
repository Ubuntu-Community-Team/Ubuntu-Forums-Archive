### Post by zeug on 2012-08-21
Got the WLAN to work, after installing [this file]("http://linuxwireless.org/download/compat-wireless-2.6/compat-wireless-2.6.tar.bz2"), with
```
make
sudo make install
sudo make unload
sudo modprobe iwlwifi
```
This was encouraged by seeing the the LiveCD for Xubuntu 12.04 used iwlwifi instead of iwlagn as the driver.

Now, all the remains is figuring out the graphics and touchpad.  The LiveCD has working graphics and also uses the i915 driver, so I am not sure why it doesn't work on 10.04 with the Oreinic backports.  Any suggestions on how to fix either of these would be appreciated.

Thanks!
