### Post by lildigiman on 2009-08-07
If this is a real theme for Ubuntu it most likely would have been created as a tar.gz, since it is not, I would suggest extracting the .zip somewhere where you can access it easily.

The quick and dirty way to open archive manager now is to right click on any folder, and select "open with other application" and select Archive Manager. It will probably give you some error, but thats ok, because you just wanted to get archive manager opened.

From there, go to Archive -> New. Create it somewhere easy to access (don't forget to end the name with tar.gz), and then copy the unzipped files from the theme over to your new tar.gz file.

Open up the theme selector and choose the tar.gz you just made!

I really hope this helps.

Lildigiman already answered the .zip question, and for the minimize question:

On the bottom toolbar, in the left hand corner, there's a sort of brown square that you click on, and it minimizes all windows. Just like in Windows there's that button to show desktop. If you don't have it, right click your panel (taskbar), select "Add to Panel", and you'll get a list of stuff you can add to your taskbar. There's one that says "Show Desktop" and below it "Hide application windows and show the desktop". Just click it and everything will be minimized.

Hope this helps. :D

### Post by yeeeev on 2009-08-07
Regarding the "minimize all windows" keyboard shortcut.
If you have CompizConfig Settings Manager (CCSM) installed, you can go to "general compiz options"->"key bindings" tab
There's a "show desktop" option there with a little keyboard next to it. You can enable it to whatever keyboard shortcut you want ("Super+d" will achieve the shortcut you mentioned)

Sorry, but I don't know a command line way to configure it. Whoever said GUI is easier???

Good luck

P.S. to install CCSM, run:
sudo aptitude install compizconfig-settings-manager
you can then run it from the System->Preferences menu.
