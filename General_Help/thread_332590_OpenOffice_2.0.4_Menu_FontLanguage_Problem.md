### Post by ajgreeny on 2007-01-06
If you can at least see the menus as separate words go to the last entry on the menu 7, which is Tool>Options.  Now go to the forth entry in the first group in that and that will have a tick box near to the top, which says, if you can read it, Use system font for user interface.  In my case that is ticked to use my chosen system font (arial 11).

I hope that may help, but if not, then I'm completely stumped.
