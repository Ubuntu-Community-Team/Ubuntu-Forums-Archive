### Post by sekinto on 2008-04-14
> We use it for e-mailing a lot.

You should be able to use any in-browser client, and if you want an external e-mail client Evolution and Thunderbird work vary well.

> And for downloading photos from an HP Photosmart  M425 Digital Camera.

I wouldn't think you would have problems, should be able to mount your camera as a flash device or similar.

> We listen to music online, and sometimes download music as well

Can listen to music online if it is streamed though a compatible technology like Flash. You can easily play downloaded content, mp3, ogg, flac, wav, you name it.

> What do you guys believe about it? Can the stream of music be 'hacked' into?

I highly, HIGHLY, doubt that. To exploit code in an audio streaming client the person hacking would have to know exactly what OS you are using and what streaming client you are using. Besides that it wouldn't work because you don't run streaming programs as root, or at least I hope you don't. Plus the concept seems kind unbelievable in the first place, since the streamer would be reading the audio as audio and not as a executable.

> We also burn CD's a lot, and is there an easy to understand CD burning program on Ubuntu.

Brasero is really easy to understand. K3b has a lot of features.

> I found the Mac OS X (Tiger 10.4) to be confusing, as just how to burn a CD.

Might Ubuntu be easier?

Burn the .iso as a DISC IMAGE.
Set your computers BIOS to boot from optical media first.
Insert the CD and start it up, click the install icon on the desktop and fallow the instructions.

> And lastly, I goof around at You Tube sometimes, is Ubuntu easy to watch You Tube like sites on?

It works just fine, although the flash plugin is a little buggy at the moment, still works though.

> I know that these are mostly really simple questions, but I'm really NEW ay Ubuntu, and I'm wondering if it can do ALL of these things.

Everyone was "new" at some point. Glad you are considering venturing into the Linux meadows, the grass is greener here... although if you want the grass to be a different color you can always edit some configuration file.
