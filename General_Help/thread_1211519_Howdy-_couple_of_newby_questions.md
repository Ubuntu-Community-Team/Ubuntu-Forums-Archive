### Post by Arand on 2009-07-12
--
.../2 versions of the kernel, one is 13, one is 11/...
Whenever a new version of the kernel is deemed better/stable enough it will make it in as an update, however, if you install -13 the -11 will still remain, since some times the newer version will break things for a number of users, hence it keeps the old version as a fallback, if the new causes problems.
If you've tested the new one and you a comfortable with it not causing any problems, you can if you want remove the old one, ("computer janitor" can do that I think.., or use synaptic {package name for kernel is linux-image-xxxx-generic})
--
.../force programs to start on a particular desktop?/...
The Compiz window effects/manager can do that, you'll need to install the compizconfig-settings-manager for that, and look under the options for "window rules" if I remember correctly.

Someone else could answer "evolution" and "phone"?

- Arand
