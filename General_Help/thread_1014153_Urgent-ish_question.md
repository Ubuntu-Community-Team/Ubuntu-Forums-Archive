### Post by HermanAB on 2008-12-17
Look at a program called 'Motion'.  It will take snapshots whenever it detects significant motion and store the snap on an Apache web server.  You can then log into the web server from anywhere using a browser and see what is going on.

Of course it is all easier said than done, but a few hours of playing around should get you going.
