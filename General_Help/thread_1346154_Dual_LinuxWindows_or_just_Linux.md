### Post by jrusso2 on 2009-12-04
You have a couple of options you could try WINE and see if those applications run, or install virtualbox and Windows as guest and then you can install them as you would in Windows because they are in Windows but running in a Virtual machine.

### Post by Snow986 on 2009-12-04
Many use VMware or another virtualization software to run windows virtually in linux.  I find it suits most all of my windows needs (which are very few these days :) ). As to which distro is best, that is up to you, but if you like ubuntu and kubuntu, you can just install one or the other (usually better to go with ubuntu) and then install KDE or gnome along side.  Then you can choose at boot whether to use gnome or KDE.
