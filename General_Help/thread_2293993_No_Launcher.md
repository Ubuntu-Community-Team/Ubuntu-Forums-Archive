### Post by Bucky Ball on 2015-09-09
*Thread moved to **General Help**.*

***New to Ubuntu*** is as the name suggests.

Open a terminal and:

```

sudo apt-get autoremove
sudo apt-get update
sudo apt-get upgrade
sudo apt-get dist-upgrade
```	

This will not upgrade your machine to a newer release. Please post any and all errors. If none, reboot.
