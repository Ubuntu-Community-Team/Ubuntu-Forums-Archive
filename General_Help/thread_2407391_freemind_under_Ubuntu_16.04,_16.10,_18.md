### Post by QIII on 2018-12-03
Have you tried going to a terminal emulator (the command line) and trying

```
sudo apt install freemind
``` to install it from Canonical's repo?

Alternatively, you can install it as a snap from [here]("https://snapcraft.io/freemind").

I'm not yet sold on snaps, so my first choice would be via the repo.

As an aside:  16.10 is no longer supported, so you are not likely to have any success installing to it.  I'd get rid of 16.10, as it is a security hazard.
