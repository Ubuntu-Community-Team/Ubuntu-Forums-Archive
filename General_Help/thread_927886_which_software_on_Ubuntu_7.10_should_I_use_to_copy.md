### Post by hyper_ch on 2008-09-23
k3b, gnomebaker, brassero... there are a few burning apps out there.

Just make sure that the downloaded ISO is not corrupted. if you use bittorrent for download then it's good. If not, then make a md5check (k3b makes that automatically, don't know about the others) and compare it with the md5 given here:

[http://mirror.globo.com/ubuntu/releases/7.10/MD5SUMS](http://mirror.globo.com/ubuntu/releases/7.10/MD5SUMS) (this is for 7.10 only)

Furthermore, try to burn the CD as slow as possible. Higher burn speeds may lead to corruption and/or older drives may not be able to read it.
