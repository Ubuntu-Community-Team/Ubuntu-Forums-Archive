### Post by Elfy on 2008-06-27
Edgy is an old version the newest version is Hardy, I usually get it with a torrent then it gets checked as it goes. I used this page to get the hardy torrent [http://releases.ubuntu.com/8.04/](http://releases.ubuntu.com/8.04/)

This is the main install page - [https://help.ubuntu.com/community/Installation](https://help.ubuntu.com/community/Installation)

This is psychocats pages which are useful as well - [http://www.psychocats.net/ubuntu/index.php](http://www.psychocats.net/ubuntu/index.php) which I found very helpful last year.

This is a newer page which gets a lot of mentions - [http://apcmag.com/how_to_dual_boot_windows_xp_and_linux_xp_installed_first.htm](http://apcmag.com/how_to_dual_boot_windows_xp_and_linux_xp_installed_first.htm)

Beryl is gone and is replaced by compix fusion, this is installed by default with Hardy.

There is plenty of information about, but the best thing is to use the Absolute Beginners forum - plenty of friendly people about to help you in your hour of need.

Good luck
