### Post by oldos2er on 2013-09-10
[https://help.ubuntu.com/community/Fonts]("https://help.ubuntu.com/community/Fonts") shows how to install fonts system-wide. If you only need fonts for your user, create a folder named *.fonts* in your home folder, move or copy the fonts you want there, then run ```
sudo fc-cache -fv
```
