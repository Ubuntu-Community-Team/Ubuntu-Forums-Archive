### Post by ajgreeny on 2009-12-30
Probably just change the theme that is currently running on the machine to one with light coloured window backgrounds.

### Post by ac0la on 2010-01-05
I thought I had tried that, but you were right--I found a theme with a lighter top menu bar, and this indeed effected the background color of the Add Bookmarks interface window. Then I dug deeper and found that the option I needed to change was "controls". It's not obvious from the Appearance interace that this will change the top menu, etc, but it does.
