### Post by Mark Phelps on 2014-11-03
> **Bob_Younkin said:**
> I have 10 laptops of various makes, all over 8 years old and most with 1-1.5 gb of RAM. The program we use to prepare taxes is now "on the cloud" so I want convert these laptops to internet appliances running Chrome browser (software doesn't support Firefox).
Problem you're likely to run into is drivers for older PCs.  What I suggest is that you create a LiveDVD of the Linux distro you want to run, boot each laptop from that, and see what driver problems you encounter on each.  With older machines, it's likely that they all will work OK as it generally newer hardware (like Integrated Graphics) that tends to be problematic.

> I need a distro with a lean footprint...
As mentioned, Lubuntu, of all the Ubuntu versions, is likely to demand the least resources.

>  and very user friendly. 

Unfortunately, folks used to using Windows PCs are going to enounter a learning curve with any Linux distro -- because, simply put, Linux is not a drop-in replacement for Windows. Whether or not Lubuntu is more or less user-friendly than Ubuntu is largely a matter of personal preferences.. so each person is likely to think differently of it.

You are most likely going to have to walk each person through using whichever distro you install, as the menus and selections are going to be different from what they are used to using in Windows.
