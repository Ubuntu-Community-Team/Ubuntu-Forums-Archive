### Post by Tipped OuT on 2009-05-14
A simple google seacrh would have solved your problem.

Here: [http://home.arcor.de/salnet/segoe_ui.zip](http://home.arcor.de/salnet/segoe_ui.zip)

> Unzip the downloaded archive and place the two .ttf files in your .fonts directory.

You can find it in your user folder. /home/$USERNAME/.fonts/

For example: /home/ruchi/.fonts

If the directory .fonts does not exist, create it.
[B]
Make sure you set hidden folders and files to be visible.[/B]
