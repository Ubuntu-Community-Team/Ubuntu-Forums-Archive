### Post by buzzingrobot on 2014-08-25
Fedora defaults to Gnome Shell, not Unity. Gnome Shell's default theme is Adwaita, which uses blue, I believe, only as a selection color. Fedora's default background is usually blue, however. 

Plenty of alternative themes for Unity are around.  You'll need an addon tool to enable them after installation. Unity Tweak Tool (in the repos) does the job well.

### Post by mcduck on 2014-08-25
> **Jahidul_Hamid said:**
> I don't think unity will allow such customization...:(

Why wouldn't it?

In fact it already does. Just go to [gnome-look.org]("http://gnome-look.org/") or any similar web site, download any GTK3 theme and icon theme you happen to like, and you're good to go. Also there are several different themes included in Ubuntu's repositories.

The default Appearance dialogue only seems to pick themes that are installed system-wide (in */usr/share/themes* for GTK themes and */usr/share/icons* for icon themes) but if you prefer installing things into your own user account only (*~/.themes* and *~/.icons*) I recommend getting the [Unity Tweak Tool]("apt:unity-tweak-tool") from Software Center (it also allows you to select the GTK theme and icon theme independently from each other, and set fonts, cursor themes and various other settings)
