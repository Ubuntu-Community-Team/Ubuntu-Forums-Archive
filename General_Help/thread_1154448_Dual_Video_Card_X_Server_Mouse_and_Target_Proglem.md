[SIZE="4"]Successes:[/SIZE]
1) Compiz works.
2) No startup problems.
3) Even with both x servers running, using twinview treats each of the monitors (of one video card) separately, thus allowing maximization.

[SIZE="4"]How I did it[/SIZE]
Having Ubuntu enable restricted drivers reverts my system back to limited graphics mode, don't do it.

[SIZE="3"]Instead do this[/SIZE]

Download the Nvidia drivers from the command line using 
```
sudo apt-get install nvidia-glx-180
```
Back up your xorg file
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```
Press: CTRL-ALT-F1 to go to console login.
Create a new Xorg file with the Nvidia program
```
sudo nvidia-xconfig
```
restart gnome
```
sudo /etc/init.d/gdm restart
```
