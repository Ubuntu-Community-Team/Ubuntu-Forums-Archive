If I want to upgrade to the next version, how to do it keeping my present set up of programs & settings intact (including Compiz), so that I need not install Individually all over again?

---

### Post by 3rdalbum on 2011-07-16
Either do the update from Update Manager or run the installer from the Ubuntu 11.04 CD and select "Upgrade" from the menu. They will preserve your per-user settings because they won't touch your home directory, and they also download new versions of your current programs.
