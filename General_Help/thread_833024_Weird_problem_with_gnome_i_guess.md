Hmm.  Well first, understand that this is not common.  I have installed many different versions of ubuntu on many many different computers, and have never had this trouble.

What it sounds like is that your panel is hanging for some reason.  You mentioned that you go and kill "explorer.exe" in windows and that works.  Well in ubuntu, the process to kill would be gnome-panel.  If you want to bring up the task manager, I think you have to configure a shortcut key for that in System > Preferences > Keyboard Shortcuts.

That might get you back running, but as to why it hangs in the first place, I'm not sure.  Are you using any applets that are out of the ordinary?
