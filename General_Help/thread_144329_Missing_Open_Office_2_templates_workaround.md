### Post by ajgreeny on 2006-03-14
I've done a search on this problem but not found a solution so thought I'd put this info here.

Like a lot of people I've got the problem in Breezy Kubuntu using OOo2, of no templates showing in the dialogue box when I want to open them even though they are there if I look using Konqueror.  What on earth is going on?

I have found a workaround to help out, but the problem does need sorting properly.

In order to use templates from the dialogue box you should make a sub-folder of your documents folder (in my case home/Office_Docs/templates) and put your templates there.  Then when you need to open a new template just navigate to that location on the dialogue box and there they will be, just as they should be in the templates folder but aren't.  The dialogue box seems to open on that sub folder when you open it next time so the system remembers everything OK, and templates you open appear as untitled documents as normal so it works quite well.

Still doesn't sort out why the templates put in the default place won't appear, but at least it means you can continue using the things.

Give it a try.
