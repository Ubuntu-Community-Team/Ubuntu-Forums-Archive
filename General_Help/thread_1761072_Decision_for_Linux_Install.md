### Post by RedRat on 2011-05-17
If you are new to Linux/Ubuntu, I would strongly urge you to install and LTS version, in your case that would be 10.04LTS. Since you seem to be a beginner, best to avoid the various other upgrades, i.e., 10.10, 11.04, and later this year 11.10. Wait on those because these interim versions should be considered "betas" to the next LTS release. Experiment with 10.04 and see if you like it, then when you feel confident then upgrade to the next and greatest. 

As to stability of your laptop, I echo what was said before here, if it is a hardware problem, e.g., failing hard drive, memory chip, etc. installing Ubuntu will not clear up your problem. If it is a virus or some malware, then yes, Ubuntu will give you relief.

If the Live CD doesn't work then there is a low probability the Alternate CD will work better. The installed system is the same in both cases. 

At this point I would recommend one of two approaches:

1. Try Ubuntu 10.04 (the stable Long-Term Support release) and possibly some other distributions from this list: [http://distrowatch.com/dwres.php?resource=major](http://distrowatch.com/dwres.php?resource=major)

2. Figure out what graphics card your computer uses and search for your computer model number and/or your graphics card model number on these forums and the Ubuntu wiki. If there is some kind of incompatibility with your graphics card, then it's a safe bet other users have encountered this problem as well. :)

Also make sure you have verified the CD integrity as I suggested above. Bad burns are all too common. Some users have better luck with a Live USB thumb drive.
