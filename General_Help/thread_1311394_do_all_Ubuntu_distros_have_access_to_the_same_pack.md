### Post by snowpine on 2009-11-02
Hi Eival, Ubuntu releases are numbered according to the date they're released. So 9.10 equals October 2009.

So, if you are still using 8.04, you're using whichever drivers/applications/etc. were stable as of April 2008.

Hope that clears things up for you. :)

### Post by Sef on 2009-11-02
1) Since upgrading software may cause problems, upgrades are not issued once the new version has been released.
