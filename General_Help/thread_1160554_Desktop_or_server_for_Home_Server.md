### Post by lisati on 2009-05-15
To start the ball rolling:
The "Desktop" edition is good for everyday use, and can have packages added later that will let it be used as a a server.
The "server" edition comes with some packages preinstalled that are useful for server functions. The downside is that it doesn't come with a GUI, but this can be easily added.
