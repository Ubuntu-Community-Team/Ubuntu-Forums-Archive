### Post by sirnigel on 2006-03-12
No need to Delete the partition, as you will be using it for the clean install.  

I have not installed nUbuntu, but I assume it intalls like Ubuntu.  Boot from the nUbuntu CD, when it comes time to partition, make sure the same partitions are chosen for /, swap, etc.  If it doesen't choose correctly, you can do it manually.  Choose to format the partitions for nUbuntu but be sure not to format your XP partition.

Choose to install Grub in the same way as you did when installing Ubuntu,(probably on the MBR of the 1st hard drive).
