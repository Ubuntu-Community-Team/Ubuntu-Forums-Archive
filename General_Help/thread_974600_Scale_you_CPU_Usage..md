### Post by sirebral on 2008-11-07
If you have not done this already you might choose to.  I was running the CPU Frequency Scaling Monitor and noticed that if I had not changed it I would stay at 800 MHz nearly all of the time.  When I scaled it to performance the bar turned green and now I am running at 1.87 GHz (really 1.86 I think) all of the time.

If you are experiencing slowness, scale your CPU.  Intrepid seems to not use the CPU as much as I would like it too so I told it to use it all.
