### Post by PocketDog on 2008-10-05
In Evolution go to Edit > Preferences
Mail Accounts > Click 'Add'

From the AOL website:

[quote=AOL]    * Your email address: [email]screenname@aol.com[/email]
      (Be sure to include the @aol.com or suffix)
    * Incoming mail (IMAP) server address: imap.aol.com
    * Outgoing mail (SMTP) server address: smtp.aol.com; port: 587
    * Your user name is your AOL screen name
    * Your password is your AOL password[/quote]
Here's how I just set it up using my old AOL account, and it appears to work fine

Server type IMAP (I've removed my screen name for obvious reasons)
