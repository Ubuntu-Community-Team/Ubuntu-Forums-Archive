### Post by ushimitsudoki on 2008-12-26
Fullscreen? You can try the "Undirect Fullscreen Windows" in general options.

You can also turn off compiz to see if it is actually the problem. This is really easy to do if you want to install *fusion-icon* from the repos. It will put a little icon in the panel and you can right-click on it.

Or, you can do something like: *metacity --replace* and *compiz --replace* to toggle the window managers as needed.
