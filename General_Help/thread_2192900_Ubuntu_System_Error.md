### Post by 3rdalbum on 2013-12-10
What other symptoms do you have, apart from the error message? Sometimes, these kinds of "errors" are fairly normal and don't cause (or indicate) any real problem.

Although I don't use 13.04 or 13.10 it does sound unusual to have two different programs disagreeing on what version you are using. What version did you install? Have you upgraded to a different version of Ubuntu? What does this command show:

```
cat /etc/lsb-release
```

Try running all updates. But once again, it doesn't necessarily indicate a problem.
