### Post by Lars Noodén on 2012-09-24
The same way you would download anything else using [wget](http://manpages.ubuntu.com/manpages/precise/en/man1/wget.1.html).  

```

wget "http://manpages.ubuntu.com/manpages/precise/en/man1/wget.1.html"

```
