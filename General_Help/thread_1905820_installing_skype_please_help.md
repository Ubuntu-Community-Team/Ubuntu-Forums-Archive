### Post by Canime on 2012-01-07
Open your terminal and type skype in. alternatively its in the internet menu of the main menu. If its not there and runs from the terminal you can include it in those menus by right clicking the main menu icon, 'edit menus'. Then navigate to Internet menu and 'add new menu item'. Type in the information about skype, such as Name = Skype, command = skype, comment = internet phone. Add a cool icon to the icon button and browse from what you have or even download the skype icon from here : [http://files.softicons.com/download/application-icons/skype-icons-by-adrian-kenny/png/512/Skype%20Blue.png](http://files.softicons.com/download/application-icons/skype-icons-by-adrian-kenny/png/512/Skype%20Blue.png)

All set.
