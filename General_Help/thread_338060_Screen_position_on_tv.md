### Post by bobpaul on 2007-03-07
In MythTV:
Settings->Setup->Appearance
You can configure the GUI Height/Width and X/Y Offsets to move it relative to the screen edges.

You can also modify the mplayer command called by mythtv. Change ```
mplayer -fs -.....
``` to ```
mplayer -geometry WidthxHeight+X+Y -.....
``` where all values are, again, in pixels.

For MythTV related questions you'll probably get an answer quicker on the MythTV Forums at MythTalk.com
