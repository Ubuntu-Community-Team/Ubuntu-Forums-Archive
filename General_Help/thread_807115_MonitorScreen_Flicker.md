### Post by kdwillia on 2008-05-25
Is this a dual boot system or a single install of Ubuntu?

You need to test the "hardware" by loading up a different O.S.  A LiveCD of another Distro would be a good option.   If you still get the flicker with another O.S. then it points pretty heavily to a bad piece of hardware.   Also make sure that you are not running through a switchbox or someother piece of hardware such as that.   Take the system down to the simplest setup possible to test items such as this.

Hope this gives you some direction.
