### Post by sanderd17 on 2011-03-20
64 bit is interesting if you have more than 3.5 GB ram, that's because a 32 bit OS can only access about 3.5 GB

2^32 bytes= 4294967296 bytes = 4 GiB

and you have to substract about 500 MB for adresses the system needs internally. You could install the PAE kernel module to get more virtual addresses, but it's still not as eficient as a 64-bit.

So for your case, 32-bit or 64-bit won't make any differance.
