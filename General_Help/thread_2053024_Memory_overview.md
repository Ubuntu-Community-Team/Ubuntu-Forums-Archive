### Post by TheFu on 2012-09-04
[http://ubuntuforums.org/showthread.php?t=1243323](http://ubuntuforums.org/showthread.php?t=1243323) seems to be related, though I like **top**, **htop** and **saidar** CLI programs for real-time data.  **System Monitor** is the main tool in that thread.

For historical performance stats, **sysusage**  [http://sysusage.darold.net/](http://sysusage.darold.net/) is an easy to use and install tool. However, it is about system utilization overall, not specific programs.
