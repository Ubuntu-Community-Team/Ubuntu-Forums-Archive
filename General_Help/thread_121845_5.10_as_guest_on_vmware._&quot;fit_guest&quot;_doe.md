### Post by korny on 2007-01-30
Hi - after a kernel upgrade, you need to recompile the vmware tools.
Also, you need the kernel headers - and for some reason, it doesn't seem to complain if you don't have them...
The process is:
[FONT="Courier New"]sudo apt-get install linux-headers-386
sudo vmware-config-tools.pl[/FONT]
and then answer the prompts as they come - the defaults should all be ok.
You also should say "Yes" to the "resize guest window" prompt (I think that's the name, i'm working from memory here)

That seems to work for me, though I haven't worked out yet how to get the mouse to automatically work across windows, nor how to get clipboard sharing...
