### Post by ago on 2008-04-24
Heavy load on the servers is expected in the next few days.

Wubi normally uses the same http servers, so if you let Wubi download the ISO for you, it will probably be very slow.

Using bittorrent the download will be faster and you will not overload the server. Once the final ISO is released, you can download it using one of the following 2 links. 

64bits: [http://releases.ubuntu.com/8.04/ubuntu-8.04-desktop-amd64.iso.torrent](http://releases.ubuntu.com/8.04/ubuntu-8.04-desktop-amd64.iso.torrent)
32bits: [http://releases.ubuntu.com/8.04/ubuntu-8.04-desktop-i386.iso.torrent](http://releases.ubuntu.com/8.04/ubuntu-8.04-desktop-i386.iso.torrent) 

Place the ISO in the same folder where you have Wubi.exe (grab it from wubi-installer.org). Wubi will then use the local ISO.
