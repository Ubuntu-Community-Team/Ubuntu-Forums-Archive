### Post by ad_267 on 2010-02-03
Yeah Kompozer is the best option. If you're keen to learn HTML and CSS don't give up too soon though, there's some good tutorials here:
[http://www.w3schools.com/html/](http://www.w3schools.com/html/)
[http://www.w3schools.com/css/](http://www.w3schools.com/css/)

A lot of people would much rather write HTML by hand rather than use a WYSIWYG (what you see is what you get) editor because the code is cleaner and you get more control. It's also nice to understand what's going on under the hood if you do use something like Kompozer.
