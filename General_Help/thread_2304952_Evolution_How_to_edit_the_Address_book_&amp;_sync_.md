Further:
The address book is different to the contacts list/db.

My address book is listed in a category called 'Personal'....... if I am group emailing, I must click 'Personal' for all my contact emails to appear.

Note: any emails sent, have their address added to the Address Book.

Questions:

1. How to edit the email addresses in the address book?

2. How to view all the entries in the Contact db?
 I can search it, for individuals, but I can't see who is in there.

This brought up the entire contacts list.

However, it is missing numerous entries from the Address book.... in particular the one I wanted to edit.
That person has 3 entries in the Address book, and only one entry in the contact list.

3. How to safely combine the two lists, so that I can add info to all the contacts that are residing in the Address book?

It does seem strange that the prog seems to be keeping two separate lists of contacts.
