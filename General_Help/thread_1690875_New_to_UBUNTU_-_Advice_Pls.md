### Post by qyot27 on 2011-02-19
The mouse pointer may be moving to a separate desktop, or the screen manager thinks you have more than one monitor connected.  There should be a long icon next to the Trash that consists of four boxes side-by-side - right click on that icon, and reduce the number of desktops to 1 (which will make the icon into a single box).  If that doesn't fix it, then I'd see about the monitor configuration.
