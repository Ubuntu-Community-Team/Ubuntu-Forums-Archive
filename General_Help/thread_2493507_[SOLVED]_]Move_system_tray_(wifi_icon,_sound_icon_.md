### Post by Dennis N on 2023-12-17
> **simtcr said:**
> Are you able to help me navigate please, I am new to desktop based linux.

Ok, we can move the entire panel to the bottom. 
First you need to have gnome-shell-extension manager installed. In your terminal, do:
```
sudo apt install gnome-shell-extension-manager
``` 
In case it is already installed, it will say it's already installed. If not yet installed, continue installing.

Next, start it and click the "Browse" button at the top. Enter "dash to panel" in the search box. When you see it in the results, click "install"
After it's installed, click the "Installed" button at the top and find "Dash to Panel"
Click the little gear icon by it to get the configuration options. Using the "Position" button at the top there is an option "Panel Screen Position" to locate the panel at the bottom of the screen. It should move immediately.
