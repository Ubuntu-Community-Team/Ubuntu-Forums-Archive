### Post by goldgin on 2007-02-06
Got it, you have to disable Scroll Acceleration using Microsoft Intellipoint, the software in Windows (yes) that configures the mouse. 

New MS mice probably control scroll acceleration using hardware instead of software, so the driver probably sends the signal for the mouse to accelerate and then the mouse 'remembers' the setting even in linux!
