### Post by NilsE on 2009-05-08
I use Keypass 1.15 on Windows and just updated to Keypassx 4.0 in Ubuntu.  I have a shared partition that I keep the data base on and share it between the 2 with no problems.

By the way, there is a beta version for Windows that is not compatible with any of the available Linux versions of KeyPassX. Atleast I have not figured out how to make the Linux version accept the signature of the Windows beta version.

So anyway, the good news is that the Windows and Linux versions can share the data base but the products do look a little different. All the functionality seems to be there in Linux.
