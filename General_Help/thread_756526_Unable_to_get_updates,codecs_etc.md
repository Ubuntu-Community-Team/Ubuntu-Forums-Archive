### Post by sekinto on 2008-04-15
If you are connected to the internet, do this:

System>Administration>Software Sources
Check all the network sources, and then uncheck the CD source.
If you want you can also check the third party Conical (hope I spelled that right) sources, but you don't need to.
It will ask you to reload the package lists... say yes.
Now everything should work fine.
