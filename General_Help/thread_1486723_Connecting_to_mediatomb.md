### Post by VeeDubb on 2010-05-19
Well, there are a number of apps that can do this.  I assume you're talking about video files.

Boxee media player can do this, although it can be hit and miss with connecting to UPnP servers like media tomb.  Frequently, it's simpler and more efficient to simply set up a network file share of some kind, and mount that on the computer you're trying to stream from that way, than to use a UPnP server.

For instance, I simply right clicked the folder where my movies are stored, and went into sharing options, and set up a very simple samba share that would allow reading with guest access.  Then I was able to browse that media from the network shortcut in nautilus on my netbook.

UPnP servers are great when you're trying to do things like stream to a network capable TV or your playstation, but don't really add much functionality when you're viewing the media from a PC.
