### Post by grahammechanical on 2011-11-09
Have you installed Ubuntu?

You can use the iso file that you download to create a Ubuntu Live CD that will load and run the operating system from the CD drive. You can use this Live CD to install Ubuntu to the hard disk.

When you boot from the Ubuntu Live CD you will be able the select the language that will be used. We do not have different Ubuntus for different languages. We have one Ubuntu for 25 languages. We choose the language we want when we install Ubuntu.

How fast is your Internet connection?

The download of 229KB is a download that stopped for some reason. Did the Internet connection break while it was downloading?

A Ubuntu download file for CD should be between 695MB and 750MB. A DVD download will be between 3GB and 4GB. The extra space allows room for all the different language files and maybe some extra programs.

English is my native language but I have never seen "dnl" in any English dictionaries. It is not an English word. Please do not remove the vowels from English words or create new spellings of English words.

I usually can understand broken English. I do not expect everyone in the world to speak perfect English or to write correct grammatical English. I am not good at these things myself. But if you wish to communicate with others who use English then you must speak or write in a language that everyone can understand.
