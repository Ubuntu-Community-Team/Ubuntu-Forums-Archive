### Post by jimmy the saint on 2009-02-03
> Rhythmbox can import m3u playlists. Just go to File-->Playlists-->Load From File... and navigate to the directory with the playlist you want to load.

You should also be able to navigate to your playlist in a file browser (if you're using Gnome, Nautilus) and just double click an m3u file and Rhythmbox should open it.

that solved it for the individual in that thread.
