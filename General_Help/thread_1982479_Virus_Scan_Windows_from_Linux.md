### Post by computeratin on 2012-05-18
Kaspersky makes a good AV. Not a lot of false positives. It's not cheap. But, in my personal opinion it's one of the best AVs on the market today. When you install it you have the option of creating a *nix "rescue" disk.
 
Although it is currently above my skill level to do so, as I understand it, it *shouldn't* be horribly difficult (since it's *nix) to rebuild the disc so that it supports your network connection. 
 
If you could figure out how to do that and combine it with one of the "repair" type *nix distros that loads in to and runs from RAM then you could set it up so that every time you needed to do a scan it would connect to the net and download the latest AV def updates. 
 
But, you'd have to set it up so that it either did not update the engine or could update the engine w/o reboot.
 
As I understand it this should be possible. And working out something like this is part of a project that I'm building. But, it'll be a while before I get that far.
 
It would be very similar to what was done on the Ultimate Boot CD. But, instead of Clam it would be KAV. And I could never get UBCD to connect to the net. (I'm pretty sure it was a driver issue.)
