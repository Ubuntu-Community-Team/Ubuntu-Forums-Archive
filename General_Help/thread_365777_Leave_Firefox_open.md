### Post by casaschi on 2007-02-20
Use alltray. when you close firefox it will minimize to the icontray instead.
I have a set up where "alltray firefox" is started at login, so immediately ready.

It's not in the repos, but you can install the .deb from [http://alltray.sourceforge.net/downloads.html](http://alltray.sourceforge.net/downloads.html)
