I'm not an Ubuntu expert but I have noticed that it is possible to...
[LIST]
[*]right click on a file

[*]choose the tab [COLOR="Purple"]**Open With**[/COLOR]

### Post by NilsE on 2008-01-04
c0met has the right idea.

In the Nautilus file browser right click on a file that has the extension you are interested in fixing and select Properties and there is a tab for open with.  You can either select one of the listed ones as the default or select add to add one.
