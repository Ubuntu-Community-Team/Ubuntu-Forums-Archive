### Post by Irony on 2008-01-10
Brushes are stored in;

```
/usr/share/gimp/2.0/brushes
```

You can make your own by copying a couple of these brushes into;

```
/home/username/.gimp-2.4/brushes
```

Then open up gimp and go to;

```
/home/username/.gimp-2.4/brushes
```

and open one of the brushes and modify them from there (for example make them bigger or replace the image with say an ubuntu symbol) and them save them.
