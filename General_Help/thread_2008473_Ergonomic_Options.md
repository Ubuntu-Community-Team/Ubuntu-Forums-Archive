### Post by MarjaE on 2012-06-22
Okay, one important fix is to disable the mouse-over scrollbars, bring back the functional ones, and make them wider.

Mouse-over features are an ergonomic nuisance at best, and given the latency issues, an egonomic disaster. I think the mouse-over scrollbars were intended to save space in an  era when scrollwheels are standard, but scrollwheels cause nasty tendon  issues, and mouse-over features only seem to save space, since you have  to avoid the whole area where the damn thing might pop up.

This thread includes the scrollbar fixes that worked for me: [http://ubuntuforums.org/showthread.php?t=1884742](http://ubuntuforums.org/showthread.php?t=1884742) I actually removed the packages. I understand there have been some problems applying the same fixes in 12.04.
