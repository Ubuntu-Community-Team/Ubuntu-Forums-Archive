### Post by Leppie on 2009-12-31
fstab is a file containing some configuration options for mounting your filesystem.

could you copy the following code and paste it in a terminal:
```
cd ~/Desktop \
&& wget 'http://sourceforge.net/projects/bootinfoscript/files/bootinfoscript/0.44/boot_info_script044.sh/download' \
&& chmod 755 boot_info_script044.sh \
&& sudo ./boot_info_script044.sh
```
and post the file RESULTS.txt which will be generated by this script.
