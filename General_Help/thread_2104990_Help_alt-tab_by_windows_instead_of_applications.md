### Post by iScrtAznMan on 2013-01-14
Hi, I'm new here and recently replaced windows with Ubuntu 12.10. 

I was wondering if anyone knows if there is a way to make alt-tab behave more like other platforms where you can change based on open windows instead of applications. I like to have multiple terminals open (I'm currently learning clisp) and find it hard to switch between them. 

I tried using compiz, but when I search for "Static Application Switcher", nothing shows up. Does anyone know if there is a way to simplify the alt-tab menu or if there is another x-windows profile that has less eye-candy?

Edit: I figured it out, I did not install the compiz-plugins. Once I installed the plugins, I was able to disable unity's alt-tab and enable 'static application switcher'.
