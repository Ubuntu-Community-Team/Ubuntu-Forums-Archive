### Post by TheFu on 2022-05-04
If it were me, I'd use an export format from the laptop calendar program - probably export to ICS, then take that file and import it on the other computers.  Another popular format is ical.

With Thunderbird, you can copy the entire profile between systems. You know that Thunderbird has a built-in calendaring system now, right?  It can use local or remote calendars.  I have a 'holiday' calendar for the next 2 yrs, that is an .ics file that I copy to 1 thunderbird system and import into my calendar server (Zimbra), then all the other clients accessing the same Zimbra account calendar have access to it.
Just dropping a file in a specific location doesn't do anything. It is the Import --> Calendar that matters.
