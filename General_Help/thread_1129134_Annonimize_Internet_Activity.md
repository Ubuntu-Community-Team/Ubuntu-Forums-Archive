### Post by lovinglinux on 2009-04-18
Moblock is not an annonimizer tool, is a firewall ip blocker. I doesn't hide your IP, it just prevents unwanted IPs to connect to you.

You could use [TOR](http://www.torproject.org/). You might want the [Torbutton extension](https://addons.mozilla.org/en-US/firefox/addon/2275) to enable and disable TOR easily in Firefox. 

Please make sure you understand the pros and cons of using it, before installing and running TOR. Also, make sure you don't abuse it with high volume traffic, because the Tor network relies on volunteers to donate bandwidth. 

I don't use it, so I can't give instructions on how to install it. [This](https://help.ubuntu.com/community/TOR) might help you tho.
