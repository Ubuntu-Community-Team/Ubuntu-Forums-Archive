### Post by dresnu on 2006-02-14
For karamba to start at kde startup with your favourite themes, open watever themes you want and save the session from Kmenu-> Save Session. That should do it.

Hi! 
I do use Superkaramba and I must say that I like it! The only thing I do is open Superkaramba (KDE->Utilities->Superkaramba) access the "theme" file and just let it in my desktop! When I logout/in the theme that I just loaded still there... but is necessary to leave that “bomb” icon in you taskbar (don't close it) so the theme will be loaded again!
Have you tried to find downloaded files in /home/you/.superkaramba ?
