### Post by jocko on 2008-08-02
> **LowSky said:**

if you need to use he most common applications, java, flash, mp3 playback, etc follow my instructions,
press Alt+F2 and copy and past this command
```

```or try LinuxMint, it's ubuntu with all the extras most need already installed

> **mikedemario17 said:**
> i did it but youtube still  says i dont have flash...

That command needs to be run in a terminal (Program --> Accessories --> Terminal). If you run it in a "run program" dialogue box (Alt+F2), you will not be able to enter your password or answer any questions you get from apt-get, so nothing will happen.
But you can also open up synaptic (System --> Administration --> Synaptic Package Manager) and search for the package "ubuntu-restricted-extras" there.

As for which os to get: If you want linux, I would say stay in Ubuntu and try to learn how it works. And if ubuntu is slower than windows xp, there must be something wrong with your ubuntu installation... I'm sure people here will help you figure out how to fix that, if you just ask the question...
Mint (based on ubuntu), Debian (which ubuntu is based on), SUSE, Mandriva and Fedora are other examples of fairly easy to use distros (in my opinion, that is. I haven't tried that many distros.).
Some (gentoo) are quite difficult to install for a beginner, while others are as easy as Ubuntu (with livecds)...
