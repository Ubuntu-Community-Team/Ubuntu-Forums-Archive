### Post by greybirds on 2007-05-29
You can restore the icons by installing packages. Start Synaptic (System > Administration > Synaptic Package Manager) and enter your password. Then search through the list in the upper right column for these two packages:

gnome-icon-theme
gnome-extra-icons

Click the box beside them and select "Mark for reinstallation", then click the Apply button. It will download and install the default icons

You can also click the Search button and enter "icons"; there's a lot of nice themes that aren't installed by default.
