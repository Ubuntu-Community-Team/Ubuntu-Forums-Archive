### Post by Robot007 on 2010-02-16
Is it possible to have Archive Manager create the specified folder when I enter it into the path field?
Like if I want to put something in "/Main/Sub1/Sub2" But "sub2" doesn't exist, can I just have Archive Manager create it and put the files there?

---

### Post by sgosnell on 2010-02-16
When you click on Extract, the file browser comes up, and in the top right corner of the window is a large button that says "Create Folder".  Click on that, and you can create a folder where you want.
