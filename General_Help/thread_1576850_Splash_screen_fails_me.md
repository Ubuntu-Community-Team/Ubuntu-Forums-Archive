### Post by themusicalduck on 2010-09-17
Are you using version 10.04?

10.04 doesn't use Usplash anymore, it uses Plymouth, so if you want to install a new splash theme it will need to be a Plymouth one.

I don't think you can replace Plymouth with Usplash either and you probably wouldn't want to try. While googling it, I just found this post advising against it - [http://ubuntuforums.org/showthread.php?t=1464173](http://ubuntuforums.org/showthread.php?t=1464173) :)
