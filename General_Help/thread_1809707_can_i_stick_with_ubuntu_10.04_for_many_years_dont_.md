### Post by mcduck on 2011-07-22
No, kernel updates aren't the only ones that matter. Every update matters, especially getting security updates for all the software you are using.

Of course, if you really want to, you can stick to 10.04 even after it's support has ended. It will just be insecure, and you won't be able to get any software from it's repositories any more. Probably a bad idea if you plan on doing it on a machine that's connected to Internet.

If you don't like Unity or Gnome Shell, then you'd better just try finding an alternative desktop environment (or window manager) that you do like and use that. XFCE4 might be something worth a try.

Anyway, it's pointless to blame Ubuntu developers for forcing you to use something else than Gnome2. There really isn't much option here, as Gnome developers aren't supporting Gnome2 any more. So the options for a Gnome-based desktop in future are either Gnome-shell or Unity.

(Also you might want to know that the Gnome-shell will be optionally installable in Ubuntu 11.10, and it's fallback mode looks quite similar to the old Gnome2 interface.)
