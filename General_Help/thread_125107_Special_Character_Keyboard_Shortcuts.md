### Post by BitTorrentBuddha on 2006-02-25
i believe you're looking for something more along the lines of ß ü é etc...
go to System -> Preferences -> Keyboard -> Layout Options and set a compose key, then just hold the compose key and make the special part of the character followed by the letter, (i.e. <compose key>+", u = ü) [here is a complete list of how to do each character](http://webcvs.freedesktop.org/*checkout*/xorg/xc/nls/Compose/en_US.UTF-8)
