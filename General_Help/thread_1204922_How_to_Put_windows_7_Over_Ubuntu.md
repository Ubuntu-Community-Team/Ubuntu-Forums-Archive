### Post by Gh0st83 on 2009-07-05
Simple steps:

1. Insert your Windows 7 DVD and boot from the disk instead of the hard drive (might have to set this in BIOS if it's not already)
2. Format the drive as you please in the wizard (this is where you can effectively remove Ubuntu or any other operating system that is installed on your drives)
3. Allow Windows to install on your selected partition(s)
4. Take out the disk and have at it with your new Windows 7 install

Total process took about 45-50 minutes on my machine when I installed Windows 7 beta release 7077.

I only use Win7 for gaming.  I much prefer Ubuntu to Windows, but to each their own!
