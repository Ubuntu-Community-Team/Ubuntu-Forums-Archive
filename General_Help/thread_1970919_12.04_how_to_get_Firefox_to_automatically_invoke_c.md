### Post by mutex023 on 2012-05-02
> **Dngrsone said:**
> You can do that?

Hmm, not directly I'm afraid.
You need to write a small shell script - 

```

#!/bin/bash

# $1 - for the amz file passed as cmd line argument to the script by firefox
gksudo /usr/bin/clamz "$1"

```Save this as say - "**myclamz"**
Give the file root permissions and **execute permissions**, copy it to **"/usr/bin"**.
Now tell Firefox to open ***.amz** files with **/usr/bin/myclamz**
