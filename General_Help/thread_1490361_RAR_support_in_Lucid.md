### Post by inameiname on 2010-05-24
The default archiver for Ubuntu is simply a right click and compress option in the right click menu of nautilus. Simple enough. If you've installed rar and unrar, it's that easy. And just make sure to change the format that you want compressed to 'rar'. It's set by default on 'tar' I believe.
