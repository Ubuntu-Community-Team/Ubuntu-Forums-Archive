### Post by strabes on 2008-02-18
Try using an alternate install CD. [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

Below the "Start Download" button, there is some text which reads: "Check here if you need the alternate desktop CD. This CD does not include the Live CD, instead it uses a text-based installer." Check the checkbox next to this text and then click "Start Download." This CD is not a live CD and the installer is text-based like in windows, but you shouldn't have any problems.
