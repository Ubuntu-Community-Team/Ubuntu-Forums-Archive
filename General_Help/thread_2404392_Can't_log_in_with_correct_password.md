### Post by Bearly Able on 2018-10-23
I've just installed Xubuntu 18.04 on an old laptop.  The install seems to have gone smoothly, and it boots to the log-in screen, but when I enter my password, it says it's incorrect.  I know it isn't, but the system won't accept it.

I've tried setting a new password using root from the boot menu, as explained here: [https://askubuntu.com/questions/24006/how-do-i-reset-a-lost-administrative-password](https://askubuntu.com/questions/24006/how-do-i-reset-a-lost-administrative-password) but it won't accept the new password, either.

Any suggestions?

### Post by Impavidus on 2018-10-23
Have you tried using the on-screen keyboard? It's possible that the keyboard layout is wrong. Maybe the system assumes you've got a US keyboard while you have a UK one.

### Post by Bearly Able on 2018-10-23
Well, that's bizarre.  

I'd closed down the laptop after numerous failed attempts.  I booted up to try your suggestion, and on autopilot typed my password as usual and hey presto! It worked. :)  I've no idea why.  I'd turned it off after several attempts this afternoon, so it can't have been simply shutting it down which did the trick.

Oh well, thanks for your help - you do seem to have fixed it. :P

(BTW, I set the keyboard layout during installation, so it should be OK.)
