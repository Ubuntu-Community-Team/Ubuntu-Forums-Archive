### Post by Moustaffa on 2010-06-14
Hi,

I've found sort of a solution..but not a good one

Normally, when you minimize it goes to the upper left corner. Open a few applications/locations, then go to Docky settings and in the standard tab, you'll find at the bottom "Manage Windows Without Launcher".

Hit that (once, twice, it doesn't matter, you'll notice that it stays checked) and close the settings window.

If all goes well, you should see that THE ALREADY OPEN APPLICATIONS will minimize to the dock. Note that if you open a new application, it will, again, minimize to the upper corner.
However, if you repeat the steps above, this new application will also minimize to the dock.

Anyone got a better solution?

Deleting this applet fixed it for me, all my applications minimize to docky, right from the start.
