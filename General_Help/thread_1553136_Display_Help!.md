### Post by earthpigg on 2010-08-14
hi,

i have a wonky workaround suggestion.

1 - create a gnome panel: right click on panel -> new panel

2 - make it huge: right click on new panel -> properties -> make the 'size' a rediculously large number of pixels

3 - and set it's orientation to be 'left'.
