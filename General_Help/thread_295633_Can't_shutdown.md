### Post by CatKiller on 2006-11-08
If you can't find out how to turn on the GUI tools to do it, ```
sudo shutdown -h now
``` will shut the computer down for you in a safer manner than just pressing the Big Red Button. ```
sudo shutdown -r now
``` will tell the computer to reboot.
