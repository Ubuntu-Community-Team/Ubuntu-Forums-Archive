### Post by MaxIBoy on 2008-10-16
Google "xubuntu," find the download link for the alternate install (DO NOT CHOOSE THE LIVECD VERSION,) download the file, burn the CD using a special ISO burner (DO NOT JUST WRITE THE ISO FILE TO THE CD WITH WINDOWS' BUILT-IN CD WRITING TOOLS,) boot the computer off the CD, delete the partition with the type "ext3" or "ext2," (DO NOT TOUCH ANY PARTITIONS WITH THE TYPES "NTFS" OR "FAT32,") and use the empty space to create a new partition to install Xubuntu in.
