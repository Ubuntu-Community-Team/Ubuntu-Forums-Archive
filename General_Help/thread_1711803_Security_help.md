### Post by flipper T on 2011-03-21
np neojack

whilst many linux users will understandably couch their answers regarding the virus threat, i think that the consensus is that there is nothing to worry about if you are using linux in isolation. that is not to say that you cannnot download and forward a windows virus with linux. merely that this virus will not effect your linux system.

the virus does remain "dangerous", however, if you move it from your linux system to a windows os, of particular note if you dual boot or have a home network including windows os.

ps linux and all other os offer no protection against other internet internet nasties such as phishing attacks. if you recieve an email from your "bank" asking for account details, no os can protect you from replying.
