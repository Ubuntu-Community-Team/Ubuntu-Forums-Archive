Yes. The older ones have the option "Install Other OS". That feature was removed from the PS3 slim. Of course, the only OS that could run on the PS3's processors (which are not intel/amd/garden-variety ppc) is some kind of Linux.

The recommended Linux for the job is actually Yellow Dog Linux, since PSUbuntu is way outdated.
