### Post by louis_nichols on 2006-05-14
xmms is very popular among Linux hardcore fans. It has a gtk interface, so it goes easy on resources. Beep media player is a hack of xmms to support gtk2 and is made in such a way that it can take winamp 2.x skins. If you REALLY want something easy, you could use mp3blaster, which is a graphical console based player (just joking here :) ). Another, pretty new, player is [Listen,]("http://listengnome.free.fr/"), which is made to behave like amarok but look like gnome. 

I personally use amarok, because of the powerful Media Library and also because the upcoming 1.4 version has support for iRiver ifp series flash based mp3 players, of which I have one.

I don't know about how RAM hungry all these are, though... You may wanna try them
