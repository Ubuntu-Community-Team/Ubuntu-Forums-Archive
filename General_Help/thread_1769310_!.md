### Post by tim73 on 2011-05-28
I removed all the packages causing errors by "sudo apt-get remove package_name_here" (like nvidia-current)

Then postgresql-client-8.4 could not be removed but using this solved the problem

[http://ubuntuforums.org/showthread.php?t=1415362](http://ubuntuforums.org/showthread.php?t=1415362)

Except I used this:

sudo gedit /var/lib/dpkg/info/postgresql-client-8.4.prerm 

I added "exit 0" after first line and then the removal went ok. This might be helpful for others because apt-get cannot install anything if it goes to this error mode.
