### Post by ajgreeny on 2016-05-06
Try the original and best, synaptic package manager.
```
sudo apt-get install synaptic
```
It used to be the default GUI package managerin all the *ubuntu family of OSs until the various software-centres arrived on the scene, and which I have never used.
Synaptic is always the first package I add to any of the OSs that do not have it by default.

As ajgreeny points out, you can use the Synaptic Package Manager. The good news is that it's part of Lubuntu 16.04 by default and so you won't need to install it: [http://packages.ubuntu.com/xenial/lubuntu-desktop](http://packages.ubuntu.com/xenial/lubuntu-desktop)

Your question doesn't tell people what feature you're interested in. There are other Software Centers that could be recommended. You'll need to be aware that a lot of dependencies may be installed just to have this other software center running. Whether that's worth it, is your call.

There is a sort of online app store but it doesn't seem to provide information for 14.10 onwards: [https://apps.ubuntu.com/](https://apps.ubuntu.com/)
