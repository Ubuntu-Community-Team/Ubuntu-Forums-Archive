### Post by dandnsmith on 2011-01-15
It sounds like you introduced the switch after your installs.
I found that at least some monitors connected through some switches cannot return EDID info - whether it's the cables or the switch I don't know.
I was lucky, in that I've only a requirement to switch between two PCs, so could use two different connection possibilities to my monitor (one VGA, one DVI) - needs 2 different switch operations, but works.
I think there has been some moderately recent posting on use of EDID with Ubuntu - try a search for it.
