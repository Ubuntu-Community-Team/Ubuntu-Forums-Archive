### Post by p_quarles on 2008-08-13
> **PsychedelicWonders said:**
> And what exaclty do exhange servers do for me and what are gtk settings?
Exchange is a server protocol published by Microsoft and used for full functionality of Outlook clients. If you used it, you would probably know.

Gtk is the building block of the Gnome desktop suite, along with many other applications. The settings I was referring to are things such as appearances and file selection dialogues. 

Thunderbird uses XUL instead of Gtk -- this is the same framework used by Firefox. 

> This is bad that thunderbird doesnt do these things that evolution does?
Only if you rely on the things that Thunderbird doesn't do.

Really, that's true of all things: the fact that your desktop lamp cannot illuminate a stadium doesn't mean it's "bad," it just means it has a different purpose. 

> Can you run IMAP on evolution?
Yes.

### Post by derrick81787 on 2008-08-13
> **PsychedelicWonders said:**
> yeah but I'm really trying to go with programs that are able to function in Ubuntu, windows and on an iphone to allow my use of one program to seem seamless, regardless what machine im on.

Evolution is Linux-only, so if you want to use the same program on both Windows and Linux, I'd recommend Thunderbird.  I don't know anything about the iphone, though, so I don't know if Thunderbird runs on there or not.

As far as the differences go between Thunderbird and Evolution, Thunderbird does mail (POP3 and IMAP) and only mail.  Evolution does mail (POP3 and IMAP and Exchange) as well as provide a calendar, a way to schedule meetings, and other things, much like Outlook.  Of course, there are extensions that can make Thunderbird do other things.  For example, the Lightning extension gives Thunderbird a calendar.

Basically, Thunderbird does what you want, and Evolution does what you want plus more.  However, Thunderbird is much more cross-platform, if that's what you're looking for.
