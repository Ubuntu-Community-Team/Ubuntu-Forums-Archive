### Post by bluntu on 2006-08-19
I have a folder and it's about 1.9Gig

I archive it (.zip) and notice that the # of files are different. The actual folder has exactly 1850 files.

```

Type: Folder
Contents: 1850 items, totalling 1.9 GB

```

I then double-clicked on the zip to see how many files are in there and it reports: 1831 files (1.9 GB)

What is really going on here?


[[edit]]
Nevermind, just found out that a folder inside a zip is not counted as an item. If you place 5 folders inside folder A, it will reports 5 items. If you zip it you will have 0 items.
