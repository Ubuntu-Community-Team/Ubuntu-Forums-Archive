### Post by rolamento on 2007-12-23
After a lot of search in how to run googleearth with compiz I have found a solution that is not perfect but works.
Maybe it could help you.

-open your /home/USER/.google-earth folder
-inside make and edit a new document with:

export DISPLAY=:0.0
sh googleearth

- Save with any name.
- right mouse click on the new file and properties
- go to Permission tab and make it executable

Thats it.

Now you can open by execute this file or you can point the googleearth icon menu aplications/internet/Google earth to this file. 

Hope it helps you.

Cheers.
