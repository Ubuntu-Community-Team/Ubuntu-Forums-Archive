### Post by talktoari on 2008-06-06
Go to System > Preferences > Keyboard > Layout Options > Alt Win key behaviour > SUper mapped to winkeys (default)

BTW Super key is the "Windows" key in windows.

Also, check the keyboard layout in case SUPER key doesnt work.

Hope this helps.
