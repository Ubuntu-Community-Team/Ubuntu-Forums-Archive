### Post by hetx on 2009-08-10
Ubuntu is much easier to set up. Arch dumps you in CLI and the rest is up to you. It's far from impossible to do and it's a good learning experience, but it still takes much more work than Ubuntu.

I'd say Ubuntu gives you an easier learning curve and still gives you all the power and control. Arch does let you configure everything just the way you want it, but that's only helpful if you know exactly what you want!

ps I use both Ubuntu and Arch
