### Post by Duggeek on 2007-03-13
It's dependent on how the API calls the "mini browser", so it's a coder issue to be sure.

There's a small feature that might help; in the upper-left corner of the mini-browser is what looks like an "edit" icon. 

Clicking that button will convert the "breadcrumb style" folder navigation to something just like Windows address bar. You can copy the path quickly, open another Nautilus window at the same time and locate the file using the standard browser. (with previews)
