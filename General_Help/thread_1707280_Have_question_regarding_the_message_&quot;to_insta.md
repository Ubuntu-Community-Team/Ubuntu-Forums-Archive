### Post by grahammechanical on 2011-03-15
Go to System>Administration>Update Manager and click Settings and then select the Other software tab. You will need to click Add to add another software source to the list.

Linux and Ubuntu are very secure operating systems. You can set Ubuntu to have automatic log in but you are the only person allowed to change the system settings. This is why you are asked at certain times to authenticate the action. Your log in password will work.

Another way to get to Software Sources is to open the Ubuntu software Centre and click edit and you will see Software sources on the bottom of the list.

Regards.
