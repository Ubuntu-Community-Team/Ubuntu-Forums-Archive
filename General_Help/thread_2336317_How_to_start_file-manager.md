### Post by ajgreeny on 2016-09-07
Just search for "files" in Activities and I am sure it should appear.
You can then try a right click on the file you want to delete completely, though I do not know if that is a default option so you may have to enable it in Preferences first.
You can also hold down shift when you delete a file and it will not go to Trash.
