### Post by erginemr on 2008-01-24
If by gtk-gnash, you mean Gnash, the open-source flash movie player, then I believe you can uninstall it with:
```
sudo aptitude purge gnash
```
Then, visiting a Flash enabled web page should bring forward a selection dialog for the installation of either "Adobe Flash" or "Gnash", from where you should be able to install Adoe Flash to put an end your suffer.
