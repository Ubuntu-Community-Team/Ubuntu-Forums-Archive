### Post by plucky on 2010-01-21
> **kportertx said:**
> I updated my installation today and noticed my laptop not running as well afterwards, is there a way to see what was recently updated or last updated on this system so that I can try revert some files to a previous problem to see if the sluggishness clears up?

**System > Administration > Synaptic Package Manager > File > History** and select the date for the last update.

Before that check the Drivers for you Video card are still loaded and working.

Good Luck
