### Post by oldrocker99 on 2013-07-07
I run 13.04, and use MATE (named after the caffeinated herb, and pronounced MAH-tay), which is an active fork of GNOME 2. It has some renamed programs (Nautilus=Caja, Gedit=Pluma, etc) so as to not intefere with GNOME 3-based systems. It's currently v1.6, and is pretty darn stable. 

Here is how to install it:

[http://www.webupd8.org/2013/04/mate-16-released-install-it-in-ubuntu.html](http://www.webupd8.org/2013/04/mate-16-released-install-it-in-ubuntu.html)

Try it. You just might like it.

### Post by waynefoutz on 2013-07-07
You could install MATE on what you have. I spent last weekend playing with a distro called Ultimate Edition LTS version 3.4, it's based on Precise, but with a newer kernel. After you run the updates after the install, it's running the latest MATE with all the compiz eye candy, including the cube, working out of the box. It has a a LOT of themes to choose from, but all of them a bit too dark for my tastes. It was a great trip down memory lane, it didn't take me long to duplicate the desktop I had from my 8.04- 10.04 days that I loved so much.
