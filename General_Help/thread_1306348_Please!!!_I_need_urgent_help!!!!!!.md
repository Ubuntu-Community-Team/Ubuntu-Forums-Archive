### Post by mechro on 2009-10-30
Using the LiveCD go to System > Administration > Synaptic Package Manager: search and install **testdisk**. It's a semi-gui application so I'd suggest you relax, take your time and get the feel of it before jumping in.

Read the instructions here...

[http://www.cgsecurity.org/wiki/TestDisk_Step_By_Step](http://www.cgsecurity.org/wiki/TestDisk_Step_By_Step)

If you need any more help post back here, somebody will help you even if it takes a little time.

### Post by cgb on 2009-10-30
You may be out of luck depending on what all was finalized in gparted.  You could try using [testdisk]("http://www.cgsecurity.org/wiki/TestDisk") and see if it will recover the deleted partitions.  I believe there is a link for a rescue live disk on that link, this is what you will need.
