### Post by 3Miro on 2009-06-27
1. That is odd, I cannot find such an option on my printer. I don't know, I never had to force black and white.

2. Are you using Ubuntu or something else. Try Totem (Movie Player from Applications -> Sound and Video). That way you will know if it is a system or player issue.

3. I have set up Skype to work on several machines, but you have to play with the options. Go to options and then sound. Try to find one by one the device that will work for you. I had to manually set up pulse as the default output and HW:something as the input. Then it all worked for me.

If those do not work, you may have a general sound setting issue.
