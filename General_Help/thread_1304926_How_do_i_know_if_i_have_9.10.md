### Post by buchan on 2009-10-29
run ```
lsb_release -a
``` from a cmd line... you should see 

```
No LSB modules are available.
Distributor ID:	Ubuntu
Description:	Ubuntu 9.10
Release:	9.10
Codename:	karmic
```
