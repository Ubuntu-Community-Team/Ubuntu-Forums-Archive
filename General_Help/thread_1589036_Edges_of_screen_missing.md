### Post by tedrogers on 2010-10-05
Hi all,

I have a Toshiba television hooked up to my PC. It's fine in Windows, but not in Ubuntu or Fedora 13 for that matter (which is usually more bleeding edge).

The GFX card is an ATI card.

Using my Linux OS of choice, Ubuntu, after install / and on Live CD, the sides of the screen (top and bottom and both sides) are missing. You can waggle the mouse over there and guess the menu's etc. and it works, but you just can't see them. Annoying!

How come in Ubuntu I can't see the edges of the screen. Linux System Rescue 1.6.1 is the same, and this is Gentoo based, so is a different test.

I looked for a video scaling application, but to no avail.

Any help appreciated.

SOLUTION:

On my Toshiba Regza 37Z3030DB, the exact scan option is hidden inside the WIDE / SUPER WIDE / CINEMA etc options menu and is only available when HDMI input is detected. Took me soooo long to work this out. Proper blonde moment I guess, but I honestly expected it to be in the PICTURE SETTINGS part of the menu system. Oh well, so pleased to have found a solution now! :)
