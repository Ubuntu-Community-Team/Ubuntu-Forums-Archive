### Post by tomeriker on 2008-05-18
I hope this helps.. I haven't played diablo for a while so this will be off of memmory..

First thing is: Diablo doesn't install correctly. You need to find an installed game on someone elses computer and copy the "soundsd2.mpq" and I think one other file of the .MPQ type. One sure way to get it right is to compare the files from your install with the other.. Pay close attention to the file size and replace any of yours that are smaller with the correct one. I am very sure there are only 2 that need to be fixed. once you have this done the game should play just as in windows.. It worked for me..
