### Post by 3rdalbum on 2009-03-13
Neither of them are the best.

Ubuntu is the original project with a Gnome desktop. The Gnome desktop focuses on simplicity and elegance.

Kubuntu is the same basic system as Ubuntu, but with a KDE desktop. KDE focuses on flexibility, looks and integration.

Edubuntu is the same basic system as Ubuntu, using the Gnome desktop, but it features software designed for children. It also integrates the Linux Terminal Server software, so a school computer lab can be run with ease.

Xubuntu is pretty much the same basic system as Ubuntu, but it uses the XFCE desktop which focuses on simplicity and low resource use. The base system of Xubuntu is also slightly modified for lower resource use than regular Ubuntu.

Basically, you probably want Ubuntu as it's the "true blue" experience. If you don't like the Gnome desktop you can change to KDE or XFCE without reinstalling.
