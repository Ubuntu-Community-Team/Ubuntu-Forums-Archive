### Post by pqwoerituytrueiwoq on 2013-05-14
The software center will tell you where to find a application once it is installed, for example:
[ATTACH=CONFIG]242597[/ATTACH]
Ubuntu comes pre-installed with software for using torrent files, it is called [transmission]("apt:transmission"), i did quite a bit of seeding for the ubuntu 13.04 release in the 64bit category

executable files are located in [FONT=courier new]/usr/bin/[/FONT]
the downloaded package used for installing is downloaded to [FONT=courier new]/var/cache/apt/archives/[/FONT]
