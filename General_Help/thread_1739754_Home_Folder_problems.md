### Post by Frogs Hair on 2011-04-26
You can try this .

1. Right click desktop and create folder.
2. Right click folder an select open with another application. 
3. Select File Browser from menu. 
4. Check the remember this application box and select open.
5. Test the items on the places menu.
6. Delete folder from desktop.
