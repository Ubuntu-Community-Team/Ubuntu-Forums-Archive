### Post by JimBuntu on 2008-02-25
Ive pondered the same question in the past. I currently have Ubu 64 set up on my AMD laptop, and Ubu 32 set up on my newly built AMD 64x2. I found that the 64 bit version takes some more setting up than the 32 bit. The 32 bit is more ready to go. I didnt see any noticeable improvements as far as speed. The only thing I had was the false impression in my head that 64 bit is 'cooler' than 32 bit. All of the programs are exactly the same with the exception that there are less drivers set up for 64 bit, probably because it is newer and used less. I would stick with 32 bit for the time being.

### Post by fragos on 2008-02-25
You won't notice a difference in speed.  Some libraries and plugins are only available in 32 bit which will make 64 bit more difficult to set up.  32 bit does have a memory addressing limit of 4GB but you aren't likely to be able to use that much memory anyway on the Desktop.  When I bought my 1st 64 bit machine I felt obligated to run 64 bit but I was wrong and got over it.
