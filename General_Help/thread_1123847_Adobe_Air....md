### Post by lovinglinux on 2009-04-12
Adobe Air is just a runtime environment (like Java for example). You need to install compatible applications to use with it. When you install the application it will ask if you want to create a desktop icon launcher. 

You can get applications at [http://www.adobe.com/cfusion/marketplace/index.cfm?event=marketplace.home&marketplaceid=1](http://www.adobe.com/cfusion/marketplace/index.cfm?event=marketplace.home&marketplaceid=1)
