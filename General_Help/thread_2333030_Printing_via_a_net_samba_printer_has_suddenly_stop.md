### Post by hawran.diskuse on 2016-09-04
Finally I got free time to try some things and the problem is solved:

For some reason I installed gadmin-samba application when I was upgrading my notebook (and printing was OK back then) and this app has messed up with my samba somehow. So the solution is as follows:
[B][FONT=courier new]service smbd status
    service smbd stop
    service smbd status
    
    sudo apt-get purge gadmin-samba
    sudo apt-get purge samba samba-common[/FONT][/B]
    
    ***RESTART***
    
    and **[FONT=courier new]synaptic[/FONT]**:
    Install: **samba**:amd64 (2:4.3.9+dfsg-0ubuntu0.16.04.2),     samba-common:amd64 (2:4.3.9+dfsg-0ubuntu0.16.04.2, automatic),     samba-common-bin:amd64 (2:4.3.9+dfsg-0ubuntu0.16.04.2, automatic)
    Install: **smbclient**:amd64 (2:4.3.9+dfsg-0ubuntu0.16.04.2)
    
    ***RESTART***
    
    **[FONT=courier new]service smbd status[/FONT]**
(to check)

Printing is OK again.
