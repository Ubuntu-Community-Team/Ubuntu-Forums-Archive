### Post by Lars Noodén on 2012-12-13
Perhaps some of the script was sending data to stderr in addition to stdout.  You can try capturing both.

```

*scriptname* > /location/textfile 2>&1

```

You can also use the [script](http://manpages.ubuntu.com/manpages/precise/en/man1/script.1.html) utility to record a session.
