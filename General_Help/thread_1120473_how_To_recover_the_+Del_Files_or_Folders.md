### Post by mb_webguy on 2009-04-09
Shift+Del is essentially permanent.  It's like deleting something and then removing it from the Recycle Bin in Windows.  You should only use it when you're sure you never, ever want to see that file again.

However, assuming you haven't made many changes to your hard drive (primarily creating new files) since you deleted the file, it *may* be possible to recover it using something like [TestDisk]("apt://testdisk"), but it won't be easy, and will certainly involve a bit of time and effort.
