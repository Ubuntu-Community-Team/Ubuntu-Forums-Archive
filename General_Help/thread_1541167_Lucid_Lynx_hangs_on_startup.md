### Post by aextance on 2010-08-15
In some ways I seem to have the same problem, only it only happens when I try and boot with a monitor attached to my laptop rather than using the monitor screen. Very strange. 

Try putting in the live CD. Does that still work? There are some folks having this issue with the live CD here:

[http://ubuntuforums.org/showthread.php?t=1472054](http://ubuntuforums.org/showthread.php?t=1472054)

Ruehara, if you want to go back to Karmic, you should be able to keep a partition with Lucid and your data on if you reinstall Karmic from the live CD. I think you might be able to do this right from the install screens, but if not, if you select "try without installing" you can install it and keep the lucid partition that has your data on. Although that can be a bit awkward, so I normally just shunt all my data onto an external hard drive and do a clean install.
