### Post by ajgreeny on 2015-08-14
To the best of my knowledge no satnavs available here (UK) will update using Linux, and that includes both Garmin and Tomtom.

I do manage very easily, however, to update my Tomtom satnav using Windows (XP used for that purpose only) in a virtual machine running on VBox.

I'm pretty certain wine will not manage to work with any USB attached items so I think that will be unsuccessful, even if you managed to get the Garmin update application to work.
