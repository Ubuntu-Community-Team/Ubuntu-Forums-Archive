### Post by geirha on 2009-06-20
If I remember correctly, that message also states that the homedir must not be writable to anyone else. 
```
ls -ld ~
```
What's the permissions and ownership of your homedir?

If the ownership is correct, either of the following commands should make it only writable to you:
```

chmod 700 ~   # No one but you have access
chmod 755 ~   # You have full access, everyone else has read access

```
