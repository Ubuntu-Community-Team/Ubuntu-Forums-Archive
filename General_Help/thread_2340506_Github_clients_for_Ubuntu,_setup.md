### Post by frostschutz on 2016-10-19
You're looking for a GUI frontend to git? Here's a list: [https://git-scm.com/download/gui/linux](https://git-scm.com/download/gui/linux) I don't know any of them, so I can't make recommendations, sorry.

For the terminal program, to get used to it at first you can google "git cheat sheet". There are several, one directly from GitHub. There are also a ton of help pages on GitHub that explain things ( [https://help.github.com](https://help.github.com) ), that's a good resource if GitHub is what you will primarily be using. If you just want to use git locally (w/o going public on github or paying github for private repositories) then I'm sure you'll find some general books on git.
