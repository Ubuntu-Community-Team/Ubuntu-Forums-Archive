### Post by blueparukia on 2008-05-21
I've always preferred doing it the terminal way.

I find that most of the time typing sudo apt-get install programName works.

I just tried this with transmission upon reading your thread, and voila it works.

Open up a terminal window (Start->Accessories->Terminal) and type:

```

sudo apt-get install transmission

```

Then press enter.

You will be asked for your password, type it in and press enter.

Transmission is less than a 1MB download and will install quickly.

Adding it via programs also works, but I don't find it as easy for some reason.
