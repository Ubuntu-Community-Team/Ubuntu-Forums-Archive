### Post by NilsE on 2007-12-27
Basically speaking if you install from update manager, Synaptic, or add/remove you should be OK and less likely to have problems.   

Installing from other than the repositories can cause odd problems even with the simplest typos. So, any manual installs or tweaking can cause short or long term problems.

If you have to edit a file to tweak something be very careful and follow instructions very closely and you will be OK.
