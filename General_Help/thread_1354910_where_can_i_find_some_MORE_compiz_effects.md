### Post by LuisGMarine on 2009-12-14
I'm guessing that you already have downloaded the Advanced Desktop Effects Settings (ccsm).  This little program allows you to change the different 'stock' effects, it has another 20+ animations.  

I don't know if by stock you mean, what Ubuntu comes with without messing around with any settings, or stock as in you already are fed up with all the animations you can change though the ccsm app.

If that's not the case, then download it from Ubuntu Software Center.  Just type in 'compiz' and it should be the second one on the list.
