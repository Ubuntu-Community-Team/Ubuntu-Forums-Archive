### Post by pdwerryhouse on 2008-08-19
You probably don't have libc6-dev installed, and I imagine there's a bunch of other development packages needed, too.

This should probably be enough to get all that you need:

```
sudo apt-get install build-essential
```

If you still have issues after that, then it might be worth installing all of the packages that Ubuntu uses to build its own version of wine:

```
sudo apt-get build-dep wine
```
