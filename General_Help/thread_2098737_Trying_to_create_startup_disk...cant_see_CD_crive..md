### Post by efflandt on 2012-12-27
**Startup Disk Creator** is not for burning CD or DVD images, it is for making a bootable live/install system on USB flash or drive (USB memory stick, SD in USB card reader, etc.)

Try **Brasero** or whatever program you have for writing CD and DVD.  However, don't use the one that might pop up when you insert the blank DVD, that only seems to want to write "files" to the disk. You want to open the full program that can write a CD or DVD image to disk.
