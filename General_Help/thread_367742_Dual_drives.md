### Post by zadehas on 2007-02-22
You'll probably have to set the BIOS boot sequence option in order to choose from which drive you want to boot. I currently have one drive with Ubuntu Edgy and one drive with Windows XP, each installation being independent (no GRUB handling the two). I use the BIOS to decide which OS should boot. This, I think, is the best option for dual booting.
