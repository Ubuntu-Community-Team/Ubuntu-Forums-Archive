### Post by mb_webguy on 2008-09-12
> **ghostdog74 said:**
> there's no need to use an anti-virus program for linux, unless you are scanning your files on a *nix environment and later using them in other platforms.

+1

As ghostdog said, you don't really need an antivirus program for Linux.  (You might want to check the "Security on Ubuntu" link in my signature.)  There are no known viruses for Linux in the wild.  The only reason you'd every really want to run any antivirus software in Linux is to avoid passing on virus-infected files to Windows users -- which is very important for a mail server, but somewhat less so for a desktop computer.

However, there are several AV applications available.  Some people like clamav, which you can find in the repositories.  It's a command-line scanner, but you can find several GUI frontends for it in the repositories as well.  I personally prefer [AVG Free for Linux]("http://free.avg.com/ww.download?prd=afl").  It has a simple interface, it's fast, and updating the virus database is extremely easy.

EDIT:
> **zohaib said:**
> cool, so i don't have to worry about spywares n trojans., have i?
No, but you're still susceptible to script-based exploits in your web browser.  Using Firefox with extensions like Adblock Plus, FlashBlock, and No-Script will take care of that, though.
