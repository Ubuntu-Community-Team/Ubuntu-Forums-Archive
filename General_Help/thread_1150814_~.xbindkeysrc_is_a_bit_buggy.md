Hi

Ive got a Logitech G5 Gaming Mouse, and Ive finally managed to map it buttons to keyboard commands.

heres a part of my .xbindkeysrc file: 
```
# Wheel Left Tilt [Button 6] = Alt_L + left
 "xvkbd -xsendevent -text "\[Alt_L]\[Left]""
 b:6

# Wheel Right Tilt [Button 7] = Alt_L + right
 "xvkbd -xsendevent -text "\[Alt_L]\[Right]""
 b:7

# Backward (big left side button) [Button 8] = Opens a new Tab in Firefox
 "xvkbd -xsendevent -text "\[Control]\[T]""
 b:8

# Forward (little left side button) [Button 9] = Open New Terminal
 "gnome-terminal"
 b:9
```

However, in firefox, on random occasions, Control+T behaves as Control+Shift+T (and opens a recently closed tab instead of a new one).

Any idea why?
