### Post by xyzzyman on 2012-01-25
That's all controlled at the mobo/BIOS level. It's not like clocking a CPU. Most modern motherboards already clock the memory to the optimal level, so unless you are trying to overclock which at that point you'd have a motherboard designed to handle that better, you're out of luck.
