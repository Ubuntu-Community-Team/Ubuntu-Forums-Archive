### Post by nkri on 2008-08-07
Go to System>Preferences>Main Menu.

Navigate to the folder you want to change, and right click on it.  Click Properties.

On the left, there should be a button showing some sort of icon.  Click on it, and it will bring up the set of icons Ubuntu uses by default or are added by applications.  Choose one of these, or browse for a custom icon from some other folder and click OK.  This should change it, although I'm away from my laptop right now and can't check for sure.

Good luck!
-nkri
