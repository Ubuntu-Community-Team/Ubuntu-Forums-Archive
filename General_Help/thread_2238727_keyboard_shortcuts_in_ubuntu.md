### Post by Paulgirardin on 2014-08-09
If you are using Ubuntu with Unity DE,hold down the super key (windows Key) for a second and the launcher will appear with number overlays on each icon.Hit the number for the app you wish to open and it will do so.The keyboard shortcut overlay appears at the same time.
