### Post by Lostincyberspace on 2007-12-18
If you burned the disk correctly. then this is how you install.
First you need to boot until you reach the desktop. then there will be an icon that says "install Ubuntu to the hardisk" or some thing to that effect. then follow the prompts until you get to the partition portion then you should choose the "format whole drive and install ubuntu" option or again something similar to that. Then just follow the rest of the prompts and it should install after a few more questions.
