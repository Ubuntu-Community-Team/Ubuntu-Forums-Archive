### Post by snowpine on 2009-03-04
> **measekite said:**
> During the time of the Long Term Version that LTS gets security updates and bug patches do the repositories maintain and update the applications to new versions as they are released or are the the versions fixed at time of release like the regular versions?

The versions are fixed at the time of release. Ubuntu is not a "rolling release" distro.

There is a "backports" repository that you can enable to get new versions of certain apps, and of course there are ways to install the latest version of a specific app (for example if you search you'll find tutorials for OpenOffice 3).
