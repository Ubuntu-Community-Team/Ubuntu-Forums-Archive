### Post by sites on 2011-02-21
Boot up a live cd and run memtest.  If RAM passes the test, reboot all the way to the live desktop and check the hard drive SMART status in Disk Utility for any issues.  Verify that your hardware isn't faulty before banging your head on the desk over software issues.
