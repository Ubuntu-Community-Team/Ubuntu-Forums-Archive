### Post by Firestone on 2008-12-16
I love irssi, but if you're already lost here you might wanna check out a gui client.
Anyway; start with '/server <server>', '/nick <nick>' and '/join #<chan>'. Then switch windows with 'alt <window number>'. 
To fully set it up you'd better check the links above and use '/help'.
