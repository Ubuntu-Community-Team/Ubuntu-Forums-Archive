### Post by winh8r on 2011-12-10
Just a little something I discovered which might be useful in a small way, if you go into your Synaptic Package Manager and select settings>repositories and look for the "download from" drop down menu, it will generally be set to the default for your country of residence/installation/locale, however if you have a look through the options for the mirrors listed you can often find an alternative which is considerably faster than the main Ubuntu server for your country.
Using another mirror does two things , it lessens the load/traffic on the main server and can speed up your update process quite remarkably.

Just thought this might help.
