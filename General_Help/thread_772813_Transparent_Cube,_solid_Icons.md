### Post by gameryoshi600 on 2008-05-01
install the compiz comfig settings manager. type the following in the terminal:
```
sudo apt-get install compizconfig-settings-manager
```
Now when its done go to system>preferences>advanced desktop effects settings.
click desktop cube go to the transparent cube tab then set opacity during rotation and opacity when not rotating. i use 80. hope i helped. 
:KS
edit: sorry you cannot make the icons transparent. i'm sure there is something out there but i do not know about that.
