Ok,  Here is a workaround:


[LIST=1]
[*]Download gimp - Image Editor
[*]Pick the image of your choice
[*]Save the image as \urs\share\background\warty-final-ubuntu.png [bascially replace the old png with this one]
[*]restart.
[/LIST]
