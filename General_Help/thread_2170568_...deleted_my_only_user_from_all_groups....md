### Post by steeldriver on 2013-08-26
Yes there is a single-user 'recovery console' for fixing this kind of thing - see [http://www.psychocats.net/ubuntu/fixsudo#recoverymode](http://www.psychocats.net/ubuntu/fixsudo#recoverymode)

Each user's *primary* group is a 'user private group' with the same name as the user. I *think* that the default secondary groups for the 'primary user' are: adm cdrom sudo dip plugdev lpadmin
