### Post by coffeecat on 2008-08-11
As far as I understand it, ubuntu-desktop is a metapackage which is used to pull in all the packages making up a default Ubuntu gnome desktop if you want to install this from Kubuntu or Xubuntu. Or, if you're using the standard Ubuntu Gnome, you need to keep it installed to ensure that everything gets updated. With it you get the Ubuntu themed and tweaked version of Gnome. With Hardy you get the 2.22 version of Gnome.

I can find no 'gnome-desktop' package in Synaptic, but there is a 'gnome-desktop-environment'. If you look carefully, you'll see that this is the 2.20 version of Gnome, the previous version. I have no idea exactly what this package does, but I would guess that apart from installing an earlier version, it would install a default Gnome desktop rather than the Ubuntu variant.

And if that means spatial mode in Nautilus, I wouldn't touch it with a 90-foot bargepole! :p

(Yes, I know you can change it with two clicks. :wink: )
