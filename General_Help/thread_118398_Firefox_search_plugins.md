### Post by aysiu on 2006-01-16
It depends, actually, on whether you're using a Ubuntu Firefox or a Mozilla Firefox and whether the plugin came with Firefox or you added it yourself. I'd look here, though:

/usr/lib/mozilla-firefox/searchplugins/
/opt/firefox/searchplugins/
~/.mozilla/firefox/*.default/searchplugins/
