### Post by kkruecke on 2011-05-10
1.From Application launcher->Choose the Applications icon, the one that looks like a magnifying glass)

2. under Installed choose "see more results"

3. scroll down to the Keyboard application

4. On the Layouts tab click "Apply System wide"

5. Still under the layouts tab, click Options button, then Alt/Win behavoir.  

6. I chose "Meta is mapped to Left Win" (you may need to click "Apply System-wide" again).

Now, when the unity applicaton laucher is present, if you press the left Win key, you will that numbers appear superimposed over those applications with shortcuts. Click the number corresponding to the application you wish to start.
