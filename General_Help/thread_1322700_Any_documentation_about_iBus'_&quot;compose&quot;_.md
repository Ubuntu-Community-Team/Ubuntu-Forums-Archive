### Post by Nicolas Raoul on 2009-11-11
Hello all,

I need to type accentuated letters on a keyboard that has no accents, so I find iBus' new "compose" input method very convenient!

As expected for an input method, it is a bit intricate, and "compose" has a lot of settings (many more than Anthy for instance). Unfortunately, there does not seem to be any documentation explaining how to get the most of it. Input methods all have their great tricks to achieve great things in an optimal way, and "compose" is probably no exception. But I can't find any documentation.

The "About" information is "Mimic Compose Key input", but this term has zero results in Google. The author is Yu Yuwei. compose allows one to  input special letters by composing a letter and diacritical mark. For instance e' becomes é. Obscure settings include things like "Switch to normal commit mode, which use space to commit", "Switch to full punction", "Switch to full letter", or "Switch to table mode".

If there is no documentation anywhere, where should I create a Wiki page for this?
