### Post by Seq on 2009-04-22
FWIW, it works fine in IMAP (I send todo notes to myself all the time).

Although mail sent by you to a mailing list doesn't work still. The mail list sends that message out to everybody (including yourself), but gmail discards that copy as it already has a copy (the copy you sent). That message thus doesn't get labelled properly, so you have incomplete conversation threads via IMAP (though it works in the web interface).
