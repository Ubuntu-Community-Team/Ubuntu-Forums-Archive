Well, for those who doesn't know:

If you want to hide the folder "MyHiddenFolder", and can't rename it to ".MyHiddenFolder" (for whatever reason), do this:

* Create a file called ".hidden" in the folder where the folder or file is placed!

* Open the file (Enable showing hidden files/folders) and write the name of the file/folder, each instance as a new line, and end it with a newline without content...
