### Post by 3rdalbum on 2011-04-25
No issues, I do the same. Netbook hardware might not last 24/7, 52 weeks a year - but a lot of people do use an older machine with Ubuntu to create a file server. I have a mini-ITX server with an Intel Atom CPU. I'd recommend using system-config-samba to easily create the Samba share, it's a lot easier than manually editing the samba.conf file.
