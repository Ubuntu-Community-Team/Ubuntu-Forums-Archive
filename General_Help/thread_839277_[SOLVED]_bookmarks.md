### Post by WRDN on 2008-06-24
In Firefox in Vista, select Bookmarks > Organise Bookmarks, then select File > Export (I think).
Then, in Ubuntu, select Bookmarks > Organise Bookmarks, and select 'Import and Backup' > 'Import HTML'
