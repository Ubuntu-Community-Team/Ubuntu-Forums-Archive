Welcome. For Skype, open 'Software and Updates' and make sure you have the 'Canonical Partners' repository enabled in the 'Other Software' tab. Once that's done, you will be asked to do an update and they you simply open Software Centre and install Skype (search and you'll find it there).

Best to try and install things from the repository where possible rather than dragging them in from third-party sources. Much of what you need is already in the Software Centre.

Never heard of Yarxi, no idea there, and your password issue should be edited out of this thread and posted in a new one (one issue per thread is preferred and avoids confusion). Lastly, you might want to change the title of this thread to something descriptive like 'Can't install Skype' (edit first post and the 'Go Advanced'). 'Skype' is generic and tells potential helpers nothing about your issue. ;)
