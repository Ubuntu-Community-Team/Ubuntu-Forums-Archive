### Post by doobious on 2006-12-19
I had the same problem with only one Alt working, noticed it in Firefox.  I was able to fix it by going to Preferences -> Keyboard -> Layout Options -> Alt/Win key Behavior.  Then I changed the Default to "Alt and Meta are on the Alt keys (default)", seems to work fine now.
