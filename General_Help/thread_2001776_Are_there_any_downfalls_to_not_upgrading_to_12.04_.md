### Post by VE6EFR on 2012-06-11
As long as it's working for you, the only downfall I can think of off hand is eventually the version you are on will no longer be supported. This means you wouldn't get any further security or software updates. This could potentially leave your system somewhat vulnerable.

---

### Post by Cheesemill on 2012-06-11
10.10 has reached end of life and therefore doesn't receive any more updates or support.

If there are any security flaws found in any of the software you have installed they will **never** be fixed, leaving you potentially vulnerable to being hacked.

If it's just the new Unity interface that you don't like you can easily use Gnome fallback instead to give you a Gnome 2 style look and feel.

### Post by leopoldbirkholm on 2012-06-11
If it is the Unity interface that bother you, I would suggest you read this thread: [http://ubuntuforums.org/showthread.php?t=1859961](http://ubuntuforums.org/showthread.php?t=1859961)

As other mentioned, you will not longer get security update for the system.

So my suggestion for you, read the thread mentioned above, take a backup of your files and do an upgrade when you have a work-free day.
