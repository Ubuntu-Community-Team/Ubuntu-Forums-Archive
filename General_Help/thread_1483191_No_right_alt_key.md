### Post by WorMzy on 2010-05-14
The right Alt key is generally an "[Alt Gr]("http://en.wikipedia.org/wiki/Alt_Gr")" key, rather than an Alt key. If you would rather it be another Alt key, you'll probably need to change something in the keyboard settings (System -> Preferences -> Keyboard).
