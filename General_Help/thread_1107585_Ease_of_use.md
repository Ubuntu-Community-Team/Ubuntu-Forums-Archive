### Post by lovinglinux on 2009-03-27
You can also right-click the script, select copy then paste in the terminal, but it is easier to just double-click the script file and select "Run in Terminal" option.

If you need to run scripts you have created, then put them under ~/bin and just type the file name in the terminal. It's much easier and faster than browsing the script with Nautilus.
