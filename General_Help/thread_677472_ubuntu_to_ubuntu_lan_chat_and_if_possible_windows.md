### Post by PeterJS on 2008-01-24
Pidgin can do that and is installed by default. The protocol for sending messages on the local network is called Bonjour. This will also work with win boxes, as you can install pidgin on them. It's interesting to have running on random public networks, never know what kinda people you can meet that way.

### Post by jpeddicord on 2008-01-24
See the package "xipmsg". It is compatible with the "msg" command on Windows boxes. It's run via the CLI, but the messages pop up in a GUI. It's a little archaic, but it works for random messaging for system notices and whatnot.

You can also just try getting Pidgin on all of them and syncing up all of the contacts... cumbersome, but it works pretty well. All of the PCs must be connected to the Internet for this.

There are some other ones out there, but the xipmsg one was the first that came to mind. Nothin beats random LAN messaging.
