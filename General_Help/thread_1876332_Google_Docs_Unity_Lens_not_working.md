### Post by twtduck.ii on 2011-11-06
I am not sure about anything below, so I may be completely wrong.


In Ubuntu 11.10, there's a new version of Compiz and, therefore, Unity, compared to 11.04. I think that the Google Docs lens is not compatible with the old version of Unity and to use it you need to upgrade to Ubuntu 11.10. 

Also, it is helpful information for you to know that very many core components of Ubuntu were updated with the 11.10 release and, though they look basically the same on the surface, Ubuntu 11.10 is VERY different from 11.04. In other words, to use the lens, you need to update to 11.10. 

It is not hard and usually only takes a couple hours (depending on computer speed and internet speed) to update to 11.10. Go to update manager, click settings, and go to updates to set it up to check for new releases as shown (attached). Then sometime soon it will tell you when you start the Update Manager that there is a new release and you can update.
