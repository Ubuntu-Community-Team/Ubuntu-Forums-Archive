### Post by janpol on 2010-10-29
Do you have your NTFS drives mounted? In order to do that, you just need to open them from the Places menú.

Also, if you want 'em to be automatically mounted every time you log into Ubuntu, you can use this guide:

[http://ubuntuforums.org/showthread.php?t=785263]("http://ubuntuforums.org/showthread.php?t=785263")

After mounting the drives, if they don't appear in the google desktop menu as standalone drives, search for the folder where ubuntu mounts 'em (normally some folder inside /mnt).

Hope it helps
