### Post by tgalati4 on 2014-08-10
Not much available, but if you are clever, you can try using *sox*.

[http://ubuntuforums.org/showthread.php?t=1576554&highlight=play+rec+pitch](http://ubuntuforums.org/showthread.php?t=1576554&highlight=play+rec+pitch)

Basically use *sox* to bend the microphone input and send it to a temporary file.  Then assign the temporary file as the input to Skype.  Because of the delays involved, talking in realtime may be a challenge because your brain can't process speech when it is delayed more than 200 or 300 milliseconds.
