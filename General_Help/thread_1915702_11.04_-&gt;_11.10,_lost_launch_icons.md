### Post by mcduck on 2012-01-28
If you install [Gnome-Tweak-Tool]("apt:gnome-tweak-tool") (which I'd highly recommended as it allows you to configure your theme, fonts and other settings that Gnome 3's option dialogs don't have yet), you'll find the option to enable desktop icons in the Desktop category.

Alternatively you can use dconf-editor (included in the [dconf-tools]("apt:dconf-tools") package) to enable the icons, or do it by running the following command:
```
gsettings set org.gnome.nautilus.desktop volumes-visible true
```
