### Post by Gremlinzzz on 2012-02-18
System Requirements

The minimum memory requirement for Ubuntu 10.04 LTS is 256 MB of memory. Note that some of your system's memory may be unavailable due to being used by the graphics card. If your computer has only the minimum amount of memory, the installation process will take longer than normal, but will complete successfully, and the system will perform adequately once installed.

### Post by claracc on 2012-02-19
I have lubuntu 11.04 in an old pentium III laptop, 1 GHz CPU, 512 MB ram (64 MB for graphics old sis card) and 20 GB HDD and I am very happy with it.

It works very well for surfing the net with epiphany browser. I had to do some tweaks and install some greasy scripts in order to watch  well flash videos in youtube (CPU resources below 50 %). For other flash videos in newspapers, they are  in slow motion but I think this is a flash problem in ancient hardware. Scripts for managing different flash containers ( brightcove, EMP in bbc videos) can be made to play flash in your mplayer software but it is a huge task proper for developers and experienced people.

I can install firefox in the system but It takes a lot of CPU resources in the old laptop and it is very slow in general. Epiphany browser is quick, simple and responsive in old hardware.

For office tasks I only use abiword, it is enough. Openoffice can be installed but it is slow.
