### Post by skwishybug on 2007-04-03
If you mean setting the system default, go to the location of the file you want to set, right click on it and select properties. In the "Open With" Tab, find the program you wish to use to open that type of file.

If the program is not there, click the "Add" button and add it in.

Click close. Double click the file name and check your selection.
