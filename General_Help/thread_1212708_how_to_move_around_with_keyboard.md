### Post by nothingspecial on 2009-07-14
Alt F1   - Opens menu
Alt F2   - Run an application
Alt F4   - Close Window
Alt F5   - Make window smaller (can`t think of the word)
Alt F7   - you can move the window with your arrows
Alt F8   - resize window
Alt F9   - minimize window
Alt F10  - maximize window

There are tons of others and you can make your own

Of course the best way of using ubuntu without a mouse is [[COLOR="Magenta"]gnome-do[/COLOR]]("http://do.davebsd.com/")

```
sudo apt-get install gnome-do
```
