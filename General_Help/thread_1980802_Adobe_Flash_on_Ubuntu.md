### Post by Rasa1111 on 2012-05-15
Just have to Install "ubuntu restricted extras", and your problem will be solved. 
Not sure if it will work from the CD, but once you have Ubuntu installed, then do it. 

Once you install Ubuntu restricted extras, you'll be good to go. 

Install it from the software center* (or from terminal)

If you install from terminal, just type 
```
sudo apt-get install ubuntu-restricted-extras
```
