### Post by drs305 on 2008-07-30
> **b-boy said:**
> Hi guys where can i set a TIME for automatic updates to run

Ubuntu has a process called crontab which can run a script or command at a give time/date. If your computer is not on all the time, there is also a process called anacrontab that takes care of things should you often turn your computer off.

You can read a bit about them by typing "man crontab" or "man anacrontab" or do a search of the forums or net.
