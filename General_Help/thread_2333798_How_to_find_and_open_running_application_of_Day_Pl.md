### Post by kologha on 2016-08-13
I know that Day Planner is running on my PC (Ubuntu 14.04) but I cannot find it in order to open it and add new events I want to be reminded of. I have it locked to the Launcher, but if I open it from the launcher it gives me this message:

"Another instance of Day Planner appears to be running. Please quit the  currently running instance of Day Planner before continuing. You may  force the other instance to quit if you wish." 

I do not want to force the running one to quit, because then I will probably have to re-input all the events currently on it. Please help me!

Thank you

### Post by howefield on 2016-08-13
Once you add an event to Day Planner it is saved as far as I can see.

Check that by opening the ~/.config/dayplanner/calendar.ics file in gedit to see your calendar entries. If you are happy that they are all present and correct take a copy of the folder just to be sure then you should be safe enough to close the "other" running instance of dayplanner.
