### Post by FRuMMaGe on 2009-12-17
To all users who may be suffering with this problem there is a very simple workaround, but if you know a fix please let me know.

For the last few months on my Jaunty installation the boot is hanging on the "Setting up console font and keymap", the problem is not actually with the console font or keymap, but the command it is trying to run afterwards, I know this because originally it occured on the "setting up nfs shares" but after removing nfs from my system, it now hangs on the keymap command, so this is probably not the original culprit.

The workaround is so simple its almost embarrassing

WORKAROUND:

Ctrl+Alt+Del

Yes, by pressing this key combination it skips this step and resumes a normal boot. It's pretty annoying that I have to do this on each and every startup, but it works.

If anyone can help in fixing this problem it would be great
