### Post by fragos on 2006-11-14
I recommend you use both as they are complementary.  The monitor is sufficient for hardwwired networks but a network manager add the features you need to control and select from available wireless networks.  The monitor will display a signal strength icon in the panel when wireless networks are in use.  From my experience I'd recommend wifi-radar as your network manager.
