### Post by timgood on 2011-04-29
Hmmm.

After I upgraded from 10.10 I was pleasantly surprised to get a reasonably flawless Unity desktop. There are some cosmetic issues with Plymouth, but I'm sure these can be fixed eventually.

There were other not so wonderful things: Skype starts, but is only shown in the notification area randomly. Very annoying. Sometimes when it starts the sound is corrupted, sometimes it's fine. Some apps don't show up in the notification area at all: Shutter and Radio Tray being the most useful (but not anymore!). The much vaunted launch bar looks OK except for showing icons that are not in the current workspace - and there is no way to change ugly icons (Boinc Manager) for other, better looking ones.

Whoever decided to put 'System Settings' with the Logout button needs shooting. Took me ages to find it. I'm finding the launcher not very intuitive and it seems that things I accomplished with a few clicks of the mouse now take several. Can't seem to use Lightscribe applications from the launcher (won't start unless run from a terminal).

I know it's early days. I'm really trying hard to like Unity at the moment, but I'm hoping that it's going to get better, quickly! Any help with the above problems would be welcome!
