### Post by zo3adams on 2011-03-03
Python is an interpreted language, so compiling is not actually necessary.

The interpretor will compile the your program to byte code at run time.  To run a python program you wrote just  make it executable and call on it. ie:

chmod a+x MyProgram.py
./MyProgram.py


<- ran from the directory the program resides in, this will change the permissions to allow any user to run it, then run it.
