### Post by reyvehn on 2008-05-10
I'm using hardy heron and just installed Advanced Desktop Effects Settings manager.  When I navigate to System>Preferences>Appearance and then the "Visual Effects" tab and select "Extra", everything works fine (wobbly windows, workspace switcher, window switcher, etc...).  

However, when I go into System>Preferences>Advanced Desktop Effects Settings and enable the desktop cube, it simply doesn't work (I've used compiz before and all the settings for Desktop Cube are as they should be).  In addition, when I do this, it disables the workspace switcher even though I can plainly see 4 workspaces on my panel.  The only way to restore the workspaces is to go back to the Visual Effects settings and choose either Extra, Normal, or None.  But then I'm back where I started.

Has anyone had this problem before or does anyone have any suggestions?  Thanks a lot in advance.

### Post by tommyhakinen on 2008-05-11
make sure you have all these plugins enabled:
Desktop Cube, Rotate Cube, Cube Reflection

after that, go to the Advanced Desktop Effects Setting and look for General Setting. Under Desktop Size tab, change the Horizontal Virtual Size to 4.
