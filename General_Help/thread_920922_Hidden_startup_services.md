### Post by Shambles on 2008-09-15
I've been getting this odd problem lately.  Every time I boot up ubuntu it starts up several programs like they were boot up services, only they aren't and I'm not sure where to look for the cause of this problem as I'm sure it severely affects my boot up time.

Each time it is the same.  It loads a terminal window, pidgin, firefox, and the file manager.  Is there a file that might hold more startup information than the services options?

---

### Post by howefield on 2008-09-15
Go to System > Preferences > Sessions and click on the Sessions Options tab, is their a check mark in the box next to Automatically remember running applications when loggin out ?

If so, uncheck it.
