### Post by matthew on 2006-01-05
I haven't used that program to convert files.

There is another available in the Ubuntu repositories that has worked very well for me. It's called "soundconverter" and can be installed using Synaptic or at a terminal by typing ```
sudo apt-get install soundconverter
```
Once it's installed you can run it from the menus at Applications->Sound & Video->Sound Converter. It has a nice GUI and is quite easy to use.

Good luck and have fun!
