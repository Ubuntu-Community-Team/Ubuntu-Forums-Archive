### Post by 3Miro on 2011-11-27
You can burn the .iso to a CD and then install it within Windows with Wubi. (this is recommended for someone completely new to Linux)

[https://wiki.ubuntu.com/WubiGuide](https://wiki.ubuntu.com/WubiGuide)

Alternatively, you can setup dual-boot where both Windows and Ubuntu have their own separate partitions. (this is recommended if you want to have the full power of both OS)

[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

During installation of Ubuntu, you will also have the option to completely remove windows (if this is what you want).
