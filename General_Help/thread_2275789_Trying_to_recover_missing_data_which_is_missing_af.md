### Post by Mark Phelps on 2015-04-28
How do you know the "D" drive is gone?

Recovering data from former Windows partitions is best done using Windows data recovery apps -- and to do that, you need a working Windows PC.

If you have that, and the drive is connected to that, then download and install NTFS GetDataBack from Runtime Software.  Run it in the deepest search mode -- and this could take hours, so best to run it overnight.  When done, see what it has found.  IF it found the files and folders you want back, you have to then go online and purchase a license -- and you then have to recover those files/folders to a different drive.
