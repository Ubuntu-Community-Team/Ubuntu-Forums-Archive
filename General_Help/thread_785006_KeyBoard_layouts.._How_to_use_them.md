### Post by sdennie on 2008-05-07
I think that by default hitting both Alt keys will swap the layout.  You can configure this via System->Preferences->Keyboard->Layouts->Layout Options->Layout Switching.  You can also add an icon to the panel that tells you what layout you are in and it will switch it by clicking it.  You can find that by right clicking on a blank part of the panel and selecting Add to Panel then adding the Keyboard Layout Indicator applet.
