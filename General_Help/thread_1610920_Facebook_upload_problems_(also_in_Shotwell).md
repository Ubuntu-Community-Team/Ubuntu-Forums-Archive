I don't think it's Ubuntu that's causing the problem.  I haven't had a Facebook account in a long time, but I remember having problems with uploading images at first. If I remember correctly it was the NoScript extension for Firefox that was part of the problem.

I think I had to make sure the correct java plugin was installed as well. As far as Flash, this [plugin]("https://addons.mozilla.org/en-US/firefox/addon/161939/") is great for getting the correct plugin for Flash.

Also, Facebook seemed to work fine, with no extra work, using the Chrome browser, so maybe try that too. 

Hopefully someone with more experience will have more ideas, but maybe these suggestions will be a good place to start.
