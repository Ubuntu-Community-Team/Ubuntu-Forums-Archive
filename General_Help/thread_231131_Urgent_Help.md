### Post by confused57 on 2006-08-07
The desktop Live CD installation probably requires at least 256 mb ram to install...the alternate cd would work better to install on systems with less than 256 mb ram.
If you want, you can try booting the Live CD up using the "safe graphics" mode & try installing.
Here's another possible method:
[http://www.ubuntuforums.org/showthread.php?t=218036&highlight=legacy](http://www.ubuntuforums.org/showthread.php?t=218036&highlight=legacy)

You really shouldn't have any problems installing with the alternate cd and you may want to consider downloading and installing the Xubuntu alternate CD iso, which is faster on lower end systems than Dapper.
