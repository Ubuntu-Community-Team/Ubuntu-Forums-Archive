### Post by nerdtron on 2013-09-12
Windows has no "recycle bin" for USB flash drives and Ubuntu has a .Trash folder which acts as a "recycle bin" for all drives (even USB drives too). Have you tried to empty the Trash after deleting from USB? Or perhaps, Shift+Delete on the file you want to delete?
