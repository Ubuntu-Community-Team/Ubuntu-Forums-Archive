On reflection it seems that a problem occurred at upgrade time and subsequent moving of photo files combined with newer version of Kphotoalbum. Each file showed "NO SEARCH" in exif info. When I highlighted some files and re-read exif data WITH UPDATE EXIF DATA checked then reordered photo order it corrected order and enabled grayed out options to be accessed.
