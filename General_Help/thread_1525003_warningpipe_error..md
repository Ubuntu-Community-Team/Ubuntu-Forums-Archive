### Post by e_l_p_i_s on 2010-07-22
In case anyone else reads this thread when searching for similar error I found it helped going to about:config in Firefox and set to false all options (except nptest) in dom.ipc.plugins for the time being until it behaves itself.

Helpful pages:
[https://wiki.mozilla.org/Firefox/Crash_Protection](https://wiki.mozilla.org/Firefox/Crash_Protection)
