### Post by Frogs Hair on 2012-02-19
Be sure have placed the icons in the correct locations. Some applications will store the icons in usr/share/pixmaps in the root file system.

One way to ensure the path to the icon you want to use is to open the main menu , enter program properties by right clicking on the line the application is listed . When the dialog box opens select the icon picture on the left and navigate to the image you want to use . Repeat if the icon is incorrect at the next login.
