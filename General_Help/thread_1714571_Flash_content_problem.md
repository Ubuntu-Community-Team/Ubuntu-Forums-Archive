### Post by ajgreeny on 2011-03-25
Try using the **flash aid** add-on for firefox from [https://addons.mozilla.org/en-US/firefox/addon/flash-aid/](https://addons.mozilla.org/en-US/firefox/addon/flash-aid/)

It will remove whatever you have already, including gnash and other open source items that might conflict with adobe flash, and install the latest version for your system.  It is a very useful add-on and the changes made by firefox will also work in chromium.
