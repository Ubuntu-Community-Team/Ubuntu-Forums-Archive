### Post by jonian_g on 2008-07-17
Go to Synaptic, check for installation the package you want and instead of clicking apply, go to the File menu and select generate download script.

But you have to do this on the computer you want to install the package to get all the dependencies. Then run the script (double click) on your computer and all the packages will be downloaded in the folder the script is.
