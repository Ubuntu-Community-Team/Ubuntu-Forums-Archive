> I have Edgy version

The resolution may be wrong. That sounds quite likely particularly since a panoramic display would probably have a non-standard native resolution. Could you check what Ubuntu has set the resolution to and see if it's correct for your display, assuming that you know what the native resolution of the display is? You can see the resolution Ubuntu is using in System > Preferences > Screen Resolution, or in the terminal by entering "xrandr". In the xrandr output, the resolution you're using is the one marked with an asterisk (*).
