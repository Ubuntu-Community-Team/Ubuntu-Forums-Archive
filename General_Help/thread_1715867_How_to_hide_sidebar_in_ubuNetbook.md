### Post by lithopsian on 2011-03-27
This desktop is called Unity.  It is the default in the Netbook edition of Ubuntu Maverick (10.10) although not in the Desktop edition.   So one way of getting rid of it would be to install the desktop edition :)

Alternatively, what desktop manager would you like?  You can install whichever one you like, but you may have to instruct it to replace Unity.  For example:
```
openbox --replace
```
