### Post by mörgæs on 2011-04-29
Take 11.04 to a long, thorough test before deciding. 10.04 is a rock-solid system, and I would think twice before throwing it away.

If you have decided, you could boot a live 11.04. From that you run Gparted and delete the old Ubuntu partitions.
