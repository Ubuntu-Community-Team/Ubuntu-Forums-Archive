### Post by CharlesA on 2013-04-07
Either your banking website sniffs HTTP headers to find out what browser you are using, or you have an addon installed that blocks cookies or javascript, such as NoScript.

Check your extensions and start firefox is safe mode to see if the problem persists.

### Post by kurt18947 on 2013-04-08
Some banks set a cookie that says "oh, this machine has been here before".  If the site doesn't find that cookie, I must go through an authentication procedure. In addition to add-ons as CharlesA mentioned,  you might look in Edit -> Preferences -> Privacy -> Accept cookies from sites.  Try adding your bank to the list that is allowed to set persistent cookies.
