the reason you are seeing ver 2.13 as the available version in the package manager is because Ubuntu is not a rolling release. This means the versions are locked when the distribution is released. Gutsy's repos were locked in October 2007, and you only get security updates.

If you want to upgrade your version you will either have to compile the program from source or get a precompiled bnary from a source such as [getdeb]("http://www.getdeb.net/").
