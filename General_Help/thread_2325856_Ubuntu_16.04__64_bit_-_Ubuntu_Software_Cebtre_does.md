### Post by grahammechanical on 2016-05-26
For information purposes. Ubuntu 16.04 no longer comes with Ubuntu Software Centre. It has been replaced by Gnome Software store and re-branded as Ubuntu Software store. The software catalogue for Ubuntu Software is still being compiled. There are also improvements to Ubuntu Software yet to come.

When you installed if you ticked the box Install third party software you would have got some third party audio & video codecs and also Microsoft core fonts. Provided you accepted the Microsoft End User License Agreement (EULA). If you missed the dialog and failed to accept or reject the offer, then this particular action is still incomplete as far as the Ubuntu package manager is concerned. 

A newly installed Ubuntu Software Centre will not have an up to date database. It can take a several minutes to re-populate the database.

Regards.
