Via video hardware in Linux is not supported very well, easiest thing to do would be to buy a cheap video card with a Nvidia or ATI chipset.

There are **no** 11.04 drivers available on the Via Linux site, the latest they have is for 10.10.

Install the **unity-2d** package if you want to have Unity.
