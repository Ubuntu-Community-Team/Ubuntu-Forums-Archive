Hi, is it possible to have vertical desktops when using the cube. I would like to have a cube with 4 horizontal sides and 2 vertical sides. If i set this amount in compiz desktop settings, i can only rotate horizontal.

### Post by mcduck on 2009-03-28
No, it's no possible. The reason for this is that the cube top&bottom would have wrong dimensions.

For example if your desktop resolution is 1024x768, the cube caps would be 1024x1024. Fitting your desktop there would require scaling it to that resolution every time you rotate the cube so Compiz designers decided to leave the caps for simple graphics only.
