### Post by cariboo on 2008-11-27
You can access cups to modify the printer options. To do this in your web browser address bar type:

```
http://localhost:631
```

Once at the cups web page click Administration-->Manage Printers-->Set Printer Options and scroll down to Starting Banner and set it to none.
