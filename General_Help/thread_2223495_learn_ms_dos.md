### Post by matt_symes on 2014-05-11
Hi

Linux doesn't use Microsoft DOS. They are different operating systems after all.

However take a look at dosbox. It's a MS DOS emulator. That may be useful for you.

You can install it from the terminal using

```
sudo apt-get install dosbox
```

You can run it by the terminal by typing

```
dosbox
```

It may also get an icon somewhere to start it.

Kind regards

### Post by sudodus on 2014-05-11
Ubuntu and many other linux distros do not use MSDOS, but bash as the shell program for terminal window commands. MSDOS is a Microsoft product, and is available in Windows.

Do you want to learn the linux bash, or do you still want to run DOS (there are other DOS alternatives, for example FREEDOS, alongside MSDOS)?
