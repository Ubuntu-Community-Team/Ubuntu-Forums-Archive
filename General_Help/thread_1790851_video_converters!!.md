### Post by papibe on 2011-06-25
> **amna543 said:**
> im trying to get films on my ipod...:(
[Handbrake]("http://handbrake.fr/") has several Apple presets like iPod, iPhone and iPad.

Best bet is to use this ppa:
```
$ sudo add-apt-repository ppa:stebbins/handbrake-snapshots
$ sudo apt-get update
$ sudo apt-get install handbrake-gtk
```
Regards.
