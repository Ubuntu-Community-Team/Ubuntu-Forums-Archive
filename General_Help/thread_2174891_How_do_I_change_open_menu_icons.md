### Post by Frogs Hair on 2013-09-16
> when I use the open command   Hello , what command are you using ?  When icons are stored in usr/share/icons they will appear in all applications.
Example if I open Thunar as root I would see the same folder icons.

When Icons are stored in home/.icons they won't appear if I open Thunar as root. You seem to be writing about both folder icons and menu icons so the question is not clear.
