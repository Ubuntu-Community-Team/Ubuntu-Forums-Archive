### Post by Frak on 2008-07-04
Most ubuntu packages are just recompiled upstream packages from Debian. Though, while Ubuntu can use most Debian packages, they are not exactly compatible due to the fact that Ubuntu is built differently than Debian.

And using root only is a breach of security, so a google search would be your best bet.
