### Post by ajgreeny on 2010-06-20
If you mean McAfee Antivirus, you can forget all about that!

1.  It will not run in linux as it is a windows security suite.

2.  It is not needed in linux as there are no viruses in the wild for linux that will do any harm to your system.  Windows viruses can sit on your linux box quite happily without doing any harm.

3.  If you want to use something to check for viruses, in case you inadvertently send them to windows users in, for example, an email, install **clamav** and/or **clamtk** and run it occasionally.
