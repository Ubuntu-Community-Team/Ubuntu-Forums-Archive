### Post by ajgreeny on 2011-03-19
A full desktop edition is no problem on most, if not all netbooks; I don't like the netbook editions of ubuntu and much prefer the standard desktop of all versions that I have tried.   However, I have just moved from a standard ubuntu 10.04 desktop edition on my netbook to the 10.10 desktop version of Lubuntu with its LXDE desktop.

It uses all the same repos as ubuntu, but has a different default range of applications, eg abiword and gnumeric for the office apps, chromium as the browser and sylpheed for email.

It is _*very*_ fast in comparison to the gnome version of ubuntu, but not as customisable, though for a netbook I think it is so much faster that it is worth it.  I have added some of the standard ubuntu applications to mine, such as openoffice (actually libreoffice, just to try it out), firefox, which I still prefer to chromium, and vlc for media, and I am now much happier with the performance of the machine.

It might be worth you giving it a try, if you can find an iso somewhere and avoid the download, though it is much smaller than ubuntu, only 545MB.
