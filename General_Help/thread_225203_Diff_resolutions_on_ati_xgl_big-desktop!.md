	Option	    "DPMS"
EndSection

	Identifier  "ATI Graphics Adapter 0"
	Driver      "fglrx"
	Option	    "(null)"

	Option	    "OpenGLOverlay" "off"

The thing to note is the "Pair Modes" option in the Device section - it tells Xinerama how to deal with the two different resolutions. It seems to work very nicely, including switching between clone mode and big desktop mode depending on which resulution you select in the System-->Preferences-->Screen Resolution dialogue.  I haven't played around too much, but I suspect that a lot of those other options in my Device section are actually unecessary.  I'm just glad it works!:D 

The next step which I will try at some point is to setup XGL/Compiz again, and see how that goes.
