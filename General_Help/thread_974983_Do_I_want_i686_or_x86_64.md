### Post by TeXtonyx on 2008-11-08
64 means 64-bit and some newer computers have the hardware to run 
a 64-bit Operating System (OS). Isn't your computer sort of old? 
If yes, you should run a 32-bit OS which is usually I386. If there 
is no I386 offered then you must use I(5/6)86. You don't want 64.

Another way to tell is when you first boot up watch the black screens 
with white letters (bios screens) which will tell you what kind of 
hard drive you have and how much ram (memory) you have installed. If 
you have 512mb or less of ram it is very unlikely you can use 64-bit.
