### Post by tr333 on 2008-02-09
How can I make Firefox act like Safari and Camino on OS X?  When pressing the [Tab] key in Safari or Camino, it just cycles through all the various form elements (textboxes and buttons).  Doing the same in Firefox cycles through all the links on the page as well, which makes this functionality much less useful for entering form data quickly via a keyboard.

---

### Post by pointone on 2008-02-09
This can be changed with the accessibility.tabfocus setting in about:config.

[quote=http://www.truerwords.net/articles/mozilla/tabkey.html]1 - Focus only moves between text fields.
2 - Focus moves between form elements other than text fields (gee, I bet that's a popular choice).
3 - Focus moves between form elements of all types (buttons, menus, text, etc).
4 - Focus only moves between links on the page.
7 - Focus moves between both form elements and links.[/quote]
