### Post by Autodave on 2018-08-31
4GB RAM should be fine with Xubuntu.  My first thought is that you said that you upgraded: things don't always work as expected on upgrades and that is the reason that I always do clean installs. You might consider doing a clean install after making sure that you have backed everything up to another HD, USB stick, etc.

You may also want to try an Ad Blocker. But having said that, if you are already using an Ad Blocker, try disabling it!  This machine that I am on right now runs much better without the Ad Blocker running.
