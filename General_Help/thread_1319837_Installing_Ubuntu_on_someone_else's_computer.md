### Post by amoeba801 on 2009-11-08
Installing the standard basic Ubuntu does not cause any licensing issues.

The problems start when you want to use multimedia. Some formats are 'restricted'(including MP3s)and royalty payments maybe required. Whether a particular format is restricted is dependent upon Country/Federal and State laws. Read more here - [IMG]https://help.ubuntu.com/community/RestrictedFormats[/IMG]

To get your grandmother up and running, why not get her used to Ubuntu by running it from the Live CD first so as not to impact her PC. Then if she likes it, install it as dual boot util she is used to it.

### Post by julianb on 2009-11-08
For someone who doesn't know how to use windows, it doesn't hurt to set the machine up as dual boot BUT...

you don't have to tell her how to get in to windows if she doesn't know how to do it herself. ;)



For those installing Ubuntu for someone who is accustomed to windows, I'd recommend the following:

set the machine up with a "start" menu in the bottom left, with running applications listed on the bottom task bar. And set them up with no top task bar. Set it up so that, as much as possible, the person doesn't have to change his/her habits in order to find things in Ubuntu. Making the screen look superficially like a windows machine might help too! 

[http://sourceforge.net/projects/auroraproject/](http://sourceforge.net/projects/auroraproject/)
