### Post by tread on 2005-05-22
You will find that most of the time, you don't need to download apps from websites.
Go to synaptic:
System->Administration->Synaptic

search for the required application, and install it. You can even browse the applications available. To increase the number of applications, add more repositories. 

See [the unofficial guide](http://ubuntuguide.org) for info on adding repositories.
