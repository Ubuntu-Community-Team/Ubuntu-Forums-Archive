### Post by TheFu on 2014-01-10
NX.
**FreeNX** on the XMBC box and the **NoMachine NX client** on the remote machine that you are sitting on. Don't expect any video or audio playback through this interface, but for remote admin stuff that **ssh** isn't enough, it works.

For CLI stuff, there is always **screen**.
