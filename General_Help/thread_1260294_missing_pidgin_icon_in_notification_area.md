It's really annoying not being able to hide my buddy list and it's only pidgin having the problem. ### Post by Brandon Williams on 2009-09-07
I run pidgin on xubuntu 9.04 and have never had to do anything special to make the icon show up. Double-check the Interface tab in the preferences dialog to make sure that it's configured to always "Show system tray icon".
