### Post by timgood on 2010-08-25
If you install the Firetray addon, not only will you get notified of incoming emails, but you will also have the option to run Thunderbird at start up. Alternatively, just add Thunderbird to the list of programs that run at startup - System/Preferences/Startup Applications.

Hope this helps.

### Post by Daniel0108 on 2010-08-25
Yea, you just have to open:
System->Preferences->Startup Applications
Then click on Add.
Enter "Thunderbird" as name and "thunderbird" as command.
Then click on ADD :)
Now Thunderbird will open on Startup ;)

### Post by timgood on 2010-08-25
Firetray minimises Thunderbird to the system tray - the icon notifies you of unread emails and maximises Thunderbird when clicked. You can install it from [here]("http://code.google.com/p/firetray/downloads/list").

Quote from developers:

 Features   

Already supported features

- adds tray icon for both firefox and thunderbird

- Show number of unread mails in the tray icon (Thunderbird)

- close button hides to tray

- ability to start minimized to tray

- when restoring if the window is iconified, deiconify it

- Custom tray icon

- Hotkey for Hide/Unhide

- Multimedia keys grabbing to allow playback control in songbird

- Handle mouse scrolls on tray icon
