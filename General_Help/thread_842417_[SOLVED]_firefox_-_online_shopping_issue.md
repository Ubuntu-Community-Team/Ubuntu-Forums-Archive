### Post by bodhi.zazen on 2008-06-27
Typically this is a problem with blocking either java (no script) or cookies.

It is easiest to allow both & complete the transaction.

If you want to be more restrictive :

allow java for session.

Allow cookies for session. Copy the url go to preferences -> cookies -> exceptions -> enter url and click "allow for session" (you can remove it later if you wish).

The tricky thing with cookies is the url usually changes to https

So you may need to allow multiple url for cookies and / or java (the most I have seen was 4 layers).
