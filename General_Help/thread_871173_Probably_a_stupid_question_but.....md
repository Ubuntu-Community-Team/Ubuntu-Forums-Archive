### Post by caljohnsmith on 2008-07-26
That's because .exe files are Windows executable programs. Some Windows programs run in Linux using Wine (you can install Wine using Synaptic Package Manager); to find out which programs work with Wine, you can visit appdb.winehq.org.

### Post by gjoellee on 2008-07-26
Ubuntu is a Linux platform using .deb and .tar and you can use .rpm if you install alien. However .exe is for Windows, but you can run .exe with WINE. You can download wine here: [http://wine.budgetdedicated.com/archive/ubuntu/hardy/wine_1.1.2~winehq0~ubuntu~8.04-2-0ubuntu1_i386.deb](http://wine.budgetdedicated.com/archive/ubuntu/hardy/wine_1.1.2~winehq0~ubuntu~8.04-2-0ubuntu1_i386.deb)
