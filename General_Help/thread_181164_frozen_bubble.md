### Post by Sef on 2006-05-23
These are Dapper files.   They should work for Breezy also.

Here's the files you need to download 

> frozen-bubble (1.0.0-6ubuntu1) [universe]
    Pop out the bubbles !
frozen-bubble-data (1.0.0-6ubuntu1) [universe]
    Data files for Frozen-Bubble

Just type these commands from the terminal:

sudo apt-get update

sudo apt-get install frozen-bubble frozen-bubble-data

Note: If you need packages, a good place to look is here:

[http://packages.ubuntu.com/]("http://packages.ubuntu.com/")
