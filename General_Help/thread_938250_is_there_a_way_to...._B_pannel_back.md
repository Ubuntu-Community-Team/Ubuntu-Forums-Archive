### Post by y-lee on 2008-10-04
If i understand you deleted your bottom panel :o so it appears you must create a new one. Right click an existing panel (you still have the top one right?) now choose new panel and gnome should create a new panel. You can now move this pane to where you want it, say the bottom if it is not created already there. But you will have to re-add all the stuff you had in it before. Right click the new panel and choose add to panel and add what ever you want, trash can and so on. Not you probably want a window list in it, so you can tell what applications you have open. you may also want to right click it after you have added the stuff you want and set the properties to whatever you had before.:)
