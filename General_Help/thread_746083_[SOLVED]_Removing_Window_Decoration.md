### Post by m60dude5 on 2008-04-05
I would say you should find a different theme. Do you use emerald? If so, I would find another theme. I have used many without problems. Unchecking "Windows Decorations" will remove the whole border, so that's not a good idea. I don't think you should disable the border, rather, you should disable the theme that caused the problems. If you mess up and the borders disappear, press alt F2 and type emerald --replace in the box and they should come back. If you don't use emerald, then right click on the desktop, choose Change background, and then click the theme tab. Then choose the default theme and start over. Let me know how this works for you.
