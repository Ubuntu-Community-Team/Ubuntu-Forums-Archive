### Post by mcduck on 2011-03-25
If you are using the driver version from AMD's web site instead of the one from Ubuntu's repositories, then that's not a bug.

The driver is supported, but as it needs some modules built specifically for the kernel version you are using, you need to reinstall the driver after every kernel update.

If you use the driver from Ubuntu repositories you don't need to worry about such things as everything should be done automatically for you.
