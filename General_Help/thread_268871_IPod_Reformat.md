### Post by tubasoldier on 2006-12-04
I know it has been a long time since you posted this question. But here is the answer.

Linux will move files to the .trash folder by default when the delete button is pressed. By going into the .trash folder and deleting them again it will simply send them back to the .trash folder. In order to completely remove the files you will need to do a SHIFT+delete. This will "Shred" the files. Hope this helps someone else in the future.
