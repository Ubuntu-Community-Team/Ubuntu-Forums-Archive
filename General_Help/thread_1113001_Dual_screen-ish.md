### Post by norwoods on 2009-04-01
you get admin privileges by starting a program with sudo or gksu for gnome programs. run in a terminal window:
gksu nvidia-settings
when the nvidia gui interface is running, click:
 X Server Display Configuration. 
click Configure...
click the TwinView radio button
click OK
set up anything you want
click Apply
click Save to X Configuration File
click Quit

### Post by kevmitch on 2009-04-02
**Do** you have an Nvidia card? If not then you probably just borked your xorg.conf. If you have trouble restarting your xserver, you might want to reset your xorg.conf:

```
dpkg-reconfigure -phigh xserver-xorg
```

What you want to do should be possible both with and without an nvidia card, but you don't want to use nvidia-settings if you don't have an nvidia card.

Have you already tried going to Settings->Preferences->Display (or maybe it's screen instead of display)?
