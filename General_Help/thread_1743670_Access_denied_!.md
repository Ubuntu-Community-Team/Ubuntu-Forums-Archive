### Post by bergfly on 2011-04-29
Easiest way to do it is open Konsole and type 

*kdesudo dolphin*

Then enter your password. This will open a dolphin window with permissions to do whatever you want. Drag and drop the file where you need to. Just be careful, running the file manager as root means you could do damage to system files....
