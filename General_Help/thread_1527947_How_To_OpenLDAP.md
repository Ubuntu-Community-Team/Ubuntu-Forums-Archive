### Post by krishnandu.sarkar on 2010-07-09
First of all switch to Ubuntu from Windows. Then open Gedit from Applications > Accessories. Copy that piece of code and paste it in Gedit and save it anywhere with the name backend.example.com.ldif

Otherwise you may do the same from Windows too, copy and paste it in notepad and save it with the name mentioned. Now go to Ubuntu, now open the file from the drive where you saved it.

Though it is recommendable to do the same from Ubuntu.
