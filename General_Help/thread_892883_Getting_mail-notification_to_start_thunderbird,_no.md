### Post by yadayada2 on 2008-08-17
Hi there,

you know there is this mail-notification package for the gnome panel. I really like it and already have changed it to support ssl, too. 

However, I don't know how to tell this app to use thunderbird instead of evolution. The project site about this little panel applet explicitly states the support of Moz apps.

But there seems to be no way to make it start thunderbird when loading the mail reader. Instead it always wants to launch evolution.

Do you have a hint for me?

Thx!

### Post by BUSHYBOB on 2008-08-17
In Preferences -  preferred applications is the default mail client thunderbird?

---

### Post by yadayada2 on 2008-08-17
Thank you. I thought about this the same time you posted your idea and yes, you were right. In the preferred applications the wrong mail reader was selected. That solved it already.

However, thanks for the hint!

### Post by ajgreeny on 2008-08-17
If you have more than one desktop environment, you may need to make sure thunderbird is the default in both, in order for this to work all the time.
