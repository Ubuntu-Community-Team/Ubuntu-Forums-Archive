### Post by mb_webguy on 2009-02-19
> **Lingonet said:**
> Hello, I have a question!

Can I install Ubuntu on any laptop I want? Is there kernels for every computer or what is the deal with it?

Thank you!

The primary issue with installing Ubuntu on a laptop is that laptops tend to have wireless network cards, some of which (darn you, Broadcom!!!) aren't natively supported in Linux, which requires you to use the Windows driver.  So make sure you have a network cable handy in case your wireless card doesn't work out of the box and you need to go online to install updates and download the Windows driver.

Other than that, installing Ubuntu on a laptop shouldn't be any different than installing it on a desktop.

### Post by donkyhotay on 2009-02-20
There aren't any kernels specific to laptops. Kernels are generally specific to the processor such as x86 and x86_64 (most common types) but also PPC, SPARC and all the various other types that I can't think of off the top of my head right now. There is nothing special about laptops that require special versions. As mentioned sometimes wireless cards don't have the necessary linux drivers and since many laptops have wireless cards it becomes a common problem but this is no different then if a desktop has a wireless card too. I've had linux on my laptop for 2-3 years now and (fortunately) my wireless card is one that 'works out of the box' and never given me an issue. Honestly if it weren't for the fact my desktop is 64bit and my laptop is 32bit I would use the exact same disc to install on both systems. The only way to know if your wireless card will work is to either search online for other people with the same card that use linux or just try it and see what happens.
