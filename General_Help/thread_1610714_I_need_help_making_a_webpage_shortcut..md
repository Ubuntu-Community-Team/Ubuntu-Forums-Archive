### Post by HiImTye on 2010-11-01
try prism
```
sudo apt-get install prism
```
it runs whatever web page you tell it to with whatever options you need. you'll find it in the internet menu and running it will open up a configuration window. put in the webpage and the options and make sure to check create a desktop shortcut
you can change the icon and such from there

firefox does not have a --fullscreen option
