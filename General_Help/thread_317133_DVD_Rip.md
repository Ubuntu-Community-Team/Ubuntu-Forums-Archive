### Post by viciouslime on 2006-12-12
IF you want an exact copy so it can be burnt later then there is no need to use ripping tools, there main purpose is to re-encode the dvd so it can be burned to smaller discs like single layer, or cds, or whatever.

If you want to be able to burn to dual layer then you can either create an iso of the file on your laptop, this can then be mounted and played as normal whenever you want to watch the film, but it can also be burned to a dual layer disc and will give you an exact replica of your original dvd with no loss of quality at all.

Or, you can just copy the vob files from the dvd, into a folder on your harddrive. You'll be able to play these back too, however, burning them back to a disc will be a little more complex because you will need to get the folder structure right, however, using an app like gnomebaker will make this task very easy.
