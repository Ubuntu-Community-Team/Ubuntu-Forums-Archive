### Post by deathadder on 2011-04-07
Some would argue that having a login prompt is a mute point, if someone has physical access to your machine then security (passwords etc) can be broken. It's shockingly easy to do, just think about using a live CD to get root access for instance. Others would argue that it helps.

Personally I sit on the fence on this one, but would say that you should ensure you require a password to install / remove software or access any keychain like software.

It really boils down to "do you trust those around you" if the answer is no then you really shouldn't be allowing auto login. :)
