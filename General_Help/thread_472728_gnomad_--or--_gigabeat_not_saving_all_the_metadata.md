### Post by trippinnik on 2007-06-13
Not sure which gigabeat you have but it probably uses MTP.  There's a Linux project [http://libmtp.sourceforge.net/](http://libmtp.sourceforge.net/) and gnomad uses this, you could also try Amarok as it has support and you can try getting the newest version from CVS and also email the devs.  I have a gigbabeat p20 (a flash based one only release here in Japan) but gave up because I couldn't get it to work without Windows and now the Battery is dead cause it wouldn't even charge.. Anyway it's good that yours isn't locked out with authentication stuff like the Zune, you can probably get it to handle the missing Meta data.
