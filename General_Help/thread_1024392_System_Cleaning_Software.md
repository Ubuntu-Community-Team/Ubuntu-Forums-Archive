### Post by deanjm1963 on 2008-12-29
Is there anything specific that you think needs cleaning?

Linux is not Windows, it cleans up after itself very well, even after a power outage.

Temporary files are deleted on each boot and software installation/de-installation doesn't leave "fragments" like Windows does, although it can leave "orphan" files that can be safely removed by Synaptic but 99% of the time they're very tiny in size and removing them makes no difference in performance.

Hope this helps. :D
