### Post by WorMzy on 2011-04-30
Chalk another vote up for Arch. It has a [damn good guide]("https://wiki.archlinux.org/index.php/Beginners%27_Guide") for newbies which walks you through installing the core system, setting up all the main config files, then setting up networking, sound, and the graphical interface, all from the command line. Other distros probably have something similar, but this is the only one I can personally vouch for.
