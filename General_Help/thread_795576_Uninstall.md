### Post by ibuclaw on 2008-05-15
The most direct way to remove Ubuntu would be to format the partition that Ubuntu is installed on, then Boot up your Windows CD and choose the Recovery option (that should only just fix your MBR to make it Windows only again).

But if you want to install Xubuntu, just install it over your current version of Ubuntu.
Backup your home folder first though, as the Ubuntu filesystem will be erased in the process.
