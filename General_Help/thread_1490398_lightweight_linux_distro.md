### Post by egeezer on 2010-05-24
I've tried a few lightweights.  Results:
SliTaz: super-compact, great as a dual-boot to rescue anything else you boot that goes wrong.
Puppy: same thing, but just run the LiveCD because you must run as root-only
Lubuntu:  I got a good burn, almost everything seems to work out, but it has no ability whatever to access other partitions.
antiX:  not very stable, caused a few lockups/freezes/etc.
TinyMe RC1: very pretty, lot of features but somewhat limited versions of each (e.g. thin Abiword)
