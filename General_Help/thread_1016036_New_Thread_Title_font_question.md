How do I get the text, that I use in my thread title, to appear in a "BOLD" font when it is listed on the forum?

---

### Post by jerome1232 on 2008-12-19
Thread's that you have already visited and haven't had any new replies since you have visted them appear in normal text while threads that you haven't visited or have had replies since you have visited them appear in bold text.
