### Post by Mazza558 on 2008-05-24
I've been hard at work putting together a vastly improved version of FutureLooks with an improved default theme and many, many other new features.

[IMG]http://img137.imageshack.us/img137/2057/futurelookswh8.jpg[/IMG]

New stuff in this release:

- Improved the original theme to make titlebars legible everywhere. This meant that I had to lower the transparency somewhat, but I wanted a better balance of usability and looks.

- Added 4 variations of the original futurelooks theme, including blue, green, silver and "glow", an experimental theme which replaces shadows with auras around each window.

- Added FutureAurora, which is based on the Aurora theme engine for everyone who prefers that to clearlooks. Also included is the first dark variation to the pack, combining Aurora Midnight and a dark version of the emerald theme. Note that AuroraMidnight has problems rendering buttons on Firefox with white web pages (e.g google).

- Added FutureMurrina for people who prefer the Murrine Engine.

- Added FutureHuman (smartboyathome's creation) which fits better with human murrine or the old Gutsy human theme.

- Added a matching Login Screen

- Added more wallpapers

- Added instructions to install Avant Window Navigator

- Documentation on every aspect of the theme pack

- Expanded credits to take into account everything new I've added.

_______________________________

Find the release on the new Google Code project page for it:

[http://code.google.com/p/futurelooks/downloads/list]("http://code.google.com/p/futurelooks/downloads/list")

Discuss the theme either here or in the old FutureLooks thread, here:

[http://ubuntuforums.org/showthread.php?p=5034221]("http://ubuntuforums.org/showthread.php?p=5034221")

Screenshots of the new themes are attached - enjoy!

Just port the code from the Murrina engine. Aurora was just an unnecessary fork from Murrina, and thus should be code-compatible. The only problem right now is that you need to recompile each GTK program with a different patch to enable transparency.

> **Mazza558 said:**

Sure you can. Look at the Ubuntu Human package (apt-get source it), and then check out how it is built. It simply puts it in /usr/share/themes and /usr/share/icons folders in order to get it to install. Replace the theme folders with your theme's folders and then build with debhelper. :)
