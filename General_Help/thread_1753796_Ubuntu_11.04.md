### Post by 3rdalbum on 2011-05-09
1. You've been using Ubuntu for a while. Ubuntu doesn't just test itself, you know - a lot of us were testing it for months before release and reported all the bugs we found, and now I at least can't observe any bugs in 11.04. Why not help test the next version, 11.10?

2. If you're after speed, I don't believe OpenSUSE is where you'll find it.

3. Have you tried some basic speed-troubleshooting techniques, such as looking in System Monitor to find if there's any programs hogging the CPU?

4. If you believe it's easy to write bug-free software, have you ever tried? It's not easy to write efficient software, and it's not easy to write non-buggy software. In fact, it's not easy to write software at all. Maybe try it some time and you might be more appreciative of what the Ubuntu team does. And test Ubuntu, next time, before release.

5. Can't see any help threads that you've started, apart from the one you solved with chmod? Maybe you should ask about some of the issues you've been having.
