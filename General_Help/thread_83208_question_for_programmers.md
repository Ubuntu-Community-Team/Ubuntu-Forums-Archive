### Post by GoldBuggie on 2005-10-28
KDevelop is really nice...it handles many languages not only c++. If you want to build nice GUI programms KDevelop works excellent with qt3designer(which is a gui designer).

Writing many programms with kate and such is also nice but the good thing with kdevelop is that it works nicely with debugging, source version control and memory leak check. Overall a nice tool that doesn't go away from the nice clean feeling of working with kate but rather adds some extra stuff that just makes work easier.
