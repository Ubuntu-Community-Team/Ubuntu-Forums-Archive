### Post by Titan8990 on 2009-04-19
I would like to start off by saying that on dual boot systems, Windows should be installed first but it is not a requirement.


The issue you are referring to is not a Linux issue. It is a common problem with computers that ship stock with Vista, and do not contain drivers for XP.

So you should ask yourself this:

Did this computer come with Vista?

Am I now trying to install XP?


If the answers to these questions and yes, you should check out a Windows support forum.
