Unless Xubuntu plays tricks on you, there must be an installer running without you knowing it.
Start system monitor and in menu View choose: All processes.
On top of the list click "Process name" to sort the names alphabetically.
Then start looking down the list. Things like synaptic, aptitude, gdebi, update-manager, gnome-apt-install.
There has to be one.
