### Post by taurus on 2008-12-05
192MB of RAM is not enough memory to run the LiveCD (desktop CD).  So if you want to install Ubuntu on that laptop, try the alternate CD.  Better yet, download the Xubuntu alternate CD because Gnome will be turtle slow on that laptop if you can manage to install it.

### Post by ibutho on 2008-12-05
Did you attempt to use the live cd to install Ubuntu? If so, then your system does not meet the minimum requirements (low ram) to run the live cd and installer. You probably should try using the alternate cd to install Ubuntu, or look at something like Xubuntu. Another option would be to upgrade the RAM first and try installing using the live disc.
