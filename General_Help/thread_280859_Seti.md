### Post by Pelekophori on 2006-10-22
Boinc is the replacement for the "classic" seti@home program.  I believe that the classic version is no longer supported, ie the seti project will not send data to, or process data from, the classic version.  

A shame as far as I'm concerned because the classic version was a model of just works simplicity, whereas I have never been able to get Bonic to work in either Windows or Ubuntu.

See here for more info: [http://setiathome.berkeley.edu/~mattl/facts.php]("http://setiathome.berkeley.edu/~mattl/facts.php")
