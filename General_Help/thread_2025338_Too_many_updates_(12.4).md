### Post by Cheesehead on 2012-07-14
> **johnniea said:**
> My Update Manager keeps wanting me to download too many updates.

It's offering only the updates you told it you want. To change that, go to Software Center --> Edit Menu --> Software Sources.

> **johnniea said:**
> Some of these are for programs I don't use or have uninstalled.

Updates are only offered for packages you have installed. You can control what you have installed using Software Center.

### Post by irv on 2012-07-14
When you first install Ubuntu it will update your system files and there will be a large amount of files. After that it will just keep them up to date. It may be just a few or at times many more. When a kernel update comes along it will require a reboot but you can wait. And yes, it only will only update what you have installed. Don't forget there are many support files for the programs you are running. (lib files and other). When you install an app it comes as a package which contains other files.
It is a good practice to keep your system up to date because some of these updates are for security and bug fixes.
