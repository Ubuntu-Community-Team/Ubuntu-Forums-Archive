### Post by mikewhatever on 2008-03-17
I get this behavior only if I click on one of the Desktop Switcher work space in the bottom right corner. By default, two work spaces are enabled, so try to make sure you click on the page first, then scroll. If you only want one work space, right click on the Desktop Switcher, then Properties and reduce the number of workspaces to 1.
