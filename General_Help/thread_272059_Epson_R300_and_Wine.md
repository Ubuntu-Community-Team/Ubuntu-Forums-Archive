### Post by tomcat1965 on 2006-11-29
Go to Applications>Graphics>Gimp.From file menu select open then browse to your background image select open.
go to Xtns>script-fu>misc>cdlabel>ok 
Now go to the background image and select script-fu>utils>cdmask.
This will overlay a CD pattern over the image.
Now select print,set up your printer and you're done.
Always use GIMP to print CD's as printing from image viewer tends just to print a small section of the image on CD
