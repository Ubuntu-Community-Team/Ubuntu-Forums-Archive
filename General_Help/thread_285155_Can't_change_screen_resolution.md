### Post by yaa13 on 2006-10-26
1680*1050@60Hz?
In addition to that you have made:
Type in terminal
```
user@myhost:~$ sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bckp
user@myhost:~$ gtf 1680 1050 60

  # 1680x1050 @ 60.00 Hz (GTF) hsync: 65.22 kHz; pclk: 147.14 MHz
  Modeline "1680x1050_60.00"  147.14  1680 1784 1968 2256  1050 1051 1054 1087  -HSync +Vsync
```
Insert row
```
Modeline "1680x1050@60.00"  147.14  1680 1784 1968 2256  1050 1051 1054 1087  -HSync +Vsync
```
in "Monitor" section (xorg.conf) and remove other lines in this section containing 1680x1050 (if existed).
Restart X server.
If such configuration will appear wrong, return to initial
```
user@myhost:~$ sudo mv -f /etc/X11/xorg.conf.bckp /etc/X11/xorg.conf
```
