Does anyone know of a ubuntu/linux equivalent for the MicroApache server?

[quote="microapache.amadis.sytes.net"]
A Portable, USB-Friendly, Micro-Sized Apache Server for Windows

MicroApache is a microscopic and heavily cut-down version of the latest, standard, GPL Apache server developed initially to meet a FloppyWeb Challenge to run Apache from a standard 1.44Mb floppy disk. My thanks to James Greenwood (no website, unfortunately) for coming up with the original idea. I have further reduced the size   by another 50% and now it weighs in at about 350kb for the most-minimal, fully-working version of Apache with a legacy version of 1.3.x weighing-in at only 170Kb.
