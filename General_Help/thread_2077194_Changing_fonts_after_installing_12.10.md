I just installed 12.10 and MyUnity is gone...
I was able to change the size of the icons in the dash, but I don't know how to change the size of the fonts in various locations in the UI.

MyUnity allowed me to do that, but it is no longer available.

### Post by raedmohammadi on 2012-10-28
install gnome-tweak-tool :)

```
sudo apt-get install gnome-tweak-tool
```

then you can change the font and size of it.
