### Post by ubuntu27 on 2012-05-09
Are you using an English (US) keyboard? Then change your keyboard layout to English (US International with dead keys), and you will able to type accents and other special characters using ALT+Letter

System Settings------> Keyboard Layout-------> + ----------->English US International with dead keys----->Add


Example:

Alt+n=Ăą
Alt+e=ĂŠ

etc

Yes, you can have more than 1 layout. You can change it anytime in the Indicators (upper right hand corner)

### Post by fballem on 2012-05-09
To change keyboards, click on the keyboard indicator on the top panel. When you click on it, a mini-menu will be displayed. Make sure you select US International with Dead keys. The text beside the indicator will change (my changes to en2).

Once the correct keyboard is selected, then try typing the tilde (~) first. Nothing will be displayed. If you then type the 'n'key, you will get Ăą displayed. As for the Alt key, remember that there are two of them - use the one on the right of the spacebar.

By the way, if you want to type a tilde (~) without the letter, then type the tilde followed by the spacebar.

Hope this helps,

### Post by haplorrhine on 2013-02-11
Alternative:

Tools -> AutoCorrect Options... -> Replace

In Replace, set certain combinations of characters to be replaced with the Spanish letters.  On the upside, you can make it an easy combination of two keys.  On the downside, it only "replaces" the text if there are spaces on each side of the text to be replaced.
