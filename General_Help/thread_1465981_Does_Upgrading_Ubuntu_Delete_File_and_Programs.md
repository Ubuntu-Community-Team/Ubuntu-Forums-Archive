I want to upgrade to ubuntu 10.04 and was wondering if upgrading deletes all programs, files, and drivers that I have already installed.  Thank you.

### Post by NMFTM on 2010-04-29
It won't "delete programs", but it will override old versions of applications with their respective new versions. Some settings may be lost. Most likely no user data will be lost either, but since computers are very complex anything can happen. So backup if your unsure.
