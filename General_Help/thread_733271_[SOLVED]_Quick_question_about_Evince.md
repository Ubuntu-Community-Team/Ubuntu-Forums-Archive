### Post by oddworld on 2008-03-23
I really like using evince for my PDF files, but is there any way evince can scroll through documents like acrobat's "hand tool".
Is there any tool in evince that allows you to "grab" the page and move it around, or are you restricted to using the scroll bar?
thanks!

---

### Post by Dennis on 2008-03-23
Goto: Edit - toolbar - drag and drop **continuous** onto the tool bar - click it - use the scroll wheel held down to move with the hand continuously through the document.

---

### Post by oddworld on 2008-03-23
Ah gotcha, the trick is to use the middle mouse button held down.
thanks!
