### Post by lordyosch on 2012-05-02
Serviio is pretty easy to install.

1. Download it
2. Extract it somewhere -home folder maybe?
3. To self start on boot add a line to the startup applications list
(give it a name, browse for the serviio.sh file in the 'bin' folder of serviio directory)
4. Either double click on serviio-console.sh and choose open in terminal, or open terminal navigate to serviio/bin/ folder and type serviio-console.sh
5. using the options within serviio console build a media library

Now you're pretty much up and running!
