### Post by akand074 on 2010-12-17
> **WarrenSH said:**
> I would like to be able to right click on any type of a file & be able to add it to a 7zip archive or what ever the .7z is called you know like a .rar or .zip. Also I want 7zip to be the default program that opens all compressed files.

7zip doesn't have a GUI front-end for Linux. 7zip is only designed as a CLI application for Linux. Luckily Nautilus and Archive Manager have support for it. So if you are trying to say that you want the actual 7zip application to do all your archiving/extracting for you, it can't be done. What's your issue with Ubuntu's archive manager? It can archive/extract all file types as long as you have the libraries installed.
