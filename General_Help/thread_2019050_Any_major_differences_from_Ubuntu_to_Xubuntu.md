In a nutshell, xubuntu's DE is Xfce (as opposed to Unity for ubuntu). Some of the default applications are different, configuration options/applications are different, look-and-feel is different, but the base is the same. As for tips, read through the xubuntu sub-forum here for some ideas.

> Also, this is a random question, but is there any way to make windows snap to half screen?  Because when i try it just switches to the next workspace. 
This feature was added in Xfce version 4.10. Xubuntu 12.10 will be the fist version of the distro to officially support 4.10. In the meantime, if you want to run 4.10, there is a PPA available for it. See: [http://www.webupd8.org/2012/05/install-xfce-410-in-xubuntu-1204.html]("http://www.webupd8.org/2012/05/install-xfce-410-in-xubuntu-1204.html")

To get it to also snap on the left and right borders, you need to disable (uncheck) the "Wrap workspaces when dragging a window off the screen" checkbox in Settings Manager -> Window Manager -> Advanced Tab.

Another difference is that xubuntu is meant for more lighter applications and is lighter overall. So if you have a system with pretty dated specs, xubuntu is meant for it due to it's lightweight trait.
