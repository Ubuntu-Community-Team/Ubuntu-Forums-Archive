### Post by aloshbennett on 2009-07-25
Any .jpg, gif or .png can act as an icon. Using the browse button you get when you click the spring, open a folder containing the pictures. It will load all the pictures in the folder and you can select from them.

Yes, you can use gimp to create new icons. You could design new ones (little tough though) or scale/crop a picture to make an icon out of it.
