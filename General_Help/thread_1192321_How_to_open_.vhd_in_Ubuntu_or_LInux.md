### Post by cariboo on 2009-06-20
You will have to install virtualbox in order to open your disk image file. Virtualbox is in the repositories, so you can use Add/Remove Programs or Synaptic Package Manager to install it.

If you need usb and want to be able to use sound and adjust the video you will need to install Virtualbox puel which is available [here]("http:///www.virtualbox.org/wiki/Downloads"), just download the application, then double click it, and gdebi will install it for you.
