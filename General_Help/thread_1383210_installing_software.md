### Post by NFblaze on 2010-01-17
Well, if your using the latest vesion and are a noob you can use the easiest method which is the **Ubuntu Software Center** which is listed in the Applications menu. Search for keywords for what you need ex. "ID3 Tag editor"

After, that the next easiest method is probably Synaptic Package Manager. Which is listed under **System > Administration**. Search for keywords for what you need.

There is also thru the terminal using the **apt-get** command. You should learn about that command. You can look it up by opening the terminal and typing man apt-get or better yet googling apt-get tutorial.

Another way is by compiling source code, although chances are very rare that you will ever need to do this. It is also tedious and a bit on the technical side.

### Post by bug67 on 2010-01-17
Ubuntu installs software a little differently than other OSs.  It (and other Linux distros) use repositories to store and add software for download to your computer.  There are several ways to do this.

1.  _Software Center_.  In the upper left hand panel, select it from the "Applications" drop-down.  From there, you can find a list of available software to add to Ubuntu.

2.  _Synaptic Package Manager_.  Again in the upper left of the panel.  However, this time select from the "system" drop down.  You'll need to enter your administrative password to use Synaptic.  In the search box, type in the name of the software you are looking for.

3.  _Command Line Interface (CLI)_.  This is beyond the scope of my knowledge of Linux.  Usually, whenever I've installed software from the CLI, it's been a matter of copy/pasting code or some command involving the phrase "apt-get install name of software."

4.  You can also download .deb packages from various web sites that you can manually install but this is *way* beyond my scope of abilities.  Google usually can be a very helpful tool in these cases.  Just a matter of "How to install Name of Software in Ubuntu."

Hope some of this helped.  These forums are a great resource.  I'm sure others with more experience will chime in.
