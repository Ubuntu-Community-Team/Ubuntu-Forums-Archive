### Post by duanedesign on 2010-04-07
I just tested this in Lucid.

Open Office writer from the menu and create a file with a 'save as' from the File menu. This file does not show up in Places > Recent Documents. However if you reopen it with a double-click in a Nautilus window or on the desktop it will show up.

I know a lot of people were hoping this would be fixed by Lucid :(
