### Post by Nick Payne on 2011-03-14
Since installing the update to acroread 9.4.2, when I open a PDF portfolio I no longer see the initial page showing the files comprising the portfolio with a thumbnail of the first page of each file. If I open the same portfolio using the Windows version of Adobe Reader 9.4.2, then I do see the initial page correctly, so the problem seems to be with the acroread version available through the Ubuntu partner repository. The portfolios displayed correctly prior to this update.

I'm running Ubuntu 10.04 amd64, and a portfolio which demonstrates the problem is at [http://www.users.on.net/~njpayne/bikestuff/SmokersGap.pdf]("http://www.users.on.net/~njpayne/bikestuff/SmokersGap.pdf") (64Kb).

AFAIK, Acroread is the only option for opening and using PDF portfolios. I'm not aware of any other PDF reader that can handle them.
