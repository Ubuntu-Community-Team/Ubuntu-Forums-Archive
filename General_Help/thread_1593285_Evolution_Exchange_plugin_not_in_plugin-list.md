### Post by todak on 2010-10-11
Start Evolution and invoke the help section by pressing F1. Scroll down to section 6 which is entitled "Connecting to Exchange Servers". It gives detailed information on how to set up an account and connect to an Exchange server, though you can configure only ONE account for Exchange in Evolution. Hope this helps. :)

### Post by todak on 2010-10-11
It won't show up in the list of plugins. You have to create a new account using the new account wizard and it will give you the option to use an Exchange server and to set the Exchange parameters. That is where the plugin is accessed and configured.
