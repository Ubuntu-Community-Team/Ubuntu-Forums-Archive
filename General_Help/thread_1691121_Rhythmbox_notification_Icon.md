### Post by ajgreeny on 2011-02-19
It is now built in to the indicator applet, which is where the volume icon is placed now, as well.   However, this is an upstream gnome change, not specifically Ubuntu, and as far as I'm aware, there is no way to change back.

As far as I can remember there has been no separate icon in the panel since 10.04, certainly it is the same in my 10.04, and always has been.  I can't remember how it was in 9.10, but I think that was the previous way, with a completely separate icon in the panel.
