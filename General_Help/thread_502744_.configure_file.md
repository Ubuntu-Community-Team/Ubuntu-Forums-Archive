### Post by avik on 2007-07-21

Do you want to make .deb file for distribution?  As far as I know, there's a guide [here]("http://www.debian.org/doc/maint-guide/") for that.

If you want to make .deb just so you can manage the program using apt-get or synaptic, then you can use checkinstall, which is in the repos. Just use sudo checkinstall when you would use sudo make install. The resulting .deb file is  adequate for personal use.
