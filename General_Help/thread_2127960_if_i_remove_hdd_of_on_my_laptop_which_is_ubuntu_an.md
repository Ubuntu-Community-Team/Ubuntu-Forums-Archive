### Post by Kirboosy on 2013-03-21
Windows will most likely not be able to read the hard drive. (Because it  doesn't support EXT3/4) I know there are open source tools out there  that enable Windows to read EXT but I never had good luck with them.

I  would recommend grabbing a liveCD of Ubuntu and booting the Windows PC  that you want to back the files up to off the CD/USB stick. Once booted  go ahead and copy the files accordingly. You could also Liveboot the "broken" computer and back up across the network if you have network storage setup.

Good luck and let me know if you need any additional help.
