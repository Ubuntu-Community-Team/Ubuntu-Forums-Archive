### Post by The Warlock on 2005-12-24
[QUOTE=Gooks]Is there an anti-virus or anti-spyware for linux?[/QUOTE]
There is no spyware for Linux, so an anti-spyware tool would be sort of useless, wouldn't it? :D

As for viruses, there are very very very few that affect Linux, and virtually none that will affect a default installation. The only one in the wild right now (IIRC) affects users with an old version of PHPNuke set up. There is a program called ClamAV that will catch what Linux viruses there are, as well as Windows viruses so you don't accidentally e-mail them to your friends.
