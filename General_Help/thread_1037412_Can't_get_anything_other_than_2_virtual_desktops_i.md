Okay, I'm using Compiz-Fusion with KDE 4.1. I ran compiz --replace, and everything seems to be working fine, but I can't use anything but 2 virtual desktops. I went into the Desktop Settings dialog and tried to change it to 4, but it just stayed at 2. I also tried 1 to confirm that it couldn't use anything else. Is there something else I need to do?

BTW, I installed Compiz-Fusion by installing the compiz-kde package.

---

### Post by Ianprime0509 on 2009-01-11
Okay, nevermind, I fixed it. To anyone else who's having this problem, just go to ccsm, General Options, Desktop Size. Now, set vertical size to 1, and horizontal size to the number of desktops you want.
