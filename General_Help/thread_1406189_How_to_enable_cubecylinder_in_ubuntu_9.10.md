### Post by Luisnux on 2010-02-13
Install simple-ccsm (also known as simple compiz-config manager) and then open it and go to the desktop tab in simple-ccsm and set the desktop columns to 4.  Please keep in mind that you should have the extra effects enabled.  If you would like for the cube to look more like a cilinder you will have to go to "compiz config settings manager" (notice that this one is different from "compiz-config manager"), then select the option that says cube reflection and defformation and that should do it, also you would need to select the 3d windows options to have your windows pop-out of the cube like in the video.  It should look like the cube the on the video.
