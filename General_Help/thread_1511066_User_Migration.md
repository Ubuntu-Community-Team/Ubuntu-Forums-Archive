### Post by 3rdalbum on 2010-06-16
Copying across the home directories of all the users will keep all this. Also, if you're fresh-installing a new version of Ubuntu, as long as you don't opt to format the disk you'll be asked if you want to preserve those home directories (it's the "migration assistant" window).
