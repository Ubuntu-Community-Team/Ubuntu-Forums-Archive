### Post by Nonno Bassotto on 2007-09-17
I'm using a laptop and the volume settings work well. The only problem is that I'd like to be able to use the multimedia keys on my laptop to set the volume more quickly.

The multimedia keys do what they're supposed to: they raise and lower the main volume. But my laptop is connected via the phones output to an external system. So I'd like to leave the main volume (which only affects the laptop speakers) always off, and have binding for the multimedia keys to the phones volume.

Can I do this? (by the way, I'm using compiz, I know this affects in some way the key bindings).

---

### Post by Nonno Bassotto on 2007-09-17
Never mind. I found how to do it. I'll write it here in case anyone has the same problem.

Just go in System->Preferencies->Sound and select Headphones as the default device.
