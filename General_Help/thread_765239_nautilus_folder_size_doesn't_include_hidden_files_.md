### Post by ogcub on 2008-04-24
Hello,
if you want to know how much space folder takes on disk, you usually right click on it and choose properties. then you can see folder size. however it doesn't include hidden files/folders. So if you have folder with one big hidden file, nautilus will show it's size as zero.
Is it a bug, or simply another gnome usability/simplicity nonsense?

---

### Post by retrow on 2008-04-24
If you can notice the difference in size and track it back to a hidden file, it defeats the whole purpose of making it a 'hidden' file to begin with. If you want to see the hidden files and the size of folder with hidden file sizes included, all you have to do in Nautilus is:

View >> Show Hidden Files
