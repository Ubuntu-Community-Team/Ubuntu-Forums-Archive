### Post by skiamakhe on 2008-05-14
Well, as you might imagine, you'll get a ubuntu-biased response here. That being said, both fedora and ubuntu provide live CDs, meaning you can boot to the CD and run the operating system without changing your vista installation at all. I would recommend you start there. The thing I would be most concerned with is to make sure all your hardware, especially video and wireless networking, is working like it needs to in your distro of choice. 

If you don't already have an ISO burning solution for windows, here's one I've used for years:

[http://isorecorder.alexfeinman.com/isorecorder.htm](http://isorecorder.alexfeinman.com/isorecorder.htm)

Make sure you get the right version (v3) for vista.

Keep in mind that booting and running a Live CD will be really slow because reading CDs is much much slower than reading a hard drive.

### Post by bingoUV on 2008-05-14
I use fedora but I would not recommend it for you. Reasons
1. If you get fedora 9 today (it was released yesterday) you will get maximum 18 months before it will be unsupported. No updates. Meanwhile fedora 11 will be out with way more features. Regular upgrades are quite frustrating for new users.
2. It is quite polished but a bit less suited for inexperienced linux users. Very new features, not tested quite well appear in fedora. It is ahead of Ubuntu on features (mostly).

Go for 8.04 Ubuntu. It is supported for about 3 years. User community is great. Appropriate for switchers fresh from windows, yet it does not try to copy windows blindly.
