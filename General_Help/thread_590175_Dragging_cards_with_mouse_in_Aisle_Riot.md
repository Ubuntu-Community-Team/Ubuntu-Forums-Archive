### Post by popch on 2007-10-24
Using Gutsy Gibbon (but not using Feisty Fawn)

I observe strange behaviour of the mouse when dragging cards in AisleRiot. The game is Spider, if that matters.

Dragging a card behaves erratically. 

Sometimes it works as expected. The cursor changes from open hand to fist and the card to be dragged moves when and where it is supposed to.

Other times, dragging works with a delay. This causes an arbitrary playing card to be moved, i.e. the one which holds the cursor at the time it changes from open hand to fist.

Still other times, the cursor changes from open hand to fist and back again when clicking, within fractions of a second. The cursor will move when moving the mouse, the cards will not move.

Lastly, the cursor fails to change shape when clicking.

I have the impression that the behaviour changes from good to worse while playing some time. At first, all works as expected, in the course of a game it becomes quite difficult to play the game, some moves requiring five or more clicks until the cursor changes shape, remains changed and the card moves.

If it's only in solitaire, I could possibly live with the problem. If it affected other programs, that would be a show stopper. I have not observed such behaviour in other programs until now.

Any advice?
