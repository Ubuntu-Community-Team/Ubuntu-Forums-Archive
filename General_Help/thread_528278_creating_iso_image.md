### Post by lgambett on 2007-08-17
Ok, maybe that on your Windows Winrar is set as the default application to open files with the extension .iso. The file you downloaded must have the extension .iso (ie the name of the file must be ubuntu-7.04-desktop-i386.iso). You have to burn the iso image on CD without opening it, then reboot your PC with the newly burnt CD. Ubuntu will start, and the install icon will appear on your desktop.

No problem / errors should arise at this stage. Consider the possibility to obtain a free Live CD.
