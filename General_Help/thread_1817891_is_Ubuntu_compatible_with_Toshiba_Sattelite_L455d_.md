### Post by 3rdalbum on 2011-08-03
> **rememberme22222 said:**
> I want to cross over to linux with my brand new (old stock) Tosh Sattelite
L455d s5976, but am leery after an awful experience with Knoppix on my last
Toshiba. 
 
Are there modem issues still going on? Knoppix I recall couldnt use the modem. 
This current laptop is using a plugin USB modem. AMD64 32 bit .

If you used Knoppix, I'm guessing it was a long time ago. Linux changes so quickly that most Linux experience you had a year ago is not applicable anymore.

What sort of modem? If it's a dialup modem, it's probably not likely that it works. Most dialup modems are "winmodems", every one requiring completely different drivers that only work on Windows. Because almost nobody is on dialup anymore, there's not much work into getting these running on Linux. I guess this is the exception to the rule I stated above :-)

If it's an ADSL or cable modem, plug it in through the Ethernet port and it will work.

If it's a mobile broadband modem or wifi adapter, it depends on the exact model number, but it will probably work (and if not today, then some point in the future). 
 
> Im reading about the heating fan program being off...is this right?

Fan control is done via your computer's BIOS, not through Ubuntu unless you install extra software. So, basically, you've been given some wrong information at some point.
 
> Id like to know before I order the DVD if there are known problems or not.

There's not much information on the Toshiba website, but I see no problems with what's listed there in terms of hardware. Of course it depends on exactly what's on the motherboard, but I'd be reasonably confident that Ubuntu will boot and be useful.

Just a note: The DVD version of Ubuntu doesn't contain any more software than the CD version; it only contains support for more languages. If the DVD is more expensive, then just buy the CD - the extra language packs can be downloaded in the unlikely event that you need them.
