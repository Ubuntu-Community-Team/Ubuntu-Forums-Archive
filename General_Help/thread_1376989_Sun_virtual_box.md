### Post by nexes128 on 2010-01-09
First open a terminal (Application -> Accessories -> Terminal)

Then do the following:

```

For Karmic 32-bit
wget http://download.virtualbox.org/virtualbox/3.1.2/virtualbox-3.1_3.1.2-56127_Ubuntu_karmic_i386.deb
dpkg -i virtualbox-3.1_3.1.2-56127_Ubuntu_karmic_i386.deb

For karmic 64-bit
http://download.virtualbox.org/virtualbox/3.1.2/virtualbox-3.1_3.1.2-56127_Ubuntu_karmic_amd64.deb
dpkg -i virtualbox-3.1_3.1.2-56127_Ubuntu_karmic_amd64.deb

```

Then just follow the on screen instructions
