### Post by gordintoronto on 2009-09-10
> **sopadeajo said:**
> New to Ubuntu, downloaded GParted application with Add/remove but i do not know how to start it.Any help?

Accessories, Terminal, sudo gparted

Quite often, you can run a downloaded application by typing its name into Terminal, but sometimes the program name is different from the package name.

In Synaptic, you can look at the Properties of a package you have installed.  One of the properties is the names of the files.  The program name has to be in there somewhere.
