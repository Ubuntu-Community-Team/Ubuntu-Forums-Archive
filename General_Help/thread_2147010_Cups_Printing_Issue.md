______________________________________________________________________

Canon supply a driver for this printer; it is called the UFR driver; for a 64bit system, it seems helpful to install some symbolic links before installing the driver

the links would be 

> [COLOR="#FF0000"]sudo ln -s /usr/lib /usr/lib64[/COLOR]
> [COLOR="#FF0000"]sudo ln -s /usr/local/lib /usr/local/lib64[/COLOR]
> [COLOR="#FF0000"]sudo cp /usr/lib64/cups/filter/pstoufr2cpca /usr/lib/cups/filter[/COLOR]
> [COLOR="#FF0000"]sudo cp /usr/lib64/libcanon* /usr/lib[/COLOR] 

you can get the UFR driver from here

[http://support-asia.canon-asia.com/contents/ASIA/EN/0100270810.html](http://support-asia.canon-asia.com/contents/ASIA/EN/0100270810.html)

it comes down as the [COLOR="#008000"]Linux_UFRII_PrinterDriver_V260_uk_EN.tar.gz[/COLOR] and I can give you more help if you choose that route: a standalone printer; connected to a standalone home computer; should be ok; a networked printer can involve many issues; you have a networked printer you tell us
