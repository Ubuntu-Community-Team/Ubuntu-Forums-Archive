### Post by papibe on 2012-07-24
qmail is MTA (message transfer agent), just like postfix.

Read [here]("https://help.ubuntu.com/community/Postfix") the section called 'What is a Mail Transfer Agent'.

My guess is that you need a mail client, like Thunderbird, or Evolution.

Regarding to the ability to install other packages, let's make a simple installation. tree is a little program that show the directory structure as a tree.

Could you post the result of this?
```
sudo apt-get install tree
```
