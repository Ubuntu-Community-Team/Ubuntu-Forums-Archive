My Ubuntu 12.04 is fully updated and installed since a long time - I can't find it either in the software center.
Try downloading the DEB file from the official website:

[http://www.skype.com/en/download-skype/skype-for-computer/](http://www.skype.com/en/download-skype/skype-for-computer/)

Note that there apparently is no package for Ubuntu 12.10, only for 10.04 and 12.04.
Double-click on the DEB file, and the Software Center will open.

Well, skype can be in the software center. You just need to enable the partner repository in sources setting.

Take a look here:
[https://help.ubuntu.com/community/Repositories/Ubuntu#Adding_Canonical_Partner_Repositories](https://help.ubuntu.com/community/Repositories/Ubuntu#Adding_Canonical_Partner_Repositories)
