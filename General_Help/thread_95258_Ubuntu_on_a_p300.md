### Post by hw-tph on 2005-11-26
It will be sluggish if you use the default Gnome user interface. You can probably gain quit a lot of performance by using some less taxing UI like the OpenBox window manager (you can add a panel separately - try [fbpanel](http://fbpanel.sourceforge.net/) for a light GTK2 panel).

Also make sure you disable services that you don't need.
