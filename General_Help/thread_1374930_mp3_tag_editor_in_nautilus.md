### Post by sgosnell on 2010-01-09
No, the reply was not too harsh.  Ubuntu does not support mp3 files at all, out of the box.  You have to install additional codecs just to play them, because mp3 is a proprietary format.  It's no surprise, then, that you have to install additional software to do anything with mp3s, and will almost certainly continue to have to.

### Post by ethanay on 2010-02-14
You can only read metadata in Nautilus.  You can edit tags in Rhythmbox.  Or if you have a lot of editing to do and need some automation/batch support, use any number of the tag editors.

Just go to Applications > Ubuntu Software Center > Sound & Video and search "tag" and you'll get plenty of options.

I use MusicBrainz Picard whenever I can.  It is simple and slick.  And if something isn't in there (~50% of the time) then it is an opportunity to add to the database for future users.  You know, in the spirit of Ubuntu :)
