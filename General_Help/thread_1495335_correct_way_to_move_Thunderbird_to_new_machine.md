You do mean the ".mozilla-thunderbird" folder and not the "thunderbird" folder .. don't you?

One very strong suggestion is that you File>Compact Folders before you start moving stuff around.

I've done this quite a few times and if memory serves me correctly the following methods works every time for me;

1/ Uninstall TB on the new machine

2/ Delete any ".mozilla-thunderbird" folder you might have in your new /home folder

3/ Copy old ".mozilla-thunderbird" folder to new machine in /home

4/ Install TB

5/ Shut down and re-boot. I think that'll work but I'm not anybody's expert.
