### Post by AnLGP on 2008-09-13
Someone please correct me if I'm wrong.

Metacity is GNOMEs (default Ubuntu Desktop Environment) Compositing Window Managing system.  [http://en.wikipedia.org/wiki/Metacity](http://en.wikipedia.org/wiki/Metacity).  You can change this by using other window managers such as compiz, fluxbox, awesome, pekwm, sawfish, etc.

I can't answer your other questions, though.  I know permissions are simply what you as a user or people as a group can do with the specific directory or file, etc.  As far as where they are I can't say 100% but with Thunar when I right click it has a "Properties" option and the permissions are under there.

Maybe it is similar for metacity and maybe it is not.

I think the good thing about Ubuntu Tweak is that it makes it easier for people new to linux & ubuntu to change things that require a bit of work and knowledge easier.  I also think it's a double edged sword because some people might get lazy and not go learn it themselves; which is always beneficial.

### Post by fragos on 2008-09-13
Ans to 3. is when you right click a file and select "Properties" you get a pop up window with a number of tabs. How many tabs and their purpose depends on the type of file. This Property window controls system operation for all files of that same mime type. Click the Permissions tab and the advanced permisions settings for that mime type are available to change.

### Post by mb_webguy on 2008-09-13
I just downloaded and installed Ubuntu Tweak to see what it's about.  From what I can tell, it doesn't do anything that you can't do without it, mostly by using gconf-editor.  Ubuntu Tweak simply provides a sleeker interface.  It also consolidates several other things like Add/Remove Programs and certain things on the System menu into one interface.

Personally, even though the interface may not be as simple, I prefer staying with gconf-editor because it provides the ability to set many more preferences than are available in Ubuntu Tweak.
