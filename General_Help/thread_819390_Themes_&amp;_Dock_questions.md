### Post by pointone on 2008-06-05
After downloading the .tar.gz files, open "System > Preferences > Appearance" and drag/drop the file into the window, or click the "Install" button and browse to the file yourself.

As for dock, AWN is probably the most widely used:

```
sudo apt-get install avant-window-navigator
```
