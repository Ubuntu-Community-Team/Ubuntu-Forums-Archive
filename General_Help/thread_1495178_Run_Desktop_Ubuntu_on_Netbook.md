### Post by ibuclaw on 2010-05-27
There really is not much difference between the two...

Infact you can install the Desktop version without the need for reinstallation.

```
sudo apt-get install ubuntu-desktop
```

Then logout, and on the bottom panel, should be a "Session" dropdown box. Select "Ubuntu Desktop Session" and login.

IMO, when it comes to window managers, Metacity (default in Ubuntu Desktop) is much lighter than Mutter (default in Ubuntu Netbook). And is very noticeable under intense UI processing conditions - ie: video/flash rendering.
