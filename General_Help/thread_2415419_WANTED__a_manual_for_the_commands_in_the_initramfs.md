### Post by CatKiller on 2019-03-25
I'm sure there are guides out there somewhere, but those tools are not necessarily sufficient to fix whatever has put you in that position. They are simply the maximum that can be made available at that time.

For the two likely scenarios that will put you there - hardware, or you've messed up the configuration - you'll want to use something else. A live DVD/USB is generally the best tool for undoing system breakage.
