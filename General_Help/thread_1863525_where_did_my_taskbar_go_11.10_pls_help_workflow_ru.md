### Post by lovinglinux on 2011-10-18
It is just a bug that affects some users. To fix that, install *Compiz Config Settings Manager*, launch it, click the Unity plugin, set the "Hide Launcher" option to "Autohide". It won't move when you drag a window next to it, but it will show up as expected when you move the mouse to the edge of the screen.

Be aware that some Compiz plugins are making Compiz crash. So be careful about what do you change with CCSM. If Compiz crashes and you can't log on Unity 3D anymore, delete the file ~/.config/compiz-1/compizconfig/config to restore it.
