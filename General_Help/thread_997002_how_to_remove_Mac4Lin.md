### Post by seren6ipity on 2008-11-29
Hey LethalBoy, I just uninstalled it. 

Go to terminal and browse (cd) to the folder containing Mac4Lin_Uninstall_v1.0_RC.sh script. The unistall script expects the entire folder Mac4Lin_v1.0_RC1 or similar name depending on version to be in \home\username.

From terminal run sudo sh Mac4Lin_Uninstall_v1.0_RC.sh 

Let me know if this does not work for you.

Manually delete fonts from /username/.fonts

Ctrl+Alt+Backspace to refresh and get the normal scroll and window max/min options back.
