### Post by dannybuntu on 2009-07-27
> **TwosComplement said:**
>  Does anyone know a more modern Windows look-alike window manager?

Well, you could make gnome look like windows xp by installing windows xp gtk+ theme look alikes. But that's hardly the solution since I the GUI design philosophy is very different.

A window manager can only take you as far and most window managers that I know of do not support highlighting, dragging things on the desktop. 

So the best course for you is to let your friend continue using windows xp. 

Making Ubuntu look like Windows XP is only a small change when it comes to the whole using your desktop thing. What are you gonna do if he needs to go into the terminal? How about untarring tar.gz.

For your friend to appreciate Ubuntu he must be willing to shed off some gui usage differences.
