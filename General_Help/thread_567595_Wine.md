this is the main wine source

[http://www.winehq.org/](http://www.winehq.org/)

with the windows programs it supports being listed, in order of playability, here

[http://appdb.winehq.org/](http://appdb.winehq.org/)

I run a few windows games (Far Cry, Half Life 2, a biological recording package, and a share-trading package). If you get into it there is plenty of help around here on the subject.
