### Post by idunno271828 on 2008-03-31
OK, this was a stupid problem with an easy solution.  I had disabled the "Enable System Beep" option and perhaps accidentally checked  the "Visual System Beep --> Flash entire screen" option.  This "visual" option just needs to be unchecked.  This can be done by accessing the Sound Preferences via System --> Preferences --> Sound --> System Beep tab.
