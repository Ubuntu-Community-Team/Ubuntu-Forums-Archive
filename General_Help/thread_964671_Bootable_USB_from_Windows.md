It's far easier to burn the ISO file to CD as an ISO image - you don't need to extract the contents.
Have a look here: [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

EDIT: Oh: you want to put it onto USB. Don't panic! there's a help page here: [https://wiki.ubuntu.com/LiveUsbPendrivePersistent#Live%20USB%20creator%20(GUI-based,%20runs%20from%20Live%20CD](https://wiki.ubuntu.com/LiveUsbPendrivePersistent#Live%20USB%20creator%20(GUI-based,%20runs%20from%20Live%20CD))
