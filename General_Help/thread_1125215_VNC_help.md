### Post by HermanAB on 2009-04-14
If your machine is running Windows XP Pro or Win2003, then enable Remote Desktop Protocol.  RDP works better than VNC and it is secure.

If your machine is running Linux, then install ssh-server and let your friend use SSH from Linux, or Xming and PuTTY from Windows.  SSH works better than VNC and it is secure.
