### Post by cariboo on 2008-12-20
I found that in a lot of cases there is no help files, using the help browser. Almost all programs have manpages, for example to check the manpage for gnucash, in a terminal type:

```
man gnucash
```

If you really don't like the terminal you can also check out the man page on the internet, in Google just search for man gnucash, and you should end up [here]("http://man.gnusquad.org/gnucash/section-1/en/").
