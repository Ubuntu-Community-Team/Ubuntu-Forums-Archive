### Post by raashell on 2008-11-30
I upgraded to the latest kernel this morning and lost my virtualbox capability.  I've tried re-installing the package through both Synaptic and the repositories with no luck.  Re-installing the generic virtualbox-ose-modules also met without success and there are no modules, that I can see, for the latest kernel.  

Anybody know how to fix this?

===============================

Minutes later, I removed all the virtual box stuff and installed virtual-box2, Sun's proprietary version and everything is working again.  If you're having a similar problem and don't see virtual-box2, you might need to add Sun's repository.  There are instructions for this at the virtualbox site.
