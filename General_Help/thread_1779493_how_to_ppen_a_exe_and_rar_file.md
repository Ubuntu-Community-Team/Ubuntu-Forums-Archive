### Post by hhh on 2011-06-10
You can't open .exe files in Linux, it's a Windows format. You can run them through Wine sometimes.

For rar, install the package unrar (sudo apt-get install unrar or search for it in Synaptic). After that you can right-click the file and choose "Extract Here". HTH
