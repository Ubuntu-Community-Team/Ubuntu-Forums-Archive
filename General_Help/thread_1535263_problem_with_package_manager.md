### Post by ankspo71 on 2010-07-21
Hi,
It is probably the automatic updater running in the background. Whenever I run into this problem I restart my computer and it works again. We never want to force quit any update or package-management processes, so I find restarting to be better.

If that doesn't help, it should be possible to disable the automatic updates (a feature I'm not seeing right now in Kubuntu 10.10 testing), but then you won't be notified of updates until you enable it again.
Hope this helps
