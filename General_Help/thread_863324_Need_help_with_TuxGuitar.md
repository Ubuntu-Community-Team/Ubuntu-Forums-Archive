### Post by DirtDawg on 2008-07-18
You're post is very vague about what happens when you try to run Tux Guitar.

However, it looks like the version in the repository (add/remove) is a little old, so using the most recent version will likely be easier than trying to fix the one you already installed. If you go to the [Tux Guitar download page]("http://www.tuxguitar.com.ar/download.html"), you will see several options. I would recommend downloading either the "Ubuntu-x86 Binary Package" or the "GNU/Linux-x86 Binary Files".

You can double-click the Ubuntu Binary and it will install itself (make sure you uninstall the older version first). Or you can try the Gnu/Linux binary, just extract and double-click the "tuxguitar" file and select "run".

If you have any other problems, post a detailed account here and maybe we can help. Have fun.
