### Post by Jonor on 2014-08-29
I would file it as a bug on Launchpad, they may then say if it is not a bug.
Within the Tweak Tool tweaks for windows there is an option to Automatically raise Windows 
which may make a difference if for some reason it has been switched off.
In the meantime there is a Window List extension for displaying at the bottom of the screen 
if the hot corner is too inconvenient.
