### Post by gordintoronto on 2011-09-15
This thread explains all:
[http://ubuntuforums.org/showthread.php?t=1632365](http://ubuntuforums.org/showthread.php?t=1632365)

However, there is an assumption that all the computers are running exactly the same version of Ubuntu. (Eg. 10.04 32-bit.) Similarly, if only one machine has Gimp installed, it is the only one which will get Gimp updates.
