### Post by tristan on 2005-06-05
Hi everyone. I've had a long interest in ubuntu. Had 4.10 dual booting with mandrake 10.1 for a while, but recently blew the hard drive up and used the occasion to go completely ubuntu (5.04). I'm very happy with that decision!

I have a Creative Zen Touch 20Gb which is supported under linux using gnomad2 (a front end to libnjb). The Gnomad2 version 2.5 supplied with Hoary works well but is not terribly fast at file transfers. However Gnomad 2.7 and libnjb2.1 have just been released. I downloaded the FC3 rpms and converted them to debs with alien, and they installed and run perfectly. Most importantly the file transfer has to be at least 200% faster! 

One thing I have noticed is that if I have the pilot applet running and scanning for my Palm (USB), then there are serious conflicts with gnomad2. It won't read the jukebox contents nor transfer files. Disabling the applet fixes the problem. I wonder if anyone else has observed the same problem?
 
Anyway, sorry for a rambling 1st post. In summary : Ubuntu 5.04 rocks, and gnomad2.7 rocks provided you don't use the pilot applet.
