### Post by papibe on 2015-03-24
Hi Lori_Imel.

A couple of thoughts:

In the file manager (Nautilus), you can use the same trick as in Firefox to increase the size of the icons and thumbnails: Ctrl++ to enlarge, and Ctrl+- to reduce. If you want a default preferred size instead, go to Preferences and set Icon View Defaults -> Default level zoom to the level you want.

Now, if you really have tons of images, the default file browser (Nautilus) is not the best way to manage them. I'd recommend an specialized tool.

I personally use gthumb (available on the Software Center). It is simple and fast. You can browse the thumbs, open a pic in fullscreen, and do slide shows. To set the thumbnails size go to Preferences -> Browser -> Thumbnail size.

Hope it helps. Let us know how that goes.
Regards.

### Post by grahammechanical on 2015-03-24
If we use File Manager to select an image file and right click that image we can choose Open with Image Viewer and then we can use the Previous and Next arrows to move from image to image through all the images in that particular folder.

Double click an image that is revealed in the Dash will open Image Viewer showing that image and again we can use the Previous and Next arrows. In the Dash right clicking an image will provide a larger view without opening Image Viewer and there are Forward and Back arrows that will take us through the images that are either in the Pictures folder or are recently used files.

Regards.
