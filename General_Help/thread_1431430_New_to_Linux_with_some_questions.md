### Post by J V on 2010-03-16
1 in XP i have short cuts to applications/internet/etc. cannot see how to create one in Linux (i found launch but that does not seem to fit).

In linux programs are started with commands, so to start firefox you just put the command "firefox" in a launcher, similarly, "firefox http://ubuntu.com" will pop open ubuntu.com in firefox... Or you could switch it to "Location" instead of "Application" and just type in the url, same difference...

Depends... visual basic not qbasic I presume? :P

Most windows programs that are written to standards should work fine through wine, but I would still advise looking for an alternative (And if you wrote them yourself, the mono project should make them work on linux, or you can rewrite in bash which would be much faster)
