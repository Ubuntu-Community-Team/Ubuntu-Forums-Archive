Okay.....first of all you need to get a good GDM theme made by someone else. Choose one in which the text fields and their positions suit your needs. Extract the tar.gz/tar.bz2 file somewhere. Then get an image as per your needs or if you have something homemade, that'll do. The third step...copy your image to the folder where you extracted the GDM theme. And then rename it with the name of the image file that already appears in the GDM theme e.g. something like 'background.jpg'/'background.png' so that your file replaces the original image file. Then create a tar.gz archive of the folder. Install the theme and logout and see for yourself.

Enjoy!!:)
