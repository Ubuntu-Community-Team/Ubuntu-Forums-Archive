### Post by chocklet on 2012-07-02
The author is shown at...
File > Properties > Created

if the name is inserted at...
Tools > Options > LibreOffice > User Data > First/Last name/Initials

AND

if the box... 
File > Properties > "Apply user data" 
is checked


To eliminate the author's name at File > Properties > Created, either 
- uncheck the File > Properties > "Apply user data" box, and
- click File > Properties > Reset
or
- eliminate the name from Tools > Options > LibreOffice> User Data > First/Last name/Initials, and
- click File > Properties > Reset
