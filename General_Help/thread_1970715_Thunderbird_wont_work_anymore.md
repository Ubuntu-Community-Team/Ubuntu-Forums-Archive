### Post by ajgreeny on 2012-05-01
Is this a pop3 or imap acount?  If pop3 I suggest you backup the hidden ~/.thunderbird folder (or at least the **Mail** folder in there), and then try a new profile by starting TB and setting up the account again.

If it is imap, it is less important to back up the ~/.thunderbird folder as all the mail is kept on your mail provider's servers, not your own drive, nevertheless it is still worth backing up, which you should always do regularly in any case.

You **do** make backups, don't you?
