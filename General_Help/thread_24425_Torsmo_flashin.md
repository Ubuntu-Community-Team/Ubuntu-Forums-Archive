### Post by PMO6022 on 2005-04-07
This can be fixed as follows:

Goto "applications -> system tools -> configuration editor"
Now select "apps -> nautilus -> preferences"
Scroll down the list to find "show desktop" and uncheck it.

Note that this will cause Nautilus to no longer draw any desktop icons.  I don't use any icons, so this fix works fine for me.
