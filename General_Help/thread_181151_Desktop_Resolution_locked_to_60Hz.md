### Post by Ramses de Norre on 2006-05-25
You've got modelines defined in your screen section, these are used to define your refresh rate. Easiest way to solve this is sudo dpkg-reconfigure xserver-xorg.

### Post by Potfur on 2006-05-25
Add HorizSync, VertRefresh and ModeLine in "Monitor" section.
Then, try to add this in "Device" section:
```
Option   "UseEDIDFreqs" "False"
Option   "ModeValidation" "NoEdidModes"
```
