### Post by zarqoon on 2009-07-27
Sound a bit like the system is not even trying to boot the cd.
Check the Bios for the Boot order and make sure you cdrom is in front of the harddisks.

Some boards also have an option to change the boot device just for one startup. Read the stuff on the screen directly after powering on the machine. It could say something like press f8 to enter boot manager.

### Post by mikewhatever on 2009-07-27
It's possible that you have a bad cd, (have you downloaded and burnt is yourself?), but I don't quite understand what you are trying to do. Can you clarify why are you trying to boot various Windows versions with the Ubuntu cd inside. Anyway, that's not the way things work. You can check out the cd while running Windows. Just put it in and wait. There should be a kind of presentation with some pictures and info about Ubuntu. If you want to try Ubuntu without making changes to disk, set the bios to boot from cdrom and then restart with the Ubuntu cd inside. Ubuntu should load from the cd, running entirely in RAM, which is good for a first impression and for testing the hardware.
