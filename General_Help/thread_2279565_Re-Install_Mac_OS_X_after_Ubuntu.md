### Post by buzzingrobot on 2015-05-24
To find out if the hidden recovery partition is still there, restart the Macbook.  When you hear the startup chimes, hold down the Command and the 'R' key.  Keep holding them down and see if you boot into Recovery Mode.  If so, from there you should be able to do restore the OS. I recall from a few years ago that you will have access to a partitioner so you can remove any Ubuntu partitions and repartition the drive correctly for OS X.

If the recovery partition was actually deleted, an Apple forum is likely a better bet for help on reinstalling OS X. A DMG, I seem to remember, is an archive.  Right click on it to see if Ubuntu knows how to extract it.  Perhaps there's a README file inside.
