### Post by albandy on 2009-03-07
you know php files are commonly used for web applications? You should run it under apache with php support installed.

Also you can run it in termianl by typing php file.php but this takes no sense in a web environment.

So copy the php file to your /var/www folder and then type in your browser: [http://localhost/myphpfile.php](http://localhost/myphpfile.php)
