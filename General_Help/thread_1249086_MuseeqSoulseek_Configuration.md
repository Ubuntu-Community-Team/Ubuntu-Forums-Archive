### Post by jesuisbenjamin on 2009-08-25
Since no one responded except for this spammer whose post has rightly been deleted, i found the answer and thought it'd be clever to post it for the sake of archiving solutions to problems.

[LIST]
[*]In order to use Museeq or any other Soulseek client, one needs to have Museekd installed (the deamon).
[*]Then it will not work if a minimum configuration has not been done, this can be done by running *musetup* in Terminal. Musetup will let you know what are the basic requirements.
[/LIST]
