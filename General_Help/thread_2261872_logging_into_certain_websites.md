### Post by raptir on 2015-01-21
There are a couple possibilities...

[LIST]
[*]It could be a browser compatibility issue. If you are using Firefox, try installing Chromium (or Chrome from Google's site) and see if that works, or vice-versa. I would say use whatever browser you were using in Windows.
[*]There could be a missing plugin, though it usually would tell you. But check if it says anything about needing Java or something else.
[*]On the flip side, it could be an extension you have installed interfering. Things like NoScript and Adblock Plus, though very useful, can sometimes cause sites to load improperly.
[*]
[/LIST]
