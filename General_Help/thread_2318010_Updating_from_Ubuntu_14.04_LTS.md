### Post by howefield on 2016-03-22
16.04 won't be released till the 21st of April. Until then it remains a development release and should only be used if you are prepared to deal with breakage.

Installing with a 16.04 DVD and selecting "Something Else" at the partitioning screen will allow you to mount the existing 14.04 / partition/disk to install directly over the 14.04 release, by not marking for formatting you will preserve your /home folder. In any event, something as risky as upgrading an operating system shouldn't be done without having a backup of your important data safely secured.

Open up Software Sources and check that you are set to be notified of Long Term release and you will be advised when it is available for you.

### Post by mastablasta on 2016-03-22
you will be prompted to update when 16.04.1 is released (sometimes in August). you can also upgrade it when it is released. even when released it is expected to have some bugs unresolved and unknown. so if you cherish the stable system wait for the upgrade prompt or use the interim releases (note do not attempt to upgrade from 14.04 to 15.10 - for some reason it gives this option :o )
