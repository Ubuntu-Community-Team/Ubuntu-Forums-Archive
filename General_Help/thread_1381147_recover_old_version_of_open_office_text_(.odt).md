### Post by mörgæs on 2010-01-15
The file itself is lost, but there could be a backup in the same directory. 

It will be a hidden file with a snake (~) added in the file name.

Hidden files can be seen with the command ```
ls -a
``` in the command line or by allowing Nautilus to show hidden files.
