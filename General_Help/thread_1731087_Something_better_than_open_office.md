### Post by earthpigg on 2011-04-16
Two hints that may be relevant to you: 

1) OpenOffice.org from the official website is different, and lacking certain features, when compared to OpenOffice.org that [we have in Ubuntu]("http://en.wikipedia.org/wiki/Go-oo"). If you went to openoffice.org and downloaded Sun's latest official release thinking it would improve your experience, it actually did the opposite. 

2) On all versions/branches/forks of OpenOffice (including LibreOffice), disabling java speeds things up. Tools -> Options -> Java -> uncheck the box.

in addition to the options already mentioned here, some people maintain a virtualbox install of windows + ms office for that "just in case" emergency situation.
