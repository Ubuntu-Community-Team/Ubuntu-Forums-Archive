### Post by urukrama on 2010-07-29
The lightest would probably be MPD . MPD is a music player daemon that runs in the background, which you control with a front end (the interface to the player). There are several good front ends for MPD: sonata, gmpc, or, if you prefer the command line, ncmpc and ncmpcpp. This will probably be the lightest you can get.

There are some guides on this forum on how to set it up. Setting it up is not all that hard and once you have it configured you can run one of its front ends just like an ordinary media player.

You might also be interested in QuodLibet. It is not as light as mpd, but a very fine audio player.
