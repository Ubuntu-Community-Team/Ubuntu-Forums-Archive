### Post by qamelian on 2007-08-17
> **kh1116 said:**
> I am using an ATI Radeon 9000, the tutorial is for ATI cards, if there is a different way to install beryl or compiz with my radeon card, please tell me.
Thanks Kevin

You don't need the restricted drivers with this card, which is just as well since ATI no longer supports it with the last several driver releases. Beryl, Compiz, and Compiz-Fusion all work fine using the default open-source drivers for this card.

The restricted drivers are needed for some cards (mostly newer cards) but not others.
