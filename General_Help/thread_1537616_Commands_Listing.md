### Post by Zeike on 2010-07-23
if you actually want the full list you can do: 

```

ls /bin; ls /usr/bin; ls /sbin; ls /usr/sbin

```

Be prepared for a lot of scrolling ;).  In fact it will probably fill your scrollback buffer up and you won't be able to read them all...
