### Post by BigSilly on 2008-05-12
I realise there are a few threads around regarding this, and I've searched through them but can't find anything to sort this one out. I've got a very old numpty monitor whose resolution is a measly 1024x768. I've been able to set the desktop and bootsplash to this just fine, but for some reason I get something higher on the login screen. It's not too bad to be fair. It fits the screen and everything and is usable, but I'd rather set it to the correct 1024x768. I don't like the jarring effect as it switches from the login to the desktop.

I've had a look through my xorg.conf, but can find no reference to screen resolutions. Here it is:

### Post by BigSilly on 2008-05-12
Fixed it. Had to add a subsection like this -

```
Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
	Defaultdepth	24
  SubSection "Display"
  depth 24
  virtual 1024 768
  EndSubSection
```

Sorted! I'll mark this as solved if I can find the button....
