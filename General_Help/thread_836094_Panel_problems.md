### Post by gandalf458 on 2008-06-21
I'm running 8.04 on a Thinkpad. Every so often when I use the mouse pad it seems to "swap screens" - I think I've fathomed out that they're called panels. I can't figure out how to control them. Having accidentally swapped panels I can't figure out how to swap back. How do I find out how panels work and can they be switched off?

---

### Post by drs305 on 2008-06-21
On the bottom panel you can add a "window switcher" icon. It displays the number of windows/workspaces available and allows you to transfer between them. It looks like a series of squares. Once you have it installed on the panel, you can right click, Preferences, and the choose the number you want. You can select "1" if you more or less want to disable this feature and just want a single workspace.

If this isn't what you are looking for you might be able to adjust the settings if you are using compiz: System, Preferences, Advanced Desktop Effects.
