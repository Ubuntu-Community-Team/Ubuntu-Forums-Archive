### Post by Topsiho on 2011-05-30
If you know vim well, and if vim knows about C++ syntax highlighting, well, go ahead and use vim :)
You could also use gedit, which is more easy to use than vim IMHO.

You also could use an ide, which might be useful for larger projects, but if you just start learning C++, probably you don't start larger projects.

Using an ide needs some learning too, so I guess (but who am I?) you better concentrate on C++ and use vim, or gedit, or maybe nano. Plus: if you know some C++ choosing an ide might be simpler.
