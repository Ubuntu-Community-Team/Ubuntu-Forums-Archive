### Post by Mark Phelps on 2012-02-28
This is one of the reasons I moved away from Ubuntu version 10.x -- nearly half of the time, it would not boot for me.

What sometimes worked for me, was booting into Recovery Mode and running the entry to Repair broken packages.  After that, I was able to get a working desktop with StartX.  And, after a reboot, it would work OK again -- for a while.

I've found the 11.x series to be a lot more reliable.
