### Post by jim_deadlock on 2015-12-04
Thunderbird has an adaptive spam filter, it "learns" as it goes. The more messages you mark as spam the more efficient it will become. Also, never "unsubscribe" from spam, it will just make it worse, just mark as spam. You can also set TB to automatically delete spam so you never even need to see it.
