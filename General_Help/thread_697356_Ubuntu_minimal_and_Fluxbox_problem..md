### Post by kerry_s on 2008-02-15
> **zaqwe said:**
> botched install of Ubuntu or Xorg?

Tried reinstalling, but no effect, except errors changed. 

New errors : 
```

(EE) AIGLX error: dlopen of /usr/lib/dri/r300_dri.so failed (/usr/lib/dri/r300_dri.so: cannot open shared object file: No such file or directory)
(EE) AIGLX: reverting to software rendering
```

wait, wait, that's a common error in xorg.0.log, meaning aiglx is not supported on your card. i get that when i install, you just turn it off.

/etc/X11/xorg.conf

```
Section "ServerFlags"
        Option  "AIGLX" "off"
EndSection
```


your problem might be xdm, it's old as hell, try going with gdm,

[B]sudo apt-get install xorg gdm fluxbox
then reboot(ctrl+alt+delete)[/B]

or you could always do with out and when you log in have x start up.

~/.bash_profile

```
[B]if [ `tty` = "/dev/tty1" ]; then
startx
fi[/B]
```
