### Post by ahndoruuu on 2009-06-28
You don't need to "uninstall" Ubuntu.

Assuming you actually have an XP CD, you just need to boot from that CD and follow the installation process, eventually it will ask you if you'd like to format the hard drive or something to that effect, at which point you should say yes. Now you're going to run into complications if you don't have an XP CD or if you don't know how to set your system to boot from CD-ROM as first priority.

What kind of computer? If custom built, what motherboard?
Brand/Model would be excellent. I'm guessing you just need to get the boot menu and choose the CD Drive from the menu. Most computers will tell you the esc key to press to get a boot order menu, or to get into Bios setup. As mentioned earlier F2 is a common choice for Bios Setup, and Esc is a common choice for Boot Order Menu.

Once you've set your CD ROM drive to be first in boot order; then, with your XP cd in the drive, press any key to boot from CD when you are prompted to do so on the black screen with the white text.

Follow the MS Windows XP install wizard, and in no time you'll be buying an Anti-Virus subscription, and battling browser hijackers with the best of them.
