### Post by Topsiho on 2014-04-22
It is quite a normal, even a low, number of updates after installing a version of Ubuntu that was released several months ago. Usually the number of updates can be far greater. I guess you installed from the last point release of 12.04 LTS, which is OK.
You should understand that not only the kernel itself is updated (security, drivers new hardware), or other system programs, but also all applications that you installed from repositories or maybe ppa's. Not only security but also improvements and other changes.

Any update poses a risk that it'll ruin something on your computer, but after nearly 10 years of using Ubuntu (from the very first release, 04.10) I remember that this happened only once to me, and this was repaired within the shortest of times. No problem at all.

As stated by howefield: Ubuntu 14.04 LTS was released a few days ago. I myself am waiting for the first point release (14.04.1 LTS) before I´ll install this in place of my current 12.04 LTS, so that most remaining problems in 14.04 may be cured.

The point releases prevent having to download too many updates, which are necessary to update a fresh install from a .iso that is some months old :).
