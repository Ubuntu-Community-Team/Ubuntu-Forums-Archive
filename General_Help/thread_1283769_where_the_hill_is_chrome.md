### Post by lovinglinux on 2009-10-06
> **fragos said:**
> I don't think it works fully but I believe Chromium calls them "user-scripts" but you'll need to add a parameter to the command line as in "chromium-browser --enable-user-scripts %U".

That would be like a built-in Greasemonkey scripting. See documentation [here](http://dev.chromium.org/developers/design-documents/user-scripts).

The extension API actually uses **--enable-extensions**. It is currently under development and you can't find any extensions in the extension page. See documentation [here](http://dev.chromium.org/developers/design-documents/extensions).
