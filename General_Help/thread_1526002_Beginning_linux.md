### Post by Darkness Des on 2010-07-07
Learn how to use the command line first (not very hard, takes very little time if you go to [http://linuxcommand.org](http://linuxcommand.org)) then learn Python (optional), C, C++ (or C#, if you must. I never use it), and Java. That should get you set pretty well. Only learn what you need though.
