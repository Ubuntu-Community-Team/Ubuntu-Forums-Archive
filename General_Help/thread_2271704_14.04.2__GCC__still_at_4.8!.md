### Post by dino99 on 2015-04-01
When a version has been released, then it gets 'security' upgrades and sometimes can get some backports.
If you want/need the latest stable version or the latest released, then dont stick with an old ubuntu flavour, even if it is still maintained.
Some ppas can provide newer versions, but then you will not be sure to be conflicts free with one or more lib/app
