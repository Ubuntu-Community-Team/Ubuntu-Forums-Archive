### Post by mickie.kext on 2009-12-20
> **shadow of the locust said:**
> alt-tab worked!

for some reason though, minimising a window doesnt minimise it to the panel.  i have no idea where it disappears to.  has it maybe been a change in some setting that minimises it elsewhere?

Right click on panel, click "add to panel", then search for item called "Window List". You probably want to add "Show Desktop" and "Trash" items too.

PS: You need "Workspace switcher" too, that is that thing for selecting workspaces. If those items become missplaced, right-click it and select "move". Then... well, move it wherever you want it to be on panel.
