### Post by Vixis on 2007-10-28
You will need *aptoncd*
in the terminal :
```
sudo apt-get install aptoncd
```

then System -> Administration -> APTonCD

More instructions: [Create backup of all installed packages using APTonCD in Ubuntu]("http://www.ubuntugeek.com/create-backup-of-all-installed-packages-using-aptoncd-in-ubuntu.html")
