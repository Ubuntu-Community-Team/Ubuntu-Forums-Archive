### Post by Yellow Pasque on 2008-11-06
If you need the latest version of an application, you can compile it from source or find a pre-existing .deb

VLC: [https://launchpad.net/~c-korn/+archive](https://launchpad.net/~c-korn/+archive)
Eclipse: [https://launchpad.net/~eclipse-team/+archive](https://launchpad.net/~eclipse-team/+archive)
Tomcat: [https://launchpad.net/~miguillo/+archive](https://launchpad.net/~miguillo/+archive)
f-spot: [http://ppa.launchpad.net/michelinux/ubuntu](http://ppa.launchpad.net/michelinux/ubuntu)
