### Post by vamega on 2010-01-19
As far as the official client is concerned, there is no way for you to get the windows version running on Linux without using Wine. You could use the linux version that they offer, by downloading the version found [here]("http://www.aim.com/get_aim/linux/latest_linux.adp"). But that seems like a half baked attempt at a client in my opinion.

If you want to communicate over AIM, I'd use pidgin, empathy or another aim client that works on Linux.
