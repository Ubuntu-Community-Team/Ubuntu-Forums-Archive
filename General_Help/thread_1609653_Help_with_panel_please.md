The Gnome panels are real easy to use. Just right click on an empty bit of your top panel (where you want the window buttons!!) and select **add to panel** in the context menu. Then scroll down the list of stuff you can add and select **Windows List**.

### Post by ajgreeny on 2010-10-30
You can delete the top panel and just use a single bottom panel if you prefer.

Here's a screenshot of mine.  Everything is added to that one panel; it gets a bit crowded at times, but that's the way I like it, and my screen is not large, I run most apps maximised, and so screen real-estate is important to me.

From left to right on the panel I have:-
  Main menu
  Home
  Workspace switcher
  Window list
                Free space on panel.
  Disk mounter
  Network monitor
  Launcher for a personal bash script
  Gnome terminal launcher
  Launcher for personal txt file
  Indicator applet
  Open Office launcher
  Notification area
  Trash
  Clock
  Synaptic package manager launcher
  Shutdown.

As I said, it is a bit crowded for some people, but that is the beauty of Ubuntu and linux in general; you can make it look exactly the way you want it.
