### Post by darkstaar on 2010-10-29
Actually, you can install both side-by-side simply by typing in a terminal window:

```
sudo apt-get install kubuntu-desktop
```

Then you can select which desktop environment you  want to use at the login screen.

Of course, all of your data will remain intact.

EDIT: Since Kubuntu is really little more than Ubuntu with the K Desktop Environment rather than Gnome, it won't take up as much extra space as you might think.
