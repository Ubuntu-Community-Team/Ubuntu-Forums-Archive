### Post by Vaphell on 2011-03-06
amd64 is a common name for 64bit flavor of x86 architecture. Any modern desktop processor is (or at least should be) compatible with it. Check if your processor is in fact 64bit and you are good to go. Alternatively you can give 64bit livecd/usb a spin - if it refuses to run then you got your answer :)
