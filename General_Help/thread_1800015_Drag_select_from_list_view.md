### Post by searchfgold6789 on 2011-07-08
I don't know how to enable it, but until you figure out how, here is something that will help you:
You can select multiple things very easily by using the Ctrl and Shift key. To select a bunch of files that are next to each other, select the first file in the list you are trying to select and then depress the Shift key. Then, click the last file in the list to select it (while holding the shift key) and then those two files, along with all of the files in between, will be selected.
To only select individual files simultaneously, hold down the Ctrl key and select each file, and all the files you click will be added to the selection. You can deselect files by clicking on them, and all the other files will still be selected.
