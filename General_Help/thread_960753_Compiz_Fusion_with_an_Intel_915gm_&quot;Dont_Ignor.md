### Post by VeeDubb on 2008-11-02
Out of curiosity, have you gotten anywhere with this?

With the frame rates you've listed, it really sounds like you're using bad open source video drivers, and I'm honestly not sure what's available for the chipset you listed.  It sounds to me like a cheap on-board video adapter.

It's quite possible, and I'm only guessing here since I've never used beryl, that the problem is that beryl was able to run with little to no 3D acceleration, and compiz is heavily dependent on proper acceleration.

If all my guesses are right, you've really only got two options.  One would be to go back to using an older OS like XP, or whatever version of linux you were running beryl on.  The second option would be to buy a real video card.

XP came out, what, 5-6 years ago?  If you've got low performance hardware, you'll be a lot happier with that, or with a 5-6 year old version of linux than with 8.04 or 8.10.  And don't even THINK about vista.
