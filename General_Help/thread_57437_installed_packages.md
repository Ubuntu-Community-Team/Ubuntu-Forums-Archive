### Post by johannes on 2005-08-16
You can add a link in the menu with [Smeg](http://ubuntuforums.org/showthread.php?t=38183). It is really simple to use and you don't need an open console when you start the program you want from the menu. You can download it with Synaptic and access it in the menu Applications/System Tools/Smeg Menu Editor.
