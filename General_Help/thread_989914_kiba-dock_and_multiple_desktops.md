### Post by prshah on 2008-11-22
> **Vangelis said:**
> I have installed kiba-dock. I love this little dock, however, it only seems to appear on the first desktop. When I change to other desktops, it is not displayed on them. Does anyone know how I can display kiba-dock on all desktops?


Unfortunately, Compiz (compositing) only works on one desktop at a time; so, if you switch desktops and want kiba to appear on that, you have to manually run Compiz by pressing Alt+F2 and then giving the command```
compiz --replace
```
