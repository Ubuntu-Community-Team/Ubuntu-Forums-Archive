### Post by OutOfReach on 2008-10-13
Goto a terminal (Applications>Accessories>Terminal):
and type in:
```
sudo apt-get install msttcorefonts
```

It will ask you for your password, type it in, it will not show you'r password, but you will still be typing it in.
