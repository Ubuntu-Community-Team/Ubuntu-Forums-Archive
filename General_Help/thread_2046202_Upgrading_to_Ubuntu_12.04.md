### Post by houseworkshy on 2012-08-24
The general experiance of upgradeing to the new version as opposed to fresh installing the new version is that the fresh install method is less likely to create problems.  I'm still on 10.04 because I use a desktop PC, external mouse ordinary lcd screen etc, and for that set up I prefer Gnome2 or KDE.  
Duel booting both versions is a pretty good idea, it would give you the chance to learn Unity whilst the old familiar Gnome 2 is still functional.  I always duel boot now, it takes the stress out of change.
