### Post by Brandon Williams on 2009-08-28
I'm not certain which dialogs you're having trouble with, so I can't look to see if there's a way to fix them, but you _can_ get to the controls. By pressing the Alt key, you can drag the dialog around the screen while the mouse is pointing anywhere within the dialog. Without the Alt key, you can only drag a window with the mouse pointing at the title bar, but with the Alt key, the mouse can be pointing anywhere.
