### Post by xXx 0wn3d xXx on 2006-07-09
Run azureus as root to allow it to update. 
> 
cd /location/where/azureus/is
sudo ./azureus

It will then apply the update and all will be well. If you still want to erase it, the main folder should be in /opt/.
