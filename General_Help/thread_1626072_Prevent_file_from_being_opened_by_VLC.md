### Post by ntesla123 on 2010-11-19
How can I prevent a video files from being opened by a specific program. 

E. g. how can I prevent an avi or flv file from being opened by VLCPlayer

---

### Post by Kulshrax on 2010-11-19
Right click on the .avi or .flv file, select "Properties," go to the "Open With" tab, and select the new program you would like to open the file with. This will change the file associations for all files of that type.
