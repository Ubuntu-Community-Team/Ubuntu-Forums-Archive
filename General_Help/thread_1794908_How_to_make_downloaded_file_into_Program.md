### Post by sddhhanover on 2011-07-01
I recently downloaded eclipse IDE and Seamonkey, which are both currently executable files, but they do not show up under 'applications' on the top panel. Is there any way for me to change this such that they can appear under 'Applications' in any of the subcategories?

---

### Post by adit on 2011-07-01
You have done manual installation.  So they won't appear under Applications Menu automatically. If you want these programs to appear under Applications menu automatically, you should install through "Ubuntu Software Center".
Type into terminal
```
alacarte
```
The menu editor will start. There you can add your "manually downloaded" programs in any subcategory you want.
