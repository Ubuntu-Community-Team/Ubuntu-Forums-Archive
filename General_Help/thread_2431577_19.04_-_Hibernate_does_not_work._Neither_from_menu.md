From the "Leave" menu all features work including "Suspend", but "Hibernate" shows no reaction. In the "LXQt Preferences" I've set "Lid" to Hibernate when closed, this also doesn't work. Do you have pointers for me, please? Searching brought no joy.

### Post by GhX6GZMB on 2019-11-20
OK, fixed it.
Hibernate needs a disk swapfile that's not asked for during the Lubuntu installation. Installing the swapfile with a size => than internal RAM size brings success.
This link describes the procedure:

[https://linuxize.com/post/how-to-add-swap-space-on-ubuntu-18-04/](https://linuxize.com/post/how-to-add-swap-space-on-ubuntu-18-04/)
