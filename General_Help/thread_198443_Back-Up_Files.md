### Post by Irony on 2006-06-17
I notice that a lot of files generate hidden back-up files, e.g. a gedit file entitled notes adds a back-up file notes~

Are these back-up files necessary for the system or just for my benefit, I notice that I can untick them in the configuration editor.

---

### Post by 5-HT on 2006-06-17
Ubuntu will periodically make backups of certain important files (look in /var/backups). 

As to gedit making it's own backups of files (appended with an ~), I disable that feature as well. Just remember to manually backup any important config file prior to editing.

While I understand the default setting of having gedit make a backup of files you are editing if they happen to be important config files, to make a backup of  *every* file you edit seems a little overkill to me.

Hope that helps.
