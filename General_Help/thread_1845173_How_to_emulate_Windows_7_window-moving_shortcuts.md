### Post by 3rdalbum on 2011-09-16
In Compizconfig Settings Manager (hint: if you don't already have it, install it!), go to General Options and then click on the Keybindings tab. You'll be able to set up the maximise and minimise shortcuts.

For the side-by-side windows, you need to go to "Grid" in the main Compizconfig window. You can change the "Put Left" and "Put Right" shortcuts there.
