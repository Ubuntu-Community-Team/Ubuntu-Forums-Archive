### Post by Spr0k3t on 2010-06-21
Something to consider...

A DoS (Denial of Service) attack is a DoS regardless of the operating system used.  If a firewall is not properly configured to handle DoS attacks or other similar related issues, the performance of the system will be the same regardless of the operating system used.

So my questions are quite simple.  What is your server for and what have you set up the firewall to do?  Do you have an external (sic) hardware firewall via router?  Is this system a *mixed-server with a user desktop or dedicated to being a server without additional applications? (*major difference there)

If you are getting bombarded with malware, viruses, or the server is under constant heavy usage from verifiable remote connections... then you may have better luck with a system that has less overhead on the operating system.  If the system is intended to be used as a true server (ie, no user desktop whatsoever), then you would see quite a significant improvement on serving up whatever it is you are using the server for.
