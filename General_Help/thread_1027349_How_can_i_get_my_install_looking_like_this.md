### Post by steveneddy on 2009-01-01
```
sudo apt-get install avant-window-navigator
```

> The avant-window-navigator is a MacOS X like panel for the GNOME Desktop
written in C. Besides launchers that can be dragged onto the bar, it features
a task bar that behaves similar to the MacOS X dock. The window navigator
uses the composite extension for transparency and other effects.
