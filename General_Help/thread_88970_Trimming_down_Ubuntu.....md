### Post by Xian on 2005-11-11
If it were my lappy I'd use the InstallCD and do a server installation, which will get you the base-only Ubuntu system. Then either just put XFCE on for your desktop environment, or even better install fluxbox & gdm, then only add those applications that you know you will be using (web browser, media player, etc.). 

If you pile on XFCE or fluxbox now you'll still have a bloated system.
So, basically it is pointless. Start over and do it right.
