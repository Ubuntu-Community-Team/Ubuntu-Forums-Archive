### Post by QIII on 2018-09-11
Canonical is not under any requirement to include Tails in their repos.  Tails is a separate Debian-based (it started out as Gentoo-based as I recall) distro meant to be booted from a live USB or DVD.

You can use the PPA (apparently), or do a direct download from [here]("https://tails.boum.org/install/index.en.html").

In any case, it's not a "program" to be run inside Ubuntu, unless you want to run it in a virtual machine.
