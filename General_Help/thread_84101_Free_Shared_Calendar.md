### Post by slazh on 2005-10-30
I've been kinda looking for the same thing for some weeks now. The best thing I've been able to find was the following 'howto'; [http://www.vetula.com/blog-radio/stories/2002/11/19/publishingEventsInMozilla.html](http://www.vetula.com/blog-radio/stories/2002/11/19/publishingEventsInMozilla.html)

Aslong als your calendar application supports exporting / publishing to .ics files, and you have access to a webserver (hosting provider?) 
you'll be able to share and check out each others calendar's even if you're not at home.

I have no idea what options Ximian supports as to exporting / publishing calendars, but I've that Outlook has no problems using .ics files.
