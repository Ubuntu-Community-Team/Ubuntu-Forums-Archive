### Post by anotherdisciple on 2008-09-30
The effects are installed by default, but the manager isn't. So, you have to open a terminal and install it. Your terminal is in (Applications > Accessories > Terminal).

Type:
```
sudo aptitude install compizconfig-setting-manager
```

After you install it... go to (System > Preferences > Advanced Desktop Effects Settings)... and there you have it!

### Post by semitone36 on 2008-09-30
Also, before you are able to use Compiz you need to enable desktop effects. Go to  System -> Preferences -> Appearance, click on the last tab "Desktop Effects", and click on either "Normal" or "Extra".  This will allow Compiz to run.
