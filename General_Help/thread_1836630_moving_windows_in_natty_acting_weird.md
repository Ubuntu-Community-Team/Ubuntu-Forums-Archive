### Post by seawolf167 on 2011-08-31
For compiz, the "front-end" is the compizconfig-settings-manager. You can install that to tweak your settings by installing and running

```
sudo apt-get install compizconfig-settings-manager
```

As for video drivers, you can check those by clicking the power button in the upper-right, select system settings, then select Additional Drivers.
