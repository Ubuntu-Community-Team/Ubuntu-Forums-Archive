### Post by ajgreeny on 2007-08-02
If you remove it you will be forced to also remove ubuntu-desktop, but that will only matter if you try to do a distro version update using apt-get.  Ubuntu-desktop is just a meta-package which makes sure everything needed for a full ubuntu system is there.  Don't worry if you get rid of it, it does nothing for your system now.
