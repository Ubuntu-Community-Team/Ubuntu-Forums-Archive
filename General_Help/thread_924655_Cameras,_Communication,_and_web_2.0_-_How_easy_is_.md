### Post by jespdj on 2008-09-20
I first had a Microsoft VX-3000 webcam, and it wasn't possible to get it working properly on Ubuntu. I now have a Logitech QuickCam, which works out of the box. Also the built-in webcam in my laptop works out of the box. Before you buy a particular brand and model of webcam, search the forums here and Internet to see if there are known problems with that particular model.

Skype for Linux 2.0 does support webcams. Since Skype is not free (libre) software, it is not in the Ubuntu repository. The easiest way to install Skype is from the [Medibuntu repository](http://medibuntu.org/).

There are a number of applications for photo editing: F-Spot for organizing your photos, and GIMP for photo editing, for example. If your camera works as an USB storage device, then it will work without problems on Ubuntu. You can also just use a card reader to transfer photos from your memory card to your harddisk. The current version of Photoshop (CS3) does not yet work under WINE, as far as I know. I think CS and CS2 do.

Webapplications and Web 2.0 all work without any problems on Ubuntu. The default Internet browser on Ubuntu is Firefox, and you can also get Opera and other browsers.

Ofcourse there are a lot of development tools available in Ubuntu. The first thing to install is the package build-essential, which includes the basic tools you need for software development. There are development environments available for many different programming languages, including C, C++, Python, Ruby, Java, C#, etc.

Ubuntu works, like almost all Linux distributions, with a package management system. You use the package manager program (Synaptic) to download and install software from the Ubuntu repository, which contains more than 20,000 packages. There's free software for almost anything you can imagine. If software is not available in the Ubuntu repository, you can install it manually.
