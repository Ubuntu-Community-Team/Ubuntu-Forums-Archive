### Post by HereInOz on 2008-12-27
Go to System > Preferences > Keyboard, click the Layouts tab then the Other Options button.  Set the Caps Lock behaviour to:

CapsLock uses internal capitalization. Shift "pauses" CapsLock

This will probably fix your problem.

BUT:  It will probably return, particularly if you are using VMWare Server.  When it returns, go back into System > Preferences Keyboard, and change the setting back to Default, then back again to the previously set behaviour.  Should work again.
