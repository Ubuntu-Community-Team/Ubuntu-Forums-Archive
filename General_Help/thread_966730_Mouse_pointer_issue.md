### Post by vytalelementz on 2008-11-01
I am having a minor issue with my mouse pointer. I have downloaded a particular theme I would like to use instead of the plain old default theme and installed it properly (I think). When I try to use it, the theme only works partially when I hover my mouse over a browser window or some other applications. The theme goes back to default when I hover my mouse over my empty desktop or the docks. I have tried restarting x like you should and rebooting my computer with no luck at all. I saw a thread with the similar issue but it hasn't been updated since August of this year with a resolution. So, I'm hoping that someone out there has experienced this just recently as well and can clue in on what can be done to resolve this. Any help is appreciated. Thanks.

---

### Post by suprman2020 on 2008-11-01
Go to the Compiz Config Settings Manager. Click on General Options. Under cursor themes type in the name of the theme (write the name of the theme as you see it the appearance settings) and it should work.
