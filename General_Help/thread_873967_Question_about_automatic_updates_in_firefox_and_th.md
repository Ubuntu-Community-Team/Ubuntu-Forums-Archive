### Post by braddcadd on 2008-07-29
If you have no add-ons for firefox/thunderbird, the apps will not ask to update them on each startup.

It the core code of firefox/thunderbird is updated in the ubuntu packages this will appear in the notification area with all other other source updates.

Some firefox/thunderbird addons are actually provided through ubuntu.  These addons will come through the notification area also.

Good luck!
