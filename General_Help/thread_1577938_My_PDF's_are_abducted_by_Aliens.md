### Post by aytech on 2010-09-23
Hi, this whole story qualifies as an x-file, so i asked if you solved it already ;)
 
As for acroread, I forgot to mention you need to add Medibuntu repository to your repository list: 
 
```
deb [http://packages.medibuntu.org/](http://packages.medibuntu.org/) lucid free non-free 
```
 
You can do this by going to **Synaptic Package Manager &#8594; Settings &#8594; Repositories**. Select **Add **and enter the above adress. 
You will also have to add the repository key. Enter this to the terminal: 
 
```
 
wget --quiet [http://packages.medibuntu.org/medibuntu-key.gpg](http://packages.medibuntu.org/medibuntu-key.gpg) -O - | sudo apt-key add -

```
 
Then try installing it again
