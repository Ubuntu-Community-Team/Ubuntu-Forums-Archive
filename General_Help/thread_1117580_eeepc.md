### Post by snowpine on 2009-04-06
An eee is no different than any other computer. The reason eeebuntu, ubuntu eee, etc exist is that earlier versions of Ubuntu did not include the correct wireless driver for the eee. Starting with Jaunty 9.04, Ubuntu has full support for the eee hardware.
