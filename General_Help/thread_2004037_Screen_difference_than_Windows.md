### Post by QIII on 2012-06-15
Unity, which is probably the Desktop Environment (DE) you saw, is radically different from the Windows interface.  It seems to be a "love it or hate it" interface and you will find all sorts of heated debate around it.  I don't use it because I don't prefer it, but I don't exactly see why some despise it like they do.

But that is not the only fruit in the basket.  There are literally dozens of different fruit of all shapes, sizes and colors.  DEs can be changed.  Many can be extensively customized.

Unity, KDE, Xfce, LXDE, Enlightenment, etc, etc.

You can have it look any way you'd like.  You can try what you want and toss out what you don't like.

You can try before you buy by running a LiveCD.  It will be slow because it is running from optical media instead of magnetic, but you can get a good feel for how things work.  Download and burn Ubuntu, Kubuntu, Xubuntu and Lubuntu for starters.  Run each one and see what you think.

Just don't expect Windows!  It's a bit wild out here in the woods.

### Post by Quarkrad on 2012-06-15
Other option, that I tried, is to install Virtualbox and install the different Ubuntu versions that you can download as iso images (you don't have to keep making CDs).  This method makes it easy to install the different versions, alter them and play around to get the 'look and feel' that you want and make your decision  which to install as your main OS.  It is easy to delete Virtualbox once you have finished.

And I second the suggestion to virtualize.  I should have said that in my post.  The thing to watch out for if you do that, however, is the fact that you will be running on the virtualization software's hardware abstraction layer and may miss some details of how a "bare metal" installation will work.  But we can help you work through any issues you may have when you do a "real hardware" installation.
