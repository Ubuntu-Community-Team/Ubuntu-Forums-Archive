### Post by bogan on 2012-02-24
Hi!, **shelvacu**,

If you are logged in to Ubuntu, or to Ubuntu 2D. all Window menu bars for the active Window, are displayed in the top bar - Global Menu - accessed by placing the mouse cursor on the top bar.

That's normal in Unity for 11.10.

**Edit**:  If you want the previous display the easy way is to log-in to Gnome Classic, but you may need to install Gnome session fallback to get that option. Look for Gnome Session Manager sub titled 'gnome-session-fallback' in the Ubuntu Software center.

Chao!, **bogan**.
