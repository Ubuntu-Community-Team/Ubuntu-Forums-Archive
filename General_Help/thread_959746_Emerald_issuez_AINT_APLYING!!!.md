### Post by loomsen on 2008-10-30
Type
```
ccsm
```

go to Effects and activate Window Decoration plugin. After you enabled it, klick on the Icon again. The settings window for window decorations plugin will show up.
Add 
```

emerald --replace

```
to Command.

As you don't seem to be too experienced with compiz, here's a screenshot:
[[img]http://img511.imageshack.us/img511/8809/screenshot26lx2.th.png[/img]](http://img511.imageshack.us/img511/8809/screenshot26lx2.png)

Then just reload the window manager with similar option provided by fusion icon.

If you really installed everything as you say, it should work.
