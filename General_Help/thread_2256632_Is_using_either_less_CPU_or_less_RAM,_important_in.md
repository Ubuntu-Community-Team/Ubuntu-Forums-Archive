### Post by QIII on 2014-12-13
Hello!

There is no practical limit to the number of "CPU cycles" or "memory operations" used.  (OK, there is, but that is something that would happen over more years than the machine would ever be useful for anyway.)

The biggest issue with regard to longevity is heat.  More CPU use generates more heat, but a good HSF is going to keep the CPU within an acceptable range and good case fans will cool the memory modules.  (We'll disregard rabid overclocking of each.)

If you were to use 100% of your CPU and memory the problem would be one of performance -- and hard crashes in the case of exceeding the available memory.

Ideally, one would hope that the least possible amount of CPU and memory capacity be used.  However, things are rarely ideal.

### Post by grahammechanical on 2014-12-13
You could run the machine with no desktop environment and just the Linux command line and it would not make any difference to how long the machine lasts. People are still using machines a lot older than that. These machines have far less memory and single core CPUs and even when new everything was running at maximum because that was the way the machine was designed. Without things running full speed the OS would have been next to unusable. I would guess that most people upgrade the hardware not because it has broken but because it cannot run newer operating systems and applications.

In decades past all computers were run for 24 hours before being shipped. It was called "burn in." It was noticed that electronic components would either burn out in 24 hours or keep working on and on. It is more likely that mechanical bits and pieces will fail before the electronic components.

Look at this link and notice that these drives have 1.5 million hours MTBF

[http://www.bhphotovideo.com/c/buy/Internal-Drives/ci/6525/N/4294542385](http://www.bhphotovideo.com/c/buy/Internal-Drives/ci/6525/N/4294542385)

MTBF = Mean Time Between Failures. It is a figure that is a result of testing and it attempts to predict how long the drive will last. And remember, it may well be that it is the mechanical parts of the drive that could be causing the failure.
