### Post by magic8ball on 2011-08-17
This is a vague question and my answer might be too simplistic, but...

If I go to my "Places" pull-down menu, I have an entry for my Windows XP NTFS drive (in my case, labeled C_DRIVE) as well as an entry for "Computer" that I can use to navigate to the Windows drive.

If you've already gotten to your Windows 7 drive from Ubuntu and can't find your data, from the "Go" pull-down menu in the File Browser window you can select "Search for Files..." and try to locate it from there.

Or, you can boot into Windows and locate the data files, then boot into Ubuntu and navigate to them.

Good Luck.
