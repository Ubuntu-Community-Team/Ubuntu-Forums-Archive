### Post by mltucker on 2009-04-13
Hi,

Here's how I've done this:

1. Get [unetbootin]("http://unetbootin.sourceforge.net/"), 

2. download your xubuntu iso

3. Run unetbootin, tell it which flash drive you want to use, and tell it where the xubuntu iso is.

4. It will format it and make it bootable.

I do this all on Vista.  Very easy.  Do you want to have a separate partition on it?  I may have misunderstood...
