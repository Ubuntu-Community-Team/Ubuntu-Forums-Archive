### Post by Impavidus on 2017-02-13
So you know how to use the hexadecimal codes, but you'd like to use the decimal codes, as those are the ones you learned by heart. I understand, converting from decimal to hexadecimal isn't easy. But have you considered using compose sequences? Easier to learn than either decimal or hexadecimal codes. As in, <compose> a e &#8594; æ, <compose> o / &#8594; ø, <compose> A A &#8594; Å, <compose> - > &#8594; &#8594;. The downside is, there's not a compose sequence for every character.
