### Post by 3rdalbum on 2009-07-16
Every line of the sources.list must begin with one of the following:

deb
deb-src
# (any line begining with # is not read by the system)

The second-last line in your file begins with "http://" etc. Remove this line, or put "deb " before it.

To prevent problems like this, you should use the Software Sources program to add repositories, rather than manually editing the sources.list file.
