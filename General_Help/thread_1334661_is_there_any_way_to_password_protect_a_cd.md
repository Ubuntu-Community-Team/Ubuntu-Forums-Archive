Hey, kooley,

One option would be to put the data you need into a TrueCrypt (encrypted) volume and burn it. That would make the data as safe as your password is strong.

If you need any help setting up TrueCrypt, let me know. I had a little confusion when I first tried, but it's smooth as silk once you have it installed. You can download the .deb here: [http://www.truecrypt.org/downloads](http://www.truecrypt.org/downloads)
