### Post by Penguinnerd on 2012-04-30
Xubuntu would be quite fast. It might not like Unity very much, so I would not expect regular Ubuntu to be as fast on it.

I've got Xubuntu 12.04 running on a very similar netbook right now, and it's definitely a speed improvement over WinXP.

You can do a "test run" by making a Ubuntu (Or Xubuntu, or Lubuntu) USB boot disk and trying it out without actually installing it.

If you're on Windows, here's how:
[http://www.ubuntu.com/download/help/create-a-usb-stick-on-windows](http://www.ubuntu.com/download/help/create-a-usb-stick-on-windows)

Just remember: When you start it up, click "try Ubuntu" (or Xubuntu) instead of "install Ubuntu".
