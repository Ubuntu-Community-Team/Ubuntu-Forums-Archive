### Post by amgeex on 2006-09-06
Ok, since Applecrow's download page is down, and I needed to install Transmission, I decided to build a new debian package based of the latest svn version. This package was built with checkinstall.

This is Transmission version 0.7 svn-r850 (built today).

This will install Transmission to /opt/transmission

You may want to do this:

```
sudo ln -s /opt/transmission/bin/transmission-gtk /usr/bin/transmission-gtk
```

Here's the download link: [http://www.filefactory.com/file/a9b8d1/](http://www.filefactory.com/file/a9b8d1/)
