### Post by jmszr on 2009-04-18

Since you stated that you wanted to install Ubuntu as a program on Windows Xp you might want to consider Wubi: [http://wubi-installer.org/](http://wubi-installer.org/) . You can learn more about it here: [http://wubi-installer.org/faq.php#use](http://wubi-installer.org/faq.php#use)
