### Post by robert.rankin.jr on 2009-03-12
Startup Manager will work with it for you, and there are other programs in the repositories (search Add or Remove Programs for Grub) which will set certain settings. The big problem that I find with the idea of a graphical frontend to grub is that if you don't know what your doing, no matter how sophisticated the interface is, you can screw your installation of Ubuntu up. But Startup Manager is fairly safe, has little enough options that nothing should go wrong, and provides enough options to make it convenient.

Robert
