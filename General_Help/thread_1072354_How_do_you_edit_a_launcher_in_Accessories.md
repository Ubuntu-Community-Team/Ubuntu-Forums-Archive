### Post by drs305 on 2009-02-17
If you are trying to edit an *existing* launcher in Accessories, right click on the *Menu* icon on the panel and select "Edit Menus". Then click on Accessories in the left panel and in the right pane highlight the particular application and click on Properties to edit it.

You can also start the editing with the command "alacarte" in terminal.

To add an application to the Accessories menu, go through the same steps and select "+New Item".
