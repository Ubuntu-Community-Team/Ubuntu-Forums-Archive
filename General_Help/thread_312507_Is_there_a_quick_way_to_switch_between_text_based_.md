Hi. I was just wondering if there is any way to go into text based mode only, and then when done going back to the GUI. Note Im not just talking about opening the terminal.

---

### Post by christhemonkey on 2006-12-04
<Ctrl>+<Alt>+<F1> for the first virtual terminal
<Ctrl>+<Alt>+<F2> for the second virtual terminal
etc

Until:
<Ctrl>+<Alt>+<F7> for the first default X server (GUI)
