### Post by DeMus on 2010-03-14
> **BenAshton24 said:**
> You should install from the PPA: [http://www.openshotvideo.com/2008/04/ppa-instructions.html](http://www.openshotvideo.com/2008/04/ppa-instructions.html)

Look on the download page of the above mentioned link.
Choose your system and OS (for me 64 bit Jaunty) and you will see this:

64 Bit / Ubuntu 9.04
Download OpenShot (DEB Installer)
Download OpenShot Help Manual (DEB Installer)
Download Dependencies (DEB Installers)

Download all three lines. The last one is a tar.gz file. (Double) Click it and it will open file-archiver. Now you see 4 deb files which you can install automatically with gdebi. Just click on the filename and gdebi will open. Install them one by one.
Then install the main program which you also have downloaded. It works like a charm and from what I have seen already it is a wonderful program, coming close to Pinnacle Studio and others.
