### Post by mlind on 2006-06-07
I'm not sure about 3rd generation, but with my 2nd generation Mini I use

[LIST]
[*]gktpod - for tranferring files
[*]Rhythmbox - for listening music from iPod
[*]amaroK, Banshee - alternative players, can handle ipod file transfers
[/LIST]

Banshee is sluggish for me, I don't use it very often. amaroK seemed very nice, it's
KDE application, so as tradeoff you need to install necessary KDE base packages.
I warn you though, it once destroyed my Ipod's database by reducing song count to
magic number 255... I was able to get this fixed using iTunes tho.

I don't know if XMMS can play from ipod, you should try it. gtkpod can use XMMS
as player when playing audio from ipod though.

you can install software from apt repositories
```

sudo aptitude install gtkpod

```
