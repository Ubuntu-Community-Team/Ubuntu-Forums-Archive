### Post by dv3500ea on 2010-05-18
Try setting as the command for the shortcut:
```
cd ~ && ./frontier.sh
```
or perhaps:
```
perl -e 'chdir $ENV{HOME}; system("./frontier.sh");'
```
replacing frontier.sh with the name of the file.

### Post by qreeves on 2010-05-19
The script needs to be run with the game directory as the working path. Modify the bloodfrontier.sh script as such:

```
# BF_DATA should refer to the directory in which Blood Frontier data files are placed.
BF_DATA=~/bloodfrontier
```

Replace ~/bloodfrontier with the directory into which you extracted it. You may also be interested in using the [prebuilt package on playdeb (http://bloodfrontier.com/playdeb)](http://bloodfrontier.com/playdeb).
