### Post by Compintuit on 2010-01-30
If you'd like a real, open POSIX-compliant cell phone, buy it. If you had no idea what that meant, but would like freedom to do everything the hardware is capable of with it, buy it. I've heard some complaints about the software, though, but I'm sure any issues will be cleared up in the future. I don't believe it's unusable in any form. Perhaps it may not have some of the curves of the iphone, but I'd sure love one. 

[http://maemo.org/](http://maemo.org/) is the main site for developement; be sure to check it out. You don't actually need nokia once you have it; you can install apps without their approval. Tons are free. I'd wager you won't regret it if you buy it with the mindset that this will get better, isn't as finished as the iphone, but will be good a very long time. Cheers!

### Post by fragos on 2010-01-31
I've an N810 which mounts like over USB allowing me move files between Ubuntu and the N810. I also use Bluetooth to move files between Ubuntu and the N810. Aps on the N810 that use sqlite or text based data are compatible between the two platforms. I run the GPE set of PIM aps on the N810 and on Ubuntu and they are file compatible. N900 being new may not have all the 3rd party aps that run on the N810 yet. There's a very active Maemo community at [http://talk.maemo.org](http://talk.maemo.org).
