### Post by issih on 2008-07-18
Its just a Compiz plugin. You probably already have Compiz installed, if you are running the latest version of ubuntu (8.04).

After that you just need to have the package compizconfig-settings-manager (ccsm) installed. From there (System->Preferences->Advanced Desktop Effects) you can configure the various plugins.

I think the one you want is called application switcher, its something liek that anyway, you can choose to have the windows shown like that or in a cover flow style.

Hope that helps
