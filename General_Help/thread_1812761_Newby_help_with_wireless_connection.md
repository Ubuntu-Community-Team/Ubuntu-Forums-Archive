### Post by walt.smith1960 on 2011-07-27
[INDENT]Live CDs (or dual installations) are good.  I'm not sure Knoppix provides the best "out-of-box" experience.  Linux Mint seems to provide a good out-of-box LiveCD experience right now.  The limitation with LiveCDs is they don't "remember" things from session to session.  A common bugaboo with wireless in Linux is the need to download restricted software.  Even if you get wireless working via downloads or enabling drivers, the install will forget everything it knew once it's rebooted.  Hence the benefit of partitioning a disk and having BOTH Windows & one or more linux distros installed.
[/INDENT]
