### Post by Sam on 2008-10-15
Hello and welcome here !!

There is no need to install any drivers, except for specific cases, which will be explained if you install the system. But in general you don't need any "external media except the web for installing stuff after the system install. Every piece of software is in "repositories" which are managed with a single application.

But you can try Ubuntu without making any changes on your hard drive and check how it look like.


I hope you'll enjoy Ubuntu ;)
