### Post by p_quarles on 2007-11-01
I can give you a few suggestions. Hope one of them works.

1) Gnome's network manager gives a lot of people trouble. I've never used this, but I understand that many have better luck with [WICD]("http://wicd.sourceforge.net/").It's an alternative Linux network manager, more capable of roaming than the default app.

2) The knetworkmanager in Kubuntu 7.10 has so far worked better than anything else I've tried. If you don't care for KDE, you might try just installing that package and seeing how that works.

3) If you have administrative access to either of these wireless routers named "linksys," you might be able to clear up all these problems by simply renaming one of them.
