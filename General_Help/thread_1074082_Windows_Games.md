### Post by mail2345 on 2009-02-18
Have you tried virtualization software, like Virtual Box?
It can run windows in a virtual pc, so you can play your games without reboot.
You could also get one of the corparately maintained wine based game running programs, like Cedega(more for games) or Crossover(more fore productivity apps).

BTW, Wine Is Not an Emulator.
It intercepts windows system calls and changes them to linux system calls.

Games also work better in newer versions of wine.
Google can find some good guides.

Not, it's not an emulator.  The name "Wine" is derived from the recursive acronym "Wine Is Not an Emulator".  "Emulator" has a very specific meaning, and Wine isn't one.  It's a compatibility layer.
