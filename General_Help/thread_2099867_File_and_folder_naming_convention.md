### Post by Vaphell on 2012-12-30
there is nothing wrong with using spaces, you just need to know what do with them in cli:
- tab autocompletion should escape spaces automatically so you don't have to type everything, seriously - use it as much as possible to cut down typos and such
- you can quote paths - no space problem
- you can escape spaces manually with \ - no space problem
