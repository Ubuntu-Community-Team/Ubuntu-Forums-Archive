### Post by lah7 on 2014-08-24
Welcome to the forums! If Crossover has been correctly installed, it should be searchable in the Unity Dash.

I don't have Crossover to know this, but I believe you need to look for "Install Windows Software" or "Manage Bottles" to access the Crossover interface.

Similarly, if you're alright with the terminal, try pressing** CTRL+ALT+T **and typing '**crossover**', which **might** open up the interface.

If none of this works, Crossover might not be probably installed. We can help if that's the case, by providing some more details on how you're installing it.
