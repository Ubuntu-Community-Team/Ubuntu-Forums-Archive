### Post by lisati on 2008-09-23
I use the default settings of iptables, together with the default firewall settings of a Netgear router and a D-Link ADSL modem, and haven't been aware of any need to change anything other than setting up some basic wireless security. I'd only look into changing the settings if I were to look into hosting a web site on one of my machines.
