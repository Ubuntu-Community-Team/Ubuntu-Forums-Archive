### Post by cmnorton on 2010-10-12
It would help if your title mentioned boot problem.

For starters, you need a rescue CD. If you installed Ubuntu from a CD or DVD, there may be rescue programs on it. It sounds like you need to re-establish your boot.

Start here -- [http://fosswire.com/post/2009/05/restoring-overwritten-grub/](http://fosswire.com/post/2009/05/restoring-overwritten-grub/) -- and also search these forums.

On the Windows side, you'll need to run rescue from Disk1 of your installation media.
