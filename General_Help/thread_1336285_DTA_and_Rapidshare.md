### Post by pablopancho on 2009-11-24
I don't think you can configure DTA like that. If you really must - try Jdownloader. It is a java program - big disadvantage but you can get used to it and there are some advantages too, like:
- it works under Ubuntu (you need java runtime environment)
- you can configure premium account
- after it downloads archives it unpacks them
- monitors clipboard for rs links

[http://jdownloader.org/](http://jdownloader.org/)

### Post by curtj on 2009-12-09
> **njmalibu313 said:**
> Hi,
Can anyone tell me how I can configure Download Them All (DTA) to work with my premium rapidshare account? I have looked in the preferences tab in DTA, and still cannot find the option to add my account. Thanks a lot

Go to Rapidshare.com and log in to your rapidshare premium account.

Go to settings and check ***[FONT=Courier New]Direct downloads, requested files are saved without redirection via RapidShare[/FONT]***

You may also have to reenter your password on this page before clicking 'save' at the bottom of the page

This will save a cookie on the browser that you use to do this.

Use DTA in this same browser to download files via rapidshare and you'll get what you're after.
