### Post by ju2wheels on 2009-04-06
Yes thats no problem, however I would recommend using the same torrent app on both OS if possible, if not you may loose data.

I would recommend that when you start the torrent, you tell it to allocate the space for it all at once instead of on the fly. 

Secondly, when you boot into the second OS, begin the torrent download normally as if you havent started it yet, except tell it to save the torrent to the exact same location as you did on the first OS. Once you start it, I would tell the torrent to validate the data that has already been downloaded so that it realizes theres data there already and it will just resume where the other OS left off.

### Post by mb_webguy on 2009-04-06
As long as both torrent clients are downloading to the same target directory, there shouldn't be any problem.  You shouldn't even have to use the same torrent client.  I did this a while back using uTorrent in Windows and Deluge in Ubuntu.

To be safe, you should probably force the client to recheck the torrent at the beginning of each session.  This should fix any problems with the status bar.
