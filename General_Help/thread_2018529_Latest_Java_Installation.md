### Post by QIII on 2012-07-06
Installing from their website is NOT the easiest method.

See the link in my sig and look for the link entitled "Using webupd8.org's strikingly simple method"

Webupd8's installer from their ppa keeps the Oracle Java version updated as new updates are released when you do your normal Ubuntu updates.  It also automatically installs and activates the browser plugin. Installing from Oracle's website requires that you watch for when they release a new update, uninstall Java and reinstall.
