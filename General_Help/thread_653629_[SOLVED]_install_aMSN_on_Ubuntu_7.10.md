> hi guys , can anyone guide me how to install latest version of aMSN on ubuntu 7.10 ? im kind of new to linux so step by step guide would be really great . thanks in advance .

The easy way on doing this is go to your "Synaptic Package Manager" And Search for aMsn!
It's in System> Administration >Synaptic Package Manager
In Synaptic Package Manager right click on aMsn and go to mark for installation.. Then click on apply.
Then when it's all done on stalling amsn go to Applications>Internet>aMsn

### Post by Sef on 2008-01-01
There is a very easy way to get the latest version from amsn.  (Synaptic has .96; the latest version is .97.)

Read the [install]("http://www.autopackage.org/docs/howto-install/") page first.

Then download the [amsn package]("http://www.amsn-project.net/linux-downloads.php") (aMSN Installer for Tcl/Tk 8.5.)
