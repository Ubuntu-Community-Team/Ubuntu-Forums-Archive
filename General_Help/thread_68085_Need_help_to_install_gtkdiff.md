It's just a quick answer, but tkdiff do the same and you can install it easily : ```
sudo apt-get install tkdiff
```For repositories problems have a look [here](http://www.frankandjacq.com/ubuntuguide/5.04/index.html#extrarepositories).

### Post by frodon on 2005-09-22
I don't know if there is an icon in the gnome menu however you can create a shortcut in gnome menu using [smeg](http://www.frankandjacq.com/ubuntuguide/5.04/index.html#menu-editor). personally i open it with command line : ```
tkdiff file1_path file2_path2 
```
