### Post by SickTwist on 2005-10-19
Ubuntu isn't "licensed" per se.  The individual programs each have their own licenses.

Since Ubuntu is based only on [Free Software]("http://www.gnu.org/philosophy/free-sw.html"), you are allowed to distribute it however you'd like. The only catch is, if you modify the source code of any programs, you may (depending on that program's license) need to include the modified source code. Otherwise you don't need to worry about that aspect since the source code is available from the repositories.

So go ahead and give it away--no worries.  That's one of the beautiful aspects of Free Software :)

### Post by SickTwist on 2005-10-19
The programs are individually licensed--Canonical can't change that. There are hundreds of programs in an Ubuntu installation so Canonical can't just say "The GPL applies to everything" because it doesn't. However, the licenses that are used (if not GPL/LGPL) are similiar to the GPL in the sense of the Freedom that they provide.

The Ubuntu name and logo are trademarked. Those are the only things Canonical can control.
