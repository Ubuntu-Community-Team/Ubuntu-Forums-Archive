### Post by m_duck on 2008-07-23
I'm not sure if it is what you want as the screen shot doesn't seem to be working but if you press alt + r, with the desired window selected, you can use the arrow keys to resize it. Likewise, you can press alt +m to move the windows with the arrow keys. Press enter when finished moving/resizing to return to the normal screen.
