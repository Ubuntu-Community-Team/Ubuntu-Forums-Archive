### Post by leg on 2010-06-09
> **lovinglinux said:**
> You could try [Ogre](http://www.ogre3d.org), which is a very popular open-source engine. I remember they were thinking about implementing it in Blender, so perhaps you could find a plugin to do that.
[Ogre]("http://www.ogre3d.org/") is a rendering engine so you would also need to use a physics engine, sound engine alongside if you went down that route. Having said that there are some great links on their site to these engines and tutorials on how to get them incorporated. [Irrlicht]("http://irrlicht.sourceforge.net/") is also a 3D rendering engine so you would be in the same position if you use that. 

Wikipedia has a [list of games]("http://en.wikipedia.org/wiki/List_of_game_engines") engines you could look through to see if any suit your needs.
