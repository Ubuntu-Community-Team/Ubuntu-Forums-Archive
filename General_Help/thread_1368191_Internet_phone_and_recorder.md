### Post by garryknight on 2009-12-30
You could try vsound. You run it from the command line, e.g. like this:
  ```
vsound -f myskypecall.wav skype
```It runs Skype, you make your call then close Skype and you should find that it has created a .wav file. You can then convert this to something like .ogg or .mp3 using lame,  SoundKonverter, or something like that.

I've used vsound in the past with RealPlayer and other apps but not with Skype so I can't guarantee that it works, but worth a try, no?
