### Post by snowpine on 2011-08-12
I have 2 netbooks with similar specs (1.6 atom with 1gb ram) and I've tried dozens of distros: Ubuntu, Debian, Fedora, Slackware, etc. I have yet to find a distro that does **not** run well on a netbook! :)

Ubuntu is a great place to start, in part because the interface that was formerly "Ubuntu Netbook" is now the standard default (Unity) for Ubuntu. In other words Ubuntu has abandoned the traditional "desktop" model and has become a netbook-centric distro. ;)

### Post by LMP900 on 2011-08-12
I use Ubuntu 11.04 with Unity on my netbook and it runs just fine. I should note that I have 2GB of RAM and a 16GB SSD installed. I like Unity because it makes efficient use of the limited resolution. I keep most windows maximized and switch applications via the launcher bar.

I've tried Joli OS and have considered Chromium OS and MeeGo, but the full functionality and efficient use of space makes Ubuntu my choice for a netbook.
