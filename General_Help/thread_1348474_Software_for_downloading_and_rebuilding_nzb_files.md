### Post by BrandonC19 on 2010-11-15
You need to open a terminal and type ```
sudo apt-get install unrar-free 
``` to unrar those .RARs.

   Also once "unrar-free" is installed, you can just right-click your .RAR's and select "Extract Here" as opposed to doing it via the command line.
