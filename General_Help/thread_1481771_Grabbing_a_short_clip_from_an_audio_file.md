### Post by t0p on 2010-05-12
I've got a short mp3 file from the BBC's *League of Gentlemen* comedy show. In the clip, Papa Lazarou tells his dwarfs to put a poster in the local shop.  Then he says "Tell them the circus is coming to town."

I'd like to grab that sentence "Tell them the circus is coming to town" so I can make it into a ringtone for my phone or something.  What simple utility is available to allow me to do this?  Something simple please, I'm not a Rock DJ or nothing.

### Post by 3rdalbum on 2010-05-12
Audacity is an audio editor; you can copy that part of the file just by dragging across it and hitting Control-C, then go to File > New and paste the clip in using Control-V.

Now go to File > Export and export it as an MP3, which I imagine your phone uses as a ringtone.

Audacity is quite powerful if you want to experiment with it, too. For instance, if this is an old sound clip or if it's quiet, you might want to run the "Amplify" filter.
