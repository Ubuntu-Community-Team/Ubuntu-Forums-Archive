### Post by WorMzy on 2010-12-17
In most cases: Alt+F2, metacity, run.

If you don't use metacity (the default window manager) then substitute it with whatever you do use. e.g. Alt+F2, compiz, run.

If it happens persistently, try running the command from a terminal and see if there's any error messages.

### Post by ghostborg on 2010-12-17
There is a compiz icon called fusion-icon in synaptic package manager that once installed is placed under Applications=>System Tools and can be loaded at startup if you wish.
Once its loaded it puts an bluebox w/ arrow icon on your panel.
You can then right click on it for options like Reload Window manager.
