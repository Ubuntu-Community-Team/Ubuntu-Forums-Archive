### Post by titis on 2011-10-09
Accidently I deleted Google search from engine default options

So, I found to restore

1. Preferences/Basics/ManageSearchEngine

2. Other search engine

3. Add in each field

```
[SIZE="2"]Google  google.com  http://www.google.com/#search&q=%s[/SIZE]
```
