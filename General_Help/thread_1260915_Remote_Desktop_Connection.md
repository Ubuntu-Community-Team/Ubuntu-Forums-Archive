### Post by P4man on 2009-09-08
the easiest to setup is just using the built-in remote desktop. Go to system > preferences > remote desktop. Enable it, and you should be able to log in from windows using any VNC client.

You may find this works rather sluggish though. If you want a faster solution, try freeNX. Step by step instructions here:

[https://help.ubuntu.com/community/FreeNX](https://help.ubuntu.com/community/FreeNX)

You need to download the windows client for your windows machine obviously.
