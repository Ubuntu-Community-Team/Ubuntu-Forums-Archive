### Post by BoneKracker on 2007-08-07
Use Gimp.  You'll also need a vector graphics package if you want to create svg files (scalable icons).  The rest is art.  Most people seem to take an existing icon theme and modify it.

While there is probably a utility for converting a deb directly into a tarball, I would simply extract the contents of the deb (dpkg), then tar them.

If you are actually interested, you might find guidance on the Gnome wiki (there used to be an active Gnome Art community there, which I think is more or less defunct, but they have probably left how-to material lying around).  You might also find someone who can give you tips at the Gnome-Look site (gnomelook.org) as well as many examples of icon themes.

I'd like to be of more help, but I lack artistic talent and have never ventured into the realm of creating icons.
