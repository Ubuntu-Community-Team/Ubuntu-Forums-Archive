### Post by dragonfly41 on 2014-03-02
These ideas are **untested **since I've only just started trying out Wuala (encrypted) secure cloud storage.

However I've been using Zotero for a while and I'm interested in managing private bookmarks and notes shared with authorised user groups.

Suggestion:

Install Firefox add-on Zotero [http://www.zotero.org/](http://www.zotero.org/)

Import into Zotero your existing Firefox database of bookmarks.

Create new account at Wuala.   [http://www.wuala.com/](http://www.wuala.com/)

Sync your Firefox Zotero database with Wuala secure cloud storage.
You could sync with zotero cloud storage but it's not encrypted.

[http://www.zotero.org/support/zotero_data](http://www.zotero.org/support/zotero_data)

> "It is mine, precious!!!" 

If you prefer to run your own server instead of using Wuala (even though encrypted) you can research deploying Zotero in your private server.

[https://groups.google.com/forum/#!topic/zotero-dev/p5Mlvnw3nXY](https://groups.google.com/forum/#!topic/zotero-dev/p5Mlvnw3nXY)

I've not yet tried this approach.
