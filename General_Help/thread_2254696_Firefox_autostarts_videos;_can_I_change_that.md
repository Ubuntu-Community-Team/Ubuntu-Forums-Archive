### Post by flaymond on 2014-11-29
I will use NoScript add-on to disable pop-up videos. (worked for me). Do you right-click the videos? I got several options when I right-clicked it. Or you can try this -         [https://addons.mozilla.org/en-US/firefox/addon/flashblock/](https://addons.mozilla.org/en-US/firefox/addon/flashblock/)  ,    [https://addons.mozilla.org/en-US/firefox/addon/image-and-flash-blocker](https://addons.mozilla.org/en-US/firefox/addon/image-and-flash-blocker)    Those are flash blocker( can configure it to stop autoplay videos by default). If still not working, try this   [https://addons.mozilla.org/en-us/firefox/addon/stop-tube/](https://addons.mozilla.org/en-us/firefox/addon/stop-tube/)      Please try these add-ons first before using about:config on firefox because if you do mistake in configuring it, you might break your Firefox.
