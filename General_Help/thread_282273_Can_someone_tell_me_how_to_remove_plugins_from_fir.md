### Post by catanzag on 2006-10-23
You are welcome, Xzallion!!!

Some time ago I had a similar problem with Java applets under Windows, and I wondered where Java plugin was hidden in the extension window....

*Plugins* extends browser capability in handling file types different from html, text and main image formats (Java, Quicktime, Realvideo, Flash) and are compiled natively for each O.S. Differently, *extensions* just extend (sorry for the repetition) browser functionality (GUI) or add mini applets (reminder, calculator, downlader and so on); they are generally platform independent (but not all, for instance lightning is specifically compiled), an they can be handled in a coherent way for installation and management.
