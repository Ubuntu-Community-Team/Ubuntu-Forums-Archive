### Post by cfrancoec on 2011-08-22
It's a simple tip for  go into a folder after making the dir in terminal.
```
mkdir somedir && cd $_
```Also you can use it for editing files
```
touch somefile && vi $_
```And also for removing the file or directory in the same line, I don't know for what, but you can use it :P
```
touch something && rm $_
```
