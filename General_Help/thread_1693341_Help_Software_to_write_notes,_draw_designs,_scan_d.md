You can also have a look here:

[http://www.osalt.com/](http://www.osalt.com/)

It is a good resource for finding

open-source alternatives to

commercial software.

Try to search for "one note"

and see what you find.

A lot of software on that page will

be available from the repositories.

Good luck.
