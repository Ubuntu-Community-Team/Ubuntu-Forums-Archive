### Post by mcduck on 2006-12-17
I suppose the script uses Metacity's shortcuts to open gnome's System Monitor? Beryl replaces Metacity as window manager that won't work.

But Beryl has it's own shortcut options:

1. Open Beryl Settings Manager
2. Under General Settings/Commands type 'gnome-system-monitor' into Command 0
3. Under General Settings/Keyboard enable 'Run Command 0' and set the keys to Control, Alt and Delete
