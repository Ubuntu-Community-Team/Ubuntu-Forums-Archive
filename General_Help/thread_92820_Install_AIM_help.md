I was pretty sure that Gaim came pre-installed with Breezy, but here goes: 
There are a couple of ways to do it. The easiest (for me at least) is to open up either gnome-terminal or konsole, and then type the following at the prompt:

sudo apt-get install gaim

then enter in your log-in password in when it asks you for one. 

Ubuntu will automatically download the correct packages and install them for you. 

you can then either type "gaim" (without the quotes) in the command line to run the program, or it should set up a menu item under "Internet", I believe. 

[Gaim]("http://gaim.sourceforge.net/") has pretty good documentation to set up and connect to AIM. It also can connect to MSN, Yahoo, and other Instant Message networks. 

You can also use Synaptic Package Manager to install Gaim, or other programs. I'll leave that as an exercise to you to figure out. :-) Hope that helps. Good luck!
