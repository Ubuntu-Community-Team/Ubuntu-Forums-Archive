### Post by Copernicus1234 on 2010-05-08
Firefox. Superior extensions and themes compared to Chrome, and the speed difference for rendering web pages is not noticeable enough to be factor for me. It starts faster than Firefox though.

I have both Chrome and Firefox on my system but keep going back to Firefox. If youre going to use it, please put on a decent theme like [Chromifox Basic]("https://addons.mozilla.org/sv-SE/firefox/addon/8782"), otherwise its ugly.

The fastest browser is Midori. But seriously, we have a ridiculous amount of really great browsers today, most of them cross platform. **Try them all**. There is true competition in that market now that Microsoft dropped the ball.
