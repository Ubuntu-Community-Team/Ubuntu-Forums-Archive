### Post by sdennie on 2008-12-26
There are two ways.  The first is to use System->Preferences->Keyboard Shortcuts and the second is to use ccsm (if you are running compiz).  To install ccsm use:

```

sudo apt-get install compizconfig-settings-manager

```

Then go to System->Preferences->Advanced Desktop Effects->General->Commands and use the keybinds and commands parts.
