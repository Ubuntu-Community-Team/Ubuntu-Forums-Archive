### Post by warl0q on 2012-04-06
ubuntu is not freezing it is just firefox so just make sure that alt+F2 is assigned in keyboard shortcuts to the "Show the run command prompt" under the system heading. Then whenever flash makes firefox freeze just hit alt+F2 and type xkill, hit enter, then click anywhere on the screen and this will close the frozen window without you having to do a hard shutdown.
