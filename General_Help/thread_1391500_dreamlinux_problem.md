### Post by ankspo71 on 2010-01-26
Hi,
Could it be that you forgot how to make burn the cd as an image? If you open your cd drive through "my computer", then open the cd drive to view the contents, and all you see on the cd is a copy of an ISO file, then this means you accidentally made a data cd. In whatever program you are using to burn the cds, you need to select something like "Burn As Image". Not all cd burning software might have this feature so you may have to install a second program. Two programs are mentioned here: [https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)
Hope this helps.

### Post by ankspo71 on 2010-01-27
Hi,
There is no need to extract the iso file. All you would need to do is launch your cd burning application, look for a menu item or button somewhere that says something like "burn an image", "burn an iso" or something to that extent. Once you click on it, it will ask you to select the image file you would like to burn, choose the file to burn (dream linux iso), then select OK or burn.

 If you were to download and install ISO Recorder, you would right click on the ISO file and select "copy image to cd" according to this tutorial [http://isorecorder.alexfeinman.com/HowTo.htm](http://isorecorder.alexfeinman.com/HowTo.htm) 
Here is the download page to ISO Recorder [http://isorecorder.alexfeinman.com/isorecorder.htm](http://isorecorder.alexfeinman.com/isorecorder.htm)

As far as I know, Vista doesn't have the ability to "burn an image" without having another program installed. I know my XP didn't.
Hope this helps.
