### Post by Zimmer on 2006-10-12
Have a look at the help file under 'Variables'
You can set a page number field with an offset  which means it will start numbering after the offset number of pages  (put in a minus sign to skip the first x pages ie. -3 would start page numbering as 1. from the 4th page)
