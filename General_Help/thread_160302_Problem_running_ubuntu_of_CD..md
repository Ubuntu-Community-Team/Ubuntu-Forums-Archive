### Post by Qrk on 2006-04-14
Make sure you have the "Live Disk," then put it inside your cd tray before you turn your computer on. You'll probably need to set the Bios to boot to the CD. When your computer first turns on, you should see something like "Press ESC to enter setup." 

Press whatever key it is, (Esc, Del, F12 are good bets) and change the boot order to put your CD drive before your hard drive. Reboot, and you should see the Ubuntu Logo.
