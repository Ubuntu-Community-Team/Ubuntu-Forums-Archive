Where can i in kubuntu manage the programs and services that is run at startup.

### Post by snova on 2009-01-28
Open System Settings, Advanced, Autostart. Click Add Program and type in **/usr/bin/nm-applet**. Then Gnome network manager will start automatically.

As for disabling KNetworkManager, I simply removed it...
