### Post by sevendogsbsd on 2018-04-28
Well, I found this:

```
Super+a
```

That's the windows key + a, opens the apps view. It's not one key but better than nothing. Might be an extension out there for this but the activities view invocation would have to change because that's what the meta (super) key does now.
