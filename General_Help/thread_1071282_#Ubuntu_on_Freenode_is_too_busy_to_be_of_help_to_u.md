### Post by Will Pittenger on 2009-02-16
I have attempted to ask for help on Ubuntu related topics on Freenode's #Ubuntu, but with 1000 users, it seems no one notices pleas for help.  There are just too many conversations going on.  I think that we should divide it up by topic.

[LIST]
[*]#Ubuntu (existing channel) -- for those considering installing Ubuntu.  Other users would be directed elsewhere.
[*]#Ubuntu-install -- For those with problems installing Ubuntu.
[*]#Ubuntu-config -- For those with problems related to configuring Ubuntu.
[*]#Ubuntu-hardware -- for those with hardware and driver related problems.
[*]#Ubuntu-Apps -- for those with problems related to applications
[*]#Ubuntu-Beta -- Granted, there is already a #Ubuntu-Develop channel.  However, I would suggest that it be restricted to coding questions while the Beta channel would handle issues about the ***installed*** instances of development versions.
[*]#Ubuntu-ui -- User interface issues.
[/LIST]
