You have posted the same thing on another thread. Firestarter is NOT a gui for ufw and is not even recommended anymore. The graphical interface for ufw is [gufw](apt:gufw).

Firestarter is just another firewall manager like ufw and they both create rules for the iptables. Don't use both, otherwise they will create conflicting rules.
