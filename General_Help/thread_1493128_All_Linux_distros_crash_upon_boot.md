### Post by Jerry N on 2010-05-25
I had a similar problem with a geForce 7600 video board with Ubuntu 9.04 and earlier.  It seems to work much better with 9.10, 10.04, and LinuxMint 9 and has never been any problem at all with Win XP and Win 7.  With Ubuntu 9.04 I had to boot the desktop live CD in some sort of compatibility mode (I don't remember exactly what it was called) and install from there.  After installation, proprietary nVidia drivers could be installed normally.   Parted Magic will not boot on that computer (Athlon 3800, 2gb RAM, Gigabyte MB) but gparted live and recent versions of Puppy linux boot OK. 

Added Comment:  A little hard to convert to Linux if you can't boot Linux!
 
Jerry
