### Post by Sammi on 2007-11-28
I think you are searching for two separate things:

*1. Something that can trace the stolen computer, so that it can be recovered.*

I guess some hardware like a built in GPS could do this. This isn't really a OS feature IMO.

*2. Something that can protect the stolen data, so it can not be read by the thief.*

Some kind of encryption software could do this, but I haven't tried any. A quick search on Google gave me a lot of choices, but these two seem particularly interesting, I think:
[http://en.wikipedia.org/wiki/TrueCrypt](http://en.wikipedia.org/wiki/TrueCrypt)
[http://en.wikipedia.org/wiki/Dm-crypt](http://en.wikipedia.org/wiki/Dm-crypt)
