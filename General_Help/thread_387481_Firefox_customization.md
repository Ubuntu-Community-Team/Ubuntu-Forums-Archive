### Post by garybrlow on 2007-03-18
Some features have been turned off in the linux version like select all on click on the url/address bar and others. You have to use the about:config option to turn them on/true some of them have been flagged false. Type about:config on the url/address bar and click enter. on the filter are type mouse. Go over them and see what has been turned off/false and toggle them. You can also change other values like the number of lines scroll etc. You may have to run Firefox as Root to make the changes permanent or maybe not. Try changing them as current user first.

Am currently using the official version of firefox not the deb installation that comes with Ubuntu. They are basically the same cept that the official version doesn't have to be installed, comes all in one folder and is executable. It shares the same configuration files as the Ubuntu/Debian version so it should work. :)
