Sounds like you don't understand how updates work in Ubuntu... Applications are updated to major new versions every six months, when you install the next release (next one will be Karmic Koala in October). You will never see a major application update (for example from 1.0 to 2.0) within a current release, only minor upgrades/bug fix/security update (for example 1.0 to 1.1). This is done to keep Ubuntu stable.

Since Firefox is such an important application, Ubuntu makes it really, really easy to get FF 3.5 in Jaunty. But it won't happen automatically; you have to make it happen. :)
