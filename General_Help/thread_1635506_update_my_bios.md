### Post by Mark Phelps on 2010-12-02
Messing with the BIOS is a sure-fire way to "brick" your PC -- if you don't know what you're doing.

Motherboard manufacturers all provide a BIOS update utility -- but in nearly all cases it is one of the following:
1) Must be written to a bootable diskette (boots in MS DOS)
2) Involves pressing some special key sequence and then reading/writing to a diskette/USB stick.

As already said, it is specific to make and model motherboard.

And, NO, it can not be run from inside Ubuntu.
