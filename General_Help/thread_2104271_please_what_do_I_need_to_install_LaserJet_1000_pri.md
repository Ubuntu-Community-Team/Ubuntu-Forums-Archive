### Post by Mark Phelps on 2013-01-12
CUPS is the thing to try to install the printer.  You do that, with the printer connected, by opening a browser and entering "localhost:631"  (without the quotes).

Once there, you will be able see menus, one of which will be allow you to add a printer.  IF CUPS sees your printer, it will be listed and you can walk through the process of adding it and choosing drivers.

I've done that with several HP printers without problems.

There is also HPLIP -- which others have used -- but I haven't had to resort to using that.
