I think anyone who has setup an exhange server with ubuntu has seen this help page: [http://nknu.net/ubuntu-14-04-exchange-configuration-thunderbird-pidgin/](http://nknu.net/ubuntu-14-04-exchange-configuration-thunderbird-pidgin/)

I continuously got Java runtime errors whenever I tried to setup davmail for my exchange server.  I searched high and low for an answer, and found one in the deepest corner of the Internet.  

The simple answer that has helped me, simply install Sun's version of Java.  I do not know why this fixes all of the certificae and runtime issues, it simply does. 

I'm posting this so that I can help others, and so I can fidn this help later for myself as well.
