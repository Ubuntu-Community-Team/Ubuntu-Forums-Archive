### Post by sudodus on 2015-01-10
There is a reason for the term 'pre-release'. These updates are still being tested. So ***the risk is greater*** than with released packages.

If you have a good backup, (or know how to repair your system in other ways), you can try them. The testers run them all the time. Most of the time they work.
