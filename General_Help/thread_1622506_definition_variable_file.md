### Post by col48 on 2010-11-15
At least two possible meanings, depending on the context.

1. a file name which is stored (as a text string) in a variable in the source code of a program.  It is therefore not fixed.

2. a file in which variables are stored, usually both names and values.  In such a file you may find a line like
defaultuser = geronimo
for example.
