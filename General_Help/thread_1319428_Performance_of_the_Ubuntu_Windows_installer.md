### Post by tom.swartz07 on 2009-11-08
personally, I find it easier just to use the livecd to create a separate partition. more recently, ubuntu has been really great at allowing you to pick your partitions and how much room you need, etc. 

there would probably be a bit of performance boost from running on its own partition too- think of it this way:

1.) you dont have to worry about losing ubuntu if windows crashes (which it will, one time or another)
2.) you have the option to customize the location of your specific partitions (you could install the os on one, then have all of your user data on another, etc) --again- data security
3.)with a livecd install, you get a full version of GRUB, rather than the gimped windows bootloader that is changed. Grub has more features, and is easier to work with if something goes wrong.


plus many other advantages.

I would def suggest going with separate partition, and install from there.
