So the key :-) keyboard shortcut is:
CTRL+ALT+TAB
With it you change the focus from the desktop to the panel, and vice versa
When the focus in in the panel you can move from icon to icon with TAB (to the right) or SHIFT+TAB (to the left). Then Enter key (sometimes F10 instead) makes the same as left-clicking with the mouse, and SHIFT+F10 makes the same as right-clicking with the mouse. ESC key (sometimes Enter instead) takes the menus out.

Important!!

If you can't do this it's probably because you're using Compiz instead of Metacity.

The workaround is to use Metacity instead.

2 easy ways to do it:
a) ALT+F2 -> gnome-appearance-properties -> Visual Effects -> None
b) ALT+F2 -> metacity --replace
