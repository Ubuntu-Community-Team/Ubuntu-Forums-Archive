### Post by diaa on 2008-02-04
If you are using mplayer you can delay the subtitles by passing
[LEFT]```
-subdelay <sec>
```
parameter, you can also check [mplayer documentation]("http://www.mplayerhq.hu/design7/info.html#docs") for more details

if you want to change the subtitle file you can use the program [*subtitleeditor*]("http://kitone.free.fr/subtitleeditor/"), I haven't tried but I think it can do this, you can install it by running
```
sudo apt-get install subtitleeditor
```
[/LEFT]
