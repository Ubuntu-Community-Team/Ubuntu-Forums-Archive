### Post by DeMus on 2011-02-27
> **rahulbest said:**
> i m new one ubuntu.....i find its interesting nd stable......
thinking of buying dell laptop with preloaded windows 7....now i want to ask u can i install ubuntu along with windows 7 and if i do that then my warrenty gets void or not???

Yes, you can and no, probably not. But ask Dell about it for sure.

You need an empty part of your hard disk, so in Windows you need to make the C:\ partition smaller. How to do that is something I can't help you with since I don't have Windows here.

Don't mess with the partition Dell placed on your hard disk to recover your Windows installation

Then install Ubuntu using the CD and choose to either install Ubuntu along side Windows or choose the manual install and point to the empty hard disk space.

Before doing this read more about it. There are lots of websites explaining how to do this.

It would be a good idea to start Ubuntu first from the live CD to see if everything works, especially videocard and wireless network. If not, you better not start using Ubuntu. Ask around here if there are any others with the same type of laptop to see if things work. People always help here.

Have fun using Ubuntu.
