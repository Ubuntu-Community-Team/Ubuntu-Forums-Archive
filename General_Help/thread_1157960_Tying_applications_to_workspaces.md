### Post by lovinglinux on 2009-05-13
You can open specific applications on specific workspaces using the "Place Windows" plugin on Compiz.

Open "CompizConfig Settings Manager", then click the "Place Windows" button, then open the "Fixed Window Placement" tab, then add the application to the "Windows with fixed viewport" list, selecting the viewport it should be placed.
