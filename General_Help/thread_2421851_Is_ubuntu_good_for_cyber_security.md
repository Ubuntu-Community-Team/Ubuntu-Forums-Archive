### Post by 1clue on 2019-06-28
You can find the tools, and you can make Ubuntu as secure as a security-specific distro. But it will be more work.

The upstream packages are all the same. Each distro will have its own rules about when to upgrade a version of some package, so Ubuntu might have newer or older versions of each package than some other distro does.

The biggest difference is that a security-specific distro has the fewest packages possible to handle security tasks. You can add more packages, but the question you need to ask is, should you?

Another difference is that a security-specific distro will almost certainly compile each package with different options enabled or disabled.
