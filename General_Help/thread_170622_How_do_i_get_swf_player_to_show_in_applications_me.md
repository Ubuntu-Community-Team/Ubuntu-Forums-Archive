### Post by skippy81 on 2006-05-05
To add an application to the menu:

1) right click your mouse where "Applications" is displayed on the top menu.
2) select "edit menus"
3) select the group you want to add to from the left hand panel.
4) click the "new entry" button
5) now you need to give it a name (swf player seems wise :) and an icon.  Most importantly you need to type in the command you use to access the player from the terminal.

Post the name of the swf player package you installed and me or someone else will be able to tell you the command name.
