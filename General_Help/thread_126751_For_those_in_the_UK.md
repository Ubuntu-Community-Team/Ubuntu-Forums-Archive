### Post by simo_lin on 2006-02-07
Hi

Have heard of the wires only approach to broadband - I think they just set your telephone line up at the exchange and then you need to get either an ADSL router and the splitter (so you can use voice on the same line) and an ADSL modem or just an ADSL router/modem and a splitter.

Netgear do a router with integrated ADSL modem.

You need to then set up the router/modem with the relevant login info and you should be online.

The router/modem in one device is a good option if you can get it as just one more power plug (!) you need to connect and also you get a firewall (of sorts) on the router and also NAT to allow many pc's within your LAN to use the router.

Think long and hard if you want to use a USB modem as I have had a hellish struggle with these - and that was under Windows not Linux - your mileage may vary though.

Hope that make sense as it is my very first post on these forums .........

### Post by Brynster on 2006-02-07
a "Wires only connection" is supplied only by DSL providers.

(BT, Wanadoo, Bulldog, etc)

This gets you ADSL subscription activated from there you need to add an ADSL modem router (may i reconmend the Netgear DG834 or the DG834G if you want a wifi connection)
thats basically it. Pick a provider and say switch me on in A nutshell

Cable providers (either Telewest or NTL) don't provide a wires only connection as you need a media converter (some times referred to as a cable modem)installing to change electronic data infomation into optical data to run down the fibre optical cables they use to pipe TV,Phone and data to you house.


Hope this clears things up for you,
