### Post by mullinsn2000 on 2010-03-26
Well the answers given were quick, short answers and details were not given.  Ubuntu is an OS.  If you should install it is up to you.  Yes you can keep your Windows XP, if you partition the HDD correctly and install Ubuntu side by side with Windows.  Do not count on being to run you Windows apps in Ubuntu.  Some Windows apps will run in WINE which is a Windows emulator.  Not all programs work in WINE and many do not work like you may want them.  Ubuntu (or Linux in general) has many programs that will do what you want.  The might not have the fancy names of your Windows programs, but they will meet or exceed your needs.  Use Office in Windows?  Try Open Office in Ubuntu, or Windows.  It is free and in my opinion works better than Office.  This is just an example of what I mean.

### Post by panti19 on 2010-03-27
you can download the disc image from ubuntu.com and burn it with say Nero or any other iso burner. 
it is a live cd which means you can try the system without any changes on your drive. just put the cd and restart, it will boot ubuntu without affecting windows. 
it will automatically load the drivers for all(or most) of your devices so you will be able to use all the applications available. However, since the system will be running from the RAM, it will be slower than if it were to run from the hard drive so don't judge it's speed.
on the desktop you will have a shortcut to the ubuntu installer which will give you information about the options you have in installing the system.

### Post by oldfred on 2010-03-27
There are three main versions of windows XP, Vista, & 7. There are versions of Ubuntu released every 6 months. The differences are not significant in most cases. So review several of the instructions on dual booting. 

With XP you can use gparted (on liveCD), other versions of windows you want to use the windows tools to resize the windows partition to make some room to install Ubuntu. It is better to do it in two steps even though the Ubuntu installer will do it all if you want. You should resize windows, then go back and boot windows to make sure it is ok with the size change. Then go back and finish the install of Ubuntu.

1. Houseclean windows
2 defrag windows
3 defrag windows again - note size used and allow at least 20% more to let windows keep working.
4. Resize windows partition to make room for Ubuntu.
5 boot windows, it may want to run chkdsk since it sees changes
6 install Ubuntu
7 Enjoy
