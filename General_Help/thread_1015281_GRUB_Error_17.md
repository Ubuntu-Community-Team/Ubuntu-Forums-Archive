### Post by pisceanreverie on 2008-12-18
My laptop computer has been able to dual boot to Windows and Ubuntu. I removed Ubuntu from the computer and now Windows will not boot. When I boot up the computer, I get a message that says GRUB Error 17. What do I need to do to allow my computer to boot into Windows?

---

### Post by Tim Sharitt on 2008-12-18
You'll need to boot from your windows cd and go to the recovery console and enter fixmbr. Your grub installation was probably removed with your ubuntu installation.
