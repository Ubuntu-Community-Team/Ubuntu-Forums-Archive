### Post by HereInOz on 2010-06-19
If you move your "Home" folder to a separate partition, as described here:

[http://www.psychocats.net/ubuntu/separatehome](http://www.psychocats.net/ubuntu/separatehome)

you will be able to then reinstall Karmic by formatting ONLY the root partition, leaving the /home partition as it is, and that will retain all of your data on the /home partition.

You MUST NOT use the "Guided" partitioning at install, you MUST use the manual partitioning option, and ensure that the /home partition is not formatted.

[SIZE=4]**_A Humble Advice_:**[/SIZE]
[COLOR=Red]**[Warning: This will save you the money & would be the safest way I can think of, but WOULD DEFINITELY COST YOU A LOT OF TIME! Bet is yours!]**[/COLOR]

[LIST=1]
[*]Download Karmik from [here]("http://releases.ubuntu.com/9.10/"), & burn it onto a CD. (choose 32bit, Desktop version, preferably torrent)
[*]Check if you can boot from that LiveCD & whether it is error-free, working fine.
[*]Disconnect both your HDDs.
[*]Get yourself a 1GB or more USB Memory Stick.
[*]Boot from the LiveCD and create a USB Startup Disk using the USB stick as the destination, and setting extra storage size to maximum (approx. 4GB, if availabl - to be able to add more programs/files if required).
[/LIST]
For next few stages, this (the USB Stick) would be your new HDD with Karmik installed!

[LIST=1]
[*]Now connect your 2nd HDD only.
[*]Boot off USB, install 7-zip utility through Synaptic.
[*]Archive all your compressible data residing on your HDD with 'Archive-Manager'. While doing so, choose '7-zip' format with 'Ultra' compression option.
[*]Don't forget to cross-check the integrity of each archive. Delete the source files after confirming the archive integrity.
[*]This should significantly reduce the size of your data, thus freeing up more available disk-space. (Unless your "personal-stuff" is compressed videos/JPGs/mp3s, etc.)
[*]Stuff like art (brushes, fonts, etc.), all your work (????), websites backups should definitely reduce to less than half their size when compressed with 7-zip. (Again- DON'T FORGET TO CROSS-CHECK ARCHIVES' INTEGRITY BEFORE DELETING ORIGINAL FILES)
[*]Now connect your Primary HDD & do the same stuff with the data on it.
[/LIST]
Now you'll have a small sized data on HDD1 & larger free space on HDD2. Simply move all the important archived data to the HDD2. Then-

[LIST]
[*]Using the LiveCD/USB, resize the first partition on HDD1 with Gparted to free up around 20GB or more free space in the beginning.
[*]Create a primary partition on this freed up space. Since this will be used for / only, 20GB should be more than sufficient for most users!
[*]Now install Karmik on this partition making it a mount point for root (/) and if possible, format the rest of the drive to create a separate partition for /home (of-course leaving a 1-2 GB space for swap).
[/LIST]
After going through all of the above, you should have all your data safe & sound along with a working installation of your beloved Karmik. But how much time you'd have wasted by then- only God can tell!
