### Post by xecure on 2008-05-04
ok i solved the problem, but just in case anyone needs the solution, all you have to do is rename the file from 'file.bin' (or sometimes just 'file' with no file extension) to 'file.docx' or what ever you think the file extension is (doc, odf, etc..)
