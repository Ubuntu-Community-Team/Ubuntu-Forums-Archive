### Post by jediborger on 2008-03-31
It sounds like you already know how to partition, so I would just try a different partitioning tool. You can use the Ubuntu desktop cd and run Gparted from the System-Administration menu. Or Gparted has a standalone live cd. I would recommend just trying Gparted's live cd since it doesn't load as much on the computer as Mandriva or Ubuntu.

[http://gpartedclonz.tuxfamily.org/index.php]("http://gpartedclonz.tuxfamily.org/index.php")
When booting just select the first option in Grub.
