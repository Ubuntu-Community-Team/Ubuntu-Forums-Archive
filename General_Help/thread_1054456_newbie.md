### Post by ajcham on 2009-01-29
I can't tell from the pic what the program is, but presumably it set to run at startup?

Go to System » Preferences » Sessions, and look down the list of startup programs - see if that jogs your memory as to what it is.  If you find it you can uncheck its entry in the list to prevent it running at startup or, having found out its name, uninstall it.
