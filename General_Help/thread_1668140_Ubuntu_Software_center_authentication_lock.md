### Post by jojothehobo on 2011-01-15
I found a solution and got more impressed with Linux and ubuntu. 

One poster on a related thread said to try to install the same package using synaptic. I tried that and synaptic threw off an error message saying in effect that the entire package management system just got corrupted (this isn't supposed to happen in Linux) but that I should run the listed script as sudo in a terminal window and all would be well. 

So I did what was suggested with no expectations, but after about two minutes the terminal prompt came back. Everything worked after that. My god, an error identified and solution that worked suggested. Amazing.
