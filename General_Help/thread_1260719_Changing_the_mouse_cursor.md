### Post by ad_267 on 2009-09-07
Do you have a theme you're trying to install or are you trying to create your own?

There are cursor themes at [http://www.gnome-look.org/index.php?xcontentmode=36](http://www.gnome-look.org/index.php?xcontentmode=36).

You should be able to drag and drop cursor themes onto the appearance window to install them, if that doesn't work you can extract the themes into ~/.icons.

Oh and you can open a .cur file with Gimp by renaming the extension to .ico.
