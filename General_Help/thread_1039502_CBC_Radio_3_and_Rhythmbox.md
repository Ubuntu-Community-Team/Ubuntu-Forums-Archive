### Post by drjonze on 2009-01-14
I had a hard time adding CBC Radio 3 as an internet radio station on Rhythmbox. The CBC had made some changes to their service and I could not get it working using the old url I had. After doing some serious digging for something that should have been really simple, I found a new url that works: 

[http://radio3.cbc.ca/nmcradio/webradioLow.m3u](http://radio3.cbc.ca/nmcradio/webradioLow.m3u)  

It is a lower bit rate stream but it sounds just fine. 

(If you don't know CBC Radio 3, its all-Canadian indie music.)
