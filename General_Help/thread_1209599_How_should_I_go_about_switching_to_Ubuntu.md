### Post by DeMus on 2009-07-10
I would not suggest using Wubi. This is an installation inside Windows. It does mean you also need Windows. From within Windows you can install Ubuntu (or any other flavor of Linux). But, when Windows crashes (and this seems to happen now and then) you also have to install Ubuntu again, meaning loosing  everything you had so far.

I would suggest trying Ubuntu from a live CD first. This means you don't install anything, the whole system runs from the CD. Your Windows installation remains as it is. Should anything don't run from the live CD because of some exotic hardware for example, you can just continue using Windows.

When things work (monitor, sound, network, mouse, keyboard) you can consider a dual boot, having Windows and Ubuntu side by side. When you boot the computer you get a menu from which you can choose which OS you want to use. It means you have to make room on your hard disk for it. How you have to do that is not so difficult. There are lots of people here who can help you with that.

When you got used to using Ubuntu it is time to say goodbye (hopefully for ever) to Windows. Remove windows and use the empty hard disk space for Ubuntu. Again, there are people here who can explain how to do that.

Have fun and welcome in the wonderful world of Linux.
