### Post by joey-elijah on 2009-07-12
1. Is there anything that you do with Windows that you cannot do with Unbuntu?

Yes there are. Nothing the average user would notice, though.

2. Do all programs run ok via Ubuntu (e.g. I-tunes, poker sites, excel, word etc)?

Why itunes? Rhythmbox, banshee, songbird are all/have native linux versions that work flawlessly. You can manage your ipod, transfer/remove songs etc with them. 

3. Is it OK for a tech idiot like me?
Most people find linux scarier because it's not layed out the same way as Windows. People forget that they also felt daunted by windows when that was new... Ubuntu is easy to use and i doubt you'll stuff it up in any major way unless you go poking around in it's nether regions!

Most hardware doesn't require you to install any drivers (they're built in) and most modern hardware is supported fine out of the box. You will need to enable 3rd party drivers to get the best from your graphics card, and you'll need to 'allow' use of 3rd party codecs for mp3 etc playback. You'll be prompted to do these so it's only a click a way.

If you ever come across an issue within 10 mins of posting it here it'll likely be solved. That's how awesome the Ubuntu Community is.

4. Can I install Ubuntu in English on my (new) Spanish language laptop? 

Yes! During installtion you can choose the install language. 

5. Will there be any headaches with the Spanish keyoboard for example?

If you set the keyboard layout to spanish during install the buttons will work as they look.

6. How do I install Ubuntu and get rid of Vista and finally get away from MS for good?

Just install Ubuntu and let it totally format your harddrive. Simple as.

Don't be scared, we were all where you are now and once you just get it over with you'll wonder what the hooha was about!
