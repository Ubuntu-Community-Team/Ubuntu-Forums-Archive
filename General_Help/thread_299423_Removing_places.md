### Post by CatKiller on 2006-11-14
You can remove the Menu Bar from the panel, and replace it with the Main Menu. This is just an icon, so the Places part of it is less obtrusive.

I find the Places menu to be quite useful - You can set a bookmark in Nautilus, and then you can quickly access that folder from the Places menu. But then I don't use the Menu Bar.
