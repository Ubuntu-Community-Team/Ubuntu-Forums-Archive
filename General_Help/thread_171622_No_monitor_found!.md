### Post by chicken101 on 2006-05-07
You should be able to fix this problem easily.  First thing I would do is try and reconfigure xserver-xorg.  Go into the console and login.

> sudo dpkg-reconfigure xserver-xorg

That will launch the xserver reconfigure wizard.  If you had some drivers that worked for you, like the frglx drivers for ati (as opposed to the default "Ati" drivers).  So instead of choosing nvidia from the list, choose whatever driver (if any) you had installed.  It will ask you for a couple more things, like screen resolution, mouse settings etc.  

If this wizard fails to run, that means that x-server isn't isntalled right.  In this case do:
(not always necessary, so you don't really have to remove everything)
> sudo apt-get remove xserver-xorg   
> sudo apt-get install xserver-xorg

Reboot, and if xserver still doesn't load right, you can try:
> sudo apt-get install ubuntu-desktop

These things should definitely fix your problem.  cheers!
