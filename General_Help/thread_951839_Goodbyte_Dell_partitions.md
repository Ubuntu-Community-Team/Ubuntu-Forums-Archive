I've long since backed these up and have only rarely used the utility partition when I thought my fan was going mad, so I'm think it's time to get rid of the things.

Anyone else do this and wish they hadn't, apart from those who felt they had to return to MS?

If you have installation media shipped with your computer, then I would suggest you get rid of the utility partition(s) that are shipped with your computer.  None of the diagnostics are of any use to the end-user.

If you don't have reinstallation media, you need to find out what kind of backup solution is provided.  Usually, you can make a system restore dvd from the backup partition.  With that done, there is no reason to keep the backup partition around.
