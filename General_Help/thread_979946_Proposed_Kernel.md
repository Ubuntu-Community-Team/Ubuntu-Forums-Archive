### Post by aaaantoine on 2008-11-12
Obligatory "Is it plugged in?" response:

Proposed means that they have come up with a patch to resolve some bugs in the system, but they would like some people to test it before they push it to everyone.  Basically if you were to get the proposed kernel, you would be beta testing said kernel for the Ubuntu devs.

If the patch is good, it will eventually be pushed to everyone.  If it breaks too many machines, it will remain on the drawing board until these problems are resolved.

Or, like was done in Hardy to my recollection, they will completely leapfrog it with a different patch entirely.  linux 2.6.24-20 was in the proposed repos for months, but I don't think it was ever released.  Instead, they released 2.6.24-21.
