### Post by henriquemaia on 2005-11-06
Try reinstalling the OO.o dictionaries through DicOOo, a macro included in OO.o2

If you're using 1.4 or 1.5, download DicOOo from [http://ftp.services.openoffice.org/pub/OpenOffice.org/contrib/dictionaries/dicooo/DicOOo.sxw](http://ftp.services.openoffice.org/pub/OpenOffice.org/contrib/dictionaries/dicooo/DicOOo.sxw)

and run the macro included in this file. This always works for me, even when I was using the OO.o betas.

Good luck.
