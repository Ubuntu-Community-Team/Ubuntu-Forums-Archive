### Post by anspectrum on 2013-01-23
I was kinda in hurry and couldn't wait long for an answer. What I did was:

1. Copied important information from Account Settings.

2. Made a backup of the emails (just in case)

3. Deleted ~/.thunderbird folder (Delete from your home folder)

4. Restart thunderbird.

5. Created the account with same settings as was earlier. Set paths etc etc.

6. Problem Solved.

I'm gonna make this thread marked as solved however, I would really like to know what could have been the possible problem. ;)
