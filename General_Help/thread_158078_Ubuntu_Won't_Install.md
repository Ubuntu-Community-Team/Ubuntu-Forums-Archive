### Post by edwina on 2006-04-10
Not really an answer, I'm afraid ...

Have you tried one of the live CDs? Maybe if you can get your machine up and running with one of those, then you'll be able to work out what's wrong with it. 

If you can't get Ubuntu to install then maybe you'll have better luck with a different distro. That is often the advice anyway - if things won't work with one, it may not be worth the battle, just try a different one. That said, it sounds like your computer is acting rather strangely for a new machine. Maybe you should see if you can sort it out some other way first - you don't want to be stuck with a new machine running so badly from the outset.

Sounds like a tricky one. Best of luck!
