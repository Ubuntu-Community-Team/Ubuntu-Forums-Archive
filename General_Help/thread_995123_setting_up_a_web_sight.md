### Post by andrew.46 on 2008-11-27
Hi krysia,

You need to learn a little html and css to start with. The page that started me off and still gives a good view of the fundamentals is:

[http://www.w3.org/MarkUp/Guide/](http://www.w3.org/MarkUp/Guide/)

You will also need an editor and I would strongly suggest bluefish:

```
sudo apt-get install bluefish
```

From there you will need to purchase a domain name and some server space but learning html and css should keep you busy for a while. (For experimentation you may find that your isp has already given you a little space anyway, this is usually free.)

All the best,

  Andrew

### Post by andrew.46 on 2008-11-28
Hi krysia,

> **krysia said:**
> you mention buying a domain address is that always the case??

There are 2 different things here:

[LIST=1]
[*]**Buying a domin name:** for a small cost you can buy for example krysia.com
[*]**Buying server space:** you can pay a company to host your site on their server.
[/LIST]

For example I own the domain name andrews-corner.org and this costs me about $20 per year. My site ([http://www.andrews-corner.org](http://www.andrews-corner.org)) is hosted by a company called Dreamhost and this costs me about $119 per year. I bought the name and the hosting in one package which is the easy way :-).

But it would be better to experiment a little first with free webhosting until you get your feet.

Andrew
