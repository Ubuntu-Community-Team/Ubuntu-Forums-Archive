### Post by Mark Phelps on 2011-10-11
Tablets, like laptops, tend to have lots of specialized hardware -- which has been the bane of non-MS Windows OSs like Linux distros and Android.

IF you do buy a Windows table, you then need to make a Live USB of the Linux Distro you want to run and boot from that -- to see how well it finds the hardware and loads the proper drivers.  Any hardware that doesn't work is going to range from simple to impossible to fix.

### Post by onla on 2011-10-11
As long as my knowledge goes, you can use dual operating systems in your tablet with the help of **Virtual Box**. It is an open-source cross platforms virtualization application that allows anyone to create a virtual machine on their computer. Just make sure that you have enough memory in your tablet before installing another operating system like Android or Ubuntu  to your Windows tablet with the help of this Virtual Box. 

You can download the Virtual Box here: [https://www.virtualbox.org](https://www.virtualbox.org)

I hope it'll help you.
