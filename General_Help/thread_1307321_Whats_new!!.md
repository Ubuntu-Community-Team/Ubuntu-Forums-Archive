### Post by Mike_IronFist on 2009-10-31
> **nileshlog said:**
> So whats new and good in ubuntu 9.10. I have my system on dual boot on windows and ubuntu, but use ubuntu less frequently since cant play games. But thats not the only criterion. I can always use windows as secondary installation. but is there anything much tempting in 9.10 to upgrade?

It's faster-booting, has a faster version of Firefox, better themes, better suspend/resume performance, and other things.

In other words, you should definitely upgrade. Upgrading is as easy as going to System -> Administration -> Update Manager and then installing the upgrade. It's not "mandatory", but if you want the latest and best stuff (and a much faster system), upgrade to 9.10
