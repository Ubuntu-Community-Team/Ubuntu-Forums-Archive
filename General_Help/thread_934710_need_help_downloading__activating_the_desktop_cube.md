### Post by Cybergod on 2008-09-30
> **napstr007 said:**
> can someone tell me where to download the desktop cube and also activate / use it ???:confused:

You will have to install compiz

```
sudo aptitude install compiz compizconfig-settings-manager
```

Then click "System"->"Preferences"->"Advanced Desktop Effects Settings".

Here you'll find the settings to enable "Desktop Cube" and "Rotate Cube".
