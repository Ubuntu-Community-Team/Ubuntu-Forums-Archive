### Post by ZhuaSD on 2008-12-16
I just managed to stop gaming long enough to let everyone know:  Civ II works great on Virtualbox on 8.04.  I got it running real nice in just a few steps.

First, download the deb file from Sun, and install it.  Then, make a new environment, and install xp on it, using an image or disc.

Then, I threw in the Civ 2 disc and installed that way.  An image would also probably work.  My virtual machine could not access my usb system, but that could be worked around.

After installing everything I also needed to install the guest additions on the Virtualbox, for graphic card support.  In the screens you will see the guest additions icon, a blue bin of some sort.

The only thing was I had to resize all the info panes when going into full screen mode, other than that it was pretty simple.

So game on!!

I now have 30 more gigs of space because I no longer need windose at all!  Civ 2 rocks on Ubuntu!!
