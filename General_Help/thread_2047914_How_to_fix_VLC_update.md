### Post by flipper T on 2012-08-25
does your currently installed version of vlc work ?

if yes, just sit tight & it will be upgraded when everything is ready.

if no, you will need to remove those ppa´s you added (google ¨remove ppa¨) and install the standard version of vlc, most easily from the ubuntu software centre.

ps, if you are not confident in the use of ppa´s, don´t use them.

that sounds harsh, but it is not meant that way. just friendly advice.

if you want to use ppas in future, look up a application called ¨y-ppa-manager¨.
