### Post by ajgreeny on 2008-11-20
Do you want to sync the partitions/folders or use the same ones for both OSs?  It is not easy, I think to do the latter and you may find it easier if you have a separate /data partition, which you will need to mount at bootup by editing /etc/fstab.  You will need to make the partition ntfs format for ease of use by both OSs.
