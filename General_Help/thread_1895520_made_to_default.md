This may be a little different depending on which desktop environment you're using, but maybe something like this:


[LIST=1]
[*]**Save** a .torrent file (do not open).
[*]Right click the .torrent file, go to "Properties."
[*]Go to the "Open with" tab
[*]Select the application you want, and press the button/check tho box that says "set as default."
[/LIST]
