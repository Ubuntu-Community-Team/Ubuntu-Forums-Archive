 As stated above, you can double-click on an Active torrent and then Files to see each file being downloaded to select the priority of each file; High, Medium, Low. For example if you want to view episode 1 just click it and make it High priority.

 It's not a good idea though because you're missing the concept of torrents which is to send many pieces of all files from the many pieces of all the files of the people seeding them. If everyone selected High priority for what they wanted to see the whole system would be pointless, it would almost be like being back to P2P (before torrents were invented). 

 Although I can see your point, I often want to see a file but it's not complete, sometimes I may make one file High priority when it's in the 90% range and then put it back to regular priority.
