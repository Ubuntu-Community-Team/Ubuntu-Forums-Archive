### Post by drs305 on 2011-07-20
Can you try booting an older kernel from the Grub menu?  If you can't see the menu during boot, hold down the SHIFT key during boot and it should appear.

If you don't see another kernel option listed on the menu, you can boot the LiveCD and run the boot info script from the following site. Post the contents of RESULTS.txt and we can take a look at the boot files.

[http://bootinfoscript.sourceforge.net]("http://bootinfoscript.sourceforge.net")
