### Post by cleentrax on 2007-06-28
A couple of responses:

First, the Live CD is great for getting the feel of Ubuntu and whether it will install properly on your PC. But it is not the best environment for installing software (Synaptic) or playing back videos.

Once you install Ubuntu to your hard drive, I think the performance/stability issues will go away. You should also consider using VLC for video playback. It has all the necessary codecs, and works better for me than Totem.
