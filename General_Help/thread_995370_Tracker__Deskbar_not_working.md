### Post by davidsrsb on 2008-11-28
Reindex the database and hover the pointer over the tracker icon to monitor the progress. You should see "ineding 4/4 folders" and then some larger number like 100/200 before after some time "Tracker : idle"

Only when it has finished can you do a search.
In indexer preferences-files I have index and watch my home directory ticked and have added /media/sda1/Documents and Settings/david/My Documents to make it index my WinXP home directory
