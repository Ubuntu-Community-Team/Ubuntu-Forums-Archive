### Post by stinkeye on 2010-01-29
An easy way is to use easystroke.
In preferences > additional buttons
click on the add button.
In the new window tick instant gestures and then click in the window
with the mouse button you want to modify.
Then go to the actions tab and choose add action and set it up as shown.
When recording the stroke the gesture doesn't matter.
You just have to be holding down the button while moving the mouse.
If it works you'll see the number of your button with a "?" in the middle.
