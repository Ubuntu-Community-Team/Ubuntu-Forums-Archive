### Post by Tamlynmac on 2011-03-19
I've experienced this issue with previous releases and decided to install [XnView ]("http://www.xnview.com/en/index.html")for a replacement. It's a little unquie to setup the first time, but works great.

If you should choose to try it and need install or setup help, just let me know in this thread. It's actually quite simple, once you realize that the download is a Tar Gz and only needs to be uncompressed directly to your home directory. The initial setup is done from the home/XnView folder/bin/xnview. Which is also the executable - when placing it in your menu. A small window will open the first time you open the executable and all one need do is select option from the tools window menu. After the initial setup, the bin executable will open the main window with thumbnails.

My wife and I both use it, as we generally don't need the complexity of Gimp. It has numerous benefits including image filtering and altering capabilities. It can convert a substantial number of formats to other formats, etc. Just wish it had more themes. :-k

Good Luck
