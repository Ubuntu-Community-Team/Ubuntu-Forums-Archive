### Post by fballem on 2008-10-25
The main differences among them is the desktop that is used by default, and the software that installed by default. To the best of my knowledge, they share common repositories, so the software that is available for one is available for all.

Ubuntu is a general purpose linux distribution, derived from Debian, that uses a GNOME desktop by default. There are two flavours - desktop and server. (see [http://www.ubuntu.com/](http://www.ubuntu.com/))

Kubuntu is a desktop linux distribution, derived from Debian, that uses a KDE desktop by default. (see [http://www.kubuntu.org/](http://www.kubuntu.org/))

Edubuntu is ubuntu with additional software installed by default that is aimed at education. This is also referred to as Ubuntu Education Edition. (see [http://www.edubuntu.org/](http://www.edubuntu.org/))

Xubuntu is ubuntu with an Xfce interface, which is a light-weight interface. This may allow older, less capable PCs to run a flavour of ubuntu and obtain the benefits from ubuntu. Its light-weight also allows new machines to run faster. (see [http://www.xubuntu.org/](http://www.xubuntu.org/))

Hope this helps,
