### Post by grumblebum2 on 2014-04-29
Should be somewhere in your applications menu.
I don't use xfce.

_Tip:_
Use terminal Tab completion when unsure of a start command.
In the terminal type grub and hit Tab a number of times and it will show all commands starting with grub.
Add a "c" so it becomes  "grub-c" and hit tab again.
Because grub-customizer is the only command starting with "grub-c" the command will be completed.
