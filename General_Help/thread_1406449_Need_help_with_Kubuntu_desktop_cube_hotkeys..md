### Post by kio_http on 2010-02-14
In KDE System settings under desktop effects there is an option to change hotkey for effects and enable/disable effect.

NOTE: Compiz can also be used with KDE. So if you really want compiz then install compiz kde. Or just stick to Kwin, its quite good as well.
