### Post by xjgnsdof on 2008-05-15
I used to run Ubuntu with 2GB of DDR 800 Corsair, and it was blazing fast. I upgraded to 4GB because it was on sale at Newegg and because I play games in Windoze on the same machine.

I not only didn't notice a difference, but because I was using 32-bit Ubuntu, the OS could only recognize 3.5GB of the 4GB that I put in. My advice: stick with what you have and focus on customizing Ubuntu so it's easier for you to find stuff and has all the apps you need.

### Post by sdennie on 2008-05-15
