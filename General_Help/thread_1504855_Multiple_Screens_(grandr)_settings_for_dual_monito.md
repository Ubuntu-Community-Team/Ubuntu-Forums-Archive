### Post by vcax on 2010-11-17
I don't know a way to do it from the gui tool (grandr), but it is only an interface to xrandr text based tool. 
So you can set your screen resolutions and positions in xrandr and than save it to /etc/kde4/kdm/Xsetup or ~/.xprofile

as it is explained in detail here:
[https://wiki.ubuntu.com/X/Config/Resolution](https://wiki.ubuntu.com/X/Config/Resolution)

(look for "Setting xrandr changes persistently")
