### Post by jonian_g on 2008-08-30
The Emerald themes come with a file .emerald
Emerald (window manager) themes can be used only when Compiz (Desktop effects) is enabled and they decorate the window border.

Gtk themes decorate everything except the window border. When compiz is not enabled the metacity (window manager) themes decorate the window border. Gtk themes and metacity themes come with an archive, usualy .tar.gz, whitch contains a folder which contains 2 more folders, gtk-2.0 and metacity-1. The first contains the gtk theme and the second the metacity theme. Though Gtk themes and metacity themes can come in seperate archives.

There is no way you can mix metacity and emerald themes because they can't be used at the same time. When you use desktop effects emerald will be used and when not, metacity.

### Post by rossjman1 on 2008-09-03
To change the panel background seach gnome-look.org for panel backgrounds. Save it somewhere (I have all mine in the Pictures directory). Right click on the panel and go to properties. In the background tab enable background image, and chose the image you want. I have included a couple images that I use, and it may go well with your theme (you may want to find one you like better, though).
As for the window title bars you need to find a metacity or emerald theme. Looking at the Gnome-look page for the gtk theme you downloaded, the description states that it comes with the Emerald theme. So download it again, and open the archive with file-roller (just double click the archive) and search for a *.emerald file. Extract that file and import it using the Emerald theme manager.
