### Post by ajgreeny on 2011-04-15
Interesting question that I had never even thought about before!

I have only just started using sylpheed in a netbook install of Lubuntu, and as far as I can make out everything is in the hidden .sylpheed-2.0 folder in my home folder.  Hit Ctrl+H to see hidden files and folders if it does not show at the moment.

The address book is in a xml file called, in my case, **addrbook-000002.xml**

I hope this helps.
