### Post by pseudonym on 2007-02-01
Attached is a working xorg.conf I had for that kind of screen setup. You will need to extract and make some edits to it to match yours (eg. the names and refresh rate values for your monitors, plus you probably want to keep the 'Input Device' sections for your keyboard and mouse the same as what you have now). To help you understand what you are doing, you can use [the nvidia.com readme]("http://us.download.nvidia.com/XFree86/Linux-x86/1.0-9746/README/appendix-p.html") . 

There are several other howtos on exactly this topic in the [Tips and Tutorials forum]("http://www.ubuntuforums.org/forumdisplay.php?f=100") if you want to read some more ;)

### Post by pseudonym on 2007-02-01
Don't forget to change the resolutions, too. That 1680x1050 is for a 22" widescreen I was using before. Native resolution for your 19" screen should be 1280x1024 for a 4:3 aspect ratio monitor, or 1440x900 for a widescreen.

Also, don't bother running nvidia-xconfig if you want to stay with this setup. It tends to overwrite things which shouldn't be. Just stick with manual editing.

to open the nvidia-settings as root.Then under the x server displays I was able to set the configuration to twinview and all the other settings to what I wanted.Then I clicked the save to X configuration button and it worked.Subsequent tinkering around borked my xorg and I went back to my handmade xorg.conf so I can't say how stable that method is.

### Post by bodhi.zazen on 2007-02-04
If you have a nvidia card, install the Beta nvidia driver with envy from here:

[http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html)

Then, as fenian suggested, type nvidia-settings in a terminal

The nvidia settings terminal will then configure your monitors, including resolution (no need to gksudo).

```
nvidia-settings &
```

Just click on "X Server Display Configuration"  ;
Then click on the "Configure" box in the "Display" section ...
