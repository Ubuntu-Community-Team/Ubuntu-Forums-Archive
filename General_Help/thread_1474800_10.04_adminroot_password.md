### Post by doas777 on 2010-05-06
just fyi, if your terminal is already pointing to ~/Desktop, and your script is on the desktop, then you want to add a './' to the beginning of the filename. 
```

ubuntu:~/Desktop$ sudo ./<filename.sh>

```
you shouldn't have to 'chmod +x' if you are using sudo, as all files are 777 to root.
