Short answer: No, they will not.

Long answer: Yes, but it will require effort and some kind of solution. Getting them to run natively is of course not possible since there is no Linux-versions of them available. Some have had success running them using Wine, me personally prefer to use them in a virtual machine i.e. using VirtualBox with a Windows XP installation inside. That is in my opinion the most pleasant solution and the one that require the least effort as far as I'm concerned. The downside is this will require memory for the VM.

I want to say that some people I've heard of have had luck with it. CS2 should work in Wine, but it might require tweaking. I've never tried it myself so there's no guarantees that it will work.
