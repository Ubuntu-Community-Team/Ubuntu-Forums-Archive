Please check the link in my signature. I built that machine for around AU$700 and it purrs.

[_ENERGY-EFFICIENT LINUX COMPUTER BUILD_]("http://ubuntuforums.org/showthread.php?t=1179877")

You can tweak the hardware components to your needs but this is a 100% Ubuntu friendly build (no Windows installed).
