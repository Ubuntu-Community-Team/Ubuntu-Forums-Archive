### Post by lovinglinux on 2011-10-20
Right-click on the toolbar, select "Customize", then drag the buttons from the customization dialog to the toolbar. If that doesn't work, click the button "Restore Defualt Set". If that doesn't work, close Firefox and delete the file *localstore.rdf* from ~/.mozilla/firefox/<profilename>/.
