### Post by stderr on 2009-01-01
You'll need to stick the CD in the drive, reboot, and it should pick it up and start the setup prog. You may need to "hit a key to boot from the CD" or something - it'll prompt you. If it doesn't, you may have your Hard Drive set as the first boot device. This is a BIOS setting, so you'll need to enter the BIOS and move the CD drive up the boot priority sequence, above your hard drive.

Alternatively, some motherboards offer a key to hit to manually select a boot device - that'd do you.
