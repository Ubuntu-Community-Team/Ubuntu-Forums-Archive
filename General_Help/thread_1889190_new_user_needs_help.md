### Post by Mike Loubet on 2011-11-30
Well, Linux and Gaming are two words that aren't normally synonymous.

If you want to play Windows games (or programmes) on Linux, there's WINE (you can install it from software centre) but go to [http://www.winehq.org/](http://www.winehq.org/) to check if the game you want to play works and for any specific instructions on its installation.

Then there are emulators like Mupen64plus if you want to play N64 games or ZSNES for SNES games.

There are of course native games as well, like [http://oilrush-game.com/](http://oilrush-game.com/) but these are much more rare.

I used to dual-boot for that very reason, but a couple of years ago stopped using windows completely since I find I can run what I want under WINE, and if not just make do.
