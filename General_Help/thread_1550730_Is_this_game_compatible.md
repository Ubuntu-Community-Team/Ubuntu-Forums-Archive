### Post by cj.surrusco on 2010-08-11
There are plenty of tutorials on how to install games with Wine, so I won't make one for you, but here are the basics.

Install wine
```
sudo apt-get install wine
```

Make the .exe executable(Right-click>Properties>Permission Tab)

Run the .exe
