### Post by oldfred on 2018-04-21
You will not be able to recovery any data that was overwritten. That is why we suggest backups.

If testdisk's deeper search does not find files, then you may be able to use photorec.
 Testdisk & photorec
[http://www.cgsecurity.org/wiki/PhotoRec](http://www.cgsecurity.org/wiki/PhotoRec)
[http://www.cgsecurity.org/wiki/PhotoRec_Step_By_Step](http://www.cgsecurity.org/wiki/PhotoRec_Step_By_Step) 

Photorec does not find full file name as it just searches drive for file types. So you have to manually rename each of your files that it finds.
I have used it and it took forever to recover files and then compare to my too old backup to see what changed. Quickly learned to have better backups.
