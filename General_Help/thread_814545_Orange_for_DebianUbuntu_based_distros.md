### Post by Sef on 2008-05-31
From the site toward the bottom:

> Sources

Source distribution includes makefiles for Unix and Macintosh, and project files for MS Visual Studio. To compile Orange, you will need an appropriate compiler (MS VC 6.0 on Windows, and gcc on Linux and Mac), Python and numpy.

    * orange-source-0.9.67.zip : sources of the last stable version
    * orange-source-snap-2008-01-04.tgz : a daily snapshot of sources from SVN (we do not even guarantee they'll compile)



I would compile from orange-source.   Read [How to Install Anything in Ubuntu]("http://monkeyblog.org/ubuntu/installing/") for how to compile from source.
