### Post by stani on 2008-05-21
This is a glossy orange theme with rounded buttons and striped bars. It uses the murrine engine, but also the clearlooks engine. It blends well with the human icons of Ubuntu and the overglossed metacity theme.

It follows these guidelines:
- orange instead of brown
- striped bars
- rounded buttons
- notebook tabs with gradients
- ...

It is derived from MurrinaOranche of Sandrea.

This is the first theme I made. I hope you like it and vote for it on gnome-look.org. Thanks Cimi for developing such a wonderful engine.

Here is the link: [http://www.gnome-look.org/content/show.php/Uborange?content=81501](http://www.gnome-look.org/content/show.php/Uborange?content=81501)

The application on the screenshot is [Phatch]("http://photobatch.stani.be"), a photo batch processor, which I develop ([http://photobatch.stani.be](http://photobatch.stani.be)). It allows you to apply rounded corners, shadows, perspective, reflections, ... to a lot of images in minutes instead of hours. As a demonstration [Phatch]("http://photobatch.stani.be") was used to pimp up the screenshots.

Some previews, of course with [Phatch]("http://photobatch.stani.be") ;-)
