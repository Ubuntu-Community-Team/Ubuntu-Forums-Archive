**Amateur Radio DX Spots via Firefox Browser: Plugin Required.**

I read amateur radio DX posts on a regular basis using W1NR's web site: *“[http://www.w1nr.net/cgi-bin/spider.cgi](http://www.w1nr.net/cgi-bin/spider.cgi)*”.  

That system uses a JAVA Spider Webclient available from "*[http://www.dxcluster.org/](http://www.dxcluster.org/)*".  The application is  supported by developers “... of DX cluster protocols and cluster software.”

In order to log-in and use the on-line system, I first had to add a plugin to my system that is able to execute JAVA applets.  After installing **“icedtea-6-plugin"** with the Synaptic Package Manager, I have been able to send and receive postings to and from the W1NR system.  Other websites using the Spider Webclient may need the plugin as well.
