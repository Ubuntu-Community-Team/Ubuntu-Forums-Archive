This is solved.  If I could figure out how to get back into the original question I would put a [SOLVED] on the front.  My memory is we used to be able to do that.
The graphic for authentication will appear IF java and java script are both shut off but will not appear if they are turned on.  That suits me fine because I prefer to have both disabled except when needed.  So, instead, I guess the question becomes:  Why must they be shut off?  It would appear it really isn't a question about Firefox but rather about the new site security mechanisms which are, unfortunately, even when functioning, somewhat awkward and a bit time-consuming to use.
