### Post by Ufo on 2005-09-22
From commandline you can use oggenc, which comes with vorbis-tools package.
Quote from info oggenc:

DESCRIPTION
       oggenc reads audio data in either raw, WAV, or AIFF format and encodes
       it into an Ogg Vorbis stream.
