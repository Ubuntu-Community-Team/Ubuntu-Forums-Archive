### Post by squenson on 2009-11-01
The file is the 'Examples' folder is owned by 'root' and therefore it opens as read-only, so you cannot change anything in it (contrary to Excel which simply disables the Save, a read-only file cannot be used in Calc). Make a copy of the file in another folder and then you should be able to use it properly. The file doesn't contain any macro, everything is done with formulas.
