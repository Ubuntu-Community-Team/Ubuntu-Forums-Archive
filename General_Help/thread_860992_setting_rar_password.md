### Post by prshah on 2008-07-16
> **psykotrol said:**
> 
whenever I go into archive manager or karchiver, and try to set the password, it does nothing.


Passwords will be applied only for files added _after_ adding the password; existing files in the archive will not be included in password protection.

### Post by prshah on 2008-07-18
> **psykotrol said:**
>  I had to create the archive, delete the files in the archive, create a new archive, save it, set a password, add the files, save it as something else...

An easier way: Create your blank archive; set a password; then drag-n-drop files into it. You don't then have to re-save it under another name.

Or, if you have an existing archive, just dump archive that into your blank password-protected archive.
