Can you use this?

[http://www.ubuntu.com/desktop/get-ubuntu/alternative-download#dvd](http://www.ubuntu.com/desktop/get-ubuntu/alternative-download#dvd)

It is on the ubuntu download page, which you could have found yourself easily, without needing to shout.

There is a disadvantage with having all on a DVD: it's contents are getting obsolete fast.

That's one point with which Ubuntu made a good start from the beginning, and a winner: while other distro's wanted you to download (or buy) a set of 5 or more CD's, Ubuntu made available repositories which were mirrored all over the globe on very fast servers.

These CD's were obsolete within the shortest of times, making installing from them a dependency hell: the dependencies were often not the newest ones.

### Post by m_zeid on 2011-03-22
OK sorry for bumping this thread, but I thought it will may be helpful for others to know what I did:

to install applications from programs and games, I've created a virtual Ubuntu using Virtualbox. Of course a fresh install will eliminate the problem of dependencies.
So I download any game or program from Synaptic not the software center, and when applying the changes I checked the "download only packages"

then going to /var/cache/apt/archives and then copy the whole contents except the "lock" file
there is no point of doing this if you are downloading more than one thing. You should do one at a time, copy it, purge the cache, and then do it again with other program :)

then you can simply organize them in folders and categories and burn them on disc ;)
