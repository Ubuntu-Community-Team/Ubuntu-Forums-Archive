### Post by itcotbtoemik on 2010-07-01
What is $TERM set to?  If it's "vt100" then you wouldn't have
an F12 defined.  On the other hand, Ubuntu is probably insisting
that your locale uses UTF-8 encoding, which can do odd things
with the line-drawing characters if they're sent to a vt100.
