### Post by azad on 2008-01-20
Hi, here's the solution:

1. Insert Ubuntu liveCD
2. Start the installer
3. During installation. choose "use entire disk"
4. When instalation is done, remove the disk and reboot

Enjoy Ubuntu. If you have any other issues, don't be afraid to ask. Cheers.
