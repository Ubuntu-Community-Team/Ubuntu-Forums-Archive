### Post by Darth_Sy on 2011-04-06
I just installed Ubuntu 10.10 on my Dell Inspiron 1420, previously running Vista.

The default brightness setting was lower than I wanted it to be, so I used the keyboard shortcuts for brightness control to increase it to maximum, and they worked just fine. However, after I leave my computer idle for a while and the screen dims, once I move the mouse again, the screen returns to the default brightness setting, not the maximum.

Is there a way to permanently set the brightness to maximum?

Thanks for your time!

### Post by techunit on 2011-04-06
There are many things in Ubuntu that aren't easy to do but you can find the screen brightness settings under the power settings. If that doesn't work or you don't find it you may want to install Ubuntu Tweak that allows you to set the brightness settings easily. It will also make using Ubuntu more convenient.
