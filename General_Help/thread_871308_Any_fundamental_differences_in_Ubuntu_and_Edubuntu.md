Edubuntu and Ubuntu both come preloaded with the Gnome Desktop Environment. The only difference between the two, are the packages that come preloaded on it. Edubuntu has educational packages installed by default. All packages that can be installed on Ubuntu, can be installed on Edubuntu, and vice-versa. The ubuntu devs release 'flavours' so that users dont have to go about installing certain packages, because they're already installed :)
