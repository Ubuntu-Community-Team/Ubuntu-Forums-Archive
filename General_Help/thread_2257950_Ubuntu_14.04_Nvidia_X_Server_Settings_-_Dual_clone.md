### Post by DuckHook on 2014-12-23
"Save to X Configuration File" should do the trick. Catalyst will ask you for your password and will then create an xorg.conf file for you in /etc/X11. Thereafter, at every boot, this file will be read and the settings you have defined in Catalyst will be loaded.

To check that you have xorg.conf, do:```
cat /etc/X11/xorg.conf
```You should get a bunch of lines. If you get nothing, then your xorg.conf was not created. Case counts in Linux so make sure you capitalize 'X' in "X11" and lower-case in "xorg.conf", or just copy and paste my code into your terminal.
