### Post by taurus on 2006-12-04
You don't have to use commandline with Ubuntu if you don't want to although I can't live without it.  You can use crontab to install the update once a week or once a day (at night) so your friends don't have to know how to use apt-get/aptitude.  And if you just want to show them what it looks like, try the LiveCD or even KnoppixCD because it comes with almost everything that you can imagine...
