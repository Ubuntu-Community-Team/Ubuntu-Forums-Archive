### Post by Doctor J on 2006-11-03
@frank_b:

I wouldn't change!  What's in the repo is just the official version, repackaged.  The only benefit i can imagine to using synaptic is the possibility of a BOINC update, in which case you wouldn't automatically be notified when it comes out, while synaptic would notify you.  If there is an update, then i would install that through synaptic...

As to specifics of how, i'm guessing here.  Probably start by selecting the Activity -> Suspend menu item.  Then quit the client & manager.  All the stuff in your Home directory should be able to stay.  The client and manager will be installed elsewhere [I'm not on Ubuntu @ the moment, so i'll have to wait to say exactly where].  Anyway, you'd likely need your admin password to delete those...
