### Post by mb_webguy on 2009-04-25
I'm assuming you don't want to deal with the command line, since if WinFF can convert to wmv but you don't like the options used, you could simply use ffmpeg in the command line with different options.

I have several GUI conversion utilities, and while most will convert *from* wmv, few of them allow conversion *to* wmv.  VLC is one of the few GUI apps I know of that will.  It won't convert in batch, but it should do the job.

### Post by mb_webguy on 2009-04-25
Well, Handbrake will handle pretty much everything I feed it, it can convert in bulk, and it has some handy presets.  So as long as I'm converting to an avi, mp4, mkv, or ogm -- which is the overwhelming majority of the time -- I use Handbrake.  

I'll occasionally use VLC to convert, but usually only if I want to strip audio or video from a file.  I also use Avidemux on occasion, but usually only if I need to edit the file as well, or if the audio and/or video stream is already in the format I want and all I want is save to a different container.
