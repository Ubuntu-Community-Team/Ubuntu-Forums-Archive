### Post by NightwishFan on 2008-03-12
Define slower.

About CPU speed, by default it is set to dynamic which allows it to work at a slower speed until you need it to be faster. So if your PC isn't working hard enough to need the extra power it will not use it. Which is a good thing. If you wish to run a newer computer at better performance I advise you try out XFCE which is lighter on resources than KDE or GNOME.
