### Post by Zalbor on 2006-10-29
The newest version of Ubuntu (6.10) comes with firefox 2 installed. So maybe you need to download it.
If you still want to install ff2 in an older version, you probably need to enter the untarred directory and give the commands:
```
./configure
make
sudo make install
```
These are the usual way to install something from source, I'm not sure if FF uses a different way (though I doubt it).
