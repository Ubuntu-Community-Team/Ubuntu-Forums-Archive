### Post by A-dogg on 2007-12-08
I've installed Celtx, a writing program, and it has icons included. Problem is, I've added the program to the start menu, but when I click to change the icon through the "launcher properties" dialog box, and navigate to the Celtx icons folder, it shows it as empty. It only let's me choose an .svn icon out of the standard icons that I first see when I click on the springboard icon in "launcher properties" to change the icon.

Not sure if this makes any sense...

Weird thing: I chose a random icon out of the default icon's that I could view after clicking on the springboard icon (the one that you're given when you add an item to the main menu before you change it). I tried to change the icon again, thinking I might now have access to the Celtx icons, but all  the dialog box showed me this time was default .png icons.

I have no idea what's going on. Can anyone help?

EDIT: I figured out how to do it. It's mentioned as a sort of bug, or just a weird setup, in various places I googled. When you open launcher to change the icon, you browse to the folder where you have the icon and hit open. Note that until you hit "open", the window will be blank and not show you any icons. Very strange.
