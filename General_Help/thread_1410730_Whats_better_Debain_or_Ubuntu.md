### Post by 3rdalbum on 2010-02-19
Try both and use what works better for you.

Debian has three versions - Stable, Testing, and Unstable. Stable is usually quite old - Debian Stable right now is like Ubuntu 8.04 but with fewer modifications. Testing is less bleeding-edge than Unstable.

Ubuntu is usually based on Unstable, but Ubuntu 10.04 is based on Testing.

Debian Stable is always more reliable and bug-free than the equivilant Ubuntu, with fewer frills. If you're a new user than I'd tend to advise you to stick with Ubuntu.
