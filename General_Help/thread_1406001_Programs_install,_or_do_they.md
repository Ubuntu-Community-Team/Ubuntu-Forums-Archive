### Post by snowpine on 2010-02-13
If I understand correctly :) you are running in "Live" mode. In which case, yes, you will lose any applications you install or documents you create each time you power down the computer. You need to install if you want the full experience. (You don't have to replace XP if you don't want to; you could set up a dual boot or install to another SD card.)
