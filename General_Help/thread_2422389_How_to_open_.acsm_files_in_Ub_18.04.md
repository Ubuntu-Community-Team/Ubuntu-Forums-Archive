### Post by Holger_Gehrke on 2019-07-07
If you're talking about files for the Adobe Digital Editions e-book reader, then no, there's no native Linux application that can use these. But the .acsm-file is actually not the book, it's just a link to it and some authorization information and possibly keys for decryption. If the e-book is not encumbered by DRM you can use Digital Editions in WINE to download it and then use any reader that can handle the format it's in for reading. winetricks has an item for installing Adobe Digital Editions, so it *should* work quite well.

Holger
