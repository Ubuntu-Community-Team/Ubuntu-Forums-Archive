### Post by marycate75 on 2007-12-25
I'm new to linux and this has been bothering me as well.
I found this for firefox and it worked.
Don't know if it will work on Opera.
Hope it helps.
> Firefox can be setup to highlight the entire URL bar within Ubuntu (and other distros). Open it up and in the url bar type:

about:config

From here you can use the filter, but what you are looking for is: browser.urlbar.clickSelectsAll

Change this to enable, restart firefox, voila
