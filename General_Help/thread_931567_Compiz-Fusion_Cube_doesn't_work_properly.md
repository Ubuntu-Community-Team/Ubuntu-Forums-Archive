### Post by Thingymebob on 2008-09-27
I'm not a KDE user but you need to add 2 more workspaces, I think in KDE you do this by:
   1.  Right-click on an empty spot on your current desktop background. From the menu choose Configure Desktop.
   2. In the Configure KDesktop window choose Multiple Desktops.
   3. Move the slider or click up the selector arrows at the top to set Number of desktops.
   4. For now, choose 4 and leave "Mouse wheel over desktop background switches desktop" unchecked. You can come back to change these options anytime.
   5. Click Apply and OK.

### Post by rmikel on 2009-02-22
**_Solution to Cube Rotate 1 Panel Problem_**

Thank you very much!:D I use Gnome desktop, however I did some research and found out how to increase my virtual desktops. If you enter more than 4 desktops you can create odd rotating polygons. An example would be a hexagon.

For Gnome user the steps to solving the 1 panel problem in cube rotate are as follows.

1. Add desktop switcher widget to panel. Right-click panel -> Add to panel... -> Type "switcher" in the search panel -> select "Workspace Switcher" -> Click Add button.

2. On the panel right-click workspace switcher -> left-click preferences.

3. Click each spin box to the desired number. Make sure both spin boxes match! A suggestion is to choose numbers equal to or above 4.
