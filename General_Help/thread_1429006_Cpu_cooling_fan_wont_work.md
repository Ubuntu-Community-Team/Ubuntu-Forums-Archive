### Post by Mark Phelps on 2010-03-14
How do you know the fan is not running?  It could be running silently and you don't hear it.

Did the laptop come with Ubuntu preinstalled? If not, what was the previous OS on it -- and did the fan run under that OS?

If it's really not running, that will burn out the CPU in your laptop in no time.  And, if you can't solve the fan problem, and want to continue to use the laptop, you will need to replace Ubuntu with a distro (or OS) in which the fan DOES work.
