### Post by nsleiman on 2007-04-13
Finally! i have Beryl working like a charm :)
This is actually not a real howto but a collection from many many howtox :) (just search the forum and you will find detailed info)


Steps i have done are:

First of all search your system for all *beryl* and delete everything related :)
(it happened that i tried some days ago to install beryl but it failed big time so some libs and links were still on my system and not remover)


Add the repos

```
sudo nano /etc/apt/sources.list
```
insert:


```
deb http://download.tuxfamily.org/3v1deb edgy beryl-svn
deb-src http://download.tuxfamily.org/3v1deb edgy beryl-svn
```

Add the PGP key
from the konsole type:

```
wget http://3v1n0.tuxfamily.org/DD800CD9.gpg -O - | sudo apt-key add - 
```

Install the Packages:

```
sudo apt-get install beryl emerald xserver-xgl
```

Edit the xorg.conf

```
sudo nano /etc/X11/xorg.conf
```

make sure you have these options:

```
Section "Extensions"
        Option      "Composite" "0"  
EndSection
```
or "Disbaled" instead of "0"


&
look for AIGLX and turn it off

```
Option AIGLX="off"
```



KDMRC


```
sudo nano /etc/kde3/kdm/kdmrc 
```

look for: 
ServerCmd=/usr/bin/X -br
change its value to 
```
ServerCmd=/usr/bin/Xgl -fullscreen :1 -ac -br -accel glx:pbuffer -accel xv:pbuffer
```

save and exit.

Put a file in the Autostart to load beryl-manager on each session automatically.

```
sudo nano .kde/Autostart/xgl
```
insert this code:

```
#!/bin/bash
if pgrep Xgl &>/dev/null; then
    /usr/bin/beryl-manager
    sleep 5
    /usr/bin/beryl-xgl --force-xgl
fi

```
save and exit.

Restart your PC and have fun :)
