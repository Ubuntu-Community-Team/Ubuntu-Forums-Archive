### Post by SonnHalter on 2008-12-30
compiz is a program that allows 3d effects. 
THING 1
go to system>prefecences>compiz settings manager (don't see compiz settings manager? go to thing 2)

scroll down the list to "workarounds" and deselect legacy fullscreen support.


THING 2

go to synaptic package manager and search compiz 

then choose "compiz settings manager"


install. 

follow thing 1
