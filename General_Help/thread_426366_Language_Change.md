### Post by Ifaistos on 2007-04-30
I had the exact same problem...

Go to System >> Keyboard >> Layout Options (<-- This is a tab)

Go to the "Group Shift/Lock behaviour" and check the option "Alt+Shift changes group".

This way you will be able to switch between Hellenic and English the way it is done in Windows.

Hope it helps :)
