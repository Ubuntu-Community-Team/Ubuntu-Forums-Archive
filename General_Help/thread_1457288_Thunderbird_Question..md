### Post by Roasted on 2010-04-18
When I go to reply to a message that came to me with some sort of a logo in the email (particularly in signatures) I get a popup asking me how I want to respond.

Send in HTML + Plain Text
Plain Text Only
HTML Only

It defaults to Plain Text Only. Is there any way I can lock it to HTML + Plain Text so I never get asked about it?

---

### Post by Detonate on 2010-04-18
In the properties of the addressee in your address book, you can choose whether the recipient prefers to receive mail as html, or plain text.  Once set, the pop-up will not come up for that address.
