When I try to install something via the software manager, I get this
 The action would require the installation of packages from not authenticated sources.

### Post by Revolutionary101 on 2010-02-02
Maybe its a problem with your software sources. Go to System>Administration>Software Sources and disable all of the sources not enable with Ubuntu by default. If it works just keep enabling sources until you find the one that is causing you the problem and delete it or fix it.
