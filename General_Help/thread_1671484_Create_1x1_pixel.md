How do you create a 1x1 pixel jpg file in ubuntu?

---

### Post by James78 on 2011-01-20
Open GIMP, go to File->New, for width and height, put in 1, press Ok, and you'll have a 1x1 pixel. You can save it as whatever filetype you want, and color it if you want also.. Just zoom in for that to see the pixel (drop down that says 100%, select 800%).
