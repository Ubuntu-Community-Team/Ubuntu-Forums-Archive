### Post by PmDematagoda on 2008-06-21
Can you post the outputs of:-
```
cat /etc/X11/xorg.conf
```
and
```
cat /var/log/Xorg.0.log
```
if possible.

If you have a Live CD, then perhaps you could execute:-
```
cat /etc/X11/xorg.conf >> ~/xorg.conf
```
and 
```
cat /var/log/Xorg.0.log >> ~/Xorg.log
```
in the terminal of the normal Ubuntu install after which you could access them through the Live CD and post them here.
