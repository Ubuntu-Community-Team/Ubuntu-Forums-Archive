Hello,

I have Windows 7 at work (without admin privileges) and I would like to remote access my home laptop (Ubuntu 11.10 ).

I used to access my home laptop (previously with Windows vista) with Logmein Free as it does not require any install on Windows 7 (just a plugin under firefox). The install of the Logmein software was required only on the host (my home laptop).

So now, How can I access my home laptop (free)? I know There is a software called Teamviewer but could not figure how to use it, does not seem very frendly.

Any guidance"?
Thanks

David

### Post by LowSky on 2011-12-21
Teamviewer is pretty simple...

Each PC gets a ID number and password. All you need to do is from one PC type in the ID of the other computer, hit connect to partner and it will then ask for the password.


I use it all the time with my own computers and for clients I can't directly access.
