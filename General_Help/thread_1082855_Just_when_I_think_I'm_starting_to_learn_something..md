### Post by lordharsha on 2009-02-28
Unless you have an IDE for python, you'll need to call it from the command line. Since Python 2.5 comes with Ubuntu, you'll need to use python3 to invoke the interactive prompt.

> $python3

I don't use an IDE, but eric is a fairly easy to use IDE. You might have to change the link to the interpreter in the IDE to use python3.
