The onboard keyboard feature seems to run at startup. How do I disable this?

Running on Gutsy.

---

### Post by Tim Sharitt on 2008-07-23
Goto System > Preferences > Preferred Applications and click the accessibility tab and make sure the box beside "Run at start' isn't checked.
If it's not, goto System > Preferences > Sessions and make sure it isn't enabled under 'Startup Programs.'
