### Post by rgb1701 on 2011-03-25
There is no such thing as a "mounter" in the sense you describe.  Your brother was mistaken, or you misinterpreted him.

You install Ubuntu 10.10 to the hard disk from its install CD or from a USB stick- no need for anything else.

You can also run Ubuntu 10.10 directly from the CD or USB stick by booting from it.

You can easily install the Ubuntu 10.10 ISO to USB stick with the Multisystem tool

[http://www.webupd8.org/2010/03/how-to-create-multiboot-liveusb-using.html](http://www.webupd8.org/2010/03/how-to-create-multiboot-liveusb-using.html)

If you need to multi-boot your PC, keeping Windows, (called dual booting), you will first need to backup your data, then use Parted Magic to resize your current Windows partition to make room for Ubuntu-

[http://partedmagic.com/doku.php?id=start](http://partedmagic.com/doku.php?id=start)

then simply boot the Ubuntu 10.10 CD or USB stick and follow the install wizard.
