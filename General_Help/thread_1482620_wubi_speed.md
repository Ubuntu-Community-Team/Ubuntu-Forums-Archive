### Post by lisati on 2010-05-13
The speed issue is unlikely to be related to your using Wubi to install Xubuntu. Although Wubi sets things up to "borrow" a portion of the Windows partition, Ubuntu doesn't actually use Windows to run. 
Looking at your system specs, it's more likely that the sluggishness is because your RAM is towards the lower end of what *ubuntu needs to run.

I haven't actually moved a wubi installation to its own partition, but have read elsewhere in the forums that it can be done. Perhaps someone else can guide you on the merits and the "how to" if that's what you want to do.

### Post by Mark Phelps on 2010-05-14
> **mje1708 said:**
> ... The laptop is a Toshiba Tecra 8100 Pentium 3 700 mhz with 384 Meg of RAM...

The RAM is not the only problem; another problem is that you're running a P3 700.

Newer versions of Ubuntu are not going to run well on a processor that slow, just as newer versions of MS Windows crawl along on older machines with slower processors.

You would probably do better with an older version (8.04?) or with one of the more lightweight distros.
