### Post by RJG098 on 2013-01-31
Figured it out! I needed to install support for webapps for Chromium.

I was confused because Firefox gets it out of the box and Chromium doesn't.

### Post by AAllInFlynn on 2013-02-01
That is correct.  You need to install unity-chromium-extension.  From the command line enter the following...

```
$ sudo apt-get install unity-chromium-extension
```
