### Post by Impavidus on 2021-10-17
Put them in some archive format. Several exist. The traditional archive format on Linux is .tar, but you could use .zip or some other instead. The traditional format on Windows is .zip, so if the recipient is using Windows, that may be a better choice. Archive manager is a GUI tool for creating and unpacking archives, but you can use the command line tools (tar, zip) if you prefer.

If the images are compressed already (usually they are), compressing the archive won't do any good. You can use an uncompressed archive.
