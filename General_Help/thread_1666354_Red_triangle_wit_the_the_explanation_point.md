### Post by akand074 on 2011-01-13
The reason for that is because old sources in your repository that no longer get updated are selected, so it's saying some of the sources have not been updated. Your image shows the theme introduced in 10.04 and later. Those errors are from 9.10 or prior, so it looks like you made an upgrade and some old sources are still selected. 

Open up the software centre and go to Edit > Software Sources and go to the "Other Software" tab, deselect/delete those two old sources they are talking about, and the error should go away.
