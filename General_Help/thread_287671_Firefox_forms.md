### Post by BitTorrentBuddha on 2006-10-29
Attached is an archive with the folder "form-widgets" and the file "forms.css". Extract these to firefox's res directory (/usr/share/firefox/res by default in Edgy.) It sounds as though you've done similar things before, but if you need further help on how to do this, just ask. You'll also need to replace these files every time you upgrade firefox.
