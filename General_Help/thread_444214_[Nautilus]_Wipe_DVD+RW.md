### Post by mchatel on 2007-05-15
> **Wim Sturkenboom said:**
> Can I wipe (reformat) a DVD+RW using Nautilus? The Nautilus help does not mention it, searches here and on internet don't mention it either. Or did I miss it?

If you just need to wipe the DVD+RW, you will have this option if you choose to burn something new to it, using Nautilus.  I do this all the time.  Nautilus prompts me if I want to erase the disc first, and I say "yes".

If you only want to wipe the disc, and not burn something new to it right away, then I can't recall if there is an "erase" right-click option on a mounted DVD+RW disc icon or not.  I'll have to check when I'm home.
