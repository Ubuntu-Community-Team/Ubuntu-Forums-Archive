### Post by venturax on 2010-07-08
i found the solution after consulting a local guru.

```
$sudo apt-get install docbook-utils
$sudo apt-get install xmlto
```

The latter is only needed to generate html files.

find the tmpl file and write
```
$docbook2html [tmplfile].tmpl
```

Voilá :popcorn:
