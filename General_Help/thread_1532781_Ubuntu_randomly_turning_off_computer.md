### Post by linux18 on 2010-07-16
A very strange problem, crashes always come with a warning in linux.

two things come to mind both of them are very rare problems:

1. CPU overheating

2. bad RAM   


for the first one, turn one the computer and time how long it takes before it crashes, if it is consistent you may need to use frequency scaling or get a better cooling system

for the second, run memtest for a few hours and see if there is bad ram.

lastly, check and post some log files ( I rarely ever experience any problems so exactly which logs to check elude me ) try the first two steps and by that time some better advice will be posted.
