### Post by buzzingrobot on 2014-07-13
Also, if you click on a window while holding down the Super/Windows key, you can drag an oversized window down to get access to the menus.  Browsers typically have an enlarge/shrink option. (Try using the alt key if the super key doesn't work. I can't double check right now.)

If the text in everything is too large, not just in the browser, perhaps you accidentally entered a key combination your machine uses to change resolution.
