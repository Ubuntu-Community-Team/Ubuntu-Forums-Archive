### Post by p_quarles on 2008-01-14
You could, but it would be a gigantic pain in the neck.

Honestly, software itself doesn't really take up that much space. I have a fully featured installation of the latest Ubuntu, and the root filesystem (everything apart from my /home folder) is using up all of 5.7 GiB. I would use the external to store data, and avoid bothering with reconfiguring your executable paths.
