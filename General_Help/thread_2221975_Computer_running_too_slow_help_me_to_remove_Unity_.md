### Post by batden on 2014-05-04
Easy way to switch to LXDE: sudo apt-get install lubuntu-desktop
Then reboot.

You don't have to remove Unity. Just log into LXDE.

_Or_ install Lubuntu over Ubuntu (don't forget to backup your important files before reinstalling):
[http://lubuntu.net/](http://lubuntu.net/)
