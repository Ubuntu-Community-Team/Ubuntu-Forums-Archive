### Post by aysiu on 2008-06-21
Well, the only way to know is to experiment with controls.

Turn off Compiz and see if the issue persists. If not, it's Compiz-related.

If so, create and log in as a new temporary test user and see if the test user account also has the same problem. If not, then it's a user-specific problem. If so, then it's a system-wide problem.
