Is there a way that I can set up an autologin, so it doesn't go for the enter password?

---

### Post by NeoLithium on 2007-06-22
Absolutely, there is a way. Just be very aware that it is not secure before you set it up.  Go to System -> Administration -> Login Window.
Go to the security tab and check the box marked "Enable Automatic Login" then choose your username from the drop-down menu.
