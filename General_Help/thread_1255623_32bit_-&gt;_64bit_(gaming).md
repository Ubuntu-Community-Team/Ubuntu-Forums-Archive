### Post by NoaHall on 2009-09-01
Flash might not work so well, apart from that, most, if not all apps will work.

Benefits are great, but mostly it's the deal with more than 3.4 GB of RAM allowed to be used. It also lets your processor deal with twice as much things at once.

### Post by falconindy on 2009-09-01
Using 64-bit is usually transparent compared to 32-bit. You'll occasionally come across programs which will require some extra libraries (the IA32 libraries come to mind) in order to run, but they're readily available in the repositories. If you get into programming, you may have some further issues, but again those can be solved.

I'll second the issue with flash but add some hope: There's finally been an alpha for a 64-bit flash player released recently, and it seems to be far more stable than the 32-bit emulated that many of us run. More info [here]("http://ubuntuforums.org/showpost.php?p=6196594&postcount=451").

Also, since you mention gaming, I'll throw it out there that I used to play WoW through WinE in 64-bit Ubuntu. There were no problems beyond the usual, and in some cases it even ran faster than when I played on WindowsXP.

**Long story short:** There are things that won't immediately run on 64-bit Ubuntu. In my case, there has always been a workaround.
