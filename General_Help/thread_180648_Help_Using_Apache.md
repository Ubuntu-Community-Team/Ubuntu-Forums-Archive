I installed Apache1 not too long ago. All I did was create a public_html folder in my home directory and put the html files in there. You should have ownership of your home directory, so no need to change permissions. To check it just go to [http://127.0.0.1/~yourhomedirectoryname](http://127.0.0.1/~yourhomedirectoryname)
