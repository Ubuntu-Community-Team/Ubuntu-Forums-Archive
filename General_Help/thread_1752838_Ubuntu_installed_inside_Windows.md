### Post by mikewhatever on 2011-05-08
Try to reset the Unity interface with the following steps:
-hit ctrl+alt+f1 (that should drop you to the console window)
-login with your username and password
-type 'unity --reset' (now wait, it can take up to a minute)
-hit ctrl+alt+f7 to return back to GUI
