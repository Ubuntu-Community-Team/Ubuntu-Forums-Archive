### Post by nick_h on 2007-10-25
You can find Evolution under Applications -> Office as zaphod777 suggested.  It is also provides the drop-down calendar when you click on the time on the menu bar.

Firefox is also installed by default and is under Applications -> Internet.  There is no problem using extensions.

Gaim was installed with Feisty - with Gutsy it is now Pidgin (also under Applications -> Internet).  I don't use it myself but Yahoo is listed as one of the protocols and it looks very easy to set up.

Thunderbird is not installed by default but you can install it by typing:
```
sudo apt-get install mozilla-thunderbird
```
