EDIT - I see what I did with Faenza.... the default Faenza I loved/hated, because I like the icons for applications but the folders with that color tan I was not a fan of at all. However, if you install F-Dark, I had it working on my desktop and it was perfect. On my laptop, I was having the issue with the application icons still being blah. On the F-Dark theme page, I found this:

You must have the above icon theme installed, as this theme only replaces the original folders.

Duuur! So F-Dark is just an "extension" with different folder icons, but it still uses Faenza's application icons. So all I need is to have Faenza and F-Dark installed and by selecting F-Darkest-Color-Black icon theme, I have a decent setup. Nice...
