### Post by grahammechanical on 2016-08-07
We can change the login language at the login screen. In the top panel at the right there should be a small square icon. Click on that and from the drop down menu we can make a selection out of the keyboard language layouts installed.

It seems that no matter what we select as the language during the install process English (US) is always installed. So, in my case I always get English (US) and English (UK). The last language used during the session becomes the default language and the one set for the login screen. In your case it seems that a user is using English (US) and then login out or shutting down.

We can add and remove different language keyboard layouts at System Settings>Keyboard and click Text Entry Settings. In the new dialog window, in the left pane select, for example, English (US) and then click the minus sign ( - ).

We can also gain access to the Text Entry Settings from the drop down menu of the language indicator icon in the top panel.
