### Post by Buffalo Soldier on 2005-03-14
To do that without hitting the INSERT key, at the status bar of your OpenOffice you will see something like this:
```
Page 1 / 1	Default	100%	**INSRT**	STD	HYP
```
Click on the **OVER** to change it to **INSRT**. Example:
```
Page 1 / 1	Default	100%	**OVER**	STD	HYP
```
Basically it changes your editing mode from OverWrite Mode to Insert Mode.
