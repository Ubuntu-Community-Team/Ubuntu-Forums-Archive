### Post by Frogs Hair on 2012-08-24
Besides the 30 GB  root disk size limit and the in ability to hibernate because of no swap partition, it should work the same .The root disk size limitation and the fact that it is inside the Windows partition is the problem , but you can backup or sync your home folder files to an external device, Ubuntu One or Dropbox.

I use Ubuntu One because of installing the latest version of Ubuntu every six months. I find syncing my folders works fine and don't use the default backup tool. Wubi is really for trial use and not intended for long term installation.
