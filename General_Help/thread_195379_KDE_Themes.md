### Post by Jucato on 2006-06-12
ok let's look one by one:

Wallpapers - we all know that one

Themes/Styles - these are widget styles, that is, how widgets look (tabs, 
scrollbars, buttons, check boxes, etc), Most of the stuff here needs to be compiled from source, AFAIK.

Window Decorations - controls how window borders and buttons (minimize, maximize, close, etc) look. Some of these will require an "engine" to be used, something like deKorator, which can be installed from the repository. (However, the version in the repository is a bit older than the current version, I think). An engine gives you some special customization features.

Icons - icon set themes, changes your icons' appearance

Clipart - just what it is, clipart.

Emoticon themes - emoticon sets for Instant Messaging apps like Kopete.

Kopete themes - well, changes how Kopete looks

(skipping to more important ones)

Color scheme - basically color themes for different things like windows, title bars, default text, selected text, etc.

Splash Screens - themes for what you see while logging into KDE (the one with "Initializing peripherals", etc)

Bootsplash Themes - themes/background images for what you see while booting Linux (the ones where text scrolls up)

Hope that clears up some of them. :D
