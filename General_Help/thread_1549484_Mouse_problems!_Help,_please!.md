### Post by OpSecShellshock on 2010-08-09
Open up Ubuntu Software Center, and in the search put in "pointing devices." That should take you to a decent alternative, which I believe is cleverly called Pointing Devices. Once installed, it can be accessed from System-->Preferences. It's got a few more options available, and may work. I have used it to successfully set up both a touch pad and a trackball (and the trackball actually has some pretty non-standard buttons on it). I've never used it on a system with multiple devices, but it seems robust enough that it would have the option you're looking for.
