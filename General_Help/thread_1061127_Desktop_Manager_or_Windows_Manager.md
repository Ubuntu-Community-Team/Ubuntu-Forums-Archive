### Post by snowpine on 2009-02-05
Hi Rytron,

Generally speaking a Windows Manager (like fluxbox, openbox, etc) is very minimal and basically just draws windows, dialog boxes, menus, etc. on the screen.

A Desktop Environment (Gnome, KDE, Xfce, etc) includes a windows manager plus some other useful applications like a terminal, file manager, text editor, etc.

People who choose a windows manager over a desktop environment are generally looking for speed and/or simplicity, and plan to add the applications of their choice, rather than wanting everything pre-bundled.

Hope that clears it up. :)

ps A great example is LXDE. This is a lightweight desktop environment that uses Openbox as its windows manager. Compare LXDE to Openbox and you will understand the difference.
