### Post by jms830 on 2006-03-28
to install programs it is somewhat different than windows. you can use synaptic package manager (applications menu> add applications) or you can use the command line by typing 
sudo apt-get install programname
I would recommend using either easyubuntu or automatix to install the basic programs/functionality you want.
here's a link to easyubuntu directions. just type the commands they list into terminal. [http://easyubuntu.freecontrib.org/get.html](http://easyubuntu.freecontrib.org/get.html)
