### Post by howefield on 2017-05-05
There is [onedrive]("https://github.com/skilion/onedrive") and also [rclone]("https://rclone.org/")

I use rclone and find it to be an excellent utility but it is a command line application so that might not suit you.
There is no Microsoft application for Onedrive that supports Linux.
