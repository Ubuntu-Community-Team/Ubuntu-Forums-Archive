### Post by ramjet_1953 on 2009-11-09
Hi, there!

By default all ports on Ubuntu are closed.

Unless you want to do anything fancy, like port forwarding, you don't really need to do anything, or require any other program to edit "iptables"

One thing that many new users are unaware of, is that Linux already comes with a built-in firewall (iptables) and that software packages like "Firestarter" are not firewalls, but merely configuration tools for "iptabels".

As I said earlier, unless you need to do something fancy, just leave iptables alone and sit back and enjoy Linus's inherent security.

If you want to block particular sites just modify your [COLOR="Blue"]/etc/hosts.deny file[/COLOR] by inserting the ip addresses of the sites that you want to block.
