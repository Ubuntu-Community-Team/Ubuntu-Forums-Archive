### Post by Dennis N on 2018-04-17
Which release of Ubuntu did you install? Fonts are set from the Gnome Tweaks tool (search for 'Tweaks' in the overview search box). The drop down menu background and text colors are determined by the theme you are using. You can select a theme for applications in the Tweaks tool in the Appearance tab.
