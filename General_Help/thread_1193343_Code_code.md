### Post by ibuclaw on 2009-06-21
[NOPARSE]```
insert code here
```[/NOPARSE]

Ends up looking like this
```
insert code here
```

For more information, look at our BBCODE cheatsheet.

[http://ubuntuforums.org/misc.php?do=bbcode](http://ubuntuforums.org/misc.php?do=bbcode)
