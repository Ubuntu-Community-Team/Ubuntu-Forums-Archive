### Post by blackjack6.21.91 on 2006-05-21
two things

first of all, you'll probably just be able to install a new version of azureus and it should overwrite all the old files.  it would basically be like a reinstall.

if your really want to *remove* it though, then try opening nautilus as root and deleting the files.  you should get a lot less errors.  you can do this by going to applications-->system tools-->run as a different user and type in nautilus
