### Post by shakin on 2005-05-09
All I can suggest is what helped me sort out a lot of printing problems: use kcontrol. Even if you are using Gnome, install kcontrol and use its printer configuration wizard to setup your printer. It's excellent and I've never had poor printing results after using it.
