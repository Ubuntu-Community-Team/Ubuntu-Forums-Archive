### Post by earthpigg on 2011-04-13
server oriented security patches come for 5 years, routine feature additions and improvements come for 3 years.

note that the guide above mentions 'production' servers - a learning tool is generally not considered a 'production' server.

if the server stuff is for personal use, learning, playing around, etc, and you aren't going to have company payroll information or social security numbers... GUI away :)
