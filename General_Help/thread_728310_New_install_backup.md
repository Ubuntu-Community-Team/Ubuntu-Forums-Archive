### Post by fifth on 2008-03-18
> **timboellis said:**
> I am about to do a fresh install of Ubuntu is there an easy way to save and restore all your settings ie. icon s themes, home folder applications etc.

If you have /home on its own partition then just make sure you dont select it for formating when you re-install and choose the same mount point for the partition, ie /home.

If not, then you'll need to make a copy of the /home folder to a dvd/cd/flash drive. Just make sure to copy all files, including hidden files.
