### Post by vanadium on 2018-02-23
That is determined by the application itself. Many applications nowadays allow to use the general file dialogs of the desktop environment, but not all apps necessarily allow that. mtpaint is an old (and probably not maintained anymore) gtk program, so you should have gtk2 dialogs there, unless, again, that program provides its own dialogs.
