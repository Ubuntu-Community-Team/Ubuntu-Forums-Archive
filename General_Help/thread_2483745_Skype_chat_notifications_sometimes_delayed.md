### Post by TheFu on 2023-02-10
Skype is from MSFT.  Maybe contact them?

OT:

I tried to use skype as my home phone a long time ago, after MSFT bought it and had started standing up skype servers inside their infrastructure.  After trying really hard to use it for 4 months, I gave up.

These days, if I were looking for a new chat/voice solution, I'd look towards Matrix.  [https://matrix.org/clients/](https://matrix.org/clients/)  While we can run our own servers, there are plenty of free, federated, servers available too. Sorta like how email is federated, without a central company trying to keep the monkeys in their cage, but with matrix, we are allowed to roam.  ;)

If you don't need security, there's meet.jit.si ... it is a free webrtc conferencing webapp.  There are fat clients for all platforms.  My extended family uses it periodically for face-to-face meetings.  My LUG uses it weekly for our online meetings.  Just needs a modern browser that supports webrtc - which almost all of them do.  I run that connecting using chromium in a private firejail sandbox to counteract some of the security failures in webrtc.  Pretty much every video conferencing tool uses webrtc these days, so they all have the same issues. At least with a firejail wrapper, I can control the allowed access, unlike with skype or zoom or nearly all the other options.
