### Post by MattPhillips on 2009-03-02
Hi,

I had something like this, discussed in one of my threads a while back--you may be able to solve it by modifying your xorg.conf file.  Specifically, make sure that your virtual screen size is consistent with the resolution of your monitors--so that if you have e.g. two monitors at 1280x1024, your virtual screen is 2560x1024.  xorg.confs where this problem doesn't arise are possible, but I don't understand things well enough to know what the difference is.

Matt

### Post by K7522 on 2009-03-03
I figured it out on my own. All I had to do was reinstall the intel drivers.

```
sudo apt-get remove --purge xserver-xorg-video-intel
sudo apt-get install xserver-xorg-video-intel
```

sudo vi /etc/X11/xorg.conf

add in Section "Device":

```
BoardName "Mobile Intel GM45 Express Chipset"
Driver "intel"
```

Save and restart.
