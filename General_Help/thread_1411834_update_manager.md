### Post by cogier on 2010-02-20
I suggest that you check that the Update Manger is installed.

Go to 

System>Administration>Synapic Package Manager

In the search box type "update-manager" (without the quotes)

It should appear at the top of the list. If it is not installed there is your problem. If it is installed then uninstall it and then reinstall.

Hope that helps.
