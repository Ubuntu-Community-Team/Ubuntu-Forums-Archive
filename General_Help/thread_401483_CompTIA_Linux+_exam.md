### Post by sysop on 2007-04-24
From [another thread]("http://ubuntuforums.org/showthread.php?t=286317&highlight=comptia") I posted to:

> I sat the Linux+ (just this morning) and passed. I found it very challenging having had only 2 years of informal experience with Linux, and having studied for 2 months prior to sitting the exam. (I used a [uCertify Practice Test ]("http://www.ucertify.com/certifications/CompTIA/Linux-plus.html")and the [Linux+ 2005 In Depth]("http://www.amazon.com/Linux%2B-2005-Depth-Jason-Eckert/dp/1592007287/ref=pd_bbs_sr_2/102-9343617-3462516?ie=UTF8&s=books&qid=1177432598&sr=8-2") book. Both good resources together with some **practical hands on**.) The test covered lots of areas and was very in depth in many of those areas. Like the LPI certs it's CLI focused. I covered CLI command parameters, various daemon and GRUB/LILO bootloader configurations, user management, file systems, protocols, and very basic Linux security principals. Knowing the exact syntax was _very important_ ...
