### Post by NoReflex on 2009-04-13
Of course there is...
In Firefox go to Bookmarks -> Organize bookmarks.
There you will see Import and Backup. Select backup and transfer the resulting file to the destination computer. 
To import the file you must select Import...

Best wishes,
