### Post by casaschi on 2006-12-21
I have a laptop with UK physical keyboard layout.
However, I condigure it using the US keyboard layout that I find more convenient to remember (same layout on any pc I use, regardless of what's shows on the keys).

The laptop has a "Alt" key on the left of the spacebar and an "Alt Gr" key on the right.
Using the US software keyboard layout I have no use for the "Alt Gr" key, would it be possible to reconfigure it somehow that "Alt Gr" would behave like a second "Alt" key for the combinations like <Alt>-XXX or <Ctrl>-<Alt>-XXX ?

Thanks.

---

### Post by darrenm on 2006-12-21
Try System > Preferences > Keyboard > Layouts > Alt/Win key behaviour.

Select Alt and Meta are on the Win keys.
