### Post by john183 on 2008-11-19
I personally find that installing windows apps in wine is rather easy. all you have to do is run the install program for the software that yo are trying to install. granted there are some programs that do not work with wine, but you are trying to run software written for a different OS, you cannot expect everything to work without a problem. To make things a little easier, check out the wine application database at [http://appdb.winehq.org/](http://appdb.winehq.org/) they list all the software that has been tried with wine and how well it works and they have guides for most programs that require extra steps to be done to install and run properly.
