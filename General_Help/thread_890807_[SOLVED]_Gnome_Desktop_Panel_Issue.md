I was in the process of customizing my Gnome desktop by configuring my panels when I accidentally deleted the bottom default panel. So I dragged to top panel to the bottom, but apparently my windows won't minimize to it anymore. How could I retrieve that feature back?

---

### Post by SpaceMaster on 2008-08-15
-> Right click your existing panel.
-> Select "New Panel"
A blank panel will appear.  To add shortcut icons, menus (i.e. Applications, System, Places) and window selectors:
-> Right click
-> Select "Add to Panel"
-> choose from the new list.
