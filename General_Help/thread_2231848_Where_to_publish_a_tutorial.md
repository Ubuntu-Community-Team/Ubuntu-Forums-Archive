### Post by tgalati4 on 2014-06-28
I would post it in 3 places.  Once in this forum as a link to a PDF that is stored somewhere (googledocs, dropbox, etc).  Second, find an appropriate page in the Ubuntu documentation wiki--you many need to create a new page or add it to an existing page.  You may have to spend some time to find a good spot for it.  Third, as a link in your Ubuntu profile, or personal webpage, or google+ page, etc.

There are quite a few djvu tools in the repositories, but no wiki pages other than the manual pages for the individual tools.  It would be nice to have a djvu overview of tools and techniques for handling djvu files.
