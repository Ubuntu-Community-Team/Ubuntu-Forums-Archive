### Post by markackerman8@gmail.com on 2017-04-28
sudo apt-get install dconf-tools

Start it now with this command:

dconf-editor &

Head to org >> gnome >> desktop >> interface, then in the left pane of the window, locate "cursor-size" which has a default value of 24. and Increase it, hit apply

Fewwww works with 4K awesomely

Ubuntu Gnome

Hope it helps:p
