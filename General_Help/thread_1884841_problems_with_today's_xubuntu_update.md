### Post by martago on 2011-11-22
After installing today the latest updates to Xubuntu 11.10, I got the following problems:
 
- Onboard Keyboard for Gnome is now smaller, but cannot be moved from its top position on the screen to another part of the screen as it was before. Plus it stays on top of other windows, so it is impossible to work with other programs because this utility blocks the view of other widows.
 
- all my icons on the desktop, that were arranged in a vertical fashion, now are more horizontal, and if I move them, they don't stay in the new position for long, they return to the new way they are now (2 semi half horizontal lines).
 
I sincerely don't understand how these situations happen. It feels a lot like amateurism in place.

### Post by martago on 2011-11-25
For those who had the same problem as I did, I found out that below the close button of the new version of onboard for Gnome (more compact than the previous one, without the title bar), there is a "move onboard" button that allows the onboard window to be moved around the desktop.
