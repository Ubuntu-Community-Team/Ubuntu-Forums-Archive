### Post by fsando on 2007-05-17
When I first installed Feisty I was very pleased that the context menu button worked on my laptop for the first time.
Somehow it has stopped working - I have no idea whether I did it myself (I could have, or an update or an installed program could be the culprit) and I'm really missing it.

I use xgl/beryl, but it doesn't work in normal gnome either.

UPDATE (After three weeks!!):
I feel so incredibly stupid: I found the option (did it mysteriously disappear and come back today?):
Keyboard preferences > Layout Options > Alt/Win key behavior > Add the standard behavior to Menu Key
Could it be more obvious!?
