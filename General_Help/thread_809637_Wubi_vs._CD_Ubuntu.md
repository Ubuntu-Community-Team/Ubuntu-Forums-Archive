### Post by Inventech on 2008-05-27
Is there a difference between Wubi and CD version of Ubuntu?

---

### Post by Game_boy on 2008-05-27
Wubi:

- Has a slight performance penalty compared to CD Ubuntu
- Is more prone to data loss from power cuts or otherwise due to the hacky nature of the filesystem
- Is vulnerable to issues in Windows that damage the filesystem
- Is much harder to upgrade to new versions of Ubuntu or any "core" applications like drivers or libraries
- Can be installed and uninstalled using a Windows utility. It is therefore dependent on having Windows, which some might say is against Ubuntu's free software philosophy.
