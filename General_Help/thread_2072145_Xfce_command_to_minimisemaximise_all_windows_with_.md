### Post by Rytron on 2012-10-17
Hi.

Do you know if it is possible to have a Xfce command to minimise/maximise all windows with "Super Key + d"?

I know you can use "Ctrl + Alt + d" at present (as default) but I'd rather use just the 2 keys mentioned earlier.

Thanks.

---

### Post by Toz on 2012-10-17
Go to Settings Manager -> Window Manager -> Keyboard, find the "Show desktop" entry, double-click it, and change the shortcut to Super+d.
