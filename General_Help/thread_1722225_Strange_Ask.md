I believe you're looking for the Linux Terminal Server Project, which allows client computers (without hard disks) to work as remote terminals on a single server machine.

There's lots of good HOWTO documents in the Ubuntu documentation start here: [https://help.ubuntu.com/community/UbuntuLTSP](https://help.ubuntu.com/community/UbuntuLTSP)
