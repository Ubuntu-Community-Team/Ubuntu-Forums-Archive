### Post by veggen on 2011-01-10
I'm pretty sure the actual location doesn't really matter, as long as you go to project configuration and add it to the classpath.
Or more generally, just make sure the jar is on the project's (or even global) classpath. For the project itself, check the classpath in it's configuration. For the global classpath, check your CLASSPATH environmental variable. Type 'env' in terminal to see all current environmental variables and type
CLASSPATH=$CLASSPATH:/path/to/jar
to add a specific jar.
