### Post by Grout58 on 2008-12-26
Hey all.  I just have a question and would like the get some feedback. One project that I really love in the OSS community is Stratagus and Wargus.  In case you are not familiar with the project it is simple.  It was to make a native Warcraft 2 client to the big 3 platforms.  Not only was the client native but it was also made modern with features like high resolutions, modern multiplayer functionality, and unit queuing.  

    These two projects have been permanently halted because the developers are now working on Bos Wars so development is now frozen.  Before being frozen it is my opinion that both these projects were fairly mature and give a very good experience.  The problem is that it is fairly complicated now especially for new Ubuntu users to get it running because you need to compile it from source and there are a lot of dependencies.  I would very much like to see a decent sized Wargus multiplater scene going on.  What do you guys think?  I can create and maintain the debs for Ubuntu if there is an interest.

NOTE: Stratagus is in the repositories but is a very old version and not compatible with the latest version of Wargus.  Also to clear up confusion between the two projects Stratagus is the actual open source engine made, and Wargus is a project that extracts all the data and maps from the Warcraft II disk.
