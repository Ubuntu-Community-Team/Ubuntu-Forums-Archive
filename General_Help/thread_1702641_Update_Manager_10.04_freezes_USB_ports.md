### Post by spillemw on 2011-03-08
Have been using 10.04 LTS for a while (to general satisfaction).  One issue that has occurred recently is that twice a day or so my system seemed to freeze and a hard reset is required.
Purely by coincidence I found out today that it seems that Update Manager, which is started automatically on my systems to check for updates, freezes my USB ports, which gave the impression the system freezes as both my keyboard and mouse are USB.
I found out as I am using a Wireless mouse and found that if I switched it off and on again, then for a second or so I could use the mouse again.  I used that to push the "Install" button of update manager, and, behold, the system "unfreezed".
