### Post by ryanhaigh on 2008-04-15
Well in that case I would check to see whether amarok has an option to disable automatic tag updating or whatever, being a kde application it should have a configuration option for just about everything. Im not sure about songbird, as you said its not a final release and I have never used it so can't state whether will or won't have the same option to disable tag modification, but it may.

As for already buying photoshop, Im sure the requirements on the box say that you need windows, which linux and even wine is not. Depending on your machines specs you could try running an instance of windows in a virutalbox/vmware/qemu virtual machine for when you need to use photoshop. Buying hardware and software for linux is a little like buying for mac, you should check whether its compatible first, of course adobe don't sell photoshop for linux and for cases such as this people have developed wine which tries to allow windows apps to run on other platforms while others have created free alternatives to those applications like the GIMP.
