### Post by TheFu on 2014-09-08
> **tierra2 said:**
> I am still figuring out how to use Ubuntu, could anybody give me any pointers?

Not a virtualization question.

The best way I know to learn any OS is to use it daily for everything. Attack every new task as a learning opportunity. The links above are great.

A primer for Windows users might help: [http://www.dedoimedo.com/computers/ultimate-linux-guide-for-windows-users.html](http://www.dedoimedo.com/computers/ultimate-linux-guide-for-windows-users.html)  - skim it to get an overview of the OS, when, why and how things are done.  Don't skip the part on users/groups and permissions - this is the core of security on any UNIX-like OS.

Spending a some time learning the Linux/UNIX philosophy is smart too - it is NOT Windows or OSX - and that is a good thing.  An appreciation of that fact will help when things get difficult.  It is a new OS and there are some issues doing certain things.  I've been using Linux since '93 - 

There may be a LUG in your city. Face to face help can bridge some gaps too. LUG = Linux Users Group.  Google for one nearby - check universities and meetup.

Youtube has lots of how-to videos for desktop users.
For servers, things are a little more complicated - start by learning the shell/CLI: [http://linuxcommand.org/tlcl.php](http://linuxcommand.org/tlcl.php)
