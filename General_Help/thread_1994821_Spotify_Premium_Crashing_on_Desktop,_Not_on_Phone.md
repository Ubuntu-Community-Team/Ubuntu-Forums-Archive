### Post by kostkon on 2012-06-03
Try deleting the spotify config folder.

Open nautilus, press CTRL+H (or select View -> Show Hidden Files), select the .config folder and delete the folder named spotify that you'll find in there. Then, when you run Spotify it should ask for your login details again.
