### Post by tsh on 2011-10-16
Am I missing something, or does unity assume that I remember the names of the apps I have installed? I guess search is fine if I am using keyboard as my primary input - what if I'm not, yet still want to use more than 10 different apps?

---

### Post by mcduck on 2011-10-16
You can add as many applications as you want into the launcher panel, and you can browse all your apps through the Dash. So no, you don't need to use the search if you don't want to.

In addition you don't actually need to remember the name of the apps, typing a word related to what you want to do is usually all you need for the search to find the right stuff. For example searching for "Photo" will reveal both Shotwell Photo Manager and Gimp, even though there is no "photo" in Gimp's name.
