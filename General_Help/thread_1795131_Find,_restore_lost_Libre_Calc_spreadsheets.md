### Post by grahammechanical on 2011-07-02
You say this:

> clicked on the launchers for my spreadsheets

Does this literally mean what it says? The launchers may have lost the path to the documents. Have you tried opening these spreadsheets from within Calc? Where do you normally store these files when you save them? In Documents or in a folder created by you. There originals should be there. Calc files have the .ods file extension.

If you open Libreoffice and go to Tools>Options>Paths you will see that Backups are stored at .libreofffice/3/user/backup but the folder may be empty as I think that the folder is only used while Libreoffice is loaded. I think that It gets flushed when the program closes down. As you experienced a crash something may still be there but then again Libreoffice should have offered to recover the documents that you were working on.

Regards.
