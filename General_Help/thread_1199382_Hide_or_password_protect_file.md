### Post by moster on 2009-06-28
You right click to file and "add archive" select zip or 7zip and put it under password.

7zip you must install trough add/remove but he is better because he can hide name of the files inside archive so somebody have to know password to even see names of files in archive.

fast solution to hide something is to add . in begining of file (or folder) name. It will become hidden and you will not see it unless you "show hidden files" in nautilus.
