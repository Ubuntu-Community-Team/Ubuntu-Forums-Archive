### Post by Thelasko on 2010-06-25
The software is downloaded from Ubuntu's [software repository]("http://en.wikipedia.org/wiki/Software_repository").  All of the software in the repository has been checked and tested.  Furthermore, Ubuntu downloads the software over a secure connection.

It's kind of like Apple's app store, except anyone can view the source code.

### Post by Dustin2128 on 2010-06-25
explain to him that the user structure of linux prevents most viruses in its self. You can't do much of anything without having root access, and you can't have root access for more than 15 minutes without actually logging into the root account which is disabled by default on ubuntu if I recall correctly. Maybe even download an AV program from the repos. Anyway, after he uses it for a few weeks or months he probably won't worry about it too much. He just needs to realize that the repos are maintained by canonical and any open source software is virus free; you can look at the source code.

### Post by jflaker on 2010-06-25
Not to say that you CAN'T get malware/virus.  But if you stick with trusted software repositories and don't download from untrusted sites, you should be 99% OK.  

in other words, don't add untrusted sources to the software source list and use common sense.
