sudo apt-get install nano
``` in a terminal or console.

### Post by mb_webguy on 2009-01-25
While I can't find a definitive list, nano is a part of the GNU Project and seems to be included in most Linux distributions.  Nano is derived from pico, which was the editor for the pine mailer, so you may fine pico on some distributions (including Ubuntu, apparently).

Snarkiness aside, I'm a nano man myself.  It's not as common as vi, which is as close to a cross-distro standard as we've got.  That hails back to the days when the admin had very little space and memory to spare and needed something lightweight, so it's common because everyone knows how to be frustrated by it!
