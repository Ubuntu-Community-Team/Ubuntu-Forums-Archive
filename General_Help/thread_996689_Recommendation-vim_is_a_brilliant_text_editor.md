### Post by sub2007 on 2008-11-29
I think vim is one of those things that once you know how to use it you can't imnagine using anything else. It's just quite difficult to start using.

Also worth checking out for beginners to vim is gvim, which has a bit of a GUI for vim (you can still use vim commands) but it definitely helps to be able to click save and then close it for when you can't quite remember that save and quit is :wq 

And don't forget to type in terminal vim-tutor (or vi-tutor) to learn the ropes with vim. It's very well designed and will show you most of the basic functions.
