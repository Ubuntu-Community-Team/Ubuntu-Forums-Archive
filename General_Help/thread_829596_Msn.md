### Post by sub2007 on 2008-06-15
I don't know what you installed - did you install emesene or aMSN? For your needs I think aMSN would be your best bet - AFAIK supports video chat, supports skinning ([http://www.amsn-project.net/skins.php](http://www.amsn-project.net/skins.php)) and has MSN Plus! support through a plugin.

You can install it with Synaptic, search "amsn". If you are unsure then do the search in Synaptic and if it has a green box next to it then you should already have it. Then you can go on to install skinning to improve the look.
