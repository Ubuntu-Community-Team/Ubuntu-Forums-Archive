### Post by ddrichardson on 2007-08-22
You need to tell it to email you if you are only subscribed to email you - in your profile there is an option at the top of the left hand menu for bugmail settings.

By default it only emails you for bugs you are assigned to. Launchpad is a great system and I'm glad it doesn't email me about all the bugs I'm subscribed to by default, only the assigned ones.
