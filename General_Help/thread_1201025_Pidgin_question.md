### Post by colau on 2009-06-30
> **Siix said:**

Last thing is my terminal. When I su and enter my password it always gives me the "su: Authentication failure" and I know my password is correct. It doesn't seem to affect me tho. I seem to have root permissions at all times. What's going on with this?

Logging as root is not permitted by default for security issue.
However you can set root password.
```

sudo passwd root

```
