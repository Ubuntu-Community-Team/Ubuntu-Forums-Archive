### Post by MattBD on 2010-10-09
Yes, VLC is in the repositories, so you can install it with sudo apt-get install vlc.

Music-wise, I like Exaile nowadays. Banshee is also pretty good, and for those who are either running KDE or don't mind using KDE apps on other desktops, Amarok is an excellent music player.

Re a CCleaner counterpart, Linux doesn't really get messed up in the same way Windows does, so it's not really needed in quite the same way. Apt can clear up after removed software packages just by running sudo apt-get autoremove && sudo apt-get clean.
