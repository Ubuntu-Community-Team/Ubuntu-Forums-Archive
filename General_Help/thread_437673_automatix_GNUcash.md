### Post by ramjet_1953 on 2007-05-09
Unfortunately, some package writers don't add the extra scripting to put in a GUI menu entry.

However, we can fix this!

Click on [COLOR="Blue"]System>Preferences>Main Menu[/COLOR]
A window will open

In the left panel, click on [COLOR="Blue"]Office[/COLOR]
On the right-hand side click on the button [COLOR="Blue"]"New Item"[/COLOR]

In the Name Box, Type in [COLOR="Red"]Gnu Cash[/COLOR]
In the Comments Box type in anything you consider relevant
In the Command Box type in [COLOR="Red"]gnucash[/COLOR]

Click on the button that says [COLOR="Blue"]"No Icon"[/COLOR]
A window should open and if you are not already there, navigate to [COLOR="Blue"]/usr/share/pixmaps[/COLOR]

Select the item [COLOR="Red"]gnucash-icon.png[/COLOR]

After that click on [COLOR="Blue"]OK[/COLOR] and you should now have a menu item in Office for Gnu Cash.

You may have to log out and log back in again for the item to appear in the menu.
