### Post by searchfgold6789 on 2010-11-06
There are some options for you.
You could boot from the CD (put the CD in and then reboot) and install the full version of Xubuntu, which will allow you to dual boot with Windows.
You could download Wubi (_**[click here for file]("http://ubuntu.mirrors.pair.com/releases//lucid/wubi.exe")**_) and install Ubuntu directly in Windows.

I do **not** recommend bios flashing for several resons:

[LIST=1]
[*]it is a difficult process not for beginners
[*]the only BIOS flashes available, even for older computers, are usually the one you already have
[*]it is usually not necessary
[/LIST]
Why not just boot from the CD and install it completely? it will keep Windows as it is and allow you to boot into either operating system quite easily.

If your computer is older than 4 or 5 years, then it is probably not possible for you to install Ubuntu to a USB Flash drive easily. Well you could install it, but not easily boot from it. And you will need a larger pen drive (4 or 8 GB) anyway.

If your situation requires that ubuntu be installed on a pen drive, I would boot from the CD, install ubuntu to your pen drive, and then install [**_PloP_**]("http://www.plop.at/"), which will let you boot from the pen drive or Windows at startup.

Unetbootin is a program that will enable you to make a Live USB stick, which is a live CD copied to a USB stick and has the same function which is to install Ubuntu. It probably won't be of much use to you unless your computer is newer and supports USB booting.
