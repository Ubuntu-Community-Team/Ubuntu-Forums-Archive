### Post by veratyr on 2007-05-09
So for laughs I popped in the latest live cd on my work computer. It's a compaq d51s. It has an integrated intel graphics card as well as a pci geforce 4 mx card. I run a dual screen setup on it with windows and was not all that surprised when the live cd would not start. X failed to start and could not find any screens. So I thought I'd share this as I see it being a large issue for a lot of people who may have multiple (different) graphics cards and screens. It's sort of a show stopper for installing ubuntu with this type of setup. I should not have to remove a piece of hardware to get it to install.
