### Post by lovinglinux on 2011-10-18
Are you sure you are running Unity 3D? Unity 2D doesn't use Compiz, so it might be reason why changing the settings make no effect.

Does your top panel has a shadow? If not, then you are using Unity 2D. In this case, you need to install the proprietary video driver for your card, through *Additional Driver*s application.
