### Post by howefield on 2014-02-07
You are not missing anything from the desktop, seems that you are following guides written for the old gnome style interface, whereas you are now running Unity (assuming you have the default Ubuntu setup).

The equivalent to Menu > System > whatever, would be to open the dash and search for the application you need, in this case, a terminal. As you start typing the word terminal you'll see terminal icon appear in the dash ready to click on.

### Post by howefield on 2014-02-07
> **Old_Brewer said:**
> How would I search for something more involved like:
Menu -> System -> Administration -> Synaptic Package Manager -> Settings -> Repositories
or Menu -> Applications -> System Tools ->

Pretty much the same way, open the Dash and start typing Synaptic... after a few letters the icon will appear in the Dash. Once launched you'll have access to the menus, such as > Settings > Repositories.

For applications that you use fairly regularly you can pin to the launcher, so that they are always there within easy access. Just counted what I have on the launcher, 24 icons which cover just about all the applications I use, it is fairly seldom that I need to use the Dash.

System tools I think was covered up above as an option in the "cog" - top right hand corner of the panel. But again, it is searchable via the Dash.
