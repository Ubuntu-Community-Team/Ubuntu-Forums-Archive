Had a similar problem with my netbook, but this was with an SD card. Every time I resumed or started up, I got a file browser window.

If I recall, the way I fixed it was to go to Edit -> Preferences in the file browser window, click the "Media" tab, and uncheck the box labeled "Browse media when inserted".

I realize an SD card isn't the same as a network location, but it might be a place to start looking for settings to change.
