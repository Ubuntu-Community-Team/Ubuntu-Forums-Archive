### Post by slooksterpsv on 2010-10-04
Encrypt files - absolutely. I use truecrypt, make a 2GB or so file, and store all my documents in it it makes the file a virtual volume, but you set a password on it to where you can only access it.

Terminal, is used for just about anything and everything. Most applications that you'll use have a GUI (graphical window), for troubleshooting purposes we may say, ok go to Applications -> Accessories -> Terminal and type in lspci and paste back what you received.

Or if you need help configuring something and there isn't a GUI app for it, we'll walk you through what to do in terminal.

I wouldn't too much about terminal, but I would keep it in the back of your  mind to learn, as it can be the most rewarding experience using it. It makes you feel Elite in the Linux world =P. Like I said though, you probably won't use it unless you need help with something that you're unable to figure out and we need you to run commands to get information.

### Post by dtfinch on 2010-10-04
I play Oblivion on Wine probably ~20-30 hours most week without trouble. I'm using Wine 1.2 (from [here](https://launchpad.net/~ubuntu-wine/+archive/ppa)) on Lucid right now, since on 1.1 it was missing sound effects, and some occasional graphical glitches appeared on 1.3, but on 1.2 everything seems fine. My video card is a Geforce GT 240.

I do use weOCPS (Oblivion crash preventer), and its logs claim to have prevented a few crashes on most days, but it did on Windows too.

Fallout should work too, since it uses the same game engine, though I haven't played it.
