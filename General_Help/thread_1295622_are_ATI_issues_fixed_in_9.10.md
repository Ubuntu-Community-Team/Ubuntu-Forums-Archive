### Post by jbrown96 on 2009-10-19
It's not an Ubuntu issue; ATI makes horrible drivers (and not just Linux ones either). Here are a couple things to mitigate the problems.
1) Turn off compiz. 
2) Use OpenGL output. I use vlc, and there is an option in the video settings to use opengl. I'm using 9.10 beta, and with opengl there is barely any tearing. It's actually getting quite good. I think that XV is the default, which is supposed to give some hardware acceleration, but it has horrible tearing. The next best mode is X11.
