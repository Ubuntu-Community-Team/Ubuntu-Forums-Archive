### Post by EXCiD3 on 2008-05-16
Its safe to enable it. All it means is that you will get beta versions of software before it is considered a stable release. Most of the time the software works completely fine but you may find some issues with it. Enabling this opens up updates for almost all the software you have installed so when you update it will check there for updates instead of the stable section, giving you slightly newer software.
