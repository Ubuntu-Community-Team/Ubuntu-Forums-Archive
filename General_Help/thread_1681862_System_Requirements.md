### Post by Rhizoid on 2011-02-04
Only 128MB of RAM is a tall order... if you can double it to 256MB you can run XUbuntu 9.10 quite happily on a PII 450 so a PIII 600 should be no problem.

Incidentally - a PIII 600Mhz would have either PC100 or PC133 ram, not 600Mhz RAM ;)
