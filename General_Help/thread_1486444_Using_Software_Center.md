Hi,

You probably need to use a faster server. 

Go to System- Administration- Software Sources. In the middle of the window it says Download From, Click the button that id's the server you are currently using, and select Other. You can either manually select a new server or you can click the Select Best Server button. It will then test to find the fastest and best server for you to use.
