### Post by Richard1979 on 2010-02-16
You can install from inside Windows if you just want to try it out.
Just put the Ubuntu CD in and use the "wubi.exe" program on the CD to install (if it doesn't auto-start).

You can uninstall it later if you don't like it by going to the Control Panel and Add/Remove software or Control Panel > Programs & Features.
