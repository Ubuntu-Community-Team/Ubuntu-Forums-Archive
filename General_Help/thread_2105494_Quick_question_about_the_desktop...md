### Post by Warren Hill on 2013-01-16
Two basic options here 

1.  Install Xubuntu instead of Ubuntu. Uses Xfce desktop.

2. Install Ubuntu then after login open terminal and run 

sudo apt-get install -y gnome-session-fallback

logout then back in again selecting "Gnome Classic" or "Gnome Classic (no effects)" as the desktop.  This is not identical to Ubuntu 10.04 but as close as you will get :D
