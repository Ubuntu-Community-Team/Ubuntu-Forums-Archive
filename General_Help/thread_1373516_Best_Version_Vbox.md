### Post by howefield on 2010-01-06
> **sandman3838 said:**
> Is one better than the other?

Don't be shy about going to the website and reading :)

The two versions are basically the same, except for the fact that the PUEL (Personal Use and Evaluation License) version on the website has some additional closed source coding giving you additional features outlined below, which the OSE (Open Source Edition) in the repository doesn't.

The performance of each should be pretty identical, so in that respect one is not "better" than the other, however if you need any of the following features..

> Remote Display Protocol (RDP) Server
This component implements a complete RDP server on top of the virtual hardware and allows users to connect to a virtual machine remotely using any RDP compatible client.

USB support
VirtualBox implements a virtual USB controller and supports passing through USB 1.1 and USB 2.0 devices to virtual machines.

USB over RDP
This is a combination of the RDP server and USB support allowing users to make USB devices available to virtual machines running remotely. 

it would be best to take the PUEL version.
