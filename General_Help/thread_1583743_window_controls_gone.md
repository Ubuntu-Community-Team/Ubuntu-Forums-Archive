### Post by JedMeister on 2010-09-28
I use an Emerald theme and have had a similar thing happen randomly a number of times. 

I've got a nifty tray app called "Compiz Fusion Icon" (its in the Universe repo). Install with: [FONT="Courier New"]sudo apt-get install fusion-icon[/FONT] then set it to auto start. Its a bit of an ugly big blue icon in the tray but it is handy for when that happens, just right click the icon and "Reload Window Manager" and all good again! It's also handy to access settings quickly. Even though its a bit ugly it almost fits with my theme (well not really but its the same blue as DropBox so I put up with it).

Hope it helps you guys too.
