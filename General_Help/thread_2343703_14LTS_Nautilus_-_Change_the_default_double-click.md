### Post by FerryGnu on 2016-11-18
I have wine installed and in Ubuntu-16LTS when I double-click in nautilus it uses wine as the default for .exe programs. If I rght-click "Launch with wine" is the first line in the popup menu, so I assume that means it's the default action.

In 14.4LTS a double-click on an .exe file launches the Archive manager. If I right click for the popup menu, The "Launch with wine" is in a sub-menu two lines down. 

How can I make that the default for nautilus on an .exe file?

I did some searching and installed utweak, but cannot find how to change that default action so that it works like 16LTS.

Is there any way I can do this?

Thanks

---

### Post by howefield on 2016-11-18
Not sure if this will do it but try navigating to one of the .exe files in Nautilus (Files) - then right click on it and select Properties > Open with > select the Wine option and set as Default.
