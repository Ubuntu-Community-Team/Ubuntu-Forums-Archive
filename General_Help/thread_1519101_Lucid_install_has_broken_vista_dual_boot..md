### Post by Irony on 2010-06-27
```
menuentry "Windows Vista (test1) (on /dev/sda1)" {
	insmod ntfs
	set root='(hd0,1)'
	chainloader +1
}
menuentry "Windows Vista (test2) (on /dev/sda2)" {
	insmod ntfs
	set root='(hd0,2)'
	chainloader +1
}
```

Add the above to */etc/grub.d/40_custom* then do;

```
sudo update-grub
```

Then reboot and scroll down to Windows Vista (test1) and click it or try Windows Vista (test2), hopefully one of them will work.

I've often found that getting rid of the floppy line does the trick.

Did I mention that Grub 2 is a horrific cludge?
