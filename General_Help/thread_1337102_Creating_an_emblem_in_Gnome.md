### Post by VCoolio on 2009-11-25
It's a bit complicated, but you could copy your self-created emblems to the emblems folder in your icon theme. They need to be the same size as the others though, and if not you need to create a different folder and edit the index file of your icon theme accordingly.
Icon theme is in either /usr/share/icons or ~/.icons.

The new emblems should show up if you restart nautilus; if not, change your icon theme and then change back.

To put one of your requests simply... do you want to change the house icon for your home folder to a personal photo? If you just attach an emblem it'll probably be too small to scare a mouse. ;)
