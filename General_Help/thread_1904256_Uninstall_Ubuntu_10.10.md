### Post by Ms. Daisy on 2012-01-04
OK, so you're going to install Windows back on the machine using an install CD/DVD? If that's the case, then you can just boot from the optical disc and start the installation. Windows will repartition and reformat using the whole drive thus eliminating Ubuntu.

If you don't have an install CD then post back.

If you're concerned about erasing any data that you had on Ubuntu, then there's a different route to go.

As for learning Linux, yup I hear you- steep learning curve.  After 4 months I'm still a noob.  If you decide to give Linux another try, maybe you can run it off of a live CD or pen drive. You'll still have Windows to do stuff you need to do, thus hopefully eliminating a source of frustration.

### Post by Basher101 on 2012-01-05
looks like you tried popping the installation DVD in while booted into ubuntu. You must change the bootorder in your BIOS so you boot from the DVD drive (or USB) first. That way you can get into the installer, delete all volumes, make a new one and let the installer deal with the rest..
