### Post by searchfgold6789 on 2010-09-24
Did you use Wubi to install ubuntu?

[LIST]
[*]If you did, try: (at some point backing up and deleting the windows files)
[LIST]
[*]make an ext4 partition for ubuntu, plus a swap one at least the size of your RAM.
[*]copy the ubuntu files to there.
[*]wait
[*]boot into a live CD and type
[LIST]
[*]sudo mount /dev/sdXX /mnt
[*]sudo mount --bind /dev  /mnt/dev
[*]sudo mount --bind /dev/pts  /mnt/dev/pts
[*]sudo mount --bind /proc /mnt/proc
[*]sudo mount --bind /sys  /mnt/sys
[*]sudo chroot /mnt
[*]sudo grub-install --root-directory=/mnt /dev/sdX (sdXX is your ubuntu partition) (sdX is your hard drive)
[/LIST]
 
[*]reboot and cross fingers and toes
[/LIST]
 
[*]If you did not, simply run gedit or a windows program to delete the windows partition. You can also use the Windows install CD to do this.
[/LIST]
I hope this helps instead of wiping your hard drive!
I don't know much about Wubi, for example if it will even detect the swap partition after you finish, but I wish you good luck.
