### Post by mushwars on 2010-02-07
the easiest way to do this is install xampp on your windows box, then click the little start filezilla server and set up a user to access the folder you want.

Now install filezilla on your linux and connect to the local ip address of your windows box, then you can start transfering files.

Good luck.
