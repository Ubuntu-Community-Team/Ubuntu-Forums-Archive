### Post by sjnovick on 2009-09-17
No need to decide at all.  If your computer has enough capability, you don't need to dual boot.  Choose one of Windows XP or Ubuntu as the ONLY OS.  Then install VirtualBox so that you can run a virtual machine with the other.  For example, you can choose Ubuntu as your only OS.  Then open the VirtualBox program and choose to run Windows XP in a new window.
