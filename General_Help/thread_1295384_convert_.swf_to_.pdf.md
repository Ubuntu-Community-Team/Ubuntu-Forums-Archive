### Post by Hamo on 2010-05-04
I don't know if you find a solution to your problem by now but I face the same problem now and the only solution I found is to install a virtual pdf printer and than open the .swf file in firefox and print it out using virtual printer.. it will save the .pdf files in pdf folder in your home directory.. 
here is the command to install the printer
```
sudo apt-get install cups-pdf
```

than just go to system -> administration -> printing 
and add new printer and choose cups pdf and than when printing choose cups pdf as a printer...
