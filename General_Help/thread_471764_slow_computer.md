### Post by DagMan on 2007-06-12
For now until you know better what caused it, go into System->Preferences->Theme and select the default human theme.  Hopefully that will resolve the immediate issue.

I haven't looked very much in Feisty and found themes I'm happy with pretty quick but often a theme won't work or lags because of one of two things.  It can have a lot of graphics in it that need to be drawn all over again everytime something moves and on top of that some people design themes poorly so that the graphic files are much bigger than they need to be.  The other reason is that some themes require you to have the 'engine' to run the theme correctly.  They are usually in Synaptic but not always named in such a way that make them easy to spot... either that or I'm just not good at searching in Synaptic.  Hopefully someone else can help with this part.

For now see if changing back to the default theme speeds things back up.
