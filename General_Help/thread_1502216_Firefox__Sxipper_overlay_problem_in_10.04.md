### Post by khoover on 2010-06-05
I've used the Sxipper plugin ([http://www.sxipper.com/]("http://www.sxipper.com/")) for Firefox since Ubuntu 8.10, and it's a great password manager. Since my upgrade to Ubuntu 10.04, however, the Sxipper overlay has become solid black, not the semi-transparent overlay that it had been on 9.10 and previous versions.

I upgraded two computers to Ubuntu 10.04 (one uses the ati display driver and one uses the nvidia driver), and both have the same problem with the overlay. The plugin still works, but the advantage of the semi-transparent overlay is you could see the form underneath before you chose to fill it in with Sxipper.

I'm using Firefox 3.6.3 and Sxipper 2.3.3.

I've removed my Firefox profile from my home directory and started with a new one and a new install of the latest version of Sxipper, but no success. I've tried using different gnome themes, but didn't get anywhere with that. Any ideas on next steps?

### Post by icon_mefisto on 2010-06-21
I was a sxipper user too. I don't have a solution to the overlay problem, but can recommend using lastpass as an alternative.

After I switched, I found I prefer it over sxipper. It needs less interaction and can fill and submit forms completely automatically. And the interface doesn't confuse my girlfriend like sxipper did. It can also import your sxipper persona data (as well as *many* other password managers and auto form fillers).

[https://addons.mozilla.org/en-US/firefox/addon/8542/](https://addons.mozilla.org/en-US/firefox/addon/8542/)
[http://lastpass.com/](http://lastpass.com/)
