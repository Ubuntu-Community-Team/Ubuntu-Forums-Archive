### Post by flanagan on 2010-07-11
One possibility is that you have adjusted the zoom level of some of these pages in an attempt to get the text to appear normally. Hitting CTRL-0 on these page should set the page back to its default zoom level.

More likely, though, is that your Firefox settings are set to override the pages' intended font and size settings. If your chosen font doesn't fit in the page divs allotted to them, you will get the results you are seeing. Better off to make sure you use the page authors' intended settings by assuring that the checkbox labeled *Allow pages to use their own fonts, instead of my selections above* is checked.
