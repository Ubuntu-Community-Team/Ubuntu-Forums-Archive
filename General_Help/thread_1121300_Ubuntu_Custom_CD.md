### Post by Tim Sharitt on 2009-04-09
You may want to try the uck utility included in the repositories. I lets you start with a standard cd image and customize it with pre-installed packages, script, settings, etc.

To install in a terminal:
```
sudo apt-get install uck
```

Or you can install from Synaptic or add/remove.

### Post by phantom3113 on 2009-04-09
You can also try out Partimage which makes allows you to burn an image of a partition of your hard drive (or all of it if it's all one partition) that you can use to restore your computer to a previous state, or to create multiple identical copies of an OS on multiple computers.
