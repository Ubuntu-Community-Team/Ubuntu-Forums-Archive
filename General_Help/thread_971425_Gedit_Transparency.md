wow,, this is from 2 years ago ... perhaps you have already found a solution for it however since I recently started to wonder the same thing and since I have not found anything(pluggins for gedit or perhaps a manual work-a-round to get the transparent effect) I decided to use VIM instead.

Vim is a very advanced text editor, you can start it from your command prompt 

$ vim or $ gvim (for a new window)

and since the program runs within gnome-terminal you get the transparent effect you want.

However vim does require a learning curve ... and in my opinion it's well worth it.

Hopefully they'll release a gedit pluggin or something soon but from what I've read in the brainstorm forums there's nothing.
