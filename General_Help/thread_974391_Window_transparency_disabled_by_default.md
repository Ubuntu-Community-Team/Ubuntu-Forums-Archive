### Post by Stoikheion on 2008-11-07
I have a fresh install of 8.10, and somehow all of my desktop effects work *execpt* for the <alt><scroll-wheel> function that makes windows transparent. I've been googling for a while about it, and it seems I'm supposed to have a "Window Transparency" tab in the General Options of the CompizConfig Settings Manager - I don't have any such tab, unfortunately. Is there some setting that enables this? Thanks in advance :)

---

### Post by snodnipper on 2008-11-09
To enable window transparency, in 'CompizConfig Settings Manager', enable the 'Opacity, Brightness and Saturation' plugin.  Alt mouse wheel should then work ok.

### Post by Stoikheion on 2008-11-09
Thanks a lot! It worked completely fine, thank you :)

I already had some experience with Compiz, so it made sense. So to elaborate on what Snodnipper meant, In order to get access to the more advanced settings of Compiz, I had to install "Advanced Desktop Effects Settings (ccsm)" using the Applications -> Add/Remove... tab. Once I installed them, I had to go to System -> Preferences -> CompizConfig Settings Manager, and 'Opacity, Brightness and Saturation' is one of the first settings. I check marked the box next to it, and alt-scroll now works.
Thanks! :D
