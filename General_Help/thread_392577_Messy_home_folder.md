### Post by Shatrat on 2007-03-24
For google earth and some others, if you install as a user it installs to your user directory by default.
I cleaned up mine a bit by moving user installed games and google earth and so forth to a seperate directory in my home called "useronlyapps" 
a lot of that other stuff it looks like you can just delete. 
I see several signing keys and other stuff that you probably dont need anymore.
