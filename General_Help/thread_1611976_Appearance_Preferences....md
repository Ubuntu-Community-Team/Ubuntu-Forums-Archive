### Post by oldrocker99 on 2010-11-02
I had VLC (the video player) try to open all my Places. The fastest way to fix it is to right-click a folder on the Desktop (create a new one if you need to) and select Open With...

You will see a menu with a number of applications; select Other Application. In that window, select Use a custom command, and type 'nautilus' (without the apostrophes) and click on the "Remember this application for folder files" and close. That fixed the problem for me.
