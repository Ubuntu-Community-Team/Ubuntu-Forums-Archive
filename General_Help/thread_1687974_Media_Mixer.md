### Post by Frogs Hair on 2011-02-14
Try System > Preferences > Sound and see if T-bird is using the Alert volume or the output volume and possibly lower or mute one or the other.

There are mixer options in the software center or you can access the system mixer from the terminal.
```
alsamixer
```
