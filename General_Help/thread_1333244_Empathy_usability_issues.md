### Post by DrIdiot on 2009-11-21
I've started trying out Empathy (in Karmic) and I've run into the following issues:

1) When someone sends you a message, it does not open a new window.  Rather, it displays a popup notification (useless if you're not actually at your computer), and the icon flashes in your buddy list (useless unless you check if frequently), and there seems to be no way of ever finding out that they sent you a message if they happen to log off.  Am I missing a way to change it in the preferences?

2) There doesn't seem to be any way to rename groups or create new groups.  The only option I see under the "Groups" menu is "Delete."  You can, however, move people from existing groups to another by dragging.

3) Copy-pasting conversations needs to be improved.  Right now, it only copies the text and doesn't copy the usernames of the people that sent the text.

It's not all bad though.  For example, I really like the new logging system and the new conversation displays.

Anyway, so I didn't post this just to complain.  I was wondering:

Is (1) a deliberate design choice?  Is it going to be changed at some point?  If it's not, I'm going to switch back to Pidgin.  If it's going to be changed (soon), I could probably stick with Empathy for now.

(2) seems a little absurd.  I'd imagine if it were truly the case, it would've been filed as a bug by now, but I didn't see such a bug.  Does anyone else have the same experience, or is my computer weird?

I've filed a bug for (3) in GNOME Bugzilla.
