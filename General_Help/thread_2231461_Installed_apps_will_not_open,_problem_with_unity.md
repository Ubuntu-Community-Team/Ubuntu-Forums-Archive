### Post by grahammechanical on 2014-06-25
What version of Ubuntu are you using? The software web site only offers a version for Ubuntu 12.04 and not Ubuntu 14.04. 

> [COLOR=#333333][FONT=Ubuntu Beta]*** This version of the Splashtop Streamer supports Ubuntu version 12.10 and 12.04; currently there is no support for other versions of Ubuntu or other Linux distributions ***[/FONT][/COLOR]

[https://apps.ubuntu.com/cat/applications/trusty/splashtop-streamer/](https://apps.ubuntu.com/cat/applications/trusty/splashtop-streamer/)


Another point to keep in mind is that the software is listed as Beta software. So, expect problems. The image that you show is what the Dash presents when we right click an application icon displayed in the Dash Applications scope. But it is missing the button labelled Launch. There must be a reason for that. What happens when you left click the application icon? Does the application load? Have you tried running this application from the terminal. Do that and you may see error messages that tell you why the application is not running.

I cannot find Splashtop Personal in the 14.04 software centre. So, if you are using 14.04 then that application is not compatible. That is my opinion.
