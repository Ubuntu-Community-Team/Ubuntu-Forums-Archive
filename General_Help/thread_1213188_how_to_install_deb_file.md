### Post by cosine352 on 2009-07-14
just double click the .deb file. But remember it is highly advised to install applications from Ubuntu repositories. 
In the terminal 
> sudo apt-get install application-name

'application-name' is the application you want to install. 
Or use the synaptic package manager to find and install applications (system --> addministration --> Synaptic package manager)
