### Post by 37fleetwood on 2012-09-12
thanks anyway, I figured it out myself from a post on the Fedora forum.
apparently when I installed xscreensaver it uninstalled the gnome screensaver which controls locking and switching the screen as well as shutting off the screen when idle. a quick re-install of gnome-screensaver seems to have fixed all of the problems.
interestingly it shows the users wallpaper with the login dialog box. I wouldn't mind finding out how to stop that.
