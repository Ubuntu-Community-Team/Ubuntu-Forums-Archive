### Post by Tomosaur on 2007-01-27
For development, you can visit the Ubuntu launchpad page, which lists bugs and feature requests, write the code to fix one, and then submit it for review: [Ubuntu Launchpad](https://launchpad.net/ubuntu/)

You can also just write some software and make it available for people, this'll help Linux in general.

Alternatively, Ubuntu also requests help with documentation and artwork. See [this page](http://www.ubuntu.com/community/participate) for more info.
