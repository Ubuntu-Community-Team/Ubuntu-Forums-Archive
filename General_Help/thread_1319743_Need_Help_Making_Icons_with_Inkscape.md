### Post by falconindy on 2009-11-08
Inkscape is a vector graphics program. It has no idea how to deal with raster images (.png) so they're imported as links.

If you want to make them into SVGs, you'll need to recreate them in Inkscape by tracing them.

P.S. Ever looked at an SVG file? It's nothing but text to define points and directions (vectors).
