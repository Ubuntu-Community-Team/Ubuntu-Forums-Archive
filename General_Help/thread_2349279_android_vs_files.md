### Post by SeijiSensei on 2017-01-13
First, install a copy of ES File Explorer from the Google Play store so you can navigate around the device.  Make sure the files are where you think they should be. Navigate to the files using ES File Explorer and click on one of them.  Select a player from the window that pops up.  Does the file play?

Do you have an SD card for additional storage?  Is there a Music folder on the card?  Are the files there?

In my experience the default Android Music app handles most any format I've thrown at it, including WAV, MP3, Ogg Vorbis and FLAC.  Are the files in one of these encodings?  Do they have the right extensions, e.g., .flac?
