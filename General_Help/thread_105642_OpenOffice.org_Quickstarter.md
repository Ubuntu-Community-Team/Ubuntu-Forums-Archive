### Post by Qrk on 2005-12-18
There are a few options. One is to install "prelink" which is in the repositories. It takes a little care to set up... but that is all available in here in the forums. Just search the how-to section. Prelinking doesn't seem to do a lot to boost performance, I've found.

An easier way to speed up OO.o2 is to disable Java. Openoffice becomes much snappier if you go to Options-->Java-->Disable java runtime environment. It cut the loading time into a third (for me) at least.
