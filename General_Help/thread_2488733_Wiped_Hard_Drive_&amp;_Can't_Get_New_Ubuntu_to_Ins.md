### Post by dragonfly41 on 2023-07-13
Quick reply. I suspect that you just wiped your HDD but overlooked formatting and preparing the partition for "Try Something Else".
You will need to use Gparted in LiveUSB (from memory it is installed in LiveUSB - otherwise install into LiveUSB).
I'm busy so cannot expand further but follow a tutorial on formatting and setting up UEFI.

### Post by oldfred on 2023-07-14
If older system, a lighter weight flavor may be required.
I tried full Ubuntu on old laptop & it just would not install. Server version installed & I add very lightweight gui.
Then installed Kubuntu which is my normal install flavor even for new systems and was surprised it worked as it is more  a mid-weight flavor.

[https://ubuntu.com/download/flavours](https://ubuntu.com/download/flavours)
Light weight flavors:
Lubuntu, xubuntu, Ubuntu MATE, Budgie
