### Post by vanadium on 2007-06-04
There are many books you can buy, and the best thing to do is go to a book store and look through different copies to try to see what would suit you most at this time. It also depends on your current focus: Do you want to learn more about the graphical tools and the application software? Do you want to get some insights into the internals of the operating system?

For learning something about the operating system, you can head over to the Linux Documentation project. ([www.tldp.org](www.tldp.org)). The book

[http://www.tldp.org/guides.html](http://www.tldp.org/guides.html), "Introduction to Linux - A Hands on Guide" is nicely written and provides a very good idea of how a Linux system is conceived. "GNU/Linux Command-Line Tools Summary" provides a nice overview of what one can do at the command line.
