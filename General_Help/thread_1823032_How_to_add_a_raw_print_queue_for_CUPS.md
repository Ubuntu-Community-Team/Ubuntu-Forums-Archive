I want to correct this: [http://ubuntuforums.org/showthread.php?t=277292]("http://ubuntuforums.org/showthread.php?t=277292") which comes up to from a google search and is wrong.

Adding a raw CUPS printer is simple. Just select select 'Generic' from the Choose Driver dialogue then 'Raw Queue' from Choose Driver. Provided Samba is set-up correctly this printer should now be available as a Windows share. The correct driver must be installed for it on the client machines connecting.
