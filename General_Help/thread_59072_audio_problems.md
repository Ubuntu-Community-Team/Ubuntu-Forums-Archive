### Post by humanity_to_others on 2005-08-23
you need to install mozilla-mplayer with w32 codec pack or can give a try to Vlc with its own plugin for mozilla.
[http://ubuntuguide.org/#mplayer](http://ubuntuguide.org/#mplayer)
[http://ubuntuguide.org/#codecs](http://ubuntuguide.org/#codecs)

vlc:
```
sudo apt-get install wxvlc 
sudo apt-get install mozilla-plugin-vlc
```
