### Post by Sankar P on 2007-07-30
Spam Assasin requires you to train not just spam but ham also. So you need to train a few non-junk messages as ham by clicking on them and clicking non-junk. if you do this for a few messages, your spam-filter should work fine.

IIRC, The same holds true for BogoFilter as well.
