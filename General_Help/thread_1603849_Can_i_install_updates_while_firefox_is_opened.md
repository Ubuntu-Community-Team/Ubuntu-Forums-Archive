### Post by coffeecat on 2010-10-23
It's not mandatory, but if there are updates that affect Firefox, you'll be prompted to restart Firefox once the updates are complete.

By the way, don't feel in a rush when the popup appears. You can always go back to System > Administration > Update Manager at your leisure. It's good to be diligent with updates but it's not a drop everything and do it NOW sort of thing. :)

### Post by ivarn on 2010-10-23
Also, the worst thing that can happen is that you will need to reinstall the firefox package which isn't a big deal really.
Because all your settings, bookmarks, history files and such are placed under /home/user/.mozilla/
