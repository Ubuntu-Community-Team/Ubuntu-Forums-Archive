### Post by merlinus on 2009-06-11
1. yes
2. yes, but you will not format your windows partition, so it will still be there.  grub, a menuing system, will be installed to let you run either of them.
3. you can delete the ubuntu partitions, and then using gparted resize windows to reclaim that space.
4. ubuntu will run quite slowly with only 256M ram, so you may wish to try xubuntu, which requires less system resources.
