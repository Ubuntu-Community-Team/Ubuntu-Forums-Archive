### Post by kc1di on 2012-04-25
It seems like support for your graphic's chipset was discontinued some time ago.

at least according to what I found on the internet.
"Submitted by Gregg Marshall on Sat, 2012-01-21 21:56
Laptop Quality Assurance Forum
I tried testing the following laptop with both Ubuntu 10.04 and 11.10 32 bit install CD's. Both failed during the boot process about the time you'd expect to see the desktop appear.

Toshiba Satellite 1135-S155
512 MB RAM
100 GB Hard Drive

Googling around it would appear that the drivers for the Intel 82852/82855 chipset installed on this computer never got updated after Ubuntu 8."

But rather than go with an edition as old as 8.04 I'd look for another distro that might work well with that old of a laptop.

Try vector or puppy there are others also.
