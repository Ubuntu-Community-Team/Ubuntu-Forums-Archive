### Post by ninjapirate89 on 2009-09-08
If vbox in windows is the same as ubuntu then just make a folder you want to put your shared files in (or use a pre-existing one) then open up virtualbox and click on the settings button for ubuntu. In the left column at the bottom you should see Shared Folders. Click on that, then on the top-right you should see an icon of a folder with a plus sign. Click on that and browse to the folder you want to share and give it a name. You should be able to access the folder when you boot into ubuntu now... theoretically :P
