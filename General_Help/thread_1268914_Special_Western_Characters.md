### Post by carlindo on 2009-09-17
I have a standard 105 key US keyboard and I need to use special the character Ç. In Ubuntu the command: "setxkbmap -option compose:lwin" assigns the combination <left win key> + <comma>, followed by C to the character Ç.

This works fine for any application running under Ubuntu but OpenOffice.

Does anybody know a fix for this?

Ç is used in Portuguese, French and maybe some other western language.

### Post by StuartN on 2009-09-17
If you choose a compose key (System-> Preferences-> Keyboard-> Layouts-> Other Options-> Compose key position), such as the right control or right alt key, then you will have access to most characters without setting up any keys yourself. Pressing Compose-,-c produces ç.
