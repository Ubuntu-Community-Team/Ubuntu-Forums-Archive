### Post by nanotube on 2006-02-15
as to sound: maybe your soundcard is not supported by ubuntu... go to System>administration>device manager and see what soundcard you have, and then post it here and people can tell you if it is supported...

as to playing media files (once you get your sounds figured out), see this link:
[https://wiki.ubuntu.com/RestrictedFormats](https://wiki.ubuntu.com/RestrictedFormats)

as to executable files: windows executables (the .exe files) do not run in linux. to find and install programs/packages use the synaptic package manager.

as to the terminal, you can find the terminal in Application>accessories>terminal
You can also switch to another virtual terminal by hitting the key combination Ctrl-Alt-F1 (to get back to gui, hit Ctrl-Alt-F7)

as to a complete noob tutorial for ubuntu... try [http://ubuntuguide.org/](http://ubuntuguide.org/) and [https://wiki.ubuntu.com/UserDocumentation](https://wiki.ubuntu.com/UserDocumentation) and [http://doc.gwos.org/index.php/BreezyStarterGuide](http://doc.gwos.org/index.php/BreezyStarterGuide)

have fun, and if you have any questions, feel free to post to the forums. :)
