### Post by amsterdamharu on 2010-02-08
You have 4 harddisks, grub2 needs to be installed on the right mbr and the right partition.

[https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD](https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD)

"Run the grub-install command as described below. This will reinstall the GRUB 2 files on the mounted **partition** to the proper location and to the **MBR** of the designated device."

Let's say grub2 is in the mbr since you can run search at the command line and search is not a command of legacy grub but is it installed in the right partition?

Your linux partition is sdc1 and your system might boot of sda (using mbr of sda). You don't have a seperate boot partition that I can see in fdisk -l. The mbr used depends of what's in your bios and if there is no bootable mbr in any of the other harddisk. I assume mbr of sda has grub2

```
sudo mount /dev/sdc1 /mnt
sudo grub-install --root-directory=/mnt/ /dev/sda
```
