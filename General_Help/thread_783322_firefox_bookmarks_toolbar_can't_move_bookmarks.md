### Post by gs_berg on 2008-05-05
You can make changes to firefox bookmarks toolbar from within firefox bookmarks sidebar, as follows:

Open the sidebar (press <Ctrl> + B or go to View -> Sidebar -> Bookmarks).

Inside the sidebar, locate the folder Bookmarks Toolbar Folder (near the top), which contains all your folders and links in the same order as in the bookmarks toolbar. Open this folder.

Inside the Bookmarks Toolbar Folder, locate the folder where you want to make changes, open it, and now you can move bookmarks as you want them to appear in your toolbar folders.  After you make changes, all your changes will be there in the firefox toolbar.  Now you can close the sidebar.
