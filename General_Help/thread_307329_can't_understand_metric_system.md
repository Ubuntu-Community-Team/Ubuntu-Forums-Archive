### Post by Cronjob on 2006-11-26
If it's in metric, your configured system locale is probably incorrect or your time and measurements are set to display in metric by default. Just go to Control Center => Accessibility => Country - Region & Language => Other => Measure system => Imperial 

I find it's just easier get used to celsius or do the math (and there's really no reason to stick to the imperial system, since everything except gas is labeled in both metric and imperial). If you don't want to get used to knowing that 28'C means pretty warm weather, then just multiply whatever kweather says by 1.8 and add 32 (so 30'C would be  86'F).
