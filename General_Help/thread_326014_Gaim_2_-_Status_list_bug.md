### Post by WiseElben on 2006-12-27
Yes, it is annoying. This bug happens because the program (or Gnome, I don't know who's fault it is) reads how far away from the screen border it is. One way to work around this bug is to make your buddy list shorter, leaving enough room for the whole list to be displayed without being "blocked" by the edge of the screen.
