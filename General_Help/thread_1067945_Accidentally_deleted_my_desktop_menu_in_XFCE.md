### Post by plucky on 2009-02-12
Did <Alt F2> open a window?

What happens when you right click on the desktop?

You should be able to create a launcher on the desktop,so create a terminal by using the command ```
xfce4-terminal
```.Then open the terminal and type in ```
xfce4-panel &
```.


See this [link](http://www.anujpathania.blogspot.com/2008/06/xubuntu-panels-disappear.html)
