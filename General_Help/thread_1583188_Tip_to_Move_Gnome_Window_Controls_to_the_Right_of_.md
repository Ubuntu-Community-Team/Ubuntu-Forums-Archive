### Post by shantiq on 2010-09-27
[from this amazing site]("http://www.ubuntuka.com/move-window-controls-right/?utm_source=feedburner&utm_medium=feed&utm_campaign=Feed:+Ubuntuka+(Ubuntuka)&utm_content=Google+Feedfetcher") 

```
gconftool-2 --type string --set /apps/metacity/general/button_layout "menu:minimize,maximize,close"

```


or if you want to be guided by the delicious NixiePixel she has a [vid here]("http://www.youtube.com/watch?v=f2wFjPy-wAA&feature=related") which contains all info but is more convoluted



anyway either way is good     i personally favour minimize,maximize,close on the right-hand side but all permutations are available
