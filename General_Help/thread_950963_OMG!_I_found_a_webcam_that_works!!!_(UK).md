### Post by sleepingdragon on 2008-10-17
I couldn't believe my luck with this little find. OK, it certainly isn't going to win awards for quality, but it is recognised by every bit of software I can find that uses a webcam - no configuration required.

It's from ASDA's Smart Price range - they have a VOIP kit for about £8.50 - it includes a headset.

I'm running on 8.04 and it fired up immediately. It runs on USB1.1, and there is a slight problem with green hue, but it does respond well to video settings in various software so that problem can be removed. The image size is 320x240, again, nothing special.

Overall, it's functional but not exceptional. I've given a slightly more detailed review over at [ubuntuhcl.org]("http://ubuntuhcl.org/browse/product+asda---supermarket-brand-smart-price-webcam?id=6506").
