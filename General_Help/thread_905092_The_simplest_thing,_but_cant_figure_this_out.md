### Post by Shazaam on 2008-08-30
I think what you are looking for is called a sub dock which will auto-expand when you hover over it. You should be able to go into cairo-dock Settings or right click it to add a sub dock. Once you get that figured out just drag/drop icons on it.
My cairo-dock is set up to have 4 sub-docks; one that I named "Virtual" has launchers for Virtualbox, VMware and Wine on it.

### Post by fabounet on 2008-09-01
you can drag and drop http adresses (or ftp or anything) in the Stack applet.
In the dock, you can "only" drop either a launcher from the Menu, or a file/folder.

if you want to put an icon in a sub-dock, you can drop into it, or you can drop it into the main dock, and then edit it's property and  select the right sub-dock in the list.
