### Post by LoneWolfJack on 2010-06-18
for the question mark:
> ctrl + shift + u + 003f

I'm not sure if this is what you consider impossible to use because the hex codes are actually shorter than decimal codes.

you could also map this to a certain key.

The Devanagari layout in the Indian layouts supports those characters that you mention as missing.
If you go to the keyboard display for the Devanagari (India) layout, you can see visually what you need to press to get /? and other characters.

For example, 
   AltGr + Shift + 1 : !

The text file for the layout is /usr/share/X11/xkb/symbols/in
