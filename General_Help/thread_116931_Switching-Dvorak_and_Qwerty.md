### Post by Gandalf on 2006-01-13
I'm not sure how to do that, because actually i don't even use KDE, but u can switch from one to another using the command setxkbmap like:
```

setxkbmap dvorak

```
or
```

setxkbmap us

```
am not sure if u can assign a shortcut to a command in KDE but if u can, u have already the commands :)
