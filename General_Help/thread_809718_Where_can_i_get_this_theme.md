### Post by spupy on 2008-05-27
From what i can see this is a **Murrina** theme. I'm not sure which one is on the screenshot, but I guess you can find it here:
[http://www.cimitan.com/murrine/themes/gallery](http://www.cimitan.com/murrine/themes/gallery)

You may need to install the engine first, though. Search in synaptic for "murrina". If it isn't there, you can download it from the website i gave you the link to. Installing it from there shouldn't be difficult.

These Murrina themes come with a configurator, so if you don't like some particular element of the theme, you can easily change it. Found also on the website above.
