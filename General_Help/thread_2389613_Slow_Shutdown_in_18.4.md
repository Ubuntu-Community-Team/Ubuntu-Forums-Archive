open4pl, if you're testing the next release then you could report or look for any issues that you come across on the ISO Testing Tracker - [http://iso.qa.ubuntu.com/qatracker/milestones/384/builds](http://iso.qa.ubuntu.com/qatracker/milestones/384/builds). If the issue is already known then it may well be fixed before 18.04's release. If the bug affects only **you** on **your** hardware then it may never be fixed.

Ubuntu developers and Canonical staff don't frequent these forums. A few pop in from time to time but they generally communicate which each other on IRC and mailing lists.

Launchpad is generally where issues need to be raised and if reported correctly they will find their way to the right develoment team. See [https://help.ubuntu.com/community/ReportingBugs](https://help.ubuntu.com/community/ReportingBugs) for further advice on the procedure. But you will always have to give a lot more information then you have so far.
