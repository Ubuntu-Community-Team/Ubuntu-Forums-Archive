### Post by Lars Noodén on 2009-10-09
In thunderbird, for receiving mail:

[INDENT][FONT="Courier New"]Edit->Account Settings->Server Settings[/FONT][/INDENT]

for sending mail:

[INDENT][FONT="Courier New"]Edit->Account Settings->Outgoing SMTP->{stmp server for that account}->Edit[/FONT][/INDENT]
