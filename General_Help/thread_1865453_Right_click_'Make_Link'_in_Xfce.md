### Post by Rytron on 2011-10-20
How can I right click and make a Link in Xfce the way it is done in GNOME2 [image attached]?

---

### Post by gsmanners on 2011-10-20
If you open File Manager (thunar), go to the file and select it, then go to the Edit menu and select Make Link (should be between Duplicate and Rename).
