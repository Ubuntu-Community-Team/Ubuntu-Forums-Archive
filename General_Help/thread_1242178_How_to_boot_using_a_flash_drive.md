### Post by earthpigg on 2009-08-16
procedure is the same for all ubuntu or ubuntu-based distributions.

get to a computer with a functional ubuntu or ubuntu-based distro install and have the .iso handy.

install usb-creator if it is not already installed.

run usb-creator.

on vanilla ubuntu, its system -> administration -> usb startup disk creator.


also, there is no such thing as "Ubuntu Ultimate". i believe you are referring to Ultimate Edition - note that 'ubuntu' is no longer anywhere in the name of that distribution of linux.
