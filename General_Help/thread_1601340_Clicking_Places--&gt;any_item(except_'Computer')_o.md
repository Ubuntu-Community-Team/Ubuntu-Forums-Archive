### Post by plucky on 2010-10-20
> **tadatma said:**
> [FONT=Georgia]I seem to have a strange problem. When I click on "Places-->any item" (except 'Computer') the folder opens in "Archive Manager". I have tried resetting the panels but the problem persists. Any suggestions?
[/FONT]

Open a terminal ( Applications > Accessories > Terminal) and type in ```
nautilus
```

In the window that opens,select any folder and right click on it and select "Open with Other Application".
On the window that opens,select File Browser and click on "Open". 

You should now be good to go.

Good Luck
