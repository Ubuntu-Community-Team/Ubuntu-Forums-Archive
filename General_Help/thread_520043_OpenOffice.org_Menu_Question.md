### Post by RedSquirrel on 2007-08-07
Hmm... this was one of the first things I changed, but I use OpenOffice so rarely these days, I'm not 100% sure how I did it.

Have a look in **View -> Toolbars**

and see if the "Bullets and Numbering" item is selected (checkbox). If it is, de-select it.
