### Post by Exdevon on 2007-08-07
> **Astrikor said:**
> Can anyone tell me how to change the default setting for "Save As" in Calc?


In Openoffice the path you need is Tools>Options>Load/Save>General
Then select Spreadsheet in "Document type" to load the calc default options menu into the "Always save as" window.  Select whichever option you want and your Openoffice calc will always list that option at the top of the "Save As" menu list. eg if you select "Excel 97/2000/xp" your calc spreadsheets will in future save as .xls by default.
