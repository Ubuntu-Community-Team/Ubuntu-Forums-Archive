### Post by rogueleader12345 on 2009-09-07
Yes, Ubuntu comes with base drivers. It should work out of the box with that system. And the difference between Kubuntu and Ubuntu is the desktop environment. Ubuntu uses Gnome, Kubuntu uses KDE. If you look at some screenshots, you will see the difference.

### Post by cholericfun on 2009-09-07
you should not need to install any drivers afterwards, 
however i've seen so much driver related nonsense in the course of one installation that i have to say you might run into prolems - depending on hardware.

### Post by howefield on 2009-09-07
