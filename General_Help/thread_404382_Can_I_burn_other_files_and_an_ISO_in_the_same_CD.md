### Post by ihavenoname on 2007-04-08
I think the best way to do this on linux is to use isomaster. You can check to make sure that the iso still has a boot record. From my experiance with it, all you have to do is make a folder on the iso and put all your files in it. Make sure to save your new iso with a new name, that way if anything got messed up you can still fix it.
