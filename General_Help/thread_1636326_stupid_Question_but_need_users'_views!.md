[LIST=1]
[*]The latest Ubuntu release contains the latest **stable** versions of packages and kernels, along with additional features. But these releases always have their share of problems and unless you are prepared to live with the problems, don't use them.
[*]Ubuntu has LTS (Long Term Support) releases that remain supported far longer than the normal releases which appear every 6 months. The LTS is for people who want stable systems that they know work and only receive Security updates over their supported life. 10.04 is the current LTS.
[/LIST]
The bottom-line is that if you want to be a crash-test dummy, use the latest Ubuntu release the minute it becomes available.

If you just want a stable tool that you can rely on to work while the forums fill up with people moaning about all the problems with the latest version, stick with the latest LTS release.
