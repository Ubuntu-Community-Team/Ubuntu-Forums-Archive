You can easily apt-get the Ubuntu version of Bluefish. It's in the Universe repository (not enabled by default).

[http://packages.ubuntu.com/hoary/web/bluefish]("http://www.webopedia.com/TERM/W/WYSIWYG.html")

As far as which is easiest to learn, I'd vote for Nvu.  It's a [WYSIWYG]("http://www.webopedia.com/TERM/W/WYSIWYG.html") editor. Bluefish is not.  Both are very good, however.

You can get a Debian version of Skype directly from Skype's web site. On the page is says it works with any Debian-based distro.

[http://www.skype.com/products/skype/linux/]("http://www.skype.com/products/skype/linux/")

And yes you should generally only install programs from an Ubuntu repository. However, that's not always possible.  More often than not though, it is.

I recommend you check out [http://ubuntuguide.org]("http://ubuntuguide.org/"). There is a lot of great info there.
