### Post by ilrudie on 2008-06-06
If you really want to learn something the best way is often not an IDE in my opinion.  The IDE covers up the inner workings in many cases.  Python I believe is interpreted (I could be wrong here) which means you don't have to compile it.  Just type python into a terminal and you get a sort of playground where you can try out little python commands.  As far as editing .py files goes gedit is solid but vi or emacs are classics.  The learning curve for those editors is nearly verticle but once you know them they have a lot to offer.
