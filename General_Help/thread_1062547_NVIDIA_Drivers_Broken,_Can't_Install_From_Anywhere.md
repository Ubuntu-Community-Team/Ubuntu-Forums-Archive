### Post by cariboo on 2009-02-08
It almost sounds like you didn't do a complete upgrade. Disable any ppa repositories in System-->Administration-->Software Sources. Open a terminal and type:

```
sudo apt-get install -f
```

then run:

```
sudo apt-get update && sudo apt-get uprade
```

Make sure that there are no errors during the update process.

Once you have things fully updated, you should be able to install the correct drivers for your version.

Jim
