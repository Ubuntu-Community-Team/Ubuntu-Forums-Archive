### Post by ronmarley1 on 2006-02-22
Type: sudo gedit /etc/apt/sources.list in a terminal window and change everything that says "breezy" to "dapper"
Save that file and (in a terminal window)
Then type: sudo apt-get update (hit Enter) 
Then type: sudo apt-get dist-upgrade (hit Enter)
Good luck, and welcome to the community!
