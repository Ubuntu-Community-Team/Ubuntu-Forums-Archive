### Post by Tim Sharitt on 2008-07-31
Launchers can be added in the awn manager which can be found in the System > Preferences menu. It is also possible to add launchers by dragging them from the desktop to the awn panel.

### Post by mondoUNC on 2008-07-31
System > Preferences > AWN Manager

Click "Launchers" then "Add"

Fill out Name and Command (Make sure not to accidentally put the command in the description box) The Icon should pop up, if not click the icon box. Make sure "Stock/Theme" is selected and scroll and choose an icon.

If you don't know the command for an app, go in to the Main Menu and find desired app; right click and "Add Launcher to desktop", right click on the desktop icon, select properties, look at the "launcher" tab and there should be the command.

You're supposed to be able to just drag and drop shortcuts to the awn dock, but it never seems to work for me. Whenever AWN restarts they disappear, but if you do it as I described you should have no problems.
