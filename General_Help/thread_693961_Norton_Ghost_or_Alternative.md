### Post by Poleh on 2008-02-11
I can highly recommend ShadowProtect - there are 3 versions:
1. Desktop edition
2. Server edition
3. IT edition

the first two are designed to be installed on the host OS and back up to removable media. The IT Edition is one that can be run simply by inserting a CD/DVD into the system, or booting off the CD/DVD.

These are all, however, pay-ware and the IT edition is not cheap :(

Acronis as mentioned previously is a good option, or if you can find it the DOS exectuable of GHOST 9/10 is still extremely good.
