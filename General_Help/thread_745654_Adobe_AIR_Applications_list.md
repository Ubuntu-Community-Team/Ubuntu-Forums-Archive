### Post by ibuclaw on 2008-04-04
AIR will be cross-platform.

It acts as a leverage to these languages:

 o **HTML** - 100% Linux Supported

 o **JavaScript** - 100% Linux Supported

 o **Flash** - Is Now 100% Linux Supported from Adobe.

 o **Ajax** - Ever heard of DOM scripts from Mozilla? That's Ajax. So 100% Linux Supported.

 o **Flex** - A Linux Port is being made by Adobe as I speak, it's currently in Alpha Stages though.

So wouldn't that be all applications (Eventually)?
