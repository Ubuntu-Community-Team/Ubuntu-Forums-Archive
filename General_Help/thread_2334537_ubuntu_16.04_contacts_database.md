### Post by Bucky Ball on 2016-08-20
Libreoffice still has a database. It's called Libreoffice Base. Install libreoffice-base.

[See here]("https://help.libreoffice.org/Common/Database_1") for further details.

### Post by Olav on 2016-08-20
If you are looking for an Outlook replacement then Evolution is probably the most capable offering e-mail, contacts, calendar etc.

See [https://en.wikipedia.org/wiki/Evolution_(software)](https://en.wikipedia.org/wiki/Evolution_(software)) to get an impression.

You could always install LibreOffice Base but then you would have to create your own contacts application with it. Not sure if you would really want to go that route.
