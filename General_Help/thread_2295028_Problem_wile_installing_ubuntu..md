### Post by grahammechanical on 2015-09-15
My BIOS boot system sees a USB memory stick with an OS on it as an external hard disk. So, I have to enter the BIOS section that selects which hard disk to boot from to put the USB stick at the top of the list.

Your BIOS may work in a similar way.

### Post by Bucky Ball on 2015-09-15
If you boot from the faulty installation disk, can you 'Try Ubuntu' and get to a desktop? If so, plug a cable in if wireless doesn't work and download a new ISO and create a bootable USB or disk that way, from a live session. How did you create the install disk in the first place and which release are you using? You should get the ISO from [here]("http://www.ubuntu.com/download/desktop"). :)

When you boot from new install media and get to the options, choose the 'Check disk for defects' option (I think that's still there) prior to installing. 

Welcome to the forums and glad you love Ubuntu. Hopefully we can help you get it installed. :)
