### Post by 3rdalbum on 2009-11-22
If I had seen this thread 14 hours ago, I would have said "Just go to System > Administration > Hardware Drivers and it will download and install an official Nvidia driver for you".

That's still the recommended way to get your Nvidia driver. If you download the installer from nvidia.com and install it, then you must reinstall the driver after every kernel update (whereas with the Ubuntu version of the driver, it stays in sync with the kernel).
