### Post by mcduck on 2012-02-14
> **qwertyjjj said:**
> When I first installed Maverick Meerkat it had the options desktop version or laptop version. I didn;t see that when installing Ocelot.

There is, and has never been, a laptop version. There used to be a *netbook* version which had a user interface designed to work better on a very low resolution displays, but since the Unity interface is now used on the desktop version, there is no more need for a separate netbook version.

Even the netbook version didn't have any performance tweaks, so if you are having such problems it's because of something else. Make sure you have graphics drivers and such installed, check log files for possible errors, and post your computer specs here so people have some idea what kind of hardware we are dealing with.... And if it's an older laptop, you might want to consider running a more lightweight desktop environment, such as XFCE or LXDE.
