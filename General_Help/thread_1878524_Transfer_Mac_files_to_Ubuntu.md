### Post by Rodney9 on 2011-11-10
You can just copy your files from the mac to a usb or dvd, good back up anyway

or

Install hfs support - HFS+ is a modernized version of Apple Computers HFS Filesystem.
Nowadays, it is widely used with more recent versions of MacOS.
hfsplus consists of a library and a set of tools that allow access
to HFS+ volumes. Then copy the files directly.
