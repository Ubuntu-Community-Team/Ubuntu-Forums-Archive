### Post by texaswriter on 2010-08-30
Hi, not sure how you are posting here or how you expect to hear about feedback, but pressing alternate function key 2, as in alt f2, will bring up a window to run a command. Type orca --setup, that's orca space bar double hyphen setup. Hit enter. This will bring up orca with preferences menu. Waiting a few seconds will result in this preferences menu having focus. IF "speak object under mouse" is what you need re-enabled, pressing tab seven times followed by space bar, followed by tab 4 times, followed by another space bar will do the trick. 

If speech is completely disabled, once you bring up orca preferences, press the right arrow key, press tab once, press space key, press SHIFT tab twice, hit enter or space bar. 

If you want key echo turned on, do you want ALL keys enabled or just echo word?

If "enable echo by word" is disabled, once you bring up orca preferences, press right arrow key three times, press tab 3 times, press space key, press tab key 5 times, and press enter. 

Please be aware that all the above key sequences assume you just started "Orca Preferences". If you can, always start each sequence after opening up orca preferences. If possible, if multiple commands are necessary, try closing the preferences menu and reopening.
