I'm pretty much new myself. That would work, however your installed programs would be lost as well as you wouldn't have a way to restore a copy of your current operating system setup. 

Here is a link that may work for you. It's not that complicated, but it could be pretty intimidating if you're unfamiliar with the file system and the command line.
