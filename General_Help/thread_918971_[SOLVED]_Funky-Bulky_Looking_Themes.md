### Post by tech13 on 2008-09-13
Assuming you have ubuntu hardy heron, go download the tarball and unarchive it, you should get 3 folders:
- Dust
- Dust Aurora
- Dust borderless

open System>Preferences>Appearance and drag either Dust or Dust Borderless into the theme window and it will say it successfully installed. go to that theme and click on it to apply. Then drag Dust Aurora into the theme window and it will ask if you want to apply the theme, click yes. Your theme should look the same as in the picture you linked to (you may need to configure your windows so that they dont show the buttons if you want it to be EXACT). I customized that theme with a different cursor and icon set (since it uses ubuntu default).

as far as the minimize/close buttons, I couldn't get those to replicate as in the picture, only could get the same ones as yours...
