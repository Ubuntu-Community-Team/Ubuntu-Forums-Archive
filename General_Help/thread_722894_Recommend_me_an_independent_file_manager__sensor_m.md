### Post by kerry_s on 2008-03-13
rox-filer with leafpad and conky

rox-filer is a lite and very flexible file manager, takes a bit of getting use to. can do desktop icons and background as well. use " rox -s " in your startup for desktop icons.

leafpad is a text editor

conky is a very configureable system monitor

sudo apt-get install rox-filer leafpad conky
