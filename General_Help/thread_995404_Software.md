### Post by perlluver on 2008-11-27
Most software can be installed through Applications>Add/Remove.  If you can't find it there, you can check [http://www.getdeb.net](http://www.getdeb.net).  .Exe's or .Msi's will usually not install in Linux.  There is Wine, A Windows software installer, but not all programs will run in there.  You will want to find .deb's.  These will install the easiest in Ubuntu.
