### Post by themusicalduck on 2009-06-19
I'm trying to import my firefox bookmarks into opera. I go to file, import and export, import netscape/firefox bookmarks. Then I pick the file bookmarks.html in my firefox profile in ~/.mozilla/firefox/qk3dxbno.default but it only imports the bookmarks that come default with firefox and none of my actual bookmarks.

Am I supposed to be using a different file from somewhere?

---

### Post by lindsay7 on 2009-06-19
Go to Bookmarks on the tool bar and click on organize. Then export the html file to your usb or a directory that you use to store data.  Close Firefox and open Opera. Import the html file you saved ( look at file import and the menu bar).
