### Post by kerry_s on 2007-03-27
You need to run this->
sudo dpkg-reconfigure -phigh xserver-xorg

Your missing a whole section called "synaptics" which is the touch pad.
You might also want to install "gsynaptics" or "qsynaptics" for better control of the touch pad. I find "qsynaptics" better if you want better finger click control.
Example:
1 finger tap is left click
2 finger tap is middle click
3 finger tap is right click

You have to add a line to the /etc/X11/xorg.conf "synaptics" section for those programs to work.

```
    Option         "SHMConfig" "true"
```
