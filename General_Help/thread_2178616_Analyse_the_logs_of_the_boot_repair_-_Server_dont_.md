### Post by oldfred on 2013-10-04
I do not know LVM. But since Boot-Repair could not see / (root) inside LVM, it just installed a Windows type boot loader not grub.
It does show it mounted the LVM and your grub says your install is in 
 root=/dev/mapper/ubuntu-root
It looks like Boot-Repair tried to run fsck to fix partition issues, but still could not totally see LVM correctly?

Are you using LVM on a 33GB drive for a reason like full drive encryption? One of LVM advantages is flexibility to resize partitions, but with 33GB you will not be resizing much. And LVM adds a level of complexity.
