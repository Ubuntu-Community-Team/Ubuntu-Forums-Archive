### Post by alvinmoneypit on 2016-10-28
deja dup backup would not report through GUI that it was not backing up to my NFTS device.  I installed luckybackup that has a GUI interface within 'modify' under 'advanced' settings within the 'command options' tab there is a checkbox titled, "destination is FAT/NFTS'.  With this checkbox checked, backup is successful, without it, it appears to work, but the files are never actually written to the device.  Adjustment may be possible through the command line to deja dup, but I'm not proficient with commands and decided to keep luckybackup as my default backup program.  I'll mark this thread as solved.
