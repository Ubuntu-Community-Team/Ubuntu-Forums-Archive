### Post by Yashiro on 2009-02-23
Enter about:config ion the Firefox address bar.
Find the value:
**browser.urlbar.clickSelectsAll 	 **

Set it as you wish.

True (default in non-Unix builds): Select the contents of the address bar when it is clicked

False (default everywhere else): Opposite of above
