### Post by cjchand on 2008-10-29
Greetings folks,

Thought I would drop a line to let you know that updated, proprietary Nvidia drivers with "preliminary" X.org 1.5 support are available:
[http://www.nvnews.net/vbulletin/forumdisplay.php?s=&forumid=14](http://www.nvnews.net/vbulletin/forumdisplay.php?s=&forumid=14)

They are indeed Beta and the word "preliminary" is in quotes for a reason.

Install went like a champ, but if I set System -> Preferences -> Appearance -> Visual Effects to "Extra," there are some issues. Namely, the title bar is missing from every window and many windows are just white (e.g.: Gnome Do, Terminal). Reverting back to "None" brings things back to normal.

Apparently this is not an isolated incident, as per [this post]("http://www.nvnews.net/vbulletin/showthread.php?t=122147"). 

The more folks we get testing these hopefully the sooner we can get a stable set of drivers. 

Happy Ubuntuing,
CJC
