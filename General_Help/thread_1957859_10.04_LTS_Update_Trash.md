### Post by 3rdalbum on 2012-04-15
> **longbeard said:**
> Ok, I understood. It is working now after plan, thank you for the support!

I'd still say that a question to the user is mandatory before triggering such far reaching actions on the system. :KS:KS

I believe it does. You insert the CD and it asks you if you want to use it with the package manager. If you choose Yes, then it makes the change to your sources.list that you observed.

Merely having the CD listed as a software source may trigger the "New version 12.04 available" message at the top of Update Manager, but if you click the regular "Install Updates" button it will just install the normal 10.04 updates. To upgrade to 12.04 you would need to press the "Upgrade" button that appears along with the 12.04 message at the top of the window.
