### Post by ajgreeny on 2011-01-19
You can use **Programs-Accessories-Archive manager** to make a large tar.gz which can be split into DVD sized chunks using the "Other options" dropdown box.  You can then burn those chunks to DVD.

There may be better ways to do it, but I have always used a large external hard disk so I've never even looked for ways.  I just give you the possibility of doing it the tarball way.

### Post by ajgreeny on 2011-01-19
You won't be splitting the individual files, only the single tarball file.  It is quite common practice to do this, so there should be no reason to worry.

Depending on what types of files are being archived as a .gz the whole backup may be a lot smaller than the size of the original files put together.
