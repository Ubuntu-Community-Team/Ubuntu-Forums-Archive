### Post by nandemonai on 2009-06-14
Sure is ;)

In terminal:

```
sudo apt-get install nautilus-image-converter
```

Once installed, log out and back in and viola!

(Should be installed by default in my humble opinion).

Alternatively, you could search for the package in Synaptic. I just find terminal a lot faster.
