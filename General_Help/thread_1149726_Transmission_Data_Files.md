### Post by HuckleSmothered on 2009-05-06
Been a while since I've done something similar, but I think you can just use the partially DL'd file and the torrent file. Make sure the partial is in DL location you have specified in Transmission though.

Transmission will detect it once you have it open the torrent file. Don't use a different torrent file though. I think that just starts a new DL.
