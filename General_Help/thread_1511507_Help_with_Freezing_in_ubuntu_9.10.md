### Post by MooPi on 2010-06-16
Is it currently working ? Have you recently updated the system ? Maybe an update didn't complete. 
This may be a hardware issue and not software. Sounds like a heat problem or faulty memory module. If you are using a laptop check to see if fans are moving or if they have become clogged with dust. On a desktop you'll need to open a side panel to see if all your system fans are functioning. Lastly pop the Ubuntu CD in and run memtest to see if there is a problem with your ram. These are all possibilities.
