### Post by ColBuendia71 on 2008-12-24
Or if you're running Gnome, you can install screenlets:

```
sudo apt-get install screenlets
```

Then I would suggest going to System>Preferences>Sessions and adding screenlets to your startup programs. I think that there is a system monitor included in the default screenlets, but if not you can download one and add it.
