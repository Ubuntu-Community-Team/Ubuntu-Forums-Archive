### Post by deadflowr on 2020-02-27
You can export from chrome/chromium.

(go to Settings > Passwords, then click on the 3 dots in the Line for Save passwords and a dialog box for Export Passwords will show, click on it.
It warns you that all passwords will be visibe to anyone who can see the file.)

The file should be saved as a .csv file or something in the location you choose to save it at.

From there importing depends on the import mechanism of whatever password manager you have.

(I'm guessing you mean the package/program passwordsafe?)
