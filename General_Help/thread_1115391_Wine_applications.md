### Post by mb_webguy on 2009-04-03
Just out of curiosity, why are you trying to install WinRAR?  The Ubuntu default Archive Manager, File Roller, can open and create rar archives as well as many others.  Install either the rar package or the p7zip-rar package to enable support for rar archives.  I suggest the latter, since it adds support for many other archive types as well.
