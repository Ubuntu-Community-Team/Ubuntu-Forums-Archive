### Post by VitalBodies on 2010-04-03
There are number of Ubuntu users and "would be" sellers that might want to stay within Ubuntu and not have to also have a Windows machine to bulk list on Ebay, their own shopping cart or Google base. Although there may be many ways to do this, I wanted to introduce a way that I found that is working for me. This took me quite a bit of time and research and I hope this thread helps others. 

In short you, have to have the right software that can do the job all the way to completion. I wanted to stay completely open source were possible. 

**Here is the software I found to be the most helpful:** (OS = Open Source)

This tutorial does not go into the countless details that are involved in setting up web sites, Ebay, shopping carts and the like. Instead this tutorial simply points the way and lets "would be" sellers that want to use open source software know there is indeed "a way" and what that way is. Again, this is not to say this is the only way, but at least you there is a way what that is... 

**Spreadsheets:** 
The key to this system is to have your listings in spreadsheet form. This allows one to bulk upload (edit, update, end, etc) ones listing on Ebay, the shopping cart, Google Base and other venues. You will end up with a spreadsheet for Ebay, one for the Shopping cart and CSV Improved will create a Google Base spreadsheet for you. 

**Open Office Spreadsheet**
In this tutorial I have suggested Open Office Spreadsheet and the tool of choice. There may be other editors that are just as nice. 

**Gedit**
There can be occasions along the way where you might need to edit in "plain text" and Gedit can help us with that. This tool can be useful to convert text or code or something you have copied onto the clipboard to plain text also. For example, you can just paste into Gedit, select what you pasted and click copy and now (like magic) what is stored in the clip board is in plain text, ready to paste into a spreadsheet or XHTML code.

**GIMP:**
This is a great tool for editing images for your listings. 

**Joomla:** 
[Joomla]("http://www.joomla.org/") is a tool for creating web sites and managing content. Joomla sites need to be styled with a template and there are many templates available but be careful not to download "free" (as in pirated) commercial templates as they tend to have viruses and other bugs. It takes some effort to set all this up and the last thing you need is your site hacked. Some templates can style both Joomla and VirtueMart. Some web sites hosts like [Dreamhost]("http://www.dreamhost.com/r.cgi?477702") offer one click installs of Joomla and easy database management so installation is easy. 

**VirtueMart: **
[VirtueMart]("http://virtuemart.net/") is a component for Joomla that adds a shopping cart into Joomla. This is nice in that you are not dealing with a cart and separate web site for the sake of simplicity and styling. 

**CSV Improved: **
[CSV Improved]("http://www.csvimproved.com") is an import export component for VirtueMart so you can import, manage and export your listings. You can also export a ready made CSV file for Google Base. 

[IMG]http://www.vitalbodies.com/site/uploads/images/graphics/CSV_Improved_Team_Work.png[/IMG]


**Ebay:**  

Ebay tends to be very proprietary and can lead one down that path - to help avoid that fate you can use Ebay File Exchange (within Ebay) rather than Turbo Lister (a software program) which tends to only run on certain Operating Systems.  If one learns Ebay File Exchange then they only really need to use Turbo Lister once to grab their listings for export so they have the item numbers to match up to the title and description in the spread sheet they edit in Open Office. That is if they have a lot of listings already on ebay, otherwise one really does not need Turbo Lister at all. At the time I needed to download my listings I did not know about open source [jAOLT]("http://code.google.com/p/jaolt/") that can also do that task. I have not tried jAOLT but it is well worth a look and I would have tried it for sure. 
Additionally, if you leave Turbo Lister behind and switch to File Exchange you are are much more ready to load your own web sites cart with a spread sheet and add your products to Google Base. This frees you from Windows, IE, Wine and Turbo Lister. I was able to do just that, using the software mentioned in this tutorial. Consider keeping the item titles at 55 characters or less so they are compatible with Ebay. 

**Firefox: **
I mention Firefox because I know it works throughout the whole process. Epiphany is nice for double checking web pages as it uses the Webkit engine to display web pages and Webkit which can pass the acid3 test for CSS.
