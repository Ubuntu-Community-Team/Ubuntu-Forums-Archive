### Post by finite9 on 2007-11-15
you can create/convert videos for use with iPods using Mencoder.  If you feel uncomfortable using the command line, then get gmencoder which is a GUI front-end to Mencoder.  See the Mencoder documentation for how to accomplish this:

[http://www.mplayerhq.hu/DOCS/HTML/en/encoding-guide.html](http://www.mplayerhq.hu/DOCS/HTML/en/encoding-guide.html)

Install the software using:

```
sudo apt-get install mplayer mencoder gmencoder
```

And don't forget to check out the iPod posts on the forums!  This post seems to have exactly what you need, but it's a bit of command line stuff involved:

[http://ubuntuforums.org/showthread.php?t=114946&highlight=ipod+video](http://ubuntuforums.org/showthread.php?t=114946&highlight=ipod+video)
