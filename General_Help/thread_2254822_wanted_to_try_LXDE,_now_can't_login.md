### Post by dry crust on 2014-11-30
Well, I discovered there was a thing called Recovery Mode, which is activated by pressing the shift key. You'll need to Google it to find out more. I used an Ethernet cable to connect to my modem. I found the commands to download the Lubuntu desktop and did that. At the end it asked which desktop manager to use and listed gdm, lightdm, and lxdm. Gdm lists the users and supports closing the laptop lid, so I chose that one. Finally typed in the reboot command and finally got the normal login screen. I chose lubutu as the environment. I think I prefer Gnome, but I'll continue on with this until I reboot. You can close this now.oh, and don't forget to do those backups!
