### Post by oldfred on 2010-03-17
Are you able to boot at all? Script does not show any boot loader at all. (it shows win95 or DOS partition so it may not see really old stuff)  Did script miss something  or are you not booting?

You have wubi, do not install grub to the MBR, you need a windows boot loader to choose windows or Ubuntu.

See the fix on link below:

[http://sourceforge.net/apps/mediawiki/bootinfoscript/index.php?title=Boot_Problems:Wubi_9.10](http://sourceforge.net/apps/mediawiki/bootinfoscript/index.php?title=Boot_Problems:Wubi_9.10)
[http://ubuntuforums.org/showthread.php?t=1376845](http://ubuntuforums.org/showthread.php?t=1376845)
Patch made available  12/23/2009
[https://bugs.launchpad.net/ubuntu/+source/lupin/+bug/477169/comments/210](https://bugs.launchpad.net/ubuntu/+source/lupin/+bug/477169/comments/210)
