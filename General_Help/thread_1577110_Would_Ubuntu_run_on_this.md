### Post by jocko on 2010-09-18
Ubuntu will run fine on 512 Mb ram, but it may perform a bit slowly if you do anything that require more memory. My experience (probably not entirely unbiased) is if it runs XP with decent performance, it will run ubuntu with (at least) as good performance.

So this post is NOT true:
> **jeremy96 said:**
> No, Ubuntu might not run on your system. The problem is that you do not have enough RAM. You need AT LEAST 1 GB of RAM to run Ubuntu.
 Recommended minimum (1Gb) and absolute minimum possible amount is NOT the same. I have successfully tried ubuntu 10.10 64 bit (when it was in alpha) on only 128 Mb ram in a virtual machine, and it RUNS (but quite poorly since it becomes completely dependent on a swap partition).
