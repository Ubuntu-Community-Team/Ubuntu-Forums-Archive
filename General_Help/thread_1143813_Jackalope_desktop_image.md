### Post by CatKiller on 2009-04-30
It's in the ubuntu-wallpapers package. If you can't find it on your cd, you can download the package from [here]("http://packages.ubuntu.com/intrepid/ubuntu-wallpapers"). Don't install it, just open it in File Roller, and you'll be able to extract the file you want. For legacy reasons (I assume) it's called warty-final-ubuntu.png, even though it's actually a JPEG.
