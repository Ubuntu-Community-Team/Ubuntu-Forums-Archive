### Post by zombifier25 on 2012-11-17
For the login logo, it is found in /usr/share/unity-greeter

If you want to change the desktop session, you don't need to install another OS like offgridguy said, you can install them right on top of your current system. There's gnome-shell, gnome-panel, plasma-desktop (KDE), xfce and lxde.
