### Post by Firetech on 2005-05-11
I run OO.o 2-beta, because it's better kde-integrated (I run kubuntu). However, I've noticed that removing OO.o 1.1.3 makes my spell checking dictionaries disappear from OO.o 2 (all of them). Reinstalling makes them appear again... So I have both installed. It's good to have 1.1.3 left just in case 2-beta starts crashing...
The main difference (AFAIK), is that OO.o 2 uses the OpenDocument standard, and OO.o 1.1.3 uses it's own (shared with StarOffice). 2-beta has support for the old system (of cource), but it recomends every document to be saved with the new system, to conserve all looks.
