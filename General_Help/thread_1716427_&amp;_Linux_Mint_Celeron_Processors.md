### Post by tgalati4 on 2011-03-28
I have a 500 MHz PIII with 768 MB of RAM that runs well with Linux Mint XFCE.  Celerons have small cache and don't seem to run as well as PIII, but they use less power (66 watts vs 45 watts typically).  Both have the same 32-bit instruction set so there is no specific kernel or distro targeted for the Celeron.
