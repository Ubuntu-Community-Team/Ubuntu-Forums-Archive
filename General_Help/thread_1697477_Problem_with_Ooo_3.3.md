### Post by Tamlynmac on 2011-02-28
If your opening them with Nautilus (file browser).
1. locate one of each of the file types (.odt, .ods, etc.) you wish to auto open in OO.
2. Right click on each file type (one at a time) and select properties.
3. Select the "Open With" tab
4. Remove all "open with" options (except open office) by selecting each and clicking on the remove button at the bottom of the window.
5. Select the open office option (spreadsheet, word processor, etc.) and left click on the close button at the bottom of the window.

If your opening them from a screen icon.
1. Right on the icon and select properties
2. Select the "Open With" tab
3. Remove all open with options (except open office) by selecting each and clicking on the remove button on the bottom of the window.
4. Select the open office option (spreadsheet, word processor, etc.) and left click on the close option at the bottom of the window.

Once you've change one file of the file type all other files of that type should auto open in the selected application. For example, by changing the open with properties of one .odt all other .odt files should default to OO. 

Good Luck & hope this helps.
