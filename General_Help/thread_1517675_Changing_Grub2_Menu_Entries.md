Thanks a lot for all of that! BURG actually works pretty similarly to GRUB so it should all be the same. Thats exactly what i needed, but i happened to find a different work around. In BURG, theres an option for folded menus, which ended up taking out most of that extra stuff.

I set

> GRUB_FOLD=true

in my /etc/default/burg.cfg

I also added

> GRUB_TITLE_OVERRIDE="/dev/sda1=Windows XP"

So that the Windows XP entry looked simpler

Then did a "sudo update-burg" and it worked.
