### Post by tgalati4 on 2010-12-21
sugarcrm has that capability.  Its email client is clunky, but it works.  You install sugarcrm on a server then access it from a web page.  You can create a list of tasks then assign those tasks to workers and then send batch emails out in the evening.

There are also several project management/code management tools that do a similar email tasking function.

You can also ssh -2 -Y and bring up any graphical email client and set up a mailing list then attach the tasker document to that mailing list.
