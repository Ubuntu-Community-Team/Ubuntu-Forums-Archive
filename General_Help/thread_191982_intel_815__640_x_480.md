### Post by givré on 2006-06-08
Try to reconfigure xorg
```
sudo dpkg-reconfigure xserver-xorg
```
It will ask you, after few question, the resolution you want, make sure you know the refresh rate of your monitor.
Check this howto to know more 
[http://doc.gwos.org/index.php/ChangeResolution](http://doc.gwos.org/index.php/ChangeResolution)
