### Post by Mark Phelps on 2012-02-07
You said it takes 30 second to show the logo -- which is not bad for Ubuntu 11.10.

So, what happens after that?  Are you then able to login?

If you are, then next time you login, open a terminal and enter "sudo update-grub" (without the quotes).  This will regenerate the default GRUB config file. Watch as it runs and you should see it detect Windows XP on your PC.

When you reboot, you should then be presented with a GRUB menu.
