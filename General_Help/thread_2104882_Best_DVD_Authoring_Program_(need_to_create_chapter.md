### Post by flying-man on 2013-09-14
I do not know if this thread is active or not, but I found DVD Styler, a pretty decent cross-platform DVD Authoring program. It allows chapters within the video file, a whole video file being a chapter or a combination of both. It allows moving backgrounds in menus, chapter selecting frames with still images from the specified chapter and allows you to put anything into the menu, like buttons text, etc. The only setback is that it only allows PAL or NTSC format and not any HD Video. Check it out [here]("http://www.dvdstyler.org/en/").
