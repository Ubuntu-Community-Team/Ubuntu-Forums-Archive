### Post by garvinrick4 on 2011-01-13
When you go into Synaptics and in upper right type plymouth.
Now does it show plymouth theme kubuntu as being installed?
right click and remove and apply.
Now install plymouth theme ubuntu with right click and apply.
## Do not remove plymouth itself has other packages needed to run ubuntu.
