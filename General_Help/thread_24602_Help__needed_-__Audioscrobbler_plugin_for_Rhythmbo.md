### Post by AndyM on 2005-05-18
You've probably figured it out by now, but in case other people are looking for an answer...

Installing the Audioscrobbler plugin (it's a separate application actually) is easy:
* Download the newest version here: [http://members.cox.net/alexrevo/](http://members.cox.net/alexrevo/)
* Unarchive it and move the directory somewhere like /usr/local/bin/
* Run rbscrobbler.py to install and begin the app
* Run rbscrobbler.py to start up the app whenever you open Rhythmbox
Alternatively, you can start up rbscrobbler when Ubuntu loads by going to System>Preferences>Sessions, clicking the Startup tab, and adding something along the lines of "/usr/local/bin/rbscrobbler-0.0.9/rbscrobbler.py"

Hope this helps someone!
Andy M.
