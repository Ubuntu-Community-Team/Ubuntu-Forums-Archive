### Post by dyfi on 2012-05-23
If your USB drive is running Ubuntu or a Debian distro, install gnome-commander from repos and open as root and from there you can change file permissions.  As a second method, if you have a CD drive, download and install Parted Magic to a CD - again this would access your files.  Hope this may help you.
