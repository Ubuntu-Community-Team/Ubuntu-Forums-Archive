### Post by r_l on 2007-08-20
What model is your computer?  How new is your computer (how long have you been using it)?  Do you have a system restore disk come with the computer?  Ubuntu or not, may be try save the system first by using a boot up disk (or a restoration disk).

In the Wuby FAQ, it says - 
"Hibernation/suspend is not supported under Wubi, moreover Wubi filesystem is more vulnerable to hardreboots (unplugging the power) than a normal filesystem, so try to avoid unplugging the power."   

Did your system went into suspend or was it not plugged so that the battery ran out (then it would be like unplugging).  Perhaps this is the one that caused problem.
