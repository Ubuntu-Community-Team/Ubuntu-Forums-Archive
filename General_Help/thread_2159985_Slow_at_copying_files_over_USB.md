### Post by Impavidus on 2013-07-05
Maybe the write doesn't write directly, but stores the data in a buffer, whilst the close flushes the buffer and waits until the write has really completed. I think that if you don't use the flush as a mount option, the close will return immediately, with the buffer being flushed at a later time.
