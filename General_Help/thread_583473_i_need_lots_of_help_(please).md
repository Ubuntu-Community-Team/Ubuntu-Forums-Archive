### Post by thirddeep on 2007-10-20
To install applications, such as pidgin (formerly gaim) for IM:

1) Click on "System"
2) Click on "Administration"
3) Click on "Synaptic Package Manager"
4) Type in your password
5) Click on "Search"
6) Type in the name -or part thereof- of the application
7) Click in the first column (s) and select "Mark for Installation" 
8) Click "Apply"

That is the GUI way, very boring.

From the command line:
```
sudo apt-cache search XXXX
sudo apt-get install XXXX

```
Far better :-)
