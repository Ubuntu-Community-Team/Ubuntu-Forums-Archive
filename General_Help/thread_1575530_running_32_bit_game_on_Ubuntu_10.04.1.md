### Post by 3rdalbum on 2010-09-15
1. 3D games don't work in virtual machines anyway, but your description of the exact problems with the virtual machines was not specific enough for us to provide any help. This is the wrong way to go about playing 3D games, anyhow.

2. Warcraft 3 works in Wine, check out [http://appdb.winehq.org/objectManager.php?sClass=version&iId=1177](http://appdb.winehq.org/objectManager.php?sClass=version&iId=1177)

3. You don't need the Ubuntu server edition to run server software. The only difference between Ubuntu Server and Ubuntu Desktop is that Desktop comes with a GUI.
