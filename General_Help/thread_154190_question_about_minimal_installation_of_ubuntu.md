If you do a server install, it's quite fast. My advice would be to do the server install and then to just install only what you need on top of that. The full *ubuntu-desktop* package can take quite a while.

### Post by aysiu on 2006-04-03
I don't think it would save time. Ubuntu's default installation takes about a half-hour to forty-five minutes for me. The time it would take to isolate the exact packages you want and install only those would be far more than the difference you would save.

You're better off doing a default install (which, honestly, is quite slim--one application per task--including Evolution, OpenOffice, and Firefox) and then *uninstalling* the applications you don't think you need.
