### Post by sailor.nir on 2011-02-15
minimize (toggle): Alt-F9
maximize (toggle): Alt-F10
restore: Alt-F5
resize: Alt-F8

For many more options, check [https://help.ubuntu.com/community/KeyboardShortcuts](https://help.ubuntu.com/community/KeyboardShortcuts)
