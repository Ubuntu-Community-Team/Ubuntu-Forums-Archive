### Post by forestpixie on 2007-09-04
you probably need the ntfs tools to allow read/write on your vista docs

go to synaptic in the system >admin menu and type ntfs in the search box - you need ntfs-3g and ntfs-config I believe

once it's installed there will be a tool in the apps >system tools menu

There are plenty of threads here - search for ntfs and go from there :)
