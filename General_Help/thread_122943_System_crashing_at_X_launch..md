### Post by tseliot on 2006-01-29
Keep pressing ESC as soon as you turn on your computer. You will access the GRUB menu.
Select "Recovery Mode" (you will boot in the command line, NO GUI)
type:
sudo nano /etc/X11/xorg.conf

Scroll down the file and get to the "Section module"
change the driver from whatever it is to "vesa"

Press CTRL+O to save
CTRL+X to exit

then type:
sudo reboot

and everything should work (you can boot normally now)

### Post by dcstar on 2006-01-31
[QUOTE=Heko]Update: after removing the glx driver and enabling Vesa, it surprisingly works. However, now I don't have 3D/2D acceleration. What is the problem with the glx driver?
......[/QUOTE]
VESA is a basic "vanilla" mode that (virtually) all video cards support, it is there as basically a "last resort" to get your X system working.

If you want the 2D/3D stuff, you have to get your native video going.

You can do:

**sudo dpkg-reconfigure xserver-xorg**

to go through the configuration again.
