Firefox2, X.org 7.1 (built in AIGLX if you use any kind of composite window manager), basically the packages are newer.  The art is much better (bootsplash doesn't make me cringe, esp the kubuntu splash which is quite pretty actually).  Plus for me the hardware detection was better and its much more stable for me (this varies system by system, many people would disagree... try the livecd and see).

To upgrade or not: If you want to keep current with new releases as a general rule, then yes, get Edgy.  If you are happy with upgrading on a more windows like schedule (more stable, less cutting edge) stick with dapper until the next LTS release, which will be a while.  I personally recommend Edgy, I think its ages better than Dapper, but I like to have the newest stuff, so take that as you will.

### Post by mcduck on 2006-12-14
Kernel in Edgy has better handling of multi-core CPU's. The boot process is faster. And it also has support for some hardware that isn't supported in Dapper. For example I got my laptops memory card reader working, and I'm also now able to use my wired network with Network Manager, in Dapper I couldn't do that. And I could once again set Gnome's panels to use transparent background pictures, that was broken in Dapper. And of course there is newer versions of many apps.
