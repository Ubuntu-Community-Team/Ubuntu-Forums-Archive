### Post by marshmallow1304 on 2010-05-05
NTFS.  Mac can read only by default (AFAIK) but you can add write as per [this article]("http://www.lifehack.org/articles/lifehack/how-to-read-and-write-ntfs-windows-partition-on-mac-os-x.html").

Edit: There are other solutions, such as adding ext3 support to Mac and Windows or adding HFS+ support to Windows, but those are both more tedious than using NTFS.
