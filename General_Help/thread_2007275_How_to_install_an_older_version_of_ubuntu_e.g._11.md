### Post by 3Miro on 2012-06-20
You cannot downgrade an existing install, but you can make a full install of an older version. From the bottom of the page, select a country that is closest to you (i.e. USA) and you can select older versions like 11.10, 11.04 ...

[http://www.ubuntu.com/download/desktop/alternative-downloads](http://www.ubuntu.com/download/desktop/alternative-downloads)
