### Post by spin2cool on 2007-03-27
EXE files are windows executable files and will not work natively on linux, without going through some sort of emulation layer or a program like WINE.  If you want to use it on linux, you should probably find the appropriate package in Synaptic, or find an alternative program, if it isn't available for linux.  

If you tell us what the program is, we may be able to help you more.

---

### Post by spin2cool on 2007-03-27
Oh, and to extract the contents of a zip file, just double click it in nautilus, or right click and choose "extract here".  To do the same thing from the command line, type "unzip <filename>".

### Post by spin2cool on 2007-03-27
The easiest way to install WINE is probably through automatix:
[http://www.getautomatix.com/](http://www.getautomatix.com/)
After that, things can sometimes get complicated.  Check the WINE AppDB to find out if your applications are supported and get more detailed instructions.   [http://appdb.winehq.org/](http://appdb.winehq.org/)

What kind of music programs are you talking about?  If you want players, amaroK, xmms, and many others are available.  If you want editing, Audacity is a solid editor, and there are other more specialized ones that are linux-native.  Search these forums and you'll find lots more info.
