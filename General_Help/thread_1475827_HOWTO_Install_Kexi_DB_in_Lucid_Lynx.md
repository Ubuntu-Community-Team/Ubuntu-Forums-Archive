Hi

In case anybody is battling to install Kexi database front end in Lucid Lynx, you have to first enable kubuntu ppa beta repository.  Enter the following in terminal:

sudo add-apt-repository ppa:kubuntu-ppa/beta 
sudo aptitude update
sudo aptitude install kexi
