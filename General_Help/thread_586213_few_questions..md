### Post by Scheater5 on 2007-10-22
Welcome to the world of Linux and Ubuntu.
Maverick02, most definitely Linux is safer than Windows.  Not only is Linux designed to be safer, but there are also significantly fewer malicious programs written for Linux.  Should you still feel the need for an antivirus, there are completely free antiviruses and firewalls in Linux (in fact, many people take old computers and turn them into firewalls using Linux - it's that safe!).  
As for downloading programs, through the magic that is the APT system (Advanced Package Management) you will find all you are likely to need is already prepared and ready for use in Ubuntu.  With an internet connection you can install them in just a few clicks - and all for free.  If you find something you need that is not prepackaged, the method will vary, but if the website doesn't have instructions for Ubuntu you'll find help her on the forums.  
Again, Welcome to Ubuntu.

### Post by MadhavS on 2007-10-22
Ubuntu is safer than windows, in terms of how many viruses/spyware that ubuntu users get compared to windows. However, windows is used much more than ubuntu, and so people are more liable to hack windows than ubuntu. However, if i understand correctly, you are asking if you need antivirus software like in windows, the answer (i believe) is generally no. Ubuntu's built-in protection (apparmor) is enough to protect it from viruses and spyware. Hope that answered your questions.
