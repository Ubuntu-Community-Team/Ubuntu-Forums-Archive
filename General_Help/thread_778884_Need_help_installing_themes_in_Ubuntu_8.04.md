### Post by halln on 2008-05-02
This what I did: Downloaded the theme from gnomelook.org and save it to desktop. Open system/pref/appearance/ install theme/select file location/desktop/open/ then back to appearance your theme should be under customize not on the theme page. I hope it works for you let me know.
