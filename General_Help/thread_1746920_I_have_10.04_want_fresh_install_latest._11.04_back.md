### Post by Ghost_Mazal on 2011-05-02
All your settings is in various folders in your /home folder. If you make a full backup of that and put it on the new install all your settings will be there. Copy the whole /home/username folder as there are many hidden folders in there.

For the applications itself I found that remastersys is best. It makes a installable iso that you can burn and it will do a install the same as if you do it with a live cd , but all you apllications is already there and don't need to be downloaded again.
