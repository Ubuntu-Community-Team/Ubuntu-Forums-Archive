### Post by kurt18947 on 2011-12-04
I'm using Gnome-shell, Unity may be somewhat different but should be similar.  I'm using a wireless connection and change DNS settings by left clicking the network icon on the upper right part of the screen then click 'network settings'. Click 'wireless', 'configure', then 'IPv4 settings'.  Enter DNS server numbers where indicated separating server addresses with a comma.  I hope this helps you.
