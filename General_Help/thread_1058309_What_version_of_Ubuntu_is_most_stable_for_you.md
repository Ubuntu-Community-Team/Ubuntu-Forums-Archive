### Post by AusIV4 on 2009-02-07
I've run MythTV on a desktop for several years. I consider it to be a critical box that should have maximum uptime, so I don't change its configuration or run dist-upgrades very often. 

I first built it when Dapper was in beta. Early on I didn't find it particularly stable, but I think that may have been a hardware issue. Eventually I replaced most of the hardware in that system to the point that I needed to do a re-install. At that point, Gutsy was out, and it has about the same end of life as Dapper, so I went with the more modern release.

Not wanting to mess with anything, I left it on Gutsy when Hardy came out. When Intrepid came out, I decided that Gutsy's end of life was drawing near, so I upgraded to Hardy, where it will stay until I have a strong motivation to upgrade it.

Long story short, if you need something that does its job and just keeps working, stick with the LTS so you won't have to make significant changes very often. If you want the most recent software, you can still get a pretty reliable system with whatever the most recent (non-alpha/beta) release of Ubuntu happens to be.
