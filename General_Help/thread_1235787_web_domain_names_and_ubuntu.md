### Post by blueridgedog on 2009-08-09
You have to register a domain with a registration agent of some type (such as network solutions or godaddy, and after that you can point the domain to your server for either DNS hosting or for content hosting (web, mail, ftp and the like). It is not possible without some type of domain registrar.

You may be able to get a subdomain from another soucre, such as mydomain.someotherdomain.com, if you have a relationship with the owner of the parent domain.  Dynamic DNS arrangements work this way.
