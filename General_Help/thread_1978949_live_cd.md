### Post by grahammechanical on 2012-05-12
You do not say what version of Ubuntu you are using. If you have Ubuntu 12.04 then you will find a utility that will create a bootable Ubuntu on a USB memory stick.

It is called USB Creator. You find it by opening the Dash and typing USB. To do what you want you need to install Ubuntu on the USB stick with Persistence. Other wise the USB stick will be a like a live CD - none of your changes will be written to the memory stick.

USB Creator makes this easy. You browse to the ISO image that you wish to use. 

Under the heading - When starting up from this disk, documents and settings will be:

You tick Stored in reserved extra space and you move the slider to set aside the amount of extra space that you think that you will need.

Do not tick Discarded on shutdown, unless you save them somewhere else.

Then click Make start up disk.

Regards.
