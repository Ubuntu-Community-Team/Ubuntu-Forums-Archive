### Post by cowbud on 2005-05-13
What type of stream are you listening to? 

If it is mp3 make sure you have gstreamer0.8-mad installed.

If you do not simply type

sudo apt-get install gstreamer0.8-mad 

on the command prompt. As far as you installing the other applications it should not have stopped rhythmbox from working.
