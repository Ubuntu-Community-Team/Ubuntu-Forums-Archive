### Post by fragos on 2009-07-01
Right click a file of the type you wish to associate and select Properties and then the Open With Tab. The new setup will work for all files of that type. You mentioned .EXE files which are Windows executable. Installing "wine" may make them executatable but they can't be run directly by Linux.
