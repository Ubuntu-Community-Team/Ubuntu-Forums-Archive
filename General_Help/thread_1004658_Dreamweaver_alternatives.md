### Post by radical3 on 2008-12-16
dont listen to these guys go with **quanta plus**, if your coming from dreamweaver quanta is the best choice, and it has all the stuff DW has , manage sites(called projects), ftp, and you know how dreamweaver has those tabs at the top to add styles, tables, divs and all those other stuff you can throw in. as well as a fantastic css editor similar to DW's css editor.

has an endless amount of configurations and can become anything you want it to become,  definitely recommend it. 

install it with

```
 sudo apt-get install kdewebdev

// sudo apt-get install quanta will ask you to install "kdewebdev"

```
 or **Add/Remove** --> search: **Quanta**
