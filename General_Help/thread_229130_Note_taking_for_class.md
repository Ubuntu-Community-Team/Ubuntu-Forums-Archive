### Post by orb9220 on 2006-08-04
gjots2 in synaptic

A simple jotter (outline processor) for X11/gtk-gnome
gjots2 is a fairly simple jotter (outline processor) application for your
desktop.

You can use gjots2 to organize your jottings into a tree structure, adding
thoughts and miscellany as you go. You can get it to spit out HTML, XML,
postscript, pdf, man etc etc if you want (see the online manual for an example
of the HTML conversion).

Gnome-think

Hierarchical organizer and outliner
Think is a hierarchical organizer, allowing you to organize text in a tree
form. It uses XML for its native file format.

All I did was search in synaptic on  outline for search term.

### Post by sethmahoney on 2006-08-04
I really like Tomboy for taking notes.  You can have a few different windows open to compile notes in (handy if the instructor is jumping around) and then work an outline window that links to those other documents.  It also tends to help (me, anyway) link up seemingly random topics.  The only problem with it is that document titles with more than one word can be dangerous, as in some circumstances they will make it crash.

### Post by jstrube on 2006-08-06
I like [Treeline]("http://www.bellz.org/treeline/") better than kjots.
But what I recently discovered is [BasKet]("http://basket.kde.org/"). It is simply great for just organizing stuff. A bit like tomboy (which maybe better for taking notes) but then you can extend the content later with additional information. Inline preview ! And it has tomboy import, so you can take notes in tomboy, then after class do some research and add content in BasKet.
Check out the beta of the basket note pad. There's no package, but IMHO it's much better than the "original" version. It has a free-form mode which just lets you just drop content somewhere on the page as you go, then later you can re-arrange it by dragging the snippets in order.
