### Post by Ramses de Norre on 2006-03-13
There should be an icon for it at one of your task panes, an open door with a red arrow. If not: right click on panel > add to panel > something like log out.. look for the door;)

You can also go for the CLI way: type sudo init 0 or sudo shutdown -h now (you can replace the now with a number for the minutes to wait before shutting down.)
