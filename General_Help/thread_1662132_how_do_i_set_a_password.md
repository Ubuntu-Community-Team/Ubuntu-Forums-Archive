### Post by 23dornot23d on 2011-01-07
That will set the Admin Password ( if it has not already been set on installation - if it has then it will allow you to change it ) ..... **sudo passwd**

but to set a password for a User logging into a Desktop Manager you need to go to  ......

 **System - Administration - Users and Groups**

and you will need the Admin Password to change the user password/s ....
