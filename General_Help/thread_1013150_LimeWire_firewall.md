### Post by amdalex on 2008-12-16
I use Frostwire and enable UPNP in Frostwire and the router, it automatically configures port forwarding. Frostwire works better than Limewire free because it connects to 5 supernodes rather than the 3 Limewire connects to. Frostwire is essentially the same program (same code and runs from Java) as Limewire but it works better. They have a deb. for Linux.
