### Post by ubnoobie on 2006-06-01
desktop is a live cd and graphical installer.
alternate is the (old style) text installer. also has a "server" install, which is a "bare" install. not to be confused with the separate "server" version iso.

the installed system is the same for the version you download (ubuntu, kubuntu, xubuntu...)
