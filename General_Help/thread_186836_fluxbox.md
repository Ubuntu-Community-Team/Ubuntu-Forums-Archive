### Post by Sutekh on 2006-06-02
I assume you don't want to go in for any big dependant-heavy applications.

You could install a program like [gtk-theme-switch](http://packages.ubuntu.com/dapper/x11/gtk-theme-switch) from the universe repository
```
sudo apt-get install gtk-theme-switch
```

or install [gtk-chtheme](http://plasmasturm.org/code/gtk-chtheme/) manually.

### Post by mscman on 2006-06-02
Do```
sudo apt-get install gtk-theme-switch
```from terminal and then run ```
switch2
```
That will let you change the gtk themes for programs.

NOTE: This information found from the Ubuntu Wiki topic located [here]("https://wiki.ubuntu.com/Fluxbox"), a great howto for fluxbox
