### Post by merlinus on 2007-07-15
You will probably have to change the mount point, for starters.  It is probably mounted at /media/storage at the moment.

Look at the ouput of these commands to get info:

```

sudo fdisk -l
mount
cat /etc/fstab

```

-merlin

### Post by merlinus on 2007-07-15
/dev/hdb3 /media/disk ext3 defaults 0 0

It is mounted at /media/disk, so you should be able to access it.  I do not understand exactly what you are wanting, or what partition is mounted to /storage, as nothing like that appears in /etc/fstab.

You might also look at:

```

mount

```
