### Post by CrushingYourHead on 2011-05-17
If you're using 10.10 or previous, it's in your Gnome menu, you can right-click the entry and the context menu should have the option to create launcher on the panel or desktop.

If its not in your menu, just add a new Chromium entry with the menu editor.
