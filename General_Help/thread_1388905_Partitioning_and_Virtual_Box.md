### Post by bpalone on 2010-01-23
First question is what OS are you wanting to install VirtualBox on?

If on Ubuntu (or windows for that matter) go and get the PUEL version from [here]("http://www.virtualbox.org/wiki/Downloads") and while you are there get the manual.  The manual tells you how to install it.

VirtualBox allows you to create a virtual machine, or a computer within a computer.  You don't have to partition, you just need enough free space to handle the virtual machines Hard Drive that it creates.  It is a great way to play with different OSes, without messing with your existing setup.  It is also good for allowing you to have access to programs, etc that you can't get for your favorite day to day OS.

Hope that helps you out a bit.

### Post by howefield on 2010-01-23
> **esosa233 said:**
> Can someone give me some info on virtual box like how to install and what it does and such, and if I have a 362 Gb memory how should partition my system

To go back to your original question, Virtualbox allows you to run different operating systems simultaneously, ie, without the need to reboot your machine.

It is cross platform. So you can install it in to Linux, Mac or Windows systems. The system you install it on becomes the host system, and the virtualised system is known as the guest operating system.

Once you have installed it and create you first virtual machine, it will take you through creating the "hard disk" for the guest. You do not need to partition your disk.

If you are running an xp guest for example, you might give it a 10 gig hard drive, or maybe 20, depends on what you want it for.

If you are more specific, you'll get more specific help.
