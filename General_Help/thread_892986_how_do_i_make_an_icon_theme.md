### Post by Genius314 on 2008-08-17
1. Copy /usr/share/icons/Human (or another icon theme, it doesn't really matter) to /home/username/.icons/YourThemeName (you might have to make the folder if it doesn't exist.)

2. Edit the index.theme file, so that the "name" and "comment" tags describe your theme.

3. Now just replace each of the files in there. I'm not too sure how to add more icons, but that should get you started. :)
