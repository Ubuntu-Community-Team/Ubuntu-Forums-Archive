Hi,
the twiki package for 8.04 is obviously old, it's v4.1 whereas the latest version according to the homepage is v5.1.

I haven't installed twiki myself but there is a guide for manual installation on the homepage: 
[http://twiki.org/cgi-bin/view/TWiki05x01/TWikiInstallationGuide](http://twiki.org/cgi-bin/view/TWiki05x01/TWikiInstallationGuide)

However this guide which I found seems better since it's specific to Ubuntu (for Ubuntu 10.10, but it should be relevant for 12.04 as well):
[http://library.linode.com/web-applications/wikis/twiki/ubuntu-10.10-maverick](http://library.linode.com/web-applications/wikis/twiki/ubuntu-10.10-maverick)

My advice would be to try the Ubuntu guide, and if you get stuck check the official guide.
