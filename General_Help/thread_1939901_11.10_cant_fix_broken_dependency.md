### Post by wojox on 2012-03-12
Take a look at this [dpkg: warning: &#8216;ldconfig&#8217; not found in PATH or not executable]("http://cksum.org/blog/149-dpkg-warning-ldconfig-not-found-in-path-or-not-executable"). It looks like you will need to edit /etc/sudoers file with "visudo" and add the two lines.
```
1 upgraded, 1 newly installed, 0 to remove and [COLOR="Red"]351 not upgraded.[/COLOR]
```

And upgrade after you fix that. :p
