### Post by daveshields on 2007-09-07
In a word, [WordPress.com]("http://wordpress.com") . I've used it for over a year to power by blog. It's free and the underlying software is also open source. See [WordPress August Wrap-up: Game, Set, Match]("http://daveshields.wordpress.com/2007/09/06/wordpress-august-wrap-up-game-set-match/")

They have recently added a new feature that supports posting source code. See [Posting source code in WordPress]("http://daveshields.wordpress.com/2007/09/03/posting-source-code-in-wordpress/"). See also [HTML is Many Things]("http://daveshields.wordpress.com/2007/09/04/html-is-many-things/") for another example of this feature in action.

I've also written a number of posts about Ubuntu: [ubuntu-posts]("http://daveshields.wordpress.com/ubuntu-posts/"). Most of my views these days are of these posts. For example, my most read post so far today is [Configuring Intel 845G video chipset on Ubuntu 7.04 Feisty Fawn desktop]("http://daveshields.wordpress.com/2007/08/21/configuring-intel-845g-video-chipset-on-ubuntu-704-feisty-fawn-desktop/")

WordPress supports the import and export of several blog formats, and you can archive your blog at any time in the form of an XML file that you can download to your machine.
