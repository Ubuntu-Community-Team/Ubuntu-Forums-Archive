### Post by klz300 on 2011-05-14
After installing Ubuntu 11.04 (64-bit) on a dual screen setup the second monitor is fuzzy and shaking.

Here is some extra info:

VGA: ATI Mobility Radeon HD 5650 with Intel GMA HD Switchable Graphics

*This happens with both open source and proprietary drivers.

*Refresh rate is properly set.

*Using RGB Monitor Cable (VGA Cable).

**EDIT: FIXED. Apparently the Switchable Graphics feature is incompatible with Ubuntu. After disabling it in the bios and doing a fresh install it works fine.**
