### Post by drs305 on 2009-08-24
The normal repositories generally don't go back that far.

You can probably be to get the old package from Launchpad. If it was once in the official repositories, Launchpad will probably have it:
[https://launchpad.net/ubuntu]("https://launchpad.net/ubuntu")

Each step is on the resultant next page except the last:

Enter the app in the "Find a package" window.
Click on the applicable name in the next page.
Click on the version under Version History
Click on the Builds section.
Find the deb file and download it.
