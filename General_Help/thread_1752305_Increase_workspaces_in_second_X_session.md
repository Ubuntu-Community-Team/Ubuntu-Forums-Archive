I have 2 video cards, two monitors, and two X sessons.  Regardless of which session I click on my workspace preferences, to increase the number of workspaces... it always increases them in screen number 0.  How do I affect the workspaces in screen number 1?

---

### Post by papibe on 2011-05-07
Install "Advanced Desktop Effects Settings (ccsm)" from the Software Center. It'll go in System -> Preferences -> CompizConfig Settings Manager. Open it on the X session you want to configure. 

Go to General Options -> Desktop Size, and increase the Horizontal Virtual Size.

I hope it helps.
