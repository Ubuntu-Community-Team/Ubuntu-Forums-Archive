### Post by matrooswolf on 2007-12-10
Picasa 2.7 for linux preview is out.

It should (finally) have web album support.

[http://picasa.google.com/linux/download.html]("http://picasa.google.com/linux/download.html")# 

You can install it with this repository

```
#Google testing repository
deb [http://dl.google.com/linux/deb/](http://dl.google.com/linux/deb/) testing non-free
```

Then:
```
sudo apt-get update
sudo apt-get install picasa
```
