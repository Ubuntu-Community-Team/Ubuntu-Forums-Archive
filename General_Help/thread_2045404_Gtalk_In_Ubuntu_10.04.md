### Post by Zvlwab on 2012-08-21
There are lots of XMPP clients (Gtalk uses the XMPP protocol)
[http://xmpp.org/xmpp-software/clients/](http://xmpp.org/xmpp-software/clients/)

And many specific Gtalk applications.
[http://www.google.com/talk/otherclients.html](http://www.google.com/talk/otherclients.html)

You shouldn't try to run the Windows Gtalk client in Ubuntu.

However if you really don't want to try any new software, you could try to run the Gtalk client in Wine.

[http://penguininside.blogspot.nl/2009/05/installing-google-talk-on-wine.html](http://penguininside.blogspot.nl/2009/05/installing-google-talk-on-wine.html)

### Post by khalafuf on 2012-08-22
I think the wine error is because you didn't give the .exe file the permission to be executable.

**Here's the fix:**

1-  Go to where you saved the .exe file.
2-  **Right click** on the .exe file, then choose **properties**.
3-  Open the "**Permission**" tab, and check the box that says any of these words: **{"execute", "is executable", or "allow to run as program"}.**
