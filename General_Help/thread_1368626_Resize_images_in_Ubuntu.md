### Post by vnstudy on 2009-12-30
I have switched to Linux for 2 months. Yes, nothing change since my desktop become freedom. Say goodbye to Windows!

But it is really not easy for me to live with Linux at the honey moon. I have removed all knowledge about Windows and try again with Linux, both click and click install process and Terminal!


 One of the most difficult thing I faced up is that Ubuntu (or Linux in common) has a handy tool to resize image. In contrast, you know, Windows has a huge alternatives.
 But today, if you are a incoming guru have been using Ubuntu for a while and trying to find a handy resizing image tool in Ubuntu, let do the same way like me. It is really easy!


 ***Note 1:*** Please write down firstly, GIMP can not help you!
 All thing you need to do is install a package that attaches to Nautilus. Its name is  [nautilus-image-converter]("http://www.bitron.ch/software/nautilus-image-converter.php").
 1. Open Terminal and enter “*sudo apt-get install nautilus-image-converter*” (without quotes please!).
 2. Log out and then log in again.
 [[IMG]http://vnstudy.org/home/wp-content/uploads/2009/12/anh1.png[/IMG]]("http://vnstudy.org/home/wp-content/uploads/2009/12/anh1.png")


 3. Right click on an image and select “Resize Image” on the context menu:
 A menu will open and you can do what you want! No more EXE nor license! Linux is always free!
 Tell me about that, you guys? Is is really easy!
 ***Please note 2:*** To resize multiple images, just select multiple files and follow the same steps

I hope you are well. Just post on my blog :

[http://vnstudy.org/home/how-to-resize-images-in-ubuntu](http://vnstudy.org/home/how-to-resize-images-in-ubuntu)
