### Post by ad_267 on 2008-06-15
Go to System - Administration - Login Window. Go to the security tab and enable automatic login.

### Post by Rocket2DMn on 2008-06-15
While this is strongly *not recommended* for security reasons, you can go to System->Administration->Login Window, then click the Security tab.  Check the box for Enable Automatic Login and select the username from the dropdown box.
