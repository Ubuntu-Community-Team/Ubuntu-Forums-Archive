### Post by Alisdair Kelly on 2007-08-08
I can report that my problem is now solved. Another user in another venue pointed out to me that in the file view pane of Kb3, you can select VIEW Hidden files to enable Kb3 to see and select them for buring.
I've smacked my forehead :confused: a couple of times since learning this. Apparently this vew hidden files must be selected each time you launch K3b, so I'll have to remember the trick/tip :)
