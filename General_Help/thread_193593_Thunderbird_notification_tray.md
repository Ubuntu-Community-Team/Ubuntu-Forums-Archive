### Post by blackjack6.21.91 on 2006-06-10
Grab [this]("http://moztraybiff.mozdev.org/") extension.  

Install it by going to T00LS --> EXTENSIONS, click Install and find the file you downloaded.

Restart Thunderbird

Go back to TOOLS --> PREFERENCES, right click the extension, select PREFERENCES, and check everything that's there.

### Post by mellowb on 2008-01-11
I tried  the biff extension but preferred the mail-notification package, which can be installed from the universe repository. The main advantage is that it sits on the system tray and doesn't need the Thunderbird or the Firefox to be running. Additionally, it is more configurable and can be used to monitor all your imap, pop3, gmail account, among others.
