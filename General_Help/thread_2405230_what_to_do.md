### Post by l33ch on 2018-11-04
I would list the packages that are missing that you would like to have and just install the packages on there. The only thing Kali is (and any Linux Distro for that matter) is a collection of programs/scripts that make up that particular distobution. The programs/tools are usually the same across flavors.  For example, under Kali's Wiki page is lists some programs/packages available/installed:

Aircrack-ng
Armitage
Burp suite
Cisco Global Exploiter, a hacking tool used to find and exploit vulnerabilities in Cisco Network systems
Ettercap
John the Ripper
Kismet
Maltego
Metasploit framework
Nmap
OWASP ZAP
Social engineering tools.
Wireshark
Hydra
Reverse Engineering tools
Binwalk
Foremost
Volatility

Unless I am mistaken, these are all packages/programs that you should be able to install on any Linux Distribution. So I would look for the ones that you don't have when you had Kali, and locate and install them on that machine--done deal. You should be able to install the majority of them via apt-get. And some I'm sure you can download from the package maintainer and dpkg/untar them.
