### Post by bnshrdr on 2009-04-20
I know this is a long shot but is there a way to determine the options that were used when configuring/making a program?

---

### Post by maheshasolkar on 2009-04-20
I believe it differs from program to program. In Vim, ':version' can give some idea. In Firefox, accessing 'about:buildconfig' via the address bar is helpful.

If you built the program yourself, you can go to the source directory and look in the config.log file. The last ./configure command is usually displayed in there - in the very first few lines.
