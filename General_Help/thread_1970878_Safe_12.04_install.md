### Post by Mark Phelps on 2012-05-03
When you first boot Ubuntu, hold down a SHIFT key to get a GRUB menu.  IF you see an entry their for Older Kernel versions, select that.  If you have a kernel version of 3.2.0-23, select that.

I've found the selecting the newest kernel does not boot on my PC, but selecting an older kernel does.
