### Post by sub2007 on 2009-05-26
Basically, Xfce is lighter and more modular than GNOME and KDE. This means that it uses less memory but also that the Xfce base is much lighter than the other two, including very few bundled applications. There are very few default applications and therefore the idea then is that you  customise the installation based on the packages that you need and not having a lots of packages installed that you don't want.

How to pronounce Xfce and what it means

“Ecks Eff See Eee”. The name Xfce originally stood for XForms Common Environment, but since then, Xfce was rewritten twice and doesn't use XForms toolkit anymore. The name survived, but the F is nolonger capitalized (not “XFce”, but “Xfce”). Currently the acronym doesn't stand for anything (suggestion: X Freakin' Cool Environment).
