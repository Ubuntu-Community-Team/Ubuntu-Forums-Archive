Where should I look to find more information about each update that is released for Ubuntu?

I used to install each and every update that Update Manager told me about, but since the two recent problems (X.org and kernel), I've been more wary. The problem is that I don't know how to find out more information about each update, to gauge whether I should install it or not.

For example, at the moment I have updates waiting to be installed for gzip, libgnutls, kernel and fglrx, but no further info about them.

The changelog in Update Manager is usually blank. Even when it does appear, it's usually very techy and not easy to read and understand. What does:

>       * CVE-2006-4535: Fix for SCTP SO_LINGER DoS.
    - Upstream GIT-SHA: b9ac86727fc02cc7117ef3fe518a4d51cd573c82
  * CVE-2006-4538: ia64 and sparc elf loader memory checking vulnerability.
    - Upstream GIT-SHA: 3a459756810912d2c2bf188cef566af255936b4d

really mean for example?

Security updates have a thread in the [Official Security Announcements]("http://ubuntuforums.org/forumdisplay.php?f=20") part of this forum, and they are usually quite easy to read, but other updates don't seem to have such threads. There's nothing on the homepage of ubuntu.com or in the support section, nothing on planet.ubuntu.com, the Weekly newsletter is not timely enough (I'd rather not wait a week or more).

Dare I say it, but Windows updates give more information. Each one has a couple of lines about it, with a read-more link to a couple of paragraphs, which links to an even longer knowledgbase article. I don't see anything like this with Ubuntu updates.

Am I looking in the wrong places?
