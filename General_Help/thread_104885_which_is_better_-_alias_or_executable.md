### Post by infoseeker on 2005-12-17
There are some commands that I use quite often and I was wondering which was better.....creating an alias in ~/.bashrc (which I have been doing) or creating a text file for each command and make them executable.

An example would be:

> alias mnt1='sudo mount -t ntfs /dev/hda1 /ntfs1'
or
> alias mnt2='sudo mount -t ntfs /dev/hda2 /ntfs2'
or
> alias mid='timidity -iA -B2,8 -Os &'

Etc, Etc........

Thanks

---

### Post by endersshadow on 2005-12-17
An alias would probably be best for single commands.  An executable would be best if you wanted to repeat a sequence of commands.
