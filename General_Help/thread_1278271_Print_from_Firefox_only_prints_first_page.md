### Post by fluffman86 on 2009-09-29
Click File > Print, and near the bottom of the Print window, is the button next to "All Pages" selected?

Also, you can copy/paste just the text you need into Applications > Text Editor or OpenOffice.org.

Or, you can highlight the text/pictures you want and click File > Print, then go to the Options tab, and select the check box next to "Print Selection Only."
