Ubuntu calendar
From Wikipedia, the free encyclopedia

Developed by 	Various
Latest release  	5.03
Platform 	        ubuntu (linux)
Available in      	English
Development status 	discontinued
License 	        Various
Website 	        ubuntu-calendar on Hardy Heron

The package ubuntu-calendar, was described as:
"...a set of wallpapers and themes that are updated every month. The selection of art photography includes some semi-nude models and artistic portrayals of people from diverse ethnic groups...."

When installed, the package used to download monthly wallpaper images. Some of these wallpapers, showing art photography of nude and semi-nude people, have resulted in Ubuntu being humorously referred to as the porn distribution, Linuxxx and other such names. Installing and using ubuntu-calendar was always optional.

The package has been discontinued, probably due to controversy, but it is still available for Ubuntu 8.04, in the universe repositories. [1]
