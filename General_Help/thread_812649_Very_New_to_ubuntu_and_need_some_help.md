### Post by werries on 2008-05-30
yes, if you download the standard Ubuntu cd, it is a LiveCD.
This can be used to install, or to just run as live.
it will not make system changes unless you choose to mount your harddrive and edit stuff yourself.
It is intended to be good to use for new people, it just loads the system onto mostly RAM, leaving your system unchanged when you shutdown.

### Post by sdennie on 2008-05-30
As said above, the LiveCD you are downloading will boot you into Ubuntu without making any changes to your machine (it will be super slow because it's using the CD though).  Another option that is new with 8.04 is, if you burn that LiveCD to a disk and then stick it into a Windows machine, it will allow you to install Ubuntu with Wubi and you won't need to repartition but instead you can install Ubuntu as what appears as a file on your Windows machine and it will ask which you want to boot when you start the machine (I've not tried this but, it's supported).
