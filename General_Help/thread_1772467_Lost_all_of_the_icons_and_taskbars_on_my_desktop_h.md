### Post by Frogs Hair on 2011-05-31
Try using the reset button on the computer next to the power button . When the login screen comes up select recovery console from the session list below the greeter box and login . A terminal will appear on the screen . Enter this code in the terminal and restart. ```
unity --repair
```
