### Post by Telengard C64 on 2011-11-04
Some programs (hopefully very few) require *root* permission to do their jobs. In Linux, *root* is the ultimate power of the system. Programs run with *root permissions* have full access to every file on the system, and the ability to alter or destroy anything.

The message is just to remind you of how to grant *root* power to a program if you absolutely must. Please read the page linked below for a more complete explanation.

[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)
