### Post by karsta62 on 2020-11-02

> That's mp3gain with the gui - anybody been able to do this?

Thanks

Yes, something like this:
sudo add-apt-repository ppa:flexiondotorg/audio (change distro to eoan in the flexiondotorg-ubuntu-audio-focal.list -file)
sudo apt update
sudo apt install mp3gain aacgain
download a gtk version of easymp3gain from [https://sourceforge.net/projects/easymp3gain/files/easymp3gain%20x86_64/](https://sourceforge.net/projects/easymp3gain/files/easymp3gain%20x86_64/)
dpkg -i the_downloaded.deb

If it gives errors, (for me it did) try:
dpkg -i --force-overwrite the_downloaded.deb
