### Post by phantom3113 on 2009-01-07
> **chamber said:**
> Try changing the horizontal virtual size, (think its called that, not at my linux computer) to 4 and see if that helps.

Chamber is correct. The number of desktops option is if you have multiple moniters connected to one computer. Virtual horizontal/vertical desktop size is the number of different desktops. (Vertical will give you multiple rows but doesn't affect the cube) Set Horizontal to 4 and you'll be all set to go.

Plus, if you're bored and have expo enabled, try setting horizontal and vertical desktops each to 10 and then activate expo. It's a wall of 100 desktop screens! :D
