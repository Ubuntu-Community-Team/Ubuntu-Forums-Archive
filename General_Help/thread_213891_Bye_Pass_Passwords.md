### Post by sgbeamer on 2006-07-11
Not in text mode but you can do so in Gnome by going to System | Administration | Login Window  

When this comes up enter your password and go to the Security Tab 

Click the first box which says "Enable Automatic Login" and select your username.  

This will take you straight in without passwords.
