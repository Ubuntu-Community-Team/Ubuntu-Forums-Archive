### Post by Sorivenul on 2008-12-12
If you need to run versions of the Adobe applications that don't run with Wine, there are other options.

1.) Dual-boot
Not my preference, but if some gamers need to do it, some artists may as well. There are many guides on how to do this around the Forums.
2.) Virtualize
Install Windows in VirtualBox, VMware, or QEMU, setup your Adobe tools there, and run Windows as a virtual OS when you need the Adobe tools.

I don't think the Ubuntu developers will devote time to making a creative suite like Adobe provides.  The Wine developers are working hard to add more compatibility and stability to their product, but their work is tedious and time-consuming.  In the meantime, with few or no suitable alternatives, workarounds must be found and accepted.
