### Post by ardchoille42 on 2007-01-12
I was looking around the forums and found this awesome app hidden in the 3rd Party Projects section. I wanted to make others aware of this great project because I feel it is very much needed.

This is Ubuntu System Panel (it's a gnome panel applet) and the project section (including Ubuntu .deb's and plugins) can be found here: [http://ubuntuforums.org/forumdisplay.php?f=156](http://ubuntuforums.org/forumdisplay.php?f=156)

The below screenshot is of USP running in Ubuntu 6.06.1 LTS (Dapper). The screenshot shows the basic installed state of USP. However, I have found that this app is extremely configurable and has a bunch of plugins you can install to enhance the UI and functionality. I have USP running on Dapper with no crashes or problems. I have replaced the panel menu, the panel clock and a bunch of panel applets with USP and I am loving it. I'm hoping that USP will find its way into the Ubuntu repos soon.

Hehe, this is what happens when you explore the forums :)

EDIT: By the way, the screenshot shows just the default look of USP version .41, there's already a newer version being tested which has more features and plugins. You can change the locations of the sections and add plugins to create new sections. The current plugins for USP version .41 can be found in this list: [http://ubuntuforums.org/showthread.php?t=272372](http://ubuntuforums.org/showthread.php?t=272372)
