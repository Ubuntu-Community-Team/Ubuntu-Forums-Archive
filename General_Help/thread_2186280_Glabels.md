### Post by shadowfax1 on 2013-11-06
Found a better way and the quality is stunning because you have more control.  If you don't already have it install 'Gimp'
You have to add the printer plugin called guten-print which can be found in the repos.

Then to make a long story short follow these instruction here..[http://www.creativelabelconcepts.com/gimp/](http://www.creativelabelconcepts.com/gimp/)

One thing he missed in order to save the canvas with an imported image as a jpg you have to export it.
In guten print you'll have to setup your printer and make sure you select CD print.
