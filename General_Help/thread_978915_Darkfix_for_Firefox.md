### Post by manuelg on 2008-11-11
I don't know how to get rid of the darkfix fix, but you could create a new firefox profile and start out clean again.

From a terminal type "firefox -P" and a window will come up to create additional profiles.  You'll loose your bookmarks, so you'll have to copy them manually.

So...backup your bookmarks from your original profile, and then from the new profile import the backup bookmarks file.

Firefox 3: "Bookmarks -> Organize Bookmarks -> Import and Backup -> Import HTML... -> from an HTML File" 

Once you've got everything copied over, you can delete the old profile.
