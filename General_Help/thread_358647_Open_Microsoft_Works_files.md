### Post by dschaller on 2008-05-11
Microsoft Works is a beast. I've done a couple of things when I've really been in a jam with this file format. 1) I've used the zamzar free online conversion utility ([www.zamzar.com](www.zamzar.com)) to convert the .wps into something I can use, or 2) I've found that Microsoft Word Viewer 2003 will run under Wine [Edit: Version 0.9.46]. It will open certain versions of .wps if they're not too old. Then you can copy and paste into another application.

Good luck. I've found MS Works to be a tough format to deal with.
