### Post by seawolf167 on 2011-07-07
Two suggestions (the first of which it sounds like you are already doing):
1. Create a persistent USB Ubuntu Live CD with [URL="http://unetbootin.sourceforge.net/"]UNetbootin
[/URL]2. Use [Remastersys ]("http://www.geekconnection.org/remastersys/")to create a Live CD of a current installation, which would then not be a Live installation cd, but rather a live cd of your installation which wouldn't have any installation prompts
