### Post by Rob Sayer on 2014-03-22
First, there is no more "ubuntu netbook remix" and there hasn't been since 11.04 or so.

Second, I just took a quick look at the specs for the Samsung n130 netbook.  It's a single core cpu with no video 3D acceleration.  Plus, the video chipset is the Intel GMA 950.  Intel video hardware is usually well supported in linux but not always.  Yours is one where it isn't.  My netbook has a newer atom cpu but the situation is similar.

Do not install ubuntu with the Unity desktop.  Or cinnamon or any other heavyweight desktop.  Eg, Unity requires the same hardware performance as Windows 7.  Trust me, you wouldn't be pleased with the results.

On a single core 1Gb machine with that gma adapter the only ubuntu version I could possibly recommend is lubuntu.  You need a lightweight desktop and LXDE is the lightest and fastest.

Don't bother with lubuntu 12.04.  It wasn't a long term support release.  14.04 is going to be an lts though.  I'd install lubuntu 13.10 and upgrade later when 14.04 comes out.  Not necessarily the first day though ...
