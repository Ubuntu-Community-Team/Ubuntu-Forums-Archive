### Post by ajgreeny on 2008-08-17
Get synaptic to the point where you accept the updating, but then go to File >> Save markings.. or Save markings as.. or Save markings as script.. depending on what system you are going to use to download the packages.  If it will be windows, which I suspect, there is little point in saving a script file as windpws will not be able to read it, as far as I know.
