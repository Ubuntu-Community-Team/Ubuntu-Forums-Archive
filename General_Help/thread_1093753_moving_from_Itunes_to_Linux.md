### Post by anlag on 2009-03-11
If you want an iTunes-like music player in Ubuntu, the default Rhythmbox is quite nice. Not sure how iTunes orders music, but you can just copy the files over in whatever structure you like and tell Rhythmbox to use that main music directory as its library. If you want episodes of now-defunct podcasts ordered under the podcasts tab, I'm not sure exactly, but there should be a way to sort that out too.

### Post by mb_webguy on 2009-04-02
Your iTunes songs are mp3 or m4a files, which can be opened by other applications.  Just copy those songs to the Ubuntu machine.  You can even keep them in the directory structure iTunes uses if you want.  Rhythmbox is the default music player on Ubuntu, though I prefer Amarok, which is the default for Kubuntu, but can be easily installed on Ubuntu.

DRM can be removed from the few of your files that have them, though it's frowned upon to discuss specifics here on the Ubuntu forums.  Google is your friend.

Podcasts should likewise be saved as individual files somewhere.  I'm not sure what format they'd be, but it's pretty likely it's a common format like flv.  Miro is a pretty decent application for podcasts.
