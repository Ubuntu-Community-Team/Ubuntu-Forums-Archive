### Post by stephan0h on 2008-04-02
Hi,

There are so many aspects to your posting so I'll answer only the ones where i think i can contribute:

(2) ... regarding moving of the files I think there are 2 options: either via a fast lan connection of by temporarily installing and mounting your old disk in your new computer. I would install the new system, create a new user account and copy all files there and test if it works. maybe use a test user which you can drop if it does not work.

(3)+(4): I don't know if it's better to have one partition or many - it also depends on the file-system that you plan to use. My setup is this: I've got one big partition on my harddisk. I've got a second harddisk residing in a removeable frame which I only use for backups. I do a generational rsync-based backup of my primary disk to this disk - for 200 GB it takes about 20 minutes. After backing up I can remove the disk and put it into a secure place.

(5) have a look at LTSP. but beware: you need a reasonable fast lan.
