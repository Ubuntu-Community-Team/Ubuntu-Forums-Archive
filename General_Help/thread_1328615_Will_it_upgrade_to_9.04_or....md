### Post by hwttdz on 2009-11-16
It will upgrade to the next release 9.04, and you can then upgrade further.  Of course a fresh install is suggested if you can swing it.  Less troubles and whatnot, plus you get the ext4 filesystem which is a big boon.

### Post by qamelian on 2009-11-16
It will upgrade to 9.04 after which you will have the option to upgrade again to 9.10. Only LTS versions allow you to skip releases as the will allow you to upgrade one one LTS to the next LTS, skipping 3 releases.
