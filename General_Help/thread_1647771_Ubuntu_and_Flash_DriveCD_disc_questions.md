### Post by C.S.Cameron on 2010-12-18
A non persistent disk image install is less than 0.75GB and should go onto a 1GB drive just fine.
Use UNetbootin, Universal, (from pendrivelinux.com), or Startup Disk Creator from the Live CD.
Do not use the persistence options.

You can burn the iso image file to CD-R using say, Nero, Disk image.
