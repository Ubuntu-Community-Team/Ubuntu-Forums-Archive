The 15.10 release documentation states that upgrading is possible from 15.04. Does this upgrade also work for upgrading 14.04 to 15.10?

---

### Post by PhilGil on 2015-10-22
No, you can only update from the previous release to the next or from one LTS to the next, so 15.04 to 15.10 or 14.04 to 16.04 (when it is released). If you want to upgrade to 15.10 it would be easier to do a fresh install than upgrade 14.04 to 14.10 to 15.04 to 15.10.
