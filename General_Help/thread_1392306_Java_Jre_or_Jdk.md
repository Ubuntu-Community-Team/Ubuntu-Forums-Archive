### Post by crichard on 2010-01-27
JRE is suitable, Do n't go for IceTea Plugin, there are few ways to install java for your browser, You can try the following code on the terminal

otherwise goto this [site]("http://sites.google.com/site/easylinuxtipsproject/java") to find infos about manual installation of latest JRE.
