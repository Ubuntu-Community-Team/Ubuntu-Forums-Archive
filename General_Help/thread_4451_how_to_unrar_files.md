### Post by mattyh on 2004-11-15
You need to enable multiverse for this one.  That can be done by going into Synaptic.  Then Settings->Repositories.  add the word multiverse after universe in the Sections line for the given repository.  Now, all you need to do is search for rar.  There is both a rar package (for making them) and unrar-nonfree (for unraring).
