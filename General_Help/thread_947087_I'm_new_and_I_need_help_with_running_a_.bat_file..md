### Post by DirtDawg on 2008-10-13
I actually think the software the previous poster was referring to is called **Wine**. To install Wine, go to Applications>Add Remove Programs. Then enter "wine" in the search bar and you should get a match that you can install.

After that, you may be able to right-click the .bat file and choose to run it with Wine. But I'm really not sure it will work in your specific case because of the complexity of running Runescape on a Linux box, but running a Windows specific script.

It's worth a shot, but if you're new to Linux it could get hairy. Good luck.
