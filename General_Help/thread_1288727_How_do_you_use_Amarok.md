### Post by CatKiller on 2009-10-11
System &#8594; Administration &#8594; Software Sources &#8594; Updates. Put a tick in the *backports* box. This is a repository that contains packages meant for the next Ubuntu release that have been made to work with the current release, generally by popular vote.

Refresh the list of available packages in either the Update Manager or Synaptic, and you should get a notification that new versions of some packages are available.

It's possible to do all of this from the command line, but this is probably the easiest way for a new user to follow.

### Post by CatKiller on 2009-10-13
AFAIK, Banshee and Rhythmbox can do whatever GStreamer can do. If you've got the MP3 codecs, then MP3s aren't a problem. Install the *ubuntu-restricted-extras* package. There are instructions all over if you run into trouble.

I can't help you with the iPod. I don't like things that are intentionally crippled, so it's not really a great fit for me.

For ripping, I used to use Sound Juicer (included by default) but now I use K3B. Then I run the ripped tracks through easyMP3gain to add ReplayGain tags.

For my collection, I've used Rhythmbox, Banshee, Listen and Amarok. They're all good, but I settled on Amarok in the end. You might settle on something different.
