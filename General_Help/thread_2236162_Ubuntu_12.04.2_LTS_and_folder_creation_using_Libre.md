### Post by Bucky Ball on 2014-07-24
Open Writer, click 'Save As', click the little folder far top right of that window to create a folder, create one on the desktop, open that folder inside the 'Save As' Window (which should still be open) and save your file.

Go to the desktop and open the new folder. Your file is there?
