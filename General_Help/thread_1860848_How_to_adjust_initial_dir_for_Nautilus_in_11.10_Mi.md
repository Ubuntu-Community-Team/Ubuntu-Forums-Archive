### Post by jackyboy633 on 2011-10-15
pstein, I know how to get the tree view back in Nautilus 3.2 (the version in Ubuntu 11.10. What you have to do is go into the global menu with Nautilus selected, then go to View>Sidebar then click 'Tree'. This should restore the tree menu for you.

This tutorial may help you do the Nautilus launcher for the /tmp folder. Just create a launcher, using the command 'nautilus /tmp' (without the quotes)
