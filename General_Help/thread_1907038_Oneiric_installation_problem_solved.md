### Post by albertsk on 2012-01-10
I just want to share my experience.

For a few days I tried to install Oneiric by upgrading Natty, but got error message indicating:

Failed fetching [http://archive.ubuntu.com/](http://archive.ubuntu.com/) .....  
(cannot remember exactly) 

There were some suggestions but not helpful.  Today I figured out why.  It was because of "Skype".  Message indicates that skype is listed to be removed but it is in the removal black list.  I do not fully  understand this.  So, I simply removed Skype and retried to upgrade. I guess the problem is because my system is amd64, but actually reason is somewhere else. Installation procedure is still on going, but at least started for downloading (about 1.5GB).

Thanks.:D
