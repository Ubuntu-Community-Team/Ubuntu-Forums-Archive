### Post by Shay Stephens on 2006-04-14
I was a happy homesite 4.5 user and had a tough time finding something that was as good in linux.  I have finally explored bluefish enough to have made the switch 100% to bluefish.  It has some pretty cool features that I am now relying on more and more.

### Post by FluffyElmo on 2007-04-25
Try Aptana ([http://aptana.com/]("http://aptana.com/")), I find it to be very similar to Homesite, plus it can be expanded with a range of Eclipse plug-ins which give a lot of flexibility. There are *lots* of excellent code editors for Linux and you'll probably be much happier in the long run making the jump to one of them than trying to work in Wine. 

A couple of other options are:
Screem: apt-get install screem ([http://www.screem.org/]("http://www.screem.org/"))
Komodo Edit (free version of commercial product): [http://www.activestate.com/products/komodo_edit/]("http://www.activestate.com/products/komodo_edit/")


If you search Synaptic or Google you'll find a lot more if these aren't to your taste...
