### Post by GhX6GZMB on 2020-04-28
Changing UUIDs on normal ext4 partitions is easy for me, but changing the UUID of a swap partition gives me problems:

```

macro@macro-pc:~$ sudo swapoff /dev/sda3
password for macro: 
macro@macro-pc:~$ sudo tune2fs /dev/sda3 -U 5a80a205-4577-40e8-8362-51509a8ceb38
tune2fs 1.45.3 (14-Jul-2019)
tune2fs: Bad magic number in super-block while trying to open /dev/sda3
/dev/sda3 contains a swap file system
macro@macro-pc:~$ sudo umount /dev/sda3
umount: /dev/sda3: not mounted.
macro@macro-pc:~$ 

```

Is it even possible?

Thanks.

---

### Post by coffeecat on 2020-04-28
```
sudo mkswap -U <UUID> /dev/sda3
```

Substitute the UUID you want for <UUID>.
