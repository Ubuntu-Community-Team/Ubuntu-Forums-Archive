### Post by Grez on 2014-07-29
Just a quickie. I recently found some old .sdw StarOffice Writer files that I need to open again. The filters have been removed from the latest LibreOffice version so I was left looking at pages of gobbledegook.

The good news is that they still happily open in the Word Processor**AbiWord**. However, this simple solution doesn't seem to be very widely known.

Abiword is available in the Synaptic Package manager or open a terminal and type **sudo apt-get install abiword**

There seem to be lots of workarounds based on installing older versions of Libre/OpenOffice which also appear to have compatibility problems with each other. Abiword has worked for me with no problems so far.
