### Post by Mark Phelps on 2011-11-14
In terms of update drivers, Catalyst 11.10 for Linux is out -- and I read that the version in Ubuntu 11.10 is v11.7 -- and buggy.

So, if you're running Ubuntu 11.10, you can try the latest Catalyst drivers to see if that helps.

I've avoided using the restricted drivers in 11.10 because I read about so many problems with them.  If you don't need 3D acceleration, or other catalyst features, you could try using the open source drivers.  They've improved a lot for Ubuntu 11.10.
