### Post by Vaphell on 2011-03-01
your nokia gets mounted in read only mode
try **gksu nautilus** in terminal to run filebrowser with admin rights (but don't use it for anything else, because you can mess up your home dir permissions and the system as a whole way too easily). Maybe it will allow you to access the device.

Also you can check mountmanager app, i think it allows to configure default settings for storage devices. It should land in administration menu.
