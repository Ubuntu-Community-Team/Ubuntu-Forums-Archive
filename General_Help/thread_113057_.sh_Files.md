### Post by MartinG on 2006-01-07
Looking at your code quotes, the first one suggests you were already running as root when you issued  "xhost +localhost". (NB the root@ and the # at the end of the prompt.)```
root@computername:/home/username/Desktop# xhost +localhost
```This command needs to be run as the logged-in user *before* you switch to root.
