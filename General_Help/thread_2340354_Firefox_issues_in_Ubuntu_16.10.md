### Post by monkeybrain20122 on 2016-10-17
No problem here.Try a different profile. Close Firefox, open the file manager, press ctrl + h or choose show hidden files. Find the hidden folder .mozilla and rename it; now open Firefox and see if the problem persists. If all goes well and you don't need any saved data (such as bookmarks) then just delete the old .mozilla (renamed) and you are good to go.
