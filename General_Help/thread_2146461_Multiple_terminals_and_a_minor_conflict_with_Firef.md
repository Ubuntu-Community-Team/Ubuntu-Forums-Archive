### Post by 2F4U on 2013-05-19
> Barring that, is there any way to change the macro that calls XTerm?

This is probably just a keyboard shortcut, which can be edited/removed.

[https://help.ubuntu.com/community/Lubuntu/Keyboard](https://help.ubuntu.com/community/Lubuntu/Keyboard)

With respect to the different terminals, if you are satisfied with lxterminal you don't need another one. They usually do not depend on each other.
