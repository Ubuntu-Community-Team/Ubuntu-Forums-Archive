Hello to the group

I'm getting Kubuntu Feisty Fawn apps working after the DVD program install. These are the add ons I needed to find and install.

For K3B mp3 plugin, add libk3b2-mp3 from the repositories- I used Adept Manager

For Frostwire, download from web site and install using Ubuntu Package Installer, additionally, install Sun Java 6 runtime from the repositories.

For Amarok, add xine extra plugins from the repositories.

Frostwire, K3B and Amarok now run well.

Hope that this saves some time.
