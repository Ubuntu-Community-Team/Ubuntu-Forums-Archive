### Post by ajgreeny on 2012-05-18
At the grub menu, with the highlight on the line you boot to, hit the "E" ket to edit the stanza for the kernel.  Use cursor keys to go to the kernel line that ends "quiet splash" and delete just those two words from the end of the line; if there is more at the end, leave that as it is, just remove quiet splash.

Next time you boot you will see the scrolling text and it should be possible to see where it all stops.  You can then ask more questions back here.

### Post by ajgreeny on 2012-05-19
If you stick with legacy grub, and I agree with wilee-nilee that grub2 is better, you can still remove the quiet splash permanently by backing up and editing the /boot/grub/menu.lst file using ```
sudo cp /boot/grub/menu.lst /boot/grub/menu.lstbak
gksu gedit /boot/grub/menu.lst
```searching for "quiet splash" and deleting those words only.

Once again it will give you a text scrolling boot, and you can see where the hangup occurs. Once you have hopefully solved the problem you can restore the old menu.lst file to remove the text boot sequence.
