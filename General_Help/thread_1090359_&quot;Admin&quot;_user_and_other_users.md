### Post by Chibone on 2009-03-08
It asks for your password as a safety feature, I wouldn't disable it.

As for creating other users, if you went to System -> Administration -> Users and Groups and made your wife user with the Desktop user profile, she wouldn't be an admin, so she can't do some things.

If there are other things you would like her to do / not do, you will have to configure them by going to System -> Administration -> Users and Groups, select her user name, and click Properties.  Then when the new window comes up, you click the User Privileges tab and adjust as you see fit. I attached an image so you can see what kinds of things you can change.
