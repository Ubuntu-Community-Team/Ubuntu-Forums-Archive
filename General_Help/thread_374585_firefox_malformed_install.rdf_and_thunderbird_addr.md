### Post by dave37 on 2007-03-02
I dual boot XP and Dapper, spending 99% of my time in Dapper.  This evening I had to go in to XP to use an app, and on rebooting to dapper firefox popped up an error message that install.rdf was malformed, I clicked "ok", and my profile was corrupted to some extent, I also could not access my thunderbird address book.  Both profiles reside on a fat32 drive that firefox and thunderbird point to in both xp and dapper.  I ran an fsck, it found errors but wouldn't correct them.  I rebooted to xp and it found errors and "corrected them" but same error popups in windows as well.  Rebooted to windows a second time and all was well, just rebooted in to dapper and no problems.  So the tip seems to be that if like I you share your profiles across 2 operating systems, back the darn things up, and if this happens to you, try (ughhh!) the double boot to windows and hopefully all will be well.
