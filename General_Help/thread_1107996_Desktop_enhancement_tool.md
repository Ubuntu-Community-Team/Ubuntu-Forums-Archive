### Post by stroyan on 2009-03-28
You might have changed the default ubuntu setting so that focus and
scrolling does not change when you hover the mouse over a window.
The "System"->"Preferences"->"Windows menu option opens the dialog with that choice.
It is labeled "Select windows when the mouse moves over them".
