### Post by thil77 on 2009-11-25
tint2 used icons given by the window manager.

in your case, the window manager doesn't have an icon and so tint2 doesn't draw anything.

this is fixed in developpement version of tint2 SVN. tint2 will draw a default icon.

so you can try SVN version or try to see if it's possible to change Dillo default icon ...
