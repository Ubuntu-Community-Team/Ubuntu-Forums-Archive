### Post by Mark76 on 2009-11-26
As we all know the default system menu can get a bit messy when you start adding other desktop environments to your system. So I was wondering if there's a way to have the menu for each DE display only the applications (shared and unique) you want to see in that DE?

---

### Post by tommcd on 2009-11-26
In Gnome you can right-click on Applications, Places, and System on the panel and choose "edit menus". You can then select or de-select those options that you want to see in the menu. 
I don't know whether this will affect the other DEs. I would think it would not, since the configuration for each DE is stored in a different directory in your home directory. There is also probably a way to do this from KDE and XFCE. I don't use those in Ubuntu though, so I am not sure.
