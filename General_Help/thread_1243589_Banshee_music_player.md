My opinion of it? It is normally a bit much but they packaged it well on Ubuntu/Debian. It uses gstreamer, so it has CD ripping/burning and file tagging built in. I would say it is more full featured and usable than Rhythmbox, but it is a bit heavier. It is a really great program, but more than I need. I like it better than Amarok2. Slightly less than Amarok1.
