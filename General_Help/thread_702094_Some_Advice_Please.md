### Post by freelinuxhelp on 2008-02-20
You could use either. Xubuntu has the same core as Ubuntu, but has a different graphical interface (Xfce) and is designed for use on lower end hardware.

For a web server, I would use one of the LTS(Long Term Support) releases and a CMS (Content Management System) such as Drupal or Joomla... Maybe even Wordpress.

Either way, you'll most likely be installing LAMP(Linux, Apache, MySQL, PHP).
The command to install all of that is:
sudo tasksel install lamp-server

Good luck!
