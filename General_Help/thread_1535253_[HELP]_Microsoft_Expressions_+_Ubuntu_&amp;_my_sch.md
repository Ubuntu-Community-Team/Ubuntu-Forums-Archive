### Post by BenAshton24 on 2010-07-21
Do it with another program and then use GIMP to make it look like you used Expressions? (That's what I'd do :) (Also, if it is a class on the abomination that is silverlight, then I would leave))

It doesn't look like you will have any luck using wine based on info from appdb. So if you want to use Expressions (or just aren't that great with GIMP) then I recommend VirtualBox. As for the specs, go with about 800mb of RAM and say 30-40gb of HD space so that you have lots of room for different projects.

EDIT: After re-reading your post it sounds like you are unsure of what virtualbox is, if so, see: [http://www.virtualbox.org/](http://www.virtualbox.org/) The idea is that you install windows on a virtual machine with VirtualBox and then install Expressions on your virtual machine. You could also dual boot ubuntu with windows but this way is more convenient.

Hope this helps
Ben.

[http://appdb.winehq.org/objectManager.php?bIsQueue=false&bIsRejected=false&sClass=application&sTitle=Browse+Applications&iItemsPerPage=25&iPage=1&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?bIsQueue=false&bIsRejected=false&sClass=application&sTitle=Browse+Applications&iItemsPerPage=25&iPage=1&sOrderBy=appName&bAscending=true")
