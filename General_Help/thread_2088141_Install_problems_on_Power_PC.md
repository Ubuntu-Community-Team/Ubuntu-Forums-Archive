### Post by Lars Noodén on 2012-11-25
I'm looking at the Ubuntu downloads and not seeing anything for PowerPC.  Neither the AMD64 or the i386 will work, the G5 is a better chip but you will likely have to turn to Lubuntu to get started.  You can grab the torrent for the PowerPC version of Lubuntu from here:
[http://cdimages.ubuntu.com/lubuntu/releases/quantal/release/](http://cdimages.ubuntu.com/lubuntu/releases/quantal/release/)

If you want to build regular Ubuntu from that, add the package 'ubuntu-desktop'

Note that there can be some problems with the graphics card during installation of 12.10.  If you run into something, don't worry it is solvable.
