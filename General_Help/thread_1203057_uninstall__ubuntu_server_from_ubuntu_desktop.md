### Post by HermanAB on 2009-07-02
Linux is Linux is Linux...

The difference between Ubuntu desktop and server versions is imperceptable.  If you really want to to change from the one to the other, just re-install from scratch, it only takes about 30 minutes, but there isn't much difference.  

The server kernel has a few tweaks to make it work a little better in a batch processing environment, but few people will ever notice.
