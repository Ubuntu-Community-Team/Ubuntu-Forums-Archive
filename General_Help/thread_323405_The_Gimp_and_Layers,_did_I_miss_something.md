### Post by userundefine on 2006-12-22
Layers *do* exist in Gimp.  However, you can't easily manipulate layer properties like you can in Photoshop.  A drawback, IMO, and dropshadows are done in a rather hack-like fashion of a plugin which creates a new layer.  Further, you can't really 'group' Layer A and Layer B (Layer A's dropshadow).

As for WYSIWYG HTML editors... you're right, they're scarce.  But Linux has the best editors for *code* itself.  You should actually learn HTML better anyway and write out the code instead of relying on WYSIWYG, because what you see isn't really ever what you get.

### Post by Shay Stephens on 2006-12-22
