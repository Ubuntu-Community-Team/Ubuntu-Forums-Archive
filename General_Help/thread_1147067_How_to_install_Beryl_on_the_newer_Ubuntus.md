### Post by piratesmack on 2009-05-03
[COLOR="DarkRed"]**STAFF NOTE:** Beryl is **no longer maintained**, and automated installation scripts are not supported. Continue **at your own risk**.[/COLOR]

I would like to share with you something I have been working on.

I have written a bash script that installs Beryl on Ubuntu 7.10 through 9.04.

Download Link: 
[http://files.filefront.com/beryltarbz2/;13699708;/fileinfo.html](http://files.filefront.com/beryltarbz2/;13699708;/fileinfo.html)

Inside the tarball, there are 3 scripts:

1. install.sh:
This script installs Beryl

2. 9.04-fix.sh:
Run this script after install.sh ONLY if you are using Ubuntu 9.04. Otherwise, Beryl Settings won't work

3. uninstall.sh:
This script uninstalls Beryl

Video (Currently still being processed):
[http://www.youtube.com/watch?v=f7LfNTH_5LU](http://www.youtube.com/watch?v=f7LfNTH_5LU)

No need to uninstall Compiz first, they can safely coexist

Enjoy!:-D
