Most websites use their own default fonts. In the "Advanced" dialogue for fonts in Firefox, there is a checkbox that says "Allow pages to choose their own fonts..."

If you remove the check, all websites will use your default font choices.

Also, you need to install Times New Roman to use it.  If you open the software center and search for "mscorefonts," you will find the package.

After it installs, you will have to agree to the licence in order to use the fonts.
~~~~~~~~~~~~

As far as language support, there is a program called "Language Support," where you can add additional language packs and change various settings.  I don't know if this will change the Firefox settings.

Well,I'm not an Arabic speaker (or reader) except for a few, so I'm really not sure.  In addition to what picolo33 has recommended, you could check in the software center under "fonts" for some nice Arabic fonts and set those as the default.

From my experience with Hebrew and Greek (I'm a Bible student), I've often observed that many fonts do not have support for non-Latin/Cyrillic characters, and pull in a set from another font that has them.  I haven't looked at Arabic fonts for a while, but if I recall correctly, I think Liberation Serif had a very attractive Arabic font (coming from someone who doesn't read it).

You might try using an Arabic sample text in a word Processor and changing fonts until you find one that you like and using that in Firefox, or searching the software center for nice Arabic fonts.
