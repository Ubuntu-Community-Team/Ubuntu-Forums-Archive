### Post by jmfal on 2012-07-20
Welcome to Ubuntu Forums

Sound like you need to install restricted-extras, it has support for mp3 and other audio formats.

Open a terminal and run this to install
```
 sudo apt-get install ubuntu-restricted-extras   
```

You can copy/paste this into the terminal.
Press enter 
enter password (it will not enter on screen)
Press enter
restart pc 
watch youtube
