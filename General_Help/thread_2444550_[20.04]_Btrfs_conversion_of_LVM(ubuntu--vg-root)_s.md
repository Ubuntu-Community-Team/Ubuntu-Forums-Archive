```
sudo btrfs-convert /dev/mappper/ubuntu--vg-root
```

sudo mkdir /mnt/hdd
sudo mount /dev/mapper/ubuntu--vg-root /mnt/hdd
```

I create subvolume @ and a @_snapshot_[date] by
```
sudo btrfs subvolume create /mnt/hdd @
sudo btrfs subvolume create /mnt/hdd @_snapshot_[date]
```
Subvolume @ is created and I do:
```
sudo umount /mnt/hdd
sudo mount -o subvol=@ /dev/mapper/ubuntu--vg-root /mnt/hdd
```
And the @ subvolume mounts cleanly and I do 
```
cd /mnt/hdd
```
and can browse thru / filesystem.

for i in dev dev/pts sys proc run; do sudo mount --bind /$i /mnt/hdd/$i; done

sudo nano etc/fstab 
```
and correct etc/fstab entry
/dev/mapper/ubuntu--vg-root / subvol=@,defaults,noatime,nodiratime 0 0
and do
```

grub-install /dev/sda
update-grub
update-initramfs -u
grub-install --recheck /dev/sda
exit
for i in run proc sys dev/pts dev; do sudo umount /mnt/hdd/$i; done
umount /mnt/hdd

```
All commands complete without error.
reboot
