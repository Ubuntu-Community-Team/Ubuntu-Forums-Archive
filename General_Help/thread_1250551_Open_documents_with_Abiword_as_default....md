### Post by scouser73 on 2009-08-26
Say you've got a PDF file and that you have Evince but Adobe Reader is your favourite for viewing the files, you'd right click on the the PDF and it would state **Open With Evince**, well to make sure that it only uses Adobe you would **Right Click** select **Properties** select the **Open With tab**, highlight the program you wish to delete from opening the file and then select **Remove**.

Deleting the program from the Open With section doesn't remove the program from your pc.
