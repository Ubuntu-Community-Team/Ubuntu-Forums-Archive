### Post by egeezer on 2015-09-07
One way is to install Xfce on top of Ubuntu by
```
sudo apt-get install xfce4 xfce4-goodies
```

The -goodies gives you the full Xfce capability but runs on top of Ubuntu with a lot of the specific Ubuntu controls.  Whether Wacom is among those I can't say, but this is more or less the best of both worlds for me (has been up to now, anyway!)
