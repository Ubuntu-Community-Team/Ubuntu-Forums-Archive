### Post by Holger_Gehrke on 2020-07-09
Do you mean "Outlook" the Windows mail application or outlook.com the email-website ? If the former, it should be simple to install the font in windows. After that it should be available to outlook. If the latter, I believe you're out of luck. What fonts you are allowed to use on a website depends on that site, and if you've got a font installed that the editor on the site isn't configured for, you can't use it. Additionally HTML-email only includes a bit of css or - if using HTML 4 - a font-tag and not the font itself. This means that if you use a font the recipient doesn't have, he or she will see some substitute font.
