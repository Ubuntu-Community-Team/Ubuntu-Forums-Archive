### Post by ajgreeny on 2009-06-21
Yes, you can stop processes running by installing and using **bum** (BootUp-manager).  This will allow you to stop various services running at boot up that you don't need; for example I have stopped bluetooth, various laptop utilities as I have a desktop, saned as I have no scanner, dns-clean as I have no modem.  If anything that is stopped is really needed, you can just enable them again using the same system.
