### Post by notwen on 2008-04-24
If you don't need 64 bit, then don't open yourself up to the possible issues it may bring.  64 bit Hardy may require a bit more work to get it working for daily usage as some things just haven't been made for the 64bit just yet and getting the 32bit things working under 64 bit is do-able, but it takes time that could be avoided if you have no need to run the 64bit version.  That said the 64 bit version is ever improving and packages are being made to make it better and more usable for everyday users.
