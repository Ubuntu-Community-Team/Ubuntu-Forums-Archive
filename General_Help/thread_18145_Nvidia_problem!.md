### Post by p!=f on 2005-03-04
"Load glx" is commented in your conf file. Uncomment it and be sure you have nvidia-glx package installed.
You might want to try to backup your current /etc/X11/xorg.conf and to let it recreate to get more clean xorg.conf. :)
```

sudo mv -v /etc/X11/xorg.conf /etc/X11/xorg.conf.old
sudo dpkg-reconfigure xserver-xorg

```
You'll be prompted for several questions.
