### Post by benguin on 2006-01-15
If you have the Limewire Linux .bin file, run it like this in a terminal window:

[change directory to where the file is, for example, if it's on the Desktop directory, do this:]
cd ~/Desktop
[then:]
sudo ./LimewireLinux.bin [or watever the filename is]
[Enter your password and follow the on screen instructions]

Hope that helps.
