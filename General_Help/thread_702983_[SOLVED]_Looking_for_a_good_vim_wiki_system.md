### Post by seventhc on 2008-02-24
After playing around with both of those again I am using the potwiki plugin.
I liked *Wiked*, but potwiki is newer, and has at least one or two features that I liked more. That being the ability to use the <Tab> key to jump to the next [COLOR=Blue]WikiWord[/COLOR], and the ability to press <*Enter*> on a [COLOR=Blue]WikiWord[/COLOR] to jump to that file or create one if it didn't exist, as opposed to a key combination. I'm sure *Wiked* could be tweaked to do the same, but potwiki did it out of the box.
I still have both on my machine, but *potwiki.vim* is the one in my .vim/plugin directory.
Obviously by the responses I got, I am like the only person here using this system lol, but if anyones here does use vim a lot, then it's definitely something worth trying.
That being said I'll go ahead and mark this thread as solved.
If anyone out there reads this at any time and have come across another wiki system for vim, please post it as I'd be interested in trying that also. For now, I am happy with potwiki and this is what I'll use. :)

Cheers everyone.
