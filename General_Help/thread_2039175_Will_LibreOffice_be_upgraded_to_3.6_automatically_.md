### Post by dcstar on 2012-08-08
> **TenLeftFingers said:**
> I know that there are PPA's out there but if it's going to be introduced after release day then I needn't bother.

Ubuntu does **not** upgrade versions of packages after initial release, it only supplies Security Updates for the versions supplied with that Ubuntu release.

You can either add the "Backports" repository and get later versions that are packaged for the subsequent Ubuntu release or add PPAs for unofficial packages not supported by Ubuntu.
