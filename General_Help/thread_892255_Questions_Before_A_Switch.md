### Post by ibutho on 2008-08-17
Yes you can hook up to a Windows network and yes you can play xvids and mpegs if you install the right codecs. You may need to use samba if you want to browse Windows shares and use Windows printers. As for multimedia codecs, look at the page for setting up [restricted formats]("https://wiki.ubuntu.com/RestrictedFormats").
