### Post by grahammechanical on 2012-11-02
First, you need to accept that there are vast differences between Ubuntu 9.04/9.10 and Ubuntu 11.10/12.04/12.10. There are even differences between 12.04 and 12.10. Ubuntu is  moving forward all the time.

Second, If during the installation you ticked the box giving approval to install third party software, then a proprietary video driver has been installed. This means that we no longer go to the Appearance utility to set Enhanced visual effects.

Third, press and hold the Super (windows) key and you will see a list of keyboard short cuts.

Fourth, Open the Ubuntu Software Centre and install CompizConfiguration Settings Manager. You can use that to make adjustments to the desktop effects and you can also make changes to the way that Unity works by using the Ubuntu Unity Plugin.

But you should also read this:

[http://smspillaz.wordpress.com/2012/09/07/maintainers-for-the-unsupported-plugins/]("http://smspillaz.wordpress.com/2012/09/07/maintainers-for-the-unsupported-plugins/")

It explains why special effects are now limited in 12.10.

Regards

P.S. Please do not shout.
