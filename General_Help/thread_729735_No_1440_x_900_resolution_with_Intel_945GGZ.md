### Post by ryanhaigh on 2008-03-21
You will need add "1400x900@75" to the list of available modes (at the start before 1024) as well as a corresponding modeline which you can generate here: [http://xtiming.sourceforge.net/cgi-bin/xtiming.pl](http://xtiming.sourceforge.net/cgi-bin/xtiming.pl)

You may also be able to use ```
 sudo dpkg --reconfigure xserver-xorg 
```
