### Post by rocuan on 2011-04-21
I tried outrec too but hated the interface.
Ubuntu has a built in sound recorder that lets you save streams to mp3, ogg etc.

Basics : [http://www.addictivetips.com/ubuntu-linux-tips/use-sound-recorder-to-record-and-play-sounds-in-ubuntu/](http://www.addictivetips.com/ubuntu-linux-tips/use-sound-recorder-to-record-and-play-sounds-in-ubuntu/)
The link is a little dated. Current versions of sound recorder allow saving as mp3

If you have trouble tweaking the sound recorder, follow this thread : [http://ubuntuforums.org/showthread.php?t=1568404](http://ubuntuforums.org/showthread.php?t=1568404)

I tied my sound recorder to a keyboard shortcut ( Ctrl + * )
So I can rip tracks on the fly when jamming amarok's trance stations 
> 
- Main Menu->System->Preferences->Keyboard Shortcuts
  - click add
- give it a name : "Sound Recorder"
- give it the command : "gnome-sound-recorder"
- give it a memorable hot-key combo( i use Ctrl + * )
