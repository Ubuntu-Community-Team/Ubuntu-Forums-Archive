Then go to Startup and add the command ```
gaim
```

### Post by Garyu on 2007-02-03
In the main menu (at the top) go through System -> Settings -> Session (or startup? don't remember for sure)

At the Sessions settings you can add any applications you want to launch when the X server (window manager) is started.
