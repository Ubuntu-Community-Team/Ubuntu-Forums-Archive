### Post by Reaved on 2008-11-27
I have this issue as well.  I've read that others fixed it by backing up bookmarks, deleting and then creating a new profile.

This issue has existed since the 2.x versions of Firefox and I am sure that it is a problem with the profile since I have brought it through OS upgrades.

**Update:** 
I found a discussion where a user suggested removing the All-in-one Gestures extension as it breaks the context menu/spelling suggestions/add to dictionary features. Upon disabling the extension I now have spelling suggestions back.

### Post by booksnmore4you on 2009-07-19
I'm running Mint, which is built on Ubuntu, as you probably know.  

I fixed the issue - at least for my machine.

In my case, it was the [3D extension FoxTab]("https://addons.mozilla.org/en-US/firefox/addon/8879").  It was altogether removing the spelling correction options from the right-click menu.  Once I disabled it, they were restored.
