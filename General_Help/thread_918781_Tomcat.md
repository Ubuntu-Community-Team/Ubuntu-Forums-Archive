### Post by northern lights on 2008-09-13
The Hardy repositories include tomcat5.5

If that is sufficient,```
sudo apt-get update && sudo apt-get install tomcat5.5 tomcat5.5-admin tomcat5.5-webapps
```will set it up. If you need 6, you'd either want to download the binary installer from the tar.gz archive or it can alternatively be installed from the intrepid repos also.
