### Post by 3rdalbum on 2011-10-20
Ubuntu 11.04 has slightly different packages to 11.10. When you upgrade to 11.10, some old packages that are no longer needed are removed. This is very normal.

What packages does it tell you are going to be removed? If it's only a short list, it's a normal process. If it's a large list, then there's something wrong.
