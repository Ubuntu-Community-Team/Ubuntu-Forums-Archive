### Post by henryjm206 on 2008-09-15
Do you have the advanced desktop effects settings installed?? if not go to the help file and click on desktop effects then follow the instructions.
once installed go to system-> preferences-> advanced desktop effects settings then go to the general options and select the desktop size tab then slide the horizontal desktops size to four then enable the desktop cube by clicking on desktop cube and selecting the box enable in the bottom left corner you can also change the destop cube settings from that menu

good luck
