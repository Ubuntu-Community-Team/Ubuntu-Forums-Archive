### Post by WlaadDyaab on 2011-03-17
> **Hero1969 said:**
> I had downloaded a picture from the web and set it as my wallpaper.  I am browsing a an old backup image I had done looking for this picture.  Where would it be located?  It is not /usr/share/backgrounds.  Any ideas where ubuntu stores these?

Right-click > Change Desktop Background

Is it not in the "Appearance Preferences" box?

If not

Try this:

Places > Search for Files...

Then type the name of the image in the search bar > Find

Or is it still lost?

Also try looking in the home folder:

Places > Home Folder

I normally find lots of files there that for some reason, they don't go to the "Downloads" folder
