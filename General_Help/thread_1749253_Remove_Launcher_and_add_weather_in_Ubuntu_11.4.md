### Post by piratemari on 2011-05-04
If you would like to remove the Unity panel, log out of your session, and on the  login screen, look at the bottom to the far right. There is an option there which lets you pick which Ubuntu you want, select Ubuntu Classic. You can re-install weather using the package manager, search for "weather" and install the program that pops up.
