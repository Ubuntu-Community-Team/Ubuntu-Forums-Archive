### Post by ian-weisser on 2015-04-18
You don't a problem with packages that need to be removed.

You have a problem with one file that you made:
/etc/init.d/make-ir-freaking-work lacks the expected LSB header, and creates a circular service dependency.
Simply edit the file.
 Add an LSB header that looks like other files in the directory.
Remove the 'ondemand' dependency, or replace it with something less disruptive.

Then try upgrading again.
