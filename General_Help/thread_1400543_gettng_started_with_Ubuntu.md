### Post by thecliff on 2010-02-07
Yes, this is one option.  However, I feel like we should really examine why you would want to use Ubuntu for online banking and web browsing after reading an online article but not want to use it for all other day-to-day computing tasks.  

To try Ubuntu without installing or making any changes to your PC whatsoever:


[LIST]
[*]Go to [http://www.ubuntu.com/GetUbuntu/download](http://www.ubuntu.com/GetUbuntu/download)
[*]Choose 32 bit or 64 bit (if you don't know the difference, select 32 bit)
[*]Download the ISO file
[*]Burn the ISO file to a disk
[LIST]
[*]This can be done easily in Windows 7 by right-clicking and selecting Burn to Disc
[*]Alternatively, you can download Nero or ISO Burner
[/LIST]
[*]Insert the disc into your disc drive and reboot your PC
[*]On the Ubuntu screen, choose to run LIVE CD instead of installing
[*]Just restart the PC and eject the CD as soon as your PC powers on to boot in Windows again
[/LIST]

You can easily run Ubuntu side by side on the same hard drive as Windows in most cases.  If you like what you see on the live disc, I recommend you set this up and try Ubuntu for every day computing.  

Oh, and Firefox is the default browser in Ubuntu.

Others failed to mention -

Ubuntu comes with firefox so you can be up and running on the internet in no time.

Your bootable disk should also come with Open Office, which is able to read and safe in Microsoft formats.

So if you're interested in using a bootable disk for ultra safe browsing, you'll have pretty much full functionality.

As the other guys said, you can even stuff onto a usb drive.

If I were you, I would just install Ubuntu!  Don't worry, you can keep Windows installed on your computer and duel boot.  All you'll have to do when you start up your computer is select which OS you want to use.  ==> maybe do this after you play with the live CD for awhile - that's what it's there for.
