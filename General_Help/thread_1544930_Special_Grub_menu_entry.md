### Post by ajgreeny on 2010-08-03
I am not sure that grub will help you with that.

I think you will need a completely separate install, or another user on your current install to whom you set up an automatic login and then use a shell script in your startup applications to put the system into standby/suspend.

Grub simply sets the kernel you boot, and can not affect the subsequent actions of that OS as far as I'm aware.  Also I have no idea how you can put the system into suspend mode by using a script, but no doubt others will know how to do that, or a forum search will tell you.
