### Post by slickymaster on 2013-03-25
[Quanta Plus]("http://quanta.kdewebdev.org/") - Quanta Plus's features include multi-document interface, WYSIWYG editing and templates. Among the advanced features your will find team development, plug-in support and a PHP debugger.

[Aptana]("http://www.aptana.com/") - Aptana is an html/javascript editor, however, it does not provide any WYSIWYG feature. It provides "Code Assist" (code completion) which is context related help showing possible java-script functions and parameters, html and css tags. Besides being available for all major platforms, Aptana can also be installed as a plug-in to eclipse.
