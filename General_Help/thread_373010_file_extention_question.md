### Post by CocoAUS on 2007-02-28
Linux file extensions are pretty few, as far as most users are concerned.

.tar is a compressed file
.tar.bz2 and
.tar.gz are bother further compressions
.deb is a Debian package
.rpm is a Redhat Package Manager package
.ogg is a media wrapper for sound and video (like .avi or .mpg)

Uh...that's pretty much what most people will run into.  One interesting note is that Linux generally doesn't care what something has as an extension, except some programs like media players.  Lots of extensions are just categories or descriptions, rather than filetypes, such as is found in /etc/apt/sources.list

Also check out this wiki list: [http://en.wikipedia.org/wiki/List_of_archive_formats](http://en.wikipedia.org/wiki/List_of_archive_formats)
