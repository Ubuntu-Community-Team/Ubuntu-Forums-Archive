Yes, there is.
Just go to [http://get.adobe.com/flashplayer/](http://get.adobe.com/flashplayer/)
It should automatically detect your OS. 
In the field below that information, choose the ".deb package for Ubuntu 8.04+" or something like that. 
After, a big yellow button should show below that and it will be for downloading the .deb package.
.deb packages are like install files for Windows. Easy to use! I reccomend you try to download all your program in .deb files atleast until you gain some expirience. 

Install the .deb package and you're ready to go.
