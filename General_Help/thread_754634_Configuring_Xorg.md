### Post by warp99 on 2008-04-14
Set a modeline string in your monitor section. Open a terminal and issue the following:

```
cvt 1280 1024
```
you get an output similar to this:
```
# 1280x1024 59.89 Hz (CVT 1.31M4) hsync: 63.67 kHz; pclk: 109.00 MHz
Modeline "1280x1024_60.00"  109.00  1280 1368 1496 1712  1024 1027 1034 1063 -hsync +vsync

```
then add the modeline to your monitor sections:
```
Section "Monitor"
	Identifier	"Configured Monitor"
	Vendorname	"Envision"
	Modelname	"EFT 720"
	Horizsync	30-72
	Vertrefresh	50-130
        [b]Modeline     "1280x1024_60.00"  109.00  1280 1368 1496 1712  1024 1027 1034 1063 -hsync +vsync
	Gamma	1.0[/b]
EndSection
```
then the resolution to your screen section:
```
Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
	Defaultdepth	24
	SubSection "Display"
**#       	Viewport	0	0**
		Depth	24
**		Modes		"1280x1024_60.00"**
	EndSubSection
EndSection
```

Changes are printed in bold. Unless you're using dual monitors you should comment out 'Viewport 0 0". You can also calculate the modeline with a different refresh rate, type 'cvt --help' for more information.

Edit:
If you want 1400x1280 make your modeline based on that resolution.
