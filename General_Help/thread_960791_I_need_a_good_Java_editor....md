### Post by jespdj on 2008-10-28
> **DivinityX said:**
> I'm learning Java through Jcreator, i've had a taste of Netbeans at school and i thought it was pretty awesome but it didn't seem to be up to date with java as it said some things were errors that weren't on Jcreator.
You could just [download the latest version of NetBeans](http://www.netbeans.org).

Also make sure that you are using Sun Java 6:
```
sudo apt-get install sun-java6-jdk
sudo update-alternatives --config java
```
Select Sun Java 6 in the menu that the second command presents.
