That is an autopackage which is a script that needs to be run, by default in Ubuntu such files cannot be run by clicking on.
In any event Xara Xtreme appears to be 7-8 years old & likely worthless. I'd suggest installing stuff from Ubuntu software center rather than searching the web for stuff you currently know nothing about.

If what's in software-center doesn't suit most of your needs than likely Ubuntu isn't for you
