### Post by Trebaruna on 2009-03-11
Double the RAM size is not really a very good measure. For instance, my laptop has 3GB of ram, and runs fine without any swap whatsoever. According to the rule, I should allocate 6GB of diskspace to swap. An operating system needs a certain amount of memory to run itself and apps smoothly. That amount does not suddenly increase when your physical memory does.

Anyway, you need at least 1GB to write your RAM to disk, of course. Considering you've got a netbook I don't think you'd need a lot of space, because you are not likely to do memory-intense things like GIMP or virtualizing Windows Vista. If you allocate 2GB you should be just fine, methinks.
