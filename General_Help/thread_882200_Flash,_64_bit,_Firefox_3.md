### Post by macaholic on 2008-08-06
The issue is Adobe doesn't make a native 64-bit flash plugin, so the plugin has to be "wrapped" in order to work on a 64-bit architecture. Unfortunately there is not a good answer to this predicament. I personally use the latest flash 10 beta, however I have experienced a boatload of problems with that aswell. For the time being, you can either switch to a free alternative, which will lose functionality but gain stability, or wait and contact Adobe and let them know you want 64-bit flash.
P.S. There is an entire 64-bit forum [here]("http://ubuntuforums.org/forumdisplay.php?f=343").
