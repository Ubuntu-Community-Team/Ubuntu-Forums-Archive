### Post by mjpatey on 2008-02-28
Sorry, off-topic-ish... I just found this thread searching for the terms RAW and Ubuntu on Google.  I'd tried ufraw, rawstudio, LightZone and others, but they all were either too artistically unintuitive (the first 2) or just took me too long to finish working on a photo (LightZone).  I've also used the built-in abilities of Picasa, but they're a little limited for RAW work.

I just installed Raw Therapee last night, and it's the best tool I've used so far.  It hits a sweet spot between ease of use and level of control.  Somehow, though the UI gives you SO many options, it never feels like too much, and I've enjoyed using it.  Each control does what it feels like it should, unlike some of the other programs.  I'm not an expert at dealing with RAW images, and RT makes it powerfully accessible for someone like me.

As far as installation, another poster here has it right: just unpack the .tgz file and run the program by double-clicking the file named "RT".

Sorry I already sound like such a raving fan, but if anyone is in my situation, having tried the more commonly-known options out there for RAW in Ubuntu, I can't recommend Raw Therapee enough!

Disclaimer:  Though it can be had for free, Raw Therapee is not FOSS (Free Open Source Software) as I understand it.  So if you're wanting a purely Open Source solution for RAW editing, I feel you're best served by UFRAW.  Editing with curves is too touchy for me, which is why I have trouble with UFRAW, but other than that it's the best out there.
