### Post by TVTrukChik on 2008-12-18
OK, this is weird... I had the same thing happen, too.  Spent a few minutes Googling and being annoyed.  And then I got a little popup notification that there were updates for some of my add-ons.  So I installed them.  Then Firefox wanted to restart again.  

After this last restart, the nagging yellow bar at the top of the screen is gone.  So, maybe try searching for updates for some of your existing add-ons, and see if updating those gets rid of the nag bar?

Oh, I don't know if it matters, but the add-ons that wanted to update were Adblock Plus, ScribeFire, and a couple of themes... Noia 2.0 and Gnome Classic.
