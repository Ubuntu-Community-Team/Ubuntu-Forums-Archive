### Post by rphilli5 on 2007-10-25
My connection to gotomypc is incredibly slow.  It works great from my windows machines, but connecting from the 7.10 box to a windows machine using the universal viewer  is unusablely slow.  I have the latest java version installed.  I have read other reviews that the gotompc service is relatively fast for other linux and Mac users.  Any ideas?

Resolution:  gotomypc defaults to apply screen scaling.  Going from 2 monitors to a decreased resoltion on 1 monitor was the issue.  Enabling "actual size" allowed normal browsing of the remote machine.
