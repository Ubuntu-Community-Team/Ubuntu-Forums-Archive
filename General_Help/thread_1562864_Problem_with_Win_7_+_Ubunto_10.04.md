### Post by johnhenry on 2010-08-29
I suggest you try pasting the following into your grub.cfg file.

menuentry "Windows 7 (loader) (on /dev/sda1)" {
	insmod ntfs
	set root='(hd0,1)'
	search --no-floppy --fs-uuid --set 40b46c97b46c9168
	chainloader +1
}

### Post by Mark Phelps on 2010-08-29
Suggest you try johnhenry's suggestion -- just to see if that works.

While it's generally a bad idea to edit grub.cfg, in this case, it's worth experimenting to see if that code will work for you.

Realize, though, that grub.cfg gets automatically overwritten when kernel updates are applied, or "update-grub" is run.  So, manually editing it is not a way to do persistent changes.
