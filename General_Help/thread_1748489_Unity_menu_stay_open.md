### Post by vanadium on 2011-07-11
Unity launchbar will hide only if an application is maximized or if a window is placed over it. You can change that behaviour using Compiz Config manager. Default is "Dodge windows". There is also "Dodge active window", where it will only be hidden if covered by an active window. "Never" and "Autohide" are two more options.
