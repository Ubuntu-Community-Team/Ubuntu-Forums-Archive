### Post by ajgreeny on 2017-01-20
Intel is easier to deal with at the moment due to some AMD driver issues which hopefully will be overcome soon.

Is the cpu of that machine Intel, and does it use integrated graphics only?
If so I suspect it will be fine for you.

Try Ubuntu with a live USB first to see if it runs well enough; if it it too slow, (but remember a live system will run slower than an installed OS), try one of the alternative DE versions, eg Xubuntu, Ubuntu-Mate, etc etc, all of which use fewer resources.
