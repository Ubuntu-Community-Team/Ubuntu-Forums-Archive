### Post by fuzetsu490 on 2010-08-01
I got it set up like this: go to language support (under administration) install the options you need for japanese (maybe only input method if thats all you need) switch input method system to ibus, go to ibus preferences (under preferences) and add anthy (m17n) as an input method. This will let you type with romaji input.
