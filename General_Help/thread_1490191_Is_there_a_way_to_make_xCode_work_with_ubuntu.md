### Post by srs5694 on 2010-05-22
Mac OS X is not Linux-based, although it is Unix-based. AFAIK, there's no way to get Apple's XCode to work under Linux. That said, XCode is built atop GCC, which is the standard development software for Linux; however, if there's a way to use GCC or other Linux tools to write iPhone applications, I don't know what it is.
