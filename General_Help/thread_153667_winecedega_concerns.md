### Post by hollywoodb on 2006-04-01
I recommend trying wine first. Personally I use both, but Cedega seems to be interested in supporting the "hot new game" which sometimes breaks compatibility with older games.  The Wine project is attempting to reach full compatibility which means it doesn't include hotfixes for the "hot new game", but will probably run more applications than Cedega.  If your games are really old, like DOS games, try dosbox or similar as well.  I still play the original Zork 1, 2, and 3 in dosbox from time to time.
