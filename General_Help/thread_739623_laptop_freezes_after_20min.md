### Post by gmjs on 2008-03-29
Hi,

Tricky one with it being a laptop.  Does it lock up after exactly 20 minutes each time?  Does it always take the full 20 minutes after restarting the machine **just after** it has frozen?

If the time is always close or exactly 20 minutes **every** time, check your screensaver to make sure it runs correctly; turning it off if necessary and testing again for another 20 minutes.

If the time varies slightly and is considerably shorter after a reboot straight away, I would guess that the problem may be overheating (possibly due to a fan not being connected correctly when the repair was carried out).  This might need further servicing (I'd never recommend taking a laptop to bits)!

Other than that, maybe you could reseat the memory (there is usaully an obvious door underneath the laptop that can be opened).  Carefully take the memory out and push it back and see if it makes a difference.

This is very much guess work, but I'd be suprised if it was a software error as this probably hasn't changed and I don't think whoever did the repair work would have, for example, installed updated video drivers or anything that could cause a problem like this.

Hope this helps.  Good luck.

Graham
