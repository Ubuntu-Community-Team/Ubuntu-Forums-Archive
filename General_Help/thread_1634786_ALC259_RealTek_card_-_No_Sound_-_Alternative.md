### Post by marcb_ro on 2010-12-23
While trying to fix bugs on my Asus U52F (the no-microphone and speakers-enabled-with-headphones-plugged-in bugs) I accidentally tried to install a Connexant HDA driver that left me without sound on my laptop. I then reinstalled the Linux kernel, via
```
sudo apt-get install --reinstall linux-image-2.6.35-24-generic
```

I then added the following line to /etc/modprobe.d/alsa-base.conf
```
options snd-hda-intel model=auto
```

Reboot and that should be it.
