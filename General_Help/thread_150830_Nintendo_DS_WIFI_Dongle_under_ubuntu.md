### Post by AndrewCaul on 2006-03-29
It's a "[dedicated wireless access point that plugs into a USB port on your PC.]("http://store.nintendo.com/webapp/wcs/stores/servlet/ProductDisplay?currency=USD&storeId=10001&jspStoreDir=NOASTORE&productId=95704&categoryId=11157&langId=-1&catalogId=10001&ddkey=SetCurrencyPreference")"
You can see more info at [http://www.nintendowifi.com/consumerservice/gettingStarted.do?entryId=usbSetup]("http://www.nintendowifi.com/consumerservice/gettingStarted.do?entryId=usbSetup").
It looks like it has a Windows installer and interface for setting up the Nintendo Wi-Fi Connection. You might be able to get it working under Wine, but it might be easier to get a wireless router.
