### Post by TheFu on 2022-01-09
Sometimes I do.
Almost always it is because I forgot to reboot after a kernel update 3 weeks ago or there is a new FF version that was installed last patch session and I haven't restarted.

The typical FF troubleshooting method is to start it in "safe mode" and see if the issue continues.  If it doesn't, then look to an out of date addon. Usually, within a week the addon gets updated and all is well. In the meantime, use Private Windows or run firefox in a **firejail --private** session.
