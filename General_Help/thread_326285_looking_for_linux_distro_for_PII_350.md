### Post by houstonbofh on 2006-12-27
If you have 128 meg of ram, and a good swap file, Ubuntu will run on that system.  It will not be speedy.  It will also need to be installed from the text installer, as the live cd need much more ram.  Xubuntu would be quicker, but is not as new user friendly as Gnome.  If at all possible, upgrade the ram, and you will notice a world of difference.  I consider 256 a minimum, and 512 quite nice.

### Post by Mr.Auer on 2006-12-29
I run Xubuntu 6.06 on my old HP laptop which is a 400 mhz Pentium 2 i think, with 192 megs of memory. Its very usable, i can watch videos / music too, in browsing use its fast enough. I tried Ubuntu with gnome on it first, that was way too slow.

Also a good option for an old machine is Damn Small Linux. That distro is only about 50 megs in size, will run on any antiquated PC and its fast also. Its got a music player, web browser, email, office programs etc. and u can add programs too. Its not nearly as easy to use or pretty as Ubuntu/Xubuntu, but it works well and is screaming fast on old hardware. So you could at least try it to see... Also you can get additional program packages for DSL to add after installation to HD. Installation following the guide on their pages was also very quick and easy. ([www.damnsmalllinux.org](www.damnsmalllinux.org) )
 If modern looks and medium speed is ok, then its Xubuntu etc.

Another small distro is Puppy Linux, i havent used it myself so cant say.
