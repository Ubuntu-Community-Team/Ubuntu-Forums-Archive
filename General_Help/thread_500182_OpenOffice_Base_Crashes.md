### Post by Huw on 2007-07-14
When using Base from the Ubuntu repos I have encountered a handful of bugs, most notably the show-stopping crash when trying to use tables.

What I did was removed it from my computer and installed from the binaries provided by the openoffice website.  It's been a lot better since then.  You have to use alien to convert the RPMs to debs but then you can just use dpkg to install very easily.
