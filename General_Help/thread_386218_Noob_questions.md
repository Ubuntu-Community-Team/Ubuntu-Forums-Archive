### Post by teaker1s on 2007-03-16
[http://www.ubuntuforums.org/showthread.php?t=263851]("http://www.ubuntuforums.org/showthread.php?t=263851")

you don't say your hardware, but above link is for nvidia cards. [B]be aware you will be playing with beta software
[/B]
firefox has a bin file in folder-clicking it should make it run providing permission to execute is allowed(right click it and select properties, tick execute) you can then create a launcher or link to fire it up

you will want to use either automatix2 or easyubuntu scripts to install codecs, unless you fancy a look here
[https://help.ubuntu.com/community/RestrictedFormats?highlight=%28formats%29%7C%28restricted%29]("https://help.ubuntu.com/community/RestrictedFormats?highlight=%28formats%29%7C%28restricted%29")

synaptic-select preferences>general tab
tick show package properties in main window

install "alacarte" this allows you to tick what menu's display-when installed menu editor will appear either in system administration or preferences.

synaptic or terminal will install-either way

sounds like you have a DNS resolution issue for slow resolving of pages, ether set up a static ip and DNS or 
[http://www.ubuntuforums.org/showthread.php?t=282034&highlight=stabilise+dns]("http://www.ubuntuforums.org/showthread.php?t=282034&highlight=stabilise+dns")
