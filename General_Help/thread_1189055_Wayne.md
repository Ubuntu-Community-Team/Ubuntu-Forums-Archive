### Post by zipperback on 2009-06-16
> **WAYNE MOORE said:**
> All my downloaded programs are located in the tmp folder.
How do i run these programs, or what do i do with them?

Your best option for installing and running applications is to install the software from one of the available repositories. There are a few options available to you with Ubuntu.

For example, click on "Applications" on your Gnome desktop menu bar, and then select "Add/Remove", this will then provide you with a pretty good selection of available applications. Select the ones you would like, then click "Apply Changes" and they will be installed for you. You should then be able to access the application by selecting it from appropriate Applications sub menu. 

There is an additional application that is available which is similar the "Add/Remove" application, and it is called "Synaptic" and it can be accessed by "System" -> "Administration" -> "Synaptic Package Manager".

In general, when you want to install or remove software using the "Add/Remove" application, or via the "Synaptic Package Manager" you will be prompted for your system password. This will tell the system that you are authorizing the system changes to take place, and the system will check to make sure that your account is authorized to make the changes.

There are additional methods which require that you are comfortable with compiling software from source code, and you are also somewhat comfortable with using the command line for system level modifications. However, I would suggest that you familiarize your self with the two above methods of software package management before moving on to compiling software from source.

If you need additional assistance, please feel free to post a message here and either myself or someone else from the community will probably be able to assist you.

When posting a request for assistance however, please be sure that you have used the search function on the ubuntuforums.org website to check for similar problems and solutions which may be of assistance to you.

In general when asking for assistance you should probably consider the following items which I've copied from the "How to ask smart questions FAQ."

[http://www.catb.org/~esr/faqs/smart-questions.html](http://www.catb.org/~esr/faqs/smart-questions.html)

   1.  Try to find an answer by searching the archives of the forum you plan to post to.

   2.  Try to find an answer by searching the Web.

   3.  Try to find an answer by reading the manual.

   4.  Try to find an answer by reading a FAQ.

   5.  Try to find an answer by inspection or experimentation.
   
   6.  Try to find an answer by asking a skilled friend.

   7.  If you're a programmer, try to find an answer by reading the source code.

Hopefully you will find the information above useful.
