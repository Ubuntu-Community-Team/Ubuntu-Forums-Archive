### Post by notgary on 2011-11-22
There are a few other editions of Ubuntu kicking about with different interfaces. There's [Xubuntu]("http://www.xubuntu.org/"), which is running a really lightweight interface called XFCE which is a fast and lightweight interface, very similar in appearance to the old Gnome 2 interface, and there's [Lubuntu]("http://http://lubuntu.net/"), which also has a lightweight interface, and lies about half way between Ubuntu and Xubuntu in terms of resource usage. Both of these are basically just the normal Ubuntu but with a different skin, so you'll be able to get all the same apps from the software centre.

If your problems aren't too serious, then maybe you'd consider hanging about with normal Ubuntu until April when the next release is supposed to have much better [support for multiple monitors]("http://http://www.omgubuntu.co.uk/2011/11/multi-monitor-support-to-improve-in-ubuntu-12-04-video/").
