### Post by Satoru-san on 2010-02-11
try this...

copy the text in firefox with control + c

then go back to vim

then do control + shift + v


you can also select the text in firefox and in vim middle click, if you dont have a middle button then you can click left and right at the same time.

This isnt really for all variants. the second method will work on all variants the first one only works on gnome and kde.

Please mark this as solved.

{just to help, when you want to save hit [escape] then type :wq}
{after you save and you want to edit again hit [i] and start typing}
