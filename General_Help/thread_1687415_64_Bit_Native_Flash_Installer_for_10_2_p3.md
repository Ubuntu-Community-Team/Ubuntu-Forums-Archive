### Post by blueridgedog on 2011-02-13
Adobe has a new native 64 bit flash out for Linux.  I have updated Romeo-Adrian Cioaba's script to download the current update, 10_2_p3.

Thanks again to Romeo for his work in automating the use of the Adobe version of Flash for 64 bit Linux systems.

To run:

Download then cd to location where you saved the file (typically ~/Downloads) then:

```
chmod 700 native-64bit-flash-installer10_2_p3.sh
./native-64bit-flash-installer10_2_p3.sh
```
