### Post by nothingspecial on 2009-08-25
Are you using an administrative account. If not you won`t be able to use them.

Use the account you set up when you installed. If that doesn`t work then you have a problem in that you don`t have an admin account.

Restart and login to recovery mode (or whatever it`s called).

Type 
```
adduser *[COLOR="Red"]username[/COLOR]* admin
```

Substitute username for your username.

Restart.
