### Post by yancek on 2015-09-06
On the Installation Type screen, you will have an option Mount point and a drop down box with a number of selections available.  The first one is a forward slash ( / ) which is the symbol for root.  If you are a new user, best not to use a separate boot partition.  The root ( / ) is all you need but should leave some space for a swap partition.  Read through the tutrorial below to get some detailed info.
