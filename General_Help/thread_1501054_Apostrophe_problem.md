### Post by cbecker78 on 2010-06-04
sorry I don't have a lot to offer here, other than I have had the same problem when I had the USintl (US nternational) keyboard layout instead of US.  " + space gives one type of double quotes, hitting the quote key twice gives a different type of quotes.  This is designed to provide the different types of accents for a standard US keyboard layout.  For example, if I type "~+space" i get the "~" symbol, but if it type "~+n" I would get an "n" with the "~" over top of it (i.e., the spanish letter that sounds like "en-yeah").

This [wiki article]("http://en.wikipedia.org/wiki/Keyboard_layout#US-International") give a better explanation of what is going on, but it sounds like you already get that.  your best bet, I think, is to try selecting a different keyboard layout to solve it.
