### Post by z_diver on 2006-04-14
There are so many ways to skin this cat.  In Gnome I prefer to right click the file and seclect "Write to Disk".  Simple as can be and works to burn both CDs and DVDs.  If you can't believe this simple operation is built into ubuntu you can always run a program to accomplish the same task. GnomeBaker does the same thing with "Burn CD Image" and "Burn DVD Image".  And if you have a very tricky format, install K3b which can do everything GnomeBaker can do and more.  

Then again, there is always the command line which can do everything K3b can do and make dinner at the same time. ;)
