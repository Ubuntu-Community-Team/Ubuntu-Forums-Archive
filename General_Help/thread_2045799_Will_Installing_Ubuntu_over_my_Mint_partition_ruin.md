Usually installing another Linux distribution will not mess with you dual boot. Installing Windows after you install Linux will mess up dual boot since Windows refuses to acknowledge Linux :)
Mint is based on Ubuntu so installing Ubuntu might not actually replace the problem, unless your Mint version is too old.
