### Post by Jose Catre-Vandis on 2010-05-06
OK Fixed it. Thanks to everyone for the replies and the pointers.

In the end it was fairly straightforward, but would have been nice to have had some direction with the kernel upgrade that it would break the nvidia kernel module.

How to fix:

When you get the low graphics mode "request" choose the previous config option to get back to the nv driver, you will then get a rasonable boot up.

Open up hardware drivers, select the NVidia driver and then click on Remove

When this has finished, click on Activate, what a bit, then reboot

All should be well. :)
