### Post by Daveth on 2007-08-09
find the file types you want vlc to play, say a movie. In the file browser, select that file and right click, to get to Properties. Then "open with" and select vlc. Then close. Then all files of that file format will open into vlc.

### Post by stchman on 2007-08-09
> **eddietours said:**
> how do l setup vlc as the default dvd player:(

Go to System--->Preferences--->Removable Drives and Media

On the Multimedia tab put:

vlc %m

in the box what to do when a video DVD is inserted.  The %m is in case you have multiple optical drives.
