### Post by Nikron on 2007-03-11
I'm interested in this.  I have found one temporary solution however.  You can move/delete 

/usr/share/applications/ubuntu-about.desktop
/usr/share/applications/gnome-about.desktop

It just deletes them off the list, presumably because the program can longer find them.  It does leave the separators though, so it looks fairly ugly.  I wish they'd just let you delete it off the Main Menu GUI program...
