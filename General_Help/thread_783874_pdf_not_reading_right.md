### Post by chewearn on 2008-05-06
Open the pdf in Windows using Adobe Reader, go to:
File > Properties > Fonts

Find out any special font being used.  Go to your Windows fonts folder, copy the *.ttf file.  Use instruction here to install the font in Ubuntu:
[https://wiki.ubuntu.com/Fonts](https://wiki.ubuntu.com/Fonts)
