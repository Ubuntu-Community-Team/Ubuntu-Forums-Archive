### Post by arsenic23 on 2009-06-24
So after seeing a good deal on a cheap 37" LCD TV on slickdeals I decided it was time to replace my aging 20something TV.  I also ended up buying a new media PC as my previous one was not capable of digital output or realtime HD decoding.  

I currantly have a nice little 5050e with Nvidia 8300 onboard video plugged into my new TV, but it is a huge headache for me.  The TV hates computers, just hates them.  If reports that it is a 1080p display to the PC despite only being 720p, and fails at just about everything hdmi related.

If I use the default xorg with Nvidia drivers I get way too high a resolution, only the very center bit of the screen displayed, and odd GUI lockups that require gdm to be rebooted.  If I use the Nvidia control panel to set the resolution correctly I get a tiny tiny image at the center of the screen and the same odd GUI lockups that require gdm to be rebooted.  If I use a custom xorg, I get a nearly functioning display, but with some major overscan.  *( I've included an image to better show these issues. )  (( Also note that I have nearly the same issues when trying to use this setup with Vista, just without the GUI lockups. ))

I attempted to correct the overscan by modifying my xorg.conf file using the post here: [http://ubuntuforums.org/showthread.php?t=1003099&page=2](http://ubuntuforums.org/showthread.php?t=1003099&page=2)
but this completely fails no matter how I set it up.  I'm greeted with the old 'x server couldn't read settings' dialog.

I also attempted to setup a simple modline in the screen section only (as this has worked for me back when I was outputting vga through my Svideo on my other media PC).  All of those attempts resulted in gibberish being displayed on the screen.


Is there any other way to correct this overscan I'm experiencing, or will I just be sitting on my thumbs waiting for Nvidia to release a driver that deals with the issue?



My (almost)working xorg.conf:
```
Section "ServerLayout"
    Identifier     "Default Layout"

### Post by NT4usB on 2009-06-24
The MythTV guys deal with this kind of stuff all the time. Might go that route in your google searches.
iirc, TV overscan is not supported (for some output formats?)
I'm guessing you need some more information to tell the driver what it's feeding. 
Stuff like:```

Section "Screen"
    bla bla bla
       Option         "TVOutFormat" "Component" #or SVIDEO Composite etc
    Option         "TVStandard" "HD480i" # "NTSC"
bla bla bla...
  
```
Also google [modlines ]("http://www.x.org/wiki/FAQVideoModes")and manual [configuring EDID]("http://wiki.linuxmce.org/index.php/EDID").
Read up on [VDPAU]("http://www.nvnews.net/vbulletin/forumdisplay.php?s=&forumid=14") enabled driver for Nvidia. Makes playing HD a snap.
Might look around the [MythTV wikki]("http://www.mythtv.org/wiki/Category:HOWTO") for display related stuff.
