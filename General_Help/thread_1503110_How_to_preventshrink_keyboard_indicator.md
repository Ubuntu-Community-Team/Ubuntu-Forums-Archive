### Post by sdennie on 2010-06-06
Those should only be present at the login screen.  If you want to remove them, they are part of the GDM theme.  You can either install a different GDM theme or see if you can find the theme you are currently using and remove those elements (I looked around and couldn't find the theme on my machine but, I'm sure it's there if you poke around enough).
