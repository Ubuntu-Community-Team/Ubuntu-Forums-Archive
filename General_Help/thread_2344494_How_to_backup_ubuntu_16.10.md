### Post by Mark Phelps on 2016-11-25
I use Clonezilla all the time to do image backups and even though my Linux drive is 120GB, of which the Linux partition I back up is 30GB, the saved image is only 4.3GB.

Clonezilla, when you do "saveparts" (the option to save partitiojns), does compression -- so the backup files end up taking up little space.  The used space on my Linux partition is 19.4GB, but as I said, the backup set is only 4.3GB.
