If you don't like the look, you can just change the theme. Lots of themes are available at [http://gnome-look.org](http://gnome-look.org) and I'm sure the old human theme will also be there somewhere :)

And, if you don't like the window buttons, then there's a quick fix for that...

1. Open up gconf-editor (Alt+F2 'gconf-editor')

2. In the gconf-editor navigate to **/apps/metacity/general**

3. There should be a key in there that should read **button_layout**

4. Edit that key (Right click->Edit) and change the value to:
```
menu:minimize,maximize,close
```

And voila, window buttons back to normal :)
