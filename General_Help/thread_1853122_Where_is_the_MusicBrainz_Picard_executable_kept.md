### Post by nrundy on 2011-10-01
I'm trying to open an mp3 file with MusicBrainz Picard by right-clicking on the file and selecting "Open With".

But MusicBrainz Picard is not showing up in the list of apps that I can open it with. To avoid having to open MusicBrainz app and then searchign for the file, does anyone know where the stupid musicbrainz exe is kept?

I've looked in /usr/bin & /opt. I've searched the file system. I can't find it anywhere.

---

### Post by carranty on 2011-10-01
Right click on file > open with > Other application

Click on the 'use custom command' arrow and type in 

[I]picard

[/I]That should do it.
