### Post by Lars Noodén on 2010-01-13
If you want to go even lighter than Xfce, then you can go with Fluxbox, Openbox, FVWM, Ratpoison or (yes, this is the real name) scrotwm.  Some of these are rather plain, but you can add modules for things like icons or task bars, or otherwise customize the heck out of them.  There are [other window managers](http://packages.ubuntu.com/search?suite=karmic&section=all&arch=any&searchon=all&keywords=window+manager), all with their advantages and disadvantages.  

One of the best illustrations of how far you can take customization is to install fvwm and see how plain it is.  Then install fvwm-crystal and see the exact same window manager (no desktop environment) but with *customizations*.

That's a way to reduce RAM and CPU requirements.  

To reduce disk space requirements, once you figure out what you want, get the alternate install cd and install a 'command line system'.  Then install *only* the packages you want, including the window manager.


As @ howefield mentioned, all packages from the Ubuntu / Kubuntu repositories should run.  Some UI's will be more based on or compatible with KDE others with GNOME but both will run eachother's software.  They just might need a few more dependencies downloaded.
