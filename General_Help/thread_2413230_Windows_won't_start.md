### Post by yancek on 2019-02-22
You might need to use boot repair to provide more information.  Go to the link below and use the 2nd option explained on that page to get boot repair.  When you run it, make sure you select only the option to Create BootInfo Summary and do not try to make any repairs.  When it finishes, it should give you a link which you can post here and members will be able to access and view the output and hopefully suggest a solution.
