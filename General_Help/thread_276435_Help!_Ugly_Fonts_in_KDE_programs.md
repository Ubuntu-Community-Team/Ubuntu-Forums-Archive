### Post by Arisna on 2006-10-12
Navigate through the K Menu to "System Settings."  Click the "Appearance" icon, choose "Fonts," check "Use anti-aliasing for fonts" if it is not already checked, and then click the configure button.  Make sure sub-pixel hinting is enabled, and up the hinting style until satisfied.  See the below thread for further help if you need it:

[http://www.ubuntuforums.org/showthread.php?t=4456&highlight=fonts+kde](http://www.ubuntuforums.org/showthread.php?t=4456&highlight=fonts+kde)
