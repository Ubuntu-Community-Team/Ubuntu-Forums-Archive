### Post by ajgreeny on 2007-01-14
You can also get an html output from the command:-
sudo lshw -html

This just gives you a listing of your hardware, but it can be useful as a file to have handy with all the information on your machine hardware.  The html output can then be copied to a text editor and saved as a named html file in your home folder.
