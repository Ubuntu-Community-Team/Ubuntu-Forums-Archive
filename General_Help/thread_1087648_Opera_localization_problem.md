### Post by Colonel Kilkenny on 2009-03-05
Probably the font used in menus doesn't have those special letters. Try to use different font. IIRC Preferences -> Advanced -> Fonts -> Browser menus might be the correct place.
If that doesn't work the problem might be in Qt as Opera uses it to font rendering.

---

### Post by BaidareW on 2009-03-06
