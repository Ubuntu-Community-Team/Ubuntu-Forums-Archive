### Post by mb_webguy on 2009-02-12
If you're doing it in your browser, then Totem, because the VLC plugin isn't that great.

If you're doing it in the application itself, I'd go with VLC, because it actually has an interface for doing that, instead of generic "Open Location" dialog.  If it's a Shoutcast station, you may have to go into Preferences, click the All radio button to show all preferences, then enable the Shoutcast services discovery module under Playlist->Services Discovery.

Amarok and Rhythmbox also allow you listen to Shoutcast stations.
