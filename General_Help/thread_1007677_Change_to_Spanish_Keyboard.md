### Post by eyefragment on 2008-12-10
To set up your computer to use an alternate keyboard, go to System >> Preferences >> keyboard, select the "Layouts" tab, and click the "+" button to add another language to your keyboard layouts.

To quickly switch between layouts, right-click the one of your panels (the bars that are at the top or bottom of the screen by default) and select "add to panel."  Add "Keyboard Indicator" to the panel.  When you want to switch keyboard layouts, you can left-click the keyboard indicator icon (which will just appear as text for whatever layout you've chosen).  I'm not sure if there's a hotkey to do this without having to go click on the panel...

There are a lot of other nifty applets you can add to your panel too.  If you haven't looked at them, you should check them out.

### Post by sdennie on 2008-12-10
The other option is to change the default keyboard layout to "USA International (with AltGr dead keys)".  That's generally a very intuitive layout for people who use an English keyboard but sometimes want things like ¿¡ñ.  Essentially the layout is the same as normal USA layout and, all of those characters are available by hitting AltGr (usually just the right Alt key) and sometimes Shift.  For example, ñ is just "Right-Alt n" and ¡ is "Right-Alt Shift !".
