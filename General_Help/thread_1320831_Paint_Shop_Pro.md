### Post by Chronon on 2009-11-09
If you do figures for LaTeX documents, I recommend Ipe.  It has snap modes, allows LaTeX for typesetting text and math and exports to PDF or EPS.  It's not great for color support, but is good for simple line drawings, geometrical constructions, etc.

### Post by floyd-humphrey on 2009-11-09
Paint Shop Pro (I own V4 - V9) seem to run mostly fine ( tested 7 and 8 ) under Wine. There will be improvements in compatibility and stability as Wine improves, but frequent saving should do.
A PSP "lite" could be PhotoFiltre [(link)]("http://photofiltre.free.fr/frames_en.htm"), also under Wine. As I now read, retrieving the link, there is a Linux project for PhotoFiltre started! Keep an eye on that. Super-lightweight and in few ways even more powerful than PSP!

### Post by Mark Phelps on 2011-04-09
The link below is to the WineHQ page for the older versions of Paint Shop Pro:

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=76]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=76")

Right-click on the testing info for the versions you want and read what other folks have done.

As to the newer versions (X1 +), they do NOT work with Wine, period.
