### Post by wdaniels on 2008-04-12
Right-click on the menu bar and select "Edit Menus", find the entry in the list, right-click and select "Properties".  Then you can click on the box showing the icon, which will allow you to select an image file to use for the icon.  If there isn't an icon file installed by the application somewhere, you can usually find one suitable on gnome-look.org or similar sites.
