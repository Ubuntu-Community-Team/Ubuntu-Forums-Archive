### Post by adamkane on 2006-04-01
Use O'Reilly to learn bash, python, php, ruby, java.

Learn how to install an apache2/mysql4/php5 framework:
[https://wiki.ubuntu.com/ApacheMySQLPHP](https://wiki.ubuntu.com/ApacheMySQLPHP)

Use the framework to install some sort of Content Management System that you would find useful:
[http://www.opensourcecms.com/index.php?option=content&task=view&id=388&Itemid=143](http://www.opensourcecms.com/index.php?option=content&task=view&id=388&Itemid=143)

A CMS would require you to learn more about Linux in order to maintain it. It's just a recommendation. There are other ways to get involved with Linux, but this way, you don't have to become an application programmer or network administrator to get involved with Linux.
