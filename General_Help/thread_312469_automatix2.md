> **cssutto said:**

Will installing automatix2 conflict with aptitude?

No, No and it might (if the repositories used by Automatix have more recent versions of an app which was installed using aptitude and the default Ubuntu repositories). Typically Automatix by policy sticks as much as possible to the default Ubuntu repos.
