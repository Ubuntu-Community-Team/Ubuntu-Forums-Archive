Why would you want to do that?

In Wubi, instead of using GRUB, it uses GRUB4DOS -- a DOS-compatible version of GRUB that can be installed in an NTFS-format partition.

If you're trying to replace that with Lilo, my guess is that even if you could install it, it wouldn't work.
