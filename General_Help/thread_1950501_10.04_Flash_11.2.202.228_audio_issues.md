### Post by dcstar on 2012-03-31
The Lucid-updates repository now has the flashplugin-installer package that installs the 64-bit version of flash for 10.04 systems, but the latest Flash release from Adobe causes audio to stutter on my system.

Installing the previous version of Flash resolves this issue, unfortunately you firstly have to remove the flashplugin-installer package and manually install the file from the old version.

You need to download the previous version from the Adobe site and extract and install the libflashplayer.so file into /usr/lib/mozilla/plugins:

[http://helpx.adobe.com/flash-player/kb/archived-flash-player-versions.html](http://helpx.adobe.com/flash-player/kb/archived-flash-player-versions.html)
