### Post by stoneage on 2011-12-20
> **CProgramming said:**
> 
I have question - how can I drag the left taskbar to the bottom? (Like windows)

You can't. If you use the 'Classic' desktop, you can have the panel on any of the four sides, but as far as I know the unity taskbar is not configurable in that way.
[http://askubuntu.com/questions/40980/can-you-move-the-ubuntu-unity-launch-bar](http://askubuntu.com/questions/40980/can-you-move-the-ubuntu-unity-launch-bar)

You can, however use the Gnome panel as well as Unity:-
[http://askubuntu.com/questions/39069/how-do-i-use-the-gnome-panel-in-unity](http://askubuntu.com/questions/39069/how-do-i-use-the-gnome-panel-in-unity)


Or there is this if you feel like an adventure, though please bear in mind it is unofficial, and you install it at your own risk:-
[http://askubuntu.com/questions/33605/can-i-move-the-unity-launcher](http://askubuntu.com/questions/33605/can-i-move-the-unity-launcher)

### Post by yetiman64 on 2011-12-20
> **CProgramming said:**
> ...I have question - how can I drag the left taskbar to the bottom? (Like windows)...
It can't be just dragged to the bottom or easily adjusted. 
You can however add a ppa (personal package archive) to your sources and install a compiz plugin called "Ubuntu Unity Plugin Rotated". This is used to replace the existing plugin "Ubuntu Unity Plugin".

Instructions can be found [[COLOR=Blue]--here--[/COLOR]]("http://www.webupd8.org/2011/11/install-ubuntu-unity-bottom-launcher.html") , please read all the instructions carefully before trying, the replacement plugin is still regarded as experimental (though I find I'm getting very good performance from it here :)).
