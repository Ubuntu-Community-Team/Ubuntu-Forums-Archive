### Post by amauk on 2008-10-16
If you don't have it already,
install the Compiz Config settings manager
(it's not installed by default)

```
sudo apt-get install compizconfig-settings-manager
```

new menu item under System > Preferences
"Advanced Desktop Effects Settings"

Run the compiz settings manager
Enable the "Window Previews" plugin
