### Post by DieB on 2008-02-25
hi
 
I guess u already tried to set up the appropiate drives in wine ( via "Applications"-"Wine"-"configure Wine") - if not try this.

To mount an iso easily there is a tool called "Gmount-iso", it's in the "universe"-repository.

Certain games I tried had similar problems due to copy-protection (wine is not supporting many), so you might have to try a "no-cd-patch" from the web. But only do it if you have a legal copy (!).

Either way ever had an look on [www.winehq.org]("http://www.winehq.org"), there you might find better informations than in this forum?
