### Post by Mike_Walsh on 2015-02-03
Hi, patrikmellq.

Howefield is absolutely correct about the 'historical' reasons. The 'amd64' tag was based on the 'Athlon 64' processor, the very first commercially successful 64-bit CPU to be released by AMD, back in 2003. It precipitated the 64-bit revolution; and the industry has never looked back.

I know this because I have one in my elderly HP/Compaq desktop! For a single-core CPU, it truly is a magnificent beast....especially when running a featherweight distro like Puppy Linux; it flies! Even 10 yrs down the line with Ubuntu's 3D Unity desktop, it's still capable of holding its own.

You certainly CAN install the i386, if you wish. It'll run, no problems; but you'll not be doing your hardware justice if you do!
