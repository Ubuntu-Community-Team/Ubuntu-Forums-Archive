### Post by megamania on 2009-09-30
> **Jondelius said:**
> If I can get the music and gaming sorted - I'd never look back to Windows.  But they are the only things that are making me slightly uncomfortable about the switch.

Why don't you start with a dual-boot?

Repartition your hard disk so that it has:
- a partition for windows
- a partition for /  (root)
- a partition for /home
- a swap partition

Install windows, then install Ubuntu. You'll find a boot selector when you start up your computer.

This way you'll have time to test linux while still being productive.
I chose the hard way 5 years ago - just deleted windows and installed Ubuntu, but a dual-boot can be a wiser solution at least at the beginning.

Also, consider installing Virtual Box in Ubuntu and running the Windows software you need from there. For games, it's better to try Wine because Virtual Box would be too slow.

### Post by random turnip on 2009-09-30
As others said, dual boot would probably be a wise choice, jumping in at the deep end can often be a mistake.

You may well after time discover that you have found replacements and emulations for everything that you need and like using them in Ubuntu, that's when you know you don't need Windows anymore, but it won't happen straight away.  

Dual booting will basically give you chance to get used to Ubuntu in your own time and not necessarily HAVING to use it for everything if you don't want to rather than forcing yourself and not enjoying the experience as much.

Dual-booting is also REALLY easy to set up, just after you've re-installed Windows (which will take longer than installing Ubuntu) stick your liveCD into the drive, restart, follow the on-screen instructions, re-boot again and you have your new OS.  Takes like 35 - 45 mins.
