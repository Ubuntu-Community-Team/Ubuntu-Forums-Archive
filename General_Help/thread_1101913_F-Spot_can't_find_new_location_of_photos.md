### Post by fieroboom on 2009-03-20
I use f-spot all the time, but I've never messed with the tags. However, searching a little, I found [this page](http://f-spot.org/User_Guide/Organize). Under the "Tags" section, it states:
> Expert Tip: F-Spot can write tags as metadata fields into JPEG files. Tags for various RAW files, PNG, TIFF, and others are written to F-Spot's database. You will have to re-tag these files if you re-import your collection.
It sucks, but that's what I've found so far. I'll keep looking, and let you know if I find something better.
