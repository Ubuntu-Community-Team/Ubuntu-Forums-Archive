### Post by taurus on 2008-11-19
I assume you have Ubuntu on that harddrive.  Why don't you boot your machine with the LiveCD and use gparted (System -> Administration) and delete all the partitions.  Then, format that one large partition (your harddrive) to ntfs filesystem.  Now, reboot and see if windows can detect your harddrive now.
