### Post by Victormd on 2008-04-13
> **dethredic said:**
> Cool, I  made a graph in OpenOffice, but I don't see how to add the line of best fit.

From Openoffice 2.3, to add best fit, curve, log fit.. etc, create a scatter plot, right click on the points in the plot>properties and go to the statistics tab. There you'll find all types of fit curves along with several statistical analyses to be included in the plot if you'd like...
