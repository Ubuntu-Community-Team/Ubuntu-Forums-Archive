### Post by plucky on 2012-03-14
> I should *** that all of this stemmed from my entering

Code:

sudo update-manager -d

and attempting to upgrade to 12.04 LTS.


Did the upgrade succeed?

Post output from a terminal for ```
lsb_release -a
```

Also open your software sources with ```
gksudo gedit /etc/apt/sources.list
``` and put a # in front of each of these lines
```
54    deb http://archive.ubuntu.com/ubuntu lucid main universe restricted multiverse
    55    deb-src http://archive.ubuntu.com/ubuntu lucid universe main multiverse restricted #Added by software-properties
    56    deb http://security.ubuntu.com/ubuntu/ lucid-security universe main multiverse restricted
    57    deb-src http://security.ubuntu.com/ubuntu/ lucid-security universe main multiverse restricted
    58    deb http://archive.ubuntu.com/ubuntu lucid-updates universe main multiverse restricted
    59    deb-src http://archive.ubuntu.com/ubuntu lucid-updates universe main multiverse restricted
```

Then run ```
sudo apt-get update
``` and see what happens

Good Luck
