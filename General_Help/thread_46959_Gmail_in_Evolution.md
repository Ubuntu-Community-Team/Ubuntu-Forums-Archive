### Post by varunus on 2005-07-06
Gmail can be configured in Evolution.  Follow the instructions here: [http://gmail.google.com/support/bin/topic.py?topic=194](http://gmail.google.com/support/bin/topic.py?topic=194)

They don't list Evolution exactly, but you can use the instructions for Netscape and Thunderbird, of course with a few small order changes, to get everything set up.  It works, just find each corresponding setting in Evolution.

Enjoy.

### Post by varunus on 2005-07-07
You should enter "pop.gmail.com" for your receiving server.  Make sure you have it set to use a secure connection (SSL) and that your username has the @gmail.com suffix on it.

EDIT:  Also, make sure you have followed the following steps to enable POP email on your gmail account: [http://gmail.google.com/support/bin/answer.py?answer=13273](http://gmail.google.com/support/bin/answer.py?answer=13273)

Good luck.
