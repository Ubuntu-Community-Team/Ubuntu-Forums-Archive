### Post by BrokeMahPC on 2010-05-03
The HP60 and HP60XL (more ink) will work in any printer that is compatible with them - it usually says on the ink cartridge box which printers will accept it.

Had a look on the HP web site for you:
HP 60 Ink Cartridges is compatible with the following products:
HP Deskjet D1660 Printer (CB770A)
HP Deskjet D2530 Printer (CB673A)
HP Deskjet D2560 Printer (CB671A)
HP Deskjet D2660 Printer (CH366A)
HP Deskjet D2680 Printer (CH396A)
HP Deskjet F4280 All-in-One Printer (CB656A)
HP Deskjet F4480 All-in-One Printer (CB745A)
HP Photosmart C4680 All-in-One Printer (Q8418A)
HP Photosmart C4780 All-in-One Printer (Q8380A)

It only lists 1 printer form each series so it's possible this is not an exhaustive list. Your printer is a 4400 series, which includes:
Model: F4480, F4483, F4488, F4450,
F4440, F4435, F4470, F4472, F4473,
F4492
I know the 4440 and 4480 take the HP60 so it's possible all the 4400 series use it but you would have to check.
