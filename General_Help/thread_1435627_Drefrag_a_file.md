### Post by soltanis on 2010-03-21
Why? Due to aggressive caching used for external flash disks (which any good system _should_ have but anyway) fragmentation rarely happens in those files, and even when it does, it shouldn't impact performance (flash devices don't need to seek like hard disks). I don't think defragment utilities exist under Linux, much less ones that target only one specific file; you may have to use Windows defrag for that.
