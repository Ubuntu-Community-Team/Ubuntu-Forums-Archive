### Post by GregBrannon on 2010-01-28
> 1) Which Free-ware offered under Ubuntu for writing programs would you suggest learning. like Code Light IDE is that Linux? or Python 3.1

IDE - Integrated Development Environment.  A program that, at a minimum, provides a programmer with a source code editor. Some add various bells and whistles to the editor's capability.  Some are useful for more than one programming language.  A few are specific to a single language or have separate versions for different languages; Eclipse, for example.  Good IDEs will provide the hooks to the compilers, linkers, and debuggers that are installed on your computer to allow editing, compiling, linking (if necessary), running, and debugging your programs without leaving the IDE interface.

You don't need to learn an IDE to program.  Some think that using an IDE is a crutch that hides some things about programming that everyone should learn.  OK, once you learn the basics, get a good IDE.

Linux is a member of the class of computer operating systems in the same way that Windows, DOS, and MacOS are operating systems.

Python is a programming language as are C, C++, Java, Perl, and others already mentioned.

> (2) are they Linux code?

Not 100% sure what you mean by "they" and the term "Linux code."  Since you started out asking about IDEs, I'll assume that "they" refers to IDEs.  Since the IDE is a program, you'd like to have an IDE that can be run on your computer's operating system (OS).  Many good, mainstream IDEs have versions that run on Windows, Linux, and MacOS.

> (3) do I write with one and compile with another program?

As I described above, the IDE is the environment or interface for doing the basic programming processes: editing, compiling, linking, running, and debugging.  Typically, other programs installed on your computer actually do the compiling, linking, running, and debugging functions.  A good IDE will provide the commands in the background to call and execute those programs for you when needed.  Alternatively, you can elect to drop to a command or terminal mode to execute those programs on the source code and object files yourself.

> (4) with Python do I down load all version as each is a upgrade?

I'm not a Python expert, but I think you would use the latest version just as you would typically use the latest version of any program you were running or using.

> (5) PDF Guide Book I have already is "Thinking like a Computer Scientist for C++ is this a good book to use?

I agree with t4thfavor, use a book that you find interesting and keeps your attention.  I also find it helpful to compare the same topics in 2 or 3 books to get the same info explained in different ways.  I'm bound to understand one of them.

Good luck.  If you have further programming questions, you might ask in the programming forum.  It's hard to find, buried near the bottom of the list, but it's there.
