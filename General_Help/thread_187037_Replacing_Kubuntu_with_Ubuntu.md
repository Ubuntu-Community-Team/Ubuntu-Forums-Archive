### Post by pRedat0r on 2006-06-02
I just posted a similar thing on a different thread, but I believe you can do sudo apt-get install ubuntu-desktop to install the default ubuntu desktop (gnome).  You might want to check that out in the synaptic installer, but I believe this will give you the option of choosing a gnome session at login time, along with the default ubuntu settings.  I havent tried this before, but I remember reading it several times on the dev forums before release.

for XFCE you would use xubuntu-desktop, or for KDE (if you arent already running Kubuntu, you would use kubuntu-desktop.

Seems like a common question, if this works you might want to help others out as well.
