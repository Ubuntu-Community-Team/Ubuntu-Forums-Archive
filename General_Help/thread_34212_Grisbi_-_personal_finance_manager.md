### Post by GOwin on 2005-05-13
[img]http://www.grisbi.org/images/logo.gif[/img]
[size=15]Grisbi[/size]
**Easy Personnal Finance Manager for All**
[http://www.grisbi.org](http://www.grisbi.org)

> Grisbi can manage multiple accounts, currencies and users. It manages third party, expenditure and receipt categories, as well as budgetary lines, financial years, and other informations that makes it quite adapted for associations (except those that require double entry accounting). The exhaustive list of Grisbi's functionalities can be found in the User's guide, paragraph 1.2.

I think this GPL'd application is worth looking into. Especialy by people who would like to see how their finances are doing and would like something more sophisticated than a spreadsheet.

It's simple, fast, stable - well, at least it hasn't crashed on me yet. (Crossing my fingers)

When I was looking for a personal accounting software in linux, Gnucash immediately came to mind. This is probably the most mature, open-source, desktop accounting software for linux. Unfortunately, I cannot even set my accounts up. I'd get an aguile 16 error or something, and it would crash and would ask me to report the bug to the site.)

I kept on it for a few days, trying to find out what it was but never got it working.

So I searched and found other software. I've downloaded kurush but I never got it to run - i didn't bother finding out why. 

Then I read about myBudget. It was good enough and I've been using it for more than a month now but it is meant for monitoring a single account (like cash on hand) and monitoring your stated budget (and how you're doing against it). But if you want to manage bank accounts, other assets and liabilities you're better off with something else.

Then I found grisbi. I wasn't actually looking for that particular softare or anything of that kind. (I was looking for software to manage SMS and phonebook of my SonyEricsson t630.) I was thinking - gosh, this is like myBudget x 3. (It does what myBudget can, PLUS more)

It supports multiple currencies, and defaults to the Euro.

If you're coming from quicken, it supports importing QIF files. It aslo import from CSV files.

The English documentation is sparse but can get you around quicly -  but if you speak French, then you're lucky. If not,  the intuitive interface will get you going in no time at all - especially if you've tried other personal finance managers.

The English labels and text in the program sometimes need some thinking but other than those little quirks, it's very useable at this stage. 

Is is being actively developed? Version 0.5.5 was released in Jan 2005. (We have 0.5.3 in the repositories but you can download and update the 0.5.5 [debian package](http://www.grisbi.org/download.en.html#deb) from their site)

Their support forum (again, mostly in French) is kinda sparse too - that could be a good thing. If something is good enough to figure it out on your own, why bother going to the forums. :)

Now go ahead and try it out.

Post here if you like it or not.

PS
They also have sub-projects to make grisbi run on the web, and MS windows.
