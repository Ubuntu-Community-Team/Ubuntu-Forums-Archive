### Post by again? on 2019-08-19
[COLOR="#808080"][/COLOR]In 19.04 you can edit /usr/share/sddm/themes/ubuntu-theme/Main.qml
It's just a matter of commenting out sections using qml syntax of "/*" and "*/" 

Make a backup before editing...
```
sudo cp /usr/share/sddm/themes/ubuntu-theme/Main.qml /usr/share/sddm/themes/ubuntu-theme/Main.qml.bak
```
If you want to revert after editing, run...
```
sudo cp /usr/share/sddm/themes/ubuntu-theme/Main.qml.bak /usr/share/sddm/themes/ubuntu-theme/Main.qml
```

Edit the default sddm theme...
```
sudoedit /usr/share/sddm/themes/ubuntu-theme/Main.qml
```
These are the[COLOR="#FF0000"] /*[/COLOR] [COLOR="#808080"]commented out sections[/COLOR] [COLOR="#FF0000"]*/[/COLOR] I made starting at line 156 to the end of the file.

```

            Row {

                anchors.margins: 5
```
