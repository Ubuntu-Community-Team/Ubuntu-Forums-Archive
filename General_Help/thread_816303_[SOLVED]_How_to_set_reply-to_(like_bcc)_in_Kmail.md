### Post by peterbrewer on 2008-06-02
Umm, if you go into the settings for kmail you can create different identities which can have different reply to addresses.  In this case when you go to reply there should be a drop down box at the top of the message which allows you to select your method for replying.

From what I understand I think this with the previous should do what you want.

Hope this helps.
