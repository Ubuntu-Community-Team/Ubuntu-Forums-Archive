### Post by micahpage on 2012-09-07
I ran a command that made an app run fullscreen in which after I quit the program, my resolution was all messed up. The only fix I could find was to actually log out and log back in to fix it.

How would you fix this error in the midst of the resolution problem with the terminal?

---

### Post by zombifier25 on 2012-09-07
```
xrandr -s 1024x768
```
or any resolution your screen is.
