### Post by ad_267 on 2008-11-04
This page here should point you in the right direction: [http://ubuntuforums.org/showthread.php?t=809695](http://ubuntuforums.org/showthread.php?t=809695)

To run emerald you should press Alt+F2 and run "emerald --replace" then you can go to System - Preferences - Emerald Theme Manager to import your new theme (assuming you have emerald installed).

An emerald theme should have the extension .emerald, so you might have to extract the .tar file to get the .emerald file. You can right click and select "Extract here". Unless this is actually a gtk or metacity theme not an emerald theme?

This is also useful: [http://linuxowns.wordpress.com/2008/06/16/guide-to-customizing-ubuntus-look-and-feel/](http://linuxowns.wordpress.com/2008/06/16/guide-to-customizing-ubuntus-look-and-feel/)

### Post by ad_267 on 2008-11-05
There's a few different types of themes in there.

The brawnee/simp directory looks like a theme for AMSN, a replacement for msn messenger.

The brawnee/TEMAK2/THEMES/brawnee directory is a gtk theme. To install this you can go to System - Preferences - Appearance and drag the brawnee folder onto that window.

brawnee/TEMAK2/brawnee is an icon theme. You can drag this onto the appearances window in the same way to install it.

The TEMAK2/EMERALD contains an emerald theme but it isn't set up correctly for emerald to open it. You need to select all the files in the brawnee/TEMAK2/EMERALD/theme directory and then right click and select "Create Archive". Call it brawnee or whatever you want and make sure the type is ".tar.gz". Now you have to rename the brawnee.tar.gz file so that it is brawnee.emerald. Now you can go to System - Preferences - Emerald Theme Manager and select import and select the brownee.emerald file.

This is a really badly put together theme. Usually it's not this hard to install a theme. Have a look at gnome-look.org for some themes. I like this gtk theme: [http://www.gnome-look.org/content/show.php/MurrineGlow?content=88406](http://www.gnome-look.org/content/show.php/MurrineGlow?content=88406)
For a nice icon theme Eikon2 from [http://drop.io/fmrbpensador](http://drop.io/fmrbpensador) is really good. You can just drag and drop those ones onto the appearance window.
Another one is [http://www.gnome-look.org/content/show.php/oxy-gnome?content=79736](http://www.gnome-look.org/content/show.php/oxy-gnome?content=79736)
