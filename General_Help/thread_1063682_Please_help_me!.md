### Post by Kevbert on 2009-02-08
Welcome to Ubuntu.
Unfortunately it's a memory problem (lack of memory resources). From the official site:
> You need 128 MB RAM to run the Live CD or 192 MB RAM to install. The Alternate Install CD only requires you to have 64 MB RAM at install time. 

You need the Alternate CD or need to try to increase your memory. Xubuntu will still run quite slow so I suggest that it may be better for you to get [[COLOR="Red"]Puppy[/COLOR]]("http://www.puppylinux.org/") instead. It runs well on my old Athlon 900 Mhz with only 256Mb RAM, Xubuntu runs quite slow on the same machine.
