### Post by tgalati4 on 2014-02-20
I like google's calendar because it works well with Android phones.  If you want a text-based calendar, then *cal* is installed.  If you want to explore other calendars, open a terminal:

```
apt-cache search calendar
```

There are a few.


It's minimal, but some people like that.
