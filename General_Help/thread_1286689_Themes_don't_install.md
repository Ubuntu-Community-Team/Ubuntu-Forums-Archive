### Post by mechro on 2009-10-09
Drag and drop the **tar.gz** file onto a blank space in the Appearance Preferences/Theme window.

If it's not a tar.gz file further work may be required.

Some themes require extra engines/libraries.

Some themes from Gnome look are buggy or require further work.
