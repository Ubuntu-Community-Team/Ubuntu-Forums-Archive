**Original Question:**
> So, I have Xubuntu-64 (running gnome-session!) installed with a nVidia quatro card.  I got it all installed and working using envyng to have dual displays with Xinerama.  Then I gave compiz a try and now it doesn't seem to work. Compiz-fusion is not running at the moment.....

I have 2 HP1955 19" displays.

When I run displayconfig-gtk, screen0 shows that it's indeed a LCD Panel 1280x1024, but the resolution only shows 2560x1024 and I can't change it.

This poses a problem because it stretches my taskbars across both screens (I hate that, xinerama had it working properly) and windows in the second display don't render correctly.  

Is there any way to fix it so that the 1280x1024 display will work at resolution 1280x1024?

**HOW TO GET IT TO WORK!**
How to get Compiz-Fusion running on Dual Monitors on an nVidia graphics card within Ubuntu! (gnome-session, xfce should be similar) 
[LIST=1]
[*]First, Follow the directions **[HERE]("http://www.ubuntugeek.com/dual-monitors-with-nvidia.html")** verbatim.  You can copy the xorg.conf from the reply below to accomplish the setting up of your dual monitors. YOU MUST USE THE UBUNTU NVIDIA Driver! aftermarket drivers downloaded from nVidia's website or installed using Envy WILL NOT WORK! (at least I couldn't get them to work)
[*]After the GLX Driver is downloaded, installed, and the X Server's xorg.conf is edited and restarted, right click on the desktop and choose Change Desktop Background.. (you can also get here from *System > Preferences > Appearance*)
[*]Click on the **Visual Effects** tab and choose Extra.  The screen will blip and ask you if you want to keep the settings.  Choose yes and Close.
[*]Compiz Fusion is now running! if you want to be able to customize it, run this command to download the setup utility:
```
sudo apt-get install compizconfig-settings-manager
```
With this installed, you can go to *Settings > Preferences > Advanced Desktop Effects Settings* to setup all the different eye candy effects!

[/LIST]

### Post by javaguy78 on 2008-09-19
So, if you are running Ubuntu and you have Dual Displays AND you want to enable compiz fusion, you should use THIS how-to to setup your display so that all the extra stuff works:

[http://www.ubuntugeek.com/dual-monitors-with-nvidia.html](http://www.ubuntugeek.com/dual-monitors-with-nvidia.html)

My xorg.conf:

```
# xorg.conf (X.Org X Window System server configuration file)
#
# This file was generated by dexconf, the Debian X Configuration tool, using
# values from the debconf database.
#
# Edit this file with caution, and see the xorg.conf manual page.
# (Type "man xorg.conf" at the shell prompt.)
#
# This file is automatically updated on xserver-xorg package upgrades *only*
# if it has not been modified since the last upgrade of the xserver-xorg
# package.
#
# If you have edited this file but would like it to be automatically updated
# again, run the following command:
#   sudo dpkg-reconfigure -phigh xserver-xorg

Section "InputDevice"
	Identifier	"Generic Keyboard"
	Driver		"kbd"
	Option		"XkbRules"	"xorg"
	Option		"XkbModel"	"pc105"
	Option		"XkbLayout"	"us"
EndSection

Section "InputDevice"
	Identifier	"Configured Mouse"
	Driver		"mouse"
	Option		"CorePointer"
EndSection

Section "Device"
	Identifier	"Configured Video Device"
	Driver		"nvidia"
	Option		"NoLogo"	"True"
EndSection

Section "Monitor"
	Identifier	"Configured Monitor"
EndSection

Section "Screen"
	Identifier	"Default Screen"
	Monitor		"Configured Monitor"
	Device		"Configured Video Device"
	Defaultdepth	24
	Option "RenderAccel" "true"
	Option "TwinView"
	Option "MetaModes" "1280x1024 1280x1024"
	Option "TwinViewOrientation" "RightOf"
EndSection

Section "ServerLayout"
	Identifier	"Default Layout"
  screen "Default Screen"
EndSection
Section "Module"
	Load		"glx"
EndSection
```

and the Compiz-Fusion works just fine on dual displays!
