Here's the simple solution.  You need a special program, called 'lzip', to unpack that .wav file.  Fortunately, it's part of a standard package.

1.  Open a terminal window
2.  run "sudo apt-get install lzip" to install the utility you need.  You'll have to give it your password.
3.  run "lzip -d filename.wav.lz" to uncompress the file.  This assumes the .lz file is in your home directory.  If not, 'cd' to the directory the file is in, then run the command.

This should leave you with the uncompressed "filename.wav" file.
