### Post by zombiepig on 2008-11-26
It's not installed by default in ubuntu, so you'll need to manually install it.

In synaptic, do a search for 'hamster'. Install hamster-applet, then right click on your panel and select 'add to panel'. It'll appear in the list of available applets as 'time tracker'.
