### Post by akakingess on 2010-01-12
> **deathseek said:**
> i want to change my username.. can anyone help me with that?...:popcorn:

Personally, I would copy everything from your current home folder (including the hidden files, which you can see by hitting CTRL+H while in the home directory and backup all of that information to another location, and then create a new user with the username that you would prefer and copy all of that same info back into the new username's home folder and all should be as it was with the previous username, just with a new user. One of the only problems I could foresee would be if you have a lot of scripts that are written specifically for the previous username, but in that case you should have still just used the generic ~/ for your home folder so there shouldn't be much of an issue. Then you can get rid of the old user once you have added the new user to the proper groups, i.e. root, admin, sambadmin (if you use it), etc.
