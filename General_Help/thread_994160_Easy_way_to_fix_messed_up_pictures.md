### Post by sedawk on 2008-11-26
Hello!

Either there was a problem during transfer, when copying the files, or
the hard disk (partition) where the pictures are now stored is broken (physically or logically).

As the computer doesn't handle pictures differenly then other files, I
think all your files are broken, not only pictures.

It might be a good idea to run a backup before trying to check or repair
the data. What file system are you using? You can try to check
if the file system is okay with "fsck" tool.
