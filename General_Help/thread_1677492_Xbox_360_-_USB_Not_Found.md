### Post by GlitchKing on 2011-01-28
When copying files like photos/videos to your USB from Ubuntu 10.10 it will not show up on the Xbox 360 console as found.

I think this has to do with the whole FAT or NTFS thing. When you make the files through a Windows OS the files show, yet when you use Ubuntu to copy they will not show up.

Please help me as I am no longer using Windows (I've went to using Ubuntu solo).

Temporary fix:

Using Xbox 360

Navigate to My Xbox > System Settings > Memory

Select Memory then find your USB drive. (Mine showed as Memory Unit)

Format your drive here and select only to use 256 MB or more depending on what you'll need.

Plug USB back into Ubuntu and simply copy the files into it.

USB should now show on your Xbox 360 dashboard.

### Post by GlitchKing on 2011-01-28
Found a fix and hope this helps.

Navigate to System > Administration > Disk Utility

Find your USB drive to the left under storage devices and click it.

Under _V_olumes click Format Volume.

Change type to FAT

Click Format and wait for it to format.
