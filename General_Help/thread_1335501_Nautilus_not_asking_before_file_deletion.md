I've just done a clean install of 9.10 amd64. In Nautilus, under file management preferences, the option "Ask before emptying the garbage bin or deleting files" is checked, but when I delete a file or files, I don't get asked, it just happens. 

If I choose to empty the garbage bin then I do get asked. I'd also like to get prompted before a file gets deleted - it's quite possible to accidentally hit the delete key instead of home or end.

### Post by t0p on 2009-11-23
Do you mean you want to be asked to confirm before a file is *deleted permanently*?  That is the option you can select in Edit > Preferences > Behavior.

If you want to be asked to confirm before a file is moved to Trash: I think you're out of luck; I can't find that option anywhere.  But there's no need to worry: if you accidently move the wrong file to Trash, you can always restore it from the Wastebasket window.  So you're not going to lose anything through accidental deletion, as long as you select *Ask before emptying Deleted Items folder or deleting files permanently* under Edit > Preferences > Behavior.
