### Post by grahammechanical on 2013-06-21
Ubuntu 13.04 + Unity does not need Compiz Configuration Settings Manager (CCSM) to be installed to change the setting for hiding the Launcher. Go to System Settings>Appearance>Behaviour tab and autohide can be switched on and the reveal location (the place to put the mouse pointer) can be altered from left side to top left corner and the reveal sensitivity can be adjusted also. It is here that we can enable workspaces. One workspace is the install default. System Settings icon is installed into the Launcher by default.

CCSM and Unity Tweak Tool are both in the Software Centre. They do a similar job but there may be variations between the two. 


If you install another desktop through the software centre you can load into that desktop at the log in screen. It will become the default desktop until you log into another desktop. There is no need to remove Unity. If you prefer one of the flavours of Ubuntu then I would recommend installing that flavour from the beginning. Some people have got into a mess by replacing Unity and then wanting it back. It is better to have a preferred flavour and add and remove desktops through the software centre then trying to change a flavour into something else and then revert back.
