### Post by Giblet5 on 2009-10-22
Dogtail (python-dogtail is the package) can make gui testing pretty easy.

It all depends on what you're really trying to do.

There are a LOT of tools for testing, but there's no such thing as "automated test software" in any real sense - not on Linux, Unix, MVS, VMS, Windows, or Mac; you still have to develop test cases and script the test implementations...
