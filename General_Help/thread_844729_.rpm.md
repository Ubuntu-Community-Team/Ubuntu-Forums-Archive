### Post by iaculallad on 2008-06-29
First thing first: By default, Ubuntu cannot read RedHat's package RPM so you need to convert it to a format readable by Ubuntu (Debian). You need to install the alien application.

```
sudo apt-get install alien
```

After you had sucesfully installed alien, drop to your terminal and convert the RPM file (change directory to the location of your RPM file)

```
sudo alien -k name-of-your-rpm-file.rpm
```

To Install:

```
sudo dpkg -i name-of-created-deb-file.deb
```

### Post by rossjman1 on 2008-06-29
RPM is not an archive type like rar or zip are. RPM is an installation file for programs for a different type of Linux distribution. Ubuntu used something like this, but instead of RPM, Ubuntu uses DEB.
Goto System > Administration > Synaptic to install AdobeReader.
