### Post by coffeecat on 2011-01-16
@james_mcl, there will be no particular hurry for you after the end of April. You will, of course, get no more updates and security patches but Hardy will still be usable. And if you want to install any new apps after eol, you can do so by editing your sources.list to use the [noparse]old-releases.ubuntu.com[/noparse] server. This link is for upgrading, but it tells you how to include the old-releases server:

[https://help.ubuntu.com/community/EOLUpgrades](https://help.ubuntu.com/community/EOLUpgrades)
