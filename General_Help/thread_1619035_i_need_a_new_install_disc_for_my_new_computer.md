### Post by Mark Phelps on 2010-11-11
If by "install disk" you mean an Ubuntu desktop CD, you get those from various websites. They are downloaded as an image file (xxx.iso) which you then burn to CD.  

If you still have the image file around from a previous download, you can just stick in a CD and burn a new copy.  

If you don't have the image file, you will have to download a new one.  Go to the link below: 

[http://www.ubuntu.com/desktop/get-ubuntu/download]("http://www.ubuntu.com/desktop/get-ubuntu/download")
