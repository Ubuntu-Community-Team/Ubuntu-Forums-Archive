### Post by johngreth on 2010-03-18
It is so slow because the graphics can't travel across the network fast enough. It is faster with a wired only connection. Wireless is terribly slow. I managed to get used to it when I used it to manage my server, but I reinstalled and have a command prompt on the sever again. That is much faster over the network because there's no graphics, just text. There may be some crazy way to minimize the graphics data sent to make it faster, but I've never seen anything like it.
