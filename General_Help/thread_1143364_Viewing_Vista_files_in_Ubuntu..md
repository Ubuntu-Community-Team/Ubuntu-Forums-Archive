### Post by 3Miro on 2009-04-29
Where are the files when he is logged under windows. Double check the user name. I would boot under windows and double-check the exact directory where the files are. He might be signing with a different name under Vista or something.

Also make sure the files are on that drive, I know XP can point My Documents to be located on a different partition altogether.
