### Post by wildmanne39 on 2011-09-08
Hi, as haqking stated install the restricted packages for media play back then you can use banshee or rythmbox to play music and vlc or gnome movies player or one of many others to watch movies.

Also for window programs that you need that do not run in wine you can install windows in virtualbox this will allow you to run ubuntu and windows at the same time so you do not need to reboot, however the limitation is that virtualbox does not play games very well but wine usually handles games pretty good but not all.

Here is a link for virtualbox that explains what it does and how to install and use it.
[http://www.virtualbox.org/](http://www.virtualbox.org/)
