### Post by fhmanas on 2008-06-15
Most of the configs I got from browsing through the forums, in any case, here is a list of links of how I got things working.

[B]
Hamachi[/B]

Go to [http://www.logmein.com/hamachi](http://www.logmein.com/hamachi) and follow instructions to install it, pretty straight forward.

If you installed Firestarter, you have to do the following (exerpt from this [link]("http://ubuntuforums.org/showthread.php?t=135036"))
> 
*NOTE FOR FIRESTARTER USERS*
If you use the Firestarter firewall, then you must add these two lines to your '/etc/firestarter/user-pre' file to accept connections on the Hamachi interface:
Code:

$IPT -A INPUT -i ham+ -j ACCEPT
$IPT -A OUTPUT -o ham+ -j ACCEPT

Then restart Firestarter:
Code:

sudo /etc/init.d/firestarter restart





Go to Neohide [http://www.neohide.com/automatic-start-up-hamachi-for-ubuntu-feisty-fawn](http://www.neohide.com/automatic-start-up-hamachi-for-ubuntu-feisty-fawn) to get it to start when you open your computer.  





NOTE: The script can be modified to start pretty much anything upon boot up and not logged in.

[B]
FreeNX[/B]

Here is a pretty straight forward HowTo on FreeNX works in Gutsy/Hardy [http://ubuntuforums.org/showthread.php?t=620057&highlight=freenx+hardy](http://ubuntuforums.org/showthread.php?t=620057&highlight=freenx+hardy)

If you are new to NX Tech, try out NoMachine's NX at [http://www.nomachine.com/download-package.php?Prod_Id=5](http://www.nomachine.com/download-package.php?Prod_Id=5) which is a lot easier to configure but limited to 2 sessions only.

With hamachi, you can skip the part of port forwarding and use your Hamachi private network to connect.  

NOTE: Since FreeNX will open a session, you will not be able to shut down your remote computer.  If you want to do this, you must setup a VNC session.  In my experience, VNC through FreeNX is slow, might as well do straight VNC (through hamachi, it's secure already) but this is still slower.  What I do is install vncviewer in the remote machine, open a session using FreeNX, open a terminal and do a vncviewer ubuntu:0 and voila you control the remote desktop, this seems faster than either of the 2 previous solutions.


**PulseAudio**

Here is a good HowTo for PulseAudio, [http://ubuntuforums.org/showthread.php?t=789578&highlight=pulse+audio+skype](http://ubuntuforums.org/showthread.php?t=789578&highlight=pulse+audio+skype) which fixes Skype also since Skype does not normally play well with PulseAudio 

**Webcam on Skype**

Since some vimicro webcams don't work with Skype and works with Ekiga, I found this tip at [http://www.ubuntugeek.com/tip-getting-your-webcam-to-work-in-ubuntu.html](http://www.ubuntugeek.com/tip-getting-your-webcam-to-work-in-ubuntu.html) that fixed it

**IPod**

I "sync" with Rhythmbox, but I had problems with Banshee not recognizing my iPod, which somehow (not sure) was solved by this link [https://bugs.launchpad.net/ubuntu/+source/banshee/+bug/139226](https://bugs.launchpad.net/ubuntu/+source/banshee/+bug/139226) particularly the part where you have to be part of the group haldaemon.


**VirtualBox**

I use VirtualBox PUEL as the OSE version has issues with USB, however the PUEL version is not without problems with the USB.  this was solved with this link [http://www.arsgeek.com/?p=2776](http://www.arsgeek.com/?p=2776)

For bridging techniques, here is a pretty easy way [http://blog.zrmt.com/2008/04/18/virtualbox-bridged-networking/](http://blog.zrmt.com/2008/04/18/virtualbox-bridged-networking/)

**WINE**

Pretty straight forward install of WINE from WineHQ repositories, to install Net 2.0, use winetricks, I installed Photoshop CS2 with wine-doors, license required.

[B]
UserFul/Multiseat[/B]

Installed Userful from [http://www2.userful.com/products/downloads/free-2-user](http://www2.userful.com/products/downloads/free-2-user), pretty straight forward install and reboot although limited to 2 seats.

If you are more courageous, here is a link to follow, [http://netpatia.blogspot.com/2008/02/multiseat-computer-with-ubuntu-804.html](http://netpatia.blogspot.com/2008/02/multiseat-computer-with-ubuntu-804.html)

**CONKY**

Installed conky from repos and chose a config from this link [http://ubuntuforums.org/showthread.php?t=281865&highlight=Conky](http://ubuntuforums.org/showthread.php?t=281865&highlight=Conky)

To autostart conky when I log in, follow the steps in this link [http://ubuntuforums.org/showthread.php?t=386078&highlight=Conky](http://ubuntuforums.org/showthread.php?t=386078&highlight=Conky)


**Skype**

Install skype from Medibuntu repos and have it autostarted when I log in by adding in System -> Preferences -> Sessions

**Canon Printer**

I had to use Turboprint, as so far the project ddiwrapper is still far from being ported to debian but is currently being developed to be part of the wine api's


Well, that's it, it's not really a HowTo, but from multiple sources, i was able to make my Hardy do what I need it to do.
