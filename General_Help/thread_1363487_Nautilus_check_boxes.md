### Post by chewearn on 2009-12-24
This might not be answering your question directly, but I am always partial to two pane file managers.  In Windows, I use FreeCommander; while In Linux, I use Krusader.

Krusader is a KDE app; don't match the Gnome UI, but I prefer function over form.  Another two pane file manager in Linux is Gnome Commander.

Why I brought up two pane file managers?  Well, you can multi-select files by tapping on the space bar.
