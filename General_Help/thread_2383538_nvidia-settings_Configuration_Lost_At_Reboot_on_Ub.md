### Post by markackerman8@gmail.com on 2018-07-23
A Simple Solution to a long time problem - WooHoo! :p

I now able to save nvidia settings across reboots.

The key to making the settings stick is to go into the System: Preferences: "Startup Applications Preferences" menu,  ...

and then select 'add an application' and just give this command line, and name it anything you want:

```
nvidia-settings --load-config-only &
```

This will load the hidden file /home/YourUserName/.nvidia-settings-rc on boot, setting the colors brightness saturation etc. into the nvidia card. 

Always trying to help, Mark :P
