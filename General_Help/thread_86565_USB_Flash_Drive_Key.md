### Post by riggsd on 2005-11-05
[QUOTE=Luffield]Actually there is a slight difference in the way Ubuntu and Windows handle my USB Key Drive - when I delete a file in Windows it gets deleted, but in Ubuntu it is moved to a folder called "luffield.ubuntu-trash" or something like that, and does not really get deleted.[/QUOTE]

System -> Preferences -> File Management -> Behavior

Enable "Include a Delete command that bypasses Trash". Delete files with a right-click and 'Delete' instead of 'Move to Trash'.
