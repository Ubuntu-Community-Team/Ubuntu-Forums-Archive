### Post by nini9012 on 2009-12-07
I haven't really had this problem, but I'll suggest a few things anyways. Under effects settings, do you have blur windows or motion blur checked. This could cause a problem if you don't have a great graphics card. I don't even use the blur window option because it makes my desktop super slow. If it isn't related to the effects settings it could possibly be a hardware issue. You could also try changing the refresh rate for your monitor by going to System-->Preference-->Display.

### Post by Sam on 2009-12-07
You must install [compizconfig-settings-manager](apt:compizconfig-settings-manager) if it's not already done, then disable "Motion blur" in System > Preferences > CompizConfig Setting Manager.
