### Post by JustinR on 2010-10-14
> **Myoga- said:**
> Up =/

Hi,

The Linux Kernel included with Ubuntu includes drivers for almost every type of computer Ubuntu will run on.

So yes, you can use an external HDD on different computers. All computer information is collected on boot and isn't hard-coded anywhere in Ubuntu and leaves Ubuntu when you shut it down.

And yes, you should be able to use a Windows oriented AntiVirus software to scan stuff.

And just a tip - it might be easier just to install Ubuntu on something more portable like a flash drive - both will work either way! Have fun.
