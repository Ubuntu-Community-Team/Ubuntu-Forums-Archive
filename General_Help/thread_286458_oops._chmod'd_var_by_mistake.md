### Post by funkyade on 2006-10-28
Thanks for the help. All sorted now.

Just for the record here's the solution -

make a VERY strong espresso

use the rescue mode off the server install ISO (any live CD with a rescue mode will do) and mounted my / partition. Then I did -

```
#> sudo chmod -R u=rwxs,g=rxs,o=rx /var
#> sudo chmod 0700 /var/run/sudo
```

set a root password so you can 'su' in the future!

and reboot. All works fine now.:-)

phew....
