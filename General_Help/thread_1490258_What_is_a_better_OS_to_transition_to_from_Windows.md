### Post by .nedberg on 2010-05-22
Give Ubuntu and Kubuntu a spin with the Live CD and see what you like. If you have a old machine with less RAM/CPU-power, also try Xubuntu. Remember that they will all run MUCH better when they are installed!

Try them out and do some basic configuring to better get the feel of what is right for you!

For me, it was Kubuntu. KDE generally gives the user more choices, and you have to press OK/Apply to make changes. Gnome have less options and they apply immediately.

I have not found one thing that can be done in Gnome that cannot be done in KDE (or vice versa). Gnome apps run fine in KDE and KDE apps run fine in Gnome (they might look a bit out of place though).
