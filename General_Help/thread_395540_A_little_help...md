### Post by arbulus on 2007-03-28
I read a review once (but can't for the life of me remember where it was) that looked at 64bit Ubuntu, and the fellow tried to get the 32bit flash plugin to work, and was never able to do so successfully.  I run on Ubuntu on a PowerPC iMac G5, and there's no Adobe support for that at all, so I can't really be of much help. 


But for must have apps, I guess it would depend on how you plan to use your box (i.e. desktop or server).  I'll try to throw out a few general ones here.

gkRellm is a great system monitor.

I prefer Thunderbird as an email client over Evolution, because I really don't need the whole Outlook-like functionality.

Songbird is a great media player, it has incredible amounts of functionality, and can sync an iPod if you have one.  Some prefer Amarok, but I think that has alot to do with desktop environment preferences in general. (Amarok being standard in KDE, but not Gnome)

For desktop customization, gDesklets is a handy app.  It gives you the ability to put OS X style widgets on your desktop.  

I always like to encourage newcomers to explore different desktop environments.  I started with Gnome, but I've played around with KDE, XFCE, and Fluxbox.  You can install all of these on the same machine and choose which one you'd like to use at your login screen under "Sessions".  

gkRellm, gDesklets, and Thunderbird are available in the Add/Remove software option, or in Synaptic Package Manager.  You can also use the command line if you prefer.  I think you'll have to visit Songbird's website to download.

As for the different desktop environments, I'm sure there are much better posts in the forums here that can explain installing those better than I can.  There's also [http://www.ubuntuguide.org](http://www.ubuntuguide.org) which has been an indispensible tool for me.  It has a lot of tips and tricks and covers and incredible amount of information. 

Hope this helps!
