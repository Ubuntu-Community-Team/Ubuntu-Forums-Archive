### Post by dvlchd3 on 2010-04-30
Try here: [https://wiki.ubuntu.com/Artwork/Incoming/Karmic/Backgrounds/Default](https://wiki.ubuntu.com/Artwork/Incoming/Karmic/Backgrounds/Default)

I believe the Karmic Rock is the closest to previous Ubuntu backgrounds, they change for every release.  Since Canonical decided to re-do all the graphics work, the "purple crap" is the new Ubuntu style.  See this for more information: [http://fridge.ubuntu.com/node/1991](http://fridge.ubuntu.com/node/1991)

I personally don't mind the new look.  I think it is clean looking and simple, that is really all I care about.  I will say, I don't dig the color purple by any means, and have changed the desktop background, but the window and menu appearance I embrace as a good change.  I also think it is a good idea to have different color sets to distinguish between Ubuntu (FOSS stuff) and Canonical (proprietary-pay for stuff).

### Post by jfloydb on 2010-04-30
For anyone who might be interested: Changing the background was easy.  I simply made a 1920 x 1266 px .jpg picture and filled it with Ubuntu Brown. I then used the picture as my Background. You cannot (or at least I couldn't) add a graphic file to the Background folder; but you can add a graphic file to the Pictures folder and, then, simply choose it when Adding a Background.
