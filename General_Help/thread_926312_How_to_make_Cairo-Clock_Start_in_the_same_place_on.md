### Post by larryfroot on 2008-09-23
You can find screenlets in synaptic and install them from there. One of the screenlets available in the app is the cairo clock. Tick boxes on bottom left to start it and to start it every time you log on. If you right click on it a context menu comes up. tick 'lock' for it to keep the same position every time you log on, and 'sticky' to make it available to your other desktops. If you enable the widget layer in compiz settings manager and the widget option in the context menu for the clock, you can show / hide the clock (and any other enabled widget) by pressing F9.
