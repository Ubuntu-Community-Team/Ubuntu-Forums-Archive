### Post by BathroomNinja on 2006-01-14
Hylafax is a great fax *server*, but I don't know if it would work well as a 'print to fax' solution.  

Other users here on the forums have reported success using efax with efax-gtk.

Both should be in universe.  the efax-gtk is a GUI front end for efax.  It gives you the print to fax feature you are looking for.

[http://efax-gtk.sourceforge.net/](http://efax-gtk.sourceforge.net/)
