### Post by pqwoerituytrueiwoq on 2018-04-28
sounds like you need to set a dns server ([See how](https://unix.stackexchange.com/questions/128220/how-do-i-set-my-dns-when-resolv-conf-is-being-overwritten#answer-128223) * i would use nano not vim)
there is a easy way to edit your sources list file, these 2 commands MUST be done in order
[FONT=courier new]sudo sed 's/us.archive.ubuntu.com/91.189.91.23/' -i /etc/apt/sources.list
sudo sed 's/archive.ubuntu.com/91.189.88.162/' -i /etc/apt/sources.list[/FONT]

as for reverting this (order does not matter):
[FONT=courier new]sudo sed 's/91.189.91.23/us.archive.ubuntu.com/' -i /etc/apt/sources.list
sudo sed 's/91.189.88.162/archive.ubuntu.com/' -i /etc/apt/sources.list[/FONT]

You could do the smart thing and make a backup copy 1st
sudo [FONT=Courier New]cp /etc/apt/sources.list /etc/apt/sources.list.bak # create backup
sudo cp /etc/apt/sources.list.bak /etc/apt/sources.list # restore backup[/FONT]
