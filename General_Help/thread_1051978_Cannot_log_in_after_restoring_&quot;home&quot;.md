### Post by timcredible on 2009-01-27
sounds like the UID for that user is wrong.  by that, i mean that when you added that user to ubuntu, it probably got the UID of 1000, and then you restored some files from a user with the same name, but a different UID.  you probably need to make the UID match, try ```

su chown -R /home/kjt kjt
su chgrp -R /home/kjt kjt

```
