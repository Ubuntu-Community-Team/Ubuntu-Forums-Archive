### Post by -kg- on 2011-06-19
The first thing of concern to me was that you listed no GPU card, so assuming that you were intending on using the on-board graphics chipset, I checked it out.

The unit uses the Intel GMA 3150 chipset.  I've read of a few problems with the chipset from my research, though none really recently.  If you do have problems with the graphics, my recommendation would be to buy an add-on graphics card (nVidea generally has the best support, if I'm not mistaken)...preferably one from the [Linux Hardware Compatibility List.]("http://linuxhcl.com/")  If you have no problem with it, you should be good to go.

### Post by c-1000 on 2011-06-19
that mini-itx board will* probably* fit ok in the case you listed, but...

if the motherboard/cpu you listed is the one you really want, i would get an itx case for it.
alternately, get an atx motherboard for the case you listed (this is what i recommend).

the case you listed has a fan, you probably wont need another one unless youre using multiple hard drives, or have a big graphics card.

i just built a system a couple of months ago; intel i3 and integrated graphics on an asus atx motherboard, everything worked 100% right out of the box with lucid, maverick, and natty.
