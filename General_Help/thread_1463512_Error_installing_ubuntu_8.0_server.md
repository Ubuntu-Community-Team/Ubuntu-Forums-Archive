### Post by 3rdalbum on 2010-04-27
Saying "Ubuntu 8" is not helping us, sorry.

There is 8.04, 8.04.1 and 8.10. **8.04.1** is the long-term service version of Ubuntu, with patches and bugfixes.

Have you tried using the "cheat codes" that you can add to the kernel boot line? I think you press F11 (or whatever F key it says to press at the bottom of the screen) while at the first menu.

Alternatively, try using the version of Ubuntu about to be released in three days - Ubuntu 10.04. It's also a long-term service version so it's appropriate for servers, and the release candidate is available to download right now.
