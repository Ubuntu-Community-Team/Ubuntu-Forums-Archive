### Post by finer recliner on 2008-08-28
if you used sudo for another command within the last 10 or 15 minutes (i dont remember what the default time is) you dont have to reenter your password. this makes it easier to enter multiple sudo commands in a row.

and you should be using gksudo to launch GUI's from the command line, not sudo. see here for more info:
[http://www.psychocats.net/ubuntu/graphicalsudo](http://www.psychocats.net/ubuntu/graphicalsudo)
