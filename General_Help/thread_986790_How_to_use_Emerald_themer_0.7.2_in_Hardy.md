### Post by trendyabinash on 2008-11-18
Hey emerald themer worked
I followed the following steps:-
go to system > preferences > Sessions
click 'add'
type
name: emerald
command: emerald --replace

I double clicked the theme which I wanted to apply and then pressed alt+f2
and typed emerald --replace and it changed
