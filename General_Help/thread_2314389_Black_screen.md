### Post by grahammechanical on 2016-02-21
As well as the hardware specification I am looking for the following information

1) Do you see a laptop splash screen?
2) Is Ubuntu the only OS on the machine? If Ubuntu is the only OS we do not get a Boot menu (Grub) but we can call it up by pressing Shift when we see the motherboard splash screen.
3) If there is a Grub boot menu what happens when Ubuntu is selected to be loaded? What happens when the other OS (if any) is selected to be loaded?
4) At the Grub boot menu what happens when Advanced options for Ubuntu is selected and then a Linux kernel with recovery mode is also selected? Do you get a recovery menu?
5) At the recovery menu what happens when Resume is selected? Does that get to a login screen?

In other words, at what point in the whole process does the screen turn black while the rest loads?
