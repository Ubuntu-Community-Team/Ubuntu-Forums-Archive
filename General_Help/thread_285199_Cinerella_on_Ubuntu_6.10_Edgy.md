### Post by Bartje on 2007-02-27
The easiest way for me was:
go to:

[http://www.kiberpipa.org/~minmax/cinelerra/builds/]("http://www.kiberpipa.org/~minmax/cinelerra/builds/") and then to the folder for your system (athlon, pentium4, pentium or sid), and download the packages. Also install the other packages needed like libguicast, libmpeg3hv-dev, libquicktemehv-dev, libsndfileh-dev, etc... install those first before you try cinerella itself. They are needed to get the install done.

You also need libavcodec1, but for me it worked with libavcodec2 too. I've download that one from somwhere, but sadly enough I don't know anymore from where.

If someone else knows.... 

good luck
