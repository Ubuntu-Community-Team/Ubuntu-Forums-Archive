i was wondering if there was a way to temporarily stop gnome-panel like i do nautilus. ( nautilus -q ). any ideas ? (then maybe on how to re-start gnome-panel)

---

### Post by fooman on 2008-10-21
go to system > preferences > sessions > current sessions tab

find gnome-panel in the list and highlight it.  click "remove"

to get it back,  press the alt-f2 keys.  in the run dialog box that pops up, type:

```
gnome-panel &
```

click "run"
