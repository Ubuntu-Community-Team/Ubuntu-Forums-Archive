### Post by Rhubarb on 2008-10-15
OOo3 can indeed open up pdfs.
But you need a plugin that lets you do so (it'd be nice if it was included by default eh?)

Grab it here:
[http://extensions.services.openoffice.org/project/pdfimport](http://extensions.services.openoffice.org/project/pdfimport)

You can install it in OOo3 thus:
Tools --> Extension Manager ...
Then click on the "add..." button and tell it to open up your plugin.

Then you can open up pdfs in draw.  :D
