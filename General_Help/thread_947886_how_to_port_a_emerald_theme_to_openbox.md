### Post by cammin on 2008-10-15
It uses config files with the option of xpm images for buttons.

The problem you're going to have is that you can't change the thickness of the title bar. You might be able to make the buttons large enough that they overlap the top border, but they'll be so large that it goes against the minimalist aspects of the whiteline theme.
