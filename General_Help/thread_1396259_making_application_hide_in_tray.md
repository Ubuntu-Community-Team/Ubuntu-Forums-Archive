I have an application that has no options for minimizing the application to the tray. Is it possible to do this for any application of my choice?

---

### Post by hemimaniac on 2010-02-01
sudo apt-get update

sudo apt-get install alltray

sudo ldconfig

Menu > Accessories > Alltray > click on the app you would like sent to the tray
