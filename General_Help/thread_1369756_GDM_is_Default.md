### Post by Brandon Williams on 2010-01-04
The 'Login Screen Settings' dialog for the new GDM only allows you to customize whether or not autologin is used. If autologin _is_ used, the dialog also allows you to configure who will be automatically logged in and after how long.

The new default theme is quite plain in comparison to the possibilities with the old gdm. I don't know what the background image looks like for Ubuntu (I run Xubuntu), but you might be describing the expected theme. It would be easier for people to answer that if you posted a screenshot of the login screen. The post I looked to previously tells you how to change the background image, the GTK theme used, and the icon set (among other things).
