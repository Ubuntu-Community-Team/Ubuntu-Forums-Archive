An idea would be to use [clonezilla]("http://ubuntuforums.org/www.clonezilla.org")

Download and burn .iso then boot system to the CD, clone your windows partition/s or disk as an image to another location (external HDD or seperate physical internal disk) then you can clone that image to a VM in virtual box when you get Ubuntu up and running, also it is a back up of your current build.

edit: has 11.04 installed already

Boot up, install virtual box and set up your old winodws build in it.

That way you get the best of both worlds ;-)

before doing any of this always have a backup of personal data though.
