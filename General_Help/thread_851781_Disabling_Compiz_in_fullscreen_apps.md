### Post by k3kiaze on 2008-07-07
Hi,

When I run programs in fullscreen (e.g Neverwinter nights 2 with WINE). The screen edges on the left and right, still belong to Compiz, so the application itself looses control of it (really annoying). 

If I set 'visual effects' to 'none' this doesn't happen. But I can't keep doing that everytime I want to run a fullscreen app that requires access to screen edges. So my question is:

a, Can i disable screen edge access to Compiz to avoid the above problem?
(or)
b. Is there a simple way to disable Compiz when I run fullscreen apps?

P.S: The reason I don't want to set 'visual effects' to 'none' is then I have to reconfigure my Compiz settings (even more annoying)

### Post by k3kiaze on 2008-07-11
This does NOT only apply to wine programs, but to any application I run e.g: Firefox, Mplayer etc. The Screen edges [left and right only] change my mouse pointer, and take away control from my apps. If visual effects = none, then this problem is fixed. Pls help me guys. thanks.

Update:

O.k, so I've found a workaround to what I was asking for: Create CompizConfig Profiles, I select No visual effects and then when I want my settings back, select 'Extra Effects' and then my profile. I only waste about 2 mins now when i want my apps to have access to screen edges.
