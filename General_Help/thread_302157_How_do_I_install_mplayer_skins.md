I downloaded and extract some mplayer skins to my desktop. I ttried dragging and dropping into /usr/share/mplayer/skins and it didnt work. can someone please help me with the command to transfer the folder of skins into the mplayer directory? my current mplayer has just the one default skin.

---

### Post by taurus on 2006-11-18
You need to be root to move stuff to /usr/share/mplayer/skins by using sudo from a command line or "gksudo nautilus" for a graphical sudo.  Otherwise, you can just install the skin to your own home directory, ~/.mplayer/Skin.
