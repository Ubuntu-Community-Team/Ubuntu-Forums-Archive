### Post by hoppings on 2011-02-26
A friend installed QCad. All seemed well but he cannot find the app in the Applications menu under any category. Strange thing is, the Software Centre shows that it is installed. How can he run it if it does not appear in the Apps menu? Can it be run from the command line? Is this a known bug?

---

### Post by howefield on 2011-02-26
Try typing qcad in a terminal and press enter.

You can create a menu launcher for the application via System > Preferences > Main Menu if needs be.

(Occasionally a logoff/logon is required before some applications show in the menu.)

### Post by seawolf167 on 2011-02-26
You can also hit [ALT-F2] then type in the application you want to run

Additionally, to edit the menus, you can right-click the Applications Menu, then select Edit Menus and add a menu entry for your missing program
