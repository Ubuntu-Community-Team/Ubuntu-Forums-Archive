### Post by mcduck on 2010-11-11
> **Arminius said:**
> I've got the galaxy live wallpaper but I'm wondering if anyone know's of other ones?
like the one that changes by a night day cycle?

Sure, there are some available in gnome-look.org (just search for "animated wallpaper", I found way more .xml wallpapers for Gnome than I'm willing to link here...), and creating one of your own isn't actually that hard either (just open one of the existing wallpaper .xml files for reference. They are basically just a list of images to use and time delays for how long to show each one or fade to next one.)

There also seems to be at least couple of GUI tools for creating the .xml files for you if you prefer that over hand-editing the files.
