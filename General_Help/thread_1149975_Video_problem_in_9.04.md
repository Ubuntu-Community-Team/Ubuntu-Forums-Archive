### Post by maheshasolkar on 2009-05-05
The log had information on the graphics hardware I was seeking.

Looking at the log, there are quite a few warnings, but I can't say for sure what is causing the crash.

It appears like you are using the EXA acceleration method. I'd suggest you try to use UXA and see if it makes any difference. To use UXA, add the following in your /etc/X11/xorg.conf (If you already have a Device section in your xorg.conf, just add the AccelMethod line):

```

Section "Device"
        Identifier    "Configured Video Device"
        # ...
        Option        "AccelMethod" "uxa"
EndSection
```

After updating the xorg.conf file, log out and log back in, so that the xserver restarts. Alternatively, you can run the following in terminal to restart the xserver:

```
  sudo /etc/init.d/gdm restart
```

UXA is known to work on GM965/GL960, at least according to:

  [https://wiki.ubuntu.com/X/UxaTesting](https://wiki.ubuntu.com/X/UxaTesting)
