### Post by Crafty Kisses on 2009-02-19
Well he is using Beryl, which has been merged into CompizFusion, which is installed by default, you can install the Compiz settings manager by running the following:
```
sudo apt-get install compizconfig-settings-manager
```
Once you have that installed you can pick and choose what effects you want. For the dock he is using Kiba dock, which I think is discontinued, but some people have gotten it to work with the newer releases of Ubuntu, look at this thread: [http://ubuntuforums.org/showthread.php?t=554127](http://ubuntuforums.org/showthread.php?t=554127).
