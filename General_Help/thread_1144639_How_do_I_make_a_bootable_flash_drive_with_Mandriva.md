### Post by razy60 on 2009-04-30
You can use unetbootin to install a iso/cdimage file from your own source, i.e download Mandriva 2009 .iso onto your PC, and use unetbotin to create the bootable flash drive from that source.
It explains how on there site.

### Post by Bradj47 on 2009-04-30
I made a bootable Ubuntu 9.04 flash drive using the USB Startup Disk Creator under System -> Administration -> USB Startup Disk Creator. (On the latest 9.04 Jaunty edition) I would guess it would work for Mandriva too.

### Post by BigSilly on 2009-05-03
Oddly enough it doesn't! 

I've just tried to do it myself for my daughters EeePC, and where the Ubuntu USB Creator in 9.04 didn't work, Unetbootin did. I was told it could be a faulty iso, but the md5 sum is correct. Ubuntu's USB Startup creator said the iso I was trying to use was not a desktop install iso, but it most certainly was.

Anyway, Unetbootin is definitely just as easy to use, so I recommend you give that a shot.
