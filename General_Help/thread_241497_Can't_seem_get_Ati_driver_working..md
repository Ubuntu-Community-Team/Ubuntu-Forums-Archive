### Post by Ramses de Norre on 2006-08-22
Sorry I see you're using some dual view config which I don't know, so my post probably wont help..


```
sudo gedit /etc/X11/xorg.conf
```
scroll down to the device section of your video card and change ```
Driver "ati"
``` to ```
Driver "fglrx"
```
Restart X and you're new driver should be loaded.

### Post by majesticturkey on 2006-08-22
Try:

```
sudo ln -s /usr/lib/dri /usr/lib/xorg/modules/dri
```

If you're using the drivers from ATI's website instead of the repos, make sure you blacklist the fglrx module from linux-restricted-modules.

```
sudo gedit /etc/default/linux-restricted-modules-common
DISABLED_MODULES="fglrx"
```
