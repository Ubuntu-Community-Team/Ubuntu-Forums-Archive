Hi all

Today i'm going to buy a used laptop and it has been a while since i bought a used one. I tried to google for a guide on what to check but most of them out there are mediocre at best. So I decided to make my own list and publish it in order to help people in the same situation in the future.

And in order to clarify before going into the list, I've decided on the model so i'm past the choosing/screening process.



[LIST]
[*]make a RAM test
[*]make a cpu burn test
[*]test the 3d card
[*]test the USB/HDMI/etc ports on the computer
[*]make sure the charger is functional and has no loose parts
[*]check the screen for cracks, dead pixels and see how it displays every RGB color
[*]check the computer itself for cracks
[*]check the harddrive
[/LIST]


Is there anything else i've missed? if so, please fill me in :-D
