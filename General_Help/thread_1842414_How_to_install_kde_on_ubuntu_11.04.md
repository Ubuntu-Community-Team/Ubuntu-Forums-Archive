### Post by 2F4U on 2011-09-11
There is a KDE desktop meta package in the software repositories. I think it is named kubuntu-desktop. This provides the complete kubuntu software collection. There is also the plain KDE packages (don't know how it is called) which you can find by using the search function in software manager.

### Post by oldos2er on 2011-09-11
> **vikash_chandola said:**
> I want to use kde on my ubuntu 11.04. How to do it?

Correct. As was mentioned, installing kubuntu-desktop gives you the desktop environment plus various KDE apps. I believe the package kde-workspace gives you the desktop environment minus the apps. If you want everything KDE including the kitchen sink, install kde-full.
