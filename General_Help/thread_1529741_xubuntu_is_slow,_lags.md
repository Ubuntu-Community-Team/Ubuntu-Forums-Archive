### Post by ghors on 2010-07-12
I installed xubuntu 10.04 on a pc that has 768mb of ram and a 1.13gz processor. When i booted the cd up and tried installing it i got an unrecoverable error and it put me in a live desktop where i used the install xubuntu icon to install it. when it was installed xubuntu was very laggy and the cursor would randomly flash and disappear, i then checked the processor speed and it would fluctuate from 1% to 100% when i had no applications running. What would be the cause of this and could it be fixed?

---

### Post by ghors on 2010-07-14
I fixed this problem myself. My I had overclocked my processor a few months ago and I thought this might be the problem so I simply set it back to the default settings and voila Xubuntu runs fine!
