### Post by mcduck on 2008-07-20
Medibuntu repositories include some packages that are not available in Ubuntu's own repositories, usually because of licensing or patent issues or other restrictions. These include w32codecs, CSS library needed for watching DVD movies etc.

Also some packages include features that have been disabled in Ubuntu versions of the same packages, for same reasons. ffmpeg is one of them.
