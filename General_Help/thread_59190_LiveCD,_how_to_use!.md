### Post by hashimoto on 2005-08-23
1. Check that the md5sum of the downloaded iso file is correct
2. Burn the downloaded iso file on a cd as an _image_. Use Nero or whatever.
3. In bios chec that CDrom is the first boot media
4. Put the cd in, reboot your computer
5. Have fun!
