### Post by akarimco on 2007-06-16
I believe all Wubi really does is store Ubuntu's file system as a normal Windows-type file... in other words, there's probably some file called ubuntu.qqq that's several gigabytes, and when you boot into Ubuntu rather than Windows, it mounts "ubuntu.qqq" as a sort of virtual harddrive.  So you're kinda always using Wubi and yet kinda never... it's all fairly transparent.
