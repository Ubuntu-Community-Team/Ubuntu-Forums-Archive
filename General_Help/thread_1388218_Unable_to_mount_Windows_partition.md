Try two things:
1. Remove the extra space in the words 'Windows' and '7' in your fstab.
2. if the 1. doesn't work, try this:
```
sudo apt-get install ntfs-3g
```
then, replace the windows partition with this (do a backup before)```
/dev/sda2 /media/Windows7 ntfs-3g defaults,force 0 0
```

Good luck!
