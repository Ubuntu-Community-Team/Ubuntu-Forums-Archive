### Post by ciphermonk on 2007-07-01
It turns out that it was the TV output that was messing me over. Fixed it  by making the following changes to xorg.conf

First add a new Section for the TV Output

Section "Monitor"
           Identifier "TVOutput"
          Option "Disable" "true"
EndSection


and then in the Device Section add the following

Option "monitor-TV" "TVOutput"


That should fix it for you
