### Post by lazerking9 on 2011-08-12
it would help alot if we knew a little more about your system. So please post back with what you know about your CPU/HDD/RAM/etc...

*hint: if you can run linux (from your HDD or from live CD), go to terminal, and run ```
sudo lshw
```. If you are a little paranoid (like me), run ```
 sudo lshw -sanitize
``` instead (this makes it impossible to read stuff like IPs, Serial #s, etc. that may be listed)
