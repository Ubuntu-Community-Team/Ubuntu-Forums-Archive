### Post by sajinkoroth on 2008-10-03
Dictionary Application which comes default with ubuntu gives pronunciations for difficult words. You can configure it to use a large number of dictionaries by going to Edit > Preferences and then adding dict.org [ HostName : dict.org , Port : 2628, Transport : Dictionary Server ] as a dictionary source. If you need pronunciation for every word then you can run Cambridge Dictionary software under wine ( if you own one Cambridge Dictionary CD :) ).
