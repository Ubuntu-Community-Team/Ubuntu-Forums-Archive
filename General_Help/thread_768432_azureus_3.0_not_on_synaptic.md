### Post by nullmind on 2008-04-26
I haven't used Azureus in Ubuntu since Gutsy, but I believe it suffers from the package management issues of most Java applications. Firstly, I love java, so I'm not hating! Azureus likes to update and modify files using it's distribution system, and that doesn't seem to work well with it being owned by root and what not.

Long story short: install sun-java6-re and download Azureus, running in it's own directory (likely in your home directory). It's a big foobar, much like Eclipse 3.0 has became to install, but it'll likely work much better in the long run.
