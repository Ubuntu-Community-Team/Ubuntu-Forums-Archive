### Post by cdwillis on 2009-07-27
To start most programs you can type the name of the program into the shell (i.e. firefox). Some common shell commands to get you started:

man (read the manual entry for a command i.e. "man ls")
ls (lists the files in the current directory)
cd (changes to another directory i.e. "cd /home/charles/Desktop")
locate (finds a file i.e. "locate sources.list")

Reading the man pages can help you out a lot, but I find a lot of the times some of the information doesn't soak into my brain very easily. That pocket guide will probably help you out quite a bit.
