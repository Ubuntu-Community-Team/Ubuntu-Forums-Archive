### Post by JadenRosen on 2011-08-02
People, I was just watching a Youtube video about Google's Chrome OS.

And a comment there, struck me. That comment said "Ubuntu can do all of this and more..."

Suddenly I got to thinking... What if Ubuntu, made a Web-untu? We can already match Chrome OS's start-up time... All we would have to do is remove things like LibreOffice and such.

Plus, UbuntuOne has already nearly paved a way for us to create our own cloud away from Google's!

Now the main focus for this, is usability. The most amazing piece of user-friendly guruship came from Google's presenter.

He said, "since everybody already knows how to use a web browser, we just thought... Why don't we make our operating system, look like a web browser?"

If we can make every application inside of Webuntu show up on a Linux version of the Google Chrome new tab page, it would be the easiest to use Distribution of Linux ever.

Since Ubuntu would be able to do so much without any of the extra programs and such, but still has the ability to install them, why would we not do this?

It seems like easy money. Webuntu would be Google Chrome OS, with the ability to install main programs.

I am going to develop this idea, but I hope some of you guys out there agree with me that this is an amazing idea! So I am officially going to start the Webuntu Project. Anybody wanna help?

Thanks!

Jaden.
