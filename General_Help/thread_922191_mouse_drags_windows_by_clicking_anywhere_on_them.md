### Post by yazifeather on 2008-09-17
I just installed and configured compiz and am loving it so far. However I have a little problem. 

Normally I can only drag my windows around by grabbing the top bar. But now for some reason whenever I click anywhere on the windows i can drag it around. That would be ok if it didn't mean that I can't click on links or use the scroll bar because the little grabber mouse icon pops up. 

There must be some compiz setting that got changed somehow but I don't know what it is. Thank you for your help.

### Post by Uniq PhoeniX on 2008-09-17
Take a look at ccsm - Uncategorized - Move Window, and there the first setting. If it is just Button1, it means you will always drag with left mouse button. I have it set as Alt+Button1 (Default).

### Post by yazifeather on 2008-09-17
Thanks for the help :)

Turns out it was the move window value that was only button 1. I did have to learn the "metacity --replace" command to change that value though.
