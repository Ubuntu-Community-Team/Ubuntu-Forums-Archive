### Post by DarknessEnemy on 2008-07-11
If you want a GTK theme, the (system> preferences> appearance) should do it. But there are another kind of themes, The one that I use is Emerald Theme Manager, which works with most of the Compiz section of ([http://www.gnome-look.org/](http://www.gnome-look.org/)).

If you want to install those themes (compiz themes), you should get Emerald Theme Manager (I dont know if you should get the whole Compiz Fusion, but maybe you should, I recommend it, something could fail) and get the Compiz Fusion Icon (Both of them should be easy to get with Synaptic Package Manager, if you are using Ubuntu 8.04, if not there are a lot of How-to's on the internet). After you install everything You should start emerald theme manager, add the theme if you already download it and start Compiz Fusion Icon and a blue Tray Icon should appear, do a Right Click on the icon and on the "Select window decorator" menu select the option Emerald. If you already did the previous steps you should see the new theme.

Dont limit your theme there, you can even modify it. (Just dont distribute remaked work, consult the creator first, you shall modify themes for your own use) ;)

Its the only way I know, but doesn't take long.

Hope it works
