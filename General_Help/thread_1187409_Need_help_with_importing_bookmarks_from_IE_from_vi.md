### Post by entikryst on 2009-06-14
In I.E.  Click file> import/export >choose export to a file> choose favorites> choose which bookmarks to export> choose a location for the file to be saved> put it on a thumb drive and import to desired browser.

To import to firefox click bookmarks> choose organize bookmarks> choose import and back up> choose import html> pointed to the file you created with I.E.
