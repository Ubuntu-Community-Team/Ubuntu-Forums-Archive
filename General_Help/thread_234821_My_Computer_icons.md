### Post by ifokkema on 2006-08-12
If it doesn't show on Places > Computer, you can also bookmark the mount point, so that it will show up under Places.
Open the file manager, and browse to the mountpoint of the partition. Now type Ctrl-D or go to 'Bookmarks' -> 'Add Bookmark' in the Nautilus menu.
