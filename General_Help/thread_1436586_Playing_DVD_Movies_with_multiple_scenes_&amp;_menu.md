### Post by asmoore82 on 2010-03-22
I would highly recommend VLC media player, it's in the software center.

It defaults to skipping the menu and playing the main title of a DVD but
if you go to "Media -> Open Disc" you can specify whether menus are skipped.

### Post by mc4man on 2010-03-22
To make it the default for dvd's there are many ways, it's set in File Management 

Several ways to get to File Management - one would be open nautilus (home folder) and go edit -> preferences. You'll find the settings under media

You can also force a pop up by inserting a dvd and folding down the shift button, set in drop down

(though File Management is worth opening up..
