### Post by stevendpclark on 2006-03-24
Yup, you can. I own a Z60t which had that. There is a little windows utility that allows you to create a bunch of cds that contain all the information in that partition. I would recommend creating them first in case you need to update the bios or some other lower level detail which I think you can only do through windows. 

But other then that there is no use, I flushed mine and all is still good.
