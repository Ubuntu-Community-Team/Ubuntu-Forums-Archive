### Post by Lutin on 2007-11-16
Well, you could use it for the cache that your browser (firefox?) uses to store all the files it downloads off the web. Then instead of having to spin up the hard disk when you load a new page, firefox would use the ram disk instead. Of course, you would lose all the files each time you shut the machine down, but with broadband, reloading a page every time should not be a problem.

So, more life out of your battery (assuming you are using a laptop) and quieter too.

Mind you, I have not done this myself yet - but I plan to. :-)
