### Post by searchfgold6789 on 2011-11-02
If it's a fresh install, you need not worry; by default the distribution upgrade tool will update to Xubuntu 12.04 when it comes out. However, if you did something to the system that radically changes the desktop libraries, etc, for example:```
sudo apt-get install ubuntu-desktop
```then you will upgrade to a different version of Ubuntu (if you don't know what I'm talking about or haven't done anything major to your system then don't worry.)
