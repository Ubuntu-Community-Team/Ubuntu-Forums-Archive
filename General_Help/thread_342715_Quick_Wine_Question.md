### Post by JamieC on 2007-01-20
This should have been done automatically... have you tried to launch an executable since installing Wine?

If you have and it does not launch it using Wine, right click an executable file and choose properties. When a dialog appears, click the 'Open With' tab, then click the 'Add' button. Choose Wine from the list of applications and apply the changes.
