Fortunately I am fairly sure that Exchange Server CALs incorporate licensed usage of Outlook, since the Outlook client comes with the Exchange software, but all the same it would be nice to have an open source client so that once these people (any many others) are released from Microsoft Office's grasp, they can move onto an entire open source desktop with Open Office just as before, and a PIM/email client that works properly with Exchange Server.
