### Post by James Willis on 2009-04-03
If you are certain that you want to get rid of windows just allow the installation to use the entire drive that windows uses.  Ubuntu installation will then reformat the drive to the desired format, probably ext3, depending on which version that you are installing.

However, I would recommend that you first run a live CD version from your CD drive an insure that the version that you are trying to install runs properly on your laptop.  Also, you might consider resizing the windows partition to allow room to set up two linux partitions (main and small swap) for ubuntu and dual boot between windows and ubuntu.

Good luck,
Jim
