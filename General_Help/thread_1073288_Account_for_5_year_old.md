### Post by hello_kitty on 2009-02-18
Unless your son has the applicable permissions specifically granted to him by your wife, he shouldn't be able to access any of her files, so checking the box *should* be risk-free.  The only way to truly know, however, is to create his account and try and do all the "naughty" things a 5-year-old would do.  If you can figure out a way around the built-in file security, chances are he might be able to as well.  Happy testing!

Here is what I learned...
For anyone who may find this thread helpful in the future:[/B]

2 Ways to "child proof" the machine.

**Method 1** - Go to **SYSTEM / ADMINISTRATION / USERS AND GROUPS**. ADD USER. Add the obvious name and password for the child. Change "profile" to **unprivileged**.
Go to User Privileges tab to make sure ALL the boxes are unchecked.
This will disable network and Internet access altogether as well as bulletproof the rest of the machine. You can supervise any surfing the kid does using your regular account.

**Method 2** - Go to the same account manager. Same thing but the ONLY User Privileges you **need** to activate are: "Connect To Internet Using Modem" as well as "Connect To Wireless And Ethernet" if it applies to the computer.
This method will let them surf the web but you will have to password protect and print any file shares that you do not want them to access on other computers.

With method 1 I can nap while the boy plays Planet Penguin...:p
