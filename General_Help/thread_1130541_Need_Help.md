### Post by -kg- on 2009-04-23
I really think your first idea was the best one...just take the .iso to another computer that can burn a CD/R from the .iso (that is, if you're in such a rush that you don't want to order a free CD).  That way, it is unlikely that you will have to go into BIOS to change anything (and run the risk of messing up vital settings) since most newer computers are already set to check the CD drive in the boot process.

Not to mention the PITA of setting up all of that.  If you want to install using the USB stick, you have to go into BIOS and set it up to check the USB ports are even in the boot sequence.  If you were to boot to the USB stick, you would need an 8 GB or larger stick, and you would have to set BIOS to boot from it every time you want to boot to Ubuntu and set it back when you want to use Windows.

It's much easier to just burn the iso to a CD/R, boot from there, install Ubuntu and have GRUB with which to select which OS you want to boot to in the MBR of your hard drive.
