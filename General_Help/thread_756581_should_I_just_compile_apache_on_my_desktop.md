### Post by wdaniels on 2008-04-16
Yes, it's perfectly normal to run apache on a desktop for development purposes.  As to compiling it rather than using the packages - I don't see any reason to expect that package updates will introduce new bugs, but it doesn't much matter either way.  Personally, I use the packages, but I've compiled from source in the past when using less well-maintained distros.
