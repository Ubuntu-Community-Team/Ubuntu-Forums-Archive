### Post by ajgreeny on 2017-08-18
If you are saying that you can not install/download software that you have previously used in Windows it is not any surprise to most of us here.

Linux and Ubuntu, or any other distro, are not and never will be direct drop-in replacements for Windows so you can not often use the same software applications that you are used to, particularly if they are specific to your industry.

There are many applications available to install to your Ubuntu system in the software repositories; we do not normally download and install applications direct from the developers; it is not always impossible to do so but generally there is no advantage to so doing.

To add software to your system open up "Software" from the dash and search using that for what you need, but there is no point in searching for something like Photoshop, for example; it is a Windows/Mac application not available for Linux.  So tell us more about your problems and we will do our best to help you.
It may also be worth looking at [https://linuxappfinder.com/](https://linuxappfinder.com/) to see the huge range of applications that are available for Linux.
