### Post by Naatan on 2007-04-17
when you say you have the drivers on cd, doyo umean linux batteris? Cus I think your talking about the standard cd provided with your video card which is almost definitely meant for windows.. so forget about the cd :p

Ati on linux, in my experience, is hell.. I haven't been able to get an Ati card to work just right, when you make it run with one app it'll stop working for the other and vice versa, there is always something wrong.. though I have to say it seems to be going the right way with feisty..

if I were you, I would just run some google querries, as there are LOADS of tutorials out there on how to get an ati card running on ubuntu, but like I said, it will never (in my experience) run right.. 

As for your sound card, try running 'alsamixer' in terminal and make sure PCM is enabled and all the volume/mute options are set right
