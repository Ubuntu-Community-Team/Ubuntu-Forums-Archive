### Post by Cavsfan on 2013-03-28
I just noticed that 3.2 was released: [http://www.linuxtoday.com/upload/cairo-dock-3.2-release-130324063504.html](http://www.linuxtoday.com/upload/cairo-dock-3.2-release-130324063504.html)

How to install it:
[http://glx-dock.org/bg_topic.php?t=6792](http://glx-dock.org/bg_topic.php?t=6792)

For Ubuntu:
```
sudo add-apt-repository ppa:cairo-dock-team 
sudo apt-get update 
sudo apt-get install cairo-dock cairo-dock-plug-ins
```
