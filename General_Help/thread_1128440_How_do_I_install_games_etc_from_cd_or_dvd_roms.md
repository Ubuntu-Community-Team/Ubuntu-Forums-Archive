### Post by mb_webguy on 2009-04-17
Ubuntu (Linux), Windows, and Mac OS X are all different operating systems, and software designed for one will not run on the others.  

[Wine]("http://www.winehq.org/") is an application interface that will allow *some* Windows applications to run on Linux to *some* degree, but it will not work for *all* Windows applications, and applications that will run under Wine may not run perfectly.  You can check the compatibility of Windows applications under Wine at the [Wine AppDB]("http://appdb.winehq.org/").
