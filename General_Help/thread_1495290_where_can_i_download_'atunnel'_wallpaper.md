### Post by PC_load_letter on 2010-05-28
From Synaptic I found out that the 'atunnel' screensaver is part of the xscreensaver-gl-extra package. So, if you wanna install it from the terminal, all you have to do is type: 

```
sudo apt-get install xscreensaver-gl-extra
```
