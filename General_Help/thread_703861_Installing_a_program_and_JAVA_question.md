### Post by rainwalker on 2008-02-21
It depends what's in the .tar.gz; usually there are instructions with it (look for text filtes named "INSTALL" or "README" or something like that). However, if it's a Windows program you need to either make sure you get the Linux version (if there is one), or install Wine to run the .exe (which is only for Windows programs and don't work with Linux).

To install Java, go to Applications > Add/Remove and install the "Ubuntu Restricted Extras" package. You may need to enable the restricted and multiverse repositories if you haven't already (click the "Preferences" button in the Add/Remove window).
