I was forced to run an older version of Wine (0.9.37) so I could run Office 2003. (I know I use open office for word processing but I still find Excel to be far superior). ### Post by 3rdalbum on 2007-10-29
Go into Synaptic and find the Wine package. Select it. I'm not at my Ubuntu machine at the moment, but there's a menu item which says "Force Version". Use that, and select the version you currently have.

Now, you won't be asked to update it.
