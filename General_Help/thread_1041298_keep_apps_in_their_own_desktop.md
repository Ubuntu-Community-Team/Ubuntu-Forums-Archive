### Post by prince_niceguy on 2009-01-16
> **firsttimeuser said:**
> kubuntu comes with two virtual desktop as default, and i hardly use the second one. I try to use the second desktop but found out the apps run on the first one still show up in the second desktop.

I am just wondering is it possible I force all apps running in one desktop only show up in that single desktop?

When you says shows up? is it the task bar of the screen? if it is task bar then change the task bar to "show tasks from current desktop only".

if it is screen then right click the application and selection to --> desktop --> your desired desktop... that should fix it.
