### Post by malacandra on 2006-05-20
As the main User (the one who has Sudo privileges) go to System->>Administration-->>Users & Groups.

Select the User you wish to enable sound for, click Properties then User Privileges Tab. You can select which services that User has available. Make sure sound is checked and any others you want.

Then log in with that User and you should be good to go!
