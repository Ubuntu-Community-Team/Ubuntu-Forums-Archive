### Post by akand074 on 2010-11-30
Go to Accessories > Terminal and copy paste these two commands one by one (entering your password if it asks)

```
sudo apt-get install compizconfig-settings-manager 
``````
sudo apt-get install compiz-fusion-plugins-extra
```Now you can go to System > Preferences > CompizConfig Settings Manager

Scroll down under "Desktop" and check the "Desktop Cube" option. You can also play around and change a whole bunch of settings/appearances from here. 

Post back if you have any more questions, otherwise mark this thread as solved from the "thread tools" option.

EDIT: I forgot to add, make sure you have your proprietary graphics drivers installed. To do this (if you haven't already it usually notifies you) go to System > Administration > Additional Drivers (or Hardware Drivers depending on the release your using) and install it from there.
