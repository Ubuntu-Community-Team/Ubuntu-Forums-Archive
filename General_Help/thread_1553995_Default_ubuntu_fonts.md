### Post by chopinhauer on 2010-08-16
> **X5-655 said:**
> 
What is the default title bar font in 10.04?  I changed it and it seems selecting themes don't restore it.


All default fonts are 10pt Sans, except window titles (10pt Sans Bold) and the fixed width font (Monospace).

And since &#8220;Sans&#8221; is just an alias for the default sans-serif font, it will always be that way.
