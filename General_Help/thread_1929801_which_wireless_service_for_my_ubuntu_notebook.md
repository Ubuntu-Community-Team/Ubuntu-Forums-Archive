### Post by imachavel on 2012-02-22
yes that all works great with linux. It's just how you set up the network connection, if you are connected directly to the internet dns server via i.s.p. modem, or routed through a router via dhcp hard wired to the i.s.p. modem, or even possible through a repeater wirelessly seeing the ssid of a router hard wired to an i.s.p. modem

Let's not get into specifics on how a network interface adapter works, if your driver is installed correctly, things should be working. just install ubuntu on whatever wireless device you choose, through a virtual drive somehow or a new partition, and take a note of you previously were connected, check your network connection in control panel. Or if you have set up a network connection with ubuntu previously then it should be easy to find out.
