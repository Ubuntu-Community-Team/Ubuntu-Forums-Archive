How can I get back the file manager icon on the taskbar?

It disappeared after I removed the Chromium icon from the taskbar.

### Post by rick_james on 2012-01-18
I don't know how to mark this thread as 'solved', but here's how I figured it out.  Whew! :-)

What a relief! :-)

Right-click on the taskbar.

Select "Add/Remove Panel Items".

Click the "Panel Applets" tab in the Panel Preferences Window.

Click "Add".

Highlight "Directory Menu" and click "Add".

"Directory Menu" is the file manager.  It will now appear on the bottom right of the screen to the right of the clock.

To move it back to the left, right-click on the taskbar again, select "Add/Remove Panel Items", click the "Panel Applets" tab in the Panel Preferences Window, highlight "Directory Menu", then click "Up" until the icon is all the way to the left near the menu icon (the icon that is similar to the 'start' button in Windoze).
