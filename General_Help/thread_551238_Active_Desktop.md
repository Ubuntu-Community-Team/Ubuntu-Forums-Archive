### Post by sumguy231 on 2007-09-14
KDE's desktop will do that. It is possible to install kdesktop (just the desktop part of KDE) and run it at GNOME startup as a working replacement for GNOME's desktop - even the pager will pick up on KDE's desktops and you'll have more features available to you. I've done this with both GNOME and XFCE since both of them are so inflexible.
