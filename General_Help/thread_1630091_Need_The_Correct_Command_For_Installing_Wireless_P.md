### Post by wojox on 2010-11-24
Try

```
sudo -s
```

Then type your administrator password and that will give you the super user privileges, then in the folder for the printer that you get off the site just drop the autorun into the terminal, and a 'windows style' installation sequence will happen after that, once all that's done it should work
