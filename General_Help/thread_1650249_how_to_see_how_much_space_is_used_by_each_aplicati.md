### Post by bobcollard on 2010-12-21
If you mean how much memory it is taking at the moment, you can find that in System Monitor>Processes Tab.  How much space it takes on the HD is more difficult to know since there are preference files, dependent packages and logs scattered all over the place.  The basic applets are stored in file system/usr/bin.
