### Post by mcduck on 2009-09-21
You could try [Dia]("http://projects.gnome.org/dia/"). (It's in the repositories).

Of course any vector graphics editor (like Inkscape) would work as well, but Dia is specifically made for drawing all kinds of diagrams.
