### Post by Ratscallion on 2008-12-28
Well, first I suggest the easiest way to modify that logo is to download Ubuntu Tweak, latest version. It includes a feature that allows you to modify the logo of the gnome panel. Depending on the icon theme you have installed, it will change on that. Link below:

[The Ubuntu Tweak Website]("http://ubuntu-tweak.com/")

Now, if you want to make your own logo, then you'll probably need to use the GIMP image editor, included in Ubuntu 8.04 and above (I think). After drawing the icon you want save it in the following format
FILENAME.ico
To any directory you choose. After doing this, load Ubuntu Tweak, go to the section where you change the GNOME panel icon. Browse for the one you want and hey Presto!
A little resizing may be in order, but it should work
