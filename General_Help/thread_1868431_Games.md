### Post by sanderd17 on 2011-10-24
You can run some windows games via wine. Check out the wine database to see if it runs: [http://appdb.winehq.org/](http://appdb.winehq.org/)

Platinum ratings are the only apps that run out of the box. All other ratings need some tweaking.

You can also use play-on-linux, that's wine with some tweaks already applied, but you can only install a limited set of games with play-on-linux.

For native linux games, take a look at the games section in the software center.
