### Post by noguyinshame on 2013-06-23
What browsers are you using? If either Firefox or Chrome (or Chromium), you can use AdBlock to block unwanted ads.
Firefox: [https://addons.mozilla.org/en-us/firefox/addon/adblock-plus/](https://addons.mozilla.org/en-us/firefox/addon/adblock-plus/)
Chrome (Chromium): [https://chrome.google.com/webstore/detail/empty-title/cfhdojbkjhnklbpkdaibdccddilifddb?hl=en-US](https://chrome.google.com/webstore/detail/empty-title/cfhdojbkjhnklbpkdaibdccddilifddb?hl=en-US)

Adblock give you the option to disable AdBlock on website that you like, and you should do so for whichever website you support as advertisement gives them money.
