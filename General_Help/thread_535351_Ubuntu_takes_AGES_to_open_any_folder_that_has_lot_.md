### Post by jim_p on 2007-08-26
About the files and ubuntu slowing down.
Try re-organizing your stuff in folders and subfolders.
Filesystems in general, not just ubuntu, "hate" thousands of files in one directory.
This will speed things up.

On my pc, I have like 1300+ zip/rar files that contain icons. For every ~100 files, I make a new folder and this speeds up things a lot.
On the other hand, I have ~700 wallpapers that are all in the same folder with no subfolder organization.
It takes a lot more time to open that folder, even with thumbnails disabled.

And windows' thumbs.db simply provide the thumbnails of image files.
Linux does the same with images, texts, pdfs
