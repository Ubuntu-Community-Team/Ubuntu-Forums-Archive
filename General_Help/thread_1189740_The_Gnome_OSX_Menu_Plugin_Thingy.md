### Post by mcduck on 2009-06-17
I bet he's talking about the unified menu applet available for Gnome-panel (which moves the window menu from the window into top panel, like in OSX).

Its a gnome-panel applet, so it won't work in any other panel (XFCE-panel maybe, as it has some special applet that allows running gnome-panel's applets).

I don't thing that one would be very easy to change for any other purpose, you will surely find some better applet to use as starting point if you are interested in creating different kind of panel menus.
