### Post by 3Miro on 2011-02-11
You mean edit code or "run" code. 

To edit the code, you can use either Gedit or the more developer friendly Geany (gedit is installed by default and Geany is in the Software Center).

If you want to host php web-pages on your computer, check this one:

[https://help.ubuntu.com/community/ApacheMySQLPHP](https://help.ubuntu.com/community/ApacheMySQLPHP)

Since PHP requires a web-server, there is no "easy" way to "run" php.

EDIT: I just noticed you are using Lubuntu. On Lubuntu Leafpad is the default text editor, but it is perhaps too simplistic. Gedit and Geany offer syntax highlighting, and you can get them from the Software Center.
