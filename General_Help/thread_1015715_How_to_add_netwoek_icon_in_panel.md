### Post by Ivo Moelans on 2008-12-19
Try *Alt + F2* and type *nm-applet --sm-disable*. Look in *System&#8594;Preferences&#8594;Sessions* if *Network Manager* is there and checked. If not make an entry yourself.
That should be it.
