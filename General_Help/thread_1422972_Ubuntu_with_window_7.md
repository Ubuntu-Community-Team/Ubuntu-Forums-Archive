### Post by Intrepid Ibex on 2010-03-06
Boot loaders were invented to have the possibility to choose which os you want to boot to after bios stage setting everything up.

This can even be used with mobile phones these days, such as HTC Hero.

By default Ubuntu creates you both options, windows and ubuntu to boot up with in grub.

Here's a screenie from Grub (GRand Unified Bootloader, most distros use it by default) (by default the look is black - not blue, but you can change it easily with e.g. startupmanager):
