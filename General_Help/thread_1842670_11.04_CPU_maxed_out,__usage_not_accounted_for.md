### Post by jwbrase on 2011-09-12
Open a terminal and type:

```
top
```

You should see a line that looks like this:

```
Cpu(s):  6.4%us,  3.0%sy,  0.0%ni, 90.3%id,  0.0%wa,  0.2%hi,  0.2%si,  0.0%st
```

Is the number by "wa" really high?
