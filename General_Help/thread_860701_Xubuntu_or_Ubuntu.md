### Post by iaculallad on 2008-07-15

> YES, I know...but whats the differnece between Xubuntu and Ubuntu!

The common difference is the desktop they both use. Ubuntu uses the GNOME desktop by default, while (x)Ubuntu uses the XFCE desktop w/c is lighter desktop that can accommodate your 225MB of RAM.

Take a look at this [site]("https://help.ubuntu.com/community/Installation/SystemRequirements") to view the system requirement to run the Ubuntu GNOME desktop version.
