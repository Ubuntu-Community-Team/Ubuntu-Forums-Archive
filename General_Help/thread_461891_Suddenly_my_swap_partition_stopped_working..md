### Post by mrcbrown on 2007-06-02
If the drive is making funny noises, I'd imagine it is a hardware issue, if you have smart tools installed, you can run a test to see what it thinks:

[http://smartmontools.sourceforge.net/](http://smartmontools.sourceforge.net/)


If your drive is smart compatible it may be able to report to you some issues - might be time to buy a new drive.
