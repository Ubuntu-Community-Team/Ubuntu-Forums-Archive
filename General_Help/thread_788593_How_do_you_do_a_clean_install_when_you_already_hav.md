### Post by tamoneya on 2008-05-09
1.  Back up data in ubuntu partitions
2. Delete ubuntu partitions.
3. tell hardy to install into unpartitioned space (what was gusty)
4.  Ubuntu will detect windows and install grub and configure for dual boot.
5.  Setup your new hardy install as you see fit.
