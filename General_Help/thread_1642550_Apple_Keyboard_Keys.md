### Post by coffeecat on 2010-12-10
> **bumder said:**
> Only problem is, that one or two keys are misplaced (ie the '@' and '"' keys are switched)

Is that with a UK layout? If it is I'll tell you how to get the # symbol, which can be a challenge on an Apple keyboard. Anyway, whatever your language/country layout, go to System > Preferences > Keyboard. Click on the Layouts tab. Make sure you have the correct country in the upper field. If not, add it and make it the default. Now click on the Keyboard Model button. Choose Apple from the Vendor list and if that is the white USB keyboard you're probably best off with 'Macintosh' from the lower list. But try others if that doesn't suit. 

This link is for the aluminium keyboard but there might be something useful for you.

[https://help.ubuntu.com/community/AppleKeyboard](https://help.ubuntu.com/community/AppleKeyboard)
