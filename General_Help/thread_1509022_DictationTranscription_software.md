### Post by issachan on 2010-06-14
After going to the first link I looked up Perlbox and found their site [http://perlbox.sourceforge.net/](http://perlbox.sourceforge.net/) where I downloaded the latest version of Perlbox-Voice. I opened the software center and downloaded Sphinx2 (I think like 4 or 5 things came up with that name so I installed them all), I also had to install Perl-tk. Then I had to install Festival which I did via the terminal. 

After I got the program running, I noticed my mic wasn't working, so I had to edit a file (which I can't remember the name of) and restart the computer. Now it works! 

It's not very good at recognizing what I'm saying, but it can pick up some words and even short phrases... but it's difficult to say it in a way that it'll understand. 

The commands are all customizable.. I say "burning fox" and it loads up Firefox and "drowning" makes Bluefish editor start. I tell it to "die" and it says "Self destructing in 10 8 7 6.. etc". I also have it set so when I say night it says "sleep well your majesty". 
:lolflag:
I also can make it talk via a built-in text box. If you don't feel like reading, you can paste the text in there and make it read it. 

It doesn't even come close to the Dragon Naturally Speaking program, but it's fun to play with.
