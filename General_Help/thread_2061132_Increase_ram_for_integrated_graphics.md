### Post by QIII on 2012-09-21
And, by the way, DON'T try to increase your performance by attempting to install the ATI driver.  Your GPU is no longer supported.

And DON'T try to use the Legacy driver because it will not work with versions of X post 2008.

Continue to use the default open source Radeon driver.
