### Post by cherok on 2010-03-02
While writing papers, I enjoy using BasketNotes to jot down notes from source material that I'm using.  In an effort to back-up all of my notes, I entered the Backup and Restore menu (File/Backup and Restore) and chose to relocate the folder that Basket Notes draws content from.

The issue:
Not only can I not locate the text content of my notes in my Filesystem (Latest Ubuntu distro) but I also cannot see any of my old notes in Basket.

Please help me recover my old Basket settings, I'm between a rock and a hard place here.

Much appreciated,

Cherok


**SOLVED**

The default folder for basketnotes is either considered a "hidden" or "backup" folder, and thus, when using the search tool those options must be checked.
