### Post by geirha on 2010-02-28
The guide at [https://help.ubuntu.com/](https://help.ubuntu.com/) should cover these questions. I recommend you give it a read. Amongst other things, it explains how to install flash.

There's a package, [ubuntu-restricted-extras](apt:ubuntu-restricted-extras), that installs several restricted packages, like adobe flash, microsoft truetype fonts, video codecs and such. You'll find it by opening the software center (Applications -> Software Center) and searching for «restricted»... or you can click the apt-url I provided here ;)

### Post by 3rdalbum on 2010-02-28
For 64-bit Flash, you DON'T want to use the "flashplugin-nonfree" package, because that installs the 32-bit version.

Either go to Adobe Labs and download the 64-bit plugin and drag it into /usr/lib/mozilla/plugins or add the Flash Player PPA ([http://ubuntu-tweak.com/source/sevenmachines-flash/](http://ubuntu-tweak.com/source/sevenmachines-flash/)).

For most things you don't need to use the terminal; and once you know how to set up an Ubuntu machine for all your restricted extras you can get the machine up-to-speed very quickly indeed. Stick with it and you'll be very happy :-)
