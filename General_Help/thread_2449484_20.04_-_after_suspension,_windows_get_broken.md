### Post by dragonfly41 on 2020-08-27
I have no idea about the cause of this. 
But right now I am configuring a freshly installed Ubuntu 20.04 (not on a laptop) and these tools I have just installed might be useful for windows tiling.

First, install Synaptic Package Manager (if not already installed).
**sudo apt install synaptic**

Second, open Applications > System Tools >  Administration > Synaptic Package Manager and search for ** gnome-tweak-tool **
to install.

Third, go to System Tools > Preferences > Tweaks > General
and try disabling .. Suspend when laptop lid is closed
to see if that changes windows layout.

Fourth (optional) install **x-tile**

[https://www.giuspen.com/x-tile/#downl](https://www.giuspen.com/x-tile/#downl)

[http://www.giuspen.com/software/x-tile_3.1-0_all.deb](http://www.giuspen.com/software/x-tile_3.1-0_all.deb)

and try using **x-tile** to layout your windows / tiles.

These tools do not explain **why** your windows are in disorder but they will help you get back into shape
