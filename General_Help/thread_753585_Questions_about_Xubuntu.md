### Post by 1/0 on 2008-04-13
The only thing that differs Xubuntu from Ubuntu is that Xubuntu has removed most of the dependencies and programs in Gnome. Instead it uses Xfce4. The OS is still the same, a ubuntu system. You can install the same stuff (even gnome if you want to). The only reason to install Xubuntu is to get rid of Gnome for something faster.

If the apps wasn't included in the fresh install of Ubuntu, its not going to be in Xubuntu. Things are slightly different but for the better, I'd say. Give it a try. Check the Xfce documentation for more info.
