### Post by SwitchDK on 2011-10-26
Hi 1keith1,

Another option is to use Google Chrome itself instead of Chromium. This is possible by going to [www.google.com/chrome](www.google.com/chrome) and download the deb package. Once downloaded double click on the deb file and it will be installed. As part of the install a Source Repository for Google Chrome is added such that Google Chrome is automatically updated whenever Google release a new version.

I hope this helps.
