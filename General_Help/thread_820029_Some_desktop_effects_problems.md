### Post by jpeddicord on 2008-06-05
You'll want to search System > Administration > Synaptic for packages relating to "compiz". Install one called "ccsm" and another that I think goes by the name of compiz-fusion-plugins-extra. Then, go to System > Preferences > Advanced Desktop Effects Settings. Enable the water plugin, and hit Shift + F9 to make it start raining on your desktop. ;)

### Post by Forlong on 2008-06-06
compiz-fusion-plugins-unsupported is not (and has never been) part of Ubuntu.
Ubuntu only provides official (and therefore supported) plugins.

The effect you are referring to is not a part of Compiz or Compiz Fusion. There has been a hack in Beryl to do that but it hasn't been ported to Compiz Fusion, because it was regarded too hacky to be maintainable.
