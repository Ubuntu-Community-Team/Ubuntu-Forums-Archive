### Post by juraj on 2006-11-17
Make sure that you have xorg-driver-fglrx and appropriate restricted kernel modules installed. Just erase and/or move out of way your xorg.conf (you should backup it), and then:

```
sudo dpkg-reconfigure xserver-xorg
```

Select fglrx on first screen, accept default answers elsewhere.

And after that, if you don't have it, add the following to end of your xorg.conf:

```
Section "Extensions"
        Option      "Composite" "0"
EndSection
```
