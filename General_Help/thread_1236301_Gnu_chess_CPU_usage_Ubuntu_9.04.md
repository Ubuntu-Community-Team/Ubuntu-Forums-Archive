### Post by jacksaff on 2009-08-10
GNU Chess can use as much cpu as it can get. The more cpu power it has the stronger it will play. 
If you don't make a living out of chess then it will probably be able to thrash you using a lot less than the power of your current machine. In this case you won't notice that it is playing stronger than it was on the low-power box seeing as both are beating you easily.
