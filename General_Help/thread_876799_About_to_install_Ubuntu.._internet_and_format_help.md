Good to see that your ISP apparently "supports" Linux - but this should be unneccessary as internet connection should "just work" with your current router settings.

When you install Ubuntu, you will see an option to use the entire disk - this will completely remove your old Windows installation so make sure you have backed up your old data to a different media **first**.

Alternatively, you can retain your old Windows partition and add Ubuntu as a "dual-boot" option.
