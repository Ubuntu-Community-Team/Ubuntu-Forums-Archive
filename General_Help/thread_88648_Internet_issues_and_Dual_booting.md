### Post by Irony on 2005-11-10
Back-up you data, defragment windows as much as possible. Look at;

[http://www3.telus.net/linux4u/ubuntu.html](http://www3.telus.net/linux4u/ubuntu.html)

and for more detail;

[http://users.bigpond.net.au/hermanzone/p3.htm](http://users.bigpond.net.au/hermanzone/p3.htm)

These describe how to use the ubuntu installation disc to shrink the windows partition and install ubuntu to the freed up space.

Network detection should happen automatically during install (make sure your internet connection is connected during install for this to happen).

GRUB should automatically detect Windows for a dual boot system.
