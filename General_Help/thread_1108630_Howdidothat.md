### Post by zvacet on 2009-03-28
If you want to type with German letters then s**ynaptic>admin>language support>check German**.In system>preferences>keyboard>layout add German language.To switch fro one language to another language on desktop upper panel click add to panel>keyboard indicator and choose language you want.

### Post by Vaphell on 2009-03-28
you can use unicode chars by CTRL+SHIFT+U (underlined u shows up), release and then enter code number which you find in the char table
ä - u 00e4
Ä - u 00c4
ü - u 00fc
Ü - u 00dc
ö - u 00f6
Ö - u 00d6
ß - u 00df
° - u 00b0

these are unicode chars, i don't know if windows users will see them in standard configurations
