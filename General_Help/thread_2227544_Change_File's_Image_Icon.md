### Post by vasa1 on 2014-06-03
IIRC, it may depend on the file manager you use and whether that file manager also manages the desktop. I think Thunar and Nautilus have (or had) the ability to allow you to slap an emblem on top of an existing icon. The existing icon though would be determined by the mimetype (I think) unless we're talking about executables.

Keep in mind that Lubuntu is meant to be a "light" distro and features found elsewhere may not be available in default Lubuntu. Also, the development team is pretty small and is made up of people volunteering their time (probably without remuneration).
