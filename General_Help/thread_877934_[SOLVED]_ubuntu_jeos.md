### Post by Vivaldi Gloria on 2008-08-02
> **jnw222 said:**
> what is ubuntu jeos 
is it like a version of ubuntu with the kernel and a couple of packages

Jeos is a very light version which has hardly any drivers or apps on it. Do NOT install it in a computer because it won't recognize your hardware. This version is supposed to be run in a virtual sytem like vmware, virtualbox, etc. "This combination of reduced size and optimised performance ensures that Ubuntu JeOS Edition delivers a highly efficient use of server resources in large virtual deployments."
