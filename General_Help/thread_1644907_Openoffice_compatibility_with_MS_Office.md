### Post by dmizer on 2010-12-13
I moved this out to it's own thread because it's more of a support request than a recurring discussion.

There are a number of things you can do to make OO.o more compatible with MS Office. The first of which is to install all the Microsoft TTF fonts like Times New Roman, and make sure that your entire document is using a Microsoft font. Free fonts and Microsoft fonts have different widths and heights so that's one reason your formatting gets screwed up. You can install these fonts with the ttf-mscorefonts-installer package found in Synaptic.

You can also include this fantastic OO.o docx plugin: [http://www.oooninja.com/2008/12/better-office-docx-converter.html](http://www.oooninja.com/2008/12/better-office-docx-converter.html)
