### Post by 3Miro on 2012-06-20
The best way to learn about Linux is to dual-boot, that is, when you boot your Mac you have the option to use Linux or OSX. Dual-booting on a Mac can be tricky due to potential issues with hardware incompatibility, do a lot of reading about specific model of Mac (i.e. which generation, cpu, video card ...)

As for Virtual Box vs Boot Camp, as far as learning Linux, there is no difference.
