### Post by megahost on 2010-03-22
There are many of options for learning programming. A way to start is to buy a book on an easy to learn language. Depending on where you are aiming, where you want to focus on desktop, or web development, there are plenty of options. If perhaps you do choose web development, W3schools has some pretty basic tutorials out there. I wouldn't recommend taking on a language like C++, Java or Python as a first language, for the can be very challenging to learn.

### Post by dyingsun on 2010-03-22
If you've already got a little HTML under your belt, then maybe you should consider building on that. Master HTML, and CSS first. Check out: [Sitepoint]("http://www.sitepoint.com/") as a source of info, and of course [Webmonkey]("http://www.webmonkey.com")

These are great foundation technologies to build on - once you've got a grip on these, you can look at JavaScript and PHP(and mysql) to add new dimensions to your webpages. The communities are very large and lively, and if you have questions, they tend to get answered very quickly. You'll find that once you're familiar with a couple of different languages, you'll understand more about  how programming works and you will be able to pick up new ones quite quickly.

Good luck, jump right in!
