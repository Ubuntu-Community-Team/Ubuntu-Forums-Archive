### Post by egeezer on 2015-04-04
If you want real stability in just about any desktop of Ubuntu, it would be well to switch to the LTS version, and install it as 14.04.2, which has the 3.16 kernel.  The interim versions between LTS releases are somewhat like beta test-beds for what is to be included in the next LTS release.  For various reasons, 14.10 had few visible new features, and you would find 14.04.2 essentially a stable version of your 14.10, whether you run it as Xubuntu or Ubuntu+Xfce (which I do so I have LibreOffice instead of Abiword).
