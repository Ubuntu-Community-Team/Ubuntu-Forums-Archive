To resolve the error you need to open a Terminal (found under Applications/Accessories) and type in:```
dpkg --configure -a
```Or you could just download the .deb for [Frostwire](http://www.frostwire.com/) instead.  Frostwire is the open source version of Limewire Pro.  

You'll need to get your Java straightened out but I'm sure we can do that.
