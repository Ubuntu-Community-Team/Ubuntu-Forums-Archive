### Post by bodhi.zazen on 2010-07-17
You obtain a root shell with sudo

```
sudo -i
```

Run graphical applications with gksu

See also:

[https://wiki.ubuntu.com/RootSudo](https://wiki.ubuntu.com/RootSudo)

Running a desktop (gnome) as root is strongly discouraged and not supported on these forums.
