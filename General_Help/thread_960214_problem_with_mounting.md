### Post by geirha on 2008-10-27
To avoid that, use UUID instead of device nodes to identify the partitions. Run ```
sudo blkid
``` to find the UUIDs of those three partitions, then replace /dev/sda1, /dev/sda5 and /dev/sdc1 with their respective UUIDs ... E.g.:
```
UUID=0123456789ABCDEF /media/big2 ntfs-3g defaults,locale=en_US.UTF-8 0 0
```
