### Post by mooreted on 2014-08-19
In the main window there is a text box for the location of the ripped content. Look there first. If you know the name of the file you ripped you can search for it. Using the command line do:

find . -iname "*name_of_the_file*" 

You don't need to type the entire name of the file, just part of it.
