### Post by RJ12 on 2010-09-17
It actually doesn't matter which one you use. Infact VMware should be able to mount the ISO directly. Just go to File > New > Virtual Machine and follow the steps. Then start it and install it the way you would normally. I am sure that there are many tutorials on the web if you need specific instructions.
