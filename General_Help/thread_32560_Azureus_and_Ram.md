### Post by JonahRowley on 2005-05-08
There is no "best" client, some are better at certain things.  Azureus has the best interface, the Gnome bt client is the simplest, and the bittorrent client is probably the most flexible.  I use the bittorrent client, either in GUI or curses mode, I don't have the need for a giant GUI app to download Linux isos.

As for Azureus, I've lost faith in.  I've lost faith in most anything written in Java, I don't care what the zealots and benchmarks say, it's slow.  And it takes too much ram for what it does.  And that's a shame, becuase there are two projects I'm very interested in (primarily Freenet and I2P) that I literally cannot run because they are too resource intensive, but if they were written in a native language, would run fine.
