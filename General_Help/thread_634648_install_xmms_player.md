### Post by mouseboyx on 2007-12-07
You don't have to do that.
you can just open the synaptic package manager and type xmms in the search box and install the correct packages.

or press Alt-F2 and run
```
sudo apt-get install xmms
``` in a terminal

### Post by zero-9376 on 2007-12-12
installing from the source code is not the reccommended method for packages that exist in the ubuntu software repositories, you would need to install the build-essential packages to compile and install xmms from source which would use more download than just installing using apt-get. 

by the way you may want to install audacious instead of xmms, its got the same winamp2 ish look but is much newer than xmms with better dialogs and easier setup.
