### Post by Tianair on 2011-05-09
I solved the problem myself thanks to this guide [http://www.webupd8.org/2011/04/things-to-tweak-fix-after-installing.html](http://www.webupd8.org/2011/04/things-to-tweak-fix-after-installing.html)
I installed the compiz manager 
```
sudo apt-get install compizconfig-settings-manager
```
and then I went to the "OpenGL" plugin and unchecked the "Sync to VBlank" option.
