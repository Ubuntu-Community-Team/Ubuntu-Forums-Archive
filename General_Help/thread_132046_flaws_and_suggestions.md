### Post by gingermark on 2006-02-17
[QUOTE=Progressive]Nvidia drivers should be installed automatically

Multimedia Plugins don't come standard and preloaded into Konqueror or Firefox[/QUOTE]
I think legal issues prevent this from being allowed, combined with a desire for the operating system to use free software. Keep in mind watching an encrypted DVD or a wmv file in Linux is considered illegal in America. 

[QUOTE=Progressive]Antivirus Software? Firewall Software?[/QUOTE]
I'm definitely with you on the latter. Guarddog should be included on the install CD. Not sure you need antivirus software on install, you can always get it later. But in order to download a firewall you need to be connected to the net! That never made sense.

[QUOTE=Progressive]During installation it should have a checklist of optional programs that you can install or not install... programs that come to mind are digikam, the GIMP, Antivirus, Firewall, Multimedia software etc[/QUOTE]
I don't see much difference between getting all this on install, or installing it afterwards where and when you need the packages. I think an extra CD full of apps might be useful for those with slow connections. And again, I think legals issues get in the way of the multimedia stuff.

[QUOTE=Progressive]Within the GRUB boot loader, there are duplicate Kubuntu entries (probably because of kernel upgrades)[/QUOTE]
If you uninstall the kernels you don't need the entries should disappear. Consider keeping some of them though. If for whatever reason you have trouble booting off your preferred kernel, they provide another option.
