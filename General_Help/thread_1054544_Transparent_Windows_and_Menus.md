### Post by itang sanjana on 2009-01-30
This is for the transparent menu,

Open CCSM,
under General Options go to
Opacity Settings > Window Opacities > click New
Put this line in your Opacity windows box
```
(type=Menu | PopupMenu | DropdownMenu)
```
Adjust Opacity windows values in your favour.
Enjoy..

PS. I have no idea about transparent windows border yet.

### Post by gjoellee on 2009-01-30
Do you want the whole windows to be transparent or just the window borders? 

You can make your window border transparent by using Emerald. Install emerald with this command: ```
sudo apt-get install emerald
```

Then find Emerald Theme Manager in the menu and start messing with your options/install additional themes from [www.gnome-look.org](www.gnome-look.org) (or exact links)

[http://www.gnome-look.org/index.php?xcontentmode=102](http://www.gnome-look.org/index.php?xcontentmode=102)

or
