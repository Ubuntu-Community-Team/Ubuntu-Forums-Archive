### Post by RTrev on 2009-10-25
Sure, there is a lot of stuff you can delete.. but unless you're really short on disk space I'm not sure you're going to gain much.

On the other hand, you might want to go through and shut down services that are running but which you don't need.  Those can chew up both cpu and memory.

If you want to delete programs, it's easiest (for me at least) to do this in Synaptic.  Once you ask to remove something, if a list pops up saying that these things will also be removed then check that list carefully.  Make sure you aren't losing something you care about.

You can also come at this problem from the other direction by getting the Alternate CD and doing a "command line" installation.  Then add packages as you see fit.  This can be a lot of work, though.

What is your goal in doing this?  Free up disk space?  Or something else?
