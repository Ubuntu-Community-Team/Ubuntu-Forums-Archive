### Post by bluexrider on 2012-01-16
Put the .ISO in the tray and when it loads you can tell by the file name IE if its a 64bit version it will appear like this "ubuntu-11.10-server-amd64.iso" or something like that. If its a 32bit it will be "ubuntu-11.10-server-i386.iso"

i386 = 32bit
amd64 = 64bit
