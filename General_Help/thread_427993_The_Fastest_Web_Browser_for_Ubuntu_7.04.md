### Post by rillip on 2007-04-30
I can't speak for everyone, but if speed is your major consideration, I don't recommend Opera.

I'm using Opera 9.2 on Dapper Drake.  Don't get me wrong, I love it.  It's the only browser I like to use.  I also have FF installed, as some pages with non-standardized CSS and things as such display correctly in FF but not Opera (Opera won't budge on standards).

However, I find Opera to be relatively slow, particularly compared to Windows Opera.  It was there before, but has become accute since the release of 9.2.  I end up waiting for a pretty good lenght of time, two seconds or so, between click and first noticable action.
