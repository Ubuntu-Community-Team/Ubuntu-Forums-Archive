### Post by Impavidus on 2014-10-31
Xubuntu and Lubuntu are mostly identical to Ubuntu. Most of the software is the same, but these lighter flavours sacrifice eye candy for speed and memory conservation. You get different, lighter default applications for spreadsheets, word processing etc., but all applications from Ubuntu are available from the repositories.

The prompt is the part before the $. Your terminal should show you that to indicate it's ready to accept input and show you the context in which this input would be used. Only the part after the prompt is part of the command you should type. The format of the prompt is by default```
username@hostname:current_directory$
```Of course, user name, host name and current directory can change. So if the downloaded file is in the Downloads directory and you start by default in /home/username, you would type the bold text here:```
username@hostname:~$ **cd Downloads**
```In your case it's in Z_ISOs, making the command```
(omitting the prompt)$ **cd Z_ISOs**
```followed by```
md5sum lubuntu-14.04.1-desktop-i386.iso
