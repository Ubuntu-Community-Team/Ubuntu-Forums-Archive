### Post by Mark Phelps on 2011-04-06
Need more details about what you want to do ...

It sounds like you want to do the following:
1) Be running apps normally in Vista
2) Launch Ubuntu (just like some other app)
3) Use Ubuntu for a while
4) Shutdown Ubuntu and continue using the other apps

If this is what you want, Wubi is not the answer because that INSTALLS Ubuntu using Vista.  But, to run Ubuntu, you have to select it from a boot menu.  You then have to reboot to get back to Windows.

To run Ubuntu inside Windows, you need to run it in a Virtual System.  So, you should read up on VirtualBox to see if that provides what you want.
