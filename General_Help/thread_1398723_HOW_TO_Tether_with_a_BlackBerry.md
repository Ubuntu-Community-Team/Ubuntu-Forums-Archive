### Post by theozzlives on 2010-02-04
First you need BlueTooth, then install BlueMan and pair with your phone. Click on setup and select Dial Up Networking (you'll need to setup you dialup info, if any in network manager). Your connection should appear in network manager. Hope this helps someone, I played hell figuring this out.
