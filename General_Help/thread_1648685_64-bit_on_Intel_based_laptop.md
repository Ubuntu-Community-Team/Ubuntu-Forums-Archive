### Post by 3Miro on 2010-12-19
AMD64 is the name of the standard, which modern AMD and Intel CPU models support.

64-bit is slightly faster (depending on the application), while at the same time it uses a bit more RAM and has the occasional compatibility issue (those are very rare nowadays).

With 2GB of RAM, you are probably better off with 32-bit.
