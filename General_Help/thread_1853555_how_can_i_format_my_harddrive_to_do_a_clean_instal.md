### Post by airper on 2011-10-02
Just boot off a live CD click the install option and then select the new install option, which will overwrite everything. It will reformat your HD and give you a clean HD will a fresh install of Ubuntu.

This is actually the easiest option, once you select this option you just have to answer a couple of questions about the keyboard, timezone etc and then sit back while it does it's stuff.

Hope this helps.

### Post by wanderingarticfox on 2011-10-02
I am going to guess that you have a ISO Image Burned onto a CD and a computer that meets the minimum requirements for Ubuntu 11.04. If you pop that CD in ad reboot [note you may need to set up your system to boot to CD if it does not do so now] the CD will guide you through the installation. If you have a large Hard Drive you might want to go manual on the disc set-up options. But Guided will ask if you want to use the entire disc just say yes and forget about your other stuff and I mean ALL of it, Pictures Movies,Documents everything. So if you have something you want to keep do not do this until the other stuff is backed up on a USB Drive or CD's.
