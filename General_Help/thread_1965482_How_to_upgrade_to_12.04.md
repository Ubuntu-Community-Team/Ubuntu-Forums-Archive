### Post by LinuxFan999 on 2012-04-25
You can go to the update manager and check for the upgrade, then install it. This procedure is not recommended and can break your Ubuntu installation. The best way to upgrade to a new Ubuntu release is to download the ISO image, burn it to a CD, and do a clean installation. Make sure to back up before upgrading.

### Post by rk0r on 2012-04-25
To upgrade from Ubuntu 11.10 or Ubuntu 10.04 LTS on a desktop system,  press Alt+F2 and type in "update-manager -d" (without the quotes) into  the command box. Update Manager should open up and tell you: New  distribution release '12.04' is available. Click Upgrade and follow the  on-screen instructions.
