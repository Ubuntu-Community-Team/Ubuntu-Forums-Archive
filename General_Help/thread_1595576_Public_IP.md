> Do I need a public IP to run a TFTP server (will not be used locally) or access my computer from an external IP

I think you're asking for an static IP right? If so, that is what probably any ISP would tell you, but you CAN do lots of stuff without it.

There's a several steps to achieve what you want:
[INDENT]A constant and stable connection.
A subscription to a dynamic DNS service.
Configuring your router and server to both enforce security and open the ports/services you need.[/INDENT]

I found this youtube tutorial VERY helpful: [Access Your Home Computer From Anywhere]("http://www.youtube.com/watch?v=bI52nF1BRNo&feature=fvw").

Good Luck!
