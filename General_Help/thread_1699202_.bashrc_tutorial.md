### Post by JKyleOKC on 2011-03-03
Here's the first three lines of my own .bashrc file: ```
# ~/.bashrc: executed by bash(1) for non-login shells.
# see /usr/share/doc/bash/examples/startup-files (in the package bash-doc)
# for examples

```Have you tried looking there? When I did, the examples directory didn't exist. You can use "sudo apt-get install bash-doc" to bring it into your system.

Any line in the file that starts with "#" is a comment, not a command, and my copy is very well commented with explanations of what each sequence of commands is intended to accomplish. For a tutorial, I'd recommend getting a good one on bash itself, which will explain all of the commands and options. There are zillions of them, most of which you will never use but are priceless when you need them. I bought a book "Learning the Bash Shell" to use as a reference...
