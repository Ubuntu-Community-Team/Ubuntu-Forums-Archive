### Post by niccholaspage on 2008-08-25
Well when I switched from Windows to Ubuntu,Firefox wouldn't take me back a page with backspace.I learned that you need to use ALT+Left Arrow Key.Even though I know this sometimes I still try to use Backspace.Is there anyway to make it functional?

---

### Post by silkstone on 2008-08-25
Type about:config into FF and then scroll down to **browser.backspace_action**.

Change the value from 2 to 0.

That should do it!
