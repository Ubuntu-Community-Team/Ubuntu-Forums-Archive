### Post by wrgb2 on 2009-11-05
Go into Synaptic Package Manager - System > Administration > Synaptic Package Manager.  Type network-manager in the search box.  Mark network-manager for install  by double clicking it.  Then press the apply button.  This will install network-manager again for you.  If it's already installed, remove it and install it again.  Log out and log back in to activate it.
