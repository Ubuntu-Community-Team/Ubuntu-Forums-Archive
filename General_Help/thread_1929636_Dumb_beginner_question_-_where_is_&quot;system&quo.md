### Post by blazemore on 2012-02-22
The tutorials you are following are from old versions of Ubuntu which ran a desktop called "Gnome 2".

Later versions of Ubuntu, like the one you are running, run a desktop called "Unity".

If you look in System Settings you will find the item you are looking for.
