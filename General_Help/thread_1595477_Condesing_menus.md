### Post by Refalm on 2010-10-13
You can choose one of these options:

1. Right click on the panel, then "Add to panel", then add the Gnome menu.

2. Install [Cardapio](https://launchpad.net/cardapio). It's want you want, plus it has an application search bar (like in Mac OS X and Windows 7) and a hotkey.
```
sudo add-apt-repository ppa:cardapio-team/unstable
sudo apt-get update
sudo apt-get install cardapio
```
