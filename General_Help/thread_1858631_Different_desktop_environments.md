### Post by IWantFroyo on 2011-10-12
```
apt-get install
```
Same command as usual.

A DE is a collection of programs and tools that work together. You can use a bunch of different programs almost like your own "DE" by choosing a window manager, a panel, and some programs.

My personal favorite WM is Openbox. In fact, there's a list of WMs linked on that page TeoBigusGeekus gave you. It's missing a few though, such as scrotwm.
