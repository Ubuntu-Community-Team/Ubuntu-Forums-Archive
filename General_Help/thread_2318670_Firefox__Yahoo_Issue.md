### Post by SeijiSensei on 2016-03-28
I'm guessing you are using Firefox, but the home page is set to Yahoo which is I believe is now the default.  You can change your preferred home page via Edit > Preferences > General and change your default search engine with Edit > Preferences > Search.

I don't believe there is a "Yahoo browser" by default.  In the bar at the top of the window, do you see "Mozilla Firefox?"
