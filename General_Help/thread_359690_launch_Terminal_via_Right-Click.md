### Post by ethan@xmlstandards.org on 2007-02-12
Hi there,

This can indeed be done using the following install:

sudo apt-get install nautilus-open-terminal

Make sure that the UNIVERSE repository has been enable (via Synaptic is easiest way).

Upon a new login this will give you a right click context menu option to open a terminal window for the currently selected window.

Hope that helps.
