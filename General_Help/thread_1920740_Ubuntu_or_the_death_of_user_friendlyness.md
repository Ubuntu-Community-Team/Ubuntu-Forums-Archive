### Post by mcduck on 2012-02-05
Ubuntu, as pretty much every Linux distribution, has a very simple, easy and user-friendly way of installing applications. The problem is that you are trying to do things like you'd do on Windows, surfing around web sites and downloading programs manually.

Try using the Ubuntu Software Center instead. It will download and install everything automatically, create menu launchers (when applicable), and also handles updates for all your installed applications.

Downloading things yourself from web sites is usually the last thing you'd want to do on Linux.
