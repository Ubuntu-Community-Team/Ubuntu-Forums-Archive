### Post by sparky-steve on 2008-10-17
Hi

I use Audio Tag Tool (tagtool in synaptic) - its very easy to use and works brilliantly (even over a network drive) but does rely on the metadata existing in the first place. Conversely, it can also create the ID3 tags from the filename, but it's a little too basic for my liking.

It also has the option to automatically move songs into directories, such as artist/album/

Hope it helps
