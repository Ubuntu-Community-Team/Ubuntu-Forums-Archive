### Post by tomillares on 2011-04-05
I'm traying puppy lucid. I'm amazed how I can use libreoffice, gimp, firefox... in such a tiny distribution!
I wanted to use gimp with gmic. It was easy. First install gimp since Quickpet. Then go to the GMIC web and download the plugin and copy in /root/.gimp/plug-ins/. Then, to make it run may install the libraries libfftw, libpng and zlib. To install them, you may allow the ubuntu-main repositories, in puppy package manager, and then install since there.
