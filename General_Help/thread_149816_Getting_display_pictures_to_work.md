### Post by f1dave on 2006-03-24
For msn:

To display people's display pics in your menu: 
Settings -> configure -> appearance -> select detailed view in contact display methods, and then check the "use contact photos where available"

To put one of your own up:
Settings -> configure -> accounts -> select your msn account -> user info tab -> select image

Hope that helps.
