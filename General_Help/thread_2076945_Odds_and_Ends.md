### Post by critin on 2012-10-27
>  I can't run Windows at present (not enough memory)?

Which version of windows?  I've run XP fine with 512.

It actually sounds as if the issues you've named are hardware or Windows faults.  Running virus scans and ccleaner might work wonders there if you wanted to keep that OS.

12.04 does take more resources to run well.  I'd suggest running something without Unity, as suggested in an above post.  There are plenty to choose from, good ones. I cannot run 12.04 Unity with 512 memory, but xubuntu is great.  Check youtube linux reviews to see different versions in action. You probably cannot run the 3D. 

Downloading time depends on your apparatus.  I use firefox with Down Them All add on, and it takes approx. 30 min. with cable wired internet. I don't use the torrents because DTA is just as fast.  I wouldn't try downloading with wireless.

I would bet that installing over windows would get rid of the slow issues, but it still would lag with Unity.

I'm not advising that if you don't have a Windows CD to reinstall if you wanted to.  If you do have the CD, just try reinstalling Win.  It might work like new.
