### Post by trikster_x on 2010-10-25
There are actually a few options for getting a windows like desktop clock.  The easiest and best supported would be the screenlets program. Simply type or copy/pasta this into a terminal:
```
sudo apt-get install screenlets
```

The screenlets program also has quite a few other interesting items in it.  If you want to see if there are alternate skins or more screenlets, simply visit this site:
[http://www.gnome-look.org]("http://www.gnome-look.org")
