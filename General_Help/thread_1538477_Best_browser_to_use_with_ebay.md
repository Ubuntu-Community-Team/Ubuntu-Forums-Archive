Does IE Tab even work on Linux? In the original version I'm pretty sure you needed to have Internet Explorer installed for it to work. Not sure about the recent versions.

In any case, I think the problem is likely to be with your network configuration rather than your browser. ### Post by boxcorner on 2010-11-11
I have yet to find any browser that works properly with eBay. I use eBay frequently, so it's very frustrating. I have probably tried all the browsers that run under Ubuntu, including all the ones listed in the Software Centre, as well as Chromium, Firefox, etc.  

The second best seems to be Firefox, which enables item descriptions to be be composed via *Text* or *HTML* tabs, but it appears to have difficulty handling some Javascript, intermittently. 

The best currently seems to be **Opera**.  Unfortunately, the *Text* tab is missing, so descriptions have to be entered as *HTML*, and I have difficulty getting it to upload pictures, but if you can put up with that inconvenience the rest seems to be fine!

So, I tend to compose my eBay items using Opera and then upload the photo(s) using Firefox.
