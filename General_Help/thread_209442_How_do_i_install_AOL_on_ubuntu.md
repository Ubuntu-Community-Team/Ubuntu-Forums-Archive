### Post by JoWilly on 2006-07-05
There are several usb adsl modems supported in linux. To help you out we need to know what modem you have.

But, as was suggested above, for any adsl connection the best option (and this is true for any os, be it linux, windows or mac,...) is to get an adsl router (with ethernet ports). You then configure your login/password in the adsl router and connections will be automatic. You then just plug that router into any computer with any os and it works without you doing anything.
