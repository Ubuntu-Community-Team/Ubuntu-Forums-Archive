### Post by Barriehie on 2008-06-08
> **paulroberts20 said:**

Yes, In the preferences I believe the last tab is Status/Idle and there is a checkbox where you can have it startup in the state it was in during the last session.  So, if you've only got the panel icon from the last session it will startup that way the next time your system is restarted.
