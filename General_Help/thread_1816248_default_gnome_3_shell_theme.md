### Post by hhh on 2011-08-02
Unless you moved it, the default theme is still in /usr/share/gnome-shell/theme . If you've lost it, save any link from here...
[http://packages.ubuntu.com/maverick/i386/gnome-shell/download](http://packages.ubuntu.com/maverick/i386/gnome-shell/download)
... extract it and the theme folder is in /usr/shell/gnome-shell.

What method did you use to switch themes in the first place?

HTH
