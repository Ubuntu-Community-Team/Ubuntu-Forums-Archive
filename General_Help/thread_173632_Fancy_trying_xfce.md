### Post by aysiu on 2006-05-10
If you just want to *try* XFCE, make sure you install it using *aptitude*, not *apt-get* or Synaptic: ```
sudo aptitude update
sudo aptitude install xubuntu-desktop
``` Using *aptitude* makes it easier to remove later.
