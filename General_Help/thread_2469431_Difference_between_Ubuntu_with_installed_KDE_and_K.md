### Post by sudodus on 2021-11-29
If you install the package **[FONT=Courier New]kubuntu-desktop[/FONT]** into standard Ubuntu, you with have Kubuntu's desktop environment plus the program selection of Kubuntu alongside **[FONT=Courier New]ubuntu-desktop[/FONT]** of standard Ubuntu Desktop.

You installed **[FONT=Courier New]kde-full[/FONT]** into standard Ubuntu Desktop. This is a slightly different desktop environment (mainly some settings and probably also wallpaper files etc differ) and I think also a different selection of programs.

There will be double tools for some tasks (for example file browser and terminal emulator), so some extra drive space will be occupied which is not a big issue nowadays with big drives. You will also be able to select 'session' at the log in screen between the Gnome desktop of Ubuntu and the KDE desktop.

But it is not easy to get rid of one of the desktop environments. If you try, you may break the desktop environment that you want to keep. So consider adding a desktop environment *irreversible*. (This is different from adding many simple program packages, that can easily be removed.)
