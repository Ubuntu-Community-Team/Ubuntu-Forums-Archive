### Post by MilkMoney on 2006-03-31
I'd like to run the gparted live program on an old computer to be able to partition the hard drive. I am unfamiliar with how to do that and am a noob to this. Can anyone give me a tutorial on what to do once I have downloaded the gparted live disk. What steps should I follow? Thank you!

### Post by Ramses de Norre on 2006-03-31
Fire up a cd burning program and select 'burn image' or 'burn iso' or something similar. You can't just copy the file to the cd!
Then start the other pc, make sure the cd-rom drive is before the hdd in the boot sequence (change it in bios otherwise), insert the cd before the POST is completed (= very soon) and then it should boot the live cd.
