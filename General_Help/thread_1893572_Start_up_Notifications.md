### Post by zero2xiii on 2011-12-10
Hay,

Yes that is one option, or just add thunderbird as a startup application by going to: System > Preferences > Startup Applications, Click on add >
Name: THunderbird
Command: thunderbird (im guessing, I dont have thunderbird installed to check this)
Comments: Leave blank

Click add and now thunderbird should be run on startup.

You could also try CloudSN, I'm using it with Thunderbird 8 in Gnome 2 (Lucid 10.04) since the Thunderbird Indicator extension stopped being updated to work with versions later than Thunderbird 3. CloudSN offers either to integrate into the messaging indicator in Unity and classic Gnome, or to run a standalone tray icon, the latter of which you would have to use in Gnome Shell, afaik.
