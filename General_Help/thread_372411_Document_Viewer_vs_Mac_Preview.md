### Post by ramjet_1953 on 2007-02-28
As stated by another happy camper, using Ubuntu Linux, whenyou lift the hood of Linux, you have a V12, not a 4 banger.

The great strength of Linux is its' configurability.

If you seach these forums for posts on fonts, you will see that it is easy to add fonts.

Also, using Automatx2 you can add all of the Microsoft core TTF's.
I wouldn't mind betting that you can also add the Mac fonts as well, if you do a bit of searching around.

If you are using an LCD screen, don't forget to check out ant-aliasing and pixel smoothing in the screen configuration settings.

It is just a matter of taking your time to learn about this wonderful new OS and don't be afraid to ask questions on these forums.
