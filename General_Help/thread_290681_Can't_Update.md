### Post by Jussi Kukkonen on 2006-11-01
You haven't necessarily done anything wrong... but it's a pretty good error message, isn't it? The package manager can't make up it's mind about some upgrade and needs to ask you if a possible solution is ok with you...

Try
```
sudo apt-get update
sudo apt-get dist-upgrade
```
in the command line and paste the output her if you don't understand what it's telling you.

EDIT: by the way: if you have added repositories into your  *sources.list *file, you should mention that.
