### Post by TDK800 on 2011-12-18
Currently when you have OpenOffice and multiple Firefox windows open when you press alt+tab it brings up 1 openoffice icon and 1 firefox icon and then you have to keep the firefox icon active or press down arrow to get a list of the open firefox windows.

Is there some way to disable this grouping so that alt+tab would give the full list of all open (Firefox) windows?

---

### Post by lechien73 on 2011-12-18
You can open CompizConfig Settings Manager, go to the Unity plug-in and disable the Switcher, then enable the Static Application Switcher. This switcher doesn't group the applications.

If you don't have the Settings Manager installed, then open a terminal window and type:

```
sudo apt-get install compizconfig-settings-manager
```
