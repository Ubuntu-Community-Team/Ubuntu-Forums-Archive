### Post by bingoUV on 2007-11-15
install mc (midnight-commander). Now just type mc in terminal, and you are in your dream environment. You can run commands as if you are in command line, yet you see all files in the current directory. Tab completion does not work, but come to think of it, you do not need it. Maybe you will like it.

In older days, mc used to be standard install. Sigh.
