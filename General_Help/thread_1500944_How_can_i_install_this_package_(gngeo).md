### Post by colorlessprism on 2010-06-03
looks like it is a "multiverse"  package: "Software restricted by copyright or legal issues (multiverse)" - Software that is "not free" and may require licensing. This software is not supported.

GUI-based repository management is normally accomplished via the "Software Sources" panel. This panel can be accessed via two menus:

Software Sources: System > Administration > Software Sources.
Synaptic: System > Administration > Synaptic >> Settings >> Repositories.
You will have to enter your password to gain access to the page.

Go to the "ubuntu software tab"
select multiverse. click "Close" to save your changes. A dialog box should appear, asking whether you'd like to update the list of repositories. Select "Reload" to update the list.
