### Post by Alex Farber on 2010-03-17
Is there fast way to run the terminal window from Nautilus in currently opened directory? Now I open the terminal, and execute cd ... command, copying cd argument from Nautilus Location box. Maybe there is something better?

---

### Post by mcduck on 2010-03-17
Sure, just install the "nautilus-open-terminal" -package and you'll get an option to open terminal in current directory in your right-click menu.

(you need to log out and back again after installing the package before it works)
