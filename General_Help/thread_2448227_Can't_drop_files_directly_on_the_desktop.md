### Post by TheFu on 2020-08-04
Sounds like you use gnome3.  The developers for that system don't think you need that so it isn't possible.
Fortunately, we have choices in Linux for the gui we run.  There are 50+ different GUIs.  20 are popular.  Pick whichever GUI you like.  No need to load a new OS, just install the GUI.

To prevent the settings of different GUIs from colliding, just create a new userid for each GUI you want to try out, then log out, on the login page, pick the other GUI, then login and see if the new GUI works for your desires.  Pretty simple, just a little time.
