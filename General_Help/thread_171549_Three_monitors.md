### Post by Ramses de Norre on 2006-05-07
First: in most cases your bios wont enable both cards by default, you'll need to go into setup and tell it do do so.

Second: make a new device section in your xorg.conf for the card with all the appropriate values like correct busid and driver for the card.

Third: define screen and monitor sections for it, make all identifiers match etc. and edit your server layout to use the three screens.
