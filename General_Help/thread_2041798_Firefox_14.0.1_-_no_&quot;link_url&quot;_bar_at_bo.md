### Post by Vaphell on 2012-08-13
what you say is somewhat strange. Firefox deprecated oldschool status bar long time ago (like several versions back) and all you can have is addon bar which is a watered down version of the old goodie. To get the full functionality back you need Status-4-evar addon
[https://addons.mozilla.org/en-US/firefox/addon/status-4-evar/](https://addons.mozilla.org/en-US/firefox/addon/status-4-evar/)
that bottom addon/status bar apparently can be toggled with CTRL+/
