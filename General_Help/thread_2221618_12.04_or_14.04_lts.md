### Post by Impavidus on 2014-05-03
I installed Xubuntu 14.04 64 bit two weeks ago. Now I'm dual booting with Xubuntu 12.04 32 bit, which is my old system. At the same time I increased the ram of my laptop, reason to go for 64 bit now. I didn't install every package of my old install on my new one, reducing the number of unused packages still starting some services and thereby increasing boot and login speed. The new system seems faster and uses more memory, but that may have to do with the other changes than just the newer release.

Xubuntu 14.04 looks good, although there are a few glitches left. It's well usable and I could make it look almost the same as my 12.04 install. Hardware compatibility should be the same in every flavour.
