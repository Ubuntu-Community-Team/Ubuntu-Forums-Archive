### Post by astroorion on 2009-02-15
Ok here's how I did this
Open Gimp then go to file and then open usally I set my save files from firefox to my document's.
Find the Hellraiser Cube Template and click on it
Next go to View then down to Zoom Select 200
Next go to tool's then down to toolbox click on toolbox on the very top left corner there's a square that say's Rectangle select tool click on that
Next go to the first square on the bottom of the hell raiser cube place the curser on the left corner then left click and hold and move right to the right corner next move down you should have a square outline covering the entire bottom cube.
Next let go of left mouse button then go to image then down to crop to selection.
You now have a section cut off two more to go just repeat the step's and you should have three section's of the Hellraiser Cube

### Post by astroorion on 2009-02-15
Ok Now how to put a Hellraiser cube together first you need to get compiz  go to system then administration then Synapatic Package Manager Click there enter your password next type in the Quick search compiz
install compiz.
Ok next go to system preferences click on compizconfig setting manager. 
Go to Desktopcube mark the box then go to Rotate Cube mark box ok next go to General option's
go to desktop Size Move the Horizontal Virtual Size to 4.
Ok next you can't have desktop icons for the cube to work if you have them just right click on the icon and move to trash
After you remove the Desk top Icons press alt-f2 then in the box type gconf-editor
 navigate to apps-nautilus-preferences
uncheck the box on the right that says show desktop
(this makes nautilus stop drawing the desktop and allows compiz to draw them instead) hence the no icons
after that is done Go to System Preferences then back to Compizconfig setting's Manager
Go down to utlity Click on wallpaper ok from the last post on how to crop the cube there should be three
Cut's SIX side's to a cube first top bottom four side's this is where the side's go I label the sides 1 and 2 the end cap's
Being 3.
Ok in wallpaper Click on new remember the first cube that was cut click on the folder browse place that one there plus change the position to scaled the other side is on the template chose either left or right on the cross 
So just hit new again and put 2  plus change position to scaled then new again Repeat 1 and 2 side.
Ok now for the next step Getting the End Cap's top and bottom of the cube.
Click on back check mark Wallpaper then head up to effect's
Check mark then Click on Cube Reflection and Deformation click on appearance you should see on the bottom Top image file's and at the Bottom
Bottom Image file's 
Click on New on the top Image Files click on the folder icon Browse place top there then repeat at the bottom file.
Ok next go to reflection uncheck everything leave Draw top Face checked and Bottom Face checked
next go to deformation select none.
You should have a Hellraiser cube next just need to add a back ground which can be done in Descktop Cube
through appearance then Skydome.
Enjoy
