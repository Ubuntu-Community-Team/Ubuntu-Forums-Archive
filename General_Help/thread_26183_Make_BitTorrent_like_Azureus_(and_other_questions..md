### Post by Kimm on 2005-04-12
if you want to lanuch the client without using the terminal you can do this:

Alt + F2

then type "btdownloadgui" in the box that aperes.

if Alt + F2 doesnt work, you can do:

Applications -> Run Application then do the same thing.

this is just about the same thing as typing it in the console but there are no annoying extra terminal windows running
