```

/home/user/.thunderbird
```


Use ctrl+h to view the hidden folders

### Post by oldfred on 2012-11-28
I copy profile folders from my Desktop  Ubuntu PC to my laptop when traveling. 

When first installed I open Firefox & Thunderbird once to create the hidden folders with the default profiles. I then copy my profile and edit profile.ini with my profile name instead of the one created.

More info:
[http://kb.mozillazine.org/Moving_your_profile_folder](http://kb.mozillazine.org/Moving_your_profile_folder)
