### Post by pmi2 on 2015-03-23
Dell laptops have a built in self test.  Have you tried running that on reboot?  
If not look it up on the Dell web site.

There could be many reasons for this, so for example if there is a problem writing the display memory.  Then later when you hover over something and that section of screen is repainted, the correct data is written, and the screen area is restored.
