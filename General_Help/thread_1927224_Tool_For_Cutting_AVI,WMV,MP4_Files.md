### Post by forrestcupp on 2012-02-17
Any of the video editors in the repos should be able to handle something simple like that. A few of them are PiTiVi, OpenShot, Kdenlive, and Kino. You might have to install Ubuntu restricted extras to deal with some of those file formats.
