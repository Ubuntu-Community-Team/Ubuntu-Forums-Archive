### Post by barisurum on 2009-07-22
If your CPU is a 64 bit one then you should download amd64 iso file otherwise if 32 bit then i386, you can download and install the i386 one on a 64 bit machine but the programs which take advantage of 64 bit computing will run slower and if you have 4gbs of ram then you wont be able to use all of it. So I suggest installing amd64 if you have a 64 bit CPU.
