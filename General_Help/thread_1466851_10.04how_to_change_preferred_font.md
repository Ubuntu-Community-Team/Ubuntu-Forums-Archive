### Post by charrah on 2010-07-25
If you can make sense of this then this might be what you're looking for. Ubuntu's font system may be so borked by now that having a conf file may not even help. 
```
https://wiki.ubuntu.com/BetterCJKSupportSpecification/FontConfig
```

If your problem is the same as mine was (lack of antialiasing or hinting in CJK fonts) then you might be able to fix it by uninstalling the package ttf-wqy-zenhei. Apparently it's been causing a lot of problems with the font system. I used the "remove completely" option, seeing as other workarounds deal with its conf file, but I'm not sure if it matters.
