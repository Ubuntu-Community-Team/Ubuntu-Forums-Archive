Did you perform any system update?  If yes, then it is probably why you are starting to see system lock-ups.  Re-install Ubuntu and do not perform any update when the update manager asks for.  You will miss any security updates, but you will have a working computer.  BTW, I have been using Linux for more than 5 years with no security issue so I think the risk is minimal if you do not perform any update.
