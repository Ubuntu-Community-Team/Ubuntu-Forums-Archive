### Post by sanderd17 on 2011-06-27
Usted puede utilizar Ubuntu dentro de Windows (con un método de instalación llamado Wubi, que es tan fácil como la instalación de un programa grande) o al lado de las ventanas (que pone en el Live CD y reiniciar el ordenador).

Ubuntu está disponible en muchos idiomas, por supuesto español. Si usted encuentra un programa que no esté traducida correctamente, se puede fijar en Launchpad: [https://translations.launchpad.net/ubuntu](https://translations.launchpad.net/ubuntu)

This message was translated with google translate. Original message: 

You can use ubuntu inside windows (with an installation method called wubi, it's just as easy like installing a big program) or next to windows (putting in the live CD and rebooting your computer).

Ubuntu is available in a lot of languages, certainly Spanish. If you do find a program that isn't translated correctly, you can fix it at Launchpad: [https://translations.launchpad.net/ubuntu](https://translations.launchpad.net/ubuntu)

### Post by sanderd17 on 2011-06-28
Ubuntu tiene un montón de ventajas, pero también algunas desventajas.

Ubuntu no tiene virus. Como resultado, usted no tiene que instalar un antivirus que ralentiza el ordenador. Si quieres un programa de hacer algo en Ubuntu, no tiene que buscar en el Internet, usted sólo tiene que ir al Centro de Software Ubuntu (comparables a las tiendas de aplicaciones para teléfonos móviles, pero Ubuntu lo ha hecho desde 2004, cuando fue fundada).

Pero la ventaja más importante de Ubuntu es la libertad. Usted tiene la libertad de hacer cualquier cosa con Ubuntu. Como ejemplo, no me gusta la nueva interfaz que crearon (que se llama la unidad), pero tengo la libertad de cambiar a la interfaz a que yo quiero, yo puedo cambiar a una interfaz clásica de GNOME, un nuevo interfaz de gnome-shell, un KDE interfaz, una interfaz XFCE ...

Literalmente todo lo que se puede adaptar, y eso te hace aprender mucho acerca de las computadoras.

Por otro lado, Linux no es Windows. Por favor, no trate de ejecutar programas de Windows en Ubuntu. Puede ejecutar la mayoría de los programas, pero sólo funciona la mitad y se ven feos. Para la mayor parte de los programas, tenemos programas de reemplazo gratuito. Por ejemplo: tenemos Firefox en lugar de Internet Explorer, en lugar de LibreOffice de Microsoft Office, Banshee como reproductor de medios. Esto es sólo el grueso de configuración por defecto, usted tiene la libertad para cambiar su navegador para cromo, para editar documentos con AbiWord o utilizar otro reproductor de medios como Rhythmbox o Amarok.

Pero no tenemos reemplazos para un montón de juegos. La mayoría de los jugadores que instalar Linux también mantener Windows en todo, sólo para jugar.

Algunas capturas de pantalla: [http://ubuntuforums.org/showthread.php?t=1772639](http://ubuntuforums.org/showthread.php?t=1772639)

ORIGINAL MESSAGE:

Ubuntu has a lot of advantages, but also some disadvantages.

Ubuntu does not have viruses. As a result, you don't have to install an antivirus that slows down your computer. If you want a program do to something on Ubuntu, you don't have to search it on the internet, you just go to the Ubuntu Software Center (comparable at the app stores for mobile phones, but Ubuntu has it since 2004, when it was founded).

But the most important advantage of Ubuntu is the freedom. You have the freedom to do anything with Ubuntu. As example, I don't like the new interface they created (it's called Unity), but I have the freedom to change to whatever interface I want, I can switch to a classic gnome interface, a new gnome-shell interface, a KDE interface, an XFCE interface ...

Literally everything can be adapted, and that makes you learn a lot about computers.

On the other hand, Linux is not Windows. Please don't try to run Windows programs on Ubuntu. You can run most programs, but they will only work half and look ugly. For the biggest part of the programs, we have free replacement software. For example: we have firefox instead of internet explorer, Libreoffice instead of Microsoft Office, Banshee as media player. This is off coarse only the default setup, you have the freedom to change your browser to chrome, to edit documents with abiword or use another media player like Rhythmbox or Amarok.

But we don't have many replacements for a lot of games. Most gamers that installed Linux also keep Windows around, just to play games.
