### Post by raja on 2007-03-30
You can use wine to run .exe files previously installed on a windows partitions or to install windows programs and run them. Check out at the wine site to see if the program you want to use works well with wine. To use notepad for example, you just navigate to the directory containing notepad.exe and either right click and choose 'run with wine' or from the terminal, do ```
wine notepad.exe
```
