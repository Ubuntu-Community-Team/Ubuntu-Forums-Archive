### Post by fragos on 2008-10-25
Global icon size is controlled in Nautilus. In the Nautilus "Edit" drop down menu select "Preferences" and then set "Default zoom level" under "Icon View Defaults".

### Post by CowzRule on 2008-10-25
> **rmcellig said:**
> Thanks. It worked but is there an easier way to access the Configuration editor?

Right click the "Foot" icon on the panel and select "Edit Menu" In the left column click on "System Tools" then in the right column look for "Configuration Editor" and click the box next to it then click "Close". Now Configuration Editor will be in System Tools when ever you need it.
