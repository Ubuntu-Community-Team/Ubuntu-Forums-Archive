The easiest way is to just enable the universe repositories ([http://ubuntuguide.org/#extrarepositories](http://ubuntuguide.org/#extrarepositories)) and then use synaptic to install wxpython.

The Ubuntu universe repositories have wxpython2.5.3 in them,
