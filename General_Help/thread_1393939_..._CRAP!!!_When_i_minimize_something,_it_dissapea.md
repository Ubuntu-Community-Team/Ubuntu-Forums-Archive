### Post by NightwishFan on 2010-01-30
See if the correct applet is on your panel. Go to the bottom panel, right click and hit "Add to Panel". Search for "Window List" and add it to the panel. Your windows should show up.

If not try to use alt-tab to see if they are still open.

If you did not uninstall any standard programs, this command will ensure that your Ubuntu has all default software. This will require your password, and it will not show up when you type it. Open a terminal and type:
```
sudo apt-get install ubuntu-desktop
```
