### Post by lithopsian on 2011-04-07
Good practice is to keep data files in a defined location, ideally on a separate partition.  If you mix them in all over your primary Ubuntu partition, it is difficult to save them if you ever have to do a complete reinstall.

I'm not sure what you mean by remove the old applications.  You can remove any application you want at any time.  If you want to go back to a "clean" install the way the Ubuntu gods intended, then just install Natty from a Live CD instead of upgrading.  But read paragraph 1 :)
