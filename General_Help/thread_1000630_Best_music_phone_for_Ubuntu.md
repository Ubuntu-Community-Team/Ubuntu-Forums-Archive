### Post by geokker on 2008-12-03
I currently have a no-frills mobile phone and a Creative stone mp3 player. My Phone is failing so I thought I'd replace it with a music-capable phone and eliminate the need for the standalone mp3 player. I'm having a hard time finding a cheap-ish phone that at least matches the functionality of my current set-up. This is what I want:

1. 2GB minimum storage for mp3 files
2. 3.5mm headphone jack
3. Mini USB connection (not micro)
4. Mounts as a USB disk
5. Drag and drop from Nautilus/Thunar i.e. no dependence on any propritary middle-man software
6. Make and receive calls and texts

The only phone that seems like it might fit the bill is the Nokia Xpress music phone 5310. I have no idea if it will support 4. and 5.

Can you advise?

### Post by poing on 2008-12-03
The Nokia 5310 is a good phone. It uses micro USB though. Ubuntu 8.10 introduced some problems mounting it as a USB disk. (See here: ([http://sudan.ubuntuforums.com/showthread.php?t=975622](http://sudan.ubuntuforums.com/showthread.php?t=975622) The fix posted there worked for me.)

Dragging and dropping files and music using Nautilus works like a charm once the phone mounts. To my knowledge there is no way of making/receiving calls texts through Ubuntu with this phone.

Hopefully this helps :)
