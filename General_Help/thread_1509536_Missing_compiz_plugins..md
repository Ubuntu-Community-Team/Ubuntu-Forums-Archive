### Post by gingivere0 on 2010-06-14
You could try
```
 sudo apt-get install compiz-fusion-plugins-extra 
```
and if that doesn't work, open up the CompizConfig Simple Settings Manager and look at the top and click the drop-down menu and put it to "Ultimate."  These steps should allow you to set your animations how you like.
