### Post by Jussi Kukkonen on 2006-10-31
Almost all distributions patch their kernel, but the patches tend to be 'details' -- you can definitely trust e.g. memory management to be the same in all linux kernels  (of the same version). There is no reason to talk about the "Ubuntu kernel" unless you are discussing some very specific details.

*The Linux Kernel* ([http://en.tldp.org/LDP/tlk/](http://en.tldp.org/LDP/tlk/)) is comprehensive but quite old. *Wonderful World of Linux 2.6* ([http://kniggit.net/wwol26.html](http://kniggit.net/wwol26.html)) is a good discussion on the new features in linux 2.6. The O'Reilly book ([http://www.oreilly.com/catalog/understandlk/](http://www.oreilly.com/catalog/understandlk/)) is probably your best bet, but it's not available on the web.
