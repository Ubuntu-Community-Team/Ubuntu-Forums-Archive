### Post by nkri on 2008-12-18
Yep...go into compizconfig-settings-manager, and click General Options (it should be the first one).  Near the top of the window, there should be an Export button.  Click this, name it what you want, and be sure to let it save the default settings.

Now, log in to the other user's account.  Go into CCSM>General>Import, find the file in /home/*first-username*/*wherever-you-saved-the-file-before*, and click OK.  This should make the settings the same for both users.

Good luck!
-nkri
