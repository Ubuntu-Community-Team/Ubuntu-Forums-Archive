### Post by ntetreau on 2008-10-18
Sorry, but I fail to see what libmtp would have to do with this problem.  Nevertheless, I've had this problem before and permently fixed it using suggestions from this thread:
[http://ubuntuforums.org/showthread.php?t=91455](http://ubuntuforums.org/showthread.php?t=91455)

It's been a few weeks so I don't remember exactly, but I think this fixed it:
Replace username by your actual username
```
chown username /home/username;
chown username /home/username/.dmrc
chmod 744 /home/username
chmod 744 /home/username/.dmrc
```
