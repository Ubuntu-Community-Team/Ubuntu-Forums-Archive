### Post by zvacet on 2009-04-15
Ubuntu can read and write on your Vista partition,but maybe it is better solution to have shared partition formated as NTFS which can both OS access.It is good idea to have separate home partition in Ubuntu.That way you will keep your settings and files in case that you want to reinstall make fresh install of new version of Ubuntu....About partitioning and installing Ubuntu you can read more [here.]("http://psychocats.s465.sureserver.com/ubuntu/index.php")

Before you install Ubuntu defragment your Vista few times and then resize HDD.You can do it with [Gparted live CD.]("http://gparted.sourceforge.net/")
