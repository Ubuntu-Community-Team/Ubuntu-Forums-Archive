### Post by grahammechanical on 2011-04-29
No need for system restore points. Linux does not usually break unless the user messes things up. Besides, you have the install disc with Ubuntu. And you are allowed to install on as many machines as you like and you can make copies of the install discs and distribute them. Restore points are Microsoft's answer to selling you an operating system but not providing the install discs. Thus preventing you from re-installing when the system breaks.
