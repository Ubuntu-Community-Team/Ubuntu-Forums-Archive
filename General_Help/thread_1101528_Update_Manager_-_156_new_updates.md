### Post by Therion on 2009-03-20
> **jeff_flynn said:**
> I have opened my Update Manager, and it tells me "You can install 156 updates". I would like to know if there is a way I can find out what each update is for?
Your update manager screen usually gives a short blurb that tells you that.  You may need to "expand" it by clicking on the dropdown menu at the bottom though.  

It's the one labeled "**Description of Update**".  See this illustrative illustration:

[http://www.techotopia.com/images/6/66/Ubuntu_linux_update_manager.jpg](http://www.techotopia.com/images/6/66/Ubuntu_linux_update_manager.jpg)
