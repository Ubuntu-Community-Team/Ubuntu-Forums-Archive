### Post by mozetti on 2007-05-03
What do you mean by "import"? Did you (1) just open the file with OO.o or did you (2) create a new OO.o spreadsheet and then choose to import data from the Excel spreadsheet?

If it's #2, then that's the way it's supposed to work. Even if you import from Excel to Excel, it only imports one sheet at a time. But, OO.o can open Excel spreadsheets, so just open the original .xls file using OO.o and then save it as an OO.o file (if that's your goal -- to have an OO.o-type spreadsheet file).
