### Post by tredegar on 2011-01-03
Your file associations are messed up. This happens all the time. I don't know why.

Open a terminal
**nautilus**
Your file browser opens.
R-click on any folder
Open with .... Other application
Choose "File Browser"
Check "Remember this".
Open.

Done
