### Post by Patb on 2008-05-16
Reader, you could try printing your "few pages" to a (postscript) file and then use Ghostscript to convert that to PDF.  

To convert I would use a command like:
```
gs -q -dCompatibilityLevel=1.3 -dNOPAUSE -dBATCH -sDEVICE=pdfwrite -sOutputFile=OutputYourFewPagesFileName.pdf InputYourFewPagesFileName.ps
```
I cannot be sure whether this will reduce the size compared to CUPS-PDF but it might be worth a try.  If it does work, please post or mark the thread solved.  

Cheers, Pat.

### Post by reader4 on 2008-05-21
This worked perfectly.  The test PDF started with 14 pages, 523 kB.  I printed only pages 5-13 using 3 methods and got:

1. Print to file (PDF) - 3.0 MB.
2. Print to PDF (Cups) - 2.3 MB.
3. Print to file (PS) and convert to PDF - 373 kB.
