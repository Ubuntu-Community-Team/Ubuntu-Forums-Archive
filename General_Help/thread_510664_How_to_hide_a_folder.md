### Post by NeuralEskimo on 2007-07-26
One solution, and the best IMHO, is to prepend the directory name with a period. For example, say you want to make a directory named "music" to make it hidden, name it ".music" instead. Works for files too. BTW, typing "ls -a" on the commandline will show you all of your hidden files and directories. Do this in your home directory. It will be fun and educational. Hope this helps....
