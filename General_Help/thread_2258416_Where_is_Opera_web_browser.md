### Post by sisco311 on 2014-12-27
Please post the error message.

The latest Opera for Linux is only available for 64-bit systems. If you have a 32-bit OS you can download an earlier release or wait until the 32-bit version is released.

You can download the latest 32-bit version from:  [http://www.opera.com/download/guide/?os=linux-i386&ver=12.16&local=y](http://www.opera.com/download/guide/?os=linux-i386&ver=12.16&local=y)
