### Post by CaptainMark on 2013-09-22
If you've only installed cinnamon then this doesn't install nemo automatically, you have to specifically install nemo too with```
sudo apt-get install nemo
```You should then probably go about making nemo the default file manager because otherwise you never know which is going to open up when, there's a good guide here [http://itsfoss.com/install-nemo-default-file-manager-ubuntu-1304/](http://itsfoss.com/install-nemo-default-file-manager-ubuntu-1304/)
