### Post by P4man on 2009-08-22
you can zero fill it to wipe it clean. But tbh, if formatting fails, maybe the drive is dying. Still, if you want to try:

to find out what your harddisk is called:

```
sudo fdisk -l
```

then 

```
sudo dd if=/dev/zero of=/dev/yourharddisk
```

this will wipe everything from that disk, no questions asked, no output or feedback until it is finished!! Beware!

A better option is probably to test your drive for defects. Ultimate bootdisk is a good tool for that:

[http://www.ultimatebootcd.com/](http://www.ultimatebootcd.com/)

It contains harddrive diagnostics programs for every HD vendor.
