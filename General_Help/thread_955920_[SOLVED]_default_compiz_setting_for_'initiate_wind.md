### Post by alwiap on 2008-10-22
I am using Linux Mint, and I came from Ubuntu.  When I was using Compiz in Ubuntu, I was able to close windows using the 'window picker' (I enabled it to be the right side of the screen, it's the 'expose' option).  However, by default in Mint, I can't close any of the windows, just select which one to switch to.  I'd like to be able to close any of the windows using middle-click, and be able to bring one of the windows to focus using right-click, like how it was in Ubuntu.  That's all, thanks.  I know it's probably an option in ccsm, but I have looked for a while and was unable to find it.

---

### Post by alwiap on 2008-10-23
I solved it by enabling the Scale Addons plugin.  By default everything I was used to worked as was with the middleclick and rightclick.
