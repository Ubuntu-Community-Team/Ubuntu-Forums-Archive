I imaged my ubuntu OS with remastersys using the "backup" option to backup the complete system and user data (about 2gig). I burned the iso to a dvd-r and booted from the dvd-r. The remastersys loads fine, but how do I actually "restore" that backup image if I have a system crash?

### Post by logos34 on 2009-01-01
When you boot from the Remastersys CD, you should get a menu screen.  Choose the option 'Install Custom Live CD'.  Or 'Start Custom Live CD in Graphical Mode', open a terminal and type **sudo ubiquity**
