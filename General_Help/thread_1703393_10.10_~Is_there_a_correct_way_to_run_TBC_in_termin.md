### Post by 3rdalbum on 2011-04-26
You need to install the 'transmission-cli' package first, and then the command you need will be inside that package. You can look at it in Synaptic Package Manager; select the package, go to Properties, then "Installed Files" and it will list all the files that have been installed in that package.
