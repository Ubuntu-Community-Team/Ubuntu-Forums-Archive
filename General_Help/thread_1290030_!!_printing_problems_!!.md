### Post by P4man on 2009-10-13
On the desktop:
system > administration > printing
Right click the printer and set it to "shared"

On the laptop system > administration > printing
Server settings, make sure "show printers shared by other systems" is checked. 

If you got problems, click the "problems?" link on the server setting page.
