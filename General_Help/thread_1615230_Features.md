### Post by snowpine on 2010-11-06
Ubuntu and Kubuntu share the same "repository" of available software. Any application you like from one can be easily installed in the other using the Software Center or the appropriate terminal commands. For example to install Gparted just type 'sudo apt-get install gparted' in the terminal (or use the Software Center).
