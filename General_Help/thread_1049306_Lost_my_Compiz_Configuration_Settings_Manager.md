### Post by Arrgoss on 2009-01-24
I lost the Compiz Configuration Settings Manager in my system as a result of a series of events... Any help will be welcome!

I had to remove Compiz from my system because due to a conflict I could not log in. From the Failsafe Xterminal I removed everything by using
```
sudo apt-get remove compiz*
```
That solved my problem and I could get back to my Ubuntu. Now, I wanted Compiz back so I reinstalled via the Synaptics. My problem now is that I don't have an access to the CompizConfig Settings Manager from my System  menu, nor I know how to get there...
Any hint?

---

### Post by Strohfeuer on 2009-01-24
Try to look at System->Preferences->Main Menu. When it's hidden you can check the box to see it in the menu.
On the other hand if you install the Compiz Settings Manager via Add/Remove it should work.
