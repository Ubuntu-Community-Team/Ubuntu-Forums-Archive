### Post by FormatSeize on 2011-06-14
You can't install Visual Basic directly. So you have two choices:[LIST=1]
[*]Use another program that is something like Visual Basic, but that requires learning how it works, which may not be what you are looking to do.
[*]Install wine, which allows you to run Windows programs, and then install Visual Basic and use it under wine.
[/LIST]
