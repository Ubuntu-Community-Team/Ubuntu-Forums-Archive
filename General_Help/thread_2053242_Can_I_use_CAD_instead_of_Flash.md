### Post by mcduck on 2012-09-05
Web browser's won't run any CAD files, unless you use some browser plugin (which most people won't have and will not install). CAD might, of course, be useful for yourself when creating the content, you just need some other way to present it on the web page.

I'd say the best options are Flash and HTML5. Mpst people would be able to access the content if you use either one of those.

And, with some reservations, Unity 3D could make similar stuff but it requires a plugin (not available for Linux yet, but will be in near future), or Chrome browser with NaCL support switched on from settings. Once again extra steps for the user to access the content, which means that they might just decide to ignore your web page instead.
