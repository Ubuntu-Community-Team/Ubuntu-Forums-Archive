### Post by red_spyder on 2009-08-31
I usually use the terminal when I'm installing a new program. Using the command
```
sudo apt-get install
```and then the program you wish to install will install said program and all of its dependencies. It just makes the whole process automated.
For example, typing
```
sudo apt-get install wine
```will install the current version of wine as well as all of its dependencies (dependencies are all of the files that programs need to run properly).
(Also, the command 'sudo' will ask for your password. This allows you to run the command 'apt-get')
