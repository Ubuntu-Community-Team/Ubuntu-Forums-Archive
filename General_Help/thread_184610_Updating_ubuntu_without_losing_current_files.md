### Post by Suzan on 2006-05-30
Yes, just update to dapper.

The easiest way is to use the ubuntu update-manager.

Be sure, that you have the newest package for the update-manager in breezy!

Then start it in the terminal:

```
sudo update-manger -d
```


But it's ALWAYS a good idea to backup your data before making an update!!!
