### Post by elmago79 on 2009-02-04
Do you have access to a USB pen?

You could simply use [UNetbootin]("http://unetbootin.sourceforge.net/") for Windows to create a Live USB drive. 

If you don't have access to a USB drive, you can use [Wubi]("http://wubi-installer.org/") to install Ubuntu. Read the [FAQ]("http://wubi-installer.org/faq.php") before you do anything with Wubi so that you may know how to use the .iso you've already downloaded.

Finally, netbooks tend to not work out of the box with Ubuntu, so be sure to check back with any problems you run into.
