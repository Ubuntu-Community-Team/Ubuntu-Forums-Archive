### Post by dcstar on 2007-02-12
> **SpanishFranks said:**
> thanks for clearing that up. i thought swiftfox was a complete browser based on firefox

It is, it is Firefox complied with particular CPU optimisations.

In previous versions of Ubuntu, components of the Firefox engine were used by the O/S, so you could not uninstall Firefox without causing major problems to your install.

I'm not too sure of the 6.10 situation, but if you install Swiftfox (and use it), there is really no need to touch the old Firefox install.

If you set up your Repositories to use the Swiftfox deb packages, then future upgrades will be delivered automatically.
