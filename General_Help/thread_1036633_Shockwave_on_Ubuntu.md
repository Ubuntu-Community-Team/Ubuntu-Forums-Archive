Flash player and Shockwave player are two separate things (although Flash is often called "Shockwave Flash).

Flash player is not able to play Shockwave content, there is no Shockwave player for Linux and most likely will never be since Shockwave is already disappearing.

(In the old days Flash was only able to handle vector graphics, no bitmap graphics (like photographs) or video. You really wouldn't have wanted to play any video material on web with the typical internet connection speeds of a modem.

Shockwave was most commonly used to create multimedia CD-ROMs, and it was able to handle bitmap graphics, video and sound, and you could also add flash content into Shockwave.

As the internet connection speeds have been growing (and use of multimedia CD-ROMs decreasing) most of the features have been copied from Shockwave to Flash, and now Flash plays video and uses pixmap graphics so there isn't much use for Shockwave any more.)

Like the dialog says, that's the Shockwave Flash-plugin (AKA Flash plugin). It plays Flash content, not Shockwave content.

Wkipedia pages about Flash and Shockwave:
[http://en.wikipedia.org/wiki/Adobe_Flash]("http://en.wikipedia.org/wiki/Adobe_Flash")
[http://en.wikipedia.org/wiki/Adobe_Shockwave]("http://en.wikipedia.org/wiki/Adobe_Shockwave")

And Adobe's download links for both plugins:
[http://get.adobe.com/flashplayer/]("http://get.adobe.com/flashplayer/")
[http://get.adobe.com/shockwave/]("http://get.adobe.com/shockwave/")

The only way to run Shockwave content on Linux is to install Windows-version of IE or Firefox, and the Shockwave plugin for Windows with Wine. Flash, on the other hand, has a native plugin for Linux.

edit: Here's also Adobe's test page for both Shockwave and Flash plugins: [http://www.adobe.com/shockwave/welcome/]("http://www.adobe.com/shockwave/welcome/"). If you really are able to see _both_ test movies on Linux, without running the Windows plugin with Wine, I'd definitely be interested in hearing how you've done it. ;)
