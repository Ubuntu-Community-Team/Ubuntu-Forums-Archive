I have just found an odd problem: I am using CGoban3 to connect to the KGS go server (CGoban3 uses Java Webstart) and when I try and type in my password it will not work. I can edit the text in the username field but not enter anything into password.

### Post by green0eggs on 2009-03-31
Oh I forgot respond. In case anyone else has the same problem, there is bodge solution I figured out:

Run CGoban3 and type username, before attempting to type the password, right-click the SCIM keyboard icon in the Notification Area and Exit. Then the password is able to be typed. Once logged in, go back to SCIM icon (which, for me, comes back after a second or two) and Reload Configuration.

Not very elegant, I know but it works.
