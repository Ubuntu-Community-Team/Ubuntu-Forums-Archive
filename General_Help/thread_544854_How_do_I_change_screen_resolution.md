### Post by JRV on 2007-09-06
I have an Acer AL2416W monitor and a Nvidia graphics card, both are capable of supporting 1920x1200 screen resolution.

I installed the nvidia driver in the Restricted Driver Manager which gave me more choices but the only wide screen resolution available is 960x600.
 
Is there a way to get 1920x1200 resolution?

Thankyou

---

### Post by smithman89 on 2007-09-06
run nvidia-settings in the terminal.
Click on the X Server Display Configuration tab.
Use the dropdown menu to find your resolution.
Click apply.
Make sure that it works, and enjoy!
