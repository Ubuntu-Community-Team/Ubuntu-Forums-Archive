### Post by An Sanct on 2011-07-05
Hi there!

Kompozer is a HTML editor, so every browser can open your saved files and display them.

If you are trying to edit php files, then you should consider installing [LAMP]("http://www.techtickle.com/install-lamp-ubuntu-maverick-lucid-easy.html"), this is a full web server with PHP and MySQL support, the kind, web developers use. :) It will make your computer a web server, in the browser you can access it via '[localhost]("http://localhost")' or [127.0.0.1]("http://127.0.0.1")
