### Post by JuanFerS on 2006-01-11
Edit: I don't know if this package comes with the default installation, but I downloaded and installed it anyways.

Download and install the kdm theme manager from kde apps.

Edit: Then go to System settings --> KDM Theme Manager

You can install new themes from there.

### Post by sharke on 2006-01-13
# Go to Control Center
# Select "Appearance & Themes"
# Select "Splash Screen"
# Click "Add..."
# Browse for the downloaded file
# Select the new Theme
# Restart KDE
