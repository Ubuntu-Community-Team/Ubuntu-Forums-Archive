### Post by JohnSaga on 2011-02-13
For Ubuntu 10.10.
Is it possible to assign a key shortcut to start a program eg: "F8 to start FireFox or "Ctrl+F" , or any other key combination to do that?

Is there a post on that already?

---

### Post by vanadium on 2011-02-13
You can do this using System - Preferences - Keyboard shortcuts.

---

### Post by JohnSaga on 2011-02-13
I did check that but how do I type in there that I want FireFox on F8?

---

### Post by vanadium on 2011-02-13
"Add" button, command: "firefox", "Name": whatever you want ("Firefox", for example). Then highlight your new entry (it will be under "custom entries"), click the column "Shortcut" and press the shortcut key of your choice.
