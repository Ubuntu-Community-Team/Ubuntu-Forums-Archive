Is it a fresh install of Ubuntu?


When you get that tan screen, press: CTRL+ALT+F2
If you get into virtual terminal, login, and type in:
```

killall compiz.real
killall compiz
metacity --replace --display :0 &
```

Then press CTRL+ALT+F7 and see if it works.
