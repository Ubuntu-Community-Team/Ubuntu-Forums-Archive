### Post by DrMelon on 2008-09-20
Are you Dual-Booting with Windows using Wubi? If you are, you may need to defragment your hard drive with Windows. Or you can run "chkdsk /r" in Windows to check for errors on the drive.

If you're not dual booting, hold F8 while the computer starts and select Ubuntu Recovery Mode. When the error happens, you should get a description of what is going on, and how to fix it.
