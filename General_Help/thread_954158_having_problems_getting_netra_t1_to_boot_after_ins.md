### Post by Twirlcan on 2009-02-03
sometimes this helps: (an example)

```

OK> devalias

```

Get your disk alias from this command.

then...

```

boot  /pci/scsi@5/disk@0:1,\boot\vmlinux.gz

```

This specifies that opfw is going to look on the pci bus on the  pci device number 5 then a disk at target 0 and partition 1 named boot ..then the file it will look for.

I always use Solaris (and sometimes netbsd) on Sun equipment so you want to check what file grub boots in your boot partition and then enter that in openfirmware.
