### Post by oofnik on 2005-11-30
I tried installing package xmms-alarm, but it appears to be broken some how. I had it running fine with Hoary, but after doing a clean install with Breezy it's very buggy and acts weird. No matter what I set for the volume it will climb up past the max and then drop back down eventually to zero. I was late for school because of that! :mad:

Anyway I fixed it by downloading the Redhat 9.0 RPM from the xmms-alarm homepage, [http://www.snika.uklinux.net/?p=xmms-alarm](http://www.snika.uklinux.net/?p=xmms-alarm) and converting it to deb with alien. Works fine now. So if anyone wanted to know.. now you do :D
