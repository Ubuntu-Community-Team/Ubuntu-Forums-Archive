### Post by FakeOutdoorsman on 2008-07-25
Your RAM is your bottleneck, but assuming you can't add more I would suggest a custom Ubuntu install using an anternate disc or the [minimal CD image]("https://help.ubuntu.com/community/Installation/MinimalCD").  These allow you to install a command-line system which you can then build up using only the packages that you want.  The resulting system will be much faster.  I've done this with a much slower system than yours using Openbox (KDE is a hog) and it works fine.  Some good resources:
[Ubuntu Installation for Low Memory Systems]("https://help.ubuntu.com/community/Installation/LowMemorySystems")

If you don't want to reinstall:
[Howto: Set up Hardy for speed, v1.0]("http://kmandla.wordpress.com/2008/05/04/howto-set-up-hardy-for-speed/")
[urukrama's Openbox guide]("http://urukrama.wordpress.com/openbox-guide/")
