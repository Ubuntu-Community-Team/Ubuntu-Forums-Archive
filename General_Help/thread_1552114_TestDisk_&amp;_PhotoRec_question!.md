### Post by aeiah on 2010-08-13
since it only offers generic linux packages, source, and RPMs (which red hat / fedora uses) you'd go with the generic linux packages (Linux, kernel 2.6.x i386/x86_64 )

however, photorec and testdisk are available in the ubuntu repositories, so it's recommended that you install from there. either search for them in synaptic or do 
```

sudo apt-get install testdisk

```

i think photorec is installed along with testdisk automatically.
