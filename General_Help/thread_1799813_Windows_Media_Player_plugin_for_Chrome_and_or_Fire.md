### Post by auscanstom on 2011-07-08
Hi all,

I am trying to watch a paid TV channel through it's online site and I am told that it only works with a Windows media Player plugin in either Chrome or Firefox as I use Ubuntu. The websites customer service team tells me that it works fine for Windows users and they do have Ubuntu users who have made it work.
Would anyone know how to install Windows Media Player plugins on Chrome or Firefox? Or any alternatives? Please help.

---

### Post by lovinglinux on 2011-07-08
You probably have a Windows Media Player compatible plugin installed, most likely Totem. Type **about[noparse]:[/noparse]plugins** in the address bar to check. I would recommend gecko-mediaplayer tho.

Anyway, most likely that the site is using Silverlight, so you need to install [Moonlight]("http://www.go-mono.com/moonlight/").

### Post by WorMzy on 2011-07-09
Install Mozilla's [Add-on Compatibility Reporter]("https://addons.mozilla.org/en-US/firefox/addon/add-on-compatibility-reporter/?src=api") and re-enable the Moonlight addon.
