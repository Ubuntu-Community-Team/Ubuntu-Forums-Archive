### Post by gsmanners on 2010-01-03
For generic installation help I would refer to this: [http://www.psychocats.net/ubuntu/](http://www.psychocats.net/ubuntu/)

For a general developer reference, you should install the following:

build-essential
manpages-dev

For Gnome or GTK:

devhelp
libgtk2.0-doc
libglib2.0-doc
libpango1.0-doc

And whatever else you need (just look for the -doc prefix).

I don't know about Qt. I suppose you could check out [http://techbase.kde.org/Welcome_to_KDE_TechBase](http://techbase.kde.org/Welcome_to_KDE_TechBase) or a related web site.
