Here's a few I found using the [Ubuntu Package Search]("http://packages.ubuntu.com/"):
[LIST]
[*]blogtk - GTK Weblogging client 
[*]blosxom - light, feature-packed weblog app with plugin extensibility 
[*]drivel - Blogging client for the GNOME desktop

[/LIST]
