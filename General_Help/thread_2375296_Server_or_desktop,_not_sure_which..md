### Post by SeijiSensei on 2017-10-23
The only real difference between a server and a desktop distribution is the presence of a GUI.  Server has no GUI; you need to do everything from the command prompt.  If you're not comfortable with using the terminal, I suggest using a desktop distro.

I control the PC (running Kubuntu 14.04) connected to my TV with a wireless keyboard and mouse from Logitech.  That was easy and simple.  Connecting over the Internet with "ssh -X" is an option, but I don't see that it has advantages over a wireless keyboard and mouse.
