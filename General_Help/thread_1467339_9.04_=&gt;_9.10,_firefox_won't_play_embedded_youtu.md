### Post by jfmxl on 2010-04-30
I ended up at 

 [https://help.ubuntu.com/community/FirefoxNewVersion](https://help.ubuntu.com/community/FirefoxNewVersion)

Where I was advised to add

 ppa:mozillateam/firefox-stable 

via the system => administration => software sources

I then ran

 apt-get update

and

 apt-get install firefox

The result is a working 64 bit version of firefox 3.6.3

Thanks for the help.
