### Post by SOULRiDER on 2008-01-16
I dont use PDF's much, but you could use Adobe's PDF reader for linux.

To install it go to [http://www.adobe.com/products/acrobat/readstep2_allversions.html](http://www.adobe.com/products/acrobat/readstep2_allversions.html) and download the DEB version, we want that since you use Ubuntu.
Double click the downloaded file to install it.
Youre ready to use Acrobat Reader now :)

If you get any errors about libgtkembedmoz.so do this:
Go to Edit > Preferences > Internet
Where it says select browser (i think its near the bottom right) add these two routes
```
/usr/bin/firefox
      and
/usr/lib/firefox
```

Good luck and ask if you ahve any questions.
