### Post by mattharris4 on 2015-04-22
Is this computer a 2012 or earlier model computer?  If it is newer and you just installed this version the drivers may not be there.  If that is the case please install Ubuntu 14.04 ASAP.  If it is at least three years old and you have been running Ubuntu 12.04 reliably for a while I would say something in the OS install is corrupted.  There are several reasons this could be but one solution is to back up all of your documents (and hope it doesn't freeze while you are doing this) and wipe/reinstall Ubuntu (this time with the current LTS version 14.04).  

You could check the hard drive (do a search here and at AskUbuntu for how to do this) before taking the step to wipe the hard drive and start over, there is a utility you can access via the terminal.  If that fails a Google search is your friend here.  Here is one write-up about checking a hard drive:  [http://askubuntu.com/questions/38566/how-can-i-check-the-health-of-my-hard-drive](http://askubuntu.com/questions/38566/how-can-i-check-the-health-of-my-hard-drive) .  Here is another:  [https://help.ubuntu.com/community/Smartmontools](https://help.ubuntu.com/community/Smartmontools) .  Certainly if the hard drive is going bad you should replace it (assuming the computer is new enough to take a SATA drive, if it is more than 10-12 years old and takes an IDE drive I would just buy another computer as soon as I could afford it if the hard drive is failing, you would be lucky to find a new IDE replacement nowadays) -- desktops take a 3.5 inch drive, laptops take a 2.5 inch drive.  An internal 500GB drive is available for less than $75 from a number of retailers and is easy to do even on most laptops (YouTube videos are appropriate here to see how it is done), it is so easy on a desktop that I could teach my eight year old niece to do it.  After installing the hard drive you just install Ubuntu using a bootable USB stick (it is the same process as doing so on a wiped drive minus the wiping), you have evidently done that at least once already so that shouldn't be a hurdle.
