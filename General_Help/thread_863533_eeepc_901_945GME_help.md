Ok all fixed:
First I had to recompile the kernel as 945GME was not being seen for some reason.
```
sudo aptitude update; sudo aptitude install linux-image
```

After I booted up I could enable compiz but it was indrect rendering. So then I did
```
sude gedit /etc/X11/xorg.conf
```
Added a new section and put this inside:
Section "DRI"
    Group        "video"
    Mode        0666
EndSection
[/code]
Now it's all working fine and I have full rendering.
