I accidentally typed something wrong when I was installing Ubuntu. Now I can't log in for the first time because I'm not sure what I accidentally typed. How can I get around this?

---

### Post by bodhi.zazen on 2008-01-21
boot to recovery mode

enter

```
passwd <user_name>
```

where <user_name> is your log in name
