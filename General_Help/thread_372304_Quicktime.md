### Post by nereid on 2007-02-28
The .lnk file won't work, because it is a Windows link and Linux doesn't know how to handle it. Check out the folder where you installed Quicktime and then just start it with wine (if the executable is named quicktime.exe) *wine quicktime.exe*.

But why do you install Quicktime? Just install the codecs and watch the movies with your favorite Linux player.
