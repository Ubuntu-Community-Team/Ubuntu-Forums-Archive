### Post by lukeiamyourfather on 2012-08-20
Looks like Internet Explorer has a pretty bad track record with Wine.

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=25](http://appdb.winehq.org/objectManager.php?sClass=application&iId=25)

A virtual machine is probably the best option at this point. I like VirtualBox quite a lot for this kind of thing, though you'll need a license for Windows. Or if you install Windows 7 without a key it will do a 120 day trial.

### Post by Mark Phelps on 2012-08-20
The latest version of IE that I've seen installed in Wine is v8 -- and even that took a lot of tweaking.  Most stuff nowadays uses IE 9 -- and I've not seen any ways to install that in Wine.

If you need to use MS products, your best bet is to use them with an MS OS; Wine is only a stopgap for some MS apps.
