### Post by Psykotik on 2006-12-29
Fullsync : download jar installer from [sourceforge]("http://sourceforge.net/project/showfiles.php?group_id=115436&package_id=125127"), and learn how to use at the [official website]("http://fullsync.sourceforge.net/"). Pretty nice software, I think it'll suit perfectly your needs.

Unison can be downloaded via synaptic. It has plenty of option, which are not easy to find. You'll have to read a lot, since the gtk interface has to cover, let's say, 5 % of its functions. It is gorgeous, it is huge, but... it doesn't support ftp protocol.

I'm pretty fond of [Capivara]("http://capivara.sourceforge.net/"), written in java (like fullsync). It has many functions that I badly need (ftp sync), but it is not yet as user friendly as Fullsync is. That's why I'm trying to "orientate" next release :)
