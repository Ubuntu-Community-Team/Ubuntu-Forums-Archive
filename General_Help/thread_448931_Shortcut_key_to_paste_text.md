### Post by gradedcheese on 2007-05-19
Do you have a three-button mouse?  Keep in mind that any text you select is automatically 'copied', when you middle-click it is pasted.  If you have a two-button mouse, try pressing both buttons at the same time.  If you have a mouse with a scroll wheel, the wheel itself may act like a button when pressed.
