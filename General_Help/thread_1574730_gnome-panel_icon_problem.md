### Post by bug67 on 2010-09-14
Although I am not using Ubuntu, I am using an Ubuntu derivitive.  Linux Mint also uses the Gnome panel.  Sometimes my panel icons freak out too.  I fix it by restarting the Gnome panel.  Do this by launching the Process Manager and selecting "Gnome Panel" and then, "End Process."  

This doesn't happen frequently but it does happen frequently enough that I have set up a keyboard shortcut to launch the Process Manager.  I'm sure there's a way to do it from terminal as well...kill all gnome-panel maybe?  I dunno.  I'm kind of a GUI person myself.  Hope this helps.
