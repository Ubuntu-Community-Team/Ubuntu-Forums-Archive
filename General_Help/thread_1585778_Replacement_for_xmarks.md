### Post by lovinglinux on 2010-10-01
There is no web server to login in Firefox Sync. You can check the latest sync in the extension toolbar button. In Firefox 3.6 it should be on the statusbar.

For Google Chrome, you can see the data in Google Docs.

BTW, Xmarks fate is not sealed yet - [http://ubuntuforums.org/showpost.php?p=9909484&postcount=27](http://ubuntuforums.org/showpost.php?p=9909484&postcount=27)
