### Post by tjwoosta on 2008-05-12
? why would you want to install beryl?

compiz fusion is the newer version of the same thing and it comes preinstalled on ubuntu.

to enable compiz go to system-preferences-appearance click the visual effects tab then choose extra

to configure settings for compiz (like the desktop cube for instance) you would need to download the "compiz config settings manager" (either through synaptic package manager or the add/remove.. list)

if you were talking about the beryl-emerald themes, they could be installed in compiz with the "emerald theme manager" its in synaptic
