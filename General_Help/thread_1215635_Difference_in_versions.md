### Post by jerome1232 on 2009-07-17
The only difference is the Desktop Environment (the gui, or look and feel)

Ubuntu uses Gnome as the default desktop environment

Kubuntu uses KDE

Xubuntu uses XFCE

You can actually install alternate desktop environments, ie if you have Ubuntu using Gnome you can install KDE and switch between the two at your login window.
