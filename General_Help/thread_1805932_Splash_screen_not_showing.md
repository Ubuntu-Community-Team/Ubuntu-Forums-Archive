The Live CD boot uses graphics drivers that work in a basic mode and will show the splash screen in almost every circumstance, in a real install hardware graphics drivers are usually installed that are sometimes not compatible with the Plymouth system that shows the splash screen.

You can find many posts on the net that can fix this in a lot of cases.
