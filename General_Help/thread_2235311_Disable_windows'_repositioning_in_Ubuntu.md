Hello everyone!

I would like to disable the windows repositioning. When I open a new window it tries to place itself in an empty spot. To me this is annoying and I would like to have all windows to open in the center of screen.

---

### Post by mooreted on 2014-07-20
Install CompizConfig Settings Manager. Under "Window Management/Place Windows" you can change how windows are placed on the screen.
