### Post by t0p on 2011-08-09
Whoa!  Learn to walk before you run!

I get the impression you don't know how to program?  Well, let's get you started.  **Python** is a good language to start off with.  So learn the ins and outs of programming there.

Now, once you've done that, it's time to do some more learning.  I dunno about Kubuntu, but most(?) Ubuntu as are written in **C** or **C++**.  So that's your next task: learning C or C++ (don't worry, once you've learnt one language it's easier to learn other languages.

There are *lots* of sites on the web that will help you learn computer languages.  A good place to start is [here]("https://ihatehate.wordpress.com/2010/03/29/how-to-search-the-internet-1-the-history-of-search/").

Good luck!

### Post by rathin.saran on 2011-08-09
Start off by learning C and C++
Links  :
    Beej's C Tutorial - [http://beej.us/guide/bgc/](http://beej.us/guide/bgc/)
    C++ Tutorial - [http://www.cplusplus.com/doc/tutorial/](http://www.cplusplus.com/doc/tutorial/)

Books :
    The C Programming Language - Ritchie & Kernighan
    ( One of the best tutorials ever written on C, a must read )
    C Primer Plus - Stephen Prata
    C++ Primer Plus - Stephen Prata

Then learn the basics of the Linux Programming
Books :
    Beginning Linux Programming - Neil Matthew, Richard Stones
    GNU/Linux  application programming - M. Tim Jones

You should be able to create your own applications with the above.

To contribute to GNOME you need to learn GTK
Links :
    [http://zetcode.com/tutorials/gtktutorial/](http://zetcode.com/tutorials/gtktutorial/)
    [www.micahcarrick.com/gtk-glade-tutorial-part-1.html](www.micahcarrick.com/gtk-glade-tutorial-part-1.html)

Books :
    Foundations of GTK+ Development - Andrew Krause
    Gnome/Gtk+ Programming Bible - Arthur Griffith
                     OR
    The Official GNOME 2 Developer's Guide - Matthias Warkus

    

The GTK+ Documentation also comes in handy for programmers
    [http://www.gtk.org/documentation.php](http://www.gtk.org/documentation.php)


To contribute to KDE you need to learn QT
Books :
    C++ GUI programming with Qt 4 - Jasmin Blanchette, Mark Summerfield
    KDE 2/Qt programming bible - Arthur Griffith  ( a bit outdated )
    Advanced Qt Programming - Mark Summerfield

Links :
    [http://techbase.kde.org/Development/Tutorials](http://techbase.kde.org/Development/Tutorials)
    [http://zetcode.com/tutorials/qt4tutorial/](http://zetcode.com/tutorials/qt4tutorial/)

The QT documentation is available here
    [http://doc.qt.nokia.com/](http://doc.qt.nokia.com/)
