### Post by lovinglinux on 2010-03-19
To install flash properly, see the [COLOR="Sienna"]**Flash Optimization**[/COLOR] section of [Firefox optimization and troubleshooting thread](http://ubuntuforums.org/showthread.php?t=1193567).

Adobe Flash plugin is proprietary, so you can't get the source code. There are open source alternatives, but in my opinion they are not real alternatives yet.

You can get the source code for [Gnash]("http://www.gnu.org/software/gnash/") and [SWFdec]("http://swfdec.freedesktop.org/wiki/").
