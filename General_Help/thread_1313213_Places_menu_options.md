### Post by drs305 on 2009-11-03
Nautilus is the Ubuntu default file browser which opens when you click on a folder in Places.

If you don't find a way to place a folder directly on the Places menu (instead of in the bookmarks section), you can add a "drawer" to the panel by right clicking the panel, Add to Panel, Drawer. You can then drag any file or folder into that dropdown menu to display them. 

It may take a bit of work though to distinguish the different folders if they look the same. To assign a different icon to each one right click the folder, select Properties, click on the icon in the upper left, and select a new icon or path/icon.
