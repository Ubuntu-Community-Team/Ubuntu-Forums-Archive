### Post by HiImTye on 2010-11-01
install compiz and emerald
```
sudo apt-get install compiz emerald
```compiz is a window decorator and has a windows like theme. you can also find other themes on other sites (like GNOME-look)

once installed run
```
compiz --replace &
emerald --replace &
exit
```
