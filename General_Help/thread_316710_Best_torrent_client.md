### Post by PurplePenguin on 2006-12-11
Here's my 2 cents.  Everybody always talks about how Azureus, being java based, takes a lot of horsepower to run well.  For somebody who downloads and seeds a large number of torrents at one time, I find that only Azureus does what I need it to do.  I tried Bittornado, but didn't like the fact that it opened up a single window per torrent... once three or four torrents were opened, my CPU was being utilized almost exclusively by bittornado.  So I could realistically have three torrents downloading/seeding at any one time... for me, not hardly enough.  With Azureus, I can have three torrents downloading, five or six seeding - all at once and in the background, and still work on the internet and do whatever else I need to do without any noticeable effect on the performance of my computer. 

And it's an older AMD 1200 with less than a gig of RAM! :D

I do, however, agree with Taurus that you should try more than one program.  Ktorrent is a great program if you're running kde (although it tends to be noticeably less stable than other programs).

### Post by bernardobraga on 2007-11-23
I tried deluge, but I'm used to bittorrent on windows, and the feature where you can select which files you wish to download from a torrent... Isn't that possible in deluge? I tried, but couldn't find!
If it's not, is there a torrent client out there that can do that?

edit:
maybe I was too tired when I looked for the first time ehehe, found it now
here's the guide for those who would like that feature:
Select "Edit" then "Preferences". A window should pop. On the "downloads" tab, you can check the "Enable selecting files before downloading" box. You must also select "Use full alocation"
There.. When you add a new torrent, a list of the files will appear... Right click the list and you'll see some options, such as "select all".. I usually select all, unselect those I wish to download and then select "Don't download" on the selected files....
if you want to select extra files while you're downloading, click the "plugins" icon on the main screen and select the "Torrent Files" plugin.
have fun!
