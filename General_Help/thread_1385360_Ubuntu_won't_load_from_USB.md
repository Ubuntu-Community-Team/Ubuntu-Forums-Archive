### Post by farsight on 2010-01-19
Hey there,

Sorry to hear the USB load won't work.

Have you ensured that the files you downloaded are as they should be, by using the MD5# sum. I believe this link provides the MD5# sums for ubuntu versions: [https://help.ubuntu.com/community/UbuntuHashes](https://help.ubuntu.com/community/UbuntuHashes)

Failing that, if you wanted to give ubuntu a go then you could consider the wubi install. Basically ubuntu is installed as an application inside of windows. Following installation you have the option to boot in to ubuntu when booting your system. Also, if you don't like what you see (which I am sure you will) you can always delete ubuntu like any other windows application.
