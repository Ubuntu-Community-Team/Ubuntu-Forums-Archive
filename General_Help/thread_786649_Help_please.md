### Post by pedro_orange on 2008-05-08
First of all; you should name your thread something that describes what your post is about so people that can help actually do.

Secondly; exe is a Windows file. You MAY run Windows files using WINE within Ubuntu:

```
sudo apt-get install wine
```

But; there may be a native Linux version of the application you require. It may also help if you actually said which exe you want to run.

Check the WineHQ AppDB for your app.
