### Post by howlingmadhowie on 2008-08-10
everything looks fine.

one thing i would probably change in your case would be the partitioning. since new versions of ubuntu come out every six months, many people like to install them rather than sticking with the version they have. to make this easier it would be good to have a partition for data. unfortunately to do this, you'll need to reinstall now and change the partitioning. i'd recommend something like:

30GB ext3 mounted on /
the rest GB ext3 mounted on /home
3 GB swap

you need the swap partition to suspend to disk because ubuntu saves the current ram image to the swap partition when you do this. 

i'd really recommend doing that before we look at other problems. 

a propos webcam. do you have one built in? if so, can you open a terminal, type 
sudo lshw
<enter password>

and copy and paste the result so we can see it?
