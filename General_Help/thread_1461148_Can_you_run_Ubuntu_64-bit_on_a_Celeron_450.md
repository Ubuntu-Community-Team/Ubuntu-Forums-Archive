### Post by bedhead75 on 2010-04-23
It's a single core processor but is 64-bit capable according to Intel specs.  You can buy Dells with 64-bit Windows pre-installed on computers with these chips, but does Ubuntu 64-bit specifically require two or more cores?  I'm asking on behalf of a friend. Thanks.

---

### Post by Revolutionary101 on 2010-04-23
No, it does not require two or more cores. Ubuntu 64-bit will run on any mainstream 64-bit Intel processor. Although, you do not need Ubuntu 64-bit unless you are going to be using more than 4 GB of RAM.
