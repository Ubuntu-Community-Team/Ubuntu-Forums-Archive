### Post by mccorkle on 2008-01-14
Hmm, I bet you can with the old ctrl-V, enter trick -- but I wouldn't recommend it.  The problem with newlines (or any other escaped character) is that some applications are going to have issues with parsing the file name and therefore have a problem reading the files.
