### Post by Shadius on 2012-06-16
You have a few options. I believe there's a way to reset the password, but I don't know much of how to do it. Another option you have is if you don't need any data from the computer, you can simply burn the lastest image of Ubuntu, which is 12.04 LTS, from Ubuntu.com and install it on the computer. It will replace the older version of Ubuntu and all files that are on there, thus, making the computer your own. You can set up your own usernames.

This link will help:
[Reset Password]("http://www.psychocats.net/ubuntu/resetpassword")

[Download Ubuntu 12.04 LTS]("http://www.ubuntu.com/download/desktop")
