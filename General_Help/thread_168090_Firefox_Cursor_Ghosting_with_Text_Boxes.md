### Post by praj.basnet on 2006-04-29
I've found a consistent issue with cursors in text boxes ghosting for Firefox v1.07, 1.5.0.1, and 1.5.0.2.

The problem is that the cursor is "ghosted" after moving out of a textbox or when selecting data form the textbox. This also occurs in other text input files (even the location bar).

You'll find this occurs if you are using the OpenSource "nv" display driver for X. If you change this over to the latest NVidia driver, the issue is resolved. Here's a link to how to do that:

[http://www.ubuntuforums.org/showthread.php?t=75074 ]("http://www.ubuntuforums.org/showthread.php?t=75074")
