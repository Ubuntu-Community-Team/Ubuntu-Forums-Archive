### Post by dagnabit dang doohickey on 2007-07-14
He needs a Dynamic DNS Provider.

I use this one because they work with the DDNS updater in my router: [http://www.no-ip.com/]("http://www.no-ip.com/")

But there are others such as: [http://www.dyndns.com/]("http://www.dyndns.com/services/dns/dyndns/")
DynDNS is more restrictive when it come to routers. They prefer software DDNS updaters.
