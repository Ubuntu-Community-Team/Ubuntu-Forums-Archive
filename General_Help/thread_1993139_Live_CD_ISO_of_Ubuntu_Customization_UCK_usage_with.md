Hi,

I would like to share the information about Ubuntu Customization Kit (UCK).

With UCK, we can customize Live CD of Ubuntu ISO. 

We need Internet while using the UCK.

Sometimes, we may not need the internet for small customization.

In that case, you may required some modified scripts in UCK library.

Please go through this link to find out more about UCK without Internet.
