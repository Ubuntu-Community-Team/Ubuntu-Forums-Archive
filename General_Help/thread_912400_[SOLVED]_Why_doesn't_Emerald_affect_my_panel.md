### Post by Genius314 on 2008-09-06
Emerald themes only affect Window borders. For everything inside the windows (buttons, text, tabs), and also the panels, you want to install a GTK theme.

[Gnome Look]("http://gnome-look.org/") has a lot of GTK themes. In the menu on the left, select "GTK 2.x." That will have a lot of themes for you to choose from.

To apply a theme, go to System>Preferences>Appearance, and (I think) just drag the tar.gz file that you downloaded into the the window. This should install the theme. If it doesn't automatically apply, just click "Customize" and select it from that list.

Hope that helps. :)
