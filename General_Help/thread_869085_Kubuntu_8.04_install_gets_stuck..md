### Post by ajgreeny on 2008-07-24
I'm pretty sure there is an option somewhere before you get to the install part to run the live CD in low graphics mode, or something similar to that.  Try that and you may get somewhere.  Failing that, you may need the Alternate Install CD, downloadable from the same place as you got the live CD.  It installs in text mode, but still gets the job done, and afterwards you can install the nvidia driver needed for your card.
