### Post by sedawk on 2010-06-15
First you can check the Application Database from the Wine project.
at www.winehq.com->AppDB->...
Check if anybody had any success with the game with your
version of wine.
There are often good instructions on how to make the game work.

(I think the last time I checked there was some trouble with
 the font display in AoE).

If you have a valid Win license and CD you can install Virtual Box
to run a virtual PC. Then you can try to install Windows
and then AoE. I think AoE is so old that it might
work with the limited graphics support that virtual PCs
currently have.
