### Post by endersshadow on 2006-05-03
The Intel chip requires the 855resolution package (915resolution in Dapper), and cannot be obtained via the LiveCD AFAIK.

Once you have it installed, simply run (in Warty, Hoary, or Breezy):
```
sudo 855resolution -l
```
Or, in Dapper:
```
sudo 915resolution -l
```

Pick which one works best for you, and remember it.  Then edit the resolution config file:
```
sudo gedit /etc/default/855resolution #again, 915resolution in Dapper
```

Enter in the correct mode, XRES, and YRES (and BIT in Dapper) that you would like.  Then do this:

```
sudo /etc/init.d/855resolution start #again, 915 in Dapper
```

Restart X.org with Ctrl+Alt+Backspace.  Voila.

### Post by ebob on 2006-05-24
I had a problem similar to this.  I needed the resolution to be 1280x800, but the only resolution available in Gnome was 1024x768.  I installed the 915resolution package from Synaptic and did the following:

> $ sudo 915resolution -l

As has been pointed out previously in this thread, this lists the resolutions.  The list that I had follows:

> BIOS: TYPE 1
Mode Table Offset: $C0000 + $269
Mode Table Entries: 36

Mode 30 : 640x480, 8 bits/pixel
Mode 32 : 800x600, 8 bits/pixel
Mode 34 : 1280x800, 8 bits/pixel
Mode 38 : 1280x1024, 32 bits/pixel
Mode 3a : 1600x1200, 8 bits/pixel
Mode 3c : 1920x1440, 8 bits/pixel
Mode 41 : 640x480, 16 bits/pixel
Mode 43 : 800x600, 16 bits/pixel
Mode 45 : 1280x800, 16 bits/pixel
Mode 49 : 1280x1024, 16 bits/pixel
Mode 4b : 1600x1200, 16 bits/pixel
Mode 4d : 1920x1440, 16 bits/pixel
Mode 50 : 640x480, 32 bits/pixel
Mode 52 : 800x600, 32 bits/pixel
Mode 54 : 1024x768, 32 bits/pixel
Mode 58 : 1280x1024, 32 bits/pixel
Mode 5a : 1600x1200, 32 bits/pixel
Mode 5c : 1920x1440, 32 bits/pixel

The next step was to edit the bootmisc.sh file to change an unused mode.  I chose to use Mode 54.

> $ sudo gedit /etc/init.d/bootmisc.sh

Around line 17, you should find a line that reads:

[ -f /etc/default/rcS ] && . /etc/default/rcS

After this line, insert the following:

> 915resolution 54 1280 800

Reboot and compute in your chosen resolution.
