### Post by ad_267 on 2008-07-25
It doesn't show up like that. You go to Tools - Manage snippets. For each language you use you can add snippets. You add a "Tab trigger" and then if you type in that trigger and hit the tab key while editing a document in that language it will insert the snippet.
