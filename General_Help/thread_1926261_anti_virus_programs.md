### Post by Paqman on 2012-02-16
> **mastablasta said:**
> wait what? i suppose now you will say there is no malware on linux kernel?

That's not what I said at all. Please don't overreact. However, as Aysiu correctly pointed out, adding an antivirus suite to an Ubuntu Linux install won't improve the security of that system to any great degree. As you say, you can if you want apply one to screen for Windows systems, but IMO that's really only necessary if you're operating a machine that has the primary role of passing data to Windows machines (ie: some kind of server). It's pretty pointless for a desktop, and a some of the more popular Linux AV suites are of dubious effectiveness.

It's important to make this point for newbies. An antivirus suite is a good idea on a Windows machine, but more often than not superfluous on a Linux one. Aysiu's link to Bodhi Zazen's seminal thread spells this in more detail.
