### Post by adam_kimber on 2008-08-14
If you go to System --> Administration --> Hardware Drivers (I am assuming you are using Hardy) and check the Nvidia driver box that will install Nvidia's proprietary Linux driver. That **should** set up high colour depth etc without you doing anything. 

The Nvidia control panel like the windows one is not avaitable for Linux per se. There is something similar available, once you install the driver, open a terminal Accessories --> Terminal and type 
```
sudo apt-get install nvidia-settings
``` 
then 
```
nvidia-settings
``` and you will have something similar to the Nvidia control panel.

### Post by todak on 2008-08-15
Your display color-bit depth is set too low and you are seeing "stepped" gradients between different shades of colors.  If your color depth is 16 bits or less, you will see the gradations (assuming the image you are viewing has a higher bit depth than your display, which in this case is true). Set your color depth to 24 bits and you will see a smooth color gradation. As far as the "jittery" desktop goes, when you scroll your mouse wheel on an empty desktop, you are changing to a different desktop and the arrows you see are showing which way you scrolled to which desktop. It will  do the same thing by clicking on the unused desktop symbol in the lower panel.
