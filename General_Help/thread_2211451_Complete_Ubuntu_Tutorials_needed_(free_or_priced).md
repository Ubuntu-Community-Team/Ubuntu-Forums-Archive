### Post by BlinkinCat on 2014-03-16
To be more specific, pick one problem, browse through the list of [PopularPages]("https://help.ubuntu.com/community/PopularPages") to get a general feel for the problem. If you can't find the information, then type in a few terms in the search box and see if some hints come up. If you are still having problems, then ask a single, specific question on these forums and see if you can get resolution to your issue.
