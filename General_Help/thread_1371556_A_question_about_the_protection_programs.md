### Post by Some Penguin on 2010-01-03
It's not much of a problem.  Mostly, 

1) use common sense regarding what you run, and

2) keep up-to-date re: security updates, so if something IS broken that allows privilege escalation or the like, that you get the fixes ASAP.


The latter -- privilege escalation -- is usually the bigger issue than 'viruses' per se; bugs in the kernel or in programs normally run as root allowing unauthorized users to achieve root privileges.
