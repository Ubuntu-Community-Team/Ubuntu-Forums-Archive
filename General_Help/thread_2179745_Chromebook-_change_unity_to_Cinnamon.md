### Post by mcduck on 2013-10-09
There is no need to install more than one Ubuntu on the machine, you can just add any desktop environments you want into the same install, and then select which on you want from the login screen. Just use the Software Cetner, or Synaptic, or whatever package amangemetn tool you prefer to install them.

What comes to installing Cinanmon, this was pretty much the first result from a google searrch, should do the trick: [http://www.itworld.com/software/307138/install-cinnamon-desktop-ubuntu-1210]("http://www.itworld.com/software/307138/install-cinnamon-desktop-ubuntu-1210")

Also I'd be surprised if Ubuntu was the *only* Linux distributon installable on C7, especially since it's using a normal Intlel CPU so you don't even need to worry about requiring a distribution with ARM support. Of course that doesn't mean that there would be many other *easy* options, or ones with enarly as good support...
