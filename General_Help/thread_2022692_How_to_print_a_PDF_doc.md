### Post by Jerry N on 2012-07-11
If you have the document opened in a PDF reader you should be able to print it just like you would any other document, unless it is protected from printing.  In that case, Okular has an option to turn off DRM rights and print the document.  Depending on where you are in the world, this could have legal implications.
