### Post by mb_webguy on 2009-02-24
The OP is suggesting that Gnome-Do be included in the distribution as a default, not an option that can be found in the repos.

Personally, while I can see how it might be useful for some people, I think Gnome-Do is something that should be left as an optional add-on.  Such things can be installed with only a couple of mouse-clicks anyway, and I think a fairly minimal-but-extensible approach is better than bloat.
