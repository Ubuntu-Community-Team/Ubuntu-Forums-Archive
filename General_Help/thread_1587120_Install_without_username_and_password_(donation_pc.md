### Post by Mykle87 on 2010-10-03
I am holding a computer building workshop with my school's technology club. We are going to build a computer from scratch and install Ubuntu on it. Since we are going to donate the computer to an underprivileged elementary school student who is yet to be determined, I don't want to setup a generic username and password. Is there anyway fully install Ubuntu and then on first boot have the student type in their desired username and password? I want to make this as easy as possible for the student. I am really excited for this event and glad that I can give back to the community. Thanks in advance for the help.
