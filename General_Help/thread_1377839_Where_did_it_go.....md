If you really want to use that application, it is not that complicated to run it via the terminal, I would first go into the "Keyboard Shortcuts" selection under "System--->Preferences" and set up a keyboard shortcut to launch the terminal (I personally use Alt+T and that opens the terminal, at which point you would just type Gphoto2 (or whatever it was named) and it should launch with no problems. Plus, getting familiar with the terminal is a good way to really start getting into the guts of Linux (any distro) and start to begin learning some more advanced stuff slowly but surely, and you will be flying around your machine in no time. In other words, please don't let the need to use the terminal as a down-side, but rather a reason to at least start using it and getting a little more comfortable with it as it can really unlock some more powerful tracking features and get you to items more quickly in the future once you do become comfortable with it, and if you need any assistance please feel free to post back here or PM me and I would be glad to help.
