### Post by viciouslime on 2007-05-05
In short: No.

Ubuntu can theoretically run on any architecture because it is open-source and uses a linux kernel which is itself highly portable... however some one actually has to do the porting work and noone has for the dell axim, nor will they. Ubuntu isn't suited to such a device. Try handhelds.org for linux distros that are.
