### Post by howefield on 2015-07-03
Sure, a 2 key combination doesn't seem so hard for someone used to learning lines, however given that it is you could go into VLC > Tools > Preferences > Show Settings - All > Hotkey Settings and map whatever combination suits the ability.

For example I don't think the letter q on its own is assigned to any other hot key so you could use that instead of the default Ctrl + q. I doubt you could use a key or combination of keys already assigned to something else.. so that rules out something like escape,..escape.
