### Post by Tomosaur on 2007-03-16
Short answer:

Download the LiveCD and check it out. You can get the LiveCD from [here](http://www.ubuntu.com/getubuntu).


Long answer:

The LiveCD will allow you to boot Ubuntu from the CD drive. Simply burn the download file as an image, put it in your drive, and reboot the machine. If Windows loads, then reboot again, and hit whichever key it tells you to enter setup. You'll need to make the CD drive the first in the boot order.

Once you're in the LiveCD, you'll see an option saying 'Start or Install Ubuntu'. Select that, and the GUI should load up (it will take a while) and you can use Ubuntu as a complete operating system, without the need to install it. It WILL run slowly though, as it's all running from RAM. The purpose of the LiveCD is to get a feel for Ubuntu, and to check whether your hardware works.

For most people - hardware is not an issue at all. Linux supports more hardware than Windows natively (ie, you don't need to install drivers, it just works). Sometimes you MAY need to find and install drivers - particularly for wireless cards. Graphics cards should work fine - but if you want 3D acceleration, you may need the official drivers from your manufacturer (nVidia are very nice towards Linux users, they've been kind enough to release very good drivers for us, so if you have an nVidia card, you should be fine).

Ubuntu comes with many different kinds of software already installed. It has:
A full Office suite (OpenOffice).

Web browsers.

Instant messengers.

Media players.

A bunch of fun games (although they're pretty basic, nothing more than time-wasters really).

An easy way to add/remove new software. All you need to do to get great new software is to open up the 'Add/Remove' application, select which programs you want from a list of thousands (divided into categories, and with search capability to find something you really want), and click apply. You don't need to go trawling the web for potentially dangerous software, everything in the repositories (ie, the database of available software) is checked and filtered to make sure it's all safe and will work with your system. It's all free, too (always a bonus!). All software is checked for upgrades automatically, and you'll get a notification if and when new updates are available (for everything, so you don't have to check for each individual program).

Multiple desktops to keep your workspace clean.

The usual administration / preferences tools.



You will not need to worry about viruses or net security. You can if you WANT to worry, but viruses are virtually non-existant on Linux (Linux is inherently more secure than Windows anyway, so even if you did get a virus, it wouldn't really be able to do much). Linux has an in-built firewall which comes pre-configured to lock out any potential attackers. There are websites which you can use to check for vulnerabilities on your system - Ubuntu in particular always comes out on top - it's essentially invisible to the outside world (unless, of course, you advertise your presence personally!).

If you have enough RAM, and a decent enough graphics card, you can get your hands on Beryl, which basically gives you lots of eye-candy, much of it is actually very useful rather than frivolous. Here's the [Beryl Website](http://www.beryl-project.org/), and here's some [videos of Beryl in action](http://www.youtube.com/results?search_query=beryl&search=Search) so you can see what all the fuss is. Don't be fooled by some of the Vista users you'll find in the comments - Linux has had Beryl for months before Vista's Aero interface was even announced. It is also much less resource intensive - and most people consider Beryl to be far superior.

There's much, much more, obviously, but you should really just give it a go. If you're heavily dependant on Windows, then it may not be for you, but you won't know until you try. If you experience any problems later on, just come back here and we'll see what we can do to help you out :)

### Post by CocoAUS on 2007-03-16
My suggestion would actually be to download the latest Linux Mint.  Linux Mint is basically Ubuntu rebranded, with a (slightly) better theme, multimedia codecs and browser plugins installed by default, etc.  If you're new to the Linux world, or the Debian world (Ubuntu is based on Debian), Linux Mint will probably be more comfortable for you.  You'll be able to watch all your movies, listen to all your music, and browse the web with it without having to install anything extra.

Why should you use Ubuntu?

Well, maybe you shouldn't.  For me, Ubuntu is a much better experience than Windows or OS X.  I can customize everything the way I want, all my software is up to date, it's secure and virtually virus-free, it lets me do more, etc.  The software is (almost) entirely free (with the exception of a few programs like Crossover), and is generally more capable than Windows equivalents.  It's faster, doesn't have general slow-down problems over time, it's easier to maintain....  Really, it's just a better experience in general.  If you want games, no matter what people tell you, Linux probably isn't the way to go.  Unless you plan on only playing, for instance, World of Warcraft or Unreal.

Hardware is generally supported out of the box, and stuff that isn't supported out of the box generally has a tutorial you can follow online to install it.  My tvcard took literally days of troubleshooting to get up and running on Windows, but Ubuntu runs it out of the box perfectly.  Wireless is the only real issue I have.

So, if you want a fast, solid operating system that you can make look and behave the way you want, doesn't bother you with DRM or licenses or other pain-in-the-____ items, and generally gives you more power, then Ubuntu might very well be just plain awesome for you.  As long as you're willing to troubleshoot your wireless card and maybe not be able to do Yahoo! video chat.
