### Post by Cheesemill on 2012-05-03
When a release reaches EOL the repositories are moved off the main servers and no more updates are released, potentially leaving the system open to newly discovered vulnerabilities which will never get patched.

If for any reason you wish to install or continue using an unsupported version then the repositories are moved to [http://old-releases.ubuntu.com/](http://old-releases.ubuntu.com/) so by changing your soures.list you will still be able to install software, but the repositories are just a snapshot of when the EOL date is reached, no further changes will be made to them.
