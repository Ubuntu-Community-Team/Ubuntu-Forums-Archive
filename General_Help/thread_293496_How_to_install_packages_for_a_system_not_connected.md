### Post by fragos on 2006-11-05
Synaptic has an option to add CD as a repository.  Everything in the repositories is downloadable as a file.  I see no reason why you can't build a CD with the bits and pieces your friends would want to install.  The Live install CD gives you most of what the average user would wan't.  Non-GPL licensed bits like Flash and Acroread aren't included on the live install CD.
