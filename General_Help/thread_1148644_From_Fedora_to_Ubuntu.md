The biggest difference is root. In Fedora you have a root user which can perform system tasks. In Ubuntu you (the user) can get root privileges for the time being to perform system tasks.
When you install Ubuntu it will ask for a user account, type a name and then the password belonging to this username. This user will automatically be granted root privileges when needed to. When for example you want to install software the system will ask you for a password. This is your password. Root does not have one. It's the password you typed during installation.
Other users which are added later are not added to the sudoers file automatically and so they can not perform system tasks. You (as semi root) can add them if you want to.
