### Post by slibuntu on 2007-12-30
Since you're using Ubuntu,try and get .deb versions of files, as these will pretty much install themselves for you. .rpm is used by other distro's of linux.

As for Firewall software and Anti-virus, Ubuntu has a built in Firewall, and doesn't need anti-virus software at all, so don't worry about it!
