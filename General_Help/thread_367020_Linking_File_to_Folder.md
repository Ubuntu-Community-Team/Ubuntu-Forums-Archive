### Post by llamakc on 2007-02-21
Let's say you have Nautilus open and you're viewing your Desktop. You can press the left mouse button down and hold it as you drag over the two items, like your file and directory. Now that they are selected you should be able to right-click and do something with them. Sorry if I can not help more: I use the command line exclusively for these sort of operations. I think you're correct about its similarity to using ctrl to select multiple items.

To your original question: a downloaded file and folder(directory) with other files inside of it can't be linked. They're separate inodes.

### Post by cmost on 2007-02-21
If you're saving web pages simply to hang onto the information displayed on them, there's certainly an easier way than saving the web page as you've described.  It's much more efficient to file-->print, then tick the box for 'print to a file'.  This saves the page as a single file (*.ps).  Later, you can always convert it to a PDF for even more ubiquity.
