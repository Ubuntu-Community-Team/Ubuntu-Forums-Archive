Generally VBox is pretty nice, just make sure you use the proper version that supports usb and not the one that's in the ubuntu download centre or you'll get annoyed.

It'll take some time getting use to but it's pretty use friendly.  Your gonna need to set some nice amount of ram and graphics card memory towards your vbox though so I hope you have a good spec computer if you want to do stuff like mmo's.  

Ubuntu can run on 512-768 ram without problems so you can always dedicate the rest of the memory towards your vbox and play your games on it.

Make sure you set a dedicated hard drive and not one that expands, it makes the performance alot better on the computer.
