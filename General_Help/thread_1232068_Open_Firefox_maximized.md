### Post by scouser73 on 2009-08-05
Hi Snuzzer, to maximize firefox, open it up, then place the cursor on the corners and stretch it outwards, once you've done that close Firefox then restart Firefox and it should open fully from now on.

### Post by fennec_fox on 2009-08-05
Unfortunately I am at work and on Windows at the moment so I can't test anything but, normally if you open the window, maximize it then close it, it stays. 

Off the top of my head I can think of a couple things that might be helpful. 
**Right now I can't test these so I would look around for other options before trying them. These methods may be out of date.**

If you're running compiz, I *think* you can set custom options for each application somewhere in the compiz manager. I think you can specify firefox and what you want the window to do.

Otherwise, you can edit the localstore.rdf file for firefox (Mozilla **does not** recommend this!). There is a line for sizemode which you can set to "maximized".
