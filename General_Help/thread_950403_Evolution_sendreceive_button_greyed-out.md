### Post by niteshifter on 2008-10-17
Hi,

Evolution set to be off-line? This causes the send/rcv icon to gray out.

Look in the bottom left hand corner of evolution's main window. If you see an icon that looks like two plugs with a red X between them, it is set off-line. Click that icon to toggle back to on-line.

No icon showing / can't find it? Then check the on- off- line state by clicking File on evolution's menu. Near the bottom of the drop down if it reads "Work Online" then evolution is in off-line mode. Click that item to toggle back to on-line.
