### Post by squenson on 2009-09-06
Prior to launching Ubuntu, in VirtualBox application (under WinXP), you must first define the shared folder(s) you want to see from your virtual machine. This is the last option on the list. Select any folder of your XP machine and it will then be visible by Ubuntu.
