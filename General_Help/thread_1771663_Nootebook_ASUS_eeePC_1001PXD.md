### Post by idoitprone on 2011-05-30
get an ubuntu iso,since you will only have 1 or 2 gb of ram, i recommend 32 bit edition

I cannot say which edition you should use, since you have not show the intergrated graphics. The new Unity have heavier graphic requirements than gnome classic or gnome 2.

OOo ddr3 ram, that is useful.

download unetbootin
[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

get a usb with 2 gb or more with a fat filesystem and make a live usb

if you want to dual boot 
[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)

make room for ubuntu; Resize the partition within windows 

 The current default filesystem for linux is ext4.

run the installer.
