### Post by epikvision on 2012-07-18
Super = the windows logo button or the mac's cmd button on the keyboard.

If you hold the super key in Ubuntu, you can see a table of useful shortcuts. That'll make you one step closer to power user.

Try the command **"ctrl + super + d"** It fades all your current windows and reveals the desktop.  Try it again to return those windows.
