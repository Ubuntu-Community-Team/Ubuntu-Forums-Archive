### Post by linuxcanuck on 2010-10-02
Tweetdeck does support multiple Twitter accounts and will install in both 32-bit and 64-bit Ubuntu. You must first install Adobe Air.

I also like the KDE app, Choqok. I have three Twitter accounts  and two Identi.ca ones and have no problem with either.

You can also try Pino which works for me in Fedora, but not as well in Kubuntu for some reason. I need to shut it down when I need to post to a different account or it posts to the wrong account. Not fun!
