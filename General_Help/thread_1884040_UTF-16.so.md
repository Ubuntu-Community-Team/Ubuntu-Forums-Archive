When you get a program, don't you read the Knowledge Base? You should, it tells you quite clearly what to do:
> 
1) Open a Terminal window by clicking Applications -> Accessories -> Terminal and enter the following commands:

sudo mkdir -p /usr/lib/gconv
sudo ln -s `sudo find / | grep UTF-16.so` /usr/lib/gconv/UTF-16.so


Source: [http://kb.eset.com/esetkb/index?page=content&id=SOLN2733](http://kb.eset.com/esetkb/index?page=content&id=SOLN2733)
