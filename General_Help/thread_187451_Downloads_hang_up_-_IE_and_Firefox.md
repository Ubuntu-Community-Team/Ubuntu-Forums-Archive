### Post by LAurel on 2006-06-03
You should try using a download manager which enables you to resume downloads. You said you're using IE, as well as Firefox - so you probably have Windows installed. One of the best download managers for Windows is [Free Download Manager](http://www.freedownloadmanager.org/). If you're running Breezy, enable the Universe repository and install Downloader for X (I love that dm :D ).
