### Post by aeiah on 2010-03-17
you'll be lucky to have all those apps or suitable replacements available in linux.. you'll also probably find that 256mb ram isnt really enough. you should probably try a lighter distro: xubuntu uses xfce instead of gnome, so is a bit lighter. lighter still, but still ubuntu based is crunchbang linux, which uses openbox. there's also lxde, which is a similar environment to the one crunchbang provides, that you could install after installing the normal ubuntu to replace gnome.

but anyway..

matlab: matlab do a linux version, so just install that

visual studio: yea, you're probably right with netbeans, although there are loads of development environments to choose from

VHDL stuff: there seem to be quite a few options available. just google. several may be in the ubuntu repositories (making installation a breeze)
