### Post by mjuhasz on 2011-11-03
The biggest issue is that you won't get security updates after the support period. Even if you update some components (e.g. browser) from a personal package archive (PPA) they are usually not built for non-supported releases so you'll have to update them manually. Your system will eventually be full of unpatched security holes.

In my opinion you'd be better off setting up the latest LTS release instead of spending the same amount of time on patching your non-supported Ubuntu version. 10.10 support will end right when 12.04 will be out which is an LTS version supported on desktop for 5 years! You set up 12.04 the way you like it and until 2017 you don't have to worry.
