### Post by ryanhaigh on 2008-04-27
1. BACKUP and edit /boot/grub/menu.lst then you can use this: [http://www.howtogeek.com/howto/ubuntu/set-windows-as-default-os-when-dual-booting-ubuntu/](http://www.howtogeek.com/howto/ubuntu/set-windows-as-default-os-when-dual-booting-ubuntu/)

2. If you have wobbly windows compiz is running already to enable extra features you will need to install compizconfig-settings-manager which will give you a new entry in system>prefs where you can configure these features.

3.There are docks available but not used by default. Try here: [www.ubuntugeek.com/howto-install-avant-window-navigator-awn-in-ubuntu-710-gutsy-gibbon.html](www.ubuntugeek.com/howto-install-avant-window-navigator-awn-in-ubuntu-710-gutsy-gibbon.html)

4. Sounds pretty cool but there are no viewers that I know of with these features, you could try some of the viewers availalbe in synaptic (system>admin>synaptic)

5. Google search or even a search of these forums could have resolved most of these issues for you as they are very common. In future try searching before creating a new post, not only will you solve the problem faster but if something remains unresolved (point 4. for example) you can create a post which deals with that specifically and get better and more helpful responses.
