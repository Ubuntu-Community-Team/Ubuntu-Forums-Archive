### Post by 3Miro on 2010-08-30
Whne Grub comes up (the Linux loader), then select Vista and as soon as you select it, you will go into the Windows loader. That is the instance when you have to hit F8 for safe mode. Basically, select Vista, hit enter and immediately pump F8. This should work.
