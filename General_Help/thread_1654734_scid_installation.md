### Post by howefield on 2011-01-02
System > Preferences > Main Menu

Highlight Games on the left, (or whatever category you wish to place the menu item in) and click the New Item button.

The "Name" and "Comment" fields can be whatever you want, but the "Command" field should be say scid (as per attached sceenshot) 

Typing scid into the command field should change the "springboard icon" to the correct rook icon. Ok your way back out..
