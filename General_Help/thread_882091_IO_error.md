### Post by Taxman415a on 2008-08-06
It's just an Input/Output error, but without knowing more details about the specific error message and what surrounded it it's hard to tell. It's possible the ISO image you downloaded got corrupted during the download. Try re-downloading it if you can, and if you can or can't do that try getting a md5sum program to check the md5 hash of the iso you downloaded to compare that to what the md5 should be. You'll find what the md5 should be where you downloaded the iso, especially if you downloaded from Ubuntu.com as recommended. If they don't match, that most likely confirms a download problem.

Some other things to try are different medium. I always get less errors using discs I've burned to dvd+rw than cd-rw. Both verify fine on burning, but the dvd+rw's never give me errors on using the verify disc option from the Ubuntu boot menu. And I always use rw medium because then I don't have to waste discs when I want to update the install discs for newer versions down the road. rw discs are slightly more expensive, but cheaper in the long run.

Another option is to burn the disc at less than the maximum speed, try half speed for example.

Finally, if you want to try to copy down the specific error message (as many other details as you can as well) that may help too.
