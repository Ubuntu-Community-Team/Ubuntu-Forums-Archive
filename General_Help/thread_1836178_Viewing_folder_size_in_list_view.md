### Post by rmcellig on 2011-08-30
One of the things I like about Mac OS X is that I can see the size of folders when I view a finder window in list view. I find that as of late I have been using Linux more and more and was wondering if there is any way I can view folder size in list view in Linux.

I am currently using the Ping-EEE distro on a laptop.

---

### Post by peter d on 2011-08-31
In Nautilus go to Edit>Preferences then choose the List Columns tab. The list tells you how many files are in a folder. If you expand the folder by clicking the [+] it tells you the size of the files.
