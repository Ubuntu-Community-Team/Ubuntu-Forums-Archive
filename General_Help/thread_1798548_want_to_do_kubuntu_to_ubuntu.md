### Post by TBABill on 2011-07-06
If you used the metapackage (kubuntu-desktop?) to install from Ubuntu, you can either remove it from a terminal or Synaptic. However, some apps that get pulled into the system may not get removed when you remove that package. If you want a clean install of just the Ubuntu package without any KDE left behind, a new install is probably easiest.
