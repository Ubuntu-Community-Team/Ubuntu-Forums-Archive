### Post by yanvolking on 2009-11-12
Hello Community,

I have 3 computers that I use in different situations and on different occasions.  However, I like their settings and software to be as similar to one another as possible.  How can I quickly synchronize the firefox settings between them.  More concretely I would like to be able to "copy/paste" the firefox bookmarks settings from one computer to another, rather than having to manually enter the pertinent website and then bookmark it on each computer.  Is there not a file where these bookmark settings are stored and copying this file from computer A to the same location on computer B will immediately give computer B the same bookmark settings as computer A?

Thanks

Yan

### Post by Tiganjero on 2009-11-12
They are stored in ~/.mozilla/firefox/(profile)/bookmarks.html. But you can just Bookmarks>Organize Bookmarks>Import and Backup>Backup (Or Import on the other computers) Or you can try using [xmarks]("http://www.xmarks.com/").
