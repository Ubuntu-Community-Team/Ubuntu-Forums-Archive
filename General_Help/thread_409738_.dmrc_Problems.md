I Accidentally Deleted the .dmrc file in my home directory and every time i log in it keeps telling me .dmrc has the wrong permissions. How do i Regenerate the .dmrc file in my home directory? This is annoying clicking a button every time i log in. Any idea on how to fix this?

---

### Post by taurus on 2007-04-14
Applications -> Accessories -> Terminal
```
sudo chmod 644 /home/**username**/.dmrc
sudo chown **username**:**username** /home/**username**/.dmrc
```
Replace **username** with your login name.
