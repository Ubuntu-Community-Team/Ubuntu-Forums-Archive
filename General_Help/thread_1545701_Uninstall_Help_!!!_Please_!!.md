### Post by silkworm2.5 on 2010-08-04
In linux can you access any of your documents from windows? because if not,then it is gone. You will have to format the hard drive because windows cannot detect linux formatted drives. If you still have the linux CD you used,boot that then try the partitioning software on it. GParted.
System > Administration > GParted
use that to format the drive to NTFS and then try windows again. Back up any of your Data first, as this will DELETE EVERYTHING ON THE DRIVE. Probably irrecoverably.
