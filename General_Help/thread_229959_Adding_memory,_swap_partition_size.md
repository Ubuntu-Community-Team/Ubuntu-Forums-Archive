### Post by h00s on 2006-08-05
When installing Ubuntu, swap partition is set to memory-size x 2.
Example:
512mb memory, 1gb swap partition size.

What happens when I want to add more memory (512mb)?
Then memory size would be equivalent to swap partition size!

Should this be concerning me? :)

---

### Post by Rui Pais on 2006-08-05
No don't worry... 
swap could have any size you want and is independent of how much RAM you have. 

In fact that rule to set swap size is just a general tip to have a starting point when make decisions on partitions schemes. 
It's a little obsolete now... it cames from the days when RAM was expensive and swap was really needed. 
Today with 0.5G of RAM your swap is hardly (luckly) touched.
 
You can even reduce that 1G for much less if you find yourself short on disc space.

Have fun.
