### Post by StefanoCole on 2006-03-28
About the modem you should go to: [http://linmodems.org/]("http://linmodems.org/")

and download the scanModem utility. You should run it and read the file ModemData.txt which is generated. Inside such file you should find info about your chipset modem and possibly about driver availability.
