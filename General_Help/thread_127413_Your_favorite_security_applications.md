### Post by klepto on 2006-02-08
Ubuntu as it is is very secure but as some of us are very paranoid 
we need just a few tools to secure what is left to secure. I don't
have any sshd/telnetd open on my laptop machine but I still see people
trying to run code on the bare services I have open. Being behind NAT
helps too.

Here's what I use:

1.) Firehol(one of my favorites, it turns simple language into a powerful iptables firewall along with some great protection from scanning and such)
2.) Snort(a must have!)
3.) Psad
4.) Bastille(harden your your machine and clean up some services that only
need to be run by root users)
5.) Rkhunter(root kit hunter, which looks at everything on your machine to make
sure that someone hasn't run a rootkit on your or something malicious.)

What do you use?
