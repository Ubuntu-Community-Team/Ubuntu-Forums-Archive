### Post by TonyPursell on 2011-10-08
I have been using an Asus EEPC 1015PEM, running Natty, for some months. Everything was OK until a few days ago. Now the keyboard is not enabled at the login screen. I can use the touchpad and its mouse keys to select myself from the list of users, but I cannot type my password. Only the Esc key seems to be working. Pressing other keys gives nothing.

The same situation is found with a USB keyboard plugged in or with the on-screen keyboard enabled.

The physical keyboard is OK. It has been tested at the GRUB boot screen, in Windows 7, using a live 11.04 on a USB stick (which has a login screen) and in Recovery Mode.

---

### Post by Krytarik on 2011-10-08
Make sure the option "Slow Keys" is not enabled in the Accessibility options - usually an icon with a man in a ring on the right side of the bottom panel at the login screen.

Regards.
