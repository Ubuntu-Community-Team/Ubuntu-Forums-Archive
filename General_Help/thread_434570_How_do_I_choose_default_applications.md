### Post by Kalixa on 2007-05-06
Hello. I was wondering how do you choose which applications should open certain files by default. For example. Instead of having totem to open .xvid files I want VLC to do so and instead of having gedit to open text files I want gvim to do so and etc.

---

### Post by zek725 on 2007-05-06
right click on the file > open with another application > choose the application you want to open with > check **use default for this kind of file**

### Post by aardwolfsgp on 2007-05-06
Right click, select "Properties", select tab "Open With", and select your default program for your file. :)
