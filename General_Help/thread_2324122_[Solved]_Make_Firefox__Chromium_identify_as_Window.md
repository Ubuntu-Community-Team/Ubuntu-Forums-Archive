### Post by mcduck on 2016-05-11
Install some user agent changer extension in Firefox, there are at least few available. Like this: [https://addons.mozilla.org/en-gb/firefox/addon/user-agent-overrider/]("https://addons.mozilla.org/en-gb/firefox/addon/user-agent-overrider/").

...however I wouldn't change the user agent unless trying to access some web page that (for some stupid reason) refuses to let you in if it recognizes you using Firefox or Linux. And even then the best option would be to contact the webmaster ask them to fix their web site. ;)
