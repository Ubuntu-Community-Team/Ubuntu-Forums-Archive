The easier path is just to download Kubuntu (640 MB).

I don't think that's the main reason. The reasoning behind Kubuntu (and Xubuntu) is tighter integration with a specific non-GNOME desktop. You may have noticed that Ubuntu is closely integrated with GNOME; it's hard to tell where GNOME ends and Ubuntu starts (best example: Synaptic and the Ubuntu Update System).
This is not true however when using KDE or XFCE with Ubuntu, so KDE/XFCE ethusiasts decided to build Ubuntu derivates which integrate KDE/XFCE on the same level as Ubuntu does with GNOME. This is totally reasonable IMHO.
