When I view the PDF with the Ubuntu Document Viewer, the bullet points appear as 'infinity' symbols (i.e. like a number 8 rotated 90 degrees). If I view the same PDF file on Adobe Reader on a Windows system, the bullet points display correctly. Can anyone suggest a reason/cure for this?

### Post by HermanAB on 2011-10-28
Howdy,

PDF files are supposed to contain all the fonts required to render the data.  Check the file with a different PDF viewer on Linux (Xpdf, Xournal or Okular) and see whether it is a font problem or a viewer problem.
