### Post by DeMus on 2009-10-24
I did find a way to uninstall it completely:
(replace items in [COLOR="Red"]red[/COLOR] with the actual values)

In terminal go to the folder where you downloaded the googleearth.bin file
```
cd /[COLOR="Red"]path[/COLOR]/[COLOR="Red"]to[/COLOR]/[COLOR="Red"]bin-file[/COLOR]
```
Then type:
```
sudo ./GoogleEarthLinux.bin --confirm
```
This will start the installation process but will ask you to confirm 2 steps:
[LIST=1]
[*]Is it okay to extract the file to a location in the /tmp folder?
[*]Is it okay to start installation?
[/LIST]
The first confirmation should be given. The bin-file will then be extracted to a folder called /tmp/[COLOR="Red"]something[/COLOR]
[INDENT]Now start up Nautilus and open the /tmp folder. You will see the folder [COLOR="Red"]something[/COLOR] which just has been created. Copy that folder to your home folder. You can do this from within Nautilus.
[/INDENT]
Cancel the second confirmation in terminal, otherwise the program Google Earth will be installed again. At this point the installation program deletes the folder [COLOR="Red"]something[/COLOR] in /tmp, but since a copy has been made this is okay.
In terminal type:
```
cd /home/[COLOR="Red"]username[/COLOR]/[COLOR="Red"]something[/COLOR]/setup.data/bin/Linux/
```
In this folder you find 3 folders, each one representing a different  version of Linux: ```
amd64, x86, x86_64
```
```
cd [COLOR="Red"]right version[/COLOR]
```
In here you type:
```
sudo ./uninstall google-earth
```
and the program will be uninstalled.
When things are done you can erase the folder [COLOR="Red"]something[/COLOR] in your home folder.

### Post by scouser73 on 2009-10-28
There is a better way of installing Google Earth, firstly install Ubuntu Tweak: [[COLOR="Red"]**Ubuntu Tweak - GetDeb**[/COLOR]]("http://www.getdeb.net/search.php?keywords=ubuntu+tweak")

Install Ubuntu Tweak, you will now find Ubuntu Tweak installed under **System Tools**.  

Click on the Ubuntu Tweak icon, click **Applications** on the Ubuntu Tweak menu, click **Third Party Sources**, click **Unlock** & enter your password & tick the box next to **Google Repository**.  

Now click **Refresh** and you will be given a choice of software to install from Google, one being Google Earth. Select Google Earth then click **Quit**

You will then find Google Earth installed under the Internet.
