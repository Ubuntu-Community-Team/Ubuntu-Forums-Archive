### Post by KinaU on 2009-08-09
I'm not sure why it's not working for you. Go back into Add/Remove in Applications tab and search "Wine". Try to re-install Wine. When you download the game, chances are there's a "Setup" icon which got downloaded with the game files or some icon which you would regulary click in Windows to install the game onto your computer. You right click that icon and find "Open With", put your mouse on that and click on "Wine Windows Program Loader". 

You should get the same install screen as you get in Windows, where you have to click "Next", "Agree", etc.

Works for me, just try it again, it's easy. Hope this helped. :D
