It looks like that you have created your installation without "persistence"

I suggest that you create another USB installation using the Start Up Disk creator application on the Ubuntu live CD.

Within this utility, there is a check box to create an area on the installation to save data etc.

### Post by tea for one on 2012-05-31
The Startup Disc creator within Ubuntu allows you to create a USB device with or without persistence.

Other utilities like Unetbootin only allow a "live" image without being able to save file. i.e without persistence.

How did you create your original installation?
