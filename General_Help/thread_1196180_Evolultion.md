### Post by Agent Smith on 2009-06-25
I use Thunderbird so i can have seperate inboxes for each pop account. When you add an account to Thunderbird the wizard asks if you want to use a "global inbox". Make sure you untick this option. Its about the 3rd or 4th setup screen in the wizard. If you leave it ticked all messages will go to the one inbox folder.Hope this helps.
