### Post by silenthatred on 2010-06-02
I got it to work on 64-bit Windows 7 by installing VMware Player 3.1 (it's free) and then installing Windows XP Pro SP3 as a VM using the USB passthrough feature.  I would imagine the same would work for Linux, although I haven't tried VMware Player in Ubuntu, but the adapter is recognized by Ubuntu when I plug it in.  Video performance was the same as it was on my older XP laptop...no lag.  Win!  I still have a LEGAL version of XP, so your mileage may vary depending on if you do or not.  Dualbooting is also another option for some.  Personally, XP just blue-screens on install on my machine before it even gets through the first file copy part.  Maybe it's my i7 mobile, but who cares, the VM works and I don't have to reboot.
