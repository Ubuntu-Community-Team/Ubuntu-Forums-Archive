### Post by coffeecat on 2011-09-26
> **penguin1 said:**
> It didn't even ask me to install a password or anything?

I think you'll find it did. It is impossible to install Ubuntu without choosing a password - unless you've found an obscure way to get around the "who are you?" page of the installer. It used to be that you could do an OEM installation from the alternate CD and the password would be chosen at first boot, but this doesn't sound like what you've done. 

This is how to reset the password when you've forgotten it:
 
[http://www.psychocats.net/ubuntu/resetpassword](http://www.psychocats.net/ubuntu/resetpassword)
