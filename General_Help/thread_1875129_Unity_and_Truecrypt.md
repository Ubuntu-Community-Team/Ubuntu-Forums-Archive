### Post by choochoousmc on 2011-11-04
In classic. If you opened a truecrypt container via the trucrypt control window,
and then clicked exit, a truecrypt icon would appear in the top menu bar.

In unity, no such icon remains in the side bar of programs.
So if you want to mount another container you can't. Because clicking on the truecrypt shortcut icon in the apps menu, says truecrypt is already running.

So how do reaccess truecrypt to unmount and remount one or more other containers?
