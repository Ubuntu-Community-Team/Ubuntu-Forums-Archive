### Post by ironwolfe on 2008-05-13
VPN is the tunnel not the actual program that does the remote desktop.

I use Hamachi, which has a pop up user list with "networks" - I like this, as it allows for easy VNC use.  I also set up gHamachi which is the GUI for linux.

Here is an awesome thread to walk you through it:

[http://ubuntuforums.org/showthread.php?t=135036&highlight=hamachi+ghamachi+GUI](http://ubuntuforums.org/showthread.php?t=135036&highlight=hamachi+ghamachi+GUI)

You can also use ssh - if you have access to the other computer and want something a little more techie.
