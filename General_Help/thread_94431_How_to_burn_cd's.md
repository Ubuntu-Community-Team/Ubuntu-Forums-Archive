### Post by olavi on 2005-11-24
Ubuntu's default tools aren't bad, but you need to install cdrdao before they work. Use Serpentine to burn audio cds. For mp3 support, download the correct gstreamer plugin.

Personally I prefer Graveman! to GnomeBaker. The interface is much cleaner yet about as powerful. However, graveman won't add itself to the applications menu automatically. K3b is a huge, Nero-like program that requires the KDE libraries, so it may not be your first choice if you're using old hardware.

You'd better simulate burn first. If you get performance troubles, you will need to enable DMA for your drive.
