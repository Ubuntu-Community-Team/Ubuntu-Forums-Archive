### Post by jerome1232 on 2008-08-17
As far as incompatibility goes the list is VERY small off the top of my head java plugins take workarounds to get working that's all I can think of. (zsnes you use to have to compile your own 32 bit binary on a 32 bit system and move it over but thanks to dfreer there's an easy to install package)

It may take a few workarounds with some programs. 

Speed you might notice doing number crunching cpu intensive tasks with programs compiled for a 64bit architecture (such as compression, video manipulation etc...), of course if you have 4GBs + ram you will have that benefit.  But regular desktop use nothing human noticeable.
