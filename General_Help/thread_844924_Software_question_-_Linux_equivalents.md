### Post by bobblehat on 2008-06-30
For Visio, try Inkscape. I used to use Visio regularly and thought it was the one thing I'd need to keep windows for. I now use Inkscape instead for professional quality diagrams though and they're actually much better looking as a result. I guess it depends what you use Visio for as Inkscape isn't a direct equivalent but does have some overlaps.

sudo apt-get install appname
``` from terminal. 
Where "appname" gets replaced by the actual application name off cause :)
