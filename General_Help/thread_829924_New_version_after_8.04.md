### Post by issih on 2008-06-15
The ubuntu release model is to release every 6 months, and you can upgrade through the update manager at any point after release.

This will involve a significant number of files being changed, and often there are teething troubles for some people, that's just the nature of being on the front line as it were.

If you want to be a bit more stable just stick with one of the long term support releases (lts). Hardy heron is the latest of these and the promise for an lts is that it will be updated for a period of 3 years.

You can therefore happily run the os you have now (if its heron) for 3 years without ever worrying about upgrade issues.

If you want to hop onboard the bus to new features and exciting new toys, I'm afraid the trade is that you might have to cope with a few more issues.

General rule installing from fresh is best, that said I have a box that has been upgraded from edgy, through feisty and gutsy and is now merrily running hardy, its still quite happy, if a it cluttered :)
