I set my keyboard to Bengali and it generally works well. The vowels get combined into ligatures with the consonants automatically :D

However, none of the conjunct consonants get combined into ligatures. My book lists a lot of these ligatures, but I can't type them.

### Post by toby_1_kenobi on 2012-08-28
Reading on in my text book I discover that in some instances you don't want a particular two consonants to join, so I guessed there must be some way to make them join manually, and then I found out about the "virama" (hasant) key which you put between two characters to join them. On the standard Ubuntu Bengali keyboard it's the 'g' key.

---

### Post by toby_1_kenobi on 2012-09-17
Now I've switched from the "Bengali" keyboard to the "Bengali (India)" keyboard. For this keyboard the virama is on the 'd' key.

### Post by zombiezparadize on 2012-10-25
I use the "Bengali (India, Baishakhi)" keyboard. I find it the most intuitive to use because the keyboard's layout is based on the sounds of the english consonants. For example, the character "b" on the English keyboard gives you access to the characters "&#2476;/&#2477;" in Bengali which is very intuitive to me.

Good luck. Bengali is my mother tongue but I was never formally educated in it. I learned to read and write on my own and I am trying to read more. The literature is so rich and I wish I could understand more of it. :)
