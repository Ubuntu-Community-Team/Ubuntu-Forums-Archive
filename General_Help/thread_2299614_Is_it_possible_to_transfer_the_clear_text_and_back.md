### Post by grahammechanical on 2015-10-19
I would not be surprised if Ubuntu Phone is not using Ubuntu font which is also used on Ubuntu desktop.

[http://font.ubuntu.com/](http://font.ubuntu.com/)

Differences in visual quality could be due to differences in screen quality between the phone screen and the desktop monitor. Differences between video adapters would also have an effect. Is it possible to connect your Ubuntu phone to a monitor? Is the quality of the font the same? 

I do not have a Ubuntu phone but in the past I have had installed on my desktop machine Ubuntu Desktop Next and I now have installed Ubuntu Personal. Both of these are the phone OS running on my X86 CPU and using Mir and Unity 8. I have not noticed any difference in font quality in comparison with Ubuntu desktop + Xserver + Unity 7 installed on the same machine. Mijnd you, I have not been looking.

On Ubuntu desktop Compiz is the compositor/window manager. On Ubuntu phone the compositor is Mir. How they compare is very much a subjective opinion. Desktop Ubuntu has benchmarking utilities that we can install such as glmark2 OpenGL (ES) 2.0 benchmark. I do not think that you will find that utility in the app store. Besides the test has to be between Xserver+Compiz+Unity 7 on a desktop machine and Mir+Unity 8 on the same desktop machine. So, we ordinary users are not yet in a position to find the answers ourselves.

I will say this about Ubuntu Personal + Mir + Unity 8 on my desktop machine it does seem faster or smoother or whatever you want to call it.

Regards.
