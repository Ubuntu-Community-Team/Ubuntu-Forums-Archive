### Post by bingoUV on 2008-09-21
If you install from the Debian testing image, it will usually be more up to date than Ubuntu. For the rest of my reply, I will assume Debian testing, as Debian stable will be very boring.

Main differences: 
1. Debian might be a bit unstable as the packages are not tested much to see how they interact with each other. In my limited experience with Debian, it works fine mostly.
2. Default theme of Debian is much better (personal opinion).
3. Debian doesn't suffer from an equivalent of the bug [https://bugs.launchpad.net/ubuntu/+source/upstart/+bug/85014](https://bugs.launchpad.net/ubuntu/+source/upstart/+bug/85014). Ubuntu is sitting on it for 19 months.

Debian installer lets you create(or carry forward) a separate /home partition much more easily than Ubuntu's.

Since Ubuntu is based on Debian, there shouldn't be any big surprises. Plunge into it.
