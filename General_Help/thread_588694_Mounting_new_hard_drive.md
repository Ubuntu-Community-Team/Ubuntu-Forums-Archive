### Post by benerivo on 2007-10-23
Type blkid in a terminal. The output should show you the UUID's for each partition you have. As you don't have an entry for it in fstab, just make a new one. If i were you i would have the entry in the same format as your existing ntfs (hda1). The only differences will be the UUID and the mount point. Choose anything you like for the mount point, such as /media/hdb5. Then when you have saved fstab, you will need to make the mount point in your linux system, so type```
sudo mkdir /media/hdb5
```in a terminal. Then reboot.
