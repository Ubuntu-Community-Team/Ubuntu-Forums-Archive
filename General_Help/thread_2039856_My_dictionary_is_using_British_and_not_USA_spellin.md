### Post by coldraven on 2012-08-09
If you mean the spell checker in Firefox then do this.
Right click in any text box that has more than one line. select "Languages" then "English (United States)". The one in use should have a black dot next to it.
If you don't see that option go to Tools>Add-ons>Get Add-ons, download the correct dictionary and do the first step again. Hope that helps
