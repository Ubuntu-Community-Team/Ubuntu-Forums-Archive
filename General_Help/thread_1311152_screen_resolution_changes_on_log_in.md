### Post by sux2bu101 on 2009-11-03
I had similar issues...

I followed this howto with a small twist:
[http://ubuntuforums.org/showthread.php?t=83973&highlight=howto+xorg.conf+resolution](http://ubuntuforums.org/showthread.php?t=83973&highlight=howto+xorg.conf+resolution)

I ended up using the nVidia Xserver to save a copy of the xorg.conf file to my home directory after I got the resolution where I wanted it (in my case 1440x900).

After the file was saved, I rebooted, at the login screen pressed "ALT + CRLT + F2" 

I logged in and I used these commands:

To make a backup of the file

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup
To use the file nVidia Xserver created. 

sudo cp /home/xorg.conf_backup /etc/X11/xorg.conf
After that I rebooted and the login and my desktop environment resolution was at 1440 x 900.

Hope this helps!  Good luck!

PS.  I am no expert, but it worked for me and BE SURE TO MAKE A BACKUP of the xorg.conf file... if you do something wrong you can recover using the same "sudo cp 'file to copy from' 'file to copy to'"
