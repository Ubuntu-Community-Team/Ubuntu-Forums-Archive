### Post by cyberphrog on 2010-08-21
How do I add a shortcut on my desktop/menu bar that will launch and direct me to a particular website?

---

### Post by Vishnu V on 2010-08-21
Right click on the desktop and select Create launcher
Set Type as Location
Enter name of shortcut in name
Enter the website address in command
Click Ok
You can drag and drop this to panel 
or right click on panel and  select add to panel and click on  create custom application launcher and complete the form
