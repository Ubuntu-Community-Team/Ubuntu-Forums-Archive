### Post by tomreyn on 2012-07-14
Hi,

please ensure your PC matches the minimum system requirements, as documented in the MegaGlest README, which is included in the distribution and also available at [http://readme.megaglest.org](http://readme.megaglest.org). In your local installation this file should be located in [/usr/share/doc/megaglest/](file:///usr/share/doc/megaglest/).

If your PC matches these requirements and you are still unable to use the game using the Ubuntu package, please file a bug report at
[https://bugs.launchpad.net/ubuntu/+source/megaglest/+filebug](https://bugs.launchpad.net/ubuntu/+source/megaglest/+filebug)

You may also try to use the upstream installer (non-packaged, installs into your home directory) or install manually from archive files, both of which is available at [http://megaglest.org](http://megaglest.org)

Hope this helps,

Tom
