### Post by Cheesemill on 2012-03-21
Just do:
```
sudo apt-get install xubuntu-desktop
```
To install all of the Xubuntu packages and then follow the 'Remove  Ubuntu' instructions here to remove all of the default Ubuntu packages:
[http://psychocats.net/ubuntu/purexfce](http://psychocats.net/ubuntu/purexfce)
