### Post by Hedgehog1 on 2011-05-09
If you are able to boot windows, you can copy your documents from the Ubuntu partition to the Windows partition using this: **_[ext2-ext3-ext4-filesystem-from-windows]("http://www.soluvas.com/read-browse-explore-open-ext2-ext3-ext4-filesystem-from-windows-7/")_**

If you cannot boot from either windows or Ubuntu, You can use a LiveCD/LiveUSB boot from, select 'TRY' and then copy files from any disk partition to an external drive or a USB stick.
