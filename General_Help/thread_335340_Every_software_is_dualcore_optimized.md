### Post by zgornel on 2007-01-10
The main benefit of the dual core architecture is that you can run 2 different threads/processes/applications on "one" cpu (e.g. listen to music or using skype while playing Quake3) with no slowdown. Single applications that use both cores are relatively rare (most video encoders and 3d rendering programs do because of their high computational requirements).
