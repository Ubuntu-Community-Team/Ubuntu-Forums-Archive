### Post by Scucci on 2008-09-15
I had the same problem for a while until I stumbled across the answer that worked for me.

I had to install the Compiz Fussion Icon package. When you start that an icon will appear in your system tray. Right click on it, go up to Select Window Manager, and change it to Compiz.

Again, that's what worked for me.


Edit: DOH! And the most important part, same thing, right click, then select window decorator, and make sure it's set to Emerald...

Can't believe I forgot that part, that's the whole reason I started the reply in the first place.
