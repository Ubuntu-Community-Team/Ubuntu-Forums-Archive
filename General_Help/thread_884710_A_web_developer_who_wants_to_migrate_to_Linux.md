### Post by mr_niceguy on 2008-08-20
I have found JEdit has enough configuration options that I am willing to put up with a few seconds delay on the initial launch. Once it is running it is fairly responsive and if it is working slowly for your tastes there are configuration options that will make it faster. For example, turning off text aliasing/smoothing helps a lot but you want to make sure to use a clean, true-type font in that case.

Kate editor is fairly feature-full as well and runs leaner, especially if you are already using the KDE desktop. But it does not have as many options as JEdit.

Bluefish is a nice editor except that if you develop large files it will slow down or even hang.

If you're a command line person you may want to check out if there is a bundles like feature for Emacs or Vim.
