### Post by 2F4U on 2011-10-02
Disconnect the laptop from its power source, remove the battery and wait a couple of minutes. Hit the power button a couple of times while you wait. Then reinsert the battery and reconnect to the power source and try to start the laptop.

Do you see the BIOS message when you start the laptop? If not, there may be a hardware problem.

### Post by 2F4U on 2011-10-03
I was pointing to the BIOS message because that happens long before any part of the operating system is involved. So, even if the unexpected shutdown has damaged parts of the OS you should see the BIOS coming up. If you don't see the BIOS I would suspect some problem with the graphics card or monitor. If you have the chance to hook up an external monitor and then see the BIOS, this would indicate a problem with the laptop's internal display. Else it may be a problem with the graphics card.
