### Post by FuturePilot on 2007-12-04
You have to cd into the directory that was created when you extracted it. So if it's on your Desktop
```
cd Desktop/whatever_its_called/
```
Then
```
sudo ./flashplayer-installer
```
When it asks you for the directory to the browser enter
```
/usr/lib/firefox/
```

### Post by barney_1 on 2007-12-04
If you can't figure out how to extract it with Archive Manager you can use this code:
```
tar -xvf yourfilename.tar.gz
```

Then follow what FuturePilot wrote above
