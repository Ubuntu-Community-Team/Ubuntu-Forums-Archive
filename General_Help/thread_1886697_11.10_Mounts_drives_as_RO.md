I recently did a fresh install of 11.10 on a machine that was previously running 10.04

Now, anytime I connect another drive, it is mounted as read only. This happens with every drive I try, not just some. This is a big problem for me because I use Ubuntu to check customers' drives and sometimes to fix things if I cannot get into Windows on their machine.

---

### Post by jjex22 on 2011-11-25
Is this just happening for windows NTFS drives? you may be missing the NTFS driver:

sudo apt-get install ntfs-3g

Hope it helps!
