### Post by scouserlady on 2008-04-07
I have installed Ubuntu on an old Evesham which previously used Windows ME.   I have an active and successful Homeplug home network through my mains wiring, using a Netgear modem router and Addon power line communicators.   I have connections to 3 desktops, 2 of which are working fine.   The third, the Evesham, worked well when it used Windows, but now it uses Ubuntu, it is not picking up the Homeplug connection at all.   It just seems to be not trying at all.

There is probably something I am not doing which I should do.   I am very much a Ubuntu newbie and an elderly, non-techie lady, so please, keep it simple.   Any help would be much appreciated.

---

### Post by scouserlady on 2008-04-13
I have sorted this problem by using a 4 port modem router and a ethernet connection directly between the router and the PC and it is now working fine.
