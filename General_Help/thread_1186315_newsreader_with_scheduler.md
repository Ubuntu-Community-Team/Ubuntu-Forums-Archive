It is perhaps not for the fainthearted but the newsreader slrn + Leafnode 2 would pretty much fill this requirement, except of course that Leafnode 2 is an external NNTP proxy server and the scheduling is run by cron. These are both relatively complex pieces of software to setup:

[Howto] Setup the commandline newsreader slrn
[http://ubuntuforums.org/showthread.php?t=475246](http://ubuntuforums.org/showthread.php?t=475246)

[Howto] Setup and use Leafnode-2 with the newsreader slrn
[http://ubuntuforums.org/showthread.php?t=676837](http://ubuntuforums.org/showthread.php?t=676837)

and perhaps this level of complexity is not quite what you are after? I run this software on my own computer (and they are my guides!) and both run beautifully.
