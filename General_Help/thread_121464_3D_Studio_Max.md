### Post by kurageart on 2008-02-28
HI! I managed running 3ds max in ubuntu using virtual box...
That's far easy, as long you don't wanna use directx mode (open gl works well).

But actually, I think that blender is very good too.
I'm getting operative with blender, after being a 3ds max user for maybe 10 years very quickly, they are plenty of video tutorials, and they are some texturing tools I've been dreaming for, using max.

There is everything you can look for, it's just hided somewhere in the interface , or you have to memorize the right shortcuts...

I see that working with blender tends to be faster, actually, since you are quite forced to learn shortcuts, while in max everything is somewhere in the interface..

The only bad thing, yet, is the lack of some serious n-gon support... even that if u need ngon, you can select several faces, press f, and then the software will handle those faces almost like n-gon..

Personally, I'm very encouraged learning, fore some serious projects i have in mind.
Creating a small render farm using blender and Ubuntu would be like 1/10 than creating it using 3ds max...

cheers
