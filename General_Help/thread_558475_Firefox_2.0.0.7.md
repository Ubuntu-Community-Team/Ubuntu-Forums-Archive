### Post by elvis on 2007-09-24
Upgrade via the Ubuntu "Add/Remove..." tool.

Usually the most recent version of Firefox will appear in the repos within 24 to 48 hours of the Mozilla release.  From a maintenance point of view, it's much smarter to wait and let the package maintenance system update it for you than manually installing it.

A note: you probably don't need 2.0.0.7 because it's a Windows-focused patch that solved a vulnerability caused through QuickTime. But Linux does not run QuickTime natively, so that patch is not needed (unless maybe that you use an emulator to run QuickTime)

I guess that's why they haven't put it on the repos and won't ever put it.
