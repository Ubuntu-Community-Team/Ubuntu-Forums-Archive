### Post by joeyjoejo on 2010-05-08
I found I also had a simila problem on a similarly powerful system with an ATI graphics card, I got round it by disabling animations on window maximising/minimising. It made it respond instantly again 

(system -> preferences -> appearence -> visual effects tab -> none)

I know it's not ideal, and I have no idea what causes the problem, but it seems to work. What's doubly odd is that I can run Nexuiz in 1440 x 900 with the graphics set to ultimate (maxing out everything) and only have fairly light slowdown - so it doesn't seem to actually be a hardware problem, nor system-wide
