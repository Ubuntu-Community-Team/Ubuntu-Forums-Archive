### Post by ibuclaw on 2008-09-29
Which archive are you getting the software from ?

In the Menu, find "Software Sources".
In Ubuntu, that is "**System->Administration->Software Sources**"

Check that the Server that you download from is in your country.

And if it already is... try selecting a different server, or click "Select Best Server" when you select "Other" in the dropdown tab.
