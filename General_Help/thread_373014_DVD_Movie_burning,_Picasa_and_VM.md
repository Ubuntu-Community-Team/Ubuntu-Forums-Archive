### Post by z987k on 2007-02-28
There is a linux version of picasa, [http://picasa.google.com/linux/](http://picasa.google.com/linux/)

Also instead of using Nero, there are a lot of foss apps out there that IMO work a lot better.
Tovid is my favorite, I don't think it's in the repositories, but someone built a deb for it out of v.29 at [http://tovid.wikia.com/wiki/Installing_tovid](http://tovid.wikia.com/wiki/Installing_tovid)
or you can install the latest from source.
I use this like every day and I highly recommend it, it has a CLI and GUI both very easy to use.  It can go from an avi encode to mpg then set up the dvd structure and burn it all in one.
Other options i've seen are avidemux and then dvdauthor.
