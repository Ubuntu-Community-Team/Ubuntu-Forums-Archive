### Post by Mardoct909 on 2008-11-16
It's possible, but unless your terribly opposed to reinstalling, it's a better idea to do it all from scratch.

Install Windows with a partition size that leaves enough for Ubuntu.

Install Ubuntu on the rest of the space.

Done! You don't need two hard drives with that much space. If you want more space anyways, though, format it as NTFS and connect it as a slave to the mobo. You can dual boot, and both OSes can access the second drive and leave data on it.
