### Post by InTheNameOfDelicious on 2011-05-10
I solved this by installing xte and then assigned a hot corner to the command
```
xte 'keydown Alt_L' 'keydown F1' 'keyup Alt_L' 'keyup F1' 
```
Actually this turned out to be less useful then I thought it would be. Oh well...
