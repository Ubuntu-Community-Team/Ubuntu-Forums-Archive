### Post by mc4man on 2010-09-28
Right click on any folder -> Open With -> Other Application, scroll down and choose 'Open Folder', that should return default to nautilus.
(if you don't have a folder available on the desktop then just create one.

In the future, if r. clicking on a folder -> Open With -> Other Application (or Open With  Other Application), uncheck the box " Remember this application for "folder" files"
