### Post by badger_fruit on 2010-05-25
Hello
A Live CD will boot the PC and allow you to read the on-board hard-drive to recover your pictures.  As for where they are, well, that depends - where did you save them?  I'm not 100% certain but check in the /home/your_username folder to start with or I suspect there'll be a decent file search utility you can use from the Live CD.

If the "graphical error" is a hardware thing then a Live CD won't fix that but you should be able to remove the Hard-drive and install it into a desktop PC THEN use a Live CD on the desktop PC to search and recover your files.

Good luck!
