### Post by stump138 on 2007-12-07
as far as navigating in your terminal remember that it is case sensitive and you most often need to be explicit.  for example getting into your desktop folder
```

cd /home/name/Desktop

```

typing "ls" (no quotes) at the prompt will give you a listing of the files in your current working directory.  There is a nice explanation here:

[http://linux.org.mt/article/terminal]("http://linux.org.mt/article/terminal")


this is a good URL to help you setup flash:
[http://www.psychocats.net/ubuntu/flashubuntu]("http://www.psychocats.net/ubuntu/flashubuntu")
