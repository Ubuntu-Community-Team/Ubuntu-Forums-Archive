### Post by lykwydchykyn on 2010-05-30
Older versions of photoshop and dreamweaver are known to work with WINE, or you might have better luck with crossover from codeweavers ([http://www.codeweavers.com/](http://www.codeweavers.com/)).  

The virtualization option involves running an install of Windows in Virtualbox and using your apps there.  It's nearly 100% compatibility because you're actually running Windows in this case, but it's not the best solution because you'll take a performance hit, and it can be a drag shuttling files in and out of the virtual environment.  Plus you have to maintain two operating systems instead of one.

GIMP has long been considered the "flagship" raster image editor for Linux, but there are alternatives that might suit you better.  Look into Krita and Cinepaint.

There's not really a good dreamweaver clone around; kompozer is about the only WYSIWIG HTML editor with a significant feature set, and most dreamweaver users aren't that impressed with it.  If you want to edit code, the options expand a bit, e.g. bluefish and quanta plus.

As for audio, not sure; I was way into Jeskola Buzz back in the day, and once got it working in WINE, but otherwise I haven't found any Linux music making apps that awaken the muse so far.  There's LMMS, but darned if I can figure out how to work with it.
