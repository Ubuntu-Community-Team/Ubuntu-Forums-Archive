### Post by zombifier25 on 2012-03-22
You can use Natty, or Lucid builds for Maverick, however it's not really recommended that you do so, since it would probably cause [Dependency hell](https://en.wikipedia.org/wiki/Dependency_hell) (my old Lucid build was rendered unusable because of a "backport" PPA I use)
My recommendation is that you keep using Maverick until Precise come out and switch to it. The GNOME 2-lookalike classic session is very functional and usable in Precise. All you need to do is install the gnome-panel package.
