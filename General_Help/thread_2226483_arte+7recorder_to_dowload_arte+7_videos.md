### Post by Mariane on 2014-05-27
The first program to do this was called arte+7recorder and it could be installed with: 

[code]
sudo add-apt-repository ppa:arte+7recorder
sudo apt-get update
sudo apt-get install arte+7recorder-5
[code]

The -5 force version 5 to install, there were problems with version 4. 

Now a new program exists to do that, it is called qarte.

People outside France may have to use a proxy in order to connect to arte's web site from an IP which seems to be in France.
