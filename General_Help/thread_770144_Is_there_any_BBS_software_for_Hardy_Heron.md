### Post by PCMan on 2008-04-27

BBS - Bullitin Board System is a historical system. Today only people live in Taiwan, China, and Hong Kong use it. So I guess you shouldn't ask this problem here. Go to Taiwan Ubuntu forum [http://www.ubuntu.org.tw/](http://www.ubuntu.org.tw/) then you'll get more help there.

BTW, PCMan BBS client is developed by me, and I also created a Linux version called pcmanx. It's already in ubuntu repo.

If you talk about KKman, which is a program used in Taiwan only, I think nobody here knows what's that.

Maybe you'll be more interested in Lazybuntu: [http://lazybuntu.openfoundry.org/](http://lazybuntu.openfoundry.org/)
This can automatically setup all frequently used programs and ensure usable Chinese environment for you. PCManX is already included.
