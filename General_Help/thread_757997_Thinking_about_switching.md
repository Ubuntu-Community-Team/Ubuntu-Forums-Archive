### Post by bingoUV on 2008-04-17
[B]Im 17 and i want to learn how to code software, is this a good OS to do that?
[/B]    The best

**WoW and Nero 8 and windows games work on it?**
    See K3b at work, and you will forget Nero. Don't know much about games, but not all windows games work nicely.

**What version has cube installed with it, and what would be best version for me?**
    Latest. For a beginner, try Ubuntu (instead of xubuntu, kubuntu etc). 

** Im tired of windows crashing, is ubuntu more stables?**
   Depends on hardware. In short, devices (graphics cards, wireless netword cards etc.) that need drivers that are not already in kernel are most risky. Post your hardware details here. You can find threads about linux friendly hardware on this forum

**And is it true you dont need to install drivers, the kernel does it by itsself?**
     Well, kernel knows about lot of hardware of itself, and for those supported hardware you do not need to install additional drivers. Old hardware works mostly better than on other OSes. But not all hardware is supported fully in this manner so you might have to install drivers.
