### Post by j.biddy on 2011-05-10
Hey all. I've been doing some research on operating systems of the past and in reading about the NeXTSTEP operating system I stumbled upon the [Shelf]("httphttp://en.wikipedia.org/wiki/Shelf_%28computing%29") interface feature. As I understand it, it's essentially a dock that is a placeholder for files, folders, or shortcuts dragged onto it. Essentially a purely graphical clipboard and launcher bar in one. 

I believe OS X has a similar function (as a NeXTSTEP successor) with the Finder but as I understand it you have to be holding the Apple key while having the Finder opened or something. The NeXT and OPENSTEP implementations even went so far as to have a customizable Shelf for each directory if you wanted too (meaning program, file, folder shortcuts could change based on the directory you navigated to). 

I'm sure some people will find this redundant or tell me to use the command line or something, but I really do think this is an extremely powerful hybrid graphical interface feature that has sadly been lost over time. Anyone out there know of any implementations of this in Ubuntu or other GNU+Linux distros?
