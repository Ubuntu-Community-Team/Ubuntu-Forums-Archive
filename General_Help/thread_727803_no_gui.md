Install the Intel package for the X-Server with:-
```
sudo apt-get install xserver-xorg-intel
```
Then kill the X-Server with:-
```
sudo /etc/init.d/gdm stop
```
reconfigure the X-Server with:-
```
sudo dpkg-reconfigure xserver-xorg
```
select the driver as "intel" and any other options you may want and then restart the X-Server with:-
```
sudo /etc/init.d/gdm start
```

See if that solves your problem.
