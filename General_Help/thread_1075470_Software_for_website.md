### Post by tombott on 2009-02-20
I use ZenPhoto.
It allows you to FTP into a folder then auto thumbnails pictures for you.
You can also set user access by album.

[http://www.zenphoto.org/](http://www.zenphoto.org/)

You'll need PHP and MySql enabled on your host server.
