### Post by jamesrl on 2008-10-05
If you don't need that specific version of the man pages, you can just go to your favorite package manger and install libstdc++X-X.X-doc .  E.g., from the terminal run
```

sudo apt-get install libstdc++5-3.3-doc

```
will install libstdc++5-3.3-doc (the documentation for libstdc++).
