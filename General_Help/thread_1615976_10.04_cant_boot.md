### Post by corrytonapple on 2010-11-07
Welcome to the forums!
OK. You may just be having some Hard Drive (HDD) issues. Boot up that live USB Image, go to the Terminal and type:
```

mount | grep sda6

```That will tell you where the file system is mounted. Then, type:
```

sudo umount /dev/sda6

```That will unmount it. Then type:
```

sudo fsck

```That will check the whole drive for errors. Also, before you unmount, I would back-up any data you have on that hard drive.

### Post by corrytonapple on 2011-01-17
Your swap partition on your HDD (Hard Drive) may be using it.  Try turning off swap using GParted.  I think it is installed by default now, so Launch it by going to **System>Administration>GParted.**  Then go to your HDD, find your Ubuntu partition, and open it so it shows its contents.  You will see a small Swap partition.  Right click it and turn it off.  Then try what you want to do again.
