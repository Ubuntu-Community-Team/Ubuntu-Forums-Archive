### Post by Dennis N on 2018-08-07
I think you are referring to the scroll bar. The scroll bar on the right depends on the theme you are using. In Lubuntu, the default one is "Lubuntu Default" and now it has a scroll bar that widens when you move over it. To get a fixed size scroll bar, you need a different theme. 

You can try some others in the "Customized Look and Feel" dialog, but some are so old they have no support to gtk3.

I suggest you install Greybird-gtk-theme from the Ubuntu repository and try that. The scroll bar is a fixed size, and larger. 

(I use Zukitwo theme which has a wider scroll bar, but it's not in the Ubuntu repository, so you have to set it up yourself. It can be found at gnome-look.org.)
