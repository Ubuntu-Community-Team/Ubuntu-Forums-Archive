### Post by grahammechanical on 2016-07-05
On Ubuntu with Unity we install gnome-session-flashback. That will give us 2 new login options Gnome Flashback (Compiz) & Gnome Flashback (Metacity). The Ubuntu symbol will appear as part of the login panel. It will be on the right and above the panel where we type our password. Click on that symbol and a menu will appear.

Gnome Classic is a Gnome 3 shell extension. So, if you install gnome classic you will get the complete Gnome 3 shell desktop. Gnome Classic is best used with the Ubuntu Gnome distribution.
