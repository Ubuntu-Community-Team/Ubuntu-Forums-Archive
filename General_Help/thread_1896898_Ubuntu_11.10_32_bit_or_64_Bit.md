### Post by 2F4U on 2011-12-18
> Will 64 bit run slow on this much ram ?

Are you kidding? 3 GB is far from being much RAM these times. Anyways, you can go with both. 64 bit will probably use more RAM than 32 bit (at least on my machines). 64 bit can make better use from 64 bit architecture than 32 bit, which runs in a compatibility mode, so 64 bit is supposed to run faster than 32 bit. Beginning with the next Ubuntu release, 64 bit may become the default architecture.
Personally, I choose 64 bit when I install on a 64 bit hardware and 32 bit otherwise.
