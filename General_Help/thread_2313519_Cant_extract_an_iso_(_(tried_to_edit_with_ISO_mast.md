### Post by sudodus on 2016-02-13
- Would it work for you with a persistent live system or an installed system (installed like into an internal drive but into a USB pendrive)?

- Do you intend to use the system in one or several computers? In UEFI or BIOS mode? To run live or to install systems?

[Try Ubuntu (Kubuntu, Lubuntu, Xubuntu, ...) before installing it](http://ubuntuforums.org/showthread.php?t=2230389)

Persistent live drives are similar to basic live drives, and in addition can save settings, installed programs and data files, but are slower (with the same hardware). The kernel cannot be upgraded (except by upgrading to a new iso file in a 'grub-n-iso' pendrive). An installed system in a USB drive can be 'completely upgraded'.
