### Post by grahammechanical on 2013-12-25
There is a reason why Linux Mint is called that and not Ubuntu Mint, or something like that. The reason is that the Mint developers have moved so far away from what is Ubuntu that they are not allowed to use the name Ubuntu in their name. Unlike the flavours of Ubuntu which are Xubuntu, Kubuntu, Lubuntu, Ubuntu Gome, Ubuntu Studio, Mythubuntu and Edubuntu.

So, a fundamental difference is that Linux Mint is not Ubuntu with a different user interface.

### Post by buzzingrobot on 2014-04-02
Mint is built on an Ubuntu base and, as long as it relies on Ubuntu's repositories, must track Ubuntu closely enough to allow those Ubuntu packages to be installed and used transparently.   

But, Cinnamon is developed by Mint for Mint, so it seems reasonable to expect it to perform best on its intended platform.  Plus, a bug filed against Cinnamon at Mint is likely to get more traction than one filed at Ubuntu, which will likely be closed as an upstream issue.

Mint also includes some Mint-specific tools and uses a different approach to updating.

Of course, Mint also releases KDE, XFCE, and Mate versions.
