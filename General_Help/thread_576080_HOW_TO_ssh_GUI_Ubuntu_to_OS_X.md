This is my first thread so sorry if it's not great.  I just want to tell you how to open GUI programs from a remote machine to the local one on the same network.

Install X11 because it won't work just in terminal, if it's not installed already(/Applications/Utilities/). 

Then open X11 type the following command:

*ssh -X username@x.x.x.x*

Login then just type firefox and there you go It appears in a window on your mac.
