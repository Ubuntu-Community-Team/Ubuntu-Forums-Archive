### Post by cstudent on 2007-01-23
Sounds like you just copied the .iso file to a CD. You can boot into Windows and insert the CD. Then go into My Computer and click on your CD drive to view the contents. If you see one file on the CD ending in .iso, then you didn't burn the CD correctly. You'll need software like Nero to burn the .iso image onto the CD. This will create a bootable install CD. There are some free software programs out there that can burn an .iso for Windows, but I'm not familiar with what they are. A little Google research should supply an answer for that.
