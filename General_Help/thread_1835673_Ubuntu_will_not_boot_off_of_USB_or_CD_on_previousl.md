### Post by andrewrost3241981 on 2011-08-29
Hello,

     I have a laptop (HP Pavilion zv6000) that has Ubuntu installed on it.  It was giving be problems booting from the hard drive.  I did not have anything terribly important on the computer, so I decided to do a reinstallation.  This is where I am now having problems.

     My attempts at installation always end up in the computer hanging when I am trying to boot from a CD or a USB stick.  I can usually get to the beginning GUI of the installation, but the computer will just hang and go nowhere after that.  I have downloaded Ubuntu 11.04 from the Ubuntu site.  My computer has had both the 32 and 64 bit versions installed on it at one time or another.  I have tried installing the operating system from both a CD and a USB flash drive.  I have tried installing both the 32 and 64 bit versions on my computer in the last two days.  I have tried downloading Ubuntu a few times thinking that the files that I downloaded were corrupt.  My subsequent attempts at downloading the OS again and installing it have made no difference.  I have tested the RAM (768MB DDR) and it has passed Memtest 86+ without problems.

     I don't think that it is a problem with the hard drive because the CD/USB boot process doesn't really do anything related to the hard drive before it starts to hang.

     I'm just hoping that it is not hardware related because my other computer's PSU died and I decided that I was going to build a new desktop rather than fix my old desktop.  If this computer's hardware is bad, I really don't want to fix it either because it is about 6 years old but I don't have any other computer to do my work on either than my parent's computer which I am using right now.

     I have been trying other boot discs to see if they are giving me the same problem, but so far they haven't.  Right now I am thinking that it is a hardware problem.  I will keep doing some analysis to see if I can determine the real cause of the problem, but I would appreciate any help if anyone can think of any possible causes.

Thanks,

Andrew

### Post by andrewrost3241981 on 2011-08-29
I had a suspicion that it was the hard drive (or something on the hard drive).  I was able to boot other discs.  I used DBAN to erase the contents of the hard drive and I am now able to proceed with the installation without any system hangs.
