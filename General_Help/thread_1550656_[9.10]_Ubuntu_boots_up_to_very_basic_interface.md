### Post by Earl_Maroon on 2010-08-11
It sounds like a problem I had a while back, where something went wrong with gconfd.

All I had to do was delete ~/.gconfd/ and reboot. I don't think it can do any harm as it is automatically generated, but you should probably back it up first before trying.

### Post by Earl_Maroon on 2010-08-11
ctrl+alt+f1 puts you into a terminal
Login with your account
Then type:

```
cp -r ~/.gconfd ~/.gconfd-backup
rm -r ~/.gconfd
```
