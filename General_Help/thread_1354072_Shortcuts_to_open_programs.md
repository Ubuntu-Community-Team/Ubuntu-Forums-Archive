### Post by rafalcieslak on 2009-12-13
Yes it is. Try System->Preferences->Shortcuts (or something similar, as I have translated menus;P )
alternatively press Alt+F2 and launch gnome-keybinding-properties ;)

### Post by goaliedude3919 on 2009-12-13
When I go to System -> Preferences -> Keyboard Shrotcuts, I don't see any options for opening up programs. There are only options for things like minimizing windows, maximizing windows, and other similar things. Also, for me, Alt + F2 launches the run application dialog box.

### Post by oldos2er on 2009-12-13
Click Add, type in a name, in command type in /usr/bin/firefox. Click Apply. Highlight your new custom shortcut, enter your key combination.
