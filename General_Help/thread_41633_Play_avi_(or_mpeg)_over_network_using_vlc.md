### Post by linuxneeewb on 2008-06-04
If you are using a shared folder, you should be able to use the command line and search for the specific files. If you need to connect to a remote server ask the administrator how to do so. From the command line look up the specific files, then type vlc <name of file>  . That should play the .avi files and mpeg files. 

For example if a file is called movie.avi 

simply type

vlc movie.avi


That should hopefully solve your problem
