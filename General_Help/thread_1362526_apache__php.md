### Post by indytim on 2009-12-23
You may want to "Google" a bit on Apache, PHP and MySQL.  There are literally 1000s of tutorials etc covering the basics.  In a concise nutshell, Apache is a webserver, PHP is what's called a parsing language where the output of PHP is parsed back to html (the language of web pages).  MySQL is a database that is integrated into Apache and PHP so that data is stored in MySQL and parsed by PHP into web pages viewed from the web server Apache.

See a link in my signature for one way to easily install LAMP (the bundled Apache-PHP-MySQL for Linux).
