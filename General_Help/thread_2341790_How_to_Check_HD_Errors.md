### Post by Rick St. George on 2016-11-10
Here is what I did. Boot with SYSTEM RESCUE CD. 
Run Gparted, Right Clik on HD and select CHECK.
It will check the disk and fix errors automatically.

Didn't seem to mess anything up thank goodness. System booted fine afterwards.
Can't tell you how many times I've helped people using that Boot Disk. A must tool to have.
