### Post by maggotbrain on 2013-04-14
In case anyone else was curious, LibreOffice 4 can be installed using the PPAs from the LibraOffice packaging team with the following commands:

```
sudo add-apt-repository ppa:libreoffice/libreoffice-4-0
sudo apt-get update
sudo apt-get dist-upgrade
```

See [here]("https://launchpad.net/~libreoffice/+archive/libreoffice-4-0") for more details.
