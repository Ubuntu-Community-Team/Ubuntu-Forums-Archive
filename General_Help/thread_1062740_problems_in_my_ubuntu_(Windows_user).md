### Post by bford16 on 2009-02-07
When you first install Ubuntu, some things might not work correctly.  You may need a special driver for video, and you may not be able to "see" data stored on NTFS drives (Windows XP uses NTFS).  However, most of the problems with Ubuntu are easily solved. I have been using Ubuntu for two years now, and have converted all of my computers to Linux successfully.  Please keep trying.  Start by posting at the General Help forum.  Post one problem at a time, and solve that problem before moving on to the next one.

Some tips:

Find and follow instructions for installing w32codecs and libdvdcss2. You will need them to play some DVDs. You can find help from <https://help.ubuntu.com/community/Medibuntu>.  (Note: it may be illegal to use libdvdcss2 in some countries.)

You may need to install "ntfs-3g" to access files store on your Windows drives.  This is easy to do, and you will find instructions on the Forum.

Have lots of fun!
