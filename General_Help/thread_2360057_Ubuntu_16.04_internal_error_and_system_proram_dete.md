### Post by ajgreeny on 2017-05-01
Is your screenshot sitting on your hard disk somewhere in your home?  That is where it must be.

If so click on the paperclip icon in the toolbar of the Reply to Thread window and a dialogue box will appear.
Click on Add files button top right, then click on Browse button top right and navigate to the screenshot file on your disk (there are size limits to the file types, so look out for that by clicking on the question mark icon also top right).
Click on Upload button top right again.

That should be all that is needed.

As for the system problem that you see, it may help us if you can run command ```
sudo apt-update
``` and show us the output from that.
Please use **Code-Tags** for terminal output. See my signature below for a **How-to**
