### Post by dannytatom on 2008-12-09
[youtube-dl]("http://www.arrakis.es/~rggi3/youtube-dl/")
> 
youtube-dl is a small command-line program to download videos from YouTube.com.


```

sudo apt-get install youtube-dl

```

You can also find it in synaptic.
