### Post by habana on 2019-05-25
I don't use the software center; I find Synaptic Package Manager a better way to find and manage the programs I install. You will find kicad and geda and no doubt others. I personally use geda which does everything I need. If you google "linux electronic design" you will find references to these and other design programs. Work out which is best for you and search for it in SPM. Good luck!

### Post by HermanAB on 2019-05-26
The best free program is probably Eagle.  The help files on the Sparkfun web site are very good.  

The advantage is that you can upgrade to the full version once your project outgrows the free one and the paid version is still cheap.  The paid version includes a very good push and shove auto router.

I've been using Eagle for about 20 years and it just keeps getting better.
