### Post by DeMus on 2009-05-09
When the boot order is correct and you don't see a message to press any(?) key to boot from CD (or DVD) then the disc is not bootable. You probably made a copy of the iso file to disc instead of extracting it while burning.

Start up Brassero. In the main screen is an option to burn an image. Use that to burn your iso file the right way. Then boot again with the disc in the player and see what happens.

If it still does not work, email [email]bill@microsoft.com[/email]

### Post by Tipped OuT on 2009-05-09
[B]Okay kids, stop arguing.

Heres what I suggest dude, make sure the ISO is bootable (should be if you downloaded it from Microsoft's website), make sure your DVD isn't defected, burn the ISO with K3B (search for it in Add/Remove Programs). Make sure to burn it at the slowest speed for best results. Once it's done, restart your computer, go into the bios, make sure you CD/DVD drive is set to boot first, then save the settings, and restart your computer. Your computer should start reading the DVD, and it'll eventually ask you if you will like to boot from it, press any key to say yes, otherwise wait it out and it'll boot to your hard drive (Ubuntu).

Done.[/B]
