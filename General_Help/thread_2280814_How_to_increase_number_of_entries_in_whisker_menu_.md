### Post by Ralph L on 2015-06-02
I like the "Recently Used" Whisker Menu item and have it set to come up when I click on Whisker Menu in the main panel.  However, it seems to limit the number of entries to 10.  I have room for about 4 more entries.  Anybody know how to increase the number of entries.

---

### Post by sisco311 on 2015-06-03
Right click on the menu -> Properties -> Behaviour -> Recently Used -> Amount of items
