### Post by DragonlordP on 2017-01-25
I solved this by using 1080p@30hz. I get flickering at 60hz, 59.9hz and 50hz at this resolution, 30hz is the highest refresh rate that works. But when I change the resolution to 720p, I can use 60hz fine. I never had this problem on Windows, I could always set it at 59 or 60hz. So now it is finally usable, but I would still like to return to 50-60hz as scrolling is quite jerky at 30hz...
