### Post by marshag63 on 2009-08-13
Volter, give the INX live-cd a try - all CLI.  You will learn to do many, many things through the tutorials, which you will be able to use in GUI environmnet as well as CLI.

[http://inx.maincontent.net/](http://inx.maincontent.net/)

Let us know how it goes for you.

If you are talking strictly in a GUI:

Alt + F1 brings up the menu - which you can either navigate with the up/down left/right arrows or start typing the first letter of the list menu item to get to it.  There are lots of keyboard shortcuts built in.

Another option is to use Gnome-Do (its a little purple box on the panel bar, if it is installed).  Press the windows key and the space bar and a dialog box pops up and reacts to every keystroke you type - start programs, search for documents, etc.

And one last option is through Alt + F2 - this brings up the run application dialog box where you can type commands.

Within in firefox:  Go to Edit > Preferences > Advanced:  check "search for text when I start typing."  This will allow you to jump anywhere on the page and to any links.

Try this:

Alt + F1 and start typing "P" to get to the Places menu and choose "Home".  This should bring up the nautilus window.  Type "do" and you should see a little box pop up at the bottom right corner of the window and the highlight should have moved to "Documents."  Press the Enter key to go into the "Documents" folder and start typing the name of a file or folder.  If you want to go back up a level, press Alt + Up arrow key and it will go up a directory.

Alt + F4 will shut down any program window you happen to be working in.
