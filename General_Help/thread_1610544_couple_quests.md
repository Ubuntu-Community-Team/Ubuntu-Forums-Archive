### Post by matsuzine on 2010-10-31
You can absolutely create a c++ program on ubuntu.  Generally, you would use the gnu compiler collection.  For the complete set of command-line build tools, you can... 

apt-get install build-essential

Then use the 'c++' compiler from the command line to build your source files:

c++ hello.cpp -o hello
