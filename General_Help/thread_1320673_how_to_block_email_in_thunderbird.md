### Post by onyxwolf on 2009-11-09
tools> Message Filters...
Or highlight message, go to message>create filter from message...

if from first option
Click New... then change the defaults (subject, contains) to (sender, is {or contains should still work here}) and copy the email address in the text box beside it. Then under perform these actions change (move to folder) to (delete message). You can also add another action with the plus button beside the first and tell it to delete it from your server. You can even create a message template that says "STOP EMAILING ME YOU (explicative removed)!!!" and set a filter to reply with this template everytime he/she writes you!

### Post by StuartN on 2009-11-09
> **mjp29 said:**
> I'd like the email to even not show up at all in my inbox.

Either copy the email address and then Tools->Message Filters, or right-click on the email address in the preview and Create Filter From Message. Give the filter a name and set the action to Delete - if you want to not even be aware of the message, then set the first action to Mark as Read and then Delete.

You can subsequently add other email addresses to the filter, to create a blacklist. You can have as many filters as you like, and move the messages to screening folders if you might wish to preview before deleting.
