### Post by Monster_user on 2006-05-09
That gray bar on the right reminds me of an issue I had with Wine.

Everytime Wine would resize my desktop, and then resize it back, it would leave that line behind the Kicker. It is not an added bar, but a spot that is not covered by the desktop itself. "Empty Space". A glitch.

Everytime I collapsed the kicker, it would show up.

I fixed it by setting all Kicker panels to Hide automatically, and then resized the desktop using 'krandrtray'. It is a program that allows you to quickly resize the desktop, when you right-click the icon in the system tray (next to the clock).

Kmenu -> Run -> krandrtray
Alt-F2 -> krandrtray

That usually cleared it up.
