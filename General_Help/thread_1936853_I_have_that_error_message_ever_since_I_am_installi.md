### Post by cortman on 2012-03-06
Simple enough fix.
Open a terminal and enter

```
gksu gedit /etc/apt/sources.list
```

This opens the sources.list file in gedit. Sources.list tells the computer the repositories it can get packages from. In your case, a couple of the repositories are no longer functioning, and so are returning errors. Delete the lines for the "Failed to fetch" repositories, save the file, and run

```
sudo apt-get update
```

To confirm that it's fixed. Good luck!
