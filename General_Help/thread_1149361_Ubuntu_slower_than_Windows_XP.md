### Post by b@sh_n3rd on 2009-05-18
I've used ubuntu for several months now and am currently on 3 Jaunty systems. It *is* true that on *some* PC's, Ubuntu is slow...But, I discovered that the primary reason for it to *be* slow is the lack of proper video drivers. This is the main point of drag on computer systems running Linux... For example; I'm using Jaunty on a system with an Intel Pentium 4 2.4GHz processor, 512MiB RAM and an intel 82845G integrated video adapter with 64MiB of VRAM. As many of us know, the Intel xorg driver in Jaunty has a problem and I had to use this PC with the vesa driver which was quite slow. However, as soon as I fixed the problem by following the "Intel performance fix in Jaunty" HOWTO thread, my idle CPU usage dropped to 5-18% Moreover, this PC runs much faster than when it did on XP...
