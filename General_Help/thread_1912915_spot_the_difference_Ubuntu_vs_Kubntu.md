### Post by snowpine on 2012-01-21
KDE and Gnome are simply different desktop environments or "skins." You should choose whichever one fits your workflow for the way you intereact with your computer. (Or, install them both, and switch between them each time you log in, depending on your mood that day.)

If you want to compare the exact differeneces in pre-installed default applications (but you can install whatever apps you like, post-install):

[http://packages.ubuntu.com/oneiric/ubuntu-desktop](http://packages.ubuntu.com/oneiric/ubuntu-desktop)
[http://packages.ubuntu.com/oneiric/kubuntu-desktop](http://packages.ubuntu.com/oneiric/kubuntu-desktop)

Is that the info you're looking for?

Your applications should run equally well in either desktop since the "core" system is the same underneath.
