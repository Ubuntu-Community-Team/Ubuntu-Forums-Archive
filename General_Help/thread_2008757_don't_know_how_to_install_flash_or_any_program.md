In general, you don't really want to be "downloading to your hard drive" and then installing files manually. The Ubuntu repos are very well stocked, and you really ought to be using the package system, via the Ubuntu Software Center, or Synaptic, or apt-get from the command line.

This way you are assured that all dependencies are met, and you get a version tuned for Ubuntu.

For Flash, start Ubuntu Software Center, type Flash in the search box, and there right at the top is Adobe Flash Plugin.
