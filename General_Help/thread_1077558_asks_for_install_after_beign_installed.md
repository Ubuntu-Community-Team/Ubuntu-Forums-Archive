### Post by hansdown on 2009-02-28
You must have an installation program instead of a bootable one.

Have you disabled boot booster in the bios?

(NOTE: If the "Please select boot device" menu does not show up, you may have to disable Boot Booster in the BIOS.  To do this, boot your Eee PC and immediately press and hold the F2 key to get the BIOS setup menu to display.  From there, select the Boot tab and disable Boot Booster.  Save your BIOS changes and, when the boot sequence resumes, press/hold the Esc key to get the "Please select boot device" menu to display).
