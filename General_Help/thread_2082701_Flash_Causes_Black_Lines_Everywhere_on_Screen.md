### Post by COMECON on 2012-11-10
I assume you installed the restricted flash plugin, so...
You could try installing Chrome (**not Chromium!**), because it ships with another flash plugin. If, after installing, you still have the problem, remove the default flash plugin searching "Flash" at the Software Center and removing it.
I hope it helps you!
EDIT: I saw you have Chrome or Chromium installed. If it's Chromium, just try installing Chrome (download the .deb at its download page and just double-click it when it's downloaded). It shouldn't be necessary to uninstall Chromium, but if you still have problems and followed all the steps. try uninstalling it... Also, you could use any other browser (Firefox, Chromium...) if installing Chrome works for you; but I think you should uninstall the default flash player because it may override chrome's plugin.
