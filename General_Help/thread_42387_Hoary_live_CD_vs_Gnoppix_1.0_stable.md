### Post by intangible on 2005-06-17
They are both the same basic idea, but two different distributions. The Ubuntu live cd is more for sampling what a full Ubuntu installation would be like, while also giving you the ability to do whatever you want without having to install it. Gnoppix is a little more versatile in that there are multiple versions of it cutomized to do specific tasks.  it's a matter of taste as to which one you use.
