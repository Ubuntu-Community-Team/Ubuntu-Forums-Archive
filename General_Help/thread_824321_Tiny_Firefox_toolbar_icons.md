### Post by cnekmp on 2008-06-10
I think that there is a icon processing problem in your Firefox. Actually you should try to make changes in toolbar:
1. View -> Toolbars -> Customize...
2. In a new window uncheck "Use small icons" , and you should see changes.

if it was unsuccessful try to reinstall your Firefox.
