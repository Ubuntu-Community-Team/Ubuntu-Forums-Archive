### Post by WaffleCode on 2009-03-03
Installing Xubuntu: *sudo apt-get install xubuntu-desktop*

Go to Synaptic Package Manager (System -> Administration) and try removing packages from there.

AbiWord takes up less space than OpenOffice/KOffice. Don't uninstall it if your trying not to use space.

You can try searching for more lightweight equivalents of various applications. Instead of using Evolution/Thunderbird, use Claws. It's better, anyway.

You can always install the applications yourself, it doesn't have to be on Xubuntu....

### Post by 3rdalbum on 2009-03-04
If Synaptic warns you that it will remove the "xubuntu-desktop" package when you try to remove Abiword, then let it. The "xubuntu-desktop" and "ubuntu-desktop" packages are only there to make it easy to install the base system, and can be safely removed without breaking anything.

### Post by Shawn K on 2009-03-04
> **WaffleCode said:**
> You can try searching for more lightweight equivalents of various applications. Instead of using Evolution/Thunderbird, use Claws. It's better, anyway.

The only snag with a statement like that is that it's purely subjective unless you can provide quantifiable data.  On what do you base that opinion?

Personally, one of the main reasons that I like Evolution over just about anything else is that it's not as top-heavy as Microsoft Outlook, but it still has the calendar/memo/contacts/tasks functions that Outlook has, and I can sync it to my Palm.  I haven't seen another lightweight email client that still has those functions - Claws included - so to me Claws isn't automatically "better".

I'm not trying to derail this into a discussion on email clients.  I'm just saying that it might be a good idea to find out what features the guy needs before broadly proclaiming that something else is "better".
