### Post by ebsbox on 2008-01-28
I just wanted to post that Dev C++ works great under wine if anyone searches for info related to it.

When using Dev C++, compiling and building is just slightly different then in actual windows.

Compile and Build your programs as normal, then open a Terminal window, navigate to the directory where your program is saved, and the executable should be there.

Happy Programming!
