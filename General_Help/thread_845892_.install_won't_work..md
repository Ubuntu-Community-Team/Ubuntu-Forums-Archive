### Post by iaculallad on 2008-07-01
That's the common error when you try to execute a script which requires an interpreter/shell that does not exist. You need to install CSH shell in order for that to execute.

If you open the install file, you will see the line of text below:

> #!/bin/csh

W/c says that its for a CSH shell.
