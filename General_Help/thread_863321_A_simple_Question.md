### Post by danwood76 on 2008-07-18
Video editing and DVD creation are quite good under linux.
Although you generally have to use several tools to suit your needs.

Avidemux is very good for converting, chopping, adding videos together and you generally need to convert videos to mpeg 2 in order to burn them.
When I went from windows to linux I found the video quality from the conversion process in avidemux to be far better than that of the Ulead software I used.

Theres a nice piece of software for the final DVD authoring called Qdvdauthor which is very easy to use GUI for a set of command line progs.

-- edit --

The best thing to do if you are thinking of switching is do a dual boot with your current system and see if you like it.
Although obviously you will need to learn quite a bit of the OS before you get a real good impression.

### Post by jimv on 2008-07-18
WINE provides some compatibility for some Windows software to run in Linux.  A lot of programs will run under it, a lot of programs won't.  It's a ongoing project.  You can read more at winehq.org.

Also, I just did some research into the Quickbooks question, and from what I can tell the consensus is that there is no alternative to Quickbooks..at least nothing that comes close to being as feature rich.  It doesn't appear to work in WINE either.
