I just created a dual boot on a Toshiba nb505 netbook.
The odd issue is that when I launch Ubuntu it does not ask for the password. I checked the settings in User and Groups and it seems correct.

### Post by Krytarik on 2011-03-13
Check in "System -> Administration -> Login Screen" if the option "Log in as ... automatically" is chosen. If so, simply untick it.
