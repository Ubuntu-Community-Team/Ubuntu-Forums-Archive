> **salcykhan said:**
> You Can have all window program running in Ubuntu, look form WINE. this a program that will all window program in ubuntu. even yahoo massanger

No. Not all Windows programs work with Wine. Not even 20% of all Windows programs work with Wine.

My suggestion would be to use Meebo. It's a web-based messenger similar to Pidgin or Empathy, but it can do audio/video chat using an external service called Tokbox. When you start an audio/video chat with someone using Yahoo, the other person will receive a link to click; when they do, their web browser will open at Tokbox and the videoconferencing will start.
