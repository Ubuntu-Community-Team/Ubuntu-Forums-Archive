### Post by ranch hand on 2010-05-10
The link given to you by louieb should be read.  If you want a less in depth intro to Grub2  use the link in my sig.

If you would read your /boot/grub/grub.cfg file you would see, at the top, do not edit this file.  That is because it is just damned silly.

That is a generated file.  It is generated by the scripts in /etc/grub.d and /etc/default/grub.  If you want to change the grub.cfg file you make sure that those are correct and run update-grub and you should also run
```

sudo grub-mkconfig

```
which will print out the generated grub.cfg file so you can see if it is right.

To rescue this from the live CD, which you will be doing if you do do a little studying, use this;

[https://wiki.ubuntu.com/Grub2#Recover%20Grub%202%20via%20LiveCD](https://wiki.ubuntu.com/Grub2#Recover%20Grub%202%20via%20LiveCD)
