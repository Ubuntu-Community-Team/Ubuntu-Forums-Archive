### Post by SnakeHips on 2007-12-13
I had this irritating problem with the keys repeating madly at any given time ,completely random. I got around it by switchin off the "repeat keys" in system/preferences/keyboard. That worked except it crerated a new problem ,whenever it fancied the system would not allow any input from the keyboard at all...so back to the drawing board. Only workaround was to reboot every five minutes or so grrrrrr

Anyway this is how i fixed the problem ,it might help you..........the receiver for the wireless keyboard *was* plugged into a USB 4 port mini-hub ,i took it out and plugged it directly into a free USB port on the motherboard and.....hey presto.....no more probs........well ,so far anyway (four days without error)
