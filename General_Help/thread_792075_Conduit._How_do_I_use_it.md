### Post by Wolki on 2008-05-12
> **Roasted said:**
> 
How in the hell do you use this program? I understand dragging files and folders over, but... where do they sync to? I have a spare hard drive in my system... I know what I WANT to do, I just don't know how to do it with a GUI like this.

Drag the Folder DataProvider to the right. Drag another Folder into the same group. Right-click both, select configure item and configure their names and folders. Select Synchronize all from the menu. That worked for me in a test run.

That said, if you just want to backup some files to another hard disk, rsync or backup software might be better, conduit is better for synchronizing between several computers and/or web services, especially for synching in both directions.
