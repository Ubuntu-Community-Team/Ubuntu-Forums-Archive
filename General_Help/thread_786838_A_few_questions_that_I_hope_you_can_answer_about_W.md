### Post by pytheas22 on 2008-05-08
> Can I install Wubi and therefore Ubuntu on my D: drive? I don't have enough space on my C: drive for it, that's why I'm asking.

Sure, when you run the wubi installer there's a menu to select which drive to use for the install.

> Wubi keeps Windows, doesn't it? And it doesn't mess up my files on Windows? And the installation runs smoothly (on most computers), and no partitioning is required?

Short of something really weird happening, there should be no problems, and Windows won't be touched.  Some people do have some issues running wubi to install (it's still beta), but if you run in to trouble, search the forums and you should be able to figure out a solution.  And no, there is no partitioning required.  Of course, you should always be backing up your data just in case something goes wrong, whether during the wubi install or normal operation of Windows, Linux or anything else.

> And, one final quesion: Does it matter that when I set the password for Ubuntu, I changed the username too?

I'm not sure exactly what you mean.  If you mean that you changed the user name while installing Ubuntu from whatever it suggested (it suggests a user name based on the "real name" that you enter), then no, it shouldn't matter.  Otherwise, please clarify.
