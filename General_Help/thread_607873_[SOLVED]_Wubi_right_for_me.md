### Post by gotee12 on 2007-11-10
I'm using Wubi's 7.10 install right now and I'm loving it. I've run 7.10 in Virtual Box for Windows but some limitations with VB's video drivers won't allow compiz to function properly. Wubi is very cool in that you don't have to goober your Windows partition (sorry, I can't be without my Windows!!!) and still run a fully functional Ubuntu OS. Like the other poster said, the new 7.10 Wubi is still in development and is kinda hit/miss for some folks, but it does work for me. [Here's the link]("http://wubi-installer.org/devel/minefield/") to the downloads. I used the latest revision.

The only problem I have, is that once I boot into Ubuntu, I have absolutely no graphical display from when GRUB starts until the the logon manager starts. The other 7.10's I've played with (at work and on VB) at least show the Ubuntu splash to let you know it's booting properly. Ultimately though, it's not a big deal and it is still in development. Just be patient if you experience what I do. You'll eventually get something to look at!
