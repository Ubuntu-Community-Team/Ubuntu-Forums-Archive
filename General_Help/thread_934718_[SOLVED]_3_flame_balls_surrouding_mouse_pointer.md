### Post by Genius314 on 2008-09-30
You need to enable Compiz. Then you need the "Magnifier" and "Show Mouse" plugin enabled. (I'm not sure if the version of Compiz included in 8.04 has those plugins. You may have to install them.)



To enable Compiz:

Go to System>Preferences>Appearance, go to the Visual Effects tab, and select "Normal" or "Extra."

Now install CCSM:
> sudo apt-get install compizconfig-settings-manager

Now go to System>Preferences>CompizConfig Settings Manager, and find the plugins "Magnifier" and "Show Mouse." Check the boxes next to each one.

You can now use the show mouse by pressing Super+K, and the magnifier by pressing Super+M.

Hope that helps! :D
