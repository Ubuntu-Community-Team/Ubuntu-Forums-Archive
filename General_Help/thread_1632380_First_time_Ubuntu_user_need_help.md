### Post by choochoousmc on 2010-11-27
what deleted files are you wanting?
If you"wiped" the hard drive, probably S.O.L.
If you just installed over windows after just deleting windows.
You might get bits and pieces of files that Ubuntu didn't write over.
Are you wanting to return to windows? Then you will need a set of windows restore
disks, that you should of made beforehand.
Most computer manufacturers no long supply restore disks with the computer. But the ISO image is on the hard drive. Normally 2-4 dvds or upto ten cd's to create.

Or do you want to recover files in Ubuntu that you accidentally deleted?
Then yes there are some.
In Ubuntu  click on applications in the upper menu bar. Scroll down to software center.
Click it let it load. In the search bar there, type in file recovery. Then scroll through the list and find one you think will work. Install it and try it. You may need to try a couple different ones.
