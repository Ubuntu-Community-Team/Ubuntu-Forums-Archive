### Post by x1a4 on 2007-09-30
You probably just changed window button settings.  Me, I use the middle mouse button to maximize vertically, right button to maximize horizontally and left mouse button to maximize in both directions.  Run whatever you use to configure the window manager (in your case it's compiz) and change your window's button settings.

### Post by qweac on 2007-09-30
I have found a solution to my problem! :)

When I start ubuntu with the cabel to my TV connected the screen maximizes to fit the TV. When I start ubuntu with the cabel unplugged, windows maximize to fit the laptop screen. So I just have to plug in/plug out the cabel and ctrl+alt+backspace when I want to switch between using my pc for watching movies on the TV and when I use it to other things! :)
