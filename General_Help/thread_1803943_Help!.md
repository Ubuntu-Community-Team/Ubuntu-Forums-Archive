### Post by Bucky Ball on 2011-07-14
Welcome back. Avoid installing from Wubi, it can be a train wreck. It is primarily intended to try out Ubuntu to see if you like, not to install. Please use descriptive thread titles as you will increase you chances of getting 'help!'.

I would advise you download the Desktop ISO of the release of your choice, boot from it and choose the option 'Try Ubuntu'. If all goes well, proceed to install using the icon on the desktop.

If you want to dual-boot, choose manual partitioning and set up the Ubuntu partitions there and don't use or format the partition your existing OS is on.

Good luck. ;)

[http://www.ubuntu.com/download/ubuntu/download](http://www.ubuntu.com/download/ubuntu/download)

64bit for dual-core. 10.04 LTS most stable with longest support.
