### Post by /////// on 2008-08-03
If what tamoneya says doesn't work you can always try to re-install wine. Just go to synaptic find wine go to uninstall. Then go to your home drive, enable show hidden files and delete the .wine directory. Then re-install wine.

### Post by sayakb on 2008-08-03
Removing some applications using Uninstall Wine Software often doesn't remove their menu entries. Goto System->Preferences->Main Menu to remove them manually.
