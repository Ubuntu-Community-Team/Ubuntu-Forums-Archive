### Post by coolamon_98 on 2007-03-17
There's a feature you need to enable for the mail. 
Settings > Configure Kontact...
Select Mail > Appearance in left pane
Select System Tray tab
Mark Enable system tray icon
Hit Apply

Now when you receive a mail, you'll see it in the system tray marking the number of unread messages.
