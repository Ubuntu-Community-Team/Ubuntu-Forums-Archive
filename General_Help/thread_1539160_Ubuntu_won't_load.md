### Post by ronnielsen1 on 2010-07-26
Sounds like you need to change the bios settings in order to boot from cdrom. This is accessed differently on different boxes. Try pressing f2 or f10 or del  maybe f1,  press these keys when powering up your system.
> **BIOS is not set to boot from CD or DVD drive**

 Some  computers are set to boot directly from the hard drive.  This should be  as simple as entering the BIOS, enable booting from the CD-ROM drive,  and making sure that the CD-ROM is before the hard drive in the boot  order. 
The  most common way to enter the BIOS is to press the DELETE key when the  computer is first booted(this seems to be becoming standard).  On other  systems it could be a different key, or combination of keys like ESC,  F1, F2, F10, F12, Ctrl-Esc, Alt-Esc, Ctrl-Alt-Esc, Ctrl-Alt-Enter, Ins  or even others.  You might have to press, press and hold, or press  multiple times. The best way to find out the details of that is to look  in the users manual or search the manufactures website. A few different [Bios screens]("http://www.hiren.info/pages/bios-boot-cdrom"). 
[https://help.ubuntu.com/community/BootFromCD](https://help.ubuntu.com/community/BootFromCD)
