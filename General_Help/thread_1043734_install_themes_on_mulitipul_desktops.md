### Post by mcduck on 2009-01-19
Are you talking about installing the theme for all users, as opposed to installing for current user only?

In that case just extract theme files into /usr/share/themes. And for icons, /usr/share/icons.

---

### Post by shrekfx on 2009-01-19
does that install them as well?

---

### Post by mcduck on 2009-01-19
Extracting themes to themes directory is all there is to install a theme.

As soon as the files are in /usr/share/themes the theme is available for all users on that computer.
