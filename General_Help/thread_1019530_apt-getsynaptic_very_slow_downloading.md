### Post by redilyn on 2008-12-23
If its just the download speed which is slow try choosing another server.

System -> Administration -> Software Sources

Where it says download from, Click on the drop down list and click other.

Then click the button that says Select best server. Wait for the test to finish. Then click ok.

That should help some.
