### Post by Dennis N on 2016-09-04
The easiest tool for this purpose is to use the program called **audio recorder**. It monitors your sound card, and will record what is playing. This program is available from a ppa:

[https://launchpad.net/~audio-recorder/+archive/ubuntu/ppa](https://launchpad.net/~audio-recorder/+archive/ubuntu/ppa)

I assume you know how to add a ppa to your software sources and get it installed. If not, ask a follow-up question.
