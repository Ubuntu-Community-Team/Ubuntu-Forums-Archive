### Post by BinaryFeast on 2009-08-17
From your description this doesn't seem to have anything to do with what os you are running. More likely the webpage is badly coded and can only run on internet explorer. You can (as mentioned above) install Internet Explorer through Wine (or even without Wine) and see if the problem persists (who knows, it might even work with Opera or Konqueror). However, if the website simply checks what browser you have and then blocks you for using firefox or running Linux, then this addon may help you: [https://addons.mozilla.org/en-US/firefox/addon/59](https://addons.mozilla.org/en-US/firefox/addon/59). With it you can spoof the webpage with regards to both browser and OS.
