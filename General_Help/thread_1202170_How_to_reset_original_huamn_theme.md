### Post by Tamlynmac on 2009-07-01
In your menu select System/Administration/Login Window

When the "Login Window Preferences" window opens select the "Local" tab and check the box next to the desired login theme. I believe the default login screen is "Human List"

Good Luck and I hope this helps.
