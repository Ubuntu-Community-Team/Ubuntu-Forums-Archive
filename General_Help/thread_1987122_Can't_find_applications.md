### Post by papibe on 2012-05-25
Hi linux12. Welcome to the forums!

You are actually installing the application from the Software Center, so you won't find a zip, rar, or exe in any place.

Instead, the application will be available for use immediately. Use the dashboard (tap the super key - or windows key) to search and launch it.

This is a youtube [video]("http://www.youtube.com/watch?v=fAcdFd0MmH0") (kind of old) but shows you the basics of the dashboard.

I hope that helps. Have fun and come here often!
