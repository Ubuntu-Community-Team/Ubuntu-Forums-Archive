### Post by Michiba on 2006-04-13
Thought I'd answer myself, 

I Changed my Multimedia System Selector from ALSA to ESD on both Default sink output and Default source imput. From the desk top click System\Preferences\Multimedia System Selector.

Hope it may help some one else

Michiba
