### Post by PMO6022 on 2005-10-06
Lynx is available in the main repository.  Another alternative is links - available in universe.  From the description: > Links is a lynx-like character mode browser. It includes support for rendering tables and frames, features background downloads, can display colors and has many other features.  I'm sure there are other text based browsers, these are the only two I've used.
