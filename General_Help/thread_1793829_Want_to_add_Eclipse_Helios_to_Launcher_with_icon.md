### Post by Azdour on 2011-07-05
Hi,

Usually with launchers to customise the icon:

[LIST]
[*]Right click on the Launcher, choose Properties
[*]On the basic tab that appears left click on the icon image and that should allow you to pick another image
[/LIST]

On mine in the Select Custom Icon there is a pixmaps directory in the left hand list that contains the basic icon images.
