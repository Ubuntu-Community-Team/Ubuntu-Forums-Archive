[SOLVED] - So bacially i want to put the main menu onto the bottom panel, but i cant find anythink to let me do this. Anyone know how or even if this is possible???

[ANSWER]

So if like me you came from windows being used to the menu being in the bottom task bar then here is how you can put it there.

**Right Click/Add New Item/**

Then Just Select **Xfce Menu**, then you can remove the one at the top by

**Right Click/Remove Item**




Hope you found it useful.
