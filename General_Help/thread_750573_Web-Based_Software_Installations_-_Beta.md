FELLOW UBUNTU USERS!

Greetings I have had the pleasure to work with a new linux software project that may be of interest to all Ubuntu users. 

[TuxSoftware.com]("http://www.tuxsoftware.com") is a web-based Linux software installation site for Ubuntu, Debian, and derived distributions such as gOS or DreamLinux. On the site you will find hundreds of the most popular Linux software applications available for installation directly from your web browser - with links, reviews, screen shots, and additional information for each. Our goal is to provide one place where you can find, research, and install the software you need as quickly and as easily as possible.

Ubuntu users may use the apturl firefox extension or our own lsi utility to safely and securely install software from your browser. Our utility is based on mintInstall from the LinuxMint distro if you are familiar with that program. The site is similar to Linspire's CNR.com only we are only focusing on popular, well documented applications rather than every package in synaptic.

Everything is still in beta and in need of testing but moving along rapidly. Check it out @ [http://www.tuxsoftware.com]("http://www.tuxsoftware.com")

We welcome all feedback, comments, suggestions, or critique. Please be kind though as I said we are still working on it. 

Thanks,
Kevin
