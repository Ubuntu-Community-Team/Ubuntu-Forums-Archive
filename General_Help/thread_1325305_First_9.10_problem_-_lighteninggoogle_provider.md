### Post by Warthaug on 2009-11-13
In past versions of ubuntu I have used the thunderbird plugins "lightening" and "google provider" from the repositories to access my calendar on google calendars.  This always worked without a flaw.

So after installing ubuntu 9.10 I repeated what I've always done - installed thunderbird, provider and lightening from the repositories.  Instillation went fine, access to email is normal, and lightening works almost perfectly.

The almost is because alarms no longer work - if I enter them in google calendar they simply don't appear.  If I enter them in lightening the alarm icon appears, but disappears as soon as lightening refreshes.  If the alarm occurs before a refresh I'll get the pop-up; otherwise that never occurs.

I searched the mozilla forums and couldn't find any info; I was wondering if anyone here has experienced this, and more importantly, knows how to fix it,

thanx

Bryan

### Post by Warthaug on 2009-11-17
I've solved it.  For whatever reason, the version of provider and lightening in the ubuntu repositories seem to have this flaw.  Reinstalling, using the official 64bit versions in the mozilla add-ons section resolved the issue.
