### Post by kanikilu on 2009-04-24
Sorry, I don't have any idea about the status/future of Giver, but surely there has to be an easier way to share files...

You are obviously using 2 linux boxes (since Giver is only available for linux), so, you could go all out with nfs, rsync, etc, but that has a bit of a learning curve. In the interim, why not just setup SSH and transfer files in any number you want and any way you want, ranging from the command line (scp), built-in GUI (nautilus), or sftp application (gftp, filezilla, etc.)?
