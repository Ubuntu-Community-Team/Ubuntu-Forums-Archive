### Post by mac9416 on 2009-11-11
Hey, cichlasoma,

You'll want to check out [Keryx]("http://keryxproject.org"). It is a cross-platform, portable .deb package manager designed for slow-connection and offline Debian/Ubuntu users. 

Check out the tutorial to get an idea how it works: [http://crashsystems.net/2009/01/keryx-tutorial/](http://crashsystems.net/2009/01/keryx-tutorial/)

If you have any trouble, let me know!
