### Post by corevette on 2006-12-09

There is another way, but this is with a windows xp cd.  if you do get a hold of one, load it up and in the command prompt i think you type: fixmbr, and then fixboot.  correct me if wrong

### Post by Malakia on 2006-12-09
If you are trying to reinstall Linux without Windows than just go into the BIOS settings and make sure that the Cd-rom is the first one to boot, if that is the way you are installing Linux. That way it will boot off of the disk instead of the hard-drive. If you are trying to reinstall Linux with Windows than the technique mentioned by corevette should work.
