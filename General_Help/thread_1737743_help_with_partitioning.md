Hi.

You won't be able to use gparted for that.

The steps is a little different.

First you will need to burn a live cd

to boot from. Then make a copy of the wubi disk (steps explained below in red)

Then follow this guide(look at post #2) for

the steps to resize your disk:

[http://ubuntuforums.org/showthread.php?t=1625371](http://ubuntuforums.org/showthread.php?t=1625371)

[COLOR="Red"]When in windows, copy the file

"root.disk" in the "C:\ubuntu" directory an external drive.

Then, if something goes wrong with the resizing, you can 

just put it back there.[/COLOR]

Let me know if you have any questions.

Good luck.
