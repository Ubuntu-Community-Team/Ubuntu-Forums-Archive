### Post by foxmulder881 on 2007-09-29
I use Pidgin 2.2.0 and Ubuntu 7.04 Feisty Fawn.

Whenever I set my status to 'Away', I usually type in a quick message to let people know when I'll be back.

Is there any way to clear out the old away messages, as each time I create a new message, it adds to my away message lists.
Now I have a huge list of old away messages that I want to clear but have no idea how to.

Any help appreciated.

### Post by defrex on 2007-09-29
If I'm assuming correctly, you're using the drop-down at the bottom of the buddy list and hitting "new" to customize your status? If so, just go to 'saved' right below it and you will get a window listing all your custom statuses. From there you can delete them.
