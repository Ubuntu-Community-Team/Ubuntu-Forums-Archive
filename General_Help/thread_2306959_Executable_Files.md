### Post by sudodus on 2015-12-20
Some Windows exe files work with wine, but many Windows programs really need Windows to work.

Sometimes you can find linux programs that can replace your old favourite Windows program, but in some cases it is difficult. You must realize that linux is not Windows for free, it is a completely different kind of operating system.

-o-

It should be possible to find the license number of your Windows 8 for your computer, and for a small amount of money you should be able to buy an iso file so that you can create a boot DVD or USB drive and install Windows again. (If you have the license number you need not pay for the whole Windows system again.) But it may be different in different countries and for different computer brand names and models.
