### Post by sabersong on 2008-10-13
Hello.

I need to install the build-essential package, but when I use aptitude or apt-get it tells me to insert the CD.  The problem is I've misplaced the CD and can't download another (no high-speed in my area and dial-up download would take way too long.

Can anyone tell me how to get build-essential without the CD?

I'm using Gutsy Gibbon.

Thanks for the help.

---

### Post by Ryadt on 2008-10-13
Go in System > Administration > Software Sources and uncheck the cdrom option. Then retry the install.

---

### Post by Sam on 2008-10-13
System / Administration / Software Sources

In "Third Party Software" tab, uncheck the line containing cdrom.
