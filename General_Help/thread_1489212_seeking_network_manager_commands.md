### Post by cariboo on 2010-05-21
Do you want to restart network manager, or just networking? to restart networking open a terminal and type:

```
sudo service networking restart
```

you can also use the start and stop commands.

You can also right click the network manager icon in the top panel, and just disable and enable networking.
