### Post by Rubi1200 on 2011-09-11
Hi and welcome to the forums :-)

Basically, no, it should not be a problem.

In fact, it is far more likely that Windows issues can affect Wubi since it is installed as a single large file within Windows.

Installing on D should not be a problem.

For more information on Wubi, see here:
[https://wiki.ubuntu.com/WubiGuide](https://wiki.ubuntu.com/WubiGuide)

This blog by forum member bcbc is also a useful resource:
[http://ubuntu-with-wubi.blogspot.com/](http://ubuntu-with-wubi.blogspot.com/)
