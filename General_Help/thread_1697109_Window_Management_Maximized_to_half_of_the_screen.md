### Post by ddnev45 on 2011-02-28
They may already be available; if not the keyboard shortcuts you need are pretty straight forward to create.

System > Preferences > Keyboard Shortcuts

will bring up your existing shortcuts (on my Debian system, I hope they are in the same locality on Ubuntu) so you can check if they already exist.

I change mine so yours aren't going to be Alt+F7 etc....

### Post by hojjat on 2011-05-06
I upgraded to Ubuntu Natty Narwhal (11.04) in the meantime. Therefore, I started using Unity as my desktop (instead of Gnome), which has the functionality I wanted in it already:

[http://askubuntu.com/questions/28086/unity-keyboard-mouse-shortcuts](http://askubuntu.com/questions/28086/unity-keyboard-mouse-shortcuts)

CTRL+ALT+4 maximizes the window and spans it on the left half of the screen
CTRL+ALT+6 similar, but to the right of the screen

and there are many more useful keyboard shortcuts.
