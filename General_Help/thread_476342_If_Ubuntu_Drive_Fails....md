### Post by Cappy on 2007-06-17
You would have to boot the windows live cd or a windows floppy and do a "fixmbr"  and "fixboot" to gain access to windows. If it failed you would either need to make sure the hard drive your computer is booting off is the windows one (in the BIOS) or take the failed hard disk out.

I recommend a backup program so you don't lose your /home/username/ directory information:
[http://www.getdeb.net/app.php?name=SBackup](http://www.getdeb.net/app.php?name=SBackup)
