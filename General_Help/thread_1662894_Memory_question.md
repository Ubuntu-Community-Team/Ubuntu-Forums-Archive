### Post by Frogs Hair on 2011-01-08
Some websites increase memory use depending what you do there. Running multiple tabs for long periods and not cleaning the cache will consume a lot of memory .

Profile data such as saved passwords for multiple sites also contribute. 
Web browser memory use benchmarks are easy to find , but it seems they all have different results.
