### Post by coldraven on 2017-11-17
You would be better off if you shrank the Windows partition from within Windows. Some of the Windows files are unmovable and Windows will calculate the largest space that can be created without causing problems. After doing that boot up with the Ubuntu Live DVD and you should have enough space for an install.
