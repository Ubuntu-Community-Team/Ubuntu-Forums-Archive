### Post by Claus7 on 2012-07-15
Hello,

I had exactly the same problem and this happened while using firefox 13.

In order not to loose my tabs I did the following (while mouse in hand mode):

1) ctrl+n creates a new window
2) alt+F4 closes windows

in case you open new windows so as to search and see what is going on, do not hesitate to close them with the aforementioned '2' shortcut from your keyboard. If you close all the windows of ff, you will see your mouse back in order. If you reopen firefox, it will open all the windows you had opened. Close the ones you did not want and leave aside your precious one with your wanted tabs open. And back to normal.

Regards!
