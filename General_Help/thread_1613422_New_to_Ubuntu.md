### Post by gordintoronto on 2010-11-04
> **PitbullMean said:**
> Am I able to just make a partition on my HDD and have windows on one and make a new one for ubuntu? with out effecting the current structure of my windows platform.

Probably. I have seen a computer with four primary partitions, and that is the maximum, which makes life very difficult. Otherwise, you can: "Back to the C: drive, I right-clicked and selected Properties.  One of the "Properties" is Tools, and one of the Tools is defragmentation. I ran that, and it only took a few minutes on a brand-new hard drive.

"In Windows Control Panel, search for "disk," and one of the items is "Create and format hard disk partitions." I clicked on it, highlighted the C: drive and selected "Action," "All Tasks," "Shrink Volume."

Then you can install Ubuntu in the space which is freed up. You really have to pay attention, and select "advanced" or "manual" (I forget the exact name) during installation.

By the way, your hardware is far past what is required to run Ubuntu --  by a factor of more than two. It looks like a great video editing machine...

### Post by edm1 on 2010-11-05
That tutorial you have found is for dual booting Windows 7 and Windows XP/Vista. You want one for dual booting Windows and Ubuntu however you probably dont need one. If you are currently running windows and would just like to give Ubuntu a try I recommend using the [Windows Ubuntu installer (Wubi)]("http://www.ubuntu.com/desktop/get-ubuntu/windows-installer") which installs Ubuntu as though it is just an application within windows and so can be easily removed the same way you would remove any program on windows.

If you try Wubi and like it or would like to go for the more permanent option of installing Ubuntu alongside Windows. Just download the [Desktop Edition]("http://www.ubuntu.com/desktop/get-ubuntu/download"), either burn it to a CD or put it on a USB pen, insert the CD or USB pen, restart and follow the easy instructions.
