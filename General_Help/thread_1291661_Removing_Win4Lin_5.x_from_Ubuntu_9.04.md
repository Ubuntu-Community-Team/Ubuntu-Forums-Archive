### Post by BETATEST on 2009-10-14
If you installed Win4Lin 5.x from the .deb package from the Win4Lin website.  ADD/REMOVE on the Applications Menu will not show it.

From a Terminal Session, type in the command:
[B]
sudo dpkg -r win4linpro[/B]

Enter the root password when asked, and Win4Lin goes away.
