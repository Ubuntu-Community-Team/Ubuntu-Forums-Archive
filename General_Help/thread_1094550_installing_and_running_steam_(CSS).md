### Post by photon_man62 on 2009-03-13
Install WINE:

Open up a terminal.
Type in "sudo apt-get install wine" without the quotes.

Download and install WINE-Doors.
[http://wddb.wine-doors.org/system/files/wine-doors_0.1.3rc1_all.deb](http://wddb.wine-doors.org/system/files/wine-doors_0.1.3rc1_all.deb)

Run WINE-Doors and Steam should be in the list of available applications.

You can just copy and paste it into the terminal window by right-clicking in the terminal window and selecting paste.
