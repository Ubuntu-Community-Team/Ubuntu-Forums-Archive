### Post by shen-an-doah on 2007-05-24
There's also this:

[http://www.fs-driver.org/](http://www.fs-driver.org/)

It works better than explore2fs as you're able to access ext2/3 partitions like you would any other drive. Though Windows still doesn't recognise my external hard drive, but that could be because it's too big (the same way Windows won't recognise a FAT32 partition over 38GB).

### Post by reckless2k2 on 2007-05-24
[http://www.fs-driver.org/](http://www.fs-driver.org/) is the best way to get at your Linux drives as long as they are not reiser file system. 

As far as ext2/3 external hard drive being seen in windows, that is a whole other issue. If it is data in a FAT32 file system over XXGB, you are correct.
