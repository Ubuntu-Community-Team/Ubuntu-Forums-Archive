### Post by grahammechanical on 2020-11-08
The version of zoom-client for Linux Ubuntu at the Zoom web site is for Ubuntu 16.04. In the Ubuntu software store there is a snap version of zoom-client that is the latest version of zoom-client and because it is a snap app it runs fine on Ubuntu 20.04. I have not tested it on 20.10.

Because of the nature of the snap format I consider the snap app to be "safer" than the version from the Zoom web site. There are settings for zoom-client in Sound Settings. The Software store at the zoom-client location allows us to change certain permissions.

There are two areas that I consider vulnerable and I think that they would apply to any similar method of video conferencing. The telephone address of the meeting's host and the password. If these are not treated as private then anyone that has this information can join the meeting. Then, there is the question of where all that data that is going backwards and forwards over the internet is being stored. Whose servers? How secure? These questions apply to any use of the internet.

At least zoom-client now has end-to-end encryption. Or it claims to have it. At what level of encryption? How secure?
