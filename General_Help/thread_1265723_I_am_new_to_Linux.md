### Post by biebel on 2009-09-13
Love that you took adobe reader as example. All you need to do in order to read a pdf in ubuntu is doubleclick the file :)

> jason@ubuntu:~$ cd desktop
bash: cd: desktop: No such file or directory

folders are case sensitive in linux so that would have to be "cd Desktop".
Or to make things even easier "cd D" and press tab to autocomplete. If it doesn't autocomplete there are either no entries that begin with what you have already typed or there are several. If you press tab twice (semi-rapidly) it will give a list with all the possibilities.

### Post by scouser73 on 2009-09-14
1 Go to the [[COLOR="Red"]**Adobe Reader Website**[/COLOR]]("http://get.adobe.com/uk/reader/?promoid=BUIGO")
2 Click on **Different Language Or Operating System**
3 Click on the **Select An Operating System** drop down menu
4 Select **Linux - x86 (.deb)**
5 Click **Continue**
6 Select **Adobe Reader 9.1.2** & Click **Download Now**

That will start installing Adobe PDF Reader for you, once installed you can find it in Office.
