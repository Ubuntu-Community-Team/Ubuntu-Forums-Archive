### Post by ajgreeny on 2011-11-02
Are you trying to open attachments directly from thunderbird?  If that is the problem see how thunderbird is set to open the attachments from Edit -Preferences, Attachments tab.

I have attached the screenshot from my system (10.04) where I have replaced the default office suite, OOo with LO, but as you see the default to open .doc files is still OOo Writer, so for a while I had the same problem you are reporting, but with different file types.  So check in the attachments tab what is set to open those file types and change the default if necessary.
