### Post by P4man on 2009-09-12
You need the "Animation addons" (i think you have to install compiz-plugins-extra for that). Then in effect settings, go to Burn, set "randomly colored fire", check "fire smoke" and set direction to "up". Set "burn" as   closing animation in "animation".

### Post by P4man on 2009-09-12
perhaps you can give me a bit more info first :) ?
Do you have the "animations add-on" plugin ?

If so, enable it (click checkbox). Then click it to change its settings, go to burn, and set the things like I explained above.

Go back to "animations", chose "close animations", double click the first entry in that list (which has "type=normal") and set close effect to "burn". Do the same with "minimize animations" if you so wish.
