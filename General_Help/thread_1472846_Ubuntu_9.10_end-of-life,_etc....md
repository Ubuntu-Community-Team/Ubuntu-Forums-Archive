(1) what is the end-of-life for Ubuntu 9.10?.

All non-LTS releases are supported for 18 months.  So you have another year.
> 
(2) Is there a way to continue using Ubuntu 9.10 past it's end-of-lie and still receive security (and other) updates?.

Not that I know of.  You can keep using it, but you won't get updates.
> 
(3) Is there a way to avoid dependency/library/software incompatibilities if I choose to do 2#?.

Since there won't be any updates, there aren't any conflicts to worry about.
