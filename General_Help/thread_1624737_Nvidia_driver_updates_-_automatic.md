### Post by dino99 on 2010-11-18
if you want the latest drivers, you can add this ppa into your synaptic repo tab:

ppa:ubuntu-x-swat/x-updates

(from: [https://launchpad.net/~ubuntu-x-swat/+archive/x-updates](https://launchpad.net/~ubuntu-x-swat/+archive/x-updates) )

then update
