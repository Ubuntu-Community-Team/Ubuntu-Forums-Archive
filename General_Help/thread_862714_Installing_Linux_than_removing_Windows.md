### Post by jonian_g on 2008-07-17
To make a live usb (must be over 1GB) use this app:

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

Works on windows too. Still you have to download the .iso but don't have to burn it on a CD.

> I want to install Linux without CD's, so this would mean I would have to make a partition and make it dual boot with windows.

No need to do that if you don't want to keep windows. You can boot and install from the usb.

How to for Dual Boot:

[https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)
