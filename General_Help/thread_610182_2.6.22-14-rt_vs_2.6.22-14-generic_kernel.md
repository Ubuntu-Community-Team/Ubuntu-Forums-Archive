### Post by justin_guerin on 2007-11-11
The package 2.6.22-14-rt is patched for Real Time performance.  Here's the package information page from Ubuntu:
[http://packages.ubuntu.com/gutsy/base/linux-image-2.6.22-14-rt](http://packages.ubuntu.com/gutsy/base/linux-image-2.6.22-14-rt)

As it says, you likely don't want to install it directly, and it sounds like you didn't.  

If you want to know more about what makes a kernel real time, see [http://rt.wiki.kernel.org/index.php/Frequently_Asked_Questions](http://rt.wiki.kernel.org/index.php/Frequently_Asked_Questions).
