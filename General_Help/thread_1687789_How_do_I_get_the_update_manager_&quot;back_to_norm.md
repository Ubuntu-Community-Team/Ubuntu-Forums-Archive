### Post by Odense on 2011-02-14
I am running Ubuntu 10.10 on a Lenovo S10e netbook
I use an Samsung external DVD writer with it.

Today I connected the DVD writer and it had an Ubuntu 10.4 LTS CD inserted. When it started I was asked if I wanted to open the CD with the Ubuntu Software Center and I thought it would be interesting and said yes. I did however not install anything but now when I run the Update Manager and says "Check" it ask for the CD. Naturally I don't want that.

How do I get the Update Manager back to look at the software on the computer and not on the CD ?

---

### Post by Cheesehead on 2011-02-14
Ubuntu Software Center -> Software Sources -> Other Software
Uncheck the CD.

### Post by Odense on 2011-02-15
Thanks - problem solved :D
I suspected it was simple - and it was :p
