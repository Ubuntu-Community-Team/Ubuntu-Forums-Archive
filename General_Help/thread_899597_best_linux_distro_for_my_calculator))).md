### Post by sub2007 on 2008-08-24
Maybe one of the minimal distros, I really like Arch. It requires you to do a bit more work initially as on install it starts with just a command line so you build up the system as you want it and by only installing the essentials your footprint is much lower. The beginners guide in the Wiki will tell you everything you need to know. If you install it with Xfce or a Window Manager (like Fluxbox or Openbox) then that would allow you to have a light system and still have the latest software rather than using older repositories.
