### Post by Roasted on 2011-07-20
I'd use a lighter weight desktop environment. In my experience, Unity and KDE have been somewhat on par with each other. Unity on 11.04 on my netbook took up nearly 400mb of RAM on a fresh boot for all system processes. Same laptop, same 11.04, but Gnome Shell and it took nearly 160mb of RAM.

For what eye candy you get with Gnome Shell, it is extremely light on resources.

Likewise, LXDE and XFCE would be thinner to run as well. If you're after the absolute lightest possible, I'd stick with LXDE, with XFCE following behind, then Gnome Classic, Gnome Shell, Unity, and KDE.

Aside from system tweaks, I haven't done anything more. My netbooks are running 11.04, one with Unity, one with Gnome Shell, and I find the performance of each quite decent. The only difference is one has 1gb of RAM, the other has 2gb of RAM, but I've never had an issue with the amount of RAM in those systems.
