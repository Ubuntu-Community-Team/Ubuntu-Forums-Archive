### Post by kc1di on 2012-09-26
Hi olasoaga and welcome to Ubuntu

As far as I know you can. it' usually just a matter of downloading the file from mozilla and extracting it.  That being said there are some steps to take if you want to integrate it with Ubuntu.  If you want the very latest build of Firefox you can add the daily build PPA and upgrade every day , but warning this may be unstable!
here the PPA page:
[http://www.ubuntuupdates.org/ppa/ubuntu_mozilla_daily_builds]("http://www.ubuntuupdates.org/ppa/ubuntu_mozilla_daily_builds")

The current version available to Ubuntu 12.04 users is 15.0.1 which is new enough for me.  the version available via the daily build repository is 18Alpha 1. Please understand Alpha builds are not stable builds. 

If you just want the latest stable release in ubuntu 12.04 just keep your system up to date and you have the latest stable release of FF available. 

Good luck
