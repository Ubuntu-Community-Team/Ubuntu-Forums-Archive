### Post by MichaëlVD on 2006-01-24
A couple of things about search here :-)

I know you can use wikipedia in the search box in both Konqueror and Firefox. I was talking about a Wikipedia toolbar (see link in previous post), that helps you edit wikipedia articles. That extension is available for Firefox, but there's no equivalent for Konqueror. Of course, that's the case for most extensions.

The "web shortcuts" thing you're talking about in Konqueror is not the same thing as the search box. It allows you to use dozens of search engines from the location bar with the use of keywords.

Google will be the default one, so just type in your search term in the location bar and hit enter.

Other search engines need keywords, which are all described in the Konqueror web shortcuts settings window. For example, you could type in "ggn ubuntu" to find Google News about Ubuntu.

In fact, this feature means that you don't need to use the search box anymore.

You can do the same thing in Firefox by right clicking the search box on the Google News website, and clicking "add a keyword for this search", and give it a keyword such as ggn. However, you'll have to put these in a seperate Bookmarks folder, otherwise your bookmarks will be cluttered with search engines, because in Firefox, keyword search is some kind of bookmark. Also, you will have to do this for all your search engines, whereas Konqueror gives you a lot of choice by default.

### Post by Jucato on 2006-01-24
Oh, ok. I misinterpreted about that wikipedia toolbar. However, that Wikipedia extension doesn't work for Firefox 1.5 anymore. I'm thinking, however, that it might be possible to install that in Konqueror, since it seems that Mozilla, Firefox, and Konqueror share some plugins. 

Regarding the Wikipedia toolbars, we are talking about two different things, two features shared by both Firefox and Konqueror. One is the **dedicated Search Toolbar**, a small search bar usually located beside the address bar. Google is, as you said, the default. But you can also install other engines. In Firefox, this means going to the [https://addons.mozilla.org/search-engines.php]("https://addons.mozilla.org/search-engines.php") site. In Konqueror, you have to Click/Right Click "G" on the tool bar and check a few options.

The second feature is the **Web Shortcuts**, which you can type on the address bar itself. Firefox calls these "Keywords". This is where stuff like "gg [insert text]" or "wp [insert text]" comes in. Konqueror handles this through the Settings > Configure Konqueror > Web Shortcuts, while Firefox handles it the way you said it. In both these features, Konqueror beats Firefox (you don't have to download an engine, and less clutter in adding/using keywords).
