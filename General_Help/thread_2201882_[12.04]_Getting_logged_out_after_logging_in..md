### Post by sbnwl on 2014-01-26
It is not recommended to run bleachbit as root user. Did you do it as root user?
Looks like the file named .Xauthority is corrupted in your home directory.
Try to remove the file first.

Hit Alt+Ctrl+F1 at the login screen, put your username, then password.
Now issue the following command to remove that file:
```
rm -f ~/.Xauthority
```
Now hit Alt+Ctrl+F7 (or F8), you should be back to your login screen. Try to login now. Post back if troubles.
