### Post by Tamlynmac on 2011-11-03
Open your file browser and make a folder called "XnView" in your home directory.

Download this [tar.gz]("http://newsgroup.xnview.com/viewtopic.php?f=60&t=24056")

Open it in the Archive Manager (probably automatic after download)

Extract  the contents of the xnview download (archive manager) and put them in  your new folder labeled "XnView" in your home directory

From the  file browser find the "Bin" sub-folder in the xnview folder and double  click on the file marked "xnview" in that folder.

It will open a small initial configuration window. In that window menu under "tools" select "options".

When  the configuration window opens the most important thing to check is to  select "Launch Browser at Startup" by selecting  Browser in the left  side config window and selecting the square box on the right hand side  next to Launch Browser (it should turn a color when selected - probably  yellow). Also under the startup directory (just below the launch browser  switch) assign a specified folder to open when launched.

The  rest of the configuration is setup by your choice and much of it is  predefined. After your done close the XnView windows and once again  double click on the bin file "xnview". It should then open your xnview  browser window.

Remember, that you must manually add it to your  apps menu, by using the location of the bin file "xnview" in the command  to setup the launcher in your menu. In essence it the executable.

Good Luck & hope this helps.
