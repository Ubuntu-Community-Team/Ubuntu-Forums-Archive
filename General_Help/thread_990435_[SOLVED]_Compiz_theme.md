### Post by Izek on 2008-11-22
Emerald takes **.emerald** files, don't know why someone would tell you that emerald would take that file.

Beryl/Emerald Themes are what you want.

[http://www.gnome-look.org/index.php?xcontentmode=103](http://www.gnome-look.org/index.php?xcontentmode=103)

After you get an emerald theme installed, it should auto-replace with that theme. If not, you need to set up the Sessions Preferences to start up with emerald --replace.
