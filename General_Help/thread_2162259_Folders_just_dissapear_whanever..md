### Post by claracc on 2013-07-16
> **archangel2003 said:**
> My daughter still uses 10.04 LTS (Lucid Lynx).
She suggests I just keep using that version because it is solid for someone who is inexperienced.

This is a wrong recommendation since as you know, 10.04 has reached is end of life support.
It means that software repositories have been moved, so you cannot install software from them, the only way is to point your repositories to old-releases.ubuntu.com, see [http://askubuntu.com/questions/101479/are-existing-updates-available-after-end-of-support](http://askubuntu.com/questions/101479/are-existing-updates-available-after-end-of-support).


Other risk is that you are not going to receive any security update, so you are going to run an obsolete and insecure version.

I suggest you to install some of the supported releases: 12.04, 12.10, 13.04
