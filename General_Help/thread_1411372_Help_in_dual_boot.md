### Post by anand1298 on 2010-02-19
Hi,

I first installed windows XP(C:/) and ubuntu(D:/). Whenever i boot, it will show a boot screen asking me which OS to load. But then i reinstalled windows XP as i had a problem and in my boot screen it doesn't show me which OS to show; it boots directly to XP. But ubuntu is still installed in D:/. Is there a way to make the boot screen appear again without having to reinstall ubuntu, so that i can choose which OS to load? If so, how?

---

### Post by oshunluvr on 2010-02-19
When you re-installed windows, you wiped out grub.

Two ways to fix this:

Download and burn a copy of SuperGrubDisk and repair your grub.

Boot a linux livecd and fix grub manually.

Do a web search on how to do either of these.
