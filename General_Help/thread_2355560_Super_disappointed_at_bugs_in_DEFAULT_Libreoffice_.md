### Post by dragonfly41 on 2017-03-14
This suggestion does not use a GUI, but instead creates an HTML5 slideshow.

[http://lab.hakim.se/reveal-js/#/](http://lab.hakim.se/reveal-js/#/)

And you can use pandoc ... [http://pandoc.org/](http://pandoc.org/) ... to create a reveal.js (or other) slideshow in markdown.

So the pipeline is ...

markdown (use an editor with markdown plugins such as Atom)

pandoc  markdown file > HTML slideshow
