### Post by Thelasko on 2010-07-15
That really depends on the program.  Can you put it on the XP machine and it just won't run?  If so, try right clicking on it, go to properties, click the compatibility tab.  Select "run this program in compatibility mode for"  and then select "Windows 98."

You can try to make it run in Linux with Wine, but it will take [some more work to make it run]("http://ubuntuforums.org/showthread.php?t=885111").  Wine too has a compatibility mode for Windows 98.
