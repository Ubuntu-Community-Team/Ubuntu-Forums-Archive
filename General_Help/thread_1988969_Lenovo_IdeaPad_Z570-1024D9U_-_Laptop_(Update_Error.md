### Post by oldfred on 2012-05-28
It looks like it is trying to download fonts and flash plugin. Do you have a good Internet connection? What server are you connecting to. I usually select a "better" server but that is based on ping values not necessarily download speed.

System, Administration, Update Manager. Click on settings button on bottom left and in first tab - Ubuntu Software is a combo box on site used to update. Click on combo box and choose other. Then Select Best Server.
