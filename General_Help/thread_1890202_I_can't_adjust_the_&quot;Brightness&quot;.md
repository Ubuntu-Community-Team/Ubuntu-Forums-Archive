Good question :)

No, you should be able to adjust the brightness from within the live session without making any changes. All of those kinds of drivers would be located in the Kernel.  It is possible however that you will need to install the proprietary driver as stated in this guide:

[https://debianhelp.wordpress.com/2011/09/12/to-do-list-after-installing-ubuntu-11-10-aka-oneiric-ocelot/](https://debianhelp.wordpress.com/2011/09/12/to-do-list-after-installing-ubuntu-11-10-aka-oneiric-ocelot/)

whereby you would need to install Ubuntu 11.10 to your Hard Drive first.. before you could download and install any proprietary drivers for your system.

The guide will require you to download quite a few packages, so hopefully you have a nice hardwired connection to the Internet (recommended to save time)
