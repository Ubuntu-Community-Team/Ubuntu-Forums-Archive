### Post by DuckHook on 2016-07-30
Welcome to the forums, seacaptain2,

I fear you are may be putting the cart before the horse. WINE is not a magic wand and, frankly, is not the first step I would recommend for a user who is completely new to Ubuntu as you seem to be. It is finicky, arcane and often requires the sort of tweaking and config modifications that overwhelm newcomers and give Linux its reputation as an OS only for geeks.

A good place to start is by reading the link in my sig: ***Linux is Not Windows***

This will adjust your expectations to what Linux&#8213;and by extension, Ubuntu&#8213;actually is and can actually deliver.

Then I suggest that you ask on this forum for specific advice on specific usage cases citing specific details.

In general, I do not recommend WINE, despite the fact that I myself use it quite successfully for a number of older Windows games. This is because the sheer complexity and non-dependability involved in shoehorning an app designed for one OS into running on a foreign OS&#8213;while an amazing achievement&#8213;is fraught with pitfalls until you are more familiar and comfortable with Linux. Instead, I recommend dual-booting for users such as you.

### Post by DuckHook on 2016-07-30
Ubuntu comes in a number of "flavours", Kubuntu being one of them. They all share the same foundations, but each run a different DE (_D_esktop _E_nvironment). Kubuntu runs KDE (hence _K_ubuntu), Xubuntu runs XFCE and Lubuntu runs LXDE. If you are using Ubuntu proper, it runs Unity. There is no need to switch between them if they are running well for you. However, given your HW, I would suggest that you try out Xubuntu, Lubuntu or Ubuntu-Mate on a LiveDVD to see if they run even faster than what you are currently experiencing (which I assume is Ubuntu).

In fact, experimenting with the different flavours by way of LiveDVDs is a very good way to start exploring the 'buntu universe. In my view, this is an easier and more user-friendly learning curve than trying to install and run WINE. Please don't get me wrong: WINE is an incredible achievement and I think you may find it very useful, especially if you don't intend on installing Windows in a dual-boot. I'm just suggesting that you approach it in measured steps rather than trying to jump over tall buildings in a single leap.

Not least, try posting your further questions each in its own new thread with a properly descriptive title for the best chance to attract the help you desire.
