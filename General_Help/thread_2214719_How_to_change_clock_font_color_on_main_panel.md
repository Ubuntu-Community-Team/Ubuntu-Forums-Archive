### Post by Dennis N on 2014-04-02
You don't say which Xubuntu you are using (12.04, 12.10, or 13.10?), so here is a general answer:

The panel text color is set by the theme. So you could choose a theme with white panel text (which also affects the clock) - Greybird (the default theme) has white panel text. 

Otherwise, there is a tool to customize some theme colors including the panel text color: **gtk-theme-config** is in the repositories for 13.10; earlier releases get it from this PPA:

[https://launchpad.net/~shimmerproject/+archive/ppa](https://launchpad.net/~shimmerproject/+archive/ppa)

Reference system: Xubuntu 12.04
