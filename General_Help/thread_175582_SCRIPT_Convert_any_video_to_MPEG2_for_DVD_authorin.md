### Post by robglinka on 2006-05-13
Ok, this is a work in progress, let me know what you guys think.

This is a nautilus script that will let you right click on any video file that mplayer can play (flv, avi, aif, wmv, etc.) and it converts it to MPEG2 video that you can import directly into a DVD or share with your non-win32 loving linux friends.

I stole the actual working code from a [Linux.com]("http://www.linux.com/article.pl?sid=06/04/17/2058219") post.

There are three things it needs work on. 
1-I haven't tested it with movies that have AC3 sound. 
2-It doesn't work if the video file is on the desktop, not sure why.
3-There is not dialog/status/working box. You just have to wait until the file refreshes in the window.
