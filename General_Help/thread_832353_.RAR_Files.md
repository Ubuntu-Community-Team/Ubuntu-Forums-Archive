### Post by TVC15 on 2008-06-21
Hi all,


Why make things difficult if there is a simple solution integrated in the GUI?
By default, Ubuntu installs the Archive Manager (look in Applications -> Accessories.
The Archive Manager handles .rar files easily.

The way I use it is to right-click on a .rar file and choose "Open with Archive manager".
As simple as that!

Have fun

It's as simple as that... assuming that unrar is installed.  If it isn't installed, even using File Roller will pop up a message telling you the archive type isn't supported.  The solution posted earlier, "sudo apt-get install rar unrar" just makes sure those libraries are installed.  Then File Roller works.
