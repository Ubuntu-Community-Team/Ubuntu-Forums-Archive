### Post by hazza96 on 2005-06-07
I had a HDD die on me the the other day and had to re-install Ubuntu. I am running Hoary and have followed the Ubuntu Guide on how to add repos and install Adobe Reader.

When I recieve a PDF attachment I used to be able to select xpdf or acroread. Now I only get the choice of xpdf.

Does anyone know how to make it so that Evolution gives the choice of acroread?

---

### Post by Bob D. on 2005-06-07
This should work: right click on any PDF and choose Acrobat (acroread) as the default program to use to open PDF's. This should add Acrobat to the list of programs you can use to read PDF's.

Bob
