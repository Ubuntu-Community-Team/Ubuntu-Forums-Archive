### Post by speedwell68 on 2009-04-10
You could try the Seamonkey Composer, it comes as part of the Seamonkey Suite from Mozilla.  It's more like the old Frontpage Express that came with IE4 than full blown Frontpage, so basically you have to do your file management yourself.

**sudo apt-get install seamonkey**

That will install the full Seamonkey Suite, which contains a browser, mail client and the Composer.

### Post by callie510 on 2009-04-11
Definitely, definitely try all these alternatives - the best thing about Ubuntu for me has been all the open-source and free alternatives I've found for programs I thought were essential (like openoffice for microsoft office, etc). What I'm suggesting is a last-ditch thing. 

If you don't like any of these and you're just most comfortable with Frontpage, I have a suggestion for you that has helped me immensely (and definitely reduced my booting into Windows): keeping VirtualBox open with Windows XP in a second desktop.

I always keep VirtualBox/Windows XP open in the background - mostly so I can run iTunes and stupid windows-specific programs like this dumb ebook reader I need for a few books for my thesis. I would have just died if I had to write my whole thesis in Windows, so having VirtualBox really helped me out. It's also very cool to switch between Ubuntu and Windows XP just by hitting ctrl-alt-arrow. 

Just suggesting...
