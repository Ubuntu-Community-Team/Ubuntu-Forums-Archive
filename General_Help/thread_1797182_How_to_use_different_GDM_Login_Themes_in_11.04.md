### Post by mcduck on 2011-07-12
> **ddastoor said:**
> ubuntu tweak helps with changing the gdm logo and the theme colors and icon, but there are many fancy GDM themes on gnome look that are very different in terms of even placement of the username/password entries etc.

The question is : how to install those (GDM themes )?
I'm talking about: 
[[COLOR="Red"]http://gnome-look.org/index.php?xcontentmode=150[/COLOR]]("http://gnome-look.org/index.php?xcontentmode=150")

At some point, Ubuntu removed the Admin -> Login Window support for easy GDM theme installing.

Any ideas ?

Your problem is that the current version of GDM doesn't support those theme any more, they were made for the old GDM version while Gnome developers had to leave out support for them to implement new features of the current GDM version.

In other words, no, you won't be able to use those themes, the change wasn't made just to remove options from the users but was a necessity for the development of new features, the choice wasn't made by Ubuntu developers but the GDM developers, and finally, they might add better themeing support to the new GDM in future but it probably won't use those old themes anyway.

> 
I thought this was LINUX.
You know, where you can customize all of your stuff, the way you want it?
Not have to be stuck with things the way the developers thought it "should be".

This *is* Linux, and you *are* free to change anything you want. That doesn't mean you might not have to learn a lot of stuff to do that, and perhaps write your own tools and programs, but that's what the freedom in Linux means. You have all the tools and source code available, and there are no legal or other reasons restricting you from doing what you want.

Being able to change anything in Linux has nothing to do with having easy tools for doing that or not. It only means that there are no other restrictions than your own skills and willingness to actually do something to make the changes you want.
