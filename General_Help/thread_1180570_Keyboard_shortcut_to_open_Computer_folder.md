---

### Post by aysiu on 2009-06-07
You can create one.

If you go to System > Preferences > Keyboard Shortcuts, you can associate a keyboard shortcut with opening the Home folder (that opens Nautilus focused on your /home/username directory).

If you prefer to open it to Computer, you can add your own custom shortcut there with the command ```
nautilus computer:///
```
