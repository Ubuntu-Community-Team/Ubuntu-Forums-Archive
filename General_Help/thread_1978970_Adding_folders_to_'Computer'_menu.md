Ok, so you can add bookmarks rather than add to the 'Computer' menu. Just go into the folder you want to add and press ctrl+D. End up with structure like this in the file browser sidebar:

BOOKMARKS
new-folder-1
new-folder-2

COMPUTER
Home
Desktop
Documents
Downloads
Music
etc....
