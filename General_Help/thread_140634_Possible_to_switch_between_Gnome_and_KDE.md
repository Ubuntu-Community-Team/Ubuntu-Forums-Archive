### Post by carlosqueso on 2006-03-06
Sure....just ```
sudo apt-get install kubuntu-desktop
``` and you'll be able to choose between the two using the sessions menu at the login screen (and use kubuntu apps in GNOME).  If you just want the environment, just type ```
sudo apt-get install kde
```

### Post by Ramses de Norre on 2006-03-06
Just install kde (sudo apt-get install kubuntu-desktop), it will aks you for your default one (gnome=gdm, kde=kdm), but you can click on sessions on the login screen to choose which you want.
