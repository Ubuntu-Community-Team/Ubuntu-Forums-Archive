### Post by leeprice01 on 2010-10-28
Hi There

I had exactly the same problem booting my older Pentium 4 computer when I tried Ubuntu 9.04 for the first time and had exactly this same error.

For me it was nothing to do with Bluetooth, instead it turned out I needed to force ACPI in grub before Ubuntu would load correctly

Ubuntu 9.04 uses the older version of Grub so you need to modify the Kernel statement within Grub and add acpi=force to the end - like this example

kernel          /boot/vmlinuz-2.6.24-19-generic root=UUID=####'s ro acpi=force

I could be wrong but from my experience I spent hours on the bluetooth angle and it turned out to be this solution.

If you try it I would enjoy hearing what happens
