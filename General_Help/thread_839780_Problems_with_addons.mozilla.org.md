### Post by BWF89 on 2008-06-24
It used to be that if an extension wasn't officially compatible with Firefox you could just either download the extension and manually change the maxversion or use an extension such as MR Tech Local Install to over ride the compatibility lock out.

But now at the Firefox extensions addons website if you have an extension that isn't compatible with your version of Firefox it won't let  you download the extension at all. So your forced to go to the authors website and see if he's made it available there also. But some extension are only on the official site.

Is there someway that I can download the extension file and install it myself?

### Post by BWF89 on 2008-06-24
Oh wait, I've already found the solution to my own problem minutes after posting.

Go to an extension that isn't compatible:
[https://addons.mozilla.org/en-US/firefox/addon/1035](https://addons.mozilla.org/en-US/firefox/addon/1035)

Than scroll down until it says "See all versions".
[https://addons.mozilla.org/en-US/firefox/addons/versions/1035](https://addons.mozilla.org/en-US/firefox/addons/versions/1035)

Than just install the newest one and use an extension like [MR Tech Toolkit]("https://addons.mozilla.org/en-US/firefox/addon/421") or [Nightly Tester Tools]("https://addons.mozilla.org/en-US/firefox/addon/6543") to over ride the compatibility lockout.
