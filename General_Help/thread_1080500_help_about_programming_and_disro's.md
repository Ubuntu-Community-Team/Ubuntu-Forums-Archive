### Post by mb_webguy on 2009-02-25
As I've constantly told my students, you shouldn't focus on learning a programming language, but rather learning to program.  Once you learn the fundamental concepts (e.g. conditional logic, flow control, etc.), then learning to use a different language is just a matter of learning the differences in syntax.  In the last ten years, I've worked with a dozen different languages, depending on which was required or most appropriate for the particular problem.

If you have no prior programming experience, I'd suggest Python (though I'm a PHP man, myself).  Python is a general-purpose scripting language that is fairly easy to learn.  Since it's a high-level language, you don't have to concern yourself with concepts like memory pointers and garbage collection as you would in lower-level languages like C.  And since it's a scripting language, you can see the result of changes in the code immediately, rather than wasting a lot of time compiling.
