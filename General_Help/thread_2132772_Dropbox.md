### Post by HiImTye on 2013-04-05
I think you misunderstand how dropbox works. in order to access a file in the cloud, you have to download it locally. Windows dropbox does it the same way as Linux dropbox, Linux just does it more transparently.

if you only want to use Dropbox to download certain files, you can always use the Dropbox web interface:
[https://www.dropbox.com/home](https://www.dropbox.com/home)
