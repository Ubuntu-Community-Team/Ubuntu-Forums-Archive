### Post by easybake on 2008-07-21
[QUOTE=LonelyTraveler;5429347]I've installed a theme that should change the entire top panel, but it only changes the color of the Application, Places and System area. What can I do to change the entire panel? It is set to use theme color.

Right click on the panel. Go to Properties>Background Tab.
You can change it to a solid color. Or create your own png for it and select Background image.
