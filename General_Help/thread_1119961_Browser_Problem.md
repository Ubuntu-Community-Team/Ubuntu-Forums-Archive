### Post by The Cog on 2009-04-08
A lot of brain-dead web sites like to set their pages to a fixed width in pixels. If you have a high resolution screen, it can be like trying to read a strip of toilet paper. You can either use Ctrl-Shift-+ to increase the zoom (works on firefox 3.1, probably also on earlier versions), or menu View->Page Style->No Style, depending on how awful the web site has styled it.
