### Post by tgalati4 on 2012-12-11
Once you have a working linux setup, it's frustrating to upgrade and have several things not working.  These are known as regressions.

I keep several machines around and update them in a rolling manner so that I don't trash my most recent distro.  An alternative is to set up dual-boot.  Keep your old distro and install the next version on a new hard disk or make a partition and install it there.  Although partitioning is generally safe there is some risk of data loss, so backups are prudent.
