### Post by madwoollything on 2009-12-21
I'd like to be able to launch htop using the gnome keyboard shortcuts but have been able to find the command syntax to make this work.

In the gnome menu htop is associated with 'run in terminal' and I don't know how to achieve the equivalent of this using the keyboard shortcuts.

---

### Post by VCoolio on 2009-12-21
try this:
gnome-terminal -e htop
