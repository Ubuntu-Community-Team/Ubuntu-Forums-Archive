Have you changed permissions to these files to make them executable?

Locate the files you wish to install with Wine and right click then select properties and in the permissions tab check the executable checkboxes. Close the properties window, then right click the file again and select *Run with Wine..*

As for where they are located.. I have no idea.. :P
But if you download them again and make sure you know where you put them try the steps above and see if the applications work.
