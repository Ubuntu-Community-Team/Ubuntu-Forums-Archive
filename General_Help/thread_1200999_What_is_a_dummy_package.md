What is a dummy package?

For example. In synaptic, if you search for totem and go to its description you get this:
*A simple media player for the GNOME desktop (dummy package)*.

---

### Post by t4thfavor on 2009-06-30
Its a package that points you to the real package, so totem is probably totem-player-1.2.2.2 (whatever) but you can install it with sudo apt-get install totem.
