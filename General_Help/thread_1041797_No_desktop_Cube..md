### Post by phantom3113 on 2009-01-19
Although everything previously listed is also correct for enabling the cube, the prime thing you need to do (if not already done) is change your number of desktops to 4. To do this, go to general options (under compiz settings manager) >> desktop size >>  horizontal virtual size and then set it to 4
