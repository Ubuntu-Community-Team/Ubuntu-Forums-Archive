### Post by fragos on 2006-11-24
Sometimes downloads come with the wrong extension for the actual mime type.  I've seen PDF files with and HTML extension.  Linux identifies the correct mime type but some aps are picky about the extension and you have to edit it to get things working.  I haven't seen Linux assign mime mismatched extensions.
