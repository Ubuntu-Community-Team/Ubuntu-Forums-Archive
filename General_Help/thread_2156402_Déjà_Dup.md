### Post by garyzw on 2013-06-21
I have a few questions about Déjà Dup.

If I back up my Home Folder using Déjà Dup--then lets say Ubuntu fails to start a few days later can I do a fresh install and then restore my Home Folder and have all my apps and things back the way they were before the failure? Or would I need to reinstall all my apps.

---

### Post by zacktu on 2013-06-21
If all of your apps are ones that are in a default installation, then you will have your apps.  If there are other apps that you installed yourself, then you'll have to install them again.  The configuration files for whatever apps you were using before will be in your home directory.
