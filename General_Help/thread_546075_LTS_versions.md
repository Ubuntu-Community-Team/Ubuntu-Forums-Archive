Well, if by "support", you mean just asking questions and getting answers, you're right, people on the forums will probably still help you with Hoary if you really wanted it. But it means more than that when we talk about "Long Term Support". For example, Hoary and Breezy no longer get security updates -- even if a major, groundbreaking vulnerability were found tomorrow, they would never get patched. The expectation is for users to update to a newer version. However, Dapper will still be getting security updates for a long, long time to come. Moreover, Breezy and Hoary's repositories don't even work anymore. Dapper's will work for five years. Bug reports for Dapper will still get attention while Hoary/Breezy will not. And so on.
Also (though I'm not 100% sure about this), I think Canonical only gives commercial support to versions still in the support period. Which is important for entities like large businesses.

Basically, the Ubuntu developer community still pays some attention to Dapper for five years, but not to the older ones.

I hope this answers your question :) The next Ubuntu LTS (Hardy Heron) will be out long before Dapper or Feisty expire, so there's no reason not to upgrade.
