### Post by ajgreeny on 2015-01-12
As it has a nvidia graphic card, you may need to use the nomodeset boot option even to get the live system running and if the graphics is a hybrid optimus system you will also need to investigate bumblebee to get that working.

There is no helpful detail of the wireless chip used for its wifi, as far as I could see, so that could also potentially be a problem, but running the live DVD or USB will help you sort out all those difficulties.
