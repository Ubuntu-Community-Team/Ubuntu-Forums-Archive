### Post by Sylos on 2012-04-05
Depending on the computer set up os configuration it could be that some of the memory is allocated to your GPU (graphics processing) hence you see a lower amount available for normal RAM use.

On some mobos it is possible to change the RAM allocation to GPU - others you cant. Might be worth a look in the BIOS to see what the settings are.

Cheers
