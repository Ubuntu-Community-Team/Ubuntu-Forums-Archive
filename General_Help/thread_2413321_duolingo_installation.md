### Post by bitsnpcs on 2019-02-25
Hello oilyfish,

this is a tutorial for doing Gnome Shell extensions from zip - [https://linuxconfig.org/how-to-install-gnome-shell-extensions-from-zip-file-using-command-line-on-ubuntu-18-04-bionic-beaver-linux](https://linuxconfig.org/how-to-install-gnome-shell-extensions-from-zip-file-using-command-line-on-ubuntu-18-04-bionic-beaver-linux)

There is another way for setting up and using Gnome shell extensions, this is the tutorial I followed for how to set up for Gnome Extensions
 - [https://itsfoss.com/gnome-shell-extensions/](https://itsfoss.com/gnome-shell-extensions/)

When it is set up this way, on the page you linked to, and for other future Gnome extensions, you would only then need to drag the slider to "On" to install the extension. It is a more efficient set up longer term.
