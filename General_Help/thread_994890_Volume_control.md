### Post by fooman on 2008-11-27
try double-clicking on the speaker (volume) icon in the top panel.

at the bottom of the volume control box that opens up,  there is an arrow next to "show switches"...click on that arrow.

that opens up more options (make sure you "maximize" that window to see all options) some of which are mic related...try checking/unchecking some of those options and see if it helps.
