### Post by oldfred on 2009-10-12
I do not know about debian, but for karmic this is the solution:

sudo apt-get install os-prober
sudo os-prober
sudo update-grub2

The other solution is to manually add a new file
gksudo gedit /etc/grub.d/06_custom

### BEGIN /etc/grub.d/21_winxp ###
menuentry "Windows XP" {
       insmod ntfs
       set root=(hd0,1)
       search --no-floppy --fs-uuid --set a2f48f73f48f490d
       drivemap -s (hd0) ${root}
       chainloader +1
}

### END /etc/grub.d/21_winxp ###

Adjust root, uuid & drivemap entries to your system
