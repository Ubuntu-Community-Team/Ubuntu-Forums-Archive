### Post by agente100gelo on 2007-12-28
I'm developing a project called apturl.net. On this site you can search and installation of software and libraries for systems with apt-url package installed. The site obtain information from official repositories and organize for the search and easy use.

The idea is to make a more common to users who have the habit of installing software through sites.

To access: 
[INDENT][www.apturl.net](http://www.apturl.net)[/INDENT]

To install apt-url:
```
sudo apt-get install apt-url 
```

Some suggestion?

### Post by agente100gelo on 2007-12-31
Hello LaRoza,

One of the advantages is that the search uses a plugin called[ Lucene]("http://framework.zend.com/manual/en/zend.search.lucene.html") that returns results by relevance. Another is that it may in the future include a ranking / comments system made by users.

The other reason is that many users, former users of Windows, have a habit site access, to search and download application. I like of "add / remove applications," but I believe that a system "Install Now" would be more friendly than select and apply method.
