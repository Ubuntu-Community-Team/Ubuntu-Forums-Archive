### Post by bcw on 2007-06-24
Hello,

Search in the forum for xstroke - a member made a package for it.  Also, search on the web for onestroke - I don't know of a package (and haven't actually tried it yet myself).

These allow handwriting recognition.

xvkbd, onboard, and gok provide on-screen keyboards.  Personally, I think gok is a mess, and I use xvkbd with the fitaly layout tweaked to get me all the punctuation keys as well.  I haven't tried onboard yet.

Jarnal and gournal provide ink text entry - jarnal includes handwriting recognition, but I haven't tested it much to see how good it is - I just use xstroke for everything (except Mozilla programs, which won't accept input from xstroke).

As a separate matter, VMware provides a free program which will extract a virtual image of a native Windows XP Tablet edition install, and another free program (vmware player) which can run that image on a Linux host.  I do this, and the handwriting recognition and voice recognition work just fine.  I believe this is valid under the Windows license, as it's the same hardware the copy of XP was provided with.

Cheers,
Bret
