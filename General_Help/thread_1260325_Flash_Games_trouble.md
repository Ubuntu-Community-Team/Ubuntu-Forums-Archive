### Post by ankspo71 on 2009-09-07
Files and streams are not always hosted on the same website that you are visiting, so the other servers might be down. This could cause a flash video or game to open up as a black screen on a site that is working, because it doesn't have access to the content you need.

Did you search the internet for other flash games to try out to see if they are working? I just searched (free online games) on Google, and the very first result had flash games that didn't work, but the second result had flash games that did. So that could be what is going on. 

I run into this black screen from flash problem every so often. I never saw it happen on youtube though.
