### Post by 3Miro on 2009-08-05
Go to System -> Administration -> Hardware Manager and see if all the drivers have been enabled. If not, enable them.

If that doesn't help, there is something else misconfigured on your machine. I have done simultaneously: a game, a numerical simulation, download torrent and post on various forums with firefox - no lag. XP dies on the second of those anyway, so I have not thoroughly tested that under windows.
