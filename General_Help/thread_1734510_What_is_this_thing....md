### Post by BertN45 on 2011-04-20
Linux is the operating system, but unlike Windows it has no integrated Graphical user interface (Gui). Many Gui's are available for Linux like Gnome, KDE, Unity, XFCE etc, they provide the desktop and windowing software. The distributions (like Ubuntu, Open Suse, Debian) combine Linux with one of the Gui's and a set of application programs. That makes the Linux world somewhat confusing in the beginning, but it is also a strong point because you have a choice. It is what YOU like and what your PC can handle.  It is not like Windows where one size fits all.

One of the applications is Apache the web server, but there are more then 10.000 applications available for $0.00.
