Hi to help you this is what I have just done (in easy steps).

1 Went to this page [http://rawtherapee.com/downloads](http://rawtherapee.com/downloads)

2 Chose the file I wanted and clicked on it. Download started automatically.

3 Located the downloaded file (in Downloads), right-clicked on it and Extract Here.

4 In the extracted files folder (Rawtherapee) found a file called 'rawtherapee', right clicked this to check the properties. In the permissions tab checked the 'Allow executing file as program' box

5 Double clicked on it. The program started.
