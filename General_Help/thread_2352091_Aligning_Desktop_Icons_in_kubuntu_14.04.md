### Post by RobGoss on 2017-02-09
If I understand you correctly you want to have the application icons on the left side of your desktop correct?

If this is correct try using something like **Docky** you can align it on what ever side of the desktop you want

You can read up on how Docky works and features [http://wiki.go-docky.com/index.php?title=Installation](http://wiki.go-docky.com/index.php?title=Installation)

There's also other docks to choose from that work great with Ubuntu distributions
