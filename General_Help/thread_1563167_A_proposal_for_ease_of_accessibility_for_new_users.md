### Post by afrowildo on 2010-08-28
One of the problems new users face when moving from Windows to Ubuntu is that they have to find replacements for the applications they were using in Windows. These applications to readily exist, however finding them can often be tricky. Empathy will replace Windows Live Messenger, Gwibber can replace social clients such as TweetDeck, Firefox -> Internet Explorer (some 60% of computer users still use IE), etc. Tomboy is also a really good notepad, which is all many computer users need for personal writing, yet it is buried away in the 'Accessories' tab. The general problem is that applications that the user will use on a daily basis are lost among the ones that will rarely (if ever) need to open. 

This problem is compounded by the fact that the name of these applications does not always reflect it's purpose, which will also confuse new users.

I propose the addition of a window in the style of the Start menu on Windows, where applications can be pinned (as part of the default installation) so that they are more visible to the user. This can be implemented by moving the category list to the right and placing the start menu in the vacant space. 

Applications that are deemed to be of most value to the average computer user can be placed here by the developers prior to the release of the latest version of Ubuntu so that they are placed 'front and centre' in the user experience. 

The names of these applications could also be changed to help new users easily identify the purpose of the app, such as 'Tomboy Notes' being renamed 'Notepad', Firefox renamed to 'Firefox Web Browser', or at least place a slot in this Start Menu for a 'Web Browser' or a 'Notepad'.

The brainstorm for this idea can be found [here]("http://brainstorm.ubuntu.com/idea/25716/"). Any and all comments, suggestions and advice are welcome.
