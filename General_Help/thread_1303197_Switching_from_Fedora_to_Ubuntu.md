### Post by jualin on 2009-10-28
You can use Synaptic Package Manager (located under System, Administration) and select multiple packages to install. Or if you are using the command line then just separate the packages with spaces > sudo apt-get install firefox vlc
That should install the packages "firefox" and "vlc". You get the idea lol?
Fedora and OpenSuse have DVD for their installation, that's why they can have more packages locally (from the DVD) without the need of downloading them. Since Ubuntu comes in a CD, it doesn't have space to put too many packages (700 mb vs almost 5000 mb).
Which packages are you trying to install?
