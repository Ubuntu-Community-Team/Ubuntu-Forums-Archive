### Post by qpieus on 2007-06-12
It looks like you are trying to run configure on a tar archive. That's not going to work. First you need to unpack the tar.gz archive. 

cd into the directory where the archive is and type
```

tar xvzf openlaszlo-3.3.3-unix(2).tar.gz
```

After the archive is unpacked, look in the directory for a file named readme or install (or something similar). It'll have any specific instructions for installing the program. Assuming a normal configure and install process, the next step is to configure, as you tried already, but this time the archive is unpacked so the configure should work properly.
```
./configure
```
Then compile with```
make
```
Then install with ```
sudo make install
```
Optionally after the installation, you can```
make clean
``` to clean up temporary files that were generated by the compiling process. Not sure if you need sudo with the make clean command.
