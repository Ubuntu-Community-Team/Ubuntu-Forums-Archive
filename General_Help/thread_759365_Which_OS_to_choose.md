### Post by sekinto on 2008-04-19
Are you talking about games native to Linux or Windows? If you are talking about Windows games you will need Wine to run them (and that might not even run all of them), you can get Wine for any distrobution. If you are talking about Linux games there are a few distros especially made for gaming. Here is one of them:
[http://supergamer.org/](http://supergamer.org/)

If you want a system that will be the best for gaming I would recommend only installing what you need to run the games you want. Maybe do a minimal Debian install and add a lightweight window manager and a few apps you feel you absolutely need.
