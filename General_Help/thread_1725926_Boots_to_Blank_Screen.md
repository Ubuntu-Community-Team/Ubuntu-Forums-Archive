### Post by tredegar on 2011-04-10
Lucid needs 256MB RAM as a minimum.

Try [xubuntu]("http://www.xubuntu.org/") which needs:
    * 500 MHz or better processor
    * 1.5GB of available disk space
    * 192MB of memory (RAM) for installation but only 128 MB of RAM to run from the installation CD
    * CD-ROM drive
    * Ethernet interface
    * VGA graphics interface

and looks like it would be better suited to your hardware.
