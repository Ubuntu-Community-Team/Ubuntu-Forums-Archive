### Post by BoostdDSM on 2010-10-17
Ok so here's the situation. I have a Dell E177FP monitor and am running Ubuntu 10.10. I want to change my screen resolution because everything looks a little too big for my tastes and when i click the menu option on my monitor it says the current resolution 1024x768 and the optimum 1280x1024.  So i would like to make the resolution 1280x1024 on Ubuntu 10.10. When i go to System, Preferences, Monitor, the highest available option for resolution is 1024x768. So how do i get a higher resolution option so i can make the resolution optimum for my monitor.

---

### Post by BoostdDSM on 2010-10-19
Ok so i fixed it myself, but i figured id post here anyways so maybe someone with the same problem will have an answer they need. i went to system, preferences, monitors, then i unchecked "same image in all monitors" then clicked on the laptop screen and turned it to off and then went back to the Dell monitor and checked the resolution and it allowed me to go to 1280x1024. So I'm guessing because i have a laptop it was detecting the resolution for the laptop screen and not the external monitor, so if anyone runs into this problem this should be able to help you.

### Post by mister_playboy on 2010-10-19
This problem arises because the "mirrored screens" need to use a resolution that both monitors support, and this is most commonly 1024x768.
