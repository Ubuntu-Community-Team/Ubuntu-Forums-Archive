### Post by hugmenot on 2007-12-04
The PDF file is broken. It seems Acroread is more error resilient here.

This is what Evince print on the terminal when opening the file:

```
Error: Mismatch between font type and embedded font file

```

I guess this means that the font is TrueType (ttf) but is declared as Type1 PostScript.

If you want file a bug here asking for more error correction:
[https://bugs.freedesktop.org/enter_bug.cgi?product=poppler](https://bugs.freedesktop.org/enter_bug.cgi?product=poppler)


EDIT: It seems that is not true. Looking into the PDF shows that the font is declared as TrueType.
```
26 0 obj<</Subtype /TrueType/FirstChar 32/LastChar 255/Widths 29 0 R/FontDescriptor 28 0 R/Encoding /WinAnsiEncoding/Type /Font/BaseFont /ABHDEJ+TimesNewRoman,B
```

You should file a bug a point out the misidentification.
