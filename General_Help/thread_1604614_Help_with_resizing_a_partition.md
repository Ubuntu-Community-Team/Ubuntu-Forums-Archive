### Post by aeronutt on 2010-10-24
install and run gparted, or boot ubuntu from CD, and run gparted.
gparted will let you resize, format, etc, partitions.

ALthough I've never had a problem with gparted while resizing, etc. I'd recommend a backup of your data prior to use.
