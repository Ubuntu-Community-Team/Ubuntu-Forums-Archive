### Post by benanzo on 2007-06-06
I have had very good luck restoring deleted data with a combination of Autopsy and Foremost.  Just boot an Ubuntu Live CD, download and install those two apps while in the Live environment and run them one after the other.  Be sure to set the directory that recovered data will be sent to as your external disk, otherwise the RAM in the laptop will fill to the brim and you'll lose what you recovered.

A long as nothing has been done to the disk since the format, you should be able to recover most-all of it.  But if it has had a new install of an OS then the odds go down significantly.

Good Luck!
