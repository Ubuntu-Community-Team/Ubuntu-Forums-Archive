### Post by lisati on 2010-01-19
> **lupeatx said:**
> How do I start hosting a web page?

You are likely to need:
[LIST]
[*]Software to help create web pages. "Kompozer" is one Ubuntu-friendly option. Open Office Writer can save documents in HTML format.
[*]A domain name. I use a free one from [www.no-ip.com](www.no-ip.com) 
[*]If you have a dynamic IP address assigned by your ISP, you'll need the means for keeping the DNS records up-to-date in case your IP changes for some reason. Some sites like no-ip can help with software to do this automatically. My modem/router has an option to take care of this automatically as well.
[/LIST]
