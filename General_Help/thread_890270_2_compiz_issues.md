### Post by 00arthuryu on 2008-08-17
Games will run choppy, as you have to remember that compiz uses the same graphic card to render its functions, wobbliness etc. 

emerald will exit when you close the terminal where you ran
```

emerald --replace

```
If you want emerald to run at startup, add it to system - admin - sessions.

or you could put emerald --replace as a keybinding-shortcut in compiz settings config manager. Its in the 'general' section under 'commands'

hope this helps
