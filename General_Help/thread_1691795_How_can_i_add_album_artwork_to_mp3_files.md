### Post by jerenept on 2011-02-20
> **pavelexpertov said:**
> What are metadata tags? Moreover, i have got a music library in windows partition and i can access it in ubuntu partition. But is it still possible to add artwork when the library is in different partition?

Metadata tags, aka ID3 Tags, are little snippets of information, (like Album/Artist) embedded into the mp3 files. If you downloaded/ripped the files from a good source (original CD's, Amazon, Ubuntu One Music Store, etc.) these should already be correct.

to your second question; Banshee is able to read and fix tags anywhere, as long as it has write permission.
