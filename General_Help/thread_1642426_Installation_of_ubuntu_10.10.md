### Post by NertSkull on 2010-12-10
You can just install linux over the windows partition.

The problem your going to run into is the limit of 4 primary partitions on a drive.  Since you already have three you need a place for you main OS and a swap space.

I always use the alternative install CD, and in there you can set up a logical volume wherein you can set up two partitions.

I don't know about the traditional install CD (its been a while).  But you can probably load it and see what it says.  Just don't commit changes.  But it has a pretty decent partition editor that you can use to help.

But as always.  BACK UP YOUR DATA.  I can't stress that enough.  Back up all your drives before trying anything new with partitions.
