### Post by acimmarusti on 2009-04-15
Ubuntu uses the gnome desktop environment which is lighter than KDE, which is the one used by Kubuntu. It has been said that KDE is superior to Gnome in that it can be more readily customized. But this, of course, depends on what you want to do. KDE has its set of applications which are different from Gnome's. Some are said to be better than their Gnome counterparts, like Amarok (music player) or K3B (burner software). Again, this is subject to what you want to do. I find rhythmbox suitable enough for what I do, so having a heavier player like amarok is not my priority.

Xubuntu is based on the light weight XCFE desktop environment. It's even lighter than Gnome. This makes Xubuntu ideal for older less powerful computers. However, it lacks many apps that make gnome and KDE awesome, so it has to rely on some gnome and KDE apps anyway.

I don't know much about the others in your list, except that Edubuntu was engineered as an educational desktop environment...probably not what you are looking for. There is also Ubuntu Studio, designed basically for artists.

That's all I know in a nutshell

### Post by paradigm2 on 2009-04-15
Generally if you are new to Linux I would recommend that you stay with Ubuntu (the default version which uses Gnome) simply because it is easier and you will be able to get the most support.

If your computer is older or you just want to be able to do core things like browse webpages as fast as possible with no lagging and you don't care about eye candy and all sorts of other thing I'd go with Xubuntu.

If you are a seasoned Linux user, well then you probably don't even have to ask this question. ;)  But if so, I would consider Kubuntu if you like the power and configurability of KDE.  It is my observation that more power user type applications seem to work natively with KDE rather than Gnome.

However, all the above and everything else that has been said thus far being said, you can install kde, gnome, and xfce on any version you install whether it is Xubuntu, Ubuntu, or Kubuntu.  They will work together and you can switch between them.  Its just that it makes things more complicated and uses more resources.

My bottomline advice:  If your system has over 1.0 GB Ghz cpu, and 512MB (this does not match what the website reports but is my personal experience) and you are new, I'd go with Ubuntu for now and learn Gnome.  If your system has less than this, you should probably go with Xubuntu unless you don't mind a little slowdown.

Consider KDE maybe when you get more advanced.
