### Post by pointone on 2008-01-22
Most routers have some sort of bandwidth monitoring available out of the box. Try logging into your router at 192.168.0.1 or 192.168.1.1 and search around in the status or logging sections. If you can't find anything, post the make and model of your router here.

### Post by Herman on 2008-01-22
That depends on what make and model of router you have. I can't tell you about yours, but I can tell you about mine.

The way I monitor my router, which is a D-Link 524 Air-Plus Wireless Router, is through my Firefox web browser.

My ADSL modem's software interface is accessible the same way too, so I presume everyone's routers and modems might be something the same.

You just need to find out the IP number for your router somehow, maybe you have a manual that came with it or a CD that you can pop in your CD drive and look for a file called 'Index.html', 'README.TXT', or maybe a .pdf file of some description that will have helpful information in it.

Once you know the IP number for your router you should be able to access it's software interface that way. 

My broadband modem/router has a fully configurable firewall which I can access in GUI mode and do quite a lot, and there's also a CLI available if I need to do something special that's not available from the command line.
It contains a firewall that seems remarkably similar to IP Tables, I would almost place a bet that it is IP Tables and probably the modem contains a Linux kernel.

My router, too has all kind of interesting settings in it available from GUI mode through my firefox web browser. 
I can set exactly which IP addresses to allow to access the internet, and even what times of day or days of the week they will be allowed or not allowed.
It has logs I can look at too.

It's certainly well worth the time spent to do a little reading on our hardware and some exploring and maybe a few experiments if we want to get the most out of our equipment.

I have my IP numbers for both my router (for my LAN), and my ADSL Broadband modem/router bookmarked in Firefox so I can always access then at the click of my mouse.

I'm not sure if that's exactly the answer you were expecting for, but I hope that will be some help anyway.

Regards, Herman :)
