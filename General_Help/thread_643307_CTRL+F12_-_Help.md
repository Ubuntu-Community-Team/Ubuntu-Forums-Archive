### Post by jespdj on 2007-12-18
Maybe Ctrl+F12 is mapped to some Compiz action.

Install the Compiz Fusion settings manager if you haven't got it already:
```
sudo apt-get install compizconfig-settings-manager
```
Then go to System / Preferences / Advanced Desktop Effects Settings and look around in the settings. Most of the plug-ins have key bindings listed on the Actions tab.
