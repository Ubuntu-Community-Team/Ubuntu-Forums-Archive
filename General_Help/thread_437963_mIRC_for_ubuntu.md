### Post by rolando2424 on 2007-05-09
Just a reminder.

YOU CAN'T INSTALL EXE ON LINUX! (ok, I'm lying, actually you can, it's just that if you just install Ubuntu, you should first get a feel of some Linux Applications that can replace almost (I dare to say "all", if you exclude some games) the Windows applications.)

Give X-chat a try.

The simplest way (for me to explain), is just open a terminal (Applications --> Accessories --> Terminal) and type this:

> sudo apt-get install xchat

And you're good to go :D

(If you REALLY can't find a replacement for a windows application, and what to install a windows version, just tell us, and we'll try to help you with it. But give some Linux applications a try :D)
