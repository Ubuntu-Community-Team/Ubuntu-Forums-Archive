### Post by Zorael on 2008-06-03
You could install **gdebi-kde**, the KDE flavor of the Gnome GUI .deb installer. Then rightclick  a .deb file -> properties -> wrench (edit file type) -> add -> pick gdebi from wherever it ended up on the kmenu -> OK.
