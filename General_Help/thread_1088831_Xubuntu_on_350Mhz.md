### Post by wildman4god on 2009-03-13
Antix is a bit heavier than puppy, if you want a more user friendly puppy, then check out the puplets on their site, they are heavier than the original puppy but they should still be lighter than antix, what specs are you running, cause if you have enough ram puppy Linux loads entirely into ram making it ultra fast when opening apps (apps open instantly) also the next version of puppy will use icewm like antix which will make it more user friendly (a desktop environment determines a lot on how user friendly a distro is)

Edit: You might also want to try tinyme, its a derivative of pclinuxos and its total size is only 200mb it uses KDE 3.5 and it looks very nice out of the box, also its rpm based so their is a larger library of software and it uses synaptic, so you can add stuff by adding repos.

Edit: Okay I checked again and tinyme doesn't use kde it uses a combo of openbox and lxpanel which looks a heck of a lot like kde 3.5. Check it out at: [http://tinymelinux.com/](http://tinymelinux.com/)
