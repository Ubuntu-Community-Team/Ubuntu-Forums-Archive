### Post by QIII on 2018-08-23
Linux Mint is based on Ubuntu.  Fedora is the test bed for Red Hat Enterprise Linux.  Both would be relatively the same as Ubuntu with regard to resource consumption.

Lubuntu is lighter because the amount of stuff piled on top of the Linux kernel (which runs using a very, very small amount of resources) is a lot less complex.

If Lubuntu does not fit your needs, you can try [Bodhi Linux]("https://www.bodhilinux.com/"), which will run on a 300MHz processor, 128MB of RAM and 2.5GB of storage.  Not that it will run well, but it will run.  It is based on LTS releases of Ubuntu.

I'd say Lubuntu is a great choice.  Your machine will surely run that.

### Post by oldfred on 2018-08-24
My old 2006 laptop with Intel Core2 Duo & 1.5GB of RAM has Ubuntu, but install flashback/fallback/gnome-panel which is/was like the old gnome2. 
It had 12.04 when I retired it for a new desktop, but recently installed 16.04 and it still worked perhaps a bit better.

But with 1.5 GB of RAM, I could only load 1 larger app and maybe a couple of smaller apps. When I was running Firefox & clicked to open something with LibreOffice, system would turn gray & pause for a few seconds, so I knew it was using swap. 
I just learned not to open lots of tabs or more than one larger app and it worked well.
