### Post by psychicdragon on 2006-03-22
I'm not sure what the build dependencies for OOo are (there are probably quite a few). Before you try to build it you should make sure you have **build-essential** installed and you can run this command to get the build dependencies for the Ubuntu OOo package (they should be mostly if not entirely the same as the upstream one):```
sudo apt-get build-dep openoffice.org2
```

### Post by Gustav on 2006-03-22
Or you could add this to your /etc/apt/sources.list:

deb [http://people.ubuntu.com/~doko/OOo2/](http://people.ubuntu.com/~doko/OOo2/) ./

And then install it via apt-get or synaptic.
It's much easier than compiling OpenOffice.org
