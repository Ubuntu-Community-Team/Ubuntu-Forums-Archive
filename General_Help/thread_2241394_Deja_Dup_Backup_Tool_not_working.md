### Post by deadflowr on 2014-08-25
Ubuntu One file sharing service no longer exists.
Open deja-dup-preferences and change the storage location.

If you are trying to retrieve files off Ubuntu One's file sharing service, then those may have already been wiped off the planet earth.
Unfortunately, I can't find the online announcement, but i can quote the email all users should have gotten:
> Remember that you will have until 31st July 2014 to collect all of your content. After that date, all remaining content will be deleted.
