### Post by srs5694 on 2010-10-11
GParted runs fine from KDE. With few exceptions, changing your desktop environment (GNOME to KDE, KDE to Xfce, or whatever) does not affect what programs you can run. The main exceptions are little applets that are tightly integrated into the desktop environment, such as clocks or CPU use monitors that appear as part of a panel. Windows managers can also be an exception. Tools such as partition managers, word processors, and text editors normally run from any environment. Some of these tools do have links to particular desktop environments, but they almost invariably run fine from other environments.

That said, libparted-based partitioning tools such as GParted may not do what you need if you've booted from a disk you want to partition. In such cases, booting from an emergency disc should enable you to repartition your disk.
