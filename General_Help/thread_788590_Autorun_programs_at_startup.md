How would I set up certain programs to automatically start when Ubuntu boots up? For example, I always open up a shell window, so it would be handy to have that happen automatically.

---

### Post by macaholic on 2008-05-09
In gnome, go to System > Preferences > Sessions. You can add any new entries you like to run at startup, including a shell if you wish.
