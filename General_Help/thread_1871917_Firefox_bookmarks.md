### Post by AndrewGen on 2011-10-29
I recently installed Ubuntu 11.10 on a new pc and would like to copy over my Firefox bookmarks from my old Linux 10.10 drive.  I found a file marked bookmarks.html ,but there is no corresponding file on my new drive.  

How do I copy over my Firefox favorites to my new 11.10 drive?

Thanks

---

### Post by RaganN on 2011-10-29
Click on bookmarks at the top of the screen, then Show All Bookmarks, then click on Import and Backup, select backup and save the bookmarks to a file.  Copy this file to the other computer, follow the same process there, except select Restore and select the file you created.
