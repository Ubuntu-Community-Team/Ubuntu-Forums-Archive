### Post by jimrz on 2007-08-11
there are MANY threads on this subject. suffice it to say "opinions vary" ...

I, personally, did not use it when I started with ubuntu because I wanted to learn how to do it "the right way". then one day, much latr on, I mangaed to break my laptop's dapper install completely and did so about 1 1/2 days before I had to leave on an extended business trip. since I had no time to (though I did make some - yeah sleep didn't happen) I did the fresh install then decided to give  it a go. on that occassion Automatix really saved my butt, and meant that I did not have to deal with 2+ weeks on the road and only xp to use. since then, I have been using it as a time saver, but avoid using it for anything that can be installed quickly and easily through the regular repo's. 

note: I always do fresh installs rather than dist  upgrades, with which I hear it gives some issues, and have never had any problem related to Automatix.
