You mean updates?

Yes, installing the updates is *definitely* a good idea, especially the security updates. If your computer is connected to the Internet then you shouldn't even consider not installing the updates unless there's a very, very good reason.

They shouldn't cause yo any troubles, though, as Ubuntu is rather strict about what kind of updates it offers (only security updates and fixes for serious bugs are offered after a Ubuntu version has been released). So if you are having troubles you sure should post information about them, there shouldn't be anything that couldn't be easily fixed. (also providing information about such problems to the developers, through a bug report, would be a good idea. They can't fix problems if nobody tells them about the problems... ;))

### Post by HermanAB on 2011-03-17
Howdy,

"Don't fix it if it ain't broke." does have a lot of merit.

In general, updates work fine if:
You own a bog standard desktop machine that by accident happens to be the same as one of the test machines in the QA lab of your favourite distro.

and they cause problems if:
You compile programs from source sometimes
You use a virtualizer like VMware or Virtualbox
you run a laptop machine

Instead of doing updates, I completely re-install my machines once a year using the latest distro version.  That avoids a whole lot of trouble.
