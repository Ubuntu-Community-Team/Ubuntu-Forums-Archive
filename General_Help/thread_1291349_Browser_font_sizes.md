### Post by StuartN on 2009-10-14
> **gideond said:**
> I'm seeing to end of problems with browser font sizes.

In Firefox select View->Zoom->Zoom In to a level that suits you.

If you want to affect every application then go to System->Preferences->Appearance and select Details on the Font tab. Change the resolution to a bigger number for bigger fonts and a smaller number for smaller fonts - 45 is tiny, 96 is the default, 180 is big.

### Post by The Cog on 2009-10-14
In firefox, **Ctrl +** (control plus) and **Ctrl -** change the zoom. **Ctrl 0** resets to normal. I use zoom on most of my regular web sites. Firefox remembers the zoom level next time you visit. In the worst cases, I use the menu View->Page Style->No Style to render pages more readable.

### Post by The Cog on 2009-10-15
If that's the issue, try from the menu: System->Preferences->Appearance, click the Fonts tab, click the Details button and increase the Dots per inch. Most things react instantly, but firefox needs restarting before it uses the new settings.
