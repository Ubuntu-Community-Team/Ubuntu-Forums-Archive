### Post by yancek on 2018-04-17
If you have a computer with xp it is probably too old to really run Ubuntu in virtual software but the only way to find out is to try.  We don't know what your hardware.
 You can boot an iso and also install it in VirtualBox or VMWare and update it.   The process of booting an Ubuntu iso and installing it on virtual software is the same as installing xp in virtual software on Ubuntu.

```
use the Ubuntu boot partition inside a window in WinXP.
```

An Ubuntu boot partition won't work on a windows filesystem partition. 

> That way anytime I update or modify Ubuntu booted from the harddrive, I  get the same results in the windowed version of Ubuntu (In WinXP),and  visa versa.
Not sure what you mean by that as it sounds like you want two instances of Ubuntu.

There was a program for Ubuntu called wubi which was similar to using virtual software.  I think the last supported version of Wubi for Ubuntu was 12.04 which is no longer supported.
