### Post by vanadium on 2011-02-07
Try the following. Open a Nautilus file manager window, and right-click any folder. Choose "Open with other application ...". Select "File Browser", make sure "Remember this application for "Folder" files" is checked, and hit "Open". This will probably restore the correct operation of your Places menu.
