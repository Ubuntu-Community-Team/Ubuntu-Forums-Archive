### Post by Tickborn on 2009-11-17
Hi,
For Trou Type is quiet easy. Just copy the .ttf file in /usr/share/truetype and restart oOffice. the new fonts will just be availible in the scroll down menu. This way you will be able to install any kind of TT font, even downloaded one and originally not included in Microsoft.
Cheers :)

### Post by scouser73 on 2009-11-17
Enter this code in **Terminal** to install Microsoft fonts.

**> sudo apt-get install msttcorefonts**
