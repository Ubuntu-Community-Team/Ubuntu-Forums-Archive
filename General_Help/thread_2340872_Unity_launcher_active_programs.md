### Post by Frogs Hair on 2016-10-22
Newly opened applications always appear below pinned/favorite applications. There are no settings I'm aware of in the Unity Tweak Tool or Compiz to change this behavior. My preference is pin very few applications and use a small icon setting so I don't have to go hunting through stacked icons. Moving the launcher to the bottom may give you more space depending on screen resolution. This can be done with the Unity Tweak Tool in 16.04 +.
