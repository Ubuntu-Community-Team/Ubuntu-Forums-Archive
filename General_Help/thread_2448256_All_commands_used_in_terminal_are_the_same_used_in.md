### Post by ajgreeny on 2020-08-05
Many basic commands are exactly the same in all distros but when we start using commands to update or install packages they may be very different depending on the package management system used by the distros.

Ubuntu and other ubuntu based distros use apt and related commands to update or install packages, fedora used to use yum, I think, but even that may have changed as I have not looked at fedora for many years.

If you have specific problems tell us in more detail. Without that information it is impossible to help you in any useful manner.
Assuming you can boot and at least get to a command line, show us the output of command
**inxi -Fzx**

### Post by Impavidus on 2020-08-05
Mostly. Not every command line tool is installed on every Linux and not always the same version, but there is a set of more or less standard tools that do exactly the same thing at least while using some standard option. I mean, on some systems these standard tools may accept non-standard options, but when given the standard options they should behave the same on every Linux. In fact, on pretty much every OS except Windows. On the other hand, the command I use to change the desktop wallpaper is specific to this version of Xubuntu.
