Hey all. I thought this would be a good place to drop this link because I remember looking for the same thing in the same place not too long ago. This document is geared towards beginners in programming and uses Python to teach it, fitting I'd say for anyone wanting to learn this skill in Ubuntu, or whatever flavor of Linux you use. Also, don't let the title scare you, this guy (Zed A. Shaw) does an exquisite job of simplifying an otherwise chaotic thing such as programming. He lays out what beginners need to focus on and doesn't cloud things up with advanced technical jargon that most beginners don't need to worry about (not yet at least!). Anyway, he does a really good job (one of the best in my opinion) of simplifying things and anyone who is serious about learning programming would do very well to read this in it's entirety. Peace.
