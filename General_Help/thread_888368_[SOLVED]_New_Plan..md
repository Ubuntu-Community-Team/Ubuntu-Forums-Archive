### Post by cariboo on 2008-08-13
The only problem with restoring from images, is that you are constantly having to create new images. If operating systems never changed this would be a great strategy, but unfortunately nothing remains the same, bug fixes no matter what operating system are a fact of life, and of course applications are constantly being upgraded.

I have a friend that uses this strategy, and it seems like he spends more time creating new images then does doing anything else. But then I think he gets great enjoyment from creating backup images. :)
