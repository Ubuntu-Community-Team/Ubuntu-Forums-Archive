### Post by blackened on 2009-09-23
It's also very bad form to start multiple threads regarding the same problem when you feel you have not been answered quickly enough. This is not Windows, you did not pay for the software nor for support, and nobody here owes you anything. We are all volunteers, so be grateful for the help you get.

That said, everything I've read on this problem, which admittedly isn't much, points to incorrect server settings. Double-check them and try again. If you're fairly certain that your server settings are correct, then you could try those same settings in Thunderbird to confirm.

If the problem persists,  you might want to file a bug report over at launchpad.
