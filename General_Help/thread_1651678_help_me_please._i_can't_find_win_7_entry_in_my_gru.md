### Post by Quackers on 2010-12-24
Your first problem with the system is that sda1 (the first partition) is not mounting for some reason. This partition is likely to have had your first 2 Windows boot files on it.
We can try to run a chkdsk on the drive and see if it will run. This will try to repair any bad sectors.
Boot from the repair disc again and instead of running startup repair select advanced options (probably bottom left on the screen) and then select the command prompt option.
In the command prompt window type ```
chkdsk C: /r
``` and hit enter. If that runs (it will take a while) and reports that it fixed errors you will need to keep running it again until no errors are reported. At that point you can try to reboot. If Windows boots then you're ok. If it doesn't boot you can then try startup repair again.
If chkdsk won't run, it may be that the disc is unwell and may need replacing.

Hopefully in the meantime somebody else may come up with a better idea.
