### Post by confused57 on 2007-03-15
Open the file, by clicking on the link, then select "File", then "Save As"...this should save a gedit file with the patch in your home directory.

Follow the directions on the website for compiling the driver, but you'd probably extract the driver...move the patch file to the source directory that was extracted...should be directions for applying the patch on the website.  Be sure to read the README file in the source directory.

You might want to read over this compiling howto:
[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)
