### Post by Shay Stephens on 2006-10-29
So there I was setting custom keyboard shortcuts.  Not really working right, so I tried a few different combos.  Then I noticed that the period key and the delete key were not working anymore....oops.

I was able to reset it by going to system - preferences - keyboard - layouts

I added my keyboard (u.s. english) and removed the old one.  Presto! my keys worked again.
