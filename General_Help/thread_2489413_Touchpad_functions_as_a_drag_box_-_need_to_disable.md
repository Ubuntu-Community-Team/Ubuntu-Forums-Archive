### Post by dark-wanderer on 2023-07-29
A week ago, I took my laptop in for repairs. The touchpad was acting like a button was constantly held down. I noticed that a corner of the touchpad was stuck, and being rather heavy-handed myself took it in to get sorted. It was reset and that should have been the end of it.

However, my touchpad is acting like the left mouse button is held down whenever it is brushed with the lightest touch. Testing the touchpad reveals that a touch still functions like a touch, and I can drag my cursor anywhere I want on screen. The problem currently is that just touching the pad is like holding down the left-click on a mouse. If I have a document open, it highlights text as I drag it. If I hover over the top of a window or over an icon, it starts dragging it around. If I wanted to do this previously, I would have needed to effectively double click and drag, but now it's doing this every time the touchpad is touched.

Is there any way to disable this, to treat touching the touchpad as just that, a single touch. I am currently using a USB mouse, which functions as it should, but it is awkward as I don't really have anywhere to set it up like I would a desktop/

---

### Post by MAFoElffen on 2023-07-29
See if the "Tap To Click" option is enabled in Settings > Mouse and Touchpad. Default is disabled.

### Post by dark-wanderer on 2023-07-29
Tried that.
Settings have been returned to default, but the only thing that's made a difference is completely disabling the touchpad there.
