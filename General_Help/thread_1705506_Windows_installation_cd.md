### Post by MARP1961 on 2011-03-12
If you are talking about Windows XP, you will need to borrow exactly the same version of Windows as the one that used to be on your PC. For example, if you had a OEM edition of Windows XP Home Edition on your computer then this is the only install disk that will work. If you try another version (for example a retail one or perhaps Windows XP Professional) it may install; but it will not be able to be activated with the Product Number you own. You may have to buy a new Windows Licence.
