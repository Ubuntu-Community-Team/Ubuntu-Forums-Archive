### Post by erginemr on 2008-02-20
Your monitor should be real good to support 75 Hz at 1280x1024, but if you are sure that it can handle it, this is what I would try:
```
        SubSection "Display"
                Modes           "1280x1024_75" "1024x768_75" "832x624" "800x600" "720x400" "640x480"
        EndSubSection
```
or
```
        SubSection "Display"
                Modes           "1280x1024@75" "1024x768@75" "832x624" "800x600" "720x400" "640x480"
        EndSubSection
```
if the first one doesn't work. 

If 60 Hz is too straining to your eyes, I would stick to 1024x768 @ 75 Hz.

### Post by erginemr on 2008-02-21
If this is your monitor:
[http://computing.kelkoo.co.uk/ps_12456775/114401.html](http://computing.kelkoo.co.uk/ps_12456775/114401.html)

then I believe there is no need for you to worry about the refresh rate. LCD (or TFT in your case) monitors are of higher technology than older CRT monitors. I have been using a CRT flat screen monitor at home, and a wide-screen LCD monitor at work. Using 60 Hz at home strains my eyes (so I use it at 75 Hz), but 60 Hz shows beautifully at work with no side effects. 

So long story short, you don't need to have 75 Hz for LCD/TFT monitors.
