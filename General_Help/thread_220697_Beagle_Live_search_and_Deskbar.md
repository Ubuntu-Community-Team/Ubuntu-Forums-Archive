### Post by chessforce on 2006-07-21
If you have both Beagle and Deskbar installed, but did not have the option to enable Beagle Live for Deskbar(displaying Beagle search results embedded within the drop-down window), you can try installing the python-beagle package from the Ubuntu repositories (e.g. `sudo apt-get install python-beagle`).  Upon installing, you should restart the Deskbar (`killall gnome-panel` would suffice if Deskbar was running as an applet on the Gnome panel), and then check the Deskbar's preferences to see if Beagle Live is available to be enabled.
