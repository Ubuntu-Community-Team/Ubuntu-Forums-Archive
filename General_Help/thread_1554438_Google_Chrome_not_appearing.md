### Post by supermario641996 on 2010-08-16
Did you get it off of ubuntu software center.  Its called Chromium Web Browser.  If thats what you got try updating your computer.   I had that same problem with a game that I installed through ubuntu software center.  When I updated my computer it appeared there.  When you open the update manager make sure you press the button called "check" or it will not show any files that need updating.  If it shows files then click install updates and restart your computer.  Good luck.
