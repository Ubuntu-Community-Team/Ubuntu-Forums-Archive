### Post by puppywhacker on 2008-12-01
Hi,

When you burn the ISO you have to make sure you "burn as image" and not as a file on the disk.

An other possibility is that the boot order on your computer is set to boot from harddisk, instead of DVD. On most computers you can press F11 to go to the boot menu as soon as the first screen appears or the computer beeps for the first time. The correct key should be printed on the first screen.

Goodluck
