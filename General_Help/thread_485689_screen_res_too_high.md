### Post by Wim Sturkenboom on 2007-06-27
Go to a console (e.g. using <ctrl><alt><F1> and edit the file /etc/X11/xorg.conf

```

Section "Screen"
Identifier "Screen0"
Device     "Card0"
Monitor    "Monitor0"
**DefaultDepth 24**
SubSection "Display"
Viewport   0 0
Depth     1
EndSubSection
SubSection "Display"
Viewport   0 0
Depth     4
EndSubSection
SubSection "Display"
Viewport   0 0
Depth     8
EndSubSection
SubSection "Display"
Viewport   0 0
Depth     15
EndSubSection
SubSection "Display"
Viewport   0 0
Depth     16
EndSubSection
SubSection "Display"
Viewport   0 0
Depth     24
[COLOR="Blue"]**modes     "1280x960" "1024x768"**[/COLOR]
EndSubSection
EndSection

```
It contains a section as shown above. Check the default depth  (marked in bold) and edit the associated subsection (line marked in blue) to refelct the capabilities of your monitor.

You can use vi, nano, pico etc to edit the file.

First make a backup.
```

sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup

```
