### Post by Ubun2to on 2012-09-28
Right click on the empty free space next to the open tabs.
Then, select "Use System Title Bar and Borders".
I found that out while using Lubuntu-using this makes it lose some screen real estate, but I have my settings synchronized across all my machines, so deselecting it affects my Ubuntu machines as well, causing them to revert to the old system.
I rarely have to use Lubuntu, so I don't really care about losing a little bit of screen real estate.
