### Post by chrisccoulson on 2008-06-30
AFAIK, it can install in Windows using [Wubi]("http://wubi-installer.org/").

Wubi installs Ubuntu like any other Windows application. The Ubuntu disk is created as a file within the NTFS partition, and Wubi modifies the Windows bootloader to allow you to load Ubuntu from this 'virtual disk'.
