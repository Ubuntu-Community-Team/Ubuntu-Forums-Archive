### Post by cscj01 on 2021-04-14
I am running Ubuntu Flashback 18.04.2 x64.

I recently upgraded from 18.04 to 20.04. Since the upgrade, I have an issue with typing the same letter consecutively, as in words like "feel", "better", "hollow", etc. When I type these words at a normal speed, the letter will only appear once, such as fel, beter, holow from the examples.  In order to get the letter to appear twice, I have to type the consecutive letters with a pause in between them.  I have tried adjusting the Repeat Keys values to no avail,  I have also tried to set them using gsettings without luck.

This is driving me crazy.  Does anyone have any suggestions to fix this?

---

### Post by cscj01 on 2021-04-14
I have resolved this issue.  There is a section in Settings>Universal Access>Typing called Typing Assist (AccessX) called Bounce Keys.  I turned this off and the problem is gone.
