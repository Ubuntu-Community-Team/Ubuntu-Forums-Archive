### Post by 67GTA on 2009-04-05
This one is a little confusing. It is not packaged like most. It has an emerald theme inside. Right click on the tar.gz and select "Extract". Then move the extracted folder to /home/yourusername/.themes. It should show up in the appearances window then. The ./themes folder is a hidden folder. You may have to tell nautilus to show hidden folders if you haven't already.
