### Post by por100pre1 on 2007-09-28
Extract it right-clicking it. Inside look for a folder with the theme name e.g. Glossy. Inside of this folder you must see folders like gtk-2.0, gtk, etc. Move the folder with the theme name inside the .themes folder in your user folder. Be sure to press Ctrl+H to see hidden files.

Hope it helps. :)

---

### Post by oldos2er on 2007-09-28
Go to System-->Preferences-->Theme, click "Install Theme," and point it to the tar.gz file.
