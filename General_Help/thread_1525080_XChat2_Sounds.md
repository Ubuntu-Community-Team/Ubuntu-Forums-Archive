### Post by AldenIsZen on 2010-10-24
I used Google's "filetype:" tool to locate the file once I knew what it would look like. I.E. "filetype: .gz xchat-sound". Just extract the contents of the file directly to the "/home/jason/.xchat2/" folder and then go to preferences to tell it what sound should play when.


[http://es.sourceforge.jp/frs/g_redir.php?m=jaist&f=%2Fmojax%2Fmojax%2Flatest%2Fxchat-sounds.tar.gz]("http://es.sourceforge.jp/frs/g_redir.php?m=jaist&f=%2Fmojax%2Fmojax%2Flatest%2Fxchat-sounds.tar.gz")[http://es.sourceforge.jp/frs/g_redir.php?m=jaist&f=%2Fmojax%2Fmojax%2Flatest%2Fxchat-sounds.tar.gz](http://es.sourceforge.jp/frs/g_redir.php?m=jaist&f=%2Fmojax%2Fmojax%2Flatest%2Fxchat-sounds.tar.gz)

Edit/ - Startup XCHAT ater you have the sound config file in the folder, otherwise it will not see it and then overwrite it with any changes you make on close, effectively leaving you with only what you changed.
