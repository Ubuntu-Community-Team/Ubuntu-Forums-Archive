### Post by seawolf167 on 2011-01-06
I'm not familiar with the Macbuntu theme, but perhaps you have the computers set to login automatically which bypasses the sign in screen.

To disable automatic login, try this:

System > Administration > Login Screen

then uncheck the login automatically line
