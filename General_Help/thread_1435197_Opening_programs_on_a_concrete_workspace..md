### Post by jahabdank on 2010-03-21
Hi Everybody,

when I am working, I always have certain applications on concrete workspaces. This helps me to find where is what, and makes my life easier when working.

The problem is that every day when I start my computer, and open a new program I have to drag it to another workspace (where I am used to have it). This might be only 5 sec of work but is quite annoying when one works with 20 different applications, and has to drag them back and forth every day, every time when the app starts.

So here is my question: is it possible to predefine Ubuntu in such way that it will always open e.g. Open office on workspace 4 and Gimp on workspace 2 rather then always opening on the active one? Additionally I would be interested that some applications when are opened could have the 'Always on visible workspace' option selected. Another thing that would be perfect if I could even specify exact window size (or fullscreen) and position of the application (e.g. Skype opens always with the top left corner at pixel 0x1000 on the workspace 3, and OpenOffice fullscreen workspace 2).

If I could predefine all my applications in that way I would save myself from doing a lot of repetitive work that I do every day when I reboot my pc.

If this is not possible to predefine the apps, maybe it is possible to do it when starting them from the command line? I would not mind that - I could write a bunch of shell scripts - that is all right. But predefining them inside Ubuntu would of course be much more handy.

I really hope this can be done. Please help! ;-)

Thanks is advance,
Jozef A. Habdank
