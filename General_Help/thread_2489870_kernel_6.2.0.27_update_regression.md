### Post by ian-weisser on 2023-08-12
"failing to load" is vague.

If you are trying to report a reversion, then...

1) Be very specific about the problem. Exact error messages. Clear descriptions of unexpected behavior. Exact release/flavor of Ubuntu. Exact version numbers. Logs. Troubleshooting you did. Steps to reproduce. If a developer cannot reproduce the issue in their environment, then they cannot fix it.

2) Check the bug reports at launchpad.net to see if anyone else has already reported it. Look at bugs reported to the 'linux' package. If so, subscribe to the bug.

3) If nobody has reported the bug, then make the first report. Just follow the prompts and add the information you collected.
```
ubuntu-bug linux
```
