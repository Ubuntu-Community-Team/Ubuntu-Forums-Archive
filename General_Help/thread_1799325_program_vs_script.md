### Post by seawolf167 on 2011-07-07
Use whatever you are most comfortable with. If I can write and debug a BASH script faster than I can do the same for a C++ program, I'm going with the faster way unless I have a specific reason to write it (learning maybe?) in a different language.

I've got a friend that writes .net programming all day, but couldn't do much more than the basics with anything but C# - just use whatever is easiest/fastest/most comfortable for you.
