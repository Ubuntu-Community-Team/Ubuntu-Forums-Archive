### Post by shakin on 2005-09-14
KMail does this. It also minimizes to the system tray nicely. AFAIK there are two ways to only check some mailboxes in KMail:

1. the Check Mail button has a dropdown where you can select which mailbox to check.

2. In the account options there is a checkbox if you want to include that account in the general mailbox check or not.
