It all depends on what you plan to use the machine for 80% of the time.  I tend to experiment and learn new apps as well as distros so in my case, I don't think so.  

One day, you may need a faster processor to run an app in ubuntu/linux and might wish you had gotten the faster one in the first place.

If you want to maximize the RAM and processor usage, you can always go 64-bit right now.

Again, if what you intend to do/use the machine is email, surfing, word processing, some music, some videos, chatting ..... then yes, even a 1.6ghz with 2 GB Ram will work wonders.  But, if you plan to edit photos/videos, run virtualization, try out some new games, I would go for a stronger processor.

Just a side note, I always believed that the bottleneck (when it comes to responsiveness) is in the FSB.

Happy Ubuntu-ing.
