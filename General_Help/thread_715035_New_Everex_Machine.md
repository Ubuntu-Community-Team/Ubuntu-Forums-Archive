### Post by gobbles414 on 2008-03-04
chough,

Welcome! Not to dampen your enthusiasm, but **the gPC is NOT a very good Linux machine.** The gPC uses a VIA pc2500 motherboard. Since VIA refuses to release decent Linux drivers -- or provide any code to open source developers -- you will not be pleased with graphics or stability on your gPC in Ubuntu. My recommendation is that you cancel your order and purchase a different PC. If you can give me an idea of what your needs are, I'd be happy to help you locate a more Linux-friendly computer.

EDIT: Also, you'll do much better with Linux if you use the real Ubuntu instead of the gOS version. I have done extensive testing of the gOS, and it still has a LONG way to go before it can compete with the real version of Ubuntu. The gOS is unstable, even on Linux-friendly hardware. And finally, Ubuntu 8.04 should be released sometime next month. The current version is Ubuntu 7.10.
