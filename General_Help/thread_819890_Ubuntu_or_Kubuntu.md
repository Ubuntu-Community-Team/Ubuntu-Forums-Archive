### Post by ChameleonDave on 2008-06-05
> **MasterNetra said:**
> Just out of curiousity what would really be better for gaming? Ubuntu or Kubuntu?

When gaming, you're going to be using fullscreen, and ignoring your entire desktop environment.  So, the desktop environment that you have becomes largely irrelevant.  If you are strapped for resources, you'll want to make sure it's something streamlined, but that's about it.

Ubuntu or Kubuntu?  Well, what you're really asking is GNOME or KDE.  The stable versions of these two DEs use about the same resources.  However, KDE 4 uses a fair bit less.

There are also even more minimalist environments and window managers you can use, such as Xfce or Fluxbox.  If you are really so worried about performance, then start X without starting a window manager at all.  To do this, choose "Failsafe" from the Sessions menu in when you log in.  Then, start your game from the command line.  Press Ctrl + Alt + Backspace to exit.
