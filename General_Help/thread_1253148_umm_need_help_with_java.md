### Post by scouser73 on 2009-08-29
> **austinmathis said:**
> umm i installed java sun but i still cant run java programs such as runescape how might i go about being able to make java work on things such as runescape

To install Java, go to **Syanptic Package Manager**, in the search box type **Sun Java** & select & install **sun-java6-bin**, **sun-java6-jre** & **sun-java6-plugin**. Now click **Apply**, a small dialogue box will appear, then click **Apply** again.

To test your Java is installed and working correctly, go to the [[COLOR="Red"]**Java Test Website**[/COLOR]]("http://www.java.com/en/download/help/testvm.xml")
