### Post by ryanhaigh on 2008-04-19
You should be able to run firefox by getting the beta from the mozilla site, extracting it and running the program directly (you don't need to 'install' it). As for kino I would assume that you need to compile the latest version although you may have issues given the age of the release you are running there may be dependencies not available to you. Other than suggesting you upgrade to gutsy (the last version of ubuntu that is going to support ppc apparently) or a recent debian ppc system I suggest that if you have trouble compiling kino, look at alternative editors, the only ones I know of are Lives and Cinerrella.
