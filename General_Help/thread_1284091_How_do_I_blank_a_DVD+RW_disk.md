### Post by hamamelis on 2009-11-03
I had same problem using K3b in Ubuntu (not Kubuntu). It will not erase DVD+RW. 
How I did it. In Nautilus open DVD then go up one level to /media. R click on cdrom0. There there are options including Erase, either fast or complete. This worked for me.
Also in K3b when Burning the DVD it was necessary to make sure that "Create image file" was clicked. Without this quite a few files were not burned. You can click "Remove image file" so that it is not retained.

I have abandoned Brasiro. There are just so many errors with DVDs. K3B seems to work fine and only requires c70Mb of KDE files to run. It is filled with options. (Except the one to blank a DVD+RW.

Ian
