### Post by sumguy231 on 2008-01-10
You can use any formatting utility, such as the one on the LiveCD, to wipe the partition. Once you've wiped the Ubuntu partition, you'll want to reinstall the Windows bootloader by booting the recovery cd and using the fixmbr command in the recovery console.
