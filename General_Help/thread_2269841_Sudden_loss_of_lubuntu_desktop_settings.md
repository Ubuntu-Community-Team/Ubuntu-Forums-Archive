---

### Post by Rex Bouwense on 2015-03-18
Go to main menu->Preferences->Desktop Preferences->Advanced.  Then uncheck Show menus provided by window managers when desktop is clicked.  
Go back to Appearance and uncheck use the same wallpaper on all desktops.  Close.
