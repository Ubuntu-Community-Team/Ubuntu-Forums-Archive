### Post by mcduck on 2010-03-14
> **Nevon said:**
> Oh. xD I had totally forgotten about that. However, will that change the language for the next person to log in (unless he or she changes it back)?

No, it's a per-user setting so if one user sets the language in login screen then he's changing *his* language (and setting his default) and it doesn't affect any other users at all.

Same with any other settings, like selecting the session, each user chooses his own language and desktop environment.

(If you use Ubuntu 9.10 or later you'll notice that these options don't even appear until you have selected the user or typed a user name)
