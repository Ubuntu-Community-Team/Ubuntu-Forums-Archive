### Post by smartboyathome on 2008-05-11
I use E17 and love it. Its the best light Window Manager (it isn't a DE) that I have used. It is still unstable though (never was an alpha release :(). Use [this guide]("http://ubuntuforums.org/showthread.php?t=546746") to install it. It may not be for you though, as it take some configuring to get it how you like it.

EDIT: Technically, it can run with GNOME since GNOME is a desktop evironment and E17 is a window manager, but E17 is better alone imo, as all you need from GNOME is the settings manager so you can use a GNOME theme which goes good with your E17 themes, which you can find [here]("http://get-e.org") and[here]("http://e17-stuff.com").

> **Kdar said:**
> Oh.. so..Window Manager is like add-on to DE?

What other Window Managers are out there? for Gnome.. or KDE?

Window managers take care of allocating windows for apps. Window decorators draw window borders and widgets that let you manipulate windows. Window environments typically pack window decorators, and desktop managers typically pack window managers.

GNOME uses Metacity/Compiz
KDE uses KWin
XFCE uses XFWM

And then there are some independent window managers like E17, Fluxbox, Openbox, Blackbox, ... These can be used with desktop environments as well.
