### Post by simple_impulse on 2016-01-23
If your Firefox crashes in Facebook often, here's a trick you can try:

1. In Firefox address line, write about:config

2. Now comes warning, click "I'll be careful, I promise"

3. Search media.fragmented-mp4.enabled and set the value to false (double-click)

4. Restart Firefox (all instances, that means all open Firefox windows!)

Why it works, I don't know. But I found this tip and thought it would be useful to share.
