### Post by Impavidus on 2013-12-10
From a live disk? I think the user name is lubuntu, maybe ubuntu. You'll see when you open a terminal, it says something like lubuntu@lubuntu:~$. That first word is your user name. It shouldn't really ask for a password, because on a live disk you're root automatically, but if it does, just hit enter. The password should be blank (that is, an empty string).
