### Post by steveneddy on 2011-02-05
I run iTunes in with Window 7 running in Virtual Box.

It just works better this way and it feels like a native install. As long as you get the idevice libs installed on Linux and you tweak the USB settings in VB then you will be good to go. Plug in the iPhone while 7 is running and it hooks up automagically and starts iTunes.

It is difficult to run iTunes in Ubuntu even with WINE. It's not that we don't want it to, it's just that iTunes is difficult to play with when not in a native Windows environment.

If you need to sync your music then use Amarok or Banshee - even RythmBox has some functionality here.

If I need my pics - I just plug the iPhone in and I find all of my pics in the DCIM folder if I'm on my Linux desktop.

Hope this helps.
