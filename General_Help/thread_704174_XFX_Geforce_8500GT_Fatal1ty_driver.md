### Post by PmDematagoda on 2008-02-22
You can install the Nvidia driver manually from [here]("http://www.nvidia.com/object/unix.html") and installing it using the given instructions, or you could use an application called [Envy]("http://albertomilone.com/nvidia_scripts1.html") to install it for you.

You also could install the Nvidia driver by going to the Restricted Drivers Manager in System>Administration>Restricted Drivers Manager and enabling the Nvidia driver.
