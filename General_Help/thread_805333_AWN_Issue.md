### Post by moonbeam on 2008-05-24
> **Exsecrabilus said:**

Open awn manager:

Under General/General  you probably have the following 3 checked.

Autohide bar when not in use.
Keep below Maximized windows when not in use.
Maximized windows don't cover the bar...

Removing one of those options should resolve the issue.



Alternately if that doesn't work there is a fixed listed here:

[https://bugs.launchpad.net/awn/+bug/222300](https://bugs.launchpad.net/awn/+bug/222300)
