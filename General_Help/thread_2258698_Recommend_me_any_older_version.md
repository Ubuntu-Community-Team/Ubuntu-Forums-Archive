### Post by grahammechanical on 2014-12-29
> [COLOR=#000000]That maybe because the latest version of Ubuntu uses Unity 3D and my 64MB video memory can't handle it...[/COLOR]

That would be accurate, especially is it shared memory being taken from system RAM. If the video adapter is unable to run Unity 3D then a fall back open source video driver is used. It is called llvmpipe. It gives an approximation of Unity 3D but it is not very fast even on hardware that can run Unity 3D.

> [COLOR=#333333][FONT=Ubuntu]* 3D Acceleration Capable Videocard with at least 256 MB[/FONT][/COLOR]

[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

You do not necessarily need an older version of Ubuntu just one of its flavours, such as Xubuntu or Lubuntu.

Regards.
