### Post by fooman on 2010-05-29
install the "kubuntu-desktop" package.  after that,  when you get to the login screen....just click once on your user name,  then at the bottom of the screen,  choose "kubuntu" from the menu.
 
install it through synaptic,  or just open a terminal and type or copy/paste:
 
```
 
sudo apt-get install kubuntu-desktop

```
 
hope that helps.
