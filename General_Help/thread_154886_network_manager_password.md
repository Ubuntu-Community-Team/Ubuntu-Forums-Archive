### Post by florizs on 2006-04-04
As for all the administrative settings, they are secured with the SUDO password.
this means you should type your own login password in the password screen or if you'd like to enter commands in shell as root you should type sudo before the command. and type your password in the dialog box.

"sudo" means superuser do. "sudo" will prompt for "Password:". Please specify user password
