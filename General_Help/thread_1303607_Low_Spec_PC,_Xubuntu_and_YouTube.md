### Post by mixmaster on 2009-10-28
From the Xubuntu page at [www.ubuntu,org:](www.ubuntu,org:)

> 
System Requirements

Xubuntu is available for PC, 64-Bit PC.

CDs require 128MB RAM to run, or 192MB RAM to install. Desktop install requires at least 1.5GB of free space on your hard disk. 


So you'll need the extra RAM if you want to actually install it.

You may be better off with one of the ultra-lightweight distributions

First thought when I read the specs is: wow, that machine must be old.

Um... not sure of the Xubuntu specs, but with 128MB RAM, you're not going to be able to do much.  Although it **might** work, I believe you would have better luck just scrapping this old machine and getting a new one.  As a tech who works with Windows machines (not much different in terms of hardware), I'd advise just getting rid of the thing.

However, from my google searches, Xubuntu needs more than 128MB ram, so add an additional 128MB ram (if you can).  However, your processing power for the internet will be severely diminished due to the processor's speed, and the age and speed of the RAM (any 128MB RAM sticks are extremely slow in comparison to newer, more recent, RAM technologies).

**SIDE NOTE: Not Necessary For You To Read**
If you're using something with 128MB RAM, I'm assuming that its (ALERT: TECHNO BABBLE) DDR1, in which case it's most likely not worth the salvage operation (in my opinion).

Although I did get an old Compaq tower to work with Ubuntu 8.10, it took 3 sticks of 128MB RAM and two (approx)10GB IDE Hard Drives to make the system work (in dual boot config with XP [don't ask me why I put XP on the thing, a client wanted it that way]).
**END SIDE NOTE**

Hi, I have that exact same computer (Latitude Cpx). The only difference is that I've upgraded my ram to 384mb. 

I did test Xubuntu at one point; it was slow but usable. I had better luck with SliTaz, an extra-light distro. Regardless of the distro, streaming flash like Youtube is unwatchable. It's better to download the video first, then play it from your hard drive. Even then, it is not a pleasant experience. 

My understanding is that Flash, being a proprietary codec, is not well-optimized for Linux, especially on older machines. The Latitude Cpx is an okay machine for checking email, surfing Ubuntu Forums, word processing with Abiword, etc. It is a bad choice for multimedia.
