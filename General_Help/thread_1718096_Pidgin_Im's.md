### Post by Krytarik on 2011-03-30
You can use its "Psychic Mode" plugin for that, it's included by default, enable/configure it via "Tools -> Plugins".

It makes a conversation window pop up, if a buddy starts typing a message to you, and there not already a window open with those buddy.

It depends on if the corresponding client sends a notification that a message is being typed, the same feature you see in the conversation window.
