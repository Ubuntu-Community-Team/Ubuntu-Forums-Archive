### Post by DaimyoKirby on 2012-02-29
I've ran into a strange problem:
I have a Dell Dimension 2350 that has two hard drives connected to it, one with Windows XP SP3, and the other with Xubuntu 11.10.
Whenever I boot the computer, everything acts normal, with the Grub boot screen showing up. Likewise, when I select Windows, it shows the Windows XP loading screen, like normal. However, once it's done loading, the loading screen will go away, and then the monitor will remain blank. It just won't show anything.

Can anyone offer me advice or insight into this?

**EDIT:** I finally got it to restore itself to a previous date, which lead me to the root of the problem: While my computer initially displayed everything through the graphics card I had installed, it then switched to the integrated graphics once booted.
Curse you, Radeon 9200! :-D
