### Post by strabes on 2007-09-03
Just so you guys know, wine is not an "emulator." It's an implementation of windows libraries so that windows programs can run natively on linux. As for the original question, I would recommend first trying wine, but if you don't have luck with that (which is likely) just use virtualbox.
