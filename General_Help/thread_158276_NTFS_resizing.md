### Post by paulipe on 2006-04-10
Well, I had resized my windows partition from the Ubuntu install CD. And I actually shrunk the windows partition to the wrong size. It asks to what size do you want to shrink the partition or something to that effect. What it actually means is, what do you want the new partition size (created from the free space) to be? So if you want to make a 60GB NTFS partition into a 20GB NTFS partition. Just answer 40GB to that question.
