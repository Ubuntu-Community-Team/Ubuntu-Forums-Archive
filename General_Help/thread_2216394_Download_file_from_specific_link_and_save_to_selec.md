### Post by ian-weisser on 2014-04-14
The 'wget' command line utility, included with every install of Ubuntu, can retrieve links to anyplace you wish. It can can also mirror entire websites, if you wish. If you need more specific download control, it's easy to script.

See 'man wget' in a terminal.
