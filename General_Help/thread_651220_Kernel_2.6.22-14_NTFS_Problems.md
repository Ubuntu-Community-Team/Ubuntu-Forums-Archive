### Post by szaka on 2007-12-28
Remove the unused EVMS package which needlessly locks the volume. This is a fairly common Ubuntu installation/upgrade problem.

If you don't have EVMS installed then some other kernel subsystem locked the volume (LDM, MD, DM, etc). You can find it from the logs under /var/log
