### Post by SammerHammer on 2008-08-26
TYPE THIS IN A TERMINAL: sudo aptitude install simple-ccsm
OR, OPEN SYNAPTIC PACKAGE MANAGER, search for simple-ccsm, select it and Apply Changes.

After the installation, right click desktop, Change desktop background.
When the window opens, click on the Desktop effects tab, then at the bottom, there will be an option for custom preferences; click that, and in that window you can change many many options about your graphics.

If you want even more tweaking, click System>Preferences>Advanced Desktop Effects Settings
