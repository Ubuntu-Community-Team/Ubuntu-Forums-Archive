### Post by sujoy on 2008-07-10
you need to have these lines at the end of the xorg.conf file, 

Section "Extensions"
	Option "Composite" "Enable"
EndSection

### Post by Rocket2DMn on 2008-07-10
Can you please post your xorg.conf file for us please?
```
cat /etc/X11/xorg.conf
```
To edit the file, run
```
gksudo gedit /etc/X11/xorg.conf
```
You should just be able to change the value of the Composite line from 0 to 1, save and close, then restart X with CTRL+ALT+BACKSPACE.
