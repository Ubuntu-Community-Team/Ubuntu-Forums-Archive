### Post by ajgreeny on 2015-06-14
Is the .iso file you are trying to use definitely a good one?  Compare the md5sum of the iso with the listed values at [UbuntuHashes - MD5sum]("https://help.ubuntu.com/community/UbuntuHashes") 

How is the USB formatted?  It must be fat32, as far as I'm aware, so check that as well, and also try using unetbootin, available as an application for Windows and Linux to "burn" iso files to USB.
