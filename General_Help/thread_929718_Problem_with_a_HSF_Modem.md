### Post by toni_lx on 2008-11-12
**SOLUTION:**
It's really simple. It’s only a matter of writing “Stupid Mode = yes” in /etc/wvdial.conf.

[COLOR="Blue"]THAKS A LOT FOR YOUR INTEREST.[/COLOR]
By the way, in Ubuntu.10 there’s no need of installing libc6-dev*.deb, patch*.deb, alsa-driver*.deb. Only hsfmodem*.deb, and that’s all!

**[COLOR="blue"]Steps:[/COLOR]**
**[COLOR="blue"]In /etc/wvdial.conf:[/COLOR]**
[Dialer Defaults]
Auto DNS = yes
Init1 = ATZ
Init2 = ATQ0 V1 E1 S0=0 &C1 &D2 +FCLASS=0
ISDN = 0
Modem Type = Analog Modem
New PPPD = yes
Modem = /dev/modem
Stupid Mode = yes
Carrier Check = no
;Baud = 460800
Phone = My telephone
Username = My username
Password = My Password

**[COLOR="blue"]In /etc/ppp/options:[/COLOR]**
asyncmap 0
noauth
crtscts
lock
hide-password
modem
proxyarp
lcp-echo-interval 30
lcp-echo-failure 4
noipx

**[COLOR="blue"]$ sudo chmod a+x /usr/sbin/pppd[/COLOR]**
$ ls -la /usr/sbin/pppd

**[COLOR="blue"]$ sudo chmod a+rw /etc/ppp/pap-secrets[/COLOR]**
