### Post by PmDematagoda on 2007-09-21
You cannot just run Windows executables and expect them to work under Linux, you need to use emulation software like Wine or Cedega, which is optimised for running windows games under Linux. There are Linux games as well but they are usually not as advanced as Windows ones.

Though you have to buy Cedega for a small price of about $15. Wine can be gotten for free.
