I have trouble with Oanda charts

They worked great on 9.10 

But now on 10.10 64bit version the applet says: 
start: applet not initialized 

I tried to install sun java jre thinking this might be the problem ?

- solved my own problem here:
I uninstalled all Icetea and JDK open stuff from software center
Then installed sun-java and sun-jre plugin stuff from software center

Simple fix and icetea appears to interfer with the jre plugin if you have them both installed.

Anyhow all is well now with Oanda charts as normal
