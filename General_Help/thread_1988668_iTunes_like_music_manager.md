### Post by Zvlwab on 2012-05-28
I know both Banshee and Amarok have this functionality.
Banshee is a GTK program, so if you use Ubuntu or Xubuntu you'll probably like that one more. I also personally use Banshee myself. :)
You can change the way the files should be sorted in preferences. If none of the defauls settings are good enough, you can entirely customize them using gconf-editor.
