### Post by warfacegod on 2010-04-18
Next is applying your themes.

The Theme and Pointer theme I posted will install themselves if you right click an area of your desktop> Select Change Desktop Background> drag and drop it into the Theme tab. The Icon theme will work the same way but you'll need to unzip it first. Just double click it then extract. You'll want the tar.gz file.

Once you've done all of that, again in the Theme tab of Appearances, click SlicknesS-Black Theme and then click Customize.

In the new window> Controls tab> Select Dust> Icons tab> Select black-red> Pointer tab> Select Anitech_Blue.

Now for the Colors tab:

The first number is for the Background and the second is for text.

Windows:          #2E2E2E    #FFFFFF

InputBoxes:       #171717    #A19D9D

Selected Items:   #565352    #FFFFFF

Tooltips:         #96968D    #323232

### Post by warfacegod on 2010-04-18
Here are the different transparencies I use for the windows.

System> Prefs.> CompizConfig Settings Manager> Check Opacity, Brightness and Saturation then click its button> Opacity tab> click New and make a new entry. Obviously there is no point to making a new entry for software you don't use so just create the ones you'll need (Tooltip... is for drop down menus). If you don't use Swiftfox as a web browser, just switch Swiftfox with the name of whatever browser you do use. By the way, if you do use Fire/Swiftfox, the _**Night Launch**_ theme add-on goes very well with this set up. Same thing for your media player. I use Amarok 1.4 because I think its the best *and* I can make it look like this.

### Post by warfacegod on 2010-04-18

Make sure "Use colors from system theme" is unchecked.

Text color: #DDDBDB

Background Color: #000000

Now go to the Background tab.

Check Transparent Background and set the slider about 3/5 of the way towards the right.
