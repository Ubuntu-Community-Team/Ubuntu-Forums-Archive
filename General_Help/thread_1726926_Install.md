### Post by bodhi.zazen on 2011-04-11
First, I highly suggest you take a look at Virtualbox. Virtualbox will allow you to test Ubuntu without installing it directly.

Second, boot the CD and make sure your hard ware works.

Then, when you are ready, install. You can install with wubi or a standard install. I suggest a standard install.

When you do a standard install you need to resize your windows partition and make new partitions for Ubuntu.

See:

Wubi : [https://help.ubuntu.com/community/Wubi](https://help.ubuntu.com/community/Wubi)

[https://help.ubuntu.com/community/GraphicalInstall](https://help.ubuntu.com/community/GraphicalInstall)

### Post by poodoopealeoaph on 2011-04-11
Though the guy above me is right, to make it simpler, just go to [http://www.ubuntu.com](http://www.ubuntu.com) or kubuntu, xubuntu, lubuntu... really which ever one you want to try and just go to the downloads page. It is fairly simple. Just select if you want x86 or x64 and download the iso. After that is downloaded you can either burn that to a cd or you can read the directions right below the download that shows you how to install the iso on a flash drive (must be above 2gb). Then just boot your computer back up and it should load the ubuntu installer before it gets to loading windows. If it doesn't load up then you just need to mess with your boot sequence in your BIOS... anyway though, once it starts up, like the guy above me said, just hit "try ubuntu" and make sure that everything seems to work. Don't expect it to be perfect though because it will be loading everything off of the boot disk you have. Then you can just select the desktop icon in the tryout version that says to install ubuntu. Then when you get into the installer you can just select the radio button that says "install along side other operating systems". This will automatically split the partitions up evenly and make your hard drive half windows and half ubuntu. Then whenever you start up it will ask you which one you want to boot in a dos menu.... If you have any other questions once you get it installed that far you should feel free to post your questions in this thread. I'll be checking back in on it. I've reinstalled and set up ubuntu's of every kind on my laptop probably about a good 100 times on the past year so I can give you some advice on things to get right out of the gate when you get it going. By the way though, I don't reinstall every couple weeks because I have problems though. Everything works perfect, I just can't make up my mind because every version of ubuntu is fantastic!
