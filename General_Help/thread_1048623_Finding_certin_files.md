### Post by mb_webguy on 2009-01-23
Ditto.

Firefox has certain configuration/information pages (some useful, some humorous) internal to the browser, which you access by typing "about:*something*" in the address bar.  A few common "about:" pages are "about:config" (which provides advanced configuration options), "about:plugins" (which provides information about the currently installed plugins), "about:cache" (which provides information about your browser's cache"), "about:robots" (which provides some interesting information about robots), and "about:mozilla" (which displays a passage from "The Book of Mozilla").

They used to have "about:kitchensink" (which displayed [this page]("http://www.mozilla.org/docs/web-developer/samples/kitchensink.xml")), but it's been removed from the current release.
