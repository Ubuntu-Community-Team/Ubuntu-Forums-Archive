### Post by mörgæs on 2017-05-14
We are thankful that you care about reporting bugs but if you are running an old version the developers are less likely to be interested.

Best you can do is to try to trigger the bug in the development release, for example in a live boot. If it can be recreated then developers are happy for a report, if it can't then we are all even happier.
