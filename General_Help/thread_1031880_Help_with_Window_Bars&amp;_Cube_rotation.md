### Post by tuxxy on 2009-01-05
Check if you video driver is still activated then check that you have desktop effects enabled again in the appearance settings.

Press ALT + F2 and enter the following command to reactivate the emerald window themes, also you can add this command to your sessions to prevent running it every bootup

```
emerald --replace
```
