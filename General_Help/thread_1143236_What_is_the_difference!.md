### Post by mb_webguy on 2009-04-29
If you're used to Windows, the differences between Ubuntu releases are going to be somewhat underwhelming.  Windows gets a complete overhaul with each release.  Vista isn't very much like XP at all, and XP was quite a bit different from 95.  But with Ubuntu, releases are incremental.  Software gets updated, and some alterations are made to appearance and functionality, but the differences aren't something you usually notice until you've used for a few days.

The differences between Jaunty and Intrepid are greater in my opinion than those between Intrepid and Hardy.  Boot time was cut down considerably.  It has better hardware support, especially for wireless cards.  There are quite a few visual improvements.  And as you noted, quite a few applications were updated -- though whether the new versions of these applications are improvements is a bit debatable.  

OpenOffice 3 is certainly a welcome change, but I don't think the same can be said of Amarok 2, at least in my opinion.  VLC and Gnome-terminal have a few regressions in the current repository versions.  I also don't agree with the change to Update Manager.  Fortunately, these "problems" can be fixed either by adding PPAs (in the case of VLC and Amarok), downgrading to the Intrepid packages (in the case of gnome-terminal), or changing configuration settings (for Update Manager).

Overall, I think Jaunty is a step in the right direction, and an improvement over previous versions of Ubuntu.  And one of the great things about Linux is that it's highly configurable, so that if you don't like a certain change, you have numerous ways to fix it.
