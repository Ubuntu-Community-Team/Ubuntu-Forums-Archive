It uses the RAM memory to install applications while running from the Live CD (I wouldn't use the word 'trial' for this since Ubuntu is free and open-source). I could run it for two weeks because I did not have a hard disk and I could see movies and browse the internet and listen to music with only 1 GB RAM.
