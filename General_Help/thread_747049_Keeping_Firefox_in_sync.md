Hey all,

on my desktop system, I run Windows Vista 64 bit (for gaming) and Ubuntu (for actually getting work done). On both systems, I use Firefox as my GUI browser. Ubuntu has full read/write access to my Vista partition, while Vista has read-only access (for obvious reasons). 

I'd like to keep my 2 instances of Firefox in sync. Both should contain exactly the same configuration and exactly the same add-ons. 

I realise that that information must be stored on the Linux partition (because of my partition access situation). 

Any tips on how I can achieve that?

### Post by LaRoza on 2008-04-06
Run the Windows version portably in Wine.

[http://portableapps.com](http://portableapps.com)

[http://portableapps.com/news/2008-03-25_-_firefox_portable_2.0.0.13](http://portableapps.com/news/2008-03-25_-_firefox_portable_2.0.0.13)

You should store it somewhere where Linux and Windows can access it. A flash drive, or another partition will work, or it can be on the Windows partition.
