### Post by solar george on 2010-10-09
Most translations are done using gettext ([http://en.wikipedia.org/wiki/Gettext](http://en.wikipedia.org/wiki/Gettext)) you'll need access to the original source packages to carry out the translations manually.
I'd suggest that if you want to find out about how the translations system works at a low level you should pick an application that hasn't got any hindi translations but does have translations for other languages to work on. 
If you want to see the work accepted into ubuntu you'll need to do the translations through the launchpad.net web interface, this is also the best way to get involved with translations if you don't care about how it works in too much detail.

p.s. You'd be best to update the title of your post to make it clear you're asking about translations as that will make it more likely that you'll get replies from people who actually do translations themselves.
