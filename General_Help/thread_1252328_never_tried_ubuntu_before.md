### Post by fela on 2009-08-28
I suggest you install it. Don't bother with livecds as these will only give a small impression, and first impressions are almost never right when it comes to operating systems and other computer programs. Plus you shouldn't think that your hardware won't be supported if it doesn't get auto detected by the livecd. I've had computers that wouldn't boot with the livecd, but once installed ran perfectly.

The only way to see if an OS meets your needs and whether or not you like it, is to try it out for a month or so. Don't give up trying if it seems a bit daunting at first. And don't think of microsoft as a fall-back operating system, if you completely forget about microsoft then you can safely learn alot about Linux and how to use it without all the engrained computer-using-styles that MS have taught you. And remember, for Linux to be better than windows it, by way of definition, has to be different. So if things don't make sense at first due to using MS for a long time, persevere and act like you've never used a computer before. Linux is surprisingly easy to use, contrary to what many people say. I have anecdotal evidence at least: when I plonked an Ubuntu installed computer (installed by me :)) in front of my mum who'd never used MS before, she instantly started to pick up on how to do everything, its layout, etc. and she really liked it.

People that consider themselves power users because they've used Windows for a long time and know their way round it like the back of their hand, are the people that are most likely to have a hard time using something different, due to their hard-engrained patterns of usage. When they have a hard time using Linux (or some other different OS), they think 'if I can't use Linux and I'm a power user, how the **** is a newbie gonna be able to?'. It's actually the other way round, people that are new to computers in general are most likely to be able to easily learn how to use Linux. You can think of it like learning a language. When you're a baby and you've never learned any language, it's really easy and completely effortless learning the language spoken by everyone around you. But years later, learning a second language by studying it is much much more difficult.

I hope you have a smashing time with Linux and that all goes well for you.
