### Post by az on 2007-10-17
If you run "alpha' software, it may work, but will probably cause you problems - the only point to releasing alpha software is to identify problems to fix.

"Beta" software works, but you still run the risk of breakage/data loss.

"Release Candidate" means that all major bugs in the Beta software has been fixed (or the feature outright removed).  You should not experience any risk of data loss running RC software.  You should not be surprised, however, if you find small bugs.
