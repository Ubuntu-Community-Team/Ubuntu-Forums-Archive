### Post by pat4 on 2016-11-24
Can anyone help sort out an annoying problem with language selection.

It appears that two languages are installed on this computer. En1- English (UK) and En2 - English (US).

My default setting should be En1 English (UK) and I am aware of how to set that through Language support in Settings and also by the use of the selection icon at the top right of my screen. 
My problem is that no matter how many times I select the UK version it changes randomly to the US version almost daily. This is annoying when typing as keyboard settings are different. Usually I don't notice the change until I have (mis)typed a document.   I know that it is easy to select the correct setting but ideally I would like to solve the problem once and for all by removing the US version. 

I have searched for solution and there are plenty of sites telling me how to install additional languages and select from them. Nowhere can I find any help in removing specific languages other than by unchecking boxes in Language Support.
In install/remove languages screen in Language Support (Settings) the only selection is "English". There is no specification for US or UK English so it is not just a case of unchecking a box in this case.

Can anyone help me remove US English from this computer please?

Thank you in anticipation.

Regards, Pat.

### Post by howefield on 2016-11-24
Try going to System Settings > Text Entry and highlight the language you wish to remove and press the minus button.

PS. The keyboard short Super + Space toggles between the languages, perhaps that key combination was the trigger to the "random" change in input.
