### Post by aysiu on 2007-02-11
I believe printing to postscript is in by default. I'm not sure how to restore it if you've deleted that capability.

For editing PDFs, you can use KWord, but the formatting may get a bit screwed up, depending on how complicated the original PDF was.

There are some other PDF programs that allow you to combine and rotate pages--*pdfwrap*, for example.
