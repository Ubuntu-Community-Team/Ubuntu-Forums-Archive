### Post by userundefine on 2006-05-15
If you have the "cdemu" kernel module checked in your kernel, you should be able to mount it.  I don't have it in my kernel by default (nor have I recompiled), so I haven't tested it.

Or you can [convert it to an ISO]("http://he.fi/bchunk/").  This program is in universe repos.
