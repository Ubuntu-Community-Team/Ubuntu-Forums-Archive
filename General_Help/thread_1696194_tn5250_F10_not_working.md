### Post by Funkey Monkey on 2011-02-27
I am currently using tn5250 in Ubuntu.

I had a problem with the F10 key not working in the program, but opening up a drop down menu for the GNOME Terminal 2.32.0

To fix this you need to disable the Terminal's Shortcut Keys.

EDIT > Keyboard Shortcut... > Then untic the options.
