### Post by musicinmybrain on 2007-05-01
Yes, but exactly which programs depend on which programming languages you're interested in using.

You have a Python interpreter already installed, and probably one for Perl as well.

For C and C++, you'll want to install and use GCC or G++, respectively. It's probably easiest to install build-essential; that should automatically bring in gcc, g++, libc6-dev, and the Ubuntu/Debian-specific dpkg-dev. You'll probably want an IDE if you're doing much development. Eclipse is a popular choice for that, although there are others.

There are plenty of other options for other languages.
