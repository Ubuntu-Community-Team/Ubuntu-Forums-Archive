### Post by Foxheadz on 2011-03-17
To get emerald to start working you have to start it. You can do this by first pressing alt+f2 and then in the box that pops up typing emerald --replace and pressing enter. To find themes i would suggest going to [http://browse.deviantart.com/customization/skins/linuxutil/](http://browse.deviantart.com/customization/skins/linuxutil/) or [www.gnome-look.org](www.gnome-look.org)

Also if your gonna use emerald at all times i would also recommend adding emerald --replace to the startup applications. System>preferences>startup applications pressing add and under command put emerald --replace.
