### Post by SuperFreak on 2013-09-16
I think you will have to install the Thunderbird that comes with Ubuntu. However you should be able to transfer your, address book, email folders, calendar etc to the new install. If you go into Account Settings and look at Server Settings it should indicate where your local directory is. You should be able to copy the   folder where your emails are stored (mine is called Thunderbird Profile TODAY) Within TB on you Win system backup the address book (Tools/Address Book/Tools/Export), Lightning also has back up settings. Transfer all of these folders to an external drive and copy into the appropraiate folders in your Ubuntu install (usually emails are stored within the Home folder, in .Thunderbird/(preceded by random letters)default/mail.

### Post by CeejRab on 2013-09-16
Hello Dennis_Guerra, 

Sad news: No, unfortunately you can't run your current thunderbird in Ubuntu mainly because Ubuntu Linux can't run the same executable files as windows. Thunderbird's main application file on windows is an .exe which won't run in Linux (unless you use wine, but that's a whole different story). 

Happy news: You can definitely back your data up however, and import it to the thunderbird in Ubuntu. Mozilla's forums and tutorials likely have a detailed description of how to do this. Some other members have listed links above which should be helpful.

All the best to you and your switch to Ubuntu. I think you'll find the switch easy and very rewarding :)
