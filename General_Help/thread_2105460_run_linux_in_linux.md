### Post by Warren Hill on 2013-01-16
To run one OS inside another you need to create a virtual machine.  There are a few to choose from see here: 
[http://www.cyberciti.biz/tips/linux-virtualization-software.html](http://www.cyberciti.biz/tips/linux-virtualization-software.html)
