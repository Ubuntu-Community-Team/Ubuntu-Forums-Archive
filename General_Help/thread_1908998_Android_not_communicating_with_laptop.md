### Post by duke.tim on 2012-01-16
If the phone charges via the vbus pin on usb, It is possible that it will receive power even if the data connection is inhibited.
The easiest way to check is to see if you can establish a data connection via another computer.

There is a setting in android to turn on the usb data connection. Double check to ensure that the usb data connection is enabled.

### Post by Mark Phelps on 2012-01-16
When I connect my Android phone via USB cable, a menu pops up asking what to do.  If I don't select anything, it defaults to Charge Only -- meaning the PC will not see it.

If your phone should display such a menu and is not, there could be a fault in the USB jack on the phone.  I had to replace my Android phone because the USB jack went bad.
