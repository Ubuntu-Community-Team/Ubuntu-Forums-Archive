### Post by darolu on 2010-08-06
No it is not more complicated than Ubuntu, actually in a way is easier if you are part of the target 'market' of it.

It's Ubuntu aimed to the educative market; it is like the regular Ubuntu but it comes with other software pre-installed, like a puzzle game for building molecules out of isolated atoms called Atomix or Gcompris a collection of educational games for children.

You can of course install all this apps to your regular Ubuntu edition too, but getting edubntu saves you the work of doing it.

### Post by ubunterooster on 2010-08-06
Edubuntu is a partner project of Ubuntu Linux, a distribution  suitable for classroom use. The aim is that an educator with limited  technical knowledge and skill will be able to set up a computer lab, or  establish an on-line learning environment, in an hour or less, and then  administer that environment without having to become a fully-fledged  Linux geek.
[http://distrowatch.com/table.php?distribution=edubuntu](http://distrowatch.com/table.php?distribution=edubuntu)
