### Post by GroogFish on 2011-03-20
I am currently running on the latest version of Fedora. I originally chose fedora so I could learn more about the Redhat based distros because I was planning on setting up my own server on a Redhat system. But I'm going to be pushing that project off a bit so I'm considering getting a different distro. Fedora serves me well but why I'm looking for something else is because I do a lot of freelancing in developing websites and it seems Ubuntu would be better for this. Most of my website use the basics- html, php, mysql, javascript, etc. But I really want to get into developing multimedia rich pages because that's what a lot of clients are looking for in pages. With higher bandwidth it's now ok to have flash heavy applications. But most of these multimedia formats like MP3 and SWF aren't "free" so Fedora doesn't offer any support for them originally. And trying to install them is really like pulling teeth (I still can't get flash or java to work on Chrome. Can't get java to work on FF). And If I'm trying to run a business I really don't want something that will cause me a day and a half to fix with my limited Linux knowledge everytime a client wants to use a different sound file type. 

So my question to the Ubuntu community: would you say Ubuntu offers more support in this field? Would it be a wise choice in my circumstances to switch to Ubuntu's more non-free software support?

WORST case scenario is I'll have to install Windows XP. It's just not the same as the Linux experience :(

### Post by SavageWolf on 2011-03-20
On an Ubuntu system you can install "ubuntu-restricted-extras", which should install flash and the needed codecs, if you want a standalone FP, download the projector here: [http://www.adobe.com/support/flashplayer/downloads.html](http://www.adobe.com/support/flashplayer/downloads.html)
