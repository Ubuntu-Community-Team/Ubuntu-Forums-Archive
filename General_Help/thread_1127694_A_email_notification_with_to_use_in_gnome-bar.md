### Post by Quintilian on 2009-04-16
Pidgin can notify you of emails if you set it to use you @hotmail.com (msn) account. also, evolution can be set to use hotmail, and that has notifications too. hotmail recently added pop support, so this is why that option is relitively newish ~2-3mo i think.
_______hotmail via evolution____________
POP server: pop3.live.com (Port 995)
POP SSL required? Yes
User name: Your Windows Live ID, for example [email]yourname@hotmail.com[/email]
Password: The password you usually use to sign in to Hotmail or Windows Live
SMTP server: smtp.live.com (Port 25)
Authentication required? Yes (this matches your POP username and password)
TLS/SSL required? Yes
