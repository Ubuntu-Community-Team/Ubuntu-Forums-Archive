### Post by wormser on 2007-12-26
It is going to depend on which drivers.  When I installed my wireless card it asked do you want to install from disk or online.  It gave the address where it was going to download from.  I copied the address and downloaded it with Firefox.  I believe sometimes the restricted drivers are in packages.  You could go into synaptic and select the package.  When you select apply check the box to download package only.  What specifically are you trying to get?

### Post by wormser on 2007-12-26
That is the same one I backed up!  Here is the address:  [http://xeve.de/down/wl_apsta.o](http://xeve.de/down/wl_apsta.o)

You can confirm it by uninstalling the firmware and rebooting.  Install the firmware again but when it asks you to download or select location click download and it will display that same address.  Copy it and put it into Firefox to download.  Now go forward with installing the firmware.
