### Post by ajgreeny on 2021-02-18
*Thread moved to **General Help**.* as this is a request for help, not a chat thread.

Telling us you "tried something but it is not working" does not help at all.
What did you try?
What hardware do you have?
Windows 8 used UEFI not BIOS: did you make sure to install Ubuntu in UEFI mode?
Boot again to the USB/DVD you used to install and from the live system please run terminal command ***inxi -Fzx*** and post the output from that back here using Code-tags. (See my signature below for a how-to). You may need to install **inxi** to be able to do that and if so run command **sudo apt install inxi**.
No password will be requested in a live system as far as I remember, but if it is simply hit Enter, leaving it blank.

### Post by mIk3_08 on 2021-02-19
You can also try re-install using your live Ubuntu Desktop O.S. [Click here]("https://ubuntu.com/tutorials/install-ubuntu-desktop#1-overview") for more info.
When you reach to the point where you have to choose which installation type window;  "Install Ubuntu alongside windows" or choose "something else" 
You may choose "Install Ubuntu alongside windows" this option will keep your MS Windows O.S. safe or if you will choose "something else" be sure that you will choose the right partition when choosing the something else radio button.
You should have to be more careful when choosing the "something else" option because you may accidentally remove your MS Windows O.S. when choosing it. 
When your installation is successful then Welcome to the Ubuntu Linux world and Good Luck.
