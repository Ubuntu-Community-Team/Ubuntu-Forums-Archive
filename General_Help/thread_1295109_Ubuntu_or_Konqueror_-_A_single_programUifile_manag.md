### Post by lovinglinux on 2009-10-19
Ubuntu is the distro, while Konqueror is the web browser/file manager. The equivalent to Konqueror in Ubuntu would be Firefox + Nautilus. The last one is the default file manager in Ubuntu.

I like being able to browse and manage files from the browser. I currently running KDE, but I prefer Dolphin over Konqueror, for file management. Besides, nothing beats Firefox for web browsing IMO, so I don't use Konqueror.

I would like to be able to browse and manage files with Firefox. There is an extension, called Firefly, that allows you do that. But last time I checked it wasn't even being maintained. Besides, it's not beautiful or practical as Dolphin or Konqueror.

Another feature that I like a lot in Dolphin and Konqueror is the ability of launch a docked terminal, plus all the window view modes. They beat Nautilus easily.

### Post by lovinglinux on 2009-10-21
> **PsychedelicWonders said:**
> And so Enlightenment is like Ubuntu then?

Is elivecd.com basically a better looking version of Ubuntu?  Is that an easy way to look at it?

No. Enlightenment is not a distro, but a [window manager](http://en.wikipedia.org/wiki/Window_manager) like Compiz. In fact, it is a little more than that, since it can be used without a [desktop environment](http://en.wikipedia.org/wiki/Desktop_environment). You can even install it on your Ubuntu. 

On the other hand, eLive is a distro like Ubuntu, that uses Enlightenment as window manager/desktop environmnet instead of Metacity and Gnome. Kubuntu is an Ubuntu flavor that uses Kwin as window manager and KDE as desktop environment.

More info at [http://en.wikipedia.org/wiki/Comparison_of_X_Window_System_desktop_environments#Desktop_comparison_information](http://en.wikipedia.org/wiki/Comparison_of_X_Window_System_desktop_environments#Desktop_comparison_information)

To install Enlightenment on Ubuntu:

```
sudo apt-get install e16
```

Then when logging in, change the session to e16.

> What would make elivecd.com with Enlightment a better distro than Ubuntu?

Keep in mind that e17 is still unstable.
