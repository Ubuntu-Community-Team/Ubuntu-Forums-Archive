### Post by 4Orbs on 2011-02-16
When you click on "Log-out" a panel pops up with the buttons for logout, restart, shut down, suspend. At the bottom of the panel is a check box with the label "Save session for future logins". Leave that box un-checked.

### Post by 4Orbs on 2011-02-16
Open the xfce4 settings manager (in the Applications menu) and click on the "Sessions and Startup" button. Under the "General" tab, un-check the option to "Automatically save session on logout". If that option is already un-checked, then go to the "Application Autostart" tab and see if the document viewer is listed as an autostarted application. If it is, then you can uncheck it and/or remove it from the list.
