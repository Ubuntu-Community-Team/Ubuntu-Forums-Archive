Today I needed to find and use a couple of antivirus software (Avira,Bitdefender,Avast) to scan my windows machine. I've noticed while bitdefender and avast have very nice frontends Avira, the one I use on my windows machine and realy trust, doesn't... So I've wrote a small shell script using zenity in order to make a gui-like "update and scan handler".

Anyway, I thought I'd share it :) 

Feel free to modify it the way you like and if you enhance it please share so I can learn to code better :)
