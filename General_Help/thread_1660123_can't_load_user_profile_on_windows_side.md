### Post by bcbc on 2011-01-05
There's nothing wrong with Windows' system restore. 

But the only way I know to get around a corrupt user profile is to create a new user, and then copy all the Documents and settings over to the new one. MS has an article on how to do this on support.microsoft.com somewhere.
