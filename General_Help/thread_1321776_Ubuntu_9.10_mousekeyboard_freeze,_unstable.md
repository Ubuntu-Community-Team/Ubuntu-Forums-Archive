### Post by hwttdz on 2009-11-10
There seem to be a lot of people who perceive this monolithic "they" who make linux/ubuntu.  "they" is in fact you the userbase.  In order for linux/ubuntu to succeed "they" depend on users trying things before the final release and submitting bugs with detailed information about what exactly happened (hardware, software, actions, output...).

If you're having a problem you're going to need to be specific in order to help people help you.  You need to say exactly what behavior is happening, and exactly what behavior you expected to happen. You're going to need to share what version of software you have installed and what hardware you're running it on.  And probably lots more, but saying "has too many bugs..." or "doesn't work" or "is slow" doesn't help because people don't know exactly what doesn't work, or exactly what is slow.  

Linux is a totally different philosophy from windows.

Bettman, you should test with a wired mouse first so you can see if you can isolate the problem.  

People who are not Bettaman (and who do not have exactly the same problem he describes), please search to see if your problem is described elsewhere, if not please start a thread with a very descriptive title (you should hit the character limit) and a very specific question to make it easier for people to help you.
