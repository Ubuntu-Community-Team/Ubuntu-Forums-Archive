### Post by ankspo71 on 2011-02-26
Hi,
If you are using Ubuntu, you should already have a torrent client installed called Transmission. There are a many others to choose from though, and I think the most popular ones for Linux would be:

Transmission [http://www.transmissionbt.com/](http://www.transmissionbt.com/)
Deluge [http://deluge-torrent.org/](http://deluge-torrent.org/)
Qbittorrent [http://qbittorrent.sourceforge.net/](http://qbittorrent.sourceforge.net/)
Ktorrent [http://ktorrent.org/](http://ktorrent.org/)
Vuze [http://www.vuze.com/](http://www.vuze.com/)

These are all available in your package manager (Software Center, Synaptic Package Manager, Kpackagekit, etc), so there is no need to download them from their homepages.

Have you tried running 'ktorrent' from the terminal to see if it got installed but it didn't create a start menu shortcut. Many times the start menu shortcuts won't appear until after we log out then log back into Ubuntu (or restart).

If Ktorrent failed to download, then I'm not sure what could have happened. It could be server problems, or maybe unmet dependencies, or something. Maybe someone here can help you out if you could post the error messages from installing ktorrent. 

Hope this helps.
