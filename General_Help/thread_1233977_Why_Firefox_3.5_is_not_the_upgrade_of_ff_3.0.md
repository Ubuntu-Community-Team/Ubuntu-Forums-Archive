### Post by snowpine on 2009-08-07
Hi Asbesto, the reason is that Ubuntu never gives you a major upgrade of a package within a release... when you upgrade to Karmic Koala in October, you'll have 3.5 as the default browser. This also means any 9.04 application that has Firefox as a dependency assumes 3.0.x. Ubuntu is set up this way for maximum stability.

You can install 3.5 now in one of two ways: either install firefox-3.5 (shiretoko) from the Ubuntu repository, or install it directly from the Mozilla website.
