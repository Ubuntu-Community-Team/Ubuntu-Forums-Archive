### Post by endotherm on 2011-01-20
no it's fine, just a video encoding script, generated by acid rip.
when you purge a program, it removes the program itself, all files that Shipped with the installation, and any post-install config files that are known to the install script. if your application however creates scripts as part of what it does, then these scripts will not likely be removed by a purge, since no subsystem knows that they exist at all or had any relation to the app being purged.

frameno.avi is a temp file generated by mencoder when performing a multi-pass encode, so the script just deletes any old temp file lying around (using 'unlink', and ignoring any errors that occure), before trying to perform a new encode. 

the second line just tells it to rip a dvd, encode the video, and output it to your home dir.
