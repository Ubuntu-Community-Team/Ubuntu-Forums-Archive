### Post by matt_symes on 2010-11-20
Hi

Open a terminal and type 

ccsm &

to open the compiz settings manager.

Under desktop there is an option to enable show desktop plugin. Ensure this is checked. Double clicking on the Show desktop icon itself and not the checkbox allows you to configure it.

Does this get you nearer? I don't use it myself so.... and this was checked on 10.04.
