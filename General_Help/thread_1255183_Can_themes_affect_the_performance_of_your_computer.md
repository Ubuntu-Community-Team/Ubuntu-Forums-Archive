### Post by fela on 2009-09-01
I remember when I had an OSX leopard icon theme that had really large icon file sizes, and opening anything in nautilus took _forever_. I doubt it would make much difference on my current machine but this was on a low RAM (256MB) machine with a pentium 4 1.8GHz.

I use the Humanity icon theme now, which has everything in svg (scalable vector) format and pretty small file sizes (it's a great icon set IMO).

I doubt having a different GTK theme would make much difference though. Also, I think that compiz actually makes your system run smoother if you have a good graphics card, due to more stuff being offloaded onto the GPU. Of course it might not be so good on integrated graphics.
