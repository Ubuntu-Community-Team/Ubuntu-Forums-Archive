### Post by earthpigg on 2009-08-16
assuming you already have ubuntu 64 bit installed, you can certainly install kubuntu.

sudo apt-get install kubuntu-desktop

it wont be a 'dual boot', however.


when you see the login screen, select 'sessions' and chose 'KDE'.
to use ubuntu with gnome, select 'GNOME'.


or do you ACTUALLY want to truly dual boot and have ubuntu installed twice on hte same hard drive (one with ubuntu/GNOME/default and one with ubuntu/KDE/aka-kubuntu)? if so, may we inquire as to 'why' to ensure you know what you are asking for and maybe help you not waste your valuable time? :D
