### Post by sandyd on 2012-08-21
Try this in the terminal
```

sudo apt-get install libicu48
sudo ln -s /usr/lib/libicuuc.so.48 /usr/lib/libicuuc.so.44 
```

### Post by sandyd on 2012-08-21
Wait, that was my bad.

```

sudo rm /usr/lib/libicuuc.so.44
sudo apt-get remove libicu48
getlibs -p libicu48
sudo ln -s /usr/lib/libicuuc.so.48 /usr/lib/libicuuc.so.44

```Im not sure if getlibs installs in lib32 or just lib.

I think its in /usr/lib, not /usr/lib32, so Ill just leave it at that.

But yes, as Zill said, Calibre is a good option for ebook readers.
