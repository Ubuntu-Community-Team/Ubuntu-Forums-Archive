I've installed some software (emusic download manager) into the home directory and created a shortcut which works well.  Even the icon image file (png format) worked.

For security reasons, I decided to hide the emusic download manager folder by putting a period in front of it and, subsequently, amended the shortcut settings to point to the hidden folder.  This has been straightforward and it works really well.

### Post by mb_webguy on 2008-09-14
Right-click on the launcher (shortcut) and select Properties.  From there, if you click on the icon you can change the location of the icon used.
