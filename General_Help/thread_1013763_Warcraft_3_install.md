### Post by /////// on 2008-12-17
To install Warcraft 3, you need an application called WINE. Go to Terminal (Applications>Accesories>Terminal) and type in "sudo apt-get install wine". After this is done, pop in your Warcraft 3 RoC cd and install it. After you install RoC, install TFT and then apply the newest patch which you can download from battle.net/war3. After you install the patch, you should be able to play.
