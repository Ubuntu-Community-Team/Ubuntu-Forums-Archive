### Post by oldos2er on 2011-07-03
You can create other user accounts after Ubuntu is installed; there's no option I'm aware of to do this at the time the OS is installed.

Not sure about "uninstallable." Are you planning to dual boot with Windows, or have Ubuntu as the only OS?

There are no real Linux viruses to speak of; there may be browser exploits or similar malware, although rarely. Social engineering is very much a concern.

If you plan to run Windows games, I'd keep Windows. You can always check the Wine app database to see if a particular program has a chance of running in Wine.
