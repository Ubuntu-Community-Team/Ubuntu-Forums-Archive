### Post by Frogs Hair on 2012-07-17
If you mean PDF or any other documents you can save them in any folder you like. If the documents are in downloads you can just manually drag them from downloads to  the desktop and then to the documents folder . 

If using Firefox the browser can be set to ask each time you download and then you can move them to any home folder you want.This can be done in any browser but the setting location is different.
