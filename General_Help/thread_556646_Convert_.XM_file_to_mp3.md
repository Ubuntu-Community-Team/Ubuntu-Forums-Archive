### Post by yabbadabbadont on 2007-09-21
The way that I did this in the past, with multiple tracker formats, was to enable the wave-writer output plugin (in XMMS actually, but it should work in Audacious too), and then play all of them.  I then used a simple script to encode all of the resulting wav files to mp3 and tag them with the file name.  It isn't a pretty solution, but it worked.  :)
