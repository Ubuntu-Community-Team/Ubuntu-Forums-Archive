### Post by tuxxy on 2008-08-20
Did you try envy to install the drivers?

to install a .run open a terminal and navigate to the files location, for example

```
cd Desktop
```

Now type

```
chmod +x filename.run
```


Now finally type this and the installer will begin

```
./filename.run
```
