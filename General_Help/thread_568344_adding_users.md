### Post by overlord.gaurav on 2007-10-05
If you want to create an extremely restricted user profile then you can select the profile as "Unprivileged". Also, you can give/take the user certain rights by clicking on "Manage Groups" in user settings and going to properties of the right and then checking/unchecking the user.
