### Post by tonyr on 2006-07-05
Keyboard layout selections are at **System->Preferences->Keyboard** under the 
**Layouts** tab.

If that doesn't work, run **xev**, put the mouse over the window that pops up,
hit the @ key and see if any of that makes sense, or non-sense, or post it here.

You can also install a package named **xkeycaps** (includes a program of the same
name) that will let you change key definitions individually.  You can read about it [here]("http://www.jwz.org/xkeycaps/").
