### Post by Naggobot on 2010-04-04
You got at least me a bit puzzled. 

> and i dont realy like the menu going down the left side of my screen.Are you referring to Applications menu which is normally closed but opens when you left click the Applications text?

All of the images I have seen from Lucid 10.04 Beta look the same as 9.10 and there should not be any static menu going down the left side. 

Now if this is the case of Applications menu opening from the top then you can note that there should be a similar bar on the bottom. By right clicking that bar you can open a context menu and from there you can select 

Add To Panel

The item you are looking for is "Main Menu" and adding this to bottom panel adds also Applications menu to bottom panel. 

I advise against randomly deleting the top panel, move it or hide it instead if you do not want to see it. 

Also be warned about the beta release. Release date is close but there are probably a lot of unfinished issues. Also make sure that you read carefully release notes when the official release is launched. Upgrade may or may not be as straight froward from beta to official version as the upgrade from 9.10 to 10.04. 

Alternatively you should be able to move the panels by selecting propertis.
