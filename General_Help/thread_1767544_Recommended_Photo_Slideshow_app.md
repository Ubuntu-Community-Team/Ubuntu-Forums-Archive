### Post by ghostborg on 2011-05-25
Imagination is in the Software Center.
Here is a link that you can play a demo of what it can do.

[http://imagination.sourceforge.net/]("http://imagination.sourceforge.net/") 
Type slideshow in the Software center to search for others.

On Windows I used a Video editor and dropped the pictures in the video track, added transitions, music etc. You have a lot of custom control and special effects that way.

Maybe try using Wine with a Windows program. I find Wine hit and miss running Windows programs.
