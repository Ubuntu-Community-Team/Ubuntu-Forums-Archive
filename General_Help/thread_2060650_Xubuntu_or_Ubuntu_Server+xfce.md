Hi there.

I'm planning to install ubuntu in an old computer (512MB of RAM) and I want it to have xfce as its DE. So a question arised:

Which of the following options would be lighter for that pc: Xubuntu or Ubuntu Server + xfce ?

Any recomendations are very welcome!

PS: I want to use xfce (not gnome, kde, lxde, etc.)

Thanks!!!

---

### Post by Lars Noodén on 2012-09-20
You can install from the Alternate disc instead of the Desktop disc and then choose F4 - Install a command-line system.  That gives you the absolute most bare-bones system.  From there, you can add XFCE.  Then you can pick and choose which apps to add in rather than taking the whole kit that Xubuntu-desktop would provide.
