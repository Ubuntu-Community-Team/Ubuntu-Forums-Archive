### Post by dictodude on 2013-08-18
This looks like an xrandr bug you are having. Try and lower your resolution and put it back again. If it doesnt work, search the wiki for xrandr and view the tutorial on manually changing resolution from the command line! Also, if you have any gnome plugins, try and disable them, and enable them one at a time. Maybe a plugin is doing this bug, so at least if it it, you will know what it is!  Hope i helped!
