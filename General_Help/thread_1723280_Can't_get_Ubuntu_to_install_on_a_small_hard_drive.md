### Post by mikewhatever on 2011-04-06
I think neither of the *buntu images will install successfully on a 2GB drive, simply because images, when uncompressed, take more space then that, plus the installer also needs some disk space. The same applies to Debian and Cranchbang images, as there is simply too much preinstalled software. You could probably pull off a minimal iso installation (or Alternate CD) of the command line system, and then add just xorg, gdm, icewm, pacmanfm and a web browser of choice.
[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)
