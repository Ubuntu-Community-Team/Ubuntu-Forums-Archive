### Post by Bucky Ball on 2011-06-08
These are all desktop environments (DEs). KDE (Kubuntu) is greediest with most bells and whistles, Gnome next (Ubuntu up to 11.xx), then Xfce (Xubuntu). 

There is nothing stopping you from running whatever DE you like. The previous three are all running on Ubuntu anyhow. There are plenty of other DEs; fluxbox, openbox, iceWM, etc ... the list goes on ... 

PS: I know nothing about how greedy Unity is as I've not used it but from what I've read, not very. Another DE. You can install the DEs from Synaptic Package Manager, logout and choose the new DE from the sessions tab, log in and you are out of Gnome and into whatever you have chosen. The permutations are many. My desktop is Ubuntu Studio with Xfce and ubuntu-desktop installed, using artwork from Ubuntu Satanic and Edubuntu!

Ahh, the flexibility ...

Neither is really better, they both serve a different purpose. KDE was created back in '96 to be an alternative to CDE, the Common Desktop Environment, which was the most popular at the time. GNOME, created in '97, is the Gnu Network Object Model Environment, originally created as a distributed object framework desktop for *Nix systems. 

It is generally accepted that while both are modern desktop environments that require more recent hardware, KDE is generally heavier and more "bloat-y", although with some fine tuning, this can be changed.

Unity is a Compiz-Fusion plug-in running on top of Gnome. Compiz is a 3D-window manager, it's what creates the wobbly-window and 3D desktop cube effects, which are Linux's claim to fame.

You can dual boot, triple boot ... Normally distros based on Ubuntu will have an easy installation wizard. For the DE: chose whatever you want, that's the beauty of Linux: infinite choice.

To give you a bit more confusion: For now, unity is build on top of (aka "a shell of") Gnome2, but Ubuntu will switch to Gnome3 in the next release (in October). So then Unity will be a shell of Gnome3. Another shell of Gnome3 is Gnome-Shell (made by the gnome community itself). Gnome-Shell does not look like the old gnome or like unity. This is an example of gnome-shell:
[IMG]http://t0.gstatic.com/images?q=tbn:ANd9GcRdm2c5ygjjXaRSnt9OfPJoY6mcZX9rIPM2vQ5HX3Dh9SAN_6LV[/IMG]

For now, Gnome3 is not included in Ubuntu (and installing Gnome3 will break Gnome2 and unity) but you can try it with other distributions as Fedora, OpenSuse or Arch. The first two offer a good live CD (so you don't have to install it).
