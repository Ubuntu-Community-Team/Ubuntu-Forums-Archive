### Post by faitswulff on 2008-08-16
I'm new to Ubuntu and linux, too, but try going to System->Preferences->Appearance->Visual Effects and select "Extra."

That alone might not work (I've only tried these two tricks in tandem) so go to the Synaptic Package Manager (System->Administration->Synaptic Package Manager) and search "Compiz manager" (without the quotation marks).  Then install "compizconfig-settings-manager" and your "Advanced Desktop Effects Settings" should show up in Preferences.

I'm really looking for some criticism and more information about what these two steps do, though, because I'm just running on intuition.

Hope it helps!
