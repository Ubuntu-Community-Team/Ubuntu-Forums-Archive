### Post by sideaway on 2009-08-26
Jaunty Jackalope (release name/version) 9 (year '09') .04 (month) etc...

So every release is alphabetically and chronologically named. Each release is done every 6 months, so is very easy to keep track of, in the 4th and 10th month of every year there is a new release, if this is too often, there are LTS (Long Term Support) versions that offer extended support (updates etc). The Server LTS versions are supported for 5 years, desktop versions, 3.
