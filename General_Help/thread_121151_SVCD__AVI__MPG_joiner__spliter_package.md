I havent had the time to try this out yet, but possible if you put the two files in the same directory in numerical order and issued the command ```
ffmpeg2theora ./*.avi

give it a try, theora will be the open standard for the Open source community, ofcourse it is still new and has alot of work ahead of it along with the need for utilities built around its codec.
