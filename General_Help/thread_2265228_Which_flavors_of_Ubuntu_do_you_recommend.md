### Post by sudodus on 2015-02-13
[SIZE=4]Flavours:[/SIZE]

You did not mention ***standard Ubuntu*** (with Unity), and you did not mention ***Ubuntu Gnome***. They are also worth considering (and they are preferred by many people). There is also the ***Mate*** desktop in Ubuntu (as well as in Linux Mint). ***Ubuntu Studio*** provides a quick way to get a system for multimedia tasks.

If you want to, you can start from the Ubuntu **mini.iso** (alias netboot iso file alias minimal iso file), and create your own system by installing only the program packages that you like. Since you like Kubuntu, you might want to build a system around the ***KDE*** desktop environment.

[SIZE=4]Versions:[/SIZE]

 I'd recommend that you start with the newest long time support version, 14.04.1 LTS. We can expect the second point version (14.04.2 LTS) to be released within a week or two. But Ubuntu and Kubuntu of the version 12.04 (and 12.04.5) LTS are supported until April 2017, and might be good alternatives too, particularly for middle-aged computers.

The versions without long time support have short lives, only 9 months.
