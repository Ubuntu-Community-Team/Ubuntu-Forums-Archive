My 10 months old daughter has found the eject button on my CD player. Is there a way to easily disable the physical button, and eject the CD in some other way?

### Post by erginemr on 2008-02-01
LOL! Fortunately Linux does not allow you to eject CD drives by their buttons when it mounts them. So all you need to do is to keep a working CD-ROM inside the drive, Ubuntu will autoload it and will disable the functionality of the eject button.

When you really need to eject the CD-ROM, all you need to do is to right-click its icon from "Main Menu -> Locations -> Computer" and select unmount (or similar).

In cases of necessity, you may also use the "eject" command from the terminal.
