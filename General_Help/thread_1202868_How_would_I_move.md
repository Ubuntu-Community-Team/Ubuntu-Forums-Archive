How would I move files between partitions? Including windows. Thanks!

### Post by earthpigg on 2009-07-02
go to places -> computer... all the partitions on any hard drive you have plugged in should show up there.

keep in mind that Ubuntu can read windows partitions (ntfs, fat 32), but windows cannot read Ubuntu partitions (ext3, ext4, others).... if you want a shared 'my documents' partition, your best bet is to use fat32 or ntfs.
