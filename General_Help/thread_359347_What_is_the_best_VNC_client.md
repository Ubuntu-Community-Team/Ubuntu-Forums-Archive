### Post by Paerez on 2007-02-11
I like tightvnc. It's free, its in the repos, it works on windows and ubuntu (and between them). It's fast and light too.

It doesn't transfer files. It does have to be installed.

A more advanced reason I like it is that you can run it through ssh, so I don't have to forward ports to make it work, as long as ssh is already forwarded.
