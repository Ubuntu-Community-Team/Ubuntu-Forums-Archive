### Post by paddydd on 2009-06-15
Hi,
  I found the solution.

  I had done all of the expected stuff such as allowing downloads from Amazon as well as pop ups. This is can be found on the Amazon website.

  But I must have missed the statement that you need to "Enable downloads in the browser" when you start the application. This option is under the Help menu choice of the download application.

  So start the download application and then enable downloads. On my system this setting did not persist.
