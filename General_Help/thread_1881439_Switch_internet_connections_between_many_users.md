### Post by kswix on 2011-11-16
It seems I solved the problem:

1) GoTo: System -> Administration -> Users and Groups. Select a user.
2) Advanced settings -> User Privilegies -> Fill in option "Connect to wireless and ethernet networks".

But I can't see connection icon in Indicator Applet. Why?

### Post by kswix on 2011-11-16
Problem solved. If you start internet connection of one user, all other users will use it.
