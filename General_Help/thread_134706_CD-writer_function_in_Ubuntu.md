### Post by 5-HT on 2006-02-22
There is CD burning capability (data and disk image) built directly into the GNOME's file browser.

The standard install also includes an application for burning music CD's.

There is also an amazing CLI CD burning app included in the standard install (for which most of the other programs are front ends to).

Furthermore, there are at least three (K3B, GnomeBaker, Graveman) GUI CD burning suites with many options that you can install, as well as a linux port of Nero (but it's not free, though if you already have Nero you can use your key for the linux version).

About needing to keep the live CD in once you've installed, it is not needed. When you install, Ubuntu resides entirely on your hard drive.

When running from the live CD however, you'll need to keep it in the drive so you'll have to have a separate burner to create CDs if you are only using the live CD.

So overall, linux is VERY burning friendly!

HTH
