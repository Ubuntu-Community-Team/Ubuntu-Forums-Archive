### Post by Sarah L on 2008-05-25
If this activity never happens when you're sitting behind the computer, then it's likely that it has something to do with either the screensaver or the suspend function. Disable the screensaver and standby and see if this still happens. Alternatively, enable both of them but set the timeout to a very short time (ie, one minute) so you can sit back and see if either of these is causing your problem.

This happens to my computer sometimes, and it's usually due to the screensaver kicking in and then crashing.
