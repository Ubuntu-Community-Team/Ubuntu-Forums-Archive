1/ Do a repair installation :-     [http://pcsupport.about.com/od/operatingsystems/ss/instxprepair1.htm](http://pcsupport.about.com/od/operatingsystems/ss/instxprepair1.htm)

This last is not a recommended course of action, it is very very risky.
