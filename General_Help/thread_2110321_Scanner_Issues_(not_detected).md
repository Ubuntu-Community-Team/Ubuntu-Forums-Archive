### Post by Petro Dawg on 2013-01-29
I have a HP deskjet 3050 which is a printer/scanner.  The printer was added to my network using a WPS pin through my router.  The printer works fine, it is detected on my network and prints in Ubuntu12.04.  However when I try to scan with Simple Scan (for example) I get a message "No scanner detected".  

When I attach the printer/scanner through the USB directly, it is detected imediately and scans fine.

So... How do I get this scanner recogonized through the network?  

I've tried other scanner programs (Skanlite and AcquireImages), with the same results.   

Thanks for your help.

Update:

I finally figured it out.  Using the "Networked Scanner" instructions from [this]("https://help.ubuntu.com/community/HpAllInOne") site, I was able to modify the cups configuration file and everything works now.
