### Post by 3rdalbum on 2010-06-17
> **Xavierdri said:**
> Hi Guys

Thanks for all the reply's and I'm sure that all my problems can be resolved but it also seems to me that Linux is an operating system that  a programer will be comfortable with it's definitely not a click and go type of platform yet

I think you're misunderstanding some stuff here.

Ubuntu doesn't include DVD playback and MP3 support by choice. Proper licenses for these things cost money, and as a free operating system Ubuntu can't really pay for however many millions of licenses for DVD playback. If it included unlicensed components, Ubuntu would be open to legal attack.

It's very very easy to remaster an Ubuntu ISO to include MP3 playback, DVD decryption, Flash, Java etc. But these are unlicensed MP3 codecs and DVD decrypters, and you can't redistribute Flash Player without permission, so these are legal barriers and not technical barriers.

Linux Mint and SuperOS are derivatives of Ubuntu that include all these components, at the distributor's risk. Use them if you want, or just install the components on Ubuntu like most people.

I'll also point out that Windows does not ship with DVD playback, Flash or Java. It does ship with MP3 playback support because Microsoft adds the cost of this to every copy of Windows that is purchased, but it doesn't include Ogg Vorbis and Theora playback support.
