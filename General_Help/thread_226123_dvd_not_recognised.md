### Post by nebc on 2006-07-30
Ubuntu doesn't have dvd codecs built into it; you need to install them yourself.  Search for libdvd in synaptic to find the libraries you're looking for.  I don't remember exactly what the minimum set to read DVDs is, but I currently have libdvdnav4, libdvdplay0, libdvdread3, and libdvdcss2.  Let me know if this helps
