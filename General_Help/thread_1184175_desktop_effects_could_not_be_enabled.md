according to this link it's an intel card, which might explain why it is blocking the desktop effects, or at least mine are blacklisted for proper 3D acceleration in jaunty.

If you want to enable 3D desktop effects, you can disable the compiz driver check for intel cards but it is not guaranteed it will work as it should, so usa at your own risk.
In my experience it crashed constantly when i had ubuntu installed inside windows. Since i removed windows completely and did a clean install, i have not encountered any problems so far (probably a dual boot will work also)

```
mkdir -p ~/.config/compiz/ && echo SKIP_CHECKS=yes >> ~/.config/compiz/compiz-manager
```
