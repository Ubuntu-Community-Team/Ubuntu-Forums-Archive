### Post by mb_webguy on 2009-03-10
The problem with clicking a menu (or right-clicking to pull up the context menu) causing a random event (usually but not always the topmost item) to be selected is a [known bug]("https://bugzilla.mozilla.org/show_bug.cgi?id=404314") with Firefox on some (but apparently not all) Linux systems.  It's apparently been fixed in the Mozilla trunk, but hasn't propagated down to the current stable release of Firefox.

One workaround is apparently to install the Mouse Gestures extension.
