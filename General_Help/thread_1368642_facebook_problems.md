### Post by LuisGMarine on 2010-01-25
If you are using Firefox, follow these quick steps.

Open up Firefox and type in **about:config** in the search box and hit enter.

search for the line called **network.dns.disableIPv6**, and set the value to **True**

Restart Firefox and you are good to go!
