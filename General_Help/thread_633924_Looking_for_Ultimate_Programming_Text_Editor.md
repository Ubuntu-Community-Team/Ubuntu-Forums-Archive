Oh boy vi/emacs flame war a comin' run for the hills. If you're using KDE I would suggest kate. It's got syntax highlighting, several auto tabbing models, code folding, and some basic name completion on identifiers you've used before it that file. And features a built-in shell and file browser.

### Post by DirtDawg on 2007-12-07
I also prefer Vim, but [Cream]("http://cream.sourceforge.net/") is also great. It's got the power of Vim with a great GUI and more standard, remapped hotkeys(ctrl+c=copy, ctrl+v=paste, for example). You may want to give it a look.
```
sudo apt-get install cream
```
