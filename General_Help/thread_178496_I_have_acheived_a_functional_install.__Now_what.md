### Post by jaclayton on 2006-05-18
Run Adept and fetch the updates.  But first you should enable some of the other repositories - that's sources for software.  By default a lot of stuff is hidden.  This [wiki page]("https://wiki.ubuntu.com/AddingRepositoriesHowto?highlight=%28repositories%29") tells you how - scroll down to find the Adept section.

Then install all the software you could ever desire!
You'll want to enable a lot of the "restricted file formats" - there's an excellent page on the [wiki]("https://wiki.ubuntu.com/RestrictedFormats?highlight=%28formats%29%7C%28restricted%29")ibut apparently [Automatix]("http://ubuntuforums.org/showthread.php?t=138405") is pretty good - I must try that one myself.  That'll give you mp3 and DVD playback to start with.  The wiki also gives very good instructions on realPlayer, movie playback within Firefox and getting the Flash player working.

One word of advice on Flash-plugin installation - Adept appeared to hang on the progress bar - if you click "Show Details" (I think that's what the button is called) you'll see it's actually waiting for some input from you.  Just press "Enter" then and it should storm through.

There's probably some nice, updated drivers for your ATI card (I run nvidia, so you'll have to search the forums for some help on that one), not so sure about the sound card.

And then there's the heady choice of software to play with!

Go forth and have fun.
