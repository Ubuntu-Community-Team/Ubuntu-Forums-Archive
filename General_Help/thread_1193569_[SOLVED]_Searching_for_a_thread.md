### Post by lovinglinux on 2009-06-21

If you like to search Ubuntu forums using Google Site Search, then download [this file](http://ubuntuforums.org/attachment.php?attachmentid=118441&d=1245590626), extract it to ~/.mozilla/firefox/yourprofile/searchplugins/ and restart Firefox. It will add a new search plugin in the Firefox search bar. It allows you to search Ubuntu forums using Google Site Search, without the need to add "site:ubuntuforms.org". Just type the keywords.
