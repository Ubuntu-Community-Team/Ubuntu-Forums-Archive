Using iTunes or RhythymBox, my favorite method for adding album art to a track (or album) is to drag the album art from the web browser to the album art display in the media player and drop it.  You might try that.

### Post by unknown user on 2012-06-15
This does appear to be a very old post, but I thought I would add to it, as I was having the same issues and have found a good way to get around it.

If you place the Album art into the same folder as the songs and then call it "cover", Rhythmbox automatically should pick it up.
