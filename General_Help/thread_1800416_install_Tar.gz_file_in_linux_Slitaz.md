### Post by mutley89 on 2011-07-08
A .tar.gz, often called a tarball is just a compressed archive format, it can have anything in it.  Extract it using ```
tar -xfz filename
```.  There will probably be a README file, and possibly one called INSTALL with instructions for installing.
