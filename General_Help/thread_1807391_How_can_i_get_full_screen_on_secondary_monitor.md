### Post by Cybie257 on 2011-07-19
If you go to:

 System -> Preferences -> Monitors

you should be able to select the secondary monitor then hit the default button and make the secondary default/primary.

The problem you are having isn't Ubuntu, but Adobe/Flash. It is far from being perfect and therefore, isn't really programmed to comply with dual screens. I've had to run a hack/fix in order to get the full screen to stay full screen on my second monitor whenever I clicked on something on the primary. Otherwise, it would go back to small screen mode, which is annoying. 

Anyhow, hope this helps resolve your problem. 

-Cybie
