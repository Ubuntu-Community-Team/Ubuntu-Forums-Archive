### Post by MG&amp;TL on 2012-02-18
Follow the instructions: [http://www.webupd8.org/2011/11/install-ubuntu-unity-bottom-launcher.html](http://www.webupd8.org/2011/11/install-ubuntu-unity-bottom-launcher.html)

When people refer to CCSM, it's an acronym for the *compizconfig-settings-manager *package.

I suggest caution, but if all else fails, you can always use Ubuntu 2d, then reset unity.

NOTE: Ignore the various windows themes available, unless they are specifically marked as compatible with recent version of Ubuntu. We went through quite a big interface & backend change recently, I imagine it has broken all of the old themes.
