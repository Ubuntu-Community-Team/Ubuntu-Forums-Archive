### Post by LaRoza on 2008-04-19
> **Exsecrabilus said:**
> So you can't download .iso files of the DVD versions right?

Aww, that sounds too good not to be free! XD

Er, if you have the internet connection for downloading a DVD, there is no reason to get the DVD (unless you plan to use it elsewhere or to give it to someone)

The repositories (some of them) are on the disk. Without the disk, Ubuntu doesn't downloads and installs the packages as requested.

For the Ubuntu CD, there is a package called "build-essential". When you try to install it, Ubuntu asks for the CD first. If the CD is removed as a source, it just downloads it from the repositories and installs it. The DVD just has more.
