### Post by Irony on 2008-05-10
Most applications will add a menu entry somwhere in the menu structure.

For the few that don't just go to System > Preferences > Main Menu; then select where you want to add it and in the command section add the command that you used in terminal... it should now pop up in the menu.

Look at some of the other menu entries whilst at it so you get an idea of what sort of commands are used.
