### Post by TheFu on 2022-09-03
You don't.

MacOS programs don't work under Linux.
You should look for an alternative.  Find a native Linux equivalent.

[https://alternativeto.net/browse/search/?q=bookhound](https://alternativeto.net/browse/search/?q=bookhound) doesn't appear to find anything for that program. Seems to be a niche tool.  There are definitely book management systems.
[https://alternativeto.net/software/filemaker-pro/about/](https://alternativeto.net/software/filemaker-pro/about/) is for Claris Filemaker. 

You might try to make the MS-Windows version work through WINE, but know that 80% of the programs do not work under WINE and of the remaining 20%, most only support the main functions.  For example, Quicken  (a financial program) was listed as "Silver" level, which means that main functionality is working, but anything outside that either doesn't work or crashes the program.  For a few years, I ran Quicken under WINE, then an update broke it.  I tried for 2-3 weeks to figure out how to make it work again, but failed. I gave up and installed it into a virtual machine running MS-Windows and it is 1 of 2 remaining programs that I still use MS-Windows to run.  It gets used about 15 minutes every other week.   It takes time and effort to migrate off other OSes.  I started over a decade ago, but some things, like simple video editing wasn't as easy as my $50 program on Windows was until about a year ago. The replacement program isn't as slick and doesn't handle some types of editing. Some of the keyboard choices aren't great, but it does a few things much better.

For ebooks, I use Calibre. It has a server, desktop program and Android clients.  Many ebook readers can access Calibre directly.  I suppose you could use it for real books by making empty files of a specific name and importing them into Calibre. I haven't tried this.   

[https://www.ubuntupit.com/best-library-management-software-for-linux/](https://www.ubuntupit.com/best-library-management-software-for-linux/) has some "Library Management" software. I can't tell if those are for full public libraries or personal.
[https://alternativeto.net/software/personal-library/](https://alternativeto.net/software/personal-library/) seems to be for personal book library management.  The term "library" on Linux will commonly become confused with programming libraries, so some careful search terms are needed.
