### Post by 3Miro on 2009-04-15
Geany for Gnome and GTK and KDevelop for QT4 and KDE.

Geany is light weight and has all the basic options (well you still have to make your own make file, but you have to do that in Visual Studio as well).

KDevelop is heavy, but has a ton of options (most of which I do not use).

### Post by dubref on 2009-04-15
Grab Eclipse IDE for C/C++ Developers (67 MB)

from
[http://www.eclipse.org/downloads/](http://www.eclipse.org/downloads/)

Home page of CDT project:
[http://www.eclipse.org/cdt/](http://www.eclipse.org/cdt/)

The nifty thing about Eclipse is that it will also run from other linux distributions and windows. Because of wonders of JVM you dont really have to install Eclipse, just unpack it. 

In theory, this means you could have Eclipse on USB stick and have the same development enviroment everywhere you go. In reality, you would have to do some configuration when you switch machines(compiler selection etc.).

All that said, for small applications you can get by with just gedit and GCC from command line.
