### Post by RJ Hythloday on 2008-03-28
I've changed the screen resolution size in display settings but my firefox or swiftweasel are too small for me and not readable. I have to use ctrl++ twice on every tab that's opened.

### Post by cdenley on 2008-03-28
In the firefox settings, you could change the default font, but if the website defines a font size, the default will be ignored. In firefox3, which comes with hardy, you can define a minimum font size. I'm not sure about previous versions of firefox.

Also, I think fonts are easier to read with the MS fonts installed.
```

sudo apt-get install msttcorefonts

```

---

### Post by TheExplorer on 2008-03-28
Yes, also in previous versions.
Go to Edit -> Prefs -> Content -> Fonts & Colors. There are several options and "minimum font size" is there.
