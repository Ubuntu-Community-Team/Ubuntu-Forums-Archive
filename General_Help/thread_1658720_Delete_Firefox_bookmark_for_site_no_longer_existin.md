### Post by lovinglinux on 2011-01-03
> **janmartin said:**
> **Delete any and all bookmarks to this domain.**

I don't know any extension that does that. Besides, this might be useful only if you save bookmarks for the home page, but if you save bookmarks for tons of forum threads for example, this would remove all bookmarks.

In my opinion, is better to use a bookmark check tool like [CheckPlaces]("https://addons.mozilla.org/en-US/firefox/addon/10897/").

### Post by lovinglinux on 2011-01-03
BTW, you don't need to open the manger or the bookmark list to delete it. If a page no longer exists and if you are not redirected, then simply click the star in the address bar and then "Remove Bookmark".
