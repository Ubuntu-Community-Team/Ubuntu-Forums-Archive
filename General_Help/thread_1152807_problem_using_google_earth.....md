### Post by blazemore on 2009-05-08
It might just be that your graphics card is not powerful enough to run the 3D effects.
Have you had success with this application in Windows?

If you have, and this is an Ubuntu specific problem, the graphics card drivers might be at fault. You need to make sure you're running the latest version of the drivers provided by the manufacturer.
You can do this by loading the Restricted Drivers Manager (System->Administration->Hardware Drivers (or similar))
You can install your driver using this tool.

Hope this helps :-)
