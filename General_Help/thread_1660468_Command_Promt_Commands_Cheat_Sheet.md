### Post by Brandon Williams on 2011-01-05
The "utilities" section of the [POSIX "Shell & Utilities" specification](http://pubs.opengroup.org/onlinepubs/009695399/utilities/contents.html) gives a good overview of each individual command, along with examples of the use of each command. I think this is a good place to start to get an overview of the most important commands on a *nix system.
