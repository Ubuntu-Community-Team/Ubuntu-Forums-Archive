### Post by cardinal548 on 2016-09-06
So I just lost 1tb of data on a 3tb external usb drive, So in the interest of not having that happen again I'm looking for advice on how to optimally setup my system.
What I have is:
1 primary internal ssd 240GB with a 40GB ext4 partition for OS, 5GB swap partition, 195ext4 for the remaining.
2 secondary internal drive 1TB with all of it ext4 partition
3 usb external 3TB which I'm not sure at this point is still salvageable. Perhaps what to do about that would also help.
4 Old IDE drive in enclosure which I'm no longer worried about and haven't used for a long time.
Drive 3 the usb external 3TB has been reclaimed and is so far has one ext4 partition.

### Post by DogMatix on 2016-09-06
I would have the SSD as one partition and install the OS to that, including /home, alternatively have a 5GB swap partition too if you feel it is necessary. Keep the bulk of your media files (music, film, photos) on the 1TB drive. Then back the whole system up to the 3TB external.
