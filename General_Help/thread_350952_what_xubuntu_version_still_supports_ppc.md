### Post by igknighted on 2007-02-01
First, all current versions of Xubuntu support PPC.  Second, no one can predict how each version will run on your hardware.  It might be an issue between the screen and X, or it could be a million other things... the point its, try a few versions.  Especially if you can boot from the LiveCD.  If YellowDog is working fine, why not use that?  You could install Xfce desktop on that and configure it like Xubuntu (probably even download the art somewhere too).

For more thorough answers, try posting this in the Mac/PPC forum, you might get someone who knows a little more about Apple hardware issues there.
