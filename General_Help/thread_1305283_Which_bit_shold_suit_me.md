### Post by Rocket2DMn on 2009-10-29
That's a good question.  Come to think of it, I've heard mixed results about the 4GB limit (I know Windows only shows about 3.2GB).  You can use all 4GB in 32 bit if you use a kernel with PAE (physical address extension), which I believe the server kernel has.  I don't recall if the latest kernels in 32 bit Ubuntu have this capability built in.  The 64 bit should support all 4GB out of the box.  The driver support should be the same, but just the same, you can post your devices here which require proprietary drivers and we can look them up.  Sorry about my initial lapse of memory there.
