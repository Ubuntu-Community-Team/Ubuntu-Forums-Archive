### Post by ankspo71 on 2009-12-14
I'm still learning commands (just the basics) but some of my favorite ones to use are 
_locate_   (to find something by filename)
_cd _       (to change directories)
_ls _        (to list the contents of directories)
_cp _       (copy)
_mv_       (move)
_apt-cache search_ <package> (Great to use with apt-get)
_apt-cache show_ <package> (Great to use with apt-get too)

```
./configure
make
sudo make install
``` to compile and install packages from source codes

Try _nano_ in the terminal to edit a text file. Try '_w3m [www.google.com](www.google.com)_' to surf the web using google (you might like '_elinks_' much better though).

'_tar_' is what I want to learn next :D

Enjoy using the command line. once you start getting used to it, it will get much better.
