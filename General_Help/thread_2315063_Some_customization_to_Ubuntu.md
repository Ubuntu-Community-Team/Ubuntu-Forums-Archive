### Post by ian-weisser on 2016-02-26
The short answer to your original question is that you make the changes in source code (and change images in the source tree) and compile binary packages from the changed source.
Ubuntu's source code is at Launchpad.net.

UCK is useful if you want a custom set of Ubuntu packages on your install media.
It is not useful if you want to edit the contents of those packages.
