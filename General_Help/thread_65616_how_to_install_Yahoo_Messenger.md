### Post by cstudent on 2005-09-14
According to the Yahoo site at [http://messenger.yahoo.com/messenger/download/unix.html](http://messenger.yahoo.com/messenger/download/unix.html)



Debian Linux:

   1. Save the file to your machine.
   2. Log in as root  and type       dpkg -i ymessenger_1.0.4_1_i386.deb
      to install the application
   3. Run /usr/bin/ymessenger from X Window to launch the application.


To run as root just type
sudo dpkg -i ymessenger_1.0.4_1_i386.deb

You'll do this from a terminal screen and you'll need to be in the directory the .deb file is stored in.
