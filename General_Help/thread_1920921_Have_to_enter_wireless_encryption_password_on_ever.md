Hi, I'm running Ubuntu 11.10.  On this latest install I've been running for a few weeks, everytime I reboot, it forgets my wireless password and I need to re-enter it.  How can I fix this?

---

### Post by ajgreeny on 2012-02-05
Right click on the network applet icon and "Edit Connections".   Make sure the wireless connection is set for "Automatically Connect" and also "Available for all users".
