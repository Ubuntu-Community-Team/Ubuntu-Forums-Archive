### Post by DeMus on 2010-01-02
> **Jimtuv said:**
> I am running Ubuntu 9.10 when I install new software it does not show up in the menu. I have to manually add it by editing the menu. Is there something that I have done that would keep it from adding things to the menu? I am new and not really sure if this is normal.

There are plenty of small programs, utilities, which don't make an entry in the menu. Simply because they expect to be started from a terminal where you can add several options to it.
So, not all installers produce an entry. There need not be anything wrong with your installation.
