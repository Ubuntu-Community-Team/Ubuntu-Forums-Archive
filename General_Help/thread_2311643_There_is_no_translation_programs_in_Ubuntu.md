### Post by blade4 on 2016-01-29
There are two options for you : 

1. Try some native linux programs. A list can be found here : [https://en.wikipedia.org/wiki/Category:Computer-assisted_translation_software_for_Linux](https://en.wikipedia.org/wiki/Category:Computer-assisted_translation_software_for_Linux)

2. WINE is a software that allows windows programs to run on linux. However it does not work well with every software and is very hit-and-miss. In fact, since memo q needs word to display text I think it will not run well with wine. But there's no harm in trying. 

I suggest going with option 1 first since it's the easiest.
