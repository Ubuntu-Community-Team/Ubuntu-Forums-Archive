### Post by Proteusiq on 2009-05-10
Hej there!

 I had the same problem, I think it is something to do with 3D card support or something! not so sure!
  BUT I have another way around it! Which I think is pretty cool since now we can use Blender with more window control on it(window)

   Download blender.exe(for Microsoft Windows)
   Then install it under wine!
       You can have wind by ***sudo apt-get install wine*** or just use ADD/REMOVE it is there!

   Then install the blender.exe with wine, it work so cool, I think better than the other one on my case, when it comes to Animation or Rendering!

P.S: Configure your wine and Have fun!
