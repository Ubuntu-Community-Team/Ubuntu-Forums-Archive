### Post by supershin on 2010-01-24
You can set it up to dual boot, that is, be able to boot into XP or ubuntu. My current system uses that. When i power on my PC, a menu comes up asking if to boot XP or ubuntu, I choose whichever I want, plus you can set a default choice that gets chosen after a few seconds. This way you don't have to erase XP, its always there when you need it.

If you want to backup XP in the event your hard drive gets wiped, you can check out driveImageXML. It makes an image of the drive. You then use a BartPE cd at boot to run driveImageXML and restore the drive.

### Post by studmf on 2010-01-24
1) so as im installing ubuntu, it will give me the option to have a dual boot? or is there external software needed to do that?

In windows in disk management/partition you should resize XP to allow an amount of drive space you think you will need for ubuntu.  I made mine 60GB, leaving windows ~240GB, up to you.  After you have resized, boot live CD and install ubuntu, when it asks were to install choose the largest unallocated space (the area you freed up).  Continue thru the steps and you should be done.  There are some guides out there with images to step you thru resizing xp.  You can do it in Ubuntu, but I DO NOT recommend that (takes forever and screws with all sorts of stuff, in my experience)

See [SIZE=5]_**[HERE]("http://www.tips4pc.com/Format%20your%20computer/how_to_shrink_a_partition_on_you.htm")**_[/SIZE] for one guide
