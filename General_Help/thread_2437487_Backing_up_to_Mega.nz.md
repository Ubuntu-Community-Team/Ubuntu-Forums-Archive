### Post by Holger_Gehrke on 2020-02-24
mega.nz offers both a graphical tool (megasync) and a set of command-line tools (mega-cmd) for various Linux distributions including Ubuntu and its flavours. You can either download the package or set mega up as an additional repository so that the usual update & upgrade mechanisms keep them current . Among the tools in mega-cmd is mega-backup which is meant to give you multi-generation scheduled backups. I haven't tried it myself and the help says it's "in BETA state".
