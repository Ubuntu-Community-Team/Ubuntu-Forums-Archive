### Post by mc4man on 2010-11-14
After fixing - whenever using the 'other application' menu, whether on files or folders, if the 'remember this application..' box is enabled then the app chosen will become the new default.

By default the box is always enabled so you'll need to uncheck each time.
(it shouldn't change defaults - upstream bug, and it shouldn't be enabled by default - something ubuntu could change but declines to do so.
