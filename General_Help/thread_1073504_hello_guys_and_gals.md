### Post by albandy on 2009-02-18
first of all verify you've java installed, if not type in the terminal:
sudo apt-get install sun-java5-jre
then
download to your home folder the linux version of jdownloader from jdownloader.org
unzip it by this console command:
unzip "JDownloader v0.5.533.zip"
then enter the JDownloader folder:
cd JDownloader
now execute it as a java application
java -jar JDownloader.jar
