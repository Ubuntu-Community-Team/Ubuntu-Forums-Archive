### Post by squaregoldfish on 2008-12-06
From the enlightenment menu, select Configuration -> Configuration Panel.

Scroll to the bottom on the left hand side and you'll see File Manager, select that, then File Manager on the right hand side.

In the resulting popup, check the 'Open Dirs In Place' option, and you'll stop getting new windows all over the place.

Documentation is a bit on the scarce side at the minute. There is an [enlightenment-users mailing list]("https://lists.sourceforge.net/lists/listinfo/enlightenment-users") which you may find useful, although I'd recommend lurking there for a while to get a feel for it. Everyone's helpful enough, but some of the messages can seem a little brusque at first.

Steve.
