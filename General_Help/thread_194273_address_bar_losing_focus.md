### Post by schilcha on 2006-06-11
I remember this happening frequently with search-engines like clusty, google or dogpile. This [I]feature[/] can be disabled by disabling java script. 

I've installed NoScript for firefox, which turns off java-script for all sites unless you put them on a white-list.
