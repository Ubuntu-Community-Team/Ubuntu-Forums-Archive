### Post by Cheiron on 2007-01-27
If you havn't already done so, enable the multiverse/restricted ubuntu repositories.

If you want the Automatix repository without the Automatix software, add this to your repo list in synaptic.

[http://www.getautomatix.com/apt](http://www.getautomatix.com/apt) edgy main

If you want some specific programmes, some like Wine have their own repositories listed on their website.

I suppose that the other repositories you might want to add depend on the type of software that you want that isn't in the ubuntu repositories. For a lot of specific stuff, you might be able to find .deb files on the internet if it isn't in any repository
