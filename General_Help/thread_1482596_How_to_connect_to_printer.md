### Post by Choragos on 2010-05-13
I presume then it's a network printer.  Go to system->administration->Printers.  Pick add, and there is a choice for find network printers (probably jet direct).  From there it should pretty much walk you through the process.

---

### Post by Choragos on 2010-05-13
Also, the Brother website has a pretty good set of drivers, packages, and scripts to help you install printers.  For my MFC-7340 it had a script that worked beautifully.
