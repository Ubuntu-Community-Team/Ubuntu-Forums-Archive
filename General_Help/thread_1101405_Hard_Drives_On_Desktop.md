### Post by kpatz on 2009-03-20
Right click on desktop and select "Create Launcher".  Then change the type to "Location".  Enter a name and the path (mountpoint) of the drive or directory you want an icon for.  Click OK.  Done.

You can create icons for any location in your file system this way--it doesn't have to be specific drives or partition.
