### Post by mikewhatever on 2009-09-10
I'd simply add Ubuntu and BSD to the HDD with Windows, if space allows. If not, use another one. Giving each OS each own HDD is not a bad idea, but is unnecessary. It might be useful if you wanted to use the drive with Ubuntu on another computer for example.
