When I drag a window from the built-in display to the monitor, the window grows, and when I drag it back, it shrinks. In display settings, I've noted the "Scale for menu and title bars:" slider, but it changes the window size in both the monitor and built-in display. Also, the "Scale all window contents to match:" option does nothing, and I tried all the options. Any suggestions?

Note: I also have a Dell Inspiron 3551 running Ubuntu 16.10, and it doesn't have this problem.
Also, switching the built-in display's resolution to 1360x768 (16:9) fixes the solution, but I would like to be able to use the high-resolution display as it was intended to be used
