### Post by ScottFW on 2007-01-30
It was a while ago, but I'm pretty sure I installed azureus via apt-get.
sudo apt-get install azureus
I don't remember it installing the mozilla browser.
You can use Synaptic to remove both, then reinstall azureus from the terminal using apt-get and see where that gets you.

I'm not a huge fan of Azureus though. Seems a lot of people like it, but my resources monitor shows that java is a hog. And my torrents seem to download faster using the default bittorrent application (not sure if this was included with the Ubuntu install or with Firefox). The benefit is that Azureus allows me to have multiple torrents running simultaneously, whereas I get errors trying to use more than one torrent with the default app.
