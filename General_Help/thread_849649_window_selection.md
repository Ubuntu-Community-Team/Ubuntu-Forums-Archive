### Post by sayakb on 2008-07-05
@OP
Are you using Compiz? If yes, then goto System->Preferences->Advanced desktop effects settings (CCSM)
If you dont have CCSM installed:
```
sudo apt-get install compizconfig-settings-manager
```
Now in ccsm, click on **Application Switcher** plugin. Click on the **Bindings** tab. From there, **disable** the key bindings for *Next Window*and *Prev Window. *Now enable the key bindings for ***Next Window (no popup) ***and ***Prev Window  (no popup)*** and set the bindings as you wish to.
