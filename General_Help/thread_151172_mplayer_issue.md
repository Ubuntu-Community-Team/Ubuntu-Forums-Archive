### Post by izmaelis on 2006-03-27
Click right mouse button over mplayer's screen, choose "preferences", hit "Ok" (if dialog window appears), go to "Video" and check if "xv" under "Available drivers:" selected. If not select it and exit preferences window by clicking "Ok" button.
Restart mplayer (just in case) so new options are applied.
