### Post by bg1256 on 2006-11-05
Try these websites:
[http://forum.beryl-project.org/index.html](http://forum.beryl-project.org/index.html)
[www.gnome-look.org](www.gnome-look.org)

At these sites, you can find themes for Beryl.  Make sure when you save them, you save them as .emerald.  Then, just drag and drop into beryl's theme manager.  

Kiba-dock is a fun dock - similar but different - to OSX's dock.  Gnome dock is out there too, but I've never been able to get that running.
Follow the howto in the forums here (run a search), and it will work for you.

Hope it helps!

edit: from gnome-look.org, you can download compiz themes for beryl, but save them as .emerald, and most (if not all) will install fine.
edit 2: you can't drag and drop into emerald theme manager, sorry forgot that.  you have to import them...but that6's fairly straightforward.
