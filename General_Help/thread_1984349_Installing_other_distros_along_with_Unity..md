### Post by kanikilu on 2012-05-21
So far I've only installed Xfce and Cinnamon other than Unity. However, I have found from past experience that it's usually best *not* to install the *-desktop packages (like *xubuntu-desktop* or *kubuntu-desktop*), unless you  know that you want to move to that desktop environment full-time.

If you only want to test them out, try installing just the base packages. For example, *xfce4* or *kde-full*.

### Post by hal8000 on 2012-05-21
If you install a desktop like window maker, LXDE, Afterstep, E17, etc there should not be a problem as long as you have enough space on your / partition.

If you install a complete Ubuntu environment like Kubuntu, Lubuntu, etc then
there could be problems (but Ive not tried this).

In the times of KDE3 and Gnome 2, you could install as many desktops and windows managers as you wanted, however with desktops like KDE4 and Gnome3 they have become very large in size compared to earlier versions.

I am sure there must be Ubuntu users who have tried this, I just have Unity, and Gnome fallback on my Ubuntu 12.04 setup at the moment.

Again, everyone has their favourite desktop and their are no losers, excpet from Steve Balmer and Microsoft.
