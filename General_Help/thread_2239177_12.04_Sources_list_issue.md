### Post by steeldriver on 2014-08-12
The error message appears to be telling you to look in the /etc/apt/**sources.list.d/tiheum-equinox-precise.list** file NOT the main /etc/apt/sources.list

Also suggest you use gksudo or gksu for graphical apps

```

gksudo /etc/apt/sources.list.d/tiheum-equinox-precise.list

```
