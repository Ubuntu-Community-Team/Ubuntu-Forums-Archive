Im logging into a server through SFTP by going to places->Connect to Server, that way I can access my files easily through nautilus.

When I login, it starts me in the root directory, but I want to start in my Home directory.  Is there any way to have that my default starting location?

Raman

### Post by Pollox on 2010-08-22
In the "Connect to server" window, specify the path to your home directory in the "folder" field.  You can also create a bookmark to that folder in nautilus, so you don't have to type in all the details of your server every time.
