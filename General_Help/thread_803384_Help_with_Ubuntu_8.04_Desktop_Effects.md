### Post by mano cazalet on 2008-05-22
store the wallpaper in some folder. If you delete it, of course it won't be there anymore. Its different from tarball themes, these are copied to specific theme and icon folders, so you can delete the tar files for themes. To install them (themes and icons) just drag them to appearance preferences / themes.

for screenlets it puts an entry in system preferences menu or applications. You can manage your screennlets from there and set them to automatically start when system starts.

for awn you will have to add it to your sessions (system/preferences/sessions) so it will start: add avant-window-navigator. I would recommend to use the search function of forums, as there are plenty of threads on configuring both awn and screenlets.

panels can be edited with a simple right click.

in Advanced Desktop Effects Settings you can also edit the key combinations for each feature.
