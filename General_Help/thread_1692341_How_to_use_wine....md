### Post by Spyderkid on 2011-02-21
[SIZE="4"]_**[COLOR="Green"]Wine[/COLOR]**_[/SIZE]

[SIZE="3"]_[COLOR="red"]Downloading wine[/COLOR]_[/SIZE]
[COLOR="Navy"]To download wine go to the software centre and search for wine and download either the beta or standard release.[/COLOR]
[COLOR="Red"]Or you can download it from _**[HERE]("http://www.winehq.org/download")**_[/COLOR] [COLOR="Lime"][COLOR="SeaGreen"]and just double click on the file when downloaded and it should open with the software centre and install. Or if you download it with firefox just choose open with software centre.[/COLOR][/COLOR]


[SIZE="3"][COLOR="Red"]_Installing Programs_[/COLOR][/SIZE]

[COLOR="Orange"]>if you want to install a file you downloaded from the internet then find it[/COLOR], [COLOR="Blue"]right click on the file, go to permissions, and click the "allow executing file as a program" box.[/COLOR] [COLOR="SeaGreen"]Then right click use, open with wine program loader. then wait and follow the installation[/COLOR]



[COLOR="Red"]If installing from a CD/DVD/CD-ROM then its a tad trickier.[/COLOR]

[COLOR="Cyan"][COLOR="DeepSkyBlue"]>insert your CD/DVD/CD-ROM AND LOOK AT YOUR DESKTOP TO FIND WHAT THE CD'S CALLED! [/COLOR][/COLOR]
[COLOR="Orange"]>open a terminal by going to application > accessories > Terminal. Then open it and enter this.... (one line at a time pressing enter after each one.) [/COLOR]
```

cd /media
cd DVD NAME

```
[COLOR="DarkRed"](look on desktop for DVD NAME (needed above) [don't type this stuff in brackets])(if the name has spaces use " " around the name.)[/COLOR]

[COLOR="Teal"]then in the same terminal after that enter this...[/COLOR]

```

wine setup.exe

```
[COLOR="Red"]follow the setup instructions[/COLOR]
[COLOR="DarkSlateGray"](If it says there's no setup.exe file browse the CD to find an .exe file similar to setup.exe it might be SETUP.EXE or SETUPNOW.EXE or something of the likes)[/COLOR]
