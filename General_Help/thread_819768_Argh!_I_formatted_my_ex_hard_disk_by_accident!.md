### Post by rbmorse on 2008-06-06
Get a copy of System Rescue CD

[http://www.sysresccd.org/Main_Page](http://www.sysresccd.org/Main_Page) 

Create the bootable CD from the .iso file and then boot the computer from SystemRescue CD.  Use the Testdisk utility.  

Here's an excerpt from the TestDisk documentation that addresses your issue (scroll to bottom), but you might want to review the entire documentation web page in detail.

[http://www.cgsecurity.org/wiki/Data_Recovery_Examples](http://www.cgsecurity.org/wiki/Data_Recovery_Examples)

This isn't universal (i.e., it doesn't work in every case) and my own personal experience is that even if the right file systems are involved the prospects for successful recovery are small.
