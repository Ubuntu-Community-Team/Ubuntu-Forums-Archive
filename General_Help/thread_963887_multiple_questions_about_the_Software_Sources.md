### Post by mikewhatever on 2008-10-30
> **eival said:**
> 
for starters in the ubuntu software tab why does the source code have a line through it and not a check?

second, on the third party software tab, the 2nd choice has (source code) at the end. does this mean it will download the actual code and i would have to manually install?

Yes, you'd be able to compile it. Reviewing code it also useful for educational reasons.

> also i had installed the restricted nVidia drivers when ubuntu first told me they were available back when i installed 8.04, but now that i see these boxes unchecked does that mean this entire time i havent been getting updates for those drivers?

Probably not, but I am not sure if Ubuntu gets nvidia updates at all. You are welcome to enable the restricted repositories and check for yourself.

> and finally under the Updates tab, the pre-released option im sure is referring to beta releases, but the choice under it "unsupported (hardy-backdports)" what does that offer?


Backports are for newer (not necessarily pre-release) software provided by Ubuntu users. That as well as pre-release is not officially supported by Ubuntu.
