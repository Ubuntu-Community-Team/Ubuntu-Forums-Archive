### Post by seanUSXIII on 2006-12-07
If you get the Damn Small Linux live disc. You can boot it, then on the desktop, right-click > apps > tools > Install to USB HDD Pen-drive. Works like a charm. You can even run it in "toram" mode which then loads the whole OS into RAM. This makes it wicked fast. :cool:
