### Post by Grenage on 2009-11-27
> 1) I am running the 175 Nvidia drivers, and Ubuntu 9.10 is only detecting my monitor and not the TV when all cables are plugged in, should I enable the 185 drivers to begin with?

I would use the latest drivers, definitely 185.

> 2) Once Ubuntu finally establishes that I have a second moniter, i.e the TV, I set this to run at 1024 and make this main desktop, same resolution as my LCD?

You'll want to choose the native res for your TV, which depends on it's size and model.

> 3) I presume I can then remove the main monitor, so when I boot Ubuntu it will come up on the LCD screen, since it has already been configured?

Yes, that should be fine, just set the TV as the primary screen.
