### Post by hulkman on 2011-05-21
To enable greater CD / DVD support open Accessories;Terminal and enter the following two commands:
sudo apt-get install ubuntu-restricted-extras
sudo /usr/share/doc/libdvdread4/install-css.sh

Also, if you don't like the new "Unity" desktop, you can log out, choose "Ubuntu Classic" and log back in to get the old 10.10 theme that you may like better

-------------------------
[http://www.dotdeb.com](http://www.dotdeb.com) is a great place for deb packages.
