### Post by samden on 2008-12-16
When burning cds from Vista, don't use the default settings - they make cds that are intended to only work on XP and Vista (I understand it is possible to get them to work under Linux too but it may be a bit of fuss). Instead, click on "Show formatting options" and select the "Mastered" format - this is a universal format that all computers can use. You should have no troubles then.
