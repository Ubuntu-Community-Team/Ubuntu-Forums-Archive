Back up xorg.conf```
cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```
To edit xorg.conf, run```
gksu gedit /etc/X11/xorg.conf
```
In the red line, change "0" to "1".

Try enabling effects again.

Alternatively, commenting the line (putting a hash/pound/# in front of it) should work also.
