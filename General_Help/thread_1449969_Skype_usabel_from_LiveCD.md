### Post by Paddy Landau on 2010-04-08
To use Skype on a Live CD, you'd have to download it and then install it. You'd need a big amount of RAM to do that; as it's an ageing computer, I strongly suspect that the answer is 'no'.

Is the hard disk too small to make a dual-boot?

Or, you could install [Wubi]("http://wubi-installer.org/") and test Skype from there. If it works, you can uninstall Wubi and install Ubuntu. Be sure to make a full backup first; I recommend [CloneZilla]("http://clonezilla.org/") to back up your entire Windows drive in case you need to reinstall it.
