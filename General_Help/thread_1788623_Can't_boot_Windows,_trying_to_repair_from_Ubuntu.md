### Post by coffeecat on 2011-06-23
> **jo9100 said:**
> 

[LIST]
[*]Don't have enough space on my Windows partition
[*]Delete the stock "recovery" partition to make space
[*]Enlarge the Win partition
[/LIST]
How did you do this? With Gparted? If so, this is where your problems probably would have started. By enlarging the Windows C: partition to the left you would have done two things: moved the partition sector boot sector which Windows does not like at all, and reconstructed the NTFS filesystem, which looks as though it has gone wrong. For the record, resizing a Windows C: partition is best always done with Windows' own Disk Management utility.

Anyway - this seems to be where your problem is:

> **jo9100 said:**
> 
sda1: __________________________________________________________________________

    File system:       
    Boot sector type:  Unknown
    Boot sector info:  
    Mounting failed:   mount: unknown filesystem type ''

... and ...

> **jo9100 said:**
> 
/dev/sda1    *         16,128   127,700,684   127,684,557   7 NTFS / exFAT / HPFS

Assuming that /dev/sda1 is your enlarged Windows partition, the partition table shows that this should be a NTFS partition with the boot flag set, but the boot script cannot detect the actual filesystem itself within the partition - which means it is corrupted.

You need a Windows 7 installation disc (a Microsoft one, not an OEM manufacturer's restore disc) or a Windows 7 repair CD. Do you have either? It is possible to create a Windows 7 repair disc from within Windows 7, but obviously only when Windows is still functioning. Did you make a repair disc? If not, do you have access to another Windows 7 installation which you can use to create a repair disc?

You need either the installation or repair disc for the repair console, from which you would need to run two things. First a chkdsk, hopefully to repair the filesystem. This may have to be run several times. Second, you may need to repair the boot sector.

Normally, I would post a link to a site where you can dopwnload the ISO for the repair disc, but the download options have been removed "pending copyright clarification". I can also post links for more detailed instructions for the repair commands, but first please confirm whether or not you have access to either an installation or repair disc.
