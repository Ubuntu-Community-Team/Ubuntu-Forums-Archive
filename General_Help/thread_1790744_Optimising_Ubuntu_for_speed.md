### Post by nzjethro on 2011-06-25
As an easy step, you should make sure your computer isn't running any resource hungry background programmes. For example, when I watch HD movies on my (reasonably old) laptop, I have to disable Compiz, else my playback is very laggy.

As an more indepth solution, perhaps consider switching to a more lightweight DE (such as LXDE or Xfce), perhaps even reinstalling with an Ubuntu minimal install, then installing Xfce overtop.  Another option is a lightweight distro, such as Puppy Linux. Doing this will reduce computer resource use, which will mean that your programmes will run more smoothly.
