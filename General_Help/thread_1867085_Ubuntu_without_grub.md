Calm down and tell us what is going wrong!

Grub2 normally works brilliantly in my opinion, and I have 5 separate OSs on my machine all of which can be booted using grub2 from either of the two disks on which the OSs reside.  Once you get used to the way it works and make sure that you only configure it the correct way, it performs faultlessly.
