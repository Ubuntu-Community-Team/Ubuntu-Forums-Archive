### Post by mb_webguy on 2009-01-31
You're missing the "I'm Feeling Lucky" button?  Odd.

The easiest thing to do is probably make a new profile.  In the terminal, type "firefox -ProfileManager", and create a new profile.  This will give you a default configuration of Firefox.  You can export your bookmarks from your original profile and import them to the new one.  You can do the same with your saved passwords if you install the Password Exporter extension.
