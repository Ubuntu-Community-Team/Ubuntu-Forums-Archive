### Post by tommyhakinen on 2008-05-05
there are many transparent themes at gnome-look. Look for gtk-2.0 themes.

> [http://www.gnome-look.org/](http://www.gnome-look.org/)

but if you already have the theme and wanted to make the panels transparent, just right click on the panel. Select "Properties". then get to the "Background" tab, choose the "Solid Color" then adjust the sliding bar to "Transparency".

If you are using Hardy, I recommend you to use Compiz-Fusion. It can do most stuff that beryl does. here how to install it.

> sudo apt-get install compizconfig-settings-manager

enter your password when prompted.
