### Post by Sin@Sin-Sacrifice on 2009-11-16
If you want you can kill the process. ```
sudo killall programname
``` Or you can open the System Monitor and right click the process and select end or kill. Not sure if it's a common problem or not... I don't use it. Google it and search the forums to see if anyone else had the problem.
