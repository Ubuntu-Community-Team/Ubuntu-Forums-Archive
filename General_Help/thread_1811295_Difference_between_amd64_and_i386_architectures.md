### Post by BitJunkie on 2011-07-24
i386 refers to 32-bit and amd64 refers to 64-bit version. Software listed as amd64 should install on any modern system (that has a 64-bit CPU), whether the CPU is Intel or AMD.

### Post by Too Late on 2011-07-24
I can't find any reason to use these most confusing names, like AMD64, unless we're talking about specific manufacturer. Unfortunately, many Ubuntu related site use them.

As the Wikipedia article says (and as the 64-bit article is _[titled](http://en.wikipedia.org/wiki/X86-64)_), there are "vendor-neutral terms" like x86-64, which could be used as well.
