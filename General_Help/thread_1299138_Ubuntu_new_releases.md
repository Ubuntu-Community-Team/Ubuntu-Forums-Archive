### Post by flipper9 on 2009-10-23
To get the benefits of the new upgrade, you'll need to reformat.  That's not always the case with upgrades, but Grub2 and EXT4 can't be upgraded (well they can, but there are major hoops you have to jump through and issues). 

Of course, you can always stay with the older release.  Nobody is forcing you to upgrade.  Older versions still get bug fixes, security fixes, and backports of newer software versions.

### Post by kanikilu on 2009-10-23
Probably an obvious reply, but if you don't care to be on the bleeding edge and want to save yourself the headache of a clean install twice a year, why not stick to the LTS releases (every 2 years)? That way you have the option of continuing to use it if you don't want to "fix what's not broken" and not worry about security updates cutting off for, what is it, 3 years? Or better yet, do a server install, add the "ubuntu-desktop" metapackage and all the other stuff you want, and then you can even wait 5 years before reinstalling if you want :)
