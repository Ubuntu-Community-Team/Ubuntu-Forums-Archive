### Post by phidia on 2008-04-16
Several people here recommend [system recovery cd]("http://www.sysresccd.org/Main_Page"). There is also [recovery software]("http://www.data-recovery-software.net/Linux_Recovery.shtml") that works from windows for this purpose  I haven't used the windows based one, but hopefully one of those will help you.
Retrieving a deleted file is tricky you probably don't want to boot the system and definitely don't want to add/remove or change anything on that partition until you've tried the recovery procedures.
