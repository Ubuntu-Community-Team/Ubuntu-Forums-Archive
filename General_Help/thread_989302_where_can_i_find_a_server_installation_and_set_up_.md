### Post by cariboo on 2008-11-21
Here is another link to a server setup:

[http://www.howtoforge.com/perfect-server-ubuntu8.04-lts](http://www.howtoforge.com/perfect-server-ubuntu8.04-lts)

You can run as many services as you need, as long as you have the hardware to handle the load. I have a home server that runs:

Apache
Php5
Mysql
Samba
NFS

I also use it to store videos and music as well as business data, all my photos as well as a backup repository of my home directory. It is an old Athlon 1Ghz with 512Mb ram that I inhereted from a former employer. I added a 400Gb raid 1 array, plus a 160Gb and 80Gb drives
