### Post by sanderd17 on 2011-07-13
Wine was originally build as an emulator (as WINdows Emulator), but then they changed the name to "Wine Is Not an Emulator" because it was impossible to copy all the odd effects of all the libraries in Windows.

So there is currently no completely functional emulator, but Wine comes close. If wine doesn't work, you can run windows virtually inside Ubuntu with Virtualbox. That way you can work with windows programs and ubuntu programs together.
