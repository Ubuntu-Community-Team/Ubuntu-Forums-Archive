### Post by aysiu on 2007-05-07
Well, most of the applications that kill non-responding applications come preinstalled--they're not separate applications you have to go out of your way to install.

As has already been mentioned, there's *xkill* and the "force quit" applet. You can also, if you're using Gnome, launch *gnome-system-monitor*--it functions a lot like Control-Alt-Delete in Windows.
