### Post by Wim Sturkenboom on 2011-08-24
Please don't forget that this poor user might not get a grub menu if he/she ain't dual booting.

If you don't get a grub menu during boot but straight away end up in a console or gui, reboot and press <shift> (maybe <esc> works as well) or hit it several times and it should show.
