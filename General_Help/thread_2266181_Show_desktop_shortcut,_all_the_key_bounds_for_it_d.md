### Post by online-0227 on 2015-02-20
Hi.
Recently I got lastest LTS version of Ubuntu (14.04.1 maybe), before that I used to use SteamOS linux desktop.
I used to use Left Ctrl+Left Alt+D to minimize all the windows and show desktop.

But in Ubuntu, even though I bound that key to do so in Setting->Keyboard->Shortcut, it doesn't work.
Actually, it seemed all the key doesn't work except default key set, Left Ctrl + Right Window key(Super Key) + D.

Left Ctrl + Right Window key(Super Key) + D only works to minimize all windows and show desktop, other any key set doesn't work.

How to solve this?

Thank you!

=---------------- SOLVED ==============
1. DOWNLOAD CompizConfig from ubuntu softwarecenter and launch it.
2. go Ubuntu Unity Plugin -> change show desktop shortcut
