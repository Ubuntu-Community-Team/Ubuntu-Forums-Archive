### Post by phantom3113 on 2009-04-06
Sometimes, changing the login theme takes a few tries. I know when I first installed and tried changing the theme, it took 2 to 3 tries. Basically, make sure that under the "local" tab, you have the second drop-down menu option as "selected only". When choosing a new login theme, if you don't pick that option, ubuntu will automatically pick a few more themes that it will randomly cycle through as the login splash, unless told otherwise.
