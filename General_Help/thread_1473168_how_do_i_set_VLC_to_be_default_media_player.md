### Post by cariboo on 2010-05-04
Open nautilus (Places->Home Folder) then click Edit->Preferences->Media, to set the default media player.

---

### Post by WorMzy on 2010-05-04
Or right-click on a media file, click Properties -> Open With, and make sure that VLC Media Player is selected.
