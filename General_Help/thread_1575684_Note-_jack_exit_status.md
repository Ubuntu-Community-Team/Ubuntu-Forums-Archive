### Post by Vigh on 2010-09-16
Hi just thought for all you linux audio users out there, when doing a distribution upgrade for any linux system jack breaks and causes it to not be installed correctly. I fixed this problem by simply installing jackd through the terminal using the following command-

sudo apt-get install jackd

fixed jack following the distribution upgrade and now can use the jack server again
