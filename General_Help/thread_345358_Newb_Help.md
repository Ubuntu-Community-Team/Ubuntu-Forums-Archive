### Post by phiphedog on 2007-01-24
-Nicotine+ is a file sharing program that is almost exclusively music. It is a an open source version of the windows program "Soulseek"

 -You should use it because it is used by people that have quite extensive music collections,  you will rarely find a fake file or incorrectly named file, and it is, in my opinion, right up there at the top of p2p programs especially for music.

-You can open a terminal by going to Applications-->Accessories-->Terminal or navigating to usr/bin/x-terminal-emulator

-You can exit a terminal by typing exit or by clicking the X at the top corner of the window

-A quick Google search for Bash Tutorial or Bash Commands or Bash for Beginners will get you what you are looking for
