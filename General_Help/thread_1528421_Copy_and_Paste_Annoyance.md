### Post by sir_robert007 on 2010-07-10
Ok let me first say that this is more of a rant about copying and pasting in Ubuntu than an actual problem.

I was uploading a file to rapidshare that I was planning on sharing with a friend. After it was done rapidhare provided a link that I could email to friends so that they may download the file. I then copied it from the website window, then closed it. Then I went to paste it in a text file planning on emailing it later when I realized that I no longer had the option to paste. I then had to re-upload the file which had taken almost 3 hours. I find in Ubuntu you have to have the program from where you are copying text from open in order to paste. Its not like in windows where you can copy text from one program and close the window and still be able to paste somewhere else.

So is there a way where I can configure copy and paste so that it is more like Windows?

---

### Post by ajgreeny on 2010-07-10
Install **parcellite** which will keep lots of copied items and allow you to paste whatever you want.
