### Post by vav on 2007-05-03
The default in firefox will reflect the default in System->Administration->Printing

Pressing File->Print brings up a window with a drop-down print list in it.

### Post by pndragon on 2007-05-04
System Settings->Computer Administration->Printing does not effect non-KDE applications so firefox is not effected by any changes made here.
