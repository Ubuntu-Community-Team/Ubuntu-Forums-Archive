### Post by bobtfish on 2010-07-25
Files should be relatively easy. I haven't got mine like this any more, but if I remember correctly, you can (while in Ubuntu) click the Places menu at the top, go to Computer, and in there you should be able to open the Windows part of your hard drive (it might have a weird name, but should be there). Then you can just move over the files you want.

Programs are a different matter. Programs that work on Windows don't work on Ubuntu, but...
You have various possibilities. Some programs do have versions that will run on Ubuntu (e.g. Firefox, VLC). For other programs, there are usually alternatives (e.g. OpenOffice.org is a set of programs with pretty much the same functionality as MS Office).
For other programs that you really can't do without, you may wish to have a look at a piece of software called Wine, which, to a certain extent, allows you to run Windows versions of programs. However, I would say this is a bit of a last resort, as it can be a bit tricky getting some things to work, and for a new user it's going to be easier to find an alternative.

Look around the Software Centre (Applications -> Ubuntu Software Centre) which allows you to look for and easily install a large amount of software for a lot a tasks. What programs are of main interest to you?

Chris
