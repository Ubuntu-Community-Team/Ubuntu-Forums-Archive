### Post by prshah on 2008-08-26
> **kircher said:**
> I am currently using feisty or gutsy at home, but I do not have the internet yet at my new house.

You can snag the gutsy DVD, then use apt-cdrom to register it as a software source, and then use apt-get as usual to install the programs you need, from the DVD.

In fact, if you insert the DVD in the drive, you will be prompted by Ubuntu Gutsy to run package manager.

If you're looking for the Hardy DVD, then it's not available from the Ubuntu download site; you will have to search for an independent torrent or buy it from Canonical.
