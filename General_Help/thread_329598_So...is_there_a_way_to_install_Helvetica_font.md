### Post by fragos on 2007-01-01
Helvetica is a proprietary font.  Apple licenses it and Microsoft uses Ariel.  You can get Ariel by installing msttcorefonts.  There is a totaly open source set of fonts called DejaVu.  You may find you like Dejavu San Condensed.
