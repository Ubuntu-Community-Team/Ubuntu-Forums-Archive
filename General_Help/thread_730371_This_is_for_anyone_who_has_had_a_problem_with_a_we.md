### Post by wookietim on 2008-03-20
I recently broke down and bought a Dalek webcam.... It's really cool looking and has neat lights, but it wouldn't work with Kopete. I kept getting errors... When I ran this command, it started working perfectly! If anyone else is having problems, give this a try....

Bring up your Terminal:
type, sudo apt-get install libjasper-runtime
Enter
type in your password
Enter
then it should fetch and install.
