### Post by angkor on 2006-06-01
```
apt-cache search 7zip
p7zip - 7-Zip is a file archiver with high compression ratio
```

This is in the Universe repositories, which you need to enable first.

[https://wiki.ubuntu.com/AddingRepositoriesHowto?](https://wiki.ubuntu.com/AddingRepositoriesHowto?)

After you've enabled universe you can install the package using Synaptic.
