### Post by Bucky Ball on 2015-03-03
Probably normal. The point release has just come out so expect some of this for a little while. You will find you are actually on 14.04.2 and the kernel should be 3.13.0-46.75, or something like that.

If it's a problem set the Software Updater to only inform of updates once a week. Good luck. ;)
