### Post by speartip on 2014-04-28
Hi I had the same problem & seem to have solved it.
Right Click on a video file, open properties, click file type options, then embedding, tick the box "show file in embedded viewer" then click apply.
You may have to do this again for different file types, but it should play ok now.
