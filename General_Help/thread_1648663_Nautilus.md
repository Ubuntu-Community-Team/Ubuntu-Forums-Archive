### Post by coffeecat on 2010-12-19
Right-click on desktop and select 'Create a Folder'. Don't bother to name it anything special. Now right-click on the folder and choose 'Open with Other Application'. First, make sure the 'remember this application' box is ticked. Now look for 'File Browser' in the list and select that. That will reassociate folders with the file browser. You can now delete the temporary folder; it has served its purpose.

You probably accidentally associated folders with your media player by selecting it from the other application list with the tick box ticked.
