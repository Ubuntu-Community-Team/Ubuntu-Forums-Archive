### Post by Holger_Gehrke on 2018-08-12
Untrusted in this context means it's a package from a source outside the official Ubuntu repositories. Might be a PPA or a program specific repository outside the control of Canonical. 

'Lexical patterns' can either refer to data needed by one stage of a compiler (the lexer) or might be data for some linguistic tools (spell or grammar checkers for example) which split words into prefix, word stem and suffix according to these patterns.

Holger
