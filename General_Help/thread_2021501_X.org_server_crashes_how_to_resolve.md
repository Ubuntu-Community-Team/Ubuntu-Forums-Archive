Can you switch to a tty after the Xserver crash? Press ctrl+alt+2 to see if you get a terminal. To go back press ctrl+alt+7.

If you can, get the xserver to crash by loading gimp and switch to a tty. After login, type in the terminal:

```
cat /var/log/Xorg.0.log | grep EE
``` 

try and write down what the result of the command is and post it here.

```
