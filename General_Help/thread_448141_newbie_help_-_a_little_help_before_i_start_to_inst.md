### Post by tuxcantfly on 2007-05-18
1. It's the size of the disk image; the bigger you make it, the more files you'll be able to store in Ubuntu. Basically, the default should be OK for everybody unless you're into video editing or need to store lots of big files for some other purpose; if so, you should make it bigger.

2. You select "Ubuntu" once you reboot; the comments for the others were just because they already had ubuntu installed, but since you only have XP, you select "Ubuntu"
