### Post by Scunizi on 2009-03-03
You asked for advantages and disadvantages.  Well it's the only way to go if you're a gamer.  Many graphic intensive games will not run in wine on linux.  If you no longer or ever play games then the only advantage is you have another OS on your system that you can use to show other MS users how to do stuff with.

If you don't play games but need the functionality of.. say IE6,7 for work and access to IE only sites, then take your copy of win and install it in a virtual machine using Virtual Box or VMWare server. You'll still have access without having to reboot.
