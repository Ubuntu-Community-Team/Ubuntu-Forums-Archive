The formua is 

du+
--- =-iw0u+
dt

with '+' the subscript.

Here is what I input:
{d %mu_+ } over {dt} = -i %omega_0 %mu_+

It asks something after '+'. How do I input the '+' as a symbol, not an operant?

### Post by rampageoberon on 2008-06-02
```
{d %mu_+%} over {dt} = -i %omega_0 %mu_+%
```
