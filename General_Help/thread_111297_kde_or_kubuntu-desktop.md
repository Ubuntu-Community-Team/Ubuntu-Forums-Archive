### Post by mazelado on 2006-01-02
The kubuntu-desktop package is just a dummy package (it doesn't actually contain anything itself) that depends on all the other packages that make up KDE/Kubuntu. Your best bet for moving from Ubuntu/Gnome to Kubuntu/KDE is to install the kubuntu-desktop package. Even if you have KDE working now, it shouldn't hurt anything to install the kubuntu-desktop package. If you're lucky, you'll already have all the dependent packages installed.
