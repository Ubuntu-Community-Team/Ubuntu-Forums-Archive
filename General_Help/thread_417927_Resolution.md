### Post by Spr0k3t on 2007-04-22
There are a couple things you can do.  One would be to change the resolution of the fonts themselves so as to effect everything running at the system level.  Another would be to change the userChrome.css file which allows you to customize the way FireFox renders many portions of web pages and/or browsers.

An easy solution is to go to your Tools :: Options :: Content :: Advanced (Fonts & Colors) and change the minimum size there.  However, there are site developers/designers who use a fixed size font which is where you will need to override their settings through the editing of userChrome.css.

Hope this helps.
