### Post by Mark Phelps on 2012-08-26
The major problem I've seen folks reporting when using Linux alternatives to MS Office is when they get stuck with MS Word 2010 documents using the .docx format.  The Linux versions do a poor job of reading such files faithfully -- and, as a result, we see a LOT of posts about this problem, with no simple solution.

So, if she regularly receives Word documents in .docx files from others, she's going to have a problem moving away from MS Office.

I get problems with [FONT=Lucida Console].docx[/FONT] in Libre Office whenever graphics are involved. Sometimes graphics are duplicated; usually graphics overlay each other; and sometimes words that are there in the original just disappear in Libre Office.

The compatibility with [FONT=Lucida Console].docx[/FONT] is shocking when graphics are involved.

> **nothingspecial said:**
> … if you wish to use Apple products then you are better off using Windows or a Mac.
Absolutely. Apple does not support Linux, period.

> **magtrask said:**
> There must be a way to run iTunes on Ubuntu.
Well… yes and no.

If you install [PlayOnLinux]("apt:playonlinux"), you can easily install iTunes version 7 (yes, version 7), but do not expect it to run without problems. If you're lucky, it will sort-of work and won't delete everything off your iPod. I don't think that you'll manage to install a version later than 7.

If your friend's hardware is sufficient (especially plenty of RAM), you can load Windows in a virtual machine and run iTunes from there; but it's not handy to do it that way.

Short story: For iTunes, use Windows or Mac.
