### Post by TheFu on 2019-07-21
What protocols does the My Cloud support?
If it supports NFS, use that. NFS storage behaves like local storage on a system.

Be very careful enabling external access to files on a My Cloud.  There have been a few bugs in the firmware that provides everyone on the internet access to the files.  
[https://www.techradar.com/news/wd-my-cloud-nas-boxes-found-to-be-vulnerable-to-online-hacks](https://www.techradar.com/news/wd-my-cloud-nas-boxes-found-to-be-vulnerable-to-online-hacks)
and there were other issues in prior years.  I wouldn't trust any similar devices like this to be secure.  Please be very careful.
