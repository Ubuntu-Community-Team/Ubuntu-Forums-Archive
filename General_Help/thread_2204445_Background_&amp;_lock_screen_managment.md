### Post by Dreamer Fithp Apprentice on 2014-08-12
If you're still interested:
Possibly I misunderstand your question, but for me, when I lock the screen, I get the same "hack" (screensaver graphic) I'd get if it kicked in because of inactivity.  A plain black screen is one of the options, but there are many more. The fellow that maintains it strongly suggests using the included gui to change the "hack". If xscreensaver isn't already running, start it and invoke the management gui thus:```
 xscreensaver &
xscreensaver-demo
```If xscreensaver is already running, skip the first line and just do```
xscreensaver-demo
```I know it sounds strange but that program is used both to preview what the "hacks" look like and to set prefs for xscreensaver. You can do it by editing the congfig file also but that isn't necessary unless you are doing something out of the ordinary, like adding your own custom written hack.
