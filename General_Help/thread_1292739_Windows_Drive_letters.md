### Post by StuartN on 2009-10-16
> **satish_j said:**
> Is there anyway to tweak ubuntu so that it displays the name of Mounted windows partition

If you have given your partition a label (right-click and select Properties in Windows), then that name will be used when mounting the drive using Nautilus file manager.
