### Post by One Quick Question on 2005-12-31
In Microsoft and Apple operating systems, there is only one desktop environment.  You don't get much say in it.
For Linux, there are a few.  The two major ones are Gnome and KDE.
(Check out [http://en.wikipedia.org/wiki/Desktop_environment](http://en.wikipedia.org/wiki/Desktop_environment) for what exactly is meant by "desktop environment")

Ubuntu uses Gnome.  Kubuntu uses KDE.  That's the only difference.

You might like KDE better.  I do.
If you want to test it out, sudo apt-get install kubuntu-desktop.  Wow, I say apt-get like a verb now.

There won't be any underling difference to the operating system if you change DE's.  (Although KDE has cool features like kioslaves that may seem like it.)  So for example, XP games still won't play.

That, I hope, is a sufficient answer.   If you want to know more about the difference between KDE and Gnome, there are plenty of flamewars you can Google up :razz:.  The best way to find out is to check it out yourself though.

### Post by GeneralZod on 2006-01-01
[QUOTE=stimpack]Some people like the simplicity of Gnome. Others like me want to be smacked around the face with features.[/QUOTE]

Hehe - couldn't have put it better myself! 

KDE is *very* configurable and most of the options are not particularly hidden from view, leading people to call KDE "cluttered" and messy, which is actually a pretty fair point - even as an absolute out-and-out KDE whore, I have to admire GNOME's gorgeously clean, spare looks.

Also, as mentioned, most of the Ubuntu development works goes into GNOME, so the KDE that comes with Kubuntu is usually fairly rough in comparison, and may contain large bugs - for example, in the original release of Kubuntu 5.10, USB pens etc would not auto-mount properly, so we needed to wait for a patch.  Such a lapse would be pretty much unthinkable with Ubuntu.
