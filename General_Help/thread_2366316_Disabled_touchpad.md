### Post by Keith_Helms on 2017-07-16
What flavor and version of Ubuntu are you using?  You should be able to navigate to the settings menu where you disabled the touchpad by using keyboard shortcuts and then the tab key to get to the correct option one you have the screen up and hit space to check/uncheck the setting.

For example, on my Xubuntu 16.04 system I disabled the touchpad, then hit alt-F1 to bring up the application menu, used the cursor keys to select "Settings Manager", then used the cursor keys to select "Mouse and Touchpad", hit tab until the "Enable this Device" setting was selected and hit space to check the box.  That reenabled it.
