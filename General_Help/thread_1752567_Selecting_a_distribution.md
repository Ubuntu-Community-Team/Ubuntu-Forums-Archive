I have a laptop that has about the same specifications.  I run lubuntu (LXDE) on it.  You can go to [http://lubuntu.net/](http://lubuntu.net/) and make yourself up a live cd.

As mentioned, lubuntu is not a Canonical-endorsed product.  Nevertheless, lubuntu uses all the same ubuntu-debian repositories atop the standard bash shell.  If you need support, you might find it here or on a specialized LXDE board.  LXDE is definitely a more obscure DE, so you may not find the help you need among the endless lists of Unity how-to questions found here on ubuntuforums.  Also, LXDE requires some knowledge of xml configuration files, which might be a bit intimidating for new users.  Give it a shot, anyway.

XFCE (xubuntu) really requires at least 1 gb of ram to work well.  Others might disagree, but that is my experience.  I strongly recommend running xubuntu rather than ubuntu Unity on newer dual core machines with 2gb or more RAM since XFCE is much lighter and easier to work with than either Unity or Gnome3.
