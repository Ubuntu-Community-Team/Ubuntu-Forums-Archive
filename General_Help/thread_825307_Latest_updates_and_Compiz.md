### Post by strick242 on 2008-06-10
I installed the latest updates today (June 10, 2006) for Hardy.  I usually update it everyday, so it would be today's updates.  After a reboot, Compiz is totally unusably slow.  Everything worked fine prior to the updates.  I have removed and reinstalled the video driver (intel for a 945gm chipset.)  I also rebuilt my xorg.conf with the command "sudo dpkg-reconfigure -phigh xserver-xorg"  If I enter the command "metacity --replace &" everything seems to work fine, just no eye candy!  Any help would be greatly appreciated.  I don't remember the exact updates or know how to find out what they were, but I remember something about the Gnome Display Manager.:confused:

Never mind...I seem to have figured out the solution.  I had to disable the effects via System>Preferences>Appearance, then re-enable them and set my Compiz options up again...seems to be working fine!:guitar:
