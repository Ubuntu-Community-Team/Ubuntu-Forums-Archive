### Post by brentoboy on 2007-04-04
edit your xorg.conf

this area is the area that needs help
```

Section "Monitor"
  Identifier "LS902U"
  Option "DPMS"
EndSection

Section "Monitor"
  Identifier "aticonfig-Monitor[0]"
  Option "VendorName" "ATI Proprietary Driver" 
  Option "ModelName" "Generic Autodetecting Monitor"
  Option "DPMS" "true"
EndSection

```

you've got 2 monitors listed.   Both are set to "auto detect" rather than listing exact horizontal and vertical refresh rates.

looking lower in your config file....
```

Section "Screen"
  Identifier "Default Screen"
  Device "ATI Technologies, Inc. Radeon R300 ND [Radeon 9700 Pro]"
  Monitor "LS902U" 
  DefaultDepth 24

```
it seems to me that the only monitor being used is the first one listed there.
I would comment out the second one using number signs...

```

Section "Monitor"
  Identifier "LS902U"
  Option "DPMS"
EndSection

# Section "Monitor"
#  Identifier "aticonfig-Monitor[0]"
#  Option "VendorName" "ATI Proprietary Driver" 
#  Option "ModelName" "Generic Autodetecting Monitor"
#  Option "DPMS" "true"
#EndSection

```

and I would look up your vertical and horzontal refresh rates and specifically state them in xorg...
I cant find your monitor's specs, but I did find this page that offers to sell you a pdf of the manaul (how stupid is that?)
[http://www.manuals-in-pdf.com/advanced_search_result.php?keywords=Iiyama+LS902U&amp;forward=plist](http://www.manuals-in-pdf.com/advanced_search_result.php?keywords=Iiyama+LS902U&amp;forward=plist)

Anyway, once you know your refresh rates, you can plug them in like this (here is mine for example)...
[code]
Section "Monitor"
    Identifier     "SyncMaster 213T"
    HorizSync       30.0 - 81.0
    VertRefresh     56.0 - 75.0
    Option         "DPMS"
EndSection
[code]

if they are plugged in there explicitly, X will stop rebuilding them and making safe estimates when it is unsure what is the best refresh rate.

by the way the overall refresh rate (like 85 hz) is derived from a combination of the Horiz & Vertical ones.  That is why there isnt a quick "just set it to 85hz" setting in xorg.conf.
