### Post by MARP1961 on 2011-08-15
You will be better off waiting until Ubuntu 11.10 (Oneric Ocelot) is released before trying Gnome Shell. Have a look at this You Tube video review: [http://www.youtube.com/watch?v=ALCUPp-5UwE]("http://www.youtube.com/watch?v=ALCUPp-5UwE")

In the meantime, if you do not like Unity then you can simply go to your user name on the top right hand corner of the top panel and click. Log off and then click your user name again. At this point you should get an option to log in to Gnome Classic which will give you a traditional two panel Ubuntu desktop.

You can install Gnome Shell in Natty Narwhal (as I have done) but there is a risk it might get rather complicated as it involves installing an extra ppa which will mean you can only run Gnome Shell. You only have to wait until late October to try Gnome Shell in Ubuntu. It will be as easy as installing from the Software Centre.

If you are very impatient, then try a live CD of Fedora 15 which already runs Gnome Shell. Finally, you need a computer that is capable of running 3D graphics to use it. Since you seem to be running Unity then I assume you already have this.
