### Post by angkor on 2006-06-05
Don't be all paranoid. :) 

This package gives *you* the choice / freedom to choose for unattended upgrades if you'd want that. By default it is not enabled of course.

Go to System -> Administration -> Software properties. Look at the 'Internet update' tab. There you can choose to install security updates unconfirmed.

If you're still paranoid just delete the package anyway. Ubuntu-desktop is just a metapackage and your system will not brake when you remove it.

So no infringement to your freedom to choose here.
