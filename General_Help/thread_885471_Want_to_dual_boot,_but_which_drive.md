### Post by WRDN on 2008-08-10
I would put the 2 OS' on the raptor drive, giving each around 33/34Gb. This value may have to be reduced so a swap can be made. Then, format the new 640Gb drive- I would personally create a /data partition because, if you chose to install another distro in the future, there may be some issues if you were mounting the 640Gb drive as /home.
