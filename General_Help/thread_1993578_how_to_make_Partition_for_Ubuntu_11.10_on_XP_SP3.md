### Post by davidvandoren on 2012-06-02
First, make sure you have enough empty space on your windows partition.
Then, defrag your windows XP several time till it shows all space further to the end of the disk empty. 
Next insert your linux install disk and at the installation dialogue chose to install Ubuntu and windows side by side.  

Play it save and don't use too much space for the Ubuntu partition. You can move a slider at the install dialogue to adjust this.
