### Post by Elktro on 2009-11-30
I don't know how to make win-key to show panel menu. But at least Win+1 can be set up as follows:

1. Go *System* -> *Preferences* -> *Keyboard Shortcuts *
2. Click at *Show the panel menu* (under Desktop) the *short cut*
3. Press the key combination e.g. Win+1

Actually Win key might work directly as ordinary not as modifier key. Just try it out.

Note If you cannot use Win-key as modifier key:
1. Go *System* -> *Preferences* -> *Keyboard*
2. *Layouts* tab
3. Select your layout 
4. click *Other Options*
5. click Alt/Win key behavior
6. Super is mapped to the Win-keys.
7. Try again
