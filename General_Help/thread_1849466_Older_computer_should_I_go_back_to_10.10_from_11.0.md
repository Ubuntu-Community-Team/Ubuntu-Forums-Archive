### Post by Topsiho on 2011-09-24
With the specs your computer has, it should be able to run well with any Ubuntu, new or older versions. I have a computer that's 6 years old now, and I used Ubuntu until recently on a computer of 1998, of which now the hardware is exit, not by failure of Ubuntu.

With very new hardware there might be a problem with drivers not yet available, but for older hardware that problem usually doesn't exist.

With 11.04 the new interface Unity expects a graphics card with 3D-acceleration, while the open source driver doesn't yet always deliver this. So you'll have to use the driver from the manufacturer, if for the older card this exists. So that might be a problem.

An LTS version is meant to be more stable than the experimental versions in between. They get rather, ehm, well, oldfashioned though, in the long run. So it might be a good idea to try Ubuntu 10.04.3 (the latest "service pack" of 10.04), be sure to use that one, to prevent having to install too many updates after the install).

On ther other hand: new hardware, such as my new USB wifi adapter, doesn't work with 10.04, and works well in 10.10 and 11.04

You mentioned Kubuntu: Read Kubuntu everywhere I wrote Ubuntu, if you want to use Kubuntu.

Topsiho
