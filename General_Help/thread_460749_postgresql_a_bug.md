### Post by jhofmann on 2007-06-01
I have postgresql running on one of my machines as a database server.  As I remember, I had to configure postgresql using its own instructions and software.   I certainly don't remember the details, but perhaps if you went to the postgresql site, they could help you.  Postgresql is a large set of programs, directories, configuration files, and so on.  It can't be configured completely by Ubuntu.  

One additional caveat:  Postgresql default configuration and directories will be different from the directory structure of Debian/Ubuntu.  This means that what you see on the postgresql site will not be exactly what you see on the ground.  As I recall, all the procedures are the same (Create a user,  create a database, and so forth), but the directories will be different in Ubuntu than they are in postgresql installed from the tarball.

Postgresql takes a little effort to get up and running.  

Hope this helps,

Jim
