### Post by ShellsOnTheFloor on 2007-03-29
Try accessing the BIOS. This is usually done by tapping F1, Delete, or Escape right as the computer turns on.

When you get to the BIOS (it's a big menu with grainy text, you'll know it when you see it) try finding the device startup order. In most setups, it's something like Hard drive, then Removable (floppy), then CD, then Network. Change it so that the order is CD, then Hard drive. Hit whatever button the BIOS says is Save and Close (on my setup it's F10), and then the computer will restart. If the Ubuntu disk is in the drive, it should boot from the disk.
