### Post by jerrrys on 2012-03-15
Yep, congratulations.  You got Oneiric.

If you want the old style look you need to install Gnome-Classic.

Open a terminal and enter:

sudo apt-get install gnome-session-fallback

Then reboot and choose gnome classic at boot (click on the gear).
