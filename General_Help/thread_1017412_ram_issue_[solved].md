Hi!

Several days ago, I had a problem with installing/trying Ubuntu from the live cd, and I first tried to use Wubi. The install worked, and then after rebooting, the desktop wouldn't load. It would check the install fine, and then I would get the brown desktop with the "goat" thingy...what was missing was the grey bars at the top and bottom and the icons...only the wallpaper would load. 

I was doing this with only 256 mb of ram (which the ubuntu website says is fine). Now that I have upgraded to 768mb of ram, it is working fine from the live cd (haven't tried to install yet). I don't know if this is an obvious fix, but the additional ram seemed to do the trick. Just thought I would let the new users know to check their ram.
