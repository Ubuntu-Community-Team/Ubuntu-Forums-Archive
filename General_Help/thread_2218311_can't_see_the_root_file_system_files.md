### Post by grahammechanical on 2014-04-21
I am not sure what the problem is. So, I am guessing. The Ubuntu 14.04 File Manager (Nautilus) should open and show a panel (side bar) on the left side of the application window. That side bar will have three headings; Places, Devices and Network. Under Devices you will see Computer. Open computer and you will see all the system folders. In earlier versions of Nautilus Computer was called File System. Are you using Ubuntu+Unity? That is what I am using. I cannot advise on other user interfaces that you may have installed.

To show hidden files go to the File Manager View menu. Or press Crtl+H. To show or hide the side bar press F9
