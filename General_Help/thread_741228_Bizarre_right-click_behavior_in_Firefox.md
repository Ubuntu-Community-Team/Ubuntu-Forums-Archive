### Post by artblakey on 2008-04-13
I'm assuming a lot with this solution... but are you by any chance using the Nightly Tester Tools extension to make the Mouse Gestures extension compatible, because the Mouse Gestures extension is buggy in Firefox 3, and can cause problems exactly like you're describing. If you are, then try disabling Mouse Gestures and restarting Firefox to see if that solves your problem.

Kind of a long shot on my part but you never know, and if you still want to be able to use mouse gestures in Firefox 3 try installing FireGestures as a replacement for Mouse Gestures, it offers virtually the same functionality and works fine in 3.04b.
