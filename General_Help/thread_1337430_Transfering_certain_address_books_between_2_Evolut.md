Running Ubuntu 9.10

I have Evolution on one primary computer and on a secondary computer. I need to update 3 of 4 address books on the secondary computer with address books from the primary computer, such as address books entitle: Business Contacts, Career Contacts, Associate  Contacts.........but not Personal Contacts.

Other than typing these in by hand from a printout, how can I transfer these three sets of contacts from my primary computer to the secondary computer so that it is updated?

Or if just transferring 3 address books of 4 is not possible, then how do I transfer all 4 address books?

### Post by plucky on 2009-11-25
> how can I transfer these three sets of contacts from my primary computer to the secondary computer so that it is updated?

Open Evolution and select **Contacts**.Then select an address book and open **File > Save Address book as Vcard**.

This will create a file called <Name of Address book>.vcf in your home directory.Then move the file to the second computer and again under contacts,import the file into the address book.

It will import all of the contacts,so you might have to delete the address book first,or all the contacts that were there originally will be duplicated.

Do the same for the other address books.

Good Luck
