### Post by ad_267 on 2009-09-06
Gif as in an animated gif? Otherwise you should use png, and Inkscape is designed for gif/png type images more than Gimp is.

When you say you're creating rectangles, it sounds to me that what you want to use is a vector editor rather than a raster editor like Gimp.

Gimp doesn't store shapes like a vector editor does, it stores the colour value of all the pixels in an image, so it has no idea that something is a rectangle. The best way to work with Gimp is to split your work up into separate layers, that way you can easily edit one layer if you messed it up, without having to redo work on all your other layers.

### Post by twright on 2009-09-06
As others have said the reason you cannot change existing shapes in gimp is because like most raster image editors, to it they are just grids of pixels - the same would be the case in photoshop or any other raster image editors.

If you are trying to learn I recommend you work through some tutorials first. This one for example explains making basic animated gifs: [http://www.gimp.org/tutorials/Simple_Animations/](http://www.gimp.org/tutorials/Simple_Animations/)
