### Post by Rubi1200 on 2010-11-05
My suggestion would be to install Windows, Backtrack, Ubuntu in that order.

However, if that is not possible or convenient, then install Backtrack telling it either not to install its bootloader (if that option is available) or reinstall GRUB when finished (as suggested above).

This link describes how you can do it:
[https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD](https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD)
