### Post by pirateghost on 2010-02-23
you are new but are using 2 different desktop environments?  you should find one you like and stick with it, or you will get confused far too easily.  once you have a grasp on things, then you can move onto other desktop environments....

what program are you trying to install?
most software that you will need can be found in the add/remove applications or in the synaptic package manager
if you download a .deb file you can doubleclick it just like in windows and install it
if you download a .exe file, you will have to run it under WINE which is a windows emulator of sorts.

before you install it in WINE, make sure there isnt a linux alternative to the program you want, my guess is there usually is.
