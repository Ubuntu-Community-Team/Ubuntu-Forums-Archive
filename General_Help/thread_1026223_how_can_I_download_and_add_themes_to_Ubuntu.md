### Post by -=-+ w1r1zu +-=- on 2008-12-30
You have to download themes. As what they've said, they have to know if you are using Ubuntu or Kubuntu.

I haven't tried Kubuntu for a long time due to major issues on the OS.

In Ubuntu, you can dowload themes from Gnome Look ([http://www.gnome-look.org](http://www.gnome-look.org)) and other as well. Just google search for it. Everything is there.

Be sure you download files with the extension .tar.gz The Ubuntu's famous file extension hehehe.

Just put it first on your desktop and then click Settings > Preferences > Appearance > Theme Tab.

You can see the different themes pre-installed there. Now I just want you to drag the downloaded file, the .tar.gz file unto the themes. If is asks for something like Apply Theme or Keep Current, congratulations it's installed. Otherwise, like some errors, it's got a package error.
