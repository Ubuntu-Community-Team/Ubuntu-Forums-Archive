### Post by tommcd on 2006-08-30
Welcome to ubuntu!!
For some good beginner info see this:
[http://psychocats.net/ubuntu/](http://psychocats.net/ubuntu/)    also:
[http://monkeyblog.org/ubuntu/installing/](http://monkeyblog.org/ubuntu/installing/)

To install vlc player, make sure universe repository is enabled in synaptic (the second link explains that very well for beginners) and just search for vlc in synaptic (System>Administration>synaptic package manager).
Click on it, select 'mark for installation', click apply, and you're done! Any software from the repos can be installed that way!
