Anyway, after some looking around and trying different scripts and programs, the only thing that worked for me was a firefox extension (which unfortunately requires registration - which fortunately takes really 5 seconds. So, the extension is [https://addons.mozilla.org/en-US/firefox/addon/11037](https://addons.mozilla.org/en-US/firefox/addon/11037)

I dont know if you can install it from addons.mozilla.org . I logged in yet found no install button.. had to go to developers website.
