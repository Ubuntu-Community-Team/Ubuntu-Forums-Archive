### Post by eilios on 2009-04-10
After wiping Windows off my system partly because I dislike windows, partly because of a hardware failure, I reinstalled Ubuntu, and noticed some things that didn't work before(Mokoi Gaming) ran flawlessly. It turns out Wubi does not run certain things. If you find things aren't running and you have Wubi, it might be a good idea to move Wubi to it's own partition.
