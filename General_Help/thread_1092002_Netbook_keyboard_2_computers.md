### Post by pytheas22 on 2009-03-09
I'm not sure exactly what you want to do, but if you go to System>Preferences>Remote Desktop, you can enable desktop sharing.  Then go to Applications>Internet>Remote Desktop Viewer on the other machine in order to connect to it remotely.  This sounds like it's what you're looking for.
