### Post by inselaffe on 2008-03-18
You need gDesklets installed:
```
sudo apt-get install gdesklets
```
Download the desklet from that site to somewhere. No need to unpack. 
Once installed, run gDesklets shell: Applications>Accessories>gDesklets
In gDesklets shell, select File>Install new package. browse to the file you downloaded. Click Open.
The desklet will appear on the right. Double-click it to launch on your desktop.
