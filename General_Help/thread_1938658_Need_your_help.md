IMO if you are interested in gaming then you should be more concerned about whether Windows supports it and not Ubuntu.

Though there are games that can be played in Linux and ones that cant can be played in Wine sometimes you are better off building a gaming pc around Windows and not linux.

If you want the best graphics and directx etc then Windows is your only real option.
