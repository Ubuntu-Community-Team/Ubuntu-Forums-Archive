### Post by akakingess on 2009-10-10
What are you trying to do, keep the wireless open and thinking that you are using it so it doesn't go to sleep or something of that nature? If so, back when I had a cellular wi-fi, to keep it open we would set up a ping in the background that would constantly ping whatever you want just so that there is some sort of activity. I think if you say for example ping [www.google.com]("http://www.google.com") -l , that is the letter l, it will ping until you tell it to stop (this is me digging into memory banks so definitely double check as far as the -l goes), but that should at least keep the connection open showing activity. Please let me know if this answers your question or if I completely misunderstood (as is known to happen ;)
