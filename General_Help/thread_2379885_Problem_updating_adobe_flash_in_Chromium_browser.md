### Post by ajgreeny on 2017-12-11
Make sure you have enabled the Partner repositories by going to **Software & Updates ->Other Software** tab and select the tick-box for Canonical-Partner repos, refresh the repositories with **sudo apt-get update** then try the installation again.

If you currently have an older version of flash installed by some other means remove that first and flash should install and work OK.
