### Post by slooksterpsv on 2011-02-08
I personally recommend learning Python first. It's easy, powerful, and great for beginners. Actually programs like Exaile (music player) is built with Python, some research labs use Python for their data, etc.

C++ is awesome, but not my cup of tea, but if you want to learn it, a really good book is C++ For the Absolute beginner. It's amazing, teaches you a lot, and makes it "game" oriented so if you want to go into the gaming side, you can see how things work together for games.

Java is good, cross-platform, cross-hardware (mobile, desktop, etc.). Granted some items may not be available in the mobile version, but still.
