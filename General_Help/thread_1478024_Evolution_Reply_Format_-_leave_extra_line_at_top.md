### Post by I.T. Super Man on 2010-10-11
It is possible to reply in the manner that you describe.  Being that I haven't used Evolution for very long as I prefer Kontact, I do not know when this feature was enabled.  However, it is possible with Evolution version 2.28.1.

I had to enable two features in Evolution to accomplish this.

Both settings are in: Edit, Preferences, Composer Preferences
```

1) Set reply style to "Inline (Outlook style)"
2) Check mark "Keep Signature above the original message on replying"
```Now, when you reply to a message, you cursor will be above the previous email.

In all actuality, you only need to enable the "Keep Signature above the original message on replying" to accomplish this.  I simply prefer the Outlook style, not because I'm a MS zealot.
