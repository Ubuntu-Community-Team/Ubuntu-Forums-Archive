### Post by TBABill on 2012-04-26
If you're running the proprietary video driver, if available, and experiencing the sluggish behavior, it may be time to consider a different desktop environment due to the compiz issues you experience. Xfce would "feel" closest to what you are used to with 10.04 and may use fewer resources, while LXDE would certainly use far less. Cinnamon is similar to the old Gnome 2 Linux Mint setup with 1 panel, but its resource usage is still a bit high right now. KDE may be a bit too heavy as well.
 
I'd suggest doing a long trial run with Xfce and LXDE and see if you like either as an alternative to Gnome.

### Post by TBABill on 2012-04-26
While your CPU and RAM usage was the same between Ubuntu and Xubuntu on 10.10, running Unity greatly increases CPU demand from Compiz that is not present in Xfce. You can enable and run Compiz in Xfce, but it's not default and it is still not supporting Unity in Xfce. That's what creates the drop in demand between X/Ubuntu from 11.04, 11.10 and now 12.04.
