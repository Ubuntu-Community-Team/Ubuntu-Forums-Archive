### Post by metalf8801 on 2010-11-27
It sounds like the partition Ubuntu is installed on has run out of space. I had the same problem when I shrunk my Ubuntu partition. Can you make the Ubuntu partition bigger? If you can't you can still log in without a gui by pressing Ctrl Alt F1 (well you can press any F key from F1 to F6) you can then use the command line to find and delete files you don't need. 

I hope this helps. If you have more questions please ask. Like if you need help using the command line. 

Good luck 
Dan


Edit if you are going to shrink Windows 7 (I think) you should use partitioning software that is built into Windows Vista and 7 because it will decrease the odds of having a problem with your Windows install.
