### Post by SeanHodges on 2009-09-22
> **bash321 said:**
> Wine is a program that im sure some of the linux community don't like. But WINE needs to be implemented for ADOBE CS4.

as ADOBE is a key software program for manipulation of pictures videos and graphic drawings! 


wine needs to be further developed for Adobe CS4, so that novice's on linux can still run there own windows programs.

Most people are well aware of what the Adobe CS4 software suite is for, how useful and/or necessary it can be for many people, and how great it would be if Adobe supported it in Linux.

But they don't.

Wine is an incredibly complex project, and whilst it is improving all the time it is always playing catch-up with Windows. For example, Adobe Photoshop CS2 only started to work reliably around 2008: [http://appdb.winehq.org/objectManager.php?sClass=version&iId=2631](http://appdb.winehq.org/objectManager.php?sClass=version&iId=2631). The lack of support for CS4 products in Wine is not because people are too busy or don't care, it's because there is simply not enough support from Adobe to make it happen.

You have 3 choices, and you don't have to pick just one:

1) Hassle Adobe LOADS, get involved in testing Photoshop/Flash CS4 in Wine and reporting your findings, and consider campaigning for awareness of the lack of Linux support from Adobe to people outside of the Linux community. I would suggest graphic designer magazines, Adobe forums, PC magazines, Adobe affiliates/partners, blogs, basically everywhere.

2) Find alternatives for the Adobe products you need. Gimp is a great image manipulation tool, and some have also found Krita a good alternative. Flash CS4 alternatives will be more tricky to find, it will really depend on your requirements - ask around to see what others are using.

3) Use Microsoft Windows when you want to use the Adobe CS4 suite. CS4 is developed, packaged, and sold as Windows-only, and as long as Adobe keep it that way your most hassle-free option is to dual-boot or install Windows in a virtual machine as others have suggested. Using and promoting Ubuntu is great; but the bottom line is you need your computer to be able perform the tasks you require from it, and if that must involve buying Windows and Adobe CS4, then so be it.
