### Post by DJ_Peng on 2008-03-09
Since you're using a *beta* of Firefox 3 then I trust that you're aware that Firefox 3 stores bookmarks in a database, not a flat file like Firefox 2 did. You need to go into **Bookmarks > Organize Bookmarks**, then when the Library window opens select **Import and Backup > Export ...** to get your current bookmarks in a file that Epiphany should be able to use.
