### Post by geuis on 2006-11-08
This is more of an idea I just had, than a problem. But I'd like to get feedback.

One of the folks I provide support for had burned a copy of Derik's Boot and Nuke(DBAN) and had a problem with running it, which we fixed.

It made me think about a possible security setup, doing a dual boot between DBAN and your regular OS, either Linux or Windows.

I have a dual boot and home and if I dont choose Windows, it goes into Ubuntu. 

So suppose I had DBAN, or something similar, installed on a 2nd empty hard drive(one that I never use for data) and use another drive for my OS(s). Make sure DBAN is first in the boot list. Every time I reboot, I HAVE to choose my OS or it will automatically load up DBAN and woosh, there goes my OS and all the data on it.

It would suck if you dont have some secure automated backup and you forgot that 1 time to select your OS. However, if you're in a pinch and have to erase your HDD fast, it might be a good solution.
