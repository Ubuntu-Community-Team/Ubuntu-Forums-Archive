Artha is an off-line thesaurus based on WordNet.
It will give you close matches if your query is not found.
Very nice app.
Sits in the panel and can display the meanings to highlighted words by hitting ctrl+alt+w.
Think it's in the repos but if it's not, there is info and a .Deb file here:
