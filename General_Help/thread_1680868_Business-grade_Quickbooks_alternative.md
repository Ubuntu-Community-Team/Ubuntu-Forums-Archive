### Post by frotzed on 2011-02-03
Hello all.  I'm a full-time PC tech and I've been in love with Linux for years.  However, the vast majority of my customers are small businesses using Windows.  Now, I understand that the vast majority of Windows functionality can be reproduced (if not exceeded) by Linux but I'm drawing a blank in one area of software: Quickbooks.

Quickbooks, I find, is virtually ubiquitous to small businesses and it's a function I'm just not sure can be replaced by an open source alternative.  In order to be a contender, the open-source software would need to have the ability to centralize the company's data on a server (either in a file like Quickbooks or even in a database) so each workstation could have a local copy of the software but all be using the same database so as to keep synchronized as a company.

A quick google search turns up several pieces of software but I was hoping that by asking this question here I might get some input from folks who have some hands-on experience with such software and could share their experiences with me.

Thank you in advance for your time and input.
