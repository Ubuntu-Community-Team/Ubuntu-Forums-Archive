### Post by byen on 2006-05-09
hey Swaroop
The most easy way to install Fonts (i.e Telugu as provided by vaartha or eenadu) is by doing the following:
1. Download the fonts from vaartha or eenadu  (since they seem to have their own set of fonts that they need you to use *.ttf)
2.Open your Home Folder> hit Control+h to show all the hidden folders (the ones that start with .foldername) and look for a folder named .fonts ... If you dont see it.. create one by the name .fonts and then copy the .ttf font that you have downloaded and paste the file into the folder. Close your browser and restart it get the fonts working.

I use vaartha and eenadu to get news from back home and this was the method i used. I hope it helps. Let us know if you run into trouble.

Goodluck!
byen
