### Post by silkstone on 2008-08-28
1. You don't need to. There are no known/active viruses for Linux - welcome to the new world! If you tranfer files to a Windows machine, the Windows virus checker should identify any malware if it is configured properly.

2. There are thousands of packages in the repositories, and these can be installed using Synaptic. I'm not sure what Adobe application you want - you can install Adobe Reader (acroread) from the repos, but most Adobe software will not run on Linux except perhaps through Wine.
