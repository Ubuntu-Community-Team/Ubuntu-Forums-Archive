### Post by llawwehttam on 2010-01-08
To be honest it is still in development ie its grub 1.95 not really grub 2. The main difference is that there is no menu.lst to edit and you have to run update-grub if you make changes.
If you use a grub password I do not recommend upgrading as that's one of the things still being worked on.
