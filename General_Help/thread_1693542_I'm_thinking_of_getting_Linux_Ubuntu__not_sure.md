It really comes down to how you want to use your PC. Linux is not a Windows replacement, it's an alternative. I've been using Ubuntu for some time now and love it (since 9.04). I actually just built a new Ubuntu machine and even added a BR drive. Works great for accessing the data on the discs. :D

I play games on consoles; watch/listen to things from my BR player, console, or Boxee Box; and use my PC as a computer :o LOL (with some fun thrown in too of course)- it accesses, creates, and manages all my "stuff".


Linux will typically just have any driver you need (except for proprietary video, but that is just an extra click as you just need to enable the driver it finds for your hardware). It just works- all I've done is plug things in and that is it. 

Linux has its own versions of software that do the same things as Windows programs, it's just free instead. Only time you'll really have an issue is very specific software like say Hallmark Studio card maker- well, that is not going to work on Linux.

I'm guessing these "people" telling you things don't currently use Linux, nor have they probably even tried a distro. You really just have to try it for yourself and see if it is right for YOU. That is all that matters.


To start, you can just run the Ubuntu Live CD (without altering your Windows machine), or set-up a dual boot with Ubuntu and Windows, or install Ubuntu on a different HDD (so you can pop-in/boot from your Windows drive if you want).
