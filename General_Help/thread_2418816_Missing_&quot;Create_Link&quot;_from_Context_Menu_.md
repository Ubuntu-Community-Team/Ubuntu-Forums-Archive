### Post by again? on 2019-05-11
There is an option in Nautilus > preferences > behaviour
The only option I see in pcmanfm is to highlight the file/directory and select from the main menu, **Edit > Create link**
Alternative is open a new pane/window then drag and drop while holding down ctrl+shift

If you're dissatisfied with current file manager give nemo a try.
It's a fork of Nautilus before gnome started "simplifying" it.
