### Post by jwbrase on 2009-03-06
It's pretty much a chance you take, just like when you're browsing the Web. You just have to use good judgement. Sites like Google and repositories like the main Ubuntu ones aren't likely to install horrid malware to your computer, but nothing is ever entirely secure. You can never be any more certain that a repository is safe than you can be that you won't die tomorrow. But you can be fairly *confident* that a repository is safe, just as you can be fairly confident that you won't die tomorrow.
