### Post by Dennis N on 2018-10-10
Icon size in dock are set from **Settings > Dock**.
Icon size on Desktop are set in File Manager: Change to icon view, reduce zoom factor* to size you prefer (this also changes icon size of icon view mode in the file manager).

*change zoom factor with the options button on header that has three dots.
