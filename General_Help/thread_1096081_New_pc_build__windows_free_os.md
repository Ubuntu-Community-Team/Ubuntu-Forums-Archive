### Post by emarkay on 2009-03-15
I have been using PC's since the Apple II, and DOS, and now have WinXP and Ubuntu 8.1 - and haven't booted into Windows in months.

The first thing to do is evaluate the specific software you want to install that's Windows based - by name and version, and see if there's a Linux equivalent (and if more than one, you can try them all - for free!), and if not, see if it has been successfully tested in WINE, the Windows Emulator.  

Feel free to post the software by name here and see what others have discovered.
