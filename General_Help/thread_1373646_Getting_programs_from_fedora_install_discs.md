### Post by bodhi.zazen on 2010-01-06
In general, no you can not do this. Fedora packages are not necessarily compatible with Ubuntu.

To be more specific, Linux systems use a number of package managers. Ubutnu is based on Debian and uses .deb , Fedora uses .rpm

You *may* be able to install some .rpm using alien, but this is not advisable, less so if you are new to Linux.

See : [InstallingSoftware - Community Ubuntu Documentation]("https://help.ubuntu.com/community/InstallingSoftware")
