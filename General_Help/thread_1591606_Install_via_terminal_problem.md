### Post by lykeion on 2010-10-09
If it's this gnomenu you mean: [https://launchpad.net/gnomenu/](https://launchpad.net/gnomenu/)
It got a PPA repository: [https://launchpad.net/~gnomenu-team/+archive/ppa](https://launchpad.net/~gnomenu-team/+archive/ppa)
Use it like this:
```
sudo add-apt-repository ppa:gnomenu-team/ppa
sudo apt-get update
sudo apt-get install gnomenu
```
