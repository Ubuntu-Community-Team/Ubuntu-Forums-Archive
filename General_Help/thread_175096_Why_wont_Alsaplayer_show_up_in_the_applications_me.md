### Post by Sutekh on 2006-05-12
You may need to add your own menu entry for the ALSAPlayer. 

If you go the Menu Editor (in the Applications -> System Tools Menu) you should be able to add a new entry for the ALSAPlayer.  The command for running the ALSASPlayer is
```
alsaplayer
```
you can use this the command part of the dialog box, when you add a new entry.
