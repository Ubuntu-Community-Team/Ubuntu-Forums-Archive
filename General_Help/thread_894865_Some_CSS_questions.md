### Post by gjoellee on 2008-08-19
I am using as much PHP as possible (check out [www.kshoster.net]("http://www.kshoster.net") ...it is under construction), try to avoid javascript, it uses much CPU and many people don't have it enabled. If you abselutely need to use a javascript please make sure you have a good noscript. Make sure they get the same info.

 ```

(javascript)
<noscript>Info for the people who don't have javascript enabled</noscript>
```

NOTE: (javascript) is supposed to mark a where the javascript so you know where to put the noscript. Do not include the "(javascript)"
