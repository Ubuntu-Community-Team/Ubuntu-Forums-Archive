### Post by sayakb on 2008-07-14
Install CCSM 
```
sudo apt-get install compizconfig-settings-manager
```
Goto System->Preferences->Advanced Desktop Effects Settings
There, **enable** the Desktop Cube, Rotate Cube and Cube Caps plugins.
Also, click on **Animations** plugin and you can set different effects (Burn, Airplane etc) for window actions there.
