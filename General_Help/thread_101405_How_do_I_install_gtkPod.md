### Post by angkor on 2005-12-09
sudo apt-get install gtkpod in a terminal 

should do the trick.

Or search for it in Synaptic (Package Manager) and mark it for installation -> Apply

Edit: Make sure you have the Universe repositories enabled. If you don't know how check:

[https://wiki.ubuntu.com/AddingRepositoriesHowto?highlight=%28repositories%29%7C%28add%29](https://wiki.ubuntu.com/AddingRepositoriesHowto?highlight=%28repositories%29%7C%28add%29)
