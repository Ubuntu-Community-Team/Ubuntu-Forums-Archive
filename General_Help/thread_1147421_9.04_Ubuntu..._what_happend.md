### Post by delcypher on 2009-05-03
You may find your autostart applications in ~/.config/autostart . Try removing them via the command line e.g.```
ls ~/.config/autostart
```

then choose what to remove e.g.```
rm ~/.config/autostart/fusion-icon.desktop 
```

The probable reason compiz isn't working for you is because you don't have graphics drivers installed.

run ```
lspci | grep -i vga
``` and post the output and someone might be able suggest how to install the drivers from command line.

If Ubuntu 8.10 worked fine for you, then you might think about moving back to 8.10, 9.04 doesn't offer that much.
