Sounds as if you burned the iso file as a data disk which is just a copy of the iso, not a DVD.

You must burn a **disk image** when you try to turn the iso into a DVD.  This is the bottom of the 5 box options on brasero, and would have burned the disk with the normal VIDEO_RM and VIDEO_TS folders with all the necessary files in them.

If you still have the iso file that devede made, try burning it again as an image this time.
