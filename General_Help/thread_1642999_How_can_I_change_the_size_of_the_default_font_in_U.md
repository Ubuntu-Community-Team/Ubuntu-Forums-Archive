Hi!  Welcome to Ubuntu Forums!

Open the CompizConfig Settings Manager (under the System> Preferences menu) and look at the Accessibility tools to find ways to make things more easily viewable for you. For instance I use the "Enhanced Zoom" quite often.

You can also change font sizes for Firefox under the Firefox menu:
Edit> Preferences> Content> Fonts & Colors.  But changing the font size preferences doesn't always help. There are always going to be some websites where the font either displays too tiny, or too huge. 

So a quick way to adjust the view in Firefox (without fussing with settings each time) is to use the Ctrl key while tapping the + or - keys. 

Hope this helps,
Jay Car
