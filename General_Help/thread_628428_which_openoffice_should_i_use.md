### Post by Juffo-Wup on 2007-12-01
If you just download the package from OO.org's download website, you will have to download it yourself every time you want an update (like a security patch), unless you could find some special repository that they might have and add it to your sources.list.

Also, I don't know if that package will be compatible with all the other OpenOffice related packages in the Ubuntu repos (and that are likely installed in your system), such as the gnome integration one.

So I would recommend you to keep the default one, it has Writer, Calc, Impress, Draw, Math and Base.
