### Post by westie457 on 2012-06-28
Hello talibali welcome to UF.

Sorry for your loss of files, the chances are they have all been lost forever.

One small glimmer of hope for getting them back is photorec. See here for details [http://www.cgsecurity.org/wiki/PhotoRec](http://www.cgsecurity.org/wiki/PhotoRec)

Stop using that hard drive and run Ubuntu from the LiveCD/USB to minimise losing more of your files.

Photorec should be run from a Live Desktop and you will need an external hard drive to write any recovered files to.

Need further help just ask.
