### Post by danastasio on 2010-01-25
no, the root password is actually locked by default, its a security measure taken by ubuntu. Some other distro's leave the password unlocked.

to change it just type

```
sudo passwd root
```

and enter your new passwords. BE  ***_-CAREFUL-_*** working in the root environment is***_ -EXTREMLEY- _***dangerous. I have my root account totally locked.
