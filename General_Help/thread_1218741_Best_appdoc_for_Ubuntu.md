### Post by fwaokda on 2009-07-21
I just went through this same problem... I ended up going with "dockbarx", you have to add a source for it to your software sources though.

Try this guide: [http://www.ubuntu-inside.me/2009/05/howto-get-windows-7-like-taskbar.html](http://www.ubuntu-inside.me/2009/05/howto-get-windows-7-like-taskbar.html)

It acts like the new Windows 7 'superbar'. It's by far the best that I've used and I've tried probably all of them.  It also works great on my netbook to eliminate the need for a top AND bottom bar.  Now I have just a bottom (or top if I wanted) bar and gives me more screen space! :)

Good Luck!

PS. Search youtube for a video about it ( dockbarx ) and you can see how it works before trying it out.
