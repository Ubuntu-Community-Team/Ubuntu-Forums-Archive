### Post by GenePayne on 2011-08-08
I had a problem in Eclipse (v3.5 Galileo) with the scrollbars in eclipse not working properly when running in Ubuntu 11.04 with the new 'pop-up' scrollbars (the thin bar is visible, but the scrollbar won't pop-up).
The eclipse version 3.5 (Galileo) was what installed from the Ubuntu Software Center.

Someone on the launchpad.net bug report (Bug #769277) recommended installing the next version (3.6 Helios) directly from the eclipse website instead of installing it from the Ubuntu Software Center.

This worked for me:P--this newer version does not have the scrollbar problem. So, I though I would post the solution here to ubuntu forums.
