### Post by Ibidem on 2010-12-17
Sounds like some of that needs Wine-see [https://help.ubuntu.com/community/Wine](https://help.ubuntu.com/community/Wine)
(unless you use Virtualbox to run Windows, or use alternatives)
MS Office: Office 2000 works the best under Wine, but 2007 may be usable.  Office 2010 will not work, at the present.
2007 was the first with the 'ribbon', which is one of my own reasons for not using Windows/MS-Office.  Softmaker Office has the old style UI (MS Office 2003/OpenOffice style), but works better with some MS document formats; commercial, ~$70
Adobe Photoshop: You might get it working if you're very lucky.  Standard advice is to use the GIMP (completely different), or to try the Pixel image editor [http://www.kanzelsberger.com/pixel/?page_id=12 ]("http://www.kanzelsberger.com/pixel/?page_id=12")(which looks similar to Photoshop; SHAREWARE at $38 )

Ipod stuff: I wouldn't be able to recommend anything myself; there are plenty of tools with varying functionality; 
iTunes won't work in Wine, IIRC.

Hope this helps,
Ibidem
