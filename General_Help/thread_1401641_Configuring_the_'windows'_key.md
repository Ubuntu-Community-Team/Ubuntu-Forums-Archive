### Post by oedipuss on 2010-02-08
You could turn it to a compiz key, or window management key. 
Have window movement set to it, from System/Preferences/Windows, and use CCSM (compiz config manager) to assign it to various plugins and actions. 

For instance, I use WIN+left mouse button to move a window, clicking from anywhere in that window (via compiz move plugin or preferences/windows), WIN + right MB to scale (via compiz scale window plugin), WIN + scroll wheel for full screen zooming (compiz ezoom ) , WIN + arrow keys for switching workspaces, and activating expo, WIN + tab for fancier window switching, and generally whatever compiz plugin needs a keyboard shortcut I usually map to WIN + something. It leaves ctrl and alt free for other tasks (in gimp for instance they're important) and is consistent.
