---

### Post by PeEllAvaj on 2010-01-19
There are probably a lot of reasons I'm missing, but from my understanding Wubi (and the linux install with it) can be uninstalled in the normal Windows way. If you don't have unallocated space on your hard drive, it can be less scary to use Wubi and install within your NTFS partition, than to try to resize your NTFS partition in order to install and dual-boot linux.
