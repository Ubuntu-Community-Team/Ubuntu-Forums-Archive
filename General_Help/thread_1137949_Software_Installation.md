### Post by lisati on 2009-04-26
1) A good place to start adding software is on the Applications->Add/Remove Software menu. If you've found a really neat program on the internet, keep in mind that Ubuntu doesn't normally run Windows programs (.EXE files) without help - you can use some (but not all) by using [Wine]("https://help.ubuntu.com/community/Wine"). Your best bet is to look for a Linux-friendly equivalent instead of Windows programs if you can.

2) The password for sudo is the same as your main Ubuntu account. Don't worry if it doesn't show when you type it - this is a normal part of Ubuntu security.

EDIT: have a look [here]("https://help.ubuntu.com/9.04/switching/applications-equivalents.html#id2947131") for some information on software for Ubuntu
