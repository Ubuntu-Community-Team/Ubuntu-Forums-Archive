### Post by r_baker on 2008-08-17
I am new to Ubuntu as well, so I can't completely answer your first question.

- Xubuntu - my understanding is that the core systems are Linux, and the other variants are more suitable for a given purpose.  For example, Edubuntu is Ubuntu Workstation with educational add-ons.

- I have a default installation of Ubuntu 9 Workstation, and you can run rdesktop from a command prompt.  You can rdp to Windows Remote Desktop systems.

- Typically, I backup only what files I create that I deem important, and these files are typically copied to another system.  I typically don't bother making a copy of an entire system, but I have successfully used tar and dd in the past.

Hope this helps.
