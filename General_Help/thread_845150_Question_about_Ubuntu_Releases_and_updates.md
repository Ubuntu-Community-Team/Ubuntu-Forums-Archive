### Post by nick_h on 2008-06-30
You have the choice to either upgrade or do a fresh install.

You can configure the upgrade options that the system shows you in System->Administration->Software Sources under the Updates tab - Release Upgrade.

---

### Post by ibutho on 2008-06-30
You can update using the iso or through the package manager when it informs you of a new version being available. Another option is to changes your apt sources when there is a new release, to point to the sources of the new releases and do a dist-upgrade.
