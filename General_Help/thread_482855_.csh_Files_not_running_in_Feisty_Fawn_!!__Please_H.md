### Post by SuperMike on 2007-06-27
In Feisty, I read somewhere that they removed making 'sh' work by default with any type of script and, instead, you must either use /bin/csh or put this in the first line of your scripts:

#!/bin/csh

Also, have you turned on the Universe and Multiverse option in your /etc/apt/sources.list (temporarily), done an apt-get update, and then done apt-get install csh, and then turned off the Universe and Multiverse options in /etc/apt/sources.list, and done apt-get update again?

### Post by stylishpants on 2007-06-28
You are barking up the wrong tree.  The install script itself is fine.  While I find it weird that your first post claims that the file extension ends with .csh, while your most recent post shows the shebang line specifying /bin/sh, none of your posted error messages are shell-related.

Your issue is that one of the executable files that is run by the setup script relies on a shared library that is not present on your system, namely libstdc++-libc6.1-1.so.2.

According to the Ubuntu package list, there are no packages in the main Ubuntu repositories that provide that particular version of libstdc++-libc6:
   [http://packages.ubuntu.com/cgi-bin/search_contents.pl?word=libstdc%2B%2B-libc6&searchmode=searchword&case=insensitive&version=feisty&arch=i386](http://packages.ubuntu.com/cgi-bin/search_contents.pl?word=libstdc%2B%2B-libc6&searchmode=searchword&case=insensitive&version=feisty&arch=i386)

In fact, it appears that that version of libstdc++-libc6 is so old, that even the first ever Ubuntu release, Warty Warthog, had a newer version.
[http://packages.ubuntu.com/cgi-bin/search_contents.pl?word=libstdc%2B%2B-libc6&searchmode=searchword&case=insensitive&version=warty&arch=i386](http://packages.ubuntu.com/cgi-bin/search_contents.pl?word=libstdc%2B%2B-libc6&searchmode=searchword&case=insensitive&version=warty&arch=i386)

Here are some threads on the OpenCascade forums about this problem:
[http://www.opencascade.org/org/forum/thread_7949/](http://www.opencascade.org/org/forum/thread_7949/)
[http://www.opencascade.org/org/forum/thread_11215/](http://www.opencascade.org/org/forum/thread_11215/)

I didn't really read them, but I did notice that the second one has a note at the end saying that it works for Ubuntu Feisty.

Good luck.
