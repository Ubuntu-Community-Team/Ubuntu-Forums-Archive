### Post by Kixtosh on 2010-09-20
Experienced users tend to prefer Terminal commands to the Software Center, but I'm still using baby steps, so try this:

1) Open the **Applications** menu from your top panel (it's on the top left of your screen, unless you already modified something).

2) Open the **Ubuntu Software Center** from the drop down menu.

3) Type "Flash" in the search box, on the top right of the window that opens up.

4) Choose "Adobe Flash plugin" from the menu that appears, and install it (make sure you have clicked the button that actually says "install").

5) Check that it's now in your "Installed Software" list, not in your "Get Software" list (choose either category on the left hand vertical pane of the Ubuntu Software Center window).

6) Restart Firefox/Chromium and try it out.

7) Report back and mark the thread as Solved (from the Thread Tools menu, right above the thread) if it does work.

Now, pat yourself on the back and go have a nice glass of wine (if you like that)! If it _doesn't_ work, cry for about five minutes, don't mark the thread as solved, and post an update when you have regained control of your emotions.
