### Post by oldfred on 2011-09-29
aura7 link is older but essentially correct.

You just also install python3, current version in synaptic.

Whatever you do, do not uninstall the current version as major parts of Ubuntu rely on that version and will not work if only version 3 is installed.

I am also trying to learn python. and many of my small programs work the same whether my code has the python3 or python except for print or print() statements :
#!/usr/bin/python3

Of course more complicated programs will have other issues with the changes.
