### Post by HDave on 2010-03-23
I've struggled with this for ages.  They have an off-line cache mode which lets you see the calendar, but you cannot edit it off-line and sync-up with you are next on-line.

AFAIK -- It simply doesn't work that way. Period.

On the other hand, you CAN create a calendar based on a file and use various synchronization tools to update it.   For example I use Funambol.
