### Post by Therion on 2009-03-20
Download source and install instructions here:  [http://www.nvidia.com/object/linux_display_ia32_180.29.html](http://www.nvidia.com/object/linux_display_ia32_180.29.html)

In short, after you have downloaded the file, save it or move it to your desktop.  
Right click on the file and copy the name of file to your clipboard (we're going to Cut and Paste in a second).

Open a Terminal:
```
cd ~/Desktop
```

Then type in: **sudo sh**, do a right-click and select Paste (this will paste in the proper filename for you).  
Then press enter and an installer will kick off and walk you through the install process.

### Post by Therion on 2009-03-21
> **piratebill said:**
> just install envy.  That should install the latest driver automatically.
I have no issues with Envy, personally, as long as a new user understands that by using Envy they are, for all intents and purposes, installing the driver from source.  This being the case, when a new kernel header comes across in an update, the video driver will bork on them and require a re-install.  I'm okay with that myself, but it's something that could freak someone who's not aware that that **will** happen, since it's not a matter of *if* but of* when*.
