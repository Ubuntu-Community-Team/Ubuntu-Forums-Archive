### Post by dearingj on 2010-05-02
I've seen this before. The problem is that for some reason, at some time probably very recently, the folder your drive was mounted at (/media/500GB) wasn't deleted. To fix it, just delete that folder using the following terminal command, and then plug in your drive:

sudo rmdir /media/500GB

edit: Just to be clear, rmdir won't remove the folder if it's not empty, so don't be afraid of losing any data that might have been accidentally placed there.
