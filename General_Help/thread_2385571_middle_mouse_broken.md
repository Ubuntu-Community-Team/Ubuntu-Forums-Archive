### Post by #&amp;thj^% on 2018-02-22
For Blender go to User Preferences (When Blender is opened), Then at the top tabs look for "Input" , now make sure the box next to Emulate 3 Button Mouse is ticked.
Note some mouse manufacturer's like Logitech need a bit more tweaking.
Caveat:: There is currently a limitation in Blender in which the 'Rotate around Selection' does not function, when you are using free flight. I do not know why this is the case, I have been after the Blender devs for a long time about this. I have found though holding the keyboard key <ctrl> will allow for rotation. (This is when using Free Flight only)
