### Post by 505 on 2008-05-15
Usually you don't download programs and install them yourself. Ubuntu has a very handy tool for installing programs. Click on Application->Add/Remove and search for the software you want to install. Simply click the button and the software is installed. That's it...

Note that most software you download from the internet is Windows software. You cannot run this software on Ubuntu, you need software that is designed for Ubuntu.
