### Post by ajgreeny on 2012-02-19
Jaunty has not been supported for a long time now, so the repositories that your system points to do not exist any more.

You can still get the packages but will need to use the end-of-life repos, and bear in mind that the packages in that are old and could give you some security risks.

See [https://help.ubuntu.com/community/EOLUpgrades](https://help.ubuntu.com/community/EOLUpgrades) and 
[https://help.ubuntu.com/community/EOLUpgrades/Jaunty](https://help.ubuntu.com/community/EOLUpgrades/Jaunty)
