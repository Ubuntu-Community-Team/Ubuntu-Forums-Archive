### Post by Vaphell on 2011-03-28
nasty stuff generally doesn't survive disk formats because all data is destroyed and later disk is overwritten during installation (chances that you experienced one that can survive is 0.0000001% and would be most likely performed by an intel agency or something, but then you'd have greater problems than some infected computer ;)), not to mention it wouldn't be terribly useful in a different kind of operating system.
I don't think an average piece of malware tries too hard, after all there are millions of unpatched windows machines to own - simple repartitioning/reformatting will do.
