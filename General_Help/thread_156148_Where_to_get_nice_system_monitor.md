### Post by taurus on 2006-04-06
They are basically identical file except one uses gzip while the other uses bzip (smaller in size).  For gkrellm-2.2.9.tar.bz2, you can unpack it as
```

tar xvjf gkrellm-2.2.9.tar.bz2

```
and for gkrellm-2.2.9.tar.gz, you do
```

tar xvzf gkrellm-2.2.9.tar.gz

```
