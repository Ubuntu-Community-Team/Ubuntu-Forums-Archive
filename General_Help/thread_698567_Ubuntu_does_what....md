### Post by crisnoh on 2008-02-16
Depends on what software you're looking to install.  The most popular software choices are often available in the repositories, in which case you can simply go into your applications menu and select "Add/Remove Programs."  This will bring up a fairly simple to use program which list all the software in the repositories to which you are subscribed.  Check the one you want to add, uncheck the ones you want to get rid of, then hit Apply.  Synaptic goes to the repos, finds the software and installs it for you.  Magic.

Software not available in the repos is, I will admit, trickier for the newly initiated.  But there is usually a readme in the extracted file which can tell you what to do.  Otherwise, searching the forums for installation instructions for that specific program or asking will usually yield promising results.
