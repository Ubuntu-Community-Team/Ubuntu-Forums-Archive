### Post by t0p on 2009-07-26
> **mooksh said:**
> Thanks for your suggestion dude. I have just reinstalled Ubuntu 9.04 on my system; file system is default and using entire system for Ubuntu 9.04.

Maybe you misunderstood what jongers said: **STOP USING THE DRIVE!**

Every time you do something on that computer, data is written to the disk.  Which means new data may overwrite old data.  If you are serious about salvaging that old data, **you must stop using that disk immediately**.  Seriously, stop now. Now!
