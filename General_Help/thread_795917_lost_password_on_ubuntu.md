### Post by EXCiD3 on 2008-05-15
Boot into recovery mode and run
```

adduser <username> admin
passwd <password>

```

and then you will have a new user account to login with.

Here is some more info on changing a forgotten password: [http://ubuntuforums.org/showthread.php?t=133102&highlight=howto+password](http://ubuntuforums.org/showthread.php?t=133102&highlight=howto+password)
