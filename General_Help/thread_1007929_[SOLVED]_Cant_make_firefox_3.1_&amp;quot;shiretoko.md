### Post by Tim Sharitt on 2008-12-13
You should be able to select 'custom' in prefered applications and then enter the full path to the firefox 3.1 executable.
For example:
```
/opt/firefox-3.1b2/firefox
```

### Post by mjheagle8 on 2008-12-13
I solved the problem.  I use custom command to launch the links by default in a new tab. If anyone else needs it, the command is
```
firefox-3.1 -new-tab %s
```
