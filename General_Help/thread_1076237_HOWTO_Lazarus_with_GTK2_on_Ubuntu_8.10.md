Hi all!

This steps will help you install Lazarus IDE with nice GTK2 interface:

1 sudo aptitude install alien
2 Download fresh nightly rpm-builds from [http://www.hu.freepascal.org/lazarus/]("http://www.hu.freepascal.org/lazarus/") of lazarus, fpc and fpc-src
3 Convert rmps (alien --scripts) of lazarus, fpc and fpc-src to debs
4 and finally install deb packages with dpkg

That is all! All works fine! :popcorn:
