### Post by theaaronc on 2010-02-26
If you right click on the launcher icon and select properties you should see the command that is used to open Chrome. Append --incognito to the end of this command and it should force Chrome to open in that mode.

For example:

Command:  /opt/google/chrome/google-chrome %U --incognito
