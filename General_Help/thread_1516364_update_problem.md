### Post by marin123 on 2010-06-24
here, try this:

[http://swiss.ubuntuforums.org/showthread.php?t=1475182](http://swiss.ubuntuforums.org/showthread.php?t=1475182)

i also had update problem and googled it... something gets messed up in the software manager...

maybe you could partially update through terminal:

```
sudo apt-get update
sudo apt-get upgrade
```
when you type upgrade, you should see available packages for upgrade... and then type y and hit enter and it should start updating... good luck! :)

EDIT:  go to Administration -> Software Sources -> Updates tab and untick Pre-released updates, you dont need that... proposed updates are work in progress and can do more damage then good...
