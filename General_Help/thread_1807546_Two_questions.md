### Post by 2F4U on 2011-07-19
As far as I know, the G3 iMacs have PowerPC processors, but Ubuntu is only available for Intel or AMD CPU's. There are, however, a few Linux distributions available which support PowerPC processors.

The second problem is likely a graphics driver problem. If, for example, your laptop has a nVidia card, Unity would require the proprietary nVidia graphics drivers to be able to show desktop effects on Unity. These are easily installed with a tool called "Additional Drivers", which is part of the default installation.
