### Post by rbc on 2011-10-26
On my installations, the path gets highlighted when I hover the mouse pointer over the Bookmark. Or open Nautilus file browser and click on Bookmarks, then Edit Bookmarks. Highlight the Bookmark you want, and the path should show up in the Location field. Is this what you’re after?
