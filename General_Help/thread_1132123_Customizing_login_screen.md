### Post by CatKiller on 2009-04-21
I'm pretty sure that you can go to System -> Administration -> Login Window, click on the Local tab and, if you've got your style set to Themed, click on Add... to add the theme. Just select the .tar.gz file.

For that matter, drag-and-drop might work, too. I know it works for Desktop themes.
