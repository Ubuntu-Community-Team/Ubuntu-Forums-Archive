### Post by wrycatcher on 2007-03-28
Welcome back!  I've had the same inkling a few times, but have managed to persevere.  I have learned much, probably more than I really wanted to, but it has been mostly a good experience.  However, I think I reach that "breaking point" in two ways:

1.  Critical or serious problem exists for a **short **period of time.  Problems such as overheating issues, device driver flakiness or unavailability, network connectivity problems...these result in an unstable or severely impaired flatform.
2.  Minor ongoing annoyances over a **long **period of time.  Example of this are subtle incompatibilities in document formatting/conversion, lack of certain conveniences you had with other OS's, web sites and file formats that only work in IE, etc.

If a user gets past the first class of problems (with or without some pain), they will certainly hit the second class of issues.  Unfortunately, reality clashes with the dream of a less Windows-centric one.  Many tools and applications are only available if you use Windows, IE, Windows Media Player, etc.  That might well change over time, but it doesn't help you *now*, does it?  These day-to-day annoyances (even *sacrifices*) take a cumulative toll on the user's patience and satisfaction, sometimes leading the user to eventually abandon Linux.  Only, like you have experienced, you escape one set of problems only to encounter a whole different set of problems...just depends on which set of problems you can endure, I suppose...
