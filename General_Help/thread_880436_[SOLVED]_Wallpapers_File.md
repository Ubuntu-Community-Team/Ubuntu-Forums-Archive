Does anyone know where the file or folder that tells the gnome-appearance-properties program which images to display as choices for wallpaper.

---

### Post by pauper on 2008-08-05
```
gedit $HOME/.gnome2/backgrounds.xml
gedit $HOME/.gconf/desktop/gnome/background/%gconf.xml
```
