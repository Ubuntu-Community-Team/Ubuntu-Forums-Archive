Hi, I own a Netbook that has a Realtek wireless card to be exact it's RTL8187b. I installed the first UNR from canonical but alas my WiFi died. The signal is getting detected but I was unable to get a working link to my router. I tried everything I know but nothing. Then along came Karmic so I thought this must have corrected the problem and did a clean install on my NB nothing it was as same as earlier [-(.

While I was going through internet searching for a solution I stumbled upon [ndiswrapper]("http://sourceforge.net/apps/mediawiki/ndiswrapper/index.php?title=Main_Page") and this site [http://ubuntuforums.org/newthread.php?do=newthread&f=331]("http://ubuntuforums.org/newthread.php?do=newthread&f=331"). After following the instructions to the letter I got my WiFi working with out any glitches. Yey thanks to ndsiwrapper windows XP drivers are working the problem was with the default driver. This might work for you guys to if you've a problem like mine. Hope this helps you.
