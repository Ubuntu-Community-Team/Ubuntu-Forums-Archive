### Post by critin on 2011-12-02
Do you mean 'update' files instead of 'upgrade'?  Did they come from the update manager or something you chose to install yourself?

If from the update manager they normally correct themselves in my experience.  I don't worry about them.  Now if it's something I chose, it's different--errors usually mean I did something wrong.

You can always run the update manager again to see if it fixes the error if you wish.
To uninstall, go to synaptic and enter the items into search, delete them from there if that's what you decide to do.
