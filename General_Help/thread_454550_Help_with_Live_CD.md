### Post by merlinus on 2007-05-25
The first option is the correct one.  Perhaps the download got corrupted, or did not burn properly.

You can try burning at a slow speed, say 4x.  Also, there is a program that will check the md5 sum for your download to make sure it is ok.  

For windows you can get it from [http://www.md5summer.org/download.html](http://www.md5summer.org/download.html)

Good luck!
