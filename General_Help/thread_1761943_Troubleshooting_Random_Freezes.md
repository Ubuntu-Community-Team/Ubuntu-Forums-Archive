### Post by Wintaru on 2011-05-23
Just wanted to post and say that since I posted this I've reinstalled Natty from disk and it works perfectly now.  I think something went wrong when I did the upgrade versus the reinstall, so if anyone else has this issue I would try a fresh reinstall first.

Also something of note, on the fresh reinstall, Natty didn't detect my monitor resolution correctly.  It's an older Sceptre monitor that should be capable of displaying 1920x1080 but for some reason can only do 1680x1050 and I think it's EID is jacked.  Anyway, I had to set the resolution manually using xrandr and this guide here and all is well again:

[https://wiki.ubuntu.com/X/Config/Resolution](https://wiki.ubuntu.com/X/Config/Resolution)
