I tried using poster ([http://linux.die.net/man/1/poster](http://linux.die.net/man/1/poster)), but it supports only ps files. And conversion of pdf to ps is lossy.

### Post by ramjet_1953 on 2009-03-29
You could use GIMP to do that.

Load the image into GIMP and Navigate to [COLOR="Blue"]Image>Scale Image[/COLOR]

You then have the options to choose different measuring methods and even the ability to alter the aspect ratio, by clicking on the chain icon.

### Post by Jose Catre-Vandis on 2009-03-29
Or you might like to try out the Rasterbator

[http://homokaasu.org/rasterbator/](http://homokaasu.org/rasterbator/)

Which will rasterise your image to any number of sheets of paper you like.

You can rasterbate on line or download a windows executable and rasterbate at home by yourself. (OK, enough of the innuendo :)) it's a really clever tool. It will apparently run in Linux using Mono as it needs .net frameworks.
