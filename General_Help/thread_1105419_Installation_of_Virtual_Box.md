### Post by snova on 2009-03-24
> **someguyatacomputer said:**
> I have installed Sun's Virtual Box using the gedi or gebi installation application.

Now I don&#8217;t know where the files have been placed or how to start the application. Can any one help me please?

Well, you seem to have it installed. If you can't find it on the menu, try running it from a terminal:

```
VirtualBox
```

> Also is there away to add the application the applications menu (i.e. the equivalent to the Windows start menu I guess.).

Yes, System -> Preferences -> Main Menu. Select the menu you would like to put it under, press the Add Item button, and set the Command field to "VirtualBox" (minus quotes, of course). That should be it.
