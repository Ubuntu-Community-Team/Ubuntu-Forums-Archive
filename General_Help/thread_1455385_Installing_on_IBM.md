### Post by 3rdalbum on 2010-04-15
The Ubuntu CD can be used not only to install Ubuntu, but also to run Ubuntu from the CD for testing purposes. Why not try that? If you have an external CD drive, or if you use the Startup Disk Creator on a flash drive and boot from the flash drive, you'll even be able to test your second battery.

I'm not sure if you'd be able to install Office 2007 into Wine on the live CD - everything you "install" while on the live CD just goes into RAM - but it wouldn't hurt to try. You would of course need to install Wine first, from Synaptic Package Manager.
