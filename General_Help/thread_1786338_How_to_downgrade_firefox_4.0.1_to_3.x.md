### Post by ajgreeny on 2011-06-19
I think you should investigate your flash problem rather than move back to FF 3.6.x.

How have you installed the flash plugin, and what other plugins do you happen to have installed that might conflict with it?

I suggest and highly recommend you get the **Flashaid** add-on for Firefox written by lovinglinux.  It will work out what is already installed, remove it if necessary and then install the best flash version and configure it best for your system.

I've been using it for some time now and think it is brilliant;  it makes what can be a confusing activity into a quick and easy one, and if my experience is anything to go by, it will not fail you.
