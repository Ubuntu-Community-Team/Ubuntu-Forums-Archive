### Post by grahammechanical on 2013-04-26
Item 1) See if what you want is already registered and vote for it or register this as a new idea.

[http://brainstorm.ubuntu.com/](http://brainstorm.ubuntu.com/)

Item 2) Is this a function of a particular application? In what way do you have a few pictures open? If you are using an application that does not have the features that you require, then apply to the developers of the application.

In my opinion, there is no need to introduce a click on launcher icon to minimize function as this is already available with the click on minimize button in every application window. It would be unnecessary duplication of effort.
