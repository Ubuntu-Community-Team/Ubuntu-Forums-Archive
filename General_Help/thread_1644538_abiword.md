### Post by Linux_junkie on 2010-12-13
It sounds like the file type for an Abiword document has not yet been assigned to Abiword which is why its asking you what to do with it.  Either that or you've accidently pressed the right mouse button on the icon to bring up the menu.

In the menu select Open with... then from there select the Abiword application that should come up.  There should also be an option to always open the file type with that application, select this option and you should not have this problem again.
