### Post by 3rdalbum on 2009-12-29
Ubuntu and Xubuntu are not as light as Windows XP. Not by far. Also you'd have to check that you have enough RAM to run Xubuntu - I think it really needs 256mb or more.

Over time, Windows gets slower and Ubuntu doesn't; that's where the difference in performance lies.

Windows programs don't run on Ubuntu. Some (a small proportion) can be run using a compatibility layer called Wine, so you'd want to try that. But if you do a lot of gaming you want either Windows or a games console.

As for drivers, you probably won't have any issues there. Ubuntu contains most of the drivers you'll need, especially for an elderly computer such as yours. And it will automatically use whatever drivers are necessary.

The only real gotcha is that graphics card - apparently it's an ATI, from what I've been reading.  ATI does not make drivers for your card anymore, so you might not get any 3D acceleration under Ubuntu; and if you do, it will be reverse-engineered 3D support. Ubuntu includes an open-source driver that does offer 3D support on some ATI GPUs, but possibly not on yours. Try it and see - the driver will be loaded on startup, and if there's 3D support you will see nice visual effects when you open and close a window.
