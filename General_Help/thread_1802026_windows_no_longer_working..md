### Post by Mark Phelps on 2011-07-11
How did you install Ubuntu? Did you make the SERIOUS mistake of allowing the Ubuntu installer to shrink the Win7 OS partition in a side-by-side installation?

That has been known to result in filesystem corruption in the Win7 OS, rendering it unbootable.

Since you probably do NOT have Win7 installation media, and did not go to the effort, ahead of time, to use the Win7 Backup feature to create and burn a Win7 Repair CD, you have no media that you can use to repair the Win7 boot loader.

Your only option at this point is to borrow a Win7 DVD from someone else, boot from it, and run Startup Repair three times.
