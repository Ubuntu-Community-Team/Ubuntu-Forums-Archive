### Post by ajgreeny on 2010-07-29
In the past when using kubuntu-desktop on top of a standard ubuntu install, Ialways used synaptic as my package manager, never the kde equivalent which I did not think was nearly as good.  If you run ```
sudo apt-get install synaptic
``` you can also use synaptic as the package manager.

Anything that runs on the one, will run on the other, no problem.

### Post by finny388 on 2010-07-29
sudo apt-get install synaptic = ubuntu software center = synaptic package manager (which btw, I prefer immensely over kpackage)

I used Kubuntu for about a year before turning to Ubuntu

Kubuntu has plenty of friendly support in forums but is WAY less active than ubuntu.

Kubuntu is highly customizable (lots of themes and LOTS of GUI tweaks can be made)

Regarding GRUB, I believe it is identical. Regarding trackpad, no idea.

I found it was a great interface, more similar to Windows than ubuntu (not a plus for me but a facts a fact)

I just found it too slow for my ageing netbook while ubuntu has worked very nicely.

also, kubuntu has less developers, making for slower updates, greater chance of unpolished implementation. Not supported by Canonical either, it is more of a step-sister than a sister of ubuntu.
