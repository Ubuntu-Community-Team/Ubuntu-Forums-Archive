### Post by az on 2005-11-27
Ubuntu does not use the root account.  You use sudo to escalate your priviledges.  The advatages include only the need to remember one password (Ubuntu is primarily a desktop system) and everything you do as sudo is logged.

Look here:

[https://wiki.ubuntu.com/RootSudo](https://wiki.ubuntu.com/RootSudo)
