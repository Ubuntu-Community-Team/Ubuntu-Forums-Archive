### Post by kidders on 2007-05-30
Hi there,

I suspect setting something like that up would probably be waaaay beyond the ability of someone whom you want to show how to browse the web. :-( Setting aside software installation & configuration, running a server on a public network requires a reasonable understanding of things like IP, routing, NAT, UPnP, and so on ... so your friend could identify how his network is managed, and act accordingly.
