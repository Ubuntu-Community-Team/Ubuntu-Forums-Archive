### Post by localzuk on 2005-12-20
You must enable the Universe repositiory (have a look at [http://www.ubuntuguide.org](http://www.ubuntuguide.org) - bear in mind that these are instructions for hoary, so replace 'hoary' with 'breezy' in the relevant places).

Now just use synaptic to install it or use the command line and type:

```

sudo apt-get install smb4k

```
