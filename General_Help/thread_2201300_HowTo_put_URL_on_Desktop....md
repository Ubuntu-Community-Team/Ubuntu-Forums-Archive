### Post by Frogs Hair on 2014-01-23
I have done this simply by drag and drop in 12,04  and on newer versions made a Unity launcher for a web site with alacarte/main menu. With the main menu open select a category and once there add a new item and create a name and add an icon. The command is as follows and you would adding the desired url.I made a Pandora Launcher for an example.  ```
 firefox%u http://www.pandora.com/

``` The new icon can be added to favorites and will open FF to the desired page or open a new tab if FF is already open . There are other methods , but I;m not a fan of desktop icons so search the forum because the question has been asked before.

```
sudo apt-get install alacarte    
```

### Post by bc.haynes on 2014-01-23
[quote=Bucky Ball]You need to 'create a url link'. Maybe use that for a search and add 'ubuntu' to it.[/quote]
I did, and it gave the solution: Go to the URL you want on your desktop. Click on the object to the left of the URL and drag it to your desktop and release it. This will put a link to the URL on the desktop. If you double-click on the link, it will take you to the web site. This is in Ubuntu 12.04. I don't know about other distros. Have fun.
