### Post by the yawner on 2009-01-04
Hi. Xubuntu uses Thunar as its file manager. You can still change file permissions by right-clicking a file. But unlike Nautilus, it only shows the the checkbox "Allow this file to run as a program" if it recognizes the file as an executable. I've tested this myself by simply renaming the extension of a text file from .txt to .sh.
