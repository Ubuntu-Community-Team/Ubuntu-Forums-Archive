### Post by fragos on 2008-04-10
Gnome doesn't to a good job of allowing you to integrate a web mail like Gmail for sending from applications. In firefox you can accomplish this with the Google tool bar but that still leaves the others aps on their own. What worked for me was to configure Evolution to send email through the Gmail server. I didn't configure it to read email so it only sends when called by an application running in the Gnome environment. Unfortunately you won't have the Gmail address book when you send with Evolution so you'll need to populate Evolutions addressbook as well as Gnome. I installed the Contacts package which uses Evolutions data base. Contacts are very strait forward and you need not bother with Evolution for contacts and still have them available for sending mail.
