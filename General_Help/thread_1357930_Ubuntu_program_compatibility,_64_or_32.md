### Post by audiomick on 2009-12-17
Hi.
There are still things that aren't happening for 64 bit systems, but very few and less all the time. All the mainstream stuff like e-mail, open office, browser ( without add-ons. add-ons is one thing where you might find some that are still 32 bit only) works fine.

Programs that are packaged as .deb files will install on any Debian based Linux, which includes all Ubuntu variants.
