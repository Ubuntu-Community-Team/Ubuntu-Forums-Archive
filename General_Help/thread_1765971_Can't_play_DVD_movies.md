### Post by nerdy_kid on 2011-05-23
Ubuntu can't play dvds out of the box for legal reasons.  The following commands will enable playback, but may be illegal in some countries.

```

sudo apt-get install ubuntu-restricted-extras
sudo /usr/share/doc/libdvdread4/install-css.sh

```
