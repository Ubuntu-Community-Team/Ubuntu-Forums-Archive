### Post by dwhitney67 on 2011-05-14
Goto System->Preferences->Startup Applications, then select the Options tab.  Check the box for "Automatically remember..."

---

### Post by dandnsmith on 2011-05-14
Another recent post triggered a memory - so Ive found that setting under System|Preferences|Startup Applications (Options).
