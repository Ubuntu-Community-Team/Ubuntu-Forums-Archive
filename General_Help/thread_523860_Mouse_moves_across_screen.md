This sounds like a hardware problem. Optical mice (I assume your mouse is optical?) have a little infrared scanner that detects motion, and if there is either a problem with the circuitry or a physical source of movement stuck inside the bottom of the mouse, it will start seeing strange stuff.

Do you have another mouse you can plug in to the same port and see if it still does this?
