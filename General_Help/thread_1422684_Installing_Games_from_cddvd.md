### Post by Intrepid Ibex on 2010-03-05
There is that kinda app, it is called wine (Wine Is Not and Emulator).

you can install it with 'apt-get install wine' (as root) or just searching it with some nice graphical tool ubuntu has to offer (the software center or synaptic).

and here is everything you need to know about wine, you are of course free to ask some more but I hope you search from here first: [https://help.ubuntu.com/community/Wine](https://help.ubuntu.com/community/Wine)
