### Post by Pollox on 2011-01-14
I am not familiar with the program you are referring to, but may I suggest you try Synaptic Package Manager if you are looking for something more advanced than Ubuntu software center? It should already be installed, and you can find it under System > Administration.

Ubuntu by default does not have a root password set. The password you enter when you type "sudo" is your user password.
