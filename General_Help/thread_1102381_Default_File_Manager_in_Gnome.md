### Post by mb_webguy on 2009-03-22
[Changing the Default File Manager in Gnome]("http://www.psychocats.net/ubuntu/nonautilusplease")

Which file manager is best is very much a matter of personal preference.  I prefer Nautilus because it integrates naturally with Gnome and is very customizable with extensions like nautilus-actions, nautilus-gksu, and nautilus-open-terminal, and custom scripts.  But other people prefer others for various reasons.  Some like dual-pane file managers like Gnome Commander, while others prefer to use a lightweight file manager like Thunar.  My suggestion is to try several and choose the one that works best for you.
