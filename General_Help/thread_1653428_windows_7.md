### Post by wilee-nilee on 2010-12-26
> **joelkat said:**
> I'm having trouble with it and win7 would help and yes

In order to install W7 on the same HD as the Ubuntu you will need to open a space to create a NTFS partition and put the boot flag on it.

Do you have the W7 disc? 

W7 has to be booted to install in other then a virtual machine.

If you could take a screen shot of the gparted partitioner in Ubuntu that would help.

If gparted is not installed open a terminal and run
sudo apt-get install gparted 
It will be in menu-system-admin-gparted.

If you don't make a pre-formatted partition on the HD W7 will make Ubuntu unallocated or overwrite it.
