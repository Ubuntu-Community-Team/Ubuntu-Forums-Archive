### Post by mucholocodawg on 2010-04-14
Zimbra desktop works great with UNR 9.10 the procedure to install is simple.

1.) Download Zimbra desktop from
     [http://www.zimbra.com/products/desktop_download.html](http://www.zimbra.com/products/desktop_download.html)
    (Make sure to grab the linux client)

2.) Open a terminal window

3.) In the terminal window type cd ~/Downloads

4.) In the terminal window type sh zdesktop<press tab> 

5.) Follow install directions and let it install the desktop icon
    but do not let it launch the desktop client in the last step
    by removing the check mark from the launch box. 

6.) Let the install finish and close it.

7.) Close the terminal window we are done with it

8.) Open the UNR main panel if you are not there navigate to 
    files & folders and open desktop

9.) Open the UNR main panel if you are not there navigate to 
    system and under preferences open main menu

10.) On the window picker (the running application icon are on the 
     top panel right click and unmaximize the desktop file browser
     and the main menu

11.) Choose the main menu window navigate to the internet icon and
     and click on it. Move the window to the side making sure that the
     items area is near the screen edge.

12.) Choose the desktop file browser window and move it around so you see
     both the items area of the main menu window and the desktop folder content.
     Click and drag on the linux.desktop icon and drag it to the the main menu
     items area.

13.) Close the main menu window and the file browser window.

14.) That is is navigate to your internet tab and you will find zimbra desktop



that is it enjoy Zimbra on your UNR.
