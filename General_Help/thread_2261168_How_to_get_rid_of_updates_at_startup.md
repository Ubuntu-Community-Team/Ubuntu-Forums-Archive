### Post by garyed on 2015-01-16
I'm running Ubuntu Studio 14.04 & every time I boot up it checks for updates. It is really getting annoying. Is there any way to turn updates completely off until I manually want to check for them?

---

### Post by Bucky Ball on 2015-01-16
Applications>System>Software Updater>Settings (bottom left)>Updates tab>'Automatically check for updates' set it to 'Never'. 

You can also get there by opening Synaptics>Settings>Repositories.

Hope that helps.

### Post by Impavidus on 2015-01-18
When you set the update manager to never check for updates, it immediately stops checking for available updates. It continues to notify you about available updates it already knows about.

Don't forget manually updating your system every week or so.
