### Post by tgalati4 on 2007-06-22
There are other distributions that are better-suited for pendrive use.  Damn Small Linux is one.  Knoppix is another.

I've never heard of Ubuntu being used from a thumb drive.  I imagine that it is possible, but I haven't seen it.

You can use a Live CD in your iMac without touching the hard drive.  The worst that could happen is that the disk will get stuck inside the slot loader.  Of course that could happen to any disk.  Just download the latest PPC or Intel/Mac version for your hardware and give it a go.

Yes, the Live CD does run slow, but its main purpose to test that all of the hardware works and so you can get a feel of how the operating system looks.  It's not designed for everyday use like DSL or Knoppix.

You might need a 2 or 4 GB memory stick since the base operating system is about 1.5 GB.  It depends on which of the 18,500 packages you decide to install.
