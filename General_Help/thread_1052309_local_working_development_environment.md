### Post by arvevans on 2009-01-27
You didn't mention whether you are running Gnome or KDE, or...?, so answers will have to be pretty generic.

There are a number of small and fast web servers available, as well as apache.  If you have the Synaptics Package Manager you can go there and do a search for "web server" to find what is available.

Depending on how you like to develop your web content, you have a number of options, from simple text editing, to WYSIWYG editors, to full development environments...complete with version control (content management) and code checking.  Most of these tools can be found by doing searches in the Synaptic Package Manager.

Since I do my web development using WYSIWYG tools I find it very comfortable to use development suites like "Amaya" or "Bluefish".  There are many more such tool suites available from the Ubuntu repositories.
