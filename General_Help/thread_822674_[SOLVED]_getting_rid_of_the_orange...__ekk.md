### Post by steveneddy on 2008-06-08
Right click the desktop, menu opens, click 

"Change Desktop Background"

Click at the top of the window to open the Theme tab.

Either select a theme that suits your needs or click the 

"Customize" button to change the options you would prefer to see.

Easy.

### Post by Genius314 on 2008-06-08
Go to System>Preferences>Appearance, and try a theme like Clearlooks or Glossy. Also, if you just want to change the titlebars, and not the buttons, click "Customize," go to the window border tab, and change them there.

### Post by Aa1usa on 2008-06-08
System ->  Preferences -> Appearance

Then select a different theme to change the windows top border color.

If you want you Icons that you had, just click Customize and add them.

Something nice is that by default Ubuntu has some blue themes.

If you want more themes go to to [http://www.gnome-look.org/](http://www.gnome-look.org/) .
