### Post by mcduck on 2012-04-16
Compiz has the "Negative" and "Color Filter" plugins which might do the trick for you. Of course assuming the laptop is otherwise capable of running Compiz/Unity desktop... ;)

If you are able to run normal Unity desktop, just install CompizConfig Settings Manager and check there if either one of those plugins would fix the colors for you.

(Negative does simple negative colors, either for one window at a time or whole desktop. Color filter allows more detailed configuration of colors, but making new definitions isn't the easiest of a task)
