### Post by ajgreeny on 2009-12-31
The make of router does not make much difference as far as linux and ubuntu is concerned, it is the wireless adapter in the computer trying to connect that is most important.  I use a Netgear DG834G router, about 3 or 4 years old and it is fine, but a Netgear WG511v2 with marvell chip is a bit of a pain, only connecting with WEP, not WPA.

Just about all routers can be set up easily from a web browser, often pointed to [http://192.168.0.1](http://192.168.0.1) and entering the name and password for admin of the router which you can find in the manual that comes with the router, so don't worry if the seller says it is not supported under linux, as that is unlikely to matter a jot.
