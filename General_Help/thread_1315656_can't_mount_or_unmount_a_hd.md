### Post by Giblet5 on 2009-11-05
> **bushtangu said:**
> how can i resolve it to not mount  or unmount with command. 
when i'm trying to unmount with right click it show me : 
Error unmounting: umount exited with exit code 1: helper failed with:
umount: only root can unmount /dev/sda1 from /media/Data

Unmount it, then mount it with another option:

```
sudo mount /dev/sda1 /media/Data -t ntfs -o nls=utf8,user
```

Now you can unmount it via ```
umount /dev/sda1
```

(no "sudo")

You can make it much easier by placing an entry for it in /etc/fstab. ```
sudo gedit /etc/fstab
```

Scroll all the way to the bottom. On a new line, add:
```
/dev/sda1  /media/Data  ntfs  nls=utf8,user,exec,noauto  0  0  1
```

Now you should be able to mount/unmount it easily as your regular user, either from the GUI, or via ```
mount /dev/sda1
```
