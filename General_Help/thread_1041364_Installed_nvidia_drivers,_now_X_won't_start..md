### Post by orlox on 2009-01-16
Almost always, you can restore the gui by switching to vesa or nv, instead of the nvidia driver. Edit your /etc/X11/xorg.conf and add this:

```

Section "Device"
Identifier "Configured Video Device"
Driver "vesa"
EndSection

```
If there is a device section in the file that has this same identifier, then change the driver to vesa. If there is a section that contains the name of your video card, then add or change the driver accordingly.

To edit the file, you can use nano (I prefer vim, but its much less intuitive)

```

sudo nano /etx/X11/xorg.conf

```

Edit the file as mentioned above, and save using ctrl+x.

The default xorg.conf in intrepid should read like this:

```

#
# This file was generated by dexconf, the Debian X Configuration tool, using
# values from the debconf database.
#
# Edit this file with caution, and see the xorg.conf manual page.
# (Type "man xorg.conf" at the shell prompt.)
#
# This file is automatically updated on xserver-xorg package upgrades *only*
# if it has not been modified since the last upgrade of the xserver-xorg
# package.
#
# Note that some configuration settings that could be done previously
# in this file, now are automatically configured by the server and settings
# here are ignored.
#
# If you have edited this file but would like it to be automatically updated
# again, run the following command:
#   sudo dpkg-reconfigure -phigh xserver-xorg

Section "Device"
    Identifier    "Configured Video Device"
EndSection

Section "Monitor"
    Identifier    "Configured Monitor"
EndSection

Section "Screen"
    Identifier    "Default Screen"
    Monitor        "Configured Monitor"
    Device        "Configured Video Device"
EndSection

```
So you should only need to add the driver line under the Identifier line of the device section. But maybe youre file is different, so you need to modify it so it ends up with the device section for your video card looking like this:
```

Section "Device"
    Identifier    "Configured Video Device"
    Driver    "vesa"
EndSection

```
Also, the nv driver might be a better alternative (meaning this, a prettier alternative)
```

Section "Device"
    Identifier    "Configured Video Device"
    Driver    "nv"
EndSection

```
You can try what fits you best...

Let us know how it goes, so we can help you get everything working with the nvidia drivers after you restore the GUI
