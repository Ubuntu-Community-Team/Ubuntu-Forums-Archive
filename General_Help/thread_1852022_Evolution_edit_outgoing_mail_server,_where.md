 mymail@mydomain.com:465

---

### Post by SparTacux on 2011-09-29
Select Edit then select preferences from the drop down menu. Then select EDIT from the Evolution Preferences  'window'. Go through tabs at top of screen to find sending details. You'd enter something like secure.mymailserver.com:465 or mail.mymailserver.com:465
