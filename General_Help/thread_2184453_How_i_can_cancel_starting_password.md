### Post by Impavidus on 2013-10-29
It seems that you encrypted your hard drive, so it needs the passphrase before you can boot Ubuntu. It shouldn't interfere with running a Windows install dvd. You may have to wipe the drive clean or create an NTFS partition with boot flag before you can install Windows though. Use an Ubuntu live disk to do so.

On the other hand, the problems you have with Ubuntu may be solvable. It may be as simple as a problem with your graphics driver. Could you post some information about your hardware?
