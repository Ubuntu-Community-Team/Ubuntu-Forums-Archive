### Post by airspirit on 2008-06-06
A nice option is the new mintmenu.  You can install it using the beta repository for the new version of Linux Mint.  Add in "http://packages.linuxmint.com elyssa main upstream import" to your repositories, then go into synaptic and install mintmenu (should be version 4.0).  It will work 100% for you.  You'll have to change the menu icon manually (it's in /usr/share/pixmaps/mintmenu.png or something like that) but otherwise is 100% functional without any other changes in Hardy.

I installed this earlier after a ton of hunting and experimenting and was surprised how mindlessly easy this solution is.  Now that it's installed I'll never go back.  :)

If you right click on your panel, select add to panel, then add the gnome menu (NOT custom menu, that has Apps/Places/System, the regular menu has just an icon) an you have your menu, its already installed. Then just right click remove the custom menu (which comes defaultly added to Ubuntu) and you're good to go. All you programs will stay there, as will any changes you made to the layout of the menu using the custom menu.
