### Post by kurt18947 on 2012-05-20
I have limited experience with this so take this for what it costs you ;).  If 100% fidelity with MSO file formats is critical,  it might be a good idea to run MSO either using WINE, play on linux,  crossover office etc.  or in a virtual machine.  From reports here and elsewhere, Libre Office is pretty good with .docx but not perfect.  If you could get everyone to use the old .doc format  you might have better luck.  The other thing I've heard/read is to open .docx files, save as and work with the file in .odt format then save as .docx to send on.
