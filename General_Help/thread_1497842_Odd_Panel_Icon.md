It should not have done anything like that, it should have merely switched from compiz, if you were running that as your window decorator, to metacity, the default gnome version.  Are you sure you do not have other problems of display, perhaps related to a graphics card driver.

You can check if compiz is running by right clicking on your desktop ->Change Desktop Background and then go to Visual Effects tab. If it shows "Normal" or "Extra" compiz is running, if "None" metacity is already running.

I also seem to remember a launchpad bug about strange icons, so do a search of that and see if you can find anymore info.
