Install gnomebaker from the Universe repository using Synaptic or Add/Remove.  Then you can go to  Applications -> Sound and Video -> Gnome Baker  and then in Gnome Baker, Tools -> Blanck CD-RW.  There are a few other programs you can use if you don't like Gnome Baker: k3b, graveman, and cdrecord come to mind.
