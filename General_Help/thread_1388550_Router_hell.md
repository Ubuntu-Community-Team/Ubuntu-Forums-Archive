### Post by zircon214 on 2010-01-23
Just because a router supports 802.11n does not mean it can run in the 5GHz band. Did the box or manual specifically say 5GHz on it?

For example here is a 802.11n router that can run at 2.4 and 5GHz:[ http://www.linksysbycisco.com/US/en/products/WRT400N]("http://www.linksysbycisco.com/US/en/products/WRT400N")

Here is a 802.11n router that can run only on 2.4GHz_:_[ http://www.linksysbycisco.com/US/en/products/WRT160N]("http://www.linksysbycisco.com/US/en/products/WRT160N")
