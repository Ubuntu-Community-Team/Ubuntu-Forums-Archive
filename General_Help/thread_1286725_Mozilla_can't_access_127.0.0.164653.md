### Post by hal10000 on 2009-10-09
You can only access 127.0.0.1 via a webbrowser if you have running a [COLOR="Red"]webserver[/COLOR] locally on your machine. I don't know if your game comes with a webserver, if not, then you will have to install a webserver like apache (or some others you might find in the repositories)

You may also try [http://localhost:64653](http://localhost:64653) instead if the ip-address.
