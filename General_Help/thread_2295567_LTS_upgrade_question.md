### Post by monkeybrain20122 on 2015-09-19
I would say when the next one comes out. Some people may disagree I think it is unreasonable for a typical desktop user (as oppose to server or "enterprise" users) to use 5 year old Ubuntu release because all the software would be ancient then and  would have reached end of life upstream long time ago (even ppas cannot provide up to date version because the dependencies will be too old)

Also forget about "upgrade", it takes long time and is risky. Back up your data and do a clean install when the time comes.

Edited: Of course if you have old hardware and don't expect the new release to work then you should stick to the current LTS until it expires (or when your hardware expires first)

### Post by grahammechanical on 2015-09-19
Actually, Ubuntu 14.04 is not supported for 5 years. It was supported for 5 years when it was first released at the end of April 2014. So, 18 months of the 5 year support period has already been used up. Every two years the April Ubuntu release is given 5 years support. So, 14.04 will still be in support for a year after 18.04 comes is released but unless you intend to do a fresh install at that time you will have to upgrade to 16.04 first and then upgrade to 18.04. The only way to avoid upgrading to the next LTS version is to do a fresh install to by-pass 16.04.

Regards.
