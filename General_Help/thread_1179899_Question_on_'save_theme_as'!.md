I have my theme just the way I want it. When I go to 'save theme as', where does it save the setting? [Picture attached]

---

### Post by Legace on 2009-06-06
It saves them into:
```
~/.themes
```

(Note that it only saves data (what icons you're using, what GTK theme you're using, etc.). It doesn't actually create a theme you could install on an other computer. You'd need to install the icons, GTK themes, etc. manually :))
