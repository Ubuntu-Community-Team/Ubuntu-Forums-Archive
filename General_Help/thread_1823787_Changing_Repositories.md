Are there any disadvantages from changing repositories ?  eg. installed a latest application present in the previous repository which was not present in the changed repository or something like that.

### Post by papibe on 2011-08-12
What you can do is to change your sources for a set that has better time response for your location. Check the section "Download Server" on this [page]("https://help.ubuntu.com/community/Repositories/Ubuntu") (Other -> Best Server). It takes a few minutes to determine which one is closer, but it's worth it.

I hope it helps.
