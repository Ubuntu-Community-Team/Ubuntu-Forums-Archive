### Post by bboston7 on 2009-09-04
It seems like you already know a lot about linux in general and have a desire to learn about debian systems.

If so, I recommend dual booting ubuntu and debian because you will be forced to learn more if you use debian.  You will still learn stuff using ubuntu, but not as much because it does more stuff for you.  I'm not saying that's a BAD thing, but it's something to think about.
