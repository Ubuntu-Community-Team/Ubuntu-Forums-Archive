### Post by JayCarman on 2008-12-19
I was able to change just the wallpaper without changing my entire theme. The default image used is /usr/share/backgrounds/warty-final-ubuntu.png  I renamed this file to /usr/share/backgrounds/warty-final-ubuntu.bak.png and copied the image I wanted to use into the /usr/share/backgrounds directory named the same as the original - warty-final-ubuntu.png

It seems like there should be an easier way to do this, though.
