### Post by MikeM709 on 2007-06-07
yes it is. in the settings, before you click install, select what drive you want to install it on in there. it should say something like:

Drive 0
some stuff about size
Drive 1
some more stuff about the size

just choose which one you want it installed on. click on the details about the drive to select it, not the drive number.
