yes that's what it said but you can install even normal Ubuntu (gnome) with 256 RAM. i have it running with 224 RAM (32 is graphics card). it's got a bit slow lately. not as fast as it was on start. Xubuntu also works well. it consumes about 134 MB ram for system to run.
 
if you want to stay with XFCE (which has descent support for applications) - then Linux Mint XFCE (based on debian testing) uses only about 100 MB for the system.
 
next choice is Chrunchbang linux (dont' believe their disclaimer). This one is based on Debian stable. it has XFCE and Openbox version. XFCE will consume (like MintXFCE) about 100 MB. Openbox is close to 80 MB but that DE is a bit more limited and everything is done with right click.
 
LXDE (and with it Lubuntu) need also about 80-100 MB RAM to run the system.
 
whichever you choose the rest will depend on applications you will use. lighter applications use less RAM and CPU processes them faster. they might not look so preety but they are functional.
 
for a really low consumpiton SliTaz is the way to go (though still young and not such a big selection of applications in repositories). the whole OS is 31MB big, it load it self into RAM and will run with 16 MB ram i believe.
 
Another similar option is Puppy linux which runs again in its own unique way.
