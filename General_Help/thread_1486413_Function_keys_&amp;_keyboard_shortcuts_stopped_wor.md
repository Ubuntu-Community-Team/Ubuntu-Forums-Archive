### Post by BslBryan on 2010-05-17
Any updates that might have caused this problem?  Your post suggests not, but just curious. :P

If not, you might want to try [FnFx]("http://sourceforge.net/projects/fnfx/").  It's a rather old program designed for Toshiba computers for Fn use on Linux, though I had success with Ubuntu 8.04 and FnFx on a HP Dv6000 notebook.  

Also, before installing that, you also might want to try a program called [XModMap]("http://www.xfree86.org/4.0/xmodmap.1.html") or, for a graphical program, [XKeyCaps]("http://www.jwz.org/xkeycaps/").
