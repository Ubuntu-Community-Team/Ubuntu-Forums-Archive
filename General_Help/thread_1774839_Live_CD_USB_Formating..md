### Post by C.S.Cameron on 2011-06-03
You do not need to reformat, just delete the data on it.
If you have multiple partitions do not format with windows, it only sees the first partition and can brick the stick.
You can extract usb-creator.exe from the 11.04 iso, (using Winiso,etc), and use this in Windows to install Ubuntu Live, persistent.
