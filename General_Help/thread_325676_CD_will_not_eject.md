### Post by fragos on 2006-12-26
System eject is very important because large file copies may appear to be complete but are actually still going on in the background.  The software eject fushes the buffers and insure the write is complete.  The eject button on the physical CD won't work untill the drive is unmounted.  There is also a manual method which shouldn't be used unless the system is powered down.  Underneath the drive door or tray should be a small round hole.  Just the right size to be used by straitening out a paper clip which you'd insert strait in and push on a manaul release which should unlock the door.
