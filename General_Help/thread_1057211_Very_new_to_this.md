### Post by The Cog on 2009-02-01
Ubuntu is prabably as good as any distro for you. One big thing in its favour is these forums. I think they are more friendly and helpful than most other forums. Just install the normal Ubuntu desktop and then install the other applications you want. **Always** check the Ubuntu repositories for software before looking on the web - this is different to the way you would behave with Windows.

As for safety, Linux is much safer than Windows. You don't really need to bother with a firewall or AV software. Really. That's another differeence to Windows, and hard for windows users to believe. You should think about using a firewall at the time you install your first service - web server, SSH server or whatever.
