I noticed quite a few people with the same problem that i have been having with the Netflix desktop app. The issue was that the video and the audio were stuck in fast-forward and before the only way to make it stop was to restart my computer. I found that if you uninstall then reinstall the netflix desktop app that the problem will be fixed.To uninstall the app open the terminal(command prompt) and enter the code: 

sudo apt-get remove netflix-desktop

The prompt will ask you if you are sure you want to continue, press y and hit enter(all this will be displayed on screen).Then go into the
Software Center and install the netflix desktop app and everything should run smoothly from there.
