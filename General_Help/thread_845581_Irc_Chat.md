Pidgin works fine for IRC certainly as well as xchat; however, I do prefer xchat myself but that is a personal preference and has nothing to do with the functionality of Pidgin. 

With Pidgin for support on ubuntu be sure that when you select an IRC protocol as luisito advises above be sure **server** is set to **irc.freenode.net**. Fill in a username and password. 

When you connect a window will open up with two tabs open, one from nickserv and one from freenode connect. In the nickserv tab type

```
/join ##beginners-help 
```

to join the official channel for help with ubuntu issues 

or 

```
/join #ubuntuforums-beginners
```

to join the channel where the beginners team hangs out at.

Also of interest are the commands

```
/help
```

and 

```
/list
```

The latter (list) will list all rooms available you can join using the join command above. Most of these Rooms will have nothing to do with ubuntu.

Hope this helps. :)
