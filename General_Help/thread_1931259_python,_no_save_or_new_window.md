### Post by greenpeace on 2012-02-25
Hi, 

You've entered into Python's "interactive mode" [1].  You will not be able to save anything you do there, bit but it is a great way to test ideas out.

If you want to save and write Python scripts, you will need a text editor.  They can be very simple, like "gedit" which will most likely already be installed on your machine, or more advanced with functions better suited to a programmer (IDLE[3] is a popular choice for python)

There are many tutorials online to get you get started[2] [3]


[1] [http://docs.python.org/tutorial/interpreter.html](http://docs.python.org/tutorial/interpreter.html)
[2] [http://en.wikipedia.org/wiki/IDLE_(Python](http://en.wikipedia.org/wiki/IDLE_(Python))
[3] [http://onecore.net/how-topython-programming-under-ubuntu.htm](http://onecore.net/how-topython-programming-under-ubuntu.htm)
[4] [http://www.trainsignal.com/blog/introduction-to-python-programming-in-ubuntu-linux](http://www.trainsignal.com/blog/introduction-to-python-programming-in-ubuntu-linux)
