### Post by Mocker on 2009-12-06
Had the same problem... looking through Synaptic, I found the package "lightning-extension" which seemed what I needed. Sure enough, installing it made the calendar reappear in Thunderbird.

If you're not sure how to intall it:

[LIST=1]
[*]Close Thunderbird
[*]Open **Ubuntu Software Center**
[*]In the search box, type: *lightning*
[*]The only entry that should show up is "Lightning Extension for Thunderbird" select it, then click the arrow button on the right side to select it.
[*]Click **Install**, and enter your password when prompted.
[/LIST]
