### Post by Inxsible on 2008-05-21
Depends on what kind of theme it is. If it a GDM theme, then open the System>>Administration>>Login Window and then click on Local (i think) tab. Now drag and drop the downloaded theme file in this space and you will have the theme installed. Alternatively, you can click the Add button and then point it to wherever you have stored the theme file.

To install a Gnome Desktop theme, you need to open up System>>Preferences>>Appearances. On the Themes tab, simply click Add and then point to the downloaded Gnome theme. Or again, you can simply drag and drop the theme file.
