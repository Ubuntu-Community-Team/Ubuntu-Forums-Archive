### Post by rawlins02 on 2010-08-08
I just set up an external monitor and laptop display as dual. My xorg.conf is below. You should be able to see the arrangement of output change between monitors using the ATI Catalyst Control Center. Under Display Manager, can you not move the two monitors around, apply change, and see displays change accordingly?

EndSection

EndSection

	Option	    "Xinerama" "off"
EndSection
