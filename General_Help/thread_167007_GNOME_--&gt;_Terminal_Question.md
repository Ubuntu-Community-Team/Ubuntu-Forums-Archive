### Post by SavageBeginner on 2006-04-27
```
sudo apt-get install nautilus-open-terminal
```
Make sure you've enabled the universe repositories.  [https://wiki.ubuntu.com/AddingRepositoriesHowto](https://wiki.ubuntu.com/AddingRepositoriesHowto)
