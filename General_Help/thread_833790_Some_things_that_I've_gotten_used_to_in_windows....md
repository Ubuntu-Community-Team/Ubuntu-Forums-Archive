### Post by ad_267 on 2008-06-18
Disk defragmenter - Don't need it as Linux has a far superior file system.

Disk cleanup - What do you want to clean? "sudo apt-get clean" removes downloaded packages and "sudo apt-get autoremove" removes packages that aren't needed.

Disk check - fsck is a command line tool, don't know if there's a GUI available for it. Drives are usually checked on startup after a certain amount of time or a certain number of mounts.

Disk compress - Don't know sorry.
