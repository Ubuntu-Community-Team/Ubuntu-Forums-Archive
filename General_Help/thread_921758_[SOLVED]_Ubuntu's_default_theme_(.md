### Post by Genius314 on 2008-09-16
GTK 2.x themes will change the insides of windows (the buttons, text, tabs, etc) and also the panels (taskbar).

Metacity themes will change the window borders and titlebars.

GTK engines are usually not needed. They're required by the GTK themes, but most of the commonly used ones are already installed with Ubuntu.

Splash screens can refer to a few things. One is the loading bar when you start up Ubuntu. Another is an image that displays when you log in. There was a third, but I don't remember what it was.

### Post by fballem on 2008-09-16
> **Anathallo said:**
> Yeah, I know how to do that.
I have been on sites like gnomeart and gnomelook, and there are things like application themes, window border, GTK+ Engines (dunno what that is) and splash screen. Do I have to find one from each of these that go well together?

If you like blue (which I do), then you might want to install blubuntu.

[LIST=1]
[*]Open Synaptic (System > Administration > Synaptic Package Manager)
[*]Search for blubuntu
[*]You will get a list like screenshot 'blubuntu1.png' (attached)
[*]Select blubuntu look and mark it for installation. It will provide a list of other dependencies that need to be installed. Select OK.
[*]Select Apply - this will download and install all of the required items.
[*]Right-click on any blank space on your desktop. This will give you a menu.
[*]From that menu, select Change Desktop Background.
[*]This will give you a panel like blubuntu2.png (attached)
[*]Select the Theme tab and locate blubuntu. Click on it once.
[*]You will see a message that a background is suggested (see blubuntu3.png - attached). Click on the Apply Background button.
[*]Select the Background Tab.
[*]On the tab, you will see Solid Color (which is still ubuntu orange). Click on that color box and a panel will open.
[*]You will see an eye dropper. If you click on the eyedropper, you can then select any color on the screen. I chose the dark blue in the upper-left corner of the panel.
[*]Click OK (to select the color), then Close to close the panel. The new theme should now be applied.
[*]If you want to change the login screen, then select System > Administration > Login Window. This will produce a panel like blubuntu4.png (attached).
[*]Select the Local tab, and set it as shown on blubuntu5.png (attached). In particular, please make sure that the radio button beside blubuntu is checked and the style and theme are set as shown.
[*]Click Close.
[*]Restart your system. The login screen should look quite different than the standard ubuntu and when you are logged in, your desktop will look quite different.
[/LIST]

Personally, I have been very pleased with this theme, and it is quite different than the standard ubuntu theme.

Hope this helps,
