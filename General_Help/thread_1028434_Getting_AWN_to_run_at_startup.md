### Post by Admiral Yi on 2009-01-02
Sorry for a slightly newbie question here, but is there any way I can get avant windows navigator to run at startup? Whenever I log out or shutdown it goes off and I have to start it up again by running the launcher.

---

### Post by CJ56 on 2009-01-02
Go to System >Preferences >Sessions 

Go to the Startup Programs tab, click on Add & copy this

avant-window-navigator

into the Command box & if you want to, put AWN in the Name box

Close everything & you're good to go next time you reboot...
