### Post by oshirowanen on 2010-07-07
I have failed miserably.

I set out to find a NetBook which can play back 1080p flash video directly from a web browser smoothly at a minimum framerate of 24fps.

Requirements:
Must be able to play 1080p flash video directly from web browser.
Must have Linux installed, or work perfectly once Linux has been installed.
Must have a HDMI port, as generally NetBook screens do not have enough resolution for 1080p, hence the HDMI port so it can be hooked up to a bigger screen for playback.
1080p flash framerate must be smooth on fullscreen.

It seems the combination of a NetBook running Linux Ubuntu does not allow 1080p flash via a browser to be played back smoothly.  Even with an ION based netbook.

Why?

1. The CPU of a netbook is not fast enough to play 1080p smoothly,
2. 1080p flash needs hardware acceleration, via a GPU, i.e. Nvidia ION,

The problem:

1. No hardware accelerated flash for Linux Ubuntu
