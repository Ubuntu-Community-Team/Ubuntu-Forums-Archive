### Post by Joseph Schwenker on 2011-08-20
You need to use Language Support in System>Administration>Language Support. From there, click on "Install / Remove Languages...", then check "Japanese", and make sure that "Input methods" and "Extra fonts" are checked. The other options shouldn't matter. Click on "Apply Changes", and everything should be installed. You may have to select "ibus" for your "Keyboard input method system" in the Language Support window. Ibus might start automatically, if not, start it by going to System>Preferences>Keyboard Input Methods, then go to the "Input Method" tab. Select Japanese - Anthy from the dropdown, and click add. Open any text input and press Ctrl+Space and you should be able to use the input method.

Also, keep in mind that Ibus doesn't jive well with games like Sauerbraten (you might get stuck in the chat) and Minecraft (you might not be able to use your keyboard at all).
