### Post by MikeM709 on 2007-06-03
When it prompts you to hit F2, do so then. It will then take you into what is known as the BIOS, which is basically the most basic instructions for your motherboard to run properly. Once that screen comes up, just hit Esc. and it should prompt you to save any changes you have made, and just select yes or save because you never actually made any changes. 

this usually works for me. Also, please include your computer's manufacturer if you have one, unless you built the rig your self. Also, please report any other text on the screen at the time this happens.
