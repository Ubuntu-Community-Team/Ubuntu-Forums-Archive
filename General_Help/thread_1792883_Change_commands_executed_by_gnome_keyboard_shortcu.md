### Post by SRabin on 2011-06-28
Hi all;

I'm running Ubuntu 10.04.2 LTS, but wanted to use Banshee instead of Rhythmbox. After running ```
sudo aptitude install banshee
``` I wanted to bind my "Media" button on my keyboard to run banshee; unfortunately, it still wants to run rhythmbox. Where can I change the default command executed by these keyboard shortcuts? I can't seem to find them in gconf-editor under apps/metacity/* and googling has proved fruitless in finding where this configuration file is located. Thanks!

-S

---

### Post by mobius129a on 2011-06-28
System | Preferences | Preferred Applications. From the dialog box, go the Multimedia tab from which you can change the default multimedia application. This should also change the program launched by your media button. :)
