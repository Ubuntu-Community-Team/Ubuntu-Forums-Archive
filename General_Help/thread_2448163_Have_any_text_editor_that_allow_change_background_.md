### Post by Impavidus on 2020-08-03
There's a huge number of text editors: gedit, nedit, kate, vim, emacs, nano, ..., not to mention the really arcane ed. Every somewhat decent text editor supports search and replace. My favourite text editor (vim) allows clicking on a url to open it in a web browser, but that's not a feature of the text editor. It's a feature of the terminal in which it runs. Other than syntax highlighting, setting text and background colours is part of markup, not text editing, so pure text editors don't support that. There can't be colours in a plain text file. Colours may be supported by word processors.
