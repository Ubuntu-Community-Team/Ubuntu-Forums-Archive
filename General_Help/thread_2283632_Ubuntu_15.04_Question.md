### Post by grahammechanical on 2015-06-23
Ubuntu uses the Unity user interface and unity has been designed to work a certain way and we are limited to the amount of modification that we can do to the way it works. We do not add applications to the top panel. Unity does not work like that. What we see in the top panel are app-indicators. Certain applications come with an app-indicator but not every application. As you have noticed the basic controls of media players are available in the Sound app-indicator including an option to restore the application to gain access to all its controls.

We can lock applications to the Launcher. First we load the application and its icon appears in the launcher. We then right click the icon and select Lock to Launcher. We cannot move the Launcher from the left side but we can set it to auto-hide.

As for media applications we open the application window to do what you want. We get more than one workspace so that it is possible to have different applications running in different workspaces and switch between workspaces to gain access to the application. We can also minimise applications to the Launcher and then restore the application with one click to gain access to its controls.

It is about six years since Unity first appeared in Ubuntu and Unity is at version 7. And it has been Unity 7 for more than a year. In a years time Ubuntu 16.04 will be released and that will have Unity 7. Unity is designed to work the way it works.  I have not seen any indication that the developers are going to change the way Unity 7 works.

If this is giving you headaches, then perhaps you could try another member of the Ubuntu family of distributions. As well as Ubuntu we have Xubuntu, Kubuntu, Lubuntu, Ubuntu Gnome, Ubuntu Mate. Some of those distributions have so many options for modifications that you might find it difficult to make up your mind.

Regards.

Try Xubuntu, Lubuntu or Mate. Sounds like you might prefer one of them. You can try all of them from the install media (disk or USB) without changing your hard drive in any way or installing to hard drive.
