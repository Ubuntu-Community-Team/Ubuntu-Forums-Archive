### Post by ramjet_1953 on 2009-03-15
As a general rule, it is hard to go past HP printers, as they actively support Linux.

This site is a good one to check-up on a particular printer:

[http://www.openprinting.org/printer_list.cgi](http://www.openprinting.org/printer_list.cgi)

That said, it is maintained by users and sometimes is not completely correct, as some entries have been entered by users who have had difficulties and have not completely investigated the problem.

An example is with the particular printer that I use, which is a HP psc1350.
The above data base has been changed recently and the listing says it is a paperweight.

This is absolute rubbish, as this printer is being used successfully by many Linux users. HPLIP recognizes it and the graphical interface allows you to access all functions.

So the bottom line is to do some research before purchase and make sure that the printer you are considering is not only in the "lists" but check on the many Linux forums for comments by users.
