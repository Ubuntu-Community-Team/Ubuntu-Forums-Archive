### Post by the real omni on 2008-08-06
> **El Viejo Lobo said:**
> The specs are 512 MB ram  and Intel 1.67 monitoring by system monitor i can see that the memory is in 80% use and the processor is jumping to full usage a lot of times. The only program on is Firefox.:confused:

First of all, if this is when you're running Windows XP right now then chances are you have malware/spyware/viruses. Do a scan with the most recently-updated version of SpyBot Search & Destroy - that'll get rid of most infections and it'll almost always at least detect problems that are too severe for it to fix (in which case you'll need a specified removal app like ComboFix).

Second, 512 is a bit low - another 512 chip wouldn't be too expensive and would remarkably increase your system performance. I'm not sure the memory footprint of Ubuntu but Windows XP eats up about 300-400 MB of RAM just by itself, leaving only 100-200 MB left for programs. That means a lot of apps end up using a paging file (ie, using the hard drive as if it were memory.. same effect, just a loooot slower). So essentially if you get another 512 chip that'll give you about 4 times the amount of application memory that you currently have - lots of elbow room. :)

If you're currently running XP and contemplating an Ubuntu migration you'll almost definitely see better system performance without getting the extra ram chip. Nevertheless, I'd recommend the extra 512 chip anyway - trust me. You'll notice the difference bigtime.

Last off, since you have an Intel processor and not an AMD processor, make sure you install the i386 (32-bit version) of Ubuntu and not the AMD64 (64-bit) version. 

I'm running 8.04 on my Pavilion dv6500 with no problems.
