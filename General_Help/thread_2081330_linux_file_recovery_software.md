### Post by strafe_sawdoffe on 2012-11-06
I've successfully used a program called Scalpel to recover deleted documents in the past. A pretty good guide for using it can be found here:
[URL="http://www.howtoforge.com/recover-deleted-files-with-scalpel"]
http://www.howtoforge.com/recover-deleted-files-with-scalpel[/URL]

The example uses a pdf file, but the principle is the same. The question here is I'm not sure what Windows actually did to the drive -- if it merely deleted the file, this has a pretty good chance of working. Not gonna lie, it can get pretty complicated, but when it works, it's beautiful. 

By the way, when waiting on scalpel to run, I've found a *very* rough rule of thumb is about 1 minute per gigabyte on the drive you're scanning.

Hope this helps.
