### Post by semitone36 on 2008-10-14
I had a similar problem a few days ago.  As was mentioned above, you need to enable "Windows Decoration" under Effects in Compiz. If you still dont have frames after that, click on the Windows Decoration button to bring up the settings.  Somewhere on the page you should see the word "Command" with a input bar underneath.  If this bar is empty then that basically means that you dont have a theme manager selected to import your theme from.  I run emerald theme manager for my themes so the command I have in the command box is:

```
emerald --replace
```

If your themes come from emerald just put this into the command box and your themes will come back.  Otherwise youll have to figure out the command that runs your theme manager.
