I was looking for this very same feature.  As I understand it, if you want to install to the hard disk, you need to have the installable CD or DVD.

I haven't navigated this site well enough to know where to make feature requests, but I can tell you that one of the nicest features of all the Knoppix based systems is the ability to boot initially from Live CD, see if things work out, then install directly to the hard disk.  SimplyMEPIS does this better than any version I know of.  As you are running a desktop from the Live CD, you can press a desktop icon, answer a few trivial questions, keep working, and in the meantime, have your software installed to disk.

I recommend this functionality to be added to all user friendly desktop Live CD systems.  It's a great way to get in with very little effort.
