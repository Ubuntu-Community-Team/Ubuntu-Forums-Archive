### Post by A-DavRoyShi-X on 2011-01-15
I've made custom launchers for programs that can only run in terminal.
They open fine within the Terminal window, but it closes right after the program is finished, before I can read the report.

Is there a way to keep the Terminal window open, using launchers? I don't want to manually type the commands after opening a Terminal.

Thanks in advance. Currently using Lucid Lynx.

---

### Post by Smart Viking on 2011-01-15
In gnome-temrinal:
Edit->Profiles->Edit->Title and Command->When command exits:->Hold the terminal open
