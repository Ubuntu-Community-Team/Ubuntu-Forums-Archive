### Post by Impavidus on 2021-09-06
You managed to save your personal files from Windows using your Ubuntu live disk. Great. Now keep in mind that any attempt to repair or reinstall Windows may destroy Ubuntu, so make sure you've got backups of all personal files on Ubuntu too. Then do with Windows whatever you like. This is an Ubuntu forum, not Windows, so we can't help with that. But in general, once you've got all your personal files, don't spend too much time fixing it, or reinstall will be quicker.

Please note that Windows 7 is no longer supported by Microsoft and not safe to use on an online system.
