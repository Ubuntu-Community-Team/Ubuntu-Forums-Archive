Hi trobert,
Well, If my google search is correct, Supermaze is a place for Jave applets?  For Java to work on Ubuntu you must first install Java on you system.  The easiest way is to go to 'Applications/ Add/Remove (to install programs).'  When that window comes up search for Java using the center search bar at which time you should see Java Runtime 6 shown.  Click away and install.  Then try Supermaze again.
