### Post by sisco311 on 2019-04-06
Maybe you have to run it from the directory where is located :
```
cd path/to/dir
./appname
```

You can set the working directiry in the .desktop file with the Path key:
```

...
Exec=myapp
Path=/path/to/dir
```
