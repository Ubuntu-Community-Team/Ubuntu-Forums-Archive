### Post by StuartN on 2009-10-08

If you mean how to combine, say, accent marks and characters, then you are better off using the Compose key than Unicode. The sequence for an umlaut is Compose, ", vowel - Compose, ", u -> ü etc. There are some more sequences (and Unicode equivalent) here - [http://hermit.org/Linux/ComposeKeys.html](http://hermit.org/Linux/ComposeKeys.html)
