### Post by sinfree on 2007-05-09
I've been researching this issue for you and so far it looks as if you are correct in your assessment, that OO does not have the capability to play music in the slide background across mulitple slides.

First, this issue is in the bug list on the OO website and has comments here: [http://qa.openoffice.org/issues/show_bug.cgi?id=24969](http://qa.openoffice.org/issues/show_bug.cgi?id=24969)

Second, there is an interesting forum discourse about it here: [http://www.oooforum.org/forum/viewtopic.phtml?p=165334](http://www.oooforum.org/forum/viewtopic.phtml?p=165334)

Third, there is a "workaround" (not really a workaround if you ask me) that seems to involve using Microsoft's free PowerPoint viewer to view the slides.  Obviously this is not a good solution.  The link about that is here: [http://www.oooforum.org/forum/viewtopic.phtml?p=165334](http://www.oooforum.org/forum/viewtopic.phtml?p=165334)

So, I would recommend checking out that first link (the OO bug list link) and at least commenting that you are having the same problem, to help give the problem more visibility.  Other than that it looks like there isn't a good solution in OO.

Right off hand I can't think of any other good linux PowerPoint alternatives.  I'll post if I find anything better.
