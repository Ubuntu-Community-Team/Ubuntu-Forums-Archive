### Post by peitschie on 2008-05-12
You need to change the monitor specifications by going to System > Administration > Screens and Graphics.  It probably says "Generic Monitor" or something in the Model drop down.  Change this to a screen that supports the resolution you want to run things at.  

Oh, if its a wide-screen make sure you have checked the "Widescreen" checkbox down the bottom of the monitor selector.
