### Post by JuergenF on 2009-09-22
If this [Lubuntu article]("http://www.linux-mag.com/cache/7520/1.html") is right, Xubuntu uses even (marginally) more ram than the 'normal' Ubuntu/Gnome.

But if you are new to Linux you probably don't want to wait until Lubuntu comes 'official' and not do the manual install mentioned at the end of the article.
