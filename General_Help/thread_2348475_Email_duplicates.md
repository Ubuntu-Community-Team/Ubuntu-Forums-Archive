### Post by amanchesterman on 2017-01-14
I use Thunderbird but I had the same (?) problem, i.e. two identical emails (same time, same recipient) in the 'Sent' box for my Hotmail account. It was because TBird was synchronising both with the 'Sent' folder and with the 'Archive' folder on the Outlook server.
I think the 'archive' folder in Outlook operates like the 'all mail' folder in Gmail, i.e. it keeps a copy of everything that's sent.
Solution: set your email client to synchronise only with the 'Sent' folder.
