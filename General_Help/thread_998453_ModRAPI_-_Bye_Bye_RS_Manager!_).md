### Post by zenikal on 2008-11-30
modrapi is a linux commandline downloader for users of rapidshare.com that have premium account.

modrapi functionality:
- links storage
- import from a webpage
- import from a file
- manual link add
- download bandwitch limiter
- free disk space checker - modrapi block download when there is not enough space
- duplicate blocker

[COLOR="Red"][SIZE="3"]how to install[/SIZE][/COLOR]

download tar archive from here: [http://rapidshare.com/files/101290881/modrapi-1.2.tar.gz](http://rapidshare.com/files/101290881/modrapi-1.2.tar.gz)

unpack it into /usr/bin

$ chmod a+x /usr/bin/modrapi
$ modrapi configure

full help in english (stable)

$ modrapi -h
