### Post by anaconda on 2006-12-29
You have to enable USB from your VMWare player. 

Open vmware, but dont start the virtual machine yet. 
Go to edit virtual machine settings, and select USB and make sure it is enabled. Then start the virtual machine, and select VM from the top, and go to removable devices and USB devices and select your USB-device..

Then ubuntu should find it..

(sorry if there are mistakes in these instructions.. I have never used vmware -player. But I do often use vmware server, which is also free. This is how it works in the server version.)
