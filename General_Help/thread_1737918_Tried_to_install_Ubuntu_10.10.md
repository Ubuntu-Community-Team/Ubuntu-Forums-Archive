### Post by coreytrice on 2011-04-27
So I ordered a USB to SATA cable. I took the drive out of the netbook and hooked it up to another computer. That let me format the drive while it was hooked up by USB to the other computer. Then I was able to install Ubuntu 10.04. Netbook is back together and working great. Bonus is that it no longer has the ever useless Windows 7 Starter on it. Totally, Ubuntu machine now.
