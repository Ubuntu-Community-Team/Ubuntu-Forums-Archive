### Post by stchman on 2008-01-24
> **Cibola said:**
> I am a novice. How do I install from the repos??

I know I will get yelled at by the other forum members but I have had great success with the Java 5 version.

Here is how you install Java.  Open a terminal and do the following:

```
sudo apt-get -y install sun-java5-bin sun-java5-fonts sun-java5-jdk sun-java5-jre sun-java5-plugin
```

This will give you everything you need.
