This is good advice! Bear in mind, however, that running off a live CD is going to be much slower than running a 'regular' installation.

Also, try doing the things you'd usually do with Windows. Initially it may be tricky finding the apps that do the stuff you're used to, but they usually exist.

### Post by malspa on 2010-06-24
Lots of people say this is a bad way to go because you don't learn as quickly, but if you can swing it, find a company like System76 that sells Linux-preinstalled computers.  Then you avoid any hardware issues that you might encounter between Linux and machines that were made for Windows.

I started out with a Linux-preinstalled computer and it enabled me to take my sweet time learning about Linux while being able to use the computer for a lot of things right away.  After a few months of that, I was sold on Linux, and knew there was no turning back.

A nice thing about a Linux-preinstalled computer is that it's usually quite friendly if you decide to install a different distro than the one it came with.

---

### Post by malspa on 2010-06-24
Also, I would suggest thinking in terms of joining the Linux community, starting the Linux journey, not necessarily joining the Ubuntu community and starting the Ubuntu journey.  Ubuntu simply isn't always the best Linux distro for every person, and the Ubuntu community isn't always the best Linux community for everyone.

Playing around with different live CDs is certainly a good way to started.
