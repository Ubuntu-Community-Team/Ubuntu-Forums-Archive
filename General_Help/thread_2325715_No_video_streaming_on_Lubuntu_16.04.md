### Post by mc4man on 2016-05-24
You'll need flash for that site so either install & use google-chrome which will play the stream without you doing anything
[https://www.google.com/chrome/browser/desktop/](https://www.google.com/chrome/browser/desktop/)

Or for firefox install flash, plus you'll need to allow it (flash) on the site, you'll get a FF prompt about that
```
sudo apt-get install flashplugin-installer
```
