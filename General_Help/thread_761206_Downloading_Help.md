### Post by chewearn on 2008-04-21
> **MosMusy said:**
> I am new to Ubuntu so help me out... every time I download something I get the source code from the archive reader, however I am not able to install only look at the code. How can I install the programs that I download?

You don't normally install program by downloading the source code.  Use "Add/Remove Programs" in the Application menu, or use Synaptic Package Manager:
Panel Menu > System > Administrations > Synaptics Package Manager


If you really need to install a program by source code, find the readme file in the tar package for instructions.  Also, I'm sure the website where you download the source code will have the instructions as well.
