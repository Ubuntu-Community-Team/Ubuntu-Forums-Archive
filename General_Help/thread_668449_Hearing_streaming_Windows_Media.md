I struggled with this issue myself. Use Synaptic Package Manager and search there 
for Movie Player. Install it. Look for the streaming package for MP.
I now get audio and video clips much better than I ever did using Windows.

The reason these Linux multimedia players do not work online is the Codec
owners will not cooperate with Linux Project groups. Codecs are proprietary. Not
cooperating is one way the Microsoft Led Computing Cartel machinates to 
blocks serious competition.

Some ingenious engineers actually watched the main codecs run, line by line, 
and used the information obtained to reverse engineer them for Linux.

When you must choose a player to stream anything Movie Player with the
imitation codecs can run "Real Player" and "Media Player" video codecs just fine. 
It works faster and it is very rare to get what you so often get with MS Media 
Player "Do not have this codec" "Wish to go online to find proper codec". Then 
you consent to the hunt which is nearly always a big waste of time. You also 
get Real Player format media without Real Player ad intrusions.

If you cannot find the package yourself, just search Google for Movie Player 
Streaming You will find a post that explains it. I find the search engine here 
is a waste of time. It cannot handle the huge post archive here. You could try 
the site search command in google. Its simple: site: paste this site's address 
search word (make sure you have a space between the address and your 
search string or keywords.
