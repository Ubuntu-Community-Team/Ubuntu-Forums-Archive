### Post by ryanhaigh on 2008-07-28
Start nvidia-settings as root using the command below and you will then be able to save to xorg.conf

```
sudo nvidia-settings
```

### Post by northern lights on 2008-07-28
> **ryanhaigh said:**


While this is theoretically true, **never** use 'sudo' to run graphical apps as root. Use 'gksu' instead.

For references, read [https://help.ubuntu.com/community/RootSudo]("https://help.ubuntu.com/community/RootSudo")
