### Post by James_Lochhead on 2009-04-20
If you have a relatively new computer the changes are basically aesthetic. They use two different graphical environments QT and GTK... therefore applications from one work in the other (provided they are not very very specific).

There are small differences in usability as well. I think the GNOME project is simpler, less resource and easier for new users. Whereas I think the KDE UI is (in some aspects) more advanced. They also try to provide a more tailored solution - with more apps based purely around KDE.

Basically it comes down to which looks better for you and which has features you like. KDE is good for me, it has a split panel file browser, nice file browser console, lots of eye candy by default (GNOME needs more customization) and widget integration.

If you don't care about eye candy either should be fine.

### Post by James_Lochhead on 2009-04-20
