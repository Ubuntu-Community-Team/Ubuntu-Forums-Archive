### Post by dickohead on 2006-02-08
sudo apt-get install kubuntu
or
sudo apt-get install kubuntu-desktop

then when you login select KDE as your session.

### Post by aysiu on 2006-02-08
Applications > Accessories > Terminal ```
sudo apt-get update
sudo apt-get install kubuntu-desktop
```
