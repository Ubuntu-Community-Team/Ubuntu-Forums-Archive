### Post by akakingess on 2010-04-19
No, unfortunately not, and I didn't have much luck getting it to work through Wine, either, although I don't like Wine, only because to me it defeats the purpose of going Open Source ;) If you really want to run Dreamweaver, I would suggest setting up a VM and running it through that. Although, now that I think about it, I was trying to run CS3 through it, so you may have more luck, and don't count out stuff like Amaya, Screem, and some of the other WYSIWYG editors that are already open source. It is all a matter of personal preference and if you do a search, just google open source HTML editors, and you will find quite a few out there. Not necessarily up to the level of Dreamweaver, but I have become happy using them ;)
