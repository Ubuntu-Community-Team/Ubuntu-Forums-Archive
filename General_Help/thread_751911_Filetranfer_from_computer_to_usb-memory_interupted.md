### Post by mrsteveman1 on 2008-04-11
Ok, so this stick is the only place you have these files, and you cant get to them but the stick works fine.

Whats the filesystem by the way?

There are a number of programs that can "undelete" things from most filesystems, photorec is one that ive used a few times (its cross platform too). The problem is most of these programs rely on signatures, as in, most files have a defined header the program can recognize.

Are they pictures? Text? Programs?

Ok, those are pretty easy since they at least have some defined structure, they are all really text files anyway.

Its menu based and pretty easy to use, i think its in the ubuntu repo but if not you can download it from the website : [http://www.cgsecurity.org/wiki/PhotoRec_Step_By_Step]("http://www.cgsecurity.org/wiki/PhotoRec_Step_By_Step")
