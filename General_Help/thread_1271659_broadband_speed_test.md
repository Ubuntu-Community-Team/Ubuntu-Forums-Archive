### Post by scragar on 2009-09-21
1000kb/s = 1Mb/s
1MB/s = 8Mb/s
1000kb/s = 0.125MB/s

B = bytes, b = bits.
(This is actually approx, most places will advertise Megibits(but call it megabits) so these numbers will be right, if not you gain an extra 24bytes per kb, 24 kb per MB etc, so 1MB = 1.05 Megibytes)
