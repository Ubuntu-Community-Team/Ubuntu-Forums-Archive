### Post by NewJack on 2007-06-27
For the most part you don't have to worry about things like that.  Check out this link for some tips: [http://ubuntuforums.org/showthread.php?t=429694&highlight=ccleaner](http://ubuntuforums.org/showthread.php?t=429694&highlight=ccleaner)

Also, if you are using Firefox you can do the following:  Edit, Preferences, Go to Privacy tab, under Private Data- check off the option to always clear private data.

Furthermore, in Synaptic check under the STATUS tab and you will see something that might say RESIDUAL.  You can delete anything under that option since it was installed more than once.

Hope that helps.

P.S.-  Linux doesn't get fragmented like Windows so you don't have to worry about that.
