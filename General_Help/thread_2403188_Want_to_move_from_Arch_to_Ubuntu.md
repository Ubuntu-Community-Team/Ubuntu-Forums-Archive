### Post by deadflowr on 2018-10-08
[https://help.ubuntu.com/community/Openbox]("https://help.ubuntu.com/community/Openbox")
Typically though if you want just a minimal openbox only install (and not one with the default Ubuntu desktop also installed) you need to use either the server install or the ubuntu mini.iso
You can get the mini.iso from here
[https://help.ubuntu.com/community/Installation/MinimalCD]("https://help.ubuntu.com/community/Installation/MinimalCD")
And I think you need the alternative server version from here
[https://www.ubuntu.com/download/alternative-downloads#alternate-ubuntu-server-installer]("https://www.ubuntu.com/download/alternative-downloads#alternate-ubuntu-server-installer")
as the regular server version has some changes
[https://ubuntuforums.org/showthread.php?t=2390785]("https://ubuntuforums.org/showthread.php?t=2390785")

I think that should be a good starting point.
The installation is pretty straight forward. (or not, I think it is, others don't; too each his own)
When you get to the section that lists packages to install you need to go down to the manual select option and find the openbox listings.

Once you have it installed if you want help figuring out various settings or what have you please feel free to start new thread on whatever specific setting you'd like to have help with.
