### Post by davmac on 2009-03-09
I'm not aware of how to fix this issue with Linux, but you can use an Ubuntu LiveCD to recover the data.

When you get the "cannot mount volume" message, it does tell you how to force the mount. Not always the wisest of things to do but if your system is totally borked anyway... Start up a terminal and type in the mount command.

I have recently gone through this with a friend's WinXP desktop machine and my daughter's Vista laptop.

Once I'd got the data recovered I then used the original windows install disks and during the boot up selected the option to "repair" the windows installation. If that doesn't work then you may have to reinstall, but at least you'll have your data recovered.

Hope this helps,

Dave Mac
