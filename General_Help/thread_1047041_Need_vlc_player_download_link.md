### Post by adamlau on 2009-01-22
Assuming Intrepid, repositories show 0.9.4 to be the latest available version: 

```
sudo apt-get update
sudo apt-get install vlc vlc-plugin-esd mozilla-plugin-vlc
```

Latest (0.9.8a) Source: **[http://download.videolan.org/pub/videolan/vlc/0.9.8a/vlc-0.9.8a.tar.bz2](http://download.videolan.org/pub/videolan/vlc/0.9.8a/vlc-0.9.8a.tar.bz2)**
