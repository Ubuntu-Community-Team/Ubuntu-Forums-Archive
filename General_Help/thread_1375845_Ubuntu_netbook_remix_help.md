### Post by Thomar on 2010-01-08
That's an older version of the Netbook Remix.  The old version had that right sidebar, and also let you switch between the interface you have and the desktop.  They took away those features in the later releases.  A poor design choice in my opinion (I just cleaned off UNR and installed normal Ubuntu because of the window maximizing thing, which caused windows to get detached from their title bars.)

[https://help.ubuntu.com/community/USB%20Installation%20Media](https://help.ubuntu.com/community/USB%20Installation%20Media)

This link explains how to use the Unetbootin application, which will go and download your image for you.  You can tell it to use a version older than 9.10, but I don't know which version had the features you're referring to.  The link also mentions an extension for .img images, which you might need.

Does anybody here know at which version they changed UNR to not have the right sidebar and Switch to Desktop features?
