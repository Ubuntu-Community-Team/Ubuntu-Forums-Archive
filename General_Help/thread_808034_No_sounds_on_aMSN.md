### Post by angry_johnnie on 2008-06-29
I was having that same issue.  Here's what I did.

Assuming you're using Hardy Heron (8.04), what you do is launch amsn, and then go to  Account > Preferences > Others > Sound Server.

I think by default it's set to 'use a different program' and the command is set to **play $sound**.

Change that command to **aplay $sound**.
