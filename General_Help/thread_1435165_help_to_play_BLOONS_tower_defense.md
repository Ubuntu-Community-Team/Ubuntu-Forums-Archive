Sorry, the game is working perfectly on my machine. Do you have the Flash plugin properly installed? (i.e. did you install the 'flashplugin-nonfree' package? That's all you need to do - it's no more 'mucking around' than installing anything else.)


> Questions 

Q) how do I install / Download a browser in Ubuntu with BUILT IN support for "normal" Flash Files ?

There's no such thing on any platform. All browsers depend on SOME plugin for Flash Player. They can't have built-in support, because Flash Player is proprietary software made by Adobe, and Adobe doesn't make a web browser.

> Q) is there any development going on for Ubuntu, so that it can play "swf" normally ? - with no ad-ons / Additional commands  ?

Well sorta - there's Gnash, but it's a reverse-engineered open-source plugin and won't work with much Flash content. That's probably causing your black box problem; just install regular Flash Player and it will work fine.
