### Post by grahammechanical on 2014-04-04
At present the Ubuntu code that can be installed on certain reference mobile devices is different from the Ubuntu desktop code. I would not attempt to install desktop Ubuntu on to a mobile phone let alone Xubuntu. The code in present day desktop Ubuntu and its flavours is not compatible with mobile phones. It does not have the hardware drivers.

Another thing to consider, is that it is one thing to take an application that was built for a mobile phone and then fix it so that it scales up for a tablet and even a desktop monitor. But it is something else again to take an application that is built for the desktop and expect it to be usable on a mobile phone.

By this time next year the Ubuntu phone/tablet code will be converged into the Ubuntu desktop code so that there is but one Ubuntu code base but the convergence is coming from the phone/tablet down to the desktop and not the other way around.

Also keep in mind that the developers of the flavours have yet to decide how much of this converged Ubuntu code they will accept into their flavours of Ubuntu. At least one set of developers have rejected out of hand Ubuntu's replacement for the Xserver (MIR) in favour of some (any) compositor built according to the Wayland protocol.
