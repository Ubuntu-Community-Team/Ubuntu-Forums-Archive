### Post by JacobRogers on 2007-10-17
I figured out what was wrong.  I hand typed the name of the font as "Liberation_s_ Serif" instead of "Liberation Serif". It must of been using the "Times" font because I didn't type the font name in correctly.

After I actually put in the right font name, it worked perfect.  Formatted the same on all computers/programs.
