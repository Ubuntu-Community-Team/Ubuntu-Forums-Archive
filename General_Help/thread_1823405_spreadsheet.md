### Post by Nimbus200 on 2011-08-12

As mentioned in the dialog box :
"Cell protection is only effective after the current sheet has been protected"

### Post by The Cog on 2011-08-12
First, remove any protection for that sheet: Tools, Protect Document, Sheet

Second, select all cells by typing Ctrl-A or clicking the top-left corner

Third, set the cells to unprotected: Format Cells, uncheck Protected. At this point, setting the sheet to protected would have no effect.

Fourth, select only the cell you want protected, choose Format Cells and set this cell back to Protected

Fifth, use Tools, Protect Document, Sheet and set the sheet to protected.

You now find that the cell you wanted protected cannot be changed but that all other cells can.
and set them to unprotected : Format Cells,
