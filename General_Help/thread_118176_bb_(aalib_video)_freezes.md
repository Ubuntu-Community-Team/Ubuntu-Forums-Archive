### Post by pyr0man99 on 2006-06-14
I had the same problem.  I tried reinstalling all the aalib stuff and bb but it still froze.  So far the only thing that has worked for me has been to disable ubuntu software sound mixing, logout, log back in and then run it.  I don't know why it doesn't like SSM but it worked.  Also do not disable SSM in BB, only disable it in the ubuntu sound preferences
