### Post by phantom3113 on 2009-04-06
If you're copying something using an AIO printer, you should be able to do all that on the printer, regardless of OS. However, if you're talking about scanning, try Xsane under Applications>>Graphics. You could also try looking here >> [http://www.avasys.jp/lx-bin2/linux_e/scan/DL1.do](http://www.avasys.jp/lx-bin2/linux_e/scan/DL1.do) for iScan, although I believe it's only for Epson printers/scanners
