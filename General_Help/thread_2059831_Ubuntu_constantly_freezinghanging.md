### Post by newb85 on 2012-09-18
Is it hanging (as in you have to wait) or freezing (as in you have to restart the machine)?

If it's constantly freezing, it could easily be a problem with your RAM.  You can check this using Memtest86.  Boot up the machine, and when the GRUB bootloader comes up, select Memtest86.  It will thoroughly test your RAM for corruption (which will take a while).

Note: If you don't have GRUB set to show on boot up, you can bring it up by hitting the Esc. repeatedly when the power-on self-test screen comes on.
