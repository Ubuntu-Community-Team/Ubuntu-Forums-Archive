I've started using amarok as my default audio player.  I've only run into one snag.  I really like dynamic mode, but I can't get it to only play songs from one playlist. It seems to always pull from my entire collection.  Here's what I'm doing, can someone please tell me what I'm missing?

1. Create smart playlist as usual. Select the Smart Playlist.

2. Click on checkbox that says Enable dynamic mode

3. Click on "Show Options"

4. Select "Playlist Shuffle" from dropdown menu.

At this point, I click on "repopulate" and watch as tracks not in the smart playlist are added to my playlist.

### Post by supenguin on 2005-12-16
I know its odd replying to myself, but I just figured it out... After turning dynamic mode on, right-click on the playlist you want.  One option is "Add to from dynamic mode".  Click that.
