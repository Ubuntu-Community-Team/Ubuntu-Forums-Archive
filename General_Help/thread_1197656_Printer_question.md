### Post by HappyFeet on 2009-06-26
I have an HP printer that works great. However, at the top of each page, it prints out:
```
File: /home/user/Desktop/new file       Page 1 of 1
```
I do not want that printed on each page. How do I stop it from doing that?

---

### Post by Tamlynmac on 2009-06-26
Is this problem limited to the text editor? For example it's not doing it in Open Office. If so open the Accessories/Text Editor and type something, then select file print and when the print window opens select the text editor tab and uncheck Page Header / Print page headers.

You do not need to save the document to have this change saved in the Text Editor. Just select "close without saving". It will save the change to the Page Header and when you open it again it won't print the header.

I do have one question. Are you using this driver?
[http://hplipopensource.com/hplip-web/install.html](http://hplipopensource.com/hplip-web/install.html)

### Post by Tamlynmac on 2009-06-26
I forgot one quick note. If you wish to turn off the headers in Firefox. Select "file/print/options tab/Header and Footer" Change the options to blank.
