### Post by tukatz on 2012-10-17
Hello, does KOBO reader work with Ubuntu or is there a Linux version of Kobo available?

Solved!Kobo does work for Linux 64bit systems! 
I installed the following from the software center, I don't know if I needed both or which one worked but as follows: installed libzip1, installed libpng12.0, went to Kobo for linux desktop and downloaded it. Opened the kobo exe file with Wine for windows program loader and followed the prompts. It works! Thanks to Chas. for help!

By the way; Linux has ebook readers as well, they work. Find 'em in Ubuntu software center.
