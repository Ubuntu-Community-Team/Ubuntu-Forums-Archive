### Post by jetsam on 2010-04-26
I use Kate sometimes for Python.  It works fine and there's not much to learn, though the menus are long.

The bootstrapping programmer crowd would have you use Idle, but Idle is best if you are just playing around in my opinion.

SPE is a bit confusing to install, but a full featured and Python native Python IDE.  Also, Stanni deserves the donations he asks for for the documentation.  More info [here]("http://ubuntuforums.org/newreply.php?do=newreply&noquote=1&p=9178483").

Eclipse is originally for Java and can be used for nearly everything, but it's over weight: really really really big.  

Vim is easier to take with [Cream]("http://cream.sourceforge.net/").  No weight gain required.  

Summing up, my recomendation: Start with Geany as in the post above, and, in the nicest way possible, learn how to read a manual.
