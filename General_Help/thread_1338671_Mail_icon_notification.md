### Post by OpSecShellshock on 2009-11-26
I'm not sure this function has ever been available. I tried finding information on the same thing about a year and a half ago with no luck. I think Evolution has to be running in order to poll the mail server for updates.

One thing you might be able to do is set Evolution on a widget layer in Compiz Config Settings Manager so that it will be hidden but still running. Then when you get a message notification you can hit the button that toggles the widget layer to read the message. It's not the most elegant solution in the history of ever, but it will save you some screen space.
