### Post by mcduck on 2009-01-11
I use Gedit (the default text editor) for all my PHP, XHTML, CSS and scripting work.

If you feel you are missing some feature, check it's plugins and enable which ones you like. There's also additional "gedit-plugins"-package in repositories which adds a load of extra plugins. And you can find even more form the net.

Gedit has one, as a plugin. ;)

Also you can install Gnome-color or Agave for the same purpose. Gnome-color simply allows you to pick a color from anywhere (or using the color circle) and then displays hex code for it, while Agave is actually a color scheme creator but is also able to pick colors from your screen.
