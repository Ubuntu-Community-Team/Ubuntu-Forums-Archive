### Post by Honeybee on 2007-07-11
Nevemind, i've fixed it! Turned out it wasn't todays update, it was just a coincidence. While the update was happening I saved an SVG file from inkscape to my desktop. I moved that file to my hole folder, and my desktop started working, but as soon as I browsed to my home folder nautilus froze again. I've now moved that SVG file to a /broken in my home folder, and everything works until I open /broken.  So it looks like Nautilus is falling over when it tries to load the thumbnail for that particular file.  I'll submit a bug report for this later today. 

Anyway, all fixed now and working.  Sorry for jumping to conclusions that it was todays update that broke it!
