### Post by friendly_guy on 2005-02-22
Slax is an interesting live disc.  It is Slackware based and currently about 188MB. 

 It can be run entirely from RAM which is useful if you only have one drive, because you can remove the CD & rescue data by burning a CD.  

You can install Slackware packages to a running copy of Slax (not many live distros you can do that with) so you can install just about any tool you need as long as you can get a package (linuxpackages.net is usually the best place).
