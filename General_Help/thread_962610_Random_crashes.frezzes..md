### Post by Helge on 2008-10-29
Go to console mode (ctrl-alt-F2) and login there. Run top to see if there is some process that steals the cpu resources. It could be some Gnome applet. If there is such a process, try to see what happens if you kill it. Alternatively, you can login remotely while you have the (blank) desktop visible, and do the same thing from there.
