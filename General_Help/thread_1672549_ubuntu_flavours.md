### Post by kittkatt on 2011-01-21
Just stick with the regular Ubuntu if you want maximum support, all of these flavors use the same repositories of software anyway so it doesn't really matter too much (you just download the programs later on that didn't already come pre-installed).

Xubuntu, Lubuntu - used for older/slower hardware (netbooks, etc) - not appropriate for you

Mythbuntu - used for TVs (aka Windows Media Center) - not appropriate

The other distros are just Ubuntu w/different choices software programs preinstalled.

Stick to dual boot until you are REALLY comfortable using linux.  There's no downside and you never know when you might have to use a Windows-only program or can't figure out something in linux.

### Post by mastablasta on 2011-01-21
Don't expect Windows programmes to work in Linux. IF they have linux client (linux version) then they work just fine. Some of them work through WINE, but it takes a bit of additional processing power to run them, so they might not run as smooth as in windows.
 
Try Ubuntu and Kubuntu (which is more like windows in the way it looks and feels). They are different desktop environments.
 
Ubuntu uses Gnome (which has largest support in applicaitons) and Kubuntu uses KDE (which is secod largest in applicaiton support and is currently more advanced of the two). also you will be able to run Gnome applicaitons in KDE and vice versa. so no problem there.
 
Basically just try them out see what you like. As a side note i should add that is quite easy to later install additional desktop environement (such as KDE, XFCE, LXDE...) and then you can switch between them before you login.
 
too many informaiton, i know. best read the Ubutnu manual.
