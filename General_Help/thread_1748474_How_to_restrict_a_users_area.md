### Post by Frogs Hair on 2011-05-03
From Users and Groups you can change user permissions . Unless the folders are public one user should not be able to look at the others folders if separate accounts are set up for each and there is no sharing turned on . If the different users are sharing one account I don't think there is much you can do that is practical.
