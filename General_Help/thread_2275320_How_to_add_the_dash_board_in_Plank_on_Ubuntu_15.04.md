### Post by mc4man on 2015-04-25
You need to  - 
install xdotool
Then create a dash.desktop & add to plank. It can go into ~/.local/share/applications for ease of editing, ect. (no root needed
basic one, click on icon opens Dash home, right click exposes lens choices - 
```
gedit ~/.local/share/applications/dash.desktop
```

```
[Desktop Entry]
Version=1.0
Type=Application
Name=Dash
Exec=xdotool key Super
NoDisplay=true
Icon=/usr/share/unity/icons/launcher_bfb.png
Actions=Apps;Files;Music;Vids;Pics;

[Desktop Action Apps]
Name=Applications
Exec=xdotool key Super+a
TargetEnvironment=Unity

[Desktop Action Files]
Name=Files
Exec=xdotool key Super+f
TargetEnvironment=Unity

[Desktop Action Music]
Name=Music
Exec=xdotool key Super+m
TargetEnvironment=Unity

[Desktop Action Vids]
Name=Videos
Exec=xdotool key Super+v
TargetEnvironment=Unity

[Desktop Action Pics]
Name=Photos
Exec=xdotool key Super+c
TargetEnvironment=Unity
```

No you cannot remove the unity launcher completely or not easily. Just hide & make pressure to expose very high..

### Post by mc4man on 2015-04-25
> **MikeMecanic said:**
> I just removed Plank because I was not able to restart the computer since i Install it.  Will try it again later.  See this post: [http://ubuntuforums.org/showthread.php?t=2275380](http://ubuntuforums.org/showthread.php?t=2275380)
That is easily fixed, the autostart just needs a 1 or 2 sec. delay which isn't uncommon for auto-started apps to act correctly

I wouldn't of bothered with that sudo ln -s /usr/share/applications/plank.desktop /etc/xdg/autostart/, would get rid of the link & the plank.desktop file  that may of showed up in ~/.config/autostart & just make one in ~/.config/autostart like below, blue of 1 or 2
```
[Desktop Entry]
Type=Application
Exec=plank
Hidden=false
X-GNOME-Autostart-enabled=true
X-GNOME-Autostart-Delay=[COLOR="#0000CD"]1[/COLOR]
Name=plank
```

Or if wanting to keep an existing ~/.config/autostart/plank.desktop then just add the delay line to it
X-GNOME-Autostart-Delay=[COLOR="#0000CD"]1[/COLOR]
