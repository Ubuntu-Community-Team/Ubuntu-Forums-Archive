### Post by igknighted on 2007-02-07
I use an ATI card and the results via the ATI driver have been terrific.  Also, there is a community based open-source ATI driver as well which support 3D acceleration (although it isn't as fast as the commercial one, it is nice to have that option.  There is no open-source driver for nvidia that supports 3D acceleration).  The drivers are just an easy install via apt (or synaptic, your choice) as they are already in debian packages in the Ubuntu repositories.  Sometimes the Live CD doesnt play nice w/ ati cards, so if you have issues just install via the alternate CD then install your proper drivers and you will be good to go.

### Post by AusIV4 on 2007-02-08
I've been using Ubuntu for a little over a year, and I can't begin to express how far it's come in that time. Breezy badger wouldn't even start on my laptop (I only used it on my desktop) because of video card issues. Now there is a decent open source driver for my card, and a good binary driver, if you're willing to run closed code on the kernel level. I use the open source driver, but I'm not a gamer, so you may need the binary driver.

There are lots of tutorials on how to get WoW running. If you have trouble, Transgaming's Cedega supports WoW, and I think it's supposed to be a breeze.

As for the blackberry, I've seen a few how-tos on syncing blackberries using Linux. I don't know if it offers full support, but I suspect it's pretty good.

Some other areas I've seen Ubuntu improve in the last year are eye candy (beryl is amazing), Flash (upgraded from 7 to 9). My wireless card was automatically recognized by Edgy. Seems like there are other things that work out of the box now that didn't work when I first started playing with Ubuntu.

I should say, Ubuntu is not an attempt at creating a free version of Windows. I assume you realize this, since you've tried using Linux before, but I've run into several people who think they're getting a free, more secure version of windows, then think Linux sucks because it doesn't behave exactly the same.
