### Post by gordintoronto on 2010-07-20
USB Startup Disk Creator can create a "persistent" flash drive, which means changes are saved on the flash drive. You can install software, change the wallpaper, create data files, etc. It will be slower than working from a hard drive. (Until we get USB 3.0...) You don't have to create the flash drive on the target computer. Just don't install an nVidia video driver and then plug the flash drive into a computer with an ATI video card.
