### Post by jeecol on 2007-06-24
Dear all, 

Recently I need to prepare some figures for writing manuscripts (for journals in microbiology field).

According to the instruction of authors (AEM, [http://aem.asm.org/misc/ifora.shtml](http://aem.asm.org/misc/ifora.shtml))

1. Figures should be in tiff or eps files.
2. for Line art, the minimal resolution is 1200 dpi



**What do I do:

1. I used RLplot for making plots and then exported plots as eps files, about 7 Kb for each (tif file at 1200 dpi was 36 Mb, oh! my God!)

2. I used gimp and set a eps file at 1200 dpi, black and white (it bacame 174 Kb from 7 Kb). I put four plots (eps files) into OOo Draw to make a figure. I printed the OOo Draw file, and the plot quality was very good. 

3. I tried it once. I put the 4 eps files into OOo Draw to make one figure (without changing resolution in gimp), and then printed it. The quality was as good as the previous one. 



**My question: 

1. I know tiff, jpeg, and png files can set the resolution, but how about EPS file? 
(It seem not!)

2. OOo Draw is capable of exporting eps file, how about the quality for publication?  


This is a milestone for me. After I finish those figures, I can tell my friends that I use ubuntu and some good software to write my thesis and manuscripts for journals.
