### Post by Agent Smith on 2011-01-16
The reason it changes to red theme when you chose emerald is that, this is the default if you have no themes added. go and choose a theme for emerald or beryl from gnome look. Download it then run up emerald manager and choose import on the right hand side. Search to where you saved the theme and choose it. It will then appear in the emerald window. Click on the new theme and it will change. Also install compiz fusion icon. Set it to run on start up and then you can use it to easly change window managers, rather than using the terminal and typing emerald --replace etc. Hope this helps.
