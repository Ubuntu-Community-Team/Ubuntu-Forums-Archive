OpenOffice 3.0 on Ubuntu 9.04

Launchpad.net is a way to get help online, but I am completely lost on that site, so I am asking here on the Ubuntu forum.  Can anyone help me out?

### Post by L a r r y on 2010-02-24
OK I found it.

Select my bitmap that has everything covered up.

Go to the Modify menu and select Arrange submenu.

There I can Bring to front, bring forward, send backward and send to back.

I did as I suggested above and moved everything off, put the bitmap in place, and dragged my first data back in place -- behind the bitmap, with an outline of the item showing it as selected.

Right-clicked and found the arrange menu item on the context menu with the choices as above.

Evidently these data pieces are not considered to be layers, for when I made the layer invisible, it made the bitmap go away while the data item was still selected.

Go figure!
