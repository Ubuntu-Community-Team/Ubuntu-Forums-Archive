### Post by Vaphell on 2012-01-29
audacious preferences > plugins
find Global Hotkeys and configure it.
media keys for pause and play are XF86AudioPause and XF86AudioPlay
It appears somewhat buggy though, I have play/pause on a dedicated button on my keyboard and audacious recognizes only the play part of the play/pause combo (track is always restarted, never paused).

i prefer to use Z X C V B keys when audacious is active.
