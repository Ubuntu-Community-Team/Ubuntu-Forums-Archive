### Post by iowabeakster on 2013-01-24
Just sounds like the file, that needed to be installed, was corrupted as you downloaded it.

The check sum is a standard way that verifies that file that got to your computer was intact.

Nothing to worry about.

When you run the app the first time, it will install two packages called mono and gecko.  Is it one of those packages where you got the error?
