### Post by guiverc on 2020-10-06
Just as you can add the binary package `ubiquity` to your system with

```
sudo apt install ubiquity
```

you can also download the source using

```
apt source ubiquity
```

You will however need source enabled (*by default it's not enabled as most people don't want it;* see [https://help.ubuntu.com/community/Repositories/Ubuntu](https://help.ubuntu.com/community/Repositories/Ubuntu) where you can tick the checkbox titled "*Source code*")

I'll also refer you to [https://ubuntuforums.org/showthread.php?t=1512568](https://ubuntuforums.org/showthread.php?t=1512568) or the thread *How to get the full Ubuntu source code*.
