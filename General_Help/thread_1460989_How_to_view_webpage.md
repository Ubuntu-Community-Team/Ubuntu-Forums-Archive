### Post by cogier on 2010-04-23
First you need to go to your web site, right click on the page and select "View source" (or similar).

Select all the text and copy it to gedit. Then save as an .html file, then open it with Firefox.

It wont show you any of the pictures unless you are online and can access them from the web site or you download each one of the pictures and save everything in the same folder.

This is only the start, there is lots more to learn but hopefully it will be a start.

### Post by Serpher on 2010-04-23
Open it up in firefox. You can do this 3 ways:

1) Use the URL bar (file:///home/<user>/Desktop/index.html for example)

2) Right click the HTML file and select "Open with..." and select your web browser

3) In Firefox go to File --> Open and navigate to the HTML file.


Also if you are using Open Office make sure you save everything as a .html file, however I suggest just using gedit or vim is even better if you're comfortable with it.
