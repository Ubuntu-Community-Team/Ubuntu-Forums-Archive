### Post by y-lee on 2008-05-26
Just a guess on the firefox issue :confused:

If you launch firefox from a launcher right-click the icon. Properties>Launcher> Change firefox %u to firefox

If you launch firefox from the menu Go to the menu editor (Right-click on the main menu, and choose Edit Menus). Go to Internet, right-click on the Firefox launcher and choose properties. Again change Change firefox %u to firefox.

If it already says firefox then try changing it to firefox %u.

The same idea may apply to kopete.
