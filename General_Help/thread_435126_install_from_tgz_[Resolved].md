### Post by aysiu on 2007-05-06
Depends on what the tgz file *is*. Is it a theme? Is it a program? What kind of theme? What kind of program?

tgz just means it's a compressed file. It'd be like saying, "How do I install from a .zip file?" in Windows.

Read this:
[http://www.psychocats.net/ubuntu/installingsoftware](http://www.psychocats.net/ubuntu/installingsoftware)

That's probably a good place to start.
