### Post by 3rdalbum on 2009-10-27
Ubuntu 9.04 has Firefox 3.5, but it installs alongside 3.0 for reasons of software stability and predictability. Look in the repositories if you don't believe me - firefox-3.5 is the package name. The actual name in the Applications menu is the Firefox 3.5 codename (Shireko or something like that) so you don't get confused.
