### Post by Anden008 on 2005-12-11
Hi.. Iam using an english keyboard for my laptop, and want to be able to write æ, ø, å (danish characters). Any way to set this up in ubuntu? or a script of a kind?

It would be perfect to use " alt - ; " as a shortcut for æ. " alt - ' " for ø and " alt - [ " for å...  

Any suggestions are very welcome.. thx

### Post by Thunk on 2005-12-11
1. Install the Danish layout System -> Preferences -> Keyboard. Move to the Layouts tab and Add the Danish one.

2. Move to the Layout Options tab and press on Group Shift/Lock Behaviour. There you can choose a combination of keys that will switch between the layouts (English / Danish). When you're on the Danish layout just press the appropriate key to whatever combination you want. You can find the appropriate keys in the picture of the Danish layout.

I hope this helps.

### Post by taygan on 2006-02-02
Special characters can be typed by crtl+shift+unicode  for example ctrl+shift+e7 gives you a ç.  applications>accessories>character map will tell you the codes.

Look under LATIN for European characters.

ctrl+shift+e6 æ
ctrl+shift+e5 å
ctrl+shift+f8 ø

enjoy!
