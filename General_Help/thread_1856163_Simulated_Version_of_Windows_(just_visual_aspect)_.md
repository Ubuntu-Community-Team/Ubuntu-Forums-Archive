### Post by Mark Phelps on 2011-10-08
First of all, there is no single "visual aspect" of Windows just as there is not single such thing of Linux.  Each version of Windows has a different UI: XP, Vista, Win7, and now, Win8.

So, the first thing you need to decide is WHICH of these you want to study.

Second, the Trial version only runs for 90 days.  If you want something long term, you will have no choice other than to buy it.

The cheapest option, and the one probably most useful these days, would be to get Win7.  If you're a student, check with your school -- they often have Win7 available for a LOT cheaper than retail rates.  And, I would get the cheapest version of Win7 you can find -- Home Basic, if you can find it.  Enterprise, is not avaiable for purchase in retail channels, and Ultimate costs a fortune.
