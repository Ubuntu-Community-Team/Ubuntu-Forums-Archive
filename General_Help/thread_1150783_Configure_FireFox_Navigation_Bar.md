### Post by lovinglinux on 2009-05-06
You could set the page you will allow to be the home page in the Firefox preferences, then remove the search and the address tools from the toolbar (right-click >> customize), then remove the customize option from the context menu using [Menu Editor](https://addons.mozilla.org/en-US/firefox/addon/710) and lock things using [Public Fox](https://addons.mozilla.org/en-US/firefox/addon/3911).
