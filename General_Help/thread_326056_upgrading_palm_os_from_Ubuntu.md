### Post by fragos on 2006-12-27
There's more than one file syncing application available.  To can however install without bothering.  Odds are your Palm has an SD slot.  If you remove the SD from the Palm and plug it into a reader on your PC you can drag and drop new applications to the SD card.  Since not all Palm applications will run off an SD card the safest thing is to return the SD card to the Palm and move the  PRC file from the SD to Palm Internal RAM.  PRCs are all self installing when you run them the first time.  I use a program on the Palm from [www.mobile-stream.com](www.mobile-stream.com) called Card Reader which makes your palm mount its SD card on Ubuntu directly.  This will work over the USB cradle.  Now your Palm is a Card Reader for the PC and don't have to remove and replace the SD card.
