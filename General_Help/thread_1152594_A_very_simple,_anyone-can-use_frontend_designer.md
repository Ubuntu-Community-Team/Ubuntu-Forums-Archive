### Post by Kennick_Tech on 2009-05-07
Hello Ubuntu community,

I would like some feedback on an idea I have (and am sure multiple users have had) to better improve Ubuntu (and all of linux) by getting the most out of user feedback. As it is obvious, user input is very, very important and there needs to be an easier way for everyone to get their ideas in, or rate existing ideas.

My idea for this is an extremely simple frontend designer, that would be included in the release and hopefully a shortcut on the desktop. This designer (for KDE and GNOME) would just have the basics; drag and drop a label, place a button here, etc. Then the user could go into detail about what the program does, what certain buttons do, and whatever else they need to explain about this program. From there, they can upload this design from the designer and users will rate and leave comments. If someone doesn't want to go through the hassle of doing all that, they can just browse through the already made ones and leave their input.

I really think this model will help Ubuntu and all of linux greatly. I find that in my spare time when I want to give back to the linux community through programming that there are well too many projects and I tend to go for ones that *I* want, rather than what the *community* wants. And those of us that do develop know how hard it is taking the very simple user input and making it into something good, but the highest rated/most popular designs would be the main focus, meaning there would be more developers for them to make it a reality.

Please give me feedback on this idea: what you would want from it, should it be included by default, how it should look, anything. I own a business and would be more than willing to dedicate company time and resources to making this happen if there is enough support for it (free, open source and no advertising obviously). So please let me know what you think!
