### Post by kvarley on 2009-03-01
Pros: Free, looks very nice, easy to get the hang of, loads of free useful applications, some amazing free multiplayer games, windows applications can be used with emulation, some commercial games availiable for Ubuntu, great helpful community for when you get stuck!

Cons: You may have some trouble with getting devices to work, drivers are sometimes sketchy, you will need some time to get to know the operating system.

Hope that helps!
