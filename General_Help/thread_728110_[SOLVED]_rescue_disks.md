Yup, any of those that you mentioned can be used. However, there are special system-rescue-cd's available if you are concerned about rescue cds... try [www.sysresccd.org](www.sysresccd.org)

But remember that carrying the Knoppix or any other live cd ISO image around with you is no good, you must have a bootable cd burnt using that ISO image.
