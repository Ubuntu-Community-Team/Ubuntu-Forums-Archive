### Post by mendhak on 2011-11-05
I believe your purchase is limited to the Ubuntu Software Center download.  That means if you buy a game, that game is meant for all of your Ubuntu installations.  

If you can find the .deb or installation package and if it runs on Fedora, then that's great, you can use it there too (but I don't think that is the case).  It also means that it won't run on Windows.

You can also email the developers, if you're desperate, and tell them that you've purchased it on Ubuntu but want to use it on Windows too.  It depends entirely on them.  However, I don't think that many of them will agree to this.
