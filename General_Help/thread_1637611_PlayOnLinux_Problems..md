### Post by daniel_w93 on 2010-12-14
PlayOnLinux is just a front end for wine.
It runs scripts so you can easily access the Installer on the Game DVD.
So when you click on install and select a programm...say Borderlands, it searches for the DVD to run the Setup.exe file. so...no dvd...no games.
