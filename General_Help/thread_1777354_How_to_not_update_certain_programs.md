### Post by joelstitch on 2011-06-07
Every time I use Update Manager it tries to update a program that I don't want to be updated because the older version has a option that the new one don't have. How would I remove the application from the list to update?. The application is ZSNES.

---

### Post by Matt__ on 2011-06-07
open synaptic, type in the program name, select it and from the 'package' drop down menu 'lock version'.
