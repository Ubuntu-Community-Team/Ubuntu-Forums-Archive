### Post by fragos on 2007-12-24
I've never seen this problem before and I've been a Linux user for a long time.  You mentioned only having 2 USB ports.  Sounds like a very old box.  Compaq had the first commercial boxes with USB.  They jumped the gun on the standard and their USB ports had real problems with USB devices designed for the standard.  If you want help I recommend the following:

1. Include all the information you can -- you didn't even tell us which distro and version you're running or what hardware you have.  No one can help you with what you provided.

2. Use a title like "Can only use one USB port at a time" so that readers have some idea about what the subject is.

3. Don't insult the people you're asking for help and then threatening to go back to Windows because it's better.  Learning any new thing takes effort.  If you show us you don't want to make the effort we won't bother to help you.
