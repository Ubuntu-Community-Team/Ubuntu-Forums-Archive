### Post by lithops on 2014-02-23
I currently have a dual boot system (Ubuntu 12.04 and Windows XP) and have at last arranged my software so that I do not need to use the Microsoft side and would like to remove it. Here are some questions I have about removing the Microsoft side:

1. Is there any software on the Microsoft side that is accessed by Ubuntu?

2. The reason for asking this question is that I have a power line adapter so I can access the modem upstairs.  I had to install software for the microsoft side but nothing for Ubuntu.

3. If the answer to 1. above is "yes" then can I clean out the Windows side (reformat) and then remove the partition and so give all of the hard drive space to Ubuntu?

Appreciate guidance.

### Post by 3rdalbum on 2014-02-23
Ubuntu is entirely self contained, it doesn't access anything from the Windows drive. In fact, the power line adapters don't require software even on Windows.

You can erase the Windows partition. You might need to run Boot Repair from the Ubuntu CD to get Ubuntu to boot again afterwards, and I'm not sure if you can incorporate the Windows partition into the start of the Ubuntu partition. But you can certainly have it as an extra partition that you can store files in.
