### Post by MooPi on 2010-04-10
Yes. You need full privileges of the drive in question. Not necessary to install but can be run from the directory. I've used Songbird and Firefox in this fashion.

---

### Post by ajgreeny on 2010-04-10
I think it is only apps like firefox, thunderbird and songbird that will run in this way as they are run directly from the executable which is in the tarball you download, and not "installed" in the normal fashion.  To the best of my knowledge it is not possible to do the same with the deb files downloaded and then installed using apt-get, or synaptic etc etc.
