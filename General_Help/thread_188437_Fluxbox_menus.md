### Post by Rui Pais on 2006-06-04
hi,

to add something to actual menu edit and add to the file ~/.fluxbox/menu

to change, add or delete global menu (for all users) change, as root, /etc/X11/fluxbox/fluxbox-menu.

edit: and there is a package named fluxconf that contains an utility, fluxmenu, that seems to offer a gui.
