### Post by Enriquecaribe on 2009-05-05
Okay so I managed to do something stupid. I installed the ume, net-book interface for ubuntu (the interface for the eeepc) on my HP Dv2000. I don't wan't to reinstall hardy because I have my laptop set up perfectly.

So! Once I uninstalled the ume/netbook interface (Because I didn't like it that much) I was stuck with every single window opening up maximized. No matter what I use (compiz and metacity is all) my computer opens up all the windows maximized. I tryed figuring out a way to fix this but can't. I looked in gconf-editor and also found nothing. So I need some help! Theres nothing wrong with all the windows opening up maximized but after awhile its really annoying.

---

### Post by decoherence on 2009-05-05
this is handled by the 'maximus' package which is installed with the netbook stuff. Uninstall that and you should be back to normal.

---

### Post by Shibblet on 2009-05-05
Go to Preferences->Startup Applications and uncheck Maximus. Logout and log back in ... it should be OK now.
