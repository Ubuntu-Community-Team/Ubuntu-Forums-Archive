### Post by Vinu Raj V on 2009-09-27
I would like to know that, is it possible to upgrade Ubuntu to Kubuntu upgrade through CD. I need both Gnome and KDE. I have a slow internet connection. Thats why I prefer this way. It takes long time to install KDE through internet.

---

### Post by Johnny B on 2009-09-27
with the cd inserted, go to system > administration > software sources
uncheck everything under 'downloadable from the internet'
check the box by the cdrom
install software.
