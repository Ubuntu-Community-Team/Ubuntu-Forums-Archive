### Post by Mad dog on 2010-05-03
Phew! I got it back.

I went into System >Preferences >Appearance   and went to the visual effects tab and guess what was selected: 'None'. So I clicked 'Extra', it searched for available drivers and put Compiz back to work.

I'm never using ```
sudo shutdown -h now
``` again. I hope nothing else is broken.

I really like 10.04 so far.  :D
