### Post by nordlöw on 2009-05-05
I find it surprising (crazy :confused:) that *extended attributes* aren't enabled by default in Ubuntu. This especially because indexing services such as *Beagle* benefits enormously from this when a *reboot* occurs. They "can never be sure" that any given file hasn't changed so they need to re-scan or re-checksum it eventhough it has already be scanned by the indexer (prior to reboot).  This is a really bad use of resources and will probably *degrade the user experience* aswell.
