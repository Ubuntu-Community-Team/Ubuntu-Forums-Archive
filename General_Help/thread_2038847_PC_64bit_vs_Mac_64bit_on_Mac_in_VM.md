### Post by mcduck on 2012-08-07
I'd assume the mac-specific version uses a mac-friendly bootloader, in which case the difference is irrelevant when you run it in a virtual machine. (actually for VM use you'd probably want the PC version which is more likely to be supported by the virtualization application you use)
