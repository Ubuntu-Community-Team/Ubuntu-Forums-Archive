### Post by mb_webguy on 2009-03-16
> **redenex said:**
> I my new laptop yesterday, HP TouchSmart TX1005 AU. Pretty decent performance. I am going to install Ubuntu 64 BIT onit today.

Just one question. The system has a 3 GB RAM, is it anyway a problem while using64 BIT OS? Or should I opt for 32 BIT?

Any guidance appreciated, since I would be starting to install in about 3 hours from now,

64-bit Ubuntu will have no problem with any amount of memory you can currently cram in your box. 32-bit only supports up to 4GB.

As for installing Ubuntu on your TouchSmart, I've seen a mixed bag of results.  Some people have reported that the touchscreen, wireless, and a host of other components won't work, and others have reported no problems after a bit of configuration.  I don't know why the wireless wouldn't work, even if you have to use the Windows driver with ndiswrapper.  As for the touchscreen, you might find [this blog post]("http://rkvsraman.blogspot.com/2008/10/gotcha-for-hp-touchsmart-pc.html") helpful.
