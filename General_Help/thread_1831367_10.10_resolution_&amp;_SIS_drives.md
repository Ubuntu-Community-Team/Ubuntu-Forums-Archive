### Post by realzippy on 2011-08-23
You have to run the xrandr command when ubuntu runs to see the maximum screen you can get in Ubuntu.

bero@linux-l264:~> xrandr -q
Screen 0: minimum 1024 x 768, current 1024 x 768, [COLOR="Lime"]maximum 1024 x 768[/COLOR]
default connected 1024x768+0+0 0mm x 0mm
1024x768 76.0* 

the * shows the currently used resolution.
