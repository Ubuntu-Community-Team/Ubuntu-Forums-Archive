### Post by mickdansforth on 2009-01-15
The juploadr (blue bird logo) is a basic uploader with no frills. Drop the photos in the box, wait for it to process the drop, and then press upload. Slightly better than the on-page uploader.

kflickr (diagonal flickr dots) and the Flickr Uploader (flickr dots as square photos)(avail in add/remove) are way better. 

kflickr will make a new set for you when asked at least half the time. With Flickr Uploader you have to have created the set prior to opening the Flickr Uploader program. Use the on-page basic uploader to upload the album cover photo  and then create the album. 

Both allow you to select photos to bulk caption and tag. You can also bulk title, but I suggest using the Thunar File Manager (avail in add/remove) to bulk rename the photos before dumping them in the uploader.  

Both upload photos in order. I have never uploaded with juploadr. I load photos in, see that it will do none of the work for me, and close the program. 

I broke my kflickr. It still works with my second flickr account, but will no longer find my main flickr account online. I tried disallowing and reallowing, but it is no go. 

You can upload photos to different sets from different folders on your computer with both uploaders. On kflickr new photos to the top, with Flickr Uploader on the bottom. 

The Flickr Uploader can handle some where around 1000 photos. More than that and it craps out. 

All the uploaders take time to find the photos, so if you are dumping in excess of 50 photos, it might take a little time to prepare the photos for upload before you can add tags, descriptions and stuff. 

kflickr is not a big fan of caps in tags. it always makes the first letter of the tag lower case. But it remembers your tags, so when you start adding a tag, it wants to finish it for you. Each tag is sepparated by the uploader. Type tag, type enter. Type next tag, type enter. Ect.

There is a Desktop Flickr Organizer (blue F on a white book) (in the add/remove). Not overly impressed as of yet. The Interface does not seem conducive to bulk captioning of previously uploaded photos. Which is what I was really hoping it would do.

[http://greggman.com/edit/editheadlines/2005-11-23.htm](http://greggman.com/edit/editheadlines/2005-11-23.htm) Someone created a Bulk Downloader for Flicker to download an entire set, pool, or stream for Windows. It would be nice if someone made one of these for Ubuntu.
