### Post by destruego on 2006-11-13
Hm, well, I can't say I know much about how it all works, but I'm using an ATI Mobility Radeon 9000, and I've read how the fglrx drivers are supposed to work for this, but ever since I started using Ubuntu (Breezy was my first version) its been using the ATI drivers that Ubuntu automatically installs, and 3D screensavers and XGL/Beryl work flawlessly (and overall 3D stuff works better than in Windows using the proprietary drivers), where as in other distros would use fglrx by default, and the 3D stuff just chopped like mad. If you're using an ATI video card, try changing the driver name to "ati" in xorg.conf if you've got it installed... I know the default ati driver works better for me than the fglrx driver, and here's hoping it will work better for you too.

Cheers. (And sorry if this doesn't help or I've come across as an idiot!)
