### Post by tea for one on 2022-07-16
Yes, you can switch workspaces in Ubuntu 22.04 using Super and mouse scroll wheel.

You can use Extensions or Extension Manager to enable/disable each extension to check if the function returns.

### Post by Dennis N on 2022-07-16
If you use "workspace indicator" gnome-shell extension, you can use mouse wheel alone on the indicator to switch from the active workspace to another workspace without visiting the Activities Overview.
