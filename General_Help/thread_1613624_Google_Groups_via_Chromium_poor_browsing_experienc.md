### Post by lovinglinux on 2010-11-04
Install Prism:

```
sudo apt-get install prism
```

Install Prism extension for Firefox:

[https://addons.mozilla.org/en-US/firefox/addon/6665/](https://addons.mozilla.org/en-US/firefox/addon/6665/)

Visit Google Groups, click "Tools >> Convert Web Site to Application", fill the necessary fields and voilà.

Since Prism is xulrunner based, it should probably render just like Firefox and since Prism creates a separate profile for each application, you can achieve the separation you need between Firefox and Google Groups.
