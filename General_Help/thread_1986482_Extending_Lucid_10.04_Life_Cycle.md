### Post by zombifier25 on 2012-05-25
Ubuntu Server supported for 5 years means that the packages typically used in Ubuntu Server will be updated for 5 years, not all packages available, so no. If you install ubuntu-desktop on your server then that package will only be supported for 3 years, not 5.

### Post by dcstar on 2012-10-01
I recently moved from 10.04 to 12.04 (I **always** wait until the LTS matures for its .1 release), removing Unity and installing the Gnome Fallback desktop.

I tried the "Mate" Gnome 2 fork and while it was ok, it was missing all the nice tweaks the Ubuntu used in the Gnome 2 environment so I gave that the flick and stuck with Gnome 3. I also keep using Evolution - just like 10.04.

This gives me virtually all the functionality of the old Gnome 2 desktop and the up to date package maintenance of a LTS along with some things not available in 10.04.

Just be careful removing all the Unity guff because one package is required to give you the Login screen.
