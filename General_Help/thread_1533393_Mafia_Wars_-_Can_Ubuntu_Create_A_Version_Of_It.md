### Post by potat on 2010-07-18
Mafia Wars is an application that runs in Facebook (not a web app or a standalone app). It is owned by Zynga. For it to be included in Ubuntu, Zynga would have to write a standalone version that is cross-platform AND open-source it. Pretty unlikely to happen.
