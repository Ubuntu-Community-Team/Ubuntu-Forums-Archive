### Post by tgeer43 on 2009-08-25
As you might expect, the professional version of Nitro PDF is not free and there doesn't appear to be a Linux version anyway. While I can't make a specific recommendation, there are quite a few good, free Linux PDF tools out there but you'll have to research to see if they contain the special features/functions that you require.

If you still have you copy of Nitro PDF Professional for Windows you could always try running it under Wine. WineHQ.org may have instructions for installing and running this app under Wine.
