Wine is in the Ubuntu repositories, or the latest version is available from the winehq repositories - both can be installed by Synaptic once you make the required changes (do a search for the details on how to achieve this).

Wine is supposed to work with most distros, but you may have to compile it yourself if someone else hasn't packaged it for a particular distro.
