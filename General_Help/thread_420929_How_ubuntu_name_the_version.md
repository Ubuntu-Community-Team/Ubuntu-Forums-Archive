### Post by lassegs on 2007-04-24
Easy. 

Each version starts with a new letter, from A - Z (actually I think they started with B, Breezy Badger). 
So the newest version is Feisty Fawn and the previous was Edgy Eft.
Along with this they get a number: Feisty Fawn is 7.04 and it means that it was released 2007 the month of April. Edgy Eft was released October 2006, so it was named 6.10. 

You get it?
