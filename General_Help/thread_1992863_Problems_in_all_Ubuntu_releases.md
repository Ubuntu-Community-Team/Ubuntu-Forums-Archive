### Post by fozturk on 2012-06-01
I think you didn't install the required plugins. For your bank, you should have a java runtime environment, jre, installed on your computer. You can find OpenJDK Java 7 runtime in software sources. This will fix your problem with the bank. I have this installed and can see the login window. 
As for the radio, it didn't work on my chrome. But it works fine on Firefox. I guess, it is related to plugins again. 
I advise you to install Ubuntu Restricted extras package. It also includes the OpenJDK Java. You can find it on Ubuntu Software center. 
As for the virus program, it clearly states that you should have any kind of windows operating system. You can always use ClamAV under linux to check for your windows drives. That is what we do in our University.
