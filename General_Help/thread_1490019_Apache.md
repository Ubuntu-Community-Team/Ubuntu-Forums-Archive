Q: How do you troubleshoot with in Apache?
A: It depends on the problem, there are many things that can go wrong. Start by checking your config files and error logs in order to locate the problem

Q: How do you set up a webserver within linux?
A: Install the web server software of your choice, configure it and add content. It will be accessible via your remote IP address.

Q: How do you access linux-based services from a linux client?
A: This is really too ambiguous to answer... you can access Linux based services via the internet using a web browser or some other software designed to communicate with said services.

Q: How do you add groups and users using CLI tools?
A: Add user = "useradd username", Create Group = "groupadd groupname"

Q: How on earth do you set up a printer?
A: You plug it in and if necessary (it rarely is) install the drivers for it. 

Q: what exactly is bash scripting?
A: Bash scripting is a scripting language used to execute commands or do simple processes. It is a highly adaptable language and can be used for anything from opening files to making miniature programs.

Q: How do you find and use file manipulatioin tools on the command line?
A: There are many tools for manipulating files on the command line. For example there is convert, a program used for manipulating images, mplayer can be used for playing music or converting movies/music... the list goes on.

Q: When is the CLI command used?
A: There's no such thing... I'm guessing you mean, when do you use the terminal? The answer is whenever. The terminal can be used for numerous tasks and can be, in many cases, more efficient than a GUI.

Q: What is a Gnome or KDE?
A: They are both desktop environments.
