### Post by hal8000b on 2015-01-02
Are you talking about kodi media center?

[http://kodi.tv/download/](http://kodi.tv/download/)

If yes then that is a complete linux distribution based on Ubuntu. The image is an ISO image
so you have proberby damaged or replaced part or most of your Ubuntu 14.04LTS

If you cannot login to Ubuntu after a reboot then you must try repair via live CD:
[https://help.ubuntu.com/community/LiveCdRecovery](https://help.ubuntu.com/community/LiveCdRecovery)

or a reinstall.
To install kodi alongside Ubuntu you need to create a separate partition andd dual boot.
