### Post by Irihapeti on 2011-06-04
Anything I say is just my opinion, so please keep that in mind.

There's no law that says you have to upgrade if the installation is doing what you want and it's still supported. 10.10 is supported until April next year, so that gives you plenty of time. Any security issues in 10.10 are addressed with the usual updates.

If it were me, I'd try 11.04 on the desktop machine using either a second partition, or a live USB with the persistence option set. That way you could see if you can resolve the drivers problem without touching the other install. If you like what you see, then you can proceed further.

From what I've seen, upgrading rather than doing a clean install is rather hit and miss, and very much miss in a lot of cases, so I wouldn't recommend it. I like cats, so I wouldn't like to see yours kicked. :)
