### Post by JonUK76 on 2012-02-09
In the Nvidia settings program under "X Server Display Configuration" there is the option to change the dual monitor configuration.  You can change it to settings like cloned (pretty useless normally) or Twinview (which I believe is expanding the desktop over two displays). 

[IMG]http://4.bp.blogspot.com/_Lb-i_VCD1tE/SvCmBK29usI/AAAAAAAAANg/MCkrQ1gv66Q/s400/nvidia-settings.png[/IMG]

Once you make changes you "Save to X Configuration file", and then log out and log back in for the changes to take effect.   Writing to the configuration file ( /etc/X11/xorg.conf ) needs administrator access so you may need to run it from a terminal as an administrator ( sudo nvidia-settings) if it doesn't run it in administrator mode by default.
