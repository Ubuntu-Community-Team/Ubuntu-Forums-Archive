### Post by Therion on 2008-10-31
If you have Windows applications that you can't find acceptable linux alternative applications for, you have a couple options:  1) Emulation using something like Wine or Crossover or, 2) Virtual Box or VMWare and running Windows in a virtual environment.  Oh, or the third option: Dual booting Ubuntu and Windows.

Which approach is best for you depends on what applications you want to run that are not native to linux, how many there are and so forth.  There's no one single solution that's best for everyone.
