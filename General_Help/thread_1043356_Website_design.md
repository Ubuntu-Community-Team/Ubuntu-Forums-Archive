Cheers.

It makes no difference what OS you use to make the web site. Besides, majority of web servers are running Linux or BSD anyway.. ;)

For the programs, that depends on what you want, learn to code the pages by hand and to understand what's actually going on, or a nice GUI app that allows you to drag&drop stuff around to make web pages with little effort and no control over what's going on in the background.. (In other words, do you want a program for coding web pages or some WYSIWYG editor like GoLive/FrontPage/DreamWeaver?)

I'm a multimedia designer myself, building web pages (as well as other media stuff) for my living. I use Inkscape and Gimp to create graphics and then code the pages with Ubuntu's default text editor, Gedit, with couple of nice extra plugins enabled.

I have my doubts you'll be able to create anything impressive solely using a WYSIWYG app. I would suggest you start by learning how to code (X)HTML, and perhaps CSS. You can then use dreamweaver to manipulate things quickly, but be prepared to dive in and edit the code to tailor things to the look you're trying to create.

I hate to say it, but if you're looking for WYSIWYG for internet, then perhaps look to Adobe Flash. It's cumbersome, closed-source, and generally linux unfriendly, but a lot can be done with little skill.

### Post by fragos on 2009-01-18
There is no silver bullet to easy good web design. There are CMS tools like Wordpress and Joomla but I found My sites load faster and have higher SEO ratings when I hand code them in HTML and CSS withs bits of javascript and PHP as required.
