### Post by oldfred on 2015-12-13
Not sure what you are looking for?
Grub is not an operating system and has just a few commands. It is a boot manager (menu) and a boot loader.

If you want to choose language when you boot you probably need to add a script at end of boot process.

### Post by yancek on 2015-12-13
The image you posted is the language selection page from the installer which is not on the system but on the installation medium.  If you want to change the language you can go to the System Settings icon in the panel on the left side of the Desktop.  It is the gear/wrench icon.  Open that and there is a Language Support option there.  Follow the steps to change or add language support.  If you want something else, you will need to expalin.
