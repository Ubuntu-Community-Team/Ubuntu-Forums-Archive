### Post by SaintDanBert on 2010-10-02
On my Jaunty laptop, there is a question mark icon on the top panel.
**Properties** tells me that the command is "yelp".  The shell tells me
```

prompt$ which yelp
/usr/bin/yelp

```

Other assistance is available from
[https://help.ubuntu.com/9.04/index.html](https://help.ubuntu.com/9.04/index.html)

and from **man pages** found on your workstation.
```

prompt$  #=== command line
prompt$  man  man

prompt$  #=== graphical
prompt$  xman


```
