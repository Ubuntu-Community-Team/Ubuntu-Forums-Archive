### Post by louieb on 2007-12-07
Not a problem. Just remember play safe and backup everything you want to keep in case you make a mistake. 
One thing though - most Linux distributions Ubuntu included want two partitions. one for the OS and one for swap.  
Start by looking here [Psychocats Ubuntu Linux Resources]("http://www.psychocats.net/ubuntu/index.php")
Look for the Desktop CD Install link
Pay attention to the manual partitioning section because that is what you need to use to install in an existing partition. 
Good Luck.
