### Post by WRDN on 2008-07-02
If you want to use Ubuntu, you could install [Xubuntu]("http://www.xubuntu.org/") which uses the Xfce Windows Manager and is great for machines with little power.
Though Xfce is quite lightweight (saving resources), one of the best Window Managers, in my opinion is [Fluxbox]("http://fluxbox.sourceforge.net/").
All current Ubuntu distributions support the requirements you mentioned.

If you wanted an even smaller (and fast!) distro, you could try something like [Puppy Linux]("http://www.puppylinux.org/"), though this only supports a limited number of wireless devices.
