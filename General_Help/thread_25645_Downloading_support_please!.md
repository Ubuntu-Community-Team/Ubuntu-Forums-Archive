### Post by artinla on 2005-04-10
In particular, apt-get and synaptic are the things you need to learn about.

In a nutshell, you set up some internet repository sites in a file called /etc/sources.list and then you poll them.  That gives you the updated list of available packages.  You can then browse and selectively install them from within synaptic.

apt-get does the same thing, from the command line.
