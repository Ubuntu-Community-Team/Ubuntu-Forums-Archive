### Post by Archmage on 2008-06-25
It deepends on the condition of your harddrive and what filesystem it is on.

If you harddrive isn't broken and on your harddrive is either NTFS (standard in XP) or FAT32 (standard in 98) you can get your files.

### Post by owlgorithm on 2008-06-25
It will work. Linux can read Windows-formatted hard drives, and it will let you read files from a separate XP HDD.

The nice thing about trying Knoppix is you will know before you even install it if it will read the hard drive -- it is a CD that can be run in live mode, which allows you to "try before you buy" (although it is free as in beer). The Ubuntu LiveCD works the same way and would allow you to verify that everything can be read. Then, you can decide how you would like to partition your system, etc.

Just download the regular Ubuntu CD :-)
