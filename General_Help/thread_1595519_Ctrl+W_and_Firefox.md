Hi All

On the Mac platform the shortcut CMD+W will close the currently focused window. This same shortcut, when focused on Firefox, will close one tab at a time and close the window when only one tab is open. On the Linux platform Ctrl+W behaves with Firefox exactly as expected. However all other windows close by use of the Alt+F4 shortcut. When I change the "Close Window" shortcut to Ctrl+W it closes windows as expected but it also closes Firefox regardless of how many tabs are open. Basically, setting the "Close Window" shortcut to Ctrl+W overrides the "Close Tab" shortcut in Firefox.

Is there someway to calibrate Ctrl+W to the "close window" action in every app except Firefox?

I'm not ranting or saying this is a flaw in Ubuntu. I understand how a single shortcut closing windows and tabs can be annoying for some people. Especially if you have a ridiculous number of tabs open. As a personal preference I'd like Ctrl+W to do both.
