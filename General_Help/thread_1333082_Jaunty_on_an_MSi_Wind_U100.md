Hello, I run Jaunty on my MSi Wind U100. I have had a little trouble with Ubuntu.

I made this thread for the purpose of:
Letting other people with this kind of computer share their bug reports and workarounds so that others can fix their problems too.

I'll start this:

Problem with Youtube videos? try this:
```
sudo apt-get install ubuntu-restricted-extras
```
That should get it back up!
It has flash player in it. It worked for me!:p
