### Post by Schalken on 2007-06-17
Ubuntu automatically uses unused memory for caching the last bunch of files that were read from the hard drive so that if they are needed again it doesnt have to read them from the hard drive a second time. For good performance the cache should be using all of the memory that isn't being used by programs, minus a bit at the end for room to move. Hence the amount of memory being used for the cache in your situation is optimal.
