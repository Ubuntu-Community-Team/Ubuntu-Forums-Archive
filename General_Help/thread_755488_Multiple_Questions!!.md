### Post by sekinto on 2008-04-14
**- How do I get Msn to work. I mean Pidgin is great and all but i need msn with all the great little thing's it has.**

Well you could run it though Wine probably, but I doubt you would want to do that.
Have you tried Kopete? It is a very nice multi-protocol messenger like Pidgin.
Have you tried aMSN? It is very similar to MSN.
Both the programs I mentioned can be downloaded using Synaptic (System>Administration>Synaptic).

**- When i try and watch a video on youtube the bar right under the video with the volume is way out of size and if I try to fast forward or whatever the video messes up.**

Did you download the flash plugin using Synaptic, for me the one in the repos was kind of buggy. I downloaded the official installer from Adobe's website, maybe it will work for you (you would want to uninstall the one in the repositories first).

**- How do i watch videos on break because well the video's don't come up the screen area where it should be playing is stays white.**

If Break uses flash the fix I suggest above "might" work. If Break has plain old video files you could install VLC and the VLC plugin for Firefox using Synaptic.
