### Post by mb_webguy on 2008-08-09
> Is there a way to make Amarok use the icon set I have established for my system rather than the KDE defaults? I haven't got KDE and prefer the loom of GNOME and would like Amarok to match.

The short answer is "no".  While Amarok will run just fine under Gnome, it's a KDE application and KDE doesn't use the same icon sets as Gnome.

The long answer is "maybe".  Since KDE and Gnome can both be installed at the same time, you *might* be able to install the kubuntu-desktop package, login with the KDE desktop, and change the appearance of Amarok there.  While you still might not be able to use the Gnome icon set, you might be able to come close.
