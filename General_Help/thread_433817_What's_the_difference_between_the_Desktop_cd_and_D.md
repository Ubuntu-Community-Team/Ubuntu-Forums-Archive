### Post by aysiu on 2007-05-05
The DVD has the entire Main and Restricted repositories (including Xubuntu, Kubuntu, and Edubuntu). The CD has an image of the live session, which can be installed to your hard drive, and then a very small set of repositories (*build-essential*, basically).

If you don't know what Main and Restricted are, read this:
[http://www.ubuntu.com/community/ubuntustory/components](http://www.ubuntu.com/community/ubuntustory/components)
