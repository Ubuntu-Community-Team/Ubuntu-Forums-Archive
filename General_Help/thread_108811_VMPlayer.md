### Post by anil_robo on 2005-12-27
NO! You cannot use your existing installation of windows with vmware. You have to have a vmware "virtual machine" for windows. While Linux virtual machines are available freely, the case is not so with windows because of the perpetual problem of $$$. You can make your own virtual machine of your copy of windows using vmware workstation. Then you can use that virtual machine with vmware player after your vmware workstation expires.

See some screenies [here!]("http://www.ubuntuforums.org/showthread.php?t=107766#4")
