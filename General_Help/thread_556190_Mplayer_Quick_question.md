### Post by extracted on 2007-09-21
In gnome, how do you set mplayer to be the default media player ?

---

### Post by wheredidrealitygo on 2007-09-21
One easy way is to do it to each file extension.

Go to the movie/video you want, [example.avi]

right click it, select properties, and go to the "open with" tab.

then choose mplayer movie player from this list.

then it will play all .avi files by default with mplayer, you may have to do it with each multimedia type though.
