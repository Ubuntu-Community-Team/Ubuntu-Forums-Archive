Both Lubuntu and Xubuntu are faster than Windows and can be run on lower-specification machines. Try those?

Xubuntu runs the XFCE desktop environment, meant for lower-spec machines. [http://www.xubuntu.org/](http://www.xubuntu.org/)
Lubuntu runs the lightweight X11 Desktop environment. Takes up less space and probably runs a little faster than the above. [http://lubuntu.net/](http://lubuntu.net/)

EDIT: Well, Ubuntu itself is faster than Windows, too, but you need something even more lightweight.

### Post by WorMzy on 2010-07-02
I'd personally install Arch and install XFCE or one of the other light/middleweight desktop managers. From there you can install Chrome, VLC and the torrent client of your choice, and you won't have any unnecessary applications and dependencies that *buntu ships with.

It's not for the faint hearted though, a fresh install of Arch gives you a command line and that's it. You need to configure X and install a desktop environment yourself.
