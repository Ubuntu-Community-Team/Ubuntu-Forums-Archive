*EDIT*
Problem solved:
for anyone else haveing a similar problem, here is what I did to fix it.

Press ctrl+alt+F1 to go to the text prompt when at the blank screen
Use a program called gtf
It should already be installed on Ubuntu
I ran this command:
gtf 1024 768 75
for 1024x768@75Hz
it gives you a line to put in xorg.conf
edit /etc/X11/xorg.conf

these are the final edited parts of my xorg.conf

Section "Monitor"
	Identifier	"Generic Monitor"
	Option		"DPMS"
	HorizSync	30-69	
	VertRefresh	50-120
	Modline  "1024x768_75.00"  81.80 1024 1080 1192 1360  768 769 772 802
	-HSync +Vsync
EndSection

	SubSection "Display"
		Depth		24
		Modes		"1024x768"
	EndSubSection

Hopefully you can figure out where those fit in the config file. The Modline part was the gtf output.
