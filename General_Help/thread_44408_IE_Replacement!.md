### Post by bored2k on 2005-06-26
You can install Internet Explorer through wine.

1. [http://www.ubuntuguide.org/#extrarepositories](http://www.ubuntuguide.org/#extrarepositories)
2. [http://www.winehq.com/site/download-deb](http://www.winehq.com/site/download-deb)
3. Using Synaptic package manager (step 2), search and install wine and anything that pops up (winesetup, winetools or something, etc).
4. [http://ubuntuforums.org/showthread.php?t=29996](http://ubuntuforums.org/showthread.php?t=29996) (winetools: how to install).
5. [http://ubuntuforums.org/showthread.php?t=35513&highlight=winetools](http://ubuntuforums.org/showthread.php?t=35513&highlight=winetools)

The easiest way is to get Crossover Office, but that is a non free application.
[http://www.codeweavers.com/products/cxoffice/](http://www.codeweavers.com/products/cxoffice/)
