Alright I update firefox on edgy to version 2.0.0.3+0dfsg-0ubuntu0.6.10, which is today's release
and it launched without a hitch.

What you should probably do is open up a terminal window.  And run:

```
firefox
```

From there.

This will allow you to see if any error messages are occuring while it starts.
I've had issues before with firefox extensions bombing out while launching
and firefox would just not start.

You may also want to check if any other instances of firefox are stuck running
from before the update:

```
pidof firefox
```

If this returns anything there's a copy of firefox running, which could cause
issues when starting a new instance after upgrade.

To close any running a simple:

```
killall firefox
```

usually does the trick.
