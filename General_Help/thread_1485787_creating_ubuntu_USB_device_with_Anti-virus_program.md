### Post by clgy15 on 2010-05-17
Well if you make a Ubuntu USB stick with a Persistance file (Using Start-up Disk Creator, its the section saying how much storage to save your files etc.) then you can just install an AntiVirus program like normal.

However keep in mind that itll most likely only scan the thumbdrive because most Anti Virus programs are designed for the OS thats currently working.

My suggestion is to use BitDefender Live AntiVirus Disk. Its designed to be able to search for viruses without booting the hard disk. Heres the link [http://download.bitdefender.com/rescue_cd/](http://download.bitdefender.com/rescue_cd/)
 
And if you dont want to use BitDefender here a Link to a good list of Antivirus live CDs
[http://www.techmixer.com/free-bootable-antivirus-rescue-cds-download-list/](http://www.techmixer.com/free-bootable-antivirus-rescue-cds-download-list/)
