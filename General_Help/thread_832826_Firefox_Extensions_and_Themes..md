### Post by RSingh on 2008-06-18
It can be a problem that the theme is not compatible with firefox 3. If it is compatible then maybe your profile is corrupted. You can try creating a new profile but that will remove all of your bookmarks and addons, so go with this only if you are sure:

Press Alt-F2
type:
```

firefox -profilemanager
```

delete your old profile and create a new default one.
