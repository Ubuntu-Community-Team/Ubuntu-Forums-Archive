### Post by cmnorton on 2008-07-14
The problem with connecting and reading from Microsoft Access databases is making sure you are creating a cursor with the right kind of record set. Equally so, Access seems to be picky on how to connect to a foreign database. 

The point to which I am trying to lead you, is you can probably find a lot of open source sql admin tools. You have to make sure those tools can deal with Access' record set types.
