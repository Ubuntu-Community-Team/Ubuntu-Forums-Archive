### Post by rpaster1 on 2011-11-03
I have a problem and hope someone here can help. First some background. I had been using a dual boot Ubuntu 11.04 (with Windows XP), from wubi and then grub, with no problems and upgraded to 11.10 in mid October. After the upgrade to 11.10, I could not get to the login in screen. I would get the wubi selection screen ( Windows Xp or Ubuntu), then the grub screen ( with Ubuntu options and Windows XP, and then a blank purple screen. I found a solution here (pressing alt+tab) and then the login screen would appear. After logging in I changed the video driver to Nvidia's restricted driver (version 96, I think) so I could try Unity with my video card (Nvidia Geforce PCX 5300). I tried Unity 3D from the login screen, but could not get it to work, so I selected Unity 2d from the login screen and it worked fine. I then switched back to the Gnome classic interface from the login screen and it worked fine also. So I was happy as a clam.
However, about a week ago, after I manually ran an update, I can not login in again. Basically I get the wubi selection screen (Windows Xp or Ubuntu), then the grub screen (with Ubuntu options and Windows XP), and then a blank purple screen. I've tried several keystrokes (alt+tab, alt+F2, alt+f3) but nothing happens. Finally, I press ctrl+alt+del and the login screen appears and then the computer reboots. If I select Ubuntu again, the same thing starts over again.
Any help would be greatly appreciated.
