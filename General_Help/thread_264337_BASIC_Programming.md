### Post by RHTopics on 2006-09-24
Cath0de,

Here are a couple of options for you to consider.

There is a BASIC programming dialect and environment included in the default installation of Ubuntu.  It is in OpenOffice.org as part of its macro facility.

The online help in OpenOffice.org does a fairly decent job of explaining the syntax and what you need to know to get started.

The [www.oooforum.org](www.oooforum.org) forum has a sub-forum dedicated to programming in OpenOffice.org.  I have found the forum members to be helpful.

Although I have not installed it in Ubuntu yet, another option is "gambas", which is available in the universe repository.  Its integrated development environment has good online help.  It also provides useful example programs.  "gambas" is a good open source alternative for those that have used Microsoft's Visual Basic.

> I'm interested in learning BASIC, but don't know much about modern dialects and such. Anyone have any good reccomendations for a good BASIC dialect and tutorial?

I recommend learning python.  Here's why:

1. It's very similar to the old BASIC you're familiar with.

2. It's an interpreted language, meaning that you can type commands into the command line and immediately see what happens, just like the old BASIC days.  

3. It's very simple to dive in and start doing useful things in minutes.  

4. It has tons of documentation online.

5. After you get through the initial learning curve, which is not steep, you can do very powerful things with it.

6. It's already installed on your Ubuntu machine, even if you don't know that it's there.

7. Many of the things that you should learn to do in unix/linux/ubuntu can be done with python, and a lot of them already are.  It's very common to find python scripts all over your linux distro, and once you know a bit of python, you will be constantly getting that warm fuzzy feeling when you see a .py file.

8. It's fun, just like BASIC.

Just my thoughts.

Oh yeah - start here:
[http://www.python.org/](http://www.python.org/)

### Post by RHTopics on 2006-09-25
Cath0de,

My first computer was a Commodore Vic 20 which booted up into a BASIC environment.  So I believe I know what you are stating about your experience using BASIC.  Things have significantly changed in many ways since those days.

Depending on your learning goals you may want to explore programming languages other than dialects of Basic.

If you are interested in learning how to program without a particular project in mind, then I concur with **rcmiv** about learning python.  Ubuntu includes a python programming tutorial in the default installation.  It is called "Dive into Python" and can be found clicking System > Help > Applications (under Other document categories) > Programming.

Other good python guides and tutorials to get you started can be found at

[www.ibiblio.org/obp/thinkCSpy/](www.ibiblio.org/obp/thinkCSpy/)   - Although it is out of date, it is a good guide for someone learning how to program.

[www.byteofpython.info](www.byteofpython.info) - This provides a good understanding of python and complements "Dive into Python" rather well.

On the other hand if automating common tasks that you do with a spreadsheet or word processor appeals to you, then learning the Macro programming environment in OpenOffice.org will be time well spent.

And, an added bonus, the Macro environment in OpenOffice.org also supports programs written in python.
