### Post by tea for one on 2009-06-03
The iso created by Remastersys is a "live" system and it also contains the Ubuntu installation software.

I suggest that you burn the image to a CD
Then test your OS as a live system
Subsequently install on a new partition 

I.e. If there is a problem, then you have not lost the original data from where you created the remastered OS
