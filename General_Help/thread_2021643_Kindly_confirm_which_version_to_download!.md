### Post by angryfirelord on 2012-07-09
i386 in the Ubuntu world means 32-bit. It's a reference to the 386 processor, which was the first 32-bit chip. You'll want to download the i386 desktop torrent.

Alternate is Ubuntu, but with a text mode installer. Server is similar to the alternate install except that it ships utilities for server (such as a slightly different kernel and the ability to install a web server from the CD).
