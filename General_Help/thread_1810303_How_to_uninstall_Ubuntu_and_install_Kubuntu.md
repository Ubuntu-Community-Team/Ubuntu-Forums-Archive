### Post by 2F4U on 2011-07-23
It is usually necessary to hit one of the Fx keys shortly after the BIOS message shows to get a temporary boot device list. Common Fx keys are F11, F12 and F2, but it depends on your computer and you can verify it in the computer's manual.
Another option would be to change the boot order permanently in the BIOS, so that it first attempts to boot from CD and if there is no CD in the drive, boot from HDD.
