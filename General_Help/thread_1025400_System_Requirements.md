### Post by KeilanS on 2008-12-30
I installed Ubuntu (not Xubuntu) on a 600MHz PIII with 256MB of RAM and it ran fine (very slowly, but fine). So that isn't your problem.

I'm new to Ubuntu, but my suggestions would be:
1) Did you MD5SUM the iso image that you used to burn the CD? If not, learn how to do so [here]("https://help.ubuntu.com/community/HowToMD5SUM").
2) Are you sure the computer is booting from CD properly? Like did you go into the Bios and set it to boot from the CD Drive?

If you did both of those, then you'll have to wait for someone with more experience.

Good luck,
