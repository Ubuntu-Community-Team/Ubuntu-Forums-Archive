### Post by Vajra Vrtti on 2007-07-13
You can boot directly from the Ubuntu CD and have a functional OS, so that you can see what Ubuntu looks like. That will not change your system anyway. Having done that, you will have an icon in the desktop to install Ubuntu in your HD. If you choose to do that, you will get a dual boot system: you will have a new screen at boot to choose the OS you want to use, Ubuntu or XP.

The *iso* file must be burned as an image, not a backup. In Nero, which I know, you do that selecting the last option in the main menu (*Open project or disk image*, or something like that).

I recommend you to download the latest Ubuntu **7.04**.
