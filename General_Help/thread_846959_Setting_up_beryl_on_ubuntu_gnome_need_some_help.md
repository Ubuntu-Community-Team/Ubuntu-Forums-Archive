### Post by Jive Turkey on 2008-07-02
Beryl is obsolete now, If you are using Gutsy or Hardy versions of Ubuntu it already has Compiz (the replacement for Beryl) installed.  Search around the forums or the web, there are zillions of tutorials for compiz that will walk you along without much difficulty.  If you need to, you should also read up on using the terminal and terminal commands so you can follow along.

### Post by kdcoetzee on 2008-07-02
For more settings on compiz-fusion install the "Advanced desktop effect settings" tool. you can tweak compiz-fusion to your liking. 


To install type in your terminal:

```
sudo apt-get install compizconfig-settings-manager
```
