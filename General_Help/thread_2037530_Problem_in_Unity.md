### Post by Dylan1473 on 2012-08-04
There are numerous other desktop environments you can use if you so choose if you just feel Unity is buggy in general. I'd recommend [XFCE]("http://www.xfce.org/") (in Xubuntu) if you want to stick with one of the bigger ones for its customizability and efficiency or _[Enlightenment]("http://www.enlightenment.org/")_ if you don't mind it being in development and having problems occasionally because it is really pretty and cool (and also extremely customizable).

Alternatively, if you just want to figure out this one specific issue in Unity, try opening up the Gnome System Monitor next time this happens and check whether the program is still running and vanished somehow or if it just crashed.
