### Post by Vaphell on 2012-07-25
you never can be 100% sure but if js scripts were disabled then most likely nothing happened as something is required to initiate the process of infection, be it jscript or flash object - something that can run the malicious code.

Root access is not mandatory for exploits, less 'ambitious' malware can plant itself in user space with user privileges and be active only when the user is logged in. It's enough to leak user's private data and that is bad in itself.

browser history (if you don't purge it regularly) should have the address of that site
