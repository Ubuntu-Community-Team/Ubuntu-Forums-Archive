### Post by dje on 2008-08-01
how did you install it? it is preinstalled in Ubuntu Hardy + Gutsy (and Feisty). Go to System >> Preferences >> Appearance and click on the Visual effects tab or press Alt + F2 and run the following:
```
compiz --replace
```

hope that helps,
dje

### Post by tuxxy on 2008-08-01
Download emerald from synaptic, navigate to

System > Prefs > Sessions



```

```

Add that command to your sessions here and remove the need to run it at ever bootup.  Now you will have compiz windows too :)
