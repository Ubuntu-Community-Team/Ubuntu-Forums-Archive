Actually the panel clock does indeed support some code for the layout, but even though it's very similar to HTML/CSS, it actually is Pango markup. Which definitely doesn't include any blinking...

[http://library.gnome.org/devel/pango/stable/PangoMarkupFormat.html]("http://library.gnome.org/devel/pango/stable/PangoMarkupFormat.html")
