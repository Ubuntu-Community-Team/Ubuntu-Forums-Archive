### Post by Dennis N on 2022-07-14
Gnome calls that the "Dash". It appears on the Activities Screen if you disable the default Ubuntu Dock (which you must have done). But then, there is no dock at all on the Desktop. You can't move the Dash. 

If you don't want to use Ubuntu Dock and want a dock similar to the Dash  on the desktop, you use the "Dash to Dock" gnome shell extension.
