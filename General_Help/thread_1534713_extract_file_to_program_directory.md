### Post by watupgroupie on 2010-07-19
Those folders are hidden by default in your home folder. To view them while using nautilus hit ctrl+h. 

I assume you're trying to extract from an archive. Just use the default file roller program and hit the extract button at the top. Navigate to your directory and you're good to go. There are also extra archive formats for installation like .rar in synaptic.
