### Post by TristanMike on 2006-03-21
In Kopete select Settings -> Configure Kopete, and under "Behavior" make sure there is a check in the "Show system tray icon" box"

In amaroK it's Settings ->Configure amaroK  Then under "General" make sure there is a check box in "Show tray icon"

In Gaim, select Tools ->Preferences, and it's a plugin called "System Tray Icon" make sure you have it installed and checked.
