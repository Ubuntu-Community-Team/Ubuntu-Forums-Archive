### Post by aldenhg on 2007-08-14
If you didn't Shift+Delete the files or rm them from the command line, they should still be in the Trash folder which can be accessed from the trashcan icon on one of your system panels.
Otherwise, they're more or less gone unless you can use a file recovery tool. Unfortunately I don't know what those tools are named in Linux.
