### Post by nikoPadilla on 2011-06-08
Hello,

You might want to try looking at your Keyboard Preferences.  Under Mouse Keys, see if you have the setting "Pointer can be controlled using the keypad."; if it is turn it off.  This may solve that keypad problem.
