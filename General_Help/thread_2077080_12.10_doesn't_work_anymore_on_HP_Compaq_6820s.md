### Post by Mark Phelps on 2012-11-05
> **alainhenry said:**
> ... The driver for the ATI Radeon RV515 is installed, but I assume I should be able to reactivate it. 

IF you're talking about the restricted driver -- NOT in Ubuntu 12.10, you won't.

Ubuntu 12.10 uses the new version of the X-server 1.13 -- and the only AMD restricted drivers that will work with that are the newest ones for the  HD 5x/6x/7x chipsets.

Yours is an older chipset, and while it MIGHT still work in 12.10 with the open-source driver, if it really is an x1300pro, it might NOT work there either.

AMD dropped restricted driver support for the x13xx series years ago.
