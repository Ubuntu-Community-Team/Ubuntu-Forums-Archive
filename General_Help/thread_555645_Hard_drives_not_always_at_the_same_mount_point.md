### Post by jocko on 2007-09-20
Try using uuid entries instead of /dev/sdxY entries for your disks.

To find the uuid's for your disks, check the output of this command in a terminal:

```
ls -l /dev/disk/by-uuid
```It will give you a line looking roughly like this for every disk:
```
lrwxrwxrwx 1 root root 10 2007-09-19 17:59 [COLOR=Red]f394b2f2-4d6e-40cc-a64a-9ffe04b63935[/COLOR] -> [COLOR=Red][COLOR=Black]../../[/COLOR]sdd1[/COLOR]
```In this example you should change this line in fstab:
```
[COLOR=Red]/dev/sdd1[/COLOR] /media/DRIVE_ONE ntfs-3g defaults,locale=en_US.UTF-8 0 0
```to:
```
[COLOR=Red]UUID=f394b2f2-4d6e-40cc-a64a-9ffe04b63935[/COLOR] /media/DRIVE_ONE ntfs-3g defaults,locale=en_US.UTF-8 0 0
```
