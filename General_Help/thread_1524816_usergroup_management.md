### Post by sisco311 on 2010-07-05
> **jeidern said:**
> My user/group manager in the system>administration menu is missing. 


Right click on the menu -> Edit Menus -> System -> Administration and make sure that the *Users and Groups* item's checkbox is selected.

> **jeidern said:**
> 
Is there a command to get to the user/group manage using alt+f2 in the desktop? 


```
users-admin
```


> **jeidern said:**
> 
Or can i download the user/group manager from Ubuntu Software Center?


gnome-system-tools
