### Post by QIII on 2009-06-27
Wow.  Lot's of questions!  We've all been there.

I'll try to answer a few.

My pc RAM is 223MB, will that be good enough?

-- "Good enough" is relative.  "P" for plenty is desirable.  The "official" answer is that 256MB is the minimum, but I have run Xubuntu with less.  You sure won't be smoking the tires.

I usually use Microsoft Word because I am a writer. Is that available? I have seen somewhere Excel in one of the screen shoot where it was opened.

-- It may run in a "sandbox" like Wine.  I've never used Wine, but there are a lot of posts about it.  If you are not particular, OpenOffice.org's Word Processor is almost passably compatible with Word documents.  I've gone back and forth with documents between Word and OOo.  There are some formatting changes that happen each time.  Irritation.  Excel is the same deal with regard to Wine.  Somewhere there is a website that rates how well different MS programs work in Wine.

Adobe Photoshop is really important to me. I see that installing it asks platform (Windows or what). It would go in installation like with Windows?

-- Haven't a clue.  Someone else will have to give you the details.

How do I install programs? (I know how to but I am not sure is XUbuntu installing everything I need.) There always are platforms sort of Windows and that and never seen XUbuntu.

-- There are many ways to add programs.  Ubuntu (Debian) has what are known as "repositories" that contain all sorts of programs from soup to nuts.  You can access them from what is known as the Synaptic Package Manager,  from Add/Remove under the Applications button on the Gnome panels, via the terminal (command line in Windows) ... Some proprietary software can be downloaded and installed per the instructions on web sites.

I know the last one didn't make any sense at all. But I need to know. And I need to do something fast or I don't know.

-- You can get a copy of the .iso file and burn yourself what is known as a LiveCD that you can run from your CD drive to see if you like Ubuntu.

Ah, and what is with desktop? I've seen in google image the way it looks and is it customable? Like a desktop image and the way that it looks so it would remind a little of Windows XP style?

-- The desktop is infinitely customizable.  There are "themes" that make everything look almost the same as a happy little XP machine.

Ok, thanks if someone would answer me. I really hope that it's ok from me to ask here. I figured here it would be faster than in any other place.

Oh, and instalation... it's similar to windows way?

-- Installation from the LiveCD is very simple and straightforward.  Easier than Windows.  There is one spot that might be tricky for newcomers, and that is the spot where you have to "partition" your drive.  If you are getting rid of Windows entirely, I'd just select "use entire disk" the first time around.

EDIT:  With that amount of RAM, the LiveCD may not work either to run as a live session or to install.  I'd use the "Alternate" disk.

if Xubuntu is anything like Ubuntu for installing programs there are a few ways to install programs. one is through 'Add/Remove' from your 'Start' menu. Another is from the Synaptic Package Manager (a more advanced version). Both these methods use the repositories for a listing of free programs to install. There are two last 'manual' ways to install programs. one is by using a .deb file (like a windows installer but for linux). the other is by 'make, make install' (but you only really have to worry about that if you are installing programs/addons you probably dont need anyway)

There is also WINE (a Windoze 'Emulator' - but don't call it that on the WINE forums). this allows you to install Windoze programs under Linux and run them as if your computer were a Windoze computer. before doing this, however, do some looking and make sure there isn't a Linux version of the program you are trying to run (or an equivalent program).
