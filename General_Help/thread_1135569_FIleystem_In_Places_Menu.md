### Post by mb_webguy on 2009-04-24
This is just a guess, but I'd say it was removed intentionally.  Ubuntu is designed to be the desktop Linux everyone can use.  Basic Ubuntu users don't really need to go messing with anything outside of their home directory any more than basic Windows users really need to mess around with Windows system files (which is why Vista's UAC protects those folders and prevents any changes to them without explicit consent from the user).  Users can still access the filesystem when necessary -- in fact it still appears in the Places sidebar in Nautilus.  But putting it in the Places menu is a waste of space for most users.
