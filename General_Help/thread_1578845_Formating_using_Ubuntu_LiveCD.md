### Post by sikander3786 on 2010-09-21
> Ubuntu seems to format so fast compared to Windows. Why is it like that?

Not really I think. You've also got a Quick Format option in Windows which formats equally fast.

> I was also wandering that if I install Windows 7 on a HDD formated by Ubuntu will all of the previous data written by older OSes be fully removed?

To completely remove all traces, you will have to do low level format I think. The data is there when you delete it or format the drive, only the block are marked empty meaning new files can be written to them. And then they over-write the previous ones. Someone correct me if I am wrong.

You can install Windows on a partition formated in Ubuntu if it is formatted as NTFS (FAT for older versions) otherwise you'll have to reformat it in Windows and the case is the same as described above.

Happy Ubuntu-ing to you and your friend.
