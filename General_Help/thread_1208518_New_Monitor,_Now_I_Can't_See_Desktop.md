### Post by beastrace91 on 2009-07-09
Can you get to a terminal login in Ubuntu (press control+alt+f1)? If you can log into the terminal and run the following: ```
sudo dpkg-reconfigure -phigh xserver-xorg
```

That should default your xorg.conf to work with your hardware.

~Jeff
