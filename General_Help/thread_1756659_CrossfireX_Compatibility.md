Firstcheck this website [http://appdb.winehq.org/](http://appdb.winehq.org/)  to see if wine can run crossfire.
If it can, install Wine (do you already have it installed?)
find it in the software center.

now, open a terminal and change to the directory the crossfire.exe file is in.

for e.g.  if the file is in a folder named 'crossfire' that resides in the drive named 'games'.   ```
cd /media/games/crossfire 
```
Then in the terminal, run ```
wine name_of_the_file.exe
```
