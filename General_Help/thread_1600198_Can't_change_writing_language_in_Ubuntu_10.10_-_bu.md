### Post by Nordvind on 2010-10-18
Set up new Ubuntu 10.10 and it's all fine but the bug I encounter from time to time. Out of the blue, I can't switch languages, neither by keyboard shortcuts, nor from the panel icon. It happens in random time, and it must be some 10.10 bug, for it never happened on 10.04.
Any ideas what might it be, and how to cure it? :mad:

---

### Post by vangelas on 2010-10-19
if you go (from the configuration editor) to "/apps/gnome_settings_daemon/plugins/keyboard", and select active in keyboard, it should be fixed.

a similar problem is discussed here: [http://ubuntu-ky.ubuntuforums.org/showthread.php?p=9822228](http://ubuntu-ky.ubuntuforums.org/showthread.php?p=9822228)

hope this helps:)
