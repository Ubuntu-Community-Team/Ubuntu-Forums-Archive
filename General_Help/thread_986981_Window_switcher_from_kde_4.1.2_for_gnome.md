### Post by prshah on 2008-11-19
> **MasterNetra said:**
> I was wondering what package(s) would i be looking for to get that kde4 window switcher (which is similar to vista's) so i can install it for use in gnome?

If you mean similar to Vista's Aero interface, then you can find it in Compiz; System-Preferences-CompizConfig Settings Manager-Window Management-Shift Switcher. Enable it, then use Windows+Tab.

Post back if you have difficulties (ex, cannot find CompizConfig Settings Manager option)
