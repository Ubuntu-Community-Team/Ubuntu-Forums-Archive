### Post by calraith on 2007-03-06
I've noticed a few threads for people asking for assistance with setting up their eggdrop bots.  They can apt-get install eggdrop, but don't really know where to go from there.  So I wrote a bash script that automates the initial setup for you, and can even set a few default values in your conf file.  Enjoy!

Download the file "eggdrop-install.sh" to your home directory.  Launch a terminal window and type the following:
```
chmod 755 eggdrop-install.sh
./eggdrop-install.sh
```
