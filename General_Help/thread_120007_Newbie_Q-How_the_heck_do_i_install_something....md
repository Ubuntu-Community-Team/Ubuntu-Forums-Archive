### Post by carlosqueso on 2006-01-20
A couple of things that may help 

1) You need to type anything in that off-colored box into a terminal, which you access by going to Applications->Accessories->Terminal.

2) sudo gives you administrator privliges for that command, you will need to enter your password the first time (and maybe again if you do this slowly).  Don't worry that nothing appears to show up, linux just doesn't display anything when you type a password.

If you have any more questions, feel free to ask them and we'll be happy to help.
