1) The alternate CD does NOT have a GUI (it's only a command-line, and you'll have to install EVERYTHING manually). I don't suggest the alternate CD, especially if you are are not confident with using the command line.

2) No video, but read this through: [https://help.ubuntu.com/community/UsingTheTerminal](https://help.ubuntu.com/community/UsingTheTerminal)

3) Ubuntu does not use the EXE and DLL files that Windows uses. Because of this reason, you can't run Windows programs on Ubuntu natively. If you want to use Windows programs on Ubuntu, try using WINE ([http://www.winehq.org/](http://www.winehq.org/)). Note that WINE will very likely require tweaking and using the command-line, and it might be a little too hard for beginners. Also note that WINE does not perfectly run Windows apps. You may find many glitches :)

Enjoy Ubuntu (or Linux). I know it will be **very** very hard at first, but you'll get it one day. Remember to be patient, and always try to learn things.. ;)

### Post by Boondoklife on 2009-05-26
Is there something in particular that you are looking to run via the command line that you think you will need help with or is it just you want to know how to use the command line? If it is the latter then you could try [youtube]("http://www.youtube.com/results?search_type=&search_query=linux+command+line+tutorial&aq=3&oq=linux+comma").

As far as windows apps go, there are plenty of linux apps that can do just as well if not better than the mainstream windows apps, but if you find that you can not find one then by all means wine is a great way to run windows apps in linux.

Wine does not work with every application, in the few cases where it does not then you may want to look into something like a virtual machine. I used vmware to run MS Visual Studio during a programming class and it worked flawlessly.
