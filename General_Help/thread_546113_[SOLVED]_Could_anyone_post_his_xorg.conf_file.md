### Post by aysiu on 2007-09-08
Other people posting their /etc/X11/xorg.conf files isn't going to help you. We all have different hardware from you.

Go to [the terminal](http://www.psychocats.net/ubuntu/terminal) and paste in this command: ```
sudo dpkg-reconfigure xserver-xorg
``` and answer the questions as best you can. When in doubt, go with the default answer.

To get a fresh sources.list, go [here](http://www.psychocats.net/ubuntu/sources).

### Post by aysiu on 2007-09-08
> **mikecomua said:**
> nO, i see the boot screen and Ubuntu is supposed to load, but then an xserver error pops up and that's it.
After the X server error pops up, can you hit *Enter* until you get to a command prompt? If so, at the command prompt, type ```
sudo dpkg-reconfigure xserver-xorg
``` and answer the questions. Afterwards, type ```
sudo /etc/init.d/gdm restart
```
