### Post by bville09 on 2010-05-12
The ubuntu-restricted-extras package does fix this problem, but it's restricted by adobe's patents (hence the name of the package) in some countries. I haven't tried the fluendo codecs (or even heard about them up until this post).

You can install the ubuntu-restriced-extras package, by opening the Ubuntu Software Center in the "Applications" menu. When it opens, do a search for:

ubuntu-restricted-extras

...or even just "restricted" and install it from there. Although, I'm pretty sure it would tell you that you need to have flash installed if you didn't already have the codec.

Edit: Yeah, it would tell you if you didn't already have them, rather than just show a black square, it would display a red error message. I don't have any other ideas, then. Sorry.
