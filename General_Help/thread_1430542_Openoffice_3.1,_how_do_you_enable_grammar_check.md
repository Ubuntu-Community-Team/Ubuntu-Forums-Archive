### Post by rainbowagent7 on 2010-03-15
I have openoffice 3.1 and use it a lot for school papers. It has an option for a spelling and grammar check, I can use the spelling check with no problems, but I cannot figure out how to get it to check grammar. Is there an option to enable this somewhere? Any help would be greatly appreciated because I am forced to check grammar manually and find out if there are mistakes after my papers are turned in, which there usually is! Thanks in advance.

---

### Post by Tom Collier on 2010-03-15
In OOWriter, go to Tools>Options>Language Settings>Writing Aids and make sure that the "check grammar as you type" box is checked.
