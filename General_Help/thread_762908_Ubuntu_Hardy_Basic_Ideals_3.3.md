### Post by Ub1476 on 2008-04-22
At the moment, this theme can't be created in Gnome. Maybe the pixmap engine and Macslows cairo hacking could accomplish something, but not far stable or fast enough for Ubuntu standard yet. 

This theme is just a mockup by Ubuntus artist, I believe. And yes, it's quite an interesting theme if you can stand all the brown:) 

Hopefully there'll be more than one new theme for Intrepid Ibex though. So yes, they will at least make a new theme for Ubuntu 8.10. This includes, GTK2 theme, Icons, Window border, Background, GDM theme, Splash, Sounds etc. Hopefully some better integration with Compiz too. 

I suggest you check out the developing and art & design forums for updates of the theme for Intrepid Ibex.
