### Post by dresnu on 2006-02-09
Right click on Desktop-> Create New-> Link to Application

Under the General tab insert the name of the application and select an icon if you wish. Then under the Application tab insert the full path of the command you want to run in the Command textbox. Click OK. Then right click on the icon you 've just created, select properties and under the Permissions tab check "Is executable". Click OK and that should be it :)
