### Post by TheFu on 2021-11-01
```
sudo ubuntu-drivers autoinstall
```

should install nvidia drivers for the actual hardware, if they are supported by nvidia still.
Regardless, the command above should install the correct drivers for the GPU for any supported LTS release of Ubuntu. It knows about nvidia, AMD and intel.  The last two are more tied to the kernel.
