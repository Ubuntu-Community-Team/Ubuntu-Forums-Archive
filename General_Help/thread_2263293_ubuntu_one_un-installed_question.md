### Post by grahammechanical on 2015-01-31
It used to be called Single Sign On. The idea was to have one account that would allow us to sign on to several Ubuntu web sites such as Ubuntu forums, AskUbuntu, Discourse Ubuntu and Launchpad as well as our Ubuntu One cloud service account.

Then, when the Ubuntu One cloud service was discontinued the label "Ubuntu One" was used instead of Ubuntu Single Sign On. If we buy a Ubuntu phone we will find that we need a Ubuntu One username and password to download apps from the apps store. The user name and password that we use to access the Ubuntu forums will work also when accessing the Ubuntu phone app store. We can also use our Ubuntu One sign on to register for attending the Ubuntu Online Summit. That is the benefit of having a Single Sign On service.
