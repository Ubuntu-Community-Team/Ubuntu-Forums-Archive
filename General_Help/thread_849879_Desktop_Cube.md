### Post by PinkFloyd102489 on 2008-07-05
Install the package "ccsm" using Synaptic. Then go to System > Preferences > Advanced Desktop Effects Settings and select the Desktop Cube, Rotate Cube, and Viewport Switcher. You can either hold Ctrl+Alt+Left-click to rotate it or middle-click (mouse wheel) on the desktop to do so.
