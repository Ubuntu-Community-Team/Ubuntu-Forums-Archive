Found the solution after a lot of try and error. Rivendell, the large software package I mentioned brings in the newest version a monitoring application, rdmonitor. This one was responsible for this box.

See here for Details: [http://comments.gmane.org/gmane.comp.radio.rivendell.user/4184](http://comments.gmane.org/gmane.comp.radio.rivendell.user/4184)
