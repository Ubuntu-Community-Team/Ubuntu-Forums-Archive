### Post by rbmorse on 2009-03-10
Maybe not.  Sometime Memtest will report errors if the system is using memory timings that are inappropriate for the RAM modules or if the voltage supplied to the modules is incorrect. 

You need to obtain the specifications for the parts from the vendor and then verify the system BIOS settings are correct for the parts that are installed in your machine.  If you find any changes are necessary be sure to run MEMTEST again afterwards to confirm the problems have been resolved.
