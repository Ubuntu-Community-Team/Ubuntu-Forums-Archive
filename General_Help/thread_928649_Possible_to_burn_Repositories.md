### Post by jonian_g on 2008-09-24
You can go to synaptic and check for installation the packages you want. Then File>Create download script and a script with all the packages and its dependencies will be created. Run the script in a computer with internet access to download the packages.

There is an app too, called APTonCD.

Description:
> **Installation disc creator for packages downloaded via APT**  
APT removable repository creator and package backup tool for Debian based systems.
This tool will allow you to create a media (CD or DVD) to use to install software via APT
in a non-connected machine, as well upgrade and install the same set of softwares in several
machines with no need to re-download the packages again.
For more information, visit [http://aptoncd.sourceforge.net]("http://aptoncd.sourceforge.net")
