### Post by donkyhotay on 2008-04-25
I've run a 64bit system for quite some time now without any problems. It used to be difficult at times for some programs to run on 64bit (I've been using 64bit ubuntu since dapper drake). By the time feisty launched only one program gave me a problem trying to run on a 64bit system. It was a flash player which actually ran fine on a 64bit system in 32 bit compatibility mode but I had to install and run a 32bit browser in compatibility mode as well which was a pain. A short while after feisty launched there was an update which fixed it and I haven't noticed any 64bit related issues since then. I upgraded to hardy and my 64bit system is going just fine.
