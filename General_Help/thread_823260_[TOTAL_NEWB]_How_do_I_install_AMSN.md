### Post by Powerman2442 on 2008-06-09
> **Cruvenium said:**
> Type in where and how?

Sorry kind of ignored the whole "[TOTAL NEWB]" thing. :P 

Anyways, there are a few ways to install aMSN. The easiest way (in my opinion) is to go to **Applications** --> **Accessories** --> **Terminal**. Once you have terminal open type in **sudo apt-get update**, Enter... (might have the type you password than enter again) then, **sudo apt-get install amsn**. Hit enter again and wait for it to work its magic. Select **Y** (for yes to install), hit enter, and it will install.

Or you can open Synaptic Package Manager. **System** --> **Administration** --> **Synaptic Package Manager**. Once that program opens click "Search" and type in amsn. It should list 3 things. Right click the first one (amsn) and click "Mark for Installation". Select "Apply" and then close the program.

You will find aMSN under **Applications** --> **Internet** --> **aMSN**.

I'd read the guide ryanhaigh posted to. Good luck.
