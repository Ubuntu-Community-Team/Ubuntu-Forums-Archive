### Post by grahammechanical on 2014-11-27
The default browser in Ubuntu is Firefox.

Ubuntu Web Browser (Browser) comes from Ubuntu for phones and tablets. It is designed differently. Do not expect the same menu options that we get in Firefox or Chromium. Browser is a web browser for mobile devices. Ubuntu Web Browser is a very long way from being a replacement for Firefox or Chromium or any of the usual desktop web browsers. I do not think that it is meant to replace the usual web browsers.

The design of Browser has changed over the last few months. At the top right of the application window you may see an icon that looks like a stack of 3 or 4 lines. Click on that and menu will appear. One option will be All Tabs. That will reveal all the tabs that were opened during the session. another option will be New tab. That will allow the inputting of a URL to open another web site. The new tab, I think is the nearest to a History in Browser. The Star icon acts like a book mark.

We certainly do not get the same menu options with any of the Ubuntu Touch apps that we get with desktop applications. 

EDIT: The icon is three horizontally stacked lines and it is at the top right of the application window. The menu has History; Open Tabs; New Tab. When Browser is opened in a Unity 8 session or in a Ubuntu Desktop Next install the same menu has an added option of Share. The Star icon is to bookmark a page and that is about it for Ubuntu Web Browser.
