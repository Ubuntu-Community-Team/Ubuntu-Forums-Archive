### Post by merlinus on 2009-06-09
If you will only be using the hdd from linux, then ext3.  If you will be using it on any windows machine, then ntfs.

---

### Post by s3MA00RRNY on 2009-06-09
If you're only using it with Ubuntu (you have to be dead sure) then format it to ext3. Ext4 is still too new and buggy (crazy vanishing files). If you want to use it with Windows, then use Windows to format it to NTFS.
