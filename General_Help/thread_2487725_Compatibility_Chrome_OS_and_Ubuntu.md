### Post by TheFu on 2023-06-13
> **ctclewis said:**
> I'm trying to make a Chromebook work with a Promethean Smartboard that doesn't support drivers for Chrome OS but does support Linux(Ubuntu)-
can you tell me which (if any) Ubuntu OS versions might be compatible with a Chromebook running v113.0.5672.134 (64bit)?

Not all chromebooks can run anything other than ChromeOS, so the answer depends on the exact model.  Look up mrchromebox and see what he says about your specific model.

Many times, once you run any other OS besides ChromeOS, you'll never be allowed to load ChromeOS on that hardware again. Beware. I have 2 chromebooks that will never run ChromeOS again.
