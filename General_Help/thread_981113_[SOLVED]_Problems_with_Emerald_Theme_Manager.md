### Post by Idefix82 on 2008-11-13
You need to tell compiz to use emerald. Type in the terminal
```
emerald --replace
```

Alternatively, install the compiz-fusion-icon. When you start it you get an icon in the top right corner and by right clicking on it you can choose emerald to be your window decoration manager.
