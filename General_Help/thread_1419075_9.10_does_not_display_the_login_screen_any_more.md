### Post by dabl on 2010-03-01
There are a couple of possible causes:

1. Full filesystem.  If you can get to the command line, ```
df -h 
```will show how much free space you have.

2. Root permissions assigned to hidden X-related files (caused by running the X server in root mode).  Sometimes you can delete the hidden files ~/.ICEauthority and ~/.Xauthority and fix it.  You'll have to do that from the command line, at the user's login:

```
rm .ICEauthority
```

```
rm .Xauthority
```

Then restart the X server with ```
sudo service gdm start
```

(assuming this is a 9.10 system)
