### Post by reduxtion on 2010-02-28
I've been working with documents using both OpenOffice and Office and was wondering what format I should use for maximum compatability between both. I don't want to go to one or the other and have something pop-up that there might be formatting that is incompatible. Is RTF the way to go? ODT?

---

### Post by Chris_cur on 2010-02-28
It depends on how often you are using Microsoft.  I personally find it much easier to keep all my WIP ( Works In Progress) as an .odt format. This is because I work exclusively on Ubuntu. When I have a final draft I save a .doc copy because my colleagues are using Windows. 

I try to stay away from RTF because a lot of my page formatting seems to get a little messed up when bouncing between the different systems and I don't know why.  But, this is all just my personal preference.

I hope this has helped.
