### Post by LewisTM on 2011-11-08
Did you search the forums?
This [thread]("http://ubuntuforums.org/showthread.php?t=1873826") and many more can be found on the subject.

You will have to use Gigolo (included in xfce4-goodies) and install gvfs and gvfs-backends.
You can set Gigolo to start at login, minimized in the tray, while autoconnecting to shares.
Otherwise you can set it up through fstab entries but that is more challenging. Tutorials can easily be found.
