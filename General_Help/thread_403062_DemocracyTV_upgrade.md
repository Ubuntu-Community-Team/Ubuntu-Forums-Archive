### Post by kadallas on 2007-04-06
The upgrade from DemocaracyTV version 8 series to DemocracyTV version 9 series, for my edgy system came from the back port repository.  When I installed it, it hung up with an error message and therefore I reinstalled the old version.

I noticed on the Democracytv home page a link to download the current version and I followed the links to the install instructuns for Ubuntu (edgy) : [http://www.getdemocracy.com/downloads/ubuntu.php](http://www.getdemocracy.com/downloads/ubuntu.php) and I noticed there is a special repository.  I followed the instructions and everything worked fine.  The new version of Democracy came up just fine.   

The new version wanted to download a lot of older versions of podcasts so I paused them and stopped each download.  I also had to open all the podcasts in my podcast directory to get them to show in the 'my collections' section.  All this took just a few minutes.  Now everything works fine.  It looks like some nice new features have been added so I am glad I went to the trouble of upgrading. :)
