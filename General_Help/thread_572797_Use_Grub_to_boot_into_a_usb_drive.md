### Post by steveneddy on 2007-10-10
I think that is a BIOS issue and not Grub.

You would have to set the BIOS to look for a bootable USB drive upon initial startup.

If there is a bootable partition on the USB drive, it will boot from the drive and you won't even see Grub.
