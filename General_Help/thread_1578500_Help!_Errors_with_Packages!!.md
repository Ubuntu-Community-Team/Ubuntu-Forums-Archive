### Post by jeremy96 on 2010-09-20
I just installed Ubuntu 10.04.1 on my laptop. I can't install any packages from the software center!! When I want to install something, the button to install it says "Use this source" instead of "Install". What do I do?

---

### Post by afrowildo on 2010-09-20
You need to press **Use this source** in order to activate the repository in which that package resides. Once you do that, it will be replaced with the install button.
