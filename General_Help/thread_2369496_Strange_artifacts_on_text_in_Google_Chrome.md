### Post by ajgreeny on 2017-08-24
What graphics card does the Thinkpad have and which driver does it use?

If it is only Chrome that has the problem you can easily try two things.
[LIST=1]
[*]Try using a different theme in your system DE.
[*]Turn off hardware acceleration in Chrome preferences; I use chromium, not Chrome, but I assume that is possible.
[/LIST]
