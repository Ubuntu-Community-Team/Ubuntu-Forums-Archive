### Post by Mark_in_Hollywood on 2008-10-17
Dumb as this sounds, I had similar problems, but with 1 gig thumb drive. I took it to a windows xp computer and reformatted it. Worked w/o problem from then on. Also, in the file manager, do Ctrl-H and look for 

.trashes

delete it. Confirm the deletion (are we adults? do we ever want to discard something?)

close the file manager and unmount the drive.

Don't forget to mark this post solved if this does the trick, please.
