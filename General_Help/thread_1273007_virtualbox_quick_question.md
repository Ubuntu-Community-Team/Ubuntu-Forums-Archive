### Post by Darkwing-Duck on 2009-09-22
Short answer no. You can run 32 bit inside of 64 bit. You can also run a 64 bit guest with a 32 bit host. For more information on Virtual Box check this tutorial here.

[http://gwos.org/udsf/doku.php/software:virtualbox](http://gwos.org/udsf/doku.php/software:virtualbox)
