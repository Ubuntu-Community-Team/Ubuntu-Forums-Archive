### Post by charlie. on 2006-05-08
QUESTION: How easy is Ubuntu to set up as a web server?
ANSWER: Easier!

Right, now for some useful information.

Ubuntu is, IMHO, easier to use than Fedora. Don't, whatever you do, use a big-bang installer that 'installs and sets up all the apps u need in one go'. That may work in Windows, it may even work in Linux - until you have problems. If you set it up that way, you will never know how to fix it when it breaks - which it will.

Which Ubuntu release (or pre-release) are you using?

I recently setup a corporate Intranet (trial) server on Ubuntu Dapper Beta 2, in a production environment. It uses SSH, Apache2, MySql, PHP (with hand-compiled *mysqli* extension) and sFTP. When I do things like this in a corporate environment, I type notes out of habit. The notes for this project will be completed some time this week. I might be able to publish them somewhere that you can read them if you have not solved your problem by that time.
