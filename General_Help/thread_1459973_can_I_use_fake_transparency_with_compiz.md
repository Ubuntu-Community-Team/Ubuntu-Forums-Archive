### Post by theBlindSniper on 2010-11-18
I hate to revive old threads, but this might keep other people from wasting time.

The short answer is, I believe, NO.

I've tried searching for a solution to this for quite a while. I have not been able to find any settings in the CompizConfig Settings Manager that address this issue, so I am assuming that if you use Compiz, you're forced to have "real transparency" or "true transparency."

As far as I know, the only way you can have "fake transparency" or "false transparency" is to disable Compiz altogether, if that's the manager you're using. It's really frustrating, to be honest. With true transparency, I tend to get a lot of text under text in my terminals, which makes transparent windows an eye-sore instead of eye-candy.

Another note: with Compiz disabled, the fake transparency looks like it redraws the entire background of the terminal, so it creates this ugly lag effect when you move windows. Compiz... can't live with it, can't live without it.
