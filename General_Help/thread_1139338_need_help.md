### Post by mb_webguy on 2009-04-27
If you install Ubuntu over the Windows partition, all of the files on that partition are gone forever.  If you then use a Windows restore disk, Ubuntu will likewise be gone forever.

My suggestion would be to boot the Ubuntu LiveCD and mount the hard drive.  Now you can back up all of the files you want.  Since you're using Linux and booting from a LiveCD besides, you don't have to worry about the viruses at this point.  Once you've backed everything up that you want, you can then format the drive and install the OS of your choice.  Just to be safe, you should scan these files using ClamAV or some other anti-virus software before exposing them to a Windows computer.

If you decide to install Windows again, you might want to suggest to your brother that he get a good anti-virus application like AVG, use some other browser than IE (such as Firefox or Opera), and be careful about his P2P downloads.
