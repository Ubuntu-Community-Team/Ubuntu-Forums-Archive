### Post by earthpigg on 2009-12-08
hi,

[http://cdimage.ubuntu.com/ports/releases/9.10/release/](http://cdimage.ubuntu.com/ports/releases/9.10/release/)

includes an alternate install cd, that will give you a command line install.

modified directions from [here]("http://sites.google.com/site/masonux/home/notes-to-myself") that will get you a fairly basic openbox desktop:

```
apt-get install xorg openbox firefox synaptic empathy gdebi jockey-gtk wicd gdm
```

empathy is the 9.10 IM client.

gdebi lets you double click to install a .deb.

wicd is a basic network manager.

gdm is the standard ubuntu login screen.

jocket-gtk is the graphical 'hardware drivers' screen.
