### Post by Jguy on 2010-04-16
While on Windows, I used to use VMWare. It was light, it worked, it did what I needed it to do.

Ever since switching to Ubuntu, I've always used virtualBox. If you're simply looking for something to emulate ONE virtual machine for stuff like testing, or using Windows programs on a Linux host OS or non-"enterprise" solutions, then something likeweight like VMWare free or VirtualBox is for you.

I've found that XenServer is mainly good for corporate or multi-VM deployments. I've worked with HyperVM (sucked when I used it) and VZ (sucked when I used it) in enterprise applications (true hosting) and it seemed better fit for those types of applications.

Yes but it's two years out of date!

As a noob I managed to install VirtualBox 64bit add the guest additions for XP and it works fine. The only problem you may find is trying to run OEM versions of Windows because it will always try to activate. In other words after 30 days windows will stop working. You can get around this (search my posts) but it's a pain.

I like the way USB devices are handled and with the guest additions being able to re-size the VM's window or use fullscreen mode.

VirtualBox also has good documentation so that dummy like me can understand what's going on.

I have not tried any of the others so I am biased to VBox.

Have fun trying them out.
