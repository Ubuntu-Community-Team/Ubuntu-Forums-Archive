### Post by pqwoerituytrueiwoq on 2011-08-03
Why not set up a cron job?
```
crontab -e
```add this line
```
01 * * * * /etc/init.d/minecraft restart
```if it needs *root* access use *sudo* on [I]crontab -e
[/I]if you really want a sh script
```
#!/bin/sh
while [ true ]; do
     sleep 3600 
     /etc/init.d/minecraft restart
done
```
