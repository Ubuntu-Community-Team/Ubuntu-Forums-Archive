### Post by yabbadabbadont on 2007-06-25
In a terminal window:
```
dd if=/dev/fd0 of=floppy.img
```
Edit: You most likely don't need to use sudo since the default user is a member of the floppy group.

### Post by yabbadabbadont on 2007-06-25
In a terminal window:
```
sudo -i
modprobe loop             (don't worry if it tells you that loop was already loaded)
dd if=/dev/zero of=/tmp/floppy.img bs=512 count=2880
losetup /dev/loop0 /tmp/floppy.img
mkdosfs /dev/loop0
dosfsck /dev/loop0             (just to be sure everything is ok)
mount -t vfat -O uid=1000,gid=1000 /dev/loop0 /media/floppy0               (/media/floppy0 is usually already there)

```
Now you should be able to copy your files to the image.  When you are done you will need to umount the image and clean up the loop device.  To do that, in a terminal window, run:
```
sudo -i
sync
umount /media/floppy0
losetup -d /dev/loop0

```
Now your image, /tmp/floppy.img, should be ready for use with Virtualbox.
