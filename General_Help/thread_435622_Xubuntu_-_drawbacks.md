### Post by mrazster on 2007-05-07
Well...I'm running it hand have been on both Breezy and Dapper..before Fiesty. The way I see it, the only drawback would be....that there aren't as many gui configuration tools/utilitys in xfce as there are in gnome e.t.c.
How ever some of them will work in both xfce and gnome.
And also if you like tinkering with themes, icons e.t.c ...then you'll Gtk and xfwm is the only thing you have to your disposal.

Other than that....I don't see any real drawbacks with xubuntu.

I agree to some extent with **rai4shu2**...therefore I have done an CLI/server install...then added all my packages manually thru ***Aptitude***...system is less bloated, fells much snappier, uses less ram and use half the disk space. However if do an install like I did for the first time it takes a while to find all packages you need to get the system running properly.

### Post by TomFumb on 2007-05-07

Maybe it's a symptom of what someone said above about the mashup of gnome features in xfce, but I just find that configuration and tweaking is much easier in gnome, as everything fits together better
