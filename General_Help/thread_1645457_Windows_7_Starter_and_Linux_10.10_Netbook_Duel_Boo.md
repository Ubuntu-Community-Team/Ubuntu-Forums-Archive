### Post by Mark Phelps on 2010-12-15
All those options are routine.

If you want to set the default to Win7, easiest way to do this is to do the following:
1) In Ubuntu, install startup-manager
2) Launch startup-manager
3) Select Windows 7 as the default.

I believe that there is also an option to reduce the wait time.  Try setting it to 2 seconds.

Then, when you boot, see if you can press the Shift key before the menu appears.

You can try reducing the wait time -- but if you make it too short, you won't be able to press the Shift key in time to get a menu to select Ubuntu!
