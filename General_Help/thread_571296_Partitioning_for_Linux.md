### Post by notwen on 2007-10-09
If all you're wanting to do is Dual-boot Ubuntu and Windows on a machine currently running XP simply follow this [guide]("http://apcmag.com/6101/dualboot_windows_xp_and_ubuntu").  

*SpyroViper*, as far as your partition goes, do you not know how that Linux partition got there?  Having XP take up your whole HDD would make your Ubuntu installation go a lot easier.  Ubuntu can resize your XP partition and then install itself into the remaining free space.  If you do resize your XP partition, be sure to run scan disk(maybe even twice) before doing so.  Hope this helps and be sure to post back if you run into any issues. =]
