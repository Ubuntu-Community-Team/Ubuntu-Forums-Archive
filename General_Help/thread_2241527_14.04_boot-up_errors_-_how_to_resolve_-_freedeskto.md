### Post by stevecro2 on 2015-01-18
Finally found it --
Found this file: 
/etc/udev/rules.d/95-hal.rules

Commented out the line that looks like this:
RUN+="socket:/org/freedesktop/hal/udev_event"

Messages don't come up any more and everything else still seems OK
