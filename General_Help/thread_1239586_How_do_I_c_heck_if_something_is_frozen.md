### Post by starcraft.man on 2009-08-13
Alt + F4 almost always works to terminate an active program, like in windows. It closes only the highlighted window, the one with the focus that all keyboard commands go to.

To see the status of a program graphically go System > Admin > System Monitor. Like the task manager in windows, see the processes tab and look for it. Under status, it should say running if it's doing operations. Right click and kill it I'd say, nothing takes over an hour to install to my knowledge. Then try again.
