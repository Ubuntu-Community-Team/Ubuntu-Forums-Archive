The network setup is pretty much just like any other Wireless HP Printer network setup (used to be an on-site Geek Squad Agent, I've done thousands of these).
1. Look at pretty driver CD.
2. Throw away pretty driver CD.
3. Fight with packaging for 10 minutes.
4. Give up, destroy box with multi-tool.
5. Get printer hooked up and run through the print-head alignment.
6. Print the network test page to get the printer's IP address.
7. Connect to the ad-hoc wireless network the printer is on.
8. Open up a web browser and go to the printer's web page via it's IP address.
9. Enter in the correct network information for my wireless network.
10. Make sure everything is 100% absolutely correct. Also make sure I gave the printer an IP Address outside of the DHCP range.
12. Hit apply button on the printer's web page.
13. Wait 30 seconds for the printer to talk to my router.
14. Add the printer via HPLIP.
15. ???
16. Profit.
