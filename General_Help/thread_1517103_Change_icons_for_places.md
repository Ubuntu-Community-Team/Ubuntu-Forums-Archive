### Post by cryptotheslow on 2010-06-25
Rather than trying to change the icon in the menu list, have you tried changing the icon for the folder via Nautilus (the usual file explorer app).

The attached shows me doing it for one of my music folders - basically find your folder in the file explorer > right-click it > Properties > Emblems Tab > Select whichever icon you want.

I haven't tried adding it to a menu, but I presume it would use the new icon there.

### Post by Pmaw10 on 2010-06-25
It didn't make much sense but I was able to do it. What I did was I went to my home folder that contained the default Music, Videos etc folders and deleted those and then created shortcuts to my real Music, Videos etc folders and moved them there. Then for some reason, when I changed the name from "Link to Music" to just "Music" it automatically changed the folder icon and I just put those in the left panel.
