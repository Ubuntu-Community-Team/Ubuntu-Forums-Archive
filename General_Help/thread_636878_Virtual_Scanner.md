### Post by Portable_Jim on 2007-12-10
Doesn't the PDF have selectable text?

If so, select the text, then copy and pase it into OpenOffice

if the PDF has copy protection, install the Cups PDF Printer ```
sudo aptitude install cups-pdf
```(then add the printer in System>>Administration>>Printers) then print the PDF to PDF to remove the copy protection.
