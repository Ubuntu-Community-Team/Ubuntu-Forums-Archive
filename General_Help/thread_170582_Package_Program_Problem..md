### Post by PhoenixP3K on 2006-05-04
Well... most programs don't automagicaly add themselves to the application menu. Sometimes even if they are added it's not until the Gnome Panel are reloaded that you see them. You may need to log off and then log back in. Or you could also use the command: [**killall gnome-panel**]("http://easylinux.info/wiki/Ubuntu#How_to_refresh_GNOME_panel")

If you are using Ubuntu 5.10 (Breezy Badger) you can easily add a shortcut menu using the menu editor, previous Ubuntu versions can also get that [Menu Editor]("http://easylinux.info/wiki/Ubuntu#How_to_install_Menu_Editor_for_GNOME")
