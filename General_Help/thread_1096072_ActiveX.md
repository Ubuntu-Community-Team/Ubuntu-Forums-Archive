### Post by cariboo on 2009-03-14
Active X is Windfows only. If you need the flash plugin for Firefox, go to System-->Administration-->Synaptic Package Manager, and search for ubuntu-restricted-extras and install it. The ubuntu-restricted-extras meta package will install most of the codecs you need to play most audio/video media, plus Flash Java and the MSCorefonts.
