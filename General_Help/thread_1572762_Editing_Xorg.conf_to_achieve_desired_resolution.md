### Post by qyot27 on 2010-09-12
> **Ihswh said:**
> OK, what is the code to back up my xorg.conf and try yours? Does yours work with 2 monitors? Thanks Claus7
I currently only have one monitor in use, although at one time I did use two.  I found maintaining a dual-monitor setup to be a pain in the neck to do, although it did eventually work.

To backup, just do:
```
sudo cp xorg.conf xorg.conf.bak
```

Then just edit or copy over the original with the new one with gedit.  To restore the backup, you'd do:

```
sudo mv xorg.conf.bak xorg.conf
```
