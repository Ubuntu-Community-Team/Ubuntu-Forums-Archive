### Post by Zackariah on 2007-10-29
> **zorch20 said:**
> How do i compile c++ programs, im very new to ubuntu and i need help

To compile C++ you need g++ compiler.

go to Applications > Accessories > Terminal
type in   sudo apt-get install g++

To compile, you need to be in Terminal.
cd to the directory of where your C++ Code is stored and type in   g++ mycode.cpp -o myoutputfile
where mycode.cpp is the name of your source code, and myoutput file is the filename of the compiled code.
