### Post by geek_Man on 2007-01-25
You can install KDE and all your Gnome apps will work in Gnome. I have AmaroK install in Gnome and it works fine, so maybe it'll be the same the other way around.

To install KDE, go to the terminal and type:

```
sudo aptitude install kubuntu-desktop
```

Make sure that all the repositories it requires are open. Otherwise it won't work.
You'll also have to do a bit of tweaking if you want to keep the Gnome boot screen. Installing KDE will change it. Hope this helps!

### Post by kebes on 2007-01-25
You will not have to re-install your apps. If you have Gnome and KDE installed, then both of them will be able to use the other one's apps.

The only thing is that KDE might not ~list~ Gnome apps in the menu (although alot of them will show up I think). But you'll be able to run them (do Alt+F2 and then type the name of the program), and you can create your own menu items for launching them.

Installing KDE is quite easy, so you could just try it. I bet your apps will work fine!
