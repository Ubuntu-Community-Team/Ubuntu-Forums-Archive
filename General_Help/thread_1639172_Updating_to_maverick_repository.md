### Post by snowpine on 2010-12-06
You can't just switch repositories; that will definitely cause problems!

If you were using Ubuntu, I would recommend you read this guide: [https://help.ubuntu.com/community/UpgradeNotes](https://help.ubuntu.com/community/UpgradeNotes)

Since you are using Mint and not Ubuntu, however, I recommend you read the documentation here: [http://www.linuxmint.com/](http://www.linuxmint.com/)
