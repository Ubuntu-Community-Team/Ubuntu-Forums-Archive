### Post by kanikilu on 2009-03-13
I don't think you can do that. Notice in the features matrix for gparted that "grow" for hsf+ is not checked:

[http://gparted.sourceforge.net/features.php](http://gparted.sourceforge.net/features.php)

If you can boot up with your Mac OS X DVD, I believe you can use "Disk Utility" to accomplish this:

[http://www.creativetechs.com/iq/leopard_disk_utility_can_resize_partitions.html](http://www.creativetechs.com/iq/leopard_disk_utility_can_resize_partitions.html)
