### Post by wkulecz on 2010-09-09
I switched my wife over to Linux starting with Ubuntu 7.10 and its worked out very well, but...

There are a lot of applications out there than are "Windows only".  My wife is taking a GIS course at the local community college and the training software is Windows only.  

Fortunately I had setup my old Windows 2000 in a Virtualbox virtual machine and could install it there for her to use.  Only real snag is it wouldn't install from the CD, but when I copied the CD to an iso file and mounted the ISO in Virtualbox it installed fine and so far has worked well for her.  She only needs it to complete the course, then the software expires.

You will have to be on your toes as these issues pop up, but overall its a great big win for me as I spend near zero time maintaining her Linux box compared to all the time I spent "fixing" her Windows systems previously.
