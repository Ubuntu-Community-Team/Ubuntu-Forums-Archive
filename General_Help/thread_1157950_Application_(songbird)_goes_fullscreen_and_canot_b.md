### Post by bacardiandwatermelon on 2009-05-13
In the past if I have ever had any issues with windows I disable the desktop effects. System->Preferences->Apperance->Visual Effects->None.
I then adjust the window I'm having an issue with, then re-enable the Visual Effects. It maybe worth a shot.
