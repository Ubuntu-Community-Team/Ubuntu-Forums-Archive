### Post by dg_w on 2006-03-25
Yep it works and looks great..
There are many threads on it,,
I think this one has to be the best .. A single script you run and thats it all done. Works great with ATI and Nvidia

[http://www.ubuntuforums.org/showthread.php?t=147049](http://www.ubuntuforums.org/showthread.php?t=147049)

Install Dapper
From a terminal (Ctrl+Alt+F1)
Login
sudo apt-get update 
sudo apt-get dist-upgrade
Reboot
Login
Download script
Exit Gnome
From terminal
cd /to where you saved the script
sudo chmod 777 inst_xgl.sh
sudo sh inst_xgl.sh 
Reboot when asked and repeat form the line above
Reboot again and all done !

Hope that helps, the above is how I installed it , though it should be just fine
installing whilst in X

Good Luck
