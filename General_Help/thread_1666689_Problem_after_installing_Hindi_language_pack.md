### Post by skvaditya on 2011-01-14
I am using Ubuntu 10.10 gnome desktop edition. Today I tried to use hindi language pack. So I did the following operations:
systems->administration->language support
Under language tab I selected hindi & installed all its components.
Then I went into text, selected hindi & applied it system wide.
It then asked me a question that to you want to change the folders such as desktop,documents, pictures,videos n others in /home to that of hindi. I selected yes.
Everything worked fine but now when I tried to switched back to english, every thing changed but I found that new folders for desktop,pictures,videos etc. are created in hindi language & & the access from places is still to them. 
If I save anything on desktop it saves in the folder named as desktop in english in /home not on desktop. I removed hindi language pack completely but nothing changed. So the access to all my folders in home went to those new folders. I deleted them but also no change. In places menu those hindi names are only coming not the old ones.
In the attached file the highlighted file shows the new folders created & those are only being shown in places menu. How to restore it back as was before installing hindi pack?
please help
