### Post by meng on 2007-05-29
Sorry I didn't follow up my initial reply. You seem to be finding the answers yourself anyway, but I'll add:

1. Migrating data from Outlook to Evolution is a real pain, and unless someone has come up with a niftier solution in the last 6 months, I've heard that the easiest thing to do is "triple shift" from Windows Outlook to Windows Thunderbird to Ubuntu/Linux Thunderbird to Ubuntu/Linux Evolution. Never done it myself though (I just archived the old mail, and may have managed to export contacts as CSV file or something). There was a program years ago for converting .pst to Evolution format, but I believe it's an abandoned project now.

2. After using Ubuntu for a while, one becomes familiar with the native Linux alternatives to Windows applications. I would recommend these above a wine-d application unless there's a missing functionality that you just can't do without. So, for example, I understand there's no Linux version of (or alternative to) Dreamweaver, so you're stuck with wine-ing it. Instead of Photoshop, there's GIMP, however, not everyone finds using GIMP as intuitive as Photoshop (I wouldn't know, I've never used Photoshop). Check out [www.winehq.com](www.winehq.com) and the AppDB to get an idea how well individual applications run in wine, and the Crossover Linux site has similar ratings on usability/compatibility.
