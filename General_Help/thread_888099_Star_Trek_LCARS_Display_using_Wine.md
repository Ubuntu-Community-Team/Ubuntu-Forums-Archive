I always run System47 screen saver on my Windows boxes: 
[http://www.mewho.com/system47](http://www.mewho.com/system47)

You can install this using wine and run it using wine also (rename the .scr in the Wine Windows/system32 directory to .exe or just send the .scr name to Wine). Unfortunately, on my Twinview set up, it only drives a single screen.

However, if you were only running a single screen, it makes a nice screen saver.
