I have an US-international keyboard by I usually type in Portuguese. In windows, for example, to type á é í ó ú, I only type ( ' + vowel ). In ubuntu, I have to type (right alt + ' + vowel). How can I configure Ubuntu so that the right alt becomes unnecessary?

### Post by prinny42 on 2009-03-01
Go to System -> Preferences -> Keyboard and go to the Layout tab.  Click the button with a plus sign on it and under country select United States.  Then, under variant, select USA Alternative International.  Once that's done, go to Other options and expand the "Layout Switching" section.  Put a check by whatever method you prefer.
