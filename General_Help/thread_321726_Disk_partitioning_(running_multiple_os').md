just go download [ubuntu]("http://www.ubuntu.com/") and burn the iso. when you put the cd inside and reboot you'll load the ubuntu live cd and when you install there is a built in partitioning, that's not the safest way thogh. I would strongly recommend using the gParted live CD for partitioning your drive.   First defragement your XP drive a few times, then use the GParted live CD which you can download from [[COLOR="DarkOrchid"]here[/COLOR]]("http://gparted.sourceforge.net/").

I have used it a bunch of times recently (as I have gone crazy and have 6 different OS's on my machine right now) and never had a problem.  Make sure you back up your data first just in case8)
