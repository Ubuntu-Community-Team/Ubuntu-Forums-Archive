### Post by drewsus on 2011-08-08
Use redshift instead. 

```
sudo add-apt-repository ppa:jonls/redshift-ppa
sudo apt-get update
sudo apt-get install redshift
```

Then, simply use this command at startup if you have already set your location with the gnome clock:
```
gtk-redshift -l gnome-clock
```

Ive been using this program on my computers for some time now. I like it a lot. 

Furthermore, if you need the info, the tray icons for redshift are stored here: /usr/share/icons/hicolor/scalable/apps/
