### Post by Petro Dawg on 2012-08-31
To put the memory size issue into perspective, a Wubi dual boot install of Ubuntu12.04 is limited to 30Gig maximum (normally).   So a 32Gig flash drive will be plenty, even if you install a lot of programs.

I do believe an installation to an external HD would respond quickly, just as if you installed to an internal HD (maybe just read/write times would be increased), but I have never done that so I can't say for certain.
