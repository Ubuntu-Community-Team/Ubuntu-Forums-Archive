### Post by gobbles414 on 2008-08-01
Strattonbrazil, I've attached a screenshot that is similar to what you need to find on your computer -- the solution is a little bit different on each computer. Notice the red _X_ over the speaker icon...?

In Volume Control, there are two places that you'll want to look. First is *File => Change Device*. Second, make sure that you can see all of the options for each device by going *Edit => Preferences*. Note, each device has it's own set of preferences. So each time you change to another device, you'll need to go into *Preferences* again.
