### Post by TeoBigusGeekus on 2009-07-22
There is no particular format you are obliged to choose - just whatever suits your needs. If you are going to use the drive only with ubuntu you could choose ext3. If on the other hand you want to be able to read it from a windows pc, you should choose ntfs or fat32. 
Unless the drive isn't already formatted, you could use gparted to format it yourself.
