### Post by p_quarles on 2007-07-28
It's really difficult to install software without an internet connection. You could download the source code and attempt to compile it, or you could try to find a Debian package (which is the closest thing to a self-extracting installer). However, neither of those methods will be able to retrieve the dependencies you are missing.

The easiest way to get Amarok (which really is quite good) without internet access is to install Kubuntu, which comes with it already installed and working.
