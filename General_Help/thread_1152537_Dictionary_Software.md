### Post by lykwydchykyn on 2009-05-07
Most of the internet-based things you have seen (I'm assuming you looked at the panel plugin, for instance) connect to a dictd server on the internet running a variety of free dictionary databases.

You can install a dictd server locally and a large number of dictionary modules directly from the repositories.  If you first install "dictd" from synaptic, then search for "dict" you'll find a number of packages whose names are in the format "dict-something" where "something" is the name of the dictionary module.  There are around 75 modules available.

Once you install this, you can just edit the settings of your dictionary plugin or whatever to point to localhost instead of whatever website it's hitting (usually dictionary.org).  You have the same functionality, no internet required.

As for the interface you describe, I haven't seen anything like that.  You'll have to be content with copying words and pasting them into a field, unless there is some way to get clever with scripting the command line client.
