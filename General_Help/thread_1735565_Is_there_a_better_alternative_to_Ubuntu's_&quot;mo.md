### Post by Mr_VeRiTy on 2011-04-21
Hi, I'm using Ubuntu's "mouse keys" feature to move my mouse cursor around the screen using the keyboard, but I was wondering is there was a different program that would do the same thing but better.

By better I mean, for example, when transitioning from moving the mouse upwards, to moving the mouse sideways, there is a pause (which happens whenever you overlap numpad key presses,) which slows me down quite a bit. Also, to be able to drag by holding the numpad 5 key and moving the mouse in a direction with another key (instead of pressing numpad 0 to initiate drag, and 5 to stop),  would be awesome.

I know this sort of behaviour would probably be troublesome to people with more serious disabilities, but is there a program that does this? Or can I configure Ubuntu's default program to do this?

### Post by Telengard C64 on 2011-04-22
I have a keyboard with a trackball.
[http://pckeyboards.stores.yahoo.net/on10.html](http://pckeyboards.stores.yahoo.net/on10.html)

They also have keyboards with a stick pointer.
[http://pckeyboards.stores.yahoo.net/migmousblac.html](http://pckeyboards.stores.yahoo.net/migmousblac.html)

I also use KDE's *Mouse Navigation* feature. I assume Ubuntu has something which works the same (or almost the same). I'll explain what exists on my Kubuntu 8.04 Hardy Heron system, but you'll have to find the equivalent settings on Ubuntu.

*K-Menu* -> *System Settings* -> *Keyboard & Mouse* -> *Mouse* -> *Mouse Navigation*

The pause you experience when switching directions can be overcome by adjusting the settings on that page.

I don't think there is any way to drag and drop without using **KP0** and **KP.**.

Hope you can find an input device which works better for you.
