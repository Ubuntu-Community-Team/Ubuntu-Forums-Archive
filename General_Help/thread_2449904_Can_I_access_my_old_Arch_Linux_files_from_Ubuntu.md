When I first started learning Linux - and for the record I'm still a beginner - a friend helped me install Arch. We had a falling out, and without her help I found it too daunting to use. I'm installing Ubuntu on my SSD, but I want to be able to transfer files over from my HDD with the Arch install and I don't have easy access to adequate storage to offload the files outside the Arch installation. Is there an easy way to transfer the files from within Ubuntu?

---

### Post by Dennis N on 2020-09-03
If the HDD is a connected disk, it should appear in the File Manager. Click "+ Other Locations" at bottom of File Manager sidebar. In the next window, you should see the partitions of connected disks listed. Click to mount. A user password for the other system may be required.
