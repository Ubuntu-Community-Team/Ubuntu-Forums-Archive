### Post by bailey86 on 2009-01-23
As an add-on to this.

When adding it to my new Dell Mini 9 with Hardy Heron the printer wasn't detected automatically.

I selected AppSocket/HP Direct and then in the host field put in the IP address of the printer.

(As a side note - I always set these printers up with a permanent IP address from either the little LCD control panel or via the web interface after it gets an initial IP via DHCP).

Everything else was then detected normally.
