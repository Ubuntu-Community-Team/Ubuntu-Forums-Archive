### Post by Marsolin on 2007-04-04
Running the software in a virtual machine is an option, but there should be some Linux alternatives. If I understand your need correctly I would look at [Quasar Accounting]("http://linuxappfinder.com/package/quasar"), [TurboCash]("http://linuxappfinder.com/package/turbocash"), [MyBooks Professional]("http://linuxappfinder.com/package/mybooks"), [Lazy8 Ledger]("http://linuxappfinder.com/package/lazy8ledger"), [jbilling]("http://linuxappfinder.com/package/jbilling"), and [Simple Invoices]("http://linuxappfinder.com/package/simpleinvoices").

If you are interested in trying a web-based interface then you could try [LedgerSMB]("http://linuxappfinder.com/package/ledgersmb") and [SQL-Ledger]("http://linuxappfinder.com/package/sql-ledger").
