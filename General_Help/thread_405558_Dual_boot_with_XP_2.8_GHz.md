### Post by m.musashi on 2007-04-09
I think your biggest problem will be HD space. They say Ubuntu can install to as little at 2-3GB but that would be almost useless. You could give it 10GB but that leaves windows with 20GB and depending on what you have on windows that might not be very usable either.

HDs are fairly cheap. By a separate one to give yourself some room to work. Of course, if the windows drive has some free space you can give it a shot. Be sure to back-up important stuff just in case. You can also boot the live CD and get some idea of how it will run on your hardware. It will be slow running off CD but unless it is REALLY slow you should be fine.

If you just want to try Ubuntu out and the live CD is a bit too slow, you might try vmware. It's free and you can install Ubuntu in a virtual machine in windows.
