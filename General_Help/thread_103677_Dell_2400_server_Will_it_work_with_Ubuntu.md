### Post by soulestream on 2005-12-14
most newer dell servers support redhat, so ubuntu shouldn't be a problem. Hardware is kernel specific, so if it supports one linux distro, it will support them all. You might have to setup your own linux drivers, depending on the distro.  Might want to do some googling. The main hurdle I can imagine would be making sure the harddrive controller (ide, scsi, sata) has linux drivers.
