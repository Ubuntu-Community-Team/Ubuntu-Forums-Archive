### Post by k-bl on 2010-04-09
This is a very complicated and long how to article.

For GUI
Make sure you have Cron/Scheduled Tasks installed
Open Terminal - Click Applications/Accessories/Terminal
Type: "sudo /usr/bin/gnome-schedule"

FOR CLI
Type: "sudo crontab -e"
See Link for Syntax: [https://help.ubuntu.com/community/CronHowto](https://help.ubuntu.com/community/CronHowto)

Schedule, Save, and close :)
