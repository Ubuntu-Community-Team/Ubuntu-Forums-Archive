I have played around with its predecessor, the Archos604.  Want I can tell you is, the Archos has built-in samba server and client.  Therefore:
1. It is possible to stream video/media from a samba server (in a linux PC) to the Archos.
2. It is also possible to stream video/media from Archos to a linux PC, with limitation.
