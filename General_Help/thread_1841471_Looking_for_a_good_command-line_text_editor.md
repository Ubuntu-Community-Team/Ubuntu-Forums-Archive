### Post by Esotera on 2011-09-09
Specifically, I'm looking for a program that always displays the entirety of what has been typed, even if it spills onto the line below.


I've tried nano and like its simplicity, but whenever the line is longer than the screen, it occludes part of it, which isn't ideal for writing paragraphs. Is there a way to emulate this behaviour in nano, or any other popular text editor, without having to constantly add a newline?

---

### Post by nothingspecial on 2011-09-09
In nano press Alt-L, this will change the behaviour so it starts a new line.

### Post by lykwydchykyn on 2011-09-09
The feature you're talking about is called "line wrapping" or "word wrapping" (the difference being whether it breaks on spaces or at a hard character count).

Emacs or vim do this by default.

---

### Post by WorMzy on 2011-09-09
ESC, followed by $, enables (or disables) soft line wrapping.
