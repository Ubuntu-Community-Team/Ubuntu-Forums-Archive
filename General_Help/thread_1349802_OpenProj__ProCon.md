### Post by Voynix on 2009-12-08
Recommended software for my students. OpenProj handles MS files (it can import .mpp and export ms xml project files), it is cross-platform, it can run portably and it is open source. It produces some critical path networks (although there are some clear bugs), and some decent reports. It does have some limitations, no resource leveling, network optimisation, and the latest editions have restricted pdf export functionality (sure, you can code it yourself if you can handle java). It is clearly a desktop application, so it does not have any multiple user access, or server-side synchronisation. I have encountered random localisation errors (pounds and dollars making costs a real righmare). CSB, WBS and RBS seem to be detailed and charting capabilities are a real treat. If you cannot export them, there is no real use, to be honest. A good learning tool, anyways. By the way, if you try to get rid of the hideous metal interface and customise it with GTK laf, nimbus or any other skins, you will have to compile the application from source. GTK laff is disabled by default.

There is also ganttproject, with a myriad of plugins and goodies. It only focuses on networks and gantt charts (some limited resource allocation), and WBS monitoring... Simple, but effective.

Gnome Planner has some potential, but at the moment it does not have many interesting features.  I have come across many project management software for other platforms (merlin, etc) but they are overpriced and extremely platform specific...

### Post by drumkitcat on 2010-02-18
I agree, I haven't used OpenProj extensively as I just learned about it recently.  I'm a PMP certified project manager - and it's always a good thing to find software that can import MS Project files!  

From what I've seen of it so far it's easy to work with - a lot of similarity to MS Project, but I find the calendar/working time features are a little nicer than MS Project.

One thing I do want to research more closely is if/how it handles custom reports other than the stock items available.  Usually those will do the trick, but sometimes you need a custom report.

Definately though, OpenProj is the nicest/most robust project management software I've seen on the Linux platform so far.
