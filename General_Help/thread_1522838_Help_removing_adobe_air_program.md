### Post by wookieshaver on 2010-07-02
The best way to remove any program in ubuntu is using the package manager.  You navigate to System>Administration>Synaptic Package Manager let it populate then search for the program and uncheck it (assuming it appears) or more correctly "mark for removal". Heres a simple guide:
[http://www.cyberciti.biz/faq/howto-delete-remove-software-using-apt-get-command/](http://www.cyberciti.biz/faq/howto-delete-remove-software-using-apt-get-command/)
