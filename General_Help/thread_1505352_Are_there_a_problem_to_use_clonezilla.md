### Post by labinnsw on 2010-06-10
I believe what you are attempting can be best achieved by using the Ubuntu Live CD and Gparted. Gparted is extremely user friendly. You can use it to resize the partition. 

Although I don't believe you will need it, I always recommend backing up. For backing up, I use the [tar gzip commands]("http://ubuntuforums.org/attachment.php?attachmentid=158063&stc=1&d=1274699954") myself, but whatever you use, the important thing is that you back up. It won't be too difficult to get assistance once you have a backup.

By the way, my current system is a partition that was resized from 20 to 120GB.
