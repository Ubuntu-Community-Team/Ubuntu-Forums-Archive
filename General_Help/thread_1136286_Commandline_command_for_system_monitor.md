### Post by dcstar on 2009-04-24
> **NekoMaster said:**
> I would like to know so I can make Ctrl+Alt+Del summon forth the system monitor just like you do with taskmanger in windows.

To find any menu item command, simply right-click the item and send it to your Desktop, then go into the properties of the launcher on the Desktop to find the command (and delete the icon once you have the information).

---

### Post by m_2009 on 2009-04-24
The commandline command is

```
gnome-system-monitor
```
