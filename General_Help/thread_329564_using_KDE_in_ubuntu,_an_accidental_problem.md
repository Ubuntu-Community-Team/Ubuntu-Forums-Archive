### Post by Keldek on 2007-01-01

### Post by meng on 2007-01-01
I suspected that you chose between kdM and gdM (the logon manager). Also, removing ubuntu-desktop doesn't remove GNOME (ubuntu-desktop is a metapackage). At the login prompt, look for an option to change the default session from GNOME to KDE.

### Post by meng on 2007-01-01

### Post by meng on 2007-01-01
Not really. Essentially you end up with the same result as if you had installed Kubuntu in the first place, rather than Ubuntu. Personally, I had wireless interface problems with Kubuntu that didn't occur with Ubuntu, although I suspect these problems could have been fairly easily fixed if I had had the know-how at the time.

### Post by aysiu on 2007-01-01
Don't use *sudo kate*. *sudo* is for terminal applications. *kdesu* and *gksudo* are for graphical applications. It should be ```
kdesu kate
``` Read more here:
[http://www.psychocats.net/ubuntu/graphicalsudo](http://www.psychocats.net/ubuntu/graphicalsudo)
