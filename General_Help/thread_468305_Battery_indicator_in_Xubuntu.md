### Post by loathsome on 2007-06-10
There's lots of programs out there. If you do **[COLOR="DarkSlateGray"]$ apt-cache search battery[/COLOR]** you'll get heaps of programs.

I recommend 'xfce4-battery-plugin' since you're using XFCE4. Install it by typing **[COLOR="DarkSlateGray"]$ sudo apt-get install xfce4-battery-plugin[/COLOR]**

You also might want to check out lots of other neat XFCE4 programs. You could for example do something like **[COLOR="DarkSlateGray"]$ apt-cache search xfce | less[/COLOR]** to see all the programs that are in the repos.
