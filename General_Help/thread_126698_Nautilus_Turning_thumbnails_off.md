### Post by dcer on 2006-02-07
Hi,

I have a folder where I store the wallpapers I use. A script picks them randomly and set them as wallpapers. Currently I have 620+ different images there. If I go with Nautilus in that folder it takes an eternity before a list of files is shown, in this time all instances of Nautilus freeze as well as my desktop (but not the panels or any other programs). I assume this is because Nautilus creates a thumbnail for every file found there. The folder is also on a FAT32 partition, so that probably adds to the slowness too.

Is there a way to turn off thumbnails in that particular folder and keeping them on for others?

Thanks!

---

### Post by dushkal on 2006-02-07
Go to 
Edit->Preferences->Preview tab->Other Previewable Files, and select Never instead of default Local Files Only.

Cheers,
Dushkal

### Post by dcer on 2006-02-07
This turns off thumbnails altogether :( still, better than nothing, the files now show up pretty quickly :) Thanks.
