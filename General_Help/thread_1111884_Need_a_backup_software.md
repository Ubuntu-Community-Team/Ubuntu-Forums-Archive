### Post by aaronp on 2009-03-31
Try Simple Backup - in the repos.

Very easy to use and works well.
Set up to do a full backup at specified intervals and then just backs up the incrementals changes at intervals in between.
Also, keeps a list of all your installed programs so if have have to start from scratch you can even restore all the apps you had.

I've had to restore a single file a few times and it was hassle free.
