### Post by gingivere0 on 2010-06-12
Do you have compiz simple settings manager installed?  
```
 sudo apt-get install simple-ccsm 
```
Then go System>Preferences>Simple CompizConfig Settings Manager.  Try setting your effects from here, that tends to work for me.  Good Luck!
