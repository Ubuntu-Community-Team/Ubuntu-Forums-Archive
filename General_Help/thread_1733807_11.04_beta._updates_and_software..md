### Post by mikewhatever on 2011-04-19
Ubuntu 11.04, aka Natty gets constant updates. The only scenario where you'd need to reinstall is if an update really breaks the system, which is pretty unlikely at this stage.
Both Apache and php are in the repositories:
[apache2]("http://packages.ubuntu.com/natty/apache2")
[php5]("http://packages.ubuntu.com/natty/php5")

### Post by ttshivers on 2011-04-19
Packages are do not depend on what desktop environment you use, but what package system you have(.deb, .rpm) so all the packages that are in the repositories for your system will work on any desktop environment.
