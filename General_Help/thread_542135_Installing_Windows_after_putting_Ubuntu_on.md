### Post by sumguy231 on 2007-09-03
You'll need to somehow make room for Windows before installing it. You can use Gparted on the LiveCD to do this. Be aware that when you do install Windows it will overwrite your bootloader and you'll have to reinstall Grub to be able to boot to Ubuntu.
