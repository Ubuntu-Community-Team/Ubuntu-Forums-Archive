### Post by cchhrriiss121212 on 2010-10-15
Download 10.04, which is the stable long term version, rather than the newest release 10.10. 
Burn to a cd and try it out using the live mode to see if you like it before installing. You will also get to check if all your hardware is supported, things like printers can be tricky. The install is easy, just point and click.
