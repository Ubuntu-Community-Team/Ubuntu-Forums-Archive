### Post by davec64 on 2010-11-27
The "^" in ^R is Ctrl R so use the Control key at the bottom right of your keyboard.

Hope that was what you were after!  :)

Just to add a bit more, position the cursor where you want to add some text.
Type the text in and press Ctrl O, to write it to the file then Ctrl X to exit.

Always best to make a backup before tinkering with a system file!
