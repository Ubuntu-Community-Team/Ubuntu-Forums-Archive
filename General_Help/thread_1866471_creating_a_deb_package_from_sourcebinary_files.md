### Post by gsmanners on 2011-10-21
Hi. Welcome to the forums. :D

The idea behind checkinstall is to make personal packages for your system (and your system only). I use it all the time, but mainly so I can use dpkg to remove apps when I'm trying out different versions of the apps.

To make a proper deb file takes a little more work, but there are guides here and there on how to do that. I suggest looking around this forum: [http://ubuntuforums.org/forumdisplay.php?f=44](http://ubuntuforums.org/forumdisplay.php?f=44)
