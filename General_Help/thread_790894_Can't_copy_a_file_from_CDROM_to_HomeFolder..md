> when I put the mouse cursor over the CDROM that error comes upyou never have to put the cursor over the file, cdrom, ect. You do the whole operation from a _dialogue box in_ konqueror, the last step being to either browse to or type the location of the bookmarks.html. (_from inside the dialogue box_). In any event you still do the whole operation from inside konqueror. (or inside of firefox if that's what you want to import to)
