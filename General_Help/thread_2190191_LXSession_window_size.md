### Post by Dennis N on 2013-11-26
Try this to access any buttons hidden below, off-screen:

Place mouse cursor somewhere in lower part of window. Hold the Alt key down and then press and hold left mouse button down at the same time. You can then drag the window up with the mouse and see what's below. To close the window, drag it back down so you can see the closing X.
