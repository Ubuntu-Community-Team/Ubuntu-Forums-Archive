Yes, it is, if you mean by increasing the grid size, making the grid finer, but it means changing the size of desktop icons, which may be what you want anyway.

Open a Nautilus window - home folder, computer, whatever - and from the Nautilus file menu go to Edit > Preferences. Now click on the 'Views' tab (if necessary). Under 'Icon View Defaults', change the value to 66%, 50% or 33%, whichever suits you. (Or 75%, etc, if you're using an older release of Ubuntu.) You'll find that you can put icons closer and get finer-grained control over their placement.
