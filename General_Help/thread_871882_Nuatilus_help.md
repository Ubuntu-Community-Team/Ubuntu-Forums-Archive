### Post by xlinuks on 2008-07-27
I ain't a professional either.
There is installing and applying fonts.
One can install them by going to Applications->Add/Remove..->select all available applications and search by the keyword "TrueType". Install "Microsoft true type" if that's what you want.
To customize where you want the fonts to appear you go to:
System->Preferences->Appearance->The "Fonts" tab.
The dialog is pretty self explanatory.
