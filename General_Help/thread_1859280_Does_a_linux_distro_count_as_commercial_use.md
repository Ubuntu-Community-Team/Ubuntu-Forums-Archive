### Post by Musmanno on 2011-10-13
Yeah, the fact that it is a Linux distro doesn't matter so much, rather it is what you are doing with it. If you're profiting in any way (advertisement, for example), then it seems pretty clear you have a commercial use. 

Carefully read the license that goes with the icons. If there is no license, then keep in mind that copyright infringement doesn't require a commercial use to be infringement.

### Post by prodigy_ on 2011-10-13
> **haqking said:**
> what counts as commercial use is using something to conduct commerce....ie in a business or corporation
Not exactly true. In theory anything that generates income directly or indirectly is commercial use. In practice this means presumption of guilt unless you're a home user. If you put something covered by such a license into a *software project* like a Linux distro and it becomes popular by any chance, it also becomes increasingly difficult for you to prove that you aren't benefiting in any possible way.

That's why you typically don't want to use anything that comes with an NC license. In the end it's not worth the trouble.
