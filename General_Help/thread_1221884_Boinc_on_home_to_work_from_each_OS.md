hi,

If you have several distro on your computer with dedicated partitions & a /home, you may wish boinc able to continue his work from all your installed OS. You need to install boinc on each distro, no matter if boinc are not the same one because only workunits are shared.

Like that, boinc is always computing, no matter if you boot on distro 1 or distro 2. I have made the changes & every things work perfectly (actually: same work from Jaunty, Karmic & Sidux)

You can follow the discussion here:
[https://bugs.launchpad.net/ubuntu/+source/boinc/+bug/402082](https://bugs.launchpad.net/ubuntu/+source/boinc/+bug/402082)
