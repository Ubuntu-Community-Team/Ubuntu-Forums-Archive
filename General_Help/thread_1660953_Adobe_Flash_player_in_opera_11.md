### Post by lovinglinux on 2011-01-06
Get [Flash-Aid]("http://www.webgapps.org/addons/flash-aid") and run it. It will detect installed flash plugins, remove them and install the best option according to your system architecture and version. Additionally, it will apply some tweaks. 

Opera will detect the plugin installed by Flash-Aid after restart.
