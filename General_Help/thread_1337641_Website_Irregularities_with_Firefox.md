### Post by t0p on 2009-11-25
Firefox is far more likely than Internet Explorer to display a web page correctly.

In the absence of any further information, I'd say that your web page will display as it does in Firefox.  Internet explorer does not adhere to the industry standards.  So visitors to your site who use IE may see the site the same as you do in IE.  But anyone who uses a proper browser will probably see the site the same way as you see it in Firefox.

I advise that you edit your site so it displays correctly in Firefox.

There are many things people can do to make a web site render differently in IE compared to (almost all) other browser.

People should check their web sites with one of the many on-line Validators and they should find that if they get their website to comply with the current standards, then they will render consistently on **all** browsers.

Having just spent quite a while slowly cleaning up a commercial web site to make it standards compliant (and render correctly in all browsers), it is well worth doing because having sites that only work in IE is so 1990s.
