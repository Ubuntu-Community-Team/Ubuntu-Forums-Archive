### Post by luptinpitman on 2007-06-04
I believe you are referring to the "Desktop Color" that is set under Desktop Background Preferences.  The default color in Ubuntu is that brown background you see.

Once logged in, you can go to System->Preferences->Desktop Background and change the color if you like.
