 I use [Jinzora]("http://jinzora.com") to stream my music to work. It's a bit of work to get set up, but it is well worth it. It has a very rich interface and it supports downsampling on-the-fly so my high-quality music is actually streamed at a bandwidth-friendly 64kbps. 

You just need apache + php on the machine to get started.
