### Post by whoop on 2009-02-17
I believe your answer is still vnc. vnc will let you connect with a current desktop but you can also use it to start a new desktop (with a new user). 

I remember starting a remote VNC session on my girlfriends xubuntu machine while she was sitting behind it surfing the web. We where both using our own session, so it should be possible.
