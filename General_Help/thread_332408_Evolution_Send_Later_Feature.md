### Post by Spin Doctor on 2007-01-06
Wizzkid,

There is no option like this in Evolution. What I understand, if you are willing to change to kmail, that program has a delay send function. 

The closest thing to duplicate this on Evolution is according to Rick Friedman:> 1. Click on your Outbox folder 
2. From the evolution menu choose: Message -> Post New Message To Folder 
3. Create the message you will eventually wish to send. 
4. Send the message. 

The message will be kept in the Outbox folder. When you are ready to 
send it to the actual recipient, click on the message. Enter the 
recipient's address and send it as normal.  Found this URL of someone with your problem: 
[http://lists.suse.com/archive/suse-linux-e/2006-Sep/4483.html](http://lists.suse.com/archive/suse-linux-e/2006-Sep/4483.html)

Have a great day!
