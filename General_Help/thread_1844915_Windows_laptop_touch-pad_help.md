### Post by claracc on 2011-09-16
You can try to run gconf-editor in a xterm, then you navigate through desktop-->gnome-->peripherals--> touchpad, in the right side of the window tick or untick in the touchpad_enable key as you want to use or not use your touchpad.

I have not any button to enable touchpad in my hp compaq 6720s, so I only know this way through gnome settings.
