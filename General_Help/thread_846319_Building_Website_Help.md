### Post by pytheas22 on 2008-07-01
A good place to start is one of the free [CSS template sites]("http://www.google.com/search?q=free+css+templates&ie=utf-8&oe=utf-8&aq=t&rls=com.ubuntu:en-US:unofficial&client=firefox-a").  A CSS file defines how your site looks and is laid out, and these sites offer free templates that you can use for whatever you want, usually under a Creative Commons license.

Once you pick a CSS scheme that you like, you just need to open the accompanying html file (probably named "index.html") in a text editor and add your content, which is relatively easy.  If you know absolutely nothing about html right now, you are going to need to learn a little bit, but really not that much if all you want to do is copy and paste into one of these templates.  [http://www.w3schools.com/](http://www.w3schools.com/) is a good place to begin learning html.

Also keep in mind that you can create web pages in OpenOffice by saving documents as html, but it's hard to make anything very pretty that way.

You also need to find a web host and, if you want your own domain, register it.  There are a lot of free hosting services, but most will place ads on your page.  You can also get a free URL in the form of yoursite.somebodyelsessite.com, but if you want just yoursite.com, you'll need to pay to register it, usually about ten dollars.

As I said, if you have 0 experience with web design, you'll need to do a little bit of research on how html and web servers work to get this going, but I'd be happy to answer questions that you can't figure out on your own.

### Post by cpetercarter on 2008-07-01
I think you need these things:

1. Your web page needs to be written in html. You have two choices. You can use a WYSIWYG web page editor where you just type stuff in, add images, colours etc and the programme writes the html for you. I am afraid I don't know which Linux programme to recommend. Others may be able to advise. The alternative is to learn enough html (and css - used to style the web page) to do the job yourself. If you have the time, you will do best to learn how to write html yourself. It isn't difficult, and can be a lot of fun. I learnt from "Head First HTML with CSS and XHTML" by Freeman and Freeman. It is still the first thing I turn to when I hit a problem.

2. You need a sandbox to play around in. The best way to do this is to set up an internal website on your own computer, which you can then see by going to [http://localhost](http://localhost) in your web browser. For this you need the apache webserver programme, but you might as well also download the php programming language and mysql database while you are at it. An excellent guide on how to do this is [https://help.ubuntu.com/community/ApacheMySQLPHP]("https://help.ubuntu.com/community/ApacheMySQLPHP")

3. When you have played around in your private sandbox and got your webpages to a state where you would like the world to see them, you need to investigate webhosting services. Your ISP may offer a free service, or for a small monthly charge there are any number of hosting providers out there. Many web hosts offer a package which includes a free domain name (ie web address), which is pretty cool. When you have got your web hosting arranged, you will need an FTP programme to transfer your web pages from your computer to your new web site. gFTP is OK, or there is an excellent FTP extension for Firefox called FireFTP.

Hope this helps.
