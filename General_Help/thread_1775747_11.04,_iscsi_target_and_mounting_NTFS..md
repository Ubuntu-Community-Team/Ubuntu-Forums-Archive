Hi,

I have a basic Ubuntu server setup, running 11.04.

I  have several disks and have created a RAID array using mdadm. I have  created a volume group, and a lun. Using iscsitarget I have linked the  lun to an iscsi target.

In WindowsXP I can access the iscsi target, and have formatted it with NTFS.  It all works fine and is reliable.

The  problem is that now I wish to mount my lun (or the raid volume) in  Ubuntu.  I have installed the NTFS support, but no matter what I try I  can not mount anything.  Ubuntu sees the lun, or raid volume as not  having a partition table.

Can anyone shed some light on a possible solution?

### Post by ChrisChambers on 2011-06-16
Not sure if this will help -

Install NTFS Programs
# apt-get install ntfsprogs

Mount the new drive
# ntfsmount /dev/sdb1 /media/ntfsdrive

That's how I mount NTFS partitions.

I had a problem mounting a ntfs partition on ubuntu 11.04 and installed MountManager from Ubuntu Software Center and solved my problem in a matter of 10 minutes, 9 reading the manual, 1 actually fixing it.
