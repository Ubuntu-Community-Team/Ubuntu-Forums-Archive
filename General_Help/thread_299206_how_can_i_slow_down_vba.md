### Post by kobewan on 2007-06-16
For anybody that the above doesn't work for (it didn't work for me):

1. Run VisualBoyAdvance by entering the command "gvba"
2. Goto Options-->Frameskip-->Throttle-->100%

Now all games should run at 100% normally, and only go faster if you press the speed button!
