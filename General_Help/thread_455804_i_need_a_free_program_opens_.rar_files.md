### Post by yabbadabbadont on 2007-05-27
```
sudo apt-get install unrar
```
It is a command line application (like unzip) so you will have to run it in a terminal window.  You will need to have the multiverse repository enabled to install it.

[https://help.ubuntu.com/community/Repositories/Ubuntu#head-5bbef89639d9a7d93fe38f6356dc17847d373096](https://help.ubuntu.com/community/Repositories/Ubuntu#head-5bbef89639d9a7d93fe38f6356dc17847d373096)
