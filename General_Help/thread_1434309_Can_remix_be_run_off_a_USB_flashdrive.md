### Post by Oldskool Slacker on 2010-03-20
Short answer: yes ;)

Longer answer:

Download the UNR .iso file, the follow the instructions on [this page]("https://help.ubuntu.com/community/Installation/FromUSBStick") to create the bootable USB stick.

With the stick in place, turn your computer on.  The first thing you see will say something like "Press F12 for boot options".  Do that, and select the "USB" option.  UNR will load.

As far as setting up a dual-boot system, my main advice is: **resize the Windows partition from inside Windows**.  Then choose "install into free space" from the Ubuntu installer.

Good luck!
