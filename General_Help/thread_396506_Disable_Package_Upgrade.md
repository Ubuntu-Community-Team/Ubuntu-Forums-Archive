### Post by chamberlain2006 on 2007-03-29
There are 2 packages that I don't want to upgrade.  I want to make it so they are disabled, like as if there wasn't an upgrade available.  How could I do so?

---

### Post by raja on 2007-03-29
Go to Synaptic package manager, choose the package you want to retain from the list of installed software and then from the package menu, choose lock version.
