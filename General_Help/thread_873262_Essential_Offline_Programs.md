### Post by eric3_14159 on 2008-07-29
It really depends on what kinds of things you want to be able to do. There are many programs which don't require any internet access, which you can download while you do have internet access using apt-get, Add/Remove Programs, or Synaptic.

Assuming you have an ordinary install, you already have lots of programs for things like word processing, playing music or videos you have stored locally, burning CDs, and image editing.

If you want games, run Synaptic and look under the Games and Amusement sections. The majority there don't require internet access to run.
