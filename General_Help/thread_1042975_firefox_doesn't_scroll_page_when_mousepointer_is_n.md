### Post by spupy on 2009-01-18
This is the default behavior of all Linux programs (Gnome at least). On windows, the place where you clicked last has focus, so to say. On linux, the place where the mouse cursor is located has focus. That has the advantage that you can scroll windows that aren't focused. 
Also I don't think there is a way to change that. This is how the [X Window System (link)]("http://en.wikipedia.org/wiki/X_Window_System") gives focus.
