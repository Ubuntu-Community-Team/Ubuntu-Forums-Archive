### Post by bodhi.zazen on 2008-04-30
Open Xchat

Xchat -> Server list

Select server -> Edit

Add channel(s) to "Channels to join" , comma delineated list (#chann1,#chann2)

Save

Then , Check off "autoconnect to this network at startup"
