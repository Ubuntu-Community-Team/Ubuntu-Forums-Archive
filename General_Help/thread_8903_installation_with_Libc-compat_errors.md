### Post by exeel on 2005-08-06
Although this reply is probably far too late to help you, I came accross this while googling the problem, so I thought I'd share my experience.

Basically, I was getting the same problem, until I installed the libstdc++2.10-glibc2.2 package. To do this, run the following command from a terminal:

sudo apt-get install libstdc++2.10-glibc2.2

Should be fine after that.
