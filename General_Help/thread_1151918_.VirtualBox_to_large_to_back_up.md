### Post by jimbob on 2009-05-07
I want to back up my .VirtualBox directory but it is too large (~10Gb) to fit on a DVD.

How can I accomplish this?

---

### Post by Joeb454 on 2009-05-07
You could compress it, though I don't know how much that would save on space.

You could try ```
 tar -cjvf ~/Desktop/vbox.tar.bz2 ~/.VirtualBox
``` which would compress the directory and put the compressed file on your desktop

---

### Post by jimbob on 2009-05-07
Thank you, that worked very well.  It cut it down to 4.0Gb believe it or not.  It took over an hour and a half to do it.

Now to write it to DVD and hope I can get it back if the time ever comes.

### Post by jespdj on 2009-05-07
To decompress the file again, do
```
tar xjvf vbox.tar.bz2
```

### Post by UBB2 on 2009-05-07
with 7zip in ultra mode (I have Ubuntu as guest on WinXP) I actually had compression rates up to 1:10, it depends, of course, on how much space is used of the partition(s). Before you compress, it could be also useful to fill the empty parts of the partitions with zeros, because this will help with the compression ratio.
