### Post by arsenic23 on 2008-05-21
What else are you running on the system?  What CPU do you have ?

I **think** that you should only see about a 30watt increase in power on the average with that card.  But when your using 3d accelleration your peak power may likely go above what that power supply can handle ( asuming your already close to using it at full capacity ).

If you have a single hard drive, a single CD drive, and a relatively modern single core CPU then you should more then likely be using a 350 watt PSU to run that card stable.  BUT if you have a machine that is already designed to run with a lower then average power consumption, you may be able to run that card without a problem.

ANYway,you can try and play with this to give you a rought idea of what you can do with what you have : [Newegg Power Calculator]("http://educations.newegg.com/tool/psucalc/index.html") -
