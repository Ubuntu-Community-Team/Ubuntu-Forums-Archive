### Post by Legace on 2009-06-22

OS 3.0 feels snappier at some tasks (like launching Settings or loading web pages), but other tasks seem a little laggier.. All in all, a pretty decent upgrade, especially for the iPhone users (like me). This update, though, is not that big. For example 1.x -> 2.x introduced a lot of new stuff, but 3.0 doesn't have that much to it. Small this fixed and added here and there, but other than that, nothing new..

For iPod Touch users, there is not much new, but I think you'll like it better than 2.x :)
