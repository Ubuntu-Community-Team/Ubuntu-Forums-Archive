### Post by accooper on 2009-05-04
Is it mounted?  If not do so.  If you would like to keep this from happening again go to add/remove programs and look for a program called Disk Manager.  Install this and run it. Tell it which drives you want mounted at start up. I have all my hard drives mounted at start up.

Hope This helps.
