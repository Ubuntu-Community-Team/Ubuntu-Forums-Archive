### Post by grahammechanical on 2014-06-18
You might be helped by TestDisk. I used it to recovery image files from a partition I could not access. My advice is not to search for all files but to limit the search by each file type that you want to recover as TestDisk copies the files to the partition it is being run from and you need enough space in that partition to hold all the data. As much space as is used by the data. I found it less confusing to recover only certain file types and not every single file on the disk.

[http://www.cgsecurity.org/wiki/TestDisk](http://www.cgsecurity.org/wiki/TestDisk)

Regards.
