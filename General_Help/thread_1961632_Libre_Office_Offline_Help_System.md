### Post by texpat on 2012-04-19
Using Libre Office I've noticed that there is no offline help system like Open Office used to have. If you click the help icon in Libre Office a browser takes you to the online help files which are, quite frankly, not so good. No functions reference to start with and the search system isn't all that useful.

I've tried looking for something I may have missed to install, but to no avail. Is there an offline help system at all?

---

### Post by lykwydchykyn on 2012-04-19
Do you have the libreoffice-help-<language> package installed? (where <language> is the language code for your language, e.g. en-us, en-gb, es, etc)

---

### Post by texpat on 2012-04-19
Thanks for the hint lykwydchykyn,

installed libreoffice-help-en-us with syanptic and got my offline help (after a reboot).

Funny it didn't install it to start with, though.
