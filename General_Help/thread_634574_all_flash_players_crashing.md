### Post by Dynaflow on 2007-12-08
Go into Synaptic Package Manager under System -> Administration, search for flashplugin-nonfree, right-click on its checkbox, and select "Remove completely."

Then download and install [ this]("http://http://launchpadlibrarian.net/10761023/flashplugin-nonfree_9.0.115.0ubuntu2_i386.deb") from Launchpad.
