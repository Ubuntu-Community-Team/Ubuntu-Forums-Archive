But with Ubuntu, I have this problem: with 1440x900, I can't see the  whole desktop (just tow thirds; the other third of desktop is hiding to  the left and the rest of the monitor screen is black). So, it doesn't  auto-adjust when I use this resolution. The best resolution I can have  with the whole desktop fitting perfectly in the monitor screen is  1280x800. And when I go back to this resolution after trying the  1440x900 resolution I see an auto-ajust message.

Also, if I make a screenshot (with the "print" key) while in the  1440x900 resolution, it will give me the whole desktop, even thouh I  can't see it!

Of course I can survive with a 1280x900 resolution, but it's a pitty not  to be able to use 1440x900, since my monitor can handle it (as it does  with Mac OS on the same computer).
