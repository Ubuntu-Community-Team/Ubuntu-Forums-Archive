### Post by unutbu on 2009-09-12
Gabox01, actually, a postgresql package should be available in the "main" Ubuntu repository. How about open a terminal (Applications>Accessories>Terminal) and type
```

apt-cache policy postgresql
lsb_release -a
cat /etc/apt/sources.list
```

and post the output. This might help us get your situation straightened out.
