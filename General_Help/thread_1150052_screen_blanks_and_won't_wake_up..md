### Post by cmnorton on 2009-05-06
Try pressing CTRL+ALT+F1. You should see the X-term (text-mode) login.
 
Then press CTRL+ALT+F7, and see if your Ubuntu screen comes back.
 
Check your screensaver settings as well. This sounds like launchpad material. Occaisionally on bootup, I'll get the same thing with the prior release on my laptop. Once in a great while I have to shutdown and reboot. I attribute it to software, not hardware.

### Post by BslBryan on 2009-05-06
Hello!  :-)

Well, on a lot of computers, the sleep function does not work.  

In System --> Preferences --> Power Management,

make sure that it, in the "On AC Power" tab, does not say "Put computer to sleep when inactive for x amount of minutes."

The sleep function decides when to work on my machine, but luckily, even though it sleeps after inactive, it wakes up beautifully all the time.  I just hope that with any upgrades, the same thing can happen. :P

Hope I helped.  Good luck.
