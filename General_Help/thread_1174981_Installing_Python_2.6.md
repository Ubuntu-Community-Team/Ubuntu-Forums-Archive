### Post by ad_267 on 2009-05-31
Python 2.6 should be installed by default. 

And you don't have to compile it to install it if it's not there, you can just install it from the Ubuntu repositories with "sudo apt-get install python" or from Synaptic.

And if you really do insist on compiling it for some reason, you need to install the build-essential package.
