### Post by Land Rover Series 3 on 2010-09-25
You can add a ppa to your system which will allow you to automatically install later versions without all the hassle of doing a manual install. Check out:

      [https://launchpad.net/ubuntu/+ppas?name_filter=thunderbird](https://launchpad.net/ubuntu/+ppas?name_filter=thunderbird)
For example, version 3... is available from [https://launchpad.net/~mozillateam/+archive/thunderbird-stable](https://launchpad.net/~mozillateam/+archive/thunderbird-stable). "You can update your system with unsupported packages from this untrusted PPA by adding ppa:mozillateam/thunderbird-stable to your system's Software Sources". It also gives instructions on how to add the ppa.

Hope that helps.
