### Post by ki4jgt on 2011-04-27
Put in the CD, restart your computer. allow it to boot into the system, choose install Ubuntu. Follow the step by step instructions on screen. When it asks how you want the hard drive, choose the circle which says use entire disk. Then continue with installation.
