I wanted to state that I used wubi on a Dell Latitude D510 and that it worked flawlessly. I don't really have a need for the software as working with an installation DVD and managing my partitions with GParted is easy enough for me, but I think it is a great initiative for new users. The only remark I have is that I  didn't see an option for installing from DVD/CD medium. Personally I would like to encourage you to try to make a MS Virtual PC clone that lets you make an ext2 or ext3 (or even fat32) partition, so you can run Ubuntu under Windows without having to reboot. And just copy paste from one OS to another. That or a Cygwin clone, where you can build Ubuntu from core to top without too much difficulty. I am confident you can (halving seen Wubi), I can't though, I lack the skills.
