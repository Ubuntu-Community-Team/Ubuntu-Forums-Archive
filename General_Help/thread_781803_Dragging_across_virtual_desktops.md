### Post by rune0077 on 2008-05-04
First, if you haven´t already, install compiz-config settings manager. Then launch it, and go to "Rotate Cube". There's a setting somewhere here called "Edge Flip" or something, which needs to be checked.

If you're using desktop wall instead of the cube, there's a similar setting for that.

Hope it helps.
