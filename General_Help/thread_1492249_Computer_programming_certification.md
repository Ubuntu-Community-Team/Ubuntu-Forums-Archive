### Post by WorMzy on 2010-05-25
Java and C++ are your best bets. In the case of Java, it's multiplatform due to the JVM, so you'd be able to write applications for Linux, Windows, Unix, OSX, etc. C++ can be compiled on Windows, Linux, etc, but it's not as easy to write multiplatform applications. C++ is the more powerful language, and it's definitely more of an industry standard than Java, but again, depending on what you're trying to do, it may not be the best language to use.

At the end of the day, I'd say that C++ would be the better of the two to learn. Python and Java are very similar, and writing Python is a lot faster than writing Java, so learn C++ and Python you're covering the best of two worlds.
