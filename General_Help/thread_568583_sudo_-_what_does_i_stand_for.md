### Post by bodhi.zazen on 2007-10-06
See man sudo 

> Sudo (su "do") allows a system administrator to give certain users (or groups of users) the ability to run some (or all) commands as root or another user while logging the commands and arguments.

[http://www.gratisoft.us/sudo/](http://www.gratisoft.us/sudo/)

su = substitute user 

If no user is specified it defaults to root or "super user"

> su - run a shell with substitute user and group IDs

[http://linux.die.net/man/1/su](http://linux.die.net/man/1/su)

So I think it's "switch user do", which sounds less grammatically correct, but I think it's more accurate.
