I'm not sure why you want to move files to the /usr folder but you can do so by opening a terminal and typing "sudo nautilus" without the quotes.  Then move them in nautilus.  The path for files on the desktop would be /home/username/Desktop.  The file paths are case sensitive.  I don't know anything about the google talk or msn messenger.  The best way to do voice calls in linux is probably with openwengo or skype.
