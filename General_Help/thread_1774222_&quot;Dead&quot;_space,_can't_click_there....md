### Post by mc4man on 2011-06-03
There has been quite a long history of this and related issues going back to Feb.
It's changed several times and had some fixes, though has reappeared in limited fashion for some.

One of the more common ways to occur is thru the use Alt+tab, either with the "Application Switcher" or "Static Application Switcher" compiz plugins
If you do use then refrain from, if you don't then maybe disable which ever one is enabled.

If disabling in ccsm it may or may not unset all the enabled plugins - safer to do in gconf-editor
(Instructions on how can be provided if needed.
