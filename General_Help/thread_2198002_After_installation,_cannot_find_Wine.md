### Post by efflandt on 2014-01-06
The link you posted does not work unless the "-and" is removed. But unless you know what you are doing, why not just install **Netflix Desktop** the easy way using **Software Center** (shopping bag icon) or netflix-desktop using synaptic if you have synaptic installed? That installs everything you need and updates automatically when you do other automatic updates. I have not added any ppa's to 12.04. Occasionally I have gotten an error when starting Netflix Desktop that it cannot find a suitable gecko version (mozilla Internet Explorer substitute), but if I just wait, it usually still works.

Especially when new to Ubuntu the Software Center should have most things you need, should install them properly, and updates are often automatic. Although, I am used to synaptic, so I install that from the Software Center and usually use synaptic to find programs by their Debian package name (or text terminal method using: sudo apt-get

While it is good to eventually learn how to do things yourself from added ppa's and sources, they try to make it easy for beginners with Software Sources.
