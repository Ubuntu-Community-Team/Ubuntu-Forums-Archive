### Post by untappedpilot2 on 2008-12-11
What do you know LimeWire is really easy to install on Ubuntu..

1. First of all make things easier on yourself by going to [http://www.java.com/]("http://www.java.com/") and downloading the latest version of Java. (LimeWire needs this runtime environment to function.)

2. Go to [http://www9.limewire.com/download/LimeWireLinux.deb]("http://www9.limewire.com/download/LimeWireLinux.deb") and download the latest limewire .deb package.

3. Open the terminal and cd to the .deb location.

4. ```
sudo dpkg -i LimeWireLinux.deb
```

5. Enter your admin password if prompted and it'll set itself up!
