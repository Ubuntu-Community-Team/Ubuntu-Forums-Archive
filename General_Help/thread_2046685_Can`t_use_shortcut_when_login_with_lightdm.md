### Post by Kopkins on 2012-08-25
What do you mean by shortcut? Are you using keyboard shortcuts? 

When you create a keyboard shortcut for your user you gain the functionality of pressing a few keys and having something happen. But this is just for your user. For example if my username is Kopkins and I created a shortcut to hibernate the computer. The other user on the computer, Larry, would not be able to use the shortcut because it is not set for his user. 

Similarly, when you are at lightdm, you are not yet logged in, so the shortcuts you have set for your user are not in effect. 

I hope all that makes sense.

Kopkins
