### Post by hikaricore on 2007-03-20
Actually there is a way to recover files from ext2/3 file systems with data recovery tools such as Phoenix Linux, IF and only IF they have not been overwritten already.  I accidently nuked my whole home directory when I was trying to remove an empty user folder in /home.  It was worth paying for as this was where my collection of MP3s were dating back to the golden years of MP3.com..  simply priceless rare music.

This of course required the use of windows which I still had at the time, I probably could have done it with wine, but I would have risked overwriting my data had I not shutdown the instant I wiped it. >.<
