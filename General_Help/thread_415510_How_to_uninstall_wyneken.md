### Post by karl1 on 2007-04-22
In the root directory of the wyneken-0.4.tar.bz2 file, there is a script named uninstall.sh. You can run this by going to that directory and typing:

./uninstall.sh

Furthermore, any software that you install outside of the Ubuntu packaging system (.deb, synaptic, etc.) will not appear in add/remove programs. (This is to the best of my understanding. Anyone who knows better, please correct me.)
