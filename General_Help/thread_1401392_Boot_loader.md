### Post by hawaiian1der on 2010-02-08
In your case, which I've been in many times before, I would just delete everything when you re-install Windows Vista and, if you know how to partition, make a new partition and re-install Ubuntu. Then again, this all depends on if you have things on Ubuntu that you want to keep or don't mind re-installing them when you get it up and running again. My advice is to delete Windows Vista completely and do nothing but Linux (i.e. Ubuntu).

I don't see the point in upgrading from an earlier version of Ubuntu. If you intend to do this though, you need to open the Update Manager and there should be a button at the top that says, " There is a new version of Ubuntu, would you like to install?" At least it should be something like that. just click the button next to that question and wait.

Hope this helps because this is my first long post that is helping someone and not asking other people what to do.  :)
