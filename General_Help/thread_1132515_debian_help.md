### Post by amingv on 2009-04-21
You may download any ISO you like, from the sets of 5 you can do with the first (wether it's KDE or GNOME)

From the webpage:
> The first CD/DVD disk contains all the files necessary to install a standard Debian system.
To avoid needless downloads, please do not download other CD or DVD image files unless you know that you need packages on them.

The other disks contain additional software that you may get if you want and/or can't get from their repositories (e.g. if you don't have an internet connection)

Any of the first 600-somethingMB cds should do.

### Post by Yellow Pasque on 2009-04-22
Debian Lenny/5.0 is the latest Debian 'stable' version. Desktop (i.e. non-server) users generally find the software in that distro too old for their liking, and use the 'testing' version (currently called 'squeeze'), perhaps with a blend of packages from Debian 'unstable' (a.k.a. sid).

There's also distros based entirely on Debian sid (like Sidux). If the 'unstable' thing scares you, keep in mind that the latest Ubuntu often uses software from there as well as Debian 'experimental'. Personally, I found Sidux to be a very good compromise between stability and up-to-date software, though it has some downsides (doesn't officially support GNOME, package management is supposed to be done from the command line).

Experiment and find what works for you.
