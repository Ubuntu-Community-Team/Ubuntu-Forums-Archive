### Post by sports fan Matt on 2008-02-21
No Script and Adblock are Firefox addons..they can be installed through here

No Script blocks certain scripts from executing: [https://addons.mozilla.org/en-US/firefox/addon/722](https://addons.mozilla.org/en-US/firefox/addon/722)

Adblock plus is a tad better and  blocks banner ads so Firefox responds swiftier..: [https://addons.mozilla.org/en-US/firefox/addon/1865](https://addons.mozilla.org/en-US/firefox/addon/1865)

Adblock plus  has an auto update extension too: [https://addons.mozilla.org/en-US/firefox/addon/1136](https://addons.mozilla.org/en-US/firefox/addon/1136)

Hope these help..

Firekeeper uses flexible rules similar to Snort ones to describe browser based attack attempts. Rules can also be used to effectively filter different kinds of unwanted content.
