### Post by minhmeoke on 2008-07-30
No, wubi will not overwrite the default ms bootloader (ntldr), it will only add an entry for ubuntu. [https://wiki.ubuntu.com/WubiGuide#head-ed7efcda824fe7fc942b07067694e69bc65d6628](https://wiki.ubuntu.com/WubiGuide#head-ed7efcda824fe7fc942b07067694e69bc65d6628)
