### Post by Yellow Pasque on 2008-10-31
Can you start it through a terminal?
```
gimp
```
If gimp is installed and works properly when launched from the terminal, then you can add the launcher to the menu yourself with alacarte.
```
sudo apt-get install alacarte
alacarte
```
