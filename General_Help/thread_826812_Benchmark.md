### Post by y-lee on 2008-06-12
There is a program called bonnie++ for HD tests. To install use synaptic or type the below into a terminal:

```
sudo apt-get install bonnie++
```


> [Bonnie++]("http://www.coker.com.au/bonnie++/") is a benchmark suite that is aimed at performing a number of simple tests of hard drive and file system performance

See also [Linux hard drive benchmark & bottleneck testing software suite for performance]("http://www.cyberciti.biz/tips/linux-hard-drive-speed-benchmark-software-howto.html")
