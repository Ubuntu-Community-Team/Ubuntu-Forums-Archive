With the new card, it seems to run fine except for one small issue. The polygons (mainly on npcs and my own character) are being stretched off the screen, making them look like large spiky lumps. I logged into guildwars after swapping cards, and there is no corruption in the graphics. Any ideas?

### Post by lowprofile on 2007-12-03
I had the same issue with an 8500 gt. Seems to be a problem with the 1.00.xx nvidia drivers.
The way I got round it was to install 'oldblivion' - [http://www.oldblivion.com/](http://www.oldblivion.com/). The idea is to make the game work with old hardware - guess that means it doesn't use the bugged features of the new drivers. Works a treat.
