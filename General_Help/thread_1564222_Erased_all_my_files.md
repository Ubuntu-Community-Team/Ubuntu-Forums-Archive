### Post by howefield on 2010-08-30
> **PianoGrand said:**
> When I wrote Ubuntu OVER windows xp, by doing so, did that eare all of my personal files?

Installing Ubuntu OVER Windows XP by definition means you have overwritten your files that were contained in windows. They are gone.

There might be some that can be recovered if they haven't been over written yet. Have a look for info on testdisk and photorec. But you really want to stop using the disk.

### Post by svenskmand on 2010-08-30
Also another word of advice. If you ever wreck you OS installation again (be it Windows or Ubuntu) then just booting from the Ubuntu-CD - that is using it as a live cd and NOT installing Ubuntu onto you hard drive - is a easy way to recover files from you broken OS installation.

PS: this will not help you now, but hopefully you will not make the mistake of overwriting your the files you want to recover, to try to recover them :S
