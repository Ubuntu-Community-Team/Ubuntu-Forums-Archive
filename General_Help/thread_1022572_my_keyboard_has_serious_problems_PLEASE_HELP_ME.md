### Post by HereInOz on 2008-12-27
Just try a new keyboard first.  That way you will know whether it is keyboard, other hardware, or software.  No point in chasing shadows, so replace the keyboard first up.

This will sound strange, but some PS2 keyboards have a very marginal connection with the PS2 socket - the pins are only just long enough to connect, and if the keyboard is not plugged in firmly, strange things happen.  

If the keyboard is one of those which are USB/PS2, with the little green adaptor on it, plug it into a USB port - you could have developed a troublesome PS2 port.

You must verify the integrity of your keyboard before chasing software issues.
