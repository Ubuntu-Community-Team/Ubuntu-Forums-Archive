### Post by stengah on 2006-10-18
Perhaps Ubuntu Media Center is what youre looking for... it provides a web interface, but I haven't tried it. Do a search for it in the forums.

Personally I've had great expereices with Slimserver for music streaming. Works excellent! There's probably a guide here somewhere. I've made one on my personal page [http://www.evolver.dk/blog/category/dk-web/]("http://www.evolver.dk/blog/category/dk-web/")
but it's not in English :-( - the install commands are universal, though! Again, you can probably find an excellent how-to here as well.

If you're looking for an ambitious package that supports audio, vidio, pics etc, you might wanna consider building a media center with Linux software like MythTV, Frevo or Elise. The latter is pretty new and yet somewhat unstable - but looks great with the Apple-like eye- candy. I've done a lot of research for this Media Center thing, and from what I've heard Freevo probably handles music files a bit better than MythTV, so my next big experiment is building a Freevo media center. 

Anyway, hope you'll figure something out - good luck!;)
