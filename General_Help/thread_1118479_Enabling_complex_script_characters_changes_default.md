### Post by VCoolio on 2009-04-07
Have you tried KKBSwitch? It is maybe dated and meant for KDE, but works also fine on Gnome for me. It will detect the keymaps you enabled in keyboard settings and you can set a keybinding to switch between them and also you can set it to change keyboard only for the active applications or window.
Of course this will mean another (very light btw) process running, but I don't know what decent command you could use for creating a launcher. Setxkbmap messes things up (see my unanswered question [here]("http://ubuntuforums.org/showthread.php?p=7025509#post7025509") folks).
