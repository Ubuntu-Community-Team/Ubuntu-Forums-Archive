### Post by mb_webguy on 2009-04-09
> **iam_newhere said:**
> does ubuntu tutorials like the official ubuntu book also apply with xubuntu(xfce environment)? 

thanks!

Terminal commands are the same regardless of what version of Linux you use, and certainly regardless of the desktop environment.  So anything dealing with the terminal will be the same.

Xubuntu uses a few of the same default applications as Ubuntu, so some of that information might apply.  I think Xubuntu uses the Synaptic Package Manager, for example.

But anything dealing with the desktop, and most things dealing with applications or system configuration GUIs will be different.
