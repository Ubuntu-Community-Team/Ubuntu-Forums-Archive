### Post by Architeuthis on 2006-11-03
I think you should use the wine version of wine hq, you can find a official repository for wine (which you can ad to your sources list) on winehq:
[http://www.winehq.com/site/download-deb](http://www.winehq.com/site/download-deb)
This way you always have the latest version of wine, available for ubuntu.
This one is better then automatix because automatix uses the version from the standard repositories (which is older then the wine repo), atleast i think it does.
