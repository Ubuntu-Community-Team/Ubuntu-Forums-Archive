### Post by jerenept on 2010-08-12
All built in :)! you just need to install the restricted extras, and you can play virtually any media file on the planet.

Open the Software centre, and search for "ubuntu-restricted-extras"

Now select the "ubuntu-resticted-extras" package and type your password.

Personally, I think that the interface that comes with Ubuntu is excellent, but if you do not like it, you can always visit [http://gnome-look.org/]("http://gnome-look.org/") for cool themes and wallpapers.

Ubuntu come with the latest Firefox pre-installed.
