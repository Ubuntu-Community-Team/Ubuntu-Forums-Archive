You can only run some Windows software via a compatibility layer called "Wine", not natively in Linux. Not all windows software will run well on wine. To see if your win 7 software will run in wine search its name in the Application Database link at winehq, [[COLOR=#0000ff]--link to winehq site--[/COLOR]]("https://www.winehq.org/"). Regards, yeti.
