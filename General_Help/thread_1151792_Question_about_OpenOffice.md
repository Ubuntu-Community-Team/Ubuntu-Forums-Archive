### Post by m_2009 on 2009-05-07
OpenOffice Base isnt installed by default in jaunty. You can install it in synaptic by searching for openoffice.org-base or open up a terminal window and run the following command

```
sudo apt-get install openoffice.org-base
```

Hope this helps.
