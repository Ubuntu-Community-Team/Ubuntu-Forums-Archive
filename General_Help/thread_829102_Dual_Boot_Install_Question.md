### Post by konungursvia on 2008-06-14
You are correct, but before proceeding, cancel the installation and boot into Windows to perform a disk defragment at least twice, to move all the Windows files to one side of the drive, physically speaking. Then you have virtually no risk of data loss on the MS side.
