Scraggy,
            The older versions of Quickbooks Pro that can run on W98 can be installed in Ubuntu using Crossover Office. Quasar is very good but is more for a shop type business, I tried it and have a howto if you need it for Ubuntu, but need something for a service type business with invoicing, cash slips and Quotes and Quickbooks does all that. It does not allow customising of your invoices etc. Gnucash is available, but is quite old and needs some serious updating to make the GUI more modern and user friendly, it is also more of a Quicken type program. If someone decides to bring it up to a clone of Quickbooks it will be a winner on linux. There is a chap in Australia that is busy writing a package that can run on OpenOffice Org 2.0.1 and it is a first release and still needs some work I think to get it up to scratch with the likes of Quickbooks, but you can have a look at it here: [http://www.ninthavenue.com.au/products/gemstone/downloads](http://www.ninthavenue.com.au/products/gemstone/downloads) 
The beauty here is that this will be cross platform and will be able to run on any linux or windows installation with OOo installed. If this one could be made as user friendly as Quickbooks I will be the first to switch over!
