### Post by PDA1 on 2011-12-16
I've had various speed problems with LibreOffice 3.4.3 most were manifested as slow speed and having to wait after adding or deleting rows in Calc.

The problem of slow Row Insert has been solved.

I'm running Ubuntu 11.04 and the above mentioned problem seems typical to 11.04

I deleted 3.4.3 which had been installed via Synaptic and the PPA stuff  that had to be added.  Also deleted was the ./LibreOffice folder and an  old ./OpenOffice folder that contained who-knows-what but I read  somewhere to delete it...so...I did.

Then, I installed the following version;

LibreOffice 3.3.4 
OOO330m19 (Build:401)
tag libreoffice-3.3.4.1

This version was found at- [http://www.libreoffice.org/download/](http://www.libreoffice.org/download/)
The file was- [LibO_3.3.4_Linux_x86_install-deb_en-US.tar.gz]("http://download.documentfoundation.org/libreoffice/stable/3.3.4/deb/x86/LibO_3.3.4_Linux_x86_install-deb_en-US.tar.gz") 146 MB

Once the file is downloaded and extracted you MUST read the README file  found in the main folder.  It describes how to install LibreOffice.  If I  can do it....so can you.  It's very easy and doesn't require all of  that esoteric ./configure, make, make install stuff that works for me  about 1% of the time.
