### Post by wyrless2002 on 2008-11-12
If you're still having problems, you might check this thread out:

[http://ubuntuforums.org/showthread.php?t=967281&highlight=CUPS-PDF](http://ubuntuforums.org/showthread.php?t=967281&highlight=CUPS-PDF)

Also, (sorry if you already know this...) you can try right clicking on the thumbnail view, go to PROPERTIES, click on the OPEN WITH button, and add evince document viewer, or Adobe Reader, or whatever applications you're using to view and edit pdf's. Don't forget to click the one you want to use as default, especially if there is more than one in the menu/box.

When I upgraded to 8.10, I had a problem with not being able to open FOLDER file types, so I couldn't get to my HOME folder from the Places menu. Some sort of a loss of file (type) associations.
