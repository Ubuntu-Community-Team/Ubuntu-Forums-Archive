### Post by lykeion on 2010-12-12
One easy thing you can do (if you have a numpad) is to set this to control the mouse.
1. Open Keyboard Prefrences (System > Preferences > Keyboard)
2. Select Mouse Keys tab
3. Enable "Pointer can be controlled using the keypad"
This will set the following numpad keys:
Num+ -> left-click
Num5 -> right-click

If you need more configurations you should be able to use xbindkeys and xdo. You can search this forum or google for how you will do this.
