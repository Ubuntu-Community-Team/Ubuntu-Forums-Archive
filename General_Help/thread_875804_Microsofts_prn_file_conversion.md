### Post by mb_webguy on 2008-08-04
I hate prn files...  I hope you know what kind of printer it was intended to be printed on, because otherwise you might be out of luck.  Here's the FileXT description:> When you print to a file the program outputs whatever it would have sent to the printer you have attached to your computer to a file instead. The idea is that later you might want to send this file to a similar printer on a different computer. So, the main use for a .PRN file is to copy it to a printer in order to get the output. You can open a .PRN file in any text editor but what you'll likely see is your text surrounded by a whole bunch of "junk" which represents all the various printer codes necessary to set margins, special spacing, graphics, etc., etc.

And here's the FileXT page on [how to print from a PRN file]("http://filext.com/faq/print_from_prn_file.php").  As for converting the .prn to something else, you might find something in [this thread]("http://ubuntuforums.org/showthread.php?t=747801").
