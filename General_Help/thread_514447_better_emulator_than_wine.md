### Post by aysiu on 2007-07-31
Wine isn't an emulator. It is a compatibility layer that allows you to run some Windows programs. The main difference from an end user's standpoint is that an emulator is slow and a compatibility layer runs at native speeds.

What exactly are you trying to do? You said you don't want the "whole frigging OS X or Windows operating system." Well, first of all, you won't be able to run OS X-only programs. Sorry. If you want to run Windows programs inside Ubuntu, you have a few options:

1. Use Wine
2. Use Crossover Office
3. Use Cedega (mainly games)
4. Use VirtualBox or VMWare to run Windows inside Ubuntu
5. Dual boot with Windows
6. Get a second computer and a KVM switch to switch between the Windows computer and the Ubuntu computer

1-3 will work for a lot of programs but not all of them. 4-6 will work for all programs, since you are essentially running Windows.
