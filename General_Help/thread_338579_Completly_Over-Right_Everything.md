> **paulyj1124 said:**

That depends on your friends disc.  You don't need the same install disc you used (factory pressed cdroms are hi-tech stone tablets, they have no way of recording new information) as activation is based on the hardware profile of your computer being tied to a CD-key, but you do need an install disc that will work with that code.  There are several different flavors of XP install discs out there (dell oem, corporate, retail, home, professional, etc, etc,) and codes for one flavor don't work for another.

I recommend installing Ubuntu while leaving room on the Hard Drive (as in don't partition all of the space).  You can then try and install Windows in the unused space afterward.  Before re-installing windows, look into how to reinstall grub once Windows squishes it.  Good luck.
