### Post by rsteinmetz70112 on 2020-12-14
As far as I can tell mc, and putty are in the repositories although you may have to enable the universe repositories also chromium is available which is Chrome without a lot of the google bits.

Open a terminal window (Ctrl+Alt+T)
```
$ sudo add-apt-repository universe
$ sudo apt update
$ sudo apt install mc putty chromium-browser
```

### Post by grahammechanical on 2020-12-14
Google Chrome web browser is not in the Ubuntu repositories. We download it with this command


```
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
```

And we install it with this command

```
sudo apt install ./google-chrome-stable_current_amd64.deb
```

[https://linuxize.com/post/how-to-install-google-chrome-web-browser-on-ubuntu-20-04/](https://linuxize.com/post/how-to-install-google-chrome-web-browser-on-ubuntu-20-04/)
