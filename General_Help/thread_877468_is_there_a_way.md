### Post by lisati on 2008-08-01
Sometimes there are Linux equivalents to Windows programs, some come in both Windows and Linux versions. When there isin't a Linux version, a program called "Wine" can be used to run *some* (but not all) Windows programs.

Have a look [here]("https://help.ubuntu.com/community/Wine") and [here]("https://help.ubuntu.com/8.04/switching/applications-equivalents.html#id2625380")
