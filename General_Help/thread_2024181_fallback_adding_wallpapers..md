### Post by Petro Dawg on 2012-07-13
Assuming I understand your situation...

The default wallpaper changer in ubuntu only changes wallpapers stored in the root/usr/share... folder.  

To easily get other wallpapers you downloaded to change,  go to the software center and download "Wallch" (be sure to add it to your start up applications to have it work automatically when you start up).  It does a pretty good job of wallpaper auto switching.
