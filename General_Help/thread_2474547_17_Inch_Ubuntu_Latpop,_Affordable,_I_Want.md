### Post by dragonfly41 on 2022-05-02
My suggestion (not a recommendation) is to consider either:

adding an external SSD (in USB 3.0 connector) to a working laptop
or
adding an external SSD (in USB 3.0 connector) to a working PC

I use the second.
The reason: I had too many failed laptops difficult to repair.
With a tower PC you can get inside a lot easier.
And if you have USB 3.0 port you can install Ubuntu on an external SSD (that is what I'm using with Windows on the internal HDD). The only caveat. I use an external power unit to power the SSD docking bay.
