### Post by Cheesemill on 2013-05-04
You should really download the CD for 12.04 or 13.04 and do a fresh installation instead.

Feisty hasn't been supported for 5 years now and although it's theoretically possible to do the several upgrades you would need to get to a supported release I can almost guarantee that it won't work as so much has changed.

To do a fresh installation will only take you 5 minutes and an 800MB CD download, to upgrade through all the different versions would probably take you a full day and several GB of downloads.
