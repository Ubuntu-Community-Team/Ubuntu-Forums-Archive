### Post by Brandon Williams on 2010-12-01
Use one that's intended to be a programmer's editor, which rules out nano. vim and emacs are the most common console-based programmer's editors, for good reason. I've used both, they're both very good, they both have pluses and minuses, and I don't intend to start the typical "flame war" that results from claiming that one is better than the other. I suggest that you spend one hour with each one and stick with whichever one you decide is proving to be easier to learn.
