### Post by yetimon_64 on 2016-04-28
I could do that with gnome-terminal and its transparencies and using gdevilspie to strip the window decorations and control the placement in 14.04 unity; but not in 16.04 unity as I can no longer control the terminal window naming. It was by being able to set each terminal profile with a separate window name that I could then use gdevilspie to differentiate between the different profiles of the terminal. I assume you don't want all instances of the terminal to look like that.

What version of Ubuntu are you running?  If you are on 14.04 then yes, otherwise I am still trying to "hack" my way through the new 16.04 release to find a way, but no luck as yet.
