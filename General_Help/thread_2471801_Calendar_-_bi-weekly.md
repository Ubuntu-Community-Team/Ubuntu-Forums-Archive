### Post by Holger_Gehrke on 2022-02-10
The calendar in both Thunderbird and Evolution can do repetition after an user defined number of days. Thunderbird actually has '14 Days' as one of it's predefined repetition periods. So if you use either of those for email anyway, you might want to start using it's calendar, too.
