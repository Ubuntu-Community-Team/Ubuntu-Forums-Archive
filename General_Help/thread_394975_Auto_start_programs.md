Is there any way to make programs launch at startup?
i.e. Just as in the StartMenu > Startup system employed by Microsoft Windows

---

### Post by kpagpa on 2007-03-27
Hi,

Yes, if you go to  system ---> preferences -----> Sessions and click the tab 'startup programs', you can add any programs that you want to start up when your Ubuntu boots up.  

You simply have to type the package name, or the name you would have to type in the terminal to get it to run.

Good luck!
