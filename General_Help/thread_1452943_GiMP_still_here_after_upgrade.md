### Post by kurp on 2010-04-12
From Wine site I know that CS is the last one, which runs in Wine almost perfectly. Most of functions of Photoshop CS2 (95%) was also fine. Next editions cannot be used with Wine without serious problems. You'll not be able to use CS4.

Upgrading Ubuntu does not cause loosing already installed applications (which is OK, I think :)). There would be no GIMP if you started with totally new installation. If you updated only - you should remove it manually.

### Post by Tamlynmac on 2010-04-12
> Another thing, does anyone know about any good photoshop alternative except gimp?

Not exact a replacement but something you wish to investigate. Easily handles photos (thumbnails, etc) and has numerous filters for improving photo imaging.

[http://www.xnview.com/en/index.html](http://www.xnview.com/en/index.html)

Just for reference, if you wish to try it:
Down the file and use the archive manager to extract the files. I made a folder called XnView and extracted the files to that folder. To start Xnview simply start the file xnview located in the bin folder. The first screen you will get will be a small setup screen which must be completed prior to using XnView. Once you've completed the setup screen and close it, the next time you open XnView it will open the browser, based on your setup settings.

The screenshot page will provide you will a good view of the browser screen. The number of formats is impressive and formats can be convert  (by saving) without leaving the program.

Good Luck
