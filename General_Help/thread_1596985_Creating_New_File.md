### Post by WorMzy on 2010-10-14
Go to ~/Templates (create it if it doesn't exist), and create a new file called "Txt File.txt".

"Txt File" should now appear under the Create Document submenu.
[[IMG]http://img844.imageshack.us/img844/3788/menuwy.th.png[/IMG]]("http://img844.imageshack.us/img844/3788/menuwy.png")

Choose it, and it'll create a new file with the .txt extension.
