### Post by tuxxy on 2008-08-07
Yes it looks like MSN, you need to get the new aMSN version 0.97.2, download the 32/64bit package here

[http://www.getdeb.net/app/aMSN](http://www.getdeb.net/app/aMSN)

Also you could try emesene as thats  similar to MSN, search for emesene in synaptic or type this in terminal

```
sudo-apt-get install emesene*
```

If you dont like either of them give virtualbox a try, search for it in synaptic or type in terminal

```
sudo apt-get install virtualbox*
```

Heres a guide to installing an OS in virtualbox

[https://help.ubuntu.com/community/VirtualBox](https://help.ubuntu.com/community/VirtualBox)
