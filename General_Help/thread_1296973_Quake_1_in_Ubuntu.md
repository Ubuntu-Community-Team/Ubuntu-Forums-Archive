### Post by white-zilla on 2009-10-21
you have 2 options.

a) install and run it through wine.  There is a TON of great documentation on running windows software through it. This is a great place to start

[http://www.winehq.org/](http://www.winehq.org/)

b)install a virtual OS.  Use a program such as "Virtual Box" to install a virtual copy of windows on to your machine, and then install quake through that.

Obviously wine will take up less of your resources and you dont need a full copy of windows, so i would recomend trying that first
