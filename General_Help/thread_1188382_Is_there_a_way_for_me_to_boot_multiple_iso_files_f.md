### Post by brncao on 2009-06-15
Download Unetbootin! No burning required at all. You can only do it one by one, no menu that shows the entire list of OS, but it still accomplishes what you need: Carry your ISO's around and install from a USB drive. Simply upload the ISO image into Unetbootin, and set the target to USB drive. Upon reboot (make sure USB is set as #1 in the boot order in the BIOS) you'll be greeted by the Unetbootin bootup screen. I don't remember the full details so check their official site guide on how to do it. Then you'll be running off the USB in no time!
