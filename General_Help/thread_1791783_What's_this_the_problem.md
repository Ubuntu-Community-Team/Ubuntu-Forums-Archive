### Post by grahammechanical on 2011-06-27
If you click on the line 80 GB Hard Disk you will see a lot of information about that hard disk. You will also get the power to do things to the hard disk, such as format it. It is for your protection that a password is required to do these things.

Are you set up on the system as Administrator or as Custom (giving you limited administrator privileges) or are you set up as a Desktop User. See Users and Groups. If you are set up as a Desktop User then you will not be allowed to make any changes to the operating system. This arrangement is to prevent unauthorized people from making changes.
