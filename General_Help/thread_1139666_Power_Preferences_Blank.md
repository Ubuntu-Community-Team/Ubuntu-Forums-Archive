### Post by gettinoriginal on 2009-04-27
I do not have this problem, so just throwing something out there that might help.  This helps in blank drop down menus, so it might help you.

System > Pref > Appearance > Fonts set rendering to "subpixel smoothing" then go to details set smoothing to "subpixel" and hinting to full.
