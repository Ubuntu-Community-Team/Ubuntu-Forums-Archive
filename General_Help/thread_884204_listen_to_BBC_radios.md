### Post by le1 on 2008-08-08
This post is for all of you who don't want to spend hours of figuring out how to listen to BBC radios on Ubuntu.


All you have to do is follow two steps:


1. install vlc, go to:

system --> administration --> synaptic package manager

type: vlc --> install it

or 

application --> add/remove


2. download attached file, unpack it, start vlc 

go to 

view --> playlist --> manage --> open playlist...

locate unpacked file... 


you're done.


have a good listing experience...
