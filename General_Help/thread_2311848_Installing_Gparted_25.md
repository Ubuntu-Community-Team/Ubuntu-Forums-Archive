### Post by sudodus on 2016-01-30
The quick fix would be to download a gparted iso file from

[http://gparted.org/download.php](http://gparted.org/download.php)

If you want to install another version than the one in the Ubuntu repository, I suggest that you remove that version before you start installing a new version. It should be possible to use a version from a deb package as well as to compile it from source code.
