### Post by Rob Sayer on 2014-08-16
It should be there ... I've installed synaptic a bunch of times and never had to enable any exotic repos to do so.  It's in the standard tested ones.

Try a different download mirror server.  I use the main server because the Canadian one isn't quite as fast.

You should be able to go into terminal and install synaptic like so:

sudo apt-get update
sudo apt-get install synaptic

which is how I install things if not using synaptic.  I haven't used muon or ubuntu software center or any other similar things in ages.  Synaptic is one of the reasons that if I wasn't using ubuntu it'd be Debian.  Great tool.  I really like the log/history capability.

I just looked for tor in synaptic and it's there.  I'm using xubuntu 14.04.
