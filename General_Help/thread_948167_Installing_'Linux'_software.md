### Post by OutOfReach on 2008-10-15
No, you can install other software from places other than the Package Manager, but they must either be a .deb file or an archive of files (.tar.gz, .tar.bz2, etc...)

The .deb files are easier to install, but the archives can require them to be compiled.

You can also add repositories to download third-party software from the package manager.
