### Post by jsowoc on 2009-04-22
From what I remember, MSN want you to pay to use MSN mail outside of your web browser. They justify it that if you hooked it up to Evolution, you wouldn't be viewing their ads.

As a work-around, you could set up another e-mail account (eg. yahoo, gmail, aol), and forward all your MSN mail to that new account. You could then set up Evolution to work with the new e-mail provider.

### Post by jsowoc on 2009-04-22
Update: It's possible to configure Evolution to use Hotmail via POP:

> Hotmail POP3 settings:
Field - - Value
Incoming POP server - - pop3.live.com
Incoming POP mail port - - 995
POP SSL required - - Yes
User name - - Your Windows Live ID (for example: [email]example555@hotmail.com[/email])
Password - - The password you use to sign in to Hotmail
Outgoing SMTP server - - smtp.live.com
Outgoing SMTP mail port - - 25 or 587
Authentication required - - Yes (your Windows Live ID and password)
TLS/SSL required - - Yes (select TLS if available, if not select SSL)
