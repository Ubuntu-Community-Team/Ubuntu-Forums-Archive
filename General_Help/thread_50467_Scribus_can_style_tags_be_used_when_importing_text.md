### Post by stig on 2005-07-20
I have installed Scribus for DTP of my newsletters but was surprised to find no obvious way of using tags to define paragraph styles when importing text.

All my past experience of DTP has been with Pagemaker. I do the basic writing and editing of articles in a text editor and add style tags ready for import. As long as I have defined a style in Pagemaker, I can use it on import, e.g. if I define "Subhead 1" in PM then insert <Subhead 1> at the start of the paragraph in the text, it will be recognized by PM. (I map the tags to keyboard keys so that inserting them in text is very fast.)

It' such a simple and foolproof method (much more reliable than importing word processor styles) that I'm surpised it's not in Scribus too (espcailly because Scribus seems to be lacking import filters). Or have I missed something? Is it different in Linux?
