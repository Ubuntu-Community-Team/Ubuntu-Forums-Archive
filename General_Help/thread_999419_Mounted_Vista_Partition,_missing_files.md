### Post by TeXtonyx on 2008-12-03
Perhaps you have been posting about this for awhile and I missed 
the thread. Why don't you boot Vista and see if the files are
really missing, I think you have to enable viewing hidden folders.

Doesn't Vista boot? If you have a Vista install cd rather than a
manufacturer provided recovery cd, you can do a Startup Repair.

If not, you can get a Vista recovery cd from
[http://neosmart.net/blog/2008/windows-vista-recovery-disc-download/](http://neosmart.net/blog/2008/windows-vista-recovery-disc-download/)
Now they switched to making a .torrent available so you'll need to 
install a torrent program to download it. 


[http://support.microsoft.com/kb/927392](http://support.microsoft.com/kb/927392)
Describes how to use the milder approach of bootrec (try first).

If you write Vista to the MBR, it overwrites grub, so you have
to reinstall grub or use Easybcd and boot both OS from Vista.
