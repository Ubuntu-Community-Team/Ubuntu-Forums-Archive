### Post by Lekensteyn on 2009-10-30
Leftclick on the icon in the topleft edge of the window.
Choose from the menu "Configure window behavior".
Go to the third tab, "Actions"?
From the "Titlebar and edge" options, change "middle mousebutton" to "To background".

Note: I'm using a Dutch localisation, so the exact translations may differ.
I'm using Kubuntu, with a KDE desktop environment.
