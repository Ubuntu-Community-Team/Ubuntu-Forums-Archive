### Post by Ted3929 on 2009-12-04
I'm using Xubuntu for the first time and find it's a refreshing change of pace from Windows 7, but I'm having a problem.

As usual, I've got a couple hundred programs installed but only use 10 to 15 on a consistent basis.  With Ubuntu I merely right clicked the program title and sent an icon to the desktop but Xubuntu doesn't allow this.  Right clicking merely activates the program.

Why am I doing wrong?

---

### Post by rbc on 2009-12-04
Right click any empty space on the desktop screen, and select Create Launcher. Either in the Name field or the Command field, start typing the name of the application. Xubuntu should start to autofill the application name, or command
