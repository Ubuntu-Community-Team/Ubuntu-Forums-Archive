### Post by Trab on 2006-02-23
yeah, u will get a feel for linux as u mess around...
dont go to ubuntuguide.org its outdated, instead check the ubuntu wiki...
[https://wiki.ubuntu.com/UserDocumentation](https://wiki.ubuntu.com/UserDocumentation)

some very helpful things: synaptic is a package manager, it can find, download, install, and allow u to config packages (programs).

AutoMatix is very helpful to install some of those "odd" packages...

the community here is VERY helpful, if u ask more specific questions, we can help guide you...

for example, the more u tell us about this wifi problem the more we can help (i would suggest starting a new topic, with a useful title  such as "Set up BroadCom Wifi card??" etc....)

ubuntu is based off of debian, so u can use .debs to install packages manually (when u get into .tar.gz and source stuff, it gets messy...)

good luck!
