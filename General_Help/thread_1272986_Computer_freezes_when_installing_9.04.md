### Post by gotix on 2009-09-22
I have installed many linux distributions on my computer and all of them work: Ubuntu 8.04, 8.10, OpenSUSE, RedHat 9, Slax and others.

Now I tried to install Ubuntu 9.10 and the system freezes with blinking caps/scroll locks after loading a bit. I just select language, install, and after a while, it freezes, before showing any other menu.
My motherboard is: Asrock 4COREDUAL-VSTA
My video card is: Asus Extreme AX550, 128 MB, 64 bit
Monitor: ACER AL 1716 A

It's interresting that I have the same problem when I start X Windows (startx) in Slax 6.1.2 (last version). The screen turns black and the caps and scroll locks are blinking. Doesn't react to anything, and I have to reset/shut down computer from the button. Exactly like in Ubuntu 9.10

I suspect the problem is the new version of X Windows that has a problem with some motherboards or video cards or monitors.
