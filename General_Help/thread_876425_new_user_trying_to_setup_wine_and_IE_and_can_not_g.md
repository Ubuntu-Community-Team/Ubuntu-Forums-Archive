### Post by dje on 2008-07-31
in a terminal (Applications >> Accessories >> Terminal) type:
```
sudo apt-get install cabextract
```
press enter and type your password if prompted (there will be no visual feedback)

I must say that ies4linux are not intended for browsing, as the author states on the website. They are intended for web developers to test websites on internet explorer. They are too slow, insecure and old for browsing ;)

hope that helps,
dje
