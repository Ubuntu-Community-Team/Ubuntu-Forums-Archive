### Post by danking12 on 2009-03-25
Hi there,

I have attached an image that is a black box that is probably an appropriate size. Below are the steps you should follow:

1. Create regular bullets.
2. Highlight the text that corresponds with the bullets you just created.
3. Right click on the highlighted text and select the "Numbering/Bullets..." option.
4. In the dialog box that opens select the "Options" tab.
5. Select Level 1 on the left side.
6. In the Numbering drop down box select "Linked graphics"
7. In the Graphics drop down box that appears select "From file"
8. Select the file you wish to use as your bullet.

You can play around with the sizing and placement options to get something that fits best for you. Unfortunately I could not find a way to add custom bullets or use images that are located in your gallery as bullets.

This is definitely something that the OO team should be adding as a feature or at the very least add a square box as a bullet item.

Hope this helps and post back if you need clarification or have any other questions.
