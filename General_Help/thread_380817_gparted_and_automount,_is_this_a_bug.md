```

```

sudo ntfs-config

```

Tick appropriate boxes and restart.

### Post by merlinus on 2007-08-31
Change the entry for it in /etc/fstab in this fashion:

/dev/sda1 /media/sda1  ntfs-3g defaults,locale=en_US.UTF-8,uid=1000,gid=1000, umask=0002 0 2

Substitute your partition for sda1.

-merlin
