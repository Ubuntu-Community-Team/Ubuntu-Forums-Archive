### Post by xlinuks on 2008-06-19
Hi,
I used both the final version of Firefox 3 for Linux and a few hours later I got the source code and compiled it (it took like half an hour or so).
I noticed that the version I compiled seems to be faster, and indeed, after I performed the SunSpider benchmark

[http://webkit.org/perf/sunspider-0.9/sunspider.html]("http://webkit.org/perf/sunspider-0.9/sunspider.html")

it turns the compiled version runs some 10-15% faster.
The downloaded version of FF3 took like 5100ms to complete while the compiled one took about 4800ms (I ran the benchmarks several times), the overall user interface seems/is faster too. Yes I'm aware of the drawbacks of having your own version rather than that from the repos, but if you're a freak like me who likes to squeeze more speed I would suggest compiling it.
