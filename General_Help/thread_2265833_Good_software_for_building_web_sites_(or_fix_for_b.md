### Post by dragonfly41 on 2015-02-18
Although I have Bluefish installed I haven't used it too often.

I had a look and true enough HTML5 tags like <header></header>
were not coloured.

It seems that Ubuntu installation of Bluefish is version 2.2.5.

I've just installed version 2.2.7 and the HTML5 tag highlighting now works.

Bluefish site seems busy so I had to download 2.2.7 from another site.


Other HTML5 compatible editors you might try include Aptana Studio 3 .. but there are too many to list.

I would just upgrade Bluefish.

[later edit]

Here is the first download site I found .. [http://www.linuxfromscratch.org/blfs/view/svn/postlfs/bluefish.html](http://www.linuxfromscratch.org/blfs/view/svn/postlfs/bluefish.html)
