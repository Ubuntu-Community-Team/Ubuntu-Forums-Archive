### Post by cocoa117 on 2010-06-22
I have been looking as well, but I don't think there is anything, not even paid for one. OneNote is unique combining following together
1. handwrite, draw or scratch on paper experience on your digital device
2. combing all sort of information together in one place, one file. handwrite, type text, audio, video, web clip, etc.
3. You can search through many of text formed, even for handwrite
4. simultaneously access by multiple device or person on single file
There are others, but I think those are the main points. Specially the last one, you can have almost identical paper/pen/desktop computer experience as you are working now. Only everything is digitalised, all back up, and easy to make copy.
