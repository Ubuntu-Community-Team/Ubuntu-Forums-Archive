Firefox Bookmarks:  Bookmarks > Organize Bookmarks  then Import and Backup > Export.  You will be able to select a folder to save it to. 

As for the other configuration files, open Nautilus and make sure view hidden files is on.  You'll then see a lot of folders that start with "." and then have a program name.  If you back up the folders for the programs that you want to save settings for then you can restore those folders once you reinstall.  Hopefully it isn't one of those folders that is causing you problems.
