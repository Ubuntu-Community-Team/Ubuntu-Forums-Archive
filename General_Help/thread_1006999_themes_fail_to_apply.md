### Post by eternalnewbee on 2008-12-10
Maybe, seemingly, nothing happens because you have compiz enabled.
If you want to have both, then I suggest you right-click on the **Compiz Fusion Icon**, under **Select Window Manager** select **Compiz**, and under **Select window Decorator** select **GTK Window Decorator**.
If you don't have Compiz Fusion Icon installed, you can do so via **Add/Remove Applications**.
