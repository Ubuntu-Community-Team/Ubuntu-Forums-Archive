### Post by fjgaude on 2009-11-21
I have my main compute setup to not require user ID or password at boot up. When I Suspend and come back out the OS requests my password. Is there a way to have it not require the password?

---

### Post by Andreas1 on 2009-11-21
run "gconf-editor", go to apps/gnome-power-manager/lock and check "suspend"
