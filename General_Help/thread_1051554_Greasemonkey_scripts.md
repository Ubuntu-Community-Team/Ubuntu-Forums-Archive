Does the version of firefox matter- say if the scripts were written only for firefox 2?

---

### Post by icheyne on 2009-01-28
Scripts are specific to different versions of Greasemonkey, but it's always best to use the latest version of Greasemonkey.

You can get the latest versions of the scripts at [http://userscripts.org](http://userscripts.org).

Firefox should automatically upgrade to the latest version of Greasemonkey. [https://addons.mozilla.org/en-US/firefox/addon/748](https://addons.mozilla.org/en-US/firefox/addon/748)
