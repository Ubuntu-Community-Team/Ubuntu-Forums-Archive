I recently installed the PDFedit, but when I use many buttons are inactivated, I dont know why, is not the permissions. any of you know this.

This happens if the pdf file you load into the editor is "linearized". Click on Tools > Delinearize and save the file with a different name. All the buttons should be accessible now. If not, exit, and restart the editor, loading the previously saved delinearized file.
