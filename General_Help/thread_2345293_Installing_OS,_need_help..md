### Post by QIII on 2016-12-02
Hello!

Setting aside the hardware environment for a moment...

You cannot install another OS while already running one.  It does not work that way unless you are installing in a virtual machine -- which is does not sound like you are doing.

USB pen drives are not memory (RAM).  They are storage media.  [Here's]("https://www.backblaze.com/blog/whats-diff-ram-vs-storage/") a pretty good article on the difference between storage and RAM. 

If you want to install another OS, there must be install media mounted when the computer starts up, mountable media to install to and no other OS can be running.

Cheers!
