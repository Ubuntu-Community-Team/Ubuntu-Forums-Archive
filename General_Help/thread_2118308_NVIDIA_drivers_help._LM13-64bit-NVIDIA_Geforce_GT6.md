### Post by Yellow Pasque on 2013-02-20
You probably have Optimus/hybrid graphics. Check output of lspci:
```
lspci | grep -i vga
```

If you see both an Intel and Nvidia GPU, then you need to completely purge any nvidia driver you installed, delete /etc/X11/xorg.conf and reboot. If graphics are now working normally, then you can proceed to install Bumblebee: [https://wiki.ubuntu.com/Bumblebee](https://wiki.ubuntu.com/Bumblebee)
