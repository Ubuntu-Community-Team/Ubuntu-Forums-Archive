### Post by dino99 on 2015-04-13
i suppose that collection is stored inside a windows folder; then copy/paste to a support (usb stick is the friendlyest), then drop that list inside a /home folder (ubuntu)
or from an ubuntu session, use nautilus to mount the windows folder, then copy/paste as above, then unmount the windows one.
