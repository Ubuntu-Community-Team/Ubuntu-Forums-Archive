### Post by jerome1232 on 2008-08-06
this did it for me since no terminal we'll have to do this gui way
place>>home>>press ctrl+h (view hidden files) and find .gconf>>apps>>gnome-terminal>>profiles remove everything here

When you open gnome-terminal again should be fine

(another option would to remove gnome-terminal in synaptic and remove config files there too)
