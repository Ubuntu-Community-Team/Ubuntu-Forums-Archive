### Post by cmost on 2008-01-03
This is a long shot, but do you by chance have an nvidia graphics card?   If so, what version of the nvidia driver are you using and list your system specs.  There's been a nasty little lockup bug in the nvidia driver in versions since 9755.  Sadly, it's STILL in the most current version of the driver.  Certain, seemingly random things, like logging into Yahoo mail or using a Java application (like Frostwire) used to trigger the freeze at times when I experienced this bug.
