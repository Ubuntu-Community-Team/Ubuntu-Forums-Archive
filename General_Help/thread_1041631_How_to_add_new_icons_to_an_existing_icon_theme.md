### Post by jesterthejedi on 2009-01-16
try using the menu to find the icon name. right click on your start menu, choose edit menus, then navigate to the location of the program icon. choose properties and pay attention to the name used in the command box, or left click on the upper icon to see that name used. Then save your icon as that in the apps folder of any icon theme. Make sure you pick the right size folder too.
