### Post by searchfgold6789 on 2012-01-17
Looks like it works with Wine, the Windows compatibility layer for Linux.
That would be the only way you can install it on Linux, being that Guitar Pro 5 is a Windows based program.
Not all software in Wine works so well, so you can check out the page for it here: [http://appdb.winehq.org/objectManager.php?sClass=version&iId=3782](http://appdb.winehq.org/objectManager.php?sClass=version&iId=3782)
