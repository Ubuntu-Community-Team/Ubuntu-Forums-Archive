### Post by Mark Phelps on 2011-01-18
> **Philip R said:**
> I need help, I, m new to Ubuntu. I downloaded 2 games and thy appear in downloads and when i try to run them they just go to extracting files progress bar?

If these are Windows-based self-extracting archives (.exe files), they are most likely NOT going to work in Ubuntu.

Also, you would first need to extract them in MS Windows to see what they are in detail.

Then, if you want to run them in Ubuntu, you would have to install Wine.  Go to the WineHQ website to read up on the details before you attempt this.  Also, use their App Compatibility Checker to look up the games you want to run -- to see if they are compatible with Wine.
