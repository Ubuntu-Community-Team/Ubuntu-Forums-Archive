[Compcache]("http://code.google.com/p/compcache/") is a nifty Google Summer of Code project that compresses a section of memory to use as swap space, avoiding reading and writing to disk. In effect, it magically adds memory. There are [plans]("https://blueprints.launchpad.net/ubuntu/+spec/compcache") to include it in [intrepid]("https://blueprints.launchpad.net/ubuntu/+spec/compcache-usage").

I've tried setting up following the instructions on the homepage, but the use_compcache script errors silently, and I'd like to set it to run automatically on boot.
