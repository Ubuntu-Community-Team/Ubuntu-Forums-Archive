### Post by PurplePenguin on 2007-02-26
I think you mean RTM, not STFU. :D

I don't use VMware, but if you want something to run each time you boot into Ubuntu, you can add the command to the list in System -> Preferences -> Sessions.  This is a list of programs that load automatically when Ubuntu loads the desktop.  For example, I added gaim to this list.
