### Post by WorMzy on 2010-06-27
You have to set them individually. Right click on the folder, click Properties, click the folder icon, then browse to the image you want to use. Default icons are in /usr/share/icons, custom icons are in ~/.icons

EDIT: You'll want to look for a folder named "places". It generally holds the folder icons.
