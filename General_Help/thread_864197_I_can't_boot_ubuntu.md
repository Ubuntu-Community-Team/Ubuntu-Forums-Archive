### Post by Taxman415a on 2008-07-19
it sounds like there were errors either in downloading or burning the iso. When you boot the disc try the option 'Check cd for defects'. Probably you need to use new media and burn it again. I have much better luck burning isos to dvd+rw media than cd-rw media, I get much less errors. I like to use rw media because I like to update my livecd's and installation media often.

Also you should verify the iso image you downloaded is correct in the first place. download an md5 utility for windows (can't think of one off the top of my head, maybe it's md5summer or something) and look for the md5 hash for the image where you downloaded it from. They should match exactly or else there was a problem in downloading it.
