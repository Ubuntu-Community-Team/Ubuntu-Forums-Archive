### Post by chrisccoulson on 2008-05-09
Where are you attempting to delete files from? If it is from your home folder, then they go to ~/.local/share/Trash. What is the output of:
```
ls -l ~/.local/share
```
..this will check the permissions

If you are deleting files outside of your home folder, then the files will go to one of two folders on the root of the filesystem that you're deleting from. The folders will most likely need to be created by the root user if you don't have write access to the root of the filesystem that you're deleting from.
