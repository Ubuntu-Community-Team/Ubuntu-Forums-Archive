### Post by PaulW2U on 2012-05-08
As Rodney9 has posted there's no longer an edition of Ubuntu specifically for netbooks. Both Lubuntu and Kubuntu do offer such a version though.

Beware if running Ubuntu on a netbook, some of the configuration dialogs are too big for a small screen and the "OK" button is often off-screen. Unity's Keyboard Shortcut Overlay that should be displayed when pressing the Windows key won't display due to the small screen size and it's a bug that won't be fixed.

Otherwise Unity runs fine. I haven't tried Gnome Shell or the classic mode though.
