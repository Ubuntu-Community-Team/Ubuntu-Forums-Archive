### Post by zami on 2009-01-23
I think Ubuntu would be a really good fit. On a fresh install, there isn't a lot that doesn't work right out the gate, and what doens't work... well these are some superb forums for asking for help.

Are there particular programs they need to run, like Microsoft Office, Photoshop, etc? Or just particular activities they need to perform, like browsing online, watching DVDs, and so forth?

If there are particular programs you might want to look them up in the Wine application database
[http://appdb.winehq.org/](http://appdb.winehq.org/)
An absolute favorite program not running can be a real deal breaker for some people.

I've seen a few (someone can probably link to one) lists of comparable Windows/Linux software.  For example, if your Mom likes Winamp for listening to internet radio, Winamp wont work but there are some comparable replacements.  Either consult one of those bigs lists (google for "windows linux equivalent list" etc), or just ask here "what can I use instead of...." and you'll get good suggestions.

On a fresh Ubuntu install, the first thing I would do is follow this thread
Comprehensive Multimedia & Video Howto
[http://ubuntuforums.org/showthread.php?t=766683](http://ubuntuforums.org/showthread.php?t=766683)
- this will get youtube, DVDs, CDs, etc going quickly. 

I find Ubuntu totally novice friendly.  I learned Linux on Ubuntu and keep coming back to Ubuntu because, if I have a problem, I finally am familiar enough with Ubuntu I can do a bit of troubleshooting on my own.

About 4 months ago I installed Ubuntu on an old PC for a friend who was deathly afraid of computers (had just never used one) - she's an internet junky now and gets a kick out of the piddly dink games like GnomeTetris, listens to internet radio and watches previews online and DVDs on her machine, etc.  She hasn't had ANY questions or problems for me, and I'm positive that would not be the case if I'd installed WindowsXP for her.  (I'm certain she would have downloaded some cute yet lethal cursor or screensaver by now!)

-zami

### Post by mb_webguy on 2009-01-23
I'd highly recommend Ubuntu.  Ubuntu focuses on usability, and is an excellent distribution for users with little or no experience with Linux.  

Ubuntu has good compatibility with most hardware.  Printer setup is not quite as straightforward as on Windows, but a huge variety of printers is supported.  

The Gnome desktop environment is sufficiently similar to XP that users should have little problem moving from one to the other.

For basic computing, Ubuntu comes with all of the software you'll need:  OpenOffice.org for word processing and other things you'd find in Microsoft Office, Totem for playing DVDs and media files (though I personally prefer VLC), Rhythmbox for playing music (though I personally prefer Amarok), the Firefox web browser, etc.  (Of course, they'll be using different software than they did in Windows, but it will do the same thing in essentially the same way.  OpenOffice.org's Writer, for example, is very similar to Word.)  For DVD and media playback, you will need to add the Medibuntu repository and some essential packages as described on [this page]("https://help.ubuntu.com/community/Medibuntu").   

Installing software from the repositories is ridiculously simple using "Add/Remove Applications".  Applications are automatically organized by function, rather than on Windows by whatever each particular developer sees fit to organize it's software, so even if your parents aren't familiar with the name of specific applications, finding the application for a particular task is easy.

Basically, if I were setting up a computer for my computer-illiterate grandmother, I'd install Ubuntu.  (And actually, I did.)

### Post by HyperHacker on 2009-01-23
I built my brother, who knew little more than how to use Firefox in Win98, a Xubuntu machine and he loved it. The only problem I believe he had was not getting sound to work in ZSnes (and I have no idea how to fix that either <_<). I don't think he does much more than surf the web (mainly Youtube and Flash games) and play games, but it was just what he needed - fast, stable, and easy to use. (The hard drive was defective though, so it crashed a few days ago. :()

HOWEVER, Ubuntu is great ONLY if it supports your hardware out of the box, or you know a nerd who can get it working. That's Linux's biggest problem right now. (And it's really the manufacturers' fault, only making Windows drivers and refusing to give out any source or documentation.) Especially if it's something like a video card (doing everything from the command line is not for newbies!) or networking (hard to download drivers for the device you need to access the Internet), it can be a real pain. Sometimes it's not obvious either. Something doesn't work on the live CD; you need to reboot to install the driver, but you can't do that on a CD, so you just go ahead and install anyway and discover the driver doesn't actually work.

tl;dr: Install it to a removable disk or something and see if you can make everything work. If it does, just go ahead and install it right overtop of Windows, you almost certainly won't need it again (unless you're into games that don't work in Wine).
