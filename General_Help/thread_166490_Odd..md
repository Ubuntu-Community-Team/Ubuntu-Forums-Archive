### Post by nanotube on 2006-04-26
depending on your hardware, there are various utilities you can install to manually control your fans and stuff. for my dell laptop, for example, there is package i8kutils. if you have something else, there is probably some other package of tools.

if you do end up turning off your fan, though, you definitely want to configure a cpu temp sensing applet, to make sure your cpu does not run too hot.
