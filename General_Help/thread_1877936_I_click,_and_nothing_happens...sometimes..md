Hi Everyone,
So I started using ubuntu about 2 months ago and now have it set up and running probably as well as any OS can really run given the number of modifications and complex networking requirements that we have here at my university. I use Ubuntu 11.10 64-bit at work, and Ubuntu 11.10 32-bit at home on my laptop. I have an array of small issues which each setup, most of them unrelated to Ubuntu itself (networking problems at work, and display/proprietary video card related issues at home with my HDTV). But there is one thing that seems consistently problematic between both computers. The mouse in Ubuntu just doesn't work as well as it does in Windows!

I've never heard this brought up anywhere before, but it's not an entirely unimportant issue. when I drag a window across from one screen to the other, I want the mouse pointer to be on the same place on the window (ie. the bar at the top) as when I started dragging, not half an inch, 2 inches, or half a monitor up or down and sideways. Also, when I click on a button like "x" at the top of a window, it's fine if the window disappears and nothing else happens, but when I click "shutdown" for example, I want the computer to shut down, not just for the window to disappear. This is a consistent problem, and I often have to click "shut down" a second time for it to register, and it's a weird problem because the click is obviously being registered (the shut down prompt disappears), but nothing happens.

Anyway, just a little rant. Let me finish by saying that Ubuntu is really growing on me, and I'm very glad I made the switch from Windows. Other than out of my virtual box, I really couldn't imagine running windows as a primary OS anymore. Having said that, it's a shame that there are still some, often simple, things that windows just does better. This is a biggie, simply because it affects everyone...
