### Post by sa-mejia on 2008-04-17
There is a Linux application (or rather a perl script): Lock-keyboard-for-baby.pl.  It locks the keyboard, while leaving the mouse enabled.

You can download the script from here: [http://linux.softpedia.com/get/Utilities/Lock-keyboard-for-Baby-23745.shtml](http://linux.softpedia.com/get/Utilities/Lock-keyboard-for-Baby-23745.shtml)

After you have download it, you can run it by simply writing in the command line: 

perl lock-keyboard-for-baby.pl 


The application (at last in my system) opens a tiny window in the top left hand side which says "Type 'QuitNow' to quit".  Which suggests, rightly, that to unlock the keyboard you must type "QuitNow".
