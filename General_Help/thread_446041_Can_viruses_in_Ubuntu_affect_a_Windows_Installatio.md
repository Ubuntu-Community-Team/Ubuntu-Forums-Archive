### Post by LaRoza on 2007-05-16
The file would not run in linux, however, if it were run in Windows it would do whatever it was designed to do.

If you have an Antivirus program, you should use it to protect Windows. If you share a lot between Linux and Windows, maybe you could set up a seperate partition, or disk, for the files to make virus detection less of hassle.

If the virus is never run in Windows, it won't affect it.
