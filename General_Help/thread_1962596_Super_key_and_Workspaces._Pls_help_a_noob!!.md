### Post by zombifier25 on 2012-04-21
The Super key is the key that has the Windows logo on most keyboard. Super - S will show all workspaces, allowing you to drag a windows to another.

### Post by raja.genupula on 2012-04-21
ok do this 

```
sudo apt-get install compizconfig-settings-manager
```

now click at General options at Left side and again general options   at the window then open that . now click at desktop size then set the value to 2 for both horizontal and vertical desktops . then 
now check with **super -s**
