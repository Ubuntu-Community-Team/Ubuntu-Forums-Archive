### Post by pauljz on 2008-11-03
I know this is kind of "is it plugged in?" advice, but make sure numlock is toggled on if you want to enter numbers with the numpad.  Failing that, it might be a problem with the keyboard layout you selected upon installation.  In System -> Preferences -> Keyboard -> Layouts try selecting a different model/layout that's appropriate for what you're using.

As for the shutdown issue, where does it hang or freeze?
