### Post by nerdtron on 2015-01-18
1. I think it would be better to have all usernames in small letters (easier to type and easy for scripting)
2. If this is the primary user account (the first account created on ubuntu installation) and if this is a fresh install, then it would be better to just reinstall ubuntu and just use your desired username.
Changing the first username can get tricky (could involve permission issues and command line) and is not worth the effort especially if since the reinstall process is just a few minutes.
