### Post by seuaniu on 2007-02-08
Check out kino.  If you have the universe repository enabled, it ought to show up in synaptic along with an "extra effects" plugin.  It can control some cameras via firewire, and has most of the basic features you'll find in windoze movie maker.  I've used it before and had a decent time with it, but at that time (3 years ago or so) it didn't have a couple of features that I needed.

If you want something a bit more hardcore, I'll second the nod for cinelerra.  Its got a bit of a....umm.... unique interface, but is wicked powerful, and has a neat renderfarm feature that you can use on other computers in the house to get that transcoding done quicker

Good luck
-seu
