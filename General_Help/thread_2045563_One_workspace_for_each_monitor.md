### Post by hojjat on 2012-08-21
I just added a second monitor to my device. I have also have two workspaces, which appear side-by-side. I want to know how I can assign each workspace to one monitor so that when I send a program into the other workspace, it would show in the other monitor.

Technical details:
ATI Radeon HD 3470
Not using ATI Catalyst (Ubuntu's free driver works just fine for me)
Monitors are Dell, each connected using an HDMI cable, both have the exact same resolution
Ubuntu version 12.04

If that is not possible, then I need to know if there is at least a way for me to use keyboard shortcuts to send a window to the other monitor.

---

### Post by hojjat on 2012-08-21
Okay i found the answer. In CCSM, I enabled PUT plug-in in Window Management and assigned a keyboard shortcut to "Put To Next Output".
