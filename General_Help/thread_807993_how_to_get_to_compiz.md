### Post by alsamman on 2008-05-26
If you can't find the Advaned Desktop Settings in System>Pref then you can install it:

```

sudo apt-get install compizconfig-settings-manager
```

Then you can change compiz and customize it from Advanced Desktop settings. To activate compiz press alt+f2 and type: compiz --replace
