### Post by ian-weisser on 2024-01-29
> **Tom_Carr said:**
> I am getting a message from the software updater that I need to enable ubuntu pro to install some updates.

"Need" is inaccurate.

Ubuntu Pro is an optional service that includes security patches for Universe software (Universe = Additional software you installed. Not part of a stock Ubuntu install.)

For some users, it's just as simple to uninstall unused software and/or other basic mitigations.


> **Tom_Carr said:**
> I go to that screen that tellls me to "enter code on  ubuntu.com/pro/attach".  There is a code, but nowhere to enter the code  on that screen.

You are being told to open a web page. Do so.

Enter the code on that website. The field is clearly labeled on that web page.
