### Post by loomsen on 2008-10-11
Right, the color choser gnome ships with doesn't have too many options.

I'm sure you'll enjoy the gnome color chooser application tho, which povides some more options.

```

sudo apt-get install gnome-color-chooser
gnome-color-chooser &

```

Enjoy :)
