### Post by lmedland on 2008-12-14
Hi,

I have installed Ubuntu 8.10 on a USB thumbdrive by using the USB startup drive wizard from a Live CD. It works great and is really useful to carry around and use on any hardware. However, when starting up Ubuntu it logs in using the default Live credentials instead of prompting for a user account. I have setup a user account for myself, but it means I have to log out of the default account and then log in as myself every time I start Ubuntu from the thumbdrive.

Do you know how to disable the default logon account on a Live Ubuntu image on a USB thumbdrive?

Thanks for reading!

Luke

---

### Post by nielz on 2008-12-14
In the System->Administration->Login Window->Security tab there should be a setting called "Enable Automatic Login" which you can disable (or perhaps change to yourself).

### Post by lmedland on 2008-12-15
Thanks, that worked! :)
