### Post by BertN45 on 2011-05-11
It all depends on what you like to do with programming and networking and what you like to program. 

In general Programming is a profession and you need a college education to really learn it good. Besides programming you have to understand the area for which you develop the programs.  Once I worked for a computer chip manufacturer. To be able to specify and write programs I needed to learn a lot about the manufacturing of those computer chips and all the problems a production department has.

If you see it as a hobby I would choose something high-level, probably python is a good choice for Linux.

For networking you have to understand the layering/stacking. Start with the OSI stack or TCP/IP stack and understand what each layer is doing. Layering is essential to understand networking and it is used also in other areas of computing.
