### Post by Tomosaur on 2006-09-01
It's not possible that it got bricked simply by plugging it in - unless he had a destructive script running which messed with mounted drives. He may have done something completely stupid like formatted it or something though, but he would have had to have done that himself.

### Post by 3rdalbum on 2006-09-02
Loading my mother's iPod with Linux was the only way I could get songs onto it (in such a way that they would play).

Tell your friend that iPods are either formatted as Fat32 or HFS, and that those filesystems are vulnerable to corruption even when you use Windows or Mac OS. Tell your friend to use the program that comes with the iPod to "restore". If that doesn't work, and if it's still under warranty, he should return it.
