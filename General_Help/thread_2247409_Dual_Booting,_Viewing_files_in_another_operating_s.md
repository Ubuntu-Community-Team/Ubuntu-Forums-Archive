### Post by ajgreeny on 2014-10-07
No, there has been no real change; Linux is still more clever than Windows and can see the files from a Windows OS, but not the other way round.

There are some third party ext drivers available for Windows that used to allow it to read and copy files from a Linux (ext#) formatted partition, butwhen I used it some years ago it was very unstable; things may now have changed but as I no longer have Windows I do not really know more about this.
