### Post by jrrader on 2009-10-04
The updater is called ubuntuzilla.  You'll need to get the [deb package]("http://sourceforge.net/projects/ubuntuzilla/files/"), install it, and then run ```
ubuntuzilla.py
``` in terminal.

[Also]("http://www.google.com/search?q=howto+update+firefox+ubuntu&ie=utf-8&oe=utf-8&aq=t&rls=org.mozilla:en-US:official&client=firefox-a")
