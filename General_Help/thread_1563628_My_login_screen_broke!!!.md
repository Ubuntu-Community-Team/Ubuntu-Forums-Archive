### Post by bergfly on 2010-08-31
Hopefully this is just a theme file problem. So log in on the test with your username and passsword and then type ```
startx
```

That should bring up the full KDE desktop. Then simply go back into the System settings and reset your login screen to the default. Once that is done it should all be working again, unless there are bigger issues at play.
