Cedaga is a little tricky.  It is created by transgaming [http://www.transgaming.com]("http://www.transgaming.com") 

They make Cedaga and Point to Play.  Point to Play is not released anymore (to my knowledge).  I never used the demo versions because so many people liked it, but I did pay my $5 per month and now I use it.  It works pretty well.  You sometimes have to check the game database to see which version it works the best with (You can download multiple versions and set up games to run specific versions).  I have been able to play Return to Castle Wolfenstein on it and WarcraftIII Reign of Chaos.  There are forums online where you can find information.  So far I'm not 100% satisfied.  I play Counter Strike: Condition Zero using steam, and thus far have not been able to successfully install it, let alone play it, but I also have not researched it much either.  I say go for it if you have the extra $5 a month, at least maybe we can help them work out the bugs.
