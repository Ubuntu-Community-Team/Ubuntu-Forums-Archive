### Post by dawhoo on 2006-03-29
After having a little trouble getting all of my firefox extensions installed, I thought I would post what I did that worked for me.

If an extension isn't installing as expected (Spellbound), try hitting ALT + F2, type "konsole" and hit enter.  Then, type "sudo firefox" and hit enter.  This will open a firefox browser with correct permissions needed for some extensions. Now, try installing those troublesome extensions. It worked for me, I hope it works for you :D

If this was covered elsewhere, forgive the repeat, but I couldn't find anything about it.
