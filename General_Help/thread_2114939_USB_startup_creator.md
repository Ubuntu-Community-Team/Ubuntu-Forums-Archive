### Post by MakeMineAPint on 2013-02-11
I have installed lubuntu onto my Dell Mini.
How do I create a USB startup. Can't find it in Applications.

---

### Post by sudodus on 2013-02-11
Try if you have (as a terminal window command)

```
usb-creator-gtk
```

If not install it with

```
sudo apt-get install usb-creator-gtk
```

---

### Post by Rex Bouwense on 2013-02-11
I thought that you could use Start-Up Disk Creator which can be found in menu->System Tools->Start-up Disk Creator.  I believe that is installed by default.  If nothing else works you can always download, install, and use UNetbootin.
