### Post by chantaspell on 2012-05-26
Found the solution to this on my own. Reinstall ubuntu and then follow these instructions to install Bumblebee, which works a treat!

```
sudo add-apt-repository ppa:bumblebee/stable
sudo add-apt-repository ppa:ubuntu-x-swat/x-updates
sudo apt-get update
sudo apt-get install bumblebee bumblebee-nvidia
```
From: [http://pleph.appspot.com/init/posts/view/2687901](http://pleph.appspot.com/init/posts/view/2687901)

Gets graphics card usable (with optirun command) andmore importantly makes battery life comparable with win7!
