### Post by WorMzy on 2010-07-04
No idea what iWeb is, but there's a multitude of software available for creating webpages on Linux. Whether you're a hands-on type person like me who prefers writing the actual code, or if you're someone who prefers working with a what-you-see-is-what-you-get editor, there's something for you.

Gedit is great as a coding tool, you can set it up as a psuedo-IDE for almost any language with the right plugins.

Komposer is a WYSIWYG editor that I see thrown around a lot on these forums, so you might want to give that a look. QuantaPlus and Amaya are two others I've heard of.
