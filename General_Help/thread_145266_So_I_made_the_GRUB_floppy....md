### Post by Herman on 2006-03-16
Hello there, SZF2001, I think you mean you'd like to learn how to use the Grub command line interface. Most people use the Graphic User Interface (Grub Menu), but it is also very handy to be able to use the command line, although it is rarely necessary. Some Grub floppies have only the command line and others include the Grub menu also. I am not absolutely certain, but I think a menu.lst file can be added to the grub floppy to give you the grub menu, it's something I've read and been meaning to experiment with soon but haven't quite gotten 'round to as yet.

Grub is far more than 'just a booter', some say it is really a small operating system itself.
