### Post by NightwishFan on 2009-08-04
Just use a live CD image, and you can copy your data.

The live cd runs from the CD and ram and does not install. (Though it can).

If you have more than one DVDRW drive, you can use Ubuntu to burn your data to DVDs. If you only have one dvd drive, I recommend puppy linux, as puppy can be removed after the live CD loads, and your burner drive can be used.

[http://www.puppylinux.org/downloads/official-releases/latest-production-release](http://www.puppylinux.org/downloads/official-releases/latest-production-release)
