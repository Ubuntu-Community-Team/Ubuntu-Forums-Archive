### Post by seawolf167 on 2011-03-03
Couple options:

[LIST]
[*][ALT][F2] to open the run applications box, then type in the name of the program
[*]Look under the Application menu, if its not there, you can add it by right-clicking the menu, then selecting "Edit Menus"
[*]You can create a launcher for it on the panel by right-clicking and selecting "Add to Panel", then selecting the application, or selecting custom application and typing in the application path
[/LIST]
