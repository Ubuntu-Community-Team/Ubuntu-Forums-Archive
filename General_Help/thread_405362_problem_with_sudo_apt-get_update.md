### Post by jfinkels on 2007-04-11
> **darkenemy said:**
> can anyone help?

Try looking here ([https://help.ubuntu.com/community/Repositories/CommandLine](https://help.ubuntu.com/community/Repositories/CommandLine)) and here ([http://ubuntu-tutorials.com/2006/11/19/how-to-add-extra-repositories-ubuntu-510-6061-610/](http://ubuntu-tutorials.com/2006/11/19/how-to-add-extra-repositories-ubuntu-510-6061-610/)) for more information on editing your */etc/apt/sources.list* file.

However, if you don't feel like reading, and just want to reset your */etc/apt/sources.list* here's the point:

In general, your standard, run-of-the-mill sources.list will look something like this (as stated in the last link above):

```

deb http://archive.ubuntu.com/ubuntu edgy main
deb http://archive.ubuntu.com/ubuntu edgy-updates main
deb http://security.ubuntu.com/ubuntu edgy-security main

#deb http://archive.ubuntu.com/ubuntu edgy universe
#deb http://archive.ubuntu.com/ubuntu edgy-updates universe
#deb http://security.ubuntu.com/ubuntu edgy-security universe

#deb http://archive.ubuntu.com/ubuntu edgy restricted
#deb http://archive.ubuntu.com/ubuntu edgy-updates restricted
#deb http://security.ubuntu.com/ubuntu edgy-security restricted

#deb http://archive.ubuntu.com/ubuntu edgy multiverse
#deb http://archive.ubuntu.com/ubuntu edgy-updates multiverse
#deb http://security.ubuntu.com/ubuntu edgy-security multiverse

```

Note that the way I have it set up here, only the main Ubuntu Edgy Eft repositories are enabled. To enable the universe, multiverse, and restricted repositories, simply edit the file */etc/apt/sources.list* with a text editor such as "gedit" or "nano" and remove the hash (#) from the beginning of each line whose repository you want to enable. I hope this helps!
