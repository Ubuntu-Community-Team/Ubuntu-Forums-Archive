Similar to [thread=1026264]this post[/thread] I am not sure if this is the correct forum for this kind of post, but I have created a useful addition for the Firefox search plugin.

Download the following file, change the extension from .txt to .xml (UF won't allow me to upload .xml) and save it to the **searchplugins** folder in your Firefox profile. The next time you start up Firefox you will be able to use the search toolbar to search these very forums.
