### Post by adriantry on 2005-08-10
MIDI isn't an audio format, so you can't rip to MIDI.

A MIDI file contains the notes and volumes to be played by another "instrument" (like a music keyboard, or a software synthesiser on a computer). In other words, a MIDI file is like sheet music for a computer. You can't just rip a CD to sheet music.

There have been a couple of complex programs written to attempt this (Windows programs), but they are not really of any practical use.

You're options?

1. There are a lot of excellent MIDI sites out there, so you might find the song you want if you google for it.

2. If you know anyone with musical talent, you might be able to create the MIDI file yourselves by playing it with a music keyboard into a program like Rosegarden or Muse.

3. Give up on the specific song you were hoping to use, and explore the MIDI files that are already available on the net.

Hope all of this helps.
