### Post by labinnsw on 2010-04-18
1. Go to System >> Preferences >> Sessions
2. Make sure the Startup Programs tab is selected
3. From the buttons on the right, select Add
4. In the Name with, give the command a name.
5. In the Command window, type/paste your command.
6. In the comment window, describe your launcher (Optional)
7. Click OK
8. Click close.

You are done.
