### Post by ActionParsnip on 2021-11-02
The trash icon holds deleted files that you didn't SHIFT + DEL to delete. This icon gives easy access to the user's recycle bin and functions exactly how it does in Windows. Deleted files are held in the bin until it is emptied.
The Gnome team are pretty against desktop icons so the trash has been shifted to the launcher bar on the side of the screen (As far as I can tell. I'm just an Ubuntu user, just like yourself).
