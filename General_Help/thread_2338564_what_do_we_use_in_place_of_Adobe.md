### Post by kurt18947 on 2016-09-30
I've run into problems printing bank statements from one bank, not others. The problem appeared to be that Firefox has its own PDF viewer and the problem bank didn't like it. Save the problem bank's PDF file then click to open from the desktop opens with Ubuntu's default PDF viewer and it worked.
