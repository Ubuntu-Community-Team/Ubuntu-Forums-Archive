### Post by Favux on 2012-03-20
Hi joshgimer,

Welcome to Ubuntu forums!

You should be OK installing Oneiric.  Plus Precise is almost out.  The TX2 TouchSmart (with N-trig digitizer) should be pretty much supported.  If you want you can then put some more effort into touch gestures.  See the [N-trig HOW TO]("http://ubuntuforums.org/showthread.php?t=1252492").  If at all possible set it up as a dual boot.  You want to keep Windows for BIOS and firmware updates.
