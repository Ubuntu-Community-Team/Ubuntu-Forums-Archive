### Post by marc321 on 2007-06-19
Hmm.  Well, what type of image are you working with?  Is it something you created in OpenOffice, a picture taken by a camera, a scanned image, etc?

Normally, OpenOffice isn't the tool of choice for working with images.  The GIMP would be a better choice. GIMP is an advanced photo editor, similar to Photoshop in ways, that will allow you to save images in a web-friendly format.  It's found under the Graphics submenu of the Applications menu.  Post back if you need help with it. :-)

If you created the image in OpenOffice (as in a vector image in Draw), use File -> Export... and in the dialog that pops up, click "File Type" and choose PNG, JPEG, or GIF.  PNG would probably be the best for a drawing.

Hope this helps.
