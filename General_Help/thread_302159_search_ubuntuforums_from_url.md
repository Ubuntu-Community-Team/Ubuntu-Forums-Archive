### Post by bollix47 on 2006-11-18
Right click in the search box and select "add a keyword for this search".  At least thats how it works in Firefox.

Give it a description and a keyword like u (for example).  Then type "u searchterm" in the url without the quotes and replacing searchterm for whatever you want to find.
