### Post by jrusso2 on 2008-05-17
You can make the .bin file executable by running the chmod a+x RealPlayer11GOLD.bin "or whatever the file name is"command from a terminal window. - Run the .bin file by typing "./RealPlayer10GOLD.bin". Make sure you are in the same directory as the file then,  Follow the prompts provided to finish installing the player. - When you launch the player for the first time, a set-up assistant will take you through configuring your player.
