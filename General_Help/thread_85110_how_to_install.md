### Post by felix_stegerman on 2005-11-01
If you're careful (you should always be that when partitioning) and know
which partition it is, you could delete it manually and then ask the ubuntu
installer to use the free space, I think. (I've done all my partitioning manually)

You can always abort the install if you think something's wrong.


Felix

### Post by Kyral on 2005-11-01
Or select "Manually Edit The Partition Table", then select the partition and in the menu that comes up select "Use as" and then choose to use it as the root (/) filesystem
