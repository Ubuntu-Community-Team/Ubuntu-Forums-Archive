### Post by yetiman64 on 2012-02-27
> Is it possible?Yes it is. With ccsm (compizconfig settings manager) disable the side cursor activation (Set "Reveal Mode" to none), then set the "Hide Launcher" to autohide. The mouse won't activate the sidebar but the key combination Alt + F1 will still bring it up (to get it to drop back use the esc key).

Edit: both of these settings are on the Ubuntu Unity Plugin. Be carefull with changing any other ccsm settings with Unity, doing so can cause lockups.
