### Post by johan_tre on 2009-07-21
Hi everyone, 

to keep you guys updated:

I have a solved problme. 
The main reason, which is probably also the the reason why xrandr has no effect on others is the following in my case:

in Nvidia-settings, you can find a setting** "GPU Scaling method"  **
it was set to stretched in my case.
Only when you set it to **"Centered"**, it is possible to change with xrandr the resolution. 
Also with the command **xrandr -s <width>x<height>**  (without the <size> as the --help states)
 So setting the resolution to something, has no effect having this setting to "stretched" or "aspect ratio scaled"

Problem with this setting though is that it is not persisted in any way.
What you need to do to overcome this issue is add a line in the xorg.conf in the "screen" or "Device" section:
[B]    Option         "FlatPanelProperties" "Scaling = Centered"
[/B]
This way, scaling stays to "centered", and has the statement **xrandr -s <width>x<height>** effect.
