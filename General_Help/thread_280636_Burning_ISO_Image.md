### Post by chalex on 2006-10-19
Which release ISO are you talking about?  Did you check the MD5 sum?

Simply run the md5sum executable and compare the numbers with the ones posted at the download site.  This will ensure you have the same exact file. 

You can get the md5sum here: [http://www.etree.org/md5com.html](http://www.etree.org/md5com.html)
