### Post by ddrichardson on 2007-07-25
Well, the first thing to do is open a terminal and uncompress it.

```
tar xvfz filename.tar.gz
```

Once uncompressed change into the directory and find look for and INSTALL or README that will tell you how to procede. It will most likely either be a binary installer you can execute or a source file to compile.
