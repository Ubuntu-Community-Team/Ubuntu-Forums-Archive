### Post by dboot on 2006-10-10
I'm not sure about installing Iceweasel, but to extract a file you can either right click the file -> extract, double click the file -> drag the file inside the .tar.gz onto your desktop, or double click -> "extract" to desktop.

If nothing happens when you double click the .tar.gz file, you may need to install an archiver. [Ubuntu guide]("http://ubuntuguide.org/wiki/Ubuntu_Edgy") and [easy ubuntu]("http://easyubuntu.freecontrib.org/index.html") are great places to start with any version of Ubuntu. Easy Ubuntu will help you install several needed packages such as an archiver.

Basically, you have a compressed file or folder within the iceweasel.tar.gz file. You have to un-compress it from the .tar.gz file by "extracting" it to some location (like your desktop).
