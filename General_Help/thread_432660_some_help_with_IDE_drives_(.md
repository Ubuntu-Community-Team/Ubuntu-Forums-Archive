### Post by disposable on 2007-05-04
I recommend methodical problem solving. Connect only the primary master HD, double-check cables and jumpers, boot. Once that works, connect the next device, double-check cables and jumpers, boot. And so on.

If that doesn't work, try again with different configurations. If _that_ doesn't work, you're probably past the point of diminishing return, have determined some devices just won't coexist and can replace that device.
