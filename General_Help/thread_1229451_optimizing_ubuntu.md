### Post by alexandari on 2009-08-02
```
ps cux
``` in the terminal
if you wanna kill a process
```
kill -9 (the PID number)
```
like - kill -9 3210

you can kill a process easyer by typing ```
killall (process name)
```

like ```
killall mplayer
```
