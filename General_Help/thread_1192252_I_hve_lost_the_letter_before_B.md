### Post by blackened on 2009-06-19
You may be using an incorrect keyboard layout. Go to System -> Preferences -> Keyboard. Select the "Layouts" tab in the subsequent dialog. Make sure that the layout is appropriate to the keyboard you're using (if you're in the US, this should probably be USA). If the layout you're using is incorrect, then you can press the "Add" button to select a new one.
