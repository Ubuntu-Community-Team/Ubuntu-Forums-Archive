### Post by Demon012 on 2006-05-03
Hi here is a link that may help you to get the apache, mysql and php installed.

[http://www.apachefriends.org/en/xampp.html](http://www.apachefriends.org/en/xampp.html)

Xampp is an easy way to install apache, mysql, php, and include phpmyadmin to do database stuff.

After you install this you will need to put your forum in a folder in the htdocs directory and then connect to the local web server using your browser and use any install.php or whatever they provide to setup the forum.

You will then need to just install the board with whatever install script they provide.

And for the other question the easiest way of setting it up so nothing else is working is using a server install when you first install ubuntu.

Hope this helps
