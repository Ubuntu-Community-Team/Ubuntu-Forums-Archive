### Post by elj4176 on 2008-02-08
The tar.bz2 file is just a an archive (think zip file). You should be able to double-click it and extract the files to another folder. 
Some of the broadcom chipsets are supported already. You may be able to go into the restricted drivers manager and enable the broadcom drivers already.
There are several tutorials on here for getting the wireless working (search is your friend) if the above doesn't work for you. I have also have more luck setting up WEP using wifi-radar instead of the network-manager but ymmv.
