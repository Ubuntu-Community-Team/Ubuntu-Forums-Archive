### Post by fyo on 2008-05-19
Using the drop-down under the map, Africa/Algiers is right there, no problems.

The reason it isn't in the "Add" section is probably because **it's alredy in the list** ;-)

You're trying to ADD a new location, which isn't the right thing to do here. You can just select the location from among the already defined ones: System -> Administration -> Time and Date -> (Unlock) -> Time Zone.

The ADD location interface seems a bit odd all around. The list appears to preclude any existing cities (in the Time Zones list), but there are plenty of examples of cities that are there, just with a different full name, e.g. Perth under Western Australia.

But, generally, the main cities are not in the ADD interface. Take Brussels, Belgium or Copenhagen, Denmark as examples. You can add "Zaventem" (where Brussels Airport is located) or "Kastrup" (where Copenhagen Airport is located), but not "Brussels" or "Copenhagen".

Since I'm not sure what the point of the Add interface is, I'm not sure if this is a feature or a bug.
