### Post by shanem on 2015-03-01
Hi,

I have recently installed lubuntu 14.10 and I notice the virtual consoles (Ctrl-Alt-F1) are very slow to refresh. For example, when I cat a file it takes about 4 seconds to scroll one screen. However, I don't have this problem when performing the same actions in a terminal on x windows. My graphics card is ATI Radeon HD 5450. Any clues what the problem might be?

I am not sure where the issue could be. Framebuffer? Kernel Mode Setting (KMS)? The Radeon driver? I don't have much experience in these areas, so if any of you gurus could suggest a way to test, or even rule one of them out - I would appreciate it.

Shane

Update : add GRUB_TERMINAL=vga_text to /etc/default/grub and run update-grub seems to have solved the issue
