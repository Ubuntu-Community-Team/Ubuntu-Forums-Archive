### Post by bwhite82 on 2008-03-13
Search for it in: [http://packages.ubuntu.com](http://packages.ubuntu.com)

Make sure you search for the version specific to your Ubuntu version. Ie, click the dropdown and select 'gutsy' for instance.

Make note of the required dependencies, as you may need to download them as well. You can cross-reference Synaptic if you need to. I've done this many times to download and then transfer packages onto a non-networked Ubuntu install.
