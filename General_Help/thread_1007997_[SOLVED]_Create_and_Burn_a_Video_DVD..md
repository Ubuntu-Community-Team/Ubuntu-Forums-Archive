You can either use Linux or Windows.

Here is how

DeVeDe

In Ubuntu get Devede from the add/remove
install it

Open devede

click make Video DVD

drag files to window

Make sure you click on the Bottom either PAL or NTSC . Use NTSC for the North American TV system.

when done 

click next

and it will ask you where you want the file (ISO) to go
then make note of where the file will go

Then click and it will start the process depending on your computer it may take awhile.

When done go to the file where the ISO was made 

Then open up Nero (in Windows) and burn as a ISO DVD

Or open up Brasero in Ubuntu and burn as a ISO DVD
