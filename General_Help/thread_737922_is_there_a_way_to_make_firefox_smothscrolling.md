### Post by Achtung on 2008-03-28
In firefox, click Edit/Preferences
In "Advanced", click the "general" tab, then tick "Use smooth scrolling".

### Post by y-lee on 2008-03-28
If Achtung's  suggestion is not smooth enough to suit your taste you might want to consider the [Smoothwheel Firefox add on]("http://smoothwheel.mozdev.org/"). The xpi file can be found on [the installation page]("http://smoothwheel.mozdev.org/installation.html"), I had to right click this file and save to desktop and then manually install. It made a difference but eventually i quit using it for no particular reason. But I did like the add on, just trying to cut back on the way too many extensions I use :)
