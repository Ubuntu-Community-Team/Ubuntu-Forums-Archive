### Post by heatblazer on 2009-04-29
Hi
Dunno how it happened but I`ve spotted 3 times a bug related to keyboard layout switching. Actually when they switch from US to Bgr for example, Bgr is displayed but still types in US. The shortcut command for switching is gone also and even if you reenable or add it again the effect remains for a while then vanishes. However - I found a way to fix it - guess it`s related to gnome. Go to
/home/username/.gnome2 and delete the files from there. Reboot and when you add the kbd layouts the bug will be fixed. I`ve seen this on 8.04 LTS and 8.10, I can`t report for 9.04.
