### Post by Maverick_Meerkat on 2010-08-02
Greetings,

Windows boot loader is peculiar and particular. Typically, it is best to install Windows first. Then install Ubuntu. At least, according to Official Ubuntu Documentation.

As a noob, myself, I am not familiar with using Grub as the primary bootloader. On my system Windows bootloader is set to use Ubuntu as the default OS. Hence, Windows bootloader loads Grub, which then loads Ubuntu. Not very elegant, but it works.

From what I understand, this process requires windows partition ("C:\" for example, maybe "D:\" in your case) to have these two files: wubildr & wubildr.mbr

These two files are created by wubi during installation.

To set Ubuntu as the default OS, edit the Windows boot.ini file to something like this:

```
[boot loader]
timeout=3
default=C:\wubildr.mbr
[operating systems]
multi(0)disk(0)rdisk(0)partition(2)\WINNT="Microsoft Windows 2000 Professional" /fastdetect
C:\wubildr.mbr = "Ubuntu"
```

For this to work Windows partition (partition 2 in your case) must be bootable. 

Unless there is a specific reason you wish to use Grub as your primary bootloader, it may be best to go this route.
