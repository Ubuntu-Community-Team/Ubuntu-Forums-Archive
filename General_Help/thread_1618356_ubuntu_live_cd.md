### Post by Rubi1200 on 2010-11-10
If you mean documents and pictures on the hard-drive itself, then the answer is theoretically yes (but you may not be able to open everything).

The LiveCD is an opportunity to test an operating system without making changes to your current setup.

But, because it is running from a CD drive, don't expect it to be lightning fast (which it would be if it were installed to the drive).
