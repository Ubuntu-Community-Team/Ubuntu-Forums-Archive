Are you using a card reader or a USB connection for your PSP's Pro Duo Memory Stick? If it's a card reader, the drivers are mostly not installed, or it's not auto mounting. 

To install drivers the easy way go to System< Administration< Hardware Drivers and make sure you are connected to the 
Internet and have all updates installed when doing this. 

If it's a USB connection, make sure your PSP is in USB mode and that the USB cable isn't damage itself. ;)

To check if it's mounted at all, go to File System< Media.

And for the WINE stuff, I don't use WINE so I wouldn't know.
