What do you mean you don't have any program?? Ubuntu comes with all the essential packages from Office suit, mail client, web browser to media players. What else are you expecting??

> i want to know what are the most important programs i have to install them (( with cods plsssssssssss  ))



The first thing which you need to install is medibuntu repository. Follow the link...

[https://help.ubuntu.com/community/Medibuntu]("https://help.ubuntu.com/community/Medibuntu")

From this link you would like to add the two codecs...

libdvdcss2
w32codecs

After you are done with medibuntu, then you would like to install some more restricted package...

> sudo apt-get install ubuntu-restricted-extras

> and ... can i download them from a site without cods ?? (( i mean as what i do with windows )) . sorry about this questions but im new user in this operating system

You need not to download your packages from any ***website***. You can download your packages either from "add/remove applications", "Synaptic package manager" or from the 'terminal'.
