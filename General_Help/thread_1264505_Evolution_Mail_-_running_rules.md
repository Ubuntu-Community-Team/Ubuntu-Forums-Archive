### Post by hfxrzw on 2009-09-12
Sorry if this is in the wrong forum, but have a question: I have been able to set rules in Evolution, but how do I run them on what's already in the inbox? In XP you had the possibility to run new rules over your mailbox or do that at a later stage, but haven't found how to do that in Evolution yet. Thanks

---

### Post by howefield on 2009-09-12
Edit > Preferences > Receiving Options > "Apply filters to new messages in INBOX on this server"

### Post by howefield on 2009-09-12
Sorry about that, brain wasn't engaged, I missed editing the account. In any event, you want to run filters against mail already in the INBOX, try pressing ctrl A followed by ctrl Y 

That should run the filters on the mail currently sitting in Evolution.
