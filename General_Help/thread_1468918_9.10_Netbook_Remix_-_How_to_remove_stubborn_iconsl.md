### Post by OzVegan on 2010-05-03
Under my user folder I found a file called [I].gtk-bookmarks
[/I]When opened it listedall the bookmarks under my Files and Folders panel. I removed the lines to places I didn't want listed.

I also added a line to make sure I got the right file. It worked.

To view the file on the command line do the following:

[LIST]
[*]```
cd ~
```
[*]```
less .gtk-bookmarks
```
[/LIST]
Or open nautilus/explorer and press Ctrl + H to show hidden files. Open the file in gedit and edit it.
