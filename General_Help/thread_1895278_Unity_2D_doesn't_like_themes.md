### Post by Frogs Hair on 2011-12-14
Unity 2D uses GTK 3 themes Like Unity 3D . Make sure the theme is GTK 3 compatible and post a link to the theme you have installed . Mac 4 Lin and Macbuntu have been known to cause problems for some users . 

GTK 3 themes can be found at the link . [http://gnome-look.org/index.php?xcontentmode=167](http://gnome-look.org/index.php?xcontentmode=167)
