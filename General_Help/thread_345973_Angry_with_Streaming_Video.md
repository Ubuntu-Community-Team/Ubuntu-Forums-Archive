### Post by Knewguy on 2007-01-25
Hello all,

I solved my problem with streaming video and thought it might help others here too.

Within my Firefox browser I had installed an add-on called Media Player Connectivity. This add-on was causing problems so I uninstalled it.

I found another FF add-on called User Agent Switcher...this switches the user agent of the browser...and installed it.

I reopened FF and clicked on a video, the gxine program came up and I had streaming video. :)
