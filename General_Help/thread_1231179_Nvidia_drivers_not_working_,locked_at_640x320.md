### Post by ibuclaw on 2009-08-04
> **Abhishek4563 said:**
> 
this is my xorg.conf
Switch to a Virtual Terminal by pressing **Ctrl+Alt+F1** and login to the console.

Stop the Xserver:
```
sudo /etc/init.d/gdm stop
```

open /etc/X11/xorg.conf with a text editor using sudo and add the following line in bold. Under '**Section "Device"**'
```

Section "Device"
**    BUSID          "PCI:3:0:0"**
    Identifier     "Device0"
    Driver         "nvidia"
    VendorName     "NVIDIA Corporation"
EndSection

```
Save, and run:
```
sudo /etc/init.d/gdm start
```
And let us know if this resolves the issue.
