### Post by mastablasta on 2019-08-05
create & activate under "*How do I add or modify a swap partition?*": [https://help.ubuntu.com/community/SwapFaq](https://help.ubuntu.com/community/SwapFaq)

if you have plenty of ram available, you can also adjust swapiness. that way partition is not used unless you are really getting low on ram. not sure what default ios nowadays.

for example i have OS (server) on USB key  which is not good for having many read & writes. just something a memory swap might do. i also have 4 GB ram and server is usually using less than 512 Mb. so i set it not to kick in the swap file unless ram is taken aproximatelly more than 3.5 GB.

also with enough ram, swap is mostly used for hibernation and similar power management events.

### Post by mastablasta on 2019-08-06
Like USB stick, SSD health also benefits from reduced swapiness.

1.5 GB ram for OS is a lot. not sure what you are using and what you removed. Ubuntu should take arround 400-500 MB on boot. KDE desktop i have takes about 500 MB on boot (64bit) and can easily be used on old 2 GB RAM machine. but you might benefit with lighter OS (desktop) such as Xubuntu or Lubuntu. they should take up have less than or close to 200 MB on boot. especially if installed with basics only e.g. only xfce package instead of full xubuntu. then you just add what you need to run mame.

another option are windows managers such as icewm, fluxbox, openbox... they will take up 100 MB or less on boot, but look a bit more rudimentary. sometimes this is a good option if you don't need a full desktop.

a good linux distro and light on install is MX-Linux. it is Mepis and AntiX tech combined. both are Debian stable based - AntiX aimed at older machines, while Mepis aimed to make Debian easier to use, but it's development stopped.

### Post by mastablasta on 2019-08-07
this is minimal install (net install): [https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)

then you install it (old, but not much changed): [https://www.psychocats.net/ubuntu/minimal](https://www.psychocats.net/ubuntu/minimal)

then you add only XFCE and maybe a desktop manager (lightdm, gdm). Not sure if desktop manager is included or not. it's been a while since i've done it.

```
sudo apt install xfce4
```

and optional:

```
sudo apt install xfce4-goodies
```

more here: [https://itsfoss.com/install-xfce-desktop-xubuntu/](https://itsfoss.com/install-xfce-desktop-xubuntu/)
or here (Debian docs): [https://wiki.debian.org/Xfce](https://wiki.debian.org/Xfce)
