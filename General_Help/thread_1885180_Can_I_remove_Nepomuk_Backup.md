### Post by Jack Brown on 2012-03-06
hi

nepomuk got installed when you installed a KDE application.

It looks like KDE apps depend on it.

If you didnt install KDE as your desktop environment, you can try uninstalling it and check if your KDE based application still works.
if not, reinstall your KDE application.

But better just to leave it since other KDE applications you install next time might need it and just reinstall it again.
