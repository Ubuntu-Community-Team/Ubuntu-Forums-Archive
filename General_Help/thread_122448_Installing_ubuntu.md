About the video card: It will work. NVidia cards are well supported by Ubuntu and work very well. You can install 3D drivers after it's installed by following [this]("https://wiki.ubuntu.com/BinaryDriverHowto/Nvidia?highlight=%28nvidia%29") guide.

About the sound: If your computer was built in the last five years or doesn't include some exotic sound card, Ubuntu definitely supports it. Ubuntu will probably support the card even if it wasn't built in the last five years, so don't worry.

A guide: The [Ubuntu wiki]("https://wiki.ubuntu.com") is your friend. To set up dual boot, follow this guide: [Windows ualBootHowTo]("https://wiki.ubuntu.com/WindowsDualBootHowTo?highlight=%28dual%29%7C%28boot%29").
