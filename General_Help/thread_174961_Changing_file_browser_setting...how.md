### Post by tonyr on 2006-05-12
If you are using Ubuntu (as opposed to Kubuntu), in your file browser
window go to **Edit->Preferences**. Under the **Views **tab, in the
**Default View** section, choose **List View **in the **View new folders using:** setting.

It didn't take effect until I changed directories (folders) in the Browser, but it
seems to work from then on.

### Post by tonyr on 2006-05-12
There is another selection there, **Tree View Defaults**, with a 
**Show only folders** checkbox.  I think that is selected by default.  Unselect
it to show all files in the list.
