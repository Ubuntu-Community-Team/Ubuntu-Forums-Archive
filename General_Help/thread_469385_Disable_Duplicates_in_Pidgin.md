### Post by hikaricore on 2007-06-11
Well I don't know about Pidgin, but in Gaim you can left-click on the duplicate name and drag it to another (same name of course) which nests them.  ^_^
I wouldn't imagine they removed such an amazing feature.

This is useful if you have similar contacts on different screen names.

p.s. If you mess up and drag them to the wrong name like i've done numerous times, you can right-click and choose Expand to show the nested screen names.
