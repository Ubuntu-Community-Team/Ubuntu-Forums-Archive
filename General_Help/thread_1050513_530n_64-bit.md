### Post by fragos on 2009-01-26
Dell ships the 32bit version called "generic" because of 64bit library compatibility for some popular functions. You can of course download the Ubuntu 64bit ISO, make a CD and boot from it. There'll be an install button on the desktop. With "n" Dell versions you get the hardware options that are best supported by Linux and the generaly available Ubuntu 8.10 works out of the box.
