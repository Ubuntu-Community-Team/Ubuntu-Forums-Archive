### Post by boom2k1 on 2007-09-03
I may have found a better solution, and this requires you to download functions for Keyconfig extension from Firefox, and another keyconfig tool from the author's website. You can then configure the keys by going into Tools -> Keyconfig..

What I find useful is that there is the f4kc_FocusNextFrame  utility for you. It is now set as a very inconvenient key, "Meta+Alt+F", but we can change it to whatever we like... 
It probably does what I want, and I am still testing it!
