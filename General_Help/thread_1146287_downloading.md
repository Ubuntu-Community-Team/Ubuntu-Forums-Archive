I made a download folder in my home folder. I set the standard download folder in Firefox to that folder and I can find everything there.

What you have to do once you have downloaded it, depends on the type of file(s) you have downloaded. A .deb file is a file you can install by double-clicking on it. A .tar.gz file is a compressed archive which needs to be extracted first. A .mp3 or .avi file you can double-click and a media program starts to play the file.
So you see it all depends on what you have downloaded.
