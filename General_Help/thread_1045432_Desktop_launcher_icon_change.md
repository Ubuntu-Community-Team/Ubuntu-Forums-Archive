### Post by Rocky1980 on 2009-01-20
Hi all,


I have just created launcher on my desktop that runs my amgia emulator. However I want to change the icon of the launcher ? Is there any way to do this without messing around with the theme (I only want this one icon changed) 

Many thanks

---

### Post by mcduck on 2009-01-20
right-click the launcher, select "Properties" and then either click the icon you see on the left side and find a picture you want to use as icon, or simply drag&drop the new picture over the current icon.
