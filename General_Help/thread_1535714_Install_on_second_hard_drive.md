I want to install Ubuntu on a second hard drive. This drive has a corrupted installation of Windows XP and data files that are important to me.

### Post by deathadder on 2010-07-21
Yes if you choose "erase and use entire disk" the installer will delete the partition, create a new partition for Ubuntu and then format it. I would suggest trying to mount the second hard drive from the Ubuntu live-cd. If it mounts you can copy all of your files onto a external hard drive / DVD etc.
