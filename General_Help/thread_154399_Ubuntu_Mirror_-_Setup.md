### Post by halitech on 2006-04-04
looks like it's just a matter of setting up the rsync to make sure you have the updated files then sending an email to [email]mirrors@ubuntu.org[/email] letting them know which you are setting up and they add you to the list of places to downlaod from. Looks like you will not only need bandwidth but a bit of space, 20 gig for Install CD mirror and 110gig for a full mirror. 

more info here: [http://www.ubuntu.com/download/mirror]("http://www.ubuntu.com/download/mirror")
