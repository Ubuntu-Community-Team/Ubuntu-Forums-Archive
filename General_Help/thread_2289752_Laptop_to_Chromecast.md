### Post by efflandt on 2015-08-07
Are you using the HDMI extender included with Chromecast device. I was able to configure mine with the Chromecast configuration app on my Android phone, but could not actually use it for anything streaming until I added the HDMI extender to apparently get a better WiFi signal from my DSL gateway (I think configuration is direct instead of through your WiFi access point to be able to configure WiFi security settings).

So at first I did not get the Chromecast icon in Android apps and could not use Chromecast from Chrome with Chromecast Extension in Linux or Windows until I added the HDMI extender. Once I added that, everything worked (Android, Linux, and Windows).
