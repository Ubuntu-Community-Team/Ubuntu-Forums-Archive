### Post by russo.mic on 2008-12-10
To be honest, I don't know of any that doesn't.  As long as you format it with a filesystem that linux can read and write to, which is most any common one I know of, you'll be fine.

Format it FAT32, that way you can use it to transfer between OSX, Windows, and Linux should you have the need.

Or format it ext3 and run a live distro off of it.

They all should work fine.

RUsso
