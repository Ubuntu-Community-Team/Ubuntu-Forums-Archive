### Post by amanchesterman on 2014-12-14
I use Zim for taking notes and find it straightforward, flexible and reliable. It lets you organise notes in a variety of ways and gives you a tree-like overview of them all; it also includes a 'to do list' feature. It's available from the Ubuntu Software Center and there is a Windows version. It doesn't automatically sync between machines but you can do that by putting the Zim notebook file in cloud storage.
I suggest you look at the web page [http://zim-wiki.org/index.html](http://zim-wiki.org/index.html) for further information (see the FAQs for syncing between machines).
Personally I would find LibreOffice too elaborate for simple note-taking. However when you start Office it displays on the screen thumbnails of all the documents you have edited recently, so that 'at a glance' overview *could* serve as a preview of notes you have made.

### Post by dragonfly41 on 2014-12-14
To add to the list of lesser known options ...

CherryTree is found in Ubuntu Software Centre.
It is a hierarchical note manager.

There is just one developer but there is an active community of users suggesting new features.

[http://www.giuspen.com/](http://www.giuspen.com/)

You can import notes from other note files.  See export/import.

If you are looking for something more advanced then try Scrivener (free linux beta).  

[http://www.literatureandlatte.com/forum/viewforum.php?f=33&sid=a8ad5faf7e8338be7c436884c78191a8](http://www.literatureandlatte.com/forum/viewforum.php?f=33&sid=a8ad5faf7e8338be7c436884c78191a8)

Linux licence free until January 2016.
But Scrivener is on a par (in size) with LibreWriter and is more for authors, writers of novels etc.

...

I use both of these.
