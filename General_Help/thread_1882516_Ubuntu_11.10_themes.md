### Post by Frogs Hair on 2011-11-17
If you're looking for themes see the link [http://gnome-look.org/index.php?xcontentmode=167&PHPSESSID=2f08d34327ffadfdd58321e14378ec27at](http://gnome-look.org/index.php?xcontentmode=167&PHPSESSID=2f08d34327ffadfdd58321e14378ec27at) 

The GTK 3 themes will work and although shell themes are somtimes included with the GTK 3 themes they are ignored when using Unity or Classic Gnome .

Installing Themes / Icons 

Install the Gnome Tweak Tool / advanced settings from the software center for making theme selections.  

(Single User) Use Ctrl + H in Nautilus to open hidden folders.  Create a .themes and .icons folder in home / hidden folders . 

(All Users) Use gksudo nautilus > File System / usr / share / themes or icons .

Download , extract the packages , and place the folders in either location making theme selections using Advanced Settings . Logout - in may be required .
