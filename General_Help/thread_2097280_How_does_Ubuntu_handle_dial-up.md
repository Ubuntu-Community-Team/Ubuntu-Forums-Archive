### Post by Impavidus on 2012-12-23
I used a dialup connection with a USB modem whilst still using Ubuntu 6.06 and if I remember correctly it required gnome-ppp (which is a graphical front-end to wvdial). It worked out of the box though. Furthermore, you just need a modem and a telephone number (and probably an account). A local one at a local ISP is probably best, but it should work with an international number as well.

I remember a dutch ISP setting up a dial-up server for egyptians last year (not offered to dutch people anymore) when president Mubarak blocked all internet access but didn't block telephone lines. Of course, calling from Egypt to the Netherlands was expensive, but the only way to get information out...
