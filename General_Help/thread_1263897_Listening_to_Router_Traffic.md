### Post by P4man on 2009-09-11
The whole point of a router is to route traffic, and not broadcast it everywhere like a hub, so only the router itself can see that. If it doesn't have any features for logging / traffic  there isn't much you can do (other than trying out an alternative firmware for the router that does support it).
