### Post by sowelie on 2008-02-05
I would try by resetting up your resolution by running:

```
sudo dpkg-reconfigure xserver-xorg
```

It will allow you to pick the correct resolution / refresh rates for your monitors. If that doesn't work you may have to manually set up your xorg.conf.

Check out this post for some dual monitor setup tips:

[http://ubuntuforums.org/showthread.php?t=221174]("http://ubuntuforums.org/showthread.php?t=221174")
