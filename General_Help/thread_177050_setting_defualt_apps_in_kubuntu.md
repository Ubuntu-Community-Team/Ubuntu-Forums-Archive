### Post by aysiu on 2006-05-15
Let's say you want MP3s to open with AmaroK by default. Just right-click the MP3, select **Open With**. Then, in the new dialogue box, select AmaroK and choose to remember the file association (or always open it with that application).

You mean a keyboard shortcut to launch the application? You can edit the in the KMenu entry itself.

If you're talking about navigating through songs (next song, previous song, stop, play, pause), you can define global keyboard shortcuts for JuK and AmaroK (I don't know if it can be done for other media players).
