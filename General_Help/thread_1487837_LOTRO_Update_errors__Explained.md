### Post by xtremesupremacy3 on 2010-05-19
Are you one of those people who plays LoTro using the CLI launcher or PyLOTRO? Have you had problems with a recurring error 'You do not have the current version of the client'?

What has happened is that with the latest updates for Lotro, certain files are not being downloaded as they should and causes the upgrade to just simply not work. Unfortunately (at least for the EU client), they have done away with patches and instead do patching through the client...which doesn't work.

Well I was able to have a working client.

If you have a virtual machine with Windows, eg VMWare, or VirtualBox, then your ready to go, if not then please install one.

Install the Book 8 client on the virtual machine and afterwards update to the latest version. Once this is done, simply copy the entire map to ~/.wine/Program Files/
And now you should be ready to go
