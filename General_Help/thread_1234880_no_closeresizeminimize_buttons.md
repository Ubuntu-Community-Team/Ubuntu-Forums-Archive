### Post by CatKiller on 2009-08-08
> **mcduck said:**
> It might help a bit if you told what exactly you removed...

And what you didn't, for that matter.

At the moment you aren't using a window decorator, which is why your windows have no decoration.

The choice of which window decorator you're likely to want to use depends on which window manager you're using at the moment. If you're using Metacity (no "Desktop Effects") then you'll want to use Metacity as your window decorator too. If you're using Compiz (with "Desktop Effects") then you can either use Metacity or Emerald (if you want effect to be applied to your window decorations too). These are the common options, but you might be using one of the other window managers that are available instead. Knowing which window manager you're using will assist us greatly in helping you to run a window decorator too.
