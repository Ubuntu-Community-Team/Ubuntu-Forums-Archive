### Post by Jivicin on 2007-01-07
I know I've disabled it before in the configuration editor, but I can't for the life of me find it right now...

Where's the setting to disable the animation box that appears when you launch an application?  I remember disabling it because it ran so slow under XGL.

Thanks!

EDIT: Found it...

1. Run the gconf-editor (Applications->System Tools->Configuration Editor)
2. Navigate into the panel options (apps->panel->global)
3. Uncheck the "enable_animations" property
4. Exit the configuration editor
