### Post by PrinceArithon on 2006-11-02
No cuz what happens is by the time it gets to the blinking cursor the operating system has no more control over the system.  

It's very similar to windows.  Like when you do the system updates late at night or something and it asks you to restart the system so you just turn off the computer and go back to it the next day.  It all still installs, because a restart is basically just a temporary shut down if you want to think of it that way.

So no you shouldn't have any problems, but if there is, then you have something else wrong.
