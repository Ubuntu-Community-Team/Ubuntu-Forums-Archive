### Post by veniatregnum on 2010-10-13
The dropbox and transmission status icons should be included in recent versions of Faenza. (check in the usr/share/icons/Faenza/status/22/ folder).

I think it also depends on what version of dropbox you're running. I don't think the latest stable release supports this, the new forum release does however.

Here are some links:
[http://forums.dropbox.com/topic.php?id=23621]("http://forums.dropbox.com/topic.php?id=23621")
