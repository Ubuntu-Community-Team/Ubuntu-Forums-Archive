### Post by linuxinstalledfromhdd on 2011-05-20
Switch back to Classic Ubuntu.

Click on top left corner of Unity. Type in the search box for Login.  Unlock. And select from the dropdown menu Ubuntu Classic. And exit to  save. Reboot to make sure the changes have taken effect.

### Post by timgood on 2011-05-20
If you log out you will find you can select 'Ubuntu Classic' which will return you to the Gnome 2 desktop. Be warned: you may need to jump through several command line hoops before you regain the functionality you had in 10.10 and you may never get all of it back! My experiences [here]("http://teknolegy.blogspot.com/"), I've gone back to 10.10 myself. YMMV.

Hope this helps.
