---

### Post by ago on 2008-04-30
If a cracker takes over from within Windows, he can access ANY file on ANY partition, including ones from other OS. Unless they are encrypted. The fact that the files are in another filesystem on its own partition or loopfile, is only a weak barrier. In this respect the security is not much different than a Ubuntu installation to dedicated partition. It is far less likely to get your machine compromised while you are in Ubuntu, but as mentioned, it can be compromised while you are in Windows. That said I am no security expert, so take it at face value.
