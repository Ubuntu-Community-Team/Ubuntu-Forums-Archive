### Post by deconstrained on 2011-03-15
If the keylogger is software (which the attacker would need superuser privileges or a livedisk and non-passworded BIOS to install) then there's most likely software out there that could detect it (maybe it would show up on rkhunter's radar). If it's hardware, then it's nearly impossible to detect except through careful inspection of the keyboard (depending on how well it's disguised).

### Post by deconstrained on 2011-03-15
Um...Like I said if a keylogger is carefully disguised and designed well so it emits no conspicuous EM frequencies that can be detected, and it's been installed into the keyboard (i.e. cleverly disguised as the ferrite choke on the cable, if your keyboard is USB) then it would be very hard to detect, let alone notice, that it was there.

If you're paranoid, check the connection between your keyboard and computer frequently. There are some keyloggers that connect *between* your keyboard and computer. And put distinguishing but hard to find/replicate marks on your keyboard that identify it, so that you can notice tampering or replacement.
