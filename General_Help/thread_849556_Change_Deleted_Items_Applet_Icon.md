### Post by Pacopag on 2008-07-04
Hi.  I'm really a noob, so don't take me too seriously.  I'll just suggest something you can try if no one else knows what to do.  If you go into the folder /usr/share/icons, you should be able to find the file containing the trash can icon (I found mine in a folder called "Places").  I'm guessing that if you replace the file with one of a blue trash can with the same exact file name, it might work.  Problem is, when I looked in my folders (I use the "Human" theme), it looks like there are several icons that all look the same.  So it might take you a while to isolate which one is being used.

Thing is, you'll have to find a file somewhere (or make one) of a blue trash can.  There are lots of icons you can download if you google it.   Or check out art.gnome.org (that's where I got some of mine).
