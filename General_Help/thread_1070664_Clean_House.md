### Post by damis648 on 2009-02-15
Generally any filesystem (like EXT3 and NTFS) will only erase the reference to a file when a file is deleted, meaning theoretically the file is still there, while the disk space where is was is shown as empty. The only way to completely consider it 'gone' would be if write zeros to empty spaces.
