### Post by DJ_Peng on 2008-12-04
I'm pretty sure that's a technical impossibility. Someone may be able to write an extension to do that, but I doubt it would be easy. You can do it in Terminal, but Firefox has to work on too many platforms for it to work. Even an extension specifically for GNOME-based platforms would be different from any other platform and I don't see anyone putting in quite that much work for such a narrow target audience. Of course anyone is welcome to prove me wrong on that point.
