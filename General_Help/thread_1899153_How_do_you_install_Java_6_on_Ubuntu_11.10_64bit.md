### Post by QIII on 2011-12-23
The best step-by-step method for installing the Sun JRE and getting the plugin to work with your browser, the one I would highly recommend as one who makes his living as a software developer in languages including Java (among others) is found here:

[http://sites.google.com/site/easylinuxtipsproject/java](http://sites.google.com/site/easylinuxtipsproject/java)

First, however, uninstall any Sun Java you may currently have installed by the same means that you installed it.

But you really need only use if it something you are doing just does not work with OpenJDK -- and that is sometimes the case.

Since this comes up so often, since Sun Java is no longer in the repositories (I don't disagree with Canonicals reasons) and since OpenJDK (a fine and commendable effort) does not always suit every need, it would be nice if this were added to a sticky somewhere.
