### Post by sarte on 2008-11-26
Actually ogg (but you probably ment vorbis, because ogg is just a wrapper) is supported by out-of-the box ubuntu. Flac needs lib, which is very easy to install through synaptic. And Theora, I'm not quite sure, but I think its supported out of the box. And basically all the other open codecs are supported. If not out-of-the box then with a very minimalistic effort.

And if you just want a view or listen them. Then install VLC it should do the trick
