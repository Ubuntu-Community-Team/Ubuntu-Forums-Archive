### Post by Zill on 2011-02-14
Praveen30:  If you want to quickly install java plus various other useful "goodies" with one command simply install [ubuntu-restricted-extras]("https://help.ubuntu.com/community/RestrictedFormats"):
```
sudo apt-get install ubuntu-restricted-extras
```
