### Post by bluevoodoo1 on 2006-05-18
The program you installed should have some sort of uninstaller. Does it?

If it does, change directories to that folder and run... 

wine uninstall.exe

(Or something similar, whatever the name actually is)

You can also, with nautilus, navigate to the "uninstall.exe" and select wine to open it.

### Post by YokoZar on 2006-05-19

This opens a winelib program that is analogous to Windows "add/remove programs"
