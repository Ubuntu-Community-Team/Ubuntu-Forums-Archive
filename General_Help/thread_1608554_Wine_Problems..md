### Post by ivarn on 2010-10-29
ok first of all.
Wine is just an emulator ok?
There are 100s of programs for linuz that can sync your ipod, play music and such.
If you depend on wine to run your programs, change back to windows.
Because, linux isn't windows ok?
Just because iTunes won't work in Wine doesn't mean you are doing something wrong.
As I said, wine is an emulator and designed to emulate windows apps and games as best as possible.
A linux program that looks just like the old iTUnes is SongBIrd.
Songbird is a great music player that works and acts like itunes did back in the old days.
You can also use Rhythmbox.
And no, wine can't emulate iTunes's iPod functionality.
Plus, Wine is great when it comes to games and apps that are not released for linux, but seriously.
There are enough ipod and music apps on linux.
EDIT: So again, if you prefer the windows software, go back to windows.

### Post by matt_symes on 2010-10-29
Hi

Just to be pedantic, wine is not an emulator. Its a set of thin wrappers around linux api calls to enable compatibility between windows api calls and linux api calls. The converse is cygwin or minsys under windows.

Sorry pedance over.
