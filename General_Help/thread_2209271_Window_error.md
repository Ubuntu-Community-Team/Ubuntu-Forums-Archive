### Post by Adam_Soloway on 2014-03-04
Whenever I click in a window (dock, panel are not affected) the cursor changes to "grab" and when I drag, the window moves. If I hold "alt" down, some of the functionality is restored, but not all. Also, right clicking anywhere on the window brings up the window menu. This is rather urgent as I need this computer working by tomorrow and i'd rather not go through another full re-install of the operating system. Thanks for any help.

### Post by Adam_Soloway on 2014-03-05
I have fixed it, in case anyone else has this problem: In the app "Tweak Tool" under "Windows" the "Modifier to use for modified window click action" must NOT be set to "disabled".
