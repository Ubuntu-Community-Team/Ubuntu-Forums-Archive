### Post by oldfred on 2011-10-14
If it is your only working system I would not just jump in. The USB flash is a good idea. I have a full install in a 16GB flash using 8GB for / (root). It is a bit slow but functional.

### Post by warmotor on 2011-10-14
Upgraded Xubuntu to Oneric today. Now when I try to compile it can't seem to link to SDL libraries. Re-installed SDL and SDL-dev and still does not work. My current project is now at a stand-still. I hate my life.

UPDATE - OK if you are compiling any code with the GNU GCC or G++ compiler, any flags for the linker have to go AFTER the code and executable names. 

I WAS COMPILING WITH:
g++ -Wall `sdl-config --cflags` `sdl-config --libs` -lSDL_ttf -o game_001 game_001.cpp

NOW I HAVE TO FORMAT AS FOLLOWS:
g++ -o game_001 game_001.cpp  -Wall `sdl-config --cflags` `sdl-config --libs` -lSDL_ttf
