### Post by buzzingrobot on 2015-01-02
When the Gnome project stopped working on the Gnome 2 interface in 2011 (used in the release 9 you liked), another group of developers who wanted to continue using that interfce eventually began working with and adapting the Gnome 2 source code so it could be used on current systems.  That's the Mate project. It's essentially where Gnome 2 might be today if the Gnome project had continued with it, so Mate looks and works very much the same as Gnome 2. (Several applications names have changed to avoid conflicts, but that's trivial.)

The Xubuntu, Lubuntu and Kubuntu spins also use the same basic menu-and-panel approach that Gnome 2 used.

Synaptic is in the repos so it can be installed if it isn't included by default.
