### Post by mc4man on 2013-08-02
gnome & gnome-core are just metapackages,  so when  removed nothing is really removed.
You can  put them to use when not installed to check if you're missing any actual package of importance by marking for install & seeing what else is to be installed.

I use ubuntu & most times never have ubuntu-desktop installed without issue, occasionally I'll mark for install in synaptic to see if anything new or important isn't installed that is a dep of ubuntu-desktop.

If upgrading from one  *ubuntu release to another then you'd want these type metapackages installed or re-installed before upgrading
