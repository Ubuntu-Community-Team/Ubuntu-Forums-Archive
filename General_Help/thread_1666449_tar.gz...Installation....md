### Post by kn0w-b1nary on 2011-01-13
What is in the .tar.gz?
If there is a file that ends in .c, then run "gcc /pathtofile/filename.c
If there is a make file, then in the directory run "sudo make install"
If the file ends in .sh, open the directory and run "sh filename.sh"
To open a directory, type in terminal "cd /home/username/Downloads/" for example.

Hope this helps!
