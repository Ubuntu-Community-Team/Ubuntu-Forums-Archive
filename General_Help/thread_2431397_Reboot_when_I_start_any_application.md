### Post by guiverc on 2019-11-15
As I suggested on Ask Ubuntu (*you posted the same question*), I would suggest booting a 'live' system and checking your hardware & file-system for errors/problems. I would then test the box using a 'live' system to validate it and check your system logs for clues as to the issue.  Assuming you didn't find anything, I'd then install a supported release of Ubuntu.

Standard releases (such as 17.04 from 2017-April or 17.10 from 2017-October) have only a 9 month supported life so need to be upgraded to the next release 6-9 months after release (17.04 moves to 17.10, which moves to 18.04).  If you don't like the regular upgrade cycle, use a LTS or long-term-support release which has 5 years of supported life.
