### Post by Carl.Spackler on 2010-09-16
I have the same machine. Love it.
 
I have not experienced the black out that you are referring to, but I will say that all of my attempts to use the perfromance video chipset with and without vgaswitcharoo are met with failure. It seems that vgaswitcharoo does not support using the accelerated driver so when it does work - an it does, you get crappier video perfomance than you would by just using the low powered chipset. It is worth noting that vgaswitcharoo is an excellent start for hybrid video machines like ours.
 
I've been searching the internet for others that are using Ubuntu on this machine and since the machine is very new I was surprised that almost everything worked out of the box.
 
I replaced my wireless card with the Broadcom 4322 because, as you probably already know, the Atheros card that is in it does not support 5Ghz wireless. I also picked up the second hard drive caddy and added the factory bluetooth module from HP spare parts.
 
One last thing. I don't think the fingerprint reader works. I have yet to find a confirmed case of someone getting it to work.
