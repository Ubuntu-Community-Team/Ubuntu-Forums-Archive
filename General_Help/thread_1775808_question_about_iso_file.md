About once every six months.  10.04.2 is the current LTS, so regularly, but seldom!  A newly downloaded version of any ubuntu, fully updated with apt-get, ends up in the same position, however, so not too much to worry about, unless you are unlucky and have hardware that will not boot with the packages in the original iso, but needs the new ones.
