follow the steps described in the links above.. it worked for me..

it boils down to the following:
- one 'device' section describing both video outputs
- one 'screen' section with optional modestrings
- one 'monitor' section with optional modestrings
- serverflags must NOT contain xinerama, as twinview emulates xinerama itself..
