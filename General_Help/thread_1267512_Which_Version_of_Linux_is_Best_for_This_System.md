### Post by earthpigg on 2009-09-15
i suggest sticking to the ubuntu family, so you can best apply whatever knowledge you already have.

xubuntu is marketed as significantly more lightweight than kubuntu or gnome, but i beg to differ.

if you are comfy with the command line, a command line install and then you adding exactly what you want would work, of course. i will assume this is not the case, or you would not have posted this thread :D

an incomplete list of potential choices within the ubuntu family off the top of my head are:

-crunchbang (very established and popular for this use)
-masonux (im biased, obviously :D )
-spri (in beta, but works fine. the lightest on this list and only one that does not use openbox or lxde.)
-lubuntu (in beta, no automatic installer iirc)
-anything listed here: [http://wiki.lxde.org/en/Ubuntu#Ubuntu_variants_with_LXDE_pre-installed](http://wiki.lxde.org/en/Ubuntu#Ubuntu_variants_with_LXDE_pre-installed)

### Post by snowpine on 2009-09-16
I second the recommendation for CrunchBang. It is very intuitive for a Linux newbie... want to browse the web? press windows+w. browse your files? win+f. play some media? win+m. Plus it has all the multimedia codecs preinstalled (no "why can't I watch youtube??") and the CrunchBang forums are super-friendly.

If that's not fast enough, Puppy is pretty much the #1 "user friendly distro for people with really, really old hardware" out there... but it sounds like ease of use may trump blazing speed for your mom's purpose. :)
