### Post by adari on 2009-07-31
I'm making this post as a help/warning to anyone who has randomly lost many pages of work for seemingly no reason at all.  I saved and closed my 22 page document in open office last night.  When I went back to it today, the document was completely unformatted and empty.  I had a back up about 12 pages earlier, which really didn't feel good enough.  After checking the properties in the file, I found that the information was still there.  

So If you happen to find a complete file blank after being in open office, you should first check properties on the file, as long as it has a size, there is information.  While there may be another program on xfce naturally that will save it, I used gedit, which recovered all of my information in full.  Unfortunately it was completely unformatted, but atleast it's back.  Does anyone know how to autoformat it?  If this doesn't work for some people, does anyone have any other suggestions in case this happens to someone?  

I consider this problem to be catastrophic, and will not use open office anymore thanks to it.  I'm on an Asus eee pc 1000, and abiword doesn't like to run if it's not freezing, so for now, I've switched to google documents, which I might suggest to anyone.

-Adari
