### Post by 3Miro on 2009-04-13
The core of Linux (and any Operating System) is called Kernel. Different kernel versions of Linux come with different drivers and improves security as well as some other possible features.

You should always run the latest kernel version (for security reasons) unless you have a driver issue or something of the sort.

You should enter recovery mode only if something in your system gets broken. If you are familiar with windows, they have the Safe Mode version, it is kind of like that. You basically load the bare minimums for the system to fix whatever problem there might be.

Basically don't worry about it, just run the top one of the Ubuntus.

### Post by lovinglinux on 2009-04-13
Sometimes when a new kernel is released some things could stop working as expected. So number 3 is the old installed kernel, which is there if you have problems with the new kernel (1). If everything is working fine, then you don't have to worry, just select the first option when you boot.
