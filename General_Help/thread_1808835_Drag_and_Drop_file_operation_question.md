### Post by aka120 on 2011-07-23
Dragging and dropping an image from a website to a local directory in Ubuntu is not noticeably different from other OS platforms, until you drag and drop a rather large image. Yes, like a 5-10mb image will present you with a file operations dialog and a copying status. (i.e. like the dialog window you get when moving a 700mb file from 1 HDD to a completely different HDD)

Much to my noobness with Ubuntu, I did not realize that right-clicking the image and selecting save image... is instantaneous no matter what size it is.
