### Post by danhm on 2007-06-11
How did you update your kernel? If you used the linux-686 (possibly linux-686-smp, since you have a dual core processor) package, that won't change anything. The current kernel in the repos supports most processor types "out of the box". That package is just legacy for people still using or upgrading from Dapper; it points to the generic kernel.


Also, I've never heard of that problem. I have a laptop with a Core Duo as well, and haven't had a problem with overheating in the year or so I've had it.
