### Post by Mark Phelps on 2011-06-16
Sorry, don't know specifically about Testdisk, but when I've used MS Windows SW to do filesystem recovery, I've often had to run the software overnight -- and that is for much smaller (<100GB) partitions.

So, running this long for a 500GB drive might not be abnormal.

Also, if you're trying to recover an MS Windows filesystem (i.e., NTFS) from the 500GB drive, you might want to look into GetDataBack for NTFS from Runtime Software.  You can download a trial version for free that will scan your drive.  That will give you an alternative to try.
