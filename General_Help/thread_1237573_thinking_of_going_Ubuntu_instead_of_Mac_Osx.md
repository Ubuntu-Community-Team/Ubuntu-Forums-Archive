### Post by tgalati4 on 2009-08-11
There are several photo management applications in Linux.  You will have to try them and get comfortable with them, but they work well.  They are different from Lightroom and iPhoto.

If you must link your iPhone to iTunes, then OS X is really the only option.  You could set up your machine with dual-boot.  That gives you the option to use either operating system.

With more effort, you can run a virtual machine and run Ubuntu under Mac OS X using parallels (a commercial product) or one of several open virtual machine apps.

I'm not sure if you can run Mac OS X from within Ubuntu in a virtual machine.  Besides, on Mac hardware, only OS X is going to allow all of your hardware to work out of the box.  It can be made to work with Ubuntu, but it requires some fiddling.

It's really hard to enumerate the pros and cons.  You just have to use both for a while and you will quickly see the differences.  

Visit an Apple store and get some hands-on time so you can see for yourself.

I have an older powerbook running Tiger and I like it.

I have a thinkpad t43p running Jaunty and I like it more.

### Post by fennec_fox on 2009-08-11
1. I'll be honest. I'm not sure how easy this really is. I believe you will have to jailbreak it. Here's a guide
[https://help.ubuntu.com/community/PortableDevices/iPhone#Set%20up%20the%20iPhone%20or%20iPod](https://help.ubuntu.com/community/PortableDevices/iPhone#Set%20up%20the%20iPhone%20or%20iPod)

Not sure when that was last updated. You may need to do firmware changes. I don't know if you are up to that sort of thing.

2. Backs are easy. There are a number of programs available. rsync is one that comes to mind.

3. GIMP is excellent. I would go as far to say photoshop is better but, IT IS arguable. GIMP is very close to photoshop. The only complaints my professionals friends have told me about is the interface doesn't suite them mainly because they have been ground into photoshop. Is the program on par though? I would say so yes. 

As far as the OS. Mac OSX is great but, I'd take ubuntu over it an time. Ubuntu is much more open. It doesn't fight you on priviledges or anything. Customization on ubuntu is much better and nothing is really proprietary. Most things you want are at your finger tips for free and I can't really think of anything OSX has that ubuntu doesn't. 

Using wine is a little bit easier with linux because it's a couple click process install of using some sort of port for mac. OSX has better itunes support though but, ubuntu has some amazing media players. Many of which I consider better and are compatible with ipods. 

As far as performance, they are mostly the same. If you're solidly looking for performance though linux is the way to go. You can install software like preload, you can get a lightweight desktop environment, you can customize your kernel and so on. You'll have ancient machines running at modern speeds. 

As far as user experience. I would say ubunu is better than any other OS available. Yes you will probably have to use the command line from time to time but, with compiz, customization options, hotkey options, utilities available you can make a system that runs exactly how you want it and takes care of all of your unique preferences to give you the user experience you want. On my own home system my productivity on ubuntu is higher than any other OS because everything has been so easily customized to fit me. I'd say my productivy is even doubled. There's gadgets/widgets, utilties, backups, encryption, keeps track of my most used programs so they load instaneously, eye candy and special effects, customized themes and hotkeys, etc. I literally barely need to use the mouse on my system unless it's for clicking hotlinks in firefox or choosing a song in my media player. 

I say ubuntu is better than any other OS simply because of the ease of tweaking things to how you want them in ubuntu. Deb files are incredibly easy, they have tons of nice guis and a lot of the customization tools are available with a couple clicks. 

That said, that's pretty much the same for debian or any ubuntu variant. 


Mainly the thing you'll want to determine is if you want to go throug the work of getting the iphone to work. I encourage it and I think it will save you a lot of money for a system you can make look just like OSX if you want but, if you want the OSX experience and you want the iphone to work with no problems then you might want to go with OSX. OSX will certainly take care of you and it is very similar (being a BSD like system). I think OSX might hold you back a little bit but, it's very well supported and has a great user experience. Maybe you don't want as much customization or anything and just want an out of box system that the iphone will work seemlessly with.

Hope that was helpful. This is coming from a person with mingles with xp, vista, osx, red hat, solaris and ubuntu every day; not by choice. I am biased toward ubuntu though. My personal systems are now 100% ubuntu. Two of my roomates are now migrating from OSX to ubuntu. One from vista to ubuntu. One of them is not computer oriented but is now convinced ubuntu is better than OSX. 

All personal opinion and how much effort you want to make your system bend. Some people are very happy to bend to their system.
