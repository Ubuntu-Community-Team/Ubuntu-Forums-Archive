### Post by earthpigg on 2009-09-09
i'll throw VLC out there as what i consider a 'must have'.

many situations - in Windows, OSX, and Linux - where VLC will play a given audio or video file that nothing else will.

do not rely on WINE for anything that you consider 'important'. if you wnat to have some fun playing WoW or whatever, go for it. but if it is work related.... no. use VirtualBox for mission critical stuff. i consider WINE a last resort, not a first resort:

1. decide i need to accomplish a task.
2. locate Free Software application to do it that runs on Linux.
3. failing that, locate proprietary application to do it that runs on Linux.
4. failing that, locate Windows app that will do it, and try to run it in VirtualBox.
5. failing that, go look back at step 2 again and make sure i didn't miss something.
6. failing all of that, run WINE.


3d games is really the only time you will *need* to go to step 6.

but yeah, if you want to try WINE and iTunes or something else, go for it... just dont expect *anything* to *ever* work.

and when it does (which it will, often), count your blessings.
