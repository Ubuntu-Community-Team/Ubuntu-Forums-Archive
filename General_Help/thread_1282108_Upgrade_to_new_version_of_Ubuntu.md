The latest version, 9.10 is still in beta testing. There are lots of problems still, just check out the karmic koala testing forum. Some of the problems are minor, some not so much. ### Post by ralphsmole on 2009-10-04
Upgrading from 9.04 is VERY easy. Just do this:
To upgrade from Ubuntu 9.04 on a [desktop[IMG]http://images.intellitxt.com/ast/adTypes/2.gif[/IMG]]("http://www.ubuntugeek.com/ubuntu-9-10-karmic-koala-beta-released.html#") system, press Alt+F2 and type in “update-manager -d” (without the quotes) into the command box. Update Manager should open up and tell you: New distribution release ‘9.10&#8242; is available. Click Upgrade and follow the on-screen instructions.

I did it earlier,worked AWESOMLY!

### Post by Tmi on 2009-10-04
As others have said, do not upgrade yet while its in alpha/beta.

Read up on the new features either here on the forum or other places and then decide if it is worth the upgrade. If you had problems setting up the current box it might be worth sticking to the old version rather than potentially facing the same kind of problems in the next version.

Personally I don't see anything special in the new version that will warrant an upgrade for my part.

And if you do upgrade, keep in mind that doing a dist-upgrade might mess stuff up. I always do a fresh install instead.
