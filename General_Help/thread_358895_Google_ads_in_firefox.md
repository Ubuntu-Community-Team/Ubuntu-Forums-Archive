### Post by racmar on 2007-02-11
The only option I have used successfully was R.I.P. for firefox [http://rip.mozdev.org/]("http://rip.mozdev.org/")

It has pre-made scripts to remove most annoying content on high traffic sites, and you can customize your own script for myway.com

I quit using it a long time ago, and simply resigned myself to living /w the google ads.  If anyone else has a better suggestion, I'm all ears too.

Also, I've been using "Adblock Plus" instead of adblock for quite some time now, as it has the subscription lists available that are regularly updated, and it has much better development resources IMHO.  you can find it here: [http://adblockplus.org/en/]("http://adblockplus.org/en/")
