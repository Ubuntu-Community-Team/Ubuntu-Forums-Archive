### Post by idoitprone on 2011-03-04
Linux will not run .exe files, since they are windows executable. Either way, if you manage to install that driver, it probably wont work with Linux, since it was made for Windows. That means, you need to find the linux driver separately. I just know that linux support ppd files. 

[http://www.linuxquestions.org/questions/linux-hardware-18/linux-driver-for-canon-mf4370-printer-856610/](http://www.linuxquestions.org/questions/linux-hardware-18/linux-driver-for-canon-mf4370-printer-856610/)

From the looks of it Canon is not so Linux friendly.

Internet Explorer is only meant for windows, since there are better native browsers eg. firefox, chrome, and etc. You can try wine, which is a windows compatibility layer. It is an attempt to run windows apps on Linux.
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=25](http://appdb.winehq.org/objectManager.php?sClass=application&iId=25)

however, installing ie8 on linux looks problematic.
