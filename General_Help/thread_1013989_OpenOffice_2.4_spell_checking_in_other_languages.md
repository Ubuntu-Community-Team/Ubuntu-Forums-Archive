### Post by whoop on 2008-12-17
I'm trying to get OpenOffice 2.4 Word Processor to check spelling for the dutch language on my English Ubuntu 64bit installation.
English spell checking seems to go allright. When I select dutch language via the Tools-->Language-->For all text-->More menu-item the spell check seems to do nothing anymore. No red underlining for any words in the document. And when I hit F7 it just says "the spell check is complete".

What am I doing wrong? Help please!!!

### Post by uidb4056 on 2008-12-17
Have you downloaded Dutch Dictionaries?, if not open Writer and under File menu select Wizards and from them select Install New Dictionaries. Then go to Menu Tools select Language and select 'for all text', choose the Dutch and now it will work.
