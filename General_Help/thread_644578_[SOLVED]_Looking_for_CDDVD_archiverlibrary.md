### Post by tshrinivasan on 2007-12-19
cdcat is a application you need.

[http://cdcat.sourceforge.net](http://cdcat.sourceforge.net) see the link for details.

sudo apt-get install cdcat
installs this in a ubuntu macchine.

### Post by Sef on 2007-12-19
Applications > Accessories > Terminal

First type in this command:

```
sudo apt-get update
```

then type in:

```
sudo apt-get install cdcat
```
