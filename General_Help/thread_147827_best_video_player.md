### Post by psychicdragon on 2006-03-21
MPlayer is the best when it comes to playing files using strange container formats and newer video codecs. Although it has a very dated UI, that can be somewhat frustrating. VLC comes in second place in this regard, but it has a much more standard interface.

If you watch a lot of anime, MPlayer from cvs is probably the way to go. Because many groups that do anime releases tend to start using the latest and greatest codec as soon as it's available, and Totem et all just aren't able to play the files.

See this HowTo if you're interested in getting the latest version of MPlayer: [MPlayer compilation from CVS]("http://ubuntuforums.org/showthread.php?t=85190")
