### Post by martrn on 2009-10-27
> is it possible to run ubuntu on usb 1.0 using a 4gb or 8gb removable drive?

Yes its possible to run ubuntu on usb 1.0 

The data transfer rate will be 1.5 MB/S or less.

USB 1.1 is 12.5MB/s or less
ATA     is 16.5 to 33 MB/s
ATAII   is 33, 66, 101 133 MB/s
USB 2.0 is 480 MB/s or less
SATA    is 1500 MB/s

So you can run ubuntu on usb 1.0 but will not see a great deal speed transfer when saving / loading / running programs or processing a large quantity of data.

It is quiet a different ubuntu experience running over usb.  You will notice your boot time.  It make take between 10 to 20 minutes to boot up, but aside from that it should be usable there after.

Are you sure its usb 1.0 not usb 1.1 ?
