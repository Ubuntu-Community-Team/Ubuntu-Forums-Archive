### Post by Sorivenul on 2009-01-19
The LiveCD documentation is for remastering the LiveCD version, while the InstallCD documentation is for remastering the Alternate Install CD version. 

For most people interested in this type of endeavour, I usually suggest [Remastersys]("http://www.remastersys.klikit-linux.com/") instead of the manual methods outlined in these documents, though all solutions are valid.

The LiveCD can be installed, as long as you keep Ubiquity and associated files, if that is your concern. Remastersys makes this easy, as it depends on Ubiquity and said files.
