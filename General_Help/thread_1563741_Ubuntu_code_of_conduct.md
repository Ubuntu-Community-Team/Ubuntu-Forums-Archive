### Post by gzarkadas on 2010-08-29
The .asc file is a digitally signed file and the default gnome action when you double-click it is to try to verify the signature. To open the file, right-click it and select 'Open with' and then in the 2nd level menu choose 'gedit', or if gedit does not appear 'Other Application...'. If you do the later, select gedit from the list that appears. The file will open. Then press Ctrl+A to select everything and Ctrl+C to copy. Go to Launchpad's page and use Ctrl+V to paste the text.
