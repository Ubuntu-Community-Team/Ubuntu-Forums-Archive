### Post by MartinJ on 2007-08-04
I think you are using the Embeddable Image Viewer part of Konqueror, which isn't very good.
Ideally, you want to use the Gwenview Image Viewer.

To change: open an image in Konqueror then click on View menu -> View Mode -> Gwenview Image Viewer.

Hope that works / makes sense!
