### Post by icedwater on 2007-05-22
Try IEs4Linux, a script written to automate the installation of IE6, IE5.5 and IE5 (or all of these) using Wine on Linux. They are minimal IE installs meant more for website testing, but I think you should be able to download ActiveX controls if needed...

I personally didn't like this solution as it flickered quite a bit on my laptop (Fujitsu Lifebook s7020), but you might be more tolerant of such things, or maybe your girlfriend's laptop has a better graphics card :P

[www.tatanka.com.br/ies4linux/page/Installation](www.tatanka.com.br/ies4linux/page/Installation) is where you might want to look for this package, hope it helps. Migration is usually tricky where advanced applications are involved :P

Good luck!
