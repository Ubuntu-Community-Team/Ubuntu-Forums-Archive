### Post by erkki70 on 2005-04-02
Hi Akrame,
I think you need to install missing font packages  
```
xfonts-100dpi-transcoded
xfonts-75dpi-transcoded
xfonts-base-transcoded
```
 through apt-get or Synaptic in order to solve this and have normal sized fonts in those apps.

Hope this helps.
