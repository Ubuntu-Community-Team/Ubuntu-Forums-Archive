### Post by wolfen69 on 2009-03-12
you need to install the [user agent switcher]("https://addons.mozilla.org/en-US/firefox/search?q=user+agent&cat=all") in firefox. it kind of fools the website into thinking you are using IE. you can't just install it and go, it must be configured first.
