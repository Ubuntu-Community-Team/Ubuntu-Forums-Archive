### Post by lah7 on 2014-08-24
The speeds for all of your torrents in use by transmission. The alternative speed limits are simply caps you can set on how fast you can download and upload your torrents. (These limits can be changed in the Preferences.)

For instance, if you're streaming video on a slow connection and don't want your active torrents causing persistent buffering, you could turn on the alternative speed limits.

### Post by ian-weisser on 2014-08-25
You can set two sets of speed limits and toggle between them.
The handy part is that the toggle can be done on the command line, too.

For example, if you have Transmission on a mobile device, you can write a script to change speed limits based of if you're using an expensive or cheap connection.

Another example, some internet usage in places is cheaper at night, so you can write a script to change speed limits based on time.
