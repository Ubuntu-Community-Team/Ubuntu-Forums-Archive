### Post by apollothethird on 2011-06-25
> **vishnumg007 said:**
> write tab? where?

The Optional write-protect tab might not be on all the sd cards.  It you look up and down the parameters of the card and it has one you should be able to easily see it on the side.  It'll probably have the word "lock" marking the spot.

If you can't find it, it's possible you have an sd card without this feature.  I have found it to be on every sd card I've ever seen.  It might not be found on a mini sd card, just the regular sd card.  I also have seen the tab on all the mini adapter sleeves (a hollow sd card for accessing the mini sd cards).

If you have a mini sd card and are not using a mini adapter sleeve it's unlikely that you have this tab.
