### Post by papibe on 2012-03-30
Hi buffay. Welcome to the forums.

I just want to make sure of a this: you can't install and use nvidia-xconfig without installing the main package: the nvidia proprietary driver.

To install it, go to 'System -> Administrator -> Hardware Drivers'. There you will be offered to install the driver. If you haven't done, do it there.

Then, open a terminal and run:
```
sudo nvidia-xconfig
```
Then restart your machine.

From now on don't use xandr, but configure your monitors using 'System -> Administrator ->  Nvidia X Server Settings'. There you can easily set both resolutions and twinview for the other monitor.

I hope that helps, and tell us if you need more help with that.
Regards.
