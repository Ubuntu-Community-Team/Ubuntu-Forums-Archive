So I have Dual Boot Installed. This problem happened to me twice but I managed to fix them. But I thought I should share this in case it happens with other users... 

So the problem occurs when you boot Ubuntu and it drops into shell unexpectedly. It has something to do with fdisk.root or something like that (can't remember). Anyway running the recovery mode doesn't fix it... The only solution is to boot into windows and go to Ubuntu Directory from C:\Ubutnu and go to winboot and copy the menu.lst and paste it into C:\Ubuntu/disks/boot/grub  and then restart and Ubuntu will boot fine... 
I don't know if this is a major problem but I discovered the solution by complete coincidence! just heads up for the people in charge
