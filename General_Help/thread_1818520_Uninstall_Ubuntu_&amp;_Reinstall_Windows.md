### Post by MARP1961 on 2011-08-05
His computer might already be set to boot from CD first. If it is, then it should be as simple as booting with the Windows CD already in the drive. Let Windows reformat and then install. You will need your Product Key to reactivate the same version of Windows that was originally on the computer.

If you are reinstalling XP, make sure you update all the way to SP3. If you are reinstalling Vista then set aside about 24 hours to watch it updating and configuring, updating and configuring, updating and configuring................ or maybe download SP1 and SP2 onto a USB flash drive and manually install these to speed things up a tiny bit. Good luck!
