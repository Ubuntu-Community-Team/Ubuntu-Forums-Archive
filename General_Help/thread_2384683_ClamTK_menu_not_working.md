### Post by coffeecat on 2018-02-10
ClamTk run from Unity doesn't have menus. If you mean the icons on the Virus Scanner window, you have to double-click them to select them. You can change that to single click, if you prefer, in settings. 

Terminal commands are case sensitive. "ClamTK" is not valid. "clamtk" is.
