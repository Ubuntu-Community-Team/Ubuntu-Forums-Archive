### Post by EmanresuusernamE on 2008-08-30
From what I can tell just by looking at the home page, it seems like an alternative to Smooth Wall.  What they do is take an old computer that you're not using anymore, and turns it into a highly configurable firewall.  It will erase everything on the hard drive of the system and then become a dedicated "hardware" firewall.


...... No wait, FreeSCO says it's uses one Floppy, so I guess it doesn't mess with your hard drive.

If you'd like to test them, try using a virtual machine to see if you like them.  VirtualBox is pretty easy to use.
