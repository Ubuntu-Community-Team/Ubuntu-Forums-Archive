### Post by mb_webguy on 2008-08-13
I'd like to second the suggestion of Deluge.  As damoxc said, it can run under Windows as well.  I've tried Azureus and Transmission, and Deluge seems considerably faster than Azureus and has more features than Transmission.  The version in the Ubuntu repository is fairly old, but the official website offers a deb package for Ubuntu.  You'll need to change some of the network settings on the version from the website, since as far as I can tell there's no setup wizard to automatically configure the settings based on your connection speed, and if you don't change the settings it can be a bandwidth hog.

Alternatively, you can use uTorrent under Wine.  uTorrent is probably the best bitTorrent client available for Windows in my opinion, and it runs just fine (and arguably even better than on Windows) under Wine.

As for your download speed problems, have you set up port forwarding on your router?  If you haven't, *any* torrent client you use will be slow.
