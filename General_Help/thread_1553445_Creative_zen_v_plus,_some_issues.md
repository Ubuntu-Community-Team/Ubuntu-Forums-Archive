### Post by Paqman on 2010-08-15
I use Banshee, so I can tell you how to sort that out:

Plug the device in, wait till it shows up in the pane on the left. Click on it and set the device to "manage manually" for music, then just drag albums from your collection onto the device. You'll get a little progress bar to tell you it's working.

Banshee's support for MTP devices is one of it's plugins, I don't recall if it's enabled by default since it's been a while since I first installed Banshee. If it doesn't work check the list of plugins to make sure it's enabled.
