### Post by prshah on 2012-02-23
> **needhelplinux said:**
> 
Can linux make use of the full 2 GB or am I still limited to 1 GB? Is it something where the other parts of the computer can't handle the ram that the motherboard can allow?

Yes, linux can use the full 2GB ram. There are no other "parts" that will not allow 2GB RAM to be used. 

As long as your BIOS motherboard shows 2GB RAM (ie, the motherboard recognises 2GB), Linux (and all Windows XP and above for that matter) can handle and will benefit from 2GB.

However, getting the correct type of RAM for this machine may be a challenge.
