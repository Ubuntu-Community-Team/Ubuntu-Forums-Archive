### Post by KlinerDraken on 2009-08-17
> **neal.m.master said:**
> Will Banshee rename directories/files to organize them?  Based on previous threads, I think it does, but does it do this in a similar way to iTunes?  

I wanted to know before I gave it a whirl.  I'm going to move completely to Ubuntu and want something that organizes files like iTunes.

It has the option to rename files and folder, go to "Edit" and click "Select None". Now click "Edit" and "Preferences" and there is a check box to Update file and folder names, with options how you want them to be renamed under that.

Hope that's what you're looking for.
