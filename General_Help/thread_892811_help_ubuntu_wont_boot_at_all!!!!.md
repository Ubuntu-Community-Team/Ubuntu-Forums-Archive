### Post by cooke77 on 2008-08-17
How are you tring to use "Live CD"?
Unfortunately, there is no instructions supplied..on how to do this.
You have to use the BIOS 'Boot Sequence' set-up...to get the "Live CD" to work.
When it starts loading, use "ESC"....during the 'Language' window.....this will change the 'Language' window......to an 'Options' window.
It is from here, that you make a choice....whether to use the 'Live CD'.
Use the 'Check CD for errors'. (Would STRONGLY suggest that you do this, FIRST!!!.....to VERIFY that the CD has NO errors).....IF CD has errors, "that all she wrote,Folks!"

You can install from here, too. (Just be very careful...of HOW you do it.)

You can install within Windows.
(I, personally, have found this...to be the safer 'option'. IF you mess up, all you have to do, is go into Windows and DELete Linux OS from 'Add and Remove' (XP Pro)/and off the Hard-drive Partition you put it onto.)

Installing, VIA the BIOS 'set-up'....tends to place the Linux GRUB...into the MBR configuration files of XP. (A cow to un-install! Maybe force you to do a complete re-install of XP, to overcome..this will include a FULL Format, as well.)

Been there, done that.

Just make certain ....that you definitely have enough Hard-drive space...to put whatever distro of Linux you have onto it. (Don't be surprised...IF you check LATER....and, find that you have suddenly USED over 20 Gig...of Hard-drive!!!

If you get past the "Language" window/the "Options" window....and computer STARTS loading Linux OS......LOOK OUT for the "Use 'ESC' to enter Menu." line (Top Left).
You have to the count of 7...to Enter!
This is very useful...IF you can NOT go past this 'point', as this is yet another 'Option' window....for you to use. (There are a few options available......to find out which ONE worls for you....this WILL require REBOOTing, as you try each 'option'...to find the one that works for YOU.)

Once right 'option' is found, the installation will proceed.
(Sounds hard to do, BUT, it is a 'safe option'.)
And, installation...does take time.

If you can print this...will serve as guide-line, later.
