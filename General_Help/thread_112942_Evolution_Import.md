### Post by cstudent on 2006-01-05
Outlook stores data in .pst files, usually buried deep in the Windows directory. If you open the folder list view in Outlook and right click on your Outlook file at the top of the tree, right click and choose properties then select the Advanced button, you can find the path where your file is stored. Unfortunatley, Evolution can't import these files. You could export your Outlook info to .csv files and import them to Evolution, but you'll probably not be happy with the results. The best solution I came up with when I switched over was to use a Yahoo Mail account. You can import Outlook files to your Yahoo Conacts and then use Yahoo's export feature to export the data to a netscape compatible file, which can be imported into Evolution.
