### Post by DJ_Peng on 2009-01-19
The GlobalMenu replicates the Mac OSX-style menus on the top panel. It doesn't work with non-GTK apps, so OpenOffice.org and all Mozilla apps (Firefox, Thunderbird, etc.) don't work with the GlobalMenu, and sudo apps like Synaptic don't use the GlobalMenu for security reasons.

I've never seen anyone modify it to work with context menus, and I doubt it would be very easy to do, but you can ask over in the [support thread]("http://ubuntuforums.org/showthread.php?t=241868"). That's also the best place in the Ubuntu world for getting updates on what's going on with it.
