### Post by bonfire89 on 2008-12-16
by going to

Utilities>ark achieving tool

you can create a zip

click on new, and create the achieve, and then add your files to the achieve.


This doesn't seem ideal though, there should be something in the context menu, but might satisfy the situation for now.
