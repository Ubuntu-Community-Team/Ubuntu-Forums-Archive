### Post by mulder_edu on 2007-10-01
For those who've had problems using Personal Ancestry File (PAF), check out this thread:
[http://ubuntuforums.org/showthread.php?p=3457310#post3457310](http://ubuntuforums.org/showthread.php?p=3457310#post3457310)

This is a problem with the fonts in Wine.  You can manually change the font in PAF to make it readable.

I'm posting it under this heading in case anyone runs into the problem and does a search for PAF.

Solution:

Open PAF > Select Tools > Click Preferences > Find the tab that has a Red A, Blue B, and Grey C in the upper left hand corner > Click each of the four buttons to your right and change the font to something English readable (I prefer FreeSans) > Click the bottom far left button (this is your Okay button) > Restart PAF.

### Post by yeats on 2008-03-03
I know you posted this months ago, but maybe this will help.  I also use Personal Ancestry File with Windows, but after seeing that LDS doesn't support Linux, I searched for alternatives for Linux.  I came across [GRAMPS]("http://www.gramps-project.org/wiki/index.php?title=Main_Page"), which you can download from your package manager.  I have a dual boot installation of XP/Gutsy and all I had to do to move my information to GRAMPS was to export it in PAF as a GEDCOM, then import it into GRAMPS.  GRAMPS has more features than PAF and works beautifully.

Hope this helps.
