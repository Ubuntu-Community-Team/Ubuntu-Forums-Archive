### Post by trikster_x on 2010-11-05
If you want that much control over your icon-set, you are going to have to learn to create themes.  What you will need to do is download a few icon-sets, look at how they are made and see if you can throw one together using the icons of your choosing.

Fair warning:

I have done this on more than one occasion, and it is generally NOT as easy as copy/pasta.  Different themes are built with different file types.  Some use .jpg, some .png, and a very rare few use .svg.  You'll need make sure you have a version of the icon for each size that might be contained in the theme.  You will also need to change the name of the image file or modify index.themes to match the changes made to any files.  It's not necessarily a complex process, just time consuming.
