### Post by CocoAUS on 2007-06-08
System > Preferences > Preferred Applications

Ubuntu used to have a problem with not changing the url handers appropriately using this method, but the problem seems to have been fixed.  Changing your e-mail client and browser this way should associate all urls and links with the right application.
