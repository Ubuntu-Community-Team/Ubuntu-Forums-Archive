### Post by StuartN on 2009-12-19
> **polardude1983 said:**
> Or is there any way for me to save as an adobe illustrator ai file in linux at all???

Not as far as I know - Inkscape will read, but will not write, Adobe Illustrator files and there is no reliable conversion program. A PDF file might act as a (single layer) preview.

If you share (e.g. by ftp) rather than mail as an attachment then the browser might open the file correctly at the client end.

[http://wiki.inkscape.org/wiki/index.php/Inkscape_for_Adobe_Illustrator_users#Formats](http://wiki.inkscape.org/wiki/index.php/Inkscape_for_Adobe_Illustrator_users#Formats)
