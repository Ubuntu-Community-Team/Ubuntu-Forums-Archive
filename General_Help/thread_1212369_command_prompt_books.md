### Post by DGortze380 on 2009-07-13
IMO, the best skill you can learn is how to read a man page.

tab complete provides good practice.

example;

```

a[tab][tab]

```

Gives you a list of available commands starting with the letter A.

man [command] gives you the manual page for the command.

have fun!
