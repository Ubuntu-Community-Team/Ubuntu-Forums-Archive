### Post by pointone on 2007-04-15
1) To run something as super-user, use the sudo command. [Click here for more sudo information.]("https://help.ubuntu.com/community/RootSudo")

2) [Click here for Cedega information.]("https://help.ubuntu.com/community/Cedega")

3... or 4, rather) The 64-bit version of Linux can indeed run 32-bit applications; you just need to install all the required 32-bit libraries and whatnot.

5) The 64-bit version of Linux can run WINE or Cedega, which can run 32-bit Windows games.

6) It should, but the only way to be certain is to test (or find someone who has).
