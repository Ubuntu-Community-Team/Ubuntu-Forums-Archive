### Post by Mahogan on 2008-12-08
My heavy handed repeating key problem is solved thanks to a very wise Ubuntu Guru...  I disabled the repeating key function altogether and it works great for me.  Here is what VWUG wrote to me in an e-mail:

try going under System > Preferences > Keyboard, and changing the "delay" under the "general" tab? If changing the delay doesn't work, try disabling the key repeat feature entirely in that dialog box. I did a quick google search on reported bugs at bugs.launchpad.net, and maybe this might help:

[https://bugs.launchpad.net/ubuntu/+s...ev/+bug/264196](https://bugs.launchpad.net/ubuntu/+s...ev/+bug/264196)
