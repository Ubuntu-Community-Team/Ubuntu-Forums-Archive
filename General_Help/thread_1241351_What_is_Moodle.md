### Post by sideaway on 2009-08-15
Moodle is just a school subject collaboration intranet page of sorts. It allows teachers (lecturers) to create some web space based solely about a subject or class. You can have the lexture material, exam dates etc all synced on alocal calender etc. There are also built in boards in which you can ask questions.

Best bit about it is, it's open source :)

It's really only irrelevant if your school/university/college uses it.
