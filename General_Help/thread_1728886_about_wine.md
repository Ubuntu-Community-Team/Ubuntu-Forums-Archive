### Post by WorMzy on 2011-04-14
Unless you happen to run

```
wine install-keylogger.exe
``` to install it, and then run ```
wine stealallmydetails.exe
``` to run the keylogger, you should be fine.

Basically, only install windows executables that you know and trust, and downloaded from a reputable source.
