### Post by daisyfoofpoof on 2008-12-10
Hey guys.  First of all, I'm using Debian, but came here because they both use Gnome, and everyone is so helpful here.  :p  Anyway, every time I open a folder (or drive...or really anything that can be opened within the file browser) it opens in a new window...I've looked through the preferences a few times and can't find it...it's REALLY annoying when you have lots of folders to browse through.  Any suggestions?

---

### Post by ajgreeny on 2008-12-10
In nautilus preferences window, behaviour tab, select *Always open in browser windows*.
