### Post by lmarmisa on 2011-07-17
Are you referring to the trash folder of an external hard drive?. In this case the trash folder is named **.Trash-1000** (really 1000 is the **uid** of the user owner of the trash) and it is located in the root directory of the partition. You can empty that folder using the command **rm**.

In the case of the trash folder belonging to the a home directory, the trash is located at **~/.local/share/Trash**. Use the command **rm** if you want to empty that folder.
