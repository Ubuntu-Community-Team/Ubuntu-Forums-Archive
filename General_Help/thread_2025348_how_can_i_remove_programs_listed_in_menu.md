### Post by Cavsfan on 2012-07-14
**System** > **Preferences** > **Main Menu** is where you can uncheck the items that you do not want to display in any menu including Games.

### Post by Rex Bouwense on 2012-07-14
They don't take up a great deal of space, but if you want them gone open the Ubuntu Software Center and select Installed Programs.  Navigate down to Ace of Penquins and select Remove from system.
Alternatively, if you want to use Synaptic Package Manager, open it and select Installed (I believe that it is under Status).  Navigate to Ace of Penquins and select completely Remove from System.  Then Apply changes.  I think that was the procedure.
