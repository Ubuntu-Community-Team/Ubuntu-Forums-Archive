### Post by matt_symes on 2010-10-22
Hi

You might have problems running itunes under Ubuntu. There is no itunes specifically for linux. You can try and run it under wine but that is problematic and last time i checked only older versions would work. Ubuntu does have other music players and comes with rythmbox when installed

Try running it with the Live CD first. This will help identify any hardware issues and you get to play with the OS

Everything else you want is feasable. VNC or ssh is the way to go for remote access.

Kind regards
