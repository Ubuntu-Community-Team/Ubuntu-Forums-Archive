Kate, Zune does not work in Ubuntu natively, or in WINE.

The only way right now to get it to work is to install VMware Server 2.0, then install Windows as a guest OS and install the Zune software in that Windows install. It's not hard to do, but it's not necessarily easy for an Ubuntu beginner. Search the forums on how to install VMware, then google how to use VMware to install Windows.

It does work though. That's how I use my Zune software/Zune device.

Also, since you're new to Ubuntu and to the forums, a kind piece of advice: 
Try searching the forums for a little while first before posting questions. Chances are, 99% of the time your problem or question has already been answered. The Ubuntu Forums are chock full tutorials on how to get stuff done. The community is very large and always willing to help!

Welcome to the Ubuntu community :)
