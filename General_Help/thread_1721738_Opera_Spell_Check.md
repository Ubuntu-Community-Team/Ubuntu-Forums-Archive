### Post by stinkeye on 2011-04-05
I'm using 10.10 with Opera 11.01 and no problems.
Is it enabled in preferences > advanced > browsing ?

Right click in an input field in a web page 
and you'll see a listing for **check spelling** and  **dictionaries** that are installed.
You can add a dictionary here.
I'm assuming the English(US) dictionary is installed by default because there's no option to remove or add it.

**[COLOR="Red"]Edit [/COLOR]**: After messing about with this a bit,
when I restart Opera I have to select a language again in the right click menu for spellcheck 
to work and have to repeat the selection for each separate web page.
Email compose seems to work fine though.
