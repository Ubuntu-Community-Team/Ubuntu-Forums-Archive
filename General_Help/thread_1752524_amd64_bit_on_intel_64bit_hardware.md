### Post by gnaservicesinc on 2011-07-28
Well, there are a few differences when comparing the two instruction sets. But true, they are very close and you can run code compiled for one on the other and it is going to run totally fine.  

The intel's lack some model-specific registers, BSF and BSR behave differently,among a handful minor differences.  But none of that is going to cause you any problems so they are a bit difference but it dose not really matter.
