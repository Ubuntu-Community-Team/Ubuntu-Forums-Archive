### Post by benmoran on 2009-10-18
Since you're having the same problem on all of them, i'm going to assume they are all the same model of PC. In that case, please post the Brand/model number. 

Have these computers been kept up to date with upgrades? If not, try updating them first. If they have been kept up to date, then perhaps a recent kernel update is what did it. There is definitely a bug somewhere. You can try hitting escape at the grub screen and choosing an older kernel version to test. 

I know beta software isn't recommended, but perhaps you should try Karmic Koala on one of those machines. If it works out, update the rest of them once it's officially released this month. I've found it to be a fair bit more stable than Jaunty on my laptop.
