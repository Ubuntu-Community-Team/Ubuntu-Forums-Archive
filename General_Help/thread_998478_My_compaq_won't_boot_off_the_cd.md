### Post by falcon61102 on 2008-11-30
One thing you can try to just get the system up and running is to download the Super GRUB Disk ( [www.supergrubdisk.org](www.supergrubdisk.org) ) and use that to repair your MBR.  From there you will at least should have a working OS to deal with.

With the Ubuntu CD, make sure you run the Check CD command to ensure that the cd is good.  If that comes back ok, try starting up the Live version of Ubuntu and make sure that it at least runs out of the box with your system.

If you get some errors with the CD, burn the CD again but burn it at 1x and make sure you are using a CD-R.  

If you can't load the Live version of Ubuntu, post your hardware specs and you might have a hardware conflict that needs to be worked around.
