### Post by edm1 on 2011-05-03
What application is it? It is only possible with certain applications. There is no global implementation for starting minimised. You can check the manual pages (eg. 'man gedit' to check the gedit manual pages) to see if there is a suitable flag for your specific application. Then, if there is a flag (lets say '--startminimised'), you launch 'gedit --startminimised'
