### Post by mb_webguy on 2009-05-06
Go to System > Preferences > Appearance, the Theme tab.  You should see the theme there.  The background should appear on the Background tab.  If you want to use the Emerald theme, you need to install the emerald package, then select the theme in System > Preferences > Emerald Theme Manager.
