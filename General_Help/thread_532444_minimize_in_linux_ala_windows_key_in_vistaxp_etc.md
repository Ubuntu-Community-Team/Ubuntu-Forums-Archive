### Post by pjkoczan on 2007-08-22
If you're using GNOME, and you can still see your panels, there's always the "show desktop" widget you can add to your panels. It will effectively minimize all windows.

If you need to bind "minimize all" to a hotkey, I can't tell you how to do that offhand, but this might help. [www.gnome.org/learn/users-guide/gnome2.0-user-guide.pdf](www.gnome.org/learn/users-guide/gnome2.0-user-guide.pdf)

Hope this helps.

### Post by strabes on 2007-08-22
You should be able to go to Preferences, keyboard shortcuts and bind Win+D to "Hide all windows and focus desktop"

If not, check [this page](http://strabes.wordpress.com/2006/11/13/create-custom-keyboard-shortcuts-in-ubuntu/). The key name for the Win key is gnome is <Mod4>. So you would use "<Mod4>d"
