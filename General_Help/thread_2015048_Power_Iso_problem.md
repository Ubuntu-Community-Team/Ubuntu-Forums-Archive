### Post by Cheesehead on 2012-07-02
According to WineHQ ( [http://appdb.winehq.org/objectManager.php?sClass=application&iId=1867](http://appdb.winehq.org/objectManager.php?sClass=application&iId=1867) ), Rosetta Stone is a Windows application, and is not natively Linux-compatible. Some versions can be run in Ubuntu using wine.

Check your version of Rosetta Stone against the link above.
If your version is compatible with wine, then install wine from the Software Center.
After installing wine, then insert the Rosetta Stone CD and try to run it.
