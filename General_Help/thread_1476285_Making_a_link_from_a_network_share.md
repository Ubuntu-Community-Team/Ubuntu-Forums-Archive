### Post by rmcellig on 2010-05-07
How can I make a link to my network share so that when I want to access the share I don't always have to go to Places/Network all the time?

---

### Post by AlanR8 on 2010-05-07
Open Nautilus then from the menu select File then Connect to server. Make sure you tick the box to Make a Bookmark!

### Post by AlanR8 on 2010-05-07
Try Windows Share. That's what I use to connect to the wife's iMac.

Also make sure you've got a shared folder on the Mac
