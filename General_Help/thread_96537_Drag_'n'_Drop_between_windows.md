### Post by Hebus95 on 2005-11-29
With Gnome, when you have two windows and one is upon the other or in different virtual desktops, how do you move a file or a link between these two windows ?
With MacOS or Windows for instance, you can use alt + tab while you are doing a drag and drop. Another method is to fly over the window which is in background to make it in foreground.
Unfortunately, none of these two methods does work. Is there another solution or a trick to enable these features ?

---

### Post by Sutekh on 2005-11-29
Just hold the mouse (and the link/file) over the virtual desktop you want long enough and it should switch for you.

### Post by Sutekh on 2005-11-29
I'm sure that it could be tweaked, but I wouldn't know where to start.  GNOME forums perhaps?

As for Alt-Tab I'd be surprised if someone hasn't made a package to use alt-tab between all virtual desktops, it's just a matter of finding it. :p

Here's a long winded way,  select the link you want.  Click Alt Ctrl and Left or Right to switch to the workspace you want then still holding down Alt hit Tab to choose the window you want.  Unfortunately you need to use copy/cut and paste, mouse clicks don't seem to want to work (plus its pretty hard to have one finger on the mouse then use one hand to do all the key pressing!)  I actually use the scroll button on my mouse to change virtual desktops, which is faster than holding it, but still no dragging.
