### Post by Michael Schmidt on 2009-07-02
Iain Cheyne wrote some good short instructions.  Here is the link.
[http://iain.cheyne.net/](http://iain.cheyne.net/)
I also just got a Sansa Clip and am running Ubuntu 9.04.  It looks like syncing (a la Windows Media Player) isn't going to work.  However, if you go to Settings>USB on the Clip and set it to MSC, you can access it just like any USB device, like a flash drive.  Default is MTP mode - you can browse the files and copy them, but drag-and-drop doesn't properly sync them.
