### Post by ofb on 2007-11-17
No idea why you wouldn't get the latest from the site, but anyway, you can install Opera via Synaptic and get auto-patches. Probably have to enable third-party repositories "partners", if you haven't already. (That's if you're using 7.10. If 7.04, they'll be called "commercial") Current version is 9.24.

EDIT: When you right click Opera in Synaptic to select it, notice the option "Mark suggested for installation>". You'll want those two Flash items.

### Post by ofb on 2007-11-17
> **tezzaa said:**
>  I'm presuming it will or it would not be on the Add/Remove Applications list.  I'm I right ?

Yup exactly right. That and Synaptic are showing you what works with your system.

Also you'll get automatic notification of bug & security updates for the applications you choose, as well as for your Ubuntu system. This is a very nice feature.

I don't know how Add/Remove handles "Mark suggested for installation>", because I tend to use Synaptic. I guess you'll find out on your first Flash site.

Incidently, you'll get bug & security updates for your installed applications, but you will not get version upgrades. You only move to new versions of apps as you move to new versions of Ubuntu. So you may lag behind in new features sometimes, but you won't lag behind in fixes.

It's handled that way to make this business manageable for the developers who are making Ubuntu work for you.

There is also an exception called Backports. You can read about it here,
[https://help.ubuntu.com/community/UbuntuBackports](https://help.ubuntu.com/community/UbuntuBackports)
