### Post by warp99 on 2009-08-05
If the install disk boots up to a desktop then your graphics card is support with the included open source video drivers. Even if there's a problem with the graphics card Ubuntu will now boot into a "safe mode" and use the VESA drivers so no more dropping to a command line.
