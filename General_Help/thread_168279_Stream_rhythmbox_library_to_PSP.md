### Post by JackandJohn on 2006-05-22
There is a better solution; You can stream an amarok library to the psp:

-Install amarok (much better than streambox imo)
-install Amarok Web Front End (you may need to install the web server)
-You can now either; save all the pls files for the albums/playlists that you want, or browse directly to the server from the psp. :)


*This works best for firmware 1.5; if you have 2.6 or above, there is an alternate way to do it: through rss feeds - but I'm not sure on how to make that happen

### Post by evolvedlight on 2006-06-05
With 2.6, you can use PSPRSS - it will make RSS streams by artist. You then browse to local webserver and download the RSS files (or you can just play them direct off the site) and then use the RSS channel menu to play music
