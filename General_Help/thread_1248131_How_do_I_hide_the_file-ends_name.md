### Post by starcraft.man on 2009-08-23
This is a Microsoft thing for Windows. On Linux, filenames aren't modified like that removing the first extension to make it pretty. I haven't run across an option to do this in Nautilus. For the record, this Windows behaviour is what lead to so many people masking exes as other files... Linux doesn't rely on extensions for program association, but Windows still does so it's best not to try and modify extensions.

You could in fact simply remove extensions from say a song and it will still play on linux. If you ever revert to Windows it will be broken though.
