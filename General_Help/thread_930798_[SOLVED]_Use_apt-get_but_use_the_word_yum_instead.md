### Post by tommygunner on 2008-09-26
I would like to continue to use apt-get for package management but would like to use the word "yum" instead. It is shorter and much easier to type and I don't have yum installed so I'm wondering if it is possible. Anyone?

---

### Post by fwre01 on 2008-09-26
you have to create an alias in your ~/bashrc file that looks like this
```
#-------------------
# Personnal Aliases
#-------------------

alias yum='apt-get'

```
