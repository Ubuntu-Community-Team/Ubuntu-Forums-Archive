### Post by drojan42 on 2008-11-21
Finally found out... you've to pull the window from the left hand side to expose the menu of tools. By default the tools aren't shown. This, along with not having tools in the edit menu make for a frustrating precursor to using the tool.

Not nearly as frustrating as trying to use it with photos though. I ran back to windows and irfanview to manipulate the photos, thankful for dual boot.
