### Post by leg on 2008-10-06
Java is in synaptic so can be installed quite easily. For C++ you will probably want to have gcc installed. The best way to do this is install build-essentials through synaptic.

IDEs you have netbeans or eclipse for Java which are available through synaptic. For C++ I quite like Code::Blocks but I am not sure of its current status and if it is in synaptic or not.
