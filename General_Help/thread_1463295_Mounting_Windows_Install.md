### Post by Mark Phelps on 2010-04-27
I don't believe there's any way to install MS Windows from inside Ubuntu.  

Even though you may be able to mount the ISO, you can't run MS Windows executable files inside Ubuntu, so there's no way to launch the installer.

And before anyone jumps in here saying you can use Wine for this -- you can't.  Wine will run SOME MS Windows programs, but you can't use it to install MS Windows.
