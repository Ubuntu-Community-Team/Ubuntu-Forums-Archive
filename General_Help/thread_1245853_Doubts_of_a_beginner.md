Those terms are used to distinguish the different versions of Ubuntu. Jaunty is the newest version (9.04), Karmic is the current beta version (9.10), and Dapper is the oldest one you've go there (6.06) unless I missed one.

Ubuntu, Kubuntu, and Xubuntu are all essentially the same except that Ubuntu uses the Gnome desktop environment, Kubuntu uses the K desktop environment (KDE), and Xubuntu uses the XFCE desktop environment. Xubuntu is also mainly used for lower end computers.
