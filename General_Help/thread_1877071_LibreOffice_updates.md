### Post by dmoconnell on 2011-11-07
I know that when you install Google Chrome it adds a PPA and it updates through the update manager. try running a manual check for updates. (go to update manager and click the check for updates button) See if that kicks anything out.
Also I should note that after checking out the LibreOffice PPA, Ubuntu Maverick (10.10) is only up to 3.3.3 (Natty (11.04) is at 3.3.4 and Ocelot(11.10) is at 3.4.4RC (release candidate)) 

Sometimes PPA's doesn't kick things out right away and (in my experience) If their are no other updates then Update manager doesn't notify you or it will not show them till you do a manual check.
hope this helps
