### Post by sgosnell on 2009-02-27
It's a hidden folder in your home directory named .tomboy.  Any file or folder with '.' as the first character is hidden.  You can find it by opening the file manager, clicking on View, then check Show Hidden Files.  It will then show all files and folders, and you should see the .tomboy directory.  Your files are inside it.
