### Post by Kethinov on 2006-04-24
Yes, Ubuntu will work with that card. You will, however, have to take some extra steps to install the proprietary nvidia driver, however. Without doing that, you will be limited to 2d graphics only.

The only difference between Ubuntu and Kubuntu is the default desktop environment. Ubuntu defaults to GNOME. Kubuntu defaults to KDE. And you can install both in either. So it doesn't really matter which one you pick.
