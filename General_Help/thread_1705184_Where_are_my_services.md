### Post by PoHandle on 2011-03-11
You can use *bootup-manager* and/or *sysv-rc-conf* to configure services.
Bootup-manager is very easy to use, but sysv-rc-conf offers many more options.

```
sudo apt-get install bum
sudo apt-get install sysv-rc-conf
```
