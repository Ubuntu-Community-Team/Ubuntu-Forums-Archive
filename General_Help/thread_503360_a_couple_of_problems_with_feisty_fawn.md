### Post by latecomer on 2007-08-15
see [here]("http://ubuntuforums.org/showthread.php?t=283356") and [here]("http://ubuntuforums.org/showthread.php?t=83739").

I've got the same problem. To me, it looks like you have a choice between either using the regular shift key (holding it down) to type numbers, which is a bit of a hassle if you type a lot of numbers, or going into System > Preferences > Keyboard > Layout Options > CapsLock key behavior, and tick the last option (CapsLock toggles Shift so all keys are affected). 
This will make the ShiftLock key do what you want it to do regarding the number keys, but for some reason it will also stop the ShiftLock indicator light on your keyboard lighting up when pressed. It does still work, though. You just won't know it's on from looking at your keyboard.
Gotta love it, eh?
