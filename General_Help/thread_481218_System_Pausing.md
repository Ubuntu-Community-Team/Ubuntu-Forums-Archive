### Post by biobrew on 2007-06-23
This worked for me:

Edit the file: /boot/grub/menu.lst

Find this section and make it look like this:

#
# examples
#
# title         Windows 95/98/NT/2000
# root          (hd0,0)
# makeactive
# chainloader   +1
#
# title         Linux
# root          (hd0,1)
# kernel        /vmlinuz root=/dev/hda2 ro
#

kernel acpi = off noacpi



I had read previously that if affected wi-fi sometimes but it had no affect to me, except tailing /var/log/messages finally got quiet
