### Post by wahoobob on 2008-06-11
I have installed Virtualbox in a seamless mode and now that I have desktop effects running the seamless mode screws the desktop.  How do I return to regular mode in VBox so the effects are able to work?

---

### Post by sdennie on 2008-06-11
If you haven't changed the default host key, hitting Right Ctrl+L will switch you back to normal mode.
