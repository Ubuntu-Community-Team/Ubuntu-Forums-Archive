### Post by megabuffen on 2011-03-31
I currently have my email, contacts and calendar on two Windows PCs and my iPhone. I also have a laptop with Ubuntu and I would really like to be able to sync my information from Outlook to this laptop because I prefer to use Ubuntu whenever possible.

Syncing with Outlook on multiple computers is easy because I just have to place the .pst file in my Dropbox to keep it synchronized, but Outlook won't work under Wine. iPhone was a little bit harder because in order to avoid iTunes I had to use Google Calendar, and that just gives me the calendar and not my contacts or email messages.

Does anyone know of a solution that would be capable of syncing Email, Contacts and Calendar between Outlook and Evolution? If not, is this possible by dropping Outlook altogether and running Evolution on my Windows computers?

I've read that Evolution is capable of using an Exchange server. Is there any open source implementation of this so that I could use my Ubuntu server for Exchange? I believe this would solve the issue of syncing e-mail, contact and calendar, but would it work with multiple e-mail addresses? I need to be able to both send and receive mail from multiple POP3 accounts. An Exchange-compatible server would be a nice solution because iPhone has native support for it.

Any help and ideas would be greatly appreciated.

Cheers.
