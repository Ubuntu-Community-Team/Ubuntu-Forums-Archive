### Post by tact on 2008-07-23
there are probably other ways to do it...  try this:
click on System>Preferences>Appearance
Click on the customise button (will customise the current theme)
Click on "Colours" tab
Beside "Input Boxes" and change the text colour

I just did a test, changed it to red...  and all my text input boxes, including firefox location bar - text turned red.  (Remember the colour code before changing in case you want to go back).

Cheers
