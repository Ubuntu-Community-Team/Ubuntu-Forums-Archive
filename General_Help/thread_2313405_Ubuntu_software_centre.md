### Post by grahammechanical on 2016-02-12
@Edward 000

We do not know what version of Ubuntu you are using or if  this is a new installation. Have you updated the system recently? You  may need to go to System Settings>Software & Updates and activate  some of the repositories there. Have you tried clicking "More." That  should present a list of  What's new.

As regards Gnome software

Ubuntu  16.04 is not yet released and the 16.04 development version has not yet  received Gnome Software to replace Ubuntu Software Centre. We have to  add a PPA to get it and Gnome Software loads but as an app store the  "shelves" are empty.
