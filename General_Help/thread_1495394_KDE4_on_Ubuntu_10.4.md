### Post by cascade9 on 2010-05-28
More than possible. 

sudo apt-get install kubuntu-desktop  

Or 

sudo apt-get install kde-minimal 

I'm pretty sure I've seen people on this forum say that kde-minimal is buggy, but I dont know, I havent tried it. 

BTW, if you d/l the kubuntu installer KDE is the standard desktop, like gnome is for ubuntu.
