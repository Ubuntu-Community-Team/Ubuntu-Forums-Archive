### Post by JECHO on 2008-09-07
> **douggiephresh said:**
> I have AMD 64 and I used that installation for the splash screen. I followed all directions, got no errors, but why when i boot up now do i Just see words? It looks like some sort of system check. What am i doing wrong with this Fingerprints splash screen?

if you havent already then go to the add/remove programs and install "startup manager"


when that is installed open it. (system > administration > start-up manager)

under the boot options tab there is a section titled "misc." and there are three check boxes. Mark the "show boot splash" and unmark the "show text during boot". close the window then reboot.

hope that helps!
