### Post by An Sanct on 2011-02-28
You have to have the $name$.torrent files and the unfinished download (right click on item in transmission and select "Open Folder"), then copy those files and in the new install open the torrents and select "select loaction", point to the copied files. (keep a backup until you are sure that transmission has accepted the older files)
