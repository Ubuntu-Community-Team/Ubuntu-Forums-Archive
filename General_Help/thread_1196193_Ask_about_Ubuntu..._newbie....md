### Post by The Cog on 2009-06-24
Ubuntu, Kubuntu and Xubuntu are basically the same operating system but with different GUI libraries. They look different. It's a bit like 'skinning' a program for look and feel. They tend to have their own programs for things like file manager, music player etc, and although any of those programs will work on any of the systems, they tend to look a litle out of place because the controls look a little different.

Ubuntu has drivers for most hardware but not all. It is best to check before committing to hardware. Windows drivers won't work in Linux.

I don't know which desktop Mandriva uses these days. It used to use KDE (same as Kubuntu). Mandriva uses a different package system (like .MSI installers) to Ubuntu though. Ubuntu uses .deb installer files, Mandriva uses .rpm files. Other than that, the main difference is in the programs they choose to install by default.

Do a Wubi install to try it out first. After you burn your Ubuntu ISO and re-insert the CD, the auto-run feature in Windows will give you a Wubi install option. This is a nice way to go about trying Ubuntu if your not certain, as it lets you have a reversible dual boot set up. If you decide Ubuntu is not for you after running it from a Wubi install, then simply uninstall it using the Windows software uninstall utility; if you decide you'd like to switch to Ubuntu completely, then that can be tackled after you have tried it out in a safe and sane manner.

As always, Back Up, Then Move Forward.

### Post by gnoob on 2009-07-18
Another option besides doing a wubi install is just trying out ubuntu using the live cd (this runs the whole operating system off the cd without making any permanent changes to your computer - this will be much slower than a real install). To do this, insert the burned iso cd and restart the computer. The ubuntu cd menu should come up and then its pretty self explanatory. once you are in the live cd environment you can test it out to see if everything works.

a note for burning your iso cd: burn at a slow speed (4x or less)  you dont want errors on your OS install disk.

Viruses: the simple answer is you dont have to worry about viruses. thats windows BS. you can do some googling if you want to know more/why.
