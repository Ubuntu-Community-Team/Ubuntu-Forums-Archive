### Post by andrewkirk on 2010-05-21
Help.

A little while ago my Firefox in Ubuntu started playing up. When I click on a link on a web-page to a pdf or some other downloadable item, it does nothing, just ignores the command. If I open up Download Manager it has no sign of any download occurring. When the trouble first started, I could sometimes get around it by choosing Open rather than Save File when I clicked the link, but now thatdialog box doesn't appear any more.

Currently, when I need to look at an online pdf or download something I need to switch to another browser like Konqueror or Chrome and use that to download, which works fine.

I have tried uninstalling Firefox and reinstalling but that didn't fix it. 

Does anybody have any ideas why this peculiar problem would occur? I have tried googling it and searching this forum with no success.

The problem computer is a 32-bit Intel-based PC  running Karmic Koala, with Firefox version 3.5.9. Display manager is Gnome.

Thanks for any help.
