### Post by Shadius on 2012-07-14
Themes & Wallpapers: [gnome-look]("gnome-look.org")

Wallpapers: [LIST][DesktopNexus]("DesktopNexus.com")[/LIST]
[LIST][DeviantArt]("DeviantArt.com")[/LIST]

You can use *Wallch* to have the wallpaper change in a set amount of time. It's in the Ubuntu Software Center. Unfortunately, I don't know if these sites have a folder of wallpapers that you can just download so you might have to browse through some wallpapers that you like, download them individually, and put them in your folder that you'll use for wallpapers. Hope that helps.
