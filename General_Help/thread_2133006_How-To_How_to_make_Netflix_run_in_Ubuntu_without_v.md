Finally, after many, many years of no Netflix support on Ubuntu, a programmer named Erich Hoover has released a PPA that will make Netflix run in a Firefox browser through a Wine back-end. 

Here is a link to the how-to:
[http://www.techrepublic.com/blog/opensource/how-to-get-netflix-streaming-on-ubuntu-1210/4019](http://www.techrepublic.com/blog/opensource/how-to-get-netflix-streaming-on-ubuntu-1210/4019)

I tried this today and it absolutely works. I dunno if anyone on these forums had heard of this development yet, so I thought I'd share it. It says it's only for 12.10, but it will work in earlier releases as well.
