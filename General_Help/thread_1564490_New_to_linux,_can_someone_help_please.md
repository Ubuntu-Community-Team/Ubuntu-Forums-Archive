With the wireless nic download, I'm assuming it was archived into one file like the equivalent of a zip file?  Once you extract it, the vendor may have some documentation for it in a readme.txt file or something to its equivalent.  

If you used Synaptic/Software Center, everything that's installed with an intended GUI will be located somewhere within the menu, while loosely categorized under its intended function, office, internet, and so forth.  Not everything you install will be located there, nor should they be (like media codecs).

In a terminal, type man hier for a better understanding of folder's in linux.
