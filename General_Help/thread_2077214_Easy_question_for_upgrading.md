Basically, I've been using 12.04 with Gnome 3 as my WM. This was fine save that XBMC and Unity didn't function (not a huge deal as it's a dual boot comp.)
I decided to upgrade via update-manager to 12.10 not thinking it'd break some stuff. Apparently though the notifications and other things are messed up so I'm looking to fix it.
When I installed, I set different partitions for /, home, and swap. So can I just use a 12.10 DVD to overwrite just root and leave home without changes?
I wouldn't mind trying Unity again and if I don't like it, Gnome is only a command line entry away.

If I do have to lose everything it's not a huge deal (only real things were Amazon music on there and all the docs are backed up) but I'd like to see if I can just overwrite the OS only.

### Post by zombifier25 on 2012-10-28
> **cincinnatus13 said:**
> When I installed, I set different partitions for /, home, and swap. So can I just use a 12.10 DVD to overwrite just root and leave home without changes?

Yes you can. When asked the location to install Ubuntu, choose to configure it manually. Set the old / partition to be the new / and tick to format it. Same for /home, but don't choose to format it.
