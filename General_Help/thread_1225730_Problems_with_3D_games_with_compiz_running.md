### Post by Dave_Jackson on 2009-07-28
Compiz is great, but it has a penchant for causing problems with 3D games. 
In my experience, one of the most common problems are full screen games for some reason losing focus during gameplay. Sure, you could turn compiz off every time you play a troublesome game, but I've found a better way.

Go to Synaptic and install CompizConfig Settings Manager. Click General Options, and uncheck Unredirect Fullscreen Windows. Next, click the Focus and Raise Behaviour tab. Set the Focus Prevention level to Normal. I found that if you set it to High or Very High, this causes problems with multiple open windows by making the focus stealing prevention too strict. 

I found this solution solved my problems in all 3D games I have tried it on, including Sauerbraten. Google it. It's a pretty good game.

While compiz has improved greatly over the years, there are still some lingering issues like these. If anyone has anything to add, please do. I hope I've been able to help someone enjoy Linux even more.
