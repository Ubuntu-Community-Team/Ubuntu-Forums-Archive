### Post by danwood76 on 2008-01-23
It depends what sort of sites you are viewing.
If its simply a browser check then you can use the user agent switching plugin to make it look like IE to the website ([https://addons.mozilla.org/en-US/firefox/addon/59](https://addons.mozilla.org/en-US/firefox/addon/59)).

If you actually need something like active X then you may be able to install it through wine, though I've never done it.
