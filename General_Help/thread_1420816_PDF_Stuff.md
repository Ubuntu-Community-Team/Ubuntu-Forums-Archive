### Post by Pollox on 2010-03-04
If you're just using acroread to print multiple pages on one sheet, you can do the same thing with evince (aka Document Viewer aka the default pdf reader), which has a print to file option.  Just go to Print and click the Page Setup tab then specify "pages per side".
