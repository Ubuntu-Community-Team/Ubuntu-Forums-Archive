### Post by taurus on 2009-01-14
How did you burn the ISO image?  Did you burn it as an image file or as a data file?  If you look at the CD, you should see a bunch of files and directories.  On the other hand, if you see one large file that you downloaded, then you have burnt it wrong.

You need to burn it as an image file, whichever burner app you are using.  Also, make sure you have CD-ROM as the first bootable device in the BIOS.
