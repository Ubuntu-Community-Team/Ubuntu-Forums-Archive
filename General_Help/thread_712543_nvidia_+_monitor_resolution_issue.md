### Post by Sam Lars on 2008-03-01
If you could post large sections of text like that within code brackets (# on the toolbar), they will be more convenient to read. :)
So you've done nothing to your xorg.conf, and you can get that resolution, but you have to pan around to get it?
First, try running
```
sudo nvidia-settings
```
I think that's the easiest way to configure Nvidia cards.  It might say something like 1440x900 @ 1280x1024, which means the resolution is 1440, but it's only displaying a 1280 chunk.  Try getting it to run straight 1440.
If that fails, and you'd like to try something else...
Try commenting out (with #) all of the modelines (in section Monitor) and modes (in section Screen) that don't say 1440 by 900.  Make sure that still works, and then run this command:
```
cvt 1440 900 60Hz
```
Comment out the modelines (in section Monitor - you can leave them in section Screen) that you left from before, and just add the one from the command.  See if that works.
