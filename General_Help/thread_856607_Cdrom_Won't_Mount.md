### Post by mc4man on 2008-07-12
i gotta run but here's some basic info, maybe helpful, maybe not.
Audio cds aren't mounted, they're accessed at a location.
There's basically 5 ways.
You put a cd in and an app autostarts, rhythmbox and both banshee versions (if installed) are auto added as a default choice. 
if autorun is disabled or doesn't happen you could
right click open with...........
a r. click open should open a file browser at the location
a d. click on icon should open a file browser at the location or 
you could go to the location in the nautilus location bar
The location should always be cdda://<device>
for instance cdda://scd0/
If it's a data cd then it should auto mount, try putting one in and the running sudo lshw -C disk and dmseg | tail to gather some info
So it would be interesting to know what happens from these possible actions and what are you running in the terminal to 'mount'
