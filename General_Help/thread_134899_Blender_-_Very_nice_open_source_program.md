### Post by pixelnate on 2006-02-23
It is really not that difficult to do once you get a handle on the basics. Learn how to put reference images in the viewports, learn how to select and create points, learn (and love) the extrude tool ('e' key), and you will be on your way. Their are tutes listed on [www.elysiun.com](www.elysiun.com) to figure out how to use ramps to create the shine on the body.

There is no time like the present to learn. Just jump into it, it really isn't that hard. Now figuring out the best way to use Yafray for radiosity, that is difficult.:-k
