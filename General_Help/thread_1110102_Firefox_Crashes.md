### Post by Confuseling on 2009-04-07
[http://ubuntuforums.org/showthread.php?t=1114573](http://ubuntuforums.org/showthread.php?t=1114573)

Some people have had good fortune with some of the pre-release versions - in fact, if your browser desists from crashing for long enough for you to google firefox and shiretoko or minefield you'll find that the general consensus is that they work very well. There are even workarounds that will let you use many plug-ins.

Read the disclaimers though - these are thoroughly prerelease (especially minefield). They cannot be used on systems that cannot afford occasional random system death. Such is the way with software testing.

However, if it were me, and I had to use FF for my livelihood, and it kept crashing, I'd be tempted to install one on a system that I could afford to reinstall every so often...
