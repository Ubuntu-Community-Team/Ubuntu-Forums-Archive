PlayOnLinux is merely a front-end for Wine. PlayOnLinux allows you to manage different Wine prefixes for different applications, so when a app malfunctions and trashes the whole Wine drive, you only need to delete said app's Wine prefix.

### Post by Mark Phelps on 2012-10-17
Sorry to break the "bad news", but Play on Linux is not a miracle solution for running the most-used Windows apps: Office 2010 and Internet Explorer v9.  The first doesn't work well in Linux, in fact, some components don't work at all.  The second doesn't work at all in Linux.

With the release of Windows 8 soon, this will all get much worse.  Office 2013 doesn't work at all in Linux, nor does Internet Explorer v10.

IMHO, the major challenge in migrating to Linux is moving away from daily use of Office apps and IE -- something that is extremely hard for a lot of people to do.  Play on Linux helps with older versions of those, but most people use the current versions in their workplaces.
