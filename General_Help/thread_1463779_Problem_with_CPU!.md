### Post by The Real Dave on 2010-04-27
Top is installed by default, in almost all Linux and Unix variants. 

Simply open a terminal [Applications>Accessories>Terminal], type 

```
top
```

and hit Enter/Return.

Take a screenshot, and post it here.

Do whilst the CPU usage is high, so we can see what's using it.
