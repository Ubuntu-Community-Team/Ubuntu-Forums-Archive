Problem is, he already has an Ubuntu installation on his computer -- he resized his Windows partition. Because of this, he has grub installed. ### Post by chicken101 on 2008-07-16
You can go ahead with the new installation of ubuntu, you shouldn't run into any problems with it.  However, I would advise that you head over to [here]("http://www.supergrubdisk.org/index.php?pid=5"). 

The supergrub disk is a terribly handy resource when dealing with GRUB problems.  Basically it is a tool (a bootable .iso) which fixes most GRUB problems in seconds.  So if you do happen to run into a snag, you won't have to muck about dealing with fixing the GRUB manually! It's a life-saver.

For more info, take a look at their wiki which can be found [here]("http://www.supergrubdisk.org/wiki/SuperGrubDiskDocumentation").
