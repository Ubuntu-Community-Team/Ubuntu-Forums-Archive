### Post by linuxman94 on 2010-05-25
I think that when you "installed" it to the drive, you really didn't install it, you just put a live environment on it.  That means that your settings and installed programs are only saved to the ram and they are erased after a reboot.  If there is an icon on the desktop that says "Install Ubuntu 10.04 LTS" then that is the case.  You should use that to install it permanently to the hard drive.
