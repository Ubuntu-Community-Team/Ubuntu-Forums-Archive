### Post by Holger_Gehrke on 2015-01-07
And one more important bit about calibre: it's not a monolith. It's made up of a collection of tools which can be used independently of each other, namely ebook-convert, ebook-device, ebook-meta, ebook-polish and ebook-viewer. All of these will give short instructions on their use if called without parameters. So even if you don't use calibre to administer your library you can still use all of the other features of the program.
