### Post by JustinR on 2010-05-30
1.MSN
What browser are you using? MSN works fine in Opera.

2. Skype 
Skype offers a linux version specifically for Ubuntu on their website

3. Agreed except most programs are easy to install (especially through the Software Center/Synaptic)

4. System > Preferences > Keyboard > Layouts then click > Options.
   System > Preferences > Keyboard Shortcutes
_

Browsing the Ubuntu forums and clicking on topics of interest will get you places, I learned a lot just by doing this. Also googling problems I've had with Ubuntu led to my understanding of _a lot_ of things, including the terminal.

In WINE you can basically run most Windows applications. Go to the[WINE Database]("http://appdb.winehq.org/"), you can type in any program you have doubts with and it will tell you if you can run it in wine or not.
Also, search for "wine tricks" on google, its a script that lets you download development programs from Windows like Visual Basic, true-type fonts, etc for games and other programs.

The thing I've found that Ubuntu can do better than windows is customization. You can customize just about anything you could ever want to in Ubuntu. The biggest customization feature is probably desktop environments like Gnome, KDE, XFCE, LXDE, etc while Windows only comes with Explorer.
There's also ***great*** support for Ubuntu on these forums compared to Windows forums.
And, most programs are free and open source.

For every kind of multimedia file imaginable, including dvd's and even flash format (.flv), I would use VLC media player. It plays just about anything, including .mp4. Its available through the Ubuntu Software Center and Synaptic.

Virtualization (Emulation)
Ubuntu can be run on multiple OS emulation programs like VMWare, VirtualBox, VirtualPC and programs like that just fine. I'm not so sure for the iPad though.

Have fun!
