### Post by Mark Phelps on 2010-10-16
Well, it's already been two hours, so if you REALLY want the files back, you have to hurry to do the following:
1) Remove the drive from your machine, NOW
2) Find an MS WIndows machine, go to the Runtime Software site, download the trial version of Recover My Files -- and install it
3) Plugin your drive to this MS Windows machine as an external drive
4) Run  Recovery My Files and see what it can find
5) If it finds the files you need, go online, purchase it -- they will email you a product key.
6) Activate that product using the key they sent -- recover the files

And next time, when you need something critical from your machine -- be sure to make a backup first!  It would have taken you far less time to make a backup of XP and restore it than messing around with file recovery software.  And, you could have done that for free by downloading the image backup/recovery software Macrium Reflect -- for free.
