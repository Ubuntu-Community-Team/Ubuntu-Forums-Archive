### Post by jespdj on 2009-04-20
In Ubuntu, there is normally not a separate root password; in fact, the root account is disabled. If you need to do things which require administrator privileges, you use sudo. See this: [https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)

When sudo asks for your password, just type in the password that you use for your own user account.
