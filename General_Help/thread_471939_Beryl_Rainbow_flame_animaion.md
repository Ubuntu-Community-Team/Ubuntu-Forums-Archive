### Post by LollYouSuckZor on 2007-06-12
Select Animations, It depends what you want it to be attached to.  I have it on right now :) You will see things such as, Minimize, or... open, Something like that, there already should be selected animations in them, something like, Magic Lamp? Something.  Open that and scroll up to "Burn".  Select that for everything if you want everything to work with it.  Now, It should be red, and it may go slow... Scroll over, on the animations tab, and select, "Burn ( Fire )", Something near that.  Scroll all the way down and select, "Random color flames".  There it is. :) Enjoy..
