### Post by mc4man on 2011-05-14
You can get exaile into natty's sound menu or just add to the systray-whitelist
Not sure which works better, you'd need to decide for yourself
see here, read down towards end
[https://bugs.launchpad.net/exaile/+bug/588858](https://bugs.launchpad.net/exaile/+bug/588858)

The working plugin is here, download the 0.5 version
[https://github.com/sunng87/Exaile-Soundmenu-Indicator](https://github.com/sunng87/Exaile-Soundmenu-Indicator)

There is one issue in installing - inside the archive is a file 'readme.md'
As is it will prevent the plugin from being installed - 2 ways around
Extract the archive (r. click > extract here), then delete the downloaded tar.gz
Go into the extracted folder and either delete the  'readme.md' or r. click on > properties > permissions > and mark as executable

Then r. click on the folder > compress and compress back to a tar.gz
You then can add the plugin thru exaile's edit > pref.'s > plugins > install plugin 

To be able to min to sound pref.'s while playing you need to enable the 2 options in screen 2 (show & min.
(and don't pin exaile to launcher

(I decided to delete the readme.md
