### Post by ajgreeny on 2022-12-17
Xfce can do the same if you enable the xfwm window-manager compositing by using left Alt+Scroll.
I use it a lot and it is, I believe, a lot lower resource using than compiz. 

I have no personal experience of Mate's window manager but it seems that it xfwm can be used in Mate as well as xfce so it could be worth trying it out to see if the zoom works in that as well.
