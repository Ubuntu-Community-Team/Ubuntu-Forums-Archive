### Post by jackmacokc on 2005-04-26
Hey, i'm pretty new to ubuntu as well...but from my experiences with both the live version and my 3 installs that i'm currently working with, I can tell you that the live cd has very limited driver support.

Wireless support is pretty easy. Check [http://ndiswrapper.sourceforge.net/phpwiki/index.php/Installation](http://ndiswrapper.sourceforge.net/phpwiki/index.php/Installation) for help on getting it going in the install version. The live version isnt designed to have all the drives in the world..its just to check out the look and feel for the most part. 

I say give ubuntu a whirl..i'm having a ton of fun learning about *nix using ubuntu. I cant say i've gotten rid of windows yet, but its definately worth having installed in a dual boot configuration. 

hope that helps..
