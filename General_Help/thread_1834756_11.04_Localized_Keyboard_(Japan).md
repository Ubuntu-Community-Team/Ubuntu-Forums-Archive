### Post by whatthefunk on 2011-08-28
My output looks like this:
rules:      evdev
model:      jp106
layout:     jp
options:    lv3:ralt_switch

No variant.  If you did something like the code below, you could get it set up like mine, which has all the keys working properly.
```
setxkbmap -layout jp -model jp106 -rules evdev -options lv3:ralt_switch
```

If it doesnt work, try putting quotation marks around the options.  ("jp"...."jp106" etc)
