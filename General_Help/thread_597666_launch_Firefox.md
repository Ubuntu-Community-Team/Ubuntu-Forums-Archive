What do I have do to launch the Firefox application automaticaly when I login to the gnome desktop?

---

### Post by taurus on 2007-10-30
System -> Preferences -> Sessions -> Startup Program -> Add and type in firefox in the command: field.  Save it.

Then, log out and back in again.
