Hello,

I'm a new linux (ubuntu) user. I had a smilar problem while installing the .deb downloaded from the official site.
In order to solve the problem, I followed the steps described in the link below.

[http://blog.mypapit.net/2007/06/how-to-install-skype-on-ubuntu.html](http://blog.mypapit.net/2007/06/how-to-install-skype-on-ubuntu.html)



[LIST=1]
[*][COLOR=Navy]First of all you need to start up Synaptic Package manager. Go to System->Synaptic Administration->Synaptic Package Manager,[/COLOR]
[*][COLOR=Navy]From Synaptic, go to Settings->Repositories. Click on Third Party Software Tab. Click Add, and put on this line : [/COLOR][COLOR=Navy]deb [http://download.skype.com/linux/repos/debian/](http://download.skype.com/linux/repos/debian/) stable non-free[/COLOR] [COLOR=Navy]Click Add Sources.  [/COLOR]
[*][COLOR=Navy]Click Close, and Click 'Reload' at the top of Synaptic. Now you can search for Skype, and install  it from Synaptic [/COLOR]
[/LIST]
 Good luck.
