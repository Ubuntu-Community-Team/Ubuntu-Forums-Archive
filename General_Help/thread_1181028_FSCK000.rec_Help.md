I did find an app that can help in this situation.
TrID located at: [http://mark0.net/soft-trid-e.html](http://mark0.net/soft-trid-e.html)

This app can possibly identify the format of a file that you cannot based on the extension. If it finds the proper format, it renames the file FSCK*.rec.<ext> with <ext> replaced with the proper extension such as doc, pdf, etc.

This can be helpful to recover those files that were renamed.

Good Luck.

PS.  You can also teach it to learn new filetypes. Be sure to download the Database also. AT the moment this app is DOS based so you have to use the command prompt. Easy enough though.
