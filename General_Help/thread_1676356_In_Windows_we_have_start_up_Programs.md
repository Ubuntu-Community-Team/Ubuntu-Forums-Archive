### Post by Vaphell on 2011-01-27
deluge = deluge
screenlets = screenlets (most probably)
dc++ - no idea, though the package is called linuxdcpp so this can be it

either way it's easy to check
go to applications menu, right click on apps you want to run and select 'add launcher to panel' or 'add launcher to desktop' - right click these launchers to see the properties, write down the commands, go to startup applications and add new entries with these commands. You can delete these launchers from panel/desktop afterwards as they are not needed anymore.
