### Post by mgol on 2009-01-22
Strange. If it breaks in all computers, it seems it can be the CD itself issue. But You said You checked it and it was ok. If I were You I'd try to record the CD once again and install it another time.

I once got a somehow broken LiveCD from ShipIt. Test run fine, but CD was very loud, unusual to this CD drive. I tried recorded CD and it was ok.

This was the only time I got a broken CD from ShipIt, though.

You can also choose a "recovery" mode during the bootup and check what's happening.
