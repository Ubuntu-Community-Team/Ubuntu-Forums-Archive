### Post by varunendra on 2014-05-28
> **baabsaget said:**
> My wifi card is the Broadcom **BCM4331** and I am using the bcmwl-kernel-source driver.

Based on above confirmation, please try this -

While being connected to internet via cable or other means, please open a terminal (Ctrl-Alt-T) and install linux-firmware-nonfree pack with the following command -
```
sudo apt-get install linux-firmware-nonfree
```
Then purge the proprietary driver -
```
sudo apt-get purge bcmwl-kernel-source
```
Reboot, and check if the wifi works better now. If not, please post a detailed diagnostics report generated by 'wireless_script' mentioned here : [http://ubuntuforums.org/showpost.php?p=13024222](http://ubuntuforums.org/showpost.php?p=13024222)
