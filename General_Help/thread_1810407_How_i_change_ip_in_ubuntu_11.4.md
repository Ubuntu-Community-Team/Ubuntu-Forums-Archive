### Post by 2F4U on 2011-07-23
You can switch-off the automatic use of the cable connection in Network Manager, if thats what you want to do. If you want to change your IP address, that can also be done in Network Manager.
Whether you get an IP address automatically assigned depends on if you have a DHCP server in your network, for example a router that acts as a DHCP server.
