### Post by borgward on 2012-08-10
How do I reinstall the NetworkManager applet. Ubuntu 10.04 . I previously moved it, but it has disappeared from the panel. I can no longer get a wired or wireless connection. I am currently running a live Ubuntu 9.04 session on the same laptop, so the laptop is not the problem.

I did NetworkManager from the command line and got report that it is running, but have no way of accessing it. I have clicked all over the panel w/out having the dropdown appear.

The last time I was able to use the NetworkManager, it appeared as the battery/power status icon. I removed that from the panel, but now neither appear in the panel

---

### Post by borgward on 2012-08-10
Problem solved.

Right click on panel. Select add to panel. Select Notification Area. The NetworkManager applet icon will reappear as well as other notifications such as the Battery Status Icon.
