### Post by hawthornso23 on 2009-09-09
> **mike555 said:**
> === you can just add an hosts file to block ad's ===

A more elegant solution is to use a filtering proxy like privoxy. A host file can only block websites by URL - it is helpless against ads referenced by IP number and can't filter on the basis of other criteria (image size - cross-site information - etc). A filtering proxy can use filtering rules that are a lot more complicated.  A host file simply blocks the ads leaving web pages filled with ugly `missing picture' icons and frames with 404 errors. A filtering proxy can extract the ad from the page completely leaving a clean page with no ad on it.  A host file (depending on what is in it and how it is used) can also cause network problems like time-out delays. Using a host file for ad blocking is basically a hack - a clever hack and useful I will conceed - but still just a hack - and not the kind of thing I'd look at for a permanent solution.
