### Post by fyo on 2008-02-19
Firefox uses the default GTK dialogue (GtkFileChooser), so this is fundamentally not a Firefox issue (although it *could* be fixed in Firefox alone, but that would not be the elegant way of doing it).

There's actually a [bug report on Launchpad](https://bugs.launchpad.net/ubuntu/+source/firefox/+bug/89381) which discusses the issue and also has a suggested way of "fixing" it (requires compiling Firefox).

The latest Firefox 3 beta has added preview support (the inelegant solution, since the dialogue should consistent across all applications), so one solution would be to get the latest Firefox 3 beta and try that out. Remember to back up your profile!!!!

Another option is to switch browser. For example, Konqueror (and presumably other KDE-based browsers) have previews.
