### Post by Infinite Indefinite on 2008-08-31
I've noticed that the lightscribe program for Windows have some pre-installed templates, while the Linux program (4L) doesn't.  One of the templates that I liked on the Windows version was one that permitted you to put four pictures on a single disc.

I decided to create two such templates myself using GIMP.  They are attached with this post.  

To use the templates:

1) Open the template with GIMP, and select the colour-selection tool.

2) Select a colour and then delete it, (Edit ---> Cut)

3) Open your picture of choice as a layer.  Move, scale and/or transform it as desired.

4) Open the Layers Dialog, (Dialogues ---> Layer) 

5) Select the Background layer, right-click and select merge down.

6) Repeat steps 1-5 for the other colours.

7) Save final image as a png file.

8) Run 4L, load the png file and lightscribe it.
