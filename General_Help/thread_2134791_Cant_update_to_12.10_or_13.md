### Post by 3rdalbum on 2013-04-12
> **Fedelrick said:**
> 
Problem 3: After the update i went into update manager again and there was no updates available? weird? so i tried to update through the terminal and it just tried to update me to 12.04 again...
How can i fix this everyone? :/

The default setting in Ubuntu 12.04 LTS is to only notify you when there's a new LTS version. That won't happen until 2014 - 14.04.

Go into Ubuntu Software Center. Go to the Edit menu, and come down to Software Sources.

Click on the Updates tab and change the popup menu at the bottom from "For long-term support versions" to "For any new version".

Reload the package list, go into Update Manager and wait a few seconds and you'll be given the option to update to 12.10. If that's what you really want; I haven't found many differences between 12.04 and 12.10 apart from some newer software versions and a couple of minor (for me) new features in Unity.
