### Post by s3a on 2009-07-26
In my experience Ubuntu excels on the RAM department when it has at least 384MB RAM. Regardless, I would recommend anyone to do:

sudo apt-get install preload

This will preload the most used applications into your RAM for faster startup. Assuming it works the way I think it does, it adapts if you have a change in your software preference pattern. The more RAM you have for applications to be preloaded the less you need to panic about it not having your specific application preloaded.

What you should use depending on the amount of RAM you have:
0.5 GB RAM: Ubuntu 32-bit
5GB: Ubuntu 64-bit
