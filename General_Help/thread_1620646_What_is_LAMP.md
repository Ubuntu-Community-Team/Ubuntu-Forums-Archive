### Post by Solostian on 2010-11-13
To expand on thinkinhurtz's definition of LAMP...

LAMP is an acronym specifying a web server made out of open-source (free) software.

The L stands for Linux, the OS the whole shebang works from.
The A stands for Apache, a very popular HTTP server.
The M stands for MySQL, a relational database management system.
The P stands for PHP, as a server-side scripting language.
You can find useful info in related Wikipedia entries.

Some of these components are sometimes replaced by others. Most notably, PHP can be replaced by Perl, Ruby or Python.
