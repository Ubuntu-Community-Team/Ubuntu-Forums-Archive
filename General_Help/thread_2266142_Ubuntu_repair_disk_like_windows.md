Is there a way to repair Ubuntu by using a repair disk like windows? I have been forced to reinstall the system several times.

---

### Post by oldfred on 2015-02-20
No.

You should always have a current version repairCD or live installer to make repairs. Windows cannot repair Linux. A few third party Windows tools do recognize Linux formats but many do not.
And even Linux can only make some mostly minor repairs to Windows as the Linux tools cannot restore proprietary Windows code.

You should not have to reinstall, but can use live installer to repair if needed.
If you have issues with an install Boot-Repair may make some fixes or usually better to post the Summary report from Boot-Repair to a thread here and let use help you.

       Post the link to the Create BootInfo summary report. Is part of Boot-Repair:
[https://help.ubuntu.com/community/Boot-Info](https://help.ubuntu.com/community/Boot-Info)
