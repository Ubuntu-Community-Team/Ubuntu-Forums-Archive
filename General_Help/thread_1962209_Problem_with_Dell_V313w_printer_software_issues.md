### Post by Alexander uit Utrecht on 2012-09-24
The Dell V313W is equivalent to the Lexmark Impact S305 printer. As of Sept 24, 2012 Lexmark have updated the driver for this printer for deb and rpm packaged linux distribution, available at [support.lexmark.com]("http://support.lexmark.com/index?page=product&segment=DOWNLOAD&productCode=LEXMARK_IMPACT_S305&locale=en&userlocale=DE#1"). I have an Ubuntu 12.04 installation and it worked immediately without any problems. (while Dell is still not supplying the linux drivers anymore for this printer)
