### Post by approaching on 2009-08-19
Do you mean that firefox isn't showing up in the task bar (the panel down at the bottom of the screen where the programs running show up)? 

try pressing alt+tab to see if you can access that window if you aren't seeing it in the task bar. 

are you seeing both the bottom panel (which should have the open applications on it) and the top panel (which has the ubuntu menu and the tray and such on it)? you might need to adjust your monitor to see the whole image, or if you deleted that panel, you should make a new panel, then add the window manager to it.
