### Post by meatpan on 2007-06-30
A lot of people have painstakingly assembled printer compatibility information for linux.  You can see a summary here:  [http://www.linuxprinting.org/printer_list.cgi](http://www.linuxprinting.org/printer_list.cgi)

HP has some decent linux support. I've been using a deskjet for a few years with debian, (most recently ubuntu), without problems.

Take a look at the link above.  You might need to know some more printer details, which should all be on a sticker underneath the printer.  Also, you might get lucky and be able to download a driver and tools directly from ububtu (this is the case with some HP printers, via the 'djtools' package).

Good luck.
