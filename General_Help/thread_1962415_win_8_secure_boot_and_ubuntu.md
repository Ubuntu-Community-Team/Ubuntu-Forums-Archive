### Post by Lars Noodén on 2012-04-21
At the very least, secure boot will [make it difficult to install Linux](http://arstechnica.com/business/news/2012/01/microsoft-mandating-secure-boot-on-arm-making-linux-installs-difficult.ars).  The FSF seems to think it is likely that secure boot [would lock out Linux](http://www.linuxfordevices.com/c/a/News/Free-Software-Foundation-frets-about-Windows-8-secure-boot-requirement/).  With the history of the players involved, including messing with the [bootloader](http://www.birdhouse.org/beos/byte/30-bootloader/), there are a lot of risks.  Keep in mind that when installing dual boot Windows/Linux, Windows does not play nice and erases parts.
