### Post by grahammechanical on 2012-04-24
The purpose of Wubi is to give Windows users a taste of what Ubuntu is like without changing the hard disk. Wubi is not meant for long term or permanent use.

As regards your classmate saying this:

> I have a classmate who told me it has alo of crashes and is slow and shuts down suddenly alot.

I say that he does not know what he is talking about.

The great thing about the Linux operating system is that you can modify it almost endlessly.

The worse thing about the Linux operating system is that you can modify it almost endlessly.

You can modify any Linux distribution to the point that you break the operating system. That is the cause of so many user's problems and not the OS itself.

Remember, it is not going to cost you any money to download Ubuntu. So, it is not like you paid for the OS when you purchased the computer and are stuck with the OS that came with the machine.

You can always delete the two partitions that a Ubuntu install will create and then resize the Windows partitions to get the space back. And you will have lost not a penny.

Ubuntu is open source software. It will not install any closed source or proprietary software  without your permission. Sometimes we need closed source software because the open source alternatives are not yet up to standard.

You will need proprietary codecs for video playback and maybe a proprietary video driver to make full use of your video card. Ubuntu makes it easy to install this needed software but it is our choice.

Ubuntu would not offer this software without it being legal to do so.

I have been using the next release of Ubuntu (12:04) all through its development period. That is from November 2011 until now. Not once has it shutdown down. Because I was using a development version there were unexpected closures of this module or that module as the OS was modified to become 12.04 but none of these crashes stopped me from continuing what I was working on and I never lost any data as a result.

You will find that Ubuntu 12.04 like earlier releases is very stable.

Regards.

I think of Wubi as something I would recommend to students in a classroom environment who have Windows preloaded on their classroom computers. They aren't allowed to modify too much. They are only learning about Ubuntu Linux for a few months, and afterwards it will get erased eventually.  If you are worried about changing your hard drive paritions or whatever, and you want to get a feel for Ubuntu I would probably suggest installing VMWare player in Windows, and then install Ubuntu from the ISO file in VMWare player. Also after you take the plunge with Ubuntu and do install it to your hard drive, you can install VMWare Player in Ubuntu Linux and install Windows in VMWare.  It does a great job at helping new Linux users transition over to Linux.
