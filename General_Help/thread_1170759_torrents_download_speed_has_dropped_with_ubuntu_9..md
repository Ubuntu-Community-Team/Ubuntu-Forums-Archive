### Post by zobcat on 2009-05-26
Open Transmission. Go to Edit > Preferences > Network Tab

You should see *"Testing Port...* and then either Port is **open** or Port is **closed**

If it's open, then I'm not sure what you can do. But, if it is closed, go to portforward.com for a tutorial on opening your ports.

Good luck.
