It didn't help that MS chose a terrible, terrible file extension for it. Anyway, well now that we know of the true extension, it seems that very little apps support that format. Even MS' own apps don't support it.

For example see this tutorial about how to 'open' php files in publisher. 

[http://www.ehow.com/how_5964848_convert-php-files-microsoft-publisher.html](http://www.ehow.com/how_5964848_convert-php-files-microsoft-publisher.html)


Anyway, since it is so hard to find support for that format even in windows, perhaps using your method is really the only way. But next time, note that printing and scanning is overkill, you could just take a screenshot of the card:

* Open the card, once it is visible, press [print screen] in windows, then paste the screenshot in mspaint and save it as png.

* You may now use the png file.

If you have MS Works 2000, you may try to use WINE or virtual box to run it in your Linux setup.
