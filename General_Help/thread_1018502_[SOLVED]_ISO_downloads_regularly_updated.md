As time goes by and security patches and so forth are released, are those updates incorporated into the down-loadable ISO's for the current release versions?

Just a question from curiosity.

---

### Post by sdennie on 2008-12-22
Not usually, no.  The exception being the LTS releases where the minor version number increments.  For example, 8.04.1 is an updated version of 8.04.  I believe this is done because LTS releases live for much longer and it makes sense to update the ISO to the latest state of the OS a few times during the life of the release.
