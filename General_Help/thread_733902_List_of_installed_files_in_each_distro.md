### Post by cdenley on 2008-03-24
That would be a very long list. In your examples, you list packages, not files. If you want a list of user applications in the menu, I think you can boot the livecd, open terminal, then run
```

ls /usr/share/menu

```
I'm not sure if KDE and XFCE would use the same type of menu configuration, though. As far as I know, there is no comprehensive list for all user applications preinstalled for each ubuntu flavor.
