### Post by howefield on 2016-02-25
> **dave157 said:**
> I just need to know if I click on this it should load and let me add the package I downloaded? I Have not as yet downloaded a package. I click on it states not packages to install and closes is this normal. 

Right click the package and select Properties, the Open With tab will tell you which application is the default installer. If you want to use gdebi to install packages then you can specify that (if it isn't already already the default) - gdebi itself would need to be installed. But yes, double clicking on a .deb package should open the default application and install your software unless there is a problem with the package.

Beware of downloading packages from non official sites/repositories.
