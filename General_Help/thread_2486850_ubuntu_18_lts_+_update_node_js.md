### Post by Impavidus on 2023-05-14
For most software, the versions supplied from the official Ubuntu repositories get frozen for a particular release of Ubuntu. Important bugfixes get backported (for software in the main repository at least). For a more recent version of some software, move to a more recent release of Ubuntu, or get the software from some other source like a PPA (not recommended). Ubuntu 18.04 will reach end of standard support end of this month, so it's best to upgrade the OS anyway.
