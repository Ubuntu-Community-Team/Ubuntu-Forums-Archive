### Post by absolutezero1287 on 2011-10-15
Hello all. I upgraded from 11.04 to 11.10. The upgrade manager reported errors so I restarted, did a dpkg --configure -a, and my touchpad wasn't working. I am using a Lenovo Ideapad Z560. The touchpad is made by elantech. I read that some people had some success using the "synclient TouchpadOff=0" command. This has to be done on ever login to enable the touchpad. I thought it simpler to just add the command using the startup applications utility. The command I used was this: "sleep 10 && synclient TouchpadOff=0". I hope it works for you guys. One thing I've noticed is that clicking on my touchpad is less responsive. I don't yet know how to fix it but it is a minor annoyance. Before I simply had to double-click and drag on my touchpad to highlight text. Now I have to triple-click and it doesn't always work.
