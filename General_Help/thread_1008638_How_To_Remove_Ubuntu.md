### Post by guitarMan666 on 2008-12-11
I'm sorry to hear you're tired of Linux.  But if you are SURE you want to remove it, you should just use GPARTED to remove the Linux partition and then resize your Windows partition back to full size.  Make sure any data you have saved in Ubuntu is backed up onto your Windows drive.

If you installed it from within Windows, it should be an option in Add/Remove Programs (that's what it's called as of XP).  

[http://www.brunolinux.com/01-First_Things_To_Know/Windows_Tools_-_Gparted.html](http://www.brunolinux.com/01-First_Things_To_Know/Windows_Tools_-_Gparted.html) has instructions for acquiring and using GPARTED for Windows users.

Thanks for trying Ubuntu!
