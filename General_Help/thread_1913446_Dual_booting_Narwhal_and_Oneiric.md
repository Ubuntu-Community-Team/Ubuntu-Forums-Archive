### Post by Double.J on 2012-01-22
Hi there, It's usually best to set up your partitions pre install.

Boot the live media choosing try instead of install and open gparted - in 11.X, you just open the dash and type gparted, on 10.X it's under system tools usually.

_Always back up before editing partitions_

get them how you want them, then run the installer.

If all you want to do is go from 10.10 to 11.04 an update may be easier - open the update manager and see if it gives you the option to update to 11.04 (from inside the installed OS)

As for upgrading to 12.04 - release is expected 26 April so a good time to upgrade would be a month or so following... unless you have no need to, then of course keep using natty until November :)
