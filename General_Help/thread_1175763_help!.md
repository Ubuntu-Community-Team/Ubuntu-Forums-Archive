### Post by dnairb on 2009-06-01
I'm willing to bet that you have tried to disable a keyboard shortcut (in System -> Preferences -> Keyboard Shortcuts) by pressing <delete> in place of the allocated key(s).
Consequently the action is now allocated to the <delete> key.
Disabling a shortcut is achieved by using <backspace> key.
