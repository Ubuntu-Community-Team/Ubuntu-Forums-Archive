### Post by MWSTX on 2009-05-21
I'm running a fresh install of 64 bit jaunty on an HP Pavilion (2gig clock & 2gig RAM).  Install went smooth (best release I've tried!) but I'm having a problem transferring files to my Blackberry Bold (I want to be able to copy music & picture to and from the Bold). 

When I plug it in, the Blackberry mounts and I can navigate the folders and see the files but I cannot open.  When I tried to copy from the Blackberry to the desktop I thought that the File Operations window (Nautilus) was freezing up because nothing was happening (just showed a status bar that said something like "0MB of 4MB transferred").  After several attempts, I finally left the File Operations window open long enough to realize that data was in fact being transferred, but just at an incredibly slow pace (9hrs for a 4MB song!).  At  this rate it will take weeks to copy my songs & pictures from the Blackberry to my computer.

I've tried transferring to and from another thumb drive and it works lightening fast.

There also seems to be an issue unmounting the Blackberry.  A window pops up that says it is writing to the device and that I shouldn't unplug it, but the window never completes, it just times out and then I get an error message (not always the same one either).  After all of this the drive seems to finally unmount itself (5min later).

I'm hoping this is a simple fix.  Any suggestions?
