### Post by lovinglinux on 2011-07-04
Start Firefox in safe mode and disable the Global Menu Bar integration extension, then restart.

```
firefox -safe-mode
```

This will probably allow you to use Firefox, however you need to figure out why the global menu package is not being downloaded.

I would try this:

```
sudo apt-get clean
sudo apt-get update
sudo apt-get upgrade
```

You should get Firefox 5, which is the current version.

### Post by lovinglinux on 2011-07-25
> **Mortesins93 said:**
> What do you mean by ISO hash?

Files can get corrupted when you download them or burn them to a CD. However, you can check the file fingerprint, to verify their integrity, using a hash tool. This is also a good practice in terms of security, because if someone tricks your browser, making it download a different file, the hash won't match.

See:

[https://help.ubuntu.com/community/HowToMD5SUM](https://help.ubuntu.com/community/HowToMD5SUM)
[https://help.ubuntu.com/community/UbuntuHashes](https://help.ubuntu.com/community/UbuntuHashes)

I also develop am extension for Firefox that can generate and check hashes:

[https://addons.mozilla.org/en-US/firefox/addon/checkit/](https://addons.mozilla.org/en-US/firefox/addon/checkit/)
