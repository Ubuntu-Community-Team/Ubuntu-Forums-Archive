### Post by taurus on 2009-04-05
One way is to download it from [http://www.mozilla.org](http://www.mozilla.org) and unpack it somewhere in your $HOME.  Then, run it from there or create a launcher on the top panel.  Of course, you need to have the ia32-libs installed first if you want to run 32bit in 64bit environment.
