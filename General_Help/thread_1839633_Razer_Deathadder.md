### Post by 3rdalbum on 2011-09-06
I've had the problem that my Deathadder's sensitivity is too high. I just turned down the mouse speed in Gnome, which has brought it down to a better level, and I bought a more grippy mousemat from Razer that helps too, but I've just learnt to live with the slightly-too-high speed.

My Deathadder is a later model that doesn't work with the software someone made for it, and all the guides I found for sensitivity didn't work. I don't remember seeing that guide Grenage linked to so I can't comment about whether it works. Might do.

Unfortunately I'd never buy a Razer mouse again. The keyboard is alright, quite good action for typing, but the experience with the mouse has soured me.
