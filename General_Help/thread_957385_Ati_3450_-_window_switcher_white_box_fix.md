### Post by namtlade on 2008-10-24
I've had this annoying problem with my dell studio 15 with a 3450 in it for a while, in that when using the window switcher (alt-tab) the window previews were white boxes due to the rubbish drivers. And I managed to chance upon the solution so I thought I'd share.

Problem:

[IMG]http://i292.photobucket.com/albums/mm16/namtlade/Windowswitcherfail.png[/IMG]

Solution:
Turn off mipmapping in advanced desktop effects in the window switcher options:

[IMG]http://i292.photobucket.com/albums/mm16/namtlade/mipmap.png[/IMG]

Problem solved.

(this is the compiz advanced settings, not sure if this problem exists with the standard built in compiz)
