### Post by geezerone on 2008-01-13
**Dynamips** is probably the best for routers though switches isn't catered for. I have tried it a bit and it isn't the easiest to setup although the support forums are helpful.

You could try Cisco's own **Packet Tracer**, I think ver 4.1 is the latest. This is very user-friendly with the ability to select hardware in a drag/drop window system. You can then click on a device and go to command line to configure it - nice. I understand a lot of CCNA's would find this useful but don't know if it is as much for studying for CCNP or similar heights.

HTH :)
