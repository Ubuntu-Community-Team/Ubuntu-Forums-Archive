### Post by coool on 2008-04-30
I think the following procedure can solve the rotating cube problem. Try it out.
Go to system -> preferences -> advanced desktop effects settings.
Then go to the rotate cube feature and click on the bindings tab. Then, under rotate cube, set the rotate left and rotate right options with the mouse icon to 'Disabled'.
I think this should do it.
