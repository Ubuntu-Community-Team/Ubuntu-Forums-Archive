### Post by costis on 2007-01-23
Hello...
I would like to ask you how it is possible to have 3 users, each one with his own language settings. For example one's language and menus to be set in German, another's in French, and the default to be in English of course. 

I am asking you, because logging in as these users I cannot see the menu System > Administration > Language Support so as to be able to change their language.

---

### Post by toGoq on 2007-01-23
When you're at your login window (gdm or kdm) there is an option in which language you'd like to log in. Of course you'll probably need to install the different language-packages, if you want full support for those languages.
