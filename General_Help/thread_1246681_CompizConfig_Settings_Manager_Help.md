### Post by tgeer43 on 2009-08-22
From the Compiz Fusion Wiki located at:

[http://wiki.compiz-fusion.org/](http://wiki.compiz-fusion.org/)

"The **Show Desktop** plugin offers an alternative animation for the *Show Desktop* action (configured in **[General Options]("http://wiki.compiz-fusion.org/GeneralOptions")**). When activated, the settings for this plugin will determine how windows behave when *Show Desktop* is triggered"

"**Fade to Desktop** adds a smooth opacity fading transition to windows when the *Show Desktop* action (set in **[General Options]("http://wiki.compiz-fusion.org/GeneralOptions")**) is triggered. If this plugin is active, then all windows will minimize with the fading effect on this event, instead of animating according to the settings for the **[Animation]("http://wiki.compiz-fusion.org/Plugins/Animation")** plugin. They will also fade back in if *Show Desktop* is triggered again"
