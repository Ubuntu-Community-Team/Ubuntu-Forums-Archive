### Post by mc4man on 2011-10-14
Well it looks like the option has been removed. Vlc will allow mutiple instances if that suits you.

If you can't find a solution then there is an obscure way (sorta
That would be to use totem-xine instead for multiple instances or in addition to an open current totem
While it was dropped pre-karmic I've kept a version around on my installs since then

There has been a slight drop in function in releases after karmic, but I find still good for a local media player & dvd player
Screen shows 2 totem-xine & 1 totem open at same time 11.10 (32 bit
Has to be built locally
