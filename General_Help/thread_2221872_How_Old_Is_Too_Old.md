### Post by sudodus on 2014-05-07
New versions of Ubuntu aim at new hardware as well as old hardware. Making it work with new hardware will sometimes make it lose some capabilities for old hardware, even if it is not the intention. I think many people see the same trend as you, and it is often a good idea to stick to old but still supported versions, like Ubuntu 12.04 LTS and flavours and re-spins based on that version of Ubuntu. See also post #4.

> Does the way I use my machine seem to be taxing it beyond its limits?  This is in a way a silly question because it functioned fine with 13.04.  It just wasn't snappy fast like I see newer low end machines operating  today. 

Yes, I think you should run fewer, maybe only one web browser window and tab in order to get snappy response with this computer. Partly it is due to the new version of Ubuntu, partly it is due to the web sites, that gradually need more horsepower and memory. Some people would say that the websites get more bloated with eye-candy. There is also a problem, that the proprietary software flashplayer for linux is not very efficient, and not well supported.

### Post by sudodus on 2014-05-17
You have many questions. I will focus on the RAM and the mass storage device.

Upgrading RAM has often the best cost to benefit ratio. Maybe 4 GB is a sweet point for many people. It does not make a big difference to get 6 or 8 GB RAM. Instead you could change your browsing habits (have fewer windows or tabs open).

Maybe the second best option is to upgrade the hard disk drive to a bigger or faster one, or to solid state drive.

But sometimes you just need to get a new computer, because there are several bottle-necks in the old one, and new computers are rather cheap.

### Post by sudodus on 2014-05-17
I looked at the more detailed specs in the first post of the 'How Old Is Too Old' thread.

If your main concern is streaming video, I think that the CPU and graphics chip/card are the most important. If they are bottlenecks, it will not help much with more RAM and faster disk. (I think 2 GB RAM is enough for streaming video.)

The Sempron processor might be too weak, unless you can use the graphics processor to take over some of the job. This should be possible with an nvidia proprietary driver and at least some video formats (for example where VDPAU can be used).
