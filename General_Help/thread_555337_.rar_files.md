### Post by bitterbug on 2007-09-20
at the command line:

sudo apt-get install unrar

That will install the command line version of unrar for you. The nice thing is that the "Extract" tool in the context (right click) menus also detects it, and will let you then extract from the gui as well by acting as a front end for it.


edit: oops. I omitted the word install in the apt-get command.

### Post by Nebby on 2007-09-20
Apt-get will only work if you're on a debian-based distro or if you've installed apt-rpm. You said in another post you're using Red Hat, so I'd guess it won't work for you.
Firstly, the syntax to install with yum is
```
yum install [name]
```
so try using that. If that doesn't work, try Add/Remove programs, which'll be  in a system or settings menu (eg in KDE Fedora it's KMenu>System>Add/Remove Software)
