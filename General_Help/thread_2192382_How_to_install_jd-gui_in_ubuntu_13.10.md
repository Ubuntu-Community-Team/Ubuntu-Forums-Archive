### Post by heir4c on 2013-12-07
Here a download page with a tar.gz file: [http://jd.benow.ca/](http://jd.benow.ca/)

You don't have to install anything. You can run it with the executable file. How?: Unpack the tar.gz and go in the created folder: jd-gui-0.3.5.linux.i686. There you find a folder 'contrib'; an executable file jd-gui; and the file: readme.txt. If you doubleclick on the executable file it must start the program.
I hope this will be helpfull.
