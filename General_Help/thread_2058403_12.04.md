### Post by Epodx64 on 2012-09-15
Maybe it's because the swap is stuck in the LVM? I'm not real familiar with LVM's but if you need to create a swap file you could always do this 
sudo dd if=/dev/zero of=/swapfile1 bs=1024 count=524288

I dunno how much ram you have but that creates a 512mb swap file

then 
sudo mkswap /swapfile1
sudo chown root:root /swapfile1
sudo chmod 0600 /swapfile1
sudo swapon /swapfile1
sudo nano /etc/fstab
change /swapfile1    swap    swap   defaults 0 0
then reboot
then from a terminal type

free -m 

and see if the new swap is activate... might be an easier way to do this like I said I don't use LVM's so I might be off base.
