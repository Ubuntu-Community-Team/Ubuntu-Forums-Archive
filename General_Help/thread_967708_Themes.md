### Post by ibutho on 2008-11-02
Hi.

You can get themes from [GNOME-Look.org]("http://www.gnome-look.org"). Most don't come with separate fonts. As for installation, you can run System -> Preferences -> Appearance and drag your themes zipped file into that tool and the theme should be installed. The drag and drop method works for most themes but occassionally you may find some themes that need to be compiled from source.
