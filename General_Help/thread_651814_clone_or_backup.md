### Post by der_joachim on 2007-12-29
[This thread]("http://ubuntuforums.org/showthread.php?t=651967") covers a number of backup methods. Personally, I'd only backup /home and /etc. Just install a fresh Ubuntu, and restore your /home and /etc partitions from your backup. 

However, if you have a windows partition as well, which you frequently use, I'd clone. Reconfiguring windows is not as easy as in linux.
