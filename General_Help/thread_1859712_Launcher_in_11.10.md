Huh? It's exactly the same procedure as in 11.04 - install Compizconfig Settings Manager and you'll find every option in there.

The launcher appearing underneath windows is a bug. It's not one that I've observed, but I found out about it while looking at Launchpad. It should be fixed in 5, 4, 3, 2, 1.... so hang tight.

In the meanwhile, try Unity 2D. You can choose it from the cog icon at the login screen. I don't believe it has that particular bug.
