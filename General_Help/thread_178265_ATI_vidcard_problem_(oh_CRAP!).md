### Post by bluevoodoo1 on 2006-05-17
From the command line you can run...

```
sudo dpkg-reconfigure xserver-xorg
```

it's somewhat of an automated/graphical walkthrough to create your xorg file.

From there choose the "ati" or even "vesa" drivers. They won't look terribly nice, but it should get you going. When you get to the part about the monitor, let it detect its settings--and hopefully it will get the correct sync rates. 

when done.... 

```
sudo /etc/init.d/gdm restart
```

(if that does not bring you back to the graphical login, try typing "startx" first)
