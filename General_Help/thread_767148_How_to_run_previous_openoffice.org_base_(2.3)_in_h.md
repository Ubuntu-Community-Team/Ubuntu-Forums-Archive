### Post by thebrotherofasis on 2008-04-29
Thank you very much, the tutorial was of great help. Additionally, I found out that to be able to download previous versions of openoffice.org, in order to manually install them in ubuntu, there is an official mirrors site at openoffice.org:

[http://distribution.openoffice.org/mirrors/#mirrors](http://distribution.openoffice.org/mirrors/#mirrors)

From there, I could download the desired .deb packages, and the installation was really easy. I now have Hardy with Openoffice.org 2.3, and it is running like a charm!
