Firstly, you need to upgrade or reinstall a fully supported version of Ubuntu as 18.10 lost all support in July this year and therefore gets no further security or application updates.

Personally, I have never found hibernate worth bothering with as it takes almost as long as a cold boot on my hardware (it used to anyway; I haven't used it on this machine so can't really comment further).

Did you install Ubuntu or was it pre-installed by Asus?
