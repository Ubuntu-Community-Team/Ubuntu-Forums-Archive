Every time I open a virtual console, I'm greeted with a Welcome to Ubuntu message. I like to customize my motd and want to change this. I tried editing /var/run/motd and removing the Welcome part of the message. Next time I open a virtual console, the message is back again! How do I remove it for good?

---

### Post by Axanon on 2010-05-06
I found some documentation and realized that Ubuntu uses a different type of motd system called update-motd. What it does is makes generates the motd through running a series of scripts. These scripts are located in:

**/etc/update-motd.d/**

The script in particular I wanted is **10-help-text**. This generates the Welcome to Ubuntu message I want to remove. So I'm removing it.... Yup seems to have worked.
