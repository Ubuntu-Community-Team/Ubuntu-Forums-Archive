### Post by yancek on 2018-06-25
Windows 10 pre-installed is almost certainly a UEFI install.  The page at the link below is the official Ubuntu documentation on dual booting with windows UEFI.  Read that and post back with questions.  You will need to have a basic understanding of Linux drive/partition naming conventions and some knowledge of filesystems to identify which drive/partitions have windows.

[https://help.ubuntu.com/community/UEFI](https://help.ubuntu.com/community/UEFI)
