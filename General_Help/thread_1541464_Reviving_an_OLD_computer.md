### Post by snowpine on 2010-07-29
Pentium 3 600mhz is not too bad; I have a laptop with a similar processor (and 384mb ram), which I use occasionally for light web surfing and word processing (using Abiword, not OpenOffice!). I see that your motherboard can hold up to 256mb of RAM; my recommendation would be to install the maximum RAM (if you haven't done so already) and try any of the "lightweight" distros such as SliTaz, Puppy, TinyCore, AntiX, CrunchBang, etc. (DSL I would avoid since the project has been "dead" for a while now.)

### Post by dreamsR4living on 2010-07-29
> The first official version of lubuntu is released as a "stable beta" marking lubuntu as a usable system that is yet undergoing continuous improvements to enhance user experience and speed. Minimum requirements for lubuntu are comparable to Pentium II or Celeron systems with a 128 Mb RAM configuration, which may yield a slow yet usable system with lubuntu. Mario Behling, founder of the project from Berlin says: "It should be possible to install and run lubuntu with less than 128 Mb memory. For browsing the Internet without flash 64 Mb RAM is probably enough on some machines, but for other applications the speed may not be suitable for practical use."

Is what the Lubuntu website says about system requirements. If you want to try Lubuntu, install it from an alternate CD and not the Live CD, which uses too much RAM
