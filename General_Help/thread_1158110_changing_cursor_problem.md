### Post by Legace on 2009-05-13
I'll show you how to do it..

Go to [http://gnome-look.org/index.php?xcontentmode=36&PHPSESSID=70eef529fc7cf29e2cc38e212f216d67](http://gnome-look.org/index.php?xcontentmode=36&PHPSESSID=70eef529fc7cf29e2cc38e212f216d67) (or whatever page to download those cursors).

Download TAR file to your Desktop.

Open the Appearance tool. Drag and drop the TAR file into the Appearance window.

Select Customize in Appearance window. Select the "Pinter" tab, and you can change the pointer theme there :)


I used this file in the above example: [http://gnome-look.org/content/download.php?content=32627&id=1&tan=46613844](http://gnome-look.org/content/download.php?content=32627&id=1&tan=46613844)

(not all themes are compatible)

### Post by Tipped OuT on 2009-05-13
**Download a cursor pack. Extract it to your desktop. Open up the Appearance window. Drag and drop the entire cursor folder in the window. Make sure you have the cursor selected, (select your current GTK theme and click customise to change the cursor) log off, and log in again. Done.**
