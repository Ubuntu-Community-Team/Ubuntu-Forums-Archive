You might want to switch to "SwiftFox" and see if the crashing persists, or if you want to check your processes to see if anything is using a lot of CPU usage and is making FireFox crash you can simply go into your terminal and type:
```
top
```
Look for the process that you don't really use and end it, you should also get something called "Conky" it's a system monitor and you can download this by going into terminal and typing:
```
sudo apt-get install conky
```
You also might want to install the latest plugins for FireFox, possibly Java and a couple other ones.
