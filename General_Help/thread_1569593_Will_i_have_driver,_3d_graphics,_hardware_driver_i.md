### Post by valbaca on 2010-09-07
> **Jadesocket said:**
> by overwriting my windows xp and 2003? I don't have much room on my computer so by making partitions, it will limit me to only a couple gigs per OS, so i wanted to dedicate all of the memory to ubuntu.

Using a LiveCD allows you to determine how compatible your system is with Ubuntu.
From what I've seen, computers that aren't brand-new and meet the minimum system requirements fair well with Ubuntu. 
I mention brand-new computers because these are typically the onces with advanced graphics cards that haven't had GOOD linux drivers written yet.
It is also useful to search extensively for your particular computer model (e.g. "Thinkpad x200s ubuntu", or "Latitude E6400 linux") these usually turn up helpful websites like ThinkWiki.org and NotebookReview.com that let you know if there is something on your system that no one in the world can figure out or if your computer will run Lucid with flying colors.

(Added after edit) If you would like better feedback, please provide more information about your particular computer. This allows users of the same model to advise.
