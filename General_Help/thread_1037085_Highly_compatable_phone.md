### Post by anthony_barker on 2009-01-12
_modem _
- most phones should work via bluetooth 2.0 as a router
Nokia - yes fine
Blackberry - requires fiddling, 
iphone - only if jailbroken I believe
Treo - yes should work (palm not windows)

_sync _
check out opensync.org and KitchenSync (KDE app)
nokia - yes 
[http://ubuntuforums.org/showthread.php?t=260676](http://ubuntuforums.org/showthread.php?t=260676)
blackberry - yes with fiddling (Barry software)
[http://www.netdirect.ca/software/packages/barry/](http://www.netdirect.ca/software/packages/barry/)
iphone - yes via yahoo website to evolution
Treo - yes should work 
Motorola - L7 SLVR - yes - KitchenSync


_storage_
nokia - yes - via minisd/opensync
blackberry - yes- via minisd/opensync
iphone no - only if jailbroken/opensync
Treo - yes

I have a blackberry - and it works ok - still a bit of a hack, I don't use much inter connectivity except sync. 
- Google Android should work - but I don't know enough about it yet. 
- Palm pre looks promising as well.

If I were in the US I would probably pick Android or Palm pre. If I lived in Europe I might go nokia. Symbian OS is also open source. I don't like the windows interface so I can't recommend windows mobile 5/6 (but kitchensync supports it).
