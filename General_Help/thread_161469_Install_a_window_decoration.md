### Post by mariux on 2006-04-17
**Download it (from kde-look.org) for example**
if its .tar.bz2, extract using tar xvjf filename.tar.bz2
if its .tar.gz, extract using tar zxvf filename.tar.gz

**Go into the folder**
cd foldername

**Compile and install**
./configure && make && sudo make install
