### Post by tredegar on 2011-04-07
I believe thaat thing works with the **ath9k_htc** module, which is in kernels >= 2.6.35

So you need to be running ubuntu 10.10, and it should work "out of the box". 10.04LTS is still on kernel 2.6.32
