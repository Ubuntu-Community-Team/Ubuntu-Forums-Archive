### Post by Cheesehead on 2008-08-25
Why would you want to?
Seems like a great way to bork your system, and the whole point of a package manager is system stability.

If you need *that much* system customization, you may as well install from source. Then you can put things wherever you want them and damn the consequences.

To answer your question:
Never seen one.
No way to change the directory in the config files.
No way to change among the flags.
