### Post by Taxman415a on 2009-01-10
It can show it's code to you so you can either study it or change it. That may not be your cup of tea, but it's a big advantage.

As far as user visable features there aren't so many that you couldn't eventually get on Windows, but many are just easier on Linux. Customizing your desktop environment is a big one. If you don't like Gnome, replace it with something lighter or different. If you like it, but just want it different, investigate it's various settings and themes. Right click on the top panel and look and the different widgets you can add to it. There's weather and all sorts of other fun stuff. Try desktop effects if you like that type of thing. 

Other than that it really depends on what you want to do with your computer. If there isn't anything you want to do and can't, just enjoy that and the fact that your open source operating system isn't spying on you and is just out of the way. If you want to do more, decide what.

### Post by KeyserSoze93 on 2009-01-10
SSH (Secure Shell Host) - that's the first thing I started using in ubuntu that Windows didn't have.

It basically allows you to use your system remotely (both command line and GUI), and transfer files to and from it over an encrypted connection.

To try in on a local LAN, just run:
sudo apt-get install openssh-server

If you want to connect from a Linux box, just run "ssh you@your-machine"
And if you want to connect from a Windows machine, you can use a program called PuTTY.

I set up my router so I can connect to my home system from anywhere (anywhere I allow in my firewall, that is), even my mobile phone (with the aid of a Java app called MidpSSH).
