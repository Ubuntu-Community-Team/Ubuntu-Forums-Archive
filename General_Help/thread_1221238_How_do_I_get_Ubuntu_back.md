### Post by crispeto on 2009-07-23
Here's the deal. I have an Asus eee 1000he. It came with two partitions. The first one with Win Xp and the second blank so I installed Ubuntu Jaunty on it. My default start up drive was the Ubuntu drive. I got the bright idea to install Win 7 on the XP partition and everything worked great except now I have no way of getting back to my Ubuntu partition. It automatically boot to Win 7. I don't get the Asus start up screen any more and I can't seem to figure out how to get into the bios from a fresh boot. I've held down the esc, F1, F2, F6, F8, F10, and the del key and nothing will take me to the bios so I can select which drive I want to boot from. Any advice? Thanks.

### Post by Ratscallion on 2009-07-23
Try going into Start Menu -> Search for System and click system. Then go to Advanced Desktp Settings... There should be a drop down and in there, if they still exist, Windows 7 AND Ubuntu. Click whichever you want to be the default selected one. If Ubuntu IS deleted, then there won't be a partition in Computer. However, if it is, there should be a drive in computer. Windows 7 usually tells you to format your harddrive (or does it automatically) on installation... So that might have happened.

### Post by crispeto on 2009-07-23
well, I ended up taking out the battery and unplugging the power supply and sure enough the asus splash screen came on and I was able to boot to another drive. Everything's working again.
