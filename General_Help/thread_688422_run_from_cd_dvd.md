### Post by mikewhatever on 2008-02-05
Yes, but making a bootable cd is a common mistake. All you have to do is burn the iso file to CD/DVD as image. [https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28iso%29](https://help.ubuntu.com/community/BurningIsoHowto?highlight=%28iso%29)
[http://en.wikipedia.org/wiki/LiveDistro](http://en.wikipedia.org/wiki/LiveDistro)

why is a mistake?

### Post by mikewhatever on 2008-02-05
> **devilonline said:**
> why is a mistake?

Well, it does not work afterwards. Some users are accustomed to make bootable cds, while others unpack their isos and burn the content as data. Both ways don't work, so it's important to follow the how-to. Once burnt, the CD will be bootable, no worries.
