### Post by danwood76 on 2007-12-21
Qdvdauthor is one of the best packages for this.
You need to encode your videos to mpeg2 first, which is easily done using avidemux2.

The only thing I will have to say about it is that the version of qdvdauthor in the repositories is a little outdated and the author of the software is just about to release the first v1.0 of the software.
Im using RC3 at the moment which is very complete, using this software gave better results than using the windows alternative I used to use, which also cost me £60 so Im much happier now with this better free alternative :)
Avidemux is especially good and powerful program once you get to know it.

You best bet is to search google for a guide to qdvdauthor, there are quite a few good ones around, you will also want to llok at a guide for encoding with avidemux2 aswell, although it has some very easy buttons which automatically setup for encoding to DVD spec mpeg2 format.

I reccomend compiling the latest source from the qdvdauthor site aswell, but looking at the other thread you may be better off just using the one from the repositories.
