### Post by cdbitesky on 2009-10-07
Sure. In synaptic you can install KDE which will install all the standard modules accosiated with KDE without the login manager. You can install KDE-core if you just want the GUI without any new applications. If you want the display manager that is integrated with KDE just install KDM, although I personally prefer the gnome one. If you do it this way you just change which GUI you want to use when logging in under the session option.
