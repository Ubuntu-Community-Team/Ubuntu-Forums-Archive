### Post by AlanRogers on 2008-11-28
You shouldn't have to install anything for any broadband provider. Buy a router and let that manage the connection. Then connect the netbook using either wireless or ethernet, and you're all done.
 
If you're trying to install anything from a CD provided by BT, it'll almost certainly be Windows only software and heavily reliant on Internet Explorer. Ergo, you won't *just* be able to install it.
 
That's not to say that you won't get it to work but the effort involved will be fair more onerous and less rewarding than the advice I gave at the start of the post!

### Post by Kevbert on 2008-11-28
It depends what BT Voyager router you have. Some people have had problems. Here's an [COLOR="Red"][example post]("http://onlyubuntu.blogspot.com/2008/08/how-to-setup-ndiswrapper-with-bt.html")[/COLOR] for Voyager 1055 which is a little involved.  You will probably need to install the firmware driver with ndiswrapper. Once this is installed you only need to set the username and password. Make sure the router password encryption is set to at least WPA (or WPA2 is even better if supported). Try not to use WEP as this is very insecure, but better than nothing.
