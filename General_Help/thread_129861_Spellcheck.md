### Post by mjh007 on 2006-05-23
I also had the same problem and I had the dictionaries installed etc.

I've just found out from [http://www.oooforum.org/forum/viewtopic.phtml?p=142385]("http://www.oooforum.org/forum/viewtopic.phtml?p=142385")   
that the language has to be set for the text itself.

To do this:
1. Select all
2. Format>Character
3. Set language on the Font tab to the desired language that has a dictonary installed.

Worked for me, so I hope it helps you
