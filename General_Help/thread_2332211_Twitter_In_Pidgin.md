### Post by Frogs Hair on 2016-07-29
There is a Twitter plugin and many others for pidgin though I've not tried them. 

```
sudo apt-get install pidgin-twitter
```

Description:> A Pidgin plugin for a microblogging services such as Twitter.
It allows you to post and read micro blogs on pidgin.
It provides many useful features.
