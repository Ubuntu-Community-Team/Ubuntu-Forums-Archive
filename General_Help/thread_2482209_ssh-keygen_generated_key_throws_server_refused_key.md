### Post by philhughes on 2022-12-24
PuTTY does not accept openssh format keys, you have to convert them first with puttygen:

[https://the.earth.li/~sgtatham/putty/0.78/htmldoc/Chapter8.html#puttygen-conversions](https://the.earth.li/~sgtatham/putty/0.78/htmldoc/Chapter8.html#puttygen-conversions)

### Post by simtcr on 2022-12-29
I resolved the issue by adding below in /etc/ssh/sshd_config

PubkeyAcceptedAlgorithms=+ssh-rsa
