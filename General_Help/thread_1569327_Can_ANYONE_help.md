### Post by quadproc on 2010-09-06
We will need some more information to be able to help.  Things such as what kind of computer, what kind of external enclosure contains the disk, what kind of disk, etc.

I have been running an external WD1001FALS drive through eSATA for a long time on Ubuntu 9.04 and 9.10 on a desktop system with no problems.  But before I got it going, I tried a Buffalo external disk drive and I found that the Buffalo drive did not work on its eSATA port, although it did work on a FireWire port.  So there are some products available that just don't work.  You might want to check the Hardware Compatibility list, and also the Hardware Incompatibility list, to see if your combinations are mentioned.
