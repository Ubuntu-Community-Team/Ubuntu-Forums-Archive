### Post by larienna on 2009-12-23
It's is not really a help request but it is general talk.
 
I am an ex computer technician and a library technician. I have installed Ubuntu lately after a long time without touching linux and I was surprised by the progress linux has made.
 
Anyways, what I want to talk about is about the system that allows you to install any open source software at distance. This system is great except for the fact that there is so much software and the classification is so bad that browsing for software to install is horrible.
 
I don't know all application that exists, so I had to pass through each application in the list, read the name, read the description, check if this could be useful and then install. It was a painful operation that took hours.
 
Of course, there is an hypertext search box, but this is good only if you know what you are looking for. But in my case, how can I search for something if I don't even know it exists.
 
With a classification scheme, it makes it easier to browse what exists and it makes it easier to sort out what you do not need. For example, If I don't chat and there is a sub-category for chatting software, I just don't look at this category. If I am searching for a bit torrent software, I can find them all at the same place.
 
So I was wondering if there was any project to make a better classification scheme and if people think that it could be useful to have a real classification scheme?
