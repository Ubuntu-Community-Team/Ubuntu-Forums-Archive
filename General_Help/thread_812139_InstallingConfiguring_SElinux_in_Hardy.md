### Post by buntunub on 2008-05-30
Your asking a question about a function which is very new to Ubuntu, so not many people are going to have the know how to be able to help you. SELinux has been used on Red Hat/Fedora for a long time now though so there is probably a treasure trove of information available on their websites. Setups should pretty much be the same for application specific stuff. Just be aware that there are underlying differences between a Fedora/Red Hat and Ubuntu/Debian system. Things like sudo/su and locations of files will be different. Easy way to find needed config files and such just use $sudo find / - name file, or slocate commands.
