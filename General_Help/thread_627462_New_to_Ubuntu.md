### Post by chewearn on 2007-11-30
Unfortunately, I'm not familiar with the applications you listed (some other forum members have to chip in here).

However, I can offer some advice.  Generally, Windows program cannot be installed in ubuntu.  You have some recourse:

1. find alternative programs that are available in ubuntu.  There are literary thousands of free applications you can find; they might not work the same as what you are used to, and they might have less (or more) features.  Be prepared for a steep initial learning curve when switching OS.

2. try run the Windows application in Ubuntu using Wine.  When I first switched from Windows to Linux, I tried this for a couple programs, but eventually find another way.  AFAIK, Wine is hit and miss; sometimes it works, sometimes it doesn't.

3. Installed Windows in a virtual machine under Ubuntu.  I recently have some success with using Kernel Virtual Machine (KVM), though this route is probably not the easiest at the moment.

Here is an excellent site to get you going:
[http://www.psychocats.net/ubuntu/](http://www.psychocats.net/ubuntu/)
