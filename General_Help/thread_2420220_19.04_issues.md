### Post by Impavidus on 2019-06-01
That screen you see is the grub menu. You can edit the settings in /etc/default/grub and run **sudo update-grub** to apply the changes. You can set GRUB_TIMEOUT to 0, so that grub loads the default entry at once without showing the menu. However, if your system is broken it can be very useful to have easy access to the grub menu. You can also hide the menu, so that it only shows if you hit a key, but gives you a few seconds to hit that key. You can set GRUB_TIMEOUT_STYLE to countdown or hidden for that effect. See```
info -f grub -n 'Simple configuration'
```for details.
