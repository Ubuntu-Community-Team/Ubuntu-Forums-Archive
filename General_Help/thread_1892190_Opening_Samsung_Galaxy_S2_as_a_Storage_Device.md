### Post by Stovey on 2011-12-07
Hi Maxim:

I have a Galaxy S so I'm shooting in the dark here, but here goes:

1.  Return "home" / press "home" button (bottom middle)
2.  Press "menu" (bottom left) and select settings.
3.  Select "Wireless and network"
4.  Select "USB settings"
5.  Select "Mass storage"
