### Post by hopugop on 2010-05-30
One great tool for remote-control is TeamViewer. It works cross-platform (Windows to Linux, Linux to Mac, etcetera) and it is very easy to use. **The software is free for non-comercial use but it is not open-source.**

Once running, the software gives the user an code and password so you can enter those on your side and connect to her computer.

You can even install a "server only" software on her side to make it simple, just run it and ask her the codes!

Google up TEAMVIEWER and check that out.
