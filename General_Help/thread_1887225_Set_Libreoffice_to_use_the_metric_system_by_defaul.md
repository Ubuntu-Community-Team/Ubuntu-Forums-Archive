### Post by grahammechanical on 2011-11-26
Open the Libroffice Help document and search for Selecting. The second line under the Selecting heading is Measurement Units. click on it and it tells you how to do it.

You open a writer document and go to Tools>Options>Libroffice Writer>General and set the measurement unit from the drop down menu called Measusrement Unit.

You do the same for each Libroffice program. Open a Calc document and so on.

Regards.
