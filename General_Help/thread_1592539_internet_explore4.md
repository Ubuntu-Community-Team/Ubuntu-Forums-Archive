### Post by modafokaxx on 2010-10-10
First, I have to ask: why on Earth would you want to do that?
Out of habit or for a real reason?

'Cause Firefox or Opera or Chrome are all far better browsers than IE.
You can even try Epiphany or Midori if you want something that feels lighter than the default Firefox.

Anyway, to answer your question, official answer is "Microsoft does not support any of its software on any other platforms than Windows".

That being that, using the WINE compatibility layer it is possible, albeit cumbersome, to run some versions of IE on Linux, including Ubuntu.
But that is usually only done by web designers who absolutely need to test their work in IE.

Another option I believe in that case is to use the IE Tab extension for Firefox.
