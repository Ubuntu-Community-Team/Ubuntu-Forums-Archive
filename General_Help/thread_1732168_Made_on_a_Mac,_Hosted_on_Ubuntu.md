I really don't have a problem, but I did want to post this. I made our own website for a little café my wife and I have on my Mac. I have no knowledge of web design nor html coding so using iWeb was the simplest way get the page up and going. Right away when you open the application up there is an image that says "Made on a Mac" at the bottom of the page which links to Apple's website. Since I host the webpage on one of my Ubuntu boxes I thought I'd make another image that says "Hosted on Ubuntu" and put it on my site and link it to Ubuntu's site. You can see how it looks here:

[www.landscafect.com](www.landscafect.com)

It's all the way at the bottom. If you want to use the image also you can save it directly from the website or download it directly from this link:

[http://ubuntuone.com/p/nHC/](http://ubuntuone.com/p/nHC/)
