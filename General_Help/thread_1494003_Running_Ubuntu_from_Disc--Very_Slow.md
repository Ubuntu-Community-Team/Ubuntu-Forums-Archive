### Post by jerome1232 on 2010-05-26
Yeah it's running slow because your running it off a dvd, dvd read access times are much, much, much, much slower than hard disk drive access times.

You can install Ubuntu alongside Windows creating a Dual Boot system, meaning your are given the option of booting into Windows or Ubuntu at boot time.

If you want to try it you can choose to install, just make sure you choose the option that mentions shrinking Windows and installing Ubuntu. You shouldn't lose any data but there is always that chance that something will go wrong, so keep backups just in case.

Here's a little video I found that showcases the installer [http://linuxfilesystem.com/uncategorized/install-dual-boot-windows-7-ubuntu-linux-1004-lucid-lynx-64bit-rc-via-usb-tutorial](http://linuxfilesystem.com/uncategorized/install-dual-boot-windows-7-ubuntu-linux-1004-lucid-lynx-64bit-rc-via-usb-tutorial)

### Post by exodus_ on 2010-05-26
Hello and welcome to ubuntu!

If you are very careful in installing, you can basically keep all of your documents and whatnot.  You can even keep windows around and dual boot while you get used to Ubuntu!

If you are using the live CD the installer is very simple, and will help you partition your drive(s) And it should even detect windows and give you an option to "install linux beside windows" so to say.

It's also normal for it to run quite slowly from the Disc.  If you install it to hard drive, it will run a lot better for you, not to mention that you will be able to keep changes across reboots, unlike the CD/DVD ;)
