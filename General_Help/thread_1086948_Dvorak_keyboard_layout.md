### Post by ASchweitzer on 2009-03-04
Well, on install you can choose a keyboard layout. After install you can change the layout for the *X server only* with the GUI under:
System | Preferences | Keyboard (Layouts tab)

What I mean by *X server only* is that a layout that is different from the installation one will only load for a specific user, AFTER the login. So you'll still have to log in with the keyboard layout you chose at install.
