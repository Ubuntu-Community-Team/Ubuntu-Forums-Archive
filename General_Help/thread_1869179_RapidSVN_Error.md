### Post by makitso on 2011-10-25
I have RapidSVN installed on several 11.10 systems, VB Ubuntu, VB Lubuntu, VB Xubuntu.  All installations work correctly.  However, I have 11.10 Ubuntu installed on my Thinkpad t60 running unity and G3 shells.  If I try to create a bookmark to an external repository I get the error "Error while refreshing filelist (generic failure).  

I have tried removing and reinstalling but no difference.  I have tried using the LAN connection instead of wireless and no change.  Other than that the 11.10 version runs very well.

Update:  Problem solved, there was a mismatch between the desktop name and the name in the /etc/hosts file.  Once corrected the RapidSVN and SVN worked.
