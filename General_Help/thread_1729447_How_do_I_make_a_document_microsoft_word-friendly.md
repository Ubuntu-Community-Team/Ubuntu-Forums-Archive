### Post by ryanguy101 on 2011-04-14

### Post by lykwydchykyn on 2011-04-14
When you save your file, change the filetype to ".doc".  The default is .odt, which only the very newest (and patched) versions of Word can read without an add-on.

---

### Post by Don Barzini on 2011-04-14
Another option other than .doc would be to save the document as an .rtf file - Rich Text Format.

### Post by earthpigg on 2011-04-14
saving it as a .docx or .doc, as suggested above me, will generally work if there is no complex formatting.

if there is complex formatting, export it as pdf for 100% perfect compatibility. 

or, do your work in google docs and make it accessible via a web address.
