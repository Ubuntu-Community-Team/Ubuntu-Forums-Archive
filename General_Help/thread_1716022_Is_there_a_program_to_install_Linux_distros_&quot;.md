### Post by rgb1701 on 2011-03-27
I think the best, most common solution to the try-before-you-install issue for distros is to just install VirtualBox for Windows and run the distro .iso within it.

[http://www.virtualbox.org/wiki/Downloads](http://www.virtualbox.org/wiki/Downloads)
