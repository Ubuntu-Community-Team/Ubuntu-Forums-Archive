### Post by coffeecat on 2009-04-29
You probably also need Compiz Fusion Icon installed as well. Go to Synaptic and install fusion-icon. To start it, go to Applications > System Tools and a blue icon will appear in your upper panel. Right-click on that and under 'Select Window Decorator' you can choose between Gtk and Emerald. Once you've ticked Emerald, the Emerald Theme Manager should work for you. You can also add Fusion Icon to System > Preferences Startup Applications if you want. You can find the startup code you need in Edit Menus. 

Here's a useful link:

[http://ubuntuforums.org/showthread.php?t=809695](http://ubuntuforums.org/showthread.php?t=809695)
