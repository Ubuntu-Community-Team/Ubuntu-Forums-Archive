### Post by starcraft.man on 2007-05-11
[Here]("https://help.ubuntu.com/community/BurningIsoHowto") you go.

Basically you have to burn it to a CD, .ISO is not an installer format like exe. Once you burn it to a CD, you have to reboot your computer with the CD in your drive, and make sure you boot to it. Thats either accomplished by setting the bios to boot from CD (usually by default) or selecting to boot to it from a menu (my old bios is like this).

Read [this]("http://www.psychocats.net/ubuntu/installing") section on how to install Ubuntu and the different steps.
