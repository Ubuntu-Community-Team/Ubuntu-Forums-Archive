This may be an odd question, but I figured I would ask anyways. I'm using Feisty Fawn, and when X loads, I have Azureus load. Now, is there any way to make Azureus continue running, even if I Ctrl-Alt-Backspace out of X, and have the now detached session be reloaded into the new session of X?

---

### Post by simonn on 2007-04-23
No, this will kill Azureus... me hearty.

However, executing gdmflexiserver will allow you to log in as a different user, so you can run Azureus in it's own session.
