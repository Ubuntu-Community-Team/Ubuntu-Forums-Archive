### Post by vincentvee on 2008-06-27
> **steve984 said:**
> I dowmloaded some programs thru synamtic and the programs arent on my menus.  where are the program files in this file system thanks
when you download something through synaptic, the files go to [COLOR=Navy]/var/cache/apt/archives[/COLOR] but they should be installed unless you selected download package files only, even if they are not on the menu, have you tried running them by pressing alt-f2 then typing the name of the program?
even if you have to re-install them, you can double click on the package files in /var/cache/apt/archives and it will install again.
