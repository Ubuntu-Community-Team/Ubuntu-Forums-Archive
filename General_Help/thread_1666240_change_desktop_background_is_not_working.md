### Post by 3Miro on 2011-01-13
If you are using the default settings, you can change the desktop background from System -> Prefs -> Appearance.

If you have disabled the default setting to setup separate wallpapers per workspace, you have to go to CCSM (compiz-config-settings manager, install it from the Software Center it is not there) and select the wallpaper plugin.
