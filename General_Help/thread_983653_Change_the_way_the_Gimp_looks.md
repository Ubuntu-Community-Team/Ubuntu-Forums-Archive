### Post by ad_267 on 2008-11-16
If you're using Gimp 2.6 (Default in intrepid), go to Edit - Preferences - Window Management. Keep above makes them stay above. Utility window makes them minimize with the rest of the application, stay above and stay out of the taskbar, but I think it only works with metacity, not compiz.
