### Post by mcduck on 2007-04-10
To choose which program to use by default to open some file just right-click one file of that type, select 'Properties' and then select the program you want to use i the 'Open With'-tab.

To get rid of the error messages install Totem again. Don't use it if you don't want, but it's used to generate thumbnails of video files and other video-related stuff so t might be a good idea having it installed at least.
