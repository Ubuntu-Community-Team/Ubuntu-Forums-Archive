### Post by timcredible on 2008-08-19
i've tried awn, cairo-dock and kiba-dock.  you need to run compiz for them to run properly (transparency and such).  imo, none of them are 'stable', but they're not bad, and they're much better than they used to be.  awn and cairo-dock are very similar, and awn even uses a couple cairo applets.  for something like mac-os/x, either will work (i have awn right now with the transparency, reflection, and curved menus), but cairo-dock has mac-os/x icons, and does the curved menus also, if you want to go that far.  i would suggest trying all 3, see which one you like.  make sure to not install awn from the ubuntu repos, they are missing the applets, have to install from ppa instead.

### Post by SZ07 on 2008-08-19
About needing compiz, that is not necessarily true. You can turn on the compositing feature in Metacity (via Configuration Editor), and the docks will work without Compiz. This is especially useful when Compiz crashes, because you are still able to use the dock.
