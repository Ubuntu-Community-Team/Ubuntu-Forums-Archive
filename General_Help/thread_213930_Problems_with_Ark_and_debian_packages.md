### Post by Jucato on 2006-07-12
You don't use Ark to install .deb files. You can do it from the command line or you can do it from Konqueror (KDE's file manager). Right-click on the .deb file and you will see an option for Kubuntu Package, which has options to install or remove the .deb package.

Hope that helps a bit.
