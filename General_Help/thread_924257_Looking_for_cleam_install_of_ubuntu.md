### Post by nvance on 2008-09-19
The install would be the same if you were doing a dual boot or not.  Reboot your machine (make sure that the BIOS is set to boot from CD) with the cd/dvd in the drive and when prompted choose to install Ubuntu from disk.  It will format your HD and then install.
