### Post by WorMzy on 2010-10-08
Which non-kernel updates have needed you to restart?

Just to make it clear: You don't /need/ to restart when prompted, it's not like Windows where it "needs" to restart to update system files. If you don't want to restart, then don't. The update has already been run, and your system will continue to run as normal. You shouldn't be prompted to restart every 20 seconds like when using Windows, so just restart when it's convenient for you.
