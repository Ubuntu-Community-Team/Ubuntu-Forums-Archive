### Post by mje1708 on 2010-04-06
I used XP on all of my systems until a month or so ago. It started with my netbook which had always been disappointing. I installed the Ubuntu Netbook remix and suddenly it was like having a new machine. Everything worked smoother, faster, better. Multi-tasking in xp was really out of the question - everything slowed down. With ubuntu it is just not an issue. I have now installed Ubuntu on my desktop and am very pleased with the result. But to answer some of your questions -

1. Yes - you can dual boot for as long as you want with no problem.
2. You will find that there are alternatives for the programs you mention - some of the data files will likely be transferable - others possibly not. 
3. The alternatives may or may not be as good/ better/ worse than the programs you are used to - some of this will be subjective and some will depend on which parts of the program you use. Open Office for instance is a great alternative to Microsoft Office. I use both - however, if pushed, I would say MS Office has the edge slightly. But, to be honest, I would wager that Open Office would be fine for 95% of users.

As far as advantages go - you will find many postings saying that Linux is "better" than Windows and also vice versa. There are clearly advantages and disadvantages to both systems. I can only give my personal views which are as follows -

1. Linux runs my (admittedly old) hardware far better than Windows - faster, smoother, more reliably and, above all it is more stable - no more blue screen of death, freezing etc
2. I like the idea of not paying to upgrade to the latest version of MS Win every few years - I don't like the way MS go about their business - I think competition is good.
3. I like the fact that there is so much help - and willingness to help - in the Linux community - something you would have to pay for in Windows - if it were even available
4. In my limited view, Ubuntu 9,10 is certainly a match for Windows XP and Windows 7. It is different and takes some getting used to but certainly comparable. Whether you agree or not depends on you - I am certainly not one of those mad people you come across trying to convert the whole world to Linux - I think choice is good. I wish there were more alternatives to Windows - I think Windows would be a better product if it did have more competition.

Anyway my advice is install Ubuntu on your machine - dual boot. But give it time - if you are an experienced windows user it will take time to get used to - things are in different places. But I haven't found yet anything Windows will do that Ubuntu can't. I have found lots of things Ubuntu can do which windows can't.

Best of luck.

Mike

I can answer your query about Office and Outlook.

Open Office is probably the most likely alternative. I'll be honest with you, it looks like you've locked yourself into a lot of Windows-only programs. Word and Excel documents can be opened in Openoffice.org, however, so you haven't lost them. You can't expect Microsoft software to work on anything except Microsoft.

> 5. Then I wonder If I can dual boot until I become familiar with it?

Oh yes, you can dual-boot. You can dual-boot until the cows come home. That's what I'd recommend you do; dual-boot until you've broken free of the proprietary software and gotten your data out of the proprietary formats into some free formats.

> 6. What about the NTFS drives?

Not a problem, we read and write those.
 
> 7. What about Internet security and don't say they are not going to try?

Firefox has anti-phishing built-in. Linux doesn't run Windows viruses. Ubuntu doesn't listen to any incoming ports by default, but if you really want to you can configure your firewall. You don't need any extra programs for "internet security".

> 8. Then there is WMP and all the music I have plus the Licenses. Is drm in your system?

No there most certainly is not; once again you've gotten yourself locked into proprietary software AND DRM, but it may be possible to break the DRM on those music files and get them into some interoperable formats. Even just MP3 would be better. Alternatively, the music store you used will probably offer those same songs in non-DRM'ed format these days, so see if they can provide you with plain MP3s.

> 10. One very Important thing is the outlook data file.  I wish it was just a regular file structure but it is a DB type. Will I be able to transfer that data file over to your e-mail and I don't want no Internet e-mail. No adds flashing around and bothering me.  I just want to communicate messages and files. 

You may be able to do something with it, but it could require some Windows shareware to get it into a format that Thunderbird and Evolution can understand. Oh; on Linux, shareware is almost impossible to find. You'll appreciate the lack of nag notices on Linux.

> 13. Zip files

Not a problem.

> 14 Seagate Clone software for backup.

Not as far as I know, there are roughly 50 billion backup programs for Linux anyhow. You wouldn't use a Windows program to back up your Linux data; it wouldn't understand anything.

> 16. one more thing.  Office 2003 and 2007 comparable because I have both file types
 
Well If we get past all that I would be interested.

You've got real problems there. All your data and even your music is locked up in proprietary formats; for instance, Turbo Tax is designed to stop you from being able to buy a competitor's product, because the competitor's product will not read Turbo Tax files (the format is undocumented). I'd still encourage you to dual-boot or use Wubi if you want to see what Ubuntu is like; but unless you can export or migrate all your data to Free or Linux-program-compatible formats you'll be stuck with Windows forever.

The difference with Linux is that, once you've got your data in a format that standard Linux programs can understand, then you've "freed" your data, and you'll never have migration problems again.

Oh, and to clarify: Only a small proportion of Windows programs will work on Linux, and even those will use some software called Wine. It is MUCH better to use equivalent Linux programs anyway - they are faster, integrate into your desktop better, and give you the genuine Linux experience. They are also more securely designed, and mostly free of cost and open-source.
