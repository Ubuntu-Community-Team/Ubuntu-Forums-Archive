### Post by grahammechanical on 2016-05-09
Your link is not loading. What version of Ubuntu are you using?

One reason for  updates not installing is the fact that the version being used has reached end of life. When a version of Ubuntu reaches end of life there are no more updates and the software repositories are closed and archived on other servers. So, the URLs in the sources list become inaccurate and error messages are given when trying to update.

The only versions of Ubuntu still receiving support are 12.04, 14.04, 15.10 & 16.04. Are you using one of those?
