### Post by ThrobbingBrain66 on 2007-11-04
To install kismet, simply issue the following command in the terminal:
```
sudo apt-get install kismet
```

The vast majority of packages you'll need are in the repos so make sure to always check Synaptic first.  If you don't find it there, then you may have to download the source code compile it yourself.  In that case, use the following guide: [http://monkeyblog.org/ubuntu/installing/](http://monkeyblog.org/ubuntu/installing/)
