### Post by mb_webguy on 2009-05-06
With only 160MB of ram, not even Xubuntu will run well.  The minimum required for Xubuntu is 192MB.

What I'd advise is using the minimal install CD and installing a lightweight window manager like Openbox.

Alternatively, you could look at some other small distributions like Puppy Linux or DSL.

### Post by -kg- on 2009-05-06

Other than that, you definitely would be able to run it for browsing and light email, and it will certainly run in 160 MB of RAM.
