### Post by mb_webguy on 2009-01-28
In my experience, Ubuntu runs just fine on Dell laptops -- I've had two myself.  The only thing about that particular computer that may pose a problem is the wireless card.  The Dell-branded wireless cards tend to use a Broadcom chipset, and Broadcom is not very Linux-friendly.  But Linux can use the Windows driver for an incompatible wireless card using a program called ndiswrapper, and the Windows drivers are readily accessible from the Dell website.

If in doubt, stick in an Ubuntu LiveCD and see how it goes.
