### Post by 2F4U on 2011-07-20
If you delete something from your system that does not affect the backups, except if you also delete the backups. Thats the reason for making backups and storing them in a save place. If something goes wrong you are able to recover from the backup. To be able to do this, it is necessary to do regular backups, so that the backup always contains the most recent changes. It is also wise to retain several versions of past backups.
