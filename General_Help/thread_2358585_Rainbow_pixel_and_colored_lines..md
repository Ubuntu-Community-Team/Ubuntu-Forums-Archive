### Post by yetimon_64 on 2017-04-15
> **herbertglen5 said:**
> New driver for the vid card?????

I have noted a similar problem in the past with an Nvidia GTX850 video card and the nouveau drivers (as are default on a new install) on ubuntu.

You still haven't mentioned what release/version/flavour of ubuntu you run. But if you are on a unity install try opening dash and search for "Additional Drivers", you can use it to check if you have proprietary drivers available; one should be marked as "recommended" and I'd suggest you give it a try if you haven't already done so. 
Even with proprietary drivers I still get some distortion on startup, but it clears as the desktop becomes fully loaded up. I no longer get borders around windows and such with the proprietary drivers in general use. Good luck.
