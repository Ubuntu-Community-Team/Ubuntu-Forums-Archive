### Post by bitsnpcs on 2019-02-24
Hello raleigh3,

using **Gnome Tweaks **available in the Ubuntu Software go to -
Top Bar>Clock>Date   
Slide it to On, this will add the month to the date.

In Ubuntu Settings go to -
 Details>Date&Time>Time Format   
Use the drop down menu to select AM/PM to display this after the time.

I am unsure how to display the year, I will have a play around and see if I can find it.

Edit -
there is this tutorial for adding the year, it will do the date too, [http://www.ubuntubuzz.com/2018/03/show-full-clock-on-ubuntu-1710-and-1804.html](http://www.ubuntubuzz.com/2018/03/show-full-clock-on-ubuntu-1710-and-1804.html)
