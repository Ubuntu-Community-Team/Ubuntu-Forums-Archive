### Post by skymt on 2006-10-14
Yes, that's exactly how Ubuntu does things. It handles all updates itself, but the software is usually slightly out of date. This is because the Ubuntu development team has tested a certain version of each package, to make sure none of them conflict. If you install a newer version that hasn't been tested by the Ubuntu team, it may break something else, or just not work.

EDIT: Later this month, the new version of Ubuntu will be released, with the newest versions of everything. I'm beta testing it now, and it's very nice. It will, in turn, be superceded in around six months, at which point most of the software will be six months old. That's the price you pay for stability.

EDIT 2: Oh, I take it (from the last sentence) that you're already using Edgy. In that case, you'll probably get the new OpenOffice soon (unless it's too close to the release date).
