I hope I have put this into a group or section that can be read and responded
to. If this is in the wrong place, please move it where it needs to be.

I have been running Ubuntu 9.04 for about two months. I guess you call me a newcomer.
I have been an old MS user for the past 29 years, from DOS to XP.
A friend convinced me to get a couple of mobile racks and take my XP drive out and put Ubuntu 9.04 and play with it. Well it did and was hooked.
I just installed 9.10 Thursday, had it all tweaked up. It would run all the video's, music file that I could throw at it.
In the process of going to watch a video I downloaded, Ubuntu 9.10 said it needed some plug-in's for it. I told it go ahead and get them.
It found what I needed, downloaded and installed them. I then restarted the player and that is when the little round thing spinning just quit after about 20 seconds.
Yes it did, just quit so I tried clicking on the video again, it still did not play the video clip. I think the clip was an avi file as best to my knowledge.
I didn't know this was going to happen so I didn't take notes.
I tried another video clip, a .avi file and the same thing happened to me, the round thing showing that it was loading the video file came on the screen for about 20 seconds and then disappeared.
I rebooted the computer, but without any success.
The next thing I did was go to the first tab, the one with the programs listed there by group, I went all the way down to the find new programs/delte button and clicked on it.
All I saw was the little round spinning circle and 20 seconds latter it disappeared. I could not choose or search for any new files to download and install.

I did have my 'notify Ubuntu' check to let the powers to be know what happened to me.
Was it a bug? I don't know. 
I ended up re-installing Ubuntu 9.10, it took about 3 hours to get all the things tweeted like I wanted. So far it is running OK.

I would like to see two things installed on Ubuntu in the future. One is what the Windows people use called 'restore points'. I would like to 'go back' in the past and reload the settings I had earlier.

I also would like to have a program to 'clone' my Ubuntu to another hard drive. This way if something really went bad I could swap out the drives in my computer and put the one in that wasn't messed up.
I know two cloning programs for the XP people, one is Casper XP and the other is Acronis True Image. As this time they don't support Ubuntu.

Thanks for reading I hope this gets to the people that can figure out what happened.

### Post by Beggar Su on 2009-11-01
Support for common media formats and other useful support file types:

**sudo apt-get install ubuntu-restricted-extras**

also you can install vlc player if totem is giving you problems:

**sudo apt-get install vlc**

Clone your drive to another by using a free image tool (partimage) you can boot from a cd:

[http://www.partimage.org/Main_Page](http://www.partimage.org/Main_Page)
