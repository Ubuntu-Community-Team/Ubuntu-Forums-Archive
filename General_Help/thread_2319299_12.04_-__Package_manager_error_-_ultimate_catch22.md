### Post by ian-weisser on 2016-04-03
Your problem seems to be lack of storage.
You won't be able to do package actions, updates, removals, etc. until the problem is solved.
Your system is likely to remain unstable until the problem is solved.
So don't get distracted. Focus on the problem.

You wrote that you tried Disk Usage Analyzer. That's a good path forward. What is hogging all the space on your drive?

### Post by Impavidus on 2016-04-03
Upgrades don't make an unstable system stable. Let's focus on your first problem.

There are some graphical tools to analyse disk space, not all equally intuitive. Command line however is easier to share. Try these:```
# How much disk space do you use?
df -Th
# The inodes too.
df -i
# A list of the largest directories. This one may take quite a while to run.
sudo du -d 2 -h / 2>/dev/null | sort -h | tail -n 25
# Maybe you have a lot of old kernels. It should be possible to uninstall those.
dpkg --list linux-image-\* linux-headers-\*
# Which kernel are you running?
uname -a
```

### Post by ian-weisser on 2016-04-04
You do indeed have a lot of old kernels...and kernel headers.
Kernels use up some space. Headers consume a lot of inodes.

Use the package manger to remove those obsolete packages.
Do NOT remove the currently running kernel. That would be very very bad.

### Post by Impavidus on 2016-04-04
With the package manager confused by your partially installed packages and broken dependencies, you'll have to use some lower-level commands. I can't read the full names of the packages in your reply. You can get them by making the terminal window wider before running the **dpkg --list** command of my previous post. The following command has what I think the package names are.```
sudo dpkg --purge linux-image-3.5.0-23-generic
# Repeat for all other versions EXCEPT 3.13.0-79 and 3.13.0-83

sudo dpkg --purge linux-headers-3.5.0-44{,-generic}
# Repeat for all other versions EXCEPT 3.13.0-79
```
That should give you some free disk space. Next try fixing the package manager by running:```
sudo dpkg --configure -a
sudo apt-get install -f
```
