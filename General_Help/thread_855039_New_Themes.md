### Post by CheeseEatingBulldog on 2008-07-10
You can change everything in gnome (ubuntu standard). For some themes go to [http://gnome-look.org/]("http://gnome-look.org/") or [http://www.compiz-themes.org/]("http://www.compiz-themes.org/") for compiz themes.

To install a theme, simply go to System->Preferences->Appearence once you have downloaded a theme, and drag and drop it in the appearence box, it should install it for you. If you want to mix and match, download a few themes, install them and then click the 'customise' button on any theme, then you can change individual elements, such as the controls, icons and colours.

To change a bootsplash and alike, download startup-manager (```
sudo aptitude install startup-manager
```) where you can change bootsplashes and various other startup stuff.

If you want a different log in screen, grab a "GDM Theme" from gnome look and drag and drop in the "login screen" option via System->Administration.

Hope that helps.
