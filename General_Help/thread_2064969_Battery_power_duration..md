### Post by TenPlus1 on 2012-09-30
First make sure that you have your video drivers installed for your Ati or Nvidia card so it can properly manage power consumption for the graphics...

Secondly, Lubuntu uses xfce4-power-manager instead of Ubuntu's own gnome-power-manager so it's up to you to set these up for your machine...

Third, if you really think it still uses more power then you can install Jupiter applet which allows you to save even more power, link below:

[http://www.webupd8.org/2011/09/jupiter-applet-finally-available-for.html](http://www.webupd8.org/2011/09/jupiter-applet-finally-available-for.html)

I have been using Lubuntu on an ancient Compaq m2000 on which the battery is flakey and probably only gives 1hr max, but on Ubuntu itself I could never get past 40 minutes...  So it probably depends on supported hardware also...
