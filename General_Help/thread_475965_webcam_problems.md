### Post by kimusabi on 2007-06-16
Use aMSN.
```
sudo apt-get install amsn 
```
That should do it for you (don't forget to plug in the webcam and select the webcam plugin from the aMSN plugin menu), or try looking [here](http://ubuntuforums.org/showthread.php?t=332003).

Hope that helps.
