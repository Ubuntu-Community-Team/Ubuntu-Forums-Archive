### Post by catlett on 2006-09-25
Go to Places<Computer<Filesystem. Open up the var folder and then open the lib folder. Find your java folder. You may have a different version of java or something else. You can't just use my path because yours may be different. You have to look for your java folder in /usr/lib and then copy the path into opera or browse to the folder/file with opera,s window box 'browse'
