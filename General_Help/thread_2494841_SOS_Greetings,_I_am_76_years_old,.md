### Post by dragonfly41 on 2024-01-29
> I am 76 years old.
And I am an octogenarian.  Started with Windows some decades ago but in later years added Ubuntu.Let us take this step by step.
You will probably find more resources in the English speaking forums due to broader Ubuntu coverage. But if you have difficulty you can translate blocks of text using Deepl.com. Or look for a Spanish forum.
This forum will need to understand what computer and software assets you have to start with.
You write that you have a Windows computer.
> Used the system in 2008, but my children gave me a computer with Windows, but I want to go back to using Ubuntu 
What exactly are:
Make of PC
32/64 bits
Windows version
Internal Disk size
USB 2 and 3 ports available?  Or just USB 2.0 ports in 2008 computer?
RAM Memory used (can it be expanded later to say 32GB).

This information can be extracted through a Windows session.
    (a) On the Search  bar on the taskbar, enter system configuration.
    (b) Select System Configuration from the list of results.
The reason for enquiring about availability of USB 3.0 ports (rather than older USB 2.0 ports) is that if you have these we can advise on setting up your computer in dual boot mode with external Ubuntu. USB 3.0 offers faster data flow. But this will require purchasing of more assets such as SSD (e.g. crucial.com) and docking bay.

If you wish to use Ubuntu in your existing gifted computer then Windows and Ubuntu will have to share the same internal hard drive (as I started with). I later learned from Windows vagaries to distance Ubuntu from Windows owned internal drive and upgrades and instead isolate Ubuntu in external containers. This is personal choice.

Start by purchasing a San Disk 32GB flash drive.
It may be pointed out that a smaller San Disk flash drive (8GB) is adequate size to &#8220;Try Ubuntu&#8221; but in my experience it is cost effective to go a bit further to purchase a 32GB flash drive which can be configured to hold data between LiveUSB sessions. Indeed they are low cost and you might buy 8GB and later 32GB sizes to experiment.

[I]But not important data on these flash drives since these flash drives can have limited life.

[/I]You might start with a low cost 8GB San Disk drive. Follow tutorial to create a LiveUSB using Rufus or Etcher in Windows, then ensure that the LiveUSB can be seen in your BIOS boot options (probably hitting F2 or other key when booting up) and boot into SAN DISK LiveUSB.

[https://ubuntu.com/tutorials/create-a-usb-stick-on-windows#1-overview](https://ubuntu.com/tutorials/create-a-usb-stick-on-windows#1-overview)   [I]
My advice is to only use the LiveUSB mode "Try Ubuntu" to get a "feel" of Ubuntu. Do not hit the "Install" button. If you wish to retain files in "Try Ubuntu" mode you will need to setup LiveUSB to retain files.  More on that later.[/I]  Back to Windows, ensure that you post PC configuration to help in receiving further advice from forum members.  There are numerous posts here on multi-boot. It helps to understand what broad themes are of interest. Education, research, etc.

Are there any user accessibility issues to consider?

I am 75 years old. I started using Ubuntu in 2007. I could not afford and did not want Microsoft Windows. I have learnt a lot from this forum. Reading the posts of other uses can help with our own problem. Even so, we will try to help anyone even if they are asking a question that has been asked many times before. This is what I like about this forum.

Open System Settings.Select Sound. What is listed as the Output Device? Is your machine a laptop with internal speakers? Ubuntu is built on a core operating system called Linux and Linux comes with a very large number of hardware drivers. So, there usually not a problem getting sound working.

Playing back video and audio files is a different matter. We usually need to install some proprietary audio and video drivers. So, is your problem with getting sound working that of not getting sound when playing audio and video files?
