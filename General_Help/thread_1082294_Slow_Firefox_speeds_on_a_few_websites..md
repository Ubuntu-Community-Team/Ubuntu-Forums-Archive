### Post by fragos on 2009-02-28
The speed that website renders is very much a funtion of the site itself. How the web site deals with images can be a factor. Including video slowws things down. For images and video the speed is also impacted by the dots per inch. If you take an image strait from a video camera it can be rescaled dynamicaly to smaller sizes but big images load slowly. A good site with scale images and use less dense DPI on the desktop before being placed in a sight. Some sites like Gmail have a lot of data base activity necessary to rener the initial page view and this makes Gmail's initial display slower than other sites. There are diferences in the speed of browsers but Firefox is amongst the faster for rendering sites. Lastly adding lots of extensions to Firefox can also effect speed. Adblockers for example.
