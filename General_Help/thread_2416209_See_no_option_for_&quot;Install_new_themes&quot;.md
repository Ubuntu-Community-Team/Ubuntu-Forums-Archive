### Post by Dennis N on 2019-04-07
After installing from software center, you have to select it from the Appearance dialog:

**Menu > Preferences > LXQt Settings > Appearance**
To Select theme used by applications:
Select **Widget Style** in the side bar, then use the drop-down menus beside "GTK 2 Theme" and "GTK 3 Theme" to select the theme.

Also in the sidebar:
**LXQt theme** affects elements like the panel and menus;
**Icons**, **Font** and **Cursor** are self-explanatory.
