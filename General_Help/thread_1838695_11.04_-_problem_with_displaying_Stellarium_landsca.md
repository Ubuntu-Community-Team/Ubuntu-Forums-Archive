### Post by daggerstab on 2011-09-04
> **chris722 said:**
> Does anyone know how to fix that?

Yes, I do. :)

This is a Stellarium bug:
[https://bugs.launchpad.net/stellarium/+bug/657455](https://bugs.launchpad.net/stellarium/+bug/657455)

It was fixed in Stellarium 0.10.6 (the last version is 0.11.0). The version packaged for Ubuntu 11.04 is Stellarium 0.10.5, so you need either to use a work-around, or install 0.10.6 from a PPA.

The workaround:

Open a terminal and start Stellarium with the following command:
```
LC_NUMERIC=C stellarium
```You can use the menu editor ("System"->"Preferences"->"Main Menu") to edit Stellarium's launcher in the menu to use this command.

The PPA:

Add this Personal Package Archive to your software sources and install Stellarium 0.10.6:
[https://launchpad.net/~stellarium/+archive/stellarium-releases]("https://launchpad.net/%7Estellarium/+archive/stellarium-releases")
As the link explains, you can do it either with the GUI or the command line.

NOTE: Again, you need to do **only one** of these, not both.
