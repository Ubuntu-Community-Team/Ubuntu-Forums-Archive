### Post by SeijiSensei on 2017-01-05
Do you have lots of folders with lots of messages in each? Thunderbird will try syncing them all each time it starts up. If you're using IMAP consider unsubscribing from large folders you rarely view.

I just leave my machine on and running T-Bird all the time. Then it maintains sync pretty consistently.
