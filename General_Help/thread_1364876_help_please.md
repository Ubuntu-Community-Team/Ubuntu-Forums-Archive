### Post by howefield on 2009-12-27
Try closing all running applications including the Amarok running icon in the notification area, then go to System > Preferences > Startup Applications, and click on the Options tab, check "Automatically remember running applications when logging out".

Close and reboot.

Do it again, only uncheck the box this time.

Sometimes it needs a clean session for the settings to "stick".
