### Post by Bucky Ball on 2017-03-17
Do an update and then install ubuntu-restricted-extras. Reboot and try it now. 

You can install it using a package manager or in a terminal with:

```
sudo apt install ubuntu-restricted-extras
```

Always pays to [have a sniff around]("https://duckduckgo.com/?q=html5+ubuntu+firefox&t=hd&ia=web") before posting as a lot people have this issue and have solved it already. :)

Let us know how you go.

PS: You are installing this stuff from the official Ubuntu repositories? That is the best and safest way. Avoid installing stuff from third-party sites where possible (unavoidable with Chrome, but there is a Ubuntu specific version in the repositories: Chromium).
