### Post by Zelut on 2006-03-12
Well since there are very, very few viruses ever written for Linux you can scratch the Virus scanner off the list.  Defrag is done automagically on x number of boots.. and spyware isn't really an issue either.

If you want your computer to do something productive while you're gone maybe check out the Folding @ Home project & donate processor time to research.  I've got all of my machines doing it.
