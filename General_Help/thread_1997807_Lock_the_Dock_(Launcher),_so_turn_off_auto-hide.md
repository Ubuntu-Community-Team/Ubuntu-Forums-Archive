### Post by sanderj on 2012-06-05
Hi,

I'm running Ubuntu 12.04 (upgraded from earlier versions). The Dock on the left side (or is called Launcher?) will auto-hide. When I want the Dock back and I hold my mouse on the left, it often takes a lot of seconds before the Dock comes back. That's very annoying.

So: how can I lock the Dock so that it is always visible? In other words: how can I disable the auto-hide?

I tried "unsettings", but I can't find the option (maybe just overlooking it).

Furthermore:
- I have a i3 machine with 4GB RAM, so the hardware is not the problem
- I believe a fresh install of 12.04 has the auto-hide turned off. So maybe the auto-hiding is the result of upgrades from previous Ubuntu versions.

Help appreciated.

---

### Post by bspjojo on 2012-06-05
system settings, appearance, behavior, then turn off "autohide the launcher"

---

### Post by sanderj on 2012-06-05
> **bspjojo said:**
> system settings, appearance, behavior, then turn off "autohide the launcher"

It was already Off. I turned it On, quit System Settings, went back in, turned it Off again, and ... bingo!!! The Dock / Launch is now always visible.

Thank you very much! This make Unity much more usable for me. 

Thank you!
