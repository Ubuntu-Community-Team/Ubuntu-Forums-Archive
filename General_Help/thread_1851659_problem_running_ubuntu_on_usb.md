### Post by linuxman94 on 2011-09-28
You made what's called a Live USB.  The live usb/cd loads into the computer's RAM so that you can test out the OS before permanently installing it.  If you used unetbootin to make the live usb, do it again but this time in the box that says "Space used to preserve all files...", put the amount in megabytes that you want to be used for storage.
