### Post by bville09 on 2010-05-12
The netbook edition is just optimized for smaller screens, not lower cpu power (other than that, it's exactly the same as regular ubuntu). I'd go with Xubuntu however, as it's far lighter than vanilla ubuntu, with the same functionality, even looks kind of the same and has the same applications for it (even works with actual gnome and gtk+ applications, last time I tried it).

See, most (if not all) netbooks can run Windows 7, which is extremely taxing on older systems (even slows down on an old Pentium Extreme with a 9250 radeon - which completely eats even the heaviest of KDE setups), the biggest critism about 7 on netbooks though is the screen (it isn't optimized for it), it still runs on netbooks, but the text and resolution are both tiny, though, same old windows underneath all the makeup. It's kind of the same with Ubuntu, the netbook edition is actually optimised for small screens, and the text and panels reflect this, but it's still the same old ubuntu under the hood.

edit: if you've absolutely gotta have ubuntu (and nothing else) tweak some options in the alternate install to make it run faster on your system. to be honest though, I'm pretty sure it may even work fine on it at stock (if you've got some good ram and maybe a video card to match it), as long as it's like a later k6 (like the ones just before the athlons came in - the k6-3) - which works fairly good for me. if however you mean to say you have an original amd k6, then i wish you luck in trying to get ubuntu to run on your system, and suggest a copy of crunchbang or even dsl.
