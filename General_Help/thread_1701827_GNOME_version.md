### Post by 3Miro on 2011-03-08
> **TimmyK. said:**
> I'm not sure. It doesn't say anything about GNOME 2.32.0. And could my problems be caused by using the wrong version?

Depends on the theme. I think M4L was pretty big thing. Most themes are version independent since they have only graphics like Icons. Some themes, however, change behavior inside the system, so they may be incompatible.

Actually, you should be able to extract only parts of the theme, like icons, windows decorations, fonts and so on. In that way you can make a partial M4L and miss only a few of the more demanding features.
