### Post by aysiu on 2007-02-25
Are you using Ubuntu or Kubuntu?

If you're using Ubuntu, press Alt-F2 and type ```
gnome-screenshot --delay 5
``` then leave the active drop-down menu open. If you're using Kubuntu, you can change the time delay with KSnapshot.

### Post by crossedout on 2007-02-26
You can use GIMP to capture the drop-down menus.

Open GIMP-> Choose File -> Choose Acquire -> Select Screenshot
A popup opens asking for 'Single Window' or 'Whole Screen'.
Choose 'Whole Screen' and choose a delay time.  You can either minimize any open windows prior to clicking 'Grab' or set the delay to a length of time long enough to minimize the windows and open the drop-down menu.

Good luck!
