### Post by Ginsu543 on 2010-08-09
Okay, now it makes sense. Simply copying the LiveCD iso itself to the computer does not install the operating system. If you want to install the operating system, you must first burn the iso image onto a CD and then run the CD. If you have a Ubuntu system at home, you can do this easily using Brasero (comes pre-installed). Just make sure to use the option "Burn Image." It will ask you to direct it to the where the iso image is located. Input the correct directory path and run it. Once you have burned the image to CD, now you have a working LiveCD. If you want to install Ubuntu on your school computer (I'm assuming you have permission to do this), then take the LiveCD to school, put it in the CD-ROM, and boot up. Once the LiveCD boots up, just follow the directions on-screen to install Ubuntu OS.

If you have a Windows box at home, you'll need CD burning software like Nero.
