### Post by ajgreeny on 2010-07-06
Have you really solved this yet?

If not download the linux tarball for your architecture (32 or 64 bit) from [http://www.mozilla.org/projects/calendar/sunbird/download.html](http://www.mozilla.org/projects/calendar/sunbird/download.html) the unpack the tarball by double clicking the file and extract it in the normal way.  Amongst the files in the sunbird folder that will be made when you extract, there will be an executable **sunbird** file, and a shell script also called **sunbird**, if I remember correctly.  If you double click that shell script or run it from terminal, sunbird should start for you.

I say all this from memory as I don't run it separately from thunderbird's lightning add-on any more, though I did in the past.
