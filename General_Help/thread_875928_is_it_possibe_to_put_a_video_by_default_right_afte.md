### Post by spupy on 2008-07-31
This is great idea! It is certainly possible. You have to add
```
mplayer -fs videoname.avi
```
as a command to the list of programs that start at login. Unfortunately I don't use Gnome right now, but someone here should be able to tell you where to put this command.
