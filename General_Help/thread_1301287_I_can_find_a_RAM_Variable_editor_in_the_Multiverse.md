### Post by Rhubarb on 2009-10-25
There's a very nice app that'll do this for you in the ubuntu repos.
It's called scanmem
[http://linux.die.net/man/1/scanmem](http://linux.die.net/man/1/scanmem)

I'm not sure if you can specify the actual address in RAM that you wish to change, but you can use that program to find the address automatically after a narrowing it down several times, then change the value in that memory location.

I've tried this out on a few games, it seems to work very well too.

Send me a PM if you need some help with using scanmem.
