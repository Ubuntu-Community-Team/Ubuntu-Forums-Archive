### Post by amohanty on 2008-04-29
havent used windows in a while so memory is a bit hazy.

right click on taskbar
open task manager
select explorer.exe and kill it (should get restarted)
if not press Win-R and type in explorer.exe

If that does not fix it, look in the task manager to see if any ubuntu installer related processes are on, and kill those.

Finally a whole bunch of windows CD writing software, usually start up tons of f...ing software to monitor and manage writing, depending on what software you are using, kill the appropriate programs and try again.

If all else fails, reboot :) after all this is windows... assuming the problem is in windows
