### Post by deadflowr on 2018-10-02
There's a printers extension you can add.
You have a choice of which way to install it.
Choice 1 is open the Software store > go to Addons >. go to shell extensions > scroll down to Printers, click install.
or
Alternatively you can install it from the gnome shell extensions web site here: 
[https://extensions.gnome.org/extension/1218/printers/]("https://extensions.gnome.org/extension/1218/printers/")

Note that the extensions installed from the web site will prompt you whenever a new update has come, and you would need to install that update manually yourself.
(You can disable the notifications for the updates, but it affects updates for all added gnome shell extensions, by going to System Settings > notifications and toggle the gnome-shell integration to off)

While I know for fact the thsoe extensions installed from the web page will prompt when new updates are available, I am not as sure about those installed through the software store, 
as I do not recall any that I have installed doing so, but they might. I'm just not sure.

Hope it helps
