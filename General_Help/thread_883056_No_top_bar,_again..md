### Post by Animeniac on 2008-08-07
Never mind. I installed xgl and everything is working fine, again. The animations are working fine and also the top bar above the windows.

I went back to the old thread about the same problem. I entered 'metacity --replace', just so that I can see the text in Terminal. I then opened another terminal and typed in 'compiz --replace'. One of the errors was 'No Xgl present' or something like that. So I went to Synaptic Package Manager and downloaded and installed the file with 'xgl' in it. I restarted the computer and then Evreything went back to normal. :)
