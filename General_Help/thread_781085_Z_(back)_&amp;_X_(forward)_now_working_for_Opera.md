### Post by Solrac924 on 2008-05-04
Since my install of Ubuntu last week, I've been getting reacquainted with Firefox. because i made the switch to Opera (on Windows) years ago, I thought i'd install it here on Linux and compare the two. Three words: It is fast!

i have run into 2 bugs & have since resolved them. they weren't big issues, nor was the trick complex. anyway, since i couldn't find the help i needed (searching the forums & Googling around) i thought i'd post the solution here.

1-in Opera, you can use the keyboard to go back & forward through the pages you surfed. i'm not talking about alt + "arrow key". that's goofy. why use 2 hands when it takes less effort to click the back arrow? i'm referring to the very handy hot-keys 'z' & 'x'. again, for me it didn't work automatically, so i had to change the keyboard setup by clicking: Tools -> Preferences -> Advanced -> Shortcuts -> change the "Keyboard Setup" to "Opera 9.2 Compatible".

2-to enable the smooth scroll by middle-clicking & mouse up/down, you must click: Tools -> Preferences -> Advanced -> Shortcuts -> Middle-Click Options & click "Start panning" 

usually these features are ready out-of-the-box in "Opera for Windows", so i just hope this post makes the transition better for Opera users new to Ubuntu. :mrgreen:
