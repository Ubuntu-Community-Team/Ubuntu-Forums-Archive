### Post by dodo3773 on 2012-05-06
There are usually 2 .bashrc files. One in /root and one in your home folder (there is also a system wide one somewhere in /etc probably). I am sure the tutorial you are following is telling you to place it in your home folder. If it does not exist feel free to create one.

### Post by jockyburns on 2012-05-06
Should be in your home folder. Open it from the side panel and also at the top of the screen select view, then from the drop down menu, select show hidden files. When you scroll down the home folder you'll see the bashrc file
