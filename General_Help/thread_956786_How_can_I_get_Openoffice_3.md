### Post by Yellow Pasque on 2008-10-23
It's possible if you build from source (this might be a large undertaking for someone not familiar with building software from source). 

Also, If you have Kubuntu 8.10, there is a pre-made .deb here:
[https://launchpad.net/~openoffice-pkgs/+archive](https://launchpad.net/~openoffice-pkgs/+archive)

---

### Post by caleb12 on 2008-10-23
The above link is a tar ball of all the necessary precompiled .debs that are needed to install oo3. Just extract the tar ball and sudo dpkg *.deb - and voila... open office 3. And it won't conflict with the preexisting 2.4 installation...
