### Post by CatKiller on 2009-06-20
> **abhilashm86 said:**
> so what can we call ubuntu?? i know its not FSF, ok fine we may call it LGPL.......

Only some components are LGPL, so calling the whole thing LGPL would not be at all accurate. Call it what it is; a distribution of software.

The reason the FSF don't consider Ubuntu to be free is because it makes it easy to install non-Free software. They would much rather that it was difficult to install the NVidia driver, for example. Only an incredibly small number of GNU/Linux distributions meet the FSF requirements to be included in their list of Free distributions.
