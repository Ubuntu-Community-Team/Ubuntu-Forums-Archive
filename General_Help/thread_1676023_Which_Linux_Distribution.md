### Post by trinitydan on 2011-01-26
> **ubudog said:**
> Ubuntu should work, but it may not be super fast.  I would recommend CrunchBang Linux (based on Ubuntu, last time I checked).

It is very lightweight, and super fast on old PC's.
I agree, CrunchBang is super fast, although not exactly "lightweight".  More like "middleweight".  You would probably see a boot time around 20-25 seconds on that machine.  However, it is nolonger Ubuntu based, it is now based off Debian Squeeze, which is still great, as you have access to .deb files and the Debian repositories.

For the OP, I am going to have to suggest that if you are new to Linux you stick with Ubuntu for a while.  It will be a little sluggish with those specs, but it will work.  Ubuntu is a great Linux distro to learn with.  Especially with all the great help that is available on this forum.  Ubuntu is like Linux with training wheels.  CrunchBang not so much.  When you feel ready to take the training wheels off, check out CrunchBang if you want a snappy and responsive distro for your older computer. :D  This may not be the best advice for everyone.  If you are just a natural with computers maybe diving right in to CrunchBang would be a good idea in the long run.
