### Post by bobcollard on 2010-05-02
> **markw10 said:**
> I just downloaded Ubuntu Netbook Edition 10.04 and am running it from a usb flash drive on my Acer laptop.
At times I'd like to use the Gnome desktop.  I know with the previous version there was a way to change to Gnome.  Is there still a way to change to Gnome on the 10.04 version and if so how do I change to Gnome and back to Netbook Edition desktop?  Thanks.
On the bottom left corner of your log-in screen is a button for Sessions.  If you have gnome desktop installed the choice is there.  If not, open Synaptic Package Manager and install gnome-desktop, restart and the choice will be available.

### Post by escottster on 2010-05-28
I was having trouble with this as well, and got the same error message about the swfdec-mozilla.  

The previous posts are correct about gnome already being installed with UNE, but you don't see the drop box with those option until you have clicked on the username you want to log into where the password box appears.  I kept looking right past it.  

If for some reason you don't see it, you can change the default UI under the login screen option of the settings tab.  Also make sure you're not set to automatically log in when the computer starts if you want to use that feature.
