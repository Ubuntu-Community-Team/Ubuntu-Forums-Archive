SKYPE IN 11.04


1. OPEN TERMINAL AND TYPE: sudo apt-get install libqt4-dbus libqt4-network libqt4-xml libasound2


2 TYPE IN TERMINAL STEP 2: sudo sh -c 'echo "deb [http://archive.canonical.com/ubuntu/](http://archive.canonical.com/ubuntu/) maverick partner" >> \
/etc/apt/sources.list.d/canonical_partner.list'

3. TYPE IN TERMINAL STEP 3: sudo-s 

4. TYPE IN TERMINAL STEP 4: synaptic or open synaptic package manager from menu in administration.

5. RELOAD IT !

6. TYPE SKYPE AND MARK FOR INSTALL.

7 APPLY
