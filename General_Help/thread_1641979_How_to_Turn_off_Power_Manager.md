### Post by Sky Aisling on 2010-12-10
Here are instructions on how to turn off Power Manager from Ali Abdalla who is the program manager.  My question to Ali was 'how do I permanently turn off Power Manager not just *--quit* for the session.

Ali writes:

[I]You can disabled it by uninstalling it, or if you don't want to do     that, you can move the file xfce4-power-manager.desktop which is in     /etc/xdg/autostart/ to another place, then you can do your tests,     once you finish and if you want to use the autostart option again     you can place the file again in that directory, the power manager is     just a userspace application that does not do anything that will     cause problems to other softwares or hardwares, so yes you can     remove it completeley if you don't fell that you actually need it.

[/I]
