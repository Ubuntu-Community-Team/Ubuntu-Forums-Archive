### Post by Tibuda on 2009-06-02
Problem 1: Can you explain what Windows Live Writer does?

Problem 2: You can use Glade to design Gtk+ windows, QtDesigner to design Qt windows and wxGlade to design wxWidgets windows. Those are not a full-featured IDE, but you can use the windows you design with these applications with almost any language. There's also the MonoDevelop IDE, which have a designer. All available on Ubuntu package manager.

Problem 3/4: Check [http://appdb.winehq.org/](http://appdb.winehq.org/) for software compatibility with the WINE, which allows you to run Windows application on Linux.

Problem 5: I've heard about some problems with Interl cards, but I can't help you with this.

Problem 6: Can't help too.
