### Post by fabounet on 2009-02-03
The v2.0 is standing out little by little, and I've made a new package of the current state (beta 2).
News compared to the previous beta are :
 the Curve view is now in openGL
 a search engine in the config panel
 window decorators for the dialog bubbles
 ability to lock icons
 the Cairo-Desklets can rotate in 3D (that is to say, *inside* your screen)
 a new plug-in illusion

any return is appreciated since it is only a beta version :-)
