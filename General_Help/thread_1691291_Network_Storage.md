### Post by jerenept on 2011-02-20
I prefer [FreeNAS]("http://freenas.org/") over Ubuntu as a Network Storage Device, but that's just my preference.....

It is a bit easier to administer, as it has a config webpage, (like a router)

[This]("http://dailycupoftech.com/howto-install-freenas/") is a great step-by-step guide to installing FreeNAS.

FreeNAS can run on 192 MB of RAM or more, and very slow processors are supported.
