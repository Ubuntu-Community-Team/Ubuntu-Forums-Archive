### Post by ibuclaw on 2008-05-21
As a developer talking to a developer.

I say that if you've got the energy and willingness to do so, code it yourself!

Although, you probably haven't seen the full potential of Sage yet.
I mean, Open Source interpretations of MATLAB and Mathematica functions are there in the language!

But I also implore that you don't start re-inventing the wheel either!
If you feel that you can improve the source of the application that's already there, then do it.

If you find a better program from, say, sourceforge and you can improve the source to meet your needs, then that is perhaps one step better!
