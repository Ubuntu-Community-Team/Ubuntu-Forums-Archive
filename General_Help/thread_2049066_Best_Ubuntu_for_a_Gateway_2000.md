### Post by Topsiho on 2012-08-28
I have googled on "Gateway 2000", and it seems that there are more than one computers with that description, some very old, even from 1994 or so. Some laptops, some desktop computers.

I think that the ram, HD-size and cpu of these machines are not up to any graphical desktop (Unity, Gnome, KDE, the smaller ones too), but probably they will run any "pure" Linux in a text screen (on the command line). See Distrowatch to find one, the smaller the better.

This is a great way just to explore the command line of Linux, and really learn a lot, if you want to do that. It's a voyage into so far unexplored (by you) territories :) As every explorer, you'll only arrive somewhere with some perseverance.

Google on "Linux command line" or so, to find information you need.
