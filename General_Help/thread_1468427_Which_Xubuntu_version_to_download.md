### Post by cascade9 on 2010-05-01
AMD64 is the marketing name AMD made up for x86_64- x86, with 64bit extensions. Since they released the 1st x86_64 CPU, they can call it whatever they like IMO. 

Intel adopted the AMD64 extensions a few years later, so intel 64bit CPU use the same instruction sets...AMD64 will runs on intel x86_64 CPUs (but not on itanium 64, IA64, that is a different instruction set)
