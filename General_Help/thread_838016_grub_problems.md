### Post by manfer on 2008-06-23
No need to reinstall ubuntu. You only lost grub boot loader after installing Windows XP and you can restore it.

If you have a Ubuntu Live CD you can restore it with the instructions on this tread:

[http://ubuntuforums.org/showpost.php?p=1308395&postcount=1](http://ubuntuforums.org/showpost.php?p=1308395&postcount=1)

*Note: Of course this is true if you install Windows XP exactly where it was (format its partition and reinstall there). If you change the disk partioning the previous grub configuration won't work and besides you could finish losing ubuntu installation if you don't know exactly what you are doing.*
