### Post by 3rdalbum on 2010-09-26
I can't find a "gpointer" package - are you talking about gpointing-device-settings?

This should appear under System > Preferences > Pointing Devices on Ubuntu, or in Pointing Devices in the KDE menu on Kubuntu.

Packages that you install from the repositories install their own menu item, where appropriate. Command-line programs, themes, panel applets, documentation, fonts and libraries do not install menu items, for obvious reasons. If you install a program and cannot find where it has installed to, you can always go to Synaptic and right-click the package, choose Properties and then Installed Files.
