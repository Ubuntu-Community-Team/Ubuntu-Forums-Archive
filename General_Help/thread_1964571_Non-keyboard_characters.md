### Post by zombifier25 on 2012-04-24
AFAIK the feature is Windows-only.
If you want to type the &#8364; symbol, you can remap one of your keys (preferable the Right Alt) to Alt-Gr and type some of the special keys by holding AltGr and press on key on the keyboard (in the euro symbol's case, it's AltGr - 5)

### Post by sanderj on 2012-04-24
I use AltGr + accent-key (for example ' ), then letter to get letters like:

é ó ö ç ò 

And with AltGr + letter key works too:

³ ¤ ¹ ¤ ’ ‘ &#731; á ß ð æ œ © ® ñ µ ø œ ï í ú ü þ ë é å ä « »

This is enabled by selecting "International Keyboard with alt-gr keys (or dead keys??)" during install of Ubuntu.
