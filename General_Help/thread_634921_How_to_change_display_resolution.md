Well, you can change resolutions with the System -> Preferences -> Screen Resolution tool. If you want to do it in some form of script, I guess you'd need to find out how XRandR works. I seem to recall that there's some kind of keyboard shortcut you can use to go to the next lowest or next highest resolution, but I really can't remember what it was.
