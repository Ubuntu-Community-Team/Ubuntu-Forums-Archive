### Post by por100pre1 on 2007-09-26
Try looking for an HP printer. [Here's]("http://www.linuxprinting.org/printer_list.cgi?make=HP") a list that shows which one work with Linux and which ones don't. Verify the recommended driver for the selected printer, look for hpijs or hplip drivers. hplip can be installed with apt-get. Other brands can be verified in the Linux Printing page as well.
