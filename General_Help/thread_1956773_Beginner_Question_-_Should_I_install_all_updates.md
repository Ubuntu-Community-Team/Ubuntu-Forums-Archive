### Post by QIII on 2012-04-11
You should install them in Windows, too.  They shouldn't "bog down" your machine except when the updates are running.  Failure to keep on top of updates is a security risk and leaves you vulnerable.

And yes, install them in Linux. They are improvements and security updates.

The only exception is when you are offered "partial updates.". Don't do updates then.  Wait a day and check again.
