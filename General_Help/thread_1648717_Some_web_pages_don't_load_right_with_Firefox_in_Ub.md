### Post by amauk on 2010-12-19
You're using a dark theme
Check if the default colours Firefox is set to use are dark as well

Some websites might just "assume" that the default browser background colour is white
If your browser's background colour is not white, these websites may not look right

Edit > Preferences > "Content" tab
Click "Colors"
