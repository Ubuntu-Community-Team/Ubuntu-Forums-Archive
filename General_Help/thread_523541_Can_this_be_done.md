### Post by gobbles414 on 2007-08-12
skoalman88,

I know that in Feisty, it is possible to go SYSTEM --> ADMINISTRATION --> SOFTWARE SOURCES and enable something called BACKPORTS.  This will give you some of the newer software available to newer versions of the OS. For example, in Feisty, I was able to access a newer version of Wine. That really improved the performance of a couple of programs. As already mentioned, you can install programs from source as well.

I've actually had to install a couple of apps from source because the versions in the repository are outdated. A good guide on how to install things in Ubuntu can be found [here]("http://monkeyblog.org/ubuntu/installing.html").
