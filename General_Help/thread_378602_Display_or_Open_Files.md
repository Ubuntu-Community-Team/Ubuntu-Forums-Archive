### Post by daz4126 on 2007-03-07
Every time I doubleclick on a .rb or html file in Nautilus to open it, I get a dialogue that says 'Do you want to run foo.rb or Display it's contents' with the options to 'run in terminal' 'display' 'cancel' or 'run'. I need to choose 'Display' to get it to open in a text-editor. Dose anybody know if it is possible to make this the default behaviour without it asking each time?

thanks,

DAZ

---

### Post by david_2001 on 2007-03-07
Select Edit | Preferences in Nautilus, click the Behaviour tab and then check "View executable text files when they are clicked".
