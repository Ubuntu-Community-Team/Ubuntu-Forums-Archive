### Post by joshzam on 2008-08-09
You can absolutely remove the taskbar (gnome-panel) if you wish.

Go to System > Preferences > Sessions and, under the Current Session tab, select gnome-panel and Remove it. Then go to the Session Options tab and make sure "Automatically remember..." is checked and click on the "Remember Currently..." button.

I did this when I was playing around with AWN is it freed up my desktop nicely. I ended up having boot problems later on and, without a definite culprit, blamed AWN and set everything back to how it was.
