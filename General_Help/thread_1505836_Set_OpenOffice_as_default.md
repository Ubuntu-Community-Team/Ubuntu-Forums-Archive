### Post by scouser73 on 2010-06-10
Right click a document that you want to have open with OpenOffice

Select the tab labelled **Open With** & choose OpenOffice Writer/Calc/Base

If there is no option to select Openoffice, then click **Add** and select the corresponding application for the file type that you want to have opened in OpenOffice.
