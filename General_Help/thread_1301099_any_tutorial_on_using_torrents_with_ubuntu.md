### Post by rockstarrem on 2009-10-25
I don't recommend using the version of Transmission that comes with Ubuntu, I'd say either get the latest deb package or build the latest one yourself. A lot of sites don't support the version that comes with Ubuntu, and it can get you banned. Transmission is a good client, though.
