### Post by doclivingston on 2005-04-12
Although booting up quicker is something that no-one would say no to, there are huge number of things that the livecd has to do when you boot up. One of the more obvious ones is to detect and configure all your hardware - it can't remember all of these things between boots, so it has to re-do it every time it starts.

There are other things which the livecd also has to do, which an installed version doesn't, which take up time. The livecds of a couple of distos I've seen all seem to take about the same time to load.
