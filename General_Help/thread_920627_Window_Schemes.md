### Post by howefield on 2008-09-15
try pressing alt and f2 keys together.

In the run window that pops up type   emerald --replace

If that works and you want it to start up on boot, you can add it to your sessions list.

---

### Post by douggiephresh on 2008-09-15
how do i add it to my sessions list?

---

### Post by howefield on 2008-09-15
Go to System > Preferences > Sessions and click add.

In the name field type Emerald
In the Command field type emerald --replace
In the Comment field type whatever you want, it doesn't matter, leave it blank even.

Then ok your way back out.
