### Post by mb_webguy on 2009-02-18
If you have a decent amount of system resources, go with either Ubuntu or Kubuntu.  Gnome and KDE are both full-featured and robust desktop environments, and the question of which is better is a matter of personal preference.

If your system has limited resources (i.e. slow processor and/or low memory), then you may instead want to go with Xubuntu.  It uses the Xfce desktop environment which is more lightweight than Gnome or KDE.

And since all of these are different flavors of the same distribution, you're not locked in to using that particular desktop environment.  You can install any of these (and others besides) regardless of which you initially choose.  You can even have more than one installed at once, and switch between them as you choose.  

For example, I installed Ubuntu (with Gnome), but then installed the Openbox window manager (which is like a desktop environment but without the desktop).  Openbox is even more lightweight than Xfce, for use when I don't need all of the bells and whistles of Gnome.  I just choose which I want to use when I login.
