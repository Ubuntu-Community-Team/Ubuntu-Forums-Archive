### Post by hojjat on 2011-11-14
The only version of Eclipse that you can download and install from the Ubuntu Software Center is Eclipse for Java Developers. I want to enable Java EE features in it. Do you know what additional software should I install from within Eclipse's interface for that?

---

### Post by hojjat on 2011-11-14
I found the answer myself. I had to install Web Tools Platform from [http://download.eclipse.org/webtools/repository/helios/](http://download.eclipse.org/webtools/repository/helios/) (already in the list of sources in Eclipse) and optionally, the Java EE tools from [http://download.eclipse.org/releases/europa/](http://download.eclipse.org/releases/europa/)
