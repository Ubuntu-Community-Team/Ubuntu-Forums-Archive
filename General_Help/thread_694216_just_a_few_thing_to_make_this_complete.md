### Post by doorknob60 on 2008-02-11
For the sounds, go to System>Preferences>Sound and go to the Sounds tab and disable the Log In sound (The question sound also plays when it asks you to log in if you want to disable that as well). For the home folder thing, you might want to check System>Preferences>Sessions and see if there's anything weird in there set to start up, but that's probably not it.
