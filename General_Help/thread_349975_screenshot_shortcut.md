### Post by aysiu on 2007-01-30
I don't think there is one defined by default in KDE. You can easily make one, though. Right-click on the KMenu, edit it, find the entry for KSnapshot, click on the box next to the keyboard shortcut, then press the key combination you want for a screenshot (PrntScreen, for example).

I think KSnapshot is installed by default, but if not, you can easily install it: ```
sudo apt-get update
sudo apt-get install ksnapshot
```
