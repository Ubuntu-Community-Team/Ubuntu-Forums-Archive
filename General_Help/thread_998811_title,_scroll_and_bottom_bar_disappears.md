### Post by ajgreeny on 2008-12-01
It certainly sounds like a compiz problem.  If you don't have it installed get compizconfig-settings-manager using apt and then in System->Preferences->Compizconfig settings manager in the menus youcan try different combinations of plugins to see if you can overcomethe problem.  Interestingly in my install of 8.10 I had the same problem until I imported the cpmpiz profile from my present 8.04 on the same machine into compiz.  I also found that I had no extxnsions in Firefox 3 that worked with the default compiz settings in 8.10, but they all do again now.

Trial and error I'm afraid, I have no idea which of the plugins is the one that caused the difficulty, I'm just glad I got it to work again.  Even though compiz adds nothing to the work the computer does, I like to use it if I can, it gives me a nice superior, comfortable feeling when windows users see it a go "WOW!"
