### Post by TheHimself on 2009-08-23
I installed Kile 2.0.1 in Gnome (by downloading the package and not using Synaptic) and it works fine. Now update manager keeps haranguing  me  to install a ton of new apps like Konqueror, Dolphin, Ruby and much more. How can I stop that?

---

### Post by scouser73 on 2009-08-23
Go to **Synaptic Package Manager**, find the software that the Update Manager is recommending you install, and click **Package** then check the box marked **Lock Version**. You will no longer see those updates in the Update Manager.

---

### Post by TheHimself on 2009-08-23
Thank you it worked.
