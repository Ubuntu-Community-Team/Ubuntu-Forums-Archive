### Post by romis on 2007-12-16
Not every program from Synaptec will show up, mainly because not every program  is a standalone program.  What I mean is some programs are command line, such as clive, and others are part of another program (for example extra skins or themes). To see which programs will show up it's usually better to go to the Applications --> Add/Remove menu .And make sure where it says "Show" it says All Available or All Open Source, depending on your preference. Then you can just  go through and check mark the ones you want and click apply and it will automatically install them and put them in the  app db.

To use WINE you just run the installation like normal. Not every piece of software works 100% yet, but  keep in mind they're always developing it and always  getting more and more Windows APIs in there. Best thing would be to go to the WINE App Database at [http://appdb.winehq.org/](http://appdb.winehq.org/) and search for your particular software to see if it's there and whether or not it works and how much of it works.
