### Post by -kg- on 2010-05-21
> **JoelOl75 said:**
> You need to install libdvdcss2 from the medibuntu repos

Exactly.  The actual Medibuntu site is located at: [http://www.medibuntu.org/]("http://www.medibuntu.org/")  The Community Documentation that Spr0k3t pointed you to is also accessible from the "Repository Howto" tab at the top of that page.

Installing libdvdcss2 is one of the automatic things I do when I install Ubuntu on any computer that I install it on (just in case!:P ), right along with "ubuntu-restricted-extras".  That will give you the ability to play an "encrypted" commercial DVD on your computer under Ubuntu.  The restricted extras will install Adobe flash player, as well as some of the "restricted" codecs and all.
