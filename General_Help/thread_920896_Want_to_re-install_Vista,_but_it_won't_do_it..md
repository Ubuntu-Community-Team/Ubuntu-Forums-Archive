### Post by benerivo on 2008-09-15
The format it needs to be in is ntfs. I can't say for certain whether Ubuntu has done something to the hard drive that Vista isn't compatible with, but it shouldn't have done. What is the error code? I would try loading ubuntu from the live cd. From there, you can run Gparted partition manager from the main menu, and format the whole disk as ntfs. Then try the Vista installation again.
