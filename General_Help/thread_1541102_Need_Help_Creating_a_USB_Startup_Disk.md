### Post by ajgreeny on 2010-07-28
The startup disk creator will only burn to usb an iso of ubuntu and a very few other distros, eg mint, which is so close to ubuntu that it could almost be called ubuntu+.  As howefield says, unetbootin will make just about any iso file of any distro into a usb bootable disk.
