### Post by Paddy Landau on 2011-04-23
Ubuntu comes in various "flavours". The most lightweight at the moment is probably [Lubuntu]("http://lubuntu.net/"). There are other Linux distributions, some of which are even lighter.


[LIST=1]
[*]*Can I get my 3 mobile dongle usb stick to work with this OS*

All Ubuntu flavours come with a "Try It" option. Burn the Live CD to a CD (or load it onto a USB); boot from the CD (or USB); do not install it, but instead choose to try it. The OS will load and let you play around without making any changes to your computer. Plug in your 3 Mobile dongle USB and see whether it works.
[*]*Will a web browser work with OS*

Gosh, yes! Ubuntu is a fully-functioning operating system. It comes by default with Firefox, although you can also use Chromium (the Linux release of Google Chrome). There are a couple of other alternatives, but I'd recommend you use Chromium (comes with Lubuntu) because it's pretty lightweight.
[*]*Can I run an arts package with this OS*

I don't know what you mean by an arts package.
[*] *Can I also run a word processor from it*

It comes by default with a word processor. Up to now, it's been Open Office, but it will soon change to Libre Office; both can read and save as Microsoft documents. Lubuntu comes with Abiword, but you can easily install Open Office or Libre Office instead should you wish.
[*] And if so can I have a link for the download please

[Lubuntu]("http://lubuntu.net/")
[/LIST]

Good luck.

### Post by Paddy Landau on 2011-04-25
> **Toolless said:**
> I belive you can run Linux inside of windows
Sort-of correct: You can [download Wubi]("http://www.ubuntu.com/desktop/get-ubuntu/windows-installer"), which will install Ubuntu in a sort-of virtual area inside Windows. After installing, each time you reboot you will get an option to boot into Windows or Ubuntu. As far as I know, Lubuntu does not come with a Wubi installer.

However, Wubi is unreliable in the long term.

So, I do not recommend it for you, (1) because of its unreliability, and more importantly (2) because it only comes in Ubuntu, which would be too slow on your machine. (But, of course, you can try it if you want to.)

Regarding all the other screenshots, I'm a little confused. The Install Lubuntu should have given you other options. I think you may have to go for a manual installation; which, sorry, I am not good at.
