Just wanted to mention there is a new version of Gfax that runs under Jaunty and Karmic.  [http://gfax.cowlug.org]("http://gfax.cowlug.org")


Whats new in gfax-0.7.7
============================================
Gfax should now work with all software that can print to a CUPS printer.

Notable change is that Gfax MUST be running BEFORE you try to use the 
fax system.  It will minimize to the system tray.

Fax tracing ability without recompile.

Gfax is now under the GPLv3 license.
