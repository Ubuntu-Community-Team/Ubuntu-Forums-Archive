_________________________________
Search 1.
Search -> Advanced Search -> Search By Key Word -> select Search Entire Posts -> and in Key Word(s): 
box, type ltsp -> check that Search In Forum(s) -> shows Search All Open Forums is highlighted,
or click on Search All Open Forums -> Search Now
The threads are ordered according to the last post's date/time.

Search 2.
Search -> Advanced Search -> Search By Key Word -> select Search Titles Only -> Key Word(s): ltsp
-> Search All Forums -> Search Now
The threads are ordered according to the opening post's date/time.
__________________________________

I set up an ltsp subscription folder like so: I logged in then at the top of the page, clicked
User CP -> Edit Options -> Messaging & Notification -> Default Thread Subscription Mode
-> select , No email notification -> Save Changes at bottom of page.

Now, if I'm logged in, and viewing a thread, I can click Thread Tools , on the right hand side,
just at the top edge of the top post -> Subscribe to this Thread
The threads I subscribe to, and the threads that I post in, I find by clicking on
User Cp -> Subscribed Threads

Then I clicked on
User CP -> Subscribed Threads -> in left side bar, under Subscribed Thread -> Edit Folders
-> Add New Folders -> Folder Name , type ltsp , or any name to refer to ltsp
-> Save Folders

User CP -> Subscribed Threads -> Subscribed Threads in Folder: Subscriptions 
-> under Notifications, on right hand side, check boxes of ltsp threads -> bottom of page,
selected threads: -> select, move to folder -> Go -> Move # items from 'Subscriptions' 
-> Folder Name: -> select, ltsp -> Move Items
Now I can
User CP -> Subscribed Threads -> Subscribed Threads in Folder: Subscriptions 
-> Folder Controls -> Jump to Folder ->  select ltsp -> (Go)

When logged in, viewing a thread, and click Thread Tools -> Subscribe to this thread
-> Nofification Type , select no email notification -> Folder , select ltsp -> Add Subscription

I find it's best to subscribe to a thread before posting to it.

If I post in an LTSP Thread that I have not subscribed to first, then
User CP -> Subscribed Threads -> Subscribed Threads in Folder: Subscriptions 
-> under Notifications, on right hand side, check boxes of ltsp threads -> bottom of page,
selected threads: -> select, move to folder -> Go -> Move # items from 'Subscriptions' 
-> Folder Name: -> select, ltsp -> Move Items
