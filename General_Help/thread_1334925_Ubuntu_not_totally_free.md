### Post by MaxIBoy on 2009-11-23
"Free" as in "Freedom." 


Some of the software in it is proprietary software which doesn't respect your freedom. For example, software which is patented or where the source code is unavailable is nonfree. Ubuntu only has a few nonfree hardware drivers.
