### Post by jpeddicord on 2008-03-02
Check your font in Edit > Current Profile. There might have been a different font set on the terminal for some reason. "Use the system fixed width font" should fix it.

If for some reason it doesn't, check System > Preferences > Appearance, Fonts tab; Fixed Width font. Change it to something else and see if it works.
