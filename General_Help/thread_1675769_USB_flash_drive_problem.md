### Post by Zimmer on 2011-01-26
[https://help.ubuntu.com/community/Installation/FromUSBStickQuick](https://help.ubuntu.com/community/Installation/FromUSBStickQuick)

Also, if you want a liveCD on USB have a look at pendrivelinux.com for lots of tutorials on how to make them (all sorts of flavours of Linux...)

You need to download a program that can properly put the installer onto a memory stick. You can't just copy the .iso file onto the memory stick.
