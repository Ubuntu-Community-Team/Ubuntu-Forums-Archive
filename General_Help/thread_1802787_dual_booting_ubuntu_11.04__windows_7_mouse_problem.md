### Post by Mark Phelps on 2011-07-13
Not a common problem. If I had to guess (and I do), this is probably a driver problem in Win7.

Suggest when you're back in Win7, see if you can update the mouse driver.  You can try doing this in Device Manager.

You could also check the website for the Mouse manufacturer and see if they have downloadable drivers that are newer than the one you have.  This might actually work better than using Device Manager to update the driver.

### Post by Mark Phelps on 2011-07-13
> **tburntburn said:**
> Any other ideas?

Well, you could see about borrowing (or buying) a cheap USB wired mouse -- to at least get you up and working OK.

Then, turn off the wireless mouse and reboot without it turned on.  Wait a few minutes and turn on the wireless mouse.  See if Ubuntu does anything about detecting the "new" mouse.

Also, check the batteries on the wireless mouse.  I've gone through a few different such rodents, and on some, unless the batteries are fully charged, I start to get anomalies.
