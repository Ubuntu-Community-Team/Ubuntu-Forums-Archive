### Post by chocklet on 2012-07-08
Process to expand fu:

1. Go through "Getting Started with Ubuntu 12.04"
[http://files.ubuntu-manual.org/manuals/getting-started-with-ubuntu/12.04/en_US/screen/Getting%20Started%20with%20Ubuntu%2012.04.pdf](http://files.ubuntu-manual.org/manuals/getting-started-with-ubuntu/12.04/en_US/screen/Getting%20Started%20with%20Ubuntu%2012.04.pdf)

2. Learn about...
how to find information you need (no fair asking!)
how to configure Unity and Classic desktops
manually editing .desktop files
all Nautilus file manager capabilities
labels
how to partition a drive
fstab
grub2
logs

3. Learn scripting for beginning beginners...
[http://maketecheasier.com/write-linux-shell-scripts/2011/06/30](http://maketecheasier.com/write-linux-shell-scripts/2011/06/30)

4. Learn some commands...
[http://www.pixelbeat.org/cmdline.html](http://www.pixelbeat.org/cmdline.html)
[http://www.debianhelp.co.uk/commands.htm](http://www.debianhelp.co.uk/commands.htm)

5. Learn more scripting to be able to use commands effectively
[http://steve-parker.org/sh/sh.shtml](http://steve-parker.org/sh/sh.shtml)
[https://supportweb.cs.bham.ac.uk/documentation/tutorials/docsystem/build/tutorials/unixscripting/unixscripting.html](https://supportweb.cs.bham.ac.uk/documentation/tutorials/docsystem/build/tutorials/unixscripting/unixscripting.html)
[http://gd.tuwien.ac.at/linuxcommand.org/writing_shell_scripts.php](http://gd.tuwien.ac.at/linuxcommand.org/writing_shell_scripts.php)

6. Finish Zangband without cheating

Personally, I tried to do #6 before #5, and years later I'm still stuck there!
