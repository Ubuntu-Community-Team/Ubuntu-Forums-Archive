### Post by RealPSL on 2008-12-18
From the menu choose: Applications -> Add/Remove. Select Install Applications from the Show menu. In search entire flash. You should see the entry for Macromedia flash. Remove the check and click apply. That will uninstall flash.

To install select All Available Applications from the Show menu. In search enter flash. Choose macromedia flash plugin. Click Apply Changes and your reinstall is complete.
