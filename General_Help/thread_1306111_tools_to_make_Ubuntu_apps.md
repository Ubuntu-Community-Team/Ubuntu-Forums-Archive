### Post by Giblet5 on 2009-10-30
It's easier than it is on Windows or Mac, but you still have to know a little about programming in general.

Do you know any programming languages?

Python or Java are good languages for beginners. You don't want to try anything large scale with those...it'll break.

C and C++ are harder to learn but yield better results.

O'Reilly sells the books to take you from user to programmer. I doubt that anyone will argue that.

A little C++ knowledge plus Qt makes it possible to write very nice GUI applications that'll run on Linux, Mac, or Windows with just a recompile.
