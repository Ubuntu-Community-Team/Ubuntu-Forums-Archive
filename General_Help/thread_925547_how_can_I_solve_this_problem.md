### Post by ju2wheels on 2008-09-21
Everything to the right of that dotted bar is a dynamic object in itself thats tied to the network icon. So you can shove other icons between the bar and the network icon but in order to move that network icon you have to move the bar.

I think the only way around this is to right click on the bar, remove it from the panel, right click on the panel and add an individual network icon. However I think this might not allow open applications to dock on the panel although I havent tried to find out.
