### Post by deadflowr on 2014-09-15
Did you enable the partner repo?

To do so, open Software and Updates. Go to Other Software and check the box(es) for canonical partners.
Then if asked reload.
It should show up in the software center, or else
in a terminal run
```
sudo apt-get install skype
```
