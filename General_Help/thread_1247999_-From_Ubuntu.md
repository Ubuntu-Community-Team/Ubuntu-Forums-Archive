If I got you right, you want to know whether to upgrade from 8.04? Or what exactly?

You can run both, ubuntu studio and a regular ubuntu desktop install next to each other, either on different partitions (see my signature on grub) or simply by choosing your session in gdm (the login window...)

### Post by snowpine on 2009-08-23
> **themarker0 said:**
> Well, i was told that there was a way to upgrade Ubuntu to Ubuntu Studios. I was asking for help on that. I am upgrading to 9.04 atm.

Also i am a huge CML newb (CML=Command line)

System->Administration->Synaptic Package Manager

Control+F and search for packages named ubuntustudio. Mark all of the packages whose names start with ubuntustudio- for installation by right clicking on them. You might also install linux-rt if you want the real time kernel. Click Apply.
