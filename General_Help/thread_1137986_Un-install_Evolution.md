### Post by sekinto on 2009-04-26
I'm using Arch right now and have a pretty virgin Gnome install. I havewn't installed Evolution yet but I plan to later.

The only package I have installed that has anything to do with Evolution is evolution-data-server, so I'm guessing that is a core Gnome component so I wouldn't mess with that. I'm pretty sure it is safe to uninstall other evolution packages, just make sure to read the remove list in Synaptic when you hit apply to make sure nothing important is being removed (ubuntu-desktop is just a metapackage, so don't worry if that is on this list).
