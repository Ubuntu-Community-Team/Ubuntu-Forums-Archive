### Post by alecjw on 2006-10-04
How can I tell gaim to start as soon as I log in? I know there's some way of starting programs *on startup*, but is there one for when you log in?

---

### Post by Anonii on 2006-10-04
System -> Preferences -> Sessions -> Startup Programs -> Add

Type "gaim" (without quotes)

-> Ok -> Close -> Enjoy.
