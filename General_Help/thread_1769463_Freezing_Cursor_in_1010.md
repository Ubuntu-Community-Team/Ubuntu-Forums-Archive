### Post by alansecker on 2011-05-28
This had been a successful install of 10:10 on an HP Laptop. It was overlaid with Macbuntu and worked well for several months. A few weeks ago the user complained that the tracker pad wasn't working but a plug-in mouse did.

I checked the machine last night and found that the tracker pad only stopped working at the point when the Gnome icons appeared on the screen. Prior to that it was OK.

I booted as 'root' and found as the administrator, it worked fine. I need to find a way to restore it for the (very irritated user). Can anyone help?

### Post by linuxinstalledfromhdd on 2011-05-28
I would uninstall Macbuntu, and try to find an updated script installation.  Hopefully they still have the original folder they used to install Macbuntu, and inside that folder should be the uninstall.sh script. Run it. Then clean the entire system really good with Ubuntu-Tweak or Bleachbit when you are done.  Restart. And then download and install the latest script or deb file for Macbuntu. Make sure you have the very latest version available.
