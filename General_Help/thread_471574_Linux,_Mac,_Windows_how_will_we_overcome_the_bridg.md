### Post by TitanKing on 2007-06-12
Adobe CS3 on Linux, how will I do this, I cant work on Windows anymore, I just cant use it as an OS if I have Linux as choice. With the release of CS3 (Photoshop, Fireworks, Flash) and all its great new feature, I was tempted to use Windows for these applications sake. I could not get along with Windows. Being a web developer, graphics designer and programmer, Linux is the complete environment, powerful and stable. I tried Vista, but I need no computer telling me what I can and cannot do especially if its a resource hog, therefore I dumped Vista.

So what shall I try next? I need something that limits me in no way, when it comes to working. So I tried the virtual machine VMWare, it seemed nice and a possible solution at first, but everything felt slower, the cursor was jumpy, resources was heave, I needed a smooth cursor when it comes to graphics designing in any ways.

So I felt a bit defeated! Finally I needed to drop my Ubuntu as I really need to ran these tools, I cannot Dual boot, nor does the Wine emulator support these applications. I gathered some energy and started browsing for some solutions one more time. I saw a few posts everywhere, praising a application called Virtualbox. But as the name implied, it seemed like yet another Virtual machine, and my last experience failed. I thought, “hell it is Open-source and I have nothing to lose.”

Anyways, I saw that Automatix for Ubuntu already had Virtualbox included as an option to install, so I clicked on it and a few minutes later it was installed. On startup of Virtualbox I was presented with a very simple screen, I ran the very simple Wizard, mounted my XP cd and a few minutes later I could run any Windows application. You might ask but thats exactly a Virtual machine, yes perhaps but heres the twist;

   1. My applications is as fast as in Windows. I can see no performance decrease.
   2. I can share and copy files to my Linux host and visa versa.
   3. My mouse pointer does not lock to the guest screen, you have one mouse pointer and with that pointer you can click guest applications as well.
   4. You can run Windows applications headless.
   5. No drivers need be installed.
   6. I can share my clipboard with my guest and visa-versa.
   7. It gives me a total feeling that no virtual host is present, its as if applications are running in Linux native.

I finally have a solution where I can enjoy the power of Linux with the software support of Windows. This is such a huge relieve for me as I really felt depressed knowing I had to let go of Linux for a while in order to use these applications. But this, has all changed very recently.

Congratulations to the Virtualbox team, you have done a splendid job! We appreciate your work!

Some resources to get you started:
[http://www.getautomatix.com/](http://www.getautomatix.com/)
[http://www.ubuntu.com/](http://www.ubuntu.com/)
[http://www.virtualbox.org/](http://www.virtualbox.org/)
