### Post by nerdtron on 2015-04-23
KVM let's you run virtual machines on the server. It also have a GUI tool which you can install on your Ubuntu desktop to control/manage/create virtual machines on the server.

This is a good link: [http://www.howtogeek.com/117635/how-to-install-kvm-and-create-virtual-machines-on-ubuntu/](http://www.howtogeek.com/117635/how-to-install-kvm-and-create-virtual-machines-on-ubuntu/)
