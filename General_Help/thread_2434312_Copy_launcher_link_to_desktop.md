### Post by Dennis N on 2020-01-03
For Lubuntu 19.10, copy the .desktop file for Filezilla to your Desktop folder. It will then automatically appear on the Desktop as well.

The .desktop file is in /usr/share/applications if you installed Filezilla with apt or muon.

OR, simply find the launcher in the applications menu and drag the entry to the Desktop, release, and chose 'copy here' from the little menu that appears. Easy.
