### Post by bblaze on 2010-12-13
I never used wubi to install ubuntu. I simply used Gparted to make an extended partition. Format two ext4 logical partitions within the extended one. And also format the Linux swap for the Boot record. Honestly it was really simple to do and I would suggest that if you dont get WUBI working.
