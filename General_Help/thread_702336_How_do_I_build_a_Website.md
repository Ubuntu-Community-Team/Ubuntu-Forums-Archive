### Post by DrMega on 2008-02-20
> **supertails said:**
> Where can I buy one without yearly cost and if you don't build one how do companies sell them?

There will be a yearly cost. You buy the domain name (lease would be a better term because you have to renew it annually of biannually). Companies that sell them update their domain name servers to point to the IP address of where your site is hosted. You can host the site at home if you have the necessary kit and network connection, but the domain name will still need to be registered for the following reasons:

1. So the whole world can access your site without having to put in your IP address and
2. So nobody else can use the same domain name while you've got it.

Don't confuse domain name registration costs with hosting costs. If you register a domain for a few dollars a year, you don't get any webspace with that. If you buy a hosting package (in my opinion a better idea) then you get the domain and the webspace. In the latter case you don't have to worry about connection speed, up-time, reliability, security etc, and your domain will automatically point to your webspace without having to update name servers.
