### Post by lovinglinux on 2010-05-16
If you have the free Wordpress account, then you can't install plugins. If you host it on godaddy or other server then you can upload through the dashboard. Expand the plugins menu, then click "Add New". You will see an option to up&#314;oad at the top of the page.
