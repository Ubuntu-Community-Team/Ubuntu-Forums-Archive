### Post by dj59 on 2007-04-01
Hi Franscisco,unfortunately the Verizon setup CD only works with windows,and the easiest way to get your account setup is on a windows machine.Verizon's website for some reason will only recognize a windows system, and or browser to register your account with the software they provide you.You could try fooling it with the User Agent Switcher add on ,if you are using Mozilla FireFox .It is supposed to make Fire Fox appear to look like other types of browers, like Internet Explorer.You can find it by going to Fire Fox web page and click on the add on's link.Or try this link [https://addons.mozilla.org/en-US/firefox/addon/59](https://addons.mozilla.org/en-US/firefox/addon/59) .Once you do setup your account though you should be able to go to the Verizon Central web site and just log in to access the account with any browser or operating system.Good Luck Hope This Helped.
