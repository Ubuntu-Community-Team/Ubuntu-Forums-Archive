### Post by snowpine on 2010-04-12
I am partial to CrunchBang Linux on older hardware, personally.

However, before you get rid of your existing Ubuntu install, experiment with some lightweight environments (XFCE, LXDE, Fluxbox, etc.) which you can do without reinstalling.

If you find there is a big performance boost with an alternate desktop environment, cool, if not, the problem might be elsewhere (for example maybe applications take a long time to load because your hard drive is slow, in which case switching distros will make no difference since Chrome, Firefox, and OpenOffice are equally large across distros).
