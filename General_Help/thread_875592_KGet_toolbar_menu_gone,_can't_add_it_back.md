So I ended up using DownThemAll! with FlashGot instead of KGet + FlashGot.  KGet is seriously messed if there isn't an easy way to bring the toolbar and menu back.  And on top of that it saves that somewhere in some pref/config files that doesn't get removed when you "complete uninstall" KGet AND Konqueror using synaptic.

I did a quick "whereis" of KGet after uninstalling and it doesn't find it anywhere, and I can't seem to locate where it stores this asinine preference, so to hell with it.  It's going in the garbage.  I'll leave the original post in case someone is willing to find the solution and others run into this.

The DTA nterface is better, I get better speeds on the same files and it also works with Premium Account, meaning that it must share the same cookie.
