### Post by techunit on 2011-05-01
You can set that up using either Ubuntu Tweak or using keyboard shortcuts. Create shortcuts to a hidden director in your home folder. (done by naming the folder with a period first. example: .hidden folder)

Then link them to shortcut.

### Post by papibe on 2011-05-01
I you can do it with System -> Preferences -> Keyboard Shorcuts: create a custom shortcut for every folder you want.

HOWEVER, I really think you should check a category of programs called "launchers", for example Gnome-Do and Kupfer. They could solve that problem for you, and you also can use it to call almost any program you want with a couple of key strokes.

I hope it helps.

EDIT: if you want to avoid the install, test, uninstall, until it works for you, go to youtube and check for demos and tutorials of both gnome-do and kupfer.
