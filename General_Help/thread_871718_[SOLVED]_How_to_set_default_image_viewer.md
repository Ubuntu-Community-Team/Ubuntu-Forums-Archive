### Post by shelbyvision on 2008-07-27
This is a minor annoyance, but when I open a folder of images and right-click on one of the image files, I get a menu that has "Open with gimp image editor" as the default, and under that is "open with>" and a side menu of several optional programs, including "Image viewer", which I would prefer as the default. How can I change the default so that when I open any folder of images and double-click a file, it will open in Image Viewer?
Steve Shelby

---

### Post by rune0077 on 2008-07-27
Right click an image file, select "Properties", go to the "Open with" tab, and select the program you want to be standard. You'll need to do this with all the different image formats.
