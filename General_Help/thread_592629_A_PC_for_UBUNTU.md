### Post by Insightfill on 2007-10-26
I've run machines with lower specs than that as a personal web/file server, with ftp, ssh and VNC access.  For remote access and file transfer, they were fine.  The better question is "what do you plan to do with it?"

For general light server use, and as a general desktop (email, web, document creation) it should be fine.  I've found that generally Flash animation on web sites is the first thing to start bringing usefulness down dramatically.  I've got a Pentium 1GHz that actually has trouble keeping up with some of the children's games on the PBS website.

Since the system you're dealing with has gobs of RAM, try the live CD first (for Xubuntu and Ubuntu both).  The first 10 minutes are pretty sluggish, with lots of caching.  After the first 10 minutes, the CDROM light shouldn't go on as often, and you'll get a better idea of how capable the machine is.
