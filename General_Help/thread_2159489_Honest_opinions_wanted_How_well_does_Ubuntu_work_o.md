### Post by oldrocker99 on 2013-07-03
I have been using Chrubuntu for two months on a 2GB Acer C7 now, and it works like a champ. I'm able to: (1) Use MIXXX do do a radio show, (2) Use GIMP for photo editing, (3) Use LibreOffice to open ANY office-type file, (4) Play lots of games (from Steam and Desura), (5) Use PLEX Media Server with my Roku, (6) download source code from SourceForge and compile it, and more things. Chrubuntu allows my $200 Acer C7 do practically anything I can do on my desktop PC.

As far as the touchpad scrolling, did you try the Mouse control panel? Mine works fine, but I'm not a touchpad lover, so I use a USB mouse.
