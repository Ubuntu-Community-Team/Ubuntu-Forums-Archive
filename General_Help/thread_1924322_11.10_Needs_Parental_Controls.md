### Post by SeijiSensei on 2012-02-12

### Post by Juan Largo on 2012-02-13
> **Cookieh said:**
> There are many plugins and applets for firefox to do that, I used them for my old pc to stop me from getting redirected to sites with viruses.
If you go the Firefox route, there are some additional steps that need to be taken:

1.  Remove all other browsers such as chrome, opera, konqueror, midori, etc.

2.  Lock down your system to prevent the kids from installing alternative browsers.

3.  Make sure Firefox is directed to a profile that is locked down.  (It's a simple thing for the kids to rename or delete a profile folder in the user account.  Firefox will then create a new profile without any user preferences.)
