### Post by DoctorMO on 2008-03-17
If you created the CD yourself you may have burned it incorrectly. remember that the iso image is not a file to be burned onto the cd but an entire cd file system (collection of files) ready to be burned onto the CD, make sure that you can access the CD in other operating system and see lots of files and folders on it. If you only see the iso file then you will have to reburn the CD using the burn image option available in most cd burning tools.
