### Post by manfer on 2008-06-17
If you want to capture all the screen you only have to press PrintScr key on your keyboard. This key is to the right of function keys (F1, F2, F3... F12)

If you want to capture only a window instead, you have to open the window you want to capture and press Alt+PrintScr.

If it is enough for you that's the easy way. You don't have to install any other program. If it is not enough tell me.

### Post by manfer on 2008-06-17
Sorry I didn't see that "portion of"

With gimp you can use File --> Acquire --> Screen Shot... to obtain that snapshot you need.

If you don't have gimp installed and you want only a program for snapshots you could install and try ksnapshot.

prompt:~$ sudo apt-get install ksnapshot
