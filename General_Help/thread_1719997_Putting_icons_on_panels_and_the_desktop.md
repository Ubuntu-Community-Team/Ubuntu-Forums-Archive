### Post by hhh on 2011-04-06
I'm pretty sure you have to create them. For the desktop, right-click the desktop, choose "Create Launcher..." If an application is installed, the Create Launcher dialog will fill in the fields for you automatically. For example, I have Audacious installed, under "Name" I start to type "Aud" and a dropdown list pops up, I choose Audacious and the rest of the fields are automatically filled...
Name: Audacious
Comment: Listen to music
Command: audacious2 %U
Icon: a

For the panel, right-click, choose "Add New Item", choose "Launcher". You'll have to fill the fields in manually but you can create a desktop launcher first as an example, change the list of icons to "Application Icons" from the "Select icon from" dropdown list in the icon field (click the existing icon to open the dialog).

HTH
