### Post by racepics on 2011-12-08
Is anyone successfully printing high quality inkjet photos on Ubuntu yet?
I've been using Ubuntu since versions 7.xx and have never been able to get remotely good photos printed. In fact the output would be laughable if it wasn't so sad. So much so that I have a windowsXP64 system specifically to run my photo printers. I run a photographic website and sell prints so they need to be good quality.
 
Whenever I've tried to print photos from Ubuntu the results have been horrible. Terrible color, lines, grainy etc. Not to mention all over the page as far as positioning goes.
I've searched this forum and tried various recommended apps (Gnome photo printer, picassa, photoprint, gimp, gthumb et-al) but none of them even come close to a reasonable output. 

I tried the Turboprint driver and that at least got the color and quality right, but borderless prints were impossible and bordered ones had uneven borders no matter how much I played with the settings and wasted consumables.

I tried to run the Canon EasyPrint software via wine but it wont install.

Its a great, simple, fast piece of software. I get a screen of thumbs, click on the ones I want to print (two of that one, three of this one etc) then choose paper type and size, layout, (one per sheet, two per sheet etc ) then get a chance to crop (proportionally) and edit colour. Then hit the print button. Perfect every time. 

If only canon would make a linux version.
