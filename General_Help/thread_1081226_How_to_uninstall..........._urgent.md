### Post by zachthejones on 2009-02-26
bigger question - did you install ubuntu from within windows (it's called the wubi installer)? If so, I believe you ca uninstall from within Windows like a normal program.

If you installed ubuntu outside of windows (a classic dual-boot situation), you have to use the windows install disc to restore the original boot sequence. Once you do that you can write over or format the partition ubuntu was using.

(note: a normal wubi [in windows] installation does not use a partition)
