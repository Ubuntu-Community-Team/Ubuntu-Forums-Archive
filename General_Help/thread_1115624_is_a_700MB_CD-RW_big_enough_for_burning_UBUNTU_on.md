### Post by hikaricore on 2009-04-04
All the proper CD iso files should fit on a 700mb disc.

If you're running any of the daily or beta releases of Jaunty they may be oversized from time to time.
You can still fit them on a 700mb CD-R by enabling the overburn option in most cases.

All fully released CD versions of Ubuntu (including Xubuntu, Kubuntu) will run on 700Mb CDs. Unreleased Alpha and Beta versions of Jaunty 9.04 will in most cases burn to these CDs (I believe that Jaunty releases on 25th April).
Be prepared to download a large number of updates once you've installed as *buntu is a proactive operating system especially when it comes to security issues.
