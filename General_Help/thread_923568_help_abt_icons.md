### Post by mcduck on 2008-09-18
If you want just icons to start programs you can simply drag applications from the menu into the desktopn(or right-click them and choose to create a luncher on desktop).

If you want Computer, Documents, Network and Trash icons on the desktop here's how to do it:

1. hit Alt-F2 and run "gconf-editor"
2. Browse to apps/nautilus/desktop
3. enable whatever icons you want. You can also change their names in the same place.

The network connection thing depends a lot on how you are connecting to Internet. For most situations I think just using the Network Manager's icon in the notification area would be the easiest way of connecting/disconnecting Internet.
