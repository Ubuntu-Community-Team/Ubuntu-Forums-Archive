### Post by DoctorMO on 2008-03-25
Sounds like a colour profile problem. When ever a document is sent to a printer it must convert between what the computer says is the colour for display and what the printer will want to print in order for them to look the same to a human.

I've never managed to fix a colour correction issue in linux so maybe someone else can help.
