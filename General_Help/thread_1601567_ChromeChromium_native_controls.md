### Post by MadMonkey234 on 2010-10-20
Is there a way to make Chrome/Chromium render web pages with native controls like Firefox does? Take a look at input or button elements and they really look ugly. If it's not possible, is there a bug report for that?

Thanks!

---

### Post by Brandon Williams on 2010-10-22
If you use a gtk-based window manager (e.g. gnome or xfce), then yes. In the preferences dialog, go to the 'personal stuff' tab. Under the 'Appearance' heading, you can select to 'Use GTK+ Theme' and 'Use System Tile Bar and Borders'.
