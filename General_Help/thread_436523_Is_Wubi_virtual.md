### Post by tuxcantfly on 2007-05-08
> Is this Wubi, basically a virtual linux that runs under windows?

NO. See the site FAQ at [http://cutlersoftware.com/ubuntusetup/wubi/en-US/faq.html](http://cutlersoftware.com/ubuntusetup/wubi/en-US/faq.html) and if you need the nitty-gritty details, check out [https://wiki.ubuntu.com/install.exe](https://wiki.ubuntu.com/install.exe)

> Is it safe to run?

It should be just as safe, or safer (no partitioning or bootloader tweaking) than standard ubuntu

> I want to install Kubuntu, but I do not know how to partition a drive yet and am thinking about using Wubi.

It installs Kubuntu fine, I'd say go for it, worst thing that could happen is it doesn't work properly, you run the uninstaller, and it undoes everything
