### Post by Impavidus on 2013-04-06
.exe files may be send to the archive manager because some .exes are auto-extracting archives. They won't auto-extract on Linux, but the archive manager can handle them. In general, however, Linux cannot handle .exe files. Some may run in wine (check the database at [http://appdb.winehq.org/](http://appdb.winehq.org/) for a list of applications known to work in wine), but the best way is to find a Linux alternative: [http://www.linuxalt.com/](http://www.linuxalt.com/) or ask here.
