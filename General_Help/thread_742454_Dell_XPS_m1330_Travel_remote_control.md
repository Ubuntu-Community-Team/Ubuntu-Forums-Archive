### Post by bryanchicken on 2008-04-02
for anyone else that is interested i found this on the dellcommunity forums (thanks ringerc)

Volume Up, Volume Down, Mute: Volume Up & Down & Mute media keys
Black arrow buttons:                       Keyboard "Inverted T" arrow keys
Silver up and down arrows:                 Page up and down keys
|< and >| buttons :                  |<< and >>| media keys
Stop button, Play/Pause Button:            Stop and Play/Pause Media keys
Back button (silver left arrow):           Backspace key (NOT delete, backspace)
<< and >> keys:                Control-Shift-B and Control-Shift-F respectively


Not had a chance to test it yet.
