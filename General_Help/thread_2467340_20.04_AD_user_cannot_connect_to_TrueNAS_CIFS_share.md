### Post by TheFu on 2021-09-23
Use NFS, not CIFS to mount shared storage between Unix systems.  NFSv4 does system-to-system Kerberos authentication and has better performance and allows native, POSIX, file permission controls.

Please don't post the same question 5 times in a few minutes.
