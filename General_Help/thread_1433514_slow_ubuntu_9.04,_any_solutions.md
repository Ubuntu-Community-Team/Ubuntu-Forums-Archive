### Post by N92 on 2010-03-19
If you want to boot up faster you can go to your toolbar System - Preferences - Startup Applications  and remove programs that you do not need to start when you boot.

You can also Go to Ubuntu software centre and remove programs that you never use.

I do not recommend that you use any computer clean up software as you can accidentally remove important packages.
