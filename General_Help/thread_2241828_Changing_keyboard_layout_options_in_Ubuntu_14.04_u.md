### Post by grahammechanical on 2014-08-28
For the record, In 14.04 we now have something that we never had in 12.04. A keyboard app indicator icon in the top panel. Click on that and select Text Entry Settings and we get the dialog to add or remove keyboard layouts. We can also go to System Settings>Keyboard and at the bottom left of the dialog are the words "Text Entry." Click and we get the same add/remove keyboard layouts dialog.

You might find Unity Tweak tool useful for Ubuntu Unity.

Regards.
