### Post by raja on 2007-04-22
If you want vmware player, I think it is in the repos and all you may have to do is ```
sudo aptitude install vmware-player
``` You will not be able to create machines or use multiple machines with the player. But it is easy to create machines at [this site]("http://www.easyvmx.com/"). You may also want to look at virtualbox, which is an open source virtualization solution.

### Post by strabes on 2007-04-23
You can get to it from Applications --> Accessories --> Terminal. Then in there, type this: ```
sudo aptitude install vmware-player
```

Check out this page for more info: [http://www.psychocats.net/ubuntu/terminal](http://www.psychocats.net/ubuntu/terminal)
