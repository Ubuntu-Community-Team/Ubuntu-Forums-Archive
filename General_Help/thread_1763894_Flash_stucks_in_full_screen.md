### Post by linuxinstalledfromhdd on 2011-05-21
Don't forget.. 

In the Firefox browser click Tools >> Add-ons >> Flash Aid >> Preferences >> Execute

If that doesn't work, click Installation Options and try the Stable version of Flash Plugin again..  in expert mode and then click execute. 

That should do it.
