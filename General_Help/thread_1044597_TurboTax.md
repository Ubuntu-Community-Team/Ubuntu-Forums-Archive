### Post by yther on 2009-01-19
I have successfully used TurboTax Online in years past, with both Firefox and Konqueror.  Make sure you permit cookies, JavaScript, and possibly Java for that site.  If using a non-Firefox browser, possibly changing the browser ID might help (for instance, Konqueror has a setting where it will tell the remote web site that it is Firefox, or Internet Explorer) if the site is improperly testing what browser you have, rather than what technologies your browser supports.  This is quite common.

Now, having said that, there's always a chance that Intuit changed their online product this year.  I haven't tried to use it yet.
