### Post by claracc on 2013-07-13
Doing a little reasearch, I have found this link: [http://www.aboutlinux.info/2011/01/cover-thumbnailer-how-to-install-in.html](http://www.aboutlinux.info/2011/01/cover-thumbnailer-how-to-install-in.html) where it is explained how to install the cover thumbnailes program, but as you can see, the ppa from which install the software is not available for the ubuntu 13.04 release: [https://launchpad.net/~flozz/+archive/flozz/+packages](https://launchpad.net/~flozz/+archive/flozz/+packages)

So, I would downloas the source package from the website: [http://projects.flogisoft.com/cover-thumbnailer/download/](http://projects.flogisoft.com/cover-thumbnailer/download/) and try to compile for your system (first extract, and then compile).

More help about extracting and compiling in this thread: [http://ubuntuforums.org/showthread.php?t=862141](http://ubuntuforums.org/showthread.php?t=862141)
