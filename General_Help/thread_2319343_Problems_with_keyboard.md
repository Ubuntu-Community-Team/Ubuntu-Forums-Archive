### Post by coldraven on 2016-04-10
Open the Dash by pressing  the Super (Windows) key or by clicking on it in the launcher bar. Then start typing "Text Entry" and you will see it's icon appear. In the Text Entry make sure that the keyboard layout that you want is top of the list.  You can also check the box "Show current input method in the menu bar" to see a keyboard switcher icon.
