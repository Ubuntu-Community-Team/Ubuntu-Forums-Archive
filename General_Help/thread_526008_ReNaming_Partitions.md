### Post by meierfra on 2007-08-15
Create a directory in /media with a name of your choice:

```
sudo mkdir /media/name_of_your_choice
```

open the file /etc/fstab:

```
sudo gedit /etc/fstab
```

in the second to last line change

/media/sda1 to /media/name_of_your_choice

(do NOT change /dev/sda1)

Save the file and reboot.
