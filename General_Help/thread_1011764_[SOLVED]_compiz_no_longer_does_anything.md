### Post by Forlong on 2008-12-15
Remove this part completely:
```

	SubSection "Display"
		Virtual	2432 864
	EndSubSection

```
In oder to do so, you will have to open the file with root privileges, e.g. like this:
```
gksu gedit /etc/X11/xorg.conf
```

After you have saved the file and closed all applications, hit [Ctrl]+[Alt]+[Backspace] to restart your X server.

Then try again.
