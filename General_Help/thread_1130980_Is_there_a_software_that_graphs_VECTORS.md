### Post by ostrm on 2009-04-20
**gnuplot** is in the repositories and allows plots of 2d vector fields. Here some demo: [http://gnuplot.sourceforge.net/demo_4.2/vector.html](http://gnuplot.sourceforge.net/demo_4.2/vector.html)

There is another tool, **ImLib3D**, a C++ library together with a viewer, that also supports plots of 3D vector fields and OpenGL output. It's not in the repositories and the learning curve is steeper, however.

If you are looking for commercial easy to use software, I'd suggest Mathematica.
