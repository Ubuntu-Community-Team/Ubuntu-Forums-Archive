### Post by jgw on 2018-04-25
I have the following setup and the following hooked together.  I have a receiver (denon avr-s530bt - kavari), a computer providing 4k display (amd a8-7650K w/radeon R7 graphics - kavari) on a samsung 65" tv.  The receiver has 2 hdmi inputs, 1 for the computer and the other for a dish network box.  The network box works just fine.  The computer, however, has problems and I need to deal with font sizes and the sizes of the top bar and the unity launcher bar on the right.  I have found that I can deal with the general fonts using ubuntu tweak or unity tweak tool.  Anyway, the computer sends a resolution of  3840 × 2160   Then the fun starts.  I have now set all the default fonts to 25 which worked.  Then I set the top bar using system settings/displays/scale for menu and title bars and that worked too.  BUT, when I go from the computer input, to the dish input, and then back to the computer input, it loses the settings that were set before I changed to dish input that were set by system/settings/displays/scale for menu and title bars.  I have no idea how to make it permanent.  

I should add that this does NOT happen when I re-boot, only when I change the receiver inputs.  All fonts that I have changed, however, stay changed no matter what.
I have also found out that when you set the /system/settings/displays/scale for menu over 3 then the fonts get very small instead of bigger

So, does anybody know how I can permanently set both the top bar and the unity launcher?

Thank you.................

### Post by jgw on 2018-04-29
I am going to mark this solved.  I have found that if I leave the /system/settings/displays/scale open and running and minimized the scale remains the same when switching from the tv to the computer and back again.  This being the case the only problem is  when the system gets re-booted which is seldom and that seems to be the way it is, at least for now, so I am a happy camper.
