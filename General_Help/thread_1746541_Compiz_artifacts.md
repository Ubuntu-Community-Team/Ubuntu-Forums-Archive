### Post by xinix on 2011-05-01
I just upgraded to Natty and encountered a problem with Compiz creating artifacts around windows.  This happens only if I have the Blur plugin enabled and set for "Normal" windows.  Basically the shadow of the window artifacts and shows stuff from the desktop or anything behind another window when there is another window behind the focussed one. I'm attaching a screenshot since my description is not clear. 

I like to use transparency for menus and my terminal screen but I also like for the portion behind these areas to be blurred so that any text in the window/menu is more legible.  Disabling alpha blur for "Normal" windows also disables it for the terminal.  This is solved by including "class=Gnome-terminal " in the "Alpha Blur Windows" setting and removing "Normal".

I just wanted to share this in case anyone else has the same problem.
