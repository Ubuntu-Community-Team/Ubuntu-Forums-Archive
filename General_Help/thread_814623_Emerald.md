### Post by Jon11393 on 2008-05-31
I've been using >  emerald --replace  to activate emerald and it would be nice if I didn't have to type it in every time I restarted my computer!
Is there a way I could automate this? Thanks!

---

### Post by PinkFloyd102489 on 2008-05-31
System -> Preferences -> Sessions

Add a new entry into the first tab named Emerald and put the replace command under the command bit. Voila, autostart.

---

### Post by NikoC on 2008-06-01
Or:
System > Preferences > Advanced Desktop Effects Settings > Window Decoration
copy/paste the following line into the command box:
emerald --replace
