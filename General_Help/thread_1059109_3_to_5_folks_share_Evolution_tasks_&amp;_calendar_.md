__________________________
"Mung" -- transitive verb and ancient text editor command; acronym M.U.N.G. meaning "mung until not good." The TECO editor could accomplish script driven file modifications. The "mung" command launched teco without any user interface to operate as a text file filter. A frequent mistake involved running a modification script against a data file that had already been modified, hence the recursive acronym. Pronounciation rhymes with "hung" not "grunge."
