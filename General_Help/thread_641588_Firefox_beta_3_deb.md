### Post by useResa on 2007-12-15
The firefox 3.0 packages are available in the Universe repositories (see attachment).

More information about the package can also be found [here]("http://packages.ubuntu.com/gutsy/web/firefox-3.0").
The bottom of the page gives access to the download locations for the deb file.
For example for the i386 architecture you can download the deb file from [here]("http://packages.ubuntu.com/cgi-bin/download.pl?arch=i386&file=pool%2Funiverse%2Ff%2Ffirefox-3.0%2Ffirefox-3.0_3.0%7Ealpha8%2Bnobinonly-0ubuntu1_i386.deb&md5sum=a6aa26f09be53c83505810b3dbb0fccf&arch=i386&type=main").
