### Post by Spyderkid on 2010-12-02
Okay it probably won't work very well with iTunes but give it a go anyway no harm in trying.

Ok...

>go to the software centre download wine
>go to iTunes and download iTunes for windows make sure to **_save_** the file somewhere like downloads or the home folder
>find the file then right click and go to **properties** then   **permissions** then check the box which says *"allow executing as a file"*
>then right click on the file and do run with wine program loader (or something similar to that anything with wine in the name should do) 
>it should install and you'll find it under applications > wine > programs
