### Post by GreatDanton on 2012-07-15
If you are on Linux then you should try startup disk creator. Select the Iso file and the installer will take care of it.

If you are on Windows, I think the best option would be to use Unetbootin (works on Linux as well).

Hope this helps

### Post by OM55 on 2012-07-15
+1 here.

If you are in Linux already, just download the ISO file from Ubuntu.com and then either use "Start Disk Creator" or "unetbootin" to create the bootable USB media.

If you are not on Linux, you will have to find and use a similar Windows based USB bootable media creator, for example: WiNToBootic ([http://www.wintobootic.com]("http://www.wintobootic.com/"))
