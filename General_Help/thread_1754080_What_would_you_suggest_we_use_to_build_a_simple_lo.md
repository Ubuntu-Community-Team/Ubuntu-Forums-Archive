### Post by keithpeter on 2011-05-09
Hello greavette

I'd suggest trying either 

seamonkey composer as the editor with the 'site manager' set up so that the pages can save back to your server

or

running a WordPress instance on the server and your daughter updating 'pages' (i.e. whole pages with separate linking rather than 'posts') via the web browser using the wysiwig-ish html editor that is in Wordpress. There are some nice Wordpress themes out there.

The fact that the audience is invitation only covers the wordpress security angle.

Sounds fun...
