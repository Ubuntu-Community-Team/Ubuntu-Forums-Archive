### Post by DuckHook on 2018-10-27
I am not surprised if there are still bugs in 18.10. It was just released a few days ago. If you want stability, stick with 18.04. If you must have the newest driver for your devices, or if you are testing, or you need the latest kernel/libraries for a must-use app, then you may have no choice but 18.10.

I don't use 18.10, so am of limited help. But you can try launching Software Center from the command line to see the error codes that are generated. If this is a fresh install, and the bug persists, then those errors will help you report the bug to Launchpad, where the developers can triage the problem and ask you for further info and help in testing.
