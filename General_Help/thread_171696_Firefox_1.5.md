### Post by glotz on 2006-05-07
Did you uninstall the existing Firefox version prior to installing Firefox 1.5? Uninstalling the old version causes some issues. I think Ubuntu 5.10 needs a Firefox 1.0.x series version for some reason. You could try installing Firefox 1.5 also, just forget about the old version. At least, that's what I did. Here's instructions [https://wiki.ubuntu.com/FirefoxNewVersion](https://wiki.ubuntu.com/FirefoxNewVersion)
