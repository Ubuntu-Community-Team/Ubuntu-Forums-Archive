### Post by hw-tph on 2006-08-27
Use **ls** to see what is in the current directory. The dot in the **./configure** command means "this directory" (it literally translates to "execute the file 'configure' in this directory") so you will need to make sure there actually is a file called "configure" in it.

Usually, there is a text file called INSTALL in the same directory, and often a README file as well. Read those for package specific instructions.


Håkan
