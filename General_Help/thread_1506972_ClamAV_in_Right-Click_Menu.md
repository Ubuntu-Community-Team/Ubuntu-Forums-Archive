### Post by Ubuntist on 2010-06-11
In a previous installation I was able to do a ClamAV virus scan on a file from the right-click menu.  In my new Lucid installation, I've installed every ClamAV package from the Ubuntu Software Center that seems relevant, but ClamAV fails to appear in the right-click menu.  Any ideas on how to get ClamAV as a right-click option?

---

### Post by todak on 2010-06-11
Did you install the package *nautilus-clamscan*? That will install a context-menu entry for scanning files. The menu entry will be *Scan for viruses...*

---

### Post by Ubuntist on 2010-06-11
No, I hadn't installed natuilus-clamscan.  Doing so did the trick!  Thanks, todak!!!
