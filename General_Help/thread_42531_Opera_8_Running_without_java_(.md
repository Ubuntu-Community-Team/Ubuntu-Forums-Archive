### Post by Xian on 2005-06-18
I'm not at my Linux box right now, so I can't be precise but....
From the main menu goto:

Tools > Preferences > Advanced > Content 

Then there is another button where you can set and validate the java path.
You just need to point that to your java install directory and it will do the rest.

I think you have to go about two or three folders deep into that directory.
Look for something like ~/lib/i386....

### Post by wvslkr on 2005-06-18
