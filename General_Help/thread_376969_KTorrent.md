### Post by gejr on 2007-03-05
Won't make a big deal out of this, but I've had trouble finding a suitable torrent client for linux.


I've tried these:

Azureus - Slow in my experience, buggy and ugly, but has most of the functions I need. When it doesn't crash.

BitTornado - Default bittorrent client in Ubuntu. No extra functions, can't even figure out how to resume a download.

Deluge - Never managed to get it working.

qBittorrent - Fairly good but too few features and it seems to crash on me once in a while.

uTorrent - I don't like running a torrent client through Wine..too much hazzle.

Ktorrent - Don't know why I haven't tried this before. Probably because I run Gnome and it's meant to be a KDE app. But I did a simple "apt-get install ktorrent" today and i'm stunned. It's better than any of the others even not being meant to run with Gnome. 

So this is my little tip for today...If you're used to good clients like windows' "BitComet" (like me) go try ktorrent out! :)
