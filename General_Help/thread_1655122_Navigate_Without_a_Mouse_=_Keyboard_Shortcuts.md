### Post by audiomick on 2010-12-29
Alt + F1 should give you the applications menu.

Look at
System> Preferences> Key Combinations
for more inspiration... ;)

Oh, it might be called something a little different to that. Mine is in German. "Tastenkombinationen".

### Post by tredegar on 2010-12-29
Try System, Preferences, Keyboard, Mouse Keys, and check "Pointer can be controlled using the keypad"
