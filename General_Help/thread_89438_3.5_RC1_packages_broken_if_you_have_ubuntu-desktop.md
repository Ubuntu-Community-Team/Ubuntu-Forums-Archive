### Post by kaarlov on 2005-11-13
I had a same problem too.
```
sudo dpkg -i --force-overwrite /var/cache/apt/archives/kdelibs-data_4%3a3.5-rc1-0ubuntu0breezy1_all.deb

```did the trick for me. It didn't want to overwrite any other files.
