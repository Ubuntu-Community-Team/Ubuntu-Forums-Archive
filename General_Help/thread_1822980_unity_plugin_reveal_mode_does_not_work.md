### Post by MARP1961 on 2011-08-16
Yes, I'm sorry to disappoint you; but I also thought this plugin was for moving the Unity Launcher. Apparently not! It can only stay on the left and is not designed to move.

What the plugin does is to change where on the screen you need to point the mouse in order to reveal the (still on the left) launcher. Watch this short demonstration video: [http://www.webupd8.org/2011/04/unity-386-brings-option-to-configure.html]("http://www.webupd8.org/2011/04/unity-386-brings-option-to-configure.html")

Wait for version 11.10 in October and install 'Gnome Shell' from Synaptic Package Manager or Ubuntu Software Centre. Doing this will give you a choice of desktop at login. Gnome Shell gives a much better desktop experience in my opinion and will be easier to customize than Unity.
