### Post by 3rdalbum on 2009-11-30
You might feel more at home with KDE; its default file manager is Dolphin, which (once you set it back to "double-click to open" mode) is closer to Windows XP's file manager than Nautilus.

You can install KDE on Ubuntu:

```
sudo apt-get install kdebase-workspace
```
