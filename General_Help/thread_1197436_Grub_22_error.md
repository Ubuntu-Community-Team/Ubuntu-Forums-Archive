From: [http://www.gnu.org/software/grub/manual/grub.html#Troubleshooting](http://www.gnu.org/software/grub/manual/grub.html#Troubleshooting)

"Error 22 : No such partition
    This error is returned if a partition is requested in the device part of a device- or full file name which isn't on the selected disk".

You could try downloading SuperGrub and use this to try and fix your problem, if you are sure you have Ubuntu installed somewhere on your system.

SuperGrub link: [http://www.supergrubdisk.org/](http://www.supergrubdisk.org/)
