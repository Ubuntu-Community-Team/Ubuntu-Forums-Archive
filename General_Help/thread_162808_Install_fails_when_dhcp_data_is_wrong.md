### Post by pacogozalez on 2006-04-19
This is what i did notice installing ubuntu:

The installation will fail when there is an erratic  DHCP server on the lan. Some error message is shown, and the GUI doesent start. CLI mode works.

There is a stray DHCP server on my lan which will give the wrong gateway IP. 
Since i never used dhcp i did not care about, and its good to keep my neighbors out of my unsecured wifi, but it breaks the ubuntu install.

Work-around: 
unplug the lan connector until ubuntu install lets you configure the lan settings manually.
