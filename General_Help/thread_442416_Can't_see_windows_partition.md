### Post by merlinus on 2007-05-13
I think that the problem is no UUID for sda1 and sda2 in /etc/fstab.

First backup fstab, and then try this.

In a terminal:

 sudo vol_id -u /dev/sda1

That will give you the UUID for sda1.  Do the same for sda2.

Then enter those UUIDs in the appropriate places in fstab and reboot.

Here is part of my fstab as an example.

```

# Entry for /dev/sda1 :
UUID=1EB88F62B88F36F5 /media/sda1 ntfs-3g defaults,locale=en_US.UTF-8 0 1

```

-merlin
