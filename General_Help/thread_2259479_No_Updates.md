I see updates for trusty sitting in the proposed repo.

I rarely ever get updates for precise, aside from ppa updates, and the rare zero-day update here and there.(I think firefox,tbird and the kernel are the only official packages that get routine updates for me, now)
Kernel updates on precise are rarer and rarer these days, and usually they come in spats, where as the initial kernel update comes and then for the next week or so the fixes for all the regressions the new kernel brought come through.

If you can install packages then I would think the repos are up and running...
No updates is not the end of the world, it's simply a sign that things are working well, or certain packages still need to be tested before releasing them, which may or may not tie down other packages.
