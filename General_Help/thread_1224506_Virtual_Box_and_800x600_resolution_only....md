### Post by lukeiamyourfather on 2009-07-27
Install the guest addons if you haven't yet (see the VirtualBox help for the full instructions). That should allow the guest and host to communicate so when you resize the window or go full screen the guest will automatically change the virtual screen resolution to match. Cheers!
