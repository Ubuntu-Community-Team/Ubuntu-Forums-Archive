### Post by Impavidus on 2017-07-07
For most applications on Ubuntu, the version is frozen when Ubuntu is released. There are only some patches to fix bugs. That's the price to pay for stability. You could however install a PPA for play on linux, which should make the new version available through synaptic &co. I don't use play on linux/wine myself, but I understand some guides on the web telling you how to install the PPA are obsolete, so make sure the information is recent. Have a look at our wine subforum: [https://ubuntuforums.org/forumdisplay.php?f=313](https://ubuntuforums.org/forumdisplay.php?f=313)
