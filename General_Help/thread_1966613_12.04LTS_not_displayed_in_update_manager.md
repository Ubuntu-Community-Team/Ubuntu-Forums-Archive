Try running ```
update-manager -d
``` in a terminal.

Beware that the servers are running slowly,so it might be a good idea to wait a while to let things settle down before doing the upgrade.10.04 is still going to be supported for another year.

Or you could download the Alternate CD and use that to run the upgrade and you have a way of doing a clean install of 12.04 if the upgrade has a problem.

Always best to be prepared for any eventuality.

Good Luck
