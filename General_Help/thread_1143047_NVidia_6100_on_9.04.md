### Post by 3rdalbum on 2009-06-29
> **kastanedowski said:**
> 
Should I install the drivers in the producers pages for all these hardware??  or I can work as it is now??

If your graphics work to your satisfaction, then you don't need to install any drivers.

If you want 3D acceleration, use the Hardware Drivers program (System > Administration > Hardware Drivers) to install the proprietary 3D driver for your graphics card.

> I can read that even there are special drivers for linux for my processor

Not true. Drivers help the operating system to know how to talk to a device. The operating system is running on the CPU, not talking to it. If you're not already using the 64-bit version of Ubuntu, you should consider installing it when 9.10 comes out as it will be optimised for 64-bit CPUs such as yours.
