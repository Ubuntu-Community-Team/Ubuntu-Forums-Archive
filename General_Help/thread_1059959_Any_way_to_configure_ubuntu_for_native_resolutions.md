### Post by snowpine on 2009-02-09
Hi Adam,

If a window doesn't fit on the eee's screen, here's what you can do: Hold down the Alt key, then double-click somewhere on the window and drag to move the window around. This is very useful if the Cancel/OK buttons are cut off at the bottom of the screen. :)

I did not have any trouble with Ubuntu automatically recognizing the 1024x600 resolution on my eee 900ha, sorry I can't help you there.

> Recommended minimum requirements

Ubuntu should run reasonably well on a computer with the following minimum hardware specification. However, features such as visual effects may not run smoothly.

    * 384 MB of system memory (RAM)
    * 8 GB of disk space
    * Graphics card capable of 1024x768 resolution
