### Post by howefield on 2009-02-14
A combination of ...

1. General options and select the number of desktops you require. 
2. Activate Desktop Cube and Rotate Cube.
3. Go to "cube reflection and deformation". Set as required. 
4. Spin the cube by pressing mouse scroll wheel and drag.

### Post by phantom3113 on 2009-02-14
Howefield is correct, although some specification might be needed on some points:
1-under general options, you'll need to alter the "virtual horizontal desktops"
4-there are different bindings for these actions. scrolling the mouse wheel does rotate it, but if you wanted to control the rotation, you'll have to check the key/mouse bindings.
