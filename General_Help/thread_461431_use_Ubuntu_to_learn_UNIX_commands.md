### Post by GrueTamer on 2007-06-01
It might.  I wouldn't say that Ubuntu is the best for learning Unix, but if you lock yourself in a place with no graphical tools to help you, I'd say that you could learn a lot of Unix commands.  But remember...

Linux was made to be LIKE Unix.
BSD was made to BE Unix.

BSD is probably better for learning Unix itself, because it's closer to Unix than Linux is.  This isn't a knock against Linux, as I do love what Linux is like, it's just a fact.
