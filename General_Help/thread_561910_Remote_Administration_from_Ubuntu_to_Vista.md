### Post by dad311 on 2007-09-28
If Vista has "Remote Desktop" installed, you can use "Terminal Server Client" which comes installed by default in Ubuntu.

If Vista does not have "Remote Desktop"  I would use VNC".


Also, to use either service, you might need to configure the router/firewall @ you mums to allow the connection.
