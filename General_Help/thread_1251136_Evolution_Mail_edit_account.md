### Post by howefield on 2009-08-27
> **rmcellig said:**
> I would like to know how to setup Gmail IMAP with Evolution Mail?

Also, once I have my Evolution mail account set up, how can I edit it?

You need to enable IMAP on your gmail account settings page, then open up Evolution and select "Preferences" from the "Edit" menu. This will start the new account wizard where you can enter the relevant info.

Tutorial here

[http://wazem.blogspot.com/2007/10/hot-to-configure-imap-on-evolution-and.html](http://wazem.blogspot.com/2007/10/hot-to-configure-imap-on-evolution-and.html)

Once I have my Gmail set up in Evolution Mail, is it possible to use the Gmail contacts I already have set up in Gmail?

---

### Post by howefield on 2009-08-27
Log in to your gmail account and click on contacts, then the export link, check the vcard format option then save the file to your desktop.

Evolution > File > Import function to import a single file to load the saved .vcf file into your evolution contacts.
