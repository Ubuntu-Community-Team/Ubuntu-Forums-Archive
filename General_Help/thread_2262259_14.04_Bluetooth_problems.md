### Post by jeremy31 on 2015-01-31
Check dmesg for errors ```
dmesg | grep -e luetooth -e btusb
```  I have a GBU bluetooth dongle that will pair and work with a headset(Philips SHB4000)
But I did create a file that may have helped```
gksudo gedit /etc/modprobe.d/btusb.conf
``` and enter ```
install usb:v0A5Cp21E8d*dc*dsc*dp*ic*isc*ip* /sbin/modprobe btusb; echo 0a5c 21e8 > /sys/bus/usb/drivers/btusb/new_id
```
Save, exit, and reboot
