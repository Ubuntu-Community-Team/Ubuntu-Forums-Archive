### Post by geraldm on 2007-11-10
Developers do not read this forum.  Please do not scold them here.
Printing is one area that does get attention before a release.  When the release comes,
certainly some printers are working within it.  There are so many different printers
that not all are going to be tested, and for some, unexpected quirks can occur. 

Try the install setup for the printers, then look for any bug reports, trying to find any
that provide the details such as you have discovered.  If you do not find any appropriate
report, post with enough detail that a reply could be helpful to you and others.
