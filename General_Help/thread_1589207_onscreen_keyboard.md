### Post by bouncingwilf on 2010-10-06
Morning all, just a quick one, the keyboard on my boat computer has died ( Pentium duo running 10.04) but it's not a major issue as a) Ubuntu doesn't seem to mind and starts quite happily without it and b) I only use it once in a blue moon. However, as you can imagine,  there are a few occasions when it's required.

Now I'd prefer to do without it and use "Onboard" ( for one thing the keyboard on the boat seems to distort the boat compass) but the question is how do you start Onboard without an existing keyboard? (usually Alt-F2 Onboard).

I'm sure I could knock up some sort of script and associate it with a desktop icon but it would be much more elegant if I could add it to the applets panel - does anybody please have any useful suggestions/ideas about how to go about this?

Bouncingwilf

---

### Post by kerry_s on 2010-10-06
in your file manager, go to /usr/share/applications
drag "onboard" to your panel.

---

### Post by bouncingwilf on 2010-10-06
Many thanks - I found the best way was to go System-> preferences-> main menu
add Onboard to the main menu; right click it and added launcher to the panel.


Bouncingwilf
