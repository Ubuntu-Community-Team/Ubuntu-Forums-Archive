### Post by themusicalduck on 2009-05-19
Go to System > Preferences > Sound

On the Sounds tab, uncheck Play alerts and sound effects

That should disable everything.

EDIT: Just noticed you're using Kubuntu.. That will work for gnome but not sure for KDE, it might be in a similar place though.

### Post by Chris Musampa on 2009-05-19
For KDE users System Settings > Notifications, either set them up per application or go to the Player Settings tab and turn the whole lot off.
