### Post by monkeybrain20122 on 2023-02-01
I think qpdfview is by far the best pdf reader on Linux. It is light weight super fast and allows basically unlimited tabs (the tab feature is a must have for me, therefore evince is practically useless) It also has a lot of keyboard shortcuts which are what you are looking for.

[https://manual.lubuntu.me/stable/2/2.2/2.2.6/qpdfview.html](https://manual.lubuntu.me/stable/2/2.2/2.2.6/qpdfview.html)

The version in the repo is old and probably buggy, get it from the developer's [ppa]("https://launchpad.net/~adamreichold/+archive/ubuntu/qpdfview-dailydeb")

Many people recommend Okular, but I find it bloated and buggy, it allows tabs too but it seems that it doesn't restore opened tabs on restart. You also have to install a lot of kde junks unless you are on Kubuntu where they are already installed.
