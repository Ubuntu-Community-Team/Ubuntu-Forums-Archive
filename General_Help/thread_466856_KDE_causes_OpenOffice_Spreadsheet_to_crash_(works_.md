### Post by carolinason on 2007-06-07
Impress is the Presentation program in OOo. Actually now that I notice it, Ubuntu does not call it Impress it's just called presentation.

Anyway, F5 should start the Slide Show in the Presentation program and not start a new document, which is Control-n.

It sounds like a key mapping issue. Try using the menu's to see if this will give you the correct function, then we can try 'hot keys'.
