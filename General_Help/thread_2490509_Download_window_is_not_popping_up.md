### Post by grahammechanical on 2023-09-06
The default folder into which downloaded file normally go is called Downloads. It is the default location in Firefox. To change the default download location in Firefox click on the hamburger icon (three horizontal lines in the top right corner of the Firefox Window. Select Settings. Then scroll down to Files and Applications. There you can change the default folder for downloads. 

Some web sites that offer files to download do offer a dialog where we can choose which folder to save the file in. For web mail you may need to choose a Save As option. With Thunderbird there is also a save as option where we can choose the folder.
