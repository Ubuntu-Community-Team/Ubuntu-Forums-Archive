### Post by Aoshi99 on 2009-01-19
I tried to convert a mixed-mode MDF into a ISO file using mdf2iso. But if I do so, the music session disappears. I suppose that ISOs can't contain both data and audio. If that's true, I would like to know if there is a way to mount MDFs in Ubuntu directly or if there is no way to do so, how to convert the MDF into another format that supports mixed-mode.

Okay, I've found an answer myself:
All you need are PowerISO and Wine. MDF files can be converted into BIN/CUE files. And these files can be directly mounted in DOSBox. So if you want to play old DOS games which use mixed-mode cds that's the way to go! ;)
