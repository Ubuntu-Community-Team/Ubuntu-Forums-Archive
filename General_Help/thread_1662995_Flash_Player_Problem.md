### Post by lovinglinux on 2011-01-09
Get [Flash-Aid]("https://addons.mozilla.org/en-US/firefox/addon/161939/") and run it. It will detect installed flash plugins, remove them and install the best option according to your system architecture and version. 

Chrome will also detect the plugin.
