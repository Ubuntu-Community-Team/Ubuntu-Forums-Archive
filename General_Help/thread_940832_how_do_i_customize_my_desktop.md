### Post by WaeV on 2008-10-07
The image doesn't need to be with the other wallpaper files to be used as a wallpaper.  Simply right-click the desktop and select 'Change Desktop Background', then navigate to your image.

As for building a theme, I don't know, but you can customize existing themes in System > Preferences > Appearance under the 'Themes' tab.
