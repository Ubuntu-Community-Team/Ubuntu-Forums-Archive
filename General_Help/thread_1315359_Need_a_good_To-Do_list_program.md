I don't know if any of you use rememberthemilk.com.  It's an awesome to-do list website.  What I really like about it is when I make an entry in it, I can then put notes in for the task.  Then every note I put in for the task gets time stamped.  All other todo lists just give you one field for notes. The only thing I've found close was OTRS ticketing system and that's a little overboard.
