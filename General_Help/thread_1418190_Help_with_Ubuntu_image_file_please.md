### Post by Richard1979 on 2010-02-28
Are you sure it's a .rar?
Hint: Press ALT if you're using Windows and go to Tools > Folder Options (I can't remember the exact menu sorry), then untick "Hide file types" or something to that effect.

Your Ubuntu download is almost certainly an .iso (Winrar will associate .iso with itself BTW so it may appear to be an archive).

To write the Ubuntu disc you can probably right click on the file and select something like "write image to CD".
