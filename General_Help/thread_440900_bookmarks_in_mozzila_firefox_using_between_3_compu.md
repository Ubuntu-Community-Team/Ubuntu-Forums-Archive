### Post by CowPizza on 2007-05-12
You can find the bookmarks file buried somewhere in your user data folder on Windows and copy it over, but the easiest way to do it would be to install FoxMarks - it'll synchronize the bookmarks automatically in the future, as well.

[http://www.foxmarks.com/](http://www.foxmarks.com/)

### Post by DJiNN on 2007-05-15
I'll second that. Foxmarks is really great, and makes it a lot easier to manage Bookmarking in general. You could also use something like del.icio.us (Social Bookmarking) or Magnolia or one of any number of social bookmarking sites, to store your favourite sites etc. 

Personally, i use a combo of the two.... it's not perfect but it's a lot better than it was! :)
