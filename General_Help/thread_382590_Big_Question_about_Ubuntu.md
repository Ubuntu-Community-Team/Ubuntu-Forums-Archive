### Post by AusIV4 on 2007-03-12
> **laidback said:**
> Use the live cd to get a feel of the system, that way you don't install anything, it leaves your hd untouched.

The Live CD can never leave you completely confident of how you like Ubuntu. It's much slower than running off disk. There's some hardware that takes quite a bit of effort to get working, and you wouldn't want to reconfigure it every time you want to reboot your computer. If you don't have a lot of RAM, installing programs from repositories to see how you like them can bog down your system fairly quickly. You're not going to get a feel for media players on a Live CD, even if you do find a way to access your other drives that contain the media. You might be able to save your work if you have an external drive, but working off an external drive is hardly the same as working off a hard disk.

I think it's great that Ubuntu has a Live CD, but it's hard to be completely confident that you'll like the OS just based off your experience with the live cd. Even though I personally never intend to go back to windows, I think it would be nice if there were some program on the live CD or something that could remove Ubuntu, expand the Windows partition, and restore the MBR. People would be a lot more inclined to try it if they knew they could go back easily.
