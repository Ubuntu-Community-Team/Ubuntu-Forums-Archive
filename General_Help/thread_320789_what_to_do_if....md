### Post by ciscosurfer on 2006-12-17
You can install Ubuntu and then install the KDE apps you like.

Or

You can install Kubuntu and then install the GNOME apps you like.

It's a matter of preference and depends on which DE (desktop environment) you'd rather do the majority of work in.

Alternatively, or addition to, if you install Ubuntu, you can also install the full KDE-Ubuntu desktop (kubuntu-desktop) and access KDE via your sessions menu on your login screen.

And vice versa, if you install Kubuntu and you also want the GNOME desktop environment (ubuntu-desktop), then you can install it and access it via your sessions menu on your login screen [same as before].

Bottom line, KDE apps work in GNOME and GNOME apps work in KDE (when you download AmaroK, for example, which is a KDE app, it will pull the necessary dependencies you need to run AmaroK under GNOME...the same is true in the reverse [gnome-terminal, for example, used under KDE]
