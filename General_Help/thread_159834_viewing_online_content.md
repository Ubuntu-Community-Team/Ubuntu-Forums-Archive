### Post by xXx 0wn3d xXx on 2006-04-13
Use Automatix to install any needed things/plugins. Type the following in Terminal. Copy and paste.
> 
wget [http://beerorkid.com/automatix/automatix_5.7-3_i386.deb](http://beerorkid.com/automatix/automatix_5.7-3_i386.deb)
sudo dpkg -i automatix_5.7-3_i386.deb

Once install is done, select it from Applications > System Tools > Automatix. Install all the plugins and stuff.
