### Post by tgalati4 on 2007-02-27
This is the one area were Windows truly excels over Ubuntu.  In fact, this one feature alone would make me a permanent Windows user.  Well, until the next lockup.  

Or the next Norton Nagware.

Or the next Spyware Warning.

Or the Windows Genuine Advantage solicitation.

Or . . .?

Seriously,  with multiple desktops (I typically use eight) and lots of RAM (ram is cheap) you can have all of your favorite applications open all the time.  Each app has it's own recent document.  In preferences you can set how many recent entries to keep.

My desktops are labeled based on task:  Files, Shells, Listen, Browse, Write, Crunch, Progam, etc

So depending on what I want to do, I just go to the appropriate desktop with the applications already loaded and "open recent" will only have appropriate documents.

With this setup, I only reboot once every 3 to 6 months for a desktop and once every couple of weeks for laptop to clean up the memory.  Over time, memory leaks will cause some slowing of responsiveness.  You can customize sessions in System-->Preferences and put your startup applications with clever switches.

You can also save your default session, so it will be recreated (mostly) upon the next boot up.

With this work flow, I don't rely on system wide "recent documents"  because it is retarded.
