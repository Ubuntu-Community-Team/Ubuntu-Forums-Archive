### Post by papibe on 2012-12-11
Hi Claypool.

Try serching 'Update Manager'. There, if you go to settings, and select the 'Update' tab, you can set the 'Release upgrade' to 'Normal releases'.

After that, an upgrade to the newest release will be offered on the Update Manager itself.

Note that it may require to update the sources first (press the 'Check' button, wait for its completion, and restart the Update Manager).

Hope it helps. Let us know how it goes.
Regards.
