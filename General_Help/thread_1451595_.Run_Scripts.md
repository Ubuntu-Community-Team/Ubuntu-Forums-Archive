### Post by sisco311 on 2010-04-10
Right click on the link, then *Save Link As...* and yes, you can download it with wget as well:
```
wget ftp://ftp.idsoftware.com/idstuff/doom3/linux/doom3-linux-1.3.1.1304.x86.run
```
