### Post by zephyrcat on 2008-11-24
When you try to install software from repositories you added, you will probably have to install that software from Synaptic Package Manager (which can be found in System > Administration > Synaptic Package Manager). This will give you a list of "packages." Just search for "twiki" or "apache" and you will probably find the right package.
