### Post by tjpearson on 2015-11-09
I have decided to return to Ubuntu and although I don't believe my laptop is that ancient, running 2 gig of ram, the system was painfully laggy.

I noticed a lot of other users were suffering with this so written for the absolute newbie, follow this to fix, it worked for me

Open Terminal and type or copy and paste the line below:

sudo apt-get install gnome-panel

Then:

sudo reboot

At login click the gear icon above your user name and select:

GNOME Flashback Metacity

The system will reboot but will not show any desktop icons, these can easily be added, but at least your system will run nicely.
