### Post by ibid49 on 2008-05-11
Hey all,
 
One more recent convert from Windows to Ubuntu, running Hardy.  

I'm a HUGE proponent of keyboard shortcuts, and I had two things that I really liked about Vista that were worth all the annoyances Vista brought:

1) Hit Start (or windows key), start typing the name of an application, and it would display all matching applications as you typed (from anywhere in the start menu - nested directories, control panel, whatever).  Once you had typed enough letters to narrow it down to one (or the top) choice, you could just hit enter and it would run it.  To run Notepad was WindowsKey+N+O+T+Enter.  So as long as you knew the name of the application, you'd never have to touch the mouse.  Nautilus seems to have what I'm looking for, but it doesn't seem to apply to the application menu.

2) The quicklaunch bar held my most important applications, and Vista had keyboard shortcuts for them: WindowsKey + 1-9 would run whatever shortcut was in the 1st through 9th place in your quicklaunch, regardless of whether you moved them around or added some, etc.  Thus, if firefox was the 5th icon in my quicklaunch, I could run it with WindowsKey+5.  I realize I can use keyTouch or something similar to set up explicit shortcuts on a per-app basis, but I'd really like to just set this once and have it in affect any time I add a new launcher to the panel.  (is that the right terminology?)

Is it possible to either replicate one or both of these, or is there a more Linux way to do it?  I've looked at Keyboard Shortcuts in the Preferences menu, but it has a limited selection of what you can assign shortcuts to, and I don't see any way to add new items.

The first item sounds like something [GnomeDo]("https://wiki.ubuntu.com/GnomeDo") can handle. I've checked it out a little and it sounds like this type of thing is right up Do's alley. You can definitely bind keys to specific applications. The second should be easily done via key bindings, and I found [this topic]("http://ubuntuforums.org/showthread.php?t=79560") which may help with that.
