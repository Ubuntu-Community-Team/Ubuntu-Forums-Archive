### Post by themusicalduck on 2010-08-20
Another difference between normal Ubuntu and Ubuntu Studio is the kernel type.

Unless this has changed in the last few releases (I used Ubuntu Studio last a while ago) it uses the realtime kernel, which is good for low latency audio. I've read it's less stable for normal PC stuff, but I don't really know how true this is, especially now. You could always install both kernels and switch between them.

As for good soundcards, I have an M-audio Audiophile 192 card which works great with Jack and Pulseaudio. It's a good card for recording, low latency and is good quality. Only thing is that the sound goes funny if you hibernate, but that's the only problem.
