### Post by Ultimate07 on 2008-07-14
I had to boot XP to write this. I have another computer with Ubuntu set up next to me having the exact same problem as well. We have a DSL connection and according to the connection information its connected at 100 Mb/s. Whenever I try to do something like update or surf the web I get absolutely nothing. Theres nothing to indicate that anything is wrong.

Is there possibly some setting that is causing this?

EDIT: Had a tech savvy friend help me with this one. He said it was one of the oddest things hes seen. The router had outdated firmware which was apparently giving 2 different IP addresses depending on which OS you were using. The router is a Linksys WRT54G ver 2 if anyone else has this problem. Updating the firmware fixes the issue.
