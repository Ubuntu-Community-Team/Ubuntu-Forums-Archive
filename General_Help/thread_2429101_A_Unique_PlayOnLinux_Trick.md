### Post by aetheralmeowstic on 2019-10-13
My laptop, before its hard drive gave out after a year of faithful usage, ran Ubuntu Studio 19.04. I had a 16GB flash drive manufactured by Onn. I wanted to install Dolphin Portable, for Windows, on tte flash drive. I did it successfully.

For anyone who wants to install any portable Windows prograb on a flash drive via PlayOnLinux, here's the method I used:

Download the .paf.exe file and use PlayOnLinux to run the installer like usual. Format the flash drive with the FAT file system, check what PlayOnLinux uses as your flash drive's drive letter on the virtual drive (for me, it was E) and set PlayOnLinux to install the portable app to that drive. Once that's done, the program should be properly installed on the flash drive in a way that will allow it to run on Windows.
