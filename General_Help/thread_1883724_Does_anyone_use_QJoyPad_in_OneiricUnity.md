### Post by neu5eeCh on 2011-11-19
OK, I can at least answer my own second question: ALT-F2 to run programs in Unity.

As for qjoypad, all my searches have turned up no information. Either I'm the only one having issues with it in Oneiric, or it doesn't work in Oneiric.

**Edit: **The workaround for running qjoypad in Unity is to run it with the --notray option. 

> gjoypad --notray

However, bugs remain. I've noticed saved profiles can't be applied from one session to the next. Each time one starts a game, the keybindings need to be reset. For more information, look [here]("http://ubuntuforums.org/showthread.php?t=1885379").
