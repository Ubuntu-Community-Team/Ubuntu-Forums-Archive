### Post by Mark Phelps on 2010-11-17
> **john1923 said:**
> Most computers come with a rescue CD. Popping that in and following the instructions should put the computer back into the state it was in when you bought it. Ie no Linux.

Not necessarily ...

These days, what most of these recovery CDs do is only boot you into WinRE and then go looking on the hard drive for the Windows image file -- which they will then use to restore the PC.  If the OP removed that partition (remember, they said no dual-boot) during the Ubuntu install, there is no image to restore FROM.
