### Post by QIII on 2012-03-02
Jeff Hoogland at Bodhi Linux has said they will be trying to get a version based on Precise out within a few months of its release.

Bodhi is an extremely light weight Ubuntu derivative that will run on a rubber band and a hamster in a wheel.

Not that I'm directing you away from Ubuntu proper, but it's an option if you want both light and based on an LTS.

As an alternative, you could do a minimal install of Precise and use the E17 desktop environment that Bodhi uses.
