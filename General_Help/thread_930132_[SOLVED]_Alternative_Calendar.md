### Post by anotherdisciple on 2008-09-26
I don't know about the default calendar, but Sunbird has a Hebrew calendar add-on that will probably do what you are trying to accomplish.

I believe it is in the repositories, so all you have to do is open a terminal and type:
```
sudo aptitude install sunbird
```

... and then get this add on...

[[COLOR="Blue"]Hebrew Calendar Add-On[/COLOR]]("https://addons.mozilla.org/en-US/sunbird/addon/1379")


Hope that helps!
