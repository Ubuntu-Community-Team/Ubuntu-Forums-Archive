### Post by ajgreeny on 2010-05-12
There is no root account by default in Ubuntu.  ```
sudo command
```is used instead, or for a gui program gksudo for gnome and kdesu for kde followed by the program name, eg ```
gksudo gedit
```
See [https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo) for details.
