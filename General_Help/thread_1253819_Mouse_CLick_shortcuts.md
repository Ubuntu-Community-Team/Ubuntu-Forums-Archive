### Post by asmoore82 on 2009-09-02
You need to install the "mouseemu" package - it comes already installed
by default on PowerPC(Mac) versions of Ubuntu.

After installing you can edit the file "/etc/default/mouseemu" to tweak its settings;
and you will have to restart to get the service running.
