### Post by WhiskyChris on 2009-03-12
Hi all,

I have downloaded a program through the repositories (a Chemistry program - Gabedit) and decided I wanted to use the latest version which I downloaded and installed from the [sourceforge page]("http://gabedit.sourceforge.net/"). I now however have two seperate versions installed on my system.

Is there anyway I can update the link under Applications - Education - Gabedit to point to the newest version? Or perhaps I need to install the new version in a different location to overwrite the older version? (It's currently in my home directory /home/christopher/gabedit)

Thanks,
Chris

---

### Post by taurus on 2009-03-12
Just use synaptic and remove the old version (repos) from your machine.  Then, you can create a launcher for the new version in your $HOME/gabedit on the top panel so when you want to run it, just click the icon.

Otherwise, I guess you can add a menu for it in System -> Preferences -> Main Menu.
