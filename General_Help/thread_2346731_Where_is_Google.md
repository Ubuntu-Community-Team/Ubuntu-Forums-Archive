### Post by Keith_Helms on 2016-12-18
Firefox:  edit menu (if you don't have view menu bar selected, click on the 3 vertical bars at top right) -> preferences -> search tab on left -> default search engine

Chromium: three vertical dots on far top right gets you a menu -> settings -> search engine listbox
