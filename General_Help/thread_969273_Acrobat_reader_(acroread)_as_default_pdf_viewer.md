What I want is simple (I guess), every time I double-click a pdf file I want it to be opened using the acrobat reader instead of the default document reader.

If I right-click the file on nautilus and choosing acrobat from the list, or opening with a "custom command" (acroread) just open the file for one time, does not change the default behavior.

Also, after playing around a bit trying to solve the problem caused "acroread" to be shown twice on my right-click->open with... menu. 

Any ideas on how to change the default pdf viewer and/or removing the duplicate entry from my menu??

### Post by empty_tank on 2008-11-03
right click on the pdf file > choose properties > choose open with > select acrobat as default.  Remove other entries you don't need or want.
