### Post by 3Miro on 2012-10-08
The way to uninstall software depends on the way you installed it. If you used the Software Center, then you should be able to uninstall any software from it. If you used a .deb file, then you can install Synaptic Package Manager (available in the Software Center) and is a tool for a very fine control over the software. Synaptic will uninstall any software from Canonical or any other place that came in a .deb file.

If you installed the software manually (i.e. from a .tar.gz file) then you need to uninstall it manually. Let us know what instructions you used to uninstall the software so we can help you reverse the process.
