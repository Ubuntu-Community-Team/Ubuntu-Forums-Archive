### Post by ryanhaigh on 2008-06-26
A quick google search indicates that there are packages for moto4lin in the ubuntu repositories. You will need to have the universe repo enabled in System>Admin>Software sources (refresh package list when prompted). Then you can use your favorite method to install the software:
[https://help.ubuntu.com/community/InstallingSoftware](https://help.ubuntu.com/community/InstallingSoftware)

Or just click this link: [apt://moto4lin]("apt://moto4lin")
