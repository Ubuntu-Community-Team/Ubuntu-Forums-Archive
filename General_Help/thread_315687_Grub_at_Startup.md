### Post by riven0 on 2006-12-09
When your computer first starts, you should be able to enter grub by continuously hitting the esc button.

EDIT: Otherwise, if you have a problem with xserver, ubuntu should automatically boot you into a console screen and you can type sudo dpkg-reconfigure xserver-xorg from there. Make sure you log in first, though.
