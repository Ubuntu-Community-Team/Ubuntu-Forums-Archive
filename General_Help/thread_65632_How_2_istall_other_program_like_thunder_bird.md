### Post by ad noiseam on 2005-09-15
[QUOTE=s_bettencourt]i cant get evolution 2 work with gmail acouts... help[/QUOTE]

Evolution works perfectly well with Gmail. Here is the deal:

1. Go to Preferences - Mail Accounts - Add

2. Name: whatever you want, Email address: [email]youraddress@gmail.com[/email]

3. Server type: POP

4. Host: pop.gmail.com, Username: [email]youraddress@gmail.com[/email], Use secure connection: always

5. Server Type: SMTP, host: smtp.gmail.com, Server requires Authentification: check, Use Secure Connection: Always, Username: [email]youraddress@gmail.com[/email]

You're set. :)
