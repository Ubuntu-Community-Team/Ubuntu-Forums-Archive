### Post by BslBryan on 2009-05-08
Hello, muctadir. :-)

Try this:
```
sudo apt-get install kopete
```

Kopete is a chat client for Windows Live, Yahoo, and AIM.  It has support for voice chat and video chat, but to get either of them working is tricky sometimes.  There are plenty of how-tos, though, on Google.

I hope I've been of some help.

Best to you, and good luck. :-)
