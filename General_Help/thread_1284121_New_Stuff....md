### Post by fluffman86 on 2009-10-06
Yeah, your computer will run faster on faster hardware.  And a processor that old will probably not work in your new motherboard, which you'll need to get for SATA hard drives and DDR2 or DDR3 RAM.  So it's probably a moot point...you'll HAVE to buy a new CPU.

That said, I run Ubuntu on a 1.6 atom on my netbook, and it's plenty fast.  Before that, I had a 1.6 P4 Mobile, which clocked down to 400Mhz to save battery.  It almost never clocked up to the full 1.6GHz.
