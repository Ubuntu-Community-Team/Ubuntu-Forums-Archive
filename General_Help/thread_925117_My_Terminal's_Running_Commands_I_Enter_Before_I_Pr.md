### Post by sayakb on 2008-09-20
While copying the commands, make sure you copy the text only, but not any line breaks within the CODE if any. Also, try right clicking and selecting Paste if it works for you, though which should not quite make a difference.
```
echo Try this command
```

### Post by bingoUV on 2008-09-20
The "Enter" that you want to press after editing the command is getting copied along with the command. Sometimes it gets difficult to do what  LinuxIsInnovation is saying, so try copying only upto the second-last character. Remember the last character, and type it yourself.
