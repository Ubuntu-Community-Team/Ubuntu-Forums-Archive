### Post by QwUo173Hy on 2008-02-17
Hi!

Welcome to Ubuntu :) You can uninstall any plugin by clicking on

Tools -> Add-ons

and choosing 'Extensions' at the top.

From here, just choose the plugin / extension you want to remove and click 'Uninstall'


To install Realplayer, try reading this
[https://help.ubuntu.com/community/RealplayerInstallationMethods](https://help.ubuntu.com/community/RealplayerInstallationMethods)
