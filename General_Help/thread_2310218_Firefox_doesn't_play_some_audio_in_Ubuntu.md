### Post by Bucky Ball on 2016-01-17
As above. Install 'ubuntu-restricted-extras' from Software Centre if you're using Ubuntu (there is a restricted extras package specific to each flavour, for instance 'xubuntu-restricted-extras' for Xubuntu).

Just to say the play button is working fine for me. (Xubuntu 14.04 LTS, Firefox 43.0.4) :-k

Which Ubuntu release are you using?
