### Post by patchwork on 2010-03-25
Try some others out (you can install different window managers or desktop environments and choose between them on the login screen)

[Link]("http://xwinman.org/") to a list of window managers and environments.

While desktop environments come with all standard applications installed, window managers are just that.  You will need to install separate file manager, network manager, etc.  Also, some window managers will take more setup than others.  Consider all the choices a perk from running Linux.
