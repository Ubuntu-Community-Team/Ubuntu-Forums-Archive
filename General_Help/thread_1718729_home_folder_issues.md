### Post by Frogs Hair on 2011-03-31
Are you referring to the folder listed on the places menu  or the file browser (nautilus) as well ?  When you have Nautilus open right click on any folder and select open with another application and make sure file browser is selected from the menu and the remember this application box is checked  and test it to see if the folder opens normally.
