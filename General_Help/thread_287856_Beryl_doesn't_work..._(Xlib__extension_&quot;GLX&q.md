### Post by Toxicity999 on 2006-10-29
you haven't loaded the glx module!

sudo gedit /etc/X11/xorg.conf

near the top you should see Section "Module"
Add:
    Load    "glx"

I have:
Section "Module"
    Load    "bitmap"
    Load    "ddc"
    Load    "dri"
    Load    "extmod"
    Load    "freetype"
    Load    "glx"
    Load    "int10"
    Load    "type1"
    Load    "vbe"
EndSection

Incase you need more of a comparison.

sudo nvidia-xconfig --add-argb-glx-visuals --composite
``` will fix the missing glx problem.
