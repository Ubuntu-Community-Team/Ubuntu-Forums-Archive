### Post by Baufo on 2008-05-31
I have a problem running programs that require input such as "wc". "wc < somefile" works fine, however I would like manual input. So when I just type "wc" I get to type text - but how do I tell bash that I am finished typing my input and want to start executing wc?

---

### Post by dagnabit dang doohickey on 2008-05-31
After entering your input, press Ctrl+D to begin executing the command.
