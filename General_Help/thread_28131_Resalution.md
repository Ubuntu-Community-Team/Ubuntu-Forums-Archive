### Post by heimo on 2005-04-19
Try this:
```
 
Section "Monitor"
Identifier "DELL E773c"
Option "DPMS"
HorizSync 30-70
VertRefresh 50-160
EndSection

``` 
 
And this:
 
```
 
SubSection "Display"
Depth 24
Modes "1024x768" "1280x1024" "800x600"
EndSubSection

``` 
 
That way you should get 1024 x 768 by default, but if it fails, you may get other resolution (and can change to higher or lower resolution in System-menu). You should get at least 75Hz for 1024x768, but your monitor can do up to 85Hz. If you want to force that refresh rate, you can change the VertRefresh line to *VertRefresh 85 (*Though you won't be able to change to higher resolution any more.)

### Post by heimo on 2005-04-19
You need to restart only the graphical environment (Xorg). Close all programs and hit CTRL+ALT+backspace. Or you could type:
```
 sudo /etc/init.d/gdm restart
``` 
EDIT: I'm not actually quite sure if that's enough - you better just hit those three buttons and "kill" it. 
 
If your xorg.conf is not correct, you may be stuck to console (only command prompt) and you'll need those instructions on the other thread to replace the configuration with the backup (if you did that, as in instructions).
