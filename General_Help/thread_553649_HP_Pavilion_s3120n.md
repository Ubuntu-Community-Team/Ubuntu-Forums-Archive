### Post by nzlee34 on 2007-09-18
nvidia and amd will be fine for the cpu/graphics/motherboard. What sort of network interface does it have? If you are planing on using wireless it would be a good idea to check if there are good native Linux drivers (although it is often possible to use ndiswrapper, this is not possible with some software). Also investigate bluetooth drivers if you intend on using that.
