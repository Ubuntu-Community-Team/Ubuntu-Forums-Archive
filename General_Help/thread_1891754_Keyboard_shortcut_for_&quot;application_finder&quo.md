Anybody knows how to configure a Keyboard shortcut for "application finder" in Xubuntu?

---

### Post by Toz on 2011-12-06
Goto Settings -> Settings Manager -> Keyboard -> Application Shortcuts.

Click on "Add" and enter **xfce4-appfinder** as the command.

Click "Ok" and another window will pop-up prompting you for a keyboard shortcut - press the key combination that you want to use.

Using that key combination should now bring up the application finder.
