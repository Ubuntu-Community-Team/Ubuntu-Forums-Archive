### Post by grahammechanical on 2011-07-15
Let us get it in the record so that it is clear to anyone viewing this thread. The purpose of the wine software is to allow anyone who has a legal copy of a program that runs on a Windows operating system to run that program on Linux without the Windows operating system being installed on the computer.

It is legal if the Windows software has been legally obtained. If the program has not been legally obtained then it would not be legal. And it would not be the developers of wine that would be breaking the law.
