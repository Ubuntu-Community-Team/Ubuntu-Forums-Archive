### Post by 3Miro on 2009-03-30
If you already have windows installed in a partition, you cannot start it fro within Ubuntu. You have to make another installation from within the VM that you want to use.

Virtual Machines run virtual hardware, so there will be all sorts of  problems even if you can boot windows.
