I want to open multiple browsers viz.**fire fox,opera,chrome all at once via terminal**. How can I do it?

---

### Post by jfparis on 2010-03-16
```
$ firefox &
$ opera &
$ google-chrome &
```don't forget the "&" at the end of each command. That will run the program in background and let you input another command right away
