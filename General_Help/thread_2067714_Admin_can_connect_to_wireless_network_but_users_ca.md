### Post by ezacharyk on 2012-10-07
I have two desktop computers with wireless network cards in them. In both of them, the admin user (the one set up during installation of Ubuntu) can connect to the wireless network. I can see my home network. I can enter my password and I can connect. 

However, I have tried to create some user accounts for my family in one and Myth TV on the other, but on neither computer the new accounts cannot connect to the network.

They can see the network available, but when I go to enter the password, the network manager rejects the password. I know I am entering it correctly. I have not changed it. I am entering the exact same password used by the admin accounts.

Is there some reason why User accounts would not be able to connect? Is there some group they are not a part of that they should be? If so, why are they not part of that group automatically?

---

### Post by zvacet on 2012-10-09
Click on network icon>edit connections and find your network and at the bottom check **available to all users**.
