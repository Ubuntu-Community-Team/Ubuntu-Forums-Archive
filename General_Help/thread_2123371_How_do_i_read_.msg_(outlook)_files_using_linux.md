I would try Outlook Express first.  It should be part of any recent Windows install.  If not that, then I would try and export the files in .pst form.  I believe there is an app for reading PST files for Linux.  Finally if that doesn't work for you you can try this PERL script at: [**_[COLOR="#FF0000"]http://www.matijs.net/software/msgconv[/COLOR]_**]("http://www.matijs.net/software/msgconv").  As the author says its a bit of a meat axe but it does provide the contents in text form.
