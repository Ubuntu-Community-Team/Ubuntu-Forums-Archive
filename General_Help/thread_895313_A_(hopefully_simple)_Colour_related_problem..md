For the color problem: System/Preferences/Appearance -> Customize -> Color-tab (or select a different Applications-theme).

The second problem could be related to using different programs.. Most streaming videos use Flash, which is know for it's heavy CPU use. It's hard to say for sure if this is your probelms, but you should check your CPU usage while playing those streaming videos..
