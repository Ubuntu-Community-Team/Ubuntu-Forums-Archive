The basic rule in Ubuntu is that after release, each Ubuntu version only gets updates to fix serious bugs and to provide security fixes. Apart from that, all package versions are frozen to what was tested to work together during the development.

The new software versions will then appear on the next Ubuntu release.

There are some exceptions to that rule, for example Firefox does get proper updates these days, and LTS releases will get some new package versions over time, and also some stuff may be backported from the next development version if they fit certain rules.

But apart from those special cases, if you want latest version of programs then use a PPA or some other third-party repository. That's what they are for. Or consider trying some Linux distribution that uses rolling releases if you prefer having latest versions of programs over stability. :)
