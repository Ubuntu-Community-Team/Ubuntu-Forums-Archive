### Post by lswb on 2009-07-19
Right click on the panel, select "Add to Panel", then select "Custom Application Launcher" Fill out the fields as required. For opening web sites for instance you could enter in the command field "firefox http://ubuntuforums.org" A tip for opening files, instead of using the program name for the file type being opened, you can use the command xdg-open, i.e. in the command field for the launcher, use "xdg-open /path/to/file" and the default application registered to open that file type will be used.
