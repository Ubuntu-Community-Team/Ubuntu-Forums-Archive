### Post by ad_267 on 2008-05-13
I second gftp for uploading files. I use geany as a text editor and the GIMP or inkscape for graphics. If you want a free design to start with then have a look at [http://opendesigns.org](http://opendesigns.org). This can also help you understand how a good design is put together if you want to make your own. If you'd rather work with a WYSIWYG editor then like tamoneya said use kompozer.
