### Post by oldsoundguy on 2008-11-16
I have A Logitech ergonomic keyboard and a Logitech cordless optical track man working through a DVI KVM switch and feeding a 8.10 Ubuntu box and a Windows XP box.  It took using a WIRED mouse and Keyboard to get the KVM to work, but once the KVM was set up, I just plugged the receiver into it.

In another location I have an MS ergonomic keyboard and another Logitech cordless optical track man through a VGA KVM switch feeding a 8.10 Ubuntu box and a Linux Mint Elyssa box.  Same situation .. had to use hard wired into the KVM to set it up, but once running, plugged in the receiver and was good to go

both setups have gone through multiple boots on both computers without having to re set the keys or mouse.

(but watch those batteries .. there is NO warning when they are about to die .. they just DIE.)

(and if you use the USB, make sure it is activated in the bios.  (I prefer to use the PS2 because Windows just forgets USB now and then!)

### Post by lovinglinux on 2008-11-16
If you have a wireless router it can interfere with the mouse if both have similar frequencies. It happened to me. The mouse scrolled pages by itself when the wireless was active.

So check any wireless equipment for frequencies and avoid those when buying the mouse.
