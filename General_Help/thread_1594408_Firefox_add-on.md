### Post by Vaphell on 2010-10-29
i think i saw in the comments that you need to save the xpi (right click on the Add to FF button, save target file) rename it to .zip (xpi is a zip file), uncompress, modify the manifesto file to allow higher version numbers, zip again and rename to xpi. It should work

> by xaminmo on February 8, 2010 #

This is one of my most often used extensions.

I too moved to FF 3.6, but to make this work, I renamed the xpi to .zip, edited the RDF files in wordpad to show 3.9.* vs 3.5.*. rezipped, and it works like a champ on FF 3.6 Gecko/20100115

There is also that addon called Nightly Tester Tools which allows you to disable version checking.
[https://addons.mozilla.org/en-us/firefox/addon/6543/](https://addons.mozilla.org/en-us/firefox/addon/6543/)
