### Post by gandaran on 2011-08-21
you probably have conflicting flash plugins installed (more than one version) if you don't know how to fix it there is an easy way
install the [firefox addon flash-aid]("https://addons.mozilla.org/en-US/firefox/addon/flash-aid/") to help remove conflicting plugins and install the correct one for your system.
