### Post by Rahu on 2006-03-02
unipage(unipage.org) Can convert a whole site(including images and such) Into a single html file. It's currently unavailable for linux, but wine has a chance of running it. If you get that working I would try opening that html file in opeenoffice and exporting to pdf. It most likely will fail when you need to use wine, but it's the only way i can think of.
