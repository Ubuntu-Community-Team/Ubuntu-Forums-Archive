### Post by bof on 2006-12-24
glabels - Too late for this year I'm afraid.

Install glables using Synaptics.
Export your Christmas list from kaddressbook as a csv file then use glabels to design and print labels.
Use Objects/Merge Properties to select fields from the csv files.

glabels has a good selection of labels and works well but tends to crash frequently so keep saving!
