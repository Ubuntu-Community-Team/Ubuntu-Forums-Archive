### Post by dexter on 2007-02-11
Does the compiler have rights to write to the places specified (--exec-prefix=/usr) ? If not, try running your compiler with sudo rights.

---

### Post by WW on 2007-02-11
A configure script should not require root privileges.

SnowPunk98, do you have gcc installed? You can get the build tools with one command:
> sudo apt-get install build-essential
This will get the C/C++ compiler, the make command, and a couple other tools.

---
