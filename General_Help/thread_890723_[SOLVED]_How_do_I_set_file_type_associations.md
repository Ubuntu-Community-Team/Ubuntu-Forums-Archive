### Post by iaculallad on 2008-08-15
> **Sam324 said:**
> How do I set up Ubuntu to open a certain file type with a certain application?  For example, .torrent.

Right-click on the file, select "Open with other Application" -> and select your Torrent client from the list of applications listed on the window that pops-up and click on "Open" button.
