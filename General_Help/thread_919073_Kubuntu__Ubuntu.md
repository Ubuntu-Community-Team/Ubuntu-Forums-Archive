### Post by Elfy on 2008-09-13
Arranging icons - to arrange horizontally, it is possible but atm I'm not using gnome and can't rememeber completely - right clcik the desktop - believe there is an option to tidy, or similar - that needs to be off.

Icon size - open nautilus and Edit > Preferences - Icon View Defaults - change size there

Fonts - I keeps font I install myself in a hidden folder in my /home - .fonts 
once fonts are installed run ```
sudo fc-cache -f -v
``` to rebuild the font cache

Taskbar - when I am using gnome - I have one taskbar with only those items I need on it.
