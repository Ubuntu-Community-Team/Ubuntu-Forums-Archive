### Post by ibuclaw on 2010-01-24
1) Install an IRC client:
```
sudo apt-get install xchat
```
2) Start the application and Connect to the Ubuntu Servers (FreeNode).
Ensuring that you have Nicknamed yourself appropriately.
3) Join the main channel.
By default you will join #ubuntu, if you type in:
```
/join #ubuntu-classroom
```
It will open a new channel tab, where you can talk and congregate with others.
