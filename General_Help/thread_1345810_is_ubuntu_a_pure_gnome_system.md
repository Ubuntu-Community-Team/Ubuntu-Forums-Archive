### Post by fluffman86 on 2009-12-04
Ubuntu uses a fairly modified Gnome.  By default, installing a plain gnome on some distros means you get the epiphany browser instead of Firefox, as well as some other weird changes.  But for the most part, Ubuntu is almost all gnome.  If you want to run KDE, I recommend installing Kubuntu from the outset, but as sakisds said, you can add any window manager or desktop environment to a vanilla ubuntu install.

The server install has no gui, and during the install it will ask if you want to install ssh, lamp (apache, mysql, php), etc.  If you want to have a GUI (mouse, windows, pictures on the screen, etc), then it's easier to install the desktop edition and then add the ssh server, lamp server, etc.
