### Post by btindie on 2011-05-17
> **bofak said:**
> What do I need to view a web page with an .asp extension?
Nothing, that's a server side scripting language that returns a HTML page. It may not work for you as you're missing a plugin that it uses. Try installing the Adobe Flash plugin (flashplugin-installer). Under FF, go to Tools -> Add-ons then Plug-ins. If you don't have Shockwave Flash listed then you'll probably need to install it.
