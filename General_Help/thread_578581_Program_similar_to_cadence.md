### Post by por100pre1 on 2007-10-17
I'm not sure if this will help, but there's a program called [Kicad]("http://www.lis.inpg.fr/realise_au_lis/kicad/").

If you want to try it, copy, paste and ENTER this command in Terminal (**Applications> Accesories> Terminal**):

```
sudo apt-get install kicad
```
