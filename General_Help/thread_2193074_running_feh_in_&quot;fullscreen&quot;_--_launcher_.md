### Post by sudodus on 2013-12-11
You can right-click on the picture and select

Options -- Fullscreen
and maybe also
Options -- Auto-Zoom

or use options at the command line. I use this alias to run a screenshow of files in the current directory tree

```
alias fehs='feh -r -F -V -d -Z -z -D 5'
```
