### Post by ugm6hr on 2009-06-27
> **wcutrombonekid said:**
> puppy is still getting updates?

Yes, it has a very active community.  Up to v4.x now.

As for Xubuntu, the most noticeable difference is thunar, which I actually prefer over nautilus, but it cannot browse network shares like nautilus can.  There is a How-to to add this functionality to thunar using fuse, but I couldn't get it to work, so just swapped to Gnome.

Obviously, the panels, menus and desktop function slightly differently, but I don't think most people would notice.

Additionally, Xubuntu is a little over-crowded with autostarted Gnome applications, so is not a great deal faster than Ubuntu.  You can choose to turn some of them off, but you will need to find appropriate alternatives for e.g. network-manager-gnome etc.

XFCE supports Compiz, but it doesn't sound like he wants that anyway.

If he wants a fast and functional desktop environment, Ubuntu minimal + XFCE or Ubuntu minimal + LXDE (see link below) are good options.
