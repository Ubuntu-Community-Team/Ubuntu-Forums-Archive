### Post by CatKiller on 2009-10-11
In general, you wouldn't. Packages install files to the appropriate place in the [filesystem tree]("http://en.wikipedia.org/wiki/Filesystem_Hierarchy_Standard"), which is where they're expected to be to work. What is it that you're trying to achieve by having those files in your Home directory?
