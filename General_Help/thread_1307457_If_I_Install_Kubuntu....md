### Post by lilbill on 2009-10-31
Installing that installs KDE the K Desktop Environment along with it's productivity software: Abiword, another File manager, Terminal etc...

On login, you will choose which session you want from the dropdown on the bottom of the page (GNOME, KDE, CLI).  

All of the software you've already installed will still be available under KDE.
