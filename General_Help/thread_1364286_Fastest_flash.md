did you try using the ubuntu-restricted-extras package? That includes a closed-source version of flash that works rather well with netbooks.

I do know that if you install google chrome browser, ([http://www.chromium.org/getting-involved/dev-channel](http://www.chromium.org/getting-involved/dev-channel)) you could add an extension to have youtube videos play in html-5 (the extension is called html5-ifier [https://chrome.google.com/extensions/detail/kchoimdlcbapmcdnheaahjcdpdjdpfco](https://chrome.google.com/extensions/detail/kchoimdlcbapmcdnheaahjcdpdjdpfco))

That makes the videos play really nicely with very little strain on the computer.

Hope this helps! Merry Christmas!

You can do something similar using [Greasemonkey](https://addons.mozilla.org/en-US/firefox/addon/748) extension with [this script](http://userscripts.org/scripts/show/50771). The script should work with Firefox, Safari, Opera, Epiphany and Google Chrome. 

Instead of using html5, it replaces YouTube video player with mplayer.

There are other options. See Flash Optimization section of [Firefox optimization and troubleshooting thread](http://ubuntuforums.org/showthread.php?t=1193567).
