### Post by jblackhall on 2009-06-30
Slim Odds is right.  They won't update the Ubuntu 9.04 repositories with new software.  It only gets security fixes and some bug fixes.  There's a chance it will get pushed into "Backports".

If you want to install it soon(ish), you can add Fabien Tassin's PPA:
[https://launchpad.net/~fta/+archive/ppa](https://launchpad.net/~fta/+archive/ppa)  It's currently building, but it will probably be ready soon.  Just install the firefox-3.5 package in Synaptic.  It will install alongside of the current version of Firefox (I think).  Also, I'd recommend removing/disabling the PPA when you're done if you use other software that he has packaged, just so that future updates don't break your Gwibber, for example.
