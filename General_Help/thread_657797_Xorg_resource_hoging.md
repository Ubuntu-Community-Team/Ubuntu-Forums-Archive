### Post by Rocket2DMn on 2008-01-04
Awesome, I found this thread on another forum that seems to fix the problem:
[http://www.nvnews.net/vbulletin/showthread.php?t=86253](http://www.nvnews.net/vbulletin/showthread.php?t=86253)
It has you disable "UseEvents" in your xorg.conf, just make a backup first!
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.backup
```

edit: PS: you will need to restart X (CTRL+ALT+BACKSPACE will kill X and restart it)
