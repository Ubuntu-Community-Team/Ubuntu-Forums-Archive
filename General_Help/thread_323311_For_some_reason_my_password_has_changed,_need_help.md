Hi, 
  I don't know why but when I moved (physically to a new location) the password for my login didn't work. I am trying to set a new one via loging in in safe mode as root but don't know what to type in the terminal to change the password of my user. Does anyone know how I can do this? Thanks, 

 The Net Duck

---

### Post by taurus on 2006-12-21
After booting into recovery mode, run

```
passwd **thenetduck**
(assuming **thenetduck** is your login name...)
```
