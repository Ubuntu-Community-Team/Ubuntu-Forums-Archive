### Post by metalf8801 on 2013-10-25
Try using a "Local alias" 
Accounts > the account you want to change > Edit Account 
Below User Options you should see Local alias and an empty text box put what you want to show up instead of [email]1835120554@chat.facebook.com[/email]/5XAoi4xq in the text box click save and see if that solves your problem.
