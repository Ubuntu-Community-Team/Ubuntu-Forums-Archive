### Post by codemaniac on 2012-06-10
GConf-editor is meant to be a way for users to easily store user preference .

You can change the window border transparency using the gconf-editor.
To change the window borders transperancy you need to have the gconf-editor installed.

```
sudo apt-get install gconf-editor
```

You can also follow the below suggestion .

[http://ubuntuforums.org/showthread.php?p=11453890](http://ubuntuforums.org/showthread.php?p=11453890)
