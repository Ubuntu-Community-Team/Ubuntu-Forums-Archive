### Post by Darkhack on 2007-08-14
No.  Ubuntu only supports x86-32, x86-64, and Sun UltraSPARC.  There is also a PowerPC version but Canonical is no longer supporting it for future releases.  Ubuntu 6.06.1 (Dapper Drake, LTS) is the last version to support PPC.
