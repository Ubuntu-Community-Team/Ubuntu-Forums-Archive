### Post by realzippy on 2011-08-01
run 

```
gksudo gedit /etc/X11/xorg.conf
```

and change in "Monitor" section:

 HorizSync       28.0 - [COLOR="Red"]33.0[/COLOR]

to

 HorizSync       28.0 - 81.0


close file saving changes,reboot (or restart Xserver).
You resolution should be available then in nvidia-settings..
