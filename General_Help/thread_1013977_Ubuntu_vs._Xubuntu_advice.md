### Post by Bucky Ball on 2008-12-17
Xubuntu definitely. In my experience faster every time, especially with older hardware; one of it's specialties. If you *really* want Gnome (notice the difference straight away) you can always drag that in later and see how you go, but if you do, do yourself a favour and install Openbox also (you can find it with a Synaptic search). Install then logoff, click 'Sessions'. In there, select 'Gnome/Openbox' then login. Choose 'Just for this session' (you can make it default next login if you like it).

This will speed things up. Openbox by itself is really quick but I haven't yet worked out how to use it! It is too simple! Or maybe that's me ... lol

Xfce desktop manager (where Xubuntu gets the name) is really fast also, making Xubuntu on of the fastest and most lightweight of the regular Ubuntu flavours, a fact (there are more exotic desktop managers/environments and various bare-bones distros). Another distro you could try is DSL (Damn Small Linux). Some people with older machines rave about it. There is also Puppy Linux.

Have fun.

### Post by samden on 2008-12-17
Go with xubuntu, but if you want better performance still install fluxbox as well. I run Ubuntu with Fluxbox and Gnome, and jump back and forth depending on what I am doing. I had Xfce on this computer too but have removed it. 

I found both xfce and gnome used exactly the same amount of RAM (125MB when idle) and I didn't see much difference in performance between them, so as I am familiar with Gnome I stuck with that. But I predominately use Fluxbox as it uses 46MB RAM when idle and is very fast. So there are a lot of options!
