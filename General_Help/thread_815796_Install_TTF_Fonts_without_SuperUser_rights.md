### Post by FuturePilot on 2008-06-02
Place them in .fonts in your Home folder
Note the . dot in the front means it's hidden. Press Ctrl+H to show hidden files. It might not be there so you might have to create it. Make sure if you do have to create it to include the . dot in front or else it won't work.
