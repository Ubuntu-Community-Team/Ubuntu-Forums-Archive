### Post by Vaphell on 2009-09-03
yes, you can have both vista and ubuntu
you have 2 installation options:
- full independent installation - you prepare some disk space for ubuntu (separate partition) and install ubuntu there. At boot time you get to choose the system to run.
- wubi - it installs ubuntu inside windows, on windows partition in a virtual filesystem that is 1 big file in reality - this overhead means it's slower than fullblown installation. Ubuntu is shown in windows' Add/remove programs etc. Again you can choose which system to run at boot time.
