Is there some way to add users to contacts after reading the email without having to go enter the info manually?

---

### Post by Zill on 2008-04-28
Not sure if this is what you want but if you right-click on the sender's email address you should have the option to "Add to Address Book".  Obviously, this will only add the email address!

If you want to add more information then the sender should include a VCard, which you can then simply click on to import full details into Contacts.
