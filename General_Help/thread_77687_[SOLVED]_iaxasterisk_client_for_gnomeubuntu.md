### Post by hscottyh on 2006-01-04
**kiax**

Get the debian package from [http://kiax.sourceforge.net/](http://kiax.sourceforge.net/)

It has a the dependancy problem just like the Skype debian package  **libqt3c102-mt** needs **libqt3-mt** :

Rebuild the package following the directions on this thread:
[https://wiki.ubuntu.com/SkypeHowto](https://wiki.ubuntu.com/SkypeHowto)

Just follow the directions in the section ***Building Your Own Skype Package*** except with a different package name of course.
