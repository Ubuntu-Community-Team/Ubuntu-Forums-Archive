### Post by ankspo71 on 2010-06-08
Hi,
If you already have vlc installed, you can go to the menu "media" > "services discovery" > "Shoutcast Radio Listings" and enable it. When you open your playlist, there will be a searchable shoutcast directory. You can have vlc automatically add the shoutcast list on startup in the advanced options. 

There is also a program called "tunapie". This application is a shoucast/icecast player, but it requires an external player such as vlc etc. This is what I use now for shoutcast radio.

Oh By the way. Songbird is coming back as Nightingale (hopefully). [http://getnightingale.org/](http://getnightingale.org/)  (it's not available for download yet)
