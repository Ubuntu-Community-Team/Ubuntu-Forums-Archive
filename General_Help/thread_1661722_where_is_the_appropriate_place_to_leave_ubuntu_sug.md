Nice question.
My first thought goes to convenience. It's easier to manipulate one file than a folder with lots of them.
Furthermore, just copying the contents of an installation cd/dvd to a thumb drive, doesn't necessarily guarantee that you will be able to run the install.exe file and proceed with the installation. Installers sometimes search for their necessary files in specific paths; paths that require preset volume names. A mounted iso image can guarantee this (99.9% of the times).
