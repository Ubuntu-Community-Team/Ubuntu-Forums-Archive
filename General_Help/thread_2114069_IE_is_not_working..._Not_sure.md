### Post by Stonecold1995 on 2013-02-09
Actually, iirc Microsoft stopped supporting Linux (and Macintosh) quite a while ago.  Also, I'd highly, highly suggest you don't ever use IE.  It is ridiculously insecure.  Firefox and Chromium are great alternatives, which are both faster, lighter, and much, much more secure.

If you really want to use IE, you won't be able to run the Windows program directly using Wine (a Linux package which can run many Windows programs) because it is tightly tied up with the rest of Windows.  If you install Wine, you *will* however be able to run an open source version designed to run under Wine.  It's NOT built for normal web browsing, but it can be used for development.
