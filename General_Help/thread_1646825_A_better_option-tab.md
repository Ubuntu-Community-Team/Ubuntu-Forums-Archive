### Post by Krytarik on 2010-12-16
If you are using Compiz you're apparently talking about the "Static Application Switcher"-plugin, via "System -> Preferences -> CompizConfig Settings Manager" (if installed).

Unfortunately there is now way to make the app-icons bigger until now, though it seems that many users are asking for it.

You may however try the "Scale"-plugin, bind its initiation to one of the desktop-edges, and change the icon-size via the option "Appearance -> Overlay Icon".
