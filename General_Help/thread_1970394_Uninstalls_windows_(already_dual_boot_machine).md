### Post by Mark Phelps on 2012-05-01
You can download an ISO of the Ubuntu version you want to install and save that to a hard drive.  Then, go to PendriveLinux.com and download their Universal USB Installer app.  Install that, stick in a USB stick, and it will create a bootable Ubuntu 12.04 installer USB for you.

When you then boot from the installer, you have the option of choosing the entire drive -- which will then remove Windows in the process.
