Reinstalling the Operating System would destroy all data that your friend has. Do not install an Operating System unless your friend gives you permission.

You don't just "get" Vista and replace an existing OS with it. You have to purchase a license from Microsoft, which costs money.

If you want to use Vista and you want to use it for iTunes only. Then it is best to buy the license from Microsoft and then use Virtual Box. If you have enough RAM you will be able to run Vista from within Ubuntu. Virtual Box is in the Software Center and you can get instructions from their manual:

[http://www.virtualbox.org/wiki/Downloads](http://www.virtualbox.org/wiki/Downloads)

the manual is verbose, but very well written.
