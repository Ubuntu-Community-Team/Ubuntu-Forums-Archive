### Post by endotherm on 2010-12-08
ok,
1) on your desktop, create a new folder
2) rcllick the new folder and select "Open With another Application..."
3) select "Open folder" or "Nautilus" from the list. 
4) ensure that "Remember this choice..." is checked
5) click OK

now test your places menu. if all works as expected, delete new folder and enjoy
