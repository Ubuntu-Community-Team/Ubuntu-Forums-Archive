### Post by Anduu on 2006-06-10
Check your processes in the system monitor and see if metacity is running...if not type metacity in a console.Your title bars should return.

Leave the terminal open and restart making sure automatically save changes to sessions is checked in session properties.

Metacity should now be loading at login from now on.

XGL is known to mess metacity up so whenever XGL is run you will have to repeat this process when you go back to normal Gnome.To get around this I just added Metacity to my startup programs in my session properties.
