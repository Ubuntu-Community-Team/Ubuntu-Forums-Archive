### Post by buzzingrobot on 2015-04-25
Unity Tweak Tool is in the official repos.  Not sure what other tool you are using for tweaking.  

The "Default Font" setting in Unity Tweak Tool, here, affects interface fonts.  Some changes may not be apparent before logging out and back in.

Unity Tweak Tool allows disabling webapps so it makes sense it complains if the schema supporting it is missing.  However, it's odd that it's missing as it's there in a default install.

Default font settings for Ubuntu's font rendering style include: DPI:96; antialiasing: on; hinting: on; subpixel hinting style: slight. None of those, though, impact font size except for the DPI setting. It needs to be 96.  Changing it *will* affect font size.  

Not sure what "recovery" involved, but perhaps a clean install is worth considering.  Your troubles with font sizing seem unusual.

Also, LCD monitors have a single optimum resolution that should be detected and used automatically.  Attempting to alter that resolution will result in degraded display quality. (Obviously, the wrong video driver will also affect the display, as in a botched proprietary driver install.  Also, some Nvidia cards do not play well with some versions of the open source Nouveau driver, but that results in a grossly and pretty obviously inadequate display (I think System Settings->Details will show the Gallium driver is in use.)
