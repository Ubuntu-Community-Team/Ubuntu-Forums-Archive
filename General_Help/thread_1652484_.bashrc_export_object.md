I don't know what hottub would mean in this context. But what I think you need to do is just edit the .bashrc file and add this line.
```
export OBJ1=done
```
Remember that when you're exporting paths, by means of
```
export var=path
```
you're creating a 'var' named variable and setting its value to 'path'.
