I've Googled *GNOME for Windows*, *F-Spot for Windows*, etc. I haven't found anything worth posting here. So I only have one idea to offer at this time. You might want to try Microsoft's Windows Live Photo Gallery (click [here]("http://get.live.com/photogallery/overview")). It is supposed to work in XP and Vista. It's not the best simple photo program out there. But it's among the best simple Windows-based photo programs. One feature that I enjoy about Windows Live Photo Gallery is that it automatically prompted me to download the appropriate RAW plugin for my Olympus digital camera. Of course, you could install Ubuntu in a virtual machine. But honestly, that seems like too much work for just one program.
