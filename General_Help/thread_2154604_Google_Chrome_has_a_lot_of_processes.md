### Post by MidnightGrey on 2013-06-16
Since nobody seems to be able to give you a straight answer (except for the lengthy blog post vasa linked to),

basically, no there is nothing wrong with your Chrome browser, that is just how chrome works. Chrome treats each Webpage/Plugin/etc as a seperate process, so yes you are bound to see multiple Chrome processes per browser.
Chrome has its own task manager which you can open to see what each of the 21-processes are doing, I dont have chrome installed right now but i believe it is something like right click on the title bar and select the Task Manager option.
