### Post by P4man on 2009-08-11
You probably need a subtitle file if its not included.
You can find subs on:

[http://www.opensubtitles.org/en](http://www.opensubtitles.org/en)
[http://subscene.com/](http://subscene.com/)

Download the file, extract the archive if needed, and put it in the same folder as the movie. Not sure if totem auto loads subs, but you can manually load them through the menu 'image" (or picture? dont have english version here :p ) > subtitles.

Most media players will automatically load the subs if they have the same name as they movie file.
