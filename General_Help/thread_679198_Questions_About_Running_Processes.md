### Post by DMK62 on 2008-01-26
Probably your browser using java or some other application. You can check your running processes from terminal by using the top command. You will probably find that most are in a sleeping mode and that is normal. If you are using the gnome version of ubuntu go to System and then Preferences and then Sessions and there is tab where you can configure what gets started during boot. I disabled trackered, bluetooth, and evolution as I had no need for them. Be careful what you disable. 

Java and flash can be somewhat resource intensive esp with older computers.
