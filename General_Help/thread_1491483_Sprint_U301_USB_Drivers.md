### Post by emanuel.b on 2010-05-24
As far as I know, sprint's windows drivers are incompatible with Ubuntu. You could try Ndiswarapper, which is a driver framework to make Wi-Fi cards compatible with Ubuntu, using it's Windows drivers. I doubt that it would work though, and I've had some troubles with Ndiswpapper.

Anyway, I think that Sprint switched the device archetecture, to support the 4G network. But have you tried using it yet? Just plug it in to your computer, and click on the networking icon in the notification area. If it dosen't appear as an option, then right-click on the menu and select "Edit Connections". Click on the mobile broadband tab, and click "Add". A window should appear, and if your device is supported, it will appear in the drop-down list. :)
