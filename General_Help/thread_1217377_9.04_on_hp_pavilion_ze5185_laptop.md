### Post by emeraldgirl08 on 2009-07-19
Jack. If you want a clean HD- google and download Killdisk. Burn that ISO image into a bootable CD and boot from it. The instructions are pretty self-explanatory and I'm pretty sure you can figure it out.

Good luck :D

**EDIT:**

Before you go cleaning out the HDD make sure this is what you really want to do! I'm not sure if you are using any other OS but if you are a Windows user try double booting. That way if you need to run something that requires Windows you can boot back into it and use it. 

If you adamantly do not want any other OS besides Jaunty on your HD just killdisk the HD and install Jaunty.
