### Post by homerhomer on 2010-02-20
So, for the longest time I noticed that my internet's DNS was very slow. This seems to have fixed the issue that I was having.  


From the terminal 

sudo apt-get install dnsmasq-base

or 

go to synaptic and install dnsmasq-base. 


Hope this helps

BTW - I'm sure there a real reason this is not working properly and there is probably a REAL fix, instead of a band-aid., but this will work for now. 

Enjoy
