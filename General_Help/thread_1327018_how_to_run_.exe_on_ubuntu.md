### Post by Mark Phelps on 2009-11-15
Since you've installed Wine, you can save yourself a lot of time and grief trying to get MS Windows programs to work by first going to the Application Compatibility website and searching for the name and version of the MS Windows app you want to run:

[http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true")

If you see no rating, or a rating less than Gold, you'll be wasting your time installing that app and trying to get it to work.

Wine is not a miracle cure -- it runs SOME MS Widows apps well, others OK, and a LOT, not at all.
