### Post by nickoli_02 on 2008-02-11
As in, copy a command and paste into a terminal? To paste into a terminal, press Shift+Ctrl+V. This method is fine for executing one command at a time. If you want to execute a series of commands easily, create a bash script. Here's a simple tutorial on bash scripting:

[http://linuxhelp.blogspot.com/2005/10/10-seconds-guide-to-bash-shell.html](http://linuxhelp.blogspot.com/2005/10/10-seconds-guide-to-bash-shell.html)

There are tons more information on bash scripting out there on the web. Remember, google is your friend.
