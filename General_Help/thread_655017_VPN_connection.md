### Post by jesushero on 2008-01-01
It usually will not appear as an icon in any of the menus. Start a terminal window and type in the name of the application (vpn or something similar I suppose, depending on what you have downloaded). If you need help in using it, in the terminal again, type man <vpn-application-name-here> or <vpn-application-name-here> --help

Hope this was helpful.
