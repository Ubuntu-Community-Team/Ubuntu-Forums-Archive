### Post by editrix on 2008-11-06
> **juan234 said:**
> I am looking for more of a way to record a browsing session.  say I am researching something and I want to save a page of a site and include it with other pages I visited.  not sure if this even exists...

Just a tip, if you're specific about what you want to do, it's easier for us to come up with answers.

Two things that might work for you:
1. Scrapbook, which is a Firefox extension that enables you to archive web pages, group them in folders, even edit or annotate them before you save them. Konqueror also archives web pages (it's built in) but you can't edit them beforehand.

2. wwwoffle, which is a web caching proxy. It will archive the web pages you visit for a time, so that you can access them offline later. But Scrapbook sounds more like what you want.
