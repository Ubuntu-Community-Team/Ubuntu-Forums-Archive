### Post by livebits on 2011-03-05
Hi all,

i have Ubuntu 10.10 64bits and i installed the new updates that Ubuntu recommended including ATI update drivers.

But when i restart it boot to nothing a black screen.

I already try "Ctrl + Alt + (F1 - F6)" and doesn't work after Ubuntu logo it goes to a black screen.

Put a  pendisk with live cd iso but it doesn't have any recovery mode option

Answer:

Problem solved, use shift while Ubunut is boot  and you will enter to a recover menu, after that i choose the first recovery menu option, installed the lasted drivers of ATI and restart. All goes normal now. 

Hope this information help more people.
