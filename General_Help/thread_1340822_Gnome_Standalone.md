Since GNOME is not a complete distro, it will not run by itself.  You also need things like a an init system, an x-server, etc.

If what you want is GNOME with as little extra stuff as possible you still need a distro, but you can look at one of the distros that allows the most customization.

Arch linux might be a good start.  It installs a very limited set of applications by default, and then you add only what is needed to make GNOME run from there.

### Post by Cope57 on 2009-11-29
Gnome did create a [liveCD]("http://live.gnome.org/GnomeLiveCd"), and also created a [installable liveDVD]("http://torrent.gnome.org/")

[Gnoppix]("http://www.gnoppix.org/") is a Live CD including a working Debian GNU/Linux extended with a GNOME 2 desktop environment.  It is a bit outdated, but it did exist as an alternative to [KNOPPIX]("http://www.knoppix.org/") which has the KDE desktop.
