### Post by Padie on 2008-04-29
I'm not much of a pro here myself when it comes to these "sessions" and xgl itself, but one way that I know how to enable it is to go into Synaptic Package Manager (System -> Administration -> Synaptic Package Manager) and search for "xserver-xgl". Install that package and if it asks to remove anything, let it. Restart your computer and see what happens then. Alternativly, install Envy and let that sort out all your graphics and Xorg woes....;)

About the sessions: I suppose there is something corrupt or not all there on your system, and therefore it is asking you to choose one of the default loaded sessions to use when logging in....maybe the above will fix it now.

Good luck!
