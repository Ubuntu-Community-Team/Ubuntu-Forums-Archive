### Post by elspiko on 2007-12-14
Depends on what you want to do.

Both rails and php are easy to install, there will be many guides  in this forum  or on google.

Rails is a lot easier to get a basic website set up and running with its database migrations and scaffolding functions, however it doesn't cope very well with alot of traffic unless you set up a web server cluster.

Dispite the fact that rails is finally becoming popular, I would suggest making the site in PHP as Rails is quite difficult to master.

[W3Schools]("http://www.w3schools.com/php") is a great beginner tutorial to PHP, and the site generally is great for web developers.
