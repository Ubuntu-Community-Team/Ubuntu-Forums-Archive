### Post by Bucky Ball on 2012-07-10
You, my friend, are well out of support. Hardy Heron, 8.04 LTS, has been dead for nearly a year and a half and I would presume that is your problem. The wake and last rites were conducted some time ago ...

Things like your Firefox and other apps would be ancient versions and anything you tried to do to them would likely fail (including installing newer versions as your kernel is too old). Your install is also a security risk.

I suggest installing the rock solid 10.04 LTS which is supported until next April (and can be upgraded to 12.04 LTS with one click when you want or it reaches end-of-life) or the new 12.04 LTS which has nearly five years support left (but you might find it problematic). 

Good luck. ;)

PS: Have you got an option in update manager to upgrade to 10.04 LTS? As support has ended so long ago, you probably haven't.

### Post by Bucky Ball on 2012-07-11
No probs. I couldn't do the upgrade on our old desktop, same prob, brain too full. I cleared space on the other drive in there and did a clean install (10.04 LTS) to that, then copied all the /home files across. Didn't take long ... what did take long was figuring how to move the data around; like those games with the missing tile where you move the pieces around. This way the old install was still stable and there in case things got hairy (which they didn't, easy). 

Good luck with it. You're going to see a few changes, but be warned; if you go for regular Ubuntu flavour it now uses the Unity desktop environment, not Gnome. Check it out first; some like, some hate. 

I use Xubuntu myself and a lot of people have changed to the xfce4 DE (which is installable from any flavour of Ubuntu and many other distros). Enjoy!
