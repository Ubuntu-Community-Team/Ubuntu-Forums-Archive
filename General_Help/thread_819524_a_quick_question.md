### Post by pointone on 2008-06-05
When using it with command-line options, such as "aptitude install <package>", there is no difference between aptitude and apt-get. aptitude simply calls apt-get.

aptitude, however, provides a (somewhat) user-friendly interface for command-line package management; apt-get does not. Try simply running "aptitude" from the command line to take a look.
