### Post by rbprogrammer on 2007-12-15
if you just did a default install, meaning you/your wife did not save files to anywhere but your own home directories, then you could just copy her user directory over to the 64bit version.

if you want to do this *simply*, then i would just boot up the 32bit version.  save her folder to a flash drive or something.  make sure you copy the whole folder.  **hidden files and all**.  then shut down and boot up the 64bit version.  if your wife already has a user created on that system, then just copy her data ( hidden files too ) into her home directory.  if she does not have a user account on that system, then create one, and then copy every thing over.
