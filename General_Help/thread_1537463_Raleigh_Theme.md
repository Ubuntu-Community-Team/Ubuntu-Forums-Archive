### Post by ankspo71 on 2010-07-23
Hi,
From my experience, the themes that use the fewest images are normally the lightest. It is easier on computers to render a theme's colors than it is to render a theme's set of images.

Try running the command 'top' in the terminal then:
Apply a theme then watch the cpu and memory usage while the computer idles for a few minutes. Then do it again for the next theme, and the next one and so on.  You should be able to see the difference between themes this way on lower end computers.

I don't know if Xfce allows anyone to choose theme engines, but I know that in Gnome, choosing a different theme engine can make a big difference too.

Xubuntu is relatively lightweight already, but it can get even lighter if you install some lighter applications. For example, if you have brasero installed, you might like using xfburn. If you have open office installed, you might want to try abiword. There are more alternatives to consider but I don't know what Xubuntu has installed. Also chromium-browser might be a lighter weight browser than firefox too.
Hope this helps.
