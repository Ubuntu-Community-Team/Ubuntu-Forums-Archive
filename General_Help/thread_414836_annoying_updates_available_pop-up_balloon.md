Anyone know if there's a way of disabling the balloon that comes up to let you know there are updates available?

### Post by smurphs on 2007-04-20
Found it! For anyone else with a simliar grievance, go to applications, add/remove, preferences, updates.

I've now set mine to a weekly check. Much more acceptable!
