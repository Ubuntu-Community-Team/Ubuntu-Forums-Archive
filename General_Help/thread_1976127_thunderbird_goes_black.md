### Post by TiberiusT on 2012-05-19
I think I've fixed this TBird/Lightning/Google calendar combo thing. TBird 12.0.1 / Lightning 1.4

Don't use the Provider for Google Calendar plugin - yu no longer need to. Instead use CalDav like this: [http://support.google.com/calendar/bin/answer.py?hl=en&answer=99358#sunbird](http://support.google.com/calendar/bin/answer.py?hl=en&answer=99358#sunbird) . Just ignore the fact that it refers to Sunbird and carry on regardless.

I now have this setup on 2 x 12.04 machines, and on XP (groan...) in the office and for the last 3 days it's worked perfecto. Now it's time to lock down and stop these crazy Mozilla updates :)
