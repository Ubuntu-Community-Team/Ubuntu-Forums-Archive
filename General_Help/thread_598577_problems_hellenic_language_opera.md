### Post by Claus7 on 2007-10-31
Hello,

In some sites I have noticed that opera is difficult to recognise the hellenic (greek) fonts. In order to avoid that then do the following :

go to tools -> Preferences -> General -> Select your preferred language for Opera and Web pages.
There choose Greek [el] and then hit the tab Details.
In the tab : Encoding to assume for pages lacking specification use : iso-8859-7

The problems that still remain are :
I cannot open pdf documents in macromolecules web page.
Some texts are cut and continue in the next line, making the text more cumbersome.
That happens also to emails.

If someone has done something better I will be glad to know.
