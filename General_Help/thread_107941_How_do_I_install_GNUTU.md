### Post by AmboyGuy on 2005-12-24
after you've downloaded the deb package
( [http://gnutu.org/download.php?id=ubuntu/gnutu_2.2-1_i386.deb](http://gnutu.org/download.php?id=ubuntu/gnutu_2.2-1_i386.deb) )
in terminal change to the directory you downloaded it to and type ```
 sudo dpkg -i gnutu_2.2-1_i386.deb 
```

### Post by chimera on 2005-12-24
download the file AmboyGuy told you, and put it on your desktop

open terminal

type:
cd /home/<YOUR USERNAME HERE>/Desktop      and press ENTER   (replace <YOUR USERNAME HERE> with your username)
sudo dpkg -i gnutu_2.2-1_i386.deb              and press ENTER
type the password and press ENTER


gnutu should now be in your applications->accesories menu.
