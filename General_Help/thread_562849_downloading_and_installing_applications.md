### Post by SeanTater on 2007-09-29
Most likely, you are downloading a program for Windows. Windows executables will not open in a text editor correctly, and they won't run correctly on Linux either. Downloading files like that is generally not the preferred method of installation for Linux.

For program installation in linux, we have an entirely different method where you choose from a list of many thousands of available programs, and a program will install it for you.

More information about that program can be found here: [https://help.ubuntu.com/community/SynapticHowto](https://help.ubuntu.com/community/SynapticHowto)

You don't need Realplayer to play RealPlayer files, to do that, look here [https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)

I hope you find that everything you do in Linux is this unexpectedly easy.
