### Post by fballem on 2009-04-01
If you Google on the following search terms: conflicker worm linux ubuntu, you will get a long list of articles, all of which indicate that conflicker exploits a bug in Windows.

If you are running pure Linux, then you should be okay.

I don't know the impact if you are running dual-boot (Windows / Linux), running WUBI (ubuntu inside Windows), a Windows Virtual Machine, or WINE. Since I am running pure Linux (ubuntu), I haven't worried about it.

I do have some Windows machines in the house, and spent the day making sure that Windows and the Anti-Virus programs are all up to date and running. I have also made sure that I can access the Microsoft sites (including Windows Update from within Internet Explorer) and Symantec, since one of the symptoms is that users are blocked from these sites in their browser.

I have also made sure that any important data is backed up (something that I do anyway, having learned the hard way many years ago).

Hope this helps,
