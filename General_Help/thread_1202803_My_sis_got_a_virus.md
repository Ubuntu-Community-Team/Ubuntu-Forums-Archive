### Post by ghat on 2009-07-02
hi

you should do what ubuntu does natively in windowz.

ubuntu natively does not allow "standard/default" users to have administrative 
privileges, whereas, windows natively allows "standard/default" users to have administrative privileges. 

This is what I recommend, 

1. clean your sis's windows with whatever antivirus works... as described by the other folks

2. Create another windowz user account for your sis, and remove all administrative and power user privileges from the account, and let her use it routinely...

3. Tell her to use the admin account only to install software which she knows., and make sure you enable a "non-blank" password for the admin account. Best is to create another account called say "gandalf" with a "non-blank" password and add it to the group administrators, and then reboot login to that account and disble/lock  the "Administrator" account.

3. Install some anti-virus software AND  these softwares..
**[*Spybot*-S&D!]("http://www.safer-networking.org/")**

**[*CCleaner*]("http://www.ccleaner.com/")**

**[*JkDefrag*]("http://www.kessels.com/Jkdefrag/")**

whever possible install their services,(tea-timer) or add scheculed tasks to routinely cleanup crap and defrad the hdd, this will prevent future problems...


There is no need to install ubuntu, if your sis is used to windows, albeit it is a very much better and robust operating system.
