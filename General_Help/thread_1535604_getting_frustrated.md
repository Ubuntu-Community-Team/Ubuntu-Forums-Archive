### Post by ankspo71 on 2010-07-21
Hi again,

Here is a new project aimed at newer Ubuntu users [http://ubuntu-manual.org/](http://ubuntu-manual.org/)
They have a free downloadable PDF manual.  It's a good project that covers the basics that any new user may need to know about Ubuntu.

Although, If you are wanting to learn about doing things using commands, then I could also recommend "Ubuntu Linux Toolbox: 1000+ Commands for Ubuntu and Debian Power Users". The title says "power users" but I personally think it could be useful for newer users and advanced users alike. It's a pretty straight forward book that goes step by step, and not too complex for me... the average ubuntu user who is interested in learning about commands. I just got it a few days ago and have only read a few pages and skimmed through the rest, but it looks to me like a valuable resource for any debian or ubuntu based linux distribution.

With that said, there are many different types of resources out there for Ubuntu, both free (online resources) and commercial (books)... one of the things I like about Ubuntu.
