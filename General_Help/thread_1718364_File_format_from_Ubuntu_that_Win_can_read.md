### Post by SeijiSensei on 2011-03-31
The most universal format is plain text, though you obviously won't have fonts or things like that.  Save in plain text with a .txt extension.  Windows will open it with Notepad.

If you want a formatted document, you have three good choices.  One is HTML.  Save the file with a .htm extension; double-clicking the file in Windows will open it in a browser.

You can save to the .doc format that Microsoft Word uses, but that asssumes everyone has either a copy of Word or the Word Viewer.

Finally there's PDF.  That would probably be my choice since most computers these days come with Adobe Reader installed at the factory.  OpenOffice has an export to PDF option that works great.

Still I'd say the best solution for you is to have a website.  Put your recipes there, with pictures and maybe a video or two, and just give those adoring ladies the URL.  Who knows, maybe you can become another Internet sensation like [Bubbe](http://www.feedmebubbe.com/).

### Post by Naggobot on 2011-03-31
If ABW refers to [Abiword]("http://www.abisource.com/help/en-US/interface/dialogsavelinux.html") then following is recommended by the help

> The most easily exchanged format is Rich Text Format (.rtf).So you need to use Save As functionality and save as .rtf file. If windows users have Word then they can open the .rtf quite well. 

.pdf is fool proof but can not be edited by the recipient. Also a plain text file .txt is pretty fool proof, can be edited by recipient but you can not include any formats and I do not know if you can save it from Abiword. Plain text file has the benefit of being editable. 

Note that you can not convert native Abiword document or .rtf file just by changing the extension of the file. You must save to correct format using your editor.
