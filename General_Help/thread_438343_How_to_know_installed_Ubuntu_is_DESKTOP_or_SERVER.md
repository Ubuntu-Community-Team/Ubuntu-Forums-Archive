### Post by beerorkid on 2007-05-10
well according to the [official upgrade method.]("http://ubuntuforums.org/showthread.php?t=286599")   It would really not matter.  And I am not sure why you would need the CD. 

Also going from 6.06 to 7.04 might be a bit of a stretch, that is going up two releases.  Some folk have issues when upgrading, especially if it is a heavily modified system.

I would recommend copying off important data and doing a fresh install.  I would think you would want desktop cuz server has no GUI.

[this thread]("http://ubuntuforums.org/showthread.php?t=361919&highlight=difference+server+desktop") pretty much explains it.  There is little difference between desktop and server besides a few packages.
