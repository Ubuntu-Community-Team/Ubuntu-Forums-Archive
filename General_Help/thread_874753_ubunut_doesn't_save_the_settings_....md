### Post by drs305 on 2008-07-30
You can add this line to fstab (after making a /media/*mountpoint*):

```
#Entry for /dev/sda2
/dev/sda2  /media/*mountpoint* auto,users,uid=1000,gid=100,utf8,dmask=027,fmask=137 0 0
```

Once you have saved it:
```

sudo umount /dev/sda2
mount /dev/sda2

```
