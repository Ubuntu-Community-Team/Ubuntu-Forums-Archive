### Post by ibuclaw on 2009-09-24
> **credobyte said:**
> ```
system("read pause"); /* Will work in Linux */
```

No ?

OK, lets just put it this way:
Depending on the OS you are running, system() calls will more or less perform the following actions:
[LIST=1]
[*]suspend your program
[*]call the operating system
[*]open an operating system shell (relaunches the O/S in a sub-process)
[*]the OS must now find the command specified in the first argument.
[*]allocate the memory to execute the command
[*]execute the command and wait for it to exit.
[*]deallocate the memory
[*]exit the OS shell
[*]resume your program
[/LIST]

Or you could just use:
```
getchar();
```
And the program will do exactly what it says on the tin.

Iain
