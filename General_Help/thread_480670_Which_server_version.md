### Post by AlexThomson_NZ on 2007-06-22
If all you are using it for is a web-server, and it is running ok, just stick with 6.06.

Unless you NEED the extra functionality of more up-to-date programs, don't bother upgrading to 7.04.

6.06 will still get security updates, so will be just as secure as 7.04
