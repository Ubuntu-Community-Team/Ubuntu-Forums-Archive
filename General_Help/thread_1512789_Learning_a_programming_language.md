### Post by potat on 2010-06-18
If you want to learn to program in order to mess around with the OS and it's components, shell scripting is interesting to learn and can really boost your productivity. 
A shell script is basically a list of commands (the same as used in the Terminal) used to accomplish a task. Anything you can do in the terminal, you can do with a shell script, and vice versa.

Check this guide out: [http://tldp.org/LDP/abs/html/](http://tldp.org/LDP/abs/html/)

If you are more interested in programming for programming's sake (now or in the future), I recommend Python. It's intuitive and has a lot of capabilities and will

C and C++ are great languages, but really only necessary for creating software. If you are programming for personal use or enjoyment, start with higher-level (and easier to interact with!) languages like those above.
