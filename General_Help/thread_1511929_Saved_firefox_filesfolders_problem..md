### Post by lovinglinux on 2010-06-19
Use [Mozilla Archive Format](https://addons.mozilla.org/en-US/firefox/addon/212/) extension. It allows you to save the entire page as a single compressed file that can be opened by Firefox like a regular html file. It also allows to save it as MHT, to be opened by IE.
