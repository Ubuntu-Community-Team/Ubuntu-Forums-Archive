### Post by abrooke18 on 2009-02-14
If you're having general problems with Firefox, especially after an upgrade, your profile may be corrupt.

Symptoms include:
Grayed out buttons: forward, next, etc.
Home page not loading.
History not listing.

Warning: the following procedure will delete ALL personal Firefox Data!

To fix this, back up your bookmarks, then:

1. Click Places > Home Folder.
2. Click Ctrl-H to show hidden files.
3. Go to the .mozilla > firefox folder.
4. Delete the profile folder: this has an odd name that ends in .default, for example, jy434pnp.default
5. Delete profile.ini file.

You'll then need to set up the browser again, including:
Toolbars
Buttons
Tools > Addons

This worked for me.
