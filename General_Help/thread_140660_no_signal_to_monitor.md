Either the X server is not running - do CTRL-ALT-F1 and see if the text login appears - or it is running at the wrong refresh rate/resolution.

Do a forum search for fixing either of these issues.

You may also need to check if the PC has two video outputs, and Ubuntu is using the wrong one.
