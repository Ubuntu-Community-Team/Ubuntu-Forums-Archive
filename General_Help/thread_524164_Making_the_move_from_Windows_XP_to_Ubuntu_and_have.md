### Post by HermanAB on 2007-08-12
The short answer is no.

The long answer is sort-of.  Thunderbird can sort-of import Outlook mail, but if you have more than just a few messages it will fail on something and if you only have a few messages then why bother?  The same applies to the utility called olm2mbox.

Evolution works quite well with Exchange.

You can also email everything in your old system to yourself on the new system.  This is the only conversion method that is guaranteed to work, but then all the From headers will be yourself, but at least the contents of all messages will be there, you can search for stuff.

The easiest solution is just to forget about moving mail and just start using the new system.  The problem is self correcting, since after a few weeks you don't need the old mail anymore, so there is not much point in transferring it all with the consequent enormous effort.

Cheers,

Herman
