### Post by ryanhaigh on 2008-10-07
there is a commandline program called fdupes that will help you find duplicate files, I have used it many times for photos. Be aware that it will only detect exact file duplicates which means if a track was encoded differently etc. it will not be picked up.

For tagging and renaming you might want to look at picard or another musicbrainz app. These work by creating an audio signature and comparing to an online database.

[http://musicbrainz.org/](http://musicbrainz.org/)
