### Post by QIII on 2010-06-09
Welcome to Ubuntu!

You will find that you will need to learn much, because you are entering a different world in some respects.

You have become accustomed to doing things in Windows, and that is well and good.  There is absolutely nothing wrong with that.

But Linux is not Windows, so be prepared to depart from the road to which you have become accustomed.  The new road will become familiar in time.

On to your questions.

1.  On installation, Ubuntu will install drivers for nearly all common sound cards.  We can help if something is amiss.  As for your webcam, I'd have to research compatibility.

2.  Yes.  I would be very surprised if not.

3.  Sorry.  Your question was a bit confusing.  It might be easier to rephrase that.  Realizing that English is likely not your first language, we'll try to help, but I'm not sure I quite get all of that.  Part of it I can answer:  You may have as many disks attached to your motherboard as it will allow and use them all.  I have all four of my SATA headers on my motherboard connected to drives.  You may want to be sure you have a power supply sufficient for the load if you want to spin them all at the same time.

4. There are literally thousands of software applications available for Linux for nearly everything you could possibly dream of.  There are alternatives to most Windows applications and even some that are compatible with Windows applications.  Unfortunately, applications that are not designed for cross-platform use and are specifically designed to run on Windows will not run in Linux.  

Please don't ask people to send you responses via email.  If correspondence is conducted that way, others will not benefit from the solutions.  Please use the forum so that others may learn.

Happy Ubuntuing!
