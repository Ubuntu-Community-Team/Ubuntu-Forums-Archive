### Post by makem2 on 2017-11-29
This machine does not have function keys to control the brightness.

Power manager does not have any way to control the default brightness whilst not idle. I have looked through all the settings available elsewhere but none are of any use to turn down the bright white windows.

### Post by makem2 on 2017-11-29
Solved.

Clicking on the panel battery icon brings up a menu. This contains a slider for screen brightness.
