### Post by Hallvor on 2007-05-14
aMule does not support files larger than 4 gb. eMule clients on the same network (from 0.47a,) does. eMule runs with Wine, but not entirely without bugs. I suggest you download the file through your Windows-partition, or you could run the latest version of eMule through Wine.
