### Post by scorp123 on 2007-10-28
> **Russellvg said:**
> if i make a ubuntu terminal server, can a windows pc connect to it??? Define "terminal server" please? I am asking because it seems many people understand different things what "terminal server" means. Generally speaking: Yes, there are server programs such as VNC that would allow a Windows machine to connect to it and get a desktop session -- provided that the Windows machine has a suitable VNC client. Same for "terminal-only" sessions via SSH: you need to have a suitable client on the Windows side, e.g. "putty" works great. For copying files between a Windows client and a Linux server running SSH you could use programs such as "WinSCP".
