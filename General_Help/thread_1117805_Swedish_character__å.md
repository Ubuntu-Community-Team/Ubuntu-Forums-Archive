### Post by 3Miro on 2009-04-06
The answer is System -> Prefs -> Keyboard, then go to the layout tab and add a Swedish layout. I have done that so I can type in Cyrillic, almost all characters are different. I know people write in east Asian languages as well. Just add the layout that you want the shortcut that lets you switch.

### Post by change_mode on 2009-04-06
Or you can press: CTRL+Shift+U
Then enter '00E5' (and space after that).

= å

For a list of all char codes, go to Accessories -> Character Map.
