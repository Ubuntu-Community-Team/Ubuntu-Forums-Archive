A friend of mine wanted to give Ubuntu a go, so I set up a little ten gig. partition for Ubuntu. My friend is now using Linux much more frequently and needs some more space. If I unallocated a chunk of the Windows partition, how do I add it to the Ubuntu partition? ---

### Post by taurus on 2007-11-14
If you want to merge it, you need to use gparted either from the LiveCD or GParted LiveCD to do that.  You cannot do it while running Ubuntu.

[http://gparted-livecd.tuxfamily.org/](http://gparted-livecd.tuxfamily.org/)
