### Post by elbowgeek on 2006-11-25
I'd guess your query found all of the similar names using google as part of their domain registration.  You'll find that many large corporations will register these odd variants so that others can't cyber squat on similar domain names which could be used in phishing attacks and the like, but not all can be accounted for.
