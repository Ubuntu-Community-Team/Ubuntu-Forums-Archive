### Post by hxx4 on 2006-06-22
I noticed that 30MB on the Ubuntu desktop CD is "Open Source Windows Applications". The menu to install these programs is very nice, but I find this hardly useful. Asside from some of these programs being out of date, I can't imagine myself installing any of them from the Ubuntu CD. The only time I could think of that one might want to install the programs from the CD is if he or she has dialup Internet and does not want to download them. Otherwise, I think there are much better uses for the space such as including additional packages with ubuntu.

### Post by cleentrax on 2006-06-22
Since it takes up less than 5% of the CD, and since 90% of people use Windows, I think it's a great idea.

Here's the migration path:

1. Try open source apps in Windows
2. Try the Live CD
3. Install Ubuntu

Step #1 makes #2 and #3 more likely, since Firefox, Open Office, the GIMP, etc., will be familiar to the user.
