### Post by Flying caveman on 2009-12-09
did you read this part of it? 

>  olegueret replied 9 months ago
This is a problem with the wmode transparent that is used by the Flash content in 12Seconds site. The proper solution is that they change it to wmode window (for the 12Seconds crew, more info in Flash Player 10 Security Changes).

In the meantime, for end users, there is a possibility to allow permission to the 12seconds site to use camera & micro using an external Flash configuration page. Just go to this page, search the website "12seconds.tv" in the Pricacy Tab -it has an icon of a monitor with an eye inside-, select it and then choose "Allways allow". Now return to 12Seconds site and reload it ("settings" popup menu will still be disabled, but it should be possible to use camera).
