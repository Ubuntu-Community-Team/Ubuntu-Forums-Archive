### Post by sports fan Matt on 2009-02-11
How easy is it to set a default Font and heading/style in Open Office 3?

---

### Post by jonlemur on 2009-02-11
If it's like 2.4, just go to Tools->Options->OpenOffice.org Write->Basic Fonts and change them there.

---

### Post by scouser73 on 2009-02-11
Click **Tools > Options > Basic Fonts (Western)** in the Default font section, tap the drop-down menu and choose a font, and then click ok. For the heading you can do the same as the Heading drop-down menu is underneath the default font drop-down menu.
