### Post by hrickh on 2010-09-26
Gnome has had an issue for a long time now - if you copy something, then close the application before you've pasted what's in the buffer, it's gone. It's a completely stupid issue that should have been fixed long ago.

The way around it is to install a clipboard manager.  You've installed Parcellite which should solve your issue. Personally I use Pastie. they're both good clipboard managers.
