### Post by grahammechanical on 2014-12-10
These system error messages usually offer to show us Details of what is the cause of the problem. I suspect that the finger of blame is pointing at Grisbi. After all, you say it is locking up and also freezing the whole system.

What is Grisbi? Is it in the Ubuntu Software Centre? Was it installed from a PPA or by downloading it from a web site? An upgrade from one version of another will usually upgrade the applications in the Ubuntu Software Centre but not applications installed from a PPA and never applications downloaded from web sites.

For all I know you could be using an application that is not compatible with Ubuntu 14.04. You could try un-installing and then re-installing Grisbi. But a lot will depend upon Grisbi being packaged for Ubuntu 14.04.

If the problem is being caused by Grisbi then all that the maintainers of Ubuntu can do when they receive these error reports is to pass them on to the developer of the software. It is up to the developers of software to fix the bugs in their software.
