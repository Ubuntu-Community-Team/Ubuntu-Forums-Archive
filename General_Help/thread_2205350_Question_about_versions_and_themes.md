### Post by ibjsb4 on 2014-02-13
Here is some excellent documentation on the Unity desktop.

[http://ubuntu-manual.org/downloads](http://ubuntu-manual.org/downloads)

Desktop environments can be easily changed.  The old desktop look (like v9) can be added to a newer version by:

```
sudo apt-get install gnome-panel
```

Then reboot and at the login window, click on the icon and choose your desktop environment.

[https://help.ubuntu.com/community/PreciseGnomeClassicTweaks](https://help.ubuntu.com/community/PreciseGnomeClassicTweaks)

Another option would be to install Xubuntu which looks like v9 out of the box.
