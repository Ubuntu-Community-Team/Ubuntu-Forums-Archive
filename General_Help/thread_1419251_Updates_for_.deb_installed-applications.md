### Post by Enigmapond on 2010-03-01
If you manually install like that. You will have to manually install any updates as well. They will not show in the update manager. There is no repository for Picasa as it's Google and not supported. However you can install the Chrome PPA and get daily builds.
[http://www.ubuntugeek.com/install-chromium-google-chrome-web-browser-in-ubuntu.htm]("http://www.ubuntugeek.com/install-chromium-google-chrome-web-browser-in-ubuntu.html")l
