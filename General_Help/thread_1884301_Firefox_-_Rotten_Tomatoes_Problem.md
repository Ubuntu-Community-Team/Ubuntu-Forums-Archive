Rotten Tomatoes looks perfect for me. I have Ubuntu 11.10 with Firefox 8. 

Perhaps this is just a bad timing problem? If not, try restarting Firefox with all add-ons disabled and see if that fixes it (it's under Firefox's help menu). If it looks fine after that, then it must be one of your extensions.
