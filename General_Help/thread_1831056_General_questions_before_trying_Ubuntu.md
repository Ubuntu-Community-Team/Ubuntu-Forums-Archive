### Post by Mark Phelps on 2011-08-23
Browsing the web (as long as you do NOT expect to use IE) and watching movies are not going to present any problems as Linux supports all the major Browsers and many of the video codecs, with a variety of video players (I've found VLC to be among the best).

Gaming is entirely another story -- especially with the newer Windows games that use DirectX 10.  IF your video drivers don't support that, even if you can install and start a game, the performance will be so bad as to be unusable.  In general, Linux is not a good platform for playing Windows games.

### Post by 3rdalbum on 2011-08-23
Ubuntu doesn't come with codecs or DVD playback installed, but these are easy to install - check out the Medibuntu website for an easy way to get them installed and keep them updated.

Most open-source software you're familiar with is also available on Linux. Most commercial software isn't, but you'd be surprised how often there are equivalent programs available for Linux, all free. The best place to start looking is the Ubuntu Software Center program that comes preinstalled on Ubuntu.

It's possible to run SOME Windows programs using a piece of software called Wine, but really I recommend leaving it as a last resort and just trying to use native Linux programs wherever possible. Wine doesn't work very often.

### Post by grahammechanical on 2011-08-23
What is called the Live CD will boot and give you the option to Try Ubuntu or Install Ubuntu.

The Try option will give you access to all the programs that would be installed with the Install option and you will be able to access files from your Windows partitions. so, you can try out the music and video players.

Keep in mind that Ubuntu is open source and so proprietary drivers and codecs are not included on the Live CD but can easily be installed during the installation process if you so choose and after installation through Additional Drivers or the Ubuntu Software Centre.

If you choose to install Ubuntu the installer will check out your hard disk and offer you options. See the links of this page:

[http://www.ubuntu.com/download/ubuntu/download]("http://www.ubuntu.com/download/ubuntu/download")

a) You can install alongside another operating system in which case your hard drive will be partitioned and the new partitions formatted but not the whole drive like in Windows. At boot time you will get a menu to choose an OS from.

b) You can install using the whole disk, In which case the complete drive will be formatted and partitioned so that only Ubuntu is on the disk. Be careful with this option. Once you have removed Windows you cannot get it back unless you have Windows install disks. Recovery disks are not enough

c) You can do something else. This is where you do the partitioning. Not a good idea until you have some experience with using the partitioning tools that come with the installer.

Regards. I hope this helps.
