### Post by Dennis N on 2017-01-17
**.bashrc** is just a plain text file. you can use a text editor (like gedit, mousepad, or geany) to open it, add the line at the bottom and re-save. You could also use an in-terminal text editor like nano if you are acquainted with using one. But you shouldn't use a word processor for this task.
