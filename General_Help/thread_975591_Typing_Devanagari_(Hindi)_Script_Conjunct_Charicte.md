Hi, I suppose my answer is coming really late, but I'm also learning Hindi and had the same problem. 

As this thread shows up in Google when searching "Ubuntu how to type/write Hindi", I think it may help others anyway.

From Wikipedia:
A final consonant is marked with the diacritic &#2381;, called the vir&#257;ma in Sanskrit, halanta in Hindi, and a "killer stroke" in English. This cancels the inherent vowel, so that from [SIZE="5"]  &#2325;&#2381;&#2344;&#2351; [/SIZE] knaya is derived [SIZE="5"] &#2325;&#2381;&#2344;&#2351;&#2381; [/SIZE] knay. The halanta will often be used for consonant clusters when typesetting ligatures is not feasible.

Well the solutions is simple. For example, to type [SIZE="5"]  &#2360;&#2381;&#2340; [/SIZE] instead of [SIZE="5"]  &#2360;&#2340; [/SIZE], cancel the inherent vowel. Type [SIZE="5"]  &#2360; + &#2381; + &#2340;.[/SIZE]

The halanta &#2381; is at the same place than the "D" key on an English keyboard layout.
