### Post by cgroza on 2011-08-11
Your best bet on Linux is WINE. It is a compatibility layer that might be able to run your applications. It is kind of hit and miss. Check the WINE database to see if your applications are supported.

If they don't work, you could try looking for a native alternative or a dual boot/virtual machine.

### Post by haqking on 2011-08-11
first of all and preferable is to see if your .exe have Linux versions such as the following:

MS Office = Libreoffice or Openoffice
Adobe Photoshop = Gimp
Media Player = Totem or VLC


etc etc

Then if not you can choose between Wine see here [http://www.winehq.org/](http://www.winehq.org/) 

or look at Virtualisation such as Virtual Box [www.virtualbox.org](www.virtualbox.org) i personally prefer virtualisation as .exe prefer there native environment so better compatabiliy than Wine

or finally Dual Boot between Windows and Linux.

Both Linux and Virtualisation require you to have a licenced copy of Windows for installation
