### Post by fahimdadream on 2008-05-03
Hey, 

To get compiz to work you should try going to System>Preferences>Appearance, then on the last tab, click extra, if that works then in synaptic install 

compizconfig-settings-manager

then System>Preferences>Advanced Desktop Effect Settings for the plugins.

If for some reason it says, "desktop effects cannot be enabled" you have to install drivers for your v.card probably restricted drivers. If this is the case, search the forums first, so you know how to solve potential problems you could run into. 

Good Luck.
