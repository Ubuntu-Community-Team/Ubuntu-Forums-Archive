### Post by Tamlynmac on 2010-10-27
To download a .wmz file, simply save it as a download file (do not try  and open said file or copy/paste). Then follow the instructions below.

A .wmz file is a Windows media zip file. To look at and extract files,  simply download the file and upon competition of download the Archive  Manager should open (see "**Note**" below). The archive manger will  permit you to extract any and all of the zipped contents. Since this  isn't a single file, but rather a zipped file, OO wouldn't know what to  do with it.

I'd suggest you make a folder for extraction purposes, keeping all your  unzipped files in said folder. You may even consider sub-folders for  each .wmz file. Thus assuring easy identification.

**Note**: Should the Archive Manager not open - open Archive Manager  from menu and select "OPEN" in toolbar menu. Find downloaded .wmz file  from the browser window and double click on it.

* Should one have trouble opening a .wmz with the Archive Manager. They  could rename the file type from .wmz to .zip using the Nautilus file  browser. For example, test.wmz (rename to) test.zip. I didn't find that  necessary as the Archive Manager recognized and opened for extraction,  all the .wmz files I downloaded.

Good Luck & hope this helps.

### Post by Tamlynmac on 2010-10-27
One other quick thought, if you just wish to place the file into a document in OO - Try:

OO/toolbar/insert/file.

Find the .wmz file in the browser and double click on it.
I'm not sure if it will insert said file, but it should.

If it doesn't, use your Nautulis File Browser and rename the file type to ".zip" instead of ".wmz". Then try and insert it into the document.

Renaming it, should not have any effect on the opening of said file by another system.

Good Luck
