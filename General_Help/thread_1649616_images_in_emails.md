### Post by rsansores on 2010-12-21
You need to enable HTML view otherwise you will get only plain text and you must be sure that the privacy settings allow displaying images.

The first point is really easy, you can go to preferences and select "HTML view" as default or in any message click the gray vertical line in the left side of the message panel to toggle between plain text or HTML (is a little hard to explain but is really easy).

The second task is go to privacy settings and tick the option "allow displaying images".
