### Post by charles.figura on 2008-08-04
I've got a problem with Thunderbird's file dialog, but the Mozilla guys say it's a Ubuntu problem and told me to try over here.

I'm running Thunderbird on Linux Ubuntu. When I go to save a file or attach a file, the file browser shows all hidden files. I don't know how to turn OFF showing hidden files. Can anyone help? I *don't* want to see the hidden files - there are too many and it's a royal pain.

Fellow on the Mozilla forums said that Thunderbird just executes a system call for the file dialog, so the configuration has to be somewhere on the Ubuntu end.  Any ideas?

---

### Post by pluviosity on 2008-08-04
Right click on the list of files in the save dialog, and unselect "Show Hidden Files."

---

### Post by charles.figura on 2008-08-04
That did the trick - big thanks.  I didn't think of looking *there*.
