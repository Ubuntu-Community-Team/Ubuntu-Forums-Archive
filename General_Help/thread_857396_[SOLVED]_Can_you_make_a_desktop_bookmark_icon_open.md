Install mozilla prism:

[http://wiki.mozilla.org/Prism](http://wiki.mozilla.org/Prism)

The latest version is 0.9 but we have version 0.8 in repos. I suggest you install the one in the repos.

Prism is the small brother of firefox. It's exactly used for the reason you want: run different sites in their own browser. They are called webapps.

The added benefit is that prism and firefox does not share cache & settings etc. so it is also safer.
