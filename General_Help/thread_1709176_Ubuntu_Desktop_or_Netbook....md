### Post by JC Cheloven on 2011-03-17
I have 2 dell mini9 netbooks (8 and 16Gb solid state disk, 1Gb ram). One has the vanilla ubuntu install, and runs great. The other has an [lubuntu]("http://lubuntu.net/") install and runs ever better (well, a bit faster even, that's all; differences become really apparent only in ram-scarce environments).

I personally dislike the netbook linux versions. I don't want my netbook to look as a pda, but it's a matter of taste.

If you have 1Gb ram, I think you'll be fine with the normal version of ubuntu. Even 512Gb should do just fine. If you are shorter of ram, I'd go for something really light, as lubuntu. It's a bit more tricky to configure but nothing huge. You can even think of puppy linux, if you want it really light but still an usable system.

Hope this makes some light ;)

### Post by TechSupportx86 on 2011-03-18
I would like to point out gnome isn't made for screens smaller than 1024x768, so if you have a 1024x600 screen "like i do" it is near impossible to setup alot of things (mail being one of them).

My advice is if your screen resolution is 1024x768 (generally a 10.1 Inch) or higher, then use whatever you want, but if you happen to use an early eee or an 8.9 inch acer, go with netbook. 

as far as hardware goes my acer aspire one runs it just as well as any other computer i've tried 10.10 on. specs below:

1.6Ghz Atom (HT)
1.5GB
250GB SATA (100/100/30)
10.10 / Win 7 Dual-boot
8.9 inch 1024x600 Screen
