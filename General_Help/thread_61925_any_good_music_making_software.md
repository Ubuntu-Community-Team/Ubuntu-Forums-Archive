### Post by fractalvibes on 2005-09-02
For algo software for generating MIDI, take a look here:
[http://www.nosuch.com/tjt/software.html](http://www.nosuch.com/tjt/software.html)
for KeyKit - an extensible algo software that has both windows and Linux versions.
Very flexible piece of software capable of doing many algo things.

Also check : [http://tamw.atari-users.net/steem.htm](http://tamw.atari-users.net/steem.htm)    There is an Atari emulator called XSTEEM for Linux.  Some of the algo MIDI software written for the Atari all those years ago is quite excellent.  I have used the windows-targeted emulator called STeem that is very good.

MIDI in no way will ever work on my little 333mhz system, so I have to explore these things in Windows.  But I hope to hear of people using these tools on Linux with success.
