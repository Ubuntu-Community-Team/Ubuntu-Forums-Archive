### Post by grahammechanical on 2016-04-11
The User Interface is also a shell. There are different styles of UI & each UI will allow a certain degree of modification. But only to the point decided by the developers of the UI. Ubuntu Unity has very limited customization options when compared with Kubuntu or Ubuntu Mate. Most of the utilities in Ubuntu are Gnome utilities so customization of utilities comes down to what the Gnome designers decide is allowable for their software.
