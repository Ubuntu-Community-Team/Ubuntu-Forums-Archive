### Post by prem1er on 2009-11-08
You need to tell your computer to boot the live CD in the bios.  When your computer is on...

1. insert the cd
2. reboot 
3. hold down the esc key (most asus it is the esc key)
4. you will be given the option to boot from the cd
5. boot into the cd and you can install from there.

Several questions come to mind --
