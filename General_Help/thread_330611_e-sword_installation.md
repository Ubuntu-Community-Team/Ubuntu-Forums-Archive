### Post by C-A on 2007-02-08
Install gnomesword.

First install: 

[ i386](http://mirrors.kernel.org/ubuntu/pool/universe/s/sword/libsword5c2a_1.5.8-8_i386.deb) or  [amd64](http://mirrors.kernel.org/ubuntu/pool/universe/s/sword/libsword5c2a_1.5.8-8_amd64.deb) or [powerpc](http://mirrors.kernel.org/ubuntu/pool/universe/s/sword/libsword5c2a_1.5.8-8_powerpc.deb)

to match your architecture.

Then install gnomesword via synaptic.
