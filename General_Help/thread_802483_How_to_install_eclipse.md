### Post by kkkkdddd on 2008-05-21
You do not have to install anything.

Simply open folder with eclipse and run eclipse file.

Hence you unpacked eclipse to ~/eclipse:
```

cd ~/eclipse
./eclipse &
```

Btw. In my opinion, Netbeans is a better IDE. Both are free of charge so you can try Netbeans too.
