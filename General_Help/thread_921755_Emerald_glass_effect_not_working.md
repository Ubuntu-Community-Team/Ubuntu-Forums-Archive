### Post by Genius314 on 2008-09-16
1. Install CCSM:
> sudo apt-get install compizconfig-settings-manager

2. Go to System>Preferences>CompizConfig Settings Manager

3. Enable the "Blur Windows" plugin.

You may have to play with the settings a bit to make it look nice. :)
