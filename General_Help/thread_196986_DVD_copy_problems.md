I have been using k3b, it is in the repository: the discription is:
A sophisticated KDE CD burning application
K3b is a GUI frontend to the CD recording programs cdrdao and cdrecord.
Its aim is to provide a very user friendly interface to all the tasks
that come with cd recording.

It can be used to copy CDs and burn:

### Post by JimS on 2006-06-17
...
I'm still using WinXP for backing up DVDs.
Most commercial DVDs are encrypted.
Therefore I use a varity of free programs to decrypt them.
For per 2006 releases I usually use DVD Decrypter.  This program
is no longer supported, but was so very good origanally that
many still use it.
DVD Fab Decrypter is now the standard.  It can tackle almost
all post 2005 commercial DVDs.
But sometimes the DVDs have other problems for which I use
IFOEdit to correct IFO files and VobBlanker to correct VOB
file issues.
I think all these programs can run under WINE, but I have
not learned how to use WINE as yet.
Good luck.  I hope you get WINE working for you.  Others have.
