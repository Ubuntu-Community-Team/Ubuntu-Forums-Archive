### Post by L815 on 2010-11-27
If you are using a downloaded ISO, you don't need to select it from the top option. Just select the radio option for ISO, browse and select your downloaded ISO.

Also, Make sure you format your USB first.


There's also the Fedora USB tool, which has a Windows version. I haven't tried it with non-Fedora distributions, but if you cannot get unetbootin to work, possibly give that a shot. [https://fedorahosted.org/liveusb-creator/](https://fedorahosted.org/liveusb-creator/)
(this one doesn't require you to format your USB)
