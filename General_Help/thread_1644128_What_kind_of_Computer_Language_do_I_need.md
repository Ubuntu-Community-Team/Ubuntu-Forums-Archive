You should not concentrate on learning a computer language, but rather how the computer works. Once you have a grasp for that, the language is irrelevant since there are only three types of languages anyway. Knowing one language of each type and understanding how the machine works, you can learn a any new language in only a few days.

I would say C/C++ is a good start, since C is tied to Unix and most of Ubuntu is written in C/C++. It is harder than Python or Bash, but more useful over the long run. Furthermore, if you hope to one day get a job as a programmer and/or go to College for Computer Science degree, C/C++ is a must have language.

On the other hand, Python and Bash would let you make more faster, but over the long run you will learn less and you will be very restricted in what you can contribute.
