I have a problem with Debian Menu and want to add screenshots to a post. How do I do this?  Thanks, Lee

### Post by taurus on 2007-03-30
Applications -> Graphics -> Gimp Image Editor -> File -> Acquire -> Screen Shot...

Save it and use the attachment option to add your screenshot here.

### Post by ardchoille42 on 2007-03-30
Open a terminal and run: gnome-screenshot --delay=10
Minimise terminal
Open a menu
Wait for screenshot to be taken
viola

You can change the "10" (above) to another number to have a longer delay (in secionds) before the screenshot is taken.
