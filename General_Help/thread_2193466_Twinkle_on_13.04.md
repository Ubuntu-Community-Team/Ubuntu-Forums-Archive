### Post by claracc on 2013-12-13
twinkle is in repositories, at least in ubuntu 12.04.

You can try to install it various ways:

Through synaptic manager, selecting the twinkle package and clicking on install

From a xterm, through the command:

```
sudo apt-get install twinkle
```, then introduce your root password and enter

Or through software center, looking for twinkle package and click on install
