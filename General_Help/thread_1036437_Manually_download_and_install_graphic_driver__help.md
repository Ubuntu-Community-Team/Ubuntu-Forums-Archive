### Post by tuxxy on 2009-01-10
You can download the 177 drivers from the following link, this is the supported drivers and most likely the ones Ubuntu would auto install had you an internet connection;

[http://packages.ubuntu.com/intrepid/nvidia-glx-177](http://packages.ubuntu.com/intrepid/nvidia-glx-177)

My advice is go with the first one your card will be supported and its in .deb format meaning you can install it with 2 mouse clicks and also heard some issues with the 180.22 drivers.
