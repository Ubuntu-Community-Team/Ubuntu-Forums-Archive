### Post by nebhead on 2007-04-26
i believe pearpc is in the respository

little howto if you want mac os x:

first of all you need a legal version of panther on cd. you have to create some virtual hard disks and be able to turn the panther install cd into an iso. you then have to go through the installation of panther, and install it to the virtual hard disk. the wiki below gives lots for information on how to set it up. i know that to change the settings (e.g where to find the virtual hard disk and how much ram etc) this file is at /etc/ppc and is called " ppc.conf " and you need to change it as root. to run the app go to the terminal and type " ppc ". i hoped that made sense, checkout the wiki for clearer details below and take a look at my screenshot off my installation. i know the wiki is for gentoo but its generally the same thing when getting pearpc working.
