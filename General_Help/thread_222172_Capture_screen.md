### Post by Rhubarb on 2008-07-03
Another good application to capture video is gtk-recordmydesktop
```
sudo aptitude install gtk-recordmydesktop
```

There are other more specialised applications for capturing OpenGL rendered video, but is not user friendly.  (And I've forgotten what it's called too).
***Edit: *** It's called glc, find it here: [http://nullkey.ath.cx/projects/glc/](http://nullkey.ath.cx/projects/glc/)
