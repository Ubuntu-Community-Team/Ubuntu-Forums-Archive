### Post by TiredBird on 2009-10-14
> **redev said:**
> ...would really like to get photoshop installed...

Like most of us, I too came from Windows. And like you, for some time I tried to use the same Windows programs I had been using. However, I quickly learned that there was probably no Windows I had ever used that did not have a comparable (and far better) counterpart in Linux.

I have never used Photoshop. It was too expensive for me. I installed Gimp instead. But Gimp is so close to Photoshop in its capabilities that I actually learned how to use it from a Photoshop book. 

What I'm saying is that if you already know Photoshop, switch to Gimp. Some of the commands may be a bit different, but the capabilities are the same and it will run far better on your Ubuntu system than Photoshop with Wine. 

I promise that once you get used to Gimp, you'll prefer it over Photoshop just like you are already seeing the advantages of Ubuntu (Linux) over Windows.

Just go to your Synaptic package manager, put 'gimp' in the search bar, click on the one to install, and voila, it will install and be ready to run. (And you'll get automatic updates with the rest of your system updates, all just as easy to install.)

I've been using Gimp, almost daily for over three years now. I have never used Photoshop so I don't really know how it compares. But it couldn't be too different if I used an Adobe Photoshop book to teach me how to use the Gimp.

By the way, although this wasn't true when I first started using it, documentation for the Gimp is now quite extensive and very user friendly. It is online at [the gimp home site]("http://docs.gimp.org/2.6/en/").

Anyway, try it. I'm sure you'll quickly forget that you ever used Photoshop.
