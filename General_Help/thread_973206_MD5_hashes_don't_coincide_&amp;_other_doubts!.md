### Post by mb_webguy on 2009-01-26
With your system resources, I'd suggest installing [Xubuntu]("http://www.xubuntu.org/"), one of Ubuntu's sister projects, instead of Ubuntu.  Xubuntu uses the Xcfe desktop environment instead of Gnome, so it's a bit smaller and uses fewer system resources.  

As for the current release versus the long-term-support release...  Ubuntu isn't like Windows, where each release is a completely different operating system.  Ubuntu releases are incremental improvements over the previous releases, and upgrading from one release to the next is therefore fairly painless.  New versions of Ubuntu are released every six months, and each  normal version is supported for 18 months.  That means that two new versions of Ubuntu will be released before a particular version is no longer officially supported.  The long-term-support versions, on the other hand, are officially supported for three years after they are released.  HOWEVER, that "official support" really only means that security updates will continue to be released.  You will still be able to get "support" for any problems you have here on this forum.  Besides, most people tend to upgrade to a new version of Ubuntu after a year or so in order to gain access to the improvements made in the operating system as well as newer versions of applications.  

So you're not really losing anything by getting the current version instead of the last LTS version, unless there's a particular reason why you wouldn't want to upgrade your operating system for another 2 1/2 years.
