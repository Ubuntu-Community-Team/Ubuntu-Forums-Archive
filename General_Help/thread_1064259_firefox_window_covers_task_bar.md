### Post by -kg- on 2009-02-08
What you're experiencing is a common Firefox Bug.

You need to install CompizConfig Settings Manager through Synpatics.  Once you have that installed, launch it either through "System/Preferences/Advanced Desktop Effects Settings" or by typing ccsm in a terminal window.

In the CCSM window, select "Utilities/Workarounds".  At the top of the page is a checkbox for "Legacy Fullscreen Support" which you should _uncheck._

This should solve your problem.
