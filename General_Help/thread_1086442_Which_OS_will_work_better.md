### Post by jacksaff on 2009-03-04
You need a light-weight distro. The computer you have does not have enough ram to run a full desktop environment like gnome (used in ubuntu) or kde (in kubuntu). It's also borderline for xfce (used in xubuntu). Processor speed also hurts, but it's the lack of ram that will make these unusable. These desktops take up more ram than you have so your computer will have to keep swapping stuff it needs into the hard-drive which will make it painfully slow.
A light-weight distro with a window manager rather than a full desktop will fix this.
DSL or Puppy will work FAST on your machine. Both are small enough that the whole distro will fit into your ram and so all programs are effectively already loaded. They are also fairly small downloads which you can burn onto a cd and boot from.

### Post by jacksaff on 2009-03-08
Outlook express won't work - it's a windows program. There are lots of alternatives though. Thunderbird is made by the same people as firefox so you might want to try that. There are lighter email clients as well that have full features but don't look as nice.

You don't use dos to run a live distro. You hold down a function key as you turn the computer on (read the initial screen when you turn on to find out which one) and then choose to boot from the cd drive or usb where you have your distro. Your hardware might be a bit old to support usb booting but you should be able to boot from a cd. If you like it you can install and boot up normally.
