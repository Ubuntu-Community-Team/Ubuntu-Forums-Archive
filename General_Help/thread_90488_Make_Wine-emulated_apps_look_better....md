### Post by haaglin on 2005-11-16
Just use Windows XP themes, this is how my wine looks like with a xp theme: 

[[IMG]http://img23.imageshack.us/img23/7978/wine4ko.th.png[/IMG]](http://img23.imageshack.us/my.php?image=wine4ko.png)

From winehq.org:

> 1. Download a Windows XP theme. Be sure it contains a .msstyles file. 

2. Create a set of new directories in your fake Windows drive: $ mkdir -p ~/.   wine/drive_c/windows/Resources/themes/name-of-your-theme 

3. Move the .msstyles to that new name-of-your-theme directory. 

4. Use the Appearance tab of winecfg to select the new theme.

But if anyone know how to get better looking fonts in Wine, please share!
