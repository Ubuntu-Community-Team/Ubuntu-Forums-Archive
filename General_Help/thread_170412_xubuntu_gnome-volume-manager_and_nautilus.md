### Post by localzuk on 2006-05-04
In what way do you want them to work? Do you want them to run when you log in? Or do you simply wish to run them by hand?

An example of using nautilus in xubuntu would be to install it first (via synaptic), then using either a terminal or by pressing alt+f2 entering the command 'nautilus --no-desktop'. This will launch nautilus without loading the icons on the desktop.
