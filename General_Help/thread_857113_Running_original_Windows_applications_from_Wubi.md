### Post by NilsHG on 2008-07-12
Hello!

I run WinXP in Virtualbox and it works great! I can copy paste between both operating systems and you can easily set them up to share folders in order to exchange files. 
Give it a try! There are plenty of Howtos on the forum and on the web on how to set it up properly. If you want USB support for your virtual OS do not use the OSE (open source edition). only the normal virtualbox version supports usb (as far as i know).

report back if you run into trouble!

edit:
what are your computer specs? a virtual machine takes up quite some ram and cpu power. without a decent system, your virtualbox experience will differ from what i described.
