Tor does have some limitations usage wise where security is concerned. [[COLOR=#0000ff]--Here--[/COLOR]]("https://www.torproject.org/download/download.html.en#") is a page with some advice for keeping it more secure if you need for it to be so. Scroll down the page to the section titled "Want Tor to really work?"

The "point" depends on your circumstances really. The browser bundle is a quick and easy way to use it compared to a full system installation; and should be more secure than just firefox if you follow the security advice on the tor site, otherwise it may very well be "pointless" to use.

I don't use the full system installation of tor, too messy to get set up right; just the browser bundle {Linux downloads, 3 rd section in the link above at the top of the page} off a USB stick (encrypted partition) and only with usage along the lines of that advice in the link. The browser bundle can be unpacked to any location and run from and updated from there eg. in a folder in your home folder or in my case off an encrypted USB drive. I find the browser bundle very flexible/easy to use and a breeze to set up and keep updated.
