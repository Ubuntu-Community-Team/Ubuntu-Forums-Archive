### Post by Lateralis on 2011-04-30
If you've nothing to lose, then go for it.  Do a fresh install and have a crack at it.  You might find that it is buggy or doesn't work well.  Well, don't worry.  Until 11.10, you can use the classic Gnome that you've been using in 10.10.  You can choose between the two - Gnome classic and Unity - at login.  So you can periodically check Unity to see if it has improved in your eyes or not.  Or, you may love it, in which case, great! 

Otherwise, if you fancy a change but Unity isn't your thing, you can install the Desktops used in the other *buntus.  For KDE just install kubuntu-desktop.  For LXDE install lubuntu-desktop and for Xfce install xubuntu-desktop.  XFCE looks pretty nice these days and KDE is very shiny.  You can in principle install them all side-by-side and choose which one to use at login.  If there is then one that you really prefer over the others you can just remove the desktop environments you don't like at your convenience.
