### Post by happyhamster on 2010-07-01
Disclaimer: I don't know anything about core files, but do know that the /proc directory is special: it contains 'virtual' files that describe the current state of the system/kernel. That means that writing into /proc will only affect one session.

To make your setting permanent, you'll have to make a change to the proper configuration file, if I would have to guess: /etc/sysctl.conf

For more info, see:
[http://www.redhat.com/docs/manuals/linux/RHL-9-Manual/ref-guide/ch-proc.html](http://www.redhat.com/docs/manuals/linux/RHL-9-Manual/ref-guide/ch-proc.html)

Good luck.
