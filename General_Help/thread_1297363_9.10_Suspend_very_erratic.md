### Post by jbrown96 on 2009-10-22
So I was doing updates manually with apt-get last night. There were a few updates,which were all related to a new kernel. They were not automatically applied and were being held back. I could manually install them by calling ```
sudo apt-get install package-name
``` Suspend is working again! No idea what fixed it, and there didn't seem to be any reports of this issue anywhere, so maybe it was my config. If you have the same problem in the next few days, you may need to force updates.```
sudo apt-get update && sudo apt-get upgrade
``` if there are updates that are being held back, then run ```
sudo apt-get install 
``` then add all those packages onto the end.
