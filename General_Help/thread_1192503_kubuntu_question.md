### Post by coffeecat on 2009-06-20
> **Zoyberk said:**
> when i use KpackageKit and i search for firefox, wine... i get nothing.
i have fealing that i can only search software which is already installed here.

Click on 'Settings' in the left pane and make sure you've got all the repositories enabled that you need. Now click on 'Software Management', then click the down arrow by 'Filters' and make sure you haven't got 'only installed' ticked. In fact, go through the filter list and make sure you aren't filtering anything out. Lastly, make sure you've got 'All Packages' selected in the drop-down list at the right. Then wine, firefox and anything else should show up when you do a search.

Having said that, personally I think that KPackagekit is GHASTLY. If you can get it to work, why not install Synaptic? This is the software management app that comes with Ubuntu/gnome. Once Synaptic is installed you can use that instead. It is much, much better.
