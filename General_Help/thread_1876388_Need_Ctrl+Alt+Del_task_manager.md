### Post by linuxuser12345 on 2011-11-06
I am starting a push to get a new task manager, or at least an update to the Gnome System Monitor, that allows even complete beginners to force quit frozen applications quickly and efficiently. A new user is going to have no idea how to use the current "Processes" tab in the Gnome System Monitor, as it shows every little application, even ones in the background.
I am pretty sure if you used to use Windows, you from time-to-time have clicked Ctrl+Alt+Del on your keyboard, hoping to force quit an application in Ubuntu, and all you got was a log out dialogue!

The new system I am proposing will have default keybindings (keyboard shortcuts) set to Ctrl+Alt+Del and Command+opt+Esc, the popular keybindings used on both Microsoft Windows and Mac OS X for their Force Quit-type applications.

Please join me in the push to get this implemented!
For more info and to officially sign up as a person who has the same problem, please go to:
[https://bugs.launchpad.net/gnome-system-monitor/+bug/886620](https://bugs.launchpad.net/gnome-system-monitor/+bug/886620)
