I think you mean du**a**l. A duel is a fight, that doesnt sound like a good thing :)

Anyway, Im prejudiced against wubi, it causes too many problems and risks and is not flexible enough. But performance wise there should very little difference.

As for the windows apps, have you looked for native alternatives? If you name the apps perhaps ppl can make suggestions. It would also interest me because most windows apps seem to work just fine on wine on my machine, just as fast as in windows. If your windows apps are not 3D, then you could also consider a different approach than dual booting. That is install windows in a virtual machine under ubuntu (virtualbox). That will give pretty much native performance (except for 3d/video) and is less cumbersome than dual booting. It does require a fair amount of RAM, Id say 1GB at least depending on the windows apps you want to run inside the VM.
