### Post by Ramses de Norre on 2006-05-22
```
sudo aptitude install xorg-driver-fglrx
sudo gedit /etc/X11/xorg.conf
```

Find the device section of your graphics card (identifier line will be something with ati) and change the driver to fglrx (probably ati or radeon now).

Save, exit and press ctrl-alt-del to restart X (closes all applications).

And I think there are many people out there with less good hardware ;)

### Post by Ramses de Norre on 2006-05-23
```
lsmod|grep -i fglrx
```
There should be a line starting with fglrx, maybe some other lines too. If not do this:
```
sudo modprobe fglrx
```
and restart X

You get the same result by adding fglrx to /etc/modules and rebooting. (but we don't like rebooting ;))
You've added it to fglrx so it should load on boot up automaticaly now.
