### Post by Sef on 2013-02-18
> Quote:
Originally Posted by OrangeVixen  
I haven't had time to upgrade from Lucid to ???, just didn't want to come across any upgrade problems because I'm so busy lately.
That would be upgrading to 12.04 LTS Precise Pangolin. You still have two months before support for Lucid runs out. When you upgrade, make sure you upgrade to the next LTS version, not the next intermediate version, which is already unsupported.

To make sure you upgrade directly to 12.04 from 10.04:

1) Open System Settings on the tool bar (Gear and Wrench)

2) Go down to System and click on software sources.

3) A new window opens, so go to Updates

4) The bottom option is 'Notify Me of New Ubuntu Version'

5)Click on 'For Long-Term Supported Versions'

That will ensure that you will upgrade to the latest LTS.
