### Post by aysiu on 2009-06-05
Earlier versions do not run lighter. In fact, 9.04 has offered the best performance I've seen (and I've been using Ubuntu since 5.04).

If you want light, don't go to an earlier version. Just use lighter-weight stuff. A window manager (IceWM, Fluxbox) instead of a desktop environment (Gnome, KDE, Xfce), Dillo instead of Firefox, Sylpheed-Claws instead of Evolution, etc.

### Post by snowpine on 2009-06-05
From a certain point of view, Debian is the "earlier version of Ubuntu" because Ubuntu was developed from Debian roots. However, Debian is still quite well maintained (unlike older versions of Ubuntu). I would choose a newer version of Debian over an older, unsupported version of Ubuntu.
