### Post by Sneaky07 on 2008-07-14
For some reason when I listen to streamed music in Amarok it plays but not well.  It skips and sounds like a scratched CD-ROM.  The music still plays and you can hear it but is very annoying when there are random pauses in the music.  I know it is not the website that is streaming because I have tried it on another system and it has worked fine.  Has anyone had this type of problem before?  Thanks in advance!! :)

### Post by Sneaky07 on 2008-07-14
Turns out the problem was that I needed to remove the following package from my Fedora system: alsa-plugins-pulseaudio (See the link here: [http://forums.fedoraforum.org/forum/showthread.php?t=188451]("http://forums.fedoraforum.org/forum/showthread.php?t=188451")).  Thanks for your help.
