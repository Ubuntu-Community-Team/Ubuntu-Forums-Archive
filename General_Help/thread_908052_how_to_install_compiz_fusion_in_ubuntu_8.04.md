### Post by yjwong on 2008-09-02
Compiz Fusion has been installed by default in Ubuntu 8.04. If you do have the appropriate video drivers installed, you should be able to enable it in System -> Preferences -> Appearance, under the "Visual Effects" tab.

### Post by Kevbert on 2008-09-02
What's your video card ?  If it's Intel, ATI or nVidia compiz will work in most cases.  If it's Via then it's not supported.
Can you set System - Preferences - Appearance - Visual Effects to Extra ?
If you don't have Advanced Desktop Effects Settings (under Systems - Preferences) you need to install the Compiz Manager.  Go to Applications - Add/Remove and Search for Compiz.  You then need to install CCSM (probably the first item displayed).
