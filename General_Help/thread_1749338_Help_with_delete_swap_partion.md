### Post by oldfred on 2011-05-05
Your boot.ini thinks windows is in partition 2 but it is in 3.

default=multi(0)disk(0)rdisk(0[COLOR=Red])partition(2)[/COLOR]\WINDOWS 

/dev/[COLOR=Red]sda3[/COLOR]          97,707,330   156,280,319    58,572,990   c W95 FAT32 (LBA)

If editing windows files like boot.ini
(Using nano instead of gedit, it's better for dos-style linebreaks)
Linux, of course, uses only LF as newline and DOS is expecting CR/LF so text files look wrong in DOS.
New versions of gedit have an combo box under save as, and choose windows format.

How to edit the Boot.ini file in Windows XP
[http://support.microsoft.com/kb/289022/](http://support.microsoft.com/kb/289022/)

Grub2 does not use boot flag. Some BIOS require a boot flag on a primary partition. But if booting windows, installing windows or repairing windows you have to have the boot flag on the windows primary partition. So move boot flag with gparted to sda3. It should boot with grub without flag, but if you ever want to use windows to boot then it has to have boot flag. Or if you use XP disk to repair it then it has to have boot flag.
