### Post by David Andersson on 2010-08-04
> **rob_38 said:**
> 
I'm looking at doing a course but the only ones available in my area are Fedora or SuSE.

If I go to them will what I learn be well implemented into Ubuntu? 

Such as commands etc.

There are a few areas where Linux distros differs.

What **desktop** does it use? For example, the standard desktop in Ubuntu is Gnome, and in Mandriva Linux it is KDE, IIRC. But you can install KDE in Ubuntu and Gnome in Mandriva if you want to. Then that difference diminish.

Another small difference is if **sudo or su** is used for administrative tasks. It can easily be diminished by installing sudo in the one system or setting a root password in the other.

**Network** management and **hardware driver** management can differ. But if you learn one way in one system, you should be able to tweak the other system so it works that way too.

The really big difference is **package management** (that is, how programs are installed) and it cannot be diminished. It can be very different in different Linux distros.

So, **if** the course use a Linux distro with **Gnome** then I think most of what you learn will be useful in Ubuntu. Even most thing about the command line and shell-scripting, if the course includes such things.
