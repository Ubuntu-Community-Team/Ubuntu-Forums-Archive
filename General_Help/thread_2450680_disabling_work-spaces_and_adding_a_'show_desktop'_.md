### Post by Dennis N on 2020-09-19
For a panel button, try [this Gnome Shell Extension]("https://extensions.gnome.org/extension/1194/show-desktop-button/"). The comment there indicates it works in Ubuntu 20.04. 

If you don't really need a button, **Super Key + Page Down** will also show the Desktop.
