### Post by .... on 2011-09-19
Ubuntu releases an LTS version every two years (8.04, 10.04, 12.04 etc.). These are officially supported for 3 years (server versions for 5). All other releases are supported for 18 months.

The Linux kernel has its own LTS policy that has nothing to do with Ubuntu (although I think Ubuntu generally tries to use an LTS kernel for its LTS versions).
