### Post by Bucky Ball on 2012-03-14
Other way is to boot from a LiveCD, 'Try Ubuntu', when at the desktop open Gparted (partition editor), delete the XP partition, expand the ubuntu partition into the free space, done. 

This may screw with grub though, depending on where you have it installed. You can't use this method on a system running from the hard drive as the partitions you are manipulating need to be unmounted (although you could kill the XP partition on a running system by unmounting it first but obviously couldn't expand / partition as the OS is running on that).
