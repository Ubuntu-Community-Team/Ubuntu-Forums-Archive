### Post by Cheiron on 2007-01-27
64-bit ubuntu is a seperate cd image download. 

When you go to the download page and select your download location, there should be several choices, including Mac/PowerPC computers and 64-bit computers. I guess there just wasn't enough room to put 32 and 64 bit on the same CD.

You probably won't see a dramatic speed increase from moving to the 64-bit versoin if your computer is just for typical home use. So if it's not worth the reinstall, I'd say don't worry about it.

### Post by ~LoKe on 2007-01-27
AMD64 is the architecture, not the actual processor.  Like the above said, if you're running a 32bit operating system, you don't want to install 64 bit applications (in fact, you can't).

I would advise **not** using the 64bit version.  For users who aren't experienced, you may have troubles getting certain things running in that environment.

Not worth the hassle.
