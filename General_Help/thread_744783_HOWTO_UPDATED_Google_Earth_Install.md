### Post by Frak on 2008-04-03
Since other threads seem to be lacking, here is the basic way to install GE.

```
wget http://dl.google.com/earth/client/current/GoogleEarthLinux.bin
bash GoogleEarthLinux.bin
```

for instructions with 64-Bit Systems, refer to [this]("http://ubuntuforums.org/showthread.php?t=544064") thread.

BTW, I use bash more than sh. Its just... bourne again ;)


Also, this howto is nothing special, but hopefully it helps someone.
