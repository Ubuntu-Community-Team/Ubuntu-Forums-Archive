### Post by p_quarles on 2007-09-08
Yeah, hardware support is the big challenge for Linux right now, especially with pre-manufactured laptops (specifically wi-fi). I'm sure you know this already, but the Linux kernel developers are doing what they can, but the components that have problems all have those problems because the manufacturers don't share the specs. 

That said, here's my experience: I recently purchased a laptop from Compaq, and had it built with parts I knew would work with Linux (which generally means as many Intel parts as possible). I haven't had a single problem with it so far. 

As for the GIMP: I've heard the newest development version has achieved CMYK color management. Don't quote in me on that, as I haven't used it, but you might want to check it out. On the other hand, you can run Photoshop 7 in Wine, and newer versions using VMware. If you can get your hands on a copy, you should be able to use it without too much fuss.
