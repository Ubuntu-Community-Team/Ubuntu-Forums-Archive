### Post by Rhubarb on 2010-02-13
> **JBR152 said:**
> ... I was wondering since I am technically on a Windows machine and am basically using Wubi as a skin to make it look like I have Ubuntu, is there anyway I can use Itunes and other programs?

Wubi isn't a skin for windows, it's more of a add/remove software shortcut to Ubuntu.
If you're running Ubuntu, you're not running windows at all.

As Dave89 said, it's possible to install an application called **wine** in Ubuntu that allows you to run many windows apps - but not all.
