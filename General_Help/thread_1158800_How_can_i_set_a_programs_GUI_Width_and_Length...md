### Post by kpkeerthi on 2009-05-14
1. Install **compizconfig-settings-manager** using synaptic 
2. Launch it from System -> Preference 
3. Add a [window matching rule]("http://wiki.opencompositing.org/WindowMatching#Window_Rules") that sets the desired width and height for the [application window of your choice]("http://wiki.opencompositing.org/WindowMatching#identify")
