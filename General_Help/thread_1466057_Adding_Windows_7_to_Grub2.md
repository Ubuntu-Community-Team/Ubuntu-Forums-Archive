### Post by limey_rick on 2010-04-30
What version of Ubuntu and GRUB are you using? 

If its Ubuntu 9.10, I think you will be using GRUB2 1.94 Beta4 or something like that. In this version, there is a bug with recognizing RAID partitions with DMRAID by GRUB. This was fixed with the GRUB in 10.04. 

I have a very similar setup and 10.04 RC1 solved the issue (which previously I solved by using the latest GRUB2 source and compiling in Karmic).

### Post by ht3k on 2010-04-30
THANK YOU!!!!!! I had this problem for a day and a half and this solved my issue. AWESOME.

As a noobie I would like to add for those that aren't familiar with linux.

Open terminal, type:

sudo gedit /etc/apt/sources.list

change the word karmic to lucid and save afterwards

close terminal and gedit

Next, go to:

System > Administration > Synaptic Package Manager

Search for grub2, grub-pc, grub-common and "remove completely"

open terminal

sudo apt-get install grub2

after it's finished reboot and it's done! :)

(Don't forget to change Lucid to Karmic again once you reboot)
