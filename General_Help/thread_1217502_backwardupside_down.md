To run it, save the script in ~/bin/upsidedown

Make it executable:
```

chmod 755 ~/bin/upsidedown
```

and run it like this:
```

upsidedown.py  "and there is a script that does this too:"
```

Note that all lowercase letters can be flipped, but not all capital letters, numbers, and punctuation have a flipped version.
