### Post by cariboo on 2008-12-28
To get aurora have a look [here]("http://gnome-look.org/content/show.php/Aurora+Gtk+Engine?content=56438"), there are also installation instructions on the page.

Make sure you have build-essential installed. Build-essential is in the repositories and you can use System-->Administration--Synaptic Package Manager to install the meta package.
