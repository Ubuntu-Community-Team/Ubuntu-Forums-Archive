### Post by Samuel-Sama on 2008-07-26
First, you need to know the socket type of the processor you are going to use, as a processor with a different socket is not compatible. For example, all recent Intel models will use socket LGA775.

Next you want to know the chipset of the motherboard, this controls the various peripherals etc. Some components, such as the processor may need a different chipset if it is not supported.

You also need to think whether you want integrated graphics or not; if you choose not to, then you will need a graphics card. Intergrated graphics usually perform worse than a dedicated graphics card, but are much cheaper and are sufficient for surfing the web etc. 

Some motherboards are designed especially for gaming and extreme users, you will have to pay a hefty premium for these and are best avoided unless you have deep pockets.

Hope this helps.

### Post by fragos on 2008-07-26
Do you wish to use memory and periferals from an existing system? If so there are changes in motherboards that may be of concern. If you want to use an existing IDE disk and DVD the mobo will need 2 IDE ports. Make sure the mobo supports SATA II. Some still available don't. My personal preference is Gigabyte moboards. I never spent more than $100 for one.
