How can I get into this area %appdata%\Mozilla\Firefox\Profiles\ I am usually using windows and would just go start, run, "%appdata%\Mozilla\Firefox\Profiles\" enter. Is there something on Ubuntu that is the equivalent to run on windows or something like that ? Thanks

### Post by earthpigg on 2009-09-14
hi,

in the file manager, click view -> show hidden files.

look around in /home/your-user-name-here/.mozilla

dig around in there for what you are after.

all of those files and folders that start with a dot (.) are called "dotfiles", and contain settings and whatnot for that particular user.

each user has his own complete set of dotfiles.


if you ever break a program, deleting its dotfiles and folders will revert it to it's default settings like the moment you installed it and ran it for the first time.
