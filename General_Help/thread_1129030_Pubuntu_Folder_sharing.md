### Post by fisa on 2009-05-13
By default, on the /mnt/C folder you have mounted the C: unit of the Windows host OS.

If not, you can access it using shared folders on windows, and connecting from the pubuntu with the IP.

danwood76: pubuntu is a portable version of Ubuntu, using coLinux, that runs cooperative with Windows (sharing resources, not like a virtual machine).
