4GB should be more than enough :-)

### Post by Djalmahal on 2011-03-01
It's easy to do but if you want it for gaming you will run into problems as the Virtualbox has to emulate the graphics driver instead of going directly to the graphics card. So you will not get good result for 3D games etc.

Still, worth a try.....
