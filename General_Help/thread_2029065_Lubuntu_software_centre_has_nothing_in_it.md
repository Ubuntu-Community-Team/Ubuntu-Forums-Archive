### Post by MG&amp;TL on 2012-07-19
LSC doesn't have support for commercial applications, hence why we have less. You can install Ubuntu Software Centre if that helps you.

Installing RAM-heavy apps won't make a difference unless you start them. Some desktops like KDE have daemons that may slow things down.
