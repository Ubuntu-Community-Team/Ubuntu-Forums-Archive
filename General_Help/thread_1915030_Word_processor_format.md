### Post by ndex477 on 2012-01-25
I use Libre office package in Mint 11 and am preparing to update my resume.  I noticed that Libre saves it's files in the .odt format. My question is how do i change the file format to .pdf or any other acceptable resume format?  Thanks

---

### Post by TheFu on 2012-01-25
Open the file in LibreOffice.
Select File --> Export to PDF

To save in other word processor formats, look at File --> Save As...

### Post by Jerry N on 2012-01-25
You can change the default format for Writer documents through Tools>Options>Load/Save>General.  There you can select the "Always Save As" format for the document type, i.e. "Text Document", that you are working with.  I don't think there is an option to always save as PDF.
