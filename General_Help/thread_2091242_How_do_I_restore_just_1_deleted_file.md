### Post by evilsoup on 2012-12-04
I don't know (sorry), but if you need to restore this file, I'd do it as soon as possible, even if it means restoring multiple other files. Deleting a file marks the bits as available, and they will eventually be over-ridden, even if just by temp files... which will make restoring the file actually impossible.
