Check out Click-n-Run. You might like it. It has a lot of software options to try out—not all of them support Ubuntu, but there are a lot of apps that it does support and CNR makes it very easy to install these applications on Linux, especially if you're a 'newbie.' :-)

[http://www.cnr.com/supportPages/aboutDownloadPlugin.seam](http://www.cnr.com/supportPages/aboutDownloadPlugin.seam)

Use the link above to download the CNR client to your version of Ubuntu Linux. Then it's a matter of searching through the library and clicking the install button.
