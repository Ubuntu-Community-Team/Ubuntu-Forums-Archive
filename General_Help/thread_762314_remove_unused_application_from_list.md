### Post by retrow on 2008-04-22
If you don't see Azureus in Synaptic but it still has a residual entry under shortcuts, you can edit the menu to remove it.

On taskbar, right click on Applications. Then *Edit Menus >>  Internet*

Uncheck Azureus from the list of internet applications. That way it won't show up in shortcuts any more.
