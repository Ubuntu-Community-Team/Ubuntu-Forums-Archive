### Post by deadflowr on 2016-04-27
> The appearance of "Ubuntu Software Center" in Ubuntu 16 has changed
and it wont install programs anymore.

Simple answer: it's not the old Ubuntu Software Center aka software-center anymore.
While it might be called that in the apps listings.
The actual program is now gnome-software.

Not a redesign, but a completely different program, entirely.
Current fixes for software installation problems are in the works.

Hang tight.
