### Post by Rubi1200 on 2010-09-19
Ubuntu can probably install quite happily with either the 6GB or 12 GB.

However, if you want to save files, use the external drive and create either a FAT32 partition to share with Windows or a separate Ext4 partition just for Ubuntu stuff.

A full Ubuntu installation takes up about 3GB.

This is just one possible way of doing it.

Hope this helps.
