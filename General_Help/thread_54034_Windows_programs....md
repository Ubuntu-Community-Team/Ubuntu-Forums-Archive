### Post by lol on 2005-08-03
Installing a windows program in linux is not possible. Well, there are way to do this, but it's never easy and it will fail with most of the programs...

Still, if you really want to try, have a look at Wine, but I honestly don't recommend it. You should rather install windows on a virtual machine, using VMware (not free) or Qemu for example. You should read this: [http://www.ubuntuforums.org/showthread.php?t=39513&highlight=windows+free](http://www.ubuntuforums.org/showthread.php?t=39513&highlight=windows+free)

If you want to play windows games, try Cedega ([http://www.transgaming.com/)](http://www.transgaming.com/)). You can also have a look at crossover for Office applications.

Good luck!

I wouldn't recommend OOo 1.9.79 that shows up in synaptic.  I installed it through synaptic as mentioned above.  However, it is still pretty rough looking (just look at the opening splash screen where the 1.1 is crossed out and 2.0 written in  :roll: ) and I couldn't open other files also made with OOo.  Admittedly, the files I was trying to open were made with a later release of OOo (m113), they were still in the same document format and should have worked.

Although the wiki page for installing the 2.0 beta looks difficult, it really isn't and with about an hour of time invested, you should be able to get a much better/up-to-date version of OOo.  I recommend using 1.9.113 (also referred to as m113).  Don't use m118 (I find it buggy and the program crashes alot).  The latest is M122 (came out this week) but I don't know how stable it is.

The wiki ([https://wiki.ubuntu.com/OpenOffice2Beta](https://wiki.ubuntu.com/OpenOffice2Beta)) is good.  If you have questions, there are several people on ubuntu forums that are very knowledgeable and helpful.  Just search for openoffice and reap the benefit of an open support community!

In particular, the forum member manicka was very helpful to me.
