### Post by WorMzy on 2010-12-19
You don't need to register to use IRC, but some servers allow you to "register" a nick to prevent others from masquerading as you.

Ubuntu has a number of channels on the Freenode server (e.g. irc.freenode.net). You don't need a client to use IRC, you can use [Mibbit]("http://02.chat.mibbit.com/") (which isn't able to connect to Freenode), or [Freenode Webchat]("http://webchat.freenode.net") (which is, obviously), but there are numerous irc clients available if you prefer to use one of them. I use weechat.

This forum, irc, and google are the only places I use in regards to Ubuntu.
