### Post by grahammechanical on 2016-08-21
> so therefore i cannot sudo apt-get update or sudo apt-get upgrade and get the battery fully charged until this is resolved.

Those commands do not re-charge the battery. The first command searches for updates to the OS and the second command downloads and installs the updates. And you will need your user passwords with any command with "sudo" in front of it.

Ubuntu 12.04 will reach end of support at the end of April next year. Perhaps you should think about doing a fresh install of Ubuntu 16.04 and set a new password. Do you have access to another computer that you can use to download the ISO image and burn it to DVD disc or USB memory stick.
