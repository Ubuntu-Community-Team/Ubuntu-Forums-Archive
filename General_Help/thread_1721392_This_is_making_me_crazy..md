### Post by Learning Linux 2011 on 2011-04-04
It looks like you already have done it, but I was just going to say that you should use the regular version of Ubuntu, not any netboook edition.

Most of the documentation is for the regular Ubuntu 10.10 or 10.04, so it is easier if you have that version.

If you want to see what version you are using, there is something in "System -> Administration" called System Monitor.  It also tells you things like how much memory you are using, etc.

Also, give yourself a good month to get used to things.  It takes time.

I responded to another person in this post:
[http://ubuntuforums.org/showpost.php?p=10598098&postcount=7](http://ubuntuforums.org/showpost.php?p=10598098&postcount=7)

It is rough, but it may help a little with some basics.

I also found this link helpful to get you started with a few things:
