### Post by XCan on 2009-08-02
So you're asking for an easy way to access files on your Windows machine from your Ubuntu machine? Sad to say, there are no easy ways, (which is kind of comical considering how weak Windows is despite the fact that machines really can't communicate with each other easily :p ). I would say that you should setup an FTP server (SSL of course) on your Windows box. If you really want to setup ssh, you can do it through cygwin. [http://ist.uwaterloo.ca/~kscully/CygwinSSHD_W2K3.html](http://ist.uwaterloo.ca/~kscully/CygwinSSHD_W2K3.html)
