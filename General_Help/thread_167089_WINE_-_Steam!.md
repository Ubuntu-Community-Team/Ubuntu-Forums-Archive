### Post by Mr Green on 2006-04-28
[QUOTE=chris062689]Whenever I boot up screen, it displays the message "Updating Steam..." but then, when I go to login, no text appears.

So I can't see what I'm doing.
Do I have to install a font first?
(How would I do that?; and which one?)[/QUOTE]

Run winecfg and enable Virtual Desktop. 
Then start steam, login, save account info. 
Exit steam, run winecfg and turn off Virtual desktop. 
Run steam again.
