### Post by howefield on 2011-03-13
Try going to Applications > Accessories > Passwords and Encryption keys.

In the window that pops up right click on Passwords:login and select change password.

You may need to type the old password but leave the new password fields blank. Then ok your way back out.

That should be it, but you may need to reboot and if the password dialogue window comes back up, leave the fields blank.
