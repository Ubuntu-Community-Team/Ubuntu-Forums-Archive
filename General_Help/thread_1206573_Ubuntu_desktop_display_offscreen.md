### Post by skinnymg1 on 2010-01-05
I had that same monitor and just recently got rid of it and i know what your problem is, this monitor is only supposed to do 1440x900@60Hz. So if you drop it to 60Hz you should be good if that doesnt work you will have to do as i did and stop gdm then do Xorg -configure and then edit xorg to meet the monitors requirements.

Under the monitor section it should look like this.
Section "Monitor"
	Identifier    "Monitor0"
	VendorName	"AOC"
	ModelName	"917Sw"
	HorizSync	30 - 83
	VertRefresh	55 - 76
EndSection

Under the screen section it should look like this.

Section "Screen"
	Identifier "Screen0"
	Device     "Card0"
	Monitor    "Monitor1"
	SubSection "Display"
		Viewport   0 0
		Depth     1
		Virtual	3120 1050
	EndSubSection
	SubSection "Display"
		Viewport   0 0
		Depth     4
		Virtual	3120 1050
	EndSubSection
	SubSection "Display"
		Viewport   0 0
		Depth     8
		Virtual	3120 1050
	EndSubSection
	SubSection "Display"
		Viewport   0 0
		Depth     15
		Virtual	3120 1050
	EndSubSection
	SubSection "Display"
		Viewport   0 0
		Depth     16
		Virtual	3120 1050
	EndSubSection
	SubSection "Display"
		Viewport   0 0
		Depth     24
		Modes	"1440x900"
		Virtual	3120 1050
	EndSubSection
EndSection
