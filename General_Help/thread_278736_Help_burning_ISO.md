### Post by CatKiller on 2006-10-16
If you copy and paste the file to a cd then you'll get one file on the cd. Not that useful.

If you manage to successfully burn the image to a disc then you'll get a proper directory structure and it will be automatically bootable. Much more useful.

They don't go for in-depth documentation on that Sonic site, do they? Anyway, from a teeny-tiny screenshot on their site it looks like you should have a Data menu on the left. The option you want is probably in there somewhere. What does the Direct-to-Disc option do?

EDIT: Are you using XP? Does that have any cd-burning stuff built in? I've never used it. If it does, it's possible that you could burn the image from the right-click menu - that's how it works under Linux. Just a shot in the dark.
