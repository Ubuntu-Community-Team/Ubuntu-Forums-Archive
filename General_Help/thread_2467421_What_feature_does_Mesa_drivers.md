### Post by grahammechanical on 2021-09-26
Mesa is not a video driver. It is a 3D graphics library. Keep in mind that hardware and software have been developing over a long period of time. There is a great variation in hardware even on machines from the same manufacturer. Also, Linux needs to work not only on modern hardware but also on much older hardware. Every installation of Linux will contain code that may not be essential for that particular machine but without that code an installation of Linux on another machine may not work correctly.

[https://www.mesa3d.org/](https://www.mesa3d.org/)
