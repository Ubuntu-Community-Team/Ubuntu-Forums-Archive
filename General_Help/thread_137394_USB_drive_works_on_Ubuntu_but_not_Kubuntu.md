### Post by tadashi on 2006-02-28
I selected all the respositories after installing Kubuntu and generally have about 130 upgrades and it did not break the system.  I just installed it about 2 days ago.  Once I install them, reboot, and it will be working.  Then I have to go install firefox, gaim, and kedit.   Then I disabled them when I was done.  If you plan to do any compiling install the build-essentials and if you plan on compiling anything that has to do with graphics (liquidweather, superkaramba, etc) then install PyQt and KDE dev files.

I also had to go to the Kubuntu KDE webpage to get the KDE repository to upgrade KDE from 3.4.3 to 3.5.1.  This fixed a few problems I was having.  If you like KDE then try this before wiping out your system (can't hurt).  If it does not work still then go to Ubuntu.
