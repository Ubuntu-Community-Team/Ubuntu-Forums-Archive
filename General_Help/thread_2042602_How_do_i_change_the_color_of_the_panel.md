### Post by stinkeye on 2012-08-21
I don't think it's possible to change the color of the panel.
You can change the transparency though to pick up the color of your desktop
using **compizconfig-settings-manager**. (ccsm)

To install in terminal
```
sudo apt-get install compizconfig-settings-manager
```

To run...alt+F2 and type in **ccsm**
Navigate to the ccsm > Ubuntu Unity Plugin > Experimental
[ATTACH]222972[/ATTACH]


There is a way to color the panel using a colored **conky** window 
under a 100% transparent panel.
Are you familiar with conky?
Interested, let me know.
