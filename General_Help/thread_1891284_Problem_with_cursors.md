### Post by Pingouin7 on 2011-12-05
For some reason, the cursors sometime become huge.
It happens when I'm using Firefox, but also when the system asks me for my password.

Here is a screenshot:
[What it looks like when messed up]("http://i.imgur.com/lXHmr.png")

The mouse cursor appears normal when I'm on the Desktop, but for some reason, if I take a screenshot, the mouse cursor appears huge on the screenshot while it isn't in reality. (So I can't take a screenshot for comparison, sorry.)

As you can see, I just installed the mac4lin theme, which I think caused the problem.

Thanks

Edit: Found the problem, it apparently is a bug in Natty that makes it so custom cursors only work when in certain applications.
In this case, the Mac4Lin cursor pack.
Disabling these cursors and going back to default fixed the problem.
