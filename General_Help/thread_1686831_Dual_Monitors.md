### Post by jwcalla on 2011-02-12
> **Ashy Knuttz said:**
> so theres no way i can run one harddrive on one monitor and then another on the other monitor?

No because you cannot simultaneously share many of the system components, such as the processor or memory.  To run both natively you'd need two computers.

However you can run one operating system virtually inside the other operating system as foxheadz has mentioned.  Sun made some software called VirtualBox to do this but there might be others out there as well.
