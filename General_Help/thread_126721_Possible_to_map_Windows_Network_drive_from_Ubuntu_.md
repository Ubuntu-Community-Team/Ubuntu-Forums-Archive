### Post by schdefan on 2006-02-07
You can search for a network with the gnome network search (don't know the exact name but its in some of the menus). I think the LiveCD supports mounting a windows network with smbfs. Once you know the server name or IP address where your data is stored open nautilus press ctrl+L and type smbfs://<servername>
