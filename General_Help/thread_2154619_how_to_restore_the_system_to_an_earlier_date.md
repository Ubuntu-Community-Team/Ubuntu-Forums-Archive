### Post by Cheesemill on 2013-06-15
The only way this is possible is by restoring a backup of your system that you made on that day.

If you don't have a full system backup of the day you wish to roll back to then no, it's not possible.

### Post by grahammechanical on 2013-06-15
What you ask for can be done if the OS is installed on a btrfs file system but the default file system for Ubuntu is still Ext4 and until btrfs is made the default file system we should consider it an experimental file system.

I am experimenting with btrfs at the moment. We do not have a GUI front-end to the management tools. Until we get a GUI management utility I would not recommend btrfs becoming the default Ubuntu file system. So, the fact is Ubuntu is months or even years away from having a simple roll back method.

Sorry.
