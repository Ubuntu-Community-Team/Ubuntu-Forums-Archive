### Post by zanglang on 2007-02-22
For example, if you want to create a folder called "Secrets", name it ".Secrets" and it ought to be hidden by default when you browse in Nautilus.

Alternatively, create a text file called .hidden (no .TXT extension!) and put a line 'Secrets' in it.
