### Post by howefield on 2011-01-03
> **Ruinofthedead said:**
> Will this compromise my Ubuntu Experience?

Only if you think Ubuntu is the same as Windows.

> Are there certain programs that won't be usable by me once I download Ubuntu? (I am a gamer. :razz:)

You are perhaps best going on the assumption that your windows software will be obsolete with Ubuntu, you will use new applications / software.

You will be able to run some Windows software via third party applications, eg Wine, PlayonLinux and others. But consider that a bonus.

> Is the security really that good?

Security is only ever as good as the user, and it is perfectly possible to secure a Windows installation, but yes, it's much easier with Linux.

> Is it totally free?

Ubuntu is and always will be free.

> Can I install my Nvidia driver with an ASUS driver disk once I have Ubuntu Installed? (I'm gonna go check myself in a sec.)

Nope.

> Does Ubuntu COME WITH a graphics driver?

Yes.

Depending on how serious a gamer you are, this will likely be your show stopper, although there are plenty games available to Linux users and a gaming forum here for you to check out.

With the information you have given, I'd recommend a dual boot with Windows.
