### Post by snowpine on 2010-07-11
Try disabling Compiz Desktop Effects, then try The Cog's suggestion instead: Alt+Click in window, drag so you can see the buttons on the bottom.

If you are looking to add the Netbook Edition feature to your existing install, the application is called "maximumus" and you can install it through the Synaptic Package Manager. It will full-screen everything.

### Post by efflandt on 2010-07-11
What The Cog mentions should work if you hold the Alt key, then left click the mouse within the window, but not on a field, and drag it.

One thing you can do is make the fonts smaller, if you can get to that under System > Preferences > Appearance > Fonts.  That should change fonts on the fly as you change each one (maybe from 10 to 8 ), so maybe the window will get smaller as you change them.

Another thing you can do is set top and/or bottom panels to Auto Hide, by right clicking on them and clicking Properties.
