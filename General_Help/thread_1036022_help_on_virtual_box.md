### Post by hyperdude111 on 2009-01-10
It is very easy.

Go to the icons and make a new machine.
Plug in your xp cd and under settings for the virtual machine tell it to use your cd/dvd drive for the image. Once it is done startup the virtual machine and do everything as if it was a normal pc. (but in a window)
