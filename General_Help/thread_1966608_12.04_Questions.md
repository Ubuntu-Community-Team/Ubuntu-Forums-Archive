### Post by zombifier25 on 2012-04-27
1. I think it's your Wubi install that's slowing down. Ubuntu 12.04 gave Wubi the boot from the CD. You should consider dualbooting.
2. To view your architecture, put this in the terminal
```
uname -m
```
3. You must install ccsm:
```
sudo apt-get install compizconfig-settings-manager
```
Then launch ccsm, choose the Unity Plugin, go to Experimental and change Background Color to whatever you like.
