### Post by Mark Phelps on 2012-02-23
Apart from systems obtained with Linux pre-installed by the system vendor, NO laptops support Linux "flawlessly".

Sorry ... but that's a fact of life.

Laptop vendors are infamous for using customized hardware, which requires customized drivers, to work -- recent example being Hybrid (or Switchable) graphics.  It take a while (months, years) for the Linux community to "catch-up" with customized drivers for these devices.

Generally, the newer the laptop, the less likely that it will work flawlessly out-of-the-box.

And unfortunately, about the ONLY way you'll find out (unless someone else has the exact same make and model and has already tried it) is to boot from a LiveCD or LiveUSB and see how well it works.
