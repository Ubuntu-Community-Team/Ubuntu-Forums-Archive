### Post by AdHavoc on 2008-03-13
I currently have Ubuntu 7.10 and Windows XP SP2 set up in a dual boot scheme.  Now that I've set up Ubuntu to my liking and ensured everything works, can I make my NTFS partition (note that the actual OS is not on it, it is a secondary hard drive) smaller and then make my ext3 partition (the one with Ubuntu on it) bigger?  Can this be done without damaging anything?

Thanks.

---

### Post by Rocket2DMn on 2008-03-13
Using GParted on the LiveCD or downloading the GParted LiveCD - [http://gparted-livecd.tuxfamily.org/](http://gparted-livecd.tuxfamily.org/) will allow you to do this.  You should always have everything backed up before you fiddle with partitions, in case something goes wrong.  Also, you should defragment your windows drive before proceeding.
