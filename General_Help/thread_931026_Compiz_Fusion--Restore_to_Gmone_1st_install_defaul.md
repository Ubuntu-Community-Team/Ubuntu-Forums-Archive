### Post by Genius314 on 2008-09-26
Default Compiz settings?
First install CCSM (You can uninstall later if you want.):
> sudo apt-get compizconfig-settings-manager

Now go to System>Preferences>CompizConfig Settings Manager. Click on "Preferences" on the left, and click the "Reset to default" button on the right.

Hope that helps. :popcorn:
