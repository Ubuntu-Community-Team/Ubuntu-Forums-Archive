### Post by IWantFroyo on 2011-07-31
Chromium is Chrome before Google gets to it, so none of Google's stuff should be on there. Also, yes, there is a version of Firebug for Chrome, which you'll find in the chrome app store (google it).

Chromium and Firefox are as safe as you want to make them. For either, you can install tons of plugins to keep them secure. I personally prefer Firefox because of its integration with my 10.04 system. Even though it's FF 3.6, it still runs faster than any other browser (or newer version of FF) that I put on my machine.
