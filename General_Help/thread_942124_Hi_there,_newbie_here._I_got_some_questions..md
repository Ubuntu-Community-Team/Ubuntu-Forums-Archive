### Post by jerome1232 on 2008-10-08
Linux is a world different from Windows so you will have a learning curve. The biggest hurdles are usually graphics and wireless, so it would help to know what model of a graphics card you have, and if you plan on connection via wifi what type of wifi card that you have.

Wine is a compatibility layer for Linux that can run some Windows programs. It just barely started doing decently with dx apps, I know CoD4 works in it  now I don't know about the other ones on your list.

Ubuntu uses the Gnome Desktop Environment.
Kubuntu uses the K Dekstop Environment
Xubuntu uses the lightweight XFCE Window Manager.

They have differences but at the core they all have access to the same software and it's really personal prefrence as to which one people prefer. (I like Gnome and used the e17 Window Manager for awhile, it's very different)

If you want to be able to play the latest games for Windows you are going to want to keep that Windows partition around, it's not happening under Linux (A few commercial games are designed for Linux however)

Welcome to the forums!
