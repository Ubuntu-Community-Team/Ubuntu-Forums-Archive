### Post by LeftyAce on 2009-05-28
Computer=>System Settings=> Keyboard and Mouse=>Global Keyboard Shortcuts
Select "Plasma Workspace" from the dropdown menu, and set "Show Dashboard" to whatever combo you like.

EDIT: errr. Assuming you're using kde 4.  I'm not sure how it's set up in other versions :-/
