### Post by farrinux on 2017-07-07
Try going to system settings, text entry,. There you will see listed the default keyboard.  Hit the + sign at the bottom to pick a different keyboard.  When I looked there were several german mac keyboards. Make sure the new one you pick is at the top of the list.  You can change the order with the arrow keys at the bottom.  Also note the settings on the right as well. Hope it helps.
