### Post by pqwoerituytrueiwoq on 2012-04-29
try editing that last section to make it look more like this
```
Section "Screen"
    Identifier     "Screen0"
    Device         "Device0"
    Monitor        "Monitor0"
    DefaultDepth    24
[B]    Option         "TwinView" "0"
    Option         "metamodes" "_1600x900_60_ +0+0"[/B]
    SubSection     "Display"
        Depth       24
    EndSubSection
EndSection
```
be sure to put the underlined part right for your screen
width x height _ hrz
then logout out and in

gksudo leafpad /etc/X11/xorg.conf

gksudo pluma /etc/X11/xorg.conf
