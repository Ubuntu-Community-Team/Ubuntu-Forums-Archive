### Post by oldfred on 2022-04-09
Your sdb is a drive, but sdb2 or sdb3 are partitions. 
Microsoft since beginning of time has confused drives & partitions.

Post your fdisk -l in code tags. 
Easy to add code tags using Forum's advanced editor and # icon.

Note that most tools that create USB flash drive installer totally erase that flash drive. Best to use a smaller flash drive.
And since not sure of drives/partitions, you really need to first make good backups of all your data & installed systems.

Are you wanting a live installer, a live installer with persistence or a full install to a USB flash drive?
Full install requires you to have a live installer on one USB drive to install to a second USB drive.
[https://askubuntu.com/questions/1217832/how-to-create-a-full-install-of-ubuntu-20-04-to-usb-device-step-by-step](https://askubuntu.com/questions/1217832/how-to-create-a-full-install-of-ubuntu-20-04-to-usb-device-step-by-step)
