### Post by viperdvman on 2011-10-06
If you want to change Desktop Environments on login, it's best to change your login to where it requires you to put in a password, or set it for Automatic Login and also set it to wait X amount of seconds for others to log in. That'll give you time to change desktop environment if need be.

Oh, and if you'e doing the traditional type in username, then type in password... then you type in your name, then go to that bottom bar and select your desktop environment before putting in your password.

I change DE's according to my mood, though I run GNOME 2.x like 95% of the time on both computers. (I run Ubuntu 11.04, btw)
