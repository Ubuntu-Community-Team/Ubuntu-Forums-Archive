### Post by sparks40 on 2009-11-26
I am trying to use an external USB flash, and/or USB HD drive to boot an alternative Linux distribution. My mother board is an Asus M2N-MX which works fine using conventional HDrives. When I open the BIOS setup to select the correct boot order, I do not find a specific listing for a USB device, only the existing HDrive and an Nvidia Boot option. Does this mean that the MB is incapable of booting from an external device? If so, will updating my BIOS enable the USB boot function?

TNX

---

### Post by dandnsmith on 2009-11-26
It isn't true that it is necessarily incapable - some of the BIOS's provide the option if they detect a bootable USB device on bootup.

There is a procedure for evaluating whether a motherboard is capable on pendrivelinux.com - it involves setting up such a device, and running the tests there described.
