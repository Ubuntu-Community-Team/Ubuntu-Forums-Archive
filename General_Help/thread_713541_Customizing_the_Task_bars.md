### Post by TheIdiot on 2008-03-03
Yep, right-click the panel and Add New Item

Default top panel items (not including the firefox and help launcher shortcuts) are:
Xfce Menu, Separator or Spacing, System Tray, Clock, Action Buttons

Default bottom panel items are:
Show Desktop, Task List, Pager, Trash Applet

If you accidentally removed your panels, press Alt+F2 and run "xfce4-panel"
