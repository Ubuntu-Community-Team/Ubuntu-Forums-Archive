### Post by kdawg on 2006-12-01
System>Preferences>Menu Layout

Then choose the section you want to put it and add a new item.
Put any name and description in and then the command you use in the terminal for the application into the command box.  Pick any icon you wish and hit ok. 
Hopefully that is what you are looking for!
