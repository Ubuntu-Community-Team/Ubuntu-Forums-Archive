### Post by Morbius1 on 2016-05-22
What I normally do is go to this site: [http://hplipopensource.com/hplip-web/install_wizard/index.html](http://hplipopensource.com/hplip-web/install_wizard/index.html)

I tell it what version of Ubuntu I'm using and the what model of HP printer I'm thinking of buying. It will tell me if the version of the driver Ubuntu installs works with the printer I want to buy.

It will also tell you how to install the driver directly from HP should Ubuntu not be current. I personally have never done that opting instead for something that is already compatible.

*Note: It doesn't look good for the 8715*
