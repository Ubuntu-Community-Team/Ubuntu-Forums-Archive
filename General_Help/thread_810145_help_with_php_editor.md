### Post by sdennie on 2008-05-28
Asking for an editor for code is likely to get you a huge variety of answers and justifications for those answers.  You are probably best off learning something like vim or that other editor that begins with an "e" (emax?  emess?).  Really, learning a generic editor for code is likely to help you in the long wrong, rather than a specialized php editor.  If you aren't afraid of a fairly high barrier to entry, I would recommend:

```

sudo apt-get install vim-gnome

```

Then, to start the editor, it's just:

```

gvim

```

It's not an easy editor to use (until you learn it) but, the following command may help if you have the time to invest in it:

```

vimtutor

```
