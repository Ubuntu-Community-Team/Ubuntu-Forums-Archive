### Post by aff92 on 2012-08-09
To view the contents of the dir you are currently at, type:
ls (in terminal)
To navigate/change current directory, type:
cd NAME_OF_DIRECTORY (in terminal, replace NAME_OF_DIRECTORY with the actual name of the dir you want to navigate to)

To run a makefile, in terminal, i think you should type:
make -f NAME_OF_MAKEFILE (replace NAME_OF_MAKEFILE with the actual name of the makefile)

Hope this helps!
