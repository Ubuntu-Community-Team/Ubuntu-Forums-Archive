### Post by cakwatu2 on 2014-11-28
Hello Ubuntu Linux Community!

I have been a long time Windows user, but have always taken an interest in Linux, more specifically Ubuntu. I remember using Ubuntu 9.10 alongside my Windows xp laptop years ago when Ubuntu gave out free CD's. After upgrading to Windows 7 and than 8 I stopped looking at Ubuntu and enjoyed the blissful world of Windows. What later frustrated me about Windows was the constant need of "Antivirus or Internet Security". Why should I be subject to have to secure my computer, and fix flaws that Microsoft cannot do themselves? When I returned to Ubuntu in 14.04, I really enjoyed it. It was very stable on my machine, except for a few bugs. But the problem for me began in 14.10

Let's get to the issue, Ubuntu 14.10 has created a plethora of bugs for me on my laptop. I assume it's the kernel their using but I'm not specifically sure. **First bug**: My brightness keys no longer work as they did in 14.04, Fn+F6 and Fn+F7. **Second bug**: When my laptop battery is full and I unplug my charger, Ubuntu suspends saying the power level is "critical", After getting my laptop to resume, I can continue working as usual. **Third bug**: The network indicator is a little buggy for me, when I disconnect from my network and try to reconnect, Ubuntu doesn't let me reconnect, which forces me to reboot to get back on my network.

All these issues aside I really like Ubuntu for it being open source and secure. Hopefully one of the many Ubuntu users and experts can help me resolve my problems on Ubuntu 14.10. The reason I stick with 14.10 is because I want to use the latest and greatest Ubuntu has to offer.

System Specs:

Toshiba Satellite L745-S4130

Intel HD 3000 Graphics

4GB Ram

Processor Intel(R) Core(TM) i3-2350M CPU @ 2.30GHz, 2300 Mhz
