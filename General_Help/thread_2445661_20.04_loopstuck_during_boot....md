### Post by oldfred on 2020-06-18
If Windows 8 or 10 is not fully shutdown, both fast start up or hibernation, it leaves the hibernation flag set on all NTFS partitions. 
If you have mount set for an NTFS partition in fstab, then that could be the issue as Linux NTFS driver will not mount a hibernated NTFS read/write and then has to time out.

Fast Start up off (always on hibernation), note that Windows turns this back on with updates, SHIFT + Shut down button
[http://ubuntuforums.org/showthread.php?t=2324331&p=13488472#post13488472](http://ubuntuforums.org/showthread.php?t=2324331&p=13488472#post13488472)
[http://www.tenforums.com/tutorials/4189-fast-startup-turn-off-windows-10-a.html](http://www.tenforums.com/tutorials/4189-fast-startup-turn-off-windows-10-a.html)
