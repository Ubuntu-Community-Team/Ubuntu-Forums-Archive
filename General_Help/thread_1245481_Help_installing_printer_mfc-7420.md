### Post by plucky on 2009-08-20
> **findit said:**
> It recognizes the printer but when i go to print a test page nothing happens. I downloaded the drivers but maybe not installing them correctly. any help appreciated

Why not use **Synaptic Package Manager** to install the drivers.
Open SPM and search for **MFC-7420** and it should find "brother-cups-wrapper-laser and brother-lpr-drivers-laser",select both and apply.

Then open **System > Administration > Printing** and add a new printer and it should find and install the printer if it is connected and powered on. 

Good Luck
