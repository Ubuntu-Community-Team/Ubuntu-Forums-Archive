### Post by houseworkshy on 2010-11-11
For system spec's a handy tool is sysinfo, it can be installed either via synaptic or with "sudo apt-get install sysinfo" in the terminal.  Once you've got it run it either from the terminal "sysinfo" or from Applications>System tools where it will have put a menu item.  Then use it's file tab and save it wherever you choose.  Posting that will help people to help you, also a lot easier than typing the whole lot out manually.

Small programs like that can be installed to ram in live sessions, as you are posting I'm guessing that you are in something that works. Obviously irrelevant if you are just using a differant machine.
