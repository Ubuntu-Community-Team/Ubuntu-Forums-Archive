### Post by QIII on 2012-01-20
It is possible, but not likely, that you could install from your "recovery" disk, provided it is actually an installation disk.  Some "recovery" disks are actually full-on installation disks.

However, there may be legal issues with using an OEM install disk to install Windows in a Virtual Machine.  Many OEM EULAs expressly proscribe the use of the disk for such purposes.

Also, some OEM disks will check the hardware to see if you are trying to install on one of their ElectroGobbler C Series machines and prevent you from installing if you aren't.

If you are able to start an installation, read the EULA carefully.

As much as I detest Microsoft's business practices, I would not want someone stealing my intellectual property so I won't steal theirs.
