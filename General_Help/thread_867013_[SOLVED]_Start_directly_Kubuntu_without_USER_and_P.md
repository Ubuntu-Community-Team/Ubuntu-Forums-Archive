### Post by aysiu on 2008-07-22
Go to KMenu > System Settings

Go to Advanced > Login Manager

Click on Administrator Mode

Go to Convenience and check Enable Auto-Login, and select the User you want to have an autologin for.

Then click Apply to apply changes.
