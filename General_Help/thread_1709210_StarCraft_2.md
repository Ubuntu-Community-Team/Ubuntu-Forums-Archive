### Post by Lars Noodén on 2011-03-18
If there is not a linux version, then look in the [WINE application database](http://appdb.winehq.org/) and search around for [Starcraft II](http://appdb.winehq.org/objectManager.php?sClass=application&iId=11123).  It looks like it is supported and should run under WINE.
