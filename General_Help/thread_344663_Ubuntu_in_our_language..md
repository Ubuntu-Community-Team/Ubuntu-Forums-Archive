### Post by sloggerkhan on 2007-02-07
There are several different things you can do.
One thing you will notice is that many applications have an item in the help menu called "translate this application."

You will go to a launchpad location for translation and be able to make an account, choose what language, and start a translation.

### Post by sloggerkhan on 2007-02-07

you can see which languages are in progress for the app and then chose what language you wish to translate for and so on.
