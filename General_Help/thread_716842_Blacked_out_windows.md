### Post by darkenemy on 2008-03-06
i dont know if this helps, but about a year ago i had this problem with an nvidia graphics card... basically when i had too many windows open (sometimes only two or three) some of them would turn black.. 

this was because there was not enough memory in the card to handle that much at once.  i found that if i minimized all windows and then only pulled up the one that i wanted.. it worked with no blacking.

for me this only became an issue once i started using compiz and beryl

now i have a new graphics card adn have not had the problem since.


you may want to do the same



hope that helps
