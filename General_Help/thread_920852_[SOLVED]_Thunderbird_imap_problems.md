With Thunderbird, if you have more than one e-mail address, you can select the FROM: email address by choosing the desired address from the drop-down list.  The drop-down list is right above the To: address box.

Thunderbird allows for multiple SMTP servers.  To an additional SMTP server, select from the menubar, Edit/Accounts/.  Scroll down to SMTP selection at the bottom.  Then click add and populate SMTP pertinent information.

With Thunderbird, each e-mail address can have its own SMTP server.  To select a specific server, Choose Edit/Accounts.  Select the first line of each account, the title should read Account Settings...  Select the desired SMTP server from the drop-down list of Outgoing Server.

Hope this response helps you out.  Good luck.
