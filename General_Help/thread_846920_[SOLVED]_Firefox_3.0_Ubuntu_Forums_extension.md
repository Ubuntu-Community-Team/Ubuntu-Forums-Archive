### Post by dcstar on 2008-07-02
The version available from the Firefox Add-ons site is configured to install only on FF 3.0pre and not later releases. The author seems to have "forgotten" to update this handy tool, so I have patched the attached version to install on later Firefox releases

Download the file to your PC, then **rename the file without the .zip extension**, and then open the .xpi file in Firefox (File-Open File) and you should get the normal add-on installation window. Once installed go into Tools-Add-ons-Ubuntu Forums Menu-Options and enable "Add menu to toolbar".

### Post by sawjew on 2008-07-20
Add the nightly tester tools extension from [here]("https://addons.mozilla.org/en-US/firefox/addon/6543") and you can then just install the standard addon from the mozilla addons site.  It will give you the option of overriding the version when you install the ubuntu forums menu.

You can use this to install any incompatible addon but be careful because some addons may break firefox.

If you can't install the addon because it says it is for older versions of firefox, open the extension page by clicking on the extension title, then browse down until you see "see all versions".  Click on this link and you can install it from the newest one on the page.

### Post by MikeDK on 2011-07-31
rename your downloaded ubuntu_forums_menu-0.69-fx.xpi.zip to ubuntu_forums_menu-0.69-fx.xpi then open the xpi file with archive-manager and mark the install.rdf file and open that with your favorite text-editor say gedit for example, then go to <em:targetApplication>

			<Description>

				<em:id>{ec8030f7-c20a-464f-9b0e-13a3a9e97384}</em:id>

				<em:minVersion>1.0</em:minVersion>

				<em:maxVersion>5.*</em:maxVersion>
and change max version to your version of firefox or a later version.

Should work that way, if your not having an older version of firefox thou.
