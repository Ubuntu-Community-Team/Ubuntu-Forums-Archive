### Post by geo909 on 2008-06-20
First of all you should install texlive. Search for texlive base in synaptic. You can easily install any other latex package from synaptic too.

 For an editor, I use kile, which I really love. It's an editor especially for latex. I also use kdvi, kpdf and there's lots of cooperation between them (you can go directly to the line that pdf shows and vice versa and other stuff). You can use kile with other viewers though, the configuration is easy.

 Another latex editor you can find in synaptic is texmaker. I haven't used it but It looks similar to kile and it seems good too. Of course you can use any editor to write tex code, but it's better to use a specialized latex editor if you are new to tex because it makes it easier for you.

 If you are into programming and you are used to vim or emacs, there are tex extensions.

 Anyway, kile+kdvi+kpdf works like charm for me.
