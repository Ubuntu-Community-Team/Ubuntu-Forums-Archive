### Post by ApUUbunU on 2008-08-29
Open a terminal by clicking the ubuntu icon in the top right, go to accessories and try the command:
> $ sudo apt-get install avant-window-navigator awn-manager 

Hopefully that should install the program. If not, try the link

[http://news.softpedia.com/news/Install-AWN-on-Hardy-Heron-82611.shtml]("http://news.softpedia.com/news/Install-AWN-on-Hardy-Heron-82611.shtml")

I found this link just be searching Google for "how to install avant window manager", information is probably already there on there net, just go look for it!
