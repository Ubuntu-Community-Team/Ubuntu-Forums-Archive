I believe the default pdf viewing application is Evince. You might want to check that it's installed, and then right click on a pdf file and select Properties -> Open With to set that as the default again.

### Post by deadflowr on 2018-12-01
> It shows the first page, of the file and that's the only thing I can access.
Per this, check that page pane is enabled in the view menu section.
That should then show the pdf pages in the left side pane, which you then click on those to switch each page.
