### Post by Furao on 2007-03-04
There's a tutorial page for the whole process in the Ubuntu documentation here:

[https://help.ubuntu.com/community/BurningIsoHowto](https://help.ubuntu.com/community/BurningIsoHowto)

Be sure you check the MD5 sum according to the directions on the page.
