Why do you think Libre Office is different to Open Office?  The developers of Libre Office were Open Office developers before they split from Oracle.  As the code is open source they took the code from Open Office and continued to develop it but because Oracle owns the name of Open Office the developers can no longer use its name (only the code) and therefore called it Libre Office.  (by the way Libre is Spanish for free).

If you really want to install the package called Open Office you will either need to revert to Ubuntu 10.04 LTS or install it from the OpenOffice.org website.
