### Post by speedwell68 on 2009-07-28
Assuming you are running Ubuntu 9.04 Jaunty Jackalope.  Simply go to the 'System'menu, choose 'Preferences' and then 'Startup Applications'.  Then it will open a small window.  Click the 'Add' button and browse to find firestarter.  Remember to give it a title and description too.

If you are using an earlier release of Ubuntu it is exactly the same except that instead of choosing 'Startup Applications' from the 'Preferences' menu, it'll be under 'Sessions'.
