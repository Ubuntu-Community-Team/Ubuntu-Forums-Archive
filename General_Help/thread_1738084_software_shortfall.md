### Post by Paddy Landau on 2011-04-24
**iPod:**

Unfortunately, Apple supports Mac and Windows only. You may have to live with dual-boot, unless you can successfully run Windows from VirtualBox.

Rhythmbox is supposed to sync with Apple devices. I have read that you  need to uninstall libimobiledevice0 and install libimobiledevice1 (you  can do this from the Ubuntu Software Centre), but I don't know whether  that will help you.

Some people have solved the problem with iPod and iPad, but it seems to be hit-and-miss. Search the forums for iPod sync, see what others have done, and whether you can duplicate the success.
