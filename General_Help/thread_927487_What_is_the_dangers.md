### Post by prshah on 2008-09-23
> **NintendoTogepi said:**
> Does using Wubi to try Ubuntu cause any serious dangers? Can it mess up my Windows XP installation or screw up my hard drive?

Short answer: No it shouldn't.

No one can say it can't, because that indicates _inability_ to screw up. Wubi/Ubuntu _can_ screw up your XP installation and/or hard drive **if** misused; eg; misuse of the "dd" command.

While Wubi/Ubuntu have the ability to "screw up" your Windows XP installation and/or hard drive, it is highly unlikely as long as you are (relatively) careful.

I suggest you go for it.

CAVEAT EMPTOR.
