### Post by Vivaldi Gloria on 2008-07-28
I suggest you simply start with kernel-19 rather than kernel-20. This is much easier to do than trying to fix kernel-20.

Press Alt+F2 and start

```
gksudo gedit /boot/grub/menu.lst
```

Kernels are at the end. It should look something like this:

```
title		Ubuntu 8.04, kernel 2.6.24-20-generic
root		(hd0,0)
kernel		/boot/vmlinuz-2.6.24-20-generic root=UUID=fb18d006-4ec8-41d3-bb25-8a1f643a99aa ro quiet splash
initrd		/boot/initrd.img-2.6.24-20-generic
quiet

title		Ubuntu 8.04, kernel 2.6.24-20-generic (recovery mode)
root		(hd0,0)
kernel		/boot/vmlinuz-2.6.24-20-generic root=UUID=fb18d006-4ec8-41d3-bb25-8a1f643a99aa ro single
initrd		/boot/initrd.img-2.6.24-20-generic

title		Ubuntu 8.04, kernel 2.6.24-19-generic
root		(hd0,0)
kernel		/boot/vmlinuz-2.6.24-19-generic root=UUID=fb18d006-4ec8-41d3-bb25-8a1f643a99aa ro quiet splash
initrd		/boot/initrd.img-2.6.24-19-generic
quiet

title		Ubuntu 8.04, kernel 2.6.24-19-generic (recovery mode)
root		(hd0,0)
kernel		/boot/vmlinuz-2.6.24-19-generic root=UUID=fb18d006-4ec8-41d3-bb25-8a1f643a99aa ro single
initrd		/boot/initrd.img-2.6.24-19-generic

```

Of course your uuids are different. Now comment out lines belonging to kernel 2.6.24-20 by putting # in front of them.

```
#title		Ubuntu 8.04, kernel 2.6.24-20-generic
#root		(hd0,0)
#kernel		/boot/vmlinuz-2.6.24-20-generic root=UUID=fb18d006-4ec8-41d3-bb25-8a1f643a99aa ro quiet splash
#initrd		/boot/initrd.img-2.6.24-20-generic
#quiet

#title		Ubuntu 8.04, kernel 2.6.24-20-generic (recovery mode)
#root		(hd0,0)
#kernel		/boot/vmlinuz-2.6.24-20-generic root=UUID=fb18d006-4ec8-41d3-bb25-8a1f643a99aa ro single
#initrd		/boot/initrd.img-2.6.24-20-generic

title		Ubuntu 8.04, kernel 2.6.24-19-generic
root		(hd0,0)
kernel		/boot/vmlinuz-2.6.24-19-generic root=UUID=fb18d006-4ec8-41d3-bb25-8a1f643a99aa ro quiet splash
initrd		/boot/initrd.img-2.6.24-19-generic
quiet

title		Ubuntu 8.04, kernel 2.6.24-19-generic (recovery mode)
root		(hd0,0)
kernel		/boot/vmlinuz-2.6.24-19-generic root=UUID=fb18d006-4ec8-41d3-bb25-8a1f643a99aa ro single
initrd		/boot/initrd.img-2.6.24-19-generic

```

Save & restart. That should do it.
