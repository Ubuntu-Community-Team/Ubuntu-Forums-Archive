### Post by Yoooder on 2007-03-01
I use Ubuntu 6.10, and there is an package in apt called "kubuntu-desktop" that, when installed, gets my the KDE/Kubuntu environment.

You can try to find an "ubuntu-desktop" package, which should give you the same net result (*if such a package exists*).

Once you get GNOME as a registered Desktop, to switch you log out, hit the "Select Session" button, and select the environment you would like to log into.  Then log in as normal and up will come your Desktop
