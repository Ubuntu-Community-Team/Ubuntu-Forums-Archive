Apparently, in new Opera versions, this option has been removed.
How can I now run it without the tray icon?

---

### Post by scouser73 on 2011-05-08
Open up Opera and in the address bar type **about:config** now in the search field type **Tray Icon** and the resulting search will be displayed, then simply untick the box & click on Save restart Opera and no more tray icon.
