### Post by uRock on 2011-03-04
Hello and welcome to the forums.

Please wait 24 hours before bumping.

If you are using the Netbook edition, then you will have to select the gnome desktop at the log in screen. Then after logging in you can right click on the desktop, then select change desktop background then click the Themes tab and you will see the Install button at the bottom of the window. You can click the install button, then navigate to where the theme is that you wnat to install and it will be installed for you.
