### Post by GadgetsGuy on 2007-09-22
I was never 100% satisfied with utorrent (via WINE) on linux ...

I love utorrent on my windows boxes, and tried everything I could to port it to Linux, but there were always little 'bugs'.

A much better option for Linux users, is to install KTorrent (a UTorrent clone for KDE).

KTorrent installs perfectly fine in gnome, and even runs from an icon in 'internet' meun, rather than a terminal (as is the case with many KDE programs you can run in gnome).

KTorrent is an awesome UTorrent replacement, and to be totally honest, I wish KTorrent was ported to use with w32 systems ...

You can find Ktorrent in the Synaptic Manager, or download a platform specific file at

[http://ktorrent.org/index.php?page=downloads](http://ktorrent.org/index.php?page=downloads)


Cheers!

### Post by Mud.Knee.Havoc on 2007-09-22
When you click the utorrent icon in the tray, it makes the utorrent window disappear.  When you click it again, it appears.  You can do the same thing with right clicking the tray icon and selecting show/hide.

Strangely, though, if you have the window open and switch workspaces/virtual desktops and then back to your desktop with utorrent open, it won't redraw the application - so you need to click the tray icon again so you can see utorrent properly.

Ktorrent's nice (when there aren't showstopping bugs), but private trackers won't usually let you run it.
