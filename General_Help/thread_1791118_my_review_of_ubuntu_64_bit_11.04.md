### Post by EgoGratis on 2011-06-26
> **mightee said:**
> I use my all applications maximized, so in windows i need only one click to jump another window, but ubuntu's this new version, i have to click at least 2 times ( first move your mouse to the left, wait for the menu appears, then find the app, then click on it (if there are multi windows you have to click one more time))

Try this 2 things:

1.) Install compizconfig-settings-manager and in Unity settings set Hide Launcher to Never.
2.) When u have a lot of applications opened try pressing key combination System Key (Windows logo on it) + W.

I think using this two things Unity does allow user to switch between application very fast. After first step you will see when you have for example six instances of FireFox browser opened and you (right) click on the launcher it bring all instances of application forward and you can easily choose the one you are searching for.

### Post by EgoGratis on 2011-06-26
> thanks man this was really helpful, but i couldnt install Unity settings :S

First of all try this. Press System Key and you will see Unity Launcher and every Launcher will have number or letter on it. Try pressing that letter on number to see what happens. Then do the same but press System + Shift + number or letter... More info:

[http://www.webupd8.org/2011/04/ubuntu-unity-keyboard-shortcuts.html](http://www.webupd8.org/2011/04/ubuntu-unity-keyboard-shortcuts.html)

About installing CCSM try:

sudo apt-get install compizconfig-settings-manager

Then find it in Application Lens or press ALT + F2 and write about:config in the search bar. Unity does get very productive once user get a grip on it!
