### Post by ajgreeny on 2016-08-04
Yes.  Gimp's save menu is unlike many other image manipulation applications.  You can either choose to overwrite or you can choose to export the edited image as something else; if you choose save it will by default save it as a new similarly named file.xcf, the gimp default image type.

---

### Post by SeijiSensei on 2016-08-04
If you imported a JPEG or PNG graphic, hitting "File > Save" in the GIMP menu will generate a file in GIMP's own .xcf format.  If you want to save it back out in its original format, you must use File > Export.
