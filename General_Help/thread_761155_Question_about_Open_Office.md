### Post by MosMusy on 2008-04-20
I know in Microsoft Word it is possible to set symbols (e.g. letters with macrons or accents) as a shortcut key  so (alt + e) would give me an e with a macron. Is that possible in open office? I work a lot with languages so such a thing would be of great help.

---

### Post by kyphi on 2008-04-21
On your desktop, you can do that by going to System -> Preferences -> Keyboard -> Layout Options.

Set a Compose Key ( I use the right windows key on my keyboard).

To print letters with diacritical marks press the compose key + the mark you want and then the letter:  á, â, à, ñ, ä, etc.

is the way to go. I have selected "Right Ctrl" as my compose key and can put, say a tilda ~ on n by pressing **Right Ctrl + ~** and then **n** to get to **ñ**. The same is also valid for Windows, you press say **AltGr+`+a** to get **à**. Once you get used to it, it will be a second nature.
