Consider your computer a file cabinet.

Open an empty drawer... set it up for a specific type of media (hanging file folders for instance) - that's your partition.. 

It's the 'basic structure' for a collection of data. Once it's formatted (like a drawer being setup for alphabetical, numeric, etc) then it's a 'volume'.. 

Now each set of papers in those hanging folders is the same as a 'folder' (or directory) in an OS, typically.

See the similarity?

PDF is the 'standard' many governments and company's use - it's a good format for compatibility.
