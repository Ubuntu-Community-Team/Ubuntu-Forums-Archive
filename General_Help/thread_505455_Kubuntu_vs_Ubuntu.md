### Post by jimisdead on 2007-07-20
I have been using kubuntu since the beginning, and found it to be a great distribution. However I was getting a few bugs and decided to attempt a switch to Ubuntu... now, less than a month later I am switching back to Kubuntu - here's why:

- I have a few accounts for different uses, and sometimes I like to switch between them. On Kubuntu this was no problem - on Ubuntu I am left with the mouse pointer on a black screen. I can't restart X, or switch to a different terminal, all I can do is a hard reboot. (nVidia 6800 go, with both the repository drivers, and the latest download)
- When I log in I like to have the windows/applications open up that I had when I logged out. KDE never had a problem with this. Gnome seems to wait 1-2 minutes after the desktop has appeared to load the applications I had running, and seems to completely forget about window sizes when using the file browser.
- Compiz works great (I know it's still being developed, but KDE still has bigger issues with it)... but after installing one user account has it on by default, while another doesn't :confused:
- Maybe I just haven't worked out how, but I need several clocks on my task bar for different locations around the world. Easy in KDE, but although I can add multiple clocks to the bar in Gnome they all have the same time.

On the plus side Ubuntu has some great little tools to get new users up and started, but they serve almost no use to a more experienced user.

With the advances in compiz-kde, it's easier wireless networking with knetworkmanager, it's better account switching, and a few other things - I'm going back to Kubuntu. But I wanted to know if the above problems were just something everyone lives with in Ubuntu, or if there was something strange going on with my installation (standard vanilla install of edgy)?
