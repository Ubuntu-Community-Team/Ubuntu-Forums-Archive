### Post by mercurus on 2004-11-02
[QUOTE=tanari]I don't want to use khotkeys
and I don't know how to make scripts[/QUOTE]

Scripting is VERY useful ... in a multiplicity of languages ... but GNOME provides an alternative.

Go to: Computer,  Desktop Preferences, Keyboard Shortcuts
Scroll down to the "Sound" area, and select the Eject line. When it is highlit, press the hot key (or combination of keys) you'd like to devote to opening and closing the cdrom drive.

Mine works nicely, although it only ejects, it won't close again - I've tried lineakd and a few other methods to get it to work under various operating systems and I think it is the drive, and not the software that refuses to close.
