### Post by pricetech on 2010-03-02
DBAN to fully wipe the drive.

Manufacturer's diagnostics may include a zero fill which will do the same thing except it won't overwrite the drive for security, but if you're planning to install to it, it shouldn't matter.

If you're concerned about the possibility of bad sectors, I'd definitely run the manufacturer's diags.
