### Post by Mark Phelps on 2011-08-01
You might NOT be able to install XP to the SD card -- it depends on how the XP installer views the card.  If it sees it as an internal hard drive, you should be OK; if it sees it as an external hard drive, you will NOT.

Unlike with Linux OSs, Windows OSs are not generally installable to removable devices because MS doesn't want you carrying around a portable version of their OS's.
