### Post by strask on 2012-05-05
You can still recover just an individual file, but you have to do it through the software; you can't for example browse through the backups using nautilus.

One thing that has annoyed me so far (I just started using it) is that in the destination file store, it keeps everything in files of about 50mb in size. So there are thousands of files in that directory; I would prefer a single or at least smaller number of files per backup.
