### Post by sigurnjak on 2009-07-03
If anyone is interested this is reply i got from developers :

>
> I did google it quite a bit but still i can not find it's equivalent
> of Firefox user profile .
> Does it have one ?

Sorry it does not.

> How do i copy it  to a another PC or back it up ?

On Linux all user data is located in .local/share/data/Arora/

> Does it have it's own chrome folder for tweaking settings ?

Currently nearly all of the settings are shown.  There are a few hidden settings, but probably only a half dozen currently and they are more as backups in case something breaks we can tell people how to disable something.  For the most part we have tried very hard to fix bugs rather then adding options to work around them (we have received a lot of requests for settings that are just to work around some bug).

> Is there a way to import firefox passwords ?

Not yet, the password support is not finished.

> Can i add more buttons to a toolbar ?

Also not yet.   As you can tell Arora is a pretty young project :)   Copying Chrome's toolbar extensions seems like it might be a good way to go in the future.
