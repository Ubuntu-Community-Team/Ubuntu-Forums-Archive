### Post by ltmon on 2005-11-15
Hi,

I use gnupod to fix my ipod when things get nasty.  I tend to hack around with mine a bit, so this has been a life saver on more than a few occassions.  It's command line only, and bit arcane to use, but it really does work.  Once I have my firmware back in shape and the iTunesDB recreated I go back to using amarok or gtkpod or whatever.  It's in the universe repository.

See: [http://www.gnu.org/software/gnupod/](http://www.gnu.org/software/gnupod/)

I'm using it on a 4G ipod photo, so YMMV.
