### Post by plucky on 2011-05-04
> **hey! said:**
> just new using ubuntu 11.04 here.....need help..... just installed ubuntu 11.04 and connected directly to a linksys router, my other pc using windows xp is connected wireless. both of them can go on-line but the thing is that anybody can connect through my wireless connection. is there a way for me to secure my connection? thanks!

That is a setting on the router to set up a password to connect to the wireless.You need to login to the router and set up a wireless password connection.

The Router user manual should tell you how to do this.

You will also need the password to login to the router to make changes.

Try to use WPA encryption as WEP is easily hacked.

Good Luck
