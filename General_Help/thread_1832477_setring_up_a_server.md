### Post by HereInOz on 2011-08-24
Try this link:

[http://www.howtoforge.com/perfect-server-ubuntu-natty-narwhal-ubuntu-11.04-ispconfig-2](http://www.howtoforge.com/perfect-server-ubuntu-natty-narwhal-ubuntu-11.04-ispconfig-2)

It is a guide to building an ISP type of webserver, where you can host multiple domains, email mailboxes, etc, so it may not be exactly what you are after, but it could give you a good start.

The process is not too difficult.  The only thing I would mention is that when you choose a password for MySQL, do not use anything other than letters and numbers.  If you put other characters in there, MySQL will fail.
