### Post by andrewsawyer on 2005-09-17
I've got aMule installed on my laptop and I've had no problems.  If you are unsure, set up a shared directory with no files in it and then just watch and see what happens.  If people start downloading off you, you can be pretty sure that it's allowing them access to more than what you want.  If there is nothing, then go from there.

aMule lets people download partially complete files off you though, so be aware that if you are downloading you can expect that there may be uploads of these same files also.

I hope that helped.
