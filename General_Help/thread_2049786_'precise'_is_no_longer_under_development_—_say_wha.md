### Post by cprofitt on 2013-02-18
I used to be under a similar mis-understanding about bugs and development.


[LIST=1]
[*]Any stable release is 'no longer under development
- currently 13.04 is under development.
[*]Being under development does not have anything to do with support
[*]Once a version of Ubuntu is release bugs require an SRU (Stable Release Update)
[https://wiki.ubuntu.com/StableReleaseUpdates](https://wiki.ubuntu.com/StableReleaseUpdates)
[*][Here]("http://ftbeowulf.wordpress.com/2011/11/30/ubuntu-user-i-have-a-problem/") is a post I wrote about this topic - it is not that there is no interest in fixing bugs, but there is a much more stringent process to doing so.
[/LIST]
