### Post by synapsys on 2009-06-19
If you are exposing a Ubuntu machine to the internet as a server you will definately want to use the server edition. You could probably do what you want with the desktop edition, but it is not nearly as secure. There are plenty of guides available on the interwebs that will show you how to set it up.

### Post by wojox on 2009-06-19
If your comfortable with the desktop then you would just need to download some other packages like apache2, mysql, php etc...

If you want to install the server edition go for it. The diffrence mainly is it uses a diffrent kernel than desktop and there is no GUI. It's all command line.
