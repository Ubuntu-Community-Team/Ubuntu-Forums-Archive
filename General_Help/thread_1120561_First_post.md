Why not boot from a USB stick.  In Windows download your Ubuntu CD image and then download a program called Unetbootin.  Here...

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

Unetbootin will copy the Ubuntu ISO to a memory stick so you can boot and install from it.  You will have to change the boot order in your PC's BIOS to make the USB stick the first bootable device.
