### Post by cak3 on 2009-09-11
You CAN get firefox 3.5 ("sudo apt-get install firefox-3.5") it will just be listed as shiretoko in the menu, since it isn't branded in jaunty (it is in karmic). It is the full and current release of firefox-3.5 though. I don't know what the other thing is, but generally you can simply download and compile the newest version of just about anything, if necessary.

You could see if its packaged elsewhere, or search packages.ubuntu.org and see if its updated in the karmic repos, but beware, installing software from repos of other releases is generally not a good idea, since it can lead to system instability (especially if it needs to install a bunch of dependencies... will probably break a lot of packages).
