### Post by Topsiho on 2011-07-10
I think it's unwise to install and try a distro that you can't fully trust. It might be OK, it might be not. If you are not SURE, then don't use it.

Why don't you use just Ubuntu, downloaded from Ubuntu itself, and check it by it's checksum(s)?

You might as well stick to Windows and download applications from all kinds of sites.

If it's Unity, or Gnome3, that you don't want, you might try Mint (which to my opinion is not better), or use 10.04, which lasts longer (after July 21st you may download the newest version, which means less updates after the install).

If you insist on 11.04 (or derivatives) then you might want to know 11.04 gives you the option of using Classic Gnome, which is the ordinary Gnome2 desktop.
