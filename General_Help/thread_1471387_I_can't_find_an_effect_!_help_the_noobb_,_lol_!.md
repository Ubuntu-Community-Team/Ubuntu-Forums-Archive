### Post by QIII on 2010-05-03
Do you have CompizConfig Settings Manager installed?

You can tell if you do by going to System | Preferences and looking for the entry "CompizConfig Settings Manager"

If it is there, go to the "Effects" section and click on "Animations".  From there, click  on "Focus Animation".

If "Dodge" is not already in the list, click "New".  In the pop-up box, select "Dodge" in the "Focus Effect" dropdown.  Set "Duration" at about 250 to begin with.

If "Dodge" is there, select "Dodge", then click Edit.

In the pop-up, move the slider for "Duration" to the left to shorten the time of the effect, right to lengthen it.  I have mine set to 250.  A larger number will make the windows move very slowly.  A shorter number will simply make the windows move so fast that it will appear as though they simply pop to the front when you click them.

If  you do not have CompizConfig Settings Manager, it can be installed via Synaptic.  Search for compizconfig-settings-manager and install it.

After installation, do as I have described above.
