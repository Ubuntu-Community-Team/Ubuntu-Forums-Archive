### Post by hyperdude111 on 2009-01-19
This happened to me in songbird (also mozilla) this is what i did, 
1. Allow hidden folders the go to "home"
2. Look for a ".mozilla" folder and delete it (this will remove all your settings, addons and bookmarks so sorry)
3. When you re-start firefox it will work like it was brand new everything as it used to be.

To be safe bake a backup of that folder so if it does not work you can restore your settings.
