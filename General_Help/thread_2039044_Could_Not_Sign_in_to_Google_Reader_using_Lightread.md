### Post by zeroseven0183 on 2012-08-08
Well,  here's the thing. The account password won't work if the two-factor  authentication is turned on. I was able to sign in to Lightread by  registering the application to my Google account.
 

To do that,
1. I logged in to my Google Reader via web browser
2. browsed to Security > Authorizing applications and sites
3. verified my account on another login page
4. generated an application specific password for Lightread
5. copied the one time password to the password field of the application
6. then signed in


 I never thought that was the way to do it.
