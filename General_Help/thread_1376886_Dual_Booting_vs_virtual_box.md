### Post by oldgeekster on 2010-01-09
> **UncleMonty said:**
> What are the advantages/ disadvantages of runnings windows xp in virtual box as opposed to dual booting?For my 2 cents, it is far easier to set up Sun Virtualbox (personal license), then run XP within a virtual machine than it is to fight dual boot and all of the issues associated with that mix.

I tried the open source "free" version from Canonical available via the Ubuntu Software Center - then ran into all of the (much discussed) problems with USB devices.  Have had no problems running Sun Virtualbox (latest version) which is "free for personal use".

There is also a certain satisfaction that comes with shutting down XP, then Virtualbox and knowing your machine doesn't need to be sterilized. ;)

**Edited**: Should have mentioned I am NOT a gamer. Most of the decision will be based upon what you do with it - there IS a very discernible performance hit when running any virtual machine. For what I do in XP it is negligible where others would find it too frustrating to deal with.
