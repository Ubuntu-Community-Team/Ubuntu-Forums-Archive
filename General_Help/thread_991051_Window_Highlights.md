### Post by loomsen on 2008-11-24
[[img]http://www.ubuntu-pics.de/thumb/6285/screenshot_03_6ZqzOD.png[/img]](http://www.ubuntu-pics.de/bild/6285/screenshot_03_6ZqzOD.png)

Rightklick on your desktop, then go to the tab themes, and hit customize.

If you need more colors, or options more likely, get gnome-color-chooser.

```
sudo apt-get install gnome-color-chooser
```

Goes to System->Preferences menu
