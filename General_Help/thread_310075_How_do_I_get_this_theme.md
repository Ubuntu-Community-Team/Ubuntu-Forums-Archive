### Post by Lord Illidan on 2006-11-30
You have to use Edgy Eft, I think.

The theme is called Silicon.. and it is in the repos : sudo apt-get install silicon-theme.

To delete top panel bar? Right click on it and click on "remove this panel", then right click on the time applet and select "move" and move it to the bottom.

As for gnome-terminal, all you need to do is click on Edit -> Profiles -> Default -> Edit -> Effects and there is the transparency option there.

To make other apps transparent, you need Beryl/Compiz.
