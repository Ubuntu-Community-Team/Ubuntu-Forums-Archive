### Post by irv on 2007-10-25
I have a question.
How does Ubuntu make an Install live CD?
The reason I ask, is because I would like to take my laptop OS just the way it is and burn it to a DVD image so I could just put it into my DVD drive and boot with it to re-install it if the need arises. I have backups but I would like to be able to put everything exactly like it is if I ever have a hard drive crash.
Irv

---

### Post by LiMaO on 2007-10-25
You may use partimage to create an image of your whole disk.. that way, if you ever want it to be restored, just have to restore the image.

Partimage should be run from a LiveCD (just use ubuntu livecd) -- both to create the image and to restore it.

That's an option =)
