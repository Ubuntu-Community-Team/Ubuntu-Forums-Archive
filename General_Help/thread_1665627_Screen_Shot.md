### Post by howefield on 2011-01-12
Pressing the printscreen key should bring up the screenshot save window so that you can name and save it to your preferred location.

Likewise using the menu launcher, Application > Accessories > Take Screenshot.

If you are using something like imagemagick via the terminal, it defaults to the home folder.
