### Post by jamesstansell on 2008-07-01
They're both worth trying.  The main difference would be in the JVM implementation, as they both use the GNU Classpath library.  What jar files do you want to run?  Ones you've created?

Another Free Software possibility is the OpenJDK, which uses the Sun class library and HotSpot JIT compiler.
