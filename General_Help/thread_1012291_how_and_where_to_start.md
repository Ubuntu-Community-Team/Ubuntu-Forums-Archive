### Post by drs305 on 2008-12-15
If you have decided on ubuntu, you could start here by looking at the specs:
[Installation/SystemRequirements]("https://help.ubuntu.com/community/Installation/SystemRequirements")

There are various distros of ubuntu, including ubuntu, kubuntu, xubuntu and more. If you look at this link it will show you how long the ubuntu versions will last (see next paragraph), but at the bottom are links to the various other ubuntu variants:
[Releases]("https://wiki.ubuntu.com/Releases")

You will have to decide which version you want.  8.04 Hardy Heron is a long-term support version which will be supported until April 2011 but will not have many updates to newer versions of applications (other than security updates or bug fixes). The newer 8.10 Intrepid Ibex will have more frequent updates but will not be supported quite as long as Hardy. The choice is yours - a bit more stability or newer software.

Next you would probably want to download the iso, so here are instructions on how to do that:
[BurningIsoHowto]("https://help.ubuntu.com/community/BurningIsoHowto")

You would have to decide how to partition your drives, and for that I would recommend you spend some time reading the information on this site. There is plenty of good info about most aspects of installing and setting up ubuntu:
[http://www.psychocats.net/ubuntu/index]("http://www.psychocats.net/ubuntu/index")

Jump on it. The water is fine and there are plenty of people around to throw you a rope if you should need one.

Welcome to ubuntu.
