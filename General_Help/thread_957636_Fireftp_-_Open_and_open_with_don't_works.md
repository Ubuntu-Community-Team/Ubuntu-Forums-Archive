### Post by cjshaw on 2009-10-24
To add gedit to open a file type in fireftp, such as a php file:
1. Right click on the file
2. Select 'Open With' -> 'Add Program'
3. Full in: Name. Gedit, Path. /usr/bin/gedit
4. Click Apply.
