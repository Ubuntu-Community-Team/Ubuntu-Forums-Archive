### Post by berniedd on 2010-04-26
I did a clean install of 9.10 version in a single core Pentium 4 without issues. I had my HP 3550 Deskjet connected to the rear USB port during the installation(*USBs are enabled in BIOS), so it would be automatically detected during the install.  However, when I try to print something, the printer would feed the paper in, but when it comes out there is nothing printed on it. I have checked the box "print black" in the printer menu options. I know it is not a printer problem because this same printer can print nicely when connected to another computer. I know it is also not the USB ports because they can read USB flashdisks connected to them. Any ideas?

---

### Post by berniedd on 2010-07-30
Sorry for the delayed update. But the cause of the problem revealed itself a few days later. It was the motherboard - on its last legs. Finally gave up the ghost. The replacement (used) motherboard was run on another install using the same live cd and voila, no more printing problems.
