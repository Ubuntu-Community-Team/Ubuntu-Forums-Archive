### Post by chrestomanci on 2006-05-28
If you mean a dial up modem, then the main thing is to make sure it is not a "winmodem" (a modem without a proper chipset, that uses the CPU for all the work, and only works with windows drivers).

Your best bet would be to buy an external modem that connects via a serial port. (not USB). as that would be garanteed to be a propper modem.
