### Post by mike2357 on 2008-05-08
If you really want them to start when you log in, then you can select System -> Preferences -> Sessions and add them.  The Command box will need the program name; you can browse to /usr/bin and find both pidgin and tomboy.

Another strategy is to put launchers on your panel.  You can right-click on your panel and select "Add to Panel."  There is already an applet for Tomboy.  Just scroll down to it, click on it, and select Add.

For Pidgen, you can select Application Launcher from the top and navigate to the menu where Pidgen lives now.

Good luck.  There are probably other solutions, too.  Don't worry about asking newbie questions.  We're all learning.
