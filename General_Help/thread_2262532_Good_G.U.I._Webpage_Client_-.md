### Post by Lars Noodén on 2015-01-25
Ok.  Bluefish is in the repository, it's popular.  Kompozer was great but has been abandoned and Bluegriffon is not yet in the repository.  I used to use Emacs myself.  

About SFTP clients, you don't need FIleZilla unless you like it.  The file managers, namely Nautilus, all support SFTP now.  Just press ctrl-L and then enter the URI.  
*sftp://user@somesite.example.org/some/path/*

### Post by Lars Noodén on 2015-01-25
The [ctrl-L will work within Nautilus](https://www.youtube.com/watch?v=9S4DV1PluzA), if you are using stock Ubuntu.  If you are using Kubuntu, Xubuntu, or Lubuntu, they have different file managers but those also support SFTP.  Once you are connected using SFTP, you can drag and drop files on the remote machine as if they were local.
