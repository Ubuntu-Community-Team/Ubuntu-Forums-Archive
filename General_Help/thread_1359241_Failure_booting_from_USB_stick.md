### Post by bit mad on 2009-12-19
If you're doing it in Windows, try the LiLi creator to make a USB stick that you can save changes to (persistent) - see the free GPL LiLi website [http://www.linuxliveusb.com/](http://www.linuxliveusb.com/)

Don't forget to format the stick as FAT32 first - you can do that from LiLi but might as well do it beforehard :)

Other things to watch out for - the USB key may be one of the unusual ones that don't work - and check the downloaded ISO was OK by checking the 'MD5 checksum'
