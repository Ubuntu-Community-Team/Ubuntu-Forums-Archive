I think it depends on the .docx file. I found a couple from doing random searches on the internet.  The problem seems to come when the file has a bunch of footnotes, graphics files etc.  You could try softmaker office, supposedly that does a good job of importing and exporting Microsoft files.

It looks like it's from the Windows 3.1/95 time frame  IMO.  It did a decent job on my problem files but didn't load one graphic.  I've actually had pretty good success with OpenOffice/LibreOffice 3.3.3 on Natty.  I'm not sure if it's the newer O_O/L_O or Gnome 3 but L_O 3.4.3 didn't seem as reliable opening complex .docx files.  That's my limited experience.
