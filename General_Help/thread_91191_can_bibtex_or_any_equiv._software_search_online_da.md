Almost all of the web interfaces to library databases offer a bibtex-formated download so even in the worst case, all you need to do is copy and paste the reference into your bibtex database. (I do this because I don't always want all the cruft that is included with the canned entries). Some of the newer bibtex management tools may offer database connectivity directly however.
