### Post by Bucky Ball on 2008-10-23
[https://help.ubuntu.com/community/DataRecovery](https://help.ubuntu.com/community/DataRecovery)

There are a number of apps that can help you find bits and pieces, but if you installed manually and you ticked the box that said format drive, well ... Try testdisk

The data isn't actually deleted unless you purposely write zeros over it, but is 'forgotten' so to speak. Marked as 'writeable' so the bits where Ubuntu hasn't written too could still be retrievable. The bits where it has of course will not be. You can safely forget the Windoze install as Ubuntu runs on EXT3 formatted drives/partitions. Windoze does not. ;(
