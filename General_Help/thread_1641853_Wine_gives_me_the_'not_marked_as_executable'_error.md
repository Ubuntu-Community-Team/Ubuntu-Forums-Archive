### Post by mc4man on 2010-12-09
Notwithstanding whether 'peggie' will run or whether you should have it on and run from your ubuntu install instead - 

right click on the .exe (or any .exe ), -> Properties -> Open With -> Add -> Use a custom command
For the command just use - 
wine
see screen
This will bypass the inane security bit policy that 'Wine Windows Program Loader' uses
