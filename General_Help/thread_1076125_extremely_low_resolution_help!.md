### Post by lswest on 2009-02-22
I fixed that issue for myself by inputting the displaysize to the Monitor section of my xorg.conf

```
Section "Monitor"
    Identifier     "Monitor0"
    VendorName     "Unknown"
    ModelName      "CRT-0"
    HorizSync       28.0 - 33.0
    VertRefresh     43.0 - 72.0
    DisplaySize     1280 1024
EndSection
```

Of course, I recommend making a backup of the file first with ```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak
``` and then making the change and restarting X.

Hope it helps,
Lswest
