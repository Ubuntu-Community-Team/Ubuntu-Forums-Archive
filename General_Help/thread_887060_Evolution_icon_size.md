I had small icons and no text set and had to remember where I changed it. I found this in the [Evolution FAQ]("http://www.go-evolution.org/FAQ#How_can_I_decide_to_show_only_text.2C_only_icons_or_text_and_icons_in_my_toolbar.3F")
> ** How can I decide to show only text, only icons or text and icons in my toolbar? **

 This is a Gnome wide setting. Either you can edit it either by using the gnome-control-center or by editing the GConf key /desktop/gnome/interface/toolbar_style. 
As far as a calendar within a Linux email client the only thing I know of is the Lightning addon in Thunderbird. But I recently switched from Thunderbird to Evolution, in part for better compatibility with my PDA, and I'm loving Evolution.
