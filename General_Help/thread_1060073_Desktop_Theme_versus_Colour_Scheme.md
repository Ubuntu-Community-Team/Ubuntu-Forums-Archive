### Post by Darkshade on 2009-02-06
> **Georgie.Mathews said:**
> Good day all

I am having a problem with the desktop theme versus the colours that KDE 4 uses (I am running 4.2 currently on Intrepid). I am using the glassified theme but this does not go too well with the available colour schemes..can't the colour scheme and desktop scheme be the same thing?

No, they cannot be the same thing. Plasma uses it's own themes for the panel, menu and plasmoids (widgets).
KDE4.2 is still fresh and there aren't many themes for it yet. Give it some time ;)
Meanwhile I would suggest you to give Aya a try. The theme uses the system colors so it integrates well with pretty much any color scheme. You can find it in [http://kde-look.org/](http://kde-look.org/) -> Plasma themes or just right clicking on your desktop -> Appearance settings -> New Theme -> Aya.
