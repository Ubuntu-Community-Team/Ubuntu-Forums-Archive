### Post by mgmiller on 2007-02-18
It could be static electricity.  I have a P-180 case running Ubuntu that is pretty sensitive to that.  When the humidity is low,  if I try to plug in a flash drive, the machine will reboot.  If I even touch parts of the case when the humidity is low, it will reboot.  I pretty well solved this by installing a large humidifier in my house.  It almost never happens now.
