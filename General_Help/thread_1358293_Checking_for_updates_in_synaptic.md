### Post by houseworkshy on 2009-12-18
If you don't the list will be out of date so yes it's best if one does.  If you are planning a big installation is also a good idea to use the quickest server.  To find it :-  System>Administration>Software Sources>Ubuntu software,  which is the first tab on which there is a drop down menu "download from"  Use that to search for the fastest one before reloading.  This can save time over the whole process.
