### Post by Mark Phelps on 2013-06-29
With a CD-RW, you can only add one session at a time -- you can't just write files to it in the same manner that you would a filesystem -- unless you write using UDF, which I believe is NOT the default.  Without using UDF, you can't remove files or directories from the CD.

And, if you use the option (intentionally or accidentally) to Finalize the session, I believe that prevents adding any more sessions.
