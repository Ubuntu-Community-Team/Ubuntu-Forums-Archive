### Post by dwasifar on 2007-01-29
Might I suggest you pass out Kubuntu CDs instead of Ubuntu?  While I myself prefer Ubuntu and Gnome, Kubuntu is probably going to be more attractive to the potential Windows convert; it's prettier and looks more Windows-like.

I would edit a little bit of the tech talk out of the instructions.  If you don't mind my presumption, here's my stab at that:

---

Replace Windows for Free! No Viruses! No Spyware! No Disk Maintenance!
Linux Ubuntu 6.06 LTS

- This CD contains a complete operating system (OS) to replace Windows - Linux Ubuntu 6.06 LTS. 

- Make as many copies as you want, completely legally.  Ubuntu is Open Source Software.

How to preview: This is a "Live CD."  You can try it without interfering with Windows.  

(1) Put this CD into your CD drive and restart your computer. Ubuntu will load and run directly from your CD drive.  Run this way, it will not change Windows or anything else on your hard drive. 

(2) If the computer still starts Windows even though you have the CD in the drive, look at your computer's manual to find out how to tell the computer to boot from the CD drive.  If you don't have the manual, just Google "change boot sequence" and you'll find tutorials.  (Don't worry. If you mess up, you can always go back and re-load "set-up defaults.")

(3) Have fun exploring! Right click on the desktop to explore configuration options.  Try out the applications and the games.  Surf the web with Firefox.  Nothing on your computer will be changed unless you decide to install Ubuntu on your hard drive.

(4) If you decide you do want to install Ubuntu, just click the install icon on the Ubuntu desktop and follow the instructions.  You can choose to install Ubuntu along with Windows, or you can opt to erase the hard drive and let Ubuntu have all the space.  Whatever you choose, Ubuntu will handle the "tech stuff" and help you through the installation.

Ubuntu is free software, and so are all the programs and games that come with it.  It even updates itself over the internet, for free.  More people every day are discovering the freedom and security of Ubuntu and Linux for the desktop, and saying goodbye to high-priced, virus-vulnerable Windows.   Try it!


---

You'll notice I completely eliminated any mention of support.  This is because Windows people think of "support" as hand-holding on the telephone, not volunteer user forums or paid service contracts to Canonical.  It will confuse and frustrate them to expect one kind of support and get another.
