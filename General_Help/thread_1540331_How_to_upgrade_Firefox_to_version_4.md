### Post by marshmallow1304 on 2010-07-27
To install FF 4 properly (i.e. not just for testing purposes):
[http://linuxhub.net/2010/07/install-firefox-4-beta-2-in-ubuntu-using-ppa/](http://linuxhub.net/2010/07/install-firefox-4-beta-2-in-ubuntu-using-ppa/)

To install FF 4 for testing purposes:
Download from the [Mozilla site]("http://www.mozilla.com/en-US/firefox/beta/") then install [FoxTester]("http://foxtester-extension.blogspot.com/") in your current FF.  Put the tar.gz in a folder, set FoxTester to watch the folder, then you can add, switch to, and even set as default the new FF.
