### Post by fballem on 2012-11-11
Are you by chance migrating from 10.04?

If you are, then please be aware that much has changed:

[LIST=1]
[*]Ubuntu now uses Gnome 3. 10.04 used Gnome 2. Most of the major distributions have migrated to Gnome 3. Check out [http://www.gnome.org/](http://www.gnome.org/) for more information.
[*]Ubuntu now uses the Unity interface. When it first came out, there was much gnashing of teeth, but as people got used to it, many found they actually liked it. You probably have a bunch of icons down the left side of your screen. This is the launcher. The top icon has the ubuntu symbol. Clicking this will lauch the dashboard application. Type what you are looking for in the dashboard, and the chances are pretty good that it will find it for you.
[*]Global menus - for whatever window is active, you will find the menu on the top panel of the screen. If you don't like it (personally I do not) then you can remove it. Open a terminal and type ```
sudo apt-get remove indicator-appmenu
``` will put the menu back into the title bar of each open window.
[*]This guide [http://www.debianadmin.com/getting-started-with-ubuntu-12-04-precise-pangolin-pdf-guide.html](http://www.debianadmin.com/getting-started-with-ubuntu-12-04-precise-pangolin-pdf-guide.html) may be helpful.
[*]Be patient - I wasn't exactly thrilled when I first encountered all of the changes, but as I've worked with 12.04, I am finding it much easier and some good decisions (along with a few bad ones) have been made along the way.
[*]I don't know if it would help at all, but I have a standard set of software that I install on ubuntu. The information on this page [https://wiki.ubuntu.com/fballem/Software%2012.04](https://wiki.ubuntu.com/fballem/Software%2012.04) may be of some help to you as well.
[/LIST]

I hope this helps,

### Post by jerome1232 on 2012-11-11
One thing to note, Unity is very search driven, usually you use search functions to get what you want. Unity is also very keyboard driven, press and hold your super key (windows) key to get a list of all the keyboard shortcuts.

To find applications, tap the super key or click the Ubuntu logo on the top left of the screen to bring out your dash home, which searches all lenses available, type to search, press enter to open the first result.

By default unity in 12.04 comes with 4 lenses, Applications, Files and Folders, Music and Videos. If you right click the Ubuntu logo you can go straight to a specific lens if you want to only search for Music for example. You can also use keyboard shortcuts which you can find by holding the super key and getting the list of shortcuts.

You can also bring up HUD by tapping alt, this is a function that searches through your application menu's, ie if I wanted to go into Firefox's preferences window, I'd tap alt and type pref and press enter (it's drilled down to the preferences option after those 4 letters) 

If you want to do some customizing, install MyUnity.

Adding a launch icon is pretty easy.

1) Start the program from the dash (you will only have to do this once).
2) While the program is running, you will see the icon for the program on the launcher.
3) Locate the icon on the launcher and right-click. A small menu will open up.
4) In the icon menu, you will see an option to Lock to Launcher. Select that option and the icon will remain when you close the program.
