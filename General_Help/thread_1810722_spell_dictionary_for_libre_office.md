### Post by grahammechanical on 2011-07-24
Libreoffice, like Openoffice, now uses Hunspell as the dictionary utility. Hunspell is backwards compatible with Myspell dictionaries.

Try opening the Ubuntu Software Centre and typing German. Click on Show 179 technical items and see what you can find. I can at this moment see a file called hunspell-de-ch-frami which is described as Swiss (German) dictionary for Hunspell ("frami" version).

There is also a Czech dictionary for Myspell. The file is called myspell-cs. I found it by searching for czech. You may need to Show 50 technical itmes to find it.

Regards.
