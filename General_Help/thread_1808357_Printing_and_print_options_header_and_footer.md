When printing labels from ebay or shipping etc. 

You can select the title or footer option to print your notes on the page if you like


Just select title / custom and then type your message so that way the shipping department or whoever will have a note on that printed page if you want.

This is great for printing ups labels from ebay or amazon if you want to print a note on the page for reference or maybe your own internal stock number.
