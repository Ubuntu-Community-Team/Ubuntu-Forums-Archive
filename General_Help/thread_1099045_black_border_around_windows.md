### Post by Ubulubu on 2009-03-17
I was having a problem in Intrepid such that black borders were showing up around all the windows (even tool-tips).  I've since fixed it so I thought I'd post the solution (since I couldn't find another post with the problem and solution).  Anyway, it was a simple fix, from System --> Preferences --> Appearance --> Visual Affects, choose "None" and that solved it.
