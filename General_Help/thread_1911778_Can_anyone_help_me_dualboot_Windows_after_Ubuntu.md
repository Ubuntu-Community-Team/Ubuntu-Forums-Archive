### Post by syerges on 2012-01-19
1. make your ubuntu partition smaller using gparted... you will have to boot from disk to do so.
2. install Windows on the unpartitioned space you just created.
3. use the website finish making Ubuntu dual boot.

( make sure you have everything you need before you start )
