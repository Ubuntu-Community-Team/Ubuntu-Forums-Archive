### Post by Ebere on 2010-05-12
> **soryu said:**
>  no os found in devices.

Sometimes it's the simpleist things that cause the problems.

If you have more than one ide drive, make sure that the jumpers on the end of it are set correctly. (Master, slave, cable select.)

Make sure you are using a hard drive cable, and not an old floppy drive cable.

Make sure the cable is plugged into the drive correctly. Red line on cable should be nearest to the power connector.

Make sure all cables are firnly seated. On BOTH ends. On the MB -and- at the drive.

If you are getting no noise from the device, check to make sure it is getting power. Try a different power connector.
