### Post by bodhi.zazen on 2010-07-08
You have several options, I suggest you install a test server and install several forums and ask for feedback.

Most software uses Apache / php5 , and mysql, most are trivial to set up as a demo.

Personally I like 

[http://www.phpbb.com/](http://www.phpbb.com/)

SMF is nice as well

[http://www.simplemachines.org/](http://www.simplemachines.org/)

See also :

[http://blog.eukhost.com/webhosting/forum-software/](http://blog.eukhost.com/webhosting/forum-software/)

So set up

[http://server/smf](http://server/smf)

htp://server/phpbb

etc 

and try them out. Be sure to look at the admin panels, as you will use them a fair amount. You probably do not need moderators as it is an internal site.
