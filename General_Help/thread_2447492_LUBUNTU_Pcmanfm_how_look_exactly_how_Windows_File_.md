### Post by guiverc on 2020-07-20
You haven't provided your release details, I'll guess you mean Lubuntu 18.04 LTS, as my Lubuntu doesn't use `pcmanfm` at all, modern Lubuntu uses `pcmanfm-qt`

If you don't like the default *file-manager*, there is nothing stopping you from using another one. I possibly prefer `thunar` (XFCE's default *file-manager*, ie. found in Xubuntu), though as it uses different libraries, it'll have a performance cost, which will or won't be noticeable depending on what resources your machine has. In this my main box I won't notice any costs, but an an old 2005 single-core thinkpad with only 1GB of ram, it's easily detected.

If using modern Lubuntu (with LXQt) `dolphin` may make more sense, but `thunar`, `caja`, `nautilus`, `nemo` or any other of the file managers will work too.

If you've have provided you release details, I'd have told you how to look for them, however the tool differs depending on whether or not your using LXDE (*legacy* Lubuntu) or LXQt (*modern* Lubuntu)  (*both tools work on all releases, but the tool provided is the lightest option for your actual release*).
