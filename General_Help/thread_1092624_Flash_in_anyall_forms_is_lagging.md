### Post by fieroboom on 2009-04-06
Hmm... This could be one of those oddities where your 8200 prefers the 173.x drivers. Instead of upgrading, try downgrading to those drivers, and see what happens. I've read that doing so can cause some firefox sluggishness when using compiz, but it load balances flash playing between both processors (or cores), and provides a much smoother playback.
If you don't have envyng, I highly suggest using it:
```
sudo apt-get install envyng-core envyng-qt
```
Then you can fire up envyng and just simply select the 173.x driver, and it will install it automagically. If that doesn't make sense, let me know, and I'll explain further.
:D
-Paul
