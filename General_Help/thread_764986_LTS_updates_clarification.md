The only reasons for package version upgrades, so far, have been security patches. Bug fixes usually don't require upgrading.
Comparing eclipse from various Ubuntu releases gives an idea what's going on with it [http://packages.ubuntu.com/gutsy/eclipse](http://packages.ubuntu.com/gutsy/eclipse). However, it doesn't mean newer versions can't be installed manually.
