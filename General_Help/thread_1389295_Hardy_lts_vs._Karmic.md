### Post by snowpine on 2010-01-24
> **RFXCasey said:**
> If I understand correctly Hardy LTS is being updated all the time as is 9.10. 

Hardy is not really "being updated all the time." All of the application versions are from April 2008 (or older): Firefox 3.0, OpenOffice 2.4, etc. All Ubuntu releases receive only minor updates: bug fixes and security patches. Because everything in Hardy is 18 months older than Karmic, it is considered more stable (more time to find and fix bugs). Both Karmic and Hardy will be supported through April 2011.

To answer your specific question, I would recommend burning two Live CDs and testing which works better on your hardware.

### Post by jmszr on 2010-01-24
I use 8.04 and it is quite stable. I haven't upgraded because the improvements in the following releases had no particular bearing on my computer usage.

I have, using PPA's and old.getdeb.net, upgraded to OO.o 3.1.1, Firefox(Namoroka)3.6 and various newer versions of applications.

That said, 8.04 is starting to get a little long in the tooth. Newer versions of some apps no longer share libs that 8.04 uses, Getdeb only has a legacy website for <9.04, and devs have pretty much lost interest in it.

It is, of course, still very useful and I would recommend it as a 1st install for those new to Ubuntu and whose requirements can still be easily met by 8.04.
