### Post by flyingbrass on 2007-02-28
You need to set Ubuntu to use local time.  See [http://doc.gwos.org/index.php/Setting_Local_Time](http://doc.gwos.org/index.php/Setting_Local_Time)

Windows doesn't get along with UTC.
