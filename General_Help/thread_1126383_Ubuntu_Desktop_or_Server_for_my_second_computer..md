### Post by CRIMPS on 2009-04-15

### Post by FluffyElmo on 2009-04-15
There is very little difference between the Server and Desktop editions, the Server edition basically just lacks the GUI. You can easily run all the server programs on a desktop install, which is what I would recommend. Use it as usual, then pop open a terminal window and you're in the 'Server':)

---

### Post by aeiah on 2009-04-15
for your purposes, you might as well install the desktop edition. the server edition does have a few things (which can pretty quickly be installed on the desktop edition), but it doesn't come with a desktop environment. you'll find it much easier to add server components to the desktop edition than desktop components to the server edition
