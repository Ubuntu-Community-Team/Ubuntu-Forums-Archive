### Post by rufius on 2008-02-05
If someone gains illegitimate access to that control panel where you can delete users, it would wreak havoc. 

Its also pretty standard to not necessarily delete a user whenever they lose access to a system that way if they need any of their files, you can recover them instead of blindly deleting them.

Hope that helps :)
