### Post by Quillz on 2007-02-27
If Dapper works for you, I see no reason to upgrade. All of the software included in Edgy, like Firefox 2, is available with 6.06, so long as you run Software Updates from time to time. In fact, you could probably go a very long time w/o ever updating the distro itself, as you already get kernel updates and software updates as it is.

Now, personally, I would upgrade to 7.04 when it's released, but only because it has improved Wi-Fi support, which is very important to me, since I use a wireless cable connection.
