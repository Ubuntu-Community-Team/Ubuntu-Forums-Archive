### Post by snowpine on 2009-07-22
You can use Unetbootin in "hard drive" mode (however I'm not sure Unetbootin supports Windows 98 ): [http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

You could also swap the hard drive into another computer with better specs, perform the install, then swap it back.

The only modern Linux distro I've tried with good support for 64mb is DSL (Damn Small Linux).

Good luck!
