### Post by howefield on 2015-07-02
Chrome and Firefox passwords are stored in plain text - for Chrome use Settings > Show Advanced Settings > Manage passwords. Not sure about Opera but I'd expect something similar, probably.

Each browser has a sync feature which might be of use, although not sure I'd recommend that route.

### Post by Rex Bouwense on 2015-07-02
Since I only use Firefox, I can only speak to it.  To save bookmarks merely, click Bookmarks and then navigate to Show all Bookmarks.  What you want is under Import and Backup.

### Post by deadflowr on 2015-07-02
Look in the profile directories
Chrome is in ~/.config/google-chrome
Opera is in ~/.config/opera
and Firefox' is in ~/.mozzila/firefox
