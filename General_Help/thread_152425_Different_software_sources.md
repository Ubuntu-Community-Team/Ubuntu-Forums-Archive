### Post by hajk on 2006-03-30
The Ubuntu installer sets archive mirrors based on locale information given early in the install procedure, so you and your friend may have given different answers at that stage. If this bothers you, you can always manually edit the /etc/apt/sources.list file to make the sources the same...;)
