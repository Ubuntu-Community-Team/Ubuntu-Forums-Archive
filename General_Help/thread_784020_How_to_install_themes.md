### Post by issih on 2008-05-06
Sadly this is not quite as simple as perhaps it might be.. I'm assuming that you might have desktop effects (and therefore compiz) enabled, but have not done anything else like installing a different window decorator (i.e. emerald etc).

In that case you want to go to:

System->Preferences->Appearance

That will bring up the appropriate window, where you can tweak things like system fonts, wallpaper image, etc

The tab you want is obviously "Theme". In the assumption that you have downlaoded a full theme, hit install, and point the installer at the .tar.gz file, then select your new theme, and that will be that.

There is a but coming however....
Themes are split into different elements, if you hit the customise button you can see they are made of:

Window Controls Style
Theme Colour (often not a functional feature with many themes)
Window border style
Icon style
Pointer style

These elements can all be installed independently as well as part of a whole theme set, so you need to know what you are installing. If it is a full theme, simply install and select it, if it is just one of these elements, you will have to install it then go into the customise screen and select the new style as part of a custom theme.

Hope that makes sense.
