I want to synchronize my time to a time server in Singapore. How can i do so with the Time and Date in System - Administration - Time and Date?

---

### Post by babur on 2010-10-28
First install "NTP" package, then you will get the option to sync the time and date with Internet servers.

System->Administration->Time and Date

- Choose "Keep synchronized with Internet servers"

- Then "Select Servers", you may add the server if it is available. Or you can add your own server.
