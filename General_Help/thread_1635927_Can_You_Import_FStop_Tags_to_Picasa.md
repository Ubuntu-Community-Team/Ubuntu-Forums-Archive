### Post by AlphaLexman on 2010-12-02
I believe 'digiKam' will do what you want. (It's in the repo's)

Goto: Settings -> Configure digiKam... -> Metadata -> Common Metadata Actions -> _S_ave image tags as "Keywords" tags in metadata embedded in files.
