### Post by CompyTheInsane on 2009-09-21
Or alternatively, you can try [gfire]("http://www.gfireproject.org"). It's a protocol plugin that lets you access Xfire from within Pidgin or any other libpurple-based IM client such as Finch. It has configurable game detection (It doesn't detect what game server you're on though) as well as support for xfire chatrooms. Downsides so far include no voice chat support. It's not a problem if you don't use xfire's voice chat though.
