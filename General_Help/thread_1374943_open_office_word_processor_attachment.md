Hi there, to transfer an OO writer document to a windows (MS-Word) user you need to save the document in OO writer as a MS-Word document.  Once you do this the file extension will be .DOC this will then be compatible with MS-Word but it might be a good idea to save the document as a MS-Word 97 - 2003 format that way whatever version the Word user is using they should be able to open it.

### Post by mcduck on 2010-01-07
OF course if the receiver only needs to be able to view the file, and not edit it, the best way is to export as PDF from OpenOffice.

PDF is pretty much the only way to make sure the file appears exactly the same regardless of the operating system and programs used to view it. Even saving in a MS Office format (.doc or .docx) doesn't ensure the file looks the same as it looks on your system, since different computers have different fonts available, render them a bit differently, and most of all the compatibility between MS Office and other programs just isn't anywhere near perfect.
