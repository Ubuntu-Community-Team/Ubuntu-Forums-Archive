### Post by grahammechanical on 2013-10-18
If you have Ubuntu 13.10 on a USB stick then try inserting the USB stick while you are loaded into 13.04. Some say that Ubuntu recognises that a newer version is on the USB stick and that it will offer to upgrade from 13.04 to 13.10 from the USB stick. I have not tested this myself.

By the way, when each new version of Ubuntu is released there is always a rush to get it and this puts a heavy load on the servers. If we wait a day or two or three things will get better.
