### Post by 73ckn797 on 2011-05-13
If there is a paper size in any options for the printer, then it should center properly by the formatting of the document. The HP printer paper feed should adjust to the size you want with the paper in the tray. You may also be able to create a custom document size in the the program you are using to create the doc. Experiment with a few trial prints to make sure everything lines up as desired. Use a draft print option to save ink.
