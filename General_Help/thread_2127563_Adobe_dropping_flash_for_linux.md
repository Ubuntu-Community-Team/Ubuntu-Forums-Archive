### Post by Impavidus on 2013-03-20
Adobe will still provide security updates, but no more new features (or that's what I read a few days ago on their site). Slowly there will be more and more flash using websites that won't properly load in Linux/Firefox. It is probably not as bad as it seems at first, because flash is mostly used for video's, which are moving towards html5 with "standard" video codecs anyway. I can watch many youtube videos without activating my flash player.

If you're really concerned, you can use chrome, which has, if I understand correctly, a more recent flash player build-in.
