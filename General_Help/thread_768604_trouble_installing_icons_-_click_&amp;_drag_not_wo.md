### Post by dbbolton on 2008-04-26
Here's how you can install this theme:

Dowload the file- it's a .tar file.

Right-click, then do Extract Here

It will make a folder with the same name as the .tar file. Open this folder.

Inside is a .tar.gz file. This is the archive that you can install by dragging into the theme manager.

Hope this helps!
