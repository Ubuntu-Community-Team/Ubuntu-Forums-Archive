### Post by RAOF on 2005-11-21
Not really, no.  That's what alien is **for**.  You should be able to install anything distributed as source, too (this is typically what you'd find in a .tar.gz).

However, you should rarely, if ever, have to deal with .debs, .rpms, or anything else.  Almost everything you'll ever want to install you can get through apt-get, or one of the frontends: synaptic or "add/remove programs" (I think it's called).  One of these is **really** the way to go: it downloads, installs, configures (often with special Ubuntu sauce) automatically.  Plus, it keeps a record of all that it does, so you get automatic updates to whatever is installed, and you can cleanly and easily remove anything you've installed through it.
