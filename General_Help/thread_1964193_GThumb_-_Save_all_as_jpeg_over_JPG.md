### Post by ajgreeny on 2012-04-23
If all your pictures are in the same folder you can select all your pics and use the tools > convert format in the menu of gthumb, and set it to create a new picture that way.  I don't think it's possible to delete the old format photo in the same operation, but you could do that easier in the terminal.
