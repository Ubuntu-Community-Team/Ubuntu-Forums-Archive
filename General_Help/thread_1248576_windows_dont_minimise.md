### Post by gldearman on 2009-08-24
Do you have Visual Effects turned on?  This is a longshot, but it's possible that Compiz believes, for whatever reason, that those windows are never to be minimized.

To troubleshoot, try turning off Visual Effects with System > Preferences > Appearance.  Do the windows now behave normally?

If they don't, or you didn't have Visual Effects enabled to begin with, then you'll have to keep looking (probably at the settings for whatever window manager you are running).

If that does solve the problem, turn Visual Effects back on, then start looking through your Compiz settings for something messed up.  I'd probably start with Window Rules, if it's enabled.

Hope this helps.
