### Post by mc4man on 2015-05-10
You have a mobile (optimus) chipset. So by default nvidia drivers are used via nvidia-prime which produces & use the xorg.conf you 'don't want'
There is no way around that.
If your currently installed drivers don't support your 840m thru nvidia-prime then  - 
look for & use drivers that do, if nothing avail. then file a bug
remove nvidia-prime & use bumblebee instead (- if bumblebee still works which is on demand, not all the time
use Intel gpu instead
