I would use dual boot because it's the only way you don't use Windows at all. As you know VM-ware runs on Windows so you still need that. Also when something goes wrong in Windows (and this seems to happen) you will also have lost your Ubuntu setup.
Using VM-ware is nice to try another OS but to have a serious installation you better use dual boot.
