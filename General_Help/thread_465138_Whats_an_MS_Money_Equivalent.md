### Post by timzak on 2007-06-05
Hi, I'm in the same boat, but use Money 06.  The program I've found that seems to be the best Linux equivalent is Moneydance.  It's not free, but it's reasonably priced and has a free trial.  If you're a newbie, installation is not automatic, so expect to ask for help to install.  I myself haven't made the switch yet...the import feature doesn't work well so I'd have to input everything from scratch.  The problem with Money is it's so proprietary that the export feature doesn't work well except when importing back into a Money product.  The website is [www.moneydance.com](www.moneydance.com).

There are some free financial packages available in Add/Remove and Synaptic, but none of them looked appealing to me.  For now I am dual-booting to Windows primarily to use MS Money.

Hope this helps.
