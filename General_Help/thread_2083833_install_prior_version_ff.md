### Post by dniMretsaM on 2012-11-13
Before you continue, I think it would be best to consider the security implications of doing that. Firefox 16 fixed fourteen security vulnerabilities from 15. 16.0.1 and 16.0.2 fixed two and one security issues respectively. All but three of those 17 issues were considered "critical." Call me paranoid, but going back to 15 probably isn't the wisest thing to do.

EDIT: I'm getting those numbers from [here](https://www.mozilla.org/security/known-vulnerabilities/firefox.html).
