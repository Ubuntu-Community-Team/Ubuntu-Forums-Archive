### Post by Dutch70 on 2011-04-08
Not sure what Wimba is, but to get java and youtube working, just go to software center, search for "restricted" and install ubuntu-restricted-extras.

Alternatively, you can use the terminal.
```
sudo apt-get install ubuntu-restricted-extras
```
