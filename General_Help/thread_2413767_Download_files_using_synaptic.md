### Post by Dennis N on 2019-03-01
> **raleigh3 said:**
> I use synaptic for most of my installations. I already have a program installed but would like to know how can I download the program to keep in my archives.

With Synaptic:
1) for an an application that's already installed, locate the application in Synaptic and mark for reinstallation.
2) Click Apply
3) On the Summary screen, check the box 'Download Package Files Only'
4) Deb file(s) are downloaded to your computer.
