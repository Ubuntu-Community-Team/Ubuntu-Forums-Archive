### Post by yoshii on 2016-06-01
Try restarting FireFox in Safe Mode (from the Help menu) and see if it then crashes on the site.  
If it doesn't then the problem is probably one of your installed Extensions/AddOns.  

You may also want to change how FireFox loads Flash from the PlugIns section of AddOns.  
You can make it so that it does NOT load Flash automatically, since Flash uses a lot of RAM.  

It's possible that you have some corrupt RAM and that it is only triggered when something large like FireFox loads and loads up a large page.
