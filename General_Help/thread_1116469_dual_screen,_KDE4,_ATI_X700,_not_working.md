### Post by stonebit on 2009-04-17
I also had this issue with an ATI card. I had a lot of problems with the ATI Linux drivers, so I just sold the card and went for NVIDIA. It's more stable and the NVIDIA monitor management is much much better than ATI's - it spits out the xorg.conf file and allows you to copy/paste the new configuration to your xorg.conf and tweak it if needed.

Anyway, I found that I had to set The X monitor manager to not clone and extend desktop AND setup the monitors in the ATI manager. It was a lot of hassle. There is a clone checkbox in the X monitor manager that kept rechecking after reboot. I can't remember what I did to get the setting to stick, but it is in the forums. I found several references when I searched for it. 

I hope that helps.
