### Post by Zorael on 2008-04-29
Well, if you install Firefox, the **ubuntu-restricted-extras** package (or kubuntu/xubuntu) as well as **sun-java6-jre**, you should get both Flash and Sun's Java.

I believe the Firefox packages (both the 3.0 beta and the stable 2.0.x) now also includes the almost-complete open Java, so you may not even need Sun's.

If it doesn't work (and you did a fresh install), sounds like you encountered a bug.
