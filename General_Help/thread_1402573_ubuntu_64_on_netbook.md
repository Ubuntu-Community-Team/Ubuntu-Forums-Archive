### Post by 3rdalbum on 2010-02-11
> **gabrielcik said:**
> I don't like unr and as far as i understood the difference between this and desktop is only in the interface.

The interface, and in the future UNR will have a slightly different default software selection.

All Intel Atom processors that are used in netbooks are 32-bit. Some that are used in nettops are 64-bit. The Atom's chipset supports a maximum of 2GiB of RAM anyway, and you wouldn't want to run intensive calculations on an Atom anyway due to the low performance, so even if you have a nettop I'd still recommend 32-bit.
