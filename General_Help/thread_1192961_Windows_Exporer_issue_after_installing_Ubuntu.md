### Post by AoSteve on 2009-06-22
Well, have you tried running   c:\windows\explorer.exe from the task manager?

That's a problem with Windoze, not ubuntu.  I've ran 6 systems now dual boot with Ubuntu, even one with two different versions of Linux and an XP.   Did you resize your main partition to add the partition for ubuntu or?

I know one good thing though, since you've got Ubuntu.  You can find the "explorer.ex_" file in one of the cab's and just rename it and copy it in place of the bad explorer.exe.  Just remember to rename the old explorer.exe to explorer.old so you have a copy of it for comparison.   

I've seen where resizing can cause file fragments to be lost.  Hence, it's almost a MUST to defrag when resizing a partition.
