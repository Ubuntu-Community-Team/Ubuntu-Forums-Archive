### Post by Ramses de Norre on 2006-10-24
You shouldn't be using the vesa driver, vesa is a driver that works on almost every video card but doesn't use any advanced features of the card.
I think you'll want the "sis" driver but I'm not sure, look around for which driver to use for your video card.

Then change this section to the appropriate driver:```
Section "Device"
Identifier "Generic Video Card"
Driver **"vesa"**
BusID "PCI:1:0:0"
EndSection
```

### Post by crumja on 2006-10-24
Vesa does things through Mesa emulation, so all OpenGL is cpu rendered. For any serious gaming or graphics work, you want a proper driver. Which driver to use will depend on which graphics card you have.
