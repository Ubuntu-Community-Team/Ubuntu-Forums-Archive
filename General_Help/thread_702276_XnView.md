### Post by julian67 on 2008-02-20
It seems like it's not Ubuntu but the XnView download page which is not for the novice ;-)

The XnView linux packages are much older versions than the windows binaries. As you have wine installed I'd suggest you download the windows  version: XnView v1.92.1 Standard version (JPEG2000) setup and install it using wine, it works pretty well.

There are similar tools available natively in GNU/Linux. Phatch is an excellent batch tool, gThumb and F-Spot are very good organisers/viewers/simple editors.  I previously used XnView windows package in wine but have been able to completely switch to free software for photo editing/managing since Phatch gained full exif data handling.
