### Post by liekomg on 2010-01-25
So, I was able to install the stable release of Firefox 3.6 on Ubuntu 9.10, but when I updated it reverted Firefox back to the Namoroka build version. Does anyone else have this problem? How do I prevent it from changing from Stable release 3.6 to Namoroka? I even try uninstalling and reinstalling Firefox but it kept installing using Namoroka or version 3.6b, I want to keep the stable version.

---

### Post by scouser73 on 2010-01-25
Go to **Synaptic Package Manager** and search for **firefox-3.5** & **firefox-3.5-branding**, highlight them, go to **Package** and select **Lock Version**.  Now reinstall Firefox 3.6 and I guess you shouldn't have anymore reverting to previous releases.
