### Post by abisdad on 2012-02-05
**_Solution!_**

Use the Tools menu, then Gallery, select the Theme and there are the images!

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Thanks both of you for replying.

I'd found the hidden file in the home folder, and there were two folders which looked promising...

They were "extensions" and "gallery", but I couldn't get any images from them!

The site I downloaded them from ( [http://extensions.libreoffice.org/extension-center/gallery-of-danger-signs](http://extensions.libreoffice.org/extension-center/gallery-of-danger-signs) ) was talking about "Cliparts" and I couldn't find anything like that in the menu.  I was also trying to insert a picture and insert a file - neither worked.

However I then found the Gallery, under Tools!

Select the Theme, and there they are!
