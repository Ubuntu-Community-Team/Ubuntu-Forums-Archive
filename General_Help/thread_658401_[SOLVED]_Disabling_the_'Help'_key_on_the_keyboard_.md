### Post by oneiota on 2008-01-04
I use a Sun keyboard, which has a 'Help' key in the upper left.  This key activates the Ubuntu Help Centre application, as one would expect. 

I would like to disable this key (without changing anything else on the keyboard) so that when the cat wishes to rest his head on the edge of the board, I don't get 400 Help windows popping up, locking Ubuntu beyond use before I've noticed the key has been depressed, erm, again.

Note this is not a keyboard shortcut, it's an actual key.

Any suggestions to change the cat's behaviour will not be considered. ;)

### Post by rubicon on 2008-01-04
Try this: System > Preferences > Keyboard Shortcuts, there you can find what you're looking for.

Or, if that won't do the work, you have another two variants: either make some program use this key for 'nothing'. For example, I installed tilda -- terminal emulator -- and set its shortcut key on F1. This not only disables help in programs, but calls tilda's window -- the best help for me :)

The second variant is far more easier -- take your screwdriver and take that key off your keyboard :)

Thank you Rubicon - that's done it.  

I had looked at the keyboard shortcuts before posting but hadn't spotted it because I was looking for something beginning with 'Ubuntu' or 'Help', etc.  I've looked again since it is the first option you had described and there it is!  

Keyboard Shortcuts application under Desktop>Launch Help Browser.

Thanks very much!
