### Post by mcduck on 2010-07-15
Jut use a theme with wider window borders. The only thing affecting the window border width is your Metacity theme.

Alt+middle mouse button allows you to resize a window from any point inside the window (instead of just the borders). Alt+left mouse button does the same thing fro moving windows.
