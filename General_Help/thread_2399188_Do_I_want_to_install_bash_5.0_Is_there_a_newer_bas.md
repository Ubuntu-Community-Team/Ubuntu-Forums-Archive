### Post by Holger_Gehrke on 2018-08-22
If you look at the changelog at [http://git.savannah.gnu.org/cgit/bash.git/commit/?h=bash-5.0-testing](http://git.savannah.gnu.org/cgit/bash.git/commit/?h=bash-5.0-testing) (it's below the list of source files), is any one of those fixed bugs bothering you ? 

The new version is labelled as 'alpha', for the previous release (4.4) there was close to a year between the alpha and the final release.

Personally I leave the testing of alpha versions of important pieces of software to braver and more knowledgeable people. If you consider yourself one such person, please go ahead ...
