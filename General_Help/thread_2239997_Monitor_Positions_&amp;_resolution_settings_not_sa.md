### Post by TheFu on 2014-08-17
TL;DR - I haven't screwed with nvidia stuff in a year or so (12.04). May do so later today on 14.04, haven't decided.

a) Stay with drivers provided by Canonical's repos. DO NOT DOWNLOAD directly from nvidia. This isn't Windows and nvidia doesn't seem to test the latest releases on Linux enough. 
b) Use **sudo /usr/bin/nvidia-settings**
c) I've never needed to screw around with any other settings elsewhere, but the /etc/X11/xorg.conf is the only file I think you need to touch ... if the GUI tool doesn't do everything.

Of course, making a backup before starting would be smart too.

/etc/X11/xorg.conf  is the default file - nothing special is needed to have X/Windows read it. It is a config file, so sourcing it, like you do above is an error.
