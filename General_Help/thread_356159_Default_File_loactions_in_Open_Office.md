### Post by sdide on 2007-02-08
Hey,
as I see it. You want the default save folder changed.

Go to top bar: Tools -> Options
Spawning the Options window. 
On the left expand the "OpenOffice.org"-tab, choose "paths".
The "My documents"-path can by edited. This should be your default path.
