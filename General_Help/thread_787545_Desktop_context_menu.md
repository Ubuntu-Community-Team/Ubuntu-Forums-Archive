### Post by amohanty on 2008-05-09
Just to get things straight the context menu in *box WMs is actually provided by the WM. In the case of gnome it **is provided by nautilus** (the file explorer). 

You can extend it using nautilus-actions, details here:
[http://www.linux.com/feature/119603](http://www.linux.com/feature/119603)

HTH
AM
