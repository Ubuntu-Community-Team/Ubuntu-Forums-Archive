### Post by UJoe4 on 2007-07-17
Go to Menu > Settings > Keyboard Settings > Shortcuts. If the only theme you have is "Default", add a new one (let's say "Custom").

In the Custom keyboard theme, add a shortcut to "xfce4-taskmanager" and bind whatever key sequence you want to it. Control-Escape is probably already bound to "Popup menu", but you can change that if you want.

Now you can press your key sequence to bring up the GUI task manager, right-click on the application you don't want and choose "Kill".
