### Post by fng on 2005-01-04
The BIOS controls wether you can boot from your cd-rom.  Not Windows.

If the WinXP-machines of your friends don't boot from the cd-rom. Do the same as with your win98 machine :). Go into the BIOS and set the cd-rom as the first boot device.
