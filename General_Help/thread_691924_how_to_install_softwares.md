### Post by zvacet on 2008-02-09
```
tar zxvf jUploadr-1.1.2-linuxGTK-i386.tar.gz
```

That will create folder.Go to that folder with this command

```
cd /folder_of_your_choice
```

Inside you will find install and read me files.Read them and you will know what to do next.
