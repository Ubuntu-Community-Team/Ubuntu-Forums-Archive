### Post by doriard on 2008-04-25
Is there a way to make Dolphin file manager remember the settings?  Every time I browse to a different folder I have to reset "details" and "show hidden files".  How can I make those settings permanent and for all folders?

---

### Post by curtis1552 on 2008-04-30
Settings->Configure Dolphin-> General -> Default View -> Details

View -> Adjust View Properties -> Properties -> Show Hidden Files
