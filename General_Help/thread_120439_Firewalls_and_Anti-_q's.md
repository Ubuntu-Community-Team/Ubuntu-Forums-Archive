1) Most likely you won't need a firewall, unless you're running some type of server (like sharing files over network, that requires a server). But a firewall isn't a bad thing to have if you want it.

2) There are very very few viruses (or like stuff) for Linux, and only one virus has been seen "in the wild". And infecting your Linux box with a virus is not an easy thing to do. So, most likely you don't need it either, no point to run it, but you can if you want.
