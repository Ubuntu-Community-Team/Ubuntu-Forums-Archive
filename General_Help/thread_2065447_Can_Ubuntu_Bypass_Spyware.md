I agree that you should completely wipe Windows just to be safe. Just stick with Ubuntu at this point. 

If you used Wubi to install Ubuntu then you can not wipe Windows at this point since a Wubi install is inside your Windows system.

If you like Ubuntu and are getting used to it, and, most importantly, you feel like you can discard Windows and only use Ubuntu, then **back up your files** and wipe your entire hard drive during a real install of Ubuntu from LiveCD or USB media. 

You can use your current Ubuntu installation to download the CD ISO of Ubuntu 12.04 and burn it to CD or Create A Startup Disk on USB disk. Once booted from the CD or USB disk, **backup** your files from your hard drive. **Once you have backed up your files**, you can go ahead and install Ubuntu and wipe your entire hard drive in the process.

Just to add one more thing, once running Ubuntu (even inside a Wubi install at present) you should be safe from any Windows spyware, so you do not have to worry. However, the spyware is still there in Windows and each time you boot into Windows you are at risk. Once you have removed Windows and are running Ubuntu from a real installation you will definitely be safe from all Windows spyware, viruses (viri?), and the like.

Good luck.

### Post by ezsit on 2012-10-03
Running a Wubi install of Ubuntu is much like booting any Linux from an ISO file. The grub2 boot loader takes control of the booting and looks somewhere in the Windows filesystem for an image file of Ubuntu to boot. Once running Ubuntu, whether from a Wubi install or a normal install, you are in Linux and safe from Windows malware. 

A boot sector virus can still infect the computer if the computer is booted into Windows, however, we are not talking boot sector virus in this case. Spyware and key loggers installed in Windows would have no effect in Ubuntu, Wubi or otherwise.

This is not to say a very enterprising cracker could not write malicious software that is installed under Windows and searches the hard drive for a Wubi installation of Ubuntu and patches the Wubi Ubuntu Linux filesystem in some way, adding malware into the Ubuntu installation. I have yet to hear of anything of this sort, but it is theoretically possible.
