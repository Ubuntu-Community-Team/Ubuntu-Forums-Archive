### Post by Yagisan on 2005-11-04
IIRC you need 48MB for an ubuntu thin client, but they have no disk so they can swap. Ubuntu will install onto that experimental computer, but it will be rather slow with only 32MB of ram, it will constantly swapping to disk, so expect to see the red light flashing a lot (not much different to Windows in that regard ;) ). If possible install more RAM on the system, otherwise happy experimenting.
