### Post by warreno on 2009-08-29
Another possibility might be to use [VirtualBox]("http://www.virtualbox.org/"), which should let you run Ubuntu in a window on your Windows machine, sort of like having a mini-computer running as a program on your main OS. I've got an install of Jaunty on my Mac Mini through VirtualBox.

It's a little weird in the setup — the documentation is somewhat out of date — but it works, once you have it up and running, or at least it seems to.

I've also got Ubuntu 9.04 on my netbook, but that was a total replacement of XP. (The netbook runs far better with Ubuntu than it did with Windows, for what it's worth.)
