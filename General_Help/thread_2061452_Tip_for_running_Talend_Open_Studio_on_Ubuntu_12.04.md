### Post by tirade on 2012-09-22
Talend Open Studio is a sophisticated open-source data integration tool that I've come to know and love at work, but using it at home gave me trouble because Linux distributions since about 2010 no longer come with one of its requirements by default: an old version of XULRunner.

I found this helpful blog post:
[http://outsidethegap.blogspot.com/2012/05/mozillafivehome-not-set-talendubuntu.html](http://outsidethegap.blogspot.com/2012/05/mozillafivehome-not-set-talendubuntu.html)


Just download and install these packages and the application will run again.
[LIST]
[*][http://nl.archive.ubuntu.com/ubuntu/pool/main/h/hunspell/libhunspell-1.2-0_1.2.14-4_amd64.deb](http://nl.archive.ubuntu.com/ubuntu/pool/main/h/hunspell/libhunspell-1.2-0_1.2.14-4_amd64.deb)
[*][http://security.ubuntu.com/ubuntu/pool/universe/x/xulrunner-1.9.2/xulrunner-1.9.2_1.9.2.28+build1+nobinonly-0ubuntu0.11.04.1_amd64.deb](http://security.ubuntu.com/ubuntu/pool/universe/x/xulrunner-1.9.2/xulrunner-1.9.2_1.9.2.28+build1+nobinonly-0ubuntu0.11.04.1_amd64.deb)
[/LIST]
