### Post by hwttdz on 2009-11-16
If you have filters set up in your gmail account they will still happen to all incoming messages.  I primarily use thunderbird, but I prefer to use gmail filters to thunderbird filters, so that I don't have to update my filters on every machine I use to read mail.  

So in sum, gmail filters still happen, it is not necessary to import them into thunderbird.
