### Post by Cheesemill on 2013-03-28
Only LTS releases get updated installation media after they have been released. For non-LTS releases (such as 12.10) the only available images are the ones that were available on release day.

You could, however, use the mini CD to perform your installation as this will download and install the latest version of packages, meaning you start with a fully up-to-date system.

You can download the mini CD from here...
[http://cdimage.ubuntu.com/netboot/quantal/](http://cdimage.ubuntu.com/netboot/quantal/)
