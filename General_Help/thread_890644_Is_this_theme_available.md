### Post by steveneddy on 2008-08-16
I would think that if you cruised through gnome-look.org and played around with Compiz Fusion settings and get the system colors just right that you could duplicate this with little difficulty.

To change the system colors, just go to the Desktop

right click

select Change Desktop Background

At the top, select the Theme tab

on the botton, button named Customize, click it

select the Colors Tab and go from there.


To get menus transparent, open up CCSM

General Options

Last tab, called Opacity Settings

Here are my setting for the Opacity Settings box

```

dock         67
Menu         90
DropdownMenu 92
popupmenu    93

```

You can use what ever you like, but these are just enough for me.

There are other settings that you can find either on UF of through Google.

Searching for **[COLOR="RoyalBlue"]Compiz Fusion Opacity[/COLOR]** or **[COLOR="RoyalBlue"]Transparency[/COLOR]** give good search results.
