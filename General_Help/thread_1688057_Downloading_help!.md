### Post by Tamlynmac on 2011-02-14
I believe the toy your describing is a [LeapFrog]("http://www.latestbuy.com.au/my-pal.html")  product that uses priority Windows based software. You may be able to  use it under wine (a program used to run some Windows apps in the  Ubuntu/Linux environment) but will require setup. I've searched and  can't locate any information regarding running this product in Linux.

If you wish to know about wine you can read [this]("http://www.makeuseof.com/tag/how-about-some-wine-with-your-linux/").

I would suggest that you download and read the [Ubuntu Pocket Guide]("http://www.ubuntupocketguide.com/download_main.html").  It's free and does a pretty good job of basic Ubuntu/Linux. It's easy  to use and a good reference for new users. One of the keys to remember  if it's a Windows (.exe) file, it won't run in Linux except potentially  in wine. Most Windows software is priority and dedicated to that  environment.


I find this [site]("http://triponic.com/guidestutorials/ultimate-beginners-guide-for-torrents/")  a nice simple description of torrents and how they work.

Ubuntu/Linux  does require a learning curve (just like Windows does). Your not alone  in the learning curve with regards to time, but I think you will truly  appreciate the OS once you've learned about it and how to use it.

):P  Welcome to the Forums and Good Luck
