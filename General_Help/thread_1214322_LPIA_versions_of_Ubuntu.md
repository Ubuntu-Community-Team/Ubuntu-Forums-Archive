### Post by hedgeborn on 2009-07-15
My Dell Latitude 2100 shipped with the LPIA version of Ubuntu 8.10. It seems that this version is specifically for Atom processors? and uses less energy.

Yet when you look to upgrade the OS, the LPIA versions are nowhere to be found. I don't see LPIA version of 9.04 or NBR anywhere. Any disadvantages or problems to be expected by updating to a non-LPIA version of the OS?

---

### Post by Brandon Williams on 2009-07-15
Most people who move away from the Dell-specific release version end up going with the i386 version of Ubuntu. There is little evidence that the lpia version gives better battery life or performance.

That said, you can install the lpia version from the alternate install CD found [here](http://cdimage.ubuntu.com/ports/releases/jaunty/release/).

---

### Post by snowpine on 2009-07-16
The Atom processor is well-supported by recent versions of the Linux kernel, which is why Debian, Fedora, Arch, etc. do not have a separate release for the Atom. Ubuntu is the only distro with a separate LPIA release, and I have a hunch it's related to the deal they made with Dell.
