### Post by SeijiSensei on 2013-10-17
It depends on what character set you are using.  My copy of TBird uses ISO-8859-1, the standard "Western European" character set.  You may be better off using UTF-8, or "Unicode."  In Thunderbird, navigate to Edt > Preferences > Display, and click the "Advanced" button in the Fonts section.  You can select a character set for both inbound and outbound messages.
