### Post by aysiu on 2007-01-21
There might be a GUI program--even a Firefox extension, but *wget* may be your friend in this matter: ```
wget -c - m http://www.nameofsite.com
```

**Edit**: Wasn't able to find a Firefox extension, but there appear to be a lot of good GUI download programs in System > Administration > Synaptic Package Manager (see attached screenshot). Just do a search by **name and description** for the word *download*.
