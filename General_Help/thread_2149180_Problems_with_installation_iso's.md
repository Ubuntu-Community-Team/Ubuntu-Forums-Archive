### Post by Megaptera on 2013-05-28
My Dell is 64 bit compatible but with only 2gb of RAM I use the i386 version with no problems. I suggest you try both as live CDs (not install) and see how they perform - note that running from live CDs will be slower than a full install.
When you do go for reinstalling, it's easier to install Windows then Ubuntu as the Windows mbr won't easily 'find' Ubuntu to give you the dual boot option, whereas Ubuntu 'grub' should see Windows easily.
Also, please remember to backup all your important data to external storage before you start 'cos reinstalling will wipe the hard drive!
