### Post by ankspo71 on 2012-02-19
Hi,
You are probably much better off creating your own topic since things have changed in Xubuntu since 2008 (when the user Torb. created this topic).

Anyhow, the only advice I can give is that 'tumbler' replaced 'thunar-thumbnailers' in more recent versions versions of Xubuntu/XFCE, so if you are using a more recent version of Xubuntu/XFCE then try installing the packages called:
"tumbler"
"tumbler-common"
"tumbler-plugins-extra"

Hope this helps.
