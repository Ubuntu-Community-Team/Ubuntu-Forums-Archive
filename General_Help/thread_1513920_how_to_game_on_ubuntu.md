### Post by NCLI on 2010-06-20
First of all, you cannot expect all Windows applications to work in Ubuntu, even with Wine. Always check the [AppDB]("http://appdb.winehq.org/") for compatibility problems and workarounds.

It looks like this particular game should work just fine, try running System->Administration->Update Manager to check for updates, then System->Administration->Hardware Drivers to check whether you have the proper drivers installed.

All drivers are handled and automatically updated by Ubuntu.

Lastly, [Linux is not Windows]("http://linux.oneandoneis2.org/LNW.htm").
