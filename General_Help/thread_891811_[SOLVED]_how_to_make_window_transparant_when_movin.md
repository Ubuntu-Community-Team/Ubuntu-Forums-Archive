### Post by howefield on 2008-08-16
Load up compiz settings manager, scroll down to the "Uncategorized" and check the box beside the move window option, click on the icon to open the settings, then move the opacity slider to the desired level of tranparency.

### Post by DaymItzJack on 2008-08-16
Go into Compiz Settings Manager (System->Preferences->), type 'move' into the search bar at the top left and select 'Move Window'. In that plugin change the opacity to 85 or whatever you wish to choose.

If you don't have compiz settings manager type in terminal:
```
sudo apt-get install compizconfig-settings-manager
```
