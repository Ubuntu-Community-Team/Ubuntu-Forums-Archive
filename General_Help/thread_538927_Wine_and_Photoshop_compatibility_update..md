### Post by FeraTech on 2007-08-30
Photoshop 7.0
Ubuntu Fiesty
Wine 0.9.44

With the new version of Wine 0.9.44 most compatibility problems have been resolved. Previously the window manager would cause all the tool windows to remain in front of all your views. This has now been fixed. Overall compatibility is near perfect.

For those that have an issue with a pause when splash screen says "loading suite" simply press enter. There is a message behind the splash that causes it to pause.

All functionality of Photoshop works. I would recommend trying it out.

3rd Party Repositories to upgrade your Wine version:

[http://wine.budgetdedicated.com/apt](http://wine.budgetdedicated.com/apt)

Simply open "Synaptic Package Manager" go to "Settings" "Repositories" and click the 3rd party tab and select Add.
