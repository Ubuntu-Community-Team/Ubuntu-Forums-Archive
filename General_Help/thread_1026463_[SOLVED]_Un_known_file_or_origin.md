### Post by drs305 on 2008-12-31
Dwhelper is a file used by the firefox extension 'Download Helper'. You can open Firefox and go to Tools > Addons. Select "DownloadHelper" and remove it with the uninstall button. You can then delete the file in your home folder and it won't reappear.
