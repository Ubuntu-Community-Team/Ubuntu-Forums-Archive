### Post by mcduck on 2011-02-07
> **Menace97 said:**
> Ok, what is wine, it is some virtualisation software right? So i can run itunes and microsoft word 2007 on linux? how does the software work and is it **_SAFE?_**

I love linux but to use it all the time i need microsoft word 2007 and some other windows only applications. (No, not itunes but others).

It's not a virtualization software, and it's not an emulator either.

Think of it as a translator, working between a windows program and a Linux system. So when the program asks for a Windows service to read a file, for example, Wine translates that to a Linux system command to read a file.

### Post by mcduck on 2011-02-08
> **Menace97 said:**
> so its pretty safe?

Yes, it's safe. And uses less resources than virtualization or emulation.

But if you happen to speak more than language, you know that translating between languages isn't always perfect. Some things can be very hard to translate, and some things are just impossible to translate from language to other. Even if you have perfect skills in both languages. And just like that, Wine runs some programs very well (sometimes even better than they'd run on Windows!), yet fails miserably with others. So it's always a good idea to check the application database at the winehq website to see if certain program works with it....

edit: OpenOffice has cover pages and stuff. :D
