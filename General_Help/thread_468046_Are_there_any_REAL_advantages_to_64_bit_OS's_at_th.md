### Post by Crakie on 2007-06-08
No real advantages at the moment, except when using 4 Gb or more of RAM (which a 32 bit OS cannot address). I sometimes hear certain high demanding video-apps may benefit from it, which makes sense given the nature of memory usage for that kind of applications. No doubt more and more 64-bits apps will start to appear and gradually replace the 32 ones.

I would not go through the hassle of upgrading to 64 bit just yet, if I were you.

### Post by rgiskard01 on 2007-06-08
In initial testing with [Rosetta ]("http://boinc.bakerlab.org/rosetta/") I found the 64-bit Ubuntu 7.04 to compute work units 2-4 hours quicker.

I'm still testing, but initially I have found....

32-bit Ubuntu 6.10 = 7-8 hrs/work unit
64-bit Ubuntu 7.04 = 4-5 hrs/work unit

Now the caveat: the 32-bit is running on an AMD64X2 3800+ 
AND, the 64-bit is running on an Intel Pentium D 925 (64, 3.0Ghz)

I could be wrong, but I think these are somewhat comparable CPUs...anyone?

I still need to do more testing...but if you do any Boinc work, it appears worth considering.
