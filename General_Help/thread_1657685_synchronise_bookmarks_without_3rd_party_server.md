### Post by TVTrukChik on 2011-01-01
Xmarks, which is available from the Mozilla add-ons page, will let you sync bookmarks using your own server.  (You set this up using the Advanced page on the settings.)  I store my bookmarks on my website so I can sync between home and work, but I don't know why it wouldn't work within your home network.  It will even sync cross-platform, so I can sync my Linux machines with my Windows one at work.
