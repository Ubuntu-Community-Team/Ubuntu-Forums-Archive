### Post by howefield on 2011-01-22
Have you installed Java ?

Search Synaptic Package Manager for sun-java6-jre sun-java6-fonts and sun-java6-plugins and install.

If you are using 10.04 or 10.10 you will need to have the Partner repository enabled.

Edit:
Should also add that there is an opensource java available if you prefer that to the proprietary version ...

[https://help.ubuntu.com/community/JavaInstallation](https://help.ubuntu.com/community/JavaInstallation)
