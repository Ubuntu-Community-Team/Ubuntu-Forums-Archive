### Post by Ralph Hinkley on 2014-05-17
Update: I have a new understanding of the way Ubuntu sorts files. By changing to the list view and unchecking everything except "type" I found that the the files are in fact being sorted by type... sort of. As many people on this forum are probably already aware, but somehow I had missed, Ubuntu puts files in general categories like "document" or "image" - not specific by file extension. Thus PNG, JPG, PSD, etc... are all grouped together, but intermingled.

It would be useful if it was possible to sort specifically by file extension, but I'm becoming less confident that there is a simple way to create that view within the file manager window.

### Post by Ralph Hinkley on 2014-05-17
Another Update: My apologies for making a lame thread that no one else needs, but on the off chance that someone else out there is only just discovering these things too, I'll post what I've learned. 

MIME type sorts the way I need. It's not an option in the main menu - only by right clicking the column headings in list view and selecting MIME type. Learn something new everyday I guess! (I feel dumb!)
