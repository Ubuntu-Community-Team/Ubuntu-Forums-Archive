Hello Ubuntu Friends,

I've been happily using Ubuntu for a few weeks. As a Firefox / Thunderbird / jEdit / Rhythmox / xmms user I've had very little trouble doing most of what I need.

There is one big hole in my linux toolbox, though -- a photo / picture organizer. I'm talking about something like Adobe Photoshop Album. Key features that I've yet to find under linux are:

* assign categories and sub-categories to pictures, search on these
