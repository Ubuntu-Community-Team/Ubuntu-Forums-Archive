### Post by Mighty_Joe on 2009-09-10
I believe the Sanza player gets the information on a particular file from the file metadata ("MP3 tag") rather than from the file system (directory and file names).  [Here's a discussion]("http://ubuntuforums.org/showthread.php?t=293087") on the best tag editor.  My personal favorite is [Tag&Rename]("http://www.softpointer.com/tr.htm"), but it's a Windows only app.
