### Post by fragos on 2008-10-27
In general Gmail is a slow web site to bring up. There's a lot of background processing that has to be done go Gmail comes up with your account and stored email. I can't speak to Vista because I haven't use anything Microsoft in many years. There is another browser called Epiphany that uses the same Gecko site rendering and is much faster than Firefox. It has less features, fewer extensions, than Firefox. You could try it. Also Gmail, a web 2.0 application, can be started in basic HTML mode which is also faster but lacks some of the gliz of full Gmail.
