### Post by slickymaster on 2015-07-09
Open a terminal window and run:```
sudo apt-get update
sudo apt-get install gnome-session-flashback
```Logout from your account and login again and choose it from the Sessions option on the login screen.

### Post by buzzingrobot on 2015-07-09
Support for the "old Gnome" (Gnome 2) ended in 2011. It's abandoned.

MATE is a slight fork of Gnome 2 that works in the current software environment. That is, it began with the Gnome 2 code base and has been modified where needed. The most visible differences with Gnome 2 are usually down to theme availability.

MATE is available in the official Ubuntu MATE release as well as via the repos.

The gnome-session-flashback is an approximation of the Gnome 2 menu-and-panel approach.
