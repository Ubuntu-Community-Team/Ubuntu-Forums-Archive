### Post by fixerdave on 2011-09-27
Oh... one more thing.  Before you do anything else, you best boot into Ubuntu and make a backup of all those "Windows" files you are desperate to get back.  The more you muck with installations, the more likely you will lose them.

With Ubuntu, you have the option of using something like DDrescue to copy everything (byte for byte) to a file on your USB drive.  If these files are really important to you, it might be a good idea to do that before installing anything else on the drive.  It's not particularly easy to do, but if losing a bunch of data will ruin your day, learning how to prevent this might be worth the effort.
