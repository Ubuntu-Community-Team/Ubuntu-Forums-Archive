### Post by EuxineTL on 2007-02-05
If you installed the driver correctly, you may use ATI tools to configure.
Backup the old xorg.conf first.

#aticonfig --initial=dual-head

Set display mode of 2nd monitor:

#aticonfig --dtop={single|mirror|clone|horizontal}

More usage can be found while you finish the driver installation.

Also refer to:
[http://gentoo-wiki.com/HOWTO_Dual_Monitors]("http://gentoo-wiki.com/HOWTO_Dual_Monitors")
