### Post by 2F4U on 2012-03-26
Since there is no particular pattern behind the freezes, my first guess would be an issue with RAM. So, what I would suggest is that you run memtest, for example from a Ubuntu liveCD.
In my experience, it makes sense to let it run for at least two iterations, since I already had RAM problems not showing up during the first cycle.
