### Post by Kateikyoushi on 2007-02-14
You can switch themes in System > preferences > themes.
To find new ones go to gnome-look.org or search with google, installation is simple just click the install theme button and select the theme you downloaded.
I know not as easy as emerald theme manager but do not know any other way.
