### Post by Drew Woodard on 2009-06-03
I think the gnome-games package is installed by default which includes the games listed [here]("http://live.gnome.org/GnomeGames/").

there are other optional games that can be installed from the repositories, some of them are listed [here]("https://help.ubuntu.com/community/Games").

you can browse Ubuntu packages [here]("http://packages.ubuntu.com/"), click on "Jaunty" to see the current version and there is a "games" subsection.
