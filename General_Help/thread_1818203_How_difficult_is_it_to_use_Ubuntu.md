### Post by lordadi on 2011-08-04
Hi MattMan,

Will you be able to use Ubuntu - in all likelihood, yes. It depends on your machine and the install is pretty foolproof anyways. Besides, you've got a community like this one to help you through it ;)

### Post by zealibib slaughter on 2011-08-04
Ubuntu is very easy to use for everyday tasks.  It is a little confusing at first because it isnt what you are use to but once you get aquainted with Ubuntu its easier to work with than windows.  Flash is easy to install with flash aid: [https://addons.mozilla.org/en-US/firefox/addon/flash-aid/](https://addons.mozilla.org/en-US/firefox/addon/flash-aid/)
As far as safe I don't use any antivirus and have never had a problem.  In general as long as you have the built in firewall enabled you will be safe.
Here are some links for working with ufw:
[https://help.ubuntu.com/community/UFW](https://help.ubuntu.com/community/UFW)
[https://help.ubuntu.com/10.04/serverguide/C/firewall.html](https://help.ubuntu.com/10.04/serverguide/C/firewall.html)
[https://wiki.ubuntu.com/UncomplicatedFirewall](https://wiki.ubuntu.com/UncomplicatedFirewall)
Also installing programs is easy in ubuntu.  You can install via the software center which is similar to apples app store, the synaptic package manager, or the command line.
 
Lastly if you ever have a problem you can more than likely get answers with a quick google search or you can ask here (but google first since the answer may be out there and its a whole lot quicker).  Hope this helps.

### Post by Herman on 2011-08-04
For people who are not very experienced in the general use of computers it's easier to just start off with Ubuntu.  That way there's no need to learn about antivirus scanners, firewalls, keeping up to date on a million new security threats every day, defragging, learning which programs to trust and which not to trust and so on.
They only need to install Ubuntu and then learn how use the applications in Ubuntu to do whatever it is they want to do.
If there's something more they want to do at any time they can just open the Ubuntu Software Center and look for a free program from an approved software repository and install it.

For people with a moderate amount of Windows experience it's actually harder. 
First, they have an ego. The biggest problem for Windows users is getting over the fact that they don't know everything and need to start at the bottom rung of the ladder and work their way back up to a comfortable knowledge level.
There's no extra knowledge needed at all really, it's more like learning a new vocabulary. Most of the applications work pretty much the same and are quite intuitive. A spreadsheet is still a spreadsheet and an image file is still an image file. The new Ubuntu user just needs to know they're looking for 'GIMP' instead of paint for working on image files and 'calc' instead of whatever they call it for a spreadsheet program in the other operating system. Then they need to get used to looking for what to them seem like different looking icons than what they're accustomed to and a different menu layout.

People who are really skilled with computers should have no problems. There are people who can pick up any computer or computer-like device and just pick it up just like that and be able to use it seemingly right away.
