### Post by mcduck on 2007-08-29
You are safe.

Windows viruses don't work on Linux, so they won't be able to spread through your Linux box automatically.

You might download an infected file with Linux and then move it into Windows machine, but the antivirus software on the Windows machine should handle it just like it handles other infected files.

If you feel like it, you may install some antivirus software on your Linux machine and use that to scan files before you move them to a Windows machine. I wouldn't bother as, like I said, the antivirus software on Windows machine should be enough.

### Post by ddrichardson on 2007-08-29
As a caveat here, if it concerns you try AVG free for Linux - although Windows viruses don't automatically infect Linux files there is the possibility of them still being present, especially things like Office macro viruses.

It wont hurt to be able to scan files in your home directory to prevent upstream users being infected.
