### Post by zombifier25 on 2012-03-30
And why would you want to remove all web browsers?
There must be AT LEAST one web browser on your system, since some packages depends on them. Maybe some really low-level hack can do the trick, but I don't recommend you do so, since you may mess up the system really bad on the way.

### Post by Irihapeti on 2012-03-30
It's certainly possible to have Ubuntu with no browsers. I've set up several Openbox installs where I've added a browser quite late in the piece, and it hasn't been from the repositories (Seamonkey direct from the Mozilla site, if that's relevant).

However, I started with a command-line install from the alternate CD and added the bits that I wanted. I suspect that it might be easier to do it that way than by trying to remove things from a full Ubuntu desktop.
