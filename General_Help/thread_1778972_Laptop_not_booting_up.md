### Post by Mark Phelps on 2011-06-13
Sound like either (1) you installed Ubuntu using the wrong keyboard setup, so it is misreading the key codes, or (2) the laptop keyboard is bad or failing.

If you've had the problem since the very beginning, it's likely (1).  If the problem is more recent (as, the laptop keyboard worked at first), it's likely (2).

If you have a USB port and keyboard, you should try plugging that in and using it.

I don't know how to change the keyboard type without a working keyboard, so you might have to reinstall to get it working.

If you can't plugin a keyboard to copy off your files, you will have to remove your hard drive and connect it to another PC.
