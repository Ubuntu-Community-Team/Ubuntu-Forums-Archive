### Post by ugm6hr on 2008-11-29
Yes it is ready.

If you are running Hardy 8.04, you need to set it to look for non-LTS upgrades.

System -> Admin -> Software Sources
Updates tab
Show new distribution releases: Normal releases

I generally recommend a fresh install rather than upgrade though (ideally with separate /home partition).
