### Post by rajames429 on 2008-01-16
How do I free the command line after starting gedit? 

I thought that I had seen this before, but I have searched and can't find tha answer. 

I usually start gedit from the command line and provide a filename to open, but then I can't use the command line until I close the file that I opened in gedit. 

I thought there was a symbol to supply on the command line in order to get a new prompt immediately?

---

### Post by p_quarles on 2008-01-16
Add the "&" character to the end:
```
gedit filename &
```
