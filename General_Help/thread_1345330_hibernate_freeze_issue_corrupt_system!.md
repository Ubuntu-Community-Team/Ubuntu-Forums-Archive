### Post by robbielink on 2009-12-03
I've been running Ubuntu (been through several versions now) on my Presario M2000 with no hibernation problems but I guess it varies model to model. A quick fix would be to turn hibernation off until someone comes up with a fix. I just installed Ubuntu 9.10 on an old Dell desktop and everything was fine until it put the monitor to sleep and the same problem you had occurred - it corrupted the install somehow. Went to System/Preferences/PowerManagement as soon as I got it reinstalled and set it to not put the display to sleep and also turned off the screen saver in case that was an issue. Later when I have time I'll try to research why this is happening but for now it seems to have solved my problem.
