### Post by yancek on 2018-10-01
The basic question first, do either of them boot from the menu you have?  Members would need a lot more information on your system.  Are you using an EFI system which is the default for a pre-insalled windows 10?  You might go to the boot repair site below and download it and run it per instructions on the page.  Do NOT try to make any repairs but select the Create BootInfo Summary option which will give you a link which you can post here and members should be able to suggest a possible solution.

 [https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)
