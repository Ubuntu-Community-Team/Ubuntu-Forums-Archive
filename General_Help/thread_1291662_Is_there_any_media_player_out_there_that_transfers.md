### Post by sirtrent on 2009-10-15
If you are using a recent version of Ubuntu then Rythmbox should be installed by default. You will probably have to install libmtp before you will be able to use mtp. I currently use Rhytmbox for my iriver clix, and except for album art it works great.

I've never used Kubuntu, but if that's what you are using Rhythmbox may not be installed, you should still be able to get install it from synaptic, however if you are using Kubuntu I'd look into Amarok (though I've never used Amarok myself).
