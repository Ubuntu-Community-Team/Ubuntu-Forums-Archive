### Post by Roasted on 2009-10-05
Whatever changes you make to the LiveCD are wiped when you reboot.

LiveCDs are meant for tinkering with to see if your hardware works and to get a feel for Ubuntu prior to installing it. All you need to do to verify your wireless is select it in network manager (if you see it) and see if you connect. If you can, open firefox, and start hitting up web sites. If it's good, you're golden.
