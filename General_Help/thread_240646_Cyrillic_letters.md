### Post by btdown on 2006-08-21
Have you tried adding a Russian keyboard then switching to it when you want to type text?
 
System -> Preferences -> Keyboard -> Layout -> Add , then select the Russian Keyboard you'd like (I like phonetic).
 
Then right click in an open part of one of your menu bars and "add to panel" the "keyboard indicator". That way you can easily see which language you are typing in, and, will also be able to change it quickly (by clicking on it).
