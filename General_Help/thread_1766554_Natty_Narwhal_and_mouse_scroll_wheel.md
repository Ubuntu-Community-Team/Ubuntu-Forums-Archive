### Post by coffeecat on 2011-05-26
> **EgoGratis said:**
> I find it hard to believe nobody has mouse with scroll wheel. Hm?

Perhaps people are reluctant to post the rather unhelpful, "works for me". Um - it works for me, on more than one machine. All applications. I've not noticed any threads describing this so I guess yours in an uncommon problem.

For interest, try this. If you have one instance of an application open, middle-clicking the launcher icon opens a new window. If you have a clickable scroll-wheel, this doubles as a middle-button. Does this work for you?

One other thought. Do you have  a usable installation of an earlier Ubuntu version or another OS to test the mouse in? It's just possible that the scroll wheel has coincidentally failed when you upgraded.
