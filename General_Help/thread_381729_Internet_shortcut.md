### Post by oomingmak on 2008-01-04
> **kenmiles said:**
> I am trying to add an internet address as a shortcut to the desktop but I only seem to be able to add a file or program as a shortcut or menu item.
When you create your launcher, just put the name of your browser's executable in the command box followed by a space and then the URL that you want to go to.

Example:  

 [COLOR=DarkRed] firefox [www.bbc.co.uk]("http://www.bbc.co.uk")[/COLOR]

or 

[COLOR=DarkRed] epiphany [www.osnews.com]("http://www.osnews.com")[/COLOR]
