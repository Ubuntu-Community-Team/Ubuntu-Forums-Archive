### Post by jpeddicord on 2007-01-30
Open the System Monitor from System > Administration > System Monitor. From there, look under the CPU column. If you see anything that has a high CPU, then shut the program down. If it is a startup program that does it, you can disable it from System > Prefrences > Sessions.
