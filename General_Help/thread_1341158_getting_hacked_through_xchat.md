### Post by neur0 on 2009-11-29
You really don't need to use a firewall unless you opened some of your ports yourself by installing some servers (daemons).
It is never impossible to be hacked, but it is highly unlikely, and using IRC won't significantly increase the chances. Just make sure you don't install any "cool" IRC scripts/add-ons and avoid DCC for file transfers. Also don't give out info about yourself to people you don't trust.
BTW, you are not logging in as root, in fact you don't even have a password set for the root account right? RIGHT !?
