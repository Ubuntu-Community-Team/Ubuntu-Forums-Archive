### Post by kpkeerthi on 2008-02-22

Search for Firefox 3 and mark for uninstallation. Firefox2 should be restored automatically unless you removed it before installing FF3. 
If you still cant run FF2, Search for Firefox 2 in synaptic and mark for installation.
