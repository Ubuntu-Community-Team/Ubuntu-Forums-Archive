### Post by jordon on 2007-08-02
"Monospace" is a term that refers to fonts whose characters are all the same width. They're also called fixed-width fonts. Ubuntu comes with a number of them; I have Bitstream Vera Sans Mono, Courier, DejaVu Sans Mono, FreeMono, Nimbus Mono, and Tlwg Typist. But for most purposes*, I recommend [Inconsolata](http://www.levien.com/type/myfonts/inconsolata.html); it's not ugly like a lot of monospace fonts I could name.

*I wouldn't recommend Inconsolata for gedit if you have syntax highlighting on; making certain keywords bold throws off the fixed-widthness. However, using Inconsolata for Firefox's source code highlighting doesn't produce that problem.
