### Post by niteshifter on 2007-12-14
A couple of things come to mind:

Did you copy it to the correct place on the card? On my Palm TX (Palm OS Garnet v 5.4.9, Treo is same/similar IIRC) .prc files are in the folder *Card_Name*/PALM/Launcher (*Card_Name* being the root directory of the card). they don't run from the root of the card.

Some Palm programs simply will not run from the card, they must be in internal memory. Try placing it there if the above doesn't work.
