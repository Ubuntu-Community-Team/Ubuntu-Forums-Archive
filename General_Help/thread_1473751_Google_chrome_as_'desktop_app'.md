### Post by WorMzy on 2010-05-05
You can use Compiz plugin "Window Rules" to make Chrome non-closable, maximized, sticky (appears on all workspaces), below (always appears below all other windows), and skip the taskbar (so it doesn't appear in the list of open programs).

Just install ccsm, run it, and set up the window rules.
