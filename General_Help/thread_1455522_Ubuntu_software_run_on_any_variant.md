### Post by mcduck on 2010-04-16
Yes, all the Ubuntu variants are one and the same operating system, and run the same software. So you can install any program, server app, desktop environment etc regardless of what disk you use to install Ubuntu in the first place.

The only difference between different variants is what is installed by default (and what options you get during the install, in case of the server & Ubuntu Studio versions).

What comes to the unofficial Ubuntu-based distros, most of them are just Ubuntu with slight modifications in theme and default installed applications. Of course some *might* be more heavily modified distro, but the basic rule is that if it uses Ubuntu's software repositories then it runs the same software.
