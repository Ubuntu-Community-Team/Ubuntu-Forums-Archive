### Post by Titan8990 on 2008-12-11
If you need features of a newer version that is not in Synaptic and you feel comfortable compiling from source then that is the best option. Compiling from source will be easy if you have already had the program installed before because you will likely already have the needed dependencies. 

If you can't wait for the new release then do so. Most of us do.

### Post by Locke_99GS on 2008-12-11
Packages found in Ubuntu's repo's are tested to be stable and function well with Ubuntu, or have some special changes made - usually to integrate with Ubuntu better. 

To upgrade packages to beyond what it offered inthe repo's, I'd first look for Ubuntu packages from the maker (their official website) and then look for Debian packages. If none are found, then you will have to download the source and compile it yourself.
