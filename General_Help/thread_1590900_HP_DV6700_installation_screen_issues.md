### Post by ehode on 2010-10-08
Hey I fixed it.  Not the most sexy fix but this works for me.

* Install OS with the Alternate install CD
* Hook up external monitor and book up installed OS
* Upon getting to the desktop, select preferences -> additional drivers and installed the nvidia closed drivers
* Reboot
* REJOICE!

I blogged about it on my site.

[http://e0d.com/blog/hp-dv6000-ubuntu-10-410-10-display-installation-issues](http://e0d.com/blog/hp-dv6000-ubuntu-10-410-10-display-installation-issues)

Hope that this works for anyone else with this issue.
