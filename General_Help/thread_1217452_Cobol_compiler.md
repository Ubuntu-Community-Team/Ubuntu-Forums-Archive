### Post by michy99 on 2009-07-19
There is a cobol compiler in the repositories called open-cobol. You can install it with a simple command in the terminal:```
sudo apt-get install open-cobol
```If you want to install the tiny cobol, you need to extract the tar.gz and look for a README file or INSTALL file. If it is source code, it will have to be compiled.
