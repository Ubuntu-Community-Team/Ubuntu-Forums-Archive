### Post by treesurf on 2011-11-08
I have searched the forums and noticed a few other people reporting the same problem, but no solutions have been suggested and since there isn't a dedicated thread yet I'm starting this one.

Symptoms:  Screen turns off as per default power saving mode.  When the screen is brought back on by user input (touchpad in my case)the system is frozen.  It is possible to move the pointer around the screen but aside from that the screen is frozen and no combination of clicks or keyboard input elicits any response.  A hard shut down is required at this point.  This occurs only every once in a while and I haven't been able to see any pattern yet.

I am running a Lenovo Ideapad Y550, but the few other reports I saw of this problem on the forum occurred on other hardware.

Any suggestions on how to solve this would be greatly appreciated.
