### Post by rumli on 2007-11-12
Hmm.  Gmail is not a mail client.  A mail client (such as Evolution and Mozilla Thunderbird) is a program that allows you to access a mail account using a mail protocol.  Gmail is a web-based mail service (similar to Yahoo mail and Hotmail).  It doesn't require a mail client and can be accessed via a web browser (such as Firefox).

In fact, until recently, the web browser was the *only* way to access Gmail.  However, Gmail has recently started supporting IMAP, which is a mail protocol that allows you to access your account via a mail client.  You can enable IMAP support in Gmail by going to your Gmail account in the web browser and go to the ["Settings" link]("https://mail.google.com/mail/#settings"), followed by the "Forwarding and POP/IMAP" link.  There are instructions on there on how to set up your mail client to access Gmail.
