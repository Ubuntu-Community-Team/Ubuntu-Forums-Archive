### Post by bobblehat on 2008-06-05
I'm guessing a little on what's happened but if it's gone full screen and you can't get it back via the menu, you can maybe fix it by going into Preferences -> Keyboard Shortcuts, finding the entry for 'Toggle fullscreen mode' and set a key for that (e.g. Alt+F3 or whatever you can find that isn't in use).

Then use that key combination to get the app out of full screen.
