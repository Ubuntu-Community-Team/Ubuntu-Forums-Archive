### Post by ckempo on 2007-04-17
Add the deskbar to a panel, and assign a shortcut key to it (I use ALT+Q but the choice is yours).

Then everything you want is just an Alt+Q away! I've got my deskbar configured so I can do everything from the keyboard - switch windows, launch programs, search the web, find files, even reboot/shutdown. Have a play with it and you should see how useful it is. :D

### Post by Smuve on 2007-04-17
You can try Katapult.  Directly from it's website - [http://katapult.kde.org/wiki/Handbook]("http://katapult.kde.org/wiki/Handbook"):

> Katapult is a KDE application that provides you with faster and easier access to your applications, bookmarks, and more. It is plugin-based and can launch anything it has a plugin for. Its plugin-driven appearance is completely customizable. **[COLOR="Blue"]Katapult was inspired by Quicksilver for OS X[/COLOR]**, and it is written in C++.

I believe it's in the repo's, so you can just
```
sudo aptitude install katapult
```
and try it out.

It's KDE based, but works perfectly fine in Gnome.
