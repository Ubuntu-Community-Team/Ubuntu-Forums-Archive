### Post by Rocket J Squirrel on 2010-04-15
I am connected to a Windows network via Samba and I know that I have at  least one shared folder on this machine. 

Software like  remastersys advises one to unmount network shares before running the backup  software, so how does one go about getting a list of any shared folders on the machine so  they can be unmounted?

### Post by plucky on 2010-04-15
> **Rocket J Squirrel said:**
> I am connected to a Windows network via Samba and I know that I have at  least one shared folder on this machine. 

Software like  remastersys advises one to unmount network shares before running the backup  software, so how does one go about getting a list of any shared folders on the machine so  they can be unmounted?

Open **Places > Network** will show you all the machines on the Network.

Just open the machine you want to find the shares on will tell you what folders are shared.

Good Luck
