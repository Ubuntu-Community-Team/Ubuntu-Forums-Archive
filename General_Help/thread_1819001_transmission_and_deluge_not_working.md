### Post by Cas07 on 2011-08-05
You have no included which browser you are using. 

There are several options to opening a torrent file: 

[LIST]
[*]Open directly from browser.

[*]Save torrent file to disk then either double-click it to open with default torrent application, right-click and 'open with' another application, or manually find it from the add torrent dialog in the application. 

[*]Copy the url or the torrent file and manually paste the link into the torrent application.
[/LIST]

It sounds like the torrent file is being saved and not opened with the application which could be an issue with the Desktop file. 

In the case of Deluge I would recommend updating to the latest version, currently 1.3.3, which is in our PPA. There is a [guide]("http://dev.deluge-torrent.org/wiki/Installing/Linux/Ubuntu#AddDelugePPARepository") on how to add a PPA repo and update Deluge.
