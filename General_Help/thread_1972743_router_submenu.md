### Post by strask on 2012-05-03
Was there an old router with the same name? If you add a new router with the same name as an old one, it would probably cause that situation. The one with the "1" after it is likely the new one.

If the old router is no longer on line, you can go to "Edit connections" in the wireless menu to remove the listing for it.
