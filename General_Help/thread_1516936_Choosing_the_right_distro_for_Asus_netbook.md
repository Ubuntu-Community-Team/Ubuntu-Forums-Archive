It really comes down to personal preference as to which linux distro you use, but the ubuntu netboot 'remix' has been explicitly designed for netbooks. ### Post by snowpine on 2010-06-24
Your English is fine, no worries! :)

I would recommend starting with Ubuntu 10.04. If you have an external CD-ROM drive, you can create a "Live CD" or if you have a USB flash drive, you can create a "Live USB."

[https://help.ubuntu.com/community/LiveCD](https://help.ubuntu.com/community/LiveCD)
[https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)

Either way, you can run Ubuntu in "Live" mode with no change to your computer. This is a good way to test drive it to make sure you like it and that it supports your hardware. You can also use the Search box at the top left of this webpage to search for "1005P"; this will tell you if other users have had positive/negative experiences with this specific model.

Kubuntu, Xubuntu, Ubuntu Netbook Edition, etc. are all just variations on Ubuntu. They have different user interfaces ("desktop environments"), appearances, and pre-installed applications (but you can install applications from one of the other 'Buntus no problem if you want). If you have enough time, you can try all the different versions to see which you like best. There are a lot of screen shots on the web that might help you decide. I recommend starting with Ubuntu first if you're not sure. Ubuntu is the original and the "captain of the team."

Then, there are other Linux "distros" apart from Ubuntu, like Debian, Fedora, Puppy, etc... but don't worry about those yet; Ubuntu is a good choice. :)
