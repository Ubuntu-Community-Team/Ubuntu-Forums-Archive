### Post by _uli on 2010-02-08
You can delete the ubuntu installation/partitions and resize the partitions with a Live CD.
On Windows Vista and Windows 7 you can go back using the Windows MBR by booting from the Windows DVD and use the Repair function.

I would give 10-15 Gb to the Ubuntu installation, then after the year has passed you can delete the windows partition and give more space to the Ubuntu installation :p
