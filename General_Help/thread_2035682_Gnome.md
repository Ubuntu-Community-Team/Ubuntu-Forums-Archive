### Post by Nytram on 2012-07-31
I'm using GS on anther distro but this should work for Ubuntu too.

It sounds like you're in Fallback mode which is similar to the old Gnome, and is there for graphics cards which aren't compatible with GS.

To switch off Fallback mode (assuming your graphics card is compatible) and get the regular GS, open *System Settings** and choose *Details* followed by *Graphics.* Here you can turn off Fallback mode and the normal GS will appear when you next login.

If this doesn't work you could try reinstalling Gnome Shell, I had to do this on my setup to get it working.

*Also available by entering *gnome-control-center* into a terminal.
