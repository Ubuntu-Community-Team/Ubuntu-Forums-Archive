### Post by ijason on 2007-11-28
i believe that you can do this. you'll need to boot off the live-cd and then see if you can mount the computers hard-drive. 

if you go to Computer in the 'Places' menu, you may see a hard drive icon that will say something similar to "60gig volume" with whatever size your drive is listed at. if you double-click the icon it should mount the drive and give you a file-explorer like interface to navigate around for files. 

the only thing i'm not sure of, is if you'll have write permissions to that drive. since you can't log in as root off the boot cd, and it may require root access to write to the ntfs (windows) hard drive. try and and let me know :)
