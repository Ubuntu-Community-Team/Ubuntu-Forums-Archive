### Post by rookie on 2005-03-13
First, I used btdownloadgui.bittornado. I didn't like that too much, so I trîed to install Azurues. It worked and Azureus is running.

However, when I click on a torrent on a web-site, the torrent is downloaded to my home directory. I then have to start Azureus and select the torrent file.

How can I make Azureus the standard bittorrent-client, so that when I click on a torrent on a web-site, Azureus starts and begins the download?

Thanks for your help

---

### Post by ow50 on 2005-03-13
In Firefox: Edit > preferences > downloads > file types. Change the action for torrent file to 'Open with azureus' instead of 'Save to disk'.
