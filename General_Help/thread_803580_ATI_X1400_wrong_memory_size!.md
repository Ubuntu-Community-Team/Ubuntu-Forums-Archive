### Post by danwood76 on 2008-05-23
You can set the amount of shared memory in the BIOS.
It will steal from your available RAM tho (I can see you have a lot so its not a problem)

When you first turn your system on enter the BIOS (this is normally done by pressing the 'Delete' button or F2)
Have a look through all the options there and you should see one that says video memory or something, set this higher and you will have more Video RAM set.
