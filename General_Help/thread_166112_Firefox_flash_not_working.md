### Post by front243 on 2006-04-25
Here is what I did on Drapper Drake:

1. Download the plugin manually from
[Flash - Linux]("http://www.macromedia.com/shockwave/download/index.cgi?P1_Prod_Version=ShockwaveFlash&P2_Platform=Linux")

2. Untar it

3. Make a "plugins" folder in your ~/.mozilla folder

4. Copy the files "flashplayer.xpt" and "libflashplayer.so" to the plugins folder you made in step 3.

5. restart firefox, now flash should work, try to type "about: plugins" in the adress-bar to check if the plugin is enabled.
