### Post by dandroid13 on 2013-04-29
Create a folder called "s" and move your font to it. Then do a Alt+F2 and type```
gksu nautilus
```locate your folder and copy it to```
/usr/local/share/fonts
```
If you want to install another font then just name a folder with the starting letter and copy it there.

### Post by ajgreeny on 2013-04-29
I think you will still need to update the font cache with ```
sudo fc-cache -f -v
``` for all your new fonts to be available for all applications, unless things have changed.
