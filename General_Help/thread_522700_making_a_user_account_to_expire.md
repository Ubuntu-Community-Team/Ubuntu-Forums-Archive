### Post by UltraMathMan on 2007-08-11
The useradd terminal application allows you to set when users expire. I think it can only set as high as the day they expire, not the exact time in that day however. Check out it's man page ```
man useradd
``` or [useradd man page]("http://linux.die.net/man/8/useradd") for more info.
