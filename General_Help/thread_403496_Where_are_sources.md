### Post by 23meg on 2007-04-07
> Where can i donwload sources?

You can download the sources for any package with

```
apt-get source package_name
```

> Is whole ubuntu under GPL?

No, but the Main component is made up of Free software that complies with the [DFSG]("http://www.debian.org/social_contract#guidelines").
