### Post by kurja on 2020-04-26
In a nutshell, using the desktop space for anything but looking at your wallpaper is no longer supported in current versions of gnome / nautilus.

There is an extension that attempts to re-create some of the functionality, at least it allows dragging and dropping of files to the desktop, but dropping urls as shortcuts does not seem to work with that either. I have no solution for you, sorry :/

[https://extensions.gnome.org/extension/2087/desktop-icons-ng-ding/](https://extensions.gnome.org/extension/2087/desktop-icons-ng-ding/)
