### Post by blackbird34 on 2013-02-25
Tools > Options > Language Settings > change the locale to whatever country you want the units of measurement from (e.g I have my LibreOffice in English with a french locale to get metric units as i live in france, not the UK or US)

### Post by TimeCrank on 2013-02-25
***************************************************************************************
=======You can select separate measurement units for **[COLOR=Blue]LibreOffice [/COLOR][COLOR=Blue]Writer[/COLOR]**,  **[COLOR=DarkOrange]LibreOffice Writer/Web[/COLOR]**,[COLOR=Green]** LibreOffice Calc**[/COLOR], [COLOR=DarkOrchid]**LibreOffice Impress**[/COLOR] and [COLOR=Lime] [COLOR=Red]**LibreOffice Draw**[/COLOR][/COLOR] documents.
 
[LIST=1]
[*]Open a document of the type for which you want to change the measurement units.
[*]Choose **Tools - Options**.
[*]In the left pane of the dialog, double-click the application for which you want to select the measurement unit.
Double-click **[COLOR=Blue]LibreOffice Writer[/COLOR]** if you want to select the measurement units for text documents.
[*]Click on **General**.
[*]On the **General** tab page, select the measurement unit.
[*] Close the dialog with **OK**.
[/LIST]
Hope this helps you out.:)
