First of all Linux is no more secure than anything else.

A virus is unlikely in a Linux environement as there are none currently in the wild though could easily be targeted with viruses if people chose too.

Knowing if you had a trojan or other malware is about vigilance and checking logs etc beware of downloads, only install from trusted sources (though trusted sources can become compromised)

See here for more information from a wiki we have been working on [https://wiki.ubuntu.com/BasicSecurity](https://wiki.ubuntu.com/BasicSecurity)

As for root, it as mentioned already is locked by default and rarely needed, to do admin tasks we use sudo see here [ROOTSUDO]("https://help.ubuntu.com/community/RootSudo")
