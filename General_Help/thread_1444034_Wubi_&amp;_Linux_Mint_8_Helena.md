### Post by mechro on 2010-03-31
Wubi isn't really a "dual-boot". It installs Ubuntu inside Windows and gives you the option to choose Windows or Ubuntu at boot time.

I think the Wubi you've downloaded only installs Ubuntu. You'd need a Wubi Mint edition to install Mint. Check the Linux Mint site to see if they do such a thing.
