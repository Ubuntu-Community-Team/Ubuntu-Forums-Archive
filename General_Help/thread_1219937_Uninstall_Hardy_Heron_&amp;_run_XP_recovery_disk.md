### Post by egalvan on 2009-07-22
OEM re-installation disks generally wipe the drive and do a factory-default install...

Some may be more generic, giving options to delete & create partitions, or recover/repair an XP install...

Generally, if you want to wipe off Linux, and HAVE SAVED ANY DATA YOU WANT TO KEEP, simply booting from the re-install CD will do the job...


And don't worry about "offending" anyone... not everyone is able to use Linux...
Some are forced (by software or hardware requirements) to use another OS...
and others simply like other OS's...

It's all about CHOICE, and true *nix geeks understand this... :)

And then there is Virtual Installation.... using a virtual machine to run XP under Linux.
Yet another choice.
