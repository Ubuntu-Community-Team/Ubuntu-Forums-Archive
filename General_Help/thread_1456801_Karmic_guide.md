### Post by luigi_mb on 2010-04-17
> **cmcanulty said:**
> I was hoping to save a pdf not print it, thanks.

In your browser click on File > Print and then select PDF. You will find the pdf file in ~/PDF. 

This assumes you have installed the "cups-pdf" package from the repositories. If not, 

```

apt-get install cups-pdf

```

/luigi
