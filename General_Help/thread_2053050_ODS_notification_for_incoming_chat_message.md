### Post by cbanakis on 2012-09-04
What messaging program are you using?
Empathy?

By default, it should pop up the same as emails.

I did not like it though, because its easy to miss the bubble, since it only pops up for a moment.

If you are using empathy, there is a setting in the options to disable the bubble notifications.
(This makes new messages pop up in a new window, so you don't miss anything)

Not sure if your getting the notifications or not, but in my case, I was getting them, but not noticing.

Which is why I disabled the bubble notifications.

Eventually, I just switched from Empathy to Pidgin. 

Pidgin is much better in my opinion.
(The the logo/icon is pretty corny)
