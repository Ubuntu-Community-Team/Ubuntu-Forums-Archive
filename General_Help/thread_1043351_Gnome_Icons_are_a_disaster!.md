2. Then when I move the theme to my .icons folder, and go enable it in Appearance, many of the icons are replaced with default GNOME or default Human icons.....

### Post by FAMUgolfer on 2009-01-18
try moving the desired icon folder to /usr/share/icons...then go back to appearance>>>customize>>>icons tab>>>and select the desired icon theme.

PS. a lot of icon themes are not complete, so if you choose an icon set that is not complete, the default gnome icon will appear.

hope this works out!!!
