I don't know Lyx, but if you can use LaTeX commands then:

1. ```
\textbf{ some text } 
```will make some text bold
   ```
 \textit{ other text} 
```will make other text italic
    ```
\section{ some text}
``` makes a header

   There's lots more you can do, just search for LaTeX fonts and the like

2.  ```
\newpage
``` makes a new page in LaTeX

I have an idea that Robin Turner's guide may be for a older version of LyX. It might be worth looking at the various tutorials and things under the help menu - the documentation is pretty good. I found I needed to take it slowly and not try to read too much at a time.

"Blocking the text" refers to selecting it with the mouse. To format a piece of text, click on the text style button (with "ab" on it) in the toolbar.
