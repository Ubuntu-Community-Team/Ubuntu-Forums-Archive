### Post by Kyral on 2005-10-23
Its not possible. And I can give the reason. There isn't enough space to fit both the i686 kernel and the K7 kernel on the install CD. So the developers went with the kernel that would run on ANY x86 machine, the i386 kernel.

So you'd have to use Synaptic/Apt-Get to install the K7 after the install
