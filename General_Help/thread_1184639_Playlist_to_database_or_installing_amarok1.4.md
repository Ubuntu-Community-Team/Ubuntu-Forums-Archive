### Post by oldtime on 2010-04-02
You can install amarok from your repository - Applications > Add/Remove Software > Amarok
You can use Coverprint to print out a playlist. Installing it was a pain for me, until I found out how to do it in Amarok - Tools > Script Manager > Get More Scripts > Install Coverprint

Hope that helps!
