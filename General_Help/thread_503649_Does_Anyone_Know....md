### Post by strabes on 2007-07-18
You don't need a firewall for ubuntu. All unnecessary ports are closed by default so installing a firewall is useless unless you need a frontend to configure it.

Unless you're a pro, the GIMP is prefectly adequate, especially if you enable [Photoshop-ish keyboard shortcuts](http://epierce.freeshell.org/gimp/gimp_ps.php).

Don't know about fruityloops or similar programs like Acid Pro. If all else fails you can dual boot or virtualize, both of which are really easy.
