### Post by Nicholas_Thomas on 2014-08-14
I had this problem too. I found my solution in the comments of [this]("http://www.moritzmolch.com/1453") article. The one that helped me was #6 by Romel Pérez:

>  			 			Hi!

 I had the same problem, but I solved it!
 I was not able to add the Anthy text entry in Ubuntu 14.04 using  Gnome Fallback. Then I switch to Gnome Shell and I add it. Finally I  switch back to Gnome Fallback and I can use it now.
 &#12414;&#12387;&#12383;&#12397;&#65281;
 
 		


I too was using Gnome Fallback, but I didn't have Gnome Shell. I installed it with > sudo apt-get install gnome-shell, logged out and logged back in with it, (it appears as "Gnome" in the menu) went to the input method settings and added Anthy, and now it works!
