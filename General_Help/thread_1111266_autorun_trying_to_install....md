### Post by mcduck on 2009-03-30
That's a Windows program. You'll have much more troubles trying to run that on Linux than just starting the installer without autorun.. ;)

Just to make this clear, the basic rule is that Windows software only works on Windows, Mac software only works on OS X and Linux software only works on Linux.

That being said, you can try if Vegas runs with Wine (which allows running some Windows programs on Linux, working as a kind of interpreter between Windows program and Linux system. But while some things run fine with Wine, others don't run at all..)

If you still wish to try, and ave Wine installed & configured, just open the CD in a file manager and file the installer file, or open the autorun.inf in a text editor to check which file it would start. Then start that with Wine an hope for the best.
