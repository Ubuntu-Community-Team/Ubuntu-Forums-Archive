### Post by ryanhaigh on 2008-07-29
You might want to check your system log to see if there are any more clues as to what is causing this. You can either look at the file directly /var/log/syslog or check using system>admin>system logs. 

If this is related to power and my guess is that it is then it is probably time to invest in a UPS. 

My parents computer was constantly locking up and I could not figure out what was causing the issue, at first I blamed windows then one day when my father was welding the lights flickered and the PC locked up on me which is how I figured out what the problem was. They now have a UPS and haven't had a lockup since. That was a windows PC but I guess the lesson is that computers react poorly to bad power.
