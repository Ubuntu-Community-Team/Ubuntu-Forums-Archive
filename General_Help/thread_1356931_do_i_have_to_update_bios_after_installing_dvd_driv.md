### Post by 3rdalbum on 2009-12-16
> **cain071546 said:**
> i cant boot from disk after installing new dvd burner
do i have to update my BIOS or what?

No, but usually the BIOS has a list of what CD/DVD drives you have, and you can select one to be in the boot order. You might need to select the new one to be in the boot order, and then make sure the boot order has the DVD burner as the first entry.
