### Post by Kapitän Rotbart on 2008-02-14
If you want to have the Pidgin spellcheck in Polish and strictly Polish, there certainly is a way. If you run the desktop in Polish (System>Admin>Languages, install the Polish language pack from there and then) Pidgin spellcheck will be strictly in Polish. Not only that, but most of your desktop will be in Polish (at the exception of some of the applications that were not included in your Ubuntu install that have not yet been translated).
