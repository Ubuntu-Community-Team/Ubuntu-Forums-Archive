### Post by bsussman on 2005-08-27
[QUOTE=anovas]I use a live Cd
How do i increase the size of the filesystem???
Mine says the limit is 650 mb..
[/QUOTE]

The filesystem referenced is your live cd itself.  You cannot alter the size of a cd.

If you must continue to use the live cd,  you will need to format and mount a disk partition.

You might find it easier to use the install cd (NOT the live cd), install on hard drive and use that - this would result in a system more tailored to your personal preferences and hardware, and will be generally easier to use since changes, customizations, etc, are preserved from boot to boot.
