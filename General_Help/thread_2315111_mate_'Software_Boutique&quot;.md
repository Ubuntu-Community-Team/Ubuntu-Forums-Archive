I'm a couple of months late here, but there is now a search feature - but this currently only searches software **within the Boutique**. This might change in 16.10 Yakkety.

The Software Boutique isn't designed to be a replacement for a software center or package manager, so you'd have to resort to other methods to search for packages like that. There is a **More Software** category that offers such utilities
