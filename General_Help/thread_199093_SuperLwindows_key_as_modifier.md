Is it possible to use the SuperL or 'windows' key as a modifier key?
i.e. to simulate "win+L" = lock screen.  I can make just the windows key lock the screen, but a single key-press seems rather prone to accidental presses, and currently I have it set up as ctrl+alt+L, but I would like to know if it's possible to use the windows key as a modifier.

### Post by mannheim on 2006-06-18
[QUOTE=Lunar_Lamp]Is it possible to use the SuperL or 'windows' key as a modifier key?
[/QUOTE]

This should work if you go to System-->Preferences-->Keyboard-->Keyboad Layout Options. Select "Super is mapped to Win-keys". Then your Win-key should be a modifioer.
