### Post by RikoW on 2005-11-22
I have no problem with printing ps files generated by psnup.

I usually create an actual file first with psnup and only then send the file to a printer, like:

```
psnup -2 paper.ps paper2.ps
lpr -P my_printer paper2.ps

```

works fine!

Do you do the same or do you pipe it directly into lpr?
If so, does the above work for you? how about looking at paper2.ps with the ps viewer like ggv?

Cheers,

             Riko

ps: I can also recommend pdfnup which works directly on pdf instead of ps ... as the name suggests! :)
