### Post by deadflowr on 2014-11-22
The package is call ubuntu-wallpapers-<codename>
replace the <codename> with that of the release version,  12.04 is precise. 11.10 was oneiric.
Ex: ubuntu-wallpapers-precise

When installed the wallpaper will be in the system folder /usr/share/backgrounds.
You can copy it by right clicking on it. > There is no need to use sudo/root to copy, if you simply copy it into your home folder.
