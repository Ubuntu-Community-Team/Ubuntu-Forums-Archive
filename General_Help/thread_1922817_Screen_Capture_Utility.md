### Post by cbanakis on 2012-02-09
compiz has a screen shot tool.

Get it in the software center (compizconfig)

Then type ccsm in terminal to open.

Select extras tab on the left.

Check the box for screen shot.

Then just hold down the windows/super key, and while holding it, left click and draw a box over what you want to capture.

and thats it, file should appear on your desktop.

### Post by Jay Car on 2012-02-09
I don't know if this might be helpful, or not.  But I often save entire pages of tutorials from websites, and because sometimes the tutorials contain commands to copy/paste, I don't want to save the web page as an image. So I just copy (ctrl+a), open a new LibreOffice doc and paste (ctrl+c) the whole web page into a document, then export to PDF.  

This gives me all images on the web page, plus links, and I can still copy/paste any commands from the PDF if I need to. 

I can also edit out any unnecessary parts before exporting to PDF. This is actually the only thing I ever use LibreOffice for, it works really well.
