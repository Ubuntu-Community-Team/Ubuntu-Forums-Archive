### Post by westie457 on 2015-03-15
When typing your password into a terminal no output shows on the screen, the keystrokes are registering. This is a security feature so no one watching the screen can get the length of your password as a parameter in cracking the password.
Type it as you would normally and hit 'Enter'.

### Post by Bucky Ball on 2015-03-15
If it is saying 'Sorry, try again', you have the wrong password and that is a new issue. As Westie says, you don't see anything when you type in your password.
