### Post by moster on 2009-07-02
Ok, you do not really need compiz fusion icon...

Here how it goes my installation of emerald. After fresh install of jaunty I install "advanced desktop effects (ccsm)" from add/remove menu, after that emerald from synaptic.

Installation is finished and last step is to make emerald default and this is done by going to system menu-preferences-compiz config and then "effects" tab and "window decorations". There you will see "command". There will say "/usr/bin/compiz-decorator" and you replace it with "/usr/bin/emerald"

restart X or whole computer.

That is safest possible way in my knowledge :)

edit:
you could install that from terminal but I am a GUI guy :)
