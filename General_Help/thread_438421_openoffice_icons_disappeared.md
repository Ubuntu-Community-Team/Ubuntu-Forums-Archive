### Post by kyphi on 2007-05-09
In Open Office, go to View -> Toolbars and place a tick next to "Standard".  That should fix it.
It is always a good idea to lock the Toolbar after that.

---

### Post by kyphi on 2007-05-10
The only way that I can loose my icons and have only text remaining is by going to View -> Toolbars -> Customise and then clicking on the Toolbar button (below New) and selecting "Text only".  Perhaps you can check your settings in that location.

---

### Post by Hagar Delest on 2007-05-10
This has been discussed on forums. It comes from the themes that have been separated from the main install by Ubuntu team. Go to Synaptic and install the themes you want to use (Tools>Options>OOo>View).

### Post by Tobu on 2007-05-13
