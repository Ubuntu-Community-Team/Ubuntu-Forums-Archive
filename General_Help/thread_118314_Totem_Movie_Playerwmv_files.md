### Post by theturner on 2006-01-16
totem-xine handles far more media formats and makes the timing problems that some people experience go away.
You can install additional codecs as described [here]("http://ubuntuguide.org/#codecs"). Note that some of these codecs are patent-encumbered or proprietary and not 100% legal to install if you live in the US.
