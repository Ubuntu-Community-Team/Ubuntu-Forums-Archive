### Post by HermanAB on 2022-10-25
Hmm, if you are a new Linux user then this kind of problem can be very hard to resolve.  During installation, a simple video driver is used, to ensure that it will work on almost any computer.  After installation, the system loads a better driver specific to the video adaptor.  This will enable better video performance - or that is the idea anyway. To fix the problem, you need to experiment with different drivers, which can be hard to do since it is difficult to do something if you cannot see the screen or if the machine is frozen.  It may be much easier to try a different Linux distribution, such as Xubuntu and see how it goes.
