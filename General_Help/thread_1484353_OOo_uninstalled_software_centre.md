### Post by Dai777 on 2010-05-15
I uninstalled OpenOffice and in the process Evolution, Pino, Banshee, Gedit, Empathy, and Software centre were uninstalled automatically not by me all I wanted was to uninstall OpenOffice. Got every thing re-installed except for software centre. How do I re-install software centre.

---

### Post by Zilioum on 2010-05-15
Go to System -> Administration -> Synaptic Package Manager. Search for "software-center". Check the box next to it and click "apply". If you will, synaptic is just an advanced version of the software-center. You can install any package via synaptic, even the software center :D 
I personally prefer to use the terminal instead of a graphical interface, you'll get the same result by tipping ```
sudo aptitude install software-center
``` into the terminal.

Cheers
