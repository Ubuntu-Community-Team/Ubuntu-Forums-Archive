### Post by collisionystm on 2011-11-16
So in case anyone was having the same issue, as me I decided to make this. I hope its not recurring. 
I have
11.10 Oneiric x64
Catalyst 11.11
Wine 1.3
WoW Cataclysm

When I would wine Wow.exe -opengl
I would get an error about 'unable to find suitable graphics interface'
Also some opengl errors.

I went to the software center
I installed libgl1-mesa-swx11

This removed the default OpenGL instances that came with Oneiric.

WoW Works fabulously. 

Unity 3D and Gnome-Shell are broken, however I realize how fast Unity 2D is. The system seems to run A LOT nicer with this different openGL and 2D. 

Good luck. Hope I helped someone. Time to get coffee.
