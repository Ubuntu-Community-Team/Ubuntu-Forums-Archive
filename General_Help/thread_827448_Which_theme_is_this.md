### Post by Daimoneze on 2008-06-12
As per the title of the video, this is Compiz-Fusion albeit heavily customized. Using your favorite package manager you can install it. Simply search for compizconfig-settings-manager or in a terminal type ```
sudo apt-get install compizconfig-settings-manager
```I have also found a package named fusion-icon to be useful.

It seems to be much easier to get it running in gnome (Ubuntu) than KDE (Kubuntu). Gnome is what is used in the video.

Forgive me if I've missed any special steps as I'm still very much a n00b.
