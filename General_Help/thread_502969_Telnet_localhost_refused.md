janipewter@ghostserv:~$ telnet localhost 3333

but...

> janipewter@ghostserv:~$ telnet 10.0.0.5 3333
Trying 10.0.0.5...

> janipewter@ghostserv:~$ telnet ghostserv 3333

Connected to ghostserv.janipewter.net.
Escape character is '^]'.


eggbot  (Eggdrop v1.6.18 (C) 1997 Robey Pointer (C) 2006 Eggheads)

How can I make it work if I telnet localhost too?

### Post by Mr. C. on 2007-07-18
There's your problem:
> tcp        0      0 **10.0.0.5:3333**           0.0.0.0:*               LISTEN     4005/eggdrop

Your server is only listening on the 10.0.0.0/24 interface.  Configure it to listen on all interfaces.

MrC

### Post by Mr. C. on 2007-07-18
In your config file, set:

```
listen 3333 all
```

instead of

```
listen 3333 10.0.0.5
```

and restart your program.

MrC
