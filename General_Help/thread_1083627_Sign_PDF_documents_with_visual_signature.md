Hi!

I am looking for a way to digitally sign PDF documents which will visually mark-up the pdf.

What I found so far:
1. acroreader - it allows you to sign PDFs but only if they are "reader enabled"
2. In order to make a pdf "reader enabled" you must have a Acrobat Professional 8 or later - open the file and from Advanced menu click on Extend Features in Adobe Reader.

Cryptonit - this program can digitally sign PDFs, but with no visual mark.

Does any one now a way to make PDFs in Ubuntu "reader enabled", eg with Extended Features in Adobe Reader?

### Post by srangelov on 2009-03-22
Yes, there is a way of singing PDFs with visual mark. And it is called OpenSignPDF. This is an open source program for Linux for visually signing PDF files. It is part of the [Open Signature - an open source project for the digital signature of documents]("http://opensignature.sourceforge.net/english.php").

I wrote a little article about how to use this application:
[Digital Signature and Signing PDF Documents with Visual Mark]("http://my.oltrans.org/en/ubuntu-faqs/41-customizing/74-digital-signature-and-signing-pdf-documents.html")

Read the last section [Signing PDF Documents with a Visual Mark]("http://my.oltrans.org/en/ubuntu-faqs/41-customizing/74-digital-signature-and-signing-pdf-documents.html").

Thank you!
