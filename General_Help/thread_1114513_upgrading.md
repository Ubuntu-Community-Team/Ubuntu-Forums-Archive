### Post by mb_webguy on 2009-04-02
If you upgrade, no.  If you do a clean installation, yes.

The easy way to upgrade to Jaunty when it's released is to open System->Administration->Software Sources, and go to the Updates tab.  In the Release Upgrade box, select "Normal Releases".  Now, when Jaunty is released, you'll automatically be prompted to upgrade, which will essentially be the same as normal system updates.
