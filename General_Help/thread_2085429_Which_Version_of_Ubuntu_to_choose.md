### Post by GreatDanton on 2012-11-18
That depends on you. If you want to have stability and long term support go with 12.04.
If you want the latest features go with 12.10.

You can also create separate root (/) and home partition. Whenever you would like to install new version, all you have to do is just delete root partition and install new version on it.

Regards.

### Post by 681ankit on 2012-11-18
A new LTS version is released every 2 years. In previous releases, a  Long Term Support (LTS) version had 3 years support on Ubuntu (Desktop)  and 5 years on Ubuntu Server. Starting with Ubuntu 12.04 LTS, **both** versions will receive 5 years support. There is no extra fee for the LTS version;..Choice however depends on you..
