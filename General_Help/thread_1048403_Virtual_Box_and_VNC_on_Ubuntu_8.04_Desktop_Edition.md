### Post by whoop on 2009-01-23
VNC is installed by default on ubuntu-desktop. System-->preferences-->remote desktop. I believe the package is called vino (you can find it in synaptic and confirm that it is installed)
I do not believe virtualbox is installed by default, you can install it with synaptic. I think the package is called virtualbox-ose
