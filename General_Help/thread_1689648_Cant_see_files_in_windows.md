### Post by coldraven on 2011-02-17
FAT32 can only support files up to 4GB.
If you copied a DVD, at 4.6GB it will have screwed up.
See the "Limits" section here [https://secure.wikimedia.org/wikipedia/en/wiki/Fat32](https://secure.wikimedia.org/wikipedia/en/wiki/Fat32)
