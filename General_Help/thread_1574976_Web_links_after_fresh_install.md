### Post by ankspo71 on 2010-09-15
Hi,
My best guess is that you might have chosen to do a manual partition install and forgot to tell Ubuntu to 'format this partition'. Without re-formatting the partitions, all files that used to be there will still be there, unless they are overwritten by newer versions of the same files. (Edit: I just reread your post and saw what you said about reformatting the partitions)
