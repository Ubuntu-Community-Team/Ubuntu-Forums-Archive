### Post by Tomosaur on 2007-03-08
Dapper is the current long term support (LTS) version of Ubuntu. It is considered the most stable. Edgy Eft is the version after Dapper - most of the bugs have been squished now so it's also very stable - but it is not LTS. Feisty Fawn is the current development version of Ubuntu. If you don't feel comfortable using an unstable version, then Dapper or Edgy are for you. Edgy has several performance improvements over Dapper, and other things are slightly different, but it'll only take you a few minutes to get used to it.

Xubuntu should be available from the main site - if it's not working, try looking for a mirror of it elsewhere.
