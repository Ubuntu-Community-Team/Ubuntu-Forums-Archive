### Post by pointone on 2008-02-14
Python is probably a good choice; it's simple yet powerful and incredibly versatile. Python is an interpreted language, so you can write programs in any text editor, then run them by either using a  shebang line (#! /usr/bin/env pythonat the beginning of the script) or by running:

```
python <script>
```

You can simply run "python" from the command line to start the Python interpreter.

You may also want to consider learning shell scripting. It's absolutely invaluable.
