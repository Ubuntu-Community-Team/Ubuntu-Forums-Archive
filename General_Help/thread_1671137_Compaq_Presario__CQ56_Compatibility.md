### Post by pslat on 2011-05-07
[LEFT]Hi matdombrock, I've been running Ubuntu 10.10 with KDE (installed  afterwards) for a few months now and it works really well. No  installation problems but had a slight power management problem causing  the system to become sluggish which was solved in my case by disabling  desktop search and turning off desktop effects. My system is now running  really well so I'd say go for it. 
I resized the partitions on the  hdd and used the extra space for Ubuntu. There are issues here as the  system comes with a recovery partition and two primary partitions which  only leaves room for one more primary partition. If  you are not familiar with partitioning do some research on this first as  I have heard of some people using the entire hdd by mistake and losing the  original os along with the recovery options.
Among other things I'm successfully running:
Usual Multimedia: flash, mpegs, audio cd's, mp3's and DVD's
Openoffice suite
Opengl apps: e.g Google earth, Art of Illusion, Celestia, Stellarium, Snowglobe (Secondlife)
Wine apps: Google sketchup (no linux version)
Games: Flightgear, Sauerbraten, Openbve Smokin' Guns. Note: High end graphic eaters such as Zero Ballistics 2.0 do not work.
VMWare Player (for those times I temporarily need to use a certain other os)
Internet: Firefox, Filezilla ftp, Thunderbird email
External usb Bluetooth connection to Mobile phone and Handheld thingy.

- I haven't have any problem using flash in full screen (with 10.2.159.1).
- The wireless card works fine but you have to enable it each time after booting (by pressing fn-f12).
There are a few different models of CQ56 from what I can gather so my system FYI:-
Pentium T4500 2.3GHz dual core Processor
3Gb Memory
Intel Mobile4 chip-set GM45
Realtec RTL8102E network card
RTL8191 Wireless card
~300Gb hdd.
It's a real pleasure having Ubuntu on this laptop. Enjoy.:smile:[/LEFT]
