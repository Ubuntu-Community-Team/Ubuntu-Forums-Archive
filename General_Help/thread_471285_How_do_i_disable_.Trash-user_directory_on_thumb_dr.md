### Post by mainalisuyog on 2007-06-11
When i delete files on my removable media, it creates a folder called .Trash-user directory, which is hidden in ubuntu, but visible when i pug the disk in windows machine. Is there any way to prevent ubuntu from automatically creating this directory? I'd prefer if all deleted files went to .Trash-user directory in my home folder instead. Thank you.

---

### Post by merlinus on 2007-06-11
You can show hidden files in ubuntu (Ctrl-H) and delete the folder on your flashdrive at any time.

You can also press Shift-Del and the files will not go into the hidden trash folder but be immediately -- and irrevocably -- gone..

-merlin
