### Post by Mr_Bumpy on 2009-10-30
I noticed that the default Xubuntu theme isn't 100% compatible with FireFox (ie., black text on black background when adding bookmark).  If you want to keep the default theme, which admittedly is quite nice, you can install a Firefox addon: the dustfox theme variant found in the [Dust-BlueHaze GTK theme]("http://www.gnome-look.org/content/show.php/Dust-BlueHaze?content=111950") looks very nice with Xubuntu's new default theme.  This also eliminates any text color issues (at least that I've found).
