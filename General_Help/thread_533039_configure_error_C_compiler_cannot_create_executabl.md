Just trying to install pidgin on my less than vanilla install (I have done full update in adept after enabling several repositories) and got the cannot create executable error.  I had only installed gcc package before attempting this.  Getting build-essential package solves that, but just wanted to note further dependencies in the interest of completeness.

libxml-parser-perl
libglib2.0-dev
libgtk2.0-dev 
libxml2-dev


If you get this error:
checking for XML::Parser... configure: error: XML::Parser perl module is required for intltool
Run this:
```
sudo apt-get install libxml-parser-perl
```

If you get this error:
You must have the GLib 2.0 development headers installed to build.
Run this:
```
sudo apt-get install libglib2.0-dev
```

If you get this error:
You must have the GTK+ 2.0 development headers installed to compile Pidgin.
If you want to build only Finch then specify --disable-gtkui when running configure.
Run this;
```
sudo apt-get install libgtk2.0-dev 
```

If you get this error:
You must have libxml2 >= 2.6.0 development headers installed to build.
Run this:
```
sudo apt-get install libxml2-dev
```

Or google "ubuntu pidgin debian package" for binary install :).
