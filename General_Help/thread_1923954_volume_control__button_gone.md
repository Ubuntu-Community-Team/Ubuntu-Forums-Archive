### Post by winh8r on 2012-02-11
Right click on panel and select "add to panel"

select "indicator applet" 

click add

should be back again


if it is not in add to panel tab then open a terminal and type:


sudo apt-get install indicator-sound

should be in add to panel tab now



once you get it back right click beside it and select "lock to panel"
