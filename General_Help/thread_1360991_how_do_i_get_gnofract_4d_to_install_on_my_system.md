### Post by catenary on 2009-12-30
There are 2 sets of binary packages for Gnofract 4D: one for distributions using python 2.6, and another for python 2.5. You must be installing the wrong one. You want [http://sourceforge.net/projects/gnofract4d/files/gnofract4d/3.12/gnofract4d-python26_3.12-1ubuntu1_i386.deb/download](http://sourceforge.net/projects/gnofract4d/files/gnofract4d/3.12/gnofract4d-python26_3.12-1ubuntu1_i386.deb/download)
