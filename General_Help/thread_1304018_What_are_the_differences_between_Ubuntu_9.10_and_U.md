That is a typo. Ubuntu Netbook Remix does not require an Atom processor (for example, it runs fine on the original Asus eee, which used a Celeron processor).

The Atom-optimized version of Ubuntu is called LPIA (low powered intel architecture) and is available here: [http://cdimage.ubuntu.com/ports/releases/9.10/release/](http://cdimage.ubuntu.com/ports/releases/9.10/release/) (It does not use the netbook interface by default, but I think it can be installed if you really want.)

Not that a distro needs to be "optimized for the Atom" anyway. The Atom is a very popular chip that is well-supported by recent versions of the Linux kernel. :)
