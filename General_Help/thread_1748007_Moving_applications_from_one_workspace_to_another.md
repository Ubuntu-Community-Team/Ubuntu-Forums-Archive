### Post by matt_symes on 2011-05-03
Hi

To move the maximised window using the keyboard you need to press 

shift + ctrl + alt

and the same time and then use the arrow keys. 

This keyboard button configuration can be changed using ccsm and editing the desktop wall: 'move window with in wall' section

Kind regards
