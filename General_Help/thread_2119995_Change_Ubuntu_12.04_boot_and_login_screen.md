### Post by kclark on 2013-02-25
You could be able to do this with super-boot-manager.

```
sudo add-apt-repository ppa:ingalex/super-boot-manager
sudo apt-get update
sudo apt-get install buc super-boot-manager
```

[http://askubuntu.com/questions/201129/change-boot-animation-in-ubuntu-12-04](http://askubuntu.com/questions/201129/change-boot-animation-in-ubuntu-12-04)

I know you said you liked the older animation, but this will let you make custom animations :)
