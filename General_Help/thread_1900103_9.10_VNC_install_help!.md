### Post by snowpine on 2011-12-25
That's because the 9.10 release reached its "end of life" back in April and is no longer supported. 

You can download and install the current 11.10 release here: [http://www.ubuntu.com](http://www.ubuntu.com)

Also as a general tip, you need to watch out for typos, for example this is wrong:

```
sudo apt=get install vncserver
```

It should be:

```
sudo apt-get install vnc4server
```

I recommend using the Copy & Paste functions for accuracy. Also you do not need to log in as "root" because "sudo" takes care of that for you. :)
