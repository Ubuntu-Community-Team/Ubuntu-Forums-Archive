### Post by cronholio on 2006-05-30
I use the MPlayer plugin and I have been able to play all videos with sound so far (I've been using it since Hoary). Try to change the sound output in the MPlayer settings to ALSA.

### Post by Denis on 2006-05-30

I use Mediaplayer Connectivity which is a Firefox extension:

[https://addons.mozilla.org/firefox/446/]("https://addons.mozilla.org/firefox/446/")
