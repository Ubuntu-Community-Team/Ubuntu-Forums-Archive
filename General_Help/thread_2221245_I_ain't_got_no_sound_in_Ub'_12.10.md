### Post by ajgreeny on 2014-05-01
Try a live system (DVD or USB) before you reinstall to see if sound works in that; if it does it should also work in the installed system.

I recommend a clean install, not an upgrade from 12.10 to 14.04. You would have to go through all the intervening versions to get to 14.04 and that is frankly not worth doing.

### Post by agemini68 on 2014-05-01
Check in sound systems, does it say dummy output? If it does, my only solution was to get a new soundcard. If it is showing output, something like analog stereo duplex, try in sound systems going into input and select your microphone.
