### Post by DGortze380 on 2008-10-22
8.04 is a stable LTS release. 

8.10 beta is, well, beta. It may still have issues, especially on particular hardware. Honestly 8.04 had issues on my laptop from alpha all the way to 8.04.1.

My suggestion is this. Get 8.04. Learn about the OS. At that point you have two choices. Reinstall when you want to upgrade (my choice), or upgrade via the Update Manager (50/50 person success rate with this).

Or, go ahead and install 8.10 beta, start learning about the OS, hope you don't have any flaky issues that make it more difficult than necessary. When the final release is out, your install will become the final release through normal updates, no upgrade process at all. There will more than likely be no issues going from beta to final like this.
