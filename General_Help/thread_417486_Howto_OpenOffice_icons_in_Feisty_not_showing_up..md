### Post by FeraTech on 2007-04-21
For those having problems with Open Office icons not showing up and are using the Tango Theme:

1. Go to synaptic package manager.
2. Search for "Tango".
3. Install the Tango OpenOffice icon theme.

The Tango icon theme is not automatically installed in the Feisty upgrade. You need to do this manually.
