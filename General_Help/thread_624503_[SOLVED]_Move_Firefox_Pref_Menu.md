### Post by oscarsfriend on 2007-11-27
Type "about:config" in the address bar, then in the filter type "home".  The top match should be browser.startup.homepage, and set the value to the URL of the homepage you want.  Also, [this add-on for FF](https://addons.mozilla.org/en-US/firefox/addon/710) might do what you want, though I have not tried it myself. HTH.

EDIT: Actually, [this might be exactly what you are after](https://addons.mozilla.org/en-US/firefox/addon/2642).
