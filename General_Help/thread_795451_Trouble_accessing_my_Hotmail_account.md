### Post by danwood76 on 2008-05-15
In each web browser they say what sort of browser they are (which user agent) when they request a page.

In firefox you can install a plugin called 'User Agent Switcher' whcih allows you to dupe servers into believing you are a different browser.

On the top bar if you select tools then add-ons you should be able to search and install it from there.
Upon a restart of firefox a new option will appear in the tools menu where you can select IE as the user agent.
