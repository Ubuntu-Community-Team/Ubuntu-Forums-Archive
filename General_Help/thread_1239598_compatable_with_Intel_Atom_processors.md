### Post by starcraft.man on 2009-08-13
Intel Atom are standard x86 instruction set CPUs. 86-64 (i.e. 64 bit) support I cannot vouch for, it seems to depend on the specific run of the Atom that is inside (from what I read). Any Atom can run the 32 bit Ubuntu, your particular experience with how well it runs will depend on general level of hardware but it is in fact compatible. So basically just install regular 32 bit, I don't see any reason to use 64 on a netbook.
