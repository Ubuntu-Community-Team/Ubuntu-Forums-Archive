Well, my mother is sick of viruses and such, so im going to install ubuntu on it, but also have vista if its needed. I know about grub and all that, but i just wonder, what do i do to clear space for ubuntu? How do i install ubuntu without risking loss of data in Vista?

---

### Post by avtolle on 2009-01-09
First, make backup copies of the data. Second, from within Vista, defrag the drive (>1 time). Third, still within Vista, use the tool provided to shrink the Vista installation to provide space for the Ubuntu install. Fourth, boot from the Ubuntu live CD; when you get to the partitioning step (step 4, IIRC) select use largest available space option. Fifth, complete the installation. Post back with any questions.
