Hello everyone:

I am half windows, half ubuntu user on my computer, mainly because I need to use certain design programs and I'm taking multimedia class at school so I use Adobe products.
Last year, we were taking a Microsoft e-learning program to learn how to use Office. For this program, we needed the use of IE (which I hate BTW). I really wanted to use Firefox, so searching on the addons I found an interesting one: Coral IE Tab.
What this add-on does, is to load pages using an IE engine, while still using Firefox. This is made by clicking on the right corner a firefox icon that appears after you install the add-on. This will change the icon to the IE one and reload the page, now with an IE engine. Using this, I know many users will finally get rid of Windows for once and for all.
I hope this Tip helps everyone in here.

Link to Coral IE Tab: [http://coralietab.mozdev.org/](http://coralietab.mozdev.org/)
