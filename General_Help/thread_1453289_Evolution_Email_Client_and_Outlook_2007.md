### Post by daniel@deckersok.com on 2010-04-13
Update.

Ubuntu 9.10
Evotution 2.28.1

I tried all day yesterday to import mail and contact info from Outlook into Evolution. I had many near successes but never could get the Home, Work & Other addresses to import ever. All the rest of the contact info imported just fine.

This morning I opened Evolution and studied it a bit. All of my contact import attempts defaulted to importing contacts into the Ubuntu One address book. I could find no way to import into the personal address book. Finally, I deleted the Ubuntu One address book completely. I was then able to import contacts into the personal address book. Presto. All the address were there. Then to finalize my success, I moved all the contacts from the Personal address book into the Ubuntu One address book. All the addresses were gone again. As a test, I moved some contacts back into the personal address book and the address were still gone. I have reported this bug. If you are not using Ubuntu One then selecting Outlook.pst as single file import works perfect.
