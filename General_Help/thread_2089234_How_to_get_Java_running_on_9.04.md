### Post by QIII on 2012-11-28
To get a secure version, if it will work at all with 9.04, see the Java wiki link in my signature.

Find "Oracle Java 7", "Command line methods" and the link "Using webupd8.org's strikingly simple method."

You are going to have to use the terminal.  I don't think you have much choice.

The "Easy Linux Tips" method is not the "easiest" and it does not automatically update as Oracle releases updates.
