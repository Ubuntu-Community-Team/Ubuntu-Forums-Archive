### Post by grahammechanical on 2013-04-11
Did you install the Linux Mint desktop on Ubuntu 10.04? That URL must have already been in your sources list before you ran the upgrade. The Ubuntu upgrade would not upgrade the Linux Mint desktop because Ubuntu is not responsible for Linux Mint. Further more, in what way were those Linux Mint items connected to the Grub Settings?

From watching the posts in this forum over the last year and seeing the problems that people have when upgrading from 10.04 to 12.04, I would suggest that people take 10.04 back to a default state before running the upgrade. I tested upgrading from 10.04 to 12.04 several times last year and I had no problems upgrading a fresh/default 10.04. But I cannot say the same for those who modify the desktop.

Regards.
