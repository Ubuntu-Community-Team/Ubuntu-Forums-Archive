### Post by sloggerkhan on 2007-08-30
You can connect to an xp computer if it has  shared drive to get files or use a VNC-type viewer to access the windows machine if it has a client. However, getting to the shared drive won't help with open files, and on a windows box logging in remotely will probably require priviliges to force the other user to log out immediately, unless you want to share a frozen screen. (I am figuring this is XP, by the way.) In any case, once the machine is locking up, it's probably too late. Maybe try setting it to autosave next time?
