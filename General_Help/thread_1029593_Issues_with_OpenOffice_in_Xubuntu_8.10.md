### Post by x1a4 on 2009-01-03
r-click a panel; click 'Add new item'; click 'Screenshot'; click 'Add'.  This will add the screenshooter plugin to your panel.  You can also install the **scrot** package which will install Scrot--a command line screenshot tool.  Scrot also has a GUI which you can get [here]("https://launchpad.net/gscrot")
