### Post by Farmer of Bricks on 2008-12-22
There are very few native KDE4 themes at [kde-look.org]("http://www.kde-look.org/index.php?xcontentmode=75"), so you might want to install compiz and emerald, and take a look at [compiz-themes.org]("http://www.compiz-themes.org/").

code:
```
sudo apt-get install compiz
sudo apt-get install emerald
```
