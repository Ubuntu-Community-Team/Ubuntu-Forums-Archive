Isn't there anything remotely better than Evince for PDFs? Evince yields several problems.

The EPS equivalent of the same figure translated by the GIMP to a tif is also huge so the file appears small due to intelligent packaging. However, it opens fine on my Windooze netbook.
Then I always run into problems printing PDFs (Too many failed attempts), seems like memory problem to. Printing the same files from the netbook is a piece a cake......

Is there anything or should I install acrobat under wine?

### Post by 3Miro on 2010-07-02
I always found acroread to be excessive in terms of resources. Try it to see if it works for you, but don't get your hopes up.

Other options are XPDF, ePDFViewer and Okular with the last one being somewhat large under Gnome (it is default for KDE and is more featured and faster, but doesn't work as well under Gnome).
