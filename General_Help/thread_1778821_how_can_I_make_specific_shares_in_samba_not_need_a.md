### Post by CrazeDIzzleD on 2011-06-09
So I'm Samba to share some partitions to my other machines. One of the shares I want everyone on my network to be able to access without needing a password.

I can do this if I set **security = share** in the Authentication section, but then all shares don't require a password. I tried leaving **security = user** in the Authentication and then doing **security = share** for the share I didn't want a password on, but that didn't work (still need a password for it).
