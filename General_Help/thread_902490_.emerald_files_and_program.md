### Post by werries on 2008-08-27
You don't have emerald set as your title manager.
You need to go to System > Preferences > Sessions, and add the command with name "emerald" and command as
```
emerald
```
And then log out/in.
OR alternatively you can just do this in terminal
```
emerald --replace
```
But that command will need to keep the terminal open and you'll have to do it every restart.
