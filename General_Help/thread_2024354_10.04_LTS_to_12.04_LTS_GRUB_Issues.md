### Post by fasterthanlight on 2012-07-13
Hello!

I am currently trying to complete an upgrade of Ubuntu 10.04 LTS to 12.04 LTS, but I am having some issues with GRUB.

I created a USB install disk of 12.04 and installed it with no problems. My issue is that I am unable to boot into this from the grub version I currently have. My computer loads grub 1.98 automatically, but my new partition with 12.04 is not listed. I've tried several fixes, including updating grub and using [boot-repair]("http://paste.ubuntu.com/1090599/") with no luck.

However, when I set my boot order to boot from USB prior to booting from my hard disk, and I insert the USB drive I used to install 12.04 LTS, I am sent to grub 1.99, which displays everything correctly.

Any ideas on how to fix this?

---

### Post by fasterthanlight on 2012-07-13
Of course, just as I post this, I've figured it out. I ended up using grub-customizer, and this, for some reason, fixed it, and that's good enough for me!
