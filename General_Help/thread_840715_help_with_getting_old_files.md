Yes, of course! But first of all, welcome to Ubuntu. :)
What you need to do is run Ubuntu from a "live CD" instead of installing it. You should be able to mount your hard drive and retrieve any files from your windows partition. It will be easiest if you have an external hard drive or USB flash drive to copy the files to.

If you have 256mb or less of ram, you will have trouble running the live CD. In that case, you can try a Gparted Live CD or a light-weight distro such as Puppy or DSL that runs from a Live CD with lower-spec computers.

Does any of that make sense? :)

### Post by avtolle on 2008-06-25
OK; hit Alt+F2, in the box input gksudo nautilus, then hit Enter. When Nautilus starts up, in the places section, you should see something like File System. Click on that, and there should be a display of files/folders come up. Right click the same, an option is to copy; and you should be able to copy them to external media. HTH.
