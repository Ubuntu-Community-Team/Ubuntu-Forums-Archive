### Post by Gaiwecoor on 2012-02-11
This may come a bit late, but I got this from a post here: [https://bugs.launchpad.net/nanny/+bug/877318](https://bugs.launchpad.net/nanny/+bug/877318)

Gnome Nanny seems to be failing because it requires gdm. To get Nanny running in Lubuntu, install gdm in the terminal:
```
sudo apt-get install gdm
```

When presented with the option for configuration, go ahead and stick with lxdm.

This seems to have let it run on my installation this morning.
