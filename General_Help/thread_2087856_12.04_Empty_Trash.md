### Post by DukeOfMixture on 2012-11-25
You just want to empty your trash folder?

Be certain that this path is true for your OS. You don't want to delete something you need: from terminal:

```
rm -r ~/.local/share/Trash/files/*
```Do you have a File Manager?

If so, it may have an option for you to right-click "Trash" and then select "Empty".

If your your File manager doesn't offer that option, it's useless. Install Xfe and hit ctrl+del
