### Post by pasipasi on 2006-12-12
If you did this, the converting would actually make the quality worse than 128kbps (which is already very low for even a decent audio system). You can't convert ANY lossy file to another format or bitrate without losing quality in the process. For example, converting a 160kbps mp3 file to 128kbps mp3 would probably make it sound like a 96kbps file or even much worse.
