### Post by An Sanct on 2011-03-27
With Firefox, you can use the [Flash Aid]("https://addons.mozilla.org/en-us/firefox/addon/flash-aid/"), this will automatically download and install the latest flash plugin.

After that, tell us if the browser still crashes.
