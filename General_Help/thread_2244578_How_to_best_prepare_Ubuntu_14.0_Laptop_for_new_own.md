### Post by deadflowr on 2014-09-17
I'd simply use a livecd/usb, and reformat the drive. I'd leave the partitions as they were, just mark them to be formatted/reformatted.. I think if normal you'd have one ext4, an extended, and a swap inside the extended.   
Then I'd leave it as is, and instead of installing I'd give them the livecd/usb to install it themselves, or at least have them be there when the install happens, if possible.
That way, there would be less confusion about setting up users and stuff.

### Post by rolkin on 2014-09-17
One suggestion would be: if you feel comfortable, and if they're willing to learn a little, sit down together and go through the installation and initial setup together. They may really appreciate the time and I think it's always a good idea to see how something like an OS gets setup (even if it's not *that* complicated to do). I'd also like to think that they'd be able to give you good feedback as to what they like and don't like, leading you to install and configure what's appropriate for them.
