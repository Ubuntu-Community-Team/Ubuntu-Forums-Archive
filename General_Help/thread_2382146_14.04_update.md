### Post by ajgreeny on 2018-01-09
The simple answer is "no", the software of your OS, be it Linux, Windows or mac, has no bearing on those CPU bugs; that is all down to hardware design, not software.

However, it is important to make sure that whichever OS you use is kept fully updated with all the patches and security updates available, so continue what you're doing now and you are as good as you can be in relation to these CPU problems.

There is, perhaps not surprisingly, a long sticky thread about these bugs at [https://ubuntuforums.org/showthread.php?t=2381801](https://ubuntuforums.org/showthread.php?t=2381801) so have a good read through that which answers a few questions as best they can be at present.
