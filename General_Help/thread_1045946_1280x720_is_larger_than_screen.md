### Post by gjoellee on 2009-01-21
You can change this section: ```
 
Option "UseModes" "1280x720" 
```Into the screen resolution you would like, ex to 1280x1024:
```
Option "UseModes" "1280x1024"
```
Remember to change the other places where it says "1280x720" with "1280x1024".

If that crashed xorg, run "xfix" see how here: [http://www.kshoster.net/content/howto-run-xfix](http://www.kshoster.net/content/howto-run-xfix)
