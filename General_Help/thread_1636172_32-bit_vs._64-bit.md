### Post by thatguruguy on 2010-12-02
It all depends on what you'll be doing with it.

It doesn't really matter on the computer my kids use. On MY computer, I occasionally rip DVDs or do graphics work or use blender, all of which works a lot better when you can multi-thread.

On the computer that I have set up as an HTPC, some game emulators (notably dolphin-emu) work a lot better if you can take advantage of multiple processors.  And, since it's an HTPC, it also processes a lot of videos, so it benefits from running a 64-bit OS.
