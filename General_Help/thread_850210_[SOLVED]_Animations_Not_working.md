### Post by sayakb on 2008-07-05
Have you recently enabled some plugin at ccsm? If yes, then open CCSM, goto **Preferences** and click **Reset to Defaults.**
If you dont have CCSM, at a terminal:
```
sudo apt-get install compizconfig-settings-manager
```
Then goto System->Preferences->Advanced desktop effects settings
