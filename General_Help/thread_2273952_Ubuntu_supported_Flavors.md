### Post by grahammechanical on 2015-04-16
There are different views on what is meant by "safe to use."

It is generally accepted that Linux is less likely to become infected by malware because criminals are more likely to attack a Microsoft OS because the fact that there are many, many more users of Windows than of Linux so it is more worth their while to target Windows. To put it another way, even supported Windows versions get infected by malware.

In a general sense, support in Ubuntu means security fixes for the Linux kernel. Once a version of Ubuntu reaches end of life it will no longer get kernel security fixes. The software repositories or archives are usually closed. So, there will be no updates of any kind and even installing software will become impossible without a workaround.

The OS is still usable but is it safe to use? That all depends. Here is the Ubuntu weekly newsletter. It has a section for Updates and Security for supported Ubuntu versions.

[http://ubuntuforums.org/showthread.php?t=2273533](http://ubuntuforums.org/showthread.php?t=2273533)

In a few days Ubuntu 15.04 will join the list and in several weeks 14.10 will drop off the list and so will 10.04 server edition. As all the flavours of Ubuntu use the same Ubuntu archives as Ubuntu so all the flavours get these updates as well.

This is why it is recommended that we install a Long Term Support version or be prepared to upgrade to the new release every 7 or 8 months. Or before the version we are using reaches end of life.
