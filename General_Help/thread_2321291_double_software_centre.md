### Post by grahammechanical on 2016-04-21
Upgrades will still have the Ubuntu Software Centre. And the icon for the new Ubuntu Software is the same as the icon for Ubuntu Software Centre. Actually Ubuntu Software is Gnome Software re-branded and as part of the re-branding the icon changed from a light blue coloured shopping bag to the Ubuntu orange coloured bag.
