### Post by EvilCowX on 2011-08-02
For Linux the configfile is created on the first run in ~/.dosbox/ The name is dosbox-version.conf where version is the version number currently installed.

See [http://www.dosbox.com/DOSBoxManual.html#ConfigFile](http://www.dosbox.com/DOSBoxManual.html#ConfigFile) for further info!
