### Post by Brandon Williams on 2009-08-15
If you're running jaunty, you can change the file manager using gconf-editor. Navigate to /desktop/gnome/session/required_components/ (I think that's the path, but I'm on a Hardy system at the moment), and you will find a key that's used to specify the program that is used as the file manager.
