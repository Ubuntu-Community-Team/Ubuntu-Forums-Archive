I was unable to boot from an Ubuntu USB stick on a Winbook TW802 tablet with Windows 8.1. BIOS boot order is OK and fast start is disabled.  I then discovered the need to use a 64-bit version and to add bootia32.efi to the EFI/BOOT folder on the USB stick ([http://infosoda.com/ubuntu-tw700-1/](http://infosoda.com/ubuntu-tw700-1/)).  It does boot now, but when I choose any item 0n the GRUB menu the tablet freezes up.  Any suggestions are most welcome.
