### Post by fragos on 2007-05-30
I don't have any problems and I have three USB cursor control devices connected at the same time -- laser mouse, standalone touchpad and Wacom tablet.  I switch between them the cursor always defaults to the one I'm using at that instant.  Do you have a KVM mouse cable plugged in?  Why don't you try the USB to PS/2 adaptor that probably came with the USB mouse.  Using these adapters a USB mouse or keyboard will work perfectly well in a PS/2 port.  I used that method when I wanted to use a PS/2 KVM.
