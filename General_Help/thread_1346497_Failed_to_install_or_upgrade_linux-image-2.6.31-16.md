### Post by Sin@Sin-Sacrifice on 2009-12-05
Sounds like the .deb for that image is corrupted. Delete it and try again. ```
sudo rm /var/cache/apt/archives/linux-image-2.6.31-16.52_i386.deb
```

Edit: If you're running x64 then change the i386 part to amd64
