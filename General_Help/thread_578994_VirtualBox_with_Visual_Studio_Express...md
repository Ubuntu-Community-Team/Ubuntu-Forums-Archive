### Post by p_quarles on 2007-10-17
I don't use Visual Studio, but it should work, since it doesn't depend on 3D acceleration or peripheral hardware, which are the main things that don't work with virtualization.

The problem is that unless you give enough RAM to the virtual machine, it's likely to be pretty slow, depending on the complexity of what you're doing with it.

Itunes, by the way, now works in the most recent version of Wine.
