### Post by bkline on 2016-02-17
Mostly for curiosity -- does anyone know why sometimes Software Updater asks for a password and sometimes it doesn't? I'm pretty sure it's not one of those "we've already asked for your password recently" situations, because sometimes the first updates since the last reboot don't require a password.

---

### Post by Dennis N on 2016-02-17
If the package is already installed, you are not asked for authentication. If you install a new package, then you are asked. So any time the update contains at least one new package, you will be asked.
