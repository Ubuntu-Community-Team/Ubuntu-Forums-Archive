### Post by upulr on 2013-05-18
Since its a touchpad, seems you're using a laptop. See if there is a Function key combo that toggles touchpad to on and off. Try it. Some newer laptops show a on screen symbol when these key combos are used.

My own issue was similar though I did not post it here. After upgrading to Ubuntu 13.10, my brightness settings adjustments cant be done through the System settings or the function keys. The funny thing is that whilst System Settings do not change brightness even slightly, the Function Key combo does reduce the brightness level one step down.

I used the help info on the Ubuntu manual to submit a bug report.
