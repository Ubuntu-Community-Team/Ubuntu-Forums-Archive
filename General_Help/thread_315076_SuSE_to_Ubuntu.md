### Post by ajgreeny on 2006-12-08
Have alook at this page.  It has a great many different links to information pages on the various versions of Ubuntu and will help a lot.

[http://doc.gwos.org/index.php/Main_Page](http://doc.gwos.org/index.php/Main_Page)

I'm newish (18 months) to ubuntu and linux in general, but I love it and will never go back to windows if I can avoid it.  The cli is just there to help out sometimes and the forum is the best I've ever come across.  I tried suse 9, but found it very slow, yast in particular was painfully slow compared to apt-get (cli), aptitude (cli or x gui) or synaptic (the best gui available, I think).

All three of these use apt as the back end but have their advantages and disadvantages.  Aptitude remembers all the dependencies installed with a prog, so if you uninstall it all those will go as well.  Apt-get and synaptic don't remember like that so occasionally you will be left with unused dependencies if you uninstall with apt-get or synaptic.  I usually use synaptic for installing almost everything.  It's incredibly fast and easy to use, but it's up to you.
