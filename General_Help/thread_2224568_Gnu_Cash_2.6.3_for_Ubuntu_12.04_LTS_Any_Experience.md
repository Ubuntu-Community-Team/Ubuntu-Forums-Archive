### Post by rbmorse on 2014-05-18
2.6.3 fixes some bugs. The feature set remains largely the same.  If you're not affected by the bugs there's no reason to upgrade. 

You can get 2.6.3 packaged for Ubuntu 14.4 from the getdeb site, but that particular build has a problem wherein it will occasionally hang on shutdown (no data loss).  

The best bet is to stay with 2.6.1. from repository.  But, if you do need to upgrade to 2.6.3 then download the source files from the Gnucash web page and build from those.
