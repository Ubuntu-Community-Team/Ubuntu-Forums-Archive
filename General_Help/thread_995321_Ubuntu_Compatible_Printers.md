### Post by dnairb on 2008-11-27
I use an Epson Stylus Photo R220 (with 6 inividual ink cartridges) which works very well.

One thing to note, however, is that when connecting it via USB, one has to ensure it is not connected via a hub or PCI/USB card; rather it needs to be connected to a mobo-mounted USB port, otherwise it "loses" data when printing large pictures. I don't know if this is the case with other USB printers.

I also use mtink (available via synaptic) to monitor the ink levels.
