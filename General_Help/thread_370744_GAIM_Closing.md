### Post by Praill on 2007-02-26
To load gAIM from terminal you just type 
```
gaim
```

But I would recommend something else. Instead of using gAIM you should try installing and using kopete. It's the KDE version of gAIM and its far superior imho.

Just open up a terminal and type in 
```
sudo apt-get install kopete
```

or you can install it with an apt-get gui
```
sudo synaptic
```
and then search for kopete.

Good Luck.
