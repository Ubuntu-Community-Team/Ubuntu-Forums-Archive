### Post by 3rdalbum on 2011-12-02
Ubuntu runs Linux software, mainly. Macintosh software is not compatible. Windows software can sometimes be made to run on Ubuntu using some software called Wine, but this doesn't always work. It's not "native" anyway, and using Windows software kinda defeats the purpose of using Ubuntu.

Some versions of Microsoft Office and some versions of Adobe CS are supported in Wine. ([http://appdb.winehq.org/](http://appdb.winehq.org/) for more details) Ubuntu comes with some office software already called Libreoffice, which is compatible with most MS Office documents. It's very capable for most people. I'm not sure what parts of the CS suite she uses, but for image editing I get along with Gimp. It's free, easy to install on Ubuntu, and also quite capable for a lot of the things people use Photoshop for.

You could install Ubuntu on your own computer to give it a test run if you like. It's quite easy to install as a trial - just download the CD image, burn it to a CD, insert it into your Windows computer and follow the instructions on screen. Once you've tested to see how it works, you can buy the netbook and remove the test-run of Ubuntu from your own computer.

### Post by Mark Phelps on 2011-12-02
Based on what you've said so far, Linux is NOT going to be a good choice for you.

Why?

Because one of the apps you mentioned, Adobe CS5, does NOT work in Wine. Period.  See the link below for details:

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=8646]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=8646")

In addition, if you are using the latest version of MS Office (2010), that does not work in Wine, either.  Some components of Office 2007 work, but stuff like Access and Publisher do not.

If you're dependent on MS products for daily use, then Linux distros are not a good way to go.

### Post by OrangeCrate on 2011-12-02
Not to pontificate too much here, but, many people look at Linux, specifically Ubuntu in this case, as a drop-in replacement for Windows. Unfortunately, that's not the case, and there is an old article - still quite relevant - that discusses this at length:

[http://linux.oneandoneis2.org/LNW.htm](http://linux.oneandoneis2.org/LNW.htm)

As has been said here, Linux may not be for you. Windows 7 is really, a pretty good operating system, and it is designed to utilize those programs you are most familiar with. Just be sure to keep it updated, and it would really help with overall security, if you use it from a standard user account, rather than from the default administrative account. If you're not familiar with using one, see here for guidelines on setting one up:

[http://www.unixwiz.net/techtips/win7-limited-user.html](http://www.unixwiz.net/techtips/win7-limited-user.html)

Also, asiyu (a moderator here on the forums) discusses this topic here:

[http://www.psychocats.net/ubuntucat/windowssecurity/](http://www.psychocats.net/ubuntucat/windowssecurity/)

(Frankly though, I disagree with him on item #6. When it comes to Windows security, a belt along with the suspenders, is, in my mind, the best strategy.)

Good luck.
