### Post by Forlong on 2008-06-07
If you don't have it already, install ccsm ```
sudo apt-get install compizconfig-settings-manager
```

Then run ```
ccsm
``` and go to **General Options**

There you disable **Hide Skip Taskbar Windows**


P.S. if you like Cairo-Clock, you'll love Screenlets (also in the repos since Hardy).
