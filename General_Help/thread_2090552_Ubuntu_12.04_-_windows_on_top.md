### Post by Aleksej on 2012-12-02
OK, before someone thinks otherwise, it has nothing to do with Microsoft. :guitar: My question is - how do I set Unity on 12.04 so that applications that are running don't go in background when I click somewhere on the side. For example, I'm running Movie Player and a web browser, and I want to set Movie Player to be visible all the time while I'm browsing. 

Thanks!

---

### Post by mcduck on 2012-12-02
Right-click on a window's title bar, and select "Always on top" from the menu.
