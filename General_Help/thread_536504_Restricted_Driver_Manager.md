### Post by sumguy231 on 2007-08-28
Try this:
1.) Open a terminal.
2.) type 'sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.bak' (without quotes of course)
3.) Run 'gksu gedit /etc/X11/xorg.conf'
4.) Find the 'Driver' section and change the part in quotes to fglrx if it isn't already.
5.) Save and close.
6.) Log out and log in again, or hit Ctrl+Alt+Backspace (Warning: the latter will suddenly restart X without any regard to anything)
7.) See if: 1.) X starts and 2.) If you can do hardware accelleration - try running 'glxgears' to test.

If it didn't work because I'm wrong or otherwise, do this from the terminal you get dumped into:
```
sudo cp /etc/X11/xorg.conf.bak /etc/X11/xorg.conf
sudo /etc/init.d/gdm stop
sudo /etc/init.d/gdm start

```
If 'sudo /etc/init.d/gdm stop' takes you to the splash screen, hit Ctrl+Alt+F1 to get back to the terminal.
