### Post by zeroseven0183 on 2009-06-13
Were you given an Install option? If not, right-click to your desktop and select **Change Desktop Background**. From the **Appearance Prefences**, go to **Theme** tab, click **Install...** and locate the file you downloaded.

Then you can Customize it from the button. You'll see it there on the window.
