### Post by 7-webmaster on 2017-01-30
On 16.04 and 16.10 the user login screen does not include the option for a Guest Session even though such a guest session has been defined and is accessible from the control panel once I have logged in to my own account.  Once I have switched to the guest account from my own account using the control panel, a process that takes almost a minute, and then log out the Guest session is displayed as an option on the login screen.  Since I do not know why the Guest session was not visible in the first place, or what the system was doing for that minute during the switch from my account to the Guest account, I do not know if that Guest login option will suddenly disappear at some time in the future.
I believe that there is a gap in the documentation regarding the Guest session.  There is documentation for manually adding a Guest session, but since the default install creates a Guest session, that documentation is low priority.  However the documentation for login to a Guest session does not address the question of how you get the default Guest session to appear on the login screen.
