### Post by Chronon on 2011-01-23
Hi.  :)

See if you can adjust input levels with pavucontrol.  You might also need to adjust levels in alsamixer.

The integrated microphone is usually attached to a separate sound device.  If you don't see it it's possible it's not being recognized.  Are you able to capture sound with it using other applications?  If not, then you can attach an external microphone and use that instead.
