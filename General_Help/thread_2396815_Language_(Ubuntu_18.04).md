### Post by Impavidus on 2018-07-21
How do you want to write those characters?

If you have a portuguese keyboard, it should support the normal portuguese characters using dead keys. Type <dead tilde> a to get ã, for example. An international keyboard layout also comes with a lot of dead keys, sufficient for most European languages. You can also use the compose key, if you configure one: <compose> ~ a gives ã. Most combinations are easy to memorise. Or type the unicode code points directly: ctrl+shift+u e 3 space gives ã. Instead of space, you can also terminate with enter or hold ctrl+shift until you're done. You need the code points in hexadecimal and it works for all of them. Or use one of those utilities for input of special characters, depending on your desktop environment. Select the character with your mouse and copy to your writing application.
