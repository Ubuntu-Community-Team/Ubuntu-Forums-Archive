### Post by Sam on 2010-05-03
Do you want to read your stuff from the harddrive plugged in your PC while using the xbox? In this case, go for EXT4.

If you need to use it with older linux distros, go for EXT3.

If it's an external drive and you plan to plug it on a Windows box sometimes, go for NTFS. But if it's not often, or from your PC, you can use EXT3 as there is a driver for Windows.

If you need the best compatibility, use FAT32, but there is a file size limit of 4GB.
