### Post by drizad on 2008-07-10
Get M-Ping. I managed to install M-Ping after few hours, trying to figure out how to install it.
[http://sourceforge.net/projects/m-ping](http://sourceforge.net/projects/m-ping)

Just download and extract it. Make sure you read the README first before installing.

You will need to have Qt 4.3.x installed before running the installation. Get it from you Synaptic package manager, choose to install [FONT="Georgia"]qt4-dev-tools[/FONT] first.

During the installation process, you may be needed to install [FONT="Georgia"]g++[/FONT], but most probably not. Just make sure you take notice of their error message if any.

After the installation, running M-Ping is a breeze!

Good luck!:)
