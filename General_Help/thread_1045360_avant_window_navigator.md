### Post by jocko on 2009-01-20
Make sure you have the package "awn-manager" installed. Start it from System --> Settings --> Awn manager. In the general preferences, just make sure "automatically start on login" is enabled.

---

### Post by phantom3113 on 2009-01-20
you may also have to add it to the list of programs that begin at startup. To do this, open System>>Preferences>>Sessions and then choose Add. Put in the title of the program, in this case Avant Window Navigator and then the command which should be avant-window-navigator. You can put a comment if you feel like it, but it's not needed.
