### Post by blazemore on 2010-05-19
Dual-CPU motherboards probably will only take enterprise-level chips like Xeons.
Check to see what "socket type" the dual-cpu motherboard has (Such as LGA 775) and see if that socket type will support your Pentium.

### Post by cascade9 on 2010-05-19
> **blazemore said:**
> Dual-CPU motherboards probably will only take enterprise-level chips like Xeons.

Pretty much these days.....

The Celeron 'a' and other variants up to 500mhz can run dual CPU, the later celerons cant. The Pentium 2 and 3 can go dual-CPU, but that was discontinued with the P4- Intel wanted to protect the valuable server market. (SiS actually made dual p4 motherboard, but intel threatened them with a big stick so they never produced it).

There is no way that you will get a P4 dual CPU system going. Well, unless you can find an engineering sample of the motherboard SiS made. 

There were Athlon dual-cpu systems, which used sort-of desktop CPUs. Technically, they were just a variant on the standard Athlon, instead of being 'Athlon XP' they were Athlon MP' (multiprocessor). There was a hack to makea  standard Atlhlon XP into a Athlon MP though.
