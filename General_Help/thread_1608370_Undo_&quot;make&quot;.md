### Post by 3Miro on 2010-10-29
If you installed the software using "make", there usually is "make clean" command that removes any compiled binaries.

If all that you have is the binary .so file, then you cannot unmake it. There is the process of "reverse engineering", however, it is almost as hard as making the program yourself and depending on the type of software that you are dealing with, it may be illegal.
