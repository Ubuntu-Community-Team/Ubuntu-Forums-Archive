### Post by 73ckn797 on 2008-12-22
> **upapilot said:**
> My evolution mail only downloads only recent/unread mails from the server. Is there any way to make it download all the messages on the server.:confused::?:


Open Evolution, go to edit/preferences/mail accounts/, then click edit for the email account in question. Go to "receiving options" and see if "Leave messages on server" has been checked or not.

That is good if you are prone to tinkering and having to reconfigure things. The next install will get all messages left on the server.

I guess that this will address what you desire.
