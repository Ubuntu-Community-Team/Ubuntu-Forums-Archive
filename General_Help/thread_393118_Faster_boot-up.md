### Post by neoaddict on 2007-03-25
[http://leetturtle.com/guides/guide_ubuntugetty.php](http://leetturtle.com/guides/guide_ubuntugetty.php)

> Getty is a virtual terminal in Ubuntu, which allows you to get to a terminal via a keyboard shortcut, typically CTRL+ALT+F1 to CTRL+ALT+F5. Disabling some of these gives you a performance boost, but don't disable them all, because you won't be able to use them if you disable all of them. To get back to your desktop from Getty, press CTRL+ALT+F7.
