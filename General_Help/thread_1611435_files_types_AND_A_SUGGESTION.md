### Post by oldos2er on 2010-11-01
tar.gz and tar.bz2 are compressed archives that could contain any type of file(s); similar to .zip.

deb is a Debian package; double-clicking it should open a package installer.

rpm is a Red Hat (and its derivatives) package. On Ubuntu you can use a program called *alien* to convert it to deb, which doesn't always work successfully.
