### Post by Noah0504 on 2011-01-28
If you have an ATI or nVidia card, you'll have to first make sure you are using the proprietary drivers.  You can see if you have them installed via the hardware driver manager.  Then, go to appearance in the preferences, and turn on the extra effects.

### Post by Habitual on 2011-01-28
Open the GNOME Configuration Editor; press Alt-F2 to open the Run Application dialog and type gconf-editor. Click Run. Navigate to Apps->metacity->general. Check the compositing_manager box, and Metacity will immediately restart with compositing!
