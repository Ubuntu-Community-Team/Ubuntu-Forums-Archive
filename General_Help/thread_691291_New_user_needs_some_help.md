### Post by meg23 on 2008-02-08
You can start the rhythmbox gui by typing* rhythmbox* into the terminal. For other ways of controlling rhythmbox by the terminal, enter *man rhythmbox *or *rhythmbox --help* into the terminal.

### Post by forrestcupp on 2008-02-08
The only problem with running it from the terminal is that if you close your terminal window, it will close whatever you are running.  If you are wanting to learn the command line, I think it's ok to run gui apps from a menu and use the terminal for all other navigating and commands.
