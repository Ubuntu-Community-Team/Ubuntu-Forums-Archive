### Post by anjilslaire on 2008-10-24
You need to resize your partition to leave some empty space to install Vista in. However, since you're new to Linux, its *far* easier to install Windows 1st, then linux. Have your friend split your drive into 2 partitions, Install Vista onto 1, then install Kubuntu on the 2nd. This will allow the Linux bootloader to easily manage both OSes & you can dual boot both.
