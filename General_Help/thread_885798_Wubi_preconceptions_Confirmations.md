### Post by James_the_dude on 2008-08-10
Hello all.

I have loved using linux, except for the one problem which has always plagued me for as long as I have linuxed, and that is getting my hardware to work properly.

As wubi installs ubuntu within windows, I assume that I won't have to deal with any linux drivers at all because wubi would work with the windows drivers. Therefore wubi would be the answer to my problems, all my hardware would work. Am I right in making these assumptions?

---

### Post by AusIV4 on 2008-08-10
No. Wubi installs on the same partition as Windows, but it is a separate boot and separate operating system, so it still cannot use Windows drivers. Windows isn't running when Ubuntu is running, so it can't use the Windows drivers.
