If they're Windows games you're running with WINE then you could move your wine C drive directory to another partition.

### Post by blueridgedog on 2009-06-14
> **recharge330 said:**
> I was wondering if and how I can set where programs are installed or move them afterwards.

In days gone by, perhaps, but not so much today under Linux.  Prior to package managers, you installed most things by hand and compiled the applications one at a time.  Now, you get a binary file and a set of install parameters so that things can be integrated into a menu system uniformly, so the install location generally had to follow the file system specification ([http://en.wikipedia.org/wiki/Filesystem_Hierarchy_Standard](http://en.wikipedia.org/wiki/Filesystem_Hierarchy_Standard))
