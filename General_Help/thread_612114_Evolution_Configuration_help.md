I have a nagging issue with my evolution config.  It's working fine and dandy using the exchange connector, but every email I send.. goes into a local sent folder rather than my exchange sent folder.. this is annoying because if I use webmail(or my blackberry) I can't see any of my sent items that I sent from my evolution client..

### Post by nulall on 2007-11-13
Edit --> Preferences --> Mail Accounts --> Edit --> Defaults --> Sent Messages Folder

Then just set it to whatever folder you want it to go into
