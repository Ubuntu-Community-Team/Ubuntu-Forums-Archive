### Post by atomkarinca on 2008-05-07
TuxGuitar is the one you should go with. And I highly suggest you to install the 1.0-rc4 version. You can find the Ubuntu package [here]("http://www.tuxguitar.com.ar/download.html").

EDIT: If you want better midi support you should install timidity and change the midi port to timidity:

```
sudo apt-get install timidity
```
