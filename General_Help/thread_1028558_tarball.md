### Post by jerome1232 on 2009-01-02
A tar file is just an archive (similiar to a .zip but it's not compressed) So it could be anything in there. Source code, installation binary, the program itself.

What is it your trying to install exactly? Do you have a link to where you downloaded the file.

Generally it's best to install from the repositories or install from a debian package (.deb)

edit--- first step would be to extract the contents of the archive, just right click it and select extract here.
