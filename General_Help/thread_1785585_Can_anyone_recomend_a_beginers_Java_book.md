### Post by lightstream on 2011-06-18
JavaScript and Java are two quite different animals. JavaScript is a scripting language (i.e. interpreted line by line as the code runs) most normally used on web pages to create dynamic effects.

Java on the other hand is a full blown object orientated language, which is compiled to so-called byte code. It will perform better than JavaScript, and is capable of much more. 

Good code design is much more important. Unfortunately I don't really bother with books, preferring online content, so can't recommend anything there.
