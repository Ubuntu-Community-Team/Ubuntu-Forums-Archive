### Post by tamoneya on 2008-05-09
I cant be positive why the ram is reading low but my best bet is an integrated graphics card is sharing the memory and therefore part of your memory is being used as graphics memory.  Either way you dont have enough ram for the normal install but you do have enough to run ubuntu.  The way you can solve this is by using the alternate install CD. This will not have the 256 MB requirement.
