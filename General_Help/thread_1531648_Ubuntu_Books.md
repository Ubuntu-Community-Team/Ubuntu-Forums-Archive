### Post by Zorgoth on 2010-07-15
Depends how savvy you are.  I thought myself pretty savvy before I started to learn Ubuntu, and compared to now I knew practically nothing.  And how do you intend to use Ubuntu?  If you are planning on serious power-use, the Ubuntu Unleashed looks best out of the ones you got; I believe also that the Ubuntu Pocket Guide can be downloaded free, although if you prefer the book format that is understandable.

If you are just looking to be a desktop user, which I would guess you aren't, don't buy a book; you won't need one.  

What applications are you planning to run?  Do you want to do bash scripting, programming, or anything requiring advanced text-editing?

And as a final note, books can certainly be helpful as an introduction, but learning the nitty-gritty details (which for most things you hopefully don't need) is all about experience, trial-and-error, and man pages.  It takes time but it is easily worth it.  I do not think I could get half the work done on a Windows machine that I do with my Ubuntu.

### Post by Zorgoth on 2010-07-16
I am certainly not a web developer so I can't help you specifically with that, but in that case I would certainly go for a substantial book like (it looks like that to me, I haven't read it) Ubuntu Unleashed 2010 or something similar.  Looking at the description of that particular book it looks like it has a lot of what you need.

Being a book about Linux, you should note that "2010" or no it was probably dated the day it came out.  There are a number of negative reviews as well as positive ones; most of the negative ones are about the book not covering 9.10 and 10.04 well.  If you are looking for the latest information, a book is not the place to get it, but for an introduction to Linux and Ubuntu, without necessarily covering all the newest features, it or another book like it will probably be fine.

Gimp has an excellent tutorial/manual if I recall correctly, but I don't know about anything else.
