### Post by Zill on 2013-01-21
jbcohen:  Welcome to the wonderful world of Free and Open Source Software (FOSS). :-)

I had similar background knowledge to you and so can understand that you want to apply your experience to Linux.  However, as you will discover, Linux is a whole new ballgame and the biggest problems seem to come from newbies who try to use Linux in the same way they would use MS Windows.

If you approach Linux with an open mind and are prepared to learn new ways of working then I am sure you will come to see what a great OS it is.  Just read the [extensive Ubuntu documentation]("https://help.ubuntu.com/community") and then ask questions on this forum if you hit any snags - we are a friendly bunch here. ;-)

The following two links give some very useful info to help you get started:
[LIST]
[*][Linux is Not Windows]("http://linux.oneandoneis2.org/LNW.htm")
[*][Installing Software in Ubuntu]("http://www.psychocats.net/ubuntu/installingsoftware")
[/LIST]
