### Post by tgalati4 on 2007-10-23
Use easytag 2.1, embed picture into mp3 file.  Use gtkpod to move file to iPod.  Play song file to see if artwork comes up.  Works for me using a Red Nano 4 GB, Mint Linux (Feisty).
