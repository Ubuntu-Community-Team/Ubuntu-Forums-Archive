### Post by amsterdamharu on 2010-02-19
Looks like ubuntu is installed but might be some problem with hardware. 

You can edit the grub menu entry by pressing e when selecting the option.

Not working on ubuntu right now but the line linux ... has many parameters you could try to add xforcevesa to it to see if that will help (vesa is used on live cd and could fix video card problems). 
Also add acpi=off 
Setting acpi off will not power off the computer, after you choose shut down you have to manually push the button when it says system halted.

There is a better way of doing this but right now I cannot find it. Do you know what kind of video card you have?
