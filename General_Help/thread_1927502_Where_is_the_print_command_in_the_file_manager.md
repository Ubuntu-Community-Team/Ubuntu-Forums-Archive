### Post by mcduck on 2012-02-18
File manager doesn't print files. You need to open a file in a program that can open that type of files, and then print it from there. /(so if you want to print an image, you open it in any program that's made for opening images and print it with that program. If you want to print a web page you open it in a web browser, if you want to print a spreadsheet you open it in spreadsheet application and so on)

I can't see any point in adding a print feature into a file manager, as that would require it to be able to open every possible type of files (and opening files sure isn't what file managers are for). It makes much more sense to just leave the printing to other programs instead.
