### Post by Jekshadow on 2009-05-10
Just treat multiple desktops as multiple desktops instead of one cloning the other.

System -> Preferences -> Display

---

### Post by fuwkej on 2009-05-10
Under Preferences>Display there is an option to extend the screen (So I have the main screen on the external monitor, and the 'right' portion of it is the actual notebook screen) So in essense I have a 8" tall 24" long screen. Works well, thanks
