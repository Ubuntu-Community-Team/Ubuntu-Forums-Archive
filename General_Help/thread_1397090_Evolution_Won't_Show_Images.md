### Post by NFblaze on 2010-02-02
Hey, I have the same problem. I didnt really bother me, but within 2 seconds of reading this thread I discovered in the View Menu there is a selection called "Load Images". You can click this or press Ctrl + I and it will load images.

Im sure there is also a way to make it default to this action in the preferences menus.
