### Post by Untitled_No4 on 2010-05-04
Are you running 32bit Flash? If so try to install the 64bit Flash from ppa:
[https://launchpad.net/~sevenmachines/+archive/flash](https://launchpad.net/~sevenmachines/+archive/flash)

You will have to remove your current Flash installation first.

EDIT:
Check this post for more detailed instructions:
[http://ubuntuforums.org/showpost.php?p=9216920&postcount=9](http://ubuntuforums.org/showpost.php?p=9216920&postcount=9)
