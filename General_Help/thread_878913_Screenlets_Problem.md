### Post by rainwalker on 2008-08-03
Install CCSM (compizconfig-settings-manager) and go to System > Preferences > Advanced Desktop Effects Settings
Enable the Widget Layer plugin
Start the screenlet(s), right click on them, and select "Widget" in the Window dropdown
Then check what the shortcut is in CCSM for showing the Widget Layer; press that and the screenlets will appear over everything else
