### Post by Eric Weir on 2008-04-29
I'm in the process of getting a fresh install of Xubuntu configured. I've installed OpenOffice Writer and Calc and they're showing up with much larger fonts than other applications or the system itself, e.g., in the menu bar, special windows, etc. 

My assumption is that this is something that should be controlled by the system, that any changes I make in OpenOffice itiself will affect only documents.

Any ideas why this is happening, or, more importantly, how I can change it?

### Post by rbmorse on 2008-04-29
Start any OOo application, then:

tools > options > view

Adjust the number in the "scaling" window (change should apply as soon as you click "ok")

On my system that was set to 125% on first run, changing the scaling figure to 100 set things back to about where they ought to be (or close enough that I don't feel the need to fsck around with it anymore).
