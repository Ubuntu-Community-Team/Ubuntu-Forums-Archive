### Post by Elfy on 2008-09-03
There is a bug it seems - [https://bugs.launchpad.net/ubuntu/+source/openoffice.org/+bug/222376](https://bugs.launchpad.net/ubuntu/+source/openoffice.org/+bug/222376)

IF you have any compiz enabled, turn it off and try again. I use a different window manager and have no problem with the wizard size.

If you are not sure if you have compiz enabled - got to Appearances - system >preferences menu - 4th tab and use normal/no effects

### Post by AlberTUX-CRC on 2008-10-05
i have found a workaround disabling the "window decoration" plugin in the "advanced desktop effect settings". you will temporarily lose the title bar and the close/minimize/resize buttons until you activate it again... or you can leave it disabled and bring the window menu using [alt + right click]
