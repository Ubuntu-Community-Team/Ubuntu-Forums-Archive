### Post by Brandel Valico on 2011-02-28
Check the easy stuff first. Are all the coax cables still good and connected tightly? Check the cable box and various other parts all working correctly? If yes then move on to checking that TVtime is set to view the correct video source. 

Often people instantly think that it's software related. But it seems you had it working then it stopped without you having changed any settings. So I'm inclined towards a hardware failure.
