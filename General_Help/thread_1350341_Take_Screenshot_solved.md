### Post by marshmugger on 2009-12-09
I posted a request on this some time back and nothing happened(!)

It would happen that when I used the keyboard shortcut **PrtSc** to take a screenshot, multiple screenshot windows would pop up and my computer hangs. I'm using UNR 9.04 on an Acer Aspire One netbook.

I changed the keyboard shortcut and have had no problems for weeks now:
1. Go to *Preferences* / *Keyboard Shortcuts*.
2. Define a new keyboard shortcut for *Take a screenshot of a window*.

I am using **Alt** + **P**

I hope someone finds this helpful.
