### Post by lorap on 2005-04-06
hi!
do this:
Computer-->System Configuration-->Synaptic Package Manager
once the synaptic's window opens do this
Settings-->Repositories
once you get repositories' window open mark all unchecked checkboxes then press Ok.
after the repositories' window closes press Reload button in synaptic's window.
once the process completes press the Search Button and in the search window write in GRIP,then press Ok button.
once the synaptic finds it out for you,mark it for install and press Apply button.
now you're done :-)
have a nice day!
