### Post by mb_webguy on 2008-08-13
While the links posted above will help give you an idea of what software is available for Linux and how it compares to Windows software, be aware that Ubuntu has many Windows software equivalents pre-installed.

[LIST]
[*]MS Office -> OpenOffice.org
[*]Outlook -> Evolution
[*]Microsoft Office Picture Manager -> F-Spot
[*]Internet Explorer -> Firefox
[*]MSN Messenger (and all other IM clients) -> Pidgin
[*]Adobe Photoshop -> GIMP
[*]Windows Media Player -> Totem (movies) and Rhythmbox (music)
[*]Nero (or other CD/DVD burning software) -> Brasero
[*]WinZip (or other file archive managers) -> FileRoller
[*]Remote Desktop -> Vinagre
[*]Vista Aero desktop effects -> Compiz-Fusion
[/LIST]

Also, Ubuntu comes with many pre-installed programs that don't have a Windows (or at least Microsoft) equivalent.

[LIST]
[*]Transmission BitTorrent client
[*]GIMP Image Editor (similar to Adobe Photoshop)
[*]Ekiga Softphone (internet telephone like Skype)
[*]Sound Juicer (easy CD-ripper)
[*]Tomboy notes (a sort of personal wiki note-taking program)
[/LIST]

All of this and more comes pre-installed with Ubuntu, and more software is available free in the repositories.  For example, I prefer Amarok (the default audio player for Kubuntu) over Rhythmbox, the Deluge BitTorrent client over Transmission, and the VLC media player to Totem.  I could install the Thunderbird email client instead of Evolution.  If I want multimedia editing software, I can install Avidemux, Cinelerra, or KdenLive.  There are several excellent games available, including Battle for Wesnoth, Frets on Fire, Alien Arena, Tremulous, and Warsow.  The Ubuntu repositories include an amazing number of applications, all of which can be installed, free of charge, with a single click.

And if that's still not enough for you, you can use Wine to install Windows programs on Ubuntu.  I have installed Internet Explorer, MS Office 2003, and several games on Ubuntu with Wine.
