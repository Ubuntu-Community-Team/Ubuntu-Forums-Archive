[SIZE=3]I found something here:

[http://keepnote.org/manual/index.shtml#spell_check](http://keepnote.org/manual/index.shtml#spell_check)

It says:

[/SIZE]> **[SIZE=3]Spell checking[/SIZE]**

[SIZE=3] Spell checking is available only on Linux and Mac OS X at this time. [GtkSpell]("http://gtkspell.sourceforge.net/") is required for  this feature to be enabled.  You can install the required dependencies on Debian (or a compatiable platform) using the following command: [/SIZE]

[SIZE=3]apt-get install python-gnome2-extras aspell aspell-en aspell-XX [/SIZE]

[SIZE=3] Where XX is your own specific language (e.g. de for German spell checking).  Once dependencies are installed, you can use the menu option Tools > Spell Check to toggle automatic spell checking on and off. [/SIZE]
