### Post by dev0nian on 2013-04-12
Hello,

I'm pretty new to Ubuntu, and I'm running a 12.10, 32-bit version. I've run into this strange problem wherein everytime I press (Shift+') it prints @ instead of ", and vice-versa for (Shift+2).
Since I'm used to touch typing, this little problem is driving me nuts, since I keep mixing asterisks and double-quotes.
Can somebody please guide me? Thanks.
P.S.: I'm using GNOME, if that matters.

---

### Post by cool110 on 2013-04-12
You have your keyboard layout set to English (UK) instead of English (US).
[https://help.gnome.org/users/gnome-help/3.5/keyboard-layouts.html.en](https://help.gnome.org/users/gnome-help/3.5/keyboard-layouts.html.en)
