---

### Post by MisterGaribaldi on 2011-02-26
Gnome, KDE, and other DEs support several different image formats to use as icons. Generally, I would recommend either SVG (which is really only an option when the image is being initially created as this is basically a vector-based image) or PNG. In both cases, don't forget about transparency!
