### Post by Mark Phelps on 2010-10-25
To get a fuller picture of what's gone wrong, you should select the Vista option and see if you can still boot into it OK.  

If you can, that narrows down the possibilities.

If you can't, you've most likely corrupted your Vista OS filesystem by shrinking it using the Ubuntu installer, and unless you can boot from a CD (and you HAVE a Vista Repair CD handy), you're not going to be able to get back into Vista easily.
