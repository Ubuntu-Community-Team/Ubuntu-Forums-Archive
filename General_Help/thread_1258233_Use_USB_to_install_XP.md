### Post by fluffman86 on 2009-09-04
Unetbootin will install an XP CD to a thumbdrive if you have an ISO.  Just select Disc Image and point it to the ISO of your choice.  

Then reboot your computer, and press a button for boot options (it's f12 on Dells), and select USB.  If your computer won't boot from USB, use the Plop Boot CD (google it).
