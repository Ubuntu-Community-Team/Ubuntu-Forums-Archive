### Post by Diod on 2009-11-30
The "install.sh" script adds an action to the context menu of Thunar that allows you to mount or unmount a selected iso file using the provided "tiso" script. 
The iso's are mounted as /mnt/<filename> (e.g. /mnt/backup.iso). Right clicking on the directory will also allow you to unmount the iso.

This script is based on [http://ubuntuforums.org/showthread.php?t=380602]("http://ubuntuforums.org/showthread.php?t=380602") by Jason Valdron, but with a few changes to make it a little more up-to-date and fix a few bugs.
It depends on gksudo and zenity.

Have you found any bugs or do you know of any improvements? Please post them here.
