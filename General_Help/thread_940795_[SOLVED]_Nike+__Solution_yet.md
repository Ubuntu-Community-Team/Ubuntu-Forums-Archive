### Post by nisidabay on 2008-10-11
Hi there,

I've got Hardy Heron installed on my system. As I needed a special software for my TomTom Go GPS I also installed Virtualbox and obviously Xp running on it. Apart from it I've got also Itunes installed and ten minutes ago I plugged my Ipod Nano and uploaded my runs on the Nike site using Itunes which sync my runs automatically. Later using IE always with VirtualBox I checked that everything was ok.

One caveat. The virtualbox package has to be the one from the [www.virtualbox.org](www.virtualbox.org) site, 'cause the one that came with ubuntu (at least with Gutsy) for some reason didn't recognise the usb ports.

Hope it helps.
