### Post by CatKiller on 2009-05-16
> **Roasted said:**
> I have a png icon right now. And the menu is not accepting it. Looks like you need .svg....

You don't need an SVG. And if you've just taken a PNG and saved it as an SVG you haven't got an SVG anyway; you've got an SVG that contains a PNG picture.

When you're selecting icons in the launcher creator, you select the directory not the file. Having selected the directory you will get a listing of the image files that are in there. Do not expect to see a listing of the image files in the Browse window until you've actually Opened the directory that contains the image that you want.
