### Post by j2fraser on 2007-03-27
Hi,

I want to be able to boot and use K/Ubuntu portably. I am looking for advice from those in the know about the best way to have a portable K/Ubuntu installation that will save settings and files.

I recently followed the [[COLOR="Red"]How-to[/COLOR]]("https://help.ubuntu.com/community/LiveCDPersistence?highlight=%28LiveCD%29") to create a USB thumb drive which will preserve my settings and create a space for me to save files to; however, I am wondering if this is the best way...

These are the advantages and disadvantages that I see:

1. LiveCD w/USB thumb drive for persistence - Using a LiveCD is loud (in my desktop's DVD drive, anyways) and pretty slow. The USB thumb drive seems to save settings just fine, although I have read that using a USB key in this way dramatically reduces its life-span. I don't want my data crapping out on me.

2. LiveUSB - I read that it is possible to install K/Ubuntu onto a USB memory stick and boot from there. Installation seems relatively complicated and, if the LiveCD using a USB thumb drive for persistence approach (as above) shortens the lifespan of a USB memory key, I can only imagine that this approach is an order of magnitude worse.

3. External USB drive - I have contemplated installing K/Ubuntu onto an external USB drive (e.g. Western Digital Passport), but I hear that these drives are generally slow and those that are a little faster (e.g. 7200rpm) are bulky and require an external power adapter -- not exactly the ideal portable computing environment.

I know that it is possible to work LiveCD persistence with a file in the root directory of a hard drive, but the system I want to use it on has MS Windows 2000 Professional installed on one big NTFS partition, which Linux can't (reliably) write to, so that option is out for me. I can't mess around with the integrity of this particular system!

Anyways, there may well be other options out there that I am unaware of, so I am looking for your ideas and feedback. If my understanding, as set out in the foregoing, is wrong, please let me know!

Thanks!
