
The hda5 partition no longer exists. It seems that the file is a bit outdated.

You need to edit /etc/fstab

```
gksudo gedit /etc/fstab
```
and remove the last line, /dev/hda5.  Then, add this one to /etc/fstab.

```
/dev/hda2   /media/hda2   ext3   defaults   0   1
```
Save the change and create a new mount point for it.

```
sudo mkdir /media/hda2
```
Then reboot.  Now, see if everything's there.

```
df -h
```
