### Post by ajgreeny on 2012-08-20
That will depend more on the applications or games you want to run with wine than whether or not wine will run.

Very often the configuration of wine falls back to WinXP which certainly runs OK on 192 MB ram, so I don't see that as a major problem, but if you wanted to run Photoshop or something similar, or a hefty game, I suspect you could be in trouble.

Try it and see!  There's nothing lost.
