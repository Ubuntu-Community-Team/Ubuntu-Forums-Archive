### Post by moreweb on 2006-11-08
Solved. There was some errors in **/etc/pam.d/common-password** and **/etc/pam.d/common-session**:

common-password
```
/lib/security/pa**[COLOR="Red"]e[/COLOR]**_unix.so => /lib/security/pam_unix.so
```

common-session
```
opt**[COLOR="Red"]a[/COLOR]**onal => optional
```

Only one question: **WHY?????** :-k :evil:
