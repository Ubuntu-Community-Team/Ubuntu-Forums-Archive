### Post by JKyleOKC on 2014-08-18
> **tubos2 said:**
> - Hi i have a ubuntu server , is it safe to install packages from other repository's?Sometimes, but not always. Like very old nautical charts sometimes said, "Here there be dragons." Be very sure that you have good backups of everything that you value, before attempting this, and be prepared to reformat your entire drive and reinstall everything in case you meet one of those dragons.
> - When & how do packages like monitorix make it to the default ubuntu repository ?That's up to the folk at Canonical who maintain the repositories. The general rule, though, is that applications do not get added between releases, and releases happen only in April and October of each year. Once a release is made, applications contained in it are updated only for security needs, not for routine feature additions.> - How are packages updated from other repository's ?Again, that's up to the repository maintainer. You may want to search the forums for "PPA" which is the abbreviation for privately maintained repository-like areas. You can add the URL for a PPA to your update manager's settings, and it will be treated the same as an official repository.

Hope this helps a bit!
