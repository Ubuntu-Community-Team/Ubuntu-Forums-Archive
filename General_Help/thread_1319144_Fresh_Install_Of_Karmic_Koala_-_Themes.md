### Post by wrgb2 on 2009-11-08
Click on System > Preference > Appearance and you can select one of the other themes that comes with Ubuntu.  If you want something slicker, go to gnome-look.org -- they have all sorts of cool themes.  To install one, download it to the desktop and drag and drop it into your Theme windows in Appearance.  For other themes for Firefox, see this link : [https://addons.mozilla.org/en-US/firefox/browse/type:2](https://addons.mozilla.org/en-US/firefox/browse/type:2)
