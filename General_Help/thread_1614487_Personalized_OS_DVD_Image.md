### Post by cariboo on 2010-11-06
In all cases, after an Ubuntu version has been released there are only security updates. There aren't newer versions of the software installed. Windows users are familiar with security updates, so just explain that Linux is much quicker at releasing security updates than Windows.

Using Ubuntu, generally the only way to get a newer version of a program is to install a newer release.
