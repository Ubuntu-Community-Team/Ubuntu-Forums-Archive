### Post by mac9416 on 2009-12-08
Hi, robertcoulson,

So you are trying to download a .deb file and install it with Gdebi? Does it say, "A later version is available in a software channel"? That means a later version of the package you are trying to install is available in the repositories. Go to System>Administration>Synaptic, search for the package, and install it from there.

Good luck!
