### Post by amauk on 2010-01-12
In a ternimal (applications > accessories)
type
```
sudo tasksel
```
From here you can select any pre-configured install
(space toggles selections)

"Kubuntu desktop" is KDE
"Xubuntu desktop" is XFCE

As far as I know, Ubuntu doesn't support any of the *box desktops (fluxbox, openbox, etc.)
So if you want these, you'll have to install them from the repos, and configure manually
