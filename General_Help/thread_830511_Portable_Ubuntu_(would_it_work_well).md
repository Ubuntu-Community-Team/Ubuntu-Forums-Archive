### Post by sdennie on 2008-06-15
> **Tpolich said:**
> I have a 16bg flash drive and I was thinking about installing ubuntu on it. I have a few questions before I try it though. 

1) I know my BIOS can boot from removable, is that common or just the newer ones?


It's fairly common for machines built in the last few years.

> 
2) How slow/fast will it run from a flash stick? I would think some were close to hard drive speeds.


The depends on the flash stick.  A rough estimate would be, "Well enough".

> 
3) What happens if I take this ubuntu install from my computer and plug it into a different computer? Say from mine with a NVIDIA card to one with an ATI card.


That probably depends on how deep into the configuration you get.  With Hardy, graphics cards are largely auto-detected so, if you make no system specific changes to your install, it might work on other machines without any problems.

> 
4) I have a 64bit possessor if I were to install the 64bit ubuntu and then plug it into a 32bit computer it would not work, right? If the above statement is true, is there anyway to get it to work or would it be way to much work for the reward?


If that is something you are worried about, just install a 32bit version to your flash disk.  32bit versions work just fine on 64bit hardware.
