### Post by pbrane on 2012-04-18
Assuming you are using Gnome you can create a new terminal profile by going to **File->New Profile...** If you can't see the menu then right click on the terminal window and select **Show Menubar**. Then you should be able to see your new profile and edit it to your taste.
