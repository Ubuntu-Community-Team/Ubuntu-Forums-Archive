### Post by lindsay7 on 2009-05-20
I guess you could install wine and try to run games with it on Ubuntu, but if they worked they would not run as well if you just ran them with windows. I am not a gamer so I am not sure of all of this. If you want to run games under Ubuntu to, my guess is that you would be better of with a dual boot system. After you use Ubuntu for a while you will want to do this anyway as it makes upgrades etc. much easer with a dual boot. You may also want to set up a separate /home directory too as this lets you keep all your personal files and other programs in it and you can upgrade or do clean installs without messing with you /home directory.
