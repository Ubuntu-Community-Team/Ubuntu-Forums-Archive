### Post by Sumo74 on 2010-01-02
> **Hwest said:**
> How can i go about installing a .tar.gz file?

You need to run the tar command in terminal, and then extract the files to a directory, move to that directory and run ./configure, make, sudo make install (The software may be different and not need the ./configure stage). That's usually how I do it.
