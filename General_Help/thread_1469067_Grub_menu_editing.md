### Post by Revolutionary101 on 2010-05-01
Open up Synaptic Package Manager and search for "startupmanager". When you find it, install it. Then after it is done installing, go to System>Administration>StartUp-Manager.

The StartUp-Manager is an easy program that provides a GUI for editing your GRUB configuration.
