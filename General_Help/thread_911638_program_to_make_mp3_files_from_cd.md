sound juicer will only rip an mp3 to 128kb a sec. i recommend ripper-x for this. also install gstreamer0.10-plugins-ugly-multiverse.

### Post by ad_267 on 2008-09-05
Applications - Sound & Video - Audio CD Extractor and set the output format to mp3 under Edit - Preferences. I think you can change the bitrate if you edit the mp3 profile and change the GStreamer pipeline.
