### Post by deadowl on 2008-12-03
Right click the folder on the drive
Select properties
Select the permissions tab
Next to all three places it says "File Access", select "Read and Write" 
(don't worry about folder access)
Click "Apply Permissions to Enclosed Files"

Voila.
