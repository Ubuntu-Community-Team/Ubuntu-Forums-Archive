### Post by bodhi.zazen on 2010-09-19
If you are intending to run a dedicated server, use the server install.

The standard desktop (Gnome / KDE/ XFCE / etc) are of limited value as most server applications are not graphical and managing a server is essentially installing servers, editing config files, and start/stop services.

If you feel you want a graphical interface for your server, take a look at the various web interfaces, such as webmin.
