### Post by pqwoerituytrueiwoq on 2012-12-11
[http://archive.canonical.com/ubuntu/pool/partner/a/adobe-flashplugin/](http://archive.canonical.com/ubuntu/pool/partner/a/adobe-flashplugin/)

you want [FONT=Courier New]adobe-flash-properties-gtk[/FONT] and [FONT=Courier New]adobe-flashplugin[/FONT]
 
if you enable canonical partners in your software sources you can get them with ```
sudo apt-get update;sudo apt-get install adobe-flashplugin adobe-flash-properties-gtk
```
