### Post by NoVista on 2008-02-23
How I fixed this:

I brought up the search tool. I hit enter, (searched for all files).
I found the file I was trying to copy, and deleted it.

Now I can go to Computer, open the drive, and the contents read fine.

Surprised no one else has had this happen.

Problem Solved !
