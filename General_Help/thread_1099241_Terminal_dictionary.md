### Post by lykwydchykyn on 2009-03-18
I installed the dictd server, and several dictionary modules.  Then installed the dict client and configured it to hit localhost.  It uses very few resources, and it's as easy as:
```

dict someword

```

If you're not sure how to install that, just ask.
