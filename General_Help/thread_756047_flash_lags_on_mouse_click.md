### Post by forrestcupp on 2008-04-15
If you are using Gutsy, are you using the version of Flash in the pre-released updates?  If not go to System->Administration->Software Sources and in the Updates tab, check the boxes next to Pre-released updates and Unsupported updates.  Then open a terminal and type:
```
sudo apt-get update
sudo apt-get upgrade
```
That should get you a newer version of Flash that may or may not help your problem.

You will have to restart your web browser before it takes effect, though.
