### Post by SirBismuth on 2010-02-23
Only time I've had mail sit in the Outbox for any period of time, is when the mail server is not accepting or there are network issues.

I would also suggest going to Edit/Preferences, and checking your settings under **"Mail Accounts"** and **"Network Preferences"**, and see if any settings there are preventing the mail from being sent.  I don't ***think*** you need to specify a port if you are using the default ports.
