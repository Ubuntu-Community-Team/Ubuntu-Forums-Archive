### Post by avanhel on 2008-08-16

The screen that you describe sounds a bit like the termninal window. 
try pressing ctrl+alt+f7 that should put you back into the desktop.
Otherwise type "sudo shutdown -r now" that will restart the computer and allow you to choose vista and reinstall wubi if needed
