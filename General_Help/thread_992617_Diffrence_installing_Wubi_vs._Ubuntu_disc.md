Wubi will work. Basically, it install Ubuntu as sort of like a Windows application. It actually installs Ubuntu on the same partition as Windows (although I think you can select where you want it to be installed, correct me if I'm wrong). The only detriment to that is that Wubi uses NTFS (NT Filesystem, same thing Windows uses) which does make it run a little slower than if you installed it natively. 

Personally, I wouldn't bother with it. Dual booting is easy enough. Check out these guides depending on what you have:

(Dual booting Ubuntu with Vista installed first): [http://apcmag.com/how_to_dualboot_vista_with_linux_vista_installed_first.htm](http://apcmag.com/how_to_dualboot_vista_with_linux_vista_installed_first.htm)

(Dual booting Ubuntu with XP installed first):
[http://apcmag.com/how_to_dual_boot_windows_xp_and_linux_xp_installed_first.htm](http://apcmag.com/how_to_dual_boot_windows_xp_and_linux_xp_installed_first.htm)

The tutorials both use Ubuntu 8.04, but I'm pretty sure it'd be mostly the same with 8.10. If there's any trouble, just post back here and I or someone else will be able to help you out. :)
