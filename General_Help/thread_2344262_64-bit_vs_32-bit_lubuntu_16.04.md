### Post by 1clue on 2016-11-23
For most people it's like everyone here says. The 64-bit builds are more thoroughly tested these days due to most recent hardware being 64-bit. And the 32-bit memory limit is really 4g but considering that a lot of hardware is mapped to or uses system memory, you generally have less than 4g for applications.

There is a mechanism called PAE which allows a 32-bit cpu to use larger memory sizes, but IMO it's a hack. The more memory you actually have in the system the more you want a 64-bit operating system.

If this were my system I'd reinstall with 64-bit, but as people said it will run fine as-is for a normal user.
