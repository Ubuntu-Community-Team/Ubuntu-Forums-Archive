### Post by kestrel1 on 2009-02-11
I ran a partial upgrade, but it managed to screw the system up so I ended up using the CD for 8.10 & installing from scratch.
Not keen on the upgrade route anyway. With Windows upgrades I have always found it better to do a complete fresh install. You always end up with stuff left over from the previous version.

### Post by drs305 on 2009-02-11
You would not be prompted to do a partitial upgrade to a new version. - that is, a 'partial upgrade' isn't going to try to do a dist-upgrade from 8.04 to 8.10. This concerns upgrades to newer packages in the same version you are currently running.  ubuntu is not going to leave you with half-hardy and half-intrepid.
