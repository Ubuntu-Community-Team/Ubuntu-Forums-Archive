### Post by snowpine on 2013-05-09
You can reinstall any important packages you might have accidentally removed with:

```
sudo apt-get update
sudo apt-get install ubuntu-desktop
```

If there are any errors, post them here. :)

(note: if you use kubuntu/xubuntu/lubuntu then modify the command above accordingly)
