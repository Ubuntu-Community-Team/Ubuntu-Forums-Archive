### Post by jerome1232 on 2011-08-04
Are you asking whether you should try Ubuntu or Fedora, or asking for installation instructions for Ubuntu or Fedora, or both?

Well if you wish to try Ubuntu you can download an iso from here [http://www.ubuntu.com/download/ubuntu/download](http://www.ubuntu.com/download/ubuntu/download)

Follow the instructions provided on the site to try/install it. I recomend feeling around from the live cd for a bit to get the hang of it, if you are uncomfortable with partitioning up your drive install from within Windows (aka Wubi) to give it a test run. If you like it Ubuntu, you can create a true dual boot later.
