When you see a flash element on a page, firefox should say something like, "Install missing plugin."  When you click on that, a window should pop up eventually allowing you to choose between Adobe Flash and Gnash. 

I installed Adobe, and it worked fine.  After you select it and install, Firefox should handle the installation automatically.  For me, Adobe's Flash hasn't crashed yet.  Hope this helps, cheers.

### Post by karth on 2007-10-23
Go to System menu > Administration > Synaptic

Search for gnash and mark the packages for removal
then search for flash in the restricted tree (you have to activate the restricted repo in System > Administration > Software Sources) and mark it for install

Click ok, enter your password and go.
