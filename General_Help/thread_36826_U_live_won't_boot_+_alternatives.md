### Post by blackworx on 2005-05-25
Disc needs to be bootable.  If you burn the tree from the ISO image this won't be the case.  What to do is use your burning software to burn the ISO image direct.  In Nero, this involves starting the Nero Burning ROM main program (usually Program Files\Ahead\Nero\nero.exe) and when you get the "New Compilation" window, click the Open button at the bottom right, then point Nero at the ISO file then hit Burn.

Dunno about other progs, but it should be similar (I used to use Easy CD Creator and the process was almost identical).  Basically, find a way of giving your burning prog the ISO file direct for burning without any intervention (like unpacking directory trees etc.) from you.

Hope that helps.  :grin:
