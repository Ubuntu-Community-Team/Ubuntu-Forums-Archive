### Post by perpetualcacophany on 2009-03-06
You can download it at [The Ubuntu Download Page](http://www.ubuntu.com/getubuntu/download). You will have the choice between Ubuntu 8.10 or Ubuntu 8.04 LTS. The newest version is, of course, 8.10, but many people will tell you to run 8.04 because the LTS (long-term support) releases are more stable. It's really a personal choice. 

If you are worried about the system requirements being a problem, you could choose the less resource heavy [Xubuntu](http://www.xubuntu.org/get) version. Actually, looking at your setup, I might recommend Xubuntu because of its lesser requirements. 

You can find out more info about each one from their websites: [Ubuntu](www.ubuntu.com) or [Xubuntu](www.xubuntu.org) 

Once you get the ISO (about 700mb), I would recommend following the instructions you find in [The Ubuntu Pocket Guide](http://www.ubuntupocketguide.com/index2.html) (a great guide in general if you want to know more about Linux and Ubuntu). Not only does it have detailed instructions for various types of installations, it also has a lot of tips which will help you out as a beginner trying to get accustomed to everything.

By the way, using Ubuntu as a student myself has been just fine. There are very few classes that will require anything that Ubuntu can't do. I would be tempted to say that Ubuntu can do most things better actually. Ubuntu comes with all of the productivity software you will ever need, all of the security you will need, and all of the stability you need. After an initial learning/unlearning period, you will be just fine for school use.

The LiveCD will allow you to boot into the operating system without actually installing it. Most of the popular linux distributions will have a LiveCD so that you can try it out. While running the LiveCD there is an option to install the operating system if you decide you like it.

Debian is one of the most famous Linux distributions out there. Ubuntu is based on Debian, and [so are a ton of others](http://en.wikipedia.org/wiki/List_of_Linux_distributions#Debian-based). From what I can find right now Debian has higher system requirements than Xubuntu does right now. That means that Xubuntu would probably run better on your computer than Debian would (unless you use Xfce with Debian, but I'm not sure you would want to get into all that yet).

Debian also adheres more strictly to the open source software ideals and therefore can take a little more work to get running initially depending on all of your different hardware (someone correct me if I'm wrong, I'm basing this on my small amount of research).
