### Post by CatKiller on 2019-11-12
> **scott_2 said:**
> If I create a Kubuntu boot disk and start up my pc, will i have the option of installing it over Mint?  

Probably not directly. The installer defaults to playing nice with other operating systems, so it will try to shrink everything down to fit in alongside. You can do it manually with the Something Else option, though. 

>  Also, can this screw up being able to boot up my pc? 

Not deliberately. Things can sometimes go wrong, and Windows goes out of its way to make things difficult. Make sure your backups are good before you start and check that Windows' Fast Startup is still disabled.
