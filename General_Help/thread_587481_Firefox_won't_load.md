### Post by JBAlaska on 2007-10-23
I believe your trouble lies in either a corrupt or conflicting Synaptics TouchPad Driver or config tool, unfortunately I do not have a touchpad to test here.

First do a **sudo apt-get update** then **sudo apt-get upgrade** if you haven't for a while, this just may cure it. If your all up to date,

You can first open synaptic package manager and put **Synaptics TouchPad Driver** in the search, then completely ununstall and reinstall, **xserver-xorg-input-synaptics** and any other installed synaptics touchpad utility's the above search shows.

If that does not allow you to start FF and other programs your having trouble with, you might try commenting out the touchpad section in xorg.conf just to see if that allows you to start FF, that way we'll be sure were on the right track. CAUTION doing this "may" stop x from starting, so make a backup of xorg.conf that you can restore with the CLI.

**If someone else uses this touchpad and knows a fix for this issue, please post here**
