### Post by Gizenshya on 2009-01-29
are *.tgz files and *.tar.gz files the same thing?  If I change the extension of a *tar.gz file to *.tgz, would it still work correctly?

---

### Post by jerome1232 on 2009-01-29
You can remove the extension all together and it'll work properly, at least from the command line. 

For the most part linux uses alternate means of determining file type, extensions are mostly there to make it easy on people to figure out what type of file it is.

But yes .tgz and .tar.gz represent the same file type.

### Post by jms1989 on 2009-01-29
Yep, even nautilus will reconise it without the filename but it'll expect a file extension to open it with the right app. For example, you could give a jpeg the pdf extension but it'll still be a jpeg. :) Linux is do cool. :)
