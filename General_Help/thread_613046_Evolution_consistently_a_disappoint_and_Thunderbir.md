### Post by wyth on 2007-11-20
There's a couple things to consider here:[LIST]
[*]Mozilla is cutting Thunderbird off so it can start its own foundation and get the TLC it needs.  There's a lot of possibilities with Thunderbird, but Mozilla's key project has obviously become Firefox, and they want to set Thunderbird off someplace where it can get more attention than it's getting now.[LIST]
[*]So maybe when that happens, it'll get some more support here.[/LIST] 
[*]Firefox is integrated in part because it uses/can use the Tango icon set.  Thunderbird has that, but it's not fully functional.  In fact, the Tango icons were making Thunderbird crash a while back, and they don't integrate with the Lightning extension.
[*]Evolution will be have Google Calendar integration soon, just like Lightning/Sunbird.  So you'll be able to add a calendar entry in Evolution and it'll show up in your Google Calendar, and vice-versa (without any third-party software making it work)[LIST]
[*] This will be nice when it happens because the Evolution calendar is also integrated into the taskbar Clock.  So you'll be able to set alarms that integrate with the desktop from Evolution or Google.[/LIST] [/LIST]There are some more features in the pipeline for Evolution 2.14, I believe.  I know the Google Calendar integration was a Google Summer of Code deal that I believe is slated for either 2.14 or 2.16.  When Thunderbird gets its own foundation running, we might see some more support in Ubuntu, but Evolution may really catch up/surpass it by that point in terms of feature integration.

I've started using Swiftdove, an optimized FLOSS version by the folks who brought your Swiftweasel, and found that it doesn't have the Tango icon issues Thunderbird did and runs quite nicely (although the Lightning integration still isn't there).  If you're interested in those, Google them; the developer is in the process of moving from Sourceforge to Tuxfamily so he can use an apt repository, and I think Swiftdove is still on Sourceforge, but will soon be in a available via apt if you add the source.

*EDIT* Evolution does need an integrated system tray feature...
