I was having the same problem. I found that if I typed out my stuff, switched off SCIM (CTRL + SPACE), and highlighted the text, it would let me change the font in OO. Kind of convoluted, but the end result is fine, at least if it is an all-Chinese document.

Hope this helps.

Added in edit: After doing the above steps on a test text, when I switched SCIM back on it let me keep my changed font.
