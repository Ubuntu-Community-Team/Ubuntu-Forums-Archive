### Post by dcstar on 2011-11-06
> **greengiant83 said:**
> I am using remote desktop viewer to remote into my windows 7 desktop. Is there a way I can configure it so that when I hit alt-tab it goes to my remote windows 7 machine instead of switching out of remote desktop and back to ubuntu?

If you install the **remmina** package to connect (basically an alternative Remote Desktop) it has a feature to send all keyboard codes to the remote client, ALT-TAB works fine with this.:

[https://launchpad.net/~llyzs/+archive/ppa](https://launchpad.net/~llyzs/+archive/ppa)

Also install the relevant **freerdp** packages.
