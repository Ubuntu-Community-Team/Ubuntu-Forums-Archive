### Post by Mark Phelps on 2013-03-05
Hey -- I understand where you're coming from -- really.  I started out years ago with Ubuntu and when I finally upgraded to 8.04, I was amazed!  That worked so well that I set about converting other machines to Ubuntu (from Windows XP) -- including a Tablet PC that came with XP pre-installed.

And then, one, day, an upgrade to a newer version "broke" stuff that I really needed to work. So, I restored 8.04.  And, every time a new release came out, I tried that -- with the same dismal result.  Eventually, I got tired of using apps and an OS that were years out of date and erased Ubuntu and installed a current Windows OS -- and everything worked -- just like it did back with Ubuntu 8.04.

You MAY be able to use PPAs, or compile from source, to overcome some of the difficulties of using an Ubuntu version that has reached end-of-life -- but eventually, the past will catch up with you -- and you will end up being stuck in a time warp with outdated, obsolete stuff.
