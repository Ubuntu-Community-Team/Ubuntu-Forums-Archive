### Post by forestpixie on 2008-04-04
You have to do it with either the partition editor on the livecd or use a livecd version of a partition editor.

You can only work on a partition when it's not mounted.

I use parted magic - [http://partedmagic.com/wiki/PartedMagic.php](http://partedmagic.com/wiki/PartedMagic.php)
