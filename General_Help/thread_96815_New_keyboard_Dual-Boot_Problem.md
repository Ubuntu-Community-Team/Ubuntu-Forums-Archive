### Post by Dkz0o7 on 2005-11-29
I bought a new keyboard (Microsoft Comfort Curve Keyboard 2000). When I startup my computer, and it shows the select Operating System, I can not press up or down on the keyboard. My PC automatically starts up into Linux Ubuntu.

I will try to get a usb/ps2 adapter. I have two, but they don't work, and both are mouse ps2 adapters I think (both have mouse symbols on them).

I've had two keyboards for this PC before, and the usb one did the same thing at startup (without linux dual-boot). My other ps2 keyboard worked fine, so that was the one I've been using for the last few months.

[My Mainboard]("http://www.royaldataservices.com/products/mboard/ecsk7sem.html")K7SEM

### Post by Dkz0o7 on 2005-11-29
Problem Solved!

I restarted my PC (K7SEM), and pressed DEL to enter setup. Got in. Ended up in features setup, where I saw USB function for DOS disabled. I enabled it, saved, and restarted. I was then able to select my OS at startup.

Thx for reading, maybe somebody could add this to some kind of guide or FAQ.
