### Post by thomasaaron on 2006-11-19
EXCELLENT.
So I guess it was a resolution problem after all.
The previously mentioned thread had the solution.
Here is what I did. Hopefully it can help someone else...

sudo gedit /boot/grub/menu.lst

Then add vga=791 to the Kernel option so that it looks something like this:

kernel	/boot/vmlinuz-2.6.15-27-386 root=/dev/hda1 ro vga=791 quiet splash

Save and reboot.

Other vga options (for different resolutions) are mentioned in the aforementioned thread.
