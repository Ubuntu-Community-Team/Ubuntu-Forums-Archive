### Post by Ramses de Norre on 2007-01-21
Mostly when that happens here it's because there are more versions of the theme in the package, in that case you have to unpack the package. You'll get a .tar.gz for each theme that's inside the original then and you can install those.
If that's not the case: contact the person who made the theme.
