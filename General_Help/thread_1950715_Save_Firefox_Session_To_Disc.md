### Post by SillySod on 2012-04-01
I've been using the Session Manager in Tab Mix Plus which lets me re-load all the tabs from the last session or the one before that.

Sometimes I want to look at certain things only once every few weeks (for example to do specific month-end things) and the only way I have to retrieve that particular session is to load it every time I start Firefox, then start a new window with what I'm doing today, then close my new window, then close the old window, so that it always appears in the Tab Mix list as my old session. 

It would be handy if I could save a copy of a whole session to my hard drive to be restored at some later point.  I can't see any option in Firefox or Tab Mix Plus for doing this, is it possible?

### Post by SillySod on 2012-06-16
In fact, I have found that "TabMixPlus" can do it: using "Save This Window" in session manager allows you to save all your current tabs with a name which can then be retrieved at any time from the session manager menu.  Right-clicking on a saved session also allows you to replace the session with the current window or delete it.
