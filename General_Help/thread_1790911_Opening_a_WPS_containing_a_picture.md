If you're using Virtual Box 4.0 (latest release) then you can use the install disc and instlall it just like you would regularly. Virtualization is weird to get used to but it's really just like a computer inside your computer. If you use the wizard to create the vm first you just go though the wizard through to the end. Make sure you give it enough resources/ the right resources to run properly in there. Then select that vm from the list in the main window and click the start button. You'll be given another wizard talking about being the first time you've run the vm, blah blah blah - and that wizard is where you can install Windows from the disc (or an iso if you have one and prefer that).

Lemme know if you run into issues and need more detail. You prob will need to register it again (yes, even the vm of it).
