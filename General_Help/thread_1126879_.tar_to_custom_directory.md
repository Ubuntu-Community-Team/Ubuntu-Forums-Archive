### Post by maheshasolkar on 2009-04-15
May be. If there is a file called *configure* in the directory, you probably are. If there is such a file, go into that directory and try:

```
  ./configure --help

```

It will tell you about a bunch of options it supports. If you want to install the application to a custom directory, you'll probably use the --prefix option, like so:

```
  ./configure --prefix=/opt/apps/myapp
  make
  make install
```

That should compile the source code and install it in /opt/apps/myapp, or the one that you choose.
