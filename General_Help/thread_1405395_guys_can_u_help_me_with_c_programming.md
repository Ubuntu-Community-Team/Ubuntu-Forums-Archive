### Post by kedah160 on 2010-02-12
The first source file is to be  named displayargs.c, and is to #include a header file named do_display.h. It is to contain a definition of the method *main()*,  which is to receive the standard three arguments. *main()* is to  pass the first two arguments to a function named *do_display()*,  and then it is to print each of the environment variables, one per line.
 The second source file is to be named do_display.c. It must also #include "do_display.h", and is to contain the definition of *do_display()*, which  is to print each of the command line arguments in **reverse order **before it returns.
 For example: 
  **$ **gcc -g  -Wall displayargs.c do_print.c -o displayargs
  **$ **./displayargs  you? are How there. Hello,
  **The command  line arguments are: Hello, there. How are you?**
