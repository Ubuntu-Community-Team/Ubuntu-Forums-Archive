### Post by WorMzy on 2011-04-21
As of the current version (10.10), the Netbook version uses Unity interface, and the Desktop version uses the GNOME 2 interface. I'd expect the Netbook version to come with less tools by default, since Netbooks are primarily designed for web browsing. Someone else may be able to rustle up an actual comparison of the two, to show you the differences side-by-side.

### Post by ajgreeny on 2011-04-21
As of 11.04 there will apparently be no separate versions, but for 10.04 and 10.10 the differences are only minor.  The UNE version does not include compiz or brasero, though both can be added if wanted.  A netbook does not normally have a CD drive, so no brasero makes some sense, but if yours has a CD just add brasero if you need it.

I looked at 10.10 UNE, but hated unity and found it very slow, so just used the classic gnome desktop, which is installed by default as well as unity on UNE; you just choose it at the login screen, after clicking on username but before password, in the session menu bottom right.

There are probably other differences between the two, but I think those are the main ones;  it really is just the DE that alters, everything else is just the same, and all the repositories are available for all, so no worries about not being able to install whatever you want.
