### Post by aysiu on 2008-09-12
The installed Ubuntu should have the same hardware detection as the live CD.

You mentioned your computer was 900 MHz but didn't say anything about the RAM. If you have 512 MB of RAM or more, the live CD should be usable and even fast. If you have less than 256 MB of RAM, you may not be able to run the live CD without it freezing up. If you have less than 192 MB of RAM, I don't think you'll be able to run the live CD at all.

### Post by fiddler616 on 2008-09-12
In case you were wondering about *how* to use the Live CD:
[LIST=1]
[*]Put the CD in your drive, turn off computer
[*]Boot up computer, enter BIOS (aka "Setup")
[*]Go to the boot order tab, and make sure that your CD drive is above the hard drive
[*]Exit BIOS
[*]An Ubuntu menu will load, with stuff like "Install Ubuntu", "Try Ubuntu" "Check CD for defects", etc.
[*]Use the keyboard to select "Try Ubuntu" (if all you see is a bunch of lanugages, hit Esc)
[*]Ubuntu will load off the Live CD. Enjoy!
[/LIST]
