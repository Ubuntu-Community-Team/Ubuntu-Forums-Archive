### Post by lovinglinux on 2009-07-02
That is a common issue, described in the troubleshooting section of the [Firefox optimization and troubleshooting thread](http://ubuntuforums.org/showthread.php?t=1193567). 

Basically you need to close Firefox, delete the file *locastore.rdf* from your Firefox profile and start Firefox. The link above provides all info you need to perform this procedure.
