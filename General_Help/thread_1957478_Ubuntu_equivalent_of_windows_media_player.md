### Post by LewisTM on 2012-04-12
The Linux philosophy is to provide single tools for single purposes, for simplicity and efficiency. You'll have a hard time finding an app that does everything.

KDE seems to be more in tune with the MS Windows multipurpose philosophy. In that respect Amarok, KDE's music manager, should do everything you need except watch videos.

It's not as clear or intuitive as Windows Media Player but it functions well.
