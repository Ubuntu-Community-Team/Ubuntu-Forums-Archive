### Post by Rob_H on 2009-12-18
If all you want to do is remote desktop sharing, there are several choices. The most flexible one is VNC. It works equally well with either Windows or Linux as the client or server and enables you and the other person to interact with the desktop. There are several VNC implementations available. Just Google and you will find some.
