### Post by nzjethro on 2011-07-03
Yes, check out [PanelSwitcher]("https://wiki.ubuntu.com/PanelSwitcher"). It lets you save and import panel layout settings. Thus, get to your desired setting, and save it! :D

Thanks! :)

I've managed to install it following this thread and it worked for me (I'm on Ubuntu 10.04 Desktop Gnome):
[http://ubuntuforums.org/showthread.php?t=892149](http://ubuntuforums.org/showthread.php?t=892149)

I've also found that all panel icons can be found on the folder:
~/user/.gnome2/panel2.d/default/launchers
What I found great:) is that by opening this folder I will find all my panel icons listed alphabetically, and as I use "compact view" on nautilus, I can now create dozens of shortcut icons that point to my favourite locations or programs! :)
Much better than cluttering my Desktop, as I don't like my icons there.
