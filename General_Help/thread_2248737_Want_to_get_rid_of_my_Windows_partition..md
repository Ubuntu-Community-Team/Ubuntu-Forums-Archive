### Post by kc1di on 2014-10-16
If you installed ubuntu using Wubi then you will delete it along with windows if you delete the windows partition wubi installs ubuntu inside a folder on the windows partition.
if you want to get rid of windows the best way would be to transfer all important files from your ubuntu install to an external media. and any documents or files you want to save from windows.  then do a complete re-install of Ubuntu (not wubi) and let it take over the whole disk.  
good luck ;)
