### Post by KeLa on 2010-10-25
Open terminal and write
whereis gnome
and if you get answer like this:
gnome: /usr/gnome /usr/share/gnome
then you got gnome installed.

Gnome is user interface (window decorations etc...)
Other popular you can use are kde, xfc and so on.
