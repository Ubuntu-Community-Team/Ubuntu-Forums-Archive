### Post by Paqman on 2010-01-03
> **djmh said:**
> is this an ok way to get that sudo aptitude install kubuntu-desktop ?


No. Installing kubuntu-desktop will add all the Kubuntu packages to your current system.

> 
or should i install on a seperate partiton ?

If you want them totally separate, yes. If you have /home on a separate partition then you could share it between the two, just create separate user accounts on each system to prevent the config files getting messed up.
