### Post by didac on 2008-06-02
Mozilla settings are found in your home directory, in a folder called .mozilla 
Notice the 'dot' at the beginning. It means is a hidden folder. To see it in Nautilus go to View => Show Hidden Files. 

Don't copy the Windows folder on the Linux folder. Go file by file. First duplicate the Linux file (copy and paste will do) and substitute it for the Windows file with the same name.

I don't have experience with Thunderbird. Most likely there is another folder inside .mozilla

If you only want to transfer Firefox bookmarks, save them in Windows to a file, and import them in Firefox. Bookmarks => Organise Bookmarks => Import

Regarding media files: what is the problem? you also have Windows Media Player playlists, or is only individual files?
