### Post by zeroseven0183 on 2010-02-02
How about you add your user to the **audio** group? Try to create another user and see if the login sound works.

See [Comprehensive Sound Problem Solutions Guide]("http://ubuntuforums.org/showthread.php?t=205449") thread for more technique

The user from [this thread]("http://ubuntuforums.org/showthread.php?t=421482") deleted the **.asoundrc** from the Home folder and was able to resolve the issue.
