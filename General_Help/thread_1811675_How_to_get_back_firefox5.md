Hi and welcome 

You can remove the Aurora build from the Synaptic Package Manager found on the administration menu or from dash if you're using unity .  Search for it and mark it for complete removal and select apply , then  go in Settings> Repositories > Other software  and remove the source . 

Try this PPA to install Firefox 5 stable .  Sorry , the code tags don't seem to be working for me .

sudo add-apt-repository ppa:mozillateam/firefox-stable

sudo apt-get update

sudo apt-get upgrade
