### Post by the ev on 2007-02-18
.EXE are Windows programs, which you can't run natively in Linux.

There are several things you can do to try and get that program running.

1) Setup Wine, which is a Windows API.  You can do this easily by downloading and installing [Automatix2]("http://www.getautomatix.com/").  Then, you can right click on the file and select "Open with Wine".

2) Setup a virtual machine using a program like VMWare.  This will emulate and entire Windows installation, rather than just the API.  You'll need a full Windows CD to build the virtual machine.  Once that's setup, you'll have a window with a full Windows installation inside it.  Then you can download and install the file as you would on a normal Windows machine.  For more complete instructions on how to do this, check [this]("http://ubuntuforums.org/showthread.php?t=183209") out.

3) Try to find a native Linux app that is an alternative to the one you're trying to use.  A good place to look is [Open Source Alternatives]("http://www.osalt.com").

[Here are some other things you can try as well]("http://lifehacker.com/software/linux/5-ways-to-run-windows-software-on-a-linux-system-234643.php").

Hope that helps!
