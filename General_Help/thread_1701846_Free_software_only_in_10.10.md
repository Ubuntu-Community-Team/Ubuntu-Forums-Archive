### Post by grahammechanical on 2011-03-07
Some people for ethical reasons only want to use open source software. So, Ubuntu is only Open Source until we, the users chose to install propriety software such as graphic card drivers and video codecs.

The Ubuntu software Centre will only offer software that is being provided with the copy right owners permission whether it is without cost or not. Such as the Adobe Flash plug in. It is our choice to use or not. I think we are still asked if we want the install process to install non-open source software. If we choose yes, then the installer will fetch many of the drivers and codecs that we need.
