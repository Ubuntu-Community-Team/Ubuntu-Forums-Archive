### Post by CA_Warren on 2009-05-20
This has bugged me for a while, too, so I hunted it down (doesn't help when developing websites):

This means that you've 'selected' the tab using the Tab Mix Plus extension. More details [here]("http://download.cnet.com/8301-2007_4-9732587-12.html"), but basically if you've selected multiple tabs to merge, the 'selected' tabs are marked by that asterisk.

You can deselect them the same way you selected them, which is by (accidentally?) holding down CTRL and left-clicking the tabs in question.


Hope that helps.
