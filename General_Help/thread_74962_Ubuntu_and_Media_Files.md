### Post by Wolki on 2005-10-13
[QUOTE=cparsons]Does anyone know what type of media files Ubuntu actually plays?[/quote]

Every file type that if free to include, meaning not covered by patents that would make the ubuntu install cd illegal in many countries.

That's why those codecs have to be on servers that are not located in such countries. There has been a lot of discussion on this subject. 

Some explanations and discussion here: [http://ubuntuforums.org/showthread.php?t=70646](http://ubuntuforums.org/showthread.php?t=70646)

> i.e it doesn't play mp3 or .wav so what does it play?

It should play .wav i think. Ogg Vorbis and FLAC of course too.

> and if so, is there a programme for windows or linux that will convert mp3 to the relevant file type.

Yes, but since that would violate the patents, it can't be included by default either. Don't know about windows software, but I'm 100% sure that there is software that can do that.

> what is this "ogg" file extension i keep seing???

Ogg Vorbis is a music format that is completely free in every sense of the word, and (at least) not worse than mp3. Here's some info: [http://en.wikipedia.org/wiki/Vorbis](http://en.wikipedia.org/wiki/Vorbis)
