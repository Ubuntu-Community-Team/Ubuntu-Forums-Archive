### Post by Rui Pais on 2006-06-12
hi,
slow internet connections are most of the time caused by
DNS problems ([check this]("http://ubuntuforums.org/showthread.php?t=168056&highlight=DNS"))
ipv6 problems ([check this]("http://ubuntuforums.org/showthread.php?t=181171&highlight=ipv6"))

If it's a specific Firefox problem maybe it's a
memory leak (check [this ]("http://tech.cybernetnews.com/2006/04/04/reducing-your-memory-usage-in-firefox/")and [this one]("http://tech.cybernetnews.com/2006/03/26/this-may-help-your-firefox-memory-leak/"))

Or, if none of those helps maybe trying with [swiftfox]("http://ubuntuforums.org/showthread.php?t=142798").
That will allow to check if another build of firefox gives the same results and, ususally, switfox is faster then the normal firefox build (either the ubuntu version as the plain mozilla site version)
