### Post by xdominex on 2011-08-06
Active support for software is a huge advantage. The Ubuntu developers only support LTS releases past their 6-month period of being the newest distro. 10.04 is the current LTS, though, not 10.10, and 11.04 still has much newer software than 10.04, even.

Another advantage is access to the new Unity interface. I personally prefer Gnome 2, but I think Unity has *potential*. I also know for a fact that Unity will be much better in the 11.10 release, so it's nice to be able to get used to Unity and start using it now. One significant advantage of Unity is the screen real estate. The title bar AND menu bar of (most) applications is integrated into the Unity panel at the top of the screen, giving you MUCH more space to work with in Firefox, Gedit, LibreOffice (though you have to install lo-menubar package for it this to work with LibreOffice), etc. If you hate Unity, the classic Ubuntu desktop is still available as an option at the login screen; if your initial impression of Unity is "nice" or better, you should read this...
[http://askubuntu.com/questions/36274/tips-and-tricks-for-unity](http://askubuntu.com/questions/36274/tips-and-tricks-for-unity)
...and install Confity (just google it).

Those are the main advantages, and though they number only two, they are big ones. I hope this has helped you make your decision.

### Post by dcstar on 2011-08-06
> **xdominex said:**
> Active support for software is a huge advantage. **The Ubuntu developers only support LTS releases past their 6-month period of being the newest distro**.


Absolute rubbish.

**All** Ubuntu releases are "supported" until their end of life dates - and "supported" means keeping the versions of software released fixed with security updates, it does **not** mean (and has never meant) upgrading to new versions.

The reality is that the **new** releases contain new versions of programs that invariably have faults (which people call "bugs") that get resolved because they are **new**, not because of any time restriction.

Once any new release gets past a certain time most of the bugs are usually solved.
