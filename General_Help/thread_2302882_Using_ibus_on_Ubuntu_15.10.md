### Post by grahammechanical on 2015-11-14
The Keyboard input method system on my install of Xenial Xerus, which is only a few days out from being Ubuntu 15.10 has ibus as the default. But then I did choose English as the default language. Ubuntu software Centre will install Fcitx for me. So, I suggest searching Ubuntu Software Centre for dbus and installing it from there. And then opening System Settings>Language Support and see if dbus is in the drop down menu of the Keyboard input method system panel.

Regards.
