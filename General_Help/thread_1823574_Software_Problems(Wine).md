### Post by grahammechanical on 2011-08-12
Can we just clarify a few things? You installed wine - yes? And then you installed these two programs through wine -? Did you install them as if you were installing them in Windows? You need to explain what you mean by nothing happened.

When wine is installed you get a utility called Uninstall Wine Software. When you run that you see a button called Install. When you click on the Install button you get a Windows type file browser. From there you can browse to the folder or disk that holds these programs. When you click on setup.exe or install.exe or something like that the Windows program will install just like an installation on Windows itself.

If you installed these two programs this way then there should be some evidence that they are installed. What version of Ubuntu are you using? 

To find out if these programs work under wine go to

[http://www.winehq.org/]("http://www.winehq.org/")

Regards.
