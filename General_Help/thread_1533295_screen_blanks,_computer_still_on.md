### Post by quixote on 2010-07-19
Is it possible the screen is gradually failing?  If you have Windows on this computer, does it have the same issue?  Ctrl-alt-del ought to reboot the computer.  Doesn't it?  

Ctrl-alt-backspace should just log you out and give you a login screen again.  That has the effect of restarting X windows, ie the graphics, and might kick the screen to life if it's a software problem.  To enable it in Karmic and Lucid, go to Preferences > Keyboard > Layout tab > Options button.  Under "Key sequence to kill the X server" tick the box for ctrl-alt-backspace.

The problem with both of those, as well as pushing the off button, is you lose all your work.  I don't know how you could jog the screen back to life more elegantly, though.  That doesn't mean there aren't better ways!

Also don't know about good hardware checks to figure out what's wrong with the screen.
