### Post by SeijiSensei on 2013-08-26
Open the directory with Dolphin, select a file, then hold down Shift and press Delete.  That deletes the file directly rather than copying it to Trash.

You can make the Delete option a permanent fixture in the right-click contextual menu.  In 12.04, follow the instructions I posted [here](http://ubuntuforums.org/showthread.php?t=2109165&p=12476243&viewfull=1#post12476243).  In 12.10 and later, use [this method](http://ubuntuforums.org/showthread.php?t=2109165&p=12478966&viewfull=1#post12478966) instead.  Now when you right-click on a file or directory entry in Dolphin, the Delete option will appear along with Move to Trash.

You can also adjust the size of the Trash from the Dolphin configuration menus.
