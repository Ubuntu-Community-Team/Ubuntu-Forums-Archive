### Post by themusicalduck on 2010-08-23
How did you try and make the partition? Remember you need to use a LiveCD session to repartition because you can't partition a drive while it is mounted.

You can tell the installer during the install process to create a partition and setup the dual-boot automatically. (It's a good idea to backup anything important on the hard drive though).

For messenger you could use Pidgin, Empathy, aMSN or Emesene.

For music, Banshee, Amarok, Rhythmbox. For video, Smplayer or VLC.

Only real alternative to Photoshop is GIMP, which works pretty great.

There are a few backup programs. Only one off the top of my head is Deja Dup.

For burning, Brasero or K3B.

Video conversion, WinFF. But there are mor.

Of course there's lots to search through in the Software Centre once you have Ubuntu installed.

Lastly, take a look at [www.ubuntu-manual.org](www.ubuntu-manual.org) it's ideal for new users of Ubuntu and even walks you through the install process.

Good luck!

Another point - remember the Netbook Edition has a completely different interface to the Desktop edition which is a bit more like Windows. (some people miss this). The Netbook interface is really good on netbooks though (I use it on mine)

### Post by davidmohammed on 2010-08-23
welcome,

some thoughts on your questions:

the default messenger that comes with ubuntu is called empathy.  You can use this for standard chat. Not sure about video chat though.

totem is the default media player for ubuntu - its preinstalled.

To make sure it works with all types of codecs, use the software centre and search for ubuntu-restricted-extras.

A good photoshop replacement is called Gimp - again use software centre to install.

I personally use Clonezilla to backup my laptop - search for it on google.

Burning software - brasero - it comes as default on the installation

Antiviruses - dont worry about viruses UNLESS you work in a mixed windows environment.  Search for "linux avg" on google if you want to install a AV product.
