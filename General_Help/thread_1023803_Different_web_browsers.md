### Post by x1a4 on 2008-12-28
[GNU IceCat]("http://www.gnu.org/software/icecat/") is Firefox without any of the non-free stuff with additional security features.  [SeaMonkey IS]("http://www.seamonkey-project.org") (formally Mozilla IS) is what Netscape evolved into.  [Flock]("http://www.flock.com/") is a social networking browser based on Firefox.  [Kazehakase]("http://kazehakase.sourceforge.jp/") is a nice light-weight GTK browser which uses Gecko.  [Midori]("http://www.twotoasts.de/index.php?/pages/midori_summary.html") is another light-weight GTK browser which uses WebKit.

### Post by cariboo on 2008-12-28
If you go to System-->Administration-->Synaptic Package Manager, click the search button and type in the search term:

```
web browser
```

The above search term will display all the web browser that are available in the repositories.
