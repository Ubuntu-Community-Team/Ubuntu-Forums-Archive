### Post by ryanhaigh on 2008-07-06
You could try using the http method for chatting to your msn contacts.
Accounts>Manage>Select your msn account>Modify>Advanced>Check the use http method box.

As an alternative, if you only use MSN you might want to try [http://www.emesene.org/](http://www.emesene.org/)

If you are using Hardy (ubuntu 8.04) you can install from synaptic/add-remove or just click here [apt://emesene]("apt://emesene")
