### Post by Rocket2DMn on 2009-04-19
To find where programs are located, there are a few commands you can use, for instance:
```
which pppoeconf
whereis pppoeconf
```
The commands above show us that the binary is located at /usr/sbin/pppoeconf

Cheers.
