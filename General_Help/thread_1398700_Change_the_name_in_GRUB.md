### Post by Leppie on 2010-02-04
best thing to do is edit your 30_os-prober script. open it with your editor:
```
gksudo gedit /etc/grub.d/30_os-prober
```
find the following section and add the red part:
```
      for LINUX in ${LINUXPROBED} ; do
        LROOT="`echo ${LINUX} | cut -d ':' -f 1`"
        LBOOT="`echo ${LINUX} | cut -d ':' -f 2`"
        LLABEL="`echo ${LINUX} | cut -d ':' -f 3 | tr '^' ' '`"
        LKERNEL="`echo ${LINUX} | cut -d ':' -f 4`"
        LINITRD="`echo ${LINUX} | cut -d ':' -f 5`"
        LPARAMS="`echo ${LINUX} | cut -d ':' -f 6- | tr '^' ' '`"

        if [ -z "${LLABEL}" ] ; then
          LLABEL="${LONGNAME}"
        fi
[COLOR=Red]
        # change menu entry to BackTrack
        if [ "${LBOOT}" = "/dev/sda1" ]; then
          LLABEL="BackTrack"
        fi[/COLOR]
```
save and exit the file, then run update-grub:
```
sudo update-grub
```
