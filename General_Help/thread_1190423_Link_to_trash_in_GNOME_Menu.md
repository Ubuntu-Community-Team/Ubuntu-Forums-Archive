### Post by jenkinbr on 2009-06-17
Yes - You can add it to places:

Open the file browser in ubuntu "Accessories >> File Browser"

Click the edit location bar and enter ~/.local/share/Trash/files

Click the Bookmarks menu and add it as a bookmark called 'Trash'

It should now be availible when you click 'Places >> Bookmarks >> Trash'
