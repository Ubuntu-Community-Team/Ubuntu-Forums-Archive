### Post by gobbles414 on 2008-08-23
In Synaptic, install *gscan2pdf* (with all recommended packages) and *libsane-extras*. If you can't find the gscan2pdf program in Synaptic, you can get it from [here]("http://gscan2pdf.sourceforge.net/") instead. Restart your computer, and then try using gscan2pdf to scan a document. NOTE, you may need to use the drop-down device selector in gscan2pdf in order to scan your document. If this idea fails, there are others to try. So don't despair!

PS: Which version of Ubuntu are you using?
