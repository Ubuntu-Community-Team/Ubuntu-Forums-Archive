### Post by interfect on 2012-01-12
You could try the [Autocomplete = on]("https://chrome.google.com/webstore/detail/ecpgkdflcnofdbbkiggklcfmgbnbabhh") extension, which re-enables saving values (and passwords) on form fields that have it set to disabled. Chrome interprets autocomplete = off on password fields as a flag disallowing users from saving the password in Chrome, while (maybe) Opera doesn't.
