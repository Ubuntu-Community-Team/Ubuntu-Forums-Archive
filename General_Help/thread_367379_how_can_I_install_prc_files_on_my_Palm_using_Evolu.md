### Post by fragos on 2007-02-23
You can install with Bluetooth using the Send to when right clicking the PRC.  Setup is a bit complicated.  The simplest way is to use an SD reader on your Linux box.  Just copy the PRC to an SD you then put in your Palm.  Place it in /PALM/Launcher and you can can run the ap off the SD card.  You can also move the PRC to the Palm RAM from the SD.  The first time you run a PRC it installs itself.

### Post by dangermouse28 on 2007-02-23
Why not use gpilot via the terminal?

e.g.  gpilot-install-file  --later appname.prc

Then sync the Palm with your PC. You'll find the application has been installed on the Palm.:)

### Post by defishguy on 2007-02-25
With my palm TX bluetooth works only about fifty percent of the time.   I set up a network share on my pc and copied all the PRC files to that share, connect to the network using wifi, and just copy the file to the palm using Wifile LT.  It works flawlessly.
