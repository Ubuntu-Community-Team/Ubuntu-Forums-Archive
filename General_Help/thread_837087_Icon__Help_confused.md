### Post by GrandpaLeaman on 2008-06-22
Did you select the new icons under the theme? If not you need to go into Appearance Preferences, and at the bottom of the window select Customize. Select the Icons tab and then scroll down till you find the icons set you installed. Select it and close. You will now have a new theme called Custom. You can leave it like that or you can select Save As and change the name to "My Theme" or whatever you want to name it.

Hope this was what you were looking for. Let me know if you have question.
