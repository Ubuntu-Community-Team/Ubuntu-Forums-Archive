The page is about the same as for normal computers, it's just smaller.
The iPhone and most phones with and HTML web browser will have either 240x320, 320x480 or 800x480 as screen resolution (the iPhone has the second one).
So you just need to make the page with a 480 pixels width, but always leave some margin, make it more like 460.
