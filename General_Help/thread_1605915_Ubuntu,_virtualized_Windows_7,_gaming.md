### Post by nerdleturtle on 2010-10-25
IMHO, your best option is a dual boot, because running two OSes at the same time really taxes hardware. I can also recommend wine to run several windows games in Ubuntu, as It works really well for certain games. Finally, I can recommend one excellent Linux native game: [http://alientrap.org/nexuiz/](http://alientrap.org/nexuiz/)
 
More info on Dual Booting with wubi here: [https://help.ubuntu.com/community/Wubi](https://help.ubuntu.com/community/Wubi)
Good Luck! And Welcome to the world of Ubuntu! :)
