### Post by john newbuntu on 2009-12-24
Run update manager and get all updates to the system.  If it still happens, check the logs located in /var/log directory.  There are several of them that start like dmesg, kern.log, syslog etc.  Most of these have timestamps.  Look them up at the time the problem happens and see if you can spot anything.
