### Post by pbpersson on 2012-02-18
Back when I did this I used webmin and it was a life saver!

You place it on the server and then you can connect to the server from any client system and easily administer the server using a simple GUI which automatically edits all the config files for you.

I have heard some people say that you need to be knowledgeable and hand-edit all those dozens of config files to be a real admin but I think life is just too short and we have too much to get accomplished.  Why not use a short cut?

### Post by winh8r on 2012-02-18
Hi,

Welcome to the forum. Firstly it is a little difficult for anyone to offer you any real help as you have not said which guides you have followed.

[https://help.ubuntu.com/11.10/serverguide/C/index.html]("https://help.ubuntu.com/11.10/serverguide/C/index.html")

is a good start, and I apologise if you have already read it.

If you are having difficulty coming to terms with setting up a server then perhaps it would be a better idea to wait until you are a little more familiar with Ubuntu in general, before attempting what can be a frustrating process, even to a more experienced user.

There are a lot of threads on this forum and you will probably get better results when you need help by wording your searches carefully and looking for results with a [SOLVED] marker on them.

My best advice would be to take things a step at a time and get to know what you are working with a little better before you beat yourself up about not being able to get the result you want right away.

All of us had to take time and learn, none of us knows all the answers to everything. 

If you cannot find an appropriate answer to a problem you are having, then by all means post a request for help but it will help people to help you if you are a bit more specific about what particular aspect of the setup you are having difficulty with and what you have already tried to resolve it.

Please do not be offended by this reply as it is not my intention to offend.
