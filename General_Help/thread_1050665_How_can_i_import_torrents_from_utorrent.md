### Post by mb_webguy on 2009-01-25
The problem is that the paths to your data are likely going to be different between uTorrent on Windows and whatever client you use on Ubuntu.  Otherwise, you could simply locate the directory where uTorrent stores its torrents, copy them, and drop them into your Ubuntu client.

If, however, you have set up your system so that both clients are looking at the same location, then it could be that simple.
