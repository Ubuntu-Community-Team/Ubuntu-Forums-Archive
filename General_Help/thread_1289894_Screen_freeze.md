### Post by 123456789123456789123456 on 2009-10-12
does not sound like a hardware problem but could be.  The OS locked up(froze) from the sound of it.  causes could be a bad install.  other reasons would be bad ram, Incorrect BIOS support for all three processors.  Check the BIOS, make sure that it is set to reconize and use all three processors.  If so, Does Ubuntu read all three processors?  When viewing system processes, does it ever read at the high 90's for cpu usage, or ever 100%?  if so, there is a problem somewhere with hardware most likely, since you have 3 processors, or a triple core processor, the cpu usage at most every stage of running the OS, should not be higher that 50% total with all processors combined.  Boot from the cd, and perform a memory test, at least 4 passes, see if there are any problems reported.  if so, does it allways accure, or even allways accure at the same point.  This information, can help track down the problem.
