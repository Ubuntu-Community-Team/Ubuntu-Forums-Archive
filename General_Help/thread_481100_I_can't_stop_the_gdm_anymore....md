Hmm I'm unsure. To be honest this is out of my league. I only know about ATi and nVidia graphics cards.
But yeah post your xorg.conf and lets have a look.

I know that there is a thing called runlevels in linux. One runlevel is single-user, and there is another one which is multi-user.

When X is set to load at singleuser level, it presents the graphical login box. When X is set to run at multiuser level, you get the text login box and type startx to go into the GUI.

In Ubuntu, the run level scripts are found at /etc/rc.2 and /etc/rc.3 and so on. Im not sure which number is which but someone here can tell you.
