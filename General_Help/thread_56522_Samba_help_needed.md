 as amohanty suggested, you have to have SAMBA user+passwords as well as ubuntu user accounts

samba needs to be installed (not just the client - and i had to restart samba to get it working, or reboot..)

i think you need to create a passwd file first

i did this (correctly? i don't know) by typing 

<sudo smbpasswd -a> 

and entering my password at the prompt

i then added users by:

<smbpasswd -a user1> 

and typing the appropriate password

i think the 'windows file sharing' bit you are talking about is found by attempting to share a directory by right-clicking on it and choosing 'share folder'

choose share using SMB and theres your 'general windows filesharing options' button

i control the access to the shares with smb.conf

i think theres a simpler way to share, by using 'security=share' in your smb.conf, but i've never tried this as i need access control
