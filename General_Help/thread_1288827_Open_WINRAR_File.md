### Post by tabibito on 2009-10-12
You can use ark, the default pre-installed archive manager in ubuntu. But first you have to enable rar support by typing the following in the terminal:

sudo apt-get install unrar

to get the propietary version of unrar or

sudo apt-get install unrar-free

for open-source version of it. Both are free though.
