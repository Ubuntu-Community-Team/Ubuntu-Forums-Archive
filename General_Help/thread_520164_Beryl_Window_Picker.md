### Post by H_out on 2007-08-16
RE: getting Beryl Window Picker to include minimized windows

I was looking for the same thing and finally found it!

Beryl Settings Manager > Window Management > Scale

Go to the Behaviour tab.  (they spelled behavior wrong?)

Scroll down to "Show Minimized Windows".

Check the box and you should be set.
