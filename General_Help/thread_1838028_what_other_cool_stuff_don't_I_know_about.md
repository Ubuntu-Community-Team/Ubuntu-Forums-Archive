From terminal: 
- start a command, press tab. If there's only one match, the command is auto-completed. If there are several matches, pressing tab again displays them. 
- type in "apropos xyz" (replace xyz for something you suspect some commands should exist, for example "apropos nvidia"). A list of related commands do appear.

Have fun :)
