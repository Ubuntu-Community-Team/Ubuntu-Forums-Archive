### Post by ipsi on 2011-03-30
I know this is unlikely to be of much use to the original poster, but to anyone else looking for a way around it, you'll need to pass the size as an option to the mount command. e.g:

```
sudo mount -t tmpfs -o size=10G tmpfs /mnt/inmemorymount/
```

Or, if putting in /etc/fstab:

```
tmpfs /mnt/inmemorymount/ tmpfs size=10G 0 0
```

(Found thanks to [http://en.wikipedia.org/wiki/Tmpfs](http://en.wikipedia.org/wiki/Tmpfs)). I have no idea why it is this way, but at least it can be worked around.
