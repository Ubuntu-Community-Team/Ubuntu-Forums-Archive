### Post by lovinglinux on 2011-09-30
If you are using Ubuntu 11.04, then just update and upgrade your system. You will get Firefox 7.

I am not using Ubuntu right now and I usually do this stuff via command line, but I suppose you can do that via Software Center. If not, then look for Update Manager in your applications list.

If you prefer the command-line, just execute these on a terminal:

```
sudo apt-get update
sudo apt-get upgrade
```
