### Post by njf on 2006-03-25
For same software, they use the same package.
Kubuntu comes with different default applications than Ubuntu (Konqueror instead of Firefox). 
You can still can get Firefox in Kubuntu by installing firefox package in the repository, and the package is the same as the one you get in Ubuntu.
