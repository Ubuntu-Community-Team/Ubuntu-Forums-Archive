### Post by kostkon on 2007-06-02
> **SheNux said:**
> Which directory do I drop my .ttf fonts in?

if you open a nautilus window, e.g. your *Home* folder, go to the *View* menu and select *Show Hidden Files*, you will be able to find the .**fonts** folder. You will, then, be able to paste any font you like there.

Alternatively, if you activate the* Location Bar* of a nautilus window and go to the address **fonts:///** you'll reach the same folder. Again paste your fonts there.
