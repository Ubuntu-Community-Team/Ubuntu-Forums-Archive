### Post by Mirge on 2009-05-24
For what it's worth, I've had no freezes in Kubuntu 9.04 on 4 computers, whereas I had several freezes/lockups in Ubuntu 9.04.

If you were primarily a windows user, I would maybe check out some screenshots of Ubuntu, Kubuntu, Xubuntu, etc and download & burn Live CD's for which you think you would be most comfortable with. Then try them out & see what you think! I'm a die-hard fan of Ubuntu 9.04 AND Kubuntu 9.04. I primarily use KDE, but GNOME's simplicity is fantastic too.

If you're interested in learning web development, I'd definitely start with HTML with a particular focus on CSS (Cascading Stylesheets). GIMP or Photoshop will be an invaluable tool when it comes to web design as well. When you start getting curious about server-side programming, I'd recommend dabbling with PHP. It's widely supported and has a huge community.

Good luck to ya!

### Post by VCoolio on 2009-05-24
Hi, welcome to Ubuntu, at least I hope so. I would definitely recommend Ubuntu, it's aimed at new linux-users and it's (one of?) the biggest linux distro's around so it has the most help to be found (eg these forums). 

As said, you could do without the terminal; on the other hand it's fast and simple, especially if one needs to explain / get something done. On these forums or on irc most help is given with commands to enter in the terminal, which is much easier than pointing out where to click in all the menus and tabs. So to know what your doing [this is a link]("http://www.linuxcommand.org/learning_the_shell.php") to get started and [this is a link]("http://www.tldp.org/LDP/abs/html/") that has pretty much everything you'll need for the next years covered.
