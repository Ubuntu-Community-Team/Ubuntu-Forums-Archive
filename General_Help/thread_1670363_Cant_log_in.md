### Post by TimEnid on 2011-01-20
i guess no one ever experienced this. but in case someone does, i was able to fix it. from that same screen, press ctrl+alt+f1, which will bring up the terminal. enter user name and password and then the following command:
sudo dpkg-reconfigure gdm
