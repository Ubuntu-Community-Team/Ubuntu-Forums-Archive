### Post by QIII on 2014-06-21
Hello!

.tar and .gz (or even .tar.gz!) files are compressed files like a "zip" in Windows.

You will need to decompress (unzip) them before you can do anything with them.

As for how to install what is in them, that is impossible to say with any certainty without knowing what the application is.  Many packages will include a "read me" file with instructions that may be unique to that application.

### Post by buzzingrobot on 2014-06-21
In Linux, as in Unix, software distributed in .tar.gz files is predominantly source code that must be compiled and installed before use.  Any dependencies must be identified, located, and installed manually.  The Windows custom of putting ready-to-run executables in a compressed archive, along with all needed support libraries, that can be launched after expansion into a single directory is rarely used in Linux.
