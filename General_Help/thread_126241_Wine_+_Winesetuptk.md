[QUOTE=Ocxic]i wish to use wine, and use winsetuptk to configure. but i cannot install both of these using synaptic. I'll mark wine, then mark winestuptk and it will unmark wine Why?[/QUOTE]
The winestuptk version in the repositories isn't certified with the later versions of wine, so it won't let you install both.

wine has it's own inbuilt configuration (winecfg), or find the "Sidenet" configuration tools and use them.
