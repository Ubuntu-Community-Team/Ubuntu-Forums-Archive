### Post by cosbear on 2007-06-01
Howdy:

I haven't heard of any problems with secondary IDE on Feisty; though I'm a relative newcomer.  Is this something you have heard others complain of?  I have installed it on 3 computers with cd writers and/or dvd writers on the secondary IDE port and haven't had a problem with any of them.  I'm just guessing here without much info to go on but I would think that it is more likely a problem in setup, or a incorrect jumper setting on the device maybe coupled with incorrect cabling.  Have you entered the bios setup utility on boot and checked to make sure the secondary IDE port is switched on? If it is a bios or hardware problem I can probably help you solve it, if it's got to do with configuring the OS I might not be the best source as I'm fairly new to it.  I'm assuming that since the cd isn't working that you did a network upgrade to feisty.  I would enter the bios setup utility on your machine at boot.  It usually lists the devices the motherboard recognized on boot.  If the cd rom shows up there then it is definately a software problem with the OS.  If your bios recognizes the device then you can probably make sure that cd is in boot priority list before the hardrive and put a feisty disc in and boot from that if it works and the cd is recognized on the live version of feisty then a clean install from the cd should solve your problem.  I you still need help figuring it out feel free to message me.  Good luck with your problem.  cosbear
