### Post by SalHelder on 2011-08-07
Try this:

1. Install Opera.

2. Open the website: [https://hr.macys.net/insite/common/logon.asp](https://hr.macys.net/insite/common/logon.asp)
It'll redirect you but it's ok.

3. Now right click the page and go to: "Edit site's preferences."

4. Go the last tab the one that says connections.

5. There look for the option that says "Mask as Internet Explorer"

6. Enter the address again: [https://hr.macys.net/insite/common/logon.asp](https://hr.macys.net/insite/common/logon.asp)

It worked here.
