### Post by Morbius1 on 2011-06-19
You might want to explore gedit with the additional plugin package:
```
gedit-plugins
```It has for example a "Session Saver" plugin that allows you to save multiple sessions ( gedit > File > Saved Sessions > Save current session )

[COLOR=Blue]**EDIT**[/COLOR]: Gedit already autosaves you just need to enable it: gedit > edit > preferences > editor > File saving
