### Post by colorlessprism on 2010-05-30
when you signed up to the forums you should have received a "welcome" message. in that message should be a link to "Ubuntu pocket guide" this little pdf provides a ton of basic information in a great format. there are as many Linux books and how-to's as there are Linux users. the only other book i can really recommend is "Ubuntu Linux toolbox" this manual provides more than 1000 terminal commands and is grouped by tasks. the hierarchy seems strange at first but with a little time it really does make sense.
