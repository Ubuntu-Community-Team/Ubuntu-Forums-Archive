### Post by MCrittenden on 2008-02-21
If you're asking if it's possible to change the default, meaning that when you open a folder from "Places" it will open in a different file manager than nautilus, then check out [http://www.psychocats.net/ubuntu/nonautilusplease](http://www.psychocats.net/ubuntu/nonautilusplease) however any folder opened directly from an icon on the desktop will still open in Nautilus. Not sure how to change this. 

If you just want to download an alternative file browser, but not set it as the default, wolfen69's recommendations are good ones. I'd also suggest pcmanfm.

### Post by MCrittenden on 2008-02-21
That link will work as long as you click on the folder from the "Places" menu rather than the desktop itself. If you're used to opening things directly from the desktop, just get in the habit of doing "places -> desktop" instead. I changed my default to Thunar like that.
