I'm using Pidgin for internet messenger but I don't know how to save and view my message archive.

### Post by fraktalek on 2009-11-24
You can enable logging in Tools -> Preferences -> Logging
There check "record all chats" and "record all instant messages".

You can then view the logs by right-clicking on a buddy in the contact list and selecting "View log".

There also are some handy plugins for logging. One for example shows you the last conversation when you start chatting to someone. Check Tools -> Plugins.

Pidgin stores the logs in ~/.purple/logs where ~ stands for your home directory which you probably know anyway.

### Post by audiomick on 2009-12-09
In your file browser in your home directory, go to view> show hidden files.

This will make lots of files the are called ".something_or_other" visible. The dot at the front of the name tells the system that the file is to be a hidden file.

Look for the .purple/logs file that fraktalek referred to. 

You should be able to just copy that file and save it to somewhere else for safe keeping.
