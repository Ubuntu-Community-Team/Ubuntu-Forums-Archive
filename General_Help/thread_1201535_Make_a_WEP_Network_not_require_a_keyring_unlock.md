### Post by PwnCakes193 on 2009-07-01
Whenever I turn on my computer, it always asks me for my password to unlock the keyring so it can connect to my network that is protected by WEP. How do I disable the requirement for a keyring so I don't have to put a password in to connect each time?

---

### Post by Legace on 2009-07-01
> **PwnCakes193 said:**
> Whenever I turn on my computer, it always asks me for my password to unlock the keyring so it can connect to my network that is protected by WEP. How do I disable the requirement for a keyring so I don't have to put a password in to connect each time?

Open System -> Preferences -> Network Connections and select the "Wireless" tab. Select your connection and press **Edit**. Make sure the **Available to all users** entry (next to "Cancel") is checked. Press Apply. Reboot, and it shouldn't ask password anymore :)
