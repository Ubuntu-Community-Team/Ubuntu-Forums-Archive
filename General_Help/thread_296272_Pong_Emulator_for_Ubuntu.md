### Post by skymt on 2006-11-09
Pong, in the original arcade version, is technically impossible to emulate (as the term "emulate" is generally used). See [the MAME FAQ](http://www.mame.net/mamefaq.html#m17) for some details. Basically, in order to run the original Pong, you would need to write a *simulator* (not an emulator) that would run only the one game. The reward/effort ratio is too low, so nobody's done it and nobody's likely to.

You will probably have to settle for [a clone](http://happypenguin.org/list?search=pong). Or, find a Pong ROM for a home console that has an emulator (Google is your friend).
