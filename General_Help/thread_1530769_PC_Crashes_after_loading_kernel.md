### Post by erythros on 2010-07-14
I have never found a bios update that corrected a BSOD issue. So I doubt that you'll find a fix in that direction. Since you isolated the HDD, you need to do the same for the other components.
 
If you have multiple sticks of RAM, remove all but one and boot the system and try to run the Ubuntu LiveCD. Swap that RAM stick with a different one untill all sticks have been cycled. This will help prove your memory is good.
 
If you can swap video cards do so.
 
The last option is to swap the motherboard but keep all the original components...
