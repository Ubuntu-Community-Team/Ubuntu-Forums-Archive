### Post by ManamiVixen on 2013-03-12
Oh, that is easy. in your home folder, click both ctrl+h at the same time to show all files. Then create a folder called .icons, if the folder is already there, just copy the download icon package .tar.gz into the folder and extract it there. Now you need a tweak tool to change the theme. Ubuntu by default dosen't have a real tweak tool. So you have to install something like ubuntu-tweak or gnome-tweak-tool.

Ubuntu-Tweak is the best, but can be dangerous is used improperly. In terminal, put "sudo add-apt-repository -y ppa:tualatrix/ppa" then "sudo apt-get update && sudo apt-get install ubuntu-tweak" Now hit the windows key and type ubuntu-tweak and click the icon. Navigate to tweaks section, then click the themes icon. There you can change the icons.
