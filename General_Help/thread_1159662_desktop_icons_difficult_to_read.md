### Post by uiberto on 2009-05-15
Thanks!

I tried this before but thought it was a dead-end. I achieved the icon text background I wanted with these settings:

# sudo gnome-color-chooser

Click on Desktop tab
Under Icon Labels, check "beautify icon labels"
Under normal, check on Background and set a color
Under normal, check Background Opacity and set it greater than 0 (I set mine to 192/255)
