### Post by p_quarles on 2007-11-02
This is going to be specific to the window manager you're using. If you're running KDE, Beryl or Compiz-Fusion, the ability to map commands to keys is built in. That said, they're not always that good at recognizing "extra" keys on newer keyboards. 

Metacity (the default Gnome WM) doesn't have as many options. From what I understand, the packages "hotkeys" and "keylaunch" are both very good at doing what you're after. I've never used either, but I know others have had good luck. Also, from what I heard, keylaunch has better GUI support.
