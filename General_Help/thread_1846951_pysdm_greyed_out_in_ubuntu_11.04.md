### Post by hpvpp on 2012-04-03
You need to click (one of) the little triangles to open the (drop down) list.  After that you can select the volume you want.

Notes

1- you need to know a lot about Linux to be able to make use of it sensibly
2- pysdm does not properly reflect the actual disk layout
3- see [http://pysdm.sourceforge.net/](http://pysdm.sourceforge.net/)
