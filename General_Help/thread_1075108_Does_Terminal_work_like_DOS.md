### Post by Psyphre on 2009-02-19
Just for convinience I'll list you a couple navigation commands:

'**cd**' is to **c**hange **d**irectory.

So your in your 'home' directory as that is where terminal starts up in.
You have a folder called 'Pictures' in your home directory. So to go there in terminal do this:

cd Pictures/

To go out of a folder (i.e go back) do this:

cd ..

To list the files and folders within a folder type:

ls

If you cant be bothered to navigate thru command entirely (or you just dont know where a file is location using terminal) then you can cheat.

Type: 'cd'  and then drag a folder or file from your browser into the terminal. Press enter and voila! I do it all the time :)

Also dont forget terminal (and linux in general) is **_case sensitive_**. The number of times I failed at using terminal when I was new at linux because I forgot this......

Anyway hope that helps. Obviously read the links posted cos there are so many useful commands.
