### Post by sikander3786 on 2010-12-23
Welcome to the forums :-)

After selecting Windows from Grub menu, immediately start pressing F8 and you might see the boot options then.

The reason I am saying this is because Grub just chainloads Windows bootloader hence transferring all the control over to it and there shouldn't be any changes in Windows boot later.
