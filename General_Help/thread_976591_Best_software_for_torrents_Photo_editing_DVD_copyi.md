### Post by lovinglinux on 2008-11-09
> **Kill Vista said:**
> I'm using Transmission for torrents, it's ok but very limited interface. I tried KGet but when I add torrents it says they are stopped and they won't start after clicking start. 

Deluge is great. I has all essential features, a nice interface and the configuration is simple. It is in the repositories.

> **Kill Vista said:**
> I'm looking for something like ACDSee for photo editing. I don't need anything fancy, and I'm not doing any real editing of the photos. Just need something that has an auto-correct, crop, resize and print features.

It is hard to beat ACDSee features, but in my opinion it's strength is about organizing photo collections, not editing.

Gimp is the best photo editor for Linux available out there. 

Phatch is a nice batch converter if you simply need to crop, auto-correct and resize pictures. The editing approach is a little bit different, because you first create the editing rules, then drag the photos to Patch interface an it will apply the editing rules to them. If you do a lot of simple editing tasks, this is great for the job. It is in the repositories.
