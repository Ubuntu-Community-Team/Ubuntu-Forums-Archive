### Post by localzuk on 2006-03-20
MP3 support is not native to Ubuntu due to license restrictions (it is a patented file format so license fee's are due if anyone creates a decoder for it).

However, if you wish to install one - take a look at [http://www.ubuntuguide.org/#codecs](http://www.ubuntuguide.org/#codecs) for information specific to codecs - you will need to look at the earlier section regarding enabling repositories (change 'hoary' to 'breezy' where applicable if you are using breezy) before this will be of much use to you.

Other than that, take a look at automatix or easyubuntu ([http://www.ubuntuforums.org/forumdisplay.php?f=77](http://www.ubuntuforums.org/forumdisplay.php?f=77) and [http://www.ubuntuforums.org/forumdisplay.php?f=86](http://www.ubuntuforums.org/forumdisplay.php?f=86) respectively)

Hope that helps

### Post by sleepkreep on 2006-03-22
Just as an update.  The company that owns the patent on mp3 format has released it as open source now with the intentions that linux distros can ship with support of mp3s.  Right now, it is licensed only to GStreamer but the source code is released under the MIT license for other multimedia engine creators to hack away with it.  check [http://www.osnews.com/comment.php?news_id=13078]("http://www.osnews.com/comment.php?news_id=13078")
