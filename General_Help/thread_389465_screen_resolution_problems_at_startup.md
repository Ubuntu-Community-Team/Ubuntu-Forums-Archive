### Post by Tobster on 2007-03-20
I had a faulty Ubuntu disk.  I would get another disk and do a complete re-installment.  Sometime if you re-install Ubuntu this can happen I think it because it impossible to fully delete old data so the old data sometimes can get mixed with the new data but a re-installment with a fresh disk should fix the problem.

It might be worth installing another OS just to delete the Ubuntu data THEN re-install Ubuntu just so that Ubuntu is writing over a different language this will stop the old Ubuntu data getting mixed up with the new.
