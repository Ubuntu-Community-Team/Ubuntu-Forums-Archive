### Post by vanadium on 2009-12-22
It is probably the way how it works. The advantage is that you have visual feedback when there are no messages. A windows suddenly closing might be considered more confusing for users. Yet, with the current behaviour, a message indicating that there are no more message would be appropriate.

Never noticed this myself, because I tend to work from the message list, and I do not delete (non-spam) messages. They just get transferred to an archive after a while.
