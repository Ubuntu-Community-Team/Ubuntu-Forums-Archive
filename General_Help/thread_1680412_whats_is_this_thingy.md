### Post by lanzi on 2011-02-02
Well, on German (all QWERTZ?) keyboards it´s a combination of AltGr + ">" keys. If you´re typing on a QWERTY one (American layout) chances are that you´d need to use the Shift + "\" combination of keys ([http://www.techpin.com/wp-content/uploads/2010/01/cool-layouts.png):](http://www.techpin.com/wp-content/uploads/2010/01/cool-layouts.png):))
