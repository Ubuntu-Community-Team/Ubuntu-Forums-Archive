### Post by ajgreeny on 2012-02-29
Yes, dead easily, provided you have all the necessary codecs required to both decode and encode both the original file-type and the one you want to encode.

**ffmpeg** is a command line application, but extremely flexible if you need it to be, or very simple if you prefer.

**Winff** is a gui for ffmpeg, so misses out on a few of the options you can use in terminal, but most users will never use or miss them.

Here's a web-page with some useful example commands if you should decide to try ffmpeg in terminal.
[http://www.catswhocode.com/blog/19-ffmpeg-commands-for-all-needs](http://www.catswhocode.com/blog/19-ffmpeg-commands-for-all-needs)
