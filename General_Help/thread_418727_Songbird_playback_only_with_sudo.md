### Post by TransitMan on 2007-04-23
When you upgraded the OS did you also upgrade Songbird to the Feisty edition, or did you leave under Edgy?
And how did you install Songbird? (It is not in the repositories, just checked!)
If you used Automatix (make sure you have the version for Feisty installed), use that program to uninstall Songbird, then reinstall Songbird.
From there you should have no problems.
