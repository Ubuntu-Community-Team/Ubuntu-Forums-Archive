### Post by sureshatt on 2010-06-08
[LIST]
[*]Use "chmod" command to change permissions of a file or a folder
[*]"chmod a+rwx TargetFile"
[*]above command enables reading,writing & execuing permissions for you.
[*]"chmod -R a+rwx TargetFolder" command will run recursively and give permissions to every folder and file inside the TargetFolder
[/LIST]
