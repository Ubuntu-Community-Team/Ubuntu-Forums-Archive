### Post by Shazaam on 2008-12-30
1gig of ram should be serviceable. What you need to do is disable unneeded services (Windows AND Linux) that start at boot and run in the background. If you don't have any Bluetooth hardware attatched, disable it. If you don't need fancy desktop effects, disable them. No printer? Disable it. Etc.
If, after disabling services and you find that it still runs too slow for you then go for the memory upgrade. One thing you have to remember is that it is all connected. Memory, hard drive speed and free space, video card memory, etc. One (or more) weak areas have an effect on total performance.
A good Vista start...
[http://www.theeldergeekvista.com/control_panel.htm](http://www.theeldergeekvista.com/control_panel.htm)
Somewhere in Administrative Tools should be a Services folder. Carefully (after researching) check and edit the ones that start up automatically. Disabling some can have drastic unwanted effects so do your research.
