### Post by deadflowr on 2018-06-04
True openbox is a window manager.
But LXDE is more than an application.
It's a set of specifically designed packages created to work best with one another.
Hence desktop environment.
applications pertaining to any desktop environment would include
text editor 
file manager
music/video player(s)
word processor
web browsers/email clients
system utilities
and more

There is usually also design perks associated with any DE,
such as panel designs and layouts.
Or if the DE uses some sort of dock or not.
As well as theme selection and what have you.

Window Managers are just a layer within the environment.
(And important layer, but a layer nonetheless)
