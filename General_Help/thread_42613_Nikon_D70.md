### Post by heimo on 2005-07-25
Seems like this questions has gone without answer for about a month. (I'll answer anyway.)

You could use gtkam and install dcraw, gimp-dcraw (and Gimp) to download jpgs and Nikon raw files (NEFs). You can use gtkam to download files and Gimp to post process. Also very handy program for converting and doing batch processing is ImageMagick
