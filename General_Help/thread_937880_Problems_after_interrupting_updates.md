### Post by admelfo on 2008-10-04
Hi -- first things first -- I've been playing with Ubuntu for all of three weeks. So, I know nothing, really, and am learning everything.

That said, from your description, if you were trying to install to a fresh hard-drive, and you lost power in the middle of it, why not just start over? If I were you, I'd want to know that I got a nice, clean, complete install -- now's the time to do it, before you actually start using the OS.

My first ubuntu install was on an 7-yr-old 4G hard-drive that had been configged for Win 2000. I've since swapped out the old HD with a new 80G drive -- so I was trying to do what you're doing. I noticed that if anything interrupted the install, while I'd eventually get into Ubuntu, the lower third of my screen was screwy -- not white, but some strange video thing going on. At any rate, I FDisked the new drive, and repeated the install -- haven't had any probs since. My only regret is that I didn't do that original install on the new hard drive with the latest version of U -- I had a disk from a prev version, and had to update to 8.04 after I installed that.
