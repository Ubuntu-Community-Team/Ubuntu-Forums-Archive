### Post by carmenin87 on 2012-06-16
There is a PDF file that I like to read in Google Chrome.

In Windows, using Chrome, I can just type c:\ in the field where you enter a web url and then browse to the location.

How would I do something like that in Ubuntu?

---

### Post by TVTrukChik on 2012-06-16
Type in "file:///home/your-user-name".  That's your home folder.  If the file's on the actual desktop, type in "/Desktop" after your user name.
