### Post by cmnorton on 2008-04-10
I use PowerTerm InterConnect for Linux -- [http://www.ericom.com/pti4linux.asp](http://www.ericom.com/pti4linux.asp) -- but it's expensive at about $150.00. There are some emulators in Open Source that are good, but there is a catch with those.

Do you have to map keys, F5, F6, F7, and so on?

If you have to map keys, I could find no emulator that would let me do that other than PowerTerm (on Linux). There are of course a lot of good terminal emulators for Windows, my favorite being Van Dyke's Secure CRT.

Here is a link to some other emulators that appear to be shareware or open source.

### Post by fragos on 2008-04-15
If it helps in your search, VT-100 is ANSI 2.64. When ASCII terminals were the vouge the VT100 was the most prevalent. I've never seen a terminal emulator that wasn't VT100 capable. At least this was true in the Windows world. A free package called "xvt" might work for you. It's in the Ubuntu repositories.
