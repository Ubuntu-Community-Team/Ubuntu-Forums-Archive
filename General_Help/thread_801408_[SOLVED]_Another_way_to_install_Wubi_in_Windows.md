Only the Live CD may be used. Actually, it is not necessary (but it is a good idea) to burn a CD to use Wubi. One may download the iso for the Desktop (i.e., Live) CD, place it into the same folder as is Wubi-8.04.exe, and start Wubi. It will install from there. To precisely answer your question, however, the Live CD may be used. There are instructions for that on the Wubi website, as I recall.

May I suggest going to the Wubi forum and looking there for some information that will assist you? [http://ubuntuforums.org/forumdisplay.php?f=234](http://ubuntuforums.org/forumdisplay.php?f=234) is the link.
