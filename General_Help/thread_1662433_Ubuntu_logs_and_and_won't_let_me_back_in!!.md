### Post by tredegar on 2011-01-08
At the grub boot screen select Recovery.
Then choose to drop to a root shell
At the root shell type **passwd *yourusername***
Give yourself a new password. Remember it.
Type **reboot**

When you get to your desktop go to System Preferences Screensaver.
Uncheck the box that says "Lock screen ...."
