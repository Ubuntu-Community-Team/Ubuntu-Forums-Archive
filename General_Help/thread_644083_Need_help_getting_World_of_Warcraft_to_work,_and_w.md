### Post by sajro on 2007-12-18
EDIT: See you're using Kubuntu. Instead of Applications -> Accessories -> Terminal go to KMenu -> System -> Konsole.

I think you misunderstood. You were probably told to use Wine and put your question in a forum (this is a forum, where you can talk to other users).

I haven't installed WoW in Wine but this should work.

So, to get Wine, go to Applications -> Accessories -> Terminal or hit alt-f2 and type "xterm" (without quotes). Now type (or copy and paste) > sudo apt-get install wine

Now insert your WoW CD. Open up "Wine File" somewhere in your Applications menu. Browse to D:\ (or whatever the CD-ROM is in Wine File) and double-click setup.exe (or whatever the setup program is). It should install properly now (rated as "Gold" on Wine's site). 

Now find World of Warcraft in your Applications menu, click it, sign in, and enjoy.
