### Post by ankspo71 on 2010-01-22
Hi, I hope I am not too late. **Don't remove python.** Most of ubuntu (including xubuntu) depends on it, and removing it could ruin your *ubuntu system. (I found this out the hard way)

Also, there are so many useful programs that you can install that are written with python, and without python you won't be able to run those programs.

### Post by doas777 on 2010-01-22
python is not just a langague, but also a runtime. if you uninstall python you lose the ability to run python programs, not the ability to write them (you could write code just fine without it, but you would have to test it elsewhere). java and the .net framework are the same way: they are there to run programs written for those frameworks, not development tools.
