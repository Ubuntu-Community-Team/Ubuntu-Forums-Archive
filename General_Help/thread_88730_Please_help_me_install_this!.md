### Post by Sykil on 2005-11-10
I'm not familiar with Cedega, but I'll try to help:

whatever.tar.gz is just like a *.zip archive in Windows.  Find the file in Nautilus (the file manager) and right click on it. You should see "extract here" on the menu. Click it.

It'll probably make a folder in the directory you're in.  Enter this folder and look for any kind of "readme" file that can offer you further instruction.

Edit: here's a wiki article, [https://wiki.ubuntu.com/Cedega](https://wiki.ubuntu.com/Cedega)
