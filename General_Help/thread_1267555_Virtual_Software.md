### Post by rocket16 on 2009-09-16
Hello and welcome to Ubuntu! Virtual Box is nice. You can simply install it by the following process:
1. Go to Applications -> Add/Remove
2. Type "VirtualBox" in the Search Box
3. Check out Virtual Box OSE and click Apply Changes
This will download and install VirtualBox.
