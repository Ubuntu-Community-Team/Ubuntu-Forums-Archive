### Post by grahammechanical on 2011-06-13
Does this happen every time that you try to open any folder? File Manager will associate files with certain programs so that files of that type will open in the associated programs. A folder is just another kind of file to file manager and somehow an association has been made with movie player.

Try this. Right click on the desktop and select Create folder. Then right click on the new untitled folder and select Open with Other Application. From the list of applications select File Browser.

This might cause File Manager to associate itself to folders and open them instead of loading Movie Player. The Trash bin and the System label are just folders. So, this should work. It did for me when I somehow caused File Manager to become confused with its file associations. In my case it was with Evince PDF viewer.
