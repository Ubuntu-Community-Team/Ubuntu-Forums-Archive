### Post by Favux on 2009-02-15
Hi newagelink,

Yes, you have the xorg.conf edited correctly.

I think you tried to edit the xorg.conf on the live CD.  Try booting without the live CD.  To edit the xorg.conf in a graphical environment:
```
gksudo gedit /etc/X11/xorg.conf
```
to do it w/o graphical environment:
```
sudo nano /etc/X11/xorg.conf
```
then ctrl-O (or F3) to save and ctrl-X (or F2) to exit.

With an Xserver problem does X offer to let you change graphics?  Try a VESA driver.  If you get a graphical shell use gksudo gedit.

When you boot do you get to the log-in screen?  If so try Options in the left lower corner and try Failsafe Gnome (gedit) or terminal (nano).  Or boot with the (recovery) kernel. Then pick "drop to root shell prompt" and use  sudo nano.
