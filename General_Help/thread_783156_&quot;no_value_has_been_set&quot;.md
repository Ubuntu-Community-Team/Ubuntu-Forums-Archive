### Post by aaronp on 2008-08-22
I know it's been a couple of months but just in case - the answer to your problem is that you need to set a 'real name' for yourself.
Your username might be 'abc' or whatever but your real name is unspecified so Ubuntu calls it 'no value has been set'

Go to System > Administration > Users and Groups
then select your username and choose properties, then fill in your real name.
You'll need to enter your admin password to effect this change.

hope this helps...
