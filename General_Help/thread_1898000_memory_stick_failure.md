### Post by Total Noob on 2011-12-20
Can a memory stick fail completely?

I picked up an old Compaq desktop, a few years old, a P4. I replaced the hard drive and installed 11.4 on there, and it worked fine, so I put it in the basement pending finding a permanent use.

When I broke it out a few months later, it was dead. Monitor would not flip on, and I got three long beeps and a flashing num lock signal on my keyboard.

I got various tips but what worked was removing one of the two identical 1 gb memory sticks that I got with the machine. Retesting showed that the computer would boot with one but not the other singly, and not with both installed in their original places or switched.

So the one stick is bad, or so it seems.

Can a memory card fail like that? Both were working before and had the same exposure to the basement, and yet one was good and the other bad. Is it just time before the second one gives up too? Is there a way to test broken and working sticks to ascertain whether I should replace them?

Thanks.

### Post by sudodus on 2011-12-20
I assume you are referring to RAM sticks. It is very unlikely that it fails, if it is not tampered with (just sitting in a computer). But it happens. There is no reason to believe that the other one will fail soon, it is likely to last as long as the motherboard or cpu.

Maybe it is humid in the basement? Can you see any corrosion on the bad memory stick? In that case, maybe you can try to remove it very carefully avoiding any static electricity. But probably it's gone.

Or maybe it is the memory slot on the motherboard, that is bad. Will the computer start with the good stick in any of the slots?
