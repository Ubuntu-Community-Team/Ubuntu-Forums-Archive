### Post by agiamba on 2007-03-02
you can add Wine into the repositories and then just sudo apt-get install wine, or you can compile it. compiling wine is fairly easy, download the .tar.bz2, extract, change your directory to the file, and then run ./compile. instructions follow from there, if you are missing libraries, you can apt-get them.
