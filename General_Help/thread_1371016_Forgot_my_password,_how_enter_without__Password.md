### Post by warfacegod on 2010-01-12
When grub loads and shows all the operating systems you have choose the newest ubuntu *recovery* heading. Once it is done loading select "drop to shell" hit enter. Then type passwd username (whatever name you use to log on). Hit enter, type new password, hit enter. Done.
