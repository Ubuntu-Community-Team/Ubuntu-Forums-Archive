Hi there:

It isn't horribly user friendly, bt you could look at using a tool called psnup (part of the psutils package):

Here is the usage:

[http://www.tu-berlin.de/zrz/software/utilities/psutils/psnup.html](http://www.tu-berlin.de/zrz/software/utilities/psutils/psnup.html)

Bascialyl, if you can print to file to create a post script file, then use this tool to create a new post script file with multiple pages per sheet. I have then used ps2pdf to create a PDF file, but this step is not strictly speaking necessary.
