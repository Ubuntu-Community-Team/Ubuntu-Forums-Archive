### Post by clementc on 2019-01-01
Hi [**[COLOR=#000000]Kris_M[/COLOR]**]("https://ubuntuforums.org/member.php?u=2014004"),

Can you please try the following and advise if this has helped resolve your issue.

1. In Terminal, run: sudo systemctl edit cups-browsed.service
2. Type in:
```
[Service]
TimeoutStopSec=5
```
3. Still in Terminal, run: sudo systemctl daemon-reload

The patch like you call it simply changes the default timeout when shutting down from 1min30sec to 5sec.
