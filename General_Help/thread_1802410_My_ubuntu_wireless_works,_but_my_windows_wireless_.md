### Post by Topsiho on 2011-07-12
In Windows the drivers for hardware must be installed first, before this hardware can be used. To do this one needs a disk or a file which contains these drivers, and which must be available during the install.

In Ubuntu this is done automatically, if supported.

Ubuntu does support much hardware, but not all. This also aplies to Windows.

If you haven't a disk with the necessary drivers, you might try and search for it on the internet, using the exact description of the hardware.

But really this is a Windows problem, not Ubuntu. And for Windows, too, you can of course try and find solutions on the internet, Google is your friend :)

Topsiho
