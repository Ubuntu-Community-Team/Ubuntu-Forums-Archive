### Post by HermanAB on 2011-12-14
Thunderbird can import Outlook mail with some success, for varying degrees of some.

The better way to do it, is to bounce the mail through an IMAP server.  Create a gmail account.  Hook Outlook to the new account using the IMAP protocol (as well as the old account using whatever protocol you used as before). Select, click drag and drop the mail from the old account to the new account. Do NOT forward the mail.  Click drag drop is the way to do it.

Go and watch a ball game...

Now run your new mail client and hook it to gmail.  Either just use gmail via IMAP from now on and avoid all future mail problems, or download the mail using the POP protocol, if you really want to face the same problem again another day.
