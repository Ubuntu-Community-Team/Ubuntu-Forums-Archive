### Post by andymorton on 2010-12-31
[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

Download the above software, enter which distro you want, the version number and put in the .iso file in the appropriate field. Then press ok or burn or whatever it says. Make the sure the USB stick has been formatted beforehand. 

You could also use the start-up disk creator included with Ubuntu if it's an Ubuntu or Ubuntu-based distro.
