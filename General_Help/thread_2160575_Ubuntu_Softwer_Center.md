### Post by gordintoronto on 2013-07-07
Install VirtualBox with the Extension Pack, install Ubuntu 12.04 in VirtualBox, download the programs you have paid for, copy  the .deb files to a flash drive, install in Mint. Watch out for dependencies: someprogram.deb might require thislib.deb, thatlib.deb, awholebunchoflibs.deb.
