### Post by Therion on 2009-02-24
If you burned the CD *correctly (as an image, not a data-CD) you simply put it in the drive and reboot your PC.  The CD contains everything your computer will need to run Ubuntu in what's called a Live Environment. It will be somewhat slower than usual since you'll only be using the CD however, so expect that.

You'll be able to test most things on your PC, but you won't have multimedia codecs installed or Flash or Java.  You won't be able to download or install software.  This is just a test-drive situation, not a proper install to the hard drive.  

If you like what you see and want to install Ubuntu to your hard drive there will be an icon on your desktop to do that.



*Browse the CD using Windows Explorer.  If there's only ONE file on the CD (ending in .iso), it has NOT been burned correctly and you will not be able to boot from it.  
If you see there are several files and some folders on the CD you have burned it correctly and it should boot your system.
