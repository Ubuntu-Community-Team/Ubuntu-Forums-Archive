### Post by mcduck on 2010-05-06
You might want to first try pressing Ctrl-0 (or select View/Zoom/Reset Zoom from the menu) to reset the zoom settings.

Recent versions of Firefox scale both text and images by default when zoomed, and that will of course reduce image quality. If the zooming is done on purpose then it might be a good idea to just increase the font sizes from preferences, or at least set the zoom to only apply to text (View/Zoom/Only Zoom Text).
