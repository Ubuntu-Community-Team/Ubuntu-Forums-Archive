### Post by nzjethro on 2011-06-13
Boot from a Live CD (the CD you used to install Ubuntu), by inserting it into your CD drive and turning your computer on. If it doesn't load the CD, enter your BIOS menu and change the boot order to put CDROM first. When asked whether to install or try, select try, and it should boot you through to Ubuntu off the disk.
When you're in Ubuntu live CD, run the bootinfo script, and post the results here.
