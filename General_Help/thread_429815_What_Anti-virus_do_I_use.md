### Post by Tomosaur on 2007-05-01
You are very, very unlikely to get a virus using Ubuntu, most people never bother with one. However, there are anti-virus programs available, just click Applications, then Add/Remove, and do a search for anti-virus programs. Many people use ClamAV, others prefer AVG.

It's entirely up to you. If you want peace of mind, then feel free, but the chances of you getting a virus are so slim as to be negligible. There ARE viruses out there for Linux, but the speed with which security holes are updated means that most will only affect really old kernel versions, while the others rely on social engineering tactics (ie, tricking you into downloading and installing the virus personally).

Well, Linux isn't Windows. Windows software does not work on Ubuntu. You need to either download the Linux version of the software, or find a replacement.

There is a chance that you can get the Windows software to work on Linux by using Wine (which pretends to be Windows, so that Windows software will work), but you should not rely on it. You can check [the Wine Application Database](http://appdb.winehq.org) to see whether your software works properly using Wine. Don't mistake no listing for bad news though, it just means nobody's reviewed how well it works. It's always worth giving it a go if you can find no suitable replacement software.
