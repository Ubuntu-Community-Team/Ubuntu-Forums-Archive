### Post by msammels on 2012-07-03
I'm not sure I understand you correctly... are you wanting to automatically log in each session as a particular user? If so simply do the following:

Click the Ubuntu icon, in the top left corner. Type "Users" and click the icon for "User Accounts".

From that window which pops up, make sure to click the padlock in the top right, entering your password when it asks, then simply switch "Automatic Login" from OFF to ON.
