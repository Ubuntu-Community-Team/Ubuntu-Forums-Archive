### Post by lovinglinux on 2010-12-03
Are you referring to the Firefox launcher icon or buttons on Firefox interface?

If you want to change the launcher icon, then go to "System >> Appearance >> Main Menu >> Internet", select Firefox in the list, right-click, select "Properties", then click the logo to choose a different image. 

I wouldn't recommend editing the original icon.

If you are referring to Firefox interface icons, then you can use a different theme, that you can download from [Mozilla Add-ons]("https://addons.mozilla.org/en-US/firefox/search/?q=Theme&cat=all&lver=any&pid=1&sort=&pp=20&lup=&advanced=") site, or you can use user script to modify the interface. You can do that by [editing a file]("http://www.mozilla.org/unix/customizing.html") in your profile or by using [Stylish]("https://addons.mozilla.org/en-US/firefox/addon/2108/") extension.
