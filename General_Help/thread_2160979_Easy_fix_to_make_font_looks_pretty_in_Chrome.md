### Post by BreezyBrooke on 2013-07-09
did you install the redistricted extras package? It contains commonly used fonts on the web. It will also install flash player and media codecs.

sudo apt-get install ubuntu-restricted-extras

after installing, reboot and all the font's online should be fixed. Ubuntu by default dosen't ship these needed fonts because they are owned by microsoft and need an EULA agreement to use and install them.
