### Post by CatKiller on 2009-07-26
The launchers for AWN are .desktop files in ~/.config/awn/launchers. You can specify the icon that gets used in there if the automatic configuration isn't working. You can either use the generic name (such as "thunderbird") in which case the icon will be theme-dependent, or you can specify the full path to the image that you want to use.
