### Post by sisco311 on 2011-09-01
From: [uhelp]community/32bit_and_64bit[/uhelp]

[I]A 32-bit computer has a word size of 32 bits, this limits the memory theoretically to 4GB. This barrier has been extended through the use of 'Physical Address Extension' (or PAE) which increases the limit to 64GB although the memory access above 4GB will be slightly slower. 

A 64-bit computer will be able to address up to 16.8 million TB (16 exabytes) although constraints are in place that limit this to around 1TB.[/I]



EDIT:
In short, the 32 bit PAE kernel will recognize all your RAM, but go with the 64-bit version. :)
