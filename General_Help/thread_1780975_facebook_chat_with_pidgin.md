### Post by sffvba[e0rt on 2011-06-12
I didn't check out the link, but to get Facebook chat to work is very easy.

Username: *yourname*@chat.facebook.com
Password: *your facebook password*

Note that you have to have setup a "handle" for facebook... so you should have access to your facebook account in the format [www.facebook.com/](www.facebook.com/)*yourloginname* (in other words, you have to be able to log into facebook using a user name and not your e-mail address).

Leave all other settings as is... oh and choose xmpp when selecting to create a new account in pidgin...
