### Post by wondering_jew on 2006-02-17
I'd been using easytag for months before I found the feature you are describing, in easytag select the files you want tags for  right click, select cddb search, then go to manual search because the automatic search sucks, enter the artist and album name, search, select an album select the tags you want and apply. I've found it to be a bit buggy everyonce in a while, for each tag you select in the top search window make sure that the corresponding song is selected in the bottom window and give the bottom window time to catch up... you'll know what I mean in you need to...
