### Post by Wim Sturkenboom on 2011-09-04
A modern monitor _should_ not give you problems; Ubuntu 8.04 configured the video correctly when I bought a new monitor (1920x1200) to replace my old CRT.

Ubuntu 10.04 (fresh install to replace 8.04 above) worked properly from the start (or after I installed the correct video card driver, not sure).

This was with a 6200 based nVidia card.

Only requirement is that the video card as well as the video card driver support the resolution.

And you don't need tweaking software. If tweaking is necessary, you can do so manually.

### Post by Duncan Williams on 2011-09-05
One thing...
Get a monitor with built in settings manager!

Most monitors will be picked up by ubuntu and as long as your video drivers are correct, will choose the correct resolution, etc.
However some monitors require you to install `settings manager' software (for > windows)

A lot of monitors have the settings manager built into the monitor (buttons, switches or touch-screen) > get one of these.

I can recommend  AOC LCD Widescreen monitors from personal experience.
