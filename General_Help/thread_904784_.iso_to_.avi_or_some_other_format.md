### Post by mb_webguy on 2008-08-29
Install and use dvd::rip or OGMrip to rip a dvd to an avi or other video type.  I used to use dvd::rip, but have recently switched to OGMrip because of its support of x264 and mkv.  Dvd::rip is fine for ripping to an avi, though.

If you'll notice, OGMrip is mentioned at the bottom of the HOWTO_DVD_to_Matroska tutorial.  OGMrip is a GUI frontend for mplayer, mencoder, ogmtools, mkvtoolnix, mp4box, oggenc, lame, and faac -- the command-line programs those tutorials use.  Of course, as a GUI frontend, OGMrip doesn't provide all of the options available by using the command line tools, but it is more than sufficient for the average user who wants to rip a dvd to a file and retain multiple audio tracks and subtitles.

OGMrip has a very simple interface.  Just load the dvd, select the video stream and audio track (and subtitle track if you're using a container that supports it), and click Extract.  (After you click Extract, you'll be given some options, but the defaults are usually good, and you can simply click Extract again.)  If you want an avi using divx or xvid video and aac audio, make sure you edit the preferences, since the default is mkv using x264/mp3.

### Post by mb_webguy on 2008-09-02
Encoding will take a while no matter what format you choose to rip to.  MKV does tend to take a bit longer than avi, but mkv gives you the option to keep multiple audio and subtitle tracks from the dvd, and results in a slightly better quality video.  But yes, except encoding to take a few hours, even if you're ripping to an avi.
