### Post by tetrafuran on 2007-11-21
Setting keyboard commands have no effect. Something is overriding this tool. What could it be?

update: It seems to be connected to which graphical effects are in use. system - preferences - advanced desktop effect settings.

another update: Certain compiz plugins seem to have the ability to inhibit keyboard commands. For instance in "desktop wall" I am able to use most of the commands I need, while in "desktope plane" switching between the deskstops uses different commads that cannot be chnged. Only in "desktop cube" I can have all the commands I need. At the moment I'm using the cube, but I had to modify it so that it wouldn't be so distracting and annoying. Of course another choice would be to disable all these nice effects (compiz). Not such a bad idea either.
