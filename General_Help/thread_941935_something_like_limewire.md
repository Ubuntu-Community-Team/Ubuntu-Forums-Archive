### Post by fooman on 2008-10-08
frostwire is in the repos...you can get an older version from synaptic (applications > administration > synaptic package manager).  or in a termnial:

```
apt-get install frostwire
```

but like i said...that will be an older version.  you can grab the latest version from here:

[http://www.frostwire.com/?id=downloads](http://www.frostwire.com/?id=downloads)

it is a .deb file,  you should be able to just double-click it to install.
