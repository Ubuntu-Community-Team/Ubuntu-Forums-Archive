### Post by Dennis N on 2014-06-25
This information may get you started:

[http://cipricuslinux.blogspot.com/2012/11/edit-shortcuts-keybindings-in-lubuntu.html](http://cipricuslinux.blogspot.com/2012/11/edit-shortcuts-keybindings-in-lubuntu.html)

Note: You don't need (and shouldn't use) sudo to edit the .xml file in question. Any plain text editor will do instead of gedit (which is not installed in Lubuntu), and you can open and save with the text editor in the normal fashion without the terminal being used.

Good luck.

Are you using PCManFM 1.2? It's significantly better than the previous ones. Please clarify.

Many of the keyboard combinations available are present in the respective menu dropdowns. Press Alt+F (when the file manager window in in focus) to see the "File" dropdown, press Alt+E to see the "Edit" dropdown, etc.
