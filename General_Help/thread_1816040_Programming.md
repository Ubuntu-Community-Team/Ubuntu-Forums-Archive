### Post by LemursDontExist on 2011-08-01
If you have some project in mind that you want to work on, I'd recommend python since it will let you write working programs with sophisticated functionality very easily and quickly.

If you just want to learn to program, and are planning to do a bunch of simple programming exercises, then I'd say start with C.  In the long run learning C will teach you the basics of how programs work, and that knowledge will be useful in whatever language you decide to program in.  

Once you've learned one language picking up a new one isn't hard, so I wouldn't put too much importance on this decision.

I would avoid Java though.  Even the simplest java program tends to be about a dozen lines long, and most of those lines of code make no sense to a beginner.
