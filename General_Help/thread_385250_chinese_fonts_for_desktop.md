### Post by kodoku on 2007-03-15
Default ubuntu fonts don't display all the characters of a Chinese character set.  You can tell when you try to use SCIM in gedit for instance, when you type in the pinyin to pull up the character candidates, you get some letters that are either boxes with numbers/letters in them, or just plain dots.  First you have to install language support (if you haven't already done so) through Menu > System > Administration > Language Support

This should install the chinese fonts that have all the characters.  Then you have to go to Menu > System > Preferences > Fonts, and then change all the fonts to a Chinese font (I personally like AR PL Zenkai Uni, but another option is AR PL ShanHeiSun Uni)

Good luck!
