### Post by geirha on 2009-03-02
The root account is, and should be, locked down. It does not have a password. If you need to run something as root, use sudo. 

[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)

---

### Post by taurus on 2009-03-02
There is a root account but it's locked so you can't log in as root unless you create a password for it.  And since we are not allowed to show you how here, just use sudo/gksudo if you need root privilege.  It works just as good.

[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)
