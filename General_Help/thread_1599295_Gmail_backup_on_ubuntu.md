### Post by spikoley on 2010-10-17
You can use POP to download all the emails.  Just make sure you check leave a copy on the server if you want to keep them online.  Any mail client can do this including Evolution.  Did you set up IMAP instead of POP?  IMAP will only download the subject of the email until you click to open it.  If you don't like Evolution then I recommend Thunderbird.

### Post by btindie on 2010-10-17
Your best option to backup Gmail is to use the IMAP protocol.
