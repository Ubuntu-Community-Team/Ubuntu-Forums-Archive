### Post by bodhi.zazen on 2009-11-16
In general wine is not always so easy to use and in general you are going to get the best information from winehq:

[http://appdb.winehq.org/](http://appdb.winehq.org/)

There is a linux native equivalent, gnomesword

it is in the ubuntu repositories

```
sudo apt-get install gnomesword xiphos
```

Home page : [http://xiphos.org/](http://xiphos.org/)

the project is now known as Xiphos ;)
