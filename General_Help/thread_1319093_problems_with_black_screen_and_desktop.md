### Post by lowpan on 2009-11-08
I had similar, not as bad as yours with the default dark theme.

With the default dark theme, I had problems with some applications, like skype, where the skype menu would be all black and not readable.  It was black on black, so I couldn't read it.  Then I went to System->preferences->appearances, in the theme tab, selected the 'ClearLooks' theme, where the background is whitish.  Then I could see all my windows.

I found the problem to be black font on black background with the default theme.
