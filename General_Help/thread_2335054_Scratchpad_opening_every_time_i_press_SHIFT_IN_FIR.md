in the last few days every time i press shift while doing anything in firefox scratch pad open which is a total pain.
i found a solution dated 2008 from someboy with the same issue, go to about:config and change
[I]devtools.scratchpad.enabled to false.
it worked for the person posting the problem but in my case it was already on false, to be safe i changed [I]devtools.scratchpad.enabledautocompleted to false but didn't do anything, hence the post without capital letters.

looks like the shortcut should be SHIFT + F4 NOT SHIFT + "ANYTHING ON THE KEYBOARD"
Anybody running into that problem? [/I]
