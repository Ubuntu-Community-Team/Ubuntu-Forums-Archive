### Post by starcraft.man on 2009-07-08
I'm not a lawyer, I can't answer technical questions on trademark and licensing. I do know plenty of people and small shops preinstall Ubuntu on machines and sell them.

It's easy to do, I'd recommend you download the alternate install disk for either x86 or 64 bit versions (depending on client preference) and then when you boot up ensure your using the OEM install option, it's somewhere on the boot splash screen.

I think the big problems occur when you start operating a large scale operation like Dell or System76 and then you need to deal with trademark. I'm not a lawyer though, so don't take my word for it.
