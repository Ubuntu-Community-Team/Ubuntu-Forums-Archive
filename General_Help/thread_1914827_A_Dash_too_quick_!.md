### Post by rg4w on 2012-01-31
There is a Unity plugin for Compiz Config Settings Manager which has the option to adjust the delay for the Launcher.  CCSM is available in the Ubuntu Software Center.

When using CCSM you may be tempted to play with other of its many settings.  Often quite fun, but please be aware that some options may conflict with Unity, so unless you're confident about what you're setting you may do well to stick with the settings in the Unity plugin only.

### Post by marin123 on 2012-02-10
Install ccsm. (search for it in Software Center)
Open it. (Alt+F2, then write ccsm and hit Enter)
Go to Ubuntu Unity Plugin.
Set Edge Reveal Timeout to 300 or more (value is in miliseconds, and this is the period that you have to keep your mouse on the left edge to show the launcher).
You can experiment with different values.
