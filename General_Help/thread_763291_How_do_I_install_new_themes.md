### Post by irunwithknives on 2008-04-22
first go to system=>preferences=>appearance.  Drag and drop the file called "aero-clone.tar.gz" into the window. it should ask if you want to install the theme.  Say yes.  Then go to System=>Preferences=>Emerald.  If you dont already have emerald theme manager, install it by searching "emerald" in synaptic.  Then, drag and drop "aero_blue.emerald"into the window.  It should show up in the window.  If the borders change, you are good.  If they dont, then you should press alt+F2 and type in
```
emerald --replace
```
This should make the window borders change
