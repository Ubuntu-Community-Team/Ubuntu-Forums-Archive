### Post by oldfred on 2013-04-13
None.

Some are more challenging than others. The high end UltraBooks have a small SSD and use RAID which causes issues. And have dual video which needs Bumlebee, although nVidia just released the first beta of a new driver that has some support for dual video.

A few will boot Windows and Ubuntu with secure boot on or off, but I have not kept track. Most only boot Windows with secure boot on. If you do not want Windows turn secure boot off. 

Best suggestion I saw was to put hard drive on shelf and buy a new SSD to install into laptop. 

I have been logging systems, but not keeping details. Most of these worked but required a fair amount of effort.
