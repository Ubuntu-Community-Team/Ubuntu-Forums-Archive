### Post by Genius314 on 2008-08-18
Okay. This should work:
Go to your home folder (click on "Places>Home Folder" in the menu).

Now press ctrl+h. This will make all of the hidden folders appear (like program settings and themes).

Find the folder named ".icons". Open it.

Inside you should see the folders for the icon themes that you installed. If you're certain that you don't want them anymore, you can delete them.
