```
Not all updates can be installed
```

---

### Post by todak on 2009-05-11
More than likely you have the proposed updates enabled. Nothing from there will update if there will be a conflict with your installed system. I'm guessing it has to do with a kernel update, but the restricted headers, etc. are not yet in the proposed updates and therefore, will be held back until all of the pieces are available. Uncheck the proposed updates in software sources in the updates tab and the message will go away.
