### Post by PaulW2U on 2018-09-01
Are you aware of GNOME Shell's extensions? They add additional functionality that you might think has been omitted by GNOME's developers.

You can install a selection of them from Ubuntu's repositories or browse hundreds of them online at [https://extensions.gnome.org/](https://extensions.gnome.org/). The [Applications Menu]("https://extensions.gnome.org/extension/6/applications-menu/") extension is probably what you are looking for. You will need to install "chrome-gnome-shell" if you don't already have it in order to install extensions from that site.
