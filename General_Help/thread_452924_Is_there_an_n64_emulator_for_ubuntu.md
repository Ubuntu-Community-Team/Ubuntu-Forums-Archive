### Post by crhylove on 2007-08-31
There is a GREAT Nintendo 64 emulator that works nearly PERFECT in wine.

I'm on Gutsy 64 bit, but I imagine it will run just as well in Feisty.

The emulator is a continuation of the Mupen emulator, and (as of right now) there is no native port (though it is open source, so there SHOULD BE eventually!!), but the controller, sound, video, and most of the games work perfectly in this emualtor.

[http://okaycreations.com/mupen64/viewforum.php?f=4](http://okaycreations.com/mupen64/viewforum.php?f=4)

To save the rest of you tons of headache and testing:

Mupen .5 (native) will not play games in 64 bit.  It does work pretty good for 32 bit though.

Under Wine:
Project 64 works well, but the controllers do not (wine issue).
1964 will crash mostly, there is a work around, but then you still don't have working controllers (same wine issue).
Mupen .51 for windows works poorly at best.

Hopefully somebody will package up that source for the new Mupen++ and give us a .deb, but until then, as I said, it runs nearly flawlessly in Wine, and the controller works perfectly as well!
