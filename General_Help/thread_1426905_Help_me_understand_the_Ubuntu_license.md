### Post by coffeecat on 2010-03-11
> **coilerman said:**
> And if there is no extra fee for the LTS version, why doesn't everyone download the LTS version?

To clarify this:

There is a new release every 6 months (April and October), but an LTS only every 2 years. The version number = release-year:release-month, so Karmic Koala = 9.10 = 2009:October.

The last LTS was Hardy Heron, 8.04, and the next one will be Lucid Lynx, 10.04. The intervening ones, 8.10 = Intrepid Ibex, 9.04 = Jaunty Jackalope, and 9.10, were all ordinary 18-month supported versions.

If you need stability - mostly in the enterprise - you go for the LTS version. The disadvantage of that is that app versions don't get upgraded. They get bugfixes and security patches, but not version upgrades. So people running Hardy Heron will be using an older version of Open Office, for example (unless they install a later version from outside the Ubuntu repositories). This is an advantage in the enterprise - it costs money to train people for newer versions of software.

But if you like the latest and greatest bells and whistles in the OS and the latest and greatest versions of your favourite apps you go for the 18-month supported releases. Many people here upgrade or reinstall every six months with the new release.
