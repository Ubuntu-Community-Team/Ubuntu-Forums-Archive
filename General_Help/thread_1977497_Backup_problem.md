### Post by HermanAB on 2012-05-10
Short answer: No.

The Windows backup program typically doesn't even work on another Windows machine, never mind on a Linux machine.

Long answer: Maybe.

Google for "windows bkf file on linux".

The problem is that there are multiple Windows backup file formats, hence why it typically never works even on Windows.
