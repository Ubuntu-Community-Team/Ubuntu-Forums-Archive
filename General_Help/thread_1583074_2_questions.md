### Post by sanderd17 on 2010-09-27
You'll get a notice when you can upgrade ubuntu via the regular software updates. If you did weird things with your ubuntu installation, it's possible the installation will break on some things. 

If your current install is a wubi install, the upgraded via the upgrade manager will be a wubi install too. 

If you wan't to see how the new ubuntu looks like, how it feels ootb, you need to do a fresh install. With an upgrade, ubuntu keeps the configuration changes you made to your installation (like wall paper, theme, menu's ...).

If you're not too curious, it's best to wait a little before you install it. If you wait until end october, the servers will be faster and some upgrade bugs will be fixed.
