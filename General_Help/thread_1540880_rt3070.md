### Post by hhh on 2010-07-29
Wicd is another network manager aimed at wifi connections. Install it via Synaptic or

```
sudo apt-get install wicd
```

It will remove network manager and you'll need to reboot.

Before you do that though, have you tried other encryption settings? Can you disable encryption on the router? I'm looking through other threads for solutions...
[http://ubuntuforums.org/showpost.php?p=9177505&postcount=3](http://ubuntuforums.org/showpost.php?p=9177505&postcount=3)
