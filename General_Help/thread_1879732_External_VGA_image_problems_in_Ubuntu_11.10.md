Hi,

I have recently updated my HP Pavilion laptop to the 64bit version of Ubuntu 11.10. I use the laptop as a media centre connected to a projector via the VGA port (there isn't a digital output). It appears to shimmer and shake slightly, and this is especially prevalent towards the bottom of the screen. I connected a netbook running ubuntu 10.10 to the VGA cable, and the image was stable, so I know it is not interference in the cables or a problem with the projector. I also tried changing the refresh rate from 60.0Hz to 75.0Hz and there was no change to the image.

The graphics card details from lspci are:

00:02.1 Display controller: Intel Corporation Mobile GM965/GL960 Integrated Graphics Controller (secondary) (rev 0c)

I've tried searching through previous forum posts but couldn't find any solutions. Does anyone have any suggestions for sorting this problem, as I use the laptop for all my media so would really like the image to be stable.
