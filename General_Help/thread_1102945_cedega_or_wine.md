### Post by CatKiller on 2009-03-22
I've never felt the need to run Cedega. As I understand it, Cedega is just Wine with some licensed copy-protection stuff. If it doesn't work in Wine with a no-cd patch then it probably won't work in Cedega either.

You can see which games work in Wine by checking their [Applications database]("http://appdb.winehq.org/").
