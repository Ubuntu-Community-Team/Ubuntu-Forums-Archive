### Post by joshsmith on 2008-05-14
hey
if you go into 
system>preferences>keyrings
on the first tab there is option for the location of application passwords, set it to login

if this doesnt do what you want, delete the current keyring (perhaps after making a note of the keys)
next time you enter a password you will be prompted to create a new keyring, then use no password. and it will unlock properly

if you want it to have a password it is still a good idea to delete you old keyring and start a fresh, as ubuntu now has a new (better) system for keyrings
