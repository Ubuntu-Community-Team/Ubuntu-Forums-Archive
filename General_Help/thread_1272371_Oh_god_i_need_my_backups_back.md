### Post by rbc on 2009-09-22
Photorec will not recover a folder, nor will testdisk, however, the list of file types it will attempt to recover is quite extensive, and certainly not just documents. Two things to mention though....do not try to recover them to that same hard drive. Secondly, the file names will not be what they were originally, so you are going to have to manually sift through all of them. Fortunately, photorec does tend to group file types together
