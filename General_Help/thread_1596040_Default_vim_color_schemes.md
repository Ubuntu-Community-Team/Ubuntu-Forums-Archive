### Post by JohnElway on 2010-10-13
Back when I was on Windows, I used gVim to do a lot of my text editing. I remember that there were a set of default color schemes that I could select from the menu. I just installed Vim (CLI only) on Ubuntu, but I have no way to see what the included schemes are. I remember the name of one that I used frequently on gVim ('pablo') and was able to set that in my .vimrc file. However I would like to try the others that come with Vim. Can anyone tell me how to find the different color schemes that are available by default?

---

### Post by JohnElway on 2010-10-13
NM, just figured it out 2 seconds later. Open Vim, type ":colorscheme", and hit TAB repeatedly to cycle through all the included color schemes.
