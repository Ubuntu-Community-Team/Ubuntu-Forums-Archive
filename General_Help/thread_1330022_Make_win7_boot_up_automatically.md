### Post by hellmet on 2009-11-18
Open /boot/grub/grub.cfg
```
#gedit /boot/grub/grub.cfg
```Count the menuentry# for Windows 7 starting the count from 0. 
Or count the Windows 7 entry when you boot.

Example GRUB:

Ubuntu blah blah - 0th entry
Ubuntu recovery - 1st entry
Ubuntu memtest
Windows 7        - 3rd entry

Once you know the menuentry #, do
```
 *sudo grub-set-default X*
```where X is the # you got previously.

More here: [http://ubuntuforums.org/showthread.php?t=1302743](http://ubuntuforums.org/showthread.php?t=1302743)

### Post by hellmet on 2009-11-18
It should. No harm done trying. Try

 ```
sudo grub-set-default 4
```since Windows 7 is your 5th entry 0,1,2,3,4.

You might have to re do this whenever you upgrade your kernel since that adds a few entries to the menu. I say this, since I see you're still running -14, when -15 is the latest in the update pipe.
