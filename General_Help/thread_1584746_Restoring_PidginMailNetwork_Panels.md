### Post by BanannaButt on 2010-09-29
I accidentally deleted the main top panel on Ubuntu that shows all the Pidgin/Mail/Network notifications.  I restarted Pidgin, and the notification icon was appearing on the bottom panel. Here's how I reconfigured it so that the panels appeared as they used to (ie, Pidgin/Network/Mail notifications on top?

1, Right-click on top, select add to Panel
2, Select Indicator Applet, followed by Indicator Session Applet

Hope this helped any noobs out there!
