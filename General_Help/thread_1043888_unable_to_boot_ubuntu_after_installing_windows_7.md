### Post by theWrkncacnter on 2009-01-18
There is a great freeware utility called [Super Grub Disk]("http://www.supergrubdisk.org/") that can restore GRUB for you. Boot up their live disk, and it is fairly straightforward from there.

Once you have the grub menu back, the windows entry will take you to the windows bootloader giving you the choice between Vista and 7.

If you have any problems with SGD post them here and we can work from there if you want.

Hope that helps!
