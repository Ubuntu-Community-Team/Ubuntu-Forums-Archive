### Post by TheFu on 2021-12-20
If you've decided to block parts of a website, the website has no mandate to work. I block about 300K tracking sites and domains. Sometimes, the websites work fine and other times, they don't work at all.  Then I'm left with a choice - to relax my rules or no access the website in that way.

You can try running in Incognito mode and accessing the website
or
you can create a new userid and try accessing the website using that userid (logout and login as the new user)
or
you could try using a different browser and try accessing the website (try dillo, lynx, chromium)
or 
you could try running your browser inside a constrained environment (I use **firejail**) and try accessing the website
or
you could try to use a URL downloader like pandoc or wallabag to gain access to the website, without the junk. Sometimes these work and sometimes they don't.
or
.... probably 50 other things to try.
