### Post by Jonam on 2007-03-28
Minidisc is not supported under Linux by Sony so there is no application to do what you are after. At best, you can run windows under Linux using VMware and then run SonicStage in this environment. There is a thread on these forums that links to a tutorial on how to do this.

[http://ubuntuforums.org/showthread.php?t=7913&highlight=minidisc+vmware](http://ubuntuforums.org/showthread.php?t=7913&highlight=minidisc+vmware)

Hope this helps. I'm stuck with dual booting Windows and Ubuntu as Sonicstage is the only application that is preventing me moving completely to Linux.

Jonam

### Post by SteinbergerNate on 2007-07-27
Hello. I too have suffered the lock-in that Sony has put on it's md and hi-md users. Using VMware has really worked well for me. It recognizes the hi-md and works flawlessly for me. I have my music folder set up as a smb share and simply access my music from sonicstage over a virtual network so I can still use Rhythmbox to listen on my computer.
