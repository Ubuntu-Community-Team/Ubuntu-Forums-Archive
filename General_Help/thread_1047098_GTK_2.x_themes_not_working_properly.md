### Post by onero on 2009-01-22
Try installing fusion-icon; you can change the Window Decorators / Managers using that, just choose the Metacity Window Decorator. For the themes, check the theme homepages what the required theme engines are, if any. Engines such as pixmap are I think installed by default, but some themes require engines such as Aurora or the latest SVN version of Murrine. I believe there are .debs available for those as well.
