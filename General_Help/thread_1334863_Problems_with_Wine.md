### Post by Scott753 on 2009-11-22
Wine is notoriously buggy when it comes to running 3rd party software. Even with the usual windows programs it can sometimes malfunction. 

If I were you, and needed to play Windows-only games, I would either have Windows installed as a dual boot, or I would use some VM like Virtualbox which can simulate the Windows OS.

However, VM's are taxing on your computer's resources. A recent review I read described it as halving RAM and OS, since you're basically running two OS at the same time.
