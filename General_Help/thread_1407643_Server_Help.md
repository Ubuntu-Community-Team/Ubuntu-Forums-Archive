thats a pretty generic question.  what is it you are wanting to know?  what do you mean by will it do anything to your computer?

to help clear the air, because there are about 100 new posts a day about server edition - everyone seems to think that the word server means something extremely special (special hardware, connections, OS, etc).  a server is any machine that provides a service to a network (beit LAN or WAN connectivity), this can be some random desktop you dug out of the trash or one of the multi-thousand dollar servers that companies buy for 24/7 uptime and support.

there is nothing special about a server OS aside from being OPTIMIZED to run services rather than desktop applications (but this does not mean it will not run desktop applications).

ubuntu server edition has no GUI which means you will need to familiarize yourself with the command line or utilize webmin to set up your server.  if you plan on installing a desktop gui then you might as well just install the desktop version of the OS.

by the way:  the search button DOES work on these forums and there are literally THOUSANDS of posts on this very same EXACT subject.  plenty of reading available for you

I think the question comes from the fact that many people are migrating from Windows, and with windows there are (substantial) differences between server and desktop editions.

The Ubuntu server edition is optimized for servers. This includes a custom kernel and by default there is no graphical interface. If you wish a graphical interface on a server take a look as on of the many web based interfaces such as webmin.
