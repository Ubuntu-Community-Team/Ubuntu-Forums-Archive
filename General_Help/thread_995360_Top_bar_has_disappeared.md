### Post by tuxxy on 2008-11-27
If you are using emerald the compiz window decorator then you could try the following command, press ALT + F2

```
emerald --replace
```

Open emerald theme manager and select your theme, if you need to download it run this

```
sudo apt-get install emerald
```
