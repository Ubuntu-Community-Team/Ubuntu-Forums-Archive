### Post by StuartN on 2010-04-29
> **screenwise said:**
> My question is how can I keep Synaptic from upgrading to 10.04?

Synaptic will not force a release upgrade. You can switch off the display of release updates in System -> Administration -> Software Sources, open the Updates tab and select "Never" for Release Upgrades.

You can probably reach the same option within Synaptic.

### Post by stoneage on 2010-04-29
If you are running 9.10 you will only get updates for 9.10. 

If you are worried about inadvertently upgrading, go to System -> Administration -> Software Sources -> Updates and for Release upgrade choose Never.
