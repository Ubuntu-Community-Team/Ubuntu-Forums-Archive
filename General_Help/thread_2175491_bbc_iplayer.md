### Post by coldraven on 2013-09-19
Nope! No Silverlight needed. I watch iPlayer all the time just using Firefox.
However if you want to download programs I do not think there is an iPlayer program from the BBC that is Linux compatible.
I just use get_iplayer, a command line utility that lets me download for viewing later.
```
sudo apt-get install get_iplayer
```
Read the documentation here:
[http://linuxcentre.net/getiplayer](http://linuxcentre.net/getiplayer)
