### Post by tredegar on 2011-04-03
I think you'll find it is called **computer-janitor**. You can run it from System-Administration-Computer Janitor.

If it is not installed, fire up System-Admininstration-Synaptic, and install it.

From the command line, you can run it as **computer-janitor**

You may regret having run it when you have found it.
