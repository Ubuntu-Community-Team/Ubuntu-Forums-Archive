### Post by David006 on 2011-11-07
Ubuntu 10.04 LTS ('long term support') aka 'Lucid' will give you the least problems.  It will also remain unchanged for another year or more.

After you install 10.04 LTS, you need to update to the latest patches, improvements, etc.  This will take a little time.  The very latest version to download (10.04.3) already has ALL updates up until 2011-Jul-20.  So you get to the same place, but faster.


What I would actually recommend is to do a conservative install of**Ubuntu 10.04.3 LTS (Lucid)**, 32bit, AND then try an install of **Ubuntu 11.10 (Oneiric)** 64bit (alongside).  See if you can live with Unity (3D), oddly placed buttons (eg. minimize, maximize, close) and other 'new' features.  This is the future, but are WE READY !?

It has been much improved in just the last 2 weeks, with most glitches and bugs stomped out.  There are still missing features: editor for launcher menu, more themes (with all icons), and full integration (to the global menu, full-screen view, etc.) for text editor, LibreOffice, etc.
