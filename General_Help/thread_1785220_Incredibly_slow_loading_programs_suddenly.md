### Post by 3rdalbum on 2011-06-18
If you're experiencing a slow computer, you should use System Monitor to check that the CPU isn't running close to capacity. Sometimes a buggy program can take up most of your CPU's time, causing the rest of the system to run slowly. Find that buggy program and kill it.
