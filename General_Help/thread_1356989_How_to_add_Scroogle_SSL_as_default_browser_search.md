### Post by User3k on 2009-12-16
I thought I would cut and paste this for anyone who is interested or might be. I always use Scroogle myself. For those unfamiliar with it. They delete access logs within 48 hours. Plus they have an SSL search which is nice.

Cut from their website and pasted here:


> *Browser support:   Firefox — many Sherlock and OpenSearch plugins;  we don't test them 

*Microsoft IE7 / IE8  — paste [http://www.scroogle.org/cgi-bin/nbbw.cgi?Gw=TEST](http://www.scroogle.org/cgi-bin/nbbw.cgi?Gw=TEST) in their box 


*Opera  —  Tools > Preferences > Search > Add. Pick a new keyword ( "x"? ) and for the "Address" 
use [http://www.scroogle.org/cgi-bin/nbbw.cgi?Gw=%s](http://www.scroogle.org/cgi-bin/nbbw.cgi?Gw=%s)   Now x search terms should launch a search 


*Chrome  —  wrench > Options > Default search Manage > Add. Paste the URL shown for Opera, 
or better yet, use [https://ssl.scroogle.org/cgi-bin/nbbwssl.cgi?Gw=%s](https://ssl.scroogle.org/cgi-bin/nbbwssl.cgi?Gw=%s)  (***_Chrome phones home a lot_***)
