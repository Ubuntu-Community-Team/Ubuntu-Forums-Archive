Hello everyone
Well, I needed 'OpenOffice.org Math' to edit some equations for a paper but I couldn't find a launcher that run that program so I thought it wasn't installed. I tried installing it but Ubuntu told me that program was already installed.
I realise that I wasn't the only one with OpenOffice.org Math installed but  no launcher to be found so I figured out how to make one:
Go to:
System > Settings > Main Menu

Once you're there Add a new launcher to the 'Office' menu with the following properties:
Name: OpenOffice.org Math
Command: ooffice -math %U

Then you select any Icon you want, OpenOffice.org icons are located here: /usr/share/icons/Human/48x48/apps/ooo-template.png.
