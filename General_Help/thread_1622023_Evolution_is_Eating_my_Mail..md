### Post by Topher88 on 2010-11-15
Evolution is fetching my email from the sbcglobal account that it is set up to fetch from, but it is not showing me the messages.  I've noticed that I haven't been getting any mail this last week, but I (stupidly) blew it off until my dad mentioned sending me an email that I never received, at which point I did a little test and fount that if I send an email to myself and log into Yahoo to check it, it's there, but if I open Evolution, it will "fetch" the mail, but not show any incoming messages, and then the message will disappear from my Yahoo inbox.  I won't ever see messages unless I check Yahoo first, which defeats the purpose of using evolution.

It started doing this randomly; it used to work fine, I just randomly stopped getting my mail recently.  Hopefully I didn't miss anything too important.  

So two questions: What is causing this, and is there any way for me to get my missed mail back?

Any ideas?  All help is greatly appreciated!

### Post by jacob.david on 2010-11-16
In Evolution Edit > preferences and in the "Receiving options" tab there is an option called "leave message on the server". If you check that option then a copy will be left in the yahoo mail. This will only work if you are using POP (I believe yahoo only provides POP access)
