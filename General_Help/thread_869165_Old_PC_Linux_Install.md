### Post by kdnewton on 2008-07-24
You sound like you'd be comfortable with a variant of Ubuntu. I'd recommend Xubuntu ([http://www.xubuntu.org/](http://www.xubuntu.org/))

I've got an old 350MHz PC w/ 64MB-RAM sitting around and I've had luck with Xubuntu in the past. You can still do without the command-line if you choose and being a slightly older PC you'll have better luck with Xubuntu rather than regular Ubuntu. Xubuntu is meant to be run on older PCs.

There are other options if you want to get away from Ubuntu and variants.

[Edit: I agree with ilovelinux33467 that you should use the [Alternate CD]("http://mirror.anl.gov/pub/ubuntu-iso/CDs-Xubuntu/8.04/release/xubuntu-8.04.1-alternate-i386.iso") because I didn't have much luck running the LiveCD on my old PC.]
