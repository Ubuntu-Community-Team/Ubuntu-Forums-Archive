### Post by scott_g on 2009-01-11
> **Lukasz Tarkowski said:**
> I have installed Ubuntu on Virtual Box successfully.
I have used Vista Virtual Box
Now How do I open the screenshots I taken?
I have used Vista Virtual Box
They have been save as sav and vdi,  I dunno what to do about that

Are you running Ubuntu in a Virtual Box on a Vista machine?  When you took the screenshot, did it come up with a message asking you where to save it? (in this case, the virtual box took the screenshot)  If so, 

1. Click somewhere other than the VirtualBox window
2. Press the Print Screen button on your keyboard.  
3. Open up word/gimp/photoshop/an image editing program
4. Press ctl-v to paste.  

It should paste the screenshot in the program.  These instructions are assuming you are running Ubuntu on Vista.  They change if you're running Ubuntu on Ubuntu...

Scott
