Hello touch screen users, 

If you would like to get multi-touch working please follow the instructions on the folling link:
[http://ubuntuforums.org/showthread.php?t=1252492&page=128](http://ubuntuforums.org/showthread.php?t=1252492&page=128)

I had difficulty configuring ginn, so this post is to save you a bit of time. To configure ginn with your own gestures, you need to edit the wishes xml file. This file is in three locations: /etc/wishes.xml, /usr/share/ginn and home/ginn/etc/wishes.xml. The only file that seems to be read is in the /usr/share/ginn folder. Explanations on gestures, and outputs are in home/ginn/etc - attributes, buttons and keys.
