### Post by Rodney9 on 2012-02-12
Lubuntu 11.10 comes with [Audacious]("http://audacious-media-player.org/"), it will play Internet Radio stations.

For just Internet Radio stations there is [Radio Tray]("http://radiotray.sourceforge.net/") in the Software Center.
