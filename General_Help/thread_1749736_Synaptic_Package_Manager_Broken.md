### Post by Tone303 on 2011-05-04
..... I found a solution that fixed it

when i went to adobe website and installed adobe using APT option on its drop down bar, in then forced the canonical into "Other Sofware" tab to exist in synaptic package manager, meaning i can now install other things such as skype, sun java, VLC media player

so simply installing adobe flash forced it there.

problem solved
