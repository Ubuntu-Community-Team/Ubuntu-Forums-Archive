### Post by rsambuca on 2009-02-05
To have it run on startup, you should be able to just add it in the "sessions" section.  Go to System -> Preferences -> Sessions, and then add it manually.  It should then autostart when you login.

As far as the System Tools menu, can't you just right-click on the top menu bar (to open the Alacarte menu editor) and add it?
