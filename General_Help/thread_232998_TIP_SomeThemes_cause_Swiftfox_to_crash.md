### Post by Psquared on 2006-08-09
There are some Firefox themes that claim compatibility with Swiftfox but will cause it to crash. I went through this 3 times last night and I could not figure out which one it is. It will say it installed properly, but if you select it Firefox won't start. Neither will Swiftfox.

Just a warning to stick with the most popular themes. If you happen to get a bad one, go into ~/home and select show hidden files. change the name of .mozilla to .mozilla_backup then restart. Swiftfox and Firefox will start back up but you will have to reinstall all themes and extensions. You can import your bookmarks from the .mozilla_backup file.
