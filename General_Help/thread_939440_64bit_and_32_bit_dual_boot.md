### Post by thefishki345 on 2008-10-05
Hi, I want to dual boot ubuntu 8.10 32 bit beta, as well as my existing 8.04 64 bit installation.

I downloaded the iso, burnt to disk, inserted it, rebooted computer, but it didnt come up with the disk menu, it just booted to the ubuntu startup screen. So, my question is how do i dual boot 2 types of ubuntu (32 and 64) temporarily (as it is for a test, I want to be able to easily remove the 32 bit one without losing my files)?

thanks.

---

### Post by mikewhatever on 2008-10-05
Make sure you've burnt the iso file correctly and set the bios to boot from the cdrom.
When installing, you'll need a separate partition for 8.10. The only thing that's safe to share is swap.
