### Post by Diabolis on 2008-08-24
Compiz is supposed to be able to do this. Its called **Trailfocus**.

You need CCSM, if you don't have it yet:
```
sudo apt-get install compizconfig-settings-manager
```

It will appear here:
**System/preferences/Advance Desktop Effects Settings**
