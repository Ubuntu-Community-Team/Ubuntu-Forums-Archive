### Post by gingermark on 2006-04-28
_my_ answer is that Ubuntu and Kubuntu share the same base. As such, you can add kubuntu to your current installation (from the internet and from the CD) OR do a fresh install,

So the answer is both, which I think is what you got before.

If you wanted to install kubuntu now do
```
sudo aptitude install kubuntu-desktop
```

and when asked choose KDM if you think you might use KDE more, or GDM if you think you'd use GNOME more.

Then restart X, and choose KDE from the sessions menu.

Or you could just do a fresh install... :)
