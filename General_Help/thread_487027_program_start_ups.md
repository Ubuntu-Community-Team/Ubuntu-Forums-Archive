### Post by Metacarpal on 2007-06-28
If you still have the program open at the time you log out (that is, if you don't close everything you have open), Ubuntu saves the session information.  When you log in again, it automatically reopens the applications you left open when you logged out.  In some ways handy, in others a complete nuisance.

If you've closed everything out, check your session settings in the System menu > Preferences > Sessions.  Make sure Opera isn't in your startup programs list.
