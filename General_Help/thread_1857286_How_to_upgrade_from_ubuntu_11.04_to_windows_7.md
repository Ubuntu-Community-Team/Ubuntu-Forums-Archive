### Post by papibe on 2011-10-10
This is what I do:
- Backup your data.
- Install Windows 7 using all the space wiping out Ubuntu.

Optional (or later):
- Install Ubuntu as a Dual Boot:
[INDENT]- Shrink your partition using Windows 7's [Disk Management]("http://www.howtogeek.com/howto/windows-vista/resize-a-partition-for-free-in-windows-vista/").
- Install Ubuntu[/INDENT]
- Or as an alternative install Ubuntu inside Windows using the [Wubi]("https://wiki.ubuntu.com/WubiGuide") installer.

I hope that helps,
Regards.
