### Post by p_quarles on 2007-09-03
You didn't list a wireless card, so I'm assuming this is a desktop system. 

I'd recommend the 32-bit edition of Ubuntu 7.04. It will work fine on a 64-bit processor, with only minor performance loss. The main reason for a 64-bit OS is that it will support more than 4 gigs of RAM, which you don't have. And, from what I hear, Ubuntu 64-bit works, but requires some extra tweaking to get it there. 

Of course, the first thing to do is download and run the live CD, and use that to test your hardware, and see if you like the system. Good luck. :)

### Post by PmDematagoda on 2007-09-03
My advice is that if you have time and a good internet connection to try both Ubuntu and Kubuntu and see which one you like best or do what I did, install KDE, Gnome and Xfce together in one OS though if you do decide to do this the largest problem I've seen so far is that the menu gets all cluttered up with the applications of KDE, Gnome and Xfce which I can handle by making the necessary changes to my menu, there are also some other problems which I haven't faced personally, but I think you can choose to delete them later on if you wish. I'm asking you to make your own decision because I personally don't like KDE and like Gnome better, so you can see that there would be an argument between me gtdaqua which is why it's better if you make your own decision as to what you want to use, KDE, Gnome or Xfce.

On the other issues you have I could help you, remember that there is only a slight chance that software may not work on certain desktops while working on others(though I have yet to find them, may be they are non existent), and also that usually applications built for different desktop usually work on other desktops as well.

Once again the desktop you want is your own decision.
