### Post by 3rdalbum on 2011-02-25
The netbook edition of Ubuntu 10.10 comes with an early version of a desktop interface called "Unity". Parts of this desktop interface are a little slow in 10.10. You could run the classic Gnome desktop instead - at the login screen, click your username and at the bottom choose the classic Gnome desktop from the popup menu.

If the whole computer is running slowly, then you might have a program running that is using up all your CPU time. Check the system Monitor to see if there is a program using a lot of your CPU.
