### Post by _Purple_ on 2009-04-17
Add the following line to your /etc/fstab, save and reboot. See if it works
```
/dev/sda1 /media/windows1 ntfs-3g defaults 0 0

```

Please backup your /etc/fstab before adding it, just in case anything goes wrong. To backup:
```
sudo cp /etc/fstab /etc/fstab_backup
```
