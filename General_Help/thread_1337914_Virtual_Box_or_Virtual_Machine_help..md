### Post by *Linuxftw* on 2009-11-25
I assume you've already got Virtualbox so run the application and click **New.**

Click **Next**. Name the Virtual Machine whatever you want.

Put in the OS Type, in your case it is **Windows XP.**

Choose base memory size. **512mb** should be fine.

Now you will have to create a new hard disk.

Choose **dynamically expanding storage**.

Name the hard disk whatever you'd like and choose the size of the hard disk.

Click finish.

Now go to** settings**. Go down to where it has CD/DVD-ROM

Select **Mount CD/DVD Drive**. Make sure ISO image is selected.

Load the ISO image. Click finish.

Now start the Virtual Machine.
