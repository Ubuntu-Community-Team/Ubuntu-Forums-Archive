### Post by Maestro23 on 2007-02-09
#cd = change directory

Ex:
# cd /home/user = takes you to the folder "user"

#pwd = tells you where you are currently at

This would be a good link to get your feet wet: [http://www.linuxcommand.org/learning_the_shell.php](http://www.linuxcommand.org/learning_the_shell.php)
