Hi,

OS - Ubuntu 16.04 gnome desktop

Thanks for your advice.  I forgot there is a small camera icon popup on the right corner of the top menu after evoking kazam.  Clicking the icon popup Start recording/Pause recording/Finishing recording bars(switch).  Checking "Pause recording" recording will be stopped.  Unchecking "Pause recording" recording will be resumed.  Selecting "Finish recording" a file named Kazam_screencast_00000.mp4 will be saved on Videos folder.

It is very easy and handy using it to record desktop working.  I suppose the .mp4 file can be edited on OpenShot?  I don't have OpenShot installed on this PC.  I'll check it another time on another PC having OpenShot running.
