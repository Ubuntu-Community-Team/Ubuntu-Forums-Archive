### Post by audiomick on 2009-12-17
Hallo, I'm still here, but not for long...:)

Your right, you can't delete the root account, and **you do not want to!**

In a Linux system, there is always a root user. Have a look at this:
[http://en.wikipedia.org/wiki/Superuser](http://en.wikipedia.org/wiki/Superuser)

In ubuntu, the root is disabled by default. There is a machanism using a command called "sudo" to allow users in the administrator group (the first user created on the system is always one, others can be added ) to perform administrative tasks.
[http://en.wikipedia.org/wiki/Sudo](http://en.wikipedia.org/wiki/Sudo)

You know when this is being evoked, because you will be asked for your password. I find this is a good solution. One of the advantages is that you can't absentmindedly do something that will break the system. When the password request pops up, it means "think carefully about what you are doing".
What it also means is that it is next to impossible to manipulate the system if you don't know the password.

### Post by audiomick on 2009-12-20
> **jayze said:**
> ..when/ if/ as anyone can suzz an easy way to play dvds I'll not just be a happy bunny...I'll be jumping over the moon

Have you looked at the second link in my last post? The one that ends in Restricted Extras?

In it, there is a link in bold red type that says

"click here to install the ubuntu-restricted-extras package".

which calls up the process apt to go off and get what you need to play DVDs and stuff, as is described in the link. Click on it and let it do what it wants and you should be able to play your DVDs.

Alternatively, you can do it via the synaptic package manager in 
System> Administration> Synaptic package manager. 
You will be asked for a password. This is the one you use to log on.

This is an example of the sudo process that we have already discussed, i.e. you are doing something in the package manager as the administrator from here on, until you close the package manager.

(at this point I should mention that I have to translate all these menu items. My computer speaks German for the benefit of my Girlfriend, so you might have to look for something similar rather than my exact wording.)

You first have to go, in the package manager, to options> package sources
A pop up window will open. On the first tab, "Ubuntu Software", you need to add a tick in the fourth box. It is called "copyright protected software-multiverse" or some such. The important word is "multiverse" Close that up again. It will tell you that sources have changed, and it needs to refresh. This is all ok.

In the top of the package manager window, there is a fast search field. Type ubuntu-restricted-extras in there, and it should appear in the field below. Click on the box to the left of the name. It will change it's symbol. An explanation of the symbol can be found in the help menu. When it is ready to install, i.e. after you have clicked on the box, click on the green "apply" arrow in the task bar. It will go of and get the package and install it.

You can also install "manually" by opening a terminal and typing in the appropriate command. You need to have enabled the "multiverse" package source. That is also possible in the terminal, but I don't know the command. I always use the package manager.

Once the source is enabled, open a terminal
Applications> accessories> terminal.
A nearly empty white window will open. That is the terminal, which is for entering typed commands.

Type in
```
sudo apt-get install ubuntu-restricted-extras
```
it will ask you for your password, which you have to type in blind, i.e. you wont see what you are typing. When you have given your password, it will go off and get the package.

You only need to do **_one_** of these options. I described both the package manager route and the terminal route to show you the two alternative "normal" methods of installing programs. 

In principle, this serves also as an example for everything on a linux machine. You nearly always have the choice, at least on Ubuntu, between doing something via the graphical interface, or in a terminal.

For instance, you open your browser, Firefox, by clicking on the symbol in the task bar. If you open a terminal and type
```
firefox
```
that will open it too. You can start any program from the terminal. The catch is, you have to know exactly what the command is to start the program.

Anyway, install the restricted extras package, and your DVDs should work.
