### Post by xdominex on 2011-08-06
Is the keyboard shortcut you set for that called "MOVE to workspace (right/left/whatever)" or "SWITCH to workspace (left/right/whatever)"? "MOVE to workspace x" moves the current window to that workspace. "SIWTCH to workspace x" switches your perspective to that workspace while leaving the window behind. If your keyboard shortcuts are set correctly, check to see if you have accidentally set the window for "Always on Visible Workspace." To do that, press Alt+Spacebar and ensure that "Only on This Workspace" is selected. I hope this has been helpful.

XDomineX
