After having tried several hours including makding searches on Internet without a breakthrough.  Finally I decided to uninstall the REPO version VLC

$ sudo apt remove vlc

and reinstalled VLC via SNAP

$ sudo snap install vlc

Now VLC is working without promble with picture and sound
