### Post by ruibernardo on 2007-12-01
From [here]("http://www.gnome.org/projects/evolution/"): "Evolution provides integrated mail, addressbook and calendaring functionality to users of [the GNOME desktop]("http://gnome.org/").". 

Evolution is integrated into Gnome, so if you use Gnome, Evolution is installed with it. You still can use Thunderbird if you want - it's hard (impossible?) to uninstall without uninstalling Gnome, just don't use it if you don't like it.
