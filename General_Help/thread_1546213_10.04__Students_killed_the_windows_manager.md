### Post by stinkeye on 2010-08-05
> Is there a command to reset the workspaces to 1 and perhaps that will fix it? If not, I will try replacing the wm somehow???

Alt+F2 and run
```
gconftool-2 --type int --set /apps/compiz/general/screen0/options/hsize 1
```

or
```
gconftool-2 --type int --set /apps/metacity/general/num_workspaces 1
```
