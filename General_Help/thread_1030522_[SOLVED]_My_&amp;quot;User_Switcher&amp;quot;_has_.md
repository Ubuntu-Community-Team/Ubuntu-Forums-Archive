### Post by craigwilson on 2009-01-04
Hi,

I recently tried Kubuntu (i.e. the KDE desktop) on top of an existing Ubuntu installation. I tried switching between KDE and GNOME using the "Session" tab, and when I did, and GNOME loaded, the user switcher application (in the top right, on the menu bar) quit unexpectedly and now I've unistalled Kubuntu and returned to GNOME and the GDM login screen, it still hasn't returned.

I should point out the GNOME power manager still works; pressing my power button brings up the appropriate options so that I can shut down cleanly.

Any tips for what to do?

---

### Post by craigwilson on 2009-01-05
Problem Solved!!

I found that now I no longer had KDM running, I could re-enable the User Switcher by right-clicking the menu bar, click "Add to Panel...", and add it through there. It now works fine!!
