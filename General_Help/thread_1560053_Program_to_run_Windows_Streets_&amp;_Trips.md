### Post by Mark Phelps on 2010-08-24
You don't mention which version, but the WineHQ site lists anything newer than 2006 as Garbage -- meaning, it's not going to run regardless of anything you do.

If you must use this app, your only choices are to use virtualbox to install a virtual version of an MS Windows OS -- and in that case, you will need a full installation CD/DVD and a legitimate license key.

Or, you can continue to dual-boot.

I continue to need MS Windows apps on a daily basis and find that the combination of dual-boot, together with a shared NTFS-formatted data partition, suits my needs.
