### Post by adamkane on 2006-04-23
This is probably one of the top 5 most asked questions.

First decide if you need php5/mysql5 or php4/mysql4. Once you decide, find a HOWTO, and stick to it. Once you install something, it is very difficult to uninstall and install a different version. Be prepared to re-install Ubuntu, if you make the wrong choice.
