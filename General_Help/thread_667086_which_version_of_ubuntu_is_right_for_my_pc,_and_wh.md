### Post by Craigus on 2008-01-14
Is your CPU core duo, or core 2 duo ?

Core 2 CPU's are 64 bit (so are standard core CPU's, internally, but it gets complicated).

If your CPU is core 2, you could use 64 bit ubuntu. You can also use the standard 32 bit version if you wish.
