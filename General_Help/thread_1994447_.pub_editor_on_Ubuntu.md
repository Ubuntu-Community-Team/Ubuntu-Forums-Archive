### Post by forrestcupp on 2012-06-02
As much as I hate to say it, it can't be done. MS Office will install with Wine, but Publisher doesn't work. The only way you can do it is to install Publisher in a virtual machine running Windows. There's absolutely nothing else out there that will edit .pub files.

### Post by forrestcupp on 2012-06-03
Believe me; I've searched high and low. I wish there were a choice. I use MS Office with Wine, and I still had to resort to running Publisher in a Windows VM. 

Probably the closest FOSS program out there to Publisher is Scribus, but it isn't actually compatible with .pub files; it just does some of the same things, minus all of the awesome templates that Publisher has. The only other suggestion I've seen is to use something like [Zamzar's online converter](http://www.zamzar.com/) to convert the .pub to a .doc, and then try to edit it in LibreOffice. I can't testify as to how well that would work because I resorted to using my VM.
