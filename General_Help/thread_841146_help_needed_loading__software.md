### Post by mcduck on 2008-06-26
That's a Windows executable. It's not made for Linux so you can't directly run it in Ubuntu.

However it might work with Wine, which is a nice program that allows running Windows programs in Linux. But there's no guarantee that it works, some programs run fine and others fail completely to work at all..

What I recommend is first trying to find out if there's any Linux program that does what you want to do. If there isn't any, then try if your program works with Wine.
