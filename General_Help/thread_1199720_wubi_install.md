### Post by Monotoko on 2009-06-29
Yeah it does, it is a little slower on the disk reading, but it should run fine just to try with :)

Basically it gives you the option at startup to choose an OS, Ubuntu or Windows. 

It uses the Windows bootloader, so it is also safe to remove ubuntu via the add/remove programs in Windows without affecting your ability to boot
