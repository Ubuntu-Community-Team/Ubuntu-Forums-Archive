Now about your issue...

after you 

```
sudo apt-get install emerald
```

you execute

```
emerald --replace
```

so it sets emerald as the window decorator (instead of compiz-decorator)

then System -> Preferences > Emerald Theme Manager

and import the themes you want from there.

Hope it helps.
