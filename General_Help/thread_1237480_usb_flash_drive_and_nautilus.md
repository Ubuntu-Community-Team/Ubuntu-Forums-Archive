### Post by jeffersonbenson on 2009-08-11
So when I put in my flash drive into my jaunty laptop for the first time, a window popped up asking me what I wanted to do with it. I didn't want it to do anything so I checked the box for do this from now on, and accidentally clicked on open nautilus. Now every time I put in my drive, I'm greeted with a nautilus window, very annoying. Any ideas on how to reset the preference for this particular drive? 
Many thanks!

---

### Post by furuno on 2009-08-11
- Open up Nautilus
- Go to the Edit menu
- Select Preferences
- Under the Media tab, check out the box that says something like "Do not ask or start a program when I insert a media" (sorry I'm on localized version), it should be the first checkbox.
- OK and you're done

### Post by mc4man on 2009-08-11
For usb drives you want to uncheck the "Browse media when inserted " box in  nautilus ->edit -> preferences -> media
