### Post by Frogs Hair on 2011-03-13
Hi ,
The Wubi installer will download 64 Bit by default . You will have no problem using 64 bit . I Win 7 32 bit and Ubuntu 64 bit.

If you want 32 bit the best option would be to download the 32 bit ISO , that way you will have the option to run as a live cd Install with Wubi or if you like Ubuntu try a dual boot . See the guide for another way to get 32 bit.
 
[https://wiki.ubuntu.com/WubiGuide](https://wiki.ubuntu.com/WubiGuide)
