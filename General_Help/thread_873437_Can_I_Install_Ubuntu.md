### Post by jediborger on 2008-07-29
Go back to the download page on ubuntu.com at [http://www.ubuntu.com/getubuntu/download]("http://www.ubuntu.com/getubuntu/download") and underneath the Start Download button there's a checkbox for the alternate install cd.
