### Post by normnmiles on 2004-10-13
I currently have universe enabled and have installed several general programs from it.  No problems so far.  The most comforting thing for me is that universe is "compiled against the libraries and using the tools that form part of main, so it should install and work well with the software in main" unlike the raw debian repos.
