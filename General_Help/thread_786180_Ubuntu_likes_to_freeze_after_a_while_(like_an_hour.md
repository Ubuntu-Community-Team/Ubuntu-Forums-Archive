### Post by stephenjbarr on 2008-05-07
Boot up a live CD and do a memory test. Sometimes instability can be due to bad RAM. My system was becoming unstable, so I downloaded a stress testing package and used it to stress the machine (cpu, I/O, RAM). Afterwards, I removed one of the DIMMs and tested it again. Doing this, you can systematically test to see if one of your DIMMs is bad. Or, if you have only one DIMM, swap it for a different one and see if you gain stability.
