### Post by Dark Hornet on 2007-11-10
--hello...to the best of my knowledge you cannot use VMware for 3D acceleration (games).  However, you can use WINE ([http://www.winehq.org/](http://www.winehq.org/)) to run MOST windows applications.  So to answer your questoin--you can use Ubuntu to run your work Apps, and web design, etc as well.  Be sure to check out the "Add/Remove Programs" as there are MANY linux native applications that can replace (and in many cases does better) than your windows "only" applications....anyway, have fun, and good luck---just ask if you have any questions.
