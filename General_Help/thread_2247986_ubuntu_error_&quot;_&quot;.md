### Post by buzzingrobot on 2014-10-11
The most common causes of install errors are faulty ISO images and/or faulty burns. Where was the image downloaded from?  How are you burning the image?

After you select the "erase disk and install ubuntu" option, where, precisely, do the question marks appear?  Do they appear on that menu, or does another graphic appear?

What is currently on the drive you are trying to install to?  

Run Ubuntu in the "Try" mode. Then, run "Disks" (tap the Windows key and enter "Disks", then press <Return> when its icon appears in the top roght corner). Is the target drive detected?
