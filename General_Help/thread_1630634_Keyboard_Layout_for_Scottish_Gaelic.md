### Post by alzaf on 2010-11-25
Just in case anybody needs to know, this can be achieved by using the 'gb- Extended - Winkeys' keyboard layout. I just confirmation from a member of the Fòram na Gàidhlig site that this layout is sufficient fuor typing in the language. 

[http://www.foramnagaidhlig.net/foram/viewtopic.php?f=5&t=1675#p12077](http://www.foramnagaidhlig.net/foram/viewtopic.php?f=5&t=1675#p12077)

For reference, the grave accent vowels are accessed via the AltGr key. The keys are as follows:

AltGr + a = á
AltGr + e = é
AltGr + i = í
AltGr + o = ó
AltGr + u = ú
Shift + AltGr + a = Á
Shift + AltGr + e = É
Shift + AltGr + i = Í
Shift + AltGr + o = Ó
Shift + AltGr + u = Ú

### Post by alzaf on 2010-11-25
managed to figure it out. If you want to get grave vowel variables, you have to press the ` (key left of the 1 key) and then type the appropriate character key:


` then a = à
` then e = è
` then i = ì
` then o = ò
` then u = ù
` then Shift + a = À
` then Shift + e = È
` then Shift + i = Ì
` then Shift + o = Ò
` then Shift + u = Ù

Please note, if you press Shift + ` you get ¬
Also holding down the ` key and holding down the character key also produces the grave accent vowel. 

If you want to get the ` character you have to click on the key twice.
