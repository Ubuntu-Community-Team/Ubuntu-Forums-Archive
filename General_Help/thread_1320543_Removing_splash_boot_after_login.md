### Post by scouser73 on 2009-11-09
Yes, you can remove the Xsplash, go to Synaptic Package Manager and search for xsplash or x splash and remove it.

### Post by Derspankster on 2009-11-09
For reasons unknown to me I don't see that splash screen on my desktop machine. I do see it on my laptop. The desktop machine runs compiz and the laptop does not. 

You might poke around in Configuration Editor. Might find a switch in there. If it's not installed on your computer, you can find it in Ubuntu Software Center in System Tools.
