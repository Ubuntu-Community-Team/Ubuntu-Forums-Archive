### Post by sanderd17 on 2011-10-17
In Gnome3, you don't just "move those icons around". With gnome 2, this cause my panel to look ugly more than a few times. But in Gnome3, you have to theme it with CSS and JavaScript. So once you themed it, it's where you want it. It's a lot better to maintain this way.

I don't know if this also works in fallback mode (I only use shell), but fallback is a lot younger than shell, so you should give it a bit of time.
