### Post by cariboo on 2010-05-18
AVG is a command line utility, you have to run it from a terminal. Unless you're serving files to Windows computers, there really isn't a need for an av-scanner. Most av-scanners only scan for Windows malware, as so far there are no Linux viruses in the wild.
