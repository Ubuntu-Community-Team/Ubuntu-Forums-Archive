### Post by Cato2 on 2009-09-07
Actually you can still use Kopete in Hardy (the default KDE 3.5.10 version) for a while, but it may stop working at some point soon.

I have been spending some time building the KDE 4.3.1 version of Kopete on Hardy, which does include a fix for this, and is now working quite well.  However, this took a couple of days to sort out and is not that easy to distribute, or to install.

The simplest 'fix', although quite a lot more work, is to upgrade to Jaunty and use the jaunty-proposed repository - see [https://bugs.launchpad.net/ubuntu/+source/kdenetwork/+bug/391763](https://bugs.launchpad.net/ubuntu/+source/kdenetwork/+bug/391763) for the Ubuntu Jaunty bug.  There don't seem to be plans to fix in Hardy, but you might want to comment on that bug to say it's in Hardy as well.
