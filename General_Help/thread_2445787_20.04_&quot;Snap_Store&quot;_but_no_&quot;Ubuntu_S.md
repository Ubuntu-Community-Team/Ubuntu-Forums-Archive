### Post by deadflowr on 2020-06-19
> Are they the same? or does Ubuntu Software have stuff (like gnome tweaks) that snap doesn't?
gnome-software can install snaps, debs, and flatpaks. It only requires the plugins for snap and flatpak to do so.
It can also install locally,
meaning a simple double click on a downloaded deb file should open the software install program.

Snap store can install snaps and sometimes debs, depending on it's mood, or so it seems.
But that's about all it can do. It cannot use the flatpak plugin to install flatpaks like gnome-software can.
And it can't, as far as I can see currently, install local debs.

### Post by deadflowr on 2020-06-19
Which old Ubuntu Software?

There is the old and deprecated software-center which might still be installable.
That's what Ubuntu used for years prior to 16.04.

More recently gnome-software was branded as Ubuntu Software.
Now in 20.04 gnome-software when installed is simply called Software.
See post #3 to install it.

But, imo, both are lacking in terms of fine nuanced package installation and management.
The best gui option for that is always synaptic.
