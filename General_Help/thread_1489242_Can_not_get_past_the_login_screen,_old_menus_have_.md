### Post by Spr0k3t on 2010-05-20
Sounds like an issue with the hard drive somewhere.  I remember about a year ago there were some seagate drives that needed a firmware update for some reason.

When installing windows on a separate partition, you do not have to install windows first then linux.  You can install linux how you want, partition the drive out, install windows to the empty space on the drive, then rebuild the MBR using a grub cd.  There are more details of how to do this on various wiki pages.

If you really must install windows... good luck with 7.  You'll need it.
