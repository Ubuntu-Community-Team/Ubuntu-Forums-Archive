### Post by NightwishFan on 2009-09-14
Ubuntu does not offer new versions of software until the next release, which currently will be this October. It 'freezes' versions, and then only applies bugfixes on them. That way the software is more compatible. If you want, you can manually update to the alpha Ubuntu to get newer versions, or you can find out how to update the software you want in the current version. Perhaps in a PPA.
