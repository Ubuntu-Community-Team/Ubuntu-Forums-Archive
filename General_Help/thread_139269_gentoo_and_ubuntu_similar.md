Gentoo and Ubuntu are not very similar.

### Post by taurus on 2006-03-03
It's like comparing an apple and orange.  Yes, they are both Linux distros but it takes more time to get Gentoo up and running while you can have Ubuntu running in less than half an hour.  For Ubuntu, you use either apt-get (terminal) or synaptic (GUI) to install and update your system while you use emerge to do that in Gentoo.  Once you have configured Gentoo the way you want it, that thing is fast.  I have both running on my computer at home...  :)
