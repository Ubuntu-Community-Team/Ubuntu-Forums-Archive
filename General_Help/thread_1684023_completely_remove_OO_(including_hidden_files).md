### Post by ajgreeny on 2011-02-08
You can easily remove all the application packages in the usual way in terminal or (my preference) synaptic.  If you choose "Remove completely" all the configuration files and folders except those in your home will be gone, which you can delete with ```
rm -rf /home/username/openoffice.org/ 
``` or just use nautilus.
