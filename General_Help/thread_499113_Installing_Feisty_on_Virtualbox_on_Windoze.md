### Post by AndyCooll on 2007-07-12
Firstly, you could dual-boot and have it default to Windows after 3 secs (for instance). That way it would give you enough time to hit some keys, but wouldn't take too long to then move on and hence annoy your missus.

Secondly, with VirtualBox how much memory have you allocated the Feisty image. If it's less than 256mb it probably won't boot into the live CD to begin an install. To install Feisty on my laptops (which have 192mb of RAM) I need to use the alternate CD.
