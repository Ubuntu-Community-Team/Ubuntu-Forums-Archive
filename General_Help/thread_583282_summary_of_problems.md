### Post by Micronaut on 2007-10-20
Hi,

I drop by the ubuntu scene at least once a year and try out the latest distribution, Unfortunately I have not yet found it compatible enough with commonly used hardware to be a viable alternative to Microsoft offerings.

Not a rant, just a post of my problems before I disappear back to Windows, in case other people are having the same difficulties.  I have had a lot of problems using 7.10 with the Intel Graphics Media Accelerator 950 graphics card that came with my Acer TravelMate 4230.  I found the instructions to compile/build a copy of the latest open source drivers on intellinuxdrivers.org very hard to follow as a new user of Linux.  I also found that 7.10 would not save my graphics driver as i810 and continually reverted back to generic vesa drivers.  I was unable to load the i810 driver at all under the 64bit version of Ubuntu 7.10, but I did manage to run it briefly on the 32bit distribution, and attempted to run unreal tournament 2004. This was very slow under opengl and ran at <20fps.  The same game runs at about 35fps at 800x600x32 under directx and xp.

In addition, I was unable to use my router with 128bit wep encryption.  Ubuntu would not connect.  It was fine when I used 64bit encryption, although occasionally it forgot the wep key, but this was not a problem.

The packages and updates through synaptic was very good.

I am going to return to windows xp but will return again for version 8.

Good luck.
