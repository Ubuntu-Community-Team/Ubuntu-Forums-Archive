### Post by sommotommo on 2008-03-05
=D> Hi, finally I discovered the method, which is very simple.

To whom is interested:


[SIZE="1"]**A**[/SIZE]       If not yet, install from repo*** openoffice.org-evolution***: this extension allows *Openoffice* to manage** Evolutions contacts**.


[SIZE="1"]**B**[/SIZE]        Open the ***Openoffice Database*** and import ** Evolutions contacts**.


[SIZE="1"]**C**[/SIZE]        From ***Database*** copy the contacts in ***Openoffice Calc*** where is easy to manage as you wish.
