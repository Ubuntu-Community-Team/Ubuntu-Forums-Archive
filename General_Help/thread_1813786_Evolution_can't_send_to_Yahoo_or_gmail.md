### Post by tom4everitt on 2011-07-28
For gmail, do this (maybe you've already done this, then this post will not be very useful):

In the web interface (gmail.com), go to Mail Settings, and then Forwarding and POP/IMAP. Be sure to activate POP and/or IMAP (I would recommend IMAP). Here you will also find the exact settings for servers, SSL, etc etc. [http://mail.google.com/support/bin/answer.py?hl=en&ctx=mail&answer=75726](http://mail.google.com/support/bin/answer.py?hl=en&ctx=mail&answer=75726)

It has no specific instructions for Evolution, but the generic ones should do fine. 

Good luck!
