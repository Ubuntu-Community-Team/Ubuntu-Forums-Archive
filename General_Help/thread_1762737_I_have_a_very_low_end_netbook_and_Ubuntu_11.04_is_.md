### Post by r-senior on 2011-05-19
Log out, then when you've selected your username on the login screen, look for a menu at the very bottom. Try "Classic with No Effects" and see if that's any better. Alternatively, if you like the Unity interface, install the unity-2d package and select Unity 2D in the same way.

### Post by Larkspur on 2011-05-19
> **alcatraz678 said:**
> I think it is because of the beautiful graphics of Ubuntu that is making it slow. How can I configure Ubuntu to start up as low settings?

You could either use Classic Gnome or Unity 2D: to use Classic Gnome, log out, select your login and, when you are asked for your password, click on "session" and select "Classic Gnome."  Unity 2D needs to be installed from the Software Center; once done, do the same as before, except select "Unity 2D."
