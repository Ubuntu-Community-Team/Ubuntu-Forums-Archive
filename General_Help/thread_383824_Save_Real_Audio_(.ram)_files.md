### Post by aky on 2007-03-20
Hi,

I've also had some troubles until I have found the solution.
However it is easier than i thought.

First you have to download the .ram file to your computer.
Then open it with a text editor.
In the second line you will find a path to a file which is .rm
copy and paste this path to your browser. (without the beginning: pnm:// )
Now you can download the file to your computer and convert it to mp3 (you can find .rm to .mp3 converters)

or copy the path of the .rm file to this site:
[http://media-convert.com](http://media-convert.com)

This is an online converter. The quality is not so good but it is working for me.
