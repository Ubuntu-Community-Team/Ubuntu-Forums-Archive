### Post by chip616 on 2008-06-07
OK, a user in the Kubuntu forum says that the OOo Base module has to be installed before mail merge will work.  Oddly enough, this is NOT part of the standard Kubuntu install.  I used Synaptic to install the Base module, and F4 now works, and View | Data Sources again appears in the drop down menus.

According to the person who gave me the tip, OOo 2.4.0 is still flaky in the mail merge, but with the Base module installed, at least it is there.
