### Post by sudodus on 2016-08-19
If the computer can't read files on disks in the DVD drive, and the computer cannot boot from the drive, I would suspect hardware problems like *gordintoronto* suggests.

But you failed to boot from a USB drive, and we don't know why yet.

- Are there problems with something else (except reading CD/DVD disks) when you run the installed system (for example to read from USB drives)?
- Have you booted from USB before in that computer?
- Which iso file did you flash into the USB drive?
- How did you create the USB boot drive?
- Does that USB boot drive work in another computer?

It will help us help you, if you specify the hardware, at least

- Brand name and model
- CPU
- RAM (size)
- graphics chip/card
- wifi chip/card
