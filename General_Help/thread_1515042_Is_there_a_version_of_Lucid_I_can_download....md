### Post by 3Miro on 2010-06-21
> **CharlesJWelsh said:**
> pre-configured for Intel 855 chipsets?

What do you mean pre-configured? The Linux kernel automatically detects and loads all the drivers for all of your hardware. There are a few exceptions when manufacturers make proprietary drivers (like video cards from NVIDIA and ATI as well as some Wi-Fi devices) and you have to install them from the Hardware Manager after you have installed Lucid. Also sometimes there are no drivers for specific hardware (like some printers), however, I have never heard of anyone having issues with a chipset.

Download Lucid from the main page and run it from a live CD (boot from the CD and select "Try without changes to your computer"). If it works, then it works.
