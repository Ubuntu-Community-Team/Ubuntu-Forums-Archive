### Post by stinkeye on 2009-12-17
[COLOR="DarkRed"]Edit[/COLOR]:This applies to normal ubuntu ...don't know about remix.
Themes use window borders of different widths.
You can resize using alt + middle mouse button
or install a theme using wider borders.

Here's a nice one I use 
[_[COLOR="DarkRed"]Polaris[/COLOR]_]("http://www.gnome-look.org/content/show.php/Polaris?content=93896")
Right click on the downloaded file and *extract here*.
Open system > preferences > appearance and drag and drop the tar.gz files in the window.
You can then select them in appearance > customize > window border
