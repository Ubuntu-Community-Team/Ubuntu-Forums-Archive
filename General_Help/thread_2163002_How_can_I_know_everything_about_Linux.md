### Post by QIII on 2013-07-17
Hello!  I might suggest installing a virtualization solution such as VirtualBox and installing Ubuntu as a virtual machine.  Rather than learning on your host OS, do all of your learning (and inevitable breaking) in the virtual machine.  If you bork the virtual machine, no big deal.  You can either restore a snapshot or simply reinstall.  That will keep you from ruining the host system and getting frustrated by not being able to use your computer.  One of the best ways to learn Linux is to break things and figure out how to fix them.  You will break things as you learn.  That is unavoidable.  So you will have plenty of learning opportunities.  Best wishes!

(By the way -- If you ever learn everything about Linux, let me know!  I'd like to ask you some questions!  ;) )
