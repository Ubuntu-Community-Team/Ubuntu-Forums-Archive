### Post by 2CV67 on 2014-10-31
Oh - OK...

To move the keyboard around:
If using the Small keyboard, as I was - press & hold the "***" key, then press the 4-way-arrows key, then drag the keyboard by the spot in the center.
Laborious, but if you need the smallest keyboard...

If using the Compact or Full keyboards the 4-way arrow key is immediately visible in the right-hand column, so is easier to find and you can drag directly with that key.
Perfect once you have found it!
