### Post by yancek on 2014-08-01
A Live CD whether on a CD or a flash drive is by default a read-only filesystem.  Unetbootin has an option to create persistence for Ubuntu and a few other systems.  There a limits as to how much you can save depending on the filesystem format.  So if you did not select persistence with unetbootin, the easiest thing to do is do it over and select the persistence option.
