### Post by sudodus on 2014-09-01
I think it should run well with Ubuntu.

1. You may need some boot option to get started. See [https://help.ubuntu.com/community/BootOptions](https://help.ubuntu.com/community/BootOptions)

2. Maybe there was an error during downloading. Check that the iso file is correct using md5sum. For more information on checking md5 hashes, please refer to [HowToMD5SUM]("https://help.ubuntu.com/community/HowToMD5SUM").  Once you have verified the md5 hash, you may want to refer to the [BurningIsoHowto]("https://help.ubuntu.com/community/BurningIsoHowto").

You find a list of md5sums here [http://releases.ubuntu.com/14.04/](http://releases.ubuntu.com/14.04/)

3. Maybe there was an error during burning the DVD. Try to burn with the slowest speed possible!

Or try to boot from a USB drive. See [https://help.ubuntu.com/community/Installation/FromUSBStick](https://help.ubuntu.com/community/Installation/FromUSBStick)
