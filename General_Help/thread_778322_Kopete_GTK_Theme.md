### Post by luisromangz on 2008-05-02
There is no such thing as a Kopete gui theme, but you can install qt/kde themes, and use qtconfig-qt3 to select it. There are KDE themes that resemble GTK themes, but I use QtCurve, which is a configurable style for Qt and GTK, so you all apps can use the same style.
