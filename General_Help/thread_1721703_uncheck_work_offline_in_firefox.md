### Post by crossedout on 2011-04-04
You could check the about:config setting.

In a new tab type: about:config, hit <enter>

Type: browser.offline in the filter/search field.

Change the setting to False if it isn't already.  If it is already set to False then I would start to look at your extensions, disable them all and re-enable one by one to see if any of them cause FF to start offline.

-Xout

**Oh I didn't catch it at first that you still don't have access after unchecking it.  In that case, it would seem to be a connection issue as mikewhatever suggested.  Does Chrome or other browser work?
