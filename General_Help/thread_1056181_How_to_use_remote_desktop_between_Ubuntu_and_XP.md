### Post by txcrackers on 2009-01-31
Yes...

Linux to Windows is accomplished by using *rdesktop* (command-line) or a GUI-based program like KDE's *krdc* (don't know what's there for Gnome or XFCE).

Going the other way (Windows to Linux), gets more complicated by the easiest is to use VNC. A little bit of research will turn up all the information you need on that.
