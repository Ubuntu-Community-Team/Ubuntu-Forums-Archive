### Post by kalikiana on 2006-11-09
Up to now, there has been no effort to backport any more current version. And as it seems, there will be no official update.

So I'd suggest you download the graphical installer from [here](http://www.xfce.org/archive/xfce-4.3.99.2/installers) which will compile the RC2 for you automatically, as long as you have the required source. You can either install it to /usr as an update which worked for me or put it into your home directory.
