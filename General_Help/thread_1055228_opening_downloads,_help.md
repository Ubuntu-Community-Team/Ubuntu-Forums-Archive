### Post by jeffjanzen on 2009-01-30
> **Jaded Misanthrope said:**
> Whilst taurus is correct in that Linux does really need an antivirus....

i'm 99% sure that's supposed to say "Linux doesn't really need an antivirus."  it really doesn't.

If you want to install new programs in ubuntu, the easiest way is to use "Add/Remove" from the Applications menu.  next easiest is to find them in Synaptic Package Manager on the System>Administration menu.  if there's something you want that you can't find either of those places, you're getting into slightly more risky territory.  you can install some programs using files that end in ".deb", but only do this if you're downloading from trusted sources.  and finally, if you're somewhat more experienced, you can compile programs from source (usually in the form of a file ending in ".tar.gz").  i would steer clear of that option for now.

pay attention to file extensions (.exe, .deb, .tar.gz). they can tell you a lot about a file and whether or not you're going to be able to use it/install it/read it.

happy linuxing!
