### Post by oldfred on 2012-07-25
Once you overwrote it there may not be much to recover. Writes are somewhat random onto drive.

But I have used photorec to recover totally lost data. It scans drive for anything that looks like a file by type. You do not recover file names. I had saved a lot of text files many times. It found all the old versions, so I never knew if I found the last saved, but with older backup and lots of grepping & comparing files got most of my data back. Very long process, both the scan & file comparing.
