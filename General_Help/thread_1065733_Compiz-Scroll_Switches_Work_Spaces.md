### Post by DevinMcElheran on 2009-02-10
Hey, I'm using a laptop, and sometime I accidentally hit the scroll, which spins my cube around rapidly. This happens frequently. Is there any way to turn this off? Just to stop it from turning when I scroll, but to still be active. Thanks.

---

### Post by pnutzh4x0r on 2009-02-10
If you have the CompizConfig Settings Manager (compizconfig-settings-manager) you can open it under Systems > Preferences > CompizConfig Settings Manager.  In there there should be an entry called Viewport Switcher.  If you select this there should then be a Desktop-based Viewport Switching tab which holds the bindings for the scrolling of viewports (i.e. using your mouse scroll on the desktop).  You can either disable/change the bindings or simply disable the whole plugin.

This is based on Intrepid, so there might be slight changes for Hardy.  Good luck.
