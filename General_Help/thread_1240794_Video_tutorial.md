### Post by P4man on 2009-08-15
installing in virtual box is really simple. If you want a video of how to do a wubi install, you could even use virtualbox, install windows in it, then do the wubi install in vb, while gtk-recordmydesktop records it all. That way you can record *everything*. Though a windows equivalent of gtk-recordmydesktop should work as well, and you van use the vb approach to make video segments of ubuntu, the booting, splash screen,  grub, etc.
