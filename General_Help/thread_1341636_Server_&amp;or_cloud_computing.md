### Post by fluffman86 on 2009-11-29
If you're running Ubuntu 9.10, then the new [http://one.ubuntu.com](http://one.ubuntu.com) is installed by default, which gives you 2GB of free online storage.  You'll pretty much need all Ubuntu machines to do this though, although you can at least access the files from any web browser.

You could also set up your own server or purchase server space (I recommend [http://site5.com](http://site5.com) ) but it would be just as or more difficult to keep files synced up as it is now with a flash drive.
