### Post by Cheesemill on 2012-07-03
Get it from the official website:
[http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop)

Are you sure you created the CD properly? Instructions are here:
[http://www.ubuntu.com/download/help/install-ubuntu-desktop](http://www.ubuntu.com/download/help/install-ubuntu-desktop)

You also need to make sure your BIOS is set to boot from CD.

As for applications you shouldn't be downloading them from a website, just install them using the Software Center.
