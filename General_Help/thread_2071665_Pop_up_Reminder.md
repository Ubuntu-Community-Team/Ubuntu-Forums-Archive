### Post by kcxzero on 2012-10-16
Do you use Google Calendar? If so it works great with Thunderbird. Add an extension to Thunderbird like G-Hub Lite so you can pull up Google Calendar. And set a remainder to pop-up for an appointment.

Otherwise a quick browse in the Ubuntu Software Center lead me to KAlarm and Alarm Clock. I have no personal experience using them, but seems to be what you're looking for.
