### Post by mb_webguy on 2009-01-28
The links in Ahadiel's post should give you all the information you need.  I was looking into that myself a while back, but in the end I decided that it wasn't worth the effort except as a novelty.  

I had initially thought about installing Mythbuntu on the PS3 to create a full-fledged HTPC, but the PS3's hypervisor restricts access to the GPU.  That pretty much nixes any use of Linux on the PS3 for multimedia purposes.  There have been some inroads in the last couple of years in bypassing the hypervisor to gain access to the GPU, but it's still essentially a hack, and I believe that you can't do it if you've updated your PS3 system software.

And really, other than the limited formats playable on the PS3 -- I'd really like to play matroska files with h.264 video and AC3 audio, for example, and better subtitle support would be nice -- it's a decent media center as-is, especially when coupled with a Linux machine running a UPnP media server like MediaTomb.

### Post by mb_webguy on 2009-01-28
It would be simpler to set up a UPnP server like MediaTomb on his computer so he can easily transfer files to the PS3.  Installing Linux on the PS3 just to download files is a bit overkill, and would require him to change operating systems to download and then play the media.  Installing a UPnP media server allows instant access to media downloaded on another computer.
