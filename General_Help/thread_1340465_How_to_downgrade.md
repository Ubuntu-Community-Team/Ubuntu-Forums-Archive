### Post by jbrown96 on 2009-11-28
I've had this same problem. I've enabled all updates (pre-release and unsupported) in Synaptic. It seems to have gotten better this last week. I've found that if I disable wireless (eirther right click on the network icon and disable wireless, or use the hardware switch) right before suspending I hardly every have any problems with suspend. It's kind of a pain, but it seems to work.

If you want to go back to 9.04, you will need to do a clean install. There's no tools to roll-back dist-upgrades.

### Post by bluesscream on 2009-11-28
