### Post by lukeiamyourfather on 2011-03-03
By default most Linux distributions use GMT/UTC which usually conflicts with other software that uses local time by default (like Windows). As long as everything uses one or the other the clocks will stay in sync.

System -> Administration -> Time and Date.  You'll need the administrative account password to make any changes.
