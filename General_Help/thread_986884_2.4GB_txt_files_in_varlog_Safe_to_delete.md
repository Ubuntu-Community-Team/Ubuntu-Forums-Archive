### Post by deadowl on 2008-11-19
Don't worry about them being updated in the past hour. If you delete them and the file isn't there to write into, the system will just create a replacement file and start fresh.

It's kind of like if you delete your .mozilla folder. All of your Firefox settings will be gone, but Firefox will reconstruct the .mozilla folder the next time you run it.

Possibly not enough information.  There is a bug listed in launchpad for this very issue.

Are you able to go back in the log to the last message before these messages pop in?  Do the messages before this seem to have anything to do with wireless?  If so, try: ```

If this isn't the case, look through that bug report.  It is ongoing, and there is a lot in there...
