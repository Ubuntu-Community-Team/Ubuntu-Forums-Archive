### Post by BillyBoa on 2011-11-16
To find the program now you have to go to the upper part of the launcher and push the dash. Then on search, write the name of the program and you will see the icon. If not on the same screen of the dash go down and push the second white... icon and go to applications. From there search all the programs you have installed.
