### Post by NilsE on 2008-05-16
Icons are stored in the /usr/share/icons folder by folder name of theme.  Between Gutsy and Hardy some icons were renamed which is why they are being picked up from Human instead of your theme.  Find the icon in Human and rename the ones you want in your theme to the same name as in Human.

An alternative is to edit the properties of the applications and select the icon you want for it.  There is no alternative for other so you may have to rename some icons.
