### Post by wael_hossam on 2007-08-15
Hello,
I just found a solution:
-Open the terminal
-Type the following:
evolution contacts

>This will open evolution with contacts opened
Now go to File menu and import any external mailbox file to a new folder under inbox

When you do that evolution will go by default to the new folder which is not the same as the crashing email

Now if you click inbox it will go again to the samle crash email but instead point the mouse to the upper frame of the message viewing window and shrink it to remove message preview
also go to the view menu and go to preview submenu and remove the check mark for message preview

Hey:
Now press on inbox and you can change the current message, delete the bad message and return and enable the preview again

I feel gr8 now
