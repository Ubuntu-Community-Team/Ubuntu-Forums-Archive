Dumping Windows for Linux because a virus hosed your system probably isn't a good enough reason to start using Linux. You need an additional reason: a desire to use Linux. This is a fundamental difference between Win-World and *nix -- you need to understand what your "role" is when using your system -- should you be a "user" or the "Super User", to perform the task at hand? Good to see you have the spell checker in OO writer fixed.

There are many hundreds of applications in the Add/Remove.  What are you looking for?

As far as Ubuntu not recognizing you as an admin you have to either do a sudo at the command line or enter your password when prompted.  Not being an admin is one of Ubuntu's security features.

If you wish to install an application at the command line (VLC for instance) type this.

sudo apt-get install vlc

You will be asked for your password and VLC will install.  You will see a VLC icon in the menu.  Ubuntu gives you a huge amount of software installed by default.  You don't need to go from website to website downloading a bunch of stuff.

Leave the application open that you are copying and pasting from.  It is very unintuitive to press CTRL-C and them immediately close the application.  Are you trying to conserve memory or something?

Ubuntu may not be for you.  If that is the case then you can go back to M$.

If you want to keep your Windows install safe do a few things:

1.  Install and keep up to date Anti Virus
2.  Do the same with Anti Spyware
3.  Do not run an account as an admin, run limited user.  There is no reason to run as admin.  Nobody need 100% software install capability.
4.  If not using a router then enable Windows firewall (done by default in SP2).
5.  NEVER use IE, use Firefox.
6.  Don't use Outlook Express, use Thunderbird or Evolution.

XP is so insecure out of the box that it only takes a few tidbits to make it more secure.
