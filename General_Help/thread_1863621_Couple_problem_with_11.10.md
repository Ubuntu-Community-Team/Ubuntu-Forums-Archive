### Post by grahammechanical on 2011-10-18
People,

the OP is using 11.10. Pushing the mouse pointer into the top left corner does not bring out the Launcher any more.

@akaicewolf

Have you installed ComprizConfigSettingsManager? That has a Ubuntu Unity Plugin that will allow you to set how the Launcher opens.

The same utility has something called Place Windows. That may help fix the problem with the application window going under the top panel.

This happens to me in 11.04. The first Writer window opens and sits up against the top panel. The second Writer window opens alongside the first. The third Writer Window opens on top of the first window and above it going under the top panel.

I have not tried to fix this as I started using 11.10 beta on another partition full time. And I have not had this problem.

As a matter of curiosity, did you do a fresh install or an upgrade? I am forming the opinion that highly modified 11.04 desktop environments cause problems when upgraded to 11.10.

I am of the opinion that before upgrading to 11.10 a person should strip 11.04 back to the standard Unity install.

I think that I shall fresh install when the time comes to move my working Ubuntu up to 11.10 in November. That is another thing. It is best to wait  a few weeks before upgrading. See what the problems are and how to avoid them. Give people a chance to work out the answers.

Regards.
