### Post by whitethorn on 2008-09-02
I don't know if this is exactly what you want, but you could give Basket Notepads a try.  It's a KDE package, but it's pretty useful if you want to keep stuff organized.  It pretty much offers you baskets where you pull images into, different files and type text.  A lot more stuff I haven't tried out. It's available in Synaptic, just search for basket.  

Here's the homepage  

[http://basket.kde.org/](http://basket.kde.org/)


Another you program you could try, is Rainlendar it's a customizable desktop calendar.  It has 3 windows, the calendar an events windows (which shows you what coming up) and a to do list.  It's pretty easy to use, and it doesn't look that bad.

Here's the homepage, they offer the file as a .deb so just dload and right click and open with Gdebi Package Manager.

[http://www.rainlendar.net/cms/index.php?option=com_frontpage&Itemid=1](http://www.rainlendar.net/cms/index.php?option=com_frontpage&Itemid=1)
