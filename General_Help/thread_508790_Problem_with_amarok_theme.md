### Post by kuja on 2007-07-24
czepluch, to change the style hit "alt + f2", run the command kcontrol. From here go to Appearanches & Themes -> and play with things from there. There are quite a few widget styles, window decorations and the like in the repositories. For more look on kde-look.org.
