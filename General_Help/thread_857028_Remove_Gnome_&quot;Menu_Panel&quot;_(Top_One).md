### Post by Truedream on 2008-07-12
well if you wanna get rid of the panels than go to preferences ---> sessions ---> current session and remove gnome-panel and click apply. And if you want to remove panels permanently so that it doesn't come up when you restart go to next tab session options and click on remember session. Make sure you close other applications i.e. Firefox, music player or any other folders that you've open because by doing this Ubuntu will automatically start those programs for you.

Once you delete panels install [awn]("http://www.ubuntugeek.com/howto-install-avant-window-navigator-awn-in-ubuntu-710-gutsy-gibbon.html") which is a dock utility. hope that helps.
