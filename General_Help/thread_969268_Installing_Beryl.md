### Post by Vadi on 2008-11-03
Beryl was renamed to Compiz. Desktop effects = Compiz. So, no need to install Beryl, you already have it working :)

You're welcome to install the settings manager ([click]("apt:compizconfig-settings-manager"), and go to system - preferences - CompizConfig Settings Manager) to tinker with your settings.
