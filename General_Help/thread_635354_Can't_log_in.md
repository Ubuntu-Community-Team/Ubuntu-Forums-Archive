### Post by spiderbatdad on 2007-12-08
I suggest ```
gksudo gedit /boot/grub/menu.lst
```
when the editor open, scroll down till you see the section below. Then remove the words "quiet" and "ro quiet splash" It may not be the ideal fix but it solved the problem for me. When you reboot and get to the login screen, select options and choose gnome-->make gnome default.

## ## End Default Options ##

title		Ubuntu 7.10, kernel 2.6.22-14-generic
root		(hd0,0)
kernel		/boot/vmlinuz-2.6.22-14-generic root=UUID=84b0ec8e-e7f2-4baf-a860-b9d31f516d0d** ro quiet splash**
initrd		/boot/initrd.img-2.6.22-14-generic
**quiet**

title		Ubuntu 7.10, kernel 2.6.22-14-generic (recovery mode)
root		(hd0,0)
kernel		/boot/vmlinuz-2.6.22-14-generic root=UUID=84b0ec8e-e7f2-4baf-a860-b9d31f516d0d ro single
initrd		/boot/initrd.img-2.6.22-14-generic

title		Ubuntu 7.10, memtest86+
root		(hd0,0)
kernel		/boot/memtest86+.bin
quiet

### END DEBIAN AUTOMAGIC KERNELS LIST[/CODE][/QUOTE]
