### Post by Frogs Hair on 2012-08-04
If using Thunderbird  Mail,  Mozilla recommends the Lightning calendar because Sunbird is no longer being developed. Lightning can be installed from tools- add-ons in Thunderbird.  [https://addons.mozilla.org/en-US/thunderbird/addon/lightning/](https://addons.mozilla.org/en-US/thunderbird/addon/lightning/)

The read me file in Sunbird states the following. > For information about installing, running and configuring Sunbird 
including a list of known issues and troubleshooting information, 
refer to: [http://www.mozilla.org/projects/calendar/](http://www.mozilla.org/projects/calendar/)
