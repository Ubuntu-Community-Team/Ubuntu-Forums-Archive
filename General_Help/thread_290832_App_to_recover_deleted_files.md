### Post by bsussman on 2006-11-01
If you go into synaptic and search for the word forensic in name and description, you will find some tools that might work.

I assume you are not currently using the disk in question, since if you were it would be  rewritten, modified and so on as we speak which lessens your chances of recovery greatly.
