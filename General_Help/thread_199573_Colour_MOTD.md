### Post by afallenhope on 2008-05-13
Yeah I have one set up for my shells.
You have to use the VIM editor.
I'd suggest you update it.

To accomplish colour:

PRESS: CTRL + V then CTRL + [ then enter your colour code.

EXAMPLE: CTRL +V CTRL + [ outputs ^[ (which is the escape char.).


```
CTRL + V
CTRL + [
[0m 
CTRL + V
CTRL + [
[31m
```

will output... red text.
 Hope this helps
