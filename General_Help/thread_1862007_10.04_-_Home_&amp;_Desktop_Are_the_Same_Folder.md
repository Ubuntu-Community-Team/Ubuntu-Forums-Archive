### Post by 73ckn797 on 2011-10-16
If you have Configuration Editor under Applications/System Tools, you can check settings in apps/nautilus/preferences and uncheck the option "desktop_is_home_dir".

If no shortcut under Applications then open terminal and enter ```
gksudo gconf-editor
``` and you can access config there.
