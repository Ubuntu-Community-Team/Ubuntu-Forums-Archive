### Post by picopir8 on 2008-03-31
Just to expand on what the others stated, you can can use the liveCD to access any drive (internal hard drive, optical drives, usb flash drives, usb hard drives, other memory card reader/writer, etc.).  A few weeks ago I used my liveCD to copy all files off a hard drive with a corrupt boot sector and put them on a USB hard drive.

Keep in mind, if you do have a virus, your files may be infected.  So while you will be able to  get them with linux, as soon as you copy them back to a windows computer, you may infect that computer.  So run a virus checker before sending them to a windows computer.
