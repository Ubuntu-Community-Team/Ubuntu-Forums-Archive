### Post by tiagosales on 2009-09-12
Go to:

Menu > System > Administration > Update Manager.

Click in Settings, Third-Party Software Tab and mark [http://archive.canonical.com/ubuntu](http://archive.canonical.com/ubuntu) jaunty partner repository.

Then search for Symphony in Synaptic.
