### Post by rob_quill on 2009-05-12
I guess try it and see what happens. My motheboard was about 4 years old so if yours is newer maybe it is the graphics card (I tried that first but it didn't help) I'd say change the slot and see what happens. If that doesn't help try another graphics card if you have one and if that doesn't work then it's probably the motherboard. I think when I put the new graphics card in it got stuck on the BIOS which was a pretty good indicator the motherboard was at fault.
