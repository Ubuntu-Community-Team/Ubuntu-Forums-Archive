### Post by U2XS on 2009-06-15
I have become very dependent on [AutoHotKey]("http://www.autohotkey.com") in Windows. It is a keyboard and mouse macro program. But in addition to that it allows you to use programming to complete your task. You can create loops, assign variables, use functions, etc... It is even integrated into Windows so that you can access DLL files directly.

Is there such an equivalent in GNU/Linux? I would hope so, but I can&#8217;t find it. I do see that Ubuntu does have Hotkeys enabled, but that pretty much means that a combination of CTRL+C is mapped to the Copy command. This is pretty basic and I am looking for more. Thoughts?

### Post by synapsys on 2009-06-15
You can configure keyboard shortcuts in System >> Administration >> Keyboard. Or it might be System >> Preferences >> Keyboard... I can't remember at this time.

### Post by VCoolio on 2009-06-15
You could take a look at [xbindkeys]("http://www.linux.com/archive/articles/59494"), very nice app; lets you assign any command to any key- / mousebinding. 
Also you can of course write your own very advanced scripts, make them executable and make keybindings to run them using synapsis' suggestion.
