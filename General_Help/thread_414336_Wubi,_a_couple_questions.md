### Post by ago on 2007-04-20
> **Taigrr said:**
> 
Because it's on C:, will that kill off any virus protection and such you get from running Linux vs Windows? Or will it be just as secure as a default Ubuntu installation?
When you run Ubuntu it is as secure as standard Ubuntu, but if you get a virus within Windows, that has control on all files/disks, it may potentially delete/modify the  linux files.  But that is not much more likely than if you have a standard dual boot.

> How much slower? And what areas are slower?
Same speed. Only harddisk access is slightly slower but you should not be able to notice that.

> And, I see that it's NOT a Virtual MAchine-line program, so it'll have full access to things like my video card and whatnot, correct?
Yes

> And, will it work with Feisty right now? Like, if I were to want to install it tonight, would it work okay?
At the moment it works with Feisty Beta, we are working to push out a Feisty Final release. I am not sure it will work right now since all old downloads were removed from the official servers (nothing to do with us), and the servers are very busy anyway. You are better off waiting for the first minefield which supports final.
