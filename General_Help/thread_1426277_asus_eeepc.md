### Post by 3rdalbum on 2010-03-10
The reason why there are no "restricted" drivers for your computer's graphics is because it uses Intel graphics, and Intel's graphics drivers are open-source and activated by default on Ubuntu.

If Kubuntu's desktop effects are running slowly, you can turn them off from (I think) the Appearance control panel; just disable "compositing". Your netbook should be able to handle this stuff, I would have thought; maybe you've got a program running in the background that is using large amounts of your CPU power?
