### Post by scouser73 on 2010-04-21
> **Kelvrin said:**
> How do I keep Synaptic from updating Thunderbird. I've tried Thunderbird 3 and don't like it so I want to keep my current version.

**1** - Go to Synaptic Package Manager

**2** - Enter **Thunderbird** into the search field

**3** - Highlight the installed package

**4** - Click on **Package** on the top row

**5** - Select **Lock Version**

Done, now Thunderbird will no longer be updated.
