All we've been able to work out so far is how bring up the workspaces, which is overkill for seeing all windows on one desktop.

A non-Netbook tailored desktop might have taken up more space, but I think the lack of obvious Netbook documentation would have made it an awful lot easier to get on with :\ Pointers to documentation and tutorials for the 10.10 flavour of Unity would be greatly appreciated, as Google only turns up the out-of-date or the "Unity Linux" OS, not the UI.
