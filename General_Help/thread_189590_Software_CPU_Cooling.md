### Post by mat1t on 2006-06-05
The way that the software coolers work is to tell the processor to go into idle/low power mode when it's not in use.

If you are running the correct Kernel (k7 I think for your case, hopefully someone else will confirm) then it should do it automatically.

If not, you may want to look in to a bigger fan or a heatsink, as if you play games the software cooler won't help in XP or Ubuntu
