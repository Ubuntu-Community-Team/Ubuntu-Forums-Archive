### Post by geirha on 2008-08-05
.tar.gz is an archive, like .zip and .rar. Double-click it to open it in the archive manager.

.sh is a common extension for scripts. Open it in an editor to see what it does. If you are confident that it does not do anything malicious, you can give it execute permission and run it.
