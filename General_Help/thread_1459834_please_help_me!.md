### Post by 3rdalbum on 2010-04-22
Kompozer is available to install in all Ubuntu variations (Ubuntu, Ubuntu Studio, Kubuntu, Edubuntu, Xubuntu etc). It can help you develop web pages.

I don't know if Kompozer has the ability to upload the pages to your hosting company, but the file browser used in Ubuntu/Ubuntu Studio and the one used in Kubuntu can upload to FTP sites, which is more than likely how your hosting company works.

As for your other thread regarding your wireless device, you can boot up the Ubuntu CD as a "live session" - that is, Ubuntu running from the CD. You can test your networking device there before you install Ubuntu.
