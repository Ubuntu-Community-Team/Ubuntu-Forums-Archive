### Post by acl123 on 2009-11-11
In Intrepid I had an icon in the top menu bar, with a drop down list of languages. I could select Japanese and type in Japanese characters.

This has disappeared in Karmic. I have tried everything to get Japanese input back, reinstalling scim and reinstalling iBus.

I am not sure what I am supposed to be doing. iBus doesnt seem to do anything. It says there should be a 'language bar' but there is no language bar. It says I can hit ctrl-space, but if I hit ctrl-space nothing happens.

How can I type in Japanese?

---

### Post by Irihapeti on 2009-11-11
I think you need to go to System -> Administration -> Language Support and select "iBus" in the Input Method box. Then you need to log out and in again (or reboot if it suits you better).

Also make sure you have ibus-anthy installed as well.
