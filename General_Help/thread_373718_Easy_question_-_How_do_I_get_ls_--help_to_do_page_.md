### Post by otazman on 2007-03-01
Trying to read a help file but when I am in terminal mode and enter ls --help it scrolls all of the information off the page. How do I put screen breaks on each page of information? i.e. like /p on windows command line.

---

### Post by taurus on 2007-03-01
```
ls --help | more
```
Then hit the Space bar for the next 24 lines.
