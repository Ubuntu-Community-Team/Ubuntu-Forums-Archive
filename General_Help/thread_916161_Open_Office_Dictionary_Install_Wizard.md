### Post by robert shearer on 2008-09-10
Try this Open open office Writer  then select Tools from the menu bar at the top.
From the drop down menu select options at the bottom.
Select/expand  Language Settings from the left hand menu.
Select Languages
Go to Default Languages for Documents  Western  and expand this.
Search for English Canadian.
If there is an abc symbol next to it then the dictionary is installed and simply select it as default and apply/close.
If no abc then you need to install the dictionary/aspell package from your synaptic package manager and repeat the above.

Repost if you need more help.
