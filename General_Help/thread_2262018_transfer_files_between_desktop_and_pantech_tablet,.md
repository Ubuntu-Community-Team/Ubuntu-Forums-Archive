### Post by Dreamer Fithp Apprentice on 2015-01-23
Gonna answer my own post and save some other fool some trouble. Pretty  sure the problem is the micro usb connector. The engineer that wrote the  specs for those things needs to go into some other line of work. Might  be an excuse for putting such a flimsy little thing on say, a watch, or  an absurdly tiny phone, but on a tablet there is plenty of room for a  full size standard type A rectangular usb port. I can't get either data  or power through the micro usb. Not the first time I've seen this, I  should have suspected it sooner. I strongly encourage anyone reading  this to think twice before buying any device in which a critical  function depends on a micro usb contiuning to work, because they tend not to.
