### Post by PaulW2U on 2018-07-21
> **mark allyn said:**
> I am running 32-bit ubuntu 12.04 OS.
Ubuntu 12.04 is no longer supported as it has been "End of life" since April 2017 - see [https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases).

The currently supported releases of Ubuntu are 14.04, 16.04 and 18.04. Instructions for installing R 3.5 on those releases can be found at [UBUNTU PACKAGES FOR R]("https://cran.r-project.org/bin/linux/ubuntu/README.html").
