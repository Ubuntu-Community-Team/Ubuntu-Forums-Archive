### Post by Calash on 2010-04-29
Start with the Software Center and search for Flash.  The Adobe installer will be in the top 5 results.  It may ask you to enable restricted apps, if so say yes (Adobe is not open source, so it has to ask you before installing)

That gets you 10.0.45 if I remember correctly.  There is a pre-release version that is newer, but I would not suggest going that route unless you find the one in the repo cannot fill your needs.
