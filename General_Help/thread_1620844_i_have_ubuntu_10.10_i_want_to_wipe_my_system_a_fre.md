### Post by Foxheadz on 2010-11-13
Download the .iso of ubuntu 10.10 [Here]("http://www.ubuntu.com/desktop/get-ubuntu/download") if you don't already have it.

Burn the disc image to a cd using Brasero disc burner. 

Boot from the cd and select install

During install when you get to the part about what part of the hard drive just select use and format entire hardrive.

or if you don't understand that follow this guide [https://help.ubuntu.com/10.10/installation-guide/i386/index.html](https://help.ubuntu.com/10.10/installation-guide/i386/index.html)
