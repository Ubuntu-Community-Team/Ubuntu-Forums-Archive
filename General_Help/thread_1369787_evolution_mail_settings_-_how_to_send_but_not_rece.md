### Post by dhillonv10 on 2010-01-01
I am afraid not, you have to enter the configuration for receiving the email first, and then it let's you go to enter sending email part. Now with gmail you can do this, gmail introduced new features that allows you to check email from multiple accounts and also the ability to send email without receiving any email from that server

### Post by rdh61 on 2010-01-02
Hi dhillonv10,

Thanks for your reply. I've found a way of doing it that works:

Edit > Preferences > Mail Accounts > Select account > Edit > Receiving Email tab > Server Type: > Select "None" from dropdown list.

Now I can send with this account but not receive. Hope this helps others.
