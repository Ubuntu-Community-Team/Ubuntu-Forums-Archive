### Post by timgood on 2010-03-20
AWN will only install if you have a compositing window manager (like Compiz installed) - and this means that you will need to install proprietary drivers if you are running a NVidia or ATI graphics card. Check out [this site]("http://linuxhelp.blogspot.com/2008/02/install-avant-window-navigator-awn-in.html") for help with installation. Hope this helps.
