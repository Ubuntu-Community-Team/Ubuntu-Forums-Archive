### Post by awk on 2009-09-26
> hello, how do i program in ubuntu, wich ones are the bestActually, you can already do amazing things in any ordinary linux terminal before you decide which programming language you will invest in learning.

consider this command, for instance:

```
clear ; while [ ! "$var" == "quit" ] ; do echo -n "I want to program, because " | tee -a reasons.txt ; read var ; echo "$var" >> reasons.txt ; done ; var= ; more reasons.txt
```After mastering the bash shell, learning an actual programming language might not even be necessary!
