### Post by mybeat on 2008-07-11
Type following in console
```

dosbox

```
Now you need to mount a path to where princeofpercia.exe is located.

```

mount c /path/to/folder/

```
Then you need to go to mounted C: 
```

C:

```
and run it
```

princeofpercia.exe

```
