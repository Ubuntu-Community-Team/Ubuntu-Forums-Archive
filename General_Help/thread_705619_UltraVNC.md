### Post by jpeddicord on 2008-02-24
This isn't UltraVNC, but the client should still be compatible. Install the package [tsclient]("apt:tsclient") and you can run it from Applications > Network > Terminal Server Client. From there you can connect to your VNC server on the other machine.
