### Post by bedtime2 on 2017-12-09
I would like to type in some keyboard characters such as:

Æ, æ, &#338;, &#339;...

But I want to do this with a keyboard remap or easy shortcut of some sort; I would rather not use the <compose> + <enter the 4 digit code here> routine, as I use these characters quite often, and pressing 5+ keys for one key, and having to remember all the codes is more than I am willing to do.

I use Ubuntu 17.10 (Wayland).

Thanx.

[B]
*** SOLVED ***[/B]

Okay, found out how; it was much easier than I had anticipated:

You set your compose key:
Tweaks -> Keyboard & Mouse -> Compose Key -> and turn it on

Press your compose key (mine is <print scrn>). Let it go. Then press the combo as such:

Æ = *A* + *E*
æ = *a* + *e*
&#338; = *O* + *E*
&#339; = *o* + [I]e

[/I]
