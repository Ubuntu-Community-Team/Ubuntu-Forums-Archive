My computer gets very hot (~70 degrees C) when I run .jar files. Is there any way to "lessen the load" when running a .jar? Thanks

### Post by amingv on 2011-09-17
jar files are, by themselves, just collections of Java classes and/or resources. These classes may contain code to do something as simple as adding two numbers or as complex as a full-blown videogame and will tax your processor/resources accordingly.

However, even when running some very intensive app your processor shouldn't really get that hot for for a long period of time. If you have a reason to believe the stress is caused by a program in a jar file, you can open your task manager and see how much CPU that given program is using. Otherwise, it might be a good time to check fans, thermal compound and case ventilation.
