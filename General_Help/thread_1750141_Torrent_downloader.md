### Post by cymbaline42 on 2011-05-06
Some cool suggestions on this thread :-)

I had used only Transmission from the time I started using Ubuntu until the time I started reading this thread.  It's very lightweight, consumes very little memory (~10 MB in my case) and has an option to enable blocklists.

I was very curious to try Tixati (since I was a huge fan of the creator's previous application until it got shut down) and I have to say I'm very impressed.  I like the interface, the display of relevant data and download speeds were much faster than Transmission.  Also like how you can add multiple blocklists by specifying URL and setting them up to update automatically at regular intervals.  The only downside was a higher memory footprint than Transmission (~60 MB) but I think I can live with it for the positives outweigh the negatives.
