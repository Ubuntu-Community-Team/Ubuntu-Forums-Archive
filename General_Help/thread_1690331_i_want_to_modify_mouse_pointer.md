### Post by rvchari on 2011-02-20
> **jolian ramos said:**
> i don't know how to install point themes file from this site i already downlaod one of them but don't know what to do then i am sorry i am kind beginner

go to gnome-appearance, select themes. keep the window open and simply drag the *.tar.gz theme into that window. it will install.

alternatively, click the install button in the theme section and point it to the theme you have downloaded.
enjoy.

NOTE: if you dont have the tar.gz format and it is in zip format, first unzip and compress that folder to yourtheme.tar.gz format. then follow the above steps.
