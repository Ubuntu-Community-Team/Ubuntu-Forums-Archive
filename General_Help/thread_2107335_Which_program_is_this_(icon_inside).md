### Post by grahammechanical on 2013-01-22
I have a similar one on my system. Green background (darker than the colour you show), white bubble, green hash (#). It is in Online Accounts. It represents IRC chat, which is not installed by default in Online Accounts. To get Internet Relay Chat in Online Accounts, search the Software Centre for account-plugin-irc and install it.
