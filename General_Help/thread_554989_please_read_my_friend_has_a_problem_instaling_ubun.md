### Post by 505 on 2007-09-19
You need to tell the computer to boot from the CD. Watch the message in the boot screen, before Windows comes up. Usually is tells you which key to press for setup, and which one for boot options (or boot order). If boot options is there, press that key and select the CD drive. If not, press the key to go into the setup (BIOS) and change the boot order. Place the CD drive before the hard drive, save, and reboot.
