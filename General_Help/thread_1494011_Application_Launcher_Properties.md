### Post by weisend on 2010-05-26
I installed the Ubuntu Netbook Edition 10.04 on my Asus Eee 1000, and its really been great so far. The one thing that I haven't been able to figure out is how to set application launcher options in the favorites and other menus. Just to clarify what I mean: In the standard desktop editions, you do this by right clicking an application icon, selecting properties, and then editing the command. I want to be able to launch with startup switches like "/opt/google/chrome/google-chrome %U **-start-maximized**."

Thanks very much for any help with this!

---

### Post by freddiespagheti on 2010-05-26
I think what you want is to go to:
System > Main Menu

Then find the item you are looking for. For example, for Google Chrome, select 'Internet' in the left-hand pane and 'Google Chrome' in the middle pane. Then, in the right-hand pane, click 'properties'. Finally, edit the 'command' box as pleases you.
