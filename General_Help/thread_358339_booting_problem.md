### Post by sdide on 2007-02-10
Hey, 

first you need to make sure, the computer even tries to boot on a CD, when you boot.
You need to check your BIOS settings, and then etiher: 
* make it boot on CD before the harddisk, or - if available - 
* put the CD-drive on the list of bootable device-menu, and then choose that in BIOS POST.

If that doesn't work, I recommend trying to boot something else (a windows bootable CD or something else bootable), to see if your problem lies with the liveCD. If you can boot another CD, there is probably something wrong with the live CD.

Each computer is different, but there is a hotkey that you push just after the computer comes on and before it goes into the OS.

Usually an F1-F12 key.  Like on my Gateway, it is F2 = BIOS and F10 = Boot options.

Check your documentation or the website for your computer manufacturer.
