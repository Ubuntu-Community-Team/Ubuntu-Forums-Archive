### Post by michaelkahl on 2008-09-13
Well lately I've been searching for a lightweight powerful and easy to use system.  Ubuntu has been good to me, but it still has more installed out-of-the-box than I care for, so my search began.
Last night I actually started playing around with Sidux, it's definately a great OS.  Unfortunately flash is busted right now and won't work with it.  So I moved on.  Since I'm enrolled in Cisco's netacademy it is important for me to have flash working.  (I installed the 64bit version, I read about people having problem with the 32bit as well so I moved on)I may even consider Sidux again later on down the road.  Everything else installed with no problems for me.
Next I was considering Arch or looking into an XFCE debian distro when I ran across an interesting article.
[http://psychocats.net/ubuntu/minimal#standard](http://psychocats.net/ubuntu/minimal#standard)
This article discussed the minimal install Ubuntu CD....I'd never heard of this.  So I downloaded the iso and went with it.  This is truly stripped down.  I've installed xfce, open office, rawtherapee, argyllcms, vlc media player, a couple of other programs I commonly use and did my little customizations to the GUI.  I feel that everything I need is here, and nothing is installed that I don't want or won't use.
I'm sure there are way more in depth ways of building your own Linux that can shrink it down even more, but that's not what I was after today.  I wanted an effective system that has what I want and that I could setup relatively quickly. 
I run Ubuntu 8.04.1 64bit at work and usually start out with around 250-300mb in system memory. This is still half of what Vista uses, but about the same as XP runs when I have everything loaded onto it.

I now boot into my session at 120mb in system memory, am still a member of the Ubuntu community and am way to addicted to Linux.
:guitar::lolflag:
This is in no way a rant or complaint about Ubuntu.  I'm very thankful for the work that has been put into this distro.  If anything I think distro's like Ubuntu are doing a wonderful job of delivering easy to use software into the hands of many. 
Ubuntu has been my gateway distro and I'm already making plans of setting up VM's to test other distros, break some distros and even play with compiling my own kernel, which I have done once so far.  
So thanks to the developers and the community for all the hard work.
