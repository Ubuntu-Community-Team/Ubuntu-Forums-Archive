### Post by &amp;KyT$0P# on 2022-09-06
I use the "English (Macintosh)" keyboard layout, which uses the right Alt key to type accents.  For example, for the umlaut, press right alt + U, then the letter you want the umlaut on: e.g. right alt + U then press e gives ë.

This is independent of desktop environment.  There are other keyboard layouts that contain similar options.
