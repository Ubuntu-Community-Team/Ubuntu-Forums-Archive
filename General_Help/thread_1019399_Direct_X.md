### Post by jespdj on 2008-12-23
Direct X is not a program. It's a set of Windows libraries to provide hardware-accelerated 3D graphics, sound and other stuff.

OpenGL is the cross-platform API for hardware-accelerated 3D graphics, which is implemented on Linux distros. If you are using the right video driver, you have OpenGL working already.

Can you explain why you're asking this question - what are you trying to achieve that requires a replacement for Direct X?
