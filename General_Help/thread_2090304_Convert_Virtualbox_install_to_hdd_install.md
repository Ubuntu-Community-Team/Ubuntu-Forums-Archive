### Post by 2F4U on 2012-12-01
> **Foxheadz said:**
> This is all just hypothetical, but I want to know if this is possible. Lets say I set up a virtualbox install of ubuntu. After a while, it all seems to go smoothly and I want to install ubuntu to a hard disk. Instead of having to redo everything in my virtualbox install, is there anyway I can just make like a bootable backup and install that to the harddrive?

Yes and no. While it is not possible to transfer the VM guest directly to physical hardware, there is a software remastersys, which allows you to create live media from your customized system. These live media may then be used to install on another system.

[http://www.remastersys.com/index.html](http://www.remastersys.com/index.html)
