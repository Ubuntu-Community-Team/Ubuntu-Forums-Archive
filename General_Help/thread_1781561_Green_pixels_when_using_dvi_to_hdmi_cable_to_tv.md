### Post by christopher.wortman on 2011-06-18
Turn off subpixel smoothing and see if that fixes things in Nvidia Control Panel. If not, try a shorter/different cord. Also do you dual-boot with Windows? Does it persist in Windows to see if it is not a bad cable? I use a $30 D-Sub Stereo-triopic 15' DVI cable for my monitor without any issue. I find cheaper cables seem to give me headaches and aren't worth my time.
