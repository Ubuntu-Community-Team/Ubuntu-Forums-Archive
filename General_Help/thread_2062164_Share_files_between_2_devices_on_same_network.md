### Post by einonm on 2012-09-24
On the Linux machine, set up a samba directory share, search for 'samba' in the Ubuntu software centre, and use the GUI app to setup then Nautilus / Windows Explorer to copy between the machines.

On XP, I believe there is a method to do the same by sharing a particular folder.

---

### Post by NCLI on 2012-09-24
Or the easy way: Right-click a folder in Ubuntu, choose to share it, allow write, then access it from the XP machine through "Network" in the Start menu.
