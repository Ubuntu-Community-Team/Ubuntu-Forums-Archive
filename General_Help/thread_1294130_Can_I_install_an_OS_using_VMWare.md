### Post by 3rdalbum on 2009-10-17
VMWare requires an x86 host CPU and an x86 guest operating system, so the first suggestion won't work. And I don't think you can swap a Mac OS 9 system from one computer to another; I think there's some model-specific foo. You could try it though if you have another PowerPC-based machine with an operating system that will run on the G3.

If you've got a Firewire CD-ROM drive and the G3 has a Firewire port, you could probably use that assuming it's made for use with the Mac. If it's not Apple-approved it might not work, from what I've heard.
