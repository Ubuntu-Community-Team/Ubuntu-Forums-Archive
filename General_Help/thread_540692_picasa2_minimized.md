Does anyone know if there is a way to autostart picasa upon login and have it minimize to the panel?

---

### Post by apetresc on 2007-09-01
Yes. If you're using KDE, simply put in a symlink to Picasa in ~/.kde/Autostart . If you're using Gnome, use the Session Manager to save a session with Picasa running the way you want it, save that session, and set it to autoload :)

Good luck
