### Post by King Blah on 2006-06-08
Hi. 
I recently installed some programs from Synaptic Package manager, and they were nowhere to be found on the Applications Menu. I saw somewhere to install "menu-xdg", and restart X with ctrl+alt+backspace. I did so, but still no menu. 
Then i tried installing "menu" as well, and restarting X. Still nothing.
How on earth do i get this thing working? When I install applications I don't want to have to launch them by using a command in the terminal..

---

### Post by matthew on 2006-06-08
Since you have the "menu" package installed, from a terminal type "sudo update-menus" and it should set it up for you. You can also use this command to update the menus later when you add new things.
