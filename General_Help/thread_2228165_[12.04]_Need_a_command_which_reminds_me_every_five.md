### Post by grumblebum2 on 2014-06-06
How about a script using a zenity progress window.
Every 300 seconds the progress window pops up and closes after 10 seconds.
```
#!/bin/bash

while [ 1 ]
	do
		sleep 300
		i=0
		(while [ $((i <= 100)) '=' 1 ]
		do
    		echo $i
    		sleep 1
    		i=$((i + 10))
		done) | zenity --progress --title="Rest Break" --text="Take 10 second break" --auto-close
		
done
```
