### Post by zephyrcat on 2009-04-19
What you are trying to do is quite complicated. I don't know of any simple way to do this, but you could definitely write a shell script. Check out this page for creating the script:

[http://nixcraft.com/shell-scripting/209-script-uploading-files-ftp-server.html](http://nixcraft.com/shell-scripting/209-script-uploading-files-ftp-server.html)

Then go to Synaptic and install gnome-schedule. This is a great GUI program for scheduling tasks (basically a front-end to cron).
