### Post by snowpine on 2008-09-10
Hi ieBrazil,
Ubuntu includes its own office suite called OpenOffice that can read and write MS Office files. MS Office will not run natively in Ubuntu, since it is a Windows application and not a Linux application. There is an application called Wine that some people use to run Windows apps in Ubuntu. I do not know how well MS Office runs in Wine, but frankly, I am 100% satisfied with OpenOffice. :)

### Post by snowpine on 2008-09-10
The only tricky part is that when you save a file, the default format is OpenOffice. If you want to save as an MS Office document, you have to choose the file format from the Save As window. You can also go Edit-->Preferences and set MS Office as your default file format if you like. That's the best idea if you frequently share files with MS users.
