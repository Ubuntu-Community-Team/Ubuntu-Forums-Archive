### Post by tenpennys on 2013-10-19
There is a very nice paperport clone for linux called MaxView.  It will stack and read paperport .max files and create them as well. 
Only problem is that the project has been discontinued.   We need to get some interest in it again to get the project going again.  It works well if  you can get it to work.  There is no support for 64 bit.  Although I was able to get it going in Linux mint 14 64 bit and in mint 14 32 bit on a laptop.  Had to do a make though.  

Sounds like there would be some support for keeping the project going again if someone was interested in picking it up.
