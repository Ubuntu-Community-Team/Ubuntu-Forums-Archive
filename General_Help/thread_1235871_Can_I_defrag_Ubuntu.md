### Post by martinbaselier on 2009-08-09
There are tools, but it' s not really necessary. Linux doesn't get fragmented easily. 

You can run fsck to see if your disk is fragmented. You need to start in recovery mode to run fsck. 

If you use ext3, you can use shake
[http://vleu.net/shake/](http://vleu.net/shake/)

For ext4 there is not yet a tool afaik.
