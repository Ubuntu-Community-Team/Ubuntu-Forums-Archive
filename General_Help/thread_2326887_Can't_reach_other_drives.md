### Post by oldos2er on 2016-06-05
You'll need to boot into Windows to fix your NTFS partition. Did you disable Windows' fast boot? See [http://www.howtogeek.com/243901/the-pros-and-cons-of-windows-10s-fast-startup-mode/](http://www.howtogeek.com/243901/the-pros-and-cons-of-windows-10s-fast-startup-mode/)

Instead of deleting Ubuntu, you could simply install Linux Mint in the same partition you currently have Ubuntu in; though Mint is very similar to Ubuntu.
