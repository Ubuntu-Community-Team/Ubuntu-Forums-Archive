### Post by C.S.Cameron on 2010-12-07
Welcome to the forums Phishhook.

You can do a Full install to flash drive just like to internal HDD.
It will act the same only a little slower. Compiz and video drivers, etc. will work fine.
A 4GB flash drive is minimum size required.

[http://ubuntuforums.org/showpost.php?p=10205977&postcount=3](http://ubuntuforums.org/showpost.php?p=10205977&postcount=3)

Compiz should work ok in your standard Persistent, Disk image install.
Install compiz settings manager and select the desired effects.
Create a launcher on your desktop with "compiz" as the command.
Run the launcher after booting and the effects you have selected should start.
