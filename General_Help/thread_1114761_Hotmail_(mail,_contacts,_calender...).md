### Post by Dejai on 2009-04-03
Firstly, welcome to the Ubuntu forums! And congratulations on switching to Ubuntu! 

I have never personally used live so I can not really comment on the availability of a calendar or address book sync, though I am sure someone should be able to help you. If your trying to do this with thunderbird I would suggest getting an IRC client: Open terminal and type:
sudo apt-get install xchat 

Then going to server:
irc.mozilla.org
channel: 
#thunderbird 

I believe mozilla offers help for there products. They should be able to give you specialised help for that application, if you have any other questions feel free to ask.

On the issue of Gmail, I personally would like to get as far away from anything Microsoft has its greedy little paws around, but in truth it is not going to matter that much, if you are used to using the live services that fine. I suggest you check out Gmail and make your own mind up on that :P Gmail does offer a lot of other services many of which are somewhat more compatible with Ubuntu applications due to the nature of Googles software.
