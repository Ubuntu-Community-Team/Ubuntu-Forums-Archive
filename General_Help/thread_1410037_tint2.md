### Post by thil77 on 2010-02-19
in tint2-0.9 or latest SVN version, you have tooltip option
[http://code.google.com/p/tint2/wiki/Configure#Tooltip](http://code.google.com/p/tint2/wiki/Configure#Tooltip)

here my tooltip config
```

#---------------------------------------------
# TOOLTIP
#---------------------------------------------
tooltip = 1
tooltip_padding = 2 2
tooltip_show_timeout = 0.7
tooltip_hide_timeout = 0.3
tooltip_background_id = 4
tooltip_font_color = #OOOOOO 80
tooltip_font = sans 10

```
