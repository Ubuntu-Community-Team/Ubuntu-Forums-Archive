### Post by kuja on 2008-03-10
You'll need to use sudo - [https://wiki.ubuntu.com/RootSudo](https://wiki.ubuntu.com/RootSudo)

Either you could use sudo with the cp/mv command(s), or you could start your file browser with gksudo/kdesudo (ie: gksudo nautilus or kdesudo konqueror).
