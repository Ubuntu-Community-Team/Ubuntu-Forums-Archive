The work being done to run Ubuntu on mobile phones will result in better memory usage for Ubuntu that will filter down to the benefit of desktop Ubuntu users. How much memory does the video adapter have? That is what will make a difference.

Do not compare the system requirements of a Microsoft OS with the system needs of a Linux distribution.

You don't need the "Windows installer" (Wubi) in order to dual-boot between Windows and Ubuntu. In fact it's better not to use it.

Download the disk image file for Xubuntu or Ubuntu and burn it to CD as a disk image (i.e. use the "Burn image to disk" function of your burning software). Set your BIOS to boot up from CD, and then boot up your computer from that disc and select "Install Ubuntu" from the menu.

Choose the option to install side-by-side with Windows and then wait.

When it's finished installing, reboot. You'll be able to choose on boot-up whether to run Windows or Ubuntu/Xubuntu.

Your computer should still run okay with Ubuntu, although Xubuntu will be a bit faster, Xubuntu is also more spartan and a bit old-fashioned.
