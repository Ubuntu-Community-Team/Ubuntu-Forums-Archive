### Post by oldfred on 2013-10-19
With my Linksys router I had no issues. I like a fixed IP for my Desktop to make it easier to copy data to laptop. But my Comcast router is giving me grief.

You cannot use any fixed addresses that are part of DHCP. But with my Comcast the entire range is DHCP and you still configure on the router which device is fixed. So some routers require settings on router and some do not, but most can let you use router settings.

My Linksys just reserved 100 thru 150 as DHCP, so I was able to use anything outside of that range.
Screenshot is with 168. range. New Comcast uses 10. range so I had lots of changes.
