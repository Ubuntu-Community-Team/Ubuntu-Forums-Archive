### Post by WWSmith36 on 2008-10-13
If you still have windows and it is on that drive, I suggest you boot into windows and cleanly shut it down. Boot into windows and shut it down this way twice and then try to mount the drive in ubuntu.

If that does not help then you will have to manually mount the drive with the force option.  This is not the recommended way to mount a drive, as it warns it may cause problems.  I would try mounting as read only first and then backup any important data before forcibly mounting the drive.
