### Post by adam814 on 2009-12-29
Are you sure you're currently using it?  Do you happen to have the "us" keyboard layout installed as well?

Try going to System > Preferences > Keyboard > Layouts

Note which layout is the default and whether the separate layout for each window box is checked.  I believe both alt keys at once changes keyboard layout, but it's also configurable under layout options.

I use the "us" and "latam" layouts myself and I've on several occasions gotten the same effect while not realizing I'm using the wrong layout.
