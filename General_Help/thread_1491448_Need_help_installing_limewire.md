### Post by sgosnell on 2010-05-23
The terminal is equivalent to the command prompt window in Windows.  You can either click Applications/Terminal or press Ctrl-Alt-T to get it.

I'm not sure installing limewire is ever a good idea, but it's your computer, so it's your decision.  I don't know what went wrong with frostwire, I've never tried it so it's hard to say.  You didn't provide much information to go on.  The easy way to install java is to open Synaptic package manager via System/Administration/Synaptic, then search for sun-java6, click on the appropriate packages, select Mark for installation, then click the Apply button.
