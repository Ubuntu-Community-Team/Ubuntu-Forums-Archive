### Post by TheFu on 2013-01-23
> **bodhi linux said:**
> how cant i recover xubuntu when someting go wrong?

In Linux, you make backups before something goes wrong. Then after something terrible happens, you restore from a backup.

There are many, many, many, great backup tools depending on your needs.  If you don't have any backups, the choices become more limited and your skill with Linux will determine whether and what can be recovered, if anything.

OTOH, seldom does anything really bad happen regardless of how it appears.  Linux is fairly hard to hurt unless you run as root or use **sudo **all the time.

Often, the easiest answer for someone having GUI issues after a change is to create a new user account and login using that instead.
