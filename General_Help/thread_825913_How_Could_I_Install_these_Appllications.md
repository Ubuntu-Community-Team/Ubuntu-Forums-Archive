### Post by yman on 2008-06-11
In the top-left corner of a standard English Ubuntu install you have a menu. In that menu, navigate as following:
Applications -> Add/Remove Applications. You can then browse the catalog of applications available from the Ubuntu repositories, and any other repositories you added.

A more advanced interface called Synaptic can be found at System -> Administration -> Synaptic Package Manager
