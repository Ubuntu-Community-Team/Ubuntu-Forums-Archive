### Post by rainwalker on 2008-06-07
I'm not sure how things work on Kubuntu, but I can explain the video.

First, they're using the Desktop Cube and 3D Windows plugins with a skydome. The reason their skydome rotates like that is because they have it set to animate. There's a checkbox somewhere for "animate skydome" but it only looks good if you have a really wide image to use. Then he set animations for minimizing/maximizing, and used the Annotate plugin with the Zoom Desktop/Enhanced Zoom Desktop plugins. For the waves, they're using the Water Effect plugin, though I don't use it (crashes my comp) so I can't point to what option it is. After that it's the Scale/Scale Addons plugin, then the Ring Switcher, then the Application Switcher. To grab the windows and peel them back like that, just maximize it and then Alt + click somewhere around the edge and drag it. I'm not sure how they took the picture of that little section of the desktop with the scribbles, but I'm guessing that was a feature in Beryl that I haven't found in or has been removed from Compiz Fusion.
Note that doing all that as quickly as that person did will not only require decent system specs but also a stable video driver (or you risk crashing everything).
