When you're at the GRUB boot list, press 'e' for Edit while on the failing kernel, and you should see something like 
```
/vmlinuz-2.6.27-9-generic root=UUID=d9e306e7-0bd3-473e-b683-a1a4317cd3e9 ro quiet splash
``` remove the words "quiet" and "splash" and then post back what's reported. (note: your UUID **will** be different than mine, do not panic, you don't have to change that.)

### Post by MighMoS on 2009-01-03

This may be a bug in usplash, you could report it on Launchpad. As a temporary solution, you can make the change permanent by editing /boot/menu.lst

Remove "quiet splash" where you see ```
# defoptions=quiet splash
``` and run ```
sudo update-grub
```
