### Post by howefield on 2015-12-11
Too long since signing it for me to remember, but just to say that there is a package that you can install that makes it easy to sign.

code-of-conduct-signing-assistant 

> Code of Conduct Signing Assistant is an open source program that helps you
sign the Ubuntu Code of Conduct easily!

No more worries about PGP keys or decrypting mail: just follow the steps
the Code of Conduct Signing Assistant guides you through and in the end
you have signed the Ubuntu Code of Conduct!
