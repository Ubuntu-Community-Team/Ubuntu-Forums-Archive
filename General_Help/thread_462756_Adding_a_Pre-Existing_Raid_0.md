### Post by pseudonym on 2007-06-03
You may need to install the package 'dmraid' to get Ubuntu to work with your RAID 0 array. There's a good howto about it [here]("https://help.ubuntu.com/community/FakeRaidHowto?highlight=%28fakeraid%29"). It deals mostly with installation but you should still find it useful.

In a Linux-only system you can normally just use what's called "Linux software RAID" but in multi-boot scenarios (or if it's just your preference) 'dmraid' is needed so that Linux can work with the RAID BIOSes contained in devices like Promise controllers.
