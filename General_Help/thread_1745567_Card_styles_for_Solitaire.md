### Post by hawthornso23 on 2011-05-01
Install the package   gnome-games-extra-data  via synaptic to get extra card themes. 

The install button in aisleriot is grayed out due to bug
[https://bugs.launchpad.net/ubuntu/+source/gnome-games/+bug/652181](https://bugs.launchpad.net/ubuntu/+source/gnome-games/+bug/652181)
