beagle-search --no-tray

### Post by Kerry_W on 2008-03-13
right click on the applications menu and choose Edit Menus. (or go System-Preferences-Main Menu)
Highlight Accessories and then click New Item
type in a name for it ie "Beagle", click browse to find the command, add a comment if you want to. 
click ok and you should be all set.

Kerry
