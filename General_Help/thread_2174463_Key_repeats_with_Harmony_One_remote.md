My overall goal is to get my Harmony One to properly control XBMC. I'd like to use the 12.2 version of XBMCbuntu. I have previously followed [this guide]("http://forum.xbmc.org/showthread.php?tid=95063") to set things up for Windows. Now I'd like to transition to Linux, but things aren't working as expected. The guide basically suggest setting up two devices in the Harmony to control XBMC. The first is the remote with the "Windows Media Center SE" profile and the second is the keyboard with the "MCE Keyboard" profile. The keyboard allows for some more advanced functionality.  Everything command from the remote is working properly, but most commands from the keyboard are not.  Commands from the "MCE Keyboard" profile repeat indefinitely.
 
I'm using genuine Microsoft components: model 1044 keyboard, model 1039 remote, and model 1040 IR receiver.

I opened a terminal window and pressed a keyboard key on the Harmony.  Sure enough, the key just repeats in the terminal window until I hit ctrl+c with a real keyboard.  Pressing the same key on the 1044 keyboard worked as expected, no repeats.
 
This same setup works on Windows 7.  I plugged in the 1040 receiver into my windows box and both the Harmony One and the 1044 keyboard sent the key command with no repeats in the command prompt.
 
So, is the problem the Harmony One or Ubunutu?  If it was Ubuntu the real 1044 keyboard wouldn't work, right?  I can't be the Harmony or it wouldn't work in Windows.  
 
Anyone have any advice on how to make this work?
