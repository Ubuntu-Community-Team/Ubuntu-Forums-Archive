### Post by taslinux on 2010-01-06
If you have several windows open in a workspace, the usual way to switch from one to the other is to use the window list on the bottom panel, although you can also use Alt+Tab on the keyboard.

There's a third way, and it's especially useful if you've set up your desktop with only vertical panels, because the Gnome window-list applet doesn't work very well as a vertical stack of buttons.

Go to System>Preferences>Windows, and under Titlebar Action, choose 'Roll up' from the drop-down menu.

When you next open a window, double-click on the titlebar. The window will roll up into the titlebar like a shade on a real (glass) window. To make the window roll down, double-click on the titlebar again or click on the window's button in the window list.

Next time you have several windows open, arrange them so their titlebars don't overlap. Use rolling up to get from one window to the other. In other words, rollup the window you're working with, the roll down the next window you want.

This might sound complicated, but you may find it's easier and faster than minimising the unused windows. Rolled up windows haven't disappeared as minimised windows - they're still on the desktop as titlebars, and accessible without going to the window list in the panel.

Rolling up is also handy with multi-window applications like the GIMP and XSane. You can roll up and roll down each window separately.
