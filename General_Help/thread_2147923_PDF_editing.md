### Post by Lars Noodén on 2013-05-23
As noted above, editing PDFs will be difficult if at all possible.  If it is at all possible work with the upstream program that created the PDFs and then export the modified files to PDF.  PDF itself was never meant for editing and is only kind of a holding area before the document heads of to either the printer or the bit bucket.
