### Post by 2CV67 on 2021-08-29
For many years, I used "Drawers" to easily launch my favorite files.
Drawers allowed me to pin my chosen files (about 15) in my chosen fixed positions in a window which opened with one click on the Applications Launcher (Dash to Panel) & closed automatically after selection.

Since Ubuntu 20.04 Drawers is no longer available.

Thanks to Dennis N in another thread, I found that Gnome Shell Extension "Files View" seemed to do just what I wanted!
I could pin any file or folder in the "Favorites" area.
I could access Files View Favorites in one click from the panel.
I could launch any Favorite in one simple click.
Files View closed automatically afterwards.

But since 21.04, Files View is no longer available!

So I am again looking for a Favorite Files Launcher...

Thanks for any suggestion!
