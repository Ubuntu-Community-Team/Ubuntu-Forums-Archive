### Post by bruce89 on 2006-05-25
K3b for KDE or Gnomebaker for Gnome.  If you want to just record data in Gnome, just use the file manager (nautilus) (by going to the Go menu CD/DVD Creator or by going Control-L, and entering burn:///)
