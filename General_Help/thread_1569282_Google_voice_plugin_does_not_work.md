### Post by fragos on 2010-09-06
I installed that plugin while running the Chrome browser and video chat was available but wouldn't work until a plugin update. In Gmail-> Settings-> Chat -> Voice and video chat: I can see my Google Talk Plugin is v1.5.1.0. Try Verify Settings to select the correct options.

If it indicates you don't have the Video Chat feature, enter about:plugins in Chromes location bar and you'll see a list of plugins that are installed. Make sure Google Talk Plugin is in that list and that it's enabled. If it isn't there, go to [http://mail.google.com/support/bin/answer.py?hl=en&ctx=mail&answer=100173](http://mail.google.com/support/bin/answer.py?hl=en&ctx=mail&answer=100173) and install again.
