### Post by Zorael on 2008-04-26
Plasma is the new desktop Thingy included in KDE4. To my knowledge, it's not possible to get it running under KDE3, but I'd be happy to be proved wrong.

So either try it out on a live cd, install it in a new partition, or install it beside your current Kubuntu installation.
```
$ sudo aptitude install kubuntu-kde4-desktop
```
