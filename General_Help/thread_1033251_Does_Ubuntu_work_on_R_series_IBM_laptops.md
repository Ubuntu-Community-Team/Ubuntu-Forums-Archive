I've seen Linux installed on IBM R Series. The best way to find out is by using a LiveCD. If everything, including networking, graphics, sounds etc works with the LiveCD, then go for the full install. Xubuntu is only recommended for PCs with low specs. Some of the applications are different than Ubuntu and its visually less appealing.
[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

I strongly recommend Hardy Heron LTS which is supported for 3 years unlike Intrepid which seems to have more bugs. Hardy 8.04.2 is releasing sometimes soon.

Good luck!!!
