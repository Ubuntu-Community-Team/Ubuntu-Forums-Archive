### Post by Sabot on 2006-11-21
make sure you uninstall the old version of libmjpegtools before you try to install the new one.  If you need the old version for some key programs, my solution for using Cinelerra may not work for you. Also, you will want to install mjpegtools with Synaptic after you get cinelerra installed.  You will get a broken package error when you install mjpegtools, but it will use the new libmjpegtools.
