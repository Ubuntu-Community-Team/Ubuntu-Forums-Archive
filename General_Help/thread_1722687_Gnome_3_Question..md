### Post by linuxyogi on 2011-04-06
In Ubuntu & some other distros installing the package called gnome-shell works. Although its not the complete thing but you can try it out.

```
apt-get install gnome-shell 
``` 

& then

type

```
gnome-shell --replace
```

Just don't close the terminal.

If elementary is not debian then just use whatever package management system it uses.

I am not sure if your distro have included the package in their repos.
