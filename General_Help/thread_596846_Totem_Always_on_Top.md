I'm having a strange problem, a couple of days ago whenever I start up Totem and play a movie Totem defaults to Always on Top in GNome. I uncheck it, close the application, restart the application and always on top is no longer enabled. Open a movie though and it reenables itself by default. Can't find anywhere in preferences where I can disable this and I don't remember it doing this any time since I upgraded to Gutsy. How can I tweak this to stop this from happening? ### Post by daengbo on 2007-10-30
Your config for Totem got set to "Always on top." Why? I don't know. Open Totem. Go to Edit -> Plugins
Uncheck the "Always on Top" plugin.

That should do it.
