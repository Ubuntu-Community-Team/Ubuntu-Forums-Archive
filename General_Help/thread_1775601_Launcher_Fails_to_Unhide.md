### Post by garvinrick4 on 2011-06-05
Open
```
ccsm
```
and check in Unity plugin, click on it. Settings there.
If you do not have ccsm installed do so.
```
sudo apt-get install compizconfig-settings-manager
```
Now open ccsm and get your settings how you like them.
