### Post by efflandt on 2013-12-02
What does "lspci" show for your card reader? Does that SD card work in a USB card reader or any other device (that can handle whatever file system it has)?

Read "man apt-get" about the difference between **upgrade** and **dist-upgrade**. "upgrade" may not install packages if dependencies change, which may have broken something else. You should likely be using "dist-upgrade" which is smarter about installing any new required dependencies. Note that dist-upgrade does not automatically upgrade to a newer distribution, it just more completely upgrades your current distro.
