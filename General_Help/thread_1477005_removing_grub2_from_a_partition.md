### Post by oldfred on 2010-05-08
While installing grub to a partition causes major problems for windows since it already has data there, an install to the Ubuntu partition does not cause any problem and you will only see it by running the script. The first sector is reserved for partition boot data. 

As long as you have installed to the MBR you should be ok.
