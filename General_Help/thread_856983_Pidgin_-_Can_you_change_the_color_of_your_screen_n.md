### Post by jonlemur on 2008-07-12
It looks like you can go to Tools->Plugins and use either the Pidgin GTK+ Theme Control or the Conversation Colors plugins.  Click on the checkbox by their name then click on Configure Plugin to change the colors.

### Post by jonlemur on 2008-07-12
I went to Tools->Plugins and checked the box for "Pidgin GTK+ Theme Control"  Then I clicked on "Configure Plugin" and selected "Sent Message Name Color"  I changed that to a non-default color and clicked "Write settings to..."  The next message I sent had the new color.  You should also be able to edit the "Received Message Name Color" in the same way.
