### Post by kaibob on 2010-01-26
There are a number of ways to do this but clonezilla is probably the most popular. Don't be put off by the cluttered interface--just accept the defaults when in doubt. Clonezilla is run from a live CD and the drive image is typically saved to an external hard drive. I don't know of any cloning programs that will boot and run from an external hard drive. 

I have been using clonezilla for about a year and have found it to be very reliable. 

[http://clonezilla.org/](http://clonezilla.org/)

BTW, be sure to use the Ubuntu-based version. It work's best with ext4.
