### Post by vinnn on 2007-03-18
Put simply... No.

You can install the VMware Server Client which is a simple client for connecting to virtual machines running in VMware Server, either locally or on a remote machine running the VM.
In other words VMware Server is still running the virtual machine but you connect to the virtual machine from the client software.

You can download the Linux package over at the [VMware Server download page]("http://register.vmware.com/content/download.html").
