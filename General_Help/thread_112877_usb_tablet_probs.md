### Post by new2hoary on 2006-01-06
This doesn't seem to be well supported in linux.

Someone has been working on a driver, but it would appear to have limited usability;

[http://www.pell.portland.or.us/~orc/Code/iPen/](http://www.pell.portland.or.us/~orc/Code/iPen/)

It looks like this driver will allow you to use it as a mouse within X but thats about it. It wouldn't be any good for use within gimp.

It also seems that this device was discontinued a long time ago - so it is unlikely to ever be well supported.

If you want a tablet for Linux look at the Wacom range - these are well supported - but check the model first - the latest models might not have support yet.
