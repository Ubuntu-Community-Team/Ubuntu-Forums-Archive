### Post by lisati on 2010-04-12
The main differences from a user's perspective are the choice of [GUI]("http://en.wikipedia.org/wiki/Graphical_user_interface") and the programs that come package with the [distribution]("http://en.wikipedia.org/wiki/Linux_distribution"). Other than that, they're pretty much the same [Operating System]("http://en.wikipedia.org/wiki/Operating_system").

The "Standard" Ubuntu uses [Gnome]("http://en.wikipedia.org/wiki/GNOME") for a GUI, while Xubuntu uses [xfce]("http://en.wikipedia.org/wiki/Xfce")

plz reply

Xubuntu uses an entirely different desktop environment.  You might like it, or you might not.

It's a bit of a puzzle, actually.  XFCE, the desktop which is used in Xubuntu, is a lightweight one.  It's geared toward older machines with less memory, slower processors, etc.  But for some reason the devs added enough bloat that it actually uses more memory than the full Gnome version.

I can't see any reason to use it unless one has an affinity for the Xfce desktop.  It looks and acts completely different, and you'll be lost at first because it's so different.

Hope this helps some.
