### Post by tredegar on 2011-01-07

> 1 Turn on the computer.
2 Press the F2 key when the VAIO logo appears.
 The BIOS setup screen appears. If not, press the F2 key several times.

### Post by Ten98 on 2011-01-12
Sadly the BIOS menu on the Vaio is somewhat limited and does not allow you to change much.

To edit some of the jucier settings like VT support, AHCI mode etc you must use a manual BIOS settings dump / upload tool like Symcmos.

This was the only way I could enable AHCI mode on my Vaio VGN-FE31M laptop. It is difficult to track down which hex values you need to change but they are all out there on the forums.
