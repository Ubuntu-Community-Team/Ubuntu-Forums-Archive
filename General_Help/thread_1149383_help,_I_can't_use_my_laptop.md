### Post by sacredeagle on 2009-05-05
Firstly: it is considered very rude to write a message entirely in **Bold** letters, and using phrases like **HELP ME PLEASE? **sure makes it hard for anyone to reply to such whining and yelling. Everyone is here to help or be helped so good manners is essential.

What I would first of all do in this case is to find a Linux installation CD that is able to start a system from the hard-disk, I think this is possible with the Ubuntu installation CD, and I know this is possible with a SuSE installation CD. 
By doing this you will be able to find out if the system is OK, and you can re-install GRUB to MBR from there.
