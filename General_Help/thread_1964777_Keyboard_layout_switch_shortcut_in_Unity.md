I have two languages in the keyboard layout program so when I want to change the keyboard layout I have to make a click in the Unity panel and then select the desired language.

### Post by Canaryguy on 2012-04-28
Well, I found how to do it.

1º go to **Language Support** application.
2º Options
3º Key(s) to change layout
4º Choose your desired key combination to change the keyboard layout.
