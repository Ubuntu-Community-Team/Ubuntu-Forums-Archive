### Post by COMECON on 2012-10-30
What do you mean with virtual drive?
If you mean a virtual machine, of course you can.
If you mean an emulated drive, then you can't, because you need to boot from the ISO so you need to burn it somewhere (you can't boot from an emulated drive).
