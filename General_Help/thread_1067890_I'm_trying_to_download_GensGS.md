### Post by kanikilu on 2009-02-12
It looks like they have a Debian package (.deb) for Ubuntu 8.04+, 32-bit. Just right-click this link and choose "Save Link As..." (it's a direct link to the file you want):

[Gens/GS Milestone 6 (Ubuntu 8.04+, 32-bit)](http://info.sonicretro.org/images/5/56/Gens_2.15.5-gs-m6-1_i386.deb)

Save this file to your desktop and then double-click it to install it.

For a guide on how to manually install deb packages, see this page:

[http://www.psychocats.net/ubuntu/installingsoftware#gdebi](http://www.psychocats.net/ubuntu/installingsoftware#gdebi)
