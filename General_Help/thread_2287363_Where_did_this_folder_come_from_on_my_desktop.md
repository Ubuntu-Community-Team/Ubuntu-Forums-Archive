### Post by yetimon_64 on 2015-07-18
Is there a "htm" or "html" file on the desktop with the folder as well ? 

The "_files" part of the folder name looks like a web (html) page has been saved. Some photo gallery applications can save to a webpage style for uploading to a website. Of course any browser can also "Save page as" a "Web page, complete".

Are the images in the folder familiar to you or totally random/unknown ? If they are familiar and you've been viewing them recently in such an application it is possible you may have inadvertently saved them in that format.

If the image themselves in the folder were unknown to me either from my filesystem or off a website I'd recently viewed and are not wanted. I'd personally delete them as a mistaken saving of some site etc.
