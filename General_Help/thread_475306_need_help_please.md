Funnily enough, I noticed the other day while I was installing Feisty that the installer lets you use a USB stick as an installation medium! Just plug it in before you run the installer.

Make sure you don't install the MBR on the harddisk at the end.

Running Linux off a USB stick sounds slow to me. Writing a lot to the stick will probably wear it out quick. Why don't you install [VMWare]("http://www.vmware.com/products/server/") in XP and run Ubuntu in a window?
