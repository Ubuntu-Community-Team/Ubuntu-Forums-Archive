I've just moved over from Gentoo and have been enjoying the speed and accessibility for a while now.

However, there are times when I miss the ability to bring in a bleeding edge package easily.

Now Gentoo can actually be installed within another distro. I've only seen this mentioned in order to convert a distro to gentoo. 

You just extract portage to /usr and then I think you start bootstrapping.

However,

As an example you could dual boot Gentoo and Ubuntu and run programs across from each. That's a simple example, I believe there could be a way to do it on the same filesystem, perhaps using /etc/fstab, perhaps using chroot.

This way you get it all, at the price of disk space, but since each distro is 6gb average/max it then that's nothing on todays 300gb drives.
