### Post by shakin on 2005-06-16
My first suggestion would be to 'apt-get install mozilla-browser' so you can search for help easier.

My second suggestion is to 'mv ~/.mozilla/firefox ~/.mozilla/firefox_old' and see if that lets you back in. You can get your bookmarks and stuff back from the firefox_old directory.

My third suggestion is to [read this how-to](http://ubuntuforums.org/showthread.php?t=40880) about installing Firefox 1.1. Actually, you can install any version of Firefox from mozilla.org that way, but I recommend 1.1 because it completely rocks :)
