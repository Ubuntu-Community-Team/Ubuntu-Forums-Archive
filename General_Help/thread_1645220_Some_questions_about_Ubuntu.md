### Post by timgood on 2010-12-14
If you download the live CD installer for Ubuntu, you can boot from the CD drive into a full working Ubuntu desktop without making any changes to your hard drive. This will enable you to check if all your hardware works correctly. If you have any specific problems with hardware not working, there are usually workarounds and you should then come back to the forum for help.

Good luck - I hope you find Ubuntu as friendly and powerful as I and many other users have!

### Post by coffeecat on 2010-12-14
Hi and welcome to the forum. 

> **HeLp.Me said:**
> I've been reading up on alternative OS and think Ubuntu sounds to be the most user friendly Linux distribution for a beginner like myself, who knows nothing about programming or networks and servers, areas where Linux seems to have a high usage.

Ubuntu is a fine system for everyday desktop use, so don't worry about not knowing about programming, networking or servers. In terms of GUI usability, Ubuntu is nearly on a par with both Windows and MacOS. If you see a preponderance of terminal commands in thread posts and internet howtos, that does not necessarily mean you have to use the terminal for everything, just that it's sometimes quicker and easier to get the task done. 

> **HeLp.Me said:**
>  I have an Acer Aspire 5050 running Windows XP, but was hoping to start new with an entire Ubuntu system and do away with Windows for good. The one hesitation I have is that Linux seems to have considerable compatibility problems with certain hardware, and that my laptop mightn't work the same.

Think about dual-booting until you are sure you want to abandon Windows. Six years after starting with Linux I still use Windows occasionally. There's no harm in that.

> **HeLp.Me said:**
> 1. Should I use standard Ubuntu Desktop or the Netbook version for my laptop?

If that's not a netbook, go for the standard desktop version.

> **HeLp.Me said:**
>  2. I have a built in 802.11g wireless adapter from Broadcom. Is this compatible with Ubuntu or can I just use a PCMCIA card, namely [this one]("http://cgi.ebay.co.uk/Ubuntu-Linux-Wireless-54MB-802-11g-Laptop-Adaptor-/290475282623?pt=UK_Computing_Networking_SM&hash=item43a1ad10bf") or this [USB adapter ]("http://cgi.ebay.ie/Ubuntu-Linux-USB-2-0-Wireless-802-11g-54MB-Dongle-/290441190211?pt=UK_Computing_Networking_SM&hash=item439fa4db43")?

Broadcom is a bit of a pain because licensing issues prevent the driver/firmware being included in a default install. Before you get totally sidetracked by an unnecessarily complicated howto on the internet, have a look at the Ubuntu community guide:

[https://help.ubuntu.com/community/WifiDocs/Driver/bcm43xx](https://help.ubuntu.com/community/WifiDocs/Driver/bcm43xx)

> **HeLp.Me said:**
> 3.  My PCMCIA / Card Reader only works on Windows XP, it's Release.txt calls it an ENE CB-712/714. Is there an alternative or way to make it work with Linux?

Have you tried it yet? It may very well 'just work'.

> **HeLp.Me said:**
>  4. I also have a Realtek audio driver, but have already found a few fixes for that online, since I don't have Ubuntu installed at the moment, I can't try it out.

Ditto for Realtek audio. It may just work. Bear in mind that many "fixes" posted to the internet are by people who have a talent for making life difficult for themselves - and others. :wink:

Try booting up with a live Ubuntu CD and see what works and what doesn't work. Then take it from there.

Good luck!
