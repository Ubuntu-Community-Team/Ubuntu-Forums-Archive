### Post by Yetisaurus Akjas on 2012-05-14
Try zooming in with Ctrl and + or in the "_V_iew" menu, at the "_Z_oom" option, select "Zoom _I_n". 

From my experience, reinstalling Firefox doesn't reset all its settings unless you delete the folder /home/(username)/.mozilla/firefox while the browser is not running (for safety). You can also delete the entire folder /home/(username)/.mozilla when the browser is not running, as it will be created again the next time you start Firefox. 

'Hope this will help you. :)

### Post by prshah on 2012-05-14
> **Nesgaard1 said:**
> I was on the  internet when my page shrunk. I think it was my thumb on the mouse as I was typing. It effected only the single page and when shut it down it came up the same way. i even reinstalled firefox to no avail.  Can anyone tell me how to repair this?

You can use Ctrl+Mouse Scroll Up to Zoom In (bigger) and Ctrl+Mouse Scroll down to Zoom out (smaller).** Press Ctrl+0 to reset to default in Firefox**.

If this doesn't work, then you are probably using an Addon such as NoSquint. Please check settings for the relevant addon, if you are in such a situation.
