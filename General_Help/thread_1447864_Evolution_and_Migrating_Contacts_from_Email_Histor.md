Evolution has a feature to automatically add each contact when replying to an e-mail.

Since you (probably) don't want to actually send e-mails just to do this, you should set up a dummy SMTP server on your system that actually doesn't not send out mail, set up Evolution to use it, and then do a bulk Reply to all those e-mails.

That should add them to the Contacts without causing too much grief.
