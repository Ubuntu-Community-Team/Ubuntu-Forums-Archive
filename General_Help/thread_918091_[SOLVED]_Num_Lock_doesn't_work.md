### Post by gmoney on 2009-05-05
Cool, thanks for the solution.  That was quite annoying.  I know for a fact that I didn't change that setting.  I did, however, just upgraded my kernel to 2.6.24-24-server.  This numlock issue happened on the first reboot after the new kernel.

This leads me to believe that this is a bug.  Any ideas on where one would submit a bug of this nature? I would think the Ubuntu launchpad, but perhaps it should be submitted to the kernel development team?

Also, for anyone else who runs into this problem in the future:  

If you're running gnome on ubuntu, click on 

system > preferences > keyboard

Then, click on the "mouse keys" tab, and deselect "Allow to control the pointer using the keyboard".

If that doesn't work, you might want to try this:
[https://help.ubuntu.com/community/NumLock](https://help.ubuntu.com/community/NumLock)

Cheers.
