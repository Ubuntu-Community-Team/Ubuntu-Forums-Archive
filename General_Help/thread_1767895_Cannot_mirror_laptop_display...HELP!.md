### Post by jerryjustly on 2011-05-26
You might give the following instructions a try.  What I've done is hooked up to an external monitor and turned it on using my keyboard and wrote the following instructions while I was doing it.

1. Press Alt + F2 (This opend the run dialog)
2. Type
gnome-display-properties
3. Press Enter
4. Wait for a few seconds (A window is opening)
5. Press Spacebar (This should put a check in the "same image in all monitors" box.
6. Press Alt + A (This Applies the changes)
