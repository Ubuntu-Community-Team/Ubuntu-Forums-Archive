### Post by 3rdalbum on 2009-12-22
Check the CD for errors at the startup menu. If it comes back as clean, then you would seem to have a hard disk problem.

One thing you could try is creating a 3 gigabyte partition at the beginning of the disk, and then installing Ubuntu in a partition after that third gigabyte. If you've got bad blocks at the start of the disk, then this may help you get going.
