### Post by bmt on 2007-01-20
Printing is problematic in Ubuntu (sorry, I can't find the thread on the support forum).

My experience is that Xara will only print to the default printer.  Try setting the CUPS-PDF printer as default (in the Ubuntu Printers settings) and printing again.

The Xara print dialog has no effect on the printer selection (in my Dapper).
