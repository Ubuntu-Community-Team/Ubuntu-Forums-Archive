### Post by C.S.Cameron on 2011-05-12
Startup Disk Creator only works with Ubuntu based iso's.
UNetbootin does not work with Windows iso's.
Making a Windows 7 install disk is easy with Ubuntu, format NTFS and set boot flag but that doesn't work with XP.
You might try the EEE PC site for info on making an XP installer.
