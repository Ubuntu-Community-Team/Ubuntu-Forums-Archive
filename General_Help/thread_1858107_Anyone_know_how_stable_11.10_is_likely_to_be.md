### Post by Mark Phelps on 2011-10-11
If Ubuntu had a "roll-back" option in which, with a single mouse-click, you could "restore" 11.04 in a few minutes, I'd say go for it -- but, alas, there is no such option.

Thus, given your situation, I would strongly recommend the following:
1) Install 11.10 in a dual-boot scenario so if the worst happens, you still have a working 11.04 setup
2) Wait until farther down the road to install 11.10 -- when you have Internet access to either research solutions to problems and/or download upgrades and patches.

### Post by mr.interested on 2011-10-16
> **wolfen69 said:**
> Please don't spread FUD. A lot of people are already using it, and it is fine. As a matter of fact, I've heard a *lot* of good things about it. I myself have been using it for weeks without issue. Seems to be a solid release. But like any other OS, your mileage may vary. Try the live cd when it is released in a couple of days and see for yourself.

The system itself is almost solid, but there are lots of driver-related issues, all of which were abstent in Ubuntu 11.04:

- Soundblaster Live! 24-bit (External USB) doesn't work properly ([http://ubuntuforums.org/showthread.php?t=1860273](http://ubuntuforums.org/showthread.php?t=1860273)).

- Adjusting screen brightness doesn't work properly ([http://ubuntuforums.org/showthread.php?p=11351226#post11351226](http://ubuntuforums.org/showthread.php?p=11351226#post11351226)).

In addition, there are problems with software, and some of them are not updated to Ubuntu 11.04. For example, there are still small issues with Dropbox.

So no, it's not spreading 'FUD'. We need to install the laster Ubuntu so that we can give feedback. Otherwise those problems will not be fixed! Having said that, if OP goes somewhere for 6 months without access to internet, I don't recommend installing Ubuntu 11.10, unless problems are fixed or are not present on OP's system.
