### Post by frodon on 2006-05-31
Yes there's an option in synaptic to add a CD as a repository So add the CD as a repository refresh and update all. When all is done don't forget to change your sources.list file to point on the dapper repositories.
You need the alternative CD for that not the desktop CD.
