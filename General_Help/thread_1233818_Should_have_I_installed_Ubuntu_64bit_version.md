### Post by 3rdalbum on 2009-08-07
It's difficult to use 4 gigabytes of RAM on a laptop, unless you copy DVDs or something (in which case the cache will fill up with DVD data, which won't really be very useful).

You might as well install the 64-bit edition anyway, it's almost exactly the same as 32-bit except for the memory.

### Post by philinux on 2009-08-07
There are only a couple of important apps to me that aren't 64 bit. Acroread and flash. Adobe have release an alpha 64 bit flash which I use just fine. For the minority 32 bit apps there's ia32libs which will let you install 32 bit apps. Firefox is 64 bit native in repo's too.

I've been running 64 bit for over a year now with zero problems.
