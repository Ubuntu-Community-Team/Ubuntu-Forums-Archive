### Post by luisromangz on 2008-03-27
Ktts is the text to speech manager for KDE. It makes possilble for KDE apps to use voice to notify events or read a pdf file or a website.

It can use several backends, being the most useful festival. A way to use festival with Gnome must exists, but as a KDE user I don't know.

There aren't useful voice recognition sofware for Linux. But I have heard recently that Google paid the Wine developers to make Wine run Dragon Naturally Speaking correctly.
