### Post by Mark Phelps on 2011-05-21
Basically -- NO. If there was a simple way to bypass the password, then there wouldn't be much point in having one, would there?

And, if you get suggestions about password crackers available from various sources, realize all the examples used to support such things use simple, easy-to-guess, very short passwords.

Any sufficiently complex (as in USEFUL) password is going to take anywhere from days to weeks to crack.
