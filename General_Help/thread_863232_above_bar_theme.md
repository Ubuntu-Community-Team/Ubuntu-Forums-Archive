### Post by NilsE on 2008-07-18
> **cyril_hoi said:**
> ok ok, thank you

but how do you install a gtk 2?? in ubuntu 8.04 compiz fusion?
GTK is running even when Compiz is.  You can drag and drop a theme you downloaded into the appearance preference to install it.  Just open the appearance tool under preferences then go to the appropriate tab under customize to install it.

Alternatively you can extract the archive to the /usr/share/themes folder. If it is an icon theme then extract it to the /usr/share/icons folder.  

You will need to do this as root so open the archive in nautilus by starting nautilus with gksudo nautilus in a terminal and clicking on the archive file.
