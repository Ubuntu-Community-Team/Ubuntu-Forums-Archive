### Post by ianmillington on 2009-09-04
Can't actually say what happened in your case.

You may find this guide to be a helpful download

[http://www.tuxradar.com/linuxstarterpack](http://www.tuxradar.com/linuxstarterpack)

So far as getting packages is concerned, in the main forget downloading, extracting and installing. That's the windows way. The Linux way is via the package manager (Add/Remove software- it has a different meaning than in Windows).  From within there you will find thousands of packages available. They are held on servers across the globe in repositories. Just click the ones you want and hit apply. The package you selected (and all dependencies) will be downloaded and installed for you. I think Opera (which I've not used myself) might already be in there.

### Post by Grenage on 2009-09-04
It sounds like you are running a live CD, rather than installing Ubuntu.  Anything installed on a live CD will be lost on reboot.  To install it, select the install option at the boot-time menu.

### Post by 3rdalbum on 2009-09-04
When you go to the Opera website, it automatically selects the appropriate package for you as an Ubuntu user. This is a Debian package, therefore the name always ends in ".deb" when you download it.

Double-click this package, and then click the "Install Package" button in the new window that opens. When it is done, you will have the Opera entry in your Applications menu, under Internet.
