### Post by 98cwitr on 2010-04-12
Georgia...not United States Georgia ;) (jp, I'm from Georgia here in the US...had to do it). Anyway...

[http://www.tips5.com/install-compiz-fusion-and-enable-cube-atlantis-effect-on-ubuntu](http://www.tips5.com/install-compiz-fusion-and-enable-cube-atlantis-effect-on-ubuntu)
[http://ubuntuforums.org/showthread.php?t=789801](http://ubuntuforums.org/showthread.php?t=789801)

You'll need these dependencies before you get started:

```
sudo apt-get install build-essential compiz-bcop compiz-dev git-core libcairo2-dev libglu1-mesa-dev libtool libxss-dev
```

it's really not hard...once you get used to it, installing applications is much easier in linux than Windows. 

As for firefox...all Linux distros come with it...now if you want to **update** firefox to 3.6.x then just follow these really simple steps

```

sudo apt-get update
sudo apt-get upgrade

```

These two simple commands will also update your system and all applications installed on it. 

You can also do this from the Update Manager under System->Administration

### Post by lovinglinux on 2010-04-12
Firefox 3.6 will be only available in the next release of Ubuntu, called Lucid Lynx. Ubuntu does not offer major updates for the applications distributed with each new release, only security updates. Which means you can update from firefox 3.5.3 to 3.5.4, but not from 3.5.x to 3.6. Nevertheless, there are a couple of ways to get the latest Firefox when it is released by Mozilla. See the [[COLOR="Sienna"]**Installing Other Versions**[/COLOR]](http://lovinglinux.megabyet.net/?page_id=220#Installing-Other-Versions-1) section of [Firefox optimization and troubleshooting thread](http://ubuntuforums.org/showthread.php?t=1193567).
