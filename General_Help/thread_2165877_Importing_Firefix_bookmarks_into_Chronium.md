### Post by nerdtron on 2013-08-06
Haven't tried importing the .json file to chrome but I think this method will work.
1. Install firefox in lubuntu.
2. Import your bookmarks there.
3. Close firefox.
4. Open Chromium and it will ask you to import bookmarks from firefox.

Or better yet. Create an account in Chromium right now and then import your existing firefox bookmarks. Then on the new install of lubuntu, sign in to Chromium and everything will be in sync.


Edit: BTW, you're still using .json as backup? Firefox (and Chromium) has a sync feature, all bookmarks can be automatically sync between different firefox versions and installations.
