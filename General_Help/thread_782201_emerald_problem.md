### Post by FastZ on 2008-05-04
What you posted a link to is the emerald theme only.  To install the emerald theme, open Emerald Theme Manager and then click the Import button I believe it is and then import that emerald theme.  If you want the gtk2 theme to go with it, you'll have to download this: [http://www.gnome-look.org/content/show.php/Darker+Ice?content=70611](http://www.gnome-look.org/content/show.php/Darker+Ice?content=70611)

The darker-ice.tar.gz file in that zip archive is your gtk2 theme.  Just import that via your System>Preferences>Appearance>Theme tab then click the install button and install the darker-ice.tar.gz archive.
