### Post by sudodus on 2016-06-13
If you are running a current version of an Ubuntu family flavour (for example Xubuntu live or installed), you can use mkusb to create a ***persistent live*** drive in [another] USB drive.

Install mkusb according to this link: [https://help.ubuntu.com/community/mkusb](https://help.ubuntu.com/community/mkusb)

Create a persistent live system according to this link: [mkusb/persistent](https://help.ubuntu.com/community/mkusb/persistent)
