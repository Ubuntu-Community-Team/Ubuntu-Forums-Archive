### Post by tkjacobsen on 2006-05-28
in Ubuntu the drivers are in the kernel (the core of the system) The Ubuntu kernel should have most drivers needed. If you miss anything explicitly post it in theese forums to get some help. (especially wireless networking will/can give you some trouble, and maybe some printer drivers)
 
Programs are installed using the package manager. In gnome it is synaptic (system -> administration -> synaptic package manager) in KDE it is adept (don't remember where it is located) You can just search for packages, and choose to install them, then just press apply. 

To add more packages you can possibly install you have to add som extra repositories. See the following link:
[https://wiki.ubuntu.com/AddingRepositoriesHowto](https://wiki.ubuntu.com/AddingRepositoriesHowto)

Post if you have any trouble, I'll be following this thread
