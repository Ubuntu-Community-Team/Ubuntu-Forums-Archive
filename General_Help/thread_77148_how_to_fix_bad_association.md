### Post by treris on 2005-10-16
In Hoary you could change file associations by going to KDE components in the Settings menu, I can't find it in the Breezy System Settings, but the Settings menu is also still available throug the system button in the taskbar.
If you've found it, go into file associations and type in the extension you would like to change preferences for and the default application with which to open it will appear on the righthand side, simply change that to whatever you like and you're all set

hope this helps
