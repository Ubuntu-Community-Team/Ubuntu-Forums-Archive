### Post by supergrav on 2011-02-20
I think there is no need for a fresh installation. You can just update your current system via the traditional method. LTS means that updates for this release will be provided for a longer time period than ordinary releases.

### Post by PaulW2U on 2011-02-20
> **Scunnered said:**
> Can you please advise me how I apply the new update to my system? When I looked at the Canonical site I found that this was being offered as an ISO download.
Type **lsb_release -a** in a terminal window, it should say that you already have 10.04.2 if you are fully updated. The new ISO will minimize the number of updates needed for anyone installing 10.04 for the first time.

### Post by howefield on 2011-02-20
LTS releases usually have several point releases mainly updating the image as PaulW2U points out, but there can also be other fixes such as the mentioned installation system corrections. These won't affect your currently up to date system but if you wanted a reinstall, it would preferable to use the point releases. I think there is 10.04.3 and 10.04.4 to come before this LTS goes end of life.
