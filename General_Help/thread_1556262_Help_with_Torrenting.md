### Post by NCLI on 2010-08-19
The torrent application Transmission is installed by default. If you want something different, just go to the Software Center.

If you want to run Windows applications(Like utorrent), just go to the Software Center and install Wine 1.2. Please note that not all Windows applications will work as Wine is still Work-in-progress, but utorrent should work just fine. 

Check [this website]("http://appdb.winehq.org/") for a list of Wine-compatible applications.
