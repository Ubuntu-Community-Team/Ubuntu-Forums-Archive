### Post by gzarkadas on 2010-05-04
For start try the manuals for:

[LIST]
[*][Bash]("http://www.gnu.org/software/bash/") (there is an online manual there, or just use the command)
[/LIST]
```
info bash
```[INDENT]to get the local version.
[/INDENT]
[LIST]
[*][Advanced Bash-Scripting Guide]("http://tldp.org/LDP/abs/html/index.html"). This is an in-depth shell scripting guide; worth to read.
[/LIST]

[LIST]
[*][GNU Coreutils]("http://www.gnu.org/software/coreutils/") (there is an online manual there, or just use the command)
[/LIST]
```
info coreutils
```
[LIST]
[*][GNU Findutils]("http://www.gnu.org/software/findutils/") (there is an online manual there, or just use the command)
[/LIST]
```
info find
```
[INDENT]to get the local version.
[/INDENT]
