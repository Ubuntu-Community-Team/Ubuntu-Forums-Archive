### Post by Lars Noodén on 2011-05-13
There's a lot of common ground, even [with other systems](http://www.levenez.com/unix/), but where to start all hinges upon what you want to do with the system.   

Maybe start with some books from when Free and Open Source Software got popular for a non-technical background.  
[list]
[*][Open Sources: Voices from the Open Source Revolution](http://oreilly.com/catalog/opensources/book/toc.html) (1999)
[*]The Hacker Ethic and the Spirit of the Information Age.  Pekka Himanen. (2001)
[*]Rebel Code: Linux and the Open Source Revolution. Glyn Moody  (2001) 
[/list]

If you want to start with technical topics, then what will it be interesting for you to do with Linux?  That will indicate where to start.
