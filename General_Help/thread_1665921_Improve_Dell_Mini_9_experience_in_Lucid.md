### Post by cariboo on 2011-01-13
Depending on what Broadcom chipset your system has, you should have no problem with the drivers, the unfortunate part is that you need a wired connection to install it. Just go to System->Administration->Additional Drivers, and choose the recommended driver.

If you have the 945 graphics chipset, the install process will install the i915 driver automatically.
