How can I change what program automatically runs when I put in a USB device?  Sometimes it's Rhythmbox and sometimes it's F-Spot, but I think I'd rather nothing come up and let me start it myself.

---

### Post by jyaan on 2008-11-22
Go to System > Preferences > Removable Drives.

You can select whichever program you like from there.
