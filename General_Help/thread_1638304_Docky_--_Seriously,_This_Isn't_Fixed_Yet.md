### Post by StaRetji on 2011-01-04
I've noticed this too.
For example I would create shortcut on desktop and will drag it to Docky.
If shortcut is deleted for desktop it gets removed from Docky too.
So, workaround was to make a directory with desired shortcuts, drag them to docky - do not delete folder and shortcuts.

I hope that helps.
