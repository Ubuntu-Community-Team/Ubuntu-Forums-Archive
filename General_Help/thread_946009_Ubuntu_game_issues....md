### Post by pytheas22 on 2008-10-12
The vast majority of these games should be available from the repositories.  Open up Synaptic (System>Administration menu) and search for the games by their names, or just search for 'game' and you should get hundreds of results.  Synaptic will automatically download and install these games for you.

Generally it should not be necessary for a casual user to ever install anything from a .tar.gz or .zip file in Ubuntu.

If there's a particular game you want but can't find in Synaptic, let us know the name and maybe we can figure out an easier way for you to install it.
