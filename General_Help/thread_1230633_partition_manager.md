### Post by rbc on 2009-08-03
Ubuntu is going to need its own partition (and swap partition) regardless of how large the hard drive is. You can create the partition ahead of time or the partitioning can be done as part of the installation process. Gparted has its own Live CD
[http://gparted.sourceforge.net/download.php](http://gparted.sourceforge.net/download.php)
Or you can run the Ubuntu Live CD, then install Gparted (into RAM)
Before you install Ubuntu though, back up your important data!!!!!! That is not to discourage you from trying Ubuntu. I have installed Ubuntu many times, as well as a couple dozen other flavours of linux, ans I have only had one install go haywire, but it can happen
