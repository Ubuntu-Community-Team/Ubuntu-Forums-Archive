### Post by 3Miro on 2009-04-05
Go to System -> Prefs -> Keyboard Layouts and you will be able to install different layouts so you can write in pretty much any language. You will be able to write in any program as well, I always skype and send e-mails in non-English.

For Dreamweaver:
Option 1: it is best if you can find a version of Dreamweaver for Linux, however, such might not exit.  

Option 2: Try to get it to run with wine, go to winehq.com and check the database for people who have tried to make the program work. Also, keep an eye on the wine version used, wine today has way more support than it did one or two years ago.

Option 3: Install windows in a virtual machine. Virtual Box is free and if you only need Dreamweaver, you can install the open source version from Add/Remove.
