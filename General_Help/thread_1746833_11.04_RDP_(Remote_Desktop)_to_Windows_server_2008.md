### Post by newschapmj1 on 2011-05-02
Problem

I was trying to connect to Windows server from 11.04
I knew the connection worked from a Windows 7 PC

terminal services client (rdp or rdpv5)
An error has occurred
Details: recv: ~Connection reset by peer

Fix
I'd forgotten about the Windows server security.

Control Panel, System & Security, System
Remote Settings, Remote Desktop has 3 radio buttons.
I changed from most secure to medium security.

I hope this saves someone some time.
