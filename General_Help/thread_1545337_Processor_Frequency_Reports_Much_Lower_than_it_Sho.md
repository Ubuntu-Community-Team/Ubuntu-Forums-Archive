### Post by Universal344 on 2010-08-03
I just built myself a new computer with an AMD Athlon 3 GHz triple-core processor and yet both Hardinfo and Sysinfo report that all three cores are running at only 800 MHz.  I was wondering if this is probably an issue with the reading or if Ubuntu is really only getting 2.4 GHz out of my processor?

---

### Post by prodigy_ on 2010-08-03
This has nothing to do with with Ubuntu. It's a built-in power-saving feature of your CPU. Disable Cool'n'Quiet in BIOS if you don't like it.

[http://en.wikipedia.org/wiki/Cool%27n%27Quiet](http://en.wikipedia.org/wiki/Cool%27n%27Quiet)
