### Post by CatKiller on 2011-07-22
Compose -> t -> h gets you þ.
Compose -> T -> H gets you Þ.

There are lots of fairly intuitive Compose key combinations like that.

Otherwise, if you already know the Unicode number of a character, you can use that directly. Lower-case thorn is 00DE, for example, so you can use Ctrl-Shift-U D E to get Þ.

If you're typing a whole passage from a different layout, you can add that layout in your keyboard preferences and use a shortcut to switch layouts. Or you can use a panel app to switch layouts if that's easier. Which layouts were you having trouble finding?
