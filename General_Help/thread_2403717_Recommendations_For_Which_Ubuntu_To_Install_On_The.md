### Post by TheFu on 2018-10-15
You can install any flavor of Ubuntu on that system, but the heavier GUIs will run better if you have a newer GPU.  A GPU with hardware decoding of h.264, h.265, and mpeg2 will be better for watching videos too.

### Post by l33ch on 2018-11-04
I use Ubuntu Studio 18.04. Was running 16.x. It's a full featured Distro Suite for Audio/Video Production/Photo Editing. Low Latency kernel which is good if you use metronomes, etc. I use it for my general use OS. The distro might have a larger footprint than other flavors (2.8gb compressed vs 1.5gb or so for others, but it's very quick, snappy and stable. Uses xfce and does not use any of that unity, but I believe GNOME3. For something more "lightweight", I also love to use Linux Mint in xfce. Not really a big fan of Mate, or KDE, or Unity. Those are just my 0.02. Happy Linuxing!
