### Post by Mr Nemo on 2010-01-14
First you could try synaptic and search for the programs. System --> Administration --> Synaptic Package Manager. Or open up a terminal and use 'sudo aptitude install <program name>. If that fails you could try to use a tar ball. Google it to find out how to install from it.

Edit: I just installed neverball from the terminal to see what it was.
