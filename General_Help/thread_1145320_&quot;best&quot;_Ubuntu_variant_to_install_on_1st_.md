### Post by FlyingBuzz on 2009-05-01
Hello!
I'm using Ubuntu 9.04 Netbook Remix on my Asus Eee PC 901. And it works perfect. I have all of the hardware work Out-Of-The-Box with no even small problems.
Before upgrading a was using Eeebuntu 2.0 (Based on Ubuntu 8.10 distribution). I can say that 9.04 NBR is even better. I had some problems on Eeebuntu. For example I could't make my touchpad work with gsynaptics even after enabling access to shared driver memory. And now it simply works.
