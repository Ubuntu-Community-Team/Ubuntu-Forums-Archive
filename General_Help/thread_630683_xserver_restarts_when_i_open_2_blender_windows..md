### Post by Stochastic on 2007-12-10
I'm sorry, but if you really want that to happen you may have to program it yourself.  Gimp functions fine as a single instance with multiple files open, I don't see a circumstance where running two gimp instances on the same xserver is of any use.  Frankly it sounds more like a feature that would cause extra issues and headaches for users with unknown gimp instances eating away at their resources in the background and the extra code to implement such silliness seems more along the lines of microsoft bloatware style than ubuntu design.  But maybe I'm wrong.

Furthermore, getting the experimental features and SVN of blender is bound to give you bugs - that's in the definition of development versions.  If you're still having issues with the repo version, please say so, but otherwise you should be asking the developers to fix the bugs in the development/experimental/svn versions via mailing lists, launchpad, and irc.
