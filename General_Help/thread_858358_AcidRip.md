With using Acidrip how do I adjust the settings so that the quality of the resulting ripped file is as good as the dvd from which it was ripped. I just finished ripped Appleseed: Ex Machina - great movie by the way - as an .avi file and resulting file is not that clear. Thanks

### Post by nikgare on 2008-07-13
The only way to make the quality the same is to use the **copy** option in the **General** tab - the others are for lossy compression, eg mp3

If you want to rip to mp3, you need to change the bit rate from 128 in the **Options** box - how high is up to you, but I can't tell the difference between an mp3 at 192 and the original file
