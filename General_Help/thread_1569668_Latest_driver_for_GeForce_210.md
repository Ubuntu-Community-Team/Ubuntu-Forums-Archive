### Post by cascade9 on 2010-09-07
It is possible to install the drivers d/led from nVidia, but its much easier to just use the hardware drivers tool-

system-> administration-> hardware drviers

Using the driver provided that way can save a lot of problems, eg, if you get a kernel upgrade. 

In 99% of cases, its not worth the hassle to install the newest nVidia drivers manually.
