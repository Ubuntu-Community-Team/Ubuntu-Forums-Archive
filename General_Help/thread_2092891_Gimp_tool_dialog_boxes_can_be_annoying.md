### Post by Buntu Bunny on 2012-12-09
Gimp 2.8.2
Xfce 4.8

When I'm working with photos and select a tool, the floating dialog box title bar parks itself in the top left corner underneath my top desktop panel. Consequently I cannot move the box and it's usually in the way. Is there anything I can do about this?

---

### Post by GreatDanton on 2012-12-09
In gimp 2.8 there is feature called single window mode. You can find it under Windows/ Single-Window mode.

Regards.

---

### Post by mcduck on 2012-12-09
Also holding down the Alt key allows you to move a window from any point in the window instead of just the titlebar. So you can use that and move the toolbox a bit downwards...
