### Post by CrackedP0t on 2013-04-28
I'm dual booting Windows 8 and Ubuntu 13.04. I've amassed bookmarks, add-ons, and things on the Windows 8 side, and I'd like to be able to use them on the Ubuntu side. I can't use the Firefox Sync option because it requires me to have access to both devices at the same time, which I can't do because they're on the same hard drive.

---

### Post by JonPaul on 2013-04-28
from the old partition Go to edit -> preferences -> sync tab -> pair a device -> don't have device with me. note down recovery key.

from the new partition go to tools -> set up sync -> i have an account -> put in username, password and key. 

And voila!! sometimes it takes a few minutes to update.
