I'm having a bizarre issue. Whenever I unplug my 10.04 notebook my Dropbox folder opens.  Why? And how do I stop that?

---

### Post by IceRayn on 2011-04-28
Try checking your keyboard shortcuts list, and see if the folder is set to a shortcut that's a special key on the keyboard. There's a chance that your laptop sets an unplug action to a shortcut, which is then activated when the conditions are met. 

Other than that... Check the Dropbox Linux settings. I've never used Dropbox on Linux, but I there might be a shortcut like that in the Dropbox settings too.
