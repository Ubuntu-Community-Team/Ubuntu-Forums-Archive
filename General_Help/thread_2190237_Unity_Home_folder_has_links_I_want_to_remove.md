### Post by grier-devon on 2013-11-26
Hello there,
What version of Ubuntu are you running. On mine when looking at the file manager GUI on the right side kind of at the top there is a little arrow button that point down. If you click on that you will see a option to "Show hidden Files". Now if you do not see that then you are using the older file manager GUI, run your cursor across the top of the screen while you have have the file manager GUI open. I believe with that one to "Show Hidden Files" is in either under "view" or "edit".

I have no problem editing stuff in my home folder, just go into you home folder and right click on the folder you would like to modify.
