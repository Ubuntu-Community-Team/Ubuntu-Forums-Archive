### Post by lolindrath on 2007-09-19
I have the same problem. It's actually a flash window that pops up and covers the whole screen and is supposed to be transparent (probably a flash under linux bug).

I used Adblock to block the flash objects (swf) and then I could see the webpage. You just install Adblock Plus, then click the red stop sign when  you're at the citi cards website. 

Then right-click on any flash objects you see and set them as blocked.

### Post by Vince329 on 2007-09-21
Just found something that works great. This is from windows secrets news letter. Install User Agent Switcher. I now can get virtual account numbers.

To install User Agent Switcher, go to the Firefox add-ons site. Click Install Now. The installer will restart Firefox when finished or prompt you to do so before the changes will take effect.

To add the User Agent Switcher button to your Firefox toolbar, right-click the toolbar and choose Customize. Drag the User Agent icon to where you want it. When selected, it offers a drop-down menu from which you can choose the browser you want to report.
