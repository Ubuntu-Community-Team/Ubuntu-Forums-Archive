### Post by Giblet5 on 2009-11-23
A lot of the problems posted on this forum are "I can't log in."

EASY PREVENTIVE TIP:

Before you tweak your beautiful Ubuntu installation to death (literally), create a second admin account.

Click System -> Administration -> Users and Groups

Create a second administrative account and password that you can use if you can't log in using your normal account.

Don't customize any settings for that account.

If you find that you can't log in to your normal account for some reason, there is always this account to fall back on. You'll have a full administrator-level account AND a graphic interface.

Do it right now. It'll be much harder to do from a Recovery Mode shell or a LiveCD.

Cheers!
