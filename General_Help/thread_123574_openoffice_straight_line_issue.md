### Post by shakin on 2006-01-30
[QUOTE=the_padawan]I forgot to add, if I try to uninstall OO from synaptic or using apt-get remove openoffice.org2, the program tells me that it will also uninstall "kubuntu-desktop". I hesitate to proceed, as that sounds particularly omenous.[/QUOTE]

You can safely remove kubuntu-desktop as it's a meta package that tells Synaptic what other packages are used for Kubuntu's desktop software. Once that software is installed the mete package doesn't do anything. You'll want it again for the next version of Kubuntu.
