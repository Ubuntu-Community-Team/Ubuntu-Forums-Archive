Click on the Ubuntu button on the left Launcher bar. Click More Apps. Any graphical programs you have installed will be listed.

For command-line programs you need to access them from the terminal, and for some they just add functionality to an existing program,  for instance 7zip which allows the default Archive Manager program to deal with 7z archives.
