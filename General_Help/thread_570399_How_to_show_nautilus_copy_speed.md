### Post by Claus7 on 2007-10-08
Hello and welcome!

In order to see how much time a command will take in order to be executed type : time cp file_you_want_to_copy copied_file
That way you can copy your files and see how much time it takes you. 

In order to do that you have to open a terminal first (Applications -> Accessories -> Terminal). 

You can use the command time with almost all the commands. 

I hope this helped,
