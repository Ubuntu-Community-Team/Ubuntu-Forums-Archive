### Post by duke.tim on 2011-10-06
Select Scan -> recursive scan 

in your home directory (say your username is ungui) 

select /home/ungui/.wine

if you can not see the .wine directory hit "ctrl+h" and you should see the hidden directories

and that is what i'd select to scan.
