I am very very used to using the backspace key as a shortcut to the back button in a browser.  Please help.

---

### Post by TVTrukChik on 2009-01-05
Open a new tab and type about:config in the address bar.  In the "filter" box, type "browser.backspace_action".  Right-click on that entry, select "modify", and change the value to 0.
