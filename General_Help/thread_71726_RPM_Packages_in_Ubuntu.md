### Post by lao_V on 2005-10-04
Ubuntu is Debian-based and as such uses deb packages. Have a look in Synaptic to see if there are any packages corresponding to your rpms. If not you will have to convert them to deb yourself. "Alien" will help you achieve this.

---

### Post by darkmatter on 2005-10-04
If you choose to use alien to convert the rpm's, be warned that the results can be unpredictable.

I would recommend trying to find the .deb equivalents for those packages first, and only run alien if all else fails.
