### Post by Impavidus on 2018-10-06
18.04 appears quite solid to me (as every release of Ubuntu, I've used it since 6.06), but for some reason upgrades sometimes go wrong. Some people recommend fresh install over upgrades, although I tried many upgrades and all worked (on a few occasions with minor fixes). Broken upgrades can be hard to fix, so the easy way out is probably a fresh install.

If you want to fix this, have a look at the log files (they are in /var/log/, or use the log viewer) and look for errors around the time a lockup happened. Also try the package manager with```
sudo apt update
sudo apt upgrade
```and show the full output.

A reboot is likely to cause more damage and rarely (but it does happen) the only way out of a lockup. Have you tried SysRq+reisub?
