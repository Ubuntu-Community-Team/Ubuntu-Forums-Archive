### Post by Dreamlocked on 2007-07-01
Greetings.

I am a fairly new Xubuntu user with only limited previous linux experience (openSuse 10.2).
So far, I am intriguingly satisfied with Xubuntu and had few to no problems; a very nice piece of OS.

The only thing that halted me, and I could not find a solution in google or various forums is the inability to start a second X session.
First, I tried this script, which supposedly would speed up framerates in games, such as World of Warcraft (**and many modified version of this script**).

```

 #!/bin/sh

 export WOW_PATH=~/".wine/drive_c/Program Files/World of Warcraft" # Edit this, if You put WoW somewher else.

 sudo X :3 -ac &   # Launches a new X session on display 3 
 cd "${WOW_PATH}"   # Goto WoW dir 
 sleep 2   # Forces the system to have a break for 2 seconds 
 DISPLAY=:3 `which wine` WoW.exe -opengl   # Launches WoW

```

Running this script and its variants gave me a blank screen, with Ctrl+Alt+Fn not working, leaving me no choice but a hard reboot.

Only running :
```
 sudo X :3 -ac &
```
or
```
 sudo xinit -- :2
``` from tty2 gave same results (blank screen, non-functional Ctrl+Alt+Fn).

My question really is : What am I doing wrong? or What have I forgotten to enable?
