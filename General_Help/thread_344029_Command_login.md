### Post by x64Jimbo on 2007-01-22
When you're entering a password in terminal, you have to realize that it won't display anything. Just type the whole password, and then hit enter. It'll work. It doesn't display any characters because even asterisks can give away the number of characters, which makes cracking a password much more simple.
