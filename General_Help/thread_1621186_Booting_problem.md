### Post by Rubi1200 on 2010-11-14
Hi and welcome to the forums :-)

It could be that some file-system damage occurred as a result of what happened.

The boot script results look fairly normal.

I recommend that you check the file-system from the LiveCD:

```
sudo e2fsck -C0 -p -f -v /dev/sda1
```if this reports errors, run this:

```
sudo e2fsck -f -y -v /dev/sda1
```Then reboot, taking the LiveCD out, and see what happens.
