The default install of 12.04 is very light on themes (settings ->appearance ->style.) What's the best/quickest way to install a decent number of alternatives?

---

### Post by Toz on 2014-04-21
There are a number of theme packages in the repositories that you can install:
- gtk-smooth-themes
- murrine-themes
- shimmer-themes
- shiki-colors
...that will provide extra GTK (appearance) themes. 

If you want to have a go at manually installing some themes, have a look at [https://wiki.xfce.org/howto/install_new_themes]("https://wiki.xfce.org/howto/install_new_themes").
