### Post by DuckHook on 2018-02-19
The keyboard layout will not confuse the system so long as the actual characters you are entering are the same. The challenge/response process is not dependent on the keyboard layout. It is tied only to the character code. It is fine to change keyboard layouts and people who must work in different languages do so regularly.

Note, however, that Ubuntu allows multiple keyboards to be registered, so when logging in, you must remember which keyboard layout you last used.

Your problems with your last install were more likely to do with multi-booting than keyboard layout. Ubuntu GRUB may have been replaced by Trisquel GRUB. We also have no idea of how you configured your filesystem layout. Too many factors to juggle. To avoid confusion, try to stay with the same keyboard layout throughout all of your different OSes.
