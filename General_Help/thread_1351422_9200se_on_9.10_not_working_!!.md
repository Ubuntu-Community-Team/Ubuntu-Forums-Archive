### Post by clhsharky on 2009-12-10
HI

Your card is supported with default driver in 9.10. AGP cards don't all way get configured automaticly.

```
man radeon
```
In a terminal will give you more info.
Also run
```
sudo dpkg-reconfigure xserver-xorg
```
then restart X could do the trick.

If that doesnt fix it you might have to edit the
> xorg.conf
file.
Let us know what you need, it should work.
