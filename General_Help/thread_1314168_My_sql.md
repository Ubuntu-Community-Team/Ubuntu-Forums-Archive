### Post by Giblet5 on 2009-11-04
It's "MySQL" or just "mysql". Your English is excellent but using the wrong product name could be confusing.

Once you installed mysql-server, you were already providing database service via port 3306 on the system where you installed it.

If you are new to MySQL, I recommend also installing the mysql-admin package (appears in Applications -> Programming -> MySQL Administrator).

You will want to run that tool and create MySQL accounts for those who will be using it.

There are many excellent books on MySQL. Check 2 or 3 out from the library and read them.

Good luck!
