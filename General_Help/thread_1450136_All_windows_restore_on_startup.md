### Post by rudy.b on 2010-04-08
Hi,

Look under System > Preferences > Startup Applications > Options and ensure that 'Automatically remember running applications ...' is unchecked.

### Post by howefield on 2010-04-09
Sometimes it needs a "clean" shutdown to make it remember your setting.

Close all running applications then go to System > Preferences > Startup Applications, and click on the Options tab, check "Automatically remember running applications when logging out".

Close and reboot.

Do it again, only uncheck the box this time.
