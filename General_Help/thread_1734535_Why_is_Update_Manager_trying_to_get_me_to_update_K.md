### Post by Mark Phelps on 2011-04-20
Ubuntu installs some of the KDE infrastructure by default, so you can have the option of choosing the KDE desktop if you want.

However, if you want to see the HUGE list of additional packages that would be installed if you switched over to using KDE, open up the package manager, select kde-desktop -- and look at the long list.
