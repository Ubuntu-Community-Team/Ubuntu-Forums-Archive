### Post by snowpine on 2011-01-23
Netbook Edition is not designed to be faster, but rather to have a different user interface designed for smaller netbook screens. Some people find it is a little faster; some find it is a little slower--it really depends on your specific hardware (tell us about yours and maybe we can make a recommendation).

Running from Live CD/USB is slow because the filesystem is compressed and must be extracted to RAM as you use each application/feature. To get the best performance, you should install Ubuntu to your hard drive.

### Post by snowpine on 2011-01-23
> **Hatryst said:**
> So, any tips to speed up my LiveUSB?

Yes, install to your hard drive. :)

If that is not an option for whatever reason, and you need to run from USB on a permanent basis, then the best advice I can give is:

1. Buy the best quality, fastest thumb drive you can afford.
2. Consider using a distro such as Puppy Linux that is specifically designed to run as a Live CD/USB and that loads itself into RAM.
