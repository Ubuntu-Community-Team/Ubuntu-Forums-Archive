### Post by oldfred on 2014-04-22
I would reinstall grub to sda with command line or Boot-Repair. But if using Boot-Repair do not use auto fix as that installs grub to every MBR. Use advanced mode and choose Ubuntu and sda. Also then choose Windows and sdb, so you could boot Windows directly from BIOS if you have grub issues.

       Boot Repair -Also handles LVM, GPT, separate /boot and UEFI dual boot.:
[https://help.ubuntu.com/community/Boot-Repair](https://help.ubuntu.com/community/Boot-Repair)
You can repair many boot issues with this or 'Create BootInfo' report (Other Options) & post the link it creates, so we can see your exact configuration and diagnose advanced problems.


 How to restore the Ubuntu/XP/Vista/7 bootloader
