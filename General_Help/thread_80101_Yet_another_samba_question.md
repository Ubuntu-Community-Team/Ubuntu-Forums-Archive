### Post by PaulBillett on 2005-10-21
[QUOTE=k_smolka]
i have been trying to install samba to let me access my windows network which i have done before in red hat.[/QUOTE]

Firstly, you need to install nothing to access a windows share... only for windows pc's to access shares on Ubuntu.

To access windows shares just open the file browser press ctrl L and enter the path to the share. ie smb://computername/foldername

Lastly, To actually share files you will need to add an ubuntu user account for every share username.  You can do that under /System/Administration/Users and Groups/
