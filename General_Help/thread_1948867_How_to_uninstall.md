### Post by Mark Phelps on 2012-03-29
> **Spewed said:**
> I'd like to be able to uninstall it and be able to log back into Windows without any issues whatsoever. Is this even possible?

Actually -- NO -- not without "any issues whatsoever".

Presuming you actually installed GRUB to your MBR, and Ubuntu to its own separate partition, when you remove Ubuntu, your PC won't boot anymore -- because it's looking for the rest of the GRUB code in the Ubuntu partition, which is now GONE.

BEFORE you do that, you need to do the following:
1) Use the Backup feature in Win7 to create and burn a Win7 Repair CD
2) Boot from the CD and run Startup Repair.  See if you now boot directly into Win7.
3) If not, reboot from the CD and run Startup Repair again.  You may have to run it a total of three times.

But eventually, it will rewrite the boot code so you no longer use GRUB at all.

When you can boot directly into Win7, use the Win7 Disk Management utility to remove the Ubuntu partition.
