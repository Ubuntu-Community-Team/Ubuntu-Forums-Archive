### Post by clintonthegeek on 2008-11-23
It sounds like an easy question, but it's very open-ended. If you've been using Ubuntu for a year than you probably already sort-of understand stuff like file permissions, how the file system is laid out, a little command line stuff, right? Skim though [this](http://linuxreviews.org/beginner/) and see if you are already familiar with that basic stuff.

After that, it's just a matter of learning individual stuff, depending on what you need to do. The internet has much more up to date and relevant info than any book you can buy, you just need to Google for what it is you want to know!

Want to learn how to do advanced command line stuff? The default command line program is called 'bash', so just Google for *bash tips* or *bash programming*. Learn how to access your computer remotely? Google *how to use ssh*, and read a few articles (remember they might have conflicting steps or info -- pay attention to what sounds safest and most right!)

If you *actually* want to learn the guts of linux, then building and configuring a [Gentoo](http://www.gentoo.org) installation is the  most effective way to learn all about that, because it forces you to do everything yourself! Try in a virtual machine or something, because it can take days to properly configure and build linux from scratch!

Books that promise to teach you linux probably will. But they'll also get old, and might not be relevant to anything you are interested in doing, or not specific enough to your distro. Learning through doing is probably a lot easier!
