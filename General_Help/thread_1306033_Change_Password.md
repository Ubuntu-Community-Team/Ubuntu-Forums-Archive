### Post by SuperSonic4 on 2009-10-30
As user (if you can log in on your current password):

```
passwd
```

### Post by P4man on 2009-10-30

"change password"

Alternatively (and if you want a very insecure short pw) run 
```
sudo passwd yourusername
```
