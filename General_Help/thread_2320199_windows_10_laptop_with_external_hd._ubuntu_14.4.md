### Post by DogMatix on 2016-04-11
Windows doesn't natively read ext3, ext4 file systems. You could try using [Ext2Read](https://sourceforge.net/projects/ext2read/), [Ext2fsd](http://www.ext2fsd.com/) or a similar third party tool.
