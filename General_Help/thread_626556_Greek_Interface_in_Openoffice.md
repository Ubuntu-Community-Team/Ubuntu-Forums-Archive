### Post by dazlari on 2008-03-23
This happened to me too! I wasn't sure if it was the recent update or playing about with options that had brought on this curse, but for weeks I struggled with this. I checked for Greek character sets such as openoffice.org-l10n-el which I didn't have loaded, so removing them was not the problem. I searched the OO help and that pointed out (in English) a checkbox in the Tools | Options | View menu (the only checkbox in the first section) entitled "Use system font for user interface". This was checked and unchecking it resolved the problem.

### Post by simosx on 2008-03-24
It's not "Greek" but what happened was the system got borked up and selected a "Symbol" font to show the string of the user interface.

The Symbol font is a legacy one; it replaces the latin letters with the Greek semi-equivalent, and it appears as if it is written in Greek. Some elemental Greek alphabet knowledge and you will be able to see that the language of the messages is actually English, German or whatever you have configured.
