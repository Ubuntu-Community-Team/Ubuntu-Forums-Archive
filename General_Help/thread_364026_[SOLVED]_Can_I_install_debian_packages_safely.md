### Post by CocoAUS on 2007-02-17
Most programs general users will use should work using standard .deb packages.  The Ubuntu kernel is customized (as most major distros' are), but that's mostly a hardware thing.  I've only found one or two Debian .deb's that didn't work, and they were for very, very obscure programs.  So obscure, I can't even remember what they were.

So, while it's best to use Ubuntu-specific .deb's, I wouldn't expect any trouble using Debian .deb's.
