### Post by mcduck on 2010-01-24
Application font is what all the user interfaces (menus, toolbars, buttons etc) in your programs use.

Document font is used for document view only, for example if you open a text file in Gedit, the user interface will use Application font while the document itself will be displayed using the Document font (of course assuming you haven't set any other font directly in Gedit's settings).

Window title font only changes the font in window titlebars.

Fixed width = monospace, the fonts used in terminals and other situations where a monospace font is required. (monospace font is a font where all letters use same amount of space).

None of these settings will have any effect on any web pages you are viewing. For that you'll need to change the font settings in your web browser.
