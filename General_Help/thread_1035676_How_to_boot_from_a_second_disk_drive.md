### Post by mahela007 on 2009-01-10
I have two disk drives . One is just a plain CD drive and the other one is a DVD-R drive. I have set the boot priority of my PC to CD but it just doesnt boot from the DVD drive. I can only boot from the cd drive. What can I do to be able to boot from the DVD drive?

---

### Post by LoneWolfJack on 2009-01-10
If you have a cheap mainboard, it's quite possible that you can only set the BIOS to "boot from CD", but without being able to specify which CD drive (you appear to be indicating exactly that). Your BIOs will then pick the first CD drive it finds to boot from.

Your solution would be to switch the bus connectors on the drives so your mainboard will see the DVD drive first. But obviously, you won't be able to boot from the CD drive then...

### Post by mikewhatever on 2009-01-10
You have to set it to also boot from the DVD. There should be options to boot from both cd/DVD-roms, so if one of them is not set, the computer wouldn't try booting from it.

### Post by todak on 2009-01-10
Unless your BIOS specifically has an option to boot from CD/DVD drive #2, you must physically swap the drives so that the DVD drive is first. That is what I had to do on my system. Actually, the second drive exists to plug the hole in the case, since I hardly use it, except for the rare occasion that I need to duplicate a CD! My main drive is a CD/DVD dual-layer RW and the second drive is just a CDRW.
