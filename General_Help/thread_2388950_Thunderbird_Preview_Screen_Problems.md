### Post by walts48 on 2018-04-10
What Thunderbird forum? There are many.

If you want to resize the header try using the Compact Header extension.

[https://addons.mozilla.org/en-US/thunderbird/addon/compactheader/](https://addons.mozilla.org/en-US/thunderbird/addon/compactheader/)

If you are using the Classic layout, the Folder Pane is on the left, with the Thread Pane above the Message Pane, to the right of the Folder Pane. If you can resize the Thread Pane (upper window), the bottom window (Message Pane) resizes with it.
