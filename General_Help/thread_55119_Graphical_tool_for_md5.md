### Post by dave9191 on 2005-08-07
Sorry, but I don't know any gui tools for md5. But it sounds like to me that you are typing in every single file name. You know that you can use regular expressions in the command line. So if you want a md5sum for all the files in the current dir you can 

md5sum *

or every zip file

md5sum *.zip

Much nicer then click on each file and selecting a md5 option assuming you can find a gui :) Maybe there is a plugin for konqueror or the gnome file browser.
