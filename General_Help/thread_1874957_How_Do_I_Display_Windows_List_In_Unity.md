### Post by Jacobonbuntu on 2011-11-20

I think you confuse gnome with Unity; Unity is not part of Gnome. What you probably did was disable the Unity plugin in the ccsm.
maybe what you would like is to keep the unity launcher visible (you can do this in the compiz manager > unity plugin > behaviour (tab) > "never hide"
you can see than the icon of the running apps, with an arrow for every window. click on it and see the windows exposed.
