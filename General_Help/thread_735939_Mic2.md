### Post by SGAZ on 2008-04-02
If you're using the **Gnome Alsa Mixer** you can select Sound Card Properties in the Edit menu and make sure all of the check boxes are checked.  That will make all of your sound card elements visible in the Mixer for selection/input/muting etc.

Gnome Alsa Mixer is available in the regular add/remove programs under Applications if you're not already using it.
