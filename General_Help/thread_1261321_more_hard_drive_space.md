### Post by rob-ward on 2009-09-08
You can resize the windows partition using an application like gparted, however you do risk dataloss. I would recommend that before you try to resize the windows partition you back everything up.

Once the windows partition has being resized the linux partition can be resized in the same way, however again you can risk something going wrong. Backup everything that you need just in case.
