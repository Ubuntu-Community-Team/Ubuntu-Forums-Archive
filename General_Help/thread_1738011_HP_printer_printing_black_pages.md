### Post by rcytopdog on 2011-04-24
Thought I'd post this as I couldn't find much about it online.  I have an HP3745 and am running Ubuntu 10.04.  For some reason I couldn't get anything from the black ink cartridge in test mode or for real. The printer would go through the motions but would return a blank page. Turns out it was because my COLOUR ink cartridge was empty!!  Solution was simply to select print in grayscale.

Hope this helps someone. :)

Also I find with this particular printer HPCUPS works better than HPLIP.
