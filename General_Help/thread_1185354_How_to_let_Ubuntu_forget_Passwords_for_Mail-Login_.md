Also I managed to connect to a GMX mail account (it is important to use the customer number as the name in this). However, when logging on, I set the checkmark to remember the password for the mail logon. Thunderbird worked fine after this.

Then I installed another mail account for another member of our family (who logs in under another user account at Ubuntu).

Later I changed the password of my first gmx account, and now I can no longer download the new mails in it with thunderbird.

Checking it with the webmail interface, I saw, that all my attempts resulted in "wrong passwords" - probably because thunderbird is still using the old password which it has memorized. There is no feedback, that the password was rejeted by the mail account.

But how can I make ubuntu forget the password, which it has memorized for thunderbird? 

I already tried to consult the Help of Thunderbird, but unfortunately this only connects me to the internet page of mozilla [www.mozilla.com/en-US](www.mozilla.com/en-US) which is not the help for Thunderbird.
