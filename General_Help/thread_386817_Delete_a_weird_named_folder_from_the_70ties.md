### Post by bernied on 2007-03-17
Can you get inside the lowest level folder (NOT the / directory!! - be very careful) and do:
```
sudo rm -r ./*
```
or from / folder:
```
sudo rm -r ./,hJ6*
```

Other idea - run your filesystem browser (Nautilus?) as root, see if it can handle it:
```
gksudo nautilus
```or is it gksu (sorry I use kde)

Do any of these get you anywhere?
