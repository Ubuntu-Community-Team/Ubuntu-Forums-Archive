### Post by WorMzy on 2011-10-12
Most drivers are built into the kernel itself, or are built as modules which can be added into the kernel when necessary.

To make sure you have the most up-to-date drivers, install the latest kernel available to you from the software centre.

If you want to live on the bleeding edge, download and build your own kernel from the git source: [https://github.com/torvalds/linux/](https://github.com/torvalds/linux/)
