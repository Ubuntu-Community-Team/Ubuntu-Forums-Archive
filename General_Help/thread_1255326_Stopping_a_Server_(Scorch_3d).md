### Post by NertSkull on 2009-09-01
So I'm trying to get more involved with ubuntu.  I tried playing a game of scorched earth 3d online.  How do I stop the game server (when I host a game) after I am done or want to change the settings.  

In windows I would just close the server, and it would go away.  But in ubuntu if I do that, and try to start a new game, it says the server is already running.  Any ideas on how to stop that?  Thanks everyone

### Post by buttertoad on 2009-09-04
The easiest way I've found is to open a terminal and type:

```
killall scorched3ds
```

The "s" at the end tells you it's the server.

Hope that helps.
