### Post by lisati on 2011-01-11
I agree that it sounds like you might have done what's known as a "wubi" installation.

You might want to have a look through the forum for threads about shifting from a "Wubi" installation to see if that sounds like the situation you're in.

**Important:** Regardless of how you installed Ubuntu, it's a good idea to make backups of your data before tinkering with or removing Windows. This includes making sure you have a set of recovery disks available for your Windows installation just in case you need to get it back for some reason. If you're in luck, you'll already have a set or have the software preinstalled on Windows to help you with making a set.

Here is a guide on how to migrate a Wubi install to the drive:

Unless you want to consider a dual-boot setup, then the easiest thing to do would be to go for a clean install of Ubuntu onto the drive (which will completely remove Windows of course).
