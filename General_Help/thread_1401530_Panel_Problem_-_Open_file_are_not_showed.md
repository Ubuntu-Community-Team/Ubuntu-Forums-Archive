### Post by rocket16 on 2010-02-08
Hmm, seems like you may have accidentally removed the Windows List from your panel. Right click the desired panel (where the list of open windows will be shown). Now, select "Add to Panel", and select "Window List" from the list of widgets below and click "Add". Hope this will help you. Thanks.
