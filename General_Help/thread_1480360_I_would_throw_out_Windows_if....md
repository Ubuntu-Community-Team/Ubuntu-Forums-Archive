### Post by Calash on 2010-05-11
For point #1, do a search for Active Directory in the Software Center.  It works fairly well.  Some oddities during the Lucid upgrade that relocated my home folders but some creative linking and adjusting permissions fixed that up fine.

One thing I have yet to find is limiting local permissions based on Active directory groups.  I can restrict access to the entire system based on groups but once they are in they get permissions based on how you configure the files regardless of what AD groups they are members of.  By default these are User only permissions.

You can set them per user so it still can be managed.


Point 2 and 3 are not something I see on my end.  My system is stable and graphics perform very well.  I have dual monitors setup as well with my screen stretched and it works well.


I can not comment on the other points as they are not issues I have seen myself.
