Hey, I apologise if this is the wrong forum, but there's a potential program I've been thinking of.

Recently, I set up a bluetooth connection to back up files (text messages, contacts, etc) on my mobile phone with the Nokia PC suite on Windows, more recently my Mum asked me to do a similar thing with a USB cable and her Blackberry device.

It suddenly occurred to me that this is a common operation, and I wasn't sure if there was a program in Linux that did something like the Nokia PC Suite. You know, only it would synch up with any mobile device and possibly synchronise stuff like calendars and contacts between mutliple devices.

I checked and Ubuntu can read files over Bluetooth from my Nokia Phone no problem, but it's a file operation rather then a standalone program.
I did some searching and I found some frameworks and tools like OpenSync and Barry, but couldn't quite find the fully featured GUI program like Nokia PC Suite.

So, am I overlooking something?

### Post by Irihapeti on 2010-05-26
Yes, there is something.

See [here]("http://antoine.ginies.free.fr/syncnokiae70/syncnokia.html") and [here]("http://antoine.ginies.free.fr/syncnokiae70/syncnokia.html") for details.

Unfortunately, the syncml plugin has been removed from Lucid. See [here]("http://ubuntuforums.org/showthread.php?t=1474031") for instructions on how to install it from Karmic repos.

If you want a gui, you can install multisync0.90

Edit: and there's also [obextools]("http://unleasheddreams.wordpress.com/2008/09/12/connect-nokia-to-linux-with-obex-tools/") that does some of the job that PC Suite does, I gather. I've never actually used PC Suite, so I'm not sure.
