Ah, ok, first thing - typically you don't have to download them to install them - you just click on "Install" when you're at the theme page...

But if you do have them locally downloaded, you need a special extension to install them:

Click on the "[Install Now]("https://addons.update.mozilla.org/extensions/moreinfo.php?id=421&vid=1383")" link.

That extension will allow you to go File->Install Theme, and then browse to your downloaded themes to install them.  :)
