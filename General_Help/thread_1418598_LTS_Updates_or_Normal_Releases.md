### Post by mikewhatever on 2010-02-28
If you want to upgrade once in two years, choose LTS. I'd say, tt's more suitable for organizations then home users.
On the other hand, if you want more frequent upgrades and newer software, follow regular releases. Please note, LTS has no advantage over regular releases other then longer support.

### Post by dcstar on 2010-02-28
> **mcse37 said:**
> I'm planning on installing the next LTS release, but I'm unsure about updates.  Under System - Administration - Software Sources - Updates tab, I have a choice between LTS updates or normal releases.  Which would I choose & why?


If you want a system that will work over a long period of time choose the LTS option.

If you want to keep having the latest "gee-wizz" version offered every six months (and all the problems that come with every single new version), choose the other.

It all really depends if you either want a stable platform that will work in a predictable manner for all the things you want to do, or you are willing to put up with all the hassles of things that don't quite work correctly until months after each new release.
