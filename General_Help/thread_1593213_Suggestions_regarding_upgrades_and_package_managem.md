### Post by KingNeil on 2010-10-11
1. Do not include "A new distribution is available" button on the upgrade manager. There are so many reports of upgrades messing things up, so this definitely should not be advertised to the average user.

2. When you first install Ubuntu, the list of packages has not been downloaded/updated/indexed. Thus, if someone was to try and install something via command line, it wouldn't work. And when someone goes into synaptic, they have to wait quite a few minutes for the packages to be shown. Thus, the list of packages should be downloaded/updated/indexed in advance.

I submitted these ideas to Ubuntu brainstorm. So if you like them, upvote them on there, once the moderators have approved them.

1. [http://brainstorm.ubuntu.com/idea/26062/](http://brainstorm.ubuntu.com/idea/26062/)
