### Post by jken146 on 2007-12-27
File names don't have extensions in Linux (although often they are included to make the names more informative).  The command ```
file blah_blah.blah
``` will make an attempt at determining the file type, based on its contents.  That said, it probably is some kind of package (sorry I couldn;t be more helpful).

### Post by FuturePilot on 2007-12-27
Make sure it's executable first.
```

```
Then try running it
```
/path/to/file.package
```
