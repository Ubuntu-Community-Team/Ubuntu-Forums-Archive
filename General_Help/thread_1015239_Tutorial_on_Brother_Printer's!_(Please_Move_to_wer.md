### Post by macintosh on 2008-12-18
If you have one of these?
[http://www.brother.com.au/multi-function-centres/images/models/DCP-130C/Brother_DCP-130C.jpg](http://www.brother.com.au/multi-function-centres/images/models/DCP-130C/Brother_DCP-130C.jpg)
you might want to know this! On the GNOME version on Ubuntu that's the one with the Menu Bar at the top and a Taskbar at the bottom. Like this
[IMG]http://img227.imageshack.us/img227/5479/picture1ui4.png[/IMG]
you may want to know how to get the whole thing working without any problems! Well hear is how you do it go to System>Administration>Synaptic Package Manager. Then go to "Search" and type Brother then look down the Brother CPUS wrapper packages install one then quit synaptic.
[IMG]http://img222.imageshack.us/img222/5614/picture2cw2.png[/IMG]
Then go to System>Administration>Printers and delete any related printers you might have then click add new printer and you will see that your printer name is in the list then you can print successfully without no problem.
