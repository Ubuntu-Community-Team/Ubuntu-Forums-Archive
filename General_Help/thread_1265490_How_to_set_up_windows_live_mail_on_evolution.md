### Post by nerdking on 2009-09-13
After a few hours i managed to get windows live mail setup to use evolution perfectly, im posting this here for anyone else wondering how to do this. (sorry if im posting this in the wrong section.)


when seting up a account for live mail, use this as the receiving mail: pop3.live.com set authentication as ssl and use your email as the username.

for sending use: smtp.live.com for authentication use TLS(ssl will not work for some reason)
and use your email for username once again. now you should be able to send and receive windows live emails with no problem.


any one reading this feel free to put these instructions into simpler form.
