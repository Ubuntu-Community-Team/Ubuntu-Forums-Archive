### Post by DanneStrat on 2011-04-21
> **storm1990 said:**
> began installing ubuntu with wubi, how much should the installation size be for a 251gig harddrive

Hi.

That depends on what you will use the wubi installation for.

I would say that 30 GB would be enough in most cases for

a wubi install. If you plan on storing personal files you could 

make the size 50 GB. Ubuntu will need about 4 GB of disk space 

for system files then you will need additional space to install 

new applications and even more if you plan to store large files.

### Post by Rubi1200 on 2011-04-21
Wubi has a maximum size of 30GB for the initial install of the root.disk; which includes all system files and your home folder.

If you want extra space, take a look here at these links about how to resize/create another virtual disk:
[http://ubuntuforums.org/showthread.php?t=1625371](http://ubuntuforums.org/showthread.php?t=1625371)
