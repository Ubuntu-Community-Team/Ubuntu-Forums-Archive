### Post by coffeecat on 2010-04-10
For an application installed in wine, go to the wine menu and search for the uninstaller for the particular application there. It may be under the entry for the application itself, but there's also an uninstall choice in the wine menu. You need to uninstall wine apps from within wine because wine uses a Windows-type registry.

For a Linux application installed with Synaptic or Software Centre, you can uninstall with Synaptic.
