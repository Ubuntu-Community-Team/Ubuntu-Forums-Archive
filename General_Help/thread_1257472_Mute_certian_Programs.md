### Post by CatKiller on 2009-09-04
> **fluffman86 said:**
>  I don't think Ubuntu has a single place to do that.

It does if you're using Pulse Audio. Install *pavucontrol* in Synaptic (it's in the Universe repository). It will put a Pulse Audio Volume Control entry in the Sound & Video menu that will let you set the per-application volume.
