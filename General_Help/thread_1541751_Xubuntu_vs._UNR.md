### Post by snowpine on 2010-07-29
Xubuntu is "lighter" (has lower system requirements) than Netbook, see here:

[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

Personally, I would stick with Xubuntu. If you are curious about the netbook launcher, you can install it from the software center; you don't have to reinstall. It is just a different interface; the core system is the same.

### Post by 3Miro on 2010-12-21
UNR takes the same resources as ubuntu-desktop, it just has a different interface, which is optimized for space and people either like it or they don't;

Xubuntu uses different desktop environment that is overall lighter.

You can install xfce4 or xubuntu-desktop (the second one pulls more things) and you can select between xfce and UNR on login. XFCE and Gnome play nice with each other, since they are based on the same GTK technology.
