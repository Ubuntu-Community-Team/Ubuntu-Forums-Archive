### Post by Topsiho on 2011-02-01
Hello,

You say that you have saved the document ("multiple times"), so it must be somewhere on your hard disk. You saved in your documents, so that's the place where to look for it.

If you can not find it there you could try and locate it, probably there is a GUI way to do it, but in the terminal you could type more easily:

locate <name of the document, or just part of it>, and maybe you so can find it's location.

If not, to be sure, you could update the database of locations of files on the hard disk using:

sudo updatedb

and then enter your password when asked.

If you did not yet give a name to the document, then it is named something like "Untitled.odt" or other, depending on the language you use on your computer. Maybe .doc in stead of .odt, if you saved it as a .doc file, if your school still insists on MS documents.

Hope this helps,

Topsiho

### Post by dno on 2011-02-01
Just go into OpenOffice and go File -> Recent Documents.  It should be right there.  If by chance your system crashed while in OpenOffice, it is smart enough to ask you if you want to recover a file from the crash the next time you use Openoffice.  Doesn't sound like the case here, but just to let you know.
