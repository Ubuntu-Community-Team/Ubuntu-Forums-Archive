### Post by KenSentMe on 2006-12-27
when you run 'ps aux' in terminal you can get a 4/5 digit numberof the process thunderbird is running on. If you then type 'kill number' e.g. kill 5764, then that process is terminated.

Make sure that you kill the right process, like this one:

username    4950  0.0  2.0 141996 42476 ?        Sl   16:26   0:14 /usr/lib/mozilla-thunderbird/mozilla-thunderbird-bin

4950 is the proces number (pid, proces id)
