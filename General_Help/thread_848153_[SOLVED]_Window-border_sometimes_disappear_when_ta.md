### Post by sayakb on 2008-07-03
Referring to the picture you enclosed, it seems like the theme you use has high transparency for inactive windows. Are you using emerald? You may like to use some different theme if the high transparency is bothering you. If it is an emerald theme, you can easily change the amount of transparency for inactive windows (Select theme in emerald and click **Edit themes** tab)
