### Post by deadflowr on 2015-07-10
I take it this about Ubuntu's default terminal.
Right?
If so, then try a mouse right click go down to the Profiles item. Then go to Profile preferences.
In General setting tab uncheck "use system fixed width fonts"
Now you can set whatever font you want and with whatever size you want.
(Setting the font size is set in the font selection page.
There are plus and minus signs in the bottom corner to adjust the size, they should be above the select button)

Again, though, if you are using a different terminal than Ubuntu's default ignore the above.
