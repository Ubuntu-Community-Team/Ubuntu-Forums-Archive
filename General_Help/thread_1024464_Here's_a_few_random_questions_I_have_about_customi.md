### Post by BeachBum on 2009-01-25
> **Rukaru said:**
> 
1) I'd like to be able to use text instead of icons for folder menus and launchers on the panel

If you go to the appearance prefs-> Interface -> Un check Show Icons in Menus, that should do it.  Also on the same page, you can change toolbar labels to be icons or just text using the drop down.  
> **Rukaru said:**
> 
3) I'd like emerald to be use shadows correctly. Right now, I have it set on a very low radius (you can barely see it now). if it's too high, there is a very visible step in the shadow that looks very glitchy. Is there a way to fix this?

I believe if you install emerald-theme-manager you can edit the shadow radius/opacity, etc

Hope that helps :D
