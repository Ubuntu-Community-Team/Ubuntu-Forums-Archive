### Post by Gaiwecoor on 2007-04-22
I'm running Kubuntu 7.04 on a tablet with full functionality now (after a little tweaking), so yes, you can make it work.  If I can get it to run (and I'm fairly new to the whole linux scene), there's a simple way to do it.

The only problem is the limited tablet software base.  Aside from Inkscape and the GIMP, there isn't very much you can do with it.  Currently, Windows XP Tablet has the superior functionality for text recognition and the like.  If you want to use the tablet *as a tablet*, I suggest sticking with MS.

### Post by buster8079 on 2007-11-28
I got Ubuntu working quite well on the [Thinkpad X61 Tablet]("http://www.thinkwiki.org/wiki/Category:X61_Tablet"). [ThinkWiki]("http://www.thinkwiki.org/wiki/ThinkWiki") was helpful with tweaking and customizing as were other [wikis]("https://wiki.ubuntu.com/ThinkFinger") and [forums]("http://ubuntuforums.org/showthread.php?t=604896").

With respect to handwriting recognition, Linux is *damn* close to MS with the help of an excellent program: [**CellWriter**]("http://risujin.org/cellwriter/"). After your characters are inked into this program, recognition mistakes are really only the fault of poorly inked samples. If you prefer, you can customize (through "options" then the "recognition" tab) the number of "samples per character."
