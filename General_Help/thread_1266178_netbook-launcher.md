### Post by NightwishFan on 2009-09-14
I used that on my desktop for a while, as I really do not care for a blank Gnome desktop with desktop icons. And KDE is a bit too much needless (for me) eye candy.

The netbook launcher is a bit cpu intensive for me. I like how it says 'Loading' when you click on an application though.

Edit: If you are on Jaunty, the netbook launcher is in the repos already. Just run:

sudo apt-get install netbook-launcher
