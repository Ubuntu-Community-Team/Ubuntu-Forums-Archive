### Post by deeZee on 2007-05-29
If you want to join  the ubuntu irc channel, go to Applications > Add/Remove... select Internet in the left column and look to your right for the  "XChat-GNOME IRC Chat" program, and install it.  The first time you run the program it will prompt you for a  nickname  (you don't have to have an account).  It will automatically connect you to the ubuntu channel.  I hope I understood/answered your question;)
