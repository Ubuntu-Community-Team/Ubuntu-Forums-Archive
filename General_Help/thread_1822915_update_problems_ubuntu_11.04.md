### Post by coffeecat on 2011-08-11
It's probably the hr.archive server if you can connect to the internet. It's not uncommon for a mirror to go down for a time - perhaps for maintenance. If you need to install something in a hurry simply change the server in Update Manager > Settings > Ubuntu Software tab. Otherwise wait for 12-24 hours and your server should be up again.

**EDIT**: [noparse]I just tried to connect to hr.archive.ubuntu.com and it timed out. Yes, it's a server problem.[/noparse]
