### Post by moto_tux on 2010-10-14
Old post I know, but might well help someone...

It looks like KolourPaint doesn't set the resolution for the image, it's specified as "unspecified" by default.  To change it select "File > Properties" then adjust the x and y value at the top.  96 worked for me, I'm not sure if it's exactly correct (or even matters that much!)
