I don't think you can have HTML on the desktop like you can in Windows. Its a shame, as that is one thing I use daily at work. There are a few options open to you;

1. You can use Evolution to sync with your Google Calender. You will be able to see your events in the Clock in Google, but it means clicking on the clock. This would be the simplest method.

2. You can install and set up conky, and then use it to display your calendar. You will have to download your calendar and script it showing the events you want to see (today's, this week's, this month's). Not trivial.

3. You can install and configure remind to issue reminders through zenity or lib-notify. There is a script which converts ical files to remind files. Again, not trivial.

4. Make a script to take a screen grab of your google calendar every day and set that as your desktop background. Not ideal, but it would work at a push.

Good luck!
