### Post by CurlyChris on 2006-04-18
Hi Davydd

Ok, I'm using Firefox 1.0.7 and have MPlayer installed, with the Mozilla-MPlayer plugin.

Now, when I go to the BBC radio site and click on the link marked "[speaker icon] Listen to Radio 1" (for example) I get the following error message:

"Could not find an appropriate hxplay or realplay in the system path to use as an embedded player"

BUT....

... when I click on the OK button (the error message comes up once more - just click OK again) I then get to the BBC Radio Player window.  Then all I have to do is click on  the link marked "Listen using stand-alone Real Player".

Then the MPlayer plugin loads, buffers and then gives me streaming radio.

Using the example above, this is the link for Radio 1; this should work as described above, unless there's some problem with Firefox 1.5 and the MPlayer plug-in:

 [http://www.bbc.co.uk/radio1/realaudio/media/r1live.ram](http://www.bbc.co.uk/radio1/realaudio/media/r1live.ram)

Hope this works for you.
