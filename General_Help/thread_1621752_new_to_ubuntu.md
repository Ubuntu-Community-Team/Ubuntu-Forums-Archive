### Post by TBABill on 2010-11-14
Yes all of them should work fine. There are specific drivers for your printer as well. When you said "my apps", as long as there are Linux versions you should be all set. If a program you use is for Windows only then you have some research to do to find out if you are able to run it in Ubuntu (or any distro) via Wine, Playonlinux, etc. 

I'd just dual boot with Windows until you are 100% sure you are ready to dump it. And there is absolutely nothing wrong with keeping it on a separate partition for any instance you may need it for. I keep Win 7 on mine but just don't ever boot into it unless I need a Windows specific app, which is virtually never. But since I paid for it with the machine I want to keep it.

Do what's best for you, but make sure you can restore Windows just in case you ever need it (if you decide to go full Ubuntu with no Win partition). Post back with threads for your different issues and you'll get tons of help. And welcome to the forums!! :)

### Post by lumore22 on 2010-11-17
Greetings-

I currently have a dual boot system. I'm unhappy about having to have Windows alongside my Linux, but once in a while I need to use Windows, so I keep it.  But I almost never use it. I much prefer Linux. 

Ubuntu/Linux users are an enthusiastic group so they will almost always recommend UbuntuLinux. You mentioned that you want all your apps to work. Which apps? 

_MS Office  -- > Open Office on Linux_

1. First, make a complete backup of all the Documents that you need to have access to. And do this preferably onto an external hard disk. Then you can open any MS Office Docs on the external hard disk through Open Office under Linux/Ubuntu. 

2. Do Step 1. Another way to do this:   If you are using MS Office, first download an open source app called Open Office (now may be called Libre Office) onto your Windows system. Convert all your docs from MS Office format (.doc, .dot) to Open Office Format (.odt for word processing. Other formats for the database, spreadsheet, presentation apps).  Basically, you open the open Office app, then you open the MS Office Document, then you save it under Open Office. There's probably a script you can get to do this or do it in the console on Windows. Then save these to the external hard disk, and you'll be able to open them under Ubuntu/Linux through the Open Office app running under Ubuntu/Linux. 

_Printer Drivers_
3. Printer Drivers - Your printer probably came with an installation CD. That CD has drivers for WIndows systems. Check if it also has drivers for Linux. You  may have to download drivers for Linux/Ubuntu. 

_Email/Skype_
4. If you use your PC for email and Skype, check the Skype web pages for info on how to set up Skype on Linux. (I had Skype on my Windows partition and had to remove it because it interfered with my web server. ) But setting it up on Windows 7  was not trivial.   So again check the Skype web pages.

_Networking_

5. Depending on which network interface card and which drivers you have, this may be a bit tricky. Check your PC network hardware to see what you currently have and which drivers are available for this hardware. For example, Broadcomm, Intel, etc. 

_Configuration_

6. If you have problems with drivers, you may have to spend some time manually reinstalling them, and reconfiguring. This can be a challenge. 

_Documentation_

6. If you migrate to Linux, be prepared to spend lots of time reading documentation. If you're not a reader, this can be a challenge. But, there is no shortage of users willing to help you through the process. It's a perpetual learning experience.  You'll just be on the forums all the time. Patience  is a big requirement. 

_Free Applications_

7. There are tons of free apps available for Linux/Ubuntu. There are state of the art database apps, statistical apps and other fun stuff. The nice thing about using Unix or Linux, is that all these apps are very compact and use very little disk space by comparison to Windows-based apps. 

Regards and Lots of Luck-:KS
