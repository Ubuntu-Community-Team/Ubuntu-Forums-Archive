### Post by fragos on 2008-12-18
Right click the bluetooth icon in the applet bar and select Preferences. The 1st tab should be your host name. There is a text box for known devices, meaning previously paired devices. It's best if any device you wish to connect with be in this list. Click the "+" button and the process to pair is started. You need to follow the requirements for pairing on the device you wish to pair with.

Since 8.10 the send to associated with the right click of a file icon no longer works. You can send file via bluetooth by right clicking the bluetooth icon and selecting send files to device.
