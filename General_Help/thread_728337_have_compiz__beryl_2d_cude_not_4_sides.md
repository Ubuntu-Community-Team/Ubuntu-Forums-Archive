### Post by BluntBox on 2008-03-19

### Post by Therion on 2008-03-19
You have three settings:  

Horizontal Virtual Size, Vertical Virtual Size and Number of Desktops.

**Horizontal Virtual Size** should be set to **4**.
**Vertical Virtual Size** should be set to **1**.
**Number of Desktops** should be set to **1**.
