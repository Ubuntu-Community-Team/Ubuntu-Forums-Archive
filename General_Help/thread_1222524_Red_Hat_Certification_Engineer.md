### Post by dhughes on 2009-07-25
Packages are handled differently, Debian (Ubuntu) based distributions use .deb and Red Hat use .rpm, other than that I'd say you're OK to practice with any distribution. 

 If it's a commercial Linux app I'd say it's probably going to be Red Hat based so you may want to focus on installing and configuring packages using .rpm , if you're going for Red Hat Engineer Cert I'd say stick to a Red Hat distro.
