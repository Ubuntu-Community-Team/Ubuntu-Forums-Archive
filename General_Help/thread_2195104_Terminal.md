### Post by Lars Noodén on 2013-12-22
This page gives some background information:

[https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo)

Basically 'root' is the superuser which has full access to all aspects of the system.  Normally you run your system (and its services) with less privileged users.  However, some activities need elevated privileges, but just for specific things.   You need root privileges to install system-wide packages, for example because it affects the whole system.
