### Post by happyhamster on 2010-08-07
Instead of killing the power outright, try the sequence:

SysRq + Ctrl + Alt + r
SysRq + Ctrl + Alt + e
SysRq + Ctrl + Alt + i
SysRq + Ctrl + Alt + s
SysRq + Ctrl + Alt + u
SysRq + Ctrl + Alt + b

This will reboot the system in a much more controlled way (reducing the chance of filesystem damage).
