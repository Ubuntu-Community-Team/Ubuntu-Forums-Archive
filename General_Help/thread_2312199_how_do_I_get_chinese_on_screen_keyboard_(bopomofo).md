### Post by anotherChris on 2016-02-03
Please read this thread:
[http://ubuntuforums.org/showthread.php?t=2311145](http://ubuntuforums.org/showthread.php?t=2311145)

If you are using a new release like 15.10, you want to use fcitx.  fcitx serves Chinese and Japanese.  I have installed the Japanese and have a virtual keyboard that will input hiragana and katakana.  So I imagine the Chinese version has a virtual keyboard with what you want.

If you search for fcitx and iBus, I think you can do it.  In Lubuntu 15.10, I had to start with language support, update it, then install the languages I want, then choose keyboard input method.  I don't know where to start on other releases of Ubuntu. Sorry and good luck.
