### Post by pay on 2006-10-17
With Amarok are you using Gnome or Kde? With Gnome just right click a ogg or mp3 file and click properties and change the program that it automatically opens with. With Kde right click the file Then properties --> General Then there is a symbol next to type that you can select and change the default app.

With firefox you will find you have better speeds if you compile it yourself with out cario. A project that does this is swiftfox.
