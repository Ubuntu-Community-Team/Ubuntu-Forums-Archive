### Post by sbassett on 2006-06-09
This sounds like the visual feedback feature. Stick with me here, as I hardly ever go into these options anymore. I believe if you select System-Preferences-Sound there are 3 tabs (?) On the middle or last tab I believe there is a check for visual feedback with system bell (some such wording, you'll know it when you see it). Make sure that is unchecked.
