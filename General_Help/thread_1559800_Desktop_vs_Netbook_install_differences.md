Welcome to the forums :popcorn:

Every OS is not Windows my friend. It is 10.04 Ubutnu Desktop

You can sure install Ubuntu Desktop on a Netbook. But it is the Netbook edition which is specially tweaked for Netbooks, small displays, less memory, wifi etc. See a list of supported Netbooks [here]("https://wiki.ubuntu.com/HardwareSupport/Machines/Netbooks"). Remember one might not be listed but might be supported. I recommend you go with the Netbook Edition for the Netbook and Desktop Edition for the Desktop.
