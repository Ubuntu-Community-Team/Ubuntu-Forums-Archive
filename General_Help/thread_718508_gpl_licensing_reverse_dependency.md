### Post by clgc on 2008-03-08
Suppose that a GPLv3 software (A) has a dependency on some other free  software (B). 
A needs B in order to run (you can imagine B providing some basic services to A).

Suppose now that I write a non-free software Z and distribute it to users of A, so that they can run A with Z without the need for B.
Moreover, Z gives users of A additional benefits, without any need to change A's source code.

Is this legal?
