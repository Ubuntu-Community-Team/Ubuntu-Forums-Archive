### Post by lovinglinux on 2011-01-19
Install [Open With]("https://addons.mozilla.org/en-US/firefox/addon/11097/") extension, go to the extension preferences and add your preferred image viewer to the list (like **/usr/bin/gimp**). To open an image link with the image viewer, right-click over the link, select "Open Link with" from the context menu.
