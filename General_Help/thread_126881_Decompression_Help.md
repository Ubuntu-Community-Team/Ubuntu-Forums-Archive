### Post by GeneralZod on 2006-02-08
I don't know about ace, but rar files can be extracted by installing unrar-free from the repos (you'll probably need to enable some extra ones; search the forums for unrar free) and opening your .rar file with arK.  

It's possible that this will also work from the context menus as soon as you install unrar-free.
