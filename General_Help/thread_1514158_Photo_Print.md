### Post by cogier on 2010-06-20
Photo print is available in Ubuntu 10.04 from the **Unbuntu Software Centre**. Just type **photoprint** in the search box alternatively run the following code in terminal```
sudo apt-get install photoprint

```.

I have managed to get my HP Photosmart to print borderless A4 photos beautifully.  

Hope that helps.
