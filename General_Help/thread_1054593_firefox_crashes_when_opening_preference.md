### Post by x33a on 2009-01-29
press alt+f2, type gnome-terminal.

then type: firefox -ProfileManager in the terminal and press enter

a small window will open up letting you create a new profile or use the current one.

create a new profile and see if your preferences crash again.

if not, then all's good.

after that go to firefox addons site and download febe.

configure febe to backup stuff of your choice (only firefox backups).

take regular backups.
