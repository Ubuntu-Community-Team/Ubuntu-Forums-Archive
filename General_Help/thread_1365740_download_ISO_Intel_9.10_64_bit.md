### Post by Some Penguin on 2009-12-27
In case it's not clear, 64-bit Linux builds are often referred to as AMD64, not because it's AMD-specific (it's not!) but simply because AMD got there first.  The AMD64 ISO will install fine on a 64-bit Intel box.

Now, you *could* build a kernel which was built with the AMD CPU family in mind instead of the generic x86 64-bit architecture.  But that's not what was done here.
