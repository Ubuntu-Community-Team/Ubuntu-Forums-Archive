### Post by sanderd17 on 2010-10-13
The iso's are just basic "packs". From each CD you can install the same software, the only difference is what software is pre-installed and what has to be downloaded.

Although you can install the same software with each CD, you should choose the CD that fits you best. If, e.g. you install regular ubuntu, and afterwards you decide to give Kubuntu a try, you can install Kubuntu with just one click. But your program menu will become a mess. Of all kinds of apps, the gnome and the KDE version will be installed:

konquerror and firefox, gedit and kate, amarok and rhythmbox, openoffice and koffice ...

Your settings will be shared too, which can give some little problems.
