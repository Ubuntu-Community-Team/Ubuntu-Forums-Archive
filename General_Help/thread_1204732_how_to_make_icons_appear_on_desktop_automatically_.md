### Post by 3rdalbum on 2009-07-05
It's a security feature, so no.

Having said that, the sorts of icons that come onto the desktop should be able to be double-clicked to execute, but Gnome will ask you if you trust the program. If you answer in the affermative, the next time you double-click the program it should run without any prompting.
