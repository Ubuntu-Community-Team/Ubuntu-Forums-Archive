### Post by TheRabbit777 on 2006-07-10
I succeeded in getting Dapper Drake ubunto up an running.  I had a problem with the java plug in.  When I went out on the net I run into sites that required this.  I dug and dug and was beginning to believe their was no solution.  I almost stumbled over the solution.  As it turns out the problem was not with the ubuntu system itself but rather a problem with mozilla firefox browser.  The way I fixed it was I inserted the ubuntu install cd, opened up the synoptic manager and run a search for mozilla  Wouldn't you know it  there was only one entry found for mozilla. I opened the manager for mozilla, scrolled down and low and behold there it was ! the Java fix I needed.  I installed it and now am no longer having any problems.  Hope this helps all of you who are experiencing the same problems I was having:razz:
