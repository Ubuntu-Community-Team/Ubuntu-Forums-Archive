### Post by cjazz on 2007-11-11
My guess is that, based on your description, what you probably saw was Ubuntu running as the host operating system and Windows running as a guest operating system in a virtual machine, a la VMWare or Virtual Box. It is also possible to run a Windows desktop via Wine. It would not be possible to run your native Windows and your Ubuntu, installed on separate partitions, at the same time.

You have dual boot setup, however you can not run both operating systems at the same time. It is possible to install virtual machine software that will allow you to run one OS in a virtual machine inside the other OS. But here and again you can only use one OS at a time.
