### Post by JEBB on 2009-05-04
I installed 9.0.4 UNR on my Dell Mini 9 using a flash drive.  All went well.  I would like to try Classic Desktop but am uncertain if and how I will be able to get back to the netbook desktop.  

So my questions are:  Will I be able to go back and How will I do that?  Will there be an easily findable and easily usable Preference to go back to the netbook desktop?  Where will I find it?  Thanks.

---

### Post by geirha on 2009-05-04
In netbook mode you should find it on the Preferences page as Switch Desktop Mode. In classic mode you'll find it under System->Preferences->Switch Desktop Mode. 

In fact, what you see on the preferences page in netbook mode, is what you'll see in the System -> Preferences menu in classic mode.

### Post by Brandon Williams on 2009-05-04
Before using the "Switch Desktop Mode" utility, install the updated version [from here](http://launchpadlibrarian.net/26020903/desktop-switcher_0.4.6_i386.deb). I don't believe this fixed version has made it into the Jaunty updates yet. If you use one from the original release, it will leave you with no window manager and no panel(s) when you next log in.
