### Post by BetterSense on 2008-10-09
When I use firefox I like to press Ctrl + twice in order to make the text bigger. I want to set it to be like this all the time, but I can't figure out how. Setting the font size in preferences doesn't seem to do anything. I must be doing it wrong.

---

### Post by AlexBellisBrown on 2008-10-09
Go to Preferences within Firefox, Content tab, Change the Font, and the size if you wish. Also, click the advanced button, and change the other 2 font sizes too. That should make it work for all kinds of text (not just certain sections). Then, press crtl + Q (quit firefox, do it this way, and not via the X in the corner). Then simply restart Firefox! Hope it works!! :):)
