### Post by binbash on 2008-11-29
you can do it via compiz wallpaper plugin BUT if you are using gnome, you cant use desktop icons.Enable wallpaper plugin at compiz, put 4 desktop wallpapers for example.THen open gconf-editor go to apps > nautilus and untick show desktop(or similar to this, i dont have gnome at the moment)
