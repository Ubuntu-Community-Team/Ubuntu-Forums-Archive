### Post by Yellow Pasque on 2008-10-12
You have a typo in the file (someone or something added a repo incorrectly). Open it for editing and remove the " in front of line 58
```
gksudo gedit /etc/apt/sources.list
```

### Post by Sam on 2008-10-12
Replace the last line```
“deb http://packages.dfreer.org feisty main”
```
with```
deb http://packages.dfreer.org feisty main
```
(that's it, no quotes)

Then run```
sudo apt-get update
```
