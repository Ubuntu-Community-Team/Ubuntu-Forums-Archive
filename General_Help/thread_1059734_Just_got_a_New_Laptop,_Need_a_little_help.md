### Post by lakersforce on 2009-02-04
Just in case you havn't done it...go into bios (at startup) and change your devices boot order. Make sure that your cd-rom drive is on the top of the list.

Perhaps you would rather install Vista first? That way you can have both operating systems on the same computer. But to make it work, you HAVE to install Vista first.
