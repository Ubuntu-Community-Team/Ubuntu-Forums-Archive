### Post by frankbooth on 2011-08-15
By default, navigating backwards in Firefox has two keybinds:
```

Alt + &#8592;
Ctrl + [

```

But there are addons that can customize the keybinds, [here]("https://addons.mozilla.org/en-US/firefox/addon/customizable-shortcuts/") is one that might suit your needs.
