### Post by rihikz on 2013-11-02
Hello. I'm not sure if these features exsist in Ubuntu 12.04, but you could just use thunderbird as an email client for gmail, which should send you notifications and display the unread message count in the launcher icon, if opened.
If you don't like unity you can just install the GNOME desktop environment and it will be the same as it used to be while keeping all the same features.
Also, both Firefox and chrome take up 100mb to 450mb of RAM, so you would be wasting a lot of RAM just because of an email notification.
I would recommend looking into the alternatives, rather than running a separate browser window.
