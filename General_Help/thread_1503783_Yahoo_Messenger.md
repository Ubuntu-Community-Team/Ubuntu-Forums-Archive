### Post by Sirron on 2010-06-07
Pidgin is your best alternative in my opinion; empathy is over-simplified.

The official Yahoo Messenger [_client for linux_]("http://in.docs.yahoo.com/messenger/download/unix.html") is very old and not very good.

Note that you won't be able to use voice chat or webcam if you use pidgin; though you can get voice chat working if you install the Windows version via wine.

If you really need webcam, you could use a Windows installation inside a Windows virtual machine; VirtualBox allows you to pass USB devices - such as your webcam - to the guest machine.
