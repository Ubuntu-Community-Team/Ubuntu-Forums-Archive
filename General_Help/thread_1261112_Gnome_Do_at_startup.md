### Post by themusicalduck on 2009-09-08
Not sure about quiet mode. I've had problems with quiet mode resetting itself before as well.

Another way to make it load on startup would be to go to System > Preferences > Startup Applications.

Click on Add and then in the command box type ```
gnome-do
``` and whatever you like in the other boxes.

Or if there is already something there called gnome do then make sure it's checked.
