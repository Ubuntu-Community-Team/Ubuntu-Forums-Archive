### Post by CeejRab on 2013-09-15
Hello Mayagrafix,

Definitely sounds like a hardware issue. Ubuntu is pretty flexible with adapting to hardware, and it wouldn't work fine and suddenly stop working. I think youre right, the internal PC hardware needs some TLC.

You can test this theory by running Linux live cd of a different distro such as Slax. Play a video or audio file and see if the same issue occurs after the computer warms up. If so, its hardware. If not and it works fine, possibly your driver needs an update but I doubt that... Hope this helped a little!
