### Post by The Cog on 2009-08-31
I would guess that somehow you accidentally moved all the disk contents to the trash. Maybe by pressing delete while the top-level folder was highlighted, or maybe by selecting the wrong option in a right-click popup menu. I've done that myself on occasion, especially when using a mouse that needs more pressure than usual to keep the button pressed. A slight finger tremor can release/re-press the button unintentionally.
