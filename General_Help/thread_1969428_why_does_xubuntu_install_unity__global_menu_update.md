### Post by brainwash on 2012-04-30
Please provide more details on this. Only Firefox and Thunderbird should get updates related to global menu integration (firefox-globalmenu, thunderbird-globalmenu). These extra packages are recommended, but not essential, therefore they can be removed without breaking Firefox/Thunderbird.

There are some gnome apps bundled with Xubuntu like the document viewer Evince, but you should not get updates for Unity.

Xubuntu does not offer the global menu functionality, nor is Unity part of the default installation.
