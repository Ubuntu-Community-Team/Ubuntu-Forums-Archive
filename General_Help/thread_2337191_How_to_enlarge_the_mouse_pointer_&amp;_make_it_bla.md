### Post by Dennis N on 2016-09-15
> Nope - doesn't work in Ubuntu Mint. 
You mean Ubuntu MATE? There is no Ubuntu Mint.

Larger (and black) mouse pointer in Ubuntu MATE:

Open the Appearance dialog (Control Center > Appearance). Your theme in use should be automatically selected (Note: if you previously customized any theme, then your theme is shown as 'Custom'). Then use the customize button and open the 'pointer' tab. Here you select the cursor theme. There is a slider there to adjust the size. However, just some cursors can be changed in size - the resizable black cursor in Ubuntu MATE would be DMZ black.
