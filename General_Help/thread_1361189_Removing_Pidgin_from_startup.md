### Post by howefield on 2009-12-21
Try System > Preferences > Startup Applications


If it isn't in there try closing all running applications then go to System > Preferences > Startup Applications, and click on the Options tab, check "Automatically remember running applications when logging out".

Close and reboot.

Do it again, only uncheck the box this time.
