### Post by wersdaluv on 2008-12-05
There's an issue with Yahoo and 3rd party clients right now. What I suggest for you is to use Gyachi for now. It's a Y!M client with Y!M features like photo sharing, webcam, audibles, etc. 

To get it, add this to your sources.list for intrepid

```
deb http://ppa.launchpad.net/loell/ubuntu intrepid main

```

for hardy 

```
deb http://ppa.launchpad.net/loell/ubuntu hardy main

```

for gutsy

```
deb http://ppa.launchpad.net/loell/ubuntu gutsy main

```

[https://launchpad.net/~loell/+archive](https://launchpad.net/~loell/+archive)
