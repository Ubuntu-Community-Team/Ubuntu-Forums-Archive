### Post by Frogs Hair on 2013-06-15
I don't know of one application. but the Gnome Tweak  Tool, Unity Tweak Tool, and Ubuntu Tweak are all front ends for settings that used to be confined to configuration editor. Compiz (CCSM) is independent of these applications except for some unity specific settings which Unity and Ubuntu Tweak include.
