### Post by cometa2k7 on 2008-04-20
> **randomAccess said:**
> Hm, it says I need to get Crossover Games which is trial version only. Will Office 2007 work after this trial has expired? :confused:

Or i can just use regular Crossover Pro instead of Games version?

Yeah, you can even uninstall CrossoverGames afterwards, you only need the trial version to get one of the files from one of the directories. Office 2007 seems to rely on this version of the file, and not work with Wine's version. It doesn't matter that it's only a trial version, you don't actually use it to run Office2007, you just need one file.

The file you need is in '/opt/cxgames/lib/wine/' It isn't too clear in the instructions. I think that it's the rpcrt4.dll.so file, you just need to rename it when you put it into '/home/user/.wine/drive_c/windows/system32/'.

Also, you need 'msxml3.msi' fromt the internet and install it with the command 'msiexec /i msxml3.msi' in a terminal, make sure you 'cd' to whereever you downloaded the file to.


Personally, I haven't tried it with anything other tha Crossover Games, so I don't know about Pro. It might be exactly the same, so you could try it.

If you already have Crossover Pro installed, you might be able to run Office 2007 with that, instead of using Wine. But I can only speak for the Wine method.
