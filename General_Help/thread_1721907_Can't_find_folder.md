### Post by deathadder on 2011-04-05
A file or directory that begins with a '.' is a hidden file / directory in Linux. The shortcut key nothingspecial gave you simply shows all hidden files.

If you're using Thunar (default file browser in Xubuntu) you can simply go to "View->Show Hidden Files" aswell.
