Why do we use "sudo apt-get update" all the time before we install any application?

---

### Post by Saint Angeles on 2008-04-30
it updates the lists of packages available on the repos so that your computer knows which files to update.

---

### Post by tamoneya on 2008-04-30
basically it makes sure that you get the most up to date version of the program you plan on installing.  If you dont do this you may install an old package and then just have to update it later.  Best to just install the new version.
