### Post by drs305 on 2007-05-27
unison is another program that can backup directories by comparing the contents.

It works in both linux and windows.

[http://www.cis.upenn.edu/~bcpierce/unison](http://www.cis.upenn.edu/~bcpierce/unison)

Hint: If you use it and want to exclude subdirectories, you use the Ignore Path function within the preference file.
