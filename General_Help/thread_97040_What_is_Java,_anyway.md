### Post by Jonne on 2005-11-30
I don't know what this blackdown java is, but i can answer some of your questions:
Java is a programming language, and the goal is that you can write a program once, and you could be certain it runs on every platform (that has the java 'virtual machine' installed).
So the people that write Azureus only have to write it once, and it works on Windows, Linux, and Mac OS (and others).

Firefox doesn't need java, unless you want to be able to run java applets imbedded in sites.

### Post by az on 2005-11-30
There are websites that depends on java.  If your browser does not have a java plugin, some sites do not display correctly.

Java is proprietary.  That means that it is someone's property.  There are free implementations of it (Sablevm, Kaffe, etc) which are coming along, but are not as fully-functional as sun's (the owner) version.  The version from Sun caintains a clause which prohibits you from distributing any other version of java if you distribute theirs.

That would mean that ubuntu could not distribute the free implementations and is not acceptable.  Sun cooperated with Blackdown to create an open-source (but not free-libre) version of java.  AFAIK, it used to have the same evil licencing problems as Sun's.

### Post by Edit on 2005-11-30
Hey linbetwin,

What most people are installing is the Java Runtime Environment (JRE).  This is basically an interpreter that enables Java based programs to run.  

Because most Java programs use the Java Runtime Interpreter to process real-time instructions to the PC, without the Runtime Environment the programs wouldn't run.

Here's a few things that require the JRE to run:

Azureus (afore mentioned)
JEdit - Programmers text editor
Most online chat rooms
Online games

And, I guess, EVERYTHING that has been programmed in Java.
