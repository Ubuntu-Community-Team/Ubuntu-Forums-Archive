### Post by sto6ma9ch on 2005-11-19
It looks like you'll need to do the following:
[LIST]
[*]Find and install this icon theme in System Settings ->  Appearance -> Icons (check [http://www.kde-look.org](http://www.kde-look.org))
[*]Find that KDE wallpaper and set it in System Settings ->  Desktop -> Background (again, check [http://www.kde-look.org](http://www.kde-look.org))
[*]Enable a transparent Panel in System Settings -> Panel -> Appearance (there's a check box at the bottom)
[*]Enable a transparent KDE menu in System Settings -> Appearance -> Style -> Effects
[*]Shrink the Panel in System Settings -> Panel -> Arrangement
[/LIST]
Post again if you have issues finding this stuff.

### Post by campusloop on 2005-11-19
I am not sure what kde theme is being used - my guess would be linspire - [http://www.kde-look.org/content/show.php?content=23871](http://www.kde-look.org/content/show.php?content=23871). there are other lots of themes at kde-look.org which you can try.

Icons are nuvola - [http://www.kde-look.org/content/show.php?content=5358](http://www.kde-look.org/content/show.php?content=5358).

menu drop shadow/transparency settings can be adjusted by - 
KDE control center(kcontrol)->appearance and themes->style->effects tab->check "menu drop shadow" and select "make translucent" in "menu effect" dropdown.

for kicker panel transparency - kcontrol->desktop->panels->appearance tab->check "Enable transparency"

to adjust bottom panel size - kcontrol->desktop->panels->arrangement->adjust the "position" and "length" appropriately to make it centered and reduce the size.
