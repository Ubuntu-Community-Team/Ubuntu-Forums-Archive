### Post by iamhugeinjapan on 2006-12-22
In my experience, sharing the same hard drive has been the most problematic. I never had any problems dual booting until I started using the same hard drive. It does usually work though, but for some reason it can go wrong sometimes. I've had Windows manage to destroy a linux partition once or refuse to boot after installing a linux distro.

But then again, I bet a bunch of people will start posting the opposite.

Upshot is that partition management wise, it's safer to have them on separate hard drives
