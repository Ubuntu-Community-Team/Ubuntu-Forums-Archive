### Post by gintovan on 2010-10-11
If you have 4GB or more ram, take the 64bit version.
Other than that you won't really see any differences as a normal user. 

In the past it's true that 64bit drivers/apps was missing or lacking behind, but this is not really the case anymore. And you can still install 32Bit applications on a 64bit OS.

I have used 64Bit Ubuntu since 8.04(I think) with no problems!

### Post by 3rdalbum on 2010-10-11
64-bit Ubuntu is just as well-supported as 32-bit Ubuntu.

Basically, it's as simple as this: Install 64-bit if your processor can handle it and you have more than 2 GiB of RAM. Otherwise, install 32-bit.
