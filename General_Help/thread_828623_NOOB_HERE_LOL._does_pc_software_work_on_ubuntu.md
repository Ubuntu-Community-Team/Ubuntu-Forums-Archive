### Post by Rhubarb on 2008-06-13
Why use Nero when there are many other cd / dvd burning applications that are available for Ubuntu for free?
Ubuntu 8.04 comes with Brasero by default - try using it, it's easy.

If you want to try a different cd / dvd burning program for Ubuntu, get:
sudo aptitude install k3b
or
sudo aptitude install gnomebaker

There is little to no reason why you'd want to use Nero (even less so the windows version) on Ubuntu.
