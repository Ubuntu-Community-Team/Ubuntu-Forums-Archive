### Post by SeijiSensei on 2012-11-30
Gwenview with the kipi-plugins add-on installed supports uploads to many sites including Facebook, Flickr, and Picasaweb among others.  It's a KDE application, though, so when you install it you'll get a bunch of additional libraries unless, like me, you're running Kubuntu.

```
sudo apt-get install gwenview kipi-plugins
```
