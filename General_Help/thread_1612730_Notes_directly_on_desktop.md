### Post by yetiman64 on 2010-11-03
I am not aware of any programs that allow writing to the desktop background itself. However the program xpad, available in the repos, is very customizable although similar in basic use to tomboy notes.

The attached pic is an example of a stickynote I just created on my desktop. To turn on or off the Windows decorations and the toolbars for the note is as simple as right clicking in the note and using the context menus. Controls for it are available from the Notification Area when it is running. 

With this note on the desktop and xpad put into my Startup applications the note returned, as it previously was, after a reboot.

Hope this is of some help to you.
