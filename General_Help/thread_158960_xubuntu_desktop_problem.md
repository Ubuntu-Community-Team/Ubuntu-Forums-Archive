### Post by krusbjorn on 2006-04-12
I think its because nautilus takes over your desktop. Kill nautilus and things should get back to normal. If you wanna use nautilus for file browser, uncheck apps->nautilus->preferences->show_desktop in the gconf editor, or always run nautilus with the command "nautilus --no-desktop"
