I have seen the [gEdit AutoComplete plugin]("http://zend.lojcomm.com.br/entries/gedit-autocomplete-0100/") in operation on the author's YouTube video, and would like very much to get it working on my system.  

What I've done:
I've downloaded the latest v0.11.0 of the plugin and installed it [as directed by the author]("https://github.com/nagaozen/gedit-plugin-autocomplete"):
> Installation
   1. Copy both autocomplete.gedit-plugin and autocomplete folder to ~/.gnome2/gedit/plugins
   2. Open gedit and click Edit -> Preferences -> Plugins
   3. Check the Autocomplete (gedit-json-completion) and hit Close
   4. Check the Word Completion and hit Close
   5. That's it! Now you can start coding and watch the popup working. Hit Return or Tab to accept the completion.
But, when I open gEdit, go to Edit->Preferences->Plugins -- it doesn't appear.

Thinking gEdit didn't have the correct permission to use the plug in, I progressively relaxed the permissions, checking for correct operation each time, until I now have full read-write permission to the containing folder and all its files.

Does anyone have this plugin working, and what advice could you give me?  Have I forgotten or misread anything?
