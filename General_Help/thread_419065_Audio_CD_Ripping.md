Hi,

 abcde has the most flexibility and will allow the options that you mentioned:

[http://people.aapt.net.au/~adjlstrong/ubuntu_cli.html#mp3](http://people.aapt.net.au/~adjlstrong/ubuntu_cli.html#mp3)

```
--preset extreme
--preset insane

```
