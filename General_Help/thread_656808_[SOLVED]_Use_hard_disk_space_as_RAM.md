### Post by PeterJS on 2008-01-02
Yeah that's what swap space is, you should have set that up during the install. But be warned that swap is **extremely slow** compared to RAM and really only should be used as a last resort.

Here's a post on adding more swap after the install:
[http://ubuntuforums.org/showpost.php?p=489196&postcount=3](http://ubuntuforums.org/showpost.php?p=489196&postcount=3)
