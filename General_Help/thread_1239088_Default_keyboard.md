### Post by Icebear on 2009-08-13
try if this will work

goto System>Preferences>Keyboard
hit the layout button

do you see 2 layouts

eg  USA
and United Kingdom

if so click on usa and then hit the remove button

If you see only usa layout add a United Kingdom layout and then remove usa layout
