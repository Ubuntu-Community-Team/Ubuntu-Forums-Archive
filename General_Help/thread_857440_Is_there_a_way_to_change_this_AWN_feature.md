### Post by crazyness003 on 2008-07-12
I eliminated this by using the Stacks applet. I have 8 app launchers i stored in a folder on my home holder and linked the Stacks app to that folder, now i have clean access to my most used apps with the addition of 1 more click. Using this method i can open (if applicable) unlimited instances of the same app.

As for firefox, i think theres a setting in firefox that tells it to only have 1 instance open at a time. this will only allow 1 window (excluding the downloads, addons, bookmarks etc sub-windows). I'd check that out.
