### Post by adit on 2011-04-05
Both archeos and kubuntu use the same **package management system**.  Still there is risk of breaking the system if you use programs packaged for one distribution in another distribution.
You should not even use programs packaged for kubuntu 10.10 in kubuntu 11.04. (same distribution, version number only changes).
