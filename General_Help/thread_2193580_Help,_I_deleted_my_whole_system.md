### Post by bashiergui on 2013-12-13
Nope. Even if by some chance the format did not write zeros on the entire drive, the Windows installation would have overwritten too much of the Mint data. You might be able to recover bits and bytes here and there, but you'd never get the whole file structure back. If the format worked as expected then you've written over the Mint data several times. 

There is no recovering data that has been overwritten.

Recover the data from a backup or recreate it from scratch.
