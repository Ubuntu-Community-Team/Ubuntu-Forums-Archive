### Post by jpeddicord on 2007-01-13
If you are using the Tango icon set, this is the file you will need to look at:
/usr/share/icons/Tango/scalable/places/start-here.svg

If you are using the default Ubuntu theme, change this one:
/usr/share/icons/Human/scalable/places/start-here.svg

For any other themes, just change the theme name after /usr/share/icons.
