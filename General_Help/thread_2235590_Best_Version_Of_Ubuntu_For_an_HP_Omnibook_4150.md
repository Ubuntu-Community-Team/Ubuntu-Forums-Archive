### Post by yancek on 2014-07-22
With 64MB of RAM and a 9.4GB hard drive, Ubuntu definitely won't work.  You won't be able to get any current full distribution to run.  The current Ubuntu recommends 1GB of RAM.  You might get AntiX or TinyCore Linux to run or if you can add RAM to the maximum of 256MB try Slitaz.  Try an online search as they have their own sites.
