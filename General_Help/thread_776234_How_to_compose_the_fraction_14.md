### Post by demetrisk on 2008-04-30
You can also try the Ctrl+Shift+U method:

**Ctrl+Shift+U** and **b** and **c** and then Space, or Enter or comma, etc.

(00BC is the Unicode hexadecimal number for fraction one quarter.  You can find such numbers in the Character Map (Alt+F2, gucharmap) by searching for the name (or part of the name) of the character.)

Compose Key seems to me great in theory, but I've found it takes some tinkering to make it work for all the characters I need in my particular layout and in my particular locale. So I use a combination of all methods and I'm happy: :-)

1. Compose Key for a few that work for me and I can remember easily
2. AltGr for those that are availabe on the layout I use
3. Ctr+Shift+U for a few whose numbers are easy to remember
4. Character Palette on the top panel for some more
