### Post by Oldsoldier2003 on 2008-06-03
> **rustyfishhook42 said:**
> how do i password protect files?

there is no "password protection" per se in Ubuntu. You can secure the directory by use of the permissions or you can use an encryption tool such as PGP to secure a file. You might also consider using a truecrypt container or partition if you are serious about maintaining the security of your data files.

---

### Post by p_quarles on 2008-06-03
As Oldsoldier2003 suggests, the easiest way to protect individual files is to encrypt them. GPG is a good and efficient way of doing this, and has the added benefit of being useful for secure communications.
