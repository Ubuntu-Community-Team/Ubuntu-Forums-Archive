### Post by amohanty on 2005-11-30
Usually it is one of:
F2
Delete
Esc

Try them all out in quick succession when the first text appears on you monitor, you know the one with:
Checking memory etc.

However if the first thing you see is a wallpaper from Dell, Asus, <insert your vendor here> then watch real carefully for something that goes:
Press <key> to enter Setup
