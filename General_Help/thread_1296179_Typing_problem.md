### Post by Giblet5 on 2009-10-20
If you touch a Synaptics TouchPad®, it will move the keyboard cursor to wherever the mouse cursor was sitting.

It will.

That is what it does.

I don't care what Windows does, that is what a touchpad does....by default

until you turn off the button click capability so that the physical buttons must be pressed to signal a buttonpress event.

I think you can do that from System->Preferences->Mouse...
