### Post by Holger_Gehrke on 2018-12-16
[Set keyboard shortcuts]("https://help.ubuntu.com/stable/ubuntu-help/keyboard-shortcuts-set.html.en") in the official documentation for Ubuntu 18.10 (should be the same for 18.04). On the same page you'll find several predefined shortcuts for screenshots (usually the 'print' key and some combination of shift, ctrl, alt or super) for taking images of the whole screen, the active window or a selected region of the screen to either a file in the pictures directory or the clipboard.

Holger
