### Post by pastalavista on 2010-03-13
Welcome to Ubuntu and the Forums. Your wireless problem is probably because there are no drivers for it installed. Did it work from the LiveCD?

You might need to connect via ethernet (wired) in order to access the repositories and update everything. The wireless card should be recognized and offered up by the Hardware Drivers app in the System | Administration menu.
