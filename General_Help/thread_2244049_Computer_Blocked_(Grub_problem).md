### Post by yancek on 2014-09-13
Did you fully shut down windows before installing Ubuntu?  Was windows 8 pre-installed?  Do you know if you selected to install Ubuntu in EFI mode?  There is not enough information in your post for anyone to do more than guess so boot up the Ubuntu installation medium, if you get an internet connection go to the site below, download and run the bootinfoscript from that page.  There is a link in the Description box leading to a page with instructions, be sure to read that.  After the script is run, it will output a results.txt file.  Post that here and someone should be able to advise you. 

[http://sourceforge.net/projects/bootinfoscript/](http://sourceforge.net/projects/bootinfoscript/)
