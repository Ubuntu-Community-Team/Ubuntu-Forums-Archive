### Post by prshah on 2010-01-12
> **Mark76 said:**
> I have the right driver installed (Nvidia 173), yet I keep getting the "Desktop effects could not be enabled" pop up every time I try to enable them.

Try to enable it from a Terminal (Applications-Accessories-Terminal) with the command```
compiz --replace
```

If it does not work, please post back the output from the command. If it works, it will stop as soon as you close the terminal. In that case, press Alt+F2 and repeat the same command again.
