### Post by agim on 2008-02-23
There isn't a fix using Ubuntu, but you don't have to reinstall windows to get things rolling.

[http://supergrub.forjamari.linex.org/](http://supergrub.forjamari.linex.org/)

This is a link to the super grub disk. There are instructions on the site, make sure you read them. You will have to download the super grub disk and burn the iso to a cd. Then run the livecd version of the super grub disk. You should have your windows booting in no time.

Edit: Oh, and generally it doesn't take this long to get a reply. Especially not for something relatively common like this.
