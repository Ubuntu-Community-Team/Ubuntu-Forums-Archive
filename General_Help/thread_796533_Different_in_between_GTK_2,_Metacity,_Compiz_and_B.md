### Post by PinkFloyd102489 on 2008-05-16
GTK2 changes the appearence of the panel, buttons, window color, etc. Compiz and Beryl are the same, now called Compiz-Fusion. You can install either Compiz or Beryl emerald themes.

Metacity is the default window decorator, which changes the titlebars and buttons in the titlebars. Compiz, when enabled, replaces Metacity as the default window decorator.


Basically, you can use a GTK2 theme and a Metacity/Compiz theme at the same time.
