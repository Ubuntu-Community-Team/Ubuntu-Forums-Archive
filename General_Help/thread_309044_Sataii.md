### Post by smoker on 2006-11-29
hi, like the avatar!

sometimes you have to enable sata in the bios before your os will pick up a sata drive. if you go into the bios (usually hit a key like del, of f2, f10, on boot up), look for an option to enable sata.

also, if you happen to be using windows at the moment, you will have to install separate sata drivers as windows dosen't detect sata without them.

hope this helps

yes, to enable sata on my board i can only use one ide channel, so in my case i have sata and ide channel 2 enabled, my dvd drives are on channel 2. i have 2 sata drives therefore one ide channel is enough for me.

as far as backward compatibility with sata, i would try your harddrive makers website, on my sata drives (maxtor diamonds) there is a jumper which can be set to run the drives at the slower speed of sata 1. maybe check your drive for a similar mode (autonegotiation) i believe tho i am no expert.
