### Post by Famicommie on 2007-04-22
If you had used the 64 bit version of Windows XP, you would have experienced the same disappointment. While 64 bit processors are rapidly becoming the norm in newer computers, many users are still on P4 or (gasp) P3 systems. Software developers want to meet the demands of as many users as possible, and since both the AMD and Intel 64 bit processors run fine in 32 bit environments, 32 bit has remained the norm. First the hardware progresses, and then the software quite a while later, unfortunately.
