### Post by p_quarles on 2007-10-31
Firefox is sluggish under Linux on some systems. The main factors are your network interface and your ISP's DNS server.

The good news is that you can configure Firefox so that it will run considerably faster. I've found that these instructions help quite a bit:
[http://www.ubuntugeek.com/speed-up-firefox-web-browser.html](http://www.ubuntugeek.com/speed-up-firefox-web-browser.html)

You can also install Swiftfox, which is a version of Firefox that's been specially compiled for various processors:
[http://getswiftfox.com/debian.htm](http://getswiftfox.com/debian.htm)

Finally, you can opt to go for a different web browser altogether: Epiphany, Opera, Kazehakase, Seamonkey, and so forth.
