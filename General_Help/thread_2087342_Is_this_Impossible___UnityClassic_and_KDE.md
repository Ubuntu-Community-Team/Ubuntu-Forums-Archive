### Post by Quarkrad on 2012-11-23
Two family members use 12.04 - one with Classic Gnome and the other with Unity.  They use Digikam as their photo manager - however, there is a function in Digikam **Settings/Themes/Configuration** that appears to only function when Digikam is running in a full kde environment.  I have download the kde Desktop that you could switch to it but that is another learning curve.  Is it possible to download and install some kde dependances so that whilst in the Unity or Gone Classic environments you could use Digikam as though it was in it's kde environment?  Or rather, Digikam had all it needed whilst running in Unity/Gnome?

---

### Post by Quarkrad on 2012-11-23
Yes it is - may be a better solution but if you install the full kde desktop on your 12.04 ubuntu system **sudo apt-get install kde-full**  then launching Digikam in Unity allows the Settings/Configuration option in Digikam.
