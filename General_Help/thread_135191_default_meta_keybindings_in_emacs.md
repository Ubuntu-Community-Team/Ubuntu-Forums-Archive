### Post by bernardfrancois on 2006-02-23
Normally it should work by default. Maybe it's your first time that you use emacs and you don't recognise the response of emacs on your actions? You could try the tutorial first, you can start it from the Help menu, then choose 'Emacs Tutorial'.

If you know what is going on and you are convinced that this is not the problem, then just download emacs from the official website... [This](http://ftp.gnu.org/pub/gnu/emacs/emacs-21.4a.tar.gz) is the download link for the latest stable version (as for today).
