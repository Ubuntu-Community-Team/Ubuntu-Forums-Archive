### Post by taurus on 2007-10-26
Do you want to mount /dev/sdb3 (ntfs) with read-write or do you just want to mount it as read-only like you do with /dev/sda1?

Edit /etc/fstab

```
gksudo gedit /etc/fstab
```
and add this line to the end of it.

```
/dev/sdb3    /media/sdb3    ntfs    umask=222,utf8   0 1  
```
Save it.  Then, create a new mount point and mount it.

```
sudo mkdir /media/sdb3
sudo mount -a
df -h
```

### Post by taurus on 2007-10-26
In that case, you need to install ntfs-3g first since that drive would allow you to write to ntfs.

```
sudo apt-get update
sudo apt-get install ntfs-3g
```
Now, edit /etc/fstab again and change the line for /dev/sdb3 to this new one.

```
/dev/sdb3   /media/sdb3   ntfs-3g   defaults,locale=en_US.utf8   0   0
```
Save it and reboot.  See if you can write to /media/sdb3 now.
