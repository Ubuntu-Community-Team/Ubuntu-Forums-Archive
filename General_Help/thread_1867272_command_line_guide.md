### Post by Spyderkid on 2011-10-25
some quick little tips....

```
man COMMAND 
```
type in the COMMAND you want to find out what it does, and then it will display all the info and options you can do

```
ls -l 
``` This command colourfully displays all the info about what is in the current folder

```
cd DIRECTORY 
``` changes what directory(folder) you are in

```
cd .. 
``` moves up a directory

```
mv FILENAME DESTINATION 
``` moves a file in the current folder to another folder

```
gedit FILENAME 
``` opens a file in gedit text editor

```
mkdir 
``` makes a new folder

**```
sudo 
``` gives you root privliages without acctually being root **

```
/FILENAME 
``` runs that file
