### Post by callodacity on 2011-02-19
Everytime I run sudo apt-get update, it updates most of my sources, and then asks for me to insert the installation disc, once I have it finishes..

Do I have to insert the disc every time? it won't finish updating without it..

---

### Post by howefield on 2011-02-19
Try System > Administration > Synaptic Package Manager > Ubuntu Software tab and check whether the Installable from CD-ROM/DVD option is checked, if so, uncheck it.
