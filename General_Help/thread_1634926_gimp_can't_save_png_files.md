### Post by lykeion on 2010-12-01
However much I love GIMP I think the Save as dialog could do with some usability refinements.

The dropdown menu on the right in the Save as dialog is only to filter what files are shown.

To save as PNG in GIMP you have to click on the plus sign next to "Select File Type" and change it from "By Extension" to "PNG image". GIMP will automatically append .png to the file name when you save.
