### Post by jerrrys on 2012-10-29
I do not run 12.10, but Im thinking its the same for boot to text.

In /etc/default/grub, change this line

GRUB_CMDLINE_LINUX_DEFAULT="splash quiet" to

GRUB_CMDLINE_LINUX_DEFAULT="text"

Run **sudo update-grub** when done

Also its possible to do away with the display manager.
