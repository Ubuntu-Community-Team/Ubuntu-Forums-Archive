### Post by grahammechanical on 2015-08-12
If you are getting the English pound sign ( £ ) then it could be that you have an English UK keyboard layout installed. Do you? The last keyboard layout used is the keyboard layout in use the next time we try to log in. But there will be at the login screen an icon that will let us switch keyboard layouts if more than one keyboard layout is installed.

I have found that when I install Ubuntu and I choose English UK for the system language and the keyboard layout I get two keyboard layouts English UK and English US. And I have to remove the English US layout. So, I expect that when you installed Ubuntu you also had an English US keyboard layout installed by default.
