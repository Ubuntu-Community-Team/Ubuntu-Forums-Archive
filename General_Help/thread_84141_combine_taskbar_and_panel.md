### Post by aysiu on 2005-10-30
This is what I do, because I have the same problem with the default setup:

1. Delete the useless panel (the one that has fewer stuff on it--just the window list, basically)--I believe this is what you call "the taskbar."

2. Move the other panel to the bottom (Right-click panel > Properties > Orientation > Bottom).

3. Right-click panel > Add to Panel > Window List, Notification Area, Main Menu.

4. Remove the Programs/Applications/System menus (that take up a lot of space).
