### Post by gezzer2 on 2024-06-01
have you tried using the pstate power governors ?

[https://lemmy.world/post/3081149](https://lemmy.world/post/3081149)

you will need to add the pstate line to grub

if its an intel cpu it will look like intel_pstate=passive
if its an amd cpu it will look like amd_pstate=passive 

there are others such as =active ... =guided
the file is in etc/default/grub and i have mine set to this by adding the pstate entry ..
GRUB_CMDLINE_LINUX_DEFAULT="amd_pstate=passive quiet splash"

then sudo update-grub
and reboot

more information here.
[https://docs.kernel.org/admin-guide/pm/amd-pstate.html](https://docs.kernel.org/admin-guide/pm/amd-pstate.html)

you can disable these settings by removing the grub entry .. update grub and reboot ..
