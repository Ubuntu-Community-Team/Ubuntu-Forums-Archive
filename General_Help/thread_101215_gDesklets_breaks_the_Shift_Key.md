### Post by MonolithTMA on 2005-12-10
I found the problem.

The key for toggling float mode was set to <shift> + <F12> and for some reason it seemed to be waiting for <F12> to be pressed whenever I hit <shift>. I set that key to something else and it works like a charm now.
