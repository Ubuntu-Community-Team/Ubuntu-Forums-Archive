### Post by gymophett on 2010-02-09
My computer saves it's state when shutting down. So when I boot back up, all the applications I had running boot again. I HATE THAT. When I shut it down, I want everything GONE on next boot up. How do I fix this?

---

### Post by lykwydchykyn on 2010-02-09
Open "system settings".  Select "advanced user settings". Open "Session Manager".

Under the "On Login" section, select "start with an empty session".  Hit 'Apply'.

Done.
