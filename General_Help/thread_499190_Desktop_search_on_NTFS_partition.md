### Post by baba_oreilly on 2007-07-12
I dualboot Feisty and XP on my laptop, and I'm interested in getting some desktop search up and running on the Ubuntu side (like beagle, tracker, or google desktop). However, all the stuff I actually want to search is on the NTFS partition (MP3's, movies, documents, pdfs). Are any of these programs able to index my NTFS partition from Feisty? I have the drivers installed to read/write to NTFS, in case that would have been an obstacle.

### Post by Depressed Man on 2007-07-12
If you have the ability to read/write to NTFS then you should be able to search NTFS drives for files. I do it myself with beagle, after setting it up, just go to its preferences and add the directories on the NTFS drives you want to search.

### Post by Espreon on 2007-07-12
Is red/write enabled for the NTFS partion if not goto Applications>Add or Remove Programs and search for NTFS and check the checkbox for NTFS configuration tool, then click apply. After installation goto Applications>System Tools>NTFS Configuration Tool, check both and enjoy! I did this and worked pretty good.
