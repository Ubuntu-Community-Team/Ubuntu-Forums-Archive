### Post by liamgh on 2009-08-25
Might be best to ask over at: [http://forum.boxee.tv](http://forum.boxee.tv) for the answers to these questions; 1 & 2 I don't know. 3 can be achieved by pressing 'OK' while playing the video, then selecting *settings* -> *video* then either select *Crop black bars*, change the value of *View Mode* or change the *Pixel Ratio* value. 4 can be achieved by pressing <left> to get the left hand main menu up, selecting *settings* then *Appearance* -> *Region* and select a country that uses metric, such as the UK.

Hope this helps!
