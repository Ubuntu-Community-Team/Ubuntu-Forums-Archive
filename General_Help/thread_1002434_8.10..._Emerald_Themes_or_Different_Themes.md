### Post by Shwefty on 2008-12-05
System -> Preferences -> Emerald Theme Manager

Click "Import" and navigate to the Emerald Theme file.  (Beryl is the right one).

Some people get it to work with click and drag onto the Emerald Theme Manager window, but...that's hit and miss for me.

Don't forget to run "emerald --replace" in Terminal to have Emerald control themes rather than Compiz.  Also, if this is permanent, set it up to run on startup inside of Sessions.
