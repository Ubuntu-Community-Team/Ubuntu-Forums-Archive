### Post by jonlemur on 2008-05-17
How do I change the program that opens a file of a certain extension by default?  For example, how would I make it so that adobe reader opens a pdf file instead of evince when I double click on the file, or vice versa?

---

### Post by jimmy the saint on 2008-05-17
right click on the item > Properties > Open with tab > select program.
