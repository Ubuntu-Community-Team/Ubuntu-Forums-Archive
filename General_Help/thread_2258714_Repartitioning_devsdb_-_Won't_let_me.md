### Post by Dennis N on 2014-12-29
To remove all the existing partitions from /dev/sdb, I would just create a new partition table on it.

Select /dev/sdb in the disk selector in upper right of gparted. then in the menu, **Device > Create New Partition Table**. After completing, the disk will then show as unallocated. Select the unallocated area, then from the menu: **Partition > New** and enter the details of type and size of your new partition, and apply.
