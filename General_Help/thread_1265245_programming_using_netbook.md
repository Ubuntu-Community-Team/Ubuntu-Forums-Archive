### Post by The Cog on 2009-09-13
In engineering terms there's no difference. Netbook Remix has a slightly different windoe manager - all applications are always full screen. But that's it.

In terms of practical programming, the only difference then is that programming on such a small screen can be hard on the eyes. And you can't get so many lines of code on the screen in the editor. It actually does make harder when you can't see an entire block of code without scrolling up and down.
