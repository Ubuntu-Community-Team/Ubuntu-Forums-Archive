### Post by rhoparkour on 2010-09-27
Hey there man!

It depends on what you want to write your srcipts in... the easiest thing to learn is bash scripting, the "native ubuntu language," since ubuntu uses bash. It is basically what you write in a terminal (Applications > Accessories > Terminal).

Some easy languages to write scripts in are python and perl. For hard calculations I like fortran90, but most prefer some flavor of C.

Whatever it is you choose to learn, you should know how to use emacs or vim (google them, I don't have tuts for these). For built in tutorial in vim, type vimtutor in a terminal.

Let me point to some tutorials!

Bash (a good one takes some patient googling but this is the one I used, I am happy to see that is now the first one in a google search!):
[http://tldp.org/HOWTO/Bash-Prog-Intro-HOWTO.html](http://tldp.org/HOWTO/Bash-Prog-Intro-HOWTO.html)

Python: 
[http://www.python.org/doc/tut/](http://www.python.org/doc/tut/)

Perl:
[http://www.perl.org/books/library.html](http://www.perl.org/books/library.html)

Also, look at this xkcd thread for useful links and comments:

[http://forums.xkcd.com/viewtopic.php?f=11&t=13849](http://forums.xkcd.com/viewtopic.php?f=11&t=13849)

You'll find in general that people respond well to questions about this kind of stuff, at least here in the Ubuntu forums and the xkcd forums.

Good luck man.
