### Post by wpshooter on 2009-05-31
I would not wait for a CD.

I would download the CD from Ubuntu website and burn it to a CD to do the installation.  You might want to consider downloading both the live/desktop version and also the ALTERNATE (text based) CD version in case you run into any problems with doing this installation from the live version.  

Do NOT forget to check the integrity of the Ubuntu CD by running the verification function which is found on the initial Ubuntu boot screen menu.

Good luck.
