### Post by ajgreeny on 2011-02-28
Do you already have another OS on the computer?  If not, by default the grub menu is hidden; only if there is already another OS will it show.  You can, however, make it show by pressing shift as soon as, or preferably just before you see a message saying starting grub (or something to that effect).

See the *grub2* links in my signature for all the details.
