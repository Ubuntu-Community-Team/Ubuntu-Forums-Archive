### Post by fluffman86 on 2009-11-25

### Post by nothingspecial on 2009-11-25
I use crunchbang on my main pc. It uses the same repos as Ubuntu so all the software available for ubuntu is the same.

Some of the default apps are different, for example abiword instead of open office, or terminator instead of gnome-terminal but you can always install the stuff you are used to.

The main difference is openbox instead of gnome. You`ll need a bit of time to learn how to configure it, but once you do its easy.

It is deffinately faster on my machine.
