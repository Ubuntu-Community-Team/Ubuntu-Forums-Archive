### Post by Dngrsone on 2012-01-15
I concur.  In answer to your question, though-- if you ever want to reinstall Win 7, then you should be able to do so using the recovery disc(s).  For most commercially-built systems, however, that means that the system would be restored back to the 'factory state'; in other words, the software and hard drive configuration you had when you first purchased the machine.

Partitioning your drive and installing Ubuntu alongside Windows allows you to go back to Win on those few occasions you actually require a Windows OS, without running a large risk of screwing up all your settings and data.

### Post by Mark Phelps on 2012-01-15
Whether or not you can completely restore Windows using the disks depends on whether they are full-restore disks, or only a boot disk.

The former contains all the SW needed to restore Windows; the latter only boots you into WindowsPE and needs the windows restore image stored in a Recovery partition on your drive in order to do the actual restore.

If you don't know which you have, you should check with where you bought your PC to find out.
