### Post by sanderd17 on 2010-12-16
I have the Dutch  version of OOo, so the translations won't be literal, but I guess you'll know where to go.

go in the menu to "markup"->"styles and markup". Select the page icon from the top row. There you should have a little list with different markups for pages. One of those should be "first page".

Right click on "first page" and select "edit". There do whatever you want that will apply to the first page. Then do the same for "default" (apply the footer for this, this might already be the case).

Then go back to your document. Be sure your fist page ends with a page break (ctrl+enter creates a normal page break). If you want the number to be reset, use the thing as described on the link to restart the numbering.

Now apply the markup profiles to your pages:

put your cursor on the first page, there you should see  the text "page 1/x" at the bottom bar of your window (not in your document) and next to that the name of the style used (in this case it's default). Right click on "default" and select the first page out of the list. Now your footer should have dissapeared on the first page, but not on the others.
