### Post by roccity1 on 2010-09-24
If you click on edit-->preferences and then edit again on the right in Evolution. Click on receiving options and is use secure connection set to SSL Encryption and remember password ticked? If these are set like this remove the tick for remember password and try clicking on send and receive again. It should ask for the password. It maybe that Evolution is saving a bad password.
