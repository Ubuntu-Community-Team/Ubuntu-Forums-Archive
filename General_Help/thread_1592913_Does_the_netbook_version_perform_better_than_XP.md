### Post by roggenschrotbrot on 2010-10-11
in this case you should be fine. you can test ubuntu before installing, though it might be a bit slower this way as it has to run from ram, it should give you an impression on if you like it or not. if unshure you can allways keep your xp installation and install in dual-boot mode. this way you can remove or ignore ubuntu if you aren't happy with it.

i regulary setup linux-only netbooks for relatives and friends, and none of them intends to switch back, but it realy depends on what you expect. if you insist on using a special piece of software way to do things you might not be happy with what ubuntu has to offer, but especially on netbooks it is worth a look imo, especially if you keep in mind that you can change pretty much every part of your desktop experience by modifying your installation to your liking, be it performance, layout, eye-candy or features.
