### Post by Max Blyss on 2012-05-05
Cairo dock will kind of run as its own desktop...

If you go into synaptic and install cairo dock (or software ctr...), when you log out and log back in you will have cairo dock session options.  It should work right away, and uses compiz.  I've been using the Cairo Dock Gnome + Effects session and am loving it.  Using 13% of 3G of ram as I type with other apps on so it's light enough.

Also, you should go get compiz config setting manager to config compiz.
