### Post by drmrgd on 2012-07-31
> **joco1500 said:**
> Thanks man
the unix command line is a lot harder than windows

Once you get accustomed to it, I don't think you'll still feel that way.  If you're comfortable with DOS commands, you'll pick up Linux Commands very quickly.  Heck, even some of them are the same.  For example, if you want to change directories in DOS, type'cd'.  If you want to do that in Linux, type 'cd'.  If you want to see the directory listing in DOS, it's 'dir'.  In fact, 'dir' will work in BASH, but the preferred directory list command is 'ls'.  Many of the basic ones are the same or at least very intuitive.  

Definitely check out the links that others have provided here and experiment.  Once you see how easy it is, and how much more you can do, I think you'll forget how comfortable you ever were in DOS.  These days, whenever I have to do something in Windows cmd, I get frustrated because I feel like I can't do quite as much as I can with the Linux command line, and I feel like DOS is somehow less intuitive.  Just my personal opinion of course!
