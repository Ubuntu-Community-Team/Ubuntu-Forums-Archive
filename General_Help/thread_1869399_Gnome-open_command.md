### Post by Orwell_T89 on 2011-10-29
I have found a solution for the problem I was having with opening my folders. I had somehow managed to set the default action for folders to be to launch a terminal, hence I couldn't open the folders directly or with gnome-open unless I used the nautilus command which corresponds the file manager I use. Fortunately for me I have gnome as well as kde installed on my ubuntu therefore I when I logged in via kde, the cause of the problem became obvious and I right clicked on one of the folders and selected open with then added the command nautilus. So that problem should be fixed now.
