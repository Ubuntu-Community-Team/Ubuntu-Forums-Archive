### Post by SickTwist on 2006-05-25
In no particular order:

ls - list files
cp - copy files
mv - move files
rm - remove (delete) files
mkdir - create a directory
find - search for files
tar - work with archives
grep - look for a pattern in a file or the output of another command
cat - display the contents of a file
less - display the contents of a file longer than the screen
chmod - change permissions
chown - change owner/group
man - manual pages for in-depth information about commands
free - memory usage
du - disk space usage
df - disk space available
mount - attach filesystem to a mountpoint on the tree
umount - detach filesystem from the tree
apt-get - update, install, and remove packages
apt-cache - search packages in repositories
sudo - run a command as root
su - switch to another user
vi - file editor
nano - another file editor with a more simple interface

Hopefully that will get you started. Also, check out [www.linuxcommand.org]("http://www.linuxcommand.org/") as it is a good place for beginners.
