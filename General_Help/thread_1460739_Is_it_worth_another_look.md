### Post by VeeDubb on 2010-04-24
I'm not interested in "selling it to you" but I will share my experience in switching to linux.

The very first time I tried linux, was when knoppix was still in beta.

Knopix, for those who don't know, is a live linux distribution, that was one of the first "live linux CD" distributions.  It was really intended to be run entirely from the live CD, but did have an option to install to the hard drive.

After a few weeks of playing with it, I installed, and then used that installation to upgrade to whichever version of debian has the 'new' stuff, rather than debian stable.  It took me about 3 weeks to completely destroy my system, I never got a lot of critical hardware and software to work correctly, and when I formatted the old hard drive, I was to reinstalled WindowsMe.

Later that year I tried Mandrake (later renamed to Mandriva due to copyright issues) and fell in love with the system.  I still managed to destroy my system requiring a reinstall every month or so, but it was a lot more fun in between.  Eventually, XP took me back.

This cycle repeated several times over, with various distros, and each time I found I stayed with linux longer, I kept my system running properly longer, got more critical hardware and software to play nice, and found more linux alternatives to other programs.

Other than a brief 3 month trial when Vista came out, I've been a full time linux user since about 2 or 3 years after I first booted knoppix.

With true 64bit flash and java (not available until recently) I never find my linux desktop lacking in terms of online media.  Cedega and Wine allow me to play plenty of games.  Commercial linux apps like BibblePro allow me to keep my photography business trucking along without windows.

We are now, for all intents and purposes, at 10.04, not 9.10.  The RC is out, and the final stable release is due in a matter of days.  If you want to try Ubuntu again, and you're not comfortable installing an RC, then I suggest waiting until the 29th.  No point in installing now, just to upgrade in 5 days.

8.10 - 10.04

3 versions, 18 months, a lifetime in the advancement of linux.

In short, if you tried linux, and just didn't like it, you probably won't like it now.  But, if you tried linux, and you liked a lot of things, but were unable to make it work for you 18 months ago, now, at the release of a new "LTS" version, is probably the best time there is.

### Post by bcz on 2010-04-24
I have used Unix and then Linux since about 1990, and run Windows only so I can support those customers who want to use it.

A few years back, after using Ubuntu for a while with no problems, I upgraded to a new release and had stability problems.  I was unable to fix them, and switched to using Fedora, and had no problems.  After more than a year, I switched back to Ubuntu (9.04 I think) and have had no problem since.

I am certain that my problems were caused by a bug with the ATI drivers, and that using a different VGA card would have resolved the problems.

I recently helped a friend run a Windows program under Linux.  (He was running Mint 8 distribution, which is Ubuntu based).  Mint 8 included Wine 1.0.  The program required version 1.1.20, and the latest available Ubuntu version is 1.1.42.  I loaded the latest version and it all ran fine.  (Version numbers are from memory, but the principles are correct)

Although one hears of Windows programs running faster under Linux than Windows, linux programs will run better under linux than equivalent Windows programs.

Always be conservative if upgrading a commercial system.  Setup a new system and check it works properly before cutting over from the current working system
