I have Ubuntu 19.10 installed on my laptop to dual boot with Windows 10. I've found that Ubuntu is a lot faster than Windows 10 on my laptop, but I have some major stability issues when transferring files or when doing something that involves high file usage.

For example, today I tried two times to transfer ~20GB of files to a USB stick. Randomly during the transfer the UI will completely freeze up for a second or two and then go back to normal. But, if I'm doing something else on the computer at the same time, like watching a video, it will completely freeze up. If it's a video, the audio will continue going and will stutter ever once in a while. Then, after a few minutes, the audio will get stuck in a loop like a broken record. I need to do a hard shutdown to get the system working again.

Using the System Monitor, I can see that the RAM usage stays fairly stable at around 2-3GB of usage (I have 4GB total) during the transfer. The CPU doesn't spike up either. My assumption is that it has something to do with high disk usage tasks. I don't have any issues like this when transferring files on Windows 10.

Are there any settings or system files that could be tweaked to prevent this issue? It makes using Ubuntu as the main operating system less and less practical.
