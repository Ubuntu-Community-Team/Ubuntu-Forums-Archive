### Post by QIII on 2015-02-11
Please do not bump your posts so quickly.  It is taken as exceptionally rude.

We are all volunteers here from all around the world, so your questions will be answered as quickly as the person with the right answer sees it.

Be patient and do not demand that your questions be answered immediately.

You have every right to use Ubuntu on any hardware you choose.  Certification only means that someone has tested a piece of hardware and confirmed that Ubuntu works on it.

Wubi is no longer under active development.  When it was, it would not work in Windows UEFI installations.  Therefore its use is no longer recommended.  Canonical decides which packages are still provided.  None of us here has any official say in the matter.

### Post by deadflowr on 2015-02-11
> **safraoumohamed said:**
> why does wubi exist in ubuntu 14 04
 
Because it still works with Windows 7.
Under development and being supported are two different things.
wubi is no longer being developed, so it is not design to work with the latest releases of Windows(8+).
But it is still, currently, supported to work on those Windows releases which it was developed for.
If that makes sense.


Also, while I'm at it, Ubuntu Hardware Certification means that a machine listed as such will run Ubuntu without the need to do anything other than install it.
Which means you won't need to download extra drivers, or fudge some configuration file to get it to work.
It'll run out of the box, as is.
