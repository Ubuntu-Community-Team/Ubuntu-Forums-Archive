### Post by iMac71 on 2012-12-27
First of all, you have to install the gnome panel:```
sudo apt-get install -y gnome-panel
```(you can do the same in the Ubuntu Software Center by typing "gnome-panel" without quotes in the search field).
After the installation, log out and from the login menu choose "Gnome classic" or "Gnome classic (no effects)".
That's all.
