### Post by Tamlynmac on 2011-01-17
Did you try and install 9.10 from the web or with a CD? I'd strongly recommend only upgrading from either a CD or my preferred choice - a mem stick (if possible). Web upgrading is risky and rarely have I had any success doing so.

If you used a CD, burn it at the slowest speed possible (I use K3b) and check for errors upon completion. I've experienced numerous CD burn failures (in the past) before getting a good burn. That's why my upgrade choice is now a mem stick.

I'd strongly suggest you start over by downloading the version .iso you wish to run and either burning a disk or using a mem stick. IMHO, no upgrades should be done from the update manager via the web.

One last thought, you might wish to test run the version prior to install, assuring video, sound, net, etc. all work. This may save some time should your system not be compatible with a specific version. I actually use a spare HDD to test all new versions prior to install. That way if a version fails (for any reason) I don't install it on my primary HDD and simply reboot unplugging the spare HDD - No harm, No foul.

Good Luck & Hope This Helps.
