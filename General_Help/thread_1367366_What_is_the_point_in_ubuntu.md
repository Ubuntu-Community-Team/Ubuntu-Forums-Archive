Linux is free, this does not always mean free as in for free - meaning you don't have to pay for it. No it is free, cause the source code of the programs (most of them anyway) is free. You can change it as you want it and then offer it back to the world so others work with your product.
With Windows you get whatever MS wants you to have, you may use it but will never be the owner. With Linux you are also not the owner because in Linuxland people hate ownership and dictatorship. ### Post by murphykieran on 2009-12-29
None of my day-to-day computer usage actually requires Windows - web browsing, email, listening to music, basic word processing, managing my photographs, etc.

And because I built my PC myself, I'd have to either engage in software piracy or pay money to use Windows and I like getting stuff for free legally.

And having just done a major upgrade and reinstallation on my computer, I can tell you that Ubuntu is generally a pleasure to install compared with the Windows XP Internet treasure hunt usually required to get networking/sound/USB/graphics driver, install latest service packs, install anti-virus, install a codec pack, install MS Office or download OpenOffice, install Firefox/Chrome etc. before you have a Windows system that meets your requirements.  Typically I need a second computer ready because ethernet and wireless won't work with a fresh Windows XP install so you need another PC with an Internet connection to hunt down your drivers.  Yeah, if you've got your drivers discs then you're ok, but in my experience people never have them.

On the other hand, when I installed Ubuntu all I had to do was activate the restricted graphics driver, install restricted extras for media and codecs, install Chrome as my browser, run system updates, reboot the system and I was ready to go. 

Your mileage may vary.  Windows Vista and Windows 7 might be a more pleasant installation experience but I've no experience with installing them and I had trouble getting wireless on my laptop working in Ubuntu, so not everything is perfect.  And if you're a PC gamer, then you're better off going with Windows.

I suppose it all depends on your needs and expectations, but plenty of us are satisfied with Ubuntu and find it an overall superior experience to Windows.
