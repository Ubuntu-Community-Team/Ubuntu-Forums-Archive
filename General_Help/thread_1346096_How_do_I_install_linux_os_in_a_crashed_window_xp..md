### Post by IllegalCharacter on 2009-12-04
Grab the Ubuntu image from Ubuntu's page: [http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)
Put that on a CD (you need a burning program that supports burning CD images aka .iso files).
Stick it in your friends computer, make sure to enable booting from CD (usually you press Del when starting up to go into the settings).
Follow the instructions, it's pretty straightforward.

### Post by StuartN on 2009-12-04
> **japlim said:**
> My cousin had this Window xp home-edition that crashed recently. He can't repair without a recovery CD. Now I wanted him to install Ubuntu Linux OS in his computer. I'm new to this, how can he reformat his computer and install Linux OS from a downloaded CD or DVD.

If your friend has no data in the XP partition then just follow the prompts to use the entire disk. All data on the disk will be permanently erased.

If he wants to rescue any data then you will have to find a recovery disc and check the filesystem using Windows first, because it is not safe to resize a partition that has not been cleanly shut down.
