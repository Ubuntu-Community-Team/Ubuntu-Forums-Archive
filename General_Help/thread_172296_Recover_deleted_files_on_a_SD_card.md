### Post by the_tiger on 2006-05-08
Testdisk is available in the universe repository. It has been specifically written to recover partitions and files from camera memory. This package also installs photorec which is the program you might like to use.

In windows you can try this program

[http://www.pcinspector.de/file_recovery/uk/download.htm](http://www.pcinspector.de/file_recovery/uk/download.htm)

it is freeware.
