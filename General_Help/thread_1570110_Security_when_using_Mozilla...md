### Post by coffeecat on 2010-09-07
I presume what you are describing is what you see in Internet Explorer when you visit a https site. The security around https has nothing much to do with the operating system you are using nor the browser. You are used to IE. Firefox presents things a bit differently, and Firefox in Windows would be much the same.

In Firefox, when you visit a https site, you'll see a padlock icon in the bottom bar towards the right. If you click on that icon, a small window opens with all the security details.

As far as online banking is concerned, I feel much safer using Linux. I will only do my online backing using Ubuntu - definitely not Windows.

By the way. Wikipedia article on https:
