### Post by synapsys on 2009-06-09
You could probably spend hours tracking this down to some sort of driver problem or xwindows problem. If it were me, and a live cd worked flawlessly, I would just install regular old Ubuntu and then install the programs I needed from there. As far as I know Ubuntu Studio is just basically Ubuntu with a bunch of multi-media programs pre-installed on it. The problem with pre-configured programs is that they are less flexible. I have installed Ubuntu on many many computers and it has never frozen on me. Except one computer which had bad ram.
