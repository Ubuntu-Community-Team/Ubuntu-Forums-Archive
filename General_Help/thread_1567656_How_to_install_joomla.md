### Post by JedMeister on 2010-09-04
If Joomla 1.5.15 is new enough version of Joomla for you then I don't think you can go past [Turnkey Linux (TKL) Joomla]("http://www.turnkeylinux.org/joomla") appliance. 

TKL is a project I'm quite involved with and they produce ready to run appliances with all the components preinstalled and preconfigured. You can either download them as a VM image (to run inside VMware or VirtualBox) or as an ISO (install to pretty much any virtualised environment, or burn to CD and install on bare metal). The ISO will even run as a live CD!

The current stable releases are built on Ubuntu Server 8.04 so if you are familiar with Ubuntu then you should find the commandline of TKL a breeze. They have heaps of other great features too such as preconfigured SSH, SFTP and Webmin. All for free! Nice! Oh and did I mention the forums are very friendly too?
