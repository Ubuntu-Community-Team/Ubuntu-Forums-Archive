### Post by DFlame on 2008-10-23
I regularly control windows based machines through LogMeIn using Ubuntu 8.04 and Firefox 3.*

If you install the ubuntu-restricted-extras package, Sun Java is included and Firefox will automatically add the plugin next time it is started. There was an issue with previous versions of Ubuntu/FF that required you to add the plugin manually, but this is now not the case.

Have fun :)
