### Post by janbockaert on 2007-07-02
First, i would like to know if you didn't do something wrong with installing the driver. Installing drivers for a not so new hp printer should not be necessary. Can you boot the live cd, and try printing from that cd? go to system, managment, printers, ad printer. Does the wizard finds a printer? Does the printer works?

I have no experience with vlc under linux. But i do have not so good experiences with gstreamer, the audio-backend ubuntu uses as standard. I changed that to xine. If you would like to do so. Open Synaptic, search for xine and select every packet that seems usefull. (i have totem-xine, libxine-kde, libxine-gnome, libxine-extracodecs, libxine-ffmpg, libxine-console, libxine1, libxine1-plugins). After you install these, it should be possible to play wma files without vlc.
