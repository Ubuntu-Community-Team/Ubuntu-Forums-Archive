### Post by cros13 on 2009-01-12
Hi macubuntu,

"i386" means you have the 32bit version I'm afraid.

For first time users I think that 32-bit is usually best to start off on.

There is very little benefit for general use in terms of performance from using a 64-bit version of any operating system.
The exception of course is if you do something that requires a more than 4GB of memory.

It also introduces issues with a few pieces of software.
You miss out on Boxee([www.boxee.tv](www.boxee.tv)) for one thing :(

Personally I keep both ubuntu versions dual-booting on my system.
A 32-bit for general use and a 64-bit for running Photoshop CS4 in wine.

On Partitioning:

The Installer can partition the drive including resizing your existing partitions.
