### Post by Bucky Ball on 2016-01-15
You can't install from disk while still 'booted up' (if you mean you are currently running an operating system). You need to reboot and hit a key at the manufacturer's splash screen immediately after boot to get to BIOS. That key depends on the model of computer. Sometimes del, sometimes F1 or F2. You need to check the motherboard or machine manual.

Many computers allow you to select a boot device without entering BIOS by hitting F12 at boot. Again, check the manual if it doesn't work.

But you seem to have been able to boot from the 14.04 install media because you state you did an upgrade from it, which only installed a couple of things. Bit confused. Please clarify. You are wanting to upgrade 12.04 to 14.04 via a net upgrade but can't log in, or you want to install or upgrade from install media but you can't boot from the disk or USB ... or both? ;)

PS: Don't think a guest has the permissions to update/upgrade the system. If you have reset the password, why do you need to login as guest?
