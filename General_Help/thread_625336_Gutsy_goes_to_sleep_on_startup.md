### Post by rsambuca on 2007-11-28
Put a number sign in front of this line:

Virtual 1400 1050

so that it looks like this:

#Virtual 1400 1050

You can edit the file using gedit, the text editor:```
gksudo gedit /etc/X11/xorg.conf
```After you have added the number sign, save the file, close gedit and any other programs you have open, and then see if it fixes the size issue upon a restart.
