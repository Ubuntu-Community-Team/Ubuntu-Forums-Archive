### Post by grahammechanical on 2016-04-11
What is the OS on that machine? Windows does not read Linux partitions. At least not without installing something. As far as I know, that is. Is the drive recognised and listed in the computer boot system (BIOS/UEFI) settings utility?

If by accessing the drive you mean loading Ubuntu from it, then you may be in a situation where Ubuntu was installed in BIOS mode and Windows on the new laptop was installed in EFI mode. And that does complicate things. If the old machine had a BIOS boot system & the new machine has a UEFI boot system then you would be in that situation.
