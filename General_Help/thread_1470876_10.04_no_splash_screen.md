### Post by suffer1989 on 2010-05-03
Hey, I upgraded to Ubuntu 10.04 from ubuntu 8.10, and mostly everything has been going well. However , on my old ubuntu 8.10, the bood loader didnt run, and instead has some text, showing the startup process, which was fine for me. However, after upgrading to 10.04, the screen remains blank throughout the entire boot process, and it only lights up once the desktop loads up. Is there any way to repair the bootscreen ?

EDIT : As seen on page 2 :

> **pcdave98 said:**
> I had the same problem - just a black screen with blinking cursor between GRUB and login screen and no Ubuntu splash.

This fixed it for me:

```
echo FRAMEBUFFER=y | sudo tee /etc/initramfs-tools/conf.d/splash
sudo update-initramfs -u 
```I found it in a couple of forum posts.

[http://ubuntuforums.org/showpost.php?p=9200675&postcount=9](http://ubuntuforums.org/showpost.php?p=9200675&postcount=9)

### Post by dino99 on 2010-05-03
do you mean: no grub menu ?

with muti-boot the menu is shown, but if you have only ubuntu then grub menu is hidden by default. You need to hold "shift" key down at the end of bios process to see it.

if not:
sudo dpkg --configure -a
sudo dpkg-reconfigure gdm
sudo dpkg-reconfigure plymouth

note: grub-legacy have to be removed/purged and menu.lst too, if still there, after purging:

sudo grub-mkconfig && update-grub

### Post by pcdave98 on 2010-05-04
I had the same problem - just a black screen with blinking cursor between GRUB and login screen and no Ubuntu splash.

This fixed it for me:

```
echo FRAMEBUFFER=y | sudo tee /etc/initramfs-tools/conf.d/splash
sudo update-initramfs -u 
```I found it in a couple of forum posts.

  <edit> Here's another way of doing the same thing - and also how to revert if it doesn't work for you.

[http://ubuntuforums.org/showpost.php?p=9200675&postcount=9](http://ubuntuforums.org/showpost.php?p=9200675&postcount=9)

### Post by PoliticKiller on 2010-05-05
Hi.

This solution works perfect for mi: 

Install startupmanager from repository  Go to System -> Administration -> StartUp-Manager. 

Wait a few seconds for the program to start, and then choose 1024x768 and 24 bits in the "Resolution" and "Color depth" drop-down menus. Also, check the "Show boot splash" option, and you can reduce the "Timeout in seconds" option to 0 (zero)! 

Close the application and wait a few seconds to apply the changes. 

Reboot your computer! 

You can play a little with the resolution, as 1024x768 may not work for everyone!

 I hope this can help everyone.
