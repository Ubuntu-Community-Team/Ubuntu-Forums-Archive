### Post by binarysys on 2008-10-02
Some how in my tinkering I have enabled hidden files to be shown when opening through an application. For example if I open up gedit and browse to a document I can see my .svn files. But when I browse through nautilus they are hidden as they should be unless I select them to be shown. 

How do I turn this feature off again?

---

### Post by _sAm_ on 2008-10-02
Try Ctrl + H when you see the files.

---

### Post by binarysys on 2008-10-02
Ah man that was too easy. Thanks. I knew about ctrl-h when looking through nautilis but didn't think to try it inside an application.
