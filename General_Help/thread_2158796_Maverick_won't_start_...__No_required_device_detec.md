### Post by Cheesemill on 2013-06-30
Ubuntu 10.10 reached end-of-life in April 2012 and is no longer supported. As such the repositories have been taken off-line and it no longer receives any security updates or bug fixes.

Although you a free to attempt to keep using it if you wish you're going to find it very difficult to get an answer to your question as no-one has been using it for over a year now, so all of the information relevant to 10.10 will have been mostly forgotten.

I highly recomend backing up any important data and doing a fresh installation of a version of Ubuntu that is still supported such as 12.04 or 13.04.
