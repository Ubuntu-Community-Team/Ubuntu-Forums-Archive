### Post by TrinitronX on 2005-12-17
well... you could always try to use tty[1,2,3...etc] to login and start another x session... I've never tried it myself.. but it could work.  Just press ctrl+alt+f1 to get to tty1.  login and then try using either 'startx' or 'gdm' or 'kdm'... whatever you're using :p.  To get back to your original one, just use ctrl+alt+f7.  I think usually ctrl+alt+f[1-6] give you the different terminal logins, while f7 should be the default one used for ubuntu.
