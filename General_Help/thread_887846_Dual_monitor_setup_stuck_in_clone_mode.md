### Post by rossjman1 on 2008-08-12
I have an ATI Mobility Radeon x1400, and if you go into the ATI Catalyst Control Center you can change from clone screen to big desktop. However, you have to do this everytime you log in.
```
sudo apt-get install amdcccle
```
Get to it through Applications > Other
