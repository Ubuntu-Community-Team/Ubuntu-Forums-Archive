### Post by funky_uncle on 2009-02-05
I just installed Ubuntu 10, and Firefox came with it. I've been using Firefox 3 before, and I always used backspace to go back one page. Suddenly this doesn't seem to work, and on Mozilla's own pages, it's listed as a keyboard shortcut in the Windows version, but not in the Linux version.

The Mozilla pages made me none the wiser as to how I may get this function back. Anybody here have an idea?

---

### Post by hwttdz on 2009-02-06
If you navigate to the address about:config in the firefox address bar.  Then search for "browser.backspace_action" and set the value to 0.  You should be good to go.
