### Post by audiomick on 2009-12-07
Hallo.
I have had an Ubuntu Studio on my computer for a while now, even though I don't use the "studio" aspect all that intensively.

Mine is a 64 bit, running on an Intel Core 2 Duo 3.16 GHz with 4 GB RAM

One of the differences was the kernel: Studio had a RT (real time) kernel until 9.04. This is desirable for short latencies for sound work.
The Kernel that is installed at the moment is a 2.6.31-16- generic (there was a Kernel update last night ). I don't know why it is not the RT kernel at the moment, but as far as I know, the intention is to re-include it when  it becomes possible, whatever the problem is / was.

There is this:
[https://help.ubuntu.com/community/What%20Is%20Ubuntu%20Studio](https://help.ubuntu.com/community/What%20Is%20Ubuntu%20Studio)
which was updated according to the footnote, in November this year, and still refers to the RT patch. Maybe it should still be there, and my update lost it for me.

Anyway, the main thing for me is, you install Ubuntu Studio, it works great, for me at least, and discover applications you didn't even know existed. It is really pretty much "all there" out of the box.
