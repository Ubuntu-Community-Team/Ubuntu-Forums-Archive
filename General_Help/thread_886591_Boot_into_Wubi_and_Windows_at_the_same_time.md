Wubi is just a Windows .exe used to install Ubuntu inside a Windows partition. Once Ubuntu is installed you no longer use wubi. You are given the choice to boot either Windows or Ubuntu. You can not boot and run Windows and Ubuntu at the same time. QEMU is a processor emulator that allows you to run a guest OS inside a host virtual machine, similar to Virtual PC or VMware.

Running both OS native with live switching is not possible to my knowledge.
