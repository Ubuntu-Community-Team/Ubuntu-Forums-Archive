### Post by ajgreeny on 2015-09-05
I do not run 15.04 but if there was a new kernel in your updates try booting to an earlier kernel version from the grub menu.

If you don't see the grub menu hold Shift at boot, and then choose Advanced and the kernel version prior to the update.  Alternatively you may simply need to reinstall a proprietary graphic driver if you use one normally, but we will need to know more about the hardware you have to tell you more.
