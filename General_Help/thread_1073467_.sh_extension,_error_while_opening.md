### Post by heindsight on 2009-02-18
given that the output from that netbeans installer script you posted said that it was "Searching for JVM on the system", I expect that it requires you to have java installed already before you install netbeans.

As I mentioned before, to run the script and see what output it produces before exiting, open a terminal and do:
```
cd /home/nemiux/Desktop
./netbeans-6.5-ml-linux.sh
```

But I would definitely recommend that you install netbeans from the ubuntu repositories instead. Go to System>Administration>Software Sources and make sure you have the 'Community-maintained Open Source Software (universe)' option checked. Then open a terminal and run the command:
```
sudo apt-get install netbeans
```
This will install netbeans as well as any packages that netbeans may depend on. Much easier than mucking about with an installer script, for which you have to manually find and install all the dependencies. Using this method also makes it a lot easier to uninstall netbeans if you no longer want it, by doing ```
sudo apt-get remove netbeans
```

If apt-get complains about not being able to get a lock, then it means some other package manager is running. Close all package managers and try again, if you didn't have any open, then it may be the update manager checking for new updates, in which case you can just wait a couple of minutes for it to finish and try again.
