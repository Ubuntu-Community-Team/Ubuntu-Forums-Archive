### Post by Zorgoth on 2011-08-30
cairo-dock is by far the best dock for Ubuntu if you are used to the Mac dock.  It is highly configurable and has OS X themes if you would like that, as well as many other themes.  It certainly could be configured to do something like what you want, with similar graphics, but I haven't been using it since unity came out and I don't remember how.  Ask here or on the cairo-dock forums.
