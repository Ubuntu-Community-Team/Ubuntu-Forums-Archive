### Post by Don9307 on 2007-07-21
I have Windows XP Professional SP-2 running in a virtual box (on a Maxtor 3200 USB Hard Drive) with Ubuntu 7.04 as the host OS with no issues at all.  In fact, I have a Shared Folder set up on my Ubuntu desktop which is mapped to drive F:\ on my WinXP virtual box so I can share files between these two simultaneously running OSes.   This was all accomplished using Innotek VirtualBox ( [www.innotek.com](www.innotek.com) ).   I keep the VM paused and minimized on my Ubuntu system until needed.   The experience is about as near native as you can get.   I found that there were just some MS Windows software packages that I absolutely couldn't live without, and Wine wouldn't even come close to running them:  for example, MS Money and MSN email software packages.
