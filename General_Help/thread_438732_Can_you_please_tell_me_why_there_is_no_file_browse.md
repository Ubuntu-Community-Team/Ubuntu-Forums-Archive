### Post by NikoC on 2007-05-10
Do you mean by a file browser something like Nautilus?

Alt + F2 and type

```
nautilus
``` 

in the appearing box.

If this is a solution you can always add it to the menu by typing ALT + F2

```
alacarte
```

And then making a new menu entry under any folder you want...

Cheers

ps. Nautilus is default for Gnome but several other options are available (e.g. Thunar)
