### Post by Rubi1200 on 2010-09-21
Ok, you can reinstall GRUB but you need to use an Ubuntu CD that has GRUB2 on it or it will not work. Version 9.10 also uses GRUB2 so maybe try burning that to CD.

Do you still have the pen drive with 10.04 on it?

If yes, you could boot the computer choosing to try Ubuntu live and then reinstall GRUB.

Here is the guide for how to do this:
[https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD](https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD)

It says from CD, but from a USB stick works just as well.

If you are unsure about anything, please ask first.
