### Post by taurus on 2009-01-21
Assuming the file is called java.bin and you've saved it to your desktop.  Open a terminal and do

```
cd ~/Desktop
chmod +x java.bin
sudo mv java.bin /opt
cd /opt
sudo ./java.bin
```
Now, it will unpack Sun's java in /opt.  There should be a new directory.  Again, it depends on what version you have downloaded so for now, I would call it jre_directory.

```
cd jre_directory/bin
sudo ln -s /opt/jre_directory/bin/java /usr/local/bin/java
```
In /opt/jre_directory, there should be a plugin directory which you need to either link it to your ~/.mozilla/plugins or copy it to your ~/.mozilla/plugins.
