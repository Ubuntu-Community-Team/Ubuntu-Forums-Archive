### Post by BslBryan on 2009-06-07
Hola, JuanCarlos92. :-)

Ir a Applicaciones --> Accesorios --> Terminal abrir el Terminal.

En el interprete de comandos, haz
```
sudo apt-get install envyng-qt
```

Próximo, ir a Applicaciones --> Herramientas de Sistema (System Tools) --> Envy

Obedece las órdenes de Envy, y es así de sencillo. :-)

Espero que he ayudado.
