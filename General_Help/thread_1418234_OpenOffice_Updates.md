### Post by labinnsw on 2010-03-07
> **Chamillionaire2 said:**
> Is it not possible to tell update manager I'm not interested in any more updates for openoffice.

Yes it is.

Go to System >> Administration >> Software sources
Under the "Third-Party Software" tab, uncheck the openoffice-pkgs repositories.
To remove them completely, highlight them and click the "Remove" button at the bottom.
