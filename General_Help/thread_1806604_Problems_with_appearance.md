### Post by claracc on 2011-07-18
About number one question:

If you are using gnome desktop, you can run in xterm gconf-editor, then go to apps -->metacity --> general--> button layout, and there, write menu:minimize,maximize,close. The change of the buttons to the right side is inmediate.

If you want the buttons in the left, you will write in button layout menu:maximize,minimize,close.
