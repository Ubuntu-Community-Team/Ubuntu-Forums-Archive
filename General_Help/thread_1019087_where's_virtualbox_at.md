### Post by drs305 on 2008-12-22
And if you are running from terminal, spell it VirtualBox for the run command and virtualbox for the other folders/files.
```

*which VirtualBox*
/usr/bin/VirtualBox
*whereis virtualbox*
virtualbox: /usr/lib/virtualbox /usr/lib64/virtualbox /usr/share/virtualbox

```
