### Post by MacDuff on 2008-01-26
Been there, done that, even got a T-shirt!

I experienced everything you are presently going through.  We "converts" from Windows have a major handicap, even those of us with some experience with Unix.  There are so many subtle (and not so subtle)  differences that when we do something in Linux that makes sense to us, given our experience, it doesn't work.  I can only tell you not to try to do too much at once.  It takes time to learn Ubuntu and it is less frustrating if you work on one thing at a time rather than trying to master everything in a week.  I have been playing with it for several months now and I still have frustration at times.

However it is worth it if you have the time to spend.  I am still forced to use Windows for some tasks but most of my work is done in Ubuntu and I love it.   Switching back and forth between Windows and Ubuntu machines has really shown me how much trouble I had with Windows, even though I knew it inside and out.  The Ubuntu machines are very forgiving and even though I don't fully know what I am doing with them, they almost never freeze and require re-booting.

If you have the time to play with Ubuntu, please stay with it.  I think that as you learn more about it you to will come to love it.   Remember that it probably took months or even years before you were fully versed in the vagaries of DOS and various Windows operating systems. 

The support on the forums, I have found to be superb.   But not all responses will be easy to understand.  Many responses will be by users who seem to prefer terminal input and so  require the use of the terminal.  It seems that the graphical user interface is relatively new to Linux users but better support is now apparent for those of us who want to use a GUI.  Some responses may be curt but don't be put off.  If you are not getting the information you need, try asking it in a different way.  Don't be shy about admitting how little you know.  

Each new release of Ubuntu  has been noticeably easier to use and supports more hardware.  Stay on board and find out how much fun it can be.
