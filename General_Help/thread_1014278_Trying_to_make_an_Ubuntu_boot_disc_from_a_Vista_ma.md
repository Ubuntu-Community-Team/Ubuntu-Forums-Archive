### Post by Powerman2442 on 2008-12-18
All you need to do is download the .iso file for the distro you want. Then, in Windows XP/Vista download some burning software that burns Image CDs or Bootable ISO CDs. (Check google or download.com) I know there are at least 6 different pograms that do this (that I've used). Currently the only one I use is the free version of Ashampoo Burning Suite, or something to that effect. You should be able to find it on download.com. 

Also, depending on the iso image you download you may have to use a CD or DVD. Ubuntu and most other distros tell you if you are downloading an iso for a DVD or CD. I've always done the CD version because it is not as large of a download. You can use whatever program you feel better using. Everyone is going to have their own program they prefer.

Good luck,
