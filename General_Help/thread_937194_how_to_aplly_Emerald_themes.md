### Post by howefield on 2008-10-03
press ctrl and f2 keys together and type the following into the window that pops up.

```
emerald --replace
```

You can add the command to your sessions list so that it is run when you boot up your computer.

System > Preferences > Sessions menu and press the add button, in the name field type Emerald, in the command field type emerald --replace and in the comments field type anything you want or leave it clear. Then ok your way back out.
