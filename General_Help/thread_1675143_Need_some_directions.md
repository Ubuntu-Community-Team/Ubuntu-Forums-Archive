### Post by sikander3786 on 2011-01-25
Welcome to the forums :-)

The best resource in my opinion is Ubuntu Pocket Guide.

[http://www.ubuntupocketguide.com/index_main.html](http://www.ubuntupocketguide.com/index_main.html)

Other than that, for most common tasks like software installation, you need to go to Applications > Software Center or System > Administration > Synaptic Package Manager.

The Linux command line named Terminal is found under Applications > Accessories > Terminal and can do any task, any task that the gui is capable of doing. It is a mighty stron thing.

Settings are found under System > Preferences or System > Administration.

Ubuntu forums is a great place to learn more. And Ubuntu community documentation is another great resource e.g,

[https://help.ubuntu.com/community/InstallingSoftware](https://help.ubuntu.com/community/InstallingSoftware)

You just need to give some of your time and attention to Ubuntu and it isn't that hard to learn.

Feel free to ask if you've got any queries :-)
