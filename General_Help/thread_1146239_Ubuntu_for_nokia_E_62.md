As far as I know, it's not possible to install any version of Ubuntu to Nokia phones. The hardware is very different from normal PC architecture.

The only Nokia handheld devices I know that can run Ubuntu are the Nokia Internet Tablets (N800/N810), which aren't actually phones. See [http://www.internettablettalk.com/forums/showthread.php?t=25975](http://www.internettablettalk.com/forums/showthread.php?t=25975) for additional information.
