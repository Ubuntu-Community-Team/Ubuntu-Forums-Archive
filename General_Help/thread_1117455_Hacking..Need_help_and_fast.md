### Post by mb_webguy on 2009-04-06
No OS is unassailable, though Linux is quite secure.  It would be extraordinarily difficult to crack a default installation of Ubuntu.  If you install any type of server or network service, you open yourself up a bit, but unless there is some vulnerability inherent to that application or protocol, you're still pretty safe.

If someone told me they'd hacked into my system, I'd ask them to prove it.  Ask them (without giving any information away yourself) to tell you something about your system that can't be determined from outside of your system.  Or even better, since getting back into a system you've already hacked is fairly easy, ask them to leave a message in a text file on your system to prove they were there.

Some people will tell you they've cracked your system as a prank, just to scare you.  Others will do so in the hopes you'll panic and let slip some bit of information that will help them actually crack your system.  Unless you have some credible evidence indicating you've been hacked, I wouldn't assume you have been.
