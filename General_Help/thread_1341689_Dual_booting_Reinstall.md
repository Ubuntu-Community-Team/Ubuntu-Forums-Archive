### Post by john newbuntu on 2009-11-30
The dell manual says it will permanently erase data on the hard drive.  But here is an excellent site from Dan Goodell that explains how a program called "DSRCheck" gets run prior to the restoration process.  I get the feeling that this might rewrite the MBR and therefore the partition table.  But I am not sure on whether it will prompt you prior to doing that.

[http://www.goodells.net/dellrestore/fixes.htm](http://www.goodells.net/dellrestore/fixes.htm)
