### Post by Rocket2DMn on 2009-05-29
The quickest way is to create a launcher on the panel at the top or bottom of the screen.  Right click and select "Add to Panel"; choose "Custom Application Launcher" and click the "Add" button.  A window will now appear where you can enter a Name, Command, and Comment.  For Type, you will probably just want Application, but you can choose Application in Terminal if you want the terminal to pop up.

In Gnome (which Ubuntu uses by default), you can right click the Applications menu area and select "Edit Menus".  From here you can make a New Menu, or add a New Item to an existing menu.  The same dialog as above will appear and you can enter the command to be run.
