### Post by amp711 on 2008-10-02
It sounds like a hardware issue IMHO.  Is this a new machine you just built?
Did it run another OS prior to the Ubuntu install?

If it is a new machine, start by removing any add on cards, run memtest, check the processor temp in the BIOS..  Maybe the BIOS needs a firmware bump?
