### Post by bingoUV on 2008-06-02
You can use it for eternity, as long as you wish. After 3 years, there will be no more updates to it. That means if some security or other issues are discovered, you will not get fixes to those issues after 3 years. 

If the system is not connected to internet, it is generally safe to run after updates are stopped. Otherwise, it is recommended to not use such OSes.
