### Post by nzjethro on 2011-06-30
Lubuntu and Ubuntu are the same operating system (Ubuntu Linux), it's just that Lubuntu has the LXDE desktop environment, and Ubuntu has the Gnome desktop environment (well, Unity now). Installing programmes is done in the same way, and the same programmes will work on both variants (if a programme being installed in Lubuntu has some Gnome dependencies it will install them at the same time). While the default apps on Lubuntu will be different, you can install Gnome apps on Lubuntu, and vice versa. [This guide]("http://maketecheasier.com/lubuntu-review-a-lightweight-ubuntu-variant/2010/05/06") should give you more information.

As for BURG, I'm fairly sure installing will be the same as with Ubuntu. Having never installed BURG with LXDE I can't comment with authority, but I imagine it would be the same.
