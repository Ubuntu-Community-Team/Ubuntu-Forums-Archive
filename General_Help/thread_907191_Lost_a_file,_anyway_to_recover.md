### Post by unutbu on 2008-09-01
First, do not run the server operating system, as any further disk activity in that partition may overwrite the deleted file.

Then go to [https://help.ubuntu.com/community/DataRecovery](https://help.ubuntu.com/community/DataRecovery)
and look at the section #6 entitled "Extract individual files from recovered image". There, it lists recovery tools that might help you.

PhotoRec is a popular tool often mentioned here in the forum. Despite its name, it can recover many kinds of files.

Good luck.
