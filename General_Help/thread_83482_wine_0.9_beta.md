### Post by zekolas on 2005-10-28
I believe wine makes a hidden folder in your /home/username directory that is simply called ".wine", however like I said it is a hidden directory so it will not show up with the "ls" command, however you can still do an "cd .wine" to go into it. There is an option on the file browser that allows you to show hidden files and directory as well.
