### Post by Toz on 2010-08-29
Text-based access can be obtained via ssh ('man ssh' and make sure your firewall is properly configured).

If you are looking for a graphical connection, then have a look at this link:
[http://www.techotopia.com/index.php/Remote_Access_to_the_Ubuntu_Linux_Desktop](http://www.techotopia.com/index.php/Remote_Access_to_the_Ubuntu_Linux_Desktop)

If you are considering a graphical connection over the internet, consider tunneling over ssh (read the section "Establishing A Secure Remote Desktop Session)

/ToZ
