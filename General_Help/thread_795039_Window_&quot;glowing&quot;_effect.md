### Post by BlueSkyNIS on 2008-05-15
That "glowing" effect is actually a bug, and it should be a drop (black) shadow as before. An update will resolve this, goto System -> Administration -> Update Manager, click Check and than Install Updates.

EDIT:
> How can this be disabled?
You can also install "Advanced Desktop Effects Settings" to further personalize the looks of GUI. Into terminal type:
```
sudo apt-get install compizconfig-settings-manager
```

That should do the trick :)
