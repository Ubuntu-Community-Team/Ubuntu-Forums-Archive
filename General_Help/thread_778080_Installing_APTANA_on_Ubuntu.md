### Post by Aearenda on 2008-05-01
In general, you use add/remove programs from the applications menu, or Synaptic Package Manager - both of these allow you to search the repositories for existing programs that you can install. This is a major difference from Windows, and much better - it it worth getting to understand these tools. 

Aptana has their own installation procedure, from what I can see - I had never heard of it before - but there are installation procedures for various Linux versions on their website, [www.aptana.com](www.aptana.com). If they package their installation as a .deb file, then yes, a double-click will install it. This sounds unlikely from their website, so please read their material.
