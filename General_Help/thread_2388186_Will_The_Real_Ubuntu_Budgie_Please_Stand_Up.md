### Post by QIII on 2018-03-29
Hello!

Budgie remix was not official, as the team was still working on getting it into the official family.

That happened with 17.10.  From 17.10 onwards, Ubuntu Budgie is an officially recognized flavor of Ubuntu.  If you download Ubuntu Budgie (17.10.1 or later) from [here]("https://ubuntubudgie.org/"), you will have the official version.

### Post by davidmohammed on 2018-03-30
On the 18.04.1 point release you will be offered an upgrade to the full Ubuntu Budgie 18.04.1 point release - i.e. all the budgie-remix packages will be uplifted to Ubuntu Budgie.

The Ubuntu Budgie team will be adding to the 18.04 release notes (nearer the release date in April) the upgrade instructions, together with optional recommendations to make your 18.04 upgrade a closer fit with the 18.04 ISO release 

e.g. how to replace the lightdm-gtk-greeter with slick-greeter, 
      how to reset the panel to get the new 18.04 layout and applets, 
      how to change to Pocillo icons and gtk theme

Hope this helps.
