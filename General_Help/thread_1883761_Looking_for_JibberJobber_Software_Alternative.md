### Post by davidsickmiller on 2011-11-30
Is your concern being able to use the software offline or the inconvenience of doing data entry between multiple browser tabs?

The only offline tool I know of is [JobTabs]("http://www.jobtabs.com/").  It's a Windows application, and I don't know if it runs under Wine.

There's also [44score]("http://www.44score.com/") which has a browser toolbar so you can record online apps and reminders without leaving the current webpage.  This does work in Firefox on Ubuntu.

Disclosure: I am the founder of 44score.
