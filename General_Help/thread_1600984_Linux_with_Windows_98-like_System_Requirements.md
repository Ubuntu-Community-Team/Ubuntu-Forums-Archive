### Post by lisati on 2010-10-19
My old machine came with Win98SE, and has a 133MHz processor, 64Mb RAM and a 3Gb HDD. I was able to run [Puppy Linux]("http://puppylinux.org/main/Overview%20and%20Getting%20Started.htm") on it, albeit a trifle slowly for my liking.

In my opinion, you're unlikely to find a version of Ubuntu complete with GUI that's going to work with those specs: I ended up putting a command-line only copy of 6.06 on mine.
