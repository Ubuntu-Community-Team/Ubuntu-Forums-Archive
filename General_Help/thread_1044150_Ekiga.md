### Post by mcduck on 2009-01-19
Ekiga is IP phone, using standard SIP protocols (as opposed to Skype, which uses it's own protocol and because of that only works with their own program).

To make free phone calls both need to be using some SIP phone solution, be it Ekiga, some other SIP phone software or actual SIP telephone machine.

If you wish to call to normal phone numbers you need to have a subscription into some SIP phone service, which will cost money. It shouldn't be a surprise that there's no (legal) way to use telephone networks for free.. (which should answer your last question as well);)

Why use Ekiga instead of Skype? Because it can be used to call anybody who has SIP-compatible software or device. Also large companies and organization can run their own SIP services and don't need to rely on Skype or some other external company to handle the service.
