### Post by cariboo on 2009-04-19
I think you have a dns problem. If you are accessing a router, you don't need to specify dns addresses as Network Manager will automatically pick up the dns addresses from the router. Check /etc/resolv.conf it should have the dns addresses provided by your isp, unless you are running your own dns service. /etc/resolv.conf should look something like this:

```
cat /etc/resolv.conf
# Generated by NetworkManager
domain aplus
search aplus
nameserver 64.59.168.13
nameserver 64.59.168.15
```

In my case the nameserver addresses are provided by my isp.
