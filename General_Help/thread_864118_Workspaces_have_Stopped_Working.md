### Post by jonlemur on 2008-07-19
Are you using Compiz or Metacity or something else?

If you're using Compiz, you may want to try Alt+F2, and run 'metacity --replace' to switch to Metacity.  You can also do that from the command line.  It won't have the pretty effects of Compiz, but it may run better, depending on your hardware.

If that fixes the problem, we can probably help fix whatever in Compiz isn't working, but it may entirely unrelated.

### Post by jonlemur on 2008-07-20
metacity is a window manager, and it deals with stuff like window borders and positioning, workspaces, and many other things.  Compiz is also a window manager, but it adds a bunch of eye-candy, like wobbly windows and a desktop cube for viewing the workspaces.  It's really nice when it works well, but it can demand a lot of resources.  I haven't had problems with it locking up the workspaces before, so you may want to try reinstalling compiz if you want to get that extra stuff working.
