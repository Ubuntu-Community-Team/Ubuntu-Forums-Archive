### Post by taurus on 2006-03-24
Codecs and xine are two different packages.  However, you need codecs if you want to play all type of video.  With one codecs, you can use mplayer, xine, vlc, etc.  So, try
```

sudo apt-get install xine mplayer
sudo apt-get install win32codecs

```
Otherwise, you can use Automatix to accomplish the same task...
