### Post by choochoousmc on 2010-05-28
Cut my teeth on computer theory back in 1969 in the USMC.
Then learned basic and and DOS. Moved into all variations windows.
But I don't like or trust Windows. Too many viruses, trojans and the like.
So for last year or so been waiting for the various UNIX spin offs to come of age.
Heard as lot about Linux (redhat) but couldn't get it to install a couple years ago.
Heard how great and stable Sun Microsystems Solaris and Open Solaris OS is.
Last month started the download and install process. NOT impressed.
Only the developers version would actually load up and see my internet.
But It would not recognize my hard drive partitions. people gave me help to "mount" them
But was a pain in the A** to keep going to the Terminal window and typing in Cryptic commands, instead of using a polished GUI. So I gave up.
Came across Ubuntu. First two installs would not work, disk kept crashing. Thought maybe the ISO was bad online, but did NOT explain others success. SO went to store bought a different brand of CD and tried again. This time the install was flawless.
Using an Acer Aspire 5532 two months old.
Ubuntu recognized my wireless network right away.
I needed to know how to resize and breakup the main Partition. Easily done with the live CD and Gpartd.  Kind person here gave me the directions. Worked perfect.
Shut down and rebooted the hard drive. And there they were!!!
Next was the sound and the printer.
I have a epson nx515 wireless printer. Clicked the printer menu. Ubuntu found the drivers for it and printed a test page. Page looks good.
Solaris I had to go find drivers, if they existed, so never looked.

Last was the sound. Opened the menu and raised volume levels. But no way to actually test the sound.
So went to Youtube to play a video.   Hit a snag. Need to update flashplayer.
went to Ubuntu programs menu. Clicked on internet or video. Upgrade flashplayer.
Only option was flashplayer ten. In the forums a person had trouble with it.
But clicked install. Closed firefox and restarted.
The selected video played great!
Only problem I see, is sound is tinny. That could be the internal speakers.
But a great addition to Ubuntu, would be a equalizer in the default install to increase the bass.
But all in all. At this point Ubuntu has been great.
The next test is installing programs I like, want and need, using them saving data and switching between windows data and Ubuntu data.
One thing I would suggest for people wanting to try Ubuntu. If you can afford it, get another notebook computer and install it there.
Inevitably if you do NOT know what you are doing, you might screw up and crash your windows system and maybe lose your data.
But if you insist on putting it on a windows unit. Build a set of windows recovery disks first. and back up your data on a standalone external hard drive or CD's or USB first.
But I do give Ubuntu a thumbs up so far!!
