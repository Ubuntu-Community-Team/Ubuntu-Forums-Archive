### Post by silkstone on 2008-11-26
It's a bone of contention with Ubuntu, and LTS releases in particular, that you get security updates but are left with outdated versions of bundled applications unless you install the new versions manually.

So OOo 3.0 etc will run perfectly on Hardy, but almost certainly will not be available from the normal repos. If you search around you'll find some guides on how to install the new versions.

I don't agree with the policy of having to install a new Operating System every six months to keep up-to-date with software, but nobody listens to me. :)

### Post by fragos on 2008-11-26
In software sources updates tab select unsupported "updates backports (intrepid-backports). Also add 3rd party software source from medibuntu.org. Lastly I monitor getdeb.net which provides release specific feature updates to many of my favorite applications like GIMP.
