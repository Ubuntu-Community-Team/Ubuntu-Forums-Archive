### Post by stinger30au on 2008-11-22
yep, piece of cake

its called virtual box

you can install your original copy of xp or vista or whatever and run it inside ubuntu

about the only major restriction you have is that windows will not let you use direct 3d apps, only direct draw

virtual box can be installed by add remove under applications

you will need to make an .iso of your original xp cd install disc (use brasero or k3b or something)

plenty of tutorials round on how to do it

once you have done it the first time you will kick yourself for how easy it really is


have fun, and welcome to virtual computing
