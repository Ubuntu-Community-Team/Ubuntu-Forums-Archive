### Post by geirha on 2010-03-23
If you are waiting for firefox-3.6 to appear in Ubuntu 9.04's repositories, then you'll be waiting forever. firefox 3.0 was the version in Ubuntu 9.04 when it went into [feature freeze](https://wiki.ubuntu.com/FeatureFreeze), and so it will have firefox 3.0 until it reaches EOL (end of life) in october 2010.

If any serious bugs or security holes are discovered in firefox 3.0 though, these will be fixed, and provided to you via updates.

If you want firefox 3.6, you'll have to either download it as a deb package, or find a PPA (personal package archive) that provides it.

EDIT: Or you can upgrade to Ubuntu 9.10, and then to Ubuntu 10.04 when it is released in late april. Ubuntu 10.04 will have Firefox 3.6.
