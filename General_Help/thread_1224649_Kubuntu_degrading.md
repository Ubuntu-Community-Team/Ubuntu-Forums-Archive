### Post by Zorael on 2009-07-29
Hard to say, as it could be a gazillion different things.

Check the System Activity window (ctrl+esc); any processes hogging cpu time?

As for Pidgin set to autostart, go into System Settings -> *Advanced* tab -> *Session Manager*, and pick *Start with an empty session*.
