### Post by darolu on 2010-05-03
The source code is in Ubuntu's repositories; anyone can access to it by enabling the Source Code in the Software Sources. Are the lines that start with deb-src in your /etc/sources.list

Source code is (or at least should be) available on each program website, i.e. [http://library.gnome.org/devel/jhbuild/](http://library.gnome.org/devel/jhbuild/)

Try Googling "Linux CDs." You'll find hits for various outfits that sell Linux CDs and DVDs. People buy these because they don't have fast or reliable Internet access, because they work for companies with archaic rules that require them to buy physical media for installation, because they think (rightly or wrongly) that purchased media will be of better physical quality than a home-burned disc, or for other reasons. I can also imagine people making impulse purchases of, say, a US$5 Linux DVD in a brick-and-mortar store, although it's been a while since I've seen such a thing. (Then too, I don't spend much time browsing commercial software in my local Best Buy....)

In sum, there is a modest but thriving community of vendors who do sell Linux CDs and DVDs. In my Google search, I recognized one company that's been around for over a decade, so I suspect at least some of them are profitable businesses.
