### Post by Mr._Hope__Change on 2016-02-03
Hi Edward,

Though not even close to being an expert with Ubuntu, here are my observations with the Lenovo Thinkpad T550 running 14.04 LTS:

1)  The machine becomes much hotter than Windows, and the fan is turns on more often as well. 
2)  Sometimes I get major hangups and the only thing that stops it is a manual reset. 

Overall, I am feeling okay with Ubuntu on this machine.  Will probably switch to either Lubuntu or Mint though, because even with 8gig RAM and an i5, the machine seems very slow.

### Post by Vladlenin5000 on 2016-02-03
@Mr._Hope_Change - if yours has the optional Nvidia graphics then it'll perform much better with proprietary Nvidia drivers which aren't installed by default. The default is the open source driver. Also a newer Ubuntu release works much better, irrespective of the graphics

To all - **The current LTS - 14.04 - is NOT the best for such hardware.** Instead, use the current release - 15.10 - and upgrade to the next LTS - 16.04 - in April -or- if you're brave and don't mind lots of daily update you can start with 16.04 right away. It's pretty much stable already.
