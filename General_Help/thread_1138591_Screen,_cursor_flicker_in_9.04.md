### Post by ironjaw33 on 2009-04-26
Hi,

I recently upgraded my Ubuntu installation from 8.10 to 9.04.  I had no problems with my 8.10 installation, but now in 9.04 my mouse cursor flickers when I hover over menu items, web browser hyperlinks, and over most any selectable GUI object.  Also, every so often a window that does not have focus will seem to "jump out" at me as if it were being maximized and given focus, but only for a fraction of a second and then it will return to normal.

I'm running the i386 version with an ATI X1900XTX with the default drivers.  I'm also running two monitors, one through each DVI port.  In 8.10 I used the proprietary ATI drivers, but apparently these are no longer available for my video card.

I've disabled compiz and all desktop effects, but this does not solve the problem.

Any ideas or suggestions?  Anyone else with this problem?

Thanks,
