### Post by rg4w on 2012-08-11
By default Dash uses the most computationally expensive rendering option, which blurs the background making the Dash contents easier to read.

If you install the My Unity system panel from the Ubuntu Software Center, it provides (among many other nifty options) a means of choosing the other two rendering options for Dash.

Of the three Blur settings, "Off" is the fastest, with the overlay still translucent but without taking the time to blur the background.

I have a first-gen Core i3 and I also wish Dash rendering was faster.  I've turned Blue off for a while, but found the opacity to be too light, with the underlying windows showing through just enough to make the Dash contents hard to read.

IMO it would be far more useful to have an opacity setting for Dash - anyone know how to adjust that?
