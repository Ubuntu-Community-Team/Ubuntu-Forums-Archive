mediainfo path/to/filename.mp4
``` (use the proper path to the file, of course) The output will show all the details needed to tell us the codecs used in creation of the mp4 file.

Both Totem (displayed in Gnome as "Videos") and gst-play use the GStreamer framework. GStreamer's plugins for media playback are divided into good, bad, and ugly, for different levels of licensing freedom and support.

Since you've already found other media players that work you could just use one of them but, if you'd prefer to use Totem, you'd need to install the plugins that provide support for the formats you use. 

They are gstreamer1.0-plugins-good, gstreamer1.0-plugins-bad, and gstreamer1.0-plugins-ugly in the Universe repository. There is also the ubuntu-restricted-addons in the Multiverse repository that brings in a couple more. You could only install the specific plugins that you need, or just install them all. I'd personally do the latter, since they aren't big and you might come across a different one that you need in the future.
