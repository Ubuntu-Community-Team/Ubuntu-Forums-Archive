### Post by Anfanglir on 2008-12-10
I installed Ubuntu UMPC 8.10 using the image at:
[http://cdimage.ubuntu.com/ubuntu-umpc/intrepid/current/](http://cdimage.ubuntu.com/ubuntu-umpc/intrepid/current/)
and by creating a bootable usb-stick as described at:
[https://wiki.ubuntu.com/MobileTeam/Mobile/HowTo/ImageWriting](https://wiki.ubuntu.com/MobileTeam/Mobile/HowTo/ImageWriting)

Was also anoyed by the windows that maximize by default. That function can be deactivated by disabling Devilspie (System/Settings/Sessions) and then rebooting.

So far everything seems to work fine on my Toshiba Libretto U100 (it has no touch-screen so I cant comment on touch-screen setup). I encountered one issue with lacking drivers for a pendrive with a ntfs-3g file system (this drive worked without any problems in Ubuntu desktop 8.10 on same system). Problem solved by following the procedure in this thread:
[http://ubuntuforums.org/showthread.php?t=217009](http://ubuntuforums.org/showthread.php?t=217009)

So far Ubuntu-UMPC looks really interesting!
