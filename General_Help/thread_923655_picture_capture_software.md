### Post by mcduck on 2008-09-18
For full-screen screenshots just press the PrtScr key o your keyboard. IF i remember right Alt+PrtScr will capture only the active window instead of the whole screen.

For more featured screenshot app you could try Gscrot: [http://www.getdeb.net/app/GScrot]("http://www.getdeb.net/app/GScrot")

### Post by mb_webguy on 2009-02-04
Try ksnapshot or xvidcap.  Xvidcap has the advantage of being able to capture video, but while it can do exactly what you're asking, it's not quite as easy to use for still screencaps as ksnapshot.  On the other hand, if you're not running Kubuntu, you have to install a lot of KDE-specific packages in order to install ksnapshot.
