### Post by Kain000 on 2008-08-15
Hey does anyone know how to create a shortcut to a folder located in another directory?  I want to make a shortcut to the directory that holds my FTP files on the desktop.

---

### Post by mgmiller on 2008-08-15
Open the file browser so you can see the directory you want a link to.  Right click it and then select, "make link" .  It will create a new file that looks like a directory, but it will say "link to (whatever the filename was)".  Just, drag that to your desktop.

---

### Post by Too Late on 2008-08-15
If you don't want to make a symbolic link (as described in previous post), you can also right-click the desktop, and select "Create Launcher...". Then select Type: Location, put the name and comment whatever you like, and in the location field, type:
```
file:///full/path/to/folder/
```

(eg. file:///usr/share/)
