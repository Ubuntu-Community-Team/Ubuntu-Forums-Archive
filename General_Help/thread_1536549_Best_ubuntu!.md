### Post by snowpine on 2010-07-22
> **hamed23 said:**
> Intel Integrated graphic Card

Which Intel graphics card specifically? If it is GMA500, you should read this thread (or stick with Windows): [http://ubuntuforums.org/showthread.php?t=1229345](http://ubuntuforums.org/showthread.php?t=1229345)

If it is anything other than GMA500, Ubuntu 10.04 should work fine. You can use the regular version or the netbook version, depending on which user interface you prefer.

No operating system can turn a slow computer into a fast computer. :) The Atom is just a slow processor no matter how you slice it, and the hard drives in low-priced netbooks aren't that fast either.. I would recommend booting up and starting Word at home, then putting the machine into Suspend mode. Then, when you get to class, you just have to open it up and start typing.
