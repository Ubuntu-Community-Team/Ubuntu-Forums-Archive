### Post by PCTPaws on 2011-12-03
Okay, here's my problem.  For some reason Ubuntu 11.10 is not recognizing that the Ubuntu CD is in my drive.  I can see it when I bring up the drive and browse around, but when it comes time to install apps from the software area, it tells me I need to insert the CD, and I can't install anything.  

This problem persists with both the internal CD-ROM as well as the external USB CD-ROM.  Both drives show CD's in them, and the correct files are displayed in the windows, but the OS won't recognize them when installing apps.  

I'm not as worried about it being a drive issue because I rarely use a CD for anything.  My question is, how can I install apps without having to use the Ubuntu CD for supplemental programs?  

I tried installing an ISO mount program, but I still need the CD to complete the installation of any app.  I've tried using the apt-get command, but it, too, tells me to insert the CD.

Any ideas?

Thanks!

You should not need the CDs in the drives when installing Applications.

Open Software Sources and un-tick the CDs as a source.It will then stop asking you to load the install cds into the drives.
