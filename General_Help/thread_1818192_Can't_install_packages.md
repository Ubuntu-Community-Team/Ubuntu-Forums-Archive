### Post by ankspo71 on 2011-08-04
Hi,
You can change your download location in the Sofware Sources application.
If you are using Kubuntu go to System Settings > Software Management > Settings > Click on the "Edit Origins" and enter your password when it asks for your password, then the Software Sources window will open.

In the first tab of Software Sources called "Kubuntu Sources", look for the dropdown menu by the words "Download From:". Click on the dropdown menu and choose "other" to select a different download location. A window will pop up that will allow you to choose a server closer to you, or you can use the "Select Best Server" button so Kubuntu can ping each server to automatically find the closest server. The "Select Best Server" method usually works well.
