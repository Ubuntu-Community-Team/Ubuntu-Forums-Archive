### Post by TygerTung on 2009-11-09
When I am using firefox, for some reason it likes to make the default page setup US letter. As I don't live in the US, I use A4 paper.

Since it always defaults to US letter, the printer doesn't like it and doesn't print, and I have to go into the printer and tell it to print to A4 instead. This is a pain in the ***. Is there any way to make the firefox default to A4 instead of US letter?

---

### Post by 101011010010 on 2009-11-09
Hello there.
Type "about:config" in the address bar in Firefox, click 'ok', then in the filter bar at the top of the page, type "letter", you will have an entry where the column "value" is set to letter. Right click on it, choose "modify", and change the value to "A4" (without quotes). 
I hope this helps.
