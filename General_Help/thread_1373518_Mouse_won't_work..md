Have you tried a differnet mouse, to is if it works?
If it does you have a problem with the old mouse.
Is the mouse a ball type or laser?
A ball type you can take the ball out and clean the rollers.
A laser type check to see if it has a red light on the bottom.
If you do have the light, check for foreign matter in the area where the light is and remove it.

If your mouse is a PS2 type when you replace it you should reboot so that the BIOS see it
If it is a USB type you can remove and replace it with rebooting.

good luck and have fun learning
