### Post by Hendrixski on 2007-06-13
For C++ just use Qt.  It's easy and fun and even has its own graphical editor  called QtAssistant.  It's embedded in KDevelop as someone pointed out earlier... just know that you have other options.

GTK I haven't played with, but if you're using GNOME, then most of your apps are written with GTK.

For java, they have all the graphical user stuff built in.  Try using Eclipse with one of its many graphical plugins, or NetBeans which I think has that functionality built in.  Java was only open sourced recently so it had a bad wrap in the open source world for a long time.  It's very cool and trendy now.  Give it a spin.

If you're in python... there's the TkInter... stands for tool kit interface.  And that lets you program some GUI's.

There are plenty more, but I haven't gotten around to them yet.
