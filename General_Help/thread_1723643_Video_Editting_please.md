### Post by Fire_Chief on 2011-04-07
Maybe try Openshot? [http://www.openshotvideo.com/]("http://www.openshotvideo.com/")

If you're on Ubuntu 9.10 or higher, you can add a PPA and install the software.
```
sudo add-apt-repository ppa:jonoomph/openshot-edge
sudo apt-get update
sudo apt-get install openshot openshot-doc
```

Cheers!
