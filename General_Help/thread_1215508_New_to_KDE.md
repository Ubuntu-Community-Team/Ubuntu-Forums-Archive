### Post by HavocXphere on 2009-07-17
Install FF via the package manager. Not sure whether kubuntu is currently using packagekit or adept.

Or just use the commandline with
sudo apt-get install firefox

The tar.gz you downloaded is just an archive. i.e. a zip file of a complete FF directory. You can extract it anywhere. Or google for FF3.5 installation instructions if you want more integeration.

It will probably have to download quite a bit of stuff.
