### Post by hikaricore on 2009-03-17
New packages are only released in our repos when a new Ubuntu release comes out.

There are exceptions such as serious bug fixes and security updates, but otherwise
the version of the application that was out around the end of the release cycle is
the one you'll be using for the next 6 months.

There are user created repos abound out there that may have updated versions of your
favorite programs such as [getdeb.net]("http://www.getdeb.net") and the [PPAs]("https://launchpad.net/ubuntu/+ppas") which you may use but
these are not official releases and may not function properly at all times.
