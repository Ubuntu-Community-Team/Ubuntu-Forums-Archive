Hey,

This is the ultimate guide to setting up plugins in Compiz Fusion. Hope you can find some helpful info in there.

Here's the link: [http://forlong.blogage.de/article/2008/4/26/How-to-set-up-Compiz-Fusion-074]("http://forlong.blogage.de/article/2008/4/26/How-to-set-up-Compiz-Fusion-074")

If you're still not getting a cube, check under "General Options" then the "Desktop Size" tab and make sure the "Horizontal Virtual Size" is 4.
