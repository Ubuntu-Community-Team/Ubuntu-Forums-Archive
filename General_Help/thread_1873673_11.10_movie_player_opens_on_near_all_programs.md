### Post by mc4man on 2011-11-01
edit this file
```
gedit ~/.local/share/applications/mimeapps.list
```

In the [Default Applications] section either remove this line entirely or change nautilus-folder-handler.desktop to nautilus-home.desktop

inode/directory=nautilus-folder-handler.desktop
to
inode/directory=nautilus-home.desktop

If also listed in the [Added Associations] section then you can also change or remove the line. 
(I'd just remove the lines in both sections

There will be a fix for this in the next week or so, *affects only upgrade installs. If you wanted to leave mimeapps.list alone then this will take care of
```
sudo ln -s /usr/share/applications/nautilus.desktop /usr/share/applications/nautilus-folder-handler.desktop
```
