### Post by Mark Phelps on 2011-10-13
> **lthomas926 said:**
> How do I go about correcting this then if that is the case?

There are several ways to do this, but with Win7 the one that generally works best is to use a Repair CD to restore the boot files and directory.

Since you probably did NOT use the option to create such a CD of your own when Win7 worked, you now have to go to the link below, select the proper Win7 version and PURCHASE the CD image:

[http://neosmart.net/blog/2009/windows-7-system-repair-discs/]("http://neosmart.net/blog/2009/windows-7-system-repair-discs/")

Then, download that image, burn it to a CD, boot from that CD, and run Startup Repair at least three times.
