Hello,

this is not a big deal but I post it in case someone face the same problem. I created opera mail client without really wanting it,  I was trying something else, and I wanted to remove it. In order to do so I opened a terminal and I typed:
```
cd .opera
```
and then
```
rm -r mail
```

The next time I opened opera the mail client was gone. From the aforementioned folder only some files were missing. Just FYI.
