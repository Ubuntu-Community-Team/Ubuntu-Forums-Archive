### Post by Forlong on 2008-06-22
You can't compare Xfwm's compositor to Compiz.
Compiz is a whole new window manager that lets you use lots of effects and enhancements no other wm is capable of.

To install Compiz on Xubuntu, simply install the following packages:
```
sudo apt-get install compiz-plugins compiz-fusion-plugins-main compiz-fusion-plugins-extra compizconfig-settings-manager sexy-python emerald
```

See here how to get a lot out of your Compiz install: [http://forlong.blogage.de/article/2008/4/26/How-to-set-up-Compiz-Fusion-074](http://forlong.blogage.de/article/2008/4/26/How-to-set-up-Compiz-Fusion-074)
