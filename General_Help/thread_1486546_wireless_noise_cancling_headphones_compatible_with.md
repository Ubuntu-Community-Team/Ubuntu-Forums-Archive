### Post by blazemore on 2010-05-18
The headphones themselves cancel out the noise, regardless of the operating system.

Any headphones that plug into a standard 3.5mm "headphone port" will work, providing you have sound working already.

USB headphones are rarer, but do rely on the operating system for some functions. If the pair of headphones you are considering connects via USB (unlikely), it's worth checking on Google or using the search function of this forum to see if anyone else has confirmed them working.

Hope this helps :-)
