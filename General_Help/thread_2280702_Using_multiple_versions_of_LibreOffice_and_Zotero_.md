### Post by grahammechanical on 2015-06-01
I am not sure I understand the problem.

I came to Ubuntu back in in 2007 when the office suite was OpenOffice. I converted some Lotus Wordpro documents to OpenOffice documents. I am still adding to and editing some of those documents and copying and pasting between them and newer documents and I have had no problem transiting from OpenOffice to Libreoffice and I have used every version of Ubuntu since 2007. Right now I am working with Libreoffice 4.4 I do not have problems with having multiple document windows open and copying and pasting between them. 

When Libreoffice goes grey it is a sign that the application is busy. The problem seems to be with the compatibility of the Zotero plugin for the web browser. 
> 
[h=3]LibreOffice 4: No response from plugin[/h]    Zotero LibreOffice Integration 3.5.3 and earlier are not compatible with  LibreOffice 4. Under these versions of Zotero LibreOffice Integration,  clicking buttons will not elicit a response from the plugin. To fix  this, Zotero for Firefox users should install the [latest version of LibreOffice plugin]("https://www.zotero.org/support/word_processor_plugin_installation#libreoffice_openofficeorg_neooffice"), while Zotero Standalone users should install the [latest version of Zotero Standalone]("https://www.zotero.org/download/"). If these components are up to date and the plugin still does not function, follow [these instructions]("https://www.zotero.org/support/word_processor_plugin_troubleshooting#zotero_toolbar_doesn_t_appear").
