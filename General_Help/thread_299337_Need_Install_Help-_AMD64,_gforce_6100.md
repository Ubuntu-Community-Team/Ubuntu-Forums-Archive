### Post by igknighted on 2006-11-14
It sounds like you are installing off of old media, I would recommend burning fresh CD's, and burn them at like 4x (slow, yes, but fewer errors).  Also, if you are having issues the last thing you need is the x86_64 edition.  i386 is much more reliable.  The x86_64 architecture is a subset of i386, so the i386 version of linux is quite native on your processor.  Good Luck!
