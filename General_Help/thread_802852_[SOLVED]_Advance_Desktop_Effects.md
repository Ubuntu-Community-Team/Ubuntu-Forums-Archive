### Post by atomkind on 2008-05-22
Hey Rajas, I'm just gonna give you a few links that take you through the steps of some of the items on your list. Hope it helps. 
So here it goes:

1) Desktop Effects
System>Preferences>Appearance>Visual Effects Tab
*Make sure proprietary drivers are enabled: System>Adminitration>Hardware Drivers

2) CompizFusion/Advanced Desktop Effects Settings
Try running the following command on Terminal
```
sudo apt-get install compiz compizconfig-settings-manager 
```
Advanced Desktop Effects should show in System>Preferences>Advanced Desktop Effects Settings
*However I recommend you read one the following HOWTOs

Here's the general HOWTO
[https://help.ubuntu.com/community/CompositeManager/CompizFusion?highlight=(compiz)|(fusion](https://help.ubuntu.com/community/CompositeManager/CompizFusion?highlight=(compiz)|(fusion))
