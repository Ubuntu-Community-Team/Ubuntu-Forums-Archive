### Post by dashnak on 2006-03-18
In case you still want to run winamp, it is inside the folder you selected for installation, which is "home/username/cxoffice" by default.. 
Inside, there should be a folder named after whatever you named the bottle in which you installed winamp (Well, this bottle thing is fairly new, so if you're using some older version, you just have to browse around "home/username/cxoffice")...
If you open it, you should see something resembling the layout of a windows c drive, meaning that you can browse all thw way down to program files and find the exe...
Then you can create a custom launcher and put it wherever you want...
