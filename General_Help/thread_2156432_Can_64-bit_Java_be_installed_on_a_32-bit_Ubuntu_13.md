### Post by TheCadaver on 2013-06-21
I have a 32-bit Ubuntu, but is it possible to install a 64-bit Java? Minecraft and other Java applications recommend it, and as of now, the 32-bit seems to be causing some weird glitching and rendering problems. Also, I was unable to install a 64-bit Ubuntu in the first place; when I installed the OS, I clicked "64-bit", and I made sure (and I have done this for every upgrade since Maverick and have had the same result) to click it, yet the OS always comes out a 32-bit. As far as I know, my CPU has 64-bit support (as seen in the product specs here; [http://www.newegg.com/Product/Product.aspx?Item=N82E16819115065](http://www.newegg.com/Product/Product.aspx?Item=N82E16819115065)). Does anyone know?

### Post by sanderj on 2013-06-21
No, not possible.

And your Intel Core i3-550 is very capable to run Ubuntu 64-bit (and Java-64bit on top of that). Here's a direct linke to 64-bit Ubuntu 13.04: [http://ftp.belnet.be/ubuntu.com/ubuntu/ubuntu/releases/13.04/ubuntu-13.04-desktop-amd64.iso](http://ftp.belnet.be/ubuntu.com/ubuntu/ubuntu/releases/13.04/ubuntu-13.04-desktop-amd64.iso)

And 12.04.2 64-bit:
