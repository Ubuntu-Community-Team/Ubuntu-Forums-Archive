### Post by mikewhatever on 2007-10-28
One way is to dual boot, in other words, create a separate partition for Ubuntu, install it, and choose which OS to boot every time you turn on the PC.
[http://www.google.co.il/search?q=ubuntu+vista+dual+boot&ie=utf-8&oe=utf-8&aq=t&rls=org.mozilla:en-GB:official&client=firefox-a](http://www.google.co.il/search?q=ubuntu+vista+dual+boot&ie=utf-8&oe=utf-8&aq=t&rls=org.mozilla:en-GB:official&client=firefox-a)

The other way is to install a virtual machine on Vista and run Ubuntu in the VM.
