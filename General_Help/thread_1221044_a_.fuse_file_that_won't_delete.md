### Post by mcduck on 2009-07-23
Fuse is used for virtual filesystems, and is used for example when mounting NTFS drives.

[http://en.wikipedia.org/wiki/Filesystem_in_Userspace]("http://en.wikipedia.org/wiki/Filesystem_in_Userspace")

As such, the file really shouldn't be removable. Most likely it will disappear on it's own, when you remount the drive if not before.

### Post by jerome1232 on 2009-07-23
It seems I posted for help to soon, I figured out how to check what process has a file open, using fuser -m /path/to/file

I closed the program and all is well.
