### Post by maniacmusician on 2008-05-05
> **ZeldaFan said:**
> Should all the hardware supported by the gnome release of Ubuntu be supported in kubuntu? I'll still check everything with the liveCD, but I was just wondering.
As far as hardware support goes, yeah, it should be the same as they both use the same underlying system (the linux kernel and associated services). However, in terms of software features that **utilize** that hardware, you may notice some differences. For instance, some people report that KNetworkManager doesn't work as well as gnome's nm-applet. Although, with KDE4 having Solid, and a new version of KNetworkManager, the playing field may be equal. Same with other similar programs. You'll just have to wait and see, pretty much.
