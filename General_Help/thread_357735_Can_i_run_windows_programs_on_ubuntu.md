### Post by GrumpyBob on 2007-02-10
I find the easiest option is to install qemu ([http://fabrice.bellard.free.fr/qemu/](http://fabrice.bellard.free.fr/qemu/)) and use that to run Windows.  In my case I'm running Win98SE this way.  Of course you need a legit copy of Windoze for this.  Other options might be to use VMware, or to bite the bullet and buy Crossover Office (or whatever they've renamed it) - I find it works pretty well and there is plenty of advice on which apps work and how well on the Crossover website.
