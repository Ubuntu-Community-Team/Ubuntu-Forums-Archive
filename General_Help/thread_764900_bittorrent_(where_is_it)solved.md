### Post by munkyeetr on 2008-04-24
To use the stock BitTorrent you can either:

1) Try *right-clicking* on the .torrent file and choose **Open with BitTorrent**

2) Open a terminal (**Applications** -> **Accessories** -> **Terminal**) and type:
```

gnome-btdownload

```

then browse to the .torrent file, then choose a destination for the downloaded file.

3) Right-click on the **Applications** menu and choose **Edit Menus**
   Select the **Internet** section
   Check the *Show* box beside BitTorrent; it will now show up in your **Applications** -> **Internet** menu.

4) If you just wanted to know the location of it on your file tree type:
```

which gnome-btdownload

```
