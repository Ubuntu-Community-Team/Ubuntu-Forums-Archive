### Post by Prospero2006 on 2006-02-17

Ubuntu is by far the most user-friendly distro I've come across, but I agree with the previous posters that it is not necessarily a 'replacement.' 

-90 percent of personal computers run windows because it is pretty much totally idiot proof. Everything works out of the box. Point, click, and be done with it. 

-Ubuntu is starting to catch up, but you still have to be willing to read and learn. I'd say with Ubuntu the learning curve is minimal, but there is a still a learning curve to overcome for the new user.
