When I boot up my machine then launch Firefox I'm asked if I want to start a new session or restore my previous session. If I select restore previous session then the tabs I had open the last time I was using my maching are loaded. Some of the sites are username and password protected such as Googlemail and my eBay account. I don't have to log on to the sites and can continue using my accounts on the respective sites just from where I left off from the last time I had my laptop switched on.

### Post by chrisccoulson on 2008-05-18
The site probably leaves a cookie. For example, I never have to log in to these forums, as my session resumes each time I open the browser. These security settings are dependant on the website you're visiting. Banking sites and sites like Paypal will require you to log in each time you open the browser, and will never keep your session open.

Googlemail is one site that will keep your session open (I use Googlemail). Ebay will keep it open for up to 24 hours.
