### Post by junglepeanut on 2006-11-25
Are you using Edgy or Dapper?

If I recall Dapper would jump to text when it got to thirty restarts to check the hard drive, but from edgy on it doesnt tell you it is checking, so  you may just want to wait for a while and see if it finishes its check. An easy way to double check that it is just doing a check would be to start single user and see that it will show the text saying it is checking. If you have a large hard drive or slow one this can take a bit.
