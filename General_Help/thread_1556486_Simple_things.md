### Post by sureshsaragadam on 2010-08-19
If you want to add a shortcut to your panel, right click the panel and you will find **+Add to Panel...**
select **Custom Application Launcher**

fill in the fields, then for the icon you have to select the appropriate location...

Note: some times you need to covert the icon to **png Format **to be displayed promptly
