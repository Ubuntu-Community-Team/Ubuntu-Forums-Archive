### Post by rmcellig on 2010-05-23
Xubuntu does use GDM themes, so you can download one from gnome-look.org. Then you go to Settings -> Login Window. In the Local tab there is an option to install more themes. Find the downloaded file through that and it should be added to the list you can choose from.
