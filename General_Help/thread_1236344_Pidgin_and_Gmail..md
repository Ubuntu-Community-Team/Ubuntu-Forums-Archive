### Post by elekrill on 2009-08-11
To be more specific, at the Windows Live ID page linked baove, there will be a a link "Sign up for an account". On the page that follows, choose "Yes" to proceed.. The rest is pretty self-eplanatory. 

Note that the password provided for your Live ID account is the one you will be using when you sign on to pidgin (or MSN for that matter). What you're doing here (as far as I know) is simply creating a Live ID account associated with your e-mail address.
