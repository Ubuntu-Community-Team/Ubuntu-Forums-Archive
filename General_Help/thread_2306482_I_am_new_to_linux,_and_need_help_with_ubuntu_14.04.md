### Post by Geoffrey_Arndt on 2015-12-15
Hello rozz2 . . . either forum (newbyLand or General) will work . . . so, no issues or anything wrong in either case.

But, you should be aware that "perhaps" 5% of all Windows programs will run in Linux using Wine/Play-on-Linux.    Wine is a remarkable program, but it is NOT windows (and never will be).    On the other hand, Virtual Box will run almost all Windows software IF installed correctly on a machine that has sufficient RAM and processing power.   There are several excellent videos on Youtube about how to configure it correctly.

A better strategy (if you're really serious about running Linux/Ubuntu) is to initially (within your first newbie year) ONLY install software from the official repositories via Ubuntu Software Center or another front end package manager like Synaptic.   Then, as you learn more, you can install scripts or other programs from a FEW reliable sources (PPA's).   Also helpful are websites like "Alternative To" . . so you can find out which alternative programs in Linux can partially or fully replace those Windows programs you've used before. 

On a final note, even though Linux now has more games & Steam, etc. . . . it is still nowhere near the gaming platform Windows is . . . and if that's what you want/need, maybe better to just stick with Windows.
