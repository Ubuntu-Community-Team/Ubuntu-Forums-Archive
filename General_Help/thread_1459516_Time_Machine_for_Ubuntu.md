### Post by 2hot6ft2 on 2010-04-22
Orson Wells Time Machine? yeah right...:)

Or something like [Apple - Mac OS X - Time Machine]("http://www.apple.com/macosx/what-is-macosx/time-machine.html")
Perhaps like [Back In Time]("http://gtk-apps.org/content/show.php/Back+In+Time?content=100087") it's in my repos. but I don't know if it's in the standard ubuntu repos.. I use [Ubuntu Ultimate Edition]("http://www.ultimateeditionoz.com/downloads/")

You can try installing it to find out if it's in the repos. by clicking on this if you're using 9.04 or up
[Back-In-Time]("apt:backintime-gnome")
Or install it in a terminal
Open a terminal
Applications > Accessories > Terminal
and run
```
sudo apt-get install backintime-gnome
```

> Back In Time is a framework for rsync, diff and cron for the purpose of
taking snapshots and backups of specified folders. It minimizes disk space use
by taking a snapshot only if the directory has been changed, and hard links
for unmodified files if it has. The user can schedule regular backups using
cron.

This is the GNOME front-end for the backintime-common package. If you also
install the package nautilus-actions, it can be integrated with the Nautilus
file manager to provide context-menu options.
Or search for backintime-gnome and install it in
System > Administration > Synaptic Package Manager

If you install it it will be in
Applications > System Tools > Back In Time
