### Post by miscellanyous on 2009-02-17
I have the same error using kubuntu 8.04 with an HP photosmart M417 camera. I also tried out the camera using ubuntu 8.04 (ie GNOME desktop instead of KDE). It works fine. So I believe this is a problem with KDE. [See here]("http://ubuntuforums.org/showthread.php?t=1057407&highlight=photosmart") for my attempts at a solution (unsuccessful so far).


EDIT:
I was able to solve the problem by changing the camera setting on the camera's internal menu from "Disk drive" to "Digital camera". Now the images transfer normally.
