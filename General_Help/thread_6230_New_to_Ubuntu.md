### Post by darrell on 2004-11-27
[QUOTE=Jongi]How does one go about installing KDE[/QUOTE]
 You don't need to install the whole of KDE to use K3B. install the following packages from  universe:

K3B
Kcontrol (the KDE control centre)
(apt or synaptic will also fetch a number of dependencies)

Kcontrol is not necessary, but you can use it to make the fonts/look and feel of K3B much nicer
