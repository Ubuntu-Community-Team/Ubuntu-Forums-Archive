### Post by uteck on 2006-04-19
From the error it is giving, "(EE) Failed to load module "pcidata" (module does not exist, 0)", the problem is the pcidata module.  Did you upgrade or remove any modules or Xorg?
From a different thread some people said reverting to an older version of Xorg fixed it.  Did you upgrade recently?  If so you should have the downloaded packages still stored.  Try;
dpkg -i /var/cache/apt/archives/xserver-xorg_6.8.1-1ubuntu6_i386.deb
Or look around in /var/cache/apt/archives/ for an older version of xserver-xorg.
