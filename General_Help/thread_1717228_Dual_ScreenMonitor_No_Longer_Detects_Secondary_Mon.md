### Post by earthmeLon on 2011-03-30
I backed up the non-working xorg.conf and then removed it to start from scratch.  I then ran **sudo nvidia-xconfig --add-argb-glx-visuals --composite**, which wrote a clean xorg.conf file.  I opened nvidia-settings and went to the 'Display Configuration' and told it to 'Save X Configuration File' to update the generic monitor settings.

After I got everything back to normal/default, I added what was in the backed up xorg.conf line by line.

The problematic part was:
```
    Option         "ConnectedMonitor" "DFP-0"
```
