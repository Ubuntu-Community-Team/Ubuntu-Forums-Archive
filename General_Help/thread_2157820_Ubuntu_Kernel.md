### Post by deadflowr on 2013-06-26
Ubuntu is made up of many different code bases, Linux being just one of them.

Here is a place to look for the source of the linux kernel used in Ubuntu
[http://packages.ubuntu.com/search?keywords=linux-source](http://packages.ubuntu.com/search?keywords=linux-source)

As an example, here's one of the linked pages from the above link.
[http://packages.ubuntu.com/raring/linux-source-3.8.0](http://packages.ubuntu.com/raring/linux-source-3.8.0)

If you look at the right side, you'll see options to download the source.

### Post by Cheesemill on 2013-06-27
Or you can use the apt-get command (if you have the source repositories enabled)...
```

```
