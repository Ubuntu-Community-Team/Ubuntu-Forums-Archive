### Post by fballem on 2009-08-17
Now that we're done insulting each other, in ubuntu, if you look under Applications > Accessories, there's a launcher called Take Screenshot. If you right-click on this and select the Add launcher to panel option, then there will be a small icon that looks like a camera added to your panel.

When ever you need a screenshot, click on the camera and you'll get a panel that will ask you if you want the whole desktop or the currently active screen. If you put a delay of 2 seconds or so, then select okay, this application will take the requested screenshot and ask you where you want to save it.

That's how to take a screenshot in ubuntu.
