### Post by ParianPete on 2010-07-25
Added following 
open /etc/default/grub
- insert GRUB_CMDLINE_LINUX="acpi_enforce_resources=lax"
- save and close
- run update-grub
- reboot

Now only have hwmon0, and all is OK, but have lost CORE temp.
