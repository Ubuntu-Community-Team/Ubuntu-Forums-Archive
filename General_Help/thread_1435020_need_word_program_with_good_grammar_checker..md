### Post by rsmudge on 2010-03-31
You have some options, but they depend on what program you're using the grammar checker with.

If you're looking for a grammar checker to use with OOo, then try LanguageTool. [http://www.languagetool.org](http://www.languagetool.org). They have rules that cover many common English writing mistakes.

I work on a product called After the Deadline. It's a spell checker, grammar checker, and style checker. It uses contextual information to make smart recommendations. If you're using web applications for your email and word processing, then our Firefox add-on might help you. [http://firefox.afterthedeadline.com](http://firefox.afterthedeadline.com)

On the command line you can use GNU Style and Diction but these programs are a bit dated. Still worth looking at though. [http://www.gnu.org/software/diction/](http://www.gnu.org/software/diction/)
