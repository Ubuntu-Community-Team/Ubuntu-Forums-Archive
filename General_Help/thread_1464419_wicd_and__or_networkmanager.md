[Network manager supports any kind of network connection.]("http://projects.gnome.org/NetworkManager/") [Not all connections are supported on wicd.]("http://wicd.sourceforge.net/")

I would only uninstall network manager if I actually detected some kind of conflict. When it works, (as it does on my machine) its awesome.
