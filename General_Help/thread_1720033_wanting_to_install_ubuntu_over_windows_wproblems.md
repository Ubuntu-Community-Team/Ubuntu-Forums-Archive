### Post by davidmohammed on 2011-04-02
I wouldnt worry too much.  Windows viruses dont infect linux machines.

You could always download "systemrescuecd" and burn that iso to a CD.  Boot from the CD and run the AV tools - this should clean up your MBR and the rest of your windows installation.
