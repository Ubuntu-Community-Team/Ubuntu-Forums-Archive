### Post by Jezze on 2005-04-08
This question has been asked before. In synaptic there is an option Package -> Force Version... set it to the choose ubuntu packages. Then you should be able to install mplayer.

---

### Post by DJ_Max on 2005-04-08
From that error message, it doesn't say the newest versions are installed. It says the package you are trying to install require versions of packages you can't install.
