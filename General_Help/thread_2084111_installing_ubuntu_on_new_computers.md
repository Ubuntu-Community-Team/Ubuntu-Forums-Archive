### Post by cwsnyder on 2012-11-14
The only lock-in was involved with 'Windows certified' vendors, and especially those using UEFI or those using both UEFI and ARM processors after the dust settled.

Build-your own and custom built will support Windows 8, but not lock in to Windows 8.
