### Post by powerpleb on 2008-06-17
My middle mouse button is awful, everytime I press it it just scrolls up or down, making it very difficult to paste text. Is there a keyboard shortcut that can do the same thing?

### Post by bingoUV on 2008-06-17
run xcutsel in a window. Press the button "copy 0 to primary". Now shift+Insert should work for pasting.
