### Post by MartinG on 2006-02-08
Not quite sure what you mean -- Konqueror is an application like any other, and you don't need to know anything about it in order to install Flash.  Just install Flash and the installer will put the files where they belong.  Unlike Windows, you don't (in general) choose where to install things in Linux.

I presume, incidentally that it's just the plugin you want?  And where are you trying to install it from, because it's available from some non-standard repositories.  The easiest way to get it is to install [Automatix ]("http://ubuntuforums.org/showthread.php?t=66563") and from its menu install Firefox plugins.  Then in Konqueror go to Settings->Configure Konqueror->Plugins, and press the "Scan for new plugins" button, and it should be there.
