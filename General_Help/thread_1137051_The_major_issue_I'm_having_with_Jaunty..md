### Post by vancouverite on 2009-04-25
If you are trying to get a bunch of files with tags into directories that make sense, you should try automating the process with something like EasyTAG (in the repos).  This handy little program will read a tag, rename the file and move it to a directory based on the tag data.  It will also go the other way, filling tags from directory/file name info.

[http://easytag.sourceforge.net/](http://easytag.sourceforge.net/)

Very useful for recovering your music off your iPod after a crash and things like that ;)

It is a gnome app, but will probably work in kubunut.
