### Post by callie510 on 2009-04-07
You can also try pressing F keys while you boot up, as usually one of these will force a menu where you can select where to boot. I think the particular function key varies between manufacturers. I have a Dell, and pressing f12 while the bios is loading forces a menu to appear where I can select to boot from CD.

Yeah, I always think I have my BIOS set to boot from Cd first, but I too have noticed that it does not always actually boot from the CD and does go straight to the hard drive... who knows why ... try googling your laptop model and stuff like "boot from CD" or "choose where to boot."

Or you can just restart a lot and try pressing F keys. Yeah, I've done this... I'd try F8 and F12 first....
