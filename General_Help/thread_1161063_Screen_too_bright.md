### Post by CatKiller on 2009-05-16
You could always lower the gamma if you don't want to use the monitor's own brightness settings.

EDIT: Don't worry about the fact that you're shown two different refresh rates. NVidia's driver lies to X for their own reasons, so their tool will display the correct rate but anything that uses X's own tools will give a peculiar number in the 50s.
