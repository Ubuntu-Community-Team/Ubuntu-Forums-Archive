### Post by grahammechanical on 2012-01-12
To fix this do this:

Right click on an odt file an select Open with Other Application and then choose Libreoffice. That will associate Libreoffice with all odt files. 

Somehow Nautilus has become associated with those files that you mentioned. Now you need to associate them with the applications that you want them to open with.

Regards.
