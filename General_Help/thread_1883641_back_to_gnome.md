### Post by sgage on 2011-11-21
I use Gnome Classic (no effects) in 11.10

```
sudo apt-get gnome-session-fallback
```

It's not exactly the same as the Gnome 2.3 panels, but quite similar, and I've been able to get it set up the way I like.
