### Post by coffeecat on 2011-07-02
If you open ccsm -> Unity Plugin -> Experimental tab, there's a button to the right of the launcher icon size slider. Hover your mouse over it and it says "Reset setting to the default value". It does what it says it will do! :)

You'll find that button for everything (or almost everything) in ccsm.
