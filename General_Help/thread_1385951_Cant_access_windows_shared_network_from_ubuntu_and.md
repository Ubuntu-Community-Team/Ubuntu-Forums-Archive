### Post by Silvertones on 2010-01-20
I have also had a little quirkyness in this area. When Id go to Place/ network/ win network the other computer(Win) would not show. To solve this first connect using "connect to server". You should then be able to see the other files. Right click on the folder and create a link. This will now stay as a permanent link under places and has never failed me. Now going from the windos to ubuntu I sometimes have problems. This is fixed, for me anyway, by first connecting from the Ubuntu computer to the Win computer. At this point I'm good to go.
