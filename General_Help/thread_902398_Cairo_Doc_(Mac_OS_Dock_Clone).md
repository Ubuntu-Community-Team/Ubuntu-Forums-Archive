### Post by coffeecat on 2008-08-27
You might find it easier to install [Avant Window Navigator]("https://launchpad.net/awn") from Synaptic (System > Administration > Synaptic). You can use Add/Remove if you stipulate 'All Open Source Applications' but I thought I'd point out the less user-friendly but more powerful Synaptic just in case you hadn't found it. :wink:

If you install AWN from the repositories you won't have any compiling to do. You do, however, need compiz running for it to work, so depending on your video hardware, this might or might not be OK for you.
