### Post by SeijiSensei on 2015-04-27
Widgets that install to the Panel need to be activated from there.  Open the panel, then choose Add Widgets.  You can also add widgets to the desktop by right-clicking on it and choosing Add Widgets. 

I suspect plasma-widget-quickaccess is already installed properly; you just need to find it in the Widgets list.
