### Post by blueshiftoverwatch on 2010-01-24
> **wrose51106 said:**
> I want to be able to control my machine. I want a beginners book not on how to install linux, but how to administer it. Something like an idiots book, get myself a good foundation before moving on up.
In that case, do not get [The Official Ubuntu Book]("http://www.amazon.com/Official-Ubuntu-Book-Benjamin-Mako/dp/0137021208/ref=sr_1_1?ie=UTF8&s=books&qid=1264350435&sr=8-1"). It's for complete newbies only. You might want to try [The Linux Bible]("http://www.amazon.com/Linux-Bible-2010-openSUSE-Distributions/dp/0470485051/ref=sr_1_2?ie=UTF8&s=books&qid=1264350405&sr=8-2") instead. They release a new and slightly updated version of it every year. There is probably little to no difference between the 2009 and 2010 editions. But you might as well go with the newer version.
