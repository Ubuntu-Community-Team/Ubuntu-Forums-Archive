### Post by Mr Frosti on 2006-10-09
GPG keys are put in place to verify, or check the "signature" of a server that is acting as a repository. You have the ability to import a key into your trusted repositories list, however this isn't necessary for software installations.

Many Ubuntu repositories advertise their GPG keys. For EasyUbuntu, check out the "Authenticated" section at this web address:

[http://easyubuntu.cafuego.net/](http://easyubuntu.cafuego.net/)
