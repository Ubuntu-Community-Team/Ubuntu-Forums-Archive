### Post by ninjapirate89 on 2009-09-08

Edit -> May be System -> Preferences -> Sessions on older versions of Ubuntu.

Edit2 -> Ubuntu Tweak has a section similar to CCleaner that removes unneeded files, configs, etc from Ubuntu. [http://ubuntu-tweak.com/downloads](http://ubuntu-tweak.com/downloads)

### Post by cariboo on 2009-09-08
It may be that fsck isn't finishing, this could cause startup to take longer. Without proper diagnostics it pretty hard to help solve your problem. Remember Linux is not Windows adding extra programs doesn't slow things down. If they have added services like the LAMP stack it will take a bit longer to start, but it shouldn't be noticable.
