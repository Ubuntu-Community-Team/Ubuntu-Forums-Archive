### Post by Mgiacchetti on 2008-08-12
considering you have only used 6.04 for a "couple of months"
I would back up whatever you have dire need of (docs pics etc) on a cd or flash drive

then
insert ubuntu 8.04
restart computer
when prompted press the appropriate key to boot from cd
follow installation through normally 
if you want 6.04
    install on a new partition in the partition manager
if not
    choose manual and keep the partition settings, just format your mountpoint and MAKE SURE you choose your swap drive,
