### Post by lian1238 on 2008-09-27
Assuming you have ccsm installed.. if not, then install it.
```

sudo apt-get install compizconfig-settings-manager
```After that, run 'ccsm' or go to *system->preferences->advanced desktop effects settings*.

Look for Cube Caps for the top and bottom images. For the background, go to Desktop Cube->Appearance tab. Play around with the background image and skydome.

If there's no background or caps, look for Png in the ccsm and check it.

Hope that leads you in the right direction.

### Post by lian1238 on 2008-09-27
You should use Cube Caps for the top/bottom images. It's easier than Desktop Cube.
To add a top image, copy the image. Then when you click Add under Top mage files, paste into the box. (No need to type)

Also enable the Image Loading plugin for the type you use, JPEG or PNG.
