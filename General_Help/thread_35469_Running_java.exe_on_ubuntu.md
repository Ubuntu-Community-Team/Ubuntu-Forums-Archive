### Post by ltmon on 2005-05-19
Files with .exe on the end are windows executables, and won't run on linux (without using some emulation software).  What you want to do is find the linux equivalent for the windows program you want to run.

[Here](http://ubuntuguide.org) is a good starting point, and it includes instructions for installing java on Ubuntu.
