### Post by Smart Viking on 2010-09-03
Ye i think they ship with flash in it, and you can enable it somewhere in about:something, but you could also launch the following command:

```
sudo apt-get install ubuntu-restricted-extras
```It will install lots of codecs n stuff including flash. When it is finished restart the browser and flash should be working(in both firefox and chrome(ium)). :)
