### Post by cornelis spronk on 2008-11-15
Here is a summary of the web page I listed earlier.

Go to the Firefox address bar and clear it.  Then type in "about:config" (without the quotes).

You will get a warning screen, so be careful. As you proceed you will get a long list of all your settings. Those which have been changed from the original are in bold type.

Scroll down to the line starting with "general.useragent.vendor" which is set to "Ubuntu".  Right click the line and you get a small window requesting you type in a new vendor.  Type in "Firefox" (without the quotes).  This will change the Ubuntu to Firefox.

Close out Firefox then bring it back up.

In my case, Hotmail became fully functional.
