### Post by MARP1961 on 2011-09-18
You will need to **boot the computer from the Windows 7 CD** then let it format the hard drive and install. Of course, if you do this it will overwrite anything already on your computer, so make sure you have copied all the files you want to keep onto some kind of storage medium (CD, DVD, external hard drive or USB stick).

Your computer must be set up to boot from CD first. Try switching on with the Windows disk already in the tray. If you're lucky it might then boot from disk. If it does not, then look at your instruction manual to see how to boot from disk (on a Dell you press F12 as you're switching on). You may need to place the CD drive at the top of the boot order in your computer BIOS. Be very careful what you do before you play with settings in the BIOS though.
