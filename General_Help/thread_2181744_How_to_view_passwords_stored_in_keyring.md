### Post by coldraven on 2013-10-20
I'm guessing that you have zero chance of retrieving your password, Linux is pretty good when security is involved.
What I suggest is that you create a text file and on different lines type what you may have chosen as a password..
So it would look something like this:
pAssword99
passWord-99
mYdogsname
mYcatsname
etc.

Then copy/paste  each one until you crack it, it will save you going crazy if you have a record of each attempt.
