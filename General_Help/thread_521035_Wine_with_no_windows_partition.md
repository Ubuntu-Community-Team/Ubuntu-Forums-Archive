### Post by hikaricore on 2007-08-08
There are 64bit builds of WINE but I can't comment on how well they work.
WINE only supports 32bit as far as I know so the 64bit binary would just be for compatibility.

You do not need a ***dows partition to install applications/games under WINE either.  ^_^

Your best bet would be to check the [WINE Application Database]("http://appdb.winehq.org") first to see how well/if your software runs.

Also it is easily possible to make desktop, menu, and panel launchers for software run via WINE.
But one step at a time, get something installed then go from there.  :)
