### Post by Frogs Hair on 2011-07-12
If you have installed the Elementary Nautilus PPA , to add Gloobus Preview you enter this code . I'm not sure which PPA you have added.    ```
sudo apt-get install gloobus-preview
```

As far as I know Gloobus only previews folders in Ubuntu and I don't know what other functions it has on Mac. E. Nautilus comes with Clutter Flow , the embedded terminal , and you can add Gloobus Preview .
