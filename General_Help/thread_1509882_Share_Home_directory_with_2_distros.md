### Post by amauk on 2010-06-14
You can share the same /home partition no problem
but use different usernames

Using the same username across different distros (or even diff. versions of same distro) is really asking for trouble

You can setup a central document store if you want to share files across distros
but it's the hidden <dot>blah files in the home folders that you need to keep separate from each other
