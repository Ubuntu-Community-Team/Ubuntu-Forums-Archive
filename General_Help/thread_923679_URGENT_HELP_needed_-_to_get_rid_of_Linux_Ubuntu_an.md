### Post by 73ckn797 on 2008-09-18
Did the computer come with Ubuntu or Windows. If Windows the use the recovery disk that came with the computer. If not then use whichever Windows CD you may have.

Insert the Windows disk in the drive, reboot and tell it to use the whole drive for installation. Make sure to back-up your documents first, either to a flash or cd(s) depending upon the size of all your documents.

I originally had Windows installed as the OEM OS on my Toshiba. I installed Ubuntu as a dual boot. There was a brief insanity to go totally back to Windows (I do not game either), so I used the OEM disk. It completely wiped the drive and installed the package. I am now completely back to Ubuntu as the only OS.
