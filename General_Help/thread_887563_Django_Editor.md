### Post by questron on 2008-09-10
Hi!

You won't find the ide-support you get for building a rails applications. 

I've used Komodo IDE ([www.activestate.com](www.activestate.com)) for a previous django app I wrote, but its code completion features are not that helpful for django. The Python support in NetBeans is not quite there yet, it's promising but still has some rough edges. 

As Komodo IDE is not free (there is a free Komodo Editor), I would suggest to use Eclipse in combination with PyDev and Aptana for django development, until something better comes along.
