### Post by szaka on 2007-06-19
If you used a partitioner on Linux then it made Vista unbootable. Ntfsfix also makes Vista unbootable. Your ntfs-3g version warned about this above. 

If this is indeed the case then you could install the latest ntfs-3g version 1.616 and using the "force" option it will fix the Vista bootability problem.

Don't use ntfsfix and partitioners with Vista! They are not Vista safe yet!
