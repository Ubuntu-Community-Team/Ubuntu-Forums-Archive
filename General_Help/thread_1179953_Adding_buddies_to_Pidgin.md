### Post by Legace on 2009-06-06
> **yonyz said:**
> When I click 'Add Buddy...', it doesn't let me enter an email address or ICQ number. Instead, it shows MY email address, si I'm ending up adding myself to my own contact list. 
This is because Pidgin supports multiple accounts. It asks you to which account you want to add a new buddy :)

Enter the buddy's email in the **username** field, and you should be fine.

### Post by labinnsw on 2009-06-07
> **yonyz said:**
> What if I want to add an ICQ or anything else thay Pidgin supports?

Open the first window (Account Type) and select ICQ
In the windows below (Username, group, First name, last name, Email) Add the details of the buddy you wish to add.

Do the same for any other IM system for which you would like to add a buddy.

This assumes that you have accounts with these other IM systems.
