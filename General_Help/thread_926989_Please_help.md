### Post by Titan8990 on 2008-09-22
In many peoples opinions Ubuntu is a superior desktop OS. I will put it like this:

If you are very into PC gaming - stick with Windows

If you are not willing to **relearn** how to use a computer - stick with Windows

If you never want to have to pay for software again -- get Ubuntu

If you never want to worry about viruses or spyware -- get Ubuntu

If you are interested in learning more about computers -- get Ubuntu


And there is my poorly formatted pros and cons.


Option 2:

Buy the laptop with Windows and then install Ubuntu on it as a dual boot set up. Very easy, takes about a hour.

### Post by jespdj on 2008-09-22
Before you buy the new laptop with Ubuntu, [download an Ubuntu Live CD](http://www.ubuntu.com/getubuntu) and try it out on your current computer - then you can see what it's like.

For almost every kind of Windows software, there's an equivalent piece of software for Ubuntu. For example, there's OpenOffice (which can also work with Microsoft Office documents). There are also a number of different instant messagging programs. There's unfortunately no iTunes for Linux, but there are other programs that work with your iPod (they don't work with Apple's iTunes Store, however).
