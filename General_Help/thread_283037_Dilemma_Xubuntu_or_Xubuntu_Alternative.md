### Post by TiredBird on 2006-10-23
I've used both but I have plenty of ram so I can't tell you whether or not the live install cd uses too much for your system. What I can tell you is that a complete install from the live CD (Xubuntu), including Abiword, occupies a total of 1.3 GB. I imagine that it used more while it was installing but that's what it settled down to.

On another partition, I started with a fresh install, then threw out the things I didn't want and have been adding back those things I use regularly. Some of the heavy-weights are parts of openoffice, (I use 'writer' and 'calc' only), VLC (for watching DVDs), Firefox and Thunderbird. I think I have a pretty usable system and I'm still only using 1.3 GB. (Yes, its the same as a fresh install but I've deleted things I don't use regularly. Just coincidence it ended up the same.)

One thing I would caution you on though. I went with a lighter pdf reader to save space. I think I've tried them all now and with the exception of 'acroread' they all do a poor job of rendering the font. I finally went back to Acrobat. The space it takes was worth it for the quality difference.
