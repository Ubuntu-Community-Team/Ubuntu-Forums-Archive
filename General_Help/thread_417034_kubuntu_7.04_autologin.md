### Post by wjston on 2007-04-21
> **Pathfinder_ said:**
> does anyone know how to enable auto login in kubuntu 7.04? i searched google and these forums and was unable to find. the only instructions i found is to go to system preferences -> login manager and select autologin there. however, as of feisty fawn there is no login manager in system setting.

In Feisty Fawn >System>Administration>Login Window>Click "Security Tab" then check "Enable Automatic Login" and then choose your user name from the drop down arrow. You should be good to go at next boot.
