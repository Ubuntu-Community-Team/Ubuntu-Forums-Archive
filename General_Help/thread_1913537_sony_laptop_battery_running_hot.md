### Post by Nixarter on 2012-01-22
Well, if you are running 11.10 or 11.04, there is a bug that makes the screen run at max brightness, regardless of settings.  Screens use most of the power of a laptop, so if you crank it up, you're going to use a lot more energy, which heats up the battery.  I hope they fix that bug soon, as it is EXTREMELY annoying to have such a short battery life (about 2 hours vs 6+).

That is the only thing I can think of that might be causing it.
