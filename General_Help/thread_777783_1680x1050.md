### Post by Woody1987 on 2008-05-02
If you are using a nvidia card and have the nvidia driver installed, you can configure your monitors with ```
sudo nvidia-settings
```

This has to be run as root so that changes can be written to xorg.conf. In this app you can set the resolution of your monitor upto the max that it can support. This is how i got my 1680x1050 resolution. If you are using an ATI card then you should be able to use their configuration manager.
