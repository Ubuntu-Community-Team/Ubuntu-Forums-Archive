### Post by ibutho on 2008-11-03
Most servers disable logging in as root for security reasons. The only way to change this is by editing the config files of the particular ftp server you are using. You probably don't even need to login as root. You could always tinker with permissions to allow a designated user, access to the files that you wish to work on as root.
