### Post by Mark Phelps on 2012-10-31
I've been using Ubuntu (and other Linux distros) for over five years now on a daily basis, and quite honestly, I do not find it "more natural" or "faster" than using MS Windows.

Every few years, Ubuntu and Microsoft change their desktops -- which then require us to learn "new" ways to do the same thing.  Whether the result is "more natural" to you is entirely a PERSONAL call.  What is "more natural" to one person can easily be "less natural" to another.

As to "faster" -- I've NOT found that to be the case across multiple versions of OSs and desktops.

IMHO, the only "useful" answer to the OP's question is -- use what "works best".  

If you're in a situation where you're obligated to use some MS Windows apps on a regular basis, and those don't work well in Wine -- then the choice is simple: stay with MS Windows.

On the other hand, if Ubuntu is able to run all the apps you need to use on a regular basis, and you don't need to struggle with getting MS Windows apps to work -- then switch over to Ubuntu.
