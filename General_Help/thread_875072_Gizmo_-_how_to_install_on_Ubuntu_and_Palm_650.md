### Post by treesurf on 2008-08-31
I've used Gizmo quite a bit on Ubuntu and I've been pretty happy with it.  You can download a .deb package from the Gizmo website.  Here's a direct link to the download:

[http://download.gizmo5.com/GizmoDownload/gizmo-project_3.1.0.79_libstdc++6_i386.deb](http://download.gizmo5.com/GizmoDownload/gizmo-project_3.1.0.79_libstdc++6_i386.deb)

Just double click on the file once it's downloaded and install it.

I spent much of last year in Hong Kong and using Gizmo was the absolute cheapest way I could find to call friends back in the USA.
