### Post by ajgreeny on 2008-07-12
Use synaptic to get **compizconfig-settings-manager** if you haven't already got it.  There will then ba an additional item in your menu called** Advanced Desktop Effects Settings** where you can set exactly what effects you want out of the huge number available.  It is also worth getting both **compiz-switch** and **compiz-check** from [Forlong]("http://forlong.blogage.de/article/pages/Compiz-Switch").  These are simple script files to do exactly what the name suggests, ie switch compiz on and off alternately, and check if your hardware is set up and able to run compiz in the first case.

After all that, and if you can use the** Advanced Desktop Effects Settings** from the menu, you will possibly find like me that the simple **Visual Effects** tab in **Appearance** show no check mark at all.
