### Post by haqking on 2011-08-07
> **ngno said:**
> anyone of you using ubuntu server or desktop in your PC for your daily work? can you share how stable is it?

You wouldnt use server for your daily work, a server is for a server to provide services and not used as a workstation.

Alot of people use Ubuntu as their Desktop successfully, if you want the most stable then choose a LTS version (which means long term support)

Check your hardware for compatability first to eliminate headaches.

If you do things as suggested in the stickies here for security and using sudo correctly then your system should run fine and stable.

no need for AV software either so no worried about infected files unless you share with windows users then it might be worth running some AV software to scan files you will share with them.

other than that it should be fine, try a Live CD/DVD or USB version before installing and read any release notes of the release you choose
