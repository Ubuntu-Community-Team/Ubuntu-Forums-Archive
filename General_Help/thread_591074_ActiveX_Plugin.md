### Post by SkitzoSekter on 2007-10-31
Hey guys i dont know if this helps or not and if your problem has been solved but il mention this anyways for the next lost soul who comes along: There is a plugin for firefox called MediaWrap [https://addons.mozilla.org/en-US/firefox/addon/1879]("https://addons.mozilla.org/en-US/firefox/addon/1879")
It wraps ActiveX into the plugins and then it should automatically start playing using whatever player you've assigned for that particular extension (asx, wmv, etc.). I know this isnt the perfect solution but the best ive come across so far.
