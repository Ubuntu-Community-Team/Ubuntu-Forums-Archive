### Post by dmizer on 2006-03-19
if you don't have any tool bar, something might be wrong with your install then.  if you are not using opera browser or skype or a few of the text editors, the tutorial you pointed me to isn't needed ... just the wiki entry.

i followed the wiki entry at: [https://wiki.ubuntu.com/JapaneseInputHowToInBreezy?highlight=%28japanese%29](https://wiki.ubuntu.com/JapaneseInputHowToInBreezy?highlight=%28japanese%29)

i haven't had any problems.

at the end of the wiki entry, it says to type:
```
im-switch -s uim_anthy
``` in a terminal.  you have to do that to make the anthy control panel show up, then you'll have to configure the tool bar to use japanese.
