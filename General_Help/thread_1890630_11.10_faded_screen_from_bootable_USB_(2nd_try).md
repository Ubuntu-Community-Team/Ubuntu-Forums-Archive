linux    /casper/vmlinuz  file=/cdrom/preseed/ubuntu.seed boot=casper iso-scan/filename=${iso_path} quiet  acpi_osi=Linux splash --

### Post by Advait on 2012-01-01
OK, I FINALLY got the problem fixed. Yay! Turns out the solution is pretty simple but I figured it out only thru trial and error. The solution is at the ACER boot screen I hold down the blue function "Fn" key and hit the "Increase Brightness" key three times quickly. This has to be done *before* the ACER boot screen goes away. That's the key point. If I hit the "Increase Brightness" keys after the ACER boot screen goes away, then Ubuntu never boots. I only figured this out by trial and error and some good luck. I hope other people having this problem can find this post to see if this solution works for them.

So now I'm able to run Ubuntu 11.10 from my USB drive. I'm a happy camper!

Thanks as always to the wonderful Ubuntu forum community for your help and suggestions. Cheers,

Advait
