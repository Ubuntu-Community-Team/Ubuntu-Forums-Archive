### Post by wt8008 on 2011-02-26
If you are interested in pinyin input, the easiest to install is the pinyin input method in ibus.

System->Preferences->Keyboard Input Methods

Under the `Input Method' tab select Chinese->Pinyin and add. Then you will be able to input Chinese using pinyin.

Other methods are also available.

If you don't like the iBus method, you can also try scim or fcitx other popular Chinese input tools. There is also a sunpinyin that runs under iBus available. You may want to try them all to see which one you like more.
