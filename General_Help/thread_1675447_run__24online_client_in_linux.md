Some ISP ( internet service ) use 24online client to make it customers  access the net,

to make this application "24online client" run under linux do the following:

1.download  the application from this link
[http://cyberoam.com/cyberoamclients.html](http://cyberoam.com/cyberoamclients.html)

choose linux os

2.move the file you download to home\"username"  folder

3.extra the file from terminal :
```
tar xvfz CyberoamLinuxClient.tar.gz  
```4.enter the folder "crclient" from terminal :
```
cd crclient
```5.to make login from terminal :
```
./crclient -u username 
```6.to make logout  from terminal :



```
./crclient -l
```i hope this information is useful .
