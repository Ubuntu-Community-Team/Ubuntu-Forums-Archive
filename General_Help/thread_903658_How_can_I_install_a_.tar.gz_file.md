### Post by aysiu on 2008-08-28
System > Preferences > Appearance

Then drag the .tar.gz file to the Appearance Properties window

click and drag the configure file
click and drag the "make" file
click and drag the "make install" onto the terminal

DONE!

make sure you have build essential and the files show the whole directory.

this is for programs NOT themes

for themes you just click and drag the tar onto the theme window and it automatically installs
