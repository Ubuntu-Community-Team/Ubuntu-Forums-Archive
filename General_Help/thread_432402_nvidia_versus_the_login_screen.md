### Post by mykrob on 2007-05-04
fixed it.. I just redid the xorg.conf file using a command that was listed in the config file. This apparently fixed problems with the modes listed in the old one from some botched nvidia driver installs. I then added the necessary lines to get the nvidia driver working on this card (nVidia Gefore Go mobile chipset)

Anyway, here is the new one for anyone else having a similar issue. This works for beryl too.
