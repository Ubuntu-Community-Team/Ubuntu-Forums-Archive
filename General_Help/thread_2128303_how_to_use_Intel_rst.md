### Post by oldfred on 2013-04-07
The Intel SRT is Windows only.

Many that use Ubuntu more than Windows just turn SRT off and install Ubuntu onto the SSD. That is usually large enough to be a / (root) with /home or data on rotating drive.
Those that like Windows turn it back on after installing Ubuntu. A few with larger SSDs seemed to install Ubuntu on SSD and re-enable SRT as that does not use a lot to cache Windows. Not sure if it is adjustable or not?
