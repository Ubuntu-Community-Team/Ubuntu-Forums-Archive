### Post by kaibob on 2008-08-17

I've run Ubuntu both with and without preload, and things seem to be just a bit quicker with it installed. So, I use it. There are some old threads on preload, and, like me, most people are ambivalent as to its benefits. 

Preload is easy to install and later remove through synaptic, so install it and decide for yourself. You probably should have at least 1 GB of memory. I've never heard of it doing any harm, but you should always have a backup of your drive.
