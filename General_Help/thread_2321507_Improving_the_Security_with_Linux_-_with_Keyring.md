**Improving the Security with Linux:**

Most of the Linux users are annoyed with this feature. But keyring has  been there for a special purpose. Once you log in to the machine, the  machine requests for a keyring or connection to a network. And then you  must enter the keyring password. Mostly Linux users are tempted to  disable this feature, by making it an empty keyring password. This is  where you go wrong, because the unencrypted information including  passwords is available to others. Keyring encrypts sensitive password  when data is sent over the wire.


Is this something we should be using? If so, how do we manifest its operation?
