### Post by sudodus on 2015-02-18
Do you want to put text into the picture itself, or do you want to put the file name into the display (but keep the picture file clean)?

***Text into the picture itself:***

Even if you don't need everything in ***Gimp***, it is easy to learn (and find) what you need, and maybe gradually learn more tools, that are available in Gimp.

An alternative is ***ImageMagick***, that can be used for batch processing (to treat many files with the same or similar things).

***File name into the display:***

You can try ***feh*** for this purpose. It is very powerful for screenshows and there is an option to show the file name.
