### Post by migs73 on 2010-06-01
1) the most ram you should use is about half your physical ram. Vbox warns you of possible problems with the host OS if you exceed this. Don't see why it should be a problem if your host (Ubuntu) is memory light, but better safe than sorry!
2) Vbox allows for an expandable virtual drive, so you can start of at say 10Gb but if you exceed that the drive will expand to accommodate it.
