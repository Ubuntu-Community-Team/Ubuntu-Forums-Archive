### Post by ago on 2008-05-07
The only "influence" wubi might have on windows is if you do not shut down cleanly, which is not really a wubi issue per se' (hard reboot can always corrupt filesystems). Those issues can normally be fixed running chkdsk /r in windows. There are of course many other possible reasons, but none directly related to wubi.
