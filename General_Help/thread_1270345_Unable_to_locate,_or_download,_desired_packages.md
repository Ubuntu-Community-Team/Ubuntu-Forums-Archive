### Post by Zill on 2009-09-19
Anquestory:  Ubuntu 6.06 LTS (Dapper Drake) reached [end-of-life]("https://wiki.ubuntu.com/Releases") on July 14 2009 and so is no longer supported.

Please post details of the specification of your PC so we can establish if it *should* run the latest versions of (K)Ubuntu.  You may also find the following link of interest as it lists the minimum requirements:

[https://help.ubuntu.com/community/Installation/SystemRequirements]("https://help.ubuntu.com/community/Installation/SystemRequirements")
(Although this specifies Intrepid (8.10) the later versions have similar requirements).

If your PC *does* meet the minimum requirements then please advise details of the problems experienced when installing the later Kubuntu version.  Specify exactly which version of Kubuntu you are installing.

Regarding KolourPaint it appears that the correct program to install for Kubuntu Jaunty (9.04) is kolourpaint4, not kolourpaint.  This is because Jaunty uses KDE4, rather than KDE3.

[http://packages.ubuntu.com/ko/jaunty/kolourpaint4]("http://packages.ubuntu.com/ko/jaunty/kolourpaint4")

If you need further explanation of Linux software installation (yes, it is different to Windows!) then take a look at this link:

[http://www.psychocats.net/ubuntu/installingsoftware]("http://www.psychocats.net/ubuntu/installingsoftware")
