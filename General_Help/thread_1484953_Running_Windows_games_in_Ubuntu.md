First of all there is category dedicated to wine related stuff right here on the page. 
Second thing is that if you have questions related to games and wine the best first step is to visit the appdb of wine. 
(i.e. Virtual Pool: 
[http://appdb.winehq.org/objectManager.php?sClass=application&iId=5421]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=5421") )
Third thing you have to keep in mind that you should install the game via wine again. Some old games might work without entries in the windows registry (wine keeps its own and doesn't have access to an external from i.e. another Windows installation) but generally they don't.
