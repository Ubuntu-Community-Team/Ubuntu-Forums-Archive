### Post by MAFoElffen on 2023-08-11
Try to use the hotkeys <Cntrl><Shift><->...

<Cntrl><Shift><+> Enlarges the screen
<Cntrl><Shift.<-> Shircks the screen.

Note that the <Cntrl> key needs to be the first keys you start holding down to start that key combination...
