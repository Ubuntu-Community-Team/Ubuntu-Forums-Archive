### Post by Alan.Brown on 2012-04-29
You can get help on Firefox from the Mozilla forums - 
**[http://forums.mozillazine.org/viewforum.php?f=38](http://forums.mozillazine.org/viewforum.php?f=38)**

IRC is the Internet Relay Chat system - a collection of networks that one can use for connecting with other people for chatting.  See - **[http://en.wikipedia.org/wiki/Irc](http://en.wikipedia.org/wiki/Irc)**

You would need a chat client such as **XChat. **  Then look for the FreeNode network then enter the command **/join #firefox**    This will connect you to a 'room' where you will be able to get help.

If you are not familiar with IRC I would try the Mozilla forums.

With respect - you sound like a Noob.   Learn to rely on Google and Wikipedia.

In Google try searching for **"Firefox forums"**.    In Wikipedia you can search for **"IRC"** for information on the chat networks.

Hope this all helps.

Alan
