### Post by p_quarles on 2008-04-28
The icons for KDE are stored in /usr/share/icons/default.kde. This directory is a symlink, so entering it will put you in the directory of whichever icon set is set as default in your distro. Find the offending icon, and simply change it out with one you want. ;)
