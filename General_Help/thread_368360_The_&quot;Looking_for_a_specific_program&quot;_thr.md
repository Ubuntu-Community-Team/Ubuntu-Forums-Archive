I'm not aware of any program which does this; the difference between Linux and Windows is that the command line is much more powerful in Linux than in Windows. There may not be the same demand for this type of application in Linux as there is in Windows as a lot of people will simply write their own scripts to make backups; its a reasonably simple process, easily customised and executed and can also be run on other machines which may not have a GUI. If you are not inclined to write your own script, you could look for a ready made script which will do what you want.
