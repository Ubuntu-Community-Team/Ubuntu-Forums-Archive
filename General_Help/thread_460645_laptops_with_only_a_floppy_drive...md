### Post by phidia on 2007-05-31
Older versions of debian can still be installed with floppies [http://www.debian.org/releases/woody/i386/ch-install-methods.en.html#s-create-floppy](http://www.debian.org/releases/woody/i386/ch-install-methods.en.html#s-create-floppy)

 If you had network capability you might be able to install that way. It does seem like this is an old machine so it's unlikely that newer versions of debian would work.
