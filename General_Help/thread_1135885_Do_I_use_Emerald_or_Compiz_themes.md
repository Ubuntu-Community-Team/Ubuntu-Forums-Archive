### Post by UbuntuNerd on 2009-04-24
If you're using "Emerald" as your Theme manager then download "Emerald Themes". if you go to Gnomelook.org and click on the compiz themes you will find out that some of them are actually emerald themes and you will need to have the effects activate for them to work. If you don't see any changes after installing a Emerald theme type this in a terminal:
```
emerald --replace
```
