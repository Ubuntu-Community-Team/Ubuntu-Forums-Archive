### Post by zi99y on 2006-04-15
I got this working by commenting out the modeline in xorg.conf like so:

```
Section "Monitor"
	Identifier	"Generic Monitor"
	Option		"DPMS"
#	Modeline	"1280x800@60" 83.91 1280 1312 1624 1656 800 816 824 841
EndSection
```

No need to change anything else, just put the # at the start of the line, and restart Xorg, voila!
