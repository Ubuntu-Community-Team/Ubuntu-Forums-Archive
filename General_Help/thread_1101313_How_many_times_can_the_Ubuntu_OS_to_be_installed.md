### Post by Anthon on 2009-03-20
I don't think there is a limit. 

Reading a CD in the CD drive does little to detoriate the surface, so there are no practical physical limitations (as long as you are careful handling the CD going from computer to computer).

This is free software. After installing it goes to the Ubuntu site to look for updates (if you are on the internet), but it doesn't ask for a license key and activation like some OS-es do.
