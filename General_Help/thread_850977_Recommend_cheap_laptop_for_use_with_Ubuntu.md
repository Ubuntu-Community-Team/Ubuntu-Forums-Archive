### Post by editrix on 2008-07-06
If you want cheap, get a used one. You're actually better off, because if the hardware is a little older, there's more likelihood there are Linux drivers for it. Also, with an older model, there'll be more posts about it around the web, and you can research how well it will work.

Check the Local Linux Users Groups in my sig. There may be one near you, and if there is, there's almost always someone with spare hardware willing to sell.

If you search "laptops" on Google Linux (also in my sig) you'll find sites that review compatibility.

### Post by StumpyMcDonut on 2008-07-07
Depending on what you want it for, you could always try an eeePC - although the keyboard is supposedly very tricky to use. Plus as they come preinstalled with Xandros, the hardware should be supported. Only problem is you'd need a usb cd drive

And yes, make sure wireless is compatible, or you'll just get headaches. And remember that a lot of the nvidia support for Linux doesn't exist any more - ATI/AMD or Intel are currently doing much better than them.

You'll be hard-pressed to find a laptop that isn't from a major manufacturer for £300 if you're planning on wiping vista/xp from it when you buy (considering the licence will be about £30/£40 or so for small OEM's), so if you really don't want one of the dells go pre-owned

### Post by furbuntu on 2008-07-08
Adding to what Stumpy's post, if you want an eeepc you should go with a 4G+ variety (the 2G is too small for a full install of Ubuntu without a workaround).

You might also want to consider the Everex Cloudbook or gBook. They're cheap and Windows free. They come with gOS, which is based on Ubuntu, so an Ubuntu install shouldn't be difficult. They're not exactly powerhouses, but they should get the job done.
