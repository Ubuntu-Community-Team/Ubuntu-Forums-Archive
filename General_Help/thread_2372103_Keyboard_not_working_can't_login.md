### Post by dragonfly41 on 2017-09-21
Interesting problem.   You don't give the keys which do not work.
However this is an idea which should give you a path to get back into your main OS.

(a) Use a LiveUSB and startup using "Try Ubuntu" option (no keyboard required so far).

(b) You may need to install Florence on LiveUSB as virtual keyboard.

(c) When you have virtual keyboard running use this procedure to change your user login password so that it does not require the two disabled keys.

[https://www.howtogeek.com/howto/linux/reset-your-ubuntu-password-easily-from-the-live-cd/](https://www.howtogeek.com/howto/linux/reset-your-ubuntu-password-easily-from-the-live-cd/)

or you might choose to create a new user account with new password.

(d) An alternative approach would be to disable password for usrlogin (until you get your keyboard working).

(e) Reboot and login to your OS with new details.
