### Post by Rhubarb on 2009-08-17
There's a good application that might be able to get this working for you.

```
sudo aptitude install devilspie
```

> This tool will find windows as they are created and perform actions
on them, such as resizing, moving to another workspace, or pinning
them to all workspaces.

I've used it before to adjust the transparency / size / window decorations of specific applications / windows that are on-screen before.

devilspie when used with a script might be able to get this working for you.
- You'll probably need to consult the devilspie man pages / google for more information with regards to the proper usage of it.

I hope this is of help to you.
- Rhubarb
