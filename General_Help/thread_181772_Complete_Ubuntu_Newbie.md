### Post by IYY on 2006-05-24
Your machine is certainly fast enough to run Ubuntu, but you may have to make some tweaks. The first thing you should consider is switching your window manager. Currently you have Gnome installed, and that's a very heavy piece of software. The alternatives are: XFCE, IceWM and Fluxbox. 

The advantage of XFCE is that it's easier to set up, and has more GUI desktop stuff built in like printing, menu editing, etc. It also has its own file manager and some other handy programs. The disadvantage is that while much lighter than Gnome, it's still quite heavy.

IceWM and Fluxbox are pretty much weightless, but may take a bit more configuration (for example, you'll have to edit the menus manually in a text editor). If you want something that looks and feels like Windows (or Gnome, KDE) use IceWM. If you want a new way of doing things, try Fluxbox.

There is a thread somewhere on this forum to make booting faster, but Ubuntu will still be quite slow in booting on a machine like yours. It doesn't bother most people, since Linux users tend to keep their machines on for loooong periods of time (months).

For the record, I am happily running Ubuntu on a machine with half the specs.
