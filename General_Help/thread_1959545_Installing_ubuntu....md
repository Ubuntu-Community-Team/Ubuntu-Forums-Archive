### Post by callmemahavir on 2012-04-16
I see lot of threads specifying problems while installing ubuntu. So I started this thread to make clear everything.

1. Partition is a task of dividing the disk into parts were each partition has its own filesystem type. The filesystem type is a way how files and folders are stored in partition.

2. Partitions are edited during installing OS like windows or ubuntu. The person have to know what the person is doing clearly when editing the partition. So complete knowledge is required about partitioning to modify the partition table.

3. Altering an existing partition may lead to inaccessible to the partition data.

4. While creating linux partition the user may choose home folder as a separate partition so home partition is safe even if the OS is corrupt.

5. Before installing the OS along with other partition it is needed to take backup all data in existing partitions.

6. The person should be clear in each step while installing OS.

7. While installing the OS ensure that you are modifying only empty partitions.

8. Also you may create bootable disk while installing OS. So any problem arises you may use this disk to boot.
