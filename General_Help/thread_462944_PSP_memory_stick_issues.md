### Post by kostkon on 2007-06-09
Now about deleting a file in one go maybe if you go to *Nautilus* preferences (open a *Nautilus* window and go to **Edit -> Preferences**), click on the **Behavior** tab and activate the **Include a Delete command that bypasses Trash** option.

Thus, the next time you want to delete a file in *PSP*, if you right click on the file you will see in the menu a **Delete** option in addition to the standard **Move to the Deleted Items folder**; I think if you delete the file selecting the **Delete** option,* Nautilus* will not put the file in the *.trash* folder but will delete it right away.

I don't know if it actually works, but you can give it a try.
