### Post by hamotahari on 2011-02-13
Salaam;
I don't know about what can be done to change &#1662;, but for no-space break, please go:

[LIST=1]
[*]System->Preferences->Keyboard
[*]Select **Layout** tab
[*]Choose **Options**
[*]Expand "**Using space key to input non-breaking space character**"
[*]Choose "**Zero-width non-joiner character at second level**"
[*]Press **Close**!
[/LIST]
