### Post by thegreenblob on 2008-12-09
You could try using a lighter weight desktop environment like XFCE.

In the terminal:
```
sudo apt-get install xfce4
```

Then when you reboot at the login screen select "session" then choose XFCE. It's not as pretty as gnome but it should use less memory. If you don't like it you can always choose "sessions" again and change it back to gnome.
