### Post by lovinglinux on 2010-03-13
I also recommend KeepassX. Nevertheless, I recently switched to Kwallet with [KDE Wallet password integration](https://addons.mozilla.org/en-US/firefox/addon/49357) extension for Firefox. It works like a charm, although I'm not sure it would be very useful on a Gnome system, since Kwallet only offers integration with KDE applications, while Gnome uses the Keyring. There is also a [Gnome-keyring password integration](https://addons.mozilla.org/en-US/firefox/addon/8737), but it hasn't been updated since August.

### Post by lovinglinux on 2010-03-13
> **aviramof said:**
> is there a toolbar extension to linux to any password manager software you mentioned?

I guess what you want is an offline password manager that integrates with Firefox through a toolbar, like [Roboform Toolbar](https://addons.mozilla.org/en-US/firefox/addon/750) right? If that is the case, then I'm afraid you won't find it. Most password managers allows you to open a web site and enter the password automatically, but you need to open the software first. Revelation offers a gnome panel applet, which makes this process a lot easier.

If you don't mind storing your passwords online, then there is [LastPass Password Manager](https://addons.mozilla.org/en-US/firefox/addon/8542) and [myVidoop Plugin Beta](https://addons.mozilla.org/en-US/firefox/addon/6130). I'm not sure if the last one works with FF 3.5.

Anyway, what do you want to achieve?
