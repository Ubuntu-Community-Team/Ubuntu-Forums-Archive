I am missing some fonts when making bullets. Any idea how to fix this?

This is the procedure I followed :-

1. Open up Abiword. 
2. Type a list such as "one, two, three" with each word on a separate line. 
3. Highlight the whole thing. 
4. Go to Format > Bullets and Numbering 
5. Use Type > Bullets from drop-down list
6. Of the 10 bullet styles given in Abiword, only some are resolving. Hand List, for example, works. But Dashed List, Implies List and others all the others given something weird like a square with some numbers or something close like that.
