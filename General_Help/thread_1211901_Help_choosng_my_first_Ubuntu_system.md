### Post by TeamJ on 2009-07-13
well you will want to start with the latest 9.04 I think, but you may want to choose the netbook remix.

about 0.5gb RAM
intel ATOM 1.6ghz
screen at least 1024x600
Any Graphics

about 1gb RAM
processor Dual Core 1.6ghz or perhaps single core at least 2.8
screen at least 1024x768 or perhaps 1280x720
NOT Intel integrated
about 80gb HDD

with that proce range I suggest a netbook, in which case get the UNR

### Post by t0p on 2009-07-13
> **Ice92 said:**
> 
So firstly I need to purchase a laptop, I have £300 and I've heard that the Asus Eee PC is a good choice, can you please advise, 

I just want a laptop with good memory, a decent processor and about 80-120GB HDD ( not a problem if bigger) 


The EeePC is not a laptop. It's a netbook, which is a very different animal.  Netbooks are much smaller than laptops.  And I advise you get a laptop with the largest screen you can afford.

Unless portability is a big deal to you.  In which case, an EeePC would be a wonderful.  I have one of the old EeePC 701s, and I love it.  But I don't think I'd like it as an only computer.

Someone else will have to advise you on models and manufacturers.  I'm not that up on new hardware.  But I *would* advise getting one with an Intel graphics chip.
