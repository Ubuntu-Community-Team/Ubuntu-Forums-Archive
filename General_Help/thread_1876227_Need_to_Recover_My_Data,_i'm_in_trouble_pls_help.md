### Post by hansdown on 2011-11-05
Hi, miasmablk.

You need a live CD; ubuntu, mepis, etc.

Running the live cd, open a terminal,

```
ctrl+alt+t
```.

Enter,

```
gksudo nautilus
```

You should be able, to drag and drop your data, to a usb, etc.
