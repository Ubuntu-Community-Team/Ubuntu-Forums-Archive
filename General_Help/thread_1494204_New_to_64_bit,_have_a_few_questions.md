### Post by jerome1232 on 2010-05-26
You don't have to use 64 bit.

It says that because there can be issues getting certain proprietary programs to run. Although I haven't run into one yet.

Most 32 bit only programs should work just fine under a 64 bit os (32 bit only programs you'll run into are not going to be open sourced, they will be proprietary)

64 bit offers faster number crunching (things like encoding/decoding, compression/decrompression, encryption/decryption) will generally go faster, and you can address more than 4 GB's of ram.
