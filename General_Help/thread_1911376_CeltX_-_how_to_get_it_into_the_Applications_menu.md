### Post by stoneage on 2012-01-18
You don't say which version you are using, so this may or may not apply.

Open a Terminal and enter > alacarte

From there you should be able to figure it out. Select the menu you want to add it to on the left and click the 'New Item' button on the right. 

Where it says command, browse to the CeltX executable.


If you are using 11.10 the procedure is possibly different.
