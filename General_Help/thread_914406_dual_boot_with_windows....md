### Post by Therion on 2008-09-08
You CAN install Windows on a previous Linux installation, but it's a royal PITA (Pain In The A--) because Windows wants to overwrite grub and render itself the sole OS on the bootable drive.

You might want to consider using Windows in a Virtual Box (unless you need to dual boot for gaming or something) which would be the easiest thing to do; or you could install Windows and THEN install Ubuntu and dual-boot (semi-PITA option).  Last option would be to wait for someone who can walk you through installing Windows over an existiing Linux install (and that's going to be a maaaaajor PITA).
