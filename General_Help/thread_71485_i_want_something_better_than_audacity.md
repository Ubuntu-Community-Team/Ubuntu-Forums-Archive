### Post by matthew on 2005-10-03
I've been looking into doing this sort of this for some time as well. Audio/Video recording and editing is an area that is still weak in Linux, but it is developing gradually. I can't answer all your questions, but here are some ideas that may help with a couple of them.

This is a thread where I asked similar questions. I think you will find mp3gain useful for the volume normalization you want to do, even though it is a command line program it isn't difficult. I have used both audacity and rezound and think rezound is a bit more refined. I have to run it from a command line initially by first doing "killall esd" and then "rezound" but then it works pretty well. You can give it a look anyway and see what you think. Both programs are available in synaptic.
