### Post by Habitual on 2017-08-23
You could install Thunderbird on the system with Outlook (does it convert Outlook Express?, IDK!!)
and let it do an import from Outlook, then copy in totality the %mozilla% directory from Windows to the intended Linux Host,
and fire up Thunderbird.

[Thunderbird - Importing Outlook Emails and Addressbooks ]("http://support.real-time.com/tbird/outlook_import.html") discussing getting Contacts into Thunderbird.
Getting the Windows hosts' %mozilla% directory over to Linux for use there is all that remains.


Good Luck.
