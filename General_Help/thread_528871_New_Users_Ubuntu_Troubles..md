With older hardware one thing to think about is either using Dapper or XFCE or both (Xubuntu 6.06).

One thing I've seen posted somewhere is older kernels and libraries tend to be a little more "back compatible" with older hardware which hasn't been used for a while or whose drivers have been deprecated from newer releases.

Also, a lighter weight windows manager tends to improve performance on older hardware.

Other than that, good luck.
