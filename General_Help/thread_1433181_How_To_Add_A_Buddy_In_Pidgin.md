### Post by curtissthompson on 2010-03-18
Turns out this is actually a feature of the Evolution Integration 2.6.2 plugin in Pidgin. I've had the plugin enabled before without having this issue, so I'd imagine it must've been an updated version of the plugin that caused the Add Buddy prompt to be different.  Disabling the plugin resolved the issue and returned to the normal Add Buddy prompt.  Thanks for the help!
