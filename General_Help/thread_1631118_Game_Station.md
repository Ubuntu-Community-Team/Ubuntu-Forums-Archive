### Post by eric3_14159 on 2010-12-23
A few ways occur to me. Possibly the simplest for you to set up would be to install Ubuntu, and in addition to root and an account for you (for standard system maintenance), create an account which will be used by the kids.

This account should be set to automatically log in, with no password. Its theme should probably have large fonts. Remove one of the panels and make the other one much bigger, and delete all panel applets except for buttons for games you have installed. Finally, lock the buttons and panels in place so that they won't accidentally get dragged around.

Ideally, you'd probably want to use one of the packages they often use in libraries that cause any changes to be reverted on logout, by using a system image of some kind. I don't personally have any experience with these, but "ubuntu revert to system image" would probably be a starting place.

There are lots of great games of this type in the Ubuntu Software Center. This is a great idea and I hope you are successful!
