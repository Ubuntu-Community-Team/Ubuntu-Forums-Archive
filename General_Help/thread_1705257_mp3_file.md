### Post by flipper T on 2011-03-11
are you sure its a mp3 file

linux and windows seem, to me, to handle file names & types in 2 distinct ways...

1 windows : if you rename a files extension, windows assumes that this renaming has changed the filetype

2 linux : changing or deleting an extension name of an audio file has no effect ...try changing example.mp3 into example.flv by renaming it...it remains and is treated as an mp3 file.

i would advise that you import you "mp3" file into audacity and export it as a mp3...this should ensure that it is what it says it is
