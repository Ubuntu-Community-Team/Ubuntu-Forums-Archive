### Post by ram2500 on 2011-01-02
I like to add my own pictures for the login screen as well, instead of the default purple background, which, to me, looks grainy. To do this, I use Ubuntu Tweak (it can do many other things as well...it is similar to TweakUI). From Ubuntu Tweak, under Startup on the left-hand pane, choose "Login Settings". Scroll down in the right pane to unlock (need admin password), and then click on the box with the background, find a picture of your choice, and it will set as the default background. As for making text, I would use GIMP; I am not sure if there is a utility for creating text-only backgrounds. 

You can also disable listing user lists through Ubuntu Tweak, in the Login Settings pane. Unfortunately, you won't be able to seamlessly integrate the login dialog into the image, unless you know how to do so.
