### Post by Nathan_M on 2008-08-08
I recommend you look into making a customised ubuntu CD (or DVD)... that way, if you're using the same hardware for all the computers you build, you can install ubuntu and all the required drivers in one easy step. Make sure you're allowed to distribute any required proprietry drivers. You could even go a step further and make a hard drive image, so you can just churn out lots of identical drives... I'm not sure if hard drive images can replicate partitions though.

The most difficult thing for people new to ubuntu (or any linux) is getting their hardware working. If you're providing printers, make sure it works for the user on their new computer. If not, make sure drivers for all common printers are installed, so the users can just plug and go. Same for other hardware too, as far as possible.
