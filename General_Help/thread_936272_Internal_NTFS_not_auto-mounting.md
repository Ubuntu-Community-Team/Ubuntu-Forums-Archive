### Post by Elfy on 2008-10-02
Make a folder to mount it in - I'll use ntfs - if you want to change it then make sure both folder and fstab match. If you want it to show on desktop leave /media as it is - if you don't wnat it to show then change /media to /mnt

```
sudo mkdir /media/ntfs
```

Bcakup fstab and open for editing

```
sudo cp /etc/fstab /etc/fstab.0210
gksudo gedit /etc/fstab
```

Add these to end of the file, save and close

```
##dev/sda1
UUID=6E64174A64171485 /media/ntfs ntfs-3g defaults,locale=en_US.utf8 0 0
```

Check that it mounts ok

```
sudo mount -a
```
