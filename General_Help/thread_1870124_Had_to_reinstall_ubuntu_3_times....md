### Post by decoherence on 2011-10-26
it's hard to suggest anything without more specific information about your problems. do you have other posts on this forum that go in to detail about each problem? i know reporting each problem can be a hassle when you're in the middle of a bunch of them :) 

here is some general info and questions

1. how did you install the nvidia driver? Did you use the "Additional Drivers" program that Ubuntu comes with? Nvidia driver is not installed by default due to Ubuntu not being able to support it (it is not open source -- ubuntu can't vet the quality of the driver.)

2. what were the other programs you tried to install and how did you install them? where did you get them?

3. what sorts of things did you do to try and restore the sidebar? fresh installs are rarely necessary but can be faster than trying to figure out how to fix something when you aren't familiar with the system.

4. do you recall at what point ubuntu's boot process was stopping?

For the record, I have been using Linux for many years and have been disappointed with recent Ubuntu releases, which in my experience have been increasingly buggy due to using immature software that doesn't work as well as what it's replacing. Mind you, most things seem to work fine for most people. I have a knack for finding bugs.

If you aren't concerned about running the very latest software, you might check out Mepis. It's a complete desktop system based on Debian stable, which is about as rock solid as you can get.

Also keep in mind that things like bad RAM can cause all sorts of strange behavior.

Good luck! And remember, you have options!
