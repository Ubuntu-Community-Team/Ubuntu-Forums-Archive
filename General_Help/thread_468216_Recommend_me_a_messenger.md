### Post by hikaricore on 2007-06-09
Problem is that getdeb's packages for pidgin are built for feisty, the OP is using dapper.

However a quick google search brought up something useful:

[http://drsjlazar.blogspot.com/2007/05/pidgin-for-dapper.html](http://drsjlazar.blogspot.com/2007/05/pidgin-for-dapper.html)

The owner of the blog suggests installing the gaim2 beta to make sure all dependencies are installed as the linked package was not built properly and will not check dependencies.
I have no doubt that it will work as long as all required libraries are installed, but just wanted to warn you, proceed with caution.  ^_^
