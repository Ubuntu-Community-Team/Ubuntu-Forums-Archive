### Post by y-lee on 2008-06-29
First of all sorry about all your problems. Some issues take a while to sort out and I know it can be frustrating. 

anyway...


> 1. There can I get real help in a timely manner (like within a week) to solve my problems other when using payd support from Cannonical $250 in addition to my book (is not linux free?)

You can get real time support on IRC,  install xchat or an irc client and on the freenode channel join ##beginners-help  or maybe #ubuntuforums-beginners (the latter is not as busy but it also is not really for support questions) and ask for help there.

> 2. Can I be GUARANTIED that after I will put substantial efforts I can make Ubuntu at least as stable as Windows XP pro provided that it will provide the same functionality (all types of sound including USB microphone) works, no crashes etc)

That depends upon your hardware. Ubuntu is more stable than windows for me and was far easer to install. There are no guaranties in life nor any with windows. You probably have never tried installing windows on a lot of machines. That too can be problematic.

> 3. Is it possible that getting to previouse version on Ubuntu can give me better stability?

It is possible I suppose. If you are using hardy now then maybe even. hardy was just released and I know it still has some problems for some people.

> 4. Do I need to built my oun kernel to make it stable and functional and where I can get the best info about doing it? 

I seriously doubt it, there are occasions where that may be necessary but it is an advanced thing to attempt to do. I haven't tried so I am not the one to ask for advance on doing so.

> 5. Is there a special hardware which can guaranty that I will be deprived from all those trouble? 

Ubuntu works on most hardware if you have any particular device google it or search here or ask here or on irc and see if there are any problems using it and linux.
