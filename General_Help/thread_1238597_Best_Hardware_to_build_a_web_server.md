### Post by P4man on 2009-08-15
there is no such thing as best hardware. And its impossible to even attempt to scale a machine when all we know is that its "5 website". You can imagine a static HTML site from a local dentist is something else as hosting wikipedia or google.

if you're already hosting those sites, you should have an idea of the cpu and ram load, and the required bandwidth. Use that as a starting point.

But honestly, if you have to ask, then the right answer is probably: get a hosting contract.
