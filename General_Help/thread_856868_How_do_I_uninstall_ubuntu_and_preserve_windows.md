### Post by cmnorton on 2008-07-11
If you don't want to preserve anything about the Ubuntu partition, then one way to do this is to use Windows' Recovery console to put back the Windows MBR. There might be more to do, and searching these forums for subjects like dual boot and windows boot recovery will be of additional help. Once you get Windows booting, you can reformat the partition onto which you installed Ubuntu.

Microsoft's documentation on Recovery Console should help you. You'll have to invoke it from the Microsoft installation media, given you probably cannot access it from the hard drive. Again, searching these forums should help you.
