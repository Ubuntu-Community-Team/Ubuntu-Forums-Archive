### Post by TheFu on 2014-09-25
Don't use a clipboard, use the X/Windows x-buffer only - that is select and paste using just the mouse.

Select something with the left mouse button, then use the center button to paste it. No formatting.

There is only 1 buffer, so you can't stack "selects" - but this is 1000000x easier than other options.
