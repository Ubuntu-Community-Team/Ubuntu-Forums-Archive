### Post by t4thfavor on 2009-06-29
Something akin to access forms for OpenOffice is what you are looking for. The first place to start would be with the access part of OpenOffice called Base, I don't think its installed by default.

As for MS works, I don't think the OOo ones will be compatible. If you want to get really snazzy learn basic PHP, and mysql (or even save it in a flat file (csv).
