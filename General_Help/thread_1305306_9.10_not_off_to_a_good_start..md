### Post by rednzl on 2009-10-31
Hi there,
I've found a solution to use desktop effects on RV250 and solve the black desktop problem in 9.10. Dont remember where I've found it but..
1. Open a terminal and paste this: 
```
sudo gedit /etc/modprobe.d/radeon-kms.conf
```
2. add these lines to the new file:
```
##enable KMS Radeon
options radeon modeset=1
```
3. save and close text editor
4. execute this line in the terminal:
```
sudo update-initramfs -k `uname -r` -u
```
5.reboot.

It works for me, on a compaq nx7000 with a Radeon RV250 Mobility FireGL 9000
I can use several Compiz effects like rotating cube, drawing with flame, etc.etc. 
The only problem I have now is that after an undefined time, all characters below icons on desktop or below the menu items start to degrade until disappear completely. 
Still looking around for a solution, but I think it could be a card failure...

### Post by rednzl on 2009-11-02
> **daldude said:**
> How do I remove the above action you suggested? I tried it and it did not fix my problem and now my fonts get messed up and that is annoying?

To get back from the previous trick, I think it'd be enough:
1. remove the /etc/modprobe.d/radeon-kms.conf
2. launch again the ```
sudo update-initramfs -k `uname -r` -u
``` 
3. reboot

The fonts problem annoyed me as well, but just last night I've found a solution.
In the /etc/X11/xorg.conf file, just add this line into the Device section:
```

Driver "radeon"
Option "RenderAccel" "off"

```


It works on my laptop and now all the fonts are correctly shown.

Hope this helps
;)
