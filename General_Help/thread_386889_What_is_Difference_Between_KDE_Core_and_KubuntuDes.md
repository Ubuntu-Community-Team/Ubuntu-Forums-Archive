### Post by kodoku on 2007-03-17
Kubuntu-desktop is KDE with a plethora of preinstalled applications.  KDE-core is just the bare minimum files that's needed to load KDE.

Basically, if you load kubuntu-desktop, it'll be like installing Kubuntu from a live CD, complete with apps like amarok, ktorrent, kontact, kopete, kmail, etc.  If you install KDE-core, you have almost no preinstalled kde apps, which is good if you like a lean and customized KDE install.

At first, KDE-core may seem faster, but only because not a lot of stuff has been preloaded yet.  Once you start adding your applications and integrating them with KDE (which it is so good at), it won't be much different than kubuntu-desktop.
