### Post by sumguy231 on 2007-09-01
If that's the exact terminal output, it looks like you're in the root of your home directory. If this isn't the directory you extracted the .tar.gz to, you will need to change to that directory. Also be aware that not everything distributed as a .tar.gz is installed with './configure, make, make install'. Always read the README and/or INSTALL files beforehand.
