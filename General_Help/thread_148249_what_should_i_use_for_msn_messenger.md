### Post by Shampyon on 2006-03-21
I agree, stick with GAIM. aMSN's nice, but GAIM supports a number of different protocols. You won't have to download a new program if you want to start using Yahoo Messenger or AOL as well as MSN.

As far as I know, no Linux messinging client has the huge array of fancy bells'n'whistles that you'll find in the Windows native protocol-specific clients. However, you will have everything you need (smilies, etc).
