### Post by rolfpal on 2004-12-07
First thing you need is the command that starts the app.

If it is "Gimp" (for example) I would pull down the computer menu and highlight "Graphics" so that its menu is showing, the RIGHT click on the Graphics menu, you get a menu starting with "add this launcher to panel", go down to "Entire Menu>" expand that and select "Add new Item to this menu"

Fill in the boxes, especially Name ("Gimp" in this case) and command ("gimp" in this case), choose an icon if you want, andf click OK.

It should be there.

Some old school stuff must be run in the terminal
