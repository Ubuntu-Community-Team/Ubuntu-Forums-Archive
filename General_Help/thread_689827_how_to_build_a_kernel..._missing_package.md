### Post by Rocket2DMn on 2008-02-06
To install by compiling from source, you need the build-essential metapackage
```
sudo apt-get install build-essential
```
Here's a resource for your reference as well - [http://monkeyblog.org/ubuntu/installing/](http://monkeyblog.org/ubuntu/installing/)
