### Post by efflandt on 2010-12-03
That usually means what it says.  A drive normally reserves space to automatically remap good sectors in place of bad sectors.  But when all the reserved sectors are used up, bad sectors may start appearing in the middle of your data, and it can only get worse.

So only use if for something non-essential because it could choke or start corrupting data at any time.
