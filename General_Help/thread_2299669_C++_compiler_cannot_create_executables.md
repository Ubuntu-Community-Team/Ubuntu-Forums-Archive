### Post by steeldriver on 2015-10-20
The C++ compiler can't create executables because it looks like you don't have one... you need to install g++ (or another C++ compiler of your choice)

Probably the best route is to install the build-essential metapackage, which includes gcc, g++ and the C development libraries
