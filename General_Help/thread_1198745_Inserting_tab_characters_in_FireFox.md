### Post by NSDragon on 2009-06-27
I do some development using modx, which uses, among other things, PHP code in textareas and such. So what I do when I want to insert tab characters is either copy/paste from a decent editor (like gedit or vim), or hit Ctrl+Shift+U and then type 09 (ASCII/Unicode for the tab character), then hit space. This way you can insert any unicode character (Ctrl+Shift+U, type the hex code, then hit space).
