### Post by SuperSonic4 on 2008-09-27
Just untar the file and then copy then into your kde icons folder (Linux -- /opt/kde3/share/icons/[chosen theme]/[size]x[size]/apps/ 

rename the existing kmenu.png as something line kmenu.png.old then take the number off the end of the replacement file. [this will ensure the old one is backed up]

Next, change to your chosen theme so that KDE re-reads the icons and that is it. Enjoy. [would this be in system settings -> icons]
