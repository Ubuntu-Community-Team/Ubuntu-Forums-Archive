### Post by grundunkus on 2009-04-30
I'm trying to take a screenshot of a right-click context menu in an application in order to prepare a tutorial for somebody, however if the menu is up, Alt+PrtScrn doesn't work. Is there a way I can get a screenshot of this easily?

---

### Post by DeMus on 2009-04-30
> **grundunkus said:**
> I'm trying to take a screenshot of a right-click context menu in an application in order to prepare a tutorial for somebody, however if the menu is up, Alt+PrtScrn doesn't work. Is there a way I can get a screenshot of this easily?

In Applications - Accessories you find a program called Take Screenshot.
Set a time after which the picture has to be taken, select Take Screenshot, open your menu and simply wait until the picture is taken.
