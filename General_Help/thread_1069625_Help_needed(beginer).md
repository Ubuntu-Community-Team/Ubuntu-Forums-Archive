### Post by dj722000 on 2009-02-14
To install COMPIZ, you have to go to APPLICATIONS -> ADD/REMOVE. "ALL OPEN SOURCE APPLICATIONS" in "SHOW". Under search type in COMPIZ. Look for  "Advanced Desktop Effects Settings (ccsm)" This is the easiest way I know how and its how installed it. Im sure there is other ways. But this is the easiest. Once installed it goes under SYSTEM -> Preferences -> compizconfig settings manager and there you control what it does.
 I think they know they need to install it, they "_dont know how_" is what there asking. Keep it easy.
