### Post by Dylnuge on 2007-06-16
Just so you know, all distros of Linux support OpenOffice. Not all distros have it in their package manager, if they have one, but they all support it. Also, anything to do with graphics and windows is an X Windows issue-and all versions of Linux, once again, support X-Windows.

Gentoo is my favorite, but **_*extremely*_** complex. I won't lie on that one-no matter what you consider your current level of expertise, you will learn from installing Gentoo. Note that I have not tried the 2007.0 installer-it may be better then the 2006.1 installer which worked poorly for me. Text install is not too hard with a browser open, but time consuming and almost guaranteed not to be problem free. I had too many problems with it on my laptop, so I left.

Linux Distributions are Linux variants coming out of different companies/teams, like Gentoo, Fedora, Slackware, Ubuntu, Linspire, Mandrake, etc.

Feisty, Edgy, Dapper, are versions of Ubuntu. They are not Linux Distros, they are Ubuntu versions.
