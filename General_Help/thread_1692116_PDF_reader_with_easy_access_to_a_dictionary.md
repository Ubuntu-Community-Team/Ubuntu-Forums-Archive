### Post by simpleblue on 2011-02-20
I read books using a PDF reader and I'd like to be able to easily look words up that I don't know. I see that the ipad has an easy to use dictionary built in and that's the type of thing I'm looking for. Perhaps being able to right click a word and select "look up in dictionary".

Does such a thing exist?

---

### Post by claracc on 2011-02-21
You can always install artha dictionary ( it is in the repositories), then configure it to see the meaning of the selected word clicking on CTRL + ALT +w

---

### Post by coldraven on 2011-02-21
+1 artha 
Install and add it to System>Prefs>Startup Applications
