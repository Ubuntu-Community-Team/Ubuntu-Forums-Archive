### Post by nothingspecial on 2011-04-27
a) The workspaces are now by default in a grid, you can switch between them using Ctrl-Alt and your arrow keys

b) Install the compizconfig-settings-manager package and open it. Right at the top in the general section, click the general options button, then the Desktop Size tab at the end. Horizontal visual size is the number of workspaces along and vertical virtual size is the number of desktops......... well,........ vertically so for 8, you coul set 2 vertically and four horizontally .......... or however you like it
