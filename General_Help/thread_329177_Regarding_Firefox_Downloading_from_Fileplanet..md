### Post by bollix47 on 2007-01-01
Is it possible that Fileplanet uses a pop-up for downloading?  If so, Firefox may be blocking it.  There should be an exclamation mark in the status bar if this is what's happening.

Try adding their domain name as 'allow' under edit-preferences-content-block pop-up windows-exceptions.

Also, there's an FAQ on their site that says:

If the file you are downloading is 2GB or higher, your system cannot download this file size without a Download Manager. Please use a Download Manager to obtain your download.
