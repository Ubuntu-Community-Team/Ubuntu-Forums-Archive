### Post by mcduck on 2010-10-08
If you prefer Gnome interface, you probably don't want the Netbook Edition. Besides, the main things that are different between the netbook version and normal desktop are design features to make the Netbook Edition more usable on a very small screen. The performance difference, if any, is quite small. ( 13" is small, but probably not small enough to justify the netbook interface)

Personally I'd go for Lubuntu, or a minimal install + Openbox or something like that, but if you want Gnome it should be usable on that machine, although you will probably want to tweak a few settings to make it a bit lighter (like enabling the reduced resources mode and disabling animations in the interface. Both can be done through gconf-editor).

### Post by jespdj on 2010-10-08
The main thing with the netbook edition is that it is especially tailored for small screens: it has a slightly different user interface, and for example automatically maximizes the window of all programs that you start up.

For a 13-inch screen, you probably don't need or want that, so I'd use the normal desktop edition instead.

On my Dell Mini 9, which has a small, 10-inch screen (resolution 1024 x 600) the netbook edition works very well.
