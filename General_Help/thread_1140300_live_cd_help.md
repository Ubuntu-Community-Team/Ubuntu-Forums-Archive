### Post by kanikilu on 2009-04-27
The very nature of live CD's means that it doesn't matter what OS is on the hard-drive since it operates independently. You can use any live CD you want, it doesn't even need to be Ubuntu based.

As for getting the latest, if you want it, just download it and burn it with Brasero, k3b, whatever...

[http://www.ubuntu.com/getubuntu/download](http://www.ubuntu.com/getubuntu/download)

[https://help.ubuntu.com/community/CdDvd/Burning](https://help.ubuntu.com/community/CdDvd/Burning)
