### Post by wd5gnr on 2008-12-22
You don't mention which version of KDE you are using, but you should be able to right click on the K menu and pick "Menu Editor". From there, you should be able to work it out. You can also run the kmenuedit command (for example, press Alt+F2 and type kmenuedit <ENTER>) .
