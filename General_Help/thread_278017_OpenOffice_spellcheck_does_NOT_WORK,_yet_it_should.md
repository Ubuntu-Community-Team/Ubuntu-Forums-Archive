### Post by lucia_engel on 2006-10-28
I used to have this problem since I didn't choose US and UK english as default.

If you have EN-US, EN-UK as default and it doesn't work,
Tools > Options > Language Settings > Writing Aids
Make sure "Check spelling as you type" is checked. Also check any features that you want.

If you do not have the two as default,
Tools > Options > Language Settings > Languages
Make sure there's a spellcheck symbol (ABC checkmark) next to the language you have selected as default. If it doesn't, you'll have to install new dictionaries. Try using US, UK and see if that work first.

If there's no symbol,
File > Wizards > Install new dictionaries
Install the dictionaries you want.
