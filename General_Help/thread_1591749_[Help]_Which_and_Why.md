### Post by snowpine on 2010-10-09
Welcome to the forums! I will answer question 1, and I'm sure someone will be along soon to answer 2 and 3. :)

Ubuntu and Ubuntu Netbook Edition are the same core system, different user interfaces. The Netbook Edition has a "launcher" with lots of big icons, kind of like an iPhone... it is designed for smaller screens. The best advice I can give is to take both for a test drive, see which you prefer. Your hardware is excellent and easily capable of running any version of Ubuntu. :)

### Post by DogMatix on 2010-10-10
They are essentially the same. The netbook version basically offers a different desktop which is geared around smaller screen size (as mentioned above). The UNR version allows you to switch between the netbook desktop and the classic desktop if you so desire. 

I have used the netbook version in the past but found I prefered the classic desktop and never used the netbook version. So when it came to time to load Ubuntu on my new netbook I opted to just install the regular version.

**Link:** [Ubuntu UNR Wiki]("https://wiki.ubuntu.com/UNR")
