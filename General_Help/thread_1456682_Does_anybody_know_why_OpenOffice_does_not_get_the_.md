### Post by autonomy on 2010-05-09
I figured out why this is. When I leave OOo Writer with the insertion bar blinking in a list or a table, that little list or table window is up, so when I return, that little list or table window sort of gets the focus; but really, the last app still has the focus. That little list or table window blocks focus from going back to the text. If I dock that little window or leave the insertion bar blinking in a regular paragraph, then OOo Writer gets the focus upon return as expected, so I'll mark this solved even though those aren't really solutions. I don't know if this is a GNOME phenomenon or Oracle. I bet it wouldn't happen in Windows as heretical as that is of me to say, but GIMP doesn't show up as two or three applications in Windows as it does in GNOME Ubuntu. This is an OOo [bug]("http://www.openoffice.org/issues/show_bug.cgi?id=111502").
