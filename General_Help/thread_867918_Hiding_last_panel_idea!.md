### Post by anotherdisciple on 2008-07-23
I have been doing a lot of stuff to hide my last gnome panel(because I use Avant Window Navigator). The last one is freakin' hard to get rid of. I WAS just setting it to autohide at a size of zero and setting the unhide delay to a really high number.

Well, when I want to turn off compiz I have to change all that stuff back in order to start using a gnome panel again.

Here's my latest idea... I don't use the widgets layer in Compiz for anything, so I set the gnome-panel as a widget. Now it hides until you press F9 to activate the widget layer. On top of that... when you turn off compiz.... your panel returns to normal.

I don't know if anyone else has beat me to this idea, but I thought it was cool... thought I would share.
