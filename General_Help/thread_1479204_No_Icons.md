### Post by cbecker78 on 2010-05-10
Reinstall the theme you are using. or else try selecting a different icon set in Apearences --> customize. Or just selecting a different theme alltogether.  It's possible that the link to the places your icons are stored at has been lost, the icons moved or corrupted themselves, or just plain missing.  If one of the above doesn't work, try installing a new theme that includes icons and see if that works.
