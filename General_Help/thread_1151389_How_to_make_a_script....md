### Post by Agent ME on 2009-05-06
You could make a .desktop file that would do this when double-clicked, but even simpler is to just make a deskbar launcher do this. Right click on an empty spot on the top panel (Applications, Places, System -- Time, User) and click "Add to Panel...". Choose Custom Application Launcher, put that command in the command slot, and some sort of name to remind you what it is in the name spot (this will be shown if you hover the mouse over the button later).
