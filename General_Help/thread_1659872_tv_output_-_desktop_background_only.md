### Post by Vermind on 2011-01-04
if you use the nvidia-settings tool,
in the X Server Display Configuration section,
click configure and make sure twinview is enabled. Then the position of the displays determines which one is primary, and you should be able to move windows and start programs between the displays.
If I remember correctly, HDMI worked ok for my nvidia laptop.
