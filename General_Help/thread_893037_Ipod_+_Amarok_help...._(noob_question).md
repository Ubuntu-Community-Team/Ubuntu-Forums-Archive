### Post by mc4man on 2008-08-17
Well i just set up an ipod last week on amarok and it was a little confusing. Was actually just doing it to figure out how to make amarok the default for music players ( auto start and auto connect when plugging in ipod)

In any event the first time i got it wrong, and since then found 2 diff. methods. ( still a little confused)
The easiest is after plugging in ipod open amarok and in the playlist box go settings -> configure amarok and then click the media devices box on lower left. (2nd screen shot

If you see a listing at top in media devices box (sdc1 or something similar) then use the drop down plugin box to pick Apple ipod...

If not then try the auto detect and then if something shows up in media box again pick the plugin ect.

If nothing shows up then use the add device, give it a name and do the plugin but **don't** enter a mount point.

After it's configured then back in the playlist screen  you can choose connect. 
If you see what's in screen shot 1 then it's configured right, if you see folders like ipod_control, calenders, ect. then it's not.

After it's configured there was a way to pick the model of ipod from a list, but that's not critical (and I can't find it again

If you get it configured right and working let me know if you want to set it up as default player

Edit : actually for xubuntu setting amarok as default should be straightforward unlike ubuntu which I was referring to above
