### Post by sixfthick on 2009-04-08
One of the annoying things that I've found about Ubuntu vs Windows is that when I copy an entire folder to my MP3 player it copies the files in disk order rather than filename order.  This means that if the files are out of order on the disk then they will play out of order on my MP3 player.

I know that I could just create a new folder on the MP3 player and copy files in that, but it would be much easier just to drag-and-drop the folder and know that it would copy the files in filename order (like Windows does).

Is there any way to change the order that files are copied when copying an entire folder?

### Post by vanadium on 2009-04-09
My primitive mp3 player still will play the files in alfabetical order, but if yours really only follows the "physical" order, then you can be helped with the tool "fatsort"

Install it using synaptic, and consult the help from a command prompt with "man fatsort".
