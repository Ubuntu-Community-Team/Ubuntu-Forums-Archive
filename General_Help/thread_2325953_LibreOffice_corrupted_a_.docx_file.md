### Post by Mark Phelps on 2016-05-27
LibreOffice does not work well with some docx files -- depends on how much embedded stuff is in the file.

WPS Open Office works better (the new name for Kingston Office) but last I checked, they quit releasing their free version and charge for it now.

It's best NOT to use MS Office alternative Linux apps with docx files, as the only app that truly works well all the time with docx files is MS Word.

And next time you want to experiment, make sure to make a copy of the file so that if it gets trashed, you still have the original around.
