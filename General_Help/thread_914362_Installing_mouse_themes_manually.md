### Post by Genius314 on 2008-09-08
Well, you can extract the theme to ~/.icons.
So the cursor theme should be in a folder like this: /home/username/.icons/themename (which should contain a "cursors" folder)

To apply the theme, just go to System>Preferences>Appearance, click "Customize," go to the Pointer tab, and click on the theme there.

That *should* work, although it's been a while since I've installed a cursor.
