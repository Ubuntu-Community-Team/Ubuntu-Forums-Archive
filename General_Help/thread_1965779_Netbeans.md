### Post by r-senior on 2012-04-26
You don't even need to recompile it. Compiled Java code runs on a Java Virtual Machine provided by the platform and is freely transferrable between platforms.

But, to answer the question, Java projects can be recompiled on other platforms without modification. Just copy across the project, open in Netbeans and compile/run as usual.
