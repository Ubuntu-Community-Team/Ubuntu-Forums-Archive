### Post by Quackers on 2010-10-09
If you are in a Windows system Windows Explorer may not "see" the .iso file. Use a .iso burning program (like ImgBurn - it's free) and select "burn image" and navigate to your downloaded file. The burning program should see the iso.

### Post by mcduck on 2010-10-09
What files did you download? There's only one file you need to download, and that *is* the ISO file itself. ;)

Depending on what archive applications you have installed on your Windows system .iso files might get an archive icon on them, for example WinRAR does this. Just ignore that and don't extract the .iso file. (Actually I _seriously_ recommend setting Windows to always show file name extensions, solves many problems..)
