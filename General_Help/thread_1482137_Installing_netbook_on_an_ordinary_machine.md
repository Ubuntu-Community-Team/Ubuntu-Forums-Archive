### Post by blazemore on 2010-05-13
The netbook edition contains essentially the same applications as the desktop edition.
It just has a nice launcher, and a few changes that make it more suitable for smaller screens.
In this respect it isn't a "true" netbook OS like ChromeOS or the original Acer and Asus distros.
However, you can install the netbook remix, either from the Netbook Edition LiveCD, or just by installating the packages on a standard Ubuntu Desktop. See [here]("http://ubuntuforums.org/showthread.php?t=1482486") for how to do that.
