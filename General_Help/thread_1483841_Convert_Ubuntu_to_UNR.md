### Post by VeeDubb on 2010-05-15
Your first instinct is correct.  You can just install it from synaptic. You have two options:

1. just install "ubuntu-netbook" from the repositories.  This will install all the missing stuff and add an option for it at login, just as you might switch between gnome and KDE.

2. Install "netbook-launcher" and "maximus" and then add each of those programs to your list of start-up applications.  This will give you all the launcher goodness of UNR, without having the panel completely locked down like it is in UNR.
