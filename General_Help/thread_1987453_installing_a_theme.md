### Post by samwillc on 2012-05-26
Ok, so I got this far.

1) Download the gtk3+ theme
2) Extract it
3) Type 'gksu nautilus' in terminal so I can move folder into file system with correct permissions
4) Move theme folder into /usr/share/themes
5) Open ubuntu tweak > tweaks > theme > select new theme

When choosing 'radiance', the top bar goes grey as do the windows etc... using Zukitwo or GrayDay, the top bar remains dark but the menus have different background colour.

I presume this is expected behaviour.
