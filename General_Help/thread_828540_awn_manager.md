### Post by rune0077 on 2008-06-14
typically the command is just the name of the program, often with "-" instead of spaces. The command for AWN for instance, is "avant-window-navigator".

For testing, hit Alt+F2, then type in the command you want to test. If the program opens, then it's the right command, if it doesn't, then it's not :)

Or tell us what program you want the command for, somebody else might already know it.
