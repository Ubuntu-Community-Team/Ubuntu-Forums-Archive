### Post by yancek on 2017-01-09
You don't download the Ubuntu iso to a thumb drive.  You need to download it to your hard drive and use some software designed to create a bootable thumb drive from the iso.  Did you do that?  If so, what software did you use.  Some common options are Rufus, Universal USB Installer and unetbootin.

If you did put Ubuntu on a flash drive and boot it, which Installation Type option did you use?  There are usually several options including install Alongside, Erase Disk and Install Ubuntu and Something Else (this is the manual install option).  If you selected the Erase Disk option, you have overwritten windows so stop booting the installed system.

You will need to provide more information before someone can make a logical suggestion.  The best way to do that is to boot the Ubuntu usb and install the boot repair software from the link below and run it.  Select the Create BootInfo Summary option and post a link to the output here.  Do not try to make any repairs.

Also, Ubuntu isn't going to automatically install itself just by booting the flash drive.  The user needs to make a selection to do so.
