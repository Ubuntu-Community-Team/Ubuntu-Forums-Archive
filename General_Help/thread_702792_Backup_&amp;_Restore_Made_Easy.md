I just wanted to share something with the community today. I'm a frequent user of Ubuntu. I use Ubuntu for server reasons as well as for desktop use. one of my biggest issues is with backup and restore. I would like to have some program simple and intuitive for this purpose. Well today I was reading an article about a new program called [**Restore**]("http://www.restore-backup.com/"). it is an easy install (they provide .deb package). It provides a web type GUI. It will work for Windows, Netware, and Unix/Linux. I would recommend this program to anyone looking for a simple to medium type configuration for backups.
