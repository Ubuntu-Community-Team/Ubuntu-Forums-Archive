### Post by Claus7 on 2008-07-05
Hello,

I would suggest you first to see what is going on with your hardware status (are the cables ok, does your memory fit well in its slot?). Also check out System Monitor and the error log files (in system - > administration). That way you will be able to see whether any application "eats" more memory or cpu than it should or to see if it is something else.
