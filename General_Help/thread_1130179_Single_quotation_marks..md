### Post by CatKiller on 2009-04-19
> **joeinnes said:**
> I reckon that my "Compose" key is either AltGr or the right-click/context menu one (after AltGr, then Super, then that other key)

I believe that that key is referred to as "Menu" in the dialogues.

You should be able to find out which key is mapped to the Compose key with System -> Preferences -> Keyboard -> Other Options... -> Compose key position. I've got "Right Alt is Compose" set in mine, and the only other thing that's non-default in there that I can see is "Press Right Alt key to choose 3rd level." The combinations that I gave earlier work as I gave them earlier on mine using AltGr as Compose.

It's probably worth pointing out that you don't press all of them together, it's a sequence (but quite a quick one).
Shift-AltGr (together)
(release)
'
