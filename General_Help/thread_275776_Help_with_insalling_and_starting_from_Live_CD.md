### Post by meng on 2006-10-11
AFAIK, and I have looked into this many times, you can't install from the LiveCD without loading the desktop first. The alternate CD does install the complete Ubuntu, it's just that the installation program itself is text-based (but the content is essentially the same as the LiveCD installer). Therefore, it may be ideal for you, particularly if you don't have anything of value on the hard drive.
