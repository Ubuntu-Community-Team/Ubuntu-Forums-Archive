### Post by Cheesemill on 2012-12-04
The real question is how did you get 3.7 on the AMD systems.

Current official kernel version for 12.10 is 3.5.0.19.22
[http://packages.ubuntu.com/search?keywords=linux&searchon=names&suite=quantal&section=all](http://packages.ubuntu.com/search?keywords=linux&searchon=names&suite=quantal&section=all)

Have you by any chance added any 3rd party repositories or enabled the backports on the systems with 3.7?

Compare the following commands on your machines and see if there are any differences:
```
cat /etc/apt/sources.list
ls -l /etc/apt/sources.list.d/
```
