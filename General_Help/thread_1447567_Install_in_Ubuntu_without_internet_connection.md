### Post by quadproc on 2010-04-05
> **Brv said:**
> Hello.
Can I install Ubuntu Studio 9.10, Ubuntu 9.10 or Kubuntu 9.10 Live CD/DVD on a computer without internet connection?

There are several ways to install without using an internet connection.  Probably the easiest is to get or create a Live CD for the version that you need.  Another possibility is to use a USB flash "disk".  And there are some ways of cloning hard disk drives.  If your various computers are on a LAN then you could install over your LAN.

The Ubuntu folks will send you a Live CD for free if you like.  See [https://shipit.ubuntu.com/](https://shipit.ubuntu.com/)
The only problem with this approach is that it takes a while.
