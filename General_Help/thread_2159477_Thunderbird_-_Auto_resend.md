### Post by SeijiSensei on 2013-07-03
Surprisingly, it [doesn't seem like]("https://groups.google.com/forum/#!topic/mozilla.support.thunderbird/Wab6e67gco4") Thunderbird automatically retries outbound messages that fail.  However I came across this [plug-in]("https://addons.mozilla.org/en-US/thunderbird/addon/blunderdelay/") that provides the functionality.  It hasn't been maintained for some time now though.  It references a more recent plugin, [SendLater]("https://addons.mozilla.org/en-US/thunderbird/addon/send-later-3/"), which adds more functionality than you need.
