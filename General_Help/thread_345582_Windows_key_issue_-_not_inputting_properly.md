### Post by gaebriel on 2007-01-24
I don't know if this will be helpful or not because I use GNOME instead of KDE, but to do keybindings in Metacity that use the Windows Key, I have to specify <mod4> as the key pressed. 
Example 1: ALT key and F1 key - <Alt>F1 
Example 2: Windows Key and Up arrow - <mod4>Up
Example 3: Windows Key and Control Key and Alt Key - <mod4><Control><Alt>
