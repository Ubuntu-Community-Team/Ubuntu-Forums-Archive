### Post by Revolutionary101 on 2010-04-16
Well with both you will probably see close to the same performance. Ubuntu Netbook Remix will, most likely, be faster (only by a bit, if anything).

The Ubuntu Netbook Remix GUI is made so it looks better and is easier to use on the small screens. Although with a resolution of 1024x600 it wouldn't be hard to use with the normal Ubuntu install.

If you did the normal Ubuntu install, I would suggest installing the Xfce (fast) or LXDE (faster than Xfce) desktop enviroments. These two desktop environments are faster than GNOME and will perform better on this low spec netbook.

Bottom line is that, between Ubuntu and UNR there is little performance difference. The question you must ask yourself is do you want a simple enviroment for small screens (UNR) or something that you are used to on other computers (Ubuntu default).

### Post by Aped on 2010-04-16
Revolutionary pretty much covered it. Allow me to simply suggest you start off with xubuntu, which comes with xfce and NO gnome, standard.
