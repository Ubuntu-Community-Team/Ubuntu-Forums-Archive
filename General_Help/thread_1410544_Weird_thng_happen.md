### Post by Topsiho on 2011-05-30
You should google on the name of this virus: WIn32:cih-C.

I got quite a number of hits, of which my WOT (World of Trust add-on for Firefox) marked a number as [COLOR="Red"]not trustworthy[/COLOR] (especially those that offer to remove the virus from your disk), so you need to be very careful. But if you are, you can get much information about this virus or trojan or whatever.
Linux, and thus Ubuntu, are not affected by Windows viruses, or even viruses at all, though trojans might be an issue if you install them yourself on your computer, and if they are not only for Windows (e.g. if they are multi platform). But I think the problem is that you use Ubuntu inside of Windows, as a Windows program, so to say, so that way even Ubuntu might be vulnerable.

Hope this information might help you.
Even in Linux you must be very careful from which source you install applications. From synaptic (I know not better) is quite safe.

And of course, you'd better install Ubuntu in the normal way, if you want to use it seriously.

Topsiho
