### Post by webbertiger on 2009-09-04
Try the following to clean up the old key and install new key:
1) check the current key list
**sudo apt-key list**
Then you should be able to see the old key in the list. It looks something like

pub   1024D/6A423791 2006-09-26 [expires: 2009-09-25]
uid                  Opera Software Archive Automatic Signing Key <hostmaster@opera.com>

2) remove the obsoleted key
**sudo apt-key del 6A423791**
Note that here 6A423791 is the key ID you find in step 1).

3) install new key
**wget -O - [http://deb.opera.com/archive.key](http://deb.opera.com/archive.key) | sudo apt-key add -**

Then the key errors should be resolved.
