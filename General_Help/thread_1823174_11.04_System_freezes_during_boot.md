### Post by MG&amp;TL on 2011-08-11
Did you install a particular piece of software beforehand? If so, use a quick live cd distribution (or a floppy) like Puppy Linux, mount the filesystem, find the offending files, delete them or whatever, take floppy or cd out, done.

Otherwise, I guess you have to take into account that Ubuntu runs to a six month release schedule and that it's never going to be perfect. I reinstall mine about once every two weeks/two months, depending on what I've been mucking around with (and how good new distro releases look).

I don't know what the command line in grub is for, but if you can access your system from it, great. Press 'C' at grub.
