### Post by lrmall01 on 2005-12-31
The desktop environments are different.

Ubuntu uses Gnome. [http://www.gnome.org/](http://www.gnome.org/)

Kubuntu used KDE. [http://www.kde.org/](http://www.kde.org/)

There won't be any difference in playing windows games on one versus the other.  The main difference will be in the desktop layout and some of the core desktop tools such as the file management and system administration.
