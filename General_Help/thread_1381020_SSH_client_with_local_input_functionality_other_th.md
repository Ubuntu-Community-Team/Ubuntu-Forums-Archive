### Post by aaberr on 2010-01-14
Greetings the community,

I'm looking and have been looking for an ssh client that has local input functionality like what you can find in "[EmTec ZOC]("http://www.emtec.com/zoc/")". Putty already has this feature but once that is activated the command history that may be used by up and down cursor can no longer be used as the command history does not exist in Putty and is done by SSH itself. ZOC emulates the command history itself once the local input is activated. 

The use of this feature is often for when the latency makes working with SSH difficult.
