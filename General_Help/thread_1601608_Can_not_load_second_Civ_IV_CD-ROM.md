### Post by sienile on 2012-08-11
Not sure if this works for Civ, but a common workaround for many games is to copy the contents of both CDs to the same directory on the HDD and run the installer from there. This will only work if no installer files are named the same.

### Post by Toz on 2012-08-11
You're trying to install it into z:/media - which is where your CD drive is mounted - right back onto your cd. Hence the error message. Try selecting C: as the drive.
