### Post by mac9416 on 2010-05-25
> **elliotn said:**
> one question can that work in a ms windows, as for most people who want to install offline will probably use an internet cafe wich most will run MS windows

elliotn, the script will only work in Linux. However, kaleem.adil could use a Firefox extension such as DownThemAll to get the URLs in the script.

Here's the problem with Synaptic's Generate Download Script tool. In order for Synaptic to know what applications are available, it must be updated - using an Internet connection.

A better solution would be to use [Keryx]("http://keryxproject.org"). It is much like Synaptic, but instead of staying on your offline computer, you can take it on a flash drive to an online computer to download packages. Plus it works on Windows.
