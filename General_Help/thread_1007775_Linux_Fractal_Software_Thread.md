### Post by djbushido on 2008-12-10
This is a thread designed to help fractal enthusiasts like myself get fractal software running, producing those beautiful images. You are asked to please read over what has been posted and make sure you aren't double posting. Also, please only post software that you have that works, or else with a disclaimer like "says it works with linux but I haven't gotten it running."

To start it off:

Gnofract4d: [http://gnofract4d.sourceforge.net/](http://gnofract4d.sourceforge.net/)

Includes a *.deb, as well as source download. Debian tested to work with Xubuntu and Ubuntu, needs some Gnome things for operation, so will need to install that from repos for Kubuntu.

Gimp Flame plugin: install Gimp from repos, then:
1. start GIMP
2. start a new picture (file->new)
3. go to filter->render->nature->flame <or> IFS fractal
4. select parameters, and then click ok
5. Gimp will render it for you then.

If anyone has any others, they would be greatly appreciated.
