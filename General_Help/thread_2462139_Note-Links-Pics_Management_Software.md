### Post by Holger_Gehrke on 2021-05-14
With evernote you're putting your notes into 'the cloud' - meaning somebody else's computer. If you're comfortable with that it's probably good enough. If you're not comfortable with that and possibly need offline access to your notes, then zim or cherrytree come to mind. I personally have a local apache2 webserver with PHP running and use dokuwiki for any notes that are likely to be needed for the medium to long term or need to be accessible from other machines - like smartphone, tablet, ebook-reader and "smart"-TV - in the local network.

Holger

### Post by dragonfly41 on 2021-05-15
Horses for courses.
In fact I use CherryTree - Zotero - and others such as Albert - in a toolchain.
No single tool meets all my workflow requirements.

With CherryTree you can mix objects .. such as embedded code snippets in nodes.
Thus all the snippets you read in this Ubuntu forum can be embedded as code in your notes.
You can choose code types - bash, Python whatever.

You can link Zotero and CherryTree through Zotero plugins.
[https://www.zotero.org/support/plugins](https://www.zotero.org/support/plugins)
