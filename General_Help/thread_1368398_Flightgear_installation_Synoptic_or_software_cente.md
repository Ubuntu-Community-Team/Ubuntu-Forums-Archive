### Post by mac9416 on 2009-12-30
It depends on which one you are more comfortable with. For an inexperienced user, Software Center is probably the way to go. An intermediate user might prefer to use Synaptic. I hardly ever use either; I typically use the comand line ('sudo apt-get install flightgear')

You really can't go wrong with either. Try them both and see what you like.  :)
