### Post by squaregoldfish on 2008-06-05
You could try installing mdbtools and mdbtools-gmdb. These give you the option of looking at, and exporting, data from Access databases, so you might be able to get the data into a format that Kexi or OO will understand.

I'm afraid I've only used these tools to look in mdb files, so I've no idea how good or otherwise the export tools are.
