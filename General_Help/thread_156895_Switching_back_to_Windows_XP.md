### Post by Buffalo Soldier on 2006-04-08
This Windows XP CD that you're trying to boot... is it the CD that comes with the laptop (usually called something like system restore CD) ? 

Or did you bought a Windows installer CD from a computer shop somewhere (usually called an OEM CD)?

Some laptop manufacturer will not let any other Windows installer CD run on their laptop. If that the case and you've lost the installer CD that comes with the laptop, then you need to contact the manufacturer.
