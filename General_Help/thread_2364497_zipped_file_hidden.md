### Post by coffeecat on 2017-06-24
Welcome to the forum.

If by hidden file you mean one whose filename is preceded by a ".", as in ".filename" and ".filename.zip", try pressing ctrl-h when viewing the list of files in the attach file/file upload window that opens when you click on the attach file icon in your email client. That will make visible all hidden files with a preceding ".". 

It's possible that your file manager is set to show hidden files, but the attach file/file upload window in your email client is not. In the version of Ubuntu that I am using, they are separately configurable.

If that doesn't work, please post back with these details: which release of Ubuntu you are using, and which flavour, i.e. Ubuntu itself, or Kubuntu, Xubuntu, etc; and what email client you are using. For example are you using a stand-alone client such as Thunderbird, or the webmail page of your email provider in a web browser such as Firefox?
