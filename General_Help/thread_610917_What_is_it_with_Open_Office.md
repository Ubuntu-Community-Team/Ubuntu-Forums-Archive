### Post by atomkarinca on 2007-11-17
> **Scroobytec said:**
> Well had a go at the linux version today - BUT BIG BUMMER the Linux version do not support Ubuntu only SLED 10  and RHEL 5. What a disappointment.

i beg to differ. i'm running it on my machine. if you haven't tried yet, there are two tricks installing it:

1. the installation doesn't like any kind of desktop effects. turn them off before you start installing.

- get the .bin file

```
chmod +x whateverthefilenameis.bin
./whateverthefilenameis
```

2. after the installation is finished,

```
gksudo nautilus
```

browse into your home directory, hit ctrl+h, right-click the .lotus folder and set the permissions for your username.

that's it. then it's working like a charm.

PS: it looks very nice. a little ram-monster though.
