### Post by PmDematagoda on 2007-11-29
Ubuntu 7.10 has more features, is more stable, is faster and has a more stable GUI than earlier Ubuntu versions do, but if the number of problems you are having is too much for this advantage then you can try downgrading the Ubuntu OS using this [guide]("https://help.ubuntu.com/community/DowngradeHowto").

But a word of advice, make sure that all of your important data and files are all backed up since there is a good chance that your entire OS may fail as suggested by the How TO itself.
