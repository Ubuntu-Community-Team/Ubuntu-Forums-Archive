### Post by heindsight on 2009-03-16
Pressing Ctrl+Alt+F8 takes you to virtual terminal number 8 (tty8)
In Ubuntu the system log daemon (syslog) uses this tty to display certain log messages. 

Messages generated by the boot scripts are usually written there.
Normally the last message you'll see there (the last boot script message) is:
```
 * Running local boot scripts (/etc/rc.local)        [OK]
```

Certain system errors may however result in error messages written to tty8
