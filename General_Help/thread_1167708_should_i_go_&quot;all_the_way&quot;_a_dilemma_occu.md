### Post by kay-man on 2009-05-23
Well, I can't tell you what to do, obviously, but I found myself facing the same choice not so long ago.

I have a desktop PC that came with Windows XP, and I've been running it in dual boot with Kubuntu for a few years (since Edgy, I believe). I found that the times I, or anybody else in my family for that matter, would actually boot up Windows became ever fewer and further apart.

So I started stripping down the Windows install, keeping only a few programs I thought I might need. But the truth was, I didn't.

So in the end I decided a few weeks ago to delete Windows altogether. I am now left with just one single NTFS partition that I used to share data between Windows and Linux. All the rest is ext3 and Linux only. And the last one will go too once I get around to it.

We have one Windows PC in the house, which my kids use for games, and my daughter has a dual boot laptop. One other desktop is Kubuntu only.

In the end, we don't do too much fancy stuff with the PC, and we don't need any specialized programs that are Windows only, such as Photoshop. And I found that mythtv does a better job than the original Hauppauge software that came with the TV card.

But there are no ready replacements for everything, so in the end it really depends on your situation. But by the sound of it, as long as you have itunes covered, I'd say you're good to go!

Enjoy!
