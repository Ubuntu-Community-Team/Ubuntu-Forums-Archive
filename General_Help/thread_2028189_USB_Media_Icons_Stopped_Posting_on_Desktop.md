### Post by OM55 on 2012-07-17
Of course you can still do that in 12.04 with gnome-fallback-sessions...


_ Here is how to do that, step by step:_

1. If you haven't done so, install ubuntu-tweak. To do that just go to: [http://ubuntu-tweak.com/](http://ubuntu-tweak.com/) download the latest version and install it.

2. Launch Ubuntu-Tweak from the menu: Applications -> System Tools -> Ubuntu Tweak

3. Select 'Desktop Icons' from the main screen (it is under Desktop)

4. At the bottom check the 'Show mounted volumes', which is by default unchecked.


That's it.

Hope that helps.
