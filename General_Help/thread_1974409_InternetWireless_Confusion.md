### Post by Old_Grey_Wolf on 2012-05-06
> **saponempotenti said:**
> I understand the following but am very confused..

I understand that there are 150mbps, 300mbps, and 600mbps for speeds in wireless cards.  mbps stands for mega bit per second. You can get speeds up to 300mbps  with wireless-n;  however, wireless-g is typically about 54mbps. 

> My router states "5.0GHz wireless frequency for fast data transfer" and Im not sure how to translate GHz to mbps(or if they are even talking about the same type of speed).  The router is also advertisting gigabit.. which i am assuming is 1000mbps?Wireless routes use two radio signal frequencies for the transmission of data. One is 2.4GHz and the other 5GHz. The radio signal frequency is not the same as the bits per second data rate. The 5GHz radio signal frequency allow for more signal bandwidth so it supports higher mbps data rates. If you want data rates above 100mbps you should use the 5GHz wireless-n option.

> My computers/internet connection got 19.51mbps on speedtest.net.  When I torrent files, they download at 2.3 MB/s (on average) 19.5 mbps is divided by 8 to get MB/s (8 bits in a Byte). So 19.5 mbps / 8 bits = 2.4 MB/s (mega bytes per second)

Hope that helps.
