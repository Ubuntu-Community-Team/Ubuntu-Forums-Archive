### Post by QIII on 2011-12-19
I have a PIII laptop with 512MB and an integrated AMD gpu.

I have an Ubuntu derivative called Bodhi Linux running on it and you would never guess it was a 10 year old machine.

Bodhi uses the Enlightenment DE, which is very light.

There are several light DEs available with Ubuntu.  If you are really worried about lag after you install and play with it, you can try LXDE or Xfce.

But I don't think you are going to have a problem.
