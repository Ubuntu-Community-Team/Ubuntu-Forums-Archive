### Post by dragonfly41 on 2016-02-22
First point I would make .. have you actually installed Ubuntu 14.04 ("alongside Windows") in a separate partition?

Second point is in your message "Unable to access WINDOWS 10" there is reference to /media/paul/WINDOWS 10
I would ensure there are no spaces in your media paths.  Use instead labels WINDOWS10 or WINDOWS_10 and not WINDOWS 10.
You can change partition labels.
