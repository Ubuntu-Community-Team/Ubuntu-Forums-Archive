That sounds like a "task manager".

You can add one to the main panel, or add a new panel and add a task manager to that new panel.

(Right click a panel, select add to panel)

I like having one panel at the top, so I added a task manager to it.

### Post by flipper9 on 2009-10-23
I found I really LOVE window-picker-applet and ditching the default "bottom toolbar".  It merges the application switching of the task-bar and the title-bar of maximized windows(integrates the title-bar of the maximized window into the task-bar giving you more vertical screen real-estate).  Really improves the usability on a small screen, such as on a net-book, giving you just enough extra vertical pixels to make using software easier.
