### Post by The Thunder Chimp on 2010-08-10
Unless you want to run a network server (like for a website) on that particular machine (which I doubt since you talked about the possibility of installing the netbook version), you will want to install the original Ubuntu version, as I have a 10,1 inch (which is 3 inches smaller than yours I believe), and I have the original Ubuntu running fine.

As for the memory, it strictly depends what you want to do with your computer. For instance, if you wish to put loads of music, pictures, and movies, you'll want to increase your memory accordingly. However, for some reason, space on Ubuntu fills up much slower than on Windows for some reason, probably because of the way your files are arranged on the hard-drive. Additionally, please note that all your data from your Windows partition will still be available from your Ubuntu Desktop (and command line of course), but only with third party software can you check your Linux files on Windows. All of these reasons put together have made me choose a 70% Windows/30% Ubuntu Hard-Drive partitioning (Total drive is 320 GB), although I'm on Ubuntu practically 95% of the time.

As for the touch-screen, I have heard of such screens working on Ubuntu, but never tried it myself. I can't help you with that, sorry.

If you need any more help, just type it here!

All the Best,
