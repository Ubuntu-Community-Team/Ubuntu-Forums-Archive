You have burned Ubuntu to a cd and also downloaded the iso?
What and how did you burn to a cd?

Okay, let's start:
Do you have free space on your harddisk(s) to install Ubuntu?

You have a iso file. Burn that onto a cd using the burn image function. This way not the iso file goes to the cd, but the contents of the iso file.
When done, put the disc into the cd-drive and re-boot the computer
Make sure the pc boots from the cd.
You will then get a choice if you want to start the live cd version or really install Ubuntu.
Choose the live cd first to see if things work, monitor, sound, internet connection, etc
When you are sure things work like they should AND you have free space available reboot again and this time choose to install.
At step 4 you have to choose where you want to install Ubuntu: Do not choose: entire disk since this will wipe XP.
After installation reboot again (after ejecting the cd) and you will get a menu with which you can choose to boot into Ubuntu or into XP.

Have fun.
