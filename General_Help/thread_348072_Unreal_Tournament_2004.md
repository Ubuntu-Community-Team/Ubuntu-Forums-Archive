### Post by KanRiNiN on 2007-03-06
Before you execute the linux-install.sh type "export SETUP_CDROM=/mnt/cdrom".  Just change the /mnt/ point to wherever your DVD-ROM is.  Then run it.  I have the DVD version which installed fine after using that command.
