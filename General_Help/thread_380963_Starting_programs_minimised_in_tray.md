### Post by cmost on 2007-03-10
I use a program called 'Alltray'.  Install it or compile if from source.  Then, you can minimize any application (or start one up) to the tray by simply prefacing the launch command with alltray.  For example, to startup Evolution in the tray, I use the following command:

alltray evolution-2.8.
