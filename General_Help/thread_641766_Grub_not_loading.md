### Post by Sef on 2007-12-15
Windows overwrote GRUB, and Windows only sees other Windows.  

To fix this problem, read [Recovering Ubuntu after installing Windows]("https://help.ubuntu.com/community/RecoveringUbuntuAfterInstallingWindows").
