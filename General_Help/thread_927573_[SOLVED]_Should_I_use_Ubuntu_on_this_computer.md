well how much RAM Ubuntu uses is dependent on how much RAM you have, if that makes sense. If you have lots of RAM, Ubuntu will simply cache more stuff in RAM to make your system run faster. Less RAM will obviously make the system less efficient - I think 256 will be fairly unusuable, but it will probably work - just not very well. i would reccomend Xubuntu.

if you do decide to go ahead with regular Ubuntu, make sure to make a swap partition/file that is at least 512MB in size - swap space is crucial for low memory machines. also, as mentioned above, you'll need to use the alternate CD to install.

### Post by snowpine on 2008-09-25
When I first made the switch from windows, I ran Ubuntu 7.10 on a 600mhz pentium 3 with 256mb of ram. It was not speedy, but it was usable (I've since moved on to lighter alternatives). Your computer is quicker, so I think you'll be fine, as long as your expectations are reasonable. Bumping the ram up to 512 will make a big difference.

One suggestion I can make is to install a lightweight windows manager such as Openbox, Fluxbox, IceWM, etc. You can continue to use the Gnome desktop for everyday tasks, but when you need to do something processor- or ram-intensive, you can switch to the light windows manager to get a little performance boost. For example, (before I gave up on Gnome) I would switch over to Fluxbox for music editing. It is nice to have options. :)
