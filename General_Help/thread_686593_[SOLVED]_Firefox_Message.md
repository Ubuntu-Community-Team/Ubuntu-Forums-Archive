Hi, is there any way to get rid of the "You are about to leave an encrypted page" message?  It doesn't show up on my Windows OS, and I was just wondering if there was a way to make it stop showing.  Is there something in Firefox I need to click, or do I need to type something in Terminal?

### Post by aysiu on 2008-02-03
As long as you leave the checkbox unchecked, it shouldn't warn you again, but you can also go to the Firefox preferences to change it:
Edit > Preferences > Security > Warning Messages > Settings
