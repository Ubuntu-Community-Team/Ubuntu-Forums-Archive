### Post by stabbyjones on 2011-02-21
It could be your library.
I was getting a problem similar to this where Rhythmbox would start and then segfault as the library opened.

Rename this Rhythmbox folder
$home/.local/share/rhythmbox
to
$home/.local/share/rhythmbox-backup

And reopen rhythmbox in the terminal so you can paste what comes out.

"rhythmbox -d" will start in debug mode which is what you want.

also install the debug package

apt-get install rhythmbox-dbg
