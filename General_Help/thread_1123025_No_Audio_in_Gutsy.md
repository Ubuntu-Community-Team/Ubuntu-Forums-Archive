### Post by tommcd on 2009-04-11
Ubuntu Gutsy 7.10 will reach end of life on 4-18-09 and will not be supported with updates after that. Plus the Gutsy mirrors will disappear, so you won't be able to install new programs either.
[http://www.ubuntu.com/news/ubuntu-7.10-eol](http://www.ubuntu.com/news/ubuntu-7.10-eol)

So rather that waste time with Gutsy, download and install Ubuntu Intrepid 8.10 Or Jaunty 8.04 beta. You could also try a dist-upgrade to 8.04 LTS. If you are having problems with Gutsy though, I would do a clean install rather that a dist-upgrade.
