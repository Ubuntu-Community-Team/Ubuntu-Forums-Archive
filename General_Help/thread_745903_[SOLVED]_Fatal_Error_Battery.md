I've got an elderly Gateway desktop pc running Gutsy. When I boot it up an message quickly flashes past that says something about a battery fatal error. I don't get a good enough look at it to say for sure the exact words. Computer appears to work fine. Could there be a small button battery on the motherboard that needs changing? Lance

### Post by niteshifter on 2008-04-05
Hi,

Yes, the CMOS backup battery is discharged below safe threshold and needs replacement. 

Do yourself a favor: Boot the machine and access the BIOS setup screen and write down (or use a camera and print the shots) all the various settings before disconnecting the battery.

---

### Post by Gillette on 2008-04-05
Thanks for the advice. I found I had a spare button battery matching the one on the mother board and put it in. The old battery had a 96 stamped on it so I imagine it has been in used for 12 years. Took photos as you suggested and booted up the computer without problems. I did not spot that battery message so it looks like problem is solved. Thanks!
