### Post by apetresc on 2008-02-12
Try installing alsamixer and seeing if everything is maxed there too. Also, use the 'M' key in it to see if any of the channels are actually muted.

If all else fails, try to install the new alsa drivers, this is a common workaround for many sound-related issues, and it's surprisingly easy to do.
