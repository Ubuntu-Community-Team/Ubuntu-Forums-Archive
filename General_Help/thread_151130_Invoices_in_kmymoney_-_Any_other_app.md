### Post by lhtown on 2006-03-28
I haven't used it for invoices, but Gnucash seems to be the premiere native Linux finance manager. 

It does seems to have suffered from a reeeeaaaaaally slow development cycle though. Using it is a totally retro experience not unlike going back to IE6 and Windows XP. UGH! As far as the latest stable version (there is indeed a new one in the works that has been released as Alpha and hopefully will make it into Dapper +1), what Gnucash lacks in obtuseness, it makes up for in difficulty of use. /end_vent

That said, however, it seems it really isn't far from being a great program for small businesses. Hopefully, the next version will offer a number of usuability enhancements and maybe the version following will offer some more general polish to get it up to speed and make it really competitive with the more popular proprietary programs.
