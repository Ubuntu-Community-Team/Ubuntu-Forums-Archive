### Post by Quackers on 2011-04-25
Welcome to UF.
Yes, you can install Ubuntu to the external drive. You do it the same way you installed it the first time, but selecting the external drive for installation and for grub to be installed to as well.
You will need to enter your pc's bios to set the boot device order so that the external drive boots before the internal drive (when it's attached).
With no cd drive you will need to burn the iso image to a usb flash drive and install from that.
