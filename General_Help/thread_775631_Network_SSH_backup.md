I'm using File Backup Manager (pybackpack) and I'm trying to figure out how to backup my laptop over my home network to my desktop. pybackpack supports SSH, and i've had a quick scout of the web but can't for the life of me figure out how to set up SSH on my desktop! ### Post by Monicker on 2008-04-30
Use synaptic or apt-get to install openssh-server.  This will install an ssh server on your desktop, and allow other machines to connect to it via ssh.
