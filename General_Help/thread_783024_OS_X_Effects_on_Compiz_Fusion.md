### Post by Zorael on 2008-05-05
To get mentioned CompizConfig settings manager (also called ccsm), install the **comzizconfig-settings-manager** package, via Synaptic or your package manager of choice.
```
$ sudo aptitude install compizconfig-settings-manager
```
