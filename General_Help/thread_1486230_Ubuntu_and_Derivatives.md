### Post by XubuRoxMySox on 2010-05-17
The differences are in the default desktop environments. 

**[COLOR="Purple"]Ubuntu[/COLOR]** uses the Gnome desktop environment, and Ubuntu configures them for the distro.

**[COLOR="DarkOrange"]Xubuntu[/COLOR]** uses the Xfce4 desktop environment: Simple, fast, kinda sorta like "Gnome Lite" you might say. Lucid Xubuntu has more lightweight applications as well. 

**[COLOR="Blue"]Lubuntu[/COLOR]** uses the LXDE desktop environment: Very lightweight and fast, but with less features and lightweight applications.

**[COLOR="DeepSkyBlue"]Kubuntu[/COLOR]** uses the KDE4 desktop environment and QT applications. The only issue with Kubuntu is that it is basically added on top of basic Ubuntu, which is "GTK-biased," or as some have called it, "Gnomecentric." On low-resource computers it might be a little slower than the others. But for the most part it's a matter of storage space on the hard drive (for libraries) rather than CPU or RAM useage.

Hope that explains a little more,
