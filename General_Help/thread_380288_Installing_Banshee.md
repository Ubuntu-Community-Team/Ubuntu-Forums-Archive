### Post by ArieT on 2007-03-09
You've downloaded the source code. That's not what you want to. Use your package manager in Linux for installing software. Press alt+f2 and type 'synaptic' or find the package manager in the menus.

### Post by DirtDawg on 2007-03-09
If you're new to Ubuntu and Linux, you really, really should browse the provided [help]("https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/index.html").

In particular, check out [chapter 3]("https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/add-applications.html"), and **don't forget** the part about [adding repositories]("https://help.ubuntu.com/6.10/ubuntu/desktopguide/C/extra-repositories.html").

Essentially, there's hundreds of software programs in an online library especially tailored for Ubuntu and you're free to install any of them. It's fun!
