### Post by oscarsfriend on 2007-12-18
Can you post the contents of your /etc/X11/xorg.conf file here?  Or at least check the following sections: The Device section should contain a line something like:

```

    Driver         "nvidia"

```

And the Screen section should contain lines something like:

```

    Modes      "1680x1050" "1280x1024" "1024x768" "800x600" "640x480"

```

It should start with (or at least include) the highest resolution your monitor supports.  Maybe yours is starting with "800x600"?  It is also worth taking a look at the Monitor section.

Also, it might be worth running nvidia's install script again, as I think this should write an xorg.conf file for you (but don't quote me on that!).

### Post by oscarsfriend on 2007-12-18
> **drewg said:**
> It probably looks like I'll have to install the driver again right?

That would probably be a good idea.  Your xorg.conf looks a bit of a mess to me.  Plus the lines that read **Driver "nv"** should read **Driver "nvidia"** for the driver downloaded from nvidia's website.

This section is most likely why you are getting the low screen res:

```
Section "Screen"
	Identifier	"Default Screen"
	Device		"Failsafe Device"
	Monitor		"Failsafe Monitor"
	Defaultdepth	24
	SubSection "Display"
		Depth	24
		Virtual	1792	1344
		Modes		[COLOR="Red"]"800x600@72"	"800x600@75"	"800x600@56"	"800x600@60"	[/COLOR]"640x480@75"	"832x624@75"	"640x480@72"	"1024x768@75"	"640x480@60"	"1024x768@70"	"1024x768@60"	"1152x864@75"	"1280x1024@75"	"1280x960@60"	"1280x1024@60"	"1280x960@75"	"1400x1050@60"	"1400x1050@75"	"1600x1200@65"	"1600x1200@60"	"1792x1344@60"
	EndSubSection
EndSection

```

If you cut and paste one of the 1024x768 entries to the front of the line (in front of the red above) and restart your X server, you should get a higher res for now, though I wouldn't count on getting 3D working just yet.
