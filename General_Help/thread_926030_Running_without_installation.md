### Post by mkvnmtr on 2008-09-21
Yes you can keep using the live disk till you wear it out. Anything you do on the live disk will not be saved when you turn the power off. You can't download stuff to mess up your system like I seem to enjoy doing because everything is done in ram. When you install on the hard disk you can change or add to the system. The installation adds a bootloader that gives you a chouce of systems. It is very simple and hard to foul up although some of us can do it.  It sounds like you are at the point where you should keep using the live disk and read a little here in the forums and in the wiki before you make the jump to install. Double click on the install and follow the steps till you get to the format stage a few times to see what you are getting into. At that point you can just press back to back out of the install without any changes to your disk.

### Post by lian1238 on 2008-09-21
Use liveCD when you're trying for the first time. If you like it, install it first, and then try some more because whatever you do while on liveCD stays in RAM. If you restart, it'll be history.

Try running the installer. You can create a new partition for Ubuntu out of the free/unpartitioned space on your harddisk or you can resize a partition to fit Ubuntu, although I've never tried that. I always do manual partition for fear that the installer mess something up, like format everything, but I'm just paranoid. ;)
