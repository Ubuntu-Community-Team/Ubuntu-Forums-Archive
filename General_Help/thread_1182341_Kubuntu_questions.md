### Post by yeats on 2009-06-08
> Can someone explain to me the differences in Ubuntu and Kubuntu. I see that Kubuntu uses different software than Ubuntu. Why is this?

KDE is built with the Qt development framework, while GNOME (used by "regular" Ubuntu) is built with GTK+.  Qt used to be licensed in such a way to make it incompatible with certain free/open source programs, which is one reason that GNOME was developed (this issue has since been resolved).  Since each environment is a "suite" (GNOME being less so), there are GNOME- and KDE-specific programs, but as the previous poster points out, each can run on the other without many issues (aside from appearance - sometimes KDE programs don't look as good in GNOME and vice-versa).

I've used both and like both, especially KDE 3.5.X (available in Hardy) and KDE 4.2 (default for Jaunty).  I had many problems with KDE 4.1.

### Post by Bigtime_Scrub on 2009-06-08
The main difference is Kubuntu uses KDE desktop and Ubuntu uses Gnome desktop.

Now I know you can use KDE apps in Gnome and Gnome apps in KDE, but I don't like doing that and I avoid it as much as possible. I stick to a desktop and use the applications native to it whenever possible to avoid bloat and unnecessary programs clogging stuff up. For most people that isn't a huge issue as hardware specs aren't normally a problem, but it can be especially if you are working with little hard drive space.

Also like someone mentioned native apps look better in their native environments.
