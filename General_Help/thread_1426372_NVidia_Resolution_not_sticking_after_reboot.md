### Post by realzippy on 2010-03-10
The modeline you added is not necessary when running NVIDIA-driver.You can delete it,although I do not know if it does any harm.

To fix the 1024x768,run (without sudo):

```
nvidia-settings
```

and select desired resolution, then hit "apply".(Not save to X config file,you have already done that).Then click *nvidia-settings Configuration* (down on the left side),and hit *Save Current configuration*.
Location should be *.nvidia-settings-rc* in your /home/user/ directory.
That should normally work.
If not,you can add a command in System/Settings/Autostart:

Name (your choice),Command:

**nvidia-settings -l**
