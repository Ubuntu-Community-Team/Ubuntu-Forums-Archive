### Post by 3rdalbum on 2011-08-05
Android phones update on-the-device - you don't need a computer to update your Android phone to the latest Android available for that device. Go to Settings, then About Phone and System Software Updates.

If you're talking about rooting your phone and installing a custom firmware that uses a later version of Android than what your phone can normally upgrade to, then Ubuntu can generally be used for this. The procedure varies depending on phone and firmware. Mine was able to be rooted using a program available for Linux, and then a custom firmware installed using Rom Manager on the phone.
