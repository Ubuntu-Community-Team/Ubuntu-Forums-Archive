If you've got a slow internet connection, apt-get install, etc, will run slowly but there isnt much you can do about that . About all you can do is only try to apt-get something when you arent using the internet fior any other reason. 

Also, if the CPU is slow enough, or you dont have much RAM, that can make apt-get install run slowly as well.
