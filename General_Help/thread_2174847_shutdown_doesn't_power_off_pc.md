### Post by sudodus on 2013-09-16
Welcome to the Ubuntu Forums :-)

Is it enough to press the power button quickly (less than one second) to shut it off? If so the processes have halted, and the computer is ready for poweroff, so it is maybe somewhat inconvenient, but not bad for the computer to poweroff with the button.

- Maybe you can change some setting in the BIOS menu system to make it power off from a system command.

- Have you tried the boot options noapic and nolapic?

The system ***should*** poweroff after the command

```
sudo shutdown -P now
``` or ```
sudo poweroff
```

Some hardware drivers for old computers were dropped after Ubuntu 12.04 LTS. So you may have better performance with 12.04 LTS than with 13.04 for poweroff, maybe also for other things.
