### Post by rozojc on 2007-09-11
Oooops, my mistake, apparently the answer to my problem is turning Beryl off... So, just for the record in case anyone has that problem again:

If you have Beryl on, the included VNC server (Vino) will not refresh screens in a Windoze viewer...
