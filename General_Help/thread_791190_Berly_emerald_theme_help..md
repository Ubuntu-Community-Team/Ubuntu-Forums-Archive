### Post by tjwoosta on 2008-05-12
to make emerald work you have to press alt-f2 and type emerald --replace (this will change the theme)

or you can add it to the startup commands by going to System-Preferences-Sessions and cick add (put your command in the command box)

if you want transparency please check out this thread because i dont wanna type it again

[http://ubuntuforums.org/showthread.php?t=791378]("http://ubuntuforums.org/showthread.php?t=791378")

(the fifth post down)

also it the pictures of the theme, they have made the terminal windows transparent by opening a terminal then right-click anywhere on it and go to edit current profile then click the effects tab clck transparent backgrounds and adjust then slider as desired
