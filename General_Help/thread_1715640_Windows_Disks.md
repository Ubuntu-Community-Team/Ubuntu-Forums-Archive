### Post by mrhhug on 2011-03-27
linux wont like to install on your NTFS formatted C: anyway. It is going to be best to create a new partition/volume to install linux into. (or just reformat the c: to a linux partition)

ubuntu has tools to let you read and edit NTFS drives without any trouble. i think they came preinstalled on lucid for me.
