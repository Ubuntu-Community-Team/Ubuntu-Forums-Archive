Crossover creates a pretend windows c drive in your home folder, so no, it isn't an ntfs partition.  If the install exited it looks like the installation failed.  Crossover is based on wine which seems to run MSoffice well, but it's a bit of a hit and miss affair with other windows applications.  I have no idea what type of application pplive is, but it's not mentioned in codeweavers database so it doesn't look like it's supported.  Have you managed to get it running with an earlier version of crossover in the past?  If so, try using that.  Alternatively, you could always try installing wine to see if pplive works with that.
