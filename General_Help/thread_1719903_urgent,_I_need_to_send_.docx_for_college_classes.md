### Post by gobbles414 on 2011-04-02
Two possible solutions to your problem...

1. If your teacher's concern is that non-docx files will display incorrectly on his/her computer, suggest that he/she allow you to submit your work via PDF files.

2. At least when using OpenOffice 3.2 in Ubuntu 10.10, it's possible to manually add a docx extension when saving in doc format. Word 2007 may or may not see the resulting file as a docx, so use this option with caution.
