### Post by gradinaruvasile on 2009-12-17
It can be done. The command is "openoffice.org3". Press alt+f2 and type it in. Press enter.
You can add a desktop icon:
Right-click on the desktop.
Select "create launcher".

Type:application
Name: Whatever you like
Command: openoffice.org3

Icon: click on the icon placeholder (the spring) and locate the icon - i dont know where it is exactly, i use the version from the openoffice site. That one has its icons named openofficeorg3-base.png Same for -calc -writer etc. The Start icon is named openofficeorg3-startcenter.png.
I attached it in case you cant find it on your system.
