### Post by mcduck on 2010-03-17
Gedit is surprisingly powerfull editor for coding, even more so if you enable/install some extra plugins.

If you want bracket matching then all you need to do is to go to Gedit's preferences, and on the View-tab turn on "Highlight matching bracket". 

..also make sure to check the plugins-tab, and install the gedit-plugins package. If you even after that feel that you are missing some feature then check the available extra plugins here: [http://live.gnome.org/Gedit/Plugins]("http://live.gnome.org/Gedit/Plugins")
