### Post by CrusaderAD on 2009-03-31
Does anyone know how to install additional sets of icons in Ubuntu? I know to go to Appearance > Customize and navigate to the Icons tab... but how do you go about installing packages?

---

### Post by marshall.robert on 2009-03-31
Download the package that contains the icons. Drag and drop the package into the Theme tab on the Appearance window. Or install it like any other theme.

---

### Post by Therion on 2009-03-31
> **marshall.robert said:**
> Drag and drop the package into the ***Theme tab*** on the Appearance window. Or install it like any other theme.
Therein lies the proverbial "rub".  You don't drop the file on the Icons tab, but rather on the Themes tab.  
It's a little counter-intuitive if you ask me, but that's how you do it.
