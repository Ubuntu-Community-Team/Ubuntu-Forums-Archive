### Post by creedce on 2010-06-06
Just found the fix for this after searching the forums. (Should of done this first I guess).

For any other person experiencing with interesting Ubuntu feature.

In Preferences -> Mouse -> Trackpad there is a option ticked to disable mouse while typing.

I guess this would be good for some of my silly friends who tend to move there trackpad while typing!

Thought this was a bug guess not.
