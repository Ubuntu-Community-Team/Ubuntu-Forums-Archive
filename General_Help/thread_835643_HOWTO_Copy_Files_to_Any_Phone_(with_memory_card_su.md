### Post by daemon3 on 2008-06-20
This was done with a Motorola Razr V3m with Verizon Wireless, but it should work for other phones as well.
[LIST=1]
[*]Buy a compatible memory card.  Depending on the size, they should be fairly cheap.  My 1G was around $30.  Go into your service provider's store to find the right memory card.
[*]Also buy a memory card reader if you don't have one already.  This should also be about $30.
[*]First make sure the phone recognizes the memory card.
[*]Good news: You don't need any drivers.  Plug the memory card into the reader and plug the reader into your favorite Linux computer (Yes, that should be Ubuntu).
[*]Navigate to your folders and drag in files.  It's helpful to know the file type.  For instance, my Razr only takes wma files (sadly, lame doesn't convert these very well).
[*]SAFELY REMOVE THE DEVICE.  Now put it back in your phone.
[/LIST]
That's right.  No bitpim or KMobileTools required.  It's fairly cheap to do, too.
