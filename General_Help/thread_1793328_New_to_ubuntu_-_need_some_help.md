1: Linux has different Desktop environments available. Ubuntu uses the GNOME desktop environment, with its own Unity shell on top. Kubuntu = Ubuntu using KDE instead, for example.

2: Linux doesn't work like Windows; there are no D, C, etc, drives. You will want to run the installer and choose "Install alongside Windows" or create a partition yourself and install it on that. If you want to view your Windows files from Linux, you should be able to.

3: Drag and Drop *is* that easy if you use Ubuntu's "Classic" desktop, which is just regular GNOME. When logging into Ubuntu, click your username. Then, on the bottom bar, change session to Ubuntu Classic. Much more stable and familiar, too. Unity is brand new and full of usability issues.

4: See my comment in #3 about the Unity issues.

5: Most popular distros depend on where you look. For the average joe, Ubuntu is the most popular. On servers, Red Hat (RHEL) and Red Hat clones (CentOS, Scientific) are the winners. For people who want a desktop OS similar to Ubuntu, but more bleeding edge and with the newest stuff, Fedora is a great choice. For people who want to build their OS from the ground up to meet their needs, Arch and Gentoo come to mind. Some people prefer Ubuntu remakes such as Mint because they are based on Ubuntu, but have some fun extras not included *by default* on Ubuntu. To me, that's pretty useless considering I can install the same things and turn Ubuntu into Mint within a few minutes. 

Distro all depends on your needs.
