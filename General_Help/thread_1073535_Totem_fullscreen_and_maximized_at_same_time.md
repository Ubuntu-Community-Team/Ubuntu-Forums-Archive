### Post by loki0347 on 2009-02-18
Hey all, I opened totem to watch some videos and it acts as if it's both full screen and maximized at the same time and of course F11 doesn't help because that would be too easy. [Screengrab]("http://php.uat.edu/~nicbernh/totem.png") shows the leave fullscreen option but it does nothing. Any ideas? Thanks and as always Linux rocks!

---

### Post by binbash on 2009-02-18
If you are using compiz, go to compizconfig settings manager > workarounds plugin and untick legacy fullscreen support

---

### Post by loki0347 on 2009-02-19
Thank you, my problem is solved.
