### Post by JorgeM93 on 2009-12-05
I saw this topic (in Spanish) [http://www.ubuntu-es.org/index.php?q=node/36213](http://www.ubuntu-es.org/index.php?q=node/36213) I haven't managed to download that game yet but the installation can be easy, but a bit risky. Just be shure to have libpng3 installed (or maybe something like libsdl too) then decompress the downloades file to your home directory. 

Use a terminal to type this commands 
```

cd /home/YOU/LOCATION OF THE DECOMPRESSED DIRECTORY
sudo cp -r usr/* /usr/

```At least that's what the tutorial in Sapnish says..... :P

EDIT: for playing type [alt]+[F2] and write *smw*
