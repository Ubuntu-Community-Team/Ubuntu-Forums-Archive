### Post by krismatth3 on 2006-10-25
The problem is you're missing a "ServerLayout" section:

```

Section "ServerLayout"
  Identifier "Multihead"
  Screen "ScreenLeft"
  Screen "Default Screen" RightOf "ScreenLeft"
  Screen "ScreenRight" LeftOf "ScreenLeft"
  Screen "ScreenRight2" RightOf "ScreenRight"
  InputDevice "Generic Keyboard"
  InputDevice "Configured Mouse"
EndSection

```

More is explained here: [http://www.paralipsis.org/2006/01/enabling-xinerama-in-ubuntu/](http://www.paralipsis.org/2006/01/enabling-xinerama-in-ubuntu/) , but you may well be able to wing it from the above sample.
