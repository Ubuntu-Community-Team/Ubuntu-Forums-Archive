### Post by Brandon Williams on 2010-06-09
If you want it to conform to the theme and put the buttons on the left, open the Options dialog and go to the 'Personal Stuff' tab. There, you can tell it to 'Use the GTK+ theme' and/or 'Use system title bar and borders'.

Yes, you should re-enabled the chrome repos to make sure you get security updates, etc.
