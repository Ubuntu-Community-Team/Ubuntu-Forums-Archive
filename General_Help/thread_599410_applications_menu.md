Someone can correct me if I'm wrong, but I'm not sure you can designate specific programs as being in the menu system wide, and some as just for certain users.

I think the closest results to what you're after would be affecting what is in the menu "Main Menu" (under Preferences, or by right clicking your menu and selecting Edit Menu") Then you will arrive at the main menu screen, where you can see a list of checkboxes for programs, checked ones will appear in this given user menu, but I *believe* when you add a new program "New Item" that will show up across the board, in each users Main Menu menu. So each user would go to their preferences and check the icons they want to show up in their menus.

As for downloading a given program and its icon not showing up, believe me, I know your pain, I like nice little specific icons to the program, etc, too. You can go New Item and select the program, either by entering in a command (which will sometimes auto-select the appropriate icon) or by browsing and picking an executable (/usr/bin has a lot).
