### Post by kokkus on 2008-09-26
I agree, do not buy vista.
If you would like to run a dual boot you should run XP with Ubuntu insted.
Buy your PC with an Ubuntu XP cd if possible or get XP from a friend or something. Install ubuntu after you have installed xp and not before.
Grub will then add ubuntu to the list, but if you install xp after ubuntu you have to recover grub or install supergrub from windows.
I have pretty much the same PC with the same graphic card and everything works fine. I run WinXP and Ubuntu 8.04.

I hope it will work fine for you as well and I'm wishing you good luck with your new PC:)
