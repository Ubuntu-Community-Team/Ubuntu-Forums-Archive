### Post by 3rdalbum on 2007-02-17
Audacity can open and allow you to edit WAV files, among many other file formats. I don't know precisely what format your tracker requires, but the command-line program "sox" (available from the repositories) will be able to convert a WAV to any obscure format you need.

For usage information, type "man sox" into the terminal after you have installed the program.
