### Post by mcduck on 2016-11-03
Depends on how much RAM you have, but the regular Ubuntu should work just fine, it's mainly a question of if it's going to feel fast enough for you and if there's going to be enough memory left for the applications you wish to use (for example web browsers tend to use loads and loads of memory, but that depends quite a lot on how you use them, how many tabs you usually keep open at the same time etc.).

Then again as long as you are not short on hard drive space, you can always start with the normal Ubuntu desktop, and then if you feel you'd want something lighter, add Lubuntu or Xubuntu desktop and switch to using that instead.

I would definitely recommend using latest LTS version, 16.04), or just the latest version, instead of picking older release. Ubuntu itself hasn't really gotten much heavier over years, it's just the desktops and apps (and web pages!) that need bit more resources these days, so running an old version doesn't really gain you anything the same way picking an older Windows version would for old hardware.
