### Post by hawkmage on 2011-01-24
It is not quite as functional as TOAD but I use the Oracle SQL Developer.  It is free and has an add-on for DB2 as well as a couple of other popular DBs as well.  It is a Java Application so you will need a JDK installed, preferably the Sun/Oracle one.

### Post by hojjat on 2011-01-24
Thanks for your quick reply. In what ways is Toad more functional than the software you just mentioned?

---

### Post by hawkmage on 2011-01-24
TOAD is more for a DBA and has more built-in DBA functions where as Oracle SQL Developer is more of a developer tool.  If you are doing simple DB stuff Oracle SQL Developer should work but if you are a DBA then TOAD is better.  You may also want to take a look at Tora on Source Forge, it is a free TOAD like tool that I some people I know use but I can not say anything about it my self since I have not used it.
