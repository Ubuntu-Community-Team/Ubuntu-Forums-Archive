### Post by nebu on 2010-08-18
this is only the case when the application is in some scripting language so they can be interpreted on the go.... 

however, most linux programs are not built in these languages, they are built in languages which complile the code into binary executables which are architecture dependant and sometime even have other dependecies which are not satisfied by default on some *nix distros ie. u cant use an app which uses apples cocoa toolkit to run on ubuntu!

so the safest way is either installing it from the synaptic repository or maybe download the source and install it from there.


or sometimes people also give out binaries of their programs if they are not licensing their products under a copyleft license... like the adobe flash player..... so u can probably search for open source alternative to these...
