### Post by detergen on 2010-12-15
I've got the same problem, but the next script produce for me  result in A4
```
#!/bin/bash
    tmp1=$(mktemp)
    tmp2=$(mktemp)

    pdftops -paper match $1 $tmp1
    psbook $tmp1 | psnup -2 > $tmp2
    ps2pdf $tmp2 $1
    rm $tmp1 $tmp2

```

ps2pdf make the target pdf in A4 from ps USletter  file
