### Post by fragos on 2007-07-27
All software has some bugs.  I rarely see them using Ubuntu.  I stick with the Ubuntu repositories.  They contain binaries that have been tested with this release of Ubuntu.  If you packages such as Automatrix for installs you may run into problems.  Automatrix isn't part of the Ubuntu distribution.  If you install a .deb package not specifically built for Ubuntu you may also have a problem.  If you are in the habit of compiling the latest versions of packages you can also run into trouble because they haven't been tested with Ubuntu.  Also since they weren't installed with the apt framework, the package managers won't know about their existence.  Not all that is open source will work with any distrobution.  Packages with version numbers starting with 0. are by definition works in progress and may work but are likely to missing planned features.
