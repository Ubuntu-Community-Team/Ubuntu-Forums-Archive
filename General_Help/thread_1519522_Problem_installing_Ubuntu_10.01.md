### Post by 3Miro on 2010-06-28
On 9.10 the menu with choice for "try/install" appears as soon as you start from the CD (a few seconds maybe). However, the menu was somewhat ugly.

On 10.04 the menu appears after some time and it is far more "polished". You should still get something that looks like the image that I have uploaded. This should be before it asks for a time zone. Ubuntu only starts to ask those questions when you choose install options.

I don't know if this is an issue with your CD or hardware or you simply missed the new menu. Give it another try and if you don't get the "try/install" screen make a CD or USB drive of your own. (You can make your own CD from within Windows XP)

### Post by ajgreeny on 2010-06-28
I suspect this eco-disk could be from Linux Format, a UK magazine or one of its offshoots, which has been using these disks for a few months now, following an abortive try last year at one point, which had to be stopped as most machines with letter-box DVD slots failed to get the disks to work.  This has apparently been solved for most people now, but I believe there are still a few drives which do not work with the disks.

However, it is also possible that you may just have some hardware that is incompatible with the new live version of Ubuntu.  I can not get the Ubuntu disk from LXF 133 to boot and run on my machine, even though I am already running Lucid without any major problems on that same machine.  Make whatever you can of that!  I installed my system from a UNR live CD that I already had, and then just added the ubuntu-desktop to it to get a standard Ubuntu system, and that caused me no problems.

Your machine can obviously read the disk, or you would not be able to get as far as you have, so maybe it is worth trying to do a disk copy from the eco-disk to a proper DVD, assuming it is not copy protected, which I think is so if it is an LXF disk.  A real DVD may work where the eco-disk will not.
