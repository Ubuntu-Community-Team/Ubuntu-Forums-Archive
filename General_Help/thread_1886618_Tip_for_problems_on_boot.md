Has your Ubuntu mysteriously started to hang on bootup? If so there is one simple thing to check. Is there is a button on your machine that turns wifi on/off? Make sure it's turned on because Ubuntu 11.10 will hang on scripts/init-bottom unless it is.

If you have a problem turning it on, try booting into Windows first (if you have it installed). Then try.

It's possible the problem maybe specific to certain hardware. I found this problem on a friends laptop, a  HP G72.
