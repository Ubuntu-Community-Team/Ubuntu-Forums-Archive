### Post by cogier on 2011-02-25
There should be no problem running 10.10 on the specs that you have. But you can check here [https://help.ubuntu.com/community/Installation/SystemRequirements]("https://help.ubuntu.com/community/Installation/SystemRequirements")

Have you tried running the CD as a Live CD to see what happens?

When you start the CD and you select your language there are also other options under the F6 menu that you can select to see what does and does not work. Further advise on this is available here [https://help.ubuntu.com/community/BootOptions]("https://help.ubuntu.com/community/BootOptions").

Don't give up there is a good chance Ubuntu will work with a little perseverance.
