### Post by timmorrow on 2005-05-11
I just upgraded mozilla-firefox to the 1.0.2-0ubuntu5.1 (hoary-security) version.

However, it seems that all dialog boxes (Tools --> Downloads or Edit --> Preferences, for example) give me some sort of XUL parsing error.  Aside from that, browsing the web still works fine.

Anyone else had this problem?

FYI, I downgraded to the 1.0.2-0ubuntu5 (hoary) version by doing the following:
  * Started Synaptic package manager
  * Highlighted mozilla-firefox
  * Selected menu Package --> Force Version
  * Selected previous version in dropdown, clicked Force Version
  * It prompts for uninstalling, mozilla-firefox-gnome-support and ubuntu-desktop.  let it.
  * Applied the changes
  * Installed mozilla-firefox-gnome-support again, forcing version in a similar manner

Got back my dialog boxes.
