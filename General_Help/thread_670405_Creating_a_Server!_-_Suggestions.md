### Post by indytim on 2008-01-18
It's a pretty broad area that you want to "specialize in".  For the web development front, I'd recommend that you could probably get up to speed on one or more of the parsers out there.  My choice is PHP.  It has a nice fit wth a MySQL backend.

Since you've got LAMP installed, the development can take place on your local web server.

As far as the web page layout etc., again you're faced with a large array of choices.  You can hack out the code for individual pages, make your pages dynamic through, again, say PHP and MySQL,  Or you could go the route of using any of a number of excellent CMS packages out there (I'm using Joomla for my public sites).

If you're leaning toward the actual server admin, then you will want to, among other things get heavily into web server config and all of the other toolsets that go with it.  Again, many areas to concentrate on.

My overall suggestion is to concentrate on one area and develop the necessary expertise before branching out beyond that baseline.  Otherwise, you will probably drive yourself nutso sooner rather than later.
