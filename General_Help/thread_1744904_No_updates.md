### Post by 3Miro on 2011-04-30
Go to Synaptic Package Manager -> Settings -> Repository then  find "sources" and pick other. After you pick other, a new window will pop up and you can select "fastest mirror" or something like that. This will find the closest server next to you and will hopefully let you install the updates.

Note that for the last couple of days, there were maybe 2 - 3 updates.
