### Post by Voynix on 2008-07-31
Hi Ampers,

Evince or any pdf viewer can open the Adobe Acrobat eBook Reader format as they  are simple pdf files. Ubuntu can open pdf files by default.

There are some utilities to convert the Microsoft E-book (lit) format to html (convertlit). Unfortunately, Microsoft Reader is a proprietary format and there are no readers for linux. 

Can you download the pdf files? Try this option first, but if you happen to have many .lit files, you can use convertlit. 

Voynix
