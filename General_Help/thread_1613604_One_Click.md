Is it possible to set up Ubuntu 10.10 where one click will open a document as opposed to double clicking ?

Bruce

### Post by 3Miro on 2010-11-04
Places -> Home. When nautilus opens, go to Edit -> Preferences. Select the Tab with "Behavior" and then "Single Click to Open Files".
