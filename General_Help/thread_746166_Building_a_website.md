### Post by st0n3cutt3r on 2008-04-05
Just make your website and don't worry about it. No one will ever (except maybe you) see the site on more than one operating system. Further, the web is a fluid thing, the very point of which is to be system-independent, and served up any way the user wants. Translation: "Don't worry about the subtle differences."

[http://www.hotdesign.com/seybold/everything.html](http://www.hotdesign.com/seybold/everything.html)   Don't use tables!
