### Post by hal736 on 2008-01-31
Current version via apt-get is wine-0.9.46.  
latest version on the  [wine website]("http://www.winehq.org/") is 0.9.54.  

If you want easy, *apt-get install wine*  is the fastest and easiest.

If you want the latest version, and you are comfortable installing from source code, go to the wine website and download the source and build it.

Sorry, you can't have both...:(
