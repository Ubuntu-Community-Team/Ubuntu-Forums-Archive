### Post by robertpolson on 2007-01-19
One of the reasons I could not switch from Windows to Linux was because the font size on my Internet Explorer in Linux was so small on my 16080x1050 resolution that it was pain to look at.

There was nothing I could to to change it. CTRL ++ or CTRL+mouse wheel did produce big enough fonts. I was forced to use Windows.

Today I finally found a simple solution.

Get Internet Explorer from here:

[http://www.tatanka.com.br/ies4linux/page/Main_Page](http://www.tatanka.com.br/ies4linux/page/Main_Page)

Once you install IE, open it and go to Tool - Option - Accessibility

There check: Ignore font sizes specified on web pages


Done.

Now you can zoom in really well.

P.S. If you enjoy using this project, please donate to the author as he has done some great work for the Linux community.
