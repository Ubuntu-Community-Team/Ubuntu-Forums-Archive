Thanks, that solved my problem as well.

I was using Sunbird v0.8 in Ubuntu 7.10. I just updated to Ubuntu 8.04, and my Sunbird v0.8 would no longer work (it wouldn't even appear to be loading up).

Turns out, I did need to install libstdc++5 as well (did a search for it in Synaptic and marked it for installation).

After that, my Sunbird 0.8 worked perfectly.
