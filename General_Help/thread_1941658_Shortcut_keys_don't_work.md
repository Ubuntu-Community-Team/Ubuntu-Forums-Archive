### Post by Ralph L on 2012-07-06
[http://maketecheasier.com/reactivate-alt-f2-shortcut-in-gnome-shell/2011/10/24](http://maketecheasier.com/reactivate-alt-f2-shortcut-in-gnome-shell/2011/10/24) 

Anyway, as with all Linux distro, you can customize almost everything you want. Here’s how you can fix the issue and reactivate the “Alt + F2&#8243; shortcut keys.
1. Go to System Settings and click the “Keyboard” icon.

2. Click on the “Shortcuts” tab. On the left panel, select “System”. On the right, select the first entry “Show the run command prompt” and click on “Disabled” to activate it. You should now see “New shortcut…”  Type Alt_F2.
