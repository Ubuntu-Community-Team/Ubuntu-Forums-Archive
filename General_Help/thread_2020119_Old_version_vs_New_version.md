### Post by msammels on 2012-07-07
Well, personally I wouldn't recommend sticking around with GNOME 2.x. however if you do install 10.04 it will come with GNOME 2.x.

Personally, I can't remember if 10.04 was LTS or not. If it was, then you'll be fine for updates, but if it wasn't I would steer clear and just go with 12.04.

However, you can install Ubuntu 12.04 and then the MATE or Cinnamon interfaces from Linux Mint. MATE is a fork of GNOME 2.x. that is kept up to date.

### Post by uRock on 2012-07-07
If you are wanting the old style Gnome panels and menus, then open Ubuntu Software Center, then search for and install gnome-panel. Once installed, log out, then click the little gnome foot icon by your login and select Classic Gnome. You will now have the 10.04 look on your 12.04 system, which will recieve security updates for the next 5 years.
