### Post by cariboo on 2008-12-27
Open up Firefox and in the address bar type:

```
http://localhost:631
```

This will open the cups web interface. Click on the printers tab, next click set printer options. You should now see a section with your printer model called General. Check Print Quality and set the print qaulity you want. I have mine set ot Fast Economy, you may want something different.
