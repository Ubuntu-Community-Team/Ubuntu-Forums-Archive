### Post by spiderbatdad on 2008-07-26
```
dmesg > ~/Desktop/dmesg.txt
```this will write a file to your desktop, containing a diagnostic message from the kernel. It includes your start up routine. You can read through it for hints. You can also upload it here by right clicking on the file and selecting: "create an archive." Upload the archive using the manage attachment tool below.
