### Post by Lekensteyn on 2009-11-05
It looks like you've set your source to a CD drive.

K Menu-> Applications -> System -> System settings
Add & Remove software

Settings > Edit Software Sources

Tab Kubuntu-software
Check everything

Tab Other software
Uncheck everything that starts with cdrom:

Confirm these changes.
