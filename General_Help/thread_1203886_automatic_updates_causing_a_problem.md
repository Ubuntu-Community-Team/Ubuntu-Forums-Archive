### Post by earthpigg on 2009-07-03
i have a dell mini 9.

dell's support for ubuntu is sub-par... worse than no support, as a matter of fact. i tossed dell's version of ubuntu about a month after i purchased it.

download the 9.04 i386 desktop .iso from ubuntu.com, and use system -> administration -> USB startup disc creator to turn a USB thumb drive into a bootable live ubuntu "CD".

boot from it, install regular normal everyday ubuntu 9.04, be free of dell's incompetence, and have an install that is much more compatible and will run any ubuntu .deb you come across.

install and try the ubuntu-netbook-remix package, if you like. i dont have it, but some prefer it.

either way, the general consensus is: dell's version of ubuntu is crap, and dell's linux support is worse than if it did not exist at all.



dont get me wrong though, the mini 9 is a great piece of hardware, and kudos to dell for at least making the attempt :D
