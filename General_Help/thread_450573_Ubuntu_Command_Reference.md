### Post by MontanaMax on 2007-05-21
The main documentation community for Ubuntu is at [https://help.ubuntu.com/community/](https://help.ubuntu.com/community/) - that includes links to the official online documenation, and all the community driven wiki managed information as well. 

[https://help.ubuntu.com/community/Wine](https://help.ubuntu.com/community/Wine) or the forums are probably the best resources for wine help. 

Command line documentation is common to almost all Linux systems, I tend to end up back at this site everyone once in a while for a overview of  tools  [http://www.karakas-online.de/gnu-linux-tools-summary/](http://www.karakas-online.de/gnu-linux-tools-summary/)

Good luck!
