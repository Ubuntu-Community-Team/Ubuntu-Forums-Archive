### Post by pbpersson on 2009-03-23
You can also go to System-->Preferences-->Main Menu and add it there.

Regardless of whether the executable is a .bin file or a .sh file, you can browse to it and specify it.  You can also add an icon of your choice or if one is included with the application you can browse to that directory and use the icon that came with it.
