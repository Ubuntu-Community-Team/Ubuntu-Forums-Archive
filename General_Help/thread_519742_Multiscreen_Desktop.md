### Post by redhead_demon on 2007-08-07
When ever I do a fresh install of ubuntu on wubi I enable desktop effects and they work great how everwhen I reboot I loose the four desktops but the effects remain working fine anybody got an idea on how I can correct this so I have all four desktops on the cube?

---

### Post by tuxcantfly on 2007-08-07
open gconf-editor

go to apps/compiz/general/screen0/options

find the key called "Desktops" and set it to 1

find the key called "hsize" and set it to 4
