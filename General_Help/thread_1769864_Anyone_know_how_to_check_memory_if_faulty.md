### Post by pythonscript on 2011-05-28
You can select the memtest option from the initial grub boot window and follow the process from there; or, if you don't have the grub window set to show up, you can always download a copy of memtest, boot it, and run the test from there.

Here's the website for memtest: [www.memtest86.com](www.memtest86.com)
