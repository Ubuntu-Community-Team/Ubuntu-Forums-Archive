### Post by BlackBaron1024 on 2008-07-01
d4x saves de tempoarary files in the same place, but with '.' before it, so that it's hidden.
For example, while downloadin "test.jpg" it has it as ".test.jpg" in the same location.
There's an option in Nautilus to view hidden files, or if you are using the console, you can type "ls -a" to list all files (including hidden).
