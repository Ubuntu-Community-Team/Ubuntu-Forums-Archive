### Post by Rubi1200 on 2010-12-14
Hi,
thanks for posting the results.

The problem is that GRUB is installed in the Windows partition on sda2:
> Grub 2 is installed in the boot sector of sda2
To rectify this you need to restore the Windows bootloader:
[http://ubuntuforums.org/showthread.php?t=1014708](http://ubuntuforums.org/showthread.php?t=1014708)

Once that is done, you need to reinstall GRUB from the LiveCD:
```
sudo mount /dev/sda5 /mnt
```

```
sudo grub-install --root-directory=/mnt/ /dev/sda
```

Once booted back into Ubuntu, run this command to pick up the Windows installation:
```
sudo update-grub
```
