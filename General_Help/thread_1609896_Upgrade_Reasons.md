### Post by trikster_x on 2010-10-31
If you are going to upgrade, I would suggest you go with the LTS or wait and see what happens with 11.04 when we get that one in 5 1/2 months.  I say this because I upgraded to 10.10 from 10.04 and have been regretting it.  There are some major performance issues I have which seem to be common in 10.10, such as:

Increased boot times
Increased shutdown times
Longer login times
Laggy directory changes in nautilus (and when I say this, I'm talking 10-15 secs, not momentary hangs)
Programs taking longer to start

And really, there were not enough improvements to warrant having to deal with these slowdowns.
