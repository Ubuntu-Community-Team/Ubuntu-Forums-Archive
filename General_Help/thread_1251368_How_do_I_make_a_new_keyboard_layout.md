The way to type Greek Polytonic in Ubuntu 9.04 (and is the preferred way for Greeks) is described at
[http://simos.info/blog/archives/888](http://simos.info/blog/archives/888)

That layout does differ from the polytonic layouts that are common outside Greek. I think those layouts were influenced from the TLG layout.
Two such examples are
[http://www.frame-poythress.org/poythress_articles/2007KeyboardGreekHebrew.htm](http://www.frame-poythress.org/poythress_articles/2007KeyboardGreekHebrew.htm)
and
[http://www.ntresources.com/](http://www.ntresources.com/) (cannot find the exact URL atm, sorry).

You can create or edit a keyboard layout using the tool at
[http://github.com/simos/keyboardlayouteditor/tree/master](http://github.com/simos/keyboardlayouteditor/tree/master)
