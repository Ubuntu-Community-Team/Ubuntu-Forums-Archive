sounds like the problem it's not so much about the Ubuntu 11 itself, but rather specific apps that is chewing up memory and cpu resources.

you should also mention the spec of your computer for people to have a goode idea what might be the issue

like, maybe the machine is too old to handle the newer OS + APPs

there are lots of things to consider
