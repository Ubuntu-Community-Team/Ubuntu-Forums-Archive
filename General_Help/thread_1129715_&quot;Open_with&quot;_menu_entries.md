### Post by TimDaniels on 2009-04-19
If I double-click on a .pdf file, Evince ("Evince Document Reader") becomes the PDF reader.  Evidently, Evince is the default PDF reader.

When I right-click on a .pdf file, and I select "Open with", the popup menu lists Adobe Reader, Firefox, Opera, and "Open with Other Application".  Selecting the latter results in the "Open With" dialog box that lists a bunch of applications, including "Document Reader" - which happens to be Evince.

How can I:

1) Make Adobe Reader the default PDF reader?

2) Change the entries in the popup menu so that just the apps that I want are listed in it?

3) Change the wording of the entries in the popup menu and in the "Open With" dialog box so that, for example, "Document Reader" becomes "Evince Reader"?

TIA,
*TimDaniels*

### Post by Tim Sharitt on 2009-04-19
Right click a .pdf file (or any other file type you would like to change) and select properties. In the properties window, click the Open with tab. From there you can add or remove applications from the list and check the radio button beside the application you would like to be the default.
