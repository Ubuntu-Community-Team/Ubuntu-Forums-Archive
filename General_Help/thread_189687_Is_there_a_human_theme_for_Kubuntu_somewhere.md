### Post by n8bounds on 2008-05-27
This is the color scheme I found "Flamenco.kcsrc"
```

[Color Scheme]
Name=Flamenco
activeBackground=241,137,28
activeBlend=241,137,28
activeForeground=255,255,255
activeTitleBtnBg=127,158,200
alternateBackground=231,236,254
background=234,235,238
buttonBackground=243,246,251
buttonForeground=40,40,40
contrast=7
foreground=40,40,40
frame=239,239,239
handle=239,239,239
inactiveBackground=234,235,238
inactiveBlend=231,232,236
inactiveForeground=172,172,172
inactiveFrame=239,239,239
inactiveHandle=239,239,239
inactiveTitleBtnBg=167,181,199
linkColor=26,77,186
selectBackground=241,137,28
selectForeground=249,249,249
shadeSortColumn=true
visitedLinkColor=109,32,33
windowBackground=255,255,255
windowForeground=40,40,40

```

Set up two panels, downloaded a kicker background from kde-look that I liked, and got the Flamenco (tangerine + tango + extra stuff) icons installed. I use the QtCurve style, which I liked better than Klearlooks.
