### Post by bodhi.zazen on 2008-04-28
> **VictorR said:**
> I have installed gnomesu and this works.
A new question is: Can I make an alias or something like this to call gnomesu when I type gksu? The reason is that gksu is present in all administrative launchers, and it is also being checked first in Nautilus scripts.

gksudo is a link to gksu , so no difference between these two:

> lrwxrwxrwx  1 root   root        4 2007-09-16 08:42 gksudo -> gksu> **ryanhaigh said:**
> Or gksudo rather than gksu.

```
alias gksu='/usr/bin/gnomesu'
```

Assuming that is the location of gnomesu. If not substitute the correct path (which gnomesu).

You can then add that to .bashrc :

```
export gksu='/usr/bin/gnomesu'
```to call gksu, use the full path : /usr/bin/gksu
