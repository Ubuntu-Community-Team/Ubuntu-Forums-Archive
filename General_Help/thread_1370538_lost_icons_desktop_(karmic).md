Hi guys, 
I don't know if this is a bug or not but it did not seem severe enough to put forward as a bug report but I am sure the forum staff will redirect it if needed.

Notwithstanding what is written in the thread below (and maybe because of it) I will put down what I found.

[http://ubuntuforums.org/showthread.php?t=803626&highlight=lost+icons+desktop]("http://ubuntuforums.org/showthread.php?t=803626&highlight=lost+icons+desktop")

I operate Ubuntu as a workstation to my Windows domain so I share directories on the windows server.

How I do this is to use the 'connect to server' link from the menu system.  After adding the credentials I then drop a bookmark to this window.  I have done this ever since Hardy (with the exception of Intrepid) and have never had any problems.

I have done this with Karmic and it seems to work just fine, however when I close the window with the connection then I lose all the icons on the display.  Please note here the icons are not removed from the 'Desktop' directory.  I can navigate to that location and see them, they are simply not displayed.  They can be restored by logging off and back on again (and it goes without saying by a reboot).

Now logging on would display a ghost icon to the shared directory on the desktop.  Clicking this icon would link me the the shared directory without any problem but once the window is closed - lo! no icons again.

Now, for me, I have got round this.  I simply removed the bookmark and work without it.

As a one time developer I would hazard a guess that the problem I have found is possibly related to the one quoted in the referenced thread.

I have not been able to give much detail on this as I am at home at the moment and therefore removed from by Ubuntu machine :( but if there is a demand I will try to recreate the symptoms next week when I am back at work.

If nothing else this thread might just help out someone else who has related problems.

Thanks, as ever, to all the guys and gals who do the real work.  Without you none of us would be here.
