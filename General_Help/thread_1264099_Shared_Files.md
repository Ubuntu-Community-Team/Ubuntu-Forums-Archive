### Post by scragar on 2009-09-11
I think the easiest way would be to make a directory that's writeable to everyone, have the users put the files in there and put read perms on the files.

Unfortunately this method would let all users see file names for files that aren't intended for them(And if you wanted to limit it further you could get groups involved, but again, problem with file names...)
