### Post by killermist on 2011-11-15
As lightweight goes, lubuntu and xubuntu are the 2 primary recommendations.
I've previously had some good success with Pepperment Linux, which is either lubuntu or xubuntu with added cloud-type functions layered in.  It still has access to the full ubuntu library, so if you can't find what you want in a cloud-type version, you can install the more local version of what you want.

As to 32 or 64 bit, that's a bit of a toss-up unless you have specific things that need one or the other.
More than 2Gb memory, probably want to go 64bit.  Doing Virtual Machines, might want 64bit over 32.
Regular basic use, either 64 or 32 should be fine.
Flash (like for flash games and such) doesn't really play well with 64bit yet, but Google Chrome (as packaged directly by Google) has Flash built-in, so if you were going to use Chrome anyway, the 64/32 distinction may not be meaningful.

As far as instability goes, unless you're overclocking that AMD processor, I don't foresee instability as a problem.  All the machines I've had AMD chips in ran with much stability.  I did regularly have various Pentium 4 chips freeze solid under load, and have since then permanently abandoned the Pentium 4 as an option.
