### Post by bored2k on 2005-08-22
1. [http://ubuntuguide.org/#extrarepositories](http://ubuntuguide.org/#extrarepositories)
2. sudo apt-get update
3. sudo apt-get sun-j2re1.5
4. sudo apt-get install azureus
5. killall gnome-panel

Select it from the menu. Azureus is kinda "heavy" on any box. If you don't like (you will, but its just massive and chubby), after the first two steps do
3. sudo apt-get install bittornado
4. killall gnome-panel
