### Post by todak on 2009-03-10
xubuntu is not a program, but rather an operating system. Ubuntu uses the Gnome desktop environment. Xubuntu use the xfce desktop environment and kubuntu uses KDE. If you want to install the xfce desktop environment and try it out, simply open terminal and type ```
sudo apt-get install xubuntu-desktop
``` and then before you log in next time, choose xfce in your session type at the log in window.
