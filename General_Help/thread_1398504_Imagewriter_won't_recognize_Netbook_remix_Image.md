### Post by Waveridr85 on 2010-02-04
I think I may have solved my problem.  The problem is with the installation instructions for Ubuntu netbook remix.  It says to burn the img file to disk using imagewriter.  the problem is you can only download an iso file like i did so it will not work.  What i did was create the disk and then  select System>Administration>USB Startup disk creator.  it should work from there :crossesfingers:
