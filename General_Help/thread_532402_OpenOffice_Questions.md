1) In OO, go to Tools > Options and in the box that opens go down to Load/Save. You should see a setting for your default file type.

2) Right click in the list of files and you'll see the option for showing hidden files. Click on it to remove the check mark.

3) Under the file menu, click on Export as PDF. You can also create fillable PDFs this way.

### Post by kebes on 2007-08-22
> **Dylnuge said:**
> 1. Is it possible to automatically have my documents saved as .doc files? I need to share a lot of files, so it is much easier for me if I already have them in a universal format (although I am already getting .docx files from people who don't realize that the whole world does not want/use Office 07).

Go into the options (on the menubar, go to: Tools > Options). Then pick the "Load/Save" group on the left, and the "General" sub-group.

Then you will see a section for "Default file format" an you can modify the "Always save as" to "Microsoft Word 97/2000/XP". The click "OK" to activate the change.

> 2. I am shown hidden directories in the Save screen (ones that begin with .) in both OpenOffice and Firefox. I tried pressing Ctrl-H, but it does not help. Any ideas?
I believe this is controlled at the OS/GUI level (and is not an OpenOffice specific thing). Are you using GNOME or KDE? There will be an option for showing/hiding hidden files by default.

> 3. Can I save to .pdf format in OpenOffice. I read that I could somewhere, but cannot find the option anywhere.
There is a "PDF" button in the toolbar, which is a quick way to export a PDF copy. Or go on the menubar: File > Export as PDF...
