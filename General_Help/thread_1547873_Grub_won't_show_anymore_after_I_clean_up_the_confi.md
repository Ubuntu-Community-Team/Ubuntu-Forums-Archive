sudo update-grub

### Post by mathfreak123 on 2012-03-26
You can try editing your /etc/default/grub file. Comment out the "GRUB_HIDDEN_TIMEOUT" line by placing a # at the start of the line, and set the "GRUB_TIMEOUT" value to -1. This should force the GRUB menu to appear.
