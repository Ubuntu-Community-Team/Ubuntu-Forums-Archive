### Post by Ben Crisford on 2010-04-16
You don't need to pay money for a decent looking set-up.  Whenever I have a fresh install I follow these steps:

1) Find a wallpaper I *really* like. ( for my newest install it was this one - [http://i42.tinypic.com/eu4mkg.png](http://i42.tinypic.com/eu4mkg.png) )
2) Find a theme that works with it. (I used ambiance, which ships with Lucid)
3) If necessary, customise the panels. (Rick Click> Properties> Background)
4) Install some panel applets and maybe a dock. (I have a weather applet on my top panel and Cairo-Dock with OpenGL)
5) Customise dock and applets.

And after following those steps I end up with something like this:
[http://i39.tinypic.com/mkj7f9.png](http://i39.tinypic.com/mkj7f9.png)
Or this:
[http://i55.photobucket.com/albums/g135/tronex/desktop-latest.png](http://i55.photobucket.com/albums/g135/tronex/desktop-latest.png)

EDIT; In the first image I use Cairo-dock (available from the Software Centre I think), in the second I use avant-window-navigator (also available in the Software Centre if I am not mistaken).

Also, if you are going to buy customised linux from ebay or somewhere, then be *very* careful what you are getting.  Because all sorts of people could be making that software, you dont know what could be lurking around in it.

### Post by House101 on 2010-04-16
First, you would need to say what is your syle. is it dark, colourful, glossy ect. Then you can download any theme you like (I prefer gnome-colors, it comes with some nice icons and links to some matching backgrounds, it also comes in lots of colours), and to install it all you do is go to settings>preferences>apperance then drag and drop the .tar or .tar.gz and so on, into the apperance window and it should import then you can click customize and be sure to check out all the tabs. 

You can also download icon sets, scripts/widgets for the desktop (they will install like regular packages), other backgrounds, and right-clicking on the main dock will give you more options
 
NOTE: This is based from jaunty and karmic they may have changed it in lucid
