### Post by TheFu on 2013-09-12
Can't really tell what the issue might be, but with both Win7 and Ubuntu showing issues, I'd lean towards a hardware problem.  
* failing hard disk
* intermittent connection - sata cable
* cracked motherboard.

I'd make a **full backup of everything** on the disk ASAP. You have been warned.

### Post by Quackers on 2013-09-12
The gave up waiting for root device can be indicative of a hard drive which is slow to spin up or slow to read (amongst other things).
I agree with TheFu in that the first thing you should do is to backup everything that you need off that drive. 
Once that's been done I would check the SMART status of the hard drive and if that's ok I would go to the HDD's manufacturer's website (not the laptop's manufacturer) and see if they have a disk checking utility you can run.
