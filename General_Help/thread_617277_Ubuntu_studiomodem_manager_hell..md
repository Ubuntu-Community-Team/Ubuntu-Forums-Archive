### Post by por100pre1 on 2007-11-19
Welcome to the forums! There are NOT stupid questions in these forums. :) Download an Ubuntu DVD ISO from your Windows partition and burn it. If using Ubuntu Studio Feisty 7.04 (old one) check [here]("http://mirror.mcs.anl.gov/pub/ubuntu-iso/DVDs/ubuntu/feisty/release/"). If using Ubuntu studio 7.10 Gutsy (current one) get [this]("http://mirror.mcs.anl.gov/pub/ubuntu-iso/DVDs/ubuntu/gutsy/release/"). Once done insert the new DVD in the Ubuntu system, it will likely ask you to use it as a repository. Accept and then install the packages, they should be there. 

BTW, check if the Ubuntu Studio DVD has already this packages before doing that.
