### Post by bored2k on 2005-02-28
aMule : *nix version of eMule [get the latest aMule 2 [here](http://download.berlios.de/amule/amule_2.0.0rc7-ubuntu1_i386.deb) (apt-get one is much older!).

Nicotine : This could be considered the Linux SoulSeek port .

Good ol' Mighty Bittornado-gui or Azureus ( java needed for this one, get Sun's 1.5 adding this repository deb [http://ubuntu.tower-net.de/ubuntu/](http://ubuntu.tower-net.de/ubuntu/) warty java )

Limewire Linux : ( ... ) . Best if using Pro version * . Limewire.org for Freebie one .


IMO, the best P2P software
