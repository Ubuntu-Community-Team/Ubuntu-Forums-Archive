### Post by mc4man on 2011-11-03
When the tracking bug shows "Fix Released" then the new packages will be available in an update, hasn't happened yet.

Atm they are still in oneiric-proposed (main)
So you can either wait till then & use 'unmount' instead or install the new packages from proposed.

If enabling the proposed repo one generally shouldn't just update all, better to only use to update specific packages & then disable the repo if not doing updates/upgrades on an individual basis

Packages go to proposed for testing and can't always be presumed to be 'good'

The current set in proposed that you'd want are these 3 - 
> linux-headers-3.0.0-13 (3.0.0-13.21)
linux-headers-3.0.0-13-generic (3.0.0-13.21)
linux-image-3.0.0-13-generic (3.0.0-13.21)

These  updates are also available but not necessary though are fine to install
> linux-tools-3.0.0-13 (3.0.0-13.21)
linux-tools-common (3.0.0-13.21)
linux-libc-dev ( 3.0.0-13.21)

If not clear on using proposed you should ask or as mentioned just wait, should show up normally in the next couple of weeks
