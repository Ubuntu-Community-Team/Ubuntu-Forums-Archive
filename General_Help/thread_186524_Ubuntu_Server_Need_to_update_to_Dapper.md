### Post by asimon on 2006-06-02
The [Breezy Badger Release Notes](https://wiki.ubuntu.com/BreezyReleaseNotes) stated that "Ubuntu provides security updates for 18 months after each release". Thus Breezy will still get security updates for some time, which is very important.

The new Dapper Drake will be supported for 5 years (3 years for desktops). Thus you don't have to update every 6 month if you're still happy with your version. But you should update when the times comes that your version will no longer get security updates. I suppose that there will be an announcement when Breezy (or other versions) reach their end of life.
