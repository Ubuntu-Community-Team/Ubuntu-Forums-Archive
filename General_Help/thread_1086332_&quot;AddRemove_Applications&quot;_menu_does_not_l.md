### Post by WaffleCode on 2009-03-03
Go to System -> Administration -> Synaptic Package Manager and uninstall them there. Not as good looking as Add & Remove, but oh well.

Or you can go to the Terminal (Applications -> Accessories -> Terminal) and type *sudo apt-get autoremove xyz* where xyz is the name of the application you wish to uninstall.
