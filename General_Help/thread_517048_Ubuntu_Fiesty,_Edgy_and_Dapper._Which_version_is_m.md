### Post by apetresc on 2007-08-04
Hello, tuxlinux, and welcome to Ubuntu!  :)

Names like "Feisty", "Edgy", and "Dapper" are simply codenames for the various versions of Ubuntu that were used during development.

Ubuntu 7.04 is the same as "Feisty"
Ubuntu 6.10 is the same as "Edgy"
Ubuntu 6.06 is the same as "Dapper"

Therefore, since you just installed Ubuntu 7.04, you want the Feisty version.

Hope this helps :)
