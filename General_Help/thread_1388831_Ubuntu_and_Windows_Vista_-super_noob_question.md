### Post by bpalone on 2010-01-23
Another method you might want to consider is getting VirtualBox and run Ubuntu in a virtual machine.  That way you don't mess with your Windows installation until you are sure you really want to use Ubuntu or any other distro.  The other advantage of using a virtual machine, is that you can create a copy of the virtual machines hard drive.  Then, if you hose the install, you can just copy the back up into place and be right back in business.

Just a thought, that might help ease your fears of dual booting.  I dual boot and use a virtual machine, the other way around.  Linux host with Windows as the guest.
