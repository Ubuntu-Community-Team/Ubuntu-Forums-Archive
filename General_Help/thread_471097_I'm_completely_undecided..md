Ok. Here comes my problem: I don't know whether to continue with Ubuntu, or Windows XP. Please don't immediately respond with "Ubuntu kickz windows' ***!". I've been using Ubuntu for a couple of months I think and am pretty comfortable with it. I love the way it looks (I have Beryl and all that neat stuff :D) and everything.

**But there are a few things that make me think about going back to Windows :(:**
1. I'm a web developer. Most of my site visitors will visit the sites from Windows, so I have to design them for that (so I must have IE 6 and 7, Opera for Win, etc). Also, I noticed there are a few (very small) differences between Firefox rendering on Windows and Ubuntu.
2. For web developing, I need graphics. I tried GIMP but to no avail, I just can't get around to using it. On Windows I have my trusty Paint.NET which has a super-simple UI, very easy to use IMO and you can create some amazing stuff with it.

5. Yahoo! Messenger. There is no fully-satisfactory port for Linux. And it doesn't work wined.
6. Last but not least, games. I do like to game a bit from time to time...

**Some things that keep me from going back:**

This thing is really great because I can create servers / programs that run in the background etc and I simply run them from a terminal.
2. The native ability to run Python (I'm just starting to learn it and it looks really great) especially the pygtk extension (it's really easy to create windows and stuff with it, I tried to learn C++ win32 API once and it was way too much to learn... I might try it again some time, though).
3. The lack of viruses and spyware. My Internet connection is going 4 times slower on Windows because of them. And I do have anti-virus (Nod32) and Adaware (I was shocked when I installed Ubuntu and I saw how fast my connection was actually going).
4. Looks, I just love how Ubuntu looks, with Beryl and everything.
5. Speed. Ubuntu is relatively faster than Windows (it's not awfully faster, but Windows ain't too slow either, when it's not full of viruses, but I can reinstall it and they usually go away with that).
6. The concept of open-source. It's a warm and fuzzy feeling when you know you're not controlled by some gates :p.


In conclusion... what should I do? :(

It might just do the trick for me... It sounds easy at the beginning of the tutorial but I did get some "huh??" reactions while reading it.... I have done this and successfully used it for a period of time.  A few problems though.  You can't really go back and forth from using windows through vmware then booting into windows because it will have you reauthenticate your windows each time because of the hardware changes.  (Vmware simulates a whole new set of hardware.)  You can get around this by backing up wpa.db and wpa.bak from your system32 folder after it has authenticated while booted in windows then after you authenticate while in vmware.  Then every time you switch from one to the other, you have to restore those files.  A big pain.

Also, you always always always have to make sure you do not accidentally boot back into ubuntu through vmware!

Finally, I stopped using vmware to do that because vmware eventually killed my windows installation where I had to completely reinstall.  Big pain.

I now do not recommend it.
