### Post by martijn_bakker on 2007-05-07
Your windows drives should be visible in the "Places" menu.

If you have given name for these drives, like "Windows" or "Boot", then they will show up with that name (one of my drives is called "New Volume" since I formatted it using Windows), If you did not give names then the drives will be called something like "sda4" (I don't know about the 4, that can be any number between 1 and 8 depending on the place your Windows partition takes in the partition table).
