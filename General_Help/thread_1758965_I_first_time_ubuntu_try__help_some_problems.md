### Post by sikander3786 on 2011-05-15
You want to delete the NTFS files. Ok. If there is no other data on your Hard Disk, at the Partitioning stage in Ubuntu installer, you can choose 'Use entire disk' and the installer would automatically delete all the previous files/partitions and create the necessary one's for Ubuntu. It is the easy way but CAUTION, it would delete all of your personal data also.

If you want to keep some data intact, you need to delete the NTFS partition and create a few new partitions in its place for Ubuntu. This page covers it more extensively.

[http://ubuntu4beginners.blogspot.com/2011/02/manual-advanced-partitioning-in-ubuntu.html](http://ubuntu4beginners.blogspot.com/2011/02/manual-advanced-partitioning-in-ubuntu.html)
