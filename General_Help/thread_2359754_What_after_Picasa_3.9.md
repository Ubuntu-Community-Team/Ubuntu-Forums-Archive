### Post by monkeybrain20122 on 2017-04-28
Try digikam. It is supposed to be good. Now it offers an appimage that doesn't require installation. [https://www.digikam.org/download.html](https://www.digikam.org/download.html) Go to official bundle provided by team, get the latest (5.5) for your architecture (either 64 or 32 bits) make the file executable (right click, properties > permission) That's it, just click to start and it will create a dashboard entry. Don't install from the repo since it is outdated and will pull in a ton of kde dependencies.

### Post by 2CV67 on 2017-04-28
I did try Digikam & Shotwell in the past, but stuck with Picasa while it worked ( & still works).

With Digikam, like all the others except Picasa if you make any edits like rotate, crop, lighten etc then you have to either:
1. Throw away your original.
or
2. Keep the original as well as the edited version, so doubling storage space needs.

With Picasa, I keep the original untouched, keep all the edits in tiny .ini files, and can undo/redo the edits any time I want.
That seems to me to be an overwhelming advantage.

Does no other app use that idea?
