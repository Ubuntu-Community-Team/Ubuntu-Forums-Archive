### Post by CatKiller on 2009-04-04
Yes and yes.

It will run slower from the live cd than it would if you installed it, since everything has to be loaded from the cd, and the cd is quite a lot slower than a hard drive. If you've got lots of RAM then things are only slow to load, but they run OK, since the information that's been read off the cd can be cached in the RAM and used while other stuff is being read from the cd. If you don't have much RAM, then whenever new data needs to be read from the cd, the old data gets flushed from the RAM, and then when you need the old data again, that needs to be read from the cd again too. This churn impacts performance severely.

That's why you've been asked to tell us the amount of RAM that you've got, so that people can give you advice on whether you'll have a good experience running from the cd.

If it runs OK from the live cd and you like it, the install option is on the Desktop. So you can still browse the Internet or play Solitaire or what-have-you while you're installing the software. Which is nice. It also means that if you have a question about the install process, you can research the answer during the install.

Your processor completely outclasses the amount of RAM that you've got in there. Put more in, and you'll have a **much** better experience, regardless of which Operating System you use.

With that said, Xubuntu should run OK. 192 MB should be enough for the live cd, but it won't be snappy. 384 MB is perfectly adequate. Technically, it's enough to run the Ubuntu live cd, but only barely. I certainly wouldn't bother.

DSL will run on pretty much anything. I wouldn't be surprised to see toasters and wristwatches that can run it. It is... really small. Not that welcoming an environment, though.

My mum runs Xubuntu on a machine with 160 MB of RAM, and it works OK.

It's probably worth starting a thread detailing exactly what your problems are when you try to run a live cd, now that you know it's possible. And put as much RAM in that machine as you can as soon as possible.
