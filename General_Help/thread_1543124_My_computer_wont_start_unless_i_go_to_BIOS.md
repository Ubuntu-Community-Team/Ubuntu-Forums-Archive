### Post by happyhamster on 2010-07-31
Perhaps just a BIOS quirk. Stuff like that is sometimes triggered by a particular hardware configuration. For example: if you have USB hardware attached to the laptop, try plugging them into other slots (or try booting without having them attached). Or: if you don't have a floppy drive, see if there's an option to disable floppy support in the BIOS.
Be careful when changing BIOS options though, because an uneasy boot is always better than no boot at all. Good luck.
