### Post by 3Miro on 2010-11-10
> **mandibabe83 said:**
> utorrent, skype yahoo messenger. How do you mark them executable?

You should not use utorrent under Ubuntu, use Transmission.

For Skype, go to their web-page and download the .deb file from Ubuntu (don't use the windows .exe).

For yahoo, you should check out Empathy or Pidgin (Empathy is already installed, you can get Pidgin from the Ubuntu Software Center).

Using .exe files under Ubuntu should be your last resort.

### Post by TBABill on 2010-11-10
If you become familiar with both the Ubuntu Software Center and Synaptic you will find many applications available directly from the repos to install. That is normally the preferred method to get apps, if available. Beyond that, .deb files are available for many popular programs and they click/install like a .exe in Windows. It is difficult to understand why you want to use .exe files within Ubuntu unless it is a matter of not quite understanding how Linux works and how it is different in operation and functionality (file systems, file types, structure, security, etc.) than Windows.
 
A great resource is [https://help.ubuntu.com](https://help.ubuntu.com) for just about any topic you can think of. Or search the forums directly or even Google search by entering your search terms as follows: ```
term1 term2 term3 site:ubuntuforums.org
```
 
Just use your search words in place of term1, term2, term3.
