If you load the usb with a app like unetbootin or others which is basically reading the ISO and has a casper-rw file for saving work these problems happen, basically the casper cannot be cleaned.

With 8 gigs I would do a full install, this will run like a regular HD and be cleanable.

The previous mentioned install type also is just for trying out ubuntu it has its limitations, such as kernel upgrades are not suggested is one, the OS is running off the no write ISO.

If you are doing full installs already it may be the error files are filling up as well.
