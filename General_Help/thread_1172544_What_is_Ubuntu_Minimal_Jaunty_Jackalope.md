### Post by Bender2k14 on 2009-05-28
The minimal images are contain the same operating system as the non-minimal images, it is just that (1) the disk image is smaller, so (2) the install should be faster, but (3) you will still install all of the regular program by downloading them after installation.  See this link: [https://help.ubuntu.com/community/Installation/MinimalCD]("https://help.ubuntu.com/community/Installation/MinimalCD")

After the upate, Ubuntu minimal and Ubuntu non-minimal will be the same, so you should use Xubuntu.
