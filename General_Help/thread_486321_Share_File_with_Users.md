### Post by jeffreyldavidson on 2007-06-27
I would like to share files on the same computer with a different user. If I log out how do I share the file in my home folder with a different user that logs on the same computer.

---

### Post by merlinus on 2007-06-27
You need to change the permissions.  Right-click on the folder and /or file, select Properties, go to the Permissions tab, and change Others to Read and Write.
