### Post by mike555 on 2012-06-10
I had that happen once , I found out you need "ubiquity" installed ...... You might be able to install it while running in "live" mode temporarily, so you can install the .iso

 If that works and you install the iso , be sure to install Ubiquity on the install before making another backup with remastersys.
