### Post by monkeybrain20122 on 2014-03-14
You can get it with a number of KDE applications like k3b or okular or god knows what. KDE's design is not modular and typically installing one kde application would pull in a lot of unrelated dependencies (one reason I don't like kde). Since 'print manager' is a kde app (else it would not be dependent on nepomuk) you should be able to remove it with no issue, as xfce must come with its own printing application.
