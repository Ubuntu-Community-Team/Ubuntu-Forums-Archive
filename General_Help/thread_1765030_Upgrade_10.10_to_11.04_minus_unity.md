### Post by Frogs Hair on 2011-05-22
When you login to your upgrade for the first time below the greeter box after the user name is entered select the Classic Gnome session from the bottom panel.

Go to login settings unlock and select Classic Gnome as default from the default session list.

It is possible to remove Unity from the Synaptic Package Manager , but don't how this may or may not affect the 11.04 installation . Perhaps someone has done this and could better advise you.
