### Post by howefield on 2016-06-17
Thread moved to the "*General Help*" forum.

Move your mouse over the top panel to reveal the application menus, so for the Files application you can select Edit > Preferences. The menus can be set to display locally in the window instead of the top panel if you prefer that to be the case.

### Post by howefield on 2016-06-17
System Settings > Appearance > Behaviour tab > Show the menus for a window and check the radial button for "In the window's title bar".

Set the "Menus visibility" as required.
