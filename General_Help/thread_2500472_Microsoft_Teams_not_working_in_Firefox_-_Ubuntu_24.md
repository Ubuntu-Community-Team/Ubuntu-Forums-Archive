Microsoft Teams stopped working in Firefox. I got the "Oops, failed to init! - Retry" message. 

I run Ubuntu 24.04.1 Gnome.

Simply disable Firefox Dark reader extension if you have one (`Ctrl-Alt-D` or in the extension itself).

I tested this several times an Dark Reader is the issue.
