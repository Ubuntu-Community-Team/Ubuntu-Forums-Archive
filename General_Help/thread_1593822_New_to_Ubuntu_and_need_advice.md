### Post by jungleexplorer on 2010-10-11
Although it has been some time since I have done this,it is fairly easy to do, or at least it was when I did it using Ubuntu Hardy Heron.  The easiest way to do this is to install it from within Windows.  There is a word for this type of installation but I can't remember what it is right now.  But basically, you can install Ubuntu from within Windows just like installing a Windows program.  It does all the work for you like partitioning and formatting the hard drive.  If I remember correctly it is set to automatically create a Linux partition of 30 GB.  I don't remember if you can modify this or not. Once it is done installing, just restart your system and you will see the option to boot to Ubuntu on your boot start up screen. Anyway, that is the simplest way to do it if you are just wanting try Ubuntu out.  Some say that that this type of installation is slower then others, but I have never been able to notice a difference and I have installed it all kinds of ways, by itself, dual boot (XP, Ubuntut), triple boot (XP, Vista, Ubuntu) and Quadruple boot (XP, Vista, Ubuntu, Kubuntu).
