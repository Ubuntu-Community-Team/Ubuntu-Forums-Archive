### Post by anotherdisciple on 2008-09-19
I'm not at a linux computer but hit ALT+F2 and type:

```
gconfig-editor
```

Then navigate to apps->nautilus->desktop.... there are likely to be options for it somewhere in there.

### Post by sayakb on 2008-09-19
Icons when aligned will stay aligned unless moved or are cleaned up by name. 
PS: If you are using "Clean up by name" as a replacement to "Refresh", you should rather use F5
