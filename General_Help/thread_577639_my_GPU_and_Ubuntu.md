### Post by qamelian on 2007-10-16
If you are using Ubuntu 7.04 or 7.10 (soon to be released), you only need to turn on the driver for your card in the Restricted Driver Manager located in System > Administration I believe. This will activate the latest Nvidia binary drivers available from the Ubuntu repositories, as well as the necessary kernel modules.
