### Post by Vermind on 2008-04-20
Hello,
I'm not very good in Italian,
but how about French or Spanish? I can repost this in French or Spanish if it helps.
what they suggest is:
1. download
[better wubi installer]("http://wubi-installer.org/devel/minefield/Wubi-8.04-beta-rev485sh.exe")
2. open start menu -> run - type in "cmd" (without the "")
3. drag- and drop the downloaded file to the cmd window
4. press <space> in the cmd window, then type --skipmemorycheck and finally press enter.

I hope this helps...

### Post by Vermind on 2008-04-20
Hello,
You only need to take the wubi file with the mouse, drag it to the cmd.
Then write the <space> --skip-memory-check and press enter.
You do not need to write the name of the wubi file, You do not need to say cd.
On the other hand,
if You want to cd there, then You can do so, but when You start typing wubi, press <tab> to allow the cmd to complete the name of the file for You. This way it is spelled correctly and hopefully found.
