### Post by LogicalDash on 2008-01-24
I'm using Kubuntu. When I changed to a different color scheme from the default, the buttons in OpenOffice no longer displayed their icons; they displayed text instead.

It turned out that the solution was to go into Tools -> Options -> View and uncheck "Use system font for user interface". It might be necessary to restart OpenOffice after that.

Why would a change in the system font, which I did not actually make, have any effect on the button icons? I don't know, I just found this fix by messing around with random options. :) I hope it's useful to someone else.
