### Post by treerink on 2011-06-08
Go to: System -> Synaptic -> Administration -> Package Manager ->
 search on lapack (and/or blas), and mark for installation:

libblas3gf
libblas-doc
libblas-dev

liblapack3gf
liblapack-doc
liblapack-dev

-> Apply
