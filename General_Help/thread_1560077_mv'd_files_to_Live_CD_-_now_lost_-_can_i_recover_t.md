### Post by pbrane on 2010-08-24
can you find them in the directory you moved them to? Moving file into the liveCD means you moved them to the (un)squashed filesystem. If you shut down they are gone. Since you got an error about disk space the move should not have completed on some of the files. Look around in the liveCD directory space and see if you can find the stuff and move it back.
