### Post by daaussie on 2009-12-06
We use Times or Arial, found that in windows when converting from .DOC (eg Primopdf) the resulting PDF was still very small.

However when we convert in Ubuntu, we find at least a 10X increase in memory size of the PDF.

Ubuntu Solution: You change the font to (for example LIBERATION SERIF) and you get a reasonable PDF size when converting from .DOC

It is therefore the fonts which cause this issue.
