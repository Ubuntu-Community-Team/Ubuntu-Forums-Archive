### Post by jpv on 2008-02-09
Both of these work for me on Gutsy.

From the command line (or create a menu entry):
[INDENT]vlc [http://provisioning.streamtheworld.com/pls/KYWAMDIALUP.pls](http://provisioning.streamtheworld.com/pls/KYWAMDIALUP.pls)[/INDENT]

That URL should also work as a "New Internet Radio Station" in Rhythmbox, but it may take quite a while to start streaming.

I originally found that via [http://www.google.com/search?q=stream+kyw1060]("http://www.google.com/search?q=stream+kyw1060")
