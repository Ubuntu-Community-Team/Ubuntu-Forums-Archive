### Post by fidelandche on 2012-05-26
Hi all,

Is there any way to set the document viewer to open PDF files automatically? At the moment when I download a PDF for work use I have to right click to get the option of opening with document viewer, I know it is not much of a hardship to do that, however it would be nice if I could just open it with document viewer without having to right click first to select the option of document viewer.

Cheers

---

### Post by hugh_bristol on 2012-05-26
Hi

Just go to any .pdf file in your file browser, right-click and select 'Properties'. In the 'Properties' dialog box, go to the 'Open With' tab (the 3rd one along) and you can choose Document Viewer as the default application for all files of this type. This will mean that double-clicking any .pdf in the future will automatically open it with Document Viewer. You can do the same with any type of file (e.g. do the same with a .mkv file, but for vlc).

Hope this answers your question!

Hugh
