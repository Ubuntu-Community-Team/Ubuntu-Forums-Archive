### Post by 1eth on 2011-05-24
Had same problem solution was this:-

In Firefox click on Tools. In drop down box click on Add-ons
When Add-on manager starts make sure get add-ons is highlighted and type flash aid in search box in top left corner of Add-on manager window. This should find Flash-aid 2.0.8. Click on install button and off you go!! 

The above worked for me last night :)
