### Post by tgrimley on 2008-10-23
Okay, almost solved.

I changed it under Appearance -> Customize -> Pointers and it works for most things, but when Fireefox is loading a page it still switches to the smaller pointing with loading instead of the larger one.  This is the only one I've noticed in the last 5 minutes or so.

I'm not sure when I'm supposed to stop posting to myself.. but I figured this might help someone out there.  Firefox remembers the pointer size properly after it is restarted.  

The only application I've encountered today which doesn't obey the pointer size is Matlab (2007a) which is obnoxious but not surprising.  I'm not sure if it extends to all Java6 apps or just Matlab.

I'll just mark this as solved soon if no one has any insight into the Java issue.
