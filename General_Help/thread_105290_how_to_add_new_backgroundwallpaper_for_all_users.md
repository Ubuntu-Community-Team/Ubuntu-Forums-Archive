### Post by JimS on 2005-12-18
...

Adding wallpaper/background procedure.
1. Find wallpaper you want.
2. Right click on the new wallpaper.
3. Select "Set as wallpaper".
4. Select " Set wallpaper".
Done.
:) 
Changing wallpaper/background procedure.
1. Right click on wallpaper.
2. Select "Change Desktop Background".
3. Select the image you want.
4. Click "Close".
Done.
