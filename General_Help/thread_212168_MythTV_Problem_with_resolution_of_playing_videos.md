### Post by jayemef on 2006-07-09
I just set up a fresh install of Ubuntu with MythTV. Everything went fairly smoothly, but I am having a problem with playing video. Be it a video file or DVD, the picture appears stretched vertically, with the bottom section missing. For example, an episode of anime might have all the subtitled part of the display cut off, with the characters unusually tall and thin. Same results in both MPlayer and xine, with and without full screen.

Other than this error, the Ubuntu desktop and mythfrontend appear perfectly normal, with the possible exception that there is a slight amount of black at the top and bottom of the screen (I can't remember if I had this before).

Some notes on my installation:
- This is running on TV-out
- video card is an ATI Radeon 9250


This installation had been running under Fedora, but due to personal annoyances with the distribution, I thought I'd give Dapper a try. Admittedly, I haven't looked into this too thoroughly, and am guessing that it's a simple xorg problem, but I want to get this solved today, so it doesn't hurt having others input as well.

Thanks in advance. Let me know if you need any more info.

However, now the entire display is, for a lack of better words, very choppy and unreadable. I'm going to fiddle with xorg.conf some more, but I'd gladly take suggestions.


Edit 2: Completely forgot about the fglrxconfig command, which is what I had run under Fedora. Ran that to set up xorg.conf. Worked like a charm. TV out is running great, as are my videos.
