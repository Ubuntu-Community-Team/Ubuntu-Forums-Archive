### Post by y-lee on 2008-04-22
Sadly I don't know of any linux software. There is windows software tho but i couldn't say anything about how safe it is or if it would work with wine. :confused:

But you can install the [It's All Text!]("https://addons.mozilla.org/en-US/firefox/addon/4125") Firefox add on and then be able to write long post in you favorite text editor. If that is gedit you could add [snippets]("http://live.gnome.org/Gedit/Plugins/Snippets") to add the code used by these forums. (if your fav editor is emacs or vim  you can probably handle it on your own :lolflag: )

by the way these forums use [bbcode]("http://en.wikipedia.org/wiki/BBCode"). I have actually thought of trying to write such a program as I know many people would find it useful but I honestly don't have time.
