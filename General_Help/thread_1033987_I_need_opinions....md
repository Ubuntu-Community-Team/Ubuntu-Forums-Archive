### Post by Tamlynmac on 2009-01-07
As lauren0718 states Ubuntu is one of if not the friendliest Linux distros and Firefox is the default internet browser. Evolution is the default installed e-mail package. A few additional apps can improve the experience for younger people. There is a number of apps in the package manager for education and pure enjoyment not to mention a number of games targeting younger users all of which are free. I can say without hesitation that the Ubuntu GUI (gnome) is extremely user friendly and much of which can be locked down to protect against - OPPs. Good Luck and should you need assistance in the GUI setup just ask.

### Post by avtolle on 2009-01-11
@OP: be aware that if your older computer has <384 mb RAM, the Live CD won't do well; in fact, it may not work on installation, as the minimum requirements to install from the Live CD is 384 mb RAM. If you have at least 256 mb RAM, Ubuntu will run; but you should install from the alternate iso burned to CD-R as an image at low speed (4x or less, IMO). Other things you need to check: check the md5sum on the iso before burning; and, before trying out the CD, check it for any errors. This will help save you some posting to the forum, if you have problems related to either of these things. Good luck and enjoy.
