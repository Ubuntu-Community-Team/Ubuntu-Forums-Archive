### Post by chopeen on 2005-06-11
I've just managed to install Dazuko (I want to configure ClamAV to perform on access scanning). To be honest, I'm not sure how I managed to do it. ;)

I can still give you some tips, though.

1. Install build-essential package, since you need to build dazuko module from sources.

2. Read this:
[http://www.dazuko.org/tgen.shtml](http://www.dazuko.org/tgen.shtml) (I followed this installation procedure)
[http://www.dazuko.org/faq.shtml](http://www.dazuko.org/faq.shtml) (I encountered a few of these errors)
[http://www.clamav.net/doc/latest/html/node18.html](http://www.clamav.net/doc/latest/html/node18.html) (and here I learned how to load a module)
