### Post by fritz_monroe on 2006-06-19
Yep sure does.  Some of them are **really** CPU intensive.  They are kind of like running DirectX screensavers under Windows.

Best advice is that if you need the CPU cycles to run something while you aren't using the system, just blank the screen, don't use a screensaver.
