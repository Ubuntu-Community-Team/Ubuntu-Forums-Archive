On my Ubuntu 10.04, if I hit Windows+N or Windows+M (aka Mod4+, Hyper+), my windows change from daytime colors to nighttime.  I was wondering where this is defined as I would like to redefine these keys.  I did not see them in "Keyboard Shortcuts"

### Post by fyo on 2010-08-11
Compiz handles that and since the shortcuts aren't shown in the Keyboard Shortcuts app, you need to go after Compiz.

Easiest is probably to install compizconfig-settings-manager.

Launch it (it's under System->Preferences) and select "Negative" under Accessibility. You can then change the shortcut to whatever you want.
