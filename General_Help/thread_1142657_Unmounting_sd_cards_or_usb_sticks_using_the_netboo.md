This seems to be a bug in the intrepid version of netbook-launcher.  In previous versions the icon on the desktop changed to a "mounted" version with an eject button.  This sometimes very occasionally works now, but is infrequent and random.

The best way seems to be to do it the way you say for the moment.  Hopefully this will be fixed in the future, as it is dangerous to unplug a mounted device if there is still data being written to it.
