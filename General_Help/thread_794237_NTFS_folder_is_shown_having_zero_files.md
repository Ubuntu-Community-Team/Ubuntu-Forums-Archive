### Post by dicepackage on 2008-05-15
I am leaving this here as my solution in case anyone searches upon this and needs a solution.  I had a hard time trying to get this working through ubuntu but I ended up installing Windows and using that to run scandisk.  Scandisk was able to detect errors on my drive and fix it so that it showed the files again.  I rebooted to ubuntu and all the files were there intact.  I am now going to have a seperate linux partition storing the music files since I think that this route would be more reliable then using the ntfs-3g driver.
