### Post by nogoodreason on 2009-05-04
I'm fed up of Windows and Macbooks are obscenely expensive, so I'm seriously considering buying a laptop with Ubuntu preinstalled.  There is only one thing preventing me from ordering one...

I work as a writer and editor, and as such the primary tasks I need my computer to perform are the ability to:
1) Open all document formats easily, including .docx (I don't want to waste time with the trouble of making converted copies or anything else that would be needlessly time-consuming)

2) Track changes.  i.e. if I am proofreading a Word document, set it so that my changes and comments are highlighted in red as I type them.

Microsoft Word 2007 handles both these jobs perfectly.  My question is: can anything else??  I'm sure OpenOffice and AbiWord are fine for the average user who simply uses them for standard writing, but how do such programs handle advanced use such as track changes?

This thread is really asking a double-barreled question: is there a Linux word processing program that can handle these advanced tasks?  And, if there isn't, how practical would it be to install MSWord under WINE?  As it would be a program I use constantly I wouldn't want any hassle, lag or other complications arising from running it within WINE, so would only consider this if I could still have it full screen and use it seamlessly.

> **nogoodreason said:**

MS-Office 2007 runs in CrossOverLinux, it gets a bronze award in the COL-database, but the development team is still working on improving it. See here : [http://www.codeweavers.com/compatibility/browse/group/?app_parent=1911;](http://www.codeweavers.com/compatibility/browse/group/?app_parent=1911;)
CrossOverLinux is based on Wine, and the company that makes CrossOverLinux is contributing back to the Wine project.

You can also run MS-Office 2007 in Wine (free of charge), there are howtos on the internet, but CrossOverLinux will offer you a much more seamless integration into the Linux desktop.

Here's some howtos for installing MS-Office 2007 in Wine :
[http://www.programmerfish.com/roffice-2007-in-linux/](http://www.programmerfish.com/roffice-2007-in-linux/)
[http://sudosys.be/?q=office2007_wine](http://sudosys.be/?q=office2007_wine)
