### Post by p3aul on 2011-06-27
If you are using the alsa driver you can type alsamixer and set up your microphone from there. Just hit the tab key to bring up the input mixer. You will have to use the keyboard to navigate. ie the arrow keys. after you have set what you need hit ESc to get back to the terminal. be sure to enter alsa ctl store to store your conf. unfortunately that's all I know as I am a newbie also.
