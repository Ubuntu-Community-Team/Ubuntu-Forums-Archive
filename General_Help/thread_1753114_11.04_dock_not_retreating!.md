### Post by grahammechanical on 2011-05-08
If you have the CompizConfig Settings Manager installed and the Ubuntu Unity Plugin ticked you can click the Unity plugin and you will see a heading called Hide Launcher. It has a drop down menu that lets you choose Never, Autohide, Dodge windows, Dodge Active Window. What setting to you have the launcher set to?

By the way, Unity uses Gnome. Who would you like to explain things for you? Would Mark Shuttleworth suffice? Check out his blog.

[http://www.markshuttleworth.com/](http://www.markshuttleworth.com/)
