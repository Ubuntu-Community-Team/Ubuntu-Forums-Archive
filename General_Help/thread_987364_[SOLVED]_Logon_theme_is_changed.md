### Post by shadow1983 on 2008-11-19
Hey, as it says on the website just extract .zip file, and drag n' drop desired themes to the login preferences window, it will appear on the list then just set it as the one you want to use, also make sure to unclick the default one either wise it will switch between the two.

For icons go to system--preferences--appearance, then on the theme that is highlighted click customise, you should then see a tab for icons, just drag the file to that window and it should install it for you

Hope that helps :)
