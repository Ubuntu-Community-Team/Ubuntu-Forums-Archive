### Post by az on 2006-01-01
[QUOTE=daller]How exactly will that work? (installation with a live-cd)

It's called ubuntu-express.  It is a package that is installed on a live cd which allows you to transplant the live installation onto a hard drive.  All you need to do is partition (now graphically!) and write the detected hardware configuration and you are done.  The live cd has always just been a base install with a modified init to allow running live. 

Whether you are running Ubuntu or Kubuntu, it still is just a matter of a few tweaks since Gnome or KDE have no idea you are running live.

The plan seems to be that Ubuntu and Kubuntu will ship together.  I think Edubuntu is kicking ass.  Will we see three cds?  Will Ubuntu press Ubuntu, Kubuntu and Edubuntu and distribute them seperately?  I dunno.

There have not been any flight (relatively stable snapsahots of Dapper) cds with ubuntu-express since they have yet to release a live cd.  I hope that enough testing will get done.
