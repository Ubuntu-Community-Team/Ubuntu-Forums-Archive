### Post by JC Cheloven on 2009-09-09
I had the same problem, but I guess that your "why?" is mostly a rhetorical question (something about the qt versions needed anyway). What you may want is how to fix it: 

DON'T install via sinaptic or apt-get. Simply download the version you like from the project page [http://kompozer.net/zip/](http://kompozer.net/zip/), uncompress in a handy place (even an external usb pendrive or hd), and run the shell script "kompozer" that you'll find inside. 

That's it, no installation, no crashes... I'm using kompozer 2008-12-18 this way without any issues, since intrepid.

Note.- where I found it:
[https://bugs.launchpad.net/ubuntu/+source/kompozer/+bug/264376](https://bugs.launchpad.net/ubuntu/+source/kompozer/+bug/264376)
