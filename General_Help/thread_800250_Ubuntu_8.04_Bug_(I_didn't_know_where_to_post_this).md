### Post by ubuntu27 on 2008-05-25
That must be a bug. A bug is an error or misconfiguration by an application.

Now that you have encountered a bug, you should fill a bug report at [LaunchPad]("https://launchpad.net/").
[Launchpad]("https://launchpad.net/") is where we fill bugs, make translations, write a blueprints (a plan for the future of an application), etc. In short it is an important site for the Ubuntu community.

So, go the the LanchPad site
Register
Search for the bug you encountered to see if someone already reported. If you find that it has already reported, write a comment that will help confirm the problem

If you did not find a relevant bug report. Then create a new bug report.



Reporting a bug is one of the important ways to contribute to an Free Software and/or Open Source software. You don't need to be a programmer to do this (if you are a programmer, then it is much helpful, you can write patch)

Here is the link of [Mozilla Firefox in launchpad]("https://launchpad.net/firefox/")
