### Post by GrouchoMarx on 2008-04-25
I used to use [Mouse Gestures]("https://addons.mozilla.org/en-US/firefox/addon/39"), until I upgraded to Hardy. Luckily after some searching I found [FireGestures]("https://addons.mozilla.org/en-US/firefox/addon/6366"), which is not only functionally similar to Mouse Gestures, but compatible with Hardy!

Here's a Bonus tip! To resolve the Firefox 3 font-size anomolies: under about:config set the variable "layout.css.dpi" to 96 (the system default), or whatever your preference might be.
