### Post by jereece on 2010-06-03
I recently installed the VLC media player.  Now how do I make this my default media player?  When I click on a video, Totem Movie Player still launches.

---

### Post by Vaphell on 2010-06-03
right click on a file you want to assign to VLC -> properties -> tab with apps able to open said filetype -> select VLC - done.
you need to do that for all types - avi, mov, mpg, mpeg...
