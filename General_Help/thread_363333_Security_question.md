### Post by meng on 2007-02-16
No, you don't need to (you can if you're extra worried about security I suppose). That first user has admin privileges, but they can't be used except with another password authorization.

### Post by meng on 2007-02-16
And by "another password authorization" I don't mean you will be asked for a different password, it's the same as the login password, it's just that in order to perform an administrative task, you'll have to authorize that by providing the password again.
