### Post by benny bronx on 2008-11-12
I use the fanboy's adblock list for Opera, found at:

[http://www.fanboy.co.nz/adblock/opera/]("http://www.fanboy.co.nz/adblock/opera/")

I use the list with stat/tracking sites.  It seems to work well and does not slow down browsing for me.  Download the file and drop it in the Opera folder in your home directory.  It will show up in the blocked content box (tools-preferences-advanced-content)
