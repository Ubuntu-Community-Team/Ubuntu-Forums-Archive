### Post by DapperMe17 on 2011-12-06
Viruses are platform specific code, and there are currently no viruses "in the wild" for Linux. 

Most viruses in the wild are comprised of code that can execute in a Windows environment. 

Yes, an previously infected file can easily be 'transported' through Linux, and still infect a Windows partition. 

That would be the same as an e'mail attachment virus...which wouldn't affect a Linux PC, but could be passed on to a 3rd party through a "forwarded" e'mail.

Yeah, it's best to check first! Enjoy Ubuntu, and don't be "too" paranoid.
