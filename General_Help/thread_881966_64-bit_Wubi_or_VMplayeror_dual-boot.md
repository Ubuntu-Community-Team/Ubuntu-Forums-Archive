With a wubi install you must reboot. The wubi install allows you to dual boot between Windows and Ubuntu. VMplayer runs a virtual machine inside your host OS, therefore you do not have to reboot to change OS. 

I personally think a wubi install and dual boot is better, but it's your choice.
