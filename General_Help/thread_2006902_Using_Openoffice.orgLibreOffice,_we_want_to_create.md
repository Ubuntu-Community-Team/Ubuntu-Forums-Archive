### Post by HiImTye on 2012-06-19
the default layout of spreadsheets look rather robotic but you can merge cells and edit the format to make it look like any document format. if you need to protect parts of a document and use things such as check boxes, etc, the easiest route is to use a spreadsheet. you edit it just as you would any spredsheet in Excel, except that you use 'LibreOffice Calc'/'OpenOffice.org Calc' instead
[http://help.libreoffice.org/Calc/Protecting_Cells_from_Changes](http://help.libreoffice.org/Calc/Protecting_Cells_from_Changes)
