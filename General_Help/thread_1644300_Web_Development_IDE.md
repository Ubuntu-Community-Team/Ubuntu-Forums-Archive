### Post by lykeion on 2010-12-13
If you don't need any advanced IDE features I think Eclipse/NetBeans might be a little bit overkill. These IDEs are more targeted for Java development than web development. 

So why not try [Bluefish]("http://bluefish.openoffice.nl/"). It's not WYSIWYG, but hey which experienced web designer need that anyway ;) - if you really *do* need it try [KompoZer]("http://kompozer.net/").
You can install both these directly in Ubuntu with Synaptic/Ubuntu Software Center.
