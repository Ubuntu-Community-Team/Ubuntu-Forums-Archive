### Post by jonlemur on 2008-07-13
I have MATLAB 2007a installed both on my Windows and my Linux partitions, and it works very in both.  In fact, I haven't found any differences in using the two.  Mathworks was very helpful in helping me install in Linux after I had some difficulties due to a previous installation.

That being said, Octave is a very good open source clone of MATLAB, and I highly recommend investigating it.  It has pretty much the same syntax, and I've written m files in MATLAB and run them very well in Octave.
