### Post by musicinmybrain on 2007-05-01
System --> Administration --> Language Support

Install anything it asks you to.

Click the Support checkbox beside English. Click Apply.

Change the default language to your favo(u)rite English flavo(u)r if you wish.

Log off. From the Options menu at the lower right, select System Default if you changed the default language; otherwise, select the appropriate variant of English. Log in.

That should do it!
