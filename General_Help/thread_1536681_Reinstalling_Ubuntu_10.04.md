### Post by pricetech on 2010-07-22
Question one:
Short answer is "same way you did it the first time".  Nothing special about a reinstall.  Probably the only thing you'll see different is that you'll be using the partition that's already there as opposed to let the install create one.
But:
You should ALWAYS back up your data before doing anything to your hard drive, so make sure you have a good backup of you files under windows before you reinstall.

Question two:
Leads me to ask you a question;  If you think you'll be doing away with windows entirely soon, what are you using it for now ??  It may be that using VirtualBox to install windows in a virtual machine would suffice for your needs, then you can put Ubuntu on your entire drive now.
