### Post by cascade9 on 2011-02-06
Actually, the recommended minimum requirements for ubuntu is 1GB-

[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

Ubuntu should still run on 750MB. Though you probably have 768MB, unless you are using onboard video. Onboard intel 845/865 video (used by most P4s) can be a bit dodgy with ubuntu, but it should work in most cases.

If you dont mind doing a mimimal install then adding a desktop enviroment, 750MB should fly. 

If that sounds to difficult, xubuntu or lubuntu should work better than standard ubuntu.
