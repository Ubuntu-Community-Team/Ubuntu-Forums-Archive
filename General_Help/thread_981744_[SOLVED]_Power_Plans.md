### Post by fab.head on 2008-11-14
Hi Yezinki,

It happened to me as well after one of the latest updates (I can't remember what one though).

Solution/workaround: 

1) export your bookmarks to an html file (so you will be able to import them again later on)

2) close firefox

3) open your /home/{YOUR USER}/ directory and delete (or just rename) the hidden folder .mozilla

4) restart firefox

5) import the bookmarks.html file which you saved at point 1


This worked for me
