### Post by 3rdalbum on 2010-04-16
It seems that something has gone wrong with the installation. This is more likely to happen when you use Wubi to install Ubuntu within a Windows partition.

If you can repartition the drive and install Ubuntu onto a seperate partition in a "real dual-boot" scenario (i.e. booting up the computer from the Ubuntu CD) you will probably find that everything will work.
