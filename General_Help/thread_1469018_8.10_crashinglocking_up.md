### Post by kmsalex on 2010-05-01
Can you still get to your files though a live cd? or has that failed already? if so then your hdd might have corrupt sectors.
  Are you using a nvidia card with compiz enabled? If so try recovery mode from the grub menu? Then type 
```
Try to fix X server
```when its finished type
```
Resume boot
```It should boot to the login screen in a lower resolution mode, from there disable compiz and restart and boot as normal.
