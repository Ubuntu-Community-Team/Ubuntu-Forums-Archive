### Post by beloved88 on 2009-12-06
I have a mini HP 1101 which has just about the exact same chip-set and hardware as the Dell Mini 10v.  I went to [http://www.broadcom.com/support/802.11/linux_sta.php](http://www.broadcom.com/support/802.11/linux_sta.php) and downloaded the driver and followed the instructions very carefully in the README file.  everything is performed as root, so ```
sudo -s
``` is very useful. post if you have any problems or error messages and I or someone else will tell you what to do to fix it
