### Post by leunam12 on 2017-12-08
I have installed and currently ran successfully Dragon Naturally Speaking 12 using WINE in the past, it worked well (it still works, I just don't use it anymore). It took a little bit of research and patience to install the pre-requisites: .NET.4 Framework and other Windows components. It was also hard to "train" the program to recognize the way I speak. It depends a lot on your accent, of course, I have a heavy Spanish accent and that made things more difficult for me. 
Recently I upgraded to Dragon Naturally Speaking 13 Home version and I bought a good headset for it. I have only tested it on Windows and it does an amazing work out of the box, I very rarely have to edit anything. I haven't tried to install it on Linux yet.

### Post by leunam12 on 2017-12-11
You are welcome. I think you will be better off with Dragon Naturally Speaking on Windows, it does an excellent job and you can use it with your whole computer. The way I have it running on WINE only works with MS Word and a few other WINE programs but it doesn't work on native Linux applications. So, if you want to use it with WINE you are also going to need Microsoft Word.
