### Post by flemur13013 on 2011-09-10
That can happen just by resizing the FF window to be larger than your desktop, something a noxious website might be capable of doing. 

Try [Nytram]("http://ubuntuforums.org/member.php?u=148512")'s F11 suggestion *twice* and it should shrink up, or, if you can get to it, click the full-screen icon on the titlebar - fullscreen should show the menubar, etc., then resize it from there. 

Disclaimer: I'm using fluxbox window manager and use the middle mouse button to resize windows; I don't have to actually click on borders or corners because it picks the closest one, which is great when the window is too big or off-screen.
