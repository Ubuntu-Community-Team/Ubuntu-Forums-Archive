### Post by Rasa1111 on 2010-09-11
Hi Ken,

 You'll need to install WINE to use it.
 I prefer GIMP anyhow though, lol
but I have used PS7 under WINE in Ubuntu in the past, just to test it out, and it worked fine.
Apparently PS elements will also work.

 See here~
[http://appdb.winehq.org/appview.php?appId=1794](http://appdb.winehq.org/appview.php?appId=1794)

 Youll need to install WINE through the software center first. 
It allows you to use windows programs in Linux/Ubuntu. 

 Then you can install PS Elements just as you would on a windows machine. 


other links~ [http://forums.overclockers.com.au/showthread.php?t=743816](http://forums.overclockers.com.au/showthread.php?t=743816)

 Good luck.
