### Post by Keith_Beef on 2016-05-21
I was updating my library catalogue in Tellico a couple of weeks ago and found that Amazon lookups were no longer working.

I had to leave it for a while because I was too busy with other stuff, but found some time this afternoon to look at it again.

It's been such a long time since I set up my  account in Amazon Web Services (AWS) that I didn't recognise anything… I had to log in to my AWS account, go to Security Credentials, create a new access key, then copy the access key and secret key into the corresponding fields in the Data Source Properties window of Tellilco.

Now Amazon lookups work again. It's so convenient, when adding new books to my library, to do a multiple ISBN lookup using a barcode scanner. I can add twenty books that way to my catalogue in the time it would take me to add just a single book if I was filling in all the fields by hand.
