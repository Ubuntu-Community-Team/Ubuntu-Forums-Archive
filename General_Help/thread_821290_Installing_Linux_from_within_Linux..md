### Post by nick_h on 2008-06-07
You can partition your hard drive as part of the Ubuntu installation.  See the links that forestpixie gave you for that option.

You could also partition the disk before the installation by using GParted from a Live CD.

There is also a [GParted Live CD](http://gparted.sourceforge.net/livecd.php) available.
