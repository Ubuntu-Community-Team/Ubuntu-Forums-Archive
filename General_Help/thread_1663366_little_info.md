### Post by wog on 2011-01-09
Firefox add-ons that help with popups include:

Ad-Block Plus: blocks Flash and Java adverts.

Ghostery: allows you to block (and identify) advert companies that collect your movement information and input data online.

No Script: blocks auto-scripting from happening on sites.

All three generally need to be set when you go to a new site, especially ones involving media.

Find all of them through the menu.
Tools --> Add-ons
