### Post by benerivo on 2008-07-13
I think you might need to have the emerald program startup when you login. If you're using Ubuntu (gnome), then you can go to System>Prefs>Sessions, and add an entry for emerald (just call the command emerald). The log out and back in again. This should make compiz use emerald as the window themer.
Or do you mean you can get emerald to work, but just not this particular theme?
