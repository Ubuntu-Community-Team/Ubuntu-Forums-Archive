Last February, we discovered an issue with the AWS SSM Agent randomly consuming all CPU resources. After months, AWS seems to have fixed the issue ([https://github.com/aws/amazon-ssm-agent/issues/353#issuecomment-926862388](https://github.com/aws/amazon-ssm-agent/issues/353#issuecomment-926862388)) . At our company, we have massive scale ups happening that already require pre-planning as apparently AWS can't handle such rapid scaling and this is causing additional hurdles and concern among teams.

I was wondering when the new SSM agent version will be upgraded on the [http://releases.ubuntu.com/bionic/](http://releases.ubuntu.com/bionic/) image.
