### Post by matthew on 2005-06-07
I have decided to make the switch from WinXP to Ubuntu final and I want to move all my data from Outlook 2003 to Evolution. I am running in a dual boot right now so I need to figure out a way to save/export my data from Outlook and then reboot into Ubuntu and import into Evolution. I've tried the following:

1) Export everything (folders like inbox, etc., and contacts) from Outlook. I've tried comma separated values, pst files, and every option given me in Outlook.

2) Import the files into Evolution.

Here's my problem: Evolution can't read any of the formats that Outlook uses in its export function. So...I need to find a way to convert into something that Evolution can read.

I was able to bring my contacts over by syncing my Palm m130 in Windows and then after a short struggle I got it working with Ubuntu and synced to Evolution. Got my notes the same way.

I just want my archived email as well as about 2 months of inbox stuff so I don't have to boot into Windows every day.

If it's possible, though, I would like to just avoid booting into windows altogether except for one or two programs that I haven't got working with Wine yet and that I only need about once a week or so.

Okay, done rambling...anyone have any ideas?

### Post by BAshworth on 2005-06-08
It's a bit of a work around, but I came across this link looking for a solution.

[http://mail.gnome.org/archives/gnome-list/2002-April/msg00043.html](http://mail.gnome.org/archives/gnome-list/2002-April/msg00043.html)

To sum up, it appears that you can install Mozilla Mail (not Thunderbird) in Windows, and then import the pst file into that, then export it all out again as an mbox file.  Evolution can import that directly.
