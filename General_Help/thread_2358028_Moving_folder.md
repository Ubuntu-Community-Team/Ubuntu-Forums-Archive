### Post by mooreted on 2017-04-08
I usually open two file manager windows side-by-side. One window is open to the location on my Windows drive and the other open to the location I want to copy the files to on the Linux drive then I just drag and drop stuff over. I don't like cut and paste since you can lose data if something goes wrong. I copy it over, verify I got everything then delete what I don't need from the other drive.
