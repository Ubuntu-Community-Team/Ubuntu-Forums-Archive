### Post by dannyboy79 on 2007-12-27
you could always try a little older nvidia driver. I think that would be nvidia-glx-new (which is 9755)

WHat does  this command return?

lspci -v | grep nVidia

Do you have DMA enabled for your hard drive? what does this command return?

sudo hdparm -I /dev/sdb

(NOTE: use /dev/hdX or /dev/sdX for whatever hard drive you're watching the movie from.)

You should see something like this on one of the lines: 
DMA: mdma0 mdma1 mdma2 udma0 udma1 udma2 udma3 udma4 udma5 *udma6
This means that the hard drive is set to DMA5.


If it's a dvd-drive, then I am not sure what to tell you. It must be related to the X server then.
