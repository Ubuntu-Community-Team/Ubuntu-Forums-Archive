### Post by PointSource on 2007-08-25
I don't know how you could get the nice picture and progress bar back but if you would like some indication of progress edit /boot/grub/menu.lst using this:
```
gksu gedit /boot/grub/menu.lst
```

and scroll to the bottom to edit a block of text that looks like this:
```
title		Ubuntu, kernel 2.6.20-15-generic
root		(hd0,0)
kernel		/boot/vmlinuz-2.6.20-15-generic root=/dev/hda1 ro quiet splash
initrd		/boot/initrd.img-2.6.20-15-generic
quiet
savedefault
```

to this

```
title		Ubuntu, kernel 2.6.20-15-generic
root		(hd0,0)
kernel		/boot/vmlinuz-2.6.20-15-generic root=/dev/hda1 ro
initrd		/boot/initrd.img-2.6.20-15-generic
quiet
savedefault
```

Now you'll get a whole lot of scrolling text instead of a picture (I actually prefer it that way)
