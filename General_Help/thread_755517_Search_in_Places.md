### Post by munkyeetr on 2008-04-15
Yes, it does.

1) Make sure that under the **Look in folder:** field you make it point to where you want to search (or above that point in the file tree).
2) Also, click on **Select more options** and make sure the **Show hidden and backup files** is enabled.
