### Post by Zemblan on 2010-05-26
How are you determining which drives are mounted? If Ubuntu is not automatically mounting a particular drive you can do it manually. 
If Ubuntu is installed on your "c" drive, then it most certainly is mounted if you are booted into Ubuntu, it is mounted under '/' in the file system. 
Perhaps a read of [http://geniushackers.com/blog/2009/06/28/linux-file-system-explained-for-beginers/](http://geniushackers.com/blog/2009/06/28/linux-file-system-explained-for-beginers/) would help.
