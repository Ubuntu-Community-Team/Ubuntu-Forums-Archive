### Post by Edho on 2007-03-03
1,- Install Wine.
2,- Configure Wine...In a terminal give the command      winecfg.
3,- Install a program... In a terminal give the command    wine programtoinstall.exe
4,- Run a program... In a terminal give the command    wine program.exe


On your desktop or bar you can make a launcher with that command.(4)
