### Post by cronos on 2009-05-07
Ok, the Crash Report incident is solved. :)

When I saw the Crash Report icon flash on top of my screen whenever Ubuntu booted, I thought an something was crashing/failing to start. But, it seems nothing was crashing during startup.

I eventually discovered that I had a crash report sitting in the /var directory.  The report was created by Simple Backup a few days ago, which I knew about.  After I deleted the report file, I no longer see the Crash Report icon.

I guess I am still learning how Ubuntu works. :)
