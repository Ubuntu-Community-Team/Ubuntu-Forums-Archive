### Post by mcduck on 2014-12-19
I believe MuPDF should be able to open OpenXPS files. Take a look if it's available in Ubuntu's repositories, and if not, there's probably a PPA repsoitory for it...

Alternatively, if you have access to a Windows mchine, Microsoft provides tools for converting OpenXPS to XPS (as OpenXPS isn't supported on Win7 or older), and after that even the default document viewer (Evince) should be able to handle the files.
