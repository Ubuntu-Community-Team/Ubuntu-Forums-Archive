### Post by ankspo71 on 2010-02-02
Hi,
Are you using Ubuntu 9.10 (aka Karmic)? If so try searching in Synaptic Package Manager for assogiate. According to [http://packages.ubuntu.com/karmic/assogiate](http://packages.ubuntu.com/karmic/assogiate) it should be listed in there. If you would rather use the command line to install it, you can open the terminal and type: ```
sudo apt-get install assogiate
```Hope this helps.

Hi again,
Ubuntu uses nautilus as the default file manager, but Xubuntu uses thunar and Kubuntu uses dolphin (I think). Try typing nautilus in the terminal. Does it tell you that it is not installed, or does it open your home folder?

This link explains how to install packages manually better than I can explain it.
[http://amitech.50webs.com/installing/index.php.html](http://amitech.50webs.com/installing/index.php.html) . Scroll down to "Source Package (.tar, .tar.gz, .tgz, .tar.bz, ...)". Most of the time there are instructions included in the package you downloaded, sometimes they are found in a 'readme' file or a 'install' file, or on the page you downloaded them from. Sometimes they don't give instructions though.
