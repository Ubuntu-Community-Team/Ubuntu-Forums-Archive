### Post by rugglesworth on 2007-03-19
For your needs, I think Ubuntu Edgy would be the best choice.

You do not want to use Feisty because it is still in development and will probably be buggy until a few weeks after its release.  I think that Edgy is a better choice than Dapper because it comes pre-installed with newer versions of OpenOffice and Firefox (although you will need to install Thunderbird yourself).

Edgy (Ubuntu 6.10) should work with almost any hardware and you should not find it difficult to install and configure.

Rugglesworth
