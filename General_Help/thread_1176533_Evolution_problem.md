### Post by uidb4056 on 2009-06-02
> **drdob said:**
> I have a problem with evolution mail when I down load email if there is a picture attached all I get is the thumbnail in the mail. I have looked on the forum, and could not see anything like it on there, so anybody, any ideas. Thank you Drdob

You have to open the mail message and select from the window menu View-Load Images, alternatively there is a shortcut that do the same ctrl-I.

Also you can go to Evolution menu Edit-Preferences and select Mail Preferences, in the tab HTML messages you can select the default behavior for loading images.
