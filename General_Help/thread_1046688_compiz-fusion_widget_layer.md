### Post by Shazaam on 2009-01-21
I used to use it when I ran cairo-dock to show a hidden panel. Very handy as cairo-dock doesn't really have a good Main Menu app like a regular gnome-panel.
You add the command you need under "Behavior>Widget Windows"

### Post by fabounet on 2009-01-22
you can set the desklets of Cairo-Dock on the Widget Layer too.
very convenient to clean your desktop/reduce the number of applets in the dock, and still easy to access.
(and by the way, Cairo-Dock has the same Main Menu app as the gnome-panel)
