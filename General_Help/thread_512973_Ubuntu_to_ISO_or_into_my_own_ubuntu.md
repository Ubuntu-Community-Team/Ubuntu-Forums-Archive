### Post by ddrichardson on 2007-07-30
There are two options:

Create a custom LiveCD - there is a HowTo [here]("https://help.ubuntu.com/community/LiveCDCustomization").

Or set up everything how you want it and use a tool like [System Recovery CD]("http://www.sysresccd.org/Download") to create a disk image that can be installed on each new system.
