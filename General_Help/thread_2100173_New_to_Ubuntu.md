### Post by kansasnoob on 2013-01-01
The best Ubuntu guide I've found is this:

[http://ubuntu-manual.org/downloads](http://ubuntu-manual.org/downloads)

You'll see there are different versions available. Since you're new to this I'd recommend version 12.04.1 because it has a much longer support cycle.

But if you really can't use or don't like the new Unity interface you may be more comfortable with [Lubuntu]("http://lubuntu.net/") or [Xubuntu]("http://xubuntu.org/").

Edit: I just thought to add one. HowtoForge has a very great series that can easily be googled with "perfect desktop distroname+version", eg; googling "perfect desktop ubuntu 12.04" renders this result:

[http://www.howtoforge.com/the-perfect-desktop-ubuntu-12.04-lts-precise-pangolin](http://www.howtoforge.com/the-perfect-desktop-ubuntu-12.04-lts-precise-pangolin)
