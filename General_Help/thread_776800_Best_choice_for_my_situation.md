### Post by tamoneya on 2008-04-30
if you are more familiar with GUI i would install the desktop version.  You should definately take some time to learn CLI but you dont want to be forced to.  If you pick the desktop version you will have to do some more configuring of LAMP since it is not installed by default.  The other difference is that since you are running a GUI it has a bit larger CPU and RAM utilization but since this seems to be a school project not a production server or something you should be fine in this case.  I am at university as well and I have my desktop running 8.04 64bit desktop and I use it as a LAMP server for my own small projects and what not.

Edit: missed the part about ubuntu vs. kubuntu.  The GUIs are very similar.  KDE is a bit prettier but they are both very complete GUIs and unless you have a preference for gnome or KDE (it seems like you dont) it shouldnt matter,  Install Ubuntu 8.04 desktop 32 bit
