### Post by geirha on 2009-02-15
Go into the bios and look for options regarding usb keyboards. The problem is most likely that the keyboard will not start working until the operating system has booted. There's usually an option in the BIOS to have it enable the use of USB-keyboard before the operating system steps in and takes it.

If you happen to have a ps/2 keyboard you can use, that should work without a fuzz.
