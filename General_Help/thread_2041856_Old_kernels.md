### Post by ajgreeny on 2012-08-13
I am one of the old-fashioned guys who can't manage without synaptic-package-manager, which I install even in 12.04.

Using that I search (name only) for, eg 3.2.0-27, which will find the kernel image and any headers of that version which I can remove completely.  I don't use ubuntu-tweak, but prefer to use the tried and tested package-manager.

You may also be able to do the removal job with ubuntu-software-center, but I have never used it so I don't even know you can search for kernel versions with that.
