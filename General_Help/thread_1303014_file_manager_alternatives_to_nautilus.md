### Post by VCoolio on 2009-10-27
I use thunar because it's light, but it doesn't have network browsing capabilities (I use nautilus for that). Also it has very easy customizable actions support. It fits in gnome very well.

Emelfm2 is also nice: it has two panes that can interact and also a terminal-like feature that is automatically located at your current position in the filemanager. It fits ok in gnome. Gnome-commander also has two panes, but is uglier and doesn't handle dark themes well in my experience.

And I use commandline for a lot of common stuff, especially for the system section where root is required.

Edit: found a [plugin]("http://goodies.xfce.org/projects/thunar-plugins/thunar-shares-plugin") for thunar that does network sharing! Yay!
