### Post by Dennis N on 2014-08-30
> I've got my printer and scanner working in ALL distros, but; I can't for the life of me see how you send anything TO the printer in Xubuntu. There's no option in the File Manager; you can't do it from the default image viewer, Ristretto.

Anybody got any suggestions?

Ristretto does not have a print option, so nothing is wrong. Xubuntu often stays with more basic choices like Abiword and Ristretto instead of Libre Office and Gimp. To print images, you need a more powerful graphics tool, like gimp or gthumb, which both offer printing. Both are installable in Xubuntu from the Software Center. Thunar don't offer printing as far as I can tell - how would that work, in a right-click menu? Anyway, it doesn't. To print a file, you need to open it first with the right application and then print it.
