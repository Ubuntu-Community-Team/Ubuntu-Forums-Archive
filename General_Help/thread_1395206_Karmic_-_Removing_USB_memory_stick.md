### Post by lithops on 2010-01-31
I back up my data with a USB memory stick. When I have finished I right click on the desktop icon and there are three choices: "Unmount"; "Eject" and "Safely Remove Device".

What is the difference between these three options? I have been using "Safely Remove Device".

---

### Post by Leppie on 2010-01-31
if there's only one partition on the usb stick, unmount and safely remove device are practically the same. safely remove device will try to unmount all the partitions on the device whereas unmount unmounts only 1 partition at a time. eject is really only for cd/dvd or something like  a zip/jazz drive.
