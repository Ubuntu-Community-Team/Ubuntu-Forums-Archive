Hi everybody.

I am trying to configure the "right-click" and "open with" behaviour so that I can open a certain file type with "nano" in a terminal.

I searched the web but I could not find my answer. I tried using "gnome-terminal" but it does not help.

Anybody has an answer? Thanks.

---

### Post by sisco311 on 2009-12-16
Use a custom command: 
```
gnome-terminal -x nano
```
