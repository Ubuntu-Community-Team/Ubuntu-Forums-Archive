I don't use closed source driver for graphic card and everything works automatically but probably i did something and this happened to me twice already and i would like to know how to fix it.

I don't know exactly why but after i boot computer resolution is set correctly in unit greater but only in one account i use after log in it is set too low! If i change it to native resolution it works fine but after next (re)boot it is set to low resolution again!

### Post by EgoGratis on 2012-01-17
OK here is what i did.

In display settings i changed resolution few times between different available options and apply it few times and then in the end selected native resolution and rebooted and all was fine! So the "key" was to change and apply different resolutions more then one time before selecting desired one in the end and apply it. Problem solved!
