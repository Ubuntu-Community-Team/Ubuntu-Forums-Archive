Here are the reasons I have determined that most people use for virtually every single Ubuntu upgrade since I started using 4.10:

"It's new and shiny - I want it I want it I want it gimmie gimmie gimmie....."

Anyone with any sense at all will wait at least 6-8 weeks to allow all the inevitable problems to be ironed out before considering install it on a system they actually use.

Anyone with a lot of sense will only use Ubuntu LTS releases and leave the 6-monthly releases to the "fanbois" who just **must** have the latest of everything.

Those who have enough time will install a Virtualization platform and run the newer release in a VM to try it out before committing their "real" system to use it.
