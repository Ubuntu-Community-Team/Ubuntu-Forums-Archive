### Post by kiemc-b on 2013-08-19
You can do three things:

1. Wait until the package is updated in the official repository.
2. Try to find a ready deb package somewhere else on the internet. For example, on pkgs.org. This site seems to be containing links to official deb package mirrors (you can check the exact source of the file before downloading), thus safe.
3. Compiling from source. I wouldn't write it down now, only if the previous points are not suitable for you. (I have checked, there is pcmanfm 1.1.2 on pkgs.org.)
