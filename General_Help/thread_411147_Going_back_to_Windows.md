### Post by jamespetts on 2007-04-16
This is not an entire solution or a full answer, and you may know it already, but there are equivalent applications to some of those that you listed that work in Linux that are, at least, very nearly as powerful as their Windows counterparts.

For image editing, there is the GIMP, which is also available for Windows. It is a highly sophisticated package, used by many professionals, and is entirely free. It has a very high proportion of the features of Photoshop.

For HTML editing, there is NVu. It is a good WYSIWYG editor, although it lacks support for frameset documents and server-side code. 

There are also CD/DVD authoring packages available in Linux, such as K3b, although I do not have any experience with them. 

I am not sure about interfacing with an XBox 360, but, the latter being a Microsoft product, it would doubtless have been specifically designed to work only with Windows computers with the express aim of discouraging people from doing exactly what you want to do: using Linux.

### Post by jimbo99 on 2007-04-16
As many have said, there are linux equivalents to many of the programs you spoke of.  But it is very clear that you are still thinking like a Windows user.  Once you get the hang of Ubuntu/Linux, you will be happier.  It takes throwing off many of the rather dull concepts of Windows in order to see and unlock the potential of Linux and Ubuntu.  Two weeks isn't that long a time.  You should not expect 100% translation of applications and features without having a product developed for both platforms.

Even tho many have stated alternatives to Windows apps some are doing you a disservice.  Although GIMP is a good product for the most part it in no way competes with the feature set of Photo shop.  On the other hand the K3B Version 1.0 is an excellent alternative to nero linux.  I have purchased the platinum package of Nero for  Windows and use it regularly but I use the linux K3b program just as often.  I use it when I don't feel like switching back to windows, which is most of the time.  I have other computers that have Nero on them and use them sometimes--all in all it is about an 1:1 ratio of windows nero to linux k3b.  That says a lot about the program.

In terms of other apps:  office, browsing, chatting, music and movie playback I find the solutions to be on par.  In fact, I have chosen VLC media player on my Mac and Windows boxes because it is nice and plays some content without me having to install codes.  Under Linux this app exists and works just as well.

Ubuntu / Linux provides a lot of customizability but this takes a considerable effort to work through in order to get a desktop that looks as sharp as many of the screen shots of others desktops.

Give it a chance, think more like Linux and less like Windows, and take it a step at a time, walk don't run, and you'll have a pleasant experience with a long Ubuntu career.  Not only that everything you learn now you can help others with and you can contribute something back.

### Post by Phrawm48 on 2007-04-16
For me the "must have" Windows programs were Adobe FrameMaker 7.0 and Acrobat.  FrameMaker runs faster under Wine than it does W2K Pro SP4 *but* I couldn't figure out how to make Wine work with Postscript fonts.

In response, I first used FontForge to convert the PS fonts to TTF.  That was okay.

Then I installed VMware Server and run W2K Pro SP4 under it.  That's a complete solution.  (Amusing side note: Windows under VMware is "shielded" by Linux such that a visit to the Shields Up! site showed all ports stealthed.  As such, Linux makes a dandy firewall for Windows....)

Also, get the O'Reilly book "Ubuntu Hacks".  As a ten-week-old newbie, I found it really informative and useful...
