> **JordanJP said:**
> 1) I heard of Avast Antivirus Linux Edition for Ubuntu, and I also heard that viruses and whatnot is uncommon for the Ubuntu environment. Would anyone recommend that antivirus software be installed ? And what is the current security threat like for the Ubuntu system ?

Personally I don't run any anti-virus on my Ubuntu machines and I don't know anybody who does. The reason for this is that Ubuntu (well, Linux and UNIX in general) is designed in a completely different way from Windows such that it is extremely difficult for a potential attacker to run anything on your machine that you haven't decided to run yourself. Of course, that doesn't protect from threats where someone convinces you to run something but nor does an anti-virus. So as long as you don't blindly copy some strange command line that someone you don't know has posted to a forum, you should be fairly safe.

> **JordanJP said:**
> 2) I've also heard that a firewall would be a good idea. Where can I procure and install a firewall for Ubuntu ? Are there anyone Firewalls that you would recommend in particular ?

Ubuntu has some firewall capabilities out of the box. A good introduction is [the Ubuntu firewall wiki page]("https://wiki.ubuntu.com/UbuntuFirewall"). But note that firewall on a netbook may get complicated: it's a mobile machine that is likely to connect to a variety of networks so firewall settings that may make sense for one network may not make sense for another.

Other than that, enjoy Ubuntu!

### Post by 3Miro on 2009-05-01
1) There are not viruses in the windows sense for Ubuntu or any Linux/Unix. There are security flaws and vulnerabilities (nothing is 100% secure) but not viruses. Avast looks for windows viruses. You can get a file with a windows virus and while it will not affect you in any way, to can send the file to someone else (that uses windows) and infect them. Unless you are on a network with windows machines (or you have a dual-boot setup), Avast is meaningless. Otherwise, it will only help protect the windows machines around you (and not you).

2) You already have a firewall installed. Linux comes with one. There are several tools that will let you mess with its precise settings, but unless you have a good reason to do it (and know what you are doing), you probably shouldn't mess with it (it will work just fine).
