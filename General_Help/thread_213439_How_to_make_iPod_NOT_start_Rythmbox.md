### Post by garton on 2006-07-11
When I plug in my iPod in Ubuntu, Rythmbox is started. How can I configure it so that Rythmbox is NOT started when I plug in the iPod?

Thanks!

---

### Post by Wolki on 2006-07-11
System -> Preferences -> Removable Drives -> Second Tab -> Uncheck Portable Media Players.

(note: backtranslated from my localized versions, things may be named slightly differently)
