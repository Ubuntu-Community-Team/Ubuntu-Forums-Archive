### Post by ewl1217 on 2006-09-29
You can probably share simple binaries with any GNU/Linux OS, provided that the processor architecture is the same (obviously an AMD64 or PowerPC binary wont work in a generic x86 computer). Any binaries made for the some processor architecture should (in theory at least) run in any GNU/Linux system with the proper dependencies (GTK+, QT, etc.). As for Windows binaries, you can run some of them in GNU/Linux using Wine. Wine lets you run run many Windows programs in Linux, but some features may not work properly, and some programs might not work at all (especially games).
