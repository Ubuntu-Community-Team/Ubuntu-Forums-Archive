### Post by vinnn on 2007-03-18
Type **about:config** in the address bar and hit enter.
Here, you can customise a ton of settings affecting how Firefox behaves.

In the Filter box type in **middle**
Double-click on **browser.tabs.opentabfor.middleclick** so the value column shows **true**.

Enjoy!
