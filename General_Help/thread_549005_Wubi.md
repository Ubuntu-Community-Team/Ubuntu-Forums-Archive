### Post by p_quarles on 2007-09-12
This might not be the answer you want, but . . . setting up a normal dual-boot configuration is actually easier than you think, and is much more stable than using Wubi. 

It's as simple as this:
1) Backup your data, and make sure you have your Windows disks handy
2) Defrag your hard drive twice
3) Boot your system from the Ubuntu installation CD, select "Install," and the installer will guide you through all the necessary steps. 
4) When the "partition" menu comes up, make sure you select the correct option ("automatically resize the existing volume and use newly freed space").

That's really all there is to it. The only thing that would make a difference is if you have Vista, in which case it's a slightly different procedure. But, really, this will do everything for you automatically, and it only *sounds* difficult because you haven't done it yet. :)
