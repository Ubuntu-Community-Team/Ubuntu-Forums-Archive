### Post by sithsasquatch on 2008-04-09
I've encountered that error before. When the Flash page is published, it creates that missing js file. There's nothing to fix on the client side; the web developer needs to move a copy of that js file to the directory containing the flash page, and set the necessary permissions for the js file.
