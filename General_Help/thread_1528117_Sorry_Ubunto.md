### Post by jwbrase on 2010-07-10
Alt-tab switches windows, ctrl-alt-f1 switches to a virtual console (press Alt-F7 to get back, if it works). If the system responds to either one of these key combinations while the mouse is frozen, that means the trouble probably is with your mouse. If it doesn't respond to either key combination, that means it's not responding to the keyboard either, and thus that the whole system is probably locked up.
