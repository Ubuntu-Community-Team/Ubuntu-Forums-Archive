### Post by jo3llopez on 2007-12-02
1- update & upgrade
code: sudo apt-get update
        sudo apt-get upgrade

2- Install Java
java code: sudo apt-get install sun-java6-jre

3- Download file from [www.frostwire.com](www.frostwire.com) to desktop

type: "cd Desktop" on terminal then press enter

next-  install frostwire

code: sudo dpkg -i frostwire-4.13.3.i586.deb

everything should be fine!
to type on search box, disable chat and disable tips at start up, everything should work fine ;)
