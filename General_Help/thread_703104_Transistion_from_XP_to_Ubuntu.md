### Post by gnelson on 2008-02-21
Use the FoxMarks extension for Firefox to transfer your bookmarks.  It will synchronize your bookmarks across every PC that you use Firefox on - Windows, Linux or otherwise.  If you use multiple PCs or dual boot, this is a must have!

### Post by ryanhaigh on 2008-02-21
If you dont need exchange integration use mozilla thunderbird for email, install it on windows and it should ask if you want to import settings and email do so and once its done you can copy your profile folder (C://Documents and settings/username/applicaton data/mozilla/thunderbird/profiles/ i think) to a usb/cd and then once ubuntu is installed install thunderbird and copy the your profile folder into /home/username/.mozilla-thunderbird/. You can do the same thing with firefox so you can keep ALL your settings not just bookmarks (addons/saved forms and passwords etc.)
