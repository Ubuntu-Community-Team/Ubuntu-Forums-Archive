Well, according to one of my friends, mounting a Windows partition in RW mode is the reason why my Windows partition starts to affect my performance in Windows. For example, I'd start getting random chkdsks and it seems that the OS in general just starts going slower and slower.

Is mounting a Windows partition in RW mode the reason Windows starts to go crazy over time?

### Post by 3Miro on 2012-06-18
If you mount the main windows drive and make a lot of changes to it, windows will start to complain. Use RW only when necessary and run windows defrag tool.

However, this shouldn't affect windows too much. If you do not boot windows regularly, there are background tasks that it will want to do and it will be slow because of that. Normally if you use Windows every day, those tasks will not build up and Windows will be faster.

Finally, there is always the possibility that Windows got something broken either in the registry or something more serious (i.e. a worm).
