### Post by prshah on 2009-02-13
> **onesojourner said:**
> Yesterday I was surfing and came across this screen shot of a kde 4.2 desktop. I love the transparent windows. Is there a theme that does this in gnome?


Yes, you can do this in Gnome if you are running Compiz; See System-Preferences-CompizConfig Settings Manager-"Opacity, Brightness and Saturation" plugin.

In the Opacity tab, set the opacity for whatever class or title of windows you want affected (set to "any" to effect all windows).

However, note that this will effect the windows as a whole, including the contents within them, so setting the opacity to nil will give you an invisible window; a low opacity settings will be OK.

You might also want to check out the "Opacify" and "ADD Helper" plugins.
