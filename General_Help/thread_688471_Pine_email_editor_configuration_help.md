### Post by mali2297 on 2008-02-05
I think a simple solution would be to use a signature:
[indent]* Create a file **.signature** in your home directory. 
* Write your custom signature, it may consist of only some empty lines if that is what you want. 
* Save the file. 
* The next time that you use Pine to compose a message, the signature will be automatically added. It is placed at the bottom, but before any copy of a replied message.
[/indent]
I hope it helps.:-)

By the way, the editor in Pine is called **Pico**. It is the original which Nano has replaced.
