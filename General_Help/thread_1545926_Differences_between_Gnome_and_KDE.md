### Post by sikander3786 on 2010-08-04
Hi.

GNOME and KDE are two different but leading desktop environments. Ubuntu by default include GNOME.

Each environment has its own look, different set of default applications etc. KDE is said to be friendly for Windows Migrators and GNOME is said to be more stable. I have no references though.

You can have both GNOME and KDE (and many more like xfce) installed on the same machine. You can switch between them at the login screen. Check out both of them and chose the one that fits you best.
