### Post by mb_webguy on 2008-09-17
It's really no different than with other operating systems.  The most recent version of Mac OS X is named "Leopard", and Windows Vista was code-named "Longhorn" before its release.

A little bit of imagery can be more useful than numbers when differentiating one release or version from another.  It might slip  your mind whether you're using Ubuntu 7.10 or Ubuntu 8.10, but you definitely know whether you're using Gutsy Gibbon or Intrepid Ibex.
