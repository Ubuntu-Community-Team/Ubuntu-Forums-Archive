### Post by inameiname on 2010-06-02
Anybody know how to add a space between all the lines in a document using gedit? In order for a specific script I have to read each specific line, there must be a space in between them. And my current document has well over 1100 different things, and as such, having to add a space between each one would take awhile by hand.

---

### Post by john newbuntu on 2010-06-02
In search->replace, enter \n for search and \n\n for replace. Hit replace all.
