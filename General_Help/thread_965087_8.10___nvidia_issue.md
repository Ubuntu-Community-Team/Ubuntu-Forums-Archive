### Post by SLA_leandrin on 2008-10-31
Guys, this is what I've done, worked for me.

After restarting my machine in safe mode, choosing the option to restore X. I started the console and installed envy;

```
sudo apt-get install envyng-core
```

Once it's installed (It's reaaaally fast) then,

```
envyng -t
```

And chose to install de Nvidia driver 173.14.12-1

It worked fine after that. Hope it helps-

Cheers.

### Post by JECHO on 2008-11-02
hey guys i had a similar problem on a clean install and i got it to work so heres how...


backup all of your important files and do a clean install. once ubuntu 8.10 is in update it through the update manager (system>administration>update manager). reboot.

once rebooted open up resricted hardware drivers (system>administration>hardware drivers) and activate the version 177 driver.

it might hang for a few minutes at "downloading and installing driver 0%" but thats ok. wait it out.


it will eventually finish and you should reboot and itll be activated... if it seems to hang at the downloading screen for a really long time. then just reboot and it should be installed and enabled (idk why? haha but it worked for me... ive instaleld it on 2 machines)



hope that helped
