### Post by hojjat on 2012-05-01
Ubuntu 12.04 comes with this new feature that changes the background of login screen to the background chosen by the user account that you select to log in. Is there a way to disable this feature?

---

### Post by GameX2 on 2012-05-01
If you use a XML as a background in your account, this will work.

I would suggest you to set an XML as a background, and in the XML file, use only a single "still" image.
Then you'll be able to change the login background, and your problem will be solved.
