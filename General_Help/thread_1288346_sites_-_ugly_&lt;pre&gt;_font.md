### Post by Vaphell on 2009-10-11
Firefox: Edit->Preferences->Content
there are font settings, you can set size and minimum size and choose default fonts for serif, sans-serif and monospace types and even prevent pages from using other fonts than these defined. Some pages can break slightly though but i prefer readability to having beautiful layouts optimized for windows users not and being able to read a thing.
