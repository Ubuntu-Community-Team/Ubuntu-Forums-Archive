### Post by fjf314 on 2010-01-21
So did you already put Ubuntu on the system and now you want to load XP back on again?

You should just be able to boot to your XP CD, format the drive as NTFS, and run through the XP installation. You actually should have been able to do this over the broken Windows installation without having to install Ubuntu on the drive first. Your XP CD should always give you the option to format the drive.
