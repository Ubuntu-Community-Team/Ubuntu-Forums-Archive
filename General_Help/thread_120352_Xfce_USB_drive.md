How do I get at the files on my USB key ring drive in Xfce?

---

### Post by Iandefor on 2006-01-21
open nautilus with the following command: ```
nautilus --no-desktop
``` you should be able to access it from there. If it doesn't mount automatically, run the following command:```
gnome-volume-manager
```
