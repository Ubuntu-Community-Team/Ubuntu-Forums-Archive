### Post by carlosqueso on 2005-12-12
The server install is a minimal install that only has a command line.  If you have an internet connection and want a GUI, you can type ```
sudo apt-get install gdm
sudo apt-get install ubuntu-desktop
```

However, if you have the resources and want all of the other programs and stuff, or if you lack an internet connection on ubuntu, it may be better to install the standard version. You can still use it as a web server if you install apache.

"We are quick and we are kind" , with those word I too welcome you to the world of ubuntu.

If your new to ubuntu and Linux you should start off the graphical-gui. 
Then when you have finished installing it you will have a menu in the upper, left corner, were you have diffrent choices. such as [Applications] [Places] and [System] under the [system] and then the sub-menu [Administration] you will find a program called "Synaptic package manager" which is the program used for adding new software to the machine. 
If you start it you can search (with the search button) for software you want and then just right-click on it and choose "install" to add the application. (followed by the "Apply" button ofcourse)
