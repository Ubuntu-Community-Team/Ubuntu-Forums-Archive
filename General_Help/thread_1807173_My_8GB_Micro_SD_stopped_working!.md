### Post by JC Cheloven on 2011-07-18
More than likely your sd card died. It's something that "always happens to other people" ... so we think ;)

Nevertheless, try gparted to create a new partition table, and then try to define a partition. 
If that fails... well, try in win2 just in case. Please read [this]("http://windows7themes.net/windows-7-might-kill-your-sd-cards-hotfix.html") first.

If that also fails, oh well, you'll learn to live without that.
