### Post by snowpine on 2008-09-05
> **exploder said:**
> It is against the forums policy to discuss this here.

I disagree; I think we should be encouraged to discuss root vs. sudo because it is a very commonly-asked question. :)

Chamkaran, many branches of Linux use a separate root account for administrative purposes, and it is understandable for a new user to expect the same from Ubuntu. However, Ubuntu uses a different security model where privileged users can temporarily assume root powers by prefacing commands with 'sudo'. 

It is possible to circumvent Ubuntu's security system and enable the root account. It's not hard to do so, but you won't find instructions to do so on these forums (that's what Exploder was referring to). Forum policy is that new users should be encouraged to use the default 'sudo' system rather than enable the root account. 

Anyways, I'm not telling you anything that's not in Ryadat's link, but I feel you deserved a personal response. So, welcome to the forums and don't be afraid to ask questions!
