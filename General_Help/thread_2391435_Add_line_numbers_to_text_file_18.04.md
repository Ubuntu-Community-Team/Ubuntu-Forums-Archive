### Post by PaulW2U on 2018-05-09
> **Otto-C said:**
> How to add line numbers to text file and or gedit 18.04
Assuming that you don't really want to add line numbers to the text file itself you can enable the display of line numbers on the View tab of Preferences.

You can find "Preferences" by clicking on "Text Editor" in the top bar.
