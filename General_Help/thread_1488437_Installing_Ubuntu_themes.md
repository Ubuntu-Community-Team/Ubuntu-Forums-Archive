### Post by blazemore on 2010-05-20
Most themes from websites such a gnome-look will provide, say, a theme file in a tar.gz format (this is a compressed folder, similar to zip)

If you right-click the desktop, go to Change Desktop Background, click the appearance tab. Drag and drop this tar.gz file onto this window.

The same goes for zipped icon packs.

### Post by Rubi1200 on 2010-05-20
You can also install some themes, icons etc. from the official repositories.

Go to Synaptic and search for gnome themes, icons and you should come up with some stuff. If interested, install and then when you right-click on the Desktop to change background etc. you will find them installed and ready to go under Appearance.
