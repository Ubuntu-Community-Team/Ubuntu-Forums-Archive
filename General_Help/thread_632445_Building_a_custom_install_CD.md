### Post by .nedberg on 2007-12-05
Have you tried g4l or Clonezilla?

The can both make a image of a drive and then put that image on a new drive. Seems to me this is similar to what you want. The do not install on the other computers though, the clone the original machine with users and files.

If you have used Ghost on windows you know what it is like.
