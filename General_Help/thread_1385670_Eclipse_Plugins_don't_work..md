### Post by adimit on 2010-02-10
Check this site for more information on Ubuntu's version of Eclipse. It's slightly different from the default install (because it's installed with root privileges, the plugins management system has to be a little different. Eclipse usually assumes to be installed with the user's privileges, or at least be able to write to its installation location.)

[https://help.ubuntu.com/community/EclipseIDE#Plugins](https://help.ubuntu.com/community/EclipseIDE#Plugins)
