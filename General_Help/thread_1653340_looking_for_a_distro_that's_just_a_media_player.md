### Post by WorMzy on 2010-12-26
Arch.

Install Arch, install alsa-utils, wham-bam-thank-you-ma'am, you have a minimal install with mp3 playing capabilities.

Although you may need gstreamer/gstreamer-ffmpeg too.

mpd/mpc could work as a player for you.
