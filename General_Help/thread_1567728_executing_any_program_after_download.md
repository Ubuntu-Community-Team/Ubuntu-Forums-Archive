### Post by JedMeister on 2010-09-04
Not that I'm aware of. Its one of the security features of Linux (one of the reasons why Linux doesn't get infected with lots of nasty malware like Windows - you can only run downloaded files you really want to). The only workaround that I'm aware of is if files are downloaded using rsync or come in a tar.gz archive (as permissions can be maintained in those instances - they still need to be set to execute initially though).

Still there may be another way around it that I'm not aware of (many things are possible in Linux if you know how).
