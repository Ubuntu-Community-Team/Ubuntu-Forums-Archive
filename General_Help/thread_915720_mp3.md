### Post by triphazard on 2008-09-10
I'm not 100% sure what you're trying to do but if it's make an audio CD from some MP3s then I personally use an app called K3B.  You can install it by typing ```
sudo apt-get install k3b
``` into a terminal window.

Alternatively, if you want to rip mp3s from a CD I think it's Sound Juicer that I use for that.  I believe it comes with the default Ubuntu installation but I could be wrong.

I hope this helps.
