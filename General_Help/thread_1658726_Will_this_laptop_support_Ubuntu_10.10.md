### Post by theasprint on 2011-01-05
Hi,

The main difference between the Netbook and Notebook version is about the interface.

The Netbook version has its interface fit to a small screen, thus things will appear to be bigger, like icons etc.
Also just for netbooks, it focuses more on Internet usage. 

However the Notebook version is the more workstation type, which actually uses more power than the Netbook version. 

For more info, you may want to compare the netbook and notebook versions in ubuntu.com. There are features you can decide on.
