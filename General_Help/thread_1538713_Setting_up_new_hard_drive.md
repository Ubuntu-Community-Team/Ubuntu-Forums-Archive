### Post by geoffjay on 2010-07-25
Windows is only capable of formatting drives in FAT and NTFS so no, you would not format first in Windows. The Ubuntu installer will do the formatting for you.

FYI, if the drive partition part of the installation process does not recognize all of your drives you may need to download the alternative installation disks. I had that problem before but it was when Lucid was still in beta so I'm not sure whether or not that's been fixed.

### Post by jellybaby on 2010-07-25
Hi

No, you don't have to format in Windows.

1. read up on Linux drive naming,
2. plug in your drive,
3. enable it in your BIOS,
4. boot from your Ubuntu CD,
5. the install routine will guide you from there

Hope this helps
