### Post by 3Miro on 2011-03-23
You should start with Lubuntu and/or Xubuntu (Lubuntu is probably better). Arch may run well, but I think they have their binaries compiled for i686, which may not run on PII. Salix OS is another good possibility, Slackware is very resource conservative.

Gentoo is probably too much hassle to install on that machine (although it will be the smallest of all and most efficient, also you can run the compilation during several nights).

At any rate, look at i386-i586 and LXDE or XFCE. You can also try IceWM, but i have not used it.
