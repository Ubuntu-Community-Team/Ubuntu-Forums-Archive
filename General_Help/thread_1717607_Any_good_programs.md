Casual games must have :

[chromium-bsu]("http://www.reptilelabour.com/software/chromium/")  (old school arcade space shooter..)

```
sudo apt-get install chromium-bsu
```

Great tool:
[easystroke]("http://sourceforge.net/apps/trac/easystroke/wiki")

..command your desktop actions (eg.close/minimize windows,shut down PC,open apps,whatever) with mouse gestures.Once familiarized with it,you will never want to abandon on this great tool.

Edit.

..if running powerful graphics card,you can spend some time tuning compiz;eg. FSAA,equirectangular skydome images..

### Post by Topsiho on 2011-03-30
Well, you could give KStars a try, which simulates the stars and so on in the sky. It also has some nice documentation which is quite worth reading.

It is of course a KDE program, so you'll need to install a lot of extra files (dependencies) if you don't have KDE on your computer.

You can install kstars via Synaptic, or just coding in the terminal:

sudo apt-get install kstars

and enter your password, and probably have to answer yes.

KStars  might keep you busy for hours, days, weeks, months or years :)

Topsiho
