### Post by taurus on 2007-02-24
Applications -> Accessories -> Terminal
```
cd /usr/local/games/quake4/q4base
sudo chmod 755 quake4-linux-1.0.2147.12.x86.run
sudo ./quake4-linux-1.0.2147.12.x86.run
```
