### Post by andaryfaysal on 2012-05-09
Hi,
In the previous versions of Ubuntu "expo mode" was accessed by the keyboard shortcut 'Super + W', which caused all windows from all workspaces to be spread on the screen so one could easily navigate between all open windows. However, after upgrading to 12.04 LTS, the same shortcut only spreads windows in the current workspace. My question is: is there a new keyboard shortcut for "expo mode", or maybe a way to bring the old functionality back?
Thank you.

### Post by flipper T on 2012-05-09
you can change this option in ccsm.

go to ¨expo¨ option, de-select the single workspace options and select the multiple workspace options.

job done.

ps compiz as a nasty habit of ¨forgetting¨ these settings, so you may have to reset.

personally i have just switched to the kde DE. expo is built in. no more compiz for me :)

### Post by stinkeye on 2012-05-09
Super + W uses the scale plugin and now, only shows windows from the current workspace.
I use the ring switcher to show windows from all workspaces.
