### Post by mb_webguy on 2009-02-04
As steeleyuk said, Adobe never released a Shockwave plugin for Linux, and since Shockwave is on its way out, it's unlikely one will be developed.

If you absolutely must have Shockwave, you'll need to install Firefox under Wine so you can use the Windows plugin.

### Post by blazemore on 2009-02-04
Unfortunately, Adobe chose not to release a version of Shockwave for Linux.

As it stands, the best thing to do is grab a copy of Firefox for Windows, and install it under Wine (sudo apt-get install wine)

You can then install the shockwave plugin.
