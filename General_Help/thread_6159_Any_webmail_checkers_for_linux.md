### Post by humina on 2004-12-02
I use the mail notification program:

[http://freshmeat.net/redir/mail-notification/44120/url_homepage/mailnotify](http://freshmeat.net/redir/mail-notification/44120/url_homepage/mailnotify)

It can check POP3, IMAP, and gmail.  I haven't seen one that can check yahoo and other webmail.  The above link wasn't working earlier so just in case here is a freshmeat link:

[http://freshmeat.net/projects/mail-notification/?branch_id=44120&release_id=179923](http://freshmeat.net/projects/mail-notification/?branch_id=44120&release_id=179923)
