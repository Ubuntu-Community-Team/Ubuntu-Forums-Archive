### Post by Blasphemist on 2011-07-22
I could go into a big description on doing this but I couldn't provide as good of information as this site. [http://members.iinet.net.au/~herman546/](http://members.iinet.net.au/~herman546/)

Herman did a great job with this. You will end up with neither windows nor ubuntu being the main or controlling installation really. You can determine the default to boot but you can set it up to allow you to choose at the time of boot. You are well positioned for this having windows installed first. You will have the choice of letting this installation of ubuntu do the partitioning itself or doing it yourself. Herman describes both options. I recommend that you follow his guide and make notes if necessary before you start the ubuntu install.
