### Post by varunendra on 2012-07-21
I think you should boot with a live cd or usb, install [boot-repair]("https://help.ubuntu.com/community/Boot-Repair"), run it (while being connected to internet) and post the pastebin link it gives you. It is also capable of fixing some basic problems automatically, so you may not need to manually troubleshoot the issue at all.

In case you are not connected to internet while running it, it should create a "Results.txt" file in root's home (but I'm not sure about the name or exact location of the file). Search it and post back its contents.
