### Post by omgDarkWillow on 2007-05-29
Can I install Ubuntu Studio with Wubi, while still keeping Ubuntu and Windows on, or do I have to remove Ubuntu and install Ubuntu Studio with Wubi?

---

### Post by tuxcantfly on 2007-05-30
If you already have Ubuntu installed, just use Lubi, instead of Wubi (it tends to have less issues with bootloader configurations and the like), and to answer the question, yes it'll keep both ubuntu and windows intact, just boot up Ubuntu and run Lubi, then it will install UbuntuStudio for you, and add an extra boot menu entry for it in GRUB.

Lubi guide here: [http://ubuntuforums.org/showthread.php?t=441918](http://ubuntuforums.org/showthread.php?t=441918)
