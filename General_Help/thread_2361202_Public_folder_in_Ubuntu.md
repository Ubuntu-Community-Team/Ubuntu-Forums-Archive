### Post by yancek on 2017-05-13
What you seem to be referring to is a shared data partition.  You can create a mount point for it, then create a separate partition and create a group and put the users you want in that group and give them whatever permissions you want.

You can basically do the same without creating a separate partition.  It would probably work best if you did this outside a specific user /home directory.
