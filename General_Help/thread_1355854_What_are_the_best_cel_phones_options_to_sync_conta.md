### Post by StuartN on 2009-12-16
I am using Gammu [http://www.gammu.org/wiki/index.php?title=Welcome_to_Gammu.org](http://www.gammu.org/wiki/index.php?title=Welcome_to_Gammu.org) to transfer my contacts between phone and computer. It seems to support almost any phone.

The photos and other media just appear as a USB flash memory when the phone is plugged in, so any ordinary file manager will copy those.
