### Post by ap9er on 2006-11-19
you can copy the log file over to /home/*username*/.gaim/logs/aim/*yourgaimusername/personmessaging/*

I believe trillian saves as .txt so the only other modification you'll need to make is to re-name the log file into one with a date as gaim does it per conversation. E.g you'd name it 2006-11-18.161525+1300NZDT.txt
It won't show up as a valid log file unless named as such.

There might be another way, but I haven't found it.
