### Post by yugnip on 2012-01-01
Removing the ppas means you will no longer get updates for apps you have installed. So you might not want to do that :)

The thing about repos and ppas is that occasionally things get a bit mucked in the download process... expect a few errors on occasion.

One cool and easy way to manage ppas and solve common problems with them is to install [Y PPA Manager]("https://launchpad.net/y-ppa-manager"). With this handy tool you can search ppas for applications, add ppas to your sources list, and repair common ppas problems like the key issue you currently have.

```
sudo add-apt-repository ppa:webupd8team/y-ppa-manager
sudo apt-get update
sudo apt-get install y-ppa-manager
```

The maintainer for this ppa and app is Andrew from  [www.webupd8.org]("www.webupd8.org"), a highly trusted source, so no worries about adding yet another ppa.

Hope this helps.
