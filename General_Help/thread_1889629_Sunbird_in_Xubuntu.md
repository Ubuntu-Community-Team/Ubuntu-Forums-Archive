### Post by Rex Bouwense on 2011-12-01
Sunbird is the Mozilla stand alone calender.  I don't think that that it is being supported any more.  The program that you want is called Lightning and it is a Thunderbird extension.

### Post by kurt18947 on 2011-12-01
Rex is correct about Lightning.  One small niggle to be aware of is some people have lightning on Thunderbird 7 and it works properly.  Then Thunderbird 8 comes along and they get a message that Lightning 7.x is not compatible with Thunderbird 8 and will be removed.  I had this happen.  When I restarted Thunderbird 8 I got a message that there was an updated Lightning extension available so I went ahead and installed it.  I only had a couple calendar  entries which made the upgrade.  Some people though have had calendar entries lost in the upgrade process so if you're still on Thunderbird 7 and get Lightning 7, be aware of potential data loss during upgrade.

### Post by DapperMe17 on 2011-12-01
After you install Lightning, it will automatically integrate into the main window as a "pane" on the right hand side of the Thunderbird window. 

It's settings manager will be included in Thunderbird's default settings manager.
