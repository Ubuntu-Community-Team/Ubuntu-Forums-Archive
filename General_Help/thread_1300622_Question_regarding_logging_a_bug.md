### Post by 3rdalbum on 2009-10-25
Well spotted. I just tried it on Karmic - the problem exists there too. (you might want to add this to your bug report)

Go to [www.launchpad.net](www.launchpad.net), and login to your Launchpad account (or register if you haven't already got one). In the search field on the main page, type "ubuntu". On the page for Ubuntu, in the bit where you can type a package name, type "gedit". Then click on the page for Gedit. Now click the Bugs link at the top of the page and have a look through to see if it has already been reported.

If it hasn't been reported, you can click the "Report a Bug" button and follow the directions. If it has already been reported, go to the page with the bug report and click the "Affects me too" link. You'll be kept up-to-date with the status of the bug and any comments posted about it.

The bug will probably be forwarded upstream to Gnome. You'll recieve an e-mail if it is. If necessary, you can also look at the bug report on the Gnome bugtracker and add more information, but you'll need to register on the Gnome bugtracker too.
