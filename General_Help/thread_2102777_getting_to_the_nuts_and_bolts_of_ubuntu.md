### Post by Calinou on 2013-01-08
To open the command line: Ctrl+Alt+T (GNOME), search Terminal in the Dash (Unity), Super+T (Xfce -- "Super" is the "Windows" key)

Search around the web and you'll find some introductions to the command line.

To install packages, just use the Ubuntu Software Center. If you want a more powerful graphical package manager, try Synaptic which can be installed from the USC. Or you can use the command line:
```
sudo apt-get install <package name>
```
(password is hidden for obvious reasons while being typed)
