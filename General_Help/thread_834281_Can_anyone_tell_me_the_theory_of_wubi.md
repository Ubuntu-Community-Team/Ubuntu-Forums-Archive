### Post by jualin on 2008-06-19
Wubi is a new way of installing a linux distro in which partitioning the Hard Drive is not necessary. Wubi creates a virtual partition image on Windows and this is where the Linux files are saved, therefore is like a "big" file on windows which holds Linux in it. 
That's in my own words so it may not be as technical. Hope this helps!
