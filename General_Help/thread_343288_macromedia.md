### Post by fakingit on 2007-01-21
I faced this problem some time ago when I made the switch. Despite my best efforts, I failed to find a free solution that even come close to Macromedia that run on linux.

However, there is a very good alternative that I found. Vmware. Search the forum for it, there's an installation tut.

Basically VMware allows you to install windows xp onto linux which you can run at the same time on your desktop. From there you simply install macromedia and whatever else you need. It's how I've managed to get everything working that I couldn't get working on linux :)
