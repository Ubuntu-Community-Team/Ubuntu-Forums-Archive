Welcome to the forums!

[This book]("http://www.amazon.com/Classic-Shell-Scripting-Arnold-Robbins/dp/0596005954/ref=sr_1_7?ie=UTF8&qid=1299590324&sr=8-7") will teach you POSIX shell scripting -- a set of shell commands that will work with a lot of different shells (bash, tcsh, ksh, etc).
[URL="http://www.amazon.com/Teach-Yourself-Shell-Programming-Hours/dp/0672323583/ref=sr_1_1?ie=UTF8&s=books&qid=1299590218&sr=8-1"]
This book[/URL] will teach you a handful of specific shells. These days Bash is the most common shell you'll find on Linux systems. It might be the easiest shell to find good documentation for. Don't write off the others, though -- plenty of people swear by the alternatives. I've used zsh a fair amount and it's pretty great.

Hope that helps!
