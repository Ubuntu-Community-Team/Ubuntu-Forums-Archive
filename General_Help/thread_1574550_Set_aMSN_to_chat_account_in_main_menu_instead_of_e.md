### Post by Thelasko on 2010-09-14
I assume you are talking about the desktop notifications.  This is done through the libnotify package.  Unfortunately, aMSN doesn't appear to have a libnotify interface.  You may want to check aMSN's settings to make sure.  However, it is possible to write one.

Pidgin does have a libnotify interface called "pidgin-libnotify" in the repositories.  Unfortunately, I don't know if file transfers work in Pidgin or not.
