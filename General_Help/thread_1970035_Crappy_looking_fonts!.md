### Post by codingman on 2012-04-30
Go to terminal and type in:

sudo apt-get install gnome-tweak-tool

Then go to the launcher and search for "advanced settings" and you should see the application of advanced settings. From there, click on fonts and then you can customize whichever fonts you want.

Hope this helps!
Codingman
