I need to create a cd with Dapper (iso) from my machine (i386) but the destination disk is for a friend that use Mac. Is this possible and what I need to do that in terms of configuration? Usually I use GnomeBaker as record cd but is not a problem to use anything else...

### Post by hw-tph on 2006-10-14
If you are burning a CD from a CD image (.iso or whatever) it doesn't really matter - the contents and layout of the CD is stored in the image and will be replicated just fine. If you are creating a data CD (with files and folders) you can create a CD just like usual - a Mac can read standard CD's just like a Wintel PC. It gets tricky if you are creating a bootable CD but a regular data CD should work without any special tricks.
