### Post by Dayofswords on 2010-08-14
```
cd ~/Pictures ; mkdir RENAMEJPGBACKUP ; cp *.JPG ./RENAMEJPGBACKUP/ ; rename -v 's/\.JPG$/\.jpg/' *.JPG
```
it works, why idk. i tried and it does work and i made it from [this]("http://tips.webdesign10.com/how-to-bulk-rename-files-in-linux-in-the-terminal")
so just in case things go wrong the first two parts copy the files into a folder called RENAMEJPGBACKUP that was just created, just as a precaution

EDIT:hundreds you say...the back up will be as big as the pictures you want to rename, but you can delete the folder once done
