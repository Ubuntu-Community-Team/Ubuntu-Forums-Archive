### Post by raptir on 2013-12-03
If it's a laptop with Nvidia Optimus you may not have that option in the BIOS. Bumblebee has worked for me in the past.

[https://wiki.ubuntu.com/Bumblebee](https://wiki.ubuntu.com/Bumblebee)

Once you get it set up you can launch a program using Bumblebee and it will use the discrete card.
