### Post by BC59 on 2011-12-12
Install Ubuntu to a USB through Unetbootin (using Windows) or burn a CD with the image.

Then boot and follow the instructions to install (not try).

Then you will reach a screen asking you, what you like to do, keep Windows and install Ubuntu besides etc. Choose the option to erase the whole disk and install Ubuntu using the whole space.
That's it.
