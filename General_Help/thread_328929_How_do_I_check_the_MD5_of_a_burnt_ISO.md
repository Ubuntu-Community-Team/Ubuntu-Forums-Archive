### Post by meng on 2006-12-31
You don't! If you burned the disk properly, it's no longer an iso file, it's a complete filesystem. If you browse the CD, you should see folders and files there. The only real way to check if this disk is well burned is to use it and see if it works.
By the way, it is a common mistake to just copy the iso file to disk, without creating the filesystem, so if you've done that, don't be embarrassed.
[www.psychocats.net/ubuntu/iso](www.psychocats.net/ubuntu/iso)
