### Post by 3rdalbum on 2008-10-03
Linux is not Windows - programs are never interchangable between operating systems. Having said that, Wine is an attempt at creating a compatibility layer by use of reverse-engineering Windows. Don't expect any program to work, just be grateful if it does.

There might be some special steps necessary to run Crysis in Wine. I suggest looking up "Crysis Wine HOWTO" on Google, and choosing the more recent document.
