### Post by johnc4510 on 2006-06-03
Download the theme, I put mine in a folder in /home
System>Preferences>Themes
There will be a button to Install Themes
Click on it and point it toward the folder you put the download in.
It will take care of the rest.
Note, No need to unzip or untar the theme, the installer will do it for you
