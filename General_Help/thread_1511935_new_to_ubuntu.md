### Post by lisati on 2010-06-17
I can't easily address the specifics of your particular hardware and software requirements without doing a small amount of research (my hardware is different to yours), but here goes with my $0.02:

I'd suggest trying a "Live CD" before installing Ubuntu, by booting from a regular installation CD and then selecting the "try Ubuntu" option. It will let you run Ubuntu without actually installing anything, and help give you an idea of what works and what might need a little bit of tinkering to get working. The experience won't be as fast as running a proper install because it will be running off CD, but will be a start. 

My own initial experience was that using the GUI was less painful start with, as it gave a small degree of familiarity that isn't present with the command line - point-and-click is pretty much the same whichever OS you use, while some of the commands available for CLI have a different name or take different options. Most of the software you'll be familiar with from your other OS will have an Ubuntu version (or at least an approximate equivalent); if you need any particular software, there will probably be a forum user who will be able to point you in the right direction.
