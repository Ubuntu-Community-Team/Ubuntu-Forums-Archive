Hi, your boot script looks good to me, there are two things you can try.
1. Unplug your usb drive, and see if it will boot then.
2. Boot into recovery with networking and run 
```
sudo apt-get update && sudo apt-get upgrade
```

Yeah the USB drive might be set to a higher priority than your hard drive. Go into BIOS and set your hard drive above everything else to avoid these problems in the future.
