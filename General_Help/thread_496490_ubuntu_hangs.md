### Post by ubergeeknz on 2007-07-09
I'd try to rule out your common hardware issues... these are easiest to troubleshoot...

Put in the install disk, reboot, and select 'memory test'.  Make sure everything passes, if memtest86 reports ANY errors, shut down the PC and pull one stick of RAM, repeat test.  Then try the other stick on its own.  If only one stick fails, replace that one, if both fail (!) replace it all.

Secondly, find a hard disk checker utility and test your HDD.  UBCD has several on it (easy to find by google).

Hope that helps :)
