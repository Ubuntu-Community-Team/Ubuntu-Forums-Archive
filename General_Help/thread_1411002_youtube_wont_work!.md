### Post by cogier on 2010-02-19
Go to Applications>Ubuntu Software Centre. In the box, top right, type "rest" you should see "Ubuntu Restricted Extras". Download and install this and you should be OK

If you can't see this have a look at the "View" menu and ensure it's set to "All Applications".
