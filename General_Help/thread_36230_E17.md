I've been using e17 (on gentoo) as my full time window manager since december of last year, and it is very nice, and improving day by day.  If you are having trouble installing any of the components, I suggest checking out [www.get-e.org;](www.get-e.org;) some members of the e team and a couple of gentoo users put up a wonderful install-configuration walk-through there with lots of information.

As far as the moving background thing goes, if you are talking about the login "e" logo that rotates/etc, that is part of the gdm replacement built for enlightenment 17, called "entrance" ... It's really very nice, and I have been using it (along with many other e17 apps) as long as I've been using e17.  If, on the other hand, you're talking about that moving wallpaper (you can see a video of that on rasterman.com/files/) with trees and smoke and whatnot, then I should inform you that that was a (rather ugly) proof-of-concept, and it doesn't perform very well on most machines -- mostly because it takes up so much memory and hardware rendering support is generally less-than-great on linux machines (the enlightenment team has built opengl support into their EVAS engine, but they claim hardware support on linux still isn't mature enough to make it stable long-term, though it seems to work okay for me).  Anyway, I've heard of that wallpaper bringing various users' systems to a crawl, so I wouldn't recommend it if you get e running at this point...though things will improve, I'm sure.
