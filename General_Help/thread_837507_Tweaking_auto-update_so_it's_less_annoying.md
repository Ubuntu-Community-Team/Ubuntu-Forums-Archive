Right now the only updates I have enabled are security updates. It's giving me an option to install an update to clam (AV), which I don't want. Do I need to remove the software (if I have it) for this to stop? Can I have auto-update ignore that one program/update?

### Post by windoze87 on 2008-06-22
you can ignore that update for the session you're in only, so whenever you turn your box back on and connect to the Internet, it'll be there again. I may be wrong. Removing it is easy though... go to System>Administration>Synaptic Package Manager. Click "Search" and type in "clamav". When the search finishes, you'll probably find that there are some green boxes filled in that have to do with clamav. Right click all of these and select "Mark For Complete Removal" and you should be set
