### Post by zeronothing on 2009-10-13
if you want to uninstall programs from your system you can use "synaptic" or the "Add/Remove Applications" program. they are located at:

System -> Administration 

when you open up either application, just search for the program you want to uninstall and choose remove or uninstall.

some cool apps to install are:

Audacity - a music/sound editor
VLC - a great media player
Deluge - a great bit torrent client
handbrake - great video encoder
miro - internet TV/podcasts
