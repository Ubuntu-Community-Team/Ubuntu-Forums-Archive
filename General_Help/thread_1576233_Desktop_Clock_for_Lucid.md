Thanks for pointing me in the direction of Cairo clock, Orographic.  It's working reasonably well for me.  One initial problem I had was it reverting back to its default position when I restarted my computer.  That was fixed by following Quoll's post here:  [http://ubuntuforums.org/showthread.php?t=926312]("http://ubuntuforums.org/showthread.php?t=926312")

I've added Cairo clock to my start-up programs, and I've noticed if I have the clock set for a theme other than default, when I start up my pc, the clock is "deformed".  Clicking to another theme and then back to the one I want fixes the "deformity".  Or I can leave it on the default theme, and it'll always load just fine.
