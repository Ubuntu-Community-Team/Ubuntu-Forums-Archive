### Post by eriqjaffe on 2008-10-26
It's possible to shuffle the login window amongst your installed GDM themes:

System -> Administration -> Login Window

On the "Local" tab, make sure all the themes are selected and set the "Theme" pull-down to "Random from selected".

Not so sure about the usplash (the loading progress-bar graphic) or the sound, though.
