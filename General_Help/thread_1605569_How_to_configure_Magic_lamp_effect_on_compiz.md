### Post by 3Miro on 2010-10-25
Go to Ubuntu Software Center and install ccsm (compiz-config-settings-manager). It will put an entry in System -> Prefs. Open ccsm, find animations. From there, you can find the magic lamp and look at the numbers behind it, you will have to change some of them.

### Post by WorMzy on 2010-10-25
Assuming you've already installed ccsm and compiz-fusion-plugins-extra, open ccsm (System -> Preferences -> CompizConfig Settings Manager), and click the Animations  plugin.

Choose which action you want the animation to be run for, e.g. minimize, and click it's tab. Click New and set the Effect to Magic Lamp, and enter which windows you want to use the animation. The option which dictates the speed on the animation is Duration. Setting this to a high number will make the animation take longer, entering a lower number will make it take less time.
