### Post by deadflowr on 2017-03-28
I've seen this.
Workaround that always works for me is to press the keyboard shortcut super + W which brings it back to normal (single window screen).
I've also gotten used to moving windows around with ctrl+shift+alt+arrow key.

It's a known issue:
[https://bugs.launchpad.net/ubuntu/+source/compiz-plugins-main/+bug/815996]("https://bugs.launchpad.net/ubuntu/+source/compiz-plugins-main/+bug/815996")
And an incomplete duplicate:
[https://bugs.launchpad.net/ubuntu/+source/unity/+bug/1301623]("https://bugs.launchpad.net/ubuntu/+source/unity/+bug/1301623")

for what it's worth I have never seen it get stuck mid-slide while using the keyboard shortcut to move, as the poster had in the first bug report.
(Though that user had been on the notoriously buggy 11.04 when it did)
