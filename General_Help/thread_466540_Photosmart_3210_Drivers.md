### Post by Ashex on 2007-06-13
You shouldn't need to install drivers for that printer. Ubuntu comes with hplip installed if I recall. Just go to printers under the system administration (or somewhere around there in gnome) and add the printer. Make sure it's plugged in first, then go through the steps, select HP as model, and browse down to Photosmart 3200.

However, if you want to install the hplip driver, you can find instructions [here](http://hplip.sourceforge.net/install/index.html).
