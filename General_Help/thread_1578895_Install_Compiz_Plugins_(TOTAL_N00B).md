### Post by Full_Auto_Legos on 2010-09-21
> **nimaid said:**
> Thanks, but I meant the 3D Windows Plug-in FOR the cube. (sorry about that)

Basically, I want to know how to get/install plug-ins for compiz and/or the cube. Thanks!

The cube is a standard choice feature you can select in the settings of Compiz, assuming you installed all the Compiz + Fusion packages.  There are a ton of choices.  Some choices do not work with other choices.  Compiz will usually tell you what can and cannot work together.  There is no "plug-in" per se for the cube.  The cube is not activated on the first start of compiz...you'll have to select it in the settings manager.

### Post by themusicalduck on 2010-09-21
Install the packages compizconfig-settings-manager and compiz-fusion-plugins-extra in Synaptic.

Then the 3D windows plugin should be listed in compizconfig settings manager.
