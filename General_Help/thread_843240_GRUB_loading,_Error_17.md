### Post by iaculallad on 2008-06-28
You need to post whatever displays when you issue the following commands below using your terminal.

```
sudo fdisk -l
cat /boot/grub/menu.lst
cat /boot/grub/device.map
cat /etc/fstab

```

So members can adapt solutions that can fit your need.

-Take time first to search the forum for solutions. There are many GRUB Error 17 threads that you can relate with.

Easy to fix; Download Super Grub Disk and follow the instructions from [http://www.supergrubdisk.org/](http://www.supergrubdisk.org/) to get back your Windows partition booting ability.
