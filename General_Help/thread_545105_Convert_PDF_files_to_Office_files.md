### Post by gobbles414 on 2007-09-29
Hi all,

If a simple copy and paste fails, I have another idea. Searching for *convert pdf to excel* in Google yields many results for Windows-based conversion utilities. Some of these may work in Ubuntu if installed in the Wine "Windows emulator". An OLDER version of Wine is available in the Synaptic Package Manager (a NEWER version is available if you enable backports in Software Sources). You can also download the NEWEST version from the [Wine homepage]("http://www.winehq.org"). The NEWEST version will require you to issue all of your commands from the terminal, since it has not been modified for Ubuntu like the versions in the Synaptic repositories.
