### Post by IWantFroyo on 2011-09-10
IRC isn't on Yahoo's servers. You won't get anywhere through trying to join a network through Yahoo, or using Empathy for it.

Go to [http://webchat.freenode.net](http://webchat.freenode.net) if you don't want to use a local client.

My personal favorite for IRC chatting is XChat. When using that, make sure you select Freenode's servers.
