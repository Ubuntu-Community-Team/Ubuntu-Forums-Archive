So first of all, and this will probably be the most str8 forward advice possible, get a sources.list for Intrepid, or Hardy at least, and run 
apt-get update
followed by 
apt-get dist-upgrade
Afterwards you should have compiz installed, and ok, maybe lets say two clicks away from your probably working driver...

OK, but if you don't want to do so... I'll try and help, even though I dont understand it, as it's for free...
OK, first I think you should open up a terminal, graphically edit your xorg.conf with su privileges, and post it here please.
```

gksudo gedit /etc/X11/xorg.conf

```

Maybe you just have to reconfigure your X-Server and everything will work out fine.

With ATI cards driver fglrx should be used if I remember correctly, I have a nVidia, so... Pls post your xorg.conf.

Then you should post, or more likely check urself if, and which parts of compiz are installed. In a terminal type:
```

dpkg -l | grep compiz

```
You should get a list with all installed pkges containing compiz as output.
Now check twice if you really have everything installed. OK.
Check again. 
Your xorg.conf should look sth like this, at least the part which is of interest:

# Uncomment if you have a wacom tablet

```

```

sudo gedit /usr/bin/compiz

"fglrx"

restart X, check the progress, if you didn't make any progress so far, upgrade.
