### Post by wolfen69 on 2011-10-11
You essentially bought an expensive paperweight if you plan on using it in linux. Kodak printers are probably the worst ones to buy for linux. You will need to have windows to use it. Personally, I would take it back if possible, and buy an HP. HP is plug and play in linux. I'm afraid you are stuck my friend.

It's good to do some research *before* buying hardware for linux. It will save you some headaches.

### Post by caiobarros on 2012-01-16
I could print a document successfully by using c2esp_23-1 (most recent version as I write) in the url given by paulnewall. It comes in a .deb package. Very easy to install.
After that I needed to find the printer manually in the list that the printer manager offers.
