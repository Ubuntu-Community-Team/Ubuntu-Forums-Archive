### Post by Rainfleet on 2011-10-19
Hello,

For APRS, take a look at xastir. Likely does all you need, and some more. For the Shortwave Soundcard modes, take a look at fldigi. Does most (all?) the modes HRD does. For logging cqrlog is awesome, and is also what I do use.

For regular packet, which I did not do for years, I used in the past LinKT, and TNT. But that was not on Ubuntu back then....

### Post by kd5mkv on 2011-10-21
Kd5mkv
I use Airmail on Ubuntu, you need to install Wine! It runs without too much 
trouble. You need to turn off the spellchecker in the message window and it 
performs as it does on windows. Xastir is the best aprs program for any OS, on Windows Ui-View is not developed as author g4ide (sk), winaprs is out of 
date. Xastir has a binary on debian website xastir-2.0.0 under unstable packages. But it is the stable version, an installing is very easy.
  A FBB program is in the synaptic called LinAMC, but I haven't spent enough
time on it, it is from europe and very little setup instructions. Fldigi can be downloaded from there website and scroll down to a PPA Ubuntu download and receive latest version.
  I also run Mixw on Ubuntu using Wine, a older version 2.09-2.12 run, but
not all the features work. But Mixw CW decoding cannot be beat by an digital suite I have used to date. Steve kd5mkv
