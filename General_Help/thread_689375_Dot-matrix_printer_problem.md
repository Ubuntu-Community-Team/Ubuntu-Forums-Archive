### Post by Fernanddo Saenz on 2008-02-06
I use an Epson LQ-570+ dot-matrix printer with continuous paper, which prints perfectly... but only the first page! Instead of going to the start of the next page, it goes way down the next page.The paper size is correct (Letter). This bahavior occurs printing from any app: Gedit, Mousepadd, OpenOffice, lpr... The printer works fine in Windows. Among the things I have tried, without success:

- Checked the printers's DIP switches: Correct for paper size.
- In the BIOS, went to "Standard Parallel Port" (was EPP, "Enhanced Par. Port")
- Re-checked the PPD file: the Epson's one from Linuxprint.org is the same one that comes with Gutsy, as a diff test shows
- In the Print Manager app I have tried different upper and lower margins, but nope!

This little annoyance is the only thing that keeps Ubuntu from being the OS of my dreams...
