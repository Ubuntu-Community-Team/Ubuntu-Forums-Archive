### Post by danwood76 on 2009-08-26
In K3b start a new video DVD project and add those .vob and .ifo files
It should then allow you to burn it to disk.

### Post by MartinEve on 2009-08-26
In addition to the above, make sure the project type is dvd *video* and put the .vob files in the VIDEO_TS folder.
