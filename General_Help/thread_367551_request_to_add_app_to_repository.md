### Post by Jussi Kukkonen on 2007-02-22
It's not free software, so the chance of getting it to the normal repositories is close to zero. There is no license easily viewable on the website or included with the program -- this means that no-one except First Class has any right to distribute the software, so currently it can't be put in any repository. 

You should contact the company and ask them to contact Canonical and make a redistribution deal through the commercial repository or CNR, whichever is used in the future.
