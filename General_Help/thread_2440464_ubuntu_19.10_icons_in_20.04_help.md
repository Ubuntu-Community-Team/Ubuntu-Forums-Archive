### Post by Dennis N on 2020-04-11
Communitheme is now Yaru theme.
You can download the yaru-19.10.5 .tar.gz file from here:
[https://github.com/ubuntu/yaru/releases](https://github.com/ubuntu/yaru/releases)
Open the archive with archive manager, and from inside the yaru-19.10.5 folder, extract the /icons/Suru folder. You only need this one folder if your interest is the icon set.
Copy it to ~/.local/share/icons (create this folder first)
Use Tweaks to change the icon theme to Suru. It should change immediately.
