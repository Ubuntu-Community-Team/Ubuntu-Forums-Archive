### Post by t-bob on 2014-02-12
Ever since I applied to update to Firefox 27 from 26 in my Ubuntu 13.10 (which I run on a Mac using Parallels Desktop 9), I don't get a right-click menu to pop up when I right click. ### Post by t-bob on 2014-02-12
I closed all of my Facebook tabs and restarted Firefox, and right-clicking started working properly for me. One of the pages I closed was a web forum that uses a Facebook social plugin for streaming and inputting comments via Facebook. Maybe that's what was causing the problem. In any case, this question is resolved for me - for now.
