### Post by MooPi on 2010-10-19
Depends. If you're able to get inside the computer you can reset cmos and delete all preset info on the motherboard to factory default. The only problem with that is you need to document your settings in bios so you can recreate them after reset. It is different from motherboard to motherboard so you'll need to either read the MB manual or contact the manufacture for details. Generally it is just a set of jumper pins and possibly pulling the MB battery to reset.

### Post by psusi on 2010-10-19
If you are talking about the hard drive password and not the bios boot password, then no; that is implemented in the drive itself.  The only way to recover if you forgot the password is to reformat the drive.
