### Post by KISSklas on 2011-12-03
"Can You Run It" might require you to be logged in as root. I have experienced this with Windows 7 as well. Need to launch Firefox as administrator to get it to work.

There is however no guarantee it will work as Linux and Windows are built differently. And the "Can You Run It" app is most likely built with Windows in mind.

If you try to login as root beware that you might be a bit vulnerable to attacks from hacks and viruses.
