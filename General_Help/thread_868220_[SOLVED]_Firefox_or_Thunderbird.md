### Post by silkstone on 2008-07-23
System > Preferences > Preferred Applications, and then make sure you have Firefox and Thunderbird selected as defaults.

### Post by silkstone on 2008-07-23
That is probably because you've installed FF2 to replace FF3 which comes with Hardy. Try selecting what it offers (firefox -new-tab etc) and it should be OK.

You need to specify a default web browser there for links in Thunderbird to know what to open!

### Post by rules on 2008-07-23
I just realised my problem.

For some reason it's still looking for Firefox 3 when I use the firefox command even though it is completely uninstalled. Did a search (on the pc) and found that Firefox 2's command is actually "firefox-2", so I just added the dash 2 and all is cool now :)

Thanks for the help.

It is strange though that it does not show Firefox 2 in the drop down menu in Preferred Applications.
