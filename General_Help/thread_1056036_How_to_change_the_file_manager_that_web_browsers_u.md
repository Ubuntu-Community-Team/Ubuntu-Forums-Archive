### Post by txcrackers on 2009-02-07
That's not Nautilus, that's the GTK File Dialog. They're two different things. If you used a KDE app, you'd see the KDE/Qt File Dialog. Ditto for Java programs. It's a pretty common "widget" found in all of the UI toolkits. And it depends upon what the particular program uses as to what it looks like.
