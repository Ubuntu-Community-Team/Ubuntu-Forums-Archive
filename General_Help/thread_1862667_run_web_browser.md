### Post by An Sanct on 2011-10-17
In older versions (Gnome) it is located under Applications>Internet, under Unity, open the applications menu and enter "opera" in the search dialog.

When Opera is running, click on the icon (in the left panel) and "pin" it to the panel, to make it a "bookmarked" application.
