### Post by burunji on 2008-05-01
You need to figure out whether your computer is capable of running AWN. Your computer needs to be capable of compositing support, which allows for "real" transparency, in that case can your computer run Compiz fusion? 
if so you must install it, Then you can install the AWN-dock

To install AWN-dock for gutsy just go to this link

[http://wiki.awn-project.org/index.php?title=Installation](http://wiki.awn-project.org/index.php?title=Installation)

when you are at this link scroll down the bottom of the page.
you will see where it says ----> Installation Type....make sure you choose

(  Binary package recommend ) ..then it will take you to another page there you choose your distro in your case Gutsy....then follow the instructions.

Hopefully this will help

Good luck
