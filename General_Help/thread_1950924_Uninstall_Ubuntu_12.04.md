### Post by Mark Phelps on 2012-04-02
Sorry ... there is no "downgrade" or "roll-back" feature to uninstall an Ubuntu version upgrade and return to the previously installed version.

All the more reason to AVOID pre-release versions in the future.

If you want to see if a new version works better than your current version, better approach is to make a Live CD/USB of the new version and boot to that.  That will leave your current version intact.

### Post by Mark Phelps on 2012-04-02
> **ZombiePiratez said:**
> So I would just reinstall via wubi? And that wouldn't mess anything up?

Depends on what you don't want "messed up" ...

A Wubi reinstall will completely replace the install you have now.  No files or settings will be retained.

However, it should not affect MS Windows in any way -- but I believe that you will have to REMOVE the current Wubi install before replacing it.  I don't believe that it will let you downgrade to a previous Ubuntu version.
