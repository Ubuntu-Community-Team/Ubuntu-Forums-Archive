### Post by mcduck on 2011-08-12
No, you don't need the server version. All Ubuntu versions are in the end one and the same, the different versions just include different things on the install disc, while all the same things can be installed to any version from Ubuntu's repositories.

You will, of course, need to install a web server for Joomla to work. Some prefer using XAMPP, but I'd recommend just going for Ubuntus own LAMP stack instead.

edit:
Here's a fairly simple guide for installing LAMP on Ubuntu: [http://www.unixmen.com/linux-distributions/4-ubuntu/1239-install-lamp-with-1-command-in-ubuntu-1010-maverick-meerkat]("http://www.unixmen.com/linux-distributions/4-ubuntu/1239-install-lamp-with-1-command-in-ubuntu-1010-maverick-meerkat")
