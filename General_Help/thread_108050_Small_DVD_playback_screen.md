### Post by &#1055;&#1054;&#1055;&#1058;&#1054;&#1053;&#1046; on 2005-12-24
Simplest thing to try is system>preferences>multimedia system settings. Click the "video" tab and experiment with the avilable options for "sink." You'll need to completely close the player (make sure, using the task monitor, there is not still a sleeping or dead version stioll open) and see if selecting one of the other video drawing methods solves it.
