### Post by Jouke74 on 2011-08-29
The linux alternative for Paint Shop Pro is Gimp. Or Paint shop pro 7 and MS word can be run using Crossover linux. This will cost some cash, but both programs will run on Ubuntu afterwards. The free alternative of Crossover is Wine (Crossover is based on Wine). This tool will basically do the same, however it is more difficult to start to comprehend. 

If you are really lucky the Testout labsim might also work with Crossover, but there are no guarantees here. It depends on what the program does and what it requires to run under Windows (Visual basic, Netframework, etc).

For the Lexmark you need to check if it is supported with Xsane scanner software and CUPS printer system.

Note that all of this takes some time to get used to, so take it slow. Read stuff before you try things out, and try to do things when you know what will be the supposed result (just like in a lab).

- For Linux you need Linux program's Windows exe's won't work by default.
- Check out the software repository and the internet for Windows alternatives,
- Linux drivers for hardware from the manufacturers are rare and difficult to install, use the distro drivers as mush as possible (most is installed automatically). 
Check if all hardware works. Including sound, SD card reader etc.  
- Get to learn how to use the terminal.
- Make a frequent backup of your distro and home directory if you start trying things.
