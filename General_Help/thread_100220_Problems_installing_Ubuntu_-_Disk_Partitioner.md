### Post by stuporglue on 2005-12-07
> a packard bell pc with 64Mb ram

There might not be enough RAM for all the processes. I'd recomend doing a server install. Depending on how much disk space you have, you can do a simple "apt-get install ubuntu-desktop" afterwards to install the GUI goodness. 

If you have the option, buy more RAM too. I've used Gnome on 64 Mb of RAM and it was AWEFULL. If you can't get more RAM, I'd do a server install, then install xubuntu-desktop. It's based on XFCE, and is much lighter. 

I use the server/xubuntu on slow computers here [http://stuporglue.org](http://stuporglue.org). They end up alright.
