### Post by aggierandy on 2008-06-11
[SIZE="3"]Hi guys. Well after setting up my hardy laptop to have different wallpapers on the four sides of my desktop cube, I set out to find cool stuff to put on them. After seeing some multiple monitor setups I was impressed and figured that I could do something similar with compiz. I google image searched panoramas and found some I liked. I then resized a bit and cut them into 4 parts using gimpshop and set them as the sides of my cube. 

My creation is attached below (sorry had to lower resolution to attach)



Here are all the parts as posted on flickr. Feel free to download them and use them as you like! They are numbered 1-4 from left to right. Hope ya'll like it!

[http://flickr.com/photos/27544612@N02/](http://flickr.com/photos/27544612@N02/)

[/SIZE]



**P.S.** if you want a similar setup it is easy. Install CCSM via terminal (sudo apt-get install compizconfig-settings-manager). Then System->Preferences->Advanced Desktop Settings. Then set desktop cube to be active and rotate cube. Set the desktop size to 4 horizontally. Setup the desktop cube with the images you want. hopefully these:) then simply tell nautilus not to draw the desktop. In terminal type gconf-editor, go to apps->nautilus->preferences-> uncheck Show Desktop.
**WARNING: You will no longer have desktop icons! However these files will be accessible by going to places->Desktop**
