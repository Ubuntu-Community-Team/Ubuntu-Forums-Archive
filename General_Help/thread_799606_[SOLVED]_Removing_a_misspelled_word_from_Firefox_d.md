### Post by bingoUV on 2008-05-19
Do you see the file persdict.dat at
~/.mozilla/firefox/xxxxxxxx.default ?

Assuming your profile is named default. Instead of the xxxxxxxx above it will be some random characters.

---

### Post by Luffield on 2008-05-19
Excellent, thanks very much!
I forgot to close Firefox before I edited the file, but after I realized that I should do it I edited the file again and all is well now.
