### Post by phillips321 on 2008-01-14
try deleting all the other res options from the xorg.cong:

back up your xorg.conf first

```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf_backup_080114
```

```
Section "Screen"
	Identifier	"Default Screen"
	Device		"ATI Technologies Inc RV350 AR [Radeon 9600]"
	Monitor		"L222W"
	DefaultDepth	24
	SubSection "Display"
		Modes		"1680x1050"
	EndSubSection
EndSection
```

you might want to also try adjusting the vert/horiz vaules, here are mine:
```
Section "Monitor"
    Identifier     "Monitor0"
    VendorName     "Unknown"
    ModelName      "Acer AL2416W"
    HorizSync       24.0 - 80.0
    VertRefresh     49.0 - 75.0
EndSection
```
