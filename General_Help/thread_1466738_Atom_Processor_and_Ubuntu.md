### Post by snowpine on 2010-04-30
I have not personally used the Acer Revo, but from the specs it is basically a "netbook in a box." I have 3 Atom-based computers (2 netbooks and a desktop) and Ubuntu runs just fine on all of them. The Atom is a popular chip that is well supported by the Linux kernel itself. Performance wise, the Atom is in the Pentium 3-Pentium 4 class, not a good choice for serious number crunching, but fine for web surfing, word processing, etc. I see that the Revo has Nvidia Ion graphics, so you will need to install the Nvidia driver for sure (well documented on the Ubuntu wiki). Use the Search feature at the top right of your screen and you'll find several threads on this machine. :)
