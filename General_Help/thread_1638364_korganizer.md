The menu bar in korganizer disappeared for me.  How does a person restore the menu bar in korganizer so it is visible again?

---

### Post by SeijiSensei on 2010-12-05
I haven't had this problem with korganizer, but I have had it with other KDE programs from time to time.  Usually there's an entry in the program's configuration file that you can delete to restore the missing menu or toolbar.  For korganizer, you want to examine ~/.kde/share/config/korganizerrc.  The .kde directory is normally "hidden" because it begins with a dot.  In Dolphin, go to your home directory then select View > Show Hidden Files.  You'll then see the .kde folder appear in the directory listing.
