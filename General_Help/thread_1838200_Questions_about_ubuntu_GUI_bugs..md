### Post by sanderd17 on 2011-09-03
I hope this annoyances aren't too bad so you can live another month with it?

Unity wasn't ready to be released in 11.04, and this is what you get. 11.10 will be a lot better.

For the first, you can do ALT+F2 and execute "killall gnome-panel" I suppose.

For the second, they introduced zero-pixel borders in 11.04. That means you see no pixels, but there is a border of (I believe) 5 pixels where you can grab it. Maybe you altered a setting. Try to go back to the default theme and see if it changes.

I don't have any experience with the third.
