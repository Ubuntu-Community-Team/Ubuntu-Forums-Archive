### Post by coffeecat on 2009-12-28
What's the complete model number of the Vaio, and how old is it? Also - which version of Ubuntu did you try? If it was pre-9.10 you might want to try 9.10. A newer version of Ubuntu means a newer version of the kernel which means newer drivers and drivers for hardware that wasn't supported in earlier versions. You do get some regressions for older hardware - plenty of threads here to prove that - but generally it's a good idea to try the latest version of Ubuntu if some hardware isn't working.

Example - on my Sony Vaio VGN-NS20S, bought last Spring, just about everything works in both Jaunty (9.04) and Karmic (9.10). I haven't tried Hardy (8.04) but I wouldn't be surprised if a lot of things (wireless, webcam) wouldn't work - hardware too recent.

By the way, remember that you can run Ubuntu live from the CD and test the hardware without committing yourself to an install and without making any changes to the hard disc.
