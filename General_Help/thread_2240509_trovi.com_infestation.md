### Post by Frogs Hair on 2014-08-20
Deleting the Chromium profile folder in home/hidden folder should do the trick but it will remove bookmarks and extensions also. This folder isn't removed when reinstalling and that is why Trovi comes back. Open the flle manager and press Ctrl + H to display hidden folders and I think the Chromium folder is inside the .config folder. When the browser is opened a new profile will be created. If using Google browser sync make sure that Trovi is not part of that data. Trovi is a spyware redirect rather than a virus and is included or bundled with software such as down-loaders.
