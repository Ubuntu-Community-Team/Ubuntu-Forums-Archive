### Post by Morkhaar on 2008-11-13
OK! solved it! 

Just in case someone has the same problem you need to go to:
Tools -> Options -> OpenOffice.org Writer -> View 
and uncheck 'Field Codes' in Display

Somehow all this was not painfully obvious.... :)

thanks for the help!
