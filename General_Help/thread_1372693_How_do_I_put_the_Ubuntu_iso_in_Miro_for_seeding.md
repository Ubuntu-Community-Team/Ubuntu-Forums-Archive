### Post by Mr. Picklesworth on 2010-01-04
Miro is for media, not generic data files. Sure, you could trick it into downloading / seeding an Ubuntu ISO, but the software is not designed for such things (it is, for that matter, intentionally pointed away from them). You would get a far nicer experience from an application like Transmission.

### Post by lovinglinux on 2010-01-04
Go to [http://www.ubuntu.com/getubuntu/downloadmirrors#bt](http://www.ubuntu.com/getubuntu/downloadmirrors#bt) and get the torrent for the ISO you have. Open it with Miro and start the download. Then pause the download, close Miro and copy the complete ISO to the folder where Miro saves the downloaded torrents. Start Miro again. I don't know much about Miro, but any decent BitTorrent client should detect that the file is now completed and start seeding it.
