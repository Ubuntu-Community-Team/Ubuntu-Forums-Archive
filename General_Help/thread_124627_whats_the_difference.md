### Post by GeneralZod on 2006-02-03
It's also worth noting that Kubuntu is a rather less mature product that Ubuntu, and has several fairly major bugs in it (the default System Settings is unusable if your resolution is below a certain size; the Networking utility has  broken "static IP" functionality; you'll need to do updates in order to enter Administrator mode in System Settings; etc).

### Post by bulldogzerofive on 2006-02-03
[QUOTE=erick_p]Questions: 

1. Is Kubuntu *exactly* the same as Ubuntu except for the desktop used?

2. If not, then how can I ask Ubuntu to use KDE (because I did see an install option?) instead of Gnome?[/QUOTE]


Use synaptic to install kubuntu-desktop.  It will download all the packages and install them by itself.  Needless to say, you need to be connected to the internet.

Then log out.  At the log in screen, click on the "session" icon and select KDE.  Then log in normally.  Then you will be in Kubuntu/KDE.

And I must add, I know GNU/Linux leaves a kind of bad taste in your mouth at first.  This is most likely because you are learning a new operating system that does things differently than windows, but since you already know how to use windows, you are going to blame this learning curve on Ubuntu.  

Trust me, it is worth it to stick it out and learn.
