### Post by TheHimself on 2009-08-29
When calling a GUI application from the command line how can one make it to come to the front (i.e. above other windows)?

---

### Post by DaithiF on 2009-08-29
wmctrl is a useful tool for window manipulation -- it can bring windows to the front, move them, size them, etc.
```
sudo apt-get install wmctrl
```
