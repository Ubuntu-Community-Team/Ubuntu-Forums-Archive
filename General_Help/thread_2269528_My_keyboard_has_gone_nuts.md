### Post by grahammechanical on 2015-03-16
Is it not your keyboard layout? It sounds like an English (US) layout instead of an English (UK) layout. When we install Ubuntu 14.04 and select a variant of English (UK) we get two keyboard layouts - English (UK) and English (US). We should be able to switch between them by clicking on the keyboard app-indicator in the top panel. To add and remove keyboard layouts select Text Entry Settings. See what Input Source is at the top of the list. Whatever is there is the default.

Regards.
