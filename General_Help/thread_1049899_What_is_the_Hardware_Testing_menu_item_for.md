### Post by mb_webguy on 2009-01-25
It's to make sure that your hardware is compatible with Ubuntu (which is useful if you're running the LiveCD before installation) and to help the Ubuntu team improve Ubuntu.  Here's the description from the [package page]("http://packages.ubuntu.com/hardy/hwtest-gtk"):  "This project provides an interface for gathering hardware details and prompting the user for tests. This information can then be sent to Launchpad."
