### Post by Vectorferret on 2010-06-09
If any other program is playing audio when the page with flash opens, then Flash has no audio. The browser crashes as soon as I try to navigate to another page. If Flash is playing when another program starts playing audio, that program plays no audio and crashes. I have tried both the standard flash and 64-bit flash from a ppa. The problem occurs with both versions. Sound card is ASUS Xonar D2X. I do not remember making any changes to configuration files for ALSA or pulseaudio. Thank you for any help you can give.

### Post by Vectorferret on 2010-06-10
Solved this with some pulseaudio wiki and other threads.

```
gedit ~/.asoundrc
```
Add the following lines
```

pcm.pulse {
    type pulse
}
ctl.pulse {
    type pulse
}
pcm.!default {
    type pulse
}
ctl.!default {
    type pulse
}

```

Make sure asound2 is installed (it should be by default afaik, but Pulse wiki said it needed to be so indicating here).
