### Post by matt_symes on 2011-06-08
Hi

Open a terminal and type

```
eject
```

If that does not work type

```
sudo eject
```

This is assuming you only have one CD/DVD player. 

If you have more than one then you will need to specify which one.

Do either of them work ?
