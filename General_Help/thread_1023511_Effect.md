### Post by lykwydchykyn on 2008-12-28
I'm assuming you have desktop effects on, and they work fine on your system.

Install the compizconfig settings manager via synaptic, if you haven't already.

Run the settings manager (command is ccsm).  

You need to enable "animations add-on", then set up your close animation to use "burn".
