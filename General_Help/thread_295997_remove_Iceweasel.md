### Post by kc8hr on 2006-11-09
If you installed Iceweasel as a .deb file, then open up a terminal window and type:

sudo apt-get remove iceweasel

hit 'enter' and it will be gone.

### Post by Sef on 2006-11-09
How did you install it?

If you installed it with sudo apt-get then uninstall sudo apt-get in removing it.  If you installed it with sudo aptitude then uninstall it with sudo aptitude.
