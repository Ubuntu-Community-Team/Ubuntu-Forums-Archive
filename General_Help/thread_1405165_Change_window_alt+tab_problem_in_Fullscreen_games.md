### Post by Techsnap on 2010-02-12
Are you using WINE for this? It's most likely that it won't be particularly stable through WINE, best advice I can give on this.

Make sure your games resolution matches your desktop resolution. If it does, then simply don't alt-tab out of the game, you're running it on an OS it's not designed to be used on to start with so it's not going to be flawless.

If you must have other programs running whilst you play, I suggest you set WINE to run it in a WINE Desktop, then put that on another Virtual Desktop and have it match the resolution of your monitor as close as you can.
