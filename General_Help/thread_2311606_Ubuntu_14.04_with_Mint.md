### Post by Dennis N on 2016-01-28
That is the Linux Mint Debian Edition. I comes with either MATE desktop or Cinnamon Desktop.

[http://www.linuxmint.com/download_lmde.php](http://www.linuxmint.com/download_lmde.php)

so, sticking to the official Ubuntu releases, I would look at Ubuntu MATE.

(if it's the menu you like, you don't get that same menu with Ubuntu MATE - looking at it closely, I am pretty sure that is the Cinnamon Desktop)

EDIT: You can install the Cinnamon Desktop Environment on Ubuntu - Google can point out some advice for doing so. One fairly recent:
[http://ubuntuhandbook.org/index.php/2015/05/install-cinnamon-2-6-in-ubuntu-15-0414-04/](http://ubuntuhandbook.org/index.php/2015/05/install-cinnamon-2-6-in-ubuntu-15-0414-04/)
