### Post by james@finnall.net on 2012-09-14
Hello all,
I have recently upgraded Firefox to 15.0 and now to 15.0.1 through the  normal uBuntu channel updates.  I am having problems displaying web  pages correctly.  When I scroll the screen up and down, random bars  appear on the page.  If I scroll it up/down again they may disappear or  at least appear somewhere else so I can read the text.  Always in the text, never in graphic images or white background.

Both machines are using Intel integrated graphics on Intel DG31PR motherboards.

The problem does not appear in any other application beside Firefox. It does not appear on all web pages, only some pages like news sites.  Possibly a font type of issue or a metrics problem using the fonts.  The fonts usually have a Times Roman type of appearance rather than the Courier format.  For instance, in searching the forums here I have not had the problem.  But other pages are currently displaying the problem.  If I zoom in to the web page and increase the size of the fonts then it will practically disappear until I zoom back to normal or zoom out to make them smaller.

Any suggestions on different settings to try?  Did not have this problem prior to FFv15.  Thought next update would resolve problem, but 15.0 and 15.0.1 remain the same.

Thank you for your consideration.
James

UPDATE:  It appears that disabling "Use smooth scrolling" in Edit->Preferences->Advanced->General will at least help, if not resolve, the issue.
James
