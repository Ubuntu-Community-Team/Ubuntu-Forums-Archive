### Post by eternalnewbee on 2008-11-24
Hi ragonz.
Say for example you've found a theme in [www.gnomelook.org](www.gnomelook.org) that you would like to use, so you download it to your desktop. What I have done, is create a new folder in my Home Directory, called **emerald themes**. After having finished downloading the theme, I right-click on it on the desktop, cut & paste in the **emerald folder**.
Then I open **Emerald Theme Manager**, and click on **Import**, and navigate to the emerald folder, and double click on the theme. Then select it in **the Emerald Theme Manager**.
Hope this helps you.

```

```

in the command prompt?  If not Compiz could still be doing the themes (make sure to set this up in Sessions if you want it on start up)
