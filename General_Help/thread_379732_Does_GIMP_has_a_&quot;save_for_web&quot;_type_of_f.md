### Post by DagMan on 2007-03-08
When you save a jpg there's a slider for quality.  It will effect the filesize.  If you've seen this in irfanview then you have an idea.. it's similar in the gimp.  I'm not all that knowing about other file types to answer any further apart from saying that you can reduce the number of colours used in a gif and it will effect the filesize.  You can reduce the number of colours in a lot of filetypes but I don't know what it does to their size.  One would think that many of them would get smaller.  There may be other ways to reduce size in non jpg formats as well but I'm very experienced with it.
