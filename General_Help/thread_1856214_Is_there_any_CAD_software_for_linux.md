### Post by Gemnoc on 2011-11-11
Hi,

Better late than never...

From the list in the link posted by Yann, here's my take:

For 2D:
Open source --> **LibreCAD** (QCad fork modernized to support Qt4, for the moment it's using Qt3 to Qt4 transitional libraries, it's got one tool QCad doesn't: polyline) available in Ubuntu repositories starting from Ubuntu 11.04, can be installed from a PPA on Lucid and Maverick [https://launchpad.net/~showard314/+archive/librecad](https://launchpad.net/~showard314/+archive/librecad)

Free for personal use/commercial --> **DraftSight**, really nice AutoCAD clone, provided you can stomach the EULA (FOSS aficionados need not apply)

For 3D:
**FreeCAD** is the most featured and promising open source parametric modeller at the moment, but it is not yet in the same league (and not by a very long shot) as SolidWorks and the like. Many features are missing, but it may be enough for your needs. It can be installed from Ubuntu repositories, but that version is outdated. The project has two PPAs on Launchpad supplying packages built from the development branch. *(Disclaimer: I participated in the creation of these PPAs and I am a member of the FreeCAD Maintainers team, but I am not a developer.)*

**HeeksCAD** (open source as well) is interesting too, although its scope is not as ambitious, it's geared more toward CAM. You'll need to compile it though. It's got some kind of "direct editing" that's cool to use.
[https://github.com/Heeks/heekscad](https://github.com/Heeks/heekscad)
