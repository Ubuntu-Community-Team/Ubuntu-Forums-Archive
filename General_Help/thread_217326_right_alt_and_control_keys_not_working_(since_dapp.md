### Post by nanotube on 2006-07-17
ok, problem has been solved using the following:
System>preferences>keyboard, go to the "layout options" tab, select "alt/win key behavior" and check the "alt and meta are on the alt keys (default) button. i guess that reset whatever problems it had (because it is listed as default anyway, so should have been there...)

well, so much for that...
