### Post by NIT006.5 on 2008-05-11
If I remember correctly, the default to zoom out is Ctrl+Alt+Left mouse button - but this only works when you click and hold the mouse on the wallpaper... not on any open windows. Moving the mouse while holding down the button will then rotate the cube. Yes, the Ctrl+Alt+Down is a different effect that "unfolds" the cube and lays it out like a wall. By default Ctrl+Alt+Left/Right will rotate the cube, but you must have over over-ridden that shortcut key when it was set for the paint fire effect. You should be able to reassign those shortcut keys though.
