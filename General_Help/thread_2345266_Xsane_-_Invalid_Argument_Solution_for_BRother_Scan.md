### Post by vidtek on 2016-12-02
If you get this error message after xsane finds the scanner, the preview picture size may be less than 64x64 pixels.  This is Brother support's answer:

When scanning I receive the error "Failed to start scanner : Invalid argument".


When the scanning area is small, you receive this error.The scan size should be more than 64*64 pixels.
