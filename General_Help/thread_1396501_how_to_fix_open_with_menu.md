### Post by Rodemire on 2010-02-02
Hi, 
Just to make sure I have understood your problem, you want to make a certain application your default application to open Pdfs?

If so, right click on a PDF file and go to Properties. On the popup that appears go to the tab with  "Open With". Under that tab are applications on your system that you can use to open PDFs. Choose the one you like and "Close" the window that appeared. From this point onwards you can then just open a PDF by double clicking on it and the application you chose will open the PDF for you. I use Document Viewer, simple and caters for my current needs. (If I'm correct, it comes by default with the installation disc.)

If you do the above you won't need to deal with the "Open with" menu anymore.

If I have misdiagnosed you please just add a little bit of details so you can get some better help.
