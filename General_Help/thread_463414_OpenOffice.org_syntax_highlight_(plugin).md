### Post by axt on 2008-02-12
Try copying from eclipse. 

If it does not work, try highligt. 
This is a program exact for your purpose, i think. Dowload it from 

[http://www.andre-simon.de/](http://www.andre-simon.de/)

There are installers for most spread operating systems.

So you put you code in this prog and save it to *.rtf. You can also save the code with line numbers. 
If you do not like *.rtf, copy your code from *.rtf to an *.odt file or to *.doc.

The program is free.

If you understand German (use a free online translator if you do not), you can also read 
[http://www.linux-user.de/ausgabe/2005/08/064-highlight/index.html](http://www.linux-user.de/ausgabe/2005/08/064-highlight/index.html)
about this prog.
