### Post by cong06 on 2010-02-27
Sounds like you want to learn how to use commands.

Probably you should start off by learning man pages:

```

man {command}

```
Often if you press "tab" you can have the command line help you figure out what commands are available.

For example, if you start typing "gno" and press tab, it'll probably help finsh: "gnome"

Start commands:
cp - copy
ls - list
cd - change directory

just use man to figure out how to use them.

Go at it! We're here to help.
