### Post by howefield on 2017-02-26
Just for info, there is also a snap package for 5.3 available in the Ubuntu repositories.

```
snap find libreoffice
Name         Version  Developer  Notes  Summary
libreoffice  5.3.0.3  canonical  -      LibreOffice is a powerful office suite including word processing and creation of spreadsheets,
slideshows and databases
``` 

```
sudo snap install libreoffice
```
