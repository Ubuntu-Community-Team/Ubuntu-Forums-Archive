### Post by Sutekh on 2006-04-06
While I highly enjoy using Kile, I should point out that *any* text editor can be used to create LaTeX documents.

So you can use gedit, nano, pico, vi, the mighty emacs.

Also check out [Lyx](http://www.lyx.org/)



This is a great document for learning the LaTeX basics.

[The Not So Short Introduction to LaTeX](www.ctan.org/tex-archive/info/lshort/english/lshort.pdf) (PDF)
