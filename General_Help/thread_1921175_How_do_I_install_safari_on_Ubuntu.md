### Post by 3Miro on 2012-02-15
> **pnguine said:**
> I just tried ctrl-n and all I got was a new tab. If I go to another desktop and click the try to start firefox it will just show me the window and tabs that are already running.

File -> New Window (the File menu will appear if you drag the mouse over the top panel towards the left corner). The default shortcut should be Ctr+N, while Ctr+T opens a tab.

You can also open a new window if you drag a tab out of its position (say down) and it will open a new window.

You can also open a new window if you middle-click on the Firefox shortcut on the Unity Dock-bar.
