### Post by nick_h on 2008-04-07
For the repeat keys, open System->Preferences->Keyboard, select the keyboard tab and then check that the "Key presses repeat when key is held down" option is selected.

I'm not sure about your window buttons (close, minimize, maximize) - as you say this is a common problem with Visual Effects.
