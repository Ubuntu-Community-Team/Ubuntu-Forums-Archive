### Post by mcduck on 2012-04-16
Compiz allows you to remove borders from a window, make it sticky so it appears on all desktops, and keep it below other windows. And if you want make it uncloseable, unminimizeable and hidden from any tasklist/pager application.

So basically all you need is any terminal application that can have a transparent background, and a few rules in CompizConfig Settings Manager. :)
