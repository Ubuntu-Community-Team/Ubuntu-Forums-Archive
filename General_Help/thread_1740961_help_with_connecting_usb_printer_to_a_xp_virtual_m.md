### Post by cek on 2011-04-27
Have you setup a USB filter to allow the device to pass through to the virtual machine?  With your printer plugged in to your USB port (and powered on), go into the virtual machine configuration (click on settings, then on USB), and you should be able to setup the printer to simply pass through the host os.
