### Post by hysteresis on 2011-02-07
You only need one video card with 2 outputs. I use a msi vn210 card (nvidia 
geforce 210) in ubuntu 10.10 . I set up 2 displays in "Nvidia X Server Settings" and use XBMC on my TV to watch videos. I had a few problems with HDMI audio, but this thread (Neon Dusk's reply) fixed it for me;
[http://ubuntuforums.org/showthread.php?t=1641771&highlight=multiple+hdmi+audio+devices](http://ubuntuforums.org/showthread.php?t=1641771&highlight=multiple+hdmi+audio+devices)

Good luck!
