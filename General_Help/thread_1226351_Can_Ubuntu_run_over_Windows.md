### Post by 3Miro on 2009-07-29
Wubi lets you install Ubuntu from within windows so that if you don't like it, you can just remove it from window's add/remove programs.

You can install Ubuntu as dual-boot, i.e. split the hard drive into several partition, give some to windows and some to Linux and then on boot you get to chose windows or Linux.

If you don't care about the windows installation you ca just tell Ubuntu to install "over it". You will lose windows and EVERYTHING that is on the harddrive and then install only Linux.

Advice: before you do anything, backup all important stuff from your computer.

### Post by lisati on 2009-07-29
Hi there BracGypsy.

You don't actually *need* Windows in your system to be able to use Ubuntu.

As you might have gathered from what others have already said, you have a 
number of options for checking out and, if you want, installing Ubuntu. These include:

[LIST=1]
[*]Running from the "Live CD". Most Ubuntu CDs come with an option to let you try Ubuntu without actually installing anything. It might not run as fast as a copy you've actually installed, but it can help give you an idea if there are any issues with your hardware that might need to be addressed. It also gives you a chance to discover if you like Ubuntu.
[*]"Wubi", also known as "within Windows", lets you install Ubuntu in a way that allows you to remove it using the Windows Add/Remove Programs option on the control pannel
[*]Full install in its own partition. Here there are options. The main ones are where you can either install Ubuntu "side-by-side" with Windows (known as "dual boot") or let it use the whole disk (wiping Windows completely)
[/LIST]
Feel free to continue asking questions.
