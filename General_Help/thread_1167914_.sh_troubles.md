### Post by taurus on 2009-05-23
What's the output of this command from a terminal?

```
ldd /usr/bin/bin/glc-play
```
Looks like the libraries for that program are installed to /usr/bin/lib.  Therefore, you need to edit /etc/ld.so.conf

```
gksudo gedit /etc/ld.so.conf
```
and add this line the the end of it.

```
/usr/bin/lib
```
Save it and then run

```
sudo ldconfig
```
Now, start up your program again and see what happens.

```
/usr/bin/bin/glc-play
```
