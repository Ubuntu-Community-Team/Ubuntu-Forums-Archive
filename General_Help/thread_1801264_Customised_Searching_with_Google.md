### Post by SteveDee on 2011-07-10
Finding answers to your questions can be difficult. The Ubuntu forum is great, but searching for information within it is tricky, and the best answers may often be found on other sites.

I'm not a huge fan of Google, especially when looking for specific technical information. However, here are 2 ways to use Google to best advantage.

1. Use Googl Ubuntu ([http://www.googlubuntu.com/](http://www.googlubuntu.com/))
This searches ubuntu.com for relevant information, and allows you to filter by a time period (e.g. past week, past month & so on).

2. Create a customised search.
Most ISPs provide users with some web space. So the idea is to first create a simple HTML web page with links to any sites that may contain relevant information.

For example, I have a "Linux" page which not only contains links to Ubuntu sites like Ubuntu, Ubuntu Geek & Psychocats, but also to other Linux sites like LinuxCommand, TuxFiles & so on.

Just create a simple web page with lots of links to relevant sites, and then paste some code into it from this site: [http://www.google.co.uk/cse/tools/create_onthefly](http://www.google.co.uk/cse/tools/create_onthefly)

As soon as you publish this page to your webspace, you can start using the custom search box, which will search all the sites listed on your page. How useful is that?
