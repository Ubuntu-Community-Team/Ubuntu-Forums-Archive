### Post by dentman on 2009-05-04
Same here.
 I removed with Add/remove then downloaded 4.2 from Sourceforge. Upon opening the Vuze_4.2.0.2_linux.tar.bz2 and extracting it I find there is no "installer". So reading the Readme yields 
RUNNING:
1. Extract the contents of this .tar.bz2 file.
2. Change to the 'azureus' directory where the files were extracted.
3. Start Azureus by running the script named 'azureus'; ex. "./azureus"
 aka double click the file azureus.
 This launched the much nicer looking 4.2.

 Now how to stash this folder and where. How to add a launcher to the Applications list.

Peace
 Rick
