### Post by drs305 on 2009-11-17
The default screenshot utility (gnome-screenshot) includes an option to delay the snapshot. So you can set the delay to 5-10 seconds and then go back and reselect Applications, Accessories and get the mouse pointer where you want before the snapshot is taken.

### Post by aysiu on 2009-11-17
**Step 1**
Press Alt-F2 to get the Run dialogue up

**Step 2**
In the dialogue, paste in the command ```
gnome-screenshot --delay 10
```

**Step 3**
Open the menu you want to take a screenshot of and hold it open for 10 seconds
