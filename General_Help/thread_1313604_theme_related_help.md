### Post by Brandon Williams on 2009-11-03
9.10 doesn't have shiki-mint, but it does have shiki-wise, which is also the shiki base theme with green highlighting that looks a lot like the linux mint screenshots. It's in the standard repos. Just install shiki-wise-theme.
