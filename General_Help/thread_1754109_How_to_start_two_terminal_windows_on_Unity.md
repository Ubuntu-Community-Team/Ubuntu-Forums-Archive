### Post by coffeecat on 2011-05-09
If you don't have a middle mouse button, for those applications that allow you to have a second or subsequent window open, move your mouse to the global menu and choose File > New or New Window or whatever shows in the File menu for opening a new window. For gnome-terminal it's File > Open Terminal.

Do you have a scroll-wheel by chance? Usually you can depress that to get the middle-button click.

### Post by Maheriano on 2011-05-09
On the top toolbar of the desktop is the menu for whichever application you're using. Under File, choose New Window. If you upgraded, the menu font probably blends into the brown background but they're there. On a fresh install it's easy to see.
