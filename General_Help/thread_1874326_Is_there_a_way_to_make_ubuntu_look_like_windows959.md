### Post by 3rdalbum on 2011-11-02
Ubuntu 11.10 only accepts GTK 3 themes, and as far as I know nobody has made a GTK 3 theme to look like ye olde Windows.

If you're using 11.04 or earlier, I'm sure there are Windows 95/98 themes - but they'll only change the window borders and/or window controls, they won't actually make a taskbar or Start Menu or anything like that. The earlier poster was wrong - you need to look for GTK 2.x and Metacity themes, **not** GDM themes.
