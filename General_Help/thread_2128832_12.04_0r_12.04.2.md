### Post by deadflowr on 2013-03-24
If the system is up-to-date then you should be running 12.04.2.
Run this in a terminal:
```
 cat /etc/lsb-release
```

And look at the DISTRIB_DESCRIPTION line. It''ll say which 12.04 you have.

### Post by steeldriver on 2013-03-24
Mine shows the same - I suspect the GUI system settings dialog uses the DISTRIB_RELEASE rather than the full DISTRIB_DESCRIPTION

You can see the full details via the command line

```
$ cat /etc/lsb-release 
DISTRIB_ID=Ubuntu
DISTRIB_RELEASE=12.04
DISTRIB_CODENAME=precise
DISTRIB_DESCRIPTION="Ubuntu 12.04.2 LTS"
```
