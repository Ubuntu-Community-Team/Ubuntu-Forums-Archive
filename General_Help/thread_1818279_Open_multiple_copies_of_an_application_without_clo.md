I'm searching for a way, using the Gnome GUI menu, to open multiple copies of an application at once.  This might seem a little unclear so I'm going to state an example:

I used Windows for a long time before getting hooked on Ubuntu (and other forms of GNU/Linux.).  There is a little-known trick, in XP and also in 2000, where you can open multiple copies of the same program without the Start menu closing immediately.  If you go to [Start, Programs, Accessories] and then SHIFT-click (instead of just click) on something, for example the calculator, it will open but keep the Start menu up, and you can immediately SHIFT-click on calculator again as many times as you want for as many copies as you want open.  This can be handy when you want multiple command prompts, or multiple notepads, or whatever.

Now that I'm exclusively using Ubuntu, there are many times when I want to use 3 or 4 calculators at the same time, or a couple of command prompts.  At the moment, I just pin one to the panel and hit it a few times.  That's fine, but it means I have to do it for any application that I want to do this with, ahead of time.  I would like to know if anyone knows how to accomplish this same sort of behavior as above, but with Gnome - is there a way to click on something but not lose the menu?  At the moment, I'm still using Gnome, not KDE and so far, not Unity, but if you happen to know the trick for those as well, hey that's great too.  I've searched around here and the web, but I can't seem to find it.  Maybe it doesn't exist, but it's worth asking the question.
