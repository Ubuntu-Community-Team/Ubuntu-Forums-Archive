### Post by JKyleOKC on 2011-03-18
Open a terminal and type "**which 'Alarm Clock'**" at the prompt. This will tell you where it is, if the program name really is two words and capitalized -- which would be very unusual for a Linux program! I would expect the name to be just one word, and all lower case.

When you find out where it is installed, you can right-click anywhere on your desktop, select "create a launcher" from the pop-up menu, and fill in the form. This should create an icon on your desktop that will launch the program.

Hope this helps!

### Post by randlieb on 2011-07-24
I think I have it fixed. I logged back into classic (gnome) desktop, opened the alarm clock, clicked on preferences and UNCHECKED both the following options:

1. start minimized
2. start the program automatically with gnome

logged back in to unity desktop, clicked on the alarm clock icon and, voila!...it opened!!

Hope that works for you also.
