### Post by Impavidus on 2016-10-09
Windows doesn't understand Linux partitions, so expect it to make a mess of your Ubuntu partition when you try to expand it. Use Windows tools to change Windows partitions, use Linux tools to change Linux partitions. I suggest you create a new live usb (or dvd, if you prefer). It's a great tool to have in case you break your system.
