The last LTS was Dapper, it was actually the first LTS or Long term support. The support is good until 2009. Dapper is a good version and very stable. I have it on my server and dont plan on replacing it until at least a month after the next lts is released as Dapper on the server is good until 2011.
But Feisty is a good version to. I only upgraded from dapper to Feisty 2 months ago on the desktop because I upgraded my ram and video card and since I had to redo the ATI drivers I figured it was time.
I think the next lts will be Gutsy +1 , or the release after Gutsy. If I were you I would either install Feisty, or wait until Gutsy is released and use it. I dont ever recommend using pre release versions and things break during development. While it may not be a problem, do you really want to fight with a broken system if you dont have to?
If you are going to do a fresh install, use the newest version. Dont install and then upgrade the version.
