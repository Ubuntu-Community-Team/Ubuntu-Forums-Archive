### Post by KeyserSoze93 on 2010-04-11
jquery is simply a Javascript library. You download jquery.js from the main site, add it in the folder with your page, and add the following line in your <head>:

```

<script type="text/javascript" src="jquery.js">

```

Then, you can use any jquery code in your page, so it is Javascript/HTML for all intents and purposes.
