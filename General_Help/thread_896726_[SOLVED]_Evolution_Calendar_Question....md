### Post by gobbles414 on 2008-08-21
I need a bit of clarification, sorry. Does the appointment occur on the same days every week -- e.g. every Monday, every Wednesday, and every Friday? Or, is it truly an every other day pattern ignoring weekends -- e.g. [week 1] Monday, Wednesday, Friday, [week 2] Tuesday, Thursday, [week 3] Monday, Wednesday Friday...?

If the same days every week, I recommend that you simply create an appointment for each day, and set it to reoccur on a weekly basis. That is, you'd create an appointment to occur every Monday, another appointment to occur every Wednesday, and the final appointment of the week to reoccur every Friday.

EDITED: If you need a truly every other day pattern ignoring weekends, you'll need to create five separate appointments. In week 1, create an appointment for Monday, one for Wednesday, and another for Friday. In week 2, create an appointment for Tuesday and another for Thursday. This next part is critical... **Set all five of these appointments to occur every 2 weeks.** That's it!
