### Post by aysiu on 2006-11-12
Windows programs are designed to work in Windows, not in Ubuntu.

There are some helper programs that can run native Windows programs in Ubuntu, but they don't work with every program, and they don't always work well. [Wine](http://www.winehq.com/) will do most basic Windows programs. [Crossover Office](http://www.codeweavers.com/) will do a lot of Windows programs that require Windows 2000 or XP. And [Cedega](http://www.transgaming.com/index.php?module=ContentExpress&func=display&ceid=2&meid=-1) will do a lot of Windows games.

You're always best finding native Linux programs, though.
