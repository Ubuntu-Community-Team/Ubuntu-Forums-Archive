NIS      
By... MathieuVilaplana     mathieu(AT)creationgif(DOT)com

NIS (Nautilus Image Script) is a nautilus script, that can resize a selection of images. Can easily resize and compress images shots with my camera to export it on my web gallery for example.
Since it is only a script it's really easy to change to your needs, and really usefull if you have a digital camera imho.
you can download it @ [http://g-scripts.sourceforge.net](http://g-scripts.sourceforge.net)

### Post by kabus on 2006-01-15
Imagemagick's covert function is great for that.
Since it is a command line app it can be easily automated to process lots of images.

Imagemagick is available from the repositories and here's some online docs :

[http://www.imagemagick.org/script/convert.php](http://www.imagemagick.org/script/convert.php)
