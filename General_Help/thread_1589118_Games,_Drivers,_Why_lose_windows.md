### Post by ridgeland on 2010-10-06
I suggest dual boot and keep Windows games on Windows until you learn more about Linux.  First step is learning about partitions.  You can do more than dual boot.  You can have many partitions with Windows on one, Ubuntu on one, Fedora on one etc and choose when you boot.
I've set up Wine for MS-Office 2007 for a friend, works great.  I use VirtualBox with WindowsXP as a virtual machine for my old Windows software (Delorme topo maps).  Both are options but I think you'll be happier learning Linux for Linux and keeping Windows on Windows for the first six months or so.

### Post by mastablasta on 2010-10-06
As it was said before the best way is to cut disk in two and put linux on one part & keep windows on the other. the wubi (install through windows) way is easy, but it can cause issues that you won't see in propper dual boot.
 
With propper dual boot there is no system slowdown. it is easy to do. also Linux doesn't need so much space for new programmes and it doesn't "grow" as much as windows, so 20 or 25 GB for Linux is actually a lot.
 
so if you have some spare disk make a 20GB partition for linux and install it there. 
 
For some info on how to do it check the nice manual with pictures (page 11 onwards; in my signature) or this informative site: [http://www.psychocats.net/ubuntu/](http://www.psychocats.net/ubuntu/)
