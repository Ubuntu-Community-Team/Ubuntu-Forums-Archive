### Post by NilsE on 2008-01-07
This is only a guess but I don't think it will speed anything up since the fonts are directly called by what is set in various config files not searched through.  

I also believe that you need to be careful when you remove fonts since they are also critical to web browser operations.  The base install of fonts is included to address not just the OS user but also what is typical for web sites around the world.  When a web site is hit that has a different font/language it looks to see if you have it then defaults to something else if you don't. Agreed that this would only impact sites that are foreign to you so you could potentially remove some but it may just not help anything but disk space.

This is just my opinion, and only my opinion but it just does not seem worth messing with.
