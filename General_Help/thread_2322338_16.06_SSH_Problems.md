### Post by steeldriver on 2016-04-27
AFAIK openssh-server isn't installed by default (and your netstat seems to confirm it isn't running) - you can confirm that from the command line

```

dpkg -l | grep openssh

```

You will need to find a way to install the package - easiest way will be by connecting to the internet, although it may be possible from the install media
