### Post by technophreak2000 on 2011-08-15
To any Ubuntu 11.04 users out there, if you would like to install gnomenu on your system without having to wait for the official ppa to be released, then visit this site [https://launchpad.net/gnomenu](https://launchpad.net/gnomenu) and download the tar.gz. After you have downloaded it, extract it and a simple "sudo make install" should do the trick on getting it installed.

It works fine for me in my Ubuntu 11.04 64-bit. I have been able to install themes with no problem yet. I'm using gnomenu version 2.9

Please be aware that this is not an official release for Ubuntu 11.04, so you may encounter bugs. This is only for people who can't wait for the official release to be released.:P Do this at your own risk!

So far for me, it's working fine.

If you want something similar to gnomenu, then you may want to Check out Cardapio. However, personally I like gnomenu better.
