### Post by tbraun on 2011-06-01
Hello!

I'm using Ubuntu 9.10 and recently upgraded to Skype 2.2.0.25.
Sound quality is good and that all works, I'm happy to report.
However, ever since I upgraded Skype versions, some of the
notification events don't produce a sound. For example, incoming
calls produce a popup, but no sound. Same with outgoing or
incoming chat messages.

Things like the ringing sound for an outgoing call work, but not
the ring for an incoming one.

And, yes, the sound files are there and accessible and the events
have been enabled. I even tried to set the outgoing ringing sound
as the one for the incoming call, but nothing to be heard. So,
apparently, it is not the sound file that causes the problem.

Does anyone have an idea what's causing this? Am I the only one
who experiences this?
