### Post by short4lif2 on 2006-04-15
For a long time I have been interested in customizing KDE such that the application menus appear at the top of my screen, like Macintosh, and when I switch applications, the top menu changes accordingly.  I have found many threads mentioning this feature, and I found the following article on KDElook.org.

[http://www.kde-look.org/content/show.php?content=12498](http://www.kde-look.org/content/show.php?content=12498)

Unfortunately, however, I am unable to install that theme and all of the other threads were not so helpful for me.  How can I enable this feature?

---

### Post by gingermark on 2006-04-15
To enable this feature right click on the desktop and select "Configure Desktop".
Then click "Behaviour", and select the "Current application's menu bar (Mac-OS bar)" option.

---

### Post by short4lif2 on 2006-04-15
Thanks, gingermark, it worked!
