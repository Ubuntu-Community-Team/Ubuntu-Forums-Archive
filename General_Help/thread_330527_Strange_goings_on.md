### Post by budgie9 on 2007-01-03
If memtest is showing errors then you can be sure your memory is not working as it should be.
I would suggest you pull each memory module in turn, so it will prove the module is faulty or not.
You have a copy of the memtest on your Ubuntu CD you can use.
I found that re-seating the modules can work sometimes. In my case Kingston memory in An Asus board was the fault. The memory was all good but they did not work will with Asus m/boards themselves. Change of memory make worked though.
