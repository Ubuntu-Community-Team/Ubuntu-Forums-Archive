### Post by Frogs Hair on 2013-02-26
You can auto hide the launcher on the left from Appearance > Behavior. The top panel can not be hidden in Unity but transparency is possible.

If you want a more traditional desktop install the session below and select gnome when you login. The XFCE and LXDE sessions are also options and have greater control of the top panel like the gnome session.   

```
sudo apt-get install gnome-session-fallback
```
