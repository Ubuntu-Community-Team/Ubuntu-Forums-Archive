### Post by lswest on 2009-01-05
Well, you could look into [CrunchBang linux]("http://crunchbanglinux.org/"), it's a distro based off Ubuntu that is pretty lightweight (runs openbox though).

You can also look into other lighter distributions if you wanted to (Puppy Linux, for example).

To answer you question though, you can probably best install it using the alternate install CD (however, if you have less than 256MB of RAM, I'd go with Xubuntu, if not CrunchBang).

*Edit* we replied around the same time, with 128MB of RAM, I'd suggest using CrunchBang and looking into [unetbootin]("http://unetbootin.sourceforge.net/")

Honestly though, you may need to go with DSL or Puppy Linux, if you're not able to up the RAM to at least 256MB of RAM.

That, or you could go all-out CLI with a minimal ubuntu install?  Course, that requires you to be very, very comfortable with the command-line.
