### Post by Turellius on 2007-05-02
I don't konw if any way to go back to a default install.  You could probably use Aptitude to uninstall things you installed and reinstall things you deleted, but I think it's probably just easier to reinstall.  I recently had box that was plugging along doing some updates, and somehow it got interrupted and all the system fonts were squares.  After some searching and trying different solutions for people with similar problems I decided to just reinstall and it seems that things are working better than they were last time.  In any case, the lesson to be learned is always make a backup before installing ANY  updates, even trivial font updates, because one false move could have you with a GUI full of gibberish.
