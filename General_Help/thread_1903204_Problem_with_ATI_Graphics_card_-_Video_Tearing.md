### Post by Synoc on 2012-01-02
64 bit is just more or less the instruction set for your processor if it supports 64 bit there is no reason not to use it at this point. There used to be back when 64 bit drivers weren't available or well polished, this is not an issue anymore. your RAM won't care. as you have a dual core processor I would think it's old enough to be 64 bit.

that said unless someone tells you "that's your problem" it might not be worth installing the 64 bit OS if you have been using this for a while...
