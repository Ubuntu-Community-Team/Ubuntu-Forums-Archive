### Post by rainwalker on 2007-11-06
[http://www.gnome-look.org/content/show.php/OSX-Tiger+theme?content=56577](http://www.gnome-look.org/content/show.php/OSX-Tiger+theme?content=56577)

Save the .tar.gz to your Desktop or wherever you want, then (assuming you're using Gutsy) go to System -> Preferences -> Appearance
Drag the .tar.gz file from wherever you saved it into that window, and it will install it

### Post by rainwalker on 2007-11-06
As far as I know, that will install a metacity (window border) theme, icon theme, and controls theme, which is basically everything.
You can customize the different parts of the theme individually, so if there's a window border you like more then you can use it but keep everything else.

Dock, no.
But, if you are able to use Compiz Fusion/Beryl, you can install Avant Window Navigator.

And if you're talking about the theme of the window border, that's a metacity theme.
