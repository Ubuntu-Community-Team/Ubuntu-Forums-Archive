### Post by bugmenot on 2005-09-28
[QUOTE=mumushi]regarding opera i installed it on machine then i can't find the icon. so everytime i wanted to use it i go to the terminal and type opera then it runs. can anyone help me how to make a shortcut to it in my desktop. one thing more when i close my terminal my opera browser also closes. 

Thanks in advance.[/QUOTE]
when you want to leave programs running after you closed the terminal prepend nohup ("no hangup") to the command. When you want to keep on using the terminal append an &. It detaches the command from the terminal leaving it running in the background.

$ nohup opera &
$

When you want to have the process back on the terminal enter fg ("foreground").

$ fg

the analog for backgrounding after you started the program is ctrl-z to halt the process and bg ("background").

^z
$ bg
