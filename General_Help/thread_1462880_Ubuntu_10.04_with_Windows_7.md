### Post by sandyd on 2010-04-26
install windows 7 first. then use the windows 7 partioning tool (control panel -> [set view properties to icons] -> administrative tools -> disk management). create free space by resizing one of the partitions. then install ubuntu using the option "use largest block of free space" when you get to the partitioning stage.
