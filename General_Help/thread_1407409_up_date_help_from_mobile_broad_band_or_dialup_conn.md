### Post by 3rdalbum on 2010-02-15
What exactly is the problem? Can you set up your mobile broadband account by right-clicking on the network manager applet toward the top-right of the panel and going to Edit Connections? Can you connect to it? Sometimes it may take multiple tries to connect properly on 9.10.

Once you have an internet connection, you can go to System > Administration > Synaptic Package Manager and hit the Reload button to get the list of packages from the Ubuntu servers. Then you can install MP3 support and everything else through Synaptic Package Manager.
