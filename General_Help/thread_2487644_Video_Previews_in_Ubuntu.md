### Post by TheFu on 2023-06-12
Being new to Linux, you'll need to relearn how OSes are put together.  

Ubuntu isn't from 1 company.  It takes thousand of independent projects and puts them together to make the "distro".  So, if something you want isn't doing what you want already, then there may be another program that does it. You just need to search for that program.  For example, if you think a file manager should show video previews and the default one on your box doesn't, you could look though the settings for it, to see if that can be enabled and which "helper" programs may be needed to accomplish it.

A few weeks ago, I loaded a new Ubuntu (new to me) and decided that I needed a file manager that does 2 specific things.  My search was on.  I tried 6 file managers before I found one that fit my needs.  Linux is about choice.  If you don't like the GUI, you can use one of 20-50 other choices.  If you don't like how a specific program works, you can probably find 2-10 competitors. Perhaps one of them does what you seek.  Or perhaps you are using the wrong tool - that's common for people coming from another OS.  They think that the way their former OS worked is the best/only way it should.  Linux and Unix have very different philosophies than MacOS or Windows.  Understanding those differences will safe much frustration.  

For example, a core idea in Unix is to make lots of small programs that do 1 thing really, really, well.  In MS-Windows, they make a huge program and add more and more features to encompass all those little things.  Neither is "right" nor "wrong". They are just different ways to look at the world.

There are multiple photo preview tools for Unix/Linux.  I deal with photos at the detail level.  My use of videos is fulfilled with a media center, Jellyfin and I'll playback videos using **mpv** or **KODI** or **OSMC**, depending on the platform in my hand at the moment.  Jellyfin has a web-app which can play videos, but I don't really use it that way.  I find browsers terrible for viewing videos.  That's me.  Lots of people prefer to watch videos in a browser, which is 100% fine too.

I did a quick search for video previewing.  Looks like there are many tools that can take a frame somewhere inside a video and show that as a thumbnail.  Some file managers allow adding custom scripts, so you could right-click on a video file and run a player that plays 5-10 seconds.  I think Thunar supports that. [http://freesoftwaremagazine.com/articles/tips_tricks_and_scripts_thunar_file_browser/](http://freesoftwaremagazine.com/articles/tips_tricks_and_scripts_thunar_file_browser/) calls them "custom actions".

The Gnome File Manager says it has a preview capability: [https://help.gnome.org/users/gnome-help/stable/nautilus-preview.html.en](https://help.gnome.org/users/gnome-help/stable/nautilus-preview.html.en)  Perhaps it isn't what you want or isn't enabled on your system?  IDK.

### Post by deadflowr on 2023-06-12
Install the package gnome-sushi to enable a quasi-live preview of files in Files.
(You have to highlight the file and then press the spacebar to invoke the preview.)

If you want something easier then run KDE (Kubuntu's desktop) as KDE's file manager dolphin has a preview pane built in.

The gnome desktop, which Ubuntu uses, is basic and they have stripped out a lot of stuff over the years to make it as dumb down as possible, to the point where I'm sometimes surprised it even has a file manager.

KDE goes in the opposite direction and keeps adding more and more features whenever they feel it works.
