> [COLOR=#000000]Does anybody have any idea what I am even talking about? If you do is there salvation?

Luckily I am on night shift so I can try. But I do not often use Firefox as primary browser.

Try to understand [Firefox Profile Manager]("https://support.mozilla.org/en-US/kb/profile-manager-create-remove-switch-firefox-profiles") where you can have multiple profiles.

Open Firefox using command   firefox -P
That is capital P.

You will probably just see one profile - default.

If tabs are screwed up you can hit Restart Firefox. But importantly try creating a fresh profile then you can switch between profiles if you are screwed. Or you might create a profile for different tasks rather than having multiple tabs in one mega session.
