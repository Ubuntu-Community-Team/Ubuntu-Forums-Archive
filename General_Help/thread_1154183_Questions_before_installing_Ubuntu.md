### Post by mikewhatever on 2009-05-09
Generally speaking, not a single Windows program can run natively on Linux simply because the two OSs are binary incompatible. VLC and MSN are probably the easiest, since VLC is also built for Linux, and Pidgin supports MSN chat (no video though). In order to run the rest, you'd either have to use wine or run Windows in a virtual machine. With such a strong dependency on Windows software, why do you want Ubuntu anyway?
