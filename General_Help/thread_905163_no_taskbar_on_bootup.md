### Post by nikobor on 2008-08-30
If you can run the terminal, you can start synaptic by 
*sudo synaptic* 
and reinstall the missing components.
If you still want to restart to reinstall ubuntu, then:
*sudo reboot*
or
*sudo shutdown -r now*
