### Post by sanderd17 on 2011-05-23
It uses especially processor power, so a light hard disk but a powerful processor would be ideal.

I don't know all details about it anymore, I ran it on my laptop for a month (since I don't have a server) but it used so much CPU that my laptop became hot and the battery drained fast.

So it's nothing for a laptop, but if you are going to build a server for it, than it's good I think.

Maybe something that you should know: on the world community grid, most calculations come from non profit groups that are not supported by universities. Most universities have their own (maybe shared with other universities) supercomputer network and prefer to use that instead of the World community grid. 

As example: this is the supercomputer network of the Flemish universities: [https://vscentrum.be/nl/en](https://vscentrum.be/nl/en)
