### Post by okplayer02 on 2010-06-26
well first plug in the usb stick into the computer then type from the command line

```
dmesg
```then you will see if the device has been recognized or not.

if it has then go to top right and find network manager and find the wireless network you want to connect to and fill in the relevant information

if it has not been recognized type the command 
```
lsusb
```and post that here so we can help you further.
