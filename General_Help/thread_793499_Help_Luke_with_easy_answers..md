### Post by chrisccoulson on 2008-05-14
Hi, for #2, a quick search of the forum brings up [this]("http://ubuntuforums.org/showthread.php?t=51003"). This is for creating debian packages from scratch (including compiling). There are other ways to create debian packages - eg, using something called checkinstall. However, checkinstall has no dependency handling and shouldn't be used to create debian packages that you're likely to distribute.

Also have a look at the Packaging section of [this Wiki page]("https://wiki.ubuntu.com/UbuntuDevelopment")
