PC crashed had Win XP and Ubuntu installed. Removed Ubuntu and now want to re-install. Will it create a further partion or just load up as before?

---

### Post by Kevbert on 2008-09-08
Just delete the old Ubuntu partition and then re-install in the free space.  You can do this via the install CD by selecting manual install, set mount point to / and format ext3.  You can leave the swap partition as is.

### Post by Elfy on 2008-09-08
Orjust use the old partition, don't need to delete old partition. Choose manual and pick old partition - edit partition, choose ext3 from Use As dropdown menu and / in Mountpoint dropdown menu.
