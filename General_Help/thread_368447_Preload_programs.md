### Post by Rui Pais on 2007-02-26
hi,
some stuff on those articles are a little obsolete.

Oo can be automatically preloaded if you run it, in the menu go :
Tools -> Options -> OpenOffice.org -> Memory and check:
Enable systray quickstarter

No need for scripts or extras apps, at least with gnome or kde.


hth.
