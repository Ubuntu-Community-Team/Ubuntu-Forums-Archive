### Post by mikewhatever on 2007-11-10
Have you tried installing 7.04 on the old PC? There is a restricted drivers manager under System>Admin that should have an nvidia driver for your card. You can check if it's there from the live cd, but can't install it, since it would require a reboot. A live cd is really not very functional and is intended to let you try the OS before installing.
