### Post by mikewhatever on 2021-05-18
There is a lot of nonsense in that article, so, don't believe all you read on the web.
I think you'll get comparable results with most modern desktop distros. They all have kernel level power management, like CPU+GPU frequency scaling, and user space options, like screen brightness and suspend to RAM.
Different Desktop Environments usually do not affect power usage, so it wouldn't matter if you use KDE or Mate, XFCE or LXQT, etc.
As said above, if you like Kubuntu better then Mate, go for it.
