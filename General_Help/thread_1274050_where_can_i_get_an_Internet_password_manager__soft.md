### Post by TekWiz2001 on 2009-10-04
SXipper plugin for firefox is a Roboform replacement... that imports your passwords from Roboform ... I imported my passwords from my winxp version of roboform... and it was read by SXipper and works great...


SXipper  works just like Roboform but in Linux  and you can import your Roboform passwords to SXipper
by using the import feature...

Its all explained in SXipper and takes about 2 min... From Roboform  under identities .. print list...
when the interface is open... I chose passcards instead of identities to get a password.html file .... just press save and you get the password.html  file that SXipper can read ... There is a section in SXipper that can read the Roboform Passcard.html file and import the Roboform info into SXipper...
