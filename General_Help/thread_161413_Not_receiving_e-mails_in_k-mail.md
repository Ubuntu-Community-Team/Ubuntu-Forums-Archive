When I first set up K-mail I checked the 'check what the server supports' option and it selected TLS and clear text. In Thunderbird I had selected SSL. So I changed the setting to SSL in K-mail and now appear to be getting mail normally. Thanks for the tip.:-D For what it's worth I'm comparing Konqueror/K-mail to Firefox/Thunderbird. Konqueror is definitely the faster browser and so far I like how K-mail works with Anti-Spam [Spamassassin and Bogofilter] and Anti-Virus [F-Prot] clients.
