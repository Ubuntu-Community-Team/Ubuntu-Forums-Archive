### Post by fi2 on 2010-09-07
Here is the solution: Instead of IMAP I set IMAP+ In Preferences->account->Edit->Receiving Email. So far so good.

Strange. I used to use Outlook with IMAP with the same mail server for ages. I know little about IMAP protocol to understand the difference, so no idea about the actual reason. Cross fingers, it would stay stable.
