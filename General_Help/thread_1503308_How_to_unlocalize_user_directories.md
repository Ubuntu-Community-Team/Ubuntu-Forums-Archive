### Post by zvacet on 2010-06-07
I think you can revert to English under system>preferences>keyboard.There set English as your default language.If that doesn't work (for any reason) remove support for your locale.

For files/folders with the space in name you can type in terminal (let say that folder name is my folder)

"my folder"

or you can just type 

"my  and then press TAb for autocomplete.
