### Post by cj.surrusco on 2010-06-12
If it's a CPU fan, you definitely don't want to turn it off. Excessive heat can cause permanent damage to a CPU. If it is a case fan, then maybe you could turn it off without any bad side effects. Just unplug the fan from the motherboard/power supply.

You should probably use a program like lm-sensors to monitor temperatures to make sure everything stays cool, if you decide to unplug a fan.
