### Post by neu5eeCh on 2014-07-06
By the way, in the future, if you're trying to build a file from source, the following should vastly simplify the procedure:

```
apt-get build-dep <package>
```

This command will download and install the packages necessary to build a source Debian package.

Also, consider using the *checkinstall* command (apt-get install checkinstall if you don't have it), rather than *make install*. By using checkinstall, you can use apt-get to remove the program (if there are problems with it) later.

```
sudu dpkg -r <package>
```
