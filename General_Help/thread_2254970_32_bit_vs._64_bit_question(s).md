### Post by BBQdave on 2014-12-01
Ubuntu Unity 14.04 (64-bit) runs smooth & fast on my Lenovo Thinkpad T400, Intel® Core™2 Duo processor & 2GB of RAM. To my understanding, you can run any 32-bit application on a 64-bit OS, the 32-bit application may pull in extra packages for support.

I would recommend matching your OS to your hardware. Ubuntu labels there 64-bit distros as amd64 - that is a nod to AMD, one of the first company's to implement 64-bit processing on personal computers. Some folks get confused with this, amd64 is 64-bit, and will run on intel machines - no worries :)

### Post by grahammechanical on 2014-12-01
> [COLOR=#000000]you just can't upgrade via the software center to a 64 bit version. [/COLOR]

That is correct. Converting a Ubuntu 32 bit install to a Ubuntu 64 bit install is described as technically possible but hard to achieve. 

[http://summit.ubuntu.com/uos-1411/meeting/22353/when-should-we-stop-making-32-bit-images/](http://summit.ubuntu.com/uos-1411/meeting/22353/when-should-we-stop-making-32-bit-images/)

I hope that you are running an LTS version. And that you are also backing up your important data. When I first installed Ubuntu 7 years ago it was the 32 bit version I installed. But I made the switch when I messed something up and saw re-installing as the easy fix option. You might find yourself in a similar situation one day.

The really important questions for you, I would say, are: Are there 64 bit versions of my favourite games and do they give a better user experience than the 32 bit versions that I am presently using?

Regards.
