### Post by FakeOutdoorsman on 2009-05-13
I recommend using the libx264 presets in FFmpeg for converting video.  You'll get smaller files with better quality than the standard FFmpeg mpeg4:
[url=http://rob.opendot.cl/index.php/useful-stuff/ipod-video-guide/]
iPod Video Encoding with FFmpeg[/url]

Unfortunately, this is only compatible with Jaunty Jackalope because FFmpeg for older Ubuntu releases is much too old.  However, you can always compile FFmpeg yourself fairly easily:
[url=http://ubuntuforums.org/showthread.php?t=786095]
HOWTO: Install and use the latest FFmpeg and x264[/url]
