### Post by jlubey on 2010-04-16
you can use the update manager to install the new distro. press alt-f2 and type in update-manager -d. If you dont have that, a simple sudo apt-get install update-manager will install it, then press alt-f2, and again type in update-manager -d to run it. Using this, you can update to the beta release 10.04 today, or else wait for the full release later this month
