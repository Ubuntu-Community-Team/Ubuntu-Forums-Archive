### Post by Yaldair on 2010-06-02
You can click and change your status (like click on "available") which opens up a bar below it where you can set a personal message.

If I'm correct there is also a third-party plugin which makes it even easier to do. But I'm not sure how it's called unfortunately...I just happened to come across it yesterday.

You can also (in that bar below) (I think) define standard messages. Like when you set your status to "watching tv" (away) your personal message instantly changes to "I'm currently watching tv"

Hope this helps, if you're still checking this thread ;)
