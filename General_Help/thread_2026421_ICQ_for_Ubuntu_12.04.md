### Post by calagan on 2012-09-04
Here's how to install the official ICQ client
[LIST]
[*]Install Adobe Air, using the method described [here]("http://ubuntuforums.org/showthread.php?t=2024914")
[*]```
wget http://ftp.icq.com/pub/ICQ_Linux/icq4linuxBeta.air
chmod +x ./icq4linuxBeta.air
Adobe\ AIR\ Application\ Installer icq4linuxBeta.air
```
[/LIST]

I've only tested it in Ubuntu 12.04 32-bit, but since the above method also allow you to install Adobe Air in 64-bit, ICQ should work too.
