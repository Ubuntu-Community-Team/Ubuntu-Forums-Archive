### Post by Keith Hedger on 2009-11-28
change the file extension to .odt (or whatever you are saving the original as) or right-click on the file select Properties->Open With click on the "Add" button scroll down thw list and select openoffice.org and then close the properties dialog from then on double clicking on a .bak file will open it in open office,be warned this may cause problems for other programs that use .bak as a backup extension
P.S.
you can also open it directly from oo by going to the file menu and selecting open then navigating to the file as normal ( to see hidden file/folders ( those beginning with a dot ) press cntrl-h when in the file chooser dialog )
