### Post by sayems on 2008-10-01
Open Firefox and type following at url box: about:plugins
and make sure you have the Shockwave Flash 9.0 or above

If you don't have latest flash installed, you can install it from here.

[http://labs.adobe.com/downloads/flashplayer10.html](http://labs.adobe.com/downloads/flashplayer10.html)

> Download Plugin for Linux (TAR.GZ, 3.78 MB)

> Double-click on the files 

> Run with terminal

> Enter...Enter....Enter.. your done!
