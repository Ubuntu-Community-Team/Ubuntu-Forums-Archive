### Post by cchhrriiss121212 on 2011-02-28
Testing, bug reports, documentation and support are all things you could start doing right away.

But while Ubuntu is an open-source project, it is also a privately owned product which has many full-time developers. For that reason I'd recommend contributing to something upstream, find a program you use and ask to help out there. An OS is only ever as good as the software it supports.
