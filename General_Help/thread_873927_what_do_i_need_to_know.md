### Post by Hawk__0 on 2008-07-29
Linux is a lot different than windows, but that doesn't mean it's harder.  In ubuntu the easiest way to install stuff is to click on applications->add/remove.  here you can find pretty much anything you will want to use.

things to know: Never log in with the root user account.  always use your user account. Windows apps/games for the most part will not work or work properly with wine (a program used to add windows APIs to linux). And... Patience :)
