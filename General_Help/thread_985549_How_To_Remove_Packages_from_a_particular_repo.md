### Post by jodef on 2008-11-17
You can do it through Synaptic:

System -> Administration -> Synaptic Package Manager (on the left hand side menu you will see a tab labelled **Origin** click on it you will see the various sources click on each source and you will see the packages installed for each source.

Hope this helps.
