### Post by forrestcupp on 2008-02-09
Well, any Windows commercial apps would have to be run with something like the Wine compatibility layer.  When I looked in their app database, I saw that some people have had some success with Quicken, but TurboTax doesn't work.  You can see their app compatibility database [here](http://appdb.winehq.org/).  Just type in the software you want to look up in the search bar on the left side.

For an open source alternative to Quicken, you might want to check out GNUCash.  It can import files from Quicken.  The only problem is that it currently doesn't have an export feature.  I use it and like it, though.  And it is cross-platform, so I can use it with my same checking file in Windows and Linux.

You're pretty much out of luck when it comes to Tax filing software.  My solution is to use the web based versions on their web sites.  They are pretty much the same thing and they work in any OS that has a web browser.
