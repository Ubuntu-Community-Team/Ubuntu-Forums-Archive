### Post by lukeiamyourfather on 2015-02-06
Probably not the answer you're looking for but it sounds like a hardware problem with the laptop. Sometimes manufacturers will include a special partition with diagnostic software that will run checks on all the sensors, memory, stress the components, etc. If your laptop has one of those utilities I'd run it to see if it finds anything. I agree with raptir, the hardware is modern so it would throttle itself instead of shutting down, older processors like Pentium 4 era would simply shutdown without warning if they got too hot (throttling wasn't a thing at that point).
