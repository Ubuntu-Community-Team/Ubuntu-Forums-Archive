### Post by daslinkard on 2012-10-05
You can try the following sudo command
```
sudo apt-get install gnome-session
```

Once it is finished try logging in.  Wile at the login screen, make sure 'Sessions' (at the bottom of the screen) is set to 'Gnome'.

Let me know if this fixes it for you.
