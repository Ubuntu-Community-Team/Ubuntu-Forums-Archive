### Post by TheFu on 2014-02-14
From the "teaching a man to fish" perspective .... 

If the normal update/upgrade didn't do what you wanted, then I'd look for a PPA from a highly trusted source. If no trusted  PPA is available, I'd stop there.  It isn't worth my effort to maintain individual software packages like that anymore. Hasn't been since 2008.  Only you can decide if a PPA is trustworthy. For me, it needs to come from the same team making the program, not some 3rd party that I've never heard of or haven't met. F/LOSS is a very small world and developers learn about each other all the time.

But other people would get the source code, download it and follow the build and installation instructions.

I use KeePassX which is v1 DB compatible across all the platforms that I use. That DB compatibility is what has kept me from v2 ... well, that and Mono. Not a fan and I avoid programs that depend on it. It is a political decision, nothing against the technology.

Does that help any?
