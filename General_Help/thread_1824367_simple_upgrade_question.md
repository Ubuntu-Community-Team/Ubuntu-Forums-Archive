### Post by Lunarmonkey on 2011-08-13
you can let the update manager do it, it wont find it unless you have it set up to find "normal releases". To do this open update manager, click on setting in bottom left corner, click on updates tab. Then navigate to bottom where it says release upgrade, click and set to normal release. Close then check and 11.04 should pop up at top as an option for update. Hope this helps.

### Post by CatKiller on 2011-08-14
I'm on a fresh install now (new computer) but on my old computer I'd done upgrades-in-place from 5.04 all the way up to 11.04. Some of them were perfectly fine and some of them had issues The upgrades that I had problems with weren't necessarily the upgrades that other people had problems with, and the upgrades that were fine for me weren't necessarily problem-free for everyone else. There isn't a hard-and-fast "it'll be rubbish" or "it'll be great." The only way to know for sure is to try it.

Installing fresh is significantly quicker than upgrading, but you do need to keep your data safe (separate /home partition) as well as remember all those system-wide tweaks that you did. The consensus that flipper T suggests doesn't exist. It comes down to what's right for you.
