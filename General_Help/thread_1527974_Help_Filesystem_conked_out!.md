### Post by bodhi.zazen on 2010-07-10
If you open a terminal, what is the output of :

```
fdisk -l
```

If that does not list your Ubuntu partition you can try to run testdisk (testdisk is in the Ubuntu repos, so you can install it while running the live CD).

[http://www.cgsecurity.org/wiki/TestDisk](http://www.cgsecurity.org/wiki/TestDisk)

[http://www.cgsecurity.org/wiki/TestDisk_Step_By_Step](http://www.cgsecurity.org/wiki/TestDisk_Step_By_Step)
