### Post by dharvell on 2009-08-13
> **muctadir said:**
> i tried that. but the kde crushes very often. i thought it was because it was conflicting with some applications of genome....

so, i want to install the kubuntu. if i install kubuntu, ubuntu will be gone....
Not if you install it as a dual boot.  Dedicate a portion of your drive to Ubuntu and a portion for Kubuntu.  When you boot your computer, you will then be given the option of which one you wish to boot to.  When you install Kubuntu, you should be given the option of how you want to install it... use remaining portion of disk, use a specified amount of the disk, or use the whole disk.
