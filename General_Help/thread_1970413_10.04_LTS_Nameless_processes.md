### Post by zero2xiii on 2012-05-01
Hay,

This is curious and I don't understand the language of your system. However, the following command will show you a "tree". This should help you identify the parent of the nameless process. When you notice the process again, run the following:

```
sudo ps -ejH
```

Sudo is not esential, but just to make sure you get the true parent if it is a higher priviledged aplication spawning it than your current user.
