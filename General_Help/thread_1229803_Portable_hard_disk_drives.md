### Post by gunksta on 2009-08-02
This one's easy. As long as the drive mounts as a standard USB drive, it will work with Ubuntu (or any other version of Linux) just fine. To the best of my knowledge, most of the little portable drives mount as a USB drive. To a computer, they look like big USB keys. Compatibility is a no-brainer. If you have an old laptop hard-drive laying around, they an be repurposed as an external drive. All you have to do is buy an empty case. I did this and was able to build an 80 GB external drive for $20.00.

### Post by tgalati4 on 2009-08-02
Avoid the ones that require Windows-only encription software.  It should say on the package--"Compatible with Windows only"  Linux can't see it because the firmware on the drive requires a Windows program to supply the encription password.
