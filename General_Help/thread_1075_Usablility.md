### Post by cseg on 2004-10-18
I think it would be useful to have a Forum dedicated to usability.

For example, I was tring to access the Windows network, and a dialog came up saying:  You must install Samba first!  (or words to that effect).

That's all very well and good, and I understand and appreciate that Ubuntu has no services turned on by default.  This is a good thing.

On the other hand, the dialog above could have said:

   To use this feature, Samba must be installed.  
   Please understand that doing this makes your computer accessible over the network and could make it vulnerable to network attacks.
   Understanding this, would you like me to install Samba for you?
   (if yes) 
   Please enter your password so I may perform administrative operations.
   (samba gets installed)
   Windows network access now available...

When people encouner situations which could have better usability, they could be discussed in this new Forum.  This would provide that extra layer of polish which would make Ubuntu a truely great distribution.
