### Post by Rubi1200 on 2010-09-09
To install Flash on Ubuntu you need to either use the terminal
```
sudo apt-get install ubuntu-restricted-extras
```or search in the Ubuntu Software Center and install it from there.

Flash is not part of the default Ubuntu installation.
