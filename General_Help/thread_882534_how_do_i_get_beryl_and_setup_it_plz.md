### Post by sayakb on 2008-08-07
Beryl is no longer supported in Ubuntu. If you are using Ubuntu 7.10 or above, you should already have Compiz-Fusion installed. At a terminal, do:
```
sudo apt-get install compizconfig-settings-manager
```
Then goto System->Preferences->Advanced desktop effects settings to manage plugins.
