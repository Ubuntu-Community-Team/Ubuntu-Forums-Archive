### Post by GregBrannon on 2010-05-20
Another possible approach: Network devices often have a reset button that will restore them to the factory defaults.  Google the manufacturer and model number, find a manual online, and look for the reset button, how to use it, and then the resulting configuration if there is a reset.

If no reset, the DSL provider may be able to reset it remotely or tell you what the settings are.

If I read this correct, the password to the internet-connection is lost. Not the password to access the router and not the password for the LAN (wireless) connection.

Sorry to say that I don't have any idea how you can retrieve it, since most routers don't actually mask the password by displaying dots, they just print a fixed number of dots if the password is set. Therefore you can't reveal the password by looking at the source of the page.

You should be able to have your password reset by your Internet Service Provider helpdesk, where you're probably have to answer a few questions about your personal details before they do so.
