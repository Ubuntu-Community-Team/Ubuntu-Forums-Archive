### Post by ceag on 2012-02-11
Dear Experts,

in Kubuntu 11.10 I can not print certain PDF files using Okular on my Kyocera FS-1010 printer. The problem first occurred dealing with the online tickets of German railway delivered as pdf documents (an important issue for my, I often need this tickets). 

Meanwhile, I found other pdfs (like Wikipedia Books) which I can not print as usual. I have no idea of what is particular in this pdf documents avoiding their printout. Other pdfs can be processed by Okular and the printer as expected.

Any attempt to print the document via Okular in the normal way effects a temporary increase of the printers cooling fan speed-but no printout. The job is displayed as well in the spooling window but it disappears later on without printing.

The reported problem has begun definitively with Kubuntu 11.10. and persists. Prior to the update to this version, i.e. using 10.04 and 10.10, I had no problem with printing pdfs.  

Currently, I am forced to send the pdfs to an online converter (pdf to doc) in order to be able to print them.

Some posts in a German Ubuntu message board shows that the reported problem seems to be limited to Kyocera users. 

There were no useful answer, no solution since 2 months.
