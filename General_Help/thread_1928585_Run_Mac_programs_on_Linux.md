### Post by 2F4U on 2012-02-20
My understanding is that Mac OSX and Linux are not binary compatible to each other. Therefore you can't run the Mac programs directly on Linux. And with respect to Wine for Mac, Apple is probably blocking such a development. However, as far as I know, you would be able to run Mac OSX in VirtualBox on a Linux host.
