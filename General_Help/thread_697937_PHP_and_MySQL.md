### Post by astrotech226 on 2008-02-15
Click System -> Administration -> Services.

There isn't a lot of feedback from this application, but I have mysql setup to start on the runlevels I want.  If I uncheck "Database Server (mysql)" the service stops.  If I check it, the service starts.  This can be proven by checking the process list.
