### Post by earthpigg on 2009-10-05
plan old ubuntu will indeed run fine on most neebooks.

But if you want to save hard drive space and get a faster-responding that doesn't eat up RAM, I personally am a big fan of [LXDE]("http://lxde.org/").

if you want it to come preinstalled with a bunch of stuff as per the *Buntu norm (word processor, image editor, etc), Lubuntu is supposed to be out later this month and is officially endorced by Canonical.

there are several unofficial spinoffs of Ubuntu that use LXDE: [http://wiki.lxde.org/en/Ubuntu#Ubuntu_variants_with_LXDE_pre-installed](http://wiki.lxde.org/en/Ubuntu#Ubuntu_variants_with_LXDE_pre-installed)

Any LXDE-based Ubuntu spinoff is bound to be snappier than Xubuntu, Kubuntu, or Ubuntu... and will still have synaptic and access to the same apps and security updates, assuming an internet connection is present.

LXDE also has a netbook-friendly interface, if you wish to use it:
[http://wiki.lxde.org/en/LXLauncher](http://wiki.lxde.org/en/LXLauncher)


there isn't much ubuntu-specific documentation for LXDE, so if you wish to go that route then the upstream LXDE wiki will need to become your friend for non-trivial customizations.
