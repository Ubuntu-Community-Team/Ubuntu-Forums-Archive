### Post by QLee on 2010-12-14
[QUOTE=samaniac]Hey, I recently installed ubuntu on an external hard disk and the time it takes to boot is killing me. After selecting ubuntu in the GRUB menu, it takes about 46 seconds to show the ubuntu 10.10 message and then about 14 seconds to allow me to type my password to login.

Well, your external might be (probably is) a USB connection, thus you are limited to USB speed (likely USB2, these days) and USB2 is not as fast as access from an internal drive. At boot, considerable information has to be input for the system to work. (Does the access led on the external flash and/or stay lit while the delay is happening?)

You'd probably find an install to a partition on an internal disk to be faster to boot.
