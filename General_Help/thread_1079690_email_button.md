### Post by lswest on 2009-02-26

I never bother use media keys on my keyboard, but I knew the shortcut by default in gnome is to open the default application to do with a group (in this case, mail clients), and so changing the default changes which is launched.  You can also make a custom launcher for the shortcut (if you're using Compiz or metacity, either works), but this is probably the easiest since it will open thunderbird for mailto: links as well.
