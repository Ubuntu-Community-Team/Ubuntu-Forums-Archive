### Post by meho_r on 2010-05-27
1. Sound applet is contained in "Indicator Applet". So, right click on the panel > Add to Panel, and then add "Indicator Applet". Now you should see two additional icons in the "menu bar on the top right" (which is, actually, "Notification Area" ;)).

2. I don't see the problem you're talking about. If you go to System > Preferences > Sound, you should see something like this:

[IMG]http://img30.imageshack.us/img30/521/screenshot1mq.png[/IMG]

So, if you see the same dialog, you should be able to increase input/output volume above 100%.
