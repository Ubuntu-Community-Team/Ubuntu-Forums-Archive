### Post by cariboo on 2008-08-28
Most programs only take up hard drive space when you are not running them. This haas nothing to do woth the amount of ram you have. Programs use ram when you are running them, and as soon as you close the program, it gives up the ram that it was using. Remember linux is not Windows. Linux does a much better job of memory management than Windows could ever hope to do. Don't get confuse ram with hard drive space, consider ram temporary storage, used by programs and hard drive storage is permanent storage use to store data.
