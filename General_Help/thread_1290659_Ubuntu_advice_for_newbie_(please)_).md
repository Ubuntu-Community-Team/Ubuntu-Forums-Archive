### Post by oldos2er on 2009-10-13
You can try the LiveCD without making any changes to your computer. It requires 384MB minimum; whether that fits your specs or not, I don't know.

 If you want to install Ubuntu, there are a couple different ways you can do it. One is called Wubi ([http://wubi-installer.org/);](http://wubi-installer.org/);) the other is a more traditional dual boot ([https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)).

 Whatever you decide to do, be sure to backup any sensitive data first.

### Post by undecim on 2009-10-13
As others have mentioned, you can try it out without changing anything on your computer.

If you like it, I would recommend installing it, because XP has become deprecated; windows won't be providing any more security updates, which means its become very susceptible to viruses, etc.

You can still keep XP as a backup system by choosing to resize XP when you install Ubuntu. but make sure you get a backup of any important files you have first. If something were to happen while that partition is being resized (power goes out, mouse munches on your power cord, cosmic rays crash your CPU, etc) then it may be difficult to restore that data.

Also, if you have a broadcom wireless card, you may need to use a hard-wired ethernet connection to download your wireless drivers, and after that you can use your wireless.

After installing, you will want to install the "Ubuntu Restricted Extras" package from the Add/Remove application (or "Software Store" in the current beta version) This will give you the codecs and software necessary to play most music and video files.

Overall, I think you will be happy with Ubuntu once you get used to it. If you think you would like something more similar to Windows, you can also try Kubuntu, which is a variant of Ubuntu that used the KDE Desktop Environment as opposed to the GNOME Desktop environment (i.e: only difference is the look and feel of your desktop; I think KDE is more like the Windows dekstop than GNOME is)

And one final hint: If you run into problems, search [https://help.ubuntu.com/](https://help.ubuntu.com/), and if you can't find anything there to help you, post your problem on the forums or the IRC chat. The best thing about Ubuntu or any other free and open source software is the community that drives it.
