### Post by earthpigg on 2009-08-16
the Ubuntu USB Startup Disk Creator is designed to make ubuntu and ubuntu-based operating system .iso files into bootable thumb drives.

windows xp is not an ubuntu-based operating system.

what i would try in your shoes:

find a computer with a functional cd drive. physically unplug the hard drive so the windows CD can't do anything retarded to your hard drive - don't trust those guys at microsoft. plug the target thumb drive in. boot from that cd.

act like the thumb drive is a hard drive, and install windows xp like normal to the thumb drive.

i have no idea if the Windows EULA legally allows this. read it yourself, and come to your own conclusions based on your understanding of Canadian law.
