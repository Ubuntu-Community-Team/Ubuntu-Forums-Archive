### Post by loklaan on 2012-05-16
Ubuntu 12.04 comes with **Simple Scan**, and as the name suggests it is pretty simple and easy to use. :)

Open it like you would any other application (hit the Windows key and type *simple scan*). Simple Scan will inform you if it cannot find a scanner, but if everything is ok you're free to start scanning!
