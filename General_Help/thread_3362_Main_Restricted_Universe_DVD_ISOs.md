### Post by jdusablon on 2004-11-05
I don't know how useful this would be, because the packages are constantly changing with newer versions, patches, etc.

It is possible, however. When you use synaptic, select all the packages you want to burn to DVD and check the box Download Packages Only. You should then be able to burn the packages to DVD.

Again, I stress that this isn't really worth the effort for a number of reasons.
