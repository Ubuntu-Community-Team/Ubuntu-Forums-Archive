### Post by timfrost on 2005-12-25
In gnome:
1: Put a blank CDR in the CDR/DVD[+-]RW drive
2: Cancel the burner tool that starts automatically (the default is a dialog box asking what type of CD/DVD you want to create, and you can click the "Cancel" button)
3: Browse (in the gnome file manager) to the location where the ISO image is
4: Right-click the ISO, and select  "Write to disc..."
5: click the "Write..." button
6: Wait for cdrecord to finish


NOTE:  The "Write to disc.." option is only present if the file name is recognisable as a disk image.
