### Post by mister_pink on 2008-04-24
A little tip- often it's recommended that when everything crashes, you should try going to another tty (Ctrl Alt F1) to kill the offending program, then try Ctrl Alt Backspace then the magic key combos. 

I discovered earlier that you can try doing the first magic key press, which is with R, to put the keyboard in raw mode, then try going to another tty again. It let me so I could then kill the offending process and go and save anything I had open. I rebooted after to make sure things were all right.
