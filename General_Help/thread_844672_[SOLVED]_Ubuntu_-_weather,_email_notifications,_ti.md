### Post by x1a4 on 2008-06-30
> **bonfire89 said:**
> but it doesn't have all the same options as in ubuntu...

That's correct.  Ubuntu uses GNOME as its default GUI, Xubuntu uses XFCE.  Also, not all panel plugins are installed by default.  Check Synaptic and install the other panel plugins.

Further, you can add the 'Xfapplet' plugin to your panel which will allow you to insert GNOME plugins. If you did a clean install of Xubuntu, you will have to install the GNOME panel plugin you want, and enable GNOME support.  You can do this through the 'Advanced' tab of the  'Sessions & Startup' dialog in 'Settings Manager'
