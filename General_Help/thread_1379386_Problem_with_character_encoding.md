### Post by quixote on 2010-01-13
Are you using Thunderbird?  If so, the setting is under Edit > Preferences > Display > Fonts & encoding button.  I have "Western ISO-8859-1".  I think that one is pretty good on diacritical marks, but I don't know whether it's the best.  Thunderbird's setting will, I believe, override Firefox's setting, if for instance you click a "mail" link in Firefox.
