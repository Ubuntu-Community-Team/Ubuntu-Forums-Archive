### Post by mcduck on 2011-11-11
> **soryu said:**
> can i use ubuntu classic(fallback) and have the special effects also?

Do you mean Unity 2D, Gnome Classic or Gnome Fallback?

The main idea of Unity 2D and the fallback mode of Gnome Shell is that they don't require graphics hardware capable of handling visual effects, so if you use those you are not going to get any effects.

Gnome Classic (not fallback) should be able to run Compiz as it's window manager, so that option would give you desktop effects.
