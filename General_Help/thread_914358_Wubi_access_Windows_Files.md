### Post by jlipscomb on 2008-09-08
Here is the deal, I installed Ubuntu by using Wubi, because I have no Blank CD so I used Deamon(Spelling?) Tools to install it. So, I have ubuntu installed and working just fine. Here is now the problem. I am unable to see the Windows Partition to get to my documents files. I understand that they way that I have it installed is currently like a virtual machine, so to speak. It is reading from c:/ubuntu from my windows directory, and that is where the home folder is. I want to know is there anyway of going back one more directory to get ubuntu to see the C: drive so I can get to my music, pictures, etc.

Moreless, I am saying is there anyway to get to my Windows Files under wubi. I read somewhere that the rest of windows files will be under /media/ but they aren't.

Thanks for your help.

### Post by Orlsend on 2008-09-09
Go to **file System>Host** *And thats it! you C drive.

### Post by psoplayer on 2008-10-18
I think Orlsend meant that in a Wubi installation it will mount your Windows drive at the folder /host

You can bookmark your windows folder by hitting Alt + F2, typing "/host", then Ctrl + D. Now you will see an entry on the left in the bookmarks and in the "Places" menu at the top of the screen.
