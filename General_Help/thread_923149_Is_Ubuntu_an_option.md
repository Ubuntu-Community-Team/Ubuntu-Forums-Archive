### Post by mb_webguy on 2008-09-18
> **bmbeeman said:**
> and I'm also going to need to get a copy of office because my one class requires that everything be in .docx format.

Ok, that's just wrong.  A professor shouldn't punish students for not having the most recent and most expensive version of a particular software application that isn't even available on all systems, especially when other, equivalent software applications are available.  This policy would require every student to go out and buy a $120 (minimum) piece of software.  It's doubtful a student with a Mac would have MS Office 2008 installed.  Since that software isn't available for Linux, that means that we Linux users would have to also buy a $120 (minimum) license for Windows.

I've worked in academia, and in most schools this wouldn't be allowed.  I would go to the instructor and ask why he's requiring that assignments be turned in using such a new and poorly supported format, as opposed to a more common format like doc.  Heck, if the instructor for some reason seems adamant about using docx, I'd go to the department head and ask why the instructor is forcibly endorsing Microsoft products.

HOWEVER, OpenOffice.org 3.0 will support the OpenDocument Format.  I know it will be able to open docx files, but I'm not sure if it will be able to create them.  The first release candidate was recently released, so you can give it a try if you want.

As far as Angel Learning, it looks like it's all web based, so it should be fairly platform-independent.

And for wireless networking, I haven't had many problems with wireless in Ubuntu since Feisty Fawn.  Intrepid Ibex is supposed to have an increased focus on mobile computing, so wireless should be even easier.

### Post by Nepherte on 2008-09-18
You can open .docx files with OpenOffice 2.4, but that requires some extra stuff to install. I believe OpenOffice 3 will support it by default. Don't expect 100% compatibility. You will be able to view it and save in that format, but it *might* look a little different when you open the same file with Microsoft Office.

wireless networking pretty much depends on the wireless card/chipset you are using. Most of them will work rightaway, but I suggest you check it in advance.

As for IChat, plenty of chat programs on Ubuntu.Pidgin supports most of the chat protocols.

As for games, I thought you can run counter strike. Don't know about the others.
