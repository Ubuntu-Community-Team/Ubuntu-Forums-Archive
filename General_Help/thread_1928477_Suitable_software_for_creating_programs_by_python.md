### Post by MG&amp;TL on 2012-02-20
> **Hiuhu said:**
> Hi am new to programming and have only read the basics of python. Someone pliz recommend a software where I could use to create my own programs using python

Hi, if by "software" you mean an IDE (Integrated Development Environment), then there are quite a few, my personal favourite for python being Geany. However, for your first programs I recommend using a "notepad" type program, the default one in ubuntu is called gedit.

However, if by "software" you mean a program that will create your code for you (Fireworks & co, I'm looking at you), then AFAIK there isn't one, and you will learn a lot more and write more efficient code if you write it manually.
