### Post by cta16 on 2010-08-18
What version of Ubuntu are you using? If you are using 10.04, then do the following:

1. Back up the old configuration. In the terminal:

```
sudo cp /boot/grub/grub.cfg /boot/grub/grub.cfg.bak
```

2. Open grub.cfg to edit

```
sudo gedit /boot/grub/grub.cfg
```

3. Scroll down until you see a line that begins with the word 'menuentry' followed by 'gnu grub version 1.98-1ubuntu5'. Change the 'gnu grub version 1.98-1ubuntu5' part to whatever you want it to show at the bootloader. 

Here's what mine looks like
```
menuentry 'Ubuntu 10.04' --class ubuntu --class gnu-linux --class gnu --class os {
     recordfail
     set gfxpayload=1280x800
     insmod ext2
     set root='(hd0,3)'
     search --no-floppy --fs-uuid --set 8326166f-d5d4-4b53-9b42-92dev291b866
     linux /boot/vmlinuz-2.6.32-24-generic root=UUID=8326166f-d5d4-4b53-9b42-92dec291b866 ro   quiet splash
     initrd /boot/initrd.img-2.6.32-24-generic
}
```

4. Save it and reboot. If anything goes wrong you can restore the old configuration by doing
```
sudo cp /boot/grub/grub.cfg.bak /boot/grub/grub.cfg
```

If you are using an older version of ubuntu, then you have to work with /boot/grub/menu.lst instead of /boot/grub/grub.cfg.

### Post by ranch hand on 2010-08-19
That is a very interesting question.  I do not know and can't find where that is generated from.

Your /boot/grub/grub.cfg file is silly to edit.  Every time update-grub is run it is over written.  You change it by editing the scripts that generate it.  These are mainly /etc/default/grub and the contents of /etc/grub.d.

As that comes at the top of the screen it should be from /etc/grub.d/00_header.  I find no mention of it there.

This leads me to believe that it may be hard coded into grub.  I would check these links;
[http://gnome-look.org/content/show.php/Dark+screen+clock?content=125464](http://gnome-look.org/content/show.php/Dark+screen+clock?content=125464)
[http://ubuntuforums.org/showthread.php?t=1287602](http://ubuntuforums.org/showthread.php?t=1287602)
[http://members.iinet.net/~herman546/p20.html](http://members.iinet.net/~herman546/p20.html)

There is a link in my sig for a very brief overview/intro to grub2.  You might want to look at the first post there just to get a better feel for how grub works.  The first link above is great, I do not think it will help with your question but you should know abut it.

The second link above may help.

You may also want to look into burg.  It is a modified, very themable version of grub.  I don't use it but have tested it and it is very, very nice work if you go for that sort of thing.
