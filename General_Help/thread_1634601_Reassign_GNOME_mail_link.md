### Post by czr114 on 2010-11-30
I believe evolution-indicator can't be reassigned, and I'm not sure if you want to completely remove indicator-message.

If you're simply looking to put Thunderbird in the tray, AllTray is a great solution.

Install alltray, and if you need Thunderbird on the panel, use the properties dialog to modify the command to 'alltray thunderbird %u'
