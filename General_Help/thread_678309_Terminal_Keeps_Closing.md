### Post by Tenken on 2008-01-28
Are you trying to look at the contents of the install script? Because double clicking on a .sh file usually just runs the script, it opens the terminal to execute then close when the script is done. To view the contents of the script you can right click it in Nautilus, choose "Open With" and select the text editor option.
