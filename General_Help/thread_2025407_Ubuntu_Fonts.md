### Post by Ubun2to on 2012-07-14
What do you mean? Do you want to install MS fonts, or do you want to install fonts that you create or download online?
To get MS fonts:
1. Open the Software Center.
2. Go to the page for Ubuntu Restricted Extras.
3. Make sure you have Installer for Microsoft TrueType core fonts checked in the Add-ons.
4. Update plugins, then install.
To install fonts you make or download:
1. Open the font file.
2. In the bottom right corner of the window, click the button that says "Install Font."
