### Post by robsoles on 2010-09-29
That's tricky because the basic run of it is that if you have a bootable CD and a PC that boots from CD/DVD-ROM then it should jolly well just boot.

I can only suggest that there is a 'weakness' in both the windows setup disk and the drive on the acer laptop where (at least perhaps) the Ubuntu CD you are using was burnt better somehow.

If I were trying this personally I would be inclined to make an ISO image of the windows setup CD and then I would burn it using the same machine that burnt the Ubuntu CD and finally see if that boots.

If a real fresh copy of the CD won't boot in the acer then I would service the CD/DVD-ROM unit - have it serviced by professionals if you don't know for sure what to do to service it.
