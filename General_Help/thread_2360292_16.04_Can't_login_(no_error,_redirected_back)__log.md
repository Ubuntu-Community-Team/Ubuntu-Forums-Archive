### Post by ajgreeny on 2017-05-03
Try booting to an older kernel from grub.
There seems to be a big problem with the 4.4.0-77 kernel for many users.

See new bug at
[https://bugs.launchpad.net/ubuntu/+source/linux/+bug/1687623](https://bugs.launchpad.net/ubuntu/+source/linux/+bug/1687623)

### Post by ajgreeny on 2017-05-03
Great.
Now run ```
sudo apt-get update
sudo apt-get dist-upgrade
```
which should bring in the additional packages that were not installed originally by update-manager along with the kernel.  That should fix the 4.4.0-77 problem.
