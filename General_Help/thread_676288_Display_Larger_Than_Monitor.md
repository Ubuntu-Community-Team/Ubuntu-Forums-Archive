### Post by mikelito on 2008-01-23
there is a chance this can be fixed via GUI, otherwise I can't help, as I don't own an NVIDIA, and often manually fiddling with xorg.conf require to experiment with the hardware you have at hands.

however, it seems you have selected to mirror the display on the second monitor, and that the resolution on the second monitor is different from the one on the first (you often get this with laptop and fixed LCD screen, for example).

anyway, go to system>administration>screen and graphics, and try to play with the config. if you really want to mirror, you should set the same resolution on both screens (if you have LCDs, probably one of the two will look bad because resolution is not the intrinsic one). 

other option, is to extend the desktop, so that you can set two different resolutions. I don't know if you'll see application bars on the second head. I never could try myself, as I own an ati with their buggy buggy buggy closed source driver.
