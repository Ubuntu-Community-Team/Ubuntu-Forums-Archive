### Post by Vox754 on 2007-01-05
Sorry, I didn't get your English the first time.

Anyway, your processor speed, RAM memory and hard drive size are mostly useless information. If the screen goes blank you should check your video hardware, video cards and motherboard.
I assume you have tried the "Safe screen install" and pressed the F-buttons to read the help, and to set screen resolution (to a small one), and deactivate ACPI options.

If that doesn't work, download Knoppix, which is a Live CD Linux distribution also based in Debian (as Ubuntu is). If Knoppix runs fine then you can pay attention to which drivers and configuration it uses at start up so you can find a way to implement those with the Ubuntu expert install CD.

Just remember, you must give **more** information; what have and haven't you tried.
