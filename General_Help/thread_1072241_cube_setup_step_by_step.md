### Post by howefield on 2009-02-17
For the 3D cube, and assuming that your system can run compiz.

Go System > Adminstration > Synaptic Package Manager and install compizconfig-settings-manager

You'll then find the application in your system > preferences menu, this will allow you to change compiz settings including the 3D cube.

1. General options and select the number of horizontal desktops you require.
2. Activate Desktop Cube and Rotate Cube.
3. Go to "cube reflection and deformation". Set as required.
4. Spin the cube by pressing mouse scroll wheel and drag.
