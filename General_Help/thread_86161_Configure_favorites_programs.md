### Post by Sheco on 2005-11-04
Right click on the PDF file you want to check, switch to the "open with" tab, then add the program you want to use to open the PDF file and select it. That's it.

---

### Post by kperkins on 2005-11-04
You can right-click on a file and choose properties, and then Open with and either choose something on the list that shows up, or click Adde and find the proggie you want to use for that type of file, add it, and choose it as the default to open it with.
