### Post by bward1 on 2007-01-28
1)  I'm not sure about your sound card, but in my experience ubuntu does a pretty good job of figuring out your hardware and helping you use it.

2)  Linux in general is a totally different ball game from windows.  In ubuntu, you primarily install programs from the ubuntu repositories.  Many of the applications you have in windows you may not have in linux, but there will be other programs which do essentially the same thing.  For example, if you are looking for a linux version of itunes, you will not find it, but there are several different programs that will allow you to play music and/or update your ipod from linux.  Ubuntu uses apt to install programs so you will learn to either type apt-get install program, or use the graphical interface called synaptic, to browse through the repositories to find and install (or uninstall) the applications you want.  Here is a good list of linux equivilents to windows applications for you. [http://ubuntuforums.org/showthread.php?t=33183](http://ubuntuforums.org/showthread.php?t=33183)

3)  If you are trying to use this ubuntu system to play battlefield 2, than it might be rather difficult but if you are using it to do more simple every day tasks like check email, browse the web, play music, write papers etc. then it is probably not that hard, it just takes time, patience, and the desire to learn.  Don't expect it to be just like windows, it will take some time to learn many of the differences.  It is most certainly a doable do if you are willing to learn.  Doing it single boot with linux is nice because you dont have to worry about breaking your windows install, so that should make things a little easier for you.  I am intentionally not putting a numerical answer here because the experience is different for everyone whether it be the hardware you are running it on, the applications that you are trying to use, or your willingness and desire to learn.
