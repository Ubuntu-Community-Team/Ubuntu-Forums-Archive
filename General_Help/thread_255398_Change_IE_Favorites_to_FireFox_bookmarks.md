### Post by BerkeleyDude on 2006-09-11
It seems that Firefox stores its bookmarks in:
~/.mozilla/firefox/*random_name*.default/bookmarks.html

See if your HTML file looks similar. If so, try replacing bookmarks.html with it, and restarting Firefox.
