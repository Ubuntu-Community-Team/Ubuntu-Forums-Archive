### Post by mcduck on 2010-03-03
System/Preferences/Appearance, on the Visual Effects-tab. (Setting it to anything else than "none" will enable Compiz, although it should already be enabled if your graphics card & drivers are capable of running it).

Install the CompizConfig Settings Manager if you want more detailed configuration options.
