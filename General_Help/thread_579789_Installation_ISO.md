That's how wubi 7.10 works even though at the moment physical CD behaviour is only partial (will enable it soon).
For the time being, just donwload the ISO and place it in the same folder as wubi. If you have a CD, use one of the many ISO extractors. 

Note that wubi 7.04 us the ALTERNATE ISO, while wubi 7.10 uses the LIVE CD ISO.
