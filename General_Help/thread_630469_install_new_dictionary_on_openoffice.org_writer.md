### Post by GregoryMA on 2007-12-03
I want to install a Canadian dictionary in openoffice.org writer.  I am tired of spell-check telling me I mis-spelled behaviour.  There is no 'install new dictionary' in File - Wizards though.  Does anybody know how to do this?

Greg.

---

### Post by metalheart on 2007-12-03
Have you set your language of choice to English(Canada) in System -> Administration -> Language. After that Ubuntu should download all relevant language packs (also for OpenOffice) for your language.
