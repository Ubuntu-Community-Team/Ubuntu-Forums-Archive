### Post by zachbb on 2009-01-22
ok..

**I fixed this!!**

To fix it, I just created a symbolic link from my plugin directory to the java directory:


cd /usr/lib/firefox-3.0.5/plugins **(check that this is your directory by navigating through your file system)** 

sudo ln -s /usr/lib/jvm/java-6-sun/jre/plugin/i386/ns7/libjavaplugin_oji.so

And there you go :)

edit: before I did this, I uninstalled all java packages on my system and reinstalled Sun Microsystem's java JRE.
