### Post by y-lee on 2008-07-27
have you tried enabling smooth scrollong?

In firefox, click Edit/Preferences
In "Advanced", click the "general" tab, then tick "Use smooth scrolling".


You might also want to consider the [Smoothwheel Firefox add on]("http://smoothwheel.mozdev.org/"). The xpi file can be found on the installation page, I had to right click this file and save to desktop and then manually install. It made a difference but eventually i quit using it for no particular reason.

The smoothwheel add on has alot of options to fine tune scrolling. So some experimentation may be needed to fine tune it.
