### Post by twisted_steel on 2007-11-04
I believe you are looking for the Sessions Preferences tool.  Go to System -> Preferences -> Sessions.  The first tab is "Startup Programs."  You should be able to click add and start your program from there.

You will need to know the location to specify in the Command field.  If you have an existing shortcut on the Desktop or on the panel that you want to auto-start, you can right-click on the shortcut and look at Properties.
