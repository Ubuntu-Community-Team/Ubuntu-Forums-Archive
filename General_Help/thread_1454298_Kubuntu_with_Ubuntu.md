### Post by lovinglinux on 2010-04-14
> **OnlyTim said:**
> 1. Can I install it and have it work side by side with my existing Ubuntu 9.10 Karmic?

You don't need to install another OS, just install **kubuntu-desktop** from Synaptic and will have the option to use one Desktop Environment or the other when you login. Additionally, you will have access to all your applications and documents while using one or the other. You will have only one system, with two Desktop Environment options.

I would recommend installing **kde-minimal** and **kdeplasma-addons**, instead of **kubuntu-desktop**. This way you won't get all the applications Kubuntu installs. Then you can install only the applications you want or use those from Gnome that you already have. Nevertheless, I recommend some applications like dolphin, kate, ktorrent and kfind,  which are much better than gnome counterparts in my opinion.

> **OnlyTim said:**
> 2. Will it give you the choice at start up of using the existing gnome and the new KDE?

Yes, if you don't use the automatic login. Otherwise you can change what you want to use on the next boot from system preferences (don't remember exactly where).

> **OnlyTim said:**
> 3. If I decide to stay with KDE, can I import my back up settings, such as email addresses, Firefox bookmarks, and money applications currently used with Ubuntu.

As long as you don't install a new OS and just install the **kubuntu-desktop** or **kde-minimal**, you won't lose your settings. They will be used when needed, depending on the application you load. For example, KDE saves config files for KDE applications under ~/.kde but if you start Firefox while running KDE desktop, Firefox will still use ~/.mozilla/firefox/yourprofile , just like when you launch Firefox using Gnome.
