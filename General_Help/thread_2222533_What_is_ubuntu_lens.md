### Post by caver12 on 2014-05-07
Unity Lenses are what Dash uses for searches. Different type of search different lens used lens. Such as the the online search lens. If it is enabled then when you do a search in Dash it will also search the internet. You can disable this one if you don't want it.
System Settings>>Security and Privacy>>Search>> Off (button)
