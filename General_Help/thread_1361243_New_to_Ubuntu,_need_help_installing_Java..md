### Post by synapsys on 2009-12-21
Forget the downloaded file.

From menu at top open System >> Administration >> Synaptic package manager

Type sun-java6 into the search box.

Mark 3 packages for installation (double-click)
> sun-java6-bin
sun-java6-jre
sun-java6-pluginClick Apply
