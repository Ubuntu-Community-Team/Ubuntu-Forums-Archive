In 10.10 version I could drag a dir icon to an open application bottom button  keeping the mouse button pressed until the application was restored to the screen and dropped it in a selected area. For example in K3b to burn a data DVD. 
I cannot do this in 12.04  This is very annoying. I have to reduce the size of file browser and keep k3b drop area visible in a second level.

In Mac, when you click on a file icon that is in a second level on your screen you can keep it in this level (not to make this your top level app displayed) until you release the mouse button. This is useful for drag and drop.
In Ubuntu, just you click something in a non top level it becames on top, covering what it was on top.  It would be nice if it happens after you release the mouse button.
