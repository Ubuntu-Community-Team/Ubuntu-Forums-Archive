**[COLOR="Blue"]Transferring Sony Ericsson's contacts to Nokia phone using Ubuntu [/COLOR]**
[COLOR="DarkRed"][COLOR="DarkRed"]First, THANKS to Wammu (Phone Manager for Sony Ericsson, Nokia, Samsung, etc., in Linux) and Ubuntu Linux Distro Developers.

Before starting this procedure, check your phone support by wammu in there webpage.

I used 1. Sony Ericsson W810I and 2. Nokia 3120 classic modes.

Further, Since Nokia Mobile is symbian based, I need to install gnapplet. To avoid this, I used bluetooth connectivity and USB cable for Sony Ericsson.

First connect any one phone, Say, I connected Sony ericsson with phone wizard. Then, retrieved phone contacts. And backup the contacts by backup &#8594; save and saved as .vcf format.

Now, one of the phone contacts are retrieved. Disconnect it and connect next phone (Nokia).

Follow same procedure as above to same the second phone contacts.

Now delete all the contacts and go to backups &#8594; import and select first phone contact .vcf file and select the option in popup window and press ok.

Now, first phone connect is downloaded in another phone. Disconnect the phone and follow same procedure to save the second phone (Nokia). 

Note:

1. Through above mentioned procedure, I got succeeded.
2. This is the way, I used the Wammu tool. I am not responsible for any lose of data / any information from the phones, by any means with this tutor.
3. Before starting this tutor, I backup-ed my contacts in Phone Memory card.
4. The contact options, that is "home", "work" etc., while saving in other phone becomes "others" option. [/COLOR][/COLOR]
