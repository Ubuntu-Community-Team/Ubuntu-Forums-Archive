### Post by darthmob on 2010-04-04
Of course! ;)
You can set them up in System -> Preferences -> Keyboard Shortcuts

Eg. I use Alt+q/w to switch to the left/right workspace and Ctrl+Alt+q/w to move the active window to the left/right workspace.
