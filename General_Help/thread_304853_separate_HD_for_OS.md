### Post by viciouslime on 2006-11-22
I agree with the previous post, 36GB SHOULD be enough for 3 or 4 OSs, but you said that you were going to upgrade XP to vista and since that is a 15GB minimum install before you start installing any other m$ bloatware you need to get a fully functional system (office etc.) then maybe 74 would be better if you could afford it. 74 is probably overkill for 3 or 4 OSs but 36 not quite enough if Vista is one of them so you could use any spare space on the 74GB for files you might like really fast access to, if you were doing some video editing for example.

It's a shame there isn't an intermediate option really, or you could just not install vista and get another 2 or 3 linux distros on the 36Gb one :p
