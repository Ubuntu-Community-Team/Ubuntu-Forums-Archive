### Post by mikko.ohtamaa on 2007-08-01
With the default Skype install, Skype eats plenty of CPU time when there are chat windows open. This is due to bug in Skype's emoticon animation code. Disable emoticons and you will reclaim your precious CPU time.

Please see [https://developer.skype.com/jira/browse/SCL-136](https://developer.skype.com/jira/browse/SCL-136) for further info.
