### Post by sayakb on 2008-08-07
GDM themes are for the Login Window. You need GTK themes (or emerald themes if you just need to theme the borders). I think the Moomex theme goes well with some emerald windows decoration that looks like Aero. Btw, you need emerald themes manager to be installed for using emerald window decorations:
```
sudo apt-get install emerald
```

### Post by sayakb on 2008-08-07
Just right click on one and select [B]Delete this panel.
[/B]You could then right click on the other one, select **Add to Panel** and add the applets you need. The one in the pic has the Main menu, the Window switcher, the notification area, clock and the trash applet added to the panel.
