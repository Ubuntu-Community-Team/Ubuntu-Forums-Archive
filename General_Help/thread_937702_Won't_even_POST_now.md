### Post by dmizer on 2008-10-08
There are a whole number of things that can prevent a machine from posting, and a great many of them are - kick myself in the pants because I can't believe I didn't realize that before - simple.

Things like, incorrectly connected IDE cables, HDD jumper settings, FDD cable connections, CPU fan power connector is in the wrong location, etc.

Start really really basic and double check everything. One machine I had wouldn't post because I cabled it for master/slave, but I had installed cable select cables. One machine wouldn't post because the BIOS didn't like having a HDD and DVD drive on the same IDE channel.

Would probably help a lot if you took a good hard look at the mainboard documentation too.

What mainboard did you purchase?
