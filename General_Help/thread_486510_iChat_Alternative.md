### Post by ®©™ on 2007-06-28
I don't know if anyone has made bonjour-support, but the .mac username is really an AIM username, so you could just get an IM client that works with AIM.  GAIM/Pidgin should work here. From [Wikipedia](http://en.wikipedia.org/wiki/Gaim):

Supported protocols
.NET Messenger Service (a.k.a. MSN)
OSCAR (AIM/ICQ/.Mac)
etc etc.
