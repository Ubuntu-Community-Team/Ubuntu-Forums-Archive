### Post by Timbers2k on 2009-02-27
This is a problem in gnome 2.24. All previous gnome versions saved the session the way it's supposed to, but they broke it in 2.24. I know that some distributions back ported the old session saving code so it still works, but Ubuntu didn't. I don't think there is any solution until the next Ubuntu comes out in April, that uses gnome 2.26, where it's been fixed.
