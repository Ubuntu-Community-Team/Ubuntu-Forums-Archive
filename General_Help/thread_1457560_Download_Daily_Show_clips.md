### Post by luisella on 2010-04-20
**rtmpdump** ( plus a stream url detector ) seems the app that can rip the streams from Dailyshow, Colber report, CBS.com ( Letterman ) etc. Here is a guide to download from comedy Central with rtmpdumb and URL Snooper ( these one is for windows, must find a linux equivalent ) [http://stream-recorder.com/forum/tutorial-dl-comedy-central-t6324.html](http://stream-recorder.com/forum/tutorial-dl-comedy-central-t6324.html)

The protocol is Adobe's proprietary RTMP ( [http://en.wikipedia.org/wiki/Real_Time_Messaging_Protocol](http://en.wikipedia.org/wiki/Real_Time_Messaging_Protocol) ). It is used by Comedy Central (Daily Show and Colbert Report), CBS.com, TV.com, Hulu.com (or any site hosting Hulu content) and  Amazon.

In 2009 Adobe filled a DMCA suit against the hosting of rtmpdump on sourceforge. ( [http://linuxcentre.net/adobe-has-issued-a-dmca-removal-request-for-rtmpdump )]("http://linuxcentre.net/adobe-has-issued-a-dmca-removal-request-for-rtmpdump")  Replay Media Catcher ( 40$ for Windows ) used to allow RTMP ripping, but was legally forced to remove it.
