### Post by Biznarie on 2008-12-13
You don't have to remember all the packages you installed, you can show only installed packages, I believe the option is in one of the menus on synaptic (can't remember atm as I'm on my gaming/windows machine). In Add/Remove... all you need to do is change the drop down box to Installed Applications Only, then you can easily uninstall any application you do not need (i recommend you do this first). Also you may want to look at Ubuntu Tweak, it's a good application for tweaking ubuntu and there's good application tweaks in there including a GUI for package cleaning, and extra repositories for things like Open Office 3.0, Amarok 2.0 nightly builds, skype, etc. It's very easy to use, here's the link. [http://ubuntu-tweak.com]("http://ubuntu-tweak.com")
