### Post by ndru on 2007-09-20
The Ext2 Installable File System ([http://www.fs-driver.org/](http://www.fs-driver.org/)) allows reading and writing of Ext2/3 partitions. I use it on XP, but it should work on Vista as well. Try the compatibility mode if it doesn't.
