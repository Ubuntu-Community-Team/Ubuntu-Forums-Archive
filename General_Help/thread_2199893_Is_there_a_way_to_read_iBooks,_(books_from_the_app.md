### Post by silvervulpus on 2014-01-16
**Formats**

 The supported [e-book]("http://en.wikipedia.org/wiki/E-book") formats by iBooks are [EPUB]("http://en.wikipedia.org/wiki/EPUB") and [PDF]("http://en.wikipedia.org/wiki/Portable_Document_Format").[SUP][[20]]("http://en.wikipedia.org/wiki/IBooks#cite_note-20")[/SUP] As of version 2.0, iBooks also supports a proprietary iBook format (IBA), generated with the [iBooks Author]("http://en.wikipedia.org/wiki/IBooks_Author") tool. This format is based upon the EPUB format but depends upon custom widget code in the iBooks app to function.[SUP][URL="http://en.wikipedia.org/wiki/IBooks#cite_note-21"][21]



FBReader will support Epub and PDF formats, but i do not believe IBA is available on ubuntu, there might be either a way to run the ibook program its self of ubuntu via some kind of compatibility software (mac uses unix programming too) or like i said, there might be some kind of IBA converter software somewhere on the web, i dont know where it would be though you might consider checking the ibooks author program for a conversion tool or setting to change IBA's into Epubs
[/URL][/SUP]
