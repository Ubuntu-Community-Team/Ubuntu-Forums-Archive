### Post by ad_267 on 2009-10-31
You need to have desktop effects enabled to do that. Check under System - Preferences - Appearance - Visual Effects.

If they're enabled, then install the compizconfig-settings-manager package and then have a look under the compiz settings to enable this option.

### Post by tweaksource on 2010-06-21
[http://ohioloco.ubuntuforums.org/showthread.php?p=8219998#post8219998](http://ohioloco.ubuntuforums.org/showthread.php?p=8219998#post8219998)


Go to System->Preferences->CompizConfig Settings Manager. Click on Viewport Switcher then click on the Desktop-based Viewport Switching. Set Move Next to Button5 and Move Previous to Button4. That's how I fixed mine.

Still works in Lucid.
