### Post by m_duck on 2010-11-13
Assuming you are using Firefox, type the following into the address bar.```
about:config
```It will show a warning page, so hit the 'I'll be careful' button. Next filter (using the box at the top of the page) for 'backspace'. The only result should be 'browser.backspace_action'. Double click this and change the number to '0' (without quotes). I think on Linux it is '2' by default. Your backspace button should now cause the browser to go back a page.
