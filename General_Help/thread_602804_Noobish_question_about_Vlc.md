### Post by drs305 on 2007-11-05
Select Settings, Switch Interface, Skins2 to get you into the 'skins' mode. That should open in the default skin. From there, I can right click on the skin and there is an option to select a different skin using the "open skin' option. 

Also, if you are still in the 'wxWdigets' mode, you can go to Settings, Preferences, Interface, Main Interface, Skins. At that point, make sure the Advanced Options is checked in the lower right corner. That will open up a place for you to input the skin location.

If you need skins or instructions, there are some here:
[http://www.videolan.org/vlc/skins.php](http://www.videolan.org/vlc/skins.php)

I download them and put them in my ~/.vlc folder but you can put them anywhere you like as long as you can find them when you want to open a different skin.
