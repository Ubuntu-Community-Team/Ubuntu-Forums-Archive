### Post by t0p on 2009-09-08
Do you mean what file system you should use to format the usb stick before putting the .iso on it?  If so, I would recommend fat32.  Why?  Because that's what I've always used, and it's never given me any problems.

Or do you mean what file system to use for the actual installation on your computer?  If that's what you mean, I suggest ext3.  Why? Because that's what I've always used.  Except if you're installing to a SSD. In that case, I would use ext2, because it involves less writes to the disk and so the SSD will last longer.  Ext4 is now out and proud, and many people like it.  So maybe you'll like it too.  Who knows?
