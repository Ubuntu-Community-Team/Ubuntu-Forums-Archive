### Post by Mark Phelps on 2011-10-24
Your post is confusing ...

Don't see how simply applying upgrades in Ubuntu would erase any picture files.  Have been doing this for years and never had any personal files erased.

What do you mean by "downloaded"? If you're attempting to copy pictures from a memory card to your PC, that's "uploading" to your PC, not downloading.

How is the memory card connected to your PC? If you have a card reader that uses a USB connection, it's not uncommon for such connections to drop, especially during a long file transfer.

In such cases, you would remove the card, reconnect it, and contine the file transfer from where you left off.
