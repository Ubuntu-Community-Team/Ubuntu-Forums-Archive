### Post by dave.biagioni on 2010-07-27
Hi all,

Like many others posting in the forums, I've been having issues with suspend hanging my (Lenovo SL500) laptop under 9.10, a 10.04 upgrade and, more recently, under a clean install of 10.04.  I tried a number of different workarounds as suggested throughout the forums (all voodoo magic to me, playing games with the display drivers and kernel versions), but to no avail.

Oddly, the last thing that I noticed before throwing in the towel was that I had left a memory card for a point-and-shoot camera in the laptop's built-in SD slot.  I removed the card, rebooted, and  suspend now works correctly.  I recently bought a new camera, whose SD card has been constantly in and out of the slot over the last few months to back up photos, so I suspect (but have no hard evidence) that this may have been causing previous issues with suspend in 9.10.

I would be surprised if this was the root cause of the suspend issue experienced by most users, but hopefully it will be relevant for some.

I should mention that the laptop does not appear to enter hibernate mode properly.  As I haven't ever used that mode on a regular basis, I don't know if this behavior has changed on account of removing the SD card.
