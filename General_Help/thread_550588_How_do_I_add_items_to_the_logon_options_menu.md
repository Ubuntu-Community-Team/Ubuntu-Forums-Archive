### Post by jamvegan on 2007-09-15
I'm not sure if this is what you're looking for, but if you go to System->Administration->Login Window, at the bottom of the General section there is a button labeled "Edit Commands...".  If you click it, there is a drop-down menu to select the command that you'd like to edit, including Custom Commands 0-9.
