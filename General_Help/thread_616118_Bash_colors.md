### Post by kdub432 on 2007-11-17
by default it should be turned on. However if you've deleted your .bashrc profile, you need to run the command "alias ls ="ls --color" . This will alias your ls command ot always print its output in color. I take it that you dont want to put the --color flog after every ls command.
