### Post by hepcat57 on 2008-12-05
FYI...I'm not affiliated with Yugma, but we do use it as the main web meeting software for our small company.

I've just received the following email from their support team:

[I]We are very happy to announce the beta release of the Yugma screen sharing program for Linux! A while back you had expressed interest in this application and I wanted to tell you personally that it is finally available. To download the Linux beta program please go to [www.yugma.com](www.yugma.com) and click on either the “Host a Meeting” button (our system should prompt you to download the Linux client) or you can click on the “Download” button and download from there (you can also access the release notes for the beta on this page). If you come across any bugs or have feedback, please send it to [email]linuxbeta@yugma.com[/email], your input would be greatly appreciated as we work make this program even better!

Thank you for your interest and we hope you enjoy Yugma for Linux![/I]

I've installed the app and it seems to work well so far.  Yugma seems to be one of the few viable multi-platform desktop sharing apps out there, with integrated conference calling.  Full-featured web meeting software was actually one of the last things making me keep a Windows VM available.

I definitely recommend them. :)
