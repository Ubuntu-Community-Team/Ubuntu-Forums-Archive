### Post by ago on 2008-04-24
/host will not appear under /media and it will not appear in mount. This is because it is a protected mountpoint (since it contains the system virtual disk).

If you want it on the desktop use symlinks:

ln -s /host ~/Desktop/host
sudo ln -s /host /media/host

### Post by ago on 2008-04-24
Yes that is only a folder link. You do not really want direct access to that device. Because, for instance, if you by mistake click on "unmount", your system will become rock frozen.

You can mount bind /host /media/host if you really want something closer... 

sudo rm /media/host
sudo mkdir /media/host
sudo mount -o bind /host /media/host
