### Post by letkemanp on 2007-07-08
Move the scripts to ~/.gnome/nautilus-scripts or ~/.gnome2/nautilus-scripts to install them. 

You can find more about nautilus scripts here [http://g-scripts.sourceforge.net/](http://g-scripts.sourceforge.net/). 

Once installed you should have a 'Scripts' option in you context menu in nautilus
