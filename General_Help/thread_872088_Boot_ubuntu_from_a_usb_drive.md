### Post by jonian_g on 2008-07-27
This can't be done with a 1GB usb drive, you need one at least 2GB for ubuntu but 1GB would do for a smaller distro.
I don't know of an app that can do this on a mac but you can boot a live cd on your mac and use UNetbootin:

[http://unetbootin.sourceforge.net/](http://unetbootin.sourceforge.net/)

It's just an executable file, double click and the app runs. It downloads the distro.iso you want and creates a live usb.

Ubuntu runs great on Eeepc. I have one with ubuntu 8.04 on. To install the audio drivers, Wifi drivers and have all the Fn keys working you will need a script that you can download it from here:

[http://www.ubuntu-eee.com/index.php5?title=How_to_use_the_ubuntu-eee_script](http://www.ubuntu-eee.com/index.php5?title=How_to_use_the_ubuntu-eee_script)

More info and tips for Ubuntu and Eeepc here:

[http://www.ubuntu-eee.com/index.php5?title=User_Guides](http://www.ubuntu-eee.com/index.php5?title=User_Guides)
