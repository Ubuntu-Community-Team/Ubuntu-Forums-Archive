### Post by explemonk on 2007-05-14
To install Lightning in Thunderbird 2.0, first download and save the .xpi file somewhere on your hard drive.  Then in Thunderbird select Tools -> Add-ons.  In the pop-up dialog, select the Extensions tab and then click the Install button. Browse to the .xpi file you have saved and open it.  It should install and then you can restart Thunderbird to have Lightning running.  This process should be the same for any other extension you want to install.
