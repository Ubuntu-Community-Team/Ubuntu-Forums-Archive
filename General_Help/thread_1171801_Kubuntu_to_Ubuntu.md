I'm running dual boot with Windows Vista and Kubuntu. I was wondering the easiest way to run it with Ubuntu with Vista instead of Kubuntu.

---

### Post by Idefix82 on 2009-05-27
Install the package ubuntu-desktop. Now when you boot into Ubuntu, in the login screen you can click on "Session" and choose, whether you want to log into KDE or Gnome. This way you will keep your settings and will be able to reuse some programs that you already have installed.

---

### Post by cdude on 2009-05-27
First install ubuntu-desktop package. Then remove the kubuntu-desktop and all its related packages - use synaptic to search and remove any package that has in it the word kubuntu, kde or qt4 (or qt3). This will give an ubuntu / gnome desktop.
