### Post by The Cog on 2019-08-30
Open the toolbox (ctrl-B) and double-click the pen icon. This pops up a tools options menu. Or at least, that's how you do it in Ubuntu 16.04, which I'm currently on.

### Post by The Cog on 2019-08-30
Edit -> Preferences -> Tool Options.
You get can save the current tool options now, or choose to save the current tool options every time you exit.
