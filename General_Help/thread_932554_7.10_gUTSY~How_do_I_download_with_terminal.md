### Post by ju2wheels on 2008-09-28
It looks like you are trying to use apt, thats only for downloading packages from the repositories.

If you want to download a file from the internet use wget:
```

wget http://somesite.com/somefile.zip

```

If you are interested in saving a youtube video in particular then you first need to install youtube-dl:
```

sudo apt-get install youtube-dl

```

Then to download a youtube video just use the link to the video:
```

youtube-dl http://www.youtube.com/?blablaVideoCode

```
