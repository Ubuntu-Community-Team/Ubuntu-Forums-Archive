### Post by lander on 2006-11-16
Hola, pues mejor te contesto en español. Solo tienes que instalar el paquete kde-i18n-es lo puedes instalar desde synaptic o desde la consola con:

sudo apt-get install kde-i18n-es

Este archivo trae el idioma español de todo el kde. Así que también te traducirá cualquier otro programa de kde.

Espero te sirva
