### Post by IWantFroyo on 2011-02-28
Pros: -It's tied with the GNU project, so getting programs to do almost anything is easy.

-It's almost impenetrable.
-The community knows what they're doing.
-If you don't like one distro, you can try another. I have used and enjoyed Debian, Fedora, and am using Backtrack with Ubuntu as a dual boot.
-Huge developer community.

Cons-
-It can break and need extra configuration. Linux Mint and Ubuntu are the most user friendly, along with Jolicloud. 
-Some distros are advanced. Don't use Arch Linux as you're first distro.
EDIT: The very above isn't necessarily bad. You just don't want to start out with it, so get your information gathered first.

### Post by 3rdalbum on 2011-02-28
Ubuntu is exciting. There's always new things in each new version, and there's new versions coming every six months. If you like living on the bleeding edge, you can run the current development version and notice the changes and additions every day or so when you get updates.

As for superiority, there are a couple of things you might have noticed over Windows:

1. Loads of built-in drivers. The most common ones that aren't built-in, are easily discoverable and installable using "Additional Drivers" in the Administration menu.

2. Fewer notifications. On Windows, almost everything you install creates a notification area icon, and almost every notification area icon will send a notification bubble at the drop of a hat. Plug in a USB device and you get three notifications, for instance. On Ubuntu, you typically only get notifications when you really need them. It's also more newbie-friendly to not be told "There's a new version of SuperProgramRunner, click here now to upgrade, or otherwise be reminded in 5 minutes".

3. Greater security. Linux has always been written with security in mind. Unlike Windows, you don't even need a firewall, because Ubuntu doesn't listen for incoming connections (unless you install something that does listen).

4. Easier software discovery and installation. You can find and download software from the "Ubuntu Software Center" - one click will download AND install it. If you want something that's not in the default set of repositories, you can add another set; and any software that needs updating will be listed with the operating system components in the Update Manager.

5. Multiple workspaces. Store your windows on different workspaces to switch between them easier and organise them into workflows.

It's also really cool to note that several Ubuntu features have been put into newer versions of Windows, and upcoming versions of Windows are gaining features that Ubuntu currently has. For instance:

Separation between user and administrator; been in Ubuntu since Ubuntu 4.10, been in Windows since Vista.

Desktop visual effects: Been in Ubuntu since 6.06, been in Windows since Vista (and Ubuntu 6.06 still had better effects available than even Windows 7.

Drivers built-in: Been in Ubuntu since 4.10, been in Windows since 7.

Easy-to-use, quicker installer with less input required: Been in Ubuntu since 6.06, been in Windows since 7.

Application store: Been in Ubuntu since 10.10, coming in Windows 8.

ARM-architecture version: Been in Ubuntu since 8.04 (I believe), coming in Windows 8.

Workspaces: Been in Ubuntu since 4.10, coming in Windows 8.

Tablet-friendly interface: Been in Ubuntu since 10.10, coming in Windows 8.
