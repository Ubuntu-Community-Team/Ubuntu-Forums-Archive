Oops, forgot this part:

> Secondly, how do I remove the password prompt for everything I do? Every time I install something, there's a password prompt, I'd like to remove that if possible.

I understand that it can be annoying if it happens too often, but you really, really don't want to totally disable password prompts. They are there for your system's security...and for your peace of mind. 

I'd recommend reading bodhi.zazen's "Ubuntu Security" sticky in the [security forum]("http://ubuntuforums.org/showthread.php?t=510812") to learn more.
