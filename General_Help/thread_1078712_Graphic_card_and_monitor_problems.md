### Post by Yellow Pasque on 2009-02-23
You'll have to use the Nvidia drivers from their website (the ones included with Ubuntu 7.10 probably aren't new enough to work with a 9500GT).

The easiest way is to use a program called 'Envy', which can be downloaded here: [http://albertomilone.com/nvidia_scripts1.html](http://albertomilone.com/nvidia_scripts1.html)
