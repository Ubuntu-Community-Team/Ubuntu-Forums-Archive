### Post by myk.dinis on 2010-01-12
Okay, in order to make your own cycling wallpaper.

1. Alt + F2 (Run Command) or Terminal -> gksudo nautilus /usr/share/backgrounds/cosmos
2. edit background-1.xml
3. replace absolute paths with path to pictures directory
4. replace picture names with your own picture names
5. Save!

If you copy the pictures to a directory in /usr/share/backgrounds be sure to change the permissions on the pictures to others:read

Enjoy!
