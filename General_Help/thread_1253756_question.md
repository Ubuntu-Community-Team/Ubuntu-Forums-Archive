### Post by x33a on 2009-08-30
just open place -> computer. select the partition which contains your music files. 

open it with you favourite music player (xmms, amarok, rhythmbox, exaile etc. are good options for linux).

note: linux doesn't have names like C-drive, D-drive, etc.

for more information on partition management:

[http://ubuntuforums.org/showthread.php?t=283131](http://ubuntuforums.org/showthread.php?t=283131)
