### Post by gunksta on 2008-02-05
Disclaimer - I'm writing this from a Windows box but it should be the same on Linux. I'll check when I get home.
[Edit] I use T-Bird on Windows and used it as my reference for this. [/EDIT]

It's probably easier to just use T-Bird's built-in writing tools, rather than OpenOffice.org's stuff.

1) Start T-Bird
2) Write a New Email
3)Menu -- Insert ->Image

There will be 4 tabs in the dialog box. The 3rd tab is where the money is hiding.

Appearance Tab

On the right you will see an area labeled: Alight Text to Image. Alighn the text how-ever you desire.

Hit OK.

You are good to go.

If you have something you are currently working on that already has the images in it, you can double click on the image to pull up the same dialog box. Either way it should work for you.

For writing a graphics intense email, just use T-Bird's authoring tools. I think using OOo or a dedicated HTML tool is over-kill.
