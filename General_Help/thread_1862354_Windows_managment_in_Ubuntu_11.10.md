### Post by Canaryguy on 2011-10-16
Hello,

Gnome3 and Kubuntu as far as I've seen have the possibility of seeing all the windows in miniature when you press the Windows key on your keyboard in the case of Gnome3 so it is really easy to go to the window that you want. Alt-Tab is ok but no so comfortable.

Is there any way of getting this in Unity. This is the feature that I miss a lot in Unity since working with several windows can be a mess.

Thanks.

---

### Post by Larkspur on 2011-10-16
You can get the windows-in-miniature effect in Unity by pressing the Windows key and W at the same time.  Alt-tab works too.
