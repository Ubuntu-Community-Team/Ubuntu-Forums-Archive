### Post by Fingers &amp; Thumbs on 2008-06-26
There are no theme installed by default, you will have to download from gnome-look.org.
 
You will be prompted to either open with emerald or save. Tell it to save to your desktop and then use the import feature of emerald.

There are so many themes available if they were in a repo you would have to install them all just to find out what they looked like, which would be impractical.
