### Post by 3rdalbum on 2010-02-22
> **phydeauxx said:**
> Hiya, so I have a problem. I'm an Ubuntu new-comer, and I managed to get ubuntu installed completely. Sadly enough, I removed everything from XP. And now, I wanna go back to XP. I have an installation CD a friend gave me; an XP installation ISO of sorts on a DVD-RW.

So it's a ISO file sitting on a DVD? That's not going to boot your machine. You need to use the "Burn image to disc" function of your burning software to actually make it properly.

Copy the ISO file to your hard disk in Ubuntu, then remove the DVD. Right-click on the ISO and then go to Burn Disc Image. The resulting disc should boot your machine.
