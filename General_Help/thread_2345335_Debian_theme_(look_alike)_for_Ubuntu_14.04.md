### Post by tea for one on 2016-12-03
Your image is Gnome Shell (as opposed to Unity) but I'm not sure if it is the default theme.

You can install it as follows:-

```
sudo apt install gnome-shell
```

If you wish to explore other themes, have a look here:-

[https://www.gnome-look.org/browse/ord/latest/](https://www.gnome-look.org/browse/ord/latest/)

### Post by speartip on 2016-12-03
If you're talking about the icon theme. It looks to me like it's just the bog standard gnome icon theme.
```
sudo apt-get install gnome-icon-theme gnome-icon-theme-extras
```
