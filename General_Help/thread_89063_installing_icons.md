### Post by varunus on 2005-11-14
Installing icons shouldn't be too hard.

Get icon sets from [www.gnome-look.org](www.gnome-look.org) and save the icon theme file somewhere.  As an example, I got the one called tango and now have a file called 31261-Tango-icons-0.4.0.tar.bz2 on my Desktop.

Then, go to System-Preferences-Theme.  Click "install theme" and then select the file you just downloaded and hit ok.  After a bit, you'll get a message saying "Icon theme Tango has been installed."

Then, to select it, go to "Theme Details," click on the icon tab, and choose your icon theme.  You can pick interface and window border themes here too, and those are installed in the same way.

Almost every theme works with this installation procedure; do you have one that isn't?  What is it?
