### Post by pauna on 2009-05-30
If you're comfortable with a web based tool instead of pure desktop application, check out Remember the Milk (i.e. RTM) at
[http://www.rememberthemilk.com/](http://www.rememberthemilk.com/).

You can access the todo-lists in RTM with multiple tools (browser, Google sidebar gadget for Windows, email etc). For ubuntuists an interesting feature is that you can use RTM lists in your conky display (tool is here in the forums).

A desktop application called Tasque can also interface with RTM, although it's not as quick and easy I'd like.
