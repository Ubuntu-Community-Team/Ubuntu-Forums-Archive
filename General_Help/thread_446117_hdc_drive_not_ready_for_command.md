### Post by MoLE on 2007-05-23
"drive not ready for command" suggests a problem with the drive itself, or the cabling.  Try swapping the IDE cable and checking the master/slave configuration is correct.  If that doesn't help, then check the hard drive with a program like badblocks - you can run this from the ubuntu liveCD or any of the major bootable CD distros (like knoppix).
