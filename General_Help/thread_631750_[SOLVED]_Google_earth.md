I know there is a strong windows mentality of going to a website downloading something, running it, and then having new software installed. This method does work under ubuntu, BUT ubuntu has a package management system and you really should try and get all your software through repositories, and failing that as Debs (debian install packages).

### Post by PeterJS on 2007-12-04
That warning is what happens when you try to manually install software that you could install via synaptic. It looks like you set up medibuntu at some point. You could use those debs and ignore the warning, it should result in the same version being installed. Since it popped up you could also open synaptic and search for and install google earth from there, in fact that's what I would recommend.
