### Post by ajgreeny on 2012-08-17
Of course you can.  You just need to make sure that the external drive is mounted to the same mountpoint folder every time because the path to the files in the playlist will not exist if it has changed from the time when the playlist was made.

Just go to **Playlist ->New Playlist** in the menu, then use the + in the audacious toolbar and navigate to the files to add them and then use **Export Playlist**, not **Save all playlists**.

If you should start the computer without the external drive mounted and attempt to play the playlist error messages will show
