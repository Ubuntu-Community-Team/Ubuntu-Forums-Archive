### Post by accooper on 2009-03-22
A couple of days ago I asked if anyone knew how to auto mount hard drives.  I figured it out and want to share with the group how I did it.

I went to add/remove programs and added a program named disk manager.  This allowed me to chose which drived I wanted to auto mount.

If there is a short coming of this it is a slower boot up time.

Hope this helps some of you with the same problems I had.
