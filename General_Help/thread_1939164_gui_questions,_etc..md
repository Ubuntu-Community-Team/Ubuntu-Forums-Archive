### Post by sudodus on 2012-03-11
I suggest that you download the iso files of the other flavours of Ubuntu with other desktop environments and try them live, booted from the install CD or USB drive.

- Kubuntu with KDE
- Xubuntu with XFCE
- Lubuntu with LXDE

It is also possible to add another desktop environment to an existing installation, for example
```
sudo apt-get install xubuntu-desktop
```
```
sudo apt-get install lxde
```
or (it is powerful but big)
```
sudo apt-get install kubuntu-desktop
```

Then you can select desktop environment at the login screen.

Finally, you can run gnome3 shell instead of Unity and you can tweak Unity within the 'vanilla' flavour.
