### Post by QwUo173Hy on 2012-05-19
Can anyone tell me how I can remove the second entry in the side pane? (see attachment)

The fstab entry I use for it is:
```
UUID=ed08278c-e181-445b-8312-52e083cfd764 /media/Video    ext4    auto,user,rw            0 0
```The second entry in the side pane gives an error:
> mount: /dev/sdc1 already mounted or /media/Video busy
mount: according to mtab, /dev/sdc1 is already mounted on /media/Video
when clicked.

---

### Post by vishnu_sresht on 2012-08-17
This seems to be a very old bug. According to [this Ask Ubuntu Post]("http://askubuntu.com/questions/130502/in-nautilus-i-have-two-identical-entries-in-my-devices-list-why") and [the bug report]("https://bugs.launchpad.net/gvfs/+bug/442130") referenced in that post, the solution is to use:

```
/dev/disk/by-uuid/ed08278c-e181-445b-8312-52e083cfd764 /media/Video    ext4    auto,user,rw            0 0
```
instead of

```
UUID=ed08278c-e181-445b-8312-52e083cfd764 /media/Video    ext4    auto,user,rw            0 0
```
