### Post by makariolewis on 2009-09-06
How did you try to set up Japanese characters? If you go to System > Administration > Install/Remove Languages, and then install the Japanese language pack, you should get everything.

By default, I believe you use Ctrl+Shift to switch back and forth between Japanese (anthy) and English. If you just type in hiragana or katakana, and hit space, it should automatically convert to kanji.
