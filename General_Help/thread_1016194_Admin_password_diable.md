### Post by lykwydchykyn on 2008-12-19
You shouldn't need it every time you do something.  You should only need it every time you do something that requires administrative privileges, such as installing software/updates or editing system-level configs.

Part of the security model of Linux is to make a clean separation between administrator and normal user, and to use administrative powers for as little as possible.  Get rid of this and you make Linux unsecure.
