### Post by Zacaretas on 2009-01-09
Hello! Sorry to dig this up, but I had this bookmarked and just got around to posting a reply (:

I ordered a tx2z a few weeks ago it should arrive here next week. I'll give you the configurations that I chose for mine, pinto531.

```

HP Tablet PC tx2z
Operating system:
Genuine Windows Vista Home Premium with Service Pack 1 (64-bit)

Processor
AMD Turion(TM) X2 Ultra Dual-Core Mobile Processor ZM-82 (2.2GHz)
I went with the AMD Turion X2 Ultra processor because in comparison to the 2.1GHz CPU the former has more cache.

Display
12.1" diagonal WXGA High-Definition HP LED BrightView Widescreen (1280x800)w/Integrated Touch-screen

Memory
4GB DDR2 System Memory (2 Dimm)
HP's website has a deal where you can upgrade to 3GB for free. I decided to go with 4GB of memory for $50.00 more through their website. I just now realized that I could have gone with the free 3GB upgrade and then purchased 4GB of OCZ laptop memory from Newegg for $44.99 - it's only $5.01 of savings, but nonetheless that's $5.01 more in your pocket and 3GB of memory for another laptop. O, woe is me (:

Graphics card
ATI Radeon(TM) HD 3200 Graphics with 64MB Display Cache Memory

Hard drive
320GB 5400RPM SATA Hard Drive
I went with the free 320GB hard drive upgrade at the time.

Personalization
Webcam + Fingerprint Reader with HP Imprint Finish (Reaction)
I went with the fingerprint reader option for no particular reason. In my opinion the software required to run the fingerprint reader negates security all together, since it stores the passwords within the system itself. Nonetheless it's an interesting feature and I figured why not ... (:

Networking
Wireless-N Card with Bluetooth

Primary DVD/CD drive
SuperMulti 8X DVD+/-R/RW with Double Layer Support
From my experience I have yet to come across a LightScribe device that I liked. I do think that the concept is rather cool; however I opted to not go for it this time around.

Primary battery
6 Cell Lithium Ion Battery
I went with the 6 cell battery. On the telephone the sales rep stated that I could purchase an 8 cell battery and later downgrade to a 6 cell if I didn't like it; however he said that I could not purchase a 6 cell battery and then upgrade to an 8 cell since they don't sell the 8 cells "aftermarket." I'm not sure how valid his statement was, but I went with the 6 cell. The 8 cell looked rather big to me.

```

I plan on removing the default Windows Vista Home Premium 64-bit and running Vista Ultimate 64-bit with a separate partition for Ubuntu 8.10 Intrepid Ibex. My main concerns are if the N-Trig technology will work using Ubuntu.

Is anyone else using this tablet PC with Ubuntu? If so, how is it? (:

### Post by glurgle on 2009-01-22
fglrx does support rotation, tablet mode works great, but there's is no automatic detection of screen orientation. I made a little tray applet to rotate the screen and stylus, but it's based on wxpython which is a pretty heavy dependancy for such a little tool.

There is no multi-touch support yet. It will require some work on the kernel drivers as well as linux wacom or maybe even a new xinput driver. I would like to be working on this more, but school and other projects are a priority for the moment.
