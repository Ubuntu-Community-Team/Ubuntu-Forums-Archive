### Post by him610 on 2017-06-19
Hello kolin6,

There are two versions of the I-9,pdf file - one is an interactive file where one can fill in the blanks and then print it out. Adobe requires some secret sauce to use it, and Linux users are not allowed to use the secret sauce (trade secrets, DMCA, etc.) 

The other version, *i-9-paper-version.pdf*, is the one you will probably need to use. Download it, print it out, and fill it out the old fashioned way using a pen with black or blue-black ink, or indelible pencil as was required when I was in the Army.

Good luck,
