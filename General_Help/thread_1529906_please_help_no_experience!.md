### Post by fiver22 on 2010-07-12
1) Get into your BIOS and make sure it is set to boot from CD. Then try your Windows CD again
if that doesn't work
2) Try a different bootable CD (you could use an Ubuntu Live CD) again, after confirming that your PC is set to boot from CD.
3) If it's still not working the CD Drive is likely not working and will need to be replaced/repaired.
