### Post by Zill on 2008-11-17
Outlook is a Microsoft email application for use with Windows.  The default Ubuntu email application is Evolution.  As far as I know, these two applications do not use common data files such as outlook.pst.

However, if you install thunderbird on both OS's you should be able to use common data files for both Windows and Linux.
