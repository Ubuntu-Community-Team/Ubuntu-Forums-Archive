1.Could no longer use nautilus to view a file folder and see nice thumbs of the pics. Now they just show an icon, which looks like a generic old fashioned slide, for all of them. Formerly they would all show a nice small thumb of the actual pic.
2.Though I would try F-Spot (have not used it before, was happy with the thumbs in Nautilus). It can not see my NAS even though it is mounted.

Questions:

1.How can I get my thumbs back in Nautilus when my pic files are on my NAS?

### Post by ubersolid on 2009-06-15
Nautilus: edit -> preferences -> preview. Set show thumbnails to always.

---

### Post by fisheater on 2009-06-15
Success! THANKS!

Ok here is what I did:

Nautilus -> edit -> Preferences -> Preview -> Other Previewable Files -> Show Thumbnails [ALWAYS] & Only for Files Smaller Than [100 mb].
