### Post by Bubba64 on 2008-05-30
Nightly tester tools will force add ons to work.
[https://addons.mozilla.org/en-US/firefox/search?q=nightly%20tester%20tools](https://addons.mozilla.org/en-US/firefox/search?q=nightly%20tester%20tools)
Also some add ons can be found outside of Mozilla (Google Search) add ons and will install if you have the aforementioned add on installed.
