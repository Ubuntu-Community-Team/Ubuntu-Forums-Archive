### Post by RuleMaker on 2008-07-27
Installation depends on theme type,
GTK, Icons, Metacity, Compiz, Mouse Pointers and similar are installed by dragging the downloaded theme on System&#8594;Preferences&#8594;Appearance

If you are trying to install GDM theme you'll have to drag it on:
System&#8594;Administration&#8594;Login Window&#8594;Local
