### Post by jbfoley on 2010-07-24
Try EasyTag, it's in synaptic, or sudo apt-get install easytag-aac

The interface takes some getting used to, but it's easy and fast once  you get the hang of it.  It lets you search CDDB for tags for multiple  files at once and tag them, similar to what winamp does.
