### Post by Roasted on 2010-01-22
PCMan and Dolphin both have the ability to set the group user ID and whatnot in the permissions menu. Is there a way to allow this for Gnome? It's -very- nice to have...

---

### Post by John Bean on 2010-01-22
If you mean in Nautilus, then the answer is yes. Run gconf-editor and browse to apps/nautilus/preferences and check the show_advanced_permissions setting.
