### Post by Foxcow on 2011-06-29
I'm not sure if you can install it to a DVD.  As far as I know, you need a CD-R.  Check out the main website for installation options.

First of all, you can try Ubuntu in the "Live CD" mode.  You can run the OS from the CD so you can get a firsthand look at Ubuntu on your computer.  If you like what you see, I recommend that if you have the drive space, install Ubuntu side-by-side with Windows.  That way, you can bounce between Ubuntu and Windows as you need or choose to.

Later on, you may decided to wipe Windows completely.  Backing up your info to to an external drive and Resizing the Ubuntu partition is easy enough.

Good luck!

"The combined install/live DVD allows you either to install Ubuntu permanently on a computer, or (by entering 'live' at the boot prompt) to try Ubuntu without changing your computer at all.

PC (Intel x86) install/live DVD
    For almost all PCs. This includes most machines with Intel/AMD/etc type processors and almost all computers that run Microsoft Windows, as well as newer Apple Macintosh systems based on Intel processors. Choose this if you are at all unsure.
64-bit PC (AMD64) install/live DVD

Windows will not affect the speed of Ubuntu in any way. Once you install (instead of running "live") it will be faster. :)

The [recommended minimum]("https://help.ubuntu.com/community/Installation/SystemRequirements") for Ubuntu is 512mb ram and 5gb hard drive space (plus whatever space you need for documents of course!)

How big is your hard drive and how much room do you have to spare?

Can you get another 512mb or so of RAM? That would help the speed a lot.

You could also try Xubuntu or Lubuntu, they use less RAM.
