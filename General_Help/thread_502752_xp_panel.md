### Post by zanglang on 2007-07-17
The panel colors isn't controlled very well by Gnome themes I think, you'd have to manually change it from the panel properties windows (Right click panel -> Properties). Check your downloaded theme and see if it comes with any specific instructions to change the panel's appearance.

### Post by mcduck on 2007-07-17
The panel color follows your application theme (GTK2-theme), not window theme (Metacity, Compiz, Emerald or whatever you are using).

Anyway, you can just set it to whatever color you want, or use any image you want as panel background.. (even transparency in PNG and SVG images works so you can create pretty amazing panels if you want..)
