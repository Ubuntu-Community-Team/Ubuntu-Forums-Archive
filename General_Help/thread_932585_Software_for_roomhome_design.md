### Post by nsche on 2008-09-28
xfig is reasonably useful for what you want.  It will not put in dimensioning but it will draw to scale and has a grid for when you want that.  It is a simple 2d drawing program.  The user interface is a little different from what you may be used to as it was developed quite a while ago.  It is available in the repositories. 
Install by "sudo apt-get install xfig" or use what ever other tool you use for installs.
