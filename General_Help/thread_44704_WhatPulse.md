### Post by christhemonkey on 2007-01-28
Ok, here is how i did it.

Install wine (Howto here: [http://www.winehq.com/site/download-deb](http://www.winehq.com/site/download-deb))

Then download the client from here:
[http://whatpulse.org/downloads/10/](http://whatpulse.org/downloads/10/)

Then open a terminal and type this:
```
cd
cd Desktop 
wine WhatPulse-1.5-Setup.exe 
```

Then complete the install and register and enter your information and you should be sorted!


EDIT: just to say, it doesnt seem to count my keystrokes, but it does count my clicks.
And at some point there will be a linux client!: [http://whatpulse.org/kb/idx.php/7/017/Linux-client/article/Where-is-it.html](http://whatpulse.org/kb/idx.php/7/017/Linux-client/article/Where-is-it.html)
