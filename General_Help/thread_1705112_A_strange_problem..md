### Post by grahammechanical on 2011-03-11
I have more than one version of Ubuntu installed. When I want to access the files on the other partitions I go to Places and I select Home Folder and in the left panel I see disc drive icons that relate to those other partitions. They are called Filesystem and are given a size in Gigabytes. I select the partition I want and then I select home and then my user name folder.

I can also do this the other way around. From one of the other installations into my standard Ubuntu home folder. You should be able to do the same with pinguy.
