### Post by JangMunho on 2008-05-20
Just double the second section of your boot.ini, then you'll get a boot menu for Windows loader. While the menu shows up, you can press F8 to get into safe mode.

---

### Post by FuturePilot on 2008-05-20
Select Windows from Grub, and then **immediately** start pressing F8.
