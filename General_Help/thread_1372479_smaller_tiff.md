### Post by kaibob on 2010-01-04
> **loki_dre said:**
> how do I compress, or reduce the resolution, of a tiff image file

NOTE: I must use tiff I can not convert to JPEG etc.
The easiest solution is the Nautilus Image Converter, which is an extension for the Nautilus file manager. It's available in the repo's. 

[http://www.bitron.ch/software/nautilus-image-converter.php](http://www.bitron.ch/software/nautilus-image-converter.php)

The standard command-line tool to resize image files is Imagemagick convert:

[http://www.imagemagick.org/script/convert.php](http://www.imagemagick.org/script/convert.php)
