### Post by polly1 on 2006-10-11
Hi gletare

Go into System>Preferences>Keyboard>Layouts and change the keyboard to the
model that you use, again. Do not press the "reset to defaults" button and close preferences. To reset, will probably revert the keyboard to the "generic model" layout.

Click System>Preferences>Keyboard Shortcuts heading and open up the Desktop
menu. Then examine each "Shortcut" code against the "Desktop" heading to
check that "space" does not appear instead of a code number.     

For Example "Home Folder" code should read Oxef. To correct, assuming that "space" does appear as a code - highlight the incorrect line with your mouse,and then press the actual shortcut on your keyboard itself to replace "space" with the correct code. You may have to press twice to  correct the position. Close down the keyboard shortcut and you should find your problem solved.

It worked for my keyboard, which is a Microsoft Multimedia model.

It is easy to press the space key instead of the keyboard shortcut key
when inserting the shortcut keys. This is how I solved the problem with my
space key problem, although in my case my home folder opened when space was pressed
Cheers
