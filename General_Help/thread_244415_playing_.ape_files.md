### Post by acl123 on 2008-05-16
Morgoth has taken xmms-mac out of the repositories since Gutsy, but you can install it by adding the edgy repository. It's probably best to remove it once you've installed xmms-mac - just so you don't accidentally install any other edgy stuff.
I can't say how stable it is, but it seems to work with the couple of ape files I have.
