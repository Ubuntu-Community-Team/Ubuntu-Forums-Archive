### Post by fragos on 2007-01-05
The slowdown occurs when DNS is accessed.  First IPv6 is tried.  After a timeout period IPv4 will be tried to access DNS.  Since DNS returns an IP addrees the trial and error test doesn't occur when accessing the site.  Using both accomodated a mixed IPv4 and IPv6 network transparently to the user.
