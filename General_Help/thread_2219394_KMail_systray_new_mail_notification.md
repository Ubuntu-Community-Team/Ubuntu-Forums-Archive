### Post by N1GHTS on 2014-04-23
I have recently installed a fresh copy of **Kubuntu 14.04**. After importing my mail from a previous installation of Kubuntu and configuring the email settings, I noticed that when I receive new mail the systray does not show the mail icon when a new message has arrived. I have also tried a plasma widget called "Incoming Message" which is supposedly supposed to indicate when a new message arrives but it too does not notice the new messages. 

Does anyone know how I can configure KMail to show an indicator on the systray like it used to do in 13.10?

---

### Post by N1GHTS on 2014-04-24
I figured out how to fix this myself. 

**Settings Menu** -> **Configure KMail...** -> **Appearance** left vertical tab -> **System Tray** tab -> **Enable system tray icon** checkbox

I guess it was turned on by default before but it was not turned on by default this time. It really ought to be the default in my opinion.
