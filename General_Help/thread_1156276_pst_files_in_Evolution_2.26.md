### Post by HippieDave on 2009-05-20
I'm setting up Ubuntu 9.04, coming from Outlook running in XP.

As far as I can tell, 9.04 still won't take the Outlook .pst file directly.  However -- and this works far better than the old method of exporting outlook into Thunderbird and then importing Thunderbird files-- you can Export files from Outlook into a Tab Delineated (Windows) file (that's one of the choices you get under Export). Save that to somewhere you can find it in Ubuntu and then import that as a Tab delineated Outlook file into Evolution.
