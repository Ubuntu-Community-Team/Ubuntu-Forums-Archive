This problem has happened while trying to: 
[LIST]
[*]open the print menu in firefox 3.0.3
[*]open the print menu in adobe acrobat reader 8.1.2
[*]open the Printer configuration menu (in a terminal, it's the system-config-printer command)
[/LIST]

The problem is independent of the default printer that is selected at the time. It makes no difference if there is a printer connected to the computer or not, or if a remote printer is being used, the problem arises anyway. It even happens while using the PDF Printer as default.  

Anyone has any idea of what could be happening here? Any log file that could provide info about the issue? Another thread describing the same problem or a similar one?
