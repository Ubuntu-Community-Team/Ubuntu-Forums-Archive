### Post by Topsiho on 2007-01-31
I really can not tell you what exactly is on the Ubuntu DVD :) but I want to react to your remark "and Ubuntu falls a little short here with only one CDs worth."
I have tried many other distro's such as RedHat, Fedora, Mandrake and such, coming on 4 or more CD's, containing a lot of software, much more of course than on the one CD of Ubuntu. But then: Ubuntu is the first one that gets my internet connection right during the installation itself, and downloads the necessary packages which are not on the CD.
In stead of the CD the repositories serve as the place where Ubuntu downloads every package that you need, AND those packages are the up to date ones, which you don't get when packages must be fetched from the CD, especially when that CD is some months (or more) old.
Using the repositories is MUCH easier (I did not mention all the troubles connected with dependencies if you use CD's) than using CD's.

In my opinion you do not need the DVD, especially not with a fast internet connection.

Topsiho

### Post by Topsiho on 2007-02-02
The problem I was referring to, in my previous post, is this:

One installs Linux from one or more CD's of a distribution.
One installs the software one needs from these CD's.
One applies the necessary security updates, and maybe some other ones.
Then one decides to install some more software, from the CD's, and then the trouble starts:
The new software may need one or more dependencies that were installed from the CD's but in the meantime are updated (and the version changed)!! Which makes the install impossible unless one, manually, undoes all the affected updates, or installs the distro again from the start. With all the software on CD's or DVD one has to decide which software one needs, and after the first updates (from the internet) one possibly cannot, never, install new software!!

With Ubuntu using the always up to date repositories this problem does not exist, and possibly is not even known by most Ubuntu users.

BTW: with my fast internet connection I can download as many bytes as I wish for not a penny extra. I do not know about the costs for Canonical, but I am very grateful to them for everything they do for our community. As a translator for KDE I feel I do something too :)

Topsiho
