### Post by MAFoElffen on 2024-02-04
I may be wrong (someone please correct me if this is so), but I think I remember change i how Nautilus (now Gnome Files) works and what the user can change...

It used to be more independent and you could change specific things in the application itself. In time, to streamline maintenance, more and more things were inderited from the Gnome Desktop. If you look at the application Preferences, there is little left there that affects the look-and-feel for customization's.

Fonts and font color is not there. It is inherited from the Gnome Desktop Theme. So change the Desktop theme, and it changes how it is inside Gnome Files.

There is a CSS file for Gnome, but I do not know if changes there can specify to change only for the Gnome Files application. i believe those changes are still global. I do not know for sure, but I believe that is true now, currently.
