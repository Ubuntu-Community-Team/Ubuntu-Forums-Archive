### Post by thnewguy on 2012-11-05
The current version of Ubuntu runs terribly slow in VirtualBox because of its 3-D requirements. You can try switching desktop environments by installing the gnome-fallback package. If you're getting slow performance on physical hardware, then make sure you have installed any third-party drivers for your card.
