### Post by bonzodog on 2007-07-01
When you run the install CD, it will get to the partition manger, and you will be shown a filesystem structure summary of both of your Hard disks.

At this point, you can choose to use the second disk for Ubuntu, or you could, if you so wished, wipe  the vista partition, and install Ubuntu there, and format the second disk for Data storage or something.

If you choose to use the second disk, then highlight that and tell it to format that drive as a Linux ext3 drive. 

Personally, I would use manual partitioning on it, and create a seperate /home partition as well as the root and swap partitions.
