### Post by Quackers on 2011-07-25
Welcome to UF.
It looks like the installer has crashed for some reason.
It isn't the best option for the Ubuntu installer to be resizing Windows partitions. That's safer done from the Windows Disk Management screen.
If you've left it for some time and there has been no improvement I think the only thing you can do is to force close the system.
But be prepared, it may have damaged Windows.
Do you have a Windows repair disc or installation disc?
