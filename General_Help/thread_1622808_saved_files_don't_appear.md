### Post by yetiman64 on 2010-11-15
> **duke.tim said:**
> I keep running into this issue that when saving (some) files that they don't appear until I logout and log back in. It's quite confusing because they appear in the terminal. any ideas

In a nautilus window at the location the files were saved try using the "Reload the current location" button, the one with 2 arrows forming a circle. This will refresh nautilus and should show your new files without the need to logout / login.

If you save to the Desktop and a file fails to show, go to Places > Desktop and use the Refresh button, then recheck on the Desktop, it should now be there.
