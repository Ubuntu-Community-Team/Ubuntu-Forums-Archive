### Post by alexfish on 2010-02-13
hi

Would need more details of the card IE: model number .or the type of modem, this can be obtained by plugging the device in (from the live cd) , then from the terminal use the Code : lsusb.

however it may work out of the box . see if any devices show in the network manager after the device is plugged in. if the device shows the try using the wizard . you will need the details from his service provider to complete the details. 

if you have a copy of Ubuntu 9.04 ,take that as well, if it fails in 9.10 but works with 9.04 then chances are it will work with Ubuntu 9.10 , after upgrading to the latest kernels

you could also try wvdial but these are not available at default; try get a copy on usb stick

there is a site here worth reading

[http://www.pcurtis.com/ubuntu-mobile.htm](http://www.pcurtis.com/ubuntu-mobile.htm)
