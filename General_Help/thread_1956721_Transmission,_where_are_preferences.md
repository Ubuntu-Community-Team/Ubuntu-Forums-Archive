### Post by SAKeeler on 2012-04-11
Open Transmission, make sure the window has focus (click on it), then move your cursor to the top left of the screen( where it says the name of the program), it should change from the title of the window to the menu where you can select edit then Preferences.

this assumes you are using Unity as the Desktop Environment.
