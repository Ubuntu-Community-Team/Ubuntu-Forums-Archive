### Post by geraldm on 2009-02-26
The USB 2.0 printer would be about 40 times slower than otherwise possible.
If the printer works in linux, it should then work. 
There were some changes in the USB standard as the standard evolved, but
no incompatible changes as far as the device is concerned.  The device is expected to be compatible with older transfer types.
