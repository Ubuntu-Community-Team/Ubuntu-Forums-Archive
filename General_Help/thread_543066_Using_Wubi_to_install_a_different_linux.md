### Post by tuxcantfly on 2007-09-04
See UNetbootin at [http://lubi.sourceforge.net/unetbootin.html](http://lubi.sourceforge.net/unetbootin.html) it supports Fedora 7; it allows you to install Fedora, Ubuntu, or Debian without a CD; the main difference between it and Wubi is that it installs to an actual partition like a normal Fedora install would, rather than a disk image.  Wubi has a lot of dependencies on Debian/Ubuntu-specific technologies, so it can't be ported to Fedora.
