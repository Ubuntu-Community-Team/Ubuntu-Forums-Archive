### Post by ajgreeny on 2014-04-22
A persistence file on a live USB is limited to 4GB simply because that is the file-size limitation of fat32.

If you can see any real point rather than a proper installation you can also make use of a separate partition which can then be increased to the size-limit of the drive used.
See [http://ubuntuforums.org/showthread.php?t=2165496](http://ubuntuforums.org/showthread.php?t=2165496) for the details.  I've never used it so can not give you any indication of how well that works.
