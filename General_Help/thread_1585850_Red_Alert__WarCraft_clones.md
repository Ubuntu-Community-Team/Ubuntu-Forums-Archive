### Post by 3Miro on 2010-10-01
There are couple of RTS games in the repos (Warzone for example) as for complete clones, no there are none. Depending on which RA you want, RA 1 and 2 will probably run under Virtual Box on a reasonable machine (RA3 will not). I have had some trouble with wine and CnC General, which somewhat odd since I can actually play Starctaft II which much newer.

I have played WC3, you have to turn off compiz while playing (you can install the Fusion icon or make a separate account), but otherwise WC3 plays fine. I have not tried WC1/2.
