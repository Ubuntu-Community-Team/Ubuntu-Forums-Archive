### Post by ntetreau on 2007-09-13
Welcome to ubuntu, I'll try to help by sending you on the right track.  Go into System > Preference > Keyboard > Layouts.

There make sure you choose the layout you are used to.  In my case, I have an British keyboard but I'm french so I need the accent.  That means I choose "United Kingdom International (with dead keys)".  Try to choose the one you had when you were using the other OS!
