### Post by Bucky Ball on 2012-03-03
> **Frogs Hair said:**
> Downgrades are not possible ...

Welcome!

A reinstall is the only way. Sorry. If you have a separate /home partition (not home folder in /) then this is no great problem. Just manual partition when you get to that part of the install (choose 'Something Else') and make sure your /home partition is not marked to format and you are installing 10.04 LTS to the partition where 11.10 is. The new 10.04 LTS install will then use the existing /home as /home and all your data will remain as was (although backing up is a good idea before doing this, nonetheless). ;)

My advice about this is always 'if it ain't broke, don't fix it!'. Unless you have some good reason to upgrade the release (it has drivers for your machine the LTS release doesn't or your machine won't work with the LTS for some reason) why bother? The latest is not always the greatest. Good luck.
