### Post by asimon on 2006-05-31
There is an easier way. Just install 'xscreensavers' and 'kscreensaver-xsavers'. Then all those xscreensavers (including Flurry) will appear in KDE's screensaver configuration dialog where you can choose them like any other KDE screensaver.
