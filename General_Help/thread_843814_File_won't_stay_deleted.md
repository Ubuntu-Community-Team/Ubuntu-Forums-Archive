### Post by sdennie on 2008-06-28
It sounds to me like whatever you used to download the iso keeps trying to download it.  If you were using a torrent, make sure the torrent isn't still added.  If you were using Firefox, make sure it's not still trying to download it and it's not on the list anywhere.
