### Post by tacantara on 2009-05-15
Since the GNOME desktop environment consists of many packages, I recommend using the Synaptic Package Manager to ensure that all packages required to run the GNOME DE will automatically install.  I entered GNOME in the Synaptic quick search, and scrolled down the list to GNOME.  When you check it (mark for installation), it automatically picks up the other packages required to make it work.
 
After GNOME is installed, log out.  This should take you to the sign-in screen that comes up when you first start the computer.  Click on Options - you should see an option to log in to a GNOME session.  Click on that, and it will take you to the GNOME desktop.  Use the same procedure to get back to KDE.
 
I started off using Ubuntu w/GNOME, and did basically the same steps to install the Kubuntu/KDE environment.  Personally, I like the look of the KDE desktop, and I actually find it easier to work in the flashy, "eye-candy" stuff on KDE.  Good luck, and let me know how you make out with this :)
