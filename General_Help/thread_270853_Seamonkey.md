### Post by jdackle on 2006-10-15
Funny how things turn around...
I left Mozilla Suite for Firefox because The Suite was bloated and slow!

But having Firefox and Thunderbird running at the same time completely blows off that point, as they duplicate a lot of common code which in Seamonkey is stored in memory only once.
A good while back ago I heard future versions of Ff and Tb would acknowledgeeach other's persence in memory and use the common code already loaded instead. But I don't know when that feature is due.
(Also from the begining of Firefox, when it exploded in popularity, Mozilla Suite fans said it rendered pages faster - I didn't benchmark that myself, but it's common saying.)

So by the time the Mozilla Foundation droped support for the Suite, I began to miss it...
So naturally I tried Seamonkey... and liked it. :)
Not much different from the suite though, dispite the "Roaming User" feature which is something brilliant IMO. And that Fff doesn't have as of yet. And of course all the security updates and other bugfixes... ;)
There's also a couple other new features when compared to the Suite, but I can't quite remember which now...

So if you plan on using Ff and Tb at the same time, you'd probably be better off with Sm. ;)

But I do miss a few handy features from Ff and Tb...
E.g. Sm doesn't have a separate search bar, which in itself is far better for me! I see no point in shortening the  URL/Search bar by spliting it in two (though I'd guess it's more "marketable" that way).
But what I do miss is the possibility to change the search engine instantly! I can do it yes, but then I'll have to go to the Edit menu, choose Preferences, ... In Ff I could easily change from a search engine to a dictionary and back to a search engine...
I also miss good extension uninstaller/iupdater. I have the ExtensionUninstall extension installed, but it doesn't always work. Also no way to automatically look for extensions' updates.

Regarding the extensions available to Sm, that is where I felt the "lack of support" most.
Some have been discontinued, others were never made available to Sm/Mozilla Suite. And to make things worse, the official Mozilla addons site will just throw in a bunch of Ff extensions in the list when you specifically do a search under Mozilla Suite extensions. You'll have to go to the "Advanced search" and specify Mozilla extensions only to overcome that.
And many of the older extensions' sites like Extension Room have practically been abandoned out there. :(

I particularly miss having a good tab-browsing preferences extension, which pretty much makes me have none. And this means, for instance, I have no way to revert focus to the last selected tab instead of the standard lef-right order. A feature I really miss. But I haven't looked for any new one in quite a while though...
From Tb I specially miss the Webmail extension that allowed me to get mail from my Hotmail account. Though I rarely use that account anyways...

As to stability, mind you:
1. I am in no way the apps-comparison-expert type!
2. I have a really slow machine: a Pentium III@450MHz, 512Mb RAM...

That said, I've had Sm virtually freeze my desktop when I have 3-4 open windows with some 10-15 open tabs each... Going to a text-console and killing Sm is the only way if you don't want to hang for half an hour, maybe one... (I have the SessionSaver extension in, so that's alright, and curiously it doesn't hang right away when I restart all those navigator windows + the Mailboxes window, unless I keep pushing it... ;))
Crashes I've had about one every other week, but I had them with Ff at about the same rate too...

Regarding additional themes, there's a good bunch around you can find. A bit of the same problems you'd have with finding extensions, but they're around and they look good. :)


So, basically, my advice would be: if you want to have Ff and Tb running at the same time, you'll probably be better off with Sm.
Note I haven't installed Chatzilla or the development tools, which sould make it lighter... ;)
If you just want the browser I'd advise you to install the Navigator part only (no Mail, no IRC, no development tools, ...) and test it for yourself. ;)

Hope that helps.

### Post by FastZ on 2006-10-17
I have SeaMonkey installed on Dapper but I dont use it that often.  I was using it on my Windows machine as a default browser until the Firefox 2.0 builds started surfacing.  I use Evolution for my email and Firefox 1.5.0.7 on my Linux machine and I dont really notice any memory problems.  Although I have 2Gb of memory on all my computers, I best not notice any problems with having "memory hogging" programs running.  SeaMonkey is a great program and I really liked how it includes a browser, IRChat client, mail/newsgroup client and a small html editor program.  If you dont like having to open more than one program to use one or more of those features, SeaMonkey is the way to run.
