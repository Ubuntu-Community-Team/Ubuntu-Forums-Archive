Possibly could be a resource issue, check what you have running at the time of the crash. 

You can view your processes running by typing:
```
top
```

The second thing you can try is downloading a program called "Conky" which tells you your CPU Status 24/7. You can install this by typing:
```
sudo apt-get install conky
```

The third thing you can try is **completely** uninstall Opera and see where that gets you. 

Hope this helps. :D
