### Post by sudodus on 2022-08-12
You can try this way: Install Ubuntu Server, boot into it and then install gnome-shell,

```

sudo apt update
sudo apt install gnome-shell

```

I have installed other desktop environments for example lubuntu-desktop into Ubuntu Server, but I have not tried with gnome-shell. However, I think it will bring a lot of packages necessary for the graphical desktop environment, and it should be enough to make things work.

Let us know how it works, whatever the result. And of course, if there are problems, there are many people here who can help you ...
