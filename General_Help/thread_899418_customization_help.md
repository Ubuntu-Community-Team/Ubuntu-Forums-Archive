### Post by unca.paka on 2008-08-24
Grab StartupManager, its in the repos. Drop this into a terminal,
```
sudo apt-get install startupmanager
```
or search for it in Synaptic.

Itll let you install new Usplash themes/grub themes.
Careful while using it though, you could screw up grub.
