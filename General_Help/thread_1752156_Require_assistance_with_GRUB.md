### Post by wilee-nilee on 2011-05-07
If you want all the OS's to show in a grub menu make sure sda6 is the controlling grub.

The script shows the mbr to be looking there.
```
 Grub 2 is installed in the MBR of /dev/sda and looks on the same drive in 
    partition #6 for (,msdos6)/boot/grub.
```

Lets reload it and see if this is really working. If you can manually boot into the sda6 partition run these two commands.
```
sudo install-grub /dev/sda
```
then 
```
sudo update-grub
```

If you want to just do this from a live cd, use a live c Ubuntu cd equal to the sda6 install and follow this link.
[https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD](https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD)
