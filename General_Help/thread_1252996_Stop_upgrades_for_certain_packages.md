### Post by sr_guy on 2009-08-29
Running 8.10 Intrepid here. Is there a way to stop ubuntu from updating certain packages? Everytime ubuntu upgrades the Nvidia-glx-177 package to v180 gdm won't start after boot, and the nvidia drivers won't reactivate.

---

### Post by scouser73 on 2009-08-29
Go to **Synaptic Package Manager**, locate the package that you don't want to upgrade, highlight the package, then click **Package** and select **Lock Version**.
