### Post by por100pre1 on 2007-09-24
**Edit> Preferences> Local Capture**

**Change Directory for captured pics:** to:

```
/home/*your-user-name*/webcam_pictures
```

or wherever folder you want to use in your user folder.

(replace *your-user-name* with your actual user name)
