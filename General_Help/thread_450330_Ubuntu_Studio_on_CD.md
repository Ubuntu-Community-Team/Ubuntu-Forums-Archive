### Post by rocket777 on 2007-05-22
If there is, it will have to be about 167 megs smaller. I believe the current iso is 867megs. I haven't seen a studio subset for a CD.

The only other way to use the iso w/o a dvd would be inside a virtual machine, like vmware or Virtual Box. I wrote a dvd and also used it in virtual box, but I found that very slow.
