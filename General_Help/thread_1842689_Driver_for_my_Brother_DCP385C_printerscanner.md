Hello malel

Open a shell console and run theses commands to download the drivers :
```
wget http://pub.brother.com/pub/com/bsc/linux/dlf/dcp385clpr-1.1.2-2.i386.deb
```

You should have now all the files needed to install your printer in your home folder. These files are **dcp385clpr-1.1.2-2.i386.deb**, **dcp385ccupswrapper-1.1.2-2.i386.deb**, **brscan3-0.2.11-4.i386.deb** and **brscan3-0.2.11-4.amd64.deb**.

Further informations on how to install the drivers here : [http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/index.html](http://welcome.solutions.brother.com/bsc/public_s/id/linux/en/index.html)

### Post by demonipuch on 2011-09-13
To install the printer drivers for 64 bits architectures, open a shell console and run :
```
sudo apt-get install ia32-libs
sudo aa-complain cupsd
sudo mkdir /usr/share/cups/model
sudo dpkg -i --force-architecture dcp385clpr-1.1.2-2.i386.deb
sudo dpkg -i --force-architecture dcp385ccupswrapper-1.1.2-2.i386.deb
```

### Post by demonipuch on 2011-09-13
Depending if your scanner is connected via USB or network you should do :

**USB**

make sure the package sane-utils is installed :
```
dpkg -l sane-utils
```
should return something like 
```
ii  sane-utils     1.0.20-13ubunt API library for scanners -- utilities
```
if not installed, do 
```
sudo apt-get install sane-utils
```
add a rule to udev :
```
echo -e "# Brother\nATTRS{idVendor}=="04f9", ENV{libsane_matched}="yes"" | sudo tee /etc/udev/rules.d/60-libsane.rules
```
restart udev :
```
sudo service udev restart
```

**Network**

add a network scanner entry :
```
brsaneconfig3 -a name=SCANNER model=DCP-385C ip=scanner_ip
```
where scanner_ip is the IP Address of your scanner
