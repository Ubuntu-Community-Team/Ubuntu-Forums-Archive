### Post by Sef on 2007-11-02
Run a mem86 test.  I have a similar problem with computer, by doing a mem86test, I have figured out that my memory module on my motherboard is bad.   You may have the same or a different problem.

To do the test, go to GRUB (push esc upon start up), and go down to mem86test.   You need to let it run overnight (or about 8 hours or so.)   It repeatedly tests the memory.  The first time or few times, may not pick anything up.
