### Post by morty35 on 2008-10-28
I was wondering if it was possible to have another user who I trust completely be able to view my desktop while I was working on a project.  I have a business partner, and it would really help if I could show him what I was doing on my desktop as I was doing it so we could discuss our project and changes to make, etc.

---

### Post by geirha on 2008-10-28
Enable VNC with System -> Preferences -> Remote desktop. 

Then have your business partner connect to your ip with vnc, if he has Ubuntu, that would be Applications -> Accessories -> Remote Desktop Viewer
