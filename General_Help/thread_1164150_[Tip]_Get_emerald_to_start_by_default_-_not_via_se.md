### Post by MadnessRed on 2009-05-19
Most tutorials I have seen seem to get emerald on startup by puttin emerald --replace in the startup, this is a but dodgy for me and doesn't work so here is an alternative that I use.

1) Install CompizConfig Setting Manager and Emerald if you don't already have them.

2) Select Effects, then Window Decoration. and in "Command", enter emerald --replace.


This works for me where the session on startup doesn't so maybe other people will find this useful too.
