Which version of Ubuntu you have? Usually it's not advised to install Gnome-Shell over Ubuntu Unity due to conflics in configuration files and packages. Unity uses Gnome as a back end and if Gnome gets modified, Unity wont work and usually will confuse Gnome-Shell in the process causing both to fail. Might need to reload, but wait for other advice first. :)
