### Post by kanikilu on 2009-05-08
When it turns black & white, that means it's not responding for some reason. Have you installed any new addon's or plugins recently?

You might want to try running firefox in safe mode to see if the problem continues. To do this, open a terminal window (Applications > Accessories > Terminal) and type:
```
firefox -safe-mode
```
If that works, then you might need to start disabling add-ons or resetting to default settings to see if you can get it working again without safe-mode.

For more information:

[http://support.mozilla.com/en-US/kb/Safe+Mode](http://support.mozilla.com/en-US/kb/Safe+Mode)
