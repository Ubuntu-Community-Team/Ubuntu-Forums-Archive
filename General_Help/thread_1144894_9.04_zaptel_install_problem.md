### Post by eliyahud on 2009-05-14
bug report:
[https://bugs.launchpad.net/debian/+source/zaptel/+bug/353024](https://bugs.launchpad.net/debian/+source/zaptel/+bug/353024)


patch:
[http://svn.debian.org/viewsvn/pkg-voip/zaptel/trunk/debian/patches/hrtimer_2628?pathrev=6683](http://svn.debian.org/viewsvn/pkg-voip/zaptel/trunk/debian/patches/hrtimer_2628?pathrev=6683)

Save that file to the zaptel source directory, use patch to patch the file, then run 
m-a -O build zaptel
to make sure it doesn't extract over your patch. Worked fine for me.
