### Post by The Cog on 2013-08-27
There are not. This page [http://fajran.web.id/2013/05/ubuntu-repository-total-packages-and-sizes.html](http://fajran.web.id/2013/05/ubuntu-repository-total-packages-and-sizes.html) shows that the full repository (for both 32 and 64 bit architectures) is around 50GB. If you're only looking at one architecture then maybe 30GB would be a closer figure (data files will of couse be shared between the two architectures). So I guess you would be looking at maybe an 8-DVD set - it's not done.
