### Post by sumguy231 on 2007-09-27
1.) Probably not.
2.) Yes.
3.) Yes, but without video
4.) Yes.
5.) See 7
6.) Never heard of it.
7.) Not shareaza, but Limewire and a slew of other p2p apps
8.) Yes.

By the way, I know that there are at least online services that will convert Office Open XML to something sane like OpenDocument.

Also remember that you can have both Ubuntu and Windows installed with a dual boot. No need to use exclusively one or the other.

Some Windows programs you may be able to run with a compatibility program called Wine - see this page for a list of how compatible specific programs are:
appdb.winehq.org

There are also Virtual PC apps like Virtualbox and VMWare which you can use to virtualize a Windows system in Ubuntu.
