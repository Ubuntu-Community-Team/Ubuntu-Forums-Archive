### Post by 23meg on 2005-04-27
you can "modify" lots of things in linux and chances are still you'll not get viruses, trojans, etc  ;-)  

if "modify something" means logging in as root and executing possibly malicious code, you're asking for trouble. just log in as a normal user to do your daily work, and you'll most likely be safe without any extra precautions. if you're running a server or a so called "mission critical" workstation, it wouldn't hurt to use antivirus software though. and you may also want to use antivirus in an effort not to pass on viruses to other (namely windows) machines that may get affected by them.

windows and linux security are two different paradigms. very roughly speaking, in windows, as long as you can run code on someone's machine, you have the ability to do just about anything you want to him. in linux, even after you've exploited a security flaw, you need root privileges to do any serious damage.
