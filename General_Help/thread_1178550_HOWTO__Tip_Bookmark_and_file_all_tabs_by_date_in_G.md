### Post by learningcurb on 2009-06-04
This is a really small tip but one that I find quite useful.  (Maybe there should be a subforum for tips that do not constitute a full blown HOWTO.)

When you are browsing a lot of tabs in Galeon, you can do Bookmarks > Add Tabs As Folder from the menubar, but then you are required to give the folder a name to describe what's in it.

If you're lazy like me, you can open a terminal window and enter ```
$ killall galeon
``` at the command line (or kill it with htop). When you restart Galeon, it will give you the option of creating a folder with the date and time of your last browsing session, saving you a bit of work.  Would it be better if you didn't have to kill Galeon to do this?  Yes, but I haven't figured out how to do this.
