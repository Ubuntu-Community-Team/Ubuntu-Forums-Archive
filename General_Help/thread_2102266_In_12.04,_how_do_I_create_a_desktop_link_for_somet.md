### Post by stinkeye on 2013-01-06
If you have the gnome-session-fallback (Gnome Classic session)installed you should be able to drag
an application from the dash(not launcher) to the desktop.
```
sudo apt-get install gnome-panel
```


[SIZE="3"]or[/SIZE]

You could got to **/usr/share/applications** in the file browser
and copy the relevant launcher to the desktop.
If it shows as a .desktop file on the desktop, right click on it and
make it executable in properties > permissions.
