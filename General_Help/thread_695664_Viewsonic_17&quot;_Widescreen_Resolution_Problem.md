### Post by taurus on 2008-02-13
You want to use 1440x900 resolution but I don't see it anywhere in your /etc/X11/xorg.conf.

What happens if you add that resolution, 1440x900, in here?

```

Modes **"1440x900@60"** "1280x768@60" "1280x720@60" "800x600@60" "800x600@56"

```
Save it and then either reboot or restart X server again with <Ctrl><Alt>Backspace.
