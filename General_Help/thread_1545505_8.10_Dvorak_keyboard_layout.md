### Post by simosx on 2010-08-04
> **Sonja said:**
> How do I switch to using Dvorak keyboard layout (rather than Qwerty) from the command prompt (not GUI) in Ubuntu 8.10?

When you say command prompt, do you really mean the Linux console that you get with Ctrl+Alt+F1 (you return to graphics with Ctrl+Alt+F7)? Some people call this 'DOS prompt' style.

If so, then see 
[http://www.linuxquestions.org/questions/slackware-14/change-keyboard-layout-in-console-187987/](http://www.linuxquestions.org/questions/slackware-14/change-keyboard-layout-in-console-187987/)
Ubuntu does not come with the console keymaps; you need to install 'console-data'. Then, find the dvorak keymap and load it with 'loadkeys'.
