### Post by raysa on 2016-03-25
I have two computers running Xubuntu 14.04, and on one of them the Network Manager icon is missing so I can't get at it to change network connections etc. I've looked at NetworkManager.conf on both machines and they are identical. I tried, as suggested in another thread on this, 'apt-get install indicator-applet-complete' and a few other icons appeared or duplicated, but still no Network Manager icon.

---

### Post by raysa on 2016-03-25
[SOLVED] Went to synaptic and noticed network-manager-gnome was not installed.  Installed it - icon now appears in the panel.
