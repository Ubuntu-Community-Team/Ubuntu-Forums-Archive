For anyone not wanting to read the other post-

1. Right click a file in Nautilus
2. Select Properties
3. Select "Open With" tab
4. If two programs are shown, select the one you want to be the default.
5. If only one is shown, click "Add", then add the path to the program. It will then be set automatically as the default.
