### Post by kidders on 2011-01-21
Hi there,

I'm not a Windows user, but afaik Microsoft has decided that certain devices should not *need* more than one partition, so Windows conveniently ignores them.

Apparently, one way around the problem is to make Windows think your memory stick is a USB hard disk (ie a device that *is* allowed to have multiple partitions). I'm pretty sure you can Google a solution to the problem, but it's very unlikely _you've_ done anything wrong.
