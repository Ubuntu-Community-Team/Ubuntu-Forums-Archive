### Post by Jussi Kukkonen on 2007-01-15
Not automatic updates, but up-to-date source code. So, if you're not ready to compile things from source don't waste time learning svn.

SVN is a source code management system. Developers push their new code to the repository. Then other developers and users can pull the changes to their local source code tree, compile the program and use the new features. 

There is a good book: [http://svnbook.red-bean.com](http://svnbook.red-bean.com)
