Is there any way I can make Update Manager not tell me certain programs need to be updated. For example, on Jaunty I have Amarok 1.14 and it keeps telling me to update to Amarok 2 but I like the old version so much more.

### Post by zvacet on 2009-04-26
Find your package in synaptic and click on it and go to the** package tab>lock version.**
