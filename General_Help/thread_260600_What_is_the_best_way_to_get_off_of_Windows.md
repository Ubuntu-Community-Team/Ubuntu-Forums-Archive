### Post by David Corrales on 2006-09-19
You could...
[LIST=1]
[*]Make DVD backups of the info
[*]Copy all the info to an external HD
[*]Make a shared FAT32/ext3 partition for Windows/Ubuntu. Just download the ext3 driver for Windows.
[*]Set up a network and transfer the files via Samba, FTP, HTTP, SSH,etc
[*]Make enough space inside the regular Ubuntu partition and copy the files using the ext3 driver
[/LIST]
