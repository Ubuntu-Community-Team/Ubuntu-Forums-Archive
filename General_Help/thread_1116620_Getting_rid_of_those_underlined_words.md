### Post by mb_webguy on 2009-04-05
Not necessarily.  There is an abundance of Greasemonkey scripts available.  It's moderately likely that someone else has already written such a script.

You could also use one of the many script-blocking extensions like NoScript, though most of these block all scripts on a page, and few allow selective disabling of scripts.

Also, the double-underlining itself is likely done independently of the script using stylesheets, so disabling the script won't by itself remove the underlining.
