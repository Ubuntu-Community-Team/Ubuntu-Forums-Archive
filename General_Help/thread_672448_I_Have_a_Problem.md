### Post by melopsittacus on 2008-01-19
Hello, when Linux is booting try pressing CTRL-ALT-F1. If the system is not frozen completely, it will give you a text terminal in which there may be error messages which may help to solve your problem.
As for the recovery partition: if it is on the same hard disk onto which you are installing Linux, than it is quite easy to lose it by accident and you should by all mens backup it. If it is on another hard disk, which you do not touch at all, than it is more difficult to delete it by accident.
