### Post by sikander3786 on 2010-11-21
Install the ubuntu-netbook package by

```
sudo apt-get install ubuntu-netbook
```

Once installed, log out. On your login screen, click you username, sessions menu will appear at the bottom of screen, choose Ubuntu Netbook Edition, type your password and login.

Make sure you have installed graphics drivers before that as it is all 3D.
