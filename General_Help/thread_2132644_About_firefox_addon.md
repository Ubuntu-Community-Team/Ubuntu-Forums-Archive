### Post by claracc on 2013-04-05
The fireRainbow add-on seems no to work with your version of firebug as it is explained in the description page of the add-on. It only works with firebug 1.5.4, 1.6, 1.7, 1.8, 1.9.

About firepicker, the author of the add-on says "Adds a small color picker dialog when editing CSS rule in Firebug", so I understand this add-on is not shown in firefox bar, but the color dialog will appear when you edit a css rule with firebug.
