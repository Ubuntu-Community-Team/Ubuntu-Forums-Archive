### Post by yabbadabbadont on 2009-04-25
Usually, you put the disk into the CD/DVD drive and then reboot.  Make sure that your computer's BIOS is set to boot from the CD/DVD drive first.  Once it boots from the installation CD, you just install it as you did before.
