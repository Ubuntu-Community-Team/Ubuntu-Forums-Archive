### Post by GSF1200S on 2010-05-02
> **TheNerdAL said:**
> My little sister wants a short cut for a web page and I want to make it and put it on the AWN Dock but it needs a command and I don't know how to do that. Thanks.

Unless Im missing something, it would be like:
```
firefox www.ubuntu.com
```
Of course, you could specify a profile or whatever if you needed to:
```
firefox -p Profile2 www.ubuntu.com
```

Google Chrome would be:
```
google-chrome www.ubuntu.com
```
Works fine for me.
