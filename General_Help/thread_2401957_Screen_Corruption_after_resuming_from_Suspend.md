### Post by ajgreeny on 2018-09-24
ubuntu 15.10 lost all support over two years ago so we can not really help you solve this problem other than telling you to move to a fully supported version.

I recommend either 16.04 or the more recent 18.04 version and would recommend a clean install rather than an upgrade which is no longer a simple task from an unsupported version such as 15.10.
Both are supported for 5 years, 16.04 already having used over two of those five, of course, so 18.04 is probably the better one to choose.
18.04 uses the gnome DE by default, but if you don't like gnome there are other DE versions available, eg Xubuntu with Xfce, Kubuntu with KDE, plasma or neon, and Lubuntu with LXDE, plus several others which you can find easily in the **Ubuntu Community** links at the top of each forum page.

EDIT:
A quick look at specs for this machine leads me very firmly to suggesting Lubuntu as the best OS to choose unless it has been greatly hardware upgraded.
