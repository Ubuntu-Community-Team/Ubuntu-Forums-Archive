Hi Srikrishnan

Update the keyboard to 'iBus Tool'.

Application -> System Tools -> iBus.

1. First download and install a tamil font which you like, then save that font as the system font (so that you can use the favorite font to your system as well)

2. Now open the 'iBus' and see the keyboard symbol on the system tray and clik on it -> Preference and add the language as tamil and add the fond location as well

3. Goiyala done! then every time if you want to use the tamil keyboard remember to set the preferences to tamil so that you will not to search for it
