Yes you can. The emails live in files in the mbox format under your thunderbird profile. The mbox data files do not have a file name extension.

### Post by Hobgoblin on 2009-03-15
In your home folder you will have a hidden folder called **.mozilla-thunderbird**, in that there is a fiolder called something like jdjdhyt.default

Copy the contents of that folder to a flash drive.  Find the same (though differently named) folder on the other system and put the contents you just copied in there.
