### Post by windscape on 2013-01-02
Hi,

You have an outdated PPA configured in Software Sources. XBMC 11.0 is included in the standard 12.10 repositories. If you want to try XBMC 12.0 Beta, add the repository for it using the instructions here: [https://launchpad.net/~team-xbmc/+archive/unstable](https://launchpad.net/~team-xbmc/+archive/unstable)

It appears that the instructions on the XBMC wiki for 12.10 are a bit out of date.
