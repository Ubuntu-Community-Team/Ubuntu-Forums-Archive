### Post by nzjethro on 2011-06-18
I'm sorry, but a console as in a command line interface? If so, hit Alt + F2, and type "gnome-terminal" (or maybe just "terminal").
Else click on the launcher, and type "terminal" and click the programme that comes up.
Or maybe, not sure if this is a default feature in Natty, hit Ctrl + Alt + t.
