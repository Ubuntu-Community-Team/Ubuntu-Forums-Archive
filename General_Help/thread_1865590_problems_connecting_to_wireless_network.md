### Post by Mark Phelps on 2011-10-20
For added security, some Wireless networks are configured to do MacAddress screening, that is, to only allow devices to connect whose addresses are already in a list stored with that access point.

You need to either check the Wireless Security features on the Router/Access Point, or if you don't have access, then check with whomever administers that Router/Access Point.
