### Post by mordi05 on 2008-04-14
How can I paste text from a browser to a file edited with vi?

I need to put a lot of text into my nethack options file :-), and on nethack.alt.org editing of this file is vi- based.

I only know how to cut/paste within a file.

---

### Post by geraldm on 2008-04-14
Use mouse-button-1 within the browser to cut the text.  That will put the cut text into the clipboard.  Change focus to the vi program -- vi must be in INSERT mode --.  Press the middle mouse button and the clipboard text should be pasted in.

Gerald

---

### Post by BaffledMollusc on 2008-04-15
Select the text normally, i.e. highlight it. Then, without highlighting anything else, change to your vi window and middle click (assuming you're in edit mode). If you have a two button mouse, the middle click can usually be emulated by pressing the two buttons simultaneously.
