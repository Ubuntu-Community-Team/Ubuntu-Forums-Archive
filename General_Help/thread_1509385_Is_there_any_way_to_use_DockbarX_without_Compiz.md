### Post by M7S on 2010-06-14
You can't use the preview feature in dockbarx x.0.39.1 or later without either compiz fusion or kwin (kde window manager).  The previous versions of dockbarx x.0.30 or x.0.39 used another method to show previews which worked with any window manager which supported compositing. I changed it because it was too slow and not really stable. 

You can use dockbarx without previews with any window manager, though.
