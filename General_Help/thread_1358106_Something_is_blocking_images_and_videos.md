### Post by wesleybailey on 2009-12-18
To enable loading of remote images, you need to change the "Block loading of remote images in mail messages" setting:

   1. Go to the menu and choose: Tools -> Options -> Privacy -> General
   2. Choose one of these two options:
         a. Remove the check mark next to "Block loading of remote images in mail messages", or
         b. Leave that setting checked, and select "Allow remote images if the sender is in my [list of address books]"
                * If you choose this option, make sure to add the address that sends email with images you want to see to the address book you select from the drop-down list.
