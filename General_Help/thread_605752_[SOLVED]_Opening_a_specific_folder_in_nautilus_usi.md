Is it possible to run nautilus and tell it to open a specific folder using the terminal? Like when I run 
```
nautilus
```
It opens "/home/username"

Can I set it so it will open to any folder of my choosing instead of just "/home/username"?

---

### Post by aysiu on 2007-11-07
```
nautilus /path/to/folder
``` should do it.
