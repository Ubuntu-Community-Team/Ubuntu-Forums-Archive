### Post by mb_webguy on 2009-03-24
The only "safe graphics mode" for Linux is the recovery console selectable at the GRUB menu.  This will put you at a shell prompt as root.  You can then edit the /etc/X11/xorg.conf file to change the video driver in use to something "safe" like vesa.

To do this, use the following command at the recovery console prompt to edit the xorg.conf file:```
nano /etc/X11/xorg.conf
```Look for the "Device" section, and the line specifying the Driver.  Change this to "vesa".  Then save and exit, and use the command "reboot now" to reboot the computer.  Now when you login normally, you will be using the basic vesa driver.
