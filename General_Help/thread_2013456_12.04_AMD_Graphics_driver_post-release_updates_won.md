### Post by QIII on 2012-06-30
The fglrx-updates and fglrx-amdcccle-updates packages have both been problematic for some users.

I would remove them this way:

```
sudo apt-get remove --purge fglrx* 
```I have been able to get the -updates driver to install and work properly, but the Catalyst Control Center will not start.

Installing fglrx and fglrx-amdcccle will get you to normal operation with 3D if you want the proprietary driver.
