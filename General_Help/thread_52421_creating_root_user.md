### Post by angkor on 2005-07-28

I have seen this happen quite a few times now, people referring to root logging into X when others ask how to enable a root account. 

It is totally unnecessary to log in as root in X to be able to mount a windows drive. It is also considered a security risk.

What's wrong with using sudo to mount your windows drive?

To enable a root account if you really need it:


[http://www.ubuntuguide.org/#setchangeenablerootpassword](http://www.ubuntuguide.org/#setchangeenablerootpassword)
