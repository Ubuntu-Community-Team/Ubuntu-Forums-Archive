### Post by mcduck on 2015-11-20
The automatic backup included in Ubuntu will every now and then create a full backup instead of incremental one, and after that it wants to test if the backup files are valid and can be restored correctly. If you have set it to encrypt your backups, it asks for you to input your encryption password.

So, assuming you are using automatic backups, and have set it to encrypt the backup, then the message would be perfectly valid & normal.
