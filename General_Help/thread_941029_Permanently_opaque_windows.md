### Post by loomsen on 2008-10-08
You may specify per-window rules with compiz. 
For compiz 0.79 go to ccsm -> accessibility -> opacity, brightness & saturation
Under Opacity add a new rule according to your preferences, which is pretty easy. Just click the + and chose if you wanna sort by title, class whatever. Now hit grab and nail the window you want to opacify. Now set the relation in case you want to make different windows opaque. and/and not - or/or not are possible values which should let you combine any desirable rules you want.
(If I remember right in an earlier version of compiz it is located under
ccsm --> general options --> TAB opacity)

Hope this helped
