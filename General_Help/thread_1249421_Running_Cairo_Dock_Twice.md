### Post by fabounet on 2009-08-26
you can have many docks in a single instance of Cairo-Dock, as you would have many gnome-panels.
jsut edit a launcher/applet, assign it to a new dock (enter a name that doesn't exist, like "dock2"), apply.
a new dock is created.
now right-click -> set up this dock, and place it anywhere you want.
you can then populate it with other launchers/applets in the same way.
