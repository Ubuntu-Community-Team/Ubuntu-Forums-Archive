### Post by pontuz on 2009-06-20
You can unmount a drive like you do a safe unject in Windows.
And you do almost never need to mount a drive because the system will mount it automatically.

And if you need to manage some partitions or something on a drive, you must unmount it first.

### Post by philinux on 2009-06-20
Unplugging a usb stick without unmounting or "safely remove" in windows can damage the file system. Same for an external drive.
