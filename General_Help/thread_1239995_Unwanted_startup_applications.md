### Post by ryanmcc on 2009-08-14
When I startup Ubuntu 9.04 and log in, I have a number of unwanted applications starting up automatically, such as Terminal, Tomboy Notes, Pidgin etc. I don't want any applications to start.

I think the problem stems from the fact that a few days back I used the "Automatically remember running applications when logging out" option under "System->Preferences->Startup Applications". I subsequently turned this feature off, but Ubuntu seems to have got confused and is still starting up these apps.

Any ideas on how I can fix this?

---

### Post by XCan on 2009-08-14
I think you will need to start Ubuntu, shut down all unnecessary programs, turn that "Automatically remember running applications when logging out" feature on, Restart, and then disable the feature.
