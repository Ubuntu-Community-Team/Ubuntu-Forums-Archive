### Post by Lateralis on 2010-09-05
Hi Vinay. 

Firstly, the Ubuntu Software Centre (Applications -> Ubuntu Software Centre) can tell you which programmes you have installed.  Synaptic will tell you all of the pacakges you have installed which isn't necessarily always what you're after. 

As for command line only programmes... Someone else may know a trick I don't, but generally I think you just have to know what the command is.  Very often it will be something pretty obvious, and more often than not exactly the same as the package name or stem of the package name you give to aptitude (sudo aptitude install <package>).  For Perl just type perl.  For Firefox, just type firefox.  If you can't figure it out then a web search will very often throw something up.  

As for onTV, it is apparently a Gnome application so a link should find its way into the applications menu somewhere or t'other.  

This only works with programmes installed via Synaptic/Aptitude.  Compiling programmes from source code is a totally different ball game.
