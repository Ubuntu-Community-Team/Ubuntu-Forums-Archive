### Post by dFlyer on 2011-06-29
You might want to take a look at bibble5pro. It will cost some money but works very well. You can use a database or just use the folder you have. Lightzone 3.9 also allows you to edit your file in the directory provided, but again will cost some money.

### Post by SirPecanGum on 2011-08-17
Shotwell does use a database but also writes metadata to the file - check the Shotwell preferences to make sure the feature is enabled. It works very well for me with 35,000 plus photos. Geeqie is also a nice little app, very handy to have.
