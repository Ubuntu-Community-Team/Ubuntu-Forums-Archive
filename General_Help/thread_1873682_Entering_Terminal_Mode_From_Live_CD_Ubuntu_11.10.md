### Post by Steve R. on 2011-11-01
Well, it took a bit of digging, but I found out how to enter Terminal Mode (Window)  from the _**Live**_ CD in Ubuntu 11.10.  It was _**NOT**_ obvious.

```
CTRL + ALT + T
```It can be found on the top horizontal bar that has the word "*desktop*". When the cursor is moved there the horizontal bar displays additional words, one of the displayed words is "*Help*". Select help, enter "Keyboard Shortcuts".

_**Note**_: I am using the live CD to inspect a Windows computer, so Ubuntu is **not **installed.
