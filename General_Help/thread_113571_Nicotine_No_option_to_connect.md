### Post by dresnu on 2006-03-17
Sure you have already solved this, but just in case you haven't in order to connect with nicotine you have to set a username and a password. Once you do it you 'll get the option to connect not "greyed out". If when trying to connect it gives an error about wrong password this means that the username you set is already taken and you have to change it.
