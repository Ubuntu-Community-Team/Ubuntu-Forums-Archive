### Post by naviathan on 2013-04-10
Ok, this is going to get confusing really fast. nVidia proprietary drivers have always limited you to two displays. The original post involved two displays off the nVidia card and one off the processor GPU. In past versions of Xorg this was possible. In the current version something has changed that has made it impossible to run multiple displays off GPUs using different drivers. For some reason if it's not off the same card or a similar pair of cards that run the same driver, Xorg will balk at it and fail. Right now, in Linux, if you want to run more than two displays your best bet is to buy ATI.
