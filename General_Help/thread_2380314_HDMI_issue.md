### Post by Kirk Schnable on 2017-12-17
The second monitor (the TV) is probably to the left or right of your laptop screen in the virtual layout.  You should be able to see this in the Display properties.

Try dragging the program off to the left or right side of the screen accordingly and it should go to the other screen.

If you want both screens to show the same thing, try putting the display in mirrored mode from the display properties.
