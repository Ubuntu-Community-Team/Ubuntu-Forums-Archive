### Post by Ginsu543 on 2010-08-08
We need a little more information to help you.

First, were you able to use the LiveCD to have a working live session? In other words, were you able to boot off the CD and get to the Ubuntu desktop and run it without actually installing it?

Second, if you were able to get to the live session, that means that most likely your hardware can run Ubuntu. So in that case, I suggest you try reinstalling Ubuntu.

Third, if you were not able to get the live session working, it may be that your hardware is incompatible with 10.04. Try downloading a previous version, such as 9.10 Karmic, and running it live session. There were some changes made between the two releases that may make one work better than the other on your hardware.

If you've tried all these methods and still can't get Ubuntu to run, come back here and post detailed information as to what you did exactly and what happened. The more specific you are, the better people here can help you.
