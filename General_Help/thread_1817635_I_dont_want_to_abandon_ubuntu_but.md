### Post by blane2 on 2011-08-03
I virtualize windows in OSX to use FL-Studio.

After installing guest additions I am able to pass the midi controller through the usb interface for use in FL-Studio. Latency is not an issue so long as I use the audio driver that installs with FL-Studio.


However, if I were you I'd install windows next to ubuntu and use windows to power my DAW. You'll get native support for your hardware and things will just work properly without and problems.

You don't have to ditch ubuntu, lots of people have dual-boots set up to use a handful of programs & hardware... I.E. Adobe programs, protools, and citiniq tablets..
