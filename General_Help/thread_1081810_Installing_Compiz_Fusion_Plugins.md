### Post by stumbleUpon on 2009-02-27
The compiz-fusion plugins are available in the ubuntu repositories. Installing from there would be safer.

```
sudo apt-get install compiz-fusion-plugins-main compiz-fusion-plugins-extra compiz-fusion-plugins-unsupported
```
