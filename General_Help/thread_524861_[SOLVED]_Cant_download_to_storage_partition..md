### Post by merlinus on 2007-08-13
```

kdesu kate /etc/fstab

```and add these lines before the cdrom ones:

/dev/sda1 /media/win1 ntfs-3g defaults,locale=en_UK.UTF-8,,uid=1000,gid=1000,umask=0002 0 2

/dev/sda2 /media/win2 ntfs-3g defaults,locale=en_UK.UTF-8,uid=1000,gid=1000,umask=0002 0 2

Save the file and restart.

-merlin
