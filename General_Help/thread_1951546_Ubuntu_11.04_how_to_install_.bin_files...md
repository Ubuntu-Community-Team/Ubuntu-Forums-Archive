### Post by HiImTye on 2012-04-03
you need to specify the folder it's in, or if it's in the folder you're currently in you need to tell it that it's in your current folder, i.e.:
```
/home/<yourusename>/Downloads/whatever.bin
```
or if it's in your current folder:
```
./whatever.bin
```
