Opera is **not** open source. The desktop version, as of v8.5, is _free_ but not open. Opera Software is still a commercial company, that part hasn't changed. 

Even before the browser went free, the main source of income has always been their mobile market and rendering engine, Presto, which is used by Adobe and Macromedia among others. Opera Software negotiated deals with Google and other companies that helps drive in revenue via the search bar. Mozilla has similar deals.

Anyway, as far as browser speeds are concerned, there is a great article [here](http://www.howtocreate.co.uk/browserSpeed.html) which shows various versions of IE, Mozilla, FF, and Opera against each other.
