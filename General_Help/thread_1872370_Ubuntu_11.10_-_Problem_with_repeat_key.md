### Post by TheOneEyedMan on 2011-11-16
I also have a problem with the repeat keys getting turned off from time to time. I have a work around but not a solution. Go to Preferences-Keyboard and make sure "key presses when key is held down" is checked. If it is not, that's likely your problem. Assuming it is, tweak the delay or speed a tiny bit. That turns repeat back on for me.
