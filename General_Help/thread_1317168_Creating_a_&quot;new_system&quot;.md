### Post by mac9416 on 2009-11-06
Hey, chokito,

When you install Ubuntu from a USB drive with persistance enabled, it will install Ubuntu without any of your persisted changes. Also, I don't know of any method of saving a persistent USB install to an ISO image.

You should look at the [Ubuntu Customization Kit]("http://uck.sourceforge.net/"). I have used it in the past, and it works like a charm.

Good luck!
