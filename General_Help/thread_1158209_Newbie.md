### Post by Tipped OuT on 2009-05-13
The Ubuntu download is never any bigger then 700 MB's so it can fit on a CD-R.

Ubuntu can use iTunes with "Wine" and "Playonlinux", but it's not worth it (very slow).

Ubuntu can use Skype.

Ubuntu has replacments for almost everything, for iTunes, it's Banshee, for Windows Live Messenger, it's aMSN and so on.

Ubuntu requires at least 384 MB's of RAM to function properly (256 MB's is minimum). You'll fine exellent performance with just 512 MB's of RAM.

Ubuntu has 32-bit and 64-bit versions of the operating system. It also has a netbook version.

Ubuntu is a completely free operating system, if you have any question or concerns, the answer is usually a few clicks away on Google. If you still can't find the solution, come over here, and pass the problem by us. :)
