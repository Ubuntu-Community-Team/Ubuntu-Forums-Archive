### Post by Dennis N on 2013-10-08
One solution is to use the **gnupg** (gnu privacy guard) program with a single password/passphrase to both encrypt and decrypt (called symmetric). It encrypts files, so if you want to encrypt a folder (or any collection of files) create an archive file first, then encrypt it.

**gnupg** available in Ubuntu repository. There are lots of tutorials on the web to learn how it is done. Google "gpg tutorial".
