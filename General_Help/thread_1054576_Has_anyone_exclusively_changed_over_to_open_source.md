### Post by mb_webguy on 2009-01-30
I've taken a look at the completely FOSS Ubuntu remaster, and I really doubt it could meet my needs.  I know it couldn't in the area of multimedia, since it only provides playback and conversion applications for open codecs and containers.  It would take a pretty hard-core FOSS supporter to actually go exclusively open-source.  That essentially means that you're limiting yourself to the main and universe Ubuntu repositories and software from third party sources that can be compiled from source.  I don't know about other Ubuntu users, but I pull a lot of software out of the restricted and multiverse repositories, Medibuntu, and various PPAs...  I'm sure that not all of that is open source, even though it's free.

I use the official Flash plugin instead of the (in my experience) currently somewhat inferior open-source implementations.  I design websites, so I have IE installed using IEs4Linux so that I can ensure cross-browser compatibility.  Actually, I don't even know that I could identify all of the software on my system that is free but not open-source.  When it all comes by Synaptic or in debs, it's hard to keep track of what's open-source and what's not.

But if you actually meant by "exclusively open-source" if we've switched completely over to Linux and free software, then count me in. The only way I'd go back to Windows if I were getting paid to do it.  And it wouldn't be cheap.
