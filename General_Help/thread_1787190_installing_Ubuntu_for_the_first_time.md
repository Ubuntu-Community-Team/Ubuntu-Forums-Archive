### Post by YesWeCan on 2011-06-20
Hi. Welcome to the wonderful world of linux. :)

The first thing I would do is to make a live CD of Ubuntu and boot off this. Unlike Windows, Ubuntu can boot off a CD or UDB without any installation being needed. You can then use it. ### Post by nzjethro on 2011-06-20
If you have a reasonable internet speed, and a USB flash drive lying around, I would recommend downloading both 10.04 (the long term support version) and 11.04 (the latest version). The reason is that 11.04 has had a lot a bad reviews lately for being buggy, and for not being compatible with all hardware. For example, my laptop is about 4 years old, and it's a bit slow to run 11.04 well, while it runs earlier vesions fine. Simply download the .iso's from [here]("http://www.ubuntu.com/download/ubuntu/download"), then make a Live USB, using unetbootin (found [here]("http://unetbootin.sourceforge.net/")). Try both for a week or so, to make sure that your computer can run them well, and decide which version you prefer.

And then when (if) you do decide to install it, ask for advice at these forums about partitioning/installation alongside xp. The first time I installed Ubuntu, I accidentally wiped my Win 7 partition, and had to restore everything. If you ask here, people will be able to provide step by step instructions to make sure your data will be fine.

### Post by nzjethro on 2011-06-20
Minimum Ubuntu size you'll need is about 4-6GB. This will give you a minimal / partition, and a small /home partition. Although, of course, it would be better to give it more room, especially if you are going to be installing programmes on it. Also, if you want to have a swap partition (used by the system as extra memory), they say you should allocate the same amount of space as you have RAM.

I have about 60GB for all my Ubuntu partitions on my primary version, but I also have a 10GB Development release install. Currently my 10GB isn't used up, but I haven't been using the development release for much; just testing.

Also, I'm not sure if it'll work for iPhone, but I have a programme called GTKpod, which allows me to sync my iPod (classic, gen4). Maybe look into to whether it is possible to use an iPhone with Ubuntu.

But like we say, try Ubuntu on a Live CD (or Live USB) first, and then if you decide to install it, ask the forums for help with your situation, and we can give you good advice for that situation. :D

### Post by nzjethro on 2011-06-21
> **louisgonick said:**
> whats the point of downloading both versions???

Like I said, so that you can try them both, and see which one best suits you. That's the joy of Ubuntu - it gives you a lot of choice. Both releases are supported for a few more years, but they are very different, so I'd encourage you to try both, and see which is more to your liking.
If you'd prefer not to download both, I'd suggest getting 10.04 LTS (Lucid Lynx). As well as being more stable than 11.04, if you do decide to upgrade down the line you can do so using the Update Manager in 10.04. If, on the other hand, you download 11.04, and decide it's not right for you, or your hardware has issues, you can't downgrade without downloading an earlier release (e.g. 10.04) and doing a clean install.

### Post by YesWeCan on 2011-06-22
Because of various licensing restrictions, Ubuntu does not come with comprehensive multimedia support pre-installed. Eg: for watching DVD movies. So you have to manually install a lot of things, see: [https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)

When you are running a live CD Ubuntu has no storage apart from RAM. Anything you install gets put in RAM and not on your hard disk. This is fine until you reboot when all your changes will be lost.

The LibreOffice suite fully supports reading and writing MS Office formats, plus a load more.

You can install and use Windows applications in at least two ways:

1) Install Wine (this is a Windows OS emulator) and is meant to be pretty good. Some swear by it and others at it. Google for itunes on wine.

2) You can run a virtual machine application that emulates a PC and you actually install Windows to it. Windows has no idea it is inside a virtual reality simulation running on linux. ### Post by Stray Wolf on 2011-06-22
The first thing a Ubuntu noob should do is install "Ubuntu Restricted Extras" which you can do from the "Ubuntu Software Center" (USC), or from the terminal by typing:
 sudo apt-get install ubuntu-restricted-extras
You'll also need the Medibuntu Repository which you install from the terminal like this:
sudo wget --output-document=/etc/apt/sources.list.d/medibuntu.list http://www.medibuntu.org/sources.list.d/$(lsb_release -cs).list && sudo apt-get --quiet update && sudo apt-get --yes --quiet --allow-unauthenticated install medibuntu-keyring && sudo apt-get --quiet update
If you need Java terminal input this:
sudo apt-get install sun-java6-jre sun-java6-plugin

Other than that, make sure you have libdvdcss2 and libdvdread4 by entering them into the search bar in USC.  And you'll be set for multi-media.  If you want to sync with another device like an iphone, search in USC first.  I typed iphone into the USC search just now and got 26 items. (itunes had 16 matching items including rhythmbox and amarock.)
