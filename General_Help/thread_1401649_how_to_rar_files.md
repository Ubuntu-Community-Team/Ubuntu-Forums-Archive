### Post by howefield on 2010-02-16
> **CatchItBaby said:**
> I hv 10 folder and i want to rar each folder in seprate archieve with size limiation is that no rar file is more than 100 mb how to do that ?

You could right click on each folder and select the compress option.

In the dialogue window that pops up, select rar as the compression type, click on other options and check the box "Split into volumes of" and enter the size requirement.

You might find it easier to deal with the archive manager gui when compressing files, if you go to System > Preferences > Main Menu and click on Accessories in the left pane, and then check the box beside Archive Manager, this will give you an icon in your Applications > Accessories menu to start the archive manager.
