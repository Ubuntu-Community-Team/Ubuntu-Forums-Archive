have you check how OpenOffice is set up to print.
Open an OpenOffice document then go to the Tools tab and select Options
select Print
check the setting for Reduce print data Setting for Printer or Print to file
Printer should be checked

if it still does not print to your printer go to System > Administrator > Printing
Is your printer there?  Is it set up right? As you state other apps can print
to your printer.
