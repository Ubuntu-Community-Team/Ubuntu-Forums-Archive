### Post by dnairb on 2010-03-22
I have XBMC working well on Ubuntu Karmic.

You need to install the restricted video drivers:

System -> Administration -> Hardware Drivers.

Select the latest recommended version. When it has downloaded and installed, reboot and all should be good.
