### Post by ibuclaw on 2008-04-18
Ubuntu Desktop are both exactly the same on all levels apart from the lowest one, the CPU.

 - Ubuntu is for 32bit Machines. ie: Intel 80386's (486/586/686) more commonly known as x86 CPUs.

 - Ubuntu 64 is for 64bit Machines. ie: AMD64 or 64bit Intel's.


Ubuntu Server differs from the Desktop.
The first thing you'll notice is the absence of a window manager. Ubuntu Server was built in mind for a Monitor-less PC, thus everything is done in the console.
Ubuntu Server also has some extra packages installed that are purely for Server needs (openssh, samba, etc, etc.)
Though that said, nothing will stop you if you do decide to install gnome on-top of it anyway!

And the same goes to Ubuntu 64 Server, exactly the same as it's counterpart, but runs on 64bit CPUs only.


64Bit CPUs are backwards compatible. So you can install both Ubuntu 32bit and Ubuntu 64bit on an AMD64, but can't install the latter on an Intel x86, as it's CPU cannot run the 64bit assembly binary.
