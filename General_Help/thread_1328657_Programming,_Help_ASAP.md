### Post by Giblet5 on 2009-11-16
I learned C++ from its inventor so I'm partial to it.

Get the book "[The C++ Programming Language]("http://www.amazon.com/C-Programming-Language-Special/dp/0201700735/ref=sr_1_1?ie=UTF8&s=books&qid=1258411018&sr=8-1")" by Bjarne Stroustrup.

The only people who don't like C++ are those people who can't program in C++.

The important Windows code is written in C++.

Ubuntu is mostly C++ and C (a subset of C++).

HPUX and Solaris are written in C++ and C.

The best applications are written in C or C++.

It is the fastest-executing high-level language.

When major enterprise level software is written in Java, the C++ developers start hovering around like sharks because the C++ guys know that the java code will inevitably be too slow or it will be too unstable, and the C++ guys can move in and charge a freakin fortune to do it the right way. They look like heroes having saved the day, too.

Fame, fortune, and glory. It's all yours if you but try.
