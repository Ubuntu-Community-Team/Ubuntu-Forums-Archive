### Post by jespdj on 2008-03-19
Note that if your friends are using Skype, you will have no choice but to get Skype too - unfortunately there is no other software than Skype itself that supports the Skype protocol.

You can go to the Skype website and download Skype for Ubuntu 7.04 (it works also on 7.10), or you can get it from the [medibuntu](http://www.medibuntu.org/) repository.
