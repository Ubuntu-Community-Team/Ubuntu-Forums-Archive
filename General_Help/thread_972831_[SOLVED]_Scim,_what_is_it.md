### Post by jrz on 2008-11-06
Under System-Preferences-Keyboard on the Layouts tab I have both a U.S. English and a Thailand keyboard shown. When I installed Ubuntu I set it up for both languages, but don't remember seeing the SCIM icon initially. If you only use one language I don't know what use the SCIM icon would be. If you right click on the icon you should be able to click on SCIM setup, and under IMEngine, clicking on Global Setup you are able to enable various keyboard languages that should be displayed when you left click on the icon. If left clicking doesn't do anything now, it may be because you don't have any languages enabled to select from.
