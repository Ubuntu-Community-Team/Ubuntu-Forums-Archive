### Post by MAFoElffen on 2023-10-17
Gnome Brasero's UI is very basic and inherits everything in the look-and-feel from whatever the them of the Desktop is. There is no Settings or preference file in Brasero that changes any of that. If you change the Desktop Theming, then Brasero will inherit those chances, whatever they may be.

I tend to find it easier on my eyes if I use a Dark Theme. I use Gnome Teaks, to adjust which theme I am using, and tweak from there.
