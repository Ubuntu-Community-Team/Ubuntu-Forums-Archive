### Post by kostkon on 2018-03-06
This is probably because Ubuntu 16.04 (Unity/Compiz) has just gained support for HiDPI (high resolution) monitors and, thus, it will now scale the interface accordingly.

You can change the scaling value in your Screen Display settings.
