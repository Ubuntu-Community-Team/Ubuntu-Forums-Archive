### Post by greeklegend on 2006-11-04
Or alternativly, if you're like me and have very little bandwidth, you can copy your /var/cache/apt/archives directory somewhere. Then, to install edgy, first install dapper on that PC, copy the backup of /var/cache/apt/archives to the same location on the new PC, then upgrade to edgy as you did before. This time, nothing will be downloaded
