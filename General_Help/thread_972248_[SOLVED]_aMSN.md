### Post by kellemes on 2008-11-05
Maybe I misunderstand.. You only want aMSN to be installed, correct?
The prefered way is to use the standard repositories..
In this case the universe repository, you can setup synaptic to include this repository in it's package management.
Or use Software-Sources from the menu's to include the universe repository.
I'm on Debian myself so not sure what the Ubuntu menu's look like these days;-)

Then use synaptic to search for amsn and install it.
Or from a terminal window..
```
sudo apt-get install amsn
```
