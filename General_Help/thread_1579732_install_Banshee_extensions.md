### Post by 23dornot23d on 2010-09-26
There are a couple of extensions that need adding .....

You can go to System - Administration - Synaptic Package Manager 
and search for Banshee 

or

if you are used to using the terminal ..... 

I find its quicker to do this .... but if you prefer search and tick them in Synaptic it
does the same job ......

[B]sudo apt-get update

sudo apt-get install aptitude[/B] [B]

sudo aptitude update

sudo aptitude instal banshee-extension-radiostationfet

sudo aptitude install banshee-extension-liveradio[/B] 

Link to Banshee site , for any more info you may require but feel free to ask .....
[**Radio Station Fetcher** ]("http://banshee.fm/download/extensions/")– Fetch radio stations from shoutcast.com  and xiph.org.

These are automatically included in the package below .....
if you like radio try this too .......

[B]sudo aptitude install streamtuner 
[/B] 
( Its older ..... but there are a lot of stations pre-set on it .... and its extremely quick to load up )

Best of luck .....
