### Post by UKBB on 2010-11-28
I put it together this weekend and it is up and running on Ubuntu 10.10 with no glitches so far. I'll need to get virtual box loaded and make a WinXP virtual machine to see how the system handles the load. On my old machine (a Dell Dimension 4600 p4 with 1.5 gig of ram) it would pretty much max the processor out the entire time that V-box was running a machine. The case was spot on with all the critiques I read on the reviews, only 2 sata power connectors from the power supply and they are short so you have to position the DVD Drive close to the hard drive or go and buy an extension. There is a crack in the case but it isn't causing an issue and it doesn't show so I'm not going to bother with getting a replacement. The one complaint I saw on the motherboard reviews was that the VGA connector is flimsy and moves around when you plug the cable in. I'll have to be careful with that so I don't break it off of the MB. It is a bit louder than the Dell it replaces but not terribly so and still better than the Quad Core Intel/WinXP machine I built back in 08. All in all I'm pleased with it so far.
