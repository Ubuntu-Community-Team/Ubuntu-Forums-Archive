### Post by ChaosParser on 2008-01-19
Use archive manager to unpack it.  Install Cedega via terminal: 

sudo apt-get install cedega 

In cedega there is an option you will see to install an engine.  Browse to the engine file, click install. 

Be aware that you are using a pirated copy of an engine that you are supposed to subscribe to use.
