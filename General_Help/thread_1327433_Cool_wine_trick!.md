### Post by ctult on 2009-11-15
You can create Windows executables in Wine with programs like AutoIT compiler, MinGW, and more. Just compile it and it becomes a Windows .exe file:KS. I uploaded a simple example made with Wine and AutoIT and one with MinGW and gcc.

EDIT: To run this in MinGW and gcc you have to:

1. Install Wine
2. Extract the tar.gz file
3. Open terminal and go to the directory you extracted the files
4. To run helloworld.exe:
```
wine helloworld.exe
```5. To run dice.exe, double-click the program or:
```
wine dice.exe
```
