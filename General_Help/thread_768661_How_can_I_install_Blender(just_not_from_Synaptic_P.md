### Post by stoneage on 2008-04-28
You could go to blender.org or graphicall.org and download from there. They run 'out of the box' and don't need to be installed. Graphicall has optimised and latest svn versions too. Blender.org has the latest official stable release. Enjoy!

(The Ubuntu .deb does render slightly faster than the official release. Check out - [http://www.eofw.org/bench/](http://www.eofw.org/bench/) and do some benchmarking)
