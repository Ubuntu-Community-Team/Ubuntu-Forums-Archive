### Post by jorgedso on 2008-11-25
Hello,

   I've created a .wav file with about 20 seconds to use it as startup/login sound and it didn't play at all (I've checked later that it was created as WAV (Microsoft) Signed 16 bit PCM). Then I've _changed it's format_ (with Audacity, in case anyone needs to know how I did it) to WAV (Microsoft) **Unsigned 8 bit PCM**, and it started to work nicely! :)
