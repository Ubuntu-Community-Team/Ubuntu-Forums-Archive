### Post by grahammechanical on 2014-12-03
You can open the Dash and search for Software Updater which will start a search for updates once it is loaded but not automatically download and install the updates. We have to confirm the action first.

You can open a terminal and type

```
sudo apt-get update
sudo apt-get upgrade
```

Again we will be asked to confirm the download and installation of any updates.
