### Post by aeronutt on 2010-12-05
Your first post is correct. You SHOULD be able to boot UBUNTU from live CD, run gparted, and create a new partition w/o loosing data. Of course, there is always a risk in loosing your previous partition, but I've done this a half dozen times or more with no problems.  Bascially, you'll resize the current partition making it smaller, then format the unused area. All pretty easy using gparted and livecd.

Of course, anytime you do anything like this, you should back up your critical data beforehand.
