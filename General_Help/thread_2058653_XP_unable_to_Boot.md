### Post by ajgreeny on 2012-09-16
Open a terminal and show us the output of ```
sudo fdisk -lu
``` and also click on the boot-info-script link in my signature and download and run the script (all instructions are on the site).

Paste contents of RESULTS.txt in a New Reply, then highlight entire file and click on # in toolbar (code tags) to make it easier to read.

Or You can generate the tags first by pressing the # icon in the New Reply Edit toolbar and then paste the contents between the generated [ CODE] paste here [ /CODE] tags.

From V60 the script has improved output formatting and requires the code tags (#) to make it legible. New Version is a zip file that you have to extract to get .sh to run.
