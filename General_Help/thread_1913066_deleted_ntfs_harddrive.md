### Post by imachavel on 2012-01-21
how did you delete it? Remove the drive and connect it via usb to sata or ide connector/adapter to working computer. Copy and paste the files. How did you delete the files? If they aren't completely gone, you can do that.

Data recovery works in a way where it recovers files with deleted address space, I don't recommend this because usually the system can't open the files if it doesn't know what the file is. Hopefully you didn't delete the files and it's just a boot problem. In the future a lot more info would be needed, what version model computer, what OS and what version of OS.
