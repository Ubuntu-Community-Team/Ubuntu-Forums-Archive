To download Ubuntu, go to [this site]("http://mirror.mcs.anl.gov/pub/ubuntu-iso/5.10/"), and click on the install you want (probably [PC (Intel x86) install CD)]("http://cdimage.ubuntu.com/releases/dapper/flight-3/dapper-install-i386.iso")). The install should begin (I recommend saving it to the desktop for easy access), and it'll take awhile, since it's over 600 megabytes.

Once it's downloaded as an ISO to your desktop, you need to install a separate tool to get it onto a CD. To burn an ISO file from windows (is that what you are using?), download [ISO Recorder v 2]("http://isorecorder.alexfeinman.com/v2.htm") and open the .msi file. The program will install itself and integrate with windows. Make sure you have a blank CD-R or CD-RW in your CD or DVD burner, and then right click on the .ISO file on your desktop and select "copy image to CD." Just follow the wizard, and you should have a bootable CD in no time! After it's burned restart your computer, install Ubuntu, and enjoy!
