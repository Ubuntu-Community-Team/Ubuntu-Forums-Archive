```

sudo mkdir /shared2

```

then add /dev/sde1 to fstab...
```

/dev/sde1 /shared2 ntfs-3g defaults,locale=en_US.utf8 0 0

```

and reboot (or sudo mount /shared2).  I don't know if the upgrade rebuilds fstab.
