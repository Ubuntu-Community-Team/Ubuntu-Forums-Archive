### Post by ridgeland on 2010-10-09
Hi SandWrrm,

I tried changing the keyboard layout too (I'm in the US).  But now I use just the US keyboard and turn on composite keys.  With composite keys you can type a ç with "," then [CompositeKey] then "c".  comme ça.
I use a odt file to keep notes each install, and start with a copy of the notes from the last install.  Here are my notes about composite keys:
> Enable composite keys  Menu -> System -> Preferences -> Keyboard.
Tab “Layouts” - select USA click [Options...] expand "Compose Key Position" 
then select [x] Left Win and [Close] [Close]
Now to compose an accented key: Press and release the "Compose Key" Left Win then the accent like ' then e -> é (that was the single quote, next to [Enter].  Opposite ` (left of 1) Left Win then ` then e -> è

Menu -> System -> Preferences -> Keyboard may let you reset.
