### Post by lovinglinux on 2011-04-30
Get [Flash-Aid]("http://www.webgapps.org/addons/flash-aid") and run it. It will detect installed flash plugins, remove them and install the best option according to your system architecture and version. Additionally, it will apply some tweaks that should improve performance and fix fullscreen issues. If you still get that issue after running Flash-Aid, then try to disable hardware acceleration. Right-click on a flash video and select "Settings" to find that option.
