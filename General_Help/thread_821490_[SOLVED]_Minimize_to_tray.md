There is a package calleg alltray.

To open it go to

System --> preference --> sessions.

### Post by Nepherte on 2008-06-07
You use alltray by placing 'alltray' in front of every program you want to dock in the notification area. For example:
```
alltray thunderbird
```

To see all the extra options for alltray:
```
man alltray
```
