### Post by ludovicc on 2008-05-17
Hello,

Try to follow the official instructions for installing Java. There are multiple versions and flavour available, but in general I would recommand you to use OpenJDK 6 or Sun java 6 for something that just works.

[https://help.ubuntu.com/community/Java](https://help.ubuntu.com/community/Java)

If you are using ubuntu on a 64 bits, the situation is still a bit difficult, and few things work. You best bet is to install openjdk6-jre and icedtea-gcjwebplugin. Then check that Java works on your browser with this page:

[http://java.com/en/download/help/testvm.xml?ff3](http://java.com/en/download/help/testvm.xml?ff3)
