### Post by brianfreytag on 2008-10-18
> **DGortze380 said:**
> Microsoft Office is designed to run on Windows (and Mac, but not the version you have). Look into WINE if you need MS Office. Open Office is included by default in Ubuntu. It's a powerful office suite and is 100% compatible with the version of MS Office you're trying to install.

It's dangerous saying it's 100% compatible, because it's not... things will break going back and forth, especially highly complicated and macro-enabled spreadsheets from MS Office.  I have timesheet software that works in MS Office, but not in OOo.

But I agree.. Using OpenOffice (which was installed when you installed Ubuntu) is a great ALTERNATIVE for MS Office.  If you need MS Office badly, you can always dual-boot or install XP or Vista as a virtual machine with VMWare or VirtualBox.
