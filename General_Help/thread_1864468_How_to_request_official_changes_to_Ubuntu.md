### Post by mcduck on 2011-10-19
The "lighter", in the sense commonly used with operating systems and programs, doesn't mean small in size, but light in resource usage (works on slow processor and small amounts of RAM). Any computer that meets the CPU and RAM requirements for Ubuntu versions is more than likely to have large enough hard drive to fit what's included on the install CD's. And making the CD image smaller in size wouldn't allow including programs for common tasks, which would contradict the idea of having an usable system straight from the install.

If you want less disc usage and smaller initial download, then the Minimal CD suggested by wojox would be a good option. That gives you an absolute minimum setup of Ubuntu, allowing you to add only the features you want. (the absolute minimum setup is command-line only, so be prepared for that :))
