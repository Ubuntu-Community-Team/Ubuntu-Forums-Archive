### Post by Giannis V. on 2012-04-29
i think i found a solution to install root with out the help of CERN home page which was no help at all. 
I download the root package from this web page

[http://scaini.org/site/projects/27/root-packages-for-debian-and-ubuntu](http://scaini.org/site/projects/27/root-packages-for-debian-and-ubuntu).
Choose  for 64 or 32 bits

With the download (open package not save) i push the install button, root was download 
In terminal i put code 
sudo ln -s /opt/root/bin/root /usr/bin/root

type root and it works
