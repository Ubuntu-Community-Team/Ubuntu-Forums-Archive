One main difference is that virtualization uses a video driver that you can't do 3D stuff, even if your card supports it. So, gaming is definitely out. Other things like CAD *may* work, but I don't know for sure.

I use VirtualBox at work, because there are certain applications I need which just plain don't run in Wine.

At home, I use Wine when possible, but also dual boot for games.
