### Post by 3rdalbum on 2009-05-14
AppArmour comes preinstalled, so you were probably prompted to update it. I believe there was a security update for AppArmour some time ago.

It it's an update, then apply it. Don't worry about what it is. The system knows what it is and knows that there's a later version, so you don't need to keep track of that stuff. That's what the package manager is there for.

If you want to worry, there's always Google... or you can always select the package in Update Manager or Synaptic and look at the description.
