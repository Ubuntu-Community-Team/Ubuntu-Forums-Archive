### Post by xarxiusxiii on 2009-07-13
If you are still having this problem, then go to  Preferences->CompizConfig Settings Manager (assuming you have it installed) and disable the **Widget Layer**. It should be under the **Desktop** category. Optionally, if you want to use your widget layer, just change the hotkey for Toggle Widget Display.

If you don't have the settings manager installed, search for it in Synaptic or install via terminal (sudo apt-get install compizconfig-settings-manager).

Hope this helps someone. :KS
