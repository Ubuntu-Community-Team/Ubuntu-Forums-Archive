### Post by shazbut on 2010-07-06
Video chat via google talk protocol (XMPP) works good in empathy, also supposed to work in pidgin (but I have not used pidgin for a long time). However, if you are on Ubuntu 8.04 the versions of empathy/pidgin available in the repos probably won't have video chat available.  You'll need to install from PPA, pre built binary, or source (in increasing order of complexity).  The PPAs are here: [Pidgin]("http://www.pidgin.im/download/ubuntu/"),  [Empathy]("https://launchpad.net/~telepathy/+archive/ppa/")

Looks like the builds of empathy for 8.04 are pretty old (2008) so pidgin might be your best bet.
