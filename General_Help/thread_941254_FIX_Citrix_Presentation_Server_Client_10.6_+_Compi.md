I've discovered a fix for a bug that's been giving me absolute hell lately (and a lot of other people it seems) 

When using the Citrix PNAgent with Compiz enabled the initial logon window appears with a window size of 0x0. 

In Compiz there is an addon named "Window Rules" which you can use to force windows to a predefined size. 

I used this to set a rule for Wfcmgr windows to have a size of 500x500 and bingo.. i finally can have my compiz enabled AND login to my citrix server at the same time :) 

Hopefully this will help some people out!
