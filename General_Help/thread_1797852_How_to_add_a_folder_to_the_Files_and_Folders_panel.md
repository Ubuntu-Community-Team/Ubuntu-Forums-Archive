### Post by ianc1 on 2011-07-05
If you mean in the file manager (Nautilus) its easy.

On mine I see a main window with all the files/folders listed in and to the left a bar with links to my home directory, desktop, File System, Network and Rubbish Bin.

Beneath this you see a think line and links to Documents, Music, Pictures, Videos and Downloads.  You can easily add to this list.  If you have a directory called /home/DIR go to home click on the DIR directory and drag it to this lower list on the left.  This forms a link to that directory.

I hope that long winded answer can be understood.  If not and its what your after I'll do a screen shoot and show you.
