### Post by deadflowr on 2019-02-21
memtest86+ is still a default installed package, but grub in UEFI mode cannot run it.
To run it you need to either install the different version (non-foss?) or switch to BIOS legacy mode.
[https://askubuntu.com/questions/917961/can-i-boot-memtest86-if-im-using-uefi]("https://askubuntu.com/questions/917961/can-i-boot-memtest86-if-im-using-uefi")
