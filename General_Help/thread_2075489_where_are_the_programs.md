### Post by kherring7383 on 2012-10-24
Just out of curiosity have you tried Synapse. Basically it's Gnome-Do on steroids. After I found this app several distros ago, this is all I use. 

sudo add-apt-repository ppa:synapse-core/ppa 
sudo apt-get update
sudo apt-get install -y synapse 

Install it and configure it to start every time you login and choose a theme.

By default pressing Ctrl-Space brings it up, simply type the first couple of letters and it lists the app and you just run it.

Sure beats using the Dash, since some apps are just not found right away.
