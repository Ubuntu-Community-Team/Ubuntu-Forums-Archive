### Post by snowpine on 2012-01-12
It is highly unlikely that the very small differences between openbox/fluxbox/jwm/dwm/etc will make a big difference to your battery life. In fact, using a lightweight windows manager but failing to install any power management will probably make your battery life worse compared with a full, pre-configured desktop environment! 

The first step is to use your favorite system monitor (I like htop) to analyze the data. You will probably see that lxde is not a big CPU/RAM hog at all, more likely it is things like Firefox, Flash, etc. 

Power management in Ubuntu is a **vast** topic! A couple of links to get you started:

[https://help.ubuntu.com/community/PowerManagement/ReducedPower](https://help.ubuntu.com/community/PowerManagement/ReducedPower)
[https://wiki.ubuntu.com/PowerManagement](https://wiki.ubuntu.com/PowerManagement)

### Post by cbanakis on 2012-01-12
Obviously, its not Ubuntu, but I recently started messing with Tiny Core.
And I must say, it is quite tiny, and impressive.

Running on a Pentium 2, I see my BIOS post screen for longer than the Tiny Core boot screen.

Its the fastest, lightest OS I have ever seen.  
(Without going back in time, and using horribly outdated software)

Out of the box, it has absolutely nothing installed except for a basic GUI, and a crude software center.

It seems to be a great stripped down starting point.

You also have the option when installing an app, so you can choose if you want it to be started at login, or on demand.

Kinda hard to explain what I mean, but basically, if you choose "on demand" the app will have no effect on the os performance unless you use it.  (In which case, it may take slightly longer for the app to launch)

Then if you choose "at login" it loads all the necessary files for your app when you login, increasing the drag on your system, but apps launch faster this way.

I could be mistaken, but thats my understanding of how it works.

Not sure if that helps at all, cause you seem to be looking for a lightweight Ubuntu based distro, but Tiny Core is as light as the come, so I figured I'd throw it out there.
