### Post by forumache on 2008-04-14
Yes, I know how you can do this.

Hint: You saw it too, when you added the Spanish keyboard, but you cannot see it now again.
..
..
..
Solution: Go to System|Preferences|Keyboard and select Layouts tab. See, no way to view/edit the Spanish layout. Never mind, pretend to Add a new one, select Spain in the new dialog and, voila, on the bottom left you have a button saying Print. Press it and you are done.

Enjoy,
Dan.
