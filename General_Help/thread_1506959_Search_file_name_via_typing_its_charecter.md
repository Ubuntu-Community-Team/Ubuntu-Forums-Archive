### Post by jesuisbenjamin on 2010-06-18
It works for me too. As Polki wrote, you need to highlight a file first and then type. No window will appear but it will move the highlight to the next file matching what you typed.
Alternatively, you can press ctrl+F and then type what you search in the address bar, it will return all files with the given entry. 
The small window popping up in Synaptic is proper to that program, it is not in Nautilus.

Let us know how it works out for you.
