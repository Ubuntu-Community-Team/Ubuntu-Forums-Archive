### Post by niteshifter on 2007-10-09
Short anwer: via macro programming – for a large set of documents. For a single (or a few) docs use the Styles and Formatting tool (press F11 in OOo).

Full answer: There isn't a OOo feature / command that synchronizes documents with an underlying template – because there is no underlying template. A document created with a template becomes a whole entity in and of itself and has no "memory" of the template from which it was created.

Poke around the OOo Forums site (  [http://www.oooforum.org/]("http://www.oooforum.org/") ) - theres a ton of useful stuff there, and IIRC that task of batch re-formatting has been covered there in days past. Another handy OOo resource is the OOoMacros site ( [http://www.ooomacros.org/]("http://www.ooomacros.org/") ).

HTH
