### Post by CatKiller on 2021-04-07
So, for the particular issue you're describing, it's because you haven't told Steam to use Proton (their Wine implementation) so it's looking for the Linux version of the game to download and install, and there isn't one. Select Properties and tell it to use a particular compatibility tool, then pick which version of Proton you want to use from the drop-down list.

However, that particular game is apparently [difficult to make work](https://www.protondb.com/app/1112400), even for experienced users. You might find it better to start with a different game that's known to work easily.

### Post by CatKiller on 2021-04-07

> i cannot find properties, can you give me a pointer, 

Find it in your library and right-click on it, then select Properties. Or select it in the library so that it's on the big pane, click on the gear icon and then select Properties. 

>  is there a list of which games will work. thankyou

The Platinum rating on ProtonDB is supposed to equate to running as well as or better than it does on Windows with no extra fiddling. Gold is supposed to equate to running as well as or better than it does on Windows with some fiddling. Start with either of those.

There's also thousands of native games, too, of course.
