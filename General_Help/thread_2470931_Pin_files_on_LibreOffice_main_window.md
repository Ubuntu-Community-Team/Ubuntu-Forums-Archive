### Post by dragonfly41 on 2022-01-17
xpad app allows floating notes.

If you want some more functionality than xpad you can install Actiona and look for a LibreOffice window title (or other property) and popup some notes or refer to a database.

Or you can use both with Actiona script.

I see now that recent opened files can easily be cleared and lost.
