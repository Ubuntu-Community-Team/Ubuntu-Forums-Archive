### Post by deadflowr on 2014-09-03
14.04 has the added (and unsupported?) option called minimize single application, which if set, will allow you to toggle thunderbird, and every other launcher app, to open and minimize by pressing the launcher button.

To enable, you could install compizconfig-settings-manager, then go to the ubuntu unity plugin section, then in the section marked "launcher" there is a listing for just that, "minimize single window application". Enable it.

So there is that, for you.
