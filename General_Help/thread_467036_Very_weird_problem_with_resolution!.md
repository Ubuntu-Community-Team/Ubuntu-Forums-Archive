### Post by ciphermonk on 2007-07-01
Make the following changes to you xorg.conf

First add a section for the TVoutput 

Section "Monitor"
              Identifier "TVOutput"
             Option "Disable" "true"
EndSection


Then in your device section add this line


Option "monitor-TV" "TVOutput"

Restart xorg... that should take care of it
