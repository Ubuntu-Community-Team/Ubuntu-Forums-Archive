### Post by Chuckaluphagus on 2007-06-13
I don't know the specifics, but what I've read is that the DVD burning software that comes standard with Vista does not burn DVDs that can be read in non-Vista machines.  Even for XP, you need to download a free update from Microsoft in order to read them.  If you want to burn DVDs in Vista that can be read by other machines, you'll need to use a third-party DVD-burning software.
