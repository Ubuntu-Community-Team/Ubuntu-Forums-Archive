### Post by dootzky on 2008-08-30
just a quick note for you all - i had the same problem - this is how I've disable the touchpad on my MSI laptop, from terminal:

```

sudo rmmod psmouse

```

and to turn it back on again, just start that module like this:

```

sudo modprobe psmouse

```

and finaly, if you want to disable it permanetly - just blacklist it.
edit this file (also from console, type):

```

sudo gedit /etc/modprobe.d/blacklist

```

and add these two lines on the end of the file:
```

# disable my boring touchpad on startup
blacklist psmouse

```

hope that helps.
cheers,
dootzky
