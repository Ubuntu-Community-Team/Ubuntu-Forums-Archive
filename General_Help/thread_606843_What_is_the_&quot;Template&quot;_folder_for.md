### Post by Irony on 2007-11-08
If you mean the Templates folder then you can delete it if you want.

To see it work put say a word file in it and call it say word.doc - now when you right click in an empty space in a folder and click create a document you will have the option of opening a word.doc just like in windows. You might have to logout then in for it to work fully.

If you want to hide the Templates folder then create a text file in your home folder called .hidden and in the file type Templates and save it and the Templates folder will become hidden.
