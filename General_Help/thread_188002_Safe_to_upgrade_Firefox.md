If you want to upgrade to the new version, you have to dowload it separately.  Once you do that, you can then use Firefox's built in update manager to update it from then on.  If you want to do it, you can follow the directions in [FirefoxNewVersion]("https://wiki.ubuntu.com/FirefoxNewVersion"), replacing wherever you use 1.5.0.3 with 1.5.0.4
