### Post by dbott67 on 2006-10-26
You can recover deleted partitions and files.

Be careful not to do anything else with the hard drive.  If the 2nd hard drive is the one that contained all of the data, see if you can boot using the 'Live CD' and access the drive.

If you cannot see any files, then you can try a free (Windows) program called 'PC File Inspector' at [http://www.pcinspector.de/file_recovery/uk/welcome.htm](http://www.pcinspector.de/file_recovery/uk/welcome.htm)

I have used the program before to recover files from a damaged hard drive and to recover accidently deleted files.  It will also recover partitions.  There are some caveats, but as long as you don't try to write anything to the hard drive, you may be able to get most of it back (if not all of it).

1. Connect the formatted drive to a system running XP (or use the existing computer if you can still boot into XP).
2. Download PC File Recovery from the link above.
3. Save it to the Windows partition & install it.
4. Start the program and run it against the 2nd physical drive.
5. Pray to the gods above! :)
