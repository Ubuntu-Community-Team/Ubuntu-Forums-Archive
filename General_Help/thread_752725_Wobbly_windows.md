### Post by keykero on 2008-04-11
If you haven't already, try installing Advanced Desktop Effects Settings:

```
sudo apt-get install compizconfig-settings-manager

```

This will appear in your System > Preferences menu.  From there, look for Wobby Windows and mess around with the sliders.  You can tweak the other effects as well.
