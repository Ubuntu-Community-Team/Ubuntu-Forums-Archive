I don't think it is possible to do so (well, nobody has done it so far).

However you might be able to get conky to show it.

Conky is a system monitor that can show you just about anything on your desktop.

If it is possible to display the weather or your emails it should be possible to get premier league score from some website.

There is a big "post your conkyrc file" thread on this forum, make your appeal there.
