### Post by gastly on 2009-09-09
If you're trying to triple boot Ubuntu Vista and XP, then do this:
First install XP and then Vista and then Ubuntu, this way Ubuntu's Grub will have an option to boot Vista and the vista's bootloader will have an option to boot to XP ;)
