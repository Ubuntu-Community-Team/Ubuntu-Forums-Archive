### Post by VCoolio on 2009-04-16
Look for gtk2.x themes. Also metacity (provides only window borders; you can activate them by choosing a theme and then via customize > borders select a metacity theme). GTK2.x can be complete themes, but sometimes they only provide parts and will complain that 'it will not look as is meant' because eg. some icon theme which fits nicely is missing. You should then look for that icon theme and install also that, or use another that you like. When it complains about other theme engines, it's another story. Google them or look for them in synaptic.
This one works pretty well I thought: [azel]("http://www.gnome-look.org/content/show.php/Azel?content=76898"). I'm currently using [this]("http://www.gnome-look.org/content/show.php/kde4.2-oxygen+%28modified%29?content=99926") one, which is copied from kde4.2,
