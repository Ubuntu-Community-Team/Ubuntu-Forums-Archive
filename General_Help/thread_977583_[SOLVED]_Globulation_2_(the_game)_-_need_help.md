### Post by mendozaro on 2008-11-10
As i was writing the help request i realized that the conf files must be in my Home folder and the probably are hidden.

So i go to my Home folder (using Gnome Commander), set the "show hidden files" option and VOILA! I found the directory ".glob2" containing the file "preferences.txt"!

All solved.
