### Post by ClientAlive on 2011-06-14
In the terminal you can type:

```

unzip /path/to/FileOrFolder/

```

Where "/path/to/FileOrFolder/" is your unique path to the folder or file you want unzipped.

you can also get more info on unzip and it's options by typing:

```

man unzip

```

in the terminal window.
