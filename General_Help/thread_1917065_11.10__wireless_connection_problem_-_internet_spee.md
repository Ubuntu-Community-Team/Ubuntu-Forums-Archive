### Post by normanjpcz on 2012-01-29
/PROBLEM SOLVED/

Apparently, this one is a long-time iwl3945 bug in many ubuntu releases.

You can see the bug here: [**launchpad-bugs**]("https://bugs.launchpad.net/ubuntu/+source/bcmwl/+bug/581936")

And the solution is pretty simple (at least for me). [**Install**]("http://wiki.geteasypeasy.com/How_to_use_Wicd_instead_of_Network_Manager") Wicd Network Manager instead of default Network Manager. And uninstall default Network Manager afterwards.

After this everything works fine, it seems that default Network Manager has a bug.

Hope this will help anybody else as well.

Thanks to all for help. Cheers.
