### Post by Hur Dur on 2011-01-16
I'm not going to lie and say that CPU is not important when using Linux, but it's much more important to have a lot of RAM. Ubuntu is usable on less than 512mb RAM, but I would recommend getting at least 2GB if you plan on multitasking. Ubuntu default install takes up around 10GB hard drive space, so you should have a decent sized hard drive. As for CPU, my Intel Celeron M 1.8ghz CPU handled Ubuntu quite nicely, but I would recommend a Core 2 Duo if you're going to be watching movies and burning DVDs.

TL;DR: Get a decent one.

Edit: Core i5 is awesome, as is the Core i7. Get them if you can afford them. Same with the 4GB RAM.
