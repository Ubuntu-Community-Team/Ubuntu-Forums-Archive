### Post by Hugh Bown on 2010-03-29
How do I view a config.log file?

When I enter ls config.log is listed.

When I type 'type config.log' it says no such file as config.log !!

---

### Post by diesch on 2010-03-29
Use
```
 less config.log
``` to view the file (inside less use "q" to exit less, "h" to get help).

*type* doesn't view files but tells you things about commands.
