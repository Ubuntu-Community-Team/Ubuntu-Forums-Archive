### Post by qpieus on 2008-02-19
If you have some experience with linux and don't need a graphical interface, I'd go with the server edition. This will install a basic system with no gui. You'll need to add the software you'll need for the server. The server edition might even prompt you during the install process if you want additional software installed (LAMP - see below).

If you have less experience and/or want a graphical interface, then go with regular ubuntu and later install the webserver software.

As far as setting up a webserver, I don't have any specific experience there, but there's tons of tutorials out there, like this one (for a server with gui):

[http://www.zaphu.com/2007/08/07/ubuntu-lamp-server-setup-guide-with-desktop-gui/](http://www.zaphu.com/2007/08/07/ubuntu-lamp-server-setup-guide-with-desktop-gui/)

If you google for ubuntu LAMP server, you find tons of resources. LAMP stands for Linux, Apache, MySQL, PHP, which are the main components of a webserver. Don't forget to search around the forums here for info on webservers. There's so much info here, it's scary. Have fun!
