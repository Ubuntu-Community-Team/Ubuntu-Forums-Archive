### Post by Rhizoid on 2011-02-06
> **3rdalbum said:**
> You might find Ubuntu to be faster than Windows XP (some do) but technically Ubuntu uses more resources. Ubuntu 10.10 is the most up-to-date operating system, Windows XP is a relic from the days of 64 megabytes of RAM.

I remain to be convinced.  Yes XP dates from the days of 64MB of RAM... but then Land Rover dates from the days of 2 litre petrol engines making 50 horsepower... but they're a far cry from that now.

XP SP3 with anti virus and a firewall installed - let alone a hog like iTunes - is going to be using approx 450MB of RAM and about the same in swap.  Installed with the defaults to HDD it'll take up in excess of 4GB of disk space before applications other than patches and security are considered.

I'm looking at system monitor on 10.04 running from a 4GB USB stick, with a little bit of free space left and a bunch of applications installed (OpenOffice, Firefox, Thunderbird, GoogleEarth to name a few), not to mention that while I type it's resizing a 300GB partition to 250GB and creating some space for a new HDD install of 10.04 while only using 360.4MB of RAM, no swap space to speak of and an average processor utilization of about 20%. (AMD Athlon 64 single core, dual threaded)

Granted this is 10.04 not 10.10 but I'll stand by XP being a slow resource hogging bloat of an operating system, particularly considering it's age, while Ubuntu is a state of the art streamlined and greased up racing snake by comparison.  Seems more than fair to be honest :D
