### Post by Abdus on 2010-02-06
I never heard of Latin Modern Sans (the up-to-date name of this font) to be released in a single .ttf package. I also failed to use TeX fonts efficiently in OO, but I'm nowhere near a font expert.

You are probably aware of .otf set existance:
[http://www.ctan.org/tex-archive/fonts/lm/fonts/opentype/public/lm/](http://www.ctan.org/tex-archive/fonts/lm/fonts/opentype/public/lm/)

You could also check here:
[http://www.pd.infn.it/www_cad/fanin/OOlatexFonts.html](http://www.pd.infn.it/www_cad/fanin/OOlatexFonts.html)
however this work is based on outdated, really old version of these fonts and could result in strange side effects, like pixelization in screen/print.

And to finish nicely, the new font family 'Inconsolata' is superior to anything I saw before in sans-serif family (sorry Helvetica). It's like Helvetica was before Arial was born - suitable for anything. And it's free. And available in .otf. Try it.
