### Post by tommcd on 2007-02-06
pppoeconf should be installed by default. If it is not you will have add it from your edgy CD. Just make sure CDROM is checked in your synaptic repository list.
Do you connect your ADSL by ethernet or usb. Usb DSL setups are difficult on linux since they usually need proprietary drivers. Hook up the modem via ethernet if you can. If you have a "smart" modem you can just enter the modems address in firefox and set up your username and password that way. pppoeconf should work though if hooked up by ethernet.
[https://help.ubuntu.com/community/ADSLPPPoE?highlight=%28pppoe%29](https://help.ubuntu.com/community/ADSLPPPoE?highlight=%28pppoe%29)
