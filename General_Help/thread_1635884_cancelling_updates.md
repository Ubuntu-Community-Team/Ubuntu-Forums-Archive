### Post by plucky on 2010-12-02
> **kanishkdudeja said:**
> I want to know if i opt to cancel updates midway in the update manager, will the updates that have been downloaded by that time install ?

or dey will have to be downloaded again and then installed ?

All the updates are downloaded to your computer before they are installed.

If you stop it after it starts to install the updates,you can break the dpkg system.

If you stop it during the download,it will only download the other updates it needs when you restart it.

Good Luck
