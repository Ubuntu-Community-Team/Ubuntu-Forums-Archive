### Post by Calash on 2010-04-07
Ubuntu server is going to give you more server related options during the install.  The last time I checked it did not install a GUI by default.

Using ether you can still get what you need.  Ubuntu is Ubuntu, it uses the same repos and has the potential to be what you need it to be.

Based on what you have said so far I would recommend going with the full Ubuntu install, or Xubuntu to get a lighter GUI.  Then you can install LAMP and get it configured to serve web pages.  Samba is also a simple install at that point.
