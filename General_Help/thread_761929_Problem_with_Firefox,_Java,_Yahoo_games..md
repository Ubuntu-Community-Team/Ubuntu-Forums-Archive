### Post by FluffyElmo on 2008-04-21
You probably have to install the java plug-in, enter the following from a terminal (*Applications->Accessories->Termina*l) hit enter and it should download and install the package:
```
sudo apt-get install sun-java6-plugin
```

You can also add most browser plug-ins by clicking Applications->Add/Remove... and then searching for the package you need. Also, If you're new to Ubuntu, [http://ubuntuguide.org/]("http://ubuntuguide.org/") has a lot of how-to articles for problems you may encounter.
