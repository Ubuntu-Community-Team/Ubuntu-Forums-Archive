### Post by ginbuntu on 2008-01-02
I asume you are using SCIM. 
Right click on the keyboard icon on the panel and select setup. On the setup window, there is a menu item called Global setup. IN there you can add hotkeys to switch between languages. 

Hope this helps.

### Post by Vixis on 2008-01-02
right click on the panel keyboard indicator -> keyboard preferences -> layout options -> Group Shift/Lock behaviour.
choose combination for shortcut.
