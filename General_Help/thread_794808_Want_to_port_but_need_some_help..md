### Post by NightwishFan on 2008-05-15
Yes actually. You can simply aptitude install the kubuntu-desktop package. Choose kde at the login screen, and it will boot, then log out and gnome is available the same. The only difference is that the menus will have both gnome and kde apps but they will work the same in both enviroments, with no performance loss.
