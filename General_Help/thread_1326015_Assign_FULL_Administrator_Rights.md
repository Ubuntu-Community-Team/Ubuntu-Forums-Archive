### Post by Windows Nerd on 2009-11-14
You have full administrator rights with your password through sudo. The Ubuntu Security Model is very different than that of Windows. It requires you type that password when performing administrative functions so that malware could not install itself without you knowing, like what happens in Windows. 

If you have in mind that Ubuntu is a free version of Windows with a different theme, *its not*. Ubuntu does things differently because it is Linux, not Windows.  

Yes, I know it may be annoying to have to type in your password so many times, but it makes up by having a more secure system overall. And, no having to buy antivirus, like you may have had to with your old Windows system.
