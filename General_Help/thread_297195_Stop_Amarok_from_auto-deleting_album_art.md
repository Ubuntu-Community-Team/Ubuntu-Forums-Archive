### Post by dissidentcitizen on 2006-11-13
i too noticed that amarok seemed to delete art that I had set manually if I set different files. I think the way to get around it is this.

1) Download the picture over the internet

2) Save the picture to same folder as the album

3) Rescan the collection

4) Restart amarok

5) The new art should be auto added

so if you change pictures always rescan the collection and restart rather than using the "set album art option"
