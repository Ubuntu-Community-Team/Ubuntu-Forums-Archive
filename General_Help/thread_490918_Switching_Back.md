### Post by lisati on 2007-07-03
Check your HD for a recovery partition. My desktop has one which has a "non-destructive restore" as an option which seems to leave the Ubuntu partition untouched, as well as user data. My laptop came with a recovery DVD but sadly it didn't have the "non-destructive" option.

*It's always a good idea to make sure you make a backup of your data every so often, particularly before major changes to your system. It can save you much time if things get messed up and you have to start from scratch with an empty computer*
