### Post by Cheesemill on 2012-06-27
What you're attempting isn't going to work. The recovery disks that you have will only work to recover the factory Windows installation onto your actual computer hardware.

To install Windows into a VM you need a Windows installation CD (or .iso), a recovery CD isn't the same thing and won't work.
