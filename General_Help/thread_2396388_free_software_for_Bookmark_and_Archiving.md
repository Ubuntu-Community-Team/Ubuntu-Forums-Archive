### Post by TheFu on 2018-07-14
I use nextcloud's bookmarks addon.  [https://github.com/nextcloud/bookmarks](https://github.com/nextcloud/bookmarks) , but it is part of a nextcloud server instance. This means any client I use has access to the same bookmarks which can be added, modified, deleted by any of my clients anywhere in the world.

Nextcloud is pretty great, actually.

But I suspect that is more than you want.
