### Post by Kateikyoushi on 2007-03-17
I could only guess why did it happen, something changed the uuid in your fstab would be my first bet.
So to make it work at boot let's change that first.

```
sudo cp /etc/fstab /etc/fstab.backup
sudo nano /etc/fstab
```

Then scroll down and change the fstab to look like this.

```
# Entry for /dev/sda4 :
/dev/sda4 none swap sw 0 0
```
CTRL+X to exit and Y to save the changes.

Then the two commands which activated your swap one more time and test whether it is active on boot with a reboot.
