### Post by ubulal on 2010-09-10
1. Start doing backups regularly. Now!
2. I presume you rm'ed it or empty'd the trash bin: only mount the partition read-only, make a working copy image of it and let some recovery app have a go on it (e.g. [http://extundelete.sourceforge.net/](http://extundelete.sourceforge.net/))
3. If that's not successful send the drive to a professional data recovery company or just live with the loss or both.
