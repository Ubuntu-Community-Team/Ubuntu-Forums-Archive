I am pleased to announce the release of Wubi 8.04 beta (rev 457).

This build is slightly different from the one you find on the beta Live CD (rev 449) since it incorporates some fixes and translations that we were not able to ship with the CD. There will probably be other beta builds in the coming days. The wubi-installer.org website has been updated accordingly.

Feedback welcome as usual.
