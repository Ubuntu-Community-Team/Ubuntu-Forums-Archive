### Post by trippinnik on 2008-01-26
Showing off some of the cool stuff in Linux is a good idea, but initially I'd recommend sticking to the things you use daily and are stable.  Compiz is pretty stable now, but there are still enough rough edges in e17 that you may be embarrassed when things start crashing or it doesn't do something right.  You may not be able to anticipate what your friend is going to run or try to do with whatever you set him up with.  
Again for 3, stick with the stuff you know really well.  If you use Ubuntu, show him Ubuntu.  You don't want to give a bad impression by not being familiar with what you are showing him.  
4. Let your friend go ahead and use OpenOffice, all the functionality is there as well as PDF output which makes certain your docs will show up the same on another pc. 
5. Check [http://appdb.winehq.org/](http://appdb.winehq.org/) The wine website has a lot of info on compatible programs and ways to get it working if necessary.
6. If you find out more details about the hardware, that'd be helpful.  GFX card, nic, or wireless are all pretty vital. Mp3 player etc...
