### Post by gettinoriginal on 2009-06-03
Compiz is default in Ubuntu.  For Compiz configuration you need to install "CompizConfig-settings-manager", after installation you will find it in System > Preferences > CompizConfig Settings Manager.  Welcome to Ubuntu :p
