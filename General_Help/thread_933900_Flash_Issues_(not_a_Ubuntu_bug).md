### Post by arkticcool on 2008-09-30
See if you have the plugin's installed correctly, and make sure they are enabled in the browsers.

In FireFox you simply type in the following in the address bar:
```

about:plugins

```

It should display a list of plugins installed and enabled in FireFox. If flash or any of the plugins are disabled or not installed simply go to: [https://addons.mozilla.org/en-US/firefox/browse/type:7]("https://addons.mozilla.org/en-US/firefox/browse/type:7") and uninstall your previous plugins and install with the plugins from the above url. I cannot assist you with Opera, never used it.
