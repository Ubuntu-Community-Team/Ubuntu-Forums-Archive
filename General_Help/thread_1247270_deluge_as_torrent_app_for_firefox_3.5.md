Can anyone point me to how to make Deluge the default bittorrent app and then to make Firefox aware of it as said default (or at least so it shows up in list of possible apps, so I can make it the default within Firefox)?

### Post by michy99 on 2009-08-22
If you have a torrent file handy, right-click on it and select Properties. On the Open With tab, select Deluge if it is listed. If it is not, Add it. The executable is located at /usr/bin/deluge.
