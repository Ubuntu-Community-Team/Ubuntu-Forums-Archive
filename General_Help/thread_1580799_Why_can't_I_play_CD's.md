### Post by 3Miro on 2010-09-23
If you go to Places -> Home you will open Nautilus (the file manager). On the left side of it (next to the big icons with Downloads/Documents and so on), there should be a list of special places: Your Username's Home, Desktop, File System, Network and hopefully Audio CD. If you click on it, on the top right corner of the window you should see "Open with Rhythmbox". Open it, add the files to the disk and enjoy.

Alternatively, you can go to Applications -> Sound and Video -> Rhythmbox and select the CD on the left.

If this fails, try another CD or a CD that doesn't have music, but files (like pictures, or the Ubuntu CD itself). There might be a problem with the CD itself.
