Sometimes you can make it work by running the windows version of your browser in WINE.  

Plugins are just binary executable code that is called by the browser for certain content.  Just because a browser is cross-platform doesn't mean the plugins necessarily are.  Like any binary executables, they are written and compiled for a specific platform.
