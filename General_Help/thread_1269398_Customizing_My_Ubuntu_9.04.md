### Post by rocket16 on 2009-09-18
Go to /var/cache/apt/archives. Now, select all packages from the windows. And, copy them to your CD or DVD (or pendrive). You are done. Now, to make it work, copy all packages from the CD to any Computer. Keep them is a folder, and open a terminal. Now, type cd /path/to/folder (if you place the folder named packages on desktop, then type "cd /home/username/Desktop/packages"). Now, type "sudo dpkg -i *.deb and wait for the command to finish. Now, all your applications are installed.
