### Post by jamaisvu on 2008-01-01
[FONT=Franklin Gothic Medium]I'd also suggest installing kgtk-wrapper (search the forums for how to do this -- it's a bit of a pain, but IMHO worth it), as this will let you have proper KDE file dialogs in Firefox.

Oh, and [FONT=Courier New]apt-get install firefox[/FONT] will only work in a root konsole session. In a normal konsole session, you'll need to type [FONT=Courier New]sudo apt-get install firefox[/FONT], or alternatively use Adept Manager (in K&#8594;System -- this is a gem of a program that will help you find all sorts of packages you never knew existed) or -- easier still -- Add/Remove Programs (should be in the first level of the K Menu).
[/FONT]
