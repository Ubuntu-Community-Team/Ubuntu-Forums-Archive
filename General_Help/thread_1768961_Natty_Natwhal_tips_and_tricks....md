### Post by ThatCoolGuy220 on 2011-05-27
How to add an application to unity 2D launcher?

Also Works with:3D

Its quite simple:

1- Open the application you would like to dock to the panel.

2- Look for the application on the launcher.

3- Right click on it.

4- Select "Keep on launcher" option.

5 - you are done.

The first link is wrong, because you shouldn't need Web8 to install applications when you can download them with a PPA, like you are supposed to do it. 

And the second link is wrong, because if you check off that setting for automatically detecting your refresh rate you -may- lock yourself out of your environment and be forced to drop to recovery to run your video in Failsafe mode to reset them.  Some new users may unfortunately follow these links and try to use them, and seriously bork their new OS.   That can be very annoying if you are a new user and don't understand the recovery mode feature.

Both of those guides above are terrible ways to add software and make changes to your system. Don't use them.
