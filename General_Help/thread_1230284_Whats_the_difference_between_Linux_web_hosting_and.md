### Post by AlexDudko on 2009-08-03
Difference is in server platform. Difference could also be in modules installed. For instance, on a linux server you won't be able to use code, written in .asp, there could be some other peculiarities. If you use for your site html+php/perl linux hosting will do just fine (perhaps, windows hosting will also provide this functionality). But if you use some platform dependent technologies, then maybe only windows hosting will suit you.
