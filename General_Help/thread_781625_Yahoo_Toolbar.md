### Post by tmshih on 2008-05-25
Sometimes the Firefox User Profile can get messed up if you crash out during a session and the toolbar will be unable to load the usual icons. 

IMPORTANT: If you have any Firefox Bookmarks (not yahoo bookmarks) back them up before changing the user profile! 

Close Firefox. 
Open Terminal from Applications, Accessories
Type "firefox -ProfileManager"

Delete the default profile and create a new profile

Note: You will have to reinstall all your extensions and redo all your settings, homepage, passwords, etc
