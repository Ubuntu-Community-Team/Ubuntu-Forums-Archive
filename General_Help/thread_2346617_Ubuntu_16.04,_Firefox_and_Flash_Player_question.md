### Post by Dennis N on 2016-12-16
Ctrl+Shift+A (or in the Firefox menu Tools > Add Ons) will display the Add-Ons Manager. Select "Plugins" on left to see what you have. Flash Player is called "Shockwave Flash" there when installed. You can choose how it behaves with the drop-down menu for the plugin on the right. You don't start it yourself. Firefox runs it when needed.
