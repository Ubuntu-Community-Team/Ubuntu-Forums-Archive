### Post by ryanhaigh on 2008-05-22
You might want to google transmission and libnotify (the system used for those popups), or check the transmission site.

EDIT: [http://trac.transmissionbt.com/ticket/627](http://trac.transmissionbt.com/ticket/627)

The bug has been marked as fixed so I would assume it would be included in the next release although you would need to check with the package maintainer as to when this might be included in ubuntu or whether you will have to install from an external source if you want libnotify integration.
