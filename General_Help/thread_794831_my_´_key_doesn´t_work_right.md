### Post by sdennie on 2008-05-15
Are you using more than one keyboard layout?  I use Spanish (Spain) and English for keyboard layouts and the ´ behavior your are describing is what happens when I'm using the Spanish keyboard layout.  By default, I believe that hitting both Alt keys will switch the keyboard layout but, you can also right click on an empty part of the panel and click Add to Panel...  Then add the Keyboard Indicator applet.  That will show what layout you are using and clicking on it will switch layouts.
