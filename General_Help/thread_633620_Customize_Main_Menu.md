### Post by oeolycus on 2007-12-06
There's not a ton of customization you can do with the gnome menu, but here's a few ideas:
[I][B]
Make a new gnome-panel and try experimenting with the various "add-ons."[/B][/I]

I don't have Ubuntu up right now, but I know there is one that is a more compact version of the default menu (the one on a fresh install or LiveCD) that's a lot like the Windows Start menu. From what I remember you can edit and add new applications and menus for applications. In the menu editing gui you can eliminate certain unused system administration/preference items but I'm not sure if you can altogether blow away the menu.

[B][I]Experiment with panels and drawers.
[/I][/B]
I made my own "poor-man's OSX dock" by playing around with menu items in a fresh panel. There's a lot you can do and it's not too difficult.
[B][I]
Install fluxbox, icewm or a different window manager[/I][/B]

Fluxbox has a really slick menu system with a lot of customizability. It's not *too* difficult to learn, but it requires some tinkering with the equivalent to Windows .ini files.
