### Post by Hazed on 2008-03-23
I dont think there is anything out there quite like what your after.

You could always buy an external USB hard drive for around £40-£50 and install windows onto it, and have your main hard drive dedicated to Ubuntu.

That way, it will boot into Ubuntu as default, but if you connect the USB hard drive you can go into the One-Time boot menu and select "Boot from External Device" to boot Windows.

I do that, but the opposite way around (Ubuntu on USB hard drive, Windows on main)
