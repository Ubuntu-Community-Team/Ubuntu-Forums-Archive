### Post by 3Miro on 2009-04-16
NVIDIA recently messed up the drivers for KDE 4.* and that was a big point against them. If you are going to use KDE, go for ATI.

It used to be the case that ATI had very poor Linux support, however, since AMD acquired ATI things have changed dramatically. I don't think either one is inferior from the support point of view.

I personally use NVIDIA.
