### Post by drs305 on 2010-04-10
I haven't used it but I installed it via Synaptic. You can then highlight it in the right pane of Synaptic, select "Properties" from the main menu, and then click on the "Installed Files" tab to see where it's files are installed.

Tip: For normal applications (but not gtalk in this case), you can find out locations and start commands by typing "which <appname>" and "whereis <appname>" in a terminal. *gtalk* isn't a standalone application which is why those commands won't work in this case.
