### Post by findlj52 on 2017-01-02
Hello,  I'm trying to restore a single file with the 'backups' program shipped with Ubuntu 14.04, which I believe is Deja Dup.  I've run periodic backups from which I'm trying to grab this file.   However, when running the 'Backups' program, the only options I see is to restore an entire folder.   I've also read to restore a single file from the 'Backups' program, I'm supposed to use the 'Files' program.   I don't see how to use 'Files' to extract a single file from the many '...difftar.gz' files listed in my backup directory.  
Thanks for the help, in advance!

---

### Post by deadflowr on 2017-01-02
In Files, don't go to where the backups are, go to where the file to be restored would be.
Then right click and you will get an option to either restore missing files (if you click on an empty area of the folder)
or restore previous version (if you click on a file in the folder)

Hope it helps
