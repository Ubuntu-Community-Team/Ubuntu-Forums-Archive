### Post by em3raldxiii on 2009-12-05
Looks like Grub2 uses /boot/grub/grub.cfg instead of menu.lst:  DO NOT EDIT THIS FILE This is the main Grub 2 file. It "replaces" Grub Legacy's /boot/grub/menu.lst. This file contains the Grub menu instructions. Unlike Grub Legacy's menu.lst file, grub.cfg is NOT MEANT TO BE EDITED!!!
