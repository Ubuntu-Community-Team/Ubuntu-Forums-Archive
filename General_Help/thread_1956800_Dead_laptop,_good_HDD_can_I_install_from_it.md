### Post by gufide on 2012-04-11
You can, from another computer, copy the /home/username folder which contain all your data and settings. All you will have to do is reinstall your applications. To copy it, simply plug them with sata wire or if you can, simply plug it with usb wire. I made my /home directory on another partition so I can format my openrating system without loosing any data, and I think everybody should, because it saved me a lot of time.
