### Post by yabbadabbadont on 2009-09-12
If you just need something to handle the NZB files, then hellanzb might be what you need.  It supports SSL and a large number of connections.

About the only decent News reader is Pan, but it doesn't support SSL, you have to create an SSL tunnel with something like stunnel, and then point it at that.
