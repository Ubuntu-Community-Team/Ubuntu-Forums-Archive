### Post by CaptnIgnit on 2010-01-04
Searched forever on how to get an icon on the desktop that will give you the shutdown menu. Every post I found said it wasn't possible, but it can be done (and very simply too)!

FYI, this was done in Karmic (9.10)

1. Right click on desktop and select "Create Launcher..."
2. Give it a name and in the command field put "gnome-session-save --shutdown-dialog"

I also found the icon "\usr\share\icons\Humanity\actions\48\application-exit.svg" to be well suited to replace the standard launcher icon.
