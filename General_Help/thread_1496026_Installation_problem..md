### Post by coldraven on 2010-05-29
Sounds like you downloaded the "Desktop" version of Ubunto 10.04
The rectangle and stick man is meant to mean "Press any key for manual install".
So try pressing a key and see what happens.
Maybe the installer will inform you of a problem.
Or download the "Alternative" version that has a text based installer.
See this page [http://releases.ubuntu.com/10.04/](http://releases.ubuntu.com/10.04/)

Make sure that you are not trying to install a 64bit version if you only have a 32bit box.
Keep trying, it will be worth it!
