Never mind.  I figured it out.  For future guidance, I did the following:

1.  Right-click panel.
2.  Panel Options > Panel Settings.
3.  Select ADD WIDGETS.
4.  Highlight the "Windows and Tasks" category.
5.  From the Windows and Tasks category, select the Task Manager widget.
6.  Right-click panel.
7.  Panel Options > Panel Settings.
8.  Move the Task Manager to the desired location on the panel.

### Post by amingv on 2009-06-06
Do you mean the one that shows the applications running in the background? If so, it's quite easy:

1. Right Click on the panel (on an empty space of it)
2. Panel Options>Add Widgets
3. Look for a widget called "System Tray" and add it.

After you're done, you may want to lock the panel/widgets so you don't mess them up by accident.
