### Post by TeoBigusGeekus on 2010-08-04
The drivers are indeed automatically installed.
You can test the compatibility of your system with the live cd.
Download it, burn it and boot with it. 
If ubuntu recognizes everything, you can proceed with a full installation (dual boot or not).
