### Post by drs305 on 2008-06-15
I opened an OO doc and then crashed the system to see how the recovery page opens. For me, the OpenOffice recovery page starts with a display of the documents that can be recovered. 

All I have to do to clear the crash and have the document recovery begin is to hit the return key. The other option is to press "S" for "S"tart Recovery. So if you boot and then open the OpenOffice program, for me at least, hitting the Enter key after OO has opened starts the recovery and opens the file when it is done. It shouldn't matter whether or not you can see the entire recovery page as long as the OpenOffice is in the current window.

It's worth a try at least. Good luck.

### Post by Francewhoa on 2010-01-15
To fix this delete the file ```
Recovery.xcu
```If you're using Open Office version 3.xx this file is located under ```
/home/YOUR-USER-NAME-HERE/.openoffice.org/3/user/registry/data/org/openoffice/Office/Recovery.xcu
```If you're using Open Office version 2.xx this file is located under ```
/home/YOUR-USER-NAME-HERE/.openoffice.org2/user/registry/data/org/openoffice/Office/Recovery.xcu
```
