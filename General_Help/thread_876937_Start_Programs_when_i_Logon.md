### Post by nisden on 2008-08-01
Just want to know if there is any way i can run programs when the computer start? Like open firefox and so on.

---

### Post by Krydahl on 2008-08-01
Yes, add the app you want to run to system->preferences->sessions.

---

### Post by Choad on 2008-08-01
additionally, you can "save" the currently running applications, or set it to always remember what apps you are running when you log out. both these options can be found in system> admin> sessions> session options
