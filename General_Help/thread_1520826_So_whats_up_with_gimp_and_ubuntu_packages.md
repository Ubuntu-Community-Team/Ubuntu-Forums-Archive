Hi,
My guess is that Ubuntu developers take the time needed to test packages, then apply patches where needed, prior to letting them into the official repositories. So some packages will probably take more time to get updated.

But I just found a PPA repository for Gimp via Google.
[http://www.howtogeek.com/howto/12047/install-gimp-2.7.1-on-lucid-lynx-using-ppa/](http://www.howtogeek.com/howto/12047/install-gimp-2.7.1-on-lucid-lynx-using-ppa/)

It is currently at gimp 2.7.3 so consider this version of Gimp to be very experimental. 
[https://launchpad.net/~matthaeus123/+archive/mrw-gimp-svn](https://launchpad.net/~matthaeus123/+archive/mrw-gimp-svn)

I'm using the default gimp version 2.6.8 in Kubuntu Lucid, so I haven't tested this PPA personally.
Hope that helps.
