### Post by aysiu on 2006-07-04
I would download the Alternate CD for two reasons:

1. I've read a lot of people having trouble with the Desktop CD's graphical installer, particularly when they choose the manually edit partition table option.

2. You can upgrade directly from the Alternate CD by using it as a local repository. You cannot do this with the Desktop CD.
