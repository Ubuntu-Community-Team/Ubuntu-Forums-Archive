If anyone can help me out, I would be very grateful.  I really don't want to re-install cause this is my work machine and I have vmware workstation and some other commercial software that is a PITA to re-install.

Everything was working under 7.10, during the upgrade it complained about not being able to upgrade Xorg.  I was able to get X fixed, but apparently not 100%.

Solution:
The 8.04 upgrade broke the symlinks /usr/lib/nvidia/libGL.so.1.xlibmesa and /usr/lib/xorg/modules/libglx.so that were probably part of the old Nvidia driver version 100.69 install.

I pointed the libGL.so.1.xlibmesa symlink to libGL.so.1.2.xlibmesa and I removed the /usr/lib/xorg/modules/libglx.so symlink.

This caused the GLX module to finally load.  Then I was getting a "Composite module could not be found" error when running compiz --replace, I then installed xserver-xgl and now everything works.
