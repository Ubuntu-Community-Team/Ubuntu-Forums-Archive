### Post by tamoneya on 2008-08-17
you need to use testdisk.  Open up terminal and type:```
sudo apt-get update
sudo apt-get install testdisk
sudo testdisk
```

Then have testdisk scan your iPod for any files.  Then browse through the files and tell it which ones to recover.
