### Post by snova on 2008-11-06

### Post by Paki on 2008-11-10
Good call, snova - that does work!

However, note that you also have to make a separate request through Adept to install Compiz; changing the Window Manager setting doesn't install or prompt you to install Compiz if it isn't there.

One bit of good news: Nothing breaks if you set things up to use Compiz instead of Kwin but don't actually have Compiz installed: The system just goes on using Kwin.

Switching from Kwin to Compiz has produced an enormous increase in desktop performance for me.
