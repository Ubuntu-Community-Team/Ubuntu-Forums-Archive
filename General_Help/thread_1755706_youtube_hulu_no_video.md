### Post by khelben1979 on 2011-05-11
If you want to try and replace flash with Mozilla Firefox, check here: > ~/.mozilla/plugins/libflashplayer.so

Download the flash package [from the Adobe website]("http://get.adobe.com/flashplayer/") and make a backup of the old file and copy in the new one to see if it helps. Google Chrome for instance have it's own version of flash inbuilt in the web browser and does not get affected by this change.
