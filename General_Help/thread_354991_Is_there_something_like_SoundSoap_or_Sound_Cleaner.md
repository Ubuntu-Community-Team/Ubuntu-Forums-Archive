### Post by FluffyElmo on 2007-02-07
Try Audacity, it's a great general purpose audio editor application. There are a large number of plug-ins available, some must address your specific needs. You should be able to find it in Synaptic, or just run *sudo apt-get install audacity* from a terminal window.

More information on the program and plug-ins available here...
[http://audacity.sourceforge.net/]("http://audacity.sourceforge.net/")
[http://audacity.sourceforge.net/download/plugins]("http://audacity.sourceforge.net/download/plugins")

Good Luck! If you have any problems just let us know...
