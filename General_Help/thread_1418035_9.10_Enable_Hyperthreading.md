[INDENT]I have looked all over the internet and have only been able to find instructions to enable HyperThreading on legacy versions of Grub, but not Grub2. Any ideas?[/INDENT]

If you're using a stock SMP kernel, then Hyperthreading should be enabled by default - no special tricks required to activate it.

In a terminal window, "cat /proc/version", and see if you've got a SMP kernel running.  Here's an example:```
lloyd@ubuntu:~$ cat /proc/version
Linux version 2.6.31-19-generic (buildd@crested) (gcc version 4.4.1 (Ubuntu 4.4.1-4ubuntu8) ) #56-Ubuntu [COLOR="Red"]SMP[/COLOR] Thu Jan 28 02:39:34 UTC 2010
```
