### Post by Holger_Gehrke on 2018-09-29
Of course there's a Fortran compiler available in the Ubuntu repositories. It's part of the Gnu Compiler Collection and uses the same code generation back-end as gcc. You can install the compiler with 'sudo apt-get install gfortran'. The documentation in gnu-info format for the package can be found in the package 'gfortan-doc'.

Holger
