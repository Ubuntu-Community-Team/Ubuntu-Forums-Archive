### Post by mcduck on 2008-07-26
You can increase the font size in the same appearances-panel where the themes are selected.

Text size is independent from the theme, I'm not sure if it's even possible for themes to define font sizes. 

640x480 resolution shouldn't be off the screen. But yeah, of course if you have two dispalys, one with high resolution and the other with 640x480, and you try to clone the big resolution desktop to the small one it has to be scrolled. This can be solved by using the same 640x480 on _both_ displays, so that your desktop _is_ only 640x480.
