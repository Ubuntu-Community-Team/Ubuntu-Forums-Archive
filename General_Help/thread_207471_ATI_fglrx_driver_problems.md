### Post by OniAnubis on 2006-07-01
So I was having this problem with the fglrxinfo command showing that I was running mesa drivers. I changed my kernel to the amd k7 kernel with synaptic and it failed to install the linux-restricted-modules for the k7 kernel. I installed them and it solved the mesa driver problem for me. I hope this helps anyone with the same problem I had. It was driving me crazy

I have an ati X800 agp card.
