### Post by markbuntu on 2008-07-19
Some people put

LIBGL_ALWAYS_INDIRECT

in the start up script for compiz to supposedly preclude video flickering in windowed mode.  Compiz uses indirect rendering with the ATI drivers, it is a stupid thing, but there you go. Unfortunately, this pushes all rendering into indirect mode and loads the cpu as you have noticed.

If you can find out where that line is, you can comment it out and get your direct rendering back.
