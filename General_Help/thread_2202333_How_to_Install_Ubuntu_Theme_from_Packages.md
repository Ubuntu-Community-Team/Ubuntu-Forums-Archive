### Post by Frogs Hair on 2014-01-28
Both locations you have placed the themes are correct meaning .themes or usr/share/themes. The themes you have selected may not be compatible with the version of Ubuntu you are using. You have to read the theme description and  an example is that GTK 2 themes won't work in Gnome 3 and Gnome 3.2 themes won't work in latter version of Gnome 3.  

The two currently supported versions of Ubuntu are 12.04 which uses GTK 3.4 Themes and 13.10 which uses GTK 3.8 themes.
Installing from the ppa is pretty simple also if you trust the source . See the Link: [http://www.noobslab.com/2011/11/themes-collection-for-ubuntu-1110-unity.html](http://www.noobslab.com/2011/11/themes-collection-for-ubuntu-1110-unity.html)
