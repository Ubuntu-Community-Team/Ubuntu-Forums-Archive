### Post by roastedmnm on 2008-05-16
I know that ubuntu is gnome based, but I finally found a suitable solution to this problem and thought I would share.

I first found my GNUcash file (instead of exporting anything) then I opened it in KMyMoney. KMyMoney then exports of qif (and has much nicer reports than GNUcash).

Since I rather like the format of GNUcash, I'll probably continue using it but when I need to export, I'll open up a new KMyMoney file.

Hope this helps someone.
