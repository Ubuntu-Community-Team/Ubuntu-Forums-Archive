EDIT: Nevermind, I figured it out. 

And just incase anyone else runs into this problem, all I did was map the old keys to new keys. So take the Hint command for example, what I did was:

:map <whateverkeyyouwant> f

And for switching between tabs:

:map <whateverkeyyouwant> gt

And so on... The other thing I had to figure out though when I was remapping these functions is that regular keys don't go inside <> brackets. Like seen in my above examples. But something like C-left(which is Control+Left) do go in the <> brackets.

So an example on keys that go in <> would be:

(This is to go back or forward a previous page) - :map <A-left> <C-o>

Pretty weird. Anyways, hope someone finds this useful.
