### Post by Joe88 on 2008-06-01
Yes, I downloaded a free program for windows which lets you assign mouse and keyboard buttons to control sticks and buttons on game pads. Heres a good program which is free to try:

[http://www.download.com/Pinnacle-Game-Profiler/3000-2121_4-10414015.html?tag=lst-1&cdlPid=10823576]("http://www.download.com/Pinnacle-Game-Profiler/3000-2121_4-10414015.html?tag=lst-1&cdlPid=10823576")

You will probably not like using a gamepad instead of a keyboad and mouse - I certainly didn't. Yes it nice to have all the game keyboard actions assigned to buttons on a controller, but the control sticks don't come close to giving you the level of movement control you get with a mouse and keyboard.
