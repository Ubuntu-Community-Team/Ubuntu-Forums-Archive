### Post by howefield on 2010-02-14
Look at appdb.winehq.org/ for info on compatibility of specific windows applications.

But as already mentioned, not all programs will work well with wine, and in that case, you could try virtualising a windows environment with something like Virtualbox.

This will allow you to boot windows from within Ubuntu, running both operating systems side by side.

There are limitations to this, such as a need for a windows license, to have the hardware capability, eg, enough RAM to run two operating systems, and a few software issues, such as running 3D games. But it is another option for you to consider.,
