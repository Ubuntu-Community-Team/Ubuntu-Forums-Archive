### Post by Tootler on 2009-02-18
I have a collection of song and tune books I have made for my own personal use. I produce these using Open Office Writer and insert the musical notation which is in the form of Windows Enhanced Meta File (*.EMF) images using insert/picture, then editing the image to fit where it is needed.

The images are created by my musical notation processor which allows export of the notation as an image but only in EMF format. No other option is given.

These display fine with OO when running under Windows, but when I tried to open the files with Linux, the images do not display properly. The stave lines and the words of songs display OK but all the notation elements are missing and are replaced by little squares approximately where the notes etc. should be.

I could try converting the images to bitmaps (EMF is a vector graphics format) but this is not practical as I have about 400 of these images and it would be a major task.

The same thing happens if I open the image in Open Office Draw, also it occurs both in Ubuntu and in Linux Lite on my netbook, so the problem could be a more general one.

I am using Open Office 3.0.1 but the problem was there in OO 2.x
