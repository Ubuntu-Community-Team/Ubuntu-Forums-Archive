### Post by prshah on 2008-09-02
> **jjahn said:**
> 
I'm looking for a good way to learn commands and be proficient with the terminal window.  Is there some sort of database with all the commands, 

At the terminal window, give the command ```
info coreutils
```

This will give you a list of common commands, tasks, information, caveats and usage. Details on each command can be had by navigating to it and pressing enter.
