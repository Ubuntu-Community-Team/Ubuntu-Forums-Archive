### Post by mikmalot on 2011-01-07
Okay, I figured it out:

The major problem was actually so incredibly simple I can't believe it didn't occur to me beforehand: switch my input settings in my sound preferences. The only problem with this is I have to reset it whenever I plug in my webcam/microphone.

The Skype Problem was actually a pretty easy fix, too: I downloaded pavucontrol and used it to set my webcam/microphone as the default input.
