### Post by vjkeito on 2008-08-10
if you go to [***getdeb***]("http://www.getdeb.net/") then you can install [***wine-doors***]("http://www.getdeb.net/search.php?keywords=wine") which makes installing some apps in wine a lot easier. (make sure you install the correct version for your ubuntu flavour - hardy/gutsy etc)

I believe WoW is one of the programs supported, though you will have to find the no-cd crack yourself.

hope that helps!
