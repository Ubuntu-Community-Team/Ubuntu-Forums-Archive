### Post by hackb0y294 on 2010-04-30
Download the theme first (check [http://gnome-look.org](http://gnome-look.org)), then right click the desktop and click Change Desktop Background, click themes, and drag and drop the .tar.gz file into the Themes window, and it should install.

### Post by Pratheek on 2010-05-05
Hey guys, i found the way to install the addictive human theme from 9.10 to 10.04 LTS 

if u use the 9.10 somewhere else, or start a live CD session,  then goto usr/share/themes, then copy ur human theme into a pen drive, or to something, then open ur ubuntu 10.04 LTS,  

paste ur human theme form pen drive, to desktop (Since, its easy to apply)


Right-Click>Change desktop background>themes and drag ur Human theme folder, to where the theme previews are shown .... 

and Voila, u should have seen something like "GNOME Human theme was successfully applied" .
