### Post by sikander3786 on 2012-05-22
You can find it here:

[http://www.libreoffice.org/download/?nodetect](http://www.libreoffice.org/download/?nodetect)

Select your architecture from the .deb installer, select language and download the version you want.

However, it is recommended to install the official repository version so that you keep receiving the updates. And I am not sure how well the old version would work with Precise.
