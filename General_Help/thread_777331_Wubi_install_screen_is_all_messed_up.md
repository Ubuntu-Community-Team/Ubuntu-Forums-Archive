### Post by ago on 2008-05-01
Wubi tends to use the default language you have on the OS (but still lets you choose the language for the installed OS). The idea being that if you use Windows in russian, you shouldn't have much trouble with Wubi in russian too.

Anyway just enter the password and click the middle button.
