### Post by drs305 on 2011-05-31
The blank screen with the blinking cursor most likely means that Grub is working properly and there is a problem after the kernel takes over. This symptom is often associated in Natty with nvidia video cards.

You seem to have already tried this (reference your post) but the normal way to get to a working desktop in this case is to edit the first entry (press 'e'), go to the end of the 'linux' line, and remove everything (quiet splash vt7.handoff) and add *nomodeset* Then press CTRL-x to boot.

If it gets to the Desktop, then press the icon in the top left of the screen, type in 'Additional' and you should see an "Additional Drivers" icon. Click on this, and try to add a video driver.

You can also try booting the recovery mode using the same method above, but instead of 'nomodeset' type 'single' then CTRL-x and when asked choose the 'failsafe graphics' mode.
