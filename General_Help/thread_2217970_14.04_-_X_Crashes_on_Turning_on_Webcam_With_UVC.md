### Post by jasonditz on 2014-04-19
update for others with similar problems:

The issue is related to AMD's proprietary Catalyst 14.x drivers. Uninstalling them and using the open source drivers resolves the problem, though obviously then you're stuck with the much slower open source drivers in 3.13.

In my case I use Skype, etc so sparingly I'm just going to leave Catalyst running, and uninstall it for those rare occasions when I really need uvc-based apps.
