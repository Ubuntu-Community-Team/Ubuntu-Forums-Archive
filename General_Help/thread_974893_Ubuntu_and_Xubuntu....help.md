### Post by TransitMan on 2008-11-07
1) Ubuntu is GNOME built, Xubuntu is XFCE with some GNOME added for functionality. 

2) If you install 8.10 in the GNOME version and then added XFCE via "sudo apt-get install xubuntu-desktop", then you have both desktop enviroments installed. No need to download 2 .iso's.
The command line installed those packages related to the functionality of Xubuntu/XFCE when you log into it.

3) Different desktop enviroments mean you need to learn how to use the XFCE over GNOME. 
The "file cabinet" icon you refer to is located where?
In Ubuntu, the default file manager is Nautilus, in Xubuntu it is Thunar.
Right click capability in Xubuntu is not going to be the same as GNOME, as both desktop enviroments are different and therefore will function differently.

I have used KDE, GNOME and XFCE at different points in my journey in Ubuntu land. At any one time, I may switch between all 3 in a given week, or stay with one DE for a month or more. It really comes down to what you are comfortable with and how much "tinkering" you want to do in each DE to get it the way "you" want.
Best advice, give it time. Learn each DE and then decide which one you really want to stick with, or just change-up every now and then for a different look.
