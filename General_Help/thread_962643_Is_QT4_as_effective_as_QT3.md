### Post by snova on 2008-10-29
Qt is a library. More specifically, it's a BIG library that encompasses everything from XML parsing, GUI's, networking, IPC, ....

Qt 4 makes significant additions to Qt 3. If you're programming, go with Qt 4.

KDE 3 is built on Qt 3. More recently, KDE 4 is written with Qt 4. Whether you want to stay with KDE 3 or jump to KDE 4 is a personal decision.

KDE 4 makes some interesting and unique technical decisions that makes the desktop function in a vastly different manner than is traditional. I think it's for the better, but try it for yourself.
