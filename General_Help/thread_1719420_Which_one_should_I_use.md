### Post by Tamlynmac on 2011-04-01
I can't speak for others, but I had netbook on my mini and absolutely hated it. I switched to 10.04 (LTS standard 32 bit version) and with just a little creativity you can make the desktop extremely user friendly and functional.

See this [thread]("http://ubuntuforums.org/showthread.php?t=1588279"), it may help help you with some creative ideas. I set my wife's 15" laptop up exactly the same way and she's very pleased with the results.

Please remember that which version you use or how you set it up is a personal decision. You can request feedback with regards to potential, however the bottom line is that it should meet your expectations.

Good Luck

The difference between the netbook and desktop is in the interface, nothing else. Interface is something personal, it depends on how you use your machine.
