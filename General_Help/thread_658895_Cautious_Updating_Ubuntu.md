Now Since Updating my system Ruined it the last time, I am being over cautious with this one.

### Post by kellemes on 2008-01-05
Take the updates one by one..
Handpick a couple of packages from Synaptic to update, see if the system still works, reboot to be sure.. and handpick the next bunch of updates.
Take it slow, there is no rush needed.

This way, if stuff goes wrong, you can probably guess what update created the trouble.
