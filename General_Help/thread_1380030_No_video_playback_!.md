### Post by khelben1979 on 2010-01-13
If I were in your situation I would learn to install the nVidia driver from [the nVidia homepage]("http://www.nvidia.com") instead. You will get a newer driver.

The X server must be shut down in order for you to be able to install the driver. If you're not an experienced Linux user, I would advice you to read through the install instructions carefully and take your time. I think this is the best way.

It will be easy to do a reinstall of the graphics driver if some update messes it up.
