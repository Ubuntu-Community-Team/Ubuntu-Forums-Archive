### Post by MacAnthony on 2008-06-24
If you have apache installed, then it should be working. The other computers should be able to access your web server by typing in the IP address of the server into a browser. If you want it accessible from outside your home network, then you would need to get access from outside your network to your server. Depending on the hardware you have in place, it would likely be a matter of setting up port forwarding to your server.

If you would like to edit the files, I believe they get installed to /var/www (some one correct me if I'm wrong since I don't have apache installed on my computer ATM).
