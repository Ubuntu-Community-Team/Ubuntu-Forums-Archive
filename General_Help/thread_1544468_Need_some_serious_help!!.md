### Post by gdonwallace on 2010-08-02
Does the board have on-board connections for the monitor?  If so, I would try plugging into that and turning it on in the BIOS.  Also something else you may want to try and hooking that monitor up to another machine / laptop if you have one that will accept an external monitor connection.  Sounds like either the video board is blown, or the monitor.
If it is a newer card, I would start with hooking the monitor up to another computer and see what happens.  Also, if you have a old video card laying around, try putting that in just to test the port you have your current card in.
