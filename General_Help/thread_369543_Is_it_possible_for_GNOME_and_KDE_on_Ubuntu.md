### Post by Dylnuge on 2007-02-24
> **j.miller565 said:**
> Since Ubuntu uses GNOME, can it also use KDE. Like as a  different session. I don't want to install Kubuntu. i just want to try out KDE without having to install Kubuntu

Installing kubuntu-desktop will install Kubuntu without uninstalling GNOME. Kubuntu is just a name for the KDE desktop with a bunch of packages you will need/want. However, it will change your loading screen when you boot your computer (also called a usplash screen).

To change it back, download Start-Up Manager, a very nice Ubuntu tool, and install it. (You must install base before gui). Open it up, change usplash theme to ubuntu, and voila!

When installing kubuntu-desktop, the system will ask you if you want gdm or kdm. Gdm is what you are currently using when you start your system, I recommend it.

Now either log out or reboot. When you get to gdm, click options in the lower left hand corner. Click Choose Session... and choose KDE. Choose "Only for this Session" (Unless, of course, you want KDE to be a Default). Login as usual.

You can switch by logging out and logging back in.
