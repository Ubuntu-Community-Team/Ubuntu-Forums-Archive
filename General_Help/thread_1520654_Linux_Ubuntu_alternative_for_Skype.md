### Post by YannBuntu on 2010-07-04
To do voice/video calls with Windows interlocutors, I recommend either :
- Ekiga (which exists also on Windows)
- Empathy (with Telepathy PPA , and ubuntu-restricted-extras package, and of course a Jabber account e.g. gmail adress), that can do voice/video calls with Windows Gtak interlocuters
- Skype (which exists also on Windows), but this one is not open-source


Of course as Skype is closed-source, the only software compatible with Skype is... Skype ;)
