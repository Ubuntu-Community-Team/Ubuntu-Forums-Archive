### Post by Ziggy72 on 2008-04-24
After quite a bit of Googling I now find that the 'new' glipper is not the same as the 'old' glipper. Getting the new glipper to work is actually quite easy. First, install glipper using synaptic (or whatever method you prefer). Second, right-click on the panel, choose 'Add to Panel' and select Clipboard Manager. Glipper is now on the panel and it works!
