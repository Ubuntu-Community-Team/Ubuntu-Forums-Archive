### Post by ean5533 on 2010-10-18
[This page]("http://code.google.com/p/mupen64plus/wiki/FileLocations") documents the file locations on various systems. It doesn't explicitly mention a cheat code file, but I'm guessing it's in the .config directory.

Remember, if a directory name starts with a period, it's hidden, so you'll need to select "Show hidden files" from the menu to see it in nautilus (or use the "-a" switch when using ls on the command line).
