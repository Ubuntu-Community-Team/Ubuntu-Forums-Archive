### Post by Achetar on 2007-04-12
This morning Ubuntu wouldn't let me login any way except through the fail-safe terminal. I managed to track the error to Gnome. I had recently modified my account, and realized that somehow i had messed it up. In the end I fixed it by logging into a failsafe terminal and ran 'sudo nautilus' to open a file explorer then managed to find my other window manager, beryl, and ran it. Then i ran the gnome panels so that I had the menus and such. Finally I realized to undo what i had done i had to undo my changes to my account. And once I did that it was gone.
Anyone else had a problem like this? Any advice on keeping it from happening again would be nice. I am sure others besides myself would like it.
