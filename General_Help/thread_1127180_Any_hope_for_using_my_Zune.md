### Post by StuartN on 2009-04-16
The Zune is almost completely tied to Microsoft's own music manager. There are a few threads reporting the inability to sync a Zune with non-Microsoft managers ([http://ubuntuforums.org/showthread.php?p=3708120](http://ubuntuforums.org/showthread.php?p=3708120)), and it looks like it will not be possible to use the Zune with any other operating system.
