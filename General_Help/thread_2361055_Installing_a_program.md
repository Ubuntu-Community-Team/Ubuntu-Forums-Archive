### Post by ian-weisser on 2017-05-11
You want to install a pre-release, undocumented, unpackaged, unsigned zipfile?

That's not wise.

Installing java .jar files has been long-discussed in many forums and on YouTube.
"Nothing seems to work" is not a helpful enough description of the problem you are having installing .jar files for us to usefully help you yet.

If you wish to print from your Android device to an Ubuntu printer, look for Android apps that offer 'CUPS' printing. CUPS is the default and networked print stack in Ubuntu. Or IPP directly to a newtwork printer.
