### Post by cptnapalm on 2008-02-03
I recently got a new machine (hmmm Quad core love...) and put this 8500GT in it and initially had some problems; tried manual install of nVidia drivers, Envy install, etc.  Some stuff would work, while other things wouldn't.  I couldn't get Ubuntu to install the restricted drivers for totally unknown reasons.  I got 3d working with the manual install, but Compiz would not run and using full resolution, part of the screen would not get drawn except for the background color.  Other times, the resolution would be fine but no 3d.

I could never figure out what was going wrong, but after a re-install I just used the default drivers from Ubuntu before installing much else and now all is golden.  Compiz works nicely as do 3d games from Ubuntu's repositories.

I think mine is an MSI, but a LOT of the cards are just nVidia's reference board so they are identical.  I don't think they spend much time thinking about how to make a low-end card super-spiffy.
