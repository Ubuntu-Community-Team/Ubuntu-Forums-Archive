### Post by metaphor- on 2007-04-17
physically testing a hard drive is not an easy task, especially if you are not formilliar with a computer.

There are several boot cd options which you can try. I really like "techiez toolkit", just do a search in google to find out more info about it. Basically its a program which preforms tests on all your pc hardware. It also containes several useful tools like partition magic, bios recovery/flash options, ram checking etc. It has to be the greatest program for running diagnostics on physical hardware, even pci or agp cards. There is one scan which tests everything on your computer and fixes it, im at work now so im not sure of that name, but with the menu its not really hard to find. This program helped me discover that a fan on my video card had broken, saved me much worry and time in running a health check. 

If you can download it, burn it to a cd and store it. ITs like a pc doctor
