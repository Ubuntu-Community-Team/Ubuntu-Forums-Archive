### Post by Hagar Delest on 2008-02-26
Have you opened the file that you had actually saved on your HD? I hope you haven't overwritten it with the 'recovered' file.

Else, you can see in your OOo user profile if there is a backup file (~/.openoffice.org2/user/backup). The option had to have been checked in Tools>Options>Load/Save>General and Always create a backup copy.

Autorecovery is sometimes strange, especially under Linux.
