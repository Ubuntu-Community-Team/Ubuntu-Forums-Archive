### Post by Jazzy_Jeff on 2010-07-30
If you click on the program in the task bar it should maximize or minimize. If you are talking about putting it in your system tray you can install a little program from the repositories called AllTray. You can use that to minimize any program to your tray. I just leave a shortcut to AllTray on my top task bar to use all the time.

### Post by ajgreeny on 2010-07-30
