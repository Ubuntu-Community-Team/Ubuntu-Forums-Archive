### Post by cszikszoy on 2008-06-10
try installing with the minimal or alternate CD.  It doesn't include any GUI for the installation but the dos-like interface is pretty easy to use.  I've had problems installing from the LiveCD on some computers and minimal or alternate CD's have worked fine.

If you get the minimal CD you'll need an internet connection to install because it downloads all of the latest packages as you install.  If you have the time this would be the best option because you don't need to update everything once you install.  I've also had the most success with the minimal CD.
