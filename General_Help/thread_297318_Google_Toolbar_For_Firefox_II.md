### Post by msak007 on 2006-11-11
If you're getting an error installing it, check out [this]("http://www.ubuntuforums.org/showthread.php?t=286961") thread. Seems to be a bug in the way the version in Ubuntu's Firefox was coded and what the toolbar installer is looking for. It's an easy quick change to make to the XPI, but there's a link to a file that's already been modified in that thread. I installed Firefox manually using a download from Mozilla's site, and had no problem installing the toolbar from Google's site. BTW my version (Help --> About Mozilla Firefox) says

```
Mozilla/5.0 (X11; U; Linux i686; en-US; rv:1.8.1) Gecko/20061010 Firefox/2.0
```
