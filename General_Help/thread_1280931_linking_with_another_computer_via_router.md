### Post by raysa on 2009-10-02
My netbook running Ubuntu goes online successfully via a wireless router link. Another computer running Windows XP uses the same router without problems. If I give shared status to folders on the Windows box, the Netbook sees them as a 'place' and if I click on them they mount OK.

But even when mounted, they do not appear as available data destinations in applications. How can I make applications 'see' and access the shared Windows folders?

Thanks, Ray

---

### Post by StuartN on 2009-10-02
> **raysa said:**
> How can I make applications 'see' and access the shared Windows folders?

To open a file on a remote share from within an application you can click ctrl-H in the open-file dialogue to show all hidden files and navigate down .gvfs (Gnome virtual filing system) in your home directory.
