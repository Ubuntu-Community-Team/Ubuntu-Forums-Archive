### Post by mohaham on 2005-05-30
by default the root password is disabled in Ubuntu.
this means you have to use your user password to access root.
you can enable the root password by typing this in the terminal:
sudo passwd root

to add  users to the sudoers file u can refer to this..
[http://ubuntuguide.org/#allowmoresudoers](http://ubuntuguide.org/#allowmoresudoers)
[http://www.ubuntulinux.org/wiki/RootSudo](http://www.ubuntulinux.org/wiki/RootSudo)
