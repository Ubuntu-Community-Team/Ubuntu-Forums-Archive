### Post by wildmanne39 on 2012-03-24
Hi, did you upgrade to 11.10? if so just install gnome fall back:
```
sudo apt-get install gnome-session-fallback
```
Reboot and then select "Gnome Classic" on the login screen.

If you want to downgrade from 11.10 back up your important files and reinstall 11.04.
