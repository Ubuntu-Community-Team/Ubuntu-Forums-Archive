### Post by Shazaam on 2008-03-15
So you want to get rid of Ubuntu? Can I ask why?
You can remove Ubuntu the same way you installed it. Boot up the Ubuntu livecd, go to System>Administration>Partition editor and delete the Ubuntu partition from there. You MAY have a problem loading Windows when you reboot. To repair that you will need a retail (store bought) Windows disk the same version as your Windows. Boot with your Windows cd, go to the Recovery Console and do a "fixmbr".
If you are having trouble with a failed install the easiest way to fix that is to re-install Ubuntu to the same partitions that you already have. It will even ask you if you want to save your old user settings.
