### Post by stinkeye on 2012-11-01
[http://sourceforge.net/apps/trac/easystroke/wiki/Documentation]("http://sourceforge.net/apps/trac/easystroke/wiki/Documentation")

The only setting I change is the gesture button to the right mouse instead of middle button.
Takes a bit of practice when drawing gestures. You need to release the button while still moving so you don't bring up 
the right click context menu.

If your button is recognized It's fairly simple to assign a button to a command or key.
eg I set button 9 to initiate the workspace switcher.
The link in my first post should be easy to follow.
