### Post by bryncoles on 2008-07-15
open a terminal and type 

```
gnome-keyboard-properties
```

(or go into system->preferences->keyboard). select the 'layout' tab, and click the 'layout options' dialogue. expand the alt/windows key behaviour tag and click 'super is mapped to the win keys'.

any luck with that?
