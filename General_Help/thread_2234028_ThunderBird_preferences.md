### Post by amanchesterman on 2014-07-12
1. I'm sorry I can't help you with this. I use the Reading Pane and I like the behaviour you describe (delete the email and it shows the next one) so I have never tried to set up TBird in the way you want it.
2. Do you have the Lightning calendar extension installed? If so, create a new event and on that dialogue window you will see an option to 'set reminder'. (It's just below the option to make the event repeat.) You can set a reminder for an existing event by opening the event for editing.
3. All your TBird data is stored in your TBird profile folder. There are step-by-step instructions on the internet to move it to a different location: just search online for 'move thunderbird profile'.
4. I assume you mean multiple signatures within a single email account? If so, there is an add-on for TBird called 'signature switch'. I've never tried it but it appears to do what you want.
