I do use Firestarter and ClamAV. Actually, Firestarter is just to configure the firewall. Ubuntu comes with one by default, iptables.
About the AV, it's not really needed if you're just using Ubuntu, but if you share files with a Windows computer or copy them to a Windows partition, you may want to check them before.

### Post by FuturePilot on 2007-11-09
Ubuntu comes with a built in firewall called iptables. The default settings are pretty secure, however you can tweak them with Firestarter which is in the repos.
Antivirus isn't really needed with Linux/Ubuntu.
You might want to read through this
