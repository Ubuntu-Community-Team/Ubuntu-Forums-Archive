### Post by Happy_Man on 2007-10-21
Well, if the OS says Gutsy, then it's Gutsy. If you upgraded a Feisty VM to Gutsy, it may just be that VMWare didn't update the name of the VM file automatically. If you can run all the Gutsy apps, and everything seems all right and stable, then don't worry.
