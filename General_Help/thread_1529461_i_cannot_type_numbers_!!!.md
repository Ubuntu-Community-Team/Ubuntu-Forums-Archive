I have a weird problem. I cannot type numbers using the numbers pane from the right side of the keyboard. Instead, the mouse pointer moves. For example, when i press the number 8 it moves upward.

### Post by audiomick on 2010-07-12
Hi.
At the top left of the number block there is a key that is probably labelled "num". Press that, it is the number lock key. Usually an indicator lights up somewhere on the keyboard when the num lock is activated.

---

### Post by nothingspecial on 2010-07-12
Go system > preferences > keyboard > mousekeys and check you don`t have keyboard moves the pointer (or whatever it`s called) ticked (checked)
