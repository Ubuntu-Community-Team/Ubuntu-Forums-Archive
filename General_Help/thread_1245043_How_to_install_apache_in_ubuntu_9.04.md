### Post by Copernicus1234 on 2009-08-20
Its not very difficult. You type this in a terminal (Applications Menu -> Accessories -> Terminal):

```
sudo apt-get install apache2
```

Done. It will be downloaded, installed and preconfigured with a default site you can access on localhost port 80. Web site root is /var/www. :)

You can also install using the package manager but this is easier, at least I think so.
