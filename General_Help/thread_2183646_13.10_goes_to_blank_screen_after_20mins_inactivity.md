### Post by stinkeye on 2013-10-25
I've been using this script I came across which toggles screen blanking on/off.
Used it previously in Raring but to use in Saucy I had to change a couple of dconf commands because
[LIST]
[*]"sleep-display-battery" became "sleep-inactive-battery-timeout"
[*] "sleep-display-ac" became "sleep-inactive-ac-timeout"
[/LIST]

See if it works for you.
_toggle_screen_blanking.sh_
```
#!/bin/bash

# This script automatically changes power settings, so that the screen will
# not turn off and the computer will not suspend e.g. when user wants to watch
# video.  


# Input argument: 'off' or 'on'.  If no input argument is specified, then
# will toggle power saving mode - the opposite of the current setting.

## In Saucy "sleep-display-battery" changed to "sleep-inactive-battery-timeout"
##       and "sleep-display-ac" changed to "sleep-inactive-ac-timeout"   
################################################################################
### Settings


# Idle time until turning off the screen (seconds)
# Only use one setting here because GUI version only has one
sleep_time=300


# Idle time until suspending (seconds) (0 means never)
suspend_bat=300
suspend_ac=0


################################################################################


# Path in gsettings
gsetpath=org.gnome.settings-daemon.plugins.power
gsetpath1=org.gnome.desktop.session


# Check value for screen turnoff
cur_sleep_bat=`gsettings get $gsetpath sleep-inactive-battery-timeout`
cur_sleep_ac=`gsettings get $gsetpath sleep-inactive-ac-timeout`


toggle=$1


# Determine whether to switch on or off
if [[ "$toggle" == '-off' || "$toggle" == "-OFF" || "$toggle" == "-Off" ]]; then
  switch='off'
elif [[ "$toggle" == '-on' || "$toggle" == "-ON" || "$toggle" == "-On" ]]; then
  switch='on'
else
  echo
  echo "-on or -off not specified: will toggle power savings mode."
  if [[ $cur_sleep_bat -gt 0 || $cur_sleep_ac -gt 0 ]]; then
    switch='off'
  else
    switch='on'
  fi
fi


# Toggle power saving on or off
echo 
if [ "$switch" == 'off' ]; then


  echo "Switching off power saving mode"
  notify-send -i video-display "Screensaver OFF"
  xset -dpms
  #echo "Screensaver OFF" > ~/.ss-status.txt


# Time until screen turns off (0 is never)
  gsettings set $gsetpath sleep-inactive-ac-timeout 0
  gsettings set $gsetpath sleep-inactive-battery-timeout 0 


# Time until suspend (0 is never)
  gsettings set $gsetpath sleep-inactive-ac-timeout 0
  gsettings set $gsetpath sleep-inactive-battery-timeout 0


# Turn off dimming of screen
  gsettings set $gsetpath1 idle-delay 0
  gsettings set $gsetpath idle-dim-ac "false"
  gsettings set $gsetpath idle-dim-battery "false"


else


  echo "Switching on power saving mode"
  notify-send -i video-display "Screensaver ON"
  xset +dpms
  #echo "Screensaver ON" > ~/.ss-status.txt


# Time until screen turns off (0 is never)
  gsettings set $gsetpath sleep-inactive-ac-timeout $sleep_time
  gsettings set $gsetpath sleep-inactive-battery-timeout $sleep_time 


# Time until suspend (0 is never)
  gsettings set $gsetpath sleep-inactive-ac-timeout $suspend_ac
  gsettings set $gsetpath sleep-inactive-battery-timeout $suspend_bat


# Turn on dimming of screen
  gsettings set $gsetpath1 idle-delay $sleep_time
  gsettings set $gsetpath idle-dim-ac "true"
  gsettings set $gsetpath idle-dim-battery "true"


fi
echo
```

### Post by stinkeye on 2013-10-26
In nautilus preferences they have changed the default action for executable text files to "view".
Change to "ask".

You can run the script as a toggle on/off by just clicking on it.

or
 run in terminal to turn power saving off with...
```
/path/to/script **-off**
```

and
 turn power saving on
```
/path/to/script **-on**
```

```
[COLOR="#008000"]glen@Raring:~$[/COLOR] **/home/glen/scripts/toggle_screen_blanking.sh**

-on or -off not specified: will toggle power savings mode.

Switching off power saving mode


[COLOR="#008000"]glen@Raring:~$[/COLOR] **/home/glen/scripts/toggle_screen_blanking.sh -on**

Switching on power saving mode


[COLOR="#008000"]glen@Raring:~$[/COLOR] **/home/glen/scripts/toggle_screen_blanking.sh -off**

Switching off power saving mode[COLOR="#008000"][/COLOR]
```

Cheers.
It may not.
All the dconf settings should remain but there is also an **xset -dpms** setting which will become enabled again after a logout.
It's usually this dpms setting that causes the monitor to blank when you disable everything else.

Also I was just checking the script and it seems some more dconf key names
 have changed so I would use this new script for **Saucy**.. ...
