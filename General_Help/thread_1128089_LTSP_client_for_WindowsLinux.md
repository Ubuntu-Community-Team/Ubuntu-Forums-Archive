### Post by s.fox on 2009-04-17
Hi and welcome to Ubuntu Forums,

You could try a program called ltsp-client.  

1) Navigate to:

**System->Administration-> Synaptic Package Manager.**

2) Search for ltsp.  You should see several ltsp programs listed,  including ltsp-client.

3) Select the program you want to install

4) Click Mark For Installation  (this will apear when you select the program)

5)  Click Apply (Green Tick)
