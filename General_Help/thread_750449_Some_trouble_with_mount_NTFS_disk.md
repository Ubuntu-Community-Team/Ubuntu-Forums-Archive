### Post by sisco311 on 2008-04-10
1.) create a mount point:
```
sudo mkdir /media/sda2
```2.) get the uuid of the partition:
```
blkid /dev/sda2
```3.) edit fstab:
```
gksu gedit /etc/fstab
```add this lines:

> 
#/dev/sda2
**UUID=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx** /media/sda2    ext3     defaults        0       2replace **xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx** with the uuid you get from 2.)

4.) mount the partition:
```
sudo mount -a
```5.) change the owner of the partition to get write permission:
```
sudo chown -R **your_username:your_username** /media/sda2
```

### Post by ryanhaigh on 2008-04-10
According to the output of fstab -l you do not have an ntfs partition! There are two physical hard disks in your system the first of which has only linux partitions and the second is  fat32 disk.

To mount the fat32 disk temporarily in /mnt do the following in the terminal:
```
sudo mount /dev/sdb1 /mnt -t vfat -o umask=0000
```
