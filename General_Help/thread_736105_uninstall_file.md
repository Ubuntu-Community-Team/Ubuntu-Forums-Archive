To uninstall Ubuntu just boot to a live cd and remove the Ubuntu partitions. The windows installer CD can't deal with Linux partitions, so by deleting them you turn the partition into unallocated space. you might find this useful: [http://ubuntujourney.wordpress.com/2008/03/25/removing-ubuntu-or-any-linux-from-a-dual-boot-xp-system/#more-4](http://ubuntujourney.wordpress.com/2008/03/25/removing-ubuntu-or-any-linux-from-a-dual-boot-xp-system/#more-4)
