### Post by tommygunner on 2009-01-07
I had issues with repetitive keystrokes not working on my keyboard with Ubuntu 8.10. When I would hold down the spacebar or any other key, it would not repeat.

I went to the Keyboard Preferences, turned stick keys on, then off and it worked. Under the General tab, repeat keys was already checked. So I basically did not do anything and fixed it. 

I'm wondering if anyone else had issues. I'm using a simple standard ps2 style keyboard.
