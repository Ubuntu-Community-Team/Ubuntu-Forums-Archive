### Post by Forlong on 2009-01-04
To use an Emerald theme you obviously need Emerald installed.
If you already have, go to *System &#8594; Preferences &#8594; Emerald Theme Manager* and add it there.

Any other themes can be installed in *System &#8594; Preferences &#8594; Appearance*

### Post by Forlong on 2009-01-04
> **Secret Neo said:**
> I got emerald now, and i imported the theme now what?
Try hitting [Alt]+[F2] and run
```
emerald --replace
```
> **Secret Neo said:**
> 
and with other themes, what do the extensions have to be to make them work?
Depends, .tar.gz will work if it's properly packaged.
Otherwise, just drag and drop the extracted folder with the theme in the Appearance application.
Just make sure it's the folder with the theme which might be a subfolder of the archive you downloaded.
