### Post by toddr on 2006-11-08
You simply have to type dgen in the terminal. You also have to have games to play on it. Dgen is only the program to run the games on. You have to have the actual game .bin files. Once you have the games, if you right click on a game, and select "open with other application" then at the bottom type in dgen.
