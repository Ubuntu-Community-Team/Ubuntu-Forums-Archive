### Post by begtognen on 2010-10-05
I have dual monitors. I'm running Gnome and Ubuntu 10.04.

I read that using the Compiz "put to next output" keyboard command allows you to assign a keyboard shortcut to moving a window from one monitor to the next monitor.

Although the keyboard shortcuts work for moving the window around within ONE monitor, they don't do anything when it comes to moving the window from one monitor to the next, using the "put to next output" command under Compiz : Window Management : Put.

I'm happy to try a different solution, if the Compiz keyboard shortcut just isn't in the cards.

Happy to answer any questions you might have (just a heads up, I'm quite new to Ubuntu) and thanks for your help.

### Post by begtognen on 2010-10-05
AHA. Okay - Compiz treats my dual monitors as *one* monitor.

Which means that what works beautifully is going into:

Compiz | Window Management | Put Within Viewport | Put Left (to put the window on the lefthand monitor)

and vice-versa for "Put Right".
