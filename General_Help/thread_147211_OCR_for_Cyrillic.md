### Post by Dragineez on 2006-03-19
I'm not aware of one under Linux if gocr doesn't do it. But gocr really isn't all that good anyway. Most major OCR packages under Windows are internationally aware - but the software ain't cheap and the language packs cost extra. The best I've ever seen at non-latin characters is ABBYY's FineReader - most definately in the non-cheap category. Good results, excellent language support, it's from a former eastern block country so I'm sure it would do Cryillic extremely well, and its the closest to an uncrashable OCR engine I've ever seen (and I've been using OCR in a professional capacity since '91). Won't win any speed awards, but it will do the job. 

Now, how do you justify the cost of the software? You should probably try to find someone that has it or see if you can get an eval version if this is just a one shot deal.
