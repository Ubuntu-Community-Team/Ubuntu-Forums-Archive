### Post by forrestcupp on 2008-04-13
If you're using ATI drivers, your best bet is going to be to turn Compiz off and make sure VLC isn't set to use the X11 video output module.  But if you want to use Compiz *and* run videos fullscreen, you'll have to use the X11 output so you won't get flickering.  So your best bet then would be to not make it fullscreen, but to use Compiz's zooming feature to zoom in on the video.  This will smooth it out for you, too.

That's just the price you have to pay when you use ATI.
