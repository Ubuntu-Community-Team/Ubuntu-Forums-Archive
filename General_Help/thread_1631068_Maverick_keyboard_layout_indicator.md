### Post by ShalomG on 2010-11-26
Prior to Maverick, the keyboard indicator in the notification area showed which layout was currently selected ("US", "RU", etc.). After upgrading to Maverick, the keyboard indicator always shows a keyboard icon, without any way of instantly telling the current layout (other than clicking the icon, showing the layout switcher). Is there a way to fix this?

### Post by groonrix on 2011-03-05
Hebrew layout problems, eh? :)
 
Anyway- There is a way to identify your current layout-
You can set up (at System > Administration > Keyboard Shortcuts) to use the Scroll Lock LED to identify your keyboard layout.

### Post by guyguyguy on 2011-04-11
I found a simpler workaround here (comment #23)
[https://bugs.launchpad.net/gnome-settings-daemon/+bug/531173](https://bugs.launchpad.net/gnome-settings-daemon/+bug/531173)

1. Right click on the panel and choose "Add to Panel..."
2. Select Indicator Applet
3. Done.

Worked for me
