### Post by Lars Noodén on 2011-12-11
If you are running Unity then you can change the settings this way:
[indent]
System Settings->Software Sources->Ubuntu Software->Download from
[/indent]
If you are using a different Desktop Environment, say which one.

Or if you want to change the settings manually, you'll find them in [font=Courier New]/etc/apt/sources.list[/font]

### Post by westie457 on 2011-12-11
Hello, one way is to open Software Center go to Edit and select Software Sources. In the pop up window in the Ubuntu Software tab look for Download From. In the box choose which server and or country you wish to use.
