### Post by werewulf75 on 2024-08-30
> **deadflowr said:**
> [I]Thread moved to **General Help**
[/I]
Will those updates ever be available as non-pro updates?
No.
However, you can get those updates if you subscribe to Ubuntu Pro which is free for personal use and up to 5 machines. Once you've subscribed you'll be issued a token. You attach your system like this IIRC. in Terminal :

```
sudo pro attach token
```

where you replace the word "token" with the actual token, and Bob's your whatsit. ;)
