### Post by Forlong on 2008-06-24
> **geezerone said:**
> What is "compiz-real" which I noticed running in background after recent compiz updates
compiz.real is the actual binary -- /usr/bin/compiz is a script to start Compiz properly on Ubuntu. 
> **geezerone said:**
> and is it needed if compiz won't run on my old gfx card?
No, it's probably a zombie. You can stop it like this:
```
kill -9 $(ps -o pid= -C compiz.real)
```
Next time you reboot, it shouldn't come up again.
