### Post by jmorsman on 2009-02-19
Hi,

 When you start up your VB, under the devices menu at the bottom, there is a line that says install guest additions. This will install the needed features so that you will be able to resize your screen. I had to do this when I installed xp as a vm.

### Post by howefield on 2009-02-19
With your guest system running, go to the devices menu on the window that your guest is running on, and select "install guest additions"
