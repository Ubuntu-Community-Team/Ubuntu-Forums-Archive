### Post by aysiu on 2008-01-01
The best way to tell if the plugins are slowing it down is to try Firefox without the plugins.

Close Firefox completely.

Then press Alt-F2 and paste in ```
firefox --profilemanager
``` Create a new profile called *test* and launch that. The new profile won't have any extensions. If it's faster, then clearly the plugins are slowing your Firefox down.

To get back to your old profile, close Firefox again, press Alt-F2 and run the same command and select *Default* instead of *test*
