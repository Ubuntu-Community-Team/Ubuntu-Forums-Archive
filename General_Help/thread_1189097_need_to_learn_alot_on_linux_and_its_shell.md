Alternative shells for UNIX/Linux...

C Shell
**sudo apt-get install csh**

T Shell
**sudo apt-get install tcsh**

K Shell
**sudo apt-get install ksh**

Z Shell
**sudo apt-get install zsh**

Bash Reference Manual **[http://www.gnu.org/software/bash/manual/bashref.html](http://www.gnu.org/software/bash/manual/bashref.html)**

### Post by trwww on 2009-06-17
Oreilly Publishing releases some of their out of print books online:

[http://oreilly.com/openbook/](http://oreilly.com/openbook/)

For example, say you know a bit about computers (finding and editing files/uploading and downloading) and you want to learn about web programming starting at square one.

Their book _CGI Programming on the World Wide Web_ is available for free at [http://oreilly.com/openbook/cgi/](http://oreilly.com/openbook/cgi/)

There have been a lot of new technologies come out since this book has come out, but all of them are based on the topics in this book.
