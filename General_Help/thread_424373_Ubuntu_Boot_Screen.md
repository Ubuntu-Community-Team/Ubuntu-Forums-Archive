### Post by panzer77 on 2007-04-26
For the boot screen go to Administration>Login Window and click on the local tab.  This is where you can change the Boot screen.  FYI go to [www.gnomelook.org](www.gnomelook.org), they have lots of great ones there.  Look at the section called GDM Themes.  Just download what you like and drag it into the local tab.  

Splash screen you will need to install gnome-splashscreen-manager.  You can install this with synaptic.  Once installed you will find it under System>Preferences>Splash Screen

Enjoy
