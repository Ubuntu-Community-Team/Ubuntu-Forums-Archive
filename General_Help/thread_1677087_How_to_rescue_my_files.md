### Post by cat0752 on 2011-01-28
You can make an ubuntu install CD or USB and boot from that.  From there you have the option of testing ubuntu before you install it. -- this runs ubuntu from your cd or usb.  From there you should be able to backup everything i think.  It should automatically see your ubuntu partition(s) as well as your windows partition.  So you should be able to copy everything from ubuntu to your windows partition no problem.

Also, did you use a Wubi installer when you installed ubuntu?  Seems like there are other threads of this same problem happening to others who used a Wubi installer.  Maybe this thread will be of some help:
[http://ubuntuforums.org/showthread.php?t=1676577](http://ubuntuforums.org/showthread.php?t=1676577)
