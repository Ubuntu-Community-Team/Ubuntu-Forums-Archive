### Post by happyhamster on 2010-06-23
Go here:
[http://gnome-look.org/index.php?xcontentmode=36](http://gnome-look.org/index.php?xcontentmode=36)

Search for, and download a theme with the pointer you want. Don't install the theme, just extract that file in the hidden .icons folder in your home folder. (To be able to see hidden files and folders in your file browser, press ctrl-h. To extract, right-click the file and choose "Extract Here")

Now, go to System-->Preferences-->Appearance-->Theme-->Customize-->Pointer, and the new pointer should be selectable. Good luck.
