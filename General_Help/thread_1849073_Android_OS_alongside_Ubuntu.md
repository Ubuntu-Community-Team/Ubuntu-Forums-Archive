### Post by LinuxFan999 on 2011-09-23
You could use a Live-CD like GParted to shrink your existing partitions to make room for Android. The boot loader which comes with Android should automatically add Ubuntu to the boot menu. If you don't want to mess around with the system partitions, you could install android on a USB stick and insert it into an unused USB port on your computer every time you want to boot Android. You may need to enter the BIOS to change the boot order if you want to boot from a USB stick.
