### Post by huygens on 2010-03-21
Gedit is a text editor, much like notepad on Windows, but with much more features such as syntax highlighting.
However, it is just a text editor and is not really meant for developing application.

I would recommend you other "user-friendly" applications to develop in C++.

First of all there is Geany ([apt:geany]("apt:geany")). It's a text editor oriented for programming. It will provide you (but it is limited) with some helps while writing your code.
But you could use the Software Centre (check under the top-left Application menu on Ubuntu). There is a dedicated "Programming" section where you can find more evolved software for programming in C++. You should notice that their complexity could increase with their features...
Check for the IDE keyword. It means Integrated Development Environment (such as Visual Studio, Eclipse, NetBeans or TurboC)

Here is a list of possible choices (more to see in the Software Centre)
[LIST]
[*]Geany ([apt:geany]("apt:geany"))
[*]Ajunta ([apt:ajunta]("apt:ajunta"))
[*]Codelite ([apt:codelite]("apt:codelite"))
[/LIST]

I did not add here Eclipse (+ the CDT plug-in), NetBeans (+ the C++ plug-in), KDevelop or Qt Creator because they are not primarily dedicated for C++ or they are better integrated in the KDE environment (Kubuntu). But they are good all the same.

Speaking of myself, I use at work Eclipse CDT, but it is quite bulky and heavy for C++. I like Geany when I just have to quickly fix a file.
