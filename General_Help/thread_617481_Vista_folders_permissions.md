### Post by Milk &amp; Toast &amp; Honey on 2007-11-19
Hi, you must install ntfs-3g to enable write support in ntfs partition.

```

sudo apt-get install ntfs-3g ntfs-config

```

Then navigate to Applications -> System Tools -> NTFS Configuration Tools, check (enable) the write support.

I don't have Vista, but I think Vista's NTFS is different from the XP's, so please check [this thread]("http://ubuntuforums.org/showthread.php?t=217009") or google for "ntfs-3g vista" to see if there's any issue, to prevent loosing your data.

### Post by Milk &amp; Toast &amp; Honey on 2007-11-21
0micron, try change the /etc/fstab in this line:
```

# Entry for /dev/sda5 :
UUID=8EF1-1027 /media/sda5 vfat defaults,utf8,umask=007,gid=46 0 1

```
to this one:
```

# Entry for /dev/sda5 :
UUID=8EF1-1027 /media/sda5 vfat defaults,**rw**,utf8,umask=007,gid=46 0 1

```

and then, remount the partition by typing this on terminal:
```

mount -o remount /media/sda5

```
