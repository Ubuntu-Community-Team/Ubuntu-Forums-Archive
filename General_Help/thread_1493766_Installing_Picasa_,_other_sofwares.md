### Post by meho_r on 2010-05-26
Ubuntu Software Center is an easy and secure way to install software onto your computer. You should use it whenever possible. In case that the application you need isn't present in repositories, you may check out if there is an official repository for it provided by developer (check app's official webpage for infos) or maybe there's a [PPA](https://launchpad.net/ubuntu/+ppas) (Personal Package Archive). Installing from a repository is always better idea than installing from a package, because you'll be notified in case of any updates or security and other patches, and security risk in general is much lower.

As for Picasa, try following [these](http://www.google.com/linuxrepositories/ubuntu704.html) instructions to add Google's repo, and then simply search for "picasa" in Ubuntu Software Center.
