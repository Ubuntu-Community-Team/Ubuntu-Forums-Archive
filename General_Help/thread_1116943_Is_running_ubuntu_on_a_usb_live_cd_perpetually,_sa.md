### Post by CatKiller on 2009-04-06
> **ubuntero9999 said:**
> what are the limitations?

will it have the same security as a regular installation?

The main limitation I can think of is that flash memory doesn't have an indefinite lifespan. There are a limited number of writes before each sector cannot be used any more.

It will have the same security as a regular installation in that someone could put it in another computer and read all the files on it, the same as if it was on a regular hard drive. It's just that it's easier to get hold of if it isn't screwed inside your computer case.
