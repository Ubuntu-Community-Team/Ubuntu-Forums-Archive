### Post by Vigh on 2010-10-31
You got graphics card drivers installed? Deleting folders is generally not a good idea if your doing everything through wine. Try updating/upgrading to latest packages/patches for ubuntu, install the recommended graphics card driver. Update to latest stable version of wine. Failing that, could be a broken package with the GUI launcher programming. Have had some problems with running old versions of python/GTK on new releases which seemed to break some of my application launchers. You could also purchase cedega or crossover, think its about $40, crossover supports the wine community and allows you to point click install applications particularly WoW, when you want to update, download the update manually and install to the WoW bottle. Good-luck.

If your running Karmic- try doing a back-up of critical data, then do a fresh install of 10.10. It is a very stable release with great new features.
