### Post by Westie1 on 2009-06-11
I want to do the unthinkable in the U.S. and open up a tiny consulting business where only Linux is available on the desktop for the employees (6-8). I have been running Ubuntu 9.04 and see no serious problems. However, the problems I do forsee fall into two areas 
(1) trying to get the following to run under Wine:
*  Quickbooks Pro (only kept on managers Linux machine)
*  Dymo label printer (400 Turbo)
*  multifunction scanner/fax/printer (Canon imageCLASS D480)
The Windows software for latter two devices probably would be set up on an old bare bones XP machine if Wine fails. Others have succeeded with Quickbooks so I am pretty hopeful.

(2) security
The security problem is that we all will be using our own proprietary data and software and cannot protect ourselves with non-compete agreements (not possible in our state). Hence,  we need a keylogger or equivalent that monitors employee use of our computers for email, web, printer, and USB flash drives. Yes, one could let an employee walk with the secrets and then try to prove in court that it was stolen, but good luck on that. I've read the posts about people (hackers, siblings, significant others) that want such software, but they are not protecting the survival of a small business. The current keylogging programs are not well supported and problematic.
The log files would only be examined when an employee leaves. With log data a court case against a former employee or a competing business is far easier to prove. 
Since all previous posts cry hacker, spy, or stupidity at the poster, I don't have high expectations but really don't want be pushed down the road to MS anymore if these above issues can be solved. We'd be happy to buy a commercial product like the ones for Windows if one existed. Thanks.
