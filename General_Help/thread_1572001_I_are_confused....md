### Post by M4rotku on 2010-09-10
Gnome is the Desktop Environment.  If you want to switch to Gnome, then this is the only element of the 3 you listed which you need.  Gnome Shell seems to be an advanced feature of Gnome which they plan to release with Gnome 3 - don't worry about it right now unless you want to look into it out of interest, it's not necessary.  Gnome Do is a quick application launcher which also works well as a dock.  It is not an integral part of the Gnome system, so you don't have to install it unless you want to check it out on it's own.

I hope this helps,
M4rotku
