### Post by Dennis N on 2020-12-31
As to the OPs question:
> So could anyone tell me where i might download a "free" image, to use as my browser's (Firefox) start page please. It could be scenic, sci-fi/fantasy as long as it's decent i don't mind. Thanks.
Use Google Image search. Search term "computer wallpaper". You can specify scenic, abstract, patterns, dinosaurs, kittens, etc. along with that if you like. There will be hundreds to choose from. Download the file to your home folder and in Firefox > Preferences > Home > custom URL, set this to your downloaded image. Or you can open the image in firefox (Ctrl + O), then select "use current pages" in preferences.  

Then, you would use bookmarks to get to internet sites. My suggestion was to consider using a custom HTML start page on your computer with an image (or several if you want) plus links to internet locations. If the OP doesn't know any HTML, or just isn't interested in exploring further, then bookmarks it is. As for me, HTML was part of a high school computer class.
