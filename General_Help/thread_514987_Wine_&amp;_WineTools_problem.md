### Post by bodhi.zazen on 2007-08-01

Winetools is no longer maintained in favor of Wine Doors, which is in pre-alpha, go figure ...

[http://www.linux.com/articles/54920](http://www.linux.com/articles/54920)

You will need to downgrade you version of wine to use that version or come up with a work around ...
