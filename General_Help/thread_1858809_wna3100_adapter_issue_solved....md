### Post by justoneguy on 2011-10-13
I just wanted to say that i have ubuntu 11.04 and I needed to connect wirelessly and I had a wna3100 adapter lying around so i slapped into my box and i am new to ubuntu so i did not have the foggiest on how to get it to run...so i was off to fix the problem...well step one is to download the drivers for the adapter for linux...the name of the gz file is Broadcom_bcm43xx_usb_32_64bit_v2.tar.gz    see attached file .... now in this zip there are 4 files, (2) .inf files and (2) sys files...unzip these files into a directory like downloads or something easy ... now you have to go to the software center in ubuntu and download "windows wireless drivers" software and install it..you then run it from your system/administration/windows wireless drivers   its the one on the bottom    then you point it to the .inf 32bit file that you unwrapped already from the gz file.  you go up to edit connection under networking and create a new wireless connection with your ssid and passkey or pass phrase (depending on method of encryption used) and bam your on friends...


Hope this helps
