### Post by Sunforge on 2007-07-08
You've probably already checked this but if you're using a CRT monitor ensure that the refresh rate is set to above 75 Hz.
 
There's a pretty good general trouble shooting guide for monitor resolutions/refresh etc to be found here:

[https://help.ubuntu.com/community/FixVideoResolutionHowto](https://help.ubuntu.com/community/FixVideoResolutionHowto)

No, it's exactly the same for me. My eyes get really tired very quickly and then I start to get really bad headaches. On Windows however, I can sit in front of the PC all day and not be affected at all.

I mucked around with various fonts and font display options in Ubuntu over a period of several days until I finally found some combinations that looked decent enough for me to be able to use Ubuntu for more than an hour or two at a time. Previous to me doing that, reading any text on my Ubuntu 7.04 install was like trying to watch a 3D movie without the special red and green glasses! My fonts were almost multi-coloured.

The combination of settings that I use now have really improved the look of the fonts (although I still wouldn't go so far as to say that it's 'good looking') but at least it's usable over long periods of time now.

The one area that I have **not **managed to solve (no matter how much fiddling I do) is the crappy looking fonts in Firefox. All fonts in Firefox look absolutely revolting to me (I'm talking about the content window, not the menus or title bar). I can barely read any web page for more about 30 minutes before having to close down the browser and go back to the Ubuntu desktop to rest my eyes.

I'm using a top of the range Eizo 24" Widescreen LCD monitor (1920 x 1200) so it's not like the issue is down to CRT convergence problems or whatever.

I've done several searches on the Mozilla forums regarding the quality (or lack thereof) of font rendering in Firefox, but I didn't find any relevant information. If I have to do any web browsing for more than an hour, then I have no choice but to switch back to  Firefox on Windows.

### Post by jespdj on 2007-07-08
> **konungursvia said:**

Try this if you are using an LCD screen: [ How to improve sub-pixel font rendering for Feisty](http://ubuntuguide.org/wiki/Ubuntu:Feisty#How_to_improve_sub-pixel_font_rendering_for_Feisty)
It made the fonts much nicer on my screen (24-inch Dell, 1920 x 1200).

There are also ways to make the fonts on Linux look exactly the same as on Windows (well, without ClearType ofcourse): [Display Microsoft fonts like on Windows](http://ubuntuforums.org/showthread.php?t=208396)
