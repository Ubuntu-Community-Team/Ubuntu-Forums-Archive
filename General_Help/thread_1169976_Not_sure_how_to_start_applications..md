### Post by drs305 on 2009-05-25
> The syntax for starting tintin is: ./tt++ [command file]

You can view the documentation by extracting the file:
```

/usr/share/doc/tintin++/tintin19.txt.gz

```

You can find out what files are associated with the package by right clicking the name in synaptic, selecting Properties, and then looking at the "Installed Files" tab.
