### Post by green_earth on 2007-06-29
Is there an easy way to check/change the default application for various file types, web, and so forth?

---

### Post by tyl on 2007-06-29
Right click the desired file in Nautilus (file manager).  Select open with other application under the drop down menu, and choose a program to open it with (you can choose to default it).  

If by web you mean default web browser, you can choose the default browser under System ->Preferences->Preferred Applications.
