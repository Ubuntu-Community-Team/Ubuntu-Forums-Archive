### Post by fragos on 2009-01-03
There's no need to remove Tracker and if you do other problems will occur. OPen the Sessions Preferences and on the startup programs tab uncheck the boxes for "Tracker" and "Tracker Applet." Next reboot and Tracker will no longer run and you can install the packages you want.
