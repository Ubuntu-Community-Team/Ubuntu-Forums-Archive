### Post by coffeecat on 2009-11-07
I agree with you about the default top left nonsense - it irritates me unreasonably.

Anyway, I know nothing about Devil's Pie, but can you run compiz on your system? If so, install Compiz Config settings manager from Synaptic. After installation you'll find it in System > Preferences. Then from CCSM, Window Management > Place Windows > Placement Mode - choose 'Centered'.

Bingo!
