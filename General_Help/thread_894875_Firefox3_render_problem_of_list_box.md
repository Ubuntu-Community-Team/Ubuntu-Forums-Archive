### Post by gjoellee on 2008-08-19
this may just be an option in Firefox that have been set on of some kinda reason.

In firefox go to Edit->Preferences and the content tab. Now click on "colors" and make sure the pages are allowed to use the color they are set fore. Also if "Use Systemcolors" is checked you should uncheck that option.
