### Post by 73ckn797 on 2012-01-20
If you install 11.10 you can then install gnome-session-fallback which will allow you to have a Gnome DE in the Gnome Shell or the more traditional Gnome Classic look and feel. Though, the Gnome 3 classic is much different than Gnome 2 but also quite similar.
