### Post by talcite on 2007-04-21
The restricted drivers manager just lets you know that you're using drivers which are not open source. 

I'm assuming those drivers are the Nvidia or ATI drivers you're using. They work but they're not officially supported since the source code is not openly available. 

Beryl is still in alpha testing. You should really avoid it if you aren't adventurous or if you need your OS for critical work. There are things in Beryl that can break your Xserver, meaning you'll be stuck with only command line. Normally this is ok if you're familiar with it, but if you're not, it can be a pain to troubleshoot the system.
