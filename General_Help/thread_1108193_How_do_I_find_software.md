### Post by IanVonSpeedfreak on 2009-03-27
Can you be more specific?  In general, I'd recommend using Synaptic Package Manager, if you have it installed.  Not only can you perform a search, but you can also browse software based on category.  Searching packages.ubuntu.com does pretty much the same thing, but you can use Synaptic to install the software, too.  If you prefer using the command line, typing *apt-cache search x* (x being whatever term you use to search) will search the repositories (packages.ubuntu.com) using apt.

Still I'm not quite sure what you're getting at, but feel free to rephrase if you're not being understood.
