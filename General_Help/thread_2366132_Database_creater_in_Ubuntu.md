### Post by monkeybrain20122 on 2017-07-14
There is [kexi]("https://www.calligra.org/kexi/") if you don't mind installing a lot of kde dependencies. It is in the repository.


Edited: instead of installing from the repo, you can get an appimage from [here]("https://s3-eu-central-1.amazonaws.com/ds9-apps/index.html#!/kexi-stable-master-appimage%2F"), in this way you don't need to install anything , make the appimage executable, then just click it. But the catch is that the appimages are only betas or alphas.
