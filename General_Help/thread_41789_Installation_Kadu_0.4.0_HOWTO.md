A lot of people were asking me about the solution for dependency problem during Kadu instalation.

All I did to install Kadu was just adding these lines to /etc/apt/source.list


[B]## Kadu
deb [http://kadu.net/download/binary/debian/sid](http://kadu.net/download/binary/debian/sid) ./

## Debian
deb [http://http.us.debian.org/debian](http://http.us.debian.org/debian) sid main contrib non-free[/B]
