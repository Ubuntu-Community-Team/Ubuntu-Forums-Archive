### Post by andymushu on 2008-08-02
I've written a custom wallpaper switcher for ubuntu that uses the image on the front page of [http://wvs.topleftpixel.com/](http://wvs.topleftpixel.com/), which changes daily. I wrote it as a bash script, then expanded it to include an install and an uninstall script. I used zenity heavily in these scripts, in order to provide a more user friendly gui for the user. Included in the attached zip archive are the install and uninstall scripts, as well as a readme, the actual program, and a cron job to make sure the wallpaper is switched. Everything should be installed correctly once the install script is run. If you wish, please download it and test it out. Any feedback is welcome, but this is my first shell script, so please don't be too harsh.
