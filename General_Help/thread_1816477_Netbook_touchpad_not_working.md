### Post by 3rdalbum on 2011-08-02
If the main use will be typing and browsing the net, do you really need a dual-core netbook? A dual-core will be able to run four threads simultaneously, a single-core will run two threads simultaneously (assuming you're talking about the Intel Atom CPU).

Pretty much any Atom-based netbook will be fine, just make sure you get one with either a hard disk or a GOOD ssd.
