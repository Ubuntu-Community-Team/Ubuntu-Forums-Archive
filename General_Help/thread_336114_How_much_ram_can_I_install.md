### Post by tribaal on 2007-01-11
Well the software limit is around 4 Gigs if I remember correctly (for 32bits architectures, more for 64 bits).

You will want to know your motherboard's limitations, though. Look into your motherboards manual if you have it, or pop open your case, and note down your motherboard's model number. With this and google, you should find the info pretty easily. :)

Usually modern motherboards support at least 4gigs of RAM, so if you plan to install only a gig or so, I guess you can just install the chips "eyes closed".

Don't forget to ground yourself while manipulating RAM chips ;)

### Post by cmost on 2007-01-11
You don't even need to pop open your case.  If the machine is still running some version of Windows, simply download AIDA32 (it's available on majorgeeks.com - do a search.)  Unzip the files and run the aida32.exe.  This program will tell you absolutely everything you need to know about your computer, its operating system and all of its hardware.  Practically every detail is conveniently reported.  Further, it will tell you every detail about your ram chips, right down to the manufacturer and serial numbers.  From this (coupled with a little Googling) you should be able to determine your motherboard's maximum RAM and which chips to buy.  If you've already installed Linux on this machine, then use TINA32 instead.  It's a KDE app that is essentially a clone of AIDA32.  It will also tell you much of the same useful information about your hardware.  Good luck!
