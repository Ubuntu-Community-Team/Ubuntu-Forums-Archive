### Post by dragonfly41 on 2011-05-16
This is one way which works for me (there might be a simpler way though)

System > Administration > Synaptic Package Manager

install nautilus-scripts-manager

then right click on desktop > menu > Scripts

open_terminal_here

and any other Nautilus scripts you care to add
