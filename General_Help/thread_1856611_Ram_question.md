### Post by quadproc on 2011-10-08
If you want to run more than 4 GB of RAM then you need a 64 bit system.  Since you (presently) have only 2 GB you are not facing that restriction; a 64 bit system will run OK with 2 GB of RAM in it.

When you set up your swap space, I would suggest making the size of swap be slightly larger than the largest RAM that you will ever have.  That way, you won't have to change the swap allocation in the future.

quadproc
