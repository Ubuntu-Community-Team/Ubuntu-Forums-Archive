### Post by grahammechanical on 2014-09-09
I do not think that there is such a thing as a Ubuntu Cloud.  We can put Ubuntu server images on other people's clouds.

[http://www.ubuntu.com/cloud/public-cloud](http://www.ubuntu.com/cloud/public-cloud)

We can use Ubuntu server and OpenStack on our own Cloud servers.

[http://www.ubuntu.com/cloud/openstack](http://www.ubuntu.com/cloud/openstack)

We can pay Canonical to manage our own cloud

[http://www.ubuntu.com/cloud/bootstack](http://www.ubuntu.com/cloud/bootstack)

But I have yet to see anything close to Canonical offering to sell us server space on their servers. Even the now discontinued Ubuntu One cloud storage service was not a true Cloud.
