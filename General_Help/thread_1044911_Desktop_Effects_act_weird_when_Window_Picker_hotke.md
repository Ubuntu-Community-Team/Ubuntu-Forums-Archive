### Post by diablo75 on 2009-01-20
Not sure what the cause of that problem is, but I prefer to setup that key binding within Compiz Config Settings Manager.  You might not have it installed.

If not, click Applications>Add/Remove...

Search for Compiz.

Select Compiz Config Settings Manager (check it off) and click Apply.  You'll then find the CCSM in System>Preferences>CompizConfig Settings Manager.

Look for the "Scale" plugin within the Windows Management category near the bottom.  Click on it to open it up.  you can set your own binding up in there.  Personally I prefer to set it up so I initiate the window picker for all windows simply by throwing the mouse into the upper right hand corner of the screen (because it's the least frequent place my mouse travels, even by accident).
