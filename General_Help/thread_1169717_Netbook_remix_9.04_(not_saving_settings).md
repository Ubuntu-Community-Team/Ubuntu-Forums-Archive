### Post by Brandon Williams on 2009-05-25
The UNR image is not intended to be persistent. If you want a persistent UNR install on a USB, you've got three options that I can think of: 1. Follow the instructions that I posted [here](http://ubuntuforums.org/showpost.php?p=7220253&postcount=5) (complicated); 2. If you have two USB sticks and one is high-quality (very good read/write performance), put the UNR installer on one stick and install to the high-quality stick; 3. Create a persistent intall of the standard i386.iso image using usb-creator and then install ubuntu-netbook-remix to it.
