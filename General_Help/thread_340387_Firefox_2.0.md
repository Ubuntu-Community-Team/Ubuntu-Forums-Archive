### Post by durant1958 on 2007-01-17
The problem lies with the Developers.  

They made some choices that resulted in FireFox being used by a number of other packages for HTML rendering.  They claim that it is not practical ( Sorry, but I read this as "too much work". ) to upgrade dapper from 1.5 to 2.0 via the repos.

There is a script available to do a side-by-side installation.  I don't think it's the correct way of dealing with this issue in light of the fact that dapper was supposed to be the LTS Ubuntu / Kubuntu.

I dropped 6.06 on my production system in favor of openSUSE.
