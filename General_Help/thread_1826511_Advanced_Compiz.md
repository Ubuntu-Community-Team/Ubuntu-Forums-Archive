### Post by lakerssuperman on 2011-08-16
Make sure you have all the Compiz plugin packs installed.  Then go into the Compiz settings manager and you should see a check box called 3D windows.  That should give you the effect you are looking for.

---

### Post by hhh on 2011-08-16
```
sudo apt-get install compiz-fusion-plugins-extra
```
After that it will appear in CCSM under "Effects".
