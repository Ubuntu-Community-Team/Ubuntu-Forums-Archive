### Post by erginemr on 2007-11-27
Hello,

I am not using a psp, but if its file transfer system is like every other usb flash drive out there, then the file transfer in Linux is done somewhat asynchronously and you need to first unmount the flash drive by right-clicking on its icon from under Nautilus / Computer and then selecting the appropriate option. 

This way, you can make sure that all files transferred have been actually written to the removable device.
