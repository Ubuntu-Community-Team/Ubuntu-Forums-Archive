### Post by poodoopealeoaph on 2010-09-23
Well, the issue with running fl studio in wine is that it will not allow use of ASIO in almost every case. So, you won't be able to have any input of sound to record. So you would have to use audacity to track and import it into fruity loops. Which i have to say is completely stupid. Also, unless you want to screw with the idea of getting JACK to work for a minimum of a week then you really have no free software in ubuntu that you can track with other than audacity. So, really, you won't be able to get it all to work unless you are a programming genius.
