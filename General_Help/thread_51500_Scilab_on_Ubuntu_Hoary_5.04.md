Hi
It's now been 3 days long that I've been trying to install scilab 3.1.1 on my Hoary distro... And I finally did it.
The only things I needed to install was **pvm**, **pvm-dev** and **libpvm** (with synaptic, of course).
Then I downloaded the scilab binary .tar.gz from the scilab site, decompressed it and ran **make install**.
This worked for me. (run **scilab** to test)
Maybe this will help you.
