### Post by CatKiller on 2009-09-25
It's called Shade. How you enable it depends on which window decorator you're using. If you're using Emerald, you can fiddle with the Emerald Theme Manager to show the function. Your Emerald theme may or may not have an image to use for the shade button. If you're using Metacity, you can fiddle with the /apps/metacity/general/button_layout key in gconf-editor.
