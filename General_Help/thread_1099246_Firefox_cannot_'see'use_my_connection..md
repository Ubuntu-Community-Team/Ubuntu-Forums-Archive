File -> Work Offline
There is most likely a checkmark there, and clicking it will remove that check and allow you to browse.

### Post by Sepero on 2010-06-16
Type about:config in the address bar, then set
toolkit.networkmanager.disable = true

Also for Intel 536ep dialup modem see
[http://groups.google.com/group/ubuntu-modems](http://groups.google.com/group/ubuntu-modems)
