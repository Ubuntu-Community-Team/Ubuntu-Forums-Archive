### Post by QIII on 2014-04-13
Hello!

AMD/ATI has discontinued Linux support for HD 2000 - HD 4000 GPUs.

You'll have to continue to use the default open source driver that was installed with Ubuntu.  It should be sufficient for normal use, but may not be all that great for gaming.

Best wishes.
