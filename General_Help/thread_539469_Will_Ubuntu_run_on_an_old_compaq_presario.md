### Post by firstcupoffreshpressed on 2007-08-31
I'm running debian sarge, full bloated gnome on a presario 1245.  It has 64 mb of ram right now and 4 striped swap drives.  [experiment]  The theory is that multiple small swap drives set in fstab to have all have the same priority will work like a raid array.  data exchange should be faster than with regular swap.  I've been reading a bit and am trying this.  Some people have pooh poohed the idea.

You are going to have to add ram that you can get for 20 or so dollars to do much with this.  Even Puppy is not going to be very happy [my experience] unless you have more rrrrram.

Debian sarge is a bit slow but very stable and I can open 7 or 8 applications all at once and no sign of a freeze or crash.  

I ordered a 128 MB ram chip for mine last night.  Presario 1245 max ram is 160MB [32 soldered]:guitar:

I may be able to use xubuntu.  So far I have tried xubuntu base system installs but the debian Sarge seems easier to me probably because of my past experience with it.

I'm also going to try putting the swap at the beginning of the hard drive.  Some folks say this helps access swap.
:guitar:
Theres lots of info if you just google and read selectively.

Don't expect greased lightening.  You can probably turn it in to a stable light system.  Add a pc card for wireless or dsl. The win modem in the 1245 for dialup is reportedly a painfor some linux users but I haven't tried it.
