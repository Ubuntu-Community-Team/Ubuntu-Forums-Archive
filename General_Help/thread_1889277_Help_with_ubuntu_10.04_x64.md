### Post by 3rdalbum on 2011-12-01
Open the Additional Drivers program from System > Administration.

It should list the driver there. Click on it, click on Install and away you go.

By the way, if you're a new user (and especially if you're planning on using 64-bit Ubuntu) you should be using the latest stable version of Ubuntu: 11.10. It has improvements in 32-bit compatibility, as well as numerous other improvements across the distro and software.
