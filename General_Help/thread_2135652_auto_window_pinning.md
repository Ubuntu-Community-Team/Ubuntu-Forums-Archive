### Post by slickymaster on 2013-04-15
Xfce4-taskbar-plugin should solve your problem. It will show up as Task Bar (Windows 7 Task Bar Mimick) in the "Add New Items" dialog for the XFCE panel.

To install, just run at a terminal:```
sudo apt-get install xfce4-taskbar-plugin

```

See [this thread]("http://ubuntuforums.org/showthread.php?t=2090875") if you encounter any issues on installing it on your system.
