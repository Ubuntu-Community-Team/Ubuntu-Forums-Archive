### Post by majoridiot on 2007-02-05
the necessary c++ packages can be installed by-

```
sudo apt-get install build-essential
```

this installs the gcc compiler package, which includes a c++ compiler
and many others.

more info @ [http://gcc.gnu.org/](http://gcc.gnu.org/)

### Post by Ramses de Norre on 2007-02-05
For java: enable multiverse repo and execute: ```
sudo aptitude install sun-java6-jre sun-java6-bin sun-java6-jdk
``` you can also replace the '6' by '5' if you want java5.
