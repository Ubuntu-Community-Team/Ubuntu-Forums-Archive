### Post by jualin on 2008-07-31
I went to Staples 3 months ago and bought an Acer 4720z and spent $299 (Display Model) However they had them for about $500. Everything worked out out of the box except the wireless which need to install the driver but it was very easy to do since it had an Atheros chipset (very common chipset for Linux). Try checking [the linux laptop]("http://www.linux-laptop.net/") website which can give you an idea of how messy you will need to get to make Linux work on your laptop. Good luck!
