### Post by ubuntu-geek on 2004-10-10
From the Ubuntu-announce readers Mailing List

> Dear Ubuntu-announce readers,

In addition to the installation CD images which have been available as part
of the preview release and ongoing daily snapshots, a live CD image is now
available:

Direct download:
	[http://cdimage.ubuntulinux.org/releases/warty/preview/live-i386.iso](http://cdimage.ubuntulinux.org/releases/warty/preview/live-i386.iso)

Bittorrent:
	[http://cdimage.ubuntulinux.org/releases/warty/preview/live-i386.iso.torrent](http://cdimage.ubuntulinux.org/releases/warty/preview/live-i386.iso.torrent)

This CD will allow you to try out Ubuntu before installing it, without
repartitioning or overwriting any existing software or data.  Just place it
in the drive and reboot your computer.

Enjoy!

-- 
 - mdz
