### Post by Ramses de Norre on 2006-05-21
Do you want to install on a flash disk? That's not really a good idea.. Flash memory has a limited read/write cycles, which isn't a problem when used for data storage but you'll drive will break pretty soon when you're running an os from it.

And have you been able to install or run anything with the machine yet? Because it looks like you've got some broken hardware when 3 different os's don't work on it..
Take a look at [this]("https://wiki.ubuntu.com/HardwareSupportMachinesLaptops").
