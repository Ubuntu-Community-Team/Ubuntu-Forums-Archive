### Post by Shazaam on 2008-01-11
cd/dvd drives usually get mounted to /media. Put the disk in and check if it's there.

Places>Computer>FileSystem>media>cdrom0 (or cdrom1)

If it is there open terminal and type in
```
gconf-editor
```

This code will open up the Gnome Configuration Editor. Go to apps>nautilus>desktop. See if  "volumes_visible" is checked.
