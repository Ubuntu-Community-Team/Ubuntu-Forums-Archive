### Post by snowpine on 2011-06-08
Most people find Xubuntu is faster, it is certainly worth a try!

You don't need to create a USB or reinstall, either. You can install Xubuntu from the Software Center, or from the terminal:

```
sudo apt-get install xubuntu-desktop
```

Then log out, and from the login screen, choose the Xfce/Xubuntu option.

Good luck! :)

### Post by keithpeter on 2011-06-12
Hello All

I've decided that I quite like xubuntu, even on relatively fast hardware. Its a bit lighter than Ubuntu and less picky about graphics resources.

You can add 'dmenu' from the 'suckless-tools' package in the repository and invoke programs by keypress in a dash-like way, see my signature for details.

I really like the Alt-F11 undecorate window option together with Ctrl-Alt to switch desktops. You can maximise screen use for your applications.

I like to set a right margin of 3 pixels in the settings | workspace dialogue, then I can middle click on the right of the screen and get a list of all the windows on all the desktops and I can switch to any running window.

I always add libreoffice to the xubuntu application set as I need an industrial strength office package at present, although I am trying to move over to LaTeX and Lyx.
