### Post by ianhaycox on 2008-12-29
In Synaptic mark the packages to be installed then under File->Generate Download Script. After unmark file files.

Run that script and copy your files over.

Haven't tried this but you may be able, using Synaptic, on the target pc to create a software source pointing to the downloaded files and then install the copied files via Synaptic in the usual way.
