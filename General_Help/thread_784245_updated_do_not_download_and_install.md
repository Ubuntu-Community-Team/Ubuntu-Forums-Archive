What output do you get with the command ```
sudo
```? Type it in a terminal (Applications-Accessories-Terminal).

This happened to me when the sudo was broken. The instructions are little convoluted, so if the output above indicates this is the problem, then I/someone can post a fix.

For your reference, if sudo is OK, you should get:```

Tue May 06 22:28:53 ~:$ sudo
usage: sudo -h | -K | -k | -L | -l | -V | -v
usage: sudo [-bEHPS] [-p prompt] [-u username|#uid] [VAR=value]
            {-i | -s | <command>}
usage: sudo -e [-S] [-p prompt] [-u username|#uid] file ...

```
