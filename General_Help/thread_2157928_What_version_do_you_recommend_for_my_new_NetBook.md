### Post by Lars Noodén on 2013-06-27
LTS means that it won't change, except for security updates, for five years starting when it is released.  If you are using the machine for work, then LTS is usually your best value.  If you go with the 12.04 LTS you can keep the same set up until 2017.  

The regular releases come more frequently and are supported only for 9 months now.  So if you go that route, you are basically committed to upgrading your machine every 6 months when a new release comes out.  

[https://wiki.ubuntu.com/Releases](https://wiki.ubuntu.com/Releases)

The next LTS is due out in April of 2014 and will be good until 2019.  If you start now with the non-LTS versions, you have to upgrade twice before landing in LTS where you can coast for five years.
