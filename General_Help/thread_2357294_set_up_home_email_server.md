### Post by bonestabone on 2017-03-31
I've contemplated setting up my own email server for years only to give up at the last minute after realizing how much of a pain it will be. I guess it depends on how old you are, and if you really want to tinker with something like this. 

I would check out this blog post, [https://www.digitalocean.com/community/tutorials/why-you-may-not-want-to-run-your-own-mail-server](https://www.digitalocean.com/community/tutorials/why-you-may-not-want-to-run-your-own-mail-server)

Make sure that's what you really want to do first, then you can look at the tools available.

I think I was leaning towards [https://mailinabox.email/](https://mailinabox.email/) or [http://www.iredmail.org/](http://www.iredmail.org/)

There are other options also but I ended up just going with fastmail because I don't want to worry about email maintenance.
