### Post by lorap on 2005-05-30
hi friend.
let's install VLC player so you won't need install any additional codecs.
open Synaptic Package Manager from menu.
go to Settings-->Repositories.
once you've the repositories window open press Advanced button at the bottom.
mark all checkboxes possible except one "download only upgradable packages" then press Ok.
once the update process completes the repositories window'll close.
now press Reload button.
wait until the process completes.
now press the Search button and write down VLC then press Ok.
after the search process picks you up the vlc player mark it and all its plugins for install then press Apply button.
have a nice day friend!
