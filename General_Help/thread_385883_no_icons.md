### Post by louis_nichols on 2007-03-16
Yeah, that is a real problem. Packages are not always well made and they don't contain the menu shourtcuts. The simple workaround to that is to install the package *menu*. This will place in your main applications menu an icon called Debian, which will contain submenus with all the apps installed on your computer. Or, at least, so it should...
