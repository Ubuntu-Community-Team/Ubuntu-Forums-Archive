### Post by TheAnachron on 2010-10-11
I would try [this version]("http://ftp.uni-muenster.de/pub/mirrors/ftp.ubuntu.com/releases//maverick/ubuntu-10.10-desktop-i386.iso"). (32 bit)
It seems like you are able to use 64 bit systems, however, I found out that the 32 bit is more stable and is a lot more solid.

If you have questions about partitioning, keep asking. I would use GParted! Have at least one NTFS partition so you can move the files to a windows system if you really need to! Don't forget to set up an swap partition so ubuntu workcache is getting enhanced.

Have fun using Ubuntu!
