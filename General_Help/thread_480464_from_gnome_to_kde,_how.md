### Post by dabl on 2007-06-21
Install the meta-package kubuntu-desktop, to get the basic KDE bundle.

I read in a post on Kubuntu forum that both the gnome and KDE bundles can live happily on the same computer, and apparently in the login process you'll get a choice of which desktop environment you want to boot into.  I would advise caution and research before you attempt to remove gnome-everything -- you could inadvertently lose something you really like and use.  Maybe you should run both side-by-side for awhile, before you start deleting stuff.  For example, K3b is giving me trouble on my Kubuntu system, but gnomebaker works great.
