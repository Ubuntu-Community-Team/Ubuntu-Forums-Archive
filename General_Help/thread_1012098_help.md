### Post by ajcham on 2008-12-15
The site was almost certainly designed for Windows machines, therefore any code would not run on your Linux box.  Also, unless you are running as root or have changed the permissions in your / directory (which is highly unlikely, and would be very foolish) there is no way a rootkit could be installed.

Note that the peculiar fullscreen behaviour has been noted several times elsewhere in these forums, and is unrelated to any attempt to compromise machines.
