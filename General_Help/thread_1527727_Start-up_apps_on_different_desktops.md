### Post by VCoolio on 2010-07-09
If you use compiz (that is, if you enabled visual effects), install and run ccsm / compiz-config-settingsmanager, check out the 'place' plugin. Else, use [devilspie](http://foosel.org/linux/devilspie).

Open ccsm, then find the 'place' plugin. Click it, then set up the apps you want to start on certain desktops. [This](http://wiki.compiz.org/Plugins/Place) may help.
