### Post by detwilerm on 2007-07-26
If you still need help with this I found a workaround in compiz. Go into the compiz config manager there is an option under Window Management called Extra WM Actions. Enable it and set the keys for Toggle Fullscreen. I set it to Ctrl+Alt+Enter. Then you will need to put your virtual machine into fullscreen by hitting Ctrl+F, then hit Ctrl again to leave the virtual machine and hit Ctrl+Altr+Enter. This worked for me, hope it helps.
