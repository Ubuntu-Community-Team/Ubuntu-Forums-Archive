### Post by grahammechanical on 2013-05-09
A lot of work went into Ubuntu and its flavours (such as Xubuntu) in the last four weeks before its release at the end of April. I tested 13.04 all through its development period and for months I could not run a live session without using an F6 option (more than one option in fact). Then about February/March something changed and I was able to run a live session in the same way that I always had been able to.

There have been issues with proprietary video drivers but the open source video driver (Nouveau) is much improved. I use it instead of Nvidia drivers. Each release of Ubuntu has newer kernels than the previous release.

I am now testing Saucy Salamander and the live session runs fine and also the installation runs fine.
