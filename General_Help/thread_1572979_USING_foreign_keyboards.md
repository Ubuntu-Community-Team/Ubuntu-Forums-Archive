### Post by NearlyALaugh on 2010-09-12
You can add layouts to GNOME via System > Preferences > Keyboard > Layouts. If you have more than one, a clickable indicator/switcher should show up in the system tray. This can also show you the keys.

Also consider using the "Compose Key" for quick accents:

[https://help.ubuntu.com/community/ComposeKey](https://help.ubuntu.com/community/ComposeKey)
