### Post by ubuntu27 on 2008-11-25
Hello Ubunteros! I have a little annoying problem. Every time I open a new instance of Firefox, it opens in a state similar to Full screen mode (F-11 for the keyboard shortcut). 
And in order to get out or to revert to the normal state, I have to press F-11 _twice_.  That is because if I press F-11 once, it goes to Full-Screen mode for real, so since it is in fulls-screen mode now, I have to press F-11 again.

Is there anyway to fix this? 


I have the following
Ubuntu 8.10
Mozilla Firefox provided by Ubuntu, version 3.0.4

HP Laptop (don't know the name)
Intel Core2 Duo

### Post by fooman on 2008-11-25
try this: open firefox and do the f11 thing twice to get to a normal size screen.

unmaximize firefox (not minimize to panel). resize it to a slightly smaller window size.

close firefox.

open firefox...then maximize firefox.

problem should be gone.
