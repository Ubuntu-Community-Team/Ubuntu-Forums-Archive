### Post by StuartN on 2009-10-14
> **melisotasmith said:**
> Hi I am trying to firgure out how to do a screen saver with a personal message on it ho do i do it

The quickest way to display a written message as a screensaver (without any command line) is to empty your folder called Pictures and go to System->Preferences->Screensaver and select the Pictures folder screensaver.

Now use any application to create text that can be saved as an image - OpenOffice Impress slides can be exported as jpg, or any image in Gimp. Just make sure to save these images to your Pictures folder (and make sure there is nothing personal saved there).
