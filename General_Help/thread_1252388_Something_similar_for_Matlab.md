### Post by JC Cheloven on 2009-08-28
Also there is Euler (EuMathT lately). I like it. It's free software, although natively for windows. 

It was ported to linux-GTK, and that version is in the repos. This is a quite outdated (but mostly suficcient) version. Otherwise, the current version runs flawlessly on wine, and has symbolic capabilities, as it has incorporated maxima.

### Post by bchurchill on 2009-08-28
Another great language is SAGE.  It's designed for linux and based on Python (although extremely fast because it uses C extensions).  It's designed more for advanced theoretical mathematics, like algebra, number theory and geometry.

You can get it from [http://www.sagemath.org/](http://www.sagemath.org/)

I didn't try the binary packages on Ubuntu (which should work fine), but it installed from source without needing any dependencies or installations.  It's entirely self-contained and doesn't modify anything outside its won directories or need anything else.
