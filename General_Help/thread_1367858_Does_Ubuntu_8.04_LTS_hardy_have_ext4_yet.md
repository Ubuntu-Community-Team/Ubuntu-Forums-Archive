### Post by dcstar on 2009-12-30
> **Spaceman9 said:**
> Does Ubuntu 8.04 LTS hardy have ext4 yet? Or does it only have ext3? Thanks for any help.

Previous versions will only get updates to existing features, not new features only available in later releases. Providing new features to older releases means you risk breaking known stable setups, so if you want new features you install a later release.

LTS basically means it will continue to get all updates for a lot longer than non-LTS releases. 10.04 is a LTS, wait a few months for that and then consider an upgrade.
