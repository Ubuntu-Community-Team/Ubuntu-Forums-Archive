### Post by mgmiller on 2008-07-24
Since upgrading to Hardy, I have lost the ability to search google or wikipedia using the deskbar.  

I found this nice site that has a few useful add-ons for Deskbar, including the 2 functions I had lost.

It's very easy to use.  Just download the tar.bz2 file to your desktop and then right click the deskbar icon and select preferences.  Drag the tar.bz2 file onto the searches tab.  It should say the item installed succesfully.  Then, scroll to the bottom of the list and put a check mark next to the new entry to enable it.  You can also change its displayed order by clicking the up down arrows.

Fast, neat easy.  I can search google and wikipedia again.

Enabling the google search requires one additional package to be installed first, so do this before installing the google addon:

```
sudo apt-get install python-simplejson 
```
The web site is here:

[http://www.k-d-w.org/node/39#wikipedia](http://www.k-d-w.org/node/39#wikipedia)
