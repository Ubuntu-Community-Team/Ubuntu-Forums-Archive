### Post by rune0077 on 2008-05-22
After you've got btnx installed, you need to run the config-program Applications > System Tools > btnx. It needs to test your mouse, so do everything the program tells you (it's pretty straight forward), and after that you can assign specific commands to each of your mouse-buttons, provided they got identified during the testing.
