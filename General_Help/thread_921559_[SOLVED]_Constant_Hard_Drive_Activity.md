### Post by eggdeng on 2008-09-16
Run
```
top
```
to see which programme is so active.

### Post by kep1616 on 2008-09-16
Thanks a bunch! I had been only using the system monitor and had not noticed anything, however when I ran that program I noticed that mythtv was running. I had uninstalled it once, but this time I used Synaptic to completely remove it. I have no idea why all of the sudden it started acting up, but it is fixed now.

Thanks again!
