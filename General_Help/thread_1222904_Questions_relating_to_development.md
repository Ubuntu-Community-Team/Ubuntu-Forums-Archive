### Post by Idefix82 on 2009-07-28
I find this a good place to start if you want to contribute: [http://live.gnome.org/GnomeLove]("http://live.gnome.org/GnomeLove")

In terms of programming, knowing Python and/or C would be very handy.

For the cosmetic side of things, developing whole new themes can be challenging but developing an icon theme, say, only requires you to be able to handle a graphics program like Inkscape. Just download an icon theme from [http://www.gnome-look.org/index.php?xcontentmode=121]("http://www.gnome-look.org/index.php?xcontentmode=121") and start changing some of them. You will quickly understand the structure of the directories.

Good luck and have fun!

Edit: One more thing - you would be immensely useful to the Ubuntu community if you installed the alpha and beta versions of the OS as soon as they come out (Karmic Koala is in its testing stage at the moment), tested and reported bugs on [https://bugs.launchpad.net/ubuntu]("https://bugs.launchpad.net/ubuntu")
