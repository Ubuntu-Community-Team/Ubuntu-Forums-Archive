### Post by Diametric on 2011-05-23
I have found the Linux learning curve to be very high, but the rewards to said learning even higher.  

There are two books that have been instrumental in helping me become a "power user"...I hope they're able to help you also.

For learning bash (critical in learning Linux):

[http://www.amazon.com/Learning-bash-Shell-Programming-Nutshell/dp/0596009658/ref=sr_1_1?ie=UTF8&qid=1306203299&sr=8-1]("http://ubuntuforums.org/Learning%20the%20bash%20shell")

and for learning everyday practical usage tools via the CLI (bash):

[http://www.amazon.com/Ubuntu-Linux-Toolbox-Commands-Debian/dp/0470082933/ref=sr_1_1?ie=UTF8&qid=1306203424&sr=8-1]("http://ubuntuforums.org/Ubuntu%20Tool%20Box")

Good luck and enjoy!
