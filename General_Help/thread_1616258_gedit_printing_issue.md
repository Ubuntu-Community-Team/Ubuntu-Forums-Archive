I'm having an issue printing from gedit under ubuntu 10.10. Specifically I am having problems that are related to setting preferences for printing, such as print font preferences and the like.

The Help document ([F1] within gedit) suggests that I should be able to set fonts and other page options for printing by choosing "File > Page Setup" to display the Page Setup dialog. That's all well and good for the help document, however there is no 'Page Setup' in the File menu under gedit 2.30.3 installed on my desktop.

On this install, the place to actually make that variety of selection is within the 'Print...' dialog on the 'Text Editor' tab. I'm not sure that this is an intuitive place for this configuration item.

A side effect of this is that the 'File > Print Preview' selection does not provide a usable preview until after you have printed a document if you decide to make such changes. This somewhat defeats the purpose of having a print preview entry under the File menu.
