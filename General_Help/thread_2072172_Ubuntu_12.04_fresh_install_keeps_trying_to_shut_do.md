### Post by Gizmonty on 2012-10-17
I have just done a fresh install of Ubuntu 12.04 32 bit on my machine that was previously running Mythbuntu 11.10 just fine.

Once it starts, I get the Unity desktop and then after a few minutes a dialog box appears stating that the system will shut down in 60 seconds. If I don't dismiss the dialog box my computer shuts down. If I do dismiss it, I can continue to use the computer but the dialog box keeps reappearing every few minutes.

Is there any way of identifying what triggers this shut down and how might I go about preventing it. The system seems to work perfectly otherwise.

---

### Post by Gizmonty on 2012-10-18
The problem turned out to be a faulty power switch. Ubuntu is hereby exonerated from blame.
