Actually Ubuntu's implementation of Xfce is harder on ram than regular Ubuntu is. I would recommend looking at [Lubuntu]("https://wiki.ubuntu.com/Lubuntu"). Unfortunately the Lubuntu project does not yet have an install disk available, but you can still install the LXDE desktop environment via:

```
sudo apt-get install lxde
```

or follow [this guide]("http://lxde.sourceforge.net/install.html").
