### Post by Frogs Hair on 2014-07-14
Although there are Linux versions of the installer on the website it doesn't specify what Linux operating systems or versions it will run on . Libraries change from one Ubuntu version to another so there is no way to know from what is visible on the download page if the software provider has updated the installer to run on 14.04.

 I would check their forum to confirm compatibility with your version of Ubuntu. Linux distros in general change rapidly and software provided from sources outside of their own repositories can't be checked for compatibility by those who manage those repositories. I found some posts for previous Ubuntu versions regarding this game , but nothing for 14.04.
