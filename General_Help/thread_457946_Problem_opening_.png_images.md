### Post by JSThePatriot on 2007-05-29
If "earlier" you could open *.png files without problem, what have you installed since the last proper opening of a PNG file? It seems you have some sort of file security check running upon opening of files. That warning is something one would get in windows on an executable. It seems really odd to me that you are getting it on PNG files. I would recommend that you look at how PNG files are supposed to be opened. The system may have changed them to an "executable" extension, and it just needs to be changed back to a graphic extension.
