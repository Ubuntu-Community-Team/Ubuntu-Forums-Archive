### Post by txwooley on 2013-05-07
I have several programs that I compiled from source.  Say for example one of them depends on libssl-dev.  If I update libssl-dev, do I have to re-compile the program or will it automatically use the latest one?

---

### Post by CatKiller on 2013-05-07
It depends. Occasionally a new version will have changes that are incompatible with older versions, but it's rare and normally that kind of thing gets you both versions in the repository named different things in case users need one or the other. Otherwise, no, you'll be fine.
