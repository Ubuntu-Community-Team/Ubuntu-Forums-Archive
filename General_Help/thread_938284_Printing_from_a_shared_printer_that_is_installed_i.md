### Post by daUbuQ on 2008-10-04
In my house, we have a desktop with Windows XP and all the printers are connected to it. All of these printers are shared. When I was using windows xp, printing on a shared printer was very easy. However, when I switched to Windows Vista, I can't print on Windows XP shared printer (I still don't know why).

I wanted to test printing from Ubuntu. It was realy very very easy. All what I have done is click on System --> Administration --> Printing

Then, I clicked on the Edit menu and choose New Printer .
After that, from the new printer window I selected Windows Printer via SAMBA .
THen browsed for the computer name that the printer is installed in it and selected the printer name.
And that's it . I can print any file from Ubuntu to that printer .. :-)
