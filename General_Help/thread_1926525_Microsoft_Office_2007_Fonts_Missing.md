### Post by orangeappeal on 2012-02-16
That didn't work unfortunately. I did find a way to get the font back though. I downloaded the .ttf font from [HTML]http://www.ufonts.com/download/times-new-roman/34369.html[/HTML] then I saved the file in my play on linux folder. To find the folder I had to press ctrl H to show hidden folders. I saved the .ttf file here PlayOnLinux/wineprefix/Office2007/drive_c/windows/Fonts/
I don't know what caused the file to go missing but this is my temporary fix and I hope it can help somebody in a pinch.
