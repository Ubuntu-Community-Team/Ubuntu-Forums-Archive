### Post by d0nny2600 on 2007-12-17
You haven't provided much information but I'll give you examples on how you can test it. 

If its a 3.5'' hdd
1. Install the hdd in a free bay in the workstation. Boot up your O/S as normal. It should be detected automatically on booting up. You may or may not have to mount the drive within Linux.
2. Change the currently used hdd and replace with the new drive. Boot in Ubuntu using the live cd and see if you can view the contents from within Ubuntu.

If its a 2.5'' hdd
1. Replace the currently used hdd and replace with the new drive. Boot in Ubuntu using the live cd and see if you can view the contents from within Ubuntu.

OR

Buy yourself a usb hdd caddy for the hdd. This will allow you to mount the hdd within ubuntu as a removeable disc. You can check the contents from there.

Hope that helps,
