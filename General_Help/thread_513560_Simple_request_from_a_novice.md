### Post by pak33m on 2007-07-30
littleones,

You could also select Alt + F2 and type ```
gnome-terminal
``` (if using gnome which is the default Ubuntu install) in the Run Application box 

Or type ```
konsole
``` (if using Kubuntu) in the Run Command box.  

Just another way to get to it.
