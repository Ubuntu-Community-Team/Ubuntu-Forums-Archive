I have Ubuntu 12.04 on one hard drive and Vista on the other drive. I was having trouble with Vista. I put the windows dvd into the drive to do a repair. Maybe it is time to update to a later version of Ubuntu.

### Post by Impavidus on 2014-05-01
Windows cannot install files on a Linux partition without first formatting it to a Windows partition. If it had done so, Ubuntu wouldn't have worked at all.  But you're free to install 14.04 anyway. Make sure you've got backups of your important files.
