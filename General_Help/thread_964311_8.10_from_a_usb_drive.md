### Post by C.S.Cameron on 2008-10-30
If you can boot the Live CD you can make an installer USB using Create a USB startup disk, (usb-creator), located in System, Administration.
You can also download usb-creator to 8.04 using sudo apt-get install usb-creator.
You can then use U-C to install the image to a bootable flash drive.
If you are using a windows machine you can make an installer USB from the iso using unibootin.

Oops sudo apt-get install usb-creator does not work in 8.04, to install it try:
[https://launchpad.net/ubuntu/intrepi...creator/0.1.10](https://launchpad.net/ubuntu/intrepi...creator/0.1.10)
