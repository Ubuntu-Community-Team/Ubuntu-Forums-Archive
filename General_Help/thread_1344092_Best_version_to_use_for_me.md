### Post by StuartN on 2009-12-02
> **prions said:**
> The thing that bothers me is that my graphics card is no longer supported past 8.10. Would this cause problems if I want to play some windows games?

The card is supported by the open source driver, which is compatible with the new version of X. The manufacturer is no longer developing the closed source, proprietary driver, and the latest (legacy) version is not compatible with the new version of X.

I have this graphics card in two machines running Ubuntu 8.10 (proprietary driver) and Ubuntu 9.10 (open source driver). Compiz graphics effects work fine in both.

All applications I have tried have worked.

I did find that the open source driver was considerably slower than the proprietary driver at the time the changeover occurred (in 9.04), although I have not compared speeds recently.
