### Post by sisco311 on 2009-12-16
> **KayakJim said:**
> I was playing around with the Profile settings available from Edit > Profiles in the toolbar for the gnome terminal in Ubuntu 9.04 amd64 Jaunty. Within those options I disabled the toolbar.

How can I get the toolbar to display or how can I reset the profile back to it's default values?

Right click on the terminal window -> Show Menubar 

then goto Edit -> Profile Preferences -> General tab -> select the *Show menubar by default in new terminals* checkbox.
