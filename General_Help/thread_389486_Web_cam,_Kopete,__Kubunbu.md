I had a lot issues with my cam with darkness and poor picture but this is how I fixed it

*Kopete is designed for KDE so set up a dual boot with Ubuntu and Kubuntu.

*Installed Kopete on Kubuntu 

*Go to 'Settings' then 'Configure'

Configuration works perfectly on Kopete within KDE (Kubuntu) and the picture could not be sharper.

All I got is a Lebtec cam - a really cheap one if not the cheapest one.  I have not got drivers for it  and use the Kopete drivers.
