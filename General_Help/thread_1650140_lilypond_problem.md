### Post by cloyd on 2011-01-02
Lilypond runs from the command line.  It really isn't that difficult. Create the ly file with gedit, and "compile" it in Lilypond. A helpful hint here. When you compile the ly file, it creates a midi file, a pdf, and a ps file. After you compile the ly file for the first time, open the ps file and keep it open as you work on the file. Each time you compile the ly file again, the ps file will automatically update without having to reopen the ps file. (If you try to monitor your progress with the pdf, you will have to close the file and reopen it in order to get it updated after each compile.)

Wish you luck. I'm just learning how to use lilypond, and like it much better than my old finale for what I do. However, there is a learning curve.

There is some documentation on the lilypond web site . . . it is a necessity that you get this. I don't think you can even start to use lilypond without the documentation.
