### Post by jml on 2007-02-12
I agree with what has been said so far.  For a Debian based CLI install, use either the Server edition of Ubuntu, or do a minimal Debian Etch net install.

If you really want to sink your teeth into Linux,  ;)   Then consider Slackware.  Its a distro with a lot of history, its very stable, (no cutting edge packages,) in fact it still uses the 2.4 kernal.  Installation and configuration is done by editing configuration files with a text editor like vi or emacs.  I've never had the guts to try it, but if you really want to immerse yourself in Linux this is one way to do it.  I once read a saying that went like this.  If you want help with your computer, ask a Linux user.  If you want help with your Linux installation, ask a Slackware user.  Good luck on your journey.
