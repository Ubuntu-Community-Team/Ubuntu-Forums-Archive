### Post by mc4man on 2010-10-21
You can install it and then use on a build specific basis

I'd start here, download all the neeeded packages into an empty folder, cd to that folder and use 
sudo dpkg -i *.deb

[http://packages.ubuntu.com/lucid/g++-4.1](http://packages.ubuntu.com/lucid/g++-4.1)

Figure 5 are needed
g++-4.1
gcc-4.1 
gcc-4.1-base
libstdc++6-4.1-dev
cpp-4.1
