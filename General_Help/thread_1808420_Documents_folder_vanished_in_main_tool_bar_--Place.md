### Post by jmszr on 2011-07-20
Mat11,

Open Nautilus by going to Places > Home Folder. Now navigate to the Documents folder, and go to Bookmarks > Add Bookmark. The Documents directory should be added to the left pane. It will also be added to the Places menu.

Hope that helps.
