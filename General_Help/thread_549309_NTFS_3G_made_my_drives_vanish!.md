### Post by dabl on 2007-09-12
> **ameya.satyajit said:**
> 

/dev/sda6 /media/Extras ntfs-3g defaults,locale=en_IN 0 0

/dev/sda5 /media/Windows\040Vista ntfs-3g defaults,locale=en_IN 0 0



When you ran ntfs-config, it screwed up your fstab file -- this stuff is gibberish.

If there is a backup of fstab, you probably can restore it and get something that is closer to working.

Otherwise, you'll have to work out what the correct fstab entries would be.  From your fdisk listing, it looks like they should be something like:

/dev/sda6 /media/sda6 ntfs defaults,locale=en_IN 0 0
/dev/sda1 /media/sda1 ntfs defaults,locale=en_IN 0 0
/dev/sda5 /media/sda5 ntfs defaults,locale=en_IN 0 0
