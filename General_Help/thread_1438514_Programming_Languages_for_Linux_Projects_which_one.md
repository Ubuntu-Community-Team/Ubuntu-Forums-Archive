### Post by Spiritof76 on 2010-03-25
It mostly depends on what you want to do. There have been lots of cyberink spilled debating the superiority of one language over the other.  That all being said Here is my opinion.

Bash is the most useful tool for making things happen in a Linux system. It is what I know I need to spend most of my effort learning.

Python is great for accomplishing tasks.. because it is an interpreted language, It is easy to learn and quick to develop. 

Pearl has simular strengths to Python, and is a better at text string handling.  but I don't find the community as useful

C I think people need to walk before the need to run.  C is work horse effiecient an fast. If one objects there is always C++ with all its structures and objects it becomes the powerthouse of the programming languages.  I believe one is better off learning classical C first before taking on the additional contexts of C++

Java, Is great if you want to right big slow programs that crash a lot. 
Your mileage may vary.
