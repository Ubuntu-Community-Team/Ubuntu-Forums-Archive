### Post by Vignesh S on 2009-07-12
Provided that the Intel integrated graphics has 64 MB VRAM (which I think it does), Doom 3 should work. But I wouldn't recommend putting it on the setting you have mentioned. Maybe a bit lower. Dunno if Doom 3 is as graphics intensive as Halo 2, but apparently, someone was able to run it on the Asus N10 (but that had a 256 MB Nvidia graphics card). Just to stay on the topic of graphics, I crashed my netbook (10", but the graphics was pretty much the same specs) after I ran way too many Compiz special effects. So be careful whenever you are running any graphics intensive apps on a netbook



Oh and by the way, the only reason one would want to downgrade is if there is either a glitch that is really annoying (not that there would be, Ubuntu is thoroughly tested before it is officially released) or if you don't want to upgrade (and download at least 1GB of Ubuntu upgrades) once every 6 months. I don't think that it really makes any difference on which version of Ubuntu you run because on every version, the drivers will be updated anyway. So you might as well stay on Ubuntu 9.04
