I am using hardy xubuntu.

Is there a way of telling FireFox 3 to use the same applications to open files as are defined in Thunar?

Late edit:
This problem occurs in Xubuntu amd Kubuntu.
It is solved by installing the 'firefox-gnome-support' package.
