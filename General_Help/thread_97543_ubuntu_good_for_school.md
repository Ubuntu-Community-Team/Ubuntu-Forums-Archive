### Post by magnusbb on 2005-12-01
You're certainly better off with Ubuntu. And for education purposes there is a special flavour of Ubuntu called Edubuntu. 

[http://www.edubuntu.org/](http://www.edubuntu.org/)

The nice thing about Ubuntu is, that it provides a nice and simple base for you to extend. If the hardware setup is right in the distributions, with all the correct drivers, it should be faster than Mandrake. Many have complained about the speed of Ubuntu Breezy, and found that the problem mainly was hardware related.

The sad thing about open source, is that on some areas the coders don't always have the end user in mind, speaking of hardware capabilities. The programmers use fast computers, and the code is not as quick on older machines. The good thing about open source is that you are able to, and allowed to, do something about that.
