### Post by tredegar on 2011-01-04
**wine** can be used to run *some* windows applications. Many do not run, because they need "real windows software".

But you say you can read the photos from your camera with linux. This is a good start.

There are then many (free) linux applications to help you with these photos.

**gimp** is a good image editing program. Takes some time to learn, but it is *very* powerful. Plenty of tutorials on the net.
**fspot** will help you manage your collection.
There are lots of other linux applications to help you. Try searching with [http://google.com/linux](http://google.com/linux)

Have fun.
