I've installed NUT on my Ubuntu machine and the shutdown command used (when power is lost) = *shutdown -h +0*

---

### Post by CatKiller on 2019-04-03
It says in how many minutes you'd like the shutdown command to run. You can also specify a particular time, or simply use now.

### Post by CatKiller on 2019-04-03
Yes, that's right.
