### Post by 3rdalbum on 2007-11-09
Since you use an ATI card, and ATI's drivers do not directly support Compiz, you will need to install a piece of software called XGL.

Go into the Synaptic Package Manager and install the package "xserver-xgl". Restart the computer and you will be able to use the Compiz visual effects.
