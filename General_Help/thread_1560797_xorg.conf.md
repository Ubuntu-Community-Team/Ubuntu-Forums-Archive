### Post by endotherm on 2010-08-25
the xorg.conf started becoming vestigial back with Intrepid, so it is only used if an app chooses to configure it. otherwise, all the info that used to be contained there is autogenerated at login. you can see a lot of that happening in your xorg.0.log if you want to look.
