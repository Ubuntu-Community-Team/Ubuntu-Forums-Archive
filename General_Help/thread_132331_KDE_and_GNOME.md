### Post by cronholio on 2006-02-18
No. But why bother right now? If you only want to try out KDE, install it through Synaptic and simply select it from the login screen. You can safely remove it later.

Moreover, you can run most Gnome apps in KDE and vice versa if you have the needed libraries installed, which should be the case when you install both Gnome and KDE.
