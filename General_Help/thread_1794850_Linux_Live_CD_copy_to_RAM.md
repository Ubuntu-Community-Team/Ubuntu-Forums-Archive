Hi, as far as I know it can not be done that way for at least two reasons.
1. You still need the livecd in the drive.
2. You will not have the packages installed to watch dvds.
I guess you could make a persistent usb pendrive of ubuntu and then install the restricted packages to it from the software center, I think that would work. Here is a link to make a persistent pen drive.
[https://wiki.ubuntu.com/LiveUsbPendrivePersistent](https://wiki.ubuntu.com/LiveUsbPendrivePersistent)
