### Post by marl30 on 2011-01-06
It won't work in a Virtual environment. You might get it to work in Wine, according to the Wine site. However, it may require a few Winetricks extras to work. [http://appdb.winehq.org/objectManager.php?sClass=version&iId=19125](http://appdb.winehq.org/objectManager.php?sClass=version&iId=19125)

### Post by James78 on 2011-01-06
You could also try PlayOnLinux. Which is a special program which utilizes WINE and creates special WINE instances for each game; where in regular WINE it's just one instance (.wine) for everything. The program also includes game setup scripts; basically it installs all the specially needed programs and whatever else automatically for you; that is if the game is listed.
