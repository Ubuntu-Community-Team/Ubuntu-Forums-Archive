AFAIK, .fon fonts are windows only. I just found this out when trying to do the same thing (saw the link of digg.com). So someone needs to convert it, or you can use a similar font, such as Proggy: 

[http://www.proggyfonts.com/index.php?menu=download](http://www.proggyfonts.com/index.php?menu=download)

[http://www.tobias-jung.de/seekingprofont/index.html](http://www.tobias-jung.de/seekingprofont/index.html)

There's plenty others out there.
