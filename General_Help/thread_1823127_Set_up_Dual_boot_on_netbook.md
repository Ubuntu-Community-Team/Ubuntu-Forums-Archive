### Post by sanderd17 on 2011-08-11
A live CD and a live USB works exactly the same. The easiest way to create a live USB is inside Ubuntu with the USBcreator program.

BTW, Octave is a good open-source alternative for Matlab, and almost completely compatible with it (although it is missing some libraries): [http://www.gnu.org/software/octave/FAQ.html#MATLAB-compatibility](http://www.gnu.org/software/octave/FAQ.html#MATLAB-compatibility)

And what is the best linux distro, that's a personal choice. It depends on how you use your computer and what hardware you have. But the netbook and the desktop version of Ubuntu have merged, so you won't find references to a "netbook" version anymore.
