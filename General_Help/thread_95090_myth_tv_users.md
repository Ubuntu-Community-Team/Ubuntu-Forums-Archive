Depends - there are a number of guides around that make setting up an Ubuntu/MythTV box, especially with the pvr150, relatively straightforward. 

With two caveats: First, most people find it easier to install the regular x86 flavor of Ubuntu rather than the AMD64 version in order to run more multimedia aps. (That's not to say it can't be done, it just sometimes requires a work around). Second, hopefully, you aren't afraid of attemptimg to compile drivers for the pvr150. (It's not that bad, as long as the directions are concise). 

As long as you're willing to tinker a little while, and/or have some experience getting things up and running on linux, you should be fine.

If you haven't seen [this webpage]("http://www.quietglow.com/docs/ubuntumythtv.html"), or [this thread]("http://www.ubuntuforums.org/showthread.php?p=520480"), they're both pretty helpful and almost exactly what you're asking for.
