### Post by aesis05401 on 2009-06-22
.bashrc is a file located in your home folder that contains parameters for adjusting bash.

Open two terminal tabs for side by side viewing - one for read-only view of the settings file where you will make the change, and one for the manual entry for bash that explains the settings.

In the first (from your home folder for read only access):
```

less .bashrc

```

In the second:
```

man bash

```

You can make all sorts of config changes with this information.

Good Luck.
