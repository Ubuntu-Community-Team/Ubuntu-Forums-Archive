### Post by blackened on 2007-06-02
First you need a theme, try [http://gnome-look.org]("http://gnome-look.org") for starters. Download the .emerald file to your desktop, or, if it's zipped, then extract it until you have the emerald file on your desktop.

Next, right-click the Beryl gem icon in the notification area and select "Emerald Theme Manager" from the context menu. When the manager opens, press the "Import" button and point the open file dialog to the emerald file on your desktop. Once back in the manager you should be able to select the newly installed theme from the list.
