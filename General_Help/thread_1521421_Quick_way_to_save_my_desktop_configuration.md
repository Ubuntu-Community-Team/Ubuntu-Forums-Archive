### Post by flick on 2010-06-30
In Ubuntu to get a list of all the stuff you have installed :

[http://ubuntuforums.org/showthread.php?p=1990911](http://ubuntuforums.org/showthread.php?p=1990911)

Then boot into Windows and defrag your existing Windows install.

Then use GParted to partition your drive :

To resize your partitions use GParted. Helpful links :

[http://www.wikihow.com/Use-Gparted](http://www.wikihow.com/Use-Gparted)

[http://gparted.sourceforge.net/documentation.php](http://gparted.sourceforge.net/documentation.php)

Then reinstall Ubuntu, install your apps, etc.
