### Post by Mark Phelps on 2010-02-24
OK, so this is probably going to sound silly ... but I had exactly this same problem from one of my clients a few weeks ago.  Playing WOW, game freezes after a while, had to cold reboot.

Installed a thermal monitor.  Booted the machine -- temps were OK.  Started WOW. Within a few minutes, temps were in the "red" range.

The problem?  Opened the case and discovered that the CPU fan, heatsink fins, and video card fan were all clogged with dust.  Cleaned them all.  Machine runs file now with no seizing or crashing.

Yeah, I know this sounds very simplistic -- but when a problem spans all the different things you tried, hardware is most generally the problem.
