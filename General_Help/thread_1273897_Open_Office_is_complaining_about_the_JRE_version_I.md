### Post by arvevans on 2009-09-23
SOLVED:

Never mind.  I found it.  In /tools/options in OO you can see which JRE(s) you have installed and then select which one OO will be using.

Seems this should be an automatic thing, but then I'm not the programmer who built OO, so guess I shouldn't complain too loudly.
