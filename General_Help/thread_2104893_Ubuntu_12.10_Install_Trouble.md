### Post by Giveingitago on 2013-01-14
Hi again,
Have a read here for install help:

[https://help.ubuntu.com/community/Installation](https://help.ubuntu.com/community/Installation)

For me I down load an Ubuntu ISO image, burn it to a CD. Test it with a live session. You may have to change you boot options in the BIOS to do this, select BOOT order DVD/CD Drive then USB and then Hard disk. This gives you the option to boot from CD or a USB drive to try the live session. If the live session goes OK there is a short cut on the Ubuntu desk top to install. I would recommend backing up any personal data on your Windows install first.
