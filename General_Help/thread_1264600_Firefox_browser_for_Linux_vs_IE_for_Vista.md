### Post by QIII on 2009-09-24
> **scouser73 said:**
> 
**3** - There is no need for anti-virus software as there are no known viruses "*In the wild*", 


Effectively true, but not quite accurate.  In 2006, Kaspersky cataloged roughly 800 Linux viruses "in the wild" (compare that to hundreds of thousands for Windows.)

More accurately, there are virtually no ***effective** *viruses.  You could "get" one if you took the time to log in as root and install one (actually simple malware would be more like it, since a "virus" implies propigation) -- say someone with malicious intent said "Hey, if you install this you'll get better performance..."  That would take some pretty swift social engineering.  But it is a good reason to avoid installing anything that does not come from a trusted source.

The Ubuntu repositories (and similarly for other distros) are carefully scrubbed for just that reason.  If you were to 

```
sudo apt-get install somethingwithmaliciouscodeinit
```you might be out of luck.  So you are virtually care free there as long as you stick to trusted sources.

There is, however, the issue of passing viruses and malware along to your Windows friends in an email or some sort of file, so it is not such a bad idea to use ClamAV or something similar to scan emails and files.
