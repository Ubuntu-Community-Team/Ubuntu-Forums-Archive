### Post by Kethinov on 2006-04-29
Check the ~/.wine/drive_c/Program Files/program_name folder for an uninstaller and run that. If there's no uninstaller, just delete the files the program installed.

---

### Post by YokoZar on 2006-04-29
Open a terminal and type "wine uninstaller"

This brings up an uninstall program similar to the Windows add/remove programs dialog.
