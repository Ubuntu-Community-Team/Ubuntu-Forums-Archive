### Post by SpaulS on 2004-12-18
I am posting this as a "FYI". I was trying to use the OpenOffice Autopilot, and it kept returning "The files files required could not be found. Please start the 
OpenOffice.org Setup and choose 'Repair'". I have not had this problem under Xandros or Windows XP OpenOffice installs. This problem is actually an OpenOffice setup issue. The default install used UK English, but I installed for US English. In OpenOffice, go to Tools -> Options, and under "Paths" make sure the paths for templates, hyphenation, dictionaries, and other files point to the appropriate directories.
