### Post by 3rdalbum on 2009-06-08
You must unmount the USB drive before pulling it out. Right-click on its icon and go to Unmount, then wait for the icon to disappear from the desktop. Very important.

You can uninstall software by going to Synaptic Package Manager and marking the unwanted software for removal.
