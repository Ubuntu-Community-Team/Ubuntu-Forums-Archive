### Post by ianmillington on 2009-09-17
Start a new document. If it is something that needs to be in a pre-defined style (eg a company report) and you have a pre-defined template start with that.

Before you type anything, set up a page style (open the stylist then open the page styles pane and select new by right clicking in the pane). When the page style dialog opens set up the characteristics of the page you want, including giving it a distinctive name. Then save the document as a template, giving it whatever name you want. 

Every time you use that template in the future, that page style will be available for use. To use it, select insert/page break and select the lanscape style. The document will continue in landscape until you select insert/page break and then select some other style.
Hope that helps

### Post by ianmillington on 2009-09-17
Yes, it does work, perhaps I have not been totally clear.

Open the sylist and select the page styles icon. Then in the white space in the stylist right click and select "new". The resulting dialog box is fully configurable and hitting OK in the dialog box will immediately send the new page style to the stylist. The decisive act is hitting Ok 

This is also covered in openoffice help under the heading

"page styles;creating and applying"
