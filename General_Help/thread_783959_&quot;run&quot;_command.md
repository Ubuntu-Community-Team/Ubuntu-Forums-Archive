### Post by Koori23 on 2008-05-06
as far as running commands using the "run" like Windows. The program is called xfrun4 and it can be accessed by using ALT + F2.

As far as downloading a packaged (.deb only), all you need to do is double click the .deb file and follow the instructions. That's a program called GDebi and it'll open automatically, similar to a Windows Setup dialog box.

Pokerstars is a Windows only exe file, it can't be directly used on an Ubuntu system without a program called WINE.

Getting back to the "run" command. This is what you'd do if you wanted to   open mousepad

ALT+F2
dialog box pops up

type in *mousepad*
