### Post by phinullfermata on 2008-12-08
You could try mutt, which is used in the terminal as far as I know.  This might help, although you'll need to change it for hotmail instead of gmail... shouldn't be TOO different:
[http://ubuntuforums.org/showthread.php?t=565326&highlight=mutt](http://ubuntuforums.org/showthread.php?t=565326&highlight=mutt)

Alternatively you could try setting up evolution or Thunderbird for mail with hotmail.  There's an interesting plugin for thunderbird out there called Muttator, although I don't think it's finished yet and I haven't tried it.

If you really like VIM you can try to use the Vimperator plugin for firefox, too, and see if you can use the <C-I> command to enter your email message into the text box.  This doesn't work for everything, though, so I don't know if it will help.  Make sure you read the documentation for both plugins before you use them though, they're quite different, and not everybody likes them.

Here's the link for both: [http://vimperator.org/trac/](http://vimperator.org/trac/)

Hope this helps.
