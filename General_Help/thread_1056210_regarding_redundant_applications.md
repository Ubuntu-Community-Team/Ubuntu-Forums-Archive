### Post by cariboo on 2009-02-01
It won't increase your performance, but it will free up some ram. You can safely stop the following:

At SPI Resgistry Wrapper -- unless you use Orca
Bluetooth
Check For New Hardware -- if you don't have any external drives
Evolution Alarm notifier -- if you don't use evolution
Print Queue  Applet
Remote Desktop
Tracker
Tracker Applet
Visual Assistance -- if you don't use Orca
