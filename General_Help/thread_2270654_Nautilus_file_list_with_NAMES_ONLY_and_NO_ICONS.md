### Post by mc4man on 2015-03-24
It's likely thumbnails not icons that are your issue?
As far as nautilus it doesn't allow [disabling thumbnails on a per file or per dir. basis]("https://bugzilla.gnome.org/show_bug.cgi?id=586739"). So it's allow or disallow, set in Edit > Preferences > Preview

( though I guess one could create a 'toggle switch' to enable/disable by clicking on.
