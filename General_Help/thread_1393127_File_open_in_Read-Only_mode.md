### Post by SecretCode on 2010-01-29
I think this is working as I would expect it to. If you want to edit a file you've received as an attachment, you should definitely be saving it somewhere, not using Thunderbird as the place to store it.

Note that you can save all attachments in one email at once, via the File > Attachments > Save all... menu or by right-clicking in a blank area of the attachments pane at the bottom of the email. I'm not sure if you can save attachments from multiple emails at once - but I think there is a Thunderbird extension for this - have a look at [Add-ons for Thunderbird](https://addons.mozilla.org/en-US/thunderbird/)
