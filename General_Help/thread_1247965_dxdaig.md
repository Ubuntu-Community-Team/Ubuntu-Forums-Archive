### Post by CatKiller on 2009-08-23
> **Sticks_uk said:**
> Universe repository ? where can i find that ? :)

System &#8594; Administration &#8594; Software Sources. Put a tick in the box (if it isn't there already) that says "Community maintained Open Source software (universe)."

If you've enabled a new repository, you'll need to hit the Reload button in Synaptic (System &#8594; Administration &#8594; Synaptic Package Manager) to get a new list of available software from the new repository before you can install any of it. You should always install software through a package manager.
