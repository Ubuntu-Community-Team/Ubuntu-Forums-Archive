### Post by thecoshman on 2009-12-04
I am sure that this has already been posted, but I found it hard to find any full help. Before people get confused, Iplayer has a standalone client that you can use to download programs to watch them off-line, or my case, watch them at all (my internet connection at uni is too slow to stream properly).

At first, I was following [[COLOR="Navy"]this[/COLOR]]("http://ubuntuforums.org/showthread.php?t=1298378") guide, but I was not able to do all the steps.

I was not able do to step 4, didn't think .bin meant much to Ubuntu.

Step 5, the first command, could not find that package.

Step 6, well at step 6 I had wondered of to other methods.

I then found a thread in which some one said they heard "air V2 beta" worked with Ubuntu. Well, I googled something like air v2 and found [[COLOR="Navy"]this[/COLOR]]("http://labs.adobe.com/downloads/air2.html") page, a download page for the latest beta version of adobe air. I installed the debian pacakge for Linux (obviously - not sure how I would use the BIN).

I was then able to go to the Iplayer install page [[COLOR="navy"]here[/COLOR]]("http://www.bbc.co.uk/iplayer/install/"). Installed fine!

**So, in conclusion**... I have installed the macromdia flash plugin, rather then the version that comes with Ubuntu (I think :-?), installed a seemingly random set of libs, installed adobe air beta version 2, then finally Iplayer.

**I would advise**

try installing the [[COLOR="navy"]air beta [/COLOR]]("http://labs.adobe.com/downloads/air2.html")first, see if you can get Iplayer to work.

if not, try using the marcomdia version of flash rather then what comes with ubuntu, [[COLOR="navy"]this script[/COLOR]]("http://www.cyberciti.biz/tips/install-flash-10-ubuntu-linux-64bit.html#comment-150888") will do it for you (copy the code into a text file, save it, make it executable and then run it).

Sorry that this is still not a definitive way of getting Iplayer to run, but If people reply with what they did (and didn't) do to get Iplayer working, hopefully (if I can edit this - I'm new to this forum) I can keep this post up-to-date with how to install Iplayer.
