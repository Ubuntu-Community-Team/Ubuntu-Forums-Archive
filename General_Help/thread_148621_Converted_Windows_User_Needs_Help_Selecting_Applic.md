### Post by siorai on 2006-03-22
You can run DVDShrink in Wine. There's a thread here: [DVD Shrink in Breezy]("http://www.ubuntuforums.org/showthread.php?t=78611&highlight=dvd+shrink"). 

Another option, what I do actually, is to run XP in a VMWare Player session. [Install XP in VMWare Player]("http://www.ubuntuforums.org/showthread.php?t=84275&highlight=vmware") This allows me to mainly deal with my PSP and convert videos for it, but I also use a few programs like DVD Shrink, VirtualDub, and most importantly run my Canon scanner. I then have a partition formatted to vfat so I can share files between the VMWare session and Ubuntu through Samba. I realize that migh be sacriligeous to many Linux zealots out there, but sometimes, Linux just doesn't cut it.
