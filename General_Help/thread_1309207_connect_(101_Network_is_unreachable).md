### Post by t0p on 2009-11-02
If you can browse the web okay, I think the repository server you're trying to connect with must be down.  So you need to use a different server.

In Hardy, you can do this by going to **System > Administration > Software Sources** and clicking on the drop-down menu where it says "Download from..."  It may be different if you're using a different version of Ubuntu.  But the option you need should be somewhere under **System > Administration > Software Sources** so I recommend you have a look through the available options there.
