### Post by VE7BGP on 2011-01-24
Hello fellow Ubuntu users. I am new to Ubuntu although I have used Mint for a while. I installed the Ubuntu Lucid Lynx on my 2 lap tops and the Install went great. On my desktop I had the problem when you go to login you had to use Failsafe Gnome then set your resolution to 800x600 or 832x624 then Gnome would Login. I was looking at the Startup Applications and noticed Bluetooth Manager was checked I do not run Bluetooth Hardware yet so I unchecked it and saved not to run in Bluetooth Startup. The next time I booted Ubuntu I had a clean & fast bootup. I also found the same fix worked for Linux Mint 9 on my other partition. I hope someone else reading this and having that nagging problem after changing many bootloader and grub settings to no avail will find the fix by shutting off Bluetooth Manager in Startup Applications. I hope that helps others
