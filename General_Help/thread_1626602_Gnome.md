### Post by Dans564 on 2010-11-20
It sounds like your just looking for a dock.  My advice would be to stick with gnome and just install one of the many dock apps for gnome.  Docky and avant window navigator are my two favorite.  Both can be installed by copying the following into terminal
```
sudo apt-get install docky
```

or

```
sudo apt-get install avant-window-navigator
```
