if you rename it to something like yourscreensaver.desktop.old it should disappear from the screensavers

unfortunately you cannot uninstall one, they come together in one package.

### Post by gazj on 2008-01-03
when I first started the post I was going to say delete them but thought I had best play it safe.  You can always get them back by reinstalling gnome-screensaver, but you are right it is not elegant.  If you want more control over screensavers I would uninstall gnome-screensaver and install xscreensaver in synaptic.  It has tick boxes to turn screensavers on and off when using random (note it comes with the same screensavers as gnome-screensaver)

You will know how to find things, it just takes time ;) keep hanging in there and it will come.
