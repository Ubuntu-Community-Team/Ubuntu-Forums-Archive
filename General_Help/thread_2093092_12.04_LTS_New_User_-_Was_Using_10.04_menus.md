### Post by ibjsb4 on 2012-12-09
Hi Steve, welcome to the forum  :)

12o4 now uses the Unity desktop.  If you want a desktop that looks like 10o4 you can install Gnome Classic and choose which desktop you want to run at boot (login).

To install Classic, [open a terminal]("https://help.ubuntu.com/community/UsingTheTerminal#Starting_a_Terminal") and enter:

```
sudo apt-get install gnome-session-fallback
```
