### Post by chessnerd on 2010-07-12
It has loaded, but you need to click on the notification icon and check Show Rhythmbox.

I share your frustration, which is why I remove the notification icon plugin so that it no longer has this behavior. Load up the Rhythmbox plugin manager and then de-select the "Status Icon" plugin. That way, Rhythmbox will behave like a normal program.
