### Post by Chris Jones on 2006-09-10
I had the same problem this morning after installing the Kubuntu desktop last night and upgrading. I suspect the X configuration got broken so after logging in to the terminal (text only) I reconfigured the Xserver using:

sudo dpkg-reconfigure  xserver-x.org

You will need to know what graphics chip you are using and the specifications of your monitor if these are not able to be detected automatically.

I hope this is of some help. I don't even know if this is the Ubuntu way as I am new to using Ubuntu.

### Post by Ramses de Norre on 2006-09-10
```
lspci -X
``` will output addresses in the same format as they are used in xorg.conf and the right command to reconfigure xorg is ```
sudo dpkg-reconfigure xserver-xorg
```
