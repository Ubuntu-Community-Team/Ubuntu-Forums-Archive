### Post by callan79 on 2007-08-03
> **-Outcast- said:**
> Is there any GUI program that will batch resize photos?

Hi There,

This was one of the things that I really missed with Ubuntu, until I discovered this:

```
sudo apt-get install nautilus-image-converter
```

And once this is done, select one or more images inside Nautilus (file browser), right click and you can RESIZE and ROTATE. Works really well.

Cheers
Callan
