Boot from a Live Linux CD / DVD - Knoppix preferably and run testdisk (read man testdisk first).

This will analyse your Hard Disk and tell you (if you have not over written the data) what all exists and what can be recovered. Specifically for specific files uses photorec (which i have not used)
