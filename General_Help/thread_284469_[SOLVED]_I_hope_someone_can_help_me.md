Hi Darkside, I'm a pink floyd fan from the early days.  Yes, I'm old, at least that's what the yunguns think.

You certainly have *a lot* of questions.  And you should.  Its not easy making the transition to a new operating system.  I'll try to answer your questions one at a time, but you have to understand that moving away from Windows is not going to be an easy thing.  Unfortunately the world is still Microsoft dependent, so you will have to want it and learn it if you are to get there.  Luckily, the transition doesn't have to be difficult, which brings me to your first question.

You can actually have Windows and Ubuntu running on the same computer.  You can't have them running at the same time, which means that you can decide, when you restart your computer, which operating system you want.  That's a cool feature because you can choose to run Windows some of the time and Ubuntu the rest of the time, and switch back and forth with relative ease, simply by restarting your computer.

As far as transferring files between Windows and Ubuntu, that's absolutely possible, but you will have to do a *little* work to get it done.  In fact, I share my Windows and Ubuntu files and neither system knows the difference.

I don't know the exact amount of space that Ubuntu takes (I'm sure there are others here who would know more about that), but I can tell you that without a doubt, Ubuntu can take *much* less space than a modern Windows system.  As far as speed is concerned, you can practically bet on the fact that Linux (Ubuntu) will run faster than Windows on most systems.  It has to do with the size and "bloat" of Windows.

So, will your older computer be able to run Ubuntu (Linux)?  In fact, I tell all my friends that if they have an older computer which is having problems with Windows or can't upgrade to the latest version of Windows, that they should install Ubuntu (or any version of Linux really).  They've all been happy with the results.

What you also have with Ubuntu, that you really don't with Windows, is a world-wide community of dedicated people, interested in helping new Ubuntu users get over the initial hump.  I hope this helps.  And in the future, you can start a new discussion and a new question by clicking on the "New Thread" button at the top of each forum page.  Be sure to title your question appropriately, and that way others will see the question and be able to answer.  Check out the "Absolute Beginners Forum" where there are many questions and answers which can help you make the transition.  A lot of your questions have probably been asked and answered before.  But no worries, ask any question you want.  You'll find everyone here very helpful.

Good luck.
