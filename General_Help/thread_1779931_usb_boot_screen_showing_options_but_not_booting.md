Hello, sometimes when downloading from a mirror the .ISO file can be corrupted. It needs to be downloaded at the fastest possible speed. A few suggestions for you that might work.

1, Download the ISO from the Ubuntu Download Page
[http://www.ubuntu.com/download/ubuntu/download]("http://www.ubuntu.com/download/ubuntu/download")
If this asks you to select a location to download from choose either "from the main server" or select a location near to you.

2, Burn the ISO to USB or burn it to CD. If burning to CD recommend that it is burned at the slowest possible speed to eliminate any errors in the burn process.

I know that it is possible to install to a USB stick as a LiveCD with persistence because I do that as well.

Hope this helps

If already running Ubuntu on a pc or laptop use the '_Startup Disk Creator_'

In Windows I used 'Isobuster' to look in the ISO and ran the 'usb-creator.exe' to create the LiveCD on a USB stick.

When used to boot you will get 2 options - Try Ubuntu or Install.

Select Try and you should get the same Desktop as from the LiveCD.

This takes a while as my USB stick is not the fastet drive for Disk reads and writes.

Hope I have not confused you too much
