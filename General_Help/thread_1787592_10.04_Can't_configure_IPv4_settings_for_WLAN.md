### Post by dozycat on 2011-06-22
if you I understand right, you are connecting to a radio wireless using a network cable, is ok?
there are two other ways to do it:

1. change the file interfaces
```

 $ more /etc/network/interfaces
# This file describes the network interfaces available on your system
# and how to activate them. For more information, see interfaces(5).

# The loopback network interface
auto lo
iface lo inet loopback

# The primary network interface

auto eth0
#iface eth0 inet dhcp
iface eth0 inet static
address 172.30.0.10
netmask 255.255.0.0
gateway 172.30.0.1


```

2. the command ifconfig:
sudo ifconfig eth0 172.30.0.10 netmask 255.255.0.0 gateway 172.30.0.1
