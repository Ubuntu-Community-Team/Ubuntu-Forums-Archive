### Post by fubadubrub on 2006-08-09
How do I change torrent files to be associated with Azureus.  I installed Azureus through the Add/Remove link in Gnome.  When I click a torrent link in Firefox it does not send the torrent to Azureus.  I want my torrents to be associated with Azureus system wide.

---

### Post by Fass on 2006-08-10
For torrent files in Firefox you'll have to change the default programme for them in the "open with" dialogue, or in "Edit - Preferences - Downloads - Download Actions."

For system wide association, right click a torrent file in Nautilus, Properties, Open With and set a new standard programme to open with.

---

### Post by nalmeth on 2006-08-10
And for files lying around on your computer, just right-click them, go into properties, and change the default application.
