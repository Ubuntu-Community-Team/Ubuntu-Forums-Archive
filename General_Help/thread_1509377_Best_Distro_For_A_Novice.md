### Post by warfacegod on 2010-06-14
Before slapping Linux of some sort onto this laptop, you should give an honest effort into repairing Windows. It's what they're used to using after all. _Ccleaner_, *Defraggler*, and _Glary Utilities_ are all excellent Windows tools and can go a long way to help getting Windows back on its feet.

Considering the specs you posted, I'd say the laptop is quite a bit older than 3 years. I think the suggestion of getting more RAM is a good one. A faster processor and bigger HDD would help as well. While your at it, a new laptop might be a good idea too.

Unfortunately, all of the truly lightweight distros that I'm familiar with, such as Slitaz, tend to have fairly steep learning curves. I think one of these solutions is probably your best bet:

9.04 Minimal install
9.04 or 9.10 Full install w/LXDE from Synaptic (you can set it to run the LXDE session from the login screen)
PCLinuxOS LXDE
Elive (Unfortunately, they want money, if you want to install it to the HDD but you can still run it from a LiveUSB without issues)
