### Post by FluffyElmo on 2006-12-18
GnomeBaker will do this as well if you want a traditional cd-burning interface. When you click "Burn Disk" you can choose to only write an .iso instead of burning it to disk. 

```
sudo apt-get install gnomebaker
```

isomaster does look more targeted at what you want, but variety is always good:)
