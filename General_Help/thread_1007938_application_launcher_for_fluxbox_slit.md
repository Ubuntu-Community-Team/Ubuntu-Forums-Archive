### Post by rawsilk on 2012-10-19
Though this forum is old I just wanted to add my favorite solution. I have gkrellm running in the slit which gives me my stats like cpu usage. I then download and compile the trayicons plugin which puts a little panel (you choose it's width) on the bottom of gkrellm where you can place icons with commands attached to them. 

Trayicons is actually very usful because it also can act like a notifier. You can have the icons change when some command returns true (like a mail check program).
