### Post by detroit/zero on 2008-10-19
How can I change the mount options for an external USB HDD and for flash drives?

I've had problems for a long time with my USB file transfer speeds, specifically the write cycle. 

In all my investigation, I've run across a multitude of solutions that do indeed seem to work for some people, but don't really do crap for me. The one I haven't been able to try is changing the synchronous read/write ability for external disks and flash memory.

How can I change the mount options for flash drives and my external hard disk to asynchronous?

---

### Post by Sam on 2008-10-19
Right click on the icon in the desktop (while the drive is mounted), then "Properties", then "Volume" tab.
