### Post by cjhabs on 2014-02-04
When you drag files between folders in the same partition, the default behavior is to move the file (leaves no trail).
When you drag files between folders on different partitions or disks, the default behavior is to copy the file (leaves a trail).
If you drag using the middle mouse button it will give you option of copying or moving or linking.
This is how it works on my system which has been customized, YMMV.
