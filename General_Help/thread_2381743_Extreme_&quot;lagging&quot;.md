### Post by TheFu on 2018-01-06
There could be 1,000 reasons for this.  What do the log files show?  Any hardware issues?  Is the USB slow v2 or faster v3?  Is the flash drive wear-leveling kicking in, slowing access down?  Flash memory gets slow over time, especially right before it completely dies.

Can you try a newer USB3 flash drive?
Can you try booting from optical media?  Is is still slow?  If not, you've discovered that the issue is with the USB storage, somewhere.
Can you install to an internal disk? Is it faster?

Of course, another option is to run a distro that loads everything into RAM.  TinyCore is one. It should be incredibly fast on any HW less than 20 yrs old.  I use TinyCore for online banking.

I think the problem is most likely the USB2 flash drive.
