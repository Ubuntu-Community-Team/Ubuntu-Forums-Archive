### Post by 50words on 2007-06-04
In MS Word, I can protect a document so that a user can only enter text into fields I have defined. This is particularly useful for things like client intake sheets.

However, I can't seem to do the same thing in OOo. Does anyone know how I might be able to do this (or something like it) in OOo?

---

### Post by steeleyuk on 2007-06-04
OOo does it slightly different to Word. Hopefully this works as you want:

Click the 'Tools' menu, select 'Options' and click on 'Security'. 

Then click 'Protect' and you'll be prompted to enter a password. At this point any text you enter will be red and the changes will be recorded. If you want it so that text can only be entered in fields, you'll want to select 'Open this document in Read-only mode'.

Once you close and then re-open the document you'll be in Read-only mode with a cut-down interface. To go back to editing mode, you need to click 'Edit File' which is the 5th button from the left on the Standard toolbar.

Hope that helps

Edit: Once you place the document in Read-only mode, you can no longer save the document with text entered in fields. It might be better to leave that option deselected and select 'Record Changes'
