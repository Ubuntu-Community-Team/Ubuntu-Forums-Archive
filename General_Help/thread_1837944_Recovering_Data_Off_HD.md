### Post by tgalati4 on 2011-09-02
If you open Nautilus (the file manager) and click on Network on the left side, it should search the local network for any visible file shares.  Click on any that you recognize and see if you can log into the file server directly.  Otherwise, find another Windows machine and browse the Network Neighborhood for the file share and connect to it and grab your files and save them to a thumb drive.
