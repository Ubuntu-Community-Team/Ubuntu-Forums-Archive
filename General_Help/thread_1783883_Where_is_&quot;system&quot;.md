### Post by aysiu on 2011-06-16
If you're using Unity, you don't even have to go to System. Just press the Windows key (or click on the Ubuntu logo and start typing whatever you want.

For example, if instructions say > Click on System > Preferences > Keyboard Shortcuts You should click on the Ubuntu logo and just start typing > keybo and the Keyboard Shortcuts item will appear.
