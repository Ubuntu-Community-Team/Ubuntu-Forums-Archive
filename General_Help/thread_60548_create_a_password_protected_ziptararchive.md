### Post by the4thchild on 2007-12-19
This is surely an old thread, but I figured I'd update it anyway since I stumbled upon in on Google.  In GNOME the archive manager (File Roller) has a password creation feature for encryption.  After creating a new archive, choose Edit > Password... .  Any new files added will be encrypted with the chosen password.  Note however that files added prior to the password creation will not be encrypted.  A little lock symbol shows which files have been password protected.
