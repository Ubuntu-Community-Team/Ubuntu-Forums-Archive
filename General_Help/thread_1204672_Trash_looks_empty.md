### Post by diablo75 on 2009-07-05
Also, I'm pretty sure each account/user (including the root/sudo user) has their own individual trash folder.  For example, if you ran nautilus as root, and then deleted something, it wouldn't go into your own trash but the root's trash, which would further require the root user to empty it as no other user would be permitted to do so by default.
