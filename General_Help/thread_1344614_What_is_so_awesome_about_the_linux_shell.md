### Post by jegerjensen on 2009-12-03
Well, I don't know how far the Windows shell has come lately, but the killer features of Bash (the default shell on ubuntu) are things like

[LIST]
[*]autocompletion of files, commands, command options, ++
[*]incremental history search (hit Ctrl-r and start typing, Ctrl-r again to search backwards through history)
[*]The vast number of command line utilities.  For Windows most utilities are point and click, with no command line interface.  For linux, on the other hand, a functional command line interface is the norm, even for apps that are primarily gui.
[/LIST]

imho at least :-P
