### Post by wmcbrine on 2010-01-26
If you can pull up the PDF in Evince, it has an option under the File menu, "Save a Copy". I believe Acrobat Reader has something similar. There's probably a way to disable the menu option in Acrobat, but I doubt if Evince would honor the prohibition. (?)

Edit: OK, I checked out the URL. That's not actually a PDF.

In Firefox, go to Tools -> Page Info -> Media, and you'll find the page images towards the bottom of the list. Which may or may not be the easiest way to grab them, but, it's a way.

Edit 2: Of course that only gets you graphics, and you wanted text. While Google clearly has a copy of the pages' text, which it uses to highlight search results, I don't think it exposes that text to visitors. But perhaps you could OCR it from the images?
