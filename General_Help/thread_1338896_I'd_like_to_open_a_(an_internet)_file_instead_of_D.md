### Post by Dennis N on 2009-11-27
In Firefox at least:
Left click on the link to the file.
You should see two choices:

1) Open with:
2) Save to:

Using (1), use the default or use the dropdown box to select another application. If it is not shown, you can browse to an application you want to use by using 'Other' in the select application dropdown list.
