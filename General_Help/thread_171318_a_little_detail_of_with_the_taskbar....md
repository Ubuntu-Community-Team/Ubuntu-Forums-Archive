### Post by orlox on 2006-05-06
I've been using gnome and xcfe4 for some time now, but i'm trying to switch to kde now. However, there's this little detail that annoys me. Is it possible to see in the taskbar only the apps that belong to the desktop you're currently seeing? I truly dislike to see it full with the apps of all the desktops, and I haven't been able to find a solution for this...

---

### Post by ajgreeny on 2006-05-06
Yes in KDE if you right click in the taskbar and then "Configure panel" then go to the Taskbar icon and unselect the option to "Show windows from all desktops".  All done!
