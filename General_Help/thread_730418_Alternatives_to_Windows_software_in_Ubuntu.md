### Post by forrestcupp on 2008-03-20
You could easily dual-boot with Vista.  I do that.  But that requires shutting down your computer and rebooting to Vista whenever you need to use it.  

If you want Vista in a virtual machine like vmware, you will actually have to reinstall Vista on a "virtual hard drive."  If you have an OEM version of Vista, it probably won't allow you to do that because it uses "virtual hardware" which shows up in the authentication as different than what you originally authenticated.  If that is the case, a dual-boot system is your best bet.  If you have a retail version of Vista, you may be able to reinstall in a virtual machine without trouble.

There should be good solutions to everything you mentioned other than CS3, though.
