### Post by michaelayland on 2010-03-14
A partial answer I have found is Windows software called V-CamShow and is provided free by Virdi Software (A google search found it).  It runs well in Wine on Ubuntu 9.10 but it is a bit buggy on installing webcams but a restart fixes it then once it is running it seems very stable.  Find a nice webcam like [www.ayland.eu/webcam.jpg](www.ayland.eu/webcam.jpg) add it to the list,  run the slide show and the software cascades though your chosen webcams.  It would be nicer in pure linux but that is what Wine is for...enjoy
