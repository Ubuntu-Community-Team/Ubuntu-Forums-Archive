Since it seems that yelp already has access to all the man pages, is there any way to call up the man page in yelp from the command line?

### Post by szaby on 2009-10-25
To open man pages in yelp, type:

```
[FONT=Verdana]yelp man:command(1)[/FONT]
```Numbers in parentheses are needed only, when there are multiple man pages with the same name.

*Source*: [http://linux.about.com/library/gnome/blgnome7n3d.htm](http://linux.about.com/library/gnome/blgnome7n3d.htm)
