### Post by kesman on 2008-04-30
There's a setting in **/server** that you can send a command to the server at connect, I think it was something like: /**server -autosendcmd** or similar.
**/help server** will give you all the options you need. Note that auth is used in Qnet only, I don't know about other servers' stuff.
