### Post by Impavidus on 2017-01-10
> **matteoraggi said:**
> 
I tryed with mega, but it has .deb only for ubuntu and not lubuntu, hubic and google drive don't have the .deb too for lubuntu.


Ubuntu and Lubuntu and any other debian-based Linux use the same kind of .deb files, although different and not necessarily compatible versions. Ubuntu and Lubuntu even use the same repositories, giving them exactly the same .deb files. The difference is mostly the set of default software. This means that a .deb file for Ubuntu should work on Lubuntu too. Sometimes it may draw in a lot of dependencies, but that's mostly a problem when mixing Ubuntu and Kubuntu. But even that works.
