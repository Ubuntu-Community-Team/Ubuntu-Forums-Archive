### Post by taurus on 2007-02-07
You can download a .deb version of webmin from [http://www.webmin.com](http://www.webmin.com).  And if you want to have some nice GUI to configure your stuff, then perhaps you should think about install Gnome window manager.

```
sudo aptitude update
sudo aptitude install ubuntu-desktop
```
Otherwise, you have to do everything from a console.
