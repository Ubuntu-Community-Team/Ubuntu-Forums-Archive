### Post by wislon on 2007-06-22
VMWare offer a package called something like P2V that is free that you can use to create a virtual machine off an existing physical one (I've only tried it with XP Pro SP2). It kind of copies it into a set of VM files that their products can load. Check out vmware.com, they have loads of stuff there, some of it free (or free for non-commercial use),
