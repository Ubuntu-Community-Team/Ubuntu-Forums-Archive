### Post by Chronon on 2010-01-07
Yes, Adept is a graphical package manager that provides a list of available and installed packages.

Did you go to "K > Applications > System" and look for a package manager there?  You should find Adept, KPackagekit or both.  If Adept isn't located there then you should be able to use KPackagekit to install it.
