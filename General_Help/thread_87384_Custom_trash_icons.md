### Post by Xian on 2005-11-07
Just goto the icon theme you are using (you'll have to locate that according to what you are using locally), and replace the images already present with those that you would like to make visible. For example:

~/.icons/Human/scalable/filesystems/gnome-fs-trash-empty.svg
