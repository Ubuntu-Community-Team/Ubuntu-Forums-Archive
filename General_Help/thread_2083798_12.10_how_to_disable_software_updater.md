I update my machine through the command line with apt-get, because it's faster and far less annoying than through the gui. ### Post by audiomick on 2012-11-13
In the settings of the Software updater, on one of the tabs in there look for a drop down that is called something like "look for updates of". I am translating out of German, so I don't know exactly what it is called in English. It offers the option "never". That might sort you out, I think.

### Post by Frogs Hair on 2012-11-13
The update manager is a GUI front end for the package system. It will show package information even when the terminal is used. The package name I think you want to know is update-notifier-common. I have no idea of the consequences if it was to be removed.
