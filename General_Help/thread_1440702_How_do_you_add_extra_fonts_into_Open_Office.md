### Post by mechro on 2010-03-27
I normally install **ubuntu-restricted-extras** which includes Times New Roman, but it also contains flashplugin and gstreamer stuff.

If you just want fonts you could install **ttf-mscorefonts-installer**.

You can find these packages in Synaptic Package Manager and you'll probably find some more font packages by searching in Synaptic.

### Post by drewsus on 2010-03-27
In addition to getting those common fonts, if you ever want to add new true type fonts to the system they go into the **.fonts** (yes, with the period, that means its hidden -> press ctrl+h to see/hide hidden folders) in your home directory. So, add the **.ttf* file into there and restart any app you want the fonts to appear in if it was already open
There are so many sites out there offering 1000s of fantastic free fonts
