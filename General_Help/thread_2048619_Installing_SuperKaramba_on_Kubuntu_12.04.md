### Post by amir786_z on 2012-08-28
I somehow managed to install it on kubuntu 12.04..

Steps 

1) download deb package from here [http://pkgs.org/ubuntu-12.04/kubuntu-backports-i386/plasma-scriptengine-superkaramba_4.9.0-0ubuntu1~precise1~ppa1_i386.deb.html](http://pkgs.org/ubuntu-12.04/kubuntu-backports-i386/plasma-scriptengine-superkaramba_4.9.0-0ubuntu1~precise1~ppa1_i386.deb.html)

2) open terminal type "cd home/amir/downloads/
now ur in download folder

3) now run the command sudo dpkg -i plasma-scriptengine-superkaramba_4.9.0-0ubuntu1~precise1~ppa1_i386.deb

some error appeared but working fine ;)
