### Post by FAMUgolfer on 2008-12-17
Is your question how can you get cairo-dock to start up automatically everytime you boot without having to do it manually? If so try this:

System-->Preferences-->Sessions--->Add-->type cairo-dock under Name & Command section (make sure it is in lower case)--> Save

Hope this helps!!
