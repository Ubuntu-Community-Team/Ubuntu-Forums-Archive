### Post by scandido on 2008-12-19
If you are interested in displaying the current time in multiple locations, that functionality is built in. Click the time to get the "calendar" drop down. At the bottom left it says "Locations" and at the bottom right there is an "Edit" button. Click the edit button and add cities.

There's also the "date" command if you don't mind the command line.
