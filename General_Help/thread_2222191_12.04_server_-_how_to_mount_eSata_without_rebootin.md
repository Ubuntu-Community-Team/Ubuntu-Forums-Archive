### Post by steeldriver on 2014-05-08
What have you tried so far? I don't have any experience with e-Sata, but I would expect that udev should try to detect it - do you see a new block device?

```

sudo blkid -c /dev/null

sudo fdisk -l

sudo parted -l

```

If not, are there any relevant messages if you watch the syslog when you plug/unplug the device?

```

tail -f /var/log/syslog

```
