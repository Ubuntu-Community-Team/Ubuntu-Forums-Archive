Hello Everybody,
 
I'm not sure if this is the right place to make this post or not, but it does follow-up on several previous "help needed" posts in the General Help section. If this is the wrong place, please feel free to move it - my apologies! ;) 
 
Anyway, [EasyBCD 1.51]("http://neosmart.net/blog/archives/304") went gold yesterday, it finally supports almost any Linux/BSD system you throw at it. For those of you that haven't tried it before, EasyBCD is dual-boot manager program for Windows Vista's bootloader. You can use it to dual-boot Linux/BSD/OS X from the Windows Vista bootloader without any coding or configuration.
 
Many thanks to **Herman** for his in-depth explanations and testing of the GRUB bootloader with Ubuntu, I'd probably still be struggling with the testing if it wasn't for him.
