### Post by Ramses de Norre on 2006-12-11
Right click > save page as will give you an html document, so perfect quality.
If you need an image format: ```
scrot -q 100 -s
``` and select the browser (you may need to install scrot). Scrot is to take screenshots and -s lets you select a window to take a shot of -q is the quality given by a number between 1 and 100.
