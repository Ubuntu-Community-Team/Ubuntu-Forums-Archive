Ubuntu 16.04, Twin HPw2228h monitors.

Twin monitor set up works fine for me as user. For a second user account - in  Display Settings:

[INDENT]Can see the 2nd monitor as an option but cannot enable it. 
The settings app display keeps reverting back to a single display.[/INDENT]

---

### Post by hundred1906 on 2018-01-17
Found that the Guest user and any other new users I set up had full access to both display.

So deleted the problem user account, but chose the option to retain existing storage for that user. On re-instating the user (same name etc) the problem was still present.

Eventually - entirely deleted the account and storage and re-instated. That worked!
