### Post by holycow131415 on 2012-01-12
Download virtualbox from [www.virtualbox.org]("http://ubuntuforums.org/www.virtualbox.org") so that usb support works. After that, go back to the website to find VirtualBox 4.1.8 Oracle VM VirtualBox Extension Pack to install it. Now you are set up to install windows in a virtual machine so that you can install itunes / whatever other software you need from windows.
