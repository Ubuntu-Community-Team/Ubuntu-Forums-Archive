### Post by NICU on 2007-04-05
Hey, I don't have any experience with isobuster, but I would suggest using any standard burning utility and burning the ISO as an image.  I know that all standard Ubuntu CD burning utilities offer this feature, and all windows burning applications I've used offer it too, just takes some looking around.  If you just burn the individual files (instead of an image) the CD will not be bootable, and if it is set to be bootable it will probably not boot correctly.  So just burn the iso as an image and you shouldn't have any problem.
