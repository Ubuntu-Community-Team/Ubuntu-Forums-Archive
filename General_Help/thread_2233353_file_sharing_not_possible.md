### Post by sudodus on 2014-07-15
It is possible to install and tweak samba, but it is a bit tricky. I hope someone else will chip in and help you with samba (I run it but I'm not good at it).

I find it easier to install the open-ssh server. Then you can use ssh, sftp, filezilla even the file browsers from other linux computers, and from Windows winscp and filezilla. See this link

[https://help.ubuntu.com/community/SSH/OpenSSH/Configuring](https://help.ubuntu.com/community/SSH/OpenSSH/Configuring)
