### Post by LemursDontExist on 2011-07-29
On linux you really don't need antivirus software.  Linux is generally much more resistant to viruses the Windows, and the few known viruses mostly don't exist in the wild, having been rendered inoperative by security updates.  

For the most part, when you hear about antivirus on Linux people are talking about antivirus software on a mail server which scans emails for *Windows* viruses.

For more details, check out the [wiki]("http://en.wikipedia.org/wiki/Linux_malware") page.

If you *are* running a mail server, and that's what you want antivirus for, I've heard good things about [ClamAV]("http://www.clamav.net/"), but can't speak from experience.
