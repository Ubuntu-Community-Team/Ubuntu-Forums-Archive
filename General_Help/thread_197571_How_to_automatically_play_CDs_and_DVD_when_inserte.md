### Post by arizonagroovejet on 2006-06-16
Try this -

System Settings > Storage Media > Notifications

If you want to insert an Audio CD and have Kaffeine automatically launch and start playing it then select 'Audio CD' from the drop down menu then select 'Play Audio CD with Kaffeine' from the list of options and click the 'Toggle as Auto Action' button.
