### Post by 3rdalbum on 2009-05-27
Yes. Go to System > Administration > Software Sources and under the Update tab, change the settings so that it will show distribution updates for "all releases" (not just Long Term Service releases).

Now run the Update Manager as normal and you will see an option to go to 8.10 (or maybe to 9.04?). You might need to reload the package list or wait a couple of minutes for the option to appear.

If it wants you to go to 8.10, you will need to do that upgrade before being able to upgrade to 9.04.
