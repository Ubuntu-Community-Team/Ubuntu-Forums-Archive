### Post by davidmohammed on 2010-07-28
personally - no I love linux - I've ditched windows long ago!

But for newbies - I would recommend running ubuntu in one of two ways:
1. install UNR onto a USB stick. You can install the programs you want and run as if installed on your netbook.
2. install ubuntu as a wubi install directly into windows.  Install whatever you want to give ubuntu a good thrashing...

when you are convinced - wipe your hard-drive and welcome to a new exciting world!
