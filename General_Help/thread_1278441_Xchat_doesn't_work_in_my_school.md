### Post by ibuclaw on 2009-09-29
Try port 8001, it is more secure.
If that fails, it's probably the proxy server that is blocking outbound access through those ports.

If that is the case, all I can suggest is that you try a web-based irc client.
ie: Mibbit, Chatzilla.

You shouldn't need to resort to proxy sites, ever. And if you are, you probably shouldn't be doing what you are trying to do in the first place.
