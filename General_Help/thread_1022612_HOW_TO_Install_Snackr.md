### Post by SonnHalter on 2008-12-27
Short guide to installing snackr a popular Rss ticker


**[SIZE="7"]step one[/SIZE]**
[SIZE="5"]getting adobe air[/SIZE]




go to: [http://get.adobe.com/air/](http://get.adobe.com/air/)
[IMG]http://i40.tinypic.com/2jfejv4.png[/IMG]



The file will now be downloaded and you can drag it to your desktop. It should be a .bin file

[B][SIZE="5"]
Edit the permissions[/SIZE][/B]

change the permissions to "execute as program" to do this right click the file and go to "properties" and then "permissions"
[IMG]http://i43.tinypic.com/b5mlq9.png[/IMG]



[SIZE="5"][B]
Installing AIR

[/B][/SIZE]

your file should now look gears. Now to run the installer.

open the terminal and then drag the file into the terminal.
[IMG]http://i42.tinypic.com/2q0u136.png[/IMG]

**[SIZE="4"]WAIT FOR IT!!![/SIZE]**

you will get a gtk error but the installer will run. go through it and you'll have air installed.

**[SIZE="6"]installing snackr[/SIZE]**

go to:[http://snackr.net/](http://snackr.net/)


Then, click the "snackr installer" the download should open with "adobe air installer" 

run through it and you'll now have snackr!!!

Alternatively, go to the folder where the *Adobe Air* bin installer file is, e.g. your Desktop, like this
```
cd Desktop
```
and then give
```
./AdobeAIRInstaller.bin
```
to run it.
