### Post by 3rdalbum on 2009-07-23
I'm not sure what you're asking, but I'll try to answer.

If you are running a fully-installed Ubuntu from a USB flash drive, then any programs you install will install to your flash drive. Ubuntu doesn't store anything on any other disks.

There are no "drive letters" in Linux too, so when you're running Ubuntu there's no such thing as a "C drive" or a "K:\" - they are probably /dev/hda or /dev/sdf.
