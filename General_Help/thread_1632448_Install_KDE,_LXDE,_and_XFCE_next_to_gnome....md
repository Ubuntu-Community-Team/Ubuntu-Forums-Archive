### Post by cascade9 on 2010-11-28
Xfce- 

apt-get install xfce4    OR  apt-get isntall xubuntu-desktop 

Lxde- 

apt-get install lxde   OR   apt-get install lubuntu-core

KDE- 

apt-get install kde-full    OR     apt-get install kubuntu-desktop. 

Adding xubuntu-desktop will add a lot more software than just xfce4. It will also probably be slower. 

[http://packages.ubuntu.com/maverick/xfce4](http://packages.ubuntu.com/maverick/xfce4)

[http://packages.ubuntu.com/maverick/xubuntu-desktop](http://packages.ubuntu.com/maverick/xubuntu-desktop)

Adding kde-full will also add a more software, but nowhere near as much as kubuntu-desktop. 

[http://packages.ubuntu.com/maverick/kde-full](http://packages.ubuntu.com/maverick/kde-full)

[http://packages.ubuntu.com/maverick/kubuntu-desktop](http://packages.ubuntu.com/maverick/kubuntu-desktop)
