### Post by taurus on 2007-03-07
Where did you mount your /etc/sdb1 to?  Maybe when you delete it, it goes into a trash bin and you need to empty the trash bin to actually free up the space.  Assuming it is mounted to /media/usbdrive, post the output of this command from a terminal.

```
du -h /media/usbdrive
```

p.s.  If you don't know where it is mounted, run this command and look at the output for /dev/sdb1.

```
df -h
```
