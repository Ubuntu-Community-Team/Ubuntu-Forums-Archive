### Post by Scunizi on 2010-05-02
There really isn't a "correct" version.. If you do a lot of video ripping, transcoding or other intensive system hogging type stuff then 64 bit will be faster if your machine is capable of it.  I converted to 64 "just because" I wanted to see how far it had come.  I occationally do video stuff, audio and also use my machine for an in-house FTP and web server so I think the 64bit version helps some.  The only place that I have found it lacking is with Citrix.  That's a remote web app I need once in a while for work.  Citrix only has a 32 bit version.
