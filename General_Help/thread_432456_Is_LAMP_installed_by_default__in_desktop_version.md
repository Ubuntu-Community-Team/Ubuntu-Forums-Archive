### Post by mcduck on 2007-05-04
If he's using Feisty, the easiest way to install LAMP is to open Synaptic, and then select Edit/Mark Packages by Task, and from the window that pops up select 'LAMP server' and then click the 'Apply'-button in Synaptic to start installing.
