```
GRUB_TIMEOUT=1
```

```
gksudo gedit /etc/default/grub
```

```
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash vga=795"
GRUB_CMDLINE_LINUX=""
```
save and close the file
then issue this command:
```
sudo update-grub
```

### Post by Leppie on 2010-01-08
open the 00_header file:
```
gksudo gedit /etc/grub.d/00_header
```
comment out the last if statement, except for the line with the normal set timeout:
```
#if [ \${recordfail} = 1 ]; then
#  set timeout=-1
#else
  set timeout=${GRUB_TIMEOUT}
#fi
```

run update-grub again:
```
sudo update-grub
```

this should force grub to *always* use a timeout of 1 second.
