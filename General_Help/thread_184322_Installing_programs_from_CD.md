### Post by Nonno Bassotto on 2006-05-29
Firefox should be already installed. Anyway you can install what you want from the network, when it works. For ndiswrapper you can download the installation file from here
[http://packages.ubuntu.com/dapper/misc/ndiswrapper-utils](http://packages.ubuntu.com/dapper/misc/ndiswrapper-utils)
and then double click to get it installed. You may also want a graphical front-end to it, you can find it here
[http://packages.ubuntu.com/dapper/net/ndisgtk](http://packages.ubuntu.com/dapper/net/ndisgtk)
