### Post by koriordan on 2008-12-27
You could try using the QGTKStyle theme for QT 4.4. It can be found here: [http://labs.trolltech.com/page/Projects/Styles/GtkStyle](http://labs.trolltech.com/page/Projects/Styles/GtkStyle) . From what I've heard, it will be packaged by default with QT 4.5 (Intrepid Ibex comes with 4.4).
