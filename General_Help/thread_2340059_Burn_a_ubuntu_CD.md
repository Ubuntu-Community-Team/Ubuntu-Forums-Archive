### Post by howefield on 2016-10-15
You really don't want either of the "15" releases as both 15.04 and 15.10 are beyond end of life and therefore no longer supported. You'd be better off with the current Long Term Support release which is 16.04.1 and supported till 2021.

The link below describes how to create the Live DVD (these days it does need to be a DVD as the iso will be too large to fit on a CD - there is also the option to create a Live media with a USB stick).

[https://www.ubuntu.com/download/desktop/burn-a-dvd-on-ubuntu](https://www.ubuntu.com/download/desktop/burn-a-dvd-on-ubuntu)

### Post by oldfred on 2016-10-15
If system is that old, you may not be able to run standard Ubuntu. New Unity needs a somewhat newer system with good graphics processor.
My laptop from 2006 would run Ubuntu but Unity was so slow as to be unusable. So I installed fallback which is similar to gnome version. Also Lubuntu & Xubuntu are lighter weight.
