### Post by RD1 on 2009-02-10
> What do I need to install to enable right click>run command from the desktop? For that matter, is there a "run command" in GNOME?

You can add the "Run Application" applet to the panel.

> Also, how can I get Nautilus to open a terminal?

Install "Ubuntu Tweak" ([http://ubuntu-tweak.com/](http://ubuntu-tweak.com/)) Under the "System - Nautilus" section, check the box for "Nautilus With Open Terminal" under "Extensions"
