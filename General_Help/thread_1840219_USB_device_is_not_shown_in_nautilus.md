### Post by Grenage on 2011-09-13
Looking at this entry:

```
[168362.055504] FAT: Filesystem error (dev sdb1)
[168362.055508]     fat_free_clusters: deleting FAT entry beyond EOF
[168362.055511] FAT: Filesystem has been set read-only
```

It's possible that the drive is not being automatically mounted because of this error.  It's probably going to be quickest and easiest to make a copy of your USB data, then repartition and format it.  You could use either gparted or fdisk/terminal for that.
