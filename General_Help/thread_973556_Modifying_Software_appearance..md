### Post by geoffjay on 2008-11-07
Assuming that you're running Gnome, you could probably change the size of the icon by replacing it with a bigger one. You'd just need to know the name of your icon theme and find it in either "/usr/share/icons/" or "~/.icons", I think the default for an Ubuntu setup is "Human/22x22/places/start-here.png". I could be wrong about the size though.

And I'm not positive, but I don't think the animated menu icon is available for any window manager or program launcher.
