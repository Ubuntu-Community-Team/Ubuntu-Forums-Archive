### Post by torgrot on 2008-01-30
Try going into the BIOS and selecting Legacy support for the keyboard and mouse.  USB has become so ubiquitous that even some laptops use it for the keyboard.  Mine often doesn't initialize the first time after I power it up.  The Legacy Support seems to help some times.
