### Post by tgalati4 on 2007-11-29
Bump up the RAM to 32 MB or 64 MB (you will have to scrounge or purchase old memory sticks) then you can run Damn Small Linux with a graphical desktop.  Otherwise, you have a curiosity.

It's helpful to find a working CD drive, as most lightweight distros can boot and run off of the CD ROM using a floppy to bootstrap the system.  Think of it as a 700 MB disk drive!
