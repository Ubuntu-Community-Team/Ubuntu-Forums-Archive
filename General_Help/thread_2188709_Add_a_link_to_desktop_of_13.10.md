### Post by 3rdalbum on 2013-11-19
> **Beetle Juice said:**
> Instead of opening my browser, can I put a link to my email account on my desktop? Would be so much faster!

```
#!/bin/bash
firefox "http://www.hotmail.com/"

```

Put that into a file on your desktop, right-click the file and set it to have execute permission.

Done!
