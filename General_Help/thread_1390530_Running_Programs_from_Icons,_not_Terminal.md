### Post by chome4 on 2010-01-25
How do I set up the ability to place my acidrip program in my panel? I can run it from Terminal but the icon wasn't placed under 'Applications'.

---

### Post by Techsnap on 2010-01-25
Right click the Menu, Click Edit Menus, Click New Item, Type the Name as the Name of the program and Type the command as the command you were entering into the command line to launch the application. You can set icons and things too.
