### Post by Vaphell on 2014-06-13
technically speaking Nautilus is Gnome's file manager. Canonical is about to write its own, given the endless string of unilateral, questionable design decisions made by the gnome people.

right click in icon mode works (there is always blank space between icons, though i agree it's ridiculous that you lose the ability when the list mode is crowded). That said, 'new dir' is ctrl+shift+n in nautilus, can't get any simpler than that. But then you are back to square one if you want 'new document'... sigh.

as for rename - i'd try F2. It's a rather universal shortcut for rename and it might work.
