### Post by meierfra. on 2008-06-14
> Is there any advantage of grub over vista bootmanager or do they both do the same job and shouldn't I bother to install vista's bootmanager back?

They do the same thing. As long as grub is working for you, just keep it.

The only difference:

Suppose Ubuntu gets corrupted for whatever reason, then you would have been better off using the Visa bootmanager.

Suppose Vista gets corrupted for whatever reason, then you are  better  off using Grub.

### Post by meierfra. on 2008-06-14
> There are dozens of posts from people with boot problems.

But I don't recall any which have been caused by using the Vista boot manager. The Vista boot manager just calls Grub to boot Ubuntu. (And Grub  just calls on the Vista boot manager  to boot Vista.)

So I really don't think using  the Vista boot manager will cause any problems.

But  I still vote for  keeping Grub.
