### Post by _H3MLOCK on 2010-01-20
Ok, what you want to do is going to require a bit more than point and click customization that youre looking for. You can build custom themes and configure your window manager using the respective menu (gconf-editor) but programming is something quite different. Since your on linux, I would recommend toying around with bash scripts a little bit to start with (it also gets you familiar with the command line) and then move onto python since you don't have to mess around with compilers... and then C/C++  if you want more power. If you descibe what you want to do in more detail, then maybe we can help more.

NOTE: Theme design and window decoration (custom) require no programming.
