### Post by colorlessprism on 2010-05-02
the easiest way to fix this is to run this [script]("http://bazaar.launchpad.net/%7Eoldman/%2Bjunk/launchpad-update/download/head%3A/launchpadupdate-20090603123623-qo8gcw2k08v3dib8-1/launchpad-update")  and save it somewhere like ~/bin  (/home/tb64/bin on my system). make it executable by  right clicking the file in nautilus and go to properties, then select  the Permissions tab and enable &#8216;Allow executing file as a program&#8217;. now click it and it should run for you. after that open a terminal and type "sudo apt-get update"

### Post by colorlessprism on 2010-05-02
at least this is a new one, before it was a ppa now its mediabuntu try this:
[FONT=Verdana][FONT=monospace]
sudo apt-get update && sudo apt-get install medibuntu-keyring && sudo apt-get update[/FONT][/FONT]

and if that doesnt work there is always this:
Download the gpg file at [http://packages.medibuntu.org/medibuntu-key.gpg](http://packages.medibuntu.org/medibuntu-key.gpg)  to your desktop or some local folder.
Go to *System -> Administration -> Software Sources*. in the  Authentication tab hit the "Import Key File.." button and locate the gpg  file. Close.
