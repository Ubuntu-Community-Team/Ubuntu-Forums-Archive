### Post by Adrian on 2006-01-01
[QUOTE=iand675@gmail.com]I'm installing KDE, and it wants to know if I want to use gdm or kdm. WIll I lose gnome some/all functionality if i install kdm?[/QUOTE]

* When using gdm, you can't shut down/restart your machine from the K menu. You have to log out and shut down from the gdm login screen.

* When using kdm, you will not be able to shut down/restart your machine from the Gnome menu.

This is the only "loss of functionality" that I know of.
