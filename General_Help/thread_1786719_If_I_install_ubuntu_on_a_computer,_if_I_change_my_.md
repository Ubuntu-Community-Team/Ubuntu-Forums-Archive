### Post by Mark Phelps on 2011-06-20
A simple way to do what you want is the following:
1) Download and install the FREE version of Macrium Reflect (MR)
2) Use the MR option to create and burn a Boot CD
3) Image off the Vista install to an external drive or USB stick
4) Set both of these aside for later.

This way, if she DOES want Vista back in the future, you boot from the CD, connect the drive/USB stick and use MR to restore the Vista setup.

A couple of things:

1 - If its an OEM computer (HP, Dell, toshiba, etc.) if you click on your Start button and type in Recovery - there should be an <OEM NAME> Recovery CD Creator. Like on my Gateway it's Gateway Recovery CD/DVD and it allowed me to burn recovery discs of my Windows 7. My moms is Acer Recovery (granted Acer and Gateway are the same (Gateway subsidiary of Acer)).

2 - You can install Ubuntu very easily by downloading it, burning it to a CD or ripping it to a USB drive (instructions are on Ubuntu.com) and the installing it via Wubi (your best option). 

The reason this is the best option, at least for your case, is it creates just a file on the Windows side and you select to boot into Vista or Ubuntu when it comes up. That way if something doesn't work or you ever need to remove Ubuntu you can do it through Programs and Features in the control panel.

It won't be as fast as installing it natively, but it will still work well and let you get hands on with Ubuntu. Then later on you can decide to dual-boot or install it purely =D.

### Post by dFlyer on 2011-06-20
Even if you are planning on dual booting, I would back up all important data from windows in case you decide to reinstall windows.
