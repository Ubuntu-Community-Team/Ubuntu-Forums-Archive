### Post by IWantFroyo on 2011-01-26
SD cards are supposed to be a drag-and-drop thing on most OS's. Are you sure it's registering your SD card? Are you sure your SD card isn't broken? If yes to both of these, search the HOWTO's. Command line IS sort of inevitable for some things, so don't just throw tutorials containing them away.
EDIT- Win32 Disk Imager isn't the best one by far. I don't trust it at all on my XP computer. Get Unetbootin or another 3rd party program.
