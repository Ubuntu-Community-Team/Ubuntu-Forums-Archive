### Post by uRock on 2010-12-03
Boot the Ubuntu LiveCD, then go into the menus to System> Administration> Gparted or Partition Editor, whichever is listed, then delete the partitions you do not want and create the NTFS partition that you do want.

After that, you should be able to install Windows 7 without issue.
