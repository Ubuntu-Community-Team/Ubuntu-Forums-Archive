### Post by mssever on 2007-01-21
Before we can answer that question, we need to know what program(s) are eating up your CPU and RAM. There are many ways to find this out, but I like top (from the command line) and gnome-system-monitor (GUI). If you use gnome-system-monitor, be sure to configure it to show all processes.

If you sort by CPU usage, then you can see what's eating your CPU. If you sort by memory, you can see what's eating RAM.
