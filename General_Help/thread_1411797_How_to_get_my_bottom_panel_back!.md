### Post by mcduck on 2010-02-20
Right-click on soem empty spot in the panel you still have, and select "New Panel". The panel will probably appear in the bottom automatically, but if not then right-click it and select "Properties" and set the correct position there.

After that just right-click the new panel, select "Add to Panel" and drag&drop all the applets you want to the panel. (If I remember right the default setup has "Show Desktop", "Window List", "Workspace Switcher" and "Trash" applets in the botom panel.
