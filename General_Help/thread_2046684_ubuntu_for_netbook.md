### Post by Lars Noodén on 2012-08-23
The lightest weight distro is Lubuntu which uses the LXDE desktop environment.  You can add in the other packages you want.  If you want lighter, you can dispose of the Desktop completely and run a plain window manager like Openbox, fluxbox, or fvwm.  Try a few out and see which is best for you.
