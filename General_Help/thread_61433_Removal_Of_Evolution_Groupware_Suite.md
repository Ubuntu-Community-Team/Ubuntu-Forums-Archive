### Post by ad1138 on 2005-08-31
That's an interesting question that I think many new users are afraid to ask. There are many 'suspect' dependencies in many distros, small programs that - when removed - seem to carry a lot of important stuff away with them. However, I never had any problem in that situation: if I don't need a program I just remove it (unless I'm sure that other programs need it). I remember when I tried to remove a program from Debian Sarge and it said that the whole gnome desktop environment needed to be remove along with it. I was scared, but I did it anyway and I had no problems at all. Probably it was just trying to tell me that **a small part of** gnome desktop environment was about to be removed. Maybe it would be easier if there were simpler messages and descriptions to tell the user exactly how what he's going to do will affect the system (without scaring him). ;)
