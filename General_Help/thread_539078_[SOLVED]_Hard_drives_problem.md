```-merlin
### Post by rsambuca on 2007-08-31
I hope you don't mind if I stick my nose in here.  My advice is:

1.  Yes, you need to keep the 1 at the end, ie sdb1

2.  It clearly says that you do not have the necessary mount points created.  Therefore you must create the folders in order to mount the drives to them.

```
sudo mkdir /media/Storage
sudo mkdir /media/Windows
```Please note that linux names are case sensitive, so /media/storage is different from /media/Storage.

Then try mounting.
```
