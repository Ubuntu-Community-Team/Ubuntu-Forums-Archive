### Post by Buntu Bunny on 2013-06-27
Is it possible to extract zip files to hidden folders in my home directory with Archive Manager? 

I need Adobe's ICC profiles in $HOME/.local/share/color/icc . (I created the color and icc folders in share). When I try to navigate to this location in Archive Manager to extract the files, none of the hidden files show up as options.

---

### Post by steeldriver on 2013-06-27
Pressing Ctrl-h in the file selection dialog should show hidden dirs (just like in the regular nautilus file manager)

---

### Post by Buntu Bunny on 2013-06-27
Yes! That worked Steeldriver. Thank you!
