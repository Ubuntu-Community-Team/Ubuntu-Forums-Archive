### Post by louis_nichols on 2006-05-13
When something hangs during boot, you can press Ctrl+C to skip it. Now, with hotplug, this might bring some more problems, but should bring the system up so you can troubleshoot the situation and make it go away.

Start with [this thread]("http://ubuntuforums.org/showthread.php?t=49925") to get a boot log and then paste the contents of that log here so ppl can have a look and see if we can figure out why this is happening.
