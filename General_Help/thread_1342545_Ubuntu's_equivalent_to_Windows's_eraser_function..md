This post is not a question.
But a function I discovered for beginners like me.
If it can help some newcomers!
In Windows there is a function called "eraser" that some may know and it's made by a company called heidi, if I don't mistake.
In Ubuntu you can install "WIPE" from synaptic and it does the same thing.
The function is called "delete" and is available from a right click on the desired file.
