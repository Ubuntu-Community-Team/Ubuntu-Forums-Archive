### Post by tranquility base on 2005-09-04
I am new, and have been running Linux for one day now.  I used Synaptic to download PuTTY for SSH my connections.  But I would like to put it on the panel so that I can launch it from there.  PuTTY doesn't show up anywhere in my 'Applications' menu, so the only way I've been able to start it is by going to 'Applications' --> 'Run Application' and typing in "putty".

Please tell me there's an easy way to make a quick launch button on the panel to PuTTY, and to choose the icon, if possible.  Thanks.

---

### Post by mcmuffy on 2005-09-04
Right click a panel / Add to panel / Custom Application Launcher then click Add.
In the next window enter the command used to launch your program in the command box and choose an icon.
Click OK and Robert is your mothers brother.
