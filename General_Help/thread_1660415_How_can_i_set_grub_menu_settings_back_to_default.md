### Post by drs305 on 2011-01-06
From your running Ubuntu OS, you can restore the default Grub2 settings with the following commands. I include the first command to ensure you have an internet connection. Do not run the other commands if you are not connected to the internet.

```
sudo apt-get udpate
sudo apt-get purge grub-common
sudo apt-get install grub-pc
```

Due to dependencies, the commands will purge both grub-common and grub-pc in both cases.

You will be warned about removing all bootloaders. Just tab to OK and continue as you will be immediately reinstalling Grub2 (which is why you need an internet connection).

Once you have done that, see what your Windows menu includes. If it doesn't find Windows initially, run "sudo update-grub".
