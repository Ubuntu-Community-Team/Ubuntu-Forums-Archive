### Post by JohnAStebbins on 2011-02-04
> **oldos2er said:**

HandBrake on windows is very close to the same speed as HandBrake on linux.  It uses multiple CPUs just as it does on linux. More likely, the settings used for encoding were not actually exactly the same or some other external factor (thermal throttling?) affected the encode time.  There is certainly not a factor of >4 difference.  If there were, that would be #1 priority for us to fix.
