Hello,

This is mainly the result of comparing FF2 and FF3, when editing web sites:

I mainly use a browser to edit CMS or wiki. Some difficulties arose when I installed Ubuntu 8.04: with Firefox 3, some GUI editing functions did not work. I kept 7.10 on some machines and waited Firefox 3 to stabilize. 
Today, there is still a problem when editing links in Moin and Plone with Firefox 3:
* with both Moin and Plone, general GUI editing seems ok;
* with Moin, editing Wiki or URL opens the dialog window, but no link is created;
* with Plone, at the first opening, the link is created, but the dialog window has to be forced closed; after that the dialog window will not open any more.

A explanation for the problem with Moin was discussed at [http://rrenomeron.wordpress.com/2008/03/20/fixing-moinmoin-for-firefox-3/]("http://http://rrenomeron.wordpress.com/2008/03/20/fixing-moinmoin-for-firefox-3/") but the proposed fix did not help.

I have now installed Firefox 2 in Hardy Heron: it works, thanks to Ubuntu and Debian, and I can alternately use FF2 and FF3. I made some tries with FF2 and FF3 on the same machine and came to the conclusion that the problem appears only with FF3.

Maybe this problem has already been posted, but the search tool gave me no positive result.

I wrote this to clarify a two-month old problem and hope that the problem with FF3 will be shortly solved.
