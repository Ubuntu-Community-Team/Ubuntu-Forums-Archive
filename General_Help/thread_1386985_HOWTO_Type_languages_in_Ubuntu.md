Hi in this thread you will come to know how to type languages other than Default English
**Step 1**
Install the following packages from Synaptic Package Manager. You can navigate to Synaptic package manager through System--> Administration--> Synaptic package manager. The packages are as follows
```
m17n-db
scim
scim-m17n
scim-gtk2-immodule

```See Fig 1 [ATTACH]144375[/ATTACH]

**Step 2**
Activate your language support by navigating through
System--> Administration-->Language support. Select your language of your choice.
See Fig 2 [ATTACH]144377[/ATTACH]

**Step 3**
Now setup the SCIM by checking the boxes as shown in the Fig 3.
You can navigate to SCIM through System--> Preferences-->SCIM Input method. [ATTACH]144378[/ATTACH]

Well to type open Gedit text editor. Click right mouse button, You will get a popup as seen in Fig 4.
 [ATTACH]144379[/ATTACH]

Select Scim input method
You will get a Language toolbar as shown in Fig 5[ATTACH]144380[/ATTACH]
Select your language and type.
Enjoy.....
