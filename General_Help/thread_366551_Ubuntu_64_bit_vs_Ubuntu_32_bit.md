### Post by bettlebrox on 2007-02-21
If you go 64-bit some software you might want (such as the FLash pluging for web browsers) are only 32 bit and will only work on 32-bit browsers. There are ways around this using chroots. If you want something easy to use go 32-bit, if you want the max performance from your processors and like having to tweak things got 64-bit.
