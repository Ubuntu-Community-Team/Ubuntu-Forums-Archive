### Post by -grubby on 2008-03-09
1: Ubuntu and GNOME are not the same thing. GNOME is a Desktop Envirornment. Kubuntu uses the KDE desktop environment, and Xubuntu uses the XFCE desktop envirornment

2: The different folders are the file layout. /usr does not stand for "User", /bin does not stand for "binary" and /etc does not stand for "etcetera" 

3: If you're referring to the GNOME menus, yes. Well, usually

4: Xubuntu and Kubuntu are modified versions of Ubuntu that use different Desktop envirornments.

5: Hardy and Gutsy are different versions, the OSs are not parellel. Hardy is still in development, but it has betas and alphas released

> 1. Is gnome and ubutnu synonymous? What's the difference. Is there something other than Gnome to run? KDE?
ubuntu comes with gnome, which is a desktop environment. other desktop environments include but not limited to: KDE/XFCE/Fluxbox/IceWM/Openbox....
> 2. What are the different folders for? Are they default destinations? Is USR user? ETC etc.? BIN binary? Is there a guide that covers this? I've looked to no avail.
read this: [http://www.freeos.com/articles/3102/](http://www.freeos.com/articles/3102/)
> 3. Once an app is installed does it go on the menus? Seems I have to go find some of them first.
yes, most of the time it will.
> 4. What is XUNBUNTU? KUBUNTU?
xubuntu is ubuntu with the XFCE desktop. [http://www.xfce.org/](http://www.xfce.org/)
kubuntu is ubuntu with the KDE desktop. [http://www.kde.org/](http://www.kde.org/)

and yes, gutsy is the latest release of ubuntu. [http://www.phoronix.com/scan.php?page=article&item=664&num=1](http://www.phoronix.com/scan.php?page=article&item=664&num=1)
