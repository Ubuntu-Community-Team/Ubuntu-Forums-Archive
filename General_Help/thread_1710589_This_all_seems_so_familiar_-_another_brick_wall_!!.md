### Post by mcduck on 2011-03-21
> **PM47 said:**
>  From what I've read so far I assume metacity and compiz are two alternative windows managers, metacity being the default ?

Paul
You are right, they are both window manaegrs. Metacity is Gnome's default window manager, but both are included in Ubuntu by default and which one you are using depends on if you have visual effects enabled or not.

The setting in the Appearance dialog's visual effects tab switch between Metacity (visual effects set to "None") and two different pre-configured Compiz setups ("Normal" and "Extra").

Metacity should run without any problems regardless of your hardware, while Compiz requires decent graphics card and working drivers for it to provide 3D graphics acceleration.
