### Post by Cheesemill on 2012-06-13
There is no way of updating directly from 11.04 to 12.10.
You would have to do:

11.04 -> 11.10
11.10 -> 12.04
12.04 -> 12.10

Also 12.10 is still in development so unless you have the ability to boot from a different OS, **and** are prepared for frequent breakages using 12.10 then I wouldn't recommend using 12.10 at all.
