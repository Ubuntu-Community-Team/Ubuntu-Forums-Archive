### Post by zhkent on 2011-11-08
Posting this to help others that may have this problem.
The Iomega external backup hard drive wouldn't show up in Ubuntu Natty.
Force mounting wouldn't work.
My curious son :roll: had plugged the Iomega into his windows machine, the problem was that then instead of safely remove hardware, he just unplugged the usb cord.
In ubuntu it showed as already mounted.
Hooking it back to windows and using safely remove hardware the Iomega then showed right up as an icon in Natty.  :D
