### Post by dazer26 on 2011-10-29
I don't have a lot of experience with the windows installer, but I would suggest doing a scandisc and defrag from windows and make sure you have enough disk space free.  If it still doesn't work try uninstalling and re-installing ubuntu (from within windows).  Alternatively you could get the regular version of Ubuntu and let it partition and install on your drive beside windows (if you have room)  but if your gonna do that then make sure to back up anything important.
