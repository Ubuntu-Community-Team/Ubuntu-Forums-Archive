### Post by albandy on 2008-12-23
file-roller, it's installed by default in ubuntu.

go to file-roller
select new
write the compressed file name
Select archive type (tar, zip, jar ...)
pick "Other options"
and  choose the size of the file

### Post by MeURi on 2008-12-23
Well, I think the "problem" is related to rar compression, since Rar is not a free tool for creating archives (but it's free if you want to decompress them).

You can choose a different archive format, like 7zip, which is better than rar and even open source. If you have to create a multi-volume archive using Rar, I think you have to purchase it from the author website.
