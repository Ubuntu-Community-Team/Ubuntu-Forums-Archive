### Post by sudodus on 2012-03-20
I think the button cell battery is gone, and this is why the clock is not surviving during power off.

If you run it as a server, you don't really need vnc or any graphics. And if you really need graphics once in a while, you can run ```
ssh -X
``` or ```
ssh -Y
``` to get remote X graphics. But if you want to, go ahead and install vnc. But remember that the server will be more vulnerable to attacks with vnc.
Read the following link
[[COLOR="Red"]_https://wiki.ubuntu.com/BasicSecurity_[/COLOR]]("https://wiki.ubuntu.com/BasicSecurity")

The following link explains how to use vnc in Ubuntu
[[COLOR="red"]_https://help.ubuntu.com/community/VNC_[/COLOR]]("https://help.ubuntu.com/community/VNC")
