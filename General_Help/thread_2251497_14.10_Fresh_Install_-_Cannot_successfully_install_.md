### Post by CantankRus on 2015-03-12
> **ben147 said:**
> I have the same problem. What do you mean by that? Should I go back to 14.04 instead of using 14.10 ?
I mean download the deb file for 14.04 and see if it works on 14.10.

Better option would be install **redshift-gtk** which is a similar program and will give you an indicator icon in Unity.
```
sudo apt-get install redshift-gtk
```
See this page for info and to configure.... [http://jonls.dk/redshift/](http://jonls.dk/redshift/)

This is my **~/.config/redshift.conf** file...
```
[redshift]
temp-day=4500
temp-night=3500
gamma=0.9
adjustment-method=randr
location-provider=manual

[manual]
lat=-32.22
lon=115.81

[vidmode]
screen=0
```

[**_[COLOR="#B22222"]Get lat long coordinates using google maps[/COLOR]_**]("https://support.google.com/maps/answer/18539?hl=en")
