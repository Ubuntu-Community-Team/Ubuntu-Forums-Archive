### Post by StuartN on 2009-10-01
You could register (for free) at DynDNS and get a lookup based on the hostname you pick, i.e. DynDNS stores the IP address for you, which is updated from home. Some routers have this function built in.

[http://www.dyndns.com/](http://www.dyndns.com/)
