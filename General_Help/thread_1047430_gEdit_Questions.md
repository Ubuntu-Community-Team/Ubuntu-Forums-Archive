### Post by ibuclaw on 2009-01-22
From your message, I presume you have the gedit-plugins installed already, but just looking through it myself, I can't see anything that would strike me as being a feature like the one you say.

But I know as a fact that Bluefish has this built in.
```
sudo apt-get install bluefish
```

Then go into **Applications->Programming->Bluefish Editor**, maximise the application and you'll  see on the left panel a folder and file listing.
Navigate to the directory you wish to search, right-click it and select **Advanced Open**. Everything you asked for is in there. (See attachment).
