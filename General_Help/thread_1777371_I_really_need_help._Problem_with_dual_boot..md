### Post by FormatSeize on 2011-06-07
[This thread]("http://ubuntuforums.org/showthread.php?t=870041") has a possible solution for you. 
 
But heed this advice: The next time you want to try out Ubuntu, don't partition your drive and install it right away. You can try it using the Live USB that you booted from without affecting your system (you can also use a Live CD). That way, you could have tested to see if Ubuntu would work with your program and not have any issues. 
 
Hope this helps!
