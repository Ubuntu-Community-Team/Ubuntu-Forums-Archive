### Post by Majlo on 2005-08-06
Hi..

For shoutcast i use streamtuner

sudo apt-get install streamtuner
killall gnome-panel

Then Application --> Sound & Video --> streamtuner

In streamtuner go to Edit --> Preferences and change Listen to a stream to 
rhythmbox %q
