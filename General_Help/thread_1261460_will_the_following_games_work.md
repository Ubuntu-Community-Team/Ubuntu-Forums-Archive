### Post by Vaphell on 2009-09-08
enemy territory has a native linux version, WoW works just fine in WINE
for any windows app check [http://appdb.winehq.org/](http://appdb.winehq.org/) to see if it can be run via WINE.

compatibility issues with laptops - it's somewhat true. They usually are sold with windows tailored specially for them. When you install Ubuntu you need to get drivers for all, even the most obscure hardware. Majority is working out of the box, but there are problems with the newest stuff.
Unfortunately often manufacturers don't bother to provide said drivers and you have to rely on open source versions which are often lacking. Manufacturers don't publish hardware specs so it's quite hard to write a fully working driver.

When you are going to use linux on your new laptop, it's wise to check compatibility issues before buying. There are many hassle-free ones.
