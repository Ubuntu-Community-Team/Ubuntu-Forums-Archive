### Post by ajgreeny on 2006-06-30
If your user name is *jacob* and your computer is named *ubuntu-jacob*, then I think you are already logged in but without a GUI such as gnome or kde.

At the *jacob@ubuntu-jacob* prompt, type startx and see what happens.  If nothing, then you must have installed the server version which has no desktop, so either type *sudo apt-get install ubuntu-desktop*, then your password when asked, (your own password, not a root password which you haven't got), and you should get the gnome ubuntu gui. You could also start again and download the newer version 6.06 which is so much better than 5.10, but make sure you get the desktop iso.
