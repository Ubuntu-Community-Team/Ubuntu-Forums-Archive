### Post by Mark Phelps on 2011-04-18
> **Kharlo said:**
> ... is there any way i could use linux to fix windows7? 

Not really.  You can do a few minor things like rewrite the MBR, but you can't do any extensive repairs.

What you need is a Win7 Repair CD which you CAN use to repair the Win7 boot problems.  Go to the link below, download and burn the appropriate CD image:

[http://neosmart.net/blog/2009/windows-7-system-repair-discs/]("http://neosmart.net/blog/2009/windows-7-system-repair-discs/")

Then, boot from that, select Startup Repair, and run it three times.

If that doesn't work and you need more detailed help with Win 7, suggest you contact sevenforums.com -- a Windows 7 forum.
