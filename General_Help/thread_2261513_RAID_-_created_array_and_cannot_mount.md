### Post by SeijiSensei on 2015-01-19
I think it's pretty likely you destroyed whatever information was on those drives when they were formatted.

You can mount the array by creating a mount point, say, /media/bigarray, then using

```
sudo mount /dev/md0 /media/bigarray
```

You can mount it automatically at boot by adding an entry to /etc/fstab.

You couldn't mount the array before because it did not have a filesystem on it until you ran mkfs.
