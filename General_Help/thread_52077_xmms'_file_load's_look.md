### Post by varunus on 2005-07-27
Um, beep-media-player is actually just a fork of XMMS.  It should work with the same plugins and the same skins.  :)

If you want beep-media-player to have the same look and feel as KDE, install the gtk2-engines-gtk-qt package from synaptic, and then go to your control center.  There'll be a new control center module talking about GTK2 apps; choose to make them look like KDE ones.  This will make everything look the same.
