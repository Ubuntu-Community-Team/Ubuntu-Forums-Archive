### Post by 73ckn797 on 2010-01-24
[khelben1979]("http://ubuntuforums.org/member.php?u=443262") has a good solution.

You could also down load the "deb" file from Open Office: [http://download.openoffice.org/index.html](http://download.openoffice.org/index.html)

Right click on the file and extract to a temp folder. From there right click on the "deb" file and select to install using "GDebi Pacakage Installer".

First go to System/Administration/Synaptic Package Manager and removed any remaining OO packages.
