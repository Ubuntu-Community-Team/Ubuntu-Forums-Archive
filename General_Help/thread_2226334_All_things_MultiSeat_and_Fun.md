Hi All,

So to start this is a forum post i will try to keep up-to-date while i conduct my experiment into multiseat.
Now what i am doing is not easy, but then it was never suppose to be. This is the first stage of a personal project that will hopefully result in a new distro with the intention of being MultiSeat/Multimedia.

What i have done to date :
1x Computer Situated in the attic with (8CPU cores AMD-FX, Nvidia 650Ti, 16Gb RAM, 256Gb SSD, 1TB spining Disk, and a Creative XFi sound card)
A HDMI/USB cable running to Office/Bedroom/Living room.

Living-room = 1xHDMI TV, Amp(HDMI Audio), USB hub, USB micro keyboard/trackpad, IR Blaster
Office = Monitor(HDMI Audio), Keyboard/Mouse, Printer, USB hub, Printer.
Bedroom = Nothing yet, just the cables run into a blanking plate.

So the problems to date!?
Well it wasn't easy to start with, Turns out the monitor i have supplies a dud EDID, it tries to initialise by default a metamode that isn't supported. And guess what, my TV does the same(Even with works windows laptop).
When the monitors are off the screens don't initialise. (there is a fix for this that i will list below).
The multiseat documentation contains holes, and is pure guess work half the time.
Sound (Well it had to rear its ugly head) - Tying the HDMI audio outputs to the actual HDMI screen sessions!

Ideal Outcome:
Ubuntu Desktop by default in the office with the ability to switch to KDE.
UbuntuTV or XMBC in the living room with the ability to fireup steam in big picture mode.
Better documentation to setup Multiseat.
