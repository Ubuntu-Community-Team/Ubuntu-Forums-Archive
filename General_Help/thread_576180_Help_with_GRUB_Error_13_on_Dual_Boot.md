### Post by Aterian on 2007-10-21
Just a quick possibility from the stuff I found out while trying to solve my problem:

> title Microsoft Windows XP Home Edition
 rootnoverify **(hd1,1)**
 savedefault
 map (hd0) **(hd2)**
 map **(hd2)** (hd0)
 makeactive
 chainloader +1

Seems like you should be "root"ing the same drive as you're mapping to hd0.  Try...
> title Microsoft Windows XP Home Edition
 rootnoverify (hd1,1)
 savedefault
 map (hd0) **(hd1)**
 map **(hd1)** (hd0)
 makeactive
 chainloader +1
...instead, since you only have two hard-drives.
