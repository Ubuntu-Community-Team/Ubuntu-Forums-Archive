### Post by PhilOSparta on 2006-04-16
One way to resize the desktop icons is to right click on the icon and select "stretch icon".   Then you can push or pull the corners to make the icon smaller.

The icons on the panels can be resized by right clicking on the individual panel and select "properties".  Then set the size.
