### Post by 3Miro on 2011-07-11
Only Kwin can compete with compiz in terms of effects and Kwin is for KDE. You may be able to get it to run under Gnome, but it will buggier than compiz.

Other Windows Managers (not decorator, decorator is just one component of the WM) don't have the effects (i.e. Metacity, XFWM4, OpenBox, IceWM ...)

### Post by mali2297 on 2011-07-11
[xcompmgr]("http://linux.die.net/man/1/xcompmgr") provides some basic eye-candy such as transparency and shadows.

[Cairo Composite Manager]("http://cairo-compmgr.tuxfamily.org/") is newer and seems to have more features but I do not know how stable it is.

Neither replaces a window manager. Instead, they work together with an ordinary window manager such as metacity or openbox.

### Post by CatKiller on 2011-07-12
> **1chess2u Christian said:**
> So, You say that I should change my window manager to KDE? 

No, he's saying that the options for a window manager with lots of bling are Compiz or KWin. If you want lots of bling, you'll need to pick one of those.

If you want some bling, then you can use one of the lighter-weight compositors with a lighter-weight window manager as mali2297 suggests.

Either way, using a different window manager involves changing your window manager. Window decorator, window manager and desktop environment are not synonyms.

Have you actually used Compiz? What problems are you experiencing?
