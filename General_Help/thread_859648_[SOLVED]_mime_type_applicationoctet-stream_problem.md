### Post by dond on 2008-07-14
I'm running Heron. I like to use Kontact for mail and Firefox for browsing but when I clicked on a weblink in an email it was Konqueror which fired up instead of FF. Using Applications>Other>File Associations I deprecated Konqueror and moved Firefox to the top. Now it launches fine but i must have done something else too because now some applications (Kontact and Rosegarden to name a couple) give the message 'Could not find mime type application/octet-stream' on launch. But they do seem to work fine if I just ok the message and carry on.

Can anyone please tell me how to remedy this situation?

Cheers

Dond

### Post by dond on 2008-07-15
Seems like this is a common problem when I googled it found lots of solutions. In the end I went back to Applications>Other>File Associations and added 'octet-stream' to the 'applications' branch of the tree. that seems to have worked.
