### Post by zeroseven0183 on 2009-12-16
Hi! You can do that by actually installing another Firefox add-on. It's called [Public Fox]("https://addons.mozilla.org/en-US/firefox/addon/3911"). It has options to lock Downloads, Add-ons, changing the about:config, etc.

I saw this add-on recommendation [here]("http://www.askvg.com/how-to-set-restrictions-in-mozilla-firefox-restrict-downloads-adding-bookmarks-add-ons-changing-options-etc/"). It works for me. I'm using Firefox 3.5.5 in Ubuntu 9.10.
