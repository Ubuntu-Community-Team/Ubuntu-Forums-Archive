### Post by bored2k on 2006-02-16
[LIST]
[*]Go to your file browser.
[*]Right click on a video and select the Properties option.
[*]From there, go to the "Open with" tab.
[*]If the mplayer option is not there, click on Add, browse through the application list and select if. If it's not there, select on enter custom command and type down **gmplayer**.
[/LIST]

SMB? Sure thing. It should.
