### Post by solitario on 2011-10-22
Hello all. I recently upgraded by Ubuntu and I have this strange problem when maximizing or enlarging windows beyond a certain size... Please see the attached images. When windows get beyond a certain size they turn all white. Also, after walking away from my computer for a while the whole screen is white. I discovered that by typing in my password and hitting enter that I can log in, but the normal login window is not "painted" properly. Any help is appreciated. Thanks.

---

### Post by solitario on 2011-10-22
I believe I've fixed it.
After the upgrade Ubuntu asked me to select a graphics driver and gave me two choices:
NVIDIA accelerated graphics driver (version 173)
NVIDIA accelerated graphics driver (version current) [Recommended]
I selected the one which was recommended. After going to System->Administration->Additional Drivers, reverting to "version 173" and restarting the problem went away. Thanks.
