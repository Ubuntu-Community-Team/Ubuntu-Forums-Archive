Alright, just wanted to say that perhaps I posted too soon.  I went into about:config, noticed some things had been changed, set everything back to default (which you can find on MozillaZine or Google) and so far so good... even DeviantArt is loading again :p

Looks like it was all my fault afterall for messing with stuff in about:config at some point.
