### Post by wsonar on 2009-04-21
Easy way

I got my domain through Easydns they handle my DNS so the world can see me
35 bucks for a year otherwise you will need to setup your own DNS server

you can use dynamic DNS but a static IP is better


I have a static IP witch I assigned to my router
then forwarded port 80 to my web server

install ubuntu server choose to install lamp server [Ubuntu Server Guide]("https://help.ubuntu.com/8.10/serverguide/C/index.html")

see link to customize apache
[https://help.ubuntu.com/8.10/serverguide/C/httpd.html]("https://help.ubuntu.com/8.10/serverguide/C/httpd.html")

default is /var/www

or you can edit the /etc/apache2/apache2.conf 
to choose your own directory

not a good description but thats the just of it

I got my web server and e-mail server both running on a cheap old box p4 2 gig ram nothing fancy but seems to work pretty good

you probably would want to get ssh access setup first so it's easier to administer remotly
through a terminal into your server
