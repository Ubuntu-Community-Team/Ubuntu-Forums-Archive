### Post by ericinwisconsin on 2008-01-09
That's a pretty old machine, and while I don't know the system requirements for Ubuntu off the top of my head, I'd be willing to bet that Ubuntu is a little too demanding for the system you have. Check out this thread:

[http://ubuntuforums.org/showthread.php?t=75868](http://ubuntuforums.org/showthread.php?t=75868)

There are some useful tips there. The best recommendation is to use a different desktop, like icewm or xfce4. Also, a less demanding web browser, like Dillo.

A good way to see how a Linux distribution will affect your system is to run a live CD. Not all distros have one, but Ubuntu does.

For my slow (700Mhz/256M RAM) laptop, I usually run Puppy Linux. It requires less resources than most versions of Linux and runs fairly decent.
