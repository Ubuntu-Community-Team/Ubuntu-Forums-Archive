This is a common problem and doesn't need a reinstall to solve. 

Right-click on the desktop and create a folder. Don't bother to name it anything special. Right-click on the folder icon and 'Open with other Application". Tick the "Remember this application" box and highlight "File Browser" in the application list. Now click on "Open". That will reassociate folders with the Nautilus file browser. You can now delete the temporary folder you made on the desktop.

When you next open a folder in a media player by right-clicking, make sure the remember box is **not** ticked. This is possibly the way the issue arose in the first place.
