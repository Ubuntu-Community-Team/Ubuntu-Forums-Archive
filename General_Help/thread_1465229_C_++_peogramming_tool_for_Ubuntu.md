### Post by overlordchin on 2010-04-29
Sunny,

It sounds like what you are looking for is an IDE for c++ programming. If you Click on "Applications" in the upper left of your screen and then "Ubuntu Software center" (the name may vary in diff version of ubuntu) You can click on the programming button to see all the various utils you can install related to programming. Code blocks, codelite and eclipse are among the more popular graphical IDEs for linux. You should be able to set up C++ color schemes and keywords in any of those applications. Also they should all be able to compile and run your code. 

You can also use command line based editors and compilers (vim / emacs and g++)

anything and everything you can do in an IDE can for the most part be replicated on the command line. Although its not always as easy as clicking a button.
