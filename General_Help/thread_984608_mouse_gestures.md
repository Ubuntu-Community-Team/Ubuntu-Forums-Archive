### Post by penquin on 2008-11-16
how do you disable the mouse gestures that change between the workspace?

---

### Post by muadnu on 2008-11-16
I'm assuming you are using Compiz. Open the CompizConfig Settings Manager. If you don't have it, install it by running
```
sudo apt-get install ccsm
```
on a terminal, and then go to System->Preferences->CompizConfig Settings Manager. Next, click on the Viewport Switcher plugin under general, then go to "Desktop-based Viewport Switching", look for the "Move Next" and "Move Prev" entries, click on the buttons saying "Button5" and "Button4", and uncheck the "Enabled" checkbox.
