### Post by Mark Phelps on 2011-07-23
If you're going to use Wine, you need to first go to the WineHQ site and look up the ratings for your apps.

I went there and only found one page -- the following:

[http://appdb.winehq.org/objectManager.php?sClass=application&iId=12049]("http://appdb.winehq.org/objectManager.php?sClass=application&iId=12049")

As you can see, it's rated garbage -- meaning it's NOT going to work, no matter what you do.

You would do better either installing Windows using Virtualbox, or going with dual-boot.
