### Post by sdowney717 on 2009-03-14
simply 
download qt creator sdk for 0.4.5.0

[http://www.qtsoftware.com/downloads](http://www.qtsoftware.com/downloads)
install the bin file and tell qt creator to use qt4.5 under projects pane

download 0.5 arora with source code

[http://code.google.com/p/arora/downloads/list](http://code.google.com/p/arora/downloads/list)

open the project file (*.pro) in qt creator
you can test to see if it will work by running the project in the ide

then just do
clean project
run qmake
build project

the resulting arora bin should now work with flash
