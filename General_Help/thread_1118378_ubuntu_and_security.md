> **willannex said:**
> If ubuntu is installed inside of windows, can virus' aimed at windows penetrate linux and thus corrupt any of my windows files?

It's unlikely: a significant portion of malware is designed for Windows systems, and doesn't run in Ubuntu without help. A bigger threat is from catching something while using Windows.

Caution and limited accounts (together with proper protection while Windows is running) are your best friends.

### Post by mb_webguy on 2009-04-06
This certainly wouldn't be as safe as running Ubuntu by itself.

It's essentially a similar situation to that faced by those of us who dual-boot Linux and Windows, and share a data partition between the two.  If you download an infected file while using Ubuntu, you don't have to worry about it infecting your Linux installation.  But if you then open or view the file in Windows, it can infect your system.

As much of a fan I am of Linux and Ubuntu, I actually think that this is a poor reason to use Linux if you plan to continue to use Windows.  If you want to use the Windows operating system, there are steps you can take to make it fairly safe and secure, such as using a web browser other than IE (such as Firefox or Opera), running a good and regularly updated anti-virus application (such as AVG, Avast, or Avira), keeping your Windows installation up-to-date with automatic updates, and not using an Administrator account for anything but actually administration.

If you want to use Linux, then use Linux -- even if by dual-booting so that you keep Windows as well.  But running a Linux inside XP in a virtual machine or something similar as a means of making Windows safer is a band-aid solution at best.
