### Post by Mark Phelps on 2011-08-03
Your BEST bet is to REMOVE all partitions from the drive, not to create empty NTFS partitions with Linux utilities.

Win7 will see the empty drive and will do its own formatting.

As to the question about booting, NO, Win7 will not boot from a drive that does not already contain an OS.
