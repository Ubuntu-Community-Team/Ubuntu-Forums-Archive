### Post by grahammechanical on 2014-05-10
I installed a Windows 8 pre-view edition. I only had Ubuntu on the machine. I downloaded the Windows 8 ISO image. I used File Manager>Write to Disk option. This is how I burn Ubuntu ISO images. In my case I burned the ISO image to a DVD disk. And I run the Windows 8 live session from that DVD and installed Windows 8 pre-view from the DVD. Try File Manager>Write to Disk.

Regards.

### Post by Mark Phelps on 2014-05-11
You can't create a "LiveUSB" for Win7 in the sense of having a Windows version that will run the full OS from the USB  You CAN create an installer on a USB stick, but all that will do is install the OS.
