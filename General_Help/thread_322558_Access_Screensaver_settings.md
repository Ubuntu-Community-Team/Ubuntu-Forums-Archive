### Post by bettlebrox on 2006-12-20
Gnome screensaver seems to be a bit braindead. ;) I can't find any with it to change the options for any of the screensaver. But, what works for me is to use Xscreensaver to set the preferences the screensaver including for GLslideshow.

Run the command xscreensaver-demo, click cancel if it asks to start the Xscreensaver daemon. Click on the advanced tab to set the PATH to where the pics are.

U may have to install xscreensaver for this to work.

### Post by EZ-man on 2006-12-28
I read your message,

Wanting to display your own pictures in the screensaver.

I wanted to do the same...

I found out, that if you go to: System/Preferences/screensaver..
The screensaver preferences box appears.
Select pictures folder in the screensaver preferences box.

Now, In your home folder.
Create a folder and title it Pictures.
Place all the photos you want in it.

 Displays your pictures...
