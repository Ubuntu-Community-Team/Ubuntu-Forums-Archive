### Post by Peepsalot on 2006-12-18
This is a trick I learned in Windows not too long ago which I found incredibly helpful.

Whenever an alert/error/warning/etc message pops up, if the window is in focus and you press Ctrl-C, the contents of the window will be placed into the clipboard.  It's a hell of a lot easier than transcribing the text of a long error message into notepad.

I noticed today in gnome however, that this doesn't seem to work.  Is there some other way to copy an error or alert message?
