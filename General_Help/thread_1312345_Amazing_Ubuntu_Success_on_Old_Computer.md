### Post by ukapolog on 2009-11-03
I have, in the last ten days put Ubuntu on two computers in which the system shares with Windows (XP in one case, Vista Premiuum in the other). In both cases the whole thing seems to have been very successful; they both now run Ubuntu 9.10.
But the amazing thing is that the XP machine is quite an old computer (2001) and I had tried every trick under the sun to make it work faster and failed. However, with Ubuntu it now*** flies*** - never thought I would see this happen and this computer was very close to being thrown away. I really want to thank Ubuntu for that, actually that old computer runs Xubuntu and my much newer Vista runs Ubuntu.
I have become highly enthusiastic about Ubuntu.
My only little complaint is that neither computer will fully shut down after I have been on Ubuntu; I mean they close down for all intents and purposes but one is left with that black page with some writing; in the case of Ubuntu on Vista, the black page lists about 8 error reports but only about three times out of every ten attempts does it then shutdown, I normally have to pull the plug.
For two days I searched all these forums on this and tried all the suggestions, but none of them helped. This has not effected my enthusiasm for Linux at all however.
 
ukapolog.
