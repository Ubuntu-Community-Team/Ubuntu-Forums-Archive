### Post by 11.04 on 2011-09-13
On the left side of the Nautilus file manager, by default there are five subdirectories of your home directory displayed -- Documents, Music, Pictures, Videos, Downloads.

Is it possible to add another subdirectory to that list so it shows up along with those 5 mentioned above?  If yes, how is it done?

---

### Post by Old_Grey_Wolf on 2011-09-13
If it is in your Home directory, in Nautilus go to the top bar and use the File menu. Select File > Create Folder, then enter the name for your new folder.

If you are trying to add a folder somewhere else you may get a permission denied error. If you do you need to post back here to get further instructions on elevating privileges.

### Post by 11.04 on 2011-09-13
What you described only shows up on the **right** frame of Nautilus as a regular folder.

I'm trying to make a new folder show up in the lower left section of Nautilus along with the other "special" folders (Documents, Videos, etc.).

---

### Post by 11.04 on 2011-09-13
I figured it out.  Just drag a folder from the right frame to the left frame, and it will show up in the left frame along with the other "special" folders, while also being visible on the right.
