I've downloaded a few icon sets from gnome-look but I have no idea where I'm supposed to save/extract the tar to.

---

### Post by eternalnewbee on 2008-11-20
Save them in a folder in your home directory. Then go to: **Main Menu > System > Preferences > Appearance**, and just drag a set into the Appearance window.
