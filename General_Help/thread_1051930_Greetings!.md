### Post by mb_webguy on 2009-01-27
Welcome to Ubuntu! (And hyper_ch is right -- you will get more responses and get them quicker if you use a descriptive subject.)

> **Poison Monkey said:**
> (1) I use a password manager in Windows called [**TK8 Safe**](http://www.tk8.com/safe). This is actually one of the only paid programs that isn't open source or free that I have left running because I quite like it.
However I am willing to switch and was wondering what is a good and secure program that is similar for Ubuntu?
I don't use one myself (though perhaps I should).  There are several available in the Ubuntu repositories, including Figaro's Password Manager (fpm), KeePassX (keepassx), GPass (gpass), ZSafe (zsafe), and Password Gorilla (password-gorilla).  Since I've never used any of them, I can't give you my opinion on which is best, but I have seen Figaro's Password Manager recommended by several people.

> **Poison Monkey said:**
> (2) I also ran a program in Windows called Key Scrambler just for added protection against key loggers. 
I understand that virus and spyware scanners aren't really needed for Ubuntu but is security good against key loggers?
I suggest you read the link on security in Ubuntu in my signature.  The short answer is that no software is ever installed in Linux unless you initiate it, so you shouldn't have to ever worry about keyloggers and the like.  The long answer is that every time you install a piece of software, you're trusting the software developer that the software works as advertised, and it is possible that you could install an application that, unbeknownst to you, contains malicious code.  

However, all of the software in the official repositories is reviewed and tested by the Ubuntu team.  As long as you install software from trusted sources (such as the official repositories, Medibuntu, GetDeb, websites for major software projects, and most Launchpad PPAs), you don't have to worry about things like that.  (When in doubt, ask on these forums whether a software source is trustworthy.)  The vast majority of the time you can find an application to suit your needs in the official repositories.

> **Poison Monkey said:**
> (3) Are there any recommended sites I should visit in regards to Ubuntu and other open source software available?
Google is your friend.  Include "Ubuntu" in your search for a particular type of application, and you'll get plenty of suggestions.  There are several websites that provide lists of Linux equivalents for Windows applications, such as [this one]("http://wiki.linuxquestions.org/wiki/Linux_software_equivalent_to_Windows_software").  This forum is also great for getting software suggestions.
