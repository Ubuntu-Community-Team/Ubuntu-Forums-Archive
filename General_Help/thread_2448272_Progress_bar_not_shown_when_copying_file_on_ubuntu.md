### Post by Dennis N on 2020-08-05
There is an transient icon that appears on the top bar of the file manager - a circle that fills as the transfer progresses. The progress and speed is also displayed in the dropdown during the transfer. See the screenshot, which was taken after the transfer had completed, so the circle is filled.
