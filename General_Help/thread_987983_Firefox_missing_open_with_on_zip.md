### Post by __Ryan__ on 2008-11-20
It looks like your mime types are incorrect.

In Firefox: Edit -> Prefrences -> Applications

Find the ZIP archive under content type and select the Archive Manager from the dropdown menu to the right. You an also edit your other files types from this screen as well.
