### Post by doas777 on 2011-05-12
dos batch scripting, like *nix BASH scripting, are interpreted langagues, rather than compiled ones. usually, only formal programming languages can be "compiled" to binary. 

I'm afraid your best bet is to rewrite the script in a more formal langague, and use a native compiler to build it. also, exe is a windows format, so yes, it would be best to compile any windows executables on windows.

### Post by Chronon on 2011-05-12
You can try using MinGW to generate Windows executables in Linux:
[http://www.mingw.org/wiki/LinuxCrossMinGW](http://www.mingw.org/wiki/LinuxCrossMinGW)

As doas777 said, you need to use a compiled language rather than an interpreted one if you want to compile a binary.
