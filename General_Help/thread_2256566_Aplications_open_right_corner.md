### Post by mc4man on 2014-12-13
Is a window already open or a window open but minimized. If so then that's typical behavior for the default place mode of "Smart"

If desired you can try other place modes in ccsm (install compizconfig-settings-manager), the  plugin is Place Windows > Placement mode > screen
