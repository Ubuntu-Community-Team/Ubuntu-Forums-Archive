### Post by theDaveTheRave on 2012-06-19
You may need to play around with the settings.

I did the same thing with my eeePC, I could see the display but it took a while to get the screen to actually show anything. This mostly came down to the physical size and higher resolution of the projector.

Check this out by going to your setting menu, then finding the display section.

You should see both "screen" then you just need to tell the system how to use them (individual screens, or only using one or the other). You may also find you need to have the screen plugged in on boot up the first time you do this, after that it seems to "remember" the settings, it obviously saves some meta data against the screen for when you plug it in again later on.
