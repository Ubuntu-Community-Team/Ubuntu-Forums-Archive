### Post by audiomick on 2010-02-05
Should be easy, I think. As far as I know, you have to mark the printer as shared on the machine it is on, and then you can choose it in the printer setup thing in
system> administration> print
on the machine you want to print from.
Don't know more precisely than this, as my network printer is attached (via usb) to my router. If your router can do that, it is a good choice. It means that the printer will still work if the other computer is not on.
