I have an xmodmap definitions file in my home directory which I load by putting a line in my "Startup Programs" (found in System-->Prefs-->Sessions).  Works great for all apps -- including Ctrl-Tab in Firefox, for example -- except one thing.  The Alt-Tab window-switching feature of Gnome doesn't work.  

I can get it to work after login if I temporarily redefine the kbd shortcut for window switching to something else (for example, Alt-1) by going into System-->Prefs-->Kbd Shortcuts, and then immediately set it back to Alt-Tab (though I'm actually pressing Alt-CapsLock on my busted kbd of course).  It then works fine for the rest of the Gnome session.  But, it's a pain to have to do this every time I login to the computer... 

Any ideas for how to get a remapped Tab key to do Alt-Tab window switching at startup?
