### Post by dioxin on 2008-07-07
Ditto on installing the drivers manually

also If you plan on using the 4850/70 with wine games you'll probably run into the checkerboard corruption problem which will make them unplayable.  So may want to hold off until the next driver version (Catalyst 8.7) or a workaround is posted.  I've done some research and have yet to find a fix(other than downgrading to 8.5 which doesn't work on the 4850.)

more info is here(about Cedega but also applies to wine and crossover):
