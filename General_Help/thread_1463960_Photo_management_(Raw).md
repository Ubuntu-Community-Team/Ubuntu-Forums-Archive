### Post by Malcy on 2010-05-03
You can try the following:

Bibble 5 pro
Picasa 3
Rawtherapee
Rawstudio
UFraw (can be used with Gimp or alone)
DigiKam
PhotoShop Elements and certain versions of PhotoShop will work with CameraRaw in Wine

There are probably others too.

My main editor is Bibble which I use for jpegs too. There is a trial version available. :)
