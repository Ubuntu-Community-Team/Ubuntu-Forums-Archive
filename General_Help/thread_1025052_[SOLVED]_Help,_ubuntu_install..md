Um hey, i loaded parted magic and resized my main partition and added a ext2 one, how can i tell ubuntu to only install inside the ext2, thanks.

---

### Post by louieb on 2008-12-29
Choose manual partitioning. Click on the ext2 partition. chose edit. Select / (root) for its mount point, check the format box.  And off you go.
