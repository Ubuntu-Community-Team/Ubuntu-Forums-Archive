### Post by zaveloff on 2009-01-05

### Post by zeronothing on 2009-01-05
its easy, open up synaptic package manager by doing this:

using the menu at the top select System -> Administration -> Synaptic

now using synaptic search for thunderbird. select the thunderbird package and click apply. this will install thunderbird for you.

---

### Post by zeronothing on 2009-01-05
oh yea, once you install thunderbird using synaptic package mananger. you will find thunderbird here:

at the menu at the top of your desktop, Applications -> Internet -> Thunderbird
