### Post by MJBoa on 2010-02-28
Did you follow the directions for reinstalling? I'll assume you've mounted your ubuntu root partition to /mnt, like it says in the directions.

You'll need to run the command like it says:

sudo grub-install --root-directory=/mnt /dev/sda

It's /dev/sda NOT /dev/sda3 because you're installing it to the MBR of the whole disk.

### Post by themusicalduck on 2010-02-28
Ok, so you have two hard drives. Am I right in thinking that Windows is installed on the 82gb hard drive? If it is, then that is the hard drive that you boot from and you could try this:

The partition that Linux is installed on is /dev/sda3.

So when you are booted into a live CD type these into a terminal.

```
sudo mkdir /media/sda3

sudo mount /dev/sda3 /media/sda3

sudo grub-install --root-directory=/media/sda3 /dev/sda

```

That should hopefully re-install grub with no problems.
