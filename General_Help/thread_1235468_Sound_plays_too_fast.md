### Post by NoaHall on 2009-09-04
It might be your parts messing with your sounds - for example, my old network adapter messed with my sound card - making the sound speed up for no apparent reason - removed the network adapter, sound works great now! Try remove any parts you don't need, see if sound works, if it doesn't, then there's not much to be done about it (except maybe buy a new sound card), if it does, slowly add the parts so you can work out which one is causing it.

Also, what CPU/motherboard/RAM do you have?
