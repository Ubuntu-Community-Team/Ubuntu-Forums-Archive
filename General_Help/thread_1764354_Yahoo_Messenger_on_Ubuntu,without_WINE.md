### Post by slooksterpsv on 2011-05-21
There's Empathy, Pidgin, etc. A direct Yahoo IM, nope.

The thing is, and I feel most people feel the same way which I could be wrong, a lot of linux users like multi-purpose tools =D like multi-im messengers. Empathy and Pidgin allows for you to add MSN, AIM, Yahoo, Skype, Jabber, etc. accounts without the need for another IM. Saves lots of memory in the long run if you use multiple IMs like I do.

I'm almost guessing that you need plugins to work like WebCam or that, I believe it does on Pidgin for yahoo.

I hope that helps =D :KS:popcorn::D:D
