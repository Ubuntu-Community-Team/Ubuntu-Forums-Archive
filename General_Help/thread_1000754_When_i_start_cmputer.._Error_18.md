### Post by lifeofcannabis on 2008-12-03
I have had this error before, i fixed it with the super grub disc, it works well for error 18 and 15, and it has tons of other useful tools, its free.

but if you have it dual boot, i wouldnt use it because it may mess up your dual boot (did for me one time)

but if your not dual boot, go here:
[http://supergrub.forjamari.linex.org/](http://supergrub.forjamari.linex.org/)

download that and burn the iso image to a cd or dvd, go to your dos setup, and make sure its set to boot from cd first.

now the grub disc menu is a little hard to understand but go with the "help" part and it tells you want each option does.
