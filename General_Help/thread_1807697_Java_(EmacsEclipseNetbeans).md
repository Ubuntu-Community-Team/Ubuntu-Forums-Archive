### Post by lykwydchykyn on 2011-07-20
What you need to do at this point is find a good book on web development and read it.  I'd recommend learning PHP since that's typically the "P" in LAMP.  The others are Linux, Apache, and MySQL.

Here's basically what you're going to need to know to work on this project:

 - SQL (you'll want to store your feed data in a database).
 - PHP or some other server-side language (most people start on PHP -- other options would include Perl, Ruby, Python, or Java)
 - HTML 
 - the XML format for RSS (it's not hard -- see [http://www.xmlfiles.com/rss/](http://www.xmlfiles.com/rss/))

As for emacs and eclipse -- they're just editors.  Web development is just making text files.  If you are already comfortable with a text editor, just use the one you are comfortable with.  It sounds to me like you have enough of a learning curve for this project without having to learn a new text editor or IDE.
