### Post by WorMzy on 2010-12-16
One thing worth baring in mind is that nVidia is better supported on Linux than ATi is. Wireless devices have still got relatively bad support, so if you're buying a wireless network card or USB stick, then check it's compatibility on this website: [http://linuxwireless.org/](http://linuxwireless.org/).

The main advantage to 64-bit is the ability to use more than 3GB of RAM without having to use a PAE kernel.
