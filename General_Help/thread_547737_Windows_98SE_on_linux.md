### Post by mannheim on 2007-09-10
The statement that VMWare "does not support Win 98" doesn't mean you can't install and run it. I haven't done it, but why shouldn't it work?

You won't get some of the extra features that are usually supported in more recent Windows versions under VMWare. That is, there won't be a suitable version of the "VMWare tools", which improve performance and provide some additional integration. But I guess that performance won't be an issue, as Windows 98 doesn't need much.

### Post by Shazaam on 2007-09-10
I've installed Win98SE in Vmware Player. It runs slow, (normal for a  Windows virtual machine) has NO 3d video rendering, and the sound is horrible. But it does run. If you have games or video intensive programs a dual boot is the only way to go. AFAIK Vmware has VERY limited 3d support. You could try Wine or something similar.
