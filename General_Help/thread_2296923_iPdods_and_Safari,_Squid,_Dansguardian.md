### Post by ecowan on 2015-09-29
So I have a small home network with a proxy server that runs Squid and Dansguardian. I have my router set up to accept access to the internet for ports 8080 and 1080 from my proxy server only. This works just fine for Ubuntu laptops with Firefox.

Now my kids have iPods, which apparently use Safari and I don't know what else, which does not use port 8080 but 1080. How can I arrange to pass web browsing through my Squid-Dansguardian proxy server?

Thanks. - Erny

---

### Post by ecowan on 2016-01-06
So I found a work around. I told my router to block traffic to port 1080 (Opera) and told the kids they had to switch to Firefox.
I'll mark this as [SOLVED].
