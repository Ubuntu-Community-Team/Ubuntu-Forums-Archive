### Post by brunobueno on 2009-04-29
Hi,

I need to define a rule/standard for login of my network users.

For example: 

My name is **Bruno Bueno**, what is the best option?

          - **brunobueno** (I like this, but users hate, I don't know why)

          - **bruno.bueno** (I hate this, because in VIM it makes the files like /etc/group or /etc/passwd hard to read, because the function "syntax on" stay wrong)

          - **bruno_bueno**

          - **many other options**

There are many ways to do this, but I want to know if there is a way that is the best, and why ?

Thanks for all

### Post by kpkeerthi on 2009-04-29
<first-char-of-first-name><full-last-name>
<full-last-name><first-char-of-first-name>

**buenob**
**bbueno**

### Post by Maheriano on 2009-04-29
Don't do the underscore, it slows down the typing of the username having to backspace because you accidentally hit equals.
