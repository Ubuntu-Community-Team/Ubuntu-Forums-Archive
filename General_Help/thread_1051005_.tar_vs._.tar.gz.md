### Post by mcduck on 2009-01-26
As for background, .tar is short for "tape archive", it was necessary in the days when tape drives were common. Since tapes can't store any directory structure (or even any proper file system) everything you wanted to write to tape had to be combined into one single file. So they were packed into a .tar archive.

Gzip, on the other hand, is only able to compress a single file. So it makes sense to use them together, tar to combine all your files & directories into one file and then Gzip to compress that file.
