### Post by mcduck on 2009-04-16
1. Go to System/PReferences/Appearance, and on the fonts-tab click "Details" to get to advanced font settings.

2. Set the correct DPI for your display (This depends on used resolution & physical dimensions of your display, so don't just use some random value but calculate the correct instead.)

3. Set font sizes to suitable.

4. Now open a file manager, and go to Edit/Preferences.

5. Under the View-tab you'll find options for default zoom level. Increase that until your desktop/file manager icons are large enough.

If these changes are not enough, you can do a bit more by hand-editing your GTK themes to use larger icons in menus etc, but in that case you really should just switch to smaller resolution.
