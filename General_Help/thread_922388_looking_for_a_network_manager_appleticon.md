### Post by robz0rz on 2008-09-17
The only way I know off to get a network manager applet is to add the **Notification Area** to the panel. If you don't have the network icon in that area yet, press *Alt+F2* and type in **nm-applet**.

If **nm-applet** doesn't start when you login, add it to your session by going to *System -> Preferences -> Sessions* and adding it there.
