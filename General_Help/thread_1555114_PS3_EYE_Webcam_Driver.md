### Post by lukeiamyourfather on 2010-08-17
There is no driver like you'd get for Windows, it will either work or it won't depending on if the device is designed to work with the video capture API in Linux.

[http://en.wikipedia.org/wiki/Video4Linux](http://en.wikipedia.org/wiki/Video4Linux)

Search around to see if someone already has one working or just plug one into a system to find out. It should work with the live disc if you just need to test without installing Ubuntu.
