### Post by jocko on 2009-02-21
Try adding this to the "ServerFlags" section in xorg.conf:
```
Section "ServerFlags"
    Option              "Xinerama" "0"
[COLOR=Blue]    Option      "blank time" "0"
    Option      "standby time" "0"
    Option      "suspend time" "0"
    Option      "off time" "600"[/COLOR]
EndSection
```
