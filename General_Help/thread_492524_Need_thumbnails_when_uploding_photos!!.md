### Post by oldmanstan on 2007-07-04
assuming you're using nautilus (in gnome, default for ubuntu), do this: view menu >> view as icons

unless you've changed something that should work

if it doesn't work, hit up this: edit menu >> preferences >> preview tab >> set the "show thumbnails" dropdown menu to "local files only" and adjust the size (below it) so that it is big enough for your files (if your files are all around 3MB for example set it to 5MB or something bigger so your files will show up as thumbnails)
