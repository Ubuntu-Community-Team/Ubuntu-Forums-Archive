### Post by dldseneviratne on 2008-07-03
Hello all,
I found the solution for this and thought of sharing it with everyone. Incase if you loose your partition table or if you delete a partition by mistake, you can recover all of them by using TestDisk at [http://www.cgsecurity.org/wiki/TestDisk_Download]("http://www.cgsecurity.org/wiki/TestDisk_Download"). It works on both Linux and Windows. I used Windows XP to get my partitions back.

Thank you very much to all the TestDisk developers! For a minute I thought I lost all my documents on all 4 partitions.

Cheers! :)
