Yes, try sudodus's method; once you are used to it you'll find it so much speedier than other methods.

But if you want to use the keyboard you need **Ctrl+Shift+C** to copy and **Ctrl+Shift+V** to paste in terminal.  It has to be that way as Ctrl+C is the keyboard shortcut for "cancel".
