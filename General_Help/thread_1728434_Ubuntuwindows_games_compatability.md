### Post by Mark Phelps on 2011-04-14
> **dustcap said:**
> Will they run OK with Ubuntu? Or do I need to add any other programs to play them?

They will not run in "native" mode on Ubuntu because they are Windows games, not Linux.  They require a compatibility layer, known as Wine, in order to run.

But ... experience with running stuff in Wine varies greatly by individual game and version.  Some work great.  Some don't work at all.

You should go to the WineHQ site linked below and do searches for the games and versions you want to run to see their ratings:

[http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true]("http://appdb.winehq.org/objectManager.php?sClass=application&sTitle=Browse%20Applications&sOrderBy=appName&bAscending=true")
