### Post by lovinglinux on 2010-02-27
I wouldn't bother. America's Army released version 3 a while back, for Windows only. Among the new features, they completely changed the authentication servers/method, so it seems you won't be able to authenticate the Linux version anymore (which is 2.5). So, no authentication, no servers to play.

I haven't tested the Linux version recently, but if you take a look at the [AA 2 Authentication and Account Support](http://forum.americasarmy.com/viewforum.php?f=81) forum, you will notice that all threads are locked.

If you really want to play it, install Windows on dual boot and install the latest version of America's Army.

You could also try to install via wine, but IMO it doesn't worth it. PunkBuster doesn't work at all, so you are only able to join servers that are not using it. Not to mention the usual mouse issues.

You can also get more info and help at [http://forum.americasarmy.com/viewforum.php?f=59](http://forum.americasarmy.com/viewforum.php?f=59)
