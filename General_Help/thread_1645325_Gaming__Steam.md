### Post by themusicalduck on 2010-12-14
In theory Wine is able to run Windows applications, but in general it can't run all of them.

This is particularly applicable to games, only a few games will run really well with Wine and many others will run with tweaks, but many won't run at all. Generally the performance will be lower than if it was on Windows. The more modern the game, the less likely it is to run well on Wine.

Steam does run in Ubuntu, but in my experience it runs quite slowly and is a bit of a hassle.

It's worth searching for an application on [http://appdb.winehq.org/](http://appdb.winehq.org/) to see if it will run or if you need to tweak it. What games do you want to run?

Also, there are lot's of good Linux native games. There are tons in the Software Centre and some you can buy. To buy, I recommend World of Goo, Savage 2 and anything by Frictional Games.
