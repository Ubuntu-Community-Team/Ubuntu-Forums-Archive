### Post by AdamShumpisxXx on 2009-05-01
By merely burning the ISO file to a DVD you will not overwrite any files on your HDD. If you are using Windows use IMGBURN to accomplish this:

[http://www.imgburn.com/](http://www.imgburn.com/)

Make sure to install and run it as Administrator if you're running Vista. I know how picky it can be.

If you are running Ubuntu however all you need to do is insert a blank DVD into your DVD Writer drive, right click your ISO file, and choose Write To Disc in the options that come up. Then choose the speed you'd like to write it at and continue.
