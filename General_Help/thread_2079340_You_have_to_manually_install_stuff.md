### Post by MG&amp;TL on 2012-11-01
Your friend is mistaken, or at least inaccurate.

Ubuntu has a *package manager* which handles all of the things to do with installing and uninstalling software. The software in the package manager will be enough for 99.9% of your needs. It also has a friendly interface (the software centre) so that you can browse the applications available.

For the 0.1% of the time you need something not (yet) in the package manager, you can still download stuff off the internet. However, it's sometimes a little more challenging as not everyone loves penguins. :(

It's a lot quicker and easier once you get used to it.

Okay, install oracles java, use webupd8team's strikinly simple method, you can thank the guys at Java for not letting us put their Java in our repositoires for having to do this workaround.

sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update 
sudo apt-get install oracle-java7-installer
```This is where I got that method from Wubupd8team's Method: [http://www.webupd8.org/2012/01/install-oracle-java-jdk-7-in-ubuntu-via.html](http://www.webupd8.org/2012/01/install-oracle-java-jdk-7-in-ubuntu-via.html)
