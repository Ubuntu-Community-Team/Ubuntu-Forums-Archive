### Post by lloyd_b on 2006-10-27
Highlight the text with the mouse, move to the vim window, make sure that you're in insert mode, then:

1. If a three-button mouse, click the middle button
2. If a two button mouse, click both buttons at once.

This also works to copy from a terminal window to an X app, or from one terminal window to another...
