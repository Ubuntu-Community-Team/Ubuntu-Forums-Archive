### Post by ftabor on 2009-08-03
> **masoud23 said:**
> Hello

I have just installed virtualbox 3.0 but i can't find it! where shuold it be and how do i start it?

Try going to System>Preferences>Main Menu.  Select the System Tools Category on the left.  On the right, Virtual Box should be a menu item.  If it is, uncheck it, then check it back.  Now it should show as a menu Item under Applications>System Tools.

If not, go back into main menu and create a new item in System Tools.  The command line will be VirtualBox.
