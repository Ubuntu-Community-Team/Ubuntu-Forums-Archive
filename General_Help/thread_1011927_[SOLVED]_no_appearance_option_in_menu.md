### Post by joey-elijah on 2008-12-18
i right clicked on the see through panel and it gave me the option to change the wallpaper, which brings up the appearance menu.

To actually get the appearance menu entry you have to enable it in the menu editor... which also ships switched off.

So add/remove > install menu editor > run it > enable the real 'menu' in the system/preference pane

now navigate to the menu and you can enable 'appearance' to show.

long winded but yarr. it allows you to install custom themes, icons etc
