### Post by mrjohnston on 2006-12-28
You should be able to go to System>Preferences>Keyboard Shortcut and find the volume up and volume down entries.  You can then click beside them where it should now say "Disable" and then hit the key you want to make the volume go up/down.  This will set that key to change the volume like you want.

If you deleted the entries themselves I would assume those may be in gconf somewhere.  There's got to be either than or a config file somewhere with them listed.  I would think the easiest bet would be to purge uninstall the package and reinstall to try and get everything back.  So then the question is which package contains it?  gnome-system-tools?
