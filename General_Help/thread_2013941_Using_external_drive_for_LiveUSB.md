### Post by oldfred on 2012-07-01
Most of the installers assume a USB flash drive and totally erase it. If it is another hard drive then you should not use the installers. 

While not as easy as the installers the procedure is relatively easy if you just follow the instructions. Once you learn how easy it is then you can boot any or all the ISOs without installing to flash or CD. You do need a partition  for the ISO and grub2's boot loader. 

This will boot an ISO from a hard drive.
ISO Booting with Grub 2 from Hard drive - drs305
[http://ubuntuforums.org/showthread.php?t=1549847](http://ubuntuforums.org/showthread.php?t=1549847)
