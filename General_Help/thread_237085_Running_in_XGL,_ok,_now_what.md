### Post by BitTorrentBuddha on 2006-08-15
Run this command: ```
gksu gedit /usr/bin/startcompiz
``` Then paste all of this into it and save it: ```
#!/bin/sh 
killall gnome-window-decorator 
wait

gnome-window-decorator & 
compiz --replace gconf &
cgwd --replace & 
```
Then make it executable and run it:```
sudo chmod +x /usr/bin/startcompiz
/usr/bin/startcompiz
```You should have compiz running now.
