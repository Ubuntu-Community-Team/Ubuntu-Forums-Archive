I would like to create a short-cut command to the keyboard short-cut program.  What would be the command to do this?

---

### Post by soro2005 on 2009-04-27
```
gnome-keybinding-properties
```
You can find the commands of the items in the menu by opening System --> Preferences --> Main Menu and looking up the "Properties" of the launcher you're interested in. Or you open Gedit and just drag the item you're interested in from the menu onto the Gedit window. Might work with other programs as well.
