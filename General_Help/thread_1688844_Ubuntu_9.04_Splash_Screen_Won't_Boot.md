### Post by SexySara on 2011-02-16
Yah sure :)

I didn't know the difference between the Grub boot screen and the splash screen. I had uninstalled Usplash and thus it never showed up on Start-Up-Manager. I reinstalled Usplash and when I opened Start-Up-Manager that option was there to change the splash screen. I then set the splash screen to the default settings and now everything works as its supposed to.
