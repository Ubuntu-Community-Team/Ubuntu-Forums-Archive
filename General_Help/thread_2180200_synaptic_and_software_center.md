### Post by Dreamer Fithp Apprentice on 2013-10-15
> **Vaphell said:**
> don't delete software center, just remove its icon from view and forget it exists. Problem solved! :-)
There are bazillion programs installed in your system you will probably never use, do you want to remove them too?
I don't disagree but I think it should be noted there are circumstances when it would be desirable to remove it. Such as if you are trying to make the system take up as little drive space as possible, perhaps because you really need every last byte on that drive or because you plan to remaster it to a cd. So it depends on what kind of "fat" you want to trim. As far as I know SC doesn't use any resources except drive space when you aren't actually useing it but sometimes drive space CAN be that tight, especially if the system is to be remastered to a live cd. The only thing I can say for SC over Synaptic is that some people might prefer it for browsing because it is less cluttered with libraries and such, although I believe Synaptic can give you a little more information.
