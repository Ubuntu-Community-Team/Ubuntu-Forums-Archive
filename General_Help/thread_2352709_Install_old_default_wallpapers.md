### Post by deadflowr on 2017-02-15
warty-final-ubuntu.png is in ubuntu-wallpapers.
It's the only wallpaper in that package, really
(newer releases contain a secondary grey version of it; xenial has Xerus_Wallpaper_Grey_4096x2304.png, fwiw)

Afaik it gets updated for each release, so if you want the version from another you'll need to dig for it.
I'd look through the old-release archives for it:
[http://old-releases.ubuntu.com/ubuntu/pool/main/u/ubuntu-wallpapers/]("http://old-releases.ubuntu.com/ubuntu/pool/main/u/ubuntu-wallpapers/")



FTR each release gets two wallpaper packages, one is the ubuntu-wallpapers package, and the other is a release-specific package, typically called ubuntu-wallpapers-version-name-here.
The second release-specific package is typically available beyond the initial release, so even on 16.04 you can install the ubuntu-wallpapers-lucid package from 10.04.

Hope that makes some sense
