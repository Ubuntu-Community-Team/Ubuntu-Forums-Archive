### Post by Mark Phelps on 2010-05-21
I don't know what you want to end up with, but you do know that "PE" is not a full-blown installation of Windows, right?  It's a minimalist version designed primarily for installation and troubleshooting.

So, if your goal is to have a complete MS Windows install resident on a USB stick -- it can't be done.  MS doesn't want folks walking around with "portable" installations of their OSs.
