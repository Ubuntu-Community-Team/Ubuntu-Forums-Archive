### Post by fragos on 2007-07-11
There are a number of factors that impact line spaceing.  Mixing fonts in the same line can impact the line spacing even if the have the same numeric size.  Also paragraph spacing is separately settable for each style.  Those would be the areas I'd check first, styles and fonts.
