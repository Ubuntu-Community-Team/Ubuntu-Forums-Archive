### Post by cmost on 2007-01-11
To be more specific, Fedora Core has built a front-end to Compiz on AIGLX (which is the rendering solution preferred by Fedora - the alternative is XGL.)  With Ubuntu Edgy Eft, the version of X.org supplied is setup with AIGLX active by default, however, there is no compositing manager installed (i.e., Compiz or Beryl) and Ubuntu has not developed any special front-ends to manage these features (though, it's reported that the next version of Ubuntu may have these utilities.)  The other respondent to your post pointed out the thread you should read to get Compiz or Beryl installed and running on your system.  Personally, I use Beryl which has many more features than Compiz but is perhaps a bit more cutting edge (read slightly less stable.)  Good luck!
