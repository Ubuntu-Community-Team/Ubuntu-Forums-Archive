### Post by Bashing-om on 2014-07-31
leslie3; Well,

> 
 will it remove everything?

Depends; What one can do is when doing an initial clean install is set up your /home directory as a separate partition, and in the new install choose NOT to format that partition. In this situation, IF all goes well, that /home partition will not be touched .. ( that does not take away from the need to keep good backups, anyway ).
This procedure is couple of steps above "new user" level; many no longer advocate to make up this separate /home; user preference rules.

Else, yeah .. the installer is going to over-write all data and install anew .
