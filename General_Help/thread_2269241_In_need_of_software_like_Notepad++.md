### Post by Holger_Gehrke on 2015-03-14
With the right settings and a few plugins the editor included with Ubuntu - gedit - is actually quite powerful, and showing line numbers is just a setting that's available even without any plugin. But if you have your heart set on Notepad++ there is a Linux lookalike called Notepadqq. Instructions for installation can be found [here]("http://sourcedigit.com/13031-install-notepadqq-notepad-text-editor-ubuntu-14-04-linux-mint-17/").
