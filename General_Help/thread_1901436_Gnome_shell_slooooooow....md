### Post by jontheid on 2012-01-02
Why did you install gnome shell?
The whole point of Lubuntu is that it is light and fast - a large part of this is that it uses LXDE for the desktop rather than gnome.
You don't need gnome shell to use docks in Lubuntu either.

If you want Lubuntu with some effects I suggest just using normal Lubuntu and then installing compiz. Should work fine on your notebook - I'm using Lubuntu + Compiz on a Toshiba L30 with 512MB of ram and it is fast and smooth.
