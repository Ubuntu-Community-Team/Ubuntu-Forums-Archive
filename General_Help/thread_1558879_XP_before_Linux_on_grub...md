### Post by drs305 on 2010-08-22
In synaptic, make sure the *universe* repository is enabled, and look for "startupmanager". 

SUM can change the default OS, which is probably what you want. If you want the Windows entry to be at the top of the menu (which isn't necessary to have it boot by default), check post #5 in this thread:
[http://ubuntuforums.org/showthread.php?t=1556200]("http://ubuntuforums.org/showthread.php?t=1556200")
