### Post by yancek on 2018-06-12
If you successfully created the iso of your installeld system with PinGuy Builder and have no install option, that is because you did not include the installer before creating the iso.  THere isn't much point in having an installer on an installed system.  YOu would first need to install the Ubiquity installer and run PinGuy again to create a new iso.
