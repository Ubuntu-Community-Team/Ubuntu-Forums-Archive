### Post by C.S.Cameron on 2010-12-30
Windows 7 install USB drive is easy using Ubuntu.

Format USB NTFS using gparted.
Set boot flag using gparted.
Copy data from DVD to USB using nautilus.
or 
Extract iso to USB using archive manager.
