### Post by deprofundis on 2008-12-27
Alt-Rightclick pulls up the window menu even though I have the Window Menu option disabled in Compiz Settings Manager (under General Options -> Keybindings) and it's set for Alt-Space under System -> Preferences -> Keyboard Shortcuts.  Alt-Space does nothing once I disabled the shortcut in Compiz Settings Manager, but Alt-RMB still pulls up the Window Menu while I'd rather it didn't so I can use the binding in Blender.
Any ideas where else to look?

If it's any added hint (or confusion), all mouse clicks were mapped as ALT-clicks (without me holding down ALT at all - VERY annoying) until I removed the check from "Enable Integration into the desktop environment" within the Compiz Settings Manager Preferences.  All Compiz bindings seemed to still work after this, but my mouse buttons started working again without the implicit ALT.
