### Post by Pipps on 2010-01-06
**Background**
I would like to begin backing up my documents and electronic working papers which are regularly accessed and updated between three computers. 

My files are entirely in either odt, ods, txt and pdf format.
My three computers are each running: Ubuntu Desktop, Ubuntu Netbook Remix and Win XP.

I have 1GB of webspace which I rent for a nominal fee each month in order to run a simple website. My website currently consists of three holding pages, and so occupies only a matter of kilobytes. I therefore have almost 1GB available for online backup purposes if it would be require.

I regret to say that I have no experience of servers or internet protocol. I therefore have a few questions, which I hope won't seem too simple.

**Questions**
1. Would it be possible to use this 1GB server space to run an online backup server which coordinates the files between the three computers?

2. Would it be necessary to install specific software on the 1GB server in order to run such a backup operation? What would be the software of choice be for these purposes?

3. Would it be necessary to reformat this server to a specific file-system format in order for any such backup to work? Would such file-system reformatting usually be expected to be possible for a simple account user renting space on someone's server?

4. Would it be possible for such a backup server to operate automatically and periodically coordinate all files in the specified directories between the three computers, requiring little user intervention once it is up and running?

5. Would setting-up such a Linux backup server be difficult for the uneducated Linux enthusiast? Are there any comprehensive guides which can assist? Or has anyone worked on a similar project themselves and be able to provide any brief guidance on the steps which should be taken?

Any comments, thoughts help or advice would be much appreciate. Thank you!

### Post by akakingess on 2010-01-06
My initial response would be not to mess with the formatting or anything else on the server, but rather centrally back up all the docs on a local computer and then take that whole directory and upload to the server. Not knowing quite all of the circumstances though, there may be a better option, but getting three computers to backup online (I assume you were going to just FTP the files into a directory on your rented space?) could be a little tricky unless the hosting company offers such a tool.
