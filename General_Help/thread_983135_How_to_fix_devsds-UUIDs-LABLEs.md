### Post by vanadium on 2008-11-16
All looks normal until now. Look also at the output of the mount command ("mount"). If that is also normal, then do not worry with the output of Kinfo center. To check wether your swap is active, use the command

cat /proc/swaps

("swapon -s" does the same thing). The active swaps are listed. My output is like

```

:~$ cat /proc/swaps
Filename				Type		Size	Used	Priority
/dev/sda5                               partition	3028212	39764	-1

```

Yours should list /dev/sda1
