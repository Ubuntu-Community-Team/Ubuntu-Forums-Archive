### Post by duffrecords on 2008-11-10
The "skipping incompatible" library error tends to happen when you're building a 32-bit app on a 64-bit system and the linker can only find 64-bit libraries.  Try installing these and see if it helps:```
sudo apt-get install gcc-multilib g++-multilib ia32-libs
```
