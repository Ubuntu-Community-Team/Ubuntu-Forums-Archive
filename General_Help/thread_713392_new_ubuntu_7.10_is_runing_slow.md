### Post by Nzer24 on 2008-03-02
That really doesn't make much sense. If your mom's computer has a new graphics card, it might not be supported. That would cause considerable lag because all graphics processes would have to be passed through the processor. If that is true, use the Restricted Drivers Manager to install them.

Other than that, I would try reinstalling Ubuntu. Never heard of anything like this before. Also, you might want to write a new CD on a lower speed. I had some issues with writing liveCDs too fast and having them crash and install incorrectly.
