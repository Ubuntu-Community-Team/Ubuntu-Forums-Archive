### Post by ankspo71 on 2010-06-30
Hi.
I'm sure other people here will have better answers soon, but have you considered using docks? "cairo-dock", "docky", and "avant-window-navigator" (awn) are the first 3 that come to mind. Some of these docks can be totally transparent leaving only the icons visible. some can be set to auto-hide and have special effects too. These will give you the look and feel of a Mac desktop. 

"DockbarX" is a nice way to add launchers and save space on your gnome panel. 

You also have the option to try out different desktop environments. I know 3 that might do what you want it to do... (see below)

There are 2 netbook editions that are designed to have simple access to applications (they also are better for smaller screens but that isn't what you are after). Ubuntu and Kubuntu (w/kde) both have netbook versions. You can search the internet to see what "ubuntu-netbook-edition" and "kubuntu-netbook-edition" looks like. 

Then there is the KDE desktop environment that lets you create application launcher "widgets" that can be locked in place, simply by right clicking on any start menu entry and selecting "add to desktop" (providing your widgets are unlocked before doing so). Conky doesn't have the same type of transparency effect with the KDE desktop though.](*,) 

Hope I gave you an idea or two.

Here's a link to themes. You'll find all kinds of cool stuff. Backgrounds, window borders, pointers, icon themes, etc. Not exactly what you where asking about but if your going to be obsessing about looks then this is the place to start. [http://gnome-look.org/](http://gnome-look.org/)

Another great way to obsess about looks is to enable Advanced Effects in Appearances and start playing with this:

```
sudo apt-get install compizconfig-settings-manager
```

and

```
sudo apt-get install compiz-fusion-plugins-extras
```
