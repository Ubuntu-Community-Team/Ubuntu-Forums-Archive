### Post by tredegar on 2011-05-24
If you are using the computer with the mains cable (wire) and the battery is flat or disconnected, if you unplug the wire _there is no time_ for the computer to hibernate. It stops immediately. There is no fix for this.

When running on battery, without the wire, you can configure the behaviour "when battery power is critically low" under System - Preferences - Power Managemen.
