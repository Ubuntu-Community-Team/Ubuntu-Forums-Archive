### Post by PhilGil on 2012-07-01
I'm not aware of a replacement for fences in the Gnome desktop environment. Gnome is designed to discourage you from using the desktop as a catch all for files and shortcuts.

If fences-like functionality is a necessity you might want to check out KDE.
