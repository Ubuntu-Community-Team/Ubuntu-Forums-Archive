### Post by colsandurz on 2008-12-15
If you want to get started ahead of time, then look up the TCP/IP stack.  You can learn it top down or bottom up - it doesn't matter as long as you go in order.  

Application (like HTTP, FTP) layer
Transport (TCP/UDP) layer
Internet Protocol layer 
Link Layer

You'll learn what makes up a 'packet,' how headers work, the problems with the unreliable channel, routing, DHCP, DNS, etc.
