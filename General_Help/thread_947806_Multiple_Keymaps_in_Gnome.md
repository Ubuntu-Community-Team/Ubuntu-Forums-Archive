### Post by Ub1476 on 2008-10-14
You can add a new layout from System>Preferences>Keyboard, and at least choose a key combination (alt+shift for instance) which changes the layout. I use that when I need Norwegian letters for my English keyboard.

So I suppose if you just add a new keyboard layout for your user that should be fixed..
