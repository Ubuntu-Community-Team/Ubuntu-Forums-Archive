AFAIK you cannot "make" apps use a specific network interface.

You can set up VMs to use specific interfaces and only run specific apps in those VMs.

Also, you say you connect your laptop is "physically connected to multiple networks", if this occurs simultaneously I would like to know how many network interfaces this laptop actually has?

If you actually only have **one** network connection then all you have is a routing issue.
