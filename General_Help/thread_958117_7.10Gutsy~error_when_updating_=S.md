### Post by plucky on 2008-10-25
You need to add the GPG key from a terminal with```
sudo apt-get update && sudo apt-get install medibuntu-keyring && sudo apt-get update
```


See the [Medibuntu Community documentation](https://help.ubuntu.com/community/Medibuntu)

Good Luck
