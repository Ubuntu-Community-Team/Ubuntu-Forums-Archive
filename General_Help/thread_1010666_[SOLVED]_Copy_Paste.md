### Post by kpkeerthi on 2008-12-14
This is on purpose for **security**. If you copy a 'content' to clipboard and then close the source application, linux will erase the clipboard content, unless the source application is still running.

Just use a clipboard manager if you want the copied content to live longer than the application from where you copied it from.

### Post by 2hot6ft2 on 2008-12-14
glipper - ever close a window after copying something to the clipboard? Ooops- it removes the item from the clipboard. Glipper will remember it for you (as long as it's text). You can even set how many entries you wish to recall.

Get it from Synaptic
System>Administration>Synaptic Package Manager
