### Post by ragiete on 2011-07-23
Hi folks!

My mother has a laptop with these specs:
- Intel Pentium 3 @ 896 MHz
- 256 MB RAM
- 27,9 GB HDD

The laptop is now running Windows XP. When reading  news online, everything hangs. XP is just not the OS to run on a laptop with these specs.

As an Ubuntu-freak myself who's always trying to get people to switch to Ubuntu, I wanted to throw Windows off her computer and install an Ubuntu-based OS instead. Though, the original Ubuntu version surely is a no-go on these specs, so I started looking into Ubuntu derivatives.

PuppyLinux seemed like the perfect match, but the problem is; it does not play nice with the wireless network adapter. I even tried to use the driver, which I got from the manufacturers website, with ndiswrapper and that did not fix the problem either. I asked the PuppyLinux community about it but they didn't know what was going on either. The adapter itself is not faulty, seeing as it works perfectly with Windows XP.

I'd like to try other Ubuntu derivatives which might be able to connect to the Internet with that adapter. Other than PuppyLinux, I don't really know which ones are suitable for that machine though.

I looked into Xubuntu but I read at its website that it can run on 256 MB RAM but that 512 MB is strongly recommended. I want the OS to be responsive so I think installing Xubuntu ain't a good idea either.

Anyone here who has some suggestions?

Thanks in advance!
