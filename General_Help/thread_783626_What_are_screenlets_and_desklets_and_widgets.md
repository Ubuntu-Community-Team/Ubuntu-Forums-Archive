### Post by mano cazalet on 2008-05-05
[Screenlets]("http://www.screenlets.org/index.php/Home")

you can search in Synaptic or ```
sudo apt-get install screenlets
``` in terminal

[Gdesklets]("http://www.gdesklets.de/")

in synaptics also or ```
sudo apt-get install gdesklets
```

both are widget packs, eyecandy to show weather, info about your system, launchers, and many many other features
