- free
- opensource
- php, perl, java etc (java preferred)

- provisions domains
- users per domain
- home directories
  - user based quotas including mail, files etc
- user self admin and web based password reset
- enable / disable login

- apache and/or tomcat website support
- dns management

- mail quotas
