### Post by Yellow Pasque on 2008-10-30
It doesn't matter too much. You can install the KDE desktop on Ubuntu if you'd like (and vice-versa).

> Also, will OpenOffice run on KDE?
Yes. You can run GNOME/GTK-based apps in KDE with the appropriate libraries installed. Also, Ubuntu has a package to make OO.o more "KDEish" [http://packages.ubuntu.com/intrepid/openoffice.org-kde](http://packages.ubuntu.com/intrepid/openoffice.org-kde)

### Post by snova on 2008-11-03
All Gnome programs will work under Kubuntu, and vice-versa. There is no technical difference. The *only* difference between Ubuntu and Kubuntu is the set of packages installed *by default*. It is also possible to go from one to the other.
