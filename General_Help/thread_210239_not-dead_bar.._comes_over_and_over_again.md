### Post by ajgreeny on 2006-07-06
I assume this is konqueror as a file manager that you have shown us.
If that's right, set the view how you want it then go to:-
Settings/Save view profile "kubuntu file manager" and follow the prompts to save how you wish, even down to window size and folder that it opens on.

The navigation panel can be toggled on and off with F9.
