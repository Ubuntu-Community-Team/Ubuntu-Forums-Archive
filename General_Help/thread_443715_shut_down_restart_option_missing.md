### Post by sawjew on 2007-05-14
You are probably now using kdm (the KDE display manager) rather than gdm (the Gnome display manager) therefore you will not have all the logout and shutdown options when using Gnome, only when you are using kde.  If you follow this link [http://ubuntuforums.org/showthread.php?t=416887&highlight=kdm+gdm]("http://ubuntuforums.org/showthread.php?t=416887&highlight=kdm+gdm")
 you will find out how to revert back to gdm in the 3rd post by Aysiu.

I had the same problem once before and this resolved it.
