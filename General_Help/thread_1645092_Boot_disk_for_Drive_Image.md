### Post by dandnsmith on 2010-12-14
I don't know which Drive Image product you are referring to - as always, in fields like this, there are a number of products which you might mean.
That said, however, most of them have proprietary formats for storing the information, so I wouldn't rate you chances as being very good.
The product itself, if any good, should have a standalone CD (either included with it, or creatable with the product), which can be used to restore a failed disk.
