### Post by pro003 on 2009-05-10
you can create new panel by right clicking on your below panel and then just add things you had before, again right click on the new panel ad select **add to panel**...

as far as for network manager - you can add it to** startup applications** in **system > preferences** click add then type name Network Manager Applet and the command is **nm-applet --sm-disable**
