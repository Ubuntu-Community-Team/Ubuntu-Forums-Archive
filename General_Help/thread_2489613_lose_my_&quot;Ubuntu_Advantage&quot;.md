### Post by guiverc on 2023-08-06
You didn't provide any release details (*though your tag shows 22.04*), but remove of the *desktop meta package* did occur for a [number of users if they upgraded during a specific window]("https://bugs.launchpad.net/ubuntu/+source/xorg/+bug/2030262").  You could resolve the issue by just re-installing that *desktop meta package*, with re-install not being required.

### Post by MAFoElffen on 2023-08-07
Here is how those 5 subscription are counted... If you look closely, it counts what has been active in the "last 24 hours".

What I was told by the Ubuntu Pro Team, is if, for some reason there is a problem, to contact them and they will detach the old machine subscription for people, from their subscriptions. 

There was a time when it first came out when, if you had a problem, you would have to generate a new machine ID, which a new install would generate that new machine id, so you should be good on both perspectives of that. 

If I have any question with Ubuntu Pro, I go to: [https://ubuntu.com/pro](https://ubuntu.com/pro), and use the "Chat Now" dialog.
