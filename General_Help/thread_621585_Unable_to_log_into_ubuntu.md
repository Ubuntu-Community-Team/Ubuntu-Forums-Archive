### Post by prince_niceguy on 2007-11-23
select safe gnome from the sessions options in the bottom left in the login screen and then try to login. if it works then there is some problem with the gnome profile. 

if that does not work then try to login into safe terminal.

if that does not work then boot up ubuntu into safe mode and change your password.

passwd <user name>
