### Post by Paddy Landau on 2012-06-03
I don't know how Mate works, but the default buttons for screen-shots on Ubuntu 12.04 are:

[LIST]
[*]Save: Full-screen screen-shot: PrintScreen
[*]Save: Window screen-shot: Alt+PrintScreen
[*]Clipboard: Window screen-shot: Ctrl+Alt+PrintScreen
[*]Clipboard: Area screen-shot: Shift+Ctrl+PrintScreen
[/LIST]
These can be changed in System Settings > Keyboard > Shortcuts > Screenshots. However, the screen-shot shortcuts are (at the moment) a little buggy, so I'd recommend against changing them.
