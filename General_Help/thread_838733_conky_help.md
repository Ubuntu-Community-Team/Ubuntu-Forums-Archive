### Post by spiderbatdad on 2008-06-23
You want to create/copy a file called .conkyrc to your home folder. Use ALT-f2 to run conky, or conky & from the terminal to fork the process to the background. You can also create a launcher for your panel if you like, or add conky to sessions to launch at start up.

See this thread for hundreds of examples of .conkyrc files:
[http://ubuntuforums.org/showthread.php?t=281865&highlight=.conkyrc](http://ubuntuforums.org/showthread.php?t=281865&highlight=.conkyrc)
