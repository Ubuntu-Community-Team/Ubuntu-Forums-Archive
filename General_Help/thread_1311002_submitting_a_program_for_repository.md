### Post by ikisham on 2009-11-02
You can make it available for other people if you have a PPA (Personal Package Archive) in Launchpad. then you can even keep it up-to-date.
For inclusion in the default repos I think you should ask for it in Ubuntu's Brainstorm if you have like an usability idea or you could open a bug in Launchpad as a wishlist if you just want some app to be included.

But wait more precise guidings.
