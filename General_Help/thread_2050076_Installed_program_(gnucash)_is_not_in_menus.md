I'm running 11.04 with the classic interface.

***

Solved, sort of.
O.K. I used the menu editor under System > Preferences and ADDED gnucash.
Of course I have some sort of default icon rather than the actual gnucash icon.

*****

Completely solved:
right-clicked on default icon in launch bar
selected properties
clicked on ICON in pop-up
found gnucash icon in resulting directory list and OPENED it!
