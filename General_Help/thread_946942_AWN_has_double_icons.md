### Post by brianfreytag on 2008-10-13
Go to System -> Administration -> System Monitor

Go to the Processes tab.

Click avant-window-navigaor and click end process

There is more than likely multiple processes causing this, so close all of them.

Then go to Applications -> Accessories -> Avant Window Navigator

This should take care of it for you!
