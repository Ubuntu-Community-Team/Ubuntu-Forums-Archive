### Post by lindsay7 on 2009-03-09
go to synatic find the file you have installed right click on it. go to properties, installed files. that will show you where they are. when you fine the file that starts the program, make a note of what that file is and where it is. if you right click on appications on the menu bar, choose edit menu, and fill in the boxes accordingly with the program name and start command. this will add it to the menu you pick and you can start it from there.
