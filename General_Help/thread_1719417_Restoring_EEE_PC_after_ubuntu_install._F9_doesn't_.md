### Post by Lateralis on 2011-04-02
Hi.  Sorry for the late reply.  

I've compared your results to the script with my own and I'm not sure I can really help you that much.  Trying to interpret your hard drive partitioning and failing somewhat.  Where is sda3?  

I would hazard the guess that your Windows partition is in either sda2 or sda5, but why the Bootinfo script cannot determine the filesystem is unclear to me.  The script detects all of my partitions and the filesystem types of those partitions quite happily.  

I have two questions for you.  When you installed Ubuntu did you shrink the Windows partition from the left to the right and install Ubuntu at the beginning of the disk?  And when you did the shrinking did you use the tools built into Windows or did you use GParted during the Ubuntu installation?  If you shrunk the partition from left to right using GParted and then installed Ubuntu at the start of the hard drive you may have borked the recovery partition.  

But that might not be the case.  You may want to try downloading a Windows Vista/7 boot recovery CD and try restoring the Windows boot loader.  If the Windows recovery partition is still intact the recovery CD should over write Grub and point to the correct partition to boot from.

### Post by Mark Phelps on 2011-04-02
Probably too late to help now ... but the script detected no Windows boot or loader files.  So, Windows appears to be gone.

As to booting from the recovery partition -- that's not always possible.  Depends on how the OEM configures that partition.  In some cases, they load WinPE and it IS bootable; in other cases, thy save a compressed image of the initial hard drive setup -- and it then is NOT bootable.

Your 15GB space is certainly large enough to hold a Win7 Image and some boot files -- but since you can't read it in Linux, you can't tell.

Problem with the Win7 Repair CD is that it is NOT an installer; instead, it can only repair the boot loader files.  If it DOES find a Win7 installation, then it might work; otherwise, you will have to contact the PC supplier to see if they will sell you something that can be used to restore the PC.  My guess is that it would have to be in USB form factor.
