### Post by insane_alien on 2007-09-25
Fluxbuntu is even lighter than Xubuntu. 

install the server edition of ubuntu and then type
```
sudo aptitude install fluxbox xdm
```

that'll give you the fluxbox environment, when it is installed just type 'startx' to go to the graphical environment.

icewm is also good, for that replace fluxbox with icewm in the previous command.
