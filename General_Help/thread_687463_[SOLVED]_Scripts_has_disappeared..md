### Post by Irony on 2008-02-04
Solved it - went to the open as root script in ~/.gnome2/nautilus-scripts and right clicked it and made it executable - this makes the Scripts option appear.

A few weeks back I recursively changed permissions of the home folder items so this is what must have done it.
