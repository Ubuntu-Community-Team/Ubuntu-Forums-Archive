### Post by mb_webguy on 2009-02-08
Try resizing the window.  Sometimes if I have numerous windows open at once, I'll get a window that doesn't refresh correctly.  Reducing the size of the window forces it to refresh and reduces the load on the video card, allowing it to refresh as it should.  I've noticed this usually happens with the GTK file dialog -- the file manager used by most non-KDE applications when you save or open a file.
