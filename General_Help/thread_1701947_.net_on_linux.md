Yes and no. Mono includes full support for ASP.NET, including ASP.NET MVC 1 and 2. However, you can't host MSSQL on Linux - as the name hints at, MSSQL is for Microsoft servers only. You could run MSSQL in a Windows virtual machine and connect to that from your Mono app, or a real Windows server for MSSQL, or port your app to use another database such as Postgresql or MySQL.
