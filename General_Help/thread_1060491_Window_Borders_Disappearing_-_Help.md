### Post by pikabuntu on 2009-02-04

### Post by liquidpele on 2009-02-04
I've seen this when compiz-fusion (or whatever it is on 8.04) dies...  you can even simulate it by killing off the compiz related processes while in gnome.  You can try manually restarting it via the command line, but I don't know if that will work or not.  Best bet is to log out and log back in to restart gnome.

To completely get rid of the problem, disable all the "desktop effects" so it's all disabled and the problem will probably go away - yes it's sad but I've had to do the same thing because of different problems.  Compiz is just too unstable still.
