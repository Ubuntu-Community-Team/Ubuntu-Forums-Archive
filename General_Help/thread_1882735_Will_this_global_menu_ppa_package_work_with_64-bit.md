### Post by ankspo71 on 2011-11-18
Hi,
They have a 64 bit version now.
[https://launchpad.net/~the-warl0ck-1989/+archive/xfce-appmenu-plugin/+packages](https://launchpad.net/~the-warl0ck-1989/+archive/xfce-appmenu-plugin/+packages)

It doesn't feel quite the same as using Unity though, because it doesn't move the window buttons and title bar into the panel like Unity does for maximized windows. If you are a Firefox and Thunderbird user, you will probably want to install 'firefox-globalmenu' and 'thunderbird-globalmenu' too.
