### Post by lhaeh on 2010-12-12
To use inline spell check for Firefox with the keyboard, just move the cursor to the misspelled word then press the menu key on your keyboard to access the right click menu. If the word you want isn't there, you can use the escape key to exit it without selecting any of the menu items.

IIRC, there are scripts for greasemonkey and stylish that restore right-click functionality for all websites. I think noscript does it too, and maybe there are some extensions that do only that.
