### Post by DanJC on 2008-08-31
You don't need to burn anything.  Here's how you go about this:

1. Download Wubi at [www.wubi-installer.org](www.wubi-installer.org).

2. Download the Ubuntu ISO from [www.ubuntu.com](www.ubuntu.com).

3. Place Wubi and the ISO **in the same folder.**

4. Run Wubi, set your disk space, username, distro, etc.

It should only take a few seconds, then you can reboot into Ubuntu.

Hope this helps. :)
