### Post by thacreeper on 2009-05-13
Hi There,

I had this issue.. Was fixed by removing all the files related to salutis from
/var/lib/dpkg/info/ . Basically what the previous poster has said, although
when I ran that find statement it didn't find the salutis files in that
directory. 

So:
cd /var/lib/dpkg/info/
sudo rm salutis*
sudo apt-get clean
sudo apt-get update
sudo apt-get remove salutis-connect

I did that, it went through the motions, threw some sort of error when the
files werent there in /var/lib/dpkg/info, cleaned up and now it works as
it should.. 

Hope this is of some use to someone!
