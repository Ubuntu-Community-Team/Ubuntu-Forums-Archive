### Post by Tamlynmac on 2009-01-16
I'm not quite certain what you mean. However, have you setup you applications in Firefox?

firefox/edit/preferences/applications tab

setup each file type to open individual applications.


Also set you file save location by:

firefox/edit/main tab/downloads/

check "Save files to" and assign a location where the file will be saved each time a download is started.

Hope this helps.
