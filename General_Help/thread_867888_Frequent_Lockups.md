I'm going to try installing Opensuse 11 tonight over my Windows partition since that already seems fried, just to try running it for a while to see if it is indeed an issue with my hardware or just Ubuntu itself.

Does anybody have any other suggestions I can try if it does turn out to be Ubuntu? I'd really like to keep using it as I've had nothing but good experiences with it, but I'm trying to do run some computation programs continuously and the lockups are impeding my progress.

Thanks in advance!
