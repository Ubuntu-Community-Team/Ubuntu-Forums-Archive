### Post by Cavsfan on 2013-11-27
I noticed that Bleachbit, which deletes cache, etc. was updated on Trusty 14.04. I booted into Precise 12.04 and noticed there was no way to get the updated version into Precise through it's repositories.
So, I found this by clicking in the link in Help About and found there is an update. I downloaded the deb package and installed it via the Ubuntu Software Center.
There are links for many distros of Linux as well as every active version of Ubuntu on that page.

[http://bleachbit.sourceforge.net/download/linux](http://bleachbit.sourceforge.net/download/linux)

```
cavsfan@cavsfan-desktop:~$ apt-cache policy bleachbit
```
