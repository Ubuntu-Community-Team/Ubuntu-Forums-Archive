### Post by CatKiller on 2020-03-22
Monitors turn themselves off to save power and prevent burn-in. If your monitor turns off regularly - every 15 minutes or so - when you aren't using the computer, that's what it's supposed to do. If it turns off regularly when you are using the computer then it's an issue with determining that you're using the computer. If it turns itself off sporadically then it's likely a cabling issue.
