...Unubunto...?

Anyway, I would recommend the Desktop Edition for an old laptop.

The easiest way to get it on a USB drive is to download [Unetbootin]("http://unetbootin.sourceforge.net/"). 

Format the USB drive, then run unetbootin, select Ubuntu 10.04 from the drop-down menu, and click "Ok".
