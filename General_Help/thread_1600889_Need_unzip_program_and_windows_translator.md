### Post by ivarn on 2010-10-19
1. there's no such thing as ubuntu 8.2. There's an 8.4 version (and I recommand you upgrade it as soon as possible since it's old and outdated).
2. to unzip (extract) an archive, right click it and you'll see the extracting options.
3. you'll find kaffeine in both synaptic and the software center, so you don't really have to install it manually.
EDIT, you can use google dictionary to translate things, or go to system>administration>languages and see if you can find your own language there.
