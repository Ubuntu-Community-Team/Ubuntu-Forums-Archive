### Post by coffeecat on 2011-04-13
What was the old card and what is the new? And did you install the proprietary driver for the old card?

If you haven't installed a proprietary driver you should be able to change cards and the system will autodetect the graphics on bootup. But a proprietary driver complicates things by forcing modules and a configuration file.

My guess is that you may have had an ATI card and proprietary driver before and you've fitted a nvidia, or nvidia + proprietary driver and you've fitted an ATI.

Either way, it's rescuable. Post the details and we can take it from there.
