### Post by Miaxi on 2009-03-27
Those aren't really "versions". Those releases only use different default desktop managers. 

Xubuntu = XFCE = good for older machines with 256MB RAM or less, looks like this: [http://geekgirlxx.files.wordpress.com/2007/11/xfce-desktop-on-ubuntu.png](http://geekgirlxx.files.wordpress.com/2007/11/xfce-desktop-on-ubuntu.png)
Ubuntu = Gnome = looks like this: [http://www.pendrivelinux.com/wp-content/uploads/ubuntu804.jpg](http://www.pendrivelinux.com/wp-content/uploads/ubuntu804.jpg)
Kubuntu = KDE = looks like this: [http://wiki.kubuntu-de.org/images/Kde4.1-desktop-krunner-add_widget-800.png](http://wiki.kubuntu-de.org/images/Kde4.1-desktop-krunner-add_widget-800.png)

You can still change your desktop manager later if you don't like the look.

### Post by Anthon on 2009-03-27
If you want to be/continue using Visual Studio 2008 you need to run things in VirtualBox/VMWare etc. Ubuntu Server is the only one I would not recommend as that normally does not include a head.

Full HD might be a problem under Linux. Depending on your board. I have not tried that. 

I recommend you load Ubuntu 8.10 under Vista in a VirtualMachine and try the applications out first ( Eclipse, Gimp for image processing, Firefox/Pidgin/OpenOffice, Blender for 3D). If that all works as you want you can switch from runnig Ubuntu under Vista to the other way around and then get the HD stuff to work.
