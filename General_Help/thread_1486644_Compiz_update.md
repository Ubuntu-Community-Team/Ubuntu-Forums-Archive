### Post by todak on 2010-05-18
Install compizconfig-settings-manager from the repositories.
```
sudo apt-get install compizconfig-settings-manager
``` Run the compiz configuration application by going to *System> Preferences> CompizConfig Settings Manager*. Have fun! :)

### Post by WorMzy on 2010-05-18
To get the full range of plugins, run the following:
```
sudo apt-get install compiz-fusion-plugins-extra
```
