### Post by loafimus on 2010-12-01
Just got one of these and had the same problem also.  Apparently there is an issue with the default kernel installed and the graphics chip.

The first thing you need to do is disable X from starting up at boot time.

```
sudo update-rc.d -f gdm remove
```

Go ahead and reboot so the screen doesn't turn black on you in the middle of the next couple steps.

Next make sure that all the sources are uncommented in /etc/apt/sources.list

Then update apt and the kernel

```
sudo apt-get update && sudo apt-get install linux-image-2.6.35-23-generic
```

Now we can turn X back on at bootup when that's done.

```
sudo update-rc.d gdm defaults
```

Reboot and you should be good.
