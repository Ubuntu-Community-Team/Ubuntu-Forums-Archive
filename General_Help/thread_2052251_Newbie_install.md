### Post by Petro Dawg on 2012-09-02
Wubi is the Windows UBuntu Installer.  

[http://www.ubuntu.com/download/desktop/windows-installer](http://www.ubuntu.com/download/desktop/windows-installer)

It'll install like a dual boot, but can be removed like any other program installed through Windows.  It's much safer than trying to mess with partitions.

I used it when I first tried out Ubuntu.

### Post by Lars Noodén on 2012-09-03
A Debian install CD has the advanced option of carrying out the [installation over SSH](http://princessleia.com/journal/?p=1357), but I think it is not possible to do that in Ubuntu.  So you might have to install Debian first then Ubuntu.

Really, I'd wait until you can visit.  NTFS is not suitable for daily use in anything, even Ubuntu.  Among other things it will eventually get fragmented and slow things down.
