### Post by TheFu on 2016-09-15
Java is commonly an issue. Where there any recent updates to the JRE?  Java has had issues with time before, though I haven't heard of there being any issues in the last few years. This isn't necessarily the answer: [https://stackoverflow.com/questions/6392/java-time-zone-is-messed-up](https://stackoverflow.com/questions/6392/java-time-zone-is-messed-up) - but might provide some clues.
Or be completely wrong and useless.

Do you have system monitoring setup that would record the activity upto, during and after the slowdown? Something like **Munin**? That's the issue with performance monitoring - it has to be setup BEFORE it is needed.

Elsewhere, some others have been discussing NTP problems, but those were on CentOS and 16.04.1 servers.
