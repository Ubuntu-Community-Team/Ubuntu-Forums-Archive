### Post by silkstone on 2008-08-26
System > Preferences > Appearance will get you to the themes. There's an install button there, but you should also be able to drag and drop themes into that window. If I remember correctly, you can just drag the tar.gz file without unpacking anything.

### Post by karasuman on 2008-08-26
Right-click on the Desktop.  Select "Change Desktop Background."  This opens the "Background" Tab of "Appearance Preferences", from which you can choose any other tab to change the other settings.

(I mention this because it's probably the most intuitive way to access these settings for a former Windows user.)

Choose the "Theme" tab, select a theme, and click "Customize" at the bottom.  From here, you can alter most of your theme settings.  If you have a theme or icon set to install, choose "Install" instead of "Customize."
