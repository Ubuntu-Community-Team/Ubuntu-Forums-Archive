### Post by Square on 2008-08-20
Hi,

if you've installed it via Synaptic then finding the launch command is fairly easy.  You need to right-click on the installed package then select Properties.  Then go to the installed files tab.  You'll have to find the file that's in a bin folder. In this case it's /usr/bin/magnifier. So magnifier would be the command you're after.

Hope that helps.
