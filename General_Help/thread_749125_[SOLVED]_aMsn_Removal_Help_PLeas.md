### Post by angry_johnnie on 2008-04-08
I think aMSN should be under "Third Party Software,"  NOT in synaptic.  Try to remove it from there.  As for Windows messenger, I don't know whether It will run under wine, although I personally don't think so.   If you want to give it a try, just install wine from synaptic, and then download the executable.  Suppose the name of the executable is "messenger.exe" and suppose you download it to the desktop.  Then all you'd have to do is this:

```
cd Desktop
wine messenger.exe
```

Again, I really doubt Windows Live Messenger is going to work in wine, but if you feel like trying it, that's how you do it.  

I personally prefer pidgin, but then some people do like aMSN better, as it looks similar to WLM and has some of its functionality.  

If you're only bothered because of all the  "newer version" alerts, you could just turn them off.  doesn't it have a box you can check so that it won't remind you for, at least, a week?  Besides, you wouldn't really need the absolute newest version of aMSN.  The functionality doesn't really change all that much.
