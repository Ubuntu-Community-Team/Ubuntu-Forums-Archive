### Post by macaholic on 2008-04-29
If I understand you correctly, you are mentioning the fire when he right clicks? If that is indeed what you are referring to, then you would set that by the following:
1. Enter CCSM
2. Click on Animations
3. Select Close Animation
4. Click New, select burn as the effect
5. CLick the '+' select 'type' under window type, and enter 'PopupMenu' udner window match, and select a desired duration. (I like 150)
Repeat this for 'Open Animation', and whenever you right click it will burn open and burn close.

Well, let me be more specific, perhaps that will help.

1.) Pull down the Applications Menu, choose the Accessories menu choice, and then click on Terminal.

[IMG]http://img514.imageshack.us/img514/7349/screenshotol9.png[/IMG]

2.) The Terminal program will open up and it looks like the screenshot below:

[IMG]http://img87.imageshack.us/img87/8237/screenshot1mw6.png[/IMG]

Now, your terminal window will say something a little different that ken@my-desktop but don't be alarmed. Also, Terminal is somewhere you can cause a LOT of trouble to your system if you mess up - BE CAREFUL.

3.) Now paste the command (below) exactly into the terminal window and press enter.

**sudo apt-get install compiz compizconfig-settings-manager**

There will be quite a bit of writing that will scroll by in the Terminal window and when it is finished you should now have a Advanced Desktop Effects Settings choice under the System -> Preferences -> menu.

I hope this is a little clearer.
