### Post by AT-AT on 2006-10-08
Why does Ubuntu have to create a folder called Trash on a removable drive (such as a USB flash drive)? Sure Ubuntu considers the trash as not takin up space but on Windows, Mac, etc it considers a folder taking up the same amount space as its original contents. Is there a way to force Ubuntu to not make these folders so I dont have to constantly erase them?

---

### Post by meng on 2006-10-08
From a file browser:
Edit > Preferences > Behavior > include an option to delete bypassing Trash.

---

### Post by z-vet on 2006-10-09
Or use Ctrl+Delete on choosen files instead of Delete, it will bypass Trash as well.
