### Post by Garyu on 2006-02-14
How do you get Ubuntu? At the top right of your screen when you are in this forum you see a link aptly named "Download Ubuntu". Follow the directions, burn it to a CD and presto - you are up and running. :)

I have an AMD64 processor, and I tried the 64-bit version of Ubuntu. Many things needs to be developed especially for the 64-bit processor, so it takes a lot longer for new drivers and codecs and applications and whatever to hit the 64-bit version. I couldn't get sound to work properly and there were a few other issues with software I tried. Also, the performance boost isn't that big.

Instead, I would recommend you do what I did - install the i386 version. It still has the best support for everything and it works just as well. Unless you are running a server and really care about that extra second you can squeeze out in performance from the 64-bit processor. For a regular user though, go with the i386 version.

As far as I have heard both i386 and 64-bit works just as well with Intel processors as with AMD. Apparently Intel is actually using AMD technology in their processors, so it's basically the same processor anyway (or so I've heard).

### Post by mgmiller on 2006-02-14
Yes, it's a boot from CD.  Make sure your BIOS is set to boot from CD before your hard drive.  The install CD can take care of partitioning your drive to dual boot windows/Ubuntu.  I have never done that, I always only installed Ubuntu by itself, but I understand it's pretty easy to follow along on the install CD.  After you have been using it as i386 for a while, youi can upgrade the kernel to 686 and pick up some performance.  I won't go into that here, the object is to get up and running first, tweak later if you feel the need.  I am currently running an AMD 64 wilth the 686 kernel and am very happy with the performance.

One other thing you can try first, is burning a "live CD".  and booting from that.  It will run Ubuntu off the CD and not touch your hard drive.  It lets you "test drive" the system and see if all your hardware is compatible.  Just be aware that because it's running off a CD, it's a lot slower than it would be if running from your hard drive.

One last thing.  you need to create the disks by having the .iso file you downloaded expand onto the CD, not just copy the .iso file to the disk.  If you have Roxio or Nero installed, you should just be able to double click the .iso file and Roxio/Nero will burn it the the CD correctly.

Have fun.  Like it says on the bottom of Frisbees....Flat flip flies straight, experiment!
