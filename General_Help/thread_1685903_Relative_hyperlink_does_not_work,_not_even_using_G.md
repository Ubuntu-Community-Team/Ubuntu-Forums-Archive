### Post by PCaddicted on 2011-04-18
I tried it and the only solution was...to open the file with OpenOffice,because indeed there seems no way to place links in a txt file.In the interface of OpenOffice go to Insert>Hyperlink,click on the "Target in document" buton,customize the options as desired and click Apply and then Close.Remember to save the file in the right format.And you'll have to save it separately if you do as described by me,because txt files do not(seem to) support links.
