### Post by Mud.Knee.Havoc on 2009-03-04
If you want the same kind of thing as Limewire, you should download Frostwire.  I forget if it's available in the repos... it's definitely at getdeb.net.

The best file downloader (for normal stuff you find on the web and want to download), in my opinion, is wget.  It's easy to use.  Just go onto the command line and type wget and paste the url.  You can resume broken transfers, which is a great feature.  Most people want glitzy graphical programs for even simple stuff like this, though.  (Ubuntu does offer various graphical downloaders!)
