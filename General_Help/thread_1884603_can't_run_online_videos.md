### Post by Epinephrin3 on 2011-11-22
What web browser are you using? In Firefox there's a very useful add-on called Flash-Aid which removes all previous flash plugins then installs the latest version with a few optimisation tweaks which can help solve some issues. Also you could try disabling hardware acceleration by right clicking on a flash video in youtube, going to settings, display tab, then untick 'Enable Hardware Acceleration'.

Link to Flash-Aid: [https://addons.mozilla.org/en-US/firefox/addon/flash-aid/](https://addons.mozilla.org/en-US/firefox/addon/flash-aid/)
