### Post by Crusty Old Fart on 2010-11-11
If this was happening to me, I think I would re-install everything but my /home directory using the alternate installation CD. But, if you don't have your /home directory in a separate partition or logical volume, you won't be able to exclude it. In such a case you might be able to access it through either the alternate installation CD or the recovery kernel, and copy it to a safe place before re-installing.
