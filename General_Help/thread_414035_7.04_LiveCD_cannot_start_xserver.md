### Post by Eric Pihopa on 2007-05-19
**This post could be related to an Ubuntu bug filed at**: [https://bugs.launchpad.net/ubuntu/+source/xorg/+bug/103945](https://bugs.launchpad.net/ubuntu/+source/xorg/+bug/103945) [br].[/br] ---------------------------- [br].[/br] [br].[/br]
				Also have the same issue with ati x1300 when loading live CD.

Found this workarouind on the Ubuntu NL forum

Works for me.

At the command prompt, do the following:

sudo -s -H
apt-get update
apt-get upgrade
apt-get install xorg-driver-fglrx
sudo aticonfig --initial
startx

Once you are in the Ubuntu Desktop, just run the Restricted drivers manager to check all is ok.
