### Post by Impavidus on 2013-03-13
The screensaver is no longer there by default (modern screens don't need saving), but you can install one with the package **xscreensaver**.

And if you can't find the settings anywhere in the menu's, using the command **xscreensaver-demo** will give you the settings editor. It's a GUI.
