### Post by greenrenault on 2008-04-25
I had this problem as well today and it really stumped me. Why would such a common feature be so obscure? Anyway I digress... 

The answer is this:
1. In Konqueror, open the web page you want to set as you home page
2. Expand the Settings menu
3. You should see options including Load View Profile and Save View Profile ...
4. The default home page, window size and other stuff is saved in the "Web Browsing" view profile.
5. If you click Save View Profile for "Web Browsing" now then that will set your home page to the current page.

You can have different profiles set up for web browsing, which is actually quite cool. But not particularly user friendly.

Anyway, hope this helps you and others!
