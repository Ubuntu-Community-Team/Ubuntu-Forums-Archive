### Post by marshmallow1304 on 2010-01-27
Yes, Java applications will work as long as Java is installed.  To run browser-based Java apps, install [sun-java6-plugin]("apt://sun-java6-plugin").  You may need to first go to System->Administration->Software Sources and enable the multiverse repository if it isn't enabled already.
