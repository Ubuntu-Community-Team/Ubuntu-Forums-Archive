### Post by ReyBrujo on 2010-06-21
Go to Edit -> Preferences -> General tab. The Startup options reads "When Firefox starts:", choose "Show my windows and tabs from last time", then Close. It will remember the windows and tabs you had opened when you quit from that point on.
