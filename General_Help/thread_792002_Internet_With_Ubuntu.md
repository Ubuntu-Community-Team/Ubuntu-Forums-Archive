### Post by brigadoon on 2008-05-12
I have a few questions.

1) What type of system do you have? PC make, model, cpu, ram, network card, graphics card, etc...

2) Prior to the install were you connected to the Internet? If so, how? Hard wire (100 base-T) or wireless.

3) Ubuntu (and windows XP or better) is pre configured for autorouting. If your router at home passes out IP addresses automatically you system should connect. Is your router configured for autorouting?

4) Do you have the Network card driver installed? Some hardware needs to be coaxed in order to get it to work with Linux.

Any other information you have would help understand your problem.
