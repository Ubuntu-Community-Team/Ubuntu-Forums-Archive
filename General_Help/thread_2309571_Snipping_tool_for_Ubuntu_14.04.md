### Post by deadflowr on 2016-01-12
I think the preferred method of launching apps in Ubuntu with the dash is to use the search function to find the app.
( I'm confused on which dropdown arrow in the dash you clicked, seems like the recently used section; the second section listing all apps would have far more than 23)

But beside that,
Screenshot has a default set of keyboard shortcuts you can utilize.
Look in System Settings >> Keyboard >> Shortcuts > Screenshot.

The app itself can also be launched from the terminal.
It's command is gnome-screenshot

I only ever use the full app over keyboard shortcuts for the ability to delay the shot and toggle whether or not to show the mouse pointer.

Any which way you go, it'll default the save location to the Pictures folder, using the date and time as the file name.
fwiw
