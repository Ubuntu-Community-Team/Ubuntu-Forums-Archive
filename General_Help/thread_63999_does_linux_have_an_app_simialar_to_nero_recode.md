### Post by dubrict on 2008-02-05
I love open source just as much as the next guy, but this is something you just gotta be honest with yourself about:
While Nero suite in it's entirety is a stunning example of bloatware, the Recode application alone is incredible.  I have yet to find an open-source application at the same level as this program.  With recode, I can compress DVD's to a stunning 250mb, while still retaining more quality than any open source program I've tried.  And I've tried them all.  But what's most impressive is that it does this in under 20 minutes!  Other applications average over an hour on my computer.

According to appdb.winehq.org, Nero recode should work with wine.  I haven't been able to get it to work, but perhaps you'll have better luck.
