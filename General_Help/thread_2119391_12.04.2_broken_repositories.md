### Post by mc4man on 2013-02-23
There are some things that need to be worked out though you could likely get around some of them in the meantime.
Notably - 
[https://bugs.launchpad.net/ubuntu/+source/apt/+bug/1130419](https://bugs.launchpad.net/ubuntu/+source/apt/+bug/1130419)

As far a vitualbox maybe search for or file a bug on.

If needing ia32-libs (ia32-libs-multiarch:i386) you could do one of 2 things

In the above linked bug there is a patch for apt, a patched/rebuilt apt may perform better.(does here

Or you could try pre-installing some i386 quantal-lts packages first, then try installing  ia32-libs-multiarch:i386

```
sudo apt-get install libgl1-mesa-glx-lts-quantal:i386

```

Look & see if both of these are pulled in with above, they should be,  if not install also
> libgl1-mesa-dri-lts-quantal:i386 
libglapi-mesa-lts-quantal:i386 

Then try ia32 as a sim, if all seems good remove the -s from command

```
sudo apt-get install [COLOR="Blue"]-s[/COLOR] ia32-libs-multiarch:i386
```

Edit:
as far as the linux-headers-generic - there is a linux-headers-generic-lts-quantal package already - it will update 3.5 linux-headers-generic package as they come - see screen

(maybe some judicious use of precise-proposed could be helpful
