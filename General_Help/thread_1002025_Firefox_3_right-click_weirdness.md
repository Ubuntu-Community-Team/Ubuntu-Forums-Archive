### Post by Arenlor on 2008-12-07
OT@ingeva: Bump is used to, well, bump a post up to the top. It's used when you have nothing whatsoever useful to contribute, but that you want the post to be answered or have more people contribute to it. General etiquette states that you don't bump it until it drops off the first page, though depending on the speed of the given forum it may have other rules stating you must wait longer (for example if a given forum usually only has 2 pages a day and almost every support person in that forum reads both pages you'd rarely ever use bump).

I've never had a problem with the context menu in FF myself, but I've noticed it with the bookmarks menus. If I remember right both the context and the Bookmarks menus both use the same XUL code, so there is probably some problem with that somewhere.
