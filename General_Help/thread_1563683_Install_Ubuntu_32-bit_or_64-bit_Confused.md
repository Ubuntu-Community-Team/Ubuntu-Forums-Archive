### Post by 3Miro on 2010-08-29
If you install 10.04 you will get to use all of your 6GB of RAM.

64-bit has some issues with Flash (it varies from person to person), some 32-bit windows codecs (rare and fixable with custom 32-bit mplayer) and some printer versions.

64-bit is about 10 - 20% faster for daily operations and much faster for code compilation, video editing and other CPU intensive jobs.

My advice is to get 64-bit, if you get some major problem (which is unlikely), reinstall into 32-bit.
