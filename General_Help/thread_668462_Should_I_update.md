### Post by ugm6hr on 2008-01-15
> **nabiscOGrahams said:**
> Im new to linux. Just downloaded ubuntu yesterday and the automatic update box says there are 164 updates available.

To download all of them is about 300 MB. I have no idea what they all are but should I always just download all the updates?

The CD you downloaded yesterday is actually 3 months old (Oct 2007). Hence the updates.  Ubuntu only updates the CD for download every 6 months when there is a new release.

If you have used Windows before - Ubuntu updates are similar.  However, Ubuntu updates *all* of the installed programs, not just the OS itself, which is why there are so many.

Ubuntu comes pre-installed with a lot of programs - if you don't need some of them, you can uninstall them.  Then run the update program again, and there should be a few less.

If you do install anything, it would be sensible to keep a backup of /var/cache/apt where all the installation files are kept, so you won't have to download them again if you need to reinstall (or install on another computer).

And welcome to Ubuntu!
