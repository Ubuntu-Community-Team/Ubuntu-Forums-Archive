### Post by SR_ELPIRATA on 2009-11-26
I would suggest googling this... :)

I've never had to ask for an ICC profile to do something, not even in Photoshop. If its a printer profile... I would think that it should come with the printer/software, but I have never done this that you ask.

I've worked in a printing company but I'm no expert. I would think that if you did the image in Blender... was done in RGB and all you would need to do is to change it to CMYK and see how it looks in the image editing program. I have seen RGB to CMYK images and they always lose 'color' is not abnormal.

Usually what we would do is create the image in the CMYK colorspace, as thats the way you plan to print it anyways, but I doubt Blender allows you that because printing is not what is intended for.
