### Post by zpletan on 2010-09-28
Applications->Ubuntu Software Center
Search for package in the search box
Install it

If it doesn't show:
(In Software Center) Edit->Software Sources
Make sure "Software restricted by copyright or legal issues (multiverse)" is checked
