### Post by snowpine on 2012-03-25
Older laptops often use part of the system RAM as video RAM, this could explain why you are seeing less than the expected 256mb. 

I recommend a machine that meets the following minimum specs, which should be available inexpensively (here in the USA $100 or less through a site like Craigslist):

> Ubuntu Desktop Edition

    1 GHz CPU (x86 processor (Pentium 4 or better))
    1 GiB RAM (system memory)
    15 GB of hard-drive space (or USB stick, memory card or external drive but see LiveCD for an alternative approach)
    800 by 600 screen resolution
    Either a CD/DVD drive or a USB port for the installer media
    Internet access is helpful 


source: [https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

Also for best performance I do not recommend WUBI; try a "normal" dual-boot (separate partition for Ubuntu) for best results. :)
