### Post by mc4man on 2008-03-14
it should show up in synaptic - ck. system>administration>software sources and make sure under ubuntu software that (universe ) is checked - 2nd one down (personally I'd have them all enabled)
You could get it here [http://packages.ubuntu.com/search?searchon=names&keywords=libglib1.2](http://packages.ubuntu.com/search?searchon=names&keywords=libglib1.2)
but better in the long run to be able to use synaptic or sudo apt-get install libglib1.2
