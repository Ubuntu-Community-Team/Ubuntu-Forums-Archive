### Post by lovinglinux on 2010-02-23
[http://ubuntuforums.org/showthread.php?t=75851](http://ubuntuforums.org/showthread.php?t=75851)

The thread above has the solution, although I never tried it, since I'm the only user of my machine.

Nevertheless, I would recommend another method. You can use [FEBE](https://addons.mozilla.org/en-US/firefox/addon/2109) extension to make a backup of all installed extensions on your profile, then use [CLEO](https://addons.mozilla.org/en-US/firefox/addon/2942) to pack them together into a single installation package. This way you can install all extensions at once on each account.
