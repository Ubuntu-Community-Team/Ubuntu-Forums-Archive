### Post by nanotube on 2006-02-18
that means you would be able to read any files on your ntfs partition, but not create new ones or save changes to old ones.

for a dual boot option, it is best to have a third partition with fat32, so that you can use it to share files between winxp and ubuntu.

i have my setup currently with ext3 for ubuntu, ntfs for winxp, and a fat32 partition for various data files.
