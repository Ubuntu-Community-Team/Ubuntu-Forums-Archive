### Post by ragtag on 2010-10-15
If by wipe the entire dive clean, you mean remove all operating systems, including Ubuntu, the easiest way is probably just to boot into the live CD, and use gparted. There is also a Gparted live cd you can download if you want to.

If you need to securly wipe the disk, you can use Darik's Boot an Nuke [http://www.dban.org/](http://www.dban.org/) , which takes much longer, but will make it hard for even computer experts to recover any data from the drive.

Both of the above methods will wipe everything from your computer, and make it useless, until you install a new operating systems. More or less all operating systems let you format the drive, as part of the installation process (including Ubuntu), so if you're installing a new OS, you can just do it then.
