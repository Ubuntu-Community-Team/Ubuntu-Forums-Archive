### Post by lovinglinux on 2011-12-16
I don't use Facebook, but I have seen many reports about Facebook problems. It is probably not a browser issue, although different browser versions and brands might behave differently. It seems that Facebook change their code frequently, so in a week it works fine and then all problems arise. It seems from the reports, that indeed it might be becoming bloaty,as jamesisin mentioned, due to excessive use of javascript.

Perhaps using [NoScript](https://addons.mozilla.org/en-US/firefox/addon/noscript/), [AdBlock Plus](https://addons.mozilla.org/en-US/firefox/addon/1865/) and [Ghostery](https://addons.mozilla.org/en-US/firefox/addon/9609/) could help.

It might also help optimizing databases and other stuff. See [Firefox optimization and troubleshooting thread](http://ubuntuforums.org/showthread.php?t=1193567).
