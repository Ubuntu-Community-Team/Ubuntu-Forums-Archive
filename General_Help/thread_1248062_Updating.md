### Post by drs305 on 2009-08-23
> **themarker0 said:**
> Update Manager doesn't give me a 8.10 or 9.04 option.

System, Administration, Software Sources. Settings (lower left): Updates tab. Release upgrades/show new distribution releases > Normal releases. It is probably currently set to display only long term releases.

This should now give you the option of doing a dist-upgrade.

You can also view the Jaunty upgrades wiki:
[https://help.ubuntu.com/community/JauntyUpgrades]("https://help.ubuntu.com/community/JauntyUpgrades")
