### Post by 2F4U on 2012-04-05
Ubuntu server environments are often based on LTS releases, so the software versions are usually older. Else the setup should be the same since Ubuntu distinguishes only in rare cases between server and other uses (mainly the kernel).

---

### Post by SeijiSensei on 2012-04-05
There's no functional difference between a desktop and server installation of the LAMP stack.  I have Apache installed on this machine for testing purposes, and simply pointing a browser at [http://localhost/](http://localhost/) brings up the default web page, /var/www/index.html.
