### Post by djxcon on 2007-07-11
Using the Alacarte menu editor, add a new "Application in Terminal".  
For name use LightZone.
For Command use  bash ./Desktop/LightZone/LightZone (Where ./...  is the location where you unpacked the tarball - in my case I have the LightZone folder/files in a folder called LightZone located on my Desktop)
Then give it a nice icon.
