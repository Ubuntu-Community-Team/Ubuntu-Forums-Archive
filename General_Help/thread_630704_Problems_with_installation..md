You'll just have to do more software installs and updates when you get your Internet connection working on the installed copy. This is why LiveCD is quite nice, because you can configure your network settings before doing an install.

### Post by cipher_nemo on 2007-12-04
From Unetbootin's help docs: "*UNetbootin is a tool that allows you to install various Linux distributions (Ubuntu, Fedora, Mandriva, OpenSuSE, Debian, ArchLinux) from a Windows or a Linux desktop over the internet (i.e., you don't need to burn the Ubuntu, Fedora, ... CDs).*"

Unetbootin allows you to install a boot loader for Windows 9x/2k/XP/Vista. It doesn't include a full installer for Ubuntu 7.10, so it asks you to download the packages after it boots to an "Ubuntu" option. I'd recommend downloading everything ahead of time if you're unable to configure your network during install.

A worthwhile thread for additional information: [http://ubuntuforums.org/showthread.php?t=427540](http://ubuntuforums.org/showthread.php?t=427540)
