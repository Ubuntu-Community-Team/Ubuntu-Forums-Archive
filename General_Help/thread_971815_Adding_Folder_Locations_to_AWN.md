### Post by xkalibur84 on 2008-11-05
I had the same issue you did(it seemed a whole lot easier a few versions back).  But anyway, you pretty much got the line right, but place "nautilus" first before that line and you should be good to go.  It should read:

nautilus /home/yourusername/Documents

I think it's important that the "Documents" has a capital "D".

If clicking the icon creates a new instance on the AWN bar and you prefer it doesn't(I didn't), then you have to put "Documents - File Browser" in either the "Name" field or the "Decription" field.  I can't remember which, so put it in both? =P

Anyway, I hope that helps you out and anyone else.  I'm new to this so my support regarding this is limited.  It's just something I picked up while searching various places on the interwebs.
