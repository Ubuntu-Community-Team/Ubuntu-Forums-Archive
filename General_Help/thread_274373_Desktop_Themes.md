### Post by puppy on 2006-10-10
You might also want to use synaptic to download and install gnome-art-manager. This is a nice little app that provides one interface to download window borders, icon sets, wallpapers etc and install them with one click. Very nice and easy to use 

Remember that you can also mix and match different theme elements to your liking under the "themes" tab in "preferences" so you can have a window border with a different icon set etc.
