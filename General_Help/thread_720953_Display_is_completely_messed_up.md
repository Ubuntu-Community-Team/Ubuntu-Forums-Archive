### Post by taurus on 2008-03-10
Try booting into recovery mode from GRUB menu and reconfigure your X server again.

```
dpkg-reconfigure -phigh xserver-xorg
shutdown -r now
```

### Post by taurus on 2008-03-10
Right now, you are using generic driver, vesa, for your graphic card.  So, go into System -> Administration -> Restricted Drivers Manager and see if your graphic card is on the list.  Then, enable it, installing a driver for it.

What kind of graphic card do you have anyway?
