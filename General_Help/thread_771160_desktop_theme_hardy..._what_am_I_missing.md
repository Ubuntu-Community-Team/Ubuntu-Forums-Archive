### Post by Moop on 2008-04-27
I think the gdm themes are for the login window. Don't extract the themes before you try and install them or it doesn't work. I've had some that didn't want to install but most I've tried worked. 

Art Manager is in the repos and offers an alternative way to download and install themes.
