### Post by catlett on 2006-11-05
There is no need for NeroLinux. Nero is just trying to cash in on it's windows user base. Windows users are used to having to install 3rd party apps to get functionality. Nero is one of if not the biggest burning applications for windows. When people switch they still think they need to pay for a 3rd party app to get functionality, so Nero throws out a low price Nero linux and tries to grab confused former windows users. If you needed NeroLinux it would be more than $20. It is $20 because Nero wants you to say "Only $20, what a bargain. Plus I already know how Nero works. I'll buy it"
You can buy it if you like but you do not need it. Kubuntu comes with K3b and Ubuntu comes with Gnomebaker. Plus there are many more. I actually like Brasero. I never heard of it before Edgy came out and now I prefer it to gnomebaker.
If you need an application, go into Synaptic Package Manager and do a search for the function. I.e. enter "burn a cd" and it will list the available packages. There are thousands of applications in Synaptic. Only a small percentage are installed on your Ubuntu system. There is a free software application for everything. The issue is it won't necessarily look like the windows application you have been using.
