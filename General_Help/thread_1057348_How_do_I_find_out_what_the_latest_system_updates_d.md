### Post by mb_webguy on 2009-02-01
If you're talking about updates from the official repos, then the only updates you're getting are security patches and bug fixes.  All software in the repository for a given released is locked in a week or so before the official release.  After that, only security updates and bug fixes are released through the official repositories.

If you're using third-party sources like PPAs, then you're dependent upon the maintainers of those sources to provide you with meaningful descriptions of updates.
