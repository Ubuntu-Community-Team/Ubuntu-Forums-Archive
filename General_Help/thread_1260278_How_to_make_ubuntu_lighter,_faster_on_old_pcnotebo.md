### Post by konqueror7 on 2009-09-08
install xfce if you have been using gnome. xfce's resemblance to the gnome desktop is quite big. and regarding your dependence to gnome-do's docky, you can still use it on the xfce desktop or any other.

if, however, your willing to start from scratch and build a much lighter ubuntu installation, there are 2 options. install a lighweight ubuntu-derivative distro like [CrunchBang]("http://crunchbanglinux.org/"), or use the alternative/server installer of ubuntu, and install only what you really need. there are many HowTos in this forums regarding lightweight installation, just google it. =)
