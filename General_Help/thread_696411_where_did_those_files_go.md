### Post by Shazaam on 2008-02-14
Some files such as system files won't show up in the menus. If you know the name open a terminal and type that name in.
For example type this in terminal....
```
nautilus
```
That will open the file browser.
Open Synaptic>Status (bottom left of Synaptic); then click "Installed". Choose any installed package and on the bottom right of Synaptic you will see 5 tabs that give a description etc. Go to "Installed Files" and that will list where the files are.
