### Post by mcduck on 2012-02-11
That's a *really* old guide, and makes things way more complicated than what's necessary. I'd recommend ignoring it.

Compiz already has a plugin made specifically for this feature, the Grid plugin. So all you need to do is install CompizConfig Settings Manager, and then use that to enable the Grid plugin. You can then configure window snapping and whatever keyboard shortcuts you might want to use in the Grid's settings.
