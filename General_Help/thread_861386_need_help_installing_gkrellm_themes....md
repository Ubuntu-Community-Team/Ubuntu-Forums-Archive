### Post by linuxwizard on 2008-07-16
This is how I install themes from Gkrellm website. Save to desktop > go to home folder > view > show hidden files > click on the folder .gkrellm2 > click on themes folder > copy download on desktop > paste into themes folder > right click on > extract here

Now right click on Gkrellm on desktop > configurations > themes > the new themes will show up
