### Post by Mark Phelps on 2012-01-16
Copying an Ubuntu ISO, or simply copying the files, to an external drive, does NOT make that drive bootable.

What you should do is get a USB stick (presuming the PC you want to install to can boot from USB) and use the Universal USB Installer from PendriveLinux.com to format that USB and copy the Ubuntu files to it.  

That will produce a bootable USB stick you can then use to install Ubuntu on a PC.
