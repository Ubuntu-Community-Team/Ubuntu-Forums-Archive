### Post by frodemt on 2007-12-14
Hi. I have made a mini-guide on how to make a automatic slideshow system with Ubuntu. This might not be relevant for an ordinary user, but perhaps for people working for a firm where something like this is needed. This is how I started working on this project at first. We had a need at my work to show presentations on a big screen at a place where it was not easy to manage the computer. So I made a Windows program that can send files from remote and start a presentation. This also makes it easy to manage the slideshow even if the user only is used to Windows environment, and the user do not need to hassle with memory sticks or CD’s to start a new presentation. It can all be done remotely. The SendSlide client can also update your Ubuntu system, restart it and shut it down completely.

If you are interested in the full guide you can visit this URL:
[http://tviberg.info/nettside/slideshow_linux_en.php](http://tviberg.info/nettside/slideshow_linux_en.php)

I have plans to open the source code for the client, but only if there is a lot of interest.

Comments are welcome. :)
