### Post by OneWingedAngel on 2006-04-03
For Konqueror (I'm guessing that's what you want, seeing as this is the Kubuntu section): look in View->Icon Size on the menu bar. You can set this independently from the desktop icon size in System Settings (eg. desktop icons 128x128, filer icons 48x48)
