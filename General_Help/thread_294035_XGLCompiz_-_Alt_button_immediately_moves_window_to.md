### Post by chriscando on 2006-11-06
Try running beryl-manager and see if you have the alt key set in the 'place windows' setting. There is a hotkey you can set to move a window to another desktop. To quickly tell just uncheck the 'Place Windows' option and see if your still getting that effect.
