### Post by Trev2HI on 2009-04-13
When in a Fullscreen Citrix session, Ctrl + F2 returns control to the host OS. So, hit Ctrl + F2, then you can use your normal shortcuts like Ctrl + Alt + D to return to desktop, Alt + TAB to switch windows, etc.
