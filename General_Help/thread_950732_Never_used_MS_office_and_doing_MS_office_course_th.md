### Post by bcom on 2008-10-17
Perhaps you should check with the school's IT/student help department.  Its possible that you can download a free version of Office 2003 through the school.  

If the school doesn't make Office 2003 available, I'm pretty sure it can still be purchased. 

Also, just asking your teacher about where you can get the version of the software needed to take the class doesn't say anything at all about what you know or don't know about computers.

He or she should have told you where and how to get the correct version needed for the course.

If you are running Ubuntu, you will need to install wine or vmware server or purchase Crossover (Crossover is the commercial version of wine which is free but not so easy to use) so you can install and run Office 2003 on Ubuntu.
