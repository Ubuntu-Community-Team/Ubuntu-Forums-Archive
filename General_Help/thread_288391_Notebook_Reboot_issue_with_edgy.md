### Post by davescafe on 2006-12-13
Hola H-Radical,

An answer to your question #1:
Firefox does not have a menu for about:plugins. It is a function that is built into Firefox's address bar. Specifically, just type
> about:plugins
in the Firefox address bar and press <Enter>. You will see the plug-ins page. 
Similarly, with Firefox you can use:
> about:config
to adjust and customize a large number of Firefox parameters.

Some sympathy for your question #2:
I don't have an answer, but I have the same problem on several of the computers I am using with Ubuntu Edgy. If I ask the machine to "reboot" it goes all the way down but then hangs on an Ubuntu screen. I have to manually push the power button to get past this point. I am still looking for reasons why this is happening, but I am confidant it is a defect. This was not a problem with the previous release of Ubuntu (6.06 Dapper Drake).
