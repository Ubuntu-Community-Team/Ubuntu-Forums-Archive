### Post by sisco311 on 2008-08-22
reboot in *recovery mode* and change the 
password in the *root shell *with the
following command:
```
passwd *username*
```
more info:
[http://www.psychocats.net/ubuntu/resetpassword](http://www.psychocats.net/ubuntu/resetpassword)
