### Post by HiImTye on 2010-05-14
just install it from the repositories, less hassle

using the command line:
```
sudo apt-get install thunderbird
```
using the GUI:
go to ubuntu software center and in the top right (in the search box) type in thunderbird. it will come up with Mozilla Thunderbird Mail/News. click install, follow the directions.
