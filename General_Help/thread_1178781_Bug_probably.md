### Post by lswb on 2009-06-04
See if you can get a "run application" dialog box by pressing <ALT-F2>. If so type "gnome-panel" in and press enter. (If you can't get the run app dialog post again) Hopefully the panel will appear with at least a main menu and some of the usual icons. If not right-click on the panel, select "add to panel" and add what what you want, make sure you add the gnome Main Menu. To add a second panel, right click on an empty area of the first one and select "New Panel." Then open Main Menu/System/Preferences/Sessions/Sessions Options and press the "Remember currently running applications" button. Log out and back in again (or restart the computer) and make sure the settings hold.
