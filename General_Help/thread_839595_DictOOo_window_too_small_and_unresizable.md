### Post by JesCed on 2008-07-11
Hello.

I was actually having the exact same problem, but I managed to find a clue to solve it. The issue is aparently caused by a problem with the default macro window size when using compiz. So, the solution is to turn off compiz in the "Visual Effects" tab in System ---> Preferences ---> Appearance. After you turn off compiz, the DicOOo window should be fine. Once you install your dictionaries, you can switch compiz back on.

Hope this helps... it worked for me!
