### Post by Ripose on 2009-03-04
You probably don't have the correct ATI driver installed.

To attempt an automatic ATI driver install, use Synaptic to install **envyng-gtk **(Gnome) or **envyng-qt** (KDE), the Envy icon will show up under System Tools.

Envy is for both ATI & Nvidia except for new model cards in both cases.
