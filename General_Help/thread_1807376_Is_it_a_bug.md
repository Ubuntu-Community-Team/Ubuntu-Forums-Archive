### Post by Joris Donders on 2011-07-19
That 'skip'-button is only for shortening the installation-proces. 

It arises when the installer is checking the installation. (At the very end of the installation-proces)

There is no  harm done when clicking this button; it's only for checking your installation. 

So: NO this is not a bug ! ;)

On the left side of that screen you can read what the installer is doing at that time.
