### Post by skippy81 on 2006-05-15
By compress I assume you just mean strip all the whitespace from between the tags?

Try downloading "bluefish editor" and "tidy", both programs are availiable on synaptic.  Bluefish has integreation with tidy so you can do everything from one program.  Tidy is a W3 consortium app, I don't know if it strips spaces etc, but it certainly reformats and cleans up html.  You can also download "weblint" which is a perl script, again compatable with bluefish, which also cleans up html.

Im sure that one of these programs can compress your HTML.  

Bluefish is my fave html and php editor ever, I find it much quicker to work with than dreamweaver.  However if you want a WYSIWYG (lol i think thats what you call it) app, then "nvu" is apparantly an awesome program.

bluefish editor, tidy, weblint and nvu are all availiable through the synaptic package manager.  

Have fun :p
