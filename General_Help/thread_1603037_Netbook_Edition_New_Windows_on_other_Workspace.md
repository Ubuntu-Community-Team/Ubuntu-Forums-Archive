### Post by Bazon on 2010-10-22
In Gnome it's easy, because there is a separation between application launchers and symbols for running applications. A left click on the application launcher opens a new window on whatever workspace you’re currently on.

(On Cairo Dock, there is also the possibility to mix launchers and symbols for running programs, there shift + left click opens a new window.)

---

### Post by kagenotatsu on 2010-10-22
You can ctrl+alt+shift+<arrow> to move an existing window from one workspace to another.

### Post by Bazon on 2010-10-22
