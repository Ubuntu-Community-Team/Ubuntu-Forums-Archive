### Post by catanzag on 2009-12-30
Hello,

just to share with all the community my workaround on subject.

I got Skype from medibuntu, and it worked fine in jaunty, but after upgrading to karmic, it crashed every time I tried to start a chat. My system is fully updated (well, even too much, I have also proposed repository enabled, and also qt 4.6 installed...)

Googling in the web didn't help me, apart the fact that, maybe, dependencies are too old for latest release.

My workaround was to remove skype package from medibuntu repository and download the [static]("http://www.skype.com/go/getskype-linux-beta-static") version from website, unpacking and using it.

After unpacking, now it works fine!!!

Hope this could help some of you
