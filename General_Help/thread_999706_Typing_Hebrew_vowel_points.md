### Post by amire80 on 2008-12-02
Is there a way to type Hebrew vowel points in Ubuntu?

In Windows they can be typed using an unusual combination:

* pressing CapsLock
* putting the cursor after the letter where a point needs to be set
* pressing Shift-<number>; each number key on the keyboard is assigned to a vowel points.

It's awkward, but it works. The Hebrew keyboard on Ubuntu doesn't support it.

Any ideas?

### Post by Eladon on 2009-02-08
Or you can just look at your keyboard layout:

System \ Preferences \ Keyboard \ Layouts tab
Click the "Israel" layout, and click "Print" to send it to your printer.
Any symbols (i.e. vowel signs) in the bottom right corner of each key are activated by the "3rd level" key.


Not sure if others found this, but I had to actually enable the third level key to get the vowel signs working.

System \ Preferences \ Keyboard \ Layouts tab \ Layout Options \ Third level choosers
Tick Press Menu key to choose 3rd level.
Tick Press any of Win-keys to choose 3rd level.
(or whatever works on your physical keyboard)
