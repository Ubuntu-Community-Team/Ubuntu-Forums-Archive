### Post by muthiahmerchant on 2009-11-21
I fixed this by downloading the 64bit beta version of flash from adobe website and copying it to the location where firefox looks for it. 

It did not work before because firefox had another version in a different location and that was overriding the 64 bit flash. 

To find all the location of flash plugin, open firefox, in the URL tab enter about:plugins, that tells you how many flash plugins are there and what the location is
