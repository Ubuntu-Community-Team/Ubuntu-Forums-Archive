### Post by snowpine on 2011-08-13
Yes, that should work OK. :) 
Note however that support for Gnome (and all GUI apps) will end April 2013. It is only the server-specific components that will be supported until 2015.
Also I don't know how well the server kernel is "tuned" for multimedia applications. 
In other words if you are not running a server, then there is little to no advantage to installing Ubuntu Server.

All that being said, there are thousands of computers in the world running unsupported operating systems. If your HTPC is working to your satisfaction in April 2013, it is not going to suddenly explode just because 10.04 reaches its end of life. :)
