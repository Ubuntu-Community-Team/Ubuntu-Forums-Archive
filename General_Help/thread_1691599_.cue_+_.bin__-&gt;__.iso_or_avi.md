### Post by lykeion on 2011-02-20
You can use *bchunk* command to convert bin/cue to iso
Open a terminal and install it with:
```
sudo apt-get install bchunk
```
Access bchunk manual (quit with q):
```
man bchunk
```
To convert to iso (using quotes because your file names have spaces in them):
```
bchunk "cd 1.bin" "cd 1.cue" "cd 1"
```
You should be able to open the resulting "cd 1.iso" with VLC directly (VLC > Media > Open File)

### Post by lykeion on 2011-02-20
Have you tried opening the .iso file in VLC?
1. Start VLC Media Player
2. Media > Open File... > Select your .iso file

Otherwise try mount the .iso file
1. Right-click .iso file > Open with > Archive Mounter
2. Access contents of the mounted iso from Places menu

Regarding converting to avi: You should know that [ISO]("http://en.wikipedia.org/wiki/ISO_image") is a disk image of an optical disc, and [AVI]("http://en.wikipedia.org/wiki/Audio_Video_Interleave") is a container format for multimedia (audio + video). There's no way to do a direct conversion between a disk image and a video.

EDIT
And just like soapytheclown says VLC should be able to play SVCD cue/bin directly. Just make sure you **select the cue file** (not the bin file), see: [http://www.afterdawn.com/guides/archive/how_to_play_bin___cue_files.cfm](http://www.afterdawn.com/guides/archive/how_to_play_bin___cue_files.cfm)
