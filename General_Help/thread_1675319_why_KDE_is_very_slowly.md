### Post by cascade9 on 2011-01-25

I am sorry, but I dont know any way easy way to see which way you installed KDE from a terminal. The easy way to tell is if you installed kubuntu-desktop, your computer should have a 'kubuntu' screen when loading, not 'ubuntu'. 

I will try to make what some of the other people have said clear though, it might help. 

Login screen- 

When you get to the login screen, you can select which desktop to use (gnome or KDE). Choosing gnome will get you back to the standard ubuntu desktop. 

Drivers- 

KDE uses the video card a lot. If you dont have the nvidia drivers installed, it could make KDE a lot slower. I dont know how to install the nvidia drivers from kubuntu. If yuo login to gnome, the drivers can be installed by going- 

system-> administration-> hardware driver ('Additional Drivers 	' in newer version of ubuntu, but since you are using 10.04 it should be called 'hardware drivers')


I would try loging out (or rebooting), choosing gnome at the login screen, then getting the nvidia drivers. Then reboot again, choose and KDE at the login screen. 

If you didnt have the nvidia drivers installed before, getting them should make KDE a lot smoother and faster. ;)
