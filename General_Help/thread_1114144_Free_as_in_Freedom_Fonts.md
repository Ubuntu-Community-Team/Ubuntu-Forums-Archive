### Post by my window broke on 2009-04-02
Does anybody know where I can find some decent fonts that are published under the GNU license? Or really, just any decent fonts that are published to be used freely in commercially used products? Also, if I use that program in Ubuntu to create my own fonts, what copyright status will it have?

### Post by jespdj on 2009-04-02
[Liberation fonts](http://en.wikipedia.org/wiki/Liberation_fonts)
> They are available under the GNU General Public License with a font embedding exception, which states that documents embedding these fonts do not automatically fall under the GNU GPL. Thus, these fonts permit FLOSS systems to have high-quality fonts that are metric-compatible with Microsoft software.

And ofcourse the fonts included with Ubuntu are free fonts.

If you use whatever program to create your own fonts, then ofcourse those fonts are yours and you decide yourself what the copyright is.
