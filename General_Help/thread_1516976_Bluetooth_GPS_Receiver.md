### Post by fragos on 2010-06-24
To start you'll need the pin code for your GPS receiver and to power the receiver on. Left click the Bluetooth icon in the applet bar and select "Set up new device..." and follow the directions. You'll also need to install some navigation software like "TangoGPS"
