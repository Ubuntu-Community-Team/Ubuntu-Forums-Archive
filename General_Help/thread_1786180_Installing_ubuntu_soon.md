### Post by jramshu on 2011-06-19
11.04 is still a bit buggy, especially if you want to use compiz. I suggest 10.04 LTS or 10.10.

All the codecs, java, etc. can be installed using
```
sudo apt-get install ubuntu-restricted-extras
```

### Post by gamblor01 on 2011-06-19
> **jramshu said:**
> 11.04 is still a bit buggy, especially if you want to use compiz. I suggest 10.04 LTS or 10.10.


I don't necessarily think that 11.04 itself is buggy...I think it's more specific to Unity.  You can still run gnome "classic" (which is the same gnome that both 10.04 and 10.10 use) with the latest release.  If you don't mind a few issues with Unity then install 11.04.  If you want to fall back on gnome without Unity then you still can.

Personally there are only a few bugs I am aware of that bother me in Unity:

[https://bugs.launchpad.net/unity/+bug/757991](https://bugs.launchpad.net/unity/+bug/757991)

[https://bugs.launchpad.net/ubuntu/+source/compiz/+bug/771875](https://bugs.launchpad.net/ubuntu/+source/compiz/+bug/771875)



Whatever you decide on you can always change.  If Unity doesn't work for you then fall back on gnome classic.  If that doesn't work for you then install gnome shell.  Choice is an amazing thing isn't it?
