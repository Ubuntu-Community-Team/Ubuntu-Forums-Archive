### Post by Jack Brooks on 2007-01-28
I don't know if this is the right place to offer feature suggestions (if not, can you point me to the right place?), but I wanted to offer a suggestion for Evolution that I think would be VERY handy.  

I work in a law firm and deal with a lot of email traffic (MS Outlook, of course).  (I'm offering this suggestion for Evolution because I think it would help give it a leg up over Outlook.)  I think a very helpful feature would be if you could add a note --like a sticky note or something--that would pop up as you put the mouse cursor on the message.  Such a note would be useful for the individual to add personal notes to specific emails that would allow him/her to keep track of whether any follow-up was required or if something occurred outside the email chain relevant to the subject matter in the chain.  I envision the note popping up like the little text boxes on some hyperlinks these days (the ones with the double-underline).

Such a note would NOT be forwarded along with the email it was on, but would be for use of the individual only.

Again, I apologize if this is not the correct place for such a suggestion, but this is something neither Outlook nor Evolution has and I thought it'd be a great feature for Evolution. (Of course, my luck is that it already exists and I just don't know how to access it.)

### Post by phersotty on 2007-01-28
You might want to forward that suggestion onto the developers of Evolution.   Also there are plug-ins available that might meet your needs.  For example mail-to-task which allows you to convert an email message into a task which you can then add notes to.  Take a look at [http://www.gnome.org/projects/evolution/eplugins.shtml]("http://www.gnome.org/projects/evolution/eplugins.shtml")
