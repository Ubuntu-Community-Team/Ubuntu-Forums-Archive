### Post by sudodus on 2015-08-17
Please tell us what kind of program you want :-)

If it is OK with the command line, I think ***ffmpeg*** is a very powerful tool. There is a steep learning curve, but in the end, you will be there. 

I use bash shellscripts with ffmeg under the hood to convert my video clips. I also use GUI tools that are simpler to use, for example ***avidemux*** and ***openshot***.
