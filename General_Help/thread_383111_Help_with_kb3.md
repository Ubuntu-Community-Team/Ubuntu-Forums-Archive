### Post by buuntuu! on 2007-03-13
what you want to do is a multisession cd
when you burn your data and hit the "burn" button, (sorry if it's not the exact word, my system isn't in english) you reach the dialogue where burn-speed etc may be set up. in the "settings" tab you have to choose "multisession".
you can add data later to a disc created like that.

### Post by granny4linux on 2007-03-16
To be able to burn mp3's in k3b, install libk3b2-mp3 from Synaptic or via sudo apt-get install libk3b2-mp3.
