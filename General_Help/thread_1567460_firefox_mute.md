### Post by JohnElway on 2010-09-03
Anytime you have an application that is using sound, you can mute it by clicking the sound applet in the top right corner, opening "Sound Preferences" and gong to the "Applications" tab. This all works perfectly in 10.04. If I remember correctly, I had to install some extra pulse audio package to get similar functionality in 9.04, but I can't remember what it was.
