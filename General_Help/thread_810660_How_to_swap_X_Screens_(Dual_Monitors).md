### Post by lukemack on 2008-05-28
Hi,

I have my laptop connected to a much larger external monitor and would like the larger monitor to be the 'default' with the desktop icons etc.

I am using nvidia-settings on Hardy and have both screens set to be separate x screens. Xinerama would not support compiz and I just prefer to have separate x screens.

Is it possible to swap the screen numbers over so that my external display is the main screen or do I HAVE to use twinview to get this to work?

### Post by Prospero2006 on 2008-05-28
Check out /etc/X11/xorg.conf

Each screen is set up in that file.

At the bottom of dual screen settings you should see something
like this
```

Section "ServerLayout"
        Identifier      "Default Layout"
        InputDevice     "Generic Keyboard"
        InputDevice     "Configured Mouse"
       Screen "screen0"
       Screen "screen1" RightOf "screen0"
EndSection

```

 If not, you could append a sequence of parameters like that, but you would need to check your screen definitions in the rest of the file.

---

### Post by lukemack on 2008-05-29
thanks. the solution was to edit the section like this:

```
Section "ServerLayout"
    Identifier     "Layout0"
    Screen      0  "Screen0" 0 0
    Screen      1  "Screen1" RightOf "Screen0"
    InputDevice    "Keyboard0" "CoreKeyboard"
    InputDevice    "Mouse0" "CorePointer"
EndSection
```

I swapped the 0 and 1 in the middle column for the two screens.
