### Post by d1337 on 2005-09-27
I use a package called sql-ledger...sorry, but it's web based.  However, it is pretty easy to install and use from just one machine, assuming that is what you meant by web based.  I run my business, with the exception of payroll, from sql-ledger.  I have 600+ customers, 1200+ items and do annual sales volume of over $1million averaging $175/invoice transaction (over 5500 annual invoices) .  Sql-ledger is also available via Synaptic if you have your repos set from the Ubuntu Guide.  Pretty helpful mailing list on sourceforge as well.
