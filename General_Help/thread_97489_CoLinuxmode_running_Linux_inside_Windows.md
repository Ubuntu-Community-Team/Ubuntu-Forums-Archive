### Post by suRoot on 2005-12-01
I assume you want to specifically run Ubuntu?   If so, you can try this:

Download the VMWare Player (free) at [http://www.vmware.com/download/player/]("http://www.vmware.com/download/player/")

Download the browser appliance virtual machine (also free) from [http://download3.vmware.com/software/vmplayer/Browser-Appliance-1.0.0b2.zip]("http://download3.vmware.com/software/vmplayer/Browser-Appliance-1.0.0b2.zip").  The browser appliance is Ubuntu.

Once you have the virtual machine up & running, you will find that it's more or less stripped down.  With *very little* work you can upgrade & install as much stuff as you want.  Then you've got Ubuntu running under Windows.

The default root password for the browser appliance is vmware.
