### Post by mkoehler on 2010-01-04
Linux doesn't have a whole lot in common with Windows, that's the point.  How long does it take to become a pro?  Well, you have so much more control over every aspect of the operating system, so while it's easy to get used to and use on a day-to-day basis, you could work forever and not know all of the intricacies of the operating system.  For example, I've been working with linux and unix commercial servers for about nine years, and I've been using linux as my personal operating system for about six years, and I still learn a few new things every day.

Long story short, if you're looking for the most power, control, and flexibility in your operating system, it's the best option.  It's easy to use for a new user,  but you can learn for years upon years.

### Post by steveneddy on 2010-01-04
> **fterh said:**
> I tried again and this time it's working. Weird... :confused:

Anyway is Linux a lot different from Windows (7)? How long will it take to be a semi-pro at Linux?

It depends on the user.

If you are already good at other things PC then you should pick things up pretty quick.

If you have a hard time understanding what hard drive, partition, bits, bytes, memory and what keyboard shortcuts are, then it may take you longer to get to your intended goal of becoming a "semi-pro" at Linux command line and administration.

I suggest getting to know your new OS by learning to install applications from the repositories, checking out the links in my sig and learning some basic programming languages link C, Perl, Java, HTML and Ruby.

Go to the local bookstore and purchase a book aimed at administering Linux, specifically Ubuntu.

Build yourself a small home network with a server serving files to other machines, Windows and Linux and maybe Macs, so you can get a grasp of this set of skills also.

Through intense study and application of your knowledge you will attain a moderate amount of knowledge to make yourself useful by developing small applications using Linux, to join UseNet groups to discuss your new knowledge and to participate in these forums as a high level moderator and administrator.

Also use a spare machine to experiment on because you will have to reinstall after you hose the system a few times (we've all done that) and try out as many different disciplines as you can handle.

What in particular makes you want to be a white hat?

I also suggest partitioning your HD and installing Ubuntu for real.

You may find yourself migrating away from Ubuntu in the future after you have gained some knowledge, but that is OK. Go, learn, make a better world for the Linux community.

It all starts with a dream..........
