### Post by ankspo71 on 2011-03-02
> **asdfqwert said:**
> Is it possible to make it so that when you switch the desktop all the programs in the taskbar from the current desktop disappear like in Gnome?

I'm using KDE 4.4.5.

Hi,
I think by default the Task Manager widget in your panel displays tasks from all desktops. You can change this by right clicking on the Task Manager widget > Task Manager Settings > "Only show tasks from current desktop"
Hope this helps.
