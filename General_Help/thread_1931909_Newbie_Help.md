### Post by sudodus on 2012-02-26
You have enough power to run any Ubuntu flavour, so download several of them and try which one suits you the best. Try also Linux Mint, which is based on Ubuntu.

I did not see any info about your graphics hardware (manufacturer and model). Can you find that using Windows Control Panel?
--
- 'Vanilla' Ubuntu uses Unity, which is a fancy new style DE.

- Kubuntu uses KDE, which is quite fancy and powerful.

- Xubuntu is light-weight and fast. It has a tradtional look and feel. It is not fancy, but many people find it efficient.

- Lubuntu is ultra light-weight and fast. It is simpler and leaner, and maybe targeting not so powerful computers. But many people like it because of the speed, and you can add whatever software you want.

- There are also other flavours for special purposes, for example Ubuntu Studio with a lot of pre-installed tools for graphics and multimedia.

[I]Edit: Download the iso files and try these flavours live, booted from a CD or USB drive before you decide what to install!
From the live session you should also check how your hard drive is partitioned. We come back to that later.[/I]
