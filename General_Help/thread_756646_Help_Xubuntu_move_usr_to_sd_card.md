### Post by eXtreem on 2008-04-16
When installing you can choose which filesystem you want to mount on which disk/partition. If you can partition the SD card with GParted, then you should could use it for a filesystem. When installing choose for manual disk configuration and put /usr on the partition you created on the SD card.
