### Post by sikander3786 on 2010-08-01
Hi.

Create a USB Startup Disk. You can create it from Ubuntu as well as Windows. [Unetbootin]("http://unetbootin.sourceforge.net") will help you.

If you create a USB Disk from Ubuntu I think all the changes you make will be saved for future sessions. Don't know if Unetbootin will do the same.
