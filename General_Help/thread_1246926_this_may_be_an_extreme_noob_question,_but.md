### Post by HiImTye on 2009-08-22
yeah, file compression (.zip, .tgz, .rar, etc) are different from media compression (avi, mp3, wma/v, etc). file compression compresses the entire file without changing the file structure (or rather using rules to decompress the original file structure) where as media compression strips down the file structure and then re-encodes it using compression schemes
