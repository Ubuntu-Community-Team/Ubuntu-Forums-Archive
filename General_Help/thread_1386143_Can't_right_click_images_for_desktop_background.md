### Post by fooman on 2010-01-20
there is an extension to nautilus that adds the right-click option to "set as wallpaper" ....is that what you mean?

it is called "nautilus-wallpaper" and it can be installed through synaptic, or open a terminal (applications > accessories > terminal) and type or copy/paste the following:

```
sudo apt-get install nautilus-wallpaper
```

you will need to log-out and back in again for the change to take effect....then just right-click on an image and you should see the option.

hope that helps.

### Post by stu644 on 2010-01-20
go to system > preferences > appearance, click on the backgrounds tab, then the "Add..." button near the bottom, from there navigate to the image you want to use, then select open.  select the image you wan to set as your background from the first screen then click close.  this should work to set a desktop background.
