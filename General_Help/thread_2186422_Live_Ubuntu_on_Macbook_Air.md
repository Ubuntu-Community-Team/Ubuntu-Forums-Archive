### Post by jamesisin on 2013-11-07
It can be somewhat troublesome to get something other than the Mac OS to boot on Apple hardware.  One problem is that they use EFI instead of BIOS.  If you Google you can find several tutorials for running/installing Ubuntu on an Intel Mac.  You will likely have to install a slightly different EFI (an upgrade?).  There is both rEFIt and rEFInd which are variously better at locating and launching other operating systems.
