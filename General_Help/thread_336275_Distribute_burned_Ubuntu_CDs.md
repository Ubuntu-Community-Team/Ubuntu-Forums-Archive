### Post by JTux on 2007-01-11
If you want to distribute CDs with the Ubuntu cover you can order some CDs from [shipit.ubuntu.com]("http://shipit.ubuntu.com"). Or you can simply print the cover included in the image (in the examples folder).

You can burn as many CDs as you like from the image you've downloaded and distribute them for free. 

And there is no such thing as a "Original" ubuntu CD. The CDs you burn from the image are exactly the same to the CDs sent to you from shipit in every sence except for the artwork printed on the CD ( and the cover).
