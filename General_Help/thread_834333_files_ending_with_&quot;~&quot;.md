### Post by ezramorris on 2008-06-19
These are backup files created every time you save a document in gedit (known in the menu as "Text editor") or kate. You can delete these files if you don't need them.

You can also disable these in gedit. Edit>preferences>editor tab>uncheck "create a backup copy of files before saving".
