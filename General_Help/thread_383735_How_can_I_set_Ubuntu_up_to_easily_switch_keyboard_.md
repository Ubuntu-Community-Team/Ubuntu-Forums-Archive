### Post by mojoman on 2007-05-01
This is probably a bit too late but I'm sure there is a panel plug-in that you can add to the Gnome panel, which then let you switch keyboards very easily. I'm not on Gnome myself so I can't walk you through the procedure or double-check it  but try to right-click on the panel (on a place where there are no icons or such) and choose Add to Panel. You'll get a list of options, I think it's divided by categories. Check for utilities, there should be a keyboard switcher there.
