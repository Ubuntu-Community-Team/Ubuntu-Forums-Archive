### Post by Impavidus on 2020-11-05
Newer versions may have better support for newer filesystems.

Anyway, every Ubuntu (flavour) live disk has the latest (at the time of live disk release) version of gparted (or KDE partition manager, I think) installed and one should have a live disk of one's currently installed version to fix it in emergencies, so if you have Ubuntu installed, you should already have a recent partitioning tool.
