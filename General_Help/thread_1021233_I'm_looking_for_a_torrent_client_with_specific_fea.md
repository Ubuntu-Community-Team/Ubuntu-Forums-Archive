### Post by mnemonik on 2008-12-25
Hi, I have been using Ubuntu for about a month now after having been using XP, so while I have almost everything set up in a way that I am happy and comfortable with, I still haven't found a torrent client that suits my needs. Right now I am using uTorrent through wine, but I would prefer not to have to do this.

The features I really want are these:

* The ability to save .torrent files in a separate folder from the torrent's contents automatically. I like to organize my downloads by type but I like to keep all my .torrent files in a single directory for easy back up.

* The ability to open a .torrent file I have already downloaded, but am no longer seeding, and select the directory where it was downloaded to and be able to start reseeding with out the client attempting to start the download over again.

* The ability to view extra information that is sometimes pushed to the side and forgotten about such as:
   - All the trackers that are being seeded
   - The peers you are connected with, and their speeds
   - % of files and pieces downloaded within the torrent

Is there anything out there that could fill my needs?

### Post by pyromanic123boom on 2008-12-25
I assume you use uTorrent in Windows, so did I. You'll find it works well in Wine, and does everything you've listed.

But if you want to run a native program, I would recommend Deluge. It's available in the repos as 'deluge-torrent'. It does everything you need and is lightweight. 

If you install deluge, then just open the preferences window and you can change it around.
