### Post by TuckLive on 2009-09-21
Welcome to UF.

First I'll say I don't know about compatability issues with i7 personally.  On your question about dual booting, yes it's pretty easy.  Though installing windows after Ubuntu sometimes messes with the master boot record, but documentation on that can be found [HERE]("https://help.ubuntu.com/community/WindowsDualBoot").  As to whether you should build this system, I say give the livecd a try first to really see if Ubuntu is for you.  It would be easier than resizing the Windows partition or reformatting.
