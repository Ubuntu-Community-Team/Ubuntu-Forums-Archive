Cool It seems you're heads up on the script, no /boot/grub/core.img in that line anymore and you can boot to Vista.

Here is the link but I will post the commands, make sure your using a 10.10 live cd.
[https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD](https://help.ubuntu.com/community/Grub2#Reinstalling%20from%20LiveCD)
```
sudo mount /dev/sda5 /mnt
```
Then
```
sudo grub-install --root-directory=/mnt /dev/sda
```
Reboot to the grub menu get to the terminal in the Ubuntu that is installed and run this to update grub.
```
sudo update-grub
```

If you at some way go the Natty way notice the links line that is a little changed from the second command posted here, this info the link goes straight to. It a modified command for Natty.
