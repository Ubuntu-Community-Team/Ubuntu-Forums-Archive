### Post by Mosey on 2005-11-25
So I install gDesklets so I could get the pretty launcher interface thing (where when you roll your mouse over the icons on the panel they grow and are all pretty and stuff)

But ever since I added Firefox to the launcher, whenever I open it up it takes me to the university of Arizonas homepage...

no matter what I set as my home...the default page is arizona.edu

Anyone know how to fix this weird problem?

!!!SOLVED!!!

If you right click the icon on the panel and select edit starter it'll take you to an options page. there is a box that says 'command: firefox %u

Delete the '%u' and it no longer takes you to arizona.edu it'll use your homepage.
