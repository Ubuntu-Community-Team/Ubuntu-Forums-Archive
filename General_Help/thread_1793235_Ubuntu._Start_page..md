### Post by grahammechanical on 2011-06-29
If you have upgraded both machines to Ubuntu 11.04, then it sounds to me if the "clunker" is running Ubuntu Classic as the User Interface and the "sorta" is running Unity (Ubuntu) as the User Interface. If so, then on this second machine move the mouse pointer over the top left of the top panel and you will see those menu options.

Alternatively, use the Ubuntu Classic User Interface on both machines.

Regards.
