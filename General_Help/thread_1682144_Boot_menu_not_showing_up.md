### Post by Rubi1200 on 2011-02-05
If Ubuntu is the only operating system on that particular computer, then the default is to boot straight into it.

You can access the GRUB menu as the computer boots by holding down Shift.

If you would like to change the default timeout, see here:
[http://ubuntuforums.org/showthread.php?t=1195275](http://ubuntuforums.org/showthread.php?t=1195275)
See section 5, /etc/default/grub for more information
