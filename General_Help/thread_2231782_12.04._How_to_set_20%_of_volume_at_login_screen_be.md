### Post by mafeuser on 2014-06-27
Hallo people.

I would like to have volume always set at 20% of maximal volume at login screen.

I am using lightdm/pulseaudio of course.
So far I did:

/etc/lightdm/lightdm.conf:
```
[SeatDefaults]
user-session=ubuntu
greeter-session=unity-greeter
greeter-setup-script=/etc/lightdm/my.greeter-setup-script.sh

```

/etc/lightdm/my.greeter-setup-script.sh:
```

#!/bin/sh
/usr/bin/pactl set-sink-volume 1 -- 20%
/usr/bin/pactl set-sink-volume 0 -- 20%
exit 0
```

but solution above does not do the job.
although when I'm logged in to my user, following command succeeds.
/usr/bin/pactl set-sink-volume 1 -- 20%

I did some investigation and it seems to me that pulseaudio does not work yet at the moment when greeter-setup-script is run. greeter-setup-script is for sure run

Do You know how to do it correctly?

regards,
Mafeus

### Post by ubudog on 2014-06-27
Try removing the "--" from your /etc/lightdm/my.greeter-setup-script.sh.

Instead, it would be:
```

#!/bin/sh
/usr/bin/pactl set-sink-volume 1 20%
/usr/bin/pactl set-sink-volume 0 20%
exit 0

```
