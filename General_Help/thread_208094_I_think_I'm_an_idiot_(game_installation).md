@aysiu:
Install Shield Wizard is one of the many Windows Installer programs.

@Pointful
The only remotely similar thing to Install Shield is any one of the many source package installers that take a source .tar.gz, compiles it for you, and then installs the binary generated.

To install programs, you can either use apt-get, aptitude, adept, or synaptic.

apt-get and aptitude are command line interfaces for downloading programs from the online repositories. Adept and Synaptic are front ends to apt-get, Adept for KDE and Synaptic for Gnome, although they can be run in each desktop environment.

See aysiu's site: [http://www.psychocats.net/ubuntu/installingsoftware.php](http://www.psychocats.net/ubuntu/installingsoftware.php)
or
[http://www.monkeyblog.org/ubuntu/installing/](http://www.monkeyblog.org/ubuntu/installing/)
