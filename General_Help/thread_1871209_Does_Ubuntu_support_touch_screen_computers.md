### Post by 11jmb on 2011-10-28
You can get many tablets working pretty smoothly with Ubuntu, and IIRC, one of the big updates in 11.04 was increased touchscreen functionality.

You can always get a livecd and use "try ubuntu without installing" that will not tell you everything, because there might be some proprietary drivers that you need to install, but there is a chance that it will work right off the bat.
