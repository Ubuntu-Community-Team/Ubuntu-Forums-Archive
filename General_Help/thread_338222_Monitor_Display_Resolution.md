### Post by jott27 on 2007-01-14
My Distro is Ubuntu6.10. After a reboot my Monitor resolution changed to 640x480. 
The "System Setting" in Administration mode does not permit a change for the Monitor setting. It is locked in as 640x480.
The /etc/X11/xorg.conf shows the following entry:
Section "Monitor"
  identifier "Generic Monitor"
  vendorname "Plug 'n' Play"
  modelname "Plug 'n' Play"
  modeline  "640x480@60" 25.2 640 656 752 800 480 490 492 525 -vsync -hsync
  gamma 1.0

How can I change the monitor setting to something more reasonable like 1024 or 1248 instead of 640 ??
Note:

### Post by ajgreeny on 2007-01-14
Did you do any updating of packages before rebooting and if so what were they?  If it included any xorg packages that may have upset things in some way.

Try reconfiguring your xorg by typing in a terminal:-
sudo dpkg-reconfigure xserver-xorg

If you're not sure of some things just accept the default, but make sure you have all the data for your monitor before you start, ie its default resolution, and vertical and horizontal refresh rates (should be in the monitor guide book), and know which graphic card you use so you can chose the correct driver.
Then either reboot or type in:

sudo /etc/init.d/gdm restart
Note: If you use KDE like me it will be kdm restart instead.

Good luck.
