### Post by geek_Man on 2006-11-27
Open Nautilus click Edit -> Preferences click the Behavior tab then click the "Always use text-entry location bar" checkbox. Then you're good to go!

### Post by CatKiller on 2006-11-27
Edit -> Preferences -> Behaviour -> "Always use text-entry location bar". There is a way to turn it on temporarily, but I can't remember what it is, since I always use the text-entry location bar...

Also, you can press **Alt-F2** and type in **nautilus /path/you/want/it/to/open/in** and it will open a Nautilus window in the specified directory.
