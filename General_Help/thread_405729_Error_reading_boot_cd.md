### Post by sammydee on 2008-02-17
When you download the cd file from the ubuntu servers, run a command like:

md5 /directory/of/file/ubuntu.iso

then compare the output of that to the md5 checksum listed on the cd download page. If they are different, there has been an error downloading the file.

Of course, using bittorrent is a much better alternative to this - as well as reducing load on the ubuntu servers, it also incorporates automatic md5 checking.

If the iso is verified correct, and you have tried burning on the slowest possible speed to decent cd media, you may have a hardware problem with your cd drive.

Sam
