To your question:
Downloading/Installation a software which makes it possible to run a virtual machine inside your computer (such as VirtualBox or VMware) let you install many operating systems inside such a virtual machine.
Windows XP (and surely Vista, too) is one of them.
So, when you use such a virtual machine running Windows (XP or Vista) there is *nearly* everthing you can do with it as if you had it installed natively on your computer.
What not will work is: 3D-Graphics. That will not work.
And depending on your computer (RAM and CPU) you might have difficulties with the performance.
So, if you have the proper hardware, you should be able to use your software on a windows running in a virtual machine.
