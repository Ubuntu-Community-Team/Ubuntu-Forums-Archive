### Post by ivanovnegro on 2011-05-11
Ok, you have to edit your menu, to put the program in there, and to give it an icon.
Go to the menu, right click on it, edit menu, then add a new program to one of the categories, you can give it a name and choose the icon if you click on the blank space where should be an icon, you need the source folder to put in as a command, where the executable file is, e.g., if it is under Downloads or Home, copy that,  close the window and the menu will be automatically updated.
Then you should find the program in the menu.

It should look like this when you want to refer to the program you want to put in the menu, e.g. right click on the executable file, properties will give you the command.
