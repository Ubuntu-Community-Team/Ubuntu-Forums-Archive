You don't need Gimp to save screenshots. Just press "Print Scr" (or "prt scr") key on the keyboard to take a screenshot and save it as ".png" file. Then just double-click to open the saved screenshot and "Save as" .jpeg (and manually change the extension name to ".jpg") > Done !

**EDIT:**
If you need some basic editing on it, then "KolourPaint" is a nice light weight program to do that easily.
