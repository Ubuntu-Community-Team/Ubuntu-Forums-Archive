### Post by kabus on 2006-04-29
You could just run the program screen in any terminal. It's a 'terminal-multiplexer' which allows you to have several windows open in the same terminal, similar to tabs in firefox.
It allows logging on a per-window basis. 
Here's the relevant part of the screen manual, just take a look to see if it does what you want :

[http://www.delorie.com/gnu/docs/screen/screen_117.html](http://www.delorie.com/gnu/docs/screen/screen_117.html)
