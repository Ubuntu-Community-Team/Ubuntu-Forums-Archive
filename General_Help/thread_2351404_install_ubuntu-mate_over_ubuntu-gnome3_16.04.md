### Post by wildmanne39 on 2017-02-03
You have to open "your software installer". Uncheck the install from cd.

After this is done just update the sources again and try to install the application again and it should work, I can not tell you exactly how to change the setting because I have never used gnome3 but I imagine you know how to open the software installer.
