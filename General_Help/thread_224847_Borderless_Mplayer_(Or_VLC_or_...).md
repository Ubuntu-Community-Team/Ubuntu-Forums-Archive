### Post by jordilin on 2006-07-28
I would try a program called alltray. When passing the option (if I remember) --borderless you'll get the program passed in the command line to alltray without border. Alltray puts an icon in the notification area of the program launched. 
It would be sth like:
```
alltray --borderless mplayer
```
Alltray has been discussed in the forums in:
[http://www.ubuntuforums.org/showthread.php?t=81727&highlight=alltray](http://www.ubuntuforums.org/showthread.php?t=81727&highlight=alltray) and other posts.
