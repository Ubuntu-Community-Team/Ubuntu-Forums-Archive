### Post by Petro Dawg on 2012-08-19
The recovery is probably the recovery partition for Windows, so you can reinstall Windows if Windows fails, not in the case of the SSD failing.

You will most likely need to make some recovery DVD's from Windows and then delete the recovery partition anyway in order to install Linux.  Be sure to make a Windows System Repair disk too before starting the partitioning process.

Be sure you read up and understand how to do all this before attempting, if you are not sure what all the partitions are, you will get your self in to mess quickly.
