### Post by Irihapeti on 2022-09-02
I was going to post the following, which you don't need right now but it might be useful for someone else (or yourself at another time).


You could try right-clicking on the Inbox folder and choosing Properties. Make sure that General Information is selected and click on Repair Folder.

Sometimes the folder index gets messed up and needs rebuilding.
