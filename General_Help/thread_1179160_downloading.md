### Post by alberto ferreira on 2009-06-05
piratebay is based on torrents and for torrents there is a program preinstalled in your applications>internet>transmission - just use that to download torrents.

To do emule downloads install amule - you have 2 alternatives:
GUI (graphic) installation: system>administration>synaptic and there you search for amule or other program you want to install
text installation: sudo apt-get install amule ( in the terminal (applications>accessories>console )
