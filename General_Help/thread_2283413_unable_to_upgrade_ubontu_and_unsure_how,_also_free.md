### Post by grahammechanical on 2015-06-21
The reason why you are having this problem is that Ubuntu 13.10 reached end of life in July 2014. The software repositories are now closed and have been moved to another location. The URLs in the list of software sources are no longer accurate. 

If you are having problems then an upgrade would not necessarily fix those problems. You should think about doing a fresh install of Ubuntu 14.04. Could the freezing be hardware related? RAM issues? Overheating issues?

Regards.
