### Post by electrojustin on 2011-05-26
Hi all! I upgraded to the latest ubuntu release a few weeks back and I'm experience problems with my favourite FPS game installed on my system, Cube 2 Sauerbraten. When in multiplayer, my ping goes into the thousands every couple of seconds and stays on this "peak" for a minute or so. This effectively makes multiplayer mode impossible to play. I tried obvious things: rebooting, reinstalling the game ect., but to no avail. I also ran a few tests. I tried connecting to my own server. It works when I connect using "local host", but actually typing in the IP gets me kicked out of my own server for high ping :(. I also tried pinging various sauerbraten servers and various http servers for comparison. The result of the sauerbraten server was the same as the result described in multiplayer mode, while the http server was only a hundred or so ms. Thanks for taking the time to read my..."long winded"... explanation of the problem and thanks in advance for any help!

---

### Post by electrojustin on 2011-06-26
I have solved my problem. I was playing around with my router's settings and turned the broadcast setting from "802.11b, 802.11g, 802.11n" to just "802.11g".
