### Post by tacutu on 2008-12-07
you probably can't. well, you can if you disable the Alt-Tab shortcut from the window manager, but who wants that?
2 things you can do, however:
- Alt-Enter puts on the command line the file which is selected in MC. (so you don't have to type the whole filename.
- Ctrl-o in MC takes you to a shell, where you can use the bash completion. 

Maybe that helps a bit.
