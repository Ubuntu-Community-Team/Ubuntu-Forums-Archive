### Post by j-strap2 on 2006-08-31
> **tjb891 said:**
> When you install the Tor and Privoxy packages does ubuntu chnage any settings? Also do they run as automatic services or do they have to be activated like in winXP?

Both run automatically. Note that you need to edit the Privoxy configuration file to get it to use Tor - this is detailed in several threads here and on the Tor site. Changes to the Privoxy config file take effect immediately (you do not need to restart Privoxy).
