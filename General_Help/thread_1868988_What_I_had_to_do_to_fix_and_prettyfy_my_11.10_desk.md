### Post by georgelappies on 2011-10-25
I finally got my ubuntu looking semi decent again.

 Had to do the following.

* install the gnome-tweak-tool and gnome-shell and gnome-panel and synaptic .
* install the Zukitwo gtk3 theme from [www.gnome-look.org](www.gnome-look.org)
* install the Neu icon set from synaptic.
* uninstall the fugly scroll bar overlay -search google for how.
* uninstall the fugly top menu bar -search google for how.
* reboot
* at the login select gnome classic.
* Once logged in, open a terminal launch gnome-tweak-tool.
* under theme select for window theme "Zukwitwo-Dark"
* under cursor theme select "Adwaita"
* under icon theme select "Neu"
* under GTK+ theme select "Zukwitwo-Dark"
* logout and login again.

Now my desktop behaves in a sensible manner that allows me to work without fighting the interface while not looking to shabby either.
