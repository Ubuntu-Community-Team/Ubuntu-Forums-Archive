### Post by BJ_Covert_Action on 2009-08-04
To change a menu item's icon follow these steps: (note, this is for ubuntu 8.04 things might be a bit different on your system).

1. Click 'System' <\next to your Applications area on your toolbar probably> 
2. Go to 'Preferences'
3. Click on the 'Main Menu' option
4. In the dialog box that opens up, navigate to the menu item you want to change the icon for.
5. Right-click on the item of interest and select 'Properties'
6. In the 'Launcher Properties' dialog that pop's up, click the picture of the current icon of the menu item. 
7. Select the new icon you want, and click close. 

Also, of note, is the fact that in that Launcher Properties dialog box, you can modify the launch command for the item of interest. For instance, you can add command-line switches and other options to it. This allows you to open applications from the menu with custom inputs and options.
