### Post by nilber on 2008-03-09
I have been using MoneyDance for about 2 years now.  I'm not going to say it is a perfect program, but is Microsoft Money?  Is Quicken perfect?  I have had a few problems, that have usually been resolved relatively quickly, or at least some sort of work around provided till it does get fixed.  For the price, I don't feel I made a mistake at all.  I paid $30 two years ago, and have gotten a number of small updates and 3 or 4 updates where I had to reinstall the software, which was seamless everytime.  If I wanted to run the current version of Money, I would have spent probably close to $120 since I bought MoneyDance.  Oh and Money doesn't work on Linux without wine or crossover office.  Moneydance does.  I recommend Moneydance for users on any platform, but for linux users, it's the only one I've found that runs directly in linux and has most of the features of the big name finance managers, as well as a nice interface.

Ultimately, only you can decide what is best for you.
