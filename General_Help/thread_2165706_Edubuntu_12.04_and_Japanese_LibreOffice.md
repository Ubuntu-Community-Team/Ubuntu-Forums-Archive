I am running Edubuntu 12.04 and want to use the Japanese version of LibreOffice. I have two user accounts for this computer, the system language for the main one is English and the secondary one is Japanese. I have configured the language input using ibus so that I can write in Japanese in both systems.

Except for LibreOffice in the secondary user system. There the UI for LibreOffice is all squares (see below picture) so I can't read anything and when I input Japanese characters it shows up as squares. This is obviously some kind of font problem specific to LibreOffice (the UI for Firefox displays the Japanese menu items just fine).

I have installed the Japanese package for LibreOffice and a bunch of different Japanese fonts, but I have no idea how to fix this problem. Can you help?

### Post by Brian1215 on 2013-08-06
[http://academicallychallenged.wordpress.com/2012/08/29/writing-in-japanese-in-libreoffice-on-ubuntu-12-04/](http://academicallychallenged.wordpress.com/2012/08/29/writing-in-japanese-in-libreoffice-on-ubuntu-12-04/)
"It turns out that there&#8217;s another set of options under Tools, Options, LibreOffice Writer, Basic Fonts (Asian).  This had been set to a Western character set font (hence the empty boxes).  Switching the default font to Droid Sans Japanese (although any other Japanese font should do) solved the problem."
