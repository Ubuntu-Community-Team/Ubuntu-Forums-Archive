### Post by 23meg on 2006-05-12
The File Roller archive manager (Applications/Accessories/Archive Manager) which is installed by default can handle rar files when you install the packages "rar" and "unrar-free" (or "unrar-nonfree" which is the non-free version that can also handle rar 3.x files).
