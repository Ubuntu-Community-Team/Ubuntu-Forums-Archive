### Post by RAOF on 2006-02-01
In short, no.  One of the nice things about Gnome is the way it has just **one** theme - giving different apps different themes would make the interface a lot less consistent (that way lies KDE ;)).

As far as I know, there's no way to use KDE themes with Gnome.

Edit: not quite true - you can use [Poofyhairguy's Knome]("http://www.ubuntuforums.org/showthread.php?t=115974") guide to use Kwin as Gnome's window manager, allowing you to use KDE window borders & get a lot of cool, Composite-based eye-candy.
