### Post by measekite on 2009-07-29
I am using Jaunty with Gimp 2.6.  Unlike earlier versions of Gimp that required one to install Gutenprint and PPP files this current version of Gimp has its own print module.

I am printing with a Canon IP4000.  All earlier versions of Gimp produced very good quality prints.

The current version of Gimp with the built in print module requires you to first do a File page to set the page size.  While the dialog is reasonably OK (not good documentation) the printed results are not good.

There appears to be vertical lines going through the print.  It is really readily apparent on solid colors.  These lines or banding are about 1/8 to 1/4 inch apart.  I have not spent a lot of time on this yet so I am wondering if others had any problems with it.

Please let me know about your print problems and what printer you are using.  Maybe there is a setting I need to know about.  

I do not have this problem with other software.
