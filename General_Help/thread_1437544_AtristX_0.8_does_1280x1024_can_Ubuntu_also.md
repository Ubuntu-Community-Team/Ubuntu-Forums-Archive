### Post by Spaceman9 on 2010-03-30
I got Kubuntu 9.10 and ArtistX 0.8 installed and working 1280x1024 by adding xforcevesa to the live cd line of boot commands. So by setting the resolution before the live cd is booted that will be the resolution you&#8217;ll get after the OS is installed from the live cd.

I have to use xforcevesa because Ubuntu stopped using the old ATI drivers with Ubuntu 9.04 that work with high resolutions with my old ATI video card. If anyone else has an old ATI video card they could use this trick to run any buntu from 9.04 on at a high resolution.

There are still 4 distros that have the old ATI driver though. Mandriva, OpenSUSE, Pardus and Mepis. I don&#8217;t understand why distros can&#8217;t have both the old drivers and the new drivers and let the user pick out which one they want to use. I think Ubuntu tries to stay too much on the cutting edge and is leaving too many people with old computers behind.
