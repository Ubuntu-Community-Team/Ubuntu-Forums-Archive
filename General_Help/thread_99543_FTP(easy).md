### Post by cilynx on 2005-12-05
In my experience, 'gftp' is the best balance of "ease of use for windows converts" and "actual usefulness".

Also, if you're running Gnome (default in Ubuntu), you can access FTP from the file manager:

Click Places -> Connect to Server...

It will pop up a window asking for a server and optional login.

### Post by aclaunch on 2005-12-05
There is a commandline application "ftp"; use "man ftp" to read the manual pages and get the syntax correct. There is also "wget" which is "a non interactive network downloader". Again "man wget" is your friend. This is a great program to set up multiple downloads. Beyond that there is gftp and various KDE based ones.

Good luck,
