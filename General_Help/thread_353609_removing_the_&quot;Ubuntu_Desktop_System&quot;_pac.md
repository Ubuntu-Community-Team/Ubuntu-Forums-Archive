### Post by dcstar on 2007-02-04
> **bmbeeman said:**
> I'm trying to get rid of software I never use, but in order to do that I have to remove the ubuntu-desktop package, will this affect my system in a negative way if it is removed?

Apparently it will only be an issue if you (later on) want to do a version upgrade of your current installation - then you may have to re-install it and all the packages you removed to do the correct upgrade.

If you don't care too much about that, then go ahead and get rid of it.
