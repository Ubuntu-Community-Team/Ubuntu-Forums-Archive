### Post by ilrudie on 2008-09-25
I don't know of anyway to actually install the drivers from within Ubuntu.  If your internet doesn't work while you are in Windows just download the driver's install file from the manufacturer while you are using Ubuntu and find some way to get it into Windows (e.g. mount your Windows partition and put it there, burn it to a cd, put it on a flash drive... whatever you're comfortable with)

I think thats your best bet.
