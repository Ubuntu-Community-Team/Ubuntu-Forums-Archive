### Post by chprpipr on 2012-11-22
This year's player was consistently quitting after several seconds of video and then complaining about technical difficulties both in Firefox and Chromium.  After a few weeks of this, I decided to dig the problem and came across some details about the CDN they're using this year.  Long story short, it appears that the version of Flash Player that Linux was been pegged at (11.2.202.251) is not compatible with the CDN.

Fortunately, if you install [Google's Chrome Browser]("http://askubuntu.com/questions/79280/how-to-install-chrome-browser-properly-via-command-line") you can successfully watch NBA League Pass 2012 as it has a more current version of Flash Player baked in.
