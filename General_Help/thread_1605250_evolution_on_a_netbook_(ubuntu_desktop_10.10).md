Hey guys, so I installed the desktop edition because I did not like the unity interface, felt really limited by the netbook environment.  I'm really enjoying the maverick desktop edition, and I want to use Evolution because of the integration, but my netbook screen is causing an issue where by activating the calendar, the whole evolution environment freaks out.

I've opened evolution --express, but is there a way to default to that every time I open the program?  Is there a way to run evolution in the background always, exiting out defaults to just hiding the window like all of the chat clients?

### Post by enray on 2010-10-27
Go into System > Preferences > Main Menu. Find the Evolution icon under Office or Internet. Right-click and choose Properties.  Add the "--express" switch to the command line for launching the program,  Worked for me, Ubuntu on a 576-pixel high netbook screen.

### Post by Maupertus on 2011-02-13
I know this is an old thread but wanted to add that the strange behavior by evolution on a netbook is due to its standard screen use. When you deactivate the switcher buttons, the problems will stop as well.
