### Post by ivarn on 2010-11-08
I wouldn't do this just so you can learn about the server thing. With Linux you can always fix and repair stuff nomatter what. Im very new to ubuntu myself but I've learned that almost everything can be solved from the command line. Therefor if you mess up something you will always be able to fix it.
Since you also run windows, you will always be able to login there and ask for help here on the forum if something goes wrong. Sure you can create a 3rd partition but it's your choice. If I were you I would make a backup point of what you have now so you can always go back right after all the server testing and stuff (if you wanna go back or if everything goes wrong). But I would never waste 20 gb on my pc just so I could learn about the ubuntu server. Install the packages and use your already installed ubuntu instead. Also, what can possibly go wrong? we are talking about a server here.
EDIT: It's not very hard to install a 3rd OS.
Just burn the cd iso file or use a usb stick, start the installation and when you come to the partition part, click advanced and set the partition size you want.
Since this isn't a OS you gonna use for other than testing and such, 20 gb is enough.
