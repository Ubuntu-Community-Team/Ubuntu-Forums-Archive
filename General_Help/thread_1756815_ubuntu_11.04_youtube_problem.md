### Post by garvinrick4 on 2011-05-12
Flash support is fine in Linux. No matter what it takes a lot of Cpu to run flash. Unity running on Compiz takes some cpu also. When running flash system-monitor package or type in "top" in a terminal to see how much is being used. Install system-monitor in Ubuntu software center
or type:
```
sudo apt-get install gnome-system-monitor 
```in a terminal

If you want to see what can be done if any. Tell us about your machine. processor, ram, model
and such. What you are running 32 or 64 bit and so on.
## Could be just to install or tweak your flash a bit.
