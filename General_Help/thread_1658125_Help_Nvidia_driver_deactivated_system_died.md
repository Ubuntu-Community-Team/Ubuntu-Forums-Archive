### Post by dino99 on 2011-01-02
here is an example for 3 screens:

[http://ubuntuforums.org/showpost.php?p=9604915&postcount=2](http://ubuntuforums.org/showpost.php?p=9604915&postcount=2)

first clean the system: sudo rm /etc/X11/xorg.conf (or rename it)

then into synaptic: remove/purge ALL the installed nvidia packages, then reinstalled nvidia-current & nvidia-common & nvidia-settings

then reboot and run nvidia-settings: it should detect & configure your hardware, if not , follow the example above to tweak the previous xorg.conf generated by nvidia-settings.

note: to use the latest driver, you can add this ppa:

[https://launchpad.net/~ubuntu-x-swat/+archive/x-updates](https://launchpad.net/~ubuntu-x-swat/+archive/x-updates)

or this one more risky:

[https://launchpad.net/~xorg-edgers/+archive/ppa](https://launchpad.net/~xorg-edgers/+archive/ppa)
