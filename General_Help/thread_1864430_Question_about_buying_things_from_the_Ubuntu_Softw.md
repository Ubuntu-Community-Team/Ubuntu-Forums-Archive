### Post by 3rdalbum on 2011-10-20
It's linked to your "Ubuntu Single Sign-in" account that you need for the Music Store and for purchases on Software Center. You shouldn't have any issues re-downloading software on that same computer (even with newer OS) as long as you're still logged into the same Ubuntu Single Sign-in account.
