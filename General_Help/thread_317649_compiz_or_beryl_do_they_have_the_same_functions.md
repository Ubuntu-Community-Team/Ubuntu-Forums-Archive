### Post by 23meg on 2006-12-12
From [http://wiki.beryl-project.org/index.php/FAQ/Beryl](http://wiki.beryl-project.org/index.php/FAQ/Beryl) :

> **What is the difference between freedesktop.org compiz and beryl?**


    * Uses flat file backend instead of gconf meaning almost no gnome dependency.
    * Needs "beryl-settings", an application to configure settings instead of gnome-xgl-settings or gconf-editor.
    * Has a lot of extra plugins and enhanced features in other plugins.
    * Starts with beryl-manager.
    * Has a custom themeable decorator (emerald), over Compiz's metacity theme support.
    * Has a theme manager (emerald-themer), over default gnome theme manager.
    * Changes frequently: new features are added on a daily basis.
    * Made by a community right here at forum.beryl-project.org, although submissions of extra plugins and enhancements from here to freedesktop would make the compiz better over all.

### Post by maniacmusician on 2006-12-13
the difference is basically Beryl gets updated much more often and has more features at the cost of less stability. A lot of people (like me) make it riskier by using the svn versions of Beryl, which are updated even more often and are always the cutting edge. Don't know about any good howto's for compiz, but here's the beryl one that worked for me: [http://wiki.beryl-project.org/wiki/Install/Ubuntu/Edgy/AiGLX](http://wiki.beryl-project.org/wiki/Install/Ubuntu/Edgy/AiGLX) , though I believe it may not have the correct unstable repositories on that page. At least, I'm using slightly different ones.

As you can tell from the url, it's only for edgy users.  here's one for dapper users ([http://wiki.beryl-project.org/wiki/Install/Ubuntu/Dapper/AiGLX](http://wiki.beryl-project.org/wiki/Install/Ubuntu/Dapper/AiGLX) ) but you'll have to install AIGLX + beryl. unless you have ATI, in which case, you'll probably want XGL. 

The main page for the differing howtos: [http://wiki.beryl-project.org/wiki/Install/Ubuntu](http://wiki.beryl-project.org/wiki/Install/Ubuntu)
