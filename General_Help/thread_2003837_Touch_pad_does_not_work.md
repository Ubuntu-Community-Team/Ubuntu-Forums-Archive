The screen then displays the desktop and the computer thereafter works normally except that the touch pad does not work at all.  (I use a blue tooth mouse to move the cursor.) ---

### Post by choochoousmc on 2012-06-14
go into mouse settings, and enable touchpad while using a mouse.
This should do it. If not you will need to use a terminal command.
However, I prefer the touch pad disabled if using the mouse. So I don't accidentally hit the pad and cause something to happen I don't want to.
