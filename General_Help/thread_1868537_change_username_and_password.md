### Post by WasMeHere on 2011-10-24
I suggest that you keep your user id as a 'guru' user with administration privileges and that you create a user for your father with his name and password. I think you know him well enough to decide if he should have administration privileges (that he is allowed to run ***sudo***). In the old gnome desktop environment you select

*System -- Administation -- Users and Groups*

to create users and change user privileges. The GUI program's name is

```
users-admin
``` and you can call it from the terminal.

Have fun finding out about Ubuntu :-)
Olle
