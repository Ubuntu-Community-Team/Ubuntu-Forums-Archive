### Post by Frogs Hair on 2012-02-28
Hello comradeluigi 

Could you give a better description of what you have changed and which icon set ?  You can change the path to an icon image that you would like to use from the main menu.

Open the main menu , locate the application , and click on the line where the application is displayed to enter properties. When properties open select the icon picture navigate to the directory where the image you would like to use is located .
