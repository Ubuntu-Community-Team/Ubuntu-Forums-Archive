To un-install boot from Live CD. Go to System/Administration/Partition Editor. 
Select the Ubuntu partition and delete it or just reformat it.

Then, using Live CD, Re-install Ubuntu back onto the same partition.

To change the boot order you will have to probably press the delete key when starting the computer and designating there the boot up order. You will want CD to be first. You may also be able to press the F10 key while booting to quickly select the boot device. Your system may be different but that is generally how you can accomplish that.
