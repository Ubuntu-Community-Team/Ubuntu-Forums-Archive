### Post by whitethorn on 2008-11-30
Right click on Cairo-dock -> Add a manual launcher -> Command to launch on click 

```

nautilus --browser /path/to/your/folder 
```
Then all need is an Icon for it. Now when you click on it, it should open Nautilus in the folder you specified.
