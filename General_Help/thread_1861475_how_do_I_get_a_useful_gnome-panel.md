### Post by ajgreeny on 2011-10-15
If you don't like unity I don't think you will like gnome-shell either.  Try using **gnome-session-fallback**, which you can install from the repos then reboot and choose the ubuntu-classic (I think it's called that) from the login screen.

That will give you two panels like the original gnome2 classic desktop, but you can remove one of them and add the applets you want to the remaining one by using Alt+Right Click in the panel.  The applets now auto-position themselves left, right or centre, but you can reorder them, again using Alt +Rt Click.
