### Post by P4man on 2010-07-29
wubi doesnt use grub, it uses windows bootloader. "reinstalling grub" wont do you much good.

Unfortunately I dont know much about wubi other than how to avoid it :). But I would consider booting a windows CD and restoring the windows MBR with the windows recovery tools.  That may remove ubuntu option in the boot menu; if so;  just add it again in boot.ini (see your own output above).

If that works, backup your ubuntu data and reinstall ubuntu but not using wubi but by booting off a livecd (or usb stick) and install ubuntu on to its own partition. 

Wubi is okay if you want to try it out but its really a bad plan to have it depend on windows filesystem. Almost impossible to recover data if anything goes wrong, cant change size etc etc.
