### Post by AgentZ86 on 2014-02-03
Hi, just reviving some posts that I've seen and youtube vids that actually didn't work for me

-download team speak 3
-install the .run file by simply typing the name of the file with a ./ in front of it
-example: ./teamspeakfile.run
- I would do this in the terminal and cd Downloads, then ls to see the file name you want
-highlight the file name and copy
-type ./ in the console and paste the file name afterwards and install BOOM you got team speak 3
- launch and right click the icon in the left of the launcher panel and lock launcher

Happy team speaking.
I have not messed with the overlay though in linux it does not appear to be installed by default I'll play with that later
