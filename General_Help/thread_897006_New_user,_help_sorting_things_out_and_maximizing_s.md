### Post by Peter09 on 2008-08-21
If Ubuntu is suggesting 400 odd updates then do it. Don't worry too much about it as the updates will run quite smoothly. I guess you have used an older release of Ubuntu.

Formatting will have removed all the old data from the hard drive so no need to worry.

I suggest that you do not remove anything until you understand Linux a bit better. You have plenty of Hard Drive space left so explore the O/S a bit. 

Go to System->Administration->Synaptic Package Manager and browse all the applications that you can install.
