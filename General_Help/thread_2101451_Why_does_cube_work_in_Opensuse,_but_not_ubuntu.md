### Post by stinkeye on 2013-01-04
> **Statia said:**
> Because Opensuse uses Gnome (or KDE) and Ubuntu uses Unity.
Ubuntu uses the Gnome Desktop Enviroment.
Unity is a plugin for the window manager compiz.
The cube works in both 12.04 and 12.10.
Setting up the cube in 12.10 is less problematic than 12.04.
