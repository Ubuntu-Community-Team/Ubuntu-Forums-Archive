Ok just figured this out by adding the created file inside "lightdm.conf.d" folder:
Add this:
```

[SeatDefaults]
greeter-hide-users=true
greeter-show-manual-login=true

```
```
sudo nano /etc/lightdm/lightdm.conf.d/00-hide-user-list.conf

```
either restart your system or the lightdm service.

Well now you got me thinking, so I wanted to see if Arch would also work with this and it dose.
Steps I took for Arch. (sholud aply for ubuntu also)
NOTE: If you have AccountsService installed on your system, then LightDM will use this.
Terminal: 
```
cd /var/lib/AccountsService/users/
```
Check to see if your user is listed:
```
ls
me

```
Now add this:
```
sudo nano me.conf
```
Change "me" to what was listed as user.
Now add this:
```
[User]
SystemAccount=true
```
save and close.
Now on a arch install there was no Directory for "lightdm.conf.d" so I made one.
```
sudo mkdir /etc/lightdm/lightdm.conf.d

```
And you now know how that now reads:
```

greeter-hide-users=true
greeter-show-manual-login=true

```
