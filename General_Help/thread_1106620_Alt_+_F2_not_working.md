### Post by ooburns on 2009-03-25
In CompizConfig Settings Manager, go to the key bindings tab under general options. Is there any combination defined for "Run dialog"?

Edit: That is, if you have compizconfig-settings-manager.

---

### Post by mb_webguy on 2009-03-25
You must have Alt+F2 set as a shortcut for a Compiz plugin.  Go into CCSM (Advanced Desktop Effects Settings) and start unchecking plugins and trying Alt+F2.  If the key combo starts working, look at that plugin to see what Alt+F2 is assigned to, and change it to something else.
