### Post by dbdexter on 2011-09-08
well, if you want to try ubuntu, you can install it on a pen drive. Personally, I've never tried using wubi, I just downloaded the .iso and then with UNetBootin I decompressed the LiveCD in my pendrive.then you can install ubuntu while keeping windows on your hard disk. it's not so difficult.
Here you can download the .torrents, i386 if your pc has a 32 bit architectire, amd64 if it's 64 bit: [http://www.ubuntu.com/download/ubuntu/alternative-download](http://www.ubuntu.com/download/ubuntu/alternative-download) 
P.S. you don't have to decompress the .iso as it is a compressed archive. Just burn it on a cd with programs like cdburnerxp or, as i said before, decompress it in a pen drive with UNetbootin.

### Post by ajgreeny on 2011-09-08
If you downloaded the iso file with a torrent client you will not need to check the md5sum; that is done by the torrent client automatically for you.

If your computer has a CD drive I suggest you use cdburnerXP or your current burning software but make sure to burn an image of the iso (not just copy the iso file) to CD.

Now boot from that CD and do a proper partitioned install.  Use the "Try Ubuntu" option first to run the live CD which will show up any hardware problems that may occur.
See [http://www.psychocats.net/ubuntu/installing](http://www.psychocats.net/ubuntu/installing)
