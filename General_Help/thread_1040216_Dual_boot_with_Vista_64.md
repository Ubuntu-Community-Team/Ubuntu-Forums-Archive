### Post by logos34 on 2009-01-15
lot of people ask that question...you can dual boot 32- and 64-bit OSs no problem (I used to have x64 xp pro and 32-bit ubuntu).

You can run a x86_64 OS only if the CPU supports it (a lot of newer dual-core processors do both).  The big advantage of x64 is it supports over 4 GB RAM.  Certain processor-intensive are faster in x64 due to 64-bit libs.  Check out this thread comparing the two (ubuntu at least):

[http://ubuntuforums.org/showthread.php?t=765428](http://ubuntuforums.org/showthread.php?t=765428)
