Hi,

Because my mobile broadband has a 5GB/month bandwidth cap, I have looked for a good GUI bandwidth monitor, but so far haven't found any. Instead, I use the command line tool iftop. It does not show what applications are accessing the Internet, but it does show what URLs are being accessed and how much bandwidth is being used now and over a period of time.
