### Post by snowpine on 2010-03-21
Ubuntu should NEVER ask for your root password. The root account is locked in Ubuntu. If you unlocked it somehow, you will have to seek advice elsewhere... :(

If you want to re-lock your root password (as the Ubuntu developers intended): [https://wiki.ubuntu.com/RootSudo](https://wiki.ubuntu.com/RootSudo)
