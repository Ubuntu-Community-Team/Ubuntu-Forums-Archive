### Post by ricardisimo on 2007-09-15
First try "alsamixer" in a terminal. Navigate with your arrow buttons to make sure the obvious outputs are not muted. "M" toggles mute in whatever channel is currently selected.

I had the same problem way back when in Edgy that was only solved when I upgraded to Feisty. I went several weeks without sound, sadly. Hopefully we can get yours fixed sooner.
