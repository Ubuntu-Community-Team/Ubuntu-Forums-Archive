### Post by Pard68 on 2011-07-07
You don't have anything to worry about in Ubuntu (or any Linux distro). No viruses will harm you. I think the last virus made to attack Ubuntu was made useless a few years ago. 

Besides that it is simply not economical to put out linux viruses. How many people run Linux when compared to Windows? Any virus you can find is geared towards Windows because the chances are much better that a person is running Windows than Linux. 

If you really are afraid of getting a virus in Ubuntu than just run in a VM and turn of sharing. 

And all users are more or less "strictly limited" in rights until you put that password in. So if a virus does get into your system it is because you gave it permission to do so.

If you want more info... [http://ubuntuforums.org/showthread.php?t=765421](http://ubuntuforums.org/showthread.php?t=765421)
