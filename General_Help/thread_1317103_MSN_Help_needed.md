### Post by sugarat on 2009-11-06
To communicate with your friend on the MSN network you will need an account on it. A number of chat clients will connect to it so you don't need the MSN Messenger program, just an account on that network.

### Post by Ownager on 2009-11-06
Hi there, you can use pidgin. It gives you opportunities for chatting with your friends in all messengers. I am sure that it supports your email for gmail. 

You can have a good looking msn. That would be Amsn

You can type this in terminal. 

```
sudo apt-get install amsn
```

When it doesn't let you to access your email address, you should uninstall amsn before you can go to 

[http://www.getdeb.com](http://www.getdeb.com)

That website has a new version for amsn. You may download it from there. 

I don't know what version for ubuntu are you using now.

### Post by Jr.Muffin on 2009-11-06
You first will need an MSN (or Windows Live) account. You can [sign up here]("https://signup.live.com/signup.aspx?ru=http%3a%2f%2fmail.live.com%2f%3frru%3dinbox&wa=wsignin1.0&rpsnv=11&ct=1257533485&rver=6.0.5285.0&wp=MBI&wreply=http:%2F%2Fmail.live.com%2Fdefault.aspx&lc=1033&id=64855&mkt=en-us&bk=1257533486&rollrs=12&lic=1").

Once you have an account, you will need a program to chat with your friend. I suggest Emesene. You can install it from the Software Center.

Once you have Emesene, the only thing you need to do is to login using your MSN account and add your friend to start chatting.
