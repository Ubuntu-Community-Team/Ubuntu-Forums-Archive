Hello everybody,

I was wondering if thunderbird has a feature like that:

I would like to send some mails regularly, to certain friends of mine.
So I would like to make a group, let's call it "pals", with all the email addresses of my friends and when I put "pals" in the "To:" field, thunderbird will send my message to all those people in the group..

Is something like that possible?

Thanks a lot in advance!

---

### Post by Denestria on 2008-08-26
Click address book.  Click on New List.
Fill in the mailing list name, description and nickname.
Add addresses by typing them in the list.

Then when you send mail to this group of friends typing the nickname in the address field should be all you need to do.
