### Post by raktarok on 2010-01-29
Please put a # at the beginning of the line 56 (ie the last line) The line should look like this now.

```
#deb http://ubuntu.org.ua/getdeb
```

Make sure you open the file like earlier:
```

gksudo /etc/apt/sources.list
```

The file holds the software sources(from where you download all the software packages). If you look at other lines, you will see that there are words like karmic at the end of each line, and line 56 does not follow this format. That is the cause of your problem, and putting # before the line means that the line will be ignored now.

In the end, do this to reload the software info. (or use the check for updates button in the update-manager).
```

sudo apt-get update
```

You should have a working system back now.
