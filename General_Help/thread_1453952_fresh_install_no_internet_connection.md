### Post by skaparate on 2010-04-14
Hello, I'm not a guru of linux, but i want to learn as well as help, so i will try to help you. First of all, if you are not behind a router, then you should try the following for DSL:

1.- Rigth click on the NetworkManager and then click on Edit Connections.
2.- Now click on DSL and then on ADD.
3.- You now have to fill the fields User, Service (if i'm rigth, this should be any name, like connection, entel, movistar, etc.) and Password wich should be provided by your Internet Service Provider (ISP).

I hope it helps you.
