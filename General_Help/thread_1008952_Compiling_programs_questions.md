Hello I just wanted to know if I compile a program from it's source
a tar file, do I have to recompile it after a kernel upgrade?

---

### Post by jespdj on 2008-12-12
No, only kernel drivers have to be recompiled.

If it would be the case that any program would have to be recompiled when there's a new kernel, then it would be almost impossible to update the kernel... (imagine what it would mean if all of the thousands of programs on your computer would have to be recompiled!).
