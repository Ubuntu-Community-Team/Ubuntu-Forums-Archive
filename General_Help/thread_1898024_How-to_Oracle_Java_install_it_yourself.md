### Post by Pjotr123 on 2011-12-20
All Oracle (Sun) Java packages have recently been removed from all repositories, because of license issues. And existing installations have been undone, because of security risks.

However, you can still install the newest Oracle Java JRE yourself: either by hand, or by applying an installation script that pulls the newest Oracle Java from the Oracle website and installs it. This script has been made by a respected member of the Dutch Ubuntu community, complete with it's own repository (duinsoft.nl). This approach is still allowed by the license (comparable with the procedure for Adobe Flash Player).

The choice is yours!

By hand (moderately easy):
[http://sites.google.com/site/easylinuxtipsproject/java](http://sites.google.com/site/easylinuxtipsproject/java)

By means of a script (very easy):
[http://www.duinsoft.nl/packages.php?t=en](http://www.duinsoft.nl/packages.php?t=en)

Have fun! :-)
