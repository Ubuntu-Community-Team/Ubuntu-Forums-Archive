### Post by unamanic on 2009-11-08
Like most Adobe products, there isn't really anything that compete feature for feature.  Some versions of Dreamweaver can run in wine if there are must have features.  As far as linux php editors, I like bluefish.  Phpeclipse isn't too bad either, but will take some work to get installed since it doesn't appear to be in the repos.

Mostly what you are going to get from linux editors is auto indenting, syntax highlighting, and some code completion.  You aren't going to find many (if any)  WYSISYG editors the do that are stable and do more than static HTML.

### Post by unamanic on 2009-11-08
Blue fish is a code editor, there is nothing on Linux (or Windows for that matter) that matches what Dreamweaver can do in the "Design" mode.  I do my edits in code, I edit on a test server on my LAN, view them in Firefox, IE (in a virtual machine), and/or Arora, and when I'm stisfied I upload to my production server.  This way edits are cleaner than a WYSIWYG editor can give you, and I have the opertunity to see how it is rendered on more than one browser.
