### Post by SecretCode on 2011-05-09
According to [Search Term Reference](http://algebraicthunk.net/~dburrows/projects/aptitude/doc/en/ch02s03s05.html)
```
aptitude search ~N
```
will list the new packages. Not sure if that is the same definition of "new" though
