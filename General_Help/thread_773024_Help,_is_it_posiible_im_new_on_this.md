### Post by eriqjaffe on 2008-04-28
The main thing to keep in mind is that Ubuntu is a completely different operating system, so (naturally) Windows applications don't run.  Just like OSX in that regard.

That being said, there is Wine, which allows **some** Windows applications to run, although there are never any guarantees of compatiblity with Wine.

The other thing to remember is that for most Windows applications, there are Linux equivalents - often more than one.  The list of media players with functionality similar to iTunes (for instance) is mind-boggling.  Sure, there are specific-purpose Windows apps that don't have an equivalent, but they are decidedly in the minority.

### Post by eriqjaffe on 2008-04-29
> **roberto2003al said:**
> i have downloaded the latest release from wine and what i received is something like a compressed package with a lot of stuff inside... i don't understand anything what am i supposed to do next?? from what it seems a lot of programs for Ubuntu seems to function the same way like for example a prog that sounded cool was beryl and when i downloaded what i thod was the program it was the same a package full of stuff how do i install or run this stuff??You have, in all likelihood, downloaded the source code (sometimes referred to as a "tarball").  If it ends in .tar.bz2 or .tar.gz, or something with ".tar" in it, it's going to be source code, which needs to be manually compiled into an executable format.  Given your unfamiliarity with Linux, I would **strongly**advise you to steer clear of this.

The best way to install software is just to go through Synaptic, which is the preferred (and, certainly, easiest) method for installing things.  Pretty much every mainstream package for Linux can be found through it.

[https://help.ubuntu.com/community/SynapticHowto](https://help.ubuntu.com/community/SynapticHowto)

If there is a package you want that isn't in Synaptic, look for things with a ".deb" extension - those are precompiled binaries, the Ubuntu equivalent of a "setup.exe" file.  You should be able to install those with a double-click.
