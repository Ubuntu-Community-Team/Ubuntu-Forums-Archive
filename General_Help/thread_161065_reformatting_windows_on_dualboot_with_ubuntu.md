### Post by hollywoodb on 2006-04-16
installing windows **WILL** remove grub so you won't be able to boot ubuntu the way you're used to.  However there are multiple threads in these forums on how to install/reinstall grub.

from my experience with WinXP installs, it has a little partition manager that should let you select which partition to install to.  since your partitions are already in place you should be able to boot your WinXP CD and select the desired partition when you get to the part of the install.  if you get there, and aren't sure what to do or can't find the correct partition/option you can still safely reboot and your system will be unaffected.
