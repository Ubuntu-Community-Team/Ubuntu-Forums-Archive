### Post by TenPlus1 on 2009-02-11
Are you using Pidgin in a virtualbox running XP, or ubuntu's native version ???

For XP, uninstall Pidgin, remove any directories it may have left behind and run ccleaner to tidy up registry (available at [www.filehippo.com](www.filehippo.com))

for Ubuntu, goto [http://www.getdeb.net/app/Pidgin](http://www.getdeb.net/app/Pidgin) to download the latest version (8.10 Ibex is default version to download, click on TYPES to select 8.04 Hardy version if needed)...  install in following order: pidgin-date, libpurple then pidgin but make sure you remove the old one 1st...
