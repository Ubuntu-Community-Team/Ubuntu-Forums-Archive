### Post by Ralph L on 2015-09-27
I am running xubuntu 14.04 LTS.  I found I could add items to the Thunar "Go" menu by setting View>Side Panel to Shortcuts and right-clicking a folder and selecting Send To "Side Panel".  Although this is somewhat awkward (since I like to keep Side Panel set to "Tree"), it works.  However, I don't know anyway to delete items from the "Go" menu.  My Go menu is getting too large.  Can anybody tell me how to delete items???

---

### Post by Toz on 2015-09-27
Switch back to Shortcuts view, right-click the links (from the shortcuts pane) that you want to remove and select "Delete Shortcut". The links that you add via Sending to Side Pane end up in the Go menu. Those are the only items in the Go Menu that can be removed.
