### Post by Yougo on 2011-01-07
did you install thunderbird from the software center, or manually from the internet? i did the latter once and it didn't play nice with the launcher either. one time a reboot made it appear in the menu (somehow it didn't refresh i guess)

if you hit alt+F2 and type'
```
thunderbird
```
and hit enter, does thunderbird start?

if so then you can create a launcher on your desktop by right clicking the desktop, choosing Create Launcher, and fillingin the needed info.
Or you could right click the applications menu and choose edit menu. there you can add a launcher in your desired place, or see if it's there already, but not active.
