### Post by Qrk on 2006-04-16
System->Administration->Login Window->Background Color

For the splash screen, you can download them at [www.gnome-look.org](www.gnome-look.org), and select them using the splash screen app in Administration. (at least I think you can in Breezy, I can't remember if Breezy had that or not. 

If not, change the seeting in Applications->System Tools->Configuration Editor->apps->gnome-sessions->splash_image 

Set this to the path to the splash screen you've downloaded.
