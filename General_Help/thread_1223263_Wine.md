### Post by themusicalduck on 2009-07-29
You don't really run wine as such. Just use .exe files as you would on windows by double clicking on them. There should be a menu under Applications called Wine, with a programs menu, configuration and shortcut to browse the virtual c drive.

Sometimes it's worth installing newer version of wine by using this guide: [http://www.winehq.org/download/deb](http://www.winehq.org/download/deb) and looking up windows programs to see how they run using this site: [http://appdb.winehq.org/](http://appdb.winehq.org/)

### Post by jerome1232 on 2009-07-29
Well you double click the .exe that you want wine to run.

If you want to create a launcher for an app, right click your panel, click custom launcher, 

fill in the details, the command will be

wine /path/to/progs/exe

I hope that helps a bit.
