### Post by Seeked on 2010-08-11
The reason Netflix doesn't support Linux is because Microsoft will not let them.  If Silverlight (Microsoft's proprietary DRM software) which is required to play the Netflix movies would run on Linux it would give PC users one less reason to run Weendows.  Microsoft doesn't want to support Linux.  However, YES there is some form of Netflix support for the Roku which is running Linux but all the smart Linux users still can't figure a hack out to export that capability to your PC (sad face).  I think Netflix would like to support Linux but maybe they are under some contractual obligation with Microsoft for Silverlight who is preventing them from letting us use their service that we are already paying for.
