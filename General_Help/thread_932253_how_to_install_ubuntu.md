### Post by steveneddy on 2008-09-28
Put in the CD and when you see a desktop, see if everything works.

If so, hit the install icon.

Make sure internet, firewire, USB - you know, all of the stuff you will need when you install it.

Just make note of the empty partition you want to put it on, and for Linux you need an extra partition called Swap. Only needs to be about 1 Gig.

### Post by howefield on 2008-09-28
The OP might struggle to run the live cd install, system requirements are for 384megs of ram. Probably need to use the alternate install cd.


System Requirements

Ubuntu is available for PC, 64-Bit PC and Intel based Mac architectures. At least 256 MB of RAM is required to run the alternate install CD (384MB of RAM is required to use the live CD based installer). Install requires at least 4 GB of disk space.
