Is probably a messed up Firefox profile. I would start by trying to [optimize the profile databases](http://www.webgapps.org/tutorials/firefox/optimization/database-optimization) and starting Firefox in safe mode, to see if an extension is causing the problem.

I would also recommend using Flashblock, NoScript, AdBlock and Ghostery add-ons to block stuff you don't need. It might be a script on the page causing the problem.
