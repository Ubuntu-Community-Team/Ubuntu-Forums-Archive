### Post by HermanAB on 2008-12-26
Well, a properly configured Linux system should not ever crash.  I have systems that are running for multiple years without ever being restarted.

So, you got to figure out what is the matter - this is usually best done through a process of elimination.  You imply that crashes are caused by Firefox, so try Opera and see if that one is stable.  If Opera is stable, then remove Firefox, delete all references to it and then install the newest version off the Mozilla web site.

### Post by HermanAB on 2008-12-26
Well, saying that 'Ubuntu is unstable' doesn't mean much.  Linux is Linux is Linux - deep down, all versions of Linux are the same.  So, you actually have a specific application program that is causing you trouble - you got to figure out which one and then either find a substitute or install an upgrade/downgrade.  

A generic complain won't get you any sympathy, since the system works properly for most everybody else.  So, you got to do the sleuthing yourself.

by the way, Debian 'unstable' means that it is still under development and packages are still changing.  It doesn't refer to reliability, but rather to changing features.

Cheers,

Herman
