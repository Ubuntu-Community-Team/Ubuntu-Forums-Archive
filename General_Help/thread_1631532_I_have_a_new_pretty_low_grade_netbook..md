### Post by themusicalduck on 2010-11-26
I'm pretty sure you can run Lubuntu as a live session.

However, I have heard that some of the cheapest netbooks are unable to boot from USB. Can you actually access the BIOS on it? Hopefully it's possible on yours.

Also, 700mhz cpu and 128mb ram is quite low. I think even Lubuntu may struggle a little, but I don't know for certain.

I installed Lubuntu onto a laptop with 256mb ram and I think a 1.6ghz cpu and it ran OK, but it wasn't lightening fast. That is double the specs that you have. It's worth trying a live session though.

There are lighter distros of Linux, I think Puppy Linux is one, others will probably have more suggestions.
