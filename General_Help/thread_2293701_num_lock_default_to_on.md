### Post by XBNCPRk on 2015-09-07
On 12.04 youll have to install it using numlockx. Rather than cut and past someone elses instructions though: [http://askubuntu.com/questions/155679/how-to-enable-numlock-at-boot-time-for-login-screen](http://askubuntu.com/questions/155679/how-to-enable-numlock-at-boot-time-for-login-screen) 

Note that there are two methods for it, one that will enable it with boot, and one that will enable it with login. 

Simply setting the number lock key behaviour in BIOS will not necessarily cause it to be enabled in your environment.
