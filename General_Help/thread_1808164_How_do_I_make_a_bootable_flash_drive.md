### Post by linuxman94 on 2011-07-20
You should be able to make a usb stick using [unetbootin]("http://unetbootin.sourceforge.net/").  Just point it to the iso you want to load.

That won't work for the thinkpad 600 because it does not support booting from usb.  You'll have to make a cd.

Edit:  You should get connochaetOS.  Deli linux was renamed to that.
