### Post by amanchesterman on 2014-09-16
Yes I get that warning too. It's because Calc allows you to set out left and right pages with different headers and footers (see the header + footer dialogue). And I assume that's because Calc is part of an office suite whereas Gnumeric isn't. In the suite, with Writer, users might want to set up left and right pages of a text document differently, and that functionality has been carried over into Calc by the developers.

I've no idea how you would turn the warning off and I doubt if you can. It's not confined to Gnumeric I think. I have a dim memory (most of my memories are dim these days!) that at one stage there was something like it in MS Word: if you used an old version of the program to open a document created with a later version, you got a warning notice.

So if you want to be happier, stick to using one program, or use .xls format. ;)
