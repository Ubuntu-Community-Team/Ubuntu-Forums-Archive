### Post by StuartN on 2010-05-31
Single-window mode should be ready in the next major Gimp version, by about December.

One simple fix is to lock your window positions where you would like them if it was a single window. Start by laying out the tool palettes and the editing window as you like them, then go to Edit -> Preferences, Window Management. Unselect "Save Window Positions on Exit" and click "Save Window Positions Now". You can still open, close and rearrange windows while working, but Gimp will always start with the configuration that you have saved - and obviously you can save this again if you need to change it.

Have a look at GimpShop for related ideas and extras [http://www.linuxlinks.com/article/20070724143611960/Gimpshop.html](http://www.linuxlinks.com/article/20070724143611960/Gimpshop.html)
