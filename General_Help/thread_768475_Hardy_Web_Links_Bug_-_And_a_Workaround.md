### Post by Bill_MI on 2008-04-26
You're at a site in Firefox and create a desktop Link of the URL by dragging the site icon (just left of the URL) to the desktop and a "Link To...." is created.

Then you use it later and it doesn't work!

I found if a "?" (question mark) appears in the URL the string will truncate right there.

No problem with program launchers with URLs passed to Firefox,
No problem with bookmarks.
Just these *.desktop files!

A possible workaround to correct the Link:
Right click -> Properties
Tab -> Link
In the URL, change all ? (question mark) to %3F (percent-3-F).
There's probably only one.
The link may now work.

EDIT: This doesn't work on all - including THIS message!
It did fix a bunch of old links I have.
All additional input welcomed.
