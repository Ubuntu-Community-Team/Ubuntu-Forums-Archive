### Post by amanchesterman on 2014-07-29
I looked into this a few months ago and yes, my LO Writer (4.2.5.2 -- same as yours) can **read** MS Publisher documents I've been sent. I don't know which version of Publisher was used to produce them, but I would guess it's recent (2010 at least) since the organisation that produces them uses an up-to-date version of MS Office. However Writer can't **edit** the files; but maybe you don't need to do that?

If Publisher has exported the document as a PDF, then yes you can open it in LO Writer; but if all you want to do is to **read** the document, then personally I find it quicker and easier to use a dedicated PDF reader / document viewer.
