### Post by DanneStrat on 2011-02-08
Hi!

You can install "compizconfig-settings-manager" and

"compiz-fusion-plugins-extra" from synaptic.

The plugins extra package will give you more effects and the 

settings manager will make it easier to configure desktop 

effects.

If you have any more questions feel free to ask.

Good luck!
