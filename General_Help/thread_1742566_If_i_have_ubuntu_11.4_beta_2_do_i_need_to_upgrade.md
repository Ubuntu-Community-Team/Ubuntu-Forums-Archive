### Post by kaldor on 2011-04-29
If you have applied all updates, you are currently on Ubuntu 11.04 :)

If not, you can update packages via this command:

```
sudo apt-get update && sudo apt-get upgrade
```

There's no distro upgrade like 10.10 to 11.04 :)
