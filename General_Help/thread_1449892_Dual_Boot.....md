### Post by aklo on 2010-04-08
Since it is a new comp, i strongly suggest you create a different partition just for kubuntu...this way you don't have to worry about anything. If it is the same partition, then do like what 2nd post said , defragment your hard disk before installing kubuntu.

Also install windows first before installing kubuntu so that dual boot is setup automatically for you with no configuration on your part. Perfect for new users.
