### Post by fooman on 2008-12-10
install the compizconfig-settings-manager if it is not installed already:

```
sudo apt-get install compizconfig-settings-manager
```

after it installs,  find it in system > preferences > compizconfig settings manager...on the left side, click: general > general options > desktop size

try the "horizontal virtual size" setting.

EDIT: oops...sorry.  in kubuntu,  find it in applications > settings > compizconfig settings manager
