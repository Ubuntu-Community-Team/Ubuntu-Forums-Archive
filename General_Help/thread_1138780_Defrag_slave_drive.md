Fragmentation is not operating system specific it's file system specific.

If your slave drive is fat/ntfs then it will fragment regardless of what operating system you use more often.

The Windows defrag tool has always been horrid. I'd suggest getting a 3rd party defrag tool.

I'm not aware of any MS file system defrag tools for Linux. I'm sure they exist.. somewhere?
