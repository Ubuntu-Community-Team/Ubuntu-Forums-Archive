> **blakep2 said:**
> can anyone point me in the right direction.  I am looking to connect to a remote network.  I do not want a remote desktop then access the network that way.  I would like to connect to the network as though i was physically connected to it.  I can designate a computer as some sort or server if needed.

You need to set up a VPN from one location to another.

This can be done if the network equipment at the remote end has VPN capability, or if some other equipment at the remote end also has that capability.

Most clients have VPN capability, but you need to check of it is compatible with whatever VPN is used at the remote end. The other option is to have the network equipment itself set up a VPN tunnel so the remote network is always accessible from your own network.
