### Post by RAOF on 2005-10-30
I'm pretty sure wine won't help you here.  Wine does not run windows under linux, it sort of translates windows programs so they run on linux*.  So the programs you run under wine can only use the stuff that your linux install can use - ie: has drivers for.  Trying to install the drivers for your usb network adapter under wine won't work.

* - This is not meant to be a techincal explanation.  It is incorrect, but it gets the point across :)
