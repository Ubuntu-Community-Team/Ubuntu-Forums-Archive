### Post by bchurchill on 2009-08-14
You are downloading a reasonably large file (probably around 700MB).  If your connection isn't stable then the download will take a long time.  If you're on a wifi network, try getting a wired connection instead; they are often much faster.  You can also order an Ubuntu CD online (I'm pretty sure it's free).

### Post by 3rdalbum on 2009-08-14
I noticed last night that the Canonical servers were running a bit slowly.

Some download managers will attempt to connect multiple times to the same server to try and download quicker - if you are using a download manager rather than just your web browser, then this might be why it's showing two simultaneous downloads.
