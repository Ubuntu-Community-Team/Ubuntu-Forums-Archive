### Post by zj3t3mju on 2008-06-28
To help all users who don't have an internet connection like me.
I writing a tool call 'wapt-get'.
It can get link of any package and its depends similar to apt-get do, but it run on windows

Project at: [http://wapt-get.googlecode.com/](http://wapt-get.googlecode.com/) 

Before use this program you must config it by modify the wapt-get.conf and sources.list

USAGE: double click to wapt-get.exe and choose a task from menu

I wish it useful!!!!!
