Just a thought: every now and then, after an update (I haven't narrowed it down to updating what, yet), I get the BBC iPlayer (both audio only and video) and various other streaming sites stop working.

Yes, I am outside the UK, and it works fine. The associated web site might be of some use to you, too.

Go to [www.reciva.com](www.reciva.com) (the BBC stations start at [https://www.reciva.com/stations/location/255?&start=119&count=10](https://www.reciva.com/stations/location/255?&start=119&count=10)) and you can see the listen again shows by clicking on the station name, or play now by clicking the Listen icon. Whichever you pick, a small player window opens which shows the staion and Playing Now. If you right-click in the little player window and View Source, then somewhere is an <iframe src=" line, which has the radio station url, e.g. this is <iframe src="http://www.bbc.co.uk/arabic/meta/tx/nb/arabic_live_au_nb.asx" for BBC Arabic.

In Rhythmbox you can then either select Music -> New Internet Radio Station at any time, or first select Radio in the Library list and then click the New icon that appears in the toolbar. Pasting just the URL [http://www.bbc.co.uk/arabic/meta/tx/nb/arabic_live_au_nb.asx](http://www.bbc.co.uk/arabic/meta/tx/nb/arabic_live_au_nb.asx) into the address works.

Yes, Ubuntu 9.10 (with Skype) playing the BBC live and BBC Listen again outside the UK using the BBC player streams from the BBC web site, and the Real Audio streams from Beebotron, and the Windows Media Audio streams from the Reciva website. The default install of Ubuntu (9.10 or any other version) requires the installation of additional packages, as described above. The Windows Media Audio required that a plugin be disabled, as described above (although obviously there are multiple alternative variations that would also work).
