### Post by Gillingham on 2010-10-02
Hi I can help you with b) I did it a few weeks ago while the unity interface had a major bug and I haven't gone back.

If you are logging into your system each time - then when the login screen is displayed in the bottom bar, a menu will say ubuntu netbook edition - click on this and it will display the available sessions.  Choose the most appropriate one for you.  

If you are logging in automatically you will need to enable the login screen first.  Use the the Login Screen application under Application > system click on one of the option that enable the login screen (either "allow x seconds for anyone else to log in first" or Show the screen for choosing who will log in).  Restart the system then follow the above instructions.
