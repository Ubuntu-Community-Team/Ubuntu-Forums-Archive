### Post by cortman on 2012-03-01
Yes.

It's called Emacs, and it was written by Richard Stallman. Need I say more?

```
sudo apt-get install emacs23-nox
```

Like any IDE, it takes effort to learn to use it, but it's been worth it in my experience.
For the best resources on learning emacs, Richard Stallman's own [Emacs manual]("http://www.gnu.org/software/emacs/manual/emacs.pdf") is the best. Plus there's always the [Emacs Reference Card]("http://www.cheat-sheets.org/saved-copy/emacs-refcard-a4.pdf").
