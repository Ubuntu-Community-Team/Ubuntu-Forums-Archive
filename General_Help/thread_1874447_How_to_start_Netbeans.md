### Post by alco75 on 2011-11-03
In Xubuntu, it gets an entry under 'Development' ...

Otherwise, start a terminal and try: ```
which netbeans
```

If that comes out with nothing try: ```
locate netbeans
```

(Since I only need Netbeans PHP, I download it from their site and install it manually. I've never actually installed it via APT or Synaptic or the Ubuntu Software Centre.)
