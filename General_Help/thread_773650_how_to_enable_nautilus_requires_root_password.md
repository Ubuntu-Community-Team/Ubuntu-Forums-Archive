### Post by amohanty on 2008-04-29
Any installer executable that tries to install files to a folder that you as a user do not have access to will either fail silently or if the installer has support for gui's prompt you for the password. As far as _any_ executable is concerned, afaik there is no way to restrict that in nautilus, unless of course you are willing to dive into the source and patch it :)

AM
