### Post by nerdtron on 2013-07-11
"I know that feeling bro..."

It's not ubuntu. It's because Files aka Nautilus have been undergoing a lot of changes (in 13.04 that is). And one of those changes was related to removing view modes.
 There's nothing much you can do, unless you install an old version, or other file managers such as Nemo (from Cinnamon), Dolphin (from KDE), Thunar from (Xfce). 
Or maybe other users have better solutions

As for me, I'm OK now. I'm on KDE.
