### Post by Dennis N on 2011-12-20
To install:

1) Open the Theme tab (System > Preferences > Appearance > Theme).

4) Use the Install button and navigate to the extracted .tar.gz file to install the theme.

5) The installer will extract the theme and place the theme's folder in the user's ~/.themes folder.

6) Then select the new theme from within the Theme tab.
