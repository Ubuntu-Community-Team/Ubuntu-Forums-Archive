### Post by 3rdalbum on 2010-02-28
> **sofasurfer said:**
> I've figured it out using google docs.
I would also like to know how I would compress a file in Linux so that a Windows user can uncompress it.

Alt-F2

Type:

```
file-roller
```

That's the default archive manager program in Ubuntu. You can use this to create a Zip or Rar archive; but surely the popular decompression programs on Windows must understand tar, gzip and bzip2 as well?
