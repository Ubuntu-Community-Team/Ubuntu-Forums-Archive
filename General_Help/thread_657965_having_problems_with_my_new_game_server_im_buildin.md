### Post by cdenley on 2008-01-04
It sounds like a CPU or motherboard problem to me. The only way to know for sure, and to find out which, is to swap each out, and see if the problem goes away. If you built the system yourself, you might want to make sure the CPU and heatsink are seated correctly with no bent pins, and make sure all fans are running.

I'm assuming you're computer doesn't run at all, just powers the devices. Does the LED on your case show any hard drive activity? Does it beep to signal the POST test has completed? Is your power supply set to the correct voltage? I had a cheap power supply that came set to 220V which caused a problem like this.
