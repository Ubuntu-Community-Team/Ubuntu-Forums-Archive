### Post by FrogPrince on 2009-07-04
Hi. What software do you want to use exactly? Here's my advice - and I think most people would agree with me on this. For almost every application you used in Windows, you'll find a Linux equivalent that is in most cases just as good and in many cases even better. For example, itunes? Forget it: use Amarok instead. Nero? Wait until you try k3b! Now, if there really is software you can't use in Linux you have basically three (or four) options:
1) Install Wine and hope it works (touch and go)
2) Install crossover, which is basically Wine but often works on programs Wine breaks its teeth on.
3) Use virtual box (if you have the Windows CD) - very good option
4) Use VMware server to be able to boot up your Windows partition while in Linux (complicated)
5) Dual boot and boot into Windows when you need to.

Did I say four options? I meant five!
But I can't emphasize enough that the best programs to use in a Linux operating system are the Linux ones - otherwise there's little point in making the switch.

Good luck!
