### Post by stir-crazy on 2009-09-20
The problem is that Amarok has switched over to last.fm for cover management.  Reason for the switch has something to due with how Amazon is now handling their libraries.

At any rate, Amarok has the latest and greatest code out there, but the Canonical developers need to incorporate this into the repository for a fix. (I have this problem with version 1.4.10 of Amarok, so it isn't limited to Amarok2).  Maybe submit a request to the Ubuntu (kubuntu) teams?

If you're comfortable with recompiling, this information is available directly from Amarok.  I haven't tried it yet myself, but that's why the cover manger doesn't work anymore.

EDIT: To easily install Amarok 2.2 (unstable), follow the directions [_[COLOR="Blue"]HERE[/COLOR]_]("https://launchpad.net/~kubuntu-ppa/+archive/beta")  I'm doing this now myself, hopefully will have covers again in a few minutes.

EDIT2: Only have 2.0.2 for Jaunty, 2.2 is available for Karmic.  I'm not an early-adopter, so will stick it out for a while.
