### Post by arvevans on 2007-05-16
Best advice / question so far is: "does it run in Live CD mode?".  The answer to that question may be quite revealing.

Also, I have found a couple of instances where persons decided to resurrect an old PC that would not run properly on Windoze and expected that running Linux would fix a hardware problem (usually defective hard disk).  Sometimes we are victims of our own hype (of course Linux will fix anything, even broken hard drives or bad bus controllers!).

Questions:
[INDENT][LIST=1]
[*]Does it run OK with Windows, or with some other Linux distro?
[*]Are you following the installation script exactly, and not trying to do type-ahead by anticipating the questions?
[*]Does the computer "hang", or just the installation process stop?
[*]And lastly...a listing of the hardware involved is necessary to determine if there is some incompatibility involved.
[*]null

[/LIST][/INDENT]
