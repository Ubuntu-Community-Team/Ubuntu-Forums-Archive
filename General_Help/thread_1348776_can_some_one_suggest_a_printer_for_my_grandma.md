### Post by Dark_Stang on 2009-12-07
Nearly any HP printer will work. The most work you'll have to do is download the latest HPLIP software and install it.
[http://hplipopensource.com/hplip-web/index.html](http://hplipopensource.com/hplip-web/index.html)
Click on supported printers if you want to verify that it will work. But I'd recommend any of the low end photosmart printers. I'm running a networked B209m (Photosmart Plus) and it works flawlessly.

### Post by ajgreeny on 2009-12-07
Yes, go for an HP All-in-One.

I have a deskjet F2180, which is not available any more, but was the bottom of the range when I got it.  It works perfectly with 9.04 and 9.10 with absolutely no problems of any sort.  Find the updated version of that and you're laughing.

Install hplip and hplip-gui, and you will get a toolbox for it which is as good as anything in windows, in my opinion, but without all the added rubbish programs which windows users are generally almost forced to install to get the printer driver.
