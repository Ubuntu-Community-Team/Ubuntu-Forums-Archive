### Post by leg on 2008-04-19
I am using a fx5200 and am able to use it through the restricted manager. The driver is actually 1.0.9639 nvidia-glx. It is in synaptic so just install it and make sure it is the one you are using through restricted manager. The reason you have to use that particular version is that this card is considered legacy and not supported in newer drivers released from nvidia.
