### Post by RHTopics on 2007-12-10
Could be a possible hardware problem with your monitor.  First make sure all the cables are securely attached.  You could also try doing a "ctrl-alt-f1" keyboard combination to see if you can bring up a text virtual monitor session.  If you can then you may a have a configuration problem with your X server (gui setup).  If you have a Live CD available, try booting it up and see how it behaves.
