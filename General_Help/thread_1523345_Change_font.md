### Post by SmokeyThePanda on 2010-07-03
It is quite possible. Go to System --> Preferences --> Appearance --> Fonts 

I believe that is how to get to it. If it's not reply and say that it isn't. 

As for the second part, it changes the font for everything related to the desktop and any programs that do not have a different font set or programs that have "Use System Theme" or something like that.
