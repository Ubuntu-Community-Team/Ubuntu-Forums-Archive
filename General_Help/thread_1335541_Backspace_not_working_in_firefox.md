### Post by coffeecat on 2009-11-23
You're right. This default behaviour in Linux versions of Firefox is tedious because the 'back' function of the backspace key is enabled by default in the Windows and MacOS versions.

Anyway, moan over, and how to fix it.

Type 'about:config' in the address bar. Dismiss the childish 'here be dragons' message and type 'back' in the filter. About four lines down you'll find 'browser.backspace_action'. Change the 2 to a zero in the last column and you're set.
