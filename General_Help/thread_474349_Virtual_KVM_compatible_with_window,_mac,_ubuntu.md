### Post by tehno on 2007-06-15
If anyone wants to use the same keyboard and mouse for a few different machines but they have different operating systems, then here is a program i found for you to use.

I myself have it going from a vista desktop to ubuntu desktop to a vista laptop.

Go To: [http://sourceforge.net/project/showfiles.php?group_id=59275&release_id=406637](http://sourceforge.net/project/showfiles.php?group_id=59275&release_id=406637)
And download the .exe file under binary.
Install that on your windows machines.

For ubuntu go to applications->add/remove
Select All Available Applications
Search for "syn"
One of the results should be QuickSynergy, select and install it.

Once the application has been installed on the machines you want, use the configurations to set your host(the computer of the keyboard and mouse you will be using) and the position of the clients monitors.
