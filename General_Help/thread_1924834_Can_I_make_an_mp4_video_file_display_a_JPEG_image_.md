### Post by Bromden on 2012-02-15
- Install Openshot
- Install Inkscape
- Open Openshot
- Import the .mp4 and put it in the timeline
- Clik on "Title" (in the menu of Openshot) and create a new title. The first one it displays is good too. Give it a name.
- Now press on "Use Advanced Editor". It will open Inkscape with the title created.
 If it doesn't work simply click on Apply, insert the Title into the timeline (on a new track, upper than the video) and then right click on the title file created (the one in the list) and select "edit with inkscape".
- Now you're on Inkscape! Delete the titles, and import/drag&drop your .jpg into Inkscape and place it where ever you want ^^
- Save (not with name) the Inkscape sessione and close Inkscape.
-Now you should have you're Title modified for your needs on Openshot.

It works, guaranteed.
