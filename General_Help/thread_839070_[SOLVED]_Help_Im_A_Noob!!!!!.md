### Post by ad_267 on 2008-06-24
They are probably meaning to enter a command into a terminal. You can go to Applications - Accessories - Terminal to try it out. Start out with some basics like "ls" lists a directory contents. "cd" changes directory. You can type "man command_name" to get information on a command.

Commands often have different options like ls -l lists files in a "long" format.

Edit: This explains it better than I can: [https://help.ubuntu.com/community/UsingTheTerminal](https://help.ubuntu.com/community/UsingTheTerminal)

Usually you can do everything through a gui but it's just simpler and faster to explain steps using the terminal.
