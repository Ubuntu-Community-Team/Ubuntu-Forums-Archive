### Post by maverickaddicted on 2011-12-29
BCM4306 802.11b/g Wireless LAN Controller and Ubuntu - Good Old Relationship. Sometimes what happen driver is not proper loaded in the system. I have also faced this problem. However, I found a solution on launchpad and made a note of all those commands. Connect to Internet using wired connection and follow this step by step.

1)```
sudo apt-get update
```

2)```
sudo apt-get remove --purge bcmwl-kernel-source
```

3)```
gksudo gedit /etc/modprobe.d/blacklist.conf
```

REMOVE the following configuration lines in the /etc/modprobe.d/blacklist.conf file:

[B]blacklist b43
blacklist b43legacy[/B]

SAVE the change to the /etc/modprobe.d/blacklist.conf file after removing the configuration lines.

4)After that run the following command
```
sudo apt-get update
```
```
sudo apt-get install b43-fwcutter firmware-b43-installer
```

5) Then reboot your PC and run the following command:

```
sudo rfkill unblock all
```
