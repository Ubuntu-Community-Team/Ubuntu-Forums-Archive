### Post by kernelles on 2011-01-04
In Maverick, a theme installation failed. Now when I try to click on certain folders in my home directory (including home itself), instead of opening the folder, the appearances and preferences dialogue box opens with this error: 

>  There was an error installing the selected file. "home" does not appear to be a valid theme 

How can I fix this?

Thanks,
Les.

---

### Post by Frogs Hair on 2011-01-04
Right click the desktop and select create folder . Right click the folder and select open with another application . Select file browser from the menu and make sure remember this application is checked. Remove the folder when finished.  

The default application for opening home folders was somehow changed to appearance preferences.
