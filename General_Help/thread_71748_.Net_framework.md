Hi Jim

From my experience (and I love to hear anyone who has managed to load the .NET framework) you won't be able to install it through Wine. If you want to give yourself the best chance though I would make sure that you have the latest Wine and also install and setup Winetools (this will ensure that your Wine install is properly setup and configured.

My guess though is that to use the framework you're going to need a Virtual Machine like VMware though.

Good Luck!
