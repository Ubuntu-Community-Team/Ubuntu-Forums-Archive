### Post by jim_deadlock on 2016-05-30
It says Mac only but the ISO will also work for Linux. They probably have software for Windows users and provide the ISO for "other" OS, and the only "other" they can think of is Mac(!) Anyway, use Startup Disk Creator to make the CD, boot it and it should work.

A couple of points though: a) backup any data you don't want to lose and b) if the SSD is already working fine then there's really no need to mess around with the firmware.
