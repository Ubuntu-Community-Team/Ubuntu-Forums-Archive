### Post by sithsasquatch on 2008-05-04
I fiddled with the compiz settings to fix it for me. "Rotate cube" was enabled but unchecked, so checking the box repaired it.
The path to get to that setting was system->preferences->advanced desktop effects settings->desktop.
Hope this works for you guys as well.
