Fintos, Photoshop was written by Adobe for use with Apple and Microsoft operating systems.  Fireworks was written by Macromedia for use on Apple and Microsoft.  More recently, Adobe bought Macromedia who produced Dreamweaver, Fireworks and Flash.

None of this software has yet been ported to Linux but you may be able to run some or all of the Windows software on Linux using VirtualBox OSE.  But VirtualBox will need a copy of the Windows install disk.

By far the simplest is to use GIMP which is well documented and just as powerful as Photoshop and Fireworks.
