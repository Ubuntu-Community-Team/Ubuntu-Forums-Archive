### Post by kostkon on 2011-02-02
Obviously ttf fonts are supported on ubuntu (and they aren't only for Windows, since the ttf standard was developed by Apple ;))and most of ubuntu's default fonts are ttf (and otf). Thus, if you are using 10.04 or newer just double click on the font file and select *Install Font*; For older versions, just copy your font to the *.fonts* folder that exists in your home. If the folder doesn't exist, then create it yourself.
