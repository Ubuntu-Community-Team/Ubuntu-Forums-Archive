### Post by flippo on 2010-02-17
The Gentoo install CD has a minimal kernel with only a terminal interface.  Portage (Gentoo's package management) is rather large, but if you minimize that you can make a very minimal, very efficient install... but Gentoo does require a dedicated user, who isn't afraid to get their hands dirty...
