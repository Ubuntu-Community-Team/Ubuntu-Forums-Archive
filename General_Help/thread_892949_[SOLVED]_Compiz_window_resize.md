### Post by Genius314 on 2008-08-17
1. First install CCSM if you haven't already:

> sudo apt-get install compizconfig-settings-manager

2. Now go to System>Preferences>CompizConfig Settings Manager, and scroll down to the "Resize Window" plugin.

3. Change the "Default Resize Mode" to either "outline" or "rectangle". It's not as nice looking, but it should work a bit faster.
