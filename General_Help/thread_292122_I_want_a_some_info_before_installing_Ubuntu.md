### Post by ckonrad on 2006-11-06
Ubuntu dapper works great, it took me awhile to figure out how to get my graphics card working good,  and i have to use ndiswrapper for the proprietary drivers for my wireless but everything else works fine. I have never used wine for WoW but i have used it for other games. It can take a bit of tweaking to get wine to work. Dont expect wine to work for everything you try it with, if you find that you are using wine all the time then you might as well just use windows. I am fine with dapper i dont see a reason to upgrade. The important thing to me is just having a stable os that does what i need it to do, i cant really see myself ever upgrading unless i really need to for some reason. You can make a dual boot system really easy, i had a 10gb partition with windows on it at first just in case i needed it for some reason but i dont really use it at all any more and to be honest windows feels wierd to me now. I feel claustrophobic using windows because its just one big monolithic graphical user interface. Its like imagine if you couldnt pop the hood on your car to look at the engine, thats kind of what using windows feels like to me now.
