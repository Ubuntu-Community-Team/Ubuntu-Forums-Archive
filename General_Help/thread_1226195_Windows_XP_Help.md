### Post by Grenage on 2009-07-29
Bah, XP networking is normally fine.  If there _is_ a problem, running the Winsockfix util is enough to get it running again.

If you can't access the Internet, but you can the network, you're looking at a DNS or gateway issue, or the firewall blocking external access.
