### Post by Steve H on 2011-11-21
Which version are you running? In everything after 11.04, press the "windows" key and type the name of the application. It should show a list of similarly named apps. Choose one.

Failing that try ctrl-alt-T, to open a terminal window and then type the name of your app.

Hope that helps.
