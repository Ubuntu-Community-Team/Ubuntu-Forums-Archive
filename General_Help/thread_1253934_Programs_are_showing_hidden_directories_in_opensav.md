### Post by gldearman on 2009-08-30
Hi, all,

I have developed an unusual problem.  All of my programs are showing hidden directories whenever I try to open or save a file. This is a pain, since I have 60 hidden directories in my home directory, and they sort first, so I have to scroll through them to get to the directory I actually want to open.

I obviously haven't checked every program I have installed, but, at least, Firefox, the Gimp, and gedit are displaying this behavior, so I think it's universal.

When I open a Nautilus file browser window, it still does not show hidden directories by default.

Can someone please tell me how to re-hide the hidden directories, so I don't have to scroll past all of them every time?

Thanks!

---

### Post by SoftwareExplorer on 2009-08-30
Open the Open file box and right click-->Uncheck show hidden files.
