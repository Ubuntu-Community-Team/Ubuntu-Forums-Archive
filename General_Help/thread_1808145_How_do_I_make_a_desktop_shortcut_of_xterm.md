### Post by fooman on 2011-07-20
don't know if there is a better way or not,  but to make a desktop shortcut/launcher....

try right-clicking on the desktop and choose "create launcher" from the menu.

when the create launcher window opens, put xterm in the name space.  then type or copy paste the following into the "command" space:

```
/usr/bin/xterm
```

click "ok" and the new launcher should appear on your desktop.

hope that helps
