### Post by bukwirm on 2006-07-26
To compile C/C++, you need to install the package "build-essential"
For Java, you'll need any of the Java comiliers and virtual machines - the Sun versions are avaible in the repositories (universe?), as well as some others.
Perl should be installed already.
For HTML, all you need is a text editor and a browser, which should already be installed.
