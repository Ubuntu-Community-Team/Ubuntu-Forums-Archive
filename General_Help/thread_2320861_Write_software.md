### Post by mcduck on 2016-04-18
Abiword is probably a good suggestion, but here's a less known trick: Libreoffice/OpenOffice has Java enabled by default, as some of it's features need it. However having it on seriously increases the startup time, and there's pretty good chance that you'll never need any of the features that need Java...

So, go to Tools->Options, then LibreOffice->Advanced and disable the "Use a Java runtime environment" if it's enabled. If you are lucky, that might be enough to speed things to tolerable level...

edit: Or, for a super-lightweight way of typing nicely formatted documents, try some markdown editor. For example this one works nice on Linux, and can export as HTML, ODT and PDF: [http://www.mdcharm.com/]("http://www.mdcharm.com/"). (The downside is that you'll ahve to learn MArkdown syntax, although it's really simple and probably quicker than using the styles in a word processor anyway. Changing the default styles would require some CSS skills though)
