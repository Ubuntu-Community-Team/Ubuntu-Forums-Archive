### Post by ad_267 on 2009-10-19
Installing the ubuntu-restricted-extras package will get you mp3 and video codecs, as well as flash. You won't be able to play encrypted DVDs yet though, for that see [https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs](https://help.ubuntu.com/community/RestrictedFormats/PlayingDVDs).

You can install the package from the Synaptic package manager or using:
```
sudo apt-get install ubuntu-restricted-extras
```
