### Post by TeXtonyx on 2008-11-15
Sometimes during the boot up, there will be a message on the screen
"press any key to boot from cd"
other times the install will start automatically.
If it a dvd install disk make sure you are booting it from a dvd
device not a cd device and vice versa. 

Try the disk on a friend's computer, set it to boot from cd first
in the bios, they can change it back later easily enough.

If it doesn't work. Boot XP without it. Then stick it into the drive
bay and use Windows Explorer to look at the directory and file
(or sometimes it is easier to use the Dos-like command prompt)
structure. Make sure it is a XP install disk only, not one of
those dvds which have multiple OS installs in separate directories.
