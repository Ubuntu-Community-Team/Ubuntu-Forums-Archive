### Post by PmDematagoda on 2008-06-14
Since you already updated I think there won't be any necessity to do it again, to install the codecs just do:-
```
sudo apt-get install ubuntu-restricted-extras
```
which will not only install the codecs but also Java, Flash, and a host of other non-free software, but if you don't need all that then just do a search on Synaptic for gstreamer and select the ones you need.
