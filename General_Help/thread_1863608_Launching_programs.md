### Post by lovinglinux on 2011-10-18
> **jllortega said:**
> Thanls for your quick answers. I am going to start working now and I will try the solution when I get back home, but still, I find this too complicated. Just compare with Windows: you install and you can usually opt for an icon on desktop, launch bar...

If you want to start a program that is not in the launcher, click the home button, the first on the top of the launcher (unity bar), to open the Dash, type the name in the Dash search field. If the program you are looking for is the first on the search result list, you don't need to click it. You can launch it by simply hitting Enter.

You can also drag a program icon from the Dash search results to the launcher. 

If you want to change the icon order, drag it to outside the launcher without releasing the mouse button and drag it back to the position you want.

It is actually very easy to use, once you get acquainted with how it works.
