### Post by lmarmisa on 2010-09-18
Do not use your downloaded version. 

Install wine from the Ubuntu repositories using Sypantic Package Manager (System -> Administration -> Synaptic Package Manager). Search for wine, select the wine package, click right button and Mark for installation. Then select Mark and Apply.
