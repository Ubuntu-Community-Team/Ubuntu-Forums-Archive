### Post by JustinR on 2010-03-31
Hi,

I was interested in this because I like running emulators like No$gba - which I've never tried on Linux before.

Okay - you said you installed wine - which is good, next, download the windows version of no$gba here:

http://nocash.emubase.de/no$gba-w.zip

After you have downloaded it, right click it > Extract Here or you could right click it > Archive Manager and extract it anywhere you want.

Now, find the folder where you extracted the program, double click the NO$GBA.exe, if it doesn't do anything right click it and click "Open with Wine Windows Program Loader". The program should load up a "Select Game ROM" dialogue, find your game file - either a gameboy file or Nintendo DS, it supports both. 

The games run nice but I've only had Nintendo DS games work so far, you can try a gameboy file though.

Goodluck!
