### Post by 3Miro on 2010-09-10
Wine is free open source software, which means that you can use it, however, you like, with virtually no restrictions (look up GPL for details, in general, simply for using wine, there are no restrictions). As for windows key, wine doesn't care at all.

In itself, wine doesn't do much. You only use wine to run windows programs. The individual programs that you use with wine will have to be legal (just like in windows).

Note: if legality is a big issue for you, be careful not to mistake wine with VirtualBox. VirtualBox requires you to have a legal copy of windows (with CD key or whatever).

### Post by Mark Phelps on 2010-09-11
Wine is not a Linux version of MS Windows; instead, it's a collection of custom-developed DLLs that replace MS Windows system calls with Linux system calls. This allows you to run SOME MS Windows apps.

So, it's not a copy of an MS Windows OS, it's basically a minimal version of an emulator.

Thus, there is no MS Windows license you have to deal with.

But the apps are a different issue. They are installed, not emulated, so you will have to deal with all the same licensing issues as if you installed them natively in MS Windows.
