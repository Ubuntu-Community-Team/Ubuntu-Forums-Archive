### Post by Het Irv on 2008-06-10
Most of the differences are between the versions are software related.  Ubuntu uses Gnome, Kubuntu uses KDE, Xubuntu uses XCFE. Ubuntu and Kubuntu are fairly similar, while Xubuntu is slightly harder to set up.  Edubuntu is the same as Ubuntu, just geared more toward education.

Ubuntu is more popular and most of the help you will find on the forums is geared towards Ubuntu and Gnome, but it is not hard to find help on Kubuntu.  Mostly the choice comes down to personal preference.
