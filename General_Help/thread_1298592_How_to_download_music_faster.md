### Post by blazemore on 2009-10-23
If you're downloading Creative Commons music using the BitTorrent protocol, you can follow this excellent guide: [http://torrentfreak.com/speed-up-your-torrents/](http://torrentfreak.com/speed-up-your-torrents/)
I recommend the Deluge BitTorrent client, as it is more powerful and configurable than Transmission, especially if you're used to uTorrent.
```
sudo apt-get remove transmission && sudo apt-get install deluge
```
