### Post by Holger_Gehrke on 2021-06-12
Not really surprising that you can't find any information on sunbird at mozilla. The project was stopped in 2010. The official calendar application by mozilla is the 'Lightning' add-on to the Thunderbird email client. I tried the [sunbird 1.0 beta 1]("http://ftp.mozilla.org/pub/calendar/sunbird/releases/1.0b1/linux-x86_64/en-US/") (the final release of the project) from mozilla's ftp and it started without any problems when I called it with './sunbird' after unpacking the archive into ~/bin and changing into the directory created by unpacking with 'cd ~/bin/sunbird'.
