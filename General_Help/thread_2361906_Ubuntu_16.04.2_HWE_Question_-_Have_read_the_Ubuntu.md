### Post by ajgreeny on 2017-05-22
From 16.04 version the hardware enablement packages that you started with by using 16.04.2 should be automatically upgraded as detailed in the info at [https://wiki.ubuntu.com/Kernel/RollingLTSEnablementStack](https://wiki.ubuntu.com/Kernel/RollingLTSEnablementStack)

I do not keep up with these HWE updates but stay with the original kernel series unless I need extra hardware support, which on my hardware is now unlikely as it's 3 years old, so I do not have any real experience of using these HWE updates, but it should all work for you seamlessly.
