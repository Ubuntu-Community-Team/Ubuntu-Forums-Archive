Hi there,
i assume you are using emerald:
go to emerald-theme-manager, then to the "Edit Themes" tab, and then to the sub-tab of "Titlebar"
you can change various colour options there, however, not the actual theme colours... If you then go to the "frame engine" sub-tab, and then "colors" you can change colour blends... however, i dont think you can fully change the colour without editing the theme yourself (as lots of the theme is images)
