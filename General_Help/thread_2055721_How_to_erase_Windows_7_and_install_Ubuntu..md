### Post by Wim Sturkenboom on 2012-09-10
Did you verify the MD5 sum of the download; you might have a corrupted download. You can find them at [http://releases.ubuntu.com/](http://releases.ubuntu.com/) ; follow the link for the version that you've downloaded, e.g. [12.04.1](http://releases.ubuntu.com/12.04.1/) and verify the MD5sum against the value in the file MD5SUMS.

How did you create the CD/USB? Just dumping the iso on it? That will not work. Or did you burn an image to CD / use a tool to create a bootable USB?
