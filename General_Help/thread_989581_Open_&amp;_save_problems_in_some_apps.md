### Post by duanedesign on 2009-01-21
Had the same Save As dialog box problem.

[http://ubuntuforums.org/showthread.php?t=984645](http://ubuntuforums.org/showthread.php?t=984645)

As the resolution to the thread points out (thankyou the_Ben). Clicking on the minimize/maximize button fixes the unresponsive window. Actually i think the problem had to do with the window not updating/redrawing. I would type in a text field, then maximize the window and the text would be there. I am speaking in paste tense because the recent update fixed the bug for me as well. That is the workaround if it ever comes back or if someone comes upon this thread with the same problem. Everyone I have encountered with this problem is on a HP G60.

Additionally some of the G60 owners have noticed a problem with Flash video, like You Tube, being choppy. Have you encountered this.?
