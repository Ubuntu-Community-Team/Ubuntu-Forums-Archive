### Post by dark_harmonics on 2008-04-03
Yea some older laptops dont have the capabilities to boot from USB. It also helps if your computer is utilizing usb2.0 right at bios. There were some that didnt support USB, then some that supported USB1.x then some that now support the really fast usb2.0. I have an ubuntu install on my USB stick (just did a default install and pointed it at the stick as the HD) and i use it as a fast booting recovery option with a full featured desktop. You wouldn't want to run an OS on it all the time, because the read/write operations might effect the device after a while.
