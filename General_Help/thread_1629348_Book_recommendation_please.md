### Post by Priswell on 2010-11-23
I liked *Ubuntu for Non-Geeks*. It shows you how to do most of the day-to-day stuff using GUI options. By the time I found this book, I had already learned quite a bit, and Ubuntu wasn't such a mystery, but I realized how good this would be for early beginners. 

Also, I suggest *The Official Ubuntu Book*, whatever the latest edition is. It's readable and understandable. A little bit more of the command line stuff, but still readable for the beginner.

I bought Ubuntu  unleashed when Ubuntu was v6.06. While not everything was included, of course many things have changed.
