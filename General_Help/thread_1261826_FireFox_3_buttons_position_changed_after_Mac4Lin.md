### Post by stumbleUpon on 2009-09-09
Thats what Mac4Lin is supposed to do...!!!

You will get a Mac OSX look if you use Mac4Lin and this includes the changing the position of the close, maximize and minimize buttons as well.

If you dont like Mac4Lin, just go to the folder from which you installed Mac4Lin (using a script called Mac4Lin_Install_v_*.*.sh). You should see an uninstallation script.

Just run that and Mac4Lin will be removed.
