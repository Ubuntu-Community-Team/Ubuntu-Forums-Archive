### Post by jpeddicord on 2007-02-03
Nautilus only supports displaying folder size in number of items, not total size. 

Baobab (Applications > Accessories > Disk Usage Analyzer) does however let you browse a directory tree by size to see the total size of a folder. It probably isn't nearly as useful as having it integrated in Nautilus would be, but it gets the job done.
