Hi James,

Kid3 should be able to accomplish this task. Conversion between ID3v2.3 and ID3v2.4 has been improved for the latest version 1.1, so I suggest to get Kid3 1.1, select the files to convert and then use the menu "Tools/Convert ID3v2.4 to ID3v2.3".
