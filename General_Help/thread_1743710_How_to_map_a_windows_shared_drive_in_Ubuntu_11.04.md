### Post by Morbius1 on 2011-04-29
Just a word to the wise. From this day forward everyone is going to have to state what desktop they are using: Unity or Classic.

If it's classic it's right where it used to be. In Unity it's:

Hover over the top task bar until it displays "File Edit View Places Help"
Select File > Connect to Server.
