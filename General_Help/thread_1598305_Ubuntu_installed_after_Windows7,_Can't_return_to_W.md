Well, according to the output there are two harddrives (sda and sdb).

```
Disk /dev/sda: 250.1 GB, 250059350016 bytes

Device Boot Start End Blocks Id System
/dev/sda1 * 1 30020 241127424 83 Linux
/dev/sda2 30020 30402 3068929 5 Extended

```

```
Disk /dev/sdb: 16.0 GB, 16022241280 bytes

Device Boot Start End Blocks Id System
/dev/sdb1 * 6 21202 15642688 7 HPFS/NTFS

```

As shown, there are 3 partitions on the first harddrive and one partition on the second one.
I suggest you try to boot up the second one and see what it contains, I'd guess it's a partition for restoring/reinstalling Windows.
If you don't feel like changing the boot order, I suppose you could mount it and look inside it in Ubuntu instead.
Type the following commands to mount sdb1.

```
sudo mkdir /mnt/hdd
sudo mount /dev/sdb1 /mnt/hdd

```
Here you can either go to /mnt/hdd using a file browser; Places -> Computer -> File System -> mnt -> hdd.

Or you could type the following command to view what's inside it using the terminal.
```
cd /mnt/hdd
ls -al
```
