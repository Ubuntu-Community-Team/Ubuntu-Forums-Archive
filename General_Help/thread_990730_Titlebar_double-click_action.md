### Post by cellulararrest on 2009-03-26
Came across this post and saw it hadn't been solved yet.

The way to change this is (assuming you're using the Emerald window decorator), you need to open up the Emerald Themer and change the settings in that. The GTK window appearance settings don't have any effect if you're using Emerald.

There is a drop down menu setting that changes this exact setting.
