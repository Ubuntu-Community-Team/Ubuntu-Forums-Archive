### Post by TheForumTroll on 2010-06-08
If in doubt if deleting something will break stuff you can always just rename it. That would make it a lot easier to fix.

Did the game come with a .deb or how is it started up? (in the Programs menu?) and did you run any scripts when you first installed it?
