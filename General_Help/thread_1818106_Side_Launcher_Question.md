### Post by Dbury5 on 2011-08-04
You can use the CompizConfig Settings Manager, select the Ubuntu Unity Plugin option and change the Hide Launcher selection to never. If you don't have Compiz you can download it from the software center. Hope this helps and was what you were looking for.
