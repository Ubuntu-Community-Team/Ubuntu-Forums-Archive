### Post by shigidab0p on 2006-12-22
Hi, I was having the exact same issue.  What fixed it for me was updating to the latest FrostWire (4.13.1 beta).  How to:
- Download: [http://www.frostwire.com/download.php?file=http://www.peercommons.com/frostwire/4.13.1/frostwire-4.13.1.4.i586.deb](http://www.frostwire.com/download.php?file=http://www.peercommons.com/frostwire/4.13.1/frostwire-4.13.1.4.i586.deb)
- Make sure FrostWire is closed
- Run the file.  It should automatically replace the old version.

Hope this has helped.  I'm a relatively new user of ubuntu as well (just over a week now), but I find it to be very good now I've got the basics.
