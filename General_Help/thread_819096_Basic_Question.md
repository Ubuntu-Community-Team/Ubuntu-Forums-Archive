### Post by RealPSL on 2008-06-07
Gutsy simply uses a version of the Linux Kernel. However there is more to Gutsy than the kernel there are many applications that make up gutsy to perform desktop and server operations. The previous versions of Ubuntu used earlier version of the linux kernel. More information about the linux kernel can be read here.
