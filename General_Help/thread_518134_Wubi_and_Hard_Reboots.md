I'd always keep an Ubuntu liveCD handy, just in case, but then again, so long as you always properly shut down your machine when using Ubuntu and Windows, no hard drive corruption should ever occur, and it isn't too likely either.

### Post by tuxcantfly on 2007-08-05
Also, another note: if Wubi's additional sensitivity to hard reboots worries you, yet you don't want to burn a CD to install Ubuntu, you might want to see my own netboot-based Ubuntu no-cd installation program, guide at [http://ubuntuforums.org/showthread.php?t=427540](http://ubuntuforums.org/showthread.php?t=427540) which simply resizes your Windows install so you can dual-boot between Ubuntu and Windows, only it doesn't use loopmounting so it isn't as susceptible to filesystem corruption from hard reboots as Wubi is.
