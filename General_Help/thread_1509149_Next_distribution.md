### Post by WorMzy on 2010-06-14
I know you've said that you're not sure if you're ready for Arch yet or not, but I recommend you to take the plunge. I've just started using it myself, and it's really not that difficult to set up, especially if you follow [this guide]("http://wiki.archlinux.org/index.php/Beginners%27_Guide#C:_Install_Input_Driver_Packages"). It only took me about four hours or so to get all the main stuff set up (alsa, X11, gdm, compiz), and most of that time was spent waiting for things to download and install (I used a barebones netinst CD image). Once you have gdm up and running, you'll hit the ground running.
