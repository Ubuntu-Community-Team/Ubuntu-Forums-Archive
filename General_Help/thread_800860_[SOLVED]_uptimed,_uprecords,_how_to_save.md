I use uptimed daemon to track my largest uptimes by typing uprecords in console. Does anyone know where these records are saved and how can I paste them to newly installed OS?

### Post by pedro_orange on 2008-05-20
```
uptimed > uptime.log
```

I'm not sure on the options you can give uptimed; but I'm sure you can pipe the output into a text file or into some sort of blob.
Read the doc I reckon.
