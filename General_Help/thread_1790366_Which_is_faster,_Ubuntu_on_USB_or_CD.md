### Post by nzjethro on 2011-06-24
Yes, a Live USB is faster. The CD involves is read physically (i.e. the disk spins, the laser find data), meaning it's speed is limited by how fast the disk can spin. A USB stick, on the other hand, has no moving parts - it is the same type of Memory as RAM or a solid state drive, and thus is faster.

Live Ubuntu will run faster on a USB, and it'll install to your HDD faster from a USB too.
