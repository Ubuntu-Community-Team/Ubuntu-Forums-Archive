### Post by zombifier25 on 2012-03-13
You can format your USB and make a new swap partition. However, I strongly, STRONGLY recommend you not to do it. Your USB will expire really fast, and you'll have to buy a new one. If you want more memory, buy more RAM. It's cheaper and faster that way.

### Post by plucky on 2012-03-14

Memory = Ram
Storage = USB/IDE/Sata drives

If you want more memory,upgrade your memory modules to larger capacity modules.

If you want more storage ,buy a hard disk drive.Hard disks are quite cheap.

Use you USB stick to make copies of data that you don't want to lose and to swap data between systems.

Or install a new distro on it and experiment with it.

> "Expire"? What do you mean? Only cost $8 for 4GIG 

USB pen drives have a limited number of read/write cycles compared with a mechanical hard disk.

### Post by Mark Phelps on 2012-03-14
Also, realize that your ONLY using the USB device for SWAP; you're not extending the system memory to include that device. So, don't expect to see any real performance improvement.

As other folks have already said, if you want real performance improvement, buy more system memory.
