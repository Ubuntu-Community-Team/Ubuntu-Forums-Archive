> **cowboy7305 said:**

You'll probably have more luck with 3 (Three); they'll usually provide the E160 or E220 as the modem, both of which work out-of-the-box with Ubuntu. Also note that you should ideally be using 9.04 or 10.04, otherwise you might have a flaky connection.

### Post by RedDirt6722 on 2010-04-26
I use a Telstra Turbo prepaid USB modem, and it's working fine with Ubuntu 9.10.

I'd be wary about using another carrier's service if you're plans are to head away from the major towns.  The real world difference in coverage area between Telstra and the others is definitely noticeable.  

I hate using Telstra, but living and travelling in remote areas it's the only realistic option.
