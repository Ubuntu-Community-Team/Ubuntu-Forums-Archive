### Post by kolinab on 2006-12-27
I too use Keepassx. If you back up the file you make when starting an account, you shouldn't have any problems. I recently reinstalled Keepassx and pointed it to my old file without any problems. When you get to the login screen and want to use your backed up Keepass file, just click cancel, then when the main program opens up select 'open database' from the file menu. This should work for you as long as you remember your password. 

As a word of warning, don't export the file to any other format unless you need to - other formats can be less secure and you would need to be very careful in deleting them securely as not to leave a nasty trail of information behind you.

Let me know if this answers your question, I'm pretty new to Keepassx but think I've got that bit figured out.
