### Post by carlc on 2005-07-15
Are there any applications that can reduce XMMS (or similar media player) to the notification area, remove XMMS from the window list, and provide an option to play songs and directories by right clicking the icon in the notification area? I am currently using the status docklet plugin. However, XMMS remains in the window list and the status docklet plugin does not provide a option to play songs or directories.

---

### Post by carlc on 2005-07-17
I came up with somewhat of a solution here. First, I deleted the window list and added the window selector to the main panel. I have never used the window selector before but I like how it works. Instead of having running program taking up space horizontally across the panel,  you simply click the window selector icon and a pop-up menu allows you to select between running programs. I know most people would prefer the ease of clicking on the window list, but I prefer less clutter. I then added gxmms to the panel so that I can control xmms. Now the only feature that I am missing is the ability to play files and folders without having to bring xmms back up.

### Post by sizzam on 2005-10-30
Another solution is to install the xmms-status-plugin

```
sudo apt-get install xmms-status-plugin
```

After you install and enable the plugin, you will have a tray icon for XMMS that you can click on to remove the main window (and remove it from the window list).
