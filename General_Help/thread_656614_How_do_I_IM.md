### Post by ~LoKe on 2008-01-02
In the terminal, type this:
```
sudo apt-get install pidgin
```
After it's installed, launch it (alt+f2 then type pidgin, or look for it in applications -> Internet).

You'll need an address, so go to hotmail.com and make an e-mail account, or you can use a .net passport to use your current e-mail address as your msn account.  Once Pidgin is started it'll ask you for this information.

If you are using a defualt installation of Ubuntu, you will already have Pidgin. Find it at:

Applications --> Network --> Pidgin

It is very easy to use. the 1st time you start it up it will ask you which protocol you want to use. Hit the MSN icon, type in your e-mail and password like you would on Windows and you'll be chatting with all your buddies.
