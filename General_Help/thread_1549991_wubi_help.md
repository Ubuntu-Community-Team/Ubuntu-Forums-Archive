### Post by schwabdale on 2010-08-10
I don't think you can migrate your settings from a WUBI install to a new install to a partition without doing some sort of imaging. 

The reason I don't think this will work is because when you install Ubuntu to a partition, its not associated with Windows whatsoever. 
Wubi is installed inside of windows. 

When you install it to a different partition, you may be able to 
boot into your new install and access the WUBI folder on your windows partition.
