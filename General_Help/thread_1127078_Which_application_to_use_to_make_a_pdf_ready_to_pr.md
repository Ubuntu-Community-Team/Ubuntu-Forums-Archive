### Post by mcduck on 2009-04-16
For any serious desktop publishing project you definitely don't want to use OpenOffice or any graphics editors, but a proper desktop publishing application instead..

I recommend giving Scribus a try. [http://www.scribus.net/]("http://www.scribus.net/")
