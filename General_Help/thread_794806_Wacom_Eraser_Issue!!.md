### Post by rutgerw on 2008-08-07
Although the eraser should be selected automatically, this doesn't necessarily have to be the case. When you use the eraser end of the pen you can select every tool you want. Thus you can also use the eraser-end to e.g. draw a rectangle.

Make sure you selected the eraser when you flip the pen. Also make sure the background-color selected is correct. If your background-color is in black, it will result in a black line being drawn.

### Post by fragos on 2008-08-07
When you flip the stylus you are still using the function assigned to the stylus side. To eraese with the eraser end you must select the eraser icon with the eraser end of the stylus. Now you can use the stlus like a regular pencil with eraser. The eraser end is realy an alternate stylus which can be assigned to any function. You'll also find that your mouse cursor can be assigned to yet another function.
