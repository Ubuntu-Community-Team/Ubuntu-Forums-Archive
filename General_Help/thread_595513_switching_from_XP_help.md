### Post by jef396 on 2007-10-28
The Ubuntu CD includes GParted which can clone harddrives, but I'm not sure if it can clone them to USB drives.  [Clonezilla]("http://clonezilla.sourceforge.net/") is another piece of software (with a liveCD) that can clone harddrives and I'm pretty sure it will clone to a USB drive.

As for a dual-boot guide, there's one located here: [https://help.ubuntu.com/community/WindowsDualBoot](https://help.ubuntu.com/community/WindowsDualBoot)
