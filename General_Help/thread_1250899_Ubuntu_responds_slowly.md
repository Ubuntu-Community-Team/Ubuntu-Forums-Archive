### Post by P4man on 2009-08-27
application launch speed is not dependant on how much ram you have (assuming you have enough to avoid swapping) and cpu speed is probably less important than disk performance. As illustration, see what an SSD can do:

[http://www.youtube.com/watch?v=bqefSHEx7kE](http://www.youtube.com/watch?v=bqefSHEx7kE)

If you feel like testing further, you could probably swap the drives and see if its the OS or the machine.. Or run "bonnie" on both machines and see if there is a large difference in hdd speed.
