### Post by mr.moch on 2008-08-17
I don't know, you may be talking about a "super user".  Super users are the root account, and possibly another admins, if you configure it.  In the terminal, to run things under root, you use the command:

```
sudo
```

This gives you root privilages.  Or under main menu, under "System Tools", select "root terminal".  You put in the password and it makes the terminal "root@<pc name>".

Hope this helps.
