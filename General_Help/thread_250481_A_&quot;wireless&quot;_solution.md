### Post by peterthewolf on 2006-09-04
This advice does not apply to mobile wifi only to office/home situations where the object is to save laying cable.

Requirements: the premises must have an electrical ring main such as has been fitted to all new buildings in the UK for over 50 years.

Purchase two or more "homeplugs", I used the Solwise PL-14PE (14 mb ve4rsion) which are available in the UK from Solwise of Hull. Install as instructed in each PC and bingo an out of the box solution with no new room to room wiring.

The cost to me was £25 per PC which is cheaper than wifi dongles, the 14mb version is plenty fast enough for my requirements, but for heavier use 85mb and 200 mb versions are available.

In non UK countries simply search the internet for "homeplug" to find a version for your local mains electricity plug type.

Oh I forgot to mention that I used them via ethernet connection although they can be connected via usb in which case you will need a linux driver which can be downloaded from a firm called Devolo.
