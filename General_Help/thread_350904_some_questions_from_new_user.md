### Post by Beej on 2007-02-01
Welcome

1. Depends what you mean, there are native versions of Limewire and audacity, Pinnicle is video editing? I belive there are video edituing software available but someone else should confirm. Wine is a setr of libraries that allow a some windows apps to run in linux, but not many applications are supported. Cedega is a pay application that uses wine technology to offer linux support for windows games. Crossover office allows windows native desktop applications to run in linux.

My advice would be to search for alternative to (application name) in the forums, there is usually a good native alternative.

2. Depends on your hardware, my wireless worked out of the box, some require the windows drivers and ndiswrapper to work, search your hardware and ndiswrapper in the wiki / forum for details I don't have much knowledge of ndis wrapper.

3. Yes many, I like K3B and gnome baker, they are both in the repositories, I can't remember what is installed as default.

### Post by TrendyDark on 2007-02-01
If you're in real need of running a Windows-based software application, you could always check and see how it works under Wine ([http://www.winehq.com](http://www.winehq.com)) or you can purchase CrossOver Linux for about $40-50 ([http://www.codeweavers.com/)](http://www.codeweavers.com/)). If neither of those will support your application, there's always the option of purchasing VMWare Workstation and running a Windows Virtual Machine to use the application.

As for your wireless, it really just depends in the wireless NIC. I have two computers running Ubuntu using TrendNet cards (cheap, $20 from Newegg.com) and Ubuntu 6.06 recognizes one with no problems and the other just needed Ndiswrapper and it's driver installed.

Oh, and Adobe Acrobat has a Linux version, I believe. I'm almost sure of that, but I haven't read a .pdf file since Horay days, so I may be wrong.
