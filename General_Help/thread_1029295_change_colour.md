### Post by drs305 on 2009-01-06
> **stmartin said:**
> I already did it, but useless. Still the orange box appearing. How to edit it??

This is obviously important to you and we'd like to help.

mcduck's instructions worked for me. just to review: 

System > Preferences > Appearance. 
Select the Theme tab, select your current theme (Human Theme).
Click on the Customize button at the bottom.
Select the Colors tab.
The Selected Items section is the one that determines the border colors of the selected window (orange, blue, etc) and the color of the highlight bar in apps such as nautilus. It is probably set to "FFA443" (human orange). Rotate the pointer or manually enter another color. 
(Note that the "Window Border" tab changes how the border appears but NOT the color.)

Once you have the desired color displayed: 
Click Ok, Close, Close.
The active window should have the border color you selected.

If it doesn't and you can explain what happens please try to tell us.
