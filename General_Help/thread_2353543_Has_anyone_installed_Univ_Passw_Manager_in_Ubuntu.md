Seems from your comment on the upm sourceforge page that you have it installed. I'd expect that you will point upm to the database residing in your local dropbox folder.

I've never used upm preferring KeepassX (in part because it is in the repositories and therefore easier to install) In KeepassX you would select "*Open Database*" and navigate through your local folders which for me is an ownCloud folder (you would navigate to your Dropbox folder containing the database). Then any changes made to the local file, ie creating new entries or whatever then automatically sync to ownCloud, or in your case it would be Dropbox.
