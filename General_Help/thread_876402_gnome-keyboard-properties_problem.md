### Post by Archimedes0212 on 2008-08-01
If you are doing this through the keyboard properties option under Applications->Other  there is a language field and a variant field.  You know the language that you want, now I suggest just playing with the variant.

I had a problem where my keyboard layout got screwed up.  It was set as US for language and none for variant.  Turns out I needed my variant to be 'basic'.  

Give some of the variants a shot - maybe they'll do the trick.  hope this helps.
