### Post by Alwimo on 2011-07-11
Double click on the rar file and there should be a button that says "Extract".
 
In order to get Ubuntu to understand more file formats, you will need to download ubuntu-restricted-extras and install that.
 
And yes, a lot of computer programs that work on Windows and OS X do not work on Linux. Although there are things that can help out with some of those, like PlayOnLinux, WINE and Crossover.
