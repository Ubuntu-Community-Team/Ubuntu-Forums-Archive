### Post by bumanie on 2010-01-21
Unless you are using grub-legacy- The command find /boot/grub/stage1 will not actually do anything as grub 2 works completely differently than grub-legacy. Grub 2 doesn't contain grub 1 or 1.5 stage. Secondly it is more likely to be a filesystem error or possibly the filesystem may be completely wrecked. Assuming the partition designation is /dev/sda8, I suggest booting a live cd and doing this code in terminal > sudo e2fsck -C0 -p -f -v /dev/sda8Then reboot from hard drive and see if that has fixed the errors.

### Post by Leppie on 2010-01-21
do not worry about the warnings, they should not cause you any troubles (other than showing up when issuing the fdisk command).
to add your windows install to the grub menu, you just need to regenerate your grub.cfg:
```
sudo update-grub
```
it should add your windows install automagically ;)
