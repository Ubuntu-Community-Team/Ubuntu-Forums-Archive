### Post by jmszr on 2010-07-15
math1960,

I used the Firefox add-on FLASH-AID: [https://addons.mozilla.org/en-US/firefox/addon/161939/](https://addons.mozilla.org/en-US/firefox/addon/161939/) to install Flash and it worked quite well.
