Well, you are right, your request seems to be a complicated one.  After a quick search I found two links that you might find useful in your search for an answer:
[http://forum.notebookreview.com/showthread.php?t=182495]("http://forum.notebookreview.com/showthread.php?t=182495")
[http://ubuntuforums.org/archive/index.php/t-413292.html]("http://ubuntuforums.org/archive/index.php/t-413292.html")

Be advised, I have not tried to accomplish what you are inquiring about and I recommend any attempt be used with extreme caution.  Make sure you backup and all that good stuff before trying anything.

As an additional bit of information, you will need to eliminate MD in order to have any scenario involving the MD button be successful since it is its own partition, thus clogging up primary partition options.

Good Luck!  Be sure to post back here if you are successful, I'm sure it would benefit others.
