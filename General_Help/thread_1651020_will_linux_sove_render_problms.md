### Post by mcduck on 2010-12-22
1. Linux is definitely a stable platform, so it might help you with crashing problems. But it will not affect the quality of the rendered video, so if your rendered clips aren't smooth the problem is in your scene or Blender's output settings.

2. Definitely 64-bit, if you have the hardware to run it. 3D rendering is one of the tasks that can get a great benefit from 64-bit computing. Of course if there's a recommendation on Blender site to use 32-bit version instead then you should probably respect that (although I couldn't see any, not for the stable and not for the 2.55 Beta).

3. I recommend getting the 64-bit 2.55 Beta from the Blender site. It comes as a .tar.gz archive which you can just extract where you want and run. [http://www.blender.org/dl/http://download.blender.org/release//Blender2.55beta/blender-2.55-beta-linux-glibc27-x86_64.tar.bz2]("http://www.blender.org/dl/http://download.blender.org/release//Blender2.55beta/blender-2.55-beta-linux-glibc27-x86_64.tar.bz2")
