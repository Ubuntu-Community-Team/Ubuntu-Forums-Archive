### Post by sudodus on 2013-01-17
Welcome to the Ubuntu Forums :-)

Do you want to make a 'big own Ubuntu re-mix' to distribute to other people, or would it be OK to make an image (compressed image, from which you can make cloned copies of your installation).

If you are interested in the latter case, you can use many tools. I suggest Clonezilla (which is also linux software, that you can download from the internet).

If you install no proprietary drivers, and no personal files, for example only a 'guru' user account and do the tweaking, then a cloned copy will be portable to most PCs that run Ubuntu.
