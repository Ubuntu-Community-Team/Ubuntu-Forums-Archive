### Post by sonic6567 on 2010-10-19
SOLVED for me

In my case, I am behind a proxy.

In previous versions of Ubuntu, this worked fine with my environment proxy variable.  Now, for whatever reason, it does not.

I had to explicitly set a proxy for wget.

see
more /etc/wgetrc
for info.
