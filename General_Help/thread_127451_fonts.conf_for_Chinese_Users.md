### Post by alvinblank on 2006-02-08
I'm not sure if anybody notice this, but the default Ubuntu fonts is a mixed bag of gylph. I tried searching for a solution, installed the ttf-arphic-uming and ttf-arphic-ukai but it does nothing. I remeber from past experience this is a problem with the priority of the fonts. Did a little digging, and TA DA!

The new fonts are not in fonts.conf let alone have higher priority than the old fonts.

		<family>AR PL ZenKai Uni</family>
		<family>AR PL ShanHeiSun Uni</family>

Add this 2 _before_ any other AR PL * you see in fonts.conf. I'd prefer Zenkai to be above as it looks nicer than ShanHeiSun.

### Post by MantenaBr on 2006-10-20
Greetings!

I had the same problem here. In ubuntu, the chinese characters were shown as a mix of different font styles. After searching and reading a lot, here is the rather simple solution I reached.

It seems that this happens because the /etc/fonts/fonts.conf file sets a hierarchy for the serif, sans serif or monospaced fonts, so, if some font doesn't have that particular glyph, the system searchs for it in the next font listed.

The problem is that we can't modify it, because any changes made are automatically lost when the file updates itself. As stated in the fonts.conf file:

	DO NOT EDIT THIS FILE.
	IT WILL BE REPLACED WHEN FONTCONFIG IS UPDATED.
	LOCAL CHANGES BELONG IN 'local.conf'.

The solution, therefore, is to create a file named local.conf in /etc/fonts/ and place there the hierarchical sequence we desire for the fonts. I used this one from Gnome.org

[local.conf]("http://www.gnome.org/fonts/local.conf")

Just drop the file local.conf into your /etc/fonts/ directory.  This will make the Bitstream fonts your default fonts for all applications using fontconfig (if sans, serif, or monospace names are used, as they often are as default values in many desktops).

Only by doing this and restarting my gnome session, the chinese fonts were already looking perfect. But if it doesn't work the same for you, or you just don't like the way Bitstream looks, you can always try and modify it. Where it reads:

                <family>sans-serif</family>
                <prefer>
                        <family>Bitstream Vera Sans</family>
                        <family>Here goes the Chinese font I like</family>
                </prefer>

Hope it helps.

Cheers

### Post by Sprengmeister on 2006-11-17
Satimis, have you read Mantenas's suggestion carefully?

Your local.conf should look like this:

```
....
....
        <alias>
                <family>serif</family>
                <prefer>
                        <family>Bitstream Vera Serif</family>
                        <family>AR PL ShanHeiSun Uni</family>
                </prefer>
        </alias>
        <alias>
                <family>sans-serif</family>
                <prefer>
                        <family>Bitstream Vera Sans</family>
                        <family>AR PL ShanHeiSun Uni</family>
                </prefer>
        </alias>
        <alias>
                <family>monospace</family>
                <prefer>
                        <family>Bitstream Vera Sans Mono</family>
                        <family>AR PL ShanHeiSun Uni</family>
                </prefer>
        </alias>
....
....
```

Hope this helps!
