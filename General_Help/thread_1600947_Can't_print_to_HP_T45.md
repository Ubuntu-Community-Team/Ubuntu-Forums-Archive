### Post by Mark Phelps on 2010-10-19
Have you tried using CUPS-Gutenprint drivers?  You access CUPS by opening a browser and entering "localhost:631".

That will get you a main page.  From there, you can go to Manage Printers or Add Printer.  From there, you will be provided options to select the printer make and model and the driver.
