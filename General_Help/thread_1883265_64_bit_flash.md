### Post by killermist on 2011-11-18
This may be COMPLETELY not any kind of solution you're looking for, but it might help, so here goes.

Google Chrome ([http://www.google.com/chrome/](http://www.google.com/chrome/)) comes with Flash pre-embedded.  This means that it doesn't rely on the OS layer copy of Flash.

This doesn't really help if you're needing Flash in other browsers or if you're resistant to Chrome (as I was initially), but for 64bit Linux, it seems to be a good solution for me.
