### Post by COMECON on 2013-01-20
Finally I discovered it. If anybody is interested, it's the **Zukitwo Dark** theme. I got it at noobslab's PPA:

```
sudo add-apt-repository ppa:noobslab/themes
sudo apt-get update
sudo apt-get install zukitwo-dark
```
