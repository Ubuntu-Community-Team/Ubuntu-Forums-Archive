### Post by metalheart on 2007-12-03
If you have Hebrew Ubuntu, then all following should be in that language.

You can change default language in System -> Administration -> Language.

To enable keyboard go to System -> Preferences -> Keyboard.

To add keyboard indicator applet: right click on the top panel and select Add to panel and select Keyboard Indicator.
