### Post by walts48 on 2020-07-06
Tried Edit > Preferences > Search **Default Search Engine** and selecting it from the drop-down list?

If you want to set is as your home page go to the site. 

Then it would be Edit > Preferences > Home and select "Custom URLs..." for "Homepage and new windows" in the drop-down list then click "Use Current Page".
