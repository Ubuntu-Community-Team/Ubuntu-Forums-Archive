### Post by cottfcfan on 2011-08-15
Open a terrminal & type:
```
sudo add-apt-repository ppa:libreoffice/ppa
```
then:
```
sudo apt-get update && sudo apt-get upgrade
```
Not sure if there is any updates at the moment in natty, but its worth leaving the ppa enabled for when there is.
