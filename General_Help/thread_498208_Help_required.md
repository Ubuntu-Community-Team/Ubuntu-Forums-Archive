### Post by DJMatty on 2007-07-11
Hi

What exactly did you download? Was it an ISO file? Have you downloaded it to your windows desktop?

If it's an ISO file then you need to burn the ISO to a CD using nero or some other burning app in windows. (Don't just burn the file to a disc, you need to burn an image)

This will burn either a live cd (which you can pop in your drive and boot from to play with ubuntu without modifying anything on your pc) or the alternate install cd which will boot to a text based installer.

I found this article that goes through the steps to get the disc burnt in windows:

[http://www.psychocats.net/ubuntu/iso](http://www.psychocats.net/ubuntu/iso)

If you boot the live CD and it's up and running you can use the install icon to install it to your hard drive.
