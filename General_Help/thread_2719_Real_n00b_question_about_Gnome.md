### Post by Rancoras on 2004-10-31
If you're using Ubuntu you really don't need the "my computer" and "trash" icons.  They're provided in the panels for you.  The computer menu in the top panel, and the trash applet in the bottom panel (by default).

Check out this thread for info on gdesklets:
[http://ubuntuforums.org/showthread.php?t=1851](http://ubuntuforums.org/showthread.php?t=1851)
