### Post by 3rdalbum on 2014-03-06
Firstly, don't run Ubuntu 13.04. It is no longer supported. Use 13.10, it is the latest version. In April, you can upgrade to 14.04 which will be supported for five years.

Your laptop's technical specifications are far above the minimum requirements, so yes, generally it should run well. There is a slight possibility that some piece of your computer's hardware will cause a problem with Ubuntu or will be unsupported, but when you boot the Ubuntu CD you will have the ability to try the system before installation and see what works and what doesn't work.

Most things just work without any fuss.
