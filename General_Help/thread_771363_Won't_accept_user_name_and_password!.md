### Post by igfud on 2008-04-27
During the installer, there should have been a step where you type in your real name, user name, password, etc.  When you fill in your real name, it would have automatically filled your user name with your first real name in lowercase.  The password would have to have been set by you, typed in twice.

If you can't remember if you changed the user name from its default, or what the password might be, you might be able to find some instructions here: [https://answers.launchpad.net/ubuntu/+question/3039](https://answers.launchpad.net/ubuntu/+question/3039)

and here:

[https://answers.launchpad.net/ubuntu/+question/8906](https://answers.launchpad.net/ubuntu/+question/8906)
