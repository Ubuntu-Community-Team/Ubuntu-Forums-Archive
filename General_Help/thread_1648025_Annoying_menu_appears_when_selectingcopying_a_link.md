### Post by Elline on 2011-01-07
I don't think it's an Opera problem, because this menu appears every time you marking an URI wherever you are in gnome (don't know about KDE). So it must be some program/plug-in.
It's quite disappointing that nobody knows what is that, especially on this forum... STOP! I found it.

If you have a "cairo-dock" and "clipper" on it, go to "properties" and uncheck "enable actions".

### Post by ias on 2012-03-01
Thanks. That was very annoying, and it works. You have to go right-click on the Clipper icon, go to Configure applet, then go to the Configuration tab, then under Actions unclick the check-box Enable Actions. 

This post was hard to find. Perhaps someone knows how to make it easier to find for others.
