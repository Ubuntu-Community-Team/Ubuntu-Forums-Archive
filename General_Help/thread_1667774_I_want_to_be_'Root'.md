### Post by Smart Viking on 2011-01-15
Launch
```
gksudo nautilus &
```
in a terminal.

---

### Post by Vi3GameHkr on 2011-01-15
```
gksudo nautilus
``` will open nautilus file manager with root permissions.

Of course if you decided you wanted to use that "1980s geek terminal environment" (which by the way is so much more powerful and faster than the stupid 2010s non-geek GUI environment) then you could simply use sudo with any command and gain root permission that way.
