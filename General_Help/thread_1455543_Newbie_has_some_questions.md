### Post by glubbdrubb on 2010-04-16
I will give the best advice that was given to me when I switched to Linux. Use Linux as much as possible, doing everything you would normally do in windows, do in Linux. 

When you face a problem, google it and ask here in the forums. You will be an expert in no time.


Also, Install "ubuntu-restricted-extras" from the software center. It contains software required to play DVD's, mp3's, Microsoft fonts, and a whole bunch of other useful goodies.

You do not really need any security software. There are very few viruses for Linux. You would need to give a virus explicit permission to run in order for it to infect you.

There is an anti-virus that runs on linux, but it is used mostly for scanning Widows computers and email on email servers. It is called ClamAV.

Ubuntu has a really good firewall built in, so you don't need to worry about that.
