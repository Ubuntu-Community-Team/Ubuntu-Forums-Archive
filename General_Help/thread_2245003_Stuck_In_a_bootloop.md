### Post by oldfred on 2014-09-20
In BIOS is hard drive still shown? If not you may have major hard drive issues.
If shown then you may have not shutdown correctly and have file corruption or other issues.

Best then to see details:
       Post the link to the Create BootInfo summary report. Is part of Boot-Repair:
