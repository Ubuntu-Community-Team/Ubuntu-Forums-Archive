### Post by xXx 0wn3d xXx on 2006-04-16
[QUOTE=ibook developer kid]will it damage data?[/QUOTE]
Not if you install the boot loader. Basically it will be something like this:

Ubuntu Linux Partition Here:

----------------------------------------------------------

Mac Partition Here:

So there is a division between the os's. You can only damage data if you choose to erase your mac parition/hard drive. You may also need to resize your mac partition to make room for Ubuntu. Otherwise, partitioning is very safe. Just to be safe you may want to back up any important data to external media.
