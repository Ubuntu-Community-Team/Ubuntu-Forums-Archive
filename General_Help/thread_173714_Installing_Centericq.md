### Post by LadyDoor on 2006-05-10
Did you install it from Synaptic (it's in the repositories)? If not, you probably have to install it first...or, you can just do

```
sudo apt-get install centericq
```

Ummm, assuming you've done all that, you just open a terminal and type "centericq," (w/o the quotes & comma, of course)...at least that's how it worked when I had it installed. Then you just follow the directions in the menus that will pop up (in the terminal--it's terminal-based). As far as I could tell, though, it stopped working with AIM (if you can get AIM working with it, totally post it!) when all the nice folks at AIM hq discontinued TOC recently (I only know this because AIM stopped working for me and I googled it, I promise). *Shrug.* That's all I know about centericq. I hope it helps! Good luck!

--Door
