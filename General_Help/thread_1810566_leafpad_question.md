### Post by haresear on 2011-07-23
I think leafpad probably only handles unformated text, whereas rtf (rich text format) files have font formatting. OpenOffice or LibreOffice can probably open an rtf file. To use leafpad, save the file as an unformatted text file (e.g. txt type).
