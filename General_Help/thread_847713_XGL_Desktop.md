### Post by rainwalker on 2008-07-04
You clearly haven't messed with it much, then. In order:
The menus disappear in those colors because he has the close animation set for "burn" and in the animation's settings checked the box for "randomly colored fire".
Winow opening is set to fade.
Window min/maximizing is set to vacuum.
He zooms in with Enhanced/Zoom Desktop.
Wobbly Windows is enabled, and he peels the window back by maximizing it then dragging at the edge like he would resize it or Alt + click and drag near an edge.
The ripples around the window are from the Water plugin.
He has the 3D cube + skydome enabled with 4 desktops and 3D Windows.
Window transparency can be set based on if it's inactive or by having the mouse over it and using Alt + scroll wheel.
He's using some sort of dock application at the top, it might be kibadock.
Alt + tab application switcher.

Just mess with things, if you break something it can always be undone :)

### Post by angry_johnnie on 2008-07-04
You will need to install compiz-plugins-extra.

Either by looking for it in synaptic (system > administration > synaptic), or by entering this in a terminal (applications > accessories > terminal):

```
sudo apt-get install compiz-plugins-extra
```

That's how you get all those extra effects, which can be configured from Advanced Desktop Effects.

Docks are separate things, not included with compiz fusion.  Search the forums for info on docks.  There's plenty to choose from.
