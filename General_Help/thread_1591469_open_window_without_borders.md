### Post by stinkeye on 2010-10-09
If your using compiz you can go to
ccsm > effects > window decoration
and change the Decoration Windows box.

eg. to make Gnome-terminal not have decorations the box would read
```
(any ) & !(class=Gnome-terminal)
```
