> **AxAeo said:**

If you're asking "Does dual-boot allow you to run both OS's at the same time?", the answer is NO.

But, as MelDJ indicated, you can install VirtualBox and run another OS inside Ubuntu -- but you would have to install it a second time.  And even then, when you ran it, it would be the instance inside VB, not the instance you installed via dual-boot.
