### Post by zombifier25 on 2012-05-04
I don't know what suits your requirements, I just want to add to those above me:
If you want to install Xfce on top of your Ubuntu install :
```
sudo apt-get install xfce4
```
The same for LXDE:
```
sudo apt-get install lxde-core
```
Or even KDE:
```
sudo apt-get install plasma-desktop
```
That will save you the trouble of installing K/X/Lubuntu.

(BTW, just IMHO, but I think that working on Unity is faster than other DEs, with features like the Dash, Expo, HUD,...)
