### Post by Woody1987 on 2009-04-06
When ever i log in to Ubuntu the gnome-terminal is auto started. This is kinda annoying. There is no record of it in the sessions menu. I presume it has some how been added to some config file somewhere but I don't know. Help appreciated.

---

### Post by kiridude on 2009-04-06
Set "Save changes to session" in the session preferences. Then, close all open programs and log out. whatever is open will be saved - which should be nothing. This should clear it up. Don't forget to disable after you log in again.
