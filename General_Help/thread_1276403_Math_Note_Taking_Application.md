### Post by cos85 on 2009-09-27
If you're a fast enough typist and enjoy pissing about with code-like things (with you being a Linux user, I'll guess that you do), give Latex a try. Install the texmaker package and look for some tutorials. If you invest the time to learn to use this, it should come in handy for work/projects that you do in the future.

Openoffice also has a "formula" package, but I've never really used it much, and I think it would  be slower and less useful in the long run.
