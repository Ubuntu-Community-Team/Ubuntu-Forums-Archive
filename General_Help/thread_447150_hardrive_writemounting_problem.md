### Post by taurus on 2007-05-18
/dev/hdb1 is ext3 while /dev/hdb2 is your swap!  I don't understand how /dev/hdb1 is ntfs from your output.

```

Disk /dev/hdb: 250.0 GB, 250059350016 bytes
255 heads, 63 sectors/track, 30401 cylinders
Units = cylinders of 16065 * 512 = 8225280 bytes

   Device Boot      Start         End      Blocks   Id  System
**[COLOR="Blue"]/dev/hdb1               1       27403   220114566   83  Linux[/COLOR]**
/dev/hdb2           30252       30401     1204875   82  Linux swap / Solaris

```

Try

```
sudo mkdir /media/hdb1
sudo mount -t ext3 /dev/hdb1 /media/hdb1
df -h
```
