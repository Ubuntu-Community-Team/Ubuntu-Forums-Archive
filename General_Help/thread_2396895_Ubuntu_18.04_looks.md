### Post by deadflowr on 2018-07-22
Themes
Install a program called gnome-tweaks (might show up as Tweaks in the software store)
Find a new theme somewhere, perhaps look at themes at gnome-look.org
place your themes in a .themes folders.
use gnome tweaks (Tweaks when installed) to switch to using the new theme.

the .themes folder needs to be created by you with that exact name with the leading dot.
The dot signifies it's a hidden folder, which can be seen in the Files program by either clicking up in the top right panel icon > and select show hidden files/folders.
Or by pressing the keyboard shortcut ctrl +_ H.

gnome-look.org holds a vast archive of themes and icons and what have you.
[https://www.opendesktop.org/s/Gnome]("https://www.opendesktop.org/s/Gnome")

Ubuntu also does come with a few optional themes beyond that which is in use.
So you can try those as well.

### Post by Dennis N on 2018-07-22
Besides a different theme, you might want to ditch the default beaver background (wallpaper). The background image is not part of any theme you might use. Some alternates can be found in **Settings > background** dialog. If you use your own background image it should be placed in you Pictures folder, where it will be detected in the **Settings > background** dialog. 

Also, you can change the icons by using a different icon theme, and even the cursor by choosing a different cursor theme. Have fun experimenting!

### Post by Ralph L on 2018-07-23
In addition to the color of the theme you might be interested in other aspects. For myself I like 
1. A colorful cheerful look, 
2. Window edges that are wide enough so that I can easily grab them (any side) and resize the window, 
3. Stepper arrows on the ends of the scroll bar, 
4. A scroll bar slider that has good contrast with the scroll bar,
5. A window title bar that is fairly narrow so as not to waste space and that has good contrast between the selected active window and the non-selected windows.
I chose Clearlooks-Phenix for my main theme. My settings for Xubuntu are:
1. Settings>Appearance>Style>Clearlooks-Phenix.
2. Settings>Appearance>Icons>Ubuntu-Mono-Light,
3. Settings>Window Manager>Style>Daloa
4. Settings>Panel>Appearance>Color>Red 242, Green 219, Blue 198.  (kind of an orange brownish color).

However, Clearlooks-Phenix has a bug in that, when scrolling the slider on Firefox, it often runs very quickly to the bottom. It does this when you grab the slider near either the right or left edge. I live with it, but it is very annoying. Also, some apps reject the Clearlooks-Phenix scroll bar design and use their own.

That's probably more information than you wanted to know...........
