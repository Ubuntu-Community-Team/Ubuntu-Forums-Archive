### Post by warp99 on 2009-03-08
I have a D-Link WDA-1320 PCI card that I use with the Atheros restricted drivers for my MythTV server. Setup is exactly the same as with a laptop.

When you buy the card just make sure that it's supported by checking some Linux HCLs:

[http://www.linuxcompatible.org/compatibility.html](http://www.linuxcompatible.org/compatibility.html)
