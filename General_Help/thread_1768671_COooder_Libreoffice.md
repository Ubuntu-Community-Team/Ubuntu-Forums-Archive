### Post by yoosofan on 2011-08-09
Configuring java for libreoffice is necessary and there is some java package which coooder is dependent to it. I had this problem too, but I could solve it, but I forgot the exact package. It seems that the following package is needed for coooder:
libreoffice-java-common
After installing this package, install coooder. It will work.
See the following sites for more information
[http://extensions.services.openoffice.org/project/coooder]("http://extensions.services.openoffice.org/project/coooder")
