### Post by Vox754 on 2006-12-25
Copy, cp, makes a copy. Does not delete the file in its original location.
Move, mv, moves the file.

Read the stickies, they may point you to a guide of basic commands. Avoid posting very simple questions. Search the forums, up to the right. A lot of questions have been answered years ago, and the threads have information that still applies today.
Here you go [https://help.ubuntu.com/community/BasicCommands](https://help.ubuntu.com/community/BasicCommands)
