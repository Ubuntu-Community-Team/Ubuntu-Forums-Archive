### Post by Sabot on 2007-04-02
Download these two files and install by double clicking on them.

libgnutls11_1.0.16-13.2sarge2_i386.deb
[http://security.debian.org/debian-security/pool/updates/main/g/gnutls11/libgnutls11_1.0.16-13.2sarge2_i386.deb]("http://security.debian.org/debian-security/pool/updates/main/g/gnutls11/libgnutls11_1.0.16-13.2sarge2_i386.deb")
It will have to install another package from the universe to install this .deb

gtk-gnutella_0.96.3-0_i386.deb
[http://ftp.unixdev.net/pub/debian-udev/pool/main/g/gtk-gnutella/gtk-gnutella_0.96.3-0_i386.deb]("http://ftp.unixdev.net/pub/debian-udev/pool/main/g/gtk-gnutella/gtk-gnutella_0.96.3-0_i386.deb")

These files are for a 32bit install of Ubuntu 6.10

Avoid using your right mouse button while using gtk-gnutella.  It seems to crash if you do.
