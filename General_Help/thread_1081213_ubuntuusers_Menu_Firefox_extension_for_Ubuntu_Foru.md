### Post by JaiBee on 2009-02-26
Hello, 

you perhaps know the Firefox extension [Ubuntu Forums Menu]("https://addons.mozilla.org/en-US/firefox/addon/2302").  

I made, above all for the German community [ubuntuusers.de]("http://ubuntuusers.de/"), a similar extension which also _contains ubuntuforums.org_ and  which is completely translated into English:** [ubuntuusers Menu]("https://addons.mozilla.org/en-US/firefox/addon/8388")**.

At the moment (v1.0.1) ubuntuusers Menu contains the following areas:

[LIST]
[*]*ubuntuusers.d**e*
[LIST]
[*]User Control Panel
[*]Private Messages
[*]Forum
[*]Wiki
[*]Ikhaya
[*]Planet
[*]Search
[/LIST]
 
[*]***ubuntuforums.org***
[LIST]
[*]Forum
[*]User Control Panel
[*]Private Messages & Subscriptions
[*]Search
[/LIST]
 
[*]*about 100 links to (mainly international)*
[LIST]
[*]Homepages of Ubuntu + Derivates,
[*]Wikis and Documentations,
[*]Ubuntu-Forums,
[*]Planets,
[*]Launchpad etc.,
[*]Websites about hardware, software etc.,
[*]Linux news,
[*]Online magazines,
[*]and much more...
[/LIST]
 
[/LIST]
Because you probably don't need the German links you can hide them (*Options > Sub Menus*).[INDENT][CENTER][[IMG]http://www.ubuntu-pics.de/thumb/9871/uu_menu_anim_en_big_8jxbIK.gif[/IMG]]("http://www.ubuntu-pics.de/bild/9871/uu_menu_anim_en_big_8jxbIK.gif")
[/CENTER]
[/INDENT]I hope you'll like it. If you find bugs or bad translations please post it here. Positive or negative feedback is also always welcome.
