### Post by Francewhoa on 2009-05-20
[B]Requirement:
[/B]Ubuntu 8.04.2 Desktop 32 bits version.[B]

Steps for absolute beginner:[/B]
 1. Shutdown the computer. Unplug the computer power cord. Install the video card.
 
 2. Plug the computer power cord. Power up the screen. Power up the computer. The first time you power up the computer the screen will stay blank for up to 2 minutes. That's normal. Wait.
 
 3. That's it. You should see Ubuntu on the screen.
 
 4. This step is optional. If you want to had extra features such as Desktop Effects and 3D. Navigate to the Ubuntu menu SYSTEM > ADMINISTRATION > HARDWARE DRIVERS
 
Check the ENABLE box beside the item 'NVIDIA accelerated graphics driver (latest cards)'
 
 5. Restart the computer.
 
 6. This step is optional too. If you want to had more extra features such as brightness and gamma, XVideo attributes, temperature, and OpenGL settings. Navigate to the Ubuntu menu APPLICATIONS > ADD REMOVE...

Click on the SHOW dropdown menu select the option ALL AVAILABLE APPLICATIONS.

Type in NVIDIA in the search field. Wait until the results appear below.

Check the box for the item NVIDIA X SERVER SETTINGS.
 
Click on APPLY CHANGES button.
 
 The new control panel can be found under SYSTEM > ADMINISTRATION > NVIDIA X SERVER SETTINGS.
