### Post by Brandon Williams on 2011-02-05
By default, lubuntu uses pcmanfm to manage the desktop. If you haven't changed that, then you should be able to right-click on the desktop in order to get a pop-up menu that includes an item titled "Desktop Settings". The dialog for selecting the desktop background is on the tab titled "Desktop".

If you're not allowing pcmanfm to manage your desktop (there's a checkbox on the "Desktop" tab in the dialog described above), then you'll have to use some other method. I use "display -window root filename.jpg" at startup to display the specified file as my desktop background.
