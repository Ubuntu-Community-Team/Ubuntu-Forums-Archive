### Post by ad_267 on 2008-06-11
As someone already said, the .tar.gz file you download actually contains many types of themes. You need to extract this first in the file browser. Inside you will find other .tar.gz files that contain the gtk and metacity themes. You can then drag and drop these to install the themes.

The gtk theme is what controls all the buttons and how applications look. Metacity is what controls the window borders. You should probably read this: [http://justanothertechblog.blogspot.com/2007/03/quick-guide-on-customizing-ubuntu.html](http://justanothertechblog.blogspot.com/2007/03/quick-guide-on-customizing-ubuntu.html) or some of the threads in the desktop customisation forum.
