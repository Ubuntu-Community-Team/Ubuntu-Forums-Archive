### Post by WorMzy on 2010-06-05
If you want the location bar to be there permanently, as opposed to the breadcrumb trail, do the following:

[LIST]
[*]Press Alt+F2
[*]Type ```
gconf-editor
``` and hit Run
[*]Browse to "/apps/nautilus/preferences/" and change "always_use_location_entry" to true (check the box)
[/LIST]

The change should happen immediately.
