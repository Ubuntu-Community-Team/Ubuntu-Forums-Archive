### Post by cbraxton on 2009-05-01
I just installed Ubuntu 9.04 Netbook Remix on my Asus Eee 1000. (Previously it had Ubuntu-eee 8.04 installed.) Overall things are working well, but an annoyance is that when plugging in a USB flash drive, the eject button no longer appears next to its icon like it did with Ubuntu-eee.

If Nautilus is brought up, the eject does appear along with the USB drive in Nautilus' bookmarks pane. This can be used to unmount the drive, but it's a bit inconvenient do it that way.

Anyone else having this problem? (I did some searching and didn't come up with anything.) Any fixes?

### Post by pastalavista on 2009-05-01
I never use those buttons. The drive is automatically unmounted when it is removed. If you're worried about data loss or unsaved operations, you can always right-click on the drive and select the unmount option.
