### Post by t0p on 2009-06-24
gparted is a "weapon of mass destruction" (the developers' words, not mine) which repeatedly warns you to back up data each time you click Apply.  As such, it is difficult-to-impossible to save data from a massacred partition.

There are utilities that can help - search for 'deleted' in Synaptic - but don't expect too much.
