### Post by alberto ferreira on 2009-05-29
Python suits that purpose really well.

I also consider a good and fun place to start a graphic calculator -> ti-84 with a really simple language called ti-basic that will teach him the basic principles of programming and then the fun is on super-optimising codes and finding math formulas that will probably will use boolean statements to do things in a much faster way than using if statements. Basically the platform is ideal to learn programming and have fun taking the machine to it's limits through ways you never thought before. That was my first programming language and you can pretty much learn by yourself - the syntax is very close to a spoken language - and it depends a lot on math formulas (which I consider very appealing for him) to achieve high execution speeds necessary to make a great game.
