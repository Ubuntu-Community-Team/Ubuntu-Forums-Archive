### Post by Rolfffeof6 on 2009-10-27
You can easily modify any picture or icon for use in OO as a menu icon (for example, the 'save-as' icon). There are two steps: 1) create the icon 2) import and designate the icon in OO.
 

 1) Create the icon
 
[LIST]
[*]paste the desired picture or icon 	into Gimp
[*]modify the image as desired
[*]from the Gimp menu at the top, 	click “Image”
[*]from the resulting drop-down, 	click “scale image”
[*]the height and width in pixels, 	for your image may be different. Change the larger of the two to 	“26”. Click the remaining value and Gimp will enter the 	proportionally scaled number of pixels in that box.
[*]Save the newly scaled icon as a 	jpg in a folder of your choice
[/LIST]
 

 2) Import into OO.  For example, assume you want to change the “save-as” icon in OO Writer.
 

 
[LIST]
[*]Open OO Writer
[*]select “tools”, then 	“customize”
[*]from the display of icons, select 	the “save-as” icon
[*]on the right, click “modify”, 	then “change icon”
[*]in the upper left, click “import”
[*]navigate to the appropriate 	folder, select the file previously saved, then click “open” at 	the bottom right
[*]in the panel of icons, yours 	should now be displayed... click it and select OK
[*]your newly created icon shou;ld 	now appear as the “save-as” icon
[/LIST]
