### Post by 3rdalbum on 2009-02-01
Just because some software is open-source, doesn't mean that anyone can just upload a malicious change to the software, just like that. It's not like Wikipedia. In reality, only known contributors to Ubuntu get the ability to change the mainline Ubuntu code.

If you want a copy of Ubuntu, go to [www.ubuntu.com](www.ubuntu.com) or a registered mirror.

Yes, it's possible for somebody on the street to hand you a burnt copy of Ubuntu that contains malicious software, but it's just as easy for them to hand you a burnt copy of Windows that contains malicious software. You don't need access to source code to do this.
