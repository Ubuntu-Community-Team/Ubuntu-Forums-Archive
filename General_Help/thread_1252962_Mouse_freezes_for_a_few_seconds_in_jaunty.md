### Post by LMZKJ on 2009-08-29
This may be off base, but you might want to run system monitor in the task bar and see if your CPU is spiked to 100% when this happens.  I had a similar issue on an older laptop where the system would spike and you had to wait for the task.  If you do see this happening, you can then run TOP at the command line and watch to see what process is spiking it.

It it is not a CPU spike, you might look at the driver being used for the mouse.

Hopes this helps.:)
