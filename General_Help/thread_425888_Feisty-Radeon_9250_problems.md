### Post by voided3 on 2007-04-28
That's strange, I have a 9250 in my comp and i've never had any difficulties like that. With the proprietary driver on Edgy my DVI output didn't work but VGA still did, and now on Feisty, since it uses xorg 7.2, ATI doesn't currently have a driver for it so I have to take a performance hit and  use the open source driver, but games do still work. Open your xorg.conf file in nano in the console with "sudo nano /etc/X11/xorg.conf" and look for anything that is out of place and paste it here. It also may be worth a shot to make a new cd and burn it at a slow speed; perhaps the one you burned had an error. Good luck!

### Post by aldenhg on 2007-04-28
Replace 'vesa' with 'radeon'. That will make it load the open source ATI driver which is in many ways superior to the closed source driver ATI makes. 
If radeon won't work for you either, run the command ```
sudo apt-get install xorg-driver-fglrx
``` and then do ```
aticonfig --initial
```
aticonfig might want you to be root, in which case just put a sudo in front of it.
Restart the computer and you should be good to go.

EDIT:
Wait wait wait. I failed to realized that you're running this off of the CD. Any changes that you make won't carry over to the next time you start up. If you really want to try out Ubuntu you should install it and try and use it as your primary OS for a week and see how you like it. You can always nuke the Linux partitions if you want to go back to Windows
