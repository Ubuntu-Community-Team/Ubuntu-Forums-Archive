### Post by oldfred on 2010-02-22
Your fstab in Ubuntu sda1 has the wrong UUID for swap. Just edit it to the correct one.

# swap was on /dev/sda5 during installation
UUID=64f967fa-7ee5-4a15-bc05-80d0089e65b8 none            swap    sw              0       0

there is no 64.. partition anymore, it is 
f9dae16c-5c3b-4278-9eea-c53f5311856f

You probably did not reuse swap but reformated swap and in reformating it changed the UUID.
