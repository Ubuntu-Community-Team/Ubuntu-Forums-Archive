### Post by SuperFreak on 2011-04-18
Solution to problem as posted at [http://forums.mozillazine.org/viewtopic.php?f=46&t=2170625&p=10704797#p10704797](http://forums.mozillazine.org/viewtopic.php?f=46&t=2170625&p=10704797#p10704797).
If you added them as a new calendar, you should see that calendar in the calendar list, right click it and choose Delete Calendar. If you can't see the calendar list, enter the calendar view, then go to View -> Calendar, and make sure that Calendar List is checked. It will show up in the lower left.
