### Post by mikewhatever on 2009-10-11
The first thing you should do is restore Vista's boot loader. I've no idea how to do it, but others will surely fill in. That done, just boot from the Ubuntu installation cd and delete the partition with the Partition Manager under System->Administration. Then, load Vista and format the unallocated space.
