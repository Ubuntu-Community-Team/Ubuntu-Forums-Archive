### Post by jocko on 2007-03-20
Try running your games like this:
```
DISPLAY=:0 /path/to/game/executable
```
That will run the game directly in Xorg instead of inside Xgl.
You won't get any window decorations on the game window, but I guess you'll want to run the game fullscreen anyway. Probably the best for now is to run Xgl as a separate session so that you can login to normal Xorg when you need direct rendering.
Let's hope AMD/ATI get their act together soon and start working on adding full support for features that they still consider not to be part of "[stock Xorg 6.8.2]("http://support.ati.com/ics/support/default.asp?deptID=894&task=knowledge&questionID=20868")", i.e. composite and aiglx.
