### Post by bville09 on 2010-05-12
If you've got a spare USB drive (like an empty one, or one you don't need) you can make a USB image and boot from that. There's also a bunch of programs you can use to mount the .iso inside Windows (or at least I'm sure you can, unless I was imagining it).

If you install Ubuntu or Mint through wubi (or whatever Mint calls it's equivalent), it automatically dual boots with XP. If you restart with the disc in the tray (rather impossible without a livecd or usb, though), you have an option in the installer to install Ubuntu or Mint side-by-side with Windows, and select them at startup.

They can also be installed on the whole hard disk, but this will delete the Windows installation, as well as any files you have on it, so it's important to always backup before you do, as everyone will tell you these days.

Hope this helps.

EDIT: actually, do you mean to install wubi seperatley and install ubuntu or mint from there, or do you mean to ask if ubuntu and mint can be installed on the entire hard drive through wubi?

I don't think you can install wubi by itself, and install through an .iso image, and I think it's impossible to install ubuntu or mint (or any linux distro) onto the entire hard drive from within Windows. I'd say the USB disk, or installing from wubi from the .iso image is the best option for you.

### Post by WorMzy on 2010-05-12
If you can't use the CD drive, then I'd recommend against playing with partitions. One mistake, and you'll be left with an unbootable copy of Windows, and no way of recovering it. My advice would be to stick with Wubi. For all intents and purposes it's as good as a Ubuntu install on a separate partition
