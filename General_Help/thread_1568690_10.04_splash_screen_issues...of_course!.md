### Post by davidmohammed on 2010-09-06
The code you should have typed was

```
sudo -i 

echo FRAMEBUFFER=y > /etc/initramfs-tools/conf.d/splash

update-initramfs -u
```

type the following

```
more /etc/initramfs-tools/conf.d/splash
```

this should display

FRAMEBUFFER=y

if it doesnt type

```
gksudo gedit /etc/initramfs-tools/conf.d/splash
```

edit the line to contain

FRAMEBUFFER=y

then type

```
update-initramfs -u
```
