### Post by manoriax on 2009-10-26
When I started with Ubuntu I used other applications than I'm using today. The reason for this is that I've explored some important features later. So I checked my emails with Thunderbird and after some time I switched to Evolution, because it's better integrated into Gnome. Maybe you will experience the same things.

**List of software I use:**
Word Processor - OpenOffice
E-Mail Client - Evolution
Browser - Firefox
Instant Messenger - Pidgin
Dictionary - the integrated one, but most times OpenOffice, too
Video Player - Totem (already integrated) and VLC
Music Player - Rhythmbox (after I tried Amarok and Banshee I started to like Rhythmbox ^^)

Image Editor - GIMP
Disc Writer - Brasero
 Torrent Client - Transmission
Address Book - Evolution
Download Manager - DownThemAll (Firefox addon)
 Archive Manager - The integrated one. ^^

Greetings
