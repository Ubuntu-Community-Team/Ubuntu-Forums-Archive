### Post by Mr. Picklesworth on 2008-01-28
Miro is great, but seriously needs a user interface overhaul. Right now it feels like a feed aggregator, which makes me wonder why I don't just use Liferea and use MiroGuide through my web browser.
If it had a sleek TV-like PVRish interface, with a quiet back-end doing the downloading, that would not be a problem.

Hopefully Elisa will have Miro integration at some point in its life, because that would be just the interface I'm looking for ;)

My only crit with Elisa right now is that it feels very much like the last version of Apple's Front Row. I think that wheel of options at the start is a dead giveaway, and should be replaced with something cooler. (For example: still spinning options, but with very exaggerated acceleration, motion blur, depth of field and a more oval shape).

Alas, I lack a TV tuner or a reason to own one, but I use Elisa very often, full screen in one of my workspaces. The music visualizations are excellent!
