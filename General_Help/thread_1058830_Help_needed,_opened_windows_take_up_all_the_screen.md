### Post by perlluver on 2009-02-03
> **defensa24 said:**
> Ok, so here I am surfing the web with Firefox but my close and minimize and maximize buttons have disappeared. I already tried windows decoration under Compiz configuration settings but they wont come back. Any suggestions on how to fix this?

You can try this command press Alt+F2 then run this ```
gtk-window-decorator --replace
``` if that doesn't bring it back, then go to View>Full Screen, then press the maximize button, close firefox, open it back up, and you should have the buttons back.
