### Post by sereneicyrain on 2009-11-14
I am using Ubuntu 9.10 Karmic Koala. I sort of deleted the utorrent link on the desktop after I put it as a shortcut on the panel. The shortcut stopped working as a result of me deleting the original utorrent icon executable file on the desktop. I was downloading torrents before this.

I have tried to reinstall it, and the install works sometimes, sometimes it doesn't. When I try to use it from clicking the link in wine, it doesn't work.  it doesn't do anything..

*I end the process of utorrent.exe in system monitor.

*I uninstall utorrent inside of wine.

*I retry clicking the link of utorrent inside wine. It says it is busy.


.........
I think I have solved it. I was lucky that I saved the same version of utorrent, that I was previously using, on my external harddrive. I used that version and it works. I think it is a compatibility issue. The Utorrent program "recognizes" what version you were using if you were still in the process of downloading, then you can't reinstall a new version on top of that OR there is a compatibility issue with linux and the newer versions of Utorrent.
