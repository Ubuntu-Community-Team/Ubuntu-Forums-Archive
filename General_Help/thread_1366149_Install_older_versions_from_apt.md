For some software I like to use a separate repository with the latest versions in it. The obvious risk with this is that a new version breaks something and I need to revert to an older version.

### Post by last1 on 2009-12-28
Do you mean specifically apt, or synaptic? Because in synaptic it's very easy, you select the package, then go to the menu at the top, select Package, then Force Version. Pick the version you'd rather have, that's pretty much it. 

Here's a youtube video tutorial on how to do it if you want a more visual explanation
