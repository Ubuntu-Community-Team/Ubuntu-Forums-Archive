### Post by ecr959 on 2008-02-07
Click on Bookmarks, goto File, click on Export,  then you can save the whole list as an HTML file.  I leave the filename "bookmark" but I add today's date in the filename, so I can pick out the latest backup.  You can then copy or save it to a pendrive or whatever,  just like you would any other kind of file.

I hope this helps you.
