### Post by robert shearer on 2012-06-07
Ok, looks like the global menu is the problem here so let us disable it for firefox...

It is very simple and can be re-enabled easily at any time.

go...Firefox (then run the mouse pointer to the top left of the screen to reveal the global menu)
then>Tools>Add-ons and disable the 'Global Menu bar integration' and restart Firefox.

You should now have the full menu bar showing within the current Firefox window and when you open the 'Library" window in your screenshot you will see the 'import and restore button'.

Try that and report back the result.
