### Post by themusicalduck on 2011-09-15
Try searching for libasound2 in the Software Centre and installing it from there.

Or if you'd prefer, run ```
sudo apt-get install libasound2
``` in a terminal.

edit: If that doesn't work, tell us which version of Ubuntu you are running too.
