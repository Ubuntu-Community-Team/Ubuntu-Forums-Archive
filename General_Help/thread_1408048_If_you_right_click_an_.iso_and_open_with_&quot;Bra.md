### Post by nerdtron on 2010-02-16
Opening the .iso file in Brasero will automatically burn the contents of the image file to the CD. When you open the CD after burning, you will find a lot of files in there and it is bootable since the image you used is bootable.

Burning the .iso file as a data on the CD will just make a copy of the .iso file on the CD. I can't find any use to this though.
