Install the gnome-tweak-tool, it has settings to put those items on the desktop (look for Advanced settings in the dash after you installed the gnome-tweak-tool) You don't need Ubuntu tweak(actually I haven't gotten much use of it since 10.10)

Unity3d is a Compiz plugin, so Compiz has to be supported in order for that to work. Unity 2d however doesn't support Compiz(neither is gnome shell in case you want to play with it)
