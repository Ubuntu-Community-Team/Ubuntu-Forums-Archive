### Post by DJ_Peng on 2008-07-17
I don't think so in the current version. If you deactivate the Launcher/Taskmanager you get rid of the icons for currently running programs, but you also lose any launchers you have added. You'd be stuck with just your applets, which pretty much defeats the purpose of using a dock. I haven't seen any way to configure the prefs to show launchers but not currently running applications.
