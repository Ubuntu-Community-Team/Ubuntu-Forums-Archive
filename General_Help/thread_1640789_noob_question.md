### Post by migs73 on 2010-12-08
try this;

Right click on any folder, select open with other application, select it to open with 'File Browser', then tick the checkbox for 'remember this application for all "folder" files'.

### Post by Frogs Hair on 2010-12-08
Once you have reached the home folder via the computer icon , right click each of  the folders and select open with another application . From the menu select file browser and make sure the remember this application box is checked. Check to see if the folders open from the places menu.
