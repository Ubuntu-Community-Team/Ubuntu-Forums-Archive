I am using Lucid and want to install qt4 (the Gui designer hopefully.)  When I search for qt4 in Synaptic I am not sure which package(s) I need.  I see several packages that start with qt4 but nothing that looks like the base package.  Can anybody point me in the right direction? Thanks.

---

### Post by Vaphell on 2011-11-27
there is no single package, you need to install things you want. There are these qt4-* metapackages that bundle tons of stuff and there are libqt* packages that provide the goods. Select qt4-designer, qt4-qmake, qt4-demos, qt4-dev-tools and you should be good to go.
