### Post by ugm6hr on 2009-02-08
> **terry@softreq.com said:**
> Actually this setup is for a friend of mine.  I don't use a printer with my system.

Is it true you can just plug in a USB cable, the printer is detected and your all done?

The list of printers supported "out-of-the-box" can be found by looking in the Printers menu (System->Admin->Printing).

Just pretend to add a new shared samba printer, and look through the list.  If it's there, then it should work with plug and play, or possibly require you to run through this wizard to select it manually (but still easy).

$100 colour printer will be inkjet only.  Colour lasers are surprisingly cheap now (can be found for under £100GBP, around $150US) - worth considering.  They are generally cheaper to run (especially if most of your printing is black and white).
