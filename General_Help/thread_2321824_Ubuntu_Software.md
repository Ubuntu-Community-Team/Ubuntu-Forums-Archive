### Post by grahammechanical on 2016-04-24
Ubuntu Software is a new replacement for Ubuntu Software Centre. Its software catalogue is incomplete when compared with what was in the Software Centre catalogue. I have never run Ubuntu Software from a Try Ubuntu session. You may need an internet connection to get it working fully.

I can tell you that even on an installed version of 16.04 you will not find Gambas in Ubuntu Software But you can find all sorts of Gambas stuff if you install Synaptic Package manager.
