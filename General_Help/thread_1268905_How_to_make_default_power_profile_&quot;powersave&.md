Hi,
Irregardless of terminology,  Preferences > Power Management has no feature to set what power profile (CPU clock frequency limit, if you will) to use once on battery power.

The only way to change it is to use the CPU Frequency Scaling Monitor Utility, however it can not be set to 'Powersave' automatically when on battery and 'Ondemand' or 'Performance' when plugged in AC power.
