### Post by gettinoriginal on 2009-06-26
Ok, number one:
System > Preferences > Startup Applications and be sure that Update Manager is not listed.
System > Administration > Software Sources and check the Update tab, see screenshot:
[ATTACH]119072[/ATTACH]
Number two:
Right click each icon and be sure that "lock to panel" is unticked. Now right click each icon and select "move" and put it where you want on the panel.  Once you have them all where you want them, the tick the "lock to panel" and they will stay there.
(You cannot move an icon past one that is locked.)

Welcome to Ubuntu, and have fun.
