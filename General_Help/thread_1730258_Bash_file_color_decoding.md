### Post by Vaphell on 2011-04-15
bright green is for executable files (+x in permissions)
red for archives - zip, tar.gz, ...
pinkish for video files and pics - avi, mpg, jpg...
darkcyan for audio - mp3, wav...
cyan for symlinks
