### Post by mssever on 2008-08-25
> **dphoenixa said:**
> Would it be better just to get the desktop version or server version...it seems to me it really wouldn't matter. It'll mostly be used as a "media" system.
The server version doesn't include a GUI, so if you're OK with the command line, and you don't plan on running any GUI programs, the server version will be fine. Otherwise, install the desktop version. (If you install the server version and later decide you want the full version, just do sudo tasksel and choose what you want.)
