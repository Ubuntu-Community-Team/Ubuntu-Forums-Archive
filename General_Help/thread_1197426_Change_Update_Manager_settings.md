Is it possible to set the Update Manager to ignore certain updates when they are available?

I am referring specifically to Kile 2.1, I do not want to install this with 9.04 because it doesn't work! I can continue to 'uncheck' the box, but it is a bit of a drag to do so.

### Post by mikewhatever on 2009-06-26
You can try locking the version, by selecting the desired package in Synaptic, then going to Package menu (top left) and ticking Lock version.
