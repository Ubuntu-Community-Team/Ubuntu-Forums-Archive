### Post by Coreigh on 2009-01-28
I have read that it doesn't always work in 8.xx versions of Ubuntu but have you tried the dpkg-reconfigure with both monitors plugged in?
```
sudo dpkg-reconfigure xserver-xorg
```

You may find the previous version of the working xorg.conf by looking in /etc/X11 it would be named like this: /etc/X11/xorg.conf~

Rename it like this:
```
sudo cp /etc/X11/xorg.conf~ /etc/X11/xorg.conf.001
```


You should back up you existing xorg.conf before trying the reconfig
```
sudo cp /etc/X11/xorg.conf /etc/X11/xorg.conf.BAK
```
This makes a copy of xorg.conf called xorg.conf.BAK in the original directory.

Then by renaming the previous version (if it exists) you can restart X and with luck be back where you started.

I know this got a little confusing.
-Back up what you have.
-Try the previous version if you have one.
-Try the dpkg-reconfigure.
