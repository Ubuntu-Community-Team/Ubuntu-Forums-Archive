### Post by Mark Phelps on 2012-10-29
I didn't catch which version of Kubuntu, but if it's 12.10 -- you need to know that your video chipset is NO LONGER supported in 12.10.  You will have to use the default open-source Radeon driver -- which probably will work.  But, that is the only driver available now in 12.10 for chipsets older than HD 5xxx.
