### Post by Monicker on 2008-04-30
Are you having problems installing or running it?  

I am not sure what gui package manager xubuntu uses, but you can install from a terminal session:

```
sudo apt-get install amarok
```


During the first run, there is a wizard which prompts where you media files are, so that they can be indexed.  Once they have been indexed, you can add files to your playlist via the Collection tab on the left.
