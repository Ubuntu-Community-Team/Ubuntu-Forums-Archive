### Post by kesman on 2008-03-20
What's a boot DISK? Do you mean the normal ubuntu liveCD? If that gives you errors, try the alternate install cd, it usually works when live fails. Also you could check the help for additional boot options in the liveCD boot menu (the one with "install" "check cd for defects" and other options). Alternate install cd is text-based, but very easy, and much faster than the normal install.
Pay extra attention for the partition part, since in this step you have to choose where to install ubuntu so your existing files in the hard drive won't get erased.
