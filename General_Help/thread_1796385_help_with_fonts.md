Hi. is there a particular font group or style that includes the symbol for degrees (temperature)? I've checked all the symbol selections I can find in Ubuntu 11.04, but have not found the degree symbol.

### Post by Dennis N on 2011-07-03
In character map, choose 'common'. It's unicode U+00B0

---

### Post by grahammechanical on 2011-07-03
In Libreoffice go to Insert>Special Character. In the Latin-1 set you will see what you are looking for. It has the code U+00BA (186) and that works for FreeSerif and Liberation Serif and others no doubt. The new Ubuntu fonts, both standard and light, also have it. In these fonts the U+00BA (186) looks a bit large and sits too low down, in my opinion. But try U+00BO (176). That looks just right.

Regards.
