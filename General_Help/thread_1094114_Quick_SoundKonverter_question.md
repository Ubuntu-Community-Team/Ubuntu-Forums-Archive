### Post by alex2399 on 2009-04-04
You could try burning it as .wav files, most CD-players can read this format since cd-audio is essentially wav files without headers if I remember correctly. Alternatively use a burning program to burn it onto the CD that automatically converts, this gives the least troubles, Most CD burning programs can do this.
