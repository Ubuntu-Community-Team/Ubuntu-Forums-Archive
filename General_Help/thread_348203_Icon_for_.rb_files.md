### Post by rko618 on 2007-02-10
What you can do is place the icon that you have in /usr/share/app-install/icons folder.  Next right click on any .rb file and select properties.  Finally click on the icon button next to the name of the file.  Navigate to the icon you what to use and there you go :).


Edit: For anyone else reading this I found some really cool icons that you can use for ruby files located here: [http://rox-ruby.sourceforge.net/cms.php/icons](http://rox-ruby.sourceforge.net/cms.php/icons)
