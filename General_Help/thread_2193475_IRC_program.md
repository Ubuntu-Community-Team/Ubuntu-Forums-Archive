### Post by varunendra on 2013-12-13
Xchat is my favourite and is in the repositories. Quassel is another popular one for those who are fond of glossy GUI (can be a bit heavy on weak systems though, xchat is very light).

If using gnome or Unity, you can install "xchat-indicator" to integrate it with the indicator icon in the system tray.
