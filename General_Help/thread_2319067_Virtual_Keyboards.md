### Post by sudodus on 2016-03-31
***Onboard***, install with

```
sudo apt-get install onboard
```

Type the following command (on the onboard keyboard) and see how it changes :-)

Examples:
```
setxkbmap fr

setxkbmap se

setxkbmap us
```

### Post by grahammechanical on 2016-03-31
System Settings>Universal Access>Typing tab there is a slider to turn the On Screen keyboard off & on. I also think that we can turn the On Screen keyboard on or off at the login screen but I cannot be sure. And yes, Onboard will change layout according to the installed keyboard layout that we select. We can add layouts in System Settings>Text Entry.
