### Post by PatrickMay16 on 2005-09-30
The hard disk you want to access... Is it a hard disk/partition you can access from windows? I guess you're duel booting Ubuntu with windows, and you want to access the data on your windows drive/partition from Ubuntu. 
If that's the case, this should help.

[https://wiki.ubuntu.com/AutomaticallyMountMSWindowsPartitions/](https://wiki.ubuntu.com/AutomaticallyMountMSWindowsPartitions/)

Otherwise, if Ubuntu is now the only operating system on this computer, you don't need to directly access your hard disks. You store your personal data and stuff in the home folder. Click the "places" menu at the top of the screen and select "home folder".

I'm new myself, but I think this might help you. I hope it does.
Good luck!
