### Post by joshgoes on 2010-06-05
Using Rhythmbox 0.12.8 in Unbuntu 10.04, if you want to disable the pop up song notification every time it starts a new song, go to:

Edit -> Plugins

and un-check the "Status Icon" plugin.

---

### Post by ghostcoil on 2010-06-10
Another option is :

Edit -> Plugins
Select Status Icon, click the 'configure' button
Choose the "never shown" option for notifications

Then you can keep the status icon and no pop-up :)
