### Post by encompass on 2006-02-13
For quickbooks, you may be able to run that with wine... and you could even try GNU cash.  I can't think of any inventory management software for Ubuntu, but there are many third part applications.  Perhaps Banana POS.  For you ubuntu server setup... goodness... there are lots of websites that can get you server up and running... took me about 30 minutes to setup a ssh,web,php server...and I love it.
Another company option is doing remote secure backups of data with Rsync.  The can use cygwin on there computers and back there systems to your server incase the have problems.  Rsync is a great tool for that.  That is what I do for extra cash... I run a backup server for companies.
Hope that gives you some ideas.
