### Post by GameX2 on 2013-04-26
When you press Delete, it always goes in the Trash, and you can always recover it.  You can also press Shift-Delete to delete the file permanently (Excluding recovery tools).
Be aware that if you simply press Delete, and that for some reason, the file can't be put in the Trash, you'll get a warning message.

### Post by grahammechanical on 2013-04-26
If you try to delete/move to rubbish bin a file on another partition you will get a warning that the file will be fully deleted and not just moved to the rubbish bin from where it can be restored.

Try opening the rubbish bin. You will see that it is just another folder. Actually, in standard Ubuntu the word 'delete' is not used. We 'move to rubbish bin.'
