### Post by unutbu on 2008-11-20
Whenever you want to step away from your machine,
click the icon in the upper right corner, or type Ctrl-Alt-Delete. A menu of icons appears. Pick 'Lock screen'. 

If a flatmate wants to use the machine, and has an account, then he types a key a login window will pop up. Underneath are buttons, one of which says 'Switch user'. For him to use the machine, he would press that button and then log in as himself.
