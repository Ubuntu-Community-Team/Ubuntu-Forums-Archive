### Post by Impavidus on 2014-12-16
The latest kernel for trusty is 3.13.0-43. If you upgrade the software on your computers they all should get this version. If not, try upgrading via the terminal and watch for error messages. Use these two commands:```
sudo apt-get update
sudo apt-get dist-upgrade
```
Sometimes the latest kernel doesn't work with particular hardware.
