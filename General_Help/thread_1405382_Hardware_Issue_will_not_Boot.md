### Post by dabl on 2010-02-12
You will need to learn what the motherboard "beep code" that you are hearing means.  You should be able to find it here: [http://www.gigabyte.com.tw/Support/Motherboard/Manual_Model.aspx?ProductID=2871](http://www.gigabyte.com.tw/Support/Motherboard/Manual_Model.aspx?ProductID=2871)

I would guess either (a) a memory fault, or (b) it is not finding a bootable device. If the BIOS screens show correctly on your display, then it does not seem to be a video card issue.
