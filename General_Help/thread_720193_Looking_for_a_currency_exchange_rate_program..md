### Post by ubuntu27 on 2008-03-17
I have a asnwer to your question. :) 

There is a program called [Qualculate!]("http://qalculate.sourceforge.net/") As the name implies, it is an calculator. Not just an ordinary calculator though, it coverts many mathematical propierties, and also it has current exchange that can be updated automatically.

[http://qalculate.sourceforge.net/](http://qalculate.sourceforge.net/)

For GNOME

```
sudo apt-get install qalculate-gtk
```

For KDE:

```
sudo apt-get install qalculate-kde
```

### Post by Endolith on 2008-07-09
KEuroCalc is a universal currency converter and calculator. It can convert from and to many currencies, either with a fixed conversion rate or a variable conversion rate. It directly downloads the latest variable rates through the Internet.

[http://opensource.bureau-cornavin.com/keurocalc/](http://opensource.bureau-cornavin.com/keurocalc/)


GNU units can do currency calculations, but the definitions are from April 2005.  :(  I'm looking into ways to keep this up to date, so it can be used as a base for a deskbar-applet calculator.
