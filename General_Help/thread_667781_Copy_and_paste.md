### Post by geraldm on 2008-01-15
The clipboard should be already set up and available for use.  Using it is easy.
Open up text file in an editor.
With the mouse drag across the text that you wish to copy to the clipboard.
The drag means at the start of the text click and hold down mouse button 1; then move the mouse to the end of the text to be copied.  Release mouse button 1.  The text should now be copied to the clipboard.
Create another application, such as an XTerm or xterminal.  
Then open another file with the text editor.  If you are using vi or vim, put the editor into INSERT mode.  Then press down on the mouse wheel or the middle mouse button.  The clipboard text should then appear in the edited file.

Gerald
