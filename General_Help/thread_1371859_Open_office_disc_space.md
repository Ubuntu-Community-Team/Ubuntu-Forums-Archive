### Post by lyall on 2010-01-03
You can check to see how much free spare you have by opening your file browser  
Click on Places > then select Home
at the bottom of the File Browser it should state your free space.

In Open Office go to the Tools tab and click on it
then select Options
go to the Load/Save  and click on the arrow in front of Load/Save
then click on General

by default Open Office set the Save Auto Recovery to 15 minutes
you can change to how many minutes you want or uncheck it so it will not try to do a Auto Save.  If you do this you should save the file several times while working on it.

good luck and have fun learning
