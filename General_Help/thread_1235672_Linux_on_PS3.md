### Post by mcduck on 2009-08-09
> **FrankQC said:**
> Can you put Ubuntu on a ps3? Dualboot it?

Yes you can, but if you do, keep in mind that running Linux doesn't turn it into a PC, so you still won't be able to run any windows games or software (or anything else that was made for x86 architecture). Also PS3 has fairly low memory for normal computer use and programs only really use the main core of the Cell processor so most of PS3's power isn't used and it feels more like a low-end system.

I've understood that Yellow Dog Linux has at least a bit better support for PS3's hardware. On the other hand Yellow Dog is Red Hat-based which is pretty much enough for me to stay away from it (to much bad experiences with RPM-based distributions)..
