### Post by Frogs Hair on 2011-02-19
If you have the Gnomenu installed , right click the start button and go to Properties.

From the install tabs migrate to the package and open it . Choose the button and menu you want to use and select ok to restart the menu.

If not installed , install the Gnomenu from the software center , right click the either panel and add it from the add to panel menu. you may delete the other menu when it is up and running.
