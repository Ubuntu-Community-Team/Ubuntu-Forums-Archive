### Post by ilrudie on 2008-07-18
Gentoo's main speed advantage when it was started came from an experimental gcc fork with a lot of nice optimizations.  I think most if not all optimizations have been accepted into gcc so much of Gentoo's speed advantages have diminished.  This is my understanding of it at least.  I could be wrong.  You should be fine with the generic kernel though.
