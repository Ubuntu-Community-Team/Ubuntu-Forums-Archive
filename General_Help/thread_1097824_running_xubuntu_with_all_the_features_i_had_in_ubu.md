### Post by namegame on 2009-03-16
The simple fact is that XFCE is not Gnome. Although you can emulate it, there is no way for XFCE to be set up identical to Gnome. 

Among other things, XFCE uses Thunar as it's file manager, Gnome uses Nautilus, XFCE uses XFWM as it's window manager, Gnome uses metacity. The differences are truly too numerous to list here.

### Post by snowpine on 2009-03-16
Xubuntu = Ubuntu plus the Xfce desktop environment.

If you are logging in to a Gnome and/or Compiz session, you are not using Xubuntu. Try logging out, then from the login screen, choose Sessions, then Xfce.

You can install as many desktop environments as you like, and switch between sessions each time you log in. There is no need to install Xubuntu and Ubuntu on separate partitions.
