On ubuntu 8.04 as part of the desktop effects, there is a little feature that i like to call the "virtual soda spill". This is where all the windows, as soon as you get them close to another window, dock, or the edge of the screen, the windows suddenly slam against and the object and become stuck to it, making you drag it down before it finally pops free from what ever it has attached itself to.

### Post by IllEagleAlien on 2008-05-24
I believe that is the "Snapping Windows" Feature.
To turn it off go to System/Preferences/Advanced Desktop Effects Settings
Then go to Window management
