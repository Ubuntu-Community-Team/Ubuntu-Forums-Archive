### Post by Gemnoc on 2011-11-12
Hi,

FreeCAD, hands down. Although it is not at SolidWorks level yet, and won't be for a long time, unless 50 open source developers all decide right now to contribute full time. ;)

Still, it's pretty nice for a small team project, and it's seen fast development over the past year. But why do you want to run it in Windows, when you can do it on Ubuntu??? Don't install the package available from Ubuntu's repositories though, as this version is outdated.

I helped the team set up a daily build PPA on Launchpad (and by help I mean I did some initial research and tests, but one of the developers did most of the work). There's also a second PPA which updates only whenever I think of it, I simply copy working packages from the daily builds PPA.

[https://sourceforge.net/apps/mediawiki/free-cad/index.php?title=Download#Ubuntu_PPA_packages](https://sourceforge.net/apps/mediawiki/free-cad/index.php?title=Download#Ubuntu_PPA_packages)

There's a small but active community in the FreeCAD forums, don't hesitate to join us and ask questions.
