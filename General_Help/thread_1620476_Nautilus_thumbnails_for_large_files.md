### Post by t3mujin on 2010-11-13
I'm having a recurring issue where some thumbnails aren't generated in Nautilus, it doesn't seem to be a question of a specific file type, their all jpeg's from the same camera and processed with the same software. The only pattern I notice is that the non-generated thumbnails are from larger files, usually larger than 10MB.

Any tips?

---

### Post by WorMzy on 2010-11-13
Open nautilus and click: Edit -> Preferences -> Preview, and set the "Only for files smaller than" option to something higher (max 4GB)
