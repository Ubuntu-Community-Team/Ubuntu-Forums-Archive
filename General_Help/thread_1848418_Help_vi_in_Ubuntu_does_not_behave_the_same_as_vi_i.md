### Post by WorMzy on 2011-09-22
I have no idea what terminal emulator Fedora uses, but there's plenty of alternatives out there if you aren't happy with gnome-terminal.

There's:
[LIST]
[*]rxvt-unicode
[*]sakura
[*]terminator
[*]xfce4-terminal
[*]lxterminal
[*]konsole
[*]xterm
[*]eterm
[*]roxterm
[/LIST]
and more.

As for vi not behaving like vi, I don't really understand what you mean by that, but try vim. You might be able to get vim to behave how you want it to by enabling and disabling functionality. What's more, you can then use your .vimrc config file on other systems, and it (should) work the same.
