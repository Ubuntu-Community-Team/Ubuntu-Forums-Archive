### Post by forger on 2008-10-15
ubuntu uses GNOME desktop manager
kubuntu uses KDE (3 or 4, the newest one is much better looking)
xubuntu uses XFCE (gnome-like but lighter, mainly for older computers that can't handle gnome)
gobuntu includes only free software based off of Ubuntu
edubuntu is an education edition: [http://edubuntu.org/FAQ](http://edubuntu.org/FAQ)

See here:
[http://www.ubuntu.com/products/whatisubuntu/desktopedition](http://www.ubuntu.com/products/whatisubuntu/desktopedition)
(and look at the menu on the left for other releases)
