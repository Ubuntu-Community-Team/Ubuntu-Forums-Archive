### Post by lukjad on 2009-05-21
It all depends on which repositories you enable/add. If you enable "lukjad007's super 1337 h4x0r" repositories from some shady game website you found, then whatever programs you install from that repository could have negative effects. However, if you only have the default repositories installed (if you have no clue what I am speaking about, you most likely do), then everything should be relatively safe. The only problem that could arise is that if the program you find is not well maintained or well written, then it could have bugs in it.
