### Post by creaturex on 2006-11-23
I'm not sure about the streaming, but I use VLC media player for all of my videos and I haven't encountered anything it can't play yet. Just go to your good old friend the terminal and type:
 sudo apt-get update
 sudo apt-get install vlc vlc-plugin-esd

This installs VLC and the plugin pack.

Hint: Make sure you have the universe repository before you try that. Search the forums for how to add that, you'll find tons of posts.
