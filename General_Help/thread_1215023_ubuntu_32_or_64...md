### Post by fragos on 2009-07-17
I recommend running the 32 bit version. All libraries are available in 32 bit but not 64 bit. Flash is a good example unless a 64 bit version has been created. As a user you'll notice no difference between 32 and 64 bit systems except you can't use more than 4GB. Putting that in perspective I only have 1GB and vertually never use swap space which means more memory is unlikely to improve my performance.
