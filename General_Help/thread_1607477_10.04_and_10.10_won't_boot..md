Any help would be appreciated.Is this a dual-boot installation? If not then grub doesn't show an OS choice window it just boots (or doesn't in your case!). If you hold down the shift key while it's starting you may get a grub menu - from which you might try Recovery Mode to see if that's any better. If it is dual-boot then it's a bit strange that it doesn't show the grub menu, again try holding down the shift key during start-up to see if that gets you a menu. If you can get a grub menu then adding "nomodeset" to the line that ends "quiet splash" (choose "e" to edit then ctrl-x to boot) may help.
