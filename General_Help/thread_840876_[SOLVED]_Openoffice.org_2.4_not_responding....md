### Post by chewearn on 2008-06-25
Run openoffice from terminal:
```
openoffice
```

and copy/paste the error message you see to your next post.

### Post by Jackie999 on 2008-06-26
Thank you....running from terminal made the difference. I did as you said and got this:

```
jackie@ubuntu:~$ openoffice
javaldx: Could not find a Java Runtime Environment! 
jackie@ubuntu:~$ 
```

Then openoffice opened with the same window about untitled1 needing restoring..but this time I had a button to click to start fixing..which it did..then the program opened properly.
