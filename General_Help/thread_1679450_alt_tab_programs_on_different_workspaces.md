### Post by Zorael on 2011-02-04
System Settings -> Window Behavior -> Task Switcher, List Windows dropdown menu.

Note that there's a tab for an alternative task switcher, whose hotkeys you can set in the bottom of the dialog window. So you can have Alt+Tab switch through programs on your current desktop and Meta+Tab (Win) switch through programs on all desktops, if so you wished.
