### Post by vayu on 2006-01-16
There is a menu option for screen resolution under system/preferences/screen resolution.  But that is a pull down list that has preset values.  I would guess it gets those values from your /etc/X11/xorg.conf file.  My screen is 1680x1050 and I set it in that file.
