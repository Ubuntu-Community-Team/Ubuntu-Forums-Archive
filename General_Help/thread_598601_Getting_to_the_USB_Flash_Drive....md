### Post by troyer81 on 2007-10-31
I don't know how your flash drive will be recognized by Ubuntu, but assuming that it is in the "known hardware" database, it should show up as a folder under /media.  My flash drive shows up under /media/FLASH, but I don't know if that's universal.  In fact, if Ubuntu is already booted up before I plug in the drive, the file browser loads to the directory.

If you want step-by-step instructions, you should be able to click on "Places" in the Menu Bar and go to "Computer".  If you're lucky, the drive will show up on this page.  If not, type in /media in the location bar and look for a folder that looks right.

If those don't work, I'll probably need more information to help you out.  Knowing the type of USB drive it is would help (make/model).

Good luck!
