### Post by ^_Pepe_^ on 2009-10-01
Hi!

I'm sure that some skilled people can talk about details, but swap is used only when RAM memory is full or when a resident RAM program is not often used.

There are some utilities to "persuade" the programs to stay in RAM rather in swap, since RAM is veeeery faster than a HDD (swap).
