### Post by Sutekh on 2005-12-21
The default user name is 'user' the default hostname is 'ubuntu'
 - user@ubuntu

Also for debian based systems - Ubuntu, they like to use the command *adduser*
```

sudo adduser <username>

```
Will add a new user.
