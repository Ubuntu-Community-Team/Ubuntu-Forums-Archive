### Post by janpetel on 2012-08-26
I have the same issue. It seems like the old debate issue whether to *change* settings and enforce them for all old and new users or to *add* a setting and give the user the option to change it (back) to whatever it was.

I strongly feel adding a setting should be default for all of these issues (I can come up with numerous changes in gnome/ubuntu where this is the case) instead of thinking 'Well, this might be better so probably no one will mind us changing it' and as an end user not being able to get back to the previously used setting,  which they are used to. 

So, Ubuntu developers, *if you change* any settings like these, *please add* a user option to get back to the previously used one and don't disappoint users who may have actually preferred the old setting to the new one.
 It's not too late to do so for many changes enforced recently...

Personally, when I upgrade to new versions, I do so to be able to get the latest *technical* enhancements and not particularly to be forced to use new layouts and loose old settings and functions.
