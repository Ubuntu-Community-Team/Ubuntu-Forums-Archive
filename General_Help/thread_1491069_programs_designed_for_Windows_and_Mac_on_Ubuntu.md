I believe you have to still add the codecs etc to be able to read and play dvd's and mp3's etc. Ubuntu Studio is adapted for multimedia programs. You could also install the packages on a regular installation of Ubuntu or [COLOR=Red]**sudo apt-get install ubuntu-studio**[/COLOR] in a terminal.

> I read on the net that there are so-called virtual machines that provide work of programs that are designed for the Windows operating system in Ubuntu.I would like to ask you to tell me something more about it and does the programs fails, and if you have any outage and the instability of these programs when running on Ubuntu.

There's various ways of running software designed for windows. 

Dosbox will run dos programs or games. I have yet to throw something at it that didn't run. [COLOR=Red]**sudo apt-get install dosbox**[/COLOR]
[http://www.dosbox.com/](http://www.dosbox.com/)

Wine will run certain windows software. It does not work on everything but alot better than it used to be. [http://appdb.winehq.org/](http://appdb.winehq.org/) 

> A **virtual machine** (**VM**) is a [software]("http://en.wikipedia.org/wiki/Software") implementation of a machine (i.e. a [computer]("http://en.wikipedia.org/wiki/Computer")) that executes programs like a physical machine.
Virtual machines like VirtualBox, Qemu and others allow you to run other guest operating systems in a linux (or Windows) host. I have Windows XP running in VirtualBox inside Ubuntu. You have to have a certain amount of hardware for this. Win XP runs fast in virtualbox on a p4 if I allocate 1G Memory to Ubuntu and 1G Memory to XP.
