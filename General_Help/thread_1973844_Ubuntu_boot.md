### Post by hal8000 on 2012-05-05
The boot screen in Ubuntu and Fedora is Grub2, grub2 can be themed or customised or you can install Burg.

The Grub2 splash screen in Ubuntu has a purple backgriound and white text,
possibly not as nice looking as Fedora, but much better than some other distributions.
Linux Mint for example, uses plain back screen with white text, that has not been seen since 1990.

If you want to change your background or customize grub2, have a look at this thread;
[http://ubuntuforums.org/showthread.php?t=1195275](http://ubuntuforums.org/showthread.php?t=1195275)

or community page;
[https://help.ubuntu.com/community/Grub2](https://help.ubuntu.com/community/Grub2)
