### Post by sgosnell on 2009-05-27
It's easiest if you have a CD drive.  With that, you can burn the .iso to a CD, and have a live CD.  Boot from the CD, select Install, and then select the drive to install on.  You'll need to have it connected at boot time, of course.  Make sure that at the next to last screen you click the Advanced button and select the USB drive as the place to put the bootloader.
