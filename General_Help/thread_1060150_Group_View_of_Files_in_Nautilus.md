### Post by krnekhelesh on 2009-02-04
In Nautilus you have the option to list the files in icon view, compact view and list view. This makes it easier for the user to find the file he's looking for.

However the option to list the files having the same file extension in groups under different heading in the same nautilus tab would make it even easier. Nautilus does have the option to view files in list view. However it does not group the files together and show a separation between different files formats.

This is similar to the feature present in Windows Vista and XP. 

If you think this idea should be implemented please vote for this idea.

[http://brainstorm.ubuntu.com/idea/17850/](http://brainstorm.ubuntu.com/idea/17850/)
