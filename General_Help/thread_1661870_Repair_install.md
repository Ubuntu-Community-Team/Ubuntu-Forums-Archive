### Post by stuwoody007 on 2011-01-07
is there a repair install that restores everything in ubuntu to default except the home folder?

---

### Post by cmoraes on 2011-01-07
AFAIK, there is no such option.  This is possible only if you had installed your /home directory on a separate partition.  If you had, then you could reinstall your OS to the root partition without affecting your home directory.

Another way to achieve what you want to do is to backup your entire home folder, reinstall the OS and then restore the home folder again.
