### Post by bootdoc on 2009-06-19
By binary file do mean .deb?  That is a binary.  Source would come in a tar.gz where you usually unpack the tarball, then cd to the source dir and issue ./configure then make then sudo make install.  If you used the .deb file then you could try sudo dpkg --uninstall google-earth.  If you installed from source you might try sudo make uninstall from the source dir.

if test ! -x "$UNINSTALL" ; then
