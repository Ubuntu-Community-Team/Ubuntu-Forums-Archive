### Post by banceu_sergiu_ione on 2016-08-19
It all depend of you and your desire. 

I use to have off the automatic updates and run via terminal like each 2-3 days the updates when I'm bored :)

You can run software updater any and many times you like. If there will be updates will get install if not, nothing.
You can upgrade via terminal if you want 

```
 sudo apt update 
```
```
 sudo apt upgrade 
```

Its all up to you.
