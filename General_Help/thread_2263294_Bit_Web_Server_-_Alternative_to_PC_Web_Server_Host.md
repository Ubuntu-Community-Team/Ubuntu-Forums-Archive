### Post by Xanthius on 2015-01-30
Android is not quite as safe as Linux. Yes, people make the mistake that Android runs the same kernel but it does not. It is a stripped down version of the real shell which is much lesser secure for servers in general.
Also, cellphones run on wireless networks that tend to be unreliable for servers.

But do whatever you please, if its sufficient for your needs and works it works right?

Personally I would recommend a raspberry pi that is really cheap, you're able to run a headless linux on and should be sufficient enough for small websites. And it's low on consuming energy.
