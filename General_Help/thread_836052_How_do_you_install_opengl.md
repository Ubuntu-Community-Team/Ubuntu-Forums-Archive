### Post by spiderbatdad on 2008-06-21
Yes. There is a very easy way to install software on Ubuntu. Navigate to System>>Administration>>Synaptic Package Manager. There is an archive of almost 25,000 packages. It is helpful to know the name of the package you are looking for, and there is a package "Search" feature to help sort through all those packages. Google is also useful for helping to determine a packages real name. 

In this case, I believe, the package you are looking for is called: python-opengl.

If you enter opengl into the search box, you will see numerous other, related, packages.
