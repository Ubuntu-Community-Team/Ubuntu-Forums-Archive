### Post by davidjmayo on 2007-07-04
In your file manager Right click on the file you want to open

use   Open with other application..

If it's in the list click the one you want

If not click use    custom command

then click 
browse
then file system 
then  usr  
then    bin    
and  click on what you want to open it with

check it works

go back to the original file and right click properties 
open with  
and change the default then Close.

Done!!

Hope this is clear
