### Post by 3rdalbum on 2012-01-25
If you are a new user, it is better to start with the latest normal version, which in this case is 11.10.

I don't see a reason for individuals to stick with an old version of Ubuntu, and I don't see any reason for anyone to stick with an old LTS when there is a newer one.

The TL;DR answer: Use 11.10, then 12.04. Don't stick with 10.04; you'd only need to update it a year later anyway.
