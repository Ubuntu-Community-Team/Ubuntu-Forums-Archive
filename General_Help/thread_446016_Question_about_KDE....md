### Post by strabes on 2007-05-16
You have to download KDE libraries to run kde apps in gnome. KDE apps will take longer to load in gnome because they have to load all of those libraries.

---

### Post by TheMono on 2007-05-16
As strabes said, there is a hit when starting the program, as it has to load the KDE libraries, and there is also a hit as to RAM usage, as you are carrying two environment's worth of libraries in RAM.

If you have enough RAM though, it will run exactly the same as in KDE.
