### Post by DoktorSeven on 2007-05-23
File a bug and help Ubuntu fix the problem.  What happens when you plug in the external drive?  What does the command **dmesg | tail** and **lsusb** return when you type it into a terminal when you plug in the drive?  

It is much better to approach issues like this by seeing what the specific problem is and work towards fixing it than placing blame.
