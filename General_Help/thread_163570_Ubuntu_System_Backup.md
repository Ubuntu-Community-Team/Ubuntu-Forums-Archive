### Post by PriceChild on 2006-04-21
I don't think that ubuntu will screw up quite as spectacularly as Windows, and so probably won't require a system backup.

However it's ALWAYS a good idea to backup your important work from your home directory on an external hard drive or something.

Because of the ease of synaptic and aptitude, i've never seen the need to backup the entire disk, when it's so easy to just download everything else again.

I have managed to copy an entire partition, and it all still worked, so I don't see any reason why you couldn't just copy the partition somewhere.
