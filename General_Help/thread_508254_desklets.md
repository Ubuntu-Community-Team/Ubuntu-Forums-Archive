### Post by huzzak on 2007-07-24
Just go to the Synaptic Pack Manager (Main Menu >> System >> Administration >> Synaptic Package Manager) and do a search for "gdesklets".  Mark "gdesklets" for installation and it'll automatically mark the other.  That is pretty easy.

Being from the universe isn't bad.
