### Post by plucky on 2009-03-16
Right click on desktop and select **Create Launcher** and in the command line put ```
gnome-terminal
``` Click OK.

This should put a terminal icon on your desktop.Double click the icon to open a terminal and in the terminal put ```
gnome-panel &
```
Then type exit,do not use the close X button at the top of the window.


See if that fixes the panel problem.


Good luck
