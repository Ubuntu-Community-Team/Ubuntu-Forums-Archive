### Post by dragonfly41 on 2023-04-22
Inkscape is top gun but there are some other tools whuch can help your endeavours.

There is SnapSVG.
Then BoxySVG.
Then Krita which is a self contained tool.

SVG is a nice format for creative work. The code can be edited.
One approach I use is to dynamically edit the SVG attributes by embedding PHP variables which can dynamically modify SVG attributes.
This tip I read years back in carto.net.

Here we go.  [https://svg.carto.net/book/start.php](https://svg.carto.net/book/start.php)

But try SVGSnap to see how it works.
