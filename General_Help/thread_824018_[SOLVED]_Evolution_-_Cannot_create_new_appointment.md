### Post by Welcomed Rain on 2008-06-09
I am trying to create a new appointment in evolution's calendar but when I select "new appointment", I get the following message:

Cannot create a new event

'Birthdays & Anniversaries' is a read-only calendar source. Switch to Calendar View and highlight a calendar that can accept appointments.

I'm stumped. Any ideas? I only have one calendar and I have been able to input appointments in the past. I tried reinstalling evolution without success.

---

### Post by Welcomed Rain on 2008-06-10
Solved my own problem. It seems that "Birthdays and Anniversaries" had been highlighted some how. This is not editable from Evolution. I selected "Personal" and was then able to add and edit appointments.

### Post by dcstar on 2009-08-22
> **Welcomed Rain said:**
> Solved my own problem. It seems that "Birthdays and Anniversaries" had been highlighted some how. This is not editable from Evolution. I selected "Personal" and was then able to add and edit appointments.

"Birthdays and Anniversaries" are generated from the dates you enter in your Evolution Contacts, and is basically a read-only Calendar.

Your (default) Personal and any other Calendar you create are able to have things added to them.
