### Post by Pollox on 2010-10-23
There are four versions of chromium in various levels of stability on Launchpad: [https://launchpad.net/chromium-browser](https://launchpad.net/chromium-browser)
Try a more stable version.

There are also three versions of chrome in the google repos.

I have not heard of this bug you refer to about the search bar being unusable, and I've been using chrome stable recently (and currently using beta). If you encounter it, just try one of the other versions.

You might also just try clearing your cache and loading facebook again.
