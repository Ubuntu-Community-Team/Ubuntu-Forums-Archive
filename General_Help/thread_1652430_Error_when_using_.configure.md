### Post by sisco311 on 2010-12-25
Welcome to the forums!

EDIT:
Make sure you have all the build dependencies installed:
```

sudo apt-get update
sudo apt-get build-dep gmameui
```


But, gmameui is in the repositories, so you can install it via Software Center or Synaptic or apt-get.

```
sudo apt-get install gmameui
```
