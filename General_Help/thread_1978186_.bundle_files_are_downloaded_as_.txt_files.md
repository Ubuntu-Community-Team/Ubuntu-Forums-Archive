### Post by drmrgd on 2012-05-11
It's not a problem.  It's just that the installation script looks like text.  Try going into a terminal window and executing it by running:

```
sudo sh /<pathto.bundlescript>/vmware<whatever_verison>.bundle
```

That should launch the installation script for you.
