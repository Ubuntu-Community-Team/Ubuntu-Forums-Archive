### Post by RitzBitzN on 2012-12-19
The easiest way to install Ubuntu is to install it inside Windows using Wubi ([http://www.ubuntu.com/download/desktop/windows-installer](http://www.ubuntu.com/download/desktop/windows-installer)), but you have a size limit of 30GB. Otherwise, you have to download the ISO ([http://www.ubuntu.com/download/desktop](http://www.ubuntu.com/download/desktop)), burn it to a CD (I use a free program called ImgBurn, [http://www.softpedia.com/get/CD-DVD-Tools/Data-CD-DVD-Burning/ImgBurn.shtml](http://www.softpedia.com/get/CD-DVD-Tools/Data-CD-DVD-Burning/ImgBurn.shtml)), and then reboot with it in your CD drive. Seeing as the USB drive did not work for you, at least one of these should.
