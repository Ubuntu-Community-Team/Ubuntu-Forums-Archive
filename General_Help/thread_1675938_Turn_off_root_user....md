### Post by QLee on 2011-01-26
> **Crux_Dissimilata said:**
> I was wondering if there is any way to turn off that I need to type in my password every time I come back to my computer (after a while it the screen turn black and I need to log in again)?
The most common cause of the symptom you wrote is that the screensaver is optioned to lock the screen when it is activated. You should check that from the system, preferences menu.

By the way, that has nothing to do with "root password", you use your user password.

### Post by lisati on 2011-01-26
I agree with QLee: it is likely to be your screensaver settings you need to check, on the System->Preferences menu.

The "root" user is turned off in Ubuntu by default: see [https://help.ubuntu.com/community/RootSudo](https://help.ubuntu.com/community/RootSudo) for more information.
