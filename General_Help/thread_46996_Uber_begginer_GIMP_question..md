### Post by skoal on 2005-07-06
Well, in GIMP it is really easy and has so many options to fine tune your results.  I've already attached a scaled image of your original and sharpened it a bit for you (see attached).  Here's what I did in GIMP:

1. open the file
2. right click over image > select Image > Scale Image
3. Key in 1024 (or 1200, or whatever) for the "width".
(optional) 4. Unclick the lock next to the width/height field if you want more precise "desktop" dimensions.
5. Choose "filters" from the menu > enhance > select "unsharp mask"
6. use default settings or play with them. clikc ok.
7. save file.

### Post by graigsmith on 2005-07-09
ok. it's impossible to do what you want with any graphics program.  No matter what, you can't get a better image out of that image because its a raster image.  it only has a certain amount of pixels, and no program can see any deeper than whats already there.  if you blow it up with gimp or photoshop, it will look more and more blurry.

however, I suggest trying to find a larger image on the web.  OR use INKSCAPE to trace the areas and then blow it up.  when you blow up a vector image it always scales perfectly.  The ONLY downside to doing this in inkscape is that you basically have to draw the map.  but since its only tracing it wont take as long.  if you want to trace the image, just drag the image right into inkscape.  then make a new layer. and start drawing the shapes.   then make another layer for all the symbols and stuff. and those should be easy, you just draw one of each shape, then copy and paste them all over as needed.
