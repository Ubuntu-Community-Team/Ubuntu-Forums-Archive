### Post by caljohnsmith on 2008-11-07
Try disabling all your antivirus/antimalware software before installing Ubuntu; Ubuntu has to change the Windows boot.ini file and other things that will most likely appear "malicious" to an antivirus/anti-malware program, so that is a likely cause of your problem. First go to "Add/Remove Programs" in the Windows Control Panel, remove Ubuntu, disable your antivirus/anti-malware programs, reinstall Ubuntu, enable antivirus/anti-malware software, and see if that works. Let me know how it goes.
