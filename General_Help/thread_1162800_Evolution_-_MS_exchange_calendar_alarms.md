### Post by chrisgpayne on 2009-12-03
Well, I have not found a way to modify an item if you use the Calendar functionality in the message (at the bottom for me). As stated, that does add the event to the Exchange calendar with no way to modify it. 

However, if you use the drop down in the attachments section next to "VCS/ICS calendar attachment (meeting.ics)" and then "Import to Calendar" to the Exchange calendar, you can indeed modify alarms. It seems to retain the link to the original event (if updates are sent, to be confirmed). Not ideal, but it does work for me.
