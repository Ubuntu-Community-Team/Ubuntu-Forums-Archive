**Only** if your LAN hardware supports VLANs. Do a web search for detailed instructions on setting up VLANs on Linux.

I have a Netbook that I tried to get VLANs working on but the Ethernet chip does not support VLANs, so you can set it up 100% correctly but you are at the mercy of your hardware as to whether it will work or not.
