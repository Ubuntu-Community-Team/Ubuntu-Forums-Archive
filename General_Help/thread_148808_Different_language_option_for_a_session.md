The default Ubuntu GDM theme has the option for choosing the session language; 

[http://shots.osdir.com/slideshows/original.php?release=305&slide=26](http://shots.osdir.com/slideshows/original.php?release=305&slide=26)

If you are not using the default GDM then perhaps try one which has the Language option.

You would also need to install a language pack too.  Have a search in Synaptic for **language**.  There are language-packs and language-support, I'm not sure which ones you actually need though.

I think it may be **language-pack-ro** and **language-pack-gnome-ro**
