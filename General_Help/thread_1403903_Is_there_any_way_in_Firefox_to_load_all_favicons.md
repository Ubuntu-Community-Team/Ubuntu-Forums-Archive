### Post by Vunutus on 2010-02-10
Maybe I'm just being obsessive here but I can't stand it when my bookmarks (especially on the toolbar) have the default icon.

Is there any way (whether via extension or otherwise) to load up the favicons for all bookmarks? The only extensions of the sort I've found let you manually add favicons but obviously that's a huge pain for several hundred bookmarks.

---

### Post by lovinglinux on 2010-02-10
I guess you already know about Favicon Picker. To load multiple favicons, go to the bookmark folder and select the option "Open All in tabs". This will open all sites under the same bookmark folder and will update the favicons.

### Post by x33a on 2010-02-10
You can also try checkplaces. it removes duplicate bookmarks, dead bookmarks, and also loads the favicons for the ones missing them. it works well.

[https://addons.mozilla.org/en-US/firefox/addon/10897](https://addons.mozilla.org/en-US/firefox/addon/10897)

[http://www.andyhalford.com/checkplaces/](http://www.andyhalford.com/checkplaces/)
