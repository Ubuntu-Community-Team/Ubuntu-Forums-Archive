### Post by forrestcupp on 2008-04-15
> **Officer Dibble said:**
> Are you sure this is the way a calculator should handle this? I've tried this on several calculators and the response I got for 133+15% = 152.95

Well, if you do the math by hand without a calculator, you shouldn't get 152.95.  15% = 0.15, so 133 + 0.15 = 133.15.

133 + (133 * 15%) = 152.95
That's the same as saying 133 + (15% of 133) = 152.95
The **of** is translated as multiplication.  If there is no **of** then it is talking about the percentage of 1.  So just plain 15% means 15% of 1 which is 0.15.

Another way you can do it is
133 * 1.15 = 152.95
