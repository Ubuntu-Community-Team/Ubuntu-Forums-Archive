Hello everyone,
My computer running 32 bit Maverick was doing fine in wine since I first began to use it. However recently I have been having trouble with a few games, one I've played fine since first installing Ubuntu. Starcraft 2 which worked normally is suddenly taking giant FPS tanks from 50-60 FPS at the begining of a game, to about 10-15 only about 5 minutes in. Thoughts on what is causing this? Another game which is having issues is Desktop Dungeon. Now this game requires almost no processing power which makes me wonder why it is having trouble as well.

### Post by Dorakara on 2011-06-15
Apparently this was not caused by wine soley, it only seemed that way because that was the only place i did any major gaming. Upon futhur inspection I realised my graphics card was running at about 96 degrees celsius almost all the time, causing severe fps tanking. I replaced my graphics card and now it runs fine. Solved.
