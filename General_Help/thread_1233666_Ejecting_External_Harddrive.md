### Post by prshah on 2009-08-07
> **seth1991 said:**
> When I unmount the volume from the File Browser menu, it works completely.

Unmounting the drive is enough. The "eject" button in nautilus does the same thing as unmounting the drive.

The spinning down you hear is the drive being powered off; this occurs only when either the USB cord or the external power (if there is such) is turned off.

I wouldn't call this normal, but there isn't anything more you can do in Ubuntu about this, I think it is a purely hardware issue.

Unmount = eject for usb drives in Ubuntu/Linux.
