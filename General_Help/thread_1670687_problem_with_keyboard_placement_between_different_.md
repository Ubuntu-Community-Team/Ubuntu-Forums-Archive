### Post by Mesqueeb on 2011-01-19
2 ways to solve this:

1)
add a keyboard layout in keyboard settings, and change your layout when typing other languages, I did it like that for more than 2 years using DVORAK keyboard + japanese keyboard with english layout!
You can even set the "change keyboard layout" to a hotkey on your keyboard.

2)
mess with the files in iBus, it's not as hard as you think, google a bit, and have a look how I did it for my japanese-dvorak keyboard, and now I never change layout again!!
it's explained here:
[http://ubuntuforums.org/showthread.php?t=1658844](http://ubuntuforums.org/showthread.php?t=1658844)

but I can't give you any other advice on how to edit it for korean keyboards, so I suggest to google for something like "key appointment korean ibus ubuntu .py file" or something, Good luck!

-Mesqueeb
