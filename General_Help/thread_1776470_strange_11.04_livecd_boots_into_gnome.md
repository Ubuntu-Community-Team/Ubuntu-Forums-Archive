### Post by Topsiho on 2011-06-06
The new Unity desktop needs an accelerated 3D graphics card, which your computer probably has not. The open source driver for NVIDIA has no 3D acceleration, so if you have an NVIDIA graphics adapter the computer boots into the classical desktop.

Which IMHO is really bad as Ubuntu this way frightens new users away, so bug#1 (put MS out of first position on the desktop) will not get solved this way ...

AFAIK :)
