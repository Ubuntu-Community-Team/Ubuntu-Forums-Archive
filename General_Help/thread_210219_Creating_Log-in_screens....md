### Post by Horizon on 2006-07-06
[QUOTE=jclmusic]i would like create a new log-in screen using a picture i have. how would i do this?[/QUOTE]
The best way would be to find a gdm login theme that you like the layout of and replace the background image in the tar file with the one you want. If there's some text that covers up a part of the picture you don't want it to or that you'd like bigger you can move or change it by editing the greeter.xml file.
