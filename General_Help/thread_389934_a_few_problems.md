### Post by gepatino on 2007-03-22
If you don't see the tray icon for gaim there could of the following two cases:

a. Gaim is not running. Go to Applications/Internet, and launch it.
b. If it's running (you can see the contacts list running), go to Tools/Plugins and select 'System Tray Icon'
