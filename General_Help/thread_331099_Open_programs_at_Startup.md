### Post by energiya on 2007-01-04
Hi
In System-Prefrences-Sessions-Startup Programs Tab click the Add button and write the terminal command (it doesn't allways need to point to a file). For swiftfox just put swiftfox (if you have the repository or deb version installed), for Gaim just put gaim, for Conky use conky, for terminal write gnome-terminal, and so on.
