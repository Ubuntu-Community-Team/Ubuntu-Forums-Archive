### Post by ad_267 on 2009-07-03
Go to System - Administration - Synaptic Package Manager. Search for ttf-mscorefonts-installer and install it. While you're there you might as well get the ubuntu-restricted-extras package which installs mp3 and other codecs and includes the ttf-mscorefonts-installer package.
