### Post by djeikyb on 2011-01-24
1) I don't see that Google Chrome supports this. Sorry. You can clear browsing data every time before closing Chrome, but it doesn't do it automagically.

2) Click the wrench, go to preferences, and on the basic page there is a search section where you can change your default search provider. Just click the drop down. You can also click Manage Search Engines and customise the engine name, the keyword (works just like firefox), and the search url.

### Post by Krytarik on 2011-01-24
You may try the current beta of Firefox 4, it should be fairly fast, I cannot really judge, because I have a bit old hardware setup:

- backup your profile dir ~/mozilla/firefox/*.default
- download the tar package: [http://www.mozilla.com/en-US/firefox/beta/?WT.mc_id=hp_1&WT.mc_ev=click](http://www.mozilla.com/en-US/firefox/beta/?WT.mc_id=hp_1&WT.mc_ev=click)
- put it into your home dir
- extract it
- run the "firefox" script in it

That way it is of course not fully integrated into your system, but you can easyly install/remove and try it.

There is another way to install it, through a "daily" updated PPA, but I don't recommend it, because this way you get all the "nightly" and thus maybe unstabled builds:
[http://www.ubuntugeek.com/how-to-install-firefox-4-0-beta-using-ubuntu-ppa.html](http://www.ubuntugeek.com/how-to-install-firefox-4-0-beta-using-ubuntu-ppa.html)
