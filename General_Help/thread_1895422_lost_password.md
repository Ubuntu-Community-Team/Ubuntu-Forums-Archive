### Post by galvatron408 on 2011-12-15
don't do that....

just boot in to SINGLE USER MODE. There is no password requirement. A lot of UBUNTU users don't know the difference between SINGLE USER MODE and maintenance or recovery mode.

So, if you are prompt for the root password, then you are not in single user mode.

I think this guy has it down. I can't quite remember it but it does look right. I would test but I'm now on MAC OS X.
