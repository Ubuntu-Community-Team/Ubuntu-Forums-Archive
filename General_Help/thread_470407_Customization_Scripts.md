### Post by disposable on 2007-06-10
Just wanted to share a couple of scripts that my friend bland and I use to set up Ubuntu boxes. These scripts set up resource files, upgrade and install packages, perform configuration and other duties. The first is for Edgy, which we and others still use. The second is for a Feisty LAMP box.

[http://www.dualisanoob.com/linux/dot/customedgy.txt](http://www.dualisanoob.com/linux/dot/customedgy.txt)

[http://www.dualisanoob.com/linux/dot/customfeistylamp.txt](http://www.dualisanoob.com/linux/dot/customfeistylamp.txt)

Just run them as root after a fresh install (we always 'sudo passwd root' first). Regular users can run them as well, with less to do obviously. Everything is in plain text -- nothing compiled from us -- so please feel free to review, borrow, change and otherwise improve for the sake of the community.

Enjoy!
