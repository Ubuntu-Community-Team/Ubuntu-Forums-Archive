### Post by oldtimer7777 on 2011-12-03
> **techstuffs27 said:**
> So.... will my windows stays intact with this and I can then have a dual boot ??? ubuntu and Windows ?

Yes, by selecting side-by-side installation of Ubuntu (it will shrink your windows parition and create a new additional Ubuntu partition during installation - which will take some time so be patient), you will indeed be able to keep your windows the way it is now.  The only thing that will be different is that Ubuntu will boot first, but you will always have the option in the bootloader to select Windows at any time when you turn on your computer.  Other than that, everything will remain the way it is now on your Windows system.

Read the guide completely, so you can get a feel for the entire process..

[https://debianhelp.wordpress.com/2011/09/12/to-do-list-after-installing-ubuntu-11-10-aka-oneiric-ocelot/](https://debianhelp.wordpress.com/2011/09/12/to-do-list-after-installing-ubuntu-11-10-aka-oneiric-ocelot/)
