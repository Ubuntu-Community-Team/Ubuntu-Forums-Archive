### Post by trpnblies7 on 2009-02-25
I was having this same problem today, and after a bit of browsing around, I realized that you have to install the other icon sets from Synaptic before you can actually use them. They're all called "openoffice.org-style-*" where * is the name of the particular set (Industrial, Crystal, and so on). Once I installed them, switching between sets worked fine.

Hope this helps if you're still having trouble.
