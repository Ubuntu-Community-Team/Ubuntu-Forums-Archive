### Post by Yellow Pasque on 2009-01-01
The current Mozilla/Firefox browser is a bit slow, but the 3.1 version will have faster JavaScript rendering. If you want an optimized version (i.e. compiled with optimization flags for your specific processor) of FF, check out Swiftweasel. (3.05 .deb packages will be released soon, now that SW's maintainer is back from a health-related absence)

GNOME's browser, Epiphany, now uses the WebKit/Safari backend and it's available as epiphany-webkit in the Intrepid/8.10 repo. The only version of Epiphany in the standard Hardy/8.04 repo uses the "gecko" backend, but you can get Epiphany with Webkit from a PPA or source, see: [http://theindexer.wordpress.com/2008/07/04/install-epiphanywebkit-on-hardy-heron/](http://theindexer.wordpress.com/2008/07/04/install-epiphanywebkit-on-hardy-heron/)
