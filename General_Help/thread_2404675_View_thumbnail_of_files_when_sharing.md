### Post by deadflowr on 2018-10-30
*Thread moved to **General Help***

In Files go to the Preferences settings >> And then over to Search and Preview.
You should be able to toggle file size limits and increase file sizes for thumbnails.

**Edit:** To add, also set it to show All Files and not just files on the computer

Hope it helps.

(Preferences should be available up in the Files dropdown at the Files title in the top panel.)
