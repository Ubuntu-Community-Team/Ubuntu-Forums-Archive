### Post by StuartN on 2009-12-21
> **madmax.santana said:**
> Is there a way I can manipulate or maybe convert images to vector graphics formats using GIMP?

You can do the exact same thing that Inkscape does - select an area and convert the selection to a path (vector), which can then be freely scaled without pixelation. Repeat for every other area of interest.

No, there is no way of automating the process for an entire image. Inkscape can't do that either.
