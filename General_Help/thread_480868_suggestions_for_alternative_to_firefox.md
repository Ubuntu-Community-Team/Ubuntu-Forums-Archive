### Post by Ex-Cyber on 2007-06-22
Unfortunately I am not aware of a browser that meets all of these points, but I wanted to suggest that maybe you should consider external programs for some of these features rather than using things integrated into the browser. This might give you a bit more flexibility in choosing a browser.

> ad blocking capabilities as easy and good as the adblock plus extension
[...]
good JavaScript controlI can't promise that it is "easy" (difficulty is very subjective), but I had a pretty good experience with [Privoxy](http://www.privoxy.org) (I discontinued using it because I was only using it in conjunction with Tor on a Windows system). It operates as a filtering HTTP proxy, so it will work with all browsers that support proxy servers. The rules are very powerful, so I suspect it can do pretty much anything that AdBlock can. The main disadvantage I can think of here is that you would not have any UI right there inside the browser for whitelisting a site, which would be inconvenient if you frequently enable JavaScript on new sites.

> a good download managerThere are probably too many standalone download managers to list them all here. Unfortunately I'm just getting back into Linux after a long time of being away from it, and I only know of the Windows download manager I liked to use.

> A spot for me to search Google withIf you're using GNOME, you might try the Deskbar applet (right-click on a Panel->Add to Panel->Accessories->Deskbar), though I'm not sure how well it supports browsers other than Firefox.
