### Post by LaRoza on 2007-06-22
When you install Ubuntu, you can easily install the capabilities of Kubuntu. Actually, you could have done it with Kubuntu but you are already downloading Ubuntu.

To get KDE
```

sudo aptitude install kde-core

```

To get all the apps and features of Kubuntu
```

sudo aptitude install kubuntu-desktop

```

If you like KDE, or some of its apps, this is a good choice.
