### Post by linfidel on 2008-06-29
Do you have CCSM (CompizConfig Settings Manager) - accessed from the menu system, Preferences?  That is where you want to set the number of desktops, not the desktop applet.

It's in General Options, Desktop Size tab (Horizontal Virtual Size).
