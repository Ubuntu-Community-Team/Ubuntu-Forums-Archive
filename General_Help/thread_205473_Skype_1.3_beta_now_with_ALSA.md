Skype just released Skype 1.3 beta. I tried it and it works!
No webcam and it still uses qt but basic phone-calling works fine.

Download here: [http://www.skype.com/download/skype/linux/13beta.html](http://www.skype.com/download/skype/linux/13beta.html) the Debian package and install by double clicking on it and press Install.

There is also a blog about it here: [http://share.skype.com/sites/linux/2006/06/open_beta.html](http://share.skype.com/sites/linux/2006/06/open_beta.html)
