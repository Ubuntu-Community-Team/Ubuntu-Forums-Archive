### Post by 3rdalbum on 2006-07-21
The TV tuner probably won't work, and it might take some tweaking to get the second monitor to do something different to the first :-)

You might have to install the latest ATI driver from their website if you need 3D; there's a bug in the version that's in the repositories, that might affect your card.

Otherwise, it will probably be fine. Don't forget to use a Live CD or preferably dual-boot until you're absolutely sure you don't want to go back.
