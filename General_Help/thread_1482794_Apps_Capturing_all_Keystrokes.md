### Post by gerowen on 2010-05-15
Just thought I'd add a comment.  I've had a problem with this feature before because, for example let's say I'm playing a full screen game and it freezes.  Alt+Tab doesn't work if you have no other windows open, so I'm forced to ctrl+alt+backspace or f3 to get to a command line.  Even though I know the underlying operating system is just fine because Linux is good at isolating apps and preventing them from freezing the whole system, it does me no good to do that if the overlying application is frozen and blocking my interactions with that operating system.  One way I used to get around this while playing WoW in wine is I set my WoW launcher script to run in the command line, so if WoW crashed I had another window, the terminal, that I could alt+tab to and regain control of my desktop environment.
