### Post by TigerWolf on 2006-11-23
Beryl wont work with cedega at the same time.

This will work: [http://transgaming.org/forum/viewtopic.php?t=7153&sid=a5120c20c16b7be7e744151a8de2e7f5](http://transgaming.org/forum/viewtopic.php?t=7153&sid=a5120c20c16b7be7e744151a8de2e7f5)

>  have a small script that some might find useful to help me automate turning off beryl/emeralrd and the powernow daemon and turning it back on again once im done playing games, its crude but it works

Create a text file in an editor (i call mine runcedega.sh)

Code:

sudo /etc/init.d/powernowd stop
killall beryl-manager
killall emerald
metacity --replace &
cedega
sudo /etc/init.d/powernowd start
killall metacity
beryl-manager


then run chmod +x (name of file)

You can then load cedega by typing ./(name of file) in the terminal

Used on Ubuntu (thats why I load metacity to replace emerald)
