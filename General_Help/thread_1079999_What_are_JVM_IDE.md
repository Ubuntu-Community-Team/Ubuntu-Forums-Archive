### Post by FluffyElmo on 2009-02-25
JVM stands for "Java Virtual Machine" and is what lets you run programs written in the Java programing language. Common Java programs include Limewire and Azureus.

IDE stands for "Integrated Development Environment" and is a program that lets you create and edit programs. Two common Java IDEs are Netbeans and Eclipse.

IDE can also stand for "Integrated Drive Electronics" and is an interface for connecting disk drives to your computer...it depends on the context you are using these terms in I guess.
