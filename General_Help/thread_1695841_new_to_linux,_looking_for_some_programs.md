### Post by lykeion on 2011-02-26

> what about a good music player, what would you guys suggest? also is there anything like DFX Audio Enhancer for linux? (something to boost the audio above max, my laptop has weak speakers).

If I remember right DFX Audio Enhancer was a plugin that kind of combined eq, compressor and reverb/spatializer.
**VLC Media Player** have eq, spatializer and normalizer builtin.
VLC > Tools > Effect and Filters > Audio Effects > Graphic EQ/Spatializer
VLC > Tools > Preferences > Audio > Effects > Volume Normalizer

### Post by rolandrock on 2011-02-28
FYI .tar files are bundles of files created with the tar (Tape ARchive) utility.  I'm not aware of a direct Windows equivalent.

To install themes, navigate to System, Preferences, Appearance.  Either drag'n'drop the tar file into the 'theme' window or use the 'install' button to select the tar file manually.

Good Luck
