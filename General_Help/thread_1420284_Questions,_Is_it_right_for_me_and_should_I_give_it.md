Hi.

Ubuntu come in a 64 bit flavor that would take full advantage of your computer hardware.

Linux installation does NOT work like windows. I think you'll have to really accept that some program won't be working on linux. You will certainly have alternatives for them.

As for the installation of ubuntu? It's easier than windows. For your very first install, I think you should just run wubi in windows. This is a linux installer that works on windows and will prepare a dual boot for you.

When you get use to linux and ubuntu, you'll be able to try out some slightly more complexe ways.

You can definitely give it a try anyway. Download ubuntu karmic koala (9.10) for amd64 (it works for all 64 bit hardware).

Give it a try using the live CD (When you boot on the CD, there's an option try ubuntu without installing)

You will have to install support for mp3 etc... but there's plenty of help on this forum. Ask pointers to tutorials, it's all in here.

Good luck

### Post by RedRat on 2010-03-02
Yes, Ubuntu and Linux are similar to Windows in many ways, but they are not Windows. It is different and there is a learning curve.

I would suggest that before you make that leap, that you pick up a couple of books (library or buy them) on Ubuntu and/or Linux. Do some research and reading first before you jump into Ubuntu. If you don't do this, you are going to be spending a lot of time in the Beginners Forum complaining about this and that, "What did I do wrong", etc. 

Another consideration is what it is you want to do on your computer. If it is playing games and video editing, you might want to stay with Windows. Programs like Adobe Elements and PhotoShop are pretty hard to beat in the Open Source community. If you are just surfing the web, email. Twitter, etc., you should have minimal problems in the Linux distros. 

Of course Ubuntu is a very good OS, but you might want to take a look at Linux Mint, since it is geared to new users a bit better than Ubuntu, you can get all the codecs for music and video with that OS. 

Just remember, do your research before you jump.

### Post by Aduntu on 2010-03-03
Hello, Aaron.

First, I want to put your mind at ease and tell you that Ubuntu/Linux is nothing to be afraid of.  I was like you not long ago, before I made the jump to Ubuntu.  I was very unsure of the whole process because I was so unfamiliar with the idea of any other operating system aside from Windows and Mac OS X.  But, after about six months of using Ubuntu and Ubuntu Netbook Remix, I can say that I now avoid using Windows at all costs.  I strictly run Mac OS X and Ubuntu. If it's absolutely necessary that I use Windows, I run it as a virtual machine inside of Ubuntu.

The best advice I can give you is to become comfortable with searching these forums for threads regarding issues you may have.  Chances are, someone has had the same problem you're having and there's a plethora of information available to walk you through how to fix it.

One last piece of advice -- look at this like a learning experience.  Don't come into this looking for a complete Windows replica.  Look at this as an opportunity to learn something completely different, with zero expectations.  A month from now, you'll be tickled to death that you went out on a limb to try something new.
