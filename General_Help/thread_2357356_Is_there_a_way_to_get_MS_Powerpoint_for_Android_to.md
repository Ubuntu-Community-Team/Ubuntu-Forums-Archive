### Post by dragonfly41 on 2017-04-01
If you only require to "load Powerpoint files" (presumably prepared on another device) for viewing
such as giving a mobile presentation then I would look to pandoc to convert PowerPoint files to HTML slides.

[http://pandoc.org/](http://pandoc.org/)

But if you wish to use PowerPoint to edit original PPT files then this option will not help you.

You can, however, edit the *markdown files* created by pandoc.
