### Post by blazemore on 2009-11-27
Well, "Normal" and "Extra" both use a compositing Window Manager called Compiz.
To switch between "None" (Metacity) and Compiz, you can use the command
compiz --replace
or
metacity --replace

But the difference between Normal and Extra is a matter of Compiz settings, not a different program to run.
