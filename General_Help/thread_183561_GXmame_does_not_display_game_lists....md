### Post by yamagami on 2006-06-01
Problem solved: 
there is a gxmame .35b version available. That one works just fine:

[PHP]wget http://surfnet.dl.sourceforge.net/sourceforge/gxmame/gxmame_0.35beta2-1_i386.deb
sudo dpkg -i gxmame_0.35beta2-1_i386.deb[/PHP]
