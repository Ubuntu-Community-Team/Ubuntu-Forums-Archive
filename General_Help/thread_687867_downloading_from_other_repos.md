### Post by Gerard Barberi on 2008-02-04
[Enabling extra repos]("http://www.psychocats.net/ubuntu/sources")

If there is a game you would like and it is not in the repos, go to the game's website and see if they have a Deb package available for you.  If not, try compiling it [from source.]("http://monkeyblog.org/ubuntu/installing/#installing_a_package_manually")

For RPMs, as mentioned in the above link, use a program called Alien to convert it into a Deb.
