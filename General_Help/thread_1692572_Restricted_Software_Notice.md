### Post by VOT Productions on 2011-02-21
It's that restricted software is simply software Ubuntu can't bundle due to copyright, legal problems etc.
You can go ahead and install it, as it is 100% safe for the user to install it.
(You can go to Ubuntu Software Center and install Ubuntu Restricted Software that way if you want)

You get that message because you're trying to

install software that has legal issues in some countries.

Software like that can not be bundled with the distribution

so you have to install it manually. You can safaly go ahead and

install it.

See this:

[https://help.ubuntu.com/community/RestrictedFormats](https://help.ubuntu.com/community/RestrictedFormats)
