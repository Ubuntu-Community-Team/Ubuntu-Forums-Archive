### Post by aewrfh on 2010-08-20
I use another monitor plugged into my laptop in place of the regular laptop monitor because my laptop monitor broke when I dropped it. 

When I changed monitor settings in the System>>Preferences>>Monitor Settings(or whatever it is), it became so that I can't change it back to what it was before I changed the monitor settings. I had it on mirror mode in which the screen display shows the same way regardless of it being laptop or external desktop monitor that is the monitor currently used.
Now, the panels do not show, so that means I don't have access to much of my computer anymore.

Does anybody know any tips of helps, or a complete solution to solve this concerning issue?

### Post by aewrfh on 2010-08-29
This problem is solved.

Problem: Panels are gone after messing with monitor settings and having an external monitor besides laptop's.

Solution:  Press alt + f1.  It selects the menu bar. Then use arrow keys to navigate even though you don't see anything, but it is working as if you don't see it. Open control center by pressing right 2x then down 3x on the arrow keys. Next open the monitor settings inside of control center. Change the settings to mirror monitor or whatever it is so both the laptop monitor and the external both display the same thing.
