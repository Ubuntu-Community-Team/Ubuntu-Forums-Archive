### Post by lechien73 on 2011-11-28
Hi,

Sorry, but I have a couple of terminal commands for you! If you could open a terminal window (press Ctrl+Alt+T if you don't have the icon anywhere) and type the following:

```
gconftool-2 --recursive-unset /apps/compiz-1
unity --reset

```

This should reset your launcher and menu bar.
