### Post by Giblet5 on 2009-11-09
Use the System->Administration->"USB Startup Disk Creator" to create a stick with persistent file store.

Boot from it. Work from it. It's Ubuntu on a stick!

(Note that the kde version of the tool doesn't have a broken interface.)
