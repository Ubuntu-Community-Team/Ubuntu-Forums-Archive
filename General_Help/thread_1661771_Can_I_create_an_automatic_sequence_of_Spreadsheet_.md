### Post by seawolf167 on 2011-01-07
Try going to Tools -> Options, and go to the Load/Save -> General tab.

Check the box that says "AutoSave Every" and enter a time. Then check the box that says "Always create a backup copy"

I haven't tried it (just checked out the options), but this should autosave your main document every XY minutes, and create a copy of the file at each save point.
