### Post by geirha on 2009-05-20
There is a key combo to change between keyboard layouts if you have more than one layout.

System -> Preferences -> Keyboard -> [Layouts]. Select a layout in the list, and click the Layout Options button. Under Layout Switching, you can set the key combo to switch from that layout to the next. Perhaps you've accidentally hit that key combo earlier?
