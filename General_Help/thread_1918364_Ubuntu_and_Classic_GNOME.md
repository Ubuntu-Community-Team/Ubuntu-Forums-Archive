### Post by grahammechanical on 2012-01-31
Who would be planning this? Canonical, the developers of Ubuntu? No. Do not hold your breath waiting for that to happen.

The name Unity was chosen because it was to be the unifying look of Ubuntu across a range of computer devices. Watch this and give up any thought that Canonical will reverse direction:

[http://www.ubuntu.com/tv]("http://www.ubuntu.com/tv")

Unity is at version 5 in 12.04 development branch. And as for Debian it uses Gnome as the default desktop environment as well. This is why Ubuntu uses it in the first place.

Since 11.10 Unity has been the default desktop user interface on both Ubuntu (3D) and its fall-back mode Ubuntu 2D.

So, remember what is sometimes said on this forum: Ubuntu is not a democracy.

### Post by mikewhatever on 2012-02-01
What you call Gnome Classic is really known as Gnome2, and for better or worse, that DE is no longer supported by the Gnome Foundation. Various projects strive to recreate the look and feel of the old desktop, by none, except one plan on supporting Gnome2.

Now, if you are after Gnome2, either use Debian or CentOS, or check out Mate: [https://github.com/Perberos/Mate-Desktop-Environment](https://github.com/Perberos/Mate-Desktop-Environment)

...and if you are after the look and feel, use Gnome Fallback Mode (gnome-session-fallback package from the repositories), available for 11.10 and later.
