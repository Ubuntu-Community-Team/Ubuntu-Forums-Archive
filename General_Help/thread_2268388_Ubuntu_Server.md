I don't understand.
Are you asking how to set up your new server as a replacement for your router?
Or are you asking how to set up your new server for specific network services? Which services?

Ubuntu Server *can* be easy to set up ...mine was... provided you are comfortable with the command line, you understand a teeny bit of basic networking, and you understand how to configure the services you want (ifthey need to be customized). Not a beginner task, but free and fun to try.

Ubuntu Server is very much like a kit of Legos. All the pieces are there, but unassembled for you. Assembling the finished product, which may be configured differently that others', takes patience and (sometimes) many steps.
