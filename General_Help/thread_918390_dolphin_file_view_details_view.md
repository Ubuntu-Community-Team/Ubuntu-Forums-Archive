In dolphin does anyone know how to make View->View Mode-> Details options stick?  It only applies to the current directory.  I would like dolphin to always show files as details view.

---

### Post by Denestria on 2008-09-13
Settings > Configure Dolphin
General > Default View Mode = Details.
