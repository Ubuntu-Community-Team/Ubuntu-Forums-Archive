### Post by Linux_junkie on 2010-06-16
Something for you to check.

In Gnome select the System -> Preferences -> keyboard option then under the accessibility tab check to see if slow keys has been selected.  If it has deselect it.

Keyboard might not have died its just waiting for you to press the keys for longer periods to work.
