### Post by lovinglinux on 2009-07-13
Instead of disabling the feature, you might want to install the [Secure Login](https://addons.mozilla.org/en-US/firefox/addon/4429) extension for Firefox. Then open it's preferences and disable "Submit Login Form Automatically". You will still be able to use the password remembering feature, by just clicking an icon in the toolbar, which makes it fill the login form and log in. It won't fill forms without you wanting to.

I use for a long time and works like a charm.
