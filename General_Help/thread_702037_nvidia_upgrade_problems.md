### Post by fowie on 2008-02-19
It looks from your xorg.conf file that it is using the nvidia driver because it says:
```

Section "Device"
Identifier "Failsafe Device"
Boardname "vesa"
Busid "PCI:5:0:0"
**Driver "nvidia"**
Screen 0
EndSection

```
but notice that your monitor section only has one mode,
```

Section "Monitor"
Identifier "Failsafe Monitor"
Vendorname "Plug 'n' Play"
Modelname "Plug 'n' Play"
modeline "640x480@60" 25.2 640 656 752 800 480 490 492 525 -vsync -hsync
Gamma 1.0
EndSection

```
so you could manually try adding a few higher-res modes and see if they work, or re-run xconfig and try to get it to detect your monitor.
Just a few suggestions.
