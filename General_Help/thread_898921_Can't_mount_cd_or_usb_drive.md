### Post by tommcd on 2008-08-23
Sorry if this sounds like a stupid question but since you did not say ehat you have tried, did you try going to system > administration > users and groups? Then click the "unlock" button at the bottom, then the properties tab for the user whose permissions you want to modify. Then go to :user privileges and check what you want, then close users and groups. Or you can do it from terminal.
[https://help.ubuntu.com/community/AddUsersHowto?highlight=(users)|(groups)|(and](https://help.ubuntu.com/community/AddUsersHowto?highlight=(users)|(groups)|(and))
