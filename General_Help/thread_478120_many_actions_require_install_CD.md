### Post by cptjaben on 2007-06-19
I just installed Ubuntu Edgy on my laptop using the alternate desktop CD that uses the text based installer. I did this because I my the Live CD kept failing to start the X server and therefore I could not install it. Anyway, everything worked out good using the alternate CD except that now many actions such as installing packages require me to insert the alternate CD, such as seen below:

[IMG]http://img529.imageshack.us/img529/6158/screenshot3nu9.png[/IMG]

Does anyone know how to fix this so that I no longer need to keep the CD around when installing packages? Thanks

---

### Post by Mothinator on 2007-06-19
Try going to System->Administration->Software Sources.

Enter your password, if it prompts you. Then navigate to the Third Party software tab. Uncheck "Cdrom with Ubuntu" and close.

I think this should solve the problem.
