### Post by tgoose on 2007-07-17
When saving (or Save as...), type .jpg on the end of the filename. For this to work it has to say Select File Type (By Extension) at the bottom. You can change this by clicking on the arrow.

If you want the GIMP to always save as JPEG (not a good idea, of course,) choose JPEG image instead of (By Extension).
