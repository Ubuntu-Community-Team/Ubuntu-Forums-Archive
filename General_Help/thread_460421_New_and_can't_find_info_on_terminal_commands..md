### Post by benanzo on 2007-05-31
the best way to learn about various terminal commands is to read the *man page* (manual) for that command.

For instance, if you want to learn how to use the *find* command, you would do this in a terminal:
```
man find
```

You can also have a look at the *info* pages for the commands.
```
info find
```
