### Post by FakeOutdoorsman on 2007-12-24
There isn't one Linux program that is like GarageBand, but several programs together can fill in some of the gaps.  [Ardour]("http://ardour.org/") is a good DAW, but if you're into MIDI, then you'll need Rosegarden too.  Hydrogen is perfect for making drum tracks.  Also, [Jokosher]("http://www.jokosher.org/") visually looks nice, but it's kind of new and I haven't tried it yet.
