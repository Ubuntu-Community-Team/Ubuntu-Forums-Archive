### Post by az on 2004-11-18
You can build a kernel just like in any other linux distribution.

You can, for convenience, use kernel-package to build and install your kernel.

Install build-essential, kernel-source, and kernel-package

You can use kernel sources from Linus if you want, instead of the packages source from Ubuntu.

You can also wait for Hoary to have a new kernel and install that,

There are many options.
