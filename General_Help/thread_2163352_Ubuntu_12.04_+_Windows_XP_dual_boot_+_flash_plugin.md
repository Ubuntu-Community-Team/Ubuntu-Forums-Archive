### Post by Impavidus on 2013-07-18
It could be that the site you visit wants flash version > 11.2. I think 11.7 is the latest on Windows. However, Adobe doesn't (and won't) provide the flash plugin version >11.2 for Linux. I think Chrome or Chromium should work with the latest flash build-in, although I don't know the details.
