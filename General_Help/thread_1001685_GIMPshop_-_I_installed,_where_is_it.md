### Post by ajcham on 2008-12-04
> How do I create a launcher? 

**In the Applications menu**
[LIST=1]
[*]In the top panel open System > Preferences > Main Menu
[*]Highlight the Graphics menu in the left hand window, then click the New Item button in the upper right.
[*]Set Type to Application (this is the default), Name to Gimpshop (or similar), and Command to gimp.
[/LIST]
[B]
On the Desktop[/B]
[LIST=1]
[*]Right click and select Create Launcher.
[*]As step 3, above
[/LIST]

That said, if Gimpshop doesn't give you the one-window interface you wanted (I thought it did, and that that was the point), I would advise you just try to get to grips with the standard Gimp package.  The reason being that Gimpshop is based on version 2.2, and the standard Gimp install from the repositories is v2.6. Unless there is some other advantage of using Gimpshop that I am not aware of?

I think once you get past the learning curve with Gimp, it's a perfectly decent digital editor.  A lot of people don't like to relearn techniques for basic operations, but I have the same trouble trying to use my sister's machine which has Paintshop Pro, as I'm now used to the Gimp's methods.  I do have an advantage insomuch as I don't really do any print work - this is the Gimp's biggest Achille's Heel as, AFAIAA, they are yet to introduce proper CMYK support.
