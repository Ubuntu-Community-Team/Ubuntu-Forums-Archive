### Post by Partyboi2 on 2009-09-18
1st. Download the Ubuntu desktop iso (by [[COLOR=Blue]torrrent[/COLOR]]("http://noncdn.releases.ubuntu.com/jaunty/") is the best way if you can)

2nd. Check that the md5sums match See [[COLOR=Blue]here[/COLOR]]("https://help.ubuntu.com/community/HowToMD5SUM") on how to do this.

3rd. If they match follow [[COLOR=Blue]this[/COLOR]]("https://help.ubuntu.com/community/BurningIsoHowto") on how to burn the iso to disk  (make sure to burn at x4 or less speed to avoid data corruption).

4th. Once the disk is burnt, boot the Ubuntu live cd and when you get to the Main Menu of the live cd choose the option to check the cd for defects.

5th. If it passes without any errors reboot into Vista and follow [[COLOR=Blue]this[/COLOR]]("http://apcmag.com/how_to_dualboot_vista_with_linux_vista_installed_first.htm") guide through to install Ubuntu.

If you get stuck at any point, please post back with what stage you are having a problem with  and what the errors are (if any).
