### Post by Tamlynmac on 2008-10-22
Check:

firefox/edit/preferences/main

About half way down the popup it says "Downloads". Which circle is checked?
"Save files to"
"Always ask me where to save files"

If save files to is checked then the location of your downloaded files is listed just to the right.

If always ask is checked you'll need to download a file and see what the default location is, if your not selecting a location when asked.

A good idea is to assign a location (folder) to the "Save files to" and all future downloads will go directly to the location (folder). For example I have a download folder that I made in the file browser (Nautilus)  and then assigned that as my "save to" in firefox. Now everything I download goes directly to that folder and is easy to locate.
