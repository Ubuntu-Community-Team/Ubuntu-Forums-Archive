### Post by murderslastcrow on 2010-02-10
Since they're working on a lot of new stuff in Karmic, the old usplash and GDM themes don't work, so you'll have to change those files manually in Karmic, within the system files (it's pretty sad, but I guess it's bound to happen when transitioning to new technologies).

[http://www.howtogeek.com/howto/7939/how-to-change-xsplash-themes-in-ubuntu-9.10/](http://www.howtogeek.com/howto/7939/how-to-change-xsplash-themes-in-ubuntu-9.10/)

That should get you on your way. Also, you can change the theme in the GDM, but it's kind of a pain in the butt, too.

[http://www.ubuntumini.com/2009/09/hack-karmics-gdm-login-screen.html](http://www.ubuntumini.com/2009/09/hack-karmics-gdm-login-screen.html)

Also, you don't really need a cleanup/defragmenting/registry-fixing manager in Ubuntu. The system does pretty well keeping itself clean and tidy without unnecessary bits. Just use the clear private data and clear recent documents buttons provided and everything should be pretty concise in your system.

Also, Linux doesn't have a registry. There is no point in having one- we just use good old configuration files that you can easily modify with GUI applications, so there's no need to mess with them in most cases.

Also, it's a good idea not to use Lucid unless you're a tester, although I'm sure it's not extremely unstable. The fact is that it will be difficult to get support for Alpha versions of Ubuntu software. There are plenty of good sources for Ubuntu already installed- what kind of programs do you want that aren't provided in the current respositories? There are a lot of exciting projects out there, after all.
