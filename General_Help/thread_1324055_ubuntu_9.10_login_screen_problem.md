To fix login screen theme after trying out high contrast, reinstall gdm.

For new users, here is how.

   1. Open System > Administration > Synaptic Package Manager
   2. In the Synaptic Package Manager, look for "gdm" package
   3. Right click on the gdm package then Mark for Reinstallation
   4. Click Apply button from the toolbar
   5. Logout to see login screen and check if it worked
