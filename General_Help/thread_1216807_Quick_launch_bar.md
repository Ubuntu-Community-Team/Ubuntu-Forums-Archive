### Post by collinp on 2009-07-18
You are probably thinking of the app called "Avant Window Navigator", a dock like bar that requires a compositing manager such as Compiz Fusion. It is in the Ubuntu repositories, to install it, run:
```
sudo apt-get install avant-window-navigator
```After that, to run it, click Applications, then Accessories, then Avant Window Navigator.
