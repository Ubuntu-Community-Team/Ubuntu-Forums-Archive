### Post by bollix47 on 2010-07-18
To reset firefox you can start it in safemode and there will be an option to reset to defaults:

```
firefox -safe-mode
```

As to the missing options you could try changing your theme ...  the new themes in 10.04 put those options on the left whereas something like clearlooks will move them back to the right.
