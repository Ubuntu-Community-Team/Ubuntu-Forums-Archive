### Post by BlueNoteMKVI on 2005-10-24
I'm accustomed to having ctrl+esc open my launch menu.  I learned this on Windows years ago, and I've been able to modify every flavor of Linux that I've tried to do the same thing.  I found the keyboard shortcut utility and assigned ctrl+esc to open the launch menu, but it does not work.  I did get a message about ctrl+esc being the default for some other action, but it removed the hotkey for that other action when I clicked "ok."  If I assign another combination to the launch menu (I tried ctrl+l) it works fine.  My other hotkey combos work just great.  What's up with this one?

---

### Post by BlueNoteMKVI on 2005-10-24
Found a workaround, but it still remains a problem.  The ctrl+esc combination is normally assigned to the task manager.  When I assigned ctrl+esc to "launch menu" it removed it from the "task manager" in the list, but apparently did not actually remove it.  "Task manager" was listed as have no hotkey.  When I reassigned "task manager" to another key combination, ctrl+esc worked fine for the launch menu.
