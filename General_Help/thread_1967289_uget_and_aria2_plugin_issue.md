### Post by vanquishedangel on 2012-04-28
Ok so I got uget but it seemed slow so I enabled the aria2 plugin and I was getting this "error: aria2.getVersion result error" and I scoured the web and found nothing on how to fix this error but found many for "aria2.getVersion failed" but I really did't want to recompile, I would rather just install from the repose so I finally found a fix here [http://ubuntudaily.net/archives/364](http://ubuntudaily.net/archives/364) thank you google translate lol

in uget go to edit>settings>plug-in>arguments and enter "--enable-rpc=true" in place of the default and it will work.
