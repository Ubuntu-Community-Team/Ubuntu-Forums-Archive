### Post by Hendrixski on 2007-04-09
:-) there's a very simple reason.

If you are going to do something wrong you do it where nobody can see you, in the shadows and not in plane sight.  when code is closed source you can hide behind a lot of stuff and do unethical things like viruses and spyware... or DRM and adware.  

If all of the code is transparent there is NOWHERE TO HIDE, so unethical programs dare not show up.  And if they do everyone can see them, and not let the criminals into their house.  Or they can modify the code to no longer have an ad, or not be malicious. 

That's why Linux has almost no malicious software.
