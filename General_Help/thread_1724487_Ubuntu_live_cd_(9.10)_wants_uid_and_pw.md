### Post by coffeecat on 2011-04-08
The username and password for the live session are:

Username = ubuntu
Password = blank (i.e. simply press enter.)

However, you should only be prompted to login in the live session if you have already used the desktop and are logging out and in again. If you are presented with the login screen when you first boot up, then something is probably wrong with the CD and login usually fails.

Try burning a fresh CD. Even better, download the ISO for 10.04 or 10.10 and burn a CD from that. 9.10 is near end-of-life and although that doesn't matter much when you are using it from the live CD, there are features in the newer versions that you may find useful.
