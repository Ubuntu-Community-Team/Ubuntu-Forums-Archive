### Post by krusbjorn on 2005-09-10
Save the script in ~/.gnome2/nautilus-scripts/, make it executable, and start nautilus. When you right click, there should now be the option "Scripts" available. 

To make the file executable, navigate to the nautilus-scripts directory in the terminal and type "chmod +x <filename>".
