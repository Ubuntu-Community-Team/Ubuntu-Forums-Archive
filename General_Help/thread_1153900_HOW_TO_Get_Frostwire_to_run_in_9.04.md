### Post by waspinator on 2009-05-09
If you've tried to install FrostWire 4.18 on Ubuntu 9.04 but get Fatal errors on startup, follow this short guide to get you up and running.

1. Install "Ubuntu Restricted Extras" from Add/Remove...

2. Go into terminal and run the following command, and choose the option corresponding to Sun Java
```
sudo update-alternatives --config java 
```

3. Reinstall Frostwire using the deb provided on frostwire.com

That's it. It should run now.
