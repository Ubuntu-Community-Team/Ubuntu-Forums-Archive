### Post by drs305 on 2009-05-31
If you are planning on a clean install, which would "wipe it all", you can use a feature of Synaptic. In Synaptic you can go to File, Save Markings and it will create a file with all your installed packages. You can then store that file somewhere that won't be formatted and on the new installation run Synaptic, File, Read Markings and point it to the saved file. It will import the packages in the list.
