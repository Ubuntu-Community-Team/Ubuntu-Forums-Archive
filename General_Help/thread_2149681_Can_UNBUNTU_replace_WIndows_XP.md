### Post by Mark Phelps on 2013-05-29
> **vanquish2012 said:**
> ... Can I re-format my HDD and install UBUNTU 13 and somehow retain the use of the Windows apps I've listed? 

The short answer is -- probably NOT.

While Wine, and its add-ons, are able to run SOME Windows apps, MS, among others, is constantly upgrading their apps and a version that might work well today might not work well after an upgrade is applied.  Office is a prime example of that: Office 2007 works pretty good, Office 2010 OK (for some apps), and Office 2013 is a nightmare.

If you really "Love Ubuntu" -- then you need to break away from MS Windows and learn to use alternative apps.  Staying with Windows apps is a disaster waiting to happen.
