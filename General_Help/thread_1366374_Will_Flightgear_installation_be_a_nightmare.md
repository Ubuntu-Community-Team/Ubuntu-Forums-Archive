If you don't have a 3D accelerated graphics card and driver then FlightGear will be slow, painfully slow, no matter what OS you use. Also the Linux version of FlightGear does not have a GUI aircraft/airport chooser. You will have to create a **.fgfsrc** file with those options or add them on the command line.
