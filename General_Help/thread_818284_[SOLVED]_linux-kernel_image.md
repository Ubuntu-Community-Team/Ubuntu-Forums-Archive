### Post by mrMango on 2008-06-05
A "dummy package" is simply a placeholder in the package repository. They're usually used for pointing apt to a correct package version, especially with the kernels. Just leave it alone; it isn't even taking up space on your system (well, the space it takes up is very negligable).

If it ain't broke don't fix it.
