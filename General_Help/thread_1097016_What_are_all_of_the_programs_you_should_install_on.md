### Post by mb_webguy on 2009-03-17
As far as getting Gnome (or any other application) to work, you shouldn't have to do anything special.  All of the packages in the repositories have a number of dependencies, which may have dependencies of their own.  So when you install the package containing Gnome, you install all of the software Gnome requires to function as well.
