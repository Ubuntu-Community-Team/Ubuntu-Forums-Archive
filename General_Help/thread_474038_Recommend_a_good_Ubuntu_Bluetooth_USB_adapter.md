### Post by butterfingers on 2007-06-15
I've got a Conceptronic which works perfectly well with Ubuntu. The model seems to be CBT200U2. 

In my case, getting Bluetooth to work with 6.06 Dapper and 7.04 Feisty has been extremely easy. If you are using Gnome, make sure Bluetooth File Sharing is installed (you can find it in the menu Applications >> Accessories >> Bluetooth File Sharing). This is the program I use, though you could use others. 

Then all I have to do is plug the USB and click on Bluetooth File Sharing. Now I can send from the mobile to the computer and viceversa (right click on the item you want to send and go to Send... and then you will see your mobile's name). Easy!
