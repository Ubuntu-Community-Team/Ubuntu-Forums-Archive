### Post by ecology2007 on 2007-04-08
> **Iron Man said:**
>  I enter my user name and get an error saying that the user name is case sensitive

Sounds to me more than a keyboard configuration issue than a username issue.
If your username is not lower case, the debian installer asks you to correct it and type a password. Then it uses it for the 1st ubuntu login.
Make sure every key of your keyboard is correctly responding in the login prompt. Check capslock too, and your numpad if your password has mixed cases and numbers in it.

You don 't need to download the iso again when you reinstall.
