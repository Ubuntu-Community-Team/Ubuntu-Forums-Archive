### Post by TheFu on 2015-02-18
Perhaps the best thing I can say is to check out youtube for step-by-step installation instructions.

Your Windows license will need to allow virtualized installs, which generally means no OEM disks which are tied to specific hardware and check the BIOS.

The [virtualbox.org instructions for creating the first VM]("https://www.virtualbox.org/manual/ch01.html#gui-createvm") are clear too.
