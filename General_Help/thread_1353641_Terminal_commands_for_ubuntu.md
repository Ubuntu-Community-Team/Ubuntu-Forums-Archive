If you want to get the details of a commands, 
you can use Linux manual with the command : man.

eg: if you want more details of the cp (copy) command you can do it like this..
1. Goto terminal : Applications -> Accessories -> Terminal  
2. Type : man cp
3. Press Enter key

Then you can see the list of details of the command which you want...
