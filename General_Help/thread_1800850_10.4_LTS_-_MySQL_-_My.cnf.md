### Post by Habitual on 2011-07-09
**MySQL permissions are NOT determined by the contents of my.cnf**
[http://ubuntuforums.org/showpost.php?p=11024613&postcount=2](http://ubuntuforums.org/showpost.php?p=11024613&postcount=2)

Mysql Server process started? 
Local:
```
(echo >/dev/tcp/localhost/3306) &>/dev/null && echo "TCP port 3306 open" || echo "TCP port 3306 close"
```
or
```
telnet localhost 3306
```
or ```
sudo pidof mysqld
```

Remote:
telnet ip.ad.dr.ess 3306 (for remote mysql server)

what happens?

Please let us know...
