### Post by bored2k on 2005-06-26
This is a simple question I have always had. For a regular user, is there any benefits on installing Sun-JDK over Sun-JRE ? Why ?

---

### Post by jcs296 on 2005-06-26
The average user should use the JRE.  If you write software in Java, or write software in another language and might want to try out Java, then you should get the JDK.  The JDK allows compiling java programs (and debugging them), this is not used by an ordinary user.
