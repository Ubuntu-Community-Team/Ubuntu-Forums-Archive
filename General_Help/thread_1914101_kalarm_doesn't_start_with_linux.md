### Post by uteck on 2012-01-24
Since this is in the [ubuntu] thread I assume you are not using kubuntu.
To get kalarm to autostart you will have to goto System->Preferences->Startup Applications
You can either manually add it, or use the options tab to remember running applications, but that will restart everything that was running when you logout.
