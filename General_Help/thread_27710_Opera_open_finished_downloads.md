### Post by zwaardmeester on 2005-06-10
Somewhere in the preferences you can tell Opera which programs to use to open finished downloads. For the "Open folder"  you can use Nautilus, and for "Execute file" (or something, I'm using Windows/firefox right now) use gnome-open. It should work now. (right click on finished downloads)
