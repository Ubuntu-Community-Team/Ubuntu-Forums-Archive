### Post by ssulaco on 2010-03-05
Go.....Places > Computer > Edit > Preferences > Views tab and look at "Icon view Defaults" to change the size,Mine is at 100%

Try right clicking the Icon > Properties > Basic tab........see if the name box is empty,or just try renaming the Icon

### Post by Itchikins on 2010-03-07
> Go.....Places > Computer > Edit > Preferences > Views tab and look at "Icon view Defaults" to change the size,Mine is at 100%

This was the problem.  Was set to 33% and apparently when at 33% it is too small to display the text under the icons.  Thanks a bunch for the tip   I didn't know that this affected the desktop.  Thought it only affected Nautilus.  I usually set my List View in Nautilus to 33% to fit more on screen and I must have accidentally change the Icon View setting too.
