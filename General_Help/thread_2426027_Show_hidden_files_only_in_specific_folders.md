### Post by #&amp;thj^% on 2019-09-03
**NOTE**:Nothing that elegant but, if you want to hide a file or folder, lets say MyFolder, just rename it to .MyFolder and it will be taken as a hidden file. So by adding dot makes it a hidden folder (including all the sub folders and files inside it).
To reveal it and the all the sub folders and files inside it use the Key combo>>{Ctrl+h}
