### Post by hansolo4949 on 2011-01-05
The simple way to partition your hdd to use several different linux distros, is to just install windows first, then install each individual version of linux, because as far as I know, when you install linux, most distros will ask you if you want to use the entire drive, or just part of it. You allocate the amount you want to give it and presto! That would be the way I would recommend if you don't want to get into some techy stuff. If you decide to use a partitioner, be very careful! one wrong mouse click, and you could delete one of your linux partitions.



 Hope that helps!
