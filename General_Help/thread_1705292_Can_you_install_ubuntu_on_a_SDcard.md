### Post by tredegar on 2011-03-12
I have an EEE701 with only 4GB of SSD.

I installed ubuntu to the 4GB SSD (it *just* fit), then plugged in a 16GB SD card, partitioned it, formatted it and then moved **/usr** and **/home** to it. Now there is space on the 4GB SSD for updates, new software etc.

It all works very well.

### Post by Lars Noodén on 2011-03-12
The SD card will be too slow for useful swap, use the built in drive for that.   

You might consider using the SD for data and the HD/SSD for the system.  Or follow tredegar's example.
