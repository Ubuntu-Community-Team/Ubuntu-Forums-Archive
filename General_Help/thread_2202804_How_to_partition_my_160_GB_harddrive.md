### Post by fantab on 2014-01-31
If you want the drive for linux then you can use Gparted, which comes with Ubuntu DVD/USB.
You will find [this tutorial]("http://www.dedoimedo.com/computers/gparted.html") useful. Its bit old but relevant.

Partitions are a very personal and only you know what you want. However, if I were you and installing only Linux on the Disk i would:
20Gb Primary Ext4 Ubuntu / (to install Ubuntu System files)
4Gb Primary LinuxSWAP
136Gb Primary Ext4 Ubuntu /home (to store your DATA)
