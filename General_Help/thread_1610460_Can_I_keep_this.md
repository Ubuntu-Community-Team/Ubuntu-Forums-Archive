### Post by wangsuda on 2010-10-31
> **lightxwings said:**
> What happens if I use the Wubi downloader?You install Ubuntu inside of Windows, and can remove Ubuntu like any other program. It's a great way to try out Ubuntu to see if you like it.>  Does that mean I can actively switch between using Ubuntu and my regular Windows?Only at start/boot up. You are given a menu to choose your operating system. Pick one!>  If I have applications like Quick Books and other things that save files (word documents and the like), will they be left intact and can I use them if I use Ubuntu (Wubi downloader)?Yes, they will be left intact, but no, you can't use them when using Ubuntu. Once you choose an operating system through wubi, that's the only system you get to use. But don't worry, Ubuntu has programs that work quite well with .doc format.
> Also, what's the deal with the different distros of Linx? Which are the best for general/main stream easy usage (that's decently powerful/fast)?It's a personal choice. My advice would be to try a few, funning them off live CDs. Or read the opinions you'll soon get here!
