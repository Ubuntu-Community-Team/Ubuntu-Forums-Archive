### Post by cariboo on 2008-10-17
Have you got the security repositories enabled in System-->Administration-->Software Sources. Most often when there is a security problem with a package, it is updated within a couple of days of the problem coming to light. The version numbers in Ubuntu don't necessarily correlate with the maintainers version. The current version in the repositories is cupsys (1.3.7-1ubuntu3.1).
