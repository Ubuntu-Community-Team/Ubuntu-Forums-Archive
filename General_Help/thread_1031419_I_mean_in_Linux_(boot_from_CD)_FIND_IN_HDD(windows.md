### Post by mlentink on 2009-01-05
Although I am not really sure I understand you correctly:

Yes, it is entirely possible to download a file (even a linux executable) in windows and then read that file after you have booted with an Ubuntu LiveCD. But you would not be able to install it, as a LiveCD is not 'persistent', all changes will be lost upon exit.
