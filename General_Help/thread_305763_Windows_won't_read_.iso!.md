### Post by meng on 2006-11-23
You don't "run" an ISO. You use the ISO to burn an image of a complete filesystem onto a CD, which will then allow you to boot the operating system contained in that filesystem IF you have set the BIOS correctly. Look here for how to burn from an ISO:
[http://www.psychocats.net/ubuntu/iso](http://www.psychocats.net/ubuntu/iso)
