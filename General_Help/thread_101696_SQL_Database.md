### Post by Rinzwind on 2005-12-10
mySql is a database that uses SQL (duh :) ). It's the best one around.

I suggest to install amarok and mysql and have amarok use a mysql database. 
That way by playing music you start a music-database AND get to see how it works.

MySql is BIG. It's not small but sql itself is fairly easy to understand.
MySql also comes with mysqladmin AND forums like phpbb can be run with a mysql database.

All of these programma (amarok, phpbb and mysqladmin) are very well documented so if you like to learn something it's best to use 1 of the best pieces of software there is.
