### Post by forrestcupp on 2008-03-09
I found out that this is just how AWN works.  But I did find something that works.  

You can't drag files from just any directory in Nautilus, but it is possible to drag files from /usr/share/applications to the dock to create a launcher.  So I moved my shortcut links to that directory and I was able to drag them to my dock.  I think this is the only way to have launchers without having a panel or an icon on the desktop.
