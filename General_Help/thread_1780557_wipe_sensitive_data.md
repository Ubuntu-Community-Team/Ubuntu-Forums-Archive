### Post by ottosykora on 2011-06-12
any file recovery software as [http://www.freeware-download.com/downloaddetails/15296.html](http://www.freeware-download.com/downloaddetails/15296.html) can make your files visible again and then you can shred them. 
But: on number of media real shredding is difficult or impossible. For example on flash sticks etc, even rewriting whole stick ten times will not destroy all data on it. On some modern hard drives it is also rather complex as the controller does not allow any direct access to the disk itself.
Shredding software does often promise more then it is able to do on recent hardware.
