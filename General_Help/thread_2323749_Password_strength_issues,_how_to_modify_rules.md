### Post by Morbius1 on 2016-05-07
```
sudo passwd morbius1
```
It will first ask for sudo's password then it will ask for the password you want to use for morbius1.

Sudo don't need no stinkin' rules so it can set it to whatever length or complexity it $@&* well pleases.
