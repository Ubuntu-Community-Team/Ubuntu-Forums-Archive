### Post by Gina on 2008-01-17
Although I'm a devoted Ubuntu user I do feel there are several areas that need sorting out before it will be a Windows beater for everyone.  Probably the worst is wireless followed by display problems.  8.04 has some way to go before it is even as good as 7.10 though there are areas where it is better.  

I have 6 wireless adapters none of which worked "out of the box".  Best was a PCMCIA card (Linksys) with a Broadcom bcm4306 chipset which only required the Restricted Devices Manager running to download and install the firmware and it worked.  Next a Belkin PCI card (in my old desktop) with a Ralink rt2500 chipset.  That needed a replacement driver downloaded and installed from the web to work at all but it still needs a script running at start up - to down the interface, set essid and up it again, before wireless works - not something for an inexperienced user!  Another PCMCIA card with an Atheros chipset isn't even recognised though lspci detects it and similarly with 2 USB sticks.  So far I have had no joy with getting these to work with Ubuntu.

8.04 Alpha 3 and daily builds since won't work on my laptop - garbled display - but Live CD works OK on desktop.  OK, I know there are known bugs and still a couple of months to get things fixed.  I'm really hoping for a big improvement when 8.04 LTS is finally released.

I love Ubuntu (and Linux generally) - it seems so much more logical than M$ bloatware - but I feel I can't unreservedly recommend it to all my less computer literate friends.  I think it's a pity!
