### Post by pytheas22 on 2008-09-18
I don't know about Ubuntu graphing-calculator programs, but I'm sure there's one around that someone else can suggest.

Ubuntu comes (I think; if not, you can install it with 'sudo apt-get install espeak') with a decent text-to-speech program that you can run by typing:
```

espeak
```

at a command line.  You can have it read a document to you by typing:
```

espeak /location/of/your/document
```

There's probably a GUI for it too if you look around.  If you want to change the voices and things (the default voice is hard to understand, in my opinion, and it reads too fast), take a look at the man page.
