### Post by flemur13013 on 2012-05-13
You should assume your files will get trashed regardless of how you set up /home or whatever: that means BACKUP THE THINGS YOU WANT TO KEEP before upgrading.  

Making /home a separate partition is generally a good idea; besides keeping your data separate from the OS, you can install more than one linux distro and have them all use the same /home.  Regardless of all that, I don't bother anymore because ...

Except for the ".wine" directory, my /home/username directory is pretty small and it's fast 'n' easy to copy somewhere else for safekeeping. I keep most files (music, etc) on a different partition (/data), and have two backup copies of that partition.  I think it's safer than keeping those files in a /home/username directory because linux has no default reasons for messing with it at all.
