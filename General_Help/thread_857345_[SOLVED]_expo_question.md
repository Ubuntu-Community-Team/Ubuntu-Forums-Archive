### Post by bobterri on 2008-07-12
Right now, expo opens all my workspaces (2) in one window when I drag the cursor to the upper right hand corner.  I want to open all the open applications in one workspace like expose does on a mac.  How can I change expo to work that way?

---

### Post by benerivo on 2008-07-12
I think you'll need to use the 'Scale' plugin to get this feature. It's the setting called 'Initiate Window Picker for Windows on Current Output', and you can set it to use screen edges/corners to initiate it.

---

### Post by bobterri on 2008-07-12
Thanks, Benerivo!  That did it.
