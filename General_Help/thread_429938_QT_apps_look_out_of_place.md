### Post by kejar31 on 2007-05-05
Ok I think I am happy now. :guitar: 

Heres what I did.

 
1. I installed kcontrol (KDE Control Center)
    This allowed me to manage my KDE color, icons and style

2. I downloaded the following color scheme "http://kde-look.org/content/show.php/Human+%28Revised%29?content=42016" and imported it via kcontrol and enabled it.

3. I set the icon theme from within kcontrol to human

4. Via synaptic ( apt-get if you wish ) I installed kde-style-klearlook. Afterwards I set the style from within kcontrol to Klearlook.
