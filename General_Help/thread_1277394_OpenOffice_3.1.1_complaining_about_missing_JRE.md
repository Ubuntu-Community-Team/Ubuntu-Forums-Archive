### Post by scouser73 on 2009-09-28
Try this:

To install Java, go to Syanptic Package Manager, in the search box type Sun Java & select & install sun-java6-bin, sun-java6-jre & sun-java6-plugin. Now click Apply, a small dialogue box will appear, then click Apply again.

To test your Java is installed and working correctly, go to the [Java Test Website]("http://www.java.com/en/download/help/testvm.xml")

### Post by Hagar Delest on 2009-09-28
Java is needed for some features (wizards for examples) but you can run OOo without. Make sure that in Tools>Options>OOo>Java there is a version _checked_. Wait a while so that OOo scans the system to find a JRE.
