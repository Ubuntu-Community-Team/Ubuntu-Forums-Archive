### Post by Mark Phelps on 2010-07-16
Unless someone responds who has used this make & model, we don't really know if it will work well or not.  The specs are certainly powerful enough, but the only REAL way to tell is to boot the machine into an Ubuntu LiveCd and see how well it works -- which is something you're not going to be able to do before you buy it.

Suggest you see if a store locally carries this item, take a liveCD with you, and see if you can try it.
