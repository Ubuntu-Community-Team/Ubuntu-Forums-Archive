### Post by akudewan on 2007-05-13
There's only a chance: You can try using something like [PhotoRec]("http://www.cgsecurity.org/wiki/PhotoRec"). (Go to "FileOpt" and just select the filetypes you are looking for, otherwise it will recover too many files)

Make sure you don't copy anything on the formatted drive, it will overwrite the data that has been deleted. If the data is really important, you can go in for professional data recovery.

You can apt-get PhotoRec by typing:
```

sudo apt-get install testdisk

```
