### Post by CorneliusSneed on 2016-12-16
Upgraded to 16.04 LTS today. The "Allow executing file as a program" box is checked, but upon right click no "run" option is available. (as it was in 14.04) 

Double click simply opens script as document. What am I missing?

---

### Post by deadflowr on 2016-12-16
You need to set it in Edit >> Preferences >> Behavior > Executable Text Files.
The default is, and has been, view.
Change it to either run or ask. Your choice.
