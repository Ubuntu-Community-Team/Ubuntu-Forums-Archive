### Post by P4man on 2009-09-03
photoshop is a windows program. Unless adobe made a linux version while I slept, your best bet is using linux programs on linux. Try gimp. 

sudo apt-get install gimp

If you can't find your way around the interface, you can try gimpshop, which puts a photoshop-like interface on gimp:

[http://www.gimpshop.com/download.shtml](http://www.gimpshop.com/download.shtml)
(get the debian package).

If you really must run photoshop, you need a windows emulation layer called wine; Its in the repo's;
