### Post by SoftwareExplorer on 2008-07-27
> **Killtodie said:**
> ok. i got emerald working.

all the themes I try only change the window outline, how can I get it to change the whole window itself?
As said before, an emerald theme changes the outline and titlebar. A GTK theme changes the buttons, scrollbars, menus, etc to whatever color (I believe it handles transparency, but I've never tried).

Its sounds like you are talking about a GTK theme. So, go to [http://gnome-look.org/content/show.p...?content=57437]("http://gnome-look.org/content/show.php/H-K+Suite+-+Emerald+Theme?content=57437") as steveneddy said and scroll down and look for the link to download that has "Gtk2 Theme" by it. Then save it in a place that you can find it and then go to System->Preferences->Appearance. Click install and find the file you just downloaded, open it. It should either say something like "theme successfully installed. Would you like to apply it?" or the theme will appear in the list of themes. Hope that helps, tell me if it does. :)
