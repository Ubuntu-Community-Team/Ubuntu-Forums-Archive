### Post by maxamillion on 2007-05-17
its possible that there is just some sort of a memory leak somewhere in an autostarted application or daemon, but that is honestly a unique case .... in my 7+ years of linux use on the desktop i have never had a performance loss as time goes on, actually had to opposite because as time goes on and you update/upgrade packages things tend to get faster or more efficient.
