### Post by sdowney717 on 2009-09-27
now I found out how.
Start playing the capture device
right click the screen select advanced controls
a red record button appears
just click it and the mpg file is stored in your home folder

[http://forum.videolan.org/viewtopic.php?f=14&t=53169](http://forum.videolan.org/viewtopic.php?f=14&t=53169)
also this

> Media -> Convert/Save... and use Capture Device tab to select your device, then press Convert/Save and choose transcoding options (if you don't know about codecs and containers, you can use for exampe Profile: Windows(wmv/asf) ). Then tick File and choose where you want to save the file and then press Save.
