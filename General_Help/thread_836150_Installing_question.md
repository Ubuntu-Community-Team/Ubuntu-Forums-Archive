### Post by myromance123 on 2008-06-21
Instead of burning a cd, you can read the iso file with an iso reader.
This link should help:
          [http://www.softwarepatch.com/windows/xpvirtualcd.html](http://www.softwarepatch.com/windows/xpvirtualcd.html)

That is Virtual CD, which should allow you to use the ISO file as if the file was on a cd and in your pc.
Install it and then run it. 
Mount is used to load the .iso file, then it will either load up like a CD or will be on your desktop. If its on your desktop, just double click it and it should run like a CD.

  Virtual CD is free (thats what I read), so have a go at it. Hopefully it works for you, and have fun with ubuntu :)
