### Post by kseise on 2011-04-17
I am posting from Natty, so I might be a little bit off on this, but...

In the GNOME main menu, go to System>Preferences>Removable Media
Look in there for the option that says to play CD's when installed.  There are several options for removable media available including copying photos from cameras, opening folders for browsing, etc.

### Post by grahammechanical on 2011-04-17
When I insert a CD that I do not want to run I hold down the shift key. An icon is then put on the desktop and the programs on the CD do not run. This works with audio CDs but I have not tried it with Rythmbox playing.

Regards.

I have just found another way. Open a Nautilus window and select Edit, From the menu select Preferences. Select the Media tab and where it says CD Audio change what is in the drop down box to do nothing. And see if that what you want.
