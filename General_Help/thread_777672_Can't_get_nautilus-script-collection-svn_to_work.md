### Post by Hokky on 2008-07-21
In case you're still looking. I found the solution here:
[http://ubuntuforums.org/showthread.php?t=302755](http://ubuntuforums.org/showthread.php?t=302755)

You need to enable the Subversion scripts with the command:

# nautilus-script-manager enable Subversion
