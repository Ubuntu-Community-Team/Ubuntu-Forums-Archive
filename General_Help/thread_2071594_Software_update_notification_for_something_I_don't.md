### Post by Frogs Hair on 2012-10-15
They are default packages necessary  for wireless. A new or even casual user would probably not know that they would have install  the wpasupplicant package to secure their wireless if it was not default.

 Though I have a desktop there are many packages related to battery use installed with my Ubuntu. There are many examples like this including Bluetooth and printing related packages. I use AMD and get updates for Intel related packages fairly often.    

A way around this would be a minimal installation and then you could ad what you want.
[https://help.ubuntu.com/community/Installation/MinimalCD](https://help.ubuntu.com/community/Installation/MinimalCD)
