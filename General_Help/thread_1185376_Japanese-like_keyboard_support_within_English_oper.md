### Post by creperum on 2009-06-12
I bought a laptop in Japan.  It came with Windows; I am planning to install Ubuntu as soon as possible.  The keyboard is a Japanese keyboard, which means that it has several keys not present on Western systems, which are used to switch between inputting different character sets.  (For example, there is a key to the right of the space bar that switches the input character set from romaji to hiragana. There is also a "kanji conversion" key, which converts a collection of hiragana to kanji, as well as several other "extra" keys.)

Ideally, I would like to be able to use these extra character switching keys when typing in Japanese, but from within my English system.  To put it another way, I'd like to be able to switch keyboard layouts from a Western layout to a Japanese layout that recognizes the character switching keys.

I have found several threads in the forums about enabling Japanese character support in English Ubuntu (such as [this excellent tutorial]("http://ubuntuforums.org/showthread.php?t=975144")), using SCIM and Anthy; however, it seems that these are designed for an English-layout keyboard.  Does anyone know of a way to switch keyboard layouts in such a way that my system will recognize the native Japanese character-switching keys?  Has anyone set up English Ubuntu on a Japanese computer and used it to input Japanese?

---

### Post by watchreader on 2009-07-02
You should just be able to change the keyboard layout without changing the system language.  It should be one of the first steps you choose when setting up Ubuntu, and should be available in the Administration preferences.
