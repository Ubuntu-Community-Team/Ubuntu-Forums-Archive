### Post by thasheep on 2006-05-31
I think it's incredibly unfare on debian (and hence ubuntu) to draw similarities with windows registry and package management. One of the problems with windows is that installing and removing files is very messy and not at all transparent. Apt, through dpkg keeps track of installed files and does a great job of installing and removing packages cleanly.
However, depending on how you've set it up, you might find a lot of (unneeded once installed) .debs in /var/cache/apt/archive. These can be removed with 'sudo apt-get clean' or 'sudo apt-get autoclean' (keeps current .debs). Dapper manages logs nicely by default (compresses them and stores them for a while before flushing them) so this isn't a problem. /tmp could get full but deleting its contents (including hidden files) is pretty safe, just reboot straight away. You can also find crud in your home directory but that's up to you, not the system.
Having said all that, if you want to have a nice official cd, why not? I haven't tried the livecd install yet but I've heard it's really really fast so you don't have much to lose. I think i might take kubuntu for a spin and if I like it, I'll clean up a partition to test the new installer in all its glory.

### Post by mgmiller on 2006-05-31
I couldn't wait and I figured I would beat the server crush by updating my 5.10 to 6.06 last night.  I used the new method alt/F2 gksudo "update-manager -d"  It was almost too easy.  There were a minimum of questions to answer.  The only post install problem I had was a broken x server caused by my incorrect use of the nvidia driver in xorg.conf and my using the wrong k7 kernel and restricted modules.  Not really Dappers fault that it couldn't deal with my ham fisted blundering.  At least it was easy to fix.  Changed the driver to nv and got x, then installed the right k7 kernel and restricted modules and nvidia fglrx.  then a quick nvidia-fglrx-config enable and restarted x.  Yay!  The only apps I had to reinstall were azureus and mplayer.  Pretty trivial.  I also was using totem xine with 5.10 and changed it to totem gstreamer0.10 with 6.06.  Just add in all the gstreamer0.10 plugins and codecs and it works great.  Total time to download and update the OS was less than an hour, & I spent another hour or so tweaking and playing.  It's smoother, faster, neater.   Ubuntu 6.06 ROCKS!   The only thing I don't like is the new screensaver app.  It doesn't let you choose which screen savers to use, it's now all or just one.  You also can't cnfigure them the way you used to be able to.  Evolution mail is much faster, allows scaling text size with the ctrl/scroll wheel, but by default uses Mozilla as a browser.  To change this to Firefox go to System > Preferences > preferred applications and change the browser setting from the default of Ubuntu Sensible Browser to Firefox.  Restart Evolution and it will now use firefox.  

Good luck all Ubuntistas on June 1st.  "U day"
