How can I increase the default font size in Firefox permanently?

---

### Post by xingmu on 2006-11-12
Go to **Edit** -> **Preferences** and click on the **Content** tab.    You can change the font size there.  You can also click on the **Advanced** tab and then change font sizes for specific languages if you find it necessary.
