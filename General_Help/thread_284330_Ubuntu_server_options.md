### Post by matthewstory on 2006-10-25
Let's see, where to start on this.  If you really want to get into it you could play with a couple of them to set up an LDAP cluster.  Or you could set up a samba server for their workstations on it.  The options are really quite high . . . depending on the nature of the company.  You could get really out there and set up a LAMP stack, install some wiki or forum software on that bad boy to help with collaberative documentation or development.  If they do really intense document editing you could set up a pretty slick subversion repository.  The sky is the limit really.
