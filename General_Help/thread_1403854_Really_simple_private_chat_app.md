### Post by jamesisin on 2010-02-11
Pidgin is a chat client and it will run chats over a good number of different protocols (including MSN, Yahoo, Google Talk, Jabber, IRC, &c).

The suggestion by Satoru-san is about setting up your own chat server (easy enough and it provides the most privacy).  It's also pretty easy to set up a Jabber server.  Either way you will still need a chat client (and either Empathy or Pidgin will work nicely).

Another option you may want to explore as you move forward is encrypted chatting.  I'm just not clear how deep you want to take this 'privacy' thing.
