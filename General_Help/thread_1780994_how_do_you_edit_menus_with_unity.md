### Post by mc4man on 2011-06-13
You can use any 'alacarte made' .desktops in unity several ways
(if you created a new menu  with alacarte that couldn't be used directly but could be recreated by other methods. (quicklist menus

When you create a new menu item in alacarte, (or edit one), there should be a .desktop created in ~/.local/share/applications/ 
You could simply drag it to the unity launcher and in most cases it will work fine

*in the long run better to learn about quicklists and how to add to existing .desktops and or create new ones from scratch
Keep in mind that the 'favorites' icons in the unity launcher are just shortcuts to an existing .desktop
