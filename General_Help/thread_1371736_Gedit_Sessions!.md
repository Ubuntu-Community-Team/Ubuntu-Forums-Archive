Do you know where the Gedit saved sessions are located? In my case I saved a session and titled it 'Redit'. I'd like to backup this 'Redit' file.

### Post by Barriehie on 2010-01-03
The saved sessions are stored in an XML file.  Mine is $HOME/.gnome2/gedit/saved-sessions.xml  This XML file is just a /path/filename for the session.
