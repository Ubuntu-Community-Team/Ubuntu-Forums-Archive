### Post by Drunky on 2008-02-22
The word you are looking for is *partition* as opposed to *petition*.

Right, spelling nazi out of the way.

Yes, you can install various linux distributions onto different partitions of your hard drive.  Most likely [Grub]("http://www.gnu.org/software/grub/") will be installed which upon booting your computer will give you the option of which OS to start.  If an option is not picked then it will start the first option after a discreet time has passed.
