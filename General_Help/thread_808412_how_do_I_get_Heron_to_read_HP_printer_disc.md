### Post by bwhite82 on 2008-05-26
.exe is a Windows executable, it can not be natively run in Ubuntu.

See the wiki: [https://help.ubuntu.com/community/Printers](https://help.ubuntu.com/community/Printers)

### Post by Het Irv on 2008-05-26
Have you tried looking for the drivers under the add a printer dialog box?  There are a ton of drivers already installed in Ubuntu, I haven't had any problems with them.

As to your ".exe", they only work in Windows.  You can use WINE or some other Windows compatibility tool, but in this case you are better off looking for something that will work natively in Linux.
