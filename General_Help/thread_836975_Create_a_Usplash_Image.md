### Post by angry_johnnie on 2008-06-22
Yes, you must create bars and throbbers as well.

Besides, Usplash is not some-image.png, but rather some-file.so.

[Here]("http://forumubuntusoftware.info/viewtopic.php?f=23&t=60") is a nice Usplash-Maker script, created by TheeMahn (from Ubuntu Ultimate Edition).

The script itself works.  But, apparently Hardy Heron (8.04) doesn't quite like it all that much.  There seems to be something wrong with imagemagick (the program that the script is using to convert the image into the appropriate color depth and resolution).

You may still want to try it... heck it just might work. :-)
If it doesn't, I guess you could still try to edit the image with the gimp (must be converted to 256 colors), and let TheeMahn's script do the rest.

Again, the script is fine.  It worked perfect under Feisty Fawn (7.04) and Gutsy Gibbon (7.10).

I still have a modified version of it.  And I still have Feisty Fawn in another computer.  If you try the script and nothing works, you may post a link to the image you want, and I'll make you an Usplash out of it and upload it here.  It takes only a minute or two. :-)
