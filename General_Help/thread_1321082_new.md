### Post by Giblet5 on 2009-11-09
Well, burning a CD is easy from Windows or Ubuntu. Just download the iso, bring up Windows Explorer or any file manager and double click it. Select burn.

Boot with that new CD in the drive. Select install. Overwrite 8.04. It's pretty obvious and easy, but there are people here that will help you 24/7.
