wine <your DOS location (be careful, you have to escape the \)>
```
optionally you can use the Linux-format...

### Post by directhex on 2009-05-24
> **Vadi said:**
> I don't think Mono supports .NET 3.0 to begin with.

.NET 3.0 is a set of add-on libraries for .NET 2.0: there are only two "real" versions of .NET, 1.0 and 2.0, the rest are just add-ons for 2.0

Mono supports some of 3.5, even some of 5.0, but not all of anything past 2.0

As for the OP's question, libmono-microsoft-visualbasic8.0-cil contains the assembly he's missing
