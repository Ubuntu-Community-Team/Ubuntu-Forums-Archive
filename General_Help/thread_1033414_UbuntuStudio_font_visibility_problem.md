For anyone who might encounter the same problem with FileZilla in UbuntuStudio:

I have discovered that I can do FTP just with the built in file browser. Just go system>places>connect to server chose "FTP with limited access (FTP mit Zugangsbeschränkung) and tick on "add bookmark" then you have the remote directory as a bookmark in the file browser. I can open my website files directly on the server with [[http://bluefish.openoffice.nl/](http://bluefish.openoffice.nl/) bluefish] and click store and it is already online. Nice Hmh?
