### Post by CatKiller on 2009-10-06
> **Anybloodyid said:**
> How do I open the file browser? where is it?

It's anywhere that you manage your files...

Places &#8594; Home folder (or pretty much anything in the Places menu) will open a Nautilus window, as will Applications &#8594; System Tools &#8594; File Browser, or running *nautilus* in a Terminal.

How do you normally move your files around? Do that.
