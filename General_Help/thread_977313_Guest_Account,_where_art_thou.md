### Post by prshah on 2008-11-10
> **Michelexub said:**
> 
Is it missing? Or do I have to enable the feature somehow?


You have to use the "Enhanced" quit user applet. Make sure you're fully updated, and you will be prompted to replace the old "quit" applet, with a new "combined" applet.

If you're sure you're fully updated, right click the panel, and choose "Add to Panel" and then select "User Switcher". A "twin" button will appear that will allow you to change users (and include a "guest user" option) and/or shutdown, hibernate, etc.
