### Post by Bucky Ball on 2016-03-12
Better to look for Edubuntu things in Software Centre, specifically edubuntu-artwork.

Anyhow, easiest way to see what you have and remove it is using Synaptic Package Manager. Install that and do a search for 'plymouth' and another for 'edubuntu'. I'd be very careful removing plymouth, though. I'm not on Ubuntu, but as far as I knew, that was installed by default and removing it may cause some serious issues. 

If you wanted a different theme, why didn't you pick one the regular way (on my Xubuntu install Apps> Settings> Appearance) or download one that suits your fancy? Plymouth has nothing to do with getting new themes or what you were trying to do (and as far as I'm aware you probably already had it installed). How did you try to install it? Software Centre? Not what you're looking for.
