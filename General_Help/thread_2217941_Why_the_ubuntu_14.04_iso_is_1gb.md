### Post by sudodus on 2014-04-19
Most Ubuntu flavour iso files are oversized for CD disks nowadays. They are intended for computers that are 'modern enough' for DVD drives and for booting from USB. But there are exceptions.

1. The Ubuntu mini.iso alias netboot installer alias Minimal CD. This iso file is very small, and can be used to install all Ubuntu flavours.

2. The Lubuntu iso files. Both the desktop iso file and the alternate iso file of the current versions (13.10 and 14.04 LTS) fit in a CD disk. *Edit 1*: with one exception, the desktop powerpc iso file.

*Edit 2:*
3. The Ubuntu server iso files (I found data to verify the size later: These fit within a CD disk.)
