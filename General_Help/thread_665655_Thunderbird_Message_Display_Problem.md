### Post by techgeek_us on 2008-01-12
Since upgrading to Ubuntu 7.10 (clean install), Thunderbird has occasional but routine problems displaying messages.  Sometimes the message will not display in the preview pane and sometimes it will not display after opening.  Sometimes it will not display in the preview pane but will if I open it.  It's not consistent with the messages either.  Some display fine and some don't.

I'm using IMAP in case that makes a difference.

The only thing that seems to work around the problem is disabling filters, cosing and re-opening Thunderbird.  Every message displays properly then, but I really can't live without the filters due to all of the messages from mailing lists I get.

I have tried recreating my profile and setting up the filters again.  That seemed to work initially, but the problem quickly manifested itself again.

Any help appreciated.

---

### Post by techgeek_us on 2008-01-20
For the record, this seems to have been caused by a corrupt IMAP folder that the filter was trying to place messages in.  Once I deleted that folder things started working normally and I haven't had the problem since.
