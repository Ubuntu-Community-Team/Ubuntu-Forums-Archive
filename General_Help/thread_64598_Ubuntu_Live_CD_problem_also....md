### Post by heimo on 2005-09-11
To install Ubuntu or to boot Ubuntu live CD you need to choose your CD drive as a boot device. How this is done varies from computer to computer (depends on BIOS). You can get to BIOS settings hitting F1 or F2 or F10 or DEL key at the very beginning of POST (when your computer starts) - sometimes theres a text which tells you which key to hit to enter BIOS. There you should find setting which controls boot order. You need to have CD drive above hard disk to boot from CD. (save settings and reboot with CD in drive) Sometimes you need to hit space to activate booting from CD - there should be instructions on your screen if this is needed.

Sometimes you have a BIOS boot device list. Pay attention to the texts just after you switch computer on. :) If you've ever installed Windows, this is just the same thing. :) Hopefully this helps.
