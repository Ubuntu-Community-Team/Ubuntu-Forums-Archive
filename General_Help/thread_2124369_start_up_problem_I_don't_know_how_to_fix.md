### Post by agillator on 2013-03-10
The obvious first: is your bios set to try to load from the CD FIRST? Assuming yes, what actually happens when you try to boot from the install CD? It can be slow, very slow, to boot from the cd so could it be booting and just taking longer than you expect? Your error, of course, is what the message says: it cannot mount the filesystem. Somehow we need to find out why (talk about the obvious!). It could be a number of things: bad file system, bad partition table, bad boot loader, etc. Do you have windows on your machine and if so will it boot?
