### Post by sprpdr on 2008-11-10
If anyone has had the issue of not having any sound in flash videos after the upgrade to intrebid, the easiest thing to do that will most likely fix this is to use synaptics package manager to install the adobe flash plugin for firefox, which will automatically remove the current flash plugin installed. Just search firefox and scroll down the list to find this plugin. This was the only thing that fixed my issue and I hope it helps you.:)
