### Post by Alterax on 2009-05-07
As standalone machines, not recommended.  On the other hand, if you're using LTSP, most of the heavy lifting is being done by the server.  So if you're using the Pentium 1 as a thin client, you should be just fine, provided you have at minimum 128 MB of RAM.
