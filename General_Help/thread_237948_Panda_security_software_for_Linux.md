### Post by canadianwriterman on 2006-08-17
It's more for the protection of any Windows connections you may make through your Ubuntu system. For example, you could, unknowingly, pass a virus to a Windows user through an e-mail you send. The same theory applies for people whose Ubuntu boxes are connected to a Windows server. Perhaps it also applies if you save files from your Ubuntu OS to a fat32 partition and then open it up in your Windows OS, but I'm not sure.
