### Post by swegner on 2008-09-10
Rhythmbox, which is the default music player in Ubuntu, has support for "Radio" streams.  It has many preset stations, which appear to use .m3u and .pls format.  You can also use your own stations.

### Post by kostkon on 2008-09-10
> **woli said:**
> I know that there are many applications that come built-in with ubuntu, but I want to ask if there is any specific application made for playing internet music streams, and that would save my "radio stations".
You can use *Rhythmbox*, it has internet radio support and allows you to add your own stations.

Another option is *Streamtuner*. It comes with support for *Shoutcast*, *Live365* and some other internet radio services so you will find a great selection of radios. You can also add your own, of course, by selecting *Stream -> New Preselection*.

If don't want to install another one media application, you can just use *Rhythmbox*. 

If you like the idea behind *Streamtuner* and the fact that comes preloaded with many radio stations then go for it.
