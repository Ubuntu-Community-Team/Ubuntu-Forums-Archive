### Post by Vaphell on 2013-12-08
if you are using the default ubunt environment (unity), why don't you go with default shortcuts associated with launcher icons.
WIN+# runs the program under the #-th icon in the launcher bar. By default first icon is nautilus so WIN+1, second is firefox so WIN+2... etc. To see them press and hold WIN key. You can pin whatever programs in whatever order and these shortcuts will work.
