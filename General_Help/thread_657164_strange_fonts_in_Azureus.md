### Post by gerben1 on 2008-01-03
All of a sudden certain words in Azureus are displayed using some strange font. I don't recall having changed anything concerning fonts recently. Does anybody know what could be the source of this problem?

[http://img146.imageshack.us/my.php?image=screenshotazureuszd5.png](http://img146.imageshack.us/my.php?image=screenshotazureuszd5.png)

...
Nevermind, I found the problem. For some unknown reason the language was set to Armenian, the first language in the list. 
Anyways, just running  "Tools-> Configuration Wizzard" and setting the language back to English solves it.
