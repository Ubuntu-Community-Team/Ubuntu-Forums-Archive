### Post by mutation on 2008-06-24
running with and nvidia video cards multi monitor works perfect.  Make sure you have the nvidia-settings installed, and then enable twinview and your set to go, i run 2 21" monitors(crt's can't afford LCD panels yet), the only problem that i run into is i have to disable one monitor if i play games(primarily through wine)
