### Post by santiagorf on 2008-10-02
Hi all,
I was downloading a folder with several files using Transmission, and accidentally I erased the folder. 
I have recovered the folder with all its files from the recycle bin, but I don't know how to add them back to Transmission in order to continue downloading them.
Any ideas?

Thank you!!
santiagorf

---

### Post by justleen on 2008-10-02
place the folder back some where, add the torrent again to transmission, and point the download dir to the folders you just restored. It should start with checking the progress.. 
try not to point to the actual dir you restored, but rather to the folder containing the restored folder.

it worked. thanks!!
