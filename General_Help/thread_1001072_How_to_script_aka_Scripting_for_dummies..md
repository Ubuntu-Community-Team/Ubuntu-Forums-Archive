### Post by strider1551 on 2008-12-04
My advice is to start with something less obscure.  I have no idea what Kixtart even is ([_wikipedia_]("http://en.wikipedia.org/wiki/KiXtart") says it's for windows?!).  The more popular a language is, the more tutorials, howto's, sample code, etc you can find about it.

If you're in the Linux world (since this *is* the ubuntu forum...), bash is by far the most common language to use for scripting.  If you want to get into some actual programming, python is a popular thing to start with.
