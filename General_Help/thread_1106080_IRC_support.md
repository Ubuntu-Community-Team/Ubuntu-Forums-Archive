### Post by crashsystems on 2009-03-25
One of the great things about IRC is that you can use any name you want when you connect, as long as it is not currently in use, and no one has registered it. So just put in the username of your choice and leave the password blank. If it is already taken, then the IRC server will append a number to it to make it unique.

[http://freenode.net/using_the_network.shtml](http://freenode.net/using_the_network.shtml) is a great intro on using IRC. Keep in mind that many of these things can be done graphically if you are using Pidgin. Also, [http://www.mibbit.com/](http://www.mibbit.com/) is a great browser based client for those who do not want to mess with client side software.

### Post by iareConfusE on 2009-03-25
If you have an IRC channel that you want to get to, you need the server name/ip address, and the channel name.  In pidgin, setup an IRC account using any username you want.  You don't need a password.

Once you've done that, you can join an IRC chat by going to Buddies > Join a Chat, and select your IRC account.  It'll prompt you for a server and a channel which you should have.  It also asks for a password, but if you know the server doesn't have a password, then just leave it blank and log in.  Thats it.

Usually IRC channels are for people of a specific group or interest.  You can browse through IRC channels, though I'm not sure how to do this in Pidgin, since I used mIRC on my main desktop with Vista.  So I hope you have a channel already in mind, or one that was given to you by a friend, so that you won't feel out of place in the chat.
