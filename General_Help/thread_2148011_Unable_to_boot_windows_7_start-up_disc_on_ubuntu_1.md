### Post by grahammechanical on 2013-05-23
Do a test. Unplug the hard drive. Switch on and put the Windows disk in the DVD drive. Now, the machine cannot load an operating system from the hard drive. It will be forced to look for the OS on the DVD drive. Does it? Do not try plugging the hard drive back in if the windows disk is being used.

When you plug in that Windows cloned drive, can the windows DVD load then? If not there the issue must surely be in the BIOS boot order. Does the DVD show up in the BIOS?

Regards.
