### Post by in·ter·punct on 2011-10-21
Not sure if this is what you want, but:

[LIST=1]
[*]Open nautilus
[*]Edit > Preferences > List Columns Tab > Location Checkbox
[/LIST]
If you're using list view, the path to the file will be displayed in one of the columns.

### Post by in·ter·punct on 2011-10-21
I'm using 11.10 but this should work in 11.04 as well.

You're using icon view, so you'll first need to select list view to be able to see the columns:

[LIST=1]
[*] Open Nautilus
[*] Edit > Preferences > Views Tab > View new folders using: List View (under Default View Heading)
[/LIST]
