### Post by mwolfe on 2010-06-30
Some websites check the browser's user agent string looking for internet explorer browsers or perhaps the microsoft windows operating system. While its possible some of these sites are completely ignorant about it, it's more likely that the site just plain won't work with your browser even if you do fake the user agent string.. That being said, its worth giving a try. Here is the add on:
[https://addons.mozilla.org/en-US/firefox/addon/59/](https://addons.mozilla.org/en-US/firefox/addon/59/)
Its called user agent switcher.
Unfortunately it only comes with a few IE user agent strings.. Sometimes it's helpful to have a firefox windows UA, you can find those online if you google for it. You'll have to add it manually though.
