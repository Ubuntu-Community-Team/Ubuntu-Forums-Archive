### Post by alberto ferreira on 2009-06-17
You must install a "program" that acts like an emulator - WINE - that allows you to run Windows apps under Linux ( however it doesn't always work )

Install it as any Ubuntu app through terminal:
sudo apt-get install wine

or in a graphic interface - GUI - go to:
System>Administration>Synaptic>Search for wine>install
