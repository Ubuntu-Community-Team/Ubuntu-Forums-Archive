### Post by VCoolio on 2009-05-01
Try his:
in system > preferences > keyboard go to tab 'layout' and add Arabic. 
Press button "Layout options" and under line "Layout switch" choose the key to change layout automatically.
Now you should be able to type arabic when you change keyboard layout with the keybinding you chose. Maybe you need to put an Arabic font in the folder /home/you/.fonts . Create that folder if it doesn't exist yet.
Hope that helps. You can also add a keyboard indicator on your panel (right click on panel > add to panel > keyboard indicator): click on it and your layout switches to the next. With right click on it you can easily open keyboard preferences.
