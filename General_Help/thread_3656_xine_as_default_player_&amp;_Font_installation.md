### Post by ulrich on 2004-11-08
1. if you right-click on a video-file in nautilus > properties >open with there you can set the default player for this filetype.

2. **apt-get install msttcorefonts** if you want the basic win-fonts. (verdana, etc...)
if you mean that you just want to be able to install any .ttf-font you want, then open up nautilus, hit **Ctrl + L** and type in **fonts:///** and there you go.
