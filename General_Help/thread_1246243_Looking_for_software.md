### Post by ibuclaw on 2009-08-21
> **KingArthur72076 said:**
> * Activesync replacement for my htc diamond
Don't really use flash mobile devices myself, but there appears to be two packages for you to use here:

1) [SynCE-GnomeVFS]("apt:synce-gnomevfs") so you can connect to the device via the File Browser (Nautilus) using the path:  synce:///
2) [SynCE-TrayIcon]("apt:synce-trayicon") so you have an app in the tray icon area. (You must add it first by right-clicking on the panel and selecting "Add").
