If you still doesn't know how to install an icon theme, try this:
Download an icon theme in gnomelook.org/open your home folder/make a new folder and rename it .icon  (if you dont have this folder yet)/drop your downloaded file on this folder/right click that file and extract it there/go to system/preferences/appearance/customize/icons/choose the icon theme that you downloaded/close/ hope this helps.
