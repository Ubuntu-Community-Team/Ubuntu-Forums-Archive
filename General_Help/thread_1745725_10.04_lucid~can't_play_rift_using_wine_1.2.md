### Post by Kenjitamura on 2011-05-01
To my knowledge Rift can't be played under wine version 1.2, you need the latest 1.3.  If you can't find a fix for 1.2 unfortunately you're going to have to uninstall wine, delete the wine directory including the Rift install, install wine 1.3 and reinstall Rift from scratch.  

In this thread you posted in I answered why you were getting the repository error and a possible fix so that you can install wine-1.3 from the package manager.  [http://ubuntuforums.org/showthread.php?p=10751066]("http://ubuntuforums.org/showthread.php?p=10751066")
