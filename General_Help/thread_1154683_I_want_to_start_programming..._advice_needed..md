### Post by 3rdalbum on 2009-05-10
> **higashi said:**
> I want a programming language that:

-is as easy to learn as possible

Python is that.

> -is extremely powerful (in the sense that it has a lot of capabilities)

C and C++ are fantastic in that regard, but are not easy to learn. Python probably comes 2nd place.

> -can be used to make fast programs, rather than laggy ones..

That depends more on how you write the program, rather than what language you use.

For instance, I wrote a program in Python that gets a directory listing from an HFS disk and puts it into a list box. The first way I programmed it, a large directory could take 10 seconds to read and put into the list box. I tried a different method that used a couple more lines of code, and could display the directory list in under 1 second.
