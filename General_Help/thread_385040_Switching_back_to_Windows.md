### Post by forethought on 2007-03-15
Windows, historically, does not play nice when trying to install to a hard drive with another operating system on it, which is why if you want to dual-boot, it is recommended that you install Windows first, and set your partitions during the install process, and after Windows is installed, put whatever other OS you want on there.

But like someone has suggested, unless you're wanting to run games and/or some intensive audio/video editing software, VMWare is probably an easier path, unless you feel like reinstalling both Windows and Ubuntu.
