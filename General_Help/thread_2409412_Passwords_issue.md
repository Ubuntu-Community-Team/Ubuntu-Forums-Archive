### Post by The Cog on 2019-01-02
You have to have usernames and passwords. It's not optional.
However, later in the installer you get the option to have the user automatically logged in as the machine boots. This does not mean you can forget the password though, as you will need to repeat it when you want to perform many administrative tasks even if you have already been logged in.

### Post by Impavidus on 2019-01-02
Yes, Linux is open source, free and highly configurable, but usernames and passwords are so deeply ingrained into the system that it's practically impossible to remove them and still have a secure system. Are you willing to rewrite large parts of the operating system to be able to use it without username and password? You're allowed to do so.

Once you know your password and stored it in motor memory, you should be able to type it in three seconds on a familiar keyboard.
