You need more RAM. Ubuntu and Ubuntu Netbook Remix both require 512mb minimum (and 1g is better), see here for more info:

[https://help.ubuntu.com/community/Installation/SystemRequirements](https://help.ubuntu.com/community/Installation/SystemRequirements)

If you can't get more RAM for whatever reason, try Xubuntu or Lubuntu.

Netbook Edition does NOT require an Atom processor, however it is not lighter/faster than "regular" Ubuntu in any way; it just uses a different interface designed for small screens.
