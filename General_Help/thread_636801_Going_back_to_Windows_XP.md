### Post by lvleph on 2007-12-10
You don't need WinRar for rar files. Run the following command in the terminal.

```
sudo apt-get install rar
```

Also, you don't need to extract anything from an ISO or burn it, you can mount them using this [tutorial](http://www.ubuntugeek.com/mount-and-unmout-iso-images-without-burning-them.html).
