### Post by editrix on 2008-09-17
Welcome! One of the joys of Linux is that you can install (and uninstall) to your heart's content. There's no registry to mess up, and in the unlikely even of a conflict between what you want to install and what you already have, apt (or aptitiude or synaptic) will tell you ahead of time.

I'd suggest using synaptic to search--using the categories feature--for stuff that sounds interesting. Also, these forums often suggest good programs/utilities, often when people are asking for help with them.

Here's a list of some of my favorite applications, utilities, and games...
[LIST]
[*]GPRename, a batch file renaming utility
[*]KDirStat, a graphical disk usage utility
[*][Deluge]("http://deluge-torrent.org/") bittorrent client
[*]Amarok music player
[*]Audacity audio editor
[*]Avidemux multimedia editor
[*]Gnome Subtitles, for fixing subtitle files
[*]MKV Files Creator (mkvmerge-gui), a nice and simple GUI for creating mkv files
[*]OGMRip, a simple DVD ripper with support for the newest and best containers and codecs
[*]VLC media player
[*]Gmount-iso, a simple GUI for mounting iso files
[*][Wine]("http://www.winehq.com/"), for software that stubbornly insists on being run in Windows
[*][IEs4Linux]("http://www.tatanka.com.br/ies4linux/page/Main_Page"), for those foolish web sites that assume everyone is a slave to Microsoft
[*]Gnome Configuration Editor (gconf-editor)
[*]Nautilus Actions Configuration, to add commands to the Nautilus context menu
[*]Glipper, a clipboard manager panel applet
[*][GridWars 2]("http://www.getdeb.net/app/GridWars+2")
[*]Frozen Bubble
