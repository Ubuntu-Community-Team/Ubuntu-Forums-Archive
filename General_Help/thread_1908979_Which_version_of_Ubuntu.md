### Post by 2F4U on 2012-01-14
If you want to use it as a Server, I would recommend the Ubuntu Server edition. Comes without GUI and with more harddisk space your PC could be a perfect file server.

If you want a GUI I would recommend that you try either Xubuntu or Lubuntu. Both are more lightweight than Ubuntu or Kubuntu but are still based on the Ubuntu repositories.
