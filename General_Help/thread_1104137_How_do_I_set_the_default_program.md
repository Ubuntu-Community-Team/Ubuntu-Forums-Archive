### Post by tbrminsanity on 2009-03-23
I prefer to use Adobe Reader to view PDFs over Evince Document Viewer.  But Evince is the default program for PDFs.  How do I change this?  I've looked under Preferred Applications, and I've looked in Gnome-Config but I don't want to mess with it until I know its safe (I've burned myself a couple of times in there).

### Post by Zimmer on 2009-03-23
My suggestion would be to right click on a .pdf file in Nautilus , select the  'Open With' tab and add Adobe Reader to the applications list(if not already there) and select it as the application preferred.
This should make all .pdf files open with Adobe by default..
