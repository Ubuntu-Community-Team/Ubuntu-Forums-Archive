### Post by Cavsfan on 2011-10-18
The latest and greatest Java version 6 update 29 just became available.
The red link in my signature will check what version you have installed.
The green link will provide instructions to remove the previous version and install the new version.
Except you will need to change the "27" to "29" as the site has not been updated yet.

Just performed the update myself and it's good to go.

If you have a 32 bit machine, the instructions are on the left, for 64 bit machines the right side has the instructions.

It says to remove the old version first and mentions the instructions for doing that are at the bottom.
Then you just start from the top and copy and paste the commands and you should be good to go. :)
