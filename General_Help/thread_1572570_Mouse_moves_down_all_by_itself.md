### Post by Gunman1982 on 2010-09-11
Is it a laptop? Check if there is something on the touchpad.

Is it a PC with an optical mouse? Check if there is something obstructing the sensor beneath the mouse, sometimes my mouse behaves strangely and when I check there is a small hear where the optical sensor is and messes around with my pointer.

Is it a PC with an old mouse with a trackball? open the mouse and check if there is something bugging with the sensor.

Otherwise: If you unplug the mouse and the cursor stops moving its a pretty sure indicator that you mouse is busted.
