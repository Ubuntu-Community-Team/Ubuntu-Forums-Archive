### Post by diablo75 on 2008-11-29
Open a terminal window and type:

```
sudo apt-get install compizconfig-settings-manager
```

This will install the advanced compiz control panel that is not included with Ubuntu by default.

It will add a new item to your System>Preferences menu.

Once you have that open, click on the "Animations" item, and you'll probably be able to go from there on your own.  Have fun!

(Check out the "Random color" option in Burn for something really cool).
