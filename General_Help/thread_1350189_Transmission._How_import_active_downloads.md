### Post by marquinos on 2009-12-11
Well, I found the solution.
Copy the Download folder of Transmission to the other computer.
Then if you open the .torrent file in the other computer, and you select the same folder copied, the download will be resume.
The .torrent file is here: ~/.config/transmission/torrents
