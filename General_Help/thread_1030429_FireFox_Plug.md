### Post by taurus on 2009-01-04
Applications -> Accessories -> Terminal
```
sudo apt-get update
sudo apt-get install ubuntu-restricted-extras
```
When you get to the licensing screen for java, press the Tab key to highlight the <OK> and Return to accept it.
