### Post by p_quarles on 2008-08-14
The icon is going to be somewhere in the /usr/share/icons/ hierarchy. I don't remember which iconset Kubuntu uses by default, but the system tray icons are 22x22 pixel icons. 

Basically, you need to find the iconset in which the icon you don't like resides, and replace it with the one you want (i.e., the original default Amarok icon). Basically, switching out the file is all you need to do.
