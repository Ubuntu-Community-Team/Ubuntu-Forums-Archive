### Post by sydbat on 2011-01-18
Two questions...

1) Do you have the NVidia X Server Settings installed (found under System > Administration)? If not, install it (you can find it in Synaptic).

2) If you do have the NVidia settings GUI already installed, have you gone into it and clicked "Detect Displays"? This should find all your video cards and allow you to configure them separately.
