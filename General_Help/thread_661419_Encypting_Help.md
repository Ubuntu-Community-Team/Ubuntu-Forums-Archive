### Post by Bllasae on 2008-01-07
If you encrypt your hard drive, you won't be able to use it. If you installed your operating system on it, that's why it erased it. The computer doesn't have access to necessary files in order to run.

### Post by Bllasae on 2008-01-07
> **G-Unot said:**
> so how can i encrypt my hard drive then?
That depends. Like I said before, if you encrypt your main hard drive, then you can't use it until you unpack it again. If it's an external hard drive, then there should be password protection software already on it. If not, you can use Winzip.
