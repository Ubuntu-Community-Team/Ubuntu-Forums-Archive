### Post by Woodgar on 2007-01-13
Another important thing to recognise is that Windows update will only offer updates for the core operating system, ie Windows itself, whilst Ubuntu update will offer updates for the core operating system, ie Linux, and in addition will offer updates for any programs you have installed via the repositories as well, ie Firefox, OpenOffice, graphical applications, games and so on.
