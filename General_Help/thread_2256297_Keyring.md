### Post by nerdtron on 2014-12-11
Keyring is probably from Chromium asking you for syncing your account. Another one is from the Software which will only happen when you install software from the software center (of course you need a password for software installation).

You can remove the annoying keyring of Chromium by following the answers on this thread [http://askubuntu.com/questions/495957/how-to-disable-the-unlock-your-keyring-popup](http://askubuntu.com/questions/495957/how-to-disable-the-unlock-your-keyring-popup)
