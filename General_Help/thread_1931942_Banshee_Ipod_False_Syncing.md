### Post by cbanakis on 2012-02-27
This is not a bug.

Apple constantly changes how the file system works on the iPhone, and iPods with every update, making it nearly impossible for 3rd party applications to write to the iPod/iPhone.

This is presumably done on purpose, so you have to use iTunes.

I suggest trying to downgrade your iPod software, 1 version at a time, till it works.

And NEVER update.

You may have noticed that most of the time, when you update your iPod, nothing is changed, or fixed.

This is because the updates only purpose, is to re-arrange the file system, so it wont work with 3rd party software.

Some of what I said is my personal opinion, some of it is fact.

All I know for certain, is that I never updated my iPod, and it has always worked in Banshee, and in Rhythmbox.

And My buddy has the same iPod, he updates it regularly, and his only worked 1 time, then never again.
