### Post by DagMan on 2007-08-09
Virtualbox is very good.  Fast, efficiant, easy to setup... well easy to install an os.  Using usb, for example, will require just a little bit of a google.  It's free.

There's a package for it in the automatix repo and an old but decent guide for ubuntu here [http://doc.gwos.org/index.php/VirtualBox](http://doc.gwos.org/index.php/VirtualBox) but lots of info on google as well.  There's no man page but a gui based help function that reminds me of Windows.  I find it difficult to sort thorough help like that, but it is really useful if you take the time to sort through it.
