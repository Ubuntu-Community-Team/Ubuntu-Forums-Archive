### Post by LSenf on 2010-11-11
Assuming that you are using proprietary drivers for your graphics card, the only thing coming to my mind here is the Clone Output plugin of Compiz.
You could first set the screens up as twinview and then use Compiz to clone the larger screen to the smaller one - that would give you the full resolution on the larger screen while showing the full output on the smaller one.
I don't know whether this plugin is included in the standard compiz package, however it should be available in one of the compiz packages in the ubuntu software sources. (Use compiz config settings manager (ccsm) for configuring that)
