### Post by geraldm on 2007-09-29
If you expect the printer to work in linux/*bsd it is best to consult a database of working 
printers.  That printer is not listed in openprinting.org database.  You could supply details
for it.  Is it a postscript printer? raster-driven printer? winprinter?
Does it have any interface(s) besides wireless?

Without details, a developer for the lexmark would be needed, so you would have to search
for some organization where those developers can be found.

Gerald
