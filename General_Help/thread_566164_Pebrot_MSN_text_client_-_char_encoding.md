Hello,

I'm using this text MSN client

[http://pebrot.sourceforge.net/](http://pebrot.sourceforge.net/)

but I've some problems with special characters like àèéìòù which I need.

Does anybody know how to make it work with Ubuntu (which is UTF-8 only I guess)?

[SOLVED]

I solved using konsole as terminal emulator, and playing with Settings -> Encoding
or by using the following command:

*$ LC_ALL=en_US.iso88591 LANG=en_US.iso88591 luit <application>*
