### Post by tgalati4 on 2015-02-13
```
cd
mkdir Projects
cp your_tar_file.xz Projects
cd Projects
tar -xzvf your_tar_file.xz
ls -la
cd your_tar_file
ls -la

```

After a while you get used to it.

Search for a BASH tutorial.  There are a few on the web.
