### Post by germanix on 2010-11-26
Any normal download of Ubuntu can be installed as a german version. When you install from the CD just choose "german" as the installing language. Alternatively you can after an install in english just change the language via language support. It will then download the "german package". Make sure you choose the option "use system wide" and all will be in german. Hope this helps.
