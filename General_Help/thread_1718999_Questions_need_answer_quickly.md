### Post by coldraven on 2011-04-01
You should always do a backup, especially when installing a new OS.
De-fragging Windows is also a good idea, because the installer will want to re-size the partition and you don't want bits of files in the way.
External USB drives are cheap, get one and start backing up!

### Post by grahammechanical on 2011-04-01
Make doubly sure that you do not tick the box that says Use entire disc.

The installer program will not make any changes to the hard disc without first reminding you of the choices you have made. Up until that point you can back out of the installation without any changes being made to the hard disc.

I found it useful to start the installation and then to back out as I got nervous. In this way I became familiar with what the installer was going to do. There are also guides that show screenshots of the install process.

Regards.
