### Post by fragos on 2007-06-27
First you need the following section, or something close, in /etc/X11/xorg.conf

Section "InputDevice"
	Identifier	"Synaptics Touchpad"
	Driver		"synaptics"
	Option		"SendCoreEvents"	"true"
	Option		"Device"		"/dev/psaux"
	Option		"Protocol"		"auto-dev"
	Option		"SHMConfig"		"true"
	Option		"MaxTapTime"		"150"
	Option		"MaxDoubleTapTime"	"150"
	Option		"VertEdgeScroll"	"1"
	Option		"HorizEdgeScroll"	"1"
EndSection

Lastly you need to add the synaptics touchpad to the server layout section.  Mine follows:

Section "ServerLayout"
	Identifier	"Default Layout"
	Screen		"Default Screen"
	InputDevice	"Generic Keyboard"
	InputDevice	"Configured Mouse"
	InputDevice     "stylus"	"SendCoreEvents"
	InputDevice     "cursor"	"SendCoreEvents"
	InputDevice     "eraser"	"SendCoreEvents"
	InputDevice     "pad"
	InputDevice	"Synaptics Touchpad"
EndSection

On my system I use a touchpad, laser mover and Wacom Graphire4 and switch betweeen them without frequently problem.  You may also want to install "gsynaptics" with gives you a GUI configuration interface that's easier to use.

### Post by speaker219 on 2007-06-27
Just to let you know, the line in xorg.conf that disables touchpad tapping looks like this:
Option "MaxTapTime" "0"
add that to the end of your mouse configuration
also, in your xorg where it says: "driver" "mouse"
change it from "mouse" to "synaptics"
