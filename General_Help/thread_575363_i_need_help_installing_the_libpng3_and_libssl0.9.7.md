### Post by yabbadabbadont on 2007-10-13
They will be installed automatically if needed by any other software that you install using apt-get, adept, aptitude, synaptic, ...

If you need them in order to compile a program from source, then use synaptic to search for them and install them.  Make sure that you install both the regular library and the *-dev version in that case.
