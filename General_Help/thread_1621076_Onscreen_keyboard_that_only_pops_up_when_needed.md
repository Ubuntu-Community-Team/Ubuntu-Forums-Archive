### Post by kerry_s on 2010-11-14
matchbox-keyboard & matchbox-keyboard-im

you need run the matchbox-keyboard-im, it's daemon to pop the keyboard up when you click a on a input field.

been some time since i played with it so read the help file & man page for it.

example:
**matchbox-keyboard-im --help**
