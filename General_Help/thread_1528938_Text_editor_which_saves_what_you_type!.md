### Post by Morbius1 on 2010-07-11
I'll admit you'd really have to like the other features available in gedit to do this but there is a way to autosave every so often and and to an already saved file:

(1) Make sure you have a /home/username/Templates folder. If you don't have one create it.

(2) Bring up gedit. Then immediately do a save to : /home/username/Templates/gedit.txt

(3) Now Right Click the desktop > Create Document > and "gedit" will be listed.

When you do that a "gedit.txt" file will appear on your desktop.
