How do I install fonts in Ubuntu 6.10?

---

### Post by thetorpedodog on 2007-05-05
First, you need to make a file in your home directory called ".fonts". This folder will be a hidden folder since it starts with a dot. Then, you can simply take font files and save them to this hidden folder. You may need to right-click on your file browser and select "show hidden files" to be able to see your .fonts folder. You can always hide the hidden files again after you have installed the fonts.
