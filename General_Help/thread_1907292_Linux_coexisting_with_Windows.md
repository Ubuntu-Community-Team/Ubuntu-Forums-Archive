### Post by MadsRC on 2012-01-11
You can do both.

Having 2 partitions with windows on one has the drawback that you can't have both systems running besides each other.

You can use VirtualBox to run a Windows (Virtual) PC inside your Linux machine. The drawback is that it consumes power, but you can have both systems running next to each other (Or actually, on top of each other).

### Post by KdotJ on 2012-01-11
Also, running Windows in a virtual machine will not give you great graphics specs, it won't be able to handle heavy games if that's what you're after. It is great though and you can snapshot the virtual machine and restore it to that point when you mess up in the future
