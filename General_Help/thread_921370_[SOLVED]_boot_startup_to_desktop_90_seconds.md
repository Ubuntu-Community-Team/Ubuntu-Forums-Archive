or it might be 60.

boot startup to desktop in 90 seconds is possible if you via enable auto login. 

I suppose you could have a simple user session profile with very controlled permissions that auto logins in the 90 seconds, and then change to a more versatile users session profile.

This is useful if you did encounter crashes, or other use, as in such a context it is advantageous to set box for auto login when attended, and uncheck auto login in by system > preferences > sessions manager if it is a consideration for being unattended.
