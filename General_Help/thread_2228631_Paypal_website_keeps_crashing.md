### Post by Impavidus on 2014-06-08
Not sure which of the many (national) paypal websites you mean, but I found one with a video on the background. It could well be that your Firefox, which is outdated, doesn't support that. The latest Firefox – and the one you get from the repositories now – is Firefox 29. I think that Firefox 16 was included on the install disk of Ubuntu 12.10 and was also the final release provided in the repositories of Ubuntu 11.04, which are both end-of-live. Ergo, I think you're running a dead version of Ubuntu. Your Firefox will have more problems (including security issues) than just being unable to load one of the paypal websites.

I recommend running a supported release of Ubuntu (14.04 is the latest one) or, if your computer can't handle that or if you don't like it, one of it's derivatives like Xubuntu. You can get the latest Firefox automatically from the repositories.
