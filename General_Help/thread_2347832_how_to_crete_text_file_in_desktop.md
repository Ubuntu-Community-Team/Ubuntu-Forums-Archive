### Post by howefield on 2016-12-29
Use nano ?

If you want to create a text file on the Desktop, use..

```
nano ~/Desktop/sample.txt
```

Change "sample.txt" to whatever you want the filename to be, enter your text, then Ctr+O keys to save, enter and Ctrl+X to exit. The key commands are at the bottom of the terminal window.

### Post by slickymaster on 2016-12-29
You can also use the **touch** command. in a terminal window run```
touch your_filename.txt
```The touch command will create an empty text file named your_filename.txt
