### Post by hawthornso23 on 2011-01-31
Lots of things can cause reboots. Memory problems and overheating can both cause spontaneous reboots. As can low voltage. 

check memory by selecting memtest from the grub menu

check voltage and temnperature by running sensors from a terminal.

Try eliminating these causes first.
