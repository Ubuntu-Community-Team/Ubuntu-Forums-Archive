### Post by Celauran on 2009-07-03
The difference (or, at least, the chief difference) is in desktop environment. Ubuntu uses GNOME, Kubuntu uses KDE, Xubuntu uses Xfce. Xubuntu is best suited to machines with limited resources (or people who find the other DEs too resource hungry). The difference between GNOME and KDE is really a matter of personal preference. Assuming you've got the space, you can easily install all three and choose which to use when you login.
