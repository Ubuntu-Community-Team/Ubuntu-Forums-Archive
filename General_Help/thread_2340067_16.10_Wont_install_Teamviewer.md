### Post by #&amp;thj^% on 2016-10-15
If you are using 64 bit Ubuntu 16.04 or higher system, then you must enable i386 architecture using the below command before continuing 

```
sudo dpkg --add-architecture i386

```
```
sudo apt-get update
```

Now install dependencies for TeamViewer.

```
sudo apt-get install libdbus-1-3:i386 libasound2:i386 libexpat1:i386 libfontconfig1:i386 libfreetype6:i386 libjpeg62:i386 libpng12-0:i386 libsm6:i386 libxdamage1:i386 libxext6:i386 libxfixes3:i386 libxinerama1:i386 libxrandr2:i386 libxrender1:i386 libxtst6:i386 zlib1g:i386 libc6:i386
```

Then install TeamViewer.

```
sudo dpkg -i teamviewer*.deb
```
That worked for me....hopefully it works for you.

### Post by #&amp;thj^% on 2016-10-15
Bear in mind I have had this installed since 16.04 was in development and now in 16.10.
So some time has passed since then...but I seem to remember struggling with the 64 bit .deb
Went through some old notes and found my install went like this.
```
sudo dpkg --add-architecture i386
```
You have already done this

```
sudo apt-get update
```
And this also is already done

```
wget http://download.teamviewer.com/download/teamviewer_i386.deb

```
I choose the 32bit .deb
```
sudo dpkg -i --force-depends teamviewer_i386.deb

```



In case “dpkg” indicates missing dependencies, complete the installation by executing the following command:

```
sudo apt-get install -f
```

complete the installation by executing the following command:

```
sudo teamviewer --daemon start
```
