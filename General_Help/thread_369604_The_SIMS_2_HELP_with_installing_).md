### Post by Maestro23 on 2007-02-24
Bumping your post so quickly is not a good way to get help.

You will need some software like Wine or Cedega to run windows games on a Linux box.

Cedega: [http://www.transgaming.com/index.php?module=ContentExpress&func=display&ceid=2&meid=-1](http://www.transgaming.com/index.php?module=ContentExpress&func=display&ceid=2&meid=-1)

Wine: [http://www.winehq.com/](http://www.winehq.com/)

*Wine can be downloaded from the Ubuntu repositories.  Cedega will cost you money.  Note, not everything will run under wine.

Good luck.
