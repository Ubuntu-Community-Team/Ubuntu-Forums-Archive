### Post by logix2 on 2020-03-18
[PDF Mix Tool]("https://scarpetta.eu/pdfmixtool/") allows merging multiple pages in a single PDF page. Use it to generate the PDF as you wish, then print it as normal.

Example screenshot from [Linux Uprising]("https://www.linuxuprising.com/2020/03/perform-common-pdf-editing-operations.html"):
