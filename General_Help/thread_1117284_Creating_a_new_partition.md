### Post by ShaunG on 2009-04-05
If you only have one partition it is NOT possible as you have to unmount the partition to be able to rezise it to create space for a new partition.

If you have a live cd, boot from that. Then in that go to Places->System->Admin->Parition Editor.

From there you can resize and create partitions.
