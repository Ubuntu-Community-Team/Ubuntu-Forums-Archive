### Post by ramjet_1953 on 2009-09-18
If you want to copy some DVD's, you will probably need to use something like K9Copy to convert from DVD-9 (Dual-Layer) to DVD-5 (Single Layer).

K9Copy does a good job of compressing and the resultant iso is almost indistinguishable from the original.

If your original DVD's are only DVD-5 you will need to use the Disk Copy function in Brasero to correctly copy the DVD.

However, if you just have some individual videos that you want to burn to a DVD, you will need to use a program like ISO Master, which is available in the repositories, to organize these files and create an ISO for burning.
