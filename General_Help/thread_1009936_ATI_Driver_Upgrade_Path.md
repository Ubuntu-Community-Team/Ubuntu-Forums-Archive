### Post by pseudonym on 2008-12-13
As the improved ATI driver is still fairly new I'd say it's worth upgrading. If you look at the release notes for each new driver you'll see that (in theory at least) bugs have been fixed and compatibility has been improved.

However, the potential sting in the tail, as with just about any software upgrade, is that new 'issues' might be created by a driver update.

But if everything is working fine for you then there's probably no need to install a new driver each month.
