### Post by chessnerd on 2010-07-09
If you want a browser that uses the same underlying system as Firefox take a look at Seamonkey Navigator and Conkeror in the Ubuntu Software Center under Internet > Web Browsers. SeaMonkey is based on the Mozilla project and Conkeror was originally an extension for Firefox and uses a very similar rendering engine.

If you just want an alternative browser, you might consider [Opera]("http://www.opera.com/browser") or [Chrome]("http://www.google.com/chrome"). Chrome is the second most popular browser for Linux after Firefox, Opera is third.

Also, if the issue is caused by the Ubuntu build of Firefox, you could download and install Firefox directly from [Mozilla's website]("http://www.mozilla.com/en-US/firefox/firefox.html").
