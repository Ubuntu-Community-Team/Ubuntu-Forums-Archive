### Post by Mark Phelps on 2011-08-09
Suggest you hold down the SHIFT key while booting. That's supposed to force the display of a GRUB menu.  

If that doesn't work, try disconnecting the external monitor and trying again.

Also, if you have a BIOS setting that determines which monitor (internal or external) is PRimary, try changing that to see what happens.
