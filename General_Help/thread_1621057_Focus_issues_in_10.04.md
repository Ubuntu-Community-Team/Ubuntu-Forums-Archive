I am rewriting this thread starter from scratch, since there was no response to the question asked, and it remains unresolved.

In just about any software of your choosing, you can **open**, **save**, and **save as** from the File Menu.

Starting with the Text Editor, gedit, open an existing text file, let's call "test.txt."

Choose Save As, and you can see the base name highlighted, the extension not included in the highlight.

Option 1:  Rename the file to something.txt by simply typing "something" on the keyboard (no quotes).
Optionally choose a different directory.
Save.

Option 2:  Choose a different directory and save the file in its original name there.

Option 3:  Here's the rub:  Choose a different directory.
The name is still highlighted as [COLOR="DarkRed"]test[/COLOR].txt
Type "newname" on the keyboard (no quotes).

If there is a long list in the directory, the directory jumps around to display the first of the names starting with each of the letters typed if there is a name to match.  The intended typing was supposed to go in the place of the name that was, and is, still highlighted.

IF FOCUS IS NOT ON THE NAME FIELD, THERE SHOULD BE NO HIGHLIGHT THERE.
