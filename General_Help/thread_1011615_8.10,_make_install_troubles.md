### Post by mc4man on 2008-12-15
Your build probably installed to /usr/local

If you wanted to install to /usr then you'd need to ```
./configure --prefix=/usr
```

Normally you should uninstall the old version first in either case.

If you want to remove (assuming you did a make install) then cd to build folder and run sudo make uninstall

Otherwise you could probably edit your menu item to reflect the new location (if that's where it was installed to

To *conside*r:

The latest version is available in jaunty and there aren't any dep issues to install to intrepid

You'd remove transmission-gtk and transmission-cli if installed and then do a direct download and install (upgrade) of transmission-common.

Then you'd do a direct dl and install for transmission-gtk and also transmission-cli if wanted


[http://packages.ubuntu.com/jaunty/transmission](http://packages.ubuntu.com/jaunty/transmission)
