/home/{user}/.config/google-chrome/Default/Bookmarks but this is a text file and I'm not sure it's wise to edit it. Tell Chrome to sync and your bookmarks will be saved in folder "Google Chrome". I sync multiple machines to the same Google docs account and it works well.
