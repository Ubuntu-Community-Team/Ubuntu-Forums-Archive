### Post by robsoles on 2010-09-19
[http://sourceforge.net/projects/recover/files/](http://sourceforge.net/projects/recover/files/) may be worth a look.

Researching the linux commands 'dd' 'df' & 'du' should yield good results regarding file recovery from the command line in terminal - I've used them but cannot recall right now (is my lunch break :)) how to use them properly. 

All file recovery relies heavily on that the absolute majority of file-space the deleted files was occupying hasn't been over-written yet. I fear it likely your data has been overwritten enough to make it unrecoverable but only trying to recover it will tell you if I am right.
