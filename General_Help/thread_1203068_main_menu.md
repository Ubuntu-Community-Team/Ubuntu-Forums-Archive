### Post by x33a on 2009-07-02
right click on the menu bar (top panel), click edit menus.

then check system -> preferences -> main menu.

if you lost the previous entry, you can create a new one:

under system -> preferences , click new item.

Type: application
Name: main menu
command: alacarte
comment: Change which applications are shown on the main menu
