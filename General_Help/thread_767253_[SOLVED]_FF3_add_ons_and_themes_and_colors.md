### Post by Bubba64 on 2008-04-25
If you have updated to Hardy and have add ons that are not recognized Nightly tester tools in Mozilla add ons will remedy some of these. 

Also in Hardy in synaptic is Gnome color chooser which allows you to color your basic menus and color changes while hovering and on your desktop

Mozilla has a FF theme download called persona's for Firefox which downloads for FF2 and early beta versions of FF3, and also the latest FF3.
[http://labs.mozilla.com/2007/12/personas-for-firefox/](http://labs.mozilla.com/2007/12/personas-for-firefox/)

Nightly tester tools and Persona's for Firefox install in your add on menu for easy removal if needed, and gnome color chooser is removable in synaptic if not needed.
