### Post by phillw on 2010-08-20
Hi, 
Lubuntu has automatic updates turned off by default, as on slower CPU's / Low RAM machines it can cause the user to think the machine has 'locked up' / 'hung' as it performs that task. As pointed out above, it can be turned on if you wish, but is off as default for that reason.

Adding Update Manager as both the icon and task is covered in [Method 2](https://wiki.ubuntu.com/Lubuntu/DocumentationHelp/KeepLubuntuUptoDate), for those who wish to use the manual method, [Method 1](https://wiki.ubuntu.com/Lubuntu/DocumentationHelp/KeepLubuntuUptoDate) has that.

Regards,

Phill.
