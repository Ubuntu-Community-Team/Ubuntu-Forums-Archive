### Post by Vadi on 2008-11-12
> **askyourpc.com said:**
> What things should a new Ubuntu user know?

If they're just starting out - an easy way to install Ubuntu is from Windows via Wubi: [http://wubi-installer.org/](http://wubi-installer.org/)

After they've installed, you can get Flash, some codecs, java and etc. working by going to Applications - Add/Remove, select "All available applications" in Show, search for "ubuntu restricted extras", and install that.

Do not use Automatix, Ultramatix, or the "Ubuntu Ultimate Edition", they have a high chance of breaking your ubuntu and ruining the experience.

You can easily theme your ubuntu by going to system - preferences - appearance. there are tons of themes available at [http://www.ubuntu-art.org/](http://www.ubuntu-art.org/) and [http://www.gnome-look.org/](http://www.gnome-look.org/). to install a theme, drag the downloaded .tar.gz file into the themes window.
