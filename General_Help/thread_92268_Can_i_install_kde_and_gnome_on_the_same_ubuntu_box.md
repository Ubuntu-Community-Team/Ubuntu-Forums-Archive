### Post by RAOF on 2005-11-19
Yup, you can get KDE and Gnome on the same Ubuntu system easily enough.  Install the (meta-)package "kubuntu-desktop" in synaptic (or apt-get), and you'll get the latest stable KDE, and all it's default applications, from the repositories.  As to getting the latest (unstable) KDE, you'd want to look at the kubuntu page [here]("http://www.kubuntu.org/announcements/kde-35beta2.php").
