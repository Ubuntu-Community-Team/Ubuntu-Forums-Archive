### Post by vanadium on 2013-12-24
As far as I am aware, Unity involves that you are using compiz, although graphic features may be turned off on cards that do not support them. Using Compiz Config settings manager, one may indeed change the task switcher. I think that nowadays, you first need to install compiz-plugins in order to have the other window switchers available.

I use the switcher that comes with Unity, but I turn off the automatic display of application windows. This way, Alt+tab only switches between applications, Alt+` switches between windows of the same application.
