Hello and welcome to the forums.

For new users, I usually recommend the LTS (Long Term Support) version. The current LTS version is 10.04. 

Did a quick search and came across this link: [http://ubuntuforums.org/showthread.php?t=1510574]("http://ubuntuforums.org/showthread.php?t=1510574")

If you end up choosing to install 10.04, you will need to follow the instructions listed there to get the graphics to work.

If you choose to go with the current version (11.04), there appears to be some good news: [http://ubuntuforums.org/showthread.php?t=1754775]("http://ubuntuforums.org/showthread.php?t=1754775")
