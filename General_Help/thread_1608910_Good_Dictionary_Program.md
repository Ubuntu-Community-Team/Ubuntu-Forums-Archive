### Post by ufugu on 2010-10-29
[Artha]("http://artha.sourceforge.net/wiki/index.php/Home") calls itself a thesaurus but it's also a great dictionary.

You can use it as needed, but optionally you can run it at startup and then select any word in any GTK app to get a definition.
