### Post by andymorton on 2010-10-31
Once you've downloaded the theme right-click on the Gnomenu icon, choose properties and then click the install button in the 'menu' section. Navigate to where you downloaded the theme to and select it. It should install. Make sure your theme is chosen in the drop-down menu and then click ok. If you get a message saying Gnomenu needs to reload and/or restart click yes. After that you should have you new theme.
