### Post by robert shearer on 2010-07-23
4) moving top panel.

Right click on space in the top panel and in the window that opens select 'properties'.
When that opens use the drop down box in 'orientation' to select 'left' (just temporary) Don't close the box!
The panel will move to the left of the screen.

Now do the same for the bottom panel, only select 'top'.(new box,right click etc,etc.)Now close the second box.

Back to the original box and select 'bottom'.

Now you should have the panels reversed with the top and all the icons at the bottom and the window indicator at the top.

Before closing the properties box for the bottom panel I usually expand the bottom panel to 45 pixels.

### Post by theOGRE on 2010-07-24
Ok I'm back. Open up a terminal (menu->accessories->terminal) then enter this code.

```
gconftool -t bool -s /apps/nautilus/desktop/trash_icon_visible "true"
```if you don't already have it, you can install gconf-editor from the repositories. It's a gui that let's you look at a whole lot of settings, and tweak around. On that program you would do...
apps->nautilus->desktop  and then click the checkbox next to 'trash_icon_visible'. That ought to do it.
