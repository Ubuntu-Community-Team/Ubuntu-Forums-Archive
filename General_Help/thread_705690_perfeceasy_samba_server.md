### Post by Lunitik on 2008-02-23
> **fedex1993 said:**
> is there an easy samba server guide ou there anywhere

If all you need to do is share files and access other computers, you can simply use Places > Network along with System > Administration > Shared Files.

If you're on a home machine, you really don't need to set up a samba domain, just a workgroup, and that's what Shared Files will set up for you.
