Next you will be asking, how do I install fonts?

I use some fonts from my old Windows 98 installation and a specialist font that is needed to read a language that is used on a web site that I visit. My method is to copy the font into my home folder. Then click on it and select Open with Font Viewer and then click the Install Font button.
