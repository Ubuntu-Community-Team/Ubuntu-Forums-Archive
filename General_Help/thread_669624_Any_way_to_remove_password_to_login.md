### Post by Gillingham on 2008-01-16
Yes if you are using gnome go to System > Administration > login window. That will open up a window (you will have to enter your password). Select the security tab and then you can select automatic login.

### Post by kdwillia on 2008-01-16
You can set the PC to automatically login.   Technically, you still have a login and password, but the user does not see them.

Assuming "Gutsy" install

Go to System > Administration > Login Window

Tab Security and the checkbox for Automatic Login

Hit the dropdown and choose the user account for that PC.

You will still need to type in the password if a system function is selected, but you will not have to "login" to the PC
