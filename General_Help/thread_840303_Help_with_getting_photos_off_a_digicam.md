### Post by owlgorithm on 2008-06-25
The easiest way might be to remove the memory card and read it through a USB card reader. Then, Ubuntu sees the files the same way it would see a flash drive, and transfer speeds are sometimes faster this way.

### Post by timcredible on 2008-06-25
in digikam, go to camera->add camera, and let it auto-detect your camera.  if that doesn't work, then change your camera's usb mode and try it again - sometimes the pictbridge mode works better.  if not, check [here]("http://ubuntuforums.org/showthread.php?t=323196").

also, make sure you install kipi-plugins if you're using digikam, it'll give you a bunch more good plugins.
