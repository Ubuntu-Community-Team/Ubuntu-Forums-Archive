### Post by 3rdalbum on 2009-08-03
If you've "never used Linux" and you're "awesome with a mouse", then you will want to install Ubuntu (or Xubuntu) desktop.

Ubuntu Server is command-line only, and is really meant for big iron, not for "pico-iron".

1. Use Ubuntu or Xubuntu Desktop. What version? The latest, of course.

2. Just choose the CF card as the destination disk in the installer. I'm not sure about your RAID - you might need to use the Alternate CD installer in order to do that.

3. When you say "an iTunes server" I'm guessing what you really mean is what's called DAAP Music Sharing. Any computer on the network can access the music library on your server. At first I used Rhythmbox to do DAAP sharing, but that requires leaving Rhythmbox open on the server - so I found a program in the repositories called Tangerine that (I believe) runs when your server starts up. It is configured through a graphical interface that doesn't need to keep running.
