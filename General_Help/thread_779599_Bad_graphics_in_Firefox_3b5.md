### Post by Enlitend on 2008-06-14
Try this :

In Firefox-->Edit-->Preferences-->Content
-Default font = DejaVu Serif : Size = 16

Click on Advanced...
Proportional = Serif : Size = 16
Serif = DejaVu Serif
Sans-serif = DejaVu Sans
Monospace = DejaVu Sans Mono : Size=12
Minimum font size = 12

and untick the check box that says " Allow pages to choose their ..bla blabla..."

Regarding the crappy graphic in firefox, before you want to zoom up the text, make sure you go to:

View-->Zoom 

and click on the text that say "Zoom Text Only".
