### Post by herteljt on 2008-04-22
If you check out the main Ubuntu website you can read up on the differences between the different versions.  The difference that you will see is the desktop manager.  Ubuntu uses Gnome, while Kubuntu uses KDE and Xubuntu uses Xfce4.  I would start with Ubuntu (unless your machine is quite old then Xubuntu might be a better option).

Here is a useful page for learning about the gnome desktop:

### Post by mister_pink on 2008-04-22
Ubuntu - uses gnome desktop, straightforward to use and simple interface.
Kubuntu - uses KDE, another desktop environment that looks different, is more customisable. Some people prefer it but its a massive ongoing debate. To confuse things more there are two version on the go at the moment, KDE3 and KDE4. 4 is newer, but 3 is more stable.
Xubuntu - uses xfce as a desktop, similar to gnome but meant to be slightly more lean on the system resources
Edubuntu - same as ubuntu but with educational packages preinstalled.
Mythbuntu - used for media centre PC type stuff.

Basically, just get ubuntu, and you can easily add KDE to it at a later date to try it out.
