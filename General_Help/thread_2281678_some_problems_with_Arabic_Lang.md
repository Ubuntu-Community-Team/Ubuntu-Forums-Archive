### Post by grahammechanical on 2015-06-08
It depends on how old those Microsoft Word documents are but did you create them using a special Arabic TrueType font. If so, then you may need to get a copy of that font on to Ubuntu and install it using Font Viewer.

If that gets the documents displaying the correct way in Libreoffice then copy the text into a new Libreoffice Writer document and format the text with a Libreoffice font and save it.

You may also need to install Microsoft's core fonts. Search Ubuntu Software Centre for mscorefonts. That will install the MS core fonts installer which will in turn install a selection of Microsoft fonts. You will need to accept the Microsoft End User Licence Agreement (EULA). A dialog will appear but it will appear behind the Ubuntu Software Centre application window and you will not see it unless you move the software centre window to one side. Accept the EULA and the download will proceed.

Regards.
